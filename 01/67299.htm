james- say: Hello Stanny,

 Thank you for using Microsoft Windows Vista Forums.

 Are you trying to use Diskpart to assign/change a letter for a Driver?  If so,  I have added some information below that will assist with that.  If this isn't what you are looking for please let us know.  Thanks!

Driver letter assignment in Windows Vista…

 

You can use Diskpart to assign driver letters, but it can be complex and cumbersome for the average user.

 

I’d recommend that you use Disk Management if you would like to assign a driver letter to a given device or volume. Disk Management has an advantage over Diskpart, for the average user, as it is run from a GUI (Graphic User Interface) versus the Command Prompt.

 

I’ve included instructions outlining how to use Disk Management to change a driver letter. However should you still want to use Diskpart I recommend you review the information found here.

 

Using Disk Management to change drive letter assignment…

1.     Click Start, right-click Computer, and select Manage.

2.     Click on Continue or provide Administrator privileges if prompted.

3.     In the left pane click on Disk Management which is located under the Storage category.

4.     If you wish to change the drive letter assignment for a removable storage device or volume:

a.     Right-click the appropriate volume, and then select Change Driver Letter and Paths.

b.     From the drop down list select the drive letter you would like to use.

c.     Please read the on-screen warning message, and click Yes if you wish to proceed.
