Windows 7 XP mode - Microsoft Community

jamesjoey say:

Windows 7 XP mode

With Professional, I believe one gets the XP mode feature.

Can I play old dos games in this XP mode?

sanmartin say:

You can play most old games in vanilla Windows 7. The only notable exception are games developed for a real DOS environment (= only compatible up to DOS 6.22, nothing else), for those you need a DOS emulator (e.g. DOSBox). Betrayal at Krondor, Lands of Lore - Throne of Chaos, Ultima Underworld, SystemShock are examples of gems which aren't compatible with any "protected mode" used by newer versions of Windows. They are perfectly playable by emulation, either via DOSBox or via game-specific emulators (e.g. Exult for Ultima VII, a game that was unplayable even on XP). All other games need the newest DirectX runtime installed (that's not the same as the DirectX included in Windows), they may need a certain compatibility mode to play them (right-click the game's *.exe file, properties > compatibility, try what works), some may need you to disable some graphical components of Windows before playing (disable desktop composition, set your theme to "Windows classic", etc.).

The Windows XP mode is thought for business applications, not for games. It doesn't support most of the graphical features even an older game may require. (And if I may repeat that: real DOS games weren't playable on XP, too, so an XP mode will add nothing to that fact.)
