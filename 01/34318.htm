ronnie vernon say:

Which folder can I copy existing shortcuts of folders/files to show up on Jump Lists, before accessing All Programs list?


Hi

The Jump Lists are not folders.

Depending on the Pinned Program, you can drag and drop files or shortcuts onto the pinned icon to pin the item to a Jump List.

For instance, if you have Windows Explorer pinned to the Taskbar, you can drag a Folder to the pinned icon and hover it over the WE icon. You will a message "Pin to Explorer.exe".

If you have Internet Explorer pinned to the Taskbar, you can drag a website shortcut and pin it to that program. If you are viewing a website, you can also drag the Icon at the beginning of the Address Bar and pin it to the IE Icon.

If you have a program pinned to the Start Menu, you can click the Arrow at the right side of the pinned program to expand. In the Recent Items list, hover over an item and click the Pushpin Icon to pin the item to that programs pinned Jump List.

Hope this helps.

malacath say:

Activision games like Ghosts and destiny don't use the standard xbox live ports which is why the nat can be open on the dashboard but moderate or strict in the game.

When you play ghosts or destiny the game will use demonware port mapping to open required ports via UPnP.

Some routers dont seem to be compatible with demonware port mapping so the NATs in each game will be strict or moderate.

I had to stop using my bt homehub 5 router for this reason.

If your router is not compatible with demonware then you need to use port forwardin to open the nats.

COD ghosts requires port 3076 to get an open NAT.

Destiny beta seemed to require port 1200 and this may be true for the full game.

Hi Ramon. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

Basic Display driver is only a placeholder driver which does not support good resolution or Sleep/Hibernate. It is only meant to be there until you can get the actual Display driver installed, or if it 's really old hardware may be the only one to use even though it's undesirable.

Update the Display driver from the PC maker's Support Downloads web page for your exact model number, HP Serial Number or Dell Service Tag - from the sticker on the PC.

While there check also for newer chipset, BIOS, network, sound, USB3 and other drivers, comparing to the ones presently installed in Device Manager reached by right clicking the Start Menu.

if this doesn't give you the latest or ideal driver for what you need, compare it to the driver offered by the Intel driver update Utility here: http://www.intel.com/content/www/us/en/support/...
or the Nvidia Update utility here: http://www.nvidia.com/Download/Scan.aspx?lang=e...
or the AMD autodect utility here: http://support.amd.com/us/gpudownload/windows/P...

For Display issues one fix that is working is to Roll back or Uninstall the Driver on the Display Device > Driver tab, restart PC to reinstall driver.

You can also try older drivers in Device Manager > Display device > Driver tab > Update Driver > Browse > Let Me Pick.

Then you will know you've tried everything in addition to Windows Update drivers. There are even more steps here for Updating a driver including tracing the hardware on unknown devices: http://answers.microsoft.com/en-us/windows/wiki...

If you still don't have the brightness slider in System Settings go over this checklist to make sure the install is set up correctly, optimized for best performance, and any needed repairs get done: http://answers.microsoft.com/en-us/windows/wiki...

Start with Step 4 to turn off Startup freeloaders which can conflict and cause issues, then Step 7 to check for infection the most thorough way, then step 10 to check for damaged System Files. Then continue with the other steps to go over your install most thoroughly.

If nothing else helps you can run a Repair Install by installing the Media Creation Tool and choosing from it to Upgrade Now. This reinstalls Windows in about an hour while saving your files, apps and most settings, solves many problems while also bringing it up to the latest version 1809 which you need anyway and by the most stable method. http://windows.microsoft.com/en-us/windows-10/m...

Check if you have brightness slider(s) in Settings > System > Power Options, then Additional Settings, then Change Plan Settings. This slider is also dependent upon display driver.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
