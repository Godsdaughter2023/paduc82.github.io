Networking

I have five Xbox One's.  How do I get the NAT open on all five?
arminatorx say:

In the Xbox One's network settings you'll find a button for advanced settings. 

Make sure that each and every Xbox One has a DIFFERENT port for Xbox Live. 

For example, the first Xbox has Port 3074 (make sure you don't have a Xbox 360 that uses the same port). Then the other five consoles need to use a different port. 

After each console has gotten a different port assigned, make sure that each console has a fixed, static IP set up in your router. The consoles MUST NOT change their IP addresses (unless you want to edit your port forwarding rules every time the IP changes). 

Then, with each Port number and IP address, set up port forwarding rules in your router to each specific console.
