Forget about the BUP and IFO files...the important video files
are the VOB files.

For reference...the following articles explain the various file
formats found on a Video DVD:

What's on a DVD?
http://server4.doom9.org/index.html?/dvd-structure.htm

How to play VOB / IFO / BUP files
http://www.afterdawn.com/guides/archive/how_to_play_vob___ifo___bup_files.cfm

If you copy the VOB files to a folder on your hard drive and
rename them to .mpg you should be able to play them in
Windows Media Player.

This way you can determine exactly which file contains the
content you wish to edit.

It's possible that you may be able to import the renamed
.mpg files into Movie Maker as is but if not...you will have to
convert them to the WMV format before importing.

The following freeware can convert VOB to WMV...

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Format Factory
http://www.pcfreetime.com/
(the file you want to download is : >FFSetup250.zip<
(FWIW...when installing...you may want to
uncheck all the boxes on the final screen)

First...after downloading and installing Format
Factory...you may want to open the program and
left click the 'Option' button on the toolbar and
select an 'Output Folder' / Apply / OK.
(this is where you will find your files after they
are converted)

Drag/drop your VOB files onto the main screen...

Select 'All to WMV' / OK...

Click...Start...on the toolbar...

(Have patience...the conversion maty take a while)

That should do it...

Good Luck...

Note If the computer is connected to a network, network policy settings may prevent you from following these steps. 0 - Turn off all locks.

3 - Activa Block Mayús and Block Num

6 - Activate Num Lock and Pause key.

7 - Activate all
