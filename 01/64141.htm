In Windows 7, an administrator is more akin to what used to be referred to as a power user rather than an actual administrator of the computer. The rights you require are now referred to as elevated privileges and are granted by right clicking your menu item or desktop shortcut and selecting Run as administrator. This new approach to what entails an administrator is probably due to a large number of end-users assigning themselves administrator rights not because they ever actually needed them, but simply because they could have them. This by itself is not an issue, but when the same end-users get onto malicious web sites and click everything that says Free Download, any malware or viruses that result from a poor choice have the administrative rights of the end-user who initiated the action.

It starts to sound like something written by Joseph Heller.
With all of that stated, you really should be asking yourself what operation is Cubase attempting that requires elevated privileges. These are usually registry writes or system .DLL replacement (during installations). Some games require these elevated privileges to run and some programs require them to make internal modifications (e.g. removing an add-in from Word), but they should not be required for normal program operation like saving a file unless you were attempting to save to a folder you did not have write permissions to.
