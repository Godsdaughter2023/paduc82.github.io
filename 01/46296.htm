There are two features -

1. Lock the Taskbar - This will not allow you to resize the task bar and move the taskbar using mouse. (you can still move Taskbar by Taskbar > Taskbar Settings > Taskbar location on screen and choosing the position even if it is locked. Point is that it will not allow mouse to move the Taskbar)
