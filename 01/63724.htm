    vibroplex say:

    Some NAT problems are subtle and don't appear in the Test Xbox Live Connection test. Most likely there's a port blocked... ports can be blocked in your gateway, or at your ISP. Since the Xbox 360 uses different ports than PCs & Macs use, you can't see the effect of a port blockage except on the Xbox. A sequential power cycle is the first step in clearing a port blockage:

    1) turn off the Xbox then

    2) unplug the power from the D-Link gateway

    3) Wait at least 30 secs then replug the gateway power.

    4) Wait 'til the light on the front of the gateway which says ONLINE, READY, or DSL is on solid (not flashing), then plug in the router.

    6) Turn on the Xbox.

    www.pcwintech.com/port-forwarding-d-link-dsl-2680-d-link-firmware may provide useful help if that does not resolve the problem. Good luck!

    Hi SudeepShakya,

     

    Thank you for choosing windows 8 and joining us on our community.

     

    From the issue description, I understand you want to know if you can boot in to Windows 8 after deleting the partition for Windows 7 and merging with windows 8.

     

    I would like to inform you that you can delete the partition and merge it with Windows 8 partition.  But it might be risky.

    So I would suggest you to create a back up disk of all the data and the Operating system before doing this.

    Here are the steps to delete the partition and merge it with another one.

    a.      Start or restart your PC, and boot into Windows 8.

    b.      Open Disk Management by typing diskmgmt.msc in search bar.

    c.       In Disk Management, delete the partition that you installed Windows 7 on so that is now unallocated space.

    d.      If you like, you could either create a new partition to use with the unallocated space, or extend the Windows 8 C: drive into the unallocated space to add it back to the C: partition

    e.      When finished, restart your PC and only have Windows 8 installed and listed in the Windows Boot Manager screen.

    f.        If you can see both the Operating system on the screen, then boot in to Windows 8.

    g.      Press Windows key + R and type msconfig and press enter.

    h.      Under boot tab, select windows 7 and click on delete and restart the computer.
