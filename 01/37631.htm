ronnie vernon say:

Hi

When you search for a file or folder in Windows Explorer, the path to the location of each result should be shown just below the found file or folder. Here is a screenshot.

http://i196.photobucket.com/albums/aa86/rvmv/SearchResults.png

If you have one of the Icon views selected, click the Change your view drop down menu at the top/right of the window and select List, Details, Tiles, or Content view (The screenshot has Content selected).

Search is a stand alone component that creates a Virtual folder that displays the results. It is not part of the folder heirarchy unless you save the search.

If you right click one of the results and select Open Folder Location, it will take you to the folder where the item is located.

To navigate from within any particular folder:

To go UP, press Alt+Up Arrow key.

To see the path to your present location, click an empty spot in the Address Bar.

Click the Arrow on the right side of any folder in the Address bar and the Parent folder will be displayed in Bold text.

Hope this helps.

Thank You for using Windows 7
