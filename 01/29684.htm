frederik long say: The native image restore process is rather basic in that it is unable to restore an image to a target disk if that target disk is smaller than the source disk. You need to use a more mature imaging tool, e.g. Acronis True Image. Perhaps some free image tool might do this too.

seanmurray47 say: Will this work if I shrink the size of the windows partition down to <128 GB?

frederik long say:
Will this work if I shrink the size of the windows partition down to <128 GB?

Yes, it will, but the native Disk Manager suffers from the same problem as the native image creation tool: It is very basic. You might find that it will not let you shrink the current 500 GByte partition to less than 250 GBytes. Mature partition managers do this with ease. I use Acronis Disk Director (costs money) but I occasionally see references to Easus Partiton Manager (free) in these forums.
