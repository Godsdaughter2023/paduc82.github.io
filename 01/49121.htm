Brenda,

You've probably reintroduced the virus by running System Restore.

When System Restore runs, it backs up Windows system files. But viruses also belong to that group and get backed up as well unfortunately.

First troubleshooting steps.
  1. Hit Windows key + R to launch the Run command.
  2. Type, or copy/paste: services.msc and click OK. This will take you to the Services Management Console. 
  3. In the list of services, find DCOM Server Process Launcher and double click it.
  4. In the General tab which opens by default, click the dropdown arrow in the menu called "Startup Type" and change "Automatic" to "Disabled".
  5. Click Apply/OK to exit.

I should warn you that some Windows functions won't work anymore with DCOM disabled, but you need to prevent the PC shutting down suddenly while you're troubleshooting.


Next steps.

  1. Hit Windows key + Pause/Break to take you to System Properties.
  2. Click the "System Restore" tab and then checkmark the box where it says "Turn off System Restore on all drives".
  3. Confirm when you see the prompt. 
  4. Click Apply/OK to exit.
This is going to delete all your previous Restore points, but since you don't know which of them contains a backup of the virus, this is the only way to ensure it's been removed. After we fix the problem, you can re-enable it of course.

The next step is to run Malwarebytes again. It should remove the infection without a problem. When you reboot, it subsequently removes the infected Registry keys as well so you don't have to worry about those.

After the system reboots, go back into the Services menu and re-enable DCOM to "Automatic".

You then need to monitor the machine to make sure it doesn't shutdown again without warning.

Provided it doesn't, you can re-enable System Restore by removing the checkmark.

Let me know how you get on please.
