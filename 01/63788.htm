cmdrkeene say:

Remote Desktop would let you use the other computer.  However, it's not going to be very useful for image editing or gaming over remote desktop.  The reason is that remote desktop sends back a video image of what the other computer is showing.  That video image lags a little bit, so you might try to turn in your video game, but then it will take a second before you actually see it turn.  Remote Desktop is really only useful for things like files and documents, typing emails/reports and stuff of that kind.

Likewise, picture-editing over a remote desktop would be irritating because you would try to draw a line (or something) and then the remote screen would lag behind what you're drawing because it would take a moment for the video image to come back to you.

This lag is less when both computers are running on the same network, but it's still going to be noticeable, especially on a wireless connection.
