@echo off

set regKey1=IconStreams
set regKey2=PastIconsStream

taskkill /IM explorer.exe /F

:verify-PastIconsStream

echo Registry key "PastIconsStream" already deleted.

goto verify-PastIconsStream

:PastIconsStream

echo.

@echo off

Hi Gaurab,

I suggest you to perform the methods below to check if there are any system file corruption that may have caused this issue to arise.

 

Method 1: SFC

System File Checker [SFC] scan on the computer which will replace the missing or corrupt files& check if the issue persists.

a. Right click on the Start Menu, locate and select Command Prompt (Admin).

Note: If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

b. Type the following command, and then press ENTER:

sfc /scannow

 

Method 2: DISM

Follow the steps to Run DISM Commands:

1) Press Windows + x on your keyboard.

2) Select Command prompt (Admin) to open Administrator Command Prompt.

3) In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:


     dism.exe /online /cleanup-image /scanhealth
     dism.exe /online /cleanup-image /restorehealth
