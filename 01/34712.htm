Login Information

Gateway - 192.168.1.1

Username - N/A

Port Forwarding

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway, 192.168.1.1 into the address bar of the internet browser. 
  4. The router will then prompt you to enter the username and password. There is no username and the default password is 1234.
  5. Click NAT from the menu on the left.
  6. Select SUA Only click the Edit Details.
  7. Enter 3074 for  both the Start Port No. and End Port No. in an empty row.
  8. Enter the IP Address of the console in IP Address column of the same row you entered 3074.
  9. Click the Save button at the bottom of the screen.

 

Disable Port Forwarding

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway, 192.168.1.1 into the address bar of the internet browser. 
  4. The router will then prompt you to enter the username and password. There is no username and the default password is 1234.
  5. Click NAT from the menu on the left.
  6. Select SUA Only click the Edit Details.
  7. Delete any entries referring to port number 3074.
  8. Delete the IP Address of the console from the same row.
  9. Click the save button at the bottom of the page.

cmdrkeene say:

Taskbar thumbnail caching is an Aero video card cache (desktop window manager) and unrelated to file thumbnail in terms of both generation and storage.

But no, you can't enable thumbnail cache only in a specific folder, only in the scopes you already mentioned.  The functionality just doesn't exist to track it on that level.

Even with all full defaults, you should not be getting any locking.  Any well-written program should abandon modification request is made using opportunistic locks to get out of the way when an application you're using wants to do something with a file or folder. But not all programs are as well-written as we might like.  Still I'd be interested to know what program you are using that is getting locked.  The thumbnails shouldn't be able to cause this in Windows 7 and later.
