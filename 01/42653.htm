Hello!

The CPU speed of the other Computer seems quite slow! Nevertheless it could be also a latency issues that results from:

a.) The distance between the 2 Computers

b.) ISP related technical issues

c.) Local issues ( one of the Computers causes latency issues in regards of hardware or internet connection settings)

add a) It is possible that both computers suffer from internet high latency because of far distance to each other. This is most likely ISP related.

add b) Running a speed & ping test can help you to analyze issues with your Internet connection.

add c) It is possible that your Hardware has issues or some related settings of your internet connection got manipulated, are malfunctioning or are not optimally set.

Please read this guide for more info on the topic: https://lag-guru.com/blog/100-tips-tricks-to-fix-lag-successfully/ The 100 tips and tricks are a bit gaming related but the majority of solutions and info applies to VOIP applications such as Skype as well.

In order to test the Computer and internet connection performance you can download Latency Optimizer from here (https://www.badosoft.com/latency-optimizer.php)

1. Use the integrated Performance Recorder and record a screen-sharing session for about 10 – 15 minutes. Stop the recorder and look at the log file. If you see any CPU, RAM, HDD spikes you know there could be the issue.

2. Run a speed & ping test on both Computers and check the results. It is possible that one of the Computers has bandwidth or ping issues.

3. You can also try to optimize some internet connection settings and latency.

Open a command prompt. (Run a command prompt as Administrator in Windows vista/7/8)
Copy the below command line and Paste.

echo Start
echo #
echo ######################## Default dirctory for x86 x64 ########################
echo #
echo this command is default system32 directory for x86 OS or x64 OS
cd %windir% & cd system32
reg delete "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v "TurboPC EX" /f
reg delete "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v "TurboPC EX" /f
echo #
echo ######################## Change the dirctory for x64 ########################
echo #
echo this command is x86 application's registry for x64 OS
cd %windir% & cd syswow64
reg delete "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v "TurboPC EX" /f
reg delete "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v "TurboPC EX" /f
echo End

Close the command prompt.
