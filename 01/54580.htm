ken blake say:

No, Microsoft no longer supports Vista.

If you want to change to a newer version of Windows (and as far as I'm concerned, you should), bear the following things in mind:

1. On such an old computer your hardware configuration is likely to be inadequate for decent performance on a modern version, and you will need to at least upgrade the hardware, if not replace the entire computer.

2. If you move to Windows 10, your old computer many not even be compatible with it.

3. You can not upgrade to a modern version of Windows. It requires doing a clean installation, and doing a clean installation means you will lose everything on the drive, including the lot of information on it you don't want to lose.

4. So before doing a clean installation, you should copy all day to to an external device, not an external hard drive. You can then copy it back, after the clean installation.

5. Your statement "it has a lot of information on it I don't want to lose" suggests that you do not do regular backups of that data. If that's the case, you've been playing with fire. As has often been said, it's not a question of *whether* you will lose everything on the drive, but *when*.
