Windows XP won't recognize my 4gig of memory how do I get it - Microsoft Community

eloisepacheco say:

Windows XP won't recognize my 4gig of memory how do I get it to?

I put 4 Gigabytes of memory in my comp...Windows XP isn't showing this memory upgrade.  Still shows 2.75 gigs. How do I tell Windows I have upgraded the Memory to 4 gigs.
ken blake say:
I put 4 Gigabytes of memory in my comp...Windows XP isn't showing this memory upgrade.  Still shows 2.75 gigs. How do I tell Windows I have upgraded the Memory to 4 gigs.

 

All 32-bit client versions of Windows (not just XP/Vista/7) have a  4GB address space (64-bit versions can use much more). That's the theoretical upper limit beyond which you can not go.

 

But you can't use the entire  address space. Even though you have a  4GB address space, you can only use *around* 3.1GB of RAM. That's because some of that space is used by hardware and is not available to the operating system and applications. The amount you can

use varies, depending on what hardware you have installed, but can range from as little as 2GB to as much as 3.5GB. It's usually around 3.1GB.

 

Note that the hardware is using the address *space*, not the actual RAM itself. If you have a greater amount of RAM, the rest of the RAM goes unused because there is no address space to map it to.
