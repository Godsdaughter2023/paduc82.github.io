The solution is very simple but does require going into the registry.

Ok I know it says Vista but I took a chance and tried it on Windows 7 64bit and it worked.
Please note in the registry path it states a key folder called 13, that is actually a variable as I have Family Pack 5 and my 13 was called 11. So it appears this issue is also on Quattro Pro 11, 12, and 13 maybe even older and newer versions too.

"Answer ID   760250   -       Why do I get a Visual C++ Runtime Library error when launching Quattro Pro®?

   

Some users report intermittent errors, such asVisual C++ Runtime Libary error causing an Abnormal Program Terimination when launching Quattro Pro® on Windows VISTA.

This issue has been reported to our development team.

   

To workaround this issue:

  1. Click Start, Programs, Accessories and Run.
  2. In the open window, type:  regedit and click OK.
  3. Choose the following path HKEY_CURRENT_USER\Software\Corel\User Assistant\13\Recent Work\Quattro Pro\
  4. Right click on the Quattro Pro folder, select delete.
  5. Say yes when prompted to delete the key.
  6. Close the Registry Editor
  7. Launch Quattro Pro"
