If this problem took place only recently, run System Restore as follows.

aakkam22 say:

Hi,

You should use a wired mouse and keyboard while you install Windows 10 to ensure you can control your computer at all times. After the installation finishes, you can reconnect your wireless hardware and install the drivers for it.

Here is how you should prepare for the installation:

1. Obtain your Windows 7 product key. You can do this by downloading and installing Magic Jellybean, which will retrieve it for you. It is freeware.

2. Visit Acer's website and check that your computer has drivers for Windows 10. Download the basic drivers (network, audio, video, etc.) and save them to a flash drive in case Windows 10 doesn't have drivers for your hardware. You should, in the very least, download the drivers for your network device so that you can access the Internet.

3. When you've secured the drivers, the next step is to secure your data on your current system. I would suggest simply copying all your documents, photos, videos, whatever else to an external hard drive or flash drive, and then copying them back when Windows 10 finishes installing. 

4. Next, download the media creation tool. This tool will help you download and install Windows 10. When you open it, select the Upgrade Now option and click Next.

5. When you get to the Ready to Install page, select Change what to keep and then click Keep Nothing. This step is important because it means you're performing a clean install of Windows 10. This ensures that the end result is fast and doesn't have any carried-over corruptions from your old system.

6. Windows 10 will install. When it's finished, install any drivers that weren't installed automatically. You can also type your old product key after Windows 10 installs in the Settings app. Just go to Update and Recovery > Activation and type your Windows 7 product key to activate.

Whether or not your old hardware works on Windows 10 depends on if the manufacturer has drivers for Windows 10. You can visit the manufacturer's website to check this information. For example, if I had a Cannon scanner and wanted to know if it worked with Windows 10, I would go to Cannon's website and check my model on their website for Windows 10 drivers.

Microsoft does not want or recommend that you disable updates in Windows 10, but you can do so by disabling the Windows Update and BITS services in the services console, after you install the OS. I did not just advise you to do this ;) ;) ;)

Hope this helped.
