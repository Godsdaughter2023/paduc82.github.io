Please note, if you are using a MAK license for Windows 7 Enterprise, it is pretty much identical to Windows 7 Ultimate.

 

The reason why the other editions of Windows 7 are failing to install could be the installation files are corrupt or the media is scratched.

Tools you will need for this task:

  • eicfg_removal_utility.zip – is a simple tool that will remove the ei.cfg from any Windows ISO disc image, thereby converting the image into a “universal disc” that will prompt the user to select an edition during setup. It’s a free download here

  • • Windows 7 Ultimate (x86)* - 32 bit

    • Windows 7 Ultimate (x64)* - 64 bit

That’s it! the ei.cfg file is now removed.

 

Burn the .ISO file to a blank DVD or create a USB thumb drive.

 

Windows 7 will boot as it normally does into the setup environment, the only difference this time is, you will be given a choice to select the edition of Windows 7 you want to reinstall.

Contacts in Facebook and Outlook.com are different. If you want to delete your contacts from a third party website such as Gmail or Facebook, you need to delete these contacts from the original website.

"If I break the connection as you suggest, will the contact remain in my Google Address Book? "

No, breaking connection from a third party website will not delete your contacts. You can always check your contacts by logging into the contacts list of your website.

"how do you break the connection"

Go to http://account.live.com, under Account, click Related accounts. Then choose Manage the other accounts you've connected. From there you'll be able to remove your connection between accounts.
