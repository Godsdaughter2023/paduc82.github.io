 12002       ERROR_INTERNET_TIMEOUT

"Timed out" suggests that there is interference on the outbound network path.  A common possible cause is security programs which don't know what they are doing and are treating the legitimate request to get an update as if it was coming from a suspected Trojan.  So, before retrying the update, uninstall all possible interferers and see if that changes the symptom.

To solve the problem, double-click the icon in the Taskbar (Left). After the program launches, click the Performance tab on top, and click disable “Link Power Management”.
