ken blake - microsoft mvp say:
I am planning on building my own pc and am wondering what will happen if my motherboard fails. I was told that with Windows home premium retail you are only given one activation license. How would I install Windows onto a new motherboard when the activation license is already being used?



Correct, you have only one license. However you can use that license and reinstall Windows as often as you need to or want to, as long as it's never on more than one computer.


However, one caveat: if you should buy an OEM copy of Windows instead of a retail copy, its license comes with a severe restriction: its license permanently ties it to the first computer it's installed on, and it may never be moved to another computer. And if you change the motherboard, Microsoft considers that it's a new computer. (However, even with a n OEM version, you *can* change other components such as the hard drive.)


You can save some money by buying an OEM copy, but you should be aware of this restriction. It's your choice as to whether it's worth saving the money. Remember that if the computer dies, Windows dies with it.
