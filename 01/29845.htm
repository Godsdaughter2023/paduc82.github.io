frederik long say:

A) back up "C" drive to, say, documents, and then copy this (and regular updates) to a USB stick. Won't this be enough for an emergency recovery?

-> An "image" contains all of drive C:. In other words, the complete Windows installation plus all your personal files. It would require as much free space on the backup medium as you currently use - anything between 20 and 60 GBytes. You need a hard disk in a USB case to do this. Your flash disk is probably way too small.

B) just backup the system image to "C" drive then copy this to a USB stick? This sounds too simple so I'm sure it's naive.

-> You cannot store the image of drive C: on drive C: itself. You need to store it in a different location.

a. user say:

You're going to need to get over your one syllable limitation, because making and restoring a system image is complicated. It's also extremely important.

Terminology: A system image (aka disc image, backup image or simply an 'image') is an exact copy of your hard drive from beginning to end. Another way to say it is that an image is a copy of your entire computer. When you restore an image, your computer instantly goes back in time to the moment the original image was made. The image itself is compressed, so it's significantly smaller in size than the size of your hard drive.

A USB flash drive is a poor choice for storing an image, even if you can physically use one. You want to keep a rotating history of your disk images, and that will quickly overwhelm a flash drive. For instance, you could keep that last 10 images or the last week of images. A USB hard drive (also known as a portable drive) is a much better, more robust (meaning it will last longer) and less expensive choice.

The very best favor that a computer user can do for themselves is to spend time on the web doing research on what a backup image is, how it works and how to make and restore an image. Then become familiar with the popular image backup programs, read the reviews, take advantage of the free trials and finally - get started. Having a recent backup image will save you from 90% of the suffering that people in this forum experience.

frederik long say:

The backup process backs up user files so that you can restore them if needed. It does not back up program files. To avoid problems with programs it is mandatory to keep all software installation files on an external disk together with the purchasing details (where purchased or downloaded, price paid, licence numbers).

You can create an image of your installation via the Control Panel / Backup and Restore. You can use this image to restore the machine exactly as it was when you created the image. Your data will be set back as it was when you created the image.

davem121 say:
Hi Kate

On Community, we are just users like yourself helping each other out . . .

It is possible you are logged into a temporary profile after that update . ..

Before you do anything or move any files around

Restart (not shut down) your PC 3 or 4 times, this will fix this issue a lot of the time . ..

Otherwise:

Please open Windows File Explorer
Navigate to C:\Users
Do you see two user folders that look like yours?
Open those folders and check inside Pictures, Documents . . . etc. and see if your files are there

If your files are there:

Create a new user profile with Admin privileges
Log into the new user profile
Copy all your data from the old profile to the new profile (Documents Library, Pictures Library . . . etc.)

If you still have not found your files, look for a Windows.old folder on your C Drive, expand that to open the users folder contained inside - check there for your files, icons . . .etc.

If you do find your files . . . etc. there, copy them out as soon as possible as Windows will remove the Windows.old folder in time

NOTE VERY IMPORTANT: Do Not copy your files into the Temporary profile you are currently in, when the situation is corrected, your files will be deleted with the Temporary Profile and will not be recoverable !!

andre da costa say:
Press Windows Key + R
Type: slmgr /dlv
Hit Enter

Look in the section of the Windows Script Host dialog that says 'License Status'

Windows 10 came preinstalled on my computer, how do I find the product key?

Press Windows key + X
Click Command Prompt (admin)
Enter the following command:
wmic path SoftwareLicensingService get OA3xOriginalProductKey
Hit Enter

The product key will be revealed, copy the product key then enter it
Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:
