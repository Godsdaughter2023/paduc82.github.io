Hi ,
I'm Ramzi, an Independent Advisor, here to help you.

You can do that via the TouchPad Fn keys.
Some laptops have an Fn key that can be used in combination with one of the F1 to F12 function keys to enable and disable the touchpad. To toggle the touchpad off and on, press both of these keys at the same time.

Or, Through the Windows menus

1-Press the Windows key, type Control Panel, and then press Enter.
2-Select Hardware and Sound.
3-Under Devices and Printers, select Mouse.
4-In the Mouse Properties window, select the tab labeled TouchPad
5-On the TouchPad tab, there should be an option for enabling and disabling the TouchPad. It may be a button or a checkbox.
6-Select the Disable option and then OK.
