dax1792 say:

This is normal.

The WinSxS folder is used by the Windows Installation and Update process. There are 'hard links' between the System32 folder and the WinSxS folder so the files in System32 and the latest one in WinSxS actually point to the same physical disk location. The older versions are there to be used if an update is uninstalled. The files with the mui extension contain the language specific content used by wlanext. This allows the same wlanext.exe to be used in any language version of WIndows. Only the .mui files differ.  

 

You can find more information on the WinSxS folder here:

http://blogs.msdn.com/b/e7/archive/2008/11/19/disk-space.aspx

http://blogs.technet.com/b/askcore/archive/2008/09/17/what-is-the-winsxs-directory-in-windows-2008-and-windows-vista-and-why-is-it-so-large.aspx

http://blogs.technet.com/b/joscon/archive/2010/08/06/should-you-delete-files-in-the-winsxs-directory-and-what-s-the-deal-with-vss.aspx

Over the years, Microsoft has provided tools that purport to transfer "files and settings" from one computer to another.  For Vista to Windows 7, this is "Windows Easy Transfer" --> https://www.microsoft.com/en-us/download/details.aspx?id=14179

I've always been a bit skeptical of these tools and have preferred to just copy files manually.  You can either copy the files to an intermediate device like a flash drive or an external hard drive or you can connect the two computers using either standard networking or a direct cable connection.  See, e.g., http://www.howtogeek.com/howto/9040/how-to-share-files-and-printers-between-windows-7-and-vista/

The time it will take to make the transfer depends on both the quantity of data and the transfer mechanism.  USB is faster than the typical home network, but there are two transfers involved (Vista to flash drive and flash drive to Win 7), not to mention the time taken to physically remove the device from one computer and connect it to the other, especially if that has to be done multiple times because you have more data than will fit on your flash drive.

As far as programs are concerned, the safe way -- and for years the only way -- to move programs is to reinstall them on the new computer using proper installation media (e.g., original CD/DVD or a downloaded installer file).  For some programs, you'll need to know the license/product key, so make sure you have this information before uninstalling anything from the old computer.

When Windows 10 was released, Microsoft did not provide a new "Windows Easy Transfer" program.  Instead, MS made a deal with LapLink to allow use of LapLink's "PC Mover Express" program.  I think it originally was free; now it's $14.95 -- but again, only http://pcmover-10.laplink.com/

Although Microsoft suggested PCMover Professional for transfers to Windows 10, as near as I can determine from the LapLink site, it should work for transfers from Vista to Windows 7 as well -- but it would be prudent for you to ask LapLink directly about that if you're thinking of buying it.

I've never used this LapLink program, so I can't tell you from personal experience if it works or not.  LapLink, however, has been around for a long time and I expect that it actually does what it's advertised to do.

Poking around the LapLink site, I noticed that there are multiple versions, at various prices:  http://www.laplink.com/index.php/individuals/pcmover-for-windows
