How do I determine the size (1.1 or 2.0) of my current USB - Microsoft Community

lawrencexz say:

How do I determine the size (1.1 or 2.0) of my current USB ports on Windowx XP?

I want to buy a flash drive, and I'd like to know what size my USB ports are.
lemp say:
It's not the size, but the speed.  If you buy a USB 2.0 flash drive, it will work on either USB 1.1 or USB 2.0. 

If you only have USB 1.1 ports, you'll get a warning when you plug in the flash drive that "This device could work faster if connected to a USB 2.0 port".

To determine what you have
  • Open Device Manager (Start > Run > devmgmt.msc > OK)
  • Click the + to the left of Universal Serial Bus Controllers (it's at the bottom of the list)
  • You will see one or more lines that include the words "Host Controller."  If any of those lines has the words "Enhanced Host Controller," then you have at least some USB 2.0 ports.  If none of the Host Controllers is "Enhanced," then they are all USB 1.1
  • Some older computers have a mix of USB 2.0 and USB 1.1 ports.  If you see "Enhanced Host Controller" in the list in Device Manager but you get the warning message when you plug in a USB 2.0 device, try a different port.  In particular, if you first tried a port on the front of the computer, try one on the back instead (or vice versa).

Hi

 

Use the desktop version of Internet Explorer.

 

Go to the Desktop and click the Internet Explorer icon on the Taskbar.

 

Navigate to the website that you want to create a shortcut and place on the desktop.

 

Right click an empty spot on the website page and select the Create Shortcut option. Click Yes in the confirmation prompt. This will place the shortcut for the website on your desktop.

 

Manually creating a shortcut is almost the same as creating a shortcut for an application. The only difference is that you type the URL for the website in the Location Box, instead of the path to the application.

With XP you can do a refresh install without really touching the current applications and data.

Refresh XP Installation.

Boot from the XP original CD. Skip the first screen that offers Console Repair, and continue as though you install a New XP.
