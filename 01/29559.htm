spacejunkie1 say: Hello moonduks,

If you are using something like Microsoft Office to write words, for example, you can vary the size of the image by using the zoom bar at the bottom right hand of the page.  If you want to change the font size of what you are writing you can use the scroll down format font size settings at the top of the page on the tool bar. 

If you want to maximize the fonts which are on the windows themselves you will need to go to the "appearance and personalization" section in the "Control Panel" and adjust the font size located under the appearance tab in the Display Properties Window. I use the Extra Large Fonts so that I do not have to use my reading glasses to see the text clearly.
 
The "magnifier" in the accessories and control panel options is an option, but less convenient to use.  Font sizes for Windows properties comes in normal, large, and extra large sizes. 

If you have a monitor which has a high screen resolution, you may try lowering the screen resolution as a final option if nothing else works. I've opted to use the extra large fonts for my windows theme when I added a 1600x1200 monitor to my system as an upgrade because I could not read the text without my reading glasses at the normal or large font levels.  There are new monitors now with higher screen resolutions; Microsoft may need to think about adding additional font sizes to the text in the windows themselve to compensate for the appearance of smaller sized printing. 

I hope this information helps you.

Good Luck
