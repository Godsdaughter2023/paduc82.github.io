how to disable unnecessary Apps \ERSvc, \HidServ

Hi,

lemp say:

What version of Windows do you have, including service pack?

What makes you think that either ERSvc or HidServ are unnecessary or contribute to any start up delay you may be experiencing?

ERSvc is the Microsoft Error Reporting Service.  It generally is not useful for end users and can easily be prevented from running.

HidServ may either be malware or the Microsoft Human Interface Device (HID) Access service.  If you see hidserv.exe in a subfolder of C:\Program Files, your computer is infected and you should take steps accordingly.  HidServ normally is disabled by default in Windows XP and thus should not affect your startup time.  If you have Windows XP and HidServ is running and is not malware, you need it to enable predefined hot buttons on some external device you have connected to your computer.

In general, you should stop services (which is what both ErSvc and HidServ are) by opening the Services applet (Start > Run > services.msc > OK) and setting the Startup Type of the service to "Disabled" (if you never want it to run) or "Manual" (if you only want it to run when it is needed). 

By default, the Startup Type of HidServ is "Disabled" and the Startup Type of ErSvc is "Automatic" (which means that it starts when Windows starts).

Changing the configuration of services without understanding what you are doing can be dangerous for the operation of your computer.  For more information, see http://www.blackviper.com/service-configurations/black-vipers-windows-xp-x86-32-bit-service-pack-3-service-configurations/ (or the appropriate page corresponding to your version of Windows and service pack).
