a. user say:

Hello,

As per description you like to create the Microsoft account using your business email.

You can convert Microsoft Account into Local User Account, then local User Account into Microsoft Account. This will help you to change the Microsoft’s account's login ID to your business email without affecting data of the User Account.

Refer to these steps to change Microsoft Account into Local User Account.

a. Press Windows and C keys together.

b. Click on Settings, then click on Change PC Settings.

c. Click on Account.

d. On your account page, under the User name and User ID, you will be able to see “Disconnect” option.

e. Click on Disconnect and follow the on-line instructions.

Once Microsoft account gets changed into Local User Account. Now convert Local User Account into Microsoft Account using business email. When you connect to Microsoft account use the proper email address.

Refer to these steps to convert Local User Account into Microsoft Account.

a. Press Windows and C keys together.

b. Click on Settings, then click on Change PC Settings.

c. Click on Account.

d. Click on Connect to Microsoft Account and follow the on-line instructions.

Hope this information is helpful. Please do let us know if you need further assistance with Windows.
