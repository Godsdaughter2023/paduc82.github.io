sam_572 say:

Hi Anthony,

Exporting the Publisher file as a PDF and opening that in Word works the best for me.

To do this go to: File>Export>Create PDF/XPS Document>Click the create button>Choose where to save the PDF and choose "Publish".

If you're on an older version of Publisher I believe you can Export to PDF by going to: File>Save As>Choose where to save and file name, but ensure that the "Save as type" is on "PDF">Save.

Now find where you have saved the PDF in Windows Explorer (Win Key + E) and right click the PDF and choose: Open with>Word. Word will attempt to convert the PDF file to an editable Word file. A dialog box should appear explaining this to you, just click "ok". Hopefully this should have converted all fine, however, sometimes Word can struggle and move some text.

I would suggest giving this a go and seeing what the outcome will be. Hope I helped.

Sam.

    a. user say:

    You would press F8 during XP boot to get to the Advanced Startup Options where you could choose things like Safe Mode, etc.

    You need to get into the BIOS of the motherboard and you need to be pressing key before XP starts, or you will miss the window of opportunity to enter the BIOS.

    The previousl mentioned manual for your mohterboard says to get into the BIOS, do this:

    Chapter 3 BIOS Setup


    BIOS Setup is an overview of the BIOS Setup Program. The program that allows users to modify the basic system configuration. This type of information is stored in battery-backed CMOS RAM so that it retains the Setup information when the power is turned off.

    ENTERING

    Powering ON the computer and pressing <Del> immediately will allow you to enter Setup. If you require more advanced BIOS settings, please go to “Advanced BIOS” setting menu.

    Now you need to pay attention because on your keyboard you may have a Delete key and a Del key (they are not the same).  So be sure you are pressing Del and not Delete.  Usually, there is a message on the screen for a few seconds and you need to press Del most urgently!

    Then refer to the manual again to browse the BIOS and set up the Boot Order to your HDD is first, CD second, and floppy last and exit the BIOS saving your settings on the way out.

    Knowing how to change the boot order is a good thing to know if one day your XP will not boot and you need to boot on a CD to fix your XP, you will have to change the Boot Order to look at the CD first.

    I have mine set to look at the CD first (I boot on CDs a lot) and then the HDD second.  If the CD drive is empty, it just skips it and boots to the HDD.  You can set yours like that too so you don't forget about it one day and get all flustered about why your system will not boot on a CD.
