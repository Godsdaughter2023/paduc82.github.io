mick mickle say:

No Internet after Windows 10 Fall Creators Update (Fixed)

I had two computers, among a myriad of those that I updated to Fall Creators Update yesterday, that had no Internet access (actually, no LAN access either) after installation of the update was complete.  One was an AMD desktop and the other a Samsung RC512-02 Intel notebook.  The biggest difference of these two computers from other workstations was that I used Hyper-V on both, so both of them have virtual switches.  The Windows troubleshooting tools did not help, and Device Manager showed all devices and drivers were working.  After some research, I found that the solution was to simply perform a Network Reset accessed through Settings>Network & Internet>Status and scroll to the bottom.  The computer reboots as part of this process.  This worked for both computers to restore connectivity and keep the Fall Creators Update.

Virtual Switches have to be created again for the Hyper-V machines.

This problem is different from the one I experienced with my older Toshiba A205-4577 notebook computer that had no connectivity after the 1703 Creators Update.  That update caused many duplicate entries in Device Manager, and the only way I found to fix it was to go back to the 1607 Anniversary Update and use wushowhide.diagcab to hide Creators Update.  Unfortunately, wushowhide.diagcab won't see a feature update to hide it until the update is offered in Windows Update, and once it's offered you may not have time to run wushowhide.diagcab before the damage has been done.  Microsoft needs to give users a better way to stop feature updates on older equipment that has reached its highest level of operational capability with particular Windows 10 version but breaks on newer versions.

kristoffer abs say:

Hi,

It is possible for the default program settings to change when an update from an app is installed. It can override your current setting depending on how the update is delivered. Most of the updates from applications are installed through patches. However, for some programs that needs major changes, it may reinstall the application and also some other components like setting the application to be used as the default program, thus causing the issue.

There are two ways in which you can change the default app on your device. The first one is by using the Default Programs, and the other one is by using Open with under the right-click options. Since you've mentioned that the first option is a bit tiresome, we recommend that you use the second option when setting a default app. To do this, follow these steps:

  1. Open the folder where your PDF files are located.
  2. Right-click on one of your files and select Open with.
  3. Under the Open with menu, select Choose another app.

  4. Select the application which you want to set as the default program to open the PDF files.
  5. Make sure to put a check under the Always use this app to open.pdf files option.
  6. Click OK.

Once done, the changes will automatically apply on all of your PDF files.

Let us know if you have any other concerns.

right-click on your sound card, and click Uninstall.

gerry c j cornell say:

If the upgrade process sticks or is not responding at 99% it is best to wait and do nothing for 5 to 6 hours. If there is still no progress close the Windows 10 Upgrade Assistant, restart the computer and try upgrading following the advice detailed below.

 

Are you using an ethernet cable or wireless to connect your computer to the router? An ethernet cable connection is faster and more reliable for large downloads. An ethernet cable is an inexpensive purchase.

 

Disable your anti-virus whilst upgrading.

 

I suggest you download the Windows 10 Media creation tool ISO and do an In-place upgrade.

 

Download the updated Windows 10 Media creation Tool Build 10586:
http://www.microsoft.com/en-us/software-download/windows10

 

How to download an ISO using Media creation Tool:
http://www.tenforums.com/tutorials/9230-windows-10-iso-download.html

 

How to perform an In Place upgrade using Method 1:
http://www.tenforums.com/tutorials/16397-repair-install-windows-10-place-upgrade.html#step6

 

If that does not work please provide more information to enable your problem to be diagnosed.

 

Please restart your computer and allow 20 minutes for the system to run before uploading information. When examining Event Viewer log files many, not all, problems show in the period immediately after the computer has been booted.

 

Please provide a copy of your System Information file. Type System Information in the Search Box above the start Button and press the ENTER key (alternative is Select Start, All Programs, Accessories, System Tools, System Information). Select File, Export and give the file a name noting where it is located. Do not place the cursor within the body of the report before exporting the file. The system creates a new System Information file each time system information is accessed. You need to allow a minute or two for the file to be fully populated before exporting a copy. Please upload the file to your OneDrive, share with everyone and post a link here. If the report is in a language other than English, please state the language.

 

Please upload and share with everyone a fresh copy of your System log from your Event Viewer to your One Drive and post a link here. It helps to avoid confusion if you remove any earlier copies of the log files from your OneDrive.

 

To access the System log select Start, Control Panel, Administrative Tools, Event Viewer, from the list in the left side of the window expand Windows Logs and select System. Place the cursor on System, select Action from the Menu and Save All Events as (the default evtx file type) and give the file a name. Do not provide filtered files. Do not place the cursor in the list of reports before selecting Action from the menu. Do not clear logs whilst you have a continuing problem.
