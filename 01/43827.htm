RD /S /Q %SystemDrive%\windows.old

(To grant currently logged on user ownership of)

takeown /F "full path of folder or drive" /R /D Y


(To grant administrators group ownership of)

takeown /F "full path of folder or drive" /A /R /D Y

Hi Dean,

I understand that your USB drives and the internet connection works properly but it is not listed properly as it should be in Device Manager and Network page.

This issue could have occurred if the drivers were not installed properly or failed to configure. As USB drive usually works if the default driver/generic drivers are present in the system. But as you rightly mentioned that gets listed under USB Controllers.

For better understanding of the issue, please provide us following information:

  1. What is the make and model of the computer?
  2. When did this problem started?

I suggest you to follow the steps below and see if that works:

Run System File Checker

System file Checker determine whether the issues you are experiencing on your computer are caused by one or more system files.

You can refer to the following Microsoft article to repair corrupt files using SFC tool. Use the System File Checker tool to repair missing or corrupted system files

Refer: http://support.microsoft.com/kb/929833/en-us

Uninstall Device drivers

  1. Press Windows key + R
  2. Type devmgmt.msc
  3. Right click on each device which is listed under Unknown category
  4. Uninstall each device and restart the computer

After restart Windows will automatically install the device driver and you should see the device listed under proper category.
