Sorry to say so but I think your only chance to recover your data is to hand your disks to a professional data recovery service and be prepared for a big bill.

At the same time I suggest you review your reasons for using a RAID array. This technology is useful in environments in which a disk failure must not cause any downtime. RAID technology is not a backup method and it provides no protection whatsoever against 99% of the problems that occur in a home environment, e.g.
- user error
- virus/malware infection
- system malfunction
- problems caused by bad software or drivers
- file system corruption.

Because of its complexity, a RAID array can and will cause problems of its own, as you see right now. If you wish to protect your data then you should use a few external backup disks. If you wish to protect your installation then you should store images of your system on external disks, using an imaging tool such as Acronis True Image. Step 1: Disable Automatic Restart.

 

Post the error code/message for assistance.

 

a)   Click on the  Start button and then on Control Panel.

b)   Click on the  System and Security link.

g)   Click OK in the Startup and Recovery window.

Const KeyOffset = 52
i = 28

Do
