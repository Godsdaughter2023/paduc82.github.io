yleclerc say:

Each and every model of computer will probably have different internal components (i.e.: network port, modem, video card and audio controls) so you can not "move" a "driver" from one to the other "directly."

 

Realtek Audio Manager is part of the Realtek drivers that are needed / used for your Vista laptop.  The drivers are provided by that laptop's manufacturer. It seems that your new Windows 7 laptop is a different model from a different manufacturer and could be using a different sound card than your original Vista laptop.,  You need to use the drivers that the Windows 7 laptop manufacturer is providing.

there is a command prompt command to tell Windows where WindowsRE (recovery environment) is located.  On one of my PCs, I had to run that command and once it was pointed back to the correct recovery partition, it created the recovery media fine.

try this (from another site):

Unable to Create Recovery USB Drive

If you have imaged your drive, there is a chance the System Reserved partition is not in the same place that your Windows installation thought it was, and attempts to create a recovery drive will fail with the following error:

We can‚Äôt create a recovery drive on this PC

Some required files are missing. To troubleshoot problems when your PC can‚Äôt start, use your Windows installation disc or media.

When this happens, you can fix this problem by telling the program that creates the recovery drive where the recovery environment is located by entering the following commands into a command prompt run with administrator privileges:

reagentc /disable

result ‚ÄúREAGENTC.EXE: Windows RE is already disabled.‚Äù

reagentc   /setreimage   /path   \\?\GLOBALROOT\device\harddisk0\partition1\Recovery\WindowsRE

This assumes your 450mb System Reserved partition is partition1.  In my case, it was partition3.  Keep trying partition1, partition2, partition3, etc until the command is successful. 

You can also go into disk management by right-clicking the Windows 10 Start icon and click on Disk Management.  Then look which partition is the 450mb System Reserved one for disk0.  For me (which was a result of a Windows 7 upgrade), it was the 3rd partition.  For a fresh installation it is probably the first partition and may be 500mb.

Then try again to create your recovery drive and it will probably work!
