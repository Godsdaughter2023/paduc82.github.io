This search system is the same one from 7, Vista, and XP... but there are some situations where it can choke on a file and cause high CPU usage (as investigated and discussed here).


A few steps you can take to try to resolve it:

  1. Use the automated diagnostic
    From your Start screen, type 'trouble' and look in the Settings category to open Troubleshooting.  In there, under the System section, run the Search and Indexing troubleshooter.  It will ask you what kind of problem you are having -- choose the "search is slowing down my computer" option and let the diagnostic tool run.


  2. If that doesn't help, reset the search system
    From your start screen, type 'index', and look into the Settings category to open Indexing Options.  In there, click the Advanced button, then click rebuild.  This will cause lots of disk and CPU usage while it rebuilds the index, so I recommend doing this before you go to bed and let it run overnight.


  3. If the problem still exists, try disabling the Search service
    Instructions for this are here
