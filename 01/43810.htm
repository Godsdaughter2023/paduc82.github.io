cmdrkeene say:

When you say C:Documents and D:Documents, can you provide the full location you're viewing?

The real gist you need to know is that libraries are not folders.  They are really just a quick search of several places.  Try this:  right-click the Documents library and go to properties.  This will tell you what locations that library is searching.  Every time you open the Documents library, it does an instant scan of those places, and tells you everything it found.  Those aren't extra copies of the files, it's just search results. If you delete something, you are deleting the one and only original, wherever it happened to be.

You might ask "well what's the point of a library?"  I'm glad you (might have) asked.  The reason is because you might have a lot of places.  Take me for example.  I have some movies on my hard drive.  I have a lot more moves on an extra drive (they won't all fit on the main drive).  Plus my roommate has some movies on his drive.  With a library, I have one Videos library that instantly searches all those drives and gives me one single combined view of all our movies.  Now I can see all my movies in my library at once, regardless of which drive they are on.

  1. Go to Start , then select Settings > Update & security > Activation.

* You can see the confirmation pop-up message on successful activation.

Method 1:

a) Open run command by pressing Windows key + R and type “devmgmt.msc” (without quotes) and hit Enter.

d). Once this is done, restart the computer to check the problem fixed. If it doesn't fix the issue, then Right click on the same driver and select “Uninstall” then restart the system and let the system install the driver automatically on Start. Check and see if it resolve the issue.

Method 2:

b. Right-click on the driver setup file and click on ‘properties’.
