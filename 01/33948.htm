Problem:  

The following error is received on the Activation page after a clean install of Windows 10.  Windows cannot be fully configured until it is activated, however it will not accept the Windows 7 or 8.x product key.

Error: 0xC004C003 - The activation server determined the specified product key has been blocked.

Cause:   


Due to performing a “clean” install using a Windows 10 CD or USB. The first time Windows 10 is installed on a Windows 7 or Windows 8.x PC, it must be “upgraded” not the “clean” install option.

Solution:

  1. Install your previous version of Windows. (7 or 8.x)
  2. Install all Windows updates.
  3. Run the Windows 10 CD or USB within Windows (not during bootup)
  4. Follow the on-screen instruction to UPGRADE to Windows 10.  The UPGRADE process converts your old Windows key into a new Windows 10 key.  It will not ask for the key, and after installation Windows 10 will be activated.  

NOTE:  Ensure you do not pick any advanced options to perform a complete wipe of your drive to perform a clean install when installing Windows 10 for the first time.
