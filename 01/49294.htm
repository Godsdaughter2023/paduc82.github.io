Reverting to Windows 7 from Windows 10 often does not work well.  The usual recommendation is to do a clean install of Windows 7.

Your problem may be caused by the fact that Windows 10 doesn't have Windows Photo Viewer in the right-click context menu by default.  Depending on how you got back to Windows 7 after having installed Windows 10, the necessary entries in your registry may still be missing.  There is a reg file download to get Photo Viewer into the Windows 10 context menu, but I don't know if the registry entries in Windows 7 would be the same as those in Windows 10.

A.  Before you get into registry edits, however, I assume you've done the following, but just in case you haven't ...

  • Right-click on an image file and select Open with > Choose default program
  • If Windows Photo Viewer is listed, select it, make sure the box to "Always use this program to open this type of file" is checked, and click OK
  • If Windows Photo Viewer isn't in the list of "Recommended Programs" be sure to check in the list of "Other Programs" as well

B.  You can also take a look in Control Panel:

  • Click Start, type Default Programs into the Search box, and press Enter
  • Click "Set your default programs"
  • Scroll down to Windows Photo Viewer is in the list, select it, and then click either "Set this program as default" or "Choose defaults for this program"

C.  If you did the above and Photo Viewer wasn't listed anywhere, first make sure that you have the necessary files (you probably do).

Look in C:\Program Files (and, if you have a 64-bit system, also look in C:\Program Files (x86)

  • Do you have a folder named WindowsPhotoViewer ?

naman r say:

Hi,

Try the steps mentioned below to resolve the issue.

Step 1:

Check if the IP Helper service is started.

IP Helper service provides tunnel connectivity using IPv6 transition technologies [6to4, ISATAP (Intra-Site Automatic Tunnel Addressing Protocol), Port Proxy, and Teredo], and IP-HTTPS. If this service is stopped, the computer will not have the enhanced connectivity benefits that these technologies offer.

1. Click on Start, type Services in the search box.

2. On the Services window, look for the IP helper service.

3. Make sure the service is started and set to Automatic.

If the service fails to start, check its dependency services first start them and then try to start the IP helper service.

Step 2:

If the issue still persists, update the Network card drivers. ISATAP and Teredo are interrelated to Network card drivers.

Update the drivers for your Network adapter from the Manufacturer’s website.
