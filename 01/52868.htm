how do I use a circle to identify a player in a sports video made with - Microsoft Community

jfpandise say:

how do I use a circle to identify a player in a sports video made with movie maker

I want to highlight my son in a sports video. How do I add a circle or transparent marker to identify him.
considerate_guy say:

There isn't a feature, within Movie Maker, to add a highlight.  Such a feature would be really nice and you might propose it at feedback.live.com.

In the meantime, I have a couple suggestions for ways to accomplish this.

METHOD 1:  Freeze Frame Highlight

The idea behind this method is to take a still snapshot from your video, edit it in a paint program to add any highlight you desire.  This could be darkening of the rest of the shot, adding a circle, adding some text or adding an arrow.

I created a sample video

which goes overboard and does all of the above, but you get the idea.

To create this you need to:

  1. Get to the point where you want to add the highlight and make a Split ('M' on the keyboard or use the Video Tools | Edit tab).
  2. Use the Snapshot feature (Home tab) to save an image of this frame.  Specify where to save the png image and remember this folder.
  3. The image will also get added to the end of your storyboard.  Drag it back up to the split point.
  4. Set the duration for the length of time you'd like it to freeze (if you don't like the 7 second default).
  5. Save your project at this point and close Movie Maker.
  6. Find the image you saved previously in step 2; open it in your favorite image editing program.  Even Paint will do.
  7. Add your desired highlight (draw a circle, add an arrow, type the player's name, etc.)
  8. Save the edits and reopen Movie Maker.  When you playback your preview you should see the updated image with the highlights.

METHOD 2:  Text Caption

This method uses the Wingdings font to add a circle just like you would any text caption.  This works best  if the person isn't moving within the frame, say at the start of the play.

  1. Click Caption on the Home tab.
  2. Choose the Wingdings font and set a size of about 200 (type it in).
  3. In the text area we are going to type a circle character as follows: 
    a. Make sure NumLock is on. 
    b. Hold down the Alt key. 
    c. On the number pad (not the top row), type 0161.
    d. Release the Alt key.
  4. Now position that circle as needed.
  5. Feel free to style it by changing the color or changing the font size bigger or smaller.
  6. Optionally, pick a text effect. Some good ones are Fade, Stretch and Fly in - right, but experiment to find a style you like.

There are other shapes you might want to try too. (069 = hand pointing left, 070 = hand pointing right, 071 = hand pointing up, 072 = hand pointing down, 0223 = arrow pointing left, 0224 = arrow pointing right, 0225 = arrow pointing up, 0226 = arrow pointing down).

You can also use this method for censoring someone's face, or covering a license plate, for example.  The method is the same, but you just want a solid shape instead (e.g. 0108 = circle, 0110 = square). Keep the text effect as None and set the duration as long as necessary.

Note: you might find other characters you like in the Wingdings font so take a look.

Summary:

If you use either of these methods, let me know how it goes. I'd be interested to know your thoughts on whether they work for you or not.
