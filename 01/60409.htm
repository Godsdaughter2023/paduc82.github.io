On each program you don't want to start automatically, check its Options to see if it has the choice not to start (make sure you actually choose the option not to run it, not just a "don't show icon" option). Many can easily and best be stopped that way. If that doesn't work, run MSCONFIG from the Start | Run line, and on the Startup tab, uncheck the programs you don't want to start automatically.

However, if I were you, I wouldn't do this just for the purpose of running the minimum number of programs. Despite what many people tell you, you should be concerned, not with how *many* of these programs you run, but *which*. Some of them can hurt performance severely, but others have no effect on performance.

Don't just stop programs from running willy-nilly. What you should do is determine what each program is, what its value is to you, and what the cost in performance is of its running all the time. You can get more information about these with internet searches and asking about specifics here.

lemp say:
"Too many" programs starting when your computer first starts isn't really the right way to look at things.  What you need to do is decide which programs you need and which you don't. 

If you're concerned about "slow" startup, the usual culprits are programs that perform some sort of scan of your computer -- which is what takes up time.  These are programs that you really want to keep, such as your antivirus program and Windows Update.  Some antivirus programs are more efficient than others as far as start up scans are concerned.

Many programs can be prevented from starting when Windows starts by using the program's "options" menu and unchecking an item that says something such as "Start Skype when I start Windows" (obviously for Skype). 

Another thing to do is click the Start button, then All Programs, then Startup.  The contents of the Startup folder are shortcuts to programs.  If you don't want a program to start, just delete its shortcut from the Startup folder (this will NOT delete the program; you can still start it either from a desktop item or from its own entry in the Start > All Programs list).

There are other methods that some programs use to start automatically, and that's why "startup manager" utilities exist.  Some of these (e.g., Autoruns) need to be used with caution so that you don't disable something that really needs to be started (even if you don't think it does).

As far as your specific question is concerned, we have no way of knowing what video you're watching or what tool you're referring to.  If you want advice about something like that, please provide the URL (web page address) that you're looking at.
