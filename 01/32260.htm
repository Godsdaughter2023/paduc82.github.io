How to migrate W-7 to replacement HD?

The HD where W-7 is currently installed is failing.  Can I migrate Windows to a replacement HD?

Thanks - Val Garuts

frederik long say: Here you go:
  1. Connect an external disk to your PC.
  2. Create an image of your installation via the Control Panel / Backup and Restore. Store it on the external disk.
  3. Burn a Windows Repair CD.
  4. Remove the old disk, connect the new disk.
  5. Use the Repair CD to boot the machine into Repair Mode.
  6. Restore the image.

uneluctable say:

Excellent.

It can be a different interface and capacity. Formatting should still be NTFS. 

It should still be activated if the only change is the HDD. If more is changed in the system, it may need to be reactivated.
