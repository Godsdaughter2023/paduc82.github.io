Hi

A data CD is fairly easy to copy.

If you just have one CD drive, you can insert the data CD, copy all of the files to a temporary folder and then burn tose files to a blank CD.

If you have a CD Reader drive and a CD Burner drive, you can select all of the files in the CD Reader drive, drag and drop them on the CD Burner drive and perform the burn.
