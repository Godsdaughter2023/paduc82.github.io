Hello,

Online steps: Update payment option details

Note Follow these steps to use a PC to change payment option details, such as your billing address or the expiration date of a credit card.

    1. Go to My Account at Xbox.com.
    2. Select Sign In (unless “Sign Out” is displayed in the upper-right corner).
    3. Enter your Microsoft account (email and password), and then select Sign In.

Note Be sure to sign in with the Microsoft account that’s associated with the Xbox account you need to update. If you don't know your Microsoft account or password, see Find your Microsoft account.

  1. Select Manage Payment Options.
  1. If you are prompted, follow the instructions to enter the security code. Then, select Edit next to the payment option you want to update.
  2. Update your information, and then select Save. If you don’t need to make any other changes, select Done.
Console steps: Update payment option details

Note Follow these steps to use your console to change payment option details, such as your billing address or the expiration date of a credit card.

  1. Sign in using the Xbox Live gamertag associated with the membership for which you want to update payment details.
  2. From the Dashboard, go to settings and select Account.
  1. Select Manage Payment Options.
  2. Select Update Payment Option.
  3. Select the payment option to update, and then update the information as needed.
Online steps: Replace the payment method used for your membership

Note Follow these steps to use a PC to replace the payment method you use to pay for your membership.

    1. Go to My Account at Xbox.com.
    2. Select Sign in (unless “Sign Out” is displayed at the top of the page). Enter your Microsoft account email address and password, and then select Sign In.

Note Be sure to sign in with the Microsoft account that’s associated with the Xbox account you need to update. If you don't know your Microsoft account or password, see Find your Microsoft account.

  1. Select Manage Payment Options.
  2. If you’re prompted, follow the instructions to enter the security code.
  3. Select Apply to Membership for the payment option you want to use to pay for your membership. After you finish, select Done.
Console steps: Replace the payment method used for your membership

Important If you want to delete the payment option currently associated with your membership, you must first add a new payment option, apply it to the membership, and then remove the old payment option.

  1. Sign in using the Xbox Live gamertag that’s associated with the membership for which you want to replace the payment method.
  2. Go to settings and select Account.
    1. Scroll right to Your Memberships and select the membership you want to update.
    2. Select Change Payment Options.

Note If this option appears grayed out, you may have to select Modify Membership to select a plan, or see Add a payment option to your Xbox Live account.

  1. Select the new payment option, and then select OK.

Note If you want to remove the previously used payment option, select Manage Payment Options and then Remove Payment Options after you select Change Payment Options (see step 4).

Help with payment problems

If you’re getting an error message and code, look for help at Xbox error code/status code search.

If you have problems when you try to change payment information, contact Xbox Support.

Cloud Computing Applications:

The applications of cloud computing are practically limitless. With the right middleware, a cloud computing system could execute all the programs a normal computer could run. Potentially, everything from generic word processing software to customized computer programs designed for a specific company could work on a cloud computing system.

 

Cloud applications are a sort of hybrid between traditional desktop applications and traditional web applications. They offer the benefits of both of these types of software without many of the drawbacks.

 

Like desktop apps, cloud applications can offer a rich user experience, immediate response to user actions, and offline mode. Like web apps, cloud applications do not need to be installed on a computer and can be updated at any time simply by uploading a new version to your web server. They also store their data in the cloud - offsite under your control.

 

Most often these applications do not rely on specific operating systems.

 

There is lot of cloud application available; I would suggest you to contact the program administrator/service to get more information about the compatibility with the specific operating system.
