This problem occurs because the Windows Media Player library database has become corrupted.
 
To resolve this problem, clear the Windows Media Player database. To do this, follow these steps:
 
1. Exit Windows Media Player.
2. Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.
3. Select all the files in the folder, and then click Delete on the File menu.
      Note You do not have to delete the folders that are in this folder.
 4. Restart Windows Media Player.
 
Note Windows Media Player automatically rebuilds the database.
 
If this does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:
 
   1. Exit Windows Media Player.
   2. Click Start, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.
   3. Select the Media Player folder, and then click Delete on the File menu.
   4. Restart Windows Media Player.

You could clone the current disk to a larger, external disk using XXClone and then switch the disks over. Buy an external caddy or a USB lead to connect the new disk. Make sure you (A) buy a similar type of hard disk and (B) remove the battery before switching the disks.

Only the professional version will do cumulative backups but this is perfect if an exact clone of the drive is needed. Get it here http://www.xxclone.com/index.htm Click ‘Download’ in the left pane and select the freeware package version. Close down any other running programs before starting. Make sure you select the correct Target Volume. Do not continue with the cloning if you receive any error messages, e.g. NTLDR or NTDETECT.COM is missing.

The only issue I have found with the program is that access is denied to some shared files/folders on a network. This is easily overcome by removing the share and then sharing again.

Hi, B&J

To delete an account from a user in Windows 10 environments using the "Configuration" option, we must follow the following process.

We access the "Settings" menu from Start.

There we select the option "Accounts".

We go to the tab "Family and other people" where we will see the user account to be deleted.

We select the account to be deleted, and we will see that additional options are activated for the user.

Click on the "Remove" button and we will see that the following warning message is displayed.

Click on the "Delete account and data" button if we are sure to delete that user. We can see that the user has been eliminated correctly.
