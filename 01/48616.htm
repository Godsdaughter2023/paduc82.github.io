aera rua say:

Hello meNESS_642,

An effective encryption ensures that a data is secured and protected at all times, but this may become an issue if you only want to encrypt certain files and not all files that you are creating or saving. As a work around to resolve this issue, we recommend that you follow the steps below:

  1. Press the Windows key + X, then select File Explorer.
  2. In the Address bar, enter %userprofile%.
  3. All the File folder paths for your user profile will show up.
  4. Right-click, select New and then Folder.
  5. Name that folder as New Desktop or name of your preference.
  6. Select then Right-click on the original Desktop Folder and choose Properties.
  7. Go to Location tab. In the path location, edit it and enter the new location of the new desktop folder.
  8. Click OK, then Apply.
  9. It will ask you to transfer all your files from that folder, if you wish to copy everything select Yes.
  10. Automatically, the new Desktop folder will take place and you’ll be able to save files without encryption.

Windows Update is actually just another Windows process so it can be stopped with these simple steps:

Open the Run command (Win + R), in it type: services.msc and press enter
From the Services list which appears find the Windows Update service and open it
In ‘Startup Type’ (under the ‘General’ tab) change it to ‘Disabled’
Restart

Or,

Open the Run command (Win + R), in it type: gpedit.msc and press enter
Navigate to: Computer Configuration -> Administrative Templates -> Windows Components -> Windows Update
Open this and change the Configure Automatic Updates setting to ‘2 - Notify for download and notify for install’
Open the Settings app (Win + I) and navigate to -> Update and Security -> Windows Updates. Click ‘Check for updates’ which applies the new configuration setting
Restart


Alternatively for Windows 10 users without group policy access:

Open the Run command (Win + R), in it type: regedit and press enter
Navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate\AU
In there create a ‘32-bit DWORD’ value called ‘AuOptions’ and under ‘Value Data’ type 2 and click ‘OK’
Open the Settings app (Win + I) and navigate to -> Update and Security -> Windows Updates. Click ‘Check for updates’ which applies the new configuration setting
Restart

a. user say:

Hi Richard,

 

Have you installed any codecs ?

 

Follow the steps below to check for any installed codecs:

1.       Go to the Start Screen and type Control.

2.       Now click on the option labeled Control Panel

3.       Click on “programs and feature”.

4.       In the list check for any installed codecs.

 

Are you able to play files using Windows Media Player?

 

Try to run Windows Media Player troubleshooter.

Open Control Panel by typing Control Panel on the start screen.

In the search box, type troubleshooter, and then click Troubleshooting. Click View all, and then click Windows Media Player Settings.

 

Hope this helps.
