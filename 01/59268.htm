Hi Tammy ,

In the back or bottom of your Router is a sticker with a UserName, Password and IP address (something like 192.002.005.001) printed on it

Note those down

Go to your PC and open a Web Browser

In the address type the IP address and hit Enter

That will bring up your routers configuration page

Somewhere on that page you will find LAN settings

From there you should change the SSID (name of your WiFi network) and password to something stronger

Before you can change that, you will need to enter the UserName and Password you noted down from that sticker on the router

That's It!

Please Note, after hanging that WiFi password, you will need to log in all your personal devices to your WiFi network, using that new password

  1. Press “Windows key + X” and click on “Device Manager”.
  2. In the list of devices, you should see the keyboard drivers.
  3. Right click on the device and then select "Uninstall".
  4. In the uninstall dialog, make sure you select the option to delete the driver package from the system. This causes the driver package installed earlier to be deleted.
  5. In the Action menu of Device manager, select “Scan for Hardware changes” option. This will cause the device driver to be reinstalled. Once the driver is installed, close all windows and restart the computer.
