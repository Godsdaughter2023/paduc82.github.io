a. user say:

Ignoring the more theatrical aspects of your post, automatic updating is a good thing. Good for us, good for the ecosystem. That's not the problem.

The problem is that Microsoft is not good at automatic updating, in at least three ways:

1- Several of Microsoft's previous updates have broken things in Windows and applications. That happens to Apple and Google also, but Microsoft typically doesn't care or pretends there is no problem.

2- While Google and Apple provide the details of what's in an update, for Microsoft it's a big secret. Besides security updates, which are described in brief corporation-speak, we don't know what's in an update.

3- Microsoft has a recent history of pushing updates through tricks and tactics usually associated with malware.

For these reasons, many people - including me - simply don't trust Microsoft to be competent and honest about updates. That's the bad part. It's very bad when users feel they have to defend themselves against Microsoft.

That's why I'm sticking with Windows 8.1 for now, although I'm still in the Insiders program so I can continue to use Windows 10 and watch it develop. If I felt I could trust Microsoft to do updates well, I would probably start using using Windows 10 full-time after the summer upgrade.

If Microsoft did updates better, I would be all in favor of automatic updates. One less thing to bother with. Chrome (the web browser) updates seamlessly in the background and I hardly notice it.
