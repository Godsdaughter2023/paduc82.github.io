    xbf erick c say:

    Hey there Roy, 

    Yes the Xbox Adaptive Controller is compatible with Windows 10 computers and also with Windows 8.1 & Windows 7 Pc's but will have limited functionality. Click on the following link for different ways to Connect the Xbox Adaptive Controller to a PC.

    Once you have connected the controller then you can go ahead and Customize the Xbox Adaptive Controller in the Xbox Accessories app and remap the buttons on the controller. The two circular pads on the controller are simply buttons, they do not have any tech that will allow you to use swiping features to type or to be used as the cursor for the PC. 

    When Gaming on a PC using the Adaptive controller you should still see the on-screen keyboard to type messages, You would then use the d-pad or joysticks to type just as you would with a standard Xbox One Controller. 

    The adaptive controller should respond to a toe's touch as the buttons on it are pretty sensitive. In order to get more functionality you can attach different devices using the 3.5mm ports on the back of the controller so that you can then Map these to specific functions.
