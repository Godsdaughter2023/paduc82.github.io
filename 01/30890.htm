a. user say:

Hi RosemarieSwinehart.
My name is Douglas and I am an Independent Advisor.
I would like to help you.

To help you with your concern, let me suggest you do a system restore from Safe Mode. There are three easy ways to do this:

Go to Settings> Update and security> Recovery. In Advanced Start, select Restart Now. From there, select Troubleshooting> Advanced options> Startup configuration> Restart. When you restart, you will see a list of options. Select 4 or F4 to start your PC in Safe Mode (choose 5 or F5 for Safe Mode with network functions).

Press the Windows + R key to open Run. Type msconfig and press Enter. Open the boot tab. Check the box next to Safe Mode. If you need networks, select it from below. Once you press Apply and close the System Settings window, you will receive a message to Restart your system. (Keep in mind that your system will start continuously in Safe Mode unless you turn off the System Settings option, repeat the same process in Safe Mode once you are sure you have solved the problem).

Restart your PC. Press F8 during the boot process to enter Safe Mode. Windows 10 has F8 Safe Mode disabled. You can sacrifice a couple of seconds during startup by enabling the F8 menu using the Command Prompt. Right-click the Start menu and select Command Prompt (Admin). Select Yes in the User Account Control dialogue, if it appears. Command Prompt should now be open. Type (or copy/paste) the following command: bcdedit /set {default} bootmenupolicy legacy And press Enter

To undo this legacy command at any time, reopen the elevated Command Prompt as per the above instructions and type:

bcdedit /set {default} bootmenupolicy standard

This returns startup to its original state

Once you are in Safe Mode, continue and type recovery in the search bar of the Start Menu. Select Retrieve from the list, then Open System Restore from the Advanced Recovery Tools menu.
