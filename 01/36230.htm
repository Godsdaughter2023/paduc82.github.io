In most cases, there's no big breaking differences between 8 and 8.1, the .1 is just an update similar to the Windows "service packs" of the past.  Although there have been cases (most notably Windows XP Service Pack 2) which did cause some breaking issues, they are usually minor.  But graphics programs are some elf the most fickle for sure.

To try the compatibility troubleshooter, right-click the shortcut icon for the program that has the trouble, and the option to troubleshoot should be on the right-click menu.

A couple notes though:

  1. I'm not sure if this option would show up if you right-click an icon on the Start screen (and I don't presently have a way to test this).  If is doesn't, choose "Open File Location" instead, and you'll get a normal folder window where you can right-click the icon again and do it from there.

  2. If the program is already installed and troubleshooting it this way doesn't work, try uninstalling the program, then troubleshooting the Setup file directly when you go to re-install the program.
