It sounds like you got a corrupted Zip file, which can do some pretty funky stuff even without being viral.  Either that, or just as likely a possibility, your hard drive may have bad sectors on it, which could also produce some bizarre results.  The fact that the location could no longer be found is a possible indicator of file system errors.

I recommend running CheckDisk to check for errors first.  Yes, it is time consuming, but if there are errors, finding them now and fixing them will save you from much more headache later on.  Be sure you have a backup copy of your important data before running CheckDisk.  While it is not likely, there is always a potential for data loss when running any utility to repair errors in the file system.  When you are ready, right-click on the drive in question, click on Properties, and on the Tools tab, click Check Now.  Then choose one of the following (each one is more rigorous and therefore more time consuming than the one before):

  • To run Chkdsk in read-only mode, click Start.
  • To run Chkdsk by using the /f parameter, select the Automatically fix file system errors check box and then click Start.
  • To run Chkdsk by using the /r parameter, select the Scan for and attempt recovery of bad sectors check box and then click Start.

  1. Do you have ownership and full permissions on all the files/folders?
  2. You may need to enable the Hidden Administrator Account.  Because of its potential for disaster if used improperly, before you resort to this level (unless you already have), try the DOS method below.

Questions:

  1. What is the exact error you get if you try to delete or rename the files?
  2. I am assuming there is a considerable amount of other data on the drive, so you can't simply reformat it.  Correct?
  3. How steady are your feet when you're walking through DOS?

  • Start an Administrator command prompt.  Quickest way with Process Explorer running, under the File menu, select Run as Administrator, and type in "cmd" and press enter. 
  • Once the command prompt is up, go back to Process Explorer, find explorer.exe, right click on it and select Kill Process.  later, you may not be able to see them.  Also note that while Explorer is not running, your Start Menu and Taskbar are not available.  That's why it's important to have Process Explorer running while Explorer isn't.
