frederik long say:

In newsgroup there was a strong hierarchy of posts which allowed you to see which message somebody responded to. This hierarchy is lost in forums. When addressing a specific respondent it is best to prefix the reply with "To xxx" in order to avoid incidents like the one above.

It is a well-known fact that you can gain access to the files stored on a disk as soon as you have physical access to a computer. This holds for most if not all operating systems, not just Windows.

You can reset a Windows password by going through the steps outlined below. The process is involved and requires some careful typing. Ask a computer-savvy friend to assist you if you lack experience in this area.

 1. Connect the disk to some other Windows PC.
 2. Open a Command Prompt.
 3.  .. 5. (not relevant in your case)
 6. Identify the correct drive letter for the netbook's Windows installation. You do it by typing the commands
    dir E:\Win* {Enter}
    dir F:\Win* {Enter}
    dir G:\Win* {Enter}
    etc.
    until you find the Windows folder.
    I will call that drive "Q:".
 7. Type these commands:
    cd /d Q:\windows\System32 {Enter}
    move  sethc.exe  \  {Enter}
    copy  cmd.exe  sethc.exe  {Enter}
 8. Put the disk back into the netbook.
 9. Reboot and hold down the Shift key permanently.
10. When challenged for a password, tap the Shift key five time. You will see a small black screen.
11. Type these commands:
    net user  {Enter}
    (To see a list of all accounts that exist on your machine)
    net user Administrator {Enter}
    (To see if the Administrator account is active)
12. Type these commands:
    net user "xxx" ppp  {Enter}
    (Replace xxx with the account for which you wish to set a new password, and ppp with the new password)
    net user Administrator /active:yes {Enter}
    (To activate the Administrator account if desired)
13. Log on under this account, using the new password.
14. Reboot the machine into Repair Mode by tapping F8 at boot time.
15. Select the Command Prompt.
16. Type these commands:
    cd /d Q:\windows\System32 {Enter}
    (The correct drive letter is probably C: but it could be D: or E:)
    copy \sethc.exe {Enter}
17. Reboot normally.
