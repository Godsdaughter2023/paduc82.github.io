techfreak_ say:
In Skype any image you see locally is mirrored (horizontally flipped). The recipient end views you normally. This behavior is intentional and no control has ever existed in Skype to reverse this behavior. If you are able to reverse the image using a driver/filter configuration or 3rd party software, the behavior would be reversed. You would seen them normally but the recipient would see you reversed. Using software that allows you to share your video feed may allow you to use alternate software simultaneously so that you can view the local feed non-mirrored, in that software.

The fonts in Windows 10 alright, but I can't read which tab.  How do you increase the size of the fonts on the menu and the favorites bar?

a. user say:

Hi Joey,

Thank you for posting in Microsoft Community.  I understand your concern and I am glad to assist you.

Please try the steps provided below and see if it helps.

  1. Press the Windows key + X.
  2. Select Control Panel from the list.
  3. Click on Display icon.
  4. Under Change only the text size, select Icons. (see screenshot below)
    Select a font size (6 to 24) you want for icon text.
  5. Check or uncheck Bold for what you want.
    When finished, click/tap on Apply.

You will now see a Please wait screen until your text size settings are applied.

If issue persist,

I suggest you update you graphic drivers and check.

   

Step 1. Press Windows and X keys together. 

Step 2. Click on Device Manager.

Step 3. Display Adaptor

Step 4. Right click on the graphic driver and tap on update driver through windows update and check if it helps
