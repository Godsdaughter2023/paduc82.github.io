HOW-TO Run regular Windows Defender Full scans - Microsoft Community

lz. say:

HOW-TO Run regular Windows Defender Full scans

Technical Level : Basic

Summary

Applies to: Windows 8.0, Windows 8.1, Windows 8.1 Update 1

Windows 8.x has an automatic mechanism to perform maintenance tasks when the computer is idle (no user interaction with the system). One of these activities scans the PC for potential virus and spywares. This is achieved by Windows Defender that has been integrated into the system since Windows 8.0.

Windows Defender offers three scanning modes: Quick, Full and Custom. Automatic maintenance actually “asks” Windows Defender to make a Quick scan by default. It is safe given the time a Full scan can take keeping in mind that Automatic maintenance activities run while the PC is idle. If a Full scan would need to be performed during idle time it would probably never end except if the PC remains “On” all time with no user interaction.

There has been a very good discussion here with regard to updating or not the existing Windows Defender maintenance task to force it to perform a Full scan instead of Quick one. At least for the reason mentioned above it's not recommended to modify this task although this remains possible.

Situation

You might want to run a Full virus and spywares scans at regular intervals:

spywares scans at regular intervals:

  • Without having to remember when it was done for the last time.
  • When you sleep to ensure your PC will not be running that scan when you need it.
  • Automatically.

Details

To address the above need a recurring task can be created with the Windows Task Scheduler.

Before you begin

  • The suggested task will only run in the security context of the user who created it - this means:
    • You must have logged on for the task to run.
    • If a different user – than the one who created the task – has logged on the task won’t run.
  • For this article it is assumed a Full scan has to run every Sunday at 01:00:00 AM.

Step 1 – Launch the Task Scheduler

1. Go to Search ([Win] + S or Display the Charm bar and select Search).

2. Make sure Everywhere appears under Search otherwise select the option in the drop-down menu.

3. Type Task scheduler.

4. Click/tap on the Task scheduler icon.

The Task Scheduler should open:

5. In the left panel click/tap on the triangle next to Task Scheduler Library to expand it:

Note: The steps below are not mandatory. There are suggested to avoid mixing system/personal tasks and to retrieve the latter easily:

  1. In the left panel click/tap on Task Scheduler Library.
  2. In the right panel click/tap on New Folder…
  3. Give the new folder a name, i.e. myTasks then hit OK.

Step 2 – Create the recurring task

1. In the left panel select the folder (i.e. myTasks) in which the new task should be stored.

2. In the right panel click/tap on Create Task…

In the General tab:

3. Give it a Name, i.e. Windef FullScan and a Description, i.e. Performs a Full virus and spyware scan every week.

4. In the Security option section select Run only when the user is logged on (1).


In the Triggers tab:

5. Click/tap on the New… button at the bottom of the dialog box.

6. Set Begin the task = On a schedule, Settings = Weekly, Start time = 01:00:00, Recur every = 1 then check Sunday.

7. Check the box Stop task if it runs longer than and enter i.e. 6 hours (2) in the text box of drop-down list.

8. Make sure the box Enabled (at the bottom of the dialog box) is checked otherwise check it:

then hit OK.

In the Actions tab:

9. Click/tap on the New… button at the bottom of the dialog box.

10. Set Action = Start a program.

11. In Program/script type "%ProgramFiles%\Windows Defender\MSASCui.exe" (with the quotes) and -FullScan -ShowSWE (there is a space between the two parameters) in the Add argument (optional) text box (3):


then hit OK.

In the Conditions tab:

12. In the Power section make sure options Start the task only if the computer is on AC power and Stop if the computer switches to battery power are checked otherwise check them.

13. Still in the Power section check Wake the computer to run this task then hit OK.

At this stage the task should be created and Ready to run at Next Run Time:

Step 3 – Check the task works

In the Task Scheduler:

1. Select the task.

2. In the right panel click/tap Run.

The Windows Defender user interface should open and show an on-going Full scan:


Task Scheduler Run Result

The Task Scheduler Last Run Result column provides the exit code returned by the task the last time it ran:

A successful task execution returns code 0x0 has shown above. Other codes are documented in the Task Scheduler Error & Success return codes - See the More information section below.

Notes

(1) Choosing Run whether user is logged on or not – instead of Run only when the user is logged on - is also an option but has implications:

  • The user password will be asked during the task creation – not really a problem.
  • Even if the -ShowSWE parameter is specified the task will run silently (the Windows Defender User Interface won’t display) - See Task Security Context in the More information section for more details.

(2) Setting a maximum run time for the task isn’t mandatory. It is suggested to ensure the task will be terminated after a “reasonable” period making the system totally available to the user it’s needed..
Setting Stop task if it runs longer than 6 hours is a suggestion as well. The value should be adjusted according to the disks’ size and their content.

(3) If the -ShowSWE parameter isn’t specified the task will run silently (the Windows Defender UI won’t display).
