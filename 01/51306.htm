Two questions come to mind. Does your laptop support dual monitors? If it does support dual monitors what display configuration have you set up? Is the external monitor set in extended desktop mode or have you cloned the desktop on both monitors (ie. both have the identical display)?

If the secondary monitor is set up in cloned mode its resolution will be restricted to whatever resolution is set for the primary monitor. In extended desktop mode you should be able to set it to any resolution that the monitor is capable of. There might be a problem if Windows cannot detect (through the video driver) what range of resolutions that monitor is capable of and so has to resort to one of its default generic monitor drivers. In that case try to acquire a monitor driver (usually just a .INF file) from the monitor manufacturer's support web site and install that in place of the generic monitor driver. You should then get the full range of resolutions available for that monitor.

As far as game resolutions go there could easily be a limitation in the resolutions available within the game, as well as in the aspect ratio. In any case I wouldn't expect the game to offer setting choices that were higher than the maximum display resolution set or detected by Windows.
