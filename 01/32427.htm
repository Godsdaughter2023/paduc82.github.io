peterleonard1 say:

Intel Optane is a super fast cache memory "drive" based on Intel 3D-Xpoint technology, it is used to cache the most often accessed files from your HDD and gives your system near SSD performance.

The Intel RST driver reports to the system that the mechanical HDD is an SSD when it is paired with an Optane module and disables defrag by default because you could cause file system issues if you defrag an Optane accelerated system.

If you really must defrag the drive disable the Optane acceleration first,  this  will re-enable defrag.

However it really isn't needed as the NTFS file system is better at handling fragmentation than previous versions of Windows file systems.

There is plenty of good info on this in the Intel Forums, also guides to installing Optane, troubleshooting issues and maintenance of Optane systems.
