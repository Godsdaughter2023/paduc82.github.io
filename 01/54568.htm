darian tab say:

When you install software that uses PROVIDER.DLL, it should automatically register the file for you. In some cases, the DLL file may not register properly, and as a result, it will provide you a "PROVIDER.DLL not registered" error. Fortunately, you can use a built-in utility called "Microsoft Register Server" (regsvr32.exe) to re-register your PROVIDER.DLL file. Here's how:

  1. Click Start.
  2. Type command in the search box, while holding Ctrl + Shift hit Enter.
  3. You will be prompted with a permission dialog box, click Yes.
  4. Type regsvr32 /u PROVIDER.DLL.
  5. Hit Enter to re-register your file.
  6. Close command prompt and restart your computer.

Let us know if you need any other help.
