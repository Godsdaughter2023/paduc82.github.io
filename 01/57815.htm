Wireless Security - Microsoft Community

xb1r athena say:

Wireless Security

Login Information

 

  • Gateway: 192.168.2.1
  • Username: admin
  • Password: admin

 

 

 

How to Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Device Table" from the menu to the left.
  5. Note down the IP Address of the WAP610N. (It will be in the format of 192.168.2.X)
  6. Open a new browser tab or window.
  7. Log into the router by entering the default gateway (192.168.2.x) into the address bar of the internet browser.
  8. When prompted, enter "admin" for the user name and "admin" for the password.
  9. Select "Wireless" from the menu at the top.
  10. Select "Wireless Security" from the sub menu.
  11. If the "Security Mode" is showing either "WPA Personal" or "WPA2 Personal", the wireless passphrase will be labeled as "Passphrase".
  12. If the "Security Mode" is showing "WEP" the wireless passphrase will be labeled as "Key 1".

 

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Device Table" from the menu to the left.
  5. Note down the IP Address of the WAP610N. (It will be in the format of 192.168.2.X)
  6. Open a new browser tab or window.
  7. Log into the router by entering the default gateway (192.168.2.x) into the address bar of the internet browser.
  8. When prompted, enter "admin" for the user name and "admin" for the password.
  9. Select "Wireless" from the menu at the top.
  10. Select "Wireless Security" from the sub menu.
  11. Select the desired wireless security type from the "Security Mode" drop down menu. (To disable the wireless security, select "Disabled".)

marilyn-- say:

Hello Pierre Roman,

 

Thank you for posting to the Microsoft Answers Forum.


It appears you can not remove a sync partnership until you have resolved all discrepancies.

 

If you select each discrepancy and resolve it by either deleting the local or remote copy you can
then select the share and deselect the sync option.

 

Please try the steps below:

1 - Control Panel -> Offline Files -> "Disable Offline Files"

2 - Click the "View your offline files."

3 - Delete folders and mapped drives you no longer want to sync. If the mapped drives are not deleted,
 then go to Windows Explorer or "Computer" select Explore and right click on mapped network drive and
 select "disconnect".

4 - Go back to the Offline Files window and click OK

5 - Restart computer

6 - Control Panel -> Offline Files -> "Enable Offline Files"

 

Please let me know if these steps have helped to resolve your issue or if you need further assistance.
