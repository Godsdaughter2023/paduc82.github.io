Can I return and reuse a Windows license from MSDN? - Microsoft Community

msmigiel say:

Can I return and reuse a Windows license from MSDN?

Hi everyone,

I might soon be purchasing a Lenovo W510 laptop with a blank hard drive. I have an MSDN license for both Windows 7 and Windows 10. While the laptop easily meets the minimum specs for Windows 10, there are no Windows 10 drivers (only Windows 7).

I would like to use Windows 10 as it's newer and will be supported longer but I am concerned that without drivers, a lot of functionality (Fn keys, volume, etc...) might not work.

If I use my Windows 10 license on this computer but later decide to go with Windows 7, can I somehow return the Windows 10 license so that I can use it on another computer in the future?

andre da costa say:

No, you can't. MSDN Windows Licenses are part of a subscription.

If the subscription is not active, you shouldn't be using any of the licenses from it, whether that is Windows 7, Windows 8 or Windows 10.

If your MSDN subscription is active, you can pick, choose and refuse whichever version and edition of Windows you want to use.

You can always perform a clean install to see how Windows 10 performs first, then perform a downgrade to Windows 7 if it doesn't work out.

Fn keys will work whatever version of Windows you are running, those are pre-programmed into the operating system for defined functionality, whether is opening help, find, volume, brightness, scroll lock etc.

Even if there are no native Windows 10 drivers, both Windows 7 and 10 share a relatively similar driver model. So, either Windows 7 or Windows 8 drivers might work.
