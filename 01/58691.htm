EML is a file extension for an e-mail message saved to a file in the MIME RFC 822 standard format by Microsoft Outlook Express as well as some other email programs. EML files can contain plain ASCII text for the headers and the main message body as well as hyperlinks and attachments.

gomtuu say:

FINAL UPDATE: I have resolved this issue. I searched again on the community website for the manufacturer of my Gateway FX 6860-UR20P PC (I had looked here a few months ago) and discovered that a few other owners of this PC had the same problem. They somehow discovered that the problem was caused by the PCI-E wireless WLAN card. Apparently this card (Anatel) is incompatible with any version of Windows 10 beyond release 1607. I would have thought that Microsoft's installer would recognize this hardware as a problem but it did not and the many previous failed attempts at upgrading Windows 10 did not provide a hint as to what the problem might be. There me be many versions of this WLAN card by Anatel. I'm not sure how to accurately identify this particular version. There's a barcode on the back with the numbers; NI10200037124016340100. There's a barcode on the component side with H77D      M1690E6B. There's silkscreening on the board itself: WN7601R-H1

Perhaps this information will help Microsoft or other PC users to identify this card in problem PCs.

At any rate after months of daily automatic attempts to upgrade my PC to release 1709 it has finally succeeded. Since my PC is connected via ethernet cable to my router, I don't need this card and have removed it.
