Connectivity - What is a Proxy Setting? - Microsoft Community

lizfg say:

Connectivity - What is a Proxy Setting?

What does a Proxy Setting mean and should it be used - if so in what circumstances?

sanghati barman say:

What does a Proxy Setting mean and should it be used - if so in what circumstances?

A proxy server is an intermediary between yourWindows 10 PC or device and the Internet. This server makes requests to websites, servers, and services on the Internet for you. For example, say that you use a web browser to visit www.wiley.com and your browser is set to use a proxy server. After you type www.wiley.com, the request is sent to the proxy server. The server then sends the request to the server where the website is hosted. The homepage of the Wiley website is returned to the proxy server which, in turn, returns the homepage to you.

One reason companies use proxy servers is that doing so helps them save precious bandwidth. Proxy servers can compress traffic, cache files and web pages from the Internet, and even strip ads from websites before they reach your computer. This allows companies to save bandwidth, especially when they have hundreds or thousands of employees accessing mostly the same popular websites. Other benefits include improved security and privacy.

james vinyard say: Bluetooth operates in the FM band at 2400MHz and higher to a certain point.  This does not mean it can receive radio stations because it is a FM Radio Controller (frequency range is too high, plus you have no tuner).

To learn more about Bluetooth, go here: http://en.wikipedia.org/wiki/Bluetooth

Your old phone had an actual FM receiver which can use the same antenna in the phone.  Electronically, it is possible though and you need a tuner for a lower frequency range of radio stations in the FM band.

Your alternative is Internet Radio which is actually pretty good.

Since you have a Pro, they do make TV/Radio tuner adapters for computers.  The Windows Media Center is compatible with some of those units and you can look into WMC at Microsoft for more information and a compatibility list of supported devices.  The adapter will plug into your USB port and work from there.

WiFi also operates at a completely different band range as well.  You can read more about both at the above web link.

Do watch out for the adapters you get though!  A lot of the TV/Radio tuner adapters on the market are JUNK!  Most don't work.  Check for a compatibility list before buying one.

henrycoetzee say: Hi guys I also sat with the problem, "black screen" after updating my intel HD graphic driver, after playing around the following solved my problem. (1) plug a extended screen into your laptop, you will notice that you can now see your display on your extended screen (laptop still black screen) (2) Open your device manager, go to your DISPLAY Adapter, open, you will see Intel (R) HD Graphics ....right click then disable........close all then reboot from your extended screen. Unplug extended when laptop is off, your laptop should now reboot normally. Hope it works for you...
