If you are receiving error about administrator access, while turning on SmartScreen, then the issue is probably caused by the UAC settings. If the User Account Control settings are set too high, then it could prevent the administrator from performing such tasks.

User Account Control (UAC) can help prevent unauthorized changes to the computer. UAC notifies the user when changes are going to be made to the computer, which require administrator permission. These changes can affect the security of the computer or can affect settings for other user on the computer.  

 

Let us try lowering the UAC settings and check if you are able to enable Windows SmartScreen. Please refer to this link to do so:

b.  Click on User Account Control settings.

d.  Click on Yes to confirm.
