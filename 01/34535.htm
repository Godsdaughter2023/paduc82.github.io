So my reply was the answer?
  
Put very simply: you don't have the permissions security on exFAT like you do NTFS.  NTFS is a security aware file system, exFAT - not so much.  In many ways it's not all that much better (for security) than FAT32.  Since your issue centered around security blocking your desired functionality - get rid of the security, gain back the functionality.

My new suggestion is for you to get a drive/external media large enough to keep a backup of your system. That can be whatever file format you want, but I suggest some NAS that you use exclusively to backup to.  With that much, I am pretty sure losing it would be at least a bad day for you.
