Hi Francois,

 

 

Here are some steps you may try and check if it helps.

 

Before following the steps, please download the drivers and save it on desktop.

 

Visit the NVidia website and download the drivers. http://www.nvidia.com/object/win8-win7-winvista-64bit-314.22-whql-driver.html

 

1.  Make sure that the Windows update do not search for new driver updates.

 

- Press Windows key and R key to open the Run prompt.

- Type SYSDM.CPL and press enter.

- Select the hardware tab.

- Click ion Device installation settings.

- Select "No, let me choose what to do".

- Place a check mark on "Never install driver software from Windows Update"

- Press apply and ok.

 

2. Uninstall  the NVidia driver from safe mode.

- In the Run prompt, type msconfig.exe.

- In the Boot tab, place a check mark on Safe boot.

- Press ok and restart the computer. It will boot in Safe mode.

- In the Run prompt, type devmgmt.msc and press enter.

- Once all the drivers and associated software is uninstalled from the computer, restart the computer.

 

Note: Install the driver in Safe mode.

 

Once the driver in installed, restart the computer in Normal mode. In msconfig, uncheck Safe boot to boot normally.
