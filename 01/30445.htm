taidara say:

Hey Dino!
You don't have to disable the cloud if you wan't to use a cloud save game on another console. The cloud you have is connected to your profile. So it doesn't matter which console your using as long as you sign in with your profile you can access the cloud anywere.

Use a cloud game save on another console

  1. Sign into your Xbox LIVE profile.
  2. Turn on Cloud Saved Games.
  3. Start the game with the cloud game save.
  4. When prompted, choose Cloud Saved Games as your storage device.
     

Now you can play on the other console, right where you left off. When you’re done playing, your game save is stored in the cloud.
