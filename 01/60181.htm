mark ole say:

Hi,

Frequent places is a list of folders and websites that you have opened recently. These items are displayed as "Frequent folders" in Quick access folder. In case you're unable to pin any folders to Quick access, you might want to check if the "Show recently opened item in the Jump lists in Start or the taskbar" is On.  To do that, follow these steps:

  1. Open Settings, click on the Personalization icon.
  2. Click on Start on the left side. Under " Show recently opened items in Jump Lists on Start or the taskbar"  toggle it to On.
  3. Close the Settings windows and try to pin the any folders to the Quick access folder.

If still the issue occurs, run the System File Checker tool. It s a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files. To run the tool, follow these steps:

  1. Right click on the Start button and choose Command Prompt (Admin)
  2. At the command prompt, type the following command, and then press ENTER:  sfc /scannow

Let us know how it goes after performing the suggested steps.
