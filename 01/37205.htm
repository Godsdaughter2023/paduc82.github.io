¡firedog say:
 
...
while signed in my events are displayed in the WLM calendar.  The events are not displayed on the Outlook Calendar.  However, when I sign out of the Microsoft account, the events disappear..
That's because they're two different calendars. When you're not signed in, you see the so-called default calendar. When you are signed in, you see the account calendar - the one associated with the Microsoft Account you signed in with. Sign in with a different Microsoft Account and you'll see yet another calendar.
Up to this point I have never signed in when using WLM..
You have experienced a curious phenomenon whose origin is shrouded in mystery. A user who never signed in is using the default calendar. However, if he should suddenly decide to sign in, the calendar entries are automatically transferred to the account calendar, and the default calendar is emptied. When the calendar was first introduced back in 2008, it was the intention that it should synchronize with the one at calendar.live.com and so be available online from any computer with an Internet connection. This synchronization was broken when outlook.com became Outlook Mail a couple of years ago. Because of this, new reminders probably won't work for entries made in Windows Live Mail; they will have to be configured in the online calendar.
I am using Version 2011 (Build 15.4.3555.0308).

Windows Live Mail 2011 has been obsolete since 2012 on all versions of Windows later than Vista. However, upgrading to the current version (16.4.3528.xxxx) on a newer OS will not help with your calendar woes.

davem121 say:
Hi GenericUser21,

When using Windows Reset, if the original Windows 8.1 recovery partition still exists, then more than likely your system will revert to Windows 8.1

A much better option is to backup your data and clean install Windows 10:

You can reinstall Windows 10 at any time and it will not cost you anything !

You should just clean install Windows 10 rather than upgrading from Windows 8.1. This will have 2 benefits, if you have to reset again, it will reset to Windows 10, not 8.1 and your system will be more responsive and reliable . . .

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
