a. user say:

Hello,

As I understand, you want to login to Windows 10 without entering a password.

I would like to inform you that,  if you are using a Microsoft Account, password is required and cannot be removed. However you can create a local account and disable the password prompt on login.

Refer to the below link to create a local account

https://support.microsoft.com/en-us/help/13951/windows-create-user-account

Follow the steps below to remove the password protection for a local account.

a)    Open the run box by pressing Windows + R key, type netplwiz and press enter.

b)    This should open User Accounts box which will display a list of all users on the computer. From here you can add, remove or see properties of any user.

c)     Select the user from this list and uncheck “Users must enter a user name and password to use this computer” and then click on Apply button.

d)    Now you will be presented with a dialog box which will prompt you to enter password. Do so and then click on OK.

e)    Also click on OK in the User Accounts dialog box.

Note: it is not recommended to remove the password. Please remove at your own risk.
