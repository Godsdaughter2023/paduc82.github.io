System Restore points are automatically created when the following actions occur:

  • When you install a new application or driver.
  • When you uninstall or install certain programs.
  • When you install new updates
  • Automatically on a daily interval
  • By manually creating one.

You can try the following steps to change the boot loader in Windows 8.1.

 

Method 1:

Let us try the steps given below and check if it helps.

 

a)     Press Windows key + X, Select Command prompt (Admin)

b)     Then try running the command.


          bcdboot C:\Windows

 

Note: C should be the letter for your Windows 8.1, change it if it's not.

 

That command will replace the boot files with the ones from Windows 8.1, set the Windows 8.1 menu entry as the default one.

If you need to remove Windows 8.1 from Windows XP, then you can remove the entry in the Windows XP Boot Configuration.

1.  Click Start
2.  In the Search text box, type msconfig
3.  Launch msconfig by selecting it from the search results above the search text box.

After msconfig is launched:

1.  Go to the Boot tab
2.  Select the boot entry you want to delete.
3.  Click the Delete button and then click OK.
