I have since sold this PC  and tried to remove windows 7 but could not reset  the pc to Factory settings ( only option was for a clean install of windows)

Does this mean I can't use this disc and keycode to install windows 7 on my new pc?

bill smithers say:

I had an old pc with windows 7 OEM   system builder pack installed on (upgraded from XP)

I have since sold this PC  and tried to remove windows 7 but could not reset  the pc to Factory settings ( only option was for a clean install of windows)

Does this mean I can't use this disc and keycode to install windows 7 on my new pc?

You can't transfer an OEM System Builders License; so the answer to your question is NO:

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

meanmachine 47 say:

Hi all!

 

Have any of you tried clearing the update cache on your xbox 360's? This could help with the freezing issues that you're experiencing. As for the "clicking", this could be a number of things but I would say it's just your console's processing unit reading the information from the disc. A lot of equipment with a processing unit will make these kinds of noises, such as: PCs, DVD & CD players and even iPods. It's nothing to worry about but if you really find it to be that much of an irratant installing your games to the hard drive may help.

 

"OK! That may be the reason as to why my console is clicking, but why is it freezing?" Right, the console reads a lot of information when running your games (updates included) and when performing other tasks. If an update is damaged or incomplete it can causes system freezes/crashes. Think of when playing a game on your PC, if you eject the disc while in-game the system freezes and closes the game as there is no further data for it to continue - If an update is damaged or incomplete when performing tasks on the xbox 360 it may freeze and crash due to there being no further data to run the process.

 

Clearing the update cache will allow the console to refresh and download a new update(s) which may be faulty. If you need help clearing the cache follow these steps:

 

  1. Navigate to the "My xbox" slide on the dashboard.
  2. Scroll right, to the last page which is the "System Settings" page.
  3. Select "Memory" from the list of options.
  4. Highlight the hard drive and hit Y on your controller.
  5. Select the option to clear the update cache and perform maintenance on the memory unit.
  6. Restart your xbox 360 system.

 

Hopefully this should resolve the freezing issues.
