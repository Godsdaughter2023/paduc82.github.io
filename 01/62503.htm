Windows 10 signs you into a temporary user account when your own user account has been damaged beyond repair. This gives you an opportunity to create a new user account. Once you have a new user account, you can transfer your personal files (e.g., documents , music, pictures, videos, downloads) from the old user account to the new user account, and then you can delete the old user account.

This is a big improvement over previous versions of Windows, where you were SOL (sadly out of luck) if your user account was damaged.

Here's what you need to do, including tips I learned by scr*wing up the first few times:

1- Sign in to a user account with Administrator permissions.

2- In the Administrator account, create a new Standard user account for yourself.

3- Still in the Administrator account, use File Explorer to copy your personal files from your old user account to the corresponding folders in the Public user account ... C:\Users\Public

4- Make sure you copy everything you want to copy. Don't ask how I learned this.

5- Sign out of the Administrator account and sign in to your brand new user account.

6- Move your files from the Public user account to the corresponding folders in your new user account.

7- Test your new user account by opening a dozen or so different types of file (documents, pictures, etc.) to make sure they open properly.

8- Once you're satisfied with your new user account, you can - if you wish - sign in to an Administrator account again and delete your old, damaged account.

With Windows 8, Microsoft had changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key.
