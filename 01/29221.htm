wunders say:

This error would indicate that the underlying command executed when you try to renew is the command "ip" which is not a recognized Windows command.  This leads to the question as to how you are renewing the connection.  (Normally you do not have to renew connections as they should auto-renew when the lease is 1/2 expired).  Try renewing it manually from a command prompt.  The errors are usually more descriptive...

Bring up a command prompt window (Start -> Run -> "cmd")
Enter the command:
  ipconfig /renew

See if any errors are generated when you attempt to renew this way.  If you do get errors, try referring to this article:

"Error Message When You Run the "Ipconfig /Renew" Command"
  < http://support.microsoft.com/kb/810606 >

  JW
