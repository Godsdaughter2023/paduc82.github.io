You should have two e-mails accounts now on Windows Mail.  They should keep separated without any effort on your part.  All you need to do is switch between them depending on which one you want to use.  One will be set as the default and will be what will open when you start Windows Mail and will also be the one used when you click on an email link on a web site (for example) to generate the reply dialog box - I assume in your case you want that one to be AOL so make sure that's the case (Tools / Accounts / Mail and see which one says default and if it's the wrong one click on the account you want and click on Properties and set it as the default account).

Besides leaving the emails on the AOL server, you can also change the time that they remain there.  If you check just leave them there, then you will need to periodically go to the site to clear up what is on the server so you don't run out of space (and again lose e-mails but for a different reason).  If you increase the timing to long enough that you are certain you will access and download the AOL messages before the time runs out, then you need not worry about periodically clearing the server (and you should also get all your e-mails - provided you remember to change the setting if you go on a long vacation beyond the timeframe you established or set it long enough so even that situation is covered except for maybe a rare very long trip).
