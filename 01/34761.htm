I managed to get file History to backup my SD card by first formatting it in NTFS and then adding Authenticated User to the permissions.

Right click the NTFS formatted SD card and select Properties then the Security Tab, under Group or User name click the Edit button then Add and enter Authenticated User followed by Check Names, OK etc.

Then create a folder on the SD card, such as SD-Documents, and right click this folder and select Include in library > Documents or what ever library you want the folder added to.

a. user say:

Hi Andy,

Thank you for your interest in Windows 10.

I suggest you to follow the steps below to uninstall and reinstall Mail app and check if it helps:

  1. Press Windows key.

  2. Type Windows PowerShell.

  3. Right click on it and click on Run as Administrator.

  4. Type in the following command ‘get-appxpackage *microsoft.windowscommunicationsapps* | remove-appxpackage’ and press Enter.

  5. Once app is uninstalled try to reinstall the app from Store.
