patrick barker say: Hi,

I assume by I/Q error you mean I/O error? Run a chkdsk on the Seagate drive:


start>search bar>type cmd right click and select run as admin

Elevated CMD should now be opened, type the following:

chkdsk x: /r

x implies your drive letter, so if your hard drive in question is letter c, it would be:

chkdsk c: /r

Restart system and let chkdsk run.

a. user say:

Hi,
 
Thank you for posting in Microsoft community.
 

I suggest you to Run the Home Group troubleshooter and check.

1) Type Network and sharing center in the search box and open it in Control Panel.
2) Now in the left pane, click on Home Group and click on start the Home Group troubleshooter and follow the on screen instructions.


Next follow these instructions and before doing that please make sure that you have these options enabled.

Go to Network and sharing center and click on Private and then make sure you have these options enabled:
1) Turn on network discovery.
2) Turn on file and printer sharing.
3) Allow Windows to manage home group connections.

Then collapse Private and expand Guest or Public and make sure you have these options set:
1) Turn off network discovery.
2) Turn off file and printer sharing.

Now you can follow these steps to manually set to private network connection.

1) Press Windows key + r and type secpol.msc.
2) Now click on Network list manager policy.
3) Double click on Network in the right pane.
4) Here you can manually change the network location from Private to Public and vice versa.
 
Hope this helps. Get back to us for further assistance.
