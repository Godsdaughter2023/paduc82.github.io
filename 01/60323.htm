Hi Tim

Your initial description is not what was called the 'classic style' in Windows XP. It was called the Windows XP Style, which was also called the Luna Theme. This was the default theme in XP. It had 3 basic color schemes, Default Blue, Olive Green, and Silver. You could also change most of the colors for individual components, using the Luna theme. By design, this Luna theme, as well as many other themes made for XP, such as Royale, Zune, Media Center, etc, are not available, and not compatible with Windows 7.

The Classic Theme in Windows 7 is exactly the same as it appeared in Windows XP. This is what Windows 98, 200, etc used as their basic style. See the following link.

Windows XP themes

The bottom line here is that you are trying to accomplish something that is not part of the design for Windows 7.

There are many third party applications that can make Windows 7 look like XP. Some of these applications have ported the XP Luna theme to make it work in Windows 7.

Use your favorite search engine to locate these applications.

mann_manohar say:

Hi Margaret

 

I understand you are trying to create a recovery disc for your Sony Vaio Computer that came with windows 8 preinstalled

 

1.      What happens when you try to create the recovery media?

2.      Is there any error message or code you are getting while creating recovery media?

 

 

Here are the steps you can follow to create a recovery disc

 

1.      Press Windows + C on the keyboard to get the charms bar

 

2.      In the charms bar select Settings, and then in the search box type Recover

 

3.      Select "Create a recover drive option"

This opens the Recovery Media Creator wizard that will enable you to create recovery media.

4.      Once you to select "Create a recovery drive option" Recover Drive window will pop up.

 

5.      After clicking Next, you’ll be prompted to connect a USB flash drive, or you’ll see a list of available drives. If you’d rather use optical media (CD-RW or recordable DVD), you can click the link titled Create a system repair disk with a CD or DVD instead.

 

6.      After selecting the drive, click Next. The wizard will warn you that everything on the drive will be deleted as part of the formatting process. Click Create.

 

7.      The wizard will prepare and format the drive, and then copy over the utilities and other files that make up the recovery environment, then create the recovery media.

 

 

 

You can also refer this Microsoft link

http://windows.microsoft.com/en-IN/windows-8/create-usb-recovery-drive

 

 

 

Hope this helps, please let us know the results. If you need any further assistance on this, please write us back, we will get back to you as soon as possible.

 

 

Thank you

arnemortensen say:

Found it!

Use -> Devices and Printers. Select a Printer, and a tab called Print Server Properties will show up.

Click the tab and the oldfashion tool will open, and from there just click the Forms tab -> Create Form and enter the name and values needed for the new form, the hit Save form and close out.

 

Simple once found.
