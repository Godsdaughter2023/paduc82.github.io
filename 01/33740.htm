idispose say:  

I recently purchased Microsoft Wireless Mouse 4000 for my Laptop DELL Latitude E6410. The laptop came with the default pointing device config application. I do not have Microsoft Intellipoint software. I have Win7 64 bit.

Here are the problems with the scroll wheel that I noticed when I bought Microsoft Wirelss Mouse 4000

o) The scroll wheel does not scroll through the combobox items when the combo is NOT dropped down but has the input focus.

e.g. Control Panel -> Mouse -> Pointers -> "Schemes" combobox.

o) The scroll wheel does not scroll the ribbon of MS Office 2007.

o) In Word 2007 the Ctrl+Scroll wheel does not zoom in out

o) The scroll wheel does not scroll the XML Notepad 2007

Solution:

This solution is not mine - it was there on internet already.

It worked for me.

I am posting this with additional steps for 'how to get there'

Go to Mouse Properties -> Hardware tab

Select the problematic mouse from the list "HID-compliant mouse"

Click "Properties"

Go to Details tab

Select the "Device Instance Path" from the combobox.

A value will be displayed e.g.: HID\VID_045E&PID_0745&MI_01&COL01\8&5538EC&0&0000

Note this value.

Open registry editor and navigate to:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\HID\VID_045E&PID_0745&MI_01&Col01\8&5538ec&0&0000\Device Parameters

The "Device Instance Path" is the path of the reg key that corresponds to this instance of the mouse.

In the Device Parameters, add the following DWORD (32 bit) reg keys

HScrollHighResolutionDisable = 1

VScrollHighResolutionDisable = 1

Delta = 120 (decimal)

Remove the transreceiver and put it back (this kind of re-initializes the driver). You can choose to restart the PC, too - whatever.

This should work.

Initially I had not put the "Delta" key. But because of that it was scrolling as if I pressed the "Home" and "End" keys i.e. either all the way down or all the way up. After putting the "Delta" key, it worked.

Hi,

Screen might dim after few moments on the computer if the setting to dim the display under Power options is enabled. Apart from this setting, Windows adaptive brightness option could also cause such scenarios. Adaptive brightness is a feature in Windows that uses a light sensor to automatically adjust the display brightness to match the lighting conditions in your computer's surroundings. To use adaptive brightness, you must have light sensors installed and enabled on your computer. Before we start troubleshooting the issue, check if the laptop enters sleep mode (if the screen becomes completely black). You may refer to the section "Make your PC sleep" to know the settings and then disable them (if they are already enabled).

If that is not the case, let’s check if the brightness setting under the power plan that is setup is set for the display to dim after a certain period of time. Follow the methods listed below:

Method 1:

Here’s how to check and disable the setting for the display to automatically change for a specific power plan:

  • Press Windows+X keys and select Power options.
  • Click on Change plan settings.
  • Set Dim the display to Never for when you are On battery and Plugged in.
  • Click on Save changes.

Also, refer to the article to disable Adaptive brightness on your laptop:

Refer to the section “To turn adaptive brightness on or off”:

http://windows.microsoft.com/en-us/windows/adjust-monitor-brightness-contrast#1TC=windows-7

Note: To open Control Panel, press Windows+X keys and select Control Panel. Choose Power options. The reset of the instructions also apply for Windows 8.1.

Restart the computer and check if it fixes the issue. If not, proceed to the next method.

Method 2:

It might happen that the power plan that you have setup could be corrupt. Let’s try creating a new power plan and verify the results. Refer to the article to know how to create a different plan under Power options:

Refer to the section “Can I create a power plan?”

http://windows.microsoft.com/en-us/windows/power-plans-faq#1TC=windows-8

Issue related to mouse and keyboard getting disabled frequently:

The USB devices might go into the suspended mode. It is as if when the computer goes into sleep mode, the devices connected through the USB ports might enter into selective suspended mode or a lower power mode. Here’s how to disable the setting under Control Panel:

  • Press Windows+X keys and select Control Panel.
  • Next to your currently selected plan, click Change Plan Settings.
  • Click Change advanced power settings. Click on Change settings that are currently unavailable.
  • Click the box to expand USB Settings, then expand USB selective suspend settings.
  • Click Plugged in, then click the drop down menu and select Disabled. Check for the same steps for hard disk tab as well.
  • As you're using a laptop, click Battery, then click the drop down menu and select Disabled.

Reboot the laptop and verify the status of this issue.  

For reference:

Make sure that the display driver installed on the laptop are updated. Refer to the Dell’s website and download the latest driver for the display card for Windows 8.1.

http://www.dell.com/support/home/us/en/19/product-support/product/alienware-18/drivers

Check if it fixes the issue. Here’s an article to know how to update drivers on Windows 8.1:

http://windows.microsoft.com/en-us/windows-8/all-drivers
