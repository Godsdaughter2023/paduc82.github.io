indyraceboy say:

PXE-E53: No boot filename received.......next line says: PXE-MOF: Exiting PXE ROM__

Please help!  My computer will not boot up at all!!!!

malkeleah say: The computer cannot find any bootable devices so it tries the last choice - a PXE server. (A PXE server allows computers to boot over the network and you don't have a PXE server.) In effect, your hard drive has died. There is the possibility that the motherboard has died and not the hard drive, too. You can test this by removing the hard drive, putting it in a USB drive enclosure, and attaching it to a working computer. If the hard drive is visible, you can test it with a drive diagnostic. If the drive tests fine, then the motherboard on the original computer has failed. If the drive isn't visible or doesn't pass diagnostic tests, it has died. Connect another hard drive to the original computer and if it is seen, the motherboard is fine.
