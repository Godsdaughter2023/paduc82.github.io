umesh p say:
Hi,

Make sure that the disk that you're restoring the backup to is the same size or larger than the disk that you backed up. But you may not be able to restore it to the disk whose size is smaller than the disk that you backed up.  
When you restore your computer from a system image backup to a disk that's larger than the original disk that was backed up, the disk size shown in the Computer folder will be the size of the disk that was backed up rather than the actual size of the disk. For example, if you backed up a disk that was 156 gigabytes (GB) and you restore it to a disk that's 200 GB, the disk size in the Computer folder will be 156 GB.
http://windows.microsoft.com/en-US/windows7/Reclaim-disk-space-after-restoring-your-computer-from-a-system-image-backup
 
If you try to change the system image file name then, you may not be able to use it. Instead you may change the folder name, in which this system image file is stored.
On internal and external hard drives, older system images will be deleted when the drive runs out of space.
Refer: http://windows.microsoft.com/en-US/windows7/Back-up-your-programs-system-settings-and-files
