How to exclude search results where location is only reason search - Microsoft Community

capitalchrisgrossman say:

How to exclude search results where location is only reason search term was found

Is there a way to exclude from the search results any results where the search term only showed up in the file location/path?

For instance, I search for the term "XYZ" and am only interested in the results that include XYZ in the file name (not path) or in the file contents.  I want to have the search screen results that show up only because "XYZ" was found in the path.

cmdrkeene say:

how to set folder views site:cmdrkeene.com

That site:cmdrkeene.com tells your Internet search to only look at that one site for those terms.  Windows lets you do the same thing across many fields to choose what to search.  So for your case, type it like this:

filename:xyz

Or if your file name has spaces in it, type it with quotes, like this:

filename:"xyz project"

This will find a file if it has those in the file name specifically.  This would include a file like "WXYZ Project" because it still has those letters in it.  If you want it to ONLY have those letters, nothing else at all (exact matches only), you can put a ~ after the colon.  That means "file name must exactly be", so this:

filename:~"xyz project"

Although that would be unlikely to find anything since most files have something after the name (like .doc or something), so you might use a * wildcard just like the old days:

filename:~"xyz project*"

--------

That's probably more than you wanted to know, but the first example is what you need :)
