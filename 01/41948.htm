Method 1:

Open Windows Media Player Settings troubleshooter

Open Windows Media Player Library troubleshooter

 

Method 2:

You may try disabling and enabling Windows Media Player (WMP) and check if it helps.

a. Click the Start button, click Control Panel, click Programs and features, and then click Turn Windows features on or off.

b. Uncheck and re-check the option for Media features, or Expand Media features, Turn off Windows Media Player option by unchecking this option. Wait until the feature is disabled, click OK and restart the computer.

c. Once the computer is restarted, turn it on in the same way by checking Windows Media Player.

d. Restart the computer and check if the problem persists.

For further information, refer the article:

Turn Windows features on or off - Windows 7

 

Method 3:

If the issue persists, try performing a clean boot and check if it helps.

A clean boot helps to verify if any third party application services or startup item is causing this issue.

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7
