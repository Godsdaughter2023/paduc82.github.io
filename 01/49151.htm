There are several reasons  why you are getting an error when launching the Game bar on your Windows 10 PC. To resolve your concern, here are methods that you can perform on your computer:

Method 1. Run the troubleshooter for Windows Apps on your computer by clicking the run troubleshooter button on this link.

Method 2. Reset Windows Store to clear store cache.  

  1. Press Windows key + X on your keyboard.      
  2. Click on Command prompt (Admin).      
  3. Type wsreset.exe, and then hit Enter.      

Method 3. Uninstall and reinstall Xbox app to update it.  

  1. Click Start, and then type Windows PowerShell on the search box.     
  2. Right-click, and then select Run as administrator.     
  3. Type the following command to uninstall Xbox app: Get-AppxPackage *xboxapp* | Remove-AppxPackage     
  4. Hit Enter.     
  5. Launch Windows Store and download the Xbox App here.     

Method 4. Disable the Game bar on Task Manager to completely restart the process.

  1. Press the Windows key on your keyboard, and then type Task Manager in the search box.
  2. Hit Enter.
  3. Under the Processes tab, look for Game bar, and then click on it.
  4. Click the End task button.
  5. Press Windows + G again to launch the Game bar.
