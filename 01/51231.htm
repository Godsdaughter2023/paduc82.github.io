You can use the syntax:

 

filename:xyz*.mdl

 

Note that file:xyz*.mdl is equivalent to filename:xyz*.mdl, but an extra symbol ~ is needed with the supposedly equivalent name:~xyz*.mdl to properly handle the wildcard.  (filename, file, and name are supposed to be the same thing)
