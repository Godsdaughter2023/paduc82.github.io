I recently found out a great way to play rtmp feeds in Windows Media Player 12.  Follow these steps:

4:  Type in the rtmp url in a separate cell

 

5:  Copy the rtmp url and append it to the string in step 3

 

6:  Copy the concatenated url string from Excel

 

7:  Paste the string into Windows Media Player's "File>Open URL" Menu
