rajithr say:

Hi,

Thank you for the post.

1. Did you setup your Windows 8 with the new Outlook.com address?

2. When you say 'email icon on windows8', are you referring to Mail app? The Mail app by default opens the same account that you use to sign-in to the Windows 8.

3. Are you accessing Outlook.com on Internet Explorer? Internet Explorer also automatically sign-in to the Outlook.com account that you use to sign-in to Windows 8. Here you have an option to sign-out of the account and sign-in to the old Outlook.com account.

If you want to change the Windows 8 user account to using an older Outlook.com account, then, use the Windows 8 feature to switch from the Microsoft Account to Use a local account.  This will ask you to log out and log back in with the new local account credentials. Now, do the same again and switch back to use a Microsoft Account.  It will then allow you to enter the correct(old Outlook.com) account username and password.

 

Follow this article and refer to the section 'To switch from a Microsoft account to a local account'
http://windows.microsoft.com/en-gb/windows-8/disable-remove-password

Note: If you have setup any apps using the new Outlook.com account, it will not be shown using you sign-up with the older Outlook.com account.

Hope this information helps. If you have further queries, we will be happy to help.
