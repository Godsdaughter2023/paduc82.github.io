considerate_guy say:

The last version of Windows that included Windows DVD Maker was Windows 7. There is no way to install it on a later version. In fact if you upgrade to Windows 10, that program is no longer in your Program Files folder. So if you don't see the option, you can't add it.

Microsoft dropped support for Windows DVD Maker awhile ago. It's similar to what Apple did for iDVD and the iLife Suite. My guess is that they view video DVDs as a legacy format (only SD resolution 480p and no HD support).

If you still need that format (say because someone only has a DVD player and no Blu-ray or HD streaming support), then you'll have to look into 3rd-party applications like DVD Flick, DVD Styler, etc.

P.S. I'm not sure what you downloaded called "Windows DVD Creator" but it isn't the program that Microsoft created. There is no separate install of Windows DVD Maker so it's something else. If it is a program for making video DVDs, you can save your movie from Movie Maker to your PC. Then give that to your other program for making a video DVD.
