Hi Kiron0,

 

Did you remove Japanese language from the Installed Services under Keyboard and language in the Control Panel or you just switched the language from Japanese to English in the language bar?

 

If you’ve just switched the language bar from Japanese to English and not removed from Installed Services, then you first step of troubleshooting is to remove Japanese from Installed Services completely.

 

Step 1: To make English as the default Input language

a.    Click on Start; Go to Control Panel.

b.    Switch to Classic View.

c.    Click on Region and Languages.

d.    Switch to Keyboard and Languages.

e.    Click on Change Keyboards.

f.     Under Installed Services on the General Tab, select Japanese (If you still have Japanese under Installed Services) and click remove.

g.    Click on OK to save changes.

h.    Restart the computer and then check if the programs install in English.

 

Step 2: Perform a System File Checker

 

The System File Checker tool (SFC.exe) scans for missing or corrupted system files and repairs them. Visit the link below to know how to perform system file checker scan on your computer.

 

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows 7

http://support.microsoft.com/kb/929833

 

Step 3: Re-register Windows Installer

 

If the programs that you’re trying to install use Windows Installer, then I would ask you to re-register Windows Installer.

a.    Click on Start; in the start search type cmd

b.    Right click on cmd.exe and choose to “Run as administrator”.

c.    Execute the following commands. After typing each command, press Enter.

msiexec /unregister

msiexec /regserver

d.    Close the command prompt window once you successfully re-register Microsoft Windows Installer.

e.    Restart the computer and then try to install programs.
