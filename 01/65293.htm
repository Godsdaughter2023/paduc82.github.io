nirmal s say:

Hi karamabo,

 

 

Method 1-

 

You need to perform the steps from the article mentioned below-

 

How to use the Bootrec.exe tool in the Windows Recovery Environment to troubleshoot and repair startup issues in Windows

http://support.microsoft.com/kb/927392

 

 

Method 2-

 


You need to run a CHKDSK on the computer to check for errors on the hard disk.
To do so, you need to follow the steps mentioned below-


a) Restart the computer and tap f8 on the keyboard till you come up with the advance boot options.
b) Now select Safe Mode with Networking.
c) Now open Command Prompt and copy and paste the command mentioned below-

 

Chkdsk /r /f


d) Restart the computer wait for the process to complete.

 

If you are unable to boot into Safe Mode with Networking then enter the Windows Recovery Environment by referring to the link mentioned below-


What are the system recovery options in Windows 7?
http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7


Now select Command Prompt and perform the steps from c) to d) as mentioned earlier.
Note: The Chkdsk scan will try to identify and repair and bad sectors found on the hard drive. Any data present on these sectors might be lost in the process. The computer can also go into a no boot state. Proceed at your own risk.

 

If none of the steps help fix your issue then, you need to re-install Windows.


You may do so by referring to the article mentioned below-


Installing and reinstalling Windows 7
http://windows.microsoft.com/en-US/windows7/Installing-and-reinstalling-Windows-7#section_7

 

Note: Make sure you perform a “Custom” install during the reinstallation of Windows and select the same partition as to where the previous version of Windows was installed.

This will save the older Operating System in a folder called “Windows.Old”. Data present in all the default folders (documents, downloads, pictures etc) will be saved in the process.

 

Microsoft is not responsible for any loss of data during the custom or clean installation of Windows.

 

 

Hope this helps!!
