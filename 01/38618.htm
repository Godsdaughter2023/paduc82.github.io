marites_p say:

Hi,

 

First, make sure that you are using the latest version of Skype. Also, please take note that the Skype for Windows 10 adapts the audio settings of your device.

Any changes or adjustment you can check under your native settings. 

 

Also, you can update your drivers in Windows 10.

 

To update:

  1. Select the Start Windows logo, type Device Manager, and select it from the list of results.
  2. Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver. For graphics cards, expand the Display adapter’s category, right click your graphics card and select Update Driver.
  3. Select Search automatically for updated driver software.
  4. If Windows doesn't find a new driver, you can try looking for one on the device manufacturer's website and follow their instructions.

 

If these steps don't work, try reinstalling the driver: Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall. Restart your PC and Windows will attempt to reinstall the driver.

 

If the issue still persists, do not hesitate to contact us back.

 

 

Thank you!

 

Marites P
