That is quite likely what is causing the issue, but that also means there's not such a great single solution.  Basically when the monitor changes, the desktop layout tries to accommodate it.  For example if you had two screens and some icons on both -- if you unplugged one of the screens, the desktop will try to move all the icons so that they are still visible on the one remaining screen.  That's just an example, the same thing happens if you only have one monitor.
