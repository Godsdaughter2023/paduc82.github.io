how do you find your WAP passphrase - Microsoft Community

lynnclayton say:

how do you find your WAP passphrase

I need awap passphrase to make my new printer wireless.  Where is ist/ How do I find it?
kosh vorlon say:

Hi Lynn,

 

This relates to the Wireless encryption security and it is located in the security section of your wireless router's configuration settings (though it is probaby not displayed and requires you to know it - sort of like a password).

 

Log to your wireless Router, check the configuration for the unique SSID (your Wireless connection name) and go to the security configuration section to set the Router's Wireless Security with a key (or to change the key, but you may need to know the old one to change it to a new one - though you'll see below that I didn't need to know the old one), and if necessary (and it will be), type the key in the laptop/printer when configuring it to connect to the router. You will not be able to access this key though the computer after input - it's just to establish the ability to connect. There's a blank where the key has been input, but it converts the actual key to a string of dots so it can't be read. If you change the key through the router, you can change the key through the computer by putting in the new one and once saved it will work (but will again revert to a string of dots so you can't read it either).

 

To configure the Wireless Router read its extended manual or go online and check out the manufacturer's website, forums, and/or technical support. I'm not entirely certain, but if you can login to the router's setup pages, you can probably change the key and I don't know if you will need to know the old key to set a new one or not - it may work or it may not or it may vary depending on the router. Usually if your login information is forgotten, the router can be reset and reconfigured with the default settings - but you'll need their help to do that as well as the default settings use default logins and passwords. Many use admin or administrator as the default username and admin or administrator or password as the default password. Once you gain access, check for where the key is set and see if you can change it without knowing the old key (or in your case, just be able to see it) or find out how to do it from them before you let them off the line. In some cases, your ISP's technical support group can assist you with this if they know about and support that particular router.

 

It's usually a matter of opening your browser and where you'd normally type an address (obviously while connected to the network), instead type 192.168.1.1 (or whatever your default gateway is which is in such a situation the address for the wireless router). If you don't know it (and what I just used which is the most common one didn't work) but can still access the network, then go to Start / All Programs / Accessories / and right click on command prompt and click run as administrator and then type IPCONFIG /ALL and enter and look for the wireless connection and the router's IP address should be the default gateway (and often also the DNS server IP and the DHCP Server IP as well). Use whatever it says in your browser to try to access the router. I just checked mine (a Lynksys) and after inputting that IP address it gave me access after requesting a username and password and, to my surprise and you'll be happy to hear, when I went to the wireless security section, it showed me the security key (not a bunch of dots but the actual key)!!

 

I hope this helps. I know the procedure in the prior paragraph works for me as I just did it to confirm it, but I'm not sure what equipment you have or if it will work the same or if you know the username and password to get to the router settings or how the key is set or displayed. Bear in mind that setting up the computer for a wireless connection is typicaly more than just knowing the key but also knowing the type of security being used (such as the security type and the encryption type and there can be many options for both and you don't want to be guessing). If not, then if the router is still under warranty I'd try the router's technical support for help. If not, I'd try the ISP technical support for help first as it would be free and they might be able to do it. They could also mess things up to the point where you have no choice but to contact the router's technical support group and be unable to connect until it is resolved (or try their website support options or forums for assistance (which once occurred to me but i had the bill sent to the ISP since they were responsible for messing it up and was surprised that I never heard about it again from either company).

 

Good luck!

increase speed of net - Microsoft Community

ankit singh rajput say:

increase speed of net

how to increase speed of net in windos 8

burrwalnut say:

The speed of the internet, which I presume you mean, depends on a number of things, e.g. the contract you have with your ISP and, for ADSL connections, the distance from the exchange.

 

However, marginal increases in speed can be accomplished by checking out a few things in Internet Explorer:

1. Changing two registry keys will make web pages display slightly quicker as the number of simultaneous downloads is increased. Create a system restore point before making the changes. Open a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\InternetSettings

Look for two keys in the right pane, MaxConnectionsPerServer and MaxConnectionsPer1_0Server. If they exist, then right-click on each in turn and Modify the value to decimal 10 (hex a).

If the strings don't exist in the right pane, they will need to be added manually. To add them, right-click in the ‘white space’ (an unoccupied part of the screen) in the right pane, select New > DWORD Value, type the first name as the string name, press Enter and then repeat the procedure for the second one. Now modify the values to decimal 10 (hex a).

As the registry key is a CURRENT_USER one, the change needs to be done for each user of the computer.

2. Turn off 'RSS Feeds' by going to Internet Options > Content Tab > Feeds and Web Slices > Settings and uncheck Automatically check feeds for updates.

3. Go to Internet Explorer > Tools > Manage add-ons and disable any unnecessary add-ons.
