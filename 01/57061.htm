ralf_g say:

Windows cannot change the router settings, not least because router configuration settings cannot be accessed without logging in to the router as a specific admin user and the correct password. In any case there isn't any such functionality in Windows AFAIK.

Resetting your router is done in the router, either by logging in to the configuration screens or by some sort of push button to reset to factory settings. Resetting to factory settings will erase the network access settings as well as  the current admin/user passwords to access the router configuration. If you don't know the default passwords and correct network configuration settings you could lock yourself out of the router as well as your internet connection.

Any number of things can interfere with your WiFi speed, including the number of devices concurrently logged into the WiFi connection, neighbouring WiFi routers/devices using the same channel, radio frequency noise from other electronic or electrical devices near the router or even someone nearby sneakily leeching off of your internet connection.

hairyfool say:

As has been said there is nothing ANY client (let alone Windows 10) can do to the router settings that will permanently leave it slower than before.

The router should have a reset button which will return it back to factory default settings. This has to by definition remove any configuration changes so even if what you suggest was possible (it's not) a reset should clear it. Unless supplied by the ISP it would need setting up afterwards.

The main way in which a client can affect throughput is what wireless protocol they use. If the router was forced down to b specification by one client the rest would drop down as well but this would be temporary. Your loss in speed is much more than this anyway.

Otherwise the only other possible external cause is interference which because of the level of decline is much more likely. One of the ways in which routers get round this is to automatically scan the bands and switch to the best possible although from my own experience this "feature" seems to be fairly ineffective and they stick on band 6 whatever. Otherwise the router can select transmission rates according to circumstances, your speed is synonymous with it being stuck on the lowest rate of 1-2Mb/sec. I would agree that it seems an unlikely coincidence but it could be a faulty router.
