3) Go to Device Manager.

5) Right-click the device and select Properties.

 

6) Click the "Driver" tab, click "Uninstall," click "OK" and wait a moment as the driver uninstallation process completes.

 

7) Run the driver that you downloaded from the manufacturers website. A driver setup file will have a .exe file extension that will launch an automatic installation process when opened. Simply follow the setup instructions and restart your computer when prompted to complete installation.
