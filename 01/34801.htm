user account control. - Microsoft Community

allenat1 say:

user account control.

user account control switches its self off every time i re start the pc, although its not life threatening or drastic, its becoming very annoying is there any way that i can set it so i dont have to reactivate it every time i switch on ?

a. user say:

Go to Start / Search box and type in msconfig.exe and enter and then double-click on the program icon that pappears.  Go to Tools and double=click on Enable User Account Protection (requires reboot) and click Apply and then OK. Reboot.  See if UAC is enabled when your PC starts.

If not, using Regedit (In start / search box type regedit.exe and enter and then double-click on the program icon that appears)

  1. Open Registry Editor.
  2. In Registry Editor, navigate to the following registry key:
    HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\System
  3. Locate the following value (DWORD): EnableLUA and give it a value of 1.

    Note: As always, before making changes to your registry you should always make sure you have a valid backup. In cases where you're supposed to delete or modify keys or values from the registry it is possible to first export that key or value(s) to a .REG file before performing the changes.

  4. Close Registry Editor. You need to reboot the computer for changes to apply.

Check to see if the problem is resolved.

If not, Go to Start / Control Panel / User Accounts and go to your User Account and click on Turn User Account Control on or off.  Make sure the checkbox is checked to use UAC and then click OK and back out of the problem.  Again reboot and see if the problem is resolved.

I feel fairly certain one of these procedures will work to resolve the problem.  If not, your user account profile may be corrupt.  To fix this use http://windows.microsoft.com/en-AU/windows-vista/Fix-a-corrupted-user-profile.  You need to be logged in as an administrator (and not the account you are fixing) to do this.  You also need to create or have at least 3 user accounts - I suggest two Administrator accounts (one for normal use and one for emergencies in case the other becomes corrupted) and your standard user account.

I hope this helps.

Good luck!
