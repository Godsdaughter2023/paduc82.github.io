You can't increase your video memory (VRAM).  If you have 2GB, then that's what you're stuck with.  If your computer is a desktop or tower, you might be able to swap the graphics card.  If you're using a laptop type device, I'm afraid you're out of luck and no upgrade of the graphics hardware will be possible.

Thats a OEM System Builder License. It looks genuine, but they can show you that image and send you something else.

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

What happens if I change my motherboard?

As it pertains to the OEM license this will invalidate the Windows 10 upgrade license because it will no longer have a previous base qualifying license which is required for the free upgrade. You will then have to purchase a full retail Windows 10 license. If the base qualifying license (Windows 7 or Windows 8.1) was a full retail version, then yes, you can transfer it.

**Post moved by the moderator to the appropriate category**

a. user say:

You bought a computer with the localized Chilean version of Windows. It will always be that way. You can change some of the interface elements to English by adding a language pack, but you will always have the Chilean version of Windows, and all future updates will be for the Chilean version of Windows. In addition, the English language pack will need to be reinstalled after many updates.

You can buy the localized version of Windows for your country and do a clean install. That will get you the correct version of Windows with no need for translation.

Could I get some help with upgrading to 64 bit, and if it is still possible for me to do so? I understand that with some computers, they can not upgrade to 64 bit after they have gotten 32 bit.

shenan stanley say:

There is no "UPGRADE" from 32-bit to 64-bit.

  

If you want to change from 32-bit to 64-bit and your hardware can handle 64-bit operating systems... and your Windows 10 install is activated now with a Digital License... Just grab the Windows 10 ISO, make bootable media, boot with it and install 64-bit Windows 10 (it should activate with your digital license later - you can essentially skip the entry of the product key.)
