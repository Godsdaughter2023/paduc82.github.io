jazzmutant say:

With Windows 8/8.1, Windows uses User Account Control to protect you from accidentally harming your computer. It does this by not running any of your programs as administer automatically.

There are a few ways to get around this, but since you just want to remove some files, I would recommend that you run Command Prompt as an administer and deleting the files that way.

To do this:

  1. Press WinKey + X
  2. Select Command Prompt (Admin)
  3. Navigate to the directory location of the files using:
    • "cd.." to go to the previous directory
    • "cd <directory name>" to enter a directory
    •  "dir" to view files/directories in a directory
  4. Once you are in the directory of the files you wish to delete, type "del <filename>" to remove the desired file

TIP: Be sure to include the file extension. If you are unsure of what it is, type "dir" to view the files.

Hope this works!
