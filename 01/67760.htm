Restore Factory Defaults

Login Information

  • Gateway - 192.168.1.1 or 192.168.0.1
  • Username - admin

Restoring Defaults


*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  8. When the Power Light stops flashing and glows steadily green, the Gateway has successfully reset.

vijay a. verma say:
Hi Freddie,
I am Vijay, an Independent Advisor. I am here to work with you on this problem.

Before proceeding with the troubleshooting steps, Connect your Printer to the computer and Switch it ON. In case of Wireless printer, Switch it ON and connect it to the Wifi network.

1. Run printer troubleshooter
HP provides a tool for HP printers to troubleshoot - HP Print and Scan Doctor. Download it, launch it and follow the onscreen directions to locate problems and fix them

Download Link - http://ftp.hp.com/pub/printers/hppsdr/HPPSdr.exe

2. Update Printer Driver
Download printer driver from printer manufacturer's website and reinstall.

3. Uninstall and reinstall the Printer (Credit to Andre Da Costa, MVP for providing steps)
A. Uninstall Printer
Click on Windows Key+X > Apps and Features > Scroll down and click Programs and Features > Select your printer > Select Uninstall
Type Printer in Windows Search box > Printers & Scanners > Select your printer > Remove device
Type Printer in Windows Search box > Click Printers & Scanners > Scroll down and click Print server properties under Related settings > Under Driver tab, look for your printer > Select it and click Remove button at the bottom

B. Install Printer
Go to the Printer manufacturer’s website and search for the latest Windows 10 drivers > Reinstall the Printer driver > Reboot the computer
Type Printer in Windows Start Search box > Click Printers & Scanners > On Right side, Click Add a printer or scanner > If Windows detects your printer, it will be listed > Select the printer and follow on screen directions to set it up (In case of Wifi printer, your computer should also be logged into Wifi network)
If Windows doesn't detect your printer, then you will get a blue message - Click The printer that I want isn't listed
If you are using Bluetooth / Wirelss printer > Select Add a Bluetooth, wireless or network discoverable printer > Choose the printer > Select your Printer and follow on screen directions
If you are using wired printer > Select Add a local printer or network printer with manual settings > Select Use an existing port > Select your Printer and follow on screen directions

4. Clear Spooler
Type Services in Windows Start Search Box > Click Services > Scroll down to Print Spooler > Right-click and choose Stop for the Print Spooler service > Go to C:\WINDOWS\System32\spool\PRINTERS. You can also type %windir%\System32\spool\PRINTERS into the address bar in Explorer if the C drive is not the default Windows partition > Delete all the files in this folder > Services console and right-click and choose Start for the Print Spooler service

Do let me know if you require any further help on this. Will be glad to help you.

Select the Critical battery alarm entry in the list then test it by pressing that button.  Then you can choose another sound to use from the dropdown list just to the left of the test button.  Note from the explanations below what the meaning of that Critical battery alarm is - the PC is about to Shut down, Hibernate or Sleep.

- - - - - - - - -

You can also set the battery power level notifications in your Control panel, Power options, click on Change plan settings on the link just to the right of the name of your active power plan, the click on Change advanced power settings.

Scroll down to the bottom of the list & click on the + to the left of the word battery.  When choosing the levels & onscreen notifications, think of the sequence as being

Low battery, Reserve battery & then Critical battery.

Low - you have reached a stage at which you should really consider plugging your PC into a power supply.  You can choose to Shut down, Hibernate , Sleep or Do nothing.  You can also choose to get an onscreen warning dialog.

Reserve - This is your final warning ["You have been told once, you will not be told again"].  This is only an onscreen notification, there is no action taken.

Critical - The PC is now going to take whatever action you have set up as the last ditch action [Shut down, Hibernate or Sleep] while it still has enough power to take those actions rather than just peter out dismally without closing anything down correctly.

Here are my settings - I hope they help make sense of my explanations above.

 

 

Property                    Example settings

Low battery level                    6%

Low battery notification          Off

Low battery action                    Do nothing

Reserve battery level*                    4%

Critical battery level                    3%

Critical battery action                    Hibernate

- - - - - - - - - - - - - -

* Reserve battery level specifies the percentage of battery capacity remaining that triggers the meaningless “reserve power mode”. 

When your battery charge reaches the reserve battery level, Windows just notifies you in a dialog labelled “Critical battery warning” that you’re running on reserve power but Windows does nothing at this stage. 

The dialog warns you that it is "about" to Hibernate but it will actually do whatever you have specified in Critical battery action and this could be Sleep, Hibernate or Shutdown.

Since Reserve battery level does nothing other than issue a warning, its best use is to act as the final warning notification before Critical battery action starts [which does not have any notification associated with it].

- - - - - - - - - - - - - -

The example settings shown above create this sequence of events:

At 6% battery level, absolutely nothing happens.

At 4% battery level, warning notification but absolutely nothing happens.

At 3% battery level, PC hibernates.

Port Forwarding - Microsoft Community

xbr poseidon say:

Port Forwarding

Cisco M20

Login Information

  • Gateway – 192.168.1.1
  • Username - admin
  • Password - admin
Port Forwarding
  1. Open a web browser.
  2. Log into the router by entering the default gateway into the address bar of the internet browser. (Example 192.168.1.1)
  3. The router will then prompt for a user name and password. The default is that user name is “admin” and the password is “admin”.
  4. Select Applications And Gaming from the top.
  5. Select Single Port Forwarding from the sub menu.
  6. In the following Screen enter a service name that indicates something to do with Xbox Live (XBL). 
  7. Set the service type to both tcp and udp. 
  8. Set both the starting port and ending port to 3074. Then enter the ip address that the Xbox console you wish to port forward has. 
  9. After that is complete click save settings.

 

Disable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway into the address bar of the internet browser. (Example 192.168.1.1)
  3. The router will then prompt for a user name and password. The default is that user name is “admin” and the password is “admin”.
  4. Select Applications And Gaming from the top.
  5. Select Single Port Forwarding from the sub menu.
  6. Uncheck the enabled box for any rules you wish to disable.
  7. Click Save Settings.
