Let me confirm that anything like files or folder that you have worked or saved on temporary User Profile will be lost once you come out of the Profile. So, there is no possibility of getting back these files and folders back, since they are removed permanently.

 

However, check with the following locations to see if you find missing files.


· Check the files in AppData and Temp folders (Press Windows Key + R and type %appdata% and %temp%)

· Check the files in OneDrive folders, make sure PC is connected to the Internet

· Check the files in other drive if available (D:, E:)

· Check the files in Recycle Bin


If you don't find any files, then it indicates that they are removed permanently from your PC.
