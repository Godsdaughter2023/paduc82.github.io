Please note: The resolution I have used is not useful for those that are not comfortable editing with regedit.

While I'm personally happy to go through this process others might not.

Manually disabling UAC from within Windows:

  1. Press WINDOWS KEY + R
  2. Type regedit
  3. Click OK
  4. Click Yes if prompted
  5. Go up to the top to File and then down to Export and save a backup somewhere you will remember
  6. In the left pane, browse to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System
  7. In the right pane, double-click EnableLUA
  8. Change Value Data to 0
  9. Click OK
  10. Close the Registry Editor 
  11. Restart the computer

    thrashzone say:

    Wpg has a simple Text search you can use to find Copies,

    First make sure your using the Default search,

    All photos and videos on the top left of the Navigation pane,

    The on the Home section click on the Text search in about the middle of the toolbar,

    Type copy in the box.

    It should list all of these pictures with the text on the ends of the names,

    Windows explorer does a better job because you can click on the first file and scroll down the page to the last file and hold down the Shift key while you click on it and all of the files between the clicks will be selected,

    wpg you will have to click on each file as you go very much a pain in the rear end.

    Notice what is listed on the ends of the files names and type this as an example "copy" or "(3)" using windows explorer.

    Click the start button and click on your User name listed near the top,

    Enter the above search "(3)" in the box at the top right, search user folders.
