slance310 say:

Hi,

That is actually a fairly common problem, but I'm sure that doesn't help much...

A lot of times it is caused by a registry setting and "Filters" for the CD\DVD drive...

Sometimes this happens because of one of the programs installed on your machine, like Roxio, CyberLink, and a number of others...

Check this registry setting and check for any upper or lower filters...

Just type regedit into the "RUN", "Cmd (Admin)", or "Power Shell (Admin)"...

This should be the very first entry ending with 10318... there are a bunch of them...

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4d36e965-e325-11ce-bfc1-08002be10318}

The filters will be listed on the right side after clinking on the key...

Anything listed there is a potential problem, but all filters don't cause a problem...only some...

Deleting the filter will usually fix the problem, but may cause one of your programs to not work properly...

You can usually do a Bing or Google search for the file in the filter, and identify which program it goes with...

Then try updating the program... As a last resort, delete it...

You may try to rebuild the Media Library and see if that helps.To rebuild your library

 

 

a) Shutdown windows media player.

b) Stop the media sharing service.

c) Go to Start; type Services.msc

d) Right-click on the “Windows Media Player Network Sharing Service” and select Stop.

e) Navigate to “c:\Users\”your username”\AppData\Local\Microsoft\Media Player.

f) Now delete the files named CurrentDatabase_***.wmdb and LocalMLS_*.wmdb.

g) Start the media sharing service.

h) Now start Windows Media Player and watch it build up your library again.

 

Also, restore the Media Library follow the steps given below.

 

 

a) Open Windows Media Player 12.

b) Go to Tools menu.

c) Select Advanced>>Restore Media Library.
