rey bel1 say:

Hi Marco,

This issue occurs when the following conditions are true:

  • The Keyboard item in Control Panel is not configured to wake the computer when you press a key on the keyboard.
  • Your computer may have encountered a system problem.
  • The Sleep key on your keyboard is configured incorrectly.

As an initial troubleshooting step, we suggest that you run the Power troubleshooter to identify and fix any underlying factors that may be causing the issue. Follow these steps to run the troubleshooter:

  1. Press the Start button.
  2. Type troubleshooter in the search box and then press Enter.
  3. Select Troubleshooting. Click on View all and select Power.
  4. Click Next and follow the on-screen instructions.

If the troubleshooter does not resolve the issue, we recommend that you completely reset and restore all default power plans to default settings and see if it helps. To do this:

  1. In the Search box, type CommandPrompt, right-click on the result and select Run as administrator.   
  2. Copy and paste the below command into the Command Prompt and press Enter.
    powercfg –restoredefaultschemes
  3. Once done, restart your device to apply the changes.
