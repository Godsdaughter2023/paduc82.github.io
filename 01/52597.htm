Besides the fact that you have left out a period in the URL (there should be a "." between "navy" and "mil"), the problem here is that the secure certificate that is used to communicate with this site is derived from a DOD certificate (DOD-21) that can not been traced back to a standard default "Root" certificate in Windows and thus Windows will not trust it.

The proper solution is to load (import) the Root DOD certificates into the "Trusted Root Certificate Authorities" certificate Store in your copy of Microsoft Windows.  Instructions for doing this can be found here:
  <https://knowledge.verisign.com/support/eca-support/index?page=content&id=SO5198 >

This will instruct you to go to this site to download certificates and instructs you as to how to load the certificates
  <http://dodpki.c3pki.chamb.disa.mil/rootca.html >

After loading the certificates and restarting Internet Explorer, you should be able to access this site.

deepak b say:

Hi,

 

I understand the frustration you experience when you are unable to use the touchscreen ability.

 

The issue may be because of the drivers.

 

Do you get any error messages?

 

I would suggest you to follow the steps mentioned below:

 

Method 1: Run the hardware troubleshooter

 

  1. Hover the mouse to the top right hand corner and click on search
  2. Type control panel and hit enter
  3. If the icons are not displayed in large size, click on category which is above the list and select large icons
  4. Now click on troubleshooting
  5. Click on Hardware and sound and click on Find and fix problems with devices and hardware and proceed further.

 

Method 2: Update the touchscreen drivers

 

a.       Hover the mouse to the top right hand corner and click on search

b.      Type devmgmt.msc and hit enter

c.       Now locate the touchscreen device, right click on it and click update.

 

Method 3: Install the touchscreen driver in compatibility mode

 

 a. Right click on the setup file of the driver and select Properties

b. Select Compatibility Tab

c. Place a check mark next to Run this program in Compatibility mode and select the operating system accordingly from the drop down list

d. Let the driver to install and then check the functionality.

 

You may refer the sites to get drivers and further assistance:

http://us.acer.com/ac/en/US/content/drivers

http://acer.custhelp.com/app/home/

 

Hope this helps. If the issue persists, come back and we would be glad to help you.
