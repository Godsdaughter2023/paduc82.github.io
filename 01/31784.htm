a. user say:

Try to reset OneDrive through PowerShell. OneDrive is part of Windows 10 and cannot be removed or altered, but we can reset it to clear the settings:

1. Open PowerShell as administrator (Search > PowerShell > Right click run as admin

Copy and paste these commands one by one:

1. Stop-Process -processname "OneDrive" -ErrorAction SilentlyContinue

2. Start-Process "$env:LOCALAPPDATA\Microsoft\OneDrive\OneDrive.exe" /reset -ErrorAction SilentlyContinue

3. Remove-Item -Path "HKCU:\Software\Microsoft\OneDrive" -Force -Recurse

4. Remove-Item -Path "$env:LOCALAPPDATA\Microsoft\OneDrive" -Force -Recurse

Ignore any errors at this point

5. Move-Item -Path "$env:ONEDRIVE" -Destination "$env:ONEDRIVE Backup"

Also ignore any errors here

6. try {Start-Process "$env:SYSTEMROOT\System32\OneDriveSetup.exe"} catch {Start-Process "$env:SYSTEMROOT\SysWOW64\OneDriveSetup.exe"}

After OneDrive finishes installing, It should be reset and you can check the settings again.

Issues after installing the Windows Update on your computer are possibly caused by corrupted update components. There are possibilities that the updates are interrupted during the installation. If you are still experiencing the same issue, reset Windows Update to restart the update services on your computer. Here's how:

  1. Press the Windows key on your keyboard, and then type Services.
  2. Click on Services, and then look for Windows Update.
  3. Right-click on Windows Update, and then click Stop.
  4. Go to File Explorer, and then click on This PC.
  5. Open the C:\ drive, and then go to the Windows folder.
  6. Look for the SoftwareDistribution folder, and right-click on it.
  7. Rename the folder to SoftwareDistribution.old.
  8. Go back to Services.
  9. Right-click on Windows Update, and then select Restart.
