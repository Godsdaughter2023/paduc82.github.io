weidhas say:

Xbox Live Account L&#228;nderwechsel

Einige von euch kennen das ja – einmal die “falsche” Region beim Xbox LIVE Account ausgewählt… und damit festgesessen. Gute Nachricht: diese Zeiten sind vorbei! Ab sofort könnt ihr eure Xbox LIVE Region ganz einfach selbst ändern. Wenn ihr also z.B. von vornherein nicht gleich Deutschland ausgewählt habt und das nachholen möchtet, wenn ihr ein Auslandssemester macht und zwischendurch mal weg seid, wenn ihr überhaupt wegzieht… dann einfach die Region wechseln, und schon könnt ihr die passenden Angebote nutzen. Aber Achtung: ihr könnt nur einmal alle 3 Monate zwischen Xbox LIVE-Regionen wechseln.

Folgendes wechselt natürlich ganz einfach mit:

  • Euer Gamerscore, freigeschaltete Erfolge und der Rest eures Xbox LIVE-Profils
  • Alle Microsoft Points, die sich bereits auf eurem Konto befinden
  • Die restliche Zeit eurer Gold-Mitgliedschaft


Folgendes kann nicht wechseln:

  • Einige Dienste, die ihr jetzt nutzt, sind in einer anderen Region möglicherweise nicht verfügbar. Um Informationen über Dienste - zum Beispiel Apps oder Fernsehanbieter - zu finden, die in einer bestimmten Region verfügbar sind, besucht einfach die Xbox.com-Website für diese Region. (HIER könnt ihr euch die Website samt aller Inhalte für jede beliebige Region ansehen.)
  • Xbox Music-Abonnements können nicht wechseln. Wenn ihr zurzeit ein Xbox Music-Abonnement habt, wird es beim Wechseln storniert. Restliche Abonnementzeiten können nicht zurückerstattet werden.
  • Einige Inhalte sind in der neuen Region möglicherweise wegen rechtlicher Verpflichtungen oder regionaler Einschränkungen gesperrt.


Bevor ihr die Region wechselt stellt unbedingt sicher, dass sich alle von euch gekauften Inhalte auch auf der Festplatte befinden, damit ihr sie beim Transfer in die neue Region nicht verliert, denn gekaufter Content kann nicht rückerstattet werden.

WICHTIGER HINWEIS: Ein Wechsel auf einen DE / US / AT / UK / CH Account (aus welchem Grund auch immer), bringt mit sich, dass ihr für einige Games keine Add-Ons mehr runterladen könnt, weil diese in DE entweder nur für die USK Version angeboten werden oder überhaupt nicht verfügbar sind, da das Spiel nicht angeboten wird. Ähnliches kann euch mit einem US oder UK Account passieren, wenn die Add-Ons bspw. speziell für die US oder UK Version des Spiels programmiert werden. Am sichersten und besten ist es daher, das Land zu wählen in dem man zu Hause ist und in dem man die IP Adresse erhält. Die meisten Apps filtern sowieso auf IP Adresse wegen der Verwendungsrechte der Inhalte dahinter. Erspart jede Menge Ärger und entspricht übrigens auch den Xbox LIVE Nutzungsbedingungen...

Und hier könnt ihr eure Region wechseln:
https://live.xbox.com/AccountMigration

Dieser Thread ist voerst für zwei Wochen gepinnt. Bei Fragen -> einfach raus damit.

Achja: Willkommen zu Hause ;-)

    Q: Automatic Updates may install the driver update again (even if you've changed your AU settings by now). If you wanna know how to avoid it, reply to this post.
    A: I would deff like to know how to avoid this.

    1a. Open Internet Explorer (only!) to http://support.microsoft.com/kb/910336 [ignore the title & Symptoms].

     

    1b. IMPORTANT! => Dismiss/close the Windows Update troubleshooter pop-up! - then...

     

    1c. Ignoring any "Not recommended" or similar warnings, run Fix It 50202 in DEFAULT and then AGGRESSIVE modes. [1]

     

    2. Reboot then run a manual check for updates. If the Optional driver update's still offered, hide it (again) - then install any security-related updates offered (if any).

     

    3. If you've not done so already, you can configure your system such that Automatic Updates will never offer/install driver updates again and/or change your Automatic Updates configuration such that you must Approve any updates before AU downloads or installs them. See...

     

       • Change how Windows installs or notifies you about updates
          http://windows.microsoft.com/en-us/windows7/Change-how-Windows-installs-or-notifies-you-about-updates

    PS: Any driver updates offered via Windows Update will be available via the hardware manufacturer's Download pages.     If you select this option, Windows recognizes when you are online and uses your Internet connection to search for downloads from the Windows Update Web site or from the Microsoft Update Web site.
