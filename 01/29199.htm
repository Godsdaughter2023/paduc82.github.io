chris yamazaki say:

To answer my own question:

1. Run Registry Editor (regedit.exe) and navigate to
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\FontLink\SystemLink

2. Backup the registry key from the File > Export menu.

3. In the right-side pane, double-click Segoe UI .

4. In the new window titled Edit Multi-String, replace the text Meiryo with Meiryo UI -- screenshot . Click OK when done.

5. Repeat for the Segoe Media Center , Segoe Media Center Light and Segoe Media Center Semibold fonts.

6. Restart Windows for the changes to take effect.
