Hi MG-1066,

 

By default the NUM LOCK status is the same as it was in when you properly shut down the computer. If it was on when you shut it down it will be on when you again turn the computer on.

You may also use the following steps to change the setting:  

1.      Run Registry Editor.

2.      Move to HKEY_USERS\.Default\Control Panel\Keyboard.

3.      Change the value for InitialKeyboardIndicators to 2.
