sharief k say:

Hi,

 

Please follow the methods provided below and check if it helps.

 

Method 1: Encryption

Make the file/folder only accessible to you by using encryption. To do so, follow the steps:

 

a.       Right click on the folder and go to its properties.

b.      Click on Advanced, and check Encrypt content to secure data.

c.       Apply the settings.

d.      When asked to confirm attribute changes for folders, select Apply changes to this folder, subfolders, and files.

e.       Click ok.

f.       The folder’s font color will change to Green to indicate the changes have been made.

g.       No one can use (open, rename, delete etc.) files in that folder except you.

 

 

Method 2: By disabling the access to folders.

 

Turn off Simple File Sharing by following these steps:

 

a.       Navigate to the following path: My Computer >> Tools >> Folder Options.

b.      Select the View Tab on menu.

c.       Scroll down and uncheck Use simple file sharing.

d.      Click ok.

e.       Right click on file/folder and go to properties.

f.       Select the security tab.

g.       Remove all the users from the list except yours.

h.      Click on apply and then ok.

i.        Turn on Simple File Sharing.

 

 

For more information, refer to the links:
