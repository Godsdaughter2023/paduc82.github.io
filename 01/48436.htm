bill smithers say:

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

anastasyija say:

Hi Zarebeth. I have the answer to your problem. The problem you have is the same problem I had, that is a folder name ending with a "." (a dot). That is THE problem, which also caused all your previous attempts to delete this folder to fail. Windows will not allow you to create such a file, but Mac DOES (you mentioned that your music library resided on an old Mac)! Try it out for yourself, create a folder in Windows called "P.O.D." and Windows will automatically rewrite this folder as "P.O.D" (note the missing last dot). MAC does not have a problem with this, you can call your folder "P.O.D." witout any problems (but also allows you to call this folder "P.O.D" (note the missing last dot)). The only problem is when a folder created on a Mac Device ends up on a Windows computer, which happened when you copied the musicfolder from your Mac onto your Windows 7 Laptop. I for example had the problem with the music files "C:\Users\Public\Music\M.I.A." and "C:\Users\Public\Music\T.I." after I copied files from my Ipod Touch (also a Mac Device of course) to my new Windows 7 operated laptop.

Now, I can only tell you how to delete this folder, I have no idea how to recover any data that remained in this "undeletable" folder (if any). What you can do of course, is go back to your old Mac, rewrite the "P.O.D." folder as "P.O.D" (note the missing last dot) which WILL be recognized by your Windows 7 laptop.


It is very simple, and you are going to wonder how the sollution could actually be this simple : )

GO to "command prompt" (Start > All Programs > Accessories > Command Prompt) and then in Command Prompt (the little black screen with the white letters) you type (in your case):

rd /s "\\?\C:\Users\Public\Music\P.O.D.

rd is like a delete function for folders (not for files).

It will then prompt you if you are sure "(Y/N)" and then of course you type "Y" and hit enter! And like that... begone is the folder that frustrated you SO much! Happy days have returned! : )

Hello Lowell,

Sorry to know that you are facing this issue. Let me assist you with his

1)      Do you get this error when performing a specific action or when working on a specific program?

2)      Did this issue start recently? Do you remember making any recent changes to the computer?

 

WerFault.exe process is a part of Windows error reporting system and helps sending errors logs from your computer. The error you are getting may be caused if the WerFault process is overloaded because of errors caused by another program or the Werfault process itself is corrupted due to a security threat. Let’s try these steps and check what is causing this issue.

 

Method 1: SFC scan

 

System File checker (SFC) scan is done to check if there are any corrupted system files that could be causing this issue. Refer to these steps to perform a SFC scan:

 

a)      Press Windows key + X, select Command prompt (Admin) to bring up elevated command prompt.

b)      In the command prompt type the following command: “sfc/scannow” and press enter.

c)       Now, restart the computer.

 

Method 2: Memory Diagnostic tool

 

If the above steps didn’t help, I would suggest you to run the memory diagnostic tool and check. Refer to these steps:

 

a)      Press Windows key +X, select Control panel.

b)      Change the view by option on the top right to Large icons.

c)       Click on “Administrative tools” and select “Windows memory diagnostic”.

d)      Click “Restart now and check for problems” .

 

Method 3:

 

If none of the above steps helps, you can temporarily disable the WerFault service to stop getting these errors. Refer to these steps:

 

a)      Press Windows key +R, type “services.msc”.

b)      In the services window, locate the “Windows Error Reporting” system.

c)       Right click on the service and select properties.

d)      In startup type, change from “Manual” to “Disabled” and click OK.

 

Hope this helps. Please get back to us with the results if you need any further assistance.
