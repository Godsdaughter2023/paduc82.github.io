cmdrkeene say:

You can set it so that it requires a longer drag to actually cause a drag to occur.  Normally if you click and hold an icon, and start to drag, you have to move 4 pixels before it actually starts dragging.  You could change this to a larger number so that it requires a very deliberate movement before it starts to drag.

Here's how.

Warning: Always back up the resistry before making any changes. To do this, in the registry editor, select File/Export. In the Export Range options, select All. Save the file to the Desktop. If something goes wrong, you can right click that file to restore the registry.

  1. Press the +R to bring up the Run dialog. Type regedit.exe in the Open box and click OK.
  2. Navigate to the following key.

    HKEY_CURRENT_USER\Control Panel\Desktop

  3.  In the right side pane, look for the following Values.

    DragHeight  and  DragWidth. Both of these values will have a value data of 4.

  4. Right click each one of these values and select Modify. In the Value Data box, type 20 and Click OK. 

You will need to Log Off/Log On to see the results.


Test the new setting to see if this works for you. If 20 doesn't work for you, increase the value to 30, etc, until you find the setting that your satisfied with.
