Hi,

I would suggest you to perform the following methods and check if that works.

 

Method 1: You may check and restart Windows installer, Windows update, Background intelligence services.

 

a.       Click “Start”, type “services.msc “in the Start Search bar, and press ENTER.

If prompted by User Account Control click Continue.

b.      Now locate the “Windows Installer”, “Windows Update”, “Cryptographics” and “Background intelligence”

c.       If they are not currently enabled right-click them, and select “Start”.

d.      If these services are already in start mode, please right click on that service and select “stop” and again start the service.

e.      Now restart your computer.

 

Method 2: You may rename the Windows Update temporary Software Distribution directory; follow the steps and check if the issue persists.

 

a.       Click Start and then type notepad in the Start Search box.

b.      Click Notepad from the Programs list.

c.       Copy and then paste the following commands into Notepad:

net stop wuauserv

cd %systemroot%

ren SoftwareDistribution SoftwareDistribution.old

net start wuauserv

b.      Save the Notepad file on the Desktop as rename.bat. To do this, follow these steps:

                                                                                       i.      In Notepad, click File, click Save As, and then type rename.bat

                                                                                     ii.      In the Save in list, click Desktop.

                                                                                    iii.      In the Save as type list, click All Files, and then click Save.

c.       On the Desktop, right-click the rename.bat file, and then click Run as administrator.

If you are prompted for an administrator password or for confirmation, type the password, or click Continue.

 

d.      Try to install updates again

e.      If the issue is resolved, delete the rename.bat file that you created in these steps.

Note: If the issue is resolved and you can successfully download and install updates, you can safely delete the SoftwareDistribution.old directory to recover disk space.
