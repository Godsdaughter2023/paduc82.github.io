Accessing Password Policy in Windows 10 - Microsoft Community

patrickbarrett2 say:

Accessing Password Policy in Windows 10

Was having trouble changing local password policy on Windows 10 home.  Using secpol.msc seems to be the popular answer but does not exist/isn't accessible for some reason on Windows 10 Home edition.  

To change this setting open cmd with administrator privileges (type "cmd" in search bar, right click the black cmd icon, and click "run as administrator")

As an example, I wanted my PC to stop asking me to change my password every 42 days so to change this:

In the command line type the following text NOT including the quotes:

"net accounts /MAXPWAGE:UNLIMITED"

This sets the maximum password age by days or Unlimited (never expires).

More policy commands:

net accounts /minpwlen:length â€“  This sets the minimum number of characters a password must contain. Replace the word length with the desired number of characters. Range is 0-14.

example: net accounts /minpwlen:7

net accounts /maxpwage:days â€“ This sets the maximum number of days after which the user will have to change the password. Replace days with the desired value. Range is from 1-999. If used unlimited, no limit is set. The value of maxpwage should always be greater than minpwage.

example: net accounts /maxpwage:30

net accounts /minpwage:days â€“ This sets the minimum number of days that must pass before the password can be changed. Replace days with the desired value. Range is from 1-999.

example: net accounts /minpwage:10

net accounts /uniquepw:number â€“ This sets the number of times after which a password can be used again. Replace number with the desired value. Maximum value is 24.

example: net accounts /uniquepw:8

To use a command, just enter it in the command prompt as shown and press enter.

Hope this helps
