The product key is usually located on the COA sticker attached to the bottom of your laptop or in the battery compartment. In some rare cases it can also be found in the memory compartment. For tower systems, look at the top or the side of the system unit. It looks like the following:

If the Certificate of Authenticity (COA) sticker is damaged or letters or not legible, you will have to consider the following options:

  • Use a digital camera to capture a picture of the COA then enlarge it on your computer.
  • If the COA sticker is damaged, then you would need to use the recovery partition to reinstall Windows 7, this will exempt the need to use the product key to activate your Windows 7 installation.
  • If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee. This will exempt the need to use the product key also.
  • If none of the above options work or is not available, then its best you purchase a new Windows 10 license:

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

There is nothing wrong with using it too, especially since all future releases of Windows will be full versions anyway.

BCCode: 101   0x00000101

Bug Check 0x101: CLOCK_WATCHDOG_TIMEOUT

The CLOCK_WATCHDOG_TIMEOUT bug check has a value of 0x00000101. This indicates that an expected clock interrupt on a secondary processor, in a multi-processor system, was not received within the allocated interval.

Important Information If You Have Received a STOP Code


If you have received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss. A hardware device, its driver, or related software might have caused this error. If your copy of Windows came with your computer, call the manufacturer of your computer. If you purchased Windows separately from your computer, Microsoft provides support. To find contact information for Microsoft or your computer manufacturer Contact Support.


If you have experience with computers and want to try to recover from this error, follow the steps provided in the Microsoft article Resolving STOP (Blue Screen) Errors in Windows.


The following actions might prevent an error like this from happening again:

  1. Download and install updates and device drivers for your computer from Windows Update.
  2. Scan your computer for computer viruses.
  3. Check your hard disk for errors.
Cause

The specified processor is not processing interrupts. Typically, this occurs when the processor is nonresponsive or is deadlocked.

snikefriend say:

Xbox.com wrong url causes issues

Hi! I was sent here by the Xbox Support. I posted this on the Xbox On Windows Support but they also guided me to post this here too. There is a problem with the Xbox website that causes some issues while using it. I would like to clarify, that this is only on Xbox.com. My Xbox 360 works fine! So, when I head to my achievements on Xbox.com, (url is https://account.xbox.com/el-GR/Achievements?xr=socialtwistnav , requires sign in) EVERY Xbox 360 game comes with no game icon. If you have a gamertag and at least one Xbox 360 game, you can see by yourself by pressing the above link. Now, what I've found out, is that this happens because the url that Xbox.com is giving toevery Xbox 360 game is wrong. I will continue by using an example. Let's say I want to check my "The Walking Dead" game achievements. I click on the above link, and choose The Walking Dead. The achievement page will open normally, except that the game will have no icon but instead, the generic Xbox gamepad icon. So, when you hover your mouse over the icon (or head to line 2332 of the html code of the specific game page) you'll see that this icon leads to http://marketplace.xbox.com/el-GR/Product/The-Walking-Dead/00000000-0000-0000-0000-000000000000 which is a false link. It redirects itself on http://marketplace.xbox.com/el-GR/ . Instead, the correct link for The Walking Dead is this http://marketplace.xbox.com/el-GR/Product/The-Walking-Dead/66acd000-77fe-1000-9115-d802584111de . With a little research, I discovered that every Xbox 360 url on achievement's page has exactly the same error, the 000000 string is attached to it instead of the real string like the one in the second url. This causes some (I could say serious for achievement hunters like me) problems:

  • I can't compare my Xbox 360 games to anyone. If I try to compare an Xbox 360 game, it will report an "Uh oh! This shouldn't have happened. It appears like you found a glitch in the system. Try again later." error.
  • All Xbox 360 games on the Achievements page appear with no icon, which makes it super hard for me to find a game that I want among many.
  • The Achievement page of the game, does not lead me to the game's marketplace page as it should lead me when I click on it, but it leads to http://marketplace.xbox.com/el-GR/ which is annoying because if I want to check a DLC from a game that I have incomplete, I have to research the Xbox Website to find it.

Can you please fix it? We take great use of Xbox.com when it comes to stuff like that! Thanks!
