"wddm not running". - Microsoft Community

beverlystewart say:

"wddm not running".

how do i repair the message "wddm not running".  I am using window 10

patricia yu say:

Hi Beverly,

The error message is related to PCs using WD portable drives. We suggest that you restart the WDDM service and check if it helps.

Here's how:

  1. Open Run by pressing Win + R, and then enter services.msc. You can also click on Start, Search, or Cortana, and then look for Services Desktop app.
  2. Locate Windows Display Driver Model Service.
  3. Right-click on the service, and then select Restart.

In addition, make sure that the service is set to start automatically.

  1. Right-click on the service, and then select Properties.
  2. Change the Startup type to Automatic.

If the issue persists, you may need to contact the drive's manufacturer.

Let us know how it turns out.

a. user say:

For anyone having the same problem, I finally figured it out:

Instead of English - International being an actual language option, it is an option under English (United States). 

Settings > Time & Language > Region & language

Click on English (United States)

Select "Options"

Select "Add a keyboard"
Select "United States-International QWERTY"

My preference is to removed the other keyboard options.
