How do I control which photo Movie Maker uses as a cover? - Microsoft Community

hdthoreau say:

How do I control which photo Movie Maker uses as a cover?

Windows Live Movie Maker tends to select the second photo in a series as its cover when the movie is saved. One in ten times it chooses the first image. How can I control this? I'd simply like the first photo as the cover.

considerate_guy say:

This isn't WMM that is doing the choosing... it is Windows.  When you have a video clip, Windows needs to pick a thumbnail, but the algorithm tries to be smart and avoids blindly picking the first frame.  Because a lot of clips start with a static title or perhaps fading from black, the first frame is rarely representative of the content.  So the routine tries to look later in the clip to find the thumbnail.

I do not know how the default thumbnail generator for .wmv files in Windows 7 chooses which video frame to use, but I suspect that it tries to find the first non-static frame at the beginning of the video. Also, because it's expensive to decode multiple video frames, it may skip frames in order to speed up the thumbnail generation.  You might play around with extending the duration of the first picture and see if that make the algorithm pick that picture or not... but in general, you aren't going to be able to control this.
