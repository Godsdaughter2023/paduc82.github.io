Hi BonsaiBill,

The event ID 11 is related to the driver detected a controller error on \Device\Harddisk.

Event ID 10010 occurs because the Windows Explorer operating system shell is not running on the terminal server. When a Component Object Model (COM) object tries to register, it posts a registration message to the Windows Explorer shell. Because the Windows Explorer shell is not running, the registration message times out. Subsequently, the COM component logs the event ID error message on the terminal server.
