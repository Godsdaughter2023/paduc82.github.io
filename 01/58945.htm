The command you ran does not generate any output. Why? Because it fails! Why does it fail? Because the target location of the output file is protected under the current versions of Windows!

Here is how you can run the command and find the output on your desktop:

  1. Mark the line below with your mouse, then press Ctrl+C to copy it.
  2. Press the Windows key + R.
  3. Press Ctrl+V to paste the code line, then press Enter.

cmd  /c  tasklist /FO CSV> "%UserProfile%\Desktop\Tasklist.csv"

cmd  /c  tasklist /FO table> "%UserProfile%\Desktop\Tasklist.txt"

Windows 8 64 Bit - Microsoft Community

harikishanmandalapu say:

Windows 8 64 Bit

I tried my laptop running 32bit windows XP and also using upgrade assistant , but I ended up buying the wrong bit of windows 8, is there any way I can attempt to download Windows 8 64bit again and use the same product key so I can get my Pc off windows 8 32bit to windows 8 64bit?

andre da costa say:

You can use the same product key with 64 bit Windows 8.

 

When performing the upgrade process the upgrade assistant determines your existing architecture, downloads the same architecture and installs.

To change from 32bit Windows to 64bit Windows a clean install will be performed (will have to reinstall applications etc…) and it will be necessary to purchase Windows 8 as a DVD.

Below is a web page that has additional information regarding Windows 8

Upgrade to Windows 8

You can go to a computer running a 64 bit version (Vista or Windows 7) and do the download there and create a bootable copy.

32 vs 64 bit

http://windows.microsoft.com/en-US/windows7/32-bit-and-64-bit-Windows-frequently-asked-questions

Windows 8 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 8 32 bit can utilize up to 3.2 GBs of RAM. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task, but you are able to do so much more, you can have more applications open, do things like run an Antivirus scan in the background without it affecting your system performance. Windows 8 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 8.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 8 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers.

If you need to upgrade to the 64 bit version, you will need to do a custom over your 32 bit installed. You said you ordered the backup DVD with your order, so you will have the 64 bit disc.

You will have to do a clean installation, there is no in place upgrade path from Windows 7 32 Bit to 64 bit. Lauch the Windows Easy Transfer utility in Windows 7 and create a WET file (Windows Easy Transfer file), if you have an external hard disk, you can have it stored there so you can access it later when you do a clean install of Windows 8 64 bit disk.

http://www.notebooks.com/2009/10/07/using-windows-easy-transfer-in-windows-7/

Once you have done that, simply do a clean install of Windows 8 64 bit disk. In Windows 8 64 bit disk, launch Windows Easy Transfer, follow the on screen wizard, point to where you stored the the WET file (external hard disk), and it will restore your personal files and settings. Reinstall any necessary applications.

HDMI Not working after Windows 10 upgrade - Microsoft Community

pe ka say:

HDMI Not working after Windows 10 upgrade

After upgrade to Windows 10, I have connected HDMI cable like I used to for Windows 8.1 but nothing happens! I can't see HDMI under the Control Panel>sound.  
avgjoe7 say:

The short answer is that it works fine now using a different (and presumably better) HDMI cable. The cable that works isn't particularly newer (it is at least a few years old) or different in any obvious way. Moreover, the cable that doesn't work with Win10 works fine for everything else.  So maybe there's something about the Win10 implementation that requires an HDMI cable that more closely meets the HDMI cable specifications whereas the Win7 implementation was more forgiving.  (I'm just speculating about that.)

I don't want to suggest that the HDMI cable was the only issue.  In fact, I suspect that there were multiple problems: Win10 anniversary update, audio and video drivers, and the HDMI cable.  But after getting all the latest updates and patches it still wasn't working with the first HDMI cable.  The reason I believed that the HDMI cable was OK was that it was worked fine when connecting a different PC (running Win7). Apparently, there's more to HDMI cables than meets the eye.  Just because an HDMI cable works with Win7 doesn't mean that it will work with Win10 with the same hardware.  I find this very surprising, but it is the case.  So my suggestion to everyone who is still struggling with this problem is to first get all the latest updates and drivers and then get a new high quality HDMI cable.  That may solve your problem.

Good luck!
