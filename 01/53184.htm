Have trouble signing into the Xbox app? - Microsoft Community

rooks [xbox] say:

Have trouble signing into the Xbox app?

We have a few things to try, if you're having trouble signing into the Xbox app:

1. Try your Microsoft Account in other apps

Launch the Store, People, or OneDrive apps to see if your Microsoft Account (MSA) is working properly in those apps. If they aren't, then there's an issue with your account that you'll need to fix (see other steps).

2. Log out of your PC account, then log back in

In the Start menu, select your profile and click on "Sign Out". This will log out of your Microsoft Account. Then log back in and see if that helps. Sometimes the Microsoft Account can get into a weird state, and logging out/in (not the same as rebooting your PC) will help.

3. Check your Microsoft Account in Settings

Launch the Settings app, go to Accounts, and make sure there isn't a "Verify" button under your account. Sometimes your account can get into a state where you need to re-verify it. Then go back to the Xbox app and try again.

4. Update your OS and Xbox app

Make sure you have the latest Xbox app from the store.  We’ve done a bunch of updates that should help with some upgrade issues we’ve noticed.  The app should automatically update, but sometimes it doesn’t. Additionally, if you were a part of the Windows Insider program, make sure your OS is fully up to date.

5. Remove/re-add your Microsoft Account

Try removing the connected Microsoft Account (MSA) you have and then re-add it. If you are signed into Windows with an MSA, you will need to convert your account to a local account, remove the MSA, then re-add it.

6. Clear your local Xbox identity data

This step has worked for a few people:

- Open File Explorer

- Copy/paste this directory location: %userprofile%\AppData\Local\Packages\Microsoft.XboxIdentityProvider_cw5n1h2txyewy\AC\TokenBroker

- Delete the Accounts and Cache folder in there (if they exist)

- Try launching the Xbox app again

7. Make sure Xbox Live services are running

Open the start menu, search for “services” and run it. (The app has a gear logo)
Look for “Xbox Live Auth Manager,” “Xbox Live Game Save,” and “Xbox Live Networking Service.”
If you notice one with a “Startup Type” set to “Disabled” you’ll need to change it to “Automatic”
1. Right-click on the item and select “Properties.” (You’ll see a popup window with more information)
2. Change “Startup type” to “Automatic”
3. Hit “OK”

After checking them all, try restarting the Xbox App.

8. If you are using a VPN or custom proxy setup, try disabling it

We've found that some VPN software interferes with the Xbox Live authentication process and causes sign-in to fail. While we look into why this is happening, try disabling it and see if it helps you sign in.

There has been a version update of a program that I had pinned to the Start Menu. Now it has (2) appended to the program name. I right click the pinned shortcut, choose prperties, then the General tab. When I try to rename the shortcut by deleting the (2), I get an error asking if I want to rename the "ProgramName (2).lnk" to "ProgramName (3).lnk"?  There is already a file with the same name in this location.

I finally used Search and found ProgramName.lnk and ProgramName (2).lnk in the C:\Users\UserName\AppData\Roming\Microsoft\Internet Explorer\Quicklaunch\User Pinned\StartMenu. (Internet Explorer??)

Now, the ProgramName link does not show on the Start Menu.

Ahh ha!! It looks like the ProgramName link points to the Program Files (x86) directory. The proram.exe is no longer in that directory. The ProgramName (2) is pointing to the Program Files directory and the exe is within that directory.

Sure enough, deleting the pinned link point to the non-existent 32 bit directory allowed me to rename the pinned link as desired. Without using Search, I never would have found the "missing link".

So...I solved my own problem, but post it so that others can learn.

Any suggestion as to how the "Disable internal pointing device when external USB pointing device is detected" setting can be maintained under Windows 10?  Thanks...

 

a. user say:

Hi,

Thank you for posting you query on Microsoft Community.

I would suggest you to follow the steps below to disable internal pointing device when external USB pointing device is attached. 

1. Make sure you back up the registry by click on the following link and then follow the next steps.

How to back up and restore the registry in Windows: http://support.microsoft.com/kb/322756

(Steps for Windows 8.1 are also applicable for Windows 10) 

2. Once you have created a backup of the registry, follow the steps. Press Windows key + R to open run command and type “regedit” and click Enter.

3. Navigate to the following path:

HKEY_LOCAL_MACHINE\SOFTWARE\Synaptics\SynTPEnh

4. Right click and select New > DWORD (32-bit) Value and name it "DisableIntPDFeature".

Modify its value to 33 in hexadecimal, or 51 in decimal to enable it to be default on.

5. Exit registry and restart windows for changes to come into effect.

Registry disclaimer

To do so: Important this section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. Kindly make sure you have created a backup of the registry before making any changes. Click on the link below: http://support.microsoft.com/kb/322756/
