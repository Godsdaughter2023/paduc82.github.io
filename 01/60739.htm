justin - say:

This command  will clear out your spool when you cancel a print file. So everytime you need to clear the Files use this command. Please let us know if this works out for you.

Create a Batch File:

 Click Start -> click All Programs -> click Accessories  -> click Notepad.

Copy the following commands, and then paste them into the opened Notepad window:

 

net stop spooler

CD \Windows\system32\spool\PRINTERS

Del *.spl

Del *.shd

net start spooler

Exit

 

On the File menu, click Save As , and then type ClearSpooler.cmd .

Click Save as type , and then select All Files (*.*) .

Save the ClearSpooler.cmd file to your desktop.

Right-click the ClearSpooler.cmd file, and then click Run as administrator. If you are prompted for an administrator password or for confirmation, type the password, or click Continue .
