Everyone does Not mean Everyone that wants to logon.

If accounts are set only for John, Jill, and Jack, using the Everyone setting in the Folder Permissions saves the effort of setting Permission separately for John, Jill, and Jack.

Everyone else is Not in the Everyone group, only John, Jill, and Jack.

moco man say:

What OS are you running?  Ratings on small 128GB memory and huge 1TB raid configured drives in line with 5.9 ratings.

Running Win 7 OS?  Believe 5.9 WEI for Primary Hard Drive Transfer Rate is product of OS.   Went from 7.9 with Vista HP OS on WD Raptor 300GB 10K RPM Sata2 drive with 3.0GB transfer rate to 5.9 using WD 600GB 10K RPM Sate3 drive with 6.0GB transfer rate using Sate3 mobo, Sate3 drivers, and Sate3 cable after change to Win 7 OS.  Forget the HD WEI; all other WEI ratings may be improved by upgrading your hardward.


The Windows 7 WEI Scores are only operating system dependant because the WEI for windows 7 is only performed using windows 7.  The results, however, are very real and system hardware and driver dependant.  If you score a 5.9 on your HDD then you have a second generation rotating disk running at 7200 RPM and it is healthy.  It is primarly determined installed hardware plus some basic read/write checks.  The only configuration of HDD that I have seen beet the 5.9 is two WD VelociRaptors running the operating system in Raid 0.  This only achieved a 6.1 rating.  I achieve a 7.6 by running two Intel SSD's in Raid 0.  Again this is limited from the 7.9 because they are running on SATA II channels which restrict the I/O of these heigh performing drives.  I expect to achieve the 7.9 only when I upgrade to the Crucial SATA III 6 GB/s SSDs next week.  These ratings in Windows 7 are really not as arbitrary as they are being made out to be.  The fact is that these scores are only benchmarks and do not neccesarily affect your personal computing experience.

7.7 = I7 930 @ 4 GHz

7.8 = 6GB 3 channel 1600 DDR3

7.9 = GTX 470 with GTS 250 Physx

7.9 = ditto ussing HDMI monitor

7.6 = Intel 80 GB SSD x2 in Raid 0
