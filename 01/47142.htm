Hello,

Suggest you to run System File Checker scan:


1. Search for 'command prompt' using Cortana or Windows Search.


2. From results, right click on Command Prompt and select Run as administrator.


3. Type sfc /scannow command and press Enter key.

Repair-WindowsImage -Online -RestoreHealth

4. Let the command complete, restart your system then.

4. Type OneDrive in the search box, and then click on the OneDrive desktop app. Re-setup OneDrive.

A USB cable has four wires.  Two for power and two for data.

If one of the data wires is broken then the device will charge but not connect to the PC.  There are USB cables which deliberately only have the power connections to allow charging with no data connection.
