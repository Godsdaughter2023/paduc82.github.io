bossdweeb say:

I reckon this community titled - Gaming on Windows 10 - is suitable for anything gaming related you wish to ask.

I'll start with the short answer. Major components only....    ;)

" CPU Cooler - I've got no idea how important this is and don't know if it's worth choosing something better

than the default choice.  CPU Thermal Paste - I have no idea what this does."

- If you are buying pre-built it won't much matter, but you should do some more homework

anyway and find out what they do and just how critically important both are.

This goes for the keyboard, cable managment, and the rest as well.

Read reviews and specs and weigh up the results with your needs. You have the terminology in your post.

Go forth and search....  :)

- Something to get you started...

That cooler should be adequate for a non-overclocked machine, but not if OC'd to 4.4GHz.

I don't OC but do tend to use 'k' series cpu's in the possibly mistaken belief that if Intel reckon this cpu

is good enough for stable OC'ing to unlock the clocks ( 'k' ) that it may be more longer lasting cpu

when not overclocked.

Regardless of how stable you get an OC to run it will shorten the life of your cpu to some extent.

This may not be an issue for OC enthusiast ( it is really an enthusiast thing. I don't reckon games benefit that much)

because they usually sell the machine in the first year or two.

To learn how to OC you will want to visit an OC'ing site.

Warranty - How likely is it to break in the first 2 years, and is it worth getting a warranty for 3 years?

- Crystal ball required. Luck of the draw. I reckon most failures happen within the first year. Then things start to

hot up again after five (gpu, HDD, motherboard).

The components would all (mostly?) have an RMA period of three years, which would require you to post

the item to the mfg of the component part for repair rather than return to base. Even with the extended warranty

you may only save postage as they would likely also do an RMA if the failed component is over a month old.

When doing an RMA many mfg's swap in another persons returned component for quick turn around though.

"Case - This isn't really important, so I should be abe to just get what's cheapest?

Power Supply - I'm not sure how many Watts should be used."

Case and power supply. The two most disrespected components. You don't need a top of the line case,

but for gaming you need good airflow.  This case is a bit short fore and aft, which can make fitting a long

graphics card it bit tricky because while the upper section of the HDD cage can be removed, there is still

a vertical bar in the way. It should be a good case, but a full tower is a bit easier to work in.

More volume can help with temps as well.

- Power supply. Avoid bottom end on a gaming machine.  The psu is the heart of the system.

Nvidia recommend 500W minimum with a GTX970. Because a machine can be under full load for

hours at a time while gaming it's desirable to keep the load on the psu at about 80%  max. This helps

keep the psu cooler and helps the psu produce a stable current.

FSP is a good brand but the 500W bronze model wouldn't be my choice.

Partly due to having two rails @18Amps ( 432W).

I would trust the FSP 500W Gold with a 970 if they had it on offer because of the little extra power 39Amps (468W)

on a single rail, and better components. However, if I had to buy from this source close to budget I would

go with with the  EVGA 650W, but the 750W, even though you don't need 750W, would be a safer option

than the 500W Bronze in the long term as well.

Something along the lines of 600w Bronze with 40+ Amps on a single +12V rail would be the best value

for money sweet spot, I reckon.

- If you think you may go for dual gpu's in future you need to add another 200+W to the psu total.

- GTX970. Great graphics processor, but the previous series. Not necessarily a problem if you don't ever want

to sell the card, but you've already lost resale value over the newer tech. That there is newer tech

in the GTX1070 is also an obvious benefit. I haven't read reviews for the 1070 yet but if the pattern of past

new models  remains, the 1070 would be equivalent to, or better in some games, than a GTX980.

- People do swear by the 'reference' coolers in the model you have chosen, but I think it must be about the 

clean lines, because these reference coolers tend to howl when the fan speeds up.

I've been using graphics cards with 'aftermarket' cooler for years. As with the aftermarket cpu coolers, when not

overclocking it's mostly about being more quiet, but it's also because of the factory overclocks that these cards

come with, which with gpu's I do like.  I would go with the ASUS Strix (first choice would be Gigabyte  though ).

- Note I did read several complaints at the Nvidia forums recently about loud fans on the ASUS Strix, but

I believe it must have been a bad batch. These cards should run relatively quiet.

- RAM. A lot of people are tricked to beveling the 'heat spreaders' actually do something other than look pretty...

They don't. Two naked 8GB Crucial sticks should do fine.

- Motherboard. Basic boards are as reliable and stable as the high end one's. You pay for the extra features

that come with more expensive boards, so no need paying for what you won't use.

I could be tempted by the next one up Z-170-A because of the 8 Phase power and if you think you may

ever want to run Nvidia SLI graphics... SLI . The 170-P can only do Crossfire (AMD dual graphics).

The audio may also be just a little bit better than the 170-P ?

SSD = Very fast compared to HDD. This is the drive you install Windows and your software to.

HDD = Still fast enough for installing games on as it only affects loading times by a few seconds.

The reason Microsoft Office files move is because, when you edit them, Word or Excel create temporary copies of the files.  When you save, the original is deleted and the new file is saved.  This is a foundation of the auto-save ability and error/crash recovery.  The side effect is that your desktop sees the original file get deleted, then the new file is created and goes to the first available spot like any new file would.

filipe_assis say:

Well, the problem is fixed.

After trying anything I could inside the system, I had to try the router once more. The problem was a simple fix, yet odd, since the connection was working before.

The router (archer C20) has 13 channels, and I left mine in the 13th, apparently the wifi board couldn't efficient connect to this channel, so it didn't show in the connections menu. So I lowered it to channel 1.

You can only share gold on one console at a time. The account with gold would set one console as their Home console. Any other profiles would be able to sign into that console and use the gold. The account with gold would be able to sign in and use their gold on the other console.

If you're using multiple accounts on both consoles, you'd need at least 1 more subscription.
