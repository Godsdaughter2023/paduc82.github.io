It depends on how Windows 10 was originally configured on your computer.

If its a full retail (NOT OEM SYSTEM BUILDER) store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

Aha.  I have figured out a diagnostic for this.  I did it in PowerShell but the same query can be issued in a cmd window (without the PowerShell command line escape)

wmic --% Process where Name="TabTip.exe" Get ReadOperationCount, ReadTransferCount  /format:list

FWIW, here are my observations from using this.   When I just issue it twice there is no change in the chosen statistics.  If the OSK is running I can even click on the TK button in the Taskbar and still see no change.  However, provided the OSK is not running, clicking on the TK button in the Taskbar does cause a change in those statistics;  it is not necessary to actually use the TK to see such a change.  Please check to see what you get if you try to reproduce these observations.   Note that you will be trying to issue the above command twice for each test, once before the thing you want to test and once after to see whether the test caused any change.
