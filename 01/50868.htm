This fixed the issue for me. Hope it helps

1. Exit your Skype for business completely.

2. Browse to: %localappdata%\Microsoft\Office\16.0\Lync      
3. Delete the folders which start with sip_.
4. Delete all the sub-fodders under the folder Tracing, but don’t delete Tracing itself.
5. Re-open Skype for business.
