allan mej say:

Hi Richard,

Some bugs or disk drive errors may cause high disk utilization. If your task manager shows 100% disk utilization, you may need to optimize Windows by following these troubleshooting methods:

Disable Windows Search

Some bug in Windows 8, 8.1 and 10 causes some "search loop" that causes maximum utilization of your disk. To disable Windows Search, kindly follow these steps:

  1. Press Windows + R to open Run.
  2. Type services.msc and press Enter.
  3. Scroll down to find Windows Search, right-click it, and select Properties.
  4. Change the Startup type to Disabled and save the changes.

Disable Superfetch Service

Superfetch service has been identified as a possible cause of 100% disk utilization on Windows 10. To disable Superfetch service, kindly follow these steps:

  1. Press Windows + X and select Command Prompt (Admin).
  2. Type net.exe stop superfetch and press Enter.
  3. Wait for a minute for the changes to apply and close the Command Prompt.

Let us know how things goes.
