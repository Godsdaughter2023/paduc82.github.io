steve winograd say:

You're using a crossover cable to connect two computers directly, without a hub, switch, or router.  Recent computers have improved circuitry in their network adapters that can connect directly to another computer using a regular network cable.  Your Windows 7 computer almost certainly does.

To fix the Your network is not complete message on Win98, install Client for Microsoft Networks . 

The Local Area Connection doesn't have a valid IP configuration message on Win7 means that you haven't assigned an IP address to the Local Area Connection.  You can ignore that message if the Win98 computer is actually running Windows 98 Second Edition.  If it is, both computers will assign themselves compatible IP addresses in the 169.254.x.x range using Automatic Private IP Addressing .  If the Win98 computer has the original version of Windows 98, or if you want to avoid the message, manually assign private IP addresses to the two computers, e.g. 192.168.137.1 and 192.168.137.2, subnet mask 255.255.255.0.

Try this to access the Win98 computer from Win7:

   1. Press WindowsKey+R .

   2. Type the Win98 computer's name in the box in this format:

          \\computer

   3. Click OK .

That should open a window showing Win98's shared folders, from which you can copy the desired files.

It isn't possible for a Win98 (or 95 or Me) computer to access shared folders belonging to a Win7 (or Vista) computer.  Microsoft changed the network authentication method in the later operating systems, and the earlier systems can't use the new authentication method.
