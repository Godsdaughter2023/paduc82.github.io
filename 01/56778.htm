Windows File backup stores its files in a number of .zip files. They might be hidden. Since your husband is an IT professional he should easily be able to find them on the backup disk.

If you have a .vhd file (Virtual Hard Disk) then it will contain a complete image of your System Drive. Your husband can mount this image as a virtual hard disk, using the Disk Manager, then copy any files or folders back to your library folders.
