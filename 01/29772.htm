jess bor say:

Hi MSKBoerdery,

An outdated driver may be a possible reason why you are unable to use your scanner. You may follow the steps suggested below in order to help fix the issue:

  1. Select the Start  button, type Device Manager, and select it from the list of results.

  2. Expand the Printers category to find the name of your device, then right-click (or tap and hold) it, and select Update Driver.

  3. Select Search automatically for updated driver software.

  4. If Windows doesn't find a new driver, you can try looking for the updated one from the device manufacturer's website and follow their instructions.

* If the steps above don't work, try uninstalling/reinstalling the driver by following these:

  1. Right-click the Windows key and click Device Manager.
  2. In the Device Manager, expand the Sound, video and games controllers, right click on any 3rd party device, and click Uninstall.
  3. In the Confirm Device Uninstall window, make sure to check the Delete the driver software for this device option, then click OK.

          

4. Restart your computer.
