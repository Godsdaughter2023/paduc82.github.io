    slance310 say:

    Hi,


    This is what I had to do for a Lexmark X5340... otherwise reinstalling it made no difference...

    There seems to be something going on with the Print Spooler. If you don't turn it off before uninstalling, the reinstallation doesn't help...


    One thing you might check first though is to make sure the Print Spooler in "SERVICES" is started, and Bidirectional Support is Enabled...  If this doesn't help, you will probably need to uninstall\reinstall.


    To find Bidirectional Support, right click on the printer in Devices and Printers, select Printer Properties, PORTS tab, and it will be near the bottom...



    1) Printer must be On for it to show up in Devices and Printers after the Upgrade - Just this one  time.  Otherwise, only the Lexmark Fax shows up and the Printer can't be REMOVED.


    2) UNHOOK the Printer from the computer.


    3) REMOVE Lexmark Printer and Lexmark Fax (if you have one) in Devices and Printers.


    4) STOP the PRINT SPOOLER.


    5) Uninstall software - TWO Firewall Security Messages Pop Up about "ALLOWING' Printer Communication..


    Select "ALLOW".... if you Allow both Shared and Private on both windows, you end up with two printers in Printer Preferences, Properties, etc. I unchecked a "Shared" on the first window.


    6) After REBOOT, STOP Print Spooler AGAIN.


    7) Reinstall Software\Drivers ... Hook Printer back up when Prompted.


    8) Unhook Printer again and REBOOT ... This step isn't necessary if Bidirectional Support is already ENABLED.


    Skip to step 10 if Enabled.


    9) Enable Bidirectional Support and Reboot


    10) Set up Printer Preferences if desired, and it's ready to go.
