The "Apply to folders" settings apply the current view setting to all folders that are optimized for the same content as the folder you have open. This setting will not apply to the folders which is not the same type.

 

I would suggest you to click on restore defaults in the folder options and try to change the folder views and click on apply to folders.

 

Restart the computer once to test the issue.

If the issue persists, you may create a new user profile and see if the issue persists there.
