bossdweeb say:

It may be a permissions issue.
c0000005 = Access denied.

StackHash errors similarly tend to indicate there is a permissions issue.

- It could be your Anti-virus or Firewall is blocking the game. Try disabling temporarily to see if that helps.

If so you need to make an exception in the AV. This may also be why you can't add the game to DEP.

- There was a Win7 Update  - KB 2670838 - back in '13 that caused StackHash errors, but I believe this has

been fixed with a newer update since then.

- One of the best ways to avoid permissions issues, particularly with games that can be modded is to

not install in the default location - Program Files (X86).

If you don't have a second HDD or another partition to install games to making a new folder in C:\ and

install games there may help.

- Failing the above try running in Clean Boot to troubleshoot potential Services or Starup program conflicts.
