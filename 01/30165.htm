No. The change from MBR to GPT is in the partition table records and affects the whole drive.

If Windows 10 was running on this system beforehand either the disk was GPT  and something has happened to the boot sectors or something has happened to the BIOS. In a few cases the BIOS can be set to work in legacy BIOS mode or UEFI mode. Check your system documentation.

If you upgraded to Windows 10 from 7 it may be that the upgrade worked as it was but a new install won't.
