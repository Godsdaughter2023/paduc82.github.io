a. user say:

KB890830 is the Microsoft Malicious Software Removal Tool.

When you let Windows Update install it, it launches and runs right now and you can see it running in Task Manger while you are waiting:

When it eventually finishes it puts a log file called mrt.log in this folder:

%WINDIR%\debug

If things are normal the mrt.log file looks like this:

The MSRT will not show up as an installed update in the list of installed Windows Updates or in the list of Programs and Features because it is not supposed to show up in those places so that is normal.  It will show up in the Update History.

Does it show Successful in your Windows Update history?

The MSRT will be offered by Windows Update many times in the lifetime of your Windows installation because it might get updates periodically so it will be offered again and again.  You can choose to install/run it or Hide it but it will just show up again later - and it will keep showing up.

If you think your MSRT is malfuntioning you should check the mrt.log file for clues.

I usually just hide it because it is considered weak and since it runs when Windows Update is running it can make you think that Windows Update is hung or broken when in fact the MSRT is running until it completes.
