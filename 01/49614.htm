How do I merge adjacent partitions on my hard drive?
shishira d say:

Hi Bob Ryder,

 

You can merge adjacent partitions on your hard drive provided that the other partition is an unallocated empty free space in Disk Management. To make it an unallocated space, you can use the shrink feature available in Windows 7.

http://windows.microsoft.com/en-US/windows7/Can-I-repartition-my-hard-disk Can I repartition my hard disk?

 

Right click on the partition which isn’t shrinked, you will find an option to Extend a partition, using which you can extend the partition.

Note: You can only extend a partition towards the right, which means the unallocated partition should be towards the right of the extending partition.

 

If you opt to merge partitions without using shrink method, you must use third party software program available online. Use your favorite search engine to find a compatible software program to do this task in Windows 7.
