xbf brad a say:

Hello Alex,

Thanks for bringing the question to the forums. With home console sharing basically the profile with gold one of the kids would need to play on.  You will make it home (as you have it set now) on the first console. On the second console you will download the same profile to the second. While the kids can play on the first one fine with any profile, on the new one they wont be able to. If your kids have separate accounts besides the one that has the gold, you could buy the gold on one of their accounts then set up home console with that profile.  For example we will call your profile A kids are B and C. Currently live is on A. They will need to play on A on the new Xbox to utilize gold on both consoles. If you purchase live for B (cancel A) then set B on the first console you owned as home and download on new console also. Profile B and C will be able to play on separate consoles and use the gold. C will have to play on the first console and B on the new console.

Hope this helps.

For File and printer sharing follow the steps given below and check if that helps.

 

Check if Network Discovery, File and Printer Sharing and Public Folder sharing is turned on in Windows 7 Network and Sharing Center.

To open Network and Sharing Center

1.      Click on the Start button

2.      Go to Control Panel.

3.      In the search box, type network, click Network and Sharing Center

4.      In the navigation pane, click Change advanced sharing settings.

5.      Click the chevron to expand the Home or Work network profile.

6.      Select the options to turn on network discovery and file sharing.

7.      Under Public folder sharing, do one of the following:

8.      To share your Public folders so that people on other computers on the network can open files in them but can't create or change files, click Turn on sharing so anyone with network access can open files.

9.      To share your Public folders so that people on other computers on the network can open files in them and also create or change files, click Turn on sharing so anyone with network access can open, change, and create files.

10.  Click Save changes.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

Also check

1.      Network Discovery should be ON (So it can see the other computers)

2.      Network set to Private (Public is for hotspots, airports, etc)

3.      File Sharing should be ON.

4.      Public Folder Sharing must be ON

5.      Password Protected: OFF (unless you want to set up identical usernames and passwords on ALL   computers in your Network) If you have it ON, and no identical usernames/passwords set up, you will be asked for a username and password when you try to access a Windows 7 computer from an Vista computer, or a Vista computer.
