mark ole say:

Hi William,

It is possible that the apps detect the default region you have set. Try changing the region in Settings > Time & language > Region & language and check.

You also try to disable the automatic keyboard layout option. To do that, follow these steps: 

  1. Press Windows key + X on your keyboard.
  2. Choose Control Panel.
  3. Under View by, select Small icon.
  4. Click Language.
  5. On the left side, choose Advance settings.
  6. Under Switching input methods, uncheck Let me set a different input method for each app window.
  7. Click Save.
  8. Restart your computer.

Note:  Some shortcut keys may cause some unwanted changes on the keyboard layout. Please refer to these shortcut keys:

Windows Key + Space key = Changes language/keyboard pairs.
Alt + Shift = Change Language.
Ctrl + Shift = Changes keyboard.

Let us know if you need further assistance.
