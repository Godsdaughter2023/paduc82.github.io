System Image Including OneDrive Files / Folders - Microsoft Community

bob gross say:

System Image Including OneDrive Files / Folders

If I create a system image of a hard drive on one device (PC) where I have OneDrive files and folders synched, will it mess things up if I have made changes to those (OneDrive) files / folders later with another device, and then restored my system image to the PC afterwards? Any changes made to files would be from the same Microsoft account. I'm hoping what would happen is that the newest version of files and folders would sync with the PC, replacing the old versions if the image were restored later, but it would be helpful if someone that has done this could verify that it works that way.

Thanks in advance,

Bob

a. user say:

That's a great question.

You probably understand that OneDrive is one of the features of a Microsoft account. Even if you didn't explicitly sign up for a Microsoft account, if you're using OneDrive, you've got one.

And you also understand that a Microsoft account is simply Microsoft's version of a cloud account, just like iCloud is Apple's version and a Google account is Google's version.

In a cloud account, everything syncs to the cloud. Any changes that you make to OneDrive on any of your devices gets synced to the cloud, so the copy of OneDrive in the cloud always represents the most recent version of the OneDrive folder, and that becomes the 'official' version of your OneDrive folder on all of your devices.

So when you restore the image, the contents of the OneDrive folder that has been restored to your PC will be synced to match what's in the cloud at the time, because what's in the cloud at the time is more recent than what's in your restored OneDrive folder at the time you restore it.

If for any reason you don't want the syncing to occur, make sure that your PC is disconnected from the internet when you restore the image.
