Google is a very good friend. But ... please keep this in mind:

A clean install is an advanced technical procedure. It's not for a beginner, no matter what someone with the word 'Microsoft' in their name might say. If you're going to try this you need to be very, very well prepared.

Before you even get started, you must backup all your personal data. Don't leave anything out, because after a clean install you'll never see it again.

wasted12345 say: Well its got to be XP windows update - see the KB article.

Anyway, I figured it out. The "C:\WINDOWS\system32\iacenc.dll is Present" means the update doesn't need to create the file if it exists. Unfortunately the file that exists is 0KB in length and the Microsoft update process is too stupid to figure that out.
