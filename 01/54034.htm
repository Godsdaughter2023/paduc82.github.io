    How to Copy the Microsoft Speech Recognition Training File to another profile

    1. Browse to the download page for the Windows Speech Recognition Profile Tool.
    2. Click the Continue button to validate your Windows installation.
    3. Then scroll down the page and click the Download button next to WSRProfile.exe.

    11. Move the program WSRProfile.exe to the new profile as well, or download it again.
    12. Run WSRProfile.exe, and click Restore my speech profile.
    13. Click the Browse button. Navigate to the removable storage device you used to move the speech recognition profile to the new profile, and double-click the profile.
    14. Click Use the current speech profile to make the backed-up speech recognition profile as the default.
    15. Click Close.

    - Whatever brand audio your machine uses you need to verify the drivers are installed. If they are installed you will find them

    listed in Programs and Features (or Settings>Apps>AppData).

    Your machine should have a profiling software for it's integrated/onboard audio, with a built in equaliser among other settings

    such as Balance.

    (Balance can also be changed in Settings>System>Sound>Device Properties>Speakers/headphones> Properties>Levels)

    - On machines with Realtek HD audio the profiling software is called  Realtek HD Audio Manager.

    There are several possible locations where you can access the Audio Manager ...

    - A shortcut icon in the Taskbar > Notification area.

    - Windows Control Panel

    - Or failing the above, navigate to -  C:\Program Files\Realtek\Audio\HDA 

    The Audio Manager launcher with my setup is titled RtkNGUI64.exe

    If you need to open the Audio Manager from this location or Control Panel you should be able to locate an option

    in the Audio Manager settings to pin Audio Manager to the notification area for quicker access in future.

    - There are a  number of different layouts for the user interface, but they all have several EQ pre-sets with

    titles such as 'Live', 'Studio', 'Club' etc, as well as a 'Custom' option.

    Select the Custom option and if an EQ Window with ten frequency sliders is not displayed there should be a button

    to click on to show the EQ sliders.

    Adjust to suit your needs, and save the profile.

    - There are also several 'Reverb/Environment' settings to choose from. You may find it best to disable this option.
