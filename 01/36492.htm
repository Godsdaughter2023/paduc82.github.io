keith spitz say: Thank you for the prompt response.

I corrected the problem by turning off and back on the Windows Search feature (see my notes below).  To answer your questions, though: 1) there had been several recent changes (by our corporate IT department), and 2) the error occurred each time the Windows Search service tried to run.  I did not try Safe Mode, but I did run SFC which did not report any problems.

To disable and then re-enable Window Search:

1. From Windows: "Start" -> "Control Panel"

2. In the Control Panel main window, click "Programs".

3. From Control Panel Programs, click "Turn Windows features on or off" (under "Programs and Features")

4. In the pop-up window, after it populates, locate "Windows Search", uncheck it, and click "OK".  This will force a restart.

5. Repeat the process, except turning the "Windows Search" feature back on.

You can check to see if the Windows Search service is running by:

1. "Start" -> "Run".  Type "services.msc" and press Enter.

2. Scroll through the list of services until you find "Windows Search".  If it's status is "Started" you are set.
