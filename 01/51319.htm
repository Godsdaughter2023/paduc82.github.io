tanoj.dell say:

Try this

  1. Click Start, click Run, type regedit, and then click OK.
  2. Locate and then click the following registry subkey:
    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\PCI\<var>1394_hc_hw_id</var>\<var>1394_instance_id</var>\Device Parameters
  3. If the SidSpeed value does not exist in the right pane after you click the Device Parameters subkey in step 2, create it. To do this, point to New on the Edit menu, click DWORD Value, type SidSpeed, and then press ENTER.
  4. Right-click SidSpeed, and then click Modify.
    1. In the Value data box, type one of the values that are shown in the following table, and then click OK.
    2. Collapse this tableExpand this table
    1. Value
    		Speed
    0 S100 speed
    1 S200 speed
    2 S400 speed (default value)
    3 S400/S800 speed (Windows XP Service Pack 1 [SP1] value)
    1. Note If you try to use a value that is larger than 3, SidSpeed will use a value of 0 (S100 speed).
    2. Quit Registry Editor.
