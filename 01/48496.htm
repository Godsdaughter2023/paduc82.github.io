Hi,

 

Thank you for writing to Microsoft Community Forums.

 

When you say local disk D or E, is it an external drive you are referring to, or a partition on an internal hard drive?

Did you recently make any changes on the computer?

 

Most OEM computers come preconfigured with disk partitions like the System (c) partition where the operating system is installed, the System Reserved partition which contains boot related information, a Recovery partition containing recovery image, an OEM partition containing OEM specific recovery tools. All these partitions except C are hidden by default. They are managed by the system and should not be altered by the user.

Based on your requirements you can create additional partitions using the built in Windows Disk management tool.

To know more about partition types, refer to the link: Hard drives and Partitions

 

To create new partitions in Windows 10, refer to the instructions given in the link: Create and format a hard disk partition

 

If you have already created any additional partitions on your computer but the partitions are not visible in File Explorer, it is possible that the partitions are hidden.

 

To unhide the partition, you just need to assign it a drive letter in disk management.

  1. Open the Rum dialog box by pressing the Windows key and the R key together.

  2. Type Diskmgmt.msc and hit the Enter key.

  3. Right click on the partition which is hidden

  4. Select change Drive Letter and Paths.

  5. Click on Add.

  6. Select the driver letter from the drop down list.

  7. Click Ok.

  8. Go to File Explorer and see if the drives are visible.

 

Hope that the information above is helpful.
