ken blake say:

Printers are neither compatible nor incompatibile with computers. Whether they are compatible with what you have depends on what operating system you are running, not with what computer you have.

It's very common that older models of printers do not work with newer versions of Windows . Note that I'm talking about older *models*, not with how long you've have the printer. This is because the printer manufacture does not write drivers for them to be used with the the newer Windows versions. The reason he doesn't want to write the drivers is because he wants to sell newer printers.
