Microsoft Built-in ISO mount utility drives to use the next available drive letter. If that drive letter is already taken up it will display the error "You don't have permission to mount the file". 

Changed external drives to S, and T and then no longer got the error message for mounting the file.

2: Whichever shortcut you are using to launch the window, right click it and select properties. Under the 'Run' drop-down, choose "Maximized".
