Hi Joshua,

Have you check the time setting by following the steps in the article?

If you don't know how to check the registry, here are some additional steps in details:

1. Press Windows + R on the keyboard.

2. Input regedit in the Run window.

3. In the Registry Editor, navigate to HKEY_USERS\.DEFAULT\Software\Microsoft\MSOIdentityCRL from the left panel.

4. Check whether there is a ClockSkew DWORD value on the right.

If yes, it indicates that the time zone setting is not correct on the computer, please adjust the setting and make sure you are using the same time zone settings for Skype for Business and the machine. The Skype for Business time zone is determined by the Office 365 settings. You can check it by going to Outlook Web App > Settings > Options > General > Region and time zone.
