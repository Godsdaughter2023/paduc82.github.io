Port Forwarding - Microsoft Community

xbr dionysus say:

Port Forwarding

Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - password

 Port Forwarding

  1. Open web browser.
  2. Login to router by entering http://192.168.0.1 into the address bar
  3. The router will then prompt you to enter the user name and password. The default user name and password are: "admin", "password".
  4. From the menu on the left select Services under Security.
  5. Select Add Custom Service
  6. In the following Screen enter a name “Xbox”. Set the type to Both. Set both the Start Port and Finish Port to 3074.
  7. Click Apply.
  8. Select Firewall Rules on the left side menu.
  9. Select Add under Inbound Services.
  10. Select Xbox from the Service drop down.
  11. Select ALLOW Always from the Action drop down.
  12. Type in the Xbox’s IP Address for Send to LAN Server.
  13. Select Any for WAN Users.
  14. Click Apply.
  15. Select Apply at the bottom of the Firewall Rules page.

 

Disable Port Forwarding

  1. Open web browser.
  2. Login to router by entering http://192.168.0.1 into the address bar
  3. The router will then prompt you to enter the user name and password. The default user name and password are: "admin", "password".
  4. Select Firewall Rules on the left side menu.
  5. Select the rule you want to remove.
  6. Click Delete.
  7. Select Apply.
