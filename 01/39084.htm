Hi Tril645

Rest assured, there is absolutely no privacy issues with the Windows Error Reporting service

That diagnostic data sent to Microsoft contains information about your hardware, device drivers, the apps running when the error occurred - not your personal data

That error data is processed by a large servers, and stored with the millions of other crash reports

That data is analyzed and if found to be a common error in Windows 10 can result in a patch being pushed out to resolve the issue

There is no human involved in the process to read your data, unless the error if found to be widespread, then someone needs to code the patch for Windows, it is an automated process to make Windows more stable, that is all . . .
