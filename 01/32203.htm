Firmware Update - Microsoft Community

xbr dionysus say:

Firmware Update

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

How to Find Firmware Version

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted to login type admin for both the username and password.
  5. Under the Basic Menu click on the Home option.
  6. The current firmware will be on this page next to Firmware Version.

 

 

 Firmware Update

 

Note: You will need to download the firmware from the ISP website or the manufacturer’s webpage.

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted to login type admin for both the username and password.
  5. Under the Advanced Menu click on the Firmware Upgrade option.
  6. Click on the Browse… button.
  7. Locate the file you downloaded from the manufactures web site and select it.
  8. Click on the Update Gateway button.
  9. Wait for the gateway to update.
  10. It may take 1 minute for the gateway to reboot after it finishes updating.
