Check to see if you have a restore point you can go back to

Once in Safe Mode, backup your data if you do not already have a backup
Open Device Manager and uninstall that driver you installed
Reboot to see if Windows will start Normally

rey g. santos say:

Try the keys "Alt+0176" or "Alt+0186". For Desktops, hold down the "Alt" key and type in the numbers from the numeric keypad. On laptop keyboards the “numeric keypad” is found by pressing the "fn" and "num lk" keys and then looking for the keys on your keyboard that double as numeric keypad keys when in NumLock mode. That is, for "Alt+0176", hold down the "Alt" key then press the keys M, J, 7, and O, which corresponds to numbers 0, 1, 7 and 6 for the symbol "°" and for "Alt+0186" press the keys M, J, 8 and O for the symbol "º". Note, the last key is letter "O" not number zero (0). When you're done deactivate the "num lk" by pressing the "fn" and "num lk" keys again or your "num lk" keys will always type numbers instead of letters.

Or if you are typing on MS Word, you can type the small letter "O", then select it, click on the Format menu, font..., on the "Font" properties, "Font" tab, under "Effects" click on the box before "Superscript" to check it. 

You can use the copy and paste facility on the symbol created on both process.
