devlin sco say:

Hi Denis,

Error code 0x8007065e can occur while downloading the latest update due to a corrupted update file. To help you resolve the issue, I suggest performing these troubleshooting steps:

Run the Windows Update troubleshooter

This automated diagnostic tool will scan your PC for update issues and fix them for you. Make sure that you're signed in as the administrator in order to run the tool.

  1. Click the Start button, go to Settings > Update & Security > Troubleshoot.
  2. Select Windows Update under Get up and running.
  3. Click Run the troubleshooter.
  4. Follow the on-screen instructions.

Disable third-security software

Some third-part security software may prevent download and installation of the latest updates for Windows 10. I suggest temporarily disabling the security software to complete the update. You can simply enable it once the update is complete.

Manually install the latest update

You can manually download and install the update from the Microsoft Update Catalog website.

  1. Go to the Windows 10 update history page.
  2. Find the most recent update KB for the version of Windows 10 you're using - it's usually at the top of the list.
  3. Take note of the KB number.
  4. Search for the KB number in Microsoft Update Catalog website.
    • Example: KB4340917
    • You can refer to this page to make sure which System type you have.
  5. Select the link for download, and then choose Download.
  6. Select Save if you are prompted.
  7. When the download complete, select Open to start the update.

Repair Windows system file corruption

  1. Click the Start button and type Command Prompt.
  2. Right-click Command Prompt and select Run as administrator.
  3. If prompted, click Yes or enter the administrator password.
  4. Enter DISM.exe /Online /Cleanup-image /Restorehealth and then press Enter.
    • note the space before each "/"
    • This step may take a few minutes to start and up to 30 minutes to run and complete.
  5. Enter sfc /scannow and then hit Enter.
    • note the space between "sfc" and "/"
  6. Wait for the process to complete.

Reset the Windows Update components
You can reset the Windows Update components using the Reset Windows Update Agent tool created by Manuel Gil. You need administrative permission to run this tool.

  1. Open the download page.
  2. Click ResetWUEng.zip.
  3. Select I agree.
  4. Select Save if you are prompted.
  5. When the download complete, open ResetWUEng.
  6. Run ResetWUEng and follow the on-screen instructions.

Let me know the results.

a. user say:

Look, I realize that this community forum is for people who 'don't get it' with computers, but between 2017 and now did you really not backup even once? Your grad school papers are vital to your education and possibly to your career, and you don't make backup copies? Nothing gets backed up in Windows automatically; this is something you have to setup yourself. You should be backing up every day, at least.

In Windows, we are supposed to store our personal files in the C:\Users folder. If you look at that folder you will see that is has a sub-folder with your user account name. That's where your files go.

Open the folder C:\Users\<your user account name> and you will see individual folders called Documents, Music, Pictures, etc. Those folders are never touched during an update, because they were set aside to hold your personal files. If you store your files anywhere else on your computer they might very well get deleted, because feature updates typically make extensive changes to your computer.

The first place to check for your files is inside C:\Users\<your user account name>. If you stored your files elsewhere there is one other place that you can check. After an update, Windows creates a folder called windows.old that is used to rollback an update. Your missing files might be inside that folder.

Files stored online are not affected by an update, because updates only change what's on your computer. So if you stored files in cloud storage like OneDrive, Google Drive, Dropbox, etc. those files will still be there.

This is a long shot, but you may have a user account on your school's servers, and there may be copies of your work there.

If I can think of anywhere else your files might be, I'll add to this post.
