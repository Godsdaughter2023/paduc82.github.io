tom_wc say:

The easiest way to create abridged and compilation albums of your favorite photos

Edit:  The November release provides a better approach - see the reply below.

Identifying your favorite pictures is not only the first step to creating compilation albums, but also saves you a lot of time when looking at individual albums.  Here's how to do so when you're first editing your pictures.  

With Photo Gallery reportedly being unsupported after January, it's only a matter of time before we'll no longer be able to use its flagging capabilities to identify favorites.  And Explorer's "select" option only works as long as no updates are occurring in Photos at the same time.

So how can we conveniently identify favorites while editing an album or folder?  Going into Explorer's Properties for each one is out of the question.  And in Photos once you select pictures you cannot go into full screen mode.  So I think we're expected to handle an album/folder in two passes, first editing it and only later selecting favorites for inclusion in a separate album.

But for those who are used to identifying favorites while editing, here's how you can do it.  Go to the album's folder in Photos but don't click on anything.  Then do the same in Explorer, sorting it by date to match Photos' sort.  Then start a second copy of Photos from Explorer.  You can edit from this second window and select favorites in the original Photos window.

Unfortunately it's not practical to then use Explorer to rate the selected pictures; it's easiest to simply add them to a new/different album.  To make it easy to locate, begin the name with something that will cause it to appear at the top of an alphabetical listing (quotes won't work but you can use parentheses or ".", "_" or "#").  You'll normally want to use the alphabetical option for compilation albums since the listing by date uses the date of the oldest photo in the album. 

Finally, what about pictures that you've rated or tagged in the past?  If you're OK with viewing them in date sequence simply cut/paste the ones you want to a temporary folder, use Photos to then load that folder to an album, then cut/paste back to the original folder (Photos won't get confused).

Is there a better approach?

  2. On the run command box, type msconfig, and then hit Enter.
  3. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then tap or click Disable all.
  4. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.

  6. Close Task Manager.
  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.
