Basically,  05 is an Access Denied (05) error.
80508015 ERROR_MP_FAILED_TO_SPYNET means the system failed to connect with Microsoft's reporting server, SpyNet, to which the scan results are sent to.
Scan ERROR: resource file://C:\pagefile.sys (code 0x00000020) may be 0x80508020 ERROR_MP_BAD_CONFIGURATION.

The Access Denied and 0x80508020 result codes most likely mean that the process did not allow the MRT to scan the file because it was in use, which is likely for pagefile.sys,  or that when the MRT is automatically run when it's downloaded by Automatic Updates, it is running restricted by KIS.

NOTE: In the Clean boot state KIS' firewall will be Disabled. Please ensure that the native Windows firewall is now On by openingControl Panel >Windows Firewall.
