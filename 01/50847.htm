Skype notification not working right on Android - Microsoft Community

tbaa say:

Skype notification not working right on Android

After recently installing Skype on my new Andriod phone I was surprised when I wasn't receiving call notifications, especially given that I had purchased a Skype Number was affecting my ability to make use of the service.

 

I started to research this problem and quickly discovered that there is a long history of this issue on Andriod.

 

I believe I have resolved this issue for me and am posting what worked in the hope that others may benefit.

 

The Problem

 

Skype will start and for a short while call notifications will be received for both Skype-to-Skype and Phone-to-Skype Number, then over time notifications will stop for both services, however, Skype-to-Skype seemed to be slightly more reliable than Phone-to-Skype Number.  Very annoying. 

 

The Fix

 

My testing quickly led me to think that the issue is somehow caused by a timeout of the connection between the device and Skype even though Skype is actively running in the background.

 

To keep this connection alive, Android uses a periodic heartbeat to prevent the connection from timing out. By default, this heartbeat occurs every 15 minutes on wifi and every 28 minutes on a mobile data connection.

This is problematic because most Wi-Fi routers and mobile carriers will automatically terminate this connection after a short time of inactivity.

 

The fix involved multiple setting changes and the installation of two applications from the play store - no root required.  It could be that not all these will be needed, but this combination works for me

 

  1. Install Push Notifications Fixer (PNF) to the phone -

https://goo.gl/BLZH8A

This application allows you to change the heartbeat interval (stored on the GSF database inside android data partition) to the value of your choice, in order to maintain the push service connection (GCM).

I set this to 1 minute - aggressive.
 

  1. Install Stay Alive to the phone -
     

https://goo.gl/Yykc7D

 

  1. Add PNF to the Stay Alive application

    I also added Skype to the Stay Alive application, but as long as its running in the background it should be sufficient as long as you set the next option
  2. Turn off the Android Battery Optimization settings for Skype, Say Alive and PNF
     

Go to Settings | Battery, tap the menu button (three vertical dots in the upper-right corner), tap Battery Optimization, tap the Not Optimized drop-down, and select the applications to disable Battery Optimization for.

 

Hope this information works for you, not every confirmation item may be needed, but this combination seems to be working for me.
