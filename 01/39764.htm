quantum foam say: First off, what do you use for an antivirus?  Some AV companies provide security suites with added firewall protection, which you may find of some use.

As far as firewalls go, Windows Firewall in Windows 7 is fairly good and you certainly can not go wrong with it.

Considering your requirement for maximum protection online, system performance, with the added need for a firewall, a more comprehensive suite might actually be preferred.  Here are a few examples (there are more to choose from, though based on what you are asking for, these three are well rated):

Kaspersky Internet Security
Bitdefender Internet Security
ESET Smart Security

If you are more budget-conscious and are looking for free firewalls, then Windows Firewall and COMODO Personal Firewall are two popular choices.  Just make sure you have a solid antivirus and antispyware program.  Popular free AVs include Microsoft Security Essentials, avast!, Avira, and AVG.
