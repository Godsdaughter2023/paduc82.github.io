jeeped say:
When I start my computer the notepad appears. I cannot do anything until I close the notepad. How can I remove the notepad from the start screen? 

By the notepad, I assume that you mean Notepad (i.e. notepad.exe) a text editor supplied with all versions of Windows.

Tap Windows key+R (i.e. Start, Run) and type msconfig then click OK. Look on the Startup tab for an entry that involves Notepad and/or a file ending in .TXT. Uncheck the entry and click OK in the lower right. Restarting Windows should not initiate Notepad from then on.

If you're having issues with an app you got from Microsoft Store, here are some things you can try:


• Update Microsoft Store: Select the Start  button > Microsoft Store . Select See more  > Downloads and updates > Get updates. If an update for Microsoft Store is available, it will start installing automatically. 

• Repair and reset the app: Select the Start  button > Settings  > Apps > Apps & features. Select the app that's not working, and then select Advanced options. On the page that opens with information about the app, select Repair. (Note: Some apps don't have this option.) If repairing the app doesn't fix the problem, select Reset.

      Pictures.pd6 is a hidden file...the following article explains
      how to show hidden files:

      http://windows.microsoft.com/en-US/windows7/Show-hidden-files

      a_user_here say:

      Just for some information.  I'm not sure why you can't get at the remaining 1TB (most likely something in the partition table), but the 2 TB limit has everything to do with the file system, and very little to do with the operating system.

      Here is a link on the maximums of given file systems.  Note that given a 512 sector, the practical limit for an NTFS partition/volume is 2 TB.

      https://technet.microsoft.com/en-us/library/cc938432.aspx?f=255&MSPPError=-2147217396

      Note just because 512 is the standard, that doesn't mean that you are restricted to it.  You could go to 1024 sectors and get to 4 TB.  But you would have to do this some how manually and then get Windows 10 to use that partition.

      P.S. Note that the Mac natively uses a Unix based file system, so it has different limitations.

      n-crum say:

      Bios/MBR based drives with NTFS are limited to 2 TB partitions:

      https://msdn.microsoft.com/en-us/library/windows/hardware/dn898504%28v=vs.85%29.aspx

      While deploying windows to HDD with GUID Partition Table (GPT) ,instead of MBR, each partition can have a total of 18 exabytes:

      https://msdn.microsoft.com/en-us/library/windows/hardware/dn898510%28v=vs.85%29.aspx

      It is the partition table used on the HDD that will limit that size of the supported disk.
