*FIX* Windows 10 OneDrive doesnt log on. Content not sync'd.

While working on my Dell Venue 8 Pro tablet with Windows 10 (10240 build RTM) 

I had an issue where One Drive appeared to just be a folder of contents. When I went to Desktop mode I noticed that the OneDrive app in the task notification was simply a grey cloud and it stated that I wasn't signed in.

I had reset the PC recently using the option to refresh (which fixed a sleep freeze issue) but I had chose to preserve my files.)

Additionally I had seen an update for the OneDrive app appear today (7/27/2015) 17.3.5907.0716

I located the install at C:\users\%userprofile%\appdata\local\microsoft\onedrive\

Here there is a folder called "Update" with the application OneDriveSetup.exe

The process I used was

1.) Kill "Microsoft OneDrive" process in taskmanager

2.) Moved files from OneDrive storage location to be safe (C:\Users\%userprofile%\OneDrive)

3.) Opened Administrative CMD

4.) Uninstalled OneDrive app  C:\users\%userprofile%\appdata\local\microsoft\onedrive\update\onedrivesetup.exe /uninstall

5.) Reviewed folder C:\users\%userprofile%\appdata\local\microsoft\onedrive\ to see most of the folders and such had been removed.

(Don't go deleting items in here to be safe)

6.) Ran the setup again for OneDrive C:\users\%userprofile%\appdata\local\microsoft\onedrive\update\onedrivesetup.exe

7.) Selected items to sync. (Obviously it was logging in at this point)

This process worked for me. Hope it helps others. So far I am loving Windows 10 otherwise!
