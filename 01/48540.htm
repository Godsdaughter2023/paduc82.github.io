spiritx say:
HI,

Upgrading to Windows 10 carries the same license you had with the OS
version you originally had. Whether OEM or Retail the Windows 10 upgrade
carries the same license. However the Windows 10 upgrade is tied to the
qualifying OS. So you cannot use the qualifying OS on a separate system at
the same time while the Windows 10 upgrade is on the upgraded one nor
can you use the qualifying OS on that same computer to dual boot.

To use the qualifying OS on the same or another systems (provided it had
that license) you would need to remove the Windows 10 upgrade.

The Windows 10 Upgrade is tied to the original license of the qualifying OS
and the hardware on which the upgrade was performed. It will not activate
on another system. However once upgrade you can perform a clean install
on the same system you originally upgraded (provided it was properly
activated.
