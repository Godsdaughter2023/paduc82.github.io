tryx3 say:

How to use the PrintScreen key

PrintScreen copies images to the clipboard, it does not print.  It has retained its pre-Windows 95 name [it used to be a printing control button].  Many people find that the Windows 7, Windows 10 utility "Snipping Tool" is a more convenient way of obtaining images from the screen [Windows Help has a decent explanation of the Snipping tool].

1  You use the poorly-named PrintScreen key to capture images of the whole screen or the active window to the clipboard.  The screen image can be retrieved from the clipboard by pasting e.g. pressing ctrl-v.  So to use PrintScreen to create a file containing a fullscreen image - press the PrtScrn key, open Paint or another graphics application, Paste, Save the image.  See also PrntScrn [for the screen] & Alt-PrntScrn [for the active window]

2  Variations:-

2.1  To capture the whole screen - PrntScrn  {this will also capture any visible Screentips and any visible right-click context menu}

2.2  To capture the current window - Alt-PrntScrn   {this will also capture any visible Screentips but not right-click context menus}

2.3  Windows 10 users can also use a WinKey-PrntScrn combination to automatically save a screen image to a file.  See Setting and checking the Screenshots folder

2.4  There used to be other variations using undocumented key combinations [Ctrl-PrntScrn, Fn-PrntScrn, Ctrl-Fn-PrntScrn] but they now have the same effect as using PrntScrn on its own.  These variations were set up by keyboard makers not Windows itself so it is possible that some of them still work.

3  If captured screen images are black or dark grey while you are using online banking, you probably have the security utility Trusteer Rapport installed.  Their support site explains how to enable screenshots without completely disabling Trusteer Rapport; I extracted their instructions and posted them here.  Personally, I do not inhibit any Trusteer Rapport functions.

4  For problem applications that refuse to run properly after PrntScrn is used [online games are the only examples I have heard about]-

4.1 Close the problem app,

4.2 Right click the application’s shortcut

4.3 Select Properties, Compatibility tab, Set Disable desktop composition.
