Thank you for posting to Microsoft Community.

I understand that you have already redeemed the code for Minecraft. I would suggest check the Order History in your Microsoft account to see if the code is already redeemed and linked your account. To do that you can refer the article to Redeem a gift card or code to your Microsoft, under ‘Troubleshooting’, please refer the section ‘The code was already redeemed’. After this, make sure to check if you have logged into Windows Store using the same Microsoft account through which the code was redeemed.

If the code is redeemed, linked to your account and if you have logged in with the same account in Windows Store on your system, but you are still not able to find the option to download and install the game, I would suggest you to follow the steps below and check if it is helpful to you.

Steps to reset Windows Store:

  1. Click on the Start button, type cmd, right-click on it and select Run as administrator.

  2. Type wsreset.exe and press Enter.

  3. The execution of this command may take some time, after which, you can check if you are able to install the game.

If the issue persists, please reply to us with the results of the suggestions provided above. We will look into this and we will be glad to assist you accordingly.
