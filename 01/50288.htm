jackthesmack say:

I was able to fix this thanks to Microsoft tech support.

  1. Hit your windows key and type in regedit and hit Enter
  2. Go to HKEY_CLASSES_ROOT\Directory\shellex\PropertySheetHandlers\Sharing
    • If the folder does not exist, create it.
  3. There should be a (Default) value with type REG_SZ and Data {f81e9010-6ea4-11ce-a7ff-00aa003ca9f6}
    • If the Data is blank, double click on it and paste in {f81e9010-6ea4-11ce-a7ff-00aa003ca9f6}
  4. Close regedit, and open the properties of a folder. The sharing tab should be there.

Alternatively, you can open notepad and paste the following in, then save as SharingFix.reg, then double click the file:

Windows Registry Editor Version 5.00


[HKEY_CLASSES_ROOT\Directory\shellex\PropertySheetHandlers\Sharing]
@="{f81e9010-6ea4-11ce-a7ff-00aa003ca9f6}"

Hi Ann

I'm about half in agreement with Mark... ultimately there's an element of risk in any form of 'back-up' but it's better to have any back-up than none at all. I use both cloud storage and two external (500 GB) hard drives on the basis it is highly unlikely all three would fail at the same time! A fourth measure (sometimes forgotten these days) is get some prints of your most important pictures - then if we do get hit by the great solar flare you still have those for sure.

How much a hard drive will cost depends where you are... there are plenty to choose from on (eg) Amazon... look at customer reviews (particularly the ratio of bad to good) and if affordable go for one that seems to perform reliably.

Don't be dissuaded from using free 'one drive' or 'i-cloud' storage as a second option, because hard drives can fail or be damaged too! - Regards, Ric.

Deleting saved game cache for Minecraft, then full reboot - still missing

Uninstalling Minecraft, full reboot, then reinstall - still missing

Starting a new world with the same name, supposedly to reboot the world minus anything that was being carried at the time - still missing
