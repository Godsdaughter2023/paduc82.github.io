a. user say:

Hello,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 Family.

From the description of the issue, I understand that you are unable to take full ownership of the files and folders.

I would recommend you to see if you are logged in administrator account, if not then enable it. Please follow the below steps.

If the issue occurs only with the downloaded files, then you can try the steps below

Try to open Internet Explorer 10 in no-add-ons mode and check if you’re able to load the websites.

1. Press the ‘Windows+R’ key on the keyboard.

2.In the Run window type ‘iexplore –extoff’ and click ‘OK’.

3. Now try to open the websites to check if you’re able to load the websites.

Method 2

Disable the option for Protected Mode. Please follow the below steps.

1. Press the Windows key + R.

2. Type inetcpl.cpl and press Enter.

3. Click the Security tab.

4. Uncheck “Enable Protected Mode”.

5. Click OK.

6. You will need to restart Internet Explorer if it is open.

However, if the above scenario does not matches with yours, I would suggest you to enable the built in administrator account.

You need to run the command from an elevated command prompt. To do that you need to do the following:

  • Press the Windows key t+ R from the keyboard to open the Run Command Dialog Box 
  • Enter cmd and right-click on the Command Prompt result that should appear.
  • This opens a list of options at the bottom. Select Run as administrator there.
  • Accept the UAC prompt
  • Enter the following command to enable the hidden administrator account: net user administrator /active:yes

Hope the above information was helpful. If you need further assistance, please do let us know and we would be happy to help you further.

Thank You.

I had the same problem. It affected not only downloading documents but opening existing files and applications. I had to change my Outlook settings to Run as Administrator in order to open it.

I solved the problem by opening the File Explorer

Browse the the C Drive

Right click and select

Click the Security tab

Under groups and user names check to see if your username is there. On my computer it showed SYSTEM, Users and administrators but did not have my actual Username

Click the Edit button

Click the Add button

Under Enter Object Names type in your Username.  If you are not sure it may be your username for your microsoft account before the @domain.com

Click OK

Highlight the name you just added and under the Permissions box click Full control

OK

It will take a long time and go through everything on your system.  

You may experience some errors on system files that cannot be changed. Just click Continue to get through those.

Note:  I had already added my Username to some individual folders prior to trying this step. It worked on those folders but not on everything.
