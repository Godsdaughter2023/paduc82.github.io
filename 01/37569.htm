How to Find SSID (Network Name)

Login Information



  • Gateway: 192.168.2.1
  • Username: None
  • Password: (blank)





How To Find SSID (Network Name)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Channel and SSID" from the menu on the left.
  5. The network name will be labeled as "SSID".

Direct the fax installer to C:\Windows\ServicePackFiles\i386

First, however, look in that folder for the missing files (probably fxsapi.dll and fxcfgwz.dll).  If, instead, you find fxsapi.dl_ and fxscfgwz.dl_, these files need to be expanded first.

To expand files:

  1. Click Start , click Run , type Cmd , and then clickOK .
  2. Type cd\ , and then press ENTER.
  3. At the command prompt, type expand <var>source destination</var> , where <var>source</var> is the path to the file that you want to expand on the Windows XP installation disk, and <var>destination</var> is the path to the location where you want to save the file, and then press ENTER.

* Which programs or application you're referring to? Did they installed successfully?

In general you can see all installed application under Programs and Features in control panel. If you're not able to see them, then there might be some issue with system files. Try the below mentioned steps and see if it helps:

  1. Press CTRL+Shift+ESC to open Task Manager.

  2. Click File > Run New Task

  3. Make sure you have a check mark beside “Create this task with administrative privileges”

  4. Type CMD

  5. Type the following commands at the CMD prompt:
            dism /online /cleanup-image /restorehealth

Let us know if problem still persist, we're glad to assist you.
