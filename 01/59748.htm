  1. Restart the computer and start tapping the F8 key about twice/second starting as soon as the lights on the keyboard flash.  If you see the Windows logo screen, you waited too long to start tapping and will have to restart and try again
  2. The Advanced Boot Options menu will appear (white type on black screen).  Use the up/down arrows to select (highlight) Safe Mode and press Enter
  3. When Safe Mode starts, you should see an icon for an account named "Administrator" in addition to the icon(s) for the account(s) that you normally use.  Select the Administrator icon, leave the password blank, and log on
  4. Once the Administrator Desktop appears, go to Control Panel > User Accounts and reset or change the password for your normal user account.
  5. Restart the computer.

In case anyone's still looking for an answer (and doesn't want to install a 3rd party app), powershell will do the trick (and it comes installed in Windows 7).  It's great for things like this, so I'd suggest anyone doing anything even a little advanced (find and replaces, copies, etc) take a look at it.

To do mass copies, open powershell (start->run->powershell <enter>)

Type in: Copy-Item <starting_path>\* <ending_path> -recurse

For example, to copy everything from your backup server dev folder to your local dev folder (which is sort of what I was just doing... a hacky version control for now)

Copy-Item \\backupserver\dev\* c:\dev\ -recurse

I wonder if you would get more targeted help/advice if you would tell us the KB number or Update number that is failing to install...

If the update is for some kind of hardware that error can mean that the device is not installed, connected or otherwise afflicted.

Generally speaking it is never a good idea to accept install hardware driver updates offered by Windows update since Microsoft does not know what the best drivers are for your system - you should only get them from the hardware manufacturer and only when you know the update will fix a specific issue.

When you see hardware driver updates like video, audio , network it is best to hide them so you never see them again or install them accidentally.

For the future there is an adjustment you can make in Windows 7 so Windows Update will not offer hardware driver updates and you can read about it here:

https://support.microsoft.com/en-us/kb/2500967

Dunno if it works as advertised so it is wise to look at all offered updates and decline/hide any that have to do with hardware or drivers.

Step 1:

To diagnose whether this issue occurs in “Safe Mode”, let’s reboot the computer in safe mode with networking and check if the issue appears there as well.

Safe mode is a diagnostic mode for Windows that starts your computer in a limited state. Only the basic files and drivers necessary to run Windows are started. This helps us in identifying whether the issue is due to some third party programs conflict. The word Safe mode appear in the concern of your monitor to identify which windows mode you are using.

Follow the steps given below and check if that helps.

1.      Unplug external devices connected to the computer expect keyboard and mouse.

2.      From login screen, press Shift key and click on the Power button and click restart to enter Advanced Boot Options.

3.      In this page select troubleshoot option.

4.      And click on Advance option.

5.      Select Startup setting from advance option.

6.      Here you will find Restart button, just click on it.

7.       In new windows press F5 to login with safe mode with networking.

If the issue does not exist, try to clean boot the system.

Step 2: Perform Clean Boot.

Placing your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue. Check this issue in clean boot state.

  1. Press Windows key + X key. And select Run.
  2. Type msconfig in the Run box and hit Enter.
  3. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then tap or click Disable all.
  4. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.
  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.
  6. Close Task Manager.
  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

For more information refer the below link.

How to perform a clean boot in Windows

Note: After troubleshooting, refer to this section "How to reset the computer to start normally after clean boot troubleshooting".
