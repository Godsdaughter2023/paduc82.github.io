If you want to back up newer files only, and not to remove any files in the backup that are no longer on the thumb drive, a command like this would be good.

(This assumes the thumb drive is T: and you want to copy it's contents to a folder on your C drive.  Run xcopy /? in a command prompt if you want to see the meaning of the switches.)

 xcopy  /s /d /y  T:\*.*   C:\MyThumbBack\
