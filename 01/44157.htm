chaelwyn say:

On the login screen quickly double-check whether or not you have a Local Account or an Associated Account. You can do this by simply looking to see if there is an e-mail address under the Account Name on the log-in screen. If there is no e-mail address displayed there this means you are using a Local Account and will want to follow these steps:

 

  1. From the Login Screen either Hold Shift on your Touch/Type Cover and, while holding the button, tap the Power Symbol and select Restart or, if you do not have a Cover, press the Accessibility symbol at the bottom left of the Login Screen and select Accessibility Keyboard. On the keyboard tap Shift so the key is highlighted, tap the Power Symbol and select Restart.
  2. The Surface will boot into a blue screen with four options.
  3. Select Troubleshoot
  4. Select Reset my PC
  5. Select Completely reinstall Windows

If at any point you see a screen asking you to enter an Activation Key you will want to select Skip this drive.

 

I hope this helps!
