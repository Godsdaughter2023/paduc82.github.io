If you have the option to set 1360x768, stick with that. The 360 will scale all output to fit the resolution selected. 720p or 1080p output will require the TV to perform scaling. That typically introduces latency and leaves you at the mercy of your TV's scaler.

Most games on the Xbox 360 are 720p Native, then upscaled internally if your selected resolution is higher than 720p (1360x768,1440x900,1680x1050, 1080i, 1080p). There are no 1280x720 (720p) HDTV's i know of, but there are 1360x768 (labeled as 720p, but really 768p) HDTV's that are common. So everyone who has an HDTV will most likely be set to higher than 720p at all times. There should NOT be internal slowdown from upscaling. If there is any framerate hit, its how the game is coded and since Crysis 1 was a PC graphics powerhouse game, it probably won't run the best on a 360 anyways. 

-

You do NOT want to use 1080i on a flat-panel TV. 1080i is an interlaced signal and gets DEinterlaced into a progressive signal, but this takes more time than scaling an already progressive signal. Keep it at 1360x768 if thats the native resolution of your TV.

Yes, there are several methods using image backup (not clones):

1) How to move windows 7 to a new or larger hard drive using Backup and Restore
http://www.pagestart.com/win7bckuprstrnhd072610.html

2) Norton Ghost: Rsstore the System,Reserved partition first and then the Windows partition. Below is a step by stap guide.

How to use the Ghost Recovery Disk’s 'Edit' option to successfully recover Windows:
1) Boot your computer using the Ghost 15 SRD disc. Accept the licensing agreement and when you see the Norton Ghost 15 “Welcome to the Symantec Recovery Disk” screen, connect your external USB drive. Wait about 15 seconds for the drive to be recognized.

2) Click on the ‘Recover My Computer’ option to start the “Recover My Computer Wizard” and click the ‘Next’ button. If you see the “No recovery points were automatically detected” message, click the ‘OK’ button.

3) From the ‘Select a Recovery Point to Restore’ window set the ‘View recovery points by:’ to the by ‘Filename’ option.  Use the ‘Browse’ button to locate the image backup folder on your USB drive by double clicking on the ‘Computer’ icon and then the USB drive.

Select the image backup folder you created earlier and click the ‘Open’ button located in the lower right corner.

4) Next set the ‘View Menu’ option (upper right of explorer window) to ‘List’.

5) You should now see two or more recovery files.
One of the files ending with a drive letter (MyComputerName _C_Drive.v2i ) and the other file having a name something like: MyComputerName_System Reserved0_1_Drive.v2i
If you have a third partition: MyComputerName_D_Drive.v2i

Note: Substitute “MyComputerName” for the actual name of your computer.

6) Click on the ‘MyComputerName_System Reserved0_1_Drive.v2i’ file and click ‘Open’.

7) In the ‘Select a Recovery Point to Restore’ window the ‘Recovery point details box should list the Drive: as ‘System Reserved (*:)’ with a size of 100MB, Click ‘Next’.

Notes: The new drive you installed “may” result in an “Initialize Disk Partition Structures’ message Window. If you see this message, place a check mark in the box for the disk (your new drive) to initialize it. Click the ‘OK’ button to close the window. Click ‘Yes’ to write the partition structures.

Here’s the first secret to success:
8) In the ‘Drive to Recover’ window click the ‘Edit’ button.

Notes: If the drive was previous partitioned and formatted the ‘Volume Label’, ‘Size’, ‘File System’ and ‘Type’ will be populated. More specifically there will not be any “Unallocated” space. In this case you must first click the ‘Delete Drive’ button and click “OK’, ‘OK’ and then ‘Edit’.

9) If you see the ‘System Reserved’ Volume Label/Partition listed, click on it and select ‘Delete Drive’ and click ‘OK’.

Note: you “may” see a warning message about destroying any existing data if there are any other partitions listed in the ‘Volume Label’ column for your new drive, delete them as well.

10) Now click on the unallocated space (it should now already be highlighted in gray and the size should represent the full size of the new drive), it should now be highlighted in Blue.

11) In the lower left, set the ‘Partition Type:’ to Primary partition. Next uncheck all the boxes and then only place a checkmarks in the ‘Set drive active’ (for booting OS) option and ‘Restore master boot record’. Click the ‘OK’ button and review the settings.

12) In the ‘Drives to Recover’ window the ‘Date’ column should reflect the date of the Image Backup you created from your old drive. The ‘Verify recovery point before recovery’ box should be selected (Checkmark). Click the ‘Next’ button.

13) At the ‘Completing the Recovery My Computer Wizard’ window, you can once again review your settings (click the ‘Back’ button if you need to make any changes).

14) DO NOT place a check mark in the ‘Reboot when finished’ box as after the System partition has been restore you still need to restore the Windows partition and any other partitions that existed on the old drive.

15) Click the ‘Finish’ button and click ‘Yes’ to start the recovery. The recovery process should start without any error message this time. Since you are only recovering the System partition it should only take about 1 minute to complete. Click the ‘Close’ button when the recovery has completed. You should see the Welcome screen displayed.

Now the next partition (as originally listed in Disk Management on your old drive) must be restored. This is typically the Windows 7 partition, but could also be the recovery partition on the original drive. For this example the next partition on my drive is where Windows 7 is installed.

16) Click on the ‘Recover My Computer’ option and from the ‘Recover My Computer Wizard’ click ‘Next’.

17) From the ‘Select a Recovery Point to Restore’ window set the ‘View recovery points by:’ to by ‘Filename’.  Use the ‘Browse’ button to locate the image backup folder; Ghost will remember the folder you choose in step 2.

18) You should once again see same set of recovery files.

19) This time click on the ‘MyComputerName_C_Drive.v2i’ file and click ‘Open’.

20) In the ‘Select a Recovery Point to Restore’ window the ‘Recovery point details’ box should list the Drive as: No Label (C:). (Or the volume label name you assigned). Click ‘Next’.

Here’s the second and final secret to success.
21) In the ‘Drive to Recover’ window click the ‘Edit’ button.

22) You see the ‘Windows’ partition listed just beneath the ‘System Reserved’ Partition, click on the Windows partition (leave the System partition untouched) and select ‘Delete Drive’. This will consolidate all the available drive space as ‘Unallocated’.

23) Note that the unallocated space should already be highlighted in gray.

24) Place a check mark in the box labeled ‘Resize drive after recover’.

25) Specify the size you want the Windows partition to be. For example if it was 300GB on the old drive and you now want a 600GB partition on the new drive calculate the size in binary that you will need to enter, using a calculator enter 600 * 1024 = 614400. Use this number to enter into the Size box.

Note: If the Windows partition is the only remaining partition to be restored just check the ‘Resize drive after recover’ option, no calculations of partition size are necessary as Ghost will automatically expand the Windows partition to the full size of the drive. If the Windows partition has already been restored and you are repeating these steps to restore your ‘Data’ or ‘Recovery’ partition and this is the last partition on the drive, no calculations of partition size are necessary as Ghost will automatically expand the Data/Recovery partition to occupy the remaining available “Unallocated” space on the drive.

26) In the lower left set the ‘Partition Type:’ to Primary partition. Next checkmark only the following option: ‘Check for file system errors after recovery’.

At this point you should have checked: Resize drive…., Primary partition and Check for file system errors …, no other options should be selected.

Click the ‘OK’ button and review the settings.

Once again place a check mark in the ‘Verify recovery point before recovery’ option box if it is not already checked. Now click the ‘Next’ button.

27) At the ‘Completing the Recovery My Computer Wizard’ windows you can once again review your setting (click the ‘Back’ button if you need to make any changes).

28) DO NOT place a check mark in the ‘Reboot when finished’ box.

29) Click the ‘Finish’ button and click ‘Yes’ to start the recovery. The recovery process should start without any error message this time. Recovering the Windows partition will take considerable longer to restore than the System partition. Click the ‘Close’ button when the recovery has completed.


30) If you have no other partitions to restore, remove the Recovery CD and click the ‘Exit’ button to reboot to Windows. Otherwise repeat steps 16 to 29 and choose the remaining “_Drive.v2i” file.

Note: In my case my old drive also included a partition I have create for storing data (D:) located after the Windows partition, so I repeated these steps except this time choosing the “’MyComputerName_D_Drive.v2i’ file.

Since this was the final partition, step 22 should read as follows; You see the ‘Data’ partition listed just beneath the ‘Windows’ Partition, click on the Data partition (leave both the System Reserved and Windows partitions untouched) and select ‘Delete Drive’. This will consolidate all the available drive space as ‘Unallocated’. I checked the ‘Resize drive’ box but left the value unchanged. The new D: partition will now occupy the remaining available space on the drive.
