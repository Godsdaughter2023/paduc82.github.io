Hello Anmol,

Thank you for posting your question on Microsoft community.

I appreciate you for providing details about the issue.

This issue may occur either due to corrupt device drivers or settings.

I would suggest you to run hardware and devices troubleshooter and check the issue. Please follow these steps.

a. Press Windows + W keys, type troubleshooting in the search box and press Enter.
b. Click "View all" and then click "Hardware and Devices".
c. Click "Next" and follow on-screen instructions.

If it does not help, try uninstalling and reinstalling all USB controllers and check the issue. Please follow these steps.

a. Press Windows + X keys and click device manager.
b. Click to expand "Universal Serial Bus Controllers".
c. Right click any device under "Universal Serial Bus Controllers" and click uninstall.
d. Similarly, also uninstall other devices.
e. Restart the computer.

All USB controllers will reinstall automatically.

I hope this information helps.

Please do let us know if you need any further assistance.

The new Microsoft Skype for Windows 10 (Skype UWP) application does not have a build-in chat export facility. but the entire chat history is saved locally on your computer and can be accessed using 3rd party applications..

 

In Skype UWP, the conversation history is saved locally in the skype.db database file, which is located in this folder:

 

%localappdata%\Packages\Microsoft.SkypeApp_kzf8qxf38zg5c\LocalState\<Skype user name>

 

skype.db is a SQLite 3 database file and can be accessed using a SQLite browser application.

 

Just one of many available SQLite browser applications is DB Browser for SQLite.

 

http://sqlitebrowser.org/

 

With such a SQLite browser, you can open the skype.db file and export the content of the messages table to a text file.
