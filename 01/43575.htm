    How to upgrade the hard disk? - Microsoft Community

    zfyytcx say:

    How to upgrade the hard disk?

    How to upgrade a Hard Drive? I have a HP Compaq 6715b and I am replacing 65GB to 250GB.  How should I proceed?  Thank You.

    a. user say:

    NOTE: These procedures only apply if you have Windows Vista - they will not work with XP.

    Install the new hard drive in the system.  Here are instructions on what to do: http://www.helpwithpcs.com/upgrading/install-hard-drive.htm.  Format the drive from your current operating system and create a partition (one will do but you can make several if you wish - just make sure you leave enough space in the primary partition to accommodate your complete restore and some additional space for further updates and more programs - you don't need to make any additional partitions at all (I don't) - one will do just fine. If you have a recovery partition and want to save it, create a second partition of equal size and you can copy and paste the files over just before you format the old drive. To format Go to Start / Accessories / Command prompt and right click on command prompt and click on run as administrator.  Then type in fomat (Drive letter) and enter. To partition go to Start / Control Panel /Administrative Tools/ Computer management / Disk Management and right click on the formatted drive. I'm assuming you have better than Vista Basic or Home Premium (you didn't mention your version - this process won't work on those versions because they can't do complete image backups.  If you do, skip to the next paragraph now). Since it's so much bigger, do a complete image backup from the old drive to the new drive.  Go to Start / Control Panel / Backup and Restore and select a complete system backup selecting the new drive as the target. Then do a restore of the image selecting the new hard drive as both the source and the target.  Once the restore is complete you can delete the restore file/folder from the new drive.  Then it's just a matter of setting the new drive as the master and the old drive as slave.  See the following for instructions: http://www.ehow.com/how_6031_change-master-slave-designation.html but it's best if you check with your hard drive manufacturers and the documentation that came with them (since they're all different when it comes to changing these settings). When that's done, gp to Start / Control panel / Administrative Tools / Computer management/ Disk management and right click on the new drive to make it active (if it isn't already so marked and make sure it says System and Boot - though it may not do that until you complete the dual boot procedure which follows). You then want to set it up as a dual boot system.  There are no printed guidelines for dual boot with both being Vista but here are the ones for starting with Vista and dual booting to Windows7 (just substitute Vista information where it calls for Windows7 information): http://apcmag.com/how_to_dual_boot_windows_vista_and_windows_7_vista_installed_first.htm. Once done (and it is a long and complicated process and you will need to download other software to help), the system will boot to the new drive (you may have to initiate the boot yourself and you will have to select the boot option when booting) and then you can format the old drive from the new drive for use as additional disk space.  You won't need an external hard drive to run backups and you won't need to re-install all your programs (or do any further restores from backup) - you'll be right where you were before with maybe one or two additional steps required (and this will be a lot cleaner than doing a clean install and running Easy Transfer (which is another option if you want and which I explain below) because this saves your programs as well as your data and settings.  Go to Start / Control Panel / Administrative Tools / Computer management / Disk Management and relabel the drives if necessary so that the new drive is called C: and the old drive is called D: (you may have to change D: to E: first and then change C: to D: and then change E: to C: - to change the drive letter just right click on it and select another letter) - all assuming that it didn't automatically relabel the new drive as C: when you changed it to master - I'm not sure if it would do that but just in case, you can do so yourself (you'll be able to tell which is which by looking at the size of the drives).  When you're done you'll need to go to Start / Computer / System Properties and at the bottom you'll need to check if the system is still activated (because you changed the hard drive).  If it requires activation, select the option to activate and run through the wizard.  If you encounter problems, continue through the wizard until you reach a screen with a phone number and a bunch of codes at the bottom.  Call the number and wade through the automatic prompts until you reach a human.  Explain that you just replaced your hard drive and need to reactivate.  You'll need to have your 25 digit product code handy when you make the call.  She can give you a code to enter on the screen which will reactivate Windows and then it's done.  Make sure you set up a backup process and schedule it to suit your needs - and it should be to a different drive or CD/RW in case the hard drive fails.  If the system is still in dual boot mode, use the relevant sections of the dual boot procedures to remove the obsolete boot option.

    Just so you know, another option is to install the hard drive and format and partition it as above.  Then backup all your data to the new hard drive.  Then install Vista on the new hard drive (I assume you have a full retail version and not an upgrade).  Then exchange master and slave as indicated above (you'll also need to make the new drive active and the system and boot drive - see the relevant parts of the dual boot procedure and the information in the prior paragraph) and reboot.  Then do an Easy Transfer of information from the old computer to the new computer. Here's the procedure http://www.microsoft.com/windows/windows-vista/get/easy-transfer.aspx.  You'll need to install a download from the site to the old drive to make the process work - but it tells you how to do that.  Remember, this process will require you to re-install all your programs.  Then follow the drive relabeling and activation procedures in the prior process.  And once again you're done (except for re-loading all your programs which you should do your data in your profile and then delete the backup you did to start this process (but you should still do a backup to a different drive or to a CD/RW and schedule it to occur periodically (as you see the need).  If the system is still in dual boot, use the relevant sections of the dual boot procedure to delete the old boot option.

    It's hard to cover everything in one post here (it's not a simple procedure), but hopefullly the linked references will fill in the blanks where the explanations are not fully described. Post back if you have any questions.

    I hope this helps.  Good luck!
