Hi RamanBut,

The error code 1603 means you're trying to install an application that is already installed on your computer or the requirements of the application is not met on the computer. With that being said, we suggest that you try the following methods:

Method 1: Check if the app is already installed on the computer. If so, uninstall and reinstall the application. 


If you previously had a desktop shortcut for an application, the shortcut may have been lost during the upgrade to Windows 10. In such cases, the application is likely still installed on the computer, resulting in this error when you attempt to reinstall that said application. You can restore the shortcut by searching for the application, and if it's found, press and hold (or right-click) the application and select Pin to Start. Or you may try uninstalling and then reinstalling the application. To search for and uninstall applications in Windows 10:

  1. On the Start menu, select Settings.
  2. In Settings, select System > Apps & features.
  3. If the app is listed then this is, select it and then select Uninstall.
  4. Follow the directions on the screen.

Method 2:  Check the website of the manufacturer/developer of the said application you're trying to install. See if your device meets the requirements needed to be able to install it on your computer.

Let us know how it goes and feel free to get back to us if you need further assistance.
