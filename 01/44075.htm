g.paz say:
Hello sbaviera
Thank you for participating in the Microsoft Community, my name is G.Paz, I am an Independent Advisor.
You can make the transfer of the license having the 25 digits of the activation key provided that your license is retail.

So the first step is to check what type of license you have, for this open a cmd as administrator and then you will execute: slmgr / dli

If in the answer you get something like this: Description Windows (R), Operating system, RETAIL channel, your license is transferable to a new PC.
However, if you get this answer: Description Windows (R), Operating system, OEM_DM channel your license is the one that appears by default when you buy a machine and in that case it is not transferred.

In the case that you have a retail license, you must unlink the license of the machine that you are not going to use at this time, for this: open a Command Prompt as administrator and execute these two commands:

slmgr / upk

slgmr / cpky
The first uninstalls the license, and the second deletes it from the registry. After restarting the computer, Windows will not have a product key and will be in an inactive and unlicensed state.

Then on the new computer once you have installed Windows, to activate it:
Open a system symbol and execute: slmgr / ipk [product key]

and then: slmgr / ato

Finally, link your new license with your Microsoft account.

Let me know if that work for you.

Windows 10 will use the “digital license” (digital entitlement) activation method, if one of the following is true:

You’re upgrading a genuine copy of Windows 7 or Windows 8.1 to Windows 10 for free.
You’ve purchased a copy of Windows 10 in the Windows Store and successfully activated Windows 10.
You’ve purchased a Windows 10 Pro upgrade in the Windows Store and successfully activated Windows 10.
You’re part of the Windows Insider Program and you have upgraded your genuine activated copy Windows 7 or Windows 8.1 to the latest Windows 10 Insider Preview build.

Any of this match for you?

Otherwise, to find a Windows 10 key license, you can go to regedit on the following route: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ and look for "DigitalProductId"
