kari finn say:

A few facts which you can easily check by yourself:

 

Moving your Windows Store Apps to another partition or drive will not help you very much. I just checked my test system with 35 apps installed, everything from games to online TV apps to useless gadgets, total storage space taken on C:\Program Files\WindowsApps only 2.6GB.

 

Moving your desktop apps and programs will not help you very much either. Check yourself how little space modern correctly coded software takes. This test system has Office 2010 and Office 2013, both full install, both Professional versions, 6 different media players, Webmatrix and Visual Studio for coding, dozen or so games etc. etc., total storage space taken on HDD is only 9GB (C:\Program Files 3.5GB and  C:\Program Files (x86) 5.5GB).

 

This test system of mine as shown above, if absolutely every application, desktop and Windows Store alike could be completely moved to another disk it would free 11.6GB of storage space.

 

What could instead really help you is to relocate / move your user folders or with sysprep move the complete Users main folder with everything on it, including AppData. This is done by changing the values of a Windows Environment Variable and is easy and fast procedure which puts all existing and future user folders to another location (http://www.eightforums.com/tutorials/4275-user-profiles-relocate-another-partition-disk.html). The Users folder is normally much larger than combined app and program folders.

 

In a normal computer the issue is not the space apps and programs take. Check yourself if you haven't done it already and you will be surprised how little space your Apps and Programs folders need.

 

Kari
