a. user say:

Hi Ken,

Home group listener is service used by the home group feature. If you haven’t configured a home group, then you can safely disable this service. Refer to these steps:

  1. Press Windows key + R, type ‘services.msc’.
  2. In the services window, locate the ‘Home group listener’ service.
  3. Right click on the service and select ‘Properties’.
  4. Set the startup type to ‘Disabled’ and then click ‘Apply’ to save changes.
