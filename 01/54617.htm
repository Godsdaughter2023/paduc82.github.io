bobby mikkelson say:

Hello Christi,

 

This issue happens because of Guide mixing problems during the configuration process. When manual configuration is used, you can get in a situation where Digital Antenna (ATSC) channels are actually tuning to Draco instead of using the ATSC tuner. This issue does not occur if you use automatic configuration for setting up the channels.

 

You can resolve this issue by editing the source of the individual channels and removing the Draco sources simply by unchecking the box beside those listings to remove them. You can also follow the instructions on Edit Sources screen to take other actions like reordering sources, remove a source and create a separate channel using split to correct the issue.
