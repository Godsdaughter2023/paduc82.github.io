    Can't restore from Windows 7 backup - Microsoft Community

    geoffrey bennett say:

    Can't restore from Windows 7 backup

    I created a backup with Windows 7 Backup. It is on a USB hard drive in the folder "WindowsImageBackup" and contains a folder with the name of my PC. It is 336 GB in size. I did that some time ago and want to restore from it now. After creating the backup I did a system restore. Unfortunately, now Windows 7 Restore can't find the backup, even though the drive is accessible. I have tried the "Select another backup to restore files from" option with no results.

    Is there a way to get access to the files in my backup?

    frederik long say:

    I suspect that you did not create a "backup" but a System Image file. If so then you can use this recipe.

    1. Use Windows Explorer to locate the .vhd (virtual hard disk) file on your USB disk. I will look like this:
      E:\WindowsImageBackup\PC1\Backup 2010-12-12 175726\0417acc3-2173-11df-9f99-806e6f6e6963.vhd
      Note the .vhd extension.
    2. Lauch the Disk Manager (diskmgmt.msc).
    3. Click Action, then Attach Virtual Hard Disk.
    4. Use the Browse button to locate the file you found in Step 1 above, then click OK.
    5. Note the new drive letter that Windows assigned. It could be G:. If no drive letter was assigned, right-click the new disk and give it a drive letter.
    6. Use Windows Explorer to open drive G: and retrieve your files.
    7. When finished, close Windows Explorer.
    8. Go back to the Disk Manager.
    9. Right-click the new disk on the left side of the panel where it says "Basic" and "Online".
    10. Click "Detach VHD".

    How to create new storage pool? - Microsoft Community

    jerome adrien say:

    How to create new storage pool?

    hi.
    i am running windows 8 and i wanted to create a new storage pool for my personal files. but how?

    i only have 500gb storage.

    current storage pools:

    OS (C:) - 460+gb

    lenovo (D:) - 20+gb

    thanks!

    a. user say:

    Hi Jerome,

    Yes, the statement means that you can use an external drive such as a hard drive or a USB drive as a storage drive. What you need to do is connect the drive, go to start screen, type storage, click on Manage Storage spaces. Click on Create a new pool and storage space and then follow the onscreen instructions (as given on the link provided on the previous response).

    Alternatively, you can also create a new volume partition on the hard drive. You have mentioned that the operating system drive is 460 GB and drive "D" takes 20 GB (or more). What we can do is create a new simple volume on the unallocated drive. If there is an unallocated disk space, we will create a new drive by taking the required amount of disk space from the unallocated disk space and then use the new drive as for your personal files.

    First of all, check if there is an unallocated disk/drive available under Disk Management. Follow the steps:

    • Press Windows+X keys and select Disk Management.
    • Check if there exists an unallocated drive with the drives. If you don't have an unallocated disk/drive, then right click on the drive (either the operating system or drive "D") and select Shrink Volume.
    • Now, you will find an unallocated drive in the lists. Right-click an unallocated region on your hard disk, and then click New Simple Volume.

    • In the New Simple Volume Wizard, click Next.

    • Type the size of the volume you want to create in megabytes (MB) or accept the maximum default size, and then click Next.

    • Accept the default drive letter or choose a different drive letter to identify the partition, and then click Next.

    • In the Format Partition dialog box, to format the volume with the default settings, click Next.

    • Review your choices, and then click Finish.

    • Now a new drive will be created and it will show up under File Explorer.

    Check if it resolves the issue.

    For reference:

    http://windows.microsoft.com/en-ca/windows/repartition-hard-disk#1TC=windows-7

    http://windows.microsoft.com/en-ca/windows/create-format-hard-disk-partition#create-format-hard-disk-partition=windows-7
