                        net start wuauserv

                        net start cryptSvc

                        net start bits

                        net start msiserver

Hi,

Thank you for the update and apologies for the inconvenience.

I would request you to create a new user account and check the status.

a. Press Windows key + R to open RUN box

b. Type control userpasswords2 and press OK

c. Under User Account window and under Users tab, click Add tab

d. Under How will this person sign-in windows, click the option at the bottom Sign in without a Microsoft account (not recommended)

e. On the next window click Local Account at the bottom

f. Choose a user name (use password if you want to or you can leave the password section blank as it is just to test)

g. Click Next and then click Finish

As you click Finish, the previous User Accounts window remains on the screen

i. Select newly created user and click Properties

j. Click Group Membership tab

k. Click to select Administrator and click Other and select Administrator from drop-down menu

l. Click Apply and then click OK

Hope this helps.
