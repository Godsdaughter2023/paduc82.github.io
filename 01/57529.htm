lemp say:

Read that sevenforums thread that you found carefully.  The likely answer is in post #6.

That is, Windows Backup does two different things.  It can make a "System Image" or it can back up individual files or folders.  By default, it does both.

Here's what the Help file says about "System Image" -- with a few important points emphasized

Windows Backup provides you with the ability to create a system image, which is an exact image of a drive. A system image includes Windows and your system settings, programs, and files. You can use a system image to restore the contents of your computer if your hard drive or computer ever stops working. When you restore your computer from a system image, it is a complete restoration; you can't choose individual items to restore, and all of your current programs, system settings, and files are replaced. Although this type of backup includes your personal files, we recommend that you back up your files regularly using Windows Backup so that you can restore individual files and folders as needed. When you set up scheduled file backup, you can choose whether you want to include a system image. This system image only includes the drives required for Windows to run. You can manually create a system image if you want to include additional data drives.

Thus, if your backup settings included a system image, that portion of the backup data would take up about the same amount of space as your existing system.  That is, your system image would have been about 200 GB.

But your backup settings probably also included some -- or perhaps all -- of your data files.  The extra backup data duplicates data included in the system image, but you can access individuals files. 

The screenshot below is an example of what I mean.  You can see that the system image is about 286.5 GB but there is an additional 68 GB of data files.  However, when I set up this backup, I picked the option to "let me choose" what files to back up -- and I only selected a few.  By default, Windows will protect a lot more (see the second screenshot below).

If you did a clean reinstall of Windows 7 and are now wanting to recover your files, you won't be (or shouldn't be) using the System Image (of course, this will only recover your data files and not your programs).  You should have gone to Backup and Restore,  clicked the "Restore my files" button and then selected the files to restore.  See below.  Is that what you did?  If so, I'm inclined to think that the error you got is not, in fact, related to "running out of space."  Do you get an error message when "the restore gets about 2/3rds done and conks out"?  If so, please provide the complete text.

The alternative would be to start all over and restore from your System Image -- and this is probably the path you should take.  It will restore your system, including both programs and data.  Here's how: 

http://windows.microsoft.com/en-us/windows/restore-computer-from-system-image-backup#1TC=windows-7

The error message that you get while trying to take a backup of your computer to an external hard disk are due to the following reasons:

 

A)      This issue occurs if you back up a library that contains custom folders that are located under your user profile.

B)       This issue occurs if the following conditions are true:

 

a)       Windows Backup tries to back up files in the LocalLow, Contacts, and Searches folders.

b)       These folders are not present on your computer.

 

C)       This issue occurs if the following conditions are true:

 

a)       The computer has an NVIDIA USB Enhanced Host Controller Interface (EHCI) chipset.

b)       The computer has at least 4 GB of RAM.

c)       You try to copy data from the computer to an external USB storage device.

 

D)      This issue may also occur when you are using USB devices rather than storage devices.

E)       This issue may also occur if you perform other input/output (I/O) operations on the USB storage device. In this situation, the I/O operations fail.

wunders say:

In most all cases, the "C" drive is your Windows System drive.  If you are booted to Windows, then you are using this drive.  If you want to reformat your System Drive, you have two options.  The first option is to look at the first pop-up screen and see if you can hit a key at this time to start a "System Recovery".  This will re-install Windows on your computer and put your computer into the as-purchased state.  As part of this process, there is usually an option to select a "quick" format of the hard drive or not.  If you do not choose the "quick" format, then your drive will be reformatted before re-installing Windows.

 

The second option is to boot your system from a Bootable CD such as a System Recovery CD or (my choice) a Hirens Boot CD:

  < http://www.hirensbootcd.org/download/ >

(the correct download link is a .zip file near the bottom)

Once you boot from a bootable CD, your C: drive will not be used and it may be formatted at this point.

Wireless Security - Microsoft Community

xb1r hydros say:

Wireless Security

Login Information

  • Gateway: 192.168.254.254
  • Username: admin
  • Password: admin

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.254.254 into the web address bar.
  4. This will take you to a login page, type admin for the “Password”.
  5. Once logged in select the General Settings on the left side.
  6. Select the WLAN Security option on the middle page.
  7. The wireless password will be shown near the bottom of the page.

Changing Wireless Security Type/ Disabling Wireless Security

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.254.254 into the web address bar.
  4. This will take you to a login page, type admin for the “Password”.
  5. Once logged in select the General Settings on the left side.
  6. Select the WLAN Security option on the middle page.
  7. On the main page you will find options for changing your security option or disabling wireless security.
  8. Once you have selected the desired choice click on Next.
  9. Select the Save option, this will reboot the device with your new configuration.
