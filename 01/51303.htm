bossdweeb say:

Win10 is much better at memory management than previous Windows, so the 128MB 'dedicated' is just a minimal amount of

system RAM set aside for graphics processing should you somehow use all of the rest of your system RAM with something else.

This does not prevent games from accessing more of the available system RAM. So even if there is a setting in BIOS

to boost the amount allocated to graphics (as used to be common), it's not really going to help gaming performance as much

as it may once have with earlier Windows. Which would not have been by more than a couple of extra fps anyway.

It's still only system RAM, not the faster VRAM (which is found only on dedicated graphics cards).

The only real fix for poor gaming performance, other than playing only older DX9 games, or modern games with basic effects,

is to have at least a mid-range graphics card, and a cpu with at least 3GHz clock speed (which yours just manages when a game

kicks it into 'boost' speed).

Unfortunately, All-In-One and laptop graphics cannot be upgraded. What you see is what you get.

- It can be tricky knowing what games will run on integrated graphics as they rarely include integrated graphics is

games system requirements.

- If you have not seen any benchmarks for the Intel HD 630 check the review below. They benchmark a good selection

of games, you can use to extrapolate from when you compare the system requirements of those games with

games you are interested in.

https://www.notebookcheck.net/Intel-HD-Graphics-630.187948.0.html

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
