http://support.microsoft.com/kb/910339

Hide Windows 7 Updates that you do not Ever Want to Install

http://technet.microsoft.com/en-us/magazine/ff382716.aspx

My recommendation would be to install the updates in batches, such 5 at a time, exclude hardware/driver updates. Avoid installing a combination of Microsoft Windows and Office updates at the same time. Also, try installing the small updates first. Avoid installing things like .NET Framework and SQL Express Updates.

Restart each time you install the updates (its a chore, but it guarantees a more reliable experience). Try starting out with Microsoft Office Updates for programs such as (Word, Excel, Outlook, PowerPoint).

How do I reset Windows Update components

<http://support.microsoft.com/kb/971058>

Suggest you download and save the Fixit. Then configure the system to Clean boot prior to running it:

How to troubleshoot a problem by performing a clean boot in Windows Vista/Windows 7

<http://support.microsoft.com/kb/929135>

I suggest before shutting down and installing any updates, disconnect from the Internet first.

canadian tech say:

I understand about the drive.  WD calls them re-certified, not refurbished.  Those are really drives who's package was opened and they took them back.

Windows Update (the program in Windows) is a backup program.  Windows Backup backs up your DATA.  It does not backup programs -- which is a waste of time because in a re-installation, you have to re-install programs anyway.  I do not recommend ANY backup program.  Here is how I recommend backups be done, as I advise my clients:

  1. Connect your backup drive
  2. Create a folder named myname&today'sdate
  3. Open a Window with that folder
  4. Open another window with the User folder on your hard drive
  5. Drag and drop (with the right button) folders to the backup folder
  6. When you drop the folder, select Copy from the menu
  7. Do this for Documents, Pictures, Videos, and Music folders.  You will seldom find anything of yours in other folders.

When backups are done in this way, any file is retrievable any time.

When doing a rebuild:

  1. Type Windows Easy Transfer into the text box that pops up when you click the start button (bottom left) -- this is on the computer BEFORE the re-build.
  2. Connect a backup drive
  3. Create a folder and name it Easy transfer&today'sdate
  4. Follow the prompts. This is the Old computer
  5. This produces a very large single file that can be used for nothing else, but 99% of the time works great.
  6. When you have your newly re-built system in place, after Windows is completely installed and updated, and applications are installed, repeat the same process for Windows Easy transfer choosing "this is the new computer."

Windows update is actually quite a mess.  The quality of what they put out as updates has suffered greatly since they fired their quality control people.  A lot of people don't do Windows update at all, most of the wise ones, have Windows Update set to NEVER..  and do updates when they decide to do it, which is usually 29 day after they are issued.  http://www.askwoody.com is an good source of advice about what and when to update.

It would not surprise me in the least,  if one of those updates messed up Windows backup.

Actually the age of your laptop is significant.   I have been helping about 150 clients with their windows computers for the last 16 years.  About 50% of laptops are no longer in use after 5 years.  If yours is say 8 years old, it is statistically likely on its last legs.

a. user say:

usstalbot,
When you receive an error message, it is absolutely essential to post the ENTIRE message VERBATIM. Paraphrasing it wouldn't help.
Meanwhile, please confirm whether you have step #5 checked and #6 Apply clicked ....
  1. When you are returned to the Internet Accounts dialog box, click the Mail tab. (If the Internet Accounts dialog box is not open, click the Tools menu and then click Accounts).
  2. Select your AOL IMAP account.
  3. Click Properties.
  4. In the Properties dialog box, click the Servers tab.
  5. In the Outgoing Mail Server section, click the My server requires authentication check box.
  6. Click Apply.

If you have done that and it does not work, please post the error message and let Gloria take a look at it.
