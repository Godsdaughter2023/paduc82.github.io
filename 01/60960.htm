1. Does it happen only with a particular child account?
2. Have you tried to set time for another child account?
3. What is the make and model of your computer?
 

I would suggest you to remove that local account and check.

Follow the steps.

Firstly, to delete a user account you should be logged in as an administrator.

Check if the Microsoft user account which you have signed in an Administrator account. Follow the below steps.

  1. Press “Windows key + X” together from the keyboard.
  2. Click on Control Panel.
  3. Click on User Accounts and Family Safety.
  4. Click on User accounts.
  5. Click on the Microsoft account and check if it is Administrator.

If the Microsoft account is an Administrator then you can delete the account by following the below steps.

When you delete a user account, you can choose whether you want to keep the files created under that account, however e-mail messages and computer settings for the account will be deleted.

  1. Press “Windows key + X” together on the keyboard.
  2. Click on Control Panel.
  3. Click on User Accounts and Family Safety.
  4. Click on User Accounts.
  5. Click  on Manage another account. (If you're prompted for an administrator password or confirmation, type the password or provide confirmation.)
  6. Click the account you want to delete, and then click Delete the account.
  7. Decide if you want to keep or delete the files created under the account by clicking Keep Files or Delete Files.
  8. Click  on Delete Account.
 
If it doesn't work I would suggest you to create new child account then try to set to use the PC during the time range.
 
To create new child account please follow the below link:
 
http://windows.microsoft.com/en-us/windows/set-up-family#set-up-family=windows-8&set-up-family-safety=windows-8
Note:  It is also applicable for windows 10 as well.
 
To Allow User to Only Use the PC During the Time Ranges follow the below steps.
 
1. Go to Control Panel then Family safety .
2. Under Family safety select User settings then Time Limits then Curfew.
 
3. Select "When can Standard-example use the Pc under this
select Standard-example can only use the pc during the time ranges. 
 
 
4. To Block hours click/tap on a white box, or click/tap and hold and drag over a group of white boxes and release, to mark it blue for each half hour of each day that you want to block.
 
5. To unblock hours click/tap on a blue box, or click/tap and hold and drag over a group of blue boxes and release, to mark it white for each half hour of each day that you want to unblock.
 
 
I hope this helps.
Let us know the status of the issue so that we can assist you better.

Thank you for posting your query in Microsoft Community.

This issue may arise due to driver conflicts. We will assist you to resolve the issue.

I suggest you to run the Hardware and Devices troubleshooter and check if it helps.

  1. Type troubleshooting in the search bar.

  2. Select Troubleshooting.

  3. Select View all on the top left corner.

  4. Click on Hardware and Devices.

  5. Follow the on-screen instructions to run the troubleshooter.

  6. Check if issue persists.

    If the issue persists, I suggest you to uninstall and reinstall drivers and check if it helps.

    Steps to uninstall and reinstall driver.

  1. Press “Windows key + X” and click on “Device Manager”.
  2. In the list of devices, you should see the bluetooth drivers.
  3. Right click on the device and then select "Uninstall".
  4. In the uninstall dialog, make sure you select the option to delete the driver package from the system. This causes the driver package installed earlier to be deleted.

If there is no updated drivers available on the computer manufacturer’s website, then download the latest drivers and try installing it in Windows 8 compatibility mode. Compatibility mode runs the program using settings from a previous version of Windows.

Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 8 operating system from the drop down menu and proceed with the installation. Once this is done, restart the computer and check if the issue persists.
