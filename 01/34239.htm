[quote user="Lostofview"]erst jetz seit Ende Janunar wird der Gold Service getrennt[/quote]

Du wirfst hier zwei verschiedene Sachen durcheinander. Der "Gold" Service wird nicht getrennt. Die Goldmitgliedschaft ist technisch gesehen nur ein Datenbank Eintrag der besagt, wie lange Du für Deine Goldmitgliedschaft bezahlt hast oder Prepaid Karten eingelöst hast.

Der kann nicht "getrennt" werden, und der Status Deiner Mitgliedschaft ist immer prüfbar, wenn Du Dich hier auf der Seite anmeldest, und oben rechts bei deinem Gamertag klickst und im Dropdown Menü auf Abonnements gehst.

Wie die anderen bereits geschrieben haben, kann es sein, dass ein allgemeines Problem bei Xbox Live vorgelegen hat, und Du komplett von dem Live Service getrennt worden bist. Das betrifft aber dann jeden bei Xbox Live. Egal ob zahlendes Gold Mitglied oder kostenloses Silber-Mitglied.

Wenn Dir im Netzwerk der Begriff "OSI Modell" nichts sagt, lass uns eine Auto-Analogie machen.

Stell Dir vor, das Internet ist das Straßennetz. Diese Straßen können von mehreren Fahrzeugen (Diensten) verwendet werden. YouTube wäre ein Lastwagen, Twitch ein Omnibus, Xbox Live ein Auto, E-Mail ein Fahrrad etc.

Xbox Live Gold ist ein Zusatzfeature zu Xbox Live, was Dir Multiplayer-Spiele und Partychat mit anderen Goldmitgliedern erlaubt. Also ein kostenpflichtiges Sonderzubehör in Deinem Auto. Sagen wir mal ein eingebautes Mobiltelefon mit einer extra eingebauten Mobilfunk-SIM in Deinem Auto.

Jetzt zu Deinem Problem mit der "ablaufenden" Goldmitgliedschaft: Microsoft weiß, dass Menschen vergeßlich sind. Und da Mitgliedschaften auch mal ein Jahr laufen, könnte jemand vergessen haben, dass sein Gold demnächst abläuft. Also bekommt jeder einen Monat vor Ablauf einer Mitgliedschaft eine Mail, dass der Dienst demnächst abläuft.

So hast Du einen ganzen Monat Zeit, Dir Gedanken zu machen, ob oder wie Du Deine Mitgliedschaft verlängerst. Ansonsten kämen die ganzen Leute mit verheulten Augen hier an und jammern: "Heute ist meine Goldmitgliedschaft abgelaufen, aber heute abend ist so ein ultrawichtiges Clan Match, hat einer DRINGEND einen 48 Stunden Goldcode? Meine Eltern wollen mich nicht zur Tankstelle fahren um eine Goldkarte zu kaufen!"

Du wirst ja nicht GEZWUNGEN zu verlängern. Du wirst NUR dran ERINNERT, dass DEMNÄCHST Dein Gold abläuft, und Du nicht vergisst es zu verlängern.

Zurück zur Auto Analogie: Du hörst auf Deine Gebühren für die SIM Karte zu bezahlen. Das bedeutet "nur", dass Du nicht mehr im Auto mit Deinen Freunden via Telefon kommunizieren kannst. Das Auto funktioniert aber weiterhin, und Du kannst Xbox Live für Marktplatzdownloads und andere Live Dienste verwenden.

[quote user="Lostofview"] seit Ende Janunar wird der ... Service getrennt ( erst dachte ich naja vielleicht mein Internet [/quote]

Wenn der Service getrennt wird, und es liegt (wie die anderen vermutet haben) eine Xbox Live Dienststörung vor, ist das Xbox Live Auto eben kaputt, und Du kannst damit nicht fahren. Damit ist auch die Telefonverbindung im Auto kaputt, und Du kannst nicht mit Deinen Freunden online Multiplayer spielen oder Dich mit ihnen unterhalten.

Das hindert Dich aber nicht daran, auf dem sonst funktionierenden Straßennetz mit dem Autobus, dem Fahrrad oder dem YouTube Lastwagen zu fahren.

Wenn Dein Internet komplett gestört ist (Unfall direkt vor der Garageneinfahrt Deines Hauses. Öl-Laster umgekippt und brennt lichterloh) bleibt Dir nichts anderes, als zuhause zu bleiben und ein gutes Buch zu lesen. Du kommst nicht mit dem Fahrrad am brennenden Inferno vorbei, und Busse halten nicht in Deiner Nähe, weil die Straße gesperrt ist.

gerry c j cornell say:

Error  0x80246008 can mean that the BITS service has been disabled.

Click on the Start button and select Control Panel, Administrative Tools, Services. Locate Background Intelligent Transfer Service in the list and check it has started. If it has not started check that the StartUp type is Automatic. If it is not set to Automatic right click and select Properties, General tab. There is a box after StartUp type. The options are available on a drop down by clicking on the arrow down to the right of the box. Make the change and click on Apply and OK. Restart the computer and check whether it has started.

Here are some notes that may help resolve problems encountered when installing Windows Updates.

The process of installing updates comprises downloading, installing and configuring the operating system to use the new updates. Some updates require the computer to be restarted and when this occurs configuring occurs as the system is shut down and is completed when the computer is restarted. It is important not to interrupt the configuration process as configuration involves making changes to the registry. If interrupted, the result can be a computer that will not boot.

A lot of problems arise from setting Windows Update to automatically update. You may prefer to change this setting. Select Start, All Programs, Windows Update, Change Settings. Under Important Updates is a box showing the current setting. Click on the Arrow down to the right and change the selection to “Check for Updates but let me choose whether to download and install them” and click on OK.

If you encounter problems updating or you are not using the Automatic Update you should manually update by selecting Start, All Programs, Windows Update, Check for Updates. Check the boxes before the updates you want to install, click on Install and click again on Install in the next window and the process of installing starts. You should let the process proceed without interruption. Multi-tasking at the time updating is in progress is unwise. The progress window will tell you what is going on. You need to be patient.

If you have a number of updates to install break the task of updating into two or more tasks. If you have been having problems with Windows Update run Disk CleanUp (or better still cCleaner) and Disk Defragmenter ( or better still Defraggler) before installing any large download.

If the measures above fail to resolve your problem select Start, All Programs, View Update History and post the Failure Code and Update reference number to the Windows Update Forum at Microsoft Answers.
