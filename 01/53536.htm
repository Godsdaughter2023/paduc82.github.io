thriftylemon72 say: If you are 100% sure your files just need to have the .dotmap extension removed, and you're comfortable with scripting, this powershell script will work. If you aren't comfortable with scripting, you may want to contact a PC repair shop and have an expert perform a repair for you.

Right click on the start menu and click Windows Powershell to open powershell.

To navigate your folder structure, you can use cd (which stands for Change Directory), and type ../ to go up a level, or type the name of a folder to go down a level.

For example, to navigate from

C:\ to C:\Users

you would navigate to the C directory (cd C:\) and type cd Users.


Anyways, navigate to a directory where you can safely test this script (preferably a folder without any important or irreplaceable documents) and type the following into Powershell:

dir *.dotmap | rename-item -newname { $_.name -replace ".dotmap","" }

What this script does:

The dir command returns a list of all items in the directory, and *.dotmap means only select files whose name ends with .dotmap
The pipe character ( | ) separates the two halves of the statement, and instructs powershell to only operate on the items returned by the first half.
The rename-item command does exactly what it sounds like, and uses a parameter called -newname.
Inside the { curly braces }, $_ targets each item from the left side of the | pipe ending in .dotmap, and the .name part tells powershell to update the name property.
-replace tells Powershell to replace the matching left side of the arguments (".dotmap") with the right side ("") - basically, anywhere it finds the characters ".dotmap" in the filename, it will replace it with an empty string of letters.

I tested this in a directory by creating an empty text file and copy/pasting it 10 times - when I ran dir *.txt | rename-item -newname { $_.name -replace ".txt".".txt2" } I ended up with 10 files with a .txt2 file extension.

I would strongly suggest testing it the same way before actually executing it against your files, to make sure the results are what you expect.
