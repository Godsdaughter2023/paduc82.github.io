shenan stanley say:

2d - part iii: means you can share resources (like shared folders, shared printers, etc.) with up to 20 other like devices.  Windows 10 is NOT a server OS.  You can also host a website, but the limit on the number of connections to it is 20.

2d - part v: seems to put a limit on who can use remote desktop to connect to the system, essentially saying it is only the owner of the device.  Although, honestly, prove it.

(This is all for the retail versions; people running a business should DEEPLY consider enterprise.)

2a: simply points out that two users cannot simultaneously utilize the OS at the same time.  Not too big of a worry, again, this is not a server OS.

If you are installing Windows 10 Pro or Home as a server: you're breaking the agreements.
