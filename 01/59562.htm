What error reports in Event Viewer?

Here are some notes about Event Viewer Reports which may help. When you have a lot of errors you need to concentrate on system errors and warnings, even if it is applications that are giving you grief. Sorting system problems first can make resolving application problems easier. Note the time the computer is last booted and deal with those at the beginning of the boot first. Correcting the earlier errors can resolve later ones.

1. Normally when an error occurs on your computer looking in Event Viewer should be your starting point for finding a solution. Most system related errors are logged and getting an exact copy of the relevant report is important. Unfortunately understanding the reports is not easy and most computer users need help with their interpretation. I have more to say later on interpretation.

2. Event Viewer comprises three main Windows logs. These are Application, Security and System. For troubleshooting purposes System is by far the most important.

3. To access the System log select Start, Control Panel, Administrative Tools, Event Viewer, from the list in the left side of the window select Windows Logs and System. Place the cursor on System, right click and select Filter Current Log. Check the box before Error and click on OK and you see only Error reports. Click on the Date and Time Column Header to sort. You may need to click a second time to see the latest Report at the top.

4. A tip for posting copies of Error Reports! Run Event Viewer and double click on the error you want to copy. Click on the Copy button on the General tab to place a copy on your Clipboard and close Event Viewer. Now start your message and paste into the body of the message. Make sure this is the first paste after exiting from Event Viewer.

5. There are three types of Report, being Information, Warning and Error reports. In most situations it is Error Reports that offer the best information but occasionally Warning Reports provide useful clues.

6. All reports have date and time stamps and when troubleshooting it is important to concentrate on more recent reports. Study reports since the point when the computer was last booted and then check whether a similar report appeared in the previous session. If errors do not repeat investigation as to why they occurred is wasted effort.

7. Within individual reports the more important information is Event ID and Source as these help when looking for help on the internet. The description is equally important and copying the exact text for use as the search criteria greatly helps getting better results when using Google. Do not paraphrase descriptions when asking others for help.

tryx3 say:

1  It might be a longshot but can you boot into Windows normally still so that you can run sfc?   The System File Checker can fix corrupt files in Win7
http://support.microsoft.com/kb/929833There is additional guidance on it in sfc.exe /scannow  Perhaps this will fix some element of Windows startup that is preventing the System Recovery Options window appearing.  I suggest this because [from memory only] I think Windows has not handed over to Dell by the stage of recovery when your system failed.  By the way, the F8 key does need to be pressed repeatedly so there was nothing going wrong at that earlier stage.

2  Did you make the recommended system repair disk & set of recovery disks when you got the PC [or after your first factory restoration]?  These can also be used to return the system to [almost] its original factory condition.

3  Did you make any other system images soon after getting the PC / Factory Restoring?

4  Did you use Dell DataSafe Local Backup [or its equivalent on a Studio 1558] to make a backup USB/DVD-R of the Factory Restore partition?

5 Does Dell Support Center or its associated PC Checkup application report any specific errors that you might try fixing?

6  Do you know anybody else who has a Studio 1558 of the same age [hence same configuration] with the same operating system as you?

7  Have you tried putting in the Win7 installation DVD then booting using it to do a general Windows repair?  And then retrying the Factory restoration.

8  If not, and I suppose that's the case or you would not have posted, then you will have lost the Factory restore facility.  I have also lost it but it is not the end of the world.

9  You are correct in thinking that you can use your Win7 installation disc.  You can then retry the Factory Restore just in case it was a Windows glitch & not a Dell one.  You will have to be careful during your Windows installation to let Windows install to its original partition and not to overwrite the Recovery partition created by Dell - it might wreck this anyway but it's worth trying to avoid doing so.  Before starting any reinstallation have a look at the partiitons in Computer management, Disk management so you can see what is what - you'll see the main C:\ partition & the 8 or 16GB Recovery partition put in by Dell as well as another very small one [just 100MB or so]

10  When you have restored the system using the Windows installation disk, you can then go to the Dell support website to get drivers & downloads [the link is the UK one but will at least illustrate what you will find] that will include all drivesr & some of the utilities you might otherwise have lost.  The original versions of these will be on the Dell Utilities CD that you will have received along with your PC - it's worth putting this in the PC & getting Windows to look at it when updating drivers but you'll also be checking for updates on the Dell website anyway so this CD is just a bit of a shortcut to the original versions.

11  You may have already created a Dell account.  If not then do so because this will also give you the chance of downloading a fresh copy of some Dell utilities.  In my case, this allowed me to download then ignore the Roxio Burn utility.  I don't know how to set this up as I must have done it when the PC first arrived but I have just found a card directing me to http://downloadstore.dell.com/media to set up "My Dell Downloads" which sounds like the right place.

12  All in all, the only utility I lost was Dell DataSafe Local Backup.  The only unique function of this is to create the USB/DVD-R I mentioned above.  So not a great loss really.

Let me know if there's anything else I can help with.  And check in Dell forums
