techie007l say:

    The Startup type option is grayed out because Microsoft wants to keep people from disabling the Windows Installer service.  However, you can still manually disable that service via the registry:

  1. Click Start, type "regedit" and press [Enter].
  2. Navigate (by double-clicking) to HKEY_LOCAL_MACHINE -> SYSTEM -> CurrentControlSet -> Services -> msiserver.
  3. In the right-hand pane, change the Start value from 3 (the default) to 4 (disabled).  To undo, change it back to a value of 3.
  4. If you reopen the Windows Installer service's properties, you should see that is now disabled.
