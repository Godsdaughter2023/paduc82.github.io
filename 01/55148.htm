Check for a newer Display driver on the PC or Display adapter Support Downloads webpage. wsrmarkoe say:

The jumping curse and is often caused by a microphone that is either too sensitive, picking up background noise, or the microphone element at the end of the bone of a headset is positioned incorrectly.

First thing you should do is run the microphone setup from control panel, speech recognition. If you are using a microphone built into the computer or a web cam these often will pick up background noise and the environment. For best accuracy a USB headset will work to optimally for WSR, Windows Speech Recognition. If you are using a USB headset microphone, make sure the microphone element at the end of the boom is at the corner of your mouth not in front. This is because if it is in front of the mouth it will pick up inhales and exhales as sounds to be interpreted as words or commands. Finally, if you're using a USB or a microphone that connects to the sound card, try a different input port.

Marty Markoe, MyMSSpeech.com
