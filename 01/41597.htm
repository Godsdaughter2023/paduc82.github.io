Hi -mehwish-,

You’ll find unknown devices and other non-functioning devices under Other devices. Unknown devices show up in the Device Manager when Windows can’t identify a piece of hardware and provide a driver for it. An unknown device is not functioning until you install the right driver for it. In order to find the right driver, you need to find the device's hardware ID to identify the device.

Here's how:

  1. Right-click the unknown device and select Properties to view more information.
  2. Click the Details tab, click the Property box, and select Hardware Ids in the list.

Perform a web search for the hardware ID. You should find the name of the piece of hardware associated with the unknown device, and that will give you the information you need to download and install the appropriate driver.

You need to be logged on as an Adminitrator
