If you want to install Windows 7 or 8 on a larger than 2TB boot drive. - Microsoft Community

airyu say:

If you want to install Windows 7 or 8 on a larger than 2TB boot drive.

If you want to install Windows 7 or 8 on a larger than 2TB boot drive.

 

You need to have a motherboard whose BIOS has UEFI support, a bootable CD-ROM/DVD-ROM drive and your Windows 7 install disc.

 

You can check your PC for support by putting the Win7 disc into the drive and booting that. Before the disc boots, enter the BIOS setup screen. Navigate to the "Boot Menu" or look for the Boot Drive Icon. If the icons (some newer BIOS are semi-graphical) show 2 CDs and one has UEFI on it, you're in. If it has text and the two CDs are listed with one that has the letters UEI or UEFI in the title, you're in.

 

Make the UEFI CD the top priority for booting and reboot.

 

If the BIOS or the install disc do not have UEFI support, you will need to use a third party UEFI suite.

 

The Win7 boot screen that comes up first is the language/time zone screen. Fill this out.

 

When the "INSTALL NOW" screen comes up, either click on the "repair" button or hit alt-F10 to enter the tools menu. At the bottom of the menu is "DOS Prompt" or "Command Prompt". Click that.

 

Next, start "DISKPART" (case of letters is unimportant - quotes and caps here for clarity)

 

Type "LIST DISK" and note the numbers that list the HDDs.

 

For each larger than 2TB drive you have, do the following three lines substituting the number for the "#" symbol.

 

"SELECT #"

"CLEAN"

"CONVERT GPT"

 

When this is complete, each drive larger than 2TB will be GPT. Smaller than 2TB can be left MBR. Windows does not care.

 

When finished, type "EXIT" and close the DOS/Command Prompt window. Restart and when the installation reaches the "Install Now" screen again, click that.

 

Windows Installer will use the UEFI info from the UEFI folder on the CD/DVD and show the choice of which drive to install to with every drive having one partition at the full size of the disc.

 

If it does not, or if you still find the drive showing up as a 2TB drive with the remainder shown as "Unallocated" under Drive Manager, your BIOS does not actually support UEFI and you will need third party software.

 

Good luck.

 

If you know 3rd party software that can allow Win XP and non-UEFI motherboards to operate with GPT drives, please feel free to add that here.
