This isn't related to OneDrive or any online service. It's a backup system that requires a local external drive or shared folder on your network to send backups to.  It's Microsoft's version of what Apple trademarks as "Time Machine" drive, a similar tool.  File History is a local backup tool that does what Windows Server used to call shadow copies or versioned backups. These are all similar tools architected different ways, but File History is what it's called in Windows 8 onward and what the message you see is related to.

Choose "Save backup copies of your files with File History"

Click "Turn Off"
