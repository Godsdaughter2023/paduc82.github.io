If you click the + sign in front of it, the rest of the sentence is revealed.  This reads something like ' Turn off hard disk after on battery: 10 minutes'. The '10 minutes' is in blue. If you hover your mouse over it, the following sentence pops up:

So, you could have the CPU in use but if the disk was not being used for 10 minutes, it would be powered down and would need to be spun up again when needed. This is to save electricity and extend the time you can run on battery.

1. Like Shawn "Cmdr" Keene stated, I unchecked 'Allow Devices to Wake the Computer' everywhere in the Device Manager, in the Network Adapters, the Mouse as well the Keyboard.

c) Click on the Mouse and Power Management, then uncheck 'Allow Devices to Wake the Computer'

d) Do the same for Network Adapters and the Keyboard if possible.

2. Edit the Power Options, this controls the computer's ability to wake and sleep. 

a) Start Button

b)Type in 'Power Options'

c) Make sure it is on 'Balanced' and then next to the 'Balanced' click on 'Change Plan Settings'

d) You can either try clicking on 'Restore default settings for this plan' as this is how the settings where when the computer was in factory condition

e) Or you can click on 'Change advanced power settings' 

f) Go on the 'Sleep' and make sure it is set to 'Sleep'

g) Go on 'Power buttons and sleep' and make sure it is set to 'Sleep'

h) Go on 'Multimedia settings', 'When sharing media' and make sure it is set on 'Allow the computer to sleep'

i) Click 'OK'.

a) Start Button

b) Type in 'Event viewer'

c) Click 'Windows Logs'

d) Click 'System'

e) On the 'Actions pane' click on 'Filter current log'

f) Drop down menu 'Event sources'

g) Click on 'Power trouble-shooter'

h) Than click on 'OK'.

By the way, to check what wakes up your computer.

a) Start button

b) Type 'Command prompt'

c) Type 'cmd'

d) Type 'powercfg -waketimers'

e) Type 'powercfg -lastwake'

To specify the boot sequence:

3. Use the arrow keys to select the BOOT tab. System devices appear in order of priority.

4. To give a USB device boot sequence priority over the hard drive, do the following:

  • Move the hard drive device to the top of the boot sequence list.
  • Expand the hard drive device to display all hard drives.
  • Move the USB device to the top of the list of hard drives.

6. Save and exit the BIOS setup utility.

7. The computer will restart with the changed settings.

 

To specify the boot priority:

 

  1. When the computer starts to boot up, after the manufacturer's ID screen, press Key associated to enter BIOS page several times. The device selection menu appears.
  2. Use the up and down arrows to select USB Flash Drive.
  3. To boot from the selected device, press ENTER.
