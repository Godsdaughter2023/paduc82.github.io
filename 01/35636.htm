Hello,

Thank you for posting your query Microsoft community.

You can set up Mouse Keys in the Ease of Access Center. This will allow you to use the keypad to move the mouse and click right or left buttons.

Follow the steps:

  • Press windows key + X key.
  • Select Control Panel.
  • Click on Ease of Access Center.
  • Select make the mouse easier to use.
  • Set up mouse keys.
  • Then Turn on Mouse Keys.
