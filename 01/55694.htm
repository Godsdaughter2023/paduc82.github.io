By your description you are currently running on the generic Windows default drivers.

There is nothing in the 1709 update that stuffs up games, but it does cause some driver issues that can be fixed by

"I rolled back my gpu drivers when i first got the fall creator"

Do not 'roll back' or update drivers via Device Manager, if this is how you did the 'roll back'

Do not allow Windows Updates to mess with your Nvidia drivers either.

If you still have a problem with your 780 after using the method below (if you have not already used this method)

1) When you reboot after uninstalling, Win10 tends to sneak in a driver from it's cache during reboot

(I believe it's the first gpu driver you ever installed on Win10).

One give away of this having happened is if you still have access to the Nvidia Control Panel but the Nvidia driver is not listed

in Programs and Features.

2 ) Similarly Windows Updates may pop in a new driver during reboot if you are connected to the internet.

- Download the current driver from Nvidia  - NVIDIA Driver Downloads - Advanced Search

- Uninstall your driver via Programs and Features, but before you reboot go off line and run either Driver Fusion or

Display Driver Uninstaller to finish the job (you can also use these tools without first uninstalling via P&F. I just prefer the old longer method)

You need to use one of these tools to prevent Windows from installing the cached driver when rebooting.

They also will uninstall the driver Windows installed if there is no Nvidia driver listed in Programs and Features.

The free version will do the job, even though you may get a message that you need the pay for version to delete some files.

You can go back on line after rebooting again.
