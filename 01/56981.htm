Cause/solution of: Media Creation tool stuck at 0 % - Microsoft Community

th3r_565 say:

Cause/solution of: Media Creation tool stuck at 0 %

I know the solution to this problem so decided to share it. In the case of Media Creation Tool being stuck at 0% while for example creating new install media (ISO/install USB), in 90% of cases it's the following services not started/functioning correctly:

- Windows Update service

- Windows Firewall service (needed for Update service to work properly, also in other scenarios)

So check here if this is your problem when you got the 0% download progress bug:

Click start> ''run'' and enter: services.msc (or start and search for ''Services'' app)

Then navigate to Windows Firewall and Windows update service, right-click them and go to Properties in the dropdown, select Automatic mode (after this the ''Start'' and stop service controls will be ungreyed and usable) now in Automatic startup mode, click the ''start'' button manually on both services.

Restart your PC when both services are in Automatic mode and started.

When PC rebooted, try running the Media Creation Tool again. It's likely to work this time.

If it does still not work and both services are however running, try this generic Windows Update problem fixer: https://support.microsoft.com/en-us/help/971058/how-do-i-reset-windows-update-components

This information on a known cause being these services and the guide I provide can be useful as Microsoft didn't document well what to do when you got Media Creation Tool stuck/hang at zero. As an IT'er Ive had countless of support request for this exact issue, and countless times it turned out to be these services not working properly, so I can safely say this problem is the root cause of these troubles creating install media.

Also, one doesn't always suspect something like Windows media creation tool, needs Update and Win firewall service to work. It makes little sense and there's no warning when its not running. However the services are heavily integrated with the creation tool and it bugs out when not. This guide can spare you lots of fiddling and troubleshooting.

This goes for any Windows version that can run the Media creation tool although I've seen the issue mostly on Windows 10 and 8.

gokul t say:

Hi WWaren,

 

To turn off Windows Media Center (WMC), follow these steps:

1. Click Start and select Control Panel.
2. Click Programs and Features.
3. Click Turn Windows features on or off on the left-pane.
4. Expand Media Features and uncheck Windows Media Center.
5. Click Yes and OK to continue.
6. Restart the computer.

WMC would be removed now.

 

Please note that WMC remains stored on your hard disk so you can turn it back on if you want to. Turning off WMC does not uninstall the feature, and it does not reduce the amount of hard disk space used by the same.

michelle vil say:

We appreciate your update. You don't need to purchase a new Windows 10 Home installation CD/DVD, you just need to follow the steps below:


  1. Insert your Windows 10 DVD/USB and boot up to the DVD/USB boot menu. 
     
  2. Press OK to get past the language settings. 
     
  3. Choose Repair Computer. 
     
  4. Choose cancel if asked to run System Restore 
     
  5. At the menu, choose Command Prompt. This will put you on a command line in the WinRE. 
     
  6. Enter "regedit" without the quotes. 
     
  7. Highlight HKEY_LOCAL_MACHINE (HKLM) in the left pane. 
     
  8. Go to the File menu and choose Load Hive. 
     
  9. In the source box, navigate to the drive where Windows 10 is installed (C: in my case). Drill down to C:\Windows\System32\Config. Choose the "System" file (with no extension). Hit OK. 
     
  10. In the next window name the hive "Broken System" without the quotes. 
     
  11. Hit the plus next to HKLM. You will see a key named " Broken System". 
     
  12. Drill down to HKLM\Broken System\ControlSet001\Control\Class\{4D36E967-E325-11CE-BFC1-08002BE10318}. 
     
  13. Look in the right pane for an Upper Filters line. If you see "snapman" without the quotes, you need to delete it 
     
  14. Only delete snapman by using the Modify on the Edit menu entry. Make sure you do not delete any other Upper Filters. 
     
  15. You should also check HKLM\Remote System\ControlSet001\Control\Class\{71A27CDD-812A-11D0-BEC7-08002BE2092F} for Upper Filter entries. If you see "snapman" remove it from there as well. 
     
  16. Go to the File menu and choose Unload Hive. This will save the changes in the Windows 10 registry. 
     
  17. Exit the command prompt and hit Restart.


Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully.
