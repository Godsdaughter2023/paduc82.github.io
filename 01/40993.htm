The best advice is to fix your partition table first and give Windows all the space you want it to have, and to do that from your Fedora Live Desktop CD.  I don't know how to access system utilities like the partition manager (gparted) in GNOME 3 (Fedora's default shell), but it should be already installed on the live desktop CD.  You want to make sure you have enough unpartitioned space for Windows to set up and live on: 15 GB is the barest minimum, and 30 GB is the most practical bare minimum.  (Windows is always hungry for more.)

rajesh govind say:

 

Hi Nikhyl,

 

It would be great if you can provide these information for a better understanding of the issue.

 

1)       Did you upgrade to Windows 8 or is it a brand-new computer that came with Windows 8 preinstalled?

2)       What are the recent changes that you have made to the computer prior to this issue?

 

From your description, it seems that this could be either due to some video card or driver issue on the computer. I would suggest you to try the following methods and check if that resolves the issue.

 

Methods 1: Run the Hardware Troubleshooter

 

The Hardware Troubleshooter is an automated tool which checks the hardware connected to the computer for any known issues and provides the details on how to fix them. Follow these steps to run the troubleshooter.

 

a)      Press “Windows Logo” + “W” keys from the keyboard.

b)      Type “Troubleshooting” in the search bar and press “Enter”.

c)       In the “Troubleshooting” window, click on “View All” on the left pane.

d)      Click on “Hardware and Devices”.

e)      Click on “Advanced” and then click on “Run as Administrator”.

f)       Click “Next” and follow the on-screen instructions to complete the troubleshooting process.

 

 Please check the detailed troubleshooting report and let’s know what it says. You may also try the suggestions given by the troubleshooter and check if that helps. If the issue still persists, then try uninstalling and reinstalling the graphics card drivers on the computer.

 

Method 2: Uninstall and Reinstall the Graphics Card Drivers

 

Step 1: Uninstall the Device Drivers

 

a)       Press “Windows Logo” + “X” keys on the keyboard.

b)       Click on “Device Manager” from that list.
c)       Search for the graphics card device from the device list, right click on it and then select “Uninstall”. On the un-installation window, if you have an option: “Delete the driver software for this device” you may select that and then remove the corrupted drivers from the computer.
d)       Follow the on-screen instructions to complete it and the restart the computer if prompted.

 

After the restart, you may again open the Device Manager and then click on the “Scan for hardware changes” button at the top of the Device Manager window and check if Windows 8 detects the graphics card device and installs an appropriate driver automatically. If the issue persists, then go with the next step.

 

Step 2: Update Driver Software from the Manufacturer’s website:

 

Go to the Samsung or Graphics Card manufacturer’s website and search for the latest Windows 8 compatible drivers available for the Graphics Card model and then install it by following the instructions given in the website. If Windows 8 compatible drivers are not available, then you may try installing the old (Windows 7) drivers in the compatibility mode and check if that helps. To install any drivers/apps in its compatibility mode, you may refer to this article:

 

Get your apps and devices working in Windows 8

 

Hope this is helpful. Please get back to us with the results.
