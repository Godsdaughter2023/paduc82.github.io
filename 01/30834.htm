ralph san say:

Hi K. Wester-Ebbinghaus,

Your correct, SSU stands for Servicing Stack Updates. We can simply call this the Quality updates, these are the monthly updates that include the fixes for recent known issues from the previous builds.

While LCU stands for Latest Cumulative Update. We can call this the Feature updates, usually released twice a year, and used to be called version upgrade. Feature update usually include new feature to Windows 10, not unlike the Quality update. Best example of Feature update is from Windows Anniversary Update (version 1603) to Windows Creators Update (version 1703).

You can also check the article below, this article best explains the overview of Windows as a service:

gerald vit say:

Hi K. Wester-Ebbinghaus,

I have to make some amendments on Ralph Sta's post above. Servicing is the name of configuring Windows updates or Windows features (installing, removing updates/features). Servicing Stacking Update (SSU) are small patches or changes in the Windows Update components that are required to ensure that the update process goes smoothly.

Latest Cumulative Updates (LCU) are quality improvements, fixes for known issues. As of this writing, the latest Cumulative Update for Windows build 1709 is KB4093112 (OS Build 16299.371) and you are required to install the SSU KB4099989. This Update for the SSU will make stability improvements for the Windows 10, version 1709 servicing stack.

Answering your question on the validity of a certain SSU after several months. As required by the LCU, you are still required to install the associated SSU to ensure that you won't have issues on the upgrade process later.

Let us know if this answers your question.
