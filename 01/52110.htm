sandeep ghatuary say:

Hi ChrisSchultz1416,


Thank you for posting your query in the Microsoft Communities. Follow these steps to make bootable USB for Windows 8.


Format a USB flash drive


In case you’re wondering how big of an USB flash drive you need, an 8GB drive is perfect since it can accommodate any of the three available ISO images. Here are the steps to format an USB flash drive.


Insert an USB flash drive into the computer.


Start a Command Prompt as an Administrator and type diskpart. This starts the Disk Partition tool which you need to use to format the flash drive. Once you see the DISKPART command prompt, type in the following commands:


a)    DISKPART> list disk                                 

b)    DISKPART> select disk #                         

c)    DISKPART> clean                                     

d)    DISKPART> create partition primary     

e)    DISKPART> select partition 1                 

f)     DISKPART> active                                   

g)    DISKPART> format FS=NTFS                 

h)    DISKPART> assign                                  

i)      DISKPART> exit                                       

 


Make the USB flash drive bootable


We just need to make the USB flash drive bootable before copying the files. To do this, you can use the Boot Sector Registration Tool (bootsect.exe) which is located in the boot folder of the Windows 8 ISO image.


Start a Command Prompt as an Administrator and CHDIR into the boot folder of the Windows 8 ISO image, e.g. I:\boot where I:\ is the drive where the ISO image is mounted


Type bootsect /nt60 E:                       /* where E: is the drive assigned to the USB flash drive */

Eg: F:\boot>bootsect.exe /nt60 E:  /*Where F: is the mounted ISO & E: is the USB flash drive*/

 


Copy the Windows 8 files onto the USB flash drive


From a command line, use XCOPY to copy the Windows 8 files to the USB flash drive. In the example below, I:\ is the drive where the ISO image is mounted. F:\ is the USB flash drive.


XCOPY I:\*.* F:\ /E /F /H

This is how I set the sharing.

I never use the Homegroup setup.

To share resources on a WORKGROUP

Right click on the resources you wish to share. eg D: (Storage drive)

Select 'Share with' - Advanced sharing.

Click on the 'Sharing' tab. Then click on 'Advanced sharing'

Enable sharing by putting a check in the box. The resource can also be named. eg Downloads

Click on 'Permissions' and put checks in boxes required. eg Full control.

In the panel above click on 'Add'.

Click on 'Advanced'. Click on 'Find now' on the right hand side.

Select objects from the list that is displayed.

As a simple example: Double click on 'Everyone'.

Click on OK. Put checks in the boxes to enable permissions for that object.

Do this also with 'Network'.

Basically you can give permission for objects or users on this PC

and also to Everyone on the Network.

Now open the Network and Sharing center.

Click on Change Advanced Sharing in the menu on the left.

Select buttons to enable Printer sharing if required.

Scroll to the bottom of the page and click the drop downfor 'All Networks'

If you select to use a password - that must be used by users on the network

to access the resources that you have enabled to share.

You can also disable the passwording there by clicking that option.

All users on the network should then be able to access the resources

that you have enabled.

This holds good for all Windows 8 and 10 systems on the network

because they have the same WORKGROUP name of 'workgroup'.

The Workgroup name can be changed to one of your choice.

eg MYGROUP or PCGROUP. Must be the same on all pc's

required to access the shared resources.

The WORKGROUP name can be changed in the System Properties

by clicking on 'Change Settings' on the right hand side.

This also holds good for Windows 7 systems required to access the shared resources.

Ensure that the same name is used for the workgroup on all the pc's.

These are the basic settings. One can add security be enabling Users / PC's etc

under the security tab of the Advanced Sharing - Archive Properties.

bill smithers say:

Two reasons why it won't work and can't be done as you want it to.

1. The OS on the old hard drive has the drivers installed for the motherboard in the failed computer.

A clean install is the way you get around it.

2. BUT, here is where your license comes into play:

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

@@@@@@@@@@@@@@@@@@@@@@@@

Buy a new Windows 7 license and DVD (sealed package), but only after you check with the manufacturer of this Windows 10 machine that they support Windows 7 on your new machine and supply Windows 7 drivers for it.

Then do a clean install of 7 > reinstall your programs from the original media (DVD, download, etc.) and copy your data across from the old old hard drive using that USB enclosure.
