wunders say: You still haven't convinced me that it isn't possible for someone to have executed that shortcut.  I do not know your network setup, but I have in the past connected to my neighbor's unsecured wireless network, discovered and connected to his computer, and gained unrestricted access to every file on his hard drive; photos, tax-returns, you name it.  The simplest way to do this is in a scenario something like this:

1) Gain network connectivity to the target machine.  This can be through a non-passworded wireless router or one that uses WEP protection (WEP is 100% better than no password, but we all know what 100% better than zero is...)  or any open network path to the target computer.

2) The easiest target is a Windows XP Pro with Simple File Sharing disabled.  This permits a network connection with Administrator privilege.  Most computers have a hidden  "Administrative Share" of the root C: drive of the computer.  To verify, simply bring up a command prompt window on the target machine (Start -> Run -> "cmd") and enter the command:
    net share
If you see a "C$" share, then the root drive and everything on it is available for connection (the "$" makes it hidden).
To connect, though, you must authenticate as an administrator on the sharing machine.  Most people create a new user for themselves and when they do, the original administrator account becomes hidden (and out of sight is out of mind).  The default password for the built-in administrator is a blank password.  So unless the password of the built-in administrator has been changed sometime in the past, the system is wide-open for anyone to silently network-connect as administrator and gain full reign and access to all files on the C: drive.

If you have sufficiently locked-down the machine, then the above may not be possible, but I know no details of your network or computer security.
