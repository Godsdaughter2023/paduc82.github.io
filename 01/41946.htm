Hi Surendar,

Thank you so much for providing all the relevant information regarding the issues that you are facing on your Windows 8.1 computer.

Issues 1: The Folder View always changes to “Details” view even when you change it to “Extra Large Icon” view.

This issue is because the default folder view settings is set to show the folder contents in “Details” view. We can change this by applying the current view settings of an open folder to all folders on your computer that are of the same type as the folder you currently have opened.

For this, you may follow these steps:

a) Open a Folder from any drives on the computer.

b) Click on the “View” tab.

c) Change the view setting to “Extra Large Icons” in that folder.

d) Click on “Options” and then click on “Change folder and search options”

e) Click on “View” tab on the “Folder Options” window.

f) Click on “Apply to Folders”.

On doing this the “Extra Large Icons” view settings will be applied to all the folders of the type that you have opened. You may need to follow these steps again for the folder types like: “Documents”, “Music” etc. to have this setting applied for all the folder of their type.

Issue 2: Unable to set the default app to open a particular type of files

You can set the default apps based on the file type or protocol of a file. This can be done from “Default Programs” from “Control Panel”. Please follow these steps:

a) Press “Windows Logo” + “X” keys on the keyboard and select “Control Panel” from that list.

b) Select either of the options: “Small icons” or “Large icons” from the “View by” menu on the upper-right corner of the “Control Panel” window.

c) Click on “Default Programs”

d) Click on the link: “Associate a file type or protocol with a program”

e) This will list all the file type existing on the computer with the current default program next to it.

f) You may now click on the file type that you would like to change and then click on the button “Change Program”.

g) You may change the default program or find an appropriate app from the Windows Store, if a particular app is not installed on the computer.

Hope this is helpful. Please get back to us with the updates.

BIOS and chipset drivers).

DriverView - Free - utility displays the list of all device drivers currently loaded
on your system. For each driver in the list, additional useful information is
displayed: load address of the driver, description, version, product name,
company that created the driver, and more.
http://www.nirsoft.net/utils/driverview.html

most current. Right Click the Start Button - Device Manager - Display Adapter - write down the make and complete model of your video adapter - double click - Driver's tab - write down the version info. Now click UPdate Driver (this may not do anything
as MS is far behind certifying drivers) - then Right Click - Uninstall - REBOOT this

if 3rd party with their own software and drivers and any other major device drivers
you have.

Now go to System Maker's site (Dell, HP, Toshiba as examples) (as rollback) and
then Device  Maker's site (Realtek, Intel, Nvidia, ATI as examples) and get their
latest versions. (Look for BIOS, Chipset and software updates at System Maker's

windows activation Backup

I've activated windows using Skype, got like the below image

Do I need to reactivate windows again after reinstall windows?

 
andre da costa say:

No.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already ownto reactivate if necessary. or use the reset function in Windows 10.
