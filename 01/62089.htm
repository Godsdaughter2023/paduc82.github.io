jeeped say:
... Is the HPIIIP driver available in Windows 7?

As an alternate to the HP support site, you should be able to get the correct drivers from Devices and Printers' Add Printer and Windows Update. The printer is available as HP,  HP LaserJet IIIP.
  1. Click Start or tap the  key and select Devices and Printers from the right-hand list. When the Devices and Printers windows opens and populates, click Add Printer from the command options across the top.
  2. When the Add Printer window open, first select Add a local printer. Next select a port and click Next.
  3. At the next window, immediately click the Windows Update button and wait a few minutes until the list repopulates itself. It will look like nothing is going on, but it is working. Once the lists have refreshed, first select HP from the manufacturer and then choose HP LaserJet IIIP from the Printers list. Note that HP LaserJet IIIP is alphabetically past the HP LaserJet 3xxx entries.
  5. Click Next and supply a name for the printer.
  6. When you click Next the printer driver should install and you will be left with the final screen where you can run a test print.
