chrisskinner say:

Device USB\VID_1BCF&PID_2C55&MI_00\6&9e1e39c&0&0000 requires further installation.

OK - this is a common error. Most people think the solution is a driver but it is not.

Web cam camera and video use "Camera Roll" in Pictures folder. The fault that causes the Camera not to store the picture in the Camera Roll folder lies in a DLL (Dynamic Library Link file) in 2 (two) separate files  USB\VID_1BCF&PID_2C55&MI_00\6&9e1e39c&0&0000 that are sysfiles (System Files) in Windows. These files have elevated permission requirements and are preventing themselves from being changed when the web cam is re-installed, even a "clean re-start" cannot get over the hurdle. The answer lies in renaming the said files and then re installing the web cam package, to do this requires elevated administrator permissions which can be gained through opening the Windows symbol key together with the "X" key and picking "control prompt (admin) and renaming the files (As in VID_1BCF&PID_2C55&MI_00 to VID_1BCF&PID_2C55&MI_00.old and same for \6&9e1e39c&0&0000 and then re-installing web camera in device manager and the new files should self create. In the properties tab for web cam in Device manager, look at the "Events Tab" and there should no longer be a message saying the device requires further installation, also the web cam should now work after one reboots/restarts one's Windows 10 PC, as the files created by either photo shot or video should now be migrated to the Camera Roll, via the new sysfiles which are now compatible.Don't ask me how the error crept in, in the first place but it was probably a missing letter or number in the kb whatever update.
For further information and video and work flow charts go to Google and cut and paste the subject header above, you will find the relevant articles in microsoft.com. At last audit, today some 12,497 people have this problem currently, so it is a pretty common problem.

Forgotten password and other sign-in problems

Step 2: Let us try to enable the built in administrator account and try to create a new user account and transfer all the user profile data of the old account.
