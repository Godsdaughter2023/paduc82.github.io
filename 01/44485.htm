The 60-day waiting period helps keep hackers from changing an account’s security info as a way to completely take over that account. This waiting period gives the real account owner enough time to be alerted of the change and to cancel it in case they didn’t start it.

Port Forwarding - Microsoft Community

xb1r hydros say:

Port Forwarding

Login Information

  • Gateway: 192.168.254.254
  • Username: admin
  • Password: admin

Port Forwarding

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.254.254 into the web address bar.
  4. This will take you to a login page, type admin for the “Password”.
  5. Once logged in select the NAT Settings on the left side.
  6. In the sub category click on Port Mapping.
  7. On the main page click on Add Item.
  8. In the Type drop down select Custom.
  9. For Protocol select TCP/UDP.
  10. For Remote Port Range type in the corresponding port number.
  11. For Local Port type in the same port number.
  12. For Status change to enable.
  13. Repeat steps 7-12 until all ports have been added.
  14. Select Submit to finish.

 

Disable Port Forwarding

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.254.254 into the web address bar.
  4. This will take you to a login page, type admin for the “Password”.
  5. Once logged in select the NAT Settings on the left side.
  6. In the sub category click on Port Mapping.
  7. Change the Enable option to Disable for each port.

    Which version of Baldur's Gate ? The original was released in '98, so if you have not tried already

    you may find that using Compatibility for Win98 will get the game going.

    ( R/click on the games .exe or shortcut and go to Properties > Compatibility. )

    - With games of this vintage there may be other issues, such as problems with colour, which may have been

    fixed with a 'community' patch. The issues vary between games.

    You would find any available  fixes on a Baldur's Gate  site/forum.

    - If the game is not installing it may be using a 16bit installer. 16bit installers will work on 32bit Windows, but

    do not work on 64bit Windows. Some games also have fan based installers available.

    - You can also get the 'Enhanced' version from Good Old Games, and Steam, which in addition to the updated

    graphics, will have been fixed to suit modern Windows.

    If you already have a digital version of the game you would do better to inquire at the Steam or GOG forums

    for fixes that are relevant to these versions.

    https://www.gog.com/game/baldurs_gate_enhanced_edition

    - Regarding older ( pre '07) disk based games in general...

    MS has come up with a 'feature' that prevents disk based games that used SafeDisk copy protection from

    running. This is built into Win10, and has been 'added' to Win7/8/8.1 by a Security Update.

    I have more details on MS's DRM Blocking malware in my post at   -- this thread --

    - Games up to '98 are mostly DOS based. These need to be run though  DOS BOX ( GOG also fixes this

    issue with these games ).
