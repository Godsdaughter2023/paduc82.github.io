      So how can I get rid of it?

      Thanks

      cmdrkeene say:

      Yes, you are the owner of the file.  That is the record that shows that your user account created the file on the hard drive.

      You can click "Remove properties and personal information" and choose to remove all information about the file.  Note that the username of whoever owns the file cannot be removed. You can change the owner if you want from the security tab, but that means you lose access to the file.

      donald ducks wife say:

      That's just too much, it really makes me so angry, that I am not allowed to decide over my own computer.

      Can I change the name of the user account then?

      cmdrkeene say:

      You can create a new account with a new name.  I would do this:

      1. Create new account
      2. Open the C:\users folder and drag your files from the old account to the new one
      3. Delete the old account.

      Keep in mind that the "created by" username on every file has been an un-removable component of Windows since Windows NT back in 1994.  You may not have noticed until Windows 8.1, but it's been there for 20 years on any hard drive formatted with "NTFS" (the NT File System).
