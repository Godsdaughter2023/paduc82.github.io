don cuthbert say:

You may have created the bootable USB wrong. Many use a utility called Rufus to make a bootable USB from the official Windows 10 ISO file. You also have to tell your PC to boot from that. Many have a "one-time boot menu" that can be activated using a key, such as F12, at power-up. This enables you to choose to boot from USB without changing the BIOS default.

If your PC is capable of running in UEFI BIOS mode with Secure Boot enabled, you should choose that. The new disk would need to be initialized as a GPT disk to boot. GPT disks have a specific partition layout, different from an MBR disk. Also, check if there are any BIOS updates available for your motherboard and upgrade your BIOS, if applicable.

The simplest would be to install the drive as drive 0, boot from Windows 10 install media on a USB stick, and install Windows 10 to the new disk (after first deleting all partitions from it).

If you really want to just clone the existing disk, there is often confusion introduced by having two boot managers detected by a UEFI BIOS (one on each disk). After cloning, it would help to disconnect the original disk until you see that the new one behaves normally on its own. I would recommend deleting the System partition from the original disk to eliminate it from being detected by your UEFI BIOS, as both will show up in boot options as "Windows 10 Boot Manager" with no easy way to tell which is which.

•Ntuser.dat
•Ntuser.dat.log
•Ntuser.ini

h) Right-click and Select Copy.

i) Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New Username is the name of the new user profile you created.

I see that a 2004 HP desktop has been donated to your nonprofit organization. What method was provided to "roll it back" to its original condition? XP installation CD? HP Recovery CD? Or a hidden recovery partition on the hard drive?

http://support.microsoft.com/kb/310353
