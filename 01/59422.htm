aaron_sum say:

Hi,

Thank you for writing to Microsoft Community Forums.

In order to get clarity and assist you with appropriate steps, Could you please confirm if there any major change on your system prior to which you did not face this issue?

This issue usually occurs when the folder is shared, but the inheritance is not enabled. I would suggest you to check by enabling the inheritance for the folder containing the subfolders which you are not able to access and see if this issue still persists. Please follow the steps below and check if it works for you.

Steps to Enable Inheritance:

  1. Right-click on the folder that you are sharing over the Network.

  2. Click on Properties at the bottom of the context menu.

  3. Click the Security tab, click Advanced, and then click the Permissions tab, select the user and check if the Inheritance is enabled at the bottom left corner of that window. If not, click on Enable Inheritance (Note: If the Inheritance is already enabled, you will find a button in the same place that says ‘Disable Inheritance’).

  4. Click Apply and then click OK.
