¡firedog say:

I have Live Mail 2012 and a comcast.net (Xfinity) mail account on my Windows 7 pc ... I can get email both places

It's not clear what you mean by this. Could it be that you see your Comcast account in the folder pane in Windows Live Mail, but you also sometimes access the same account using a browser to visit the Comcast webmail page?

... how do I transfer the address book to my new Windows 10 pc. 

There is as far as I know no way to export the contacts list from an Xfinity account, which is what you would have to do to have a straightforward way of importing them into the contacts list in Windows Live Mail. Here's a rather tedious workaround, though:

  1. Open your Comcast address book and select the first 20 contacts
  2. Click More options, select Send as vCard and email the message to yourself.
  3. When the message arrives, open it, then right-click on one of the attachments and select Save all. It's best to create a new folder to save them into.
  4. Repeat for the remaining contacts, in batches of 20.
  5. In Windows Live Mail's contacts window, click Import and select Business card (.VCF). Navigate to the folder where you saved the .vcf files, select the ones you want to import and Open.
     
    Note that Windows Live Mail will not check whether a contact already exists in the list. Only select the ones you don't have already.

When your contacts list is complete in Windows Live Mail, you can use the Export function to save it either as a single CSV file or as multiple VCF files, one per contact. You are now ready to import into whichever mail application you want to use in Windows 10.
