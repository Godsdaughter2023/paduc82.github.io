As an administrator:

a.     Click start

b.     Select Run

c.      Type in gpedit.msc

This will give you the swipe up to login again.

a. user say:

Hi Jacky,


This issue could be either due to some issues with the power settings or corrupted system files on the computer. Try the methods below to troubleshoot the issue.


Method 1: Run the Power Troubleshooter

1.   Open Control Panel.
2.   Type Troubleshooting in the search bar and press Enter.
3.   In the Troubleshooting window, click View All.
4.   Click Power.
5.   Click on Advanced and then click Run as Administrator.
6.   Click Next and follow the on-screen instructions.

Method 2: It could be a compatibility issue with the new Fast Startup component.

1.   Open Control Panel, choose Power Options.
2.   On the Left side menu, select Choose what the power buttons do.
3.   Select Change settings that are currently unavailable.
4.   Scroll down to the Shutdown settings section.
5.   Remove the check mark from the Turn on Fast Startup option.
6.   Select the Save Changes Button.
Shut Down the computer, wait a couple of minutes and then use the Power Button to start the computer.

Method 3: Restore default settings of power plan.

1.   Open Control Panel.
2.   Click the Power Options icon.
3.   Click Change advanced power settings.
4.   Select the option Restore default settings for this plan.
