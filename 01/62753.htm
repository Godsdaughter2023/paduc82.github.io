michaelorschel say: Managed to fix it on my own via regedit.

If anyone else is missing the default program events, you can add them back through regedit:

go to start and type in regedit to the search bar. Press enter.

In the regedit window, go to
HKEY_CURRENT_USER\AppEvents\Schemes\Apps\.Default

and add the following key names inside the .Default directory:

http://i.imgur.com/RldufWe.png

after that you can open Sounds and the program events (for default sounds at least) will be present once again. For other program event sounds you can add them in a similar way, however I can't tell you what the key names would be exactly.
