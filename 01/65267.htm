Hey Louw,

I recommend you try to go into Safe Mode to delete the file, or create Windows installation media to boot into, because it won't be growing when that Windows installation is running, hopefully.

You can prepare Windows installation media by using the Media Creation Tool, here: https://www.microsoft.com/en-us/software-downlo...

You will need a flash drive of 8 GB or larger in capacity.

When you reach the Windows Setup window when booting into the flash drive, hit SHIFT + F10, and you should be in Command Prompt. Enter in "C:" without quotes, depending on what drive letter you have. If you're not familiar with Command Prompt commands, I can give a brief summary of the commands you should be using.

cd = This enables you to navigate to a directory.
del = In shorthand, this is to delete a file.
dir = See in which directory you're currently in, as well as a list of directories and files which are in that directory.

Now when I told you to enter in "C:" or whatever your drive letter is, you start at the root of your hard or solid state drive.

Enter into the prompt, without quotes, "cd Users", then enter "dir" to see where you are. You should then enter "cd foldername", foldername being your user name, and use dir to guide you as you cd through various directories. You need to traverse through the directories to get to that Video folder, and that file.

When the file is on sight, enter "del file", the file will be the name of it, with its extension (though the extension is optional).

I realize this is a bit of a complicated solution, so if you need help with this, do not hesitate to let me know. Good luck.
