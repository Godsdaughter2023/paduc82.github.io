thealtarpastor say:

Hi LiveLifeup

I prayed and labored over this same issue, and finally the wisdom came.

reboot your system, while rebooting keep pressing F8 until the option comes up for 'system repair', it will try and repair it, hit 'Cancel' you will then have the option to choose ,View Advance Repair Options', click on that, it will take you to an option to access 'DOS Command Prompt'. Then follow the instructions below. If it locates a bad file it will try to recover it. Let it recover it, and complete the check and repair process, however, if after it finishes and you reboot, but still have the same problem, go through the same process again listed below, but this time when you do the chkdsk c:\f and it locates and deletes the same file again, don't allow it go any further (it will try and recover the file that is causing you trouble ,again), 'X" out closing the window, then reboot, that should take care of the problem, and you should be able to get window back up.

Open a dos prompt window, and type "chkdsk c:" (or chkdsk d: depending if your file is on C: or D: or E:)
It will tell you "you did not provide the /f flag, I will only look without reparing".
No problem, let it look, and if it says "no errors", then chkdsk is not the solution.
If it finds a problem, run it again with the /f option, like this :
chkdsk d: /F

sometimes it's useful, sometimes not.

By the way, this is the harmless way you can use the CHKDSK utility.

Hi AlanDummer,

Thank you for posting your query in Microsoft Community forums.

From the issue description, I see that files and folder on CD-RW became read only.

I will surely help you with this.

When a file is written to a CD, the attributes are automatically changed to read only. This is entirely appropriate, as you can't further change the file and write it back to the CD again. That's just the nature of files on a standard CD or DVD.

When you burn a CD you're not making a temporary recording. You're making a CD that is unchangeable. When you burn a CD the software makes them Read Only file.

If you want to edit a file you have on a CD then you have to copy them on the local hard drive and then you should be able to edit it and then you need to again burn them on to a different disk.

 

If you face the issue even after copying the files to the hard drive then maybe it’s because the read only attribute has not changed. To change attributes follow the steps.

a)      Using Windows Explorer, locate the files on your hard drive that you copied from the CD.

b)      Right-click mouse on one of the files. A shortcut menu will appear.

c)      From the shortcut menu select Properties. A Properties dialog will appear.

d)      On the General tab, at the bottom of the Properties dialog, uncheck the “Read-only” attribute.

e)      Click OK.
