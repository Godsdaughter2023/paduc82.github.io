cmdrkeene say:

You don't technically have to format the HD, the setup process will do that for you.  I normally do it like this:

  1. Use the tool on the Windows 10 Media Creation Tool webpage to create your own Windows 10 installer disc (or USB stick).  This lets you make your own installer drive to re-install Windows on your computer.
  2. Boot the computer from the install disk and click "Install Windows"
  3. When it shows choices of which hard drive partition you want to install onto, I usually remove every partition that exists.  This destroys all data on the disk, so make sure you have your files stored somewhere safe and sound first.
  4. When the only listing left is "Unallocated space", just click Next and the setup program will automatically create the partitions required and install the OS (this process takes about 15 minutes).

When it's done, it will be at the part of setup where you set your time zone and such, and that part takes another 5-10 minutes to finish up before you are at your pristine "like new" desktop, ready to install your favorite programs/browsers and copy your files back into place.
