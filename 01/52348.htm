jess bor say:

Hi Alan,

We understand your sentiments. For the error "could not configure one or more system components" that you are getting, you may try the following troubleshooting suggestions:

1. Create a Windows 10 USB Bootable flash drive using this support link.

When using a USB drive, make sure to set up your BIOS to boot from the USB drive.

2. After booting into setup, choose the Repair Computer option > Troubleshoot > Advanced Options > Command Prompt. If you are using a recovery drive, click Troubleshoot > Advanced Options > Command Prompt.

On the command prompt, you will need to issue a set of commands to find and resolve issues that might be preventing your computer from booting.

3. Type BOOTREC /FIXMBR and click Enter.

This command will attempt to fix any corruption issues with the master boot record.

4. Next, type BOOTREC /FIXBOOT then click Enter.

This will attempt to write a new boot sector on the hard disk if Windows detects damage. Usually, this happens in cases where an older version of Windows was installed, or a non-compatible operating system such as Linux was installed.

5. If you see "Boot Manager is Missing" error, then the BOOTREC /RebuildBcd command may be able to fix it. This operation can also restore boot entries for older versions of Windows if you have a dual boot configuration.

6. If BOOTREC /RebuildBcd does not work, then Microsoft recommends you back up your BCD (Boot Configuration Data) store, then run the BOOTREC /RebuildBcd command again. Here are the commands:

bcdedit /export C:\BCD_Backup  then click Enter.
c:  then click Enter.
cd boot  then click Enter.
attrib bcd -s -h -r  then click Enter.
ren c:\boot\bcd bcd.old  then click Enter.
bootrec /RebuildBcd  then click Enter.

7. If you are dual booting with older versions of Windows such as Windows 7, you may use the BOOTREC /ScanOs command. The ScanOS command can find and restore entries for older versions of Windows.

8. After all these commands, you may try running the chkdsk utility on your Windows 10 computer. Please refer to the post Gerry C J Cornell Community Moderator  | Article Author by clicking this link.

Note:

It is always safe to back up your personal files to avoid any inconvenience.

Let us know the result for us to assist you further.
