Steps below

  1. Disconnect any Ethernet cables
  2. Uninstall the utility software.
  3. Re-boot the machine
  4. Re-install the utility software
  5. Re-boot the machine
  6. Sign in - do not open any programs apps etc.
  7. Using the utility software configure the connection.
  8. Wait for the connection to be established - this can take a bit of time, do not open any apps or programs while waiting
  9. As soon as the connection is established right click the adaptor icon in the right hand corner of the screen
  10. Select Network and sharing centre
  11. Click the Wi-Fi connection - should bring up the wireless status box
  12.  Click wireless properties
  13. On the connection tab select  "Connect automatically when this network is in range" and "Connect even if this Network is not broadcasting its name (SSID)"
  14. On the connection tab de-select "Look for other wireless connections while connected to this network" and "Enable WLAN connection settings"

    15. Hope this helps.
