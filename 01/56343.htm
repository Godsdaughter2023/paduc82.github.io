considerate_guy say:

You must be familiar with the "classic" version of Movie Maker where you had a collection (clip library) and could make initial edits on your clips before adding them to your movie.  It also had a traditional timeline where you could drag clips on top of each other to increase the transition time.  You could do the same with songs to make one fade out as the other faded in.

 

This is not the exact case with the newer versions of Movie Maker which have a different look.  The goal was to make it easy for novices to edit a bunch of photos (primarily) and a few videos into a sharable slideshow type movie.  The collection was removed in favor of just doing that work directly in the workspace.  Additionally, the "timeline" was replaced with a hybrid storyboard / filmstrip.

 

So that explains the last two items.  For cutting sections of a clip, this is done with a split.  You would either use the 'Split' option on the Video Tools | Edit tab, or just press 'M' when the playback marker is at your desired split point.  So for example, if you had a clip that had good (G) parts and bad (B) parts:

 

GGGGBBBGGGGG

 

You would play it to the end of the good part and make a split (M).

 

GGGG | BBBGGGGG

 

Go to the end of that and make another split:

 

GGGG | BBB | GGGGG

 

In the storyboard/filmstrip area you'll now see 3 clips.  Select the middle one, right-click and choose 'Remove'.

 

GGGG | GGGGG

 

Finally, if you are familiar with the prior version and would be more comfortable with that, you can load Windows Movie Maker 6.0 on Windows 7 or Windows 8.  And don't forget that this version allows you to add custom titles, transitions and effects where the newer version has no published customization API.

lemp say:

Strange that there is an open system file in an area that should be considered to only contain user documents

You misunderstand what is in the "Documents and Settings" directory.  As the name suggests, it is not just user documents, but also system information for each user of the Windows XP multi-user operating system.

A user's documents are stored (by default) in a subdirectory of Documents and Settings, i.e., C:\Documents and Settings\<username>\My Documents.  The "My Documents" directory includes subdirectories for "My Pictures" and "My Music", so copying the "My Documents" directory will copy those files as well.

One problem is that Microsoft hides non-user files and directories by default, so if you look at C:\Documents and Settings (even if you look at a particular user's subdirectory), you'll only see what looks like user files.  For example:

Directory of C:\Documents and Settings\AnyUser

charliews say:

Chinese handwriting installed in Windows 8 the easy way

There has been quite a few requests for directions to enable Chinese handwriting, and the answers are not exactly up to the point.

 

Here is the step by step way to enable Chinese handwriting in Windows 8

 

1. Go to the "Charm"

2. Select "Control Panel"

3. Select "Language"

4. From "Language", select "Add"

5. Now add whichever language you want to add - Traditional HK, Taiwan or Maucau or Simplified. You don't need to download anything.

6. Go to "Desktop"

7. At the right hand bottom corner, you will see a "Language Icon", probably labelled as "EN"

8. Click on the "EN" icon and select the Chinese language you want

9. A new icon will now show up on the left hand side of the "Old EN" icon

10. Right click this new icon and select "IME pad".

11. Left click this new icon will switch between English keyboard and Chinese keyboard

 

Hope this will help

 

Charles

Manually look at manufacturer's sites for drivers - and Device Maker's sites.
