TV &quot;Lag&quot; When Changing Channels: SOLVED - Microsoft Community

stargeezertim say:

TV &quot;Lag&quot; When Changing Channels: SOLVED

Problem:  When changing channels, the XBox will often lag or pause for anywhere from 5 to as much as 30 seconds while attempting to adjust resolution differences between the Cable/Satellite box and the console.  Near as I can tell, the XBox is buffering the TV signal and up-/down-scaling the image to match the resolution so that no matter what is showing, the OneGuide displays properly.  This can often cause a delay while it attempts to shuffle the resolution and then re-buffer the stream.


Solution: This involves a bit of fiddling both with the output settings on your cable box as well as the XBox One settings.


Step 1: On your cable or satellite box, navigate to display resolutions and set it to display your highest resolution only (so in most cases 1080. It doesn't appear to matter whether it's 1080i or 1080p.)  Deselect any lower resolution settings so that only the highest is selected.


Step 2: On your XBox One, go into Settings > Display & Sound.  Under "TV Resolution", make sure it matches whatever you set your cable box to (again, mostly likely 1080p.)  Then under "TV Connection", select HDMI.  The Auto-Detect seems to be part of the problem, so make sure you select HDMI.  I'm not certain "Color Depth" has a real effect here, but you can fiddle with this a bit and see if your eyes see any difference (mine don't, but I'm old, soooo...) 


Step 3: Go back to TV mode on the XBox, and hit your Show Menu button (the right-side button, what would be the Start button on older controllers.)  Select "Display Mode" and choose "Native."


This has eliminated all the resolution shifting "lag" between my cable box and the console.  Hope this helps. :)

Wireless Security - Microsoft Community

xb1r athena say:

Wireless Security

Login Information

  • Default Gateway - 192.168.0.1
  • Login Username - Select admin in the dropdown
  • Login Password - leave blank by default

How To Find Wireless Passphrase (WPA/WEP)

The default firmware does not allow for the viewing of the security key, please use alternative methods.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser and in the address bar enter 192.168.0.1 and enter admin for the username and leave the password blank when prompted.
  3. Select Setup from the top menu.
  4. Select Wireless Settings from the menu on the left.
  5. Click the Manual Wireless Network Setup button.
  6. To change/disable the wireless security from the Security Mode drop down select the desired security type (WEP, WPA-Personal or None).
  7. If WEP is selected, select the WEP Key Length (either 64-bit or 128-bit).
  8. If WPA-Personal is selected, select the WPA Mode (Auto, WPA or WPA2) and the Cipher Type (TKIP, AES or Both).  WPA2 with AES is recommended.
  9. Enter the Pre-Shared Key or WEP Key.
  10. If WEP security is selected the password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  11. If WPA or WPA2 was selected the password can be between 8-63 characters, is case sensative and can use any character on your keyboard.
  12. Once the passphrase has been entered click Save Settings at the top.
