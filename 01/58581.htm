Adding AIF files to the Windows Media Player library - Microsoft Community

sam_i_am_dada say:

Adding AIF files to the Windows Media Player library

I can't add AIF files to my library, nor edit their properties in the Now Playing window.

It's possible to play them, but that's all.  Anyone know anything about this?  I'm using Windows Media Player 11.

tim de baets say:

Copy/paste the following into Notepad, save it as  WMP_AIF.reg , and double click this file to add it to the registry:

Windows Registry Editor Version 5.00
 
[HKEY_CLASSES_ROOT\.aif]
"PerceivedType"="audio"
 
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Multimedia\WMPlayer\Extensions\.aif]
"PerceivedType"="audio"
"Permissions"=dword:0000000f
"Runtime"=dword:00000007
 
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\MediaPlayer\MLS\Extensions]
"aif"="audio"

After that, WMP should recognize the AIF files and add them to the Music section of your library. However, it still won't be able to read/write any of the AIF tags (including length), simply because WMP doesn't have any tagging support for AIF.
