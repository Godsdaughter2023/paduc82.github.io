robert aldwinckle on forums say:

How can I download Microsoft Hosted Network Virtual Adapter and from where.

First check to see if you need to.   Using PowerShell

PS>Get-NetAdapter -IncludeHidden | where InterfaceDescription -match "Hosted" | Select DriverName

Then start from  C:\Windows  and perform this search

PS>cd $env:windir
PS>LS -Re -Fi vwifimp.sys -Fo -Ea "SilentlyContinue" | sort LastWriteTime | ft Length, FullName

Note that your Command line prefix may be different from mine and you should not include PS> in anything that you Paste into yours.
