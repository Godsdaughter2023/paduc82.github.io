There are several methods, but the easiest is to use the printer's front panel controls either to display the IP address in the printer's LCD display or to print a "network status" page.  Directions for these operations will be in your printer's User Guide.

phant0m`` say:

When you toggle control for ' Show desktop icons ', Windows Desktop feature. It modifies associate registry value data of' HideIcons ' value name under Explorer section.

0 to Show Desktop Icons (default state)

1 to Hide Desktop Icons

When you change the states for this feature, doing it the way you normally been doing it. It should instantly update the registry to reflect this change.

If you don't see this change, then something like security system or type of cleaner could be potentially reverting / undoing this change you making. Another scenario could be that location in the registry could of have had the permissions tampered with, or corrupted.

I won't ask you to explorer the registry, when you can simply query this information via Command Prompt with regular access rights.

When you make a change, do the following to see the state in the registry;

REG QUERY HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced /v HideIcons
