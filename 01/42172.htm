Hi ermonce,

Thank you for visiting Microsoft Windows Vista answers forum.

If the option “Word Wrap” under “Format” in Notepad is selected then the Status Bar option will be grayed out. Uncheck the Option “Word Wrap” and then you will be able to select Status Bar.

If you want both the option to be enabled at the same time, then there is a registry change that needs to be done.

Let the option Word wrap be selected and then follow the steps as below.

Note: Please back up the registry before performing the steps mentioned below:
For more information about how to back up and restore the registry, follow the steps as in the KB Article:
http://support.microsoft.com/kb/322756/

1. Click Start and then type regedit in the Start Search box.

If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

2. Locate the following key in the left pane:

HKEY_CURRENT_USER\Software\Microsoft\Notepad

3. In right-side pane, change value of StatusBar to 1.

4. Exit from the registry and the now check the notepad, both the options will be enabled.
