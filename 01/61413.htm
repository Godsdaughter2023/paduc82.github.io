From the issue description, I understand that you want to change the home location on Windows 8 computer.

 

Follow these steps to make sure the location service is turned on.

 

Step 1

Make sure that Allow apps to use my location is turned on.

a. Press the Windows + C to activate the charms menu and click on Search.

b. Click on Settings and type location, select Allow apps to use my location.

c. Turn on the option to Allow apps to use my location.

 

Step 2

Change the regional settings and check if the issue persists.

a. Press the Windows + C to activate the charms menu and click on Search.

b. Click on Settings and type location and select Change location.

c. Change the ‘Home location’.
