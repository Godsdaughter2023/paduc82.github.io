a. user say:

A Windows 10 PC needs two accounts:

1- A user account with Administrator permissions: This is typically known as an Administrator account, although technically, the correct name is an Administrators account (with an extra 's.')

This type of account is needed to perform tasks that require elevated permissions - basically anything that affects the computer as a whole. In actual use, you will almost never need to sign in to your Administrators account.

2- A user account with standard permissions: This is typically known as a Standard account or a Limited account, although the technically correct name is a Users account.

This is the account for your day-to-day computer use and it's where you store your personal files. It has fewer permissions than an Administrators account, which makes it more secure, particularly when you're on the internet. A Users account can only make changes that affect that particular account. Each computer user can have their own Standard/Limited/Users account.

If a Users account needs Administrators permission just temporarily - to install software, for example - they can right-click on the file they need to use and select 'Run as Administrator.' They'll be prompted to enter the password for the Administrators account, and they'll have Administrators permissions for that task only.

An Administrators account and a Users account can be either a local account or a Microsoft account. A local account can only be used on the computer where it was created, while a Microsoft account can be used on any computer that can connect to the internet. Every user needs to have a Microsoft account, but you don't need to actually use it. You need a Microsoft account username and password in order to use many of Microsoft's cloud-based services, like OneDrive or Windows Store, but you don't need to use your Microsoft account on your Windows PC. You can use a local account, as I do.

So you could have, for example, a local Administrators account and a Microsoft Users account, or a Microsoft Administrators account and a local Users account. There's nothing like having too many choices to turn what should be easy into something that becomes complicated  ;-)

andre for directly say:
Windows 10 offers two methods of signing in: local account and Microsoft account. Windows 10 setup only provides the option to set up a Microsoft Account if you are connected to the Internet. If you are not connected to one, setup will default to a local account.

Local Account – this account works like credentials used to log onto Windows 7 and earlier versions of Windows. A local account doesn’t include the modern benefits of Windows 10, such as PC Syncing, the ability to download apps from the Windows Store and two-factor authentication.
