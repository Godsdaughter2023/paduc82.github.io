    Hi Chris,

    Disabling the lock screen in Windows 10 can be facilitated by performing changes in the registry of your device as well as editing user account settings. To better understand your concern, we'd like to know the following:

    • The edition, version, and build of Windows 10 that you're using on your device. To check this, you can perform the steps posted on this link.
    • If your device is managed by a domain or an organization.

    The simplest approach would be to remove the existing password that your account is using. If you're using a local account, you can perform the steps posted under the To remove an existing password from a local account section of this link. If you're using a Microsoft account, you'd need to switch to a local account to facilitate your request. The link will also provide the steps to complete this.

    See other options below

    If you're using a home/personal computer running on Windows 10 Home (version 1703), you can perform the following:

    1. Right-click the Start button.
    2. Click Search.
    3. Type regedit and hit Enter on your keyboard.
    4. Double-click HKEY_LOCAL_MACHINE.
    5. Double-click SOFTWARE.
    6. Double-click Policies.
    7. Double-click Microsoft.
    8. Right-click Windows.
    9. Click New.
    10. Click Key.
    11. Type Personalization and hit Enter on your keyboard.
    12. Right-click the Personalization key you just created.
    13. Click New.
    14. Click DWORD (32-bit) Value.
    15. Type NoLockScreen and hit Enter on your keyboard.
    16. Double-click the NoLockScreen DWORD you just created.
    17. Type 1 in the Value data field.
    18. Click OK.

    For personal/home computers running Windows 10 Pro, you can perform the following:

    1. Open Run command box by simultaneously pressing Windows logo and R keys (Windows+R). In the Run dialog box, type Netplwiz and then press Enter key.
    2. You will be prompted with the User Accounts dialog, select your user account first and then uncheck the option labelled Users must enter a user name and password to use this computer at the top. Click Apply button to see Automatically sign in box.
    3. On the "Automatically sign in box" confirm your credentials and then click OK. Restart the computer and check.

    Also, you can have Windows 10 to automatically log in your account without the need to manually entering your password. To do this, you can perform the following:

    Step 1: First step is to open Run command box by simultaneously pressing Windows logo and R keys (Windows+R). In the Run dialog box, type Netplwiz and then press Enter key.

    Step 2: In the resulting User Accounts dialog, select your user account first and then uncheck the option labelled Users must enter a user name and password to use this computer. Click Apply button to see Automatically sign in box.


    Step 3: In the Automatically sign in dialog, type your password and then re-enter the password to confirm the same.
