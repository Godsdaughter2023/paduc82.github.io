We set up WDA in meeting rooms. Already configured the device to ask for a pin, but my question is, is it possible to force the device to always ask for a pin ?

edjay rust say:

Hi Rodrique,

If you're using a device running on Windows, by design, it remembers connected devices, so your wireless display adapter won't prompt you to enter a PIN if you connect to it using the same device.

It will prompt to enter a PIN if you try to connect to the adapter using a new device. This means that it will depend on the device that you're trying to connect to the wireless display adapter. The configuration to force to ask for a PIN will not be on the adapter, but on the device that you're trying to connect.

If you try to connect to the adapter using a new device, your HDTV, monitor, or projector will display a randomly generated PIN. Enter the PIN in the Microsoft Wireless Display Adapter app on your device, and select Next to connect.

However, if you're always using the same computer (running on Windows) to connect to your wireless display adapter, you can go to Settings > System > Projecting to this PC > enable Require PIN for pairing.
