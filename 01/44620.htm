I suggest you download Process Explorer. This freeware utility performs a similar function to the Windows Task Manager but it goes much further.

For further information about Process Explorer see here:
http://www.microsoft.com/technet/sysinternals/SystemInformation/ProcessExplorer.mspx

When you open Process Explorer you need to custome settings. You do this by checking options on the View menu drop down. Click on Select Columns and then the Process Image tab. Check the boxes before Description, Image Path, Image Type, and Command Line. I suggest you also click on two other options on the View dropdown, namely Show Processes from All Users and Show Fractional CPU.

The extent to which you can customise is considerable. You can dramatically change what information is displayed and this leads to confusion and frustration at times.

With the CPU column you will see  System Idle Process frequently showing a high figure.. It is important to always remember at all times that this figure represents CPU capacity not being used. The figure displayed is 100.00 less the CPU being used by all running processes.

The Processes are listed in the left column. Some names appears to be duplicated. This is not so. An example of an apparent duplication is svchost.exe. You commonly something like 10 instances listed. If you right click on one and select Properties, and the Image tab you will find that the Command line differs for each instance of svchost.exe. Each instance of svchost.exe serves a different group of services. You can see a list of those Services if you click on the Services tab.

When you are experiencing high CPU usage you can see which process is using the CPU. What you should then do is post the command line of the process you have identified as using the CPU. We can then suggest how you should investigate further in order to discover the cause of high CPU activity.

susan_s. say:

Hello SurenderK,

Welcome to the Skype Community!

I know how important this is to you. I'd be glad to assist you with this.

Using the pre-installed Skype for Windows 10 (Skype UWP), the only options to set the connection status are "Do not Disturb" and "Do not show my presence" (Invisible).

The said connection status can be change from the Profile window, which you can open by clicking on your Profile icon.

If you want to set your status to Offline, then you need to sign out of the application.

The Away status will automatically be shown 5 minutes after the Skype UWP application is minimized or closed. Unfortunately, there is no manual option to set Away status.
