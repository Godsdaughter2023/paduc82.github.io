Information on Treesize

http://www.jam-software.com/treesize_free/

When you open Treesize select Scan from the Menu and you will see a list of drives. Click on C and it will generate a list of Folders. Post the names and the sizes of the 6 largest folders and the total at the top of the list.

To reconcile figures from Treesize with the size of a partition (or drive) you need to make allowances according to this formula:
C partition + System Volume Information folder* + Free space + gain from file compression = Disk or partition capacity

*Since 13 October 2011 with the introduction of version 2.6 the size of the System Volume Information folder is being reported so you no longer have to include it in your reconciliation.

To ascertain the size of the System Volume Information folder in Vista select Start, All Programs, Accessories, right click on Command Prompt, and select Run as Administrator. Type "vssadmin List ShadowStorage" without the quotes and press the ENTER key. The information provided is as follows:

 Used Shadow Storage space.

 Allocated Shadow Storage space.

 Maximum Shadow Storage space.
