Windows Installation on a PC with an ExpressCache SSD (GUIDE) - Microsoft Community

zfollette say:

Windows Installation on a PC with an ExpressCache SSD (GUIDE)

This is a guide on how to clean install Windows 7 or 8 on a PC with an ExpressCache SSD.


With this problem, if you are trying to install Windows 7, it would give a 'BOOTMGR is missing' error or something like it. If you are trying to install Windows 8, you might be getting a BCD error on startup. Your computer might even just shut down when you try and boot it. These are all the things I have faced. Here is how to fix it. (The problem is that Windows installs the bootloader onto the SSD, which can't be booted because of hardware restrictions)


If you don't even have an ISO file to boot, here is a few links to the tools you will need.


Windows 7 ISOs

Windows 8.1 Enterprise Evaluation

Microsoft has not released any other Windows 8 ISOs.


If you are on Windows 7 or 8, and you are using a DVD for install, just right click the ISO file you downloaded and open it with "Windows Disk Image Burner", then select your DVD drive and let it do its thing.


If you are on any other Windows OS, you can extract the ISO with winRAR or some other Compression/Decompression tool, then drag all of the files to your USB or DVD.


You may need to set the boot flag for your USB, I recommend using GParted.


If you are on a Linux OS, you should do the following:

Make a folder in /mnt/ called 'iso': sudo mkdir /mnt/iso

Mount your ISO file into that folder: sudo mount /path/to/ISO/file.iso /mnt/iso

Then copy all of the files in /mnt/iso/ to your USB/DVD manually.

Finally, unmount the ISO: sudo umount /path/to/ISO/file.iso

You may need to set the boot flag for your USB, I recommend using GParted.


If you are on a Mac, you may want to check out some guides online. I am not an experienced Mac user. I assume the process could be similar to Linux, because Mac OSX are Unix based OSs.


Ok, now that we have the bootable USB/DVD, we can get into the actual process.


Boot up the installation as normal, when Windows loads all of the files hit: SHIFT+F10 this will open up the Command Prompt.

In the Command Prompt, type the command: diskpart

This will load the DiskPart Utility.


Now, in diskpart, you want to display all of your disks: list disk

You want to find your hard drive (Usually the largest one)

Now select your hard drive: sel disk n

Let 'n' represent what disk number your hard drive is.

Now you want to clean it, and make two partitions. Follow these next commands.

clean

create partition primary size=100 (for Windows 7) (This is the boot partition)

create partition primary size=350 (for Windows 8) (This is the boot partition)

sel partition 1

format fs=ntfs quick

active

create partition primary (This is your hard drive space)

sel partition 2

format fs=ntfs quick


Now select your solid state drive: sel disk n

Let 'n' represent what disk number your solid state drive is.

Now you want to clean it: clean


Now you want to set you boot partition letter to F: and you hard drive space partition to C:

You want to list all of the volumes on the system: list vol

If you did the previous steps correctly and have no media plugged into the computer, other than your install disk, you should have 3 or 4 volumes. This depends on if you have a DVD drive or not.


You want to select the boot partition volume: sel vol n

Let n represent the Volume number for your boot partition volume.

You now want to set the letter to 'F' (You may have to change other letters around to make 'F' available): assign letter=F

Now, you select the hard drive space partition volume: sel vol n
Let n represent the Volume number for your hard drive space partition volume.

You now want to set the letter to 'C' (You may have to change other letters around to make 'C' available): assign letter=C

After all of this is done, you should continue the installation as normal. 


When Windows restarts, boot up the installation disk again.

You want to enter diskpart and reset the volume letters to C and F (they will have changed on restart) (see previous step to set volume letters)

You will notice that there are more volumes on your computer now. Disregard them.

Now, exit diskpart: exit


In the regular command line, you want to run the command: bcdboot C:\Windows\ /s F:

Once you enter that command it should say something like "The boot files were copied successfully"

You are now free to restart your computer and Windows should start right up and finish installing.

Once you get the computer up and running, you can freely partition your SSD without any problem.


I really hope my guide has helped anyone that had to go through the same problem that I did.

Make sure you open Pictures Library instead of Pictures folder.

Open File Explorer, click View > Navigation Pane > Show libraries.

You won't find it on the View Ribbon (where Sort & Group are).
