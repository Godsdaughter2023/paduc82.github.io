hairyfool say:

Android is an operating system the same as Windows RT, only one can be installed so the version of Android is irrelevant.

 

For it to be installed on another system the operating system would have to be available as a software package similar to Windows 7 etc., this is not available with Windows RT.

 

The x86 PC hardware platform complies with a strict set of specifications originally set down in the 1980's although expanded a number of times to add features such as VESA, PCI, AGP, PCI-Express, ATX etc. All PCs have to meet these specifications in order to ensure operating systems such as DOS, Windows, Unix, NeXT etc will run on them.

 

Although various devices have ARM processors in them the same design constraints are not in place, how an iPad, Android or Windows RT tablet is designed varies and as such even when the physical processor is the same other aspects differ which would prevent cross installation of operating systems. Porting of an O/S to a different platform is a major task and restricted by license agreements. You can't install Playstation on Wii or XBox. Despite being both Intel x86 you can't install Windows directly on a Mac (boot camp is a workaround) or OSX on a PC (Hackintosh is illegal)

Follow the steps mentioned below to customize a theme.

2.      Click the theme that you want to change to apply it to your desktop.

Do one or more of the following:

·         To change the background, click Desktop Background, select the check box for the image that you want to use and then click Save changes.

·         To change the color of window borders, click Window Color, click the color that you want to use, adjust the intensity and then click Save changes.

·         To change the sounds for the theme, click Sounds, click an item in the Sound Schemes list and then click OK.

·         To add or change a screen saver, click Screen Saver, click an item in the Screen saver list, change any settings that you want to change and then click OK.

Your revised theme will appear under My Themes as an unsaved theme.

To save a revised theme for your use only

To save your revised theme to use on your computer, follow these steps:

1.      Open Personalization by clicking the Start button and then clicking Control Panel. In the search box, type personalization and then click Personalization.

2.      Click your revised theme to apply it to the desktop.

3.      Click Save theme.

4.      Type a name for your theme and then click Save.

The theme will appear under My Themes. Note that themes saved this way can't be shared with other people.
