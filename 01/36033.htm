jeffrey roach say:

Windows 8 and Windows RT use Connected Standby (S0iX) which is a new power state similar to what we have on Smart Phones, currently supported on ARM and Intel's Clover Trail SoC and will come to Intel's Core i-CPUs with the Haswell Chipset. This allows for email, and other connected apps to receive data while it is in a low power sleep state. This way we don't need to wait for information to download.

 

This is by design, you can adjust the applications or shut the computer completely off....also if your Network Connection is metered it will not download information while in the Connected Standby state.
