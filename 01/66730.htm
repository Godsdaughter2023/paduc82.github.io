guest794 say:

Actually, Second Nature works just fine with Windows 10 ... although it might take a little tweaking.  Basically, the procedure is as follows:

1. Install a fresh copy of one of the various Second Nature samplers (preferably with the latest version of the Second Nature software - 4.4.7 or 4.4.8?).  Although the installation app will ask you for an email address, it's not used to validate anything, so the installation will work just fine.

2. You should have already saved your unpacked collections - they were originally stored in a subdirectory called "Images".  Replace the "Images" subdirectory under the new "Second Nature" folder that you just installed with the previously saved "Images" subdirectory and then start the program.  Select one of the pictures to be the new wallpaper.

3.  Go to the Windows customization page, then choose the Lock Screen option.  At the bottom of the page is a selection for "Screensaver".  Select that, and tou will see that one of the drop-down options will be "Second Nature".

Hope this helps ...
