Port Forwarding - Microsoft Community

xbr poseidon say:

Port Forwarding

Login Information

  • Default Gateway - 192.168.1.1
  • Login Username - "Leave Blank"
  • Login Password - admin

Port Forwarding

  1. Set a manual IP address on the xbox. (click here to find out how to do this)
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.

  4. When prompted, leave the user name blank and enter  "admin" for the password.

  5. Select Applications And Gaming from the menu at the top.

  6. Select Port Range Forwarding from the sub menu.
  7. In the first available empty box type "Xbox360" for the Application.
  8. Enter "3074" in both the Start and End boxes.
  9. For Protocol select "Both" from the dropdown.
  10. Type in the last octet of the IP address that was set in the xbox.
  11. Check the Enable box.
  12. Click Save Settings.

 

Disabling Port Forwarding

  1. Set a manual IP address on the xbox. (click here to find out how to do this)
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.

  4. When prompted, leave the user name blank and enter  "admin" for the password.

  5. Select Applications And Gaming from the menu at the top.

  6. Select Port Range Forwarding from the sub menu.
  7. Uncheck any of the Enable boxes for rules set up for the xbox.
  8. Click Save Settings.

    terryiorns say:

    RandyCO - this is a related issue I also had. In my case I solved the issue of syncing not finding the emails you expected. Syncing does not automatically copy all emails from the folder, just those from a recent time period. If you want to get the older ones, you have to change a setting.

    1. Click on the gearbox in Windows Mail at the bottom left.

    2. Click on Manage Accounts on the pane that comes up to the right.

    3. Click on the account you want - probably Outlook.com

    4. Click on Change Mailbox Sync Settings

    5. Under Download Email From select "any time"

    Click Done at the bottom

    This should get you the old emails when you sync a specific folder - at least it did for me.
