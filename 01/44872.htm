greg urban say: I had this problem in XP. 
The problem is that when you set up the language switching hotkey, MS assigns some other hotkeys as well for switching between Chinese shapes and symbols that are pretty easy to hit by accident and you probably didn't notice. (At least I didn't!)

The solution was:

1) to go into
Settings > Control Panel > Regional and Language options >Languages tab > Details button > Key Settings...

2) In "Hot keys for input language" section, click "Change Key Sequence" and assign the key selections  that are set to something like "Ctrl + period" and change them to something that you will not hit by accident, (like something with an F1 key). You have to assign something, it won't let you assign (None) to all of the selections. It will just switch back after you leave the dialogue box.

Leave the "Switch between input languages option" on 'Left Alt + Shift'  or 'Ctrl + Shift' if you still want to be able to switch between languages.

Best,

Greg
