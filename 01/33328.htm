considerate_guy say:

Trying to compress video to fit as an email attachment is nearly impossible, unless you have a really short clip and don't mind lowering the resolution and bitrate.

That's about the only way to get a clip that is small enough to email. Movie Maker will try it's best to do this if you choose the "for email" output profile. Use File > Save movie and choose "for email". The file will be something like 426 x 240 pixels and at a low bitrate of about 160 kbps.

But again there are no guarantees even doing that will make it small enough.

Your best bet is to upload the video to a video sharing site (YouTube, Vimeo, OneDrive, etc.) and then email a *link* to that file. Then you can keep the HD resolution and not have to lower the bitrate immensely.

xb1r athena say:

Change Wireless Channel

Login Information

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

 

 

Change Wireless Channel

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the gateway address “192.168.1.1” into the web address bar.
  4. When prompted for a username and password type in “admin” for both the username and the password.
  5. On the left hand side of the web page select the “Advanced Settings” option.
  6. Select the “Wireless” subcategory below.
  7. Under the “Wireless – General” Section select the “Channel” drop down box and select the desired channel.
  8. Click the “Apply” button on the bottom of the page.

a. user say:

Hi Ivan,

Thank you for posting your query on Microsoft Community.

I would suggest you try the following steps and see if it helps.

Method 1: Run Windows Updates Troubleshooter 

1. Press Windows key + X and click on Control Panel.

2. Change View by from Category to Large Icons/Small Icons.

3. Click on Troubleshooting and then select View all.

4. Select Windows Update.

5. Follow the on-screen instructions.

Method 2: Manually reset Windows Updates
If method 1 doesn't help, you may try to manually reset Windows updates. Follow the steps below.

1. Press Windows key + X and click on Command Prompt (Admin).

2. Stop the BITS, Cryptographic, MSI Installer and the Windows Update Services. To do this, type the following commands at a command prompt. Press the “ENTER” key after you type each command.

  • net stop wuauserv
  • net stop cryptSvc
  • net stop bits
  • net stop msiserver

3. Now rename the SoftwareDistribution and Catroot2 folder. You can do this by typing the following commands in the Command Prompt. Press the “ENTER” key after you type each command:   

  • ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
  • ren C:\Windows\System32\catroot2 Catroot2.old

Note: Do not rename the Catroot folder, you need to rename only the Catroot2 folder

4. Now, restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt. Press the “ENTER” key after you type each command. 

  • net start wuauserv
  • net start cryptSvc
  • net start bits
    net start msiserver

5. Close command prompt and check for updates again.

Hope this helps. Please respond if you have further related queries.

Regards,

Jesinta Rozario
