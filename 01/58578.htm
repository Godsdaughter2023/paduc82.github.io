SafetyNet allows for safe and worry-free configurations and can also be useful for troubleshooting. The actions for which undo information is stored can be found and edited on the SafetyNet tab of the Options window.

How do I use SafetyNet?

Accessing SafetyNet
Click the SafetyNet link on the red bar menu of the main System Mechanic interface to open the SafetyNet window.

Viewing actions
The SafetyNet window displays the actions that System Mechanic has processed. To help you pick the right action, the following information is displayed for each:

Action - The type of System Mechanic action; for example, Optimize Registry.

Tool - The System Mechanic tool that performed the action; for example Security Flaw Repair.

Performed - The date and time the action was run.

Source - The area of System Mechanic that ran the action; for example, Wizard or Homepage.


Undoing selected actions

1. Select the action you want to undo:

  • To select a single action, click on the row to highlight it.
  • To select multiple actions, select the check box next to each action.
  • To select all of the displayed actions, click Select All from the Edit menu. All of the check boxes will be selected. (Select None clears all selections.)

2. Undo the selected actions:

  • If you have a single action highlighted, click the Undo this action link found on the left side of the window.
  • If you have multiple actions selected, click Undo to open the Undo menu then click Selected (X actions) where X is the number of actions you have selected.


Removing actions from the display
To help you keep the action list a manageable size the Purge option can be used to remove an action from the list. Once an item is purged from the display, you cannot undo it.

1. Select the action you want to purge:

  • To select a single action, click on the row to highlight it.
  • To select multiple actions, select the check box next to each action.
  • To select all of the displayed actions, click Select All from the Edit menu. All of the check boxes will be selected. (Select None clears all selections.)

2. Purge the selected actions:

  • If you have a single action highlighted, click the Purge this action link on the left side of the window.
  • If you have multiple actions selected click Purge to open the Purge menu, then click Selected (X actions) where X is the number of actions you have selected.


Exporting results
The Export option allows you to export results for an action to an XML file. Only a single action can be exported.

  1. Select the action that you want to export.
  2. From the Exportmenu, click the action name.
  3. A standard Save as window opens. Navigate to the desired location and click Save.
