Hi Russ, if Windows 10 was previously installed and activated on that PC, you can reinstall Windows 10 at any time, you will not need a product key, activation will be automatic and it will not cost you anything !

Please Note, when installing Windows 10 on the new drive, disconnect all other drives until the installation completes, otherwise you will have problems booting Windows 10

Once the installation completes, you can re-connect the other drive(s) and access the data on those drive(s)

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

Hi,

I'm not sure what type or problem you are having with your memory after reading your post, but the PageFile is not RAM, and is not memory...

The PageFile is a hidden System File on your OS hard drive partition that serves as a "Memory Overflow"...

If somehow you should exceed the amount of available hardware memory the pagefile is used...

Your Cached memory looks a bit high, but this would vary with what is running on the machine... I attached my memory profile from Taskmanager for comparison...

You will notice that mine is showing 2.9/31.5 Committed even though I only have 16GB... That is because half of the 32GB is the PageFile...

BTW... Your VRAM is not displayed in the Memory area... It is possible under some circumstances that your PC may use some of the VRAM, but that is not an everyday type scenario... The VRAM is normally only used by the graphics processor(s)...
