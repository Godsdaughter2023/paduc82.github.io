ronnie vernon say:

Hi

You can delete scheduled tasks, but it is definitely not a good idea.

All of the many maintenance routines and reporting of system status, including errors or potential destructive behavior come from the Task Scheduler.

 

If you have deleted all of the default system tasks you may be able to recover them.

 

1. Check the Recycle Bin. If you deleted any sub-folders in the System32\Tasks folder, they should be in the Recycle Bin where you can restore them. The same is true if you deleted a task or folder from within the Task Scheduler.

Recover files from the Recycle Bin:
http://windows.microsoft.com/en-us/windows7/Recover-files-from-the-Recycle-Bin

2. Try a System Restore to recover the deleted folders. These are system folders and should be replaced if there is an available Restore Point created prior to the deletion.

 

System Restore FAQ:

http://windows.microsoft.com/en-us/windows7/System-Restore-frequently-asked-questions

 

Let us know the results.

To access the System Restore, Refresh and Reset options using the F12 option at start up, perform the following:

2. Now restart the computer by pressing the power button - IMMEDIATELY start tapping the F12 key on the keyboard until the "Boot Menu" screen appears.
