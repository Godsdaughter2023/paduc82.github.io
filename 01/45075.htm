Hi Ally Bean, it is not easy to remove a file association in Windows 10,

On the view Ribbon (tab) check the box marked 'File Name Extensions'

Close File Explorer

Change the file extension on that new file from .txt to .exe

a bamboo eater say:

Hey,

Sorry to hear your having some trouble deleting and creating a new account. I'll give you the link to the steps, as well as outline the procedure here:

    1. Go to Settings, and then select System.
    2. Select Storage.
    3. Select All Devices.
    4. Select Gamer Profiles.
    5. Select the gamertag that you want to delete.
    6. Select Delete.
    7. Select one of the following:
    • Select Delete Profile Only. This deletes the profile but leaves saved games and achievements.
    • Select Delete Profile and Items to delete the profile and the associated saved games and achievement.

theunwritten96 say:

Hello and sorry to hear that you are facing this issue, if you mean you would like to create a new Xbox LIVE account under the email address that you already have an account under then unfortunately cannot. At the minute there is currently no way to delete a LIVE account permanently but you can:

- Log into the the your original Xbox LIVE account (if you can) and change your Windows Live ID that is already associated to it, if you haven't got another one then I suggest you perhaps create a new temporary one.

You can do this by:

  1. Booting up your Xbox and connect to Xbox LIVE.
  2. On your homescreen scroll along to settings.
  3. Then select 'Account'. Your 'Account Management' screen should the load.
  4. Scroll along the until you find the option 'Account Security' and select that.
  5. From there you can change the Microsoft account that is associated with your gamertag.

- You then should be able to create a new LIVE account with your original email address that used to be associated with  your old gamertag.

If you need anymore assistance please do not hesitate to post back.
