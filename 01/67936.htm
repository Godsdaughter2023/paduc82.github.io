Easy...the freeware IrfanView can combine photos
Edge To Edge (vertically or horizontally)

Restore point before installing software or updates)

Download IrfanView
http://www.software.com/irfanview-9337-1

Get the plug-ins too.....

IrFanView Graphic Viewer PlugIns
http://www.software.com/irfanview-plugins-9374-1

See the following instuctions (it's easier to do than it is to explain)

Open IrfanView and go to...Image / Create Panorama Image...
(this will open the "Create panorama image" screen)

On the "Create panorama image" screen...left click the "Add images" button.
(This will display the "Open" screen. Now...drill down to the
folder that contains your saved Photos of the Vets.

Now...left click (highlight) the two images you wish to join.
(you can multiple select if you hold your Ctrl key)
 
With the two images highlighted...left click the "Open" button.
(Or...you can add the images one at a time...which ever is easier)

Now you should be back to the "Create panorama image" screen
and the file names of the selected photos should be in the
"Input Images" field.

Now with the two file names in the "Input Images" field...
you can left click the "Create Image" button.

(the left and right positions of these images can be reversed by
selecting a file name and using the "Move Images Up" / "Move
Images Down"...buttons)

Now you should see a display of the combined image.
Go to...File / Save As...

Choose a save location / Enter a file name / Choose a format...
Left click the "Save" button...

Hi Vincent,

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. You must be logged on as an administrator to perform these steps. If you aren't logged in as an administrator, you can only change settings that apply to your user account:

a. Press Windows key + R and then type “regedit” without quotes and press ENTER.

If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

b. Locate and click the key or sub key that you want to back up.

c. Click the File menu, and then click Export.

d. In the Save in box, select the location where you want to save the backup copy to, and then type a name for the backup file in the File name box.

e. Click Save.

 

The registry location for Sound settings is: HKEY_CURRENT_USER\ControlPanel\Sounds

 

You will be able to hide the settings for changing the volume. Follow these steps:

Step 1:

Hide the Volume icon and notifications from System Tray.

a.     Right click on the Task bar.

b.    Click on Properties.

c.     Click on Customize next to Notification Area.

d.    Next to Volume, click on Hide con and notifications.

e.    Click on OK and again OK.

 

Step 2:

Hide the option of adjusting the system volume through Group Policy:

a.     Press Windows key + R on the keyboard.

b.    Type “Gpedit.msc” without quotes and press Enter.

c.     Under User configuration, Expand Administrative template.

d.    Click on Control Panel.

e.    On the right hand side, click on Hide specific control panel items.

f.      Next to Supported on it should be At least Windows 2000.

g.    Click on Enabled.

h.    Under Options, click on Show.

i.      Under Value, type “Microsoft.Sound” without quotes and click on OK.

j.      Click on Apply and OK.

How do I import my Dance Central songs into Dance Central 2?

   Launch Dance Central 2 game on your Xbox console.

   Make sure you are signed into an Xbox LIVE enabled game profile.

   From the main game menu, select OPTIONS.

   Select REDEEM CODE.

   Enter the 16 digit code printed on the back of your original Dance Central game manual. If you pre-ordered Dance Central 2 at a participating retailer, enter the token number printed on the digital receipt you received with the game (you might have this, you might not! If not, move on!)

   Follow the on screen instructions to purchase the Dance Central Import Pack.

   If you have a 25-digit pre-order code, follow the instructions you received to redeem that code for the 400 point value.

   Song files will begin downloading automatically and will be added to your song library upon completion.

Q&A

Can I import my entire Dance Central song library?

   Yes, you can import the entire on disc soundtrack from the original Dance Central.

   Your previously purchased Dance Central downloadable content will automatically be available in your Dance Central 2 library.

How much storage space will it require on my Xbox hard drive?

   You will need to have an Xbox with a hard drive or USB storage device and approximately 200-250 MB of free storage space.

Do I need an Xbox LIVE subscription?

   You will need to have an Xbox LIVE Silver (free) subscription and internet access to download the import token.

What happens if I purchase a used copy of Dance Central? Will the import code work?

   Import codes are unique to each game manual included with the original Dance Central and are one time use. Consumers will need to purchase a new retail copy of Dance Central to take advantage of the import functionality.

Hi MIchael

You can try resetting Windows 10, or clean install Windows 10 . . .

Do this twice

Go to Troubleshoot - Advanced Options - Reset this PC
Choose to keep your files and apps, just your files or to keep nothing
Then start the Reset

2.

to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
