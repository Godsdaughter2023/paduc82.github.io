Original title: Printer

 

This Canon I believe that it is an IP 2700  will only print using my color cartridge, not the black and white.  My daughter is writing a report and it will only print the picture attached to it but not the words in the report.  Help!

winston m say:

Hi,

 

 

Thank you for posting the question on Microsoft Community Forums.

 

As per the description, you are unable to print the words in the report. Let's analyze and identify the cause of the issue.

 

Which is the operating system installed on the computer?

 

Let’s first check if the color cartridge/toner is not empty or blocked.

 

Open your printer driver's Properties dialog (see above) and look for a color option that uses black only.

 

Uncheck it or select an option that enables color. Some color printers have different resolutions for color and black-and-white. Some applications will force a printer to use a quality mode that affects this option, and if the resolution is forced to its highest the color output may be disabled.

 

Also,

 

1. Enable color as gray scale.
This check box displays for color printers.

 

2. Use simplified colors.
This check box displays for black-and-white printers.

 

If you are seeing the first checkbox and you are using a color printer, then checking the box should tell Origin to send all color information and let the printer handle any conversions that may come up. In earlier versions of Origin there was a rather cryptic "Print color on black and white printers" checkbox which did the same thing.

 

 

Refer:

http://windows.microsoft.com/en-US/windows7/Change-color-management-settings#TopOfPageTarget

 

 

Change an ink or toner cartridge

http://windows.microsoft.com/en-US/windows-vista/Change-an-ink-or-toner-cartridge
