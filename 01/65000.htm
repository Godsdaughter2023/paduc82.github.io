naman r say:

Hi,

 

AutoPlay is a Windows feature that lets you choose which program to use to start different kinds of media, such as music CDs, or CDs and DVDs containing photos. There are several reasons which might be causing this problem.

 

a) AutoPlay might be restricted by Group Policy.

b) It could be canceled by a program that is running.

c) The service that AutoPlay needs to start might not be running.

 

Try these set of troubleshooting steps and check if they help in resolving the issue.

 

Method 1

a. Type "services.msc" in the start search box, click on OK.

b. When the Services window loads, navigate to the "Shell Hardware Detection" service, the double click on it to open it.

c. Make sure the "Startup Type" says "Automatic". If it doesn't, use the drop down menu to change it.

 

Method 2

Let’s follow the steps given in the section  AutoPlay doesn't start when I insert a disc or plug in a device’  from this link.

 

Troubleshoot AutoPlay problems

 

Method 3

 

You may also hold down the Shift key when you insert devices or media into your computer to open AutoPlay, regardless of the default setting.

 

Additionally, as you have mentioned auto play is enabled in control panel make sure that in the list next to the device or media, ‘Ask me every time’ option is selected from the dropdown list for all  device & media.

 

Refer:

AutoPlay: frequently asked questions

 

Hope the information helps. Please post back and let us know.
