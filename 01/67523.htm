Thank you for posting your query in Microsoft Community.

I understand that you need to uninstall an already installed update in order to install a new one (Optional). Ideally, updates that you receive through Optional updates, are not actually related to Windows security updates. Optional updates are related to any hardware or third party software installed in the computer. That is when you receive updates related to that hardware/software as an optional update.

This (Optional update) could create conflict between the one (optional) needs to be installed with the one (already installed). Hence, you may have to uninstall the one already installed in order to install the new one.
