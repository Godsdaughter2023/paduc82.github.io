Hi

In Windows 7, some of the default user folders that you saw in XP have a different name and location.

The name and location for the old Application Data folder is now  C:\Users\Ron\AppData\Roaming.

You will still see an  Application Data  folder in Windows 7, but this is not a real folder. It is called a  Junction  and it is only there for backward compatibility. Older XP programs that are hard coded to look for that old folder name will be redirected to the new location (Roaming).

All of these Junctions have the permissions set to  Everyone: Access Denied. If you attempt to change the permissions on these Junctions, you will see some unexpected results, including some programs that may quit working or refuse to install. This is why you will see either a Lock or Shortcut Arrow on these Junctions.

You will see many of these Junctions in Windows 7, including:

Documents and Settings
Local Settings
Etc.

NTFS Junction Point
