How To Find Wireless Passphrase (WPA/WEP)

  1.  Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. Your passphrase will be listed as Key 1 or Pre-Shared key

 

Changing Wireless Security Type/ Disabling Wireless Security

  1.  Login to the router by entering the default gateway IP address.
  2. The router will then prompt you to enter the user name and password.the default user name and password are: "admin", "admin".
  3. From the menu along the top select Interface Setup.
  4. From the sub-menu select Wireless.
  5. Use the Authentication Type drop down to choose your desired security
  6. If WEP is selected, select the WEP Key Length (either 64-bit or 128-bit).  The password will need to be 10 charecters in length (if 64-bit) or 26 characters in length (if 128-bit) and can only use 0-9 and A-F.
  7. If WPA is selected use the Encryption drop down to select your desired cipher type (TKIP preferred).  The password can be between 8-63 characters, is case sensitive and can use any character on your keyboard.
  8. Select Apply after inputting your desired passphrase.
