Hi John,

Could you please do the following to put the machine in a Clean Boot state.

  1. Hit Windows key + R to launch the Run command and then type msconfig and click OK.
  2. Click the Services tab, checkmark the box bottom left to "Hide all Microsoft services" and then click "Disable all" on the right.
  3. Next, click the Startup tab and then disable each item. The "Disable" button remains inactive until you click an item.
  4. Click the OK button and then restart the machine.
  5. Put the computer to sleep for how ever long it usually takes for the issue to arise and then wake it up.

If the connection works normally in a Clean Boot state, then the problem lies with a third party application or driver.

To return the system to normal startup, run msconfig again, click the General tab and checkmark "Normal startup", click OK and then restart.

Launching the Run command is a shorter method of accessing Msconfig than the method illustrated in the article, but you can do either.
