The reason for the issue to occur can be due to SNMP (Simple Network Management Protocol). SNMP manages devices on a network. Some Printers may not support this protocol, because this obtains the printer's status regardless of whether or not they support SNMP, although it only supplies simple status information for those devices.

I suggest you to disable the SNMP option from the printer properties. To do that please follow the steps below:

  1. Open Control Panel.
  2. Select Hardware and Printers option.
  3. Right click on the Network printer (your printer).

  5. Go to the Ports tab and choose Configure.
  6. Uncheck SNMP Status Enable option.
  7. Restart the computer and check if the issue persists.

How do I reformat a mac HD to NTFS for use with Windows 8? - Microsoft Community

robert k m say:

How do I reformat a mac HD to NTFS for use with Windows 8?

Hi, I have a Seagate external USB hard drive that I formatted for use with a Mac computer. How do I reformat it to NTFS for use with Windows 8?

a. user say:

Hi Robert,

 

 

The NTFS file system provides better performance and security for data on hard disks.  You can try the following steps to convert the hard disk to NTFS format.

 

 

a.    Press Windows key + X and select Command prompt(admin).

b.    At the command prompt, type convert drive_letter: /fs:ntfs, where drive_letter is the letter of the drive you want to convert, and then press Enter. For example, convert E: /fs:ntfs would convert drive E to the NTFS format.

c.    Type the name of the volume you want to convert, and then press Enter. You must use the existing name of the volume or the conversion will be canceled. You can view all available volumes in the Computer folder under Hard Disk Drives.

d.    Restart the computer.

 

 

 

Note: After you convert a partition to NTFS, you cannot convert it back. Although the chance of corruption or data loss during a conversion is minimal, you should back up all data on the partition before you begin.

 

 

Hope this helps.
