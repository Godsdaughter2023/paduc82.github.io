Hi,

Windows 10 can in fact be installed to a "GPT" disk... It is actually a requirement for newer machines to use UEFI Bios.. There is a trick to getting it to work though... The Installation Media needs to be properly created, and a possibility a "UEFI" boot device may need to be selected in Boot Options...

A lot of times the drive needs to be imaged\cloned\migrated using bootable media to get around some of the issues you mention... Some of the "Free" versions have limitations...

The problem is that most Imaging\Cloning programs require installation to create the bootable media, although if you search hard enough it may be possible to download bootable media that someone else has created from their installation...

A quote from Microsoft FAQ'a concerning GPT Drives...

Can Windows Vista, Windows Server 2008, and later read, write, and boot from GPT disks?

Yes, all versions can use GPT partitioned disks for data. Booting is only supported for 64-bit editions on UEFI-based systems.

End Quote...

The Installation media needs to be created for a UEFI\GPT machine for the GPT drive to be recognized...

A "UEFI" Boot Device needs to be selected as a Boot Option when booting with the installation media...

Some machines, because of their Bios, will not recognize a DVD as legitimate UEFI installation media...

The best way is to use a Flash Drive created with the Windows 10 Media Creation Tool...

Just as a note... A Flash Drive for UEFI Bios needs to be a GPT\FAT32 drive...

A Flash Drive for Legacy Bios needs to be a MBR\NTFS drive...

Hello tankerus,

in most cases this error code 10 means

1. that you have not the latest version of the driver, 2. the installed driver is not compatble with your windows version or 3. that a windows update is missing.

So first use the "windows update" from "Start=>ControlPanel=>All Programs" to check for the latest windows updates.

If thats ok, use the device manager to get some more iformation about what is the problem:

Click "Start=>ControlPanel=>DeviceManager" and look for the the TPM chip in the list, right click on it and click "properties" and look in "common" for some messages.

You can use the device manger also to check for updates but in most cases this won't do anything, so that you have to look for the latest and correct drivers on the website either of the vendor of your PC (Laptop) or of the vendor of the TPM chip which might be Atmel, Infineon etc. I recommend to try the Pc vendor first espacially if it is Laptop!

Make sure to download 1. the latest driver and 2. the corresponding driver for your windows version and 3. for the correct PC (Laptop) model!

Uninstall your TPM driver by using the "Device Manager" before installing the new driver!

Try this and see if it solves your problem.
