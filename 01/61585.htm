Follow these steps to do so.


a. Press Windows key + R to open the Run Dialog.

c. Look for a folder called AUInstallAgent. If it is missing in the Windows folder, then right click on an empty space and click on New folder.
d. Rename the folder as AUInstallAgent.

e. Check for the presence of C:\Windows\AppReadiness - if it doesn't exist, create it.

f. Restart the Computer and check if you are able to install apps.

If right clicking the Start button didn't work, you may have some corrupted files on the machine. To test that, hit Windows key + X (which gets you the same menu as right clicking the Start button) and then click "Command Prompt (Admin)".
