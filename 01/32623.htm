There are two kinds of Restore Points:

Event-Triggered restore points

Automatic restore points

https://msdn.microsoft.com/en-us/library/windows/desktop/aa378910%28v=vs.85%29.aspx?f=255&MSPPError=-2147217396

You can disable (or enable) Automatic restore points but I don't think you can disable Event-Triggered restore points.  I guess Windows is trying to keep you from shooting yourself in the foot by not having a way to undo some installation of batch of updates that might not work properly.  If that were allowed then somebody would complain that there is no way to undo a botched update installation.

In Task Scheduler is you right click the choose to disable the task named "SR" there is no "Finish" to click and if you again right click the task named "SR" there will now be an Enable option.  There will be no Disable option because it is already disabled.
