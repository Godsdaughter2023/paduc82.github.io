Hi Marvin,

Touchpad issues can be caused by an outdated driver.

If updating the driver did not resolve the issue, try to uninstall it and see if the issue will persist:

  1. Go to Device Manager.
  2. Right-click on the touchpad driver and select Uninstall.
  3. Restart your device and Windows will reinstall the driver.

If that didn't work, try using the generic driver that comes with Windows:


  1. In Device Manager, right-click (or press and hold) your touchpad driver.
  2. Click Update driver software.
  3. Select Browse my computer for driver software.
  4. Choose Let me pick from a list of device drivers on my computer.
  5. Select HID compliant mouse.
  6. Click Next, and follow the instructions to install it.
