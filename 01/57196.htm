Organising the All Apps Menu? - Microsoft Community

jus and windows say:

Organising the All Apps Menu?

Is it possible to organise the apps in the "All Apps" menu? I really hate how some desktop apps are under folders like Windows System, and Windows Accessories, and I would like to know if I could move those out of that folder. If I could, is it safe, because I have a computer that I need to use everyday, and I can't afford losing it.
cmdrkeene say:

You sure can -- it's actually nearly identical to previous versions of Windows (at least, all the ones that had Start menus).  You see, the Start menu is just a regular file folder full of shortcuts.  You can open that folder and add or remove shortcuts however you like, with one exception:  apps from the Windows app store will always show at the top level of the list, and can't be put into subfolders.

One other note:  This isn't new, but it's not common knowledge.  The Start menu list that you see when you click "All Apps" (or "All Programs" in older versions) is actually a combined view of two file folders:  your start menu shortcuts, plus the "all users" start menu shortcuts that show up for everyone (it's used primarily by schools and businesses so they can put an icon in that folder and it shows up for everyone all at once).   The shortcuts you want to move around and customize may be in either location.

So, here's how to get started:

  1. Hold down Windows logo and type R to open the Run box.  Copy and paste in the following line and click OK.
    This is your personal start menu folder of shortcuts.

    %appdata%\microsoft\windows\start Menu


  2. Open the Run box again (Win+R) and copy this line and click OK.
    This is the folder of shortcuts that shows for "all users", and only an administrator can make changes here since it impacts everyone that uses the computer.

    %programdata%\microsoft\windows\Start Menu

At this point, you should have two folders open.  Now simply re-arrange the shortcuts in these folders however you please.  Feel free to make new folders too.  Remember, anything in the 2nd folder will show up for everyone that logs into the computer.  If you want an icon to only show for yourself, use the 1st folder.

mouneshwar r say:

Hi Jeff,

 

Choosing the option “Turn on automatic unlocking for this drive on this computer” would not unlock the drive. You need to turn off the bitlocker for your C: drive and then decrypt the drive in order to access the drive content. In other words, fixed data drives can be automatically unlocked after the operating system drive has been unlocked.

 

Note: Fixed data drives can be configured to automatically unlock when the operating system drive is encrypted, to unlock after a password is supplied, or to unlock after a smart card is inserted. Turning off password for that particular drive will still make the drive inaccessible.

 

Steps to turn off bitlocker encrypted drive

a.    Open Bitlocker Drive Encryption by clicking the ‘Start’ button, clicking ‘Control Panel’, clicking ‘Security’, and then clicking ‘Bitlocker Drive Encryption’.‌ If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

b.    To turn off Bitlocker and decrypt the volume, click ‘Turn off Bitlocker’, and then clickDecrypt the volume.
