hairyfool say:

No Bootable Device is a BIOS message not Windows.

You need to go into BIOS setup and check that it is seeing the reinstated drive, the message is anindication that Power On Self Test (POST) has completed but can find no drive to hand the boot to. This also demonstrated that the BIOS cannot see the optical drive either.

It is highly likely that Windows installed on a drive in one system will not run on another but if that was the case it would normally generate BSODs. At worst if the laptop could see the drive but could not boot from it it would display Missing Operating System.

slance310 say:

Hi,

You're machine should have a F key to enter UEFI Bios at boot or restart...

It should also have a key to access the one time boot options, where you could choose your Flash Drive... This only works once... If you want to boot with the Flash Drive frequently, you need to enter the UEFI Bios settings and designate it as the 1st boot device...

Different machines use different keys for this... My HP Laptop requires you hit ESC to get a menu with the rest of the options...

Mt Desktop uses F2 or DEL to enter Bios, and F12 to access the boot menu...

You mat also have better luck doing this stuff if you turn off Fast Startup...

It has a way of blocking booting from something other than what it was running when it was shut down...

This feature only works when you do a SHUTDOWN and then Boot.

It doesn't effect a RESTART.

To turn it off or on...

Go to Control panel...Power Options, and select Choose What the Power Buttons Do on the left.

Then select Change Settings That Are Currently Unavailable near the top center of screen...

Lower down on the window, uncheck Fast Startup.
