Display problems probably means you will need to update the drivers in order to run Windows 10.

When a driver is faulty or not working properly, it can be identified by the yellow exclamation mark indication on the driver in Device manager.

Method 1:

Step 1: Check for faulty drivers from device manager by following the steps below:

  1. Press Windows key + X and click on Device Manager.
  2. Expand each and every driver and check for the yellow exclamation mark.
  3. Right click on the driver with yellow exclamation mark and select Uninstall option.
  4. Restart the computer for automatic installation of the drivers and check if the changes are effective.

Step 2: If the issue persist, update the display driver from Device Manager manually by following the steps below and check if the issue is resolved.

  1. Press Windows key + X and click on Device Manager.
  2. Expand Display Adapter option.
  3. Right click on the display driver and select Update.
  4. Restart the computer and check if the changes is effective.
