Some OEM disks from computer manufacturers don't have the repair option. But you can make a repair disk:

 

Creating a System Repair and Boot Disc Disc in Windows 7 (if you can't boot into Windows 7 use another computer with Windows 7)

   1. Click on Start -> All Programs -> Maintenance.
   2. Click on the Create a System Repair Disc shortcut.
   3. Choose your optical disc drive from the Drive: drop-down box.
   4. Insert a blank CD or DVD in your optical drive.
   5. Click the Create disc button. Windows 7 will now create the System Repair Disc.
   6. After the System Repair Disc creation is complete, Windows 7 displays a dialog box that you can close by clicking the Close button.



Then:

Try Repairing The Windows 7 Installation (the steps might be slightly different if using the System Repair Disk)

1. Insert the Windows 7 DVD (or System Repair Disk) and restart your computer
2. Boot from the DVD.
3. Choose your language and click Next.
4. Click Repair Your Computer and then select the operating system you want to repair.
5. Click on the Startup Repair link from list of recovery tools in System Recovery Options.
