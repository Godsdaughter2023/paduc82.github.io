trevor j lawrence say:

You must understand that Store Apps (Metro Apps) do NOT behave as desktop applications now or in the past.  A Store App that is not currently displayed on the screen is in a suspended state and uses NO resources other than some RAM.  If and only if the OS doesn't have enough RAM for its other operational requirements, the App will be closed down and the RAM retrieved.  If the OS does have enough RAM, using some for a suspended app has no effect on system performance.

 

I know its a new mind-set and difficult to get used to, but it works.  In the one or two circumstances where I feel a need to close an App, Alt-F4 works if you have a keyboard.  If not a swipe from top to bottom is quick and easy.
