Hi,

The explanation of why does a hard drive storage reflect less total memory is due to the binary numbering system of Windows.

See details below:

  • 8 bits = 1 byte
  • 1 megabyte (MB) = 1,048,576 bytes
  • 1 gigabyte (GB) = 1,073,741,824 bytes
  • 1 terabyte (TB) = 1,099,511,627,776 bytes
  • 500GB = 476,837 MB (500,000,000 / 1,048,576 bytes)
  • 1TB = 953,574 MB or 953GB (1,000,000,000,000 / 1,048,576 bytes)

This concept applies to all memory storage that is readable in Windows. If your storage device doesn’t match with this computation anymore, it means that the durability of this hard disk had been already reduced. This happens overtime every time especially if the hard disk has deleted lots of files many times.
