Hi,
Welcome to Microsoft Community and we are happy to help you.

As per the query, you want to know if you can log in to the stand alone computer after you disable netlogon service on the computer.
 
I would like to let you know that Windows allows computers to interact on a network, signing onto a domain to interface for work and other purposes. These computers use the Netlogon service to log into the domain. The service runs, providing security for the link made between the individual computer and the network. It handles permissions and login requests from the network as well. Without the netlogon service, the computer cannot operate on the network.

Stopping netlogon will prevent you from running a network computer, because you cannot log onto the network. You use the Internet or other programs linked to the network. jeeped say:

It is my understanding that you wish to essentially strip the xyz123_ portion of the filename from all files within a folder, leaving the remainder of the filename. I do not believe that you can use the RENAME command directly at the command prompt for this task. Some form of variable assignment and loop will have to look at each individually and parse a new filename by stripping the first 7 characters.

I would normally use a PowerShell or Visual Basic Script for this but those might require installation of some back-end compiler software on your system. Additionally, you have specifically asked for a RENAME based solution so here is a .CMD batch file that fits the bill. A .CMD file is for all intents and purposes a .BAT file; it is a text based file that runs a series of commands from a command prompt and if you would prefer to use the .BAT file extension, you can.

Open NotePad and paste the following into a new text document.

@echo off
setlocal enableDelayedExpansion
for %%F in (XYZ123_*.WAV) do (
  set "name=%%F"
  rename "!name!" "!name:XYZ123_=!"
)

Save the file in the folder with the xyz123_*.wav files as my_rename.cmd (or my_rename.bat). You will need to change the Save as

Open a command prompt to the folder that contains the file and run the batch file. Your rename operation should complete as expected.
