cmdrkeene say:
The taskbar will not hide if any running program has requested the "need attention" state.  Programs that typically do this include instant messengers, email or calendar reminders, or antivirus programs.  You can look for any running program that is highlighted.  Or, if that is not the case, you can check near the clock for any icons that are needing attention.
