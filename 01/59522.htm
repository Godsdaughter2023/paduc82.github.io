Hi Martyn

This is a well reported issue with the System Restore process in Windows 10

You have no option but to perform a hard shut down, then see if your system boots correctly . . .

Press and hold the Power button on your PC for 5 - 10 seconds to perform a hard shut down

Then boot your system up to see if it will successfully boot

If not, try running another System Restore operation, to see if that completes . . .

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

How do you create 2 different scheduled backups under Window 7 x64 or x86 pro

kenmcc1 say:

To create 2 backup tasks under windows 7 (both x86 and x64) Pro (and higher, I don't know about more basic versions).

>>Use Windows 7 "Backup and Recovery" to schedule the Image backup for 1 time every week on a day you select. Note: I only checked the small box at the bottom for include Image and did not check the box for "C:" drive as this should already be included. You (should) will find the link for setting the schedule day and time. You can run this config using "Backup Now" and see the results in the "D:" drive in directory/folder "WindowsImageBackup" (in my case about 33 GB). 

>>Ok, now that you have configured the Image backup with a schedule the next steps are: 1. Using Windows Explorer go to C:\Windows\System32\Tasks\Microsoft\Windows\ and create a new folder (right click New ) "WindowsImage". 2.  Navigate (click on) “WindowsBackup” and then copy the File “AutomaticBackup” to “WindowsImage” folder.  3. Rename this File in “WindowsImage” to “AutomaticImageBackup”. 

>>Next Steps: 1. Start "Task Scheduler" and naviate to \Microsoft\Windows\ and then on menu click "Action" and select "New Folder" and enter name "WindowsImage". 2. Click on the new Folder "WindowsImage" and then go back and click on "Action" and select "Import Task" an navigate to C:\Windows\System32\Tasks\Microsoft\Windows\WindowsImage change file type to *.* and select AutomaticImageBackup click OK/Select. 

>>Now you have the Image backup you want finish configuring the Task. 1. Click on General in the middle screen and then in the right hand screen click on "Properties". 2. Under General Tab input the Description you want to describe this Task. 3. Under Trigger Tab use Edit to set the schedule you desire (make sure "Enable" box at the bottom is checked.  4. Under Conditions Tab ( I selected) check the 2 boxes under "Power". Click OK and you are finished with - Image Backup with a schedule-.

>>For the “data/user/files/“ backup setup; Go back and use Windows 7 "Backup and Recovery" to schedule. 1. Select “Change settings”, then select “D:” drive and “Next” button. 2.  Select “Let me choose” (I did; you could Let Windows choose) and “Next” button. 3.  Expand the “C:” drive and select all the Folders you want backed up daily. (I choose to have my own Folders for Data Storage as this computer shares data/applications with 3 other users). I selected  Data (contains all common data for all users), Pictures (same reason), QK (Quicken shared users), Users (all user files on computer are under this folder; including “Data Files” listed first). 4.  Make sure the box “Include a system image of drives –etc” is not selected and click on “Next”. 6. Select “Change schedule” in the middle, check the “Run backup on a schedule and select (I do Daily) at 11:00 P.M. (Note: if you want to select the minutes you must go back to Task Scheduler) and click OK.   7. Now “Save settings and exit” and you are finished. 

Note: under” Manage Space” it will say your Image schedule is not set, but this is because it cannot handle cleanly 2 different scheduled backups. You (the Administrator) must keep track of how much file space is being used on “D:” drive and delete only files. I am still learning on this part. 

All the above " " should not be included as they are for highlighting!

Windows 8 Apps are built under an App system similar to the ones with Smart Phones and Mac
