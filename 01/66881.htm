tedmittelstaedt say:

This is an old problem with updates.  The troubleshooting steps above usually don't fix it.  Often this happens when you are building a system from scratch.  I can get it to re-occur for example by installing Vista, then SP1 for Vista, then Windows 7 Upgrade, then SP 2 for Windows 7, without allowing the system to download and fully patch each minor version before moving on to the next version."

I think the root cause is parts of the system being updated to newer versions while other parts have original versions of files.   The "by the book" way of doing Windows installs is to install Windows, then set it to "download and install windows updates automatically" every night, then go into power management and make sure the system doesn't go into hibernation at night, then leave the system turned on for a couple weeks and ignore it.  It will work away by itself and eventually get everything updated.   But most people who build systems from scratch want to get them completely patched before releasing them to users so they will manually update them.

One thing you can try doing under Windows 7 is:

If Windows Updates says "You receive updates For Windows Only" then click the  Find out More in  "get updates for other Microsoft products" and install the active x control, then rerun updates and if it fails again then go into Change Settings under Windows Update, and uncheck the box for "give me updates for other Microsoft Products

You are basically turning Other Updates on then off.  That will sometimes cause it to start working.

Once all updates have been installed then Windows Updates works a lot better and you can turn on the updates for other Microsoft products.  No doubt because there are updates to Windows Updates itself.

Couple other quirks - right now win 7 has something like 200 updates.  Do not try to install all the updates at once on a fresh system install them in groups of 40.  Later updates are dependent on earlier updates and Windows Updates will happily install 5 updates then fail the remaining 195, then reboot and install another 20 or so updates then fail the remaining 175 and so on.

Also, the download and install percentage bars have no bearing on any reality of how far along the update is in installing.

If the system goes into a "checking for new updates" then leave it be.  I've see a Core2 with a new load of windows 7 take 3 days to finish that before coming back with an update list.

If the system seems to hang during shutdown in between installing updates, leave it alone.  I've seen that take 3-4 hours or longer before actually finishing the shutdown.

If you can, manually download and install the very latest network drivers for your system before running windows updates for the first time.  Note that with AMD chips and the integrated network card the latest drivers may require most of the system to be updated first.

Lastly, if your system is not on a power UPS, get one.  Disorderly shutdowns can trash updates.

How to do a repair upgrade:


A repair upgrade is the simple process of installing a new build over the existing installation of Windows 7, using your installation DVD. What this does is replace broken operating system files while preserving your personal files, settings and installed applications.

Let us take a look at doing that.

Note: If you initially installed Windows 7 from a RTM disc (when it was just released), then updated to Service Pack 1, you won’t be able to use the RTM disc to do the repair upgrade because the version installed is newer than what is on the disc. The work around for this is to uninstall Service Pack 1.

To do that:

Click Start, then type: view installed updates.

Hit Enter on your keyboard.

Search for KB976932.

Select it and click Uninstall.

After your computer restarts, your computer will be restored to the RTM release of Windows 7. If you have not updated to Windows 7 Service Pack 1, you can skip this section. Another option: If you can obtain a Windows 7 retail disc that has been integrated with the SP1 code that matches your currently installed edition, then you can just use that instead.

Doing the Repair Update

The repair upgrade is a straightforward process. Simply boot to your Windows 7 desktop. Insert your Windows 7 installation DVD, then click Install Now.

When the setup menu appears, click Upgrade.

Setup will begin copying files. This will also replace any broken operating system files that might be preventing Windows from functioning properly.

There you have it: your personal files and installed applications are preserved. I found this especially handy for fixing corrupt operating system features. For example, I had disabled some features in Turn Windows Features on or off. When I tried to re-enable them, they would not work. Repair upgrade was able to fix this.

Of course, this means you will need to reapply any updates, since it basically replaces certain applications. It is an inconvenience, but achieves a functioning system if all else fails.

canadian tech say:

Wireless mice are more trouble than it is worth putting up with.  First recommendation is to use a wired one. 

Wireless mice are common problems because they depend on a radio signal between it and a receiver device.  If either device has weak batteries, you will have a problem.  The radio signal between the devices is very weak and depends on being quite close and most importantly no radio signal interference.  Signal interference can come from a lot of sources, but the computer itself can be the worst offender.  Try moving the receiver quite close (a few inches) to the mouse and see if that helps.

In your case, it sounds like interference.
