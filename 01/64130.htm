Upgrading from Home Basic to Home Premium

I purchased Windows 7 Home Premium Upgrade in the US. I bought a computer while traveling in Europe. It had Home Basic. Can I use this Windows 7 Home Premuim Upgrade to upgrade from Home Basic to Home Premium? What are the potential problems?

 

andre da costa say:

Is the computer you bought in Europe running Windows 7 Home Basic or Windows Vista Home Basic?

Upgrade paths:
When making significant changes to your computer such as an upgrade, it is always recommended you backup your system prior to installing a new version of Windows. In your case, you can upgrade to Windows 7 depending on edition and architecture of Windows Vista you have installed. If you are running a 32 bit version of Vista and want to upgrade to a 64 bit version (edition of Windows 7), you will have to a do clean install, this applies vice-versa.

If you are running consumer editions of Windows Vista such as Home Basic or Premium, you can only upgrade to either Windows 7 Home Premium or Ultimate. Also, you cannot do downgrades, which means, if you are running Windows Vista Ultimate now, you cannot downgrade to Windows 7 Home Premium. You must do logical upgrades:

Windows Vista Home Basic > Windows 7 Home Basic, Premium or Ultimate
Windows Vista Home Premium > Windows 7 Home Premium or Ultimate
Windows Vista Business > Windows 7 Professional, Enterprise or Ultimate
Windows Vista Enterprise > Windows 7 Enterprise
Windows Vista Ultimate > Windows 7 Ultimate.

  Upgrades to Windows 7 from the following operating systems are not supported:

·      Windows 95, Windows 98, Windows Millennium Edition, Windows XP, Windows Vista® RTM, Windows Vista Starter, Windows 7 M3, Windows 7 Beta, Windows 7 RC, or Windows 7 IDS

·      Windows NT® Server 4.0, Windows 2000 Server, Windows Server® 2003, Windows Server 2008, or Windows Server 2008 R2

·      Cross-architecture in-place upgrades (for example, x86 to x64) are not supported.

·      Cross-language in-place upgrades (for example, en-us to de-de) are not supported.

·      Cross-SKU upgrades (for example, Windows 7 N to Windows 7 K) are not supported.

·      Upgrades from Windows Vista to Windows N, Windows K, Windows KN, or Windows E are not supported.

·      Cross-build type in-place upgrades (for example, fre to chk) are not supported.

·      Pre-release in-place upgrades across milestones (for example, Windows 7 RC to Windows 7 RTM) are not supported.
For more details check out the link http://technet.microsoft.com/en-us/library/dd772579(WS.10).aspx

One thing I should note, you should be able to use the Windows 7 Home Premium key to do an Anytime Upgrade from Windows 7 Home Basic to Windows 7 Home Premium. Click Start, type Anytime Upgrade, follow the onscreen wizard, enter the Windows 7 Home Premium key when requested, click and restart your computer when prompted. Within 10 mins, you should be upgraded to Windows 7 Home Premium and your files and settings preserved.

mark ole say:

Hi,

This issue may occur if the device is set to go to sleep mode when the lid is closed.

To resolve this, go to power settings and make sure that the WiFi adapter remains enabled at all times. Go to Control Panel > Network and Sharing Center > Change adapter settings > right click on connected WiFi network > Properties > Configure > Power Management > uncheck Allow computer to turn off this device to save power.

Also, follow these steps:

  1. Press the Windows Key button on your keyboard

  2. Type Control Panel

  3. Click the Control Panel icon in the top-right hand corner

  4. Click on Power Options

  5. Under the selected power plan you have configured, click on "change plan settings"

  6. Click on Change advanced power settings

  7. Under the "Advanced Settings" tab, please click the [+] button next to Power buttons and lid

  8. Click the [+] button next to Lid close action

  9. Change "On Battery" and "Plugged in" to "Do nothing" instead of sleep

Let us know if you need further assistance.
