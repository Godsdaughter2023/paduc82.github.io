lemp say:

Here are the instructions from The Print Shop User Guide:

To install from a retail disc:

  1. Insert the disc into the DVD drive in your computer.
  2. The installation should begin automatically or give a pop-up asking you what you would like to do.  Select Run Setup.exe.
    Note: If the installation does not begin automatically follow these steps:
    1. Click on Start.
    2. Click on Computer
    3. Right Click on the DVD drive containing The Print Shop 4.0 disc.
    4. Left Click on Open.
    5. Double Left Click on Setup.exe.
  4. The installer will now search your computer to make sure you have the required Windows Updates in order to install The Print Shop 4.0.
  5. If it detects that you are missing an update the installer will prompt you to install these now.  Once the updates are complete the installer will move to the next step automatically.
  6. From here you should follow the on screen prompts to complete the installation.
