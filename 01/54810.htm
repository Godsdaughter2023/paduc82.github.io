a. user say:

Hi,

Thanks for posting your query on Microsoft community.

We understand the inconvenience caused and we will try our best to resolve the issue.

We appreciate you for trying the troubleshooting steps.

This issue might occur when the graphics card is not supporting the resolution which you are trying to set.

  

I suggest you to try setting the screen resolution through Windows registry and check if it helps.

Follow the below steps:

1) Press Windows key + R, type regedit and hit Enter.

2) Locate for the below path in registry.

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Hardware Profiles\UnitedVideo\CONTROL\SERVICES\BASICDISPLAY

3) There are multiple keys under this folder, the entries you want are named:

DefaultSettings.YResolution

DefaultSettings.XResolution

4) Change the resolution to your desired resolution.

javacado say: I had this exact same problem. I traced it back to using AMD's driver cleanup utility to remove the catalyst software 13.1 before installing 13.3.  A brief investigation turned up a few threads on other message boards with others having the same issues. I don't know for certain that this is what caused your problem because you didn't mention anything about AMD software, but that was my root cause. 

To fix the issue I just used the system recovery tool to restore my system to a point before I had used the utility. Then to remove the drivers I used the instructions given here: http://support.amd.com/us/kbarticles/Pages/GPU57RemoveOldGraphicsDrivers.aspx. This removed the drivers without the adverse effects of the AMD removal tool. I'm now using catalyst 13.5 with no problems. If you don't have a good system restore point available you may have to reinstall the OS.
