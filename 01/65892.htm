My suggestion is to exit all your programs normally by using the X button or Exit/Quit under the menu. This allow programs to properly save their data which help prevent data corruption and lost.

Alternatively, you might want to try increase the time wait for a program to shut down by edit of the registry. You might want to try the steps below. Backup all your data prior to making change and read all disclaimers.

NOTE:
Modifying REGISTRY settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of REGISTRY settings can be solved. Modifications of these settings are at your own risk.

3. In regedit, go to: (See screenshot below)

6. Close regedit.
NOTE: If you have problems with programs from your computer shutting down to quickly, then repeat the above steps and increase the time (step 5) a bit.

Disable automatic restart on system failure

Safe Mode

Safe Mode with Command Prompt

Enable VGA mode

Disable automatic restart on system failure

Start Windows Normally

Disable automatic restart on system failure
