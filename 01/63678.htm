Yes, it is an issue with Win10, but it is intentional. This cannot be fixed. You cannot force wide screen in Win10.

Games which use a resolution that is not native to a wide screen format ( eg- 1024x768, 1280x1024, etc )

will not run  full/wide screen on Win10. 

Win10 screen scaling was designed to override your graphics processors scaling options, and forces the

aspect ratio that suits (native to) the resolution.

My understanding is this was done to make scaling on external devices such as phones more reliable.

- The only way to get a full screen image with low resolution is to use a low res that has a 16:9 aspect ratio

(assuming a typical desktop monitor). If the game has an option for such resolutions

(COD 2 has an option for 1280x720 which runs full wide screen).

"should i downgrade to windows 8.1 again???"

I you want or need the game wide screen are are using a resolution that is native to 4:3 or 5:4 aspect ratio...

Hello Steven,

After changing Internet Provider, your device needs to reconfigure its network settings to be able to connect to the internet. Meanwhile, here are some tips you may want to try to get rid of being offline:

  • Run the Network troubleshooter followed by networking commands.
  • Update the network adapter driver.
  • Roll back the network adapter driver if updating it didn't work.
  • Temporarily turn off firewalls.
  • Temporarily turn off any antivirus or malware-prevention software.
  • Uninstall the network driver and restart.
  • Use network reset to reinstall network devices.
