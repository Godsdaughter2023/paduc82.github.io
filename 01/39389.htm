gabe1972 say:

Unfortunately, you can't use a system image from 8 on a computer that is currently running 8.1, or vice versa.  I ran into this as well.  First, don't discard the saved Windows 8 system image because there is a sort of workaround, but only if you have a Windows 8 installation disk or Windows 8 recovery disk that came with your computer.  Just reinstall Windows 8 using the installation disk or recovery disk, then you can use the system image because it will recognize that you are using the image from the same version that you now have installed.  As soon as you are done installing Windows 8, don't even bother with anything else, including activating it, just do the system image restore.  You won't have to do the tedious task of reinstalling all of the drivers and updates and programs that you had on it before.  This is only going to work though if you have a full version of the windows 8 installation disk or a recovery disk with windows 8 that came with your computer.  If you have the upgrade version, you can still do it, but you are going to have to install whatever you started with, then upgrade to 8, then restore it with the windows 8 system image.  Yeah, you have to reinstall windows 8, but then using the system image from 8 you won't have to download all of the stuff from before. 


Now, this worked for me, but that still isn't a 100% assurance that it will work for you.  It should though, as when you go to restore it with the system image, it will recognize that you are trying to restore Windows 8 with a system image of Windows 8.  


It will take a few minutes, but it is better than the alternative.


This is probably the only thing you can do, as Microsoft made it this way so you can't use a system image from another operating system that you don't own.  Can you image if you only had Windows 7 but you were able to restore your computer with someone else's system image of 8?  Talk about a free upgrade.
