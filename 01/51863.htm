lemp say:

The "Libraries" folder is located at C:\Users\yourusername\AppData\Roaming\Microsoft\Windows\Libraries.

On the other hand, none of your files are actually in this "Libraries folder." Instead, the contents of this folder are "library descriptor files" written in XML.  Among other information, these descriptor files include the locations where the files are actually stored.  If you were to go to the Libraries folder, you would have a hard time opening the descriptor files to view the XML code, because Windows Explorer "knows" that these are "special" files and when you try to "open" them, simply shows you the various files and folders in the library.

Windows 7 Libraries are collections of files and folders.  Microsoft calls them "virtual containers for users’ content" that "aggregate content from multiple storage locations into a single, unified presentation."

See https://docs.microsoft.com/en-us/windows/client-management/windows-libraries
