lemp say:
You may be referring to the log files created when Windows "Hotfixes" are installed by Windows Update (e.g., KB961503.log).  If you have Windows Explorer configured to "Hide extensions for known file types" you won't see the .log extension, but the files will still probably open in Notepad when you double click them.

The first few lines of such files should be something like:

[KB2559049-IE8.log]
3.359: ================================================================================
3.359: 2011/08/09 23:58:28.231 (local)
