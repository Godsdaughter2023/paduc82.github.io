How to search for multiple patterns - Microsoft Community

'''''' say:

How to search for multiple patterns

If I'm searching in Windows 7 for several types of file at once, e.g. *.mp3, *.wav, what do I need to type to tell Explorer that they're alternative patterns to match, not part of the same pattern?  I've tried separating them with a comma and a semicolon but each causes it to return no results.
¡firedog say:

You're looking for files with particular attributes, in this case file extensions. The basic query syntax is attribute:value, so you search for a file with the file extension mp3 by using the search query extension:mp3. This can be abbreviated to ext:mp3.

To find files with one attribute or another, use the operator OR (always in uppercase). You are looking for files whose extension is either mp3 or wav, so the value is mp3 OR wav.

A space between elements in a search query is treated as AND.

If no attribute: element is present, Search will assume you're looking for files containing the search string(s) given in either the file properties or the file properties and content, depending on your settings.

You extend an attribute: to cover more than one argument by enclosing the values in brackets ( ), thus creating a single complex element.

Your query, then, should be ext:(mp3 OR wav)

Noel
