richard eccles (tas) say:

Hi StealthyNinja473,

I am not sure if you found a resolution as yet. I had the same issue and was able to resolve it.

Windows 10 removed the AuthenTec software for the Eikon mini finderprint reader. Windows compatibility checker removed the software automatically, but did it in a very orderly fashion.  The AuthenTec Eikon Mini driver still appeared under Biometric devices.

The way to resolve it is not to use the AuthenTec software as AuthenTec have not released updated software to my knowledge. Instead use Windows Windows Settings to add fingerprints on the Eikon Mini. To do this do the following:

1. Go to Windows Settings

2. Select Accounts

3. Select Sign-in Options

4. Select Set up under the Fingerprint heading
5. Select Get Started
6. Enter your PIN (don't hit enter) or password

7. Swipe your finger across the reader several times as prompted

8. Select Add Another Finger if you want to add more fingerprints

9. Select Close

10. Close Settings

11. Do cartwheel and rejoice, your Eikon Mini Fingerprint Reader is working again :)

Cyclic redundancy check errors normally occur from a damaged or faulty disc.

We suggest checking the drive for system file errors by following the steps below:

1. Press Windows key + X.
