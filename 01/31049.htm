Only 9 GB for the Windows folder (if that's what you have) is a remarkably small amount for Windows 7.

One place you likely haven't checked is the System Volume Information folder, although this should not be very large on a recently-installed Windows drive ... unless you created some System Restore points during the installation of your software.  The easiest way to check SVI is to do the following:

  1. Click Start, right-click Computer, and select Properties
  2. Click the "System Protection" link in the left pane

  4. Look at "Disk Space Usage" (in general, the default setting for this can be reduced quite a bit)

lemp say:

The System Volume Information folder is a "super hidden" folder.  You can access it with some difficulty, but manually deleting files from this folder will likely break some Windows functionality, at least temporarily.

In addition to System Restore points, data in the SVI folder includes file indexing information, volume shadow copy information and other Windows system data.

It's better not to mess with the SVI folder directly.

Have you rebooted since you set the space for restore points to 0 and clicked the "Delete" button under "Disk Space Usage"?

Click Start, type System Restore into the Search box, and press Enter

At the first window, click Next

At the second window, click to check the box to "Show more restore points"

How many restore points are listed?

Cancel out of this.

Although it shows 0 bytes, it's odd that you have a "WindowsImageBackup" folder on your C: drive.  You should not be using your main system drive as a backup; that defeats the purpose of a backup if you have a hardware failure.

The SVI folder on a hard drive that is (or has been) used for Windows System Images may store system image information.

Click Start, type Backup and Restore in the Search box, and press Enter

What, if anything, is shown as the "Backup location"?

If there is something configured as the Backup location, click "Manage Space"

Click "Change settings" in the "System Image" section.  Select the bottom option ("Keep only  the latest ...") and click OK
