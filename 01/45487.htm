I applied one theme 'Alienware Skin Pack 1.0' now I uninstall it , but the icon are not the default .I want to restore my folder icons,Computer icon,Start Menu icon etc.. What I need to do?

liza.z say:

Hi,

 

Method 1: Set Default Windows theme:

 

  1. Click the "Start" button.
  2. Select the "Control Panel" menu option.
  3. Click "Change the Theme."
  4. Select the "Windows 7" theme under the "Aero Themes" list.

 

 

Method 2:  Restoring the Folder pictures

 

  1. Right-Click the Folder,
  2. Select  Properties ,
  3. Go to Customize Tab,
  4. Click the Restore Default Button,
  5. Finally, Click the Apply Button.

 

Method 3: Restoring the Folder icon

 

  1. Right-Click the Folder,
  2. Select  Properties ,
  3. Go to Customize Tab,
  4. In the Folder icons Group, Click Change Icon,
  5. Click the Restore Defaults Button ,
  6. Select OK Button ,
  7. Finally the Apply Button.

 

Hope it helps.

One question:  when I install the same apps in my wife's account on the same computer, does Windows 8 actually install another copy of the apps?  It seems to be odd to install two copies of the same apps on the same computer, just because there are two users (my wife and me).  It does not k=make too much sense.


Hi

 

The apps are only installed once on each computer, regardless of how many users use the apps. The apps can be installed on any number of User Accounts on the same computer. This applies to Free as well as Paid apps.

 

The apps are installed in the C:\Program Files\WindowsApps folder.

 

However, when an App is added to another User Account, a folder for the app is created in the C:\Users\user name\AppData\Local\Packages folder. This folder contains the Configuration Files for the app.  This allows the App to load the Personal Settings and Usage Statistics for each individual user.
