erika aro say:

Hi Howard,

Error 0xC000000E is a boot manager error due to a misconfiguration in the Boot Configuration Data (BCD) registry. To repair the boot manager, you'll need to boot from the original DVD or USB installer of Windows 10 or Media Creation Tool. If you don't have one, you can create a Media Creation Tool by following this link. After that, follow the steps below:

  1. Boot your computer using your bootable DVD or USB installer and select the option to repair.

  2. Go to Command Prompt in Advanced Options, then enter the following commands:

    bootrec /fixmbr
    bootrec /fixboot
    bootrec /rebuildbcd

  3. Press the Y key when it asks you to accept, then Exit.

  4. Reboot your computer.

In addition to this, you can check this link to have another way of repairing boot manager.

Port Forwarding - Microsoft Community

xb1r athena say:

Port Forwarding

Login Information

  • Gateway - 192.168.0.1
  • Username - cusadmin/chtruser/chtrtech
  • Password - password/charter

Port Forwarding

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.0.1
  3. If asked for a user name and password use (cusadmin/password). If your ISP is Charter you can use chtruser/charter or chtrtech/charter.
  4. In the menu at the top, select Setup.
  5. On the left side select Port Forwarding.
  6. In local IP Adr type in the IP Address of the Xbox.
  7. In Start Port and End Port type in 3074.
  8. In Protocol drop-down box select both.
  9. Put a check in the enable box.
  10. Select Apply.

Disable Port Forwarding

  1. Open a web browser.
  2. In the address bar at the top, type in 192.168.0.1
  3. If asked for a user name and password use (cusadmin/password). If your ISP is Charter you can use chtruser/charter or chtrtech/charter.
  4. In the menu at the top, select Setup.
  5. On the left side select Port Forwarding.
  6. Uncheck the Enable box of the ports you want to remove.
  7. Select Apply.

If you have access to another computer running Windows 7, you could try creating a system repair disc from there:

How to Create a System Repair Disc in Windows 7 - Notebooks.com

Method 1:

　

===============

　

1. Put the Windows 7 installation disc into the disc drive, and then start the computer.

　

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

　

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

　

4. Click Repair your computer.

　

5. Click the operating system that you want to repair, and then click Next.

　

6. In the System Recovery Options dialog box, click Command Prompt.

　

7. Type sfc /scannow, and then press ENTER.

　

　

Method 2:

　

===============

　

1. Put the Windows Windows 7 installation disc in the disc drive, and then start the computer.

　

2. Press any key when the message indicating "Press any key to boot from CD or DVD …". appears.

　

3. Select a language, time, currency, and a keyboard or another input method. Then click Next.

　

4. Click Repair your computer.

　

5. Click the operating system that you want to repair, and then click Next.

　

6. In the System Recovery Options dialog box, click Command Prompt.

　

7. Type Bootrec /RebuildBcd, and then press ENTER.

　

　

Method 3:

　

===============

　

1. Put the Windows Windows 7 installation disc into the disc drive, and then start the computer.

　

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

　

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

　

4. Click Repair your computer.

　

5. Click the operating system that you want to repair, and then click Next.

　

6. In the System Recovery Options dialog box, click Command Prompt.

　

7. Type BOOTREC /FIXMBR, and then press ENTER.

　

8. Type BOOTREC /FIXBOOT, and then press ENTER.

　

9. Type Drive:\boot\Bootsect.exe /NT60 All, and then press ENTER.

　

　

Note: In this command, Drive is the drive where the Windows 7 installation media is located.
