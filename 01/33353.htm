If you can't find the toggle to disable Secure Boot, a firmware update for the UEFI/BIOS may be required. This can usually be found in the manufacturer's website.

After disabling Secure Boot, boot normally into Windows and check if you can install the programs. If you're successful, boot back into the UEFI/BIOS and re-enable Secure Boot since this is an important security feature.

Once you're done creating an installation media, follow the steps below to access WinRe.

  1. Insert the installation media.
  2. Reboot your computer.
  3. Press any key to proceed when prompted.
  4. On the Install Now page, click Repair your computer.

  5. Choose Troubleshoot.
  6. Select Advanced Options.
  7. Click Command Prompt.
  8. On the Command Prompt, type this command:
         Net user administrator /active:yes
  9. Close the Command Prompt.

Once you're done with the steps, reboot your computer and log in using the hidden administrator that you have enabled.

Create a new user account (administrator) and then disable the hidden administrator. To disable the hidden administrator, log in to the newly created user account (administrator). Open Command Prompt (Admin), and then type this command:

Net user administrator /active:no
