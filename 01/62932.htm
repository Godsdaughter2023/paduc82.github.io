Overview

Login Information

 

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 

 

How To Find SSID (Network Name)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Basic" from the sub menu.
  6. The wireless network name will be labeled as "SSID".

 

How to Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the sub menu.
  6. If the "Network Authentication" is showing "802.1X" the passphrase will be based on the "Current Network Key". (By default it will be key 1, and the wireless passphrase will be "Network Key 1".)
  7. If the "Network Authentication" is showing "WPA-PSK", "WPA2-PSK", or "Mixed WPA2/WPA-PSK", the wireless passphrase will be labeled "WPA Pre-Shared Key".

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the sub menu.
  6. Select the desired security type from the "Network Authentication" drop down menu. (To disable security, select "Open" and then "Disabled" from the "WEP Encryption" drop down menu.)
  7. Select Save/Apply.

 

Change Wireless Channel

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Advanced" from the sub menu.
  6. Select the desired wireless channel from the "Channel" drop down menu. (To allow the device to select the channel automatically, select "Auto".)
  7. Select Save/Apply.

 

Change Wireless Mode

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Advanced" from the sub menu.
  6. Select the desired wireless mode from the "54g Mode" drop down menu.
  7. Select Save/Apply.

DHCP Reservation

 

This feature is not available on this device. 

 

 

DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "DMZ Host" below that.
  6. Enter the IP address of the Xbox in the "DMZ Host IP Address" box.
  7. Select Save/Apply.

 

Disable DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "DMZ Host" below that.
  6. Delete the IP address that is entered in the "DMZ Host IP Address" box.
  7. Select Save/Apply.

 

Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select Add.
  7. Select the radio button for "Custom Service" and enter Xbox.
  8. Enter the IP address of the Xbox into the "Server IP Address" box.
  9. Enter 3074 into the "External Port Start", "External Port End", "Internal Port Start", and "Internal Port End" in row 1.
  10. Select "TCP/UDP" from the "Protocol" drop down in row 1.
  11. Select Apply/Save

 

Disable Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select the radio or check box next to the rule you would like to remove.
  7. Select Remove.


 

Firewall

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Quick Setup" from the menu on the left.
  5. Uncheck the "DSL Auto-connect" check box.
  6. Select Next.
  7. The information on the WAN IP settings page should be already configured. Select Next.
  8. Check or uncheck the "Enable Firewall" check box based on current configuration. (By default the Firewall will be enabled if the device is configured with PPPoE.)
  9. Select Next leaving the other settings as default until the quick setup is complete.
  10. Select Save/Reboot on the last screen.

 

 

MAC Filtering

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted enter "root" for the username and "12345" for the password.
  4. Select "Wireless" from the menu on the left.
  5. Select "MAC Filter" from the sub menu.
  6. Select the radio button for the "Disabled" option.
  7. Select Save/Apply.

 

Bridging Guide

 

  1.  Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted enter "root" for the username and "12345" for the password.
  4. Select "Quick Setup" from the menu on the left.
  5. Enter the PVC Index (provided by ISP). Select Next.
  6. Enter the VPI and VCI values (provided by the ISP). Select Next.
  7. Select the "Bridging" radio button and select Next.
  8. Check the "Enable Bridge Service" check box, and enter the "Service Name" (Provided by ISP). Select Next.
  9. Leave the "IP Address" and "Subnet Mask" as their defaults. Select Next.
  10. Uncheck the "Enable Wireless" check box. Select Next.
  11. Select Save/Reboot.

This feature is not available on this device. 

 

 

Restore Factory Defaults

 

Through Firmware

 

  1. Plug the computer directly into the devices LAN port via an Ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  4. When prompted, enter "root" for the username and "12345" for the password.
  5. Select "Management" from the menu on the left.
  6. Select "Settings" from the sub menu, then "Restore Default" below that.
  7. Select "Restore Default Settings"

 

Through Hardware

 

  1. Plug the computer directly into the devices LAN port via an Ethernet cable.
  2. Locate the Reset button on the device.
  3. Using a toothpick/pen/paper clip, press and hold the button for 10 seconds.
