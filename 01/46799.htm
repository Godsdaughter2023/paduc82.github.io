  1. start any UWP application (in my example the windows calc.exe, the default Windows 10 calculator application)

  4. repeat the 1-2-3 steps 5000 times

rgharper911 say:

If they are similar addresses, say for example 192.168.1.1 for the router and 192.168.1.37 for the printer, this should work file.  If they are more than just a little different - say for example instead of 192.168.1.37 the printer's address is 192.168.0.100 or 169.54.45.21 - then likely not.  In most home networks the first three octets of the IP address must be the same with the fourth octet being different for every computer or device attached to the network.
