cbarnhorst say:

The file you refer to is another Electronic Software Delivery file (notice the ".esd" extension at the end) but not the one that makes the ISO file.  The file you list is the temporary file from the download. 

ESD file:

The file you refer to is not the ESD file the tutorial describes.  You have a hidden folder in the root of your system drive (usually C:) called Windows-BT.  To see it you need to make sure you have View Hidden Items checked in File Explorer. 

Inside the Windows-BT folder is the Sources folder and inside that is the install.esd file referred to in the tutorial.  If after making sure that you can view hidden files you do not have a Windows-BT file in your root directory then you have to use the MCT.  The MCT is located by scrolling the webpage down and clicking Download Tool Now button.  You will start to see options.

Media Creation Tool:

To use the MCT to make media you let a new one download and start.  One of the screens that appears will ask if you want to Install Now or make an ISO for another computer.  It will also have a checkbox for making the ISO with the same characteristics as the computer you are running.  That is what you want to do.  Just read all of your options as you go and you will come to what you want.  At any point in these early stages you can cancel or use the back button.
