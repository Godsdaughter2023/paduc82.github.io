This solution has worked for the last 2 months. I don't think this problem is limited to Windows (certainly not only Windows 10). It's something we install or do on the system. 

In the Windows Registry, look for keys that have numbers corresponding to country/region codes in Windows. Remove any key of the keyboard layout you don't want (in my case, 0x1009 to 0x0c0C).
