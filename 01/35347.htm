    ramesh srinivasan say:

    The analog clock can be brought back using a registry edit.

    Start Regedit.exe and navigate to the following branch:

    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\ImmersiveShell

    Create a new DWORD (32-bit) value named UseWin32TrayClockExperience

    Double-click UseWin32TrayClockExperience, and set its value data to 1.

    Exit the Registry Editor.

    How to Get Back the Classic Tray Clock in Windows 10
