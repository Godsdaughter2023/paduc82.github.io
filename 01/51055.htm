Ideally, backups are created in sets known as backup periods. To help maximize your disk space, Windows Backup backs up all selected folders the first time it runs and then it only backs up files that are new or have been modified since the last backup was made.

 

Periodically, Windows creates a new, full back up. Each full backup is known as a backup period. When you view your file backups, you see all of the backup periods labeled with date ranges. If you decide to delete file backups, you should always keep the most recent file backup.
