bossdweeb say: No,  you only  have to follow the instructions in the link to re-set your services.
Your software is not affected other than starting (launching) with windows if
you choose to disable it in Startup  permanently, and you can always re-enable
any Starup apps by typing   msconfig  in  Start menu > Run to bring up
System Configuration.

With the Startup applications though it's best to only leave those that you need
to have launch with Windows ticked (enabled). This does not stop you from
running those programs by launching them with a shortcut. It only stops them
from running in the background and using resources unnecessarily.

Programs like 'advanced system care' are not really necessary, and should not
be left to launch with Windows. Just run Windows disk cleanup frequently and
keep your HDD defragmented.
Having programs like this running in the background can be counter productive
in relation to system performance as they use resources when doing so.
Run the 'advanced' system care occasionally if you wish, but leave any registry 'cleaning' it may do alone unless you are after a specific registry entry that you want removed . Registries don't really need cleaning. That is more a thing of the past, when HDD's were very small and any sort of optimisation was
important. Programs like this can do more harm than good.

You also should not run more than one Anti-Virus at a time, and I see that
ASC also does scans. Running more than one AV actively can cause conflicts
and slow your machine. It's OK to run a second AV scan manually on occasion
as a cross reference, but not 'in resident' (always on ) with another AV .

Do you know when this problem began?  Try a System Restore to a point in timeBEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work. sharonmary say: Hi dar2,

Yes, I think you have answered your own question. Your taskbar is working exactly the way it should.

Just a little explanation: when you open different programs (Word, Excel, etc.), or if you have multiple windows open in Internet Explorer, they are shown on the Taskbar side by side so that you can jump easily between them. The one that is highlighted on the Taskbar tells you exactly where you are. So, yes, if you are in this forum starting a new question, that's exactly what you will see on the button.

This kind of Taskbar display suits some people but not others. If you have a lot of windows open it can look very cluttered.

The selection you made to Always combine, hide labels is a way of tidying up the taskbar. What happens then is that all the Internet Explorer sessions are grouped into one icon and all you will see is the Internet Explorer icon; if you have several Word documents open, they are also grouped together in separate icon and all you will see is the Word icon. (If you look closely at these icons you will see that it shows "cascade" effect to indicate that there is more than one window open).

What happens then is that you need to point to each of these groups and a little preview of all your IE windows, or all of your Word documents will show ... you can then click on the little preview window to jump to whichever one you want.
