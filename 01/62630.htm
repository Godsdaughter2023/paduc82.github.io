0x80080005 CO_E_SERVER_EXEC_FAILURE
BITS service could not be started

First, ensure that no proxy server is causing this issue.
Click the Start orb, in the Search field type in cmd
Right click cmd.exe and choose 'Run as administrator'. Agree to the UAC prompt.

At the prompt, enter the following command exactly as written including the spaces, and then press Enter

netsh winhttp show proxy

If this message appears, type exit, then press Enter

Direct Access <no proxy server>

If a proxy is listed, then type in the following and press Enter

netsh winhttp reset proxy

Then type in exit, press Enter to close the command prompt.
See if the system can update now.
