a. user say:

Here's a truth about all operating systems, even non-Windows operating systems:

A clean install correctly performed on supported hardware always succeeds. So there are two components:

1- Supported hardware

For the best chance of success, your hardware should be supported on Windows 10. The only way to find out 100% for sure if that's the case is to ask the hardware manufacturers, because it's the manufacturers who decide whether or not they will support Windows 10 on their hardware. Microsoft doesn't make that call.

Your hardware doesn't have to be supported on Windows 10. Some of my computer's hardware isn't. (Shhh - don't tell anyone!) But if it isn't supported then the manufacturer doesn't guarantee that it will work with Windows 10. Maybe it does, maybe it doesn't.

If you decide to check, I can help you. It's really not that difficult to do.

2- A clean install correctly performed

Did you pick up on that caveat? A clean install is no stroll in the park. It requires technical know-how (that you can learn, not to worry) and lots of preparation (otherwise you'll need lots of Preparation H.) Fortunately there's lots of good information on the internet about how to do a clean install the right way. You owe it to yourself to read up. Just remember, there's more than one way to do this, so don't get thrown if you see minor variations.

A clean install is so great because it gives you a fresh start with a clean slate. I wish I could've had a fresh start with a certain girl from high school, but now I'm getting off topic.

If all this sounds like a lot of work, yes it is. But it beats the heck out of doing 4 installations in a month and your computer still doesn't work.

Once you have your clean install on your supported hardware, how do you keep the magic going?

3- Supported software

Only install software that the developer supports on Windows 10. Not kidding about that. Older software might work, probably works, or might not work and crash your computer. Not worth it. Want to run that sick game from 1997? You can - there's a way. (You can learn about virtual machines another time.)

4- Don't make uninformed changes to your computer

People who post here have done all kinds of crazy things to their computers. Don't be one of them. Take the time to learn how to use your computer correctly. It's not rocket science. It doesn't even have to be computer science (unless you want to go to the Dork Side.) Get a book or whatever, but don't make any changes to your computer unless you really understand what you're doing.

And for those times when you don't understand what you're doing ... backup first!

a. user say:

Unofficial testing on a Lenovo ThinkPad T61p (Windows 7) d MacBook

http://lifehacker.com/5526542/how-much-battery-life-does-sleep-mode-really-drain

FYI - Windows Vista Power Management

 Sleep mode consumes some power from the battery, though far less than the normal On state. If the computer is in Sleep mode while being powered from the battery, and the battery charge runs low, the computer will turn completely Off, and will lose the open programs and documents in memory. In this case, any changes to documents that were not yet saved to disk would be lost.

Hybrid Sleep offers an additional measure of convenience. If the computer's battery becomes fully discharged while the computer is in Hybrid Sleep mode, the saved state in the computer's memory will be lost, but Windows will be able to retrieve the saved state from the hard disk drive -- though somewhat more slowly.

Empty your Recycle Bin

When you delete a file, it is usually moved to the Recycle Bin. To permanently delete the files in your Recycle Bin, follow these steps:

·        On the Start menu, click Settings > System > Storage.

·        Under Storage, click This PC > Temporary files.

·        Under Recycle bin, click Empty recycle bin > Yes, I'm sure.

·        Tip To see what's in your Recycle Bin before you empty it, open it from your desktop.

Save new files to another drive

Instead of saving all your new files to the same drive, save some files to a different drive. To save files to a different drive, follow these steps:

·        On the Start menu, click Settings > System > Storage.

·        Under Save locations, select a drive from the drop-down menu. If the drive you want to use isn't listed there, make sure it's connected to your PC.

Save fewer OneDrive files on your PC

It's convenient to have your OneDrive files saved offline so you can use them when you're not connected to the Internet. But if you're running low on drive space, you can save fewer OneDrive folders offline. You'll still be able to get to them from OneDrive.com, but they won't take up space on your PC. To save fewer OneDrive folders offline, follow these steps:

·        Go to the right side of the taskbar and find the OneDrive icon. If you don't see it, click the Show hidden icons arrow and see if it's there.

·        Right-click the OneDrive icon, and then click Settings > Choose folders > Choose folders.

·        Clear the check boxes for the folders that you don't want saved on your PC, and then click OK.
