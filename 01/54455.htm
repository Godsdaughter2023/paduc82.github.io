Hi Poetboy,

An incorrect version of the Wmpui.dll file is registered on your computer. This may occur when you install some third-party programs.

In order to diagnose the issue, you may check for errors in event viewer.

How to view and manage event logs in Event Viewer in Windows XP

You may also follow these steps & check if the issue persists.

To resolve this issue, register the correct Wmpui.dll file again:

a. Click Start, and then click Run.

b. In the Open box, type the following command, and then click OK:

regsvr32 %SystemRoot%\system32\wmpui.dll

Note: You receive a message that is similar to:

DllRegisterServer in C:\WINNT\system32\wmpui.dll succeeded.

c. Click OK, and then start Windows Media Player again.
