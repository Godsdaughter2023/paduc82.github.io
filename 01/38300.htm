mahesh babu g say:

Hi Pooya,

 

Thank you for posting your question in Microsoft Community.

 

With the Windows Startup Settings screen (formerly advanced boot options) you can start Windows in different advanced troubleshooting modes so you can find and fix problems on your PC. In previous versions of Windows, you could get to this screen by pressing F8 before Windows started up. Windows 8 PCs start up quickly so there's not enough time to press F8.

 

Follow the methods:

 

Method 1:

 

There are two ways to get to the Windows Startup Settings screen in Windows 8 from the link:

 

Windows Startup Settings (including safe mode):

http://windows.microsoft.com/en-IN/windows-8/windows-startup-settings-including-safe-mode

 

If you need further assistance on this particular issue, let us know and we will be glad to assist you.

 

Method 2:

 

You can also use command prompt to to get to the Windows Startup Settings screen in Windows 8 by following the steps:

 

a. Open an elevated command prompt by pressing Windows + X keys and click Command Prompt (Admin).

b. In the command prompt, type bcdedit, and press Enter.

c. Type bcdedit /set {current} bootmenupolicy Standard and press Enter.

d. Type Exit and press Enter and restart the computer to make the changes to take effect.

 

If you need further assistance on this particular issue, let us know and we will be glad to assist you.
