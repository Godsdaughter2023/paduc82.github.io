johnptd say:

There is a bug in the Creators Update v1703 which causes Network Discovery to not function properly. Usually what you will see is the first computer that you boot may show all the other computers but other computers booted after that may only show some other or even no other computers. I had one case where it did not even show itself. You can try entering the computer name on the address line of the File Explorer Network section. Replace "Network" with "\\" followed by the other computer's name (such as "\\Computer2"). See if that works. Unfortunately you will probably have to repeat this each time your system reboots.
