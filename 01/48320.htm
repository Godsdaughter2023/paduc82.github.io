xb1r apollo say:

Overview

Zyxel VSG1432

Login Information

Gateway - 192.168.1.1 or 192.168.1.254

Username - admin

Password - 1234 or telus

 

How to Find SSID (Network Name)

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  4. Enter the username (admin/1234) and password (admin/telus) click Login in the upper right-hand corner.
  5. Select Wireless.
  6. The network name is listed in the Wireless Network Name (SSID) field.

 

 

How To Find Wireless Passphrase (WPA/WEP)

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  4. Enter the username (admin/1234) and password (admin/telus) click Login in the upper right-hand corner.
  5. Select Wireless.
  6. The passphrase will be listed as Password under the Security Mode section.  

 

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  4. Enter the username (admin/1234) and password (admin/telus) click Login in the upper right-hand corner.
  5. Select Wireless.
  6. From the Security Mode drop down select the desired security type. If no wireless security is desired select No Security.
  7. Click the Apply button at the bottom.

 

 

MAC Filtering

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  4. Enter the username (admin/1234) and password (admin/telus) click Login in the upper right-hand corner.
  5. Select Wireless.
  6. Select MAC Authentication from the submenu.
  7. Select Disable from the MAC Restrict Mode options.
  8. Click the Apply button.

 

 

Change Wireless Mode

 

  1. Open a web browser.
  1. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  2. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  3. Select Network Settings from the menu at the bottom.
  4. Select Wireless from the submenu.
  5. From the 802.11 Mode drop down select 802.11G only mode.
  6. Click the Apply button.

 

 

Change Wireless Band (width)

 

  1. Open a web browser.
  1. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  2. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  3. Select Network settings from the bottom of the screen.
  4. Select Wireless settings next.
  5. On the General tab select the “Bandwidth” drop down menu.
  6. Select the “20 Mhz” option.
  7. Click the “Apply” button.

 

 

DHCP Reservation

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network Settings from the menu at the bottom.
  5. Select Static DHCP from the submenu.
  6. Select "Add New Static Lease".
  7. Check the box for Active and input the MAC Address and IP Address of the Xbox.
  8. Select Apply.

  

 

UPnP

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network Settings from the menu at the bottom.
  5. Select Home Network from the submenu.
  6. Select UPnP from the next submenu.
  7. Toggle UPnP.
  8. Select Apply.

 

 

DMZ

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network Settings from the menu at the bottom.
  5. Select NAT from the submenu.
  6. Select DMZ from the next submenu.
  7. Enter the IP Address of the Xbox in the Default Server Address.
  8. Select Apply.

 

Disable DMZ

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network Settings from the menu at the bottom.
  5. Select NAT from the submenu.
  6. Select DMZ from the next submenu.
  7. Remove the IP Address of the Xbox from Default Server Address.
  8. Select Apply

 

 

Port Forwarding

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network Settings from the menu at the bottom.
  5. Select NAT from the submenu.
  6. Select Port Forwarding from the next submenu.
  7. Select Add New Rule.
  8. Check the box for Active and input Xbox for the Service Name.
  9. Input 3074 for the External Start/End Port and Internal Start/End Port.
  10. Input the IP Address of the Xbox and select TCP/UDP for the Protocol.
  11. Select Apply.

 

 Disable Port Forwarding

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network Settings from the menu at the bottom.
  5. Select NAT from the submenu.
  6. Select Port Forwarding from the next submenu.
  7. Click the Trash Can icon to the right of the entry that uses port 3074.

 

 

Firewall

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Security Settings from the menu at the bottom.
  5. Select Firewall from the submenu.
  6. Drag the firewall level to Low.
  7. Select Apply.

 

How to Find Firmware Version

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Maintenance from the menu at the bottom.
  5. The firmware version is located next to firmware version.

 

 

Firmware Update

 

Firmware updates are installed automatically by the Internet Service Provider.

 

 

Restore Factory Defaults

 

Through firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar. (Or click one of the links) Enter the username and password if the gateway asks. (admin/1234, admin/telus).
  4. Select Maintenance from the menu at the bottom.
  5. Select Configuration from the submenu.
  6. Select the Reset button.

 

Through Hardware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the reset button on the device.
  3. With paperclip/pen/toothpick hold down the reset button down for 30 seconds.
  4. Wait for lights to turn off then return to normal.

 

 

Bridging Guide 

 

*Bridge mode should only be used if you have a router to plug into the device.*

 

*Please have PPPoE information available if needed to enter into the router (contact ISP if needed to get this information)*

  2. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar (or click one of the links).
  3. Click Login in the upper right-hand corner and enter the username and password. (admin/1234, admin/telus).
  4. Select Network settings from the bottom of the screen.
  5. Select Broadband from the following menu.
  6. Click the edit button under the modify column.
  7. In the mode drop down menu select Bridge
  8. Click the Apply button on the bottom of the page.

 

 

Known Issues

 

 There are no known issues for this device.

 

 

Multiple Console Support

 

We have confirmed that this network device is unable to support more than one console with an Open NAT.

jimthecru say: Hello,

I believe this has to do with your virtual memory utilization. Virtual memory is a growing or shrinking amount of hard drive space that is set aside for open programs to use. So it will grow when more programs are open and shrink when fewer are open. It is a good thing to keep so do not turn of virtual memory no matter how much memory you have in your system. You can change the values so there is a set low and set high amount the system uses. There is nothing to fix, it is part of all Windows OS's. Since your only talking 2-3 gigs of space that it fluctuates that's not something to worry about, It is minimal.
