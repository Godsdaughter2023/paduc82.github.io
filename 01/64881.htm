marlon_a say: I have had the same problem as you and searched everywhere for a positive answer and haven't got one. I was about to return the netgear extender when I, fortunately, found the way around this strange problem.
In fact it's really simple, and I really would encourage everyone that eventually get this problem to complain to netgear about not having their setup process detailed in the guide they ship with the product.

In my case I have a WEP encrypted network.

If you cannot access the extender's web interface than you need to reset the device by inserting a paper clip on the small hole next to the WPS button(if you don't know that already).

When you are prompted for the encryption key and passphrase for the existing network(original one), you MUST WRITE THE PASSPHRASE ON THE KEY INPUT FIELD you select and not on the field for the passphrase. You may select any of the 4 key input fields but you MUST write the passphrase on one of them and leave the passphrase field blank(I believe this is just for WEP encryption which is my case).

I also changed the encryption type for the new "_EXT" network from WEP to WPA/TKIP but kept the same password.

After the connection completed the extender successfully checked for updates and all IP addresses were assigned automatically via DHCP.

I hope I answered in time before you return the extender because I don't have any kind of complains after doing this.

Best of luck,
Marlon.

Not sure what computer you have (I'm assuming you're on Window 10 though) so I'll answer your question in 2 parts. If you've got a Sony VAIO like me with the 9600 GT (or 9600M GT) installed inside it, I recommend ignoring 'Answer 1' below and just scrolling down to 'Answer 2'. Answer 1 is referring to if you can install drivers on your graphics card without having to rely on the computer manufacturer to provide their own specific model updates.

Answer 1

If you go to Nvidia's website and navigate to their drivers section : http://www.geforce.com/drivers

Once on the Nvidia driver page, in the 'Manual Driver Search' box fill it out as shown in the screen grab below:

::NOTE for screen grab above:: Obviously, in you're in a different country to the UK simply select the relevant country for you from the drop down list (second from bottom, above the 'Start Search' button.

Once you've pressed search the page will load a list of all drivers available for that 9 series model (in your case the 9600M GT) There's no Windows 10 driver available for it - but an 8.1 driver will work just fine.

The most recent one is the: GeForce 341.44 Driver WHQL (24th February 2015).

So, try to install this - if it works - then that's it. Latest driver, for that card is now installed :-D

Answer 2 - The following ONLY applies to a VAIO VGN-AW11Z'B'

If you have a VAIO VGN-AW11Z'B' like myself... the first thing that you should notice after upgrading to Windows 10 is that Windows has gone ahead and installed a generic display driver in place of your Nvidia cards driver - This makes everything appear large and blurry on your screen.

After I noticed this had occurred, I simply installed the relevant graphics driver (nVidia) - 8.15.11.8640_0910130 from Sony's support page... http://www.sony.co.uk/support/en/content/cnt-dwnl/VGN-AW11Z/list

The latest driver is a windows 7 driver (don't look at this as a problem though, the most important thing is getting the system to recognise that there's an Nvidia card inside your laptop).

There is also an 'Nvidia Guard' application for Windows 8 on the same support page (download, install this as well).

Once those are installed onto your VAIO VGN-AW11Z'B' the Nvidia graphics card will now work perfectly.

:: IMPORTANT ::

If you have a different Sony VAIO model, or even another manufacturers model like an ASUS or a Lenovo or Dell or whatever, please go to your laptop manufacturers relevant support page for your specific laptop model. There will be an Nvidia graphics card driver there for you if the 9600M GT is the card that shipped originally with your laptop (it may well not be a Windows 10 driver((it could be but the chances of that are very, very slim as it's an old card)) but even if it's for Windows 8/8.1 or even Windows 7) installing the latest driver version that is available to you will fix the problem that I'm guessing you're having which is that Windows 10 has installed a generic display driver in place of your Nvidia driver after you upgraded to Windows 10 which is making your graphics and screen look not so great at all.

Hope this helps you out.

praseetha k say:

Hi sheema88,

 

1. Are you trying to delete the songs copied to media player library?

2. What exactly happens when you try to delete the music?

3. Do you receive any error messages or codes?

 

You won't be able to delete songs on your memory card using windows media player. You can only delete the songs in Windows media player library.

Refer the link:

Remove items from the Windows Media Player Library

http://windows.microsoft.com/en-US/windows7/Remove-items-from-the-Windows-Media-Player-Library

 

You have to go the original location and choose music from there to delete it.

a. Insert your memory card into a supported card reader connected to the PC.

b. Select "Open folder to view files" in the dialog window that opens on the desktop.

c. Or go to the "Start" menu and select the "Computer" folder. Double-click on the removable disk.

d. Locate the music file to be deleted. Right-click on the file and select Delete.

 

Hope this information is helpful.
