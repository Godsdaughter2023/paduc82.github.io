Thank you for the fast response. As long as you select Normal startup under Startup selection in the General tab, it should resume all Microsoft services. Another isolation method that you can perform is to boot your PC in safe mode. This starts Windows in a basic state, using a limited set of files and drivers. Regarding the error message you encountered with the Photos app, you can run the Windows Store Apps troubleshooter. This diagnostic tool will scan and fix your system for any issues found with your apps. You need administrative permission to run this tool.

  1. Right-click the Start button, select Settings > Update & Security > Troubleshoot.
  2. Select Windows Store Appsunder Find and fix other problems.
  3. Click Run the troubleshooter.
  4. If prompted, click Yes or enter the administrator password.
  5. Follow the on-screen instructions.

Should the issue persists, you can perform the rest of the troubleshooting methods outlined in this support article. The page has multiple methods that can help you resolve the issue with your Windows apps.

To make sure that you're PC is free of corrupted or damaged Windows system files, you can run both the DISM command-line tool and the SFC tool. These tools will scan your system for corruptions in Windows system files and restore corrupted files. You also need administrative permission to perform this procedure.

  1. Click the Start button and type Command Prompt.
  2. Right-click Command Prompt and choose Run as administrator.
  3. Enter DISM.exe /Online /Cleanup-image /Restorehealth and then press
    • Note the space before each "/".
    • This step may take a few minutes to start and up to 30 minutes to run and complete.
  4. Enter sfc /scannow and then hit Enter.
    • Note the space between "sfc" and "/".
  5. Restart your PC.
