taurarian say:

Although not solving the problem, you can hide updates.


Bit rusty on XP, but I believe all you have to do is uncheck the update you don"t want to install and then the download button changes to a close button. After you hit close you are told that the unselected updates will NOT be installed then it gives you the option to never be notified of that update again

 

Also you'll need to clear the temporary download folder:-

1. Stop the Automatic Updates service:

Start > Run > (type in) services.msc > [OK]
Double-click Automatic Updates > Click on Stop
(Stopping the service will take a moment)

2. Delete the contents of the Download folder:

Start > Run > (type in) %windir%\SoftwareDistribution > [OK]
Open the Download folder and delete its contents
Close the window.

3. Start the Automatic Updates service:

Start > Run > (type in) services.msc > [OK]
Double-click Automatic Updates > Click on Start
(Starting the service will take a moment)
