all the found keys should exist under:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings

3)hit F3(find next)...you will find several keys that need to be changed this way (as i said one for each profile)

you will also find instances that dont contain value of 4b0.These keys represent other preferances in power profiles so they shouldnt be changed.

Hi Adithya,

Telemetry is a feature that uploads data that will help us improve the features and overall experience of the operating system. You have an option to turn if off if you don't want to participate in sending automatic feedback. To do this, follow these steps:

  1. Press Windows key + S and type Services.
  2. Locate Diagnostics Tracking Service and double-click it.
  3. Click Stop then change the Startup type to Disabled.
  4. Click Apply and OK.

Additionally, you can adjust the performance of Windows. This method will help improve the system performance of Windows 10, and will gradually decrease the freeze issues.

  1. In Search tab of the Start menu, type Adjust the appearance and performance of Windows and press Enter.
  2. Go to Advanced tab and click Change in the Virtual memory section.
  3. At the top, uncheck the box right before Automatically manage paging file size for all drives option.
  4. Select drive C: (or the drive where Windows 10 is installed).
  5. Select Custom size: by ticking its radio button, and change the Initial size (MB) and Maximum size (MB) to the recommended value that you will find just below; on the Total paging file size for all drivers section.

Remember: Changing paging file size into larger than which was recommended by Windows is not going to be helpful.
