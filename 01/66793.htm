Reinstall Windows Preinstalled On Computer

When Windows comes installed with the computer, OEM computer manufacturers are required to provide at least one way for you to reinstall Windows if you buy a computer with Windows preinstalled - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks. ﻿Most computer manufacturers also have a feature to make your own install disks.

If the restore partition is still there and was not removed, it will reinstall Windows. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to. Or if you have install disks - insert them, restart and boot from them to start the reinstall process.

If you restore it to factory condition, it will be like it was when you first bought it - all preinstalled programs will be installed. Anything you installed after buying it must be reinstalled. This will remove any personal files on the computer, back them up first.

the larch say:

From my notes on Windows 7 Search, here is a set of briefly described tips that I have collected:

 

General tips to try when Windows 7 Search seems defective:

(1) If "Use natural language search" is on ("Folder and search options", Search), then turn it off.
(2) Rightclick on the Taskbar, click Properties, Start Menu tab, Customize, scroll down and find "Search other files and libraries", choose Search with public folders.  Have checkmark at Search programs and Control Panel, click OK.
(3)  If the Windows Registry HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\FolderTypes has {ef87b4cb-f2ce-4785-8658-4ca6c63e38c6}, then delete it.
(4) Turn Windows 7 Search off (click Start, Control Panel, Programs, Turn Windows features on or off, click checkbox of Windows Search, restart computer) and then repeat the procedure to turn it on again.
(5) In Indexing Options, delete all Included Locations items, click Advanced, Select New to rechoose.
(6) Rebuild the index (in Indexing Options, click Advanced, Rebuild, OK, and wait).
(7) Turn indexing off (as described in the paragraph titled "How to turn off Windows 7 Search indexing").  Delete all the files in the Windows and Temp folders in C:\ProgramData\Microsoft\Search\Data\Applications first if you will turn it back on.  To be able to search in Libraries windows at all, the Windows Search indexing service needs to be on.
(8) Consider trying System Restore.
(9) Create and use an entirely new user account.

How to turn off Windows 7 Search indexing:

Windows 7 Search still works without indexing (except in Libraries windows).  To turn indexing off, click the Start button, Control Panel, Administrative tools, Services, scroll down to find Windows Search in the window that appears and doubleclick it, and click the Stop button.  Indexing is made inactive for all user accounts.  To stop indexing from starting again, click the down-arrow button by “Startup type:”, click Disabled, and click the OK button.  (To turn indexing back on, change “Startup type:” to Automatic, then click the Apply, Start, and OK buttons.)  The Start menu Search box will then give results limited to programs and Control Panel items (any new programs installed will be searchable regardless of the indexing being off); no document files of any kind show up.  Searching in windows’ Search boxes mostly behaves as before. However, any changes that you might have done in Indexing Options will no longer be effective.  Also, the “Arrange by” feature in the Libaries windows will no longer work.

First, let's see if you really are missing the program -- or only the shortcut to it.

  • Do a search of your C: drive for "all files and folders" looking for mspaint

  • Click the "More advanced options" button and check the boxes to look in System Folders, Hidden Files and Folders, and Subfolders.

You should get several hits, but the important one should be mspaint.exe located in C:\Windows\system32.

If you find mspaint.exe in C:\Windows\system32, here's how you can put a shortcut on your desktop

1.  Right-click on an empty spot on your desktop and select New > Shortcut from the popup menus

2.  The Create Shortcut Wizard starts.  Copy the line below and paste it in the box that says "type the location of the item."  Copy it exactly as is.

                    %SystemRoot%\System32\mspaint.exe

3.  Click Next

4.  Type Paint in the box that says "Type a name for this shortcut."

5.  Click Finish.
