Can I install two Operating System on different hard drives.

Original title : Install two versions of windows

i have another ssd and two other hd discs, a third HD plugged into case socket

i want to install an old 32 bit version of windows on the same computer - is this possible?

Motherboard is P8-Z77 PRO with 32 gb CPU Intel 7037

mike hall... always say:

Yes, you can install two different operating systems and presumably you have already checked that the older OS is supported.

 

Discconnect every hard drive but NOT the one that you want the older OS on.

Install the older OS, and then run all updates, and install an anti-virus solution 

Now reconnect the other drives.

You can switch between the two operating systems by using the temporary boot option from your computer's initialisation screen.

 

Remeber to disable the action of each operating systems defrag utility on the OTHER OS..

 

Note that the above is NOT dual boot. It keeps both operating systems separate but still able to use common resources and files..

considerate_guy say:

It's on your keyboard...  on a standard QWERTY keyboard it would be to the right of the 'N' key and centered below the 'J' and 'K' keys.

It's the keyboard shortcut for making splits in the selected clip. I would try it out on a test clip first. Play the clip to where you want to make a split, then press the 'M' key. If you don't like the split, then use Undo (or Ctrl+Z on your keyboard).

You can do the same thing with music. Just be sure you have selected the music clip when you are playing back, not the video clip. Once a clip is split, you can move them independently on the timeline. You can also delete a selected clip (music, video or photo) by pressing the 'Delete' key (again on your keyboard).
