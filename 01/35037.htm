a. user say:

Hi Michael,

 

Thank you for posting your query on Microsoft Community Forums.

 

I see that you are unable to use mouse in media player on windows 8 computer.

 

I would like you to follow the methods provided to fix the issue:

 

 

Method 1: update the drivers:

 

a.      Press windows key + x.

b.      Choose device manager from the context menu.

c.       Look for the mouse drivers.

d.      Right click on the drivers and click on update drivers.

e.      Click Ok.

 

 

Method 2: run the media player troubleshooter.

 

a.      Press Windows Key + X.

b.      Choose control panel from the context menu.

c.       Click on troubleshooting.

d.      Click on view all from the left pane.

e.      Choose Windows Media Player setting.

f.        Click Next.

 

 

Method 3: Disable and Re-enable the windows media player.

 

a. Press the Windows + R key on the keyboard.
b) In the ‘Run’ windows type appwiz.cpl and click ‘Ok’.
c) Select the option Turn Windows features on or off.
d) Expand Media features.
e) Uncheck the media player.

 

Note: Restart the computer. After restart follow the same steps to check the media player from the Windows feature.

 

 

If you need any further assistance with windows, please reply and we will be happy to help.

Hello
Welcome to Microsoft answers

(A).If you want to  boot the computer without asking you who is the user try these steps.
1. Click start, type netplwiz into the start menu search box, hit enter.
2. Uncheck the box that says “Users must enter a user name and password to use this computer”, and then hit Apply.
(Or) select the username that you don’t want to display at the system startup and click remove.

(B). You may also disable Built-in Administrator Account and check if it helps.
1. Click start, type cmd, right click on cmd and click “ Run as administrator”.
2. Type the following command:
net user administrator /active:no
The administrator account will now be disabled, and shouldn’t show up on the login screen anymore.
