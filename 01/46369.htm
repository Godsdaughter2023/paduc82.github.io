Hello Anil,

Having run into some similar issues myself I may have a possible work around.

If I understand you correctly you have a working Windows 8/8.1 Intel graphics driver.

To "pin" or "freeze" your device driver you can tell Windows to not update device drivers.

By default Windows 10 will search Windows updates for device driver patches which it sounds like is happening to you and the Windows 10 driver is running into trouble.

If you want to try this out you can do so by going here:  

When you do that you should see the following window appear:

You will want to change the "yes" to a "no" and click "save changes".

If you are able to do this it will change Windows update's behavior such that it will no longer download and install device driver updates.

After doing that go ahead and install the legacy driver again and now it shouldn't be replaced by Windows update any more.

I have a similar issue in that an older Intel graphics driver works but the "Windows 10 version" has problems.

I was forced to "pin" the driver to a legacy version at least until a fix is deployed.

Have you confirmed that the missing folders or messages that were in them are no longer visible in Windows Live Mail? Make sure that Storage folders is selected (highlighted) on the View tab.

  • When viewing your message list, press Ctrl-Y to reveal the entire folder tree. Look for your missing folders and messages, possibly several layers deep under Recovered items or inside another folder.
  • Use the Search for a message box to look for one of the missing messages. Use a search term in the Subject or From fields for quickest results.

When you've found them, leave the program running for an hour or two to see whether any of the messages find their way back into their proper folders. If they don't you can do it manually - select a batch of messages, then right-click on the selection and choose Move to folder. Take it gently to allow the program to index the moved messages before moving another batch.

The Recovering unused disk space message will only appear if the program has been closed down, so this must have happened inadvertently. It doesn't work very well, so it would be as well to disable it once you've recovered your messages.
