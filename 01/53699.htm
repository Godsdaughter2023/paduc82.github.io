Vista and Windows 7 changed permissions so that even though you are the Administrator you do not normally run with Administrative priveledges but as a normal user.  When a program requires Administrative permission it asks for it.  That is a big change from XP.

foxfifth say: The following instructions assume that you want to have both users on the computer and you only want to move some of the files from one user to the other user.  If that isn't the case, ignore the following and post a reply.

If the above assumption is correct, log in with Admin privileges or as the user that currently has the files and go to Start > Computer (in the Start menu list at the right under Documents, Pictures, etc) and in Computer double click on the drive where the files are located and then double click on the "Users" folder and then double click on the original user's folder and work your way to the files you want to move.  Leave that window open (and optionally drag it to the right side of your screen until the mouse hits the edge and it will automatically resize to fit the right half of your screen or you can resize it manually).  Then start the same instructions over again (Start > Computer etc) but the second time when you open the "Users" folder, navigate to the new user.  With both windows open you can now drag files from one window to the other.
