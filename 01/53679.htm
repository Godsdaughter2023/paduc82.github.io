shenan stanley say:

Other than timelysupport's correct answer, you might do things *methodically* and in the following order:

  1. Uninstall Skype.
  2. Update your motherboard chipset driver from the manufacturers web page.*
  3. Update your audio device driver from the manufacturers web page.*
  4. Update your video device driver from the manufacturers web page.*
  5. Update your network device driver(s) from the manufacturer(s) web page(s).*
  6. Reboot and install CCleaner.  Run it and cleanup any loose files it finds.
  7. Download and install both MalwareBytes and SuperAntiSpyware (FREE for BOTH) and run full system scans with each of them (one at a time) and remove everything each of them finds.
  8. Open the registry and find the Skype entries and remove them.  Any you find.  HKLM and HKCU areas in particular.
  9. Reboot again.
  10. Perform a CHKDSK on all drives
  11. Perform a Defragmentation on all drives
  12. Reboot again.
  13. Visit http://winodwsupdate.microsoft.com/ and install all updates that are not optional (Optional Software and Optional Hardware can be ignored) and that do not update something you do not want to update (like Internet Explorer, etc.)
  14. Reboot and repeat 13 and 14 until you have no more non-optional updates you need to install.  Reboot once more for good measure.
  15. Download the latest version of Skype and install it. Reboot before you run it.

* Whenever possible, go to the actual manufacturer - not of the entire machine - but of the device.  For example, if you have an Intel chipset based motherboard, get the drivers for it from Intel.  If you have an nVidia video device, get the drivers for it from nVidia.  If you have a Broadcom network device, get the drivers for it from Broadcom.  If you have a RealTek audio device, get the drivers for it from RealTek.  And so on...

If, after doing all that in that order, it still does not work - it is likely either a hardware issue or a Skype issue.  However, you could try ONE last thing.  SFC /SCANNOW.  You will need your Windows XP CD.

After that - you really are left with hardware/Skype or something so messed up you might resign yourself to try a full wipe and install of everything fresh (although I would not do it if I could not image my disk and return it back to where it was before I tried.)
