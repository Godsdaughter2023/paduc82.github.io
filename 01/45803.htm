why is windows 7 adding locks to new folders that I create? - Microsoft Community

locked-down-123 say:

why is windows 7 adding locks to new folders that I create?

Hi to all,

Here is the situation I am in and I am all out of ideas.  Whenever I create a new folder on my desktop a lock icon appears with the folder icon.  Then I go to check the security tab and under the group or user names entry field there is listed; SYSTEM, My own profile name and finally Account Unknown(S-1-5-5-0-xxxxx) with a red question mark icon.  I am able to remove the Account Unknown entry, but then when I create a new folder the lock still appears along the Account Unknown entry in the security tab.  Why would it do this? And how would I fix this?

locked-down-123 say:

To the one person who has this issue, I figured it out.  It has to do with the permissions that are allowed for your parent folder that the new folders are being created in.  For example if you are having this issue when you create the new folders on your desktop, then check you permissions in the desktop folder.

In my case it was the desktop folder, so what I did was this,  I opened windows explorer, went to the security tab of the desktop folder, and saw that none of my groups or users had any permission for the desktop.  That is why when I created a new folder the Account Unknown user would be in the groups or users area of the security tab.

To Remedy this I did the following:

  • Go to desktop security tab.
  • Clicked Edit, and gave full control to SYSTEM, Administrators and my two users that I created.
  • Click apply then OK to close.

The lock on the new folder that I created earlier was gone. 

I am not quite sure how my permission got messed up, but i think it had something to do with me having attempted to move the desktop folder of the standard user to the desktop of the Administrator while I was logged into the Admin account.  I tried recreating this but I was unable to.  Either way it fixed now.
