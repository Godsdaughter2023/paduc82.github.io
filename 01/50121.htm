vanessa yar say:

Hi,

Based on the error message you've shared, it appears that the My Document folder on your Windows 7 laptop is password protected. To turn the password off, follow these steps:

  1. Open Control Panel and click Network and Sharing Center.
  2. On the left pane, click Change advanced sharing settings.
  3. Click the drop-down arrow next to Home or Work (Current Profile) to expand the network profile.
  4. Select Turn off password protected sharing.
  5. Click Save Changes.

In case this won't help, we recommend that you check and review the permissions set on your shared folder. Make sure to log in as Administrator. Here's how:

  1. On your Windows 7 laptop, right-click the shared folder (in your case, My Documents) and select Properties.
  2. Once open, click the Security tab. This will display all the permissions set on the computer.
  3. Click Edit to change the existing permissions. In case your Windows 10 is not yet added to the list, click Add, Advanced, and then Find Now to find all the users and groups on the computer.
  4. Select the user name that you want to add or adjust the permission, then click Change Permissions.
  5. Once done, click Apply to save the changes.

Let us know how it goes.
