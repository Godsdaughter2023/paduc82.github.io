If you boot in Safe Mode, what you will see is a bunch of white text on a black background rolling by on your screen as files and drivers are loaded, you will get a reminder message that you are booting in Safe Mode and asked to continue (or not), you will see the words "Safe Mode" in all four corners of your screen, your desktop icons are likely to be skewed around, and you will probably not see your background image/wallpaper - just a black background.

If you are not seeing that, you are probably not in Safe Mode and if you are in Safe Mode, the Microsoft engaged Support Engineer "expert" advice Method 1 to run the System File Checker will not work since SFC will not even run in Safe Mode - even though I have told then that 100 times, they keep suggesting it.  You do not describe any symptoms consistent with a need to run SFC.  It makes even less sense if you don't have a genuine bootable XP installation CD that is the same Service Pack as your installed Service Pack (which you indicate).  Suggesting SFC is akin to saying "I have no idea what the problem is or how to fix it".

Maybe you will get lucky with those ideas though.

You need to tell us exactly what the message says that you are seeing without paraphrasing.

For example, when you try to run msconfig, does it really say "cant find" or does it say something else and what exactly is that something else you are seeing?

Maybe you are seeing a message like this:

Windows Installer may be corrupted or your computer is in safe mode

or perhaps this message:

The Windows Installer Service could not be accessed.
This can occur if you are running Windows in safe mode, or if the Windows Installer is not correctly installed. Contact your support personnel for assistance.

If that is what you are seeing, this makes more sense:

First do this:

Click Start, Run and in the box enter:

services.msc

Click OK to open the Services applet.

Look in the Services applet to see if the Windows Installer Service is even listed and you will find that it is probably not listed (but it should be listed).

If the Windows Installer Service is listed, make sure it is not Disabled and if it is Disabled, change the Startup type to Manual.  You don't have to set it to Automatic as it will start on it's own when it needs to start and you don't need it starting every time you restart your system and running all day when there is nothing for it to do.

The Service will start when it needs to start if it is able to start.  Click OK enough times to save your adjustments and close the Services applet, restart your system to be sure the changes "stick" and see how things look now.

If the Windows Installer Service is completely missing from Services, you need to add it as follows:

Click Start, Run and in the box enter these commands one at a time clicking the OK button for each:

msiexec   /unregister    (or you can type:   msiexec /unreg)
msiexec   /regserver

Click OK after each command   (there will be no message about what happened)

Check in the Services applet again to see if the Windows Installer Service is now listed and the Startup type is set to Manual, restart your system to be sure the changes "stick" and see how things look now.

This particular adjustment is typically used when connecting LCD displays using a VGA cable or other analog video source in order to align the display pixels, but it should be unneccessary when using a DVI cable to connect with (not sure about HDMI but suspect the same). Composite sync would be moot with a DVI connection and in my experience the display positioning and resizing options in ATI Catalyst are also only operational for analog connections. Many of the symptoms you've described so far are typical for what happens on CRT monitors when the display position and size are misaligned (out of their normal range) on the monitor or in the display driver. Again, most of these settings are unavailable/disabled when using a DVI (digital) connection to the computer. Instead, these display settings are locked in to fixed defaults, which is why I was curious if you'd tried changing the connection type and what the result was. Factory reset on the monitor would both reset it to its defaults and erase any stored display settings that could be contributing to your issue.

There are other settings in Catalyst which might be relevent to your problems, such as LCD overdrive, or scaling. I suspect the driver for your series card also probably offers a few more configurable options than I get with my 4500 series card.
