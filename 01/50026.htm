screen saver problem - Microsoft Community

diamond383 say:

screen saver problem

what's the problem when screen saver won't display on desktop
arya s asok say:

Hello Dimond,
 
Thank you for writing to Microsoft Vista Community Forums.
 
The screen saver may not activate for several reasons:

- A background application is preventing the screen saver from activating.
- Power management is enabled and configured to activate before the screen saver.
- An anti-virus utility is currently running an active scan.
- Third-party wireless keyboards and mice may affect screen saver performance.
 
Try changing the screen saver in screen saver settings (try to change it to bubble) and verify if it works. Also ensure all available display drivers and software updates have been downloaded and installed. You may also try to check for updates and try to disconnect the wireless receiver for the keyboard and mouse.
 
If the issue persists, try to put the computer in clean boot state to make sure that third party applications are not interfering with the screen saver and verify if it works.

To perform a clean boot, follow the steps below:
1. Click Start, type msconfig in the Start Search box, and then press ENTER.
 (If you are prompted for an administrator password or for a confirmation, type the password, or click Continue).
2. On the General tab, click Selective Startup.
3. Under Selective Startup, click to clear the Load Startup Items check box.
4. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.
5. Click OK.
6. When you are prompted, click Restart.

If your issue is resolved after performing Clean Boot and then installing the updates, then follow the steps mentioned in the following KB article to narrow down the exact source:
http://support.microsoft.com/kb/929135

Also, see the section on how return your computer to a Normal startup mode by following the steps under “Reset the computer to start as usual”.

Hope it helps.
