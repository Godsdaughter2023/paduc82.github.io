Two big shortcomings of OneDrive on Windows 8.1 - Microsoft Community

emilio bresk say:

Two big shortcomings of OneDrive on Windows 8.1

Hello,

I've just upgraded from Windows 8 to 8.1, and have been highly disappointed by a couple of features of the new OneDrive functionality:

  1. In order to function on W8.1, OneDrive requires a Microsoft account to be connected to my computer.
    It is not possible to sign into the OneDrive App separately (i.e.: one cannot choose a Windows account to be used specifically for this app, without affecting other apps or the rest of the OS).

    I do understand this is supposed to be a functionality improvement, and it might in fact be quite convenient in most cases.
    But it can also be a trouble in some scenarios. Particularly, if you want to use a different Microsoft account for some other app (such as -in my case- the Xbox app).

    The Microsoft account I use for Xbox is different from the one I use for everything else (e-mail, OneDrive, etc.), so I need the ability to sign into the Xbox app and the OneDrive app with different Microsoft accounts.
    However, this is not possible. The Xbox app does allow for separate sign in, as long as you don't have a Microsoft account connected to your computer (but then the OneDrive app won't work). On the other hand, if I do connect a Microsoft account to my computer (so that I can use the OneDrive app), then the Xbox app will default to that account, and I cannot change it...

    To make things even worse, the OneDrive desktop app is now "integrated" into the OS, which means one just cannot configure it (or even sign into it) separately. That is: you can still see and use the OneDrive folder on the Desktop File Explorer, and manage your files from there, but only as long as you are using the OneDrive 'modern-style' app.
    In other words, if you want to use OneDrive at all, you're stuck with connecting a Microsoft account to your computer.


  2. The functionality to fetch files from a connected computer is no longer available in OneDrive, for computers running Windows 8.1.
    This isn't something that I used very often, but I did once or twice, and I found it to be a really handy feature. It's quite unfortunate to see it removed on W8.1.

Thanks,

Emilio

If you have a wireless network connection on the computer, then you can keep it secured by taking these security precautions. They are:

 

1) Use a network security key

 

If you have a wireless network, you should set up a network security key, which turns on encryption. With encryption, people can't connect to your network without the security key. Also, any information that's sent across your network is encrypted so that only computers that have the key to decrypt the information can read it. This can help avert attempts to access your network and files without your permission. Wi‑Fi Protected Access (WPA or WPA2) is the recommended wireless network encryption method.

 

2) Change the default administrator name and password on your router or access point

 

If you have a router or access point, you probably used a default name and password to set up the equipment. Most manufacturers use the same default name and password for all of their equipment, which someone could use to access your router or access point without your knowledge. To avoid that risk, change the default administrator user name and password for your router.

 

3) Change the default SSID

 

Routers and access points use a wireless network name known as a service set identifier (SSID). Most manufacturers use the same SSID for all of their routers and access points. We recommend that you change the default SSID to keep your wireless network from overlapping with other wireless networks that might be using the default SSID. It makes it easier for you to identify which wireless network is yours, if there's more than one nearby, because the SSID is typically shown in the list of available networks.

 

4) Position your router or access point carefully

 

Wireless signals can transmit a few hundred feet, so the signal from your network could be broadcast outside of your home. You can help limit the area that your wireless signal reaches by positioning your router or access point close to the center of your home rather than near an outside wall or window.
