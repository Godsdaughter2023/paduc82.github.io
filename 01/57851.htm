Hello Dr. Ramos,

Thank you for the Microsoft Answers Forum.  You didn’t say for sure but I am suspecting that this is a laptop computer that is giving you this problem?  The reason that pressing the “fn” key allows you to type normally is because somehow the “number lock” has inadvertently gotten turned on.  On a laptop keyboard there is no number pad on the right hand side of the keyboard like there is on desktop computer keyboards.  In order to have the number pad capability the number keys are assigned to regular “qwerty” keys.  When the number lock is on and you type using those keys you will get numbers instead of the letters on those keys.  To turn off the number lock you will need to press the “fn” key and the “num lock” key simultaneously.  It is on different keys for every brand of manufacturer so unfortunately I cannot locate it for you.  Usually the Function or “fn” keys are a different color.  On my personal Dell laptop the “fn” key and the Function Keys are blue.  You might also have a “number lock” indicator on your computer to let you know if this feature has been activated.  I hope this information is helpful.
