When you say restore your Windows 7 32 bit backup, what do you mean exactly? Please note, when migrating from Windows 7 32 bit to 64 bit, you will have to do a clean install. This means you will need to backup your personal files and settings using Windows Easy Transfer:

Using the Windows 7 64 bit disk, install the Windows 7 Windows Easy Transfer Utility from the DVD (To find it, just navigate to x:\support\migwiz, 'x:' represents the name of your disk drive. Once you have done that, simply do a clean install of Windows 7 64 bit disk.     armouredminion say:

    I had the same issue, very frustrated.  The way I got around the issue was:

    1) Define a static IP address for the Xbox in the DHCP table by adding a reservation to the Mac Address

    2) Set the pervious defined IP address in the DMZ (bypassing the firewall)

    3) On the Xbox, go in to settings, DVD & Blu-Ray and select Clear Persistent Data (do that twice)

    4) On the Xbox, go in to Setting, Network, Advance, Alternative Mac Address and Clear (This will reset your Xbox)

    (You may have to go in to the settings deeper if I've missed a step!)

    This has resolved my constant game dropping and may hopefully work for you.  If this does not work, then you may wish to setup a connection test on your line.  Go to www.TestMy.net from a PC and setup a test on your Network connection for 24 hours at 15 minute intervals.  If you find errors on you connection, you can then approach Virgin with the results.

    Best of luck, I hope you're back gaming soon....
