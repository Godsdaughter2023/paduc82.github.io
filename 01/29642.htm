Why does windows only let curtain web sites or icons on the desktop be brought down and pinned to the taskbar. It won't let me pin my bank icons, e-mail icon, movie theater icon and many others. It's confusing why some can and some can't be pinned. Anyone have an answer? Cause support doesn't.


Hi

 

Basically, you can only pin Programs or Utilities to the Taskbar.

 

If you have your Browser pinned to the Taskbar, then you can pin any Website to the right click Jump List for the Browser.

 

An easy solution is to create a new Folder somewhere in your User account folders, name the folder Desktop Shortcuts and move all of the shortcuts that cannot be pinned to the Taskbar into that folder.

 

Next, right click the Taskbar and select Toolbars/New Toolbar.

 

Select the Desktop Shortcuts folder that you created and click Select Folder.

 

This will give you an expandable toolbar that contains all of those shortcuts.
