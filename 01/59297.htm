Win 10 100% Disk Usage Solution - Microsoft Community

baranmavzer say:

Win 10 100% Disk Usage Solution

As many others, I had this issue for far too long. I tried every method mentioned before and none of them worked (or worked for a shot while). I finally found a permanent solution. I would like to share it here just in case other methods did not work for you (here is the link for my reply to previous thread). 

First, here is the source. 

Apparently, the issue is Microsoft Compatibility Telemetry. I will also paste the solution here just in case (use either methods, they are alternatives of each other):

Method 1 – Group Policy Editor (This method worked for me)

1) On your keyboard, press the Windows logo key  and R at the same time to invoke a Run command.

2) Type gpedit.msc and click OK.



3) Go to Computer Configuration >  Administrative Templates > Windows Components > Data Collection and Preview Builds.



4) Double-click Allow Telemetry.



5) Select Disabled, then click OK.



Method 2 – Registry Editor

1) On your keyboard, press the Windows logo key  and R at the same time to invoke a Run command.

2) Type regedit in the box and click OK.

3) Click YES when prompted by User Account Control.

4) Go to HKEY_LOCAL_MACHINE > SOFTWARE > Policies > Microsoft > Windows > DataCollection

5) Double-click Allow Telemetry.

NOTE: If you can’t find Allow Telemetry, create it: Right-click on DataCollection and choose New > DWORD (32-bit) Value. Then name the new value Allow Telemetry.

6) Set Value date to 0 (zero) and click OK.


Method 1 worked for me after trying all the other solutions. But here is one more method just in case: https://ugetfix.com/ask/how-to-disable-microsoft-compatibility-telemetry-compattelrunner-exe/
