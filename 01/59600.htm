j w stuart say:

1) Verify that KB3035583 and KB2976978 are installed.

　

Open Windows Update as follows:

Windows 7: Click on ‘Start’ / Control Panel / Windows Updates and in the bottom left corner click on the ‘Installed Updates’ link.

　

Windows 8.1: Right click on ‘Start’ / Control Panel / Windows Updates and in the bottom left corner click on the ‘Installed Updates’ link.

　

In the upper right corner there is a ‘Search Installed Update’ box. Enter the appropriate pair of KB numbers (one at a time) for your version of Windows. Example: KB3035583

　

Right click on each update to un-install it.

　

2) Open Windows ‘Task Scheduler’ (Start / Computer / Manage) and navigate to:

Computer Management / System Tools / Task Scheduler / Task Scheduler Library / Microsoft / Windows / Application Experience

　

3) Right click on ‘Microsoft Compatibility Appraiser’ and select the ‘Delete’ option.

　

4) Open Windows Explorer (File Explorer in Windows 8.1) and navigate to the C:\Windows\System32\GWX folder. Right click on the GWX.exe file and delete it.

　

5) Manually run Windows update and hide the two updates (KB3035583 and KB2976978) so they don't get automatically reinstalled.

　

6) Restart your computer, the GWX icon should be gone.

naman r say:

Hi Geoff0807,

This issue may be caused due to driver incompatibility. Make sure that you have the latest Windows 7 compatible drivers installed. If you still face the issue after the installation of the latest drivers, you may try the steps below:

1. Click Start and select Run.
2. Type cmd and Right click on cmd and click on Run as Administrator.
3. Type ipconfig /flushdns and hit Enter.

You should get a message that says: Successfully flushed the DNS Resolver Cache.

The ipconfig /flushdns command provides you with a means to flush and reset the contents of the DNS client resolver cache.

Also make sure that the DHCP is enabled on the computer. Don’t forget to update the MAC ID in the router so that you can access it every time.
