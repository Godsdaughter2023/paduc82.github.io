lemp say:

I assume that the error message you see actually refers to DAQexp.dll.

This error message is related to malware that has been partially dealt with by your antivirus/antimalware program.  Your security software has deleted the file DAQexp.dll but has not removed the part of the malware that tries to use that file.

If you would like guided help on removing the rest of the malware (which also will make that error go away), go to ONE of the following forums, read the "before you post" directions, and then ask for help.

https://forums.malwarebytes.com/topic/9573-im-infected-what-do-i-do-now/

https://www.bleepingcomputer.com/forums/t/678742/please-read-am-i-infected-is-closed-help-is-going-back-to-virus-removal-forum/

If you want to try doing things yourself:

Run a scan with free ESET online virus scanner:  https://www.eset.com/us/home/online-scanner/

Download, install, update, and run Malwarebytes:  https://www.malwarebytes.com/premium/

Note:  you will be installing a 14-day free trial program.  You can either uninstall it immediately after running it or you can keep it.  If you keep it but decide not to pay, after the 14-day trial is over, the program reverts to an on-demand only malware scanner rather than the real-time protection available with the paid product.

Download and run ADWcleaner:  https://www.malwarebytes.com/adwcleaner/

If after running those scans you still get the startup error, do the following:

Click Start, type Programs and Features, press Enter

  • Uninstall anything that relates to Wondershare

Download and run AutoRuns:  https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns

  • Click the "Logon" tab and look in the "Image Path" column for anything relating to WonderShare or DAQ or wshelper.
  • If you find any of those items, UNcheck the box at the left end of the line.
  • If you don't find any entries that include Wondershare, DAQ or wshelper, select the "Everything" tab and use the search function (CTRL+F) to look for them.

Unchecking a line prevents the entry from running the next time Windows starts.  Unchecking is reversible; only if you're certain a line item is undesirable should you delete it.
