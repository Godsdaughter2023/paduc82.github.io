almin ibrahimovic say:

Display Refresh Rate Switching - case of screen flicker for Dell XPS 15Z - solved

in relation to a post: https://answers.microsoft.com/en-us/windows/forum/windows_10-other_settings/windows-10-display-garbled-intermittently/d382c672-cdf0-4a14-843a-ea17cd349280 

the issue is caused by running on battery with Intel Graphics and Media Control Panel settings set with "Display Refresh Rate Switching" set to enable on battery profile by default.

To correct this do the following:

1. Open the "Intel Graphics and Media Control Panel", usually located bottom right corner of the taskbar, has a tool tip "Intel HD Graphics"

1.1 Go to "Power" this is on the left hand side of the "Intel Graphics and Media Control Panel"

1.2 Select "On Battery" from the "Power Source" drop down list.

1.3 under "Power Conservation Features" ensure that you untick / do not select the option for "Display Refresh Rate Switching"

Don't forget to apply the changes/save these.

Also if you have Dell 15Z laptop with Nvidia graphics onboard as well as the Intel 3000 HD, best you make nvdia your primary GPU to do this please follow:

2. Open Nvidia Settings

2.1 Select "Manage 3D Settings"

2.2 Select "High-performance NVIDIA processor" under "Preferred Graphics Processor" drop down menu, then apply changes

rajithr say:

Hi Mike,

Yes, you are right there is no direct download of the printer driver from the website.

Windows usually finds devices automatically when you plug them in or turn them on. If it doesn't, follow these steps:

  1. Go to Start and enter Devices.

  2. From the list, select Device Manager.

  3. Select Add a device.

If the Windows PC does not detect the driver automatically then, you may remove the printer driver and setup the printer again and check if that helps.

internet expert say:

Some things to check:

  • Remove any objects that can cause signal interference with the controller.
    • Physical objects that break the line of sight between the controller and the Xbox. You might need to reposition your Xbox or other things.
    • Other devices/appliances that broadcast or emit radio signals nearby, especially in the 2.4 GHz range (routers, microwaves, Bluetooth, wireless landline phones, etc)
  • Ensure the 3.5mm port in the controller is clear of dust. Slight vibrations or movement can interrupt the audio.
  • Connect the controller to the Xbox via USB and check for an update.
