Windows 10 Start Menu Modern Apps, Search etc won't open - Microsoft Community

lforbes say:

Windows 10 Start Menu Modern Apps, Search etc won't open

Hi,

I had an issue where One user on a computer who was an administrator had a broken Start Menu after the latest Build. Other users were not affected by this issue.

I did the Powershell commands posted on other websites that reinstalled the Modern Apps but that did not work.

Here is what I found that did work for those who are having issues.

1> Create a 2nd Administrator account and View all Hidden Files and Operating System Files.

2> Logoff the problematic account and logon with new account.

3> Navigate to C:\Users\ProblematicUserAccountName\AppData\Local\Microsoft\Windows

4> Create a folder called "old".

5> Move the UsrClass.dat, and all the log files and the WebCacheLock.dat that are in the root of the directory above into the old folder you just created.

6> Logoff and log back on as problematic user.
