The console is looking for a previous version of the system update. Follow these steps:

    1. Find a USB flash drive with at least 90 MB of free space.
    2. Plug the USB flash drive into a USB port on your computer.


Note The USB flash drive must use the FAT32 file system (Microsoft Windows).

  1. Download the system update for error code xxxx-3151-xxxx-xxxx-xxxx-xxxx.
  2. Click Save to save the console update .zip file on your computer.
  3. Unzip the file. If you are using Windows XP, Windows Vista or Windows 7, double-click the .zip file to unzip it.
  4. Copy the contents of the .zip file to the root directory of your USB flash drive.
  5. Unplug the USB flash drive from your computer.
  6. If there is a disc in the console, remove it.
  7. Turn off the console.
  8. Remove all console storage devices, including any USB flash drives, external hard drives or the internal hard drive on the Xbox 360 S console. Learn how to remove the hard drive.
  9. Plug the USB flash drive with the update into a USB port on console.
  10. Turn on the console. When the console starts, the software update should automatically start.
