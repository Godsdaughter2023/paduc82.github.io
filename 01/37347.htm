lemp say:

The "top part" of the Windows 7 Start Menu are items (shortcuts) that are "pinned" to the Start Menu.  That is, once there, they stay there until you specifically remove them

The "bottom part" of the Windows 7 Start Menu are shortcuts that Windows puts there automatically as "most frequently used" applications. 

To remove a pinned shortcut from the top part of the Start Menu, right-click it and select "Unpin from Start Menu."

You can't manually place shortcuts on the bottom part of the menu although you can right-click on any item there and select "Remove from this list."  This will permit the next "most frequently used" program's shortcut to appear at the bottom of the list.
