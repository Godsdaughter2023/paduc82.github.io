Hi MitchBaxter1
I am Vijay, an Independent Advisor. I am here to work with you on this problem.

Please try these options one by one and check after each option.

1. If you are able to open MS Store, open MS Store > Click on your profile picture on top right and sign-out. Then sign-in again.

2. Run Windows Store Apps Troubleshooter
Windows Key+X > Click Settings > Click Update & security > Click Troubleshoot > Scroll down to the bottom > Click Windows Store Apps > Click Run the Troubleshooter

3. Reset Windows Store through Command Prompt
Type cmd in Windows Search box > Right click on Command Prompt > Run As Administrator > Type WSreset.exe and click Enter > Reboot your computer

4. Re-register All Store apps (You will get many Reds, ignore them)
Windows Key+X > Windows Powershell (Admin) > Copy the following from below and right click in Powershell to paste > Enter > Restart your computer

Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

5. Uninstall & Reinstall Store
Windows Key+X > Windows Powershell (Admin) > Copy the following from below and right click in Powershell to paste > Enter

Get-AppxPackage -allusers Microsoft.WindowsStore | Remove-AppxPackage

Copy the following from below and right click in Powershell to paste > Enter > Reboot your computer

Get-AppxPackage -allusers Microsoft.WindowsStore | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

6. Check your Antivirus
Temporarily uninstall / disable AV. Enable after troubleshooting.

Do let me know if you have any more question or require further help.

"The internet says 10 should have DirectX 12 and you can down grade it back to 11"

Direct X does not work like that. There is no need to downgrade DirectX, if you could.

Win 10 is the first Windows to have DX12, but as with Win7/8/8.1, Win10 also has Direct X 9, 10 and 11.

The game chooses which Direct X files it requires from many different DX file options.

- Sometimes one, or several Direct X files may be 'missing' in a recent 'clean' install of Win7.

I don't know if Win10 would have any missing files, as I did an upgrade from Win7, but if it does...

The Direct X Web Installer will install any 'missing' files -

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

The Dirext X redist (June 2010) installer will reinstall all Direct X files (except the couple of DX12 files),

and replace any corrupt files in the process -

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

With Direct X is sorted out....

- The graphics card needs to be able to use specific levels of DX.  An old DX9 card cannot use DX10 or DX11.

The only graphics cards that can make full use of DX12 are the most recent series.

- The transition from DX 11 to DX 12 is a bit more complex than previous forms of Direct X.

DX12 has a few 'feature' sets that will work with DX11 graphics cards.

It would not allow you to select 5.1 uncompressed when set to auto because your TV does not support 5.1 uncompressed, which the TV reported during the HDMI handshake (generally, uncompressed 5.1 would require a receiver).
