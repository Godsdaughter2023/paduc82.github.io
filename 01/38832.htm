    In most cases, Windows Photo Gallery stores information such as captions, dates, descriptive tags, geotags and people tags back to the file as metadata using industry accepted standards such as EXIF, IPTC and XMP. This is a good thing, in which it makes this information portable as it "travels within the image file". Here is an example of the metadata of a photo edited in Windows Photo Gallery and uploaded into Flickr. Here is a good article on the subject of file metadata when it comes to photo organization.

    Sadly, Windows Photo Gallery is no longer supported, so you should consider comparable alternatives which meet your archiving needs. Digikam, Photo Supreme, ACDSee as well as Adobe Lightroom or Photoshop Elemens are good options.

    I also have a sizable photo collection composed of old scanned images and have spent more than a decade adding relevant metadata to them. If you are concerned about future generations having access to the files, do make sure it is being stored within the file in industry accepted standards. Backup often using different storage mediums. Also, as a redundancy, I also routinely create an export of the file metadata to a tab-delimited text file which I described in this blog post: Accessing Windows Photo Gallery Metadata using Exiftool 

    Hope this helps.

    brian-- say:

    Hi Lauretta1965,

    What is the model of your DVD drive? You may need to check the manufacture’s website for the latest driver for your model. You can also do the following to check the status of your DVD drive and check for updated drivers:

    1.       Click Start

    2.       In ‘Start Search’ type Device Manager, press Enter (provide administrative credentials if prompted)

    3.       Click the “+” next to DVD/CD-ROM drives to drop down the menu

    4.       Check the status of your DVD drive by right-clicking the device, clicking Properties, and then under the General tab check what it says under “Device status”. Let us know what the status was.

    5.       Then check for updated drivers by right-clicking that device and then clicking Update Driver Software… (this step may take a moment so be patient). 

    Even if this method doesn’t find a need for an updated driver, you may still check at the manufacture’s website to see if the latest one they have resolves the issue. 

     

    If you don't see the DVD drive listed in the DVD/CD-ROM drives category, see if you have a category for Unknown Devices.

    Microsoft Print To PDF - Possible Fix For Missing Files - Microsoft Community

    pete mc say:

    Microsoft Print To PDF - Possible Fix For Missing Files

    I've been having problems with Microsoft Print To PDF where it appears to create a PDF (I can name it and pick a location), but the file is never actually created.

    And I have read the various (older) posts here which seem to refer to deleting and recreating the MS Print To PDF printer in Devices and Printers.  None of those options worked for me.

    So I started with an older HP nw8440 and upgraded to Windows 10 will all updates.  The slowly went through my normal setup of all the Windows Options.  Eventually I got to a change I've been making since XP when temp files were not handled very well.  I changed the Environment Variables for both the User variables and System variables for TEMP and TMP to C:\Temp

    After making the change, Microsoft Print To PDF again would appear to create a PDF, but the file was no where to be found on the entire drive.  I changed all the variables back and discovered after changing one at a time that if you change:

    User variable for [Name]; TMP Variable from %USERPROFILE%\AppData\Local\Temp to C:\Temp the PDF will no longer be created.  Change it back and all is good again.

    Hope this helps anyone else that happened to change this variable.
