  • The surface book dock is only compatible with monitors that support DisplayPort.  The article says a workaround is to use an HDMI adapter and connect to the monitor using HDMI.  This partially resolved my issue.  It reduced the number of times the dock lose connection with the external monitor 95% of the time.  It still loses connection sometimes.

Understanding and troubleshooting Live Mail “CCC” error codes
Network error codes
Network / Internet error codes in Live Mail will all start with 0x80072, followed by a combination of 3 letters / numbers – which give the root cause.

Example: Error: 0x80072F81

Root cause: ERROR_INTERNET_SERVER_UNREACHABLE

Troubleshooting: check internet connectivity, flush DNS cache, check mail server (possible outage)

HTTP error codes
HTTP errors are indicated by 0x80AA4** or 0x80194** (where ** = a combination of 2 letters / numbers)

The last 3 digits of these error codes are standard Hex codes – which give the root cause.

Example:

• Error: 0x80AA41F7

• Hex --> decimal: 1F7 = 503

• HTTP error 503 = service is temporarily unavailable.

To convert hex to decimal:

http://www.binaryhexconverter.com/hex-to-decimal-converter

For list of common HTTP errors codes and their meanings:

http://en.wikipedia.org/wiki/List_of_HTTP_status_codes

Passport / Authentication error codes
Passport / Authentication errors will start with 0x000488** or 0x800488** (where ** = a combination of 2 letters / numbers)

For Hotmail accounts – 0x800488** error codes usually indicate an issue on an account-level, for example:

• 0x800488fb – Account suspended

• 0x800488fe – Account compromised

Troubleshooting: Check the account!

Exception: 0x80048849 – MTU / network issue!
