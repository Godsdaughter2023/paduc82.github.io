There may be some utility application running in the background that is generating files on D. 

Make sure that the Indexing Service is disabled (http://lifehacker.com/031440/turn-off-indexing-and-speed-up-windows-xp).  I'd disable Indexing for all drives, but certainly for D.

Take a look at D in Windows Explorer after first making sure the following options are selected in Tools > Folder Options > View

"Show hidden files and folders"
UNcheck "Hide extensions for known file types"
UNcheck "Hide protected operating system files"

If D is the restore partition, there shouldn't be a lot of top-level folders, so you may be able to diagnose things manually.  Otherwise, download and run either JDiskReport or TreeSize.
