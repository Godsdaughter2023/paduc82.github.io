internet expert say:

When turned on, the Xbox is usually running fairly warm. Roughly 40C (~105F?) is pretty normal and still somewhat cool. On a gaming computer (for comparison), 35-40C can be normal idle temps for powerful hardware with standard air cooling.

The Xbox will give off warm air in normal operation, and likely still does especially when you have a game suspended in the background or maybe downloading and installing an update. The concern for generated heat should only be a worry if the fan is kicking into constant high gear and the Xbox is SCORCHING hot.

You have it well ventilated (as you say), hopefully it's mounted horizontally (as it was designed; it's not meant for standing vertically), and your temps are seemingly normal -- there should be nothing to worry about. Like the last guy said, it also helps to clean the dust off the Xbox once in a while. Dust obstructs airflow, leading to heat buildup. Also make sure the console is not in direct sunlight or near other heat sources.

"which services I need running"


Just  to keep the terminology correct -

Services is the area where you Stopped the bitlocker service.

Startup is where you keep things from loading when you boot into Windows.

- For the most part disabling what is in Startup is a personal need thing, but there are some things that

do need to be left.

I leave the Intel common user interface, and Installshield update service, mainly because I haven't bothered

finding out what they do.

I also leave Anti-Virus,  Realtek audio, keyboard profile software and a couple of other thing active.

I don't have Adobe Flash loading with Windows, but I don't use it a lot, and when I do the software that

uses it launches it automatically.

The main thing here is if you have software you use all the time, it is more convenient to have it launch with

Windows.

- Regarding Services. This also has to do with personal preferences to some degree, but most stuff can be left alone.

I do not use the Services tab in System Configuration, as I believe this is only a one time setting.

(Clean Boot, linked earlier, is the better tool for checking conflicts with Services)

For manually stopping or disabling a Service, type - services.msc  - into the Run command to open

the main Services Window.

There is an Adobe Flash updater in Services which should be kept on, but when Adobe does an update

I select the option to inform me, but not to install automatically.

I've always preferred this with any update, including Windows updates as there is less chance of a corrupt

update if you are aware it is happening, and not doing something  system intensive while the update is being installed, particularly when an update requires reboot to finish.

- Registry. Type -  regedit  - into the Run command and browse the registry using the file path in the

previous post and follow the instructions. You can backup the key you are altering if you wish, as using the

backup to change the value back to whatever is current would be quicker than browsing back to the key again.

- When toying around with things you are not familiar with creating a System Restore point before you

make those changes is a good insurance policy.

If you don't know how to create a System Restore point - Time to do some research.

Actually, all of the above is covered by tons of tutorials out there in the www.

SSD vs HDD - The system specifications for you machine should be in your user manual or on the

mfg's web site.

Prefetch is useful with HDD's, as it improves performance somewhat,  but Prefetch isn't really necessary with SSD's,

and can shorten SSD lifespan somewhat (there is debate about the seriousness)

More research for details. ;D

- If you are using an HDD and have Prefetch enabled,  the Prefetch will be rebuilt over time, if you delete the files.

- Here's another trick for when you are not sure of the impact of deleting files.

Rename the file/folder. You can do something such as add another couple of P's to the front, or a 1a which will move

the file to the top of the Window. If nothing goes wrong over the next couple of days you can delete the folder.

This method protects against emptying the trash bin accidentally and loosing the file/folder.

- Speaking of deleting file. A large TEMP folder can slow things down as well.

Run Windows Disk Cleanup . This will delete many of the files in the Temp folder.

Or, for a full clean out,  go to - your User\AppData\Local\Temp

Open Temp folder and delete everything that can be deleted ( files that are in use cannot be deleted).

- Defrag your HDD

- Run the toolkit Zig-zag had you use again, and analyze the results.

amrita m say:

Hi ahmadnaziF,

 

 

You cannot prevent the computer to get into sleep mode when Bittorrent or download manager is running as there is no activity that happens in the foreground that might keep the computer awake.

 

The download happens in the background and also it does not require the computer to be awake. Download will continue even if the computer is in sleep mode.

 

Also, you may change the power plan of not letting the computer to sleep manually when these programs are running and again set it back to the normal setting when the programs are not running.

For more information follow the link given below.

Change, create, or delete a power plan (scheme)

http://windows.microsoft.com/en-US/windows7/Change-create-or-delete-a-power-plan-scheme

 

You may also check the option ‚ÄòAllow this device to wake up the computer‚Äô for network card. This will allow the computer to wake up from sleep whenever the network card is in use.

2.       Expand Network Adapter tab

3.       Right click on the network adapter

4.       Select Properties and go to Power Management

5.       Check the option ‚ÄòAllow this device to wake up the computer‚Äô

Check if that helps.
