considerate_guy say:

The .wlmp is your project file.  It's not a media file at all since all it contains is a list of the filenames of each photos, videos and songs used in your movie, the list of transitions applied, captions added, etc.  It's like a recipe for making your movie, but it contains no media within it (much like a recipe is just printed words, not something edible).  That explains why you can't play it anywhere else.

 

What you want to do is to turn that project into a movie using the File > Save movie option.  That will follow the instructions in the recipe (project file) and turn it into a cake (final movie).

 

However, you mentioned one other thing which is you tried this and you saw it stuck at 0% progress.  This is an indicator that Movie Maker is stuck on a file right at the beginning of your project.  Usually this means the source file (video or song) is incompatible with Movie Maker.  The solution is to convert that source file to another format and use the converted file instead of the original.

 

I use Freemake Video Converter to turn all my source videos into .wmv files.  I then use Freemake Audio Converter to turn my songs into .wma files.  When I edit with .wmv and .wma files, I have no problems saving my final movie.  So give that a try and let us know how it works out.

a. user say:

Hello Joyce,

 

Thank you for posting your query on Microsoft Community. We are glad to assist you.

 

I would like to inform you that the built-in option to change the Desktop Fonts and Colors was removed in Windows, starting with Windows Vista.

The reason is that any font color would become invisible, depending on the Desktop Wallpaper that was being used. An example would be a picture or multi-color wallpaper. If the font is a dark color and the text was positioned over a dark part of the wallpaper.

Windows uses a White font with a Drop Shadow effect. This keeps the font visible, regardless of the background color of the wallpaper. In extreme cases where the desktop is a brilliant white, or other very light color, the fonts will turn Black.

 

However, you may try using the various themes available for Windows 8.1 provided in the following link.

 

Themes

http://windows.microsoft.com/en-US/windows/themes

 

And about the 3D effect settings, I would like to inform you that these settings are totally dependent on the display/graphics adapter that you’re using. However, even if the graphics adapter installed on your computer does support 3D effects, but is not compatible to Windows 8, you would still not be able to use the 3D style and effects.
