Click Next.

'mike p say:

to install windows you would typically delete drive 0, each (all) of the drive 0 partitions showing, then install into that empty space.

However - in an install of windows from boot like you are attempting , there will be no windows.old and you will completely delete all of your personal files and they will be non-recoverable unless they happen to be in OneDrive or saved somewhere else.  Which you can do by the copy or move command from the repair environment command prompt manually.

You should look in your bios/setup and may find the setting for uefi or legacy, might say uefi or mbr -  make sure your boot is set to uefi, which uses a gpt setup hdd or ssd - if its saying the drive is gpt, sounds like your bios is looking for a MBR system hard drive.

slance310 say:

Hi,


What I would suggest based on your last reply, would be to do a Custom Install and delete everything on the drive... Your machine appears to be using UEFI Bios... This requires a GPT partition style...

I'm not sure why the errors with the partition selection or the BcdEdit commands...

One possible problem may be the way the Installation Media drive was created or launched...
A UEFI installation needs to be created for a UEFI Bios machine specifically, or a Legacy Bios machine specifically...

You may need to also select the drive properly in "Boot Options" as a "UEFI Flash Drive" or simply "Flash Drive"...

I know you have files you need to recover... You should be able to do this with your installation media...

Either to another internal drive, or a USB drive...

Copy\Paste From Bootable Command Prompt...

This basically works the same as any Copy\Past Function...
The screens that you work from just look different...

The drives letters will be different than from within Windows,
but their names will be the same..

Boot computer from installation media...
Select "Repair" on the lower left of the "INSTALL" screen...

Select Troubleshoot.... Command Prompt
Enter NotePad into Command Prompt and hit Enter...

Click on "FILE" Tab and select "OPEN"
Click on ThisPC or use the Dropdown menu to browse to the files you want to copy...

Right click on desired files and select "Copy" as normal...
Browse to the location you wish to save the files and select "Paste", again as normal...

Ignore anything on the bottom of the window concerning "SAVE", "CANCEL", "File Type", and ETC...
