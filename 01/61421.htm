I'm running Windows 10 upgraded from Windows 8.1 on my desktop and upgraded from Windows 7 on my Notebook, both were presenting the same problem. What helped me was deleting the "Search Data" Folder. I believe it contains the search indexes and for some unknown reason those indexes were corrupted.

Below are the steps I used (Sorry, I'm running the Brazilian Portuguese version so some items may be labeled different in English):

  1. Open Configuration/Settings
  2. Type Services in the search bar
  3. Click on Show Local Services
  4. On the Services window, look for the Windows Search service and stop it
  5. Open Windows Explorer
  6. Navigate to C:\ProgramData\Microsoft\Search
    • Note that you may need Administrator credentials
    • Note also that ProgramData is a hidden folder
  7. Delete the Data folder
  8. Restart the PC

Let me known if this solves your problems. ;)
