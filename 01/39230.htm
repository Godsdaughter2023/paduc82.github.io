a. user say:

Despite what was mentioned above it can be possible to swap saves between profiles. However it will depend on the game and you are unable to gain achievements while using the copied save.

I'll describe how to do it using a game I know it works on, Mass Effect.

Sign in with the account you want the save copied to. Then go to My Xbox and scroll to the end and pick System Settings. Now pick Memory and on the new screen the device the save is saved on.

Now choose games and scroll until you find the game, Mass Effect in our example, and enter the folder by pressing A.

Find the save you want copied and press A. You'll now be presented with three options but we only want the first, Copy.

Press A on Copy and choose to save it to the same location.

Now if the game allows the save to be copied you will have a new save created belonging to the new profile. Remember though that you wont be able to gain achievements by using that save, so your girlfriend will have to decide if she wants achievements or to retain her progress.
