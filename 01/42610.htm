For your school WiFi, you will never be able to connect to it if you need to install a program for it.

There cannot be a work-a-round for it since the processor is ARM class and the Windows 8.1 installed on it is actually called "Windows RT".  It is a recompiled version of Windows 8 and will not run any Windows software except programs available through the "Store".

You may be able to sell the tablet and get another one that has the ATOM processor for about the same price as the RT and it will run regular Windows software.  Microsoft does have an online store and they sell a wide variety of those tablets.
