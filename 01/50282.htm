a. user say:

Hi,

The "Remember each folder's view settings" is not an option in  windows 7. It is not possible to set Windows Explorer to remember individual folder view settings for folders included in libraries.

Refer to the discussion on the link : http://answers.microsoft.com/en-us/windows/forum/windows_7-files/is-remember-each-folders-view-settings/633192ab-e435-48cd-8fcd-999e7195653c

Though that option is not there but you can set a similar kind of view for similar folders. Follow the below steps for that:

You can customize a folder to appear the way you like it in File Explorer, and then apply the view to other, similar folders (folders that are optimized for the same kind of file).

1)      Tap or click to open Computer.

2)      Expand a location to get to a folder that isn't in a library, and open the folder.

 

Note: Changing the view of folders that you access through libraries works a little differently. If you open a folder under Libraries in the navigation pane in File Explorer, and then change the view, that view is applied to all the other folders that you access through the library as well. You can also change the kind of file for which the library is optimized. Select the library in the navigation pane, tap or click the Library Tools tab, and pick a kind of file under Optimize library for.

 

3)      If the folder isn't optimized for the right kind of file, change it. Tap or click the Home tab, tap or click Properties, tap or click the Customize tab, and pick one of the choices in the Optimize this folder for box.

4)      Use the View tab on the ribbon to change the folder view—the layout (icons, list, details, and so on), sorting, grouping, the columns that appear, and the other file info that appears.If you don’t have the ribbon ,press Alt key on the keyboard and it will bring up the options.

5)      Click on the Start button and go to Control Panel. In there open up Folder Options.

6)      In the Folder Options dialog box, tap or click the View tab, tap or click Apply to Folders, tap or click Yes, and then tap or click OK.

7)      If you want to view all folders on your PC using the same view, repeat steps 1-5 for each of the five folder types (General items, Documents, Pictures, Music, and Videos).

 

Note: The Apply to Folders button won't be available if the folder you selected in step 2 is in a library.

Note: When you apply a view to folders that are optimized for a particular kind of file, the view will also be applied to any new folders of that type.

For more information ,click on the  Start button and type Help in the search box and open up Windows Help and Support . In the search box at the top type in ‘Change folder options’ .Browse through the options for more help.
