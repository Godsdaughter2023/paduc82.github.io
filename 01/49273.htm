uvarllalla say: i have not read all the replies but i too have been struggling and searching around the net for answers, and with help from all of them, i was finally able to find the problem and fix it(please note i am not an expert and i do not know the consequences of doing this, but since it worked im happy and i think you will be too).

FIND THE PROBLEM
-first, you can check your event viewer to see what the problem is, and heres how to do that:
-search for event viewer in the start button.
-next, expand the 'windows logs' part and click on 'application'.
-press ctrl+F to open the search button, and then search for "explorer".(if nothing is found,try opening file explorer one more time until it restarts so that the event logger will pick it up.)

once i done this, i was able to find that 'UIRibbon.dll' located in C:\windows\system32 was causing the problem.
 
FIX THE PROBLEM
-open your 7-zip program and navigate to the drive where the dvd is (mine was F:\ in this case and yours should be different)
-look for the 'sources' folder and enter it.
-look for 'install.wim' and double click it.
-go to 'windows' , then , 'system32' inside the windows folder.
-your address bar should look like this now - F:\sources\install.wim\Windows\System32\
-look for UIRibbon.dll and extract that to your desktop or whatever folder you have admin power over and know the address of.
-you can get an address by making a shortcut on your desktop and then browse to wherever you extracted the dll file to(like 'desktop' for example), and then just copy that address there and click 'cancel' to cancel the shortcut.

time to act like an IT expert:

open your command prompt as administrator and type/copy and paste the following:
takeown /f C:\windows\system32\UIRibbon.dll
 then type:
icacls C:\windows\system32\UIRibbon.dll /grant administrators:F

this is so that the file can be replaced without getting a 'no access' situation.
then type the following:
copy C:\(your desktop address from the shortcut)\UIRibbon.dll C:\windows\System32

type 'y' and then, celebrate because the problem is fixed. if you struggle, just use the following links to guide the 7-zip extraction process and the command prompt process:

diego rios_06 say: Hi,

I'm Diego Rios, an Independent Advisor willing to help other Microsoft customers.

Change Time Zone in Settings
1. Open the Settings app, and click/tap on the Time & Language icon.
2. Click/tap on Date & Time on the left side, and turn off Set time zone automatically on the right side.
3. Select the time zone you want to use in the drop down menu on the right side, and turn on (default) or off to Adjust for daylight saving time automatically.
4. If you like, turn on Set time zone automatically on the right side
5. When finished, you can close Settings if you like.

Change Time Zone in Control Panel
1. Open the Control Panel (icons view), and click/tap on the Date and Time icon.
2. Click/tap on the Change time zone button under the Time zone section.
3. Select the time zone you want to use in the drop down menu, and click/tap on OK.
4. Click/tap on OK.
Please let me know if this helped and works.

Diego R.

davem121 say:
Hi JL

Do you want to totally erase Windows from that system or reset Windows?

1.

Totally erase Windows:

Do you have Windows bootable installation Media?

If so, boot your System from that

Then Press Shift +F10 to open Command Prompt

In command prompt, run these commands one at a time and hit Enter after each one:

diskpart

list disk

(Note down the disc number of that disc and replace x in the next command)

select disk X

clean

When that completes, that drive will be wiped



2.

Reset Windows to remove yourself

Press the Shift Key and click Restart
Your PC will boot into the Windows Recovery Environment
Go to Troubleshoot - Advanced Options - Reset this PC
Choose to keep nothing
Then start the Reset
