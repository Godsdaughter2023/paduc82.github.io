olaf engelke say:

Zoom happens in certain applications (like Internet Explorer), if you double tap  on them.

You can try to disable double tap for Internet Explorer by following method:

  • In notepad, create a new text file and give it the name nodtap.css (or any other name you find appropriate, the extension is the important part).
  • Paste following lines into it:

body {
-ms-touch-action: manipulation !important;
}

  • Now in Internet Explorer navigate to Internet Options 
  • In the lower right corner of the General tab click the button Accessibility.
  • In the section User style sheet tick the checkbox "Format documents using my style sheet".
  • Then click the Browse button and select your CSS file created from above.
  • Click OK and close Internet Explorer.

If you reopen the browser, it should no longer respond to double taps at all with zooming.

3. On the "Start" menu, click "Run". 

4. In the "Open" box, type "msiexec /regserver", and then press ENTER.

Step 2: Locate the .inf file

Locate the .inf file that is included with the driver installation program. Generally, the .inf file is located in one of the following folders on the driver installation media:

  • In the same folder as the Setup program.
  • In a subfolder that is in the folder that contains the Setup program.

If the driver files are packaged in a single executable file, you may be able to extract them or otherwise access them.

Hide Windows Update
To hide an update, follow these steps:
1. Click Start, point to All Programs, and then click either Windows Update or Microsoft Update.
2. Click Custom.
3. Click to clear the check box that is next to the update that you want to hide.
4. Expand the update that you want to hide, and then click the "Don't show this update again" check box.
5. Click Review and install updates.
