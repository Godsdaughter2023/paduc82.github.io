How to Stop and Start OneDrive Syncing - my simple method - Microsoft Community

pierrecornelissen say:

How to Stop and Start OneDrive Syncing - my simple method

I have created a simple solution for me to stop and start OneDrive Syncing by using the Windows Task Scheduler. 

I have created a basic task to start OneDrive with the following:

  • Task Name: Start OneDrive
  • Trigger: Daily
  • Scheduling: Your prefered start time (enter the date when you want the syncing to commence)
  • Action: Start a program
  • Script: Where the OneDrive image resides, normally in C:\Users\XXX\AppData\Local\Microsoft\OneDrive\OneDrive.exe

 

When OneDrive starts it will open the OneDrive window and if set the OneDrive taskbar icon will appear.  I work from home, so I do not have to log off – you have to try that one by yourself. DO NOT USE THE ADMINISTRATIVE OPTION.

 

 

The second basic task is to stop OneDrive:

  • Task Name: Stop OneDrive
  • Trigger: Daily
  • Scheduling: Your prefer stop time (enter the date when you want the syncing to commence)
  • Action: Start a program
  • Script: Taskkill.exe with the arguments /im OneDrive.exe /F

Once OneDrive is stopped, when set, the OneDrive icon may still be shown as a toolbar icon, but as soon as you move the mouse over the icon it will disappear.

Save and test check with a start and stop time close to each other – make it 10 minutes between start and stop and verify - then setup the real times.

If anyone has a better idea it would be great to hear from you.

Hope this will help you all.

tombbabe say:

Drive C is full, how do I make Drive E (empty) start to be used?

Drive C is full. How do I make Drive E work now?

shane-- say: Tom,

  Everything that is using the C: drive is going to continue to use it.  Nothing automatic is going to happen and start using the E: drive.  If you have a lot of files stored on the C: drive you can manually move them to the E: drive to free up space.  If you want programs that are currently installed on C: to be on the E: drive, you will have to first uninstall them and then do a custom re-install; selecting the E: drive for the installation.  I hope this helps; and please let us know if you have any other questions.
