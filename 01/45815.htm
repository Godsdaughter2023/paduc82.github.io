greginmich say:

Others might not be quite so satisfied with the answer provided here, because it doesn’t actually give you the ability to scan single files – and it’s also pretty clumsy in comparison to this batch-file method. This ultra-simple batch file can also be used to add the required functionality to the right-click context menu without any downloads or registry modifications:

 

Open Notepad and copy and paste these two lines:

 

"C:\Program Files\Windows Defender\MpCmdRun.exe" -Scan -ScanType 3 -File %1

Pause

 

Name the file something like “Defender Scan”; give it a .bat file extension; and save it to your desktop. Now simply drag and drop the icon for any single file, folder, or flash drive onto the batch file icon for a scan. If you’d prefer to use the context menu, then simply move the batch file to the C:\Users\<account name>\AppData\Roaming\Microsoft\Windows\SendTo folder, as suggested by manfeildman in this thread:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-do-you-scan-a-file-in-windows-8/3277ad77-b400-47c1-8508-1dfbf1729d3d
