You are unable to the start the WiFi Intel Centrino Ultimate-N 6300 AGN device getting error code 10 in Windows 8.

 

The Code 10 error code is generated when Device Manager has insufficient information or the error that is bubbled up by the device drivers is not recognized by Device Manager. Additionally, this error may occur when one of the drivers that the device needs does not start.  

 

Please follow the methods.

 

Method 1:

The most common resolution is to update the drivers that are associated with the failing device. To do this, follow these steps:

a)      Press Windows Key + R

b)      In the run box, type devmgmt.msc and press enter.

c)       In the device's Properties dialog box, click the Driver tab.

d)      Click Update Driver to start the Hardware Update Wizard.

e)      Follow the instructions to update the driver.

 

Method 2:

If you suspect that the Code 10 error is related to incompatible, outdated, or corrupted device drivers, try the following troubleshooting methods.

Update the hardware's device driver through Windows Update. To access Windows Update, visit the following Microsoft Web site:

http://update.microsoft.com

 

Method 3:

Update the device drivers from the vendor's Web site. Follow their installation or update instructions.

If the device was preinstalled on the computer, visit the computer manufacturer's Web site.

If the device was installed after the purchase of the computer, visit the device manufacturer's Web site.

If the device was preinstalled on the computer, and the computer manufacturer does not have an updated driver for the device, visit the device manufacturer's Web site.
