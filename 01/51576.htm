How to capture a Windows Live Writer Log File - Microsoft Community

a. user say:

How to capture a Windows Live Writer Log File

How to capture a Windows Live Writer Log File

a. user say:

The log file is a text document that contains logs of actions taken on Windows Live Writer.  You can capture the log file by following these steps:

Writer 2009 (Windows XP)
  1. Open Windows Live Writer and then click on the Help menu.
  2. Select About Windows Live Writer.  A small window showing your version of Writer should pop-up.
  3. Click on the Show log file link.
  4. Another window should open containing a Windows Live Writer text document. This is your Windows Live Writer log file.

 Writer 2011 (Windows Vista and Windows 7)

  1. Open Windows Live Writer and then click on the menu button located in the upper left-hand corner of the window.
  2. Select About Windows Live Writer.  A small window showing your version of Writer should pop-up.
  3. Click on the Show log file link.
  4. Another window should open containing a Windows Live Writer text document. This is your Windows Live Writer log file.

Below is an alternative way for those who cannot get the log file by following the steps above such as not being able to open Writer, etc.  Note that the steps are for PC only.

  1. Click Start
  2. Click Run
  3. If you are using Windows XP:  Copy and paste, %userprofile%\Local Settings\Application Data\Windows Live Writer\Windows Live Writer.log  
    If you are using Windows Vista or Windows 7:  Copy and paste:  %userprofile%\AppData\Local\Windows Live Writer\Windows Live Writer.log
  4. Hit <Enter> on your keyboard
  5. Save the content of the notepad
