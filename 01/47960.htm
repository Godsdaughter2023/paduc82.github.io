=======================================
I would suspect Paint is having a problem with the
.PNG graphic filter. Reinstalling the filter may be the fix.

Before proceeding it would be a good idea to create a
System Restore point.

Windows 7 - Create a Restore Point
http://windows.microsoft.com/en-US/windows7/Create-a-restore-point

Since the filters are shared files...a simple reinstall of
a program will not work. First the graphics filter must
be found and renamed. Then a reinstall of a program
that includes graphics filters should replace it.

PNG filter     -       PNG32.FLT

Do a search for .flt or try the following path:
C:\Program Files\Common Files\Microsoft Shared\Grphflt

Example...rename the .png filter from :
PNG32.FLT  -  to:  PNG32.OLD

Now install a program that includes graphics filters...

The freeware IrfanView for example.

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Download IrfanView
http://majorgeeks.com/IrfanView_d4253.html
(filename  :  iview436_setup.exe)
(uncheck the box if you do not want Google Chrome)

Get the plug-ins too...

IrfanView Plug-ins
http://majorgeeks.com/IrfanView_PlugIns_d4908.html
(filename  :  irfanview_plugins_436_setup.exe)

Good luck...

Disposable culture? The Microsoft Sculpt Comfort Mouse appears not - Microsoft Community

rupert_w say:

Disposable culture? The Microsoft Sculpt Comfort Mouse appears not to be designed to work reliably with rechargeable AA/HR6 batteries.

As you can see from the photos below, the Microsoft Sculpt Comfort Mouse uses the commonly available AA size batteries and has a label in one of the battery compartments that states AA/LR6 cells and nominal electrical power of 3V 125mA DC.

Many consumers who buy products that accept AA cells expect to be able to use rechargeable batteries interchangeably.

For some, this is merely a matter of convenience or cost, but for an increasing number of consumers, disposable single-use batteries are anathema on environmental grounds, regardless of how frequently, or not they may need replacing.

Improved chemistries have overcome the earlier problems of limited capacity and self-discharge such that Panasonic now produce a Ni-MH AA (HR6) 1900 mAh cell that retains 70% of its charge after 10 years and can be recharged up to 2100 times - an appealing alternative to disposable cells, I'm sure you'd agree.

Despite the improvements mentioned above, rechargeable AA cells are rated for a nominal Voltage of 1.2 Volts and non-rechargeable cells are rated at 1.5 Volts. This can cause a problem if the device has not been designed to take account of this difference.

The Voltage from cells varies depending on many factors, particularly state of charge, and all cells begin with an output above their nominal rating, which then drops off in use. However, rechargeable cells will reach a steady Voltage state below that of a non-rechargeable cell and this is where the problem can lie. Most devices will work with fully charged rechargeable cells for a period, but may become problematic as the Voltage levels out around 1.2 Volts.

Has Microsoft designed the Sculpt Comfort Mouse to work reliably with 1.2 Volt cells or must disposable 1.5 Volt cells be used?
