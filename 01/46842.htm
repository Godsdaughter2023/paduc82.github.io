Actually you can just put them in the same folder in Windows 8.1 that they were in inside Windows 7 (they didn't move the location for these types of things).

C:\Users\your personal folder\AppData\Roaming\Microsoft\Sticky Notes

Put your .SNT files in that folder and the next time you run stickynot.exe it will read from and open them.

sharief k say:

Hi,

 

Installed memory is the amount of memory that you have in your computer. Usable memory is the amount of memory that's available to Windows.

 

In some cases, the amount of memory that's available to Windows is less than the amount of installed memory because your computer hardware reserves some memory for its own use, which makes it unavailable to Windows.

 

For more information, refer to the link.

 

http://support.microsoft.com/kb/978610

 

Note: Applies to Windows 7, however holds good for Windows 8.1

 

Hope this information helps. If you have any concerns, please let us know.
