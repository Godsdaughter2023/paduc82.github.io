¡firedog say:

The accelerator keys given by Ken only apply to WLMail 2011 and later.

I don't know why the configuration file isn't being delivered to your XP machines. Perhaps Ken could ask his product specialist.

Meanwhile, you can download the proofing tools for the language you want using the URLs given in the 2011 file. Use the lower versions. For example, the German tools for WLMail 2009 are at
http://download.microsoft.com/download/2/6/1/261207BB-401C-4C7A-9065-635646CF297E/spl_dede7.cab 

(dede7 indicates German/Germany, version 7)


Extract the three files to a temporary folder.

With WLMail closed, the three files should be placed in a folder called

%AppData%\Local Settings\Microsoft\Windows Live Mail\Proof\prf0007\7

where 0007 indicates the last two digits of the locale ID (LCID) for Germany in hex. Germany (1031) is 0407, Spain (1034) is 040A and so on, so the proof folder for Spanish would be prf000A. The last folder in the path is the version number.

Launch WLMail and see whether the languages are now available for installation on the Spelling tab.

[Incidentally, these are the same tools as used in Office versions before 2007, so other languages not included in oeconfig.xml can also be added if the Office proofing tools are available. If your users have custom dictionaries in Office, these can also be ported over to WLMail, but that's another story...]

andre da costa say:

You will have to do a clean installation, there is no in place upgrade path from Windows 7 32 Bit to 64 bit.

Using the Windows 7 64 bit disk, install the Windows 7 Windows Easy Transfer Utility from the DVD (To find it, just navigate to x:\support\migwiz, 'x:' represents the name of your disk drive. In this folder, you will see a collection of files, the important one you need to look for and double click is 'migsetup'.) , run it and create a WET file (Windows Easy Transfer file), if you have an external hard disk, you can have it stored there so you can access it later when you do a clean install of Windows 7 64 bit disk.

Once you have done that, simply do a clean install of Windows 7 64 bit disk. In Windows 7 64 bit disk, launch Windows Easy Transfer, follow the on screen wizard, point to where you stored the the WET file (external hard disk), and it will restore your personal files and settings. Reinstall any necessary applications.

As for the product disk, it is the manufacturers responsibility to provide installation media for Windows 7 installations that came preloaded with the OS. 

Here is a work around:

Download the ISO Image of Windows 7
Windows 7 Professional x64 (64-bit)

English: http://msft-dnl.digitalrivercontent.net/msvista/pub/X15-65805/X15-65805.iso

Create a Bootable Windows 7 DVD using the .ISO file
Burning a Windows 7 ISO File on a DVD
