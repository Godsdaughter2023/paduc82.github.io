Remote desktop from Win 8 RT to Win 7 - Microsoft Community

jhinoc say:

Remote desktop from Win 8 RT to Win 7

It's been trying so far to find documentation or proper help on how to set up a remote desktop connection, so hopefully someone can provide some basic assistance. 

 

I have a MS Surface RT, and I would like to be able to connect to my Win 7 Pro laptop remotely.  The Win 7 Pro laptop may be at home or at work (behind a firewall of course), and I may use my Surface from anywhere.  I downloaded Remote Desktop app, but there is scant information on how to connect to my laptop.  It asks for a PC name, but beyond that I am at a loss.

 

Any help would be much appreciated.

 

Thanks, John

hairyfool say:

The 1st thing you need to do is make sure you can run RDP locally.

 

On the Win7 PC rt click My Computer Select Properties then Remote settings, tick Allow Remote.... then choose Allow Connections

 

In the Remote Desktop App Start a session to the PC either by name or IP and it should work.

 

On your router there should be an option to ensure the laptop always gets the same address by reserving it against the MAC address of the laptop.

 

Next on the router there will be a section on Port Forwarding. You need to set up the router to allow inbound connections on port 3389 and forward them to the laptop IP you have reserved above.

 

If you have a static IP then the next bit is simple. Also on the router will be the address assigned to its external (WAN) port. Either this will be a fixed value set up on the router or if still set to DHCP should always be the same value. Setting up another RDP session but to the external IP address of the router should then connect through to the laptop.

 

If however you have dynamic addressing from your ISP (which is more common) then this external address will change. This means that when you are outside the network you do not have a way to find the address of the router. To get round this you need to sign up to a Dynamic DNS service, this gives you a web style name for the external port of your router which is updated with the actual IP address each time it changes. Some routers have the ability to run the DNS update itself so look for an option in its menus. Ther are several free Dynamic DNS Services available.
