Clear c: drive from being so full - Microsoft Community

mike dub say:

Clear c: drive from being so full

How do i MAKE IT SO MY DRIVE IS NOT SO FULL?? I already did clean up and erased restore points??
gokul t say:

Hi Mike Dub,

 

a. What’s the size of the C drive?

b. What’s the edition of Windows 7 are you using? Is it 32-bit or 64-bit edition?

c. What exactly do you mean by “MAKE IT SO MY DRIVE IS NOT SO FULL”? How full is the drive?

 

If you have installed various applications and save other files on C drive that would take up space and hence you may run out of space.

 

The question is rather tricky.

 

1. Run Disk Cleanup. This tool removes temporary files, empties the Recycle Bin, and removes a variety of system files and other items that you no longer need. For more information, see Delete files using Disk Cleanup.

 

2. Uninstall old programs. In addition to deleting old files you no longer want, you can do the same for programs that you no longer use. For more information, see Uninstall or change a program.

 

3. Delete all but the most recent restore point. System Restore uses restore points to return your system files to the state they were in at an earlier point in time. If your computer is running normally, you can save disk space by deleting the earlier restore points. For more information, see Delete files using Disk Cleanup.

 

4. Archive old files to CD or DVD. If you have old photos that you want to keep but don't necessarily want on your computer, consider archiving them to removable media, such as a DVD or CD. That way, you’ll be able to view them whenever you'd like, but they won't take up space on your computer. For more information, see Burn a CD or DVD.

 

5. Add another hard disk drive. Adding an external hard disk drive is an easy way to create plenty of extra space to store digital photos, videos, music, and other files and to clear some space on your primary hard disk. To install an external hard disk drive, all you have to do is plug it in to your computer and connect the power cord. Most of these hard disk drives plug into a USB port, but some plug into a Firewire port (also known as IEEE 1394). For additional instructions, check the information that came with your external hard disk drive.

 

Hope this information helps.

A lot of times when it's "on" but not working, it's because there's another program that disabled it.  This is typically something like Citrix GoToMeeting or GoToMyPC, Cisco WebEx, Citrix Receiver, Skype, Google Hangouts, TeamViewer, or some other kind of remote desktop/support/presentation software.  It disables viewing while dragging to optimize for faster video bandwidth, but they notoriously don't uncheck the box, so you get in this transient state where the system has it disabled at the request of the programs, but the checkbox is still reflecting that it's enabled.

First off it now collects "Identifiable" data. That is data that can identify YOU or your usage habits (sites you visit programs you use etc...). All of this data is transmitted to 3rd party companies (Companies other than MSFT). Its up to you to decide if you trust that data will never be used in a manner you dont like. Some of us have went through great lengths using firewalls and extra software to stop or limit this and we still cant stop it all.

In 10 you have the added problem that windows now forces specific drivers from Windows Update.     susankuban say:

    Want file names to function as captions

    Hello, I am preparing to create a DVD from photos and music from an audio CD.  I've never used Windows Media Center before, but plan to use it for this project.  Before I get too far, I'd like to know how to have my file names show on each photo - as in a caption.  And I do Not want the file extension to show.  Can anyone tell me how to save the photos properly for this purpose?  Thank you, Susan
    volunteer j say:

    The freeware FastStone Image Viewer can add
    filenames to the face of your photos in a batch.

    (FWIW...it's always a good idea to create a System
    Restore point before installing software or updates)

    FastStone Image Viewer
    http://www.faststone.org/FSViewerDetail.htm
    (Windows XP / Vista / 7) (on the download screen...
    try...  exe (site 2) for the easiest download)

    I suggest that you create a new folder and add
    copies of your photos for experimental purposes.
    If you are unhappy with the result your originals
    will be untouched.

    After FastStone is downloaded and installed...
    open the program and go to...

    Tools / Open Batch Convert / Batch Convert tab...

    Check the box...Use advanced options...

    Advanced Options button / Text tab /
    Check the box...Add Text...

    (You'll have to experiment with position and font
    size and color to achieve the result you desire.)
    (text size will have to be adjusted depending on
    the size of the photos)

    Open the drop window..."Insert a Variable"
    choose...File Name / Without Extension...
    (in the white field you should  see    ($C1)
    Left click...OK

    On the Batch Convert tab...in the left field...
    Browse to and select the photos you wish to
    append.

    Left click the Add button to move the files to
    the right field.

    Choose an Output Format...

    Choose an Output folder...

    Click the Convert button...

    This is much easier to do than it is to explain so
    give it a try before you say "Good Grief...that's too
    much work".

    BTW...when your photos are prepared...try importing
    them into Windows DVD Maker to create your video
    slide  show.

    Windows Vista - Windows DVD Maker - Burn A Video Disc
    http://windows.microsoft.com/en-US/windows-vista/Burn-a-DVD-Video-disc
