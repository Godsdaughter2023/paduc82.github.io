Brother HL-5040 laser printer won't print in Windows 7 64-bit - Microsoft Community

tbneff say:

Brother HL-5040 laser printer won't print in Windows 7 64-bit

I had this problem and Brother tech support solved it. Sharing the answer here.

First, the printer is compatible with Windows 7 64-bit, i.e. this is not correct:

http://www.microsoft.com/windows/compatibility/windows-7/en-us/Search.aspx?l=en-us&type=Hardware&s=brother%20HL-5040

viz:

http://www.brother-usa.com/lp/windows7compatibility.aspx

This is an older (but great) printer whose driver isn't preloaded on Windows 7. But it's available through Windows Update (won't walk through that here, but it's straightforward. Instructions here:

http://www.brother-usa.com/lp/windows7compatibility.aspx#WinUpdate

Note that I experienced a couple of wrinkles in the process, but it generally worked as directed. So now you have your driver.

But the test page still didn't print! I ended up on the phone with Brother support. The problem is that when you download the driver from Windows Update, it defaults the printer port to LPT1. This is a parallel port which few modern laptops, if any, have. You need to switch it to the USB port

So: go to Start --> Devices and Printers

Find the Brother HL-5040, right click, and select Printer Properties in the drop-down that magically appears.

Click on the Ports tab.

You'll see that LPT1 is checked. Scroll down and check one that says USB002 (at least in my case it was 002; anything with USB00x should work). Hit Apply.

At that point my test page printed. Good luck.

Has one of the underlying files gone corrupt? I ran a ChkDsk and it fixed 2 or 3 files, but not this problem. Should I go back to a restore point? umesh p say:

Hi rgreenenf,     

a. Did you make any changes to your computer recently?
b. Have you installed any wireless network adapter software's on the computer? If yes, then uninstall it and then check if the issue persists.

You may try the below step and check if this helps.
 
Also please verify with the WLAN service is started  and set to automatic, if the WLAN service is stopped, I would suggest you to start the service and put it to automatic.

Please follow the steps to start the WLAN service.

a)      Click on Start, type in services.msc
b)      Select WLAN Autoconfig from the list, Right click properties
c)      Change the startup type to Automatic
d)      Click on Start button
e)      Click on Apply, Ok.
f)       Restart the computer and verify if it works.

Also check if all the dependencies are also running. Try the steps to check Dependencies.

1. Right-click on WLAN Autoconfig and select Properties.
2. Click on Dependencies, you will find all the dependency services under box labeled This service depends on the following system components.

If the issue persists, then I would suggest you to download the latest network adaptor drivers from the manufacturer’s website and install the same on the computer and check whether it helps you to fix the issue.

For more information visit http://windows.microsoft.com/en-us/windows7/Update-a-driver-for-hardware-that-isnt-working-properly

Hope this information is helpful.
