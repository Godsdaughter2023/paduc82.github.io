Salut,

 

Ce topic a pour seul but de sécuriser votre compte Xbox Live et de vous informez sur les méthodes qui pourraient être utilisées pour voler votre compte. Ce thread est un bref résumé de cette page du support Xbox, disponible en Français. Certains extraits de cette page seront cités dans cette article.

 

Pour commencer, voici les différents types de vols de compte:

 

  • Partage de comptes
    • Il s'agit de la forme la plus répandue. Lorsque vous partagez vos identifiants, une tierce personne peut avoir accès à votre compte.
  • Importez votre profil sur une autre console
    • Lorsque vous téléchargez votre profil sur la console d'un ami, celui ci peut avoir accès à votre profil.
  • Programme malveillant sur votre PC
    • Il s'agit de tout type de logiciels non sollicités. Une fois installés, ceux ci peuvent voler vos identifiants
  • Offres de niveaux/de succès
    • Il s'agit de personnes peu scrupuleuse qui vous promettent de vous faire monter de niveau ou de débloquer certains succès. Mais cela implique de donner vos identifiants.
  • Ingénierie social
    • Sous ce terme barbare se cache une technique qui, au détour d'une conversation banale, vous incitera à dévoiler vos identifiants
  • Failles de sécurité tierces
    • Il s'agit d'utiliser les mêmes identifiants sur plusieurs sites Web.
  • Phishing
    • Il s'agit de messages ou sites vous incitant à révéler vos identifiants

 

[quote user="Support"]

Autres méthodes de phishing :
 
  • Courriers électroniques qui semblent provenir d’un collègue ou d’un ami avec des liens vers un site Web ou demandant des informations sur vous ou votre compte.
  • Un message instantané qui semble venir d’un membre de votre liste d’amis avec un lien vers un site Web.
  • Un courrier électronique qui semble provenir de Xbox ou Microsoft.
  • Des messages en cours de partie affirmant venir du support Xbox.
  • Un faux site Web, prétendant être un site Web Xbox LIVE, avec une page de connexion Windows Live ID.
  • Sites Web offrant de booster votre score de joueur.

  • Partage de comptes
    • La méthode la plus simple est de définir un code d'accès Xbox Live. Vous pouvez apprendre sa mise en oeuvre en allant sur cette page.
  • Importez votre profil sur une autre console
    • Vous pouvez protèger celui ci avec un mot de passe. Rendez vous sur cette page pour sa configuration.
  • Programme malveillant sur votre PC
