sirangry say:

It's an interesting question, but a conundrum that I feel isn't going to be as big an issue as maybe some might fear. Firstly the Blu-ray drive speed will be fast enough to meet the BRDA's standards for movie playback, and that is all it'll be used for in the Xbox One any way. Games are still being installed onto your HDD.

.

So lets talk about that HDD. As far as I could tell, from the moment Microsoft said the HDD wasn't removable I said to myself, and others on here, that there had to be a reason. It's such an easy thing to implement that I couldn't quite wrap my head around not doing it. Unless the HDD can with a rather large cache of flash memory to bring read write speeds up to those of say a SSD. Guess what? The Xbox One HDD comes with supposedly 8GB's of flash memory. Now, that could be primarily for OS storage, to make all that switching snappy, but my guess is it is also where the game data will be loaded into for use.

.

Now not all SSD's and flash memory are born equal, but a data transfer rate of 6Gbits/s to 7GBits/s isn't out of the question here. That compares to a 12x Blu-ray drives data transfer of about 55MBits/s I believe. As you can see, flash memory is way quicker. Now a 7200rpm HDD has a data transfer rate of 1GBits/s, so again it's no slouch in terms of data transfer, but it isn't as rapid as flash memory. Obviously there is some latency involved in terms of seek times for all systems used, but these can vary greatly between devices and or how the data is packaged on them. Suffice to say, if the flash memory really is being used to store data for use then you won't need to worry about HDD rpms and Blu-ray drive speeds.

    davem121 say:
    Hi mxmxm, you can reinstall Windows 10 at any time and it will not cost you anything !

    Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

    Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

    Click this link:
    https://www.microsoft.com/en-us/software-downlo...
    to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

    Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

    If you have problems booting from a boot disc, you may have UEFI BIOS:
    Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
    1. Disable Secure Boot
    2. Enable Legacy Boot
    3. If Available enable CSM
    4. If Required enable USB Boot
    5. Move the device with the bootable disc to the top of the boot order
    6. Save BIOS changes, restart your System and it should boot from the Installation Media
