Don't assume . . .

There are many ways to format a partition, even when Windows is no longer working. Here are a few of them:
  • Boot the machine into Windows Repair Mode, either by pressing F8 at boot time or by booting it with your Windows Repair CD, then select a Command Prompt. Now use format.exe to format the System drive (which could be drive C:, D: or E:).
  • Remove the hard disk, then connect it to some other machine and do your formatting. Depending on what machines you have, you may need to install the disk in an external USB case.
  • Borrow a friend's Windows 7 installation DVD, then boot the machine with this disk. When prompted, allow the disk to be repartitioned and/or formatted. A WinXP CD might do too, depending on the type of hard disk you have (IDE/SATA).
  • Boot the machine with an Ubuntu boot CD (http://www.howtogeek.com/howto/windows-vista/use-ubuntu-live-cd-to-backup-files-from-your-dead-windows-computer/), then format it under Ubuntu.
  • Obtain one of the many partition management tools such as Acronis Disk Director. Some are free. All have a facility to partition/format hard disks.
  • Boot the machine with your Windows Repair CD, open a Command Prompt, then move into the System32 folder on the hard disk. It is usually on drives C: or D: but never on drive X:. Now use the "clean" command in diskpart.exe to format your disk. Note - I haven't tried this particular option.

Using Windows Vista Media Center to view photos.

 

You can view photos and videos stored on your computer using Windows Media Center. You simply need to browse to those folders, and select the picture you want to view. If a picture you would like to view is not listed it is most likely stored in a non-typical location.

 

The default locations for pictures and video are below:

·          C:\Users\Public\Music

·          C:\Users\Public\Pictures

·          C:\Users\Public\Video

·          C:\Users\<Username>\Music

·          C:\Users\<Username>\Pictures

·          C:\Users\<Username>\Video

How to add pictures, music, and videos to Media Center.

 

1.     On the Start screen, roll down to Tasks, and click on Settings.

2.     Click on Library Setup, and select Add folder to watch.

3.     Select the location you want to add from:

·          Add folders on this computer

·          Add shared folders from another computer

·          Add folders from both locations

4.     Locate the folder(s) you want to include, select them, and click Next.

5.     Click Finish, and then click OK when prompted.

6.     You can continue to use Media Center while your Library is updated.

7.     Once complete the newly added media should be available through the Picture Library.

 

If adding the folder to the watch list fails to resolve your issue please refer to this Windows Help and How-to article. Please also confirm that this behavior is occurring with all pictures or restricted to particular ones.

 

Also, the file type JPG is supported so that should not be an issue, but for troubleshooting purposes you will want to try viewing different file types in Windows Media Center. For a list of supported file types please refer to this document.
