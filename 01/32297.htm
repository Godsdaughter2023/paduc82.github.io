Verticle scrolling jumps after installing Microsoft Keyboard and Mouse Center v3.2.116

I have two different wireless Microsoft-branded mice I'm testing against Microsoft Mouse and Keyboard Center v3.2.116.  Without the MS Mouse & Keyboard Center installed, these mice work just fine.  Some users, however, need the extra features from this software.  The problem is that with this software (and drivers) installed, vertical scrolling jumps several screens at a time with the mouse wheel instead of 1-3 lines as typical.  This only occurs immediately following a reboot but BEFORE Microsoft Mouse and Keyboard Center is launched.  Once the software is launched, vertical scrolling with the mouse instantly returns to normal function.

This seems like a driver bug of some sort.  I tested this futher by leaving the Mouse and Keyboard Center program installed BUT I changed the mouse driver back to the generic HID driver.  I could not reproduce the scrolling issue until after I put the Mouse and Keyboard Center driver back for the mouse.

martin brule jr. say:

The latest article suggested above wasn't any help but I did figure out a workaround.  There are two Mouse and Keyboard Center processes that need to be running at and after user login.  itype.exe and ipoint.exe, both of which are in the main install folder for Mouse and Keyboard Center, need to be running to avoid abnormal mouse behavior.

I simply created shortcuts to these two executable files and put those shortcuts into the all users startup folder, typically found at:

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

That seems to have taken care of this issue for me!

Hi N.prasanna,

 

Thank you for posting on the forums.

Windows Vista Compatibility Center

http://www.microsoft.com/windows/compatibility/Default.aspx?type=Software

 

Your software is not showing being compatible with Vista.

 

You can try extracting the file from an XP cd if you still have your old one and I could give you the steps on how to do it but, you really need to contact your program’s manufacturer for additional support for their product ok.

 

Here’s are the steps on how to

Extract the Msvcp50.dll file to the computer

Extract a new copy of the Msvcp50.dll file from the Windows XP disk to the %systemroot%\system32 folder on the computer's hard disk. To do this, follow these steps:

1.       Insert the Windows XP disk in the computer's CD or DVD drive. Press and hold down the SHIFT key as you insert the disk to prevent Windows XP installation from starting automatically.

2.       Click Start, click Run, type cmd in the Open box, and then click OK.

3.       At the command prompt, type expand CD drive:\i386\msvcp50.dl_ %systemroot%\system32\msvcp50.dll, and then press ENTER.

Notes

o    Make sure that there is a character space between "msvcp50.dl_" and "%systemroot%."

o    CD drive represents the drive letter of the computer's CD or DVD drive that contains the Windows XP disk. Additionally, %systemroot% represents the path and folder in which the Windows system files are located. For example, if the letter of the computer's CD drive is E, and if the folder that contains the Windows system files is C:\Windows, type the following at the command prompt, and then press ENTER:

expand E:\i386\msvcp50.dl_ C:\Windows\system32\msvcp50.dll

4.       Type exit, and then press ENTER to close the command prompt.

5.       Restart the computer.

 

Hope that helps, please post back and let us know.
