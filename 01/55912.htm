Though the same steps holds good for Windows 7, here is how you can disable Key sequence in Windows 7:

ken blake say:

On Mon, 16 Aug 2010 10:13:16 +0000, Strugeonmaster7 wrote:

Wondering if I can use the same Windows xp install disk I have for another computer to install windows xp to a differant one.

Your question isn't clear. Do you want to have XP installed on two
computers at once?

If so, no, you may not do that. The rule is quite clear. It's one copy
(or one license) for each computer. There's nothing new here. This is
exactly the same rule that's been in effect on every version of
Windows starting at least with Windows 3.1. The only thing new,
starting with XP, is that there's now an enforcement mechanism.

Or do you mean that you want to move your installation from the
current computer to a different one?

If so, the answer is it depends. If you have a retail copy, yes, you
may do that as often as you need to or want to. But if yours is an OEM
version (for example if it came preinstalled on your computer), no you
may not. The license for an OEM version ties it permanently to the
first computer it's installed on, and it may never be moved to
another.

Ken Blake

smlunatick say:

You can not use the very same CD key on two computers, at the same time.  Nor can you "officially" transfer a previously installed OEM CD key to a different computer, as per the OEM EULA details.

If both computers have their own CD key, has the same style of Windows (Home or Pro) and are both the same distribution (OEM or "Microsoft,") then it would seems that YES you can use the same install CD.

Please note:  You can only do the "above" if the disk is a true XP install disk and not a "recovery" disk.
