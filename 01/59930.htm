cmdrkeene say:
It's inside the XP Mode VHD (Virtual Hard Drive) file, which you can think of as pretty much like a ZIP file that contains an entire drive structure instead of just a few files.  This VHD file is usually stored under your user folder in a Virtual Machines folder, for example mine was in C:\Users\SKeene\Virtual Machines.

And it is my guilty nerdy pleasure to inform you that you can indeed browse this file like it was a hard drive, without booting up the XP Mode.  However, security will apply: so if your "My Docs" was restricted, it might be difficult to access (although you may be able to Take Ownership of the folder and gain access).

Here's how:
  1. Open the Start Menu, right-click Computer, and select Manage. In the Computer Management window that opens, choose "Disk Management".

  2. On the right side, click "More Actions" then "Attach VHD" and select your XP Mode's VHD file.  You might consider making a backup copy of the VHD file before you do this, just in case!

  3. After you've attached the VHD, an Auto-Run dialog may appear.  If not, just click Start/Computer to see your new "Local Disk".
I can at least confirm that this worked flawlessly for me, and I gained access to my "XPMode User" MyDocs folder without issue.
