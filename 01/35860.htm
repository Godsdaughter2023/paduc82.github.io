Thank you for posting on Microsoft Community.

For this type of concern, seems that there has to be a broken components with your Windows' Boot Configuration Data or BCD. It has the stored information of your computer's peripherals when booting up.

You can use the Startup Repair option in the Windows Recovery Environment (WinRE) to repair the BCD store. To do this, follow these steps:

  1.  Put the Windows 7 installation disc in the disc drive, and then start the computer.
  2.  Press any keys when you are prompted. 
  3.  Select a language, a time, a currency, and a keyboard or another input method, and then click Next.
  4.  On a windows which has an Install message, click Repair your computer at the bottom left corner.
  5.  Click the operating system that you want to repair, and then click Next.
  6.  In the System Recovery Options dialog box, click Command Prompt.

Once you have the Command Prompt up, you can type the following commands one by one and enter each line:

  1. bootrec /fixmbr
  2. bootrec /fixboot
  3. bootrec /rebuildbcd

On the third command, it will prompt you to continue by typing Y or Yes and pressing the Enter key.

The rebuild process should be completed. Close the Command Prompt and restart the computer.

This should resolve your concern with being stuck on the Windows screen.

rifdhan say:

To all who require this:

Here is a list of possible solutions to your problem(s). Follow them in order.

 

Step 1: Ensure Hardware Acceleration Is Set To Full

1. Click the Start button and search for 'screen resolution'. Open the link titled "adjust screen resolution".

2. Click "Advanced" at the bottom-left.

3. Go to the "Troubleshoot" tab. If there is an option to do so, click "Change Settings". After that move the "Hardware Acceleration" slider to the maximum. if this option is not available, please skip ahead to step 2 below.

4. Click "OK". If it prompts to restart, restart your PC.

Now try checking the DirexctX Diagnostics (search 'dxdiag' in the Start menu). If you can enable all features (Direct3D, DirectDraw, etc) then your problem is solved. If not, keep reading.

 

Step 2: Install latest DirectX Runtimes

1. Go to the following Microsoft website:

http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=35

2. Download the DirectX web setup wizard.

3. After downloading, run the installer, and follow its instructions to install the latest DirectX runtimes on your PC.

4. When it has completed, restart your computer.

Now try checking the DirexctX Diagnostics (search 'dxdiag' in the Start menu) again. If you can enable all features (Direct3D, DirectDraw, etc) then your problem is solved. If not, try repeating step 1 again. If the issue still persists, keep reading.

 

If you have followed both steps above and you still cannot enable DirectX features, please say so and I will attempt to find further solutions. Also, if you can, please state your motherboard and chipset's manufacturer. This issue seems to persist specifically on VIA chipsets and motherboards, and I want to check if that is the case with you.

 

Hope that helps,

Rifdhan
