IMPORTANT: Go to the section: Reset the computer to start normally after clean boot troubleshooting to boot the PC in normal startup after identifying the cause of the issue.

Install the latest updates for Windows 10

Running Windows Update will have your machine receive the latest fixes and security updates, helping your device run efficiently and stay protected. Refer to Windows 10 update history for the latest updates for Windows 10.

Download the latest installer from the software developer website

It is possible that your current installer package is corrupted. You can download a fresh one from the software developer's website.

Run the DISM command-line tool and the SFC tool

Both tools are designed to scan for corruptions in Windows system files and restore corrupted files. You will need administrative rights to perform this task.

  1. Click the Start button and type Command Prompt.
  2. Right-click Command Prompt and select Run as administrator.
  3. If prompted, choose Yes or enter the administrator password.
  4. Enter DISM.exe /Online /Cleanup-image /Restorehealth and then press Enter.
    • Note the space before each "/".
    • This procedure may take a few minutes to start and up to 30 minutes to complete.
  5. Enter sfc /scannow and then hit Enter.
    • Note the space between "sfc" and "/".
  6. Restart your PC and reinstall the program.
