Windows Media Player is not installed properly - A Possible Fix - Microsoft Community

p.bear say:

Windows Media Player is not installed properly - A Possible Fix

Error Message: "Windows Media Player is not installed properly and must be reinstalled"

Questions (and answers) about this error message are posted all over the web -- it seems to be a very common problem with Windows Media Player -- and many have posted fixes involving uninstalling and reinstalling WMP, as well as unregistering and reregistering it.  None of these fixes seem to have worked for several people receiving this error message when trying to start Windows Media Player on their computers.  The problem seems to affect only the 32-bit version of WMP, as many have found out by manually starting the 64-bit version in the "Program Files" folder (it is the 32-bit version, in "Program Files (x86)", that is still used for the default shortcut in Windows' Start Menu).

None of the posted fixes I found worked for me.  I stumbled upon the fix that did work for me, almost by accident.  I happened to notice, in my "Program Files (x86" folder, underneath the subfolder for "Windows Media Player" was another one for "Windows Media Player Plus!".  I'd forgotten that I had installed Windows Media Player Plus! years ago, when it was sold as an add-on for Windows Media Player that could be purchased directly from Microsoft (on CD).  It had been copied over onto every upgrade I'd made to Windows since, probably, Windows XP -- and it occurred to me that it was, now, terribly old.  I uninstalled Windows Media Player Plus! and the 32-bit version of Windows Media Player immediately opened again without any problem.

So, if anyone else happens to have Windows Media Player Plus! installed on your version of Windows, uninstall it!  It might fix your problem.

By the way, Microsoft no longer sells it but its original developer (BF-Productions, apparently in the Netherlands) still offers it as a free, downloadable add-on to Windows Media Player on their web site (it is also available from all the popular software download sites, like Softpedia and Major Geeks).  I installed the current version, v2.10, on my Windows 10 Pro machine and it works, without problem, exactly like it used to.  The problem was that I still had an old version of WMP Plus! (2.08) installed that is no longer compatible with current installations of Windows Media Player.

Note:  It appears that Windows Media Player Plus! installs as an add-on only for the 32-bit version of Windows Media Player.  On my machine, it is not operational in, or accessible from, the 64-bit version of Windows Media Player at all.
