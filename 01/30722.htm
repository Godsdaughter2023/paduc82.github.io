On another computer with a recent Windows version:

  1. Create a directory directly from the root and give it a convenient name.
    E.g., c:\ztemp
  2. Navigate to ....\windows\fonts. Change to Details view of the folder. Locate the font name on the list. Right mouse click and choose Copy from the context menu.
  3. Paste to c:\ztemp    by whatever name called.
  4. Repeat for all desired fonts.
  5. Copy those fonts to a USB flashdrive or burn to a CD or DVD or transfer over a network or..... to a temporary directory on the target computer.
  6. In the usual way, install them on the target computer.

Note that if the fonts are already installed and somehow corrupted, this operation might fail.

Sorry for the inconvenience caused.

Have you made any changes to the computer prior to the issue? If so, please let us know.

Try uninstalling the drivers and reinstall them in compatible mode and check.

1. Right-click on the driver setup file and click on ‘properties’.         

2. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 7/8 operating system from the drop down menu and proceed with the installation.

3. Once this is done, restart the computer and check if the issue persists.

 

Also check for any Windows pending updates. Restart PC if necessary.

If the issue persist, perform a clean boot and check.

http://social.technet.microsoft.com/wiki/contents/articles/29876.how-to-perform-a-clean-boot-in-windows-10.aspx

Reset the computer back to Normal Mode once you are done with the troubleshooting by following the section "How to reset the computer to start normally after clean boot troubleshooting" from the same article.

 

I hope this post is helpful. Get back to us with the results. We are happy to help.
