How can I automatically send a screen shot to my printer when the - Microsoft Community

aduncan2 say:

How can I automatically send a screen shot to my printer when the print screen key is hit? Is there a way to do this without downloading an app?

I had it set up on another computer a long time ago but don't remember how it was done.

Thanks for your help!

tryx3 say:

The PrntScrn key is badly-named.  PrntScrn does not print the screen.  It sends a screenshot to the clipboard for you to paste into Paint or other graphics app so you can either save it as a file or print it from Paint [Paint is a Windows 7 component so you have it already].

PrntScrn copies the entire screen

Alt-PrntScrn copies the active window

Ctrl-Fn-PrntScrn [on my PC] copies the whole screen including any menus that are onscreen at the time [so you can capture, for example, right-click context menus].  This key combination is not documented and you will need to experiment to find out if it works on your PC.
