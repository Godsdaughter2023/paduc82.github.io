techfreak_ say:

Jumping back requires the functionality to be added back in and the history to exist. I am going to be focusing more on the history half of your inquiry.

If you are trying to view history older than 2 years, you would need to have a copy of your main.db file from a classic version of Skype.

Since you were using older classic versions of Skype (7.x), your older history would have been stored "forever" in a main.db file located on your hard drive. It would be contained in your Skype name folder in %appdata%\Skype. You can also simply search your system for "main.db" files. If the file is still present and not damaged, you can export that history and view it in your web browser. This export tool is in the latest 8.x desktop versions of Skype. 

Another option is using third party software like SkypeLogViewer [nirsoft.net] to view the contents of the main.db chat history. 

That latest versions of Skype no longer use main.db files to update and store data forever locally. The latest versions of Skype will only display up to 2 years of cloud-based chat history. Currently that cutoff day is 2 years from the current date as the feature was enabled over 2 years ago, This makes it June 2017 when it was originally ~April 2017. If you want to export history from now through June 2017, you can use the online export tool and viewer mentioned in this FAQ:

How do I export my Skype files and chat history?

When viewing history using the export tools provided by Skype (in the client/FAQ) or third party tools, you might not have the ability to jump back, but you will have the option to search the entire history. This can allow you to search for content or dates that should jump you back to what you want to see.
