  1. I do the install of win7

  • Switch to advanced and Delete all partitions, only one logical partition – C:, which will be created by the installer.

frederik long say:

Your requirement is a little exotic . . .

Here is a method that will work:

  • Create a scheduled task that will shut down the PC some x seconds after the automatic log on.
  • Set the BIOS so that it starts a boot-up sequence when power is resumed.
  • Use an external controller that cycles power with this pattern: 3 minutes on, 1 minute off.

The sequence will then work like so:

  1. The controller applies power.
  2. The BIOS boots up the PC.
  3. The scheduled task shuts down the PC.
  4. The controller turns off the power.
  5. Back to Step 1.

The external controller could be a PLC (Programmable Logic Controller), if you can afford to buy one. If you have the skills to build an electronic circuit yourself then you can get the same functionality with a few chips and a relay.

You aren't.
  
You may be *an* administrator - but likely the built-in true "administrator" is not one you even have enabled for logon.
  
Press the Windows Logo Key and R at the same time...
Type in the following - as shown (exactly... check spaces, etc.)
  
cmd /k net localgroup administrators
  
Click OK.
  
That should bring up (in Desktop mode) a command prompt window that will show you all the members of the "administrators" group.  
  
Is the user you are logged in as in that list?
  Yes - you are *an* administrator
  No or it errored in some way - you are not *an* administrator.
  
Another way - more GUI (graphical user interface) based desired?

Press the Windows Logo Key and R at the same time...
Type in the following - as shown (exactly... check spaces, etc.)
  
control userpasswords2
  
Click OK.
In the window that comes up, click on the "Advanced" tab.
Under the "Advanced user management" section, click on Advanced".
On the left side of the window that comes up, click on "Groups".
On the right side of the Window (or middle depending on the view) double-click on the word "Administrators".
Look under the "Members" section for a list of users in that group.

Is the user you are logged in as in that list?
  Yes - you are *an* administrator
  No or it errored in some way [preventing you to get that far] - you are not *an* administrator.
  
P.S. - all that was basically the same in Windows XP, Windows Vista, Windows 7, etc.  It was Windows Vista and later that added the UAC stuff that made people start wondering if they were an administrator - because it asks for a second 'authorization' to perform certain tasks.
