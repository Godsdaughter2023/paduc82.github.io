Hello Mason,

This issue can potentially be caused by the following reasons:

- The user profile is corrupted.

- The UsrClass.dat file at the following location is missing: %userprofile%\AppData\Local\Microsoft\Windows.

- Folder redirection is enabled to redirect Application Data folder into a Roaming Profile.

To help resolve the issue, we suggest that you locate the folder: %userprofile%\AppData\Local\Microsoft\Windows, and verify if UsrClass.dat is present. Normally, this is being resolved by creating a new user profile like what you have done. However, if the issue persists, you can try the following workaround:

1. Disable the "User Profile Service".

2. Reboot.

3. Log on with local admin account.

4. Remove the problematic profiles.

5. Remove reference to the specific problematic profiles from registry at:

 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileListLogoff

6. Log on with local admin account.

7. Set "User profile Service" to start automatically.

8. Reboot.

Feel free to post back should you have other concerns.

Adding music to windows movie maker - Microsoft Community

cherimaynard say:

Adding music to windows movie maker

I downloaded some music from limewire pro (I even payed for a 1 yr subscription) then transferred that music to window media player.  However, I can't figure out how to add it to my movie. I click on add music, but can't seem to find the file.  Forgive me, I'm a rookie.  Do you have any suggestions?
considerate_guy say:

Open up Windows Media Player and find the music you want to use. Now right-click on it and choose 'Properties'.  You should see the full path name of the file and where it exists on your computer.  You can highlight that complete name and copy it (Ctrl+C).  Now when you are in Windows Live Movie Maker and adding music, just paste (Ctrl+V) that filename into the filename box.

An alternative, perhaps faster way to do this, is to right-click on the song in Windows Media Player and choose 'Open file location'.  That will open up Windows Explorer with the file selected.  Now just drag that file right into your Windows Live Movie Maker window at the spot where you'd like the music to start.
