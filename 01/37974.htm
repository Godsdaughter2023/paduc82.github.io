danstuffe say: The problem is the WIFI driver that Windows 8 identifies as suitable for the included hardware as part of the installation.  It's sufficiently bad to crash the machine with the IRQL_.. error before the install finishes, so you cannot replace it.

The key is to provide the driver before the install takes place.  Choose a custom install where you get to choose the drive that you will instal windows on, and you will see a button marked "load driver".  Insert a USB stick with the downloaded drivers from the Acer site and direct the browse to this directory.  This should pick up and identify an Atheros WIFI device, and pre-load an updated driver preventing it from choosing the dodgy one during install.

Install will now finish, and you may need to also update your SATA AHCI and Bluetooth after it completes, but it won't stop you getting to the point where you can access Windows Update in order to do it normally.
