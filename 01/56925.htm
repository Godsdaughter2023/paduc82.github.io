cmdrkeene say:

Windows check disk doesn't put any type of security lock on a drive.  Your account must not be associated with an allowed user on the drive Access Control List any longer. Here's how to fix it.

  1. Right-click the Drive, choose Properties
  2. On the Security tab, click Advanced.
  3. At the top, change the owner to yourself (if it's not already).

  4. Close all dialogs (click OK until they're all gone).  This is necessary after changing the owner, or the next steps will fail.

  5. Right-click the drive again, go to Properties once more.
  6. Go to Security tab again, and this time, make sure that your account has "full control".
  7. Access the drive again.

4.    Restart Windows Media Player.

Note Windows Media Player automatically rebuilds the database.

1.    Exit Windows Media Player.

2.   If you are running Windows 7, click Start, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.

3.    Select the Media Player folder, and then click Delete on the File menu.

4.    Restart Windows Media Player.
