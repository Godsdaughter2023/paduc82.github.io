Fujitsu notebook user problem. - Microsoft Community

yixuanbai say:

Fujitsu notebook user problem.

Here is a problem with many Fujitsu laptop users: NVIDIA has stopped updating graphic drivers for all Fujitsu laptops. This means all GeForce Drivers after a certain version will cause system malfunction. For me (GeForce 640M LE), driver version after 347.88 will cause black screen during booting or updating to a newer version.

This problem will occur no matter which version of Windows you are running. This may also mean that CUDA Toolkit after 6.5 will not be supported on Fujitsu laptops any more (not sure).

The reason why this was seldom reported before is because if you are not a CUDA programmer, you would not bother to upgrade you graphic drivers. However, the auto-update mechanism of Windows 10 will force unmatched (namely the newest) driver to be installed., which will cause disastrous problem during upgrading from older versions of Windows to Windows 10 because you can not enter safe mode to disable or uninstall the graphic driver since the operating system installation fails.

For fellows run into this problem after installation (when Windows 10 install the newest NVIDIA driver during system updating), you can enter the safe mode (by repeatedly hot-rebooting your machine to force a booting manual to show up ) and disable GPU using the device manager. After that, you can use an older version of GPU driver to override the newer driver. This will solve the problem temporarily, but due to the auto-update, long-term solution is not guaranteed.

Also, this question may come to help: http://answers.microsoft.com/en-us/windows/forum/windows_10-windows_install/fujitsu-lifebook-freezes-on-initial-setup/50477bc3-c46a-4172-bff1-ef9763ed4840

For Microsoft guys, stop asking Fujitsu laptop users to solve this problem by upgrading the newest driver. Instead, you should provide alternative options so that people can choose whatever upgrade they need, especially when it comes to hardware drivers which Microsoft can't not guarantee a thing.
