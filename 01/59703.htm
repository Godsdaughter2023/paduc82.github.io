video99 say:

Success!

 

After the Windows7 install (update only, so no lost settings), then replacing a few utilities for the ASUS motherboard which were giving errors after the Windows install, external USB drives were working fine.  Then allowed Windows to do all of its updates, and external drives are still working fine.

So that was it, Windows7 was corrupted.
