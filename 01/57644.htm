shane-- say:  

 This issue is occurring because some of the games that you are playing need to “Run as Administrator.”  While the game is running, or is the “Active Window” for Windows Vista, User Access Control( A Windows Vista Security feature) prevents any other programs(in this case, Ventrilo) from accessing your keyboard and mouse.  This causes any keys you have bound within Ventrilo to not function.  The solution for this is to run Ventrilo as an administrator also because Windows Vista will see both programs running as the same user.  This will allow Ventrillo to access the keyboard and mouse while your game is running.  We have provided the steps for this below.

  

Run Ventrilo as an Administrator:

  1. Right click on the Ventrilo shortcut and select Properties
  2. Click the Compatibility tab
  3. Click the box next to Run as Administrator
  4. Click OK
