canadian tech say:

First a bit of background duplicate IP address errors

 

Your Internet service provides a unique IP address to your modem.  The router you have either in the modem or a separate device has that address.  The router hands out individual IP internal addresses to the various devices you allow on your local wireless area network.  That's actually a bit of a firewall.

 

When ever a device that is not on the network logs in with the appropriate password, it is granted an IP address by the router.  That address is given up when that device goes off the network.  These addresses also change over time as they are granted for a maximum period of time.

 

At any one point of time you are likely to have several devices on your network.

 

So, if there is IP address conflict, somehow you have two or more devices on your network that are using the same address.  That can't work!

 

The problem is almost always in the router.

 

First, most likely solution is to shut down the router -- remove power.   Wait a few seconds and then start it up again.  It will, as part of its setup, assign new addresses to each device it finds authorized to be on your network.

 

If that does not help, it is possible that your router is having a problem

Difference between resetting and clean install of Windows 8 - Microsoft Community

theking2 say:

Difference between resetting and clean install of Windows 8

Original Title: resetting or clean install

 

Hi!

 

I understand the difference between refreshing and resetting windows 8. but what is the difference between formatting the volume where Windows 8 is on and reinstalling it and a reset?

patrick barker say: Hi,

The difference between refresh and reset in Windows 8 is fairly simple.

Reset completely reinstalls Windows 8 and saves nothing. Think of it like essentially formatting the drive and doing a clean install.

Refresh does a clean install of Windows 8, however it copies your Metro Apps, personal files, etc, to a temporary location which will be added to the new install afterwards.

If you choose the reset option, you'll be asked whether or not you want to remove your files or clean the drive. You'll in most cases always choose 'remove my files' rather than 'clean the drive' as the second option is intended more for when you are selling the computer to someone else.
