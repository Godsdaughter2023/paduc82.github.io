Hi

1. Set the View to Details.

If the name is not in the list, click the More item to see the complete list in alphabetical order. Place a check mark in the column names you want to appear.

After you have the columns configured, whenever you click a column name, it will reverse the order of the files and folders.

Example: You want the the Date Modified to be ordered oldest to newest and it currently shows newest to oldest. Click the name of that column and the sort order will be reversed and show the newest to oldest.

a. user say:

Libraries were first introduced in Windows 7. They work exactly the same way in Windows 8 and Windows 10.

This was the idea behind libraries: We store our personal files and folders in different places - our local computer, other computers on a network, network attached storage (NAS), etc. Without a library, we have to search for a file or folder in each location separately. By adding all these places to a library, we only have to search the library. The library is not a physical thing, it's a way to organize files as if they were all in the same place.

You can get lots more details with a web search. Here are some search results I found:

Understanding the Libraries Feature in Windows 7


How to use libraries in Windows 7 and 8 to more easily organise your files

How to Use Libraries for Managing Groups of Files & Folders in Windows

The ironic thing about libraries is that after trying hard to persuade everyone to use libraries, Microsoft changed their mind starting with Windows 8 and now tries to persuade everybody store all their files and folders in the cloud ... like, how about in OneDrive?

You see, in the technology industry, the definition of a good idea can change depending on a company's latest marketing goals.
