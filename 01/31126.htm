steve winograd say: Connect the DSL modem to the router's WAN (Internet) port.  Connect the computers to the router's LAN ports.

The DSL modem's DHCP server will assign an IP address to the router's WAN interface, but not to the computers.

The router's DHCP server will assign IP addresses to the computers.

Make sure that the DSL modem uses a different subnet than the router.  For example, if both of them use 192.168.0.x, change one of them to 192.168.1.x.
