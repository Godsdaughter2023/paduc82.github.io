uselysses say:

Hello,

I just found a workaround.

If you open regedit and navigate to

Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\User\PowerSchemes

you will be able to find out the GUID of the Ryzen balanced power plan. Just click through the individual sub folders until you found the one that mentions Ryzen on the right side and copy from the address bar. After that, launch the text editor and type in

@echo off

powercfg -s 

Paste what you got from the registry editor into the second line with a space behind the -s. Save that as "Ryzen balanced.bat" or something similar, as long as it is a .bat file.

Press the Windows key + R and enter shell:startup into the line. An explorer window will open up. Drag and drop that file you just created into there.

Now Windows will run a file that switches to Ryzen balanced every time you start it. Hope this helps.

I got most of the relevant info from this link: https://stackoverflow.com/questions/30807801/create-a-batch-file-to-change-the-power-options-in-windows-7

    lemp say:

    Generally, compatibility questions should be directed to the publisher of non-Microsoft software that runs on Windows rather than to Microsoft.

    Also generally, if such software is compatible with a version of Windows (e.g., Windows 7), it should also be compatible with any service pack updates for that version of Windows.

    ZoomText 9.1 is an older version of ZoomText.  At least versions 10 and 11 were subsequently released.  The current product is ZoomText 2018.  I'm surprised you can find ZoomText 9.1 for sale. 

    According to its User Guide (https://zoomtext.zendesk.com/hc/en-us/articles/115005416783-ZoomText-9-1-Product-Guides), ZoomText 9.1 requires Windows Vista, Windows XP with Service Pack 2 or later, or Windows 2000 with Service Pack 4 or later.  There is no indication that it will run on Windows 7 with or without Service Pack 1, but if a program runs on Vista it will very likely run on Windows 7 SP1.

    Page 16 of the User Guide for ZoomText 2018 says (http://www.aisquared.com/docs/zt2018/ZoomText_2018_User_Guide_English_US.pdf):

    The following hardware and software are required to run ZoomText 2018:

    • Windows 10, 8.1 or 7 with Service Pack 1 (or later).
    • Processor: 2 GHz i3 dual core processor or similar. i5 or i7 recommended.
    • 4 GB Ram. 8 GB recommended.
    • Video card with support for DirectX 10 or higher.
    • Recommended: 256 MB dedicated video memory.
    • 400 MB hard drive space. SSD drive recommended.
    • 5-point multi-touch display for touch screen support.
    • USB 2.0 for ZoomText Camera support.

    In summary, if you have any doubts at all, ask ZoomText support:  https://www.zoomtext.com/help/contact-us/
