Hi Kathryn,

The other folder? If it's only 358 MB of information, it's likely the recovery partition that came about in a recent update to Windows, more specifically the April 2018 update, version 1803.

If you haven't seen Local Disk D: before the update, you should perform the following workaround to remove the mapping and stop the low disk space notifications:

Hit Windows key + R together to go into Run. Type in the following: diskpart

Enter the following in the new DiskPart window:

select volume D
remove letter=D

Does not work after a reboot? Try entering this instead in an administrator session of Command Prompt: mountvol D: /D
