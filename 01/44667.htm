How do you get email links to use outlook instead of the windows mail app?

I have Outlook (part of office 365) and Windows 8 - but when I click on any "email link" it uses windows email. In older versions it would bring up outlook dialogue.  Is there a way to change that?
k.c.s. say: Because you have Outlook I am assuming you want to do all "email related" stuff with Outlook.  If so:

  • Go to the Start screen
  • Start typing "Default Programs" without the quotes
  • Select "Default Programs" when it comes up
  • Select "Set your default programs"
  • Find "Outlook (desktop)" in the list on the left
  • Select "Set this program as default"


If for some reason that last step does not work you can try to replace that step with these:

  • Select "Choose defaults for this program"
  • Put a checkmark next to the "MAILTO  URL:mailto" option and Save it
