Random keystrokes

This post is for the community as I have already solved the problem. For months I have been trying to solve it so I am hoping to save someone the trouble.

I noticed that while using my Windows 10 based PC that sometimes random keystrokes would start appearing. Symptoms included browser jumping around, context menus popping up, and when in a text entry control seemingly random letters would appear. I tried rebooting, changing keyboards (including a wired keyboard), etc. The problem continued to occur. It would even occur with NO KEYBOARD plugged and ALL KEYBOARD DEVICES UNINSTALLED from Device Manager.

I finally figured it out today!

The problem was that *somehow* a macro had been recorded during a video gaming session and assigned to the back button on my mouse. Every time I pressed it (intending to go back) it would start playback of a very long series of keystrokes.

SOLUTION: Open up Microsoft Keyboard and Mouse Center, carefully examine all button assignments and remove any macros.

If you've been waiting an unusually long time for a document to print, or if you see an error message about the print spooler, you might need to restart the print spooler on your computer.

 

The simplest way to restart the spooler is to save your work and restart the computer. You can also try these steps:

 

To restart the Print Spooler service

 

You must be logged on as an administrator to perform these steps.

 

a. Click Start, click Run. Type in services.msc and press Enter.

b. Double-click Services. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

c. Right-click the Print Spooler service, and then click Properties.

d. On the General tab, next to Startup type, make sure that Automatic is selected.

4. Windows Key+I > System > Notification & actions > Scroll down and see if Skype is ON or not

5. Reset the Skype
Windows Key+I > Apps > Locate Skype > Advanced Options > Reset

6. Update Skype
Go to Microsoft Store and check if there is any update for Skype. If yes, update it

7. Try with classic version of Skype to isolate the problem
Download and install Classic Skype - https://go.skype.com/classic.skype

8. Run Audio troubleshooter
Windows Key+I > Update & security > Troubleshoot > Playing audio > Run the torubleshooter
