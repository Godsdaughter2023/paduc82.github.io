It probably won't help you one bit!

steve cochran say:

I think that one means your inbox is corrupted.  Try moving all the messages from the inbox to another folder (create one) and then close OE and delete the inbox.dbx. file (hidden).  OE will then create a new one.

What might be of better advice for the future would be to go to File | Identities and setup a new Identity.  Then add your mail account there (watch out for antivirus inteference -- see www.oehelp.com/OETips.aspx#3).  If it works okay then you can use File | Import | Messages and get them from the older Identity.  The rationale for doing that is to give you a new, uncorrupt message store, as yours at the moment is leaning towards corruption and message loss. 

OE folders are quite susceptible to corruption and they should be kept to within a few hundred megabytes at most, and users should backup their messages frequently.
