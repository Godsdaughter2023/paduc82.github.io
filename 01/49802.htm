Original title - How do I establish if my PC has a Bluetooth facility?__If it hasn't, is it possible to download the software in order to connect to my mobile?*

a. user say:

Bluetooth is more than just software - it is a hardware device that communicates with bluetooth devices.  Software alone isn't going to do the trick.

Go to Start / Control Panel / Device Manager and see if there is any listing for a bluetooth device driver.  If not, you probably don't have bluetooth.  The best way to check is to contact the computer manufacturer and provide them with your serial number and have them check your system to see if bluetooth is installed.  Also, if you have it there is probably a Bluetooth configuration icon in Control Panel - but it may be a separate program or included under some other category.  If not, you need both a bluetooth device that is compatible with your computer and your version of Vista (and appropriate software, firmware, and/or drivers) - plus you need to set it up and configure it - and whatever is required by the mobile to establish a bluetooth connection.  Here's an article that covers this in some detail: http://windows.microsoft.com/en-US/windows-vista/Set-up-a-Bluetooth-enabled-device.

I hope this helps.

Good luck!

frederik long say: I do not know enough about your installation to answer your questions. It is likely that the behaviour you observe is caused by some third-party software you have installed. Naming the problem folder might help. You can try this method:

1. Reboot into Safe Mode.
2. Log on under an admin account.
3. Click Start
4. Type the three letters cmd and press Enter.
5. Type the following commands and press Enter after each:
    takeown  /f   "X:\FolderName"   /a   /r   /d y
    icacls "X:\FolderName"  /t /c /grant administrators:F  System:F  everyone:F
    (Answer "yes" if prompted "Are you sure?")
6. rd  /s  /q  "X:\FolderName"

Replace "X:\FolderName" with the correct folder name. Be very careful with Step 6 because the process is not reversible.
