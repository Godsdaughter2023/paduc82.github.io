Gmail stores sent messages in the Sent mail folder at the server by default.

Right-click on the account name in the folder pane and select Properties. On the IMAP tab, select Store special folders on IMAP server, then fill in the boxes with the names of the folders as they are given in Gmail (probably Sent mail, Drafts, Bin and Spam). When viewing your message list, press Ctrl-Shift-O for Options. On the Advanced tab, select Use the 'Deleted items' folder for IMAP accounts if appropriate.
