Hi Walkup,

Windows Media Player 12 is a built in feature in Windows 7. It is not possible to uninstall this program and install the older version of WMP on Windows 7.

However you can disable this program on the computer. Installing the older version of WMP i.e. either 11 or 10 is not possible.

Turning off a feature does not uninstall the feature, and it does not reduce the amount of hard disk space used by Windows features.

To turn Windows features on or off, follow these steps:

1.      Click Start and type optionalfeatures.exe in Start Search.
2.      Expand Media Features in Windows Features window.
3.      Now uncheck the Windows Media Player and click OK.
4.      For the changes to take effect you need to restart the computer.

To re-install Windows Media Player, you have to follow the steps 1 and 2 and check the option for Windows Media Player under Media Features.

Windows Update error 8024001b
http://windowshelp.microsoft.com/Windows/en-US/Help/4fdf8e8c-bdf5-417e-a465-dab62fe182391033.mspx

Usually indicates that Windows Update itself is being updated and/or installed. This could take several minutes - suggest you try installing updates later.

If you receive Windows Update error 8024001b while trying to install updates, an update for Windows Update itself is being installed. Depending on how many programs are currently running and which operating system files or folders are affected while this update is being installed, it might take several minutes before you can install other updates for your computer. Try installing updates again later.

If the problem persists after you try installing updates later, go to the Windows Update Support Center ( http://support.microsoft.com/common/international.aspx?rdpath=1&gprid=6527 ) website to request support by e‑mail.
