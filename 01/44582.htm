The Estrangelo Edessa font was originally added to Windows to support Middle Eastern languages written with the Syriac script. Because Windows has many fonts provided primarily for particular languages, but any given user doesn't speak all of those languages, many fonts have been moved into optional features. The Estrangelo Edessa font is one of these: it has been moved into an optional feature called "Syriac Supplemental Fonts".

You can add these language-related optional font features by enabling the associated language in your user profile. If that's the case, then you can also enable the optional feature separately, without modifying your language settings.

andre da costa say:

New computers that come preinstalled with Windows often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

This is how the recovery partition is accessed for the most popular makes..

For Dell, press CTRL + F11 directly after powering up the unit
