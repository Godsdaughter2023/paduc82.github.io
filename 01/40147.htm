sandeep ghatuary say:

Hi jintsfan28,


Here are some steps which you may follow and check if they help you in fixing the issue:

Step 1: Verify that the default audio device is set correctly and speakers are enabled.

To do this, follow these steps:
1. Click the Start button, click Control Panel, click Hardware and Sound, and then click Sound.

2. Click the Playback tab. Make sure the speaker is enabled.

3. If the speaker is enabled, check if it is the default sound device.

4. Select speaker and then click Set Default.

5. On the Playback tab, select the playback device, click Configure, and then make sure the configurations are set correctly.

6. Click OK to save your settings.


Step 2: Verify that proper audio driver is installed

Without an audio driver, you won't hear sound. To make sure that an audio driver is installed, follow these steps:
1. Click the Start button, click Control Panel, click System and Maintenance, and then click Device Manager. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

2. In the Device Manager dialog box, double-click “Sound, video, and game controllers”.

3. If there is a speaker icon next to the audio driver, the driver is installed. If there is a yellow question mark next to the audio driver, the driver isn't installed or working properly.

4. Go to Step 3 to reinstall the audio driver.



Step 3: Install latest drivers for Windows 7 as new operating system require latest drivers for the device to work properly

1. Click the Start button, in the start search box, type in devmgmt.msc and press enter.
2. In the Device Manager window, expand “Sound, Video and Game Controller” and select your sound card.
3. Right click on the display card and click uninstall.
4. Click the OK button.
5. Download the audio drivers from manufacturer’s website.
6. Install the driver.
7. Restart the computer.
