considerate_guy say:

Movie Maker is designed to create a *movie* with a visual element and optional audio elements. So it requires you to start with some sort of visual element. As you discovered, you can always start by adding a single photo and then having its duration match the added audio (use Project > File to music).

Then you can make edits to the audio portion using trim/split/delete, etc. for the audio track.

When you are done, in Movie Maker 2012, there is an option to save in a Audio-only format which will create an .m4a file (or optionally a .wma file).

So, while it isn't designed for audio-only editing, you can make it do that. Alternatively, I would look into a dedicated audio editing program (say Audacity) if you plan to do this frequently.
