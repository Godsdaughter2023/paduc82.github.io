ac74f001 say:
Creating a storage space might be failing because the controller you're using is reporting multiple drives with the same "UniqueId"


Try the following PowerShell command to see if thats the case:

Get-PhysicalDisk | ft FriendlyName, UniqueId -auto

If you have multiple drives with the same UniqueId, there's not much you can do. 
You could wait for a driver update that fixes this issue (might well not ever happen), or
some controllers have older drivers that report UniqueIds properly but there's always a catch: the Supermicro AOC-SASLP-MV8 sas/sata raid card works with the older 3.1.0.22 driver, but only supports drives larger than 2TB with the 4.0.0.1200 driver which has the UniqueId reporting issue.

I'm in the same boat, and am looking for a new controller card for a couple of days, but unfortunately I can't find a list of ones that have fully working drivers for storage spaces. The best I could do is say avoid the Jmicron JMB36X controller as well.

    canadian tech say:

    If you were my client, I would buy and install a new hard drive.  Laptop hard drives have a life expectancy of about 5 years.  Some last longer, but at 7 years, if it is not yet failing, it will soon.

    On top of that, all really good Windows professionals re-install their systems about every 3 years.  The system fills up with all kinds of detritus that slows it down.

    The reason I asked for the hard drive model number is so that I can give you a link for the drive's manufacturer's HARDWARE test.  You can find it by opening Device manager. 

    1. Click the start button (bottom left)
    2. Type device into the text box that pops up
    3. Click on Device manager
    4. Click the tiny arrow to the left of the word Disk

    However, since it is 7 years old, I would replace the drive any way.

    There are many other diagnostic tests that you can run that may even help you find specific problems, but if  you want a system that operates like new, you will follow my recommendation.   In the end it takes less time and has an almost guaranteed happiness at the end.

    The reason I asked for a list of installed apps is that I can then point out which ones are resource hogs that may cause poor performance.  If you got a load of apps you do not use, remove them.

    You should also download and run the malware SCANNER, ADWcleaner periodically.  It is not a program you install and therefore will not conflict.  It is free, and fast.  Do it monthly or so.
