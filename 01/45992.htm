The i5/8gb/256 is the sweet spot version, imo.  I do a lot of media dev and it handles the task just fine.  It will run  hot at times, but that's to be expected.

Personally, I've always thought the i7 (hyper threading) was nothing more than a marketing ploy as the supposed performance gain doesn't justify the extra cost.

But there is one important caveat for all versions of the SP to be aware of.  Legacy Windows software wasn't designed for the pixel density of these tablets.  You'll likely get a migraine trying to read the UI elements.  Photoshop for example, the menu items are insanely tiny.  I've found that changing the resolution to 1600x1050 makes it bearable, but not for extended periods (hours).

Office 2013 looks great, but it's optimized to take advantage of the pixel density.

This will continue to be a problem with the SP3 until software vendors start optimizing for Win8.
