Rather than upgrading from Windows 8.1, you would be much better served by doing a clean install of Windows 10. This is without doubt the best way to install an operating system.

With a clean install (which Microsoft calls a 'custom' install) your hard drive is overwritten with a fresh copy of Windows. In effect, you get a fresh start. Any problems or mistakes in your existing Windows 8.1 installation are swept away and you get to start over.

Whether upgrading or doing a clean install, one caveat exists: You must be certain that your hardware is compatible with Windows 10. Installing Windows 10 on incompatible hardware is like building a house over a ticking bomb.

If you're doing a clean install and you're sure your hardware is compatible, here is an outline of the procedure:

1- Backup all your personal data - everything that you don't want to lose.

2- Assemble product licenses for software that you purchased.

3- Install the Windows 10 media creation tool on a USB flash drive. (I tried to give you the German version.)

4- Boot your computer from the media creation tool and do a 'custom' install.

5- After the install is over, run Windows Update until you are completely up to date.

6- Reinstall your software programs, being sure to only install programs and versions that are supported on Windows 10.

7- Restore your personal data.

jeeped say:
Those beeps are BIOS POST (Power On Self-Test) error codes. A series of one long and three short is usually video related.

This is usually caused by a problem with the video card, or the memory on the video card. It can also be a motherboard issue. I don't know how this is related to updating video drivers unless something went catastrophically wrong.

If this is a PC with an independent video card, you may want to try physically re-seating the card or install another card so you can get access to the computer for additional diagnostics. There isn't much else that can be performed short of replacing hardware components (e.g. motherboard, video card, etc).
