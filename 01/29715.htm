    Task Manager shows CPU speed that is higher than its maximum - Microsoft Community

    sharath srinivasa say:

    Task Manager shows CPU speed that is higher than its maximum

    The CPU processor speed displayed in the Task Manager is higher than the listed maximum speed.

     

     

    Why is this happening?  The Resource Monitor also reports that the CPU is running faster than its maximum speed.
    vidyaranjan a v say:

    Hi,

    The “Maximum speed” that is shown on the right is the frequency the processor advertises, as reported directly from the processor.  However processors have the ability to use “turbo” states where it can temporarily run faster than its advertised maximum.  Most modern processors will do this, and the difference can be even higher than what is shown in the image.

     

    In addition, different algorithms are used to measure processor speed.  For example, the algorithm used by Intel in its branded maximum speed is different than the algorithm used by Task Manager to directly measure the speed of the CPU.  You can use other processor utilities, such as the Intel Processor Identification Utility or CPU-Z, for additional processor speed measurements.

     

    So what you are seeing is likely a difference in calculation method – Windows is not running the CPU over the recommended frequency (unless you are manually overclocking the CPU within the BIOS/UEFI settings).

    Like most recent wireless headphones, the BoAt Rockerz 400 uses Bluetooth 4.x (also called Bluetooth Low Energy or Bluetooth LE).   See, for example, https://techtopick.com/boat-rockerz-400-audio-quality-bass-test/

    The Bluetooth support built into Windows 7 only supports Bluetooth 2.x.  As far as I can determine, the HP Compaq 6710b has an "HP Integrated Module with Bluetooth Wireless Technology" that also only supports Bluetooth 2.1.  See the "System Requirements" section here:  https://support.hp.com/us-en/drivers/selfservice/swdetails/hp-compaq-6710b-notebook-pc/3356620/swItemId/ob-76382-2

    To confirm, please do the following.

    1.  Open Device Manager on your computer (click Start, type Device Manager in the Search box, and press Enter)

    2.  Double-click the Bluetooth category to expand it

    3.  Right-click your Bluetooth adapter (receiver, transceiver, etc.) and select Properties

    4.  Click the "Details" tab.

    5.  Select "Hardware Ids" in the drop-down list

    6.  Confirm that you have one of the following (if you don't please report what you do have)

    USB\VID_03F0&PID_171D

    USB\VID_03F0&PID_2A1D

    USB\VID_03F0&PID_231D

    7.  Select the "Advanced" tab

    Check the LMP version

    • LMP 0.x - Bluetooth 1.0b
    • LMP 1.x - Bluetooth 1.1
    • LMP 2.x - Bluetooth 1.2
    • LMP 3.x - Bluetooth 2.0 + EDR
    • LMP 4.x - Bluetooth 2.1 + EDR
    • LMP 5.x - Bluetooth 3.0 + HS
    • LMP 6.x - Bluetooth 4.0
    • LMP 7.x - Bluetooth 4.1
    • LMP ​8.x - ​Bluetooth 4.2
    • LMP 9.x - Bluetooth 5.0

    If you don't have LMP 6.x or better and the headset uses Bluetooth 4, that's your problem.

    Assuming that you determine in Step 7 that you do NOT have Bluetooth 4 or better, you can buy an inexpensive Bluetooth USB transceiver that will work with Windows 7.  For example: https://www.newegg.com/Product/ProductList.aspx?Submit=ENE&N=100158097%20600315449&IsNodeId=1&Description=bluetooth&bop=And&PageSize=96&order=RATING

    If you do get a Bluetooth transceiver, disable the onboard Bluetooth in your computer.
