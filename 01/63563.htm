a. user say:

literally the post Right Before this post, is asking the exact same thing.  please check at least a little ways back before posting, as you might find the answer you needed without having to ask at all.

a lot of gamers on TrueAchievements were also wondering this.  what we figured out is that some of us were able to get the new version, but some ended up installing the Win8 version, using the exact same link.

we came to the conclusion that the new version only seems to be available for people running newer builds of Windows 10.  you can either wait for more updates to be released, or join the Windows Insider Program to receive a preview build.  note that it is advised to look into this a little for yourself, as some newer build have certain features disabled or unavailable due to bugs or other issues.  also it is not the most convenient thing if you decide you want to go back to a previous build.

if you decide you do want to update to a pre-release version of Windows 10, you can do this by going to Settings, Updates & Security, and then the Windows Insider Program section.  when you enable this, you will have to restart once, then you can either choose Fast (newer, less tested builds) or Slow (less new, but more stable builds).  after choosing, it may take a little while before it goes into effect, but then you can press Check for Updates on the Windows Update section and it should begin downloading the updates to be installed.
