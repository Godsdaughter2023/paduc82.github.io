      Bluetooth Arc Touch Mouse isnt working after installing Windows 10 Creators update

      Technical Level : Basic

      Summary

      Updates can cause programs and devices to stop working, bring up new bugs. However they do, also, fix those old bugs we have witnessed kicking about on the software. This article will help you fix the problem with Bluetooth Arc mouse not working after the Creators Update. The contents within this Wiki will be taken from this thread here: My Bluetooth Arc Touch mouse isn't working after Installing Windows 10 Creators Update

      People who contributed, and solutions I have placed within this article, and acknowledgements go to: Jestoni Mac (Microsoft Forum Moderator) & SirMulia (Windows Insider) and some extra content - such as event viewer - is my own.


      Details

      Method 1 - Using the troubleshooter within Windows 10 to find and fix the problem(s)     (Posted by:  Jestoni Mac )

      Windows 10 has a built in troubleshooter, that anyone can use, and can sometimes help find and fix problems. Thus meaning, you don't have to be an IT professional to use it. Saving you time.

      To access the troubleshooter, you will need to:

      1. Go to Settings,
        You can use Cortana by typing Settings or by using your start menu and looking on the bottom/lower left hand side
      2. Navigating to Update and security,
      3. Then to Troubleshoot which can be found under Fix and Find other problems,

      Once the troubleshooter begins, it will then try and find the problems that need to be fixed on your computer/device. Once it finds them, it will begin trying to fix them. If it is not successful, then it will leave a report in the Activity Log in Control Panel > Administrative tools > Event Viewer

      Any reports in Event Viewer can be really useful in your posts as this will outline error codes and other logs about what is happening within the Device and Operating System.

      Method 2 - Making sure the latest updates are installed for the device                                        (Posted by SirMulia)

      When it comes to different software on your device, it is often that we need to ensure it is all updates and kept well up to date. This is to ensure that it works as it should, and that the latest version of the drivers are installed. For this method, please follow these instructions:

      1. Please make sure you have the latest software download for the device, by going to the download page: arc mouse software
      2. Then, open Microsoft Mouse and Keyboard Centre,
      3. Then Select your device,
      4. Then you should be able to see a setting: Restore to Defaults,

      ***Please Note: Microsoft are aware of problems that some users have experienced, especially with the device disconnecting and struggling to connect to the computer/device. This includes: Microsoft Arc Touch Mouse Surface Edition and Arc Touch Bluetooth Mouse. While a fix is been developed, you can still use the Creators Update but with a different device or roll back your device. To know how to do this, this article may help: Recovery options in Windows 10.
