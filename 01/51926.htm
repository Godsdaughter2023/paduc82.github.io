rlyttle say: Found a fix - if anyone else is having this problem:

Problem: Photoshop CS6 is not an option as default program for jpg/jpeg in Windows 7 Professional 64-bit. Not even when you browse to the program.

Solution: If you pin a jpg from your desktop to PS you can then right-click on the jpg and set the default program to Photoshop CS6.*
When you click and drag the file down to the Ps icon. Hover over the Ps icon and you are given the option to pin the file to photoshop via tool hint. When you see that option, release. If you right-click on the Ps icon you will see a list of recent items and your pinned file should be at the top.

This fix brings photoshop into the list of 'Other programs' so that you can now enable other files to open in Photoshop as well.

note*: This is also the fix if you are experiencing a similar problem with using InDesign's "Open in External Editor" function and the jpg/jpeg opens in some program other than photoshop. Just pin a jpg to photoshop.

Hi Brahma

 

You can create a detailed report of the battery status to double check the condition of the battery.

 

1. Go to Start / All Programs / Accessories. Right click the Command Prompt and select Run As Administrator.

 

2. In the command prompt window, copy/paste the following command.

 

powercfg -energy -output %userprofile%\desktop\energy-report.html

 

Press ENTER.

 

This will create a file named energy-report.html and place it on your Desktop.

 

3. Go to the Desktop and double click the file to open.

 

Scroll down to the Battery: Battery Information section.

 

You will see:

Design Capacity xxxxx

Last Full Charge xxxxx

(Where xxxxx is a number)

 

Copy this section and paste it in a reply here, for analysis.
