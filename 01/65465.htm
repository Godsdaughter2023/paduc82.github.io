How to increase RAM? - Microsoft Community

arpitjhunjhunwala say:

How to increase RAM?

I want to increase my RAM from 2gb to atleast 8 gb. But I don't want to buy new RAM . So, please tell me whether I can use my hard disk drive which is 250 gb to increase my RAM in order to improve performance. Please see that it's 100% working and it's effect can be seen. ken blake - microsoft mvp say:
I want to increase my RAM from 2gb to atleast 8 gb. But I don't want to buy new RAM . So, please tell me whether I can use my hard disk drive which is 250 gb to increase my RAM in order to improve performance. Please see that it's 100% working and it's effect can be seen. RAM information should change in computer properties.I have much available free space in my hard disk drive.


Several points:

 

1. You can't increase your RAM other than by buying and installing more RAM. It's like the number of dollars in your wallet; you can't increase it except by putting more dollars into it.

 

2. The hard drive is not a substitute for RAM, except in the sense of using some of it as a page file. But using a hard drive page file doesn't increase performance, it *decreases* it (the hard drive is much slower than RAM). The purpose of a page file is not to increase performance  but to let you do things that you otherwise couldn't do.

 

3. Are you running 32-bit Windows or 64-bit Windows? Unless it's 64-bit, you can't go above 4GB of RAM, and of that 4GB, you can only use about 3.1GB.

 

4. Why do you want to increase RAM? Do not assume that it will automatically increase performance. It depends on what apps you run, and for many people going above 2GB will do little or nothing for them.

 

 

mike hall... always say:

Not possible. Your hard drive is already supplementing the memory by way of the page file, but there is no way to make it as fast as RAM.

 

Even if you were to replace your hard drive with an SSD type drive, file access would be quicker and the computer would appear to be faster but it would still not be close to the speed of extra RAM modules..
