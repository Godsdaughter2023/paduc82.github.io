daizwei say:

Simple answer, you can't!  BT's crappy hub doesn't allow you to override a setting that you may have set for another device.  I've got the same problem as you.  What you could do, and what has worked for me in order to get an open NAT, is to make sure, first and foremost, that your Xbox One is on energy saver mode.  Whereas the console fully shuts down when you turn it off.  Secondly, make sure you place your Xbox One in the DMZ via your router settings.  Thirdly, make sure you set a static IP for the mac address that's attributed to your Xbox One.  And lastly, having set the static IP the router should state that you should restart the device that you have set the static IP to, namely the Xbox One.  Once your Xbox One is rebooted, go to Network Settings, Test Multiplayer Connection.  Once the connection test is completed hold down both RT, LT, LB and RB simultaneously.  You should see a screen come up with detailed network statistics.  The NAT Type field should only have an ellipses (...) in the field, but if you wait for a minute a description of your NAT type should pop up.  Once this occurs simply press A to see your lovely new OPEN NAT status.
