First, check the type of license installed on your system.

If it mentions OEM <method>, you’ve OEM license installed.

If it says Retail, you’ve retail license installed.

If it mentions GVLK, VOLUME_<method> then you’ve volume license installed. Note that GVLK stands for Generic Volume License Key.

Retail license have the transfer rights, while OEM license is tied to motherboard and can't be transferred to other machine.

the larch say:

It looks like Windows Search is using far more attributes than name.  To limit results to folders using only name attribute, try:

kind:folder name:=wordpress-4.0.1

The equals sign enforces exact matching -- otherwise the symbols hyphen and periods would act as general separators of search terms wordpress, 4, 0, and 1.  For some other symbols, quotation marks would also be required around the search terms, for example "item(4).0.1" to properly search for the brackets included.  If searching with a partial name, use ~= combination, as in name:~=wordpress-4.  If wanting to use wildcard characters * and ?, use just ~ as in name:~*press?4*

hover cat dave say: Is there settings for this so I don't have to type all that stuff?
