With end of support Microsoft is not telling you that your operating systems won't work.

  • net start cryptSvc
  • net start bits
  • net start msiserver

Editing Music File Speed in Windows Movie Maker. - Microsoft Community

bestoldies say:

Editing Music File Speed in Windows Movie Maker.

Original Title: Editing Music File Speed

Is it possible change the speed of a music file in Movie Maker?  I do not want to change the speed of the video just the music file.

considerate_guy say:

The short answer is there isn't a way to directly change the speed of an added audio file. The only thing you can affect is the speed of the video track and its included audio will also be changed.

So one workaround would be to create a movie with any video element (say a single picture) and then your music. On the Project tab be sure to choose Fit to Music so the picture goes to the end of the song. Now save that as a movie.

Now import that "movie" as a video clip in a new project. On the Video Tools | Edit tab, change the speed. Make it faster or slower as needed. Save that back out as an Audio-only file.

Now you can add this music, with the changed speed, back into the project where you needed it.

Alternatively, you could just use an audio editor like Audacity to change the song first before importing it in Movie Maker.

When exactly do you get the error message?

This problem generally occurs, if you add random access memory (RAM) to the computer while it is in hibernation.  When the computer resumes from hibernation and loads the Hiberfil.sys file, Microsoft Windows detects the change in memory state, and then you receive  the error message. The Video Card provides an additional memory buffer to execute the High Definition display required by various gaming programs or other software. So, you may receive this error if you add the additional memory when the system is in hibernation.

Method 1:

Boot from the Windows 7 disc to the Windows Recovery Environment and open the Command Prompt.

  1. In the Command Prompt, type the following commands, and then press ENTER after every command:
    C:\
    ren hiberfil.sys hiberfil.old
    exit
  2. Reboot your computer and check if you are able to log in to the startup screen.

You can contact the computer manufacturer for updated BIOS
