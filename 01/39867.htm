a. user say:

Windows 10 has 3 ways to backup and restore ...

Actually, no. Windows 10 has only one way to backup and restore, and that's File History.

File History was designed to backup earlier versions of individual files - and does that well - but it was not designed to backup all your files at the same time. You can use File History to backup all your folders and files at the same time, but its awkward and doesn't always work well. If you want to backup all your personal files at once, you need a real backup program from a third-party.

Looking at Windows 10's other offerings:

System Restore is not a backup, and it never was.

Backup and Restore (Windows 7) was included in Windows 10 so that people who used it to create a backup in Windows 7 can restore that backup in Windows 10. It's not supported for making new backups.

System Image Builder was deprecated with the Fall Creators Update (v.1709) in the fall of 2017. Microsoft recommends third-party software for making image backups.

1. File History can't make a backup of your C: drive. For that you need image backup software. File History is for backing up individual files. You might be able to finagle it to do three entire partitions, but that's not what it was designed to do.

Given the choice between a kludge and real backup software, I'll take real backup software every day. Even if you can somehow make it work with File History, why bend over backwards when it's so much easier and convenient to backup with a real backup program?

2. No Microsoft backup utility has ever made incremental backups. Only third-party software does that. Almost any backup program can do that.

Not that I'm a defender of Microsoft, but backup isn't their main business. There's a whole backup industry composed of developers who excel in that field. Why should Microsoft re-invent the wheel? On the other hand, there are plenty of people who don't know the first thing about installing software and aren't inclined to try it, so Microsoft adds something to Windows - at no extra charge - to give them a basic amount of backup capability for their personal files.

3. Don't go comparing Windows 10 to Windows XP. Windows - and technology as a whole - has changed enormously during that time. Don't ask me about Macs. I get mine at McDonalds.

People often confuse an image with a clone. Perhaps it's just semantics. Both are exact copies of a partition, but image software is smart enough to not copy empty sectors, bad sectors, the page file and the hibernation file (I think.) A clone copies everything. If you're in law enforcement and you just got your hands on a perp's computer, you want to make a clone. The rest of us want an image.

Macrium Reflect is very popular (and my choice) but there's lots of other good image backup software, like Acronis True Image and EaseUS ToDo Backup, among others. Just search the web for professional reviews of image backup software.

4. A restore point is not a copy of system software and it's not made every time you install software. It's a copy of a set of key system files, drivers and registry keys that if restored might bring back enough functionality to a messed up computer for a user to attempt to fix it with Windows' built in tools. It doesn't hold a candle to an image, but when System Restore made its debut in Windows ME, imaging was real esoteric. Today there's so much consumer image backup software it's no surprise that System Restore is turned off by default.

First off read the stickies/other forum posts/contact Xbox live support to see if its a common issue. However im going to assume that you've done that already and their was no "easy" fix.

I had a problem similar to this at the begining of the year, of which XBL Support could not handle, as they didn't know the error code. I'm going to assume that you can clear your System Cash (SP?)  so as to redo the update.

With a windows PC you can use a local area connection to install the update. This lets your computer "share" it's internet connection to your xbox, allowing it to miss any security settings that may be hampering the download/install (As i think was happening to me).

First off you need to hard wire your xbox to your PC. Unplug the ethernet cable from the wall jack and into the PC. (The side of a laptop or back of a desktop tower). From here on teh computer

1) Open up your Network and Sharing Center from the control panel.

2)Than select the advanced options

3) You should see icons, highlight them both and right click

4) Select the Enable bridge connection or something similar to that.

5) Wait until it says that the bridge is connceted/active

You should now be good to go. Re-download the update and install. Once the install is complete your good to go back to wifi or the wall jack for your xbox.

Hope this helps, as it took over a month to figure out for me. Xbox live support never gave me this as a diaognostic option/fix after repeated attempts with them, however there is no faults or risks by doing it this way

If this does work, dont be afraid to share this to help others!  This works for any updates, game or system, that you may need to install
