blu3_lightning say:

Minecraft, Mojang and Mirosoft

Soooooooo......

Well, it's now common knowledge about Mojang being bought by Microsoft and I would like to ask some questions add my 2 cents to this discussion.

Minecraft became popular as it provided players with freedom. Freedom to play how they wanted, without pre-set goals. This is the one and only reason minecraft has flourished the way it has.
The problem is, players are concerned about minecraft in the hands of you guys, because they are uncertain of how you will run it. Personally, I would like some insight on some main points. Here are my questions, in no particular order:

Will you allow players to continue to host their own servers?

The main reason most people play is because of the original content found on servers. Any player can create them and can host them from their own computer or a paid host. Mojang has attempted to provide competition to these servers by creating the Realms feature, but it was an utter failure due to the popularity of these servers.

Will you create an easy to use server modding program to replace Bukkit?

Mojang recently came into possession of the server mod called Bukkit. This allowed players to create modifications for their servers called plugins. These plugins allowed unique features to be added, such as changing what items did, but without having to download any install files to use them. As of Mojang taking over Bukkit, many of the Plugin Creators quit. The reason bukkit became popular is because it was open source project. There are other server modding programs, but will you provide an open source program to create these mods? 

Do you plan to change the EULA to allow servers to sell perks and abilities?

A recent point of anger to the minecraft community was the anger due to Mojang enforcing it's EULA (End User License Agreement). Servers used to pay for hosting due to "Donations". These were effectively the owners selling perks and items to players which allowed them increased convenience or an advantage over other players. The EULA was enforced and now only allows servers to sell cosmetic perks, which has lead to a severe drop in donations and has forced many servers to shut down. If you were to provide a hosting service this could work to your advantage.

Will you allow players to continue to create modifications (or mods)?

Many players play minecraft using modifications, which are game files that can be installed to add new features and textures. These mods allow players to choose what they would like to add to minecraft, without it being official.

Will you advertise on the Minecraft launcher?

Now, I'd imagine this isn't the case as Minecraft is already a paid game, but a definite answer would be good.

Do you plan to add more ingame items?

The minecraft community are not always fans of minecraft updates. Sometimes these updates add useless features, such as Mojang's Sea Lantern, which was a re-textured Glowstone with a different crafting recipe. Successful updates have included features that have had significant game mechanics added. Such as enchanting, command blocks, pistons and anvils. Also, items which do not fit minecraft's theme, such as robots or guns will not be accepted by the community.
