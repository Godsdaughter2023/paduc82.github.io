    3. Click Camera in the left pane. If it is off, turn it on.

    Solution 2: Check for camera drivers

    The camera cannot work properly if there is a problem with the camera driver. For many cases, updating driver would resolve the problem. But before that, ensure that your camera can work with the system version. To check this, follow these steps.

    3. Go to Driver tab and click Driver Details tab.

    4. In Driver File Details dialog box, look for a file name that includes stream.sys. If you find the file there, your camera was designed before Windows 7. In this case, you are required to replace it with a newer camera.
