I have had this issue, however, I have found a quick, temporary fix that should bring everything back into fully working order for a short time.

  1. Open "MSconfig" by right-clicking the start button, and click "Run"
  2. Type in "Run": "MSconfig.exe"
  3. Once in "MSconfig" click the "Boot tab"
  4. Once you have selected Boot, select "Safe Boot" and select "Network"
  5. Click apply and restart your PC
  6. Once PC starts again, it should be in safe mode.  You will know if its safe mode if the screen resolution is lower than normal, and it says "safe mode" in each corner.  If this is what you see, log in as per normal
  7. Your taskbar should work again, so click the windows button and search as you normally would: "MSconfig"
    • If your taskbar still fails to work, repeat steps 2 - 3
  8. Once in "Boot Tab", untick "Safe Boot" and click apply
  9. Restart your PC again and your computer should start again as per normal, log in and your taskbar should work again as per normal

Tip: Just remember that this is only a temporary fix and for something perminant, you should try what Rakesh Narayanaswamy suggested and run the command he provided in Command Prompt

Have you now or have you ever used any third party applications on your system where the product description includes such descriptive phrases as:

cleaner, mechanic, detective, doctor, cure, optimize, inspector, clean up, tune up, speed up, fix up, compact, compress, faster, power, boost, booster, eraser, muncher. one click fix, new, intuitive, mistakes, stops crashes, stops errors, improves, saves you money, risk free, guaranteed or any third party program that has an icon resembling a gear, cog, tool belt, wrench, spanner, screwdriver or swooping arrows?

C:\Windows\PCHealth\HelpCtr\Binaries\pchsvc.dll

Is that bar stuck to the top of your Desktop

Press Ctrl + Shift + Esc to open Task Manager

Scroll down to find Wb. exe - on the right it should say Web Bar in the Description
