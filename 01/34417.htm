Print Spooler Error 1068 - Microsoft Community

stanleyedmondson say:

Print Spooler Error 1068

I am using windows xp and have attempted to download a printer driver but I get a message that says my spooler has stopped. I tried to restart my spooler but got a error 1068 message. Can anyone help me with this issue?

a. user say:

If you no longer use the LexMark printer, you need to tell the Spooler Service that it only has one dependency and that would be the RPC Service.

When you queried the Print Spooler from the Command Prompt, did you see the no longer needed LexMark dependency listed?  If yes, you need to get rid if it if you don't need it.

It is easiest to just tell the Print Spooler what it does depend on (that would be the RPC Service).

To do that, from your Command Prompt, enter the following command:

sc config spooler depend=  rpcss

Be sure to include the space after the equals sign or the command will not work.

Then when you query the print spooler dependencies again, you should see the only dependency listed is the rpcss Service.

Start your Print Spooler now and set it to Automatic if that is your desire, then reboot your system and check the spooler Service again to be sure the changes "stick" on a reboot.
