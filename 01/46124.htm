How will I know if my computer can run Windows 10?

System requirements

If you want to upgrade to Windows 10 on your PC or tablet, here’s what it takes.

Latest OS:  Make sure you are running the latest version either Windows 7 SP1 or Windows 8.1 Update.

For Windows 8.1, see the following article:

http://windows.microsoft.com/en-us/windows-8/install-latest-update-windows-8-1

For Windows 7, see the following article:

http://windows.microsoft.com/en-US/windows7/learn-how-to-install-windows-7-service-pack-1-sp1

Processor:  1 gigahertz (GHz) or faster processor or SoC

RAM:  1 gigabyte (GB) for 32-bit or 2 GB for 64-bit
Hard disk space:  16 GB for 32-bit OS 20 GB for 64-bit OS
Graphics card:  DirectX 9 or later with WDDM 1.0 driver
Display:  1024x600

How much actual free disk space is required by Windows 10 setup?

Windows 10 uses approximately 10 GBs of disk space. The minimum disk space available is to facilitate the installation process. Some install files are compressed and need to be expanded during setup. Some files are temporary and are deleted during or after installation. Also, because you might have data and apps on the hard disk, they will be moved around during migration.

Learn more:

http://www.microsoft.com/en-US/windows/windows-10-specifications

My Tablet does not have 16 or 20 GBs of free space and I am not able to free up any additional space. What should I do?

Windows 10 will support systems with compressed installations. Windows 10 can upgrade such systems with as little as 8.5 GBs of free space.

Also see:

How to free up disk space in Windows

What happens to my previous operating system when I upgrade to Windows 10?

Windows 10 replaces your previous version of Windows by moving it to the Windows.old folder. Windows 10 becomes the default operating system.

¡firedog say:

Carbonite is not a good choice as a backup for Windows Live Mail, unless it's carefully configured to save vital files.

Contacts are stored in databases each consisting of a principal repository file called contacts.edb together with essential auxiliary files called edb.chk and (in a separate folder) edb.log. Files with the extensions .chk and .log are by default ignored by Carbonite.

The location of these files varies with the version of WLMail. In WLMail 2012, they are in a hidden system folder called DBStore at
C:\Users\{user}\AppData\Local\Microsoft\Windows Live\Contacts\Default\15.5

 

 

¡firedog say:
All I really want is to be able to access one email address that somehow got wiped out.


Log on to Windows as a new user with admin permissions and launch Windows Live Mail. Don't bother about signing in or adding accounts unless you have to. This will set up an empty contacts database for this user.

 

In Explorer, replace the \Default\15.5\DBStore folder in the new user's profile by the one containing the address that you need. Launch Windows Live Mail and check whether the old contacts list has loaded. If it has, drag any contacts you need out to the desktop to create vCard files (.vcf) containing all their data. Copy these files to a folder (e.g. the desktop) for the old user who needs it.

Log off Windows and log on as the old user. Launch Windows Live Mail, then find a .vcf file, right-click on it and select Open with > Windows Live Mail. Click Add contact to incorporate the contact into your current contacts list. 

 

Please let us know whether this worked and did what you want.


For future reference, Windows Live Mail incorporates a free, automatic regular online backup for contacts; all you have to do is sign in with a Microsoft Account. The contacts list will then be regularly backed up, and be available to you from any computer with an Internet connection. This will obviate the need to jump through hoops should anything happen to your computer.
