Can I access the registry from a Windows installation on a disk that - Microsoft Community

al marion say:

Can I access the registry from a Windows installation on a disk that is not the primary Windows disk?

The original installation was Windows 2000 Server. The old motherboard failed. The disk is fine.

There is information in the original registry that I would like to import into the new Server 2008 registry.

Can I access that original registry if the disk on which it lives is seen by another installation as separate disk?

ie: I have the original server disk installed on a bench PC. That bench PC sees the original server disk as Drive E:

I am able to see the entire disk structure of the original server disk.

Can I access the Windows Registry from that disk?

Can I export a few keys from that Registry to import them into the new server's registry?

wunders say: Yes.

You can do this by using Regedit and mounting your Disk E: registry as a "hive" in your active registry.  To do this, follow the following procedure:

  • First, Back up your current registry in case things go wrong.  Then
  • Start Regedit  (Start -> Run -> "regedit")
  • When Regedit starts up, click once on "HKEY_LOCAL_MACHINE" (or HKEY_USERS) to select it
  • Select   File -> Load Hive
  • Navigate to the Registry file that you want to extract entries from and double-click on it
    (Probably  E:\Windows\System32\Config\...)
  • A window will pop up asking for a Key Name.  Enter anything unique (I'll use "ABC" here)
  • Now look in "HKEY_LOCAL_MACHINE".  You will see a "ABC" entry.  Navigate into this entry and you will find entries to the registry hive that you selected.  You can now extract portions of it with File -> Export
  • Grab what you want then when done, click once on the "ABC" entry, and do a File -> Unload Hive.
  • Quit Regedit
  • After exporting the files, use a text editor to change the path in the exported file to where you want to put them in your registry and double-click on them to merge them in.
Hope this gets you to where you want to be,
   JW
