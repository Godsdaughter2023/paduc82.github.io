Skype "Find in current conversation" later interferes with - Microsoft Community

rseiler say:

Skype "Find in current conversation" later interferes with single clicking

This reproduces in recent 8.x versions through 8.32 on Win10 (and likely Win7).

1) Click into an existing chat with someone.

2) Narrow the window so that you no longer see other contacts.

3) Ctrl+F (or click Find).

4) Type in any common word that you know exists in the conversation and wait for the search to complete.

5) Click Cancel.

6) Now try to single click the back arrow in the upper left to go back to the contacts list. You won't be able to until you click a second time.

7) Now that you're back at the contacts list, try to single click back into the same chat. You won't be able to, even with a double click, unless you click into another chat first, go back to the contacts list, and then go into the original chat.

At this point, things will be back to normal. Until you do another search....

ßsod say:

Hello Ramu,

Another alternative is Previous Versions which is one of my favorite features in Windows Vista, and in its most basic form is simply System Restore "for specific files and folders". In your case you can restore "several versions" of the user profile you just deleted with only a few clicks!

Now granted this all depends on whether or not you have valid System Restore points, but by default all the necessary features should be enabled.

 

Restoring files using Previous Versions

1.     Click on Start and then Computer.

2.     Select the C:\ drive, and locate the Users folder.

3.     Right-click on Users, and select Restore previous versions.

4.     Highlight a folder whose date is "before" the deletion occurred and click Open.

- or -

5.     You can click Copy which allows you to copy that folder to any location on the computer.

Note: If you selected Open a new window will be display which represents the folder as it was "on that date and time". You can now simply drag-and-drop the files you want to keep (or) the entire folder.

 

Again, this all relies on System Restore and the points that have been created previously. Thus, if System Restore is disabled or not functioning as intended this will not work. Also, if you've erased or have missing/damaged Restore Points this will not function.
