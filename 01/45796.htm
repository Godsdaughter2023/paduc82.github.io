aj don say:

Hello Thana,

Thank you for posting you concern on our Community forum page.

The blue screen problem what you experienced was caused by a driver failure. By now you already know that there are two graphics card on your Surface Book 2 and both are strategically placed. The Intel graphics card is located at the back of the clipboard (screen) and the NVIDIA GeForce graphics card is at the bottom of the base (keyboard). This is the reason why you're experiencing the said blue screen error pertaining to a driver failure whenever you use the streaming app while the clipboard is detached. The app was set to only utilize the NVIDIA graphics card. Thus, resulting to the error when the said graphics card was not being detected by the app. 

This can be avoided by configuring the the device to globally utilize the Intel HD graphics card. The option can be set when you access the NVIDIA Control Panel, you can access this setting in two ways. One is by doing a right click anywhere on the desktop and choose NVIDIA Control Panel and the second way is through the NVIDIA logo that can be found in the system tray located on the bottom right corner of the desktop. 

Once you accessed the UI of NVIDIA, you'll find 3D Settings on the left. Collapse that tray and click Manage 3D settings. Under Manage 3D Settings, you will see 2 tabs go to Global Settings and set the Preferred graphics processor to Auto-select. Then go to the Program Settings tab and click Restore, this will set the apps to utilize the appropriate graphics card for it. This is tab is also the place where you can set an app's preferred graphic processor. 

The Intel HD graphics card can provide a crisp resolution when streaming videos as much as the NVIDIA GeForce does. However, when it comes to playing games or doing other task that require 3D renderring then the NVIDIA GeForce would perform best. 

Hope this information helps. 

Thank you.

Activation Error for FSX Gold on Windows 10

My computer running Vista went phutt recently, so I bought a new PC running Windows 10. I installed FSX Gold successfully. putting in the registration key when asked. Upon running FSX, a message appears 'Activation error code 1-80004005'. It is apparent that the activation data is not reaching Microsoft's server, as after 30 minutes FSX stops as it is considered not regstered. FSX was fine on Vista.

Have scoured the internet for help, but with no success.  Any help gratefully received.

gerry c j cornell say:

Unspecified error. (Error code: 0x80004005)

 

If you see this error, select the Start Start button icon button, then select Settings > Update & security > Activation, then select Activate to try and manually activate Windows. If this doesn’t work, you might need to reset your PC. For more info, see the section in Recovery options in Windows 10

http://windows.microsoft.com/en-gb/windows-10/activation-errors-windows-10

chris348 say:

Thanks for taking the trouble to reply. Found the following in another part of microsoft support. With much fear and trembling (am scared of the registry), gave it a go ..... and it worked.

1. run regedit (Windowskey +R)
2. Expand the following registry key: HKEY_LOCAL_MACHINE\SYSTEM\CURRENTCONTROLSET\CONTROL\CLASS\{4D36E967-
E325-11CE-BFC1-08002BE10318}
3. After you find the folder, double-click Upperfilters on the right side.
4. Delete all other values except PartMgr
5. Close Registry Editor
6. Start FSX / Accel and you should get the registration wizard.

Thanks again

Chris

Method 1: Check in safe mode and clean boot mode of the computer

Step 1: Check in Safe mode

Safe mode is a troubleshooting option for Windows that starts your computer in a limited state. Only the basic files and drivers necessary to run Windows are started. The words Safe Mode appear in the corners of your monitor to identify which Windows mode you're using.

Follow the steps given below and check if that helps.

  1. Type settings in search and click on Settings.
  2. Click Update and recovery, and then tap or click Recovery.
  3. Under Advanced startup, tap or click Restart now.
  4. Once your PC restarts, on the Choose an option screen, tap or click Troubleshoot.
  5. If you don't see the Startup Settings option, tap or click advanced options.
  6. Tap or click Startup Settings.
  7. Tap or click Restart.
  8. On the Startup Settings screen, choose the Safe Mode.
  9. Sign in to your PC with a user account that has administrator rights.
  10. You can restart the computer to put it back in normal mode.

If the issue doesn’t occur in safe mode, follow step 2.

Step 2: Perform Clean Boot

Placing your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue. You need to follow the steps from the article mentioned below to perform a Clean Boot.

How to perform a “clean boot” in Windows  

Refer to Windows 8.1 steps, as they are applicable to Windows 10.

Note: Refer "How to reset the computer to start normally after clean boot troubleshooting" to reset the computer to start as normal after troubleshooting.

If the issue persists, follow method 2.

Method 2: Update display adapter driver.

Step 1: Check for the latest driver update through Windows Updates.

At the Start screen, type Windows Update.

Click or tap Settings on the right side of the screen, and then select Install optional updates from the left results.

When the Windows Update control panel opens, click on “Check for Updates” on the top left corner to receive a fresh list of updates.

Incompatible or not properly installed graphics card drivers are the most common cause of this issue.

Step 2: Follow the steps below to update the keyboard drivers.

  1. Press Windows + X key on the keyboard.
  2. Go to device manager.
  3. Right-click your keyboard device.
  4. Click Update Driver Software and then follow the steps in the wizard that appears.
  5. Restart the computer and check.

If the issue persists, follow method 3.

Method 3: Check in compatibility mode.

Download the drivers from the Manufacture and follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 8 operating system from the drop down menu and proceed with the installation.

d. Once this is done, restart the computer and check if the issue persists.
