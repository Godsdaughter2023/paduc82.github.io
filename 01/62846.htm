Hi Michel,
This is Vijay, an Onedrive advisor. Try following one by one and check after each

A. Reset - Try to reset your Onedrive (Please note that no data will be lost because of below step)

Windows Key+R > Copy the below and paste and Enter
%localappdata%\Microsoft\OneDrive\onedrive.exe /reset

After few minutes start Onedrive again. if this doesn't appear, copy the below and paste into Run and Enter
%localappdata%\Microsoft\OneDrive\onedrive.exe

B. Reinstall - If Reset doesn't work - Uninstall Onedrive and install latest version -

Let's try to completely uninstall OneDrive:

Windows Key+X > Windows Powershell (Admin).

Copy the below line, paste into Powershell and Enter
taskkill /f /im OneDrive.exe
This will terminate any OneDrive processe

Copy the below, paste into Powershell and Enter

if you’re using 32-bit Windows 10
%SystemRoot%\System32\OneDriveSetup.exe /uninstall
if you’re using 64-bit Windows 10
%SystemRoot%\SysWOW64\OneDriveSetup.exe /uninstall

(Note - It is recommended that you create a backup of your registry before modifying registry as suggested below so that if you commit some mistake, you will be able to restore - https://support.microsoft.com/en-us/help/322756...)

Now Windows Key+R > Type Regedit and Enter
Go to this path
HKEY_CURRENT_USER\Software\Microsoft\OneDrive
You can remove all folders that include OneDrive numeric version (example - 18.172.0826.0010 folder)
and from the Accounts folder, remove all the folders inside

After that restart computer and install One Drive latest version from this link:

https://go.microsoft.com/fwlink/?linkid=860984

Hi Sharon,


If you have a problem seeing other PC's in the Network Section in File Explorer:

Click your Start Button, type Services and hit Enter

When the Services app opens, scroll down to find the following services, then double click each one in turn, set their startup type to Automatic and start the service if necessary

1. Function Discovery Provider Host

2. Function Discovery Resource Publication

3. SSDP Discovery

4. UPnP Device Host

Save that and close the Services app

That's It!



If you get a Network Credentials popup, you should be able to turn off that requirement . . .

Open Control Panel

Go to Network and Sharing Center

Select Change advanced sharing settings in the left pane

Expand the network you want to disable password protected sharing on by clicking the down arrow on the right of the profile

Select 'Turn off password protected sharing' and then click Save changes.

¡firedog say:

My first observation would be that changing the view of messages stored in Windows Live Mail will not change the messages themselves - only the way they're displayed in the program. Changing the view won't delete messages or bring back messages that have been deleted.

Deleting a message is a two-phase operation in Windows Live Mail. The first - selecting a message, then using one of the options available to delete it - will normally simply move the message to a Deleted items folder, whence it can be retrieved if you change your mind. I say normally, because it is possible to delete a message permanently by selecting it and pressing Shift-Delete on your keyboard. This bypasses the Deleted items folder, but it will require confirmation before it's implemented:

Pressing buttons more-or-less indiscriminately is not a particularly fruitful way of proceeding. The only one I can probably identify is the button down in the bottom left hand corner of the screen, which suggests that you were viewing the program in some other display mode than Mail, so when you clicked on the Mail icon, you were returned to the Mail window. The icon you clicked on was the first of five, correct?

To delete messages from your inbox, select them and press the Delete key on your keyboard. This will move the selected messages to the Deleted items folder. To get rid of them for good, right-click on the Deleted items folder name and select Empty 'Deleted items' folder.

These keystrokes work in most lists in Windows:

  • Click on an item to select it;
  • Click on an item to select it, hold Ctrl down and click on another to select both;
  • Click on an item to select it, hold Shift down and click on another to select both and all in between;
  • Click on an item to select it, hold Ctrl down and press A to select All items in the folder.

 
Press DEL to delete the selection.
