chagood say:

Why is Windows Explorer applying LZW compression to my 48bit RGB TIFF files?

OS:  WIndows 7 Pro 64-bit

Hardware: Intel Core i7-2600 with 8 GB Ram

Scanning software: VueScan 64-bit version 9.1.13

Scanner: PlusTek 7600i

 

I scan color negatives and save them in TIFF format with 48-bit RGB (16-bits per color).

 

The VueScan software correctly saves the files without LZW compression because LZW does not work well with 16-bit "noisy" data -- LZW "compression" frequently makes the files larger because of the large number of distinct values that must be added to the dictionary.

 

If I use Windows Explorer to modify the Date Taken tag for a file, it changes the TIFF Compression flag from 1 (uncompressed) to 5 (LZW compression) and applies the compression.  The file INCREASES in size by about 20% (from 95.6 MB to 115.8 MB).

 

Now I understand that one might argue that Explorer is trying to make efficient use of disk space IF the compression made the file smaller but what is actually happening is the worse than it was before because the file takes MORE space AND the system now has the added burden of "decompressing" the file when it is opened.

 

Windows Explorer should NOT be adding compression to TIFF files (or any files in my opinion) unless specifically requested to do so.

 

Thanks,

Chris

Hi All,

 

Welcome to Microsoft Community. Your interest in Windows 10 is much appreciated. 

Please follow the below methods and check if it helps:

Method 1: Run the SFC.

Follow the below steps:

  1. Open Command Prompt.
  2. Select “Run as Administrator”.

Now check for the issue.

Method 2: Run the DISM Tool.

If the issue persists, I would have you run the DISM tool to check the system health and will try to restore the files. 

  1. Click on 'Start'.
  2. Enter 'Command prompt' in the Search box.  
  3. In the list of results, swipe down on or right-click Command prompt, and then tap or click 'Run as administrator'. 
  4. In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

DISM.exe /Online /Cleanup-image /Scanhealth 

DISM.exe /Online /Cleanup-image /Restorehealth

To close the Administrator: Command Prompt window, type Exit, and then press Enter.

Note: The tool might take 15-20 minutes to finish running, so please do not cancel it.

Method 3:

Clean Boot

Put your computer in clean boot and check for the issue.
How to troubleshoot a problem by performing a clean boot in Windows 8 or in Windows 8.1

Note: Once you are done with clean boot, please follow step 7 in the article to put your computer back to normal startup.

How to obtain Vista recovery Media and/or use the Vista Recovery Partition on your computer.

 

There is no legal free Vista download available.

Contact your Computer Manufacturer and ask them to send you a set of Vista Recovery disk/s.

 

They normally do that for a small $ cost.

 

Also, ask them if you have a Recovery Partition on your Computer/Laptop to restore it to Factory settings.

See if a Manual came with the computer, or go to the Manufacturer's website, email or phone them for info on how to do a Recovery.

 

Normally, you would press F10 or F11 at Startup to commence the Recovery Process.. 

Another way I have seen on some models is to tap F8, and go to a list of Startup options, and start a Factory Standards Recovery from in there, by selecting the Repair computer option.

Ask them if you can also make Recovery Disk/s from the Recovery Partition in case of Hard Driver failure or System Crash.

They will tell you how to do that.

 

Each Computer Manufacturer has their way of making Recovery Disk/s.
