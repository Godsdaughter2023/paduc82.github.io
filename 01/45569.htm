chriso747 say:

The Macks,

If the message for an 'end of trial period' comes up, it means that you have not activated FSX and/or Acceleration.   You may have had the activation screen come up when in the installation phase and provided your product key, but it may not have registered.   The option for Phone activation appears to no longer apply as MS have washed their hands off this great Sim.

The only fix for this problem is to exit FSX and amend an entry in your System Registry (take care when working in the Registry - you may want to back it up if unsure).   However, some have still had this problem of non-activation of FSX and some have not been able to activate Acceleration at all, no matter what they've tried.   So, the fix is as follows:

1. Close FSX and run Registry Editor (press Windows key + R and type   regedit   and   Ok).

2. Expand the following registry key:            

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4d36e967-e325-11ce-bfc1-08002be10318}      

3. After you find the folder, double-click Upperfilters on the right side.
4. Change the value from  0  to   PartMgr  (note, this is case sensitive) and delete any other word that may be next to it and click OK.  

5. Close Registry Editor and reboot Computer.
6. Start FSX and check whether the Activation box comes up.

If FSX runs okay now after activation but Acceleration gives problems, I'm afraid that there's not much else to suggest except to uninstall Acceleration, re-check the Registry entry as above, reboot Computer, and launch FSX (i.e. without Acceleration installed) and at least make sure that FSX still works well.   Then you could try and re-install Acceleration and see if there's still a problem with it.   If there is, you may need to either uninstall Acceleration and install Service Packs 1 & 2 for FSX, or remove FSX completely from your system and purchase FSX-Steam Edition, which is basically an upgraded FSX with Acceleration included (this is a downloadable program - not boxed and no activation necessary).   If you need further advice on this, you can write again.
