How to remove Bonjour Service

Adobe newest Creative Suite 3 installs Apple’s Bonjour service even if you don’t install Version Cue. Its main goal is to provide zero-configuration connectivity between Version Cue server and the suite’s applications.
 When installed by Creative Suite 3 applications, the name used by Bonjour for Windows in the services control panel is:
 ##Id_String2.6844F930_1628_4223_B5CC_5BB94B879762##
 If you are using Version Cue, you can correct the service name just by running two commands at a run prompt box (Start - Run):
 C:\Program Files\Bonjour\mDNSResponder.exe” -remove
 “C:\Program Files\Bonjour\mDNSResponder.exe” -install

The service will now be displayed with the correct name in the Services control panel.

 Open a Windows Run prompt box (Start - Run) and type the following command: “C:\Program Files\Bonjour\mDNSResponder.exe” -remove
Navigate to the following folder in Windows Explorer: C:\Program Files\Bonjour
Rename the mdnsNSP.dll file in that folder to mdnsNSP.old
Restart your computer

You can't run 16bit programs on any 64bit version of Windows. More accurately, you can't run 16bit executables on 64bit Windows.

If a 32bit program/game (I know this works with some games) uses a 16bit installer, it can often be installed on a machine

with 32bit Windows and then copied from the 32bit machine to an external HDD/SSD/USB and then to a

machine using 64bit Windows, and it will run.

- You may also need to locate a Registry key/s the program may have created, and copy them to the 64bit Windows in some cases.

theblureffect say:

Hi ,

You can try the following, had the same problem:

In order to hear other players you must make sure that the following is set from the Xbox 360 dashboard: First, make sure that your XBOX Live Talkback Cable is firmly connected to both your XBOX 360 controller and your headset.  This is the cable with 2.5mm plugs on each end.  NOTE: This cable is permanently connected to the XL1 and XC1 headsets.

1.  From the Xbox 360 Dashboard, press the Xbox Guide button (the X button in the center of your controller) to access your Xbox Live Settings.  

2.  Navigate to the "Settings" selection.

3.  Highlight "Prefrences", and Press "A"

4.  Select "Voice" from the following menu, and press "A".  

5.  On the Voice screen, make sure that "Play Through Headset" is selected for Voice Output selection and that volume is set to 10. It will also work on "Play Through Both" however, you might hear an 'echo' effect generated from the voice duplication.  

6.  You can fine tune game audio and chat volumes using the Headset amplifier's  chat volume controls.

If this is not working, please try testing this headset out on more than one XBOX 360 controllers.  If one particular controller does not work, you know that controller has a worn out or damaged XBOX Live Chat jack.  

If chat is not working properly on any XBOX 360 controller, you may have a bad chat cable.

Hope this works.

Cheers!
