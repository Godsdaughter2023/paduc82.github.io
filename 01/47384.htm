Hello Disaacson,

Thank you for contacting Microsoft Community.

We understand your concern in this regard.

The error code 0xc06d007e, indicates that the problem is due to incorrect file settings for the program that you are trying to use or might be with misconfigured system files. I see you telling that the problem is started after Windows Update KB3135173. But this update is rolled out on 9 February 2016. There are another two Cumulative Windows Updates rolled out after this particular Update.

  • March 8, 2016 — KB3140768 (OS Build 10586.164)
  • March 1, 2016 — KB3140743 (OS Build 10586.122)

Suggest you check and install these Windows Updates to see if they fix the issue. Check Updates.

If it doesn't fix the issue or if you really think that the problem started after installing Windows Update, then try to remove that particular Windows Update to see if it makes any difference.

To uninstall an unwanted Windows Update:

  1. Type “View Installed Updates” in the Search box and then click on View Installed Updates – Control Panel from the Search results.
  2. To uninstall the unwanted update, select it from the list and then click Uninstall.

  3. If it prompt for confirmation, just confirm the same. Once it complete uninstalling, reboot your PC and see if it makes any difference.

  4. If uninstalling an update fixes the issue, temporarily block the update from automatically installing again.

In case problem still persist, then try to perform a System Restore. If an app or update that you recently installed caused problems with your PC, you can restore Windows back to an earlier point in time, called a restore point. System Restore doesn’t change your personal files, but it might remove recently installed apps and drivers.

Note: Windows automatically creates a restore point when you install desktop apps and new Windows updates, if the last restore point is older than 7 days. You can also create a restore point manually at any time.

To restore your PC to an earlier point in time

* Press Windows Key + R to open run command

* Type rstrui and hit Enter key.

* Select Recommended Restore or any other restore point prior to the date when problem started and click Next

* Follow the instructions to restore your PC to earlier. Now see if everything works fine after System Restore.

Keep us posted if you require further assistance.
