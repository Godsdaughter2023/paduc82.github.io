obsessive power say:

Xbox One discs are Dual Layer BD-ROM discs 50 GB, which is pretty much standard now.  In addition, the One S and the One X can also read UHD Blu-ray feature film discs. Neither the One S or One X can read BDXL discs, I don't know enough about the hardware to say for sure if this is due to a hardware or software issue.  Note that despite the similar storage capabilities, UHD Movie Blu-ray discs are factory pressed, while BDXL is a writable format largely for burning discs with a PC.

Despite the One S and the One X's ability to read UHD discs, they will not be used for Game media.
