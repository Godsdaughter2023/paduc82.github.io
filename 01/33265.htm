It requires more storage than an installation disk.  An installation disk fits on a dvd.  A recovery drive requires more storage than a typical dvd can hold (if you check the box).  

It includes repair tools, which I think is all you get if you don't check the box to copy system files.  In that configuration it would serve to repair startup problems.  If you check the box you get repair tools, system files and (I think) the specific drivers used by your individual system.  

It is designed to restore your computer to "factory condition" after upgrading to Windows 10 from Windows 7 or 8.  If you have an old recovery partition it will give you the option of removing it after you create the recovery drive.  

lowtech56 say: Can I do the same thing for my Windows 7 desktop, that is, create a USB recovery drive for it?  Or is WIN7 a different animal?  Continued Thanks for your assistance.

cbarnhorst say: No.  The software that makes the recovery drive isn't in Windows 7.  Also, the computer has to support booting from usb in the BIOS in order to use it.  If it does, you might check online to see if there is a third-party program that does the same thing.

saltgrass say:

While you were messing with the memory, did you change anything in the bios?  You might just make sure the correct drive or boot manager is listed in the bios as primary.  If it is UEFI, it should be a Windows Boot Manager, if Legacy, it would be the hard drive.

I have seen many folks with the message you are getting, but not really sure what happened to those.  I was wondering if it was related to the UEFI firmware, but the BCD store is kept in the EFI System partition if you are running UEFI.

The bootrec.exe utility mentioned is usually available from a recovery command prompt window.  If the BCD store is missing, you may need to use bcdboot.exe, as was mentioned to replace it.  The exact commands will depend if you have recovery media so let us know if you do and whether you have Advanced Options for Startup Repair.

If you don't have some type of OEM bootable options for recovery, you will probably need some type of recovery or install media.
