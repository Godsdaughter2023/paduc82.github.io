Family settings can block the display of content that is rated lower than the setting for the console. Make sure that content is not hidden because of your family settings. You can adjust your settings either using your Xbox 360 console or online.

Adjust your family settings using your console

considerate_guy say:

There are two things that might be a problem:

  1. You might be trying to burn a CD instead of a DVD.  The capacity of a CD is much less and also they aren't designed to go into a DVD player.
  2. You might be burning the movie file directly to a disc (CD or DVD) as a data file.  If you do that, you end up with a data disc which won't play in most standard DVD players.  This is especially true if you are trying to burn an HD version of your movie to disc.

Instead, what you need to do is to use software to create a video DVD.  This is a specialized format with MPEG2 video stored in .vob files in a TS_VIDEO folder, with an optional menu, etc.  If you are on Windows Vista or Windows 7, see if you have Windows DVD Maker since it will do this for you.

 

If you make a video DVD this way, the resulting MPEG-2 compression should mean that you can get about 1½ to 2 hours of video on the disc.  That's because the DVD standard is not HD but is standard definition (since DVDs were created before HDTVs were commonplace).  The lower resolution means less space used up.  You should be able to do this with a single layer DVD, by the way, and should only need a dual layer DVD for a really long movie.
