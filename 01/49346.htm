considerate_guy say:

It's exactly the same in the latest version of Movie Maker 2012.  You can make splits (M on the keyboard) and start/end points (I for in point, O for out point).  Once the clip is edited to your liking, save the movie using File > Save movie.  However, if you have multiple clips you want from one original long clip, you have to do a lot of rework.

 

One way to do this efficiently if you want to save multiple parts of the movie as separate clips is to start a project, make all the splits that you want.  Then save the project.  I would actually save two copies in case you need to start again.

 

Open a copy of the project, delete all but one clip (right-click, choose Remove) and then save that as a movie for that one clip.  Now undo and pare it down to just the second clip, save as a movie.  Repeat for 3rd, 4th, 5th, etc.
