george1009 say: you are using a Dell OEM SLP key on a retail motherboard. that won't work.

OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the special instructions found only on that Manufacturer's computer hardware. So when Windows was installed using the OEM SLP key (at the factory) Windows 7 looks at the motherboard and sees the special instructions and Self-Activates. (that is why you don't have the slui 4 option)

an OEM license is not transferable to another motherboard except under special circumstances.

hula_baloo say:

 This is most likely caused by a CMOS battery that needs to be replaced.  This is a small disc shaped battery on your motherboard that keeps the clock running when power is off.  They aren't expensive, but do require opening the case to access them.

 

 I hope this helps.  Good luck.

smlunatick say: Internal clock relies on the CMOS / BIOS built-in clock.  Most computer have a coin-cell lithium battery that will power the CMOS / BIOS "chip' so as to save the info when the computer is powered down.  You need find out if the CMOS / BIOS clock is correctly set and is storing the date / time when the computer is powered down.

Before anyone whose clock is running slow rushes out to buy a new
battery, he should first take note of whether he is losing time while
the computer is running or while it's powered off. If it's while
powered off, the problem is very likely the battery. But if it's
while running, it can not be the battery, because the battery isn't
used while the computer is running.
