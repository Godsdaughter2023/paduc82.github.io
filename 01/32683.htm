The error 0x80070646 or 646 is an MSI installer error that will only occur when the updates are installed via the Windows Update control Panel.  Previous workaround to resolve the immediate problem requires the installation of the update from a standalone package from the Microsoft Download Center.  We need you to stop using this workaround as it does not prevent future Office updates to fail again with the same error.

When Windows installs large updates, your user profile is copied to a temporary profile, after the update completes, your files are copied back - that is the theory!!

Sometimes Windows reboots before the files are copied back and the user ends up signed into a temporary profile - that is when the problems start . . .

Create a new folder on your C drive - Call it My Data or some appropriate name
Create a folder structure inside that folder and store all your files in there rather than in the Documents, Pictures . . . Libraries

Then if a Windows Update causes havoc again, all your files will be safe inside that folder as it is not attached to your user profile . . .
