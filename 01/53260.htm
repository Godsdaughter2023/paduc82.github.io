steve winograd say: Windows 7 does sort things out for you.  It automatically installs programs in the right folder: "Program Files" for 64-bit programs, and "Program Files (x86)" for 32-bit programs.  You shouldn't change that, and you don't need to un-install or re-install anything.

Most programs are only available in a 32-bit version.  They install in "Program Files (x86)", and they will run fine on your 64-bit system.

If a program is available in both versions, get the 64-bit version (which is sometimes called "x64").  An example of that is Prevx 3.0 .

The advantage of running a 64-bit version of Windows basically exists only
if you also run 64-bit applications under it. Bear in mind that there are

very little for you at present--puts you into a better position for the
future.

One additional point: the 64-bit version lets you use more than the
approximately 3.1GB of RAM that the 32-bit version can use. Very few people
need or can make effective use of more than 3.1GB, but if you are one of
those who can, that's something else to consider.
