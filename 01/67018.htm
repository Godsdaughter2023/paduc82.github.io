Okay, the steps below worked for me.

1. open a command window with admin privileges and attempt a "net stop wuauserv", if that doesn't work, you might want to use the GUI (I did) and go into Start -> Computer Management -> Services and look for the Windows Update service

2. It might say something like "stopping" or "starting", change it's status to DISABLED and hit OK

3. reboot

4. upon starting Windows, go back to Start -> Computer Management -> Services and ensure that the Windows Update service is in a DISABLED state and not running (keep this window open)

5. navigate to c:\windows (or installed directory)

6. look for a folder named "SoftwareDistribution"

7. rename it to "SoftwareDistribution.OLD" (other people suggest deleting it, but I didn't in case it was different on Windows 10)

8. Return to the Windows Update service and change it's startup type to "Manual"

9. go to Start - Settings (Cog icon) - Update & security and run Windows Update again.

It should start checking and downloading updates, hope it works for others!
