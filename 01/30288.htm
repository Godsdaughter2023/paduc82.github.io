Does hibernate save windows clipboard copy file ?

The clipboard is NOT a file. It holds its data in memory.

The clipboard data does survive a hibernation.

You could easily answer the question yourself like so:

  1. Press PrtSc
  2. Hibernate
  3. Return from hibernation
  4. Open Word
  5. Press Ctrl+V to paste the clipboard's contents.

a. user say:

Frederik explained it well. In Hibernation, the contents of RAM are saved to a special file on the hard drive called hiberfil.sys. This allows for greater power savings than 'Sleep' because power is not needed to maintain the contents of RAM.

When the computer 'wakes' from hibernation, the contents of hiberfil.sys - which includes the clipboard - are copied back to RAM.
