    Thank you for posting on our Microsoft Community.

    The system might have got corrupted when you forced stop the scan with AVG.

    Try the steps below to fix corrupted recycle bin.

    1. Go to Start menu, type Command Prompt.

    2. Right-click Command Prompt > Select "run cmd as administrator".

    4. Reboot computer and then you can reuse the Recycle Bin again.
