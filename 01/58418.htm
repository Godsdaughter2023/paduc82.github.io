Hi Kathy,

 

1.  Have you made any changes to your computer recently?

2.  When are you getting this error message?

 

The uncdms.dll file is associated with the Windows Desktop Search Application that is used to find and preview different types of files, such as documents, email messages, photos, and music files on a computer.

 

I would suggest you to follow the steps and check if it helps.

 

Method 1: Uninstall and reinstall Windows Desktop Search

The uncdms.dll file is installed on your PC when you download the Windows Desktop Search application. Problems with files of this application may cause a uncdms.dll missing error to appear on your system.

To uninstall the Windows Desktop Search from your computer, you need to complete the following two tasks:

 

Step1: Remove Windows Desktop Search from startup sequence

To achieve this, follow the steps below:

a. Click the Start menu and then click Run.

b. In the Open box, type the msconfig command and then press Enter.

c. In the System Configuration Utility dialog box, click the Startup tab.

d. Locate and clear the checkbox for any option that is related to Windows Desktop Search or uncdms.dll.

e. Restart your computer.

 

Step 2: Uninstall the Windows Desktop Search Application

To uninstall the Windows Desktop Search using the Add or Remove Programs utility, perform the following steps:

a. Click the Start menu and select the Control Panel utility.

b. Double-click the Add or Remove Programs icon.

c. In the Add or Remove Programs window, locate and select the Windows Desktop Search.

d. Next, press the Remove button.

e. Restart your computer.

 

 

Method 2: I would suggest to you do an online scan and check.
