I managed to get a solution to remove the update.

run CMD as admin and enter this command

DISM /online /get-packages /format:table | findstr KB3022345

then it will give you some entries

copy the entry that is similar to this Package_2_for_KB3022345~31bf3856ad364e35~amd64~~6.3.1.5

and enter it after this dism /online /remove-package /packagename:

for example:

dism /online /remove-package /packagename:Package_2_for_KB3022345~31bf3856ad364e35~amd64~~6.3.1.5

after pressing enter the update will be removed.

repeat the step for each entry the very first command will display and reboot the pc when all done.

You'll never get through trying to use SSL on port 110. I've never come across a mail server that supports it; the standard port for secure POP3 is 995.

 

Time-out problems are notoriously difficult to diagnose and fix. There could be so many reasons why the server doesn't respond within a reasonable period. A couple of things spring to mind:

  1. I assume that you've tried extending the server timeout to its maximum. If not, you'll find the setting on the Advanced tab of the account's properties dialogue (where you also disable SSL).
  2. Do you have any anti-virus, anti-spyware or anti-spam software running that scans mail? This would not normally cause a timeout on receiving - any delay caused by scanning would occur when the messages start arriving after making the connection to the server. But you never know, so try disabling any such mail scanning; you don't need your anti-virus to scan email anyway.
  3. If there are a lot of messages in the mailbox, listing them might take too long, especially if any of them is large or dodgy in some respect. To test this case, log on to the account in webmail and move all the messages out of the inbox and into a temporary user-created folder. Then try send/receive in Windows Live Mail again. Do you still get a timeout error?
  4. You can test the POP3 connection to the server using Telnet, which bypasses your mail client program. We'll do that if points 1-3 don't help.
