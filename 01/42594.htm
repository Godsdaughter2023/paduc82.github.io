sathish s v say:

Hi,

 

By default in all laptop’s it goes to sleep mode when you close the display lid. Even if the laptop is connected the external monitors, it will go to sleep mode if you close the lid.

 

In this case you will not be able to wake the computer using either external mouse or Keyboard.

 

You may have to change the power settings to make sure the laptop doesn’t goes to sleep mode when the lid is closed (this is only when you use external monitor).

 

These are the operations that can be performed when you close the lid.

 

-   Do Nothing

-   Sleep

-   Hibernate

-   Shutdown

 

You can edit the current power plan or create your own power plan to avoid laptop going to sleep when you close the lid.

 

Follow these steps to edit current power plan:

a.       Press “Windows key + W” from the keyboard.

b.      Type “Power” without quotes in the search box.

c.       Select “Edit power plan” from the left panel.

d.      You can change the options in “when I close the lid:” section to “Do nothing”.

e.      Then click on “Save changes”.

f.        Now the laptop will not go to sleep mode when you close the lid.

 

Follow these steps to create your own power plan.

a.       Press “Windows key + W” from the keyboard.

b.      Type “power options” without quotes in the search box and hit Enter.

c.       Select “create a power plan” from the left panel and follow the on-screen instructions.

Note: Make sure “Do nothing” option is selected under “When I close the lid:” section.

 

I hope this helps you to fix the issue. If the issue persists or you have any other issues with Windows, please do post it in Microsoft Community. We would be happy to help you.

I apologize.  I should have explained more carefully.

Before reading further, I also should have asked if you are attempting to install the printer software while logged in to a user account that has "administrator" privileges.  If you click Start, type user accounts into the Search box, and press Enter, you'll get to the "Make changes to your user account" screen.  Under the name of your account, you'll see either "Administrator" or "Standard user."  You need to install printers from an "Administrator" account.

Assuming that you are using an "Administrator" account, read on.

The key part of the response at that thread is this:

HKEY_CURRENT_USER32 = "HKEY_CURRENT_USER\Software\Wow6432Node" or "HKEY_CURRENT_USER\Software\" depending on the OS and the Template Summary of the MSI (if its an MSI).

What that means is that there really isn't a key named "HKEY_CURRENT_USER32"  (and no, adding it manually won't help).  Instead, if you have a 64-bit version of Windows 7, you want to go to:

HKEY_CURRENT_USER\Software\Wow6432Node\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_BROWSER_EMULATION

And if you have a 32-bit system, you should go to:

HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main\FeatureControl\FEATURE_BROWSER_EMULATION

Go to the appropriate one of those keys (or go to each of them in turn), right-click, and select "Permissions"

You'll probably see that your user name either isn't listed in the top box under "Group or user names" or, if it is there, when you select your name you'll notice that "Full" is NOT checked under "Allow" in the lower portion of the window.  The instructions at the howtogeek link explain how to give yourself "Full" permission, which should alleviate the problem that you're experiencing.

What was really wrong was my setting for the colour depth and I'm sure allot of people will also find this useful as otherwise they will run into the same issue I did.


Any modern TV supports what is known as "deep colour" and although technically changing the colour depth should not effect anything per say but it does in fact improve the picture imo.


 I had my colour depth set to 8 bit also known as 24 bits per pixel. Your average TV setting at limited and the standard rgb setting on the X Box One makes excellent use of this in fact it looks so good that I'm sure most people wont bother to switch over. When you change to PC RGB mode and full mode on your tv I find that colour setting crushes blacks.


So when you turn on PC RGB you should also change the colour depth to whatever your TV Supports, You can look this up usually on the internet or through experimentation see for yourself. If the setting you choose such as 10 bit or 12 bit only fades to a black screen then your tv does not support that colour depth. Makes it fairly easy to discover your maximum depth quickly.


So after adjusting my settings to 36 bit aka 12 bits per pixel and turning both the tv and console on to full rgb wala the picture looks great in full. Due to the fact that I did not originally understand X Box One's system options I overlooked this and it was silly of me but its an easy mistake to make when you don't know what your doing or talking about.
