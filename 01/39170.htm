mraverage say: It would turn out that after I posted that question, I found the solution that you mentioned somewhere else (although I spent 2 days researching this mess!).

The solution worked, but it was only part of the total solution.  Here's where there was a catch.....

The original install was from a SP2 OEM disc.  The system had since been upgraded to SP3 and all the latest XP updates.  So when a "repair" installation was made using the original SP2 disc, the system partially reverted to SP2 complete with IE6  (the latter being a key point here).

Here is the total solution in the correct order  (assuming a "repair" installation was made on a more updated system using an old install disc like I did):

1. After the "repair" install, run the SP3 upgrades from a local file. You will not be able to download them on that system with IE6 and you can not upgrade to IE8 without SP3 (catch 22).

2. After the SP3 updates, you will be locked out.  No activation time exists.

---Now as mentioned above by David--

3. Boot to Safe Mode.
  - Normal Safe mode is fine... no need to start in "Safe Mode with Command prompt" and then invoke Explorer because regular Safe Mode is just that, minus the command to start Explorer.  This saves a step.

4. Click Start / Run then type into the box "Rundll32.exe syssetup,SetupOobeBnk" and click OK. This resets the activation time to 30 days.

5. Click Start / Run then type into the box "regedit" and search for the "wpaevents" folder.
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\WPAEvents

6. Right-click and select "Permissions" and add "User" (or "Everyone") for full control.

7, Similarly, change the permissions on the root of the "C" drive for "User", "Administrator", and "System" for "Full control". 

NOTE!!  XP Home Edition users may need to install "SCESP4I.EXE" to allow the setting of permissions on files and folders.  This can be installed in Safe Mode and despite the prompt, there IS NO need to reboot before being able to set permissions.
See: http://www.dougknox.com/xp/tips/xp_home_sectab.htm
  - Find and download the "SCESP4I.EXE" file onto a flash drive on another machine.
  - Run "SCESP4I.EXE" on the affected machine to extract files to a folder.
  - Right-click "setup.inf" and select "Install".
  - You should now be able to set folder and file permissions in XP Home Edition.

8. At this point, you should be able to reboot and log into Windows normally and the activation period has been reset to 30 days.  But you still can't activate Windows because the Activation window (MSOOBE.EXE) won't run.

9. Install the IE8 installation from a local installation file.  You still won't be able to download it from MS on that machine using IE6.   Elect NOT to install the updates. The required updates were installed by SP3.  Otherwise, the install will hang.

10. Reboot after IE8 installation.  The Activation Window should now run after logging on and clicking "Activate Windows".

11. Activate Windows.

12. Now install all other MS Updates.

Done.
