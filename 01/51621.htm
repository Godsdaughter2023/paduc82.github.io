xircal say:

Hi,

The program will be listed in the Windows registry with a code. You need to copy that code and then run this Fixit to uninstall the application using its code:

https://support.microsoft.com/en-us/mats/program_install_and_uninstall?wa=wsignin1.0

First of all, create a manual Restore point. This is important since you're going to make changes to the Windows registry. If you inadvertently delete something you can run System Restore and revert to the time prior to making the change by clicking the Restore point you created manually which will be recognizable by the name you gave it. Here are the steps to take.

  1. Right click the Start button and click System.
  2. Click the link to System protection on the left.
  3. Click the Create button at the bottom of the next menu.
  4. Give your Restore point a name, something you can remember easily and then click Next.
  5. After the Restore point has been created close all menus and then hit Windows key + R to launch the Run command. Type: regedit and click OK.
  6. Navigate to the following location in the registry: HKEY_LOCAL_MACHINE ---> SOFTWARE ---> Microsoft ---> Windows ---> CurrentVersion ---> Uninstall.
  7. When you expand the Uninstall key (folders are called keys in the registry) in the left hand pane, you'll get the codes of all the programs which are installed on your computer. What you have to do next is to click each one in turn and then look in the right hand pane for the description of what that code belongs to. For example, in the screenshot the code which is highlighted in the left hand pane belongs to my HP printer which you can see on the right hand side.
  8. When you've identified your program, right click the code and click Copy Key Name. In addition to the code it'll also copy the full path to it. You don't need that so open Notepad and then right click and click Paste. Then copy the code including the squiggly brackets on either side.
  9. Close the registry and then click the link to the Fixit above. Since your program is unlikely to be present in the list, click the option at the bottom called Not Listed.This will take you to the next menu where it asks you to insert the code you copied from the registry.
  10. Click next after you enter it and the Fixit will uninstall the program.

Here's the screenshot I mentioned in step 7 showing the highlighted code for my HP printer.

Oh, and if there's anything you don't understand please don't hesitate to ask.
