anannya podder say:

Hi Artegon,

Here is how you can have two, or more, Internet Explorer icons pointing to a different web page.

1. Log into your user account in Windows 7.

2. Click Windows \"Start\" button and open \"All Programs.\"
3. Right-click on \"Internet Explorer\" in the program list and choose \"Send To\" > \"Desktop (create shortcut)\" from the context menu to create another Internet Explorer icon on Desktop.

4. Right-click on \"Internet Explorer\" icon created in Step 3 and choose \"Properties\" from the context menu.

5. Place the mouse cursor in the end of the text string in the field \"Target.\"
 That text line normally looks like this:
 \"C:\\Program Files\\Internet Explorer\\iexplore.exe\"

6. Type \" \" (space) and a URL address of the home page (for example \"http://www.bing.com\") you wish to associate with this icon. The string in the field \"Target\" line has to be exactly as:
 \"C:\\Program Files\\Internet Explorer\\iexplore.exe\" http://www.bing.com

7. Click \"OK.\"

8. Right-click again on the same \"Internet Explorer\" icon and choose \"Rename\" from the context menu.

9. Type a descriptive name for this icon, for example \"Bing,\" and press \"Enter.\"

10. Click on the icon created to launch Internet Explorer with the specified home page.

Hope this helps. Let us know if you have any questions.

You will need administrative privileges to do this.

1) Disable the Bosh Corporation location sensor BMA2x2 Accelerometer
Start Control Panel by right clicking on Start button.
Right click or hold your finger on Device Manager for 1 second.

(Note: leave this windows open becasue you will need to enable it after setp 3)

copy "c:\windows\system32\DRIVERS\UMDF\BMA2x2Accelerometer.dll "c:\windows\system32\DRIVERS\UMDF\BMA2x2Accelerometer.dll.win10_backup"

copy "c:\windows.old\windows\system32\DRIVERS\UMDF\BMA2x2Accelerometer.dll" "c:\windows\system32\DRIVERS\UMDF\BMA2x2Accelerometer.dll"

Start Control Panel by right clicking on Start button.
Right click or hold your finger on Device Manager for 1 second.
Select run as Administrator > Sensors > BMA2x2 > Driver > Enable.

No reboot is required.

    a. user say:

    Hi Thomas,

    Are you looking for the feature in Windows RT by which you can record the screen and output a video file?

    You can create a Screen Capture by pressing the Windows button on the Surface (the Windows Logo found on the front of the device just below the viewscreen) and the volume down button, found on the upper right side of the Surface.

    We also have an inbuilt application named as “Steps Recorder” which can automatically capture the steps you take on the computer. It also creates a text description of where you clicked and the screen shot during each click. This application was also available in the Windows 7. You can have more information from this link:

    http://windows.microsoft.com/en-IN/windows7/How-do-I-use-Problem-Steps-Recorder

    Note: Holds good for Windows 8.1.

    Note: You will not be able to create short videos of the screen recording using this application.

    To launch this application, you just need to type “Step” from the Start Screen and you will be able to find the “Steps Recorder” in the search result.

    To search for the screen recording apps in Windows Store, you may follow these steps:

    a) Tap on Windows Store App icon from the Start Screen to open it.

    b) Swipe in from the right edge of the screen and then tap on “Search” option from the Charms bar.

    c) Type “Screen recording” in the search box and search for an app which can record the screen and output to a video file.

    Hope this answers your query. You can write back to us for further queries/issues related to windows and we will be glad to assist you further.
