First of all without providing any information about what kind of business you are planning to use your Windows Licence for it is impossible to give you more then a generic answer.

Your OEM licence means that you can not make many changes in hardware until you need to contact MS for confirmation that your hardware change isn't a complete new computer. Mainboard change would trigger this. Change of video card, RAM etc should not trigger this normally, but this is based on a algorithm where mainboard have the largest impact. If you need to change mainboard due to a repair then it might be accepted, but you have to contact MS CS. That is the main difference between OEM and Retail, that Retail can be moved to another computer.

As for how you use your computer and your Windows 10 Home OEM licence it is no problem to use it for work or private. The main difference would be if you had a big company that might be able to purchase cheaper licence in bulk [Volume Licensing Service Center (VLSC)] and to have several identical OS deployed at once, then it would be possible that it also had some kind of restriction. An example are Academic Institution which offer cheaper Windows licence (and other apps) as long as you are a registered student or part of academic professional working there .  If you had been given a computer through work with a windows licence that is part of a bulk licence (VLSC), then you might also need to change licence to Retail or an OEM (OEM licence are actually meant only for computer builder to be sold as part of pre build computer).  Companies that have volume licence and later sell or let employees take over work computer need to be aware that their licence is bound to that company and need to get OEM or Retail licence if they should follow EULA as expected.

Basically OEM and Retail licence are both independent licence in that sense which can be used for home or small work place as long as there is no special need for what is offered for connecting to network or domain with those version of Windows.

Windows Home lack features like encryption of hard disk, don't have that much of tools for administration of network as professional version have and there are also some other minor difference. Just be aware that Windows 10 is continuously in change, so features that is exclusive for each version might change over time and what is main difference might be less in the future.

For larger business or work place related OS, there is also server version, but that is based on another kernel then Home an Pro.

If you really want to be sure about what you can and can not do, then read EULA and TOS which you should have seen when you activated Windows for the first time. You have to accept those terms to even start Windows, so there you have your legal terms. What MVP tell you will make no difference as it is those terms you have to follow as long those terms aren't contradicting local laws.
