How to set the window mail default setting to no border?

Open a new mail message with your signature. Click on the image and use the Picture tools to format it as you wish. Then click Set to default in the Settings section of the Format ribbon. The settings you have chosen will be used by default for images you insert in the future, until you change them and click Set to default again.
