Hi,

Thank you for contacting Microsoft Community.

I understand that you want to merge the two drives.

I will certainly help you with this.

I would like to inform you that, if you have two different drive, it will always be good so that it will not impact on performance, also if something happen to System drive, you will lose all your data, so it’s recommended to have two different drives. But if you want to merge, I would suggest you to go through the below steps:

Steps:

1. Back up to an external device any important data on D and, optionally, move the same important data from D to C.
2. Press Windows key+X.
3. Click on Disk Management,
4. Right-click the graphic of the D partition and select Delete.
5. The partition returns to Unallocated for a Primary Partition or Free Space if it‘s a Logical Drive within an Extended Partition. Delete the partition if it is an extended partition.
6. Now right-click the graphic of the C partition and select Extend.

Note: Make sure that the data files are backed up. Converting a hard disk partition or volume from NTFS to FAT32 requires that you reformat the partition, which deletes any data on it. Be sure to back up any data you want to keep before you begin.

jason-h. say: When you plug in your printer, does it show in Devices and Printers?  (Click 'Start' > Click 'Device and Printers')

If so, try the steps below:

Click 'Start' > Click 'Device and Printers'
Right-click your printer and select 'Printer Properties'
Select the 'Advanced' Tab
Click 'New Driver' > Click 'Next' and follow through with the Vista Driver install.

When the Vista driver is installed, use the arrow down button on the Advanced tab to select the Vista driver.

Please let us know if this did or didn't help to resolve the problem.
