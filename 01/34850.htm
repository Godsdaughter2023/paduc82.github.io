The Spotlight images are stored in this folder:

andre da costa say:

Option 1:

If the COA sticker is damaged, then you would need to use the recovery partition to reinstall Windows 7, this will exempt the need to use the product key to activate your Windows 7 installation.


Option 2:

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee. This will exempt the need to use the product key also.


Option 3:

If the neither options 1 or 2 is possible, then you need to invest in a new retail full version license.

You cannot use a copy of an OEM version from a Dell computer in an HP computer. It violates the terms of the OEM licence as an OEM licence ties the copy of windows to the computer on which it is first installed.


You are likely also to get the same problem after you replace the motherboard in the Dell computer as changing the motherboard constitutes a new computer. You either have to purchase a new copy of Windows or install an identical Dell motherboard. The latter can be impractical because identical motherboards cease to be available. You should discuss the matter with Dell Technical Support to get the best advice.
