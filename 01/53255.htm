Hello,

Thank you for posting your question on Microsoft Community forum.

As per the description, I understand that you want to know the file location for Voice Recognition and Handwriting Recognition in the system.

I would suggest you to follow the methods and check if it helps.

Method 1:

To download the Windows Speech Recognition Profile tool, please follow the link provided below.

Windows Speech Recognition Profile Tool

http://www.microsoft.com/en-us/download/details.aspx?id=16296

Note:  The file format is .wav file format.

After downloading the tool, please follow method 2

Method 2:

How to Copy the Microsoft Speech Recognition Training File to another Computer, please follow the steps given below.

How to Copy the Microsoft Speech Recognition Training File to another Computer

  1. Browse to the download page for the Windows Speech Recognition Profile Tool.

  2. Click the "Continue" button to validate your Windows installation.

  3. Then scroll down the page and click the "Download" button next to "WSRProfile.exe."

  4. Save the file to the desktop.

  5. Double-click the file "WSRProfile.exe" on the desktop.

  6. Click "Backup my speech profile."

  7. Click "Default Speech Profile," or the name of the speech recognition profile that you want to back up. Click "Next."

  8. Click the "Browse" button, and navigate to the removable storage device that you would like to use to move the profile to the new computer.

  9. Type a name such as "Speech Profile," then click "Save." Windows saves your profile. Click "Close."

  10. Move the backed-up speech recognition profile to the second computer using a removable storage device like a CD or USB drive.

  11.  Move the program "WSRProfile.exe" to the second computer as well, or download it again.

  12. Run "WSRProfile.exe" on the second computer, and click "Restore my speech profile."

  13. Click the "Browse" button. Navigate to the removable storage device you used to move the speech recognition profile to the second computer, and double-click the profile.

  14. Click "Use the current speech profile" to make the backed-up speech recognition profile the default on the second computer.

  15. Click "Close."

Hope this resolves the issue. If issue persists, you can write to us and we will be glad to assist you further.

Thank you.

Hi

A message in the Wireless little Window that says Connected does not means that you really have a valid functional connection.

If everything above is OK you have to be able to connect to the Wireless Router. (WIFI Hot Spot).

Connecting to the Router means that you can enter the Router's core IP into an address bar of a browse, be able to connect, see, and configure the Router's menus.

If it does not connecting to your own Wireless Router, log from any computer that can connect to Wireless Router with a Wire, disable the Wireless Security, make sure that the Wireless broadcast SSID is On,  and try to connect with No Wireless security.

Enable the Wireless security after you mange to establish a functional connection.
