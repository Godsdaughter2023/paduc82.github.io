hanymf say:

Dear All including Microsoft Agents

Code 12 has two different issues 

The Normal one whereas everybody think it is conflict in resources and accordingly everybody try to reply using Microsoft tools and systematic diagnostic steps

But,...

Actually I discovered it is a power management issue, in simple way Microsoft starting with windows 7 , 8 , 10 some Build versions have a problem in recognizing the correct power distribution for the devices due to the fact that each laptop motherboard has its own power distribution, due to the different ways of devices related connections and in some cases it might be solved by updating Bios, but mostly not, and in most cases will be either for the LAN/Wireless cards or Graphic cards.

So Microsoft windows read the devices according to connections and repeat (Double) some main chipset devices, and using the resources for the double once, hence the other devices in chain connection will not get the enough power and resources and in some cases will return code 45, and code 12

The most common case in that issue in the motherboards which has PCMCIA due to chain connection, so it will not get enough power to all the devices connected to the PCMCIA card

To solve this problem until Microsoft fix this bug in the software , you have to find the way to redistribute the power to your devices , and the simple way to do so

1. In the device manager view by connection not by type

2. Most probably you will find that the device you have the problem with; would be in a chain connection include more than 3 devices connect to one only for example PCMCIA card

3. Buy external PCMCIA to USB card which have extra USB power cable 

4. Insert the Card , connect to it the USB power cable, and connect the other USB end to other USB in the same laptop; This will redistribute and give extra power to the PCMCIA card needed for the connected chain devices 

5. Restart the laptop , then you will find that Window will read correctly your devices and distribute the resources correctly

Please note that do not remove the cards afterward from the slots as it should be there all the time 

I hope that Microsoft now understand the original problem and try to solve it either alone or with Motherboard manufacturers like Intel for example , taking into consideration that they have to update the software drivers for the motherboards including the old models

Hany

Check and reseat all cards, memory and cables (on both ends when possible) actually
remove and replace - do not just snug. On a laptop about all you can do is reseat the
memory and possibly check the contacts/connection to the hard drive. Memory could
be the problem so  if you have more than 1 stick swap them in and out to see if the
problem follows one stick or try new memory. Antivirus/anti-spyware/security programs
have been known to cause or have a role in this issue.

Most general fix is updating drivers, BIOS, and low level chipset drivers and other control software.

Also check cabling to hard disk.

=========================================================

Bug Check 0xF4: CRITICAL_OBJECT_TERMINATION

The CRITICAL_OBJECT_TERMINATION bug check has a value of 0x000000F4. This indicates that a process or thread crucial to system operation has unexpectedly exited or been terminated.

Important Info If You Have Received a STOP Code

If you received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss. A hardware device, its driver, or related software might have caused this error. If your copy of Windows came with your computer, call the manufacturer of your computer. If you purchased Windows separately from your computer, Microsoft provides support. To find contact info for Microsoft or your computer manufacturer, Contact Support.


If you have experience with computers and want to try to recover from this error, follow the steps provided in the Microsoft article Resolving STOP (Blue Screen) Errors in Windows.


These actions might prevent an error like this from happening again:

  1. Download and install updates and device drivers for your computer from Windows Update.
  2. Scan your computer for computer viruses.
  3. Check your hard disk for errors.
CRITICAL_OBJECT_TERMINATION Parameters


The following parameters are displayed on the blue screen.

Parameter Description

1

The terminating object type:

0x3: Process

0x6: Thread

2

The terminating object

3

The process image file name

4

Pointer to an ASCII string containing an explanatory message

Cause


Several processes and threads are necessary for the operation of the system. When they are terminated for any reason, the system can no longer function.
