burrwalnut say:

Press the Windows Logo key+X (or right-click off the bottom left corner) and choose Power Options > Choose what the power buttons do > Choose settings that are currently unavailable > Scroll down and put a check mark in Show Hibernate > Save changes.

 

If the option is still not showing, perhaps Hibernation has been turned off. Turn it back on via a Run window (Windows Logo key+R), type powercfg.exe -h on (note the two spaces) and press Enter.

 

Also, on the above screen you can select whether you want to enter a password on wakeup.

atthisaddress say:

Hello vinodchem123,

 

The Sleep mode in Windows 8 is not the same sleep mode as previous versions of Windows, it is a controlled suspension, more akin to hibernation than to sleep. That's why you won't find any settings for Hibernation in Windows 8, it has more or less been folded into and renamed as Sleep.
