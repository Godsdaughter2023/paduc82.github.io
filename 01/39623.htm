Step 1: Boot in Safe Mode with Networking and determine if issue is present. In my case, it wasn't.

Step 2: Run "msconfig" and disable all non-microsoft services, reboot and determine if issue is still present. In my case, it was, so re-enable all services and reboot.

Step 3: Run "task manager" and disable all startup items. Determine if issue is present. In my case, it wasn't so the culprit is a startup item.

Step 4: Run "task manager" and disable startup items 5 at a time and determine if issue persists. Once you get to a group of items that triggers the issue, restart and disable one by one until you have pinpointed the specific item.

In my case, the culprit was "Dell QuickSet" which, as mentionned by Robert, is related to the "Ambient keyboard light sensor". I have a couple of steps which should assist you to fix the issue.

 

Method 1:

 

Try the steps mentioned in this link and check if you are able to see the pictures:

 

Show hidden files

http://windows.microsoft.com/en-us/windows7/Show-hidden-files


Along with this try to uncheck these below the show hidden files temporarily and check:

 

i. Hide empty drives in the Computer folder.

ii. Hide extensions for known file types.

iii. Hide protected operating system files(Recommended).

 

Note: After the files are transferred make sure the options mentioned above are changed back to the default. 

 

Note: If you are able to see the files in the pen drive, do a right click on the folder and click properties. uncheck hidden from the General folder and then transfer the files.

 

Method 2:

 

Run a virus scanner and check:

 

Virus is known to cause these issues. Scan your computer from Microsoft Safety Scanner, which helps to get rid of virus, spyware, and malicious software.

 

http://www.microsoft.com/security/scanner/en-us/default.aspx

Note: The Microsoft Safety Scanner expires after 10 days from the date of installation.

Note: Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.
