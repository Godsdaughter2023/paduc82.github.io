a. user say:

A computer gets 'low on memory,' that is, low on available RAM because of applications that run in the background.

The list of applications in the Settings app is deficient in two ways: 1) It only lists apps that you can choose to run in the background vs. apps that actually are running in the background; and 2) it only shows legitimate software.

Task Manager shows you what is actually running in the background, which may be more than you thought. Your first step is to identify what software processes are using the most RAM.

Excessive use of RAM has two basic causes: 1) a so-called memory leak occurs when an application closes but doesn't release all the RAM it was previously using; and 2) malware - which doesn't care if it uses RAM efficiently. Memory leaks can occur due to faulty programming (no one ever said that coding was easy) and using software versions that are not supported under the version of Windows currently in use.
