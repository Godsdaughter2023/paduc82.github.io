Hi Yaypony,

When there are 2 or more computers connected in a network using a router/ switch. If you are able to get a valid IP address 192.XXX.xxx.xxx but the computers are not connected to the internet. Then, you receive a message local only which means that your computers are able to see each other locally but cannot connect to the internet.

However, if your router or switch is unable to provide a valid IP address of 19.xxx.xxx.xxx and you get an IP address of 169.xxx.xxx.xxx instead which means that your computer cannot communicate to any of the local machines as well. Then you will get the message "limited connectivity"

Local Only: http://support.microsoft.com/kb/947041
