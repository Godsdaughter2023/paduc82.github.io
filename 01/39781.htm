frederik long say:
I did boot off the flash drive (USB) without any password

System Restore is a wonderful tool - when it works. Unfortunately it fails far too often. You have one more option to fix your problem. It involves using a backup copy of the registry which might still be intact. The process is involved and requires some careful typing. Ask a computer-savvy friend to assist you if you lack experience in this area. Note that it might reset your password to an earlier value.

1. Use your bootable flash disk to boot the machine into Windows Repair Mode.
2. Select Command Prompt.

3. - 5.  (not relevant in your case)

6. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing these commands and pressing Enter after each:
    dir C:\Win*
    dir D:\Win*
    dir E:\Win*
    until you find the Windows folder.
    I shall call this drive "Q:" even though it won't be Q:.
7. Type these commands exactly and press Enter after each:
    cd  /d  Q:\windows\System32\config
    xcopy  *.*  Q:\MyRegBack\
    cd  RegBack
    dir
8. Examine the dates of the Software, System and SAM files. 

    Were the files created before or after your problem became apparent? 

    Are their sizes larger than 0 bytes?

    Only proceed if the answer is Yes to both questions.

9. Type these commands and press Enter after each:
    copy /y software  ..
    copy /y System  ..
    copy /y Sam ..
    (the two dots are part of each command)
10. Reboot normally. If things do not work out then you can backtrack by copying the original registry files from Q:\MyRegBack to Q:\Windows\System32\config.
