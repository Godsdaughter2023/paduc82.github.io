naman r say:

Hello,

Thank you for writing to us at Microsoft Community Forum.

The error code "The specified device name is invalid" appears if you try to name a folder with a name which is a restricted MSDOS device name.

MS-DOS Device Driver names cannot be used as file names in Windows. As a result, we are unable to name folders as con, aux, nul, etc.

Microsoft MS-DOS had reserved these names for these system device drivers.

  • CON : Keyboard and display
  • PRN : System list device, usually a parallel port
  • AUX : Auxiliary device, usually a serial port
  • CLOCK$ : System real-time clock
  • NUL: Bit-bucket device
  • A:-Z: : Drive letters
  • COM1 : First serial communications port
  • LPT1 : First parallel printer port
  • LPT2 : Second parallel printer port
  • LPT3 : Third parallel printer port
  • COM2 : Second serial communications port
  • COM3 : Third serial communications port
  • COM4 : Fourth serial communications port

For more information you can also refer:

http://support.microsoft.com/kb/74496/en-us

Hope the information provided helps. Do let us know in case you need any help/information related to Windows.

As it's a new PC, it could be as simple as the CD drive isn't set to automatically run. 

You can enable it from Start>PC settings>Devices>AutoPlay, make sure the slider is 'On', then in the Removable Drive drop down box select your preferred option.

You can manually run the installation from File Explorer. Double click on the drive to display its contents then look for a file called install.exe or setup.exe, or something similar (the file extensions may or may not be displayed). If there's one called autorun.inf , open it with Notepad and look for the entry Open= which will tell you the name of the install file. When you've found it just double click on it and the installation process should begin.
