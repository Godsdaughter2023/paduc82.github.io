Here are some of the ways and methods of how you can open the command prompt with elevated privileges and run it as an administrator.

advice

For all the methods to run the command prompt as administrator, when the steps involve right clicking on the item and then selecting "Run as Administrator", you can replace it by pressing Ctrl + Shift + Enter simultaneously. It is the general keyboard shortcut, which causes the elevation to "Run as Administrator".


Method 1
1. Click on the Start button. Find the menu item Command Prompt (buried deep within "Accessories" under "All Programs" or displayed in the program's access history). In Windows with the Modern Metro interface, such as Windows 8, Windows 8.1 or Windows 10, it is located under the Windows System on the Start screen.

Alternatively, press the Win key on the keyboard or click the Start button, and then type cmd in the text box to start the search to find the Command Prompt.

2. Right click on Command Prompt.
3. In the right-click pop-up menu, choose Run as administrator.

4. Press Continue or Yes to confirm the UAC elevation warning message (the path C: \ WINDOWS \ system32> will appear on the command line instead of the user's path).


Method 2
Note: This method only works if the UAC (User Account Control) is disabled.

1. Right-click on Start and select Run.
An alternative way to open the Execute command is by pressing the Win + R keys at the same time, useful in Windows versions that do not have a Start menu.

2. Type cmd in the "Open" text box.
3. You should see "This task is created with administrative privileges".
4. Click on OK (OK) or press Enter.

Method 3
Note: This method only works on Windows 8, Windows 8.1, Windows 10 or later and their equivalent servers.

1. Right click on the Taskbar and select Task Manager.
2. Click More details if the Task Manager is showing "Less details" (ie no menu bar).
3. In the menu bar, choose File> Run New Task.

4. A "Create new task" dialog box is opened, which is identical to the "Execute" dialog, but with an additional option. Type cmd in the Open text box and select the Create this task with administrative privileges setting.
5. Press Enter or OK when you are finished.
