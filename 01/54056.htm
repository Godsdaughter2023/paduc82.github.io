I have this same issue when I import photos from my Olympus Digital Camera.  It automatically adds a caption with the camera name in Photo Gallery.  Then if you bring those into Movie Maker, you get the captions showing there.

 

There are two solutions.   The first would be to turn off the auto importing of captions.  To do that, go to the File menu (Alt+F) and choose Options.  On the Captions tab there is a setting for whether or not you want to have it automatically bring in captions with your pictures.  If you turn that off, you'll no longer get the "Olympus Digital Camera" tag added.

 

The other option, if you have already added the titles, is to go to your storyboard.  You'll see the captions as rectangles below your timeline.  Select the first one and then type Ctrl+A to select them all.  Right-click and choose Remove which should then remove them all.

frederik long say:

But how can I get access to the "system restore" ? Is it automatically launched by the CD ? 

When you boot the machine off the Repair CD then you get a menu with several options. One of them is the System Restore option.

To boot off the CD you must either make the CD drive your primary boot device, or else select the boot device at boot time. This is not entirely trivial for someone who has never done it before.

Be careful when dealing with the boot menus. The System Restore option will restore Windows to a previous point in time, provided that System Restore was active. The Image Restore option will restore Windows to its factory condition, with the total loss of all data.

If System Restore was not active when your battery failed then you could use a more complicated method to fix your corrupted registry file.
