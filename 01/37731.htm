jase-in-atl say:

This can happen if you do NOT  have the "copy files to my itunes media folder when adding to my itunes library." selected and you have a networked drive mounted from which you added files. If you launch iTunes and that same drive is not mounted , iTunes thinks the files are missing.

The same is true for any situation where the path to your media files changes: i.e. you mount a network drive as "Shared Media Folder" and later change the share name to "Media Folder"... This happens with local directories as well if you do not let iTunes manage your media files (which you should NEVER do) and you do not copy files to your iTunes media folder. Changing the location of your media files or any part of the path to those files gives you this error.

This can also happen if you did not complete the transfer of songs to your iTunes media folder---even though iTunes shows the files in your media list. I added about 200 discs to my iTunes library (which I had carefully managed for years with full ID3v2 v1 APE, and Custom tagging support embedded in every file). When my computer restarted because Microsoft installed updates automatically (though I had that option turned off...), I lost access to all of those files though they show up in my library. Trying to add them again gives me the duplicate error message. 

iTunes is the worst.

My only workaround is to delete all of the files with the ! and restart Windows. Launch itunes, and then sign out of your account. Sign back in to your iTunes account and add the "missing" folders again. If you don't have to use iTunes, perhaps you could use a more robust solution like Helium...by far and away my favorite.

Good luck with your media library. Best wishes,

jase

maverick.69 say:
I wanted to prevent Windows Live Messenger login window from popping up whenever I logged in to Windows 7.

So how would I go about deactivating apps that launch on startup?

To disable Windows Live Messenger upon system bootup, you can either use the msconfig utility or look in live messenger itself to disable it. You will most likely NOT find windows live messenger in the startup folder.  I seldom use live messenger these days but if I recall correctly, there should be an option within messenger to disable it with each system bootup. If not, msconfig should easily take care of that problem.

I can't seem to find the target location of the Startup shortcuts.  It doesn't show when I right-click the folder and select Properties.

How can I find the target location of these "shortcuts"?

They're called junction points in Vista & Win7.  To find the path to where these junction points lead to, open up the command shell. type 'cmd' at the run command line.  At the C:\ prompt, type 'dir /a' without the apostrophy.

To the right of the directory (or folder name) it will say <junction> and then display the full path right next to it.

 

In XP, I would go to the Startup folders for the current user and for All Users to see if it gets launched from those folders.  However, the paths to the Startup folders includes the Documents and Settings folder, which is not accessible in Windows 7. 

There is a very easy way to access the Startup folder.  Once you are logged on, simply type 'startup folder' into the search menu.  Even simpler, you can click on Win7 Start button.  > All Programs > Startup.

The startup folder is actually still present in Vista & Win7.  The reason why you are getting all confused is that you were clicking on the junction points.  I know it's a bit confusing at first.  But once you understand why these junction points exist and where they lead to, it will make complete sense.

I would do a search on the forums here to further understand this.  Here's a good start.
