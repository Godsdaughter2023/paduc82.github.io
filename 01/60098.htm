lemp say:
Click on the link at the top of that Gateway support page that says:
Need to know the hardware on your system to choose the right driver? Download Hardware Vendor Detection Utility

Run the file that you downloaded, and it will tell you whether your "VGA" device is made by nVidia, Intel, ATI, or Gateway. 

Alternately, or to double check, you can use either of these tools that are part of Windows:

A.  Device Manager
Open Device Manager (Start > Run > devmgmt.msc > OK)
Click the + next to "Display adapters"
Look at the name of your display adapter (video card) to see whether it is from nVidia, Intel, ATI, or Gateway

B.  System Information
Open System Information (Start > Run > msinfo32 > OK)
Click the + next to "Components"
Click "Display"
Look at the information in the right pane to determine the maker of your video card

Back at the Gateway support page, select your operating system from the drop-down box (for the video driver, it doesn't make any difference which version of XP you select; they all use the same driver).  If you don't know which version of XP you have, right-click on My Computer and select Properties.

Select the "VGA Driver" corresponding to the maker of your video card.  If there is more than one, use the newest one.

After you download the zip file, extract the contents (right-click and "Extract All") and then find and read the readme.txt file.  This will give you the installation instructions.

Failures within the first 30 minutes typically are CPU related. This can be due to one or more issues.

Core CPU voltage is too low, enter the BIOS and increase the CPU voltage in very small steps and see if it stabilized and Prime can run for at least 3 hours. Note: Since you have a laptop there may not be a BIOS CPU voltage option, also check for EIST, is it enabled or disabled.

Memtest does not catch everything: Download and install CPU-ID (CPUZ): http://www.cpuid.com/cpuz.php
This utility shows the clock speed of the CPU under various load conditions
(when using Intel's EIST speed step technology and Prime 95 for instance).
Note #1:
CPU-ID has two tabs (Memory and SPD) which show the actual memory speed and 'SPD' tab shows the rated speeds
for each memory slot that is populated.
Note #2:
Compair the two values, the actual speed should not exceed the rated speed of your memory.

Core Temp:
Monitor the temperature of each core of your processor.
Note: For non overclockers using the stock Intel/AMD heatsink and cooling fan you can expect a temperature range of
35 to 40C at idle and from 60C to 65C max when running Prime95.
http://www.alcpu.com/CoreTemp/
