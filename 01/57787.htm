how to save new apps by changing default location?

a. user say:

Hello,

 

 

In Windows 10, applications that are downloaded from Windows Store are automatically installed in the Windows Apps Folder. However, if you want your apps to be installed on another drive you can follow the steps bellow:

  1. Click on Start > Settings > System > Storage.
  2. Go to the Save Locations section,
  3. Click on the drop down menu below New apps will save to and then select the desired partition or drive.

Windows 10 will automatically install apps downloaded from the Windows Store in the new installation folder. For the ones that were already installed in your system, you can move them manually to the new location by:

  1. Go to Start > Settings > System > App & features.
  2. Launch a search by typing the name of the desired app into the Type an app name field. Once found, select the app from the list and click on the Move button.
  3. Select the destination drive and click Move.

 

Should you have further questions please let us know.

Yes this a common thing with the console. There are a few things to look at first I'm assuming you talking about the original Xbox One and not the S model.  The original Xbox one uses touch sensitive switch to power the unit on and off, they are sensitive to a ground fault condition.  This can be caused by having the console plugged into a power strip that is a surge suppressor.  The Xbox One console have build in surge suppressors so they need to pulled directly into the wall outlet and not a surge suppressor power strip ( https://support.xbox.com/en-US/xbox-one/console/connect-power-supply ).  Also make sure it's not on a circuit that has a GFI breaker, these are the ones like to bathrooms usually.
