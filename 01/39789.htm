ralf_g say:

Could be you need to change the display adapter setting from clone (duplicate) mode to extended mode?

Having the same display on both video outputs requires both monitors to use identical resolutions. Perhaps the TV is not compatible with the 1366x768 resolution of your monitor (ie blanks out). 800x600 is the Windows Safe Mode resolution IINM. During boot the computer hardware handles the display output, which is at low VGA resolutions, until Windows begins to load at which point Windows takes over control of the display device. Initial resolutions are still kept low until the display adapter drivers are loaded and then Windows switches to your selected resolution. In extended display mode each screen can have different resolution settings but the Windows Desktop only resides on the one you designate as the Primary display.

ralf_g say:
This was all tried as well as changing the frequency.  Again, it works periodically, sometimes I can simply unplug the HDMI cable from the PC and re-plug it and there will be an image on the TV.

All in all, and considering your first post where you stated that a different TV did work, it's looking like the problem is at the Phillips TV end of the HDMI cable. Looking at the available specs for both devices it really should 'just work' using any of several common display resolution settings. Your various display scenarios are all symptomatic of the TV momentarily being in sync with the Dell's output as the computer detects the video connections being broken and remade. Once the video card establishes the preset output values the Phillips loses sync and goes blank.

I see that Dell is a veritable collapsed mine for useful information. :-) Your model comes in 2 main types with a possibility of 5 different video cards or on-board video chips. Most are ATI but Intel is also used. No range of display output options are listed in this manual I'm looking at other than the maximum setting is 1366x768@ 60Hz. Makes some sense in that each video card is liable to have somewhat different capabilities, including whether they have HDMI audio or not.  

60hz is a common frequency for LCDs and TVs so most should actually be plug and play without tweaking the video settings. It appears to be the only frequency that your Phillips is compatible with anyway. If your laptop forces a different frequency (say 59Hz) at the chosen display resolution then cloning the displays might not be possible.  1360x768@60Hz is the closest resolution the TV has to match with your laptop. It might require creating a custom setting for the laptop's display if you want to clone the desktop on the TV at near maximum laptop resolution. In theory you could also use 1280x768, 1024x768, 800x600 or 640x480, all at 60Hz on both the TV and laptop in clone display mode. In extended display mode you should be able to set the TV resolution to 1920x1080 or any of the HD resolutions (720p, 1080i or 1080p).

If the laptop has ATI Catalyst control center installed you can add custom HD resolutions that aren't automatically detected. It might prevent you from setting resolutions that the DELL display is not compatible with though. Incompatible refresh rates at the various display resolutions are also indicated in CCC (eg. 60Hz is the only frequency compatible with 1360x768 on my monitor). CCC can only be installed with an AMD/ATI driver, not MS drivers or non ATI graphics chips. I don't know what display options are available for the Intel graphics chipset.

Basic Troubleshooting for Wireless Network related issues:

1.       First try and power off your modem, router and desktop/laptop. Then power on the modem, then the router, and then finally the desktop/laptop. Please do this in that order, allowing 60 seconds in between each device to enable the device to be fully booted and able to accept connections.

 

2.       Update the drivers for your wireless network adapter via Windows Update, or by using the website for either the company you bought your PC/Laptop from or the manufacturer of your networking device.

 

3.       Ensure that your router is set to broadcast it’s network name (SSID), ensure that you are connecting to the correct wireless network (SSID) and that you are using the correct WEP/WPA  passkey on both devices.

 

a.       Please use “Windows Connect Now” or “Wi-Fi Protected Setup” if available on your wireless router to ensure that the connection is setup correctly.

 

4.       If you are trying to use WPA or WPA2, please ensure that both your network adapter and wireless router support this protocol. If you are having issues detecting your wireless network when using WPA or WPA2, try configuring your router to use WEP instead and see if you can connect.

 

a.       Please note that the change between WEP/WPA/WPA2 will need to be made on both your computer and wireless router.

b.      If you are only able to connect using WEP, then your networking hardware may need a firmware upgrade. If you are not comfortable doing this yourself, please call your hardware/router manufacturer for help in doing this.

 

5.       If you are having a problem with your wireless connection after resuming from hibernate or sleep, or are having intermittent connectivity issues, try to adjust your Power Management profile to change your Wireless Adapter Settings to Maximum Performance.

 

a.       To do this click “Change Plan Settings” next to your Power Profile, and choose “Wireless Adapter Settings” and set it to Maximum Performance. You should do this with all of your power profiles. Additional guidance is available here: http://windows.microsoft.com/en-us/windows7/products/features/power-management

 

6.       Call the support line for your router manufacturer. It is likely that neither Windows nor the router are having problems, it is simply an issue of getting the configuration settings on both set up to work correctly together. To delete the User account through Command Prompt:

  1. Click Start.

  2. Type cmd, and then press CTRL+SHIFT+ENTER to open elevated command prompt(I.e. Command prompt with admin rights) or right click on the Start and select Command Prompt(Admin) from the list.

  3. If the User Account Control dialog box appears, confirm that the action it displays is what you want, and then click Continue or Yes.

  4. Type or copy paste the below command lines into it without quotes and hit Enter key:

        "net user"

This command will brings list of all User accounts that are available on your PC. Now if you're able to see the account which you want to remove, then execute the below command.

        "net user userprofile /delete"

Note : User Profile in the above command will be name of your User account. Example: If you want to delete User profile named Nalintha then the command will be net user Nalintha /delete. 

Reboot your PC and see if the User account still appears there.
