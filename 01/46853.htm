Hi Umar,

Thank you for posting your query in Microsoft Community.

Memory Management error occurs when the system memory malfunctions. Every system has at least two memory banks and the memory is installed on it. If the problem is in one of them then the total system memory decreases and the operating system gives you the blue screen error of Memory Management.

Let's try the following steps and check.

Step 1: Start the computer in safe mode  

Safe mode is a troubleshooting option for Windows that starts your computer in a limited state. Only the basic files and drivers necessary to run Windows are started. The words Safe Mode appear in the corners of your monitor to identify which Windows mode you're using.

a. When you hit the power button in Windows 10ssss, hold down the Shift key when clicking the Restart button.

b. The next screen will present you with three options. Click Troubleshoot.

c. Next, hit Advanced options followed by Windows Startup Settings from the next screen.

d. Your PC will now be rebooted to enable the advanced parameters, which include Safe Mode as well. Once it restarts, you’ll get nine options whereby the following commands apply:

  • F4 for Safe Mode
  • F5 for Safe Mode with Networking
  • F6 for Safe Mode with Command Prompt 

Step 2: Run Windows Memory Diagnostic Tool

Type Windows Memory Diagnostic in search bar and click it

In the Windows Memory Diagnostic window, select the first option which is recommended. As a result, your system will restart right away and will check for problems.

After the computer has restarted, you will see that the Windows Memory Diagnostic Tool will be checking for memory problems. This diagnosis might take several minutes. Once the process has been completed, the system will restart and will open in normal condition.

If the error occurs again, simply replace your memory modules. And that's it. In this manner, you can easily resolve the blue screen error of memory management.

if no errors are found it has nothing to do with hardware and more to do with drivers.

Right click CMD
Click Run as administrator

Type in at the prompt OR Copy and Paste these one at a time : (Hit enter after each)

netsh wlan start hostednetwork

Exit Command Prompt

Tagging to a saved file is something which can not be done for all file types by Windows OS. In general, if Microsoft has a program which has generated this file, you can put tags. Hence you can put tags for Text, Docx, Xlsx etc..
