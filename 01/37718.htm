1. Did you try to change the CMOS Battery on the computer?

2. Do you have any third-party clock applications installed on your computer?

There are three main reasons as to why your computer Clock loses its time.

1. CMOS Battery

This is the most likely scenario, especially if your computer is not brand new.

The CMOS battery sits on your computer’s motherboard and provides power to the CMOS chip. The CMOS chip stores information about the system configuration, including the date and time. If the battery goes bad, the chip starts losing information and one of the symptoms is that your computer no longer maintains its time and date.

Contact the computer manufacturer and get the CMOS Battery checked.

2. Time Zone

This is an easy to fix cause for when your computer clock loses time.

Your computer may simply be set to the wrong time zone and every time you fix the time, it resets itself to that time zone when you reboot. If the minutes are correct and only the hour is bad, that’s probably the issue you’re dealing with.

In Windows 7 you can easily fix the time zone. Right-click the system clock in your taskbar and select > Adjust date/time. Under the headline > Time Zone check whether the information is correct. If not, click the > Change time zone button and set your time zone. Don’t forget to confirm your changes..

3. Malware

This is the least pleasant and most difficult to manage scenario.

Your computer may have been hijacked by a virus that messes with your computer time. To fix it, you need to gather a few tools. First, make sure your anti-virus program is up to date with the latest virus definitions. And then perform a Full computer scan in Safe mode.

I would suggest you to follow the troubleshooting steps from the articles below and check if that helps you in fixing the issue on your computer.

Method 1

If Internet time is set and you are connected to the Internet, the computer connects to an Internet time server and synchronizes the time to the time on the server. Internet time can provide an accurate method of keeping time and is normally recommended. However, in some cases, Internet time might actually cause problems. Use the following steps to disable or enable the Internet time setting depending on what is currently selected.

1. Click Start, Control Panel, Clock, Language, and Region, and then click Date and Time.

2. Click the Date and Time tab.

3. Click Change time zone. Make sure the correct time zone is selected. Place a checkmark next to automatically adjust clock for Daylight Saving Time if it is not already selected, and then click OK.

4. Click the Internet Time tab, and then click Change settings.

If you are prompted for an Administrator password or confirmation, type the password or provide confirmation.

5. Make sure that Synchronize with an Internet time server is checked.

6. Make sure the computer is connected to the Internet and click the Update Now button.

The computer connects to an Internet server and updates the time on the computer.

7. Click OK and try using the computer for a few days. If time continues to be lost and causes problems, continue using the rest of the steps in this document.

Welcome to the Microsoft Windows Vista Communities site.

 

 

Device Manager Error (Code 10)

 

Code 10 is a default error code that Device Manager generates when the operating system cannot determine which specific error code to return after an issue occurs. If the reason for the failure is undetermined this generic message displayed is: This device cannot start. (Code 10)

 

Recommended resolutions for your scenario

 

I was unable to locate the specified memory card reader in the Windows Vista Compatibility Center (or) on Sony’s Support Website. The typical recommendation for (Code 10) is to update your drivers – as they may be corrupt, incompatible, or outdated. If you have not already done this I would recommend that you do so now.

 

However, given that the devices work upon redetect, but do not on boot up I’d like you to check for “non-present device drivers”. You will then want to uninstall both non-present and current device drivers - pertaining to this memory card reader. After you’ve done that you will want to Scan for hardware changes, and let the system redetect your memory card reader. You will then need to test the functionality of the card reader to see if the issue is resolved.

 

How to show and remove non-present device drivers…

1.     Click Start, click All Programs, click Accessories, right-click the Command Prompt, and select Run as administrator.

2.     At the command prompt type the following and press Enter after each line:

set devmgr_show_nonpresent_devices=1

start devmgmt.msc

3.     The device manager should open, and list non-present devices.

4.     Now click on View at the top, and then Show hidden devices.

5.     Locate the driver(s) associated with your device – the entry icon should appear transparent.

6.     Right-click the corresponding entry and select Uninstall.

7.     Repeat this process until all entries pertaining to said device have been removed.

8.     Once complete click Action at the top, and then Scan for hardware changes.

9.     The device(s) should now install correctly.
