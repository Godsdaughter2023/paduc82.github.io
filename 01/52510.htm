Hi Kristi,

Thank you for posting your query in Microsoft Community.

Sorry for the inconvenience caused.
 
What are the troubleshooting steps that you tried to resolve the issue?
 
Let’s try the below methods and check.

Method 1: Windows 10 has a built-in troubleshooter to check and fix issues with Update troubleshooter. I would suggest you to run this troubleshooter to check if the issue is with your Windows update.
Refer these steps:
1. Press Windows key +X, select Control panel.
1. Change the view by option on the top right to Large icons.
2. Click on troubleshooting and click on the view all option on the left panel.
3. Run the Windows update troubleshooter.
 
If the issue still persists then follow Method 2.

Method 2:
Please follow the below article on How do I reset Windows update components?
https://support.microsoft.com/en-us/kb/971058
The above link also applies to Windows 10.
 
Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry.
 
Refer to the following article for instructions on how to back up registry:
https://support.microsoft.com/en-us/kb/322756
The above link also applies to Windows 10.
 
Method 3: If the issue still persists, let us try to run the Deployment Image Servicing and Management command (DISM).

It detects incorrect manifests, cabinets or registry data, it may replace the incorrect data with corrected version available.
1. Press Windows key + X.
2. Click command prompt (Run as administrator).
3. In the Administrator: Command Prompt window, type the following commands. Press Enter key after each command:
DISM.exe /Online /Cleanup-image /Scanhealth   
DISM.exe /Online /Cleanup-image /Restorehealth
4. To close the Administrator: Command prompt window, type Exit, and then press Enter.

One other note - there are other issues with windows 10 version 1903 that you might see - Fans running faster than normal and your CPU and GPU might have increased in temps - this is not one to just let go - high temps on your CPU and GPU increase the chance of your computer shutting down or even worse costly!  When you notice this issue make sure to open your task manager and looks at the tabs - 1st one is Processes - this shows you the percentage of your CPU, Memory, Disk, Network etc. - The other is Startup tab - which shows you all the programs that start up when you turn on your computer - Here you can enable or disable programs that will cause your computer to run harder - Windows 10 version 1903 has a number that you do not need for example Cortana - an nice program but a resource hog.  Make sure you know what you are disabling and when in doubt ask :)
