wunders say:

In addition to the prior post, please notice that if you have XP Home or if you have XP Pro and Simple File Sharing is not disabled, then all network connections to your computer authenticate as the "Guest" user which, by default, does not have a password.  This allows anyone to connect and authenticate to your machine as "Guest".  If you have XP Pro and have disabled Simple File Sharing, then in addition to Guest, there is also the Built-in "Administrator" user whose password is blank by default.  Make sure both Guest and Administrator users have a password set....

 

"Setting up password for the Guest User in Windows XP"
