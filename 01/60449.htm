It is quite easy to copy registry settings, e.g. like so:
  1. Log on under the "Source" account.
  2. Open the registry editor.
  3. Select the key you wish to copy.
  4. Export it.
  5. Log on under the "Target account".
  6. Import the file you created in Step 4.
There are several problems with this approach:
  • Without having intimate knowledge of the application whose settings you wish to copy, you would have no idea which keys you must copy. My current machine which has only a moderate amount of software installed has more than 9,000 keys and 28,000 values in the HKCU hive (=ntuser.dat).
  • Some of the copied registry data might contain hard-coded user profile names, perhaps in binary form which is hard to detect.
  • You would need to know what folders and files you must to copy from the source to the target profile folder.
