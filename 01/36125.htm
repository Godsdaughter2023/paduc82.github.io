  • The PC boots into Windows RE (Recovery Environment).

There are two ways of getting to the on-screen keyboard, and as it turns out they are different keyboards. If you use the on-screen keyboard app you get the keyboard without the emoji button, if you use the keyboard button in your system tray (normally bottom right of your screen) you get the keyboard with the emoji button.

If you've upgraded to Windows 10 version 1709 (you can check by pressing the start button, typing 'winver' and pressing <Enter>) you may find the keyboard button in the system tray has disappeared (it did for me.) If this has happened right-click a blank area on your taskbar and select 'Show touch keyboard button' - you should now see the button in your system tray. Click that and you should see the on-screen keyboard that includes the emoji button.
