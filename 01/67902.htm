Type Disk CleanUp in the Search Box above the Start Button and press the ENTER key. Click on OK and after the space calculation you will see a window with a list of Files to delete. http://www.sevenforums.com/tutorials/122262-windows-7-sp1-disk-cleanup-tool.html


Not much has been written yet about the Windows Update CleanUp  function added to Disk CleanUp in October 2013. The first time it is run it can increase free disk space by something like 4 to 5gb. The free disk space recovered if you run after  subsequent monthly updates is not so dramatic but significant,


Each time you run this new feature when you shutdown the system is reconfigured both on shutdown and when the computer is next booted. This is similar to what occurs when you  install many windows updates. However, the configuration can take several hours, which is much longer than when you have only updated. It is important not to press the power button whilst configuration is in progress. If you have a laptop make sure the computer is connected to a mains electricity power supply. In addition the second time you shutdown the process occurs again, only this time after configuration there is a reference to CleanUp and when that completes the computer automatically restarts of it's own volition without any user input. When the computer has rebooted the computer is available for use.


No documentation that I have seen warns the user what to expect and my initial reaction was something had gone wrong. It had not, and should not as long as the user does not intervene.
