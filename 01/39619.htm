How can you reverse a program compatibility setting in windows 7 - Microsoft Community

b-891 say:

How can you reverse a program compatibility setting in windows 7

A program was accidentally changed to work in XP instead of windows 7. How can you reverse this.
palcouk say:
  1. Quit the program set for comaptibility
  2. From the Start menu, choose My Computer (or double-click My Computer on the Desktop).
    Windows Vista and Windows 7: From the Start menu, choose Computer.
  3. Open the C:\ drive (or whichever disk Windows is installed on).
  4. Open the Program Files folder or "Program Files (x86)" if you are using a 64-bit version of Windows Vista or Windows 7).
  5. Open that missbehaveing program folder.
  6. Right-click on that program application (the.exe file) and from the shortcut menu. Click Properties.
  7. Click the Compatibility tab.
  8. Deselect the checkbox under Compatibility mode if it's selected.
  9. Windows Vista: Click "Show Settings for All Users."
    Windows 7: Click "Change Settings for All Users."
  10. Windows Vista and Windows 7: Deselect the checkbox under Compatibility mode if it's selected and click OK.
  11. Click OK.

brekar say:

I heard back from customer support of Tom Tom.

Their newer devices have been reclassified as network devices.  They are no longer storage devices, which is why the operating system will not recognize it with a drive letter.

Per customer support, the only way to get updates to the newer network device is through the  "Tom Tom MyDrive Connect" interface.

Because no drive letter is assigned, 3rd party updates (using a drive letter) cannot be applied to these newer devices.
