aakkam22 say:

Hi Jim Hogue. I wrote this explanation of the START MENU for you to use. I hope you find it helpful and can use it to solve your problems.

Opening the START MENU

The START MENU is located in the bottom left corner of your screen. To open the START MENU, click the Windows logo.

Anatomy of the START MENU

The START MENU is split into two sides. On the left side is a list of all the programs that are installed on your computer. You can scroll through this list with your mouse wheel. 

On the right side, there are colorful squares called tiles. Each tile is a shortcut to a program. 

Using the START MENU to open programs

You can open a program by clicking it from the list on the left. You can also open a program by clicking its tile on the right side.

Searching for programs with the START MENU

The START MENU can help you find programs that are installed on your computer. When you open the START MENU, start typing the name of a program that is installed on your computer, and you will see the program in a list of results. Then you can click the program to open it.

Pinning programs to the right side of the START MENU

Pinning programs creates a tile for that program on the right side of the START MENU. You can pin programs to your START MENU for easy access. To pin a program, first you have to find it, either from the list on the left side or by searching for it. When you find the program, right-click it and select Pin to START. The next time you open the START MENU, you will see a tile for that program on the right side. 

Unpinning programs from the right side of the START MENU

You can organize the right side of the START MENU to fit your needs. If there is a tile you want to remove, right-click it and select Unpin from START. 

To organize the tiles into groups, click and drag a tile to move it next to other tiles.
