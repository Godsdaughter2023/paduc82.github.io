andre da costa say:
Is Windows 7 Professional installed on the computer now or are you attempting to use the Windows 7 Professional key to convert Starter? You must first do an Anytime Upgrade from Starter to Professional in order to use it.

If the license for Windows 7 Professional you have access to is either a retail upgrade or full version, you can use the product key and Anytime Upgrade to move from Windows 7 Starter  to Windows 7 Professional. Click Start, type Anytime Upgrade, follow the onscreen wizard, enter the Windows 7 Professional key when requested, click next and restart your computer when prompted. Within 10 mins, you should be upgraded to Windows 7 Professional and your files and settings preserved.

Please note that Anytime Upgrade does not support OEM, MSDN-Academic, Volume License (Student faculty licenses). If you are in position of any of these media, you will have to back up and do a custom clean install. If you wish to do this, please backup your Windows 7 Home Premium installation first. Certain licenses for Windows licenses distributed to students at the tertiary level are not perpetual, meaning you can't continue using the software after completing a course or graduating.

Just in case: Windows Anytime Upgrade Fail:
Download the hotfix containing a tool called CheckSUR, this tool will look into the servicing and package folders and fix any corrupted data the tool is listed under kb947821 it can be found at the following link http://support.microsoft.com/?kbid=947821
