How can I make files easier to find? - Microsoft Community

waseemulla shariff say:

How can I make files easier to find?

I keep most of my files in the Documents folder, but it is hard to find the one I am looking for. How do I make it easier? I keep most of my files in the Documents folder, but it is hard to find the one I am looking for. How do I make it easier?
john rubdy say:

There are two ways to make files within a folder easier to find, sorting and filtering by properties.

All files have properties by default, but you may want to add more.

 

To add properties to files:

You can add or modify the most common file properties, such as tags, author names, and ratings, from the Details pane at the bottom of the folder window.

1.       Click Start, and then click Documents.

2.       Change the folder view to Details.

3.       Click a file to apply properties.

4.       In the Details pane at the bottom of the folder window, click the property that you want to change, type the new property, and then click Save.

 

To rate a file by using the ratings property, click the star that represents the rating you want to apply.

 

To sort files:

1.       Click Start, and then click Documents.

2.       Change the folder view to Details.

3.        Click on the property name (such as name, Date Modified, type, or Size) at the top of the Detail ane.

To filter files:

1.       Open the folder that contains the files you want to filter.

2.        Click the arrow to the right of the heading that you want to filter by.

3.        Click the property that you want to filter by. If you want to filter by two or more properties, select the check box for each property that you want to filter by.

e.  Then check if you are able to backup

“What are permissions?” http://windows.microsoft.com/en-us/windows7/What-are-permissions
