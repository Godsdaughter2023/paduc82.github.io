Not all backup utilities accept flash drives as a valid medium, in fact maybe none, and trying to fit everything is a small space means using maximum compression which in turn takes forever..

An external hard drive has enough space such that the images and data backups don't have to be compressed to death. Backups and restores are way quicker when normal compression is used.

The System image backup is your way of getting use back onto your computer when everything has failed.  The scenario may include a failed hard drive. For this, you need a bootable CD or flash drive which has the tools to recognise the backup and also knows where the backup is located. Every backup utility worth anything has a function to create the access media, and it is one of the steps that you should take when creating the backups. There is nothing worse than having a backup but no way to get to it..

System restore is a Windows function which rolls the OS back to an earlier date.

Restoring data. This can be done via your backup program, assuming that the computer is in a workable state.

Startup repair.. The repair from a disk function ended with XP I think. Any reference to startup repair now refers to a function of the OS in which I have little faith, but this is why we make full system images. It saves having to try a startup repair multiple times before it dawns on us that it simply isn't going to work.

If the Wake Source says “Power Button,” that indicates that the power button on the PC was pressed to wake it up—an action you most likely took yourself.
If the Wake Source says something like “Device–HID-Compliant Mouse (or Keyboard),” that indicates the PC is configured for key presses and mouse movements to wake it.
If the Wake Source lists your network adapter, that indicates your PC is configured so that incoming network activity can wake it up—something that’s useful if you like having your PC go to sleep but still need it available to other network devices at times.
If the Wake Source says “Timer,” it means that a scheduled task woke the computer. The source information usually includes some indication about the task that woke the PC. For example, in the previous screenshot, I can tell that my PC was woken up in order to do a scheduled restart after an update.
You may also see something like “Wake Source: Unknown,” which is a bit more cryptic but at least it tells when the PC was awoken.
