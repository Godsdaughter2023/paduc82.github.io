When the trial is over, the trial of McAfee will start bugging you and asking you to purchase the subscription. You'll want to go into Control Panel and uninstall the Trial McAfee Lifesafe.

It's actually very bad to have more than one antivirus installed at once, they can conflict and cause big security holes, and even if not, they slow down the computer significantly because everything is trying to scan every file that gets used. My personal recommendation is to avoid McAfee and just use the Windows 8 built-in antivirus (Defender).  Windows 8 is the first one that has this, and it's quite good (same protection used by giant businesses like Dell and Microsoft).  Plus, you never have to pay for Defender, because it's built in.

So long story short:  if you want to use McAfee, just be sure to uninstall the Live Safe one and install the proper MSN one when your trial is over.

Hello Edison,

In the Fall Creators Update, the code for the touch keyboard was rewritten and reimplemented.  The full touch keyboard layout was not originally planned to be completed in the first wave of features, but we’ve heard your feedback requesting its return.  The team has been working to add the most commonly used options for the Fall Creators Update.  Currently, we’ve included a full touch keyboard option for English (United States), French (France), Italian (Italy), German (Germany), Spanish (Spain), and for English countries that share the US keyboard (like Australia and New Zealand).  In these languages, you can access the full keyboard layout by tapping the button in the top left of the touch keyboard.  We are planning on returning the full keyboard option to additional languages and regions in future Windows Updates.

If your current language is English (United Kingdom), as a temporary workaround until English (UK) is added in a future update, you can add the English (US) keyboard to access the full touch keyboard.  If adding one of the currently supported languages isn’t an option for you, you can also consider temporarily using the On-Screen Keyboard.  Type “OSK” in Search to open the On-Screen Keyboard.  You can also pin it to your taskbar for easy access.

Note:  The full touch keyboard is also available in some languages where it is the only available touch keyboard option:  Amharic Input Method, United States-Dvorak for left hand, United States-Dvorak for right hand, Yi Input Method, Cherokee Nation Phonetic, Greek Polytonic, Georgian (Old Alphabets), Devanagari – INSCRIPT, Tigrinya Input Method, Chinese Traditional Array (layout is same as US keyboard), Chinese Traditional DaYi (US keyboard layout with labels).
