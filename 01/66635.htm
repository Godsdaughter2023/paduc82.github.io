smlunatick say: Whenever you pair a Bluetooth device, it is usually paired to the last Bluetooth system  There is a very limited number of Bluetooth devices that will pair to more than one Bluetooth receiver at the same time.  And, any "multi-boot" systems, each occurrence of a Windows operating system, they all receive a unique Bluetooth system.  Each Bluetooth 'system" would require a "re-pairing" of the Bluetooth device.

rgm janssen say: Yes, this is what I experienced. But since the two occurrences of Windows  are never active at the same time, can I not give both of them the same Bluetooth identification so the device thinks there is only one?

smlunatick say: NO.  The Bluetooth "system" would probably be considered as two separate individual Bluetooth.  Each would have their own distinct list of paired Bluetooth devices.
