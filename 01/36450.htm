morc001 say:   Hey Newbie,

  It may be too late, but try this (I had the same problem):

  Navigate to %WinDir%\inf in Explorer (which is usually C:\Windows\Inf) and search for qmgr.inf. Right click and select Install. You'll need the Windows install disk, although if qmgr.dll is in %WinDir%\System32 then you can potentially point to that file. Let the install finish. At this point the service name is again Background Intell... not @%SystemRoot%..., but it won't work yet.
  Copy the Parameters key of BITS from ControlSet001 to CurrentControlSet - the INF install does not create this key for some reason, or it did not for me to say the least.
  Now open a Command Prompt, cd into %WinDir%\System32 and run regsvr32 qmgr.dll. A pop-up should tell you that the operation was successful.

  Now you should be able to start BITS from Services.

I've numbered each para so you can refer to them in any response.  I think you have viable options.

1  The factory restore would take it back to Vista anyway.  Vista is still supported [see the Windows lifecycle fact sheet - Windows Help] so you might happily keep using it until next year but after that OS vulnerabilities will not get fixed so it will rapidly become a hacker's dream.  The previous owner might have deleted the factory restore facility after upgrading to Windows 7 or might have corrupted it and it is not possible to be certain that the facility is fully functional without trying it to find out.  That would carry the risk of failure and leave you without any OS [this is not particularly likely but it is possible].

2  I assume that you are running in a standard user account rather than an admin-level user account or the name of that account would have appeared in the list you saw when trying to do the factory image restore.  In this forum, we are not allowed to advise people about that situation as it amounts to bypassing Windows user account password protection.  That's why I suggested getting a friend to help.  It might be that you are being automatically logged in to an admin account [the previous owner's name] on start-up as that facility exists but you'd need to check in Control panel, User accounts.  Does it say you are in an Admin-level user account or a Standard user account?  If it is an admin account then a couple of additional options become viable as long as you can sort out the product keys [note for later - discuss repair install procedures & built-in admin].

3  If its Windows 7 is activated [check this in the Control panel, System screen] then you should find out the product key that was used for activation.  Use the Belarc tool to retrieve this as it is suitable for these circumstances.  Just for future reference, this tool can only retrieve a product key that has been entered manually - it cannot retrieve a product key for a preinstalled Windows 7 [the tool will fail to warn you of this and will actually provide a product key - but one that is useless and cannot be used for activation].  Belarc also retrieves other product keys such as the MS Office product key if that is installed.

4  Because you do not know if the Windows 7 upgrade was done using a full licence or an upgrade licence, you should keep a note of the Vista product key as well. 

5  You'd need to search the forum for guidance on what you can do in the event of a reinstallation being needed [it would certainly seem desirable] as I do not know about upgrades.  It might be as simple as just having to put in both product keys after a clean installation of Windows 7 for all I know.  You could try searching for reinstall / clean install an upgraded system or similar.

6  If you find that you can run a clean install of Windows 7 after checking for advice in the forum then this can be very cheap.  You would only need to get a reinstallation disk for five quid not a full licence - see Amazon reinstallation disks.  I could advise you on the installation procedure itself but, as I said, I do not know how a Vista upgrade would activate a clean installation of Windows 7. 

7  A clean installation would give you the best performance that the computer is capable of but you might still find that unsatisfactory if you are used to using a newer computer.  You would also need to sort out the most appropriate drivers but that is not too difficult - see Where to get hardware drivers.
