mayank gupta10 say:

Hi Perry,

Thank you for posting your query on Microsoft Community.

As per the description, I understand that your external hard drive is not recognized by your computer.

This might have occurred due to some system conflicts.

I need some information.

  1. What is the make and model of your computer/ External hard drive?
  2. Do you get any error code or error message?

Try the methods.

Method 1: Run the Hardware and Devices troubleshooter

Open the Hardware and Devices troubleshooter

Method 2:

Step 1: I suggest you to check in device manager if the external hard drive is listed there.

  1. Connect the external hard drive to the computer.
  2. To open device manager, type devmgmt.msc in start search box and hit Enter.
  3. Check if the external hard drive is listed.
  4. In device manager, expand the hard drive listed in the device list.
  5. If the hard drive is listed, right click on it >click Properties.
  6. In General, under device status check for the message. Post back if there is any error message.

Step 2: Check if the external drive is getting detected in Disk Management.

If it the hard drive recognizes in the disc management and if there is no drive letter assigned, I would suggest you right click on the drive and assign a drive letter and check if it helps:

Refer: http://windows.microsoft.com/en-US/windows7/Change-add-or-remove-a-drive-letter

Hope this helps. Please let us know the results. We will be glad to assist you further.

davem121 say:
Hi Terravital

Click your Start Button, type winver and hit Enter

On the resulting dialog, what version of Windows 10 are you running?

If you are not running version 1809 of Windows 10, then it is normal to have a border on the active window in the same color as the accent color in Windows 10

Though the border should definitely not be that thick . ..

That border was visible in all previous versions of Windows 10, a lot of users complained about it and so Microsoft removed it in version 1809

If you are not on version 1809 of Windows 10, you should upgrade to that version to remove the borders form the active Window

Click the link below, then click 'Update Now' on that page to immediately get 1809 and bring your PC right up to date

https://www.microsoft.com/en-us/software-downlo...

Dear DaveM121,

I managed to find a solution to my problem!

The blue lines around the active browser windows were applied by a security software called Hitman.Pro to indicate that the browser was secured against internet threats.

After removing this software the blue lines disappeared.
