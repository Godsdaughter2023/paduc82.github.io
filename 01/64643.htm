Method 1: Run the Windows Update and make sure Windows is up to date

b) Type “Updates” in the search bar and click on the icon “Windows Update” from the search result.

The “Installed Updates” list shows the updates installed on the computer under different categories. For example: Windows, Office etc. However, the “View Update History” list does not show them in different categories. They show all the updates in a single list and this is by design.

 

Moreover, the “View Update History” also shows the Microsoft Defender Definition Updates which are not listed under “Installed Updates”. The reason for this is because the definition updates are merely changes to the files that Windows Defender uses to determine what's good or bad. It doesn't change the program, it just helps redefine what it should be looking for and is just like the definition updates for any other third party security program.
