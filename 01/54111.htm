Hi Matt

 

In Windows 8 and 8.1 the new Memory Manager takes care of the issues you described.

 

If you simply 'switch away' (start another app) from the current app, after 5 seconds the Memory Manager will Suspend the app. Another way to Suspend an app is to move the cursor to the top of the screen and Drag the app window to the bottom of the screen and immediately release it.

 

The app will remain in Memory. This saves the current state of the app allowing you to return to the app without losing any unsaved information.

 

You can see the current Suspended state in the Task Manager. Open Task Manager, click the Processes Tab. Click the View Tab and select Status values / Show suspended status.

 

 

 

In Windows 8.1 there are several options to completely Close an app.

 

1. Keyboard: Press the Alt+F4 keys.

 

2. Move the cursor to the top of the screen, Drag the app window to the bottom of the screen and hold it there for 2 Seconds until the app window 'Flips' before releasing it (This is new to 8.1).

 

3. Switch away from the app to another screen. Press the Ctrl+WinKey+Tab keys to display the Running Apps menu. Right click the app and select the Close option.

 

4. In Task Manager, Right click an app and select the End Task option.

 

If the new Memory Manager detects a Low Memory state on the system, it will immediately start closing suspended apps (Starting with the suspended app that has reserved the most memory) until the low memory state is resolved.
