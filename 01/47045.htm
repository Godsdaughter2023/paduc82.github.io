DDR2 OR DDR3 FOR MEMORY WITH WINDOWS 7 ???

Hello. I am planning to build a computer using Windows 7 (32bit) home preimum as OS. Looking at Dell and HP I realize they use DDR3 as their memory specification. But can I use DDR2 instead?  That is my first question. My second is: I know that installing 4 GB will only cause windows 7 to recognize 3GB. But will it "hurt" the system in any way?  or will DDR2 memory work like as if I install DDR3 memory ??? Thanks

dominicp say:

Current motherboards use either DDR2 or DDR3. The slots for the memory types are different and you must use the required type for the motherboard.

In practice, memory speed is not usually a limiting factor and it is more important to have enough memory. The speed advantage of DDR3 would only matter if you wanted something like professional video processing, where keeping an (expensive) CPU busy provided a worthwhile speed advantage.

Only 32-bit Windows is limited with a 4GB address space, so that graphics and other hardware which reserve address space leave less than 4GB available to map to the memory. Unless you have software which will not work under a 64-bit OS I recommend you go for 64-bit Windows 7.
