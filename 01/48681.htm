rizwansarwar say:

I was experiencing the same problem but with the People app which was showing in Arabic instead of English. I am using both English and Arabic as input methods and managed to fix the issue by doing the following:

  1. Go to Control Panel\Clock, Language and Region\Language
  2. Ensure that English is at the top of the list of languages
  3. Go to Advanced settings
  4. Select English in the "Override for Windows display language"
  5. Hit Save and then restart computer

People app is now displaying in English.
