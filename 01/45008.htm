Hi,

There are several ways to switch off the Connect automatically when in range option, such as through the Control Panel, Command Prompt or through Settings. For us to isolate the issue, we would like to ask the following questions:

  • When was the last time that this option worked properly?

  • Can you provide the screenshot that you have taken?

In the meantime, you can check this methods on how to switch off the Connect automatically when in range option:

Method 1: Through Settings.

  1. Open Settings.
  2. Click on Network & Internet.
  3. Click on Wi-Fi on the left pane.
  4. On the right pane, click on the network name.
  5. Toggle the Connect automatically when in range option to off.

Method 2: Through Control Panel.

  1. Press Windows key + Rto open Run
  2. Type ncpa.cpl, and then click on OK to open Network Connections window.
  3. Click on the wireless network.
  4. Click on the Wireless Properties button.
  5. Uncheck the Connect automatically when this network is in range option.
  6. Click on OK button.

Method 3: Through Command Prompt.

  1. Click on Start.
  2. Type Command Prompt.
  3. Right-click on the Command Prompt from the search results.
  4. Select Run as Administrator.
  5. Enter this command: netsh wlan show profiles
  6. Press Enter, and make note of the wireless profile name that you want to disable the automatic connection.
  7. Enter this command: netsh wlan show profile "name"
  8. Press Enter, and see if the Connection mode of this wireless network is currently set to Connect automatically or Connect manually.
  9. Enter this command: netsh wlan set profileparameter name="name" connectionmode=manual.
  10. Close the Command Prompt window.
