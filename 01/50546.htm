Method 1:
I would suggest you to run the chkdsk command from the desktop and check if it helps.
1. Click on Start and then type cmd in the search box.
2. Right click on the command prompt icon and select “Run as administrator”.
3. Type “chkdsk c: /f /r” without quotes and press enter.
 
Refer: Check a drive for errors
http://windows.microsoft.com/en-us/windows7/check-a-drive-for-errors
 
 
Method 2:
If chkdsk doesn’t help, then try to rename the partition using diskpart commands.
Follow these steps:
Open Command Prompt.
1. Type:
diskpart
2. At the DISKPART prompt, type:
list volume
Make note of the number of the simple volume whose drive letter you want to assign, change, or remove.
3. At the DISKPART prompt, type:
select volume n
Select the volume, where n is the volume's number, whose drive letter you want to assign, change, or remove.
4. At the DISKPART prompt, type one of the following:
assign letter=L
Where L is the drive letter you want to assign or change.
remove letter=L
Where L is the drive letter you want to remove.
 
Refer this link for help:
http://technet.microsoft.com/en-us/library/cc757491(v=ws.10).aspx
