lemp say:
Although Microsoft recommends 1 gigabyte (GB) RAM (32-bit Windows 7) or 2 GB RAM (64-bit Windows 7), that's not the only requirement.

Before you go too far, download and run the Windows 7 Upgrade Advisor to determine if the rest of the hardware is sufficient.

With respect to RAM, your Dell Dimension 4600 can support up to 4 GB of RAM.  It has 4 slots, so before you can know how much RAM to buy, you have to know in more detail what you have.

For example, if your current 512 MB is a single 512 MB module in one slot, you could buy a second 512 MB module for a total of 1 GB or 3 512 MB modules for a total of 4 GB.

More likely, however, is that your current 512 MB comprises 2 256 MB modules (or even 4 128 MB modules). 

The best place to start is here --> http://www.crucial.com/store/listparts.aspx?model=Dimension%204600&Cat=SSD

If you instead download and run Crucial's scanner tool, you'll get to a page similar to the one above but which also shows what you have and then customizes the offers accordingly.
