techie007l say:

    Now onto phase #1 of the installation instructions:

  1. Plug the Windows 10 TH2 (November 2015) USB flash drive in and reboot (or power on) the laptop.
  2. Press [F2] to enter BIOS setup.
  3. Set the current date to July 28, 2016, and then hit [F10], save changes and exit setup.
  4. As soon as the TOSHIBA logo appears, press/hold [F12] to open the boot menu.
  5. Select to boot from the USB flash drive.  It will probably be labeled "USB Memory".
  6. Follow the prompts to start installing Windows 10.
  7. When asked, enter the Windows 7 license key found on your laptop's COA sticker.
  8. When asked, choose [Custom install].
  9. At the drive selection screen, press [Shift]+[F10] to open a command window.  (Since this is a laptop, you may need to press [Shift]+[Fn]+[F10] if nothing happens.)
  10. Type the following command and press [Enter]: diskpart
  11. Type the following command and press [Enter]: list disk
  12. Verify that the 128GB SSD is listed as Dis the risk; the updates can always be downloaded later.
  13. When presented with the privacy options, I recommend clicking Customize and turning all of them off.  Follow the rest of the prompts to finish setting up Windows.
  14. Once on the desktop, right-click the clock (in the system tray) and click Adjust date/time.
  15. Switch Set time automatically to the Off position and close the Settings window.
  16. Connect the laptop to the Internet.
  17. Click the Windows/Start button -> Settings -> Update & security -> Check for updates.
  18. While it is checking for updates, click on Activation (in the panel on the left) and verify that Windows is activated.  If it is, you're ready for phase #2.  Go ahead and let the updates install, reboot, and make sure that everything works well, and we'll take it from there.

    The reason I hadn't pressed is because I'm not sure that it will solve the Office installation issue.  But it's certainly worth a try and would be good for other reasons.  Assuming that you have nothing important stored on the laptop now, here's how to proceed with phase #2 (the last phase):

  1. Plug the Windows 10 RS1 (August 2016) USB flash drive in and reboot (or power on) the laptop.  This is the same USB drive you used to load Windows 10 onto the Dell PC in the other thread.
  2. As soon as the TOSHIBA logo appears, press/hold [F12] to open the boot menu.
  3. Select to boot from the USB flash drive.  It will probably be labeled "USB Memory".
  4. Follow the prompts to start installing Windows 10.
  5. When asked for a license key, choose to skip.
  6. When asked what edition of Windows 10 to install, choose Windows 10 Home.
  7. When asked, choose [Custom install].
  8. At the drive selection screen, press [Shift]+[F10] to open a command window.
  9. Type the following command and press [Enter]: diskpart
  10. Type the following command and press [Enter]: list disk
  11. Verify that the 128GB SSD is listed as Disk 0.  It will probably be shown as 119GB since Window Customize and turning all of them off.  Follow the rest of the prompts to finish setting up Windows.
  12. Connect the laptop to the Internet.
  13. Click the Windows/Start button -> Settings -> Update & security -> Check for updates.
  14. While it is checking for updates, click on Activation (in the panel on the left) and verify that Windows is activated.
  15. Go back to Windows Update and wait for it to finish downloading and installing updates, and then reboot.
  16. Check for updates again and reboot again if needed.
  17. Right-click the Windows/Start button and click Device Manager.
  18. Verify that all the categories are closed with no devices showing an exclamation mark.  Expand Display adapters and verify that it's not using the Microsoft Basic Display Adapter.  If something's showing with an exclamation mark, or it's using the Microsoft Basic Display Adapter, we need to find drivers.  Take a screenshot and post it here to help give an idea of what's needed.
  19. Otherwise, you're done!
