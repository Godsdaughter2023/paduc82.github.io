lemp say:
Got a Lexmark or a Dell-badged Lexmark that you recently removed?

First, do a search for spoolsv.exe.  Normally, it should be located in C:\Windows\system32 and C:\Windows\system32\dllcache (the latter is a hidden folder)

Download, install, update, and run full scans with each of these two free programs:

MalwareBytes AntiMailware
SUPERAntiSpyware

Don't run both scans simultaneously.  Each will take some time, so start it and then go do something else for a while.  Remove anything that either program finds.  You can uninstall these scanners when done if you want.

If you found spoolsv.exe, do the step below under "Special Procedure for computers that have or had a Lexmark or Dell-badged Lexmark printer" and then start at "Clean up printers."

If you were unable to find spoolsv.exe, -- and assuming that you have XP service pack 3 and subsequent critical updates -- download a copy of spoolsv.exe from http://sdrv.ms/M2wWjt   Note that if you use Internet Explorer, you may not see the exe extension at the end of the file, but it is there, and the file when downloaded will have the correct extension.  Put a copy in C:\Windows\system32 and another copy in C:\Windows\system32\dllcache

Then do the "Special Procedure" and then the "Clean up."

Clean up printers

NOTE:  If after completing step A the print spooler still does not stay running after you have issued the "net start spooler" command, you will not be able to follow all of the steps in "Article One."  Use the Special Procedure for Lexmark printers described below and then follow Article One.  If still no joy, use the alternative method linked below.

A.  Clean out the pending print jobs

  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Type the following in the black Command Prompt window and press Enter after each line

                           net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

B.  Clean out the old printer drivers and install the latest drivers using the directions in Article One. 

Special Procedure for computers that have or had a Lexmark or Dell-badged Lexmark printer.

Open a Command Prompt window (Start > Run > cmd > OK)

Type the following in the black Command Prompt window and press Enter

      sc config spooler depend= RPCSS
      exit

note that there is no space before the =

If you want to use a Lexmark or Dell-badged Lexmark printer after doing the above, you'll have to reinstall it.

Alternative to Article One's method: http://members.shaw.ca/bsanders/CleanPrinterDrivers.htm

a. user say:

can i reset my pc without loosing window os

The whole idea of a reset is that you install a fresh copy of Windows. In a reset, your hard drive is erased. What happens next depends on what type of reset you choose.

In a 'remove everything' reset you'll only get a fresh copy of Windows 10. All your personal files, all other software and all your settings are gone. You'll have to setup Windows 10 all over again and reinstall all your software. Anything that you haven't backed up - including your personal files - is gone.

In a 'keep my files' reset you'll also get a fresh copy of Windows 10 and the personal files you stored inside your own user account will be copied and restored by Microsoft. Apps you obtained from Microsoft Store will be re-installed - but not any other software that you have - and the basic structure of user accounts is re-created. You'll still have to setup Windows from scratch and re-install any software that didn't originally come from Microsoft Store. Even though Microsoft copies and restores your personal files, it's still a good idea to make a backup copy on your own.

Clearly, a reset is not a preferred solution. Rather, it's something you do as a last resort, after everything else fails. If you're thinking of doing a reset, the better you understand what will happen - and the better you prepare - the more satisfied you'll be with the results.
