It really depends on what you mean by "saved as a Wordpad document."  I suspect that you mean that when you double-click on a file that you created and saved in Word, it opens in Wordpad instead.

If that is correctly set but you still have the same problem, then

  1. Right-click on a document saved as a *.doc file
  2. Select Open With and click on "Select Program"
  3. Find and select "Microsoft Office Word" in the list of Programs and go to step 5
  4. If Word is not in the list (unusual), click the "Browse" button and browse to and select Programs > Microsoft Office > Officenn > Winword.exe, then click "Open"
  5. Back on the first screen with the list of programs, be sure to check the box to "Always use the selected program to open this kind of file"
  6. Click OK.
If you have to use the Browse button to find Winword.exe, the directory under Microsoft Office will be "Officenn" where nn is 10, 11, 12 or some other number depending on the version of Microsoft Office you have installed.  Office 2002 (Office XP) is Office 10.  Office 2003 is Office11.  And Office 2007 is Office12.

omoikane1122 say:
Hello my name is Bradford and I'm an Independent Advisor for the Microsoft Community and I will try to resolve your problem with drive compression.

Looking into the issue there have been reports of this occurring from other users but no specific cause found (according to posts on Tenforums and Windows Central) .

Here are the steps to uncompress your drive:

First, make sure that the user you are logged in as is an administrator on that computer.

1. Press the Windows key + R
2. Type netplwiz then click OK.
If the User Accounts window appears, you are logged in as an administrator. If you're prompted to enter your credentials, you're not logged in as an administrator.

Next, open an elevated command prompt by these steps:

1. Press Windows Key +R (to open the Run Dialogue)
2. Type:
cmd
3. Then press on keyboard Ctrl+Shift+Enter to open an elevated command prompt. If you only press enter you will not get a command prompt with a high enough access level.
4. Then type the following command in and press "enter" on your keyboard after typing:
compact /compactos:never
5. If your system is currently compressed, the above command will decompress the drive.
6. If your system is not already compressed then you will receive an error message.
