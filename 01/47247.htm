tryx3 say:

If you have booted from an external USB/DVD such as an installation disk then the drive letters displayed are of no significance.  The temporary OS established by booting this way makes its own mind up about what to call each drive it finds and this has no effect on what Windows calls them when you boot normally afterwards.

As you work through the restore system image procedure, you will see on the System recovery options dialog which drive it is calling the one Windows is installed on.  Note what it says there so you can make sure it restores to the same place in the final confirmation dialog, the Re-image your computer dialog.  Once you finish & restart, it will be called whatever it has always been called by Windows.

Denis

tryx3 say:

I have just managed to get a Windows 10 computer to show abnormal drive letters using the same procedure as you - booting through Settings, Update, Recovery, Advanced startup - Restart now and then moving through Troubleshoot, Advanced options, Command prompt {as if I was checking it before going into System image recovery}.

I had always thought that abnormal drive assignments were confined to booting externally but I am clearly wrong.  So the comments I made before do apply generally. 

Since the System recovery options dialog does not appear using the procedure you are in then you could check its drive letter assignments using the Command prompt instead.  You can check which drive letters refer to which real drives by working through dir c:, dir d:, ... {but not dir x: as that is the drive letter used by the temporary OS established in this Windows recovery environment}.  But checking all this is not essential because the final confirmation dialog, the Re-image your computer dialog, will show the correct drive letters i.e. those that apply normally within Windows itself.

You can proceed and, provided that you do not select any of the format / repartition options you are offered along the way, the system image will restore to the correct real drive irrespective of what it is calling it at the moment. 

Denis
