davem121 say:
Hi Yefan, you can prevent the Search Indexer running on battery in Windows 10, but it will require a registry fix:

Click your Start Button, type regedit and hit Enter

In the left pane, navigate to:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

Check to see if you have a sub key of Windows in the left pane named Windows Search

If so, click that

If not, right click Windows in the left pane and choose New - Key

Name that Windows Search

Then click that new key

In the right hand pane, right click and choose New - 32-bit DWORD value

Name that PreventIndexOnBattery

Then double click that new DWORD and set its value to 1 and click OK

That's It - restart your PC for that change to take effect . ..
