Error 80070002 occurs when you try to install Games for Windows Marketplace

Problem

You see the following error code and message when you try to install Games for Windows Marketplace:

Sorry, Xbox Live profiles can't be downloaded right now. Try again later.

Error code: 80070002

This may mean that there's a problem with the Xliveredist.msi file you downloaded to set up the Games for Windows Marketplace application.

Solutions

To resolve this problem, go to the folder where you saved the Xliveredist.msi file and delete the file. Then, redownload the file and try the installation again.

If you keep getting the error, contact Xbox Support.
