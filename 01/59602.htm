Here you go:

  1. Log on under an admin account.
  2. Create a desktop shortcut for cmd.exe.
  3. Right-click this shortcut, then left-click "Run as administrator".
  4. Accept the challenge.
  5. Type these commands and press Enter after each:
    cd    /d   %SystemRoot%
    cd  System32
    takeown  /f  cmd.exe
    cacls  cmd.exe  /g  everyone:F
    (anwer "Y" when asked "Are you sure?")
    ren  cmd.exe   cmd1.exe

And here is how to reverse the above steps:

  1. Log on under an admin account.
  2. Create a desktop shortcut for cmd1.exe.
  3. Right-click this shortcut, then left-click "Run as administrator".
  4. Accept the challenge.
  5. Type these commands and press Enter after each:
    cd    /d   %SystemRoot%
    cd  System32
    ren  cmd1.exe   cmd.exe

If the pop-up occurs frequently then you could leave your own command prompt open for an hour or two. In this case you would only need the line in italics to reverse your action.
