A "Cyclic Redundancy Check" (CRC) error means that there was an error in the transmission of data.  If you get such an error when attempting to read a physical CD, it generally means that the CD is defective.  You can check to see if the problem is the CD itself or your computer's hardware by attempting to read other CDs, preferably CDs that have been commercially produced.  If other CDs can be successfully read, then the problem almost certainly is with your Print Perfect Deluxe CD

If the CD was commercially produced (rather than created on a home computer), there may be a scratch or dirt on the CD and you may be able to fix this by carefully cleaning the disc.  Either ask for directions or use your favorite search engine before touching the CD.

Print Perfect Deluxe is a rather ancient program, originally released for Windows 98 and said to work for versions of Windows through Windows XP.  It may or may not be compatible with Windows 7.  If your CD is indeed bad and you nevertheless want to take a chance on the program working on Windows 7, PPD is available very inexpensively from Amazon:  https://www.amazon.com/gp/offer-listing/B000ANCUMS/ref=dp_olp_all_mbc?ie=UTF8&condition=all

Get-AppXPackage -Name Microsoft.Windows.Cortana | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

Uninstall and reinstall the Printer (Credit to Andre Da Costa, MVP for providing steps)
A. Uninstall Printer
Click on Windows Key+X > Apps and Features > Scroll down and click Programs and Features > Select your printer > Select Uninstall
Type Printer in Windows Search box > Printers & Scanners > Select your printer > Remove device

If you are using Bluetooth / Wirelss printer > Select Add a Bluetooth, wireless or network discoverable printer > Choose the printer > Select your Printer and follow on screen directions
If you are using wired printer > Select Add a local printer or network printer with manual settings > Select Use an existing port > Select your Printer and follow on screen directions
