lemp say:
The "read only" box on folders does not do what it says; ignore it.

A folder will not delete if it contains a locked file.  You'll have to unlock and delete the file first.

"Some files" will not delete because they are locked -- i.e., they are in use by some application.  There is a utility called "unlocker" that used to be recommended to fix this situation, but the download now includes adware that may be flagged by your antivirus/antimalware application as dangerous.

An "empty" folder that will not delete may be fixed if you run chkdisk.  Click Start > Run and type the following, then click OK

    chkdisk /f

A window will open informing you that chkdisk /f can not be run while Windows is active and offering to schedule chkdisk for the next reboot.  Answer Yes, and then reboot.  After chkdisk completes, delete the folder.

Files that appear to be in use may be deleted by one or more of the following techniques:

A.  Delete from a Command Prompt window.  You need to know the full path name of the file in question.  For example, if you are attempting to delete a file named test.txt that is in a folder named Non Empty Folder on your Desktop and your user name is NR16020, the full path name is C:\Documents and Settings\NR16020\Desktop\Non Empty Folder\\test.txt
  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Use the del command for files and the rd command for folders.  If the full path name includes spaces, enclose it with double spaces.  Using the example:
    del "C:\Documents and Settings\NR16020\Desktop\Non Empty Folder\\test.txt"
    rd "C:\Documents and Settings\NR16020\Desktop\Non Empty Folder"
B.  Delete from Safe Mode.  To get to Safe Mode, reboot the computer and repeatedly tap the F8 key immediately after the power-on sequence is completed.  If you see the Windows logo, you waited too long and will have to try again.

C.  Delete from a Command Prompt with Windows Explorer closed.  If the procedure in A does not work because the file is locked, do the following:
  • Close all running applications
  • Open Task Manager
  • Open a Command Prompt window
  • In the Task Manager Processes tab, right-click on explorer.exe and select End Process
  • Go to the Command Prompt window and follow the steps listed in A above
  • In Task Manager, click File > New task (Run ...), type explorer.exe and click OK

lemp say:

I believe what you're referring to is "autoscroll."  Is this (or something similar) the icon you're referring to:

If you have a mouse with a center "scroll wheel," depressing the wheel rather than rotating it will enable "autoscroll."  When this icon is present, the page scrolls (moves up and down) to follow the mouse movement.  This is similar, but not the same as, the "Scroll Lock" button on the keyboard (if your keyboard even has this key).

This can also happen with laptop touchpads.

You can turn this off by depressing the scroll wheel again.

If you want to turn this "feature" off permanently, the first place to look is Control Panel > Mouse.

If you're lucky, you'll find a setting in the Mouse or Touchpad properties, either under "Scrolling" or "Wheel" that will allow you to disable auto scroll.

If your Mouse driver doesn't include this setting, you may be able to turn it off in your browser.  Firefox, for example, has a "Use autoscrolling" checkbox in Options > Advanced.  I don't think that there is a comparable setting in Internet Explorer (and I have no idea about Edge, which is what you are probably using in Windows 10).

There are also third-party utilities that may give you more control over your mouse:  https://www.raymond.cc/blog/changing-and-customizing-your-mouse-buttons-actions/

Although I haven't used any of those utilities, the User Guide for X-Mouse Button Control (http://vps2.highrez.co.uk/downloads/X-Mouse%20Button%20Control%20User%20Guide.pdf) seems to indicate that it will be capable of changing what happens when you depress the scroll wheel.
