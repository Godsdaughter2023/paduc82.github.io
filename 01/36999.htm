cmdrkeene say:

Yes the Lock icon means that the folder is private only for you to see.  To keep from cluttering up the entire folder with many lock icons, the lock only shows up if it is the first level folder to be private for yourself.  So if the parent folder had a lock, the folder below would not show the lock anymore because it's already known and assumed to be private.

So short answer: go up a folder, and set the parent folder to share with > no one, and the lock will move up to that higher level.

If it was the Manufacturer's Recovery partition, this is the advice for that:

Your Recovery Partition was imaged on your Hard Drive during the Manufacture of your Computer by the Manufacturer for these reasons.

 

1.To reinstall Windows from in case of a System crash, by pressing particular Key or Keys at Boot.

F10 or F11 or Alt + F10 are a couple of sequences different Manufacturers use.

You will have to ask your manufacturer for the correct sequence.

 

2. To make Recovery Disks from in case your Hard Drive fails, so that you can then reinstall the Operating System on to a new Hard Drive.

Also ask them how to make the Disks.

 

Your D: Recovery Drive is not to be used for backups, defragged, etc.

Backups should be kept on an External Hard Drive.

 

To fix any problems that have occurred with the D: Recovery Partition, you will have to contact your Computer Manufacturer to remove whatever has been added to it.
