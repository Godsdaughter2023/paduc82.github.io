malkeleah say: You cloned the drives incorrectly and you'll need to redo the cloning procedure. You can't change the new drive's Windows install any other way. The usual cloning procedure is to have the target drive connected (Slave if IDE, SATA 1 if SATA), boot with the cloning CD, and clone your original drive to the replacement. Shut down the computer. Then REMOVE THE ORIGINAL DRIVE AND PUT THE NEW ONE IN ITS PLACE (Master if IDE, SATA 0 if SATA). This is the part you probably forgot. Now boot the system and the new (single) drive will be C:\ and everything will work. After you have done that, if you want to use the old drive for storage you can install it in the secondary spot (Slave if IDE, SATA 1 if SATA). Format it from your newly running Windows.

You may have dragged it off the screen by accident, in which case it's sort of "minimized" and needs to be dragged back. Put your mouse right up to the side of the screen where the taskbar used to be (the very edge). Click and drag toward the center of the screen.
