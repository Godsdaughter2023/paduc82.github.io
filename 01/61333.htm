Importing of photos and videos is handled by an App. In Windows 10, the default application for importing from a device is the Photos application. While the download is gone from the Microsoft servers, you can find archived copies of the Windows Essentials 2012 offline installer. I'd recommend installing that and then switching your default import behavior so it imports using Photo Gallery instead of Photos.

Basically you need to change the Autoplay settings for your camera storage and media cards to "Import using Photo Gallery".

ralf_g say:

Depending on how popular a program was its fans may have created patches allowing the software to be run in newer versions of Windows. A web search should turn that up if such a thing exists for your program.

The most common way to deal with this kind of situation though is to use  virtualization software to create a virtual computer within Windows and install a copy of Windows 98 in that. There are free versions of VMware Player and Virtual Box, possibly others, which you can install in Windows 10 Home edition (Pro edition includes Hyper-V). It may have changed but the last time I used Virtual Box it had poor or zero support for MSDos based Windows versions (win3.x through 98) so my preference is VMware. You'd have to source a copy of Windows 98 (or whatever other OS) separately.

A couple of things to note, first being that the virtual computer will use a fixed configuration of virtual hardware components. Your program's performance (thinking games) might not be as high as you're used to. The second thing is that a lot of older programs were written for displays that were only 640x480 pixels, or 800x600 pixels, so that is the size that they would typically appear on your screen however large it is. If the program is able to run in windowed mode then you could stretch the virtual PC's display to a larger size on your screen. Some programs will only run at a fixed display size, which could leave you limited to running the program in a small 640x480 program window.

ruizell rey say:


Microsoft has identified a known issue that may swap the battery order in the battery status fly out after installing the Creators Update.  We are currently working on resolving this in a future version of Windows 10.

 

Some customers have reported they were successfully able to set the battery order in the battery status display by uninstalling the batteries with Device Manager and then reinstalling them in the desired order.  To do this:

 

  1. Open Device Manager.  (Type Device Manager in Cortana, and then select the Control Panel item in the search results.)
  2. Expand Batteries. 
  3. You should see one or more items that say Microsoft ACPI-Compliant Control Method Battery (or something similar).  Right-click each one and select Uninstall.
  4. Physically disconnect the secondary battery and make sure only the primary battery that you want to be displayed as “Battery #1” is attached.  Then, in Device Manager, select Action > Scan for hardware changes.
Scan for hardware changes.  The secondary battery should now be displayed as Battery 2.
