When it was newly installed my router picked up my neighbour's routers that also had WPS enabled. A bit disconcerting to see the external network connections show up on my LAN but there was no cross communication as there wasn't any shared security key either way. The router simply reported the existence of potential WPS network connections.

 

 Disabling WPS shouldn't be a problem, especially if you don't actually use it to connect any devices to your network (Verizon freeloaders notwithstanding).

 

Since WPS was apparently cracked a few years ago it is probably better if you do disable it in any case.

DelegateExecute will only work on Windows 7 and WMP 12. As you have WMP 11, and WMP 11 is not available on Windows 7, what version of Windows are you running then? XP or Vista? (in both cases, this is the wrong forum)

Copy-paste everything between the dashes below into Notepad and save it as FLVEnqueue.reg . Then double-click this new file to add it to the registry. After that, you may need to re-associate FLV files with WMP, through Control Panel -  Default Programs .

-----------------------

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\.flv]
@="flvfile"

[HKEY_CLASSES_ROOT\flvfile\shell\Enqueue]
@="&Add to Windows Media Player list"
"MUIVerb"=hex(2):40,00,25,00,53,00,79,00,73,00,74,00,65,00,6d,00,52,00,6f,00,\
  6f,00,74,00,25,00,5c,00,73,00,79,00,73,00,74,00,65,00,6d,00,33,00,32,00,5c,\
  00,75,00,6e,00,72,00,65,00,67,00,6d,00,70,00,32,00,2e,00,65,00,78,00,65,00,\
  2c,00,2d,00,39,00,38,00,30,00,30,00,00,00

[HKEY_CLASSES_ROOT\flvfile\shell\Enqueue\command]
"DelegateExecute"="{45597c98-80f6-4549-84ff-752cf55e2d29}"
