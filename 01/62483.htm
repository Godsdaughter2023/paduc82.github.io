cmdrkeene say:

File History is not a total mirror, it's a backup of your "files" only (documents/pictures/videos).  It doesn't include program files. If you only relied on File History and your hard drive crashed, you would need to re-install your programs again (by downloading on the web or using the original CD disc).   If you're familiar with it, File History on Microsoft Windows is nearly identical to "time machine" feature on the Apple Mac.

The "Windows Backup" is a very old technology that I wouldn't recommend using. File History is not only more reliable, but it also doesn't require the legacy backup program to extract your data, meaning you could restore from your backups more easily regardless of your operating system you're using at the moment.

System Image Backup is the only option that makes a "exact copy" of the drive, but it can't be updated. You'll need to make a new, complete, totally separate copy (image) every time you want a new full image backup.

One suggestion is to make a system image after you've installed all your favorite programs (Office, favorite web browser, whatever else), but without any "personal files" (actual documents, pictures, etc.).  That way if you ever get a new hard drive, you could restore the image and instantly have all your programs "pre installed" as it were.  Sure they might need some updates since that image could be a little older, but it saves the hassle of installing them again.  As for the files, you can copy those back from File History or (if using online storage) just like OneDrive/DropBox/Whatever sync the latest copy down again.
