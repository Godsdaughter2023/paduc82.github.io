So how do I determine which ones that I absolutely MUST ALLOW to run on startup? 

Every computer is different - your computer has different programs then others. Keep your antivirus and other programs you know you need set to auto start. If you stop one you end up needing, you can reset it to start.


Reduce Start Up Programs:
A) Click Start, type MSCONFIG in the Start Search box, and then press Enter.
OK User Account Control permission and enter password if any

C) Next change tabs by clicking the Startup tab
D) Uncheck the programs you don't want to start when your computer boots (do not uncheck your antivirus program).
E) Click OK, then restart the computer.

lemp say:
On 6/22/2010 7:36 PM, augie56 wrote:
> forgot password for my desktop can anything be done
>
 
Yes.
 
Unless you were using the built-in "Administrator" account for your
everyday usage, log into the Administrator account, go to Control Panel
> User Accounts and change the password for your user account. This
time, write it down someplace.
 
To get to the built-in Administrator account:
 
- In XP Home, you must re-start in Safe Mode. Reboot the computer and
tap the F8 key *before* you see the Windows startup screen. Pick "safe
mode" from the menu. Click on the icon for the Administrator account.
The default password is blank, unless you changed it.
 
- In XP Pro or Media Center Edition, you can either start in Safe Mode
as above, or press CTRL-ALT-DEL twice at the Welcome screen to get the
logon Window. Select Administrator from the drop-down list. Leave the
password blank.
 
If you have been using the Administrator account on a daily basis and
that's the password that you forgot (or if you set a non-blank password
for the Administrator account and have forgotten that one as well),
there are ways to reset the Administrator password, but Microsoft will
not permit us to discuss them in this forum. Google is your friend.
 
--
Lem
