I've encountered this problem and it seems that a way to fix it is to:

  1. Open Registry Editor by pressing Start or Windows + R (Run command) and writing regedit.
  2. In Registry Editor drill down to HKEY_CURRENT_USER/SOFTWARE/Policies/Microsoft/Windows/CurrentVersion/PushNotifications
  3. Set value on NoToastApplicationNotification to 0 by double clicking this entry and writing 0 to the data field and then confirming change with OK.
  4. Rebooting your machine or logging out of your system.

danno mck. say:

[quote user="aMalfunction"]I know that I could go up to 16GB but I'm greedy and an American so, more is always better...[/quote]

If you let the console set up the 16 GB on a 32 GB flash drive, it puts the ~512 MB reserved space outside the 16 GB. So you will get more usable Xbox 360 storage than if you just use a 16GB flash drive.

 

(I don't know whether you get back the GB lost to the formatting difference between 16,000,000,000 bytes and 16 x 1,024 x 1,024 x 1,024 on a "16 GB" flash drive, though. That is, a "16 GB" flash drive will be 16 billion bytes, but since the console uses the 1,024 counting you only get 14.9 GB usable and then it reserves ~512 MB of that for the system.)
