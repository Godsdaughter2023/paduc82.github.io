The "Switch between Windows" button on the Quick Launch toolbar is enabled by a file called "Window Switcher.lnk" in your Quick Launch directory. If this file is missing, the icon will not appear in the Quick Launch Toolbar.

You are probably missing this file, and need to replace it.

You can get a copy of the original file here:

 "C:\Users\Default\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch\Window Switcher.lnk"

This location is normally hidden in Explorer (you won't see it).

To put it on your own Quick launch toolbar, go to a command prompt and enter these commands (replace "<user>" with your real user name, of course! :-) -

C:\>cd C:\Users\<user>\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch

C:\Users\<user>\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch>copy "C:\Users\Default\AppData\Roaming\Microsoft\Internet Explorer\Quick Launch\Window Switcher.lnk"

You should see a message "1 file(s) copied." when the file is successfully copied. Then type "exit" to leave the command prompt.

The "Switch between Windows" icon should now re-appear in your Quick launch menu.
