Hi, Brent,

 

Memory Limits for Windows Releases (Windows)

When your computer's physically installed random-access memory (RAM) is running low, Windows adds available memory by using a paging file, generally known as virtual memory, on the hard disk to simulate physical RAM. By default, the virtual memory paging file that is created during installation is 1.5 times the physical RAM on your computer. Therefore, a computer that has 1GB of installed RAM will have 1.5GB of virtual memory.
http://support.microsoft.com/kb/308417

 

 

 

Edit:  "In computing, commit charge is a term used in Microsoft Windows operating systems to describe the total amount of pageable virtual address space for which no backing store is assigned other than the pagefile. On systems with a pagefile, it may be thought of as the maximum potential pagefile usage. On systems with no pagefile, it is still counted, but all such virtual address space must remain in physical memory (RAM) at all times."

http://en.wikipedia.org/wiki/Commit_charge

How to change account to administrator? - Microsoft Community

brian knaggs say:

How to change account to administrator?

Original Title: *** Email address is removed for privacy ***

 

I recently got a 2002 Dell Inspiron laptop but I only have a "limited account" and I need to change that to an "administrator" account in order to download programs. I am the only account on the computer. How do I change my account type without an administrator????

lemp say:
First of all, anyone who acquires a used computer is well-advised to do a complete re-installation of Windows, using either the mechanism provided by the manufacturer (Dell, in you case) to restore the computer to its "fresh from the factory" state or by using a genuine Windows installation CD.  You simply don't know what's lurking on that hard drive.

If you don't do that, you can (maybe) log into the built-in account named Administrator.  This account cannot be deleted.  If you have XP Home, restart the computer in Safe Mode (tap F8 starting immediately after powering on the computer; if you see the Windows logo, you waited too long to start tapping and will have to try again).  When the computer boots into Safe Mode, click on the icon for "Administrator"  The password -- unless the previous owner changed it -- is blank (i.e., no password).  If the password has changed, Microsoft's policy forbids us from telling you how to get around that situation, although it is possible.

If you have XP Pro (or Media Center), you can either use the Safe Mode approach as above or, while at the Welcome screen with no user logged on, press CTRL+ALT+DEL twice.  The "classic" logon screen will appear.  Enter Administrator as the user and leave the password blank.

Once logged into the Administrator account, go to Control Panel > User Accounts and create a new user with "Computer administrator" privileges.  Do NOT use the Administrator account for day to day purposes.

But do give serious consideration to reinstalling Windows.
