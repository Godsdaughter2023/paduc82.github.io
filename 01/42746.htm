tricky300 say:

XP doesn't have native support for SATA disks in AHCI or RAID modes.

 

To install XP you have to do one of three things.

  1. Change BIOS setting from AHCI or RAID to IDE (IDE might be called ATA)
  2. Provide AHCI drivers on floppy disk via the F6 prompt (just after CD boots)
  3. Slipstream AHCI/RAID drivers to create a new Windows installation boot CD with AHCI/RAID supported (for example by using nLite - http://www.nliteos.com/)

If what you describe as the XP installation CD is a manufacturer's recovery CD or CD set (as opposed to a hologrammed Microsoft XP installation CD) then your only option is probably the first of the three above.

  4. Highlight the hard drive and hit Y on your controller.
  5. Select the option to clear the update cache and perform maintenance on the memory unit.
  6. Restart your xbox 360 console.
