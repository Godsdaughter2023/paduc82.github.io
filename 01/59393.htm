jimwynne say:

If you haven't assigned a drive letter to an external device, Windows will use the next available letter.  This means that if you connected the device yesterday and Windows assigned the drive letter "G," and today you have another external device connected (which was assigned "G"), Windows will assign another letter and the Indexing service will recognize the drive as having not been indexed and start over. 

Connect the drive you want to index.  From the desktop, do Win key + x and select Drive Manager, then right-click on the drive and select "Change drive letter and path."  You will then be able to assign a drive letter (use something out past "G" in the alphabet).   This should mean that the drive will be recognized as the drive letter you chose each time it's connected, and the indexer should pick up where it left off the last time.

Hi,

Windows uses the index to perform very fast searches of the most common types of files on your computer. Instead of looking through your entire hard disk for a file name or property, Windows scans the index.

 

If you're searching in locations that aren't indexed, the search might be slow because Windows must inspect each file in those locations during the search. You can add those locations to the index to speed up future searches.

 

Indexer cannot pick up from where it left off, you need to keep the external hard drive connected to your computer until it finishes the indexing. Refer this steps to rebuild the indexing.

  1. Move the mouse pointer to the lower-left corner of the screen until you see the Start tile appear.
  2. Right-click the Start tile and select Control Panel.
  3. Select Small icons or Large icons on the View by menu in the upper-right corner of the Control Panel window.
  4. Click Indexing Options.
  5. Click the Advanced button at the bottom of the Indexing Options window. This displays a new window titled Advanced Options.
  6. Click the Index Settings tab if it isn’t selected already.
  7. Click the Rebuild button to delete and rebuild the file index.
  8. Click OK to confirm.

 

Hope this helps. If you need further assistance with Windows, let us know and will be glad to help.
