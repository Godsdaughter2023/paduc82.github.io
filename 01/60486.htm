I've come across a temp fix which is working for now. Rather than removing the drivers associated with the GT 750M NVIDIA graphics card (as in number 3 above), disabling the card outright from device manager seems to solve the problem.

It seems the Windows updates don't allow you to choose which updates are allowed (or I haven't found a way to do so yet), but this means that simply removing the driver for the card is pointless as Windows update then automatically reinstalls a faulty driver. Hopefully this helps someone else who is having any similar problems.
