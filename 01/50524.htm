jeffreylau m h say:

WindowsSearch.exe. - Unable to Locate component.

Detailed description:  This application failed to start because uncdms.dll was not found. Re-installing the application may fix this problem. 

bikesh t say:

Hi,

  • When do you receive this error message?
  • What were the changes made before the issue started?

The uncdms.dll file is associated with the Windows Desktop Search Application that is used to find and preview different types of files, such as documents, email messages, photos, and music files on a computer.

By default, the uncdms.dll file is found in C:\Windows\System32 folder. The file is most commonly found in 143,872 bytes size.

Uncdms.dll Errors

When you start your system, you may encounter the following uncdms.dll errors:

“Windowssearch.exe – cannot find the component. This application has failed to start because uncdms.dll was not found. Reinstalling the application may fix the problem.”

“The file uncdms.dll is missing.”

Common Causes of Uncdms.dll Errors

These Uncdms.dll errors may occur when the uncdms.dll file is removed or corrupted. Some of the common causes of uncdms.dll missing errors are:

Missing uncdms.dll file

Malware corruption

Registry issues

Resolution Methods

Depending on the cause of the uncdms.dll error, you may use one of the following methods to fix the issue on your computer:

Method 1: Uninstall and reinstall Windows Desktop Search

The uncdms.dll file is installed on your PC when you download the Windows Desktop Search application. Problems with files of this application may cause a uncdms.dll missing error to appear on your system.

To uninstall the Windows Desktop Search from your computer, you need to complete the following two tasks:

Step1: Remove Windows Desktop Search from startup sequence

To achieve this, follow the steps below:

a. Click the Start menu and then click Run.

b. In the Open box, type the msconfig command and then press Enter.

c. In the System Configuration Utility dialog box, click the Startup tab.

d. Locate and clear the checkbox for any option that is related to Windows Desktop Search or uncdms.dll.

e. Restart your computer.

Step 2: Uninstall the Windows Desktop Search Application

To uninstall the Windows Desktop Search using the Add or Remove Programs utility, perform the following steps:

a. Click the Start menu and select the Control Panel utility.

b. Double-click the Add or Remove Programs icon.

c. In the Add or Remove Programs window, locate and select the Windows Desktop Search.

d. Next, press the Remove button.

e. Restart your computer.

1. Windows 7 comes with *no* e-mail program. If your Windows 7 computer came with an e-mail program, it was because your computer manufacturer bundled one with it, not because it comes with Windows 7.


2. Almost certainly what you are trying to use is *not* Windows Mail. Windows Mail supposedly doesn't run under Windows 7, but you *can* get it from a Windows Vista machine and use it under Windows 7. No manufacturer bundles Windows Mail with Windows 7.


3. Perhaps what you are running is Windows *Live* Mail  (which is a different program from Windows Mail, and is often bundled with Windows 7 computers). Is that correct? If that's what it is, each to his own, but that's a program I don't like at all.


4. You are certainly not stuck with whatever e-mail program you are using, and do *not* have to change your e-mail address to use another one.


5. The last version of Windows that Outlook Express ran under was Windows XP. And there is no way to run it under Windows 7. However essentially  Windows Mail is just a later version of Outlook Express with a different name, and Windows Live Mail is a still later version with a still different name.


6. There are many e-mail programs you can download on the internet and use under Windows 7. Some are free and some can be nought (usually inexpensively). Common examples are Thunderbird, Forte Agent, Eudora, and Pegasus. You can read about many of them here: http://en.wikipedia.org/wiki/Comparison_of_email_clients


7. You can also use Microsoft Outlook (can be bought by itself or aas part of Microsoft Office). I do.


8. What e-mail program you use is entirely up to you. As I said, there are many choices, and you should choose the one you like best ( not the one that someone else likes best and recommends to you). You might even want to try several before making up your mind as to which one you want.


9. If you can't get e-mail messages with your e-mail program, you have it set up incorrectly. Call your ISP and ask them what the correct settings are.
