b.w.larson say:

I finally found the problem.  I had (long ago) used Tuneup Utilities 2013 to remove the arrows from the shortcut icons, using the "windows styler" feature - or something like that.  For some reason I still can't figure out, TuneUp Utilities was uninstalled from my pc, and left a dead-end in my registry, with the shortcut icons not knowing what to do.

I found the fix on a "sevenforums" page regarding shortcut arrow issues (change/remove/restore)

I went into regedit, navigated to:

HKEY_LOCAL_MACHINE

SOFTWARE

Microsoft

Windows

CurrentVersion

explorer

Shell Icons

Under the value "29", the data showed "c:\Utilities\Tuneup Utilities 2013\...", pointing to the file that Tuneup Utilities used to "erase" the shortcut arrows on the icons.  Being this file was uninstalled, it confused the system.

I replaced that data (c:\Utilities...) with the data string that was on the sevenforums page above:

%SystemRoot%\\System32\\shell32.dll,-50

I rebooted and, voila, everything was back to normal!!
