You absolutely do not want to have only one user account. ¡firedog say:

... how to back up windows live mail to an external hard drive? 

One easy way is to use File History. To keep a backup of your mail messages, include the mail folders in File History. You can do this in several ways, but first, locate your store folder: in Windows Live Mail, when viewing your message list press Ctrl-Shift-O for Options. On the Advanced tab, click Maintenance and then Store folder. The path is shown in the box.

  • In File Explorer (Windows key  + E), ensure that both Hidden items are selected on the View tab.
     
  • Navigate to the store folder, then right-click on it and select and posts, the message database index, account settings, calendars, feeds messages and proofing dictionaries.
     
  • Alternatively, to back up only mail messages, create a new library and add only the folders containing mail messages. These are (a) folders containing a folder called Inbox and (b) the folder called Storage folders. 
     
  • With Details selected, you can adjust the view of the files in a library to make it possible to see only the important details of each message, e.g. From, To, Subject and Date. Right-click on a column heading and select More to choose the columns to show. You can drag column headings right and left to change their order. Enabling Preview pane will show you the content of the message selected, although some images may be obscured for security reasons.
     
    Once the view is set up, File Explorer will remember it:

  • Otherwise, in the File History options, Add the individual email folders you want to back up.

Messages included in a library will be indexed by the Windows Search service by default, so search should be able to find any message From a particular sender, To a particular recipient, by Date or by keyword in the Subject or body of the message.

Of course, it rather depends on what you want to use the backup for, e.g. to restore user data after a catastrophic failure, to maintain an email archive outside the program itself or to transfer user data to a different installation. Different methods may be appropriate in each case.
