peter.r. say:

Hi Paul,

I hope the following guide will help - sorry for the delay.

The Manual for the Canon PIXMA MP600 scanner reveals that Windows Live Mail is not in the list of supported e-mail programs. That is why it is not listed by default in MP Navigator's list of Mail Programs. Nevertheless, given that Windows Live Mail is a Simple MAPI client, which is all that a Canon scanner requires, Windows Live Mail can work correctly with Canon's scanning software MP Navigator.

We can modify an .ini configuration file to add Windows Live Mail to the list of compatible Mail Programs that work with MP Navigator. This enables the Canon scanner's E-MAIL button (if it has one), and MP Navigator's E-mail functions to work correctly with Windows Live Mail (a Windows Live Mail compose window will open with the resultant file attached - although MAPI 'attached' image files are treated differently in WLMail 2011 & 2012 (they are not actually 'attached').

Below, I will outline in detail the method for adding Windows Live Mail to the compatible Mail Programs list in Canon's scanning program MP Navigator. For the benefit of others, I will generalise the method somewhat, as it can be applied to any Canon scanner using the MP Navigator scanning program.

Basically, with a PIXMA MP600 scanner, it involves adding the property Mailer5=Windows Live Mail to the [Application] section of Device sub-folder of

1. MP Navigator has one or more .ini configuration files named according to its compatible scanner models, at the following location:

C:\Program Files (x86)\Canon\MP Navigator X.X\Device (In 32-bit Systems, it will be located in C:\Program Files\Canon\MP Navigator X.X\Device) - where X.X is the version number of MP Navigator - in this example it will be MP Navigator EX 3.0

So navigate to that location, then copy and paste to your Desktop (or some other convenient location), the .ini file that is named according to your model of scanner - in this example it will be mp600.ini

Note that by default in Windows, the option to show file extensions is disabled in Windows Explorer, so if you cannot see the .ini file extension, please open Windows Explorer (i.e. open any folder) and enable viewing of file extensions:

Windows 7

If necessary, show the Menu bar: Organize->Layout->Menu bar

On the Menu Bar: Tools->Folder options...->View [tab]->Advanced settings: [section]->[ ] Hide extensions for known file types [uncheck].

Windows 8.1

If necessary, show the Ribbon: Ctrl+F1

On the Ribbon: View [tab]->Show/hide [section]->[ ]

2. Open (double-click) the .ini file that you placed on your Desktop (it should open in Notepad), and look for the [Application] section (using Edit->Find… will help). It will look something like this:

[Application]
Mailer0=Eudora
Mailer1=Outlook Express
Mailer2=Microsoft Outlook
Mailer3=Netscape
Mailer4=Windows Mail

3. Below "Mailer4=Windows Mail", add a new line (Enter key) and type (or paste) the following:

Mailer5=Windows Live Mail

Note: use the correct consecutive number if there are more, or less e-mail programs under [Application].

It should then look something like this:

[Application]
Mailer0=Eudora
Mailer1=Outlook Express
Mailer2=Microsoft Outlook
Mailer3=Netscape
Mailer4=Windows Mail
Mailer5=Windows Live Mail

Now save this edited .ini file...

4. Back at the location C:\Program Files (86)\Canon\MP Navigator X.X\Device (where X.X is the version number of MP Navigator  - in this example it will be MP Navigator EX 3.0) - rename the old .ini file by adding .bak to the file extension - in this example, mp600.ini is renamed to mp600.ini.bak

Note that if the .bak extension is added to the original .ini configuration file without first showing the file extensions in Windows Explorer (see Note in Step 1), the file will in fact be renamed to mp600.bak.ini which will interfere with the solution - MP Navigator will actually read the mp600.bak.ini configuration file and ignore the edited mp600.ini configuration file - hence, 'Windows Live Mail' will not appear in MP Navigator.

5. Now Copy and paste the .ini file that you edited, back into its original location at C:\Program Files (86)\Canon\MP Navigator X.X\Device (where X.X is the version number of MP Navigator EX - in this example it will be MP Navigator EX 3.0).

If you ever upgrade MP Navigator, you will need to repeat this process, because the upgrade will overwrite your edited .ini file.

6. Now select the Windows Live Mail entry from the list of Mail Programs within MP Navigator:


MP Navigator->One-click->Button Settings->Specify actions for one-click buttons. [section]->Attach to E-mail->Attachment Settings [section]->(o) Live Mail [select]->OK [button].

7. You will also want to select the Windows Live Mail entry within the Preferences dialog box of MP Navigator:

MP Navigator->Preferences [button]->General [tab]->Application to run [section]->Attach to E-mail: [drop down arrow]->Windows Live Mail [select]->OK [button].

8. To otherwise test if Windows Live Mail is the default e-mail program for MAPI:

Right-click a .jpg or .pdf file and select Send to->Mail recipient

A Windows Live Mail compose window should open with the file attached - although MAPI 'attached' image files are treated differently in WLMail 2011 & 2012 (they are not actually 'attached').

Cheers,
Peter.R.

glenprouty say:

Hi o_a_b,

A common cause for the increase in the usage of your HDD can be the result 

of something that is creating Restore Points. Or if you commonly create them.

You can verify this by displaying the "current usage" of your System Volume

Information.

Click "start" and type "restore point".  Click on "Create a restore point".

Select OS(C) System and then select "Configure".

Observe the "Current Usage". This is the amount of HDD space that is occupied

by your system volume information. (Restore Points).

As you loose HDD capacity you can compare this figure to that loss, to see

if this is the cause.

You can adjust the amount of space allowed for "Sys Vol" by changing the

"Max Usage" slider.  Restore Points will overlay once this Max Usage is reached.

You really don't need to reserve any more than enough space to hold about

5 or 6 restore points. Probably less than 15 GB. Going back further than 5 or 6

is a little risky. When you go back, you will be missing the changes made to

your PC in the interim.

Regards,  Glen
