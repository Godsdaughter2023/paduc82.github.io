lukeb0030 say:

I think the final answer to this is as follows;

Clipboard history support a history of copied images and text (25 in total). An image or text can be copied and will remain in the clipboard history for future use until; the limit of 25 is reached or; an item is manually deleted from the history or; the computer is rebooted.

Clipboard Sync supports syncing of Text only between computers, where the feature is enabled on both computers, and the computers are signed into the same MS account. Clipboard sync simply does not support images.

The documentation is not clear (IMO) because history and sync are typically discussed in the same article but they have different limitations (ie history supports images and sync only support text).

The solution for this (for me) was to use MS OneNote, as i can paste an image into OneNote on my desktop PC and annotate / mark up on my Surface (the image syncs across OneNote if they are both signed into the same account), and then use that marked up image on either device (the marking up syncs via OneNote as well, almost instantly).

Another option appears to be OneDrive; https://support.office.com/en-us/article/save-screenshots-to-onedrive-automatically-d04df71c-1cb0-4ad6-9f9c-b08494d79d6a

This could have worked for me also, but i'm happy with the OneNote solution that i'm using
