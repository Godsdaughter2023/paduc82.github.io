tryx3 say:

I am not quite sure what you mean.  I think you are saying that you upgraded Vista some time ago and that your problem is therefore only related to the recent Windows update that you ran.  If I am correct then running Start-up repairs might fix things for you.

There are no legal downloads of Windows 7 System repair disks.  There used to be some sources but MS worked its way around closing them down as they were operating without licences.

Your Windows 7 upgrade disk [or, indeed, any Windows 7 installation disk as long as it matches the 32 bit / 64 bit of your Windows installation] can do the job of a Windows 7 System repair disk.  Boot from it, select language-keyboard then at the install screen, you'll see a control on the lower-left Repair your computer.  Click on Repair your computer and follow the menu through and it will show you the option to run Start-up repairs.

  1. Disable the Scheduled Task - Does not stop it running
