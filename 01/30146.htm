Windows Live Movie Maker - Best way to output for PAL DVD - Microsoft Community

thornie say:

Windows Live Movie Maker - Best way to output for PAL DVD


There is no support for PAL format in any of the Windows Live Movie Maker versions. For the previous version, 2011, the Microsoft development team created a 720x576 prx.  file which created a satisfactory wmv file that could be imported into DVD Maker with satisfactory results.

Although I was able to obtain this file, I was never given permission from Microsoft to release it, so I just kept it to myself and shared with friends and students.

I expected the new version of Windows Live Movie Maker 2012 to show support for the PAL format after all the complaints the earlier versions received. ( It does seem, by the way, that Microsoft no longer even has any support staff dedicated to WLMM questions, anymore.)

The PAL prx file that worked with the earlier versions of WLMM will not work with the 2012 version. WLMM 2012 does not recognize prx files anymore.

I have been unable to contact Microsoft regarding this matter as they seem to have dropped all support for the program.

No doubt they intend to remove it in the future, even for NTSC users.

 

After a great deal of experimenting, this is the best result I can get with WLMM for PAL users and share it with anyone trying to get some satisfaction.

You should first make sure, in my opinion, that your project is set to the new 16:9 widescreen settings.

  1. After completing your project, click on the Share down arrow and hover your mouse over the recommended settings and write down the width and height details. eg- 1920 x1080 pixels, or 1280 x 720, or whatever the recommended suggests.
  2. Now go to the bottom of the list and click on "Create Custom Settings"
  3. Leave the Setting as <New> and give the new setting a name you will recognize, such as 1280x720PAL
  4. Then write the details you took down into the Width and Height, set the Bit rate to 8000 and change the Frame Rate to 25 fps
  5. Save the new wlvs file setting that you just created.
  6. Close the "Create Custom Setting" dialogue box.
  7. Now click on the "Save movie" down arrow and you will see the new setting you just created under "Recent Settings"
  8. Save the movie in a location of your choice and save it as an mp4 for saving to your computer or later uploading to YouTube or other site.
  9. If you want to burn it to a DVD using Windows DVD Maker or other software, follow step 8, but
  10. Once you click on Save, the file will be created on your computer.
  11. If you want to create a DVD, open Windows DVD Maker and import the file. If you have Windows 8 you will need to use a third party DVD maker. Most DVD makers should recognize a wmv file.

I have had a lot of frustration getting satisfactory results with WLMM and recommend that if you get serious purchase a program such as Adobe Premiere Elements for better results.

In the meantime, if you just want a quick and free result, following my steps should give you the best output.

Hope this has been helpful

j w stuart say:

It could be a sub-process or application that's running in the background
and taking all the CPU resources.

To find and display what could be the problem try Process Explorer:
http://technet.microsoft.com/en-us/sysinternals/bb896653.aspx
SysInternals Forums: http://forum.sysinternals.com/

Worth Reading: The Case of the System Process CPU Spikes
http://blogs.technet.com/markrussinovich/archive/2008/04/07/3031251.aspx

Once you have Process Explorer installed and running:
In the taskbar select View and check:
'Show Process Tree' and the 'Show Lower Pane' options.
Move your mouse cursor over any column in the right hand pane and
right click and check the following boxes:
'Command Line' and 'Version'.
Then expand the process named 'Explorer' (click on the + sign)
In the column on the left named 'CPU', look for any high CPU usage.
Next click on the CPU column to sort the processes by %CPU usage
(Highest to Lowest).

You can also mouse over and check any spike or high/low point in the CPU graph.

Move the mouse cursor over any process,
you should see a popup with some detailed info.
Then mouse over the process that's using most or all the CPU %.
Then click on that process to highlight it,
Now that  it's highlighted, right click and from the options listed select:
'Search Online'.
This should display what out there on the web about that process.
You can also double click on any process to open up a more detailed
'Properties' window.
Note: some entries like Explorer, System/Services, and Svchost
may need to be expanded to show the detail (sub processes),
in this case click on the + located to the left of the entry.

An alternate method when using Process Explorer

zigzag3143 (ms -mvp) say:
[quote]
"It's not a true crash, in the sense that the bluescreen was initiated only because the combination of video driver and video hardware was being unresponsive, and not because of any synchronous processing exception".

Since Vista, the "Timeout Detection and Recovery" (TDR) components of the OS video subsystem have been capable of doing some truly impressive things to try to recover from issues which would have caused earlier OS's like XP to crash.

As a last resort, the TDR subsystem sends the video driver a "please restart yourself now!" command and waits a few seconds.

If there's no response, the OS concludes that the video driver/hardware combo has truly collapsed in a heap, and it fires off that stop 0x116 BSOD.

If playing with video driver versions hasn't helped, make sure the box is not overheating.

Try removing a side panel and aiming a big mains fan straight at the motherboard and GPU.

Run it like that for a few hours or days - long enough to ascertain whether cooler temperatures make a difference.

If so, it might be as simple as dust buildup and subsequently inadequate cooling.

I would download cpu-z and gpu-z (both free) and keep an eye on the video temps
[/quote]

Change Wireless Mode

Login Information

  • Gateway - 192.168.0.1
  • Username - (blank)
  • Password - (blank)

Change Wireless Mode

  1. Open a web browser.
  2. Enter http://192.168.0.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (Default is blank.)
  3. Select Wireless Setup from the menu at the top.
  4. Select 802.11b/g mode from the side menu.
  5. Set the Wireless Mode to 802.11g only.
  6. Select Apply.
