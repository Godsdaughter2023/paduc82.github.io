ruz san say:

The programs need to be uninstalled one at a time. Each program is individually installed on the user account, requiring individual uninstallation. You can open Programs and Features in Control Panel to manage the programs. Here's how:

  1. Sign in to your son's account.
  2. Right-click on the Windows icon (Start button) on the home screen.
  3. Select Programs and Features.
  4. Select the programs you want to remove.
  5. Click on Uninstall.

Note that some of the programs are necessary for Windows and certain apps or services to run well on the user account.

The other apps you see when clicking on Start which you are unable to remove are part of Windows 10. These programs can only be uninstalled using Windows PowerShell. If you want to uninstall the built-in apps in Windows 10, follow these steps:

  1. Using the child account, click on Start.
  2. Type PowerShell then press Enter.
  3. Copy the command below and paste it in the PowerShell window:
    • Get-AppxPackage *xboxapp* | Remove-AppxPackage
  4. Press Enter.

You can get the full list of program codes by typing Get-AppxPackage in Windows PowerShell then pressing Enter. These program codes are what you need to type along with the command given above to uninstall other programs, replacing *xboxapp*. You can post a list of the built-in apps you'd like to remove and we'll give you the codes if you are unable to locate the codes yourself.

If you find the steps tedious, you can try to just delete your son's account and create a new one. Doing so will create a child account without the unwanted programs, but not those that come built-in with Windows 10.

For a System Image we recommend you use a USB external Hard Drive for that.

A System Image includes the Operating System, the Updates for it, your installed Programs, your Data, etc.

If your system files get corrupted and you can not repair the operating system or your hard drive fails, a System Image backup saves you a lot of time to bring your machine back up to where you were before the crash, etc. without installing updates, programs, etc.

a. user say:

I would not feel very confident about any program that split my information among multiple CDs - especially if that program was from Microsoft.

Microsoft is not in the backup/restore/system image business - that is not what they do but they had to come up with something and what they have is primitive, archaic and inflexible.

You should think about what you are going to do if your system has a catastrophic failure someday and you actually need to use your system image to get running again.  Are you going to feel good about restoring your image from 15+ CDs?

Are you going to make another image next month and use another 15+ CDs?

You would be best served by using products from a company that is in the backup/restore/system image business and that is all they do all day long - and has some good customer service and support behind them.

Popular these days is the free version of Macrium Reflect which you can download from here:

http://www.macrium.com/reflectfree.aspx

Of course they have a full featured version that you can buy but the free edition is enough to get started and see how it works.

Then (or actually first) get yourself a reputable external USB hard drive for storing your image(s) and lots of other things keeping it separate from the hard drive Windows is installed on.  That way if your Windows hard drive crashes terribly some day you won't have to worry as much about recovering because you will know your latest image is safe on a totally separate drive some place else.

Macrium will let you create a bootable CD or USB thumb drive so that when the time comes to restore your image you will have everything you need to get going again.
