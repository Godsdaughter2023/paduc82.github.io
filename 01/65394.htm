I fixed this by doing the following...

 

1. Open Regedit and navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tree

2. Export the TaskCache key to assurance.

3. Rename the Tree Key to Tree.old

4. Open Task Scheduler to see if the message still appears. It should not. Close Task Scheduler, Now rename the Tree key back as it was, then open Task Scheduler again the message should appear. This means an entry under the Tree Key is causing the issue. Close Task Scheduler.

5. There shouldn't be too many keys under Tree, rename each one to .old starting with the 3rd Party tasks i.e. AdobeFlashPlayerUpdate.  each time you rename 1 open task scheduler to see if the message still appears, keep doing this until no message appears.  I found that one of the AdobeFlashPlayerUpdates were causing the issue. I had 3 entries, I deleted the offender and all is now working.
