molesm say:

Just spent a few days solving the same error with a different cause on Windows Server 2008 R2 and in case anyone else runs into this issue and none of the above help, it may be due to the following:

It seems that the new agent requires TLS 1.2 to be enabled, for some reason it had been expressly disabled on our servers

To Check

  1. open regedit.exe (be careful!)
  2. Browse to “HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\SecurityProviders\SCHANNEL\Protocols\TLS 1.2\Client\”
  3. If do not have TLS 1.2\Client folder then this is not your issue, if you do and have the following keys and values in the folder then it could be your issue

DisabledByDefault=1

Enabled=0

To Fix:

Just delete the entire TLS 1.2 folder solved the issue, alternatively I expect expressly enabling TLS 1.2 by changing disabled to 0 and enabled to 1 would solve the issue.

create a batch file with the command and use task scheduler to run the batch file.

open notepad and type (or paste) the following

@echo off

then in task scheduler, edit your existing task, edit the trigger

remove the arguments 

remove start and use the browse button to locate your batch file

when finished right click on the task and select run, it should run the batch file and you should here 12.mp3 playing.

[edit] if you want to use windows media player rather than groove use the following (on one line with a space between .exe" and "c:\)

start  /min "" "%ProgramFiles(x86)%\Windows Media Player\wmplayer.exe" "c:\users\jonny\desktop\randomstuff\photosandvideos\projectforxmas\12.mp3"

martmcd say:

ok, I have had a look around and have noticed a number of people suggesting a registry change, in order to achieve what BulldogXX suggested.

in regedit navigate to HKEY_CURRENT_USER\AppEvents\EventLabels\WindowsLogon.

Double-click on the ExcludeFromCPL value and change it from 1 to 0

then use BulldogXX's suggestion 

Control Panel > Sound > Sounds tab > Program Events > Start Windows > Browse > choose your .wav > Apply.

then turn off fast startup (because the startup sound is bypassed if fast startup  is enabled).

click start, and type control panel

in control panel select power options

click on 'choose what the power buttons do'

click 'change settings that are currently unavailable'

untick 'turn on fast startup'
