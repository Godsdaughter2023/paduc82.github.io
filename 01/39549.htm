kevin d souza say:

Hi Janet,

 

- Why exactly are you trying to get into registry and what registry files are you referring to?

- Do you get any error messages when working on the computer or are you just worried about the errors showing in the Event Viewer?

- What happens when you try to use HomeGroup? Do you get an error? Are you trying to create a HomeGroup or join a WorkGroup?

- Is TuneUp Utilities 2013 functioning properly?

 

 

For issue related to HomeGroup, I would suggest you to go through these links which have some troubleshooting that you may try and get the issue sorted out.

http://windows.microsoft.com/en-in/windows7/open-the-homegroup-troubleshooter

 

http://windows.microsoft.com/en-in/windows7/why-cant-i-access-my-homegroup

 

 

For issues related to VSS service and the WMPNetworkSvc service as per the Event Viewer logs, I would suggest you to Start these services and also check if their dependency Services are started.

- In the Start Search, type Services.msc to get into Service.

- Locate Volume Shadow Service and right click on it and select Start.

- Locate Windows Media Player Network Sharing Service and then start the service.

- You may even right click on these services and click on properties and select the Dependency services and start those services as well for proper functioning.

 

 

Regarding the default values for some registry keys, those may be the settings made by the program or Windows for proper functioning of the software as well as Windows. I would suggest you NOT TO edit the registry unless you have a backup of the registry.

http://windows.microsoft.com/en-in/windows7/back-up-the-registry

 

 

Let us know if this information is helpful. If you have any concerns, please do let us know.
