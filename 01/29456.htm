Cause

The INACCESSIBLE_BOOT_DEVICE bug check frequently occurs because of a boot device failure. a. user say:

Hi,

If this is a bios password, contact HP.  Only they can change it.

Try this

Step 1: Logon your Windows as administrator

(You can also logon using Safe Mode with Command Prompt, pressing F8 when restart your computer, hit up/down key to choose Safe Mode with Command Prompt and hit Enter).
Step 2: Click on Start, type cmd in the Run box or Search box and press Enter.
Step 3: Type net user, all user accounts of your computer will be listed.
Step 4: Type net user "UserName" "NewPassword"(replace UerName and NewPassword with yours), press Enter, then the password will be reset as new one and you can logon Windows with it.

If not able to log on using the above, perform System Recovery to factory defaults.
