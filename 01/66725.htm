beleza, primeiro precisamos resetar o modem, com ele ligado, procure um orificio ou botão pequeno normalmente está escrito reset, fica atrás ou embaixo do modem, com uma caneta ou agulha enfie dentro desse orificio ou aperte o botão por 30 segundos ou até todas as luzes piscarem. configurando o modem como bridge:

conecte o cabo de rede no modem e no pc

abra então o seu navegador e entre com a seguinte URL 192.168.1.1 ou 192.168.254.254

Informe o : Nome do Usuário = TMAR#ZTV5.5.0a e Senha = ZTV5.5.0a

ou Nome do Usuário = ZXDSL  e Senha = ZXDSL

clique em "Quick Setup".

depois coloque VPI: 0, VCI: 33 clique em next

em "Protocol:" marque "Bridging", em "Encapsulation Mode:" escolha "LLC/SNAP" e depois next

em "Configure LAN side Settings" em "Primary IP address:" e "Subnet Mask:" não é necessário alterar os números. Mantenha sem seleção "Configure secondary IP address and subnet mask". Depois clique em "Next".

clique em "Apply" para concluir o modem será reiniciado, então até aí beleza, basta tirar o cabo de rede do pc

configurando o roteador como pppoe:

conecte o cabo de rede no pc e no roteador em qualquer uma das postar lan 1, 2, 3 ou 4

abra então o seu navegador e entre com a seguinte URL 192.168.0.1

Informe o : Nome do Usuário = admin e Senha = admin

clique em setup wizard, depois em next

marque gateway, depois next

habilite "enable ntp cliente update", na opção time zone select escolha: GMT-03:00, clique em next

em LAN Interface setup não mude nada, clique em next

em wan acessa type escolha pppoe, em username e password digite seu login  senha do velox(caso não tenha ou esqueceu digite o padrão, usarname: oi@oi e em password digite: oi), clique em next

em wireless settings não mexa em nada, clique em next

em wireless security tambem não mude nada, clique em finished, o roteador será reiniciado, retire o cabo de rede do pc

beleza, agora é só conectar o cabo de rede que sai do modem na porta WAN no roteador

Most of the Internet connectivity problems arise out of corrupt Winsock settings. Windows sockets settings may get corrupted due to the installation of a networking software, or perhaps due to Malware infestation. You might not be able connect to the Internet, or the packets won't transfer back and forth.

Follow the following methods:

Method 1

Let’s reset the winsock entries and TCP/IP stack and to troubleshoot the issue further.

Follow the steps:

a) Press Windows Key + X together and select Command Prompt (admin).

b) Type the following command and hit Enter after each:

netsh winsock reset catalog (reset winsock entries)

netsh int ip reset reset.log hit (reset TCP/IP stack)

Reboot the computer and check.

Method 2

Manually start the Wlan AutoConfig Service.Refer to the link below.

  1. Click on Start, type in services.msc.
  2. Select WLAN Autoconfig from the list, Right click properties.
  3. Change the startup type to Automatic.
  4. Click on Start button.
  5. Click on Apply, Ok.

 

Method 3
 
Run the System File checker Tool.

System File Checker is a utility in Microsoft Windows that allows users to scan for and restore corruptions in Windows system files. Perform System File Check (SFC), and then check if this fixes the issue

  1. Press the Windows button + X on your keyboard, a mini dialogue box will appear.
  2. Choose Command Prompt (Admin). You must run the command prompt in Administrator level.
  3. On the command prompt type sfc /scannow and hit enter.
  4. If everything went okay and no errors were found, restart your computer and see if the issue re-occurs.

 

Restart the computer and verify if it works.
