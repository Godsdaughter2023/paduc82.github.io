To mute the sound in a video clip, go to the Video Tools | Edit tab. Set the Video volume of that clip so it is zero. Then you'll have no sound coming from the clip. If you have multiple clips, you can repeat this for other clips as well.

An alternate way to do this, if you know that you want no original audio from any of your video clips is to go to the Project tab and use the Audio mix button. Set the audio mix so that it is to the far right with nothing coming from the original video clips but only added music, narration, etc.

  7. You need also to delete this file in the Recycle Bin, then restart the computer.
