Hello

There are three versions of Skype. Skype for Windows desktop, Skype for Windows 10 and new Skype. These three versions have different status behaviors.

  • In Skype for Windows desktop, the status will change to away depending on the minutes of inactivity the user will set on the General settings.
  • In Skype for Windows 10, the status will change to away automatically when left idle. Usually, the status change depending on the screen time out of the device.
  • In the new Skype, the status will disappear if the user opens a different window or browser. The status will just change back to online if the user hover the mouse to the application itself.
