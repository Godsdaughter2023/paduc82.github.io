maritzacapiro say:
Hi.

ASHX files are files used with ASP.NET programming and can be opened with any program that codes in ASP.NET, such as Microsoft Visual Studio and Microsoft Visual Community.

As they are text files, you can also open ASHX files with a text editor program. Use this list of the best free text editors to see our favorites.

ASHX files are not intended to be viewed or opened by a web browser. If you downloaded an ASHX file and expected it to contain information (such as a document or other saved data), it is likely that something is wrong with the website and instead of generating usable information, you provided this file on the server side.

Note: Technically you can see the text of an ASHX file using some web browsers, but that does not mean that the file assumes it opens in that way. In other words, a real ASHX file, which contains readable text for ASP.NET applications, can be viewed in your browser, but not all .ASHX files are actually ASP.NET Web Handler files. There is more about this below.

The best trick with an ASHX file is simply to rename it to the type of file you expected. It seems that many are supposed to be PDF files, so, for example, if you download an ASHX file from your electric company or bank, simply rename it as a statement.pdf and open it. Applies the same logic to a music file, image file, etc.

When these problems occur, the website you are visiting that is running the ASHX file is having some kind of problem and this last step, where it is supposed to be the ASHX file to be renamed to whatever. Whatever is not happening. Therefore, renaming the file is only you doing the last step yourself.

If this happens a lot when you download PDF files specifically, there may be a problem with the PDF add-on that your browser is using. You should be able to fix it by changing the browser to use the Adobe PDF plug-in instead.

Note: It is important to understand that you can not change the name of any file to have a different extension and expect it to do so. work properly. For example, you can not change the name of a .PDF file to a .DOCX file and assume that it will open correctly in a word processor. A conversion tool is needed for real file conversions.

Let us know if these steps help you to resolve the issue.

brianh1988 say:

I know this is an old thread, but I feel it's necessary to put in a few words of wisdom from my own personal experience. Most often, it is NOT your hard drive. It isn't even your XBox 360 unit in specific.

The real reason for this is severalfold:

1) Microsoft has a very dated release system that was crammed into the BIOS prior to launch, and either could not be updated or was benignly kept after the fact was found out. There's been practically no modification to this BIOS whatsoever, as it would cause a good deal of things to break and is written in well over 10,000 lines of boot code.

2) The hard drive itself is a must with pretty well all 360's now, if not all. If it detects no hard drive, then you are automatically kicked offline and must insert another (compatibly inferior) device. I'd like to take this moment to point out the USB thumb drives have a maximum average yield of around 15-30 (depending on the reader) MB's a second. This is far inferior to the clocked rate of a 360 drive (~102 MB's a second). If anything, that would complicate  the problem in troubleshooting now (and at the time of prior post).

3) The BIOS HAS been updated every time there was a new UI (user interface) update. This includes fixes to bugs as well. As any computer-savvy person is probably aware of, whenever files are transferred online, there is always the possibility for interference. It seldom happens to a full update, but over time these can add up.

4) We've been told, "Backup, backup, backup!" from day one on Windows and XBox but unfortunately, in my experience, seldom do people actually back up all their data. This means things ultimately get broken as they're shuffled about on the hard drive.. I myself have been a 360 owner since Feb of 2009, so I've seen four UI changes thus far, two on the current console.

All this adds up to the point where the system gets a few corrupt lines of code, your XBox 360 HD hasn't been formatted since it was shipped, and thus the hard drive is the prime suspect. Just like Windows, it doesn't run forever. The hard drive still often runs perfectly, it simply needs to be factory-wiped so it can start from scratch and not compound on load times.for everything.
