lemp say:

If your husband's computer had only one user account, that account is almost certainly an "administrator" account (unless you changed it when you create your own "administrator" account).

DO NOT, UNDER ANY CIRCUMSTANCES, REMOVE ALL "administrator" accounts from the computer.  In fact, best practices are that you should have a spare administrator account that is not regularly used.  Such an account needs to be available for emergency recovery purposes and for those instances when "administrator" privileges are temporarily required.

I suggest that you leave your "administrator" account active on the computer.  If you want your husband to be able to just turn the computer on and have it automatically go to his Desktop without entering a password, see this article:  https://www.lifewire.com/how-do-i-auto-login-to-windows-2626066

naman r say: Hi,
 
If you still have issue connecting to the network or internet follow this steps which fixes most connectivity issues. 
Method 1:
Run a Network troubleshooter.
Using the Network troubleshooter in Windows 7
http://windows.microsoft.com/en-US/windows7/Using-the-Network-troubleshooter-in-Windows-7
Method 2:
Disable the DHCP Broadcast Flag:
http://support.microsoft.com/default.aspx/kb/928233    
Check if the issue is resolved.
Method 3:
Reset the TCP/IP stack go to this article and either click on "Fix it for me" or follow the instructions to fix it yourself:
How to reset Internet Protocol (TCP/IP)
http://support.microsoft.com/kb/299357    
Method 4:
Step 1:
Disable the IP Helper service:
a) Hold the Windows key and type R, enter "services.msc" (without the quotes) and press Enter
b) Scroll down to the IP Helper service, right click on it and select Properties
c) In the dropdown box that says "Automatic" or "Manual", set it to Disabled and then click on "Apply"
d) Then click on "Stop" to stop the service from running in the current session
e) Click OK to exit the dialog
Step 2:
Disable IPv6:
Try uninstalling IPv6 on all interfaces, removing any IPv6 virtual adapters, and resetting the TCP/IP stack. To remove IPv6, go into the properties for each network adapter and either uncheck the box next to the protocol "Internet Protocol version 6 (TCP/IPv6), that will disable it, or select it and click uninstall, which will remove it off the computer. Then go into device manager and remove any 4to6 adapters, WUN miniport adapters, or tunnel adapters.
 NOTE: You should do this for each network connection. 
 
Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. 322756  (http://support.microsoft.com/kb/322756/   )  How to back up and restore the registry in Windows

a) To flush the DNS cache:
b) Click Start, type cmd in the Start Search box, and then click cmd in the Programs list.
