Save that and close the Services app

How to install Windows 10 on devices with limited disk space? - Microsoft Community

basith m say:

How to install Windows 10 on devices with limited disk space?

Hello,

Some laptops and tablets have very limited disk space.  Even the factory images for some models can have as little as 4 GB of free disk space.  Since upgrading to Windows 10 version 1607 requires more free space than this, how can I upgrade to the latest version of Windows on devices with limited disk space?

sharath srinivasa say:

Hi,

If you don’t have enough disk space to install the Anniversary Update through Windows Update, first try running Disk Cleanup to see if you can free up at least 8 GB of free space.  You can also find a list of helpful tips at Free up drive space in Windows 10. 
 
If you have at least 8 GB of free space, you can run the Media Creation Tool on the Software Download Center to install the Anniversary Update:

  1. Go to https://www.microsoft.com/en-us/software-download/windows10.
  2. Click Download tool now and select Run.  (You need to be on an administrator account to run the tool.)
  3. When you’re asked “What do you want to do?”, select Upgrade this PC now.
  4. The tool will then walk you through the rest of the installation process.  When asked, make sure to select Keep personal files and apps to prevent losing any of your data.
  5. Windows Setup will start, and you may get a message that says, “Windows needs more space”.  If so, attach an external drive for additional storage space and select Choose another drive or attach an external drive…  You can then proceed to install Windows.

 
If you do not have at least 8 GB of free space, you can still install the Anniversary Update, but you’ll need access to another PC, you’ll need a USB flash drive or DVD with at least 4 GB of space on it, and you’ll need an additional external drive:

  1. Sign into another PC that has enough free space, and go to https://www.microsoft.com/en-us/software-download/windows10.
  2. Click Download tool now and select Run.  (You need to be on an administrator account to run the tool.)
  3. When you’re asked “What do you want to do?”, select Create installation media for another PC.
  4. The tool will walk you through creating installation media.  If you need help, you can find detailed instructions on the Software Download website.
  5. Once you’ve created the installation media, insert it into the PC you want to upgrade.  Using File Explorer, run setup.exe from the installation media.
  6. Windows Setup will start, and you’ll get a message that says, “Windows needs more space”.  Select Choose another drive or attach an external drive…  You can then proceed to install Windows.
