Some Security Management packages also have similar software functionality to mitigate leaving a computer unattended and putting it into a sleep state which requires a password upon waking.

3. The next batch of Windows 10 and Window 7 power management settings are handled by the Control Panel. The  net effect of using these features is:

You can power down system components (desktop or laptops) including the display and/or hard drives. Establish low-power alarms for low power status of notebook batteries or UPS batteries. Also enabling sleep or hibernation modes for no activity conditions, and (like my notebook) configuring the behavior or the power button. The control panel settings are similar to the screen saver settings. Any prescribed fix down that path is probably not going to end well.

papa.mick say:

One more investigative step you might want to pursue is to first check the event logs for your computer. Look for errors or other anomalies which are noted. Start by doing a normal login first thing on a new day.  Log in and let the PC do its thing unattended and go into its dormant mode.  Either complete power down, hibernation, or a sleep mode. 

Then power up again and when it is in a normal state go to the event logs using an administrator account to access the Event logs looking for errors or other warnings that may have been recorded starting with that days entries.

Run the Event Viewer, and start reviewing the system event message entries. Any number of events can trigger warnings or errors which standout.  But normal entries may also give clues to what startup items are in place for system power management.  So pay attention to both normal as well as abnormal events.

sumit (independent advisor) say:
Hi,

Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

However, this is not same for retail licenses i.e. you purchased the license through Microsoft store. That does carry transfer rights. If that is the case, please see the below steps:

Run the activation troubleshooter on the new device using the below steps:
Go to Settings>Update and security>Activation and run the troubleshooter. and select I recently changed hardware on this device.

Please note that Activation troubleshooter requires you to have digital license linked to Microsoft Account.

Alternatively, if that does not help, you would have to call Microsoft Support.
