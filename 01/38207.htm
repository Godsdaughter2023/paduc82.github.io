p.bear say:
When using Microsoft Paint to scan a document/image, it wants to default image file type as a .png. How can I tell it to default to a .jpg the next time I close & open the program to make scans?


Apparently there's no way to change the default Save format, but there is an easy way to create a "Save As JPEG" shortcut in the Quick Access toolbar in Paint:

In the dropdown list in the Paint menu (click the arrow on the left-most icon on the menu bar, above the ribbon), hover your mouse over the "Save As..." command; when the list of Save options appears on the right, right-click on the one that says "JPEG picture" and choose the option to "Add to Quick Access toolbar."  Now you'll have an icon on the toolbar at the top of Paint that, when you click on it, will bring up the "Save As..." dialog with JPEG already chosen as the file type.

(AFAIK, there's no way to create a keyboard shortcut for the new icon and using the standard <Ctrl-S> keyboard shortcut will still bring up the default "Save As..." dialog with PNG chosen as the file type.)
