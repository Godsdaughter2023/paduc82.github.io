Reset again and follow the path to get your RT working again.

Never use the troubleshooting tool to fix updating.

The sequence for getting the RT back is at the bottom of this page.

https://surfacetip.com/install-windows-rt-8-1-update-3-2017/

Download the three stand alone KB update installers from the page and put them on a USB drive or the SD card installed on your RT.   KB3173424  KB3172614 and KB3097667

The sequence talks about downloading the recovery image from Microsoft. This is the same as the recovery  image that is stored in the recovery partition on your RT. I have had better results using what is already on the RT.

Reset your RT from within Windows.  Reset, not refresh.

https://www.microsoft.com/surface/en-us/support/warranty-service-and-recovery/refresh-or-reset-surface-rt#within

There is a page in this setup that wants the Bit Locker key. This is not necessary. Select 'skip this drive'

You can make all of this faster by only downloading and installing the Windows updates. You can hide all the Office updates and install them when this is finished.

Start the steps to install the updates.  Some of the searches for updates take a while, be patient and check progress.

If KB3097667  is not installed after step 16 turn off Wi-Fi and install KB3097667. After the restart, right-click on Taskbar > Properties, Start Menu.  Check the box, Use the Start Menu instead of the Start Screen for no more metro screen.

If you have any questions about this just ask.
