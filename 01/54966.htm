a. user say:

Hi,

Welcome to the Microsoft Community.

Runtime error is a software or hardware problem that prevents a program from working correctly. Runtime errors might cause you to lose information in the file you're working on, cause errors in the file (corrupt the file) so you can't work with it, or prevent you from using a feature.
