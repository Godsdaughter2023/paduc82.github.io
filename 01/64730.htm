    devlin sco say:

    Hello Christine,

    Running Windows Update usually corrects this type of issue in Windows. That being said, can you tell me all of the troubleshooting steps you've performed so far?

    If you're already signed in as the administrator, I advise starting your Windows 10 PC in safe mode to modify the system time. Should you have difficulty following these steps, please seek the assistance of other family members.

    1. Right-click the Start  button and go to Settings > Update & Security > Recovery .
    2. Under Advanced startup, select Restart now.
    3. After your PC restarts to the Choose an option screen, select Troubleshoot > Advanced options > Startup Settings > Restart.
    4. After your PC restarts, you'll see a list of options. Select 5 or F5 for Safe Mode with Networking.
    5. Right-click the Start  button again, go to Settings > Time & Language > Date & time.
    6. Under Related settings, select Add clocks for different time zones.
    7. Go to the Date and Time tab and make sure that the Time zone is set correctly. You can click Change time zone > select the correct time zone and then click OK to modify it.
    8. Go to the Internet Time tab and click Change settings.
    9. Select the Synchronize with an Internet time server check box and then choose time.windows.com for the Server.
    10. Click Update now until the clock successfully synchronizes with the server.
    11. Click OK.
    12. Once the system time is corrected, click Apply and then OK to save the changes.
    13. Restart your PC.

    If you have additional questions, or if the issue persists after the troubleshooting steps above, please let me know by replying to this thread. This way, I'll get notified and respond back to you as soon as I can.

    Language Accent Marks

    I feel compelled to start this discussion because of a very logical and intuitive feature of Windows 10 that I don't think gets enough attention. If your keyboard is set to American English (I haven't tried this with other English keyboards, it may work for them as well), you can type many vowels and consonants with various accent marks. For example:

    Ctrl + ~ followed by a = ã

    Ctrl + ` followed by e = è

    Ctrl + ' followed by i = í

    Ctrl + : followed by o = ö

    Ctrl + ^ followed by u = û

    Ctrl + , followed by c = ç

    Ctrl + ~ followed by n = ñ

    ...and so on. The only downside is that it only works in certain text boxes and not in others. I don't know whether an extensive list of these shortcuts exists, but if it does, where is it? I just happened upon this feature by chance. If there isn't a comprehensive list, perhaps we could compile one here.
