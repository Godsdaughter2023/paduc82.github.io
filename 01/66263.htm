Bad0 Nonp means something like "bad(x) non-paged" which just means a non-paged pool memory leak is going on. Could be a device driver issue

Have you made any changes to the PC recently that may have caused this issue to occur? Like a network card update or graphic card update or BIOS - etc.

You may try to isolate the issue by disabling your device drivers 1 by 1 and watch if the memory leak continues. You can do this in device manager. Start with a network card driver and disable it. If that doesn't seem to net you anything, you may just want to delete drivers for any device you can find a driver to, then reinstall them yourself with pristine updated drivers.

tryx3 say:

Hurmia,

To reinstall Windows 10, you boot from the installation disk.  This requires no Admin rights.

If you do not have an installation disk then, on the login screen, click on the power icon then hold down the Shift key as you click on Restart.  That will take you into a Windows recovery environment [WRE] where you select Troubleshoot and then Reset this PC.  A Reset is supposed to be about the same as a reinstallation but I believe that it offers you the chance to keep your existing files.  See Recovery options in Windows 10 - Windows Help

When you are up & running again, you can make an installation disk using http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

Denis

tryx3 say:

Hurmia,

Oh, yes I can see it now.  You tried to get into the WRE using Shirt-Restart before.

There are three methods of entering the WRE using Shift-Restart to do a Reset. 

1  While in Windows, click on the Start menu then the power icon then hold down Shift as you click on the Restart entry [and then let go of Shift].

2  Restarting the computer to reach the login screen then, instead of logging in, using the power icon on the login screen in the same way.

3  Start menu, Settings, Update & security, Recovery, Advanced startup - Restart now.

In all three cases, in the WRE you click on Troubleshoot then Reset this PC.

I think, judging by your explanations, that you might have already tried all three ways but I am not absolutely sure.
