tryx3 say:

Actually, I can see that I did not write all that I should have done.

Before editing anything in the Registry, you are always advised to make a full backup.  I do not suppose you are going to.  I don't either.

But what I do is save the key I am working on before starting - in this example, you are working on

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Notifications\Settings\Microsoft.Explorer.Notification.{.....}

so, before starting, right click on the key & select Export then browse to a convenient folder and save it.  The default file type, as you will see, is .reg and you should leave it as that.  [You can also Export after you have finished work so that you can keep both before & after records.]  When you have saved the file[s] you can:-

1  Right-click on the .reg file, select Edit and you will see its contents in Notepad {in a format that is much, much easier to read than would have been the case if you had messed about trying to Export the key as a .txt file type}.  It is perfectly safe to open it in Notepad and have a look around the various entries as long as you do not change anything that you do not understand.

2  Double-click on it, give it a couple of confirmations to proceed then await its completion dialog - running the .reg file rewrites all the values in the key to what they are in the .reg file.  So if you make a pig's ear out of your Registry editing, you can run the before version of the .reg file to get back to where you started.  Similarly, you can run the after version of the .reg file if you ever reset it to defaults [or if a Windows update does that for you].

Denis
