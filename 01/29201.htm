cmdrkeene say:

It shouldn't be OneDrive. Perhaps something like Outlook but even that should respect the checkbox under the "personal" section of the Skype settings.

Have you ever heard of the Microsoft tool called AutoRuns?  It's a tool (they acquired many years ago) that helps explore all the places something can auto-run from.  It'll scan the places we've already looked like Startup folders, but also registry keys and task scheduler triggers and all other various places that can be used as a entry point for automatically running software.

You can reach it from this link: https://docs.microsoft.com/en-us/sysinternals/downloads/autoruns

Scroll down all the way to the very bottom and click the "Run now from SysInternals Live" link, and run the program when your browser prompts you.  Give it a few seconds to load up, and then just type "skype" in the filter box up top and hit enter, see what it finds. If nothing, also search for "lync" just to see, that's the old program name for what is now known as Skype for Business.   Fingers cross this tool helps find it.

If this AutoRuns does find it, just uncheck the box on the left side to disable it.

cmdrkeene say:
PS - Be super careful with this tool.  You might think you can get a really "lean and fast" system by disabling things, but it's a recipe for disaster.
