Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery or installlation disk if the system prompt doesn’t work (as would seem to be the case).

If the System Restore doesn't work, do a Startup Repair by booting to the genuine Windows Vista Installation Disk (or one you can borrow from ANYONE) or from a Recovery Disk.  Here's the procedure: http://www.bleepingcomputer.com/tutorials/tutorial148.html.  To boot to the CD you may need to change the BIOS to make the CD-drive first in the boot sequence.  To do that, wait for the screen that tells you the F key to push to access the boot menu or boot setup.  Push it quickly. Since you're knowledgeable about computers, you should be using third-party backup programs instead of Microsoft's clumsy and inconvenient backup utilities.

You probably know that there are two kinds of backup: an image is a copy of your hard drive compressed into a single file and a file-and-folder backup copies individual files and folders that you choose.

Imaging is a little harder to understand at first because it's not as intuitive as simply copying files and folders, but the rewards are immense. With an image you have a complete copy of your computer. You'll never have to reinstall Windows or your software -- just restore an image and your computer goes back in time to when the image was made.

With a file and folder backup you can restore your vital documents, irreplaceable photos and videos and carefully curated music collection or your own or another computer. Both types of backup are important. To guard yourself against ransomeware - which you realize is completely avoidable - store at least some of your backups in a location that is not physically connected to your computer.

A good image or backup program makes it easy - once you've learned the software - to do the following things (that Windows' own utilities cannot do:)

  • create and save as many different backup jobs as you like
  • run backups automatically, on a schedule
  • run backups while you continue to work on the computer
  • create differential and incremental backups
  • create a retention schedule to save - for example - the last 10 backups or the last 30 days of backups
  • verify the integrity of a backup
  • easily restore an image even if your computer is completely compromised, even if it doesn't boot

and much more. You'll wonder how you used a computer without it.

In Microsoft's defense - they're not in the backup business. Windows' own backup utilities, like other Windows built-in utilities, are primarily for people who don't know how or are too afraid to install third-party software. You can do better.
