If you want to encrypt a file or folder using Encrypting File System (EFS), but the Advanced button isn't displayed on the Properties dialog box or the Encrypt contents to secure data check box is grayed out, it could be because of one of the following reasons

If Windows Update Troubleshooter and resetting Windows Update Components doesn’t fixes the issue, then put your computer in a Clean Boot state and then install the update.

Note: A Clean Boot starts Windows with a minimal set of drivers and startup programs, so that you can determine whether a background program is interfering with your game or program. Your computer may temporarily lose some functionality while in a clean boot environment, however, that functionality will return when you reset the computer to start normally after you have finished your troubleshooting.
