In WMP 12, it's not possible anymore to force a library update. However, you shouldn't need this in most cases - WMP does a pretty good job of automatically updating the library when something changes with the files.

> How can I select add previously deleted files to library?

That's still possible:  Tools -  Advanced - Restore deleted library items (press Ctrl+M if you don't see the Tools menu).
