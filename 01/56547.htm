brian tillman [outlook mvp 2007-2019] say:

This problem doesn't appear to have anything to do with Office.  It is an issue with Windows and your mail client.  Apparently Lotus Notes did not register a handler for the email recipient "Send To".  So, add a Send To that contains the needed shortcut to use Notes.  Your Notes documentation or the support web site for Notes should give you the answer on how to do that.  Basically, you should add a shortcut to Notes to your Send To folder.  The folder's path is %AppData%\Microsoft\Windows\SendTo for Windows 7, but you haven't mentioned your Windows version so if it's different, the path may be different.
