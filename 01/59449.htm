Why did you install a System Builder version of Windows on a Dell Inspiron One 2330?

A System Builder version comes with it's own 25 character product key and once activated that product key is tied to that installation.  If you activated on the other system then that is it for that product key - you can't use that product key on another system.  I would not think to install that on a store bought OEM system.

On the Dell I would have downloaded a plain Windows 7P1 Home Premium and used the 25 character product key that came with the Dell (on the sticker) to activate.

But using the 25 character Dell product key you can't download an installation ISO from the Microsoft site (you can download System Builder ISOs from Microsoft though).

You can download a plain Windows 7P1 Home Premium ISO from this link:

https://heidoc.net/joomla/technology-science/microsoft/67-microsoft-windows-and-office-iso-download-tool

Install THAT on the Dell and use the 25 character product key that came with the Dell and it will activate.
