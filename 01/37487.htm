lemp say:

Your Windows 7 system will not turn into a pumpkin on January 15, 2020.  All that will happen is that Microsoft will stop issuing security updates.  Thus, over time, your Windows 7 system will become more and more vulnerable to attack as malware authors discover new "exploits."

If you have a good antivirus program -- unfortunately, after a while, a/v programs may stop being provided for Windows 7 -- and only go to a few trusted websites (e.g., you only download new embroidery designs from reputable web sites and don't use the Windows 7 computer to "surf the 'Net"), you should be just fine.

Assuming that your current computer came with Windows 7 pre-installed, your Windows 7 license is tied to that machine and can't be transferred.  Microsoft's policy is that you can replace almost any part of your computer except the motherboard and your license will remain valid; if you replace the motherboard, Microsoft considers that you have a "new computer."  If and when you replace the motherboard, you will eventually get the "Windows is not genuine" notice and your screen periodically will turn black (although most functionality will remain -- see https://www.howtogeek.com/212956/why-your-windows-pc-isnt-genuine-and-exactly-how-that-limits-you/ ).

Although legitimate copies of Windows 7 are hard to find these days (and correspondingly expensive), you might consider buying a copy now (if you can find one) so that when your hardware finally dies, you will be able to install Windows 7 on new hardware.  Even if there are no Windows 7 drivers for whatever hardware is then available, you will always be able to create a "virtual machine" and install Windows 7 on that -- as long as you have a legitimate Windows 7 installation DVD and its corresponding product key.

Sometimes files downloaded from other computers, or the Internet, have an "Unblock" check box that needs to be ticked, this can be seen when you right-click on the file and select Properties (it is near the bottom of the Properties page).

The hard drive manufacturer will have free diagnostic software that is bootable, or that  may even run from within Windows, that will thoroughly check the condition of your hard drive.
