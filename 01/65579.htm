marilyn-- say:  

From: Ronnie Vernon MVP

 

Bill

 

You simply need to elevate the command prompt manually. Go to Start/All Programs/Accessories. Locate the 'Command Prompt' shortcut, right click this shortcut and select the 'Run As Administrator' option.

 

Somewhere in that book, the changes in the way user accounts work is explained. The most notable change is that when you are running with an administrator account, you are actually running with standard user privileges. You must explicitly elevate a process to get administrator privileges.

temhotabot say:

You can not have Port Forwarding AND uPnP enabled at the same time. If you do you will only cause more conflicts. What I recommend you do is first check the manufacturers website and see if they have an updated version of the router firmware. If they do, download it and install it as per their instructions. Then I would reset the router back to factory settings to clear out any jumbled settings you may have made. On most routers there is a reset switch on the back or bottom of the router, if there is none on your router then check within the router software.

 

Using uPnP should be sufficient enough to give you an OPEN NAT. If it is not then disable uPnP and setup your Port Forwarding as per the instructions on portforward.com.
