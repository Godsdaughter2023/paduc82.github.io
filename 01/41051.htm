bossdweeb say: If you have only one Partition try making a new folder in you C:\ drive and try
installing the game in a sub-folder there first (if you make a sub-folder then you can install more games to this new 'Game' folder later).
Program Files (x86) is integrated into Windows, and is a 'secure' folder, so
making a new folder should  have the same effect as a separate partition.
Excerpt from Wiki - "Some application installers reject spaces within the install path location"
This may be why Doom 3, and some other games won't install to the default location ( Just a guess. All I really know is there can be problems).

No guarantee it will work with your game. Games don't all respond the same.
What size Hard Drive do you have, and how much data is on your HDD  ?
Go to Start > Computer  and look at the amount of available space on the HDD ( C:\) .
I've not made a new partition on a disk with data on a single partition already.
Partitions can be made via Windows. -
If you do decide to try and make a new Partition -
Go to - Start > All Programs > Administrative Tools > Computer Managment
and find the  Storage > Disk Management tabs.
Partitions can be created here. If there is unallocated space it is easy to
create a new partition in that space, but I've not tried shrinking an existing
single partitioned HDD. This is a bit more complex than when starting with a
blank HDD as you have to worry about possible data loss.
You can get instruction by clicking on the Help tab at the top of the Window.
Go to Help Topics, click  Search  and the Window will show -  List Topics, and read on.
You may also  want do do a search for some articles.

There are also third party softwares made for partition managing, but to get
one that has all features enabled you usually have to pay for the program.

btw - Steam does not have RA2....

bossdweeb say: When you install games they give an option for 'custom' install (or similar term)
you can then Browse to the folder of your choice wherever that may be, and install there.
Some game make a folder to install to but, I generally make a folder for the
game before installing and then browse to that when I install just in case the
game does not give that option. Things can get a bit messy if several games
install  without making their own  folder as the files for different games get mixed  in with one another.
When the game starts to install if it says something like  'there is already a folder' just tell it to go ahead and install.

If you make a new partition and don't want to keep it you can delete it  and add
that unallocated space back to the original Partition or merge it into the original.
The instructions are there in the help and there are more
articles on the i-net that detail the methods of partitioning a HDD.
There's no harm in having another Partition especially with HDD's of 500GB or more.  I have several partitions. It makes keeping things organised easier.
It can take a while to work out the optimal Partition sizing for your needs, but
having more than one huge partition can be handy.
