trekdozer say: You can use "Advanced System Settings" to increase your page file, which is virtual memory stored on the hard drive.

Increase Pagefile Size
1. Open control panel and choose system and security.
2. Click System, then click Advanced System Settings.
3. Click the Advanced tab.
4. In the performance section click on Settings.
5. Click Advanced and choose Change in the Virtual Memory section.
6. Uncheck Automatically Manage Pagefile size
7. Select Custom, enter a new larger size, and press Set.
8. Click OK and restart your computer





Adding RAM requires buying RAM and installing it on the computer.

﻿First make sure your motherboard supports the amount of RAM you want to install, and which type of RAM the computer can use. If you don't know the type and maximum RAM for your computer check the manual. To use over 4GB of RAM requires 64 bit Windows.
