a. user say:

For defragmentation search Windows for Disk Defragmenter. SSDs are not (and should not be) defragmented, but Windows runs the TRIM command.

I remember when it was a good idea to run CHKDSK weekly! That hasn't been the case for years. There's no reason to run CHKDSK now. Besides, if you ran CHKDSK on today's multi-terabyte drives you could be in for a long wait. Hard disks are extremely reliable now but they can still develop physical problems (overheat, head crash, etc.) If that concerns you, run the drive manufacturer's own utility.

There was never a need to delete temp files. There was never a definition of "junk" files; that's a creation of the computer cleaner industry. Likewise there was never a need to 'clean' the Windows registry.

It can be a good idea to empty your web browser's cache - that often solves problems with the way pages display. Emptying the cache forces the browser to download a new copy of the web page. But you only need to do this if you're experiencing a problem.

On the other hand, backing up regularly is extremely important. You should backup your data folders (Documents, Pictures, etc.) but you can also backup your complete disk with disk imaging software. Once you learn to do this, you'll wonder how you lived without it.

Let's suppose I try to 'optimize' my computer - which you should avoid, trust me on this - and I completely mess things up. I simply restore a disk image from before my little experiment and I'm back at work like nothing happened.
