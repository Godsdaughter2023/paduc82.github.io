burrwalnut say:

1. Check Folder Options > View Tab that "Restore Previous Folder Windows On Login" is unchecked.

 

2. Also, after every login your profile is loaded into the registry and a certain number of programs are started from the HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run key. When Windows is unable to load a particular program, say, the path of the program doesn't exist anymore, it sometimes opens up the C:\Program Files folder by default.

 

To solve it, in a Run window (Windows Logo key+R), type regedit, press Enter and navigate to the above key and check if every value is known to you. Then do the same for HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run.

 

Note, it is prudent to create a system restore point before making any changes, just in case something goes wrong.

It's possible that you'll be able to change the password through your own administrator account. Just to confirm, is your daughter's account a Microsoft account, or a local user account?

If it's a local account, remove or change the password from the Control Panel. Sign in using your administrator account, then follow these steps:

  1. Click Start.
  2. Type Control Panel and press Enter.
  3. Click User Accounts.
  4. Click Manage another account.
  5. If the User Account Control window pops up, click Yes.
  6. Choose the local account that you want.
  7. Click Change the password.
  8. Leave the password fields empty if you want to remove the password. Otherwise, enter the new password that you want to use, as wells as a hint and click Change password.

If it's a Microsoft family account, follow these steps to remove your child's account:

  1. Log in using your adult account here.

a. user say:

The only way to make a backup of Windows is to make an image of your C: drive.

An image is an exact copy of your hard disk. You'll need image software to make an image.

Since the image is an exact copy of your C: drive, it contains an already-activated copy of Windows. Thus, when you restore the image, you are restoring an already-activated copy of Windows.

Bit of background:

Everything that you have on your computer - all your software, all your programs, configuration settings, data etc. - is really just a very long string of 0's and 1's. Image software copies all the 0's and 1's. When you restore an image, you are replacing the 0's and 1's currently on the hard disk with the 0's and 1's in your image. Which means that your computer is restored to exactly the way it was at the time you made the image.

Windows has its own image utility (System Image) but it's inconvenient - first you reinstall Windows then you restore the image - and it doesn't have the appealing features of third-party imaging software. A third-party program makes it easy to make and restore an image and to create a schedule to automatically make backups. It takes a while to learn how to use imaging software, but it's time well spent.
