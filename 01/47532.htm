For networks where IP addresses are fixed (statically assigned), ensure each local host is configured with a unique IP address.

How to Release / Renew IP Addresses on Windows

Start, Settings, Accounts, Family and other User or Account, add another Account, not Family, do not enter an E-Mail or Telefone Number, hit, I don't know the Login Information of that person, at the bottom, create User without MS Account , put in Visitor or any other acceptable Name, the Programm will tell you if it likes Guest or not, don't put in a password.
