Can't play some midi files

I have some midi files that will not play in Media Player 11. They were playing fine before my hard disk crashed, but now that I have re-installed XP, they will not play.

 

Tom

fixer upper say:

Hi,

Ok. So first of all playback of .mid/.midi files are provided by DirectX and the Sound card- not the player. You will want to make sure you have the latest version of Direct X as well as the updated drivers for your sound card first up.

 

Here are the troubleshooting steps:

 

1. Try to play a different MIDI file to see if it is an issue of a corrupt .midi file.

2. Let the file play briefly to see if you get any sound, sometimes there will be a brief silence with header information at the beginning of the .midi file which will play as silence.

3. Locate the speaker icon in the right bottom notification area in task bar. Make sure that the volume slider for MIDI/SW Synth is not on mute and turn it up.

4. Reset the file association in the player. In WMP, click Tools/options. On the File types tab/file types area- clear the MIDI file check box. Click "Apply". Select the MIDI file check box and then click apply again. This is especially useful for clearing up the "This page cannot be displayed" error.

5.

a. Control panel/ Sounds and Audio devices/ properties : click "audio" tab. In the MIDI music playback

b. In the MIDI music playback area, make sure the default device is not set to MPU-401 or External MIDI (unless you are actually porting MIDI to a keyboard or similar).

 

That covers a lot of options. Most likely it will be MIDI mute in volume settings as this is common.

 

Reference: 

http://www.microsoft.com/windows/windowsmedia/knowledgecenter/mediaadvice/0067.mspx#540

 

Hope this helps,
