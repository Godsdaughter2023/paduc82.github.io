hairyfool say:

Have not seen Win8 on Dell but they normally provide a recovery disk creation tool which will make the factory disk set. True this is not the current state after you have set everything up. Windows 8 no longer uses backup or even system image as its preferred tool instead everything is based around the Microsoft Account and the File History utility. Neither of these actually generates a full system backup but the screen for the latter is the key.

 

In the left corner of the File History control panel applet is an option "Windows 7 File Recovery" which looks like just an option to get files out of a Windows 7 backup but is in fact the full System Image creation utility

 

As James says there is a fairly well known "bug" with the Windows Setup routine in that the System Reserved partition created can be too small for the Volume Shadow copy Service (VSS) to succeed during a backup. If you go through the System Image creation process but untick the System Reserved drive the utility should complete.

 

I would only describe this as a workaround, the fix is to grow the System Reserved partition to a more sensible size but this is not a novice solution, the real answer is the setup routine should create the SR partition a sensible size in the 1st place but as that is in the installation DVD Microsoft are not going to do that any time soon.

 

As to why you are not expected to do a full system backup any more, I suspect the theory is instead you reinstall your system, reinstall your programs,  so all you need from backup are your files hence File History. Don't be surprised if the "Windows 7 File Recovery also disappears in the future.
