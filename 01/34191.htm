Restore Factory Defaults - Microsoft Community

xb1r apollo say:

Restore Factory Defaults

Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - user | admin | root     | user   |   root    | root
  • Password - pass | admin | 12345 | 12345 | central | ZOOM-DSL

Restore Factory Defaults

Through firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter the admin for the username and admin for the password if the gateway asks.
  4. From the menu on the left, select Management.
  5. Select Settings from the submenu on the left.
  6. Select Restore Defaults from the next submenu.
  7. Select the Restore Default Settings button.

Through Hardware

  1. Locate the reset button on the back of the device.
  2. Using a toothpick or ballpoint pen press and hold the reset button for 30 seconds.
  3. Wait for the device to restart.
