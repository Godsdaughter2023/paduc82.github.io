wunders say:

The "System Volume Information" folder is a super-hidden folder in the root folder.  Its purpose is to store System Restore files so that you can recover to the system state at a prior date should problems occur with your machine.  This file is normal and should be left alone.  You can delete most of the contents of this folder by turning of System Restore (but then you won't have anything to fall back on should you need to restore).

Right click on Windows Start > Right click on Windows Powershell (Admin) > Copy the following from below and right click in Powershell to paste > Enter

Get-AppxPackage -allusers *WindowsStore* | Remove-AppxPackage

Copy the following from below and right click in Powershell to paste > Enter > Reboot your computer

Unless Microsoft fix their website or give you a download link as they did for Lazenby, the only alternative is for you to borrow English & Spanish DVDs from somebody else and transfer them to USB sticks.  Please note that this is permissible under the terms of the Windows 7 licence [we are allowed to make a backup of installation media].  Note also:-

1     The DVDs would have to be the right "Type" [32 bit or 64 bit] for your intended computers. 

2     You'd need to use DiskPart procedures to prepare the USBs. 

3     It would be a bit simpler to borrow Windows 7 Ultimate DVDs but others can be used [you have to do a particular edit on the USBs after copying the DVDs' contents across].  Such multi-Edition installation USBs would apparently have limited repair capabilities compared to the proper Ultimate DVDs but I have never managed to find an explanation of just what those limitations would be.

4     It is possible that a Retail installation DVD might give you the choice of both English & Spanish installations; I have never used a Retail installation DVD so I simply do not know. An installation USB made from an English language Digital River ISO certainly does not allow this but an installation USB made from my Dell OEM installation DVD does. 

5     There ought to be nothing to stop you using OEM or System builder DVDs of the right Type.  It's your Product key that confers your Retail licence transferability rights rather than the DVD itself.  You might get annoyed with an OEM DVD insisting on installing the hardware drivers it expects to need but I would imagine that Windows update would resolve any problems that created [but I have never had to try this myself].

If you do borrow somebody else's DVDs, post back here and, if you need them, I will dig out my notes of the procedures I used successfully in paras 2 & 3.

We will edit the image (ISO) and remove the ei.cfg file so we can reveal all editions of Windows 7 during the installation.

After downloading the eicfg removal utility, you need to extract it, since its in a .zip file. Just right click it and click Extract All.
