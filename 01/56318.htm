If the app launches on Windows Startup, it is in either of there 2 places

1.

Press Ctrl + Shift + Esc keys to open Task Manager, on the Startup Tab, see if it is in the list

If it is:

  1. Firstly, right click the entry and click 'Open File Location' to note the location of the file
  2. Then select the entry on the startup tab in Task manager and click 'Disable' and close Task Manager
  3. Go to where the file was located, see if there is an uninstall.exe file there, if there is double click it to uninstall the app, if not simply delete the program folder

2. 

  1. Press Windows + X keys together, type msconfig and hit enter
  2. When the dialog opens, click on the services tab
  3. Check the box marked 'Hide all Microsoft Services'
  4. Run down through the list of remaining services and see if you can find the entry that is starting the app
  5. If you find it, uncheck it, click apply and reboot
  6. You will have to manually find where the file was located, see if there is an uninstall.exe file there, if there is double click it to uninstall the app, if not simply delete the program folder

The Xbox One is a not a DLNA DMR unless you install the Xbox Music and the Xbox Video apps on it from the market store. Once you did that, it should be discovered and you can do PlayTo scenarios against it.

However, sometimes (especially when you update your OS from 8 to 8.1 and the device was discovered by Win8) the device does not get populated in the charm or classic PlayTo menu. To overcome this, you need to go to Classic Control Panel-->Devices and Printers and look for your XboxOne device in that list. Remove any entry for Xboxone and choose again Add a Device after this.
