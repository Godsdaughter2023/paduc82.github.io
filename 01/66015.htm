1.
Move all your User Library folders to D:

Open Windows File Explorer

In the right hand pane, right click your Documents folder and choose Properties

Select the Location tab

In the Location box, type D:\Documents then click Apply and Ok

Each time, provide a new folder location, for example, Pictures will go to D:\Pictures

Move the location of all future content downloaded from the Microsoft Store

Open the Settings App (gear icon on your Start Menu)

Go to System - Storage - Change where new content is saved:

Change all options you find there to D

3.
If you are a Steam Gamer

Stop all processes associated with Steam

Move the Steam folder to your D drive manually

Then open the Steam Control Panel and point that to the new location


4.
Please Note: this should ideally be done before you begin installing any software

For 3rd party applications, some will allow you to choose an installation location, for those, create a new Program files folder on your D drive, then for each application, create a new sub-folder of that folder

Other 3rd party applications will not allow you to change the installation location, and there is no safe way to move those to your D drive . . .

If you didn't already do it, I would recommend running the cleanup tool a 2nd time and choosing the "clean up system files" option, which can free a lot more space than just the regular cleanup.

As for how to move files to another drive, here's how I like to do it:

  1. Open File Explorer and go to the "This PC" section, which will show you your drives and folders.  Go into your Pictures, Music, or other folder where the files you want to move are located.

  2. Press Ctrl+N, which will open a 2nd File Explorer window.  This time, go to the "This PC" section, and open your other hard drive.


  3. Position the two Explorer windows side-by-side, then drag files from one to the other.

mlg63 say: What model computer? How many hard drives--not partitions--separate drives? There were computers from a few years ago that separated the operating system and data on one hard drive by putting the OS on a tiny C partition and all the data on the D partition. If you have one of those it will show in Drive Management. The only way around this is to delete the D partition in Disk Management (this will delete all your files along with it) and then extend the small C partition to include all the unallocated space left from the former D partition. The D partition must be adjacent to the C partition--on the right side of C. Get a techie friend to help if this sounds too complicated.
