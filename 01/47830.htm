Problem:  

The following error is received on the Activation page after a clean install of Windows 10.  Windows cannot be fully configured until it is activated, however it will not accept the Windows 7 or 8.x product key.

Error: 0xC004C003 - The activation server determined the specified product key has been blocked.

Cause:   


Due to performing a “clean” install using a Windows 10 CD or USB. The first time Windows 10 is installed on a Windows 7 or Windows 8.x PC, it must be “upgraded” not the “clean” install option.

Solution:

  1. Install your previous version of Windows. (7 or 8.x)
  2. Install all Windows updates.
  3. Run the Windows 10 CD or USB within Windows (not during bootup)
  4. Follow the on-screen instruction to UPGRADE to Windows 10.  The UPGRADE process converts your old Windows key into a new Windows 10 key.  It will not ask for the key, and after installation Windows 10 will be activated.  

NOTE:  Ensure you do not pick any advanced options to perform a complete wipe of your drive to perform a clean install when installing Windows 10 for the first time.

Meanwhile, you can follow the steps below on how to troubleshoot startup issues in Windows 10:

Method 1: Run the boot recovery commands in Command Prompt

  1. Turn on your computer.
  2. When you see the loading dots, immediately turn off your computer.
  3. Repeat steps one and two. Thrice will be enough for the computer to think that you are having issues with startup.
  4. Once the computer boots up for the third time, you will see the loading dots on the screen and the message "Preparing automatic repair".
  5. Afterwards, your PC may turn blank then it will show "Diagnosing your PC".
  6. It will ask you to restart or choose Advanced options, choose Advanced Options.
  7. Select Command Prompt.
  8. On the Command Prompt, type or copy each command lines followed by hitting Enter:
       bootrec.exe /rebuildbcd
       bootrec.exe /fixmbr
       bootrec.exe /fixboot

Method 2: Run chkdsk.

  1. Open Command Prompt in WinRE. (Follow the steps under Method 1 to access Command Prompt.)
  2. Copy or type these commands and then hit Enter:
       chkdsk /r c:

Method 3: Disable Early launch anti-malware protection.

  1. Access the Boot Options by restart your computer at least 3times. (Follow steps 1,2 & 3 in Method 1.)
  2. In the Boot Menu, choose to Troubleshoot > Advanced Options >Startup Settings. (Click See more recovery options if don't see the Startup Settings option.)
  3. You will be prompted with a list of options and will ask you to restart, click Restart.
  4. Choose Disable early launch anti-malware protection by clicking the F-keys or the number that corresponds to it.

There is no built-in "Lock" status application within most Windows versions.  It is the responsibility of the keyboard manufacturer, or laptop maker, to provide this style of application.  Logitech and Microsoft keyboards do provide a "Locks" status module as part of their enhanced drivers.  Logitech is in their SetPoint drivers and Microsoft provide these in the Intellitype drivers.

Also, with several Bluetooth "protocol" drivers (not Microsoft's) can also provide the "Locks" status applet.  It is usually for use with most Bluetooth keyboards but these can respond the regular keyboards.
