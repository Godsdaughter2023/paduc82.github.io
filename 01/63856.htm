xircal say:

The system can charge the battery to 100%, but that doesn't mean that the battery is still working optimally.

As you can see from the following screenshot, my own battery has a design capacity of 48,400 Mwh, but since the laptop is 15 months old now, that has reduced to 35,574 mWh. It will still charge to 100%, but that's 100% of the lower capacity, not the original design figure.

In your particular case, your laptop battery may failing and that would show in the report in the same way. For example, if the design capacity was also 48,400 mWh, but the capacity remaining now was only 15,000 mWh, then that means you should replace the battery.

You should also bear in mind that if the battery loses its charge completely, you won't be able to start the laptop even if it's plugged in.

If you're having trouble copying or pasting the command, you can type it instead, but it must be exactly as I've written it:

powercfg /batteryreport

There's a space between the word "powercfg" and the forward slash attached to "/batteryreport" which is also one word.

regile reb say:

Hello Raj,

Password expiration is a feature in Windows that forces a local account on the PC to change their passwords when a specified maximum (42 days by default) and minimum ( 0 days by default) password age has been reached. Follow the steps provided below to disable password expiration:

  1. Press Windows key + X > Go to Command Prompt (Admin).
  2. Type wmic UserAccount set PasswordExpires=False > Hit Enter.
  3. Close the window and check if the error still appears.

Note: Replace UserAccount with the actual user name of the local account that you want to disable the password expiration.
