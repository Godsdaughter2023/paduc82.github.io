This is correct. However, since the name of the profile folder is mostly invisible to the user this should not really matter. If it matters to you then you can do this:

  1. Reboot the machine.
  2. Log on under an admin account.
  3. Rename the user's profile folder to C:\Users\User.bad
  4. Delete the corrupted account.
  5. Create a new account.
  6. Log on under the new account.
  7. Move the files and folders from User.bad.
