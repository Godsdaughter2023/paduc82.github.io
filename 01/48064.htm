  1. Open Start.
  2. Search for Command Prompt.
  3. Right-click the result and select Run as administrator.

  4. Type the following command to start DiskPart and press Enter:

    diskpart

  5. Type the following command to list all the volumes and press Enter:

    list volume

  6. Type the following command to select the volume in question (for example, E) and press Enter:

    select volume E

    Important: Proceed with caution, and replace "E" with the letter that appears in the newly created partition.

  7. Type the following command to remove the letter and hide the partition in question and press Enter:

    remove letter=E

  8. Use the exit command to close DiskPart.


After completing these steps, the drive will disappear from File Explorer, but it'll continue to be present on your computer.
