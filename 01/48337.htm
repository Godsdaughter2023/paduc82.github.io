  • How many user accounts do have on your computer?

When you set up Windows, you'll be required to create a user account. This account is an administrator account that allows you to set up your computer and install any programs that you would like to use. Once you have finished setting up your computer, we recommend that you use a standard user account for your day-to-day computing. It's more secure to use a standard user account instead of an administrator account.

You need to login using the available administrator account to change the user account type.

Let us try the following methods, and check if it helps.

Method 1:

1) Press Windows + R and type netplwiz, press Enter.

2) Check for the Administrator account.

3) Log Off from your computer and Login through Administrator account.

4) Again press Windows + R and type netplwiz, press Enter.

5) Select your Microsoft Account and click on Properties button.

6) Click on Group Membership tab and check on Administrator.

7) Log off from your computer and try logging in through Microsoft account and check if it helps.

Method 2:

I suggest you enable the build in administrator account, change the exiting account as administrator and then try creating a new user account.

Follow the below steps:

Step 1: Enable Build in administration account.

1) Press Windows + X key together on your desktop screen.

2) Click on Command Prompt (Admin) to open Command Prompt.

3) Type the following command and hit Enter:

     net user administrator /active: yes (Check for the spaces).

4) Now log off from the current user.

Step 2: Log in to your computer, through Build in Administrator account.

1) Press Windows + R and type netplwiz press Enter.

2) Select your existing user account.

3) Click on Properties button.

4) Click on Group Membership tab and check on Administrator.

Hope it helps.
