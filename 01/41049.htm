oddessa say:

Posted this on tecnet forum so will post fix here.

OK i've finally fixed the problem.

The error was caused by invalid or corrupt registry entries in the following:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tree\Microsoft\Windows\Media Center

When i removed WMC it only removed some of the entries not all, the following were left in the registry:

mcupdate_scheduled

StartRecording

PvrScheduleTask

So i deleted C:\ProgramData\Microsoft\ehome & removed WMC, I then made a backup and deleted the Media Center key. I then reinstalled WMC and run through the TV setup, checked the registry and found the Media Center key had been replaced with new entries.

Finally tested LiveTV etc, scheduled programs to record and finally SUCCESS, all scheduled recordings recorded without issue.

Remember to always back up the registry or create a restore point, before attempting to edit the registry.
