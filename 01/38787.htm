This bug check indicates that the requested page of kernel data from the paging file could not be read into memory.

ERROR_CODE: (NTSTATUS) 0xc000003f - {Bad CRC}  A cyclic redundancy check (CRC) checksum error occurred.

What is a CRC? A cyclic redundancy check works by taking a block of data that is supposed to be written to a CD, DVD, or hard disk, and then it calculates the checksum and writes the number with all the original data to a CD, DVD, or hard disk. It ensures that when the data is going to be read, the same calculation happens, and in case a miscalculation occurs, then there is an error, because the data that is read is not the one that was originally written.
