Now you have "the best of both worlds" -- working hardware and access to the extra features provided by IntelliPoint and IntelliType.

General Update Rule 1:
Don't update drivers unless you have a specific problem that the new driver is known to fix or unless the new driver offers a feature you must have ("If it ain't broke, don't fix it").

General Update Rule 2:
If you decide to update drivers, get them from the device manufacturer rather than Windows Update (of course, here the manufacturer is Microsoft, so see Rule 1).

General Update Rule 3:
Do NOT install any "optional" or "hardware" updates offered by Windows Update.  (This seems a bit less black & white for Windows 7 than it was for XP.  In particular, some printer drivers that not offered as downloads by the printer mfr and also not included "in the box" when you install Win 7 become available if you let the Win 7 printer install wizard connect to the MS update server.)
