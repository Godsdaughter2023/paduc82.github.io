a. user say:

A missing Help and Support service does not fit the symptoms of a corrupt user profile.  The Help and Support Service is system wide and does not belong to a particular user, and if you think your user profile is corrupted, you better not just delete it unless you want to lose all the stuff belonging to the user you are deleting.   That is not a good way to handle a corrupt or suspicious user profile - unless you want angry users!

Have you ever used any third party applications on your system where the product description includes such descriptive phrases as:

cleaner, mechanic, detective, cure, optimize, clean up, tune up, speed up, fix up, compact, compress, faster, power, boost, booster, guaranteed.

The Help and Support Center sometimes get broken with overzealous cleaning and there are lots of things to "try" that sometimes work and sometimes they don't. Sometimes people just give up and never use Help and Support Center again, but it can always be repaired.

We can reinstall the missing Help and Support Service in a couple of ways, let's start with the least complicated:

Using Windows Explorer, browse to and locate c:\windows\inf\pchealth.inf.  THat is the Help and Support installation information file.  

Right click the pchealth.inf file and choose to Install.

You may be prompted to enter your XP installation CD as it looks for the installation files.  The files are on your XP installation CD or, if you have no XP installation CD, the files will already on your system from the original installation (if Help and Support was working before) - you just need to tell the installation where to look for them.

You may want to locate the files on your system before you start the installation so you will know where to tell the installation to look when it needs a file.

If you don't have an XP installation CD, change the "Copy file from" dialogue (by clicking the Browse and Retry buttons) to point to the appropriate folder for each requested file.  Usually, the files are in these folders.

dataspec.xml (c:\windows\servicepackfiles\i386)
pchdt_w3.cab (c:\windows\pchealth\helpctr\binaries)
HCAppRes.dll (c:\windows\pchealth\helpctr\binaries)
atrace.dll   (c:\windows\system32)

If those files are not in the same place on your system, you need to do a Windows Search on your system for the file you need and point the installation to the correct folder for your system.

Reboot your system to be sure the installation is complete and check in the Services now to see the condition of the Help and Support Service.

If the Help and Support service is not running, right click it, Start it and then be sure the Properties are set to Automatic, reboot again and be sure it starts on it's own.

If for some reason that does not get your Help and Support Service installed (maybe you can't find the files) you can follow these other instructions from Ramesh:

http://windowsxp.mvps.org/helpsvc.htm

a. user say:

When Windows has a problem reading the profile information for a user it will log that user into a temporary profile (but you know that already).

In addition to the brief message (which may be missed) other symptoms are a desktop with all your usual icons, shortcuts, etc. missing and a panic situation where you might think all your programs are gone and all your personal files (documents, pictures, music, etc.) are "missing".

It is unlikely that anything is really missing - the temporary profile just doesn't have access to the same desktop, programs and files and things the regular user profile does if it was working properly.

The issue may just be a fluke and restarting your system and logging out and back in will get things going again but sometimes a user profile can get corrupted and the user will never be able to log in normally.

You can't fix a corrupted user profile but you can create a new user account (which will make a new profile) and copy things from the corrupted user account into the new user and then you will have to login with the new user account from now on.  Later when you are comfortable with the new user account you can delete the corrupted user account to keep your system tidy.

Here are some generic ways to test for a corrupted user profile and what do to if you need to create a new user account and copy things:

If you see the taskbar and desktop with just a few icons and it appear that all or most of your installed programs and personal files are "missing" it could be that you have a corrupted user profile...

It is extremely unlikely that all your documents (or anything else) has vanished or been deleted...  they are just in a place where you can't see or find them.

This condition sounds like a corrupted user profile.

Sometimes (for some reason) Windows has a problem reading your profile information when you login and that condition could just be temporary and simply restarting your system will get things going again and if that works you should keep an eye on things because it could be a sign of a bigger problem.

When your user profile is corrupted Windows will log you in to a temporary profile and that temporary profile is not going to have any of the things that you are used to seeing - but you can fix it.

Every user on your system has a user profile and under each user profile are all the things what belong to that user - the Windows desktop, the taskbar, the All Programs menus, Documents, Music, Images, etc.  Each user has their own set of these and if Windows has logged you in to a temporary profile you are not going to see any of these things.

You can use Explorer to navigate to the C:\Users folder and look for a folder that has the same name as the user profile you are used to using - that is where all your "missing" items are.  You might see something like this in the Users folder:
