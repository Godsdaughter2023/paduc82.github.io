Ethan,

The user account you created when you first booted up your Windows 10 computer was automatically made an Admin user account.

There is no such thing as "the" Admin account. 

  • You can make as many as you want. 
  • My own advice about making new accounts is in a thread in this forum, see paras 1 & 2.
  • I believe that my advice about user account creation is better than that provided by Microsoft.

As you may already be aware, Windows creates a Built-In Admin account that is, by default, disabled and which you cannot delete. 

  • This account has the username Administrator.
  • This account should be left disabled and unused [account profiles can only be corrupted when accounts are in use so by leaving it disabled & unused you are protecting it]. 
  • This account is a system recovery tool and has no other purpose.  It will automatically get enabled by the system & displayed at the Safe mode login if Windows detects that there are no other functioning Admin accounts on the computer.
  • Some people talk a lot of rubbish about this account. 
  • The Admin accounts you have created are more powerful than this Built-In Admin.  This was not the case in Windows 7; then the Built-In Admin was equally as powerful as the Admin accounts that users created.
  • If you have enabled the Built-In Admin then I would urge you to disable it again.
