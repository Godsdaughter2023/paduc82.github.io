Hi, Glenn.
 
When we boot from the DVD to install Windows 7 (or Vista), Setup has no idea what drive letters are already assigned, so it starts from scratch and assigns the letter C: to whichever volume (either a primary partition or a logical drive in an extended partition) you choose for the installation.  As J. If we already have a version of Windows, such as Windows XP, installed on the computer, we can boot into that OS and assign the drive letters that we want to each existing volume.  For example, we can assign the letter D: to the third volume on the second HDD.  Then we insert the DVD into the drive and run Win7�??s Setup.exe from the WinXP desktop.  In this way, Setup can see the letters that WinXP has already assigned and respect those.  When it asks where you want to install Win7, point it to Drive D:.
