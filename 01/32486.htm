j w stuart say:

Boot to Windows 7 and format the partition XP is currently installed on. (This assumes XP is on the 1st partition)

Since the Windows 7 bootloader (BCD store was on the XP partition) you will need to restore this to the Windows 7 partition.

Restart your computer with the Windows 7 DVD in the CD/DVD drive. wait for Windows to start (this may take a while if you have a multi-boot computers), Change your language.Time/Keyboard if necessary otherwise click 'Next'. At the 'Install Now' prompt (Do not pick pick the Install Now option) instead choose the 'Repair your computer' option located in the lower left of the dispay.

Next select the 'Use recovery tool' option and click 'Next'.
Now you will see a list of 'Recovery Tools'. Choose the 'Command Prompt' option.

At the DOS prompt type the following three commands:

bootrec.exe  /fixmbr

bootrec.exe  /fixboot (may return an 'Eliment not found' message)

bootrec.exe  /RebuildBcd

You should now see the number of Windows Installations that were identified
For a tripple boot computer, after XP has been deleted you will see the number 2 (the 2 remaining Windows 7/Vista installations)

If you see an 'Add installation to boot list?' prompt, enter  Y  (Yes) for each installation.

Remove the Windows 7 DVD. Close the command prompt window and click 'Restart' to reboot the PC.

Reboot and XP should be gone.

Methode #3
If the partition which XP was installed has been badly corrupted or deleted.

Restart your computer with the Windows 7 DVD in the CD/DVD drive. wait for Windows to start (this may take a while if you have a multi-boot computers), Change your language.Time/Keyboard if necessary otherwise click 'Next'. At the 'Install Now' prompt pick the 'Install Now' option and again wait for the 'Which type of installion' prompt.

Now select the 'Custom (Advanced)' option.
You should now see a list of all the existing partitions and unallocated space where Windows XP once resided.
Pick (highlight) the space and then in the lower left click on 'Drive Options'.

Now with the unallocated space highlighted click 'New' and 'Apply' and 'Format'.
Close out the 'Where do you want to install Windows?' prompt by clicking on the red X in the upper right.
Click 'Yes' to the ;Are you sure you want to cancle Windows installation' prompt.
Next click on the 'Repair you computer' option in the lower left.

Next select the 'Use recovery tool' option and click 'Next'.
Now you will see a list of 'Recovery Tools'. Choose the 'Command Prompt' option.

At the DOS prompt type the following three commands:

bootrec.exe  /fixmbr

bootrec.exe  /fixboot (may return an 'Eliment not found' message)
