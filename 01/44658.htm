theplectrum say: Dow load and install Audacity, a freeware audio editor. Install also the Lame mp3 codec that you can get from the same site.

Open it up and load up your mp3 file. Select > All > choose Export As from the FIle Menu > then export as an mp3 and choose a low bit rate. If the Current bit rate is 320 bps then choose 128 bps and make the name slightly different so you don't ver-write the original.

The volume of the new file will be much lower than the original (the higher bit rate) and may fit onto a CD. However, you will compromise the quality by using a lower bitrate.

Or :

If you don't want to compromise quality, using Audacity, locate a point in the mp3 file where you can cut the file into two parts, making sure that each part (700 mb maximum) can fit onto a CD.
