Once Safe Mode loads, open a Command Prompt Window

 

Press Windows Flag key + R to open Run Box

Type CMD

Press Enter

 

Type sfc /scannow

Press Enter

Note:  There is a space between sfc and /scannow

You may be asked for your installation media if corrupt files are found.  System File Checker will replace the files

 

Open CMD window again by following the steps listed above

 

Type chkdsk /F /R

Press Enter

You'll receive a message saying the volume needs to be locked and would you like to scan on restart.  Press Y and Press Enter

Restart computer to begin the scan

 

If there are corrupt files, chkdsk will repair them if it can

When the scan is finished, Windows will reload

 

Let us know if you need further help.

Pop-up Notifications in Windows 10 block the clock - Microsoft Community

nitramor say:

Pop-up Notifications in Windows 10 block the clock

I'm new to Windows 10 on a new machine. Like it mostly so far. One annoyance is the notification pop-up for appointments that comes up on the right lower corner of the screen. I'm having a couple of problems with it:

- One, it pops up 15 minutes before an appointment, but doesn't count the time down (like the Outlook reminders do, which tell you how many minutes remain or overdue the appointment is), so It's not really useful for knowing when it's time to pay attention to it.

- More importantly, it pops up right over the clock and blocks it so I can't see how close to the appointment I actually am.

Is there a way to shut off notifications like this - or at least move it a little so it doesn't block the clock?

Thanks!

a. user say:

Hi,

Thank you for posting your query in Microsoft Community. Appreciate your interest in using Windows 10.

There are two methods to disable notifications to pop-up.

Method 1:

  1. Click on Start>Settings>System>Notifications & actions
  2. Set the value as “off” for which the notification is not required.

Method 2:

       Click on Notifications icon in System Tray, click on Expand and select the option for Quiet Hours. This will disable all the notifications from popping.

Hope the information is helpful. Feel free to write to us for any other assistance with Windows, we’ll be glad to assist you.
