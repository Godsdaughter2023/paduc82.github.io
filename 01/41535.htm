a. user say:

Hi Thanatica,

Thank you for posting in Microsoft Community.

Have you done Clean installation or upgrade of Windows 10?

If you have done a Clean installation of Windows 10, then the product key which you are entering to activate the Windows will not work. Since product key is of Windows 7 and you have to purchase a key from retailer/ Microsoft store to get Windows activate. if you do a Clean install.

The error code which was provided shows that you are entering a wrong product code to get Windows activate. So, I suggest you to do a upgrade instead of Clean install.

You can install Windows 7 which will be a Clean install again and then activate it using key and then later try to perform upgrade to Windows 10. If you do an in place upgrade then it will be a free upgrade to Windows 10. But during this data may be lost. so, I suggest you to take the backup of data and applications which you feel is importanat and then perform installation.

You can refer to the Microsoft help article for more information and check if it helps.

http://windows.microsoft.com/en-us/windows-10/support

Hope it helps.

Thanks.
