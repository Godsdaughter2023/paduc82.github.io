a. user say:

Hi Stuart,

Welcome to Microsoft Community.

I can understand that you are unable to configure virtual port COM3 onto your notebook.

This can happen when virtual port is incorrectly installed and the new hardware is not located by the system.

  1. Did you make any recent changes on the computer, prior to this issue?
  2. Are you getting any error message?

Let’s try some steps to resolve the issue:

Step 1. Try to connect the device in to different USB ports and check.  

Step 2. You can also try the steps below and check.

  1. Press Windows and X key, select Control Panel.
  2. Click on Hardware and Sound, click Devices and Printers, if you see your device, right click on it and troubleshoot.

 Step 3. Check if the GPS receiver is compatible with Windows 8.1 or not.

Below is the reference link:

http://www.microsoft.com/en-us/windows/compatibility/CompatCenter/Home

If it is compatible and still the issue persists, try the next steps.

Step 4. I suggest you to uninstall the COM3 port from Device Manager completely and reinstall it.

  1. Press Windows and X key and select Device Manager.
  2. Expand Ports (COM & LPT), locate Communications Port (COM3).
  3. Right click and uninstall.

Now let’s try to reinstall it on the system.

  1. Open Windows Start Screen, in search option type cmd and right click on command prompt (cmd) and select Run As Administrator.
  2. Type hdwwiz.exe and press enter.
  3. The Add Hardware window will appear. Click on Next button in the Add Hardware window.
  4. Click on Install hardware that I manually select from a list (Advanced) radio button, then click Next.
  5. Select Ports (Com & LPT) and click on Next.
  6. Select (Standard port types) under the Manufacturer list and then select Communications Port. Click on Next button, then click on Next again. Click Finish to complete the Com port installation.
  7. Open the Device Manager again by pressing Windows and X key
  8. Expand Ports (COM & LPT), right click on new COM from the lists.
  9. Click properties.
  10. Click on Port Settings tab and then the Advanced button. Click the Com Port Number drop-down box and select COM3. Click OK , then click OK button again. "Com" port 3 is now installed.

azeez n say: Hi Tony Scarpelli,

If you switch Hard Disk Drive, Windows will perform a full back up since it doesn’t recognize the new disk. Windows will not do incremental backups at that point. But after that initial full backup, Windows backup should have no problem backing up to a different drive as long as they have the same drive letter.
     
You can change, add, or remove a drive letter by using Computer Management.
       
     
1. Open Computer Management by clicking the Start button, clicking Control Panel, clicking System and Maintenance, clicking Administrative Tools, and then double-clicking Computer Management. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

2. In the Navigation pane, click Disk Management.

3. Right-click the partition or drive that you want to change, and then click Change Drive Letter and Paths.

4. Do one of the following:

• To assign a drive letter if one has not already been assigned, click Add, click the letter that you want to use, and then click OK.

• To change a drive letter, click Change, click Assign the following drive letter, click the letter that you want to use, and then click OK.

• To remove a drive letter, click Remove, and then click Yes to confirm that you want to remove it.

Once you have done this, look in Computer, now simply right click to create a shortcut from there and it will put it on the desktop!

You may also want to look at the last post on this TechNet thread: http://social.technet.microsoft.com/Forums/en-US/w7itprogeneral/thread/d59e3120-9267-4c62-b0ac-2c62afe1da8e

If that did not work, try changing the drive letter using command prompt.
Follow the steps in the link to do that : http://technet.microsoft.com/en-us/library/cc753321.aspx

If the above 2 suggestions do not work, you may format the flash drive and then try and change the drive letter on the flash drive.

Note : In case you have important data on your flash drive, back it up first before formatting your flash drive
