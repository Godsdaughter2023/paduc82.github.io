balaji kundalam say:

Now, you're using Windows 7 with legacy mode, which means Windows 7 has been installed in legacy mode only. As I mentioned in my previous post, if Windows was installed in UEFI or legacy mode, it'll continue to work in that mode only and will fail to work if the mode is changed.

So, Windows 7 doesn't support UEFI and when you enable UEFI, the BIOS fail to boot Windows 7 as Windows 7 doesn't have the necessary boot files to support UEFI boot (legacy boot and UEFI boot involve different process). When an operating system fails to boot from the hard drive, BIOS will go for the next availble options, like USB boot or network boot and in your case, it's network boot and therefore, you're seeing the two options.

To resolve the issue, you have two options:

1. Continue using Windows 7 in legacy mode

2. Reinstall Windows 7 in UEFI mode (clean install, which requires complete formatting of the hard drive) to use Windows 7 in UEFI mode
