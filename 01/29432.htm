mike hall... always say:

Try a known working power supply on it first.

No LEDs? No initialization screen? No fans?

If the above is true, the motherboard is most likely dead. They tend not to be repairable..

 

Buying a new motherboard..

What make of computer do you have? What are the specs?

You need to know what is on the motherboard, processor make, socket type, memory type (DDR, DDR2, DDR3) to determine if you can re-use any of them. It is never too easy to get replacements for older motherboards from computer stores.

 

The computer manufacturer could replace the motherboard maybe, but it would be an expensive option and you would have to send it in to them. 

 

Another problem you have is with the Windows 7 installation. If you have a pre-installed OEM version of Windows 7, it will not activate on a motherboard that is not the same make and type as the original. You will have to buy another Windows 7 edition..
