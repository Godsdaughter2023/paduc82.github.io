a. user say:

Hi Milan,

Welcome to Microsoft Community Forum.

You may remove Ghost devices from device manager which may include drives for that unknown device.

When one remove hardware or a service, then sometimes Windows thinks that the hardware or service still exists and they roam about as ghost devices in the Device Manager. Ghost devices can cause slow
boot and other erratic behavior because Windows tries to communicate with a device which isn't there.
Show ghost devices in the Device Manager.

  1. Open a command prompt (cmd.exe) with administrator rights.
  2. Click on Start
  3. Type cmd in search box.
  4. Right click on cmd and click on Run as
    Administrator.
  5. Set the special environment variable with this command: SET
    DEVMGR_SHOW_NONPRESENT_DEVICES=1
  6. Start the Device Manager with this command devmgmt.msc.
  7. In the Device Manager enable Show Hidden Devices in the View-menu
    Now it should show all devices registered in your system, making it
    possible to remove any ghosts which shouldn't be there. Devices that are greyed
    out are inactive(not started), if the same device is listed multiple time then
    consider removing the inactive ones.
Important: You need to be careful while removing the Ghost
devices as if not done in correct way and uninstalled some other device driver,
that device would stop working.

Also run this fixit too:
http://support.microsoft.com/Mats/Program_Install_and_Uninstall

Hope this helps. Please get back to us if you have any more questions. We
are here to assist you.

How to back up and restore the registry in Windows
