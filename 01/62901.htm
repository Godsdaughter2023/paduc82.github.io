a. user say:

Hi,

 

This issue may occur because of issues with the connection protocols. Please refer to the steps below on how to configure your Gmail account to Windows Live Mail:

 

1. Enable POP in the Gmail account.

 

a. Sign in to the Gmail account.

b. At the upper part of any Gmail page, click Settings.

c. In the Settings page, on the Forwarding and POP/IMAP tab, click Enable POP for all mail or click Enable POP only for mail that arrives from now on.

d. In the When messages are accessed with POP list, click an option.

 

2. Configure the Gmail account in Windows Live Mail.

 

a. Start Windows Live Mail.

b. On the Accounts menu, click Email. If the user has already created a Gmail account, right-click the Gmail account, and then click Remove account.

c. On the Accounts tab, click Email to configure a Gmail account in Windows Live Mail.

d. Type your name, email address, and password.

e. If you want Windows Live Mail to remember the password, select the Remember this password check box.

f. Select the Manually configure server settings check box, and then click Next.

g. From the Server type list, select POP.

h. In the Ingoing server information section, in the Server address box, type pop.gmail.com.

i. In the Ingoing server information section, in the Port box, enter a port.

j. Select the Requires a secure connection (SSL) check box.

k. In the Port box, type 995.

l. For the outgoing server information, in the Server address box, type smtp.gmail.com.

m. Select the Requires a secure connection (SSL) check box.

n. Under Port, type 465, and then click Next.

o. Click Finish. Your Gmail account has been successfully created.

 

3. Test the Gmail account.

 

a. Return to the Settings page, and then click Save Changes.

b. In the folder list in Windows Live Mail, click the account.

c. On the Home tab, in the Tools group, click Send and Receive.

 

4. Configure the Gmail account in Windows Live Mail.

 

a. Start Windows Live Mail.

b. On the Accounts menu, click Email. If the user has already created a Gmail account, right-click the Gmail account, and then click Remove account.

c. On the Accounts tab, click Email to configure a Gmail account in Windows Live Mail.

d. Type your name, email address, and password.

e. If you want Windows Live Mail to remember the password, select the Remember this password check box.

f. Select the Manually configure server settings check box, and then click Next.

g. From the Server type list, select POP.

h. In the Ingoing server information section, in the Server address box, type pop.gmail.com.

i. In the Ingoing server information section, in the Port box, enter a port.

j. Select the Requires a secure connection (SSL) check box.

k. In the Port box, type 995.

l. For the outgoing server information, in the Server address box, type smtp.gmail.com.

m. Select the Requires a secure connection (SSL) check box.

n. Under Port, type 465, and then click Next.

o. Click Finish. Your Gmail account has been successfully created.

 

5. Test the Gmail account.

 

a. Return to the Settings page, and then click Save Changes.

b. In the folder list in Windows Live Mail, click the account.

c. On the Home tab, in the Tools group, click Send and Receive.

d. Windows Live Mail will display the status as the program synchronizes with your Gmail account.

 

Thanks!
