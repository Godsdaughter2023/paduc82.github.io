Does the issue occur on a specific file or project? Does the issue occur on other user accounts?

Also, most crashes are usually caused by outdated or corrupted display adapter driver. Try to update the display driver.

  • Select the Start  button, type Device Manager, and select it from the list of results.

  • Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver. For graphics cards, expand the Display adapters category, right click your graphics card and select Update Driver.

  • Select Search automatically for updated driver software.

If updating the driver did not resolve the issue, manually uninstall and reinstall it.

  1. Select the search box in the taskbar, enter device manager, and in the search results, select Device Manager.
  2. Select the arrow to the left of Display adapters and press and hold (or right-click) the Intel HD Graphics driver.
  3. Select Uninstall.
  4. Restart the device to reinstall the drivers.

Install the latest Surface and Windows updates.
