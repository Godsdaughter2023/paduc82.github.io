umesh p say: Hi CalicoCrow,     

Sorry for the pervious response. Queue length refers to the number of processes that are waiting to use the hard disk. Using the disk time percentage in conjunction with the disk queue length will tell you not only how much the hard disk is being used but also if the heavy usage is a problem. For example, if the disk time is 40 percent but the average disk queue length has a factor of two or less, then the hard disk is keeping up with the demand that’s being placed on it.

On the other hand, if the percentage of disk time and average disk queue length are both high (above 80 percent disk time and a factor of two or more for the average disk queue length), it means that processes are waiting for the hard disk to become available. Anytime a process has to wait for a component to become available, a user will experience less-than-desirable performance.

Disk represents the RAM.

      databaseben say:

      it is possible that the motherboard battery (cmos battery) is low.  if it is then the clues are that windows cannot maintain accurate time on the clock and the bios looses its settings.

      in any case, when you click f2 and the basic input output system (bios) screen pops up, it will display the hardware/devices physically connected to the motherboard.

      there is always a screen that displays the harddisk(s) detected and usually there is another screen that displays bootup order.

      harddisk0 is the main harddrive and you should see some fields with information like type of harddrive and how large it is.  if these settings are blank, then this is a possible problem. 

      so look around for legend or sub menu of f key assignements and press the one that sets the "defaults" and will subsequently populate empty fields in the bios with data. 

      also, check to see if you have a screen to assign boot up order and ensure that disk0 is first on the list and the cd drive is second.

      also, check to see if the time/clock is accurate. if it is not, then you will need to manually input the time.

      after the bios has its settings, the next f key to look for is "save and exit".
