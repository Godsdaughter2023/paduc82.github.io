I haven't heard of the windows.old troubleshooter but I know Windows.old is a folder created automatically when you perform a clean installation of Windows without formatting the hard drive or, in your case, refreshing the operating system.


Navigate to C:\Windows.old\Users\Your_user_name_folder\Desktop. Are your pictures there?


If not,


a) Do you have another hard drive or a second partition on the same drive? Maybe Windows was set to save by default user files on the second partition. If this is the case, your photos must be located at D:\Desktop or E:\Your_name\Desktop or something similar (where E: or D: the letter assigned to the second drive or partition). Can you find them there?


b) Did you run the disk clean up tool or has it been more than 28 days since you refreshed? The disk clean up utility can delete the windows.old folder depending on the options selected. Windows will also delete this folder after 28 days to restore disk space.


c) Are you sure you refreshed? There are two options: Refresh and Reset. Reset will wipe out everything, including user files, and then will reinstall Windows. Both operations are marked with buttons that display "Get started" so it's possible you pressed the wrong button.


If your case is b) or c) or none of the above and still can't find your files, try installing Recuva. It is a small utility that searches your hard drive for deleted files. You can easily set it to search by file type (eg. images). It is not a 100% certain that it will be able to recover them, especially if they have been overwritten, but it is the last thing you can do. If these files are of extreme importance, you can contact a technician or search for special companies that provide file restoration, but this service usually costs a lot.
