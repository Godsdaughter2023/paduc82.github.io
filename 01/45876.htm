Does system restore point is created automatically on Windows 8.1?

Original title: System Restore

Does system restore point is created automatically on Windows 8.1 (I'd recently upgraded from 8-OEM)? 

a. user say:

Hi Pankaj,

 

Restore points are automatically created by default just before significant system events, such as the installation of some programs, device driver, Windows Update, and by using a scheduled task once every seven days if no other restore points were created in the previous seven days. In Windows 8.1 also it is automatically created. You can also manually create restore points at any time.

You may check if the System Restore Protection is turned on by following the steps:

a. Press Windows key + X on the desktop Screen and select Device manager.

b. Open the Control Panel (icons view), click on the Recovery icon, click on the Configure System Restore.

c. Click on the System protection link in the left pane.

d. In System protection tab, check for Local disk C: if Protection is on or off.

e. If disabled, then you may click on Configure for configure restore points and select Turn on System Protection.
