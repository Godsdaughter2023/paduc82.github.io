Automatic Repair checks for the errors which is preventing Windows from booting up and will fix it. Create installation media for Windows 8.1

ethan b. say:
Hi HeglandHegland,

It's not unusual to be concerned about getting a new PC and seeing if your stuff works. I'm here to help, hopefully to ease some of these anxieties.

When the time comes, you'll indeed have to use Compatibility Mode, and I can tell you how to do it step-by-step. No web articles or anything.

Right-click on the shortcut once you get it installed (on the desktop, Start menu, wherever you see Photoshop) and click Properties on the bottom.

You will see your shortcut's Properties window. Click the Compatibility tab at the top.

There you see a whole page dedicated to compatibility mode. Check "Run this program in compatibility mode for:" and there's a dropdown menu. You can choose Windows 7 if you wish, since it last worked there.

If you run software on a 64-bit platform, they're backwards compatible with the 32-bit or x86 architecture, so it should generally be okay.
