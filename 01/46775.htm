NVIDIA GeForce GT 640M not compatible with Windows 8.1?

Since upgrading from Windows 8 to Windows 8.1, my display adapter which was working with Windows 8 no longer works and Device Manager shows :

 Additionally, whenever I run the manufacturer's driver update I receive the following error:


I did not run the windows compatibility for Windows 8.1 prior to installation as it seemed intuitive that the same devices that worked for 8 should work for the .1 upgrade. Upgrade to Windows 8.1 was conducted while connected to the internet.

Computer Model XPS L521X

Is there anything that can be done to allow me to access this display adapter hardware?

jseale say:

Oh sorry, my screenshots did not load.

Device Manager showed Microsoft Basic Display Adapter with the yellow triangle and Nvidia upgrade continually halted stating that no compatible hardware could be found.

I solved the issue.

Although the manufacturer's website lists the latest version of the  GeForce GT 640M driver it will not detect the existing hardware for Windows 8.1 WITHOUT the presence of the nVIDIA Graphics GeForce GT 630M/640M Driver version 9.18.13.2702 on the system.

This was found on the Dell website at http://www.dell.com/support/drivers/us/en/04/DriverDetails/Product/xps-15-l521x?driverId=NXJFD&osCode=WB64A&fileId=3290117795&languageCode=EN&categoryId=VI


Once this update was run and installed, the GeForce R331 Game Ready Driver was able to detect the hardware.
