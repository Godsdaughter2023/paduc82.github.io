regile reb say:

Hello James,

You may network your computers, but not using the USB ports. What you can do is set up HomeGroup in Windows 10. HomeGroup is a network sharing feature.

Note: To make sure you will be able to configure everything correctly, shut down all other the computers on the network, leaving only the computer you'll use to create the new HomeGroup turned on.

Additionally, you want to make sure all your devices are running Internet Protocol Version 6 (TCP/IPv6). Otherwise, you won't be able to join any computers to HomeGroup. See the steps below to set your devices to TCP/IPv6:

  1. Press Windows key + X > Click Network Connections.
  2. Right-click the network adapter you use to connect to the internet > Select Properties.
  3. Select Internet Protocol Version 6 (TCP/IPv6) option to put a check on the box on its left.
  4. Click OK.

To create a HomeGroup on Windows 10, follow these steps:

  1. Type HomeGroup and press Enter.
  2. Click Create a homegroup > Next.
  3. Choose what to share on the network > Next.
  4. Click Finish.

Note: You'll be presented with a HomeGroup password, which is needed to allow other computers to access files and printers. Write down or click the link to print the password.

Let us know what happens after following the suggested troubleshooting steps.

cmdrkeene say:

F1 to F12 are not assigned on a global level.  Every application gets to decide for themselves.  Example:  F2 is "rename a file" when you're inside a folder, but F2 is also "formula edit" inside Excel and "move current paragraph" inside Word.

There's no way to "unassign" the F2 key: it changes automatically depending on what program you're looking at.

When you press F5, the keyboard sends that command to the currently-active program.  The program can decide to do anything it wants with it (or ignore it entirely).  So the person who built the app you're using needs to add a "what to do when an F key is pressed" subroutine to the program.

BUT!!!  You can also use a program like AutoHotKey to create a global macro.  So you can set it so that when you press F3, AutoHotKey kicks in, gobbles up that command and sends a Ctrl-End signal instead.  Pretty cool functionality and it might do what you need.

Windows Update error 800f0826 is displayed when an update for Windows has failed to install  and is preventing you from installing additional updates. Error 800f0826 does not identify the update that has caused the problem. The update that has caused the problem will display a different error. This problem affects updates for Windows only and does not prevent you from installing or automatically receiving updates for other Microsoft programs, such as the Microsoft Office System.
