nick adsl uk say: Here are the steps to remove the UpperFilters and LowerFilters to fix the code 39 error in device manager:
1. Go to run (or hit Win+R), type Regedit and click OK.
2. Expand My Computer in Registry Editor, and then expand HKEY_LOCAL_MACHINE.
2. Expand SYSTEM, and then expand CurrentControlSet.
3. Expand Control, and then expand Class.
4. Under Class, click {4D36E965-E325-11CE-BFC1-08002BE10318}.
5. In the details pane of Registry Editor, on the right side, delete UpperFilters and also LowerFilters.

ronnie vernon say:

All I've been able to find online is how to enable the hidden admin account which is no problem. I want to make my current User the built-in admin. Is it possible for me to do that?

Hi

The special Built-in Administrator account has specific functions that cannot be applied to a normal Administrator account.

Example: If all other administrator accounts are removed or corrupted, you can boot into Safe Mode and the built-in administrator account will automatically appear on the logon screen and you can use that account to recover and create another admin account. This acts as a safety net for the system.

If you mean that you want to use this account as your normal account, this is not recommended. If something happens to this account you can lose all of the data on the system, if something goes wrong.
