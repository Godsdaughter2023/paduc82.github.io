  3. In the left column, select a listed program that you want to change its default file associations for.
  4. Select Windows Photo Viewer and click Set this program as default.

Hi Corina Maria,

Follow the steps below to update the display adapter:

  • Go to the manufacturer's website and save the most recent display driver to your desktop
  • Complete the steps below:
  1. Click on the Start button
  2. Right-click My Computer
  3. Select Manage
  4. Select Device Manager
  5. Expand Display Adapter by clicking on the + sign next to the device
  6. Right-click the adapter
  7. Select Update driver
  8. Select No, not this time from the options to connect to updates on the Wizard
  9. Click Next
  10. Select Install from a list or specified location
  11. Click Next
  12. Select Include this location in search and click Browse to go to where you saved the driver
  13. Click Next
  14. Once the driver is updated, click Finish
  • Try changing the screen resolution
