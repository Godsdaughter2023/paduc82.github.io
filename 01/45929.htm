xircal say:

Hi Michael,

There are two possible reasons why you would get code 87 when trying to run DISM.

The first one is that you ran it from an ordinary command prompt rather than an elevated one. The difference can be seen from the prompt. An ordinary users prompt looks like this:

C:\Users\Username>

And elevated command prompt looks like this:

C:\Windows\System32>

The other reason for getting the error is that you typed it incorrectly. In your opening post you posted that you ran online/cleanup-image/restore health. In that syntax, there are two missing spaces and a space where there shouldn't be one. It should look like this:

DISM.exe/Online /Cleanup-image /Restorehealth

So there's a space before each forward slash except the first one, or in other words "DISM.exe/Online" is all one word. If you copy/paste the one above into an elevated command prompt you'll see that it works. By copy/paste, I mean right click and paste, not Ctrl+V. The latter will also get you an error.

You can find an elevated command prompt quickly by right clicking the Start button and going to Command Prompt (Admin).
