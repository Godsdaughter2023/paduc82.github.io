Hi Kathryn

You cannot place an accent mark over an existing letter. You must use the information in the Character Map to copy/paste or use a keystroke to enter the letter with the accent mark already in place.

Go to Start / All Programs / Accessories / System Tools and select the Character Map.

Select the Font drop down menu at the top and then select the Font that you are using to type the text.

Locate the letter with the accent mark that you wish to use and select it.

You can either Copy/Paste that accented letter into your document or you can look at the bottom right corner of the Character Map window and note the Keystroke for that accented character.

Click inside the document to place the insertion point where you wish the accented character to appear and then use the number Keypad to enter the Keystroke for the accented character.

Turn the NumLock on to activate the number Keypad and press the keystroke combination on the Keypad. 

Example: To enter the accented y in the Verdana Font, the keystroke is Alt+0253. With the NumLock on, I press and hold the Atl key and type 0253 on the Keypad. When I release the Alt key, this is the result.  ý

Hope this helps.

Thank You for using Windows 7
