For anyone to troubleshoot the issue, they will need your minidump files.

1. Click "Start", input "SYSDM.CPL" (without quotation marks) in the “Search” bar and press “Enter”.

3. Copy all the memory dump files (looks like [Mini092008-01.dmp]) in Minidump to this folder.
4. Right-click on the Dump folder, click "Send To", and click "Compressed (zipped) Folder".

Finally a solution to the problem! I found it here:   http://www.sevenforums.com/media-center/10816-media-center-hangs-movies-movie-library.html

I took the .avi file and put it together with a .jpg file which I copied from here:  http://www.themoviedb.org/

When you find the image on the website, right click on the image. Then go to the bottom and delete the name in the box. Then when you save the image, rename it FOLDER.

I then combined the 2 files into a new folder that I renamed after the movie I had. 

When that's all done go back to 7WC and go through the process of adding folders in Media Library.

The icons will be found :)
