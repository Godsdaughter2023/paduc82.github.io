Quick question just to verify, you’re modem does not get to handshake with any fax number that you’re trying to fax to? Have you check your dialing rules if you’re setup properly?

 

From Article:

Troubleshoot fax problems

http://windowshelp.microsoft.com/Windows/en-US/Help/b180239b-e83f-4acd-8bcd-3341789b4dd31033.mspx#EJBAC

 

To be able to send a fax using a dialing rule, your recipient fax numbers need to be stored in the canonical address format for phone numbers. Windows Fax and Scan applies dialing rules only to fax numbers that are stored in this format. Canonical format describes the country or region code, area code, and subscriber number (fax number) for each recipient. You must also precede the country or region code with the "+" symbol so that the program recognizes the full canonical number, like this:

+Country/RegionCode (AreaCode) SubscriberNumber

For example, this is how you would enter a number for a subscriber in the United States in canonical address format:

+1 (425) 555-0123

 

For more information about canonical numbers, go to Canonical address format for phone numbers on the Microsoft TechNet website.

 

Please double check just to make sure that you don’t have any dialing rules that’s already setup. If you know that you’re setup is good, then I would say its you’re modem.

josephv_a say:

Hello Marcell,

Regarding your queries:

How could a person tell if they are blocked on skype?

There is no specific indication to know that one has been blocked on Skype.

Would they still be able to see profile picture?

Yes, they’d still be able to see the profile photo. However, if one has been blocked, it will prevent him from calling, his messages will not be received by the blocker and he won’t be able to see the blocker’s status.

Does last active still show when last online/active?

Once a contact is blocked, he will not be able to see the blocker’s status, moving forward. Thus it will only show the last day his contact was online, the exact day when he was blocked.

After how many days does skype show status as "days ago without saying for example 4 days ago?

When a user gets inactive for more than a week, he will be shown as "Last seen days ago" or if a user becomes inactive for a month or more, he will appear as "Last seen 30 days ago" on the app.
