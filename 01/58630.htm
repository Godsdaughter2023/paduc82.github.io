    Silvia,

    Honestly, Windows Movie Maker is a fragile application from Microsoft. It's important to note that installing system updates and other software, or reinstalling Microsoft Windows Essentials can derail Windows Movie Makers ability to handle certain media types; it's just a ubiquitous reality and sad nature of the beast.

    If you are using MP4 and AVHD; there's couple of factors that come into play there on whether or not the video files are properly formed; to say there's no defects in the source video files.  As a suggestion, if not already, copy all the SOURCE media you want to use with Windows Movie Maker to you local hard drive. Create a folder and put everything in that folder so that Windows Movie Maker has access to it as fast as possible. Grabbing source data from external sources like jumpdrive, External Hard Drive, SDCards, etc and result in performance issues; causing undue delays and stress. 


    Being able to import media directly in to Windows Movie Maker is the most efficient process [using the proper codecs], but if you are running in to issues with being able to work with various types of media, you might want to think about converting the media to something ubiquitous to Windows Movie Maker. Realizing that the conversion of video has been suggested before. 

    I personally would not convert video to another format until I've tried testing loading the proper codecs. Windows Movie Maker technically supports many formats; but what's not provided to the public [regular users] is that Windows Movie Maker doesn't recognize those various formats unless the CODEC for that format is installed in the operating system; it's not built-in to Windows Movie Maker. And Windows Movie Maker doesn't make any attempts to identify, collect or load those codecs for you; Microsoft leaves that to you to resolve.
