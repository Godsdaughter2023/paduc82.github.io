lonely wanderer say:

I had the same problem and I found that even after uninstalling Daemon tools with Revo Uninstaller, it proved unsuccessful. The solution that I have found is by disabling the shell integration of Daemon tools. You may have to uninstall Daemon tools with Revo Uninstaller beforehand, I had to use Forced Uninstall within the program.

You can download a shell extension manager at http://www.nirsoft.net/utils/shexview.html.

After downloading, open the zip file and run shexview.exe, run down the list and select:

-DaemonShellExtDrive Class

-DaemonShellExtImage Class

-Image Catalog

You can disable them by going to the upper left corner and selecting File, then Disable Selected Items.

After following these instructions, Windows Explorer should run fine and there would be no more pop-ups.

Hi,

A couple of things...

It is difficult to find a download of a specific build of Windows 7... The link from Microsoft requires a valid retail key... I doubt that the one from Dell would work... You might check with Dell.. Here is the Microsoft link...

https://www.microsoft.com/en-us/software-download/windows7

Before you reinstall anything, you need to understand about the difference's between Legacy Bios and UEFI Bios, and determining which your machine is using... or which you prefer to use...

The installation methods are a bit different, and some of the steps are not interchangeable...

Does your HDD have a small 100-250MB EFI Partition ? If so you have UEFI Bios.. If it does not, you have Legacy Bios...

I point this out because booting and installing from a Installation DVD when using UEFI Bios is a bit tricky... The installation will normally try to install as Legacy Bios, and then fail because the HDD is not created properly..

You might try opening Boot Options at the Dell screen and look for both a "CD\DVD Drive" and a "UEFI CD\DVD Drive"...

If your machine has four partitions I'm guessing that it is using UEFI Bios, but that is just a guess...

As far as deleting the partitions and etc. to start with a blank disc...

You can do this by selecting a "Custom Install"... you will get that option during the Installation process..

I don't know if is a good idea to delete everything... There may be something on one of the partitions from Dell that you may want to keep...

You can just point your installation to the Windows partition and install there... All of your old windows files will be saved in a C:\Windows.old folder that can be cleaned later...
