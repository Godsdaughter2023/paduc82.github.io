If you currently have a qualifying operating system installed on a computer at home (Windows 7 w/SP1, Windows 8.1 with Update 1 - neither "Enterprise") - then you qualify for a free upgrade to Windows 10 on that home computer.  If you have 10 computers at home with a qualifying operating system (Windows 7 w/SP1, Windows 8.1 with Update 1 - neither "Enterprise") - then you qualify for a free upgrade to Windows 10 on each of the qualifying home computers. 

You will have to upgrade each one individually. 

Personal information does not have to be involved.

Once you upgrade a given computer to Windows 10 from whatever qualifying OS it was before - _that system_ will forever (freely) have rights to the Windows 10 OS you installed on it.  Change the motherboard (the part that ties everything else together) and it will be a new system - so - you'd have to treat it as such.  Change the memory, hard drive, video card, network card, audio card, etc. - no problem (unless one of those involved changing the motherboard.)  The 'free' upgrade is tied to that particular system for the 'life of the system' - the definition for which, in the past by various EULAs, the 'system' is the motherboard.
