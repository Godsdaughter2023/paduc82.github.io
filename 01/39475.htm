a. user say:

Hi,

A metered connection is an Internet connection that has a data limit associated with it. Mobile data connections are set as metered by default. WiFi network connections can be set to metered, but aren’t by default. Some apps and features in Windows will behave differently on a metered connection to help reduce your data usage.

Follow the below steps for the metered connection.

1. Type Settings in the Search bar and then Enter.

2. Click on Network and Internet.

3. Click on WiFi on the left pane.

4. Click/tap on the Manage known networks link under Wi-Fi on the right side.

5. Click/tap on the wireless network name you want to manage, click/tap on Properties

6. Turn on or off to Set as metered connection for this Wi-Fi network.

I also suggest you to ask the ISP provider regarding to limit data received to 500 kbps.

Refer to the article on Metered Internet connections: FAQ and check if it helps

tauheedul ali say: Check that your laptop settings have the correct keyboard layout set for the type of keyboard you are using.

If you're in the UK, you need to have the UK Keyboard Layout, and if you're in the US, you need the US Keyboard layout set.

On Windows 7 or Windows Vista
  1. Click Start, type intl.cpl in the Start Search box, and then press ENTER.
  2. On the Keyboards and Language tab, click Change keyboards.
  3. Click Add.
  4. Expand the language that you want. For example, English (United States).
  5. Expand Keyboard list, select the United States-International check box, and then click OK.
  6. In the Default input language list, click  - United States-International (where is the language that you selected in step 4), and then click OK two times.
  7. In the Regional and Language Options dialog box, click OK. 

    Notice that the Language bar appears on the taskbar. When you position the mouse pointer over it, a ToolTip appears that describes the active keyboard layout.
  8. Click the Language bar, and then click United States-International on the shortcut menu that appears.
  9. The United States-International keyboard layout is selected.
Or, see how to do it here...
http://support.microsoft.com/kb/306560
