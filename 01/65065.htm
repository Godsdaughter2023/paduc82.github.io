Hi Paul,

I was able to replicate your issue and fix it.

It does involve some registry editing, be aware making wrong changes to te registry might get you into more trouble so please read carefully before you do this

start regedit.exe

The target for 'downloads' is located here:

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders\{374DE290-123F-4565-9164-39C4925E467B}

This value probably now points to your documents folder, edit it to point to your downloads folder

Do the same for this key:

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders\{374DE290-123F-4565-9164-39C4925E467B}

and sign of and on again (or restart)

That should fix the target location for your 'downloads'

You might still see the 'Documents' display name next to the Downloads Icon, if so delete the desktop.ini file in your downloads folder (this is a hidden system file) and restart or sign off again.

Hope this helps you,
