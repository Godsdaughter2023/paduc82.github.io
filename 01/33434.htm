Desktop Start Menu has dupicate items - Microsoft Community

gibsoncarter say:

Desktop Start Menu has dupicate items

The Desktop start menu is displaying duplicate items.  How can I edit this menu?


 

kevin d souza say:

Hi Gibson,

 

Please go to the following location and delete the duplicate entries you have there.

 

C:\Users\Username\AppData\Local\Microsoft\Windows\WinX\Group3

 

Note: user name is the user account using which you have logged into computer.

 

Appdata is a hidden folder. So to get to the above location, you need to enable the hidden folders. To do that, in the File Explorer window, click on the View tab and place a check mark for Hidden folders.

 

Once you have deleted the duplicate entries, restart Windows Explorer process from task manager.

- In Task manager, search for Windows Explorer process. Right click on it and select End Task.

- In the File menu, click on Run new task. Type Explorer.exe and press enter.

- You may even restart the computer so that the changes are applied. (Restarting Windows Explorer is a must so that the changes can be applied).

 

If you face any issues, you may always download any third party Win X menu editor software to add/delete the Windows Start menu entries.
