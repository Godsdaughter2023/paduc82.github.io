jeffrey brooks say:

Task Scheduler is corrupt

After deciding to go back to Windows 7 from Windows 10, my task scheduler gives me a message that, the task scheduler is corrupt or has been tampered with". Is there a software program to correct this, or must I learn how to manually correct this problem? Please help. 

a. user say:

A popular issue with similar cases. 

In this MS Community I have never seen any one of the Microsoft engaged Support Engineer "experts" fully comprehend or successfully resolve the entire issue but there is very good hope that has worked well in many cases when you understand a little about the problem.

The problem is that when reverting back to Windows 7 sometimes the registry parts of the Scheduled Tasks are restored but the Windows 10 tasks files themselves are not restored so now there is a conflict and hence the corruption - things are not matching up like they should.  The problem usually surfaces first when folks try to set their backup schedule up again.

The chances are very high that ALL your Scheduled Tasks are corrupted (about 80 of them) and some of them are pretty important if you want your Windows 7 to behave properly.  You can open Task Scheduler and poke around in some of the Tasks and see if they are functional or do they also report corruption?

Click the Start button/orb and in the box enter:

Task Scheduler

Choose to Run as administrator:

How many tasks are reported as corrupted?  All of them?

Some nice person named Dijji has already figured out this problem and developed a nifty method/program to straighten out the Scheduled Tasks and people report that this method works pretty well.

It might help to understand the problem better if you to read through the Home and Documentation tabs first so you can walk through the repair process.  Note that there is a separate Discussion tab where folks report success, issues, problems and questions.  The Discussions are helpful too when you read them and see what kinds of things folks like you have run into and how to fix them.

You can read about it here:

https://repairtasks.codeplex.com/

In the Windows 7 community most of the time the CodePlex method works just fine (whew!).

Examples:

http://answers.microsoft.com/en-us/windows/forum/windows_7-update/task-scheduler-multiple-image-is-corrupt-or-has/6205cb04-a749-4f4f-a5be-632e01989158

http://answers.microsoft.com/en-us/windows/forum/windows_7-update/task-image-is-corrupt-or-has-been-tampered-with/2d384c24-ed5c-4958-a531-80c8cce32a66

But if there are any issues or problems consider pursuing them in the Discussion tab on their page first (they know what they are doing).

Finally you have to consider that if reverting back to Windows 7 corrupted all your Scheduled Tasks what other things on your system are now afflicted/broken that you don't know about yet?  In the future if you have a problem are you always going to wonder if the problem was caused by some Windows 10 leftovers?

Some frequent contributors in the Windows 7 community feel the only way to be 100% sure your Windows 7 system is whole again is to backup all your personal files and folders, whack your hard drive and do a clean/fresh install of Windows from scratch but that is a lot of work.

http://windows.microsoft.com/en-US/windows/downloads/upgrade-advisor

2. Backup all your files and settings to avoid losing docs, photos, and other information. Windows Easy Transfer is a free tool to copy files (not programs) from your PC, and then transfer them back after you install Windows 7. If you don't want to use Windows Easy Transfer, you can copy your files to a USB drive, or CD/DVD. Note: Easy Transfer will not work when changing languages, or when going from 64 bit to 32 bit (it will work going from 32 bit to 64 bit).

http://windows.microsoft.com/windows-easy-transfer

4. Put the Windows 7 disk into your DVD drive. A window should pop up starting the install process, when asked the type of installation do you want click Custom (advanced). Note: 32 to 64 bit requires you to restart and boot from the DVD.

5. After Windows 7 is installed you can use Windows Easy Transfer (or another backup method you used) to restore your files & settings, and use your installation disks to reinstall all your programs. Then you're done - enjoy Windows 7.
