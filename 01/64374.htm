lemp say:

Unless you're planning to use this machine as a stand-alone device not connected to the Internet, it's not worth your time and effort.  Microsoft stopped supporting Windows XP -- i.e., stopped provided security updates -- in April 2014.  It's not particularly safe to use a Windows XP machine on the Internet.  In addition, many current versions of programs won't work on XP.

If I were you, I'd install some version of Linux instead.

That said, different computer manufacturers used different mechanisms to recover to factory condition.  The best way to determine this is to search on the web site of the computer's manufacturer.  The recovery mechanism might be a set of CDs that came with the computer or it might be a partition hidden on the hard drive, which is invoked by pressing some Fx key during the initial boot-up phase.  For example:

Acer – Alt + F10

Asus – F9

Dell/Alienware – F8

HP – F11

Lenovo – F11

MSI – F3

Samsung – F4

Sony – F10

Toshiba – 0 (not numpad) while turning on, release key when Toshiba logo appears

If these keys don't work then look out for a message in the screen during startup which might indicate how to start the recovery process.

If the recovery mechanism was through CDs rather than a hidden partition, you'll have to call the manufacturer, although odds are very low that they will supply XP recovery CDs anymore.

Any free trial antivirus software must be removed using the antivirus manufacturer's removal tool.  The trial period will have long since expired.

In addition, if the original installation of Windows XP did not include Service Pack 3, you would have to download and install Service Pack 3 (and Service Pack 2, unless the computer already has SP 1 or SP 2).  These are only available from the Microsoft Update Catalog:  http://catalog.update.microsoft.com/v7/site/Search.aspx?q=xp%20service%20pack

Once Service Pack 3 has been installed, you'd have to visit Windows Update and install all of the critical updates issued after SP 3 and before April 8, 2014.

And to repeat, even after installing SP 3 and all of those updates, Windows XP would still be unsafe to use on the Internet.

One of the first things you need to do before deciding on any backup solution is what you require of that solution when (if) things go wrong. There are two main reasons for a restore being needed...
