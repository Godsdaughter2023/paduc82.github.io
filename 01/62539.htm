My HTC Desire X can use HTC Sync Manager but I have tried both and neither of them is very useful software.  Both of them, for example, sync IE Favorites into a flat list in a single folder [i.e. they lose the folder structure that you might use in IE Favorites to be able to manage the potentially large number of items you have in there].

Good third-party syncing software - MyPhoneExplorer, BookmarkSB & MeercelRSSDemon

If you install HTC Sync, you will find the drivers useful as they will enable you to use third-party sync software such as MyPhoneExplorer.  MPE enables direct syncing over a USB connection so you are not using up your phone's data allowance syncing with your own data [You can also sync using your phone's WiFi Hotspot or a WiFi network].  I have used MPE with both Android 4.0 & 4.1.  You may also want to see their English language forum section for the very useful guide to setting things up.  If you install MPE while the phone is connected then the necessary MPE "client" should automatically install onto the phone as well - otherwise go to MyPhoneExplorer Client - Android Apps on Google Play to get the client.

Syncing MS Office - Outlook 2007

MPE can sync Outlook Calendar, Contacts, Notes & Tasks.

Syncing IE Favorites

If what you are trying to sync includes IE Favorites [Bookmarks] then you can use MPE to sync your IE Favorites exported list [I can explain that if you want] to the phone then use BookmarkSB to import your IE Favorites through to the Android's default browser [and then, if applicable, Chrome can import them into itself from there].   

Syncing IE RSS Feeds

I export my IE RSS Feeds into an opml file then sync that file to the phone & import it into Meercel's RSS Demon.  This is cumbersome but I don't add extra RSS Feeds very often so it is not too much bother. 

Syncing other files

I use MPE for all my syncing including music & Office-type files [I don't know if your phone comes with an MS Office-equivalent app].  You'll want to look at MPE, File, Settings, MultiSync.  If you have  a particular example of syncing that you want to do then post it here & I'll try to explain useful settings for it.
