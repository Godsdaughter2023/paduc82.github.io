aileen alf say:

In this case, you might find the following suggestions helpful:

Method 1: Rebuild your search index.

  1. Press Windows key + R, type control.exe srchadmin.dll, and click on OK.
  2. Click on the Advanced button.
  3. From the Advanced dialogue box, select Rebuild. Selecting Rebuild will begin the process of deleting, and then rebuilding your PC’s search index.

Note: While the steps to initiate the rebuilding process are simple, the actual process can take quite some time depending on how much data Windows has to index.

Method 2: Create  a File Explorer shortcut on your desktop.

  1. On your Windows 10 desktop, right-click with the mouse.
  2. Select New > Shortcut.
  3. Type explorer.exe in the Location box.
  4. Click on Next button.
  5. Click on Finish button. You should now have a shortcut on your desktop.
  6. Right-click the new shortcut, and select Properties.
  7. Under Target line, change it to (note the space between C:\Windows\explorer.exe and /n):
    C:\Windows\explorer.exe /n
  8. Click the OK button. Now you can double-click it like your regular File Explorer app.

Method 3: Reset the File Explorer view to the default settings across all your folders.

  1. Launch File Explorer, click the File menu, and select Change folder and search options.
  2. Click on the View tab.
  3. Under Folder views, click on Reset Folders.
  4. Click the OK button.

Hi Scott, you have 2 options - neither will break any Microsoft Terms and Conditions!

Use a 3rd party software to clone the 128GB SSD onto the 512GB SSD. In truth this does cause complications in a lot of cases. it causes system failures, driver problems and sometimes it is hard to get back the rest of the 512GB SSD


Your best option by far is a clean install of Windows 10 onto the 512GB SSD

This method is a lot more work, but will result in a more stable and resilient system!

Backup you files from the 128GB SSD to the 1TB HDD

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

IMPORTANT
Make sure you install the same Edition of Windows 10 (Home, Pro . . . etc.) you previously had installed to avoid Activation issues

Click this link:
https://www.microsoft.com/en-gb/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Remove the SSD and the HDD from your System

Connect the new SSD to the same SATA port on your Motherboard

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

Begin the installation . . .

During the installation process, your system will reboot, at that point, you may need to go into your BIOS and change the boot order so your system boots from your Hard Drive, otherwise the installation may start all over again - do not remove the installation media at this point - only remove the installation Media when you reach the Desktop

Once Windows is installed, load any required drivers and put it on the Internet so it Activates from the Microsoft Servers

Finally, shut your PC down, reconnect the 1TB HDD and start your system again

Yes, this method will take time, but you will end up with a stable and responsive system !

andre da costa say:
Open Mobile hotspot settings (Settings > Network & Internet > Mobile hotspot)
Turn on mobile hotspot
Under "Related Settings", click Change adapter options
Take note of which adapter is connected to the internet and which adapter is your mobile hotspot (typically named "Local Area Connection* [#]")
Right click the adapter that is connected to the internet and click Properties
Go to the Sharing tab
Uncheck "Allow other network users to connect through this computer's Internet connection". If it is already unchecked, skip this step.
Click OK
Again, open Properties > Sharing for the same adapter.
Check "Allow other network users to connect through this computer's Internet connection"
(If you get a popup that says "Internet Connection Sharing is currently enabled...", click OK.)
Under "Home networking connection:", select the mobile hotspot adapter
Click OK

Open Start then type: regedit
Right click Regedit then click Run as administrator
Navigate to the following registry key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WlanSvc\Parameters\HostedNetworkSettings\

Delete the following key:
HostedNetworkSettings

Restart then try again.
