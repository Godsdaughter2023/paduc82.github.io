Windows Update error code "0x643" and Windows Installer error code "1603" are generic error codes. Typically, these error codes mean that the update was not installed. These errors codes are usually caused by a corruption in the .NET Framework installation or by an inconsistency on the MSI database state.

To do this, follow these steps:

  1. Click the following link to download the .NET Framework Cleanup Tool:
    http://blogs.msdn.com/b/astebner/archive/2008/08/28/8904493.aspx (http://blogs.msdn.com/b/astebner/archive/2008/08/28/8904493.aspx)
  2. Use the .NET Framework Cleanup Tool to uninstall the .NET Framework versions 1.0, 1.1, 2.0, 3.0, and 3.5
    Note The cleanup tool does not let you remove the .NET Framework 2.0 in Windows Vista or later versions because the .NET Framework is installed as an operating system component.
  3. Restart the computer.
  4. Download and install the following components:
    • .Net Framework 1.1
      http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=262d25e3-f589-4842-8157-034d1e7cf3a3 (http://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=262d25e3-f589-4842-8157-034d1e7cf3a3)

reece coombes say:

Hi,

Since it stopped working straight after the upgrade, I am assuming it's a driver issue. First, restart the PC. If this doesn't work, try the following methods

Method 1: Update the driver

  1. Search Cortana for 'Device Manager' and hit enter
  2. Expand 'Disk Drives'
  3. Right click on your drive's driver
  4. Click 'Update Driver Software...'
  5. Select the 'Search automatically for driver sofware' option and wait for it to finish
  6. If it says that it installed a new driver, disconnect (if already plugged in) and reconnect your hard drive
  7. See if it works now - if not, try method 2

Method 2: Roll back the driver

  1. Open Device manager like in Method 1
  2. Locate your drive again, but instead of updating it like before, right click and select 'Properties'
  3. Go to the 'Driver' tab
  4. Click on the 'Roll back driver' button
  5. Wait for it to complete
  6. Disconnect and reconnect the hard drive and see if it works now

Well, the Toshiba Satellite L750-16X is a discontinued model.  I doubt that Windows 10 drivers are forthcoming from Toshiba.  I do see some sites with Windows 10 Toshiba Satellite drivers.

The specifications page for the Toshiba Satellite L750-16X does say that there is an internal webcam with a built in microphone.  Does the microphone work?  Internal hardware does go bad.  But I would still like to have you go through the steps of reinstalling the webcam driver.

Look to see if the microphone is enabled, then test it.  Right click on your volume icon in the system tray.  Click on Recording Devices.  Do you see a Microphone listed?  Right click on the white space on the Sound window and make sure checks are in the two boxes.  If the microphone is listed, click on it and click on the Properties button.  At the bottom of the window, make sure the microphone is enabled, click apply then ok if you need to enable it. 

This webcam is integrated on the motherboard as is the graphics card. The microphone is included on the webcam component so if the microphone is also not working, this could be bad hardware.
