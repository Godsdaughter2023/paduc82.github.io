xb1r athena say:

Port Forwarding

Login Information:

  • Default Gateway - 192.168.100.1 / 192.168.0.1
  • Username – user / admin / root
  • Password – user / cableroot / root

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. On the left had side click "Forwarding"
  5. Enter in the ports and IP address for the Xbox
  6. (In the "Public Interface IP" section, just leave it at 0.0.0.0.)
  7. Select Apply.

Disable Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.0.1 or http://192.168.100.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. (user/user, admin/cableroot, root/root)
  3. Select Gateway from the menu at the top.
  4. On the left had side click "Forwarding"
  5. Uncheck the Enable box of the ports you want to remove.
  6. Select Apply.

The DRIVER_POWER_STATE_FAILURE bug check has a value of 0x0000009F. This bug check indicates that the driver is in an inconsistent or invalid power state.

http://msdn.microsoft.com/en-us/library/windows/hardware/ff559329(v=VS.85).aspx

 

This behavior occurs when drivers do not handle power state transition requests properly. The error message most often occurs during one of the following actions:

  • Shutting down
  • Suspending or resuming from Standby mode
  • Suspending or resuming from Hibernate mode
