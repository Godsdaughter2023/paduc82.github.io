Most laptops have a VGA port if not an HDMI, so you should be able to attach a monitor to start it and copy out your files, even use it as a desktop. 

The external monitor won't work in boot mode so methods to Rescue Files when Windows Won't Start  using Boot disks wont' work.

Have you tried disassembling the laptop enough to check that it's monitor cable is connected firmly ,as these can come loose.  Most laptops have disassembly videos or tutorials which are searchable on the web using [make] [model] disassembly or substitute disassembly with replace Monitor, replace hard drive, add RAM, replace heat sink paste, replace CMOS battery, etc. which will often give you enough to access the monitor cable that is often one of the first pieces that must be removed.  Another model found in such a search may be similar to yours. 

If you're going to junk the laptop anyway you might as well remove the monitor to see if you can reattach its cable or replace it with another monitor, even temporarily.

A VGA cable is only to connect an external monitor on most laptops.
