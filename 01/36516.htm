Hey there Trophic!

You can do as ROZILLA suggested, this will work as well. However, you need to direct your Xbox Primary DNS to a DNS server. This can not be done by having your Xbox DNS settings set to automatic. The DNS server is your routers IP address or a public DNS server. You can open the Terminal window and type "dig" (without quotes). There you will find "Server" followed by its IP address (the router) and port number (#53) listed as well. Like this: ;; SERVER: 192.168.1.1#53(192.168.1.1).

You do not need to setup manually DHCP, only configure with DHCP. Open Terminal window and type ifconfig en0 (en0=wired, 0=zero). Below your APIPA address (169.254.xxx.xxx) you'll find the IP address for the airport (mine is currently 192.168.2.1). For your Xbox IP address you need to type in your inet number but with the last number being one digit higher (e.g. 192.168.2.2).

You will have a moderate NAT when sharing your internet connection using a Mac.
