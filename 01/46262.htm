If you are sharing music form Windows Media Player you will first need to stop the Windows Media Player Network Service.
Click 'Start' > Type services in the search field > Click the top result 'Services'

If you receive a permission denied error the “WMPNetworksvc” may not have been stopped or you may need to restart your machine and start again.

Once the wmdb file has been deleted, restart Windows Media Player and the database should begin to repopulate.
