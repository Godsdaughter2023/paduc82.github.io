Expanding a little on the other contributions, here is what you could to:

  1. Run these commands just once on each workstation:
    net  use  *  /del  /yes
    net use  /persisten:no
    They will prevent Windows from remembering shared connections and avoid the error messages.
  2. Place this batch file into the Startup folder you seel below:
    @echo off
    timeout  10 > nul
    net use  S:  \\YourServer\Staff
    net use  H:  \\YourServer\HumanResource
    The first command will insert a delay of 10 seconds.
    Here is the Startup folder:
    C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

If the machines are in a domain then you should adjust the domain logon script accordingly.

    2. Click on Hardware and Sound.

    4.Your HDMI driver will typically be named High Definition Audio Device. If you have a graphics card, it may have the name of the graphics card’s manufacturer before it.

    7.Windows will then search for a new driver.
