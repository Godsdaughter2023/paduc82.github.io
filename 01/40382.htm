If you are running Windows 10 version 1709 or later, the new way to enable and manage Remote Desktop is through the Configuration application. To check, click the Start button and select Settings (the gear icon in the lower left of the Start menu). Then, from the Configuration window, select System and choose Remote Desktop from the list of options on the left.
