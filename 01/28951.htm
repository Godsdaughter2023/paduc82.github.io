What is the difference between Windows 7 OEM and Windows 7 Retail - Microsoft Community

steve-5 say:

What is the difference between Windows 7 OEM and Windows 7 Retail version?

Can someone tell me, what's the difference between windows 7 OEM and retail vs ?
andre da costa say:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

 

What is OEM software?:
http://support.microsoft.com/gp/oemsupport_1/en-gb

Licensing FAQs:
http://www.microsoft.com/OEM/en/licensing/sblicensing/Pages/licensing_faq.aspx

 

With a retail version, you can transfer it to another computer, change your motherboard.

http://www.microsoft.com/oem/en/licensing/sblicensing/pages/licensing_for_hobbyists.aspx

j w stuart say:

You can buy either the Retail or OEM or your computer may have come with Windows 7 (OEM) pre-installed. Both will run just

fine and there is no difference as far as the "end user" (you in this case) in the way Windows performs.

If you purchase an OEM version DVD you either must specify either 32 or 64-bit media as only one Windows 7 install disk is included. The license ties it permanently to the first computer it's installed on. It can never legally be moved to another computer, sold, or given away (except with the original computer).

Retail Boxed version include both a 32 and 64-Bit DVD however only one may be installed and activated.
Retail (full or Upgrade version) is transferable to another or new computer as long as it removed from the original computer.

More info on this below.

1) For the OEM DVD version that you purchased: "You" provide your own support. However you still can use forums or news groups. For a pre-installed OEM version the PC manufacture provides the support.
2) The OEM version DVD is either 32 or 64-bit (not both).
3) OEM versions cannot be used to upgrade a previous version of Windows (XP or Vista).
4) All OEM versions (DVD or pre-installed by the Vendor) lives and dies with your motherboard.
Should the motherboard fail then the OEM software will not activate on a replacement motherboard unless it is an identical make, model, revision, etc and that's not a 100% certainty as sometimes a small change in the motherboard design even it’s the same replacement part number is still considered a different board and Windows OEM will not activate. o your OEM is now worthless.
5) Once installed, you can not to move/install an OEM version to another PC or non identical replacement motherboard.
6) Retail (full version as you called it) is transferable to another computer or the same computer when upgraded with "any" make and model motherboard, the OEM version is not.

Case in point I have a retail version of Windows XP which has been moved from the original 600MHz P3 computer to a 900MHz P3 and then a 2.8GHz Intel P4 and currently a Dual Core 3.3GHz Intel Wolfdale. So my initial outlay of a few dollars more has repaid itself many times over the last 11 years.

For a cost conparison the above mentioned hardware cost me more than $4,000 the XP software or in todays prices for a "Full" retail version of Windows 7 Ultimate $300.
So trying to save a few Dollars or Euros on the OEM version make no cents, pence or loonies.


Also if you are building your own PC, I suggest if you do decide to buy the OEM version don't activate Windows (disconnect the internet/Lan cable) until you are certain there are no problems with the hardware.

andre da costa say:

Product keys are not architecture specific, they work regardless its 32 or 64 bit, as long as it is the same edition. Changing editions will not change the architecture either.

Windows 7 64 bit comes in handy when you need to address at least 4 GBs or more of RAM. Windows 7 32 bit can utilize up to 3.2 GBs of RAM. Because the memory address space is much larger for 64 bit Windows, that means, you need twice as much memory than 32 bit Windows to accomplish some of the same task, but you are able to do so much more, you can have more applications open, do things like run an Antivirus scan in the background without it affecting your system performance. Windows 7 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 7.

As for compatibility, you will need 64 bit device drivers for any hardware devices you might have. Also, there is no 16 bit subsystem in Windows 7 64 bit, which means, your applications must be 32 bit only, not 16 bit installer or uninstallers. Also, if you decide to move to Windows 7 64 bit in the future, there is no upgrade path from 32 bit Windows, clean install only.

 

See instructions how to perform a custom install:

How to: Perform a custom installation of Windows

What sort of access does your user have to these locations?  The Start Menu's user location is

C:\Users\usernamexxxx\AppData\Roaming\Microsoft\Windows\Start Menu\  for the specific user.

While the general Start Menu used by most installers and accessed by all the users is C:\ProgramData\Microsoft\Windows\Start Menu.
