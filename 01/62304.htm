shutdown  /s  /f  /t 300

The command will generate a pop-up panel at the specified time, then shut the computer down 300 seconds (=5 minutes) later. To find out how to create a scheduled task, click Start, then Help, then look for help on scheduled task. When creating the task, make sure it runs under the "System" account. Initially you should specify a time 15 minutes from now so that you can test the task shortly after creating it.

If you find it daunting to create a scheduled task then here is a simpler method:
1. Log on under an admin account.
2. Click the Start globe.
3. Type the three letters cmd into the Search box.
4. Press Ctrl+Shift+Enter
5. Click "Run as Administrator".
6. Type the following commands and press Enter after each:
    at   22:00  /every:m,t,w,th,f,s,su   shutdown.exe   /s   /f   /t 300
    (to create the task described above. It will run at 10 pm.)
    at
    (to see a list of all tasks created with this method)
    at  1  /del
    (To delete task #1)
