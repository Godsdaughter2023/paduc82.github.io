could u explain forward look up and reverse lookup - Microsoft Community

server8 say:

could u explain forward look up and reverse lookup

could u explain forward look up and reverse lookup in DNS

a. user say:

Yes, sure.

A DNS lookup is when your computer wants to go to let's say, a website for instance.

We will use Microsoft.com as an example. Now if you just type that into the url address field, your computer says, "What the heck is this? I don't know how to go here."

So it sends a request for its i.p. address to your internet service provider or to the dns server you have your adapter or your router set to, instead. (OpenDNS servers are faster than most isp servers.)

The DNS server returns the correct ip address, gives it to your browser and then your browser now knows how to go to Microsoft.com.

If you did not disable the DNS Client service, your computer will not save that info so it doesn't need to look it up again....until the cache is flushed.

--------------------------------

Reverse DNS is when you have an i.p. address already and you want to see which domain names are present at that address.

For instance, if you did a reverse DNS lookup on 65.55.58.201 - you can learn that Microsoft is the owner.

http://www.dnswatch.info/dns/dnslookup?la=en&host=65.55.58.201&type=A&submit=Resolve

-------------------------------

Hope this helps ya out with learning. :)

We apologize for the delay in our response. To make the accounts as child account to your parent in Windows 8, please follow these steps:

Note: Make sure you have an internet connection to create a user using a Microsoft account.

1. Press the Windows key using your keyboard to get to the Apps screen.

2. Type Create an account (you will be directed to Search apps screen automatically).

3. Click Settings (second option with gear icon).

6. You will be prompted to add a user, enter the Microsoft account email address at the text box provided and then clickNext.

7. Tick Is this a child account? Turn on Family Safety to get reports of their PC use.

8. Click Finish.

After creating the child account, you can then go to the Family Safety website, login using the parent account and then manage your child's restrictions and monitoring options.

Did you make any recent changes prior to the issue?

 

The volume control in the taskbar controls the volume level for the entire system. If this control is grayed out, another program has disabled it to avoid conflicting user inputs.

 

Method 1: Run the Hardware troubleshooter

 

a) Press the ‘Windows + W’ key on the keyboard.

b) Type troubleshooter in the search box and then press enter.

c) Click hardware and sound and run the hardware and device troubleshooter.

 

Method 2: Uninstall unused sound programs

 

If you believe the volume control is grayed out in error, it may be a remnant from an older, unused audio program. Uninstall any unused programs from the "Control Panel," or turn off unnecessary sound device drivers from the "Device Manager" to restore functionality.

 

To uninstall unused programs:

 

a)      Go to Control Panel

b)      Select “Programs and features”

c)       If you find any unused applications, please select and choose “Uninstall”

 

To turn off unnecessary sound device driver from the "Device Manager”:

 

a)      Go to Control Panel

b)      Select “Device manager”

c)       Right click on any unused device and select “Properties”

d)      Select “Driver” and choose “Disable”

 


Method 3: Run SFC Scan

 

a.       Type Windows key +W

b.      Command Prompt (Admin)

c.       Type cmd

d.      At the Command Prompt, type: sfc/scannow

 

This will check for any integrity violations. Restart your system

 

 

Method 4: Update the sound driver

Update the audio drivers for the sound volume issue:

 

Update a driver for hardware that isn't working properly

http://windows.microsoft.com/en-us/windows7/Update-a-driver-for-hardware-that-isnt-working-properly

 

Automatically get recommended drivers and updates for your hardware

http://windows.microsoft.com/en-US/windows7/Automatically-get-recommended-drivers-and-updates-for-your-hardware

 

If the issue persists, please contact your system manufacturer.
