xbr dionysus say:

Restore Factory Defaults

Login Information

  • Gateway - 192.168.10.1
  • Username – admin
  • Password – admin

 

Restoring Defaults

Through firmware

  1. Plug the computer into the LAN port of the router.
  2. Open web browser.
  3. Login to router by entering http://192.168.10.1 into the address bar.
  4. If prompted for a username and password use admin/admin.
  5. Select Tools on the left.
  6. Select Settings from the sub-menu on the left.
  7. Click the Restore button.
  8. Click ok.

 

Through Hardware

  1. Locate the Power button on the device.
  2. Hold the Power button lights start to flash.
  3. Wait for lights to turn off then return to normal.
