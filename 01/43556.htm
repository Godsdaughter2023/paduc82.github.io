To install Windows 8 on SSD you should prepare the SSD first. FORMAT is not recommended as that command was written for IDE SATA (hdd) disks. Boot to your Windows 8 Setup DVD.

 

When the first Windows 8 setup screen opens...
SHIFT-F10  (opens Elevated Command Window)
DISKPART (opens Diskpart)
LIST DISK  (shows list of connected disks in data-cable hardwire order)
SELECT DISK N (disk number N is where you want Windows to install)
CLEAN ALL (wipes all data and prepares disk for OS alignment)
EXIT  (closes Diskpart)
EXIT  (closes command windows, returning focus to Windows 8 Setup)

 

Immediately after Windows 8 setup completes you must validate your copy of Windows 8...
Right-click in bottom left corner of screen and select Command Prompt (Admin)
SFC /SCANNOW (verifies that your Windows is original)
REGEDIT  (goto HKLM / Software / Microsoft / Windows / Current Version / Setup / OOBE / mediabootinstall  ... change  1  to  0)
SLMGR /REARM (Control Panel / System shows "Windows is activated" above a 20-digit alphanumeric key)

 

That's all there is to it. Diskpart is "LOW LEVEL" command operation, not yet integrated in Windows setup. Search with Google for SSD preparation for Windows Installation, and this will provide a wide range of similar methods. A few complete as the outline above.

  • The contact is removed from your contact list but will still appear in the Recent tab with a question mark. Right-click the contact name to see the options: Add to Contacts or Block This Person.

For new Skype (8.xxx), please follow the steps below:

  • Find the person's profile you want to view in your recent chats or use Search Skype.
  • Right-click their name, and then select View profile. Alternatively, from within a chat, tap the chat header to view their profile. Then, select to delete this person from your Skype contacts. 

Should you have further questions, do not hesitate to contact us back.
