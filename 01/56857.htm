Yes, Microsoft will stop supporting Windows XP in April 2014.  That simply means that no more updates will be supplied, but your installation of Windows XP will not turn into a pumpkin: you can continue to use it.

Microsoft classifies the updates it makes available at Windows Update as Important, Recommended, Optional, and Featured. Here's what they mean:

  • Important updates offer significant benefits, such as improved security, privacy, and reliability. They should be installed as they become available, and can be installed automatically with Windows Update.

  • Recommended updates address non-critical problems or help enhance your computing experience. While these updates do not address fundamental issues with your computer or Windows software, they can offer meaningful improvements. These can be installed automatically.

  • Optional updates can include updates, drivers, or new software from Microsoft to enhance your computing experience. You can only install these manually.

You will also see the following classifications:

  • Security updates. A broadly released fix for a product-specific security-related vulnerability. Security vulnerabilities are rated based on their severity, which is indicated in the Microsoft security bulletin as critical, important, moderate, or low.

  • Critical updates. A broadly released fix for a specific problem addressing a critical, non-security related bug.

Generally, you should install all "Important" updates, and especially "Security updates."  The sad fact is that Windows is inherently vulnerable and new avenues of attack are constantly being developed. 


Although I'm not privy to inside information and therefore can't provide definitive proof, I'm pretty certain that Microsoft releases "Security updates" AFTER someone outside of Microsoft has discovered a vulnerability and used it to successfully compromise Windows. 


Thus, the more "Security updates" you fail to install, the more likely it becomes that your computer will become infected.  The use of good antivirus and antimalware programs together with safe computing habits can minimize the risk, but IMO, not installing the security patches is simply asking for trouble.


The fact that your computer is currently stable doesn't mean that it isn't also vulnerable to attack.  As for the "non-security bugs" that are addressed by "Critical updates," you may not be using (yet) whatever Windows feature contains a bug that is fixed by one of these patches.

On the other hand, you are certainly not alone in experiencing problems -- often quite severe -- after installing some Windows updates.  Often, this means that something is already amiss in your system and the problem has now been exposed by the new patch. 

There are experts who frequent the Windows Update forum (and I am not one) who can help you determine why your computer is not working properly after you install a Windows update. 

But the choice is yours.  If you don't install Windows security patches, you may have no problems at all.  Or you may end up with all of your personal data uploaded to the Ukraine.  As the old Fram oil filter commercial used to say, "You can pay me now or pay me later."
