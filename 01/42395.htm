The first thing, do not use the drive you deleted the files from. If you are like most pc users your files are on the C: drive. When you delete a file the Operating System will use that space on the hard drive for new files so it is important that you do not download anything or copy/move anything to that drive.

You can download Recuva from Piriform. https://www.piriform.com/recuva/download/standard. This is a free file recovery program that can scan for deleted files. When you download the program you should download it to a drive that did not have the deleted files on them. If you have an external USB drive or a thumb drive download it to that drive. You can choose to scan for all deleted files or specify the type of file you are looking for. If you are not sure choose all files.

The last part of the wizard will allow you to do a deep scan. This scan is the most thorough and it also takes the longest. It could be hours before it finishes.
