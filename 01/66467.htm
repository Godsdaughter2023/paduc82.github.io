If you are installing Windows 10 from bootable installation media (a USB drive or a DVD) it doesn't matter how messed up your computer is. Your hard drive could be completely unbootable - even blank. lisuhi0bae say: Start System Restore from a command prompt
Applies to Windows 7

System Restore allows you to undo changes to your computer, such as software updates or driver installations, which might cause your computer to run slowly or not work properly. You can start System Restore from a command prompt window if you cannot start your computer in any mode other than Safe Mode with Command Prompt.

You need to have an administrator account to perform these steps.

  1. Start your computer in Safe Mode with Command Prompt. To learn how to do this, see Start your computer in safe mode.

  2. Log on to the computer.

  3. At the command prompt, type rstrui.exe, and then press Enter.


To start the System Restore tool when you cannot start your Windows Me-based computer normally or in Safe mode, you can temporarily change the Windows shell from Explorer.exe to Progman.exe:

  1. Start your computer by using the Windows Me Startup disk.
  2. At the Startup menu, choose Minimum Boot.
  3. At the command prompt, type edit c:\windows\system.ini, and then press ENTER.
  4. Edit the shell= line so that it looks like this:
    shell=progman.exe
  5. Press ALT+F, and then press S to save the changes to the System.ini file.
  6. Press ALT+F, and then press X.
  7. Remove your Windows Me Startup disk, and then restart your computer. When your computer restarts, Program Manager should start. If Program Manager does not start, repeat steps 1 through 7, being careful to follow these steps exactly, and then continue to the next step.
  8. On the File menu, click Run, type msconfig in the Command Line box, and then press ENTER.
  9. Click Launch System Restore to begin restoring your computer to a previous, functional state.
Notes
  • After you configure your computer to start Program Manager (Progman.exe), you can also start the System Restore tool by typing c:\windows\system\restore\rstrui.exe at a command prompt, and then pressing ENTER.
  • If you still cannot use System Restore, repeat steps 1 through 6. In step 4, change the shell= line so that it reads:
    shell=explorer.exe
    This sets the Windows shell back to Explorer.exe so that you can continue troubleshooting.

xircal say:

Some files can be deleted from the Temp folder, but not the folder itself because Windows uses it to hold temporary files in.

It's a bit like your car and it's gas tank. You can drain it, but you still have a fully functional automobile. But if you remove the gas tank, then you're not going to be going anywhere until you replace it.

You now find yourself in that situation with your computer. Until you replace the Temp folder and its contents, your machine is not going to function properly.

I think you're looking at having to Refresh the system in order to get it going again. What you'll have to do first is to create a Recovery disk and copy the manufacturer's partition to that. This means you'll have to go buy an external hard drive with at least 60GB free. But bearing in mind that the cost of a 60GB hard disk and one with a 500GB capacity is barely a matter of cents, you'll be better off with the large capacity drive which you can use for other purposes later.

To create the Recovery drive, hit Windows key to open the Start screen with the tiles and start typing the word "Recovery". You'll see the option called "Create a recovery drive" appear on the right. Click that and then click Next. Here's a screenshot.

Then follow instructions in this article: http://windows.microsoft.com/en-gb/windows-8/restore-refresh-reset-pc
