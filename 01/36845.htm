On a computer that is running a 64-bit version of Windows 7, the usable memory may be less than the installed memory (RAM).

This is expected behavior on computers that are running Windows 7. The reduction in available system memory depends on the configuration of the following:
1. The devices that are installed in the computer and on the memory reserved by those devices
2. The ability of the motherboard to handle memory
3. The BIOS version and settings
4. Other system settings
 
To view how the installed memory is allocated in Windows 7, follow these steps:
1. Click Start, type resource monitor in the Search programs and files box, and then click Resource Monitor in the Programs list.
2. Click the Memory tab, and then view the Physical Memory section on the bottom of the page.
