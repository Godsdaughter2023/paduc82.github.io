euozzylegend say:

Just updating this thread with a solution for anyone who might come across the same issue. Also looks to be a random problem when using HDMI between console and TV

Solution 1. Starting the console from the controller with the XBox button will circumvent this issue

Solution 2. If that does not work, in dash board (main menu) go to Settings -> Console Settings -> Display Discovery and then turn the "auto display discovery" OFF. Apparently sometimes the console defaults to DVI mode which is correct for a PC monitor but not appropriate for TV. Restart/reboot console for new settings. If display comes back as low res, go back into settings and manually change the display option back to 1080p or as appropriate for your display
