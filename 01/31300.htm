Edit Metadata and Tag Photos in Windows 8 - Microsoft Community

barb bowman say:

Edit Metadata and Tag Photos in Windows 8

Technical Level : Basic

Summary
Technical Level: Basic Applies to: Windows 8, Windows 8.1, Windows RT, Windows RT 8.1
Details

 

Background

 

There have been many people in the Microsoft Community Forums complaining about not being able to add or display keywords, tags, ratings or other information using the Photos App in Windows 8/Windows RT. For Windows RT. At this time, editing or creating metadata and/or tags is not supported in the Photos App. For all versions, these tasks can be performed easily from Windows File Explorer using the classic desktop.

 

 

Show the Library in Windows 8.1

 

1.  If you are using Windows 8/Windows 8 RT, Libraries are shown by default. If you are using Windows 8.2/ Windows RT 8.1, you may need to enable the libraries view (see http://answers.microsoft.com/en-us/windows/wiki/windows8_1_pr-pictures/show-and-restore-broken-libraries-in-windows-81/ed57b9dd-49d8-4f41-b008-8dd51dcaf76b)

 

2.  From the classic desktop, navigate to the Library

 

3.  Select the View tab

 

 

 

4.  Insure that the details pane is selected (if you don’t see the ribbon and menus, press the ALT key on your keyboard and enable the items you need).

 

5.  Navigate to your Pictures library

 

6.  Extra Large Icons and Details Pane should be selected in the View tab (select if needed)

 

 

 

 

 

Select an image and edit metadata

 

7.  Click or Tap an individual image.

 

8.  The details pane will display all the metadata and allow editing/adding of tags, author, subject, etc.

 

 

Note: Photos stored on SkyDrive can display EXIF information, but tags and ratings are currently not supported.
