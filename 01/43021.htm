Install RT Updates and Prevent the Black Screen when Restoring, and - Microsoft Community

garym123 say:

Install RT Updates and Prevent the Black Screen when Restoring, and other Tips

There is a way to easily get Surface RT devices around the current Windows Update conflict that leads booting up to a black screen after a refresh or reset, takes less than two hours.

This applies to the Surface RT and Surface 2

When the device restarts following a reset or refresh:

  • Search for updates, the first updates found include a firmware update. When the device restarts after the firmware update is installed.
  • Turn Wi-Fi off
  • Install the four MSU files in this order
  • KB2919355 HERE
  • KB3173424 HERE.
  • KB3172614 – (KB3033055)
– (KB3033055)

  • Turn Wi-Fi on
  • Finish updates or let the device do it on its own.
  • Right-click on Taskbar, Properties, Start Menu.   Check the box, Use the Start Menu instead of the Start Screen for no more metro start screen.

These are all from Microsoft servers.

Verify you are installing in the correct order. When these are in a folder together they do not display in this order.  The first one takes at least 30 minutes to finish. The others are much quicker.

Make a folder for them on your recovery USB drive or on your installed SD card.

************************************

The conflict between two updates causes Surface RT devices to boot up to a black screen with no obvious way to continue startup.

This problem started in January with the release of 

2018-01 Security Monthly Quality Rollup for Windows RT 8.1 for ARM KB4056895   and continues with 

2018-04 Security Monthly Quality Rollup for Windows RT 8.1 for ARM KB4093114

2018-05 Security Monthly Quality Rollup

Optional update, KB3033055, has to be installed first. If update 3 for RT, KB3033055, is installed following any of these rollup updates Surface RT devices boot to a black screen.

At the onset of the problem most users blamed KB3033055 for the problem, which is not the case. It has been in use since 2015.

Automatic updates install the Monthly Rollup updates first. When updating if finished users install optional updates and get a black screen.

I submitted this problem to TechNet March 26.

*************************************

Restore/refresh Surface with Windows RT with the recovery image (booting from the USB drive):

Method:-

  • Download the Windows RT recovery image
