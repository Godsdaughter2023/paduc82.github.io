mohammad _ imran say:

Hi Ma32206,

Dell monitors automatically go into power-save mode after a preset amount of time when the computer is inactive. The Dell monitor will go completely black, instead of showing a screensaver. This allows your monitor to essentially turn off until it's needed again.

A.   Press any key on your keyboard or move your mouse. Either will turn off power-save mode. Alternately, you can press the power button on your Dell computer tower or laptop.

B.   Press any key a second time if the monitor goes from power-save to stand-by mode. Stand-by mode allows the screensaver to appear.

Some possible good news for you, depending on what you might have available:

I am currently using a fix that appears to be working.  You'll want to first get a Hard Drive to USB adapter (assuming laptop, a 2.5 -> USB) which can be purchased on amazon if not already owned.  

Not required, but you may want to do an image backup of the corrupted drive (I did it but I think it failed), Google and download USB Image Tool.  This software creates a backup of the Hard Drive in it's current state.  Plug in and back up the HD.  
Download Recuva.  This is a program made by the same people as CCleaner.  Download it from the Piriform website, don't buy the pro version (unneeded) and don't download it from a 3rd party website.  

Go ahead and run Recuva and see if it can do a scan of the bad Hard Drive.  You may need to check some of the advanced options.  If it finds items on the hard drive, recover them.  More than likely, it'll say it cannot find the disk partition or file system.  If you see this, you'll need to Quick Format the drive.  Format it by going into This PC >> Find the HD In question (probably not labelled C: or D:) >> Format.  Make sure "Quick Format" Is checked, this will rebuild the file system for you but not delete anything.  If you do a full format, it might lose everything. Whatever you do, don't install, move files to, or otherwise add anything to the attached hard drive after formatting, it will now be available but empty.  Re-run Recuva; I suggest using the Advanced Options, do a deep search on the USB attached HD.   This part is going to take a long long time, so turn any sleep settings off on the computer.  I'm currently 3 hours in and it's 10% done with 233,000 files found (Not yet recovered). It estimates 17 hours left

The Microsoft store does offer an option that can possibly recover data;  you won't like it.  If they can recover anything, it's $150, otherwise they return the drive and do not charge.  It takes 48-72 hours to do, so it's 2 trips there and back.  If they fail but the data must be recovered, they can send it to a lab to repair, might run $1500.
