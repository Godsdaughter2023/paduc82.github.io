mouneshwar r say: Hi purpledaffodil,

Welcome to Microsoft Answers!

I would suggest you to try the following steps

         1. Click on Start; in the start search type devmgmt.msc and hit Enter.
         2. Uninstall the device driver.
         3. Unplug the USB cord (if this is an external IR receiver) and reboot.

Once it reboots and you are logged in, insert the device back into the USB port (if it's external).  If possible, use a different USB port.  If the IR receiver is built-in, then the previous step may not be able to be carried out.

If this did not work, then remove the batteries and place a metal object between the + and - terminals.  This shorts out the remote and somehow resets it.  It has worked numerous times.
