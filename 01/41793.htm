The site likely sold you an MSDN key that has been resold and exhausted:

About MSDN or TechNet Product Keys

They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. The main purpose is for evaluation purposes. The great thing about them, unlike trial Microsoft software, MSDN or TechNet keys don't expire. Because the agreement under which the subscription is provided is a single license, none of the software should be distributed outside of it. Even though its $50,000 worth of licenses, it is for one person only to use and no one else. Unfortunately, regardless of the licensing terms, persons still abuse the program, either giveaway product keys or resell it on auction sites. Microsoft licenses it in good faith that customers won't do so, but I guess human nature wins out.

In your case, what probably happened is, you bought an MSDN licensed key, which carries up to 10 activations unlike full packaged retail licenses which only carry 1 activation. The person who sold it to you probably sold it to 10 other persons. Somewhere along the way, one of those persons might have installed it on a second system, activated it, because it went past 10 activation threshold, Microsoft detected it that it was being abused and blocked the key from further use.

You are right, you should only purchase keys directly from Microsoft or Amazon (NOT the marketplace section).

A Windows Vista/7 retail .iso may be downloaded from my Microsoft Windows and Microsoft Office wiki:

http://en.community.dell.com/support-forums/software-os/w/microsoft_os/microsoft-windows-and-microsoft-office-downloads.aspx

This may be used to reinstall OEM licenses (with phone activation only), Upgrade licenses (using phone/internet activation) and finally Retail licenses (using phone/internet activation) assuming the product key is known. For an OEM license this is usually attached to a sticker on the base of the system. Likewise for Microsoft Office, providing the product key is known.

Next some major retailers taking Dell for example have their own discs, these discs unlock and activate when installed through a Dell BIOs, they will not unlock if a Dell BIOs is not detected. However due to the large number of versions of Vista/7 to cover all systems they will need an assortment of discs. A big shop no doubt will have many of these discs.

• Windows 7 Starter
• Windows 7 Home Premium x86
• Windows 7 Home Premium x64
• Windows 7 Professional x86
• Windows 7 Professional x64
• Windows 7 Ultimate x86
• Windows 7 Ultimate x64

• Windows Vista Starter
• Windows Vista Home Basic x86
• Windows Vista Home Premium x86
• Windows Vista Home Premium x64
• Windows Vista Business x86
• Windows Vista Business x64
• Windows 7 Ultimate x86
• Windows 7 Ultimate x64

They will need these for the rest of the major brands such as HP, Acer etc. Several of these systems now have the ability to restore to factory settings. Depending on the problem the end-user comes into the store with these may or may not be accessible anymore.

However as most visiters to a computer store will be home users (most businesses have dedicated in-house I.T. support), then I guess smaller ones will only have copies of the Home Premium Discs.

Now for Windows XP, there was a generic OEM disc for Windows XP, however this again required input of a product key which could be read from the COA provided that it was not worn. Dell again had their own discs which automatically unlocked and activated when installed in a Dell BIOs. There were 3 major versions of XP:

• Windows XP Home Edition
• Windows XP Professional
• Windows XP Media Centre Edition (much rarer than the other 2 versions)

There was also the 64 bit version of XP Professional but it was rare to the extent that only tech enthusiasts used it who probably would install Windows XP themselves rather than visit the local computer store. The generic XP Home OEM and generic XP Professional disc were generally the discs purchased with OEM licenses (the ones which mention the system builders).

• http://www.amazon.co.uk/Microsoft-Windows-XP-Home-OEM/dp/B0020ZTU1I/ref=sr_1_2?ie=UTF8&qid=1324023625&sr=8-2

• http://www.amazon.co.uk/WINDOWS-XP-Pro-software-intended/dp/B0018RCAD4/ref=pd_sim_sw_h__1

 

Generic in a sense that the discs were not locked to a Dell BIOs or a particular manufacturers BIOs i.e. unbranded (except for Microsoft of course). These can be used to reinstall any XP OEM (Home or Professional) by phone activation.

 

A good well-developed computer store should have a copy of all the reinstall discs but going by the size of main computer manufacturers and Windows Versions this may prove hard to obtain all the media particularly for small stores.

A bad computer store will install Windows 7 Home Premium on all machines, which will incorrectly license the ones shipped with other versions of Windows. Either that they will be unable to help you.

In the current climate I would imagine the retail .isos are used alot for Vista and 7 (or merely the factory settings). However I know many of the systems that shipped with Windows Vista (laptops) had a tendancy to produce a lot of heat and the COA stickers become worn to the extent the 25 product key is illegible. For this case there is the ability to speak to a Microsoft Support Agent when activating by phone, I guess then they probably send Microsoft a picture of the worn COA and be issued a new key.

 

Oh and the system drivers can be typically downloaded from the computer's manufacturers website. e.g. www.dell.co.uk > home > support > drivers and downloads and similar for HP and other manufacturers.

sandeep ghatuary say:

Hi computersaurus,

·         Which version of operating system is installed on the computer?

Microsoft does not provide any generic set of windows disc. OEM (Original Equipment Manufacturer) and retail version need different discs.

Windows operating system products that are acquired through a retail store (physical or online) are individually licensed and are activated. Retail version of Windows generally can be purchased in two license pricing levels, namely Full or Upgrade license.

OEM (Original Equipment Manufacturer) license is a restricted version of Windows that included with a new computer. Manufacturers and vendors ship Windows with OEM license as it’s deeply discounted when compared with retail copy. OEM license has limitation that it can only perform a clean install or custom install, but not upgrade.

OEM license also has another variant called System Builder OEM license, which is also an OEM license with lesser discount, has all the same restrictions and limitations of OEM license. But System Builder OEM license may be available to general customer who purchased a new computer (or hardware) from a small system builder via retail store.

Hardware vendors perform OEM activation by associating the operating system to the firmware (basic input/output system, or BIOS) of the computer prior to shipping new computer to customer. So no additional actions are required by the user to activate Windows, except when users choose to activate Windows with COA (Certificate of Authenticity) product key, which normally pasted on the casing of the hardware, in the case of not installing Windows with OEM provided image.

Support for Windows with OEM license is provided by OEM or system builder.
