andre da costa say:

The version you choose will depend on your future needs.

OEM versions of Windows 10 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Both provide the same functionality, but it depends on your needs: affordability vs transfer to another computer in the future.  You should avoid auction sites such as ebay, they are notorious for reselling non-genuine copies of Windows 10, Windows 8.1 and Windows 7. Instead, you should stick purchasing from Microsoft or Amazon.com.
