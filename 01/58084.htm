Error 27850: unable to manage network component. Operating system corruption may be preventing installation.

resolution was an easy fix in registry for controlset.

The maximum number of network  filter drivers in Windows 64 bit operating system is 14, however by default, it is set to 8.

I increased it to 8 in the the following registry key:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Network\MaxNumFilters
