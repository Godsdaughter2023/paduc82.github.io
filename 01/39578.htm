spiritx say:

Hi,

Instead of formatting both just swap the drives (move the cables) so the WD is the C:  (primary) and the
Hitachi is the D: (secondary). Then format and install Windows on the new C:.

Then you still have the installation on D: which you could swap back to if needed. Once you have the system
running on the new drive and have all the data from the old drive on it then you can format D: to remove
everything on it.

Remember you need backups of any data from any drive OFF of that drive before you format it which wipes
the drive clean.

Hope this helps.
