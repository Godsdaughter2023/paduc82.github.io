First Verify if the WLAN service is started and set to automatic, if the WLAN service is stopped, I suggest you to start the service and put it to automatic.

  4. Click on Start button.

  5. Click on Apply, Ok.

  1. Click start.

  2. Type devmgmt.msc in Search and select it.

  3. In Network adapter you will find the device, right click on it and select the option uninstall.
