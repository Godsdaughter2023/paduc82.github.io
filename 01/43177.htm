How-to restore Windows Live Mail contacts from backup - Microsoft Community

scottalbrecht say:

How-to restore Windows Live Mail contacts from backup

OK, not a question any longer but since I didn't find the answer online I thought I owed it to the community to write it up....   

Background:

Needed to reinstall Windows, essentailly I was using the Windows 7 beta and bought Ultimate. So I made a copy of all the personal data on the machine.  I actually made two copies: 1) using Windows backup and 2) simply copying the files which was a good thing because I wasn't able to restore the Windows backup!

The latest version of Windows Live Mail stores its contacts information by default in c:\users\[Windows user folder]\AppData\Local\Microsoft\Windows Live Contacts\{Some GUID ???}\DBStore

Inside this folder are a few files, I presume the contacts.edb one is the primary but I didn't need to identity the particular role of each so I didn't.

I'm going to assume that simply saving and replacing the contents of the DBStore folder is a sufficient backup if *NOT* reinstalling the operating system or Windows  Live Mail.  That said, I didn't try it so I recommend just backing up the entire [Windows user folder].

Problem:

After resintalling the Windows and Windows Live Mail, copying the previous files did work and there is not import Windows Live Contacts from file option.  What would hapeen when I replaced the contents of the DBStore folder is I would get an error when running the Windows Live Mail Client and it wouldn't see any of the previous contacts even though I could see from a binary inspection of the contacts.edb file they were there.

Solution:

What I noticed was the GUID in the Windows Live Contacts folder was now different than the one in my backup.   Now I was trying to replace the files in that folder so if the data files were independent the GUID change itself wouldn't matter.  What I discovered is it does!   So what I did was copy the original [GUID] folder in my backup and placed it along side the new one.   I then used RegEdit to change the string value C:\Users\[Windows user name]\Contacts\Default\ in the registry key HKEY_CURRENT_USER\Software\Microsoft\Windows Live Contacts\Database to have the string value of the original registry GUID (it was the valule of the new one). 

Restarting the Windows Live Mail client, I could now see all the contacts.

Additional Notes:

At this point, I leave it up to you as to how to proceed.   I'd suggest either exporting your contacts so they can be reimported under the new GUID.  You might also be able to just continue using the old GUID but I don't know the ramifications of this and wouldn't recommend it.   My goal was to get the data so the rest I leave up to you!    Best of luck and hope this helps someone.
