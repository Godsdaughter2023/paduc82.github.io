a. user say:

Hi Stephanie,

There maybe a couple of reasons behind this misbehavior. An outdated driver is the most probable cause as it works directly with your devices. It could also be a corrupted system file that causes misrepresentation on your display and functions. Here are some proposed methods that you can follow to troubleshoot the concern.

Method 1: Troubleshooter

First, you may run the Microsoft Easy fix diagnostic tool for network to automatically detect and fix any incompatibilities with your device and driver. Here are some that may apply to you:

  • Find and fix problems with devices and hardware
  • Find and fix problems with connecting to the Internet or to websites
  • Find and fix problems with wireless and other network adapters

More easy fix solutions are available on this article: How to use Microsoft easy fix solutions.

Now, to run the manual network troubleshooter, please follow these steps:

  1. Go to Control Panel, on the Search Control Panel bar located on the upper-right corner of the Control Panel window, type Troubleshooting.
  2. Click Troubleshooting on the results pane.
  3. On the left pane, click View all.
  4. Look for Connecting to a Network, right-click, and have it Run as administrator.
  5. Follow the onscreen instructions.
  6. Once done, please check if issue still persists.
  7. Please repeat the steps, and then run troubleshooter for Network adapter.

Method 2: Update driver

It is suggested to update a device driver to resolve any incompatibility issues with the latest version of Windows 10.

  1. To Update a driver, please go to the Cortana search bar beside the Start button. Type devmgmt.msc. Hit Enter.
  2. ExpandNetwork adapters. Look for your adapter.
  3. Right-click, then select Properties.
  4. Navigate to Driver tab. then click Update driver.
  5. If options is grayed out, click Update driver instead.
  6. Choose Search automatically for updated driver software.
  7. Follow the onscreen instructions.

For more troubleshooting steps about Network concerns, please refer to this article: Fix network connection issues in Windows 10.

Hope this helps. If you have any other concerns, don't hesitate to come back to us.
