a. user say:

A Windows PC starts in Safe Mode because it is not able to start normally.

When your computer starts in Safe Mode it is running only Microsoft software. If a computer can only start successfully in Safe Mode, it means that third-party software that loads alongside Windows is preventing the computer from starting normally. Your job now is to figure out what third-party software is preventing Windows from starting normally. It's either a third-party hardware driver or a third-party program.

This can be easy or it can be hard. I always like to start with the easy fixes first, so here's what you can do:

You'll need to sign in to a user account with Administrator permissions. Then open Task Manager [CTRL+SHIFT+ESC] and switch to the Startup tab. Disable everything you see there - you'll have to do this one at a time. Then restart your computer.

If your computer restarts normally, then you know that one or more of the items you disabled is preventing your computer from starting normally. So Enable each item, one at a time, and restart your computer. Do this until you finally find the item that prevents your computer from starting normally.
