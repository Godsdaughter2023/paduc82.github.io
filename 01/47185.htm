Convert Second Nature Screen saver images (*.SNX) into - Microsoft Community

memorysense say:

Convert Second Nature Screen saver images (*.SNX) into *JPG

Greetings all,

I remembered I had a screen saver software called "Second Nature" from the early 2000s that had phenomenally beautiful images.  There were hundreds of collections to select from, and I purchased about 24 of them.  The "Second Nature" screen saver software has long been defunct and I was curious if there was a way to convert the images into jpeg format just for personal use or use in another screen saver software.  I did some research online and found there were a few other people who had similar issues and wanted to accomplish the same.  After using my own computers and some trial and error, I did find a way to do so and it is surprisingly not too difficult.

All SNX files are readable by a software known as irfanview which is freeware and to the best of my knowledge, malware-free.  A quick google search will direct you to the site.

To read snx files, you need to base install package (~9 Mb) and the plugins (~24 Mb) both are available on their site.

The second nature images are stored in a folder known as

\[Program install path]\Images\[Collection name]\*.SNX

Start irfanview, and drag and drop an SNX file into the picture area.  You will get a warning about a wrong file extension, but go ahead and click yes to ignore it.  (All SNX files appear to be jpeg images with some kind of proprietary file header).  The image should display correctly and from then on, you can just save it as a different file name and format which will effectively convert the SNX to JPG (or whatever format you wish)

If you have a lot of SNX files to convert, you can use the batch convert in irfanview.  Not sure if it is a programming glitch, but the batch convert won't work unless irfanview is ran as an administrator.  (At least that was the case for me)
