ronnie vernon say:


Hello. I have a question about the file directories in Windows 7. When I open Windows explorer and type in c:\users\(current user)\documents the contents of the c:\users\(current user)\document folder is displayed.

1. The above path is correct for that folder.

When I type in c:\users\(current user)\documents\my music or c:\users\(current user)\documents\my videos the contents of the my music and my videos folders is displayed.

2. When you type either of the above paths and press Enter, you should see an Access Denied error message. The reason is that these paths were used in Windows XP, but they have been changed in Windows 7.  SCREENSHOT

If you have selected to Show Hidden folders in Folder Options, what you should see under the My Documents folder are the My Music and My Videos folders with a folder icon that has a Shortcut Arrow overlay. SCREENSHOT

These icons represent  Junction Points  which are pointers to the new location. Junctions do not contain any files, they are only present for  backward compatibility  purposes. The new locations are C:\Users\(current user)\My Music and C:\Users\(current user)\My Videos.

When I typed in c:\users\documents in windows explorer these folders do not appear.

See "2.". These Junctions will not appear unless Show Hidden Folders is enabled.

When I click on the the Start button then click on the (current user) name and then right click on My Music the location is listed as c:\users\(current user).

The above is the correct path to that folder.

I would like to know if it is possible disable the contents of the My Music or My Videos displayed when the user types in c:\users\(current user)\documents\My Music or c:\users\(current user)\documents\My Videos ?

What has probably happened is that the Permissions for those Junctions have been changed. Junctions should always have the 'Everyone - Access Denied'  permission set. This prevents accidental deletion of those Junctions.
