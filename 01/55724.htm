Screen saver activates when there is no activity on your device for a certain amount of time. It then goes to sleep that's why the music is being cut off. Here's a workaround to not activate screen save if you really want to continue listening to music while away from your device:

Set both options to Never.

I couldn't respond to a similar post so I will give my solution here. 

My mouse would freeze and then there would be two beeps.  The beeps sounded like something being plugged or unplugged into the usb slot.  It would happen randomly and frequently. 

I must have tried at least 10 suggestions I found on other websites and not only did they not work, but I couldn't find anyone who actually had a cure for it other than a reinstall.  It wasn't a hardware problem like voltage or else there would be some kind of pattern to it. That would be the same for a software cause.

If another program was running, it would most likely be running in the background and not actually doing something that would create the random beeps. 

I went into my task manager to close a program that wasn't responding and noticed I was running two instances of explorer.exe.  I decided to close the one with the least amount of memory being used and bam, it stop instantly. It made sense to me right after why that was causing it.  The explorer.exe changes memory usage constantly unless you are not trying to open and close programs etc. 

It also made sense that the more programs I had running, the worse the pauses would become since each explorer program was battling each other for memory and for control of the MOUSE. 

Try it.  I went and opened up the explorer.exe so there were two running again and the problem came back.
