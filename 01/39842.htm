a_user_here say:

You can defrag the drive if you like.  But in fact Windows 10 does this on periodic basis anyways.

But looking at your Task Manager I would say that fragmentation isn't your problem.

You will notice that your C disk is running at 100%.  That is what is slowing everything down.

You really need to track down what is constantly accessing that drive.

If you select Open Resource Monitor, you might be able to get a feel at least for which process is constantly using the disk.  Given the sounds from the drive it could also be that you have a drive that is failing and that is causing the problem.

With a RAID 0 setup I would be very afraid of any kind of disk failure because if you lose one drive in the RAID you lose all of your data.

You might also want to find a program to test your disk with.

Failure to install an operating system is usually caused by (in order of probability):

1. You are attempting to install with other devices connected to the computer. Disconnect them before trying to do an installation.

2. Faulty installation media. Check the installation CD/DVD for scratches, smudges. Test in another machine.

3. Faulty optical drive. Test with another bootable CD/DVD or swap it out for a known-working one.

4. Faulty RAM. Test with Memtest86+ from www.memtest.org. You will download the .iso to make a bootable cd. Burn the file with third-party burning software - as an image, not as data. 5. Faulty hard drive. Test with a diagnostic utility downloaded from the drive mftr.'s website. You will create a bootable CD with the file you download. Burn the file with third-party burning software - as an image, not as data. Boot with the CD you made and do a thorough test of the drive. If it fails any physical tests, replace it.

6. Other hardware components are faulty or your computer's hardware doesn't support the operating system which you are trying to install.

How do I know if Windows 8.1 Update 1 (KB2919355) has been - Microsoft Community

williamtaylor_963 say:

How do I know if Windows 8.1 Update 1 (KB2919355) has been installed on my computer?

Kind of getting mixed messages here. According to this link I have it since I see the search button (magnifying glass icon) on my Start page near my account name. Yet, I don't see KB2919355 listed in my list of Window updates even after searches.

If I have it, is it hidden to prevent an "accidental" uninstall?

shenan stanley say:

Press the Windows Logo key + R to bring up the RUN dialog box.

Type in the following:

cmd /k wmic qfe get hotfixid | find "KB2919355"

... and click OK.  It should open a command prompt.  If it has "KB2919355" in the top left corner, you have it installed.

... OR ...

Press the Windows Logo key + R to bring up the RUN dialog box.

Type in the following:

cmd /k wmic qfe | find "KB2919355"

... and click OK.  It should open a command prompt.  If it has "http://support.microsoft.com/?kbid=2919355" in the top left corner followed by your machine name and other stuff, you have it installed.

... OR ...

For a more manual search, you can create the following text file...

Press the Windows Logo key + R to bring up the RUN dialog box.

Type in the following:

cmd /c systeminfo >> %USERPROFILE%\Desktop\SystemInfo.txt

... and click OK.  It will open and close a command prompt and create a TEXT file on your desktop you can open to search in NOTEPAD or some other text editor for KB2919355.

That's just a few ways.  ;-)
