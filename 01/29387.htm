From what you're describing, you can't get into anything in the OS that would help you, so you've got two options.  The first:  boot from external media (like an install DVD or recovery USB) and perform a startup repair from the troubleshoot option.  If you don't have any external install media to boot from, then option two:

 

Do three hard shut-downs and power-ups in a row with your power button.  On the third power-up, that should trigger the recovery environment; blue screen with a handful of tiles and a "choose an option" prompt.  Click Troubleshoot / Advanced options / Startup repair and then just follow the prompts.  This will attempt to replace or fix whatever is broken with your startup and hopefully get you running again.  If not, you'll have to make a bootable USB or DVD and reinstall from that.

rdrewd2 say:

A clean install is probably the best way to go from here, though you'll lose all your personal files and installed programs that you had before (hopefully you have a file backup and install discs so that you can easily replace these?).  If you had your personal files stored in your OneDrive and can reload/re-install your previous programs, then no worries.  This will install a fresh, clean version of Windows 10; you won't need an activation key, just an active internet connection.  From a working computer, download the Media Creation Tool from here (click on "Download tool now") and follow the steps to install the Windows 10 image to a blank DVD (or USB stick, though the DVD is easier).  This will create a bootable Windows ISO that you can reinstall your OS from.  Insert the disc or USB into the crippled PC, boot from that, then just follow the prompts.  The only things you'll need to know are whether you are 32 or 64 bit, and whether you're reloading the Home or Pro version.  It's easier than it seems, I've done it on a few computers and it's never failed me.  But for really detailed instructions, read these:
