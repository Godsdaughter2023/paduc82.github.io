Hi BLO1,

Method 1:

 

Step 1:
As you say that it is a third party software conflict causing the problem, we can make sure if any of the third party service is causing the issue by checking for the issue in Safe Mode with Networking.

Access the below link to know more about ‘Safe Mode with Networking’:

http://windows.microsoft.com/en-US/windows-vista/Advanced-startup-options-including-safe-mode

Step 2:
If the issue does not persist in ‘Safe Mode with Networking’, you may have to put the computer in a Clean Boot and check for the issue. Below are the steps to put the computer in a Clean Boot State:

To perform a Clean Boot on a computer that is running Windows 7:



1. Click Start, type msconfig in the Start Search box, and then press ENTER.

 If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

2. On the General tab, click Selective Startup.

3. Under Selective Startup, click to clear the Load Startup Items check box.

4. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

5. Click OK.

6. When you are prompted, click Restart.

7. After the computer starts, check if the issue is fixed.

If your issue is resolved after performing Clean Boot, then follow the steps mentioned in the following KB article to narrow down the exact source:
http://support.microsoft.com/kb/929135

Also, see the section on how return your computer to a Normal startup mode by following the steps under “Reset the computer to start as usual”, after performing the clean boot troubleshooting.

Method 2:

If the issue still persists, you try to flush DNS by performing the following steps and check if it helps.

1. Open Command Prompt by clicking Start, then type cmd in the search box.

2. At the Command Prompt window, type:
ipconfig /flushdns

If you are prompted for Administrator password, key in administrator password

Note:
a. To view the complete syntax for this command, at a command prompt, type:

ipconfig /help

b. The ipconfig /flushdns command provides you with a means to flush and reset the contents of the DNS client resolver cache. During DNS troubleshooting, if necessary, you can use this procedure to discard negative cache entries from the cache, as well as, any other dynamically added entries.

c. Resetting the cache does not eliminate entries that are preloaded from the local Hosts file. To eliminate those entries from the cache, remove them from this file.

Hope this helps.

Hi there!

When playing offline, your profile is only saved locally which means the achievement record tied to it is only stored on your console. Once you delete that profile on your console and redownloaded a fresh copy of your profile from Xbox Live, it will retrieve a copy of your profile from when you last connected it online. In our case, since you've been playing it offline, the online record of your achievements has not been updated to reflect it. If you hadn't signed in online and have deleted the offline copy, then I'm afraid the profile it will retrieve is the outdated one hence the loss of achievements.

I hope this bit of information clarifies things for you. Sorry to hear about the inconvenience.
