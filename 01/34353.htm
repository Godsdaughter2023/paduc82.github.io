¡firedog say:

Windows Live Mail doesn't accept my new password.  

Windows Live Mail doesn't accept or reject your password: it passes it to the server concerned. If the server is happy, the program will go ahead with what it was trying to do - receive or send messages, for example. If the server isn't happy with the password, it will tell Windows Live Mail so, and the program will then ask you to enter it again so it can try submitting it a second time, and so on.

To change the password for any account, you have to visit the account's website and find the 'change password' option there. For example, if the email account concerned belongs to a Microsoft Account (i.e. the address is e.g. @hotmail.*, @outlook.*, @live.* or @msn.com), you change its password here: https://account.live.com/password/change. For a Gmail account, here: https://myaccount.google.com/intro/signinoptions/password.  

Once you've done that, you then change it in Windows Live Mail.

For dealing with Blue Screens, general guidance follows (you may already have seen some of it):

Consider running chkdsk on all partitions.  Let chkdsk complete on each partition and see if that helps.

Also consider running SFC /SCANNOW.

Other common suggestions include ensuring drivers are up to date (including video drivers), as well as ensuring that you're  using the latest BIOS.

Consider testing memory with Windows Memory Diagnostic or memtest86. Note that memory that passes tests is not necessarily  good memory - it just hasn't failed a test.  Consider systematically eliminating RAM from the system - run with a couple of  modules for a while, and see how things go. Then try the other modules.

Then, reboot and use the system as you normally would, and wait for a problem.  In the event that the system does not boot  completely after enabling driver verifier, boot into Safe Mode and run driver verifier, and tell it to delete the changes.

Wait for a bugcheck to occur after enabling verifier as described, and then upload it to your SkyDrive, and provide a link.

shenan stanley say:

A decent password is 15+ characters long using at least three of the following four things: uppercase, lowercase, numbers and symbols.  Since you should not be repeating passwords, you will probably need some sort of application to help you keep up with all the passwords you will be using - that way you remember one and can get to the rest.  I like KeePass.  It allows you to store additional information about the account, the link to the web page is clickable, etc.

Backup whatever is important to you (include product keys for software, contact lists, documents, music, pictures, etc.)  I would not backup *any* executables that you can download or have safely stored away elsewhere.  I would make a list of hardware you have installed on your system so you can later download the proper hardware device drivers for them.

Disconnect the computer from the Internet, zero-write the hard drive and install everything cleanly - making sure you have the firewall on and a decent AntiVirus installed before connecting it back to the Internet.  You *will* need external restoration/recovery/installation media for this for everything you want to keep on your system and you will need the product keys/serial numbers for everything you purchased.  If you do not have this for the stuff that came with your computer - you need to contact the computer manufacturer and figure out how to obtain that.

Any passwords set on the computer itself should be at least 15 characters long and different than any other password you use.  This is for every account.  I suggest having at least two administrative level accounts on the machine, neither that get used daily.  The built-in acount can be one with you creating another for you to use for installations and such.  Your account should be a user-level only account.

Disconnect your home router from the Internet and reset it to factory defaults.  Leave it (and everything) disconnected from your high-speed Internet at least 24 hours.  Configure it so its administrative password is 15+ characters and different than any other password you use.  Configure it so any wireless password is also 15+ characters long and different than any other password you use.

Connect to and change *every* online accounts password you utilize.  All banking, credit cards, facebook, email, etc and so on accounts.  Whenever possible, use a password that is 15+ characters long and different than any other password you use.  That will not always be possible, do the best you can following this rule as a secondary to the original I gave - 8+ characters including 3 of the following four: uppercase, lowercase, numbers and symbols.  Even that, unfortunately, will not be possible everywhere.

The longer you utilzie the corrupt system, the worse off.  Partial cleaning for temporary purposes is likely hurting more than helping.  You may think you cannot do all of this now - but the longer you put it off, the more damage could be done.

You may also want to keep close tabs on all of your credit card accounts, bank accounts and credit reporting agencies reports on you.

srimadhwa b say:

Hi,

You want to know more about Aero glass feature in Windows 8. I will help you with this issue.

 

Aero Peek featured is disabled by default in Windows 8. This is because, in Windows 8, moving the mouse cursor to the upper or lower-right corner of the screen brings up the charms bar, the bar that allows you quickly open Start, share, search and other settings.

 

Here is how to enable Aero Peek feature in Windows 8:

Step 1: Switch to desktop, right-click on the taskbar and then click Properties to open Taskbar Properties.

Step 2: Under Taskbar tab, tick Use Peek to preview the desktop when you move your mouse to the Show Desktop button at the end of the taskbar checkbox to enable the feature.

Step 3: Click Apply button. You are done!

 

Note that the Show Desktop button may not appear on the right of the taskbar even after enabling the feature but the feature will work. Just move the mouse cursor to the extreme right of the taskbar to peek the desktop. Also note that the Aero Peek keyboard shortcut (Windows + Spacebar) doesn’t work in Windows 8.

 

Hope this helps.
