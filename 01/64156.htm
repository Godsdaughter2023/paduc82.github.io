    Vista - Is there a way to roll back to XP? - Microsoft Community

    lhendri479 say:

    Vista - Is there a way to roll back to XP?

    Can you roll Vista back to XP?  If so, how do I do this?  Doable even if XP was never on computer?

    waseemulla shariff say:

     

    Hi lhendri479,

    Welcome to Microsoft Windows Vista Forums.

    If you have upgraded Windows XP to any version of Windows Vista, then it is not possible to roll back Vista operating system (any Version). The only possible way is to reinstall XP. It is not possible to install XP over Vista as Vista is the latest and XP is the oldest operating system. But, it is possible to install XP as the second OS, if the hard disk is having a second partition. However a clean install of Windows XP can be performed.

     

    There is another scenario where we can revert back to Windows XP, if the Windows Vista is installed on the XP machine in such a way that, XP files and folders are backed up in Windows.old folder.

    For more information, do have a look at the following article: http://support.microsoft.com/kb/933168

    Hope this helps.
