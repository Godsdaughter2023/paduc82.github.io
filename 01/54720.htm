andre da costa say:

Step 1: create a system repair disc:

Create a system repair disc - Windows - Microsoft

Step 2: Download Ease us partition manager

Install Ease us then launch it and use it to delete that partition.

After deleting the partition, click Apply the confirm changes.

Restart when prompted.

You will notice you can't boot anymore. This is because the boot information for Windows 7 is on C:\

The System Repair disc you were asked to create earlier is important. Boot from it then perform a Startup repair 3 times.

1. Put the Windows Windows 7 installation disc into the disc drive, and then start the computer.

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

4. Click Repair your computer.

5. Click the operating system that you want to repair, and then click Next.

6. In the System Recovery Options dialog box, click Command Prompt.

7. Type BOOTREC /FIXMBR, and then press ENTER.

8. Type BOOTREC /FIXBOOT, and then press ENTER.

9. Type Drive:\boot\Bootsect.exe /NT60 All, and then press ENTER.
