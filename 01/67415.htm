Right-Click on START

Select RUN and enter REGEDIT

In the LEFT pane:

  Select HKEY_LOCAL_MACHINE and expand

  Select SOFTWARE and expand

  Select WOW6432Node and expand

  Select MICROSOFT and expand

  Select INTERNET EXPLORER

In the RIGHT pane:

    Click VERSION to highlight

    Click EDIT on the menu bar at the top of the screen

    Select MODITY...

    Change the setting from

                                            9.11.10240.16384

                                        to

                                            9.11.10240.0

    Click OK

Close the Registry Editor and reboot your system

I found this to allow me to open MS MONEY without the nagging IE6 error message and access my files.  Do not yet know if there are any other hidden issues by doing this but none have yet been noted on the developer's pages.

Hope this helps.

Hi 32Ping,

When SFC recognizes an issue and it is unable to repair the Windows installation then using DISM (Deployment Imaging and Servicing Management tool) would be the next step to try before taking more drastic measures (such as reinstalling the OS completely).

The commands below can be run from an administrator command prompt or PowerShell environment (I prefer using PowerShell for this). Use the commands in order and wait for the results listed with each before attempting the next command (not just a 100% shown in a progress bar), some commands will take longer to complete than others. Here's the full list, follow the info shown pertaining to each result:

Dism.exe /Online /Cleanup-Image /CheckHealth
Dism.exe /Online /Cleanup-image /Scanhealth
Dism.exe /Online /Cleanup-Image /RestoreHealth

When completed (if/as successful) it would be a good idea to check the component store for any cleanup recommended by DISM. To do so, use the command below:

Dism.exe /Online /Cleanup-Image /AnalyzeComponentStore

If the result recommends cleanup use the command below to perform the cleanup:

Dism.exe /online /Cleanup-Image /StartComponentCleanup

More complete info and tutorial can be found at the links below:

https://www.tenforums.com/tutorials/7808-use-dism-repair-windows-10-image.html

https://www.ghacks.net/2016/03/14/analyze-the-winsxs-folder/

Windows 7 only gives you limited choices to customize your display.

Right-click on an empty spot on the Desktop and select Personalize.  You can also get to the same "Personalization" screen directly from Control Panel by clicking "Personalization" (use Control Panel view set to large or small icons).

Then click the "Display" link at the lower left of the window.  You can also get to the "Display" screen directly from Control Panel by clicking "Display."

At the Display screen, you can select from 100%, 125%, or 150% for the size of the font and icons.  You can get a bit more control over the size multiplier by clicking "Set custom text size (DPI)" at the upper left of the "Display" screen.

Back at the "Personalization" screen, click "Window Color" at the bottom of the window, then click "Advanced Appearance Settings."  You may recall this dialog box from Windows XP.  You can set sizes and colors of various display elements here.  Unfortunately, however, changes made here only apply if you are using the Windows 7 Basic theme or an Ease of Access thems -- none of which are likely to be what you want.  Sorry!
