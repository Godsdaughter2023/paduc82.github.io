In the meantime, the workaround I use is based on the SendTo folder.

 

1  You can keep "Windows Photo Viewer" as the default for opening graphics files if that is what you want.  I noticed that you got your terms a bit mixed up - an alternative utility is "Microsoft Office Picture Manager" [which is the only MS Office 2007 application that works correctly in the SendTo folder].  I have hardly ever used "Windows Photo Viewer" but I use "Microsoft Office Picture Manager" all the time.

  1. Plugged laptop into Ethernet.
  2. Open cmd and type ipconfig.
  3. For me it showed an extra adapter starting with "tunnel.GUID". I searched the registry for the GUID and it returned one reference to Wcmsvc. After searching the internet this could have been a VPN adapter i had installed. Uninstalling VyprVPN did not fix it.

  5. After rebooting the WiFi now works and the WiFi icon shows a list of WiFi networks now.

So check if you have any advanced network tools that automatically manage your WiFi settings, VPN (these may have kill switches), or security firewall. Uninstall these and see if that helps.
