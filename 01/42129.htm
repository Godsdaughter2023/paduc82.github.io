How do I import my Outlook Express or Windows Mail accounts into - Microsoft Community

bobbymi say:

How do I import my Outlook Express or Windows Mail accounts into Windows 7 and Windows Live Mail?

A new Import Mail Troubleshooting pack is now available to help import your mail and contacts from Outlook Express or Windows Mail. These mail accounts can be imported to Windows Live Mail in Windows 7. In order to import your Outlook Express data or your Windows Mail data you will need to install Windows Live Mail as your Windows 7 e-mail application.

What do I need in order to use the Import troubleshooter?

In order to use the troubleshooter you need to have upgraded to Windows 7 by either Installing a new or clean installation of Windows 7 to your Windows XP computer (this type of installation creates a Windows.old folder on your hard drive that contains the Windows XP files) or you need to have upgraded from Windows Vista to Windows 7.

How do I use the troubleshooter?

Step 1: Install Windows Live Mail:

You must have Windows Live Mail installed on your computer. For more information about Windows Live Mail and other Windows Live applications visit the following

Microsoft site:

http://download.live.com/windows7

Step 2: Download the Import Mail Troubleshooter:

You need to download the Import Mail Troubleshooter to the Windows 7 computer. To download the Import Mail Troubleshooting pack, click the following link and save the file to your desktop.

http://download.microsoft.com/download/6/C/9/6C970550-32AB-4235-9CDD-7FC9DD848BBB/WLMImport.diagcab

Step 3: Run the Import Mail Troubleshooter:

Double-click the downloaded file on your desktop and follow the given instructions

What does the troubleshooter do?

The troubleshooter checks your current Windows installation for Outlook Express or Windows Mail accounts. If these accounts are found, setup will ask you which one you want to import and then launch Windows Live Mail, and walk you through the import process for your mail and contacts.

The cause for the error This device is not configured correctly. (Code 1) is because the device has no drivers installed on the computer, or the drivers are configured incorrectly.

Uninstall and Reinstall the USB Controller

a) Disconnect the USB Flash Drive from the computer.

b) Press Windows Logo + R keys on the keyboard and type devmgmt.msc in the Run Command box and press “Enter”.

c) Expand “Universal Serial Bus Controllers”.

d) Right-click on every device under the “Universal Serial Bus Controllers” node, and then click “Uninstall” to remove them one at a time.

e) Follow the on-screen instructions to complete it and the restart the computer if prompted.

After the restart, you may again open the Device Manager and then click on the “Scan for hardware changes” button at the top of the Device Manager window and check if Windows detects the USB Controller devices on the computer and installs the appropriate drivers.

Do get back to us for any issues pertaining in the future. We will be happy to help.
