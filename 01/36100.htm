¡firedog say:

Server Response: 421 Cannot establish SSL with SMTP server 208.84.244.140:465, SSL_connect error 5:connection closed

That message is not coming from your server; it's coming from Avast's mail shield. Avast always wants to manage the secure connection when the mail shield is active, so just turn it off. You don't need it, and it often causes problems. You're still protected so long as you're careful.
