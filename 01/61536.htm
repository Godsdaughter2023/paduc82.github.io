  1. First called AuthenticationCapabilities and you must give it a value of 0x00003020 (or 12320 in decimal)
  2. Second is called CoInitializeSecurityParam and it must have value of 1.

Once you completed all steps above, reboot the computer and the problem should resolve.

jan bottorff (jbottorff) say:
I EASILY reproduced it and submitted a bug with an analysis using the Feedback hub. The problem is file explorer stops summing the total size if it encounters a path that's >= 260 characters. My bug report included some powershell code that shows the real tree size, and repros the incorrect tree size if it stopped summing when it hit a long pathname. The resulting calculated tree size from the PS repro experiment came very close to the size reported by explorer.

Simple PS code to sum a tree is:

(Get-ChildItem -Path C:\Users\aUsername\ -recurse -force) | Measure-Object -sum -Property Length

Code to repro this issue is:

$size = 0; $done = $false; Get-ChildItem -Path C:\Users\aUsername\ -recurse -force | ForEach-Object {if (($_
.Fullname).Length -ge 260) {$done = $true} elseif (!$done) {$size += $_.Length}};$size

A big boundary in calculated size happens at >= 260. I have 100 gigabytes in my download directory, which explorer ignores.

What will move with an account?

Account migration will move your entire profile.  This means that all Gamerscore, achievements, friends, MSP, remaining Gold time and profile settings will be moving with you. However, while remaining Gold subscriptions will move, if LIVE Gold is not available in your target region, it may not be functional.


What will not move with a migrated account?

This one’s especially important! No digital content is guaranteed to work after a migration.  This includes things like avatar items, games, DLC, TV and Video content.  If it’s not available in your target region, you’re not going to be able to get it back.  So, before you migrate, you’ll definitely want to make sure everything you want is downloaded onto your console.  (It’s not a bad idea to back it up on a USB as well, just in case!)

All Xbox Music and Zune Music passes are automatically canceled when you move, and remaining subscription time cannot be refunded.  That means you won’t be able to listen to music you’ve downloaded, stream music, or use any other features of these subscriptions,  until you restart those subscriptions!


When you will not be able to change regions?

There’s a few things that can stop an account from being able to migrate.

  • If an account is suspended (either due to Enforcement action or due to billing) then it can’t migrate until that is resolved. 
  • Any accounts that are currently a part of an Xbox Entertainment for All plan won’t be able to change regions until that subscription is over.
  • If you are a minor in your current region and would not be a minor in the target region (or vice-a-verse-a) you will also be able to change.  So it’s got to be minor in both or adult in both!
  • Finally, region migration is only available once every 90 days, so if you’ve migrated your account before less than 90 days ago, that’s going to also prevent the migration from going through.
