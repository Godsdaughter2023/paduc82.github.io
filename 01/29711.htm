   1. Click Start, type msconfig in the Start Search box, and then press ENTER.

   5. Click OK.
   6. When you are prompted, click Restart.
   7. After the computer starts, check whether the problem is resolved.

After you determine the startup item or the service that causes the problem, contact the program manufacturer to determine whether the problem can be resolved. Or, run the System Configuration Utility, and then click to clear the check box for the problem item.

danida_u say:

Hi

There are multiple reasons why calls to mobile and/or landline numbers fail.

Q: I have the subscription for UK, Australia and New Zealand but in UK a few numbers come a invalid numbers, why is that so

A: One possible reason why it comes as an invalid number is because you might have dialed it incorrectly. Try dialing the number in this format: Enter the "+" icon and then country code plus the number you want to call.

Q: Also a few come as network error, when there is no problem with the internet.

A: Skype relies on the strength of the bandwidth you are using. It would be better if you move to an area where the signal is stronger if you are on Wi-Fi or connect to another network.

Q: Also a few numbers just get disconnected after one ring giving the message that it is unavailable

A: Due to fluctuations in the public switched telephone network we cannot always guarantee the quality or reliability of calls made from Skype. Another thing is that some phone service providers are restricting calls made from unknown callers to protect their users from spam and unwanted calls. It would be better if you set up caller ID to prevent this from happening.

For additional details, you may refer to the following support links.

  • Why am I having trouble calling a mobile or landline?
  • What is caller identification and how do I set it up?
  • How much bandwidth does Skype need?

Make sure you are using the latest version of Skype available for your device and a stable connection. If the issue persists, try calling the numbers you are having issues with using a regular phone and let us know what happens.
