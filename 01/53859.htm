steve winograd say:

Yes, Windows 7 can share its Internet connection with XP.  I think that we can get everything working by starting over with all of the settings.  I've added some that we might have missed before and tested the settings on my computers.

On the Windows XP computer, configure these network connection settings manually:

      IP Address = 192.168.137.2
      Subnet Mask = 255.255.255.0
      Default Gateway = 192.168.137.1
      Preferred DNS Server = 192.168.137.1

On the Windows 7 computer:

1. Disable Internet Connection Sharing.

2. Enable Internet Connection Sharing on the cable modem connection.  If it asks you to select a private network connection, select the LAN connection.

3. In the LAN connection's TCP/IPv4 properties:

   a. Set the default gateway to 192.168.137.2

   b. Click the Advanced button , un-check the Automatic metric box, and enter 999 in the Interface metric box.
