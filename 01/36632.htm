Pin a folder to the Start Menu

How can I pin a folder (a folder from My Computer or Windows Explorer) to my Start Menu? I'd like an for windows XP Professional and for Windows XP Home Edition. Thanks. pgs654321
sathish s v say:

Hi,

 

Pinning an item to Start menu helps you in accessing that item quickly and easily. By default Windows only allows programs or shortcuts to be pinned to Start menu.

 

By making a small change in the registry value which can enable "Pin to Start Menu" option for folders as well. So that you can pin them to “Start menu” without any problem.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click this article: How to back up and restore the registry in Windows XP

 

Follow the steps mentioned below to pin a folder to start menu.

a.       Press “windows key + R” from your keyboard.

b.      Type “regedit” without quotes in the search box and hit enter.

c.       Navigate to this registry value “HKEY_CLASSES_ROOT\Folder\ShellEx\ContextMenuHandlers\”.

d.      Right click under “ContextMenuHandlers” and select “New” and select “Key”.

e.      Create a key with “{a2a9545d-a0c2-42b4-9708-a0b2badd77c8}” name including the parenthesis { } and without quotes.

f.        Close window.

g.       Now hold “shift” key then right click on any folder.

h.      You will get an option to “pint o Start menu” option.
