If your system will not boot up far enough for you to get to the F8 Advanced Boot Options menu and into at least Safe Mode (does it?), you will not be running a System Restore.

You can start/stop Superfetch service in administrative Command Prompt.

To start the service, run following command:

sc config “SysMain” start=auto & sc start “SysMain”

To stop the service, run following command:

sc stop "SysMain" & sc config "SysMain" start=disabled

Could you please do the following.

Right click the Start button and go to Control Panel. If you have the Category view, change it to "Small  icons" via the link top right and then click the Windows Update link.

Next, click the link called "View update history" and then see if you have KB2911106 installed, It fixed many issues with the Windows Store and was published in January this year. See this article: http://support.microsoft.com/kb/2911106/en-us

Also, could you please check system time and date as follows.

  1. Click the clock on the right hand side of the Taskbar and go to "Change date and time settings".
  2. In the next menu, click the "Internet time" tab and then click the "Change settings" button.
  3. It should look like you see it in the screenshot. If anything differs, make corrections and then click the "Update now" button.

Has this problem just started, or have you had it for longer than say a month? If you were able to connect recently and now this issue as arisen, you could try running System Restore to wind back the clock to a time prior to the problem occurring like this.

  1. Right click the Start button and go to System.
  2. Next, click the link top left to "System Protection" and then click the System Restore button.
  3. In the next menu, checkmark the alternative option at the bottom to "Choose another Restore point" and then click Next.
  4. From the calendar which appears, choose a date when you didn't have a problem accessing the store.(Checkmark the box at the bottom to get more Restore points if you can't go back far enough). Click Next to go ahead and then confirm when you see the final warning.

It can take around ten minutes of so to run during which time the machine will reboot. At some particular point, you'll be faced with a plain black screen where nothing appears to be happening. But don't interrupt the process and eventually you'll get the usual login screen again.
