Recently we had an issue with Skype where in Skype users lose connectivity to the application using Windows XP. This has been fully-resolved. We do apologize for the inconvenience. You may follow the steps below if you still having issues signing in on the Skype application.

1. Make sure that Skype is no longer running. Quit Skype.
2. On your keyboard, press the Windows key + R key. Then, the run windows will appear.
3. On the run window, type in %appdata%\Skype and hit enter. Look for a file named shared or shared.xml and delete it.
4. Open the folder that has your Skype name.
5. Locate and delete the config or config.xml
6. Open Control Panel and look for Internet Options.
7. Go to the advanced tab and select Reset.
8. Put a check mark under "Delete Personal Settings", then hit okay and enter.
9. Close it when prompted.
10. Restart Skype.

 

If you still having the issue after the troubleshooting above, try to delete host file on Skype.

 

1. Quit on Skype, right click the Skype icon then select Quit Skype
2. Press windows key and R on your key board then type in %temp%\Skype
3. Locate DBTemp Folder then delete it
4. Press windows key and R on your key board then type in C:/Windows/System32/Drivers/Etc
5. Locate hosts files then rename or delete it

Hi,

This error appears when you  run a software which require the Microsoft Visual C++ Redistributable. The redistributable version can be easily downloaded on the Microsoft website as x86 or x64 edition. Depending on the software you wish to install you need to install either the 32 bit or the 64 bit version.

Method 1:

Download Visual C++ Redistributable

http://www.microsoft.com/en-us/download/details.aspx?id=30679#

Method 2: You may run System File Checker [SFC] scan on the computer which will replace the missing or corrupt files & check if the issue persists.

1. Press Windows key + X, click Command Prompt (Admin).
2. In the Command Prompt, type the following command, sfc /scannow and then press Enter.
3. Wait until the process completes. This usually doesn’t take longer than 20 minutes with a regular hard disk and takes much shorter when you have a fast SSD drive.
