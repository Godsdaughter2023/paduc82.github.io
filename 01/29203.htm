The reason you got blue screen was due to the change of SSD on the motherboard. Windows is specifically designed so you won’t be able to run it on an external drive. When you removed the old SSD which had Windows 7 from the computer, it became external drive.

Bootrec /fixmbr and press Enter.

Bootrec /fixboot and press Enter.

Bootrec /rebuildbcd and press Enter.

6. Restart the computer.

Check if you are able to boot Windows 10. You can reinstall Windows 7 on the old SSD. Make sure both the SSD’s are connected on the motherboard to use dual boot.
