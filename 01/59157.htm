sfc /scannow

sfc /scannow

Method 1:

I would suggest you to once again check if all the options to run the scheduled disk defragmentation have been disabled -

  1. Open Disk Defragmenter by clicking the Start button. In the search box, type Disk Defragmenter, and then, in the list of results, click Disk Defragmenter.‌ If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  2. Click Configure schedule, and then uncheck the option to Run on schedule (recommended):

Refer to the below link -

http://windows.microsoft.com/en-US/windows7/Schedule-Disk-Defragmenter-to-run-regularly

 

Method 2:

I would suggest you to perform an SFC scan and then check if it lists any issues -
Perform a SFC Scan 

1.      Click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow

2.      Type the following command, and then press ENTER:

sfc /scannow 

A message will appear stating that 'The system scan will begin'. Be patient because the scan may take some time

3.      If any files require a replace SFC will replace them. You may be asked to insert your Win7 DVD for this process to continue

4.      If everything is okay you should, after the scan, see the following message "Windows resource protection did not find any integrity violations" 

5.      After the scan has completed, close the command prompt window, restart the computer and check.

Check the below mentioned link for further guidance on SFC scan -http://support.microsoft.com/kb/929833

1. Click Start and Choose Run.
2. In the Run box, type services.msc and Click OK.
3. Right-click the Disk Defragmenter service. Click Stop.
Stopping the service will take a moment.

Now check if the issue persists. If the issue resolves you may start the service again after troubleshooting is done and check if the defragmentation for the SSD device persists or not.
