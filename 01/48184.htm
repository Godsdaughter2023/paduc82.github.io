kosh vorlon say:

Hi Elizabeth,

 

I do not believe this is an infection, but merely a result of resetting everything to the defaults when you did the re-load and lost what might have originally been done to prevent or alleviate this problem.

 

Does your laptop have a touchpad to use instead of a mouse and do you do that or do you have a separate mouse attached?  I had this very same problem and it drove me crazy until I realized the problem was that as I typed, my palms or thumbs were sometimes brushing up against the touchpad and that was enough for it to activate it and move the cursor.  While it had sensitivity settings, even those didn't work well enough (for me anyway), so I got a wireless laser mouse with a USB plug in, updated the drivers for the laptop's touchpad as well as for the mouse, and discovered new entries that allowed me to not only adjust the touchpad sensitivity but turn if off it it detected another pointer device was attached.  If you just reloaded Windows, you may not have the most current drivers, firmware, and/or software as it would have returned to the original ones. That was the end of my problem.

 

It really does vary so without the specific manufacturers, makes and specific models of everything involved I cannot provide specific advice, but in general, this is how it would probably work:

 

Go to Control Panel / Mouse and there should be a setting there somewhere to disable the touchpad when a mouse is connected and/or adjusts the touchpad sensitivity. It may not be in a tab where you expect to find it - so check them all. It may also read differently, so look closely at all the options. If it can be done, it is most likely there (and it almost always can be done somehow - eventually).  This may help in general: http://www.ehow.com/how_8251998_adjust-windows-7-touchpad.html.  Instructions from the manufacturer will of course be more specific and detailed.

 

If you can't find anything like that, go to your computer manufacturer's web site and update the drivers for your mouse and/or touchpad (if the mouse was purchased separately, you may need to get the mouse drivers, software, and/or firmware from the mouse manufacturer's web site. Then reboot and check to see if the option now appears. I had to do that myself when I added a USB mouse to my system since the original touchpad drivers did not include the option, but the updated ones worked like a charm.  To update drivers in Windows 7, see the following: http://pcsupport.about.com/od/driverssupport/ss/update-drivers-windows-7-tutorial.htm and/or follow the instructions that come with the drivers.   There may be drivers, firmware and/or software for such devices, so be sure to get all that applies for the EXACT computer and operating system you are using as well as for the EXACT mouse you are using if you are doing that.

 

I hope this helps.

 

Good luck!
