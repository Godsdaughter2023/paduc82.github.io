Las actualizaciones de controladores para Windows 10 y muchos dispositivos (como Mouse) se descargan automáticamente y se instalan a través de Windows Update.

Es probable que ya tengas el controlador más reciente, pero si tienes problemas con un dispositivo, puedes intentar corregirlos actualizando o volviendo a instalar el controlador. Esta es la manera de hacerlo:

1. Selecciona Inicio, en Buscar, escribe Administrador de dispositivos y selecciónalo en la lista de resultados.

2. Expande una categoría (en este caso Mouse) para buscar el nombre de tu dispositivo, haz clic con el botón derecho en él (o mantenlo pulsado) y selecciona Actualizar controlador. Por ejemplo, para actualizar un controlador de tarjetas gráficas, expande la categoría Adaptadores de pantalla, haz clic con el botón derecho en la tarjeta gráfica y selecciona Actualizar controlador.

3. Selecciona Buscar software de controlador actualizado automáticamente.

4. Si Windows no encuentra un controlador nuevo, puedes intentar buscar uno en el sitio web del fabricante del dispositivo y seguir sus instrucciones.

Si estos pasos no funcionan, intenta volver a instalar el controlador: abre Administrador de dispositivos,haz clic con el botón derecho (o mantén pulsado) el nombre del dispositivo y selecciona Desinstalar. Reinicia el PC y Windows intentará reinstalar el controlador. Si este paso no da resultado, prueba con el Solucionador de problemas de Windows Update, en el siguiente enlace: https://support.microsoft.com/es-mx/help/402732...
