b69ca say:

Hi

That's a strange one and a couple of things could have caused it.  Is this your own computer or part of a network and is it possible that any policies have been applied to it.  It's possible for an administrator for example to remove the user's ability to view hidden files and folders.  If that's the case, you will have to have the policy removed to get the option back.

The second thing, although I suspect not is that the registry for this option has been corrupted.  You can try this though if you want.  Paste the following to a file called hidden.reg, save the file and double click it -

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced\Folder\Hidden]
"Text"="@shell32.dll,-30499"
"Type"="group"
"Bitmap"=hex(2):25,00,53,00,79,00,73,00,74,00,65,00,6d,00,52,00,6f,00,6f,00,74,\
  00,25,00,5c,00,73,00,79,00,73,00,74,00,65,00,6d,00,33,00,32,00,5c,00,73,00,\
  68,00,65,00,6c,00,6c,00,33,00,32,00,2e,00,64,00,6c,00,6c,00,2c,00,34,00,00,\
  00
"HelpID"="shell.hlp#51131"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced\Folder\Hidden\NOHIDDEN]
"RegPath"="Software\\Microsoft\\Windows\\CurrentVersion\\Explorer\\Advanced"
"Text"="@shell32.dll,-30501"
"Type"="radio"
"CheckedValue"=dword:00000002
"ValueName"="Hidden"
"DefaultValue"=dword:00000002
"HKeyRoot"=dword:80000001
"HelpID"="shell.hlp#51104"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Advanced\Folder\Hidden\SHOWALL]

Failing all of this you can access the folder by typing your full path in the explorer window address bar and if you have access to a command prompt, you could do something like attrib c:\folder -h which would remove the hidden attribute from the folder.

greg carmack - mvp 2010-2020 say:

First question is:  Are you using the Windows Reset or a specially added Factory Reset that's appended to that Windows menu by the PC maker to run Factory Recovery in Windows 10?  This is important because we were able to persuade Windows 10 developers to give users a clean reset instead of being forced to run the PC maker's REcovery larded with all the bloatware that corrupts and throttles Windows. 

If you're not specifically choosing the Factory Reset but only the Windows Reset and you get that message then the recovery image Install.wim  might have become corrupted.  You can try repairing it using DISM - Repair Windows 10 Image - Windows 10 Forums to see if that is enough to force Reset to run.  If not then it's likely in the Recovery staging partition that itself has become corrupted.

An alternative to Reset is to install Media Creation Tool and run an Upgrade with the latest  Creators Version media which reinstalls or UPgrades Windows while saving files, programs and settings. 

But if you choose at this screen to Change What to Keep and choose Nothing (except maybe files) then it is very close to a Clean install:

However be aware that in all of the time you've spent here you could have several times done a Clean Install Windows 10 and had the best install and learning experience of all.  There is no rush to install all your programs as long as you make the preparations thoroughly, and in fact taking your time to see how program installs, settings changes, etc. affect the install is the smartest way to do it - stretching it out as long as you want so that when it's finally done it is worth saving as an image so you never need to do another.
