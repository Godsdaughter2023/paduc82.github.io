Hello Kim, thank you for your message! My name is Ted and I'm an Independent Adviser.

There are several reasons that can cause this issue such as lack of administration rights or User Account Control or security software installed on your PC.
I want you to try the troubleshooting methods below:
Method 1:
Try running the driver file as administrator
Right click on the driver file, Properties, Compatibility tab
Under Privilege Level make a check on Run this program as an administrator
Click on Apply and then OK
Then you can try to install the driver. If it doesn’t help, follow the steps below.
Method 2:
Try enabling the built-in administrator account
Click on Start icon and type in Command Prompt and right click on the result
Select Run as administrator
Type in the following command and press Enter:
net user administrator /active:yes
Sign out and sign in to the administrator account
Install the driver
Sign out and sign in back to your user account
Disable the built-in administrator account by following the steps:
1. Click on Start icon and type in Command Prompt and right click on the result.
2. Select Run as administrator.
3. Type the following command and hit Enter:
net user administrator /active:no
