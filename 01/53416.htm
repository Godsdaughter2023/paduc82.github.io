lemp say:

Windows Update will NOT work for an XP system unless Service Pack 3 is already installed.

Service Pack 3 requires that Service Pack 1 OR Service Pack 2 be installed first

Service Pack 2 does NOT require Service Pack 1 be installed first.

Thus ...

  • if the XP installation CD that you have is "XP Gold" (no service packs), then you will need SP2 and SP3
  • if the XP installation CD that you have includes SP1 or SP2, you only need SP3.

You can download both XP Service Packs from the Microsoft Update Catalog.  You will need to use Internet Explorer v. 6 or newer.

http://catalog.update.microsoft.com/v7/site/Search.aspx?q=xp%20service%20pack

Click the "Add" button for each SP that you want.  Click "View basket."  Click the "Download" button.  In the window that pops up, "Browse" to a download location, respond to the UAC prompt if necessary (depends on which version of Windows you're using to access the Catalog), and click "Continue."

To create a slipstreamed installation CD that includes all the post-SP3 updates (requires that you first create an installation CD that includes SP3, and if necessary SP2), see http://xdot.tk/

Even if you don't create the post-SP3 CD, that site seems to imply that Windows Update may not work -- even after you've installed SP3 -- unless you first manually install the first 3 updates listed.  The list also includes the update to get IE 8 if you need that.

lemp say:

Slipstreaming in this context means to integrate service packs and other updates into an existing Windows installation CD.

The xdot.tx site I mentioned describes how, starting with an installation CD that includes Service Pack 3, to integrate all of the post-SP3 updates into that installation CD.  If you had such a CD, you would use the script provided to download all of the post-SP3 updates and then use that to create an installation CD that includes both SP3 and the subsequent updates.  I referenced the site mainly for its listing of 3 updates needed for Windows Update to work properly.

As I wrote earlier, for a running XP system, SP3 can't be installed unless either SP 1 or SP 2 has been installed.  Apparently to create a slipstreamed installation CD, you can just add SP3 to an installation CD that has no service packs ... at least according to this comment (I have no personal experience with this):  http://www.msfn.org/board/topic/151085-can-you-slipstream-xp-sp2-and-sp3-at-same-time-with-nlite/ (but read all the way through the thread).

Finally, I don't know if you can start with your no-service-pack XP CD and integrate both SP3 and all of the post-SP3 updates.

You can read about nlite, one of the commonly-used tools for slipstreaming, here --> http://www.nliteos.com/guide/

Good luck.
