slance310 say:

Ho,

I think you're talking about a CLONE of your internal drive to a second drive...

And there is nothing wrong with that at all...

The discussion was about Imaging though, and that is not quite the same.

An Image needs to be recovered to another drive... like a replacement internal HDD for instance...

If there is nothing on the new HDD, you need to boot with Acronis media to RECOVER the image... Acronis TI will create a bootable CD or Flash drive that has all of the features of the installed version...

The point being is that the Windows process creates Images, not Clones...

And like Acronis, if the internal HDD dies, you need to boot with some type of Recovery media to get the Image back into the machine...

I currently have about 15TBs of backup Acronis Images for various machines...

I don't delete the old ones when I create a new one... There have been many times that I wished I had not deleted the old one, so now I don't...

slance310 say:

Hi,

In addition to what BulldogXX has mentioned... you will need bootable media to reload the image... Just like you would with Acronis...

I have recently tried the built-in imaging feature, and was actually impressed... It is not as versatile as Acronis, but works well...

You will need to create bootable repair media of some type... I just use installation media like a flash drive or DVD and select REPAIR at the INSTALL screen...

You have to use Repair Media that is the same version as the image...

Win 8.1 media for a Win 8.1 image, and Win 10 media for Win 10... I have not yet tested as to whether they need to be the SAME BUILD or not, but I wouldn't think so..

You could probably just make a RECOVERY CD instead, but I would test it first.

I think it is then Troubleshoot, then Advanced options...

This will bring up a screen to Recover the Image...

BTW... I found the hard way that #1... if you make a 2nd image it will overwrite the 1st...

and #2, If you rename the 1st, Windows will no longer recognize it..

Does Windows 7 full system backup and restore allow a backup to - Microsoft Community

carguy_1 say:

Does Windows 7 full system backup and restore allow a backup to restore to a different size hard disk than what was originally backed up?

Original title: Question about System backup

Does Windows 7 full system backup and restore allow a backup to restore to a different size hard disk than what was originally backed up?

shinmila h say:

Hi CarGuy_1,



Yes you can restore files from a backup that was created on another computer in Windows 7. However make sure that the disk where you're restoring the backup to is the same size or larger than the disk that you backed up.

 

Note: When you restore your computer from a system image, it's a complete restoration. You can't choose individual items to restore, and all of your programs, system settings, and files are replaced with those on the system image.

 

 Here’s how you do it.

1.    Click on start and type Backup and Restore in the field provided.

2.    Click on  Backup and Restore to open it

3.    Click Select another backup to restore files from from the new window and   then follow the steps in the wizard.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

To find files that were restored from a backup made on another computer

If you're restoring files from a backup that was made on another computer, the files will be restored in a folder under the user name that was used to create the backup. If the user names are different, you'll need to navigate to the folder where the files are restored. For example, if your user name was Molly on the computer that the backup was made on but your user name is MollyC on the computer that the backup is being restored on, the restored files will be saved in a folder labelled Molly. You can find the restored files by following these steps:

1.    Click to open Computer.

2.    Double-click the icon of the drive that the files are saved on, for example C:\.

3.    Double-click the Users folder.

You will see a folder for each user account.

4.    Double-click the folder for the user name that was used to create the backup.

The restored files will be in the various folders based on where they were located originally.

To restore files from a file backup after restoring your computer from a system image backup

After you restore your computer from a system image backup, there might be newer versions of some of your files in a file backup that you want to restore. Because you've reverted your computer to the state that it was in at the time the system image backup was created, Windows Backup won't show any file backups in the Restore Files wizard that were created after the system image backup was created. To restore files from a file backup that was created after the system image backup was created, follow these steps:

1.    Click to open Backup and Restore.

2.    Click Select another backup to restore files from.  If you're prompted for an   administrator password or confirmation, type the password or provide confirmation.

3.    In the Backup Period, select the date range of the backup that contains the files that you want to restore, and then follow the steps in the wizard.

For more information, see

 

Restore your computer from a system image backup

http://windows.microsoft.com/en-US/windows7/Restore-your-computer-from-a-system-image-backup
