¡firedog say:

Assuming you're referring to the English spelling dictionary, there is a curious bug that disables the Edit button if the custom dictionary contains more than about 60 words. I don't know that it affects other language dictionaries.

The dictionary is a simple text file, so you can edit it in Notepad or any other plain text editor (i.e. not Word or Wordpad). You'll find it in the folder at %userprofile%\Microsoft\Windows Live Mail\Proof. The file is called langXXXX.lex, where XXXX is the Locale ID (LCID) in hex. 0409 is English (US), 0809 English (GB), 0C09 English (AU) and 1009 English (CA). More LCIDs here: Microsoft Locale ID Values
