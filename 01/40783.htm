2.      In the Open box, type run, and then click OK.
3.      If you are prompted for an administrator password or for confirmation, type the password, or click Continue.
4.      In the Open box, type %systemdrive%\programdata\microsoft\ehome , and then click OK.
5.      In the details pane, right-click mediacenterdatastore.db, and then click Delete.
6.      Click yes to confirm that you want to move the Mediacenterdatastore.db file to the Recycle Bin.
7.      Restart the computer for the changes to take effect after executing the steps.
8.      Start Windows Media Center.
