lefsto say:

Hello,

I had the same problem with my civilisation 4 game.

I solved it on this way.

1.) I copied the file SECDRV.SYS to "\WINDOWS\SYSTEM32\DRIVERS "

2.) In DOS prompt (with administration rights) I entered the next command.

     "SC config SECDRV type= filesys start= demand" (Mind the space between equal sing and command.)

3.) Restart Windows 10 while holding down the "SHIFT"-key.

     An option menu pops up. Browse through the options until you find "Start up option", select that option.

     Choose restart and after a while you get new options and choose option 7.

     BE CAREFUL. From this point your computer starts up unprotected.

     Execute only programs that you trust.

4.) Open DOS prompt again. execute the line "SC start secdrv"

Now it should be possible to play Civilisation 4. And other programs that uses Secure drive.

Each time when you want to play games with Secure drive, step 3 and 4 should be done again.

AFTER PLAYING and before you use other programs, I recommend to restart the computer, so that it would be protected again. 

Where to find "SECDRV.SYS".
I updated my WIndows 8.1 to Windows 10 and I was able to find that file in "\WINDOWS.OLD\WINDOWS\SYSTEM32\DRIVERS"

Or look on the web. But watch out for ill minded sites.

In my register the secure drive was already mentioned.

When not, open DOS prompt (with administration rights) and execute "SC create SECDRV binpath= C:\WINDOWS\SYSTEM32\DRIVERS\SECDRV.SYS" (Mind the space between equal sing and command.)

And than follow step 1 to 4

Good luck.
