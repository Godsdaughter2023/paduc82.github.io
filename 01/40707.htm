The error code 0x800706BA indicates that the remote procedure call (RPC) server is unavailable.

Provide us additional information to assist you better with this issue:

  1. Do you get this error only when printing to PDF or while printing to a physical printer as well?

  2. What is the error message you have received while trying to start the Print Spooler service?

  3. Which version of Windows is running on your device?

You may refer the article Which Windows operating system am I running for the information.

Regarding the issue with the Print Spooler service which is not starting, try the below suggested steps. As you have already verified the Spooler dependency services try the below command on a command prompt window to fix the issue.

Method 1

  1. Click on Start, type in cmd.

  2. Right click on it and Run as administrator.

  3. Type the command

    sc config spooler depend= RPCSS

(Note the space after the = but not before)

    4.  Restart the Print Spooler and check the status of the issue.

Method 2

You might have already tried this based on the troubleshooting article that you have mentioned. However, please let us know if you get the same error once you try the below mentioned steps.

  1. Open the run command. Type in services.msc.

  2. Double click on the Print Spooler service and click on Log On option.

  3. Uncheck the box “Allow service to interact with desktop”

You may also refer the article Fix printer problems for more information.
