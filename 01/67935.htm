Live mail adds underscore to Word extension, i.e. docx becomes _docx - Microsoft Community

spudnick09 say:

Live mail adds underscore to Word extension, i.e. docx becomes _docx

Live mail adds underscore to Word extension, i.e. docx becomes _docx  recipient cant open document without saving and renaming.
¡firedog say:

Live mail adds underscore to Word extension, ...

I think if you look closely, the file name given in the message source actually contains more than one dot (period, full stop, '.'). Because this used to be a problem in some situations, in the days when only one was permitted in Windows filenames, Windows Live Mail helpfully converts one of them to an underline (underscore, '_') character. Sadly, it converts the wrong one, so instead of changing oldletter..docx to oldletter_.docx, it changes it to oldletter._docx

The obvious way of dealing with this is to use the 'save attachments' function to Save as, taking the opportunity to correct the file name.

Another way of dealing with it is to save the file with its current, undefined extension, ._docx, say, then use Explorer's Open with function to select Word to open the file with and Always use this program to open this type of file.

The problem often arises because users allow programs to select file names. Word, for example, will take the first line of a document or part thereof as its suggested file name; if this happens to be a title or heading ending in a dot, then that dot will be included before the dot-extension is added.   

No amount of repairing the installation or uninstalling and re-installing will have the slightest effect on this fault.
