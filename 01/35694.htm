I have figured out how to fix this problem now.  Try the following; it worked for me:

1. Download and install the Microsoft Windows SDK for Windows 7 and .NET Framework 3.5 SP1 from http://www.microsoft.com/en-us/download/confirmation.aspx?id=3138 (unless you are a .NET application developer, you are unlikely to have this; unfortunately it's about a 1Gb download and we only need it for one file, CorFlags.exe, but I have not found that available separately).

2. During the installation, you can uncheck Samples and Documentation as you do not need these, but leave other components selected (as they are by default -- I do not know which actual component the required file is in, nor whether it has any dependencies on other files/components), and Microsoft Redistributables unselected (as it is by default -- you do not need these).

3. Open a Command Prompt as an administrator (click the Start button/icon, then All Programs, then Accessories, then right-click on Command Propmpt and select Run as administrator, and confirm that you want to allow it to make changes to your computer, if asked).

4. In the Command Prompt window, carefully enter the follow commands one by one (you may copy and paste from here -- check if you get any error messages as it may mean you've miscopied the command):
cd \
cd "Program Files (x86)"
cd PitchPad
copy PitchPadP.exe PitchPadP.exe.bak
"c:\Program Files\Microsoft SDKs\Windows\v7.0\bin\CorFlags.exe" PitchPadP.exe /32BIT+

5. Close the Command Prompt window and now run PitchPad.

I now have the program running on Windows 7 x64.  I hope this helps many cricket fans, clubs and scorers around the world...
