How can I identify which surface model I have - Microsoft Community

billreidy say:

How can I identify which surface model I have

how can identify which surface i have
barb bowman say:

There are a few ways to tell, for any Surface Device:

  1. Use the Surface app. That will tell you. If you don't have this installed, you can get it from the Microsoft Store.
  2. Visit the Microsoft Device service and repair site to register your device serial number. See Find the serial number on Surface if you are unsure where this is on your device.
  3. If you are unable to get online, use the System Information app to view your system info. From Search, type system and then select System Information from the list of results.
