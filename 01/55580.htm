Fortunately I did figure this out, thank goodness.  I was going to try and revert to Windows 7; but, luckily I didn't have to.   I had a vague memory about the cert created for the encryption.

Here's how I recovered.

First: When you created the certificate, Microsoft creates the personally signed certificate and prompts you for a location, filename, and optional password to store a copy of the certification.

The file ends in .pfx, If you don't know where it is, you should start looking for it, because it *was* created.  Thinking security, you may have chosen a USB Drive, A CD, or an external drive of some sort, or even the hard drive itself (hopefully you didn't store it encrypted - if that's possible).    Personally, I had also left a text file in the cert folder that contained a password clue only I would understand... (pats self on back)... after 3 tries, I got it right.

For longevity, I created a new certificate, saved it, and encrypted the files.

To do this:  Search for, and use the Windows 10 app "Manage File Encryption Certificates"
