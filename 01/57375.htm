To work around this error, we'd recommend that you create a new user account and start the upgrade there. If the upgrade is successful in the new user account, move all your files to the new account by following the steps below:

  1. Log in to your newly created account.
  2. In Windows File Explorer, click View > Options > Change folder and search option, click View tab and select the Show hidden files, folders and drives, uncheck the Hide protected operating system files check box, and then click OK.
  3. Locate the C:\Users\Old_Username folder, where C is the drive on which Windows is installed, and Old_Username is the name of the profile you want to copy the user data from.
  4. Press and hold down the CTRL key while you click each file and sub-folder in the folder, except the following files:

  • Ntuser.dat
  • Ntuser.dat.log
  • Ntuser.ini

 5. Right click on the selected files and select Copy.

 6. Locate the C:\Users\New_Username folder, and then paste the files inside the new user folder.

Note: Make sure to create a backup of all the important files in your devices before proceeding to the upgrade process.

shishira d say:

Hi drug Boban,

Welcome to Microsoft Answers Community Forums.

To assist you better I would like to ask few question's regarding this issue.

When was the last time it was working fine before?

Did you make any hardware or software changes to your computer?

I would suggest the following steps to help resolve your issue.

Method 1:-

I would suggest you to boot your computer in "Safe mode with networking" and check if you are able to upload the video.

Safe mode is a diagnostic mode of Windows Environment which boots your computer with minimal services and drivers required to boot your computer.

1. Restart your computer and Press F8 key continuously when the system starts and before the windows logon screen appears.

2. You will be prompted with Advanced options, select safe mode to enter into it.

3. In safe mode download and update your video card drivers to the latest version from your  manufacturer's website.

Method 2:-

Perform a clean boot on your computer and then try to upload video, follow these steps to perform clean boot.

  1. Click Start and type msconfig in the Start Search box, and then press ENTER.
    (If you are prompted for an administrator password or for a confirmation, type the password, or click Continue ).
  2. On the General tab, click Selective Startup .
  3. Under Selective Startup , click to clear the Load Startup Items check box.
  4. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All .
  5. Click OK .
  6. When you are prompted, click Restart .

After you have followed these steps boot your computer in Normal mode and check if there is any change, if you notice any change after doing these steps, please have a look in article "How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7" for further troubleshooting using clean boot which you can found from this link, http://support.microsoft.com/kb/929135 .

If you have attached any new hardware component to your computer recently, remove it and check if the problem still persists.

To return you computer from clean boot to normal mode,
   
  1. Click Start and type msconfig in the Start Search box, and then press ENTER.
    (If you are prompted for an administrator password or for a confirmation, type the password, or click Continue ).
  2. On the General tab, click Normal Startup- load all device drivers and services , and then click OK .
  3. When you are prompted, click Restart to restart the computer


Hope this information helped  you.
