jessen p say: Hello,

This error message: An unexpected I/O error has occurred. Status: (0Xc00000e9) is a hardware error pertaining to your system’s BIOS.  Basically it is telling you that the computer is not able to reliably communicate with the hard drive. We can try a few steps and check if it works:
 
Method 1:
Step 1:
I would suggest you to run the Start up Repair Tool from Windows Recovery and check if it helps:
http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7

Step 2:
Also run the SFC (System File checker) Scan in Windows Recovery mode.
Use the System File Checker tool (SFC.exe) to determine which file is causing the issue, and then replace the file. To do this, follow these steps:
a)      Open Command prompt from Windows Recovery Mode
b)      Type C: and press enter. On C:\ prompt type cd windows\system32\ and press enter.
c)       Type the following command, and then press ENTER: sfc /scannow
 
Method 2:
I would suggest you to rebuilt BCD(Boot Configuration data) . To do this, follow these steps:
To run the Bootrec.exe tool, you must start Windows RE. To do this, follow these steps:
a)      Put the Windows Vista or Windows 7 installation disc in the disc drive, and then start the computer.
b)      Press a key when you are prompted.
c)       Select a language, a time, a currency, a keyboard or an input method, and then click Next.
d)      Click Repair your computer.
e)      Click the operating system that you want to repair, and then click Next.
f)       In the System Recovery Options dialog box, click Command Prompt.
g)      Type Bootrec.exe, and then press ENTER.
 h)       Type bootrec /RebuildBcd , and press Enter.
 
For more information refer the link given below:
 
How to use the Bootrec.exe tool in the Windows Recovery Environment to troubleshoot and repair startup issues in Windows
http://support.microsoft.com/kb/927392
 
Hope this information helps.

¡firedog say:

Right-click on the account name in the folder pane and select Properties.

On the Servers tab,

  • change the server for outgoing mail (SMTP) to smtp.live.com
  • Ensure that the Email username is your full MSN email address
  • Ensure that Log on using clear text authentication is selected
  • Ensure that My server requires authentication is selected.

On the Advanced tab,

  • Ensure that This server requires a secure connection (SSL) is selected for both incoming and outgoing mail
  • Ensure that the server port for outgoing mail (SMTP) is 587.

Remove any unsent messages from the outbox (below the account folders in the folder pane), then close down Windows Live Mail and wait a few minutes while the program does its housekeeping and stores the new settings. Is It Ok to run processor at 100% minimum State..? And Is this - Microsoft Community

syed ali imamjafri say:

Is It Ok to run processor at 100% minimum State..? And Is this thing effect on Processor Life or not.?

I have Core i3-3220 Processor.

 

1.If I set processor speed at 100% minimum state so will it effect on processor life or not..?

 

Another Question is

I have Kingston Hyper X black 4 GB 1600 MHz RAM. It is an over locking feature.

  

2. What does over locking means.? And what should the configuration for over locking.?  

hairyfool say:

The 100% figure is purely the clock speed and used to be the only way CPUs would work. Speedstep or throttling allows the CPU to reduce speed to save energy. The CPU is designed to run at that speed and will not suffer any harm. Even when a CPU has no threads to run it cannot actually stop and has to run no-op cycles (similar to soldiers marching time)

 

I think you mean Overclocking. Memory is sold as being able to run at a nominal speed or frequency which is normally it maximum for 100% reliability although they can run slower. They need a specific voltage to run at that speed, overclocking will allow you to vary the voltage to get the memory to run at a higher speed. This is normally done to also overclock the CPU to get better performance.

 

Personally I find the performance benefits of overclocking are so marginal in terms of real use (not benchmarking) that the extra headache of improved cooling needed particularly for the CPU make it not worth it.
