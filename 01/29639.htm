techie007l say:

    Click Start -> Settings -> Update & security -> Recovery.  You should see an option to go back to a previous version of Windows.  Click the appropriate button to start the rollback process.

    If you can't do the above because the Start Menu won't open, try this:

  1. Right-click a blank area on the desktop and click Display Settings.
  2. The Settings app should open.  Click the gear icon in the upper-left side.
  3. Click Update & security -> Recovery.
  4. Click the appropriate button to start the rollback process.

    If the desktop context menu doesn't work either, here's yet another way to launch the Settings app:

  1. Copy all of the text inside of the box above to your clipboard.
  2. Press [Ctrl]+[Shift]+[Esc].  Task Manager should open.  If the taskbar works, you could also right-click/hold-release over an empty spot on the taskbar and click/tap Task Manager.
  3. If Task Manager is tiny and doesn't have a menubar, click the More details dropdown to fully open Task Manager.
  4. Click File -> Run new task.  In the dialog that appears, make sure that the Create this task with administrative privileges option is unticked.
  5. Press [Ctrl]+[V] to paste the text you copied earlier into the Run box, and press [Enter].
  6. The Settings app should open.  Navigate to Update & Security -> Recovery and click the appropriate button to start the rollback process.

    By the way, I fully support your decision to go back to Windows 7.  I have been previewing Windows 10 for almost 10 months now as part of the Insider program, and I regret to say that despite all of our feedback, Windows 10 is a mess, not to mention a privacy nightmare!
