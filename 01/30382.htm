You didn't mention whether you uninstalled AVG before installing Panda. Two antivirus programs on the same machine is a big No-No because each thinks the other is a virus and tries to block it. That in turn causes system instability.

AVG in particular needs to be uninstalled using their own removal tool which you can download from here: http://www.avg.com/eu-en/utilities

Even if you uninstalled AVG already, you should still run the removal tool to delete its registry entries and remove it entirely from your computer.

Try and run sfc /scannow from an elevated command prompt in Safe Mode: https://support.microsoft.com/en-gb/kb/929833
