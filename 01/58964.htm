frederik long say: Whether you get a NAS or a USB connected disk is irrelevant. The difference is purely in the connection method.

You can store several images on one disk. However, with the lower flavours of Windows 7 you can store the image on a local disk only. Networked media are out. This restriction does not apply to third-party imaging products.

frederik long say:

Also with the System Image route I would have to buy Imaging software for both my PC's ($60) + an external HD.

-> Imaging software is included in Windows 7. It's basic but it works (well, most of the time). An external HD is almost compulsory for backup purposes. It is ridiculously cheap.


So is the System Image process really that much better?

-> It depends. Experience says that it takes around one day to rebuild a machine from scratch and about one week to get everything back the way it was. If you have oodles of time and do not mind starting afresh (complete with downloading 200-odd automatic upgrades) then you can skip the image creation bit. I wouldn't - in fact I'm happy to pay for a more powerful and more dependable image management tool instead of using the basic native tool.
