The Windows Action Centre really only recognizes the AntiVirus program (Windows Defender) inbuilt in the operating system. So, after installing AVG Security Program it might have disabled the Windows Defender and that may be the reason why the Windows Action Centre says that the AntiVirus is “OFF”.

The same goes for the Firewall as well, make sure only one is in operation.

You can do this by following these steps:

 

a)       Press “Windows Logo” + “X” keys on the keyboard and select “Control Panel” from the menu.

 

b)       On the “Control Panel” window, change the “View by” to “Large icons”.

 

c)       Click on “Action Center” icon and then click on the link “Change Action Center Settings” from the left pane of the window.

 

d)       On this window under “Security messages” unselect “Internet Security Settings”, “Spyware and unwanted software protection” and “Virus protection” to turn off the Action Center Alerts for these features.
