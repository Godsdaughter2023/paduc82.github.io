Which Antivirus product to choose?

The “Security” software installed on your computer may be causing problems that can not be diagnosed or fixed.  Try removing it temporarily.

There are a couple of prolific chipsets i've seen, the older chips sets don't seem to work with the new driver on the prolific website.

 

Which version: Rev_0300 or Rev_0400 

1. Plug in the adapter and open the device manager.

  • If the driver is loaded, you'll see the device under ports, probably with a Yellow !
  • If the driver isn't loaded it will likely appear under Other (or unknown?) Devices.

2. Double click the adapter to open its properties.   

3. Go to the details tab in the top right.

4. Click the drop down box and choose Harware IDs.

5. Note if it says Rev_0300 or Rev_0400

 

Which Driver Version

  • Rev_0400 works with the lasest drivers from the Prolific Website (Xp, Win7 32 or 64.) 
  • Rev_0300 needs older drivers XP: driver version 2.0.2.24.  Windows 7 (32 and 64) version 3.0.1.0.  (I used NAB Y-105 USB driver from the ReleaCD on the PCEFTPOS website.)

Loading the Drivers:

 

Win7

1. Run the application VISTA.EXE to install the drivers.  Open the device manager and the adapter should be present under Port and may be showing a yellow ! exclamation mark showing in the device manager and reporting Code 10.  (If it looks normal, it's already good to go)

 

2. Right click the device (in the device manager) and choose update driver. Don't let windows update automatically - you need to pick from a list. 

 

After clicking update drvier choose:

  • Browse My Computer for Drvier Sowftware
  • Let me pick from a list of device drivers on my computer
  • Prolific USB-to-Serial Comm Port version [version] > Next

 

 

Windows XP

1. Right click the device (in the device manager) and choose update driver navigating to the list of hardware.  Don't let windows update automatically - you need to pick from a list. 

 

2. After clicking update driver choose:  

Not this time

Install from a specific location

Don't search. I will choose the driver to install

Click Have Disk button

Click Browse and naviagte to your correct version prolific driver files and pick the INF file.

Click open after selecting the INF file

Click Ok now that windows is pointing to the correct file

The device should appear in the list now, select it and click Next

 

If your device was showing under Other Devices (not Ports) in the device manager, when you choose "pick form a list" you will first have to choose a hardware type: Choose Ports, then click next, then the Have Disk button.

 

Remember to use the pick from a list option, not browse directly or automatically install.

 

The files for your driver should then copy over and hopefully the device then appears normally in the device manager.  If it does - it's good to go.  (Also windows may demand restart.)

 

XP:

Rev_0300 use driver version 2.0.0.24

Rev_0400 use driver version 2.1.44.226

 

Win7

Rev_0300 use driver version 3.0.1.0

Rev_0400 use driver version 3.4.48.272

 

Notes:

Windows7 64 drivers must be digitally signed.

I recommend changing the latency settings in ports tab/advanced button both to one for improved stability.
