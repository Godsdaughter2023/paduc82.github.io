a. user say:

Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Hopefully you have a rstore point going back that far.

If that doesn't work, go to Start / Computer / C:\Users\<username>\desktop (and you may also need to go to C:\Users\All Users\Desktop) and see if they contain your desktop items.  If so, copy them all from the folders and paste them onto your desktop.  You will probably need to rearrange them, but that should get them back.

If that doesn't work, your user profile may be corrupt.  To fix this use http://windows.microsoft.com/en-AU/windows-vista/Fix-a-corrupted-user-profile.  If that is the only available Administrator Profile (you need to be an administrator to fix this), enable the Hidden Administrator Account (HAA) using http://www.microsoft.com/communities/newsgroups/en-us/default.aspx?&lang=en&cr=US&guid=&sloc=en-us&dg=microsoft.public.windows.vista.administration_accounts_passwords&p=1&tid=d20f9db4-7b2c-48be-a087-7835dc2a9055&mid=d20f9db4-7b2c-48be-a087-7835dc2a9055.  If you can't remember the password, try nothing as that's probably what you (or the vendor) did during installation.  Once done, be sure to deactivate the HAA to save it in case it's needed again and for security purposes (since people often try to hack into systems using this account).  Do NOT use the HAA as your administrator account because if you lose the only administrator account on the system or it becomes corrupt again then you are hosed.

I hope this helps.

Good luck!

a. user say:

Hello,

Welcome to Microsoft Community. To understand the issue better, I would like to ask you a few questions.

  1. Have you installed Live Photo Gallery?

  2. When was the last time, you were able to connect the SD card without any errors?

  3. Are you able to open/access any picture from the SD card?

 

I would suggest you to try the following methods.

 

Method 1:

If you have installed Windows Live Photo gallery. I would suggest you to try the following steps.

  1. Open the folder containing all the pictures.

  2. Right click on any one of the pictures and select open with

  3. From the list of programs, choose any other program that opens pictures like Windows Photo Viewer and check if you are able to open the picture.

 

If you are able to open the picture on any other program. Refer to the following Microsoft article to set that application as default to open the pictures.

http://windows.microsoft.com/en-us/windows-8/set-program-access-computer-defaults

 

Method 2:

If you are using Windows Live Photo Gallery, refer to the following thread to diagnose and fix the issue.

 

http://answers.microsoft.com/en-us/windowslive/forum/gallery-program/windows-live-photo-gallery-error-when-trying-to/6bafe721-2bbb-4e45-876d-53169a16355b?page=1&tm=1350958953671

 

Let us know the results of the steps mentioned above. Hope this information was helpful.
