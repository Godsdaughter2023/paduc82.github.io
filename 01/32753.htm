Hi spannermonkey,

This error is due to the following:

  • Update is superseded by a newer version that is already on your system.
  • Update is already installed.
  • Wrong update for Windows version and/or CPU architecture.
  • Missing prerequisite update.

Resolution:

  • Make sure that the standalone installer matches your current Windows 10 version (1703, 1709, or 1803). Check this by running winver command in Cortana search.
  • Verify that the package that you are installing matches the processor architecture of the Windows version that you are using (x86 or x64). Check your processor architecture by running msinfo command in Cortana search, then look for the System Type section.
  • Download the Media Feature Pack for N versions of Windows 10 from this link.

Please refer to this article for more information about this error.

m. keithlabove say:

NealSD

Not Microsoft here, but a couple of things I'd mention that may help. (I had similar issues on an earlier upgrade). 

To add Chrome (or any other executable) to the taskbar or to pin to Start, first right click the executable, choose Send to, select Desktop, and that will create a shortcut to the program on your desktop.  Then you should be able to right click the shortcut, and pin it to either taskbar or Start.

Secondly, In addition to the location you mentioned (ProgramData), it seems the  All Apps menu also looks in C:\ User\[your name] \AppData\Roaming\Microsoft\Windows\Start Menu\Programs.  Try copying that whole Programs folder elsewhere (for safekeeping), and then delete everything any it.  You can copy any missing program shortcuts from your backup location to Programs under C:\ProgramData, and they may then show up in the All Apps.  Maybe.

A kludge of a workaround that I used for a while:  Right click the taskbar, choose Toolbars, New Toolbar, and then point to the Programs folder you mentioned under Program Data. This will at least let you access the shortcuts that are there.

I found that Windows 10 All Apps menu doesn't play nice with Start Menu folders nested more than one deep.

Hope some of that helps you work around this.

cbarnhorst say:

No, you no longer have the one-year to upgrade.  You have forever.

The upgrader has registered your computer as a Windows 10 Device.  That is permanent.  You can check what edition of Windows 10 you have by typing "winver" in the Search the web and Windows box (next to the Start button).  A small page will open.  Read the paragraph and it will mention Windows 10 Home or Windows 10 Pro.  The edition of Windows 10 you have is the same as your Windows 7 (all home editions become W10 Home) and both W7 Pro and Ultimate become W10 Pro.

To make sure your activation is permanent, type "command" in Search... and click the top result.  A black window opens with a blinking cursor.  Type

slmgr /xpr 

Hit Enter.  Another page opens which should say "Permanent Activation."

At any time you can Reinstall Windows 10 and it will activate.  You do not need a Product Key even if you are asked for one during the install.  If a page ask for a pk, make no entry and "skip" or "Next" or whatever gets the installer back to work.

You can create media for a dvd or usb to do the re-installation or upgrade when you decide to do so.  Go to

https://www.microsoft.com/en-us/software-download/windows10

and follow the directions to create dvd or usb for future installation.

You are out of the Get Windows 10 program now and have claimed your free Windows 10
