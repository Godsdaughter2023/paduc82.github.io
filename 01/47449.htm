princess mat say:

For you to enable your PC into a WiFi Hotspot, you need to have a compatible and updated Windows 7 wireless network adapter driver. Once you have it, you can follow these steps to turn your laptop into a WiFi Hotspot:

  1. Click the Start button, type Command Prompt, right-click and choose Run as Administrator.
  2. Type netsh wlan set hostednetwork mode=allow ssid=MyNetworkhere key=Password
  3. Press Enter. Change MyNetworkhere with your desired WiFi name and create a password of your choice.
  4. Type netsh wlan start hostednetwork in the Command Prompt, press Enter.
  5. Go to Control Panel, select Network and Internet > Network and Sharing Centre > Change Adapter Settings.
  6. Right-click your Internet connection, choose Properties.
  7. Select the Sharing tab, check Allow other network users to connect and choose the name of your WiFi adapter.

Let us know if you need anything else. We're here to help.

You can reset Windows Update components by running the Fixit on this page. But, if there's malware present, it will continue to reset the connection to the update servers:

How do I reset Windows Update components

http://support.microsoft.com/kb/971058

Suggest you download and save the Fixit. Then configure the system to Clean boot prior to running it:

How to troubleshoot a problem by performing a clean boot in Windows Vista/Windows 7

http://support.microsoft.com/kb/929135

Once the Fixit has been downloaded and the system booted to the Clean state, check to see that the native Windows firewall is now On if a 3rd party firewall was being used previously. Now run the Fixit and choose the Default mode. Restart once it's done and see if the system can connect to the update servers. If it can not, then run the Fixit again and choose the Aggressive mode. Restart when it's finished running and check for updates.

I suggest before shutting down and installing any updates, disconnect from the Internet first.
