It is technically possible, if the machine meets the minimum system requirements to run Windows 10 and supports modern hardware requirements, then, you should be able to install it.

How will I know if my computer can run Windows 10?

System requirements
Processor: 1 gigahertz (GHz) or faster processor or SoC

RAM: 1 gigabyte (GB) for 32-bit or 2 GB for 64-bit
Hard disk space: 16 GB for 32-bit OS 20 GB for 64-bit OS
Graphics card: DirectX 9 or later with WDDM 1.0 driver
Display: 1024x600

To install Windows 10 on your PC, the processor (CPU) must support the following features: Physical Address Extension (PAE), NX, and SSE2.

Once you have satisfied these requirements, there are only two options for installing WIndows 10:

Option 1. A clean install - Windows 10 setup is incompatible with Windows 95, so you would have to boot from the install media, then perform a clean install:

Step 1: How to download official Windows 10 ISO files
http://answers.microsoft.com/en-us/windows/wiki...

Step 2: How to: Perform a clean install of Windows 10
http://answers.microsoft.com/en-us/insider/wiki...

Option 2: Perform a marathon upgrade:

Step 1: upgrade from Windows 95 to Windows 2000 Professional or Windows 98 SE (if your end goal is to install a Home version of Windows 10).

Step 2: upgrade from Windows 2000 Professional to Windows XP Professional (if you chose 98 SE, upgrade to Windows XP Home)

Step 3: upgrade from Windows XP to Windows 7

Step 4: Upgrade from Windows 7 to Windows 10

As you might imagine, this is not guaranteed to work out perfectly. But if this is a fun project you just want to try out for experimental purposes, then these are your options.

Otherwise, you should purchase a brand new laptop running Windows 10 Pro in 2018.

Hello,

A few possible reasons can be the cause of this error code. It is possible that this is due to system file corruption after a failed reset. To diagnose and fix this problem, let's start your PC in safe mode from the Windows recovery screen. After three failed boot attempts, Windows goes into recovery mode. It is a blue-colored screen with a title Choose an Option. To start in safe mode, follow the steps below:

  1. Use the power button to restart your computer three times. This will start the Windows Recovery Environment.
  2. On the Choose an option screen, select Troubleshoot > Advanced options >Startup Settings > Restart.
  3. After your PC restarts, you'll see a list of options. Select 5 or F5 for Safe Mode with Networking.

If the issue persists, we suggest that you repair your PC using an installation USB or disc. To create an installation media and use it to repair your computer, follow the steps below: (You may skip steps 1-5 if you already have an installation media)

  1. On a working PC, go to the Microsoft software download website.
  2. Download the media creation tool and then run it.
  3. Select Create installation media for another PC.
  4. Choose a language, edition, and architecture (64-bit or 32-bit).
  5. Follow the steps to create installation media, and then select Finish.
  6. Connect the installation media you created to your nonfunctional PC, and then turn it on.
  7. On the initial setup screen, enter your language and other preferences, and then select Next. If you do not see the setup screen, your PC might not be set up to boot from a drive. Check your PC manufacturer's website for info on how to change your PC's boot order, and then try again.
  8. Select Repair your computer. You will then be able to perform Startup Repair to repair your PC.
  9. On the Choose an option screen, select Troubleshoot > Advanced options > Startup Repair.
  10. You can also use the installation media to start your PC in safe mode or reset your PC.
