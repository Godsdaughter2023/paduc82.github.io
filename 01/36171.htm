I suggest you to run the Powershell commands to uninstall the apps and check if it resolves the issue.

1. Press Windows Key + Q, and in the search box, type Powershell.

3. Type Get-AppxPackage and press Enter.

(This command will output a long list of all installed apps containing lots of information about each app).

Note:

Get-AppxPackage -allusers PackageFullName | Remove-AppxPackage

If you want to remove the app from any particular user account, use following command:

Get-AppxPackage -user username PackageFullName | Remove-AppxPackage

(Replace username in the above command with the correct user account name).

lemp say:
I have a Gateway computer, running Windows XP SP3, that has an internal SDHC card reader and it is not reading my 8 GB SDHC card.  It reads a 32 MB SDHC card just fine; I don't have any intermediate size cards to try to see which other sizes might work and which ones won't.  From other questions posted it seems there may be problems with SDHC cards once the size reaches 2 or 4 GB.  Could this be an addressing problem, not being able to address memory greater than this size?  Questions and answers about maximum memory for Windows XP 32-bit suggest that there is a 4 GB limit for physical memory in the computer; is there any reason this would apply to SDHC memory cards?

The problem with reading SD cards > 4GB is that these are High Capacity (SDHC) cards, and older SD-card readers may not be able to read them.  If your card reader can read a 32 GB SD card, then that isn't your problem.

The 4 GB limit for RAM for 32-bit operating systems like Windows XP has nothing to do with the ability to read a mass storage media device like your SDHC card.  

This sounds as if this particular card is bad.  The only practical method of testing that I can think of at the moment is to try the card on another computer.  Does the 8 GB card work in your camera or other device?

Do you have an external card reader in addition to the built-in one?  

If you have data on the card that's important to you, it might be worthwhile either to purchase an inexpensive USB SDHC card reader or to take the card to someone who has one (I suspect that FedEx/Kinko or Staples might even be able to copy your files off for you -- if the card is not bad).
