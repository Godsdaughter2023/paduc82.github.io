The reason you are facing this error is because the mouse drivers are not updated. I suggest you to go to device manager and update the mouse drivers. Also make sure all your Windows updates are up to date. Most of the issues are fixed by installing Windows Updates make sure that your system is up-to-date. Type Windows Updates in search bar and select Check for updates from the results. If updates are available, try installing them and check.

I would also suggest you to run the hardware troubleshooter and check if it helps.

  • Type Troubleshooting and click Troubleshooting under Settings.

  • Now type Hardware Troubleshooter in the search option.

  • Click on Hardware troubleshooter to run the troubleshooter.
