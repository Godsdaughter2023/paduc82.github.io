SpeedFan

 

SpeedFan is a program that monitors voltages, fan speeds and temperatures in computers with hardware monitor chips. SpeedFan can even access S.M.A.R.T. info and show hard disk temperatures. SpeedFan supports SCSI disks too. SpeedFan can even change the FSB on some hardware (but this should be considered a bonus feature). SpeedFan works fine with Windows 9x, ME, NT, 2000, 2003, XP, Vista, Windows 7, 2008, Windows 8, Windows 10 and Windows Server 2012. It works with Windows 64 bit too.

 

SpeedFan - Access temperature sensor in your computer

lemp say:

I don't know where you found gifview.rar, but *.rar files are a form of compressed files (like *.zip files).  You often find *.rar files on dubious websites, and you should always scan these files before using them.  Where on the Internet did you find gifview in the first place?

You would have had to use WinRAR, 7-zip, or another "archiver" program to extract the files that were contained in gifview.rar because the archive function built into Windows doesn't work with *.rar files.

Yes, you should delete gifview.exe, the shortcut to gifview.exe, and gifview.rar.  I don't know what you mean by "leftovers etc."  I have no way of knowing what might have been packaged into gifview.rar.

I don't know what gifview.exe.3.3.2.0. is but you should probably delete that also.  Before you delete it, however, right-click on it and select "Properties" and select the "Details" tab.  Is there any useful information there?

rundll32.exe and shimgvw.dll are Windows system files that should be in C:\Windows\System32.  Shimgvw.dll is Windows Photo Viewer.  You should NOT remove these files if they are in C:\Windows\System32.  However, if these files are located elsewhere on your computer, they possibly are malware.

If you are going to download strange files from the Internet, you should have strong antivirus software installed.  What are you using for an antivirus program?  What other antimalware or security software is installed?
