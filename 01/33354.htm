bobmcfly247 say:

As Rozilla already stated:

It is not possible, to use dmz and portforwarding at the same time, even if you may be able to configure both on your router. As soon as a portforwarding is set up, these ports are not longer available for dmz.

So you really need a router which is able to serve all four Xboxes by only UPnp. Best experiences on multiple console support can be found within DD-WRT able routers.

So there is a really good solution possible  if you are a little bit tech savvy at all. Set up a DD-WRT router whether on a virtual (Hyper-V) machine or on a small (and even older) PC who has Linux supported hardware.
