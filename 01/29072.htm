Hi,

The D: partition is a separate partition on your hard drive used to reinstall the operating system back to new condition and you leave it alone.

Read this about it:

Your Recovery Partition was imaged on your Hard Drive during the Manufacture of your Computer by the Manufacturer for these reasons.

 

1.To reinstall Windows from in case of a System crash, by pressing particular Key or Keys at Boot.

F10 or F11 or Alt + F10 are a couple of sequences different Manufacturers use.

You will have to ask your manufacturer for the correct sequence.

 

2. To make Recovery Disks from in case your Hard Drive fails, so that you can then reinstall the Operating System on to a new Hard Drive.

Also ask them how to make the Disks.

 

Your D: Recovery Drive is not to be used for backups, defragged, etc.

Backups should be kept on an External Hard Drive.

 

To fix any problems that have occurred with the D: Recovery Partition, you will have to contact your Computer Manufacturer to remove whatever has been added to it.

Problems with decryption may occur if a different login was used to encrypt the files which you are attempting to decrypt or due to a possible file corruption. Kindly try decrypting your files using the Cipher command with the steps below:

  1. Type CMD in Cortana/Search and press Ctrl + Shift + Enter to run the Command Prompt as an administrator.
  2. On Command Prompt, navigate to the folder where the encrypted files are using the CD command.
    • For example, if the folder where the encrypted files are is at C:\Users\Username\Documents folder, type the following command: CD C:\Users\Username\Documents
  3. Type the following command:
    • cipher /d *
  4. If you wish to decrypt a specific folder's contents and its sub-directories, run the following command instead:
    • cipher /d /s:"<insert folder name here>"
