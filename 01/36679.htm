larrypaise say:

If you got the full version of Partition Wizard it allows you to batch operations. Get into its help system and explore the following operations.

You will have to first shrink the partition with free space to unallocate space that you ultimately want to add to the Reserved System Partition. Then, before committing that change, you can actually use the UI to slid the now unallocated space around virtually on the disk to make it adjacent to the Reserved System Partiiton in the UI. Once it is adjacent (either immediately in front of or behind) the Reserved System Partition you should be able to tell Partition Wizard to extend/enlarge the Reserved System Partition.

I had to play around with it a bit to get the right set of operations. The "Undo" feature in the UI is very handy if you do something you want to change before actually committing it.

Once you have it exactly the way you want it, you can "Apply" all the changes and Partition Wizard will apply all the necessary operations in one batch set, one after the other.

As for buying a Windows 10 Upgrade, you could do that and reformat your drive from scratch but then you lose all you in place data and apps. Even with a backup -- Ugh!!
