Check and reseat all cards, memory and cables (on both ends when possible) actually
remove and replace - do not just snug. On a laptop about all you can do is reseat the

lemp say:

You have to be careful moving or deleting files from AppData.  There's a reason that the AppData directory is hidden -- Windows doesn't want casual users making changes here.

Windows uses 3 different types of re-direct devices:  hard links, junctions, and symbolic links.  Which of these you use depends on whether the "target" (the actual object) is a file or a folder and whether it is on the same volume as the link.

You've moved the object to a different volume and as I understand you, you've moved a single file rather than a folder.  Thus, you need a symbolic  or "soft" link.

Assume that the file you moved was C:\Users\Blaine\AppData\Local\Some Program\Large File

Assume that it is now E:\Moved Files\Large File

You should then do the following:

1.  Open an Elevated Command Prompt (click Start, type Command Prompt in the Search box; right-click on the link to Command Prompt and select "Run as administrator"; respond as needed to the UAC prompt)

2.  Type the following line in the black Command Prompt window and press Enter

        mklink "C:\Users\Blaine\AppData\Local\Some Program\Large File" "E:\Moved Files\Large File"
