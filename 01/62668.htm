hairyfool say:

Superfetch is a mechanism whereby commonly used files are preloaded to improve performance.

 

Safe mode is going to do little to help performance as most drivers and services do not load in safe mode so many devices will not work properly. My boot times are actually longer in safe mode than normal.

 

There are a number of logical steps to identify why a computer is slow

 

Identify and remove all unused programs.

 

Remove programs that preload during startup that are not required such as some software updaters.

 

Consider the amount of installed memory and if an increase would help.

 

Make sure the boot drive is not full, move files off to a second drive if neccessary. Windows defragmenter needs at least 15% free working space or look at an alternative defragger as the Windows version is not very effective.

 

Consider alternative A/V solutions, Norton and McAfee are renowned for slowing systems down. If you want to keep then because of a subscription uninstall them, use the uninstaller from the respective website to clean up and reinstall with the latest version to optimise the software.

Hi,  

 

Thank you for posting your query in Microsoft Community.

 

I understand the inconvenience you are facing currently. Let me help you in resolving the issue.

I would like to suggest that you perform Check disk to fix any errors that may be present  in your hard drive. To do so follow the steps mentioned below:

Steps:

1. Insert the USB which contains the Media Creation Tool. Access Windows "Recovery Environment" .

2. Once there locate and select "Command Prompt" .

3. Type the command below and allow the system to check your hard drive for any errors

4. Type at the command prompt CHKDSK /P and press ENTER to scan the disk.
5. Then type CHKDSK C: /F /R and press ENTER to repair corrupt sectors, if there are any.

NOTE: The root drive has to be C: and the command may schedule a task at the system startup, if it does, then press Y to schedule a task and restart the computer to perform chkdsk.
Once chkdsk completes, boot to the desktop and then check.

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.

Warning: Microsoft does not recommend interrupting the CHKDSK process. Microsoft does not guarantee the integrity of the disk if the CHKDSK program is interrupted .For safety you can back up your data before using the check disk command.
