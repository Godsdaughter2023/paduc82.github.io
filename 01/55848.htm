I have a couple of suggestions.  There's a few places the system keeps track of any ongoing installations, so that two can't run at once.  Obviously you never want to install two things at the same time, but it sounds like the flag for "in progress" is stuck on.

Depending on where it's stuck means we may need to check a few places.  Here's what I'd recommend first:

  1. Try restarting the Installer service.
    To do this, right-click Start and choose Run.  Type "services.msc" and then click OK.
    Find "Windows Installer" in the list and double-click it.
    Finally, stop the service with the "stop" button, then click OK and close the Services list.
    Now try installing your program (like Skype or the Sims) again.


    if that doesn't work, the next thing I'd try is this:


  2. Remove the InProgress registry key.
    Right-click Start and choose Run. Type "regedit" and then click OK.
    In the left side, open the folders to get to this path:
       HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Installer\
    Look under the Installer folder (expand it to see the folders underneath it).  If you see a "InProgress" folder there, right-click it and Export it to your desktop (to save a backup of it), then right-click it again and delete it.
    In my example below I do not have an "InProgress" folder (it should only show while an installer is running or just recently ran but is waiting for a reboot).



    After deleting the InProgress folder, restart your computer normally and then try install your program again.

Please let me know if either of these help, or if you have any questions.

System Restore is a wonderful tool - if it works. Unfortunately it fails far too often. I suggest you do this:

  1. Run the command SystemPropertiesProtection.exe to open the System Restore configuration panel.
  2. Turn off System Restore on all partitions. This will delete all existing restore points.
  3. Turn on System Restore on the system partition. This should fix the restore problem.
  4. Plug in your USB backup disk.
  5. Open the Control Panel / File History.
  6. Click Create a system image.
  7. Plug in a small flash disk (1 GByte will do)
  8. Click Create a System Repair Disk. Do NOT copy the system files.

If you ever need to re-install Windows then you can do it quickly like so:

  1. Make sure you have a recent backup of your data files.
  2. Boot the machine with the small flash disk.
  3. Restore the image from the USB backup disk.
  4. Restore your data from your backup.

11.1 In Windows explorer, go to your desktop or any convenient folder.  Select the folder in the navigation pane [the left-hand pane].  Press Shift-RightClick, select Open command prompt here.

How do I set up a regular(weekly) restore point? - Microsoft Community

raymcmurtrie say:

How do I set up a regular(weekly) restore point?

I want to set up a regular restore point because I deleted a file after trying to shift it, but only sent a shortcut, & then deleted the original file. I could not find any restore point at all. How do I set it up?

frederik long say:

I want to set up a regular restore point because I deleted a file after trying to shift it, but only sent a shortcut, & then deleted the original file. I could not find any restore point at all. How do I set it up?

System Restore is not a suitable tool for file backups. There are several reasons:

  • System Restore runs when Windows decides it's time to run. You could, of course, create a scheduled task for this purpose but this is not trivial.
  • Windows will delete System Restore points according to rules that we do not know, in particular after automatic updates.
  • System Restore is designed to back up system files. While it also backs up some user files, we do not know which ones it selects or ignores.

I recommend you set up a robust backup scheme and pay attention to the iron rule of data processing: Important files must at all times be stored on two independent media.

canadian tech say:

Let me add to Fredrik's excellent advice...

Restore is for SYSTEM restore, not a backup.  It can be a saviour in a case when you have installed some miscreant software, but sometimes, if you do not catch it quickly enough, a useful restore point is no longer available.

The default restore point space allotment in Windows 7 is 1 to 3% of drive capacity.  Typically, this does not allow for more than a few restore points.  In almost all cases, the computer is using a very small proportion of the space available. That makes it very practical to raise that 3% to say 8 to 10%.  This strategy will yield far more restore points that will give you much more flexibility in recovering a bad situation.

To change the allotment, type create in the text box above the start globe.  Choose Create a restore point.  Click configure.  Move the slide to the right as much as you see fit.  Setting it at 50 Gigabytes will have no affect if you have a 500 Gigabyte drive and are only using about 100 or so, which is highly typical.
