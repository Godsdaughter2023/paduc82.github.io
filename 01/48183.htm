The ipconfig /all command displays Windows TCP/IP settings for all your network adapters.

• ipconfig /renew in the command prompt and press ENTER

Hello Miller,

 

Thank you for posting on Microsoft Community. We will assist you in resolving the issue.

 

Before assisting further, I would like to ask you few questions:

  1. Did you check if the files and folders are present on the account setup as soon as the upgrade was completed?

  2. Did you check if you are able to find the lost files and folders in windows.old folder?

 

Query 1:

When you set up a Microsoft account in one of the device, the files and folders you save on one machine, can be accessed on another machine, anywhere, anytime when logged in into the Microsoft account as your PC is connected to cloud storage online.

 

It is not necessary that you have to create a Microsoft account to use Windows 10. One Microsoft account on a device is sufficient. However Windows 10 supports more than one Microsoft account. Logging into Microsoft account will give you the benefits to use all the Windows 10 features.

 

To not use a Microsoft account, I suggest you to create a Local account by following the steps below:

 

  1. Type Settings in the search box next to Start button, and select the top most search result.

  2. Select Accounts from the options shown in the window.

  3. Click on Family & other users from the left side panel, and click on Add someone else to this PC option under Other Users.

  4. Click on I don’t have this person’s sign-in information located at the bottom of the window.

  5. Click on Add a user without a Microsoft Account located at the bottom of the window, and follow the on-screen instructions.

 

Query 2:

When you upgrade to Windows 10, the previous Windows files and folders are automatically stored in C:\Windows.old folder. This folder is kept by Windows only for one month. After one month Windows 10 will automatically delete the folder. So you can restore previous Windows version only within one month after Windows 10 upgrade. Also if you run Disk Cleanup in Windows 10, make sure to uncheck "Previous Windows installation(s)" option. If you don't uncheck it, Windows will delete all files and folders present in C:\Windows.old folder and you'll be unable to rollback to previous version of Windows.

 

Let's try to search the files and folders lost using Cortana and Windows Explorer search.

 

Step 1: Using Cortana

 

  1. Press Windows icon key on the keyboard and type the file or folder name in the search box.

  2. Select the search result which shows folder and drive location of the program.

  3. Follow the file or folder path of the picture or video.

 

Step 2: Using Windows explorer

 

  1. Press Windows logo + E keys on the keyboard to open Windows Explorer.

  2. Click on This PC option from the left side panel of the window.

  3. Type the file name in the Search This PC box located at the top right corner of the window.

  4. Follow the file or folder path of the picture or video.
