  5. Select  Microsoft XPS Document Writer and click Remove

Windows 7 and Windows 10 Trusted installer or Tiworker issues - Microsoft Community

rob fulford say:

Windows 7 and Windows 10 Trusted installer or Tiworker issues

I noticed this issue awhile back when cleaning up a PC that was having issues.

I had the task manager open and noticed that Trusted Installer would move to the top of the memory usage as disk clean up was at the point of deleting stored windows updates.  After disk clean up finished the Trusted installer process would drop down in the list.  I would do a restart and windows would finish installing updates during the reboot.  This was when I realized that other users who had high CPU usage from trusted installer could benefit from my fix.  I done this fix on hundreds of machines including my home machine with it fixing just about any network, application, and security issue I was having.  Here is the fix.

From the start menu type disk clean up.
It will run for a few minutes and then pop-up the files to delete window.  Click on find System files button, and wait for it to finish.  I usually choose everything but the issue seems to be with the Windows update Cleanup. After selecting the files click delete.

While this is running I go into the Windows/Temp folder and delete it's contents and if prompted to skip or retry I skip when files are in use.  Also under the C:/ drive in file explorer you might see file folders that are just Numbers and Letters you can delete these files.

I also go into the control panel and open configuration manger select the Cache tab.  Select the Configure settings button, Note I usually change the Amount of disk space to use to 12,000, select the Delete Files button.    When you select the delete files button another window appears.   Check the box to include "Delete persisted cache content" and then click the delete button. and wait for it to finish.  Close this window.

Now open a task manager window and chose the process tab. at the bottom choose the button to show processes for all users.  Sort the process by memory usage by clicking at the top of that column and you will see Trusted installer at the top along with cleanmgr.exe.  Wait for them both of these processes to finish. 

Note this can take awhile I've had it take an hour for the memory to drop down from the hundred thousands.  This is usually when the disk cleanup window goes away.
Now do a restart and watch as windows will run updates and reboot.
Note: This also works in Windows 10 but the process is Tiworker, instead of TrustedInstaller. and the above process will work and most times will show a little notification by the time stating that windows has installed new updates and needs to reboot.
