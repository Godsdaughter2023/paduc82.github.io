jacobbarbarow say:

I was trying to do the same thing, this is my solution.

Open notepad and type this in, substituting Wi-Fi for the name of your adapter, leave the "".

netsh interface set interface name="Wi-Fi" admin=disabled
netsh wlan set hostednetwork mode=allow
netsh wlan start hostednetwork
netsh interface set interface name="Wi-Fi" admin=enabled

Save this as a batch file, so put in whatever name you want and put .bat on the end.

Run the batch file as administrator every time or it will not work.

This should do everything you asked for

In addition, if you would like it to stop the network when you want it to then add this beneath the last line.

pause

netsh wlan stop hostednetwork

This will leave the command prompt open and waiting for any key to be pressed to carry out the command.

Remember to share the connection of the adapter that has internet to the hosted network or the devices you connect won't have internet access.

peer_sam say:

Hi,

Apologies for the delayed response.

While installing Sublime text, you will get an option ‘Add to explorer menu’, check the checkbox to add the option to explorer menu (Right click).

  • To set Sublime text as default program for .vhdl files, right click on the vhdl file and select Open with,

  • If Sublime text is not listed in compatible programs list, click on Choose another app.

  • If you still do not see Sublime text in the list, click on More apps option,

  • If Sublime text is listed in the list of apps, select it and check the checkbox for Always use this app to open .vhdl files.

  • Click Ok to save the settings.

  • If the program is not listed, click on Look for another app in this PC and navigate to C:\Program Files (86)\Sublime Text\sublimetext.

  • Make sure that sublimetext (application) is selected under Sublime Text folder and click on Open.

  • This should set Sublime text as a default text editor for .vhdl files.

Hope it helps.
