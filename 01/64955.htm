Windows Update not working

This is a very common problem lately.  My best guess is that it has a 10% probability.  In most cases, the following rather simple and quick process works.

http://answers.microsoft.com/en-us/windows/forum/windows8_1-update/windows-update-problem-error-code-80080008/de8114b5-1487-4b31-9b28-f11974718df1?auth=1

This thread essentially lists a series of commands that I have found usually fixes WU.

  1. Start, All Programs, Accessories
  2. Right-click on Command prompt
  3. Choose Run as administrator
  4. Yes
  5. net stop wuauserv
  6. net stop cryptSvc
  7. net stop bits
  8. net stop msiserver
  9. ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
  10. ren C:\Windows\System32\catroot2 catroot2.old
  11. net start wuauserv
  12. net start cryptSvc
  13. net start bits
  14. net start msiserver
  15. pause
  16. Exit

Now re-start Windows Update.  Be very, very patient, it will likely take a long time.

If that does not produce results, you should download and run the following.  It takes a very long time to run as well.

The 64 bit version Microsoft’s “System Readiness tool” 

The 32 bit version.

curse1 say:

I personally would do a power cycle for the system to see if that clears the issue up. If that doesn't do the trick I would uninstall the game and do a fresh install to ensure no goofy business is happening. If you are not familiar with the power cycle process I will list the steps out for you:

Follow these steps to power cycle your Xbox One Console:

  1. While the console is on, press and hold the console’s Xbox button for 5 seconds.
  2. The console will power down.
  3. ***Extra step I personally take***- Unplug the power cable from the Xbox to ensure no power is going to the system. 
  4. Wait 30 seconds, and then turn your console back on by again pressing the Xbox button.

Note If your console is in Instant-On power mode, the above steps will shut the console completely off. Instant-On mode, or the ability to turn on your console by saying “Xbox On,” won’t be enabled until you restart the console.
