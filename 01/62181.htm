Retail Product keys are normally handled differently than Digital Licenses.  They can migrate without needing a MS Account just by moving them to different hardware.  If the activation server sees a previous install still associated with that key, it will ask "How many PC's is this key installed upon?" for which the only accepted answer is "One" unless it is a rare multiple-use key.  This gives MS the go-ahead to deactivate it on any other hardware still holding association with that key.  At least this is how it worked previously.  Now that we are seeing retail Product Keys also become DL's there is more to learn.

Always delete the old install first thing so there is no confusion.   If you want to keep the install then if necessary use the Activation Troubleshooter if a DL is associated with your MS Account, use Contact Support app if any problems.  Reactivating Windows 10 after a hardware change
