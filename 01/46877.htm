a. user say:

Hi Lee,

Thank you for posting in Microsoft Community. I understand your concern and I am glad to assist you.

The answer would be Yes to your first question.

The OEM Partition ( usually labelled D ) on your old drive would have contained the files necessary to reinstall the operating system on the main C Partition without having to use your Recovery Discs ( which are a bootable copy of this ).

 An OEM partition can be a "diagnostic" partition which holds a program or other information used to run diagnostics without booting into Windows. It is usually formatted with some weird file system. It can also be a "system partition" set up by Vista or 7 which holds files needed for the boot process. That type of partition is usually just about 100 mb in size. You can delete the diagnostic partition but if you delete a system partition the drive may no longer boot.
