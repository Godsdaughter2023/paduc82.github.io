cmdrkeene say:

Registry doesn't really contain files, but yes it could leave behind some references in the registry database. However that's of no concern:  unless something specifically queries the database and asks for the information, it isn't actually slowing down your computer or using any resources.  It's far better to leave it than to try to 'clean' it.

If the software vendor has removal instructions, or a list of files the program uses as reference, it's good to use that.  Another option is an uninstaller utility like Revo Uninstaller.  Sometimes you can even re-install the program normally, then use the uninstaller to remove it the recommended way (unless that's what left stuff behind the first time).

But I can't stress this enough: don't try to remove hives from the registry database. If you are feeling risky, at least make a full backup first.  But the database just having some entries won't affect your performance in any way, the registry works like Google: it only provides results you ask for. If you never query for the other entries, it doesn't matter if they exist or not, it's not impacting your performance.

        davem121 say:

        If all the ways you have found on the Web do not work, try this:

        1. In Windows File Explorer, on the View Tab, click 'Options'
        2. In the resulting dialog scroll down through the list and uncheck the option 'Hide protected operating system files'
        3. Click Apply and Ok
        4. Back in Windows File Explorer, open the folder C:\$Recycle.Bin
        5. Inside that folder there is a Recycle Bin Folder
        6. Right click that folder and choose Send To - Desktop (Create Shortcut)
        7. Now you will have a shortcut to your Recycle Bin on your Desktop

        For safety sake, you need to hide operating system files again

        1. In Windows File Explorer, on the View Tab, click 'Options'
        2. In the resulting dialog scroll down through the list and check the option 'Hide protected operating system files'
        3. Click Apply and Ok
