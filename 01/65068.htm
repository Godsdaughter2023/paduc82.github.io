andre for directly say:
Volume License keys come in two activation methods:

KMS - Key Management Server - this method requires connecting to a server host with an organization every 180 days (6 months) to reactivate. If Microsoft determines this is an illegal KMS host, it can shut it down and it will no longer be able activate clients.

MAK - Multiple Activation Key - an organization can purchase a MAK license which is similar to a retail license. A key can carry 5, 10, 25, 100 or even a thousand activations.

Id the product key is sold multiple times and is exhausted, it could easily compromise the license and Microsoft blocks it.
