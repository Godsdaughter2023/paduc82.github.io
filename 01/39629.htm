Hi Munro,

Welcome to the Microsoft Community forum, we’re here to help you with this issue.

The built-in mail app automatically retrieves all emails available in the email accounts that you add. But in this case, as the emails are available only as an offline copy, saved in the Outlook express, these emails needs to be exported as a file and then imported using an email client like Windows live mail which is available for Windows 8.1.

Follow these steps to export the messages from outlook express:

a)      The folder that contains the Outlook Express messages is marked as a hidden folder in Windows. To view this hidden folder, click Start, and then click Control Panel. Click Appearance and Themes, and then click Folder Options. Click the View tab, and under Hidden files and folders, click Show hidden files and folders.

b)       Locate the folder named Outlook Express. In Windows XP, this folder is located here by default: C:\Documents and Settings\your user name\Local Settings\Application Data\Identities\{your Windows user identity number}\Microsoft\Outlook Express.

c)      Copy the Outlook Express folder and all its contents to a thumb drive/pen drive.


Now, download and install Windows live mail in Windows 8.1:


http://windows.microsoft.com/en-in/windows-live/essentials-other#essentials=mailother


Note: The Windows Live mail client comes as a part of the Live essentials.


Once you have configured live mail, connect the outlook folders to a location in your Windows 8.1 computer and then follow these steps to import the mails.

a)      Open Windows Live Mail 2012 and click the File menu and then click on 'Import messages'.

b)      Click Microsoft Outlook Express 6, and then click Next.

c)      Click Browse, find the Outlook Express folder you copied to this computer, and then click the folder once to select it. (Do not open the folder when you select it.)

d)      Click Select Folder, and then click Next.

e)       Click All folders to import all the folders, or click Selected folders and select which message folders you want to import, and then click Next.


Try these steps and let us know the results. We’d be happy to help if you need any further assistance.

The reason is it has been just a month since windows 10 release and I guess you have just upgraded and even if there is a restore image created in your PC it is not going to be of much use to solve the bugs.

So the solution? well clean installing the OS can solve your problem. clean installed windows 10 is fast and stable.

To clean install just make sure your current upgrade OS is activated by going to SETTINGS>UPDATE AND SECURITY>ACTIVATION or you can look it up in the system properties.

please Back up all your important data and software before doing this.

STEPS TO CLEAN INSTALL WINDOWS10:

1)use your pc or laptop and download the media creation tool from

2)make sure to download the appropriate version i.e. either 32 bit tool or 64 bit tool. download that architecture of the tool which the your computer os  has which you want to use to run the tool.

3)after running the tool you will have to choose "create installation media for another pc" option and follow the steps. MAKE SURE THAT YOU CHOOSE TO CREATE THE EXACT SAME VERSION OF WINDOWS 10 THAT YOUR PC RUNS OR YOU UPGRADED TO i.e. if it runs windows 10 pro 64 bit then choose windows 10 pro 64 bit.

4)choose either usb or dvd disk as per your choice (I would advice to go with USB) to create installation media.

5)once the procedure is completed then remove and reinsert the installation media and restart your pc.

6)start your pc and as soon as you see the boot screen press F2 or F12 to access the bios. from the boot options select the usb or dvd disk on which media creation tool is created and then press enter.

7)installation procedure will start. if it asks for product key select skip option. select the drive on which you want to install and just clean install the os.when clean installing, you will be asked twice to enter Product key or license. First when on initial start up choose "SKIP THIS" option and when asked after installation but just before opening choose "DO THIS LATER" option.

i hope it helps  and have a go at the F.A.Q's on media creation tool on Microsoft website.

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install?ocid=ms_wol_win10

having a Microsoft account connected to your current free copy of windows 10 makes your OS validity more reliable.

I hope this helps you
