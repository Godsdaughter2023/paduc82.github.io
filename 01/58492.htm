Windows 10 updated Asus Laptop speaker icon button increase or - Microsoft Community

fortech say:

Windows 10 updated Asus Laptop speaker icon button increase or decrease doesnt work

Windows 10 updated Asus Laptop speaker icon button increase or decrease doesn't work

miguelchiappori say: Hi,

Connect to Windows Update and try to install all available Windows updates including optional updates and see if this solves the problem. If the problem persists, check if the sound is restored after applying one of these methods:

Method 1

1. Press the Windows + X keys and select Control Panel.

2. Click on Search and correct problems.

3. Click on View all in the left panel.

4. Click on Hardware and devices.

5. Click Advanced and Next. Follow the instructions on the screen to solve the problem. If the sound problem persists, go to method 2.

Method 2

While on the screen Solving problems with the computer, where you were in method 1, click now on the Audio playback solver, and follow the instructions of the solver.

Method 3

Uninstall the sound adapter driver from the Device Manager and restart the computer. Follow these steps:

1. Press the Windows + X keys and select Device Manager

2. Double click on Sound and video controllers and game devices to expand it

3. Right click on the sound device and select Uninstall. Check the box (if it appears) Uninstall also sun sound software.

4. Restart the computer.

5. Observe when restarting if the sound has been restored on the equipment.

Method 4

Update the sound driver from the Device Manager. Follow points 1 and 2 of method 3. Next, right click on the sound device and select Update driver software. Wait for the driver update process to complete and restart the system.

Method 5

Check the sound reproduction options. Follow these steps:

1. Press the Windows + R keys

2. In the Open box, type mmsys.cpl and press Enter.

3. On the Sounds tab, note that the Sound Combination is set to Windows Default. If not, rectify it and apply the changes.

4. On the Playback tab, select Speakers and Properties.

5. In the Speaker Properties window, General tab, note that Use this device is enabled. On the Improvements tab, check the Disable all sound effects box; and on the Advanced tab, it is a matter of testing the speed and depth in bits of the sound. It is possible that by raising or lowering the sound quality, it will be restored. Close the properties window and see if these changes in the sound configuration solve the problem.

On the other hand enable System Restore:

1. Press the Windows + X keys and select System

2. Click on System protection

3. Select your operating system and click Configure ...

4. Mark Activate system protection and establish a minimum of 10% maximum usage. Save the changes
