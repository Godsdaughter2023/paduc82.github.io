    Hi Lsarro, in version 1803 of Windows 10, Microsoft, without warning removed the Homegroup functionality . ..

    If you have a problem seeing each PC in the Network Section in File Explorer:

    Click your Start Button, type Services and hit Enter

    When the Services app opens, scroll down to find the following services, then double click each one in turn, set their startup type to Automatic and start the service if necessary

    1. Function Discovery Provider Host

    2. Function Discovery Resource Publication

    3. SSDP Discovery

    4. UPnP Device Host

    Save that and close the Services app

    That's It!



    If you get a Network Credentials popup, you should be able to turn off that requirement . . .

    Open Control Panel

    Go to Network and Sharing Center

    Select Change advanced sharing settings in the left pane

    Expand the network you want to disable password protected sharing on by clicking the down arrow on the right of the profile

    Select 'Turn off password protected sharing' and then click Save changes.

    bossdweeb say:

    "If the app is "not installed" according to the regedit then can I just simply delete the fold that the app is on so that it doesn't run? "

    Yes. The main reason for uninstalling via Programs and Features is to remove the name from the list, and

    the Registry keys (although some don't clear their Registry keys either).

    Uninstalling via P&F also facilitates reinstalling, as you cannot reinstall if Windows does not know you

    deleted the files.

    Even when uninstalling properly some software will leave folders with your settings or other information

    behind in your User \ Documents or App Data folder, as well as the install location. These have to be deleted

    manually.


    However, if the program is still launching with Windows ( verified in Task Manager > Processes, or by pop-ups, etc. )

    your software probably falls into the category of PUP (potentially unwanted program), or malware, which

    is a fairly common issue with some 'free' software.

    These programs sometimes hide themselves away (mostly popups, rather than the full program) somewhere

    outside the 'official' install folder, much as  does a virus/malware.

    This sort of program can also be installed along with another free software you did want (including legitimate

    software such as Adobe Flash Player ), and can easily happen from inattention during the setup when installing.

    They often default with a tic in the box next to an offer to install the program in question. 

    eg - Bing, Chrome (comes with Adobe Flash) among others, so you have to be careful to clear the

    tic box before installing.

    Many such programs have tutorials available on how to fully remove them, and this appears to be one of them.

    https://www.google.com.au/?gws_rd=ssl#q=bytefence+removal&*

    Trusted Platform Module (TPM) Management is a new feature set in Windows to administer the TPM security hardware in your computer. A TPM is a microchip designed to provide basic security-related functions, primarily involving encryption keys. The TPM is usually installed on the motherboard of a computer or laptop, and communicates with the rest of the system using a hardware bus.

     

    A Code 10 error is generated in Device Manager in one of the following situations:

    1. Device Manager cannot start the device.

    2. One of the drivers that the device needs does not start.

    3. Device Manager has insufficient information to recognize the error that is bubbled up by the device driver.

     

    Note: In this situation, the Code 10 error can be a very generic message that indicated some kind of unspecified driver or device problem.

     

    Refer to the link below on “This device cannot start” Code 10 error in Device Manager in Windows and check if the article helps in resolving the issue:
