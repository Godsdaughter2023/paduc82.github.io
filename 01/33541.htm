Did you install any programs or updates prior to the freezing?

You can try doing a clean boot to determine if another program is causing the freezing – this will disable all non-Microsoft services/programs.

Go to Start, type “msconfig” in the search box and press enter. You may be prompted to continue. On the System Configuration screen, go to the Services tab, check the Hide all Microsoft Services option and click Disable All. Next, go to the Startup tab and press Disable All, then restart your computer and test.

If the problem goes away, another program is causing the problem to happen. To identify what one, you will need to enable half of the services/programs at a time and test again to narrow it down.

To restore your computer to a normal boot, simply run the “msconfig” program again and select the Normal Startup option on the General tab.
