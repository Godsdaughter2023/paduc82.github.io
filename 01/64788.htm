Since this is a recent issue (worked before and is now a problem), have you considerd doing a System Restore to a point before this issue started?

Running the System Restore utility…

1.     Click Start, click All Programs, click Accessories, click System Tools, and then click System Restore.

2.     Click Continue or provide Administrator credentials if necessary.

3.     Click Choose a different restore point, and then click Next.

4.     Select a Date and Time prior to first sign of the issue you’ve described.

5.     If the date you need is in excess of (5) days click the Show restore points older than 5 days check box.

 

Important Notice: That System Restore affects Windows system files, programs, and registry settings. It can also make changes to scripts, batch files, and other types of executable files on your computer. Thus, any recently installed programs, updates, or changes to configuration will be lost if you use System Restore. However, make note that System Restore does not affect personal files, such as e-mail, documents, or photos, so it cannot help you restore a deleted file.

Before you try a System Restore, lets try a Selective Startup (Clean Boot) to make sure a Service or Startup Program is not causing this issue:

How-to run Windows in Selective Startup (Clean Boot)…

1.     Click Start, click Control Panel, click System and Maintenance, click Administrative Tools, and then double-click System Configuration.

2.     Click Continue, or provide Administrator credentials if prompted.

3.     Click the General tab, and click Selective startup.

4.     Under Selective startup uncheck Load startup items.

5.     Click the Services tab, check the Hide all Microsoft services box, and then click Disable all.

6.     Click the Apply, then OK, and then Restart.

If it works just fine in the Clean Boot procedure listed above, then a Service or Startup program is causing this issue.  Start de-selecting the Services (one at a time) using the steps listed above until you find the Service that is causing the issue, the same goes for the Startup Programs.  Once you are completed doing all of that, to put the computer back into Normal Startup, follow the steps listed above for the Clean Boot and this time Click on Normal Startup click apply then click ok.

rajithr say:

Hi Jordan,

 

Error Code 18: Reinstall the drivers for this device.

 

Recommended resolution for error code 18: Updating the driver.  If that does not work, reinstall the driver for this device through Device Manager.

I would suggest you to follow these methods and check:

Method 1:

Step 1: Follow the steps to update the keyboard driver.

 

a) Move the mouse to bottom left corner and right-click on the ‘Start icon’.

b) Choose ‘Device manager’.

c) Expand ‘Keyboard and right-click on ‘keyboard device and select ‘Update.

d) Close the Device Manager and restart the tablet and check.

 

Step 2: Follow these steps to uninstall and reinstall keyboard driver.
a) Move the mouse to bottom left corner and right-click on the ‘Start icon’.

b) Choose ‘Device manager’.

c) Expand ‘Keyboard and right-click on ‘keyboard device and select ‘Uninstall’.

d) Now, restart the tablet, the driver for the keyboard should install automatically.

 

The Samsung website shows that there are no downloads available currently for this device, so I would suggest you to check with Samsung support.

http://www.samsung.com/us/support/owners/product/XE500T1C-A01US

http://www.samsung.com/us/support/contact

 

Follow the steps and let us know if that helps.  If the issue persists, reply and we will be happy to help you.
