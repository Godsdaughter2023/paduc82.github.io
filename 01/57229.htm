In most cases, you can install the new motherboard, connect the hard disk and Windows 10 Pro will automatically reconfigure itself. It doesn't work 100% of the time, but in most cases it does. Please note;

It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

Determining filename lengths - Microsoft Community

philip54 say:

Determining filename lengths

Is it possible in Windows 7 to determine the length of the filenames in a older and its subfolders?

 

 I tried some freeware programs that are supposed to do this, but with no much luck.

 

Thanks

the larch say:

Sounds like simple enough job for someone who knows how to write scripts that run in Windows, in something like Visual Basic.  Until that person comes to help here, I'll mention that a method that I have found is to go to the desired folder, copy the filepaths of the items there, paste into an Excel sheet, and use formulas to process the results.

 

More detail:  Upon opening the folder (or upon doing a search in that folder), click Organize, click Select All, press and hold Shift key and rightclick on any filename, click Copy as path, go to an Excel sheet and paste into cell A2 to see a column fill up with file paths.  Because we want to count the characters of the filenames and not the whole paths, we have to do some tricky Excel formula making.

 

I had made column labels for cells A1 to E1:  entirepath filename extension pathlength namelength.  The cells B2 to E2 contain these formulas:

 

B2

=IF(ISERROR(FIND("\",A2)),A2,RIGHT(A2,LEN(A2) - FIND("|", SUBSTITUTE(A2,"\","|",LEN(A2) - LEN(SUBSTITUTE(A2,"\",""))))))

 

C2

=IF(ISERROR(FIND(".",B2)),"",RIGHT(B2,LEN(B2) - FIND("|", SUBSTITUTE(B2,".","|",LEN(B2) - LEN(SUBSTITUTE(B2,".",""))))))

 

D2

=LEN(A2)

 

E2

=LEN(B2)

 

Note that the namelength will include the file extension characters, or at least it does for me -- I always keep the setting to display the file extension in Windows.

 

To have all of the rows show information based on the formulas, select cells B2 to E2, copy, select cells of the lower rows from C2 to the last row that has a filepath showing, and paste.
