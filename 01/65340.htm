Here is what I do:

To have the cmd.exe (Command Prompt) open to the folder of your choice such as:

c:\windows\system32

Before making any changes to your registry by hand or with third party tools, be sure to make a backup of the registry first.

There is no undo or quit without saving option in regedit.

Here is a link to a popular registry backup tool:

http://www.snapfiles.com/get/erunt.html

Click OK to open the Registry Editor.

Navigate to this location:
 

HKEY_CURRENT_USER\Software\Microsoft\Command Processor

If there is not a REG_SZ (String Value) variable called Autorun, you need to create one by right clicking an open area on the right pane, choose New, String Value and name the new entry:

Autorun

Double click the Autorun variable to set the value, for example:

If you want the Command Prompt to open in the C:\ folder, set the value of Autorun to

cd  /d  c:\

If you want the Command Prompt to open in the C:\windows\system32 folder, set the value of Autorun to

cd  /d  c:\windows\system32

Close the registry editor, check the results and reboot to be sure the changes "stick"

Windows 10 has the capability to install the default driver for a peripheral that is connected to your computer. Open the Device Manager, right-click the name of the device (digital scanner) and then select Uninstall. Restart your computer and Windows will attempt to reinstall the generic driver for the scanner.
