khronos1 say: I've finally managed to fix the problem after years of being unable to use Windows Backup.

My PC is currently in a dual boot configuration with Ubuntu 11.10 and using Grub as the boot loader.

Yesterday I was trying to fix a problem where my PC was unable to go into Sleep mode (S3 Power State). After a lot of searching I finally managed to find a solution which was to ensure that my first Windows partition has the boot flag set. At that time, the partition where Windows was installed carried the boot flag. However Windows had also created another 100MB partition called 'System Reserved' which preceded the partition where Windows was installed. So I simply booted into Linux, opened GParted and set the 100MB 'System Reserved' partition to carry the boot flag. After doing so my PC was immediately able to go into Sleep mode properly and the Windows Backup issue was also immediately fixed.

However this fix only has the possibility of working if you are in a dual boot configuration so it probably won't apply to the average Windows user. However this very simple fix did resolve 2 problems for me that I've been having for a very long time.
