a_user_here say:

First off the versions of Windows that have N after them are versions that don't contain things like Windows Media player.  The version was created to satisfy the EU's lawsuit to "decouple" certain parts of Windows to allow other companies to compete/supply these.  If you use these versions you would have to download  the programs from Microsoft or a separate vender.  Seems ridiculous to me since even with the non N versions you can choose to use a third party program, it is just the default that would be Microsoft, but that is what it is.

RTM stands for Release To Manufacturing.  As in is the version released to the general public.

And that version is build 10240.  The same one you should have as an insider if you have been keeping up with the build installs.  You can click on the Start button and type: winver {Enter} and it will show you the version.  If you are on 10240 that install should be good to go.

On the product Ids.  Depending on how you created that Windows 10 insider VM you may have 1 or 2 of them.

If you copied your Windows 8.1 VM and then upgraded it to Windows 10 with the insider builds.  You should have the same product Id  in both of them (at least that is how I understand it).  Note is also possible that they tie the Windows 8.1 product Id to a new Windows 10, one so they might be different, but you can't have two machines that were upgraded from the Windows 8.1 product Id.  So if that is true clearly you should just continue using your Windows 10 VM and stop using the Windows 8.1 VM.  But if you actually created a Windows 10 VM without updating you should have two product Ids.  It was asked and answered by Microsoft what would happen if a person never had Windows before and joined the insider program, and installed from scratch, would they get a free copy?  The answer was yes.  So even though there have been remarks by some people in this forum that you couldn't do that, what Microsoft has said seems to be contrary to that and someone in your position might in fact now have two valid product Ids.  And so the Windows 8.1 Pro VM should be upgradable too.

As for the 8.1 Pro update, the edition you would and want of course be Windows 10 Pro, and if you like you could have it update by running the MediaCreationTool on the Windows 8.1 Pro VM.  Or you could download separately and have it update it from USB/ISO/DVD you created with the MediaCreationTool.

To see the product Id in Windows 10 open the Settings -> System -> About

To get the product Id for the Windows 8.1 Pro, look on the install media packaging.

I had the same problem. The touch keyboard would not work any more. I had not changed anything other then updating windows 10. I found the solution to the problem Here Windows 10 touch keyboard not popping up? Fix it!

"How to make the Windows 10 touch keyboard pop up automatically:
  • Press WinKey + R to pop up the “Run” menu and enter services.msc

  • Hit OK and you will reach the Services panel. There in the list, find Touch Keyboard and Handwriting Panel Service.
  • Right click and select Properties. In the General tab, change the Startup Type to Automatic or Automatic (Delayed Start). Either of those will do.
  • Now check if the keyboard pops up when you try to enter text.
  • If it does not, restart your computer and see if it works.
  • If it still does not, again go to the Touch Keyboard and Handwriting Panel Service and open properties. This time, open the Dependencies tab and you will find a list of processes quoted as “The service depends on the following components”.
  • Now close the properties window and check the list of Services and make sure that the services mentioned above are either “Running” or in “Automatic” mode. If they are disabled, you should enable them.
  • Now again find the Touch Keyboard service and enable it as Automatic.
  • See if this works for you!"

I hope this helps.
