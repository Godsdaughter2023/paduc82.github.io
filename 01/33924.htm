Suggestion 1:

Run the Windows Update troubleshooter

  • Press the Windows key + C to access the Charms bar.
  • Select Settings
  • Select Control Panel
  • Select Troubleshooting. Under the System and Security heading, select Fix problems with Windows Update.

Suggestion 2:

Perform a clean boot and try Windows updates - http://support2.microsoft.com/kb/929135

  • Press the ‘Windows + R’ key on the keyboard.
  • In the ‘Run’ window type ’MSCONFIG’ and click ‘Ok’.
  • Click the ‘Boot’ tab and uncheck ‘Safe Boot’ option.
  • On the General tab, click to select the option ‘Selective startup’, and then click to clear the option Load startup items check box.
  • On the ‘Services tab’, click to select the ‘Hide all Microsoft services’ check box, and then click Disable all.
  • On the Startup tab click the link to Open Task Manager.
  • Select each application and click on the Disable button until all of the applications are disabled.
  • Close Task Manager
  • On the System Configuration, click OK, and then click Restart.

Note: After performing the troubleshooting steps in clean boot, follow these steps to return the computer to Normal startup mode.

  • Press the ‘Windows + R’ key on the keyboard.
  • In the ‘Run’ windows type ’MSCONFIG’ and click ‘Ok’.
  • On the General tab, click the Normal Startup option
  • On the Startup tab, select each application and enable each one until all of the applications are enabled.
  • Close Task Manager and click Apply and OK on the System Configuration window.
  • When you are prompted to restart the computer, click Restart.

Method 1
For DNS server follow the below steps and check for any errors listed in event viewer–

a.      Click Start, Type cmd in the Start Search box, and then press ENTER. 

b.      At the command prompt, type the following command, and then press ENTER:
ipconfig /flushdns (It will show the message successfully flushed the DNS resolver cache)

c.      At the command prompt, type the following command, and then press ENTER:
ipconfig /registerdns (there is a space between ipconfig and / )

d.      At the command prompt, type the following command, and then press ENTER:

ipconfig /all (there is a space between ipconfig and /)
The ipconfig /all command displays Windows TCP/IP settings for all your network adapters.

If your IP address for your Local Area Connection is either 0.0.0.0 or 169.x.x.x (where x is any number), then your computer is not getting an IP address from the router. 
If that is the case, try this:

·         ipconfig /release in the command prompt and press ENTER

·         ipconfig /renew in the command prompt and press ENTER

Test it again by typing ipconfig /all to see what the address is?  (If you are on a home router, it should start with either 192.168.1.x or 192.168.0.x or 10.x.x.x)

 

Method 2
If the issue persists you may follow the steps.

a. Click on Start and then Control Panel.
b. Go to Networking and sharing center and then click on Change adapter settings.
c. Right click on Local Area Connection and choose properties.
d. Highlight Internet Protocol Version 4 and click on Properties.
e. Choose Obtain DNS servers address automatically and press Ok.
f. Choose Obtain IP address automatically.
g. Repeat the steps for Internet Protocol version 6 as well.
