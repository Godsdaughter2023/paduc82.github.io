waseemulla shariff say: Hi,
 
The licensing protocol handles the communication of licensing information between your computer and the remote computer. If there is an error in the licensing protocol, your computer cannot be connected to the remote computer.
 
To fix this issue, restart the computer and try connecting to the remote computer again. If you continue to receive this error message, contact the server administrator.
 
This also happens if one of the related registry entries is corrupt. Try these steps to fix this.
1. Click Start, type regedit and hit enter.
2. On registry editor window, navigate to Computer\HKEY_LOCAL_MACHINE\Software\Microsoft\MSLicensing
3. Right click on MSLicensing, click Export and give a name to file e.g. RemoteKeyBackup and click Save it on the Desktop.
4. Right click on HardwareID and Store keys folder under MSLicensing and click Delete.
5. Restart the computer for the changes to take effect. Restarting the computer will create new keys.
 
NOTE: If the issue still persists, then you can take backup of the MSLicensing registry key and load it on the problem computer, this will fix the issue.

http://technet.microsoft.com/en-us/library/cc756826(WS.10).aspx
