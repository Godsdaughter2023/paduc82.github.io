canadian tech say:

Chances are, your laptop has TWO pieces of memory.  Chances are if there is a memory failure, it is just one of them.  

  1. Shut down
  2. remove power
  3. Remove battery
  4. Hold the on button down for 7 seconds
  5. Open the PC
  6. Remove one of the memory pieces
  7. Reverse the procedure and test

If the problem is repeated, the one  you removed is the good one.  If not, go back and repeat the procedure but remove the other piece of memory and install the one you removed before

Reverse the procedure closing it up again.  If the problem is gone, you know which one has failed.  If the problem repeats, either both pieces of memory are defective (which would be very unusual) or the problem is not memory hardware.

When you replace memory, it must match the one that remains.

ronnie vernon say:

Hi

 

Yes, you can do this.

 

Open the folder where the .MP3 files are stored.

 

In Details View, right click the Name header at the top of the Name column and select More.

 

Scroll down to the Length item and place a check mark in the box.

 

Click OK.

 

This will add the Length Column.
