bossdweeb say:

I assume you are trying to reinstall DX because you have been shown an error message from a game.

If so, the title of the game, and the brand and model graphics processor your machine is using may be helpful

in determining the problem with the game.

- Win10 has DX9,10 & 11 already, but if your Win10 is 'missing' any files the

easy option. This will install any missing files directly.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

- From your post I assume you have been trying to reinstall DX with the DirectX Redist (June 2010),

and not the older legacy DX9.0c.

This installer has all available DX9,10 & 11 files, and overwrites all DX files, so this is what you would want if

there were corrupt DX files that need fixing/replacing.

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

The DX installer package is a two stage process. When you first run the downloaded directx_June2010_redist.exe

it extracts files to a location of your choice.

I generally make a new folder on the desktop because this folder gets deleted after...

Step 2... Open the folder the files have been extracted to and run theDXSETUP.exe to finalise reinstalling DX.

- If you have already extracted the files to somewhere in C:\ you can still run the DXSETUP.exe from that location.

Hopefully you haven't extracted the files to the base of the C:\Program Files folder as they will be more difficult

to find and delete.

(If you have problems you can extract the files to a new folder and compare with what you see in C:\Program Files.

- Note that DirectX files are actually installed to the System32 and sysWOW64 folders. The files below are what

you will find if you look in System32.

They are in alphabetical order and will start with d3dx9 - 24 > 43 . Then d3dx10 - 33 > 43 & finally d3dx11 - 42 > 43.
There should also be - d3d9, d3d10 and d3d11 (among others).
These come before the D3Dcompilers-33>43+47.
- Win10 currently has only two d3d 12 files. One in System 32,and SysWOW64 .
- There are more DX files, such as input and audio .dll's , as well, but that covers the majority of them.
