Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

The error message that you received when you played the game means that the graphics card driver needs an update. To resolve your concern, we suggest that you update the graphics card driver by following the steps below:

  1. Select the Start button, type Device Manager, and select it from the list of results. 
  2. Expand one of the categories to find the name of your display adapter, then right-click (or tap and hold) it, and select Update Driver. For graphics cards, expand the Display adapters category, right-click your graphics card and select Update Driver.
