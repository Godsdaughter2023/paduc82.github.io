a. user say:

Windows 10 actually runs a little faster on my 8-year old computer than Windows 8.1 and 8. I don't remember any longer how fast it was when it was running Windows 7. With Windows 10 I can boot to a working desktop in 35 seconds, which is good for any computer but especially for one that recently celebrated its eighth birthday.

So Windows 10 is not your problem. And hardware is not your problem. That leaves two culprits: configuration settings and software.

Windows 10 - all versions of Windows, actually - are installed by default with every bell and whistle turned on. This shows off the operating system's features but also uses a lot of system resources. One way I've kept my computer running quickly over the years is by turning off the eye candy and other parts of Windows I don't use.

Since you got a new computer you also got all the bloatware that OEMs love to add in. This too grabs your system resources like one of those hoarders on television. Get rid of the bloatware.

Another factor that slows a new computer is all the software applications that start together with Windows and run in the background. Software developers also offer their products with all the components turned  on, and this creates something akin to Ross Perot's 'giant sucking sound' on your system resources. Go through your applications and turn off all the options you don't want, which is most of them.

Once all the clutter is gone, you'll see much faster performance. These are the things you need to do, not useless things like manually defragmenting, which offers very little benefit to modern hard drives (and is downright harmful to SSDs.). Another advantage to de-cluttering is that you'll have a leaner configuration so there are fewer things to go wrong or get in the way.

In fact, if you're not seeing a noticeable improvement, you're doing something wrong.
