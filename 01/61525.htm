bossdweeb say: [EDIT: Didn't notice Mike had responded already while I was writting, but to elaborate....]

Basically anything that loads when Windows launches, particularly  Startup programs, but also
large files such as photos, or a very high resolution wallpaper that you may have on the desktop
will slow Windows boot time to  some degree.

The number of Startup programs can also slow Windows performance, so it is generally advised
to limit the number of Startup programs to those that you really need to have running when
Windows has loaded.
Similarly if you have setup Windows to re-open the files you are working on when you shut
Windows down this will impact on performance to some degree.

It comes down to how much system RAM and cpu cycles is being used by whatever processes
you have running at any one time.
You can monitor how much various processes use your system resources with
Task Manager\ Resource Monitor, but programs such as  
Process Monitor   are more detailed.
Autoruns for Windows   is also a good tool for detailed information of Starup programs .,
 
The refresh rate of your monitor is more to do with the monitor itself.
A monitor with a 120Hz refresh rate does not stress the machines resources any more
than a monitor that refreshes @ 60Hz.
However, a monitor with higher resolution will use more graphics processor resources,
particularly the VRAM .
- A U3011 with it's   2560 x 1600  resolution would certainly use more VRAM than a
monitor with 1920x1080 resolution when dealing with open Windows and photos.
When running 3D applications such as games a resolution would  add far more
load to the gpu as well.
.

Ninety processes is not unusual and many are doing relatively little anyway. Relatively little does NOT mean unimportant, by the way..

bossdweeb say: Yes, you have to be especially careful about moving folders with data that is linked to a software
as the  'path' to that folder is set by the software in question when it installs.
So when that software goes looking for files in that location and can't find them - trouble.

As an example - Make a short-cut to a folder and then move that folder to another partition,
or put it in another folder. Or, make a short-cut to a file and move that file to another folder.
The result is a  borked short-cut. 
In all these examples you can change the path in the shortcut to suit the new location, and it will
work again (or, obviously,  make a new short-cut from the new location),
but it's not always so easy to make a software look elsewhere (registry edit).

andre da costa say:

Your best option would be to purchase a retail upgrade or full version Windows 7 Professional license, then use the product key from the package with the Anytime Upgrade wizard in Windows 7 Home Premium.

If the license for Windows 7 Professional you have access to is either a retail upgrade or full version, you can use the product key and Anytime Upgrade to move from Windows 7 Home Premium to Windows 7 Professional. First, make sure the edition of Windows 7 you are upgrading from is already activated (if its not you will run into complications and you might have start all over by doing a clean install).

Click Start, type Anytime Upgrade, click the option to enter a key, enter the Windows 7 Professional key when requested, click Next, wait while the key is verified, accept the license agreement, click upgrade, wait while the software upgrades, (it might take 10 minutes or more depending on if updates are needed), your computer will automatically restart, after restarting, Windows 7 will notify you its updating the computer, the computer will restart one more time automatically and the upgrade will be completed, a windows will pop up notifying the upgrade was successful and your computer is ready to use, click close, you should be upgraded to Windows 7 Professional your files, programs and settings preserved.

Please note, you cannot do this with OEM System Builder product keys, it must be a retail box license.
