andre da costa say:

First of all, is the correct edition of Windows 10 installed?

To find out if you have the right edition installed:

Click Start > Settings > Update & security > Activation > Edition.

Windows 10 came preinstalled on my computer, how do I find the product key?

Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

The product key will be revealed, copy the product key then enter it

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support

Default port values for commonly used TCP/IP services have values lower than 255 and Well Known Ports have numbers that range from 0 to 1023. Registered Ports range from 1024 to 49151 and Dynamic/Private Ports range from 49152 to 65535. wunders say: It almost sounds like he is using Remote Desktop, but XP Home does not allow that.

Since you have XP Home, Simple File Sharing is enforced and remote users can only authenticate as the "Guest" user.  You say that you have the Guest user "Turned Off".  How did you do that?  If you did it from the User Control Panel, that does not disable network access -- only for interactive access.  To truly de-activate the guest user and disallow network access, you should bring up a command prompt window (Start ->Run -> "cmd") and enter the command:
    net user Guest  /active:no

Another approach would be to put a password on the Guest user:
"Setting up password for the Guest user in Windows XP"
  < http://support.microsoft.com/kb/555520 >

You should also make sure that the built-in "administrator" user has been given a password as well, or he can log in and undo all you have done.  Also check the "Administrators" group and make sure that he does not have access to any of the accounts listed there.  To list members of the Administrators group, bring up a command prompt window as above and enter the command:
   net  localgroup  administrators
which will list all users of the administrators group.

To see who is connected to your computer, right-click on "My Computer" -> manage -> User Sessions

A user profile of "Account Unknown" just means that a user that used to have a profile was removed from the machine.

Regarding FAT to NTFS:
  "How to convert a FAT16 volume or a FAT32 volume to an NTFS file system in Windows XP"
  < http://support.microsoft.com/kb/307881 >
