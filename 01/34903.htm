Using Classic Shell To Restore Some Features To Windows Explorer

After installing the free Classic Shell you can customize Windows Explorer closer to how you want it, the quote below is just one way to do so. Click here to see some of the options you can customize, and here is the FAQ page for Classic Shell.

A solution which did work for restoring some features.

1) Install the free Classic Shell.  This is very customizable, and works as a mod for Windows Explorer rather than a full blown replacement. You didn't need to install the Classic Start Menu option, but you can if you want.
2) Reboot the computer.
3) Launch Windows Explorer (Start - Computer) and turn on "Always Show Menus" (Organize - Layout - check Menu Bar). This is just for configuration, you can turn it off again afterwards if you want.
4) Right click on the menu bar and turn on the "Classic Explorer Bar". This adds a series of cut/copy/paste icons to Explorer. Crucially, it also adds the "Configuration" icon.
5) Go into the configuration and from here you can tweak things to your heart's content. To kill the horrible Windows 7 folder-jumping, switch the "Navigation Pane Style" to Vista. You can even knock it back to XP mode if you want (this adds in the lines to the folder tree and adds a classic "+" icon for expanding folders). The Windows Vista style folder view looks pretty much the same as the Windows 7 view, but gets rid of the horrible jumping.
6) Tweak any other settings you want.

After this, you can turn the toolbar and menus back off in Explorer if you wish. Windows Explorer looks largely the same, but folders no longer jump around. Vote for Microsoft to fix this bug.

davem121 say:
Hi RGP1942

When you moved your library folders, did you move them to a folder on your E drive or are the files now all over your E Drive?


If you moved each library to a folder on your E Drive and the expected files are in those folders:

In the left pane, click Quick Access
Then in the right pane at the top, right click each library folder
Choose Properties.
On the location Tab, click Restore Default, then apply and OK
If asked do you want to merge content, choose No



If your files are all over your E drive:

Create a new folder on the E drive for each library folder you moved
Put whatever files and folder should be each library into each of these new folders
In the left pane, click Quick Access
Then in the right pane at the top, right click each library folder
Choose Properties.
On the location Tab, click Move
Navigate to the appropriate new folder you just created
If asked do you want to merge content, choose No
Click Apply and OK
Do this for each of the library folders you moved

When all done, move the library folders back onto your C Drive
In the left pane, click Quick Access
Then in the right pane at the top, right click each library folder
Choose Properties.
On the location Tab, click Restore Default, then apply and OK
If asked do you want to merge content, choose No

a. user say:

Hi Dennis,

What is the make and model of the computer?

Windows 8/8.1 includes an automatically loaded updated version of Windows Defender which is a full antivirus based on Microsoft Security Essentials. Windows Defender is disabled automatically when you install a 3rd party antivirus.

After you install other antivirus application on Windows 8.1, you see "The Windows Security Center Service could not stop Windows Defender" written to the Application event log every five minutes.

This message is informational only, and you can safely disregard it.

Refer to the link for more information:

Using Windows Defender

http://windows.microsoft.com/en-us/windows/using-defender#1TC=windows-8

The sleep issue might occur due to incorrect settings in the power options. Make sure that you have the most current drivers for all your devices. A hardware device might be keeping your computer awake. USB flash drives and USB mice are common devices that can keep your computer awake. By default, Power Option settings do not allow scheduled tasks to wake the computer from sleep or hibernate.

If the sleep mode is not working as expected, you may follow these steps to resolve the issue:

Method 1:

Run the Power troubleshooter.

Follow these steps:

a. Press Windows Key + W, type troubleshooting in the search bar, and press enter.
b. Click on system and security and select the power troubleshooter.
c. Follow the steps from the wizard to run the troubleshooter.

Method 2:

If the issue persists, then try to restore the power plan settings to default and check.

Follow these steps:

a. Press Windows key + W and type Power.

b. Select Power Options.

c. Check if the preferred plan is Balanced.

d. Click on Change when the computer sleeps and select Restore Default settings for this plan.

Hope this answers your query. You can write back to us for further queries/issues related to windows and we will be glad to assist you further.
