    After you have UPGRADED from Windows 7 to Windows 10, you can do a clean install of Windows 10 if you want.  Start the computer with the media and when you come to the product key screen make NO entry and skip.

    Once a computer is a registered Windows 10 Device the user can install and re-install Windows 10 as often as the user wants and no product key is needed.  

    This business about "Registered Windows 10 Device" is a completely new licensing technology.  The product key that comes with your free copy is only a placeholder and everybody who participates in the Get Windows 10 program has the same product key in his system.  That's why no product keys are distributed with the free offer.

    donna men say:

    Hi BuzzBumble,

    We received multiple reports regarding this issue after updating to April 2018 Update (1803). To resolve this, simply follow these steps:

      1. Simultaneously press Windows key + R.

      2. On the command prompt, type services.msc and then press Enter.

      3. Find each of the services in the list below, and on each service, right-click, select Properties, and set Startup type to Automatic (Delayed start), then click Apply and OK.
           a. Computer Browser (Browser)
           b. Function Discovery Provider Host (FDPHost)
           c. Function Discovery Resource Publication (FDResPub)
           d. Network Connections (NetMan)
           e. UPnP Device Host (UPnPHost)
           f. Peer Name Resolution Protocol (PNRPSvc)
           g. Peer Networking Grouping (P2PSvc)
           h. Peer Networking Identity Manager (P2PIMSvc)

      4. Reboot the device and check the Network tab again.

    Hope it helps.
