frederik long say:

Steps to perform Startup Repair from the Windows Recovery Environment (WinRE):
1. Set your BIOS to use the DVD drive as the primary boot device.
2. Boot with a Windows Repair CD.
3. Press a key when the message indicating "Press any key to boot from CD or DVD …" appears.
4. Select a language, a time and currency, and a keyboard or input method, and then click Next.
5. Select an Operating System, then click Next. You are now in Repair Mode.
6. In the System Recovery Options dialog box, choose the drive of your Windows installation and click Next.
7. In the System Recovery Options dialog box, click Startup Repair.

You may have to go through this process twice.

Instead using a Windows Repair CD in Step 2, you can use your Windows 7 Installation DVD. The steps are similar to the ones above.

frederik long say:

Partitions can be deleted without affecting the data that is stored on them. There are various partition undelete tools that can restore deleted partitions. Use Google to see what's around or start a new thread under a subject line such as "Partition undelete utility".

You cannot format an unpartitioned disk. The process goes like so:
- Create one or several partitions on the disk.
- Format each partition.
- Each formated partition is now visible as a drive such as drive C: or drive E:.
Note the distinction between "disk" (the physical thing that you can touch) and "drive" (the formatted partition). Laptops have room for one disk but they can have several formatted drives.

You might have to treat this unfortunate incident as a harsh reminder that important / irreplaceable files must be backed up regularly (e.g. weekly) to an independent storage medium.
