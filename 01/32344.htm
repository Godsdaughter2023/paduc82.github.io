lemp say:

What happens when you boot from the system repair disc?  If necessary, you may have to go into your BIOS configuration utility and make your optical drive the first boot device.

The system image you created includes everything:  your operating system files, your program application file, and your data.  Assuming that you've already been using your system for some time and thus Windows, your installed programs, and your user files total more than 128 GB, the image won't fit on the SSD.

If everything but your user files totals less than 128 GB, then one possible scenario would be to back up all of your user data files to some external medium, confirm that the backup is valid, and then delete those files from the current hard drive.  Then make a system image of what's left (which is now less than 128 GB) and restore that image to the SSD.  Then you would create a new user profile, copy your backed-up data files to the 1 TB drive, and link your user profile to the user data (see tip below).

Another alternative, especially if you have more than 128 GB used even without your data, is to do a clean install of Windows 7 on the SSD, reinstall all of your application files, specifying that as much as possible of the installations be located on the 1 TB drive, and link your profile to the user data on the 1 TB drive. 

This tip from sevenforums.com may help with the issue of having Windows on the SSD and your data on the spinning drive:

If you are just wanting to save HDD space from the files in your user folders, then you might consider this below to be able to access the files from your libraries with the files actually located where you like instead. This way you will not have to worry about any potential issues that come with moving user folder locations.

  • Create a new folder at any location you want.
  • Move any files you want into the new folder.
  • Include the new folder into a library of your choice.
  • If you like, you can also set the new folder to be the default "save" to location for the library as well

To do a clean install of Windows 7 on your SSD, follow the directions here to obtain a Windows 7 ISO file for the same version of Windows 7 that came pre-installed on your computer:  https://www.heidoc.net/joomla/technology-science/microsoft/67-microsoft-windows-and-office-iso-download-tool  Burn the image to a DVD (Note: you can't just copy the files; use the Windows 7 built-in burn-to-dvd tool or free third party tool like ImgBurn).  Use the product key from the label on your computer (not the key retrieved using a key finder program) to activate Windows.  See https://techingiteasy.wordpress.com/2012/04/13/how-to-activate-windows-7-oem-license-using-a-retail-disc/

If all of the above seems complicated -- it is, and I have not provided all the gory details -- you may want to consider getting help from a local independent computer repair shop (not a chain store).  Just make sure that you get a genuine Windows 7 installation.
