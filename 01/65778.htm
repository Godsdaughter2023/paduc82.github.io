Usually, this issue is caused when the Windows Time service has gone corrupt.

I would suggest you to follow the methods given below and check.

Note: You must be signed in as an administrator to be able to change the date and time.

Method 1:

Check the status of Windows time service.

a. In the desktop window, Press windows key + R

b. Type services.msc in run window.

c. From the services window find 'windows time' and double click on it.

d. If service status is stopped, click on start and press ok.

Method 2:

Check if the date and the timezone is set correctly.

Try restarting the windows time service and check:

a. Single click on time and date display at the right bottom of screen.

b. Select internet time tab and click change settings and click update now and press ok.

Method 3:

Check if the correct time is set in the BIOS from the device manual.

If you are not comfortable accessing the BIOS, please contact the device manufacturer.
