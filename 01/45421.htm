lemp say:
What version of Windows do you have, including service pack?  I'll assume that you have some variety of Windows 7, given that you posted in a Win 7 forum.

I assume that you want to print directly (wirelessly) to the printer rather than using the ePrint feature.

Do you ever turn the printer off or do you leave it powered on all of the time?

I suspect that what is happening is that your printer is getting a new IP address assigned by your router.  This can easily happen if you turn the printer off and also have a number of other network devices (e.g., your laptops) that are not always present on the network..  Both the diagnosis and solution to this problem are relatively easy (it just takes a lot of words to describe them)

Diagnosis
Print a network configuration page using the printer's front panel controls:

1. Press the Wireless button to display the Wireless Menu.
2. From the Wireless Menu, select Print Reports.
3. From the Print Reports menu select Configuration

Make a note of the printer's IP address, subnet mask, and Default Gateway shown on the network configuration page

On a computer on which you had installed the printer, go to Devices and Printers
Right click on the icon for the HP 3052A and select Printer Properties (NOT the same as "Printer Preferences" or "Properties")
Select the "Ports" tab
Being careful to NOT click inside the list of ports, use the slider on the right side of the list so that you can see the highlighted and checked line that indicates the port in use for this printer.
If necessary, click and drag on the faint vertical line in the header between "Port" and "Description" so that you can see the complete port identification.
Make a note of the port.
For now, click Cancel to exit the printer properties.

If the port shown on the network configuration page and the port shown in the printer's properties dialog are different, that's your problem.

Solution (assuming the wrong port is the problem)

Printers and other similar devices connected to a network should always be assigned a static (fixed) IP address so that the users of those devices (e.g., computers connected to the network) can find them.

Open a browser on a computer connected to your network and enter the printer's IP address that was shown on the network configuration page.
You should be connected to the printer's web-based configuration utility.  Follow the directions at this HP support page to set a static IP address for the printer.

You can see the range of IP addresses used by your DLink DGL4500 by entering its web-based configuration utility (in your browser, go to 192.168.0.1) and look at the "Basic" network tab (see p. 30 of the manual).  It seems that the router's default is to assign addresses from the range 192.168.0.100 through 192.168.0.199.  Use an address outside that range for your printer, e.g., 192.168.0.25 or 192.168.0.250.  It is ESSENTIAL that the first 3 "octets" of the IP address remain the same (i.e., 192.168.0).  The only part of the IP address that you should change is the last "octet."

You will need to do the steps in the following paragraph on EACH computer that you want to use to print to this printer.

In some printer software, HP seems to have included a utility named "Update IP Address" (see the HP directions linked above).  I'm not familiar with that utility, but from its name, if you have it and run it, you should then be able to print.  If you do not have the utility, you'll have to manually update the IP address that Windows uses for your printer.  You do that by going to the "Ports" tab as described above and then clicking the "Configure Port" button.  On the next screen, enter the new static IP address that you assigned to the printer and OK your way out.
