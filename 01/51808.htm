considerate_guy say:

Windows Movie Maker 2.6 is old and based on the Windows XP version of Windows Movie Maker 2.1. It is very tempermental when it comes to complicated projects and/or unusual file types.

Given that the project (.mswmm) gets to 20% during saving says you should be looking about 1/5 of the way into your timeline to figure out what is different there. Check file types that might be different or where you might have applied a number of transitions/effects.

One thing you might try is to make multiple copies of your project file (.mswmm). Working in a copy (not the original project file), cut out the last 4/5 of your project and see if you can save the first 20% as a movie. If that works, take the second copy and delete the first 20% that you've already saved as a movie and see if the remaining 80% will save as a movie. If it will, great. If it won't try trimming down from the end until you can get a portion to save as a movie. Repeat this process until you can get the individual parts saved as individual movies.

Assuming that all works, start a new project, import the individual movies (.wmv) and create one final movie from joining them together.
