russell m smith say: Probably the game doesn't really need administrative privileges to run. Either it's tripping UAC auto-elevation accidentally or the 'requested execution level' in the application manifest is configured to require administrative rights (requireAdministrator). Either way, you can modify the game's executable file and force it to launch asInvoker, i.e. as the user that launches it. It should then run as a standard user without a UAC prompt for administrative credentials. Be aware that the game may launch more than one .exe file on startup.

You'll need Resource Tuner (you can download a free 30-day trial), and follow the instructions given to either add the necessary information to the application manifest, or modify the existing requestedExecutionLevel. Instead of specifying requireAdministrator as in the instructions, set the requestExecutionLevel to asInvoker.
