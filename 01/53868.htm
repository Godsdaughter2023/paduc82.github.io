rajesh govind say:

Hi Yiannis,

Usually, you get this error code: “0xC004C008”, when you install a copy of Windows Operating System on multiple PCs.

The product keys for the Windows Operating System are normally single user license keys. This means that they can be installed and used only on one computer. If your product key is being used on more PCs than the Microsoft Software License Terms allow, you'll need to buy a product key or copy of Windows for each of your PCs in order to activate them.

When you use the Windows product key to activate a copy of Windows Operating System, your computer generates a unique identifying code based on its hardware configuration. This code provides no identifying information to Microsoft; it is merely a summary of the hardware you have installed on the computer. Windows sends this code to Microsoft’s servers along with the product key, linking the two in the database. This ties your particular license key to your PC, ensuring that any attempt to use that license on another machine will fail.

So, in this case, your product key was already assigned to a set of hardware combination when you activated Windows for the first time using this key on your first computer. But, when you connected the same hard drive on another PC, it found a mismatch of the hardware combination and when this exceeded for a number times, the activation server might have blocked your product key and that could be the reason why you are unable to use it on any of the computers. You may please refer to this Microsoft article to know on how the Windows Activation works:

Microsoft Product Activation

 You may also refer to this article: Why can't I activate Windows?

For further assistance on this issue, I would suggest you to get in touch with the Windows Activation Support Team using the article given below:

How to activate a Microsoft product by using Microsoft Product Activation Center

Hope this is helpful. If you need any further assistance with Windows Operating System, please post your query on Microsoft Community Forums and we’ll be glad to assist you with it.

The Software Fix
 

Step 1

Download and install a computer fan-control program like SpeedFan on your laptop. (See Resources.) Many of these kinds of programs can be found all over the Internet. If you have any problems installing the software, consult the documentation included with it.

Step 2

Open the program and navigate to your main CPU fan. This is usually the fan that makes the most noise. Right-click on the fan and go to the controller settings. In this menu, you can control limits for this fan.

Step 3

 

Turn down the speed limits on the CPU fan to keep it from being so loud. Keep in mind that, if you have cooling problems with your laptop, this is probably not the best idea.

The Hardware Fix

Step 1

Purchase an ultra-quiet laptop CPU fan that is compatible with your laptop.

Step 2

Open your laptop case. This is usually done by removing the battery and unscrewing the mounting screws underneath your computer. Keep in mind that all laptop computer open differently, so it's wise to consult your computer's manual before completing this step.

Step 3

Once the case is open, unscrew the mounting bolts on your current CPU fan. Do this carefully, as you do not want to strip the post holes.

Step 4

Remove the fan and unclip the power cord from it.

Step 5

Place the new CPU fan on the mounting brackets and secure it with the screws you previously removed.

Step 6

Reattach the power cord to the CPU fan . Make sure it clips in all the way.
