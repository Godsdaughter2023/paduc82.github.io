a. user say:

The only way to send your computer back in time to any date you choose is to restore an image backup that you made on that date.

By now, daily backups should be standard procedure, particularly in organizations.

But now it's not at all certain that you can obtain an official ISO for an earlier version of Windows 10. If your can, them you will be able to do a clean install of that version of Windows. But that also means that you'll have to backup each user's data individually, restore it and then reinstall all your applications, set them up as you like, and update everything.

In an organization setting, relying on rolling back is an unwise choice. The fact is, the more a computer has been used since being updated, the less likely it is to have a successful rollback, regardless of how many days have passed.
