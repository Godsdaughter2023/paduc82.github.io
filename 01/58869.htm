Export Wireless Network Profile

1) Go to Start -> All Programs -> Accessories -> then click on Command Prompt (cmd.exe).

2) Command prompt window will open, then type netsh wlan show profiles and hit Enter button to show available wireless network profiles on this computer.

3) Let’s say you want to export home-network profile to a thumb drive for example (G:\wireless-profile), In that case you have to issue the command netsh wlan export profile name="home-network" folder="g:\wireless-profile"

Home network is just an example, you have to enter your networks name in place of home network.

Note: This command usage: netsh wlan export profile name="profile-name" folder="the-folder-you-want-to-export-to"

4) Then you have to go to your thumb drive folder for example (G:wireless-profile) and see there is an exported profile. Your network profile will be created as Wireless Network connection-"profile-name".xml

Import Wireless Network Profile

1) Open the command prompt again, type netstat wlan show profile, and you will see no profile here. Then you type netsh wlan add profile filename="g:\wireless-profile\Wireless Network Connection-home-network.xml", you will then add the profile successfully on this computer. By default it will add to all user profiles, so everybody on this computer can see this new profile and use it.

Note: This command usage: netsh wlan add profile filename="location-of-the profile-you-want-to-import" user=current/all

2) If you open your Manage Wireless Networks window, you will see the new profile there.

 

 

 

Windows 7 makes it easy to export your wireless settings from one machine to another.The backup of the wireless networks settings can be easily done from Windows 7, and the settings can be saved to a USB stick, and then the settings can be imported on laptops & netbooks running Windows XP, Windows Vista and Windows 7.

How to Save Your Wireless Network Settings

1.    First, open the Network and Sharing Center. You can do this by clicking on the wireless network symbol from the taskbar and clicking on ‘Open Network and Sharing Center’. An alternative would be to go to Control Panel -> Network and Internet -> Network and Sharing Center.

 

2.    On the left side of the Network and Sharing Center window, click on the ‘Manage wireless networks’ link.

 

3.    In the Manage Wireless Networks window you will see a list with all wireless networks to which you have connected in the past. Double click on the one which interests you.

 

4.    Now you will see a window with all its properties. In order to save them on a flash drive, click on the bottom link which says ‘Copy this network profile to a USB flash drive’.

 

5.    The Copy Network Settings wizard will now start. Insert the USB flash drive, wait until it is detected and the Next button is no longer gray.

 

6.    The copying process will take only a few seconds. When done, click on Close.

 

7.    If you will look on your USB flash drive you will see a setupSNK.exe file and a SMRTNTKY folder. Make sure you don’t delete them from the memory stick.

 

How to Import Your Wireless Network Settings

1.    First, plug in the USB flash drive in the laptop where you want to import the settings and run the setupSNK.exe file. When the Wireless Network Setup Wizard launches, click on Yes.

 

2.    The settings will be imported in seconds and when receiving the notification below, click on OK and start using the wireless network.

 

3.    This procedure is very helpful for both public and hidden wireless networks. Instead of typing long authentication keys, it is easier to plug in an USB memory stick and run a setup file.

Validation Windows 10 - Microsoft Community

henrythl say:

Validation Windows 10

After my PC no longer worked I connected the hard drive in another PC with the operating system still on it.

I now get a valadation error 0x803F7001

***Post moved by the moderator to the appropriate forum category.***

cbarnhorst say:

You cannot put a hard drive with Windows 10 into a device which has never had Windows 10 (same edition) and expect Windows to remain activated.

Hard drives play no role in activation.  There are no Windows 10 activation files kept on the local hard drive.  It is all about the device now.  This is a big change from previous versions of Windows.  The activation data is now kept on Microsoft's servers, not on the computer.  This is an anti-piracy measure.  No files to hack.  Instead, Microsoft now uses the unique hardware identifiers (motherboard serial number, etc.).  The Windows license is tied to those.  When you moved the hard drive the new device's unique identifiers did not match any data on Microsoft's Windows 10 registration servers anymore.  That's why simply moving the hard drive is causing you the messages you are getting.

A device (desktop, laptop, tablet, etc.) must be a registered Windows 10 device for the edition installed on the HDD in order for Windows 10 to activate and remain activated.  Registration involves sending the unique hardware identifiers (from the motherboard) to Microsoft and once these are registered with Microsoft it is permanent.  In your case, this can be done by entering a retail product key in Settings/Update and Security/Activation.  If you don't have a retail product key you can purchase Windows 10 from the Microsoft Online Store.  Product keys are not legitimately sold separately.

henrythl say:

Thanks for your reply.

Where do I find my product key?

My Windows 10 is a legitimate upgrade fron Windows 7

cbarnhorst say:

You cannot move the free Windows 7 to Windows 10 upgrade from the old computer to the new one without purchasing a new license.

The upgrade from Windows 7 to Windows 10 did not have a unique product key.  It was activated by digital entitlement, meaning the device is recorded with Microsoft as a Windows 10 device.  It is not transferrable from one computer to a new computer.  The product key embedded in it is a dummy key that helps Microsoft identify your Windows as the free upgrade should you need support.

Only retail Windows (purchased from a store) has a unique product key that can be used to activate Windows on a new computer.  You need to purchase a new copy of Windows (same edition) from the Microsoft Store online.  You do not need to re-install Windows, just put the new product key in on the Activation page.  Sorry.

Once again, the copy of Windows does not matter.  Activation is now based on the device.  Specifically the motherboard.
