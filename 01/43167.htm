job sam varkey say:

Hi,

 

The taskhost.exe is a Task Host which is a generic Host Process for Windows 7 32-bit Services. The full path to this file should be shown in TUT asC:\Windows\System32\Taskhost.exe. TASKHOST is a generic process which acts as a host for processes that run from DLLs rather than EXEs. At startup TASKHOST checks the Services portion of the Registry to construct a list of DLL-based services that it needs to load, and then loads them. There can be many instances of TASKHOST running, as there will be one instance of TASKHOST for every DLL-based service or grouping of services.

1) Uninstall the program if it's already installed. Then when you install/reinstall it instead of double clicking on the install file right click it and select "Troubleshoot Compatibility"

2) Then click "Try Recommended Settings" and in the next window click "Start the Program" to install it.

3) After it installs see if it works correctly. Please note - some programs may not be able to run in Windows 7.

More details about how to make older programs run in this version of Windows:

http://windows.microsoft.com/en-us/windows7/Make-older-programs-run-in-this-version-of-Windows

bossdweeb say: There are a lot of threads relating to The Sims problems.  I haven't actually followed them
enough to know what the most common fixes are, but if you type The Sims 3 into the search
box  a page with all the threads will pop up.

If the game was working before it is likely there is a conflict with another software you have
installed, or an update to a software that installed around the time the problem started.

If you only have this problem when trying to launch the game off line it may be you have
installed the 'optional' Windows update - KB 2670838 .
I don't know if MS has fixed it with a more recent update, but this update will cause games not
to run when off line as it makes them try to search for an update.
Note that this update is required if you use IE10, but if you don't use IE 10 you can uninstall it.

Anti-Virus also sometimes do strange things, so see if disabling your AV temporarily helps.

To help verify if other software or a Service is conflicting with the game try using the Clean Boot
procedure. One of the Startup programs may be the culprit -
How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

Games save can also become corrupt, but generally if this happens the game will load
as far as the setup screens, and only fail when trying to load the save.
With some games the old saves don't get deleted when uninstalling/reinstalling, so
if the game loads to the setup screen try using an earlier save instead of the last one.

HOW TO: Install a Software Application using Compatibility Mode

 

If a program written for an earlier version of Windows doesn't run correctly, you can try changing the compatibility settings for the program, either manually or by using the Program Compatibility troubleshooter.


To run the Program Compatibility troubleshooter:

 

1. Open the Program Compatibility troubleshooter by clicking theStart button, and then clicking Control Panel. In the search box, typetroubleshooter, and then click Troubleshooting. UnderPrograms, click Run programs made for previous versions of Windows.

 

2. Follow the instructions in the troubleshooter.

If you cannot install a program, insert the installation disc for the program and, using the troubleshooter, browse to the program's setup file, usually called Setup.exe, Install.exe, or something similar. The troubleshooter is not designed to work on programs that have an .msi file name extension.
