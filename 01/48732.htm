Windows 10 Installation Media:

  1. Insert the Media (DVD/USB) in your PC and restart.
  2. Boot from the media.
  3. Select Repair Your Computer.

Select Troubleshoot.

  1. Choose Command Prompt from the menu:

Type in the command:

Diskpart

Type in the command:

List disk            (Note which disk is your Boot drive number mine is 0)

Type in the command:

Sel disk 0

Type in the command:

List vol               (Note which volume is the EFI partition mine is 4)

Type in the command:

Sel vol 4

Type in the command:

assign letter=V:

Type in the command:

Exit

Type in the command:

V:

After you have assigned a drive letter Using Diskpart You can format the EFI partition:

Example: if you assigned a letter V to the partition the command would be:

format V: /FS:FAT32

After the format you need to recreate the EFI directory structure with the command:

MD \EFI\Microsoft\Boot

Then change to the Boot directory with:

cd /d V:\EFI\Microsoft\Boot\

Then run:

bootrec /FixBoot

Finally run:

bcdboot c:\Windows /l en-us /s V: /f All

Type in the command:

bcdboot C:\windows /s V: /f UEFI      (This replaces the above crossed out lines and works in Win 10 1709)

Just to set proper expectations, we may go through a series of troubleshooting steps to resolve the issue. First, leave your device plugged in to the power supply included with the device, but kindly remove all external devices including USB drivers, SD cards, or even keyboard / mouse dongles. External devices can sometimes cause the device to boot incorrectly.

 

If you’re experiencing the same issue, we recommend that you perform a force shut down and restart on your Surface Pro (Newest).  To do this, you can press and hold the power button until the screen turns off (about 10 seconds), then release the power button. Then, press and release the power button to turn your Surface back on. You should see the Microsoft logo. Your device may be in a “hung” state. Force Surface to completely turn off all power and restart. Performing a Forced Shutdown is like removing the battery from a laptop or unplugging a desktop PC. It resets the hardware and clears all the cache and memory for your hardware. Observe if the issue persists.

 

If the issue persists, we suggest that you force your device to open advanced repair options. To perform this, Force the device to shut down. Turn on Surface by pressing the power button. When the Surface logo appears, force a shut down by pressing and holding the power button. After two consecutive failed attempts to start Windows, "Preparing Automatic Repair" will appear on the screen. Select Advanced repair options to open a Choose an option menu.

 

  1. Advanced repair option: Startup repair- From the choose an option menu, select Troubleshoot > Advanced options > Startup Repair. This will attempt to fix problems that keep Windows from loading. This won’t affect personal files.
  2. Advanced repair option: Reset the PC- (If it has the same issue, then we recommend that you reset your Surface.) From the Choose an option menu, select Troubleshoot > Reset the PC > Keep my files. This will remove apps and settings and reinstall Windows and won’t affect your personal files.

 

If you continue to have this issue even after performing the above steps, please reply to this thread. The system will notify us, so we can get back to you and provide you more answers.
