How to Find SSID (Network Name)

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. From the menu on the left choose Wireless Status.

 

 

How To Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 in the address bar. (Or click the link) Enter the username and password if the gateway asks. ( admin/password, admin/password1, admin/password on sticker, admin/serial number begining with CSJ).
  3. Click Wireless Settings.
  4. Verify wireless passphrase on the status page.

 

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. Select Basis Security Settings
  7. In the Turn WEP On section select off.
  8. Click Apply. (If no wireless security is desired stop here)
  9. Select Advanced Security from the menu on the left.
  10. In the Level 1 section select WPA.
  11. On the following screen set the Authentication method to Personal(Pre-Shared Key)
  12. Set the WPA Type to WPA Any.
  13. Enter the desired wireless passphrase into the WPA Shared Key space.
  14. Click the Apply button.

 

 

MAC Filtering

can't change windows display language in windows 10 home single language version

 I'am can't change windows display language English to Japanese   in  windows 10 home single language version. Please  give me idea  about it ..
sherry bel say:

Hello Denish,

In Windows 10, if you need to change the default language of your computer, you need to install the language first, then set that language as default. You can follow the steps provided to install a language in Settings app:

  1. Press Windows key + I on your keyboard to open the Settings app.
  2. Click on Time & language.
  3. Under Region and language tab, click on Add a language to open the available language you can install on your computer.
  4.  After installing a new language, click on it and you’ll see three options: Set as default, Remove and Options. Under Options, click Download to download the language pack and keyboard for that language.
  5. Then if you go back to Control Panel you can see there the language that you just installed.
  6. Go back to Setting apps to change the language of the user account. Under Region and language, select a language, then click Set as default. You will see a notification under the language that says “ Will be display language after next sign-in”.
  7. Sign out of Windows and sign in back and your new display language will be set.

Reminder: Make sure that you sign in to Windows 10 using an administrative account when installing a new language.

Let us know how it goes.
