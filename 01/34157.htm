  4. Close the window and check if the changes are effective.

 

You can also set different photo viewer applications for different picture formats by following the steps below:

 

  1. Press Windows logo key on the keyboard, type Settings and select the top most search result.

  3. Scroll down the page and click on Choose default apps by file type.
