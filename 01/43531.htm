Hi,

As far as I understand, in order to get a faster boot time in Windows 10, it doesn't completely shut down your computer, but instead, merely sends it into hibernation and then cuts off the power. This is why you get no sound, because Windows 10 doesn't have the option for logging in and out sounds, which it is what you are doing when the "fast startup" option is enabled (it is, by default). You would need to disable that option to fully shutdown your system and then you should get a sound when starting Windows.

a. user say:

Hello Shafaat,

We regret the inconvenience. Let me assist you.

Service Host(svchost.exe) is a generic service hosting container. Any number of required system services can be running in them, so I would not get rid of them. You can use Process Explorer to determine what services are running in the host container. The related services might be even the Windows Update or the BITS.

Now, process Explorer is a more advanced way of looking for services running in the back ground. You can start with clean boot to identify any third party applications running which might consume the network bandwidth.
