Hi

jnlp are java files they are not usually the actual target for download.

Many sites use jnlp script to initiate the actual file's download, in many cases when the download from a site is not working well it will try to download the jnpl rather than what you actually want to Download resulting in error.

Make sure that Java is enabled on your browser.
