This is very different from uninstalling via Programs and Features) them does not generally help performance much. This is mostly beneficial for regaining some Hard Drive space.

You can deal with Startup programs either directly or via Clean Boot.
- To find out what is loading with Windows  type   msconfig  into the Start Menu Run Command
or search bar. This will load System Configuration.
In System Configuration there is a Startup tab where you can find what is loading with Windows,
and stop those programs that you don't need to load with Windows from launching.
- Clean Boot is a method for troubleshooting problems that may stem from some Services or Startup programs -

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

Two important measures for keeping Windows running smoothly are -
Windows Disk Cleanup and Defragmenter. Both should be run on a regular basis.
It is a good idea to run the Defragmenter after installing a new software.
- Anti-Virus software can occasionally slow system performance , as can other software, including
some so called system optimisation programs.

  1. "Open Windows Defender by clicking the Start button . In the search box, type Defender, and then, in the list of results, click Windows Defender.

  2. Click Tools, and then click Options.

  3. Click Administrator, select or clear the Use this program check box, and then click Save.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation."

If you have a problem enabling Defender as above try logging on as Administrator, if you
are nor already.

xbox360italy say:

Hello,

When you run the Xbox LIVE Connection Test, you see one of the following errors:
DNS Error

-or-

Your DNS server can't resolve the names of the Xbox LIVE servers or Xbox.com.



The Domain Name System (DNS) allows computers to locate other computers on the Internet. A DNS problem means that your Xbox 360 console cannot locate the Xbox LIVE servers.
To solve this problem, try the following solutions in order.

Note If you are connecting to a secure wireless network, you will need your wireless network password to complete the steps below.

Solution 1: Reset network settings to the defaults Note Before you reset the Xbox 360 console’s network settings, make sure that you write down the current settings in case you have to restore them later.
  1. Turn off your Xbox 360 console.
  2. Wireless connection only: Unplug the Xbox 360 Wireless Networking Adapter from the USB port (if applicable).
  3. Turn on your console.
  4. Press the Guide button on your controller, go to Settings, and then select System Settings.
  5. Select Network Settings.
  6. Select Wired Network or the name of your wireless network, if you are prompted to do so.
  7. Select Configure Network.
  8. On the Additional Settings tab, select Restore to Factory Defaults.

    Note This will only reset the network settings and will not change other console settings.
  9. Turn off your console.
  10. If you unplugged an Xbox 360 Wireless Networking Adapter, plug it back into your console.
  11. Turn on your console.

    Wireless only: You will receive a message to configure wireless settings. Follow the on-screen instructions to connect to your wireless network.
  12. Test your Xbox LIVE connection. To do this, Press the Guide button on your controller, go to Settings, and then select System Settings. SelectNetwork Settings, select Wired Network or your wireless network name (if you are prompted to do so), and then select Test Xbox LIVE Connection.

If you still receive a DNS error, try the next solution.

Solution 2: Restart your network equipment
  1. Turn off your Xbox 360 console and all network hardware (for example, your modem and router).
  2. Wait one minute.
  3. Turn on your modem and wait for it to come back online (one more minute).
  4. Turn on the next piece of hardware (for example, the router), and wait another minute.
  5. Repeat step 4 for every additional piece of network hardware.
  6. Press the Guide button on your controller, go to Settings, and then select System Settings.
  7. Select Network Settings.
  8. Select Wired Network or the name of your wireless network, if you are prompted to do so.
  9. Select Test Xbox LIVE Connection.

If you still receive a DNS error, try the next solution.

Solution 3: Use manual DNS settings If your computer can browse the Internet, let's use the same DNS settings that your computer is using.
  1. On your computer, click Start, type cmd, and then press ENTER.

    Note If you are using Windows XP, click Start, and then click Run. Type cmd, and then press ENTER.
  2. At the prompt, type ipconfig /all, and then press ENTER.



  3. Under your network name, look for a line that begins with DNS Servers.



  4. Write down your DNS server numbers (primary and secondary).

Now enter these DNS server numbers on your console. Here's how:
  1. Press the Guide button on your controller, go to Settings, and then select System Settings.
  2. Select Network Settings.
  3. Select Wired Network or the name of your wireless network, if you are prompted to do so.
  4. Select Configure Network.
  5. On the Basic Settings tab, select DNS Settings.
  6. Select Manual.
  7. Select Primary DNS Server.
  8. Enter the first number you wrote down, and then select Done.
  9. Select Secondary DNS Server.
  10. Enter the second number you wrote down, and then select Done.
  11. Select Done again.
  12. Press the B button on your controller, and then select Test Xbox LIVE Connection.

If you still receive a DNS error, try the next solution.

Solution 4: Try a modem connection This temporary solution will help determine the cause of the problem. If this solution helps you connect to Xbox LIVE, the problem might be a router configuration issue.

Try connecting your Xbox 360 console directly to your modem instead of going through your router. Here's how:
  1. Connect your console to your modem with a network cable.
  2. Press the Guide button on your controller, go to Settings, and then select System Settings.
  3. Select Network Settings.
  4. Select Wired Network or the name of your wireless network, if you are prompted to do so.
  5. Select Configure Network.
  6. On the Basic Settings tab, select DNS Settings.
  7. Select Automatic.
  8. Test your Xbox LIVE connection. To do this, press the Guide button on your controller, go to Settings, and then select System Settings. SelectNetwork Settings, select Wired Network or your wireless network name (if you are prompted to do so), and then select Test Xbox LIVE Connection.

If you still receive a DNS error, try the next solution.

Solution 5: Contact your Internet service provider (ISP) If you receive a DNS error when your Xbox 360 console is connected directly to your modem, there might be a routing table problem. DNS servers use routing tables to direct network traffic. If the tables have incorrect information, your console might not be able to find our Xbox LIVE servers.

Contact your ISP to see if they can help resolve the problem.

Surface Pro and SketchUp - Microsoft Community

guzatsketchup say:

Surface Pro and SketchUp

We will often see issues with SketchUp displaying a blank workspace on Windows computers. 

It may just be a white, grey, or green field with no "FaceMe" component or XYZ lines. 

This is often seen on systems that have only an integrated video card and no dedicated GPU (Graphic Processor Unit). 

If you are using a Surface Pro with only an Intel video card, there is a good chance that you will run into this issue.

The workaround is to go into your SketchUp Preferences > OpenGL > Fast Feedback and uncheck that. 

This is the feature that offloads most of the geometric calculations to the GPU. 

If you are looking for more than a workaround, you'll want to consider the upgraded version of the Surfacebook Pro.

A good example is the Surface Pro with an i7 processor that is docked into its keyboard. It shouldn’t have a problem running SketchUp Pro. This configuration includes a GeForce 1060 6GB discrete GPU and that’s more than enough power to do the job (in most cases).

IF you have the i7 and you are running in Tablet mode OR you are using a Surface Book Pro with a lesser processor (Intel i5 for example), it will not have a discrete GPU with its own dedicated vRAM and you will not be able to properly run SketchUp Pro 2017+.
