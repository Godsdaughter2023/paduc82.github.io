databaseben say:

hmm?

 

standby mode is more like a screen saver mode w/o the screen saver being displayed.

 

while the hibernation mode has the computer to completely power down (except for the trickle used to monitor the mouse wake up mode)

 

in any case, when the computer goes into hibernation, it downloads an exact copy of the data inside the ram memory chips onto the hard drive, inside a file called hiberfil.sys

 

afterwards the computer shuts off the power.

 

subsequently, when the computer is powered up, the data from the hiberfil.sys file will reload the ram memory and your computer session is restored exactly to the state it was.

 

so if you had an open document or program or were in the middle of an email at the time the machine hibernated, all those windows will be restored as though the computer had never shut down.

 

ideally, hibernation is much better than standby.   bascially, if you want a standby mode, simply turn on your screen saver feature instead, because it is the same difference.

 

in any case, don't you have a hibernation mode?  you should, however.
