Clean install invokes the normal Windows setup dialogues common since Vista, including if the Custom Install option is selected the choose the destination dialogue.

There you can erase current partitions and define new ones. Don't forget that when you set the C drive partition Windows will also create System partitioning needed by Windows to boot from. Only around 500MB but will be taken from the size you allocate to C:

Although the logic of creating a separate data partition so it is isolated from the O/S if you have to do a reinstall is undeniable don't forget it is still the same physical device. Good backups are still essential. The biggest problem with partitioning is deciding how much space to allocate, over a significant period of time it rarely works out right.

tryx3 say:

Paul,

This might simply be the battery at fault.  It might be [or might have always been] defective.  Batteries can be badly made - the impressive advances in materials science & their attendant manufacturing processes that have made this amount of mobile power available to us are expensive.  Dodgy battery manufacturers hope to sell defective products that last long enough for you to think that they were okay when delivered. 

So I agree with Frederick, altering Windows [using factory reset or any other adjustments] is not relevant unless you can prove that it is not the battery at fault. 

To confirm that you can blame the battery, set your Low, Reserve & Critical battery notifications to something very large [90%, 85%, 80%].  If they appear correctly then you have narrowed down the cause of the problem to the battery itself.

Once you have proved that the battery is at fault then you can play around with the levels until you find a set of levels that you are happy with.  And you might decide to buy a new battery from a supplier who has good reviews from existing customers.

I have recently had a similar problem and changing battery levels sorted things out or me.  I have old batteries & a very new battery.  The old batteries were crashing the computer without any warning just as yours is.  I did the test I have just suggested to you and found that the old batteries were going from just under 25% to nothing in the blink of an eye.  So I bought the new one but have set my levels to cater for both so I can squeeze the last remaining bit of life out of the old ones:

Low battery level - 30% - notification but no action

Reserve battery level - 25% - [this is a notification-only item]

Critical battery level 3% - Hibernate

If I am working and have an old battery fitted then I translate the Reserve battery notification as meaning that I must connect to AC or swap batteries straightaway.  If I have the new battery fitted then I can continue working as I know that the new battery will hibernate correctly at 3% so there's no panic. 

Do please note that the Reserve battery notification is badly worded and can cause confusion.  This is what it looks like [I had its level set to 7% at the time]

davem121 say:
Hi DCB_1, Microsoft removed the homegroup functionality in version 1803 of Windows 10 . . .

If you have a problem seeing other PC's in the Network Section in File Explorer:

Click your Start Button, type Services and hit Enter

When the Services app opens, scroll down to find the following services, then double click each one in turn, set their startup type to Automatic and start the service if necessary

1. Function Discovery Provider Host

2. Function Discovery Resource Publication

3. SSDP Discovery

4. UPnP Device Host

Save that and close the Services app

That's It!
