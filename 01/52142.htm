ganesh achar b say:

Hi,

 

I would suggest you to go through these steps.

 

 

Requirement:
USB drive (minimum 4GB)

A. Format USB drive as FAT32 on a Win7/Win8 PC.

a. Click on start.
b. Type in diskpart.exe
c. List disk (Find USB device)
d. Select disk 1 (careful! select USB drive
e. Clean  
f. Create partition primary
g. Select partition 1
h. Active
i. Format quick fs=fat32 (UEFI can only USB boot to FAT32, not NTFS)
j. Assign (In this example, USB mount point become F: when assigned.)
k. Exit.

B. Copy Windows 8 x64 ISO content to USB

Mount Windows8-ConsumerPreview-64bit-English.iso (In this example, it's H:) Note, If using Windows 7, use your favorite ISO mounting software. Copy the content using xcopy <ISO drive> <USB drive>, for example.

xcopy H:\* F:\ /s /e




C. Disable UEFI Secure Boot.

Press and hold Volume-Up and Volume-Down, then power up or reboot the slate. It should go into BIOS setup.
In "Advanced Configuration", change "Attempt Secure Boot" to Disabled.
Press Volume-Up and Volume-Down to exit, then "Save Changes and Reset".
Power off the slate.

D. Plug-in USB drive to slate. Press and Hold Volume-Up button, then power up the slate. It should boot into USB drive now.

E. Install Windows 8. When it’s done, USB drive can be removed now.

F. Re-enable UEFI Secure Boot.
Set BIOS "Attempt Secure Boot" back to Enabled.
