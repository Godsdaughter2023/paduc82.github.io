djyahtzee say:

Open regedit (registry editor) by winkey +r and type "regedit.exe" without the quotes. Regedit looks like a regular file manager display. Navigate the left side of the display to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate. There should be a folder (or a "key") titled OSUpgrade. Double click it to open the key. If it doesn't exist, right click anywhere in the window to bring up a context menu. The only option you should see is "New...". Click or hover over it and sub-menu appears. The top selection will be "Key". Click it and a new key (folder) will appear under the WindowsUpdate key. Title it "OSUpgrade" (again, no quotes) and press enter. Double click the new (or existing) key to enter it.

Within the OSUpgrade key, if the AllowOSUpgrade value doesn't exist, again follow the steps to create a DWORD value of that name. Once created, double-click on the value name and set it at 0x00000001. If the value exists already but has a different value, change it the same way. then reboot to get windows to recognize the change during boot.

It's easier than it sounds. If you are familiar with Windows Explorer or Windows File Manager you shouldn't have any problems. It might be smart to back up the registry as soon as you open it, before making changes. To do that, simply click "File/Export..." in the top menu bar. Give the reg file a name, remember where it is. If your computer starts to mess up immediately after the change, navigate to the reg file, double-click it, tell it yes and ok, then reboot.

Hope this helps you.
