hughreaves say:

does defrag work in a raid 0 setup?

shenan stanley say:
I'm using dual 750GB WD Black Caviar hard drives.


Yes.

RAID0 (not really a RAID to me - JBoD maybe) is nothing but a striped set of disks so that it is PRESENTED as a single disk.  Defragmentation can sort the data on array of disks just as well as it can a single disk - since it is the same thing.  I guess it is aptly named if you take it as a whole (there's no redundancy, so RAID ZERO. heh)

I wouldn't recommned striping at all.  When you defragment you do moving around of data which can uncover issues with a drive - which could break the "RAID" set and thus - make you lose all your data.  However, if you have chosen to do so - it contains data just like anything els e- defragmentation is something you should do because the file system will work more effectively if the drive is defragmented.

Guessing you are doing some sort of video/audio editing and you are trying to squeeze out some extra performance as well as have 1.5TB (more like 1.4TB usable) of contiguous space too?  (Two drives... probably between 0% and 15% performance increase depending on what you are doing on them as opposed to using just one of those drives.)

What is the exact make and model of your computer?
