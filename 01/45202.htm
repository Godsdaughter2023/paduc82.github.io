Hi Tomst,

Did you make any changes on the computer prior to this issue?

I can understand the frustration when things don’t work as expected.

If you receive Windows Update error 80246002 while checking for updates, files required by

Windows Update might be corrupt. This often can be corrected by renaming the Windows Update download folder, and then running Windows Update again.

Method 1 : Rename the Windows Update download folder

1. Click Start, and then, in the Search box, type command prompt.

2. In the list of results, right-click Command Prompt, and then click Run as Administrator. If you are prompted for an administrator password or    confirmation, type the password or provide confirmation.

3. In the Administrator: Command Prompt window, type the following commands. Press ENTER after each command:

net stop wuauserv

cd %systemroot%\SoftwareDistribution

ren Download Download.old

net start wuauserv

4. Type Exit, and then press ENTER to close the Command prompt window.

5. Run Windows Update again.

If renaming the Windows Update download folder doesn't solve the problem, download and install the specific update that is causing the error. To do this, follow these steps to identify, download, and install the update you need.

Method 2 : Download and install an update manually

1. Open Windows Update by clicking the Start button, clicking All Programs, and then clicking Windows Update.

2. In the left pane, click View update history.

3. Starting from the most recent dates, find an update for Windows that has failed to install, and then double-click the update to view more information.

Updates that have failed to install will display Failed under theStatus column.

4. In the Windows Update dialog box, find Error details and review the error code for the failed update. If the description doesn't contain error 80246002, click Close.

Repeat steps 3 and 4 until you identify a failed update that includes the 80246002 error code in the description, and then note the number in the title that begins with "KB."

Go to the Microsoft download website, type the "KB" number you noted in step 4 into the download search box, and then click Go.

5. In the list of search results, click the Knowledge Base (KB) article link to go to the download page. If there's more than one listing, look for a link that goes to the Microsoft Download Center.

6. If there are multiple versions on the download page, find the appropriate one for your computer. Click the Download button, and save the update to your desktop.

7. When prompted, click Open to install the update.

Method 3 :

If you continue to see this error, I suggest you run the automatic troubleshooter and check if it helps.

Hope this helps, if the issue persists with Windows updates, please get back to us. We are here to assist you.

a. user say:

You can indeed and with no need to be signed in on the site.

 

Press the Guide button, go to the Settings blade and select System Settings.

 

Choose the top option of Console Settings, then Startup and Shutdown.

 

Here pick the last on the list of Background Downloads and Enable it.

 

Then just press the off button on your console.

 

This will let your console download in what is essentially a standby mode and when it is complete, the console will turn itself off fully.
