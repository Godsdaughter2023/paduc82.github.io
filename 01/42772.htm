daniel_schär say:

Toggle action for TabTip

I'm installing a Laptop for an old Lady, who prefers tab-mode with a Keyboard and no mouse, so TabTip keeps unwantingly poping up, when she taps on text-fields.

I found no solutions in forums, but found out, that TabTip is enabeled by the service called "TabletInputService", so I wrote a batch file with toggle action to disable/enable  "TabletInputService" (see below).

It works fine and touchscreen input seems not to be affected (I can still draw in "Notes"; don't know however, how it is with a Wacom tablet or similar devices).

The batch-file has to be run in administrator mode (see below for tricks).

This is my batch-file (ToggleTabTip.bat), which should be easy to adjudst to your needs (e.g if you don't want my cool German prompt message):

-------------------------------------------------------------------------------

rem
@echo off
rem *
rem *      Toggeling action for disabeling/enabeling
rem *                               TabTip
rem *
rem *      THIS HAS TO RUN IN ADMINISTRATOR MODE
rem *
rem *  service:
rem *   TabletInputService
rem *
rem *  commands:
rem * sc stop "Name of Service"
rem * sc start "Name of Service"
rem * sc config "Name of Service" start= [disabled¦demand]
rem *           The space after the "start= " is important
rem *
IF NOT EXIST c:\baf\scripts\flags\tabon GOTO ACTI
GOTO DEACT

:DEACT
c:
del c:\baf\scripts\flags\tabon
sc config "TabletInputService" start= disabled
sc stop "TabletInputService"
SET TTMSG=aus
GOTO END

:ACTI
sc config "TabletInputService" start= demand
sc start "TabletInputService"
echo tabon>c:\baf\scripts\flags\tabon
SET TTMSG=ein
GOTO END

:END
echo off
rem TIMEOUT /T 35
cls
echo *
echo *
echo *
echo *    Die Bildschirmtastatur ist %TTMSG%geschaltet
echo *
echo *
echo *
echo *
echo *
TIMEOUT /T 5 > NUL
rem Pause
rem *** Email address is removed for privacy *** - 2016

-------------------------------------------------------------------------------

As stated above, you have to run the file in admininistrator mode.

To make this easier, you can create a link to the bat-file and check the "Run as administrator" box in the extended properties. This link can be called with a 2nd bat-file, that can run in user mode (just 1 card:

DRIVE:\\PATH\NAMEOFLINK.lnk ), so that any link to this 2nd bat-file will start the process.

NOTE: Do not put the files and links for this procedure into a protected directory (like "C:\Program Files" or "C:\Windows") or a subdirectory of such a directory, as this will cause blocks from internet-security (the user notifications on my computers are on the lowest level "never notify", as constructs like the one described here cause a lot of popups otherwise).

You can download pre-configured scripts from my website:

http://bafweb.ch/bafmat/taptiptoggle.zip

Simply unzip "taptiptoggle.zip" to C:\ and then copy the link "Toggle TabTip" from C:\baf\ to any desired location.

I hope, this helps someone.

Hi,

 

Welcome and thank you for posting your query in Microsoft Community.

Apologize for the difficulties that you're experiencing in Windows 10.

Not to worry, we’re here to help you out.

 

I suggest you to run Startup repair from Advance boot option.

It may ask you to insert the Installation DVD/USB while doing startup repair.

You have to create an installation media for that.

 

You may download an ISO from the below link:

https://www.microsoft.com/en-us/software-download/windows10

 

Once your media is ready follow the steps.

  • After you boot your computer using Windows 10 CD/ DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.

  • Select the correct time and Keyboard type.

  • Click on Repair your computer in the lower left corner.

  • Select Troubleshoot from Choose an option screen.

  • Click on Advanced options in Troubleshoot screen.

  • Click on Automatic Repair from Advanced Options screen.

  • Next, choose the operating system by clicking on its button.

  • Windows 10 will look for problems on hard drive(s) and verify that all required files are intact.

 

Hope it helps.

 

Let us know the status, we are glad to assist you further.
