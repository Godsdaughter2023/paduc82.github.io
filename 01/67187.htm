The Hardware ID is generated by Windows Instrumentation as a result of interrogating the drive itself. This is held in the BIOS of the drive and cannot be changed in any way by the user, the drive interface blocks any non factory changes.

If it is that important to you the only way to present an alternative is to replace the drive. However all drives of this type will have the same hardware ID so their gamer ident system must have an alternative method of knowing you as an individual.
