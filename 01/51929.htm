rob koch say:

Typically this would be due to a sticking "2" key, but you appear to have already tried the obvious swapping of the keyboard itself to attempt to confirm whether it was causing the problem.

Try completely disconnecting the physical keyboard and use the On-Screen Keyboard instead.

Use the On-Screen Keyboard (OSK) to type - Windows Help

That document tells you how to access the OSK from the Windows 10 sign-in screen if you need to type a password as follows.

To open the OSK from the sign-in screen, select the Ease of Access button in the lower-right corner of the sign-in screen, and then select On-Screen Keyboard.

To access the OSK once logged into Windows 10 with only your mouse:

Go to Start , then select Settings > Ease of Access > Keyboard, and then move the slider under On-Screen Keyboard. A keyboard appears on the screen that can be used to move around the screen and enter text. The keyboard will remain on the screen until you close it.

If the On-Screen Keyboard works OK, then your problem is with either the keyboard, the motherboard computer hardware that accepts input from it or possibly the keyboard driver software.

If the OSK doesn't work either, then the problem may be something damaged in either the drivers or more likely Windows itself, at which point the original cause doesn't really matter.

The only way you'd likely recover from such damage is a complete reinstallation of Windows 10, which should wipe out whatever software damage has occurred along with everything else contained in the existing copy of Windows, including any malware or other corruption.
