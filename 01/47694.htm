Check your antivirus software

In your antivirus app, look for settings related to blocking access or permission to use your webcam. For help, visit your antivirus software company’s website.
Check for an outdated webcam driver

Select the Start button, type Device Manager, and then select Device Manager from the search results.
Find your webcam under Imaging devices or Sound, video and game controllers.
Press and hold (or right-click) the name of your webcam, and then select Properties.

tiffany mcleod say: If you are using Vista Business or Vista Ultimate, and have Shadow Copy enabled (this is often enabled by default) you can try the "Restore Previous Versions" feature:

Open Windows Explorer and navigate to the file in question.

If you want to keep the new file as well as restore the old version, copy the file to another location before proceeding further.

Place your mouse over the file and Right-click.

From the menu, choose "Restore Previous Versions".

Wait for the computer to access the previous versions. This could take a few moments.

Select the desired previous version and click the "Restore" button. (Or the Open or Copy buttons if you want to do that instead.) Click "Ok" as prompted.

The file will now be overwritten with the older version. You will lose all changes made since the older version you selected above.

If you are using Vista Home Premium or Vista Basic Editions, you may find the third party software Shadow Explorer useful (I have not used this software personally).

ch.theriault say:
Hi jorj1,

I had a problem similar to yours and I successfully addressed the issue, for my case at least. But it might as well be your case too!

It was a spyware installed as a Browser Helper Object (see http://en.wikipedia.org/wiki/Browser_Helper_Object) for more info. This is the strategy used to install toolbar plug-in in Internet Explorer. These plugins have privileged access to the content of you web page. They can insert ads in the web pages you are visiting (as it is in our cases), but some of them can work silently and to steal your password as you key them in.... You detected the presence of the one which pushes ads, because of the spy cookies.

You can see/activate/inactivate all plugins installed in Internet Explorer from the menu "Tools", item "Manage Add-ons...". Then, pay attention to components which have no known signature, one of them MIGHT be the culprit. Mine was a control with no name (suspicious eh...!) and was identified as "flvdome". Stuff from Microsoft, Adobe, Sun Microsystems, Simtec, ... are safe. Try to disable the suspicious ones, one  at a time, in order to turn off only the bad one.

Good luck
