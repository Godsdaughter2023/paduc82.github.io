Remove Proxy Servers Settings - Windows

Using your Windows PC press and hold the Windows Key and hit "R" at the same time, it will pull up the Run Prompt, it is a blank space, on that type in:

regedit

and hit enter.

You'll get the registry editor,  just a disclaimer that this is the computer's registry please follow the instructions carefully to avoid serious issues.

The Registry Editor has this folders on the left pane, follow the key line(keys under folder under folder)

HKEY_CURRENT_USER ->
SOFTWARE ->
Microsoft ->
Windows ->
Current Version ->
Internet Settings.

Under Internet Settings key folder, look on the right pane you'll see a lot of keys and script like things there, high light them all and delete, except one which is the very first key named as "default" (it is color red to be specific).

Once done with this step, hit F5 key to refresh the registry, some times there are malwares that configure these keys and they keep on going back here(Proxy Server registry). If the keys stays on default, you may test your browser or any application that connects to the internet.

By the way this step resolves the Tile App issues that was unable to connect to the internet, such as Mail app, Store app, Weather app and so on.

seancollins2 say:

A slight compromise but it works.

  1. In the recycle bin properties, untick "Display delete confirmation dialogue"
  2. Open the folder containing photos to review in windows explorer
  3. From the menu bar, set the View to extra large icons
  4. From the menu bar, click the preview pane and open the preview as large as you can at the side
  5. you can now select individual photos or multiple photos by holding shift key and then press delete 

Another useful tip, if you have sub folders, start at the top folder as above, and in the search option on the right

enter *.jpg or *.tif (which ever image format you use) and it will show all images in the whole folder hierarchy.

This doesn't seem to remove them from the search results, but they are deleted.

(re do the search and you will see deleted images have gone.)

Hope this helps.
