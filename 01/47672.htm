a. user say:

Have you printed to this printer before or was it just installed?  Lexmark printers have problems running on Vista becuase they often don't have Vista drivers available for the printers.

Go to Start / Control Panel / Printers and right click on the Lexmark Printe (make sure it is set as the default printer 0 to do so right click on it and click on set ss default printer - if the option is not available it is already the default printer) and then click on Print a Test Page.  Does the page print?

Go to Start / All programs / Accessories / Notepad and double click to open it.  Type "This is a test of the Lexmark printer" and then click on File and click on Print.  Does the document print?  If so, try printing again from Word and see if it works.

If nothing prints, there is something wrong with the printer, the printer drivers, or the printer set-up.  If it was working before and isn't now or both pages print but you still can't print from Word, do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

In that case you need to start by updating the drivers which are appropriate for the printer and your version of Vista.  Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (and you do for the Lexmakr Printer), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows: http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows: http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html. 

If both of these pages print there is something wrong with how Microsoft Word is set to print or with the program itslef.  If that's the case, please repost your question in the Word section of the Office Discussion Forum at: http://www.microsoft.com/office/community/en-us/flyoutoverview.mspx Select the Word General  Discussion group as they have the most participation so rreplies are likely to be faster

I hope this helps.

If this is a particular CD that you are not able to rip, the ideally would not have anything to do with the Windows Media Player or the settings. you may receive the error that Windows Media Player cannot rip one or more tracks from the CD. The error is often caused when you try to rip the CD to a low quality.

Try the below steps and ensure you've got an improved audio quality recording.

Method 1:

a.     Open Windows Media Player.
