Changing from Windows 7 32 bit to 64 bit - Microsoft Community

gordontate say:

Changing from Windows 7 32 bit to 64 bit

I have a Dell E5420 Latitude laptop which came installed with Windows 32 bit OS.  Having checked, it appears that the laptop is capable of using a 64 bit system.  I have a Windows 7 64 bit installation disk and understand that to change from 32 bit to 64 bit involves a clean install, i.e not an upgrade.  I also have an external hard drive for backups.  

There are a couple of questions I would appreciate guidance on before I undertake the change from 32 to 64 bit.

1.  Am I correct in my belief that the Windows 7 64 bit installation disk will format the hard drive before it installs the 64 bit system?

2.  Could some kindly tell me the best way to go about copying all of my existing user files plus downloaded apps so that I can transfer them back onto the laptop when I have installed Windows 7 64bit.   Could I clone the 32 bit system using Macrium Free Edition and maybe transfer files from that, or could I use Windows Easy Transfer? Perhaps there are better ways and things to be careful of. 

ken blake say:

Several points/questions:

1. You are correct. There is no way to upgrade from 32-bit to 64-bit. You need to do a clean installation.

2a. Perhaps the easiest way to transfer your files is simply to copy them to external media before installing 64-it Windows. Then copy them back after the installation. You can use an external hard drive, thumb drive, CDs, DVDs, etc. Which device is best depends mostly on how much you have to transfer, but since you already have an have external drive, I would just use that.

You can also use Macrium or Easy Transfer, but as far as I'm concerned, simply copying is easier.

2b.  You can not transfer programs. They will have to be reinstalled from their original media (or from the downloaded files if you still have them--re-download them if you don't).

Why do you want to do this? How much RAM do you have? What programs do you run? There's a very good chance that any difference in performance will be so small that you wouldn't notice it. You might get a somewhat better improvement in performance if you would replace all your programs with 64-bit version, but that could be a considerable expense.

Since you will have to reinstall all your programs, there will be a lot of work doing that and reconfiguring Widows and all your programs the way you like them.  In my view, it's probably not worth the effort.

Note that most programs that run on  32-bit Windows will also run on 64-bit Windows, but there's an occasional one that won't--mostly some games and some utilities

You say "it appears that the laptop is capable of using a 64 bit system." If the word "appears" means you are not sure, be *very* careful. There's only one requirement; to do this, you must have a 64-bit CPU. Please make *sure* it's 64-bit.

canadian tech say:

There is a small benefit going from 32 to 64.  About 15% improvement.  More memory will yield a bigger improvement.

You cannot transfer an image.  That will not work.

There is not much benefit from this change unless you have applications that need it and/or will run better.

Use Windows Easy Transfer.  But backup your data to your external drive.  Simply drag and drop copies from the C: drive to the external drive.

You cannot backup applications.   They must be re-installed.

Yes, this will require a complete loss of what is currently stored there.  Strongly recommend when the install gets to the point when it asks you where to put the installation -- click Advanced and delete ALL partitions.  The installer will take care of all else that needs to be done.

If your laptop hard drive exceeds 5 years or is close to it, I strongly recommend you replace the drive as part of this.   Life expectancy of these drives is not much more than 5 years.  Hard drives are not that expensive and easy to replace.
