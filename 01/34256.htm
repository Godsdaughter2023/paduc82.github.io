bossdweeb say:

"I don't notice anything. As so my FPS did not increase. Is this part of it or not"

Theoretically, but in practice, probably not. A lot has to do with your hardware and the game in question.

For the most part, Game Mode is MS hype. All show and no go.

All Game Mode does is disable some stuff that is running in the background, which is supposed to free up some cpu cycles. You can do part of this procedure yourself by disabling programs in Startup (in Task Manager) that you don't use on a daily basis

to prevent them loading when you boot into Windows.

It can also automatically end some tasks in Processes that may not be required while you are running the game, but

of this is piddling on modern machines.

Some games are more cpu intensive than others, so if you have a low end cpu and are running a cpu intensive game

and also don't have a dedicated graphics card you may get a few extra fps increase with Game Mode enabled.

It's also supposed to transfer some graphics processing load from the gpu to cpu if a low end gpu is overwhelmed, but you can't squeeze

blood from a stone. If your machine is not up to the task of running a game properly in the first place Game Mode is no panacea.

If you wish to run graphics intensive games with high settings smoothly you need high end hardware. There are no shortcuts.


Then there is the downside. Unless MS have done something different since introducing Game Mode, tests with various games

have actually shown it can be counter productive, so some games will loose a few fps, or do fun things like stutter more than

they would if Game Mode is not enabled.

- As you may already know, you can't upgrade graphics on a laptop. What you see is what you get. Personally, Game Mode

is just another gimmick MS is using to try and help justify their claim that Win10 is 'the best gaming Windows ever'.
