Do you mean a wireless network encryption key/pass-phrase/password?

If so...

Open the Network and Sharing Center, click on Manage wireless networks at the left, highlight and right click on your network and select Properties. Click on the Security tab.

http://sdrv.ms/LzWbgO

Alternatively...

If you have another wireless Win 7/Vista/XP computer connected via WiFi to your network you an use Windows Connect Now [WCN] to put the wireless profile on a flash drive. You can then plug in the flash drive to the Win 7 computer and the profile will load.

http://theillustratednetwork.mvps.org/LAN/SoHoWirelessSecurity.html#Windows_Connect_Now

Some routers also allow you to see the network encryption key/pass-phrase/password in plain text. Login to your router admin pages and check the wireless security settings pages. This page lists common routers, their default IP addresses and the default admin passwords (if you never changed yours).

http://windows.microsoft.com/en-US/windows7/set-up-a-wireless-router

As a last resort you may need to simply restore your router to its factory defaults and start over setting up the network. Check your routers users manual or manufacturers support pages for help with that...

http://windows.microsoft.com/en-us/windows7/Setting-up-a-wireless-network

If your talking about the router admin password your best bet is to probably restore the router to its factory default and redo the admin password and wireless security encryption, ie. type and key/pass-phrase/password. Save off the new profile on a flash drive using WCN for later use...

Otherwise please give us more detail as to what you really mean...

http://support.microsoft.com/kb/555375

 Any help would be apprecriated.  I have looked around but most people seem to be going in circles once they get this message.

diana d say:

Hi Richard,

A) This can happen when related components still exist even if a program is uninstalled. To completely remove the software or a program, you may use the Windows Clean up utility tool.

1. Visit the following link to download the Clean up utility: http://download.microsoft.com/download/E/9/D/E9D80355-7AB4-45B8-80E8-983A48D5E1BD/msicuu2.exe      
2. Save the file to the computer.
3. Install this tool on the computer.
4. Click Start -> (All) Programs -> Windows Installer Clean Up.
5. See if you can find any components related to the program which you have uninstalled. If so, remove them.
6. Restart the computer.


B) If the above does not work, then perform a clean boot. In Clean Boot, windows use a minimal set of drivers and startup programs so that software conflict is eliminated. To put the computer in clean boot state follow the steps in the article “How to perform Clean Boot” http://support.microsoft.com/kb/929135      

Note: Please ensure that the computer is set to start as usual by following the step 7 from the article http://support.microsoft.com/kb/929135
