16 words is a very, very long name for a computer! You can most likely fix the problem by using System Restore. Here is how it works:
1. Keep tapping F8 during the early boot phase.
2. Select Repair from the menu.
3. Select an admin account, then enter its password.
4. Select System Restore when prompted.
5. Set Windows back to a point before you changed the name.
