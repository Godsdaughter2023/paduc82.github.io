Hi Sam.desilva,

 

Thank you for posting your question on Windows Vista Forums.

 

GUID stands for Globally Unique Identifier is a special type of identifier used in software applications to provide a reference number which is unique in any context (hence, "globally"),

 

While each generated GUID is not guaranteed to be unique, the total number of unique keys (2128 or 3.4×1038) is so large that the probability of the same number being generated twice is extremely small.

This is the reason that you find a different GUID every time.

 

The generation of GUID’s depends on multiple factors like the software, the platform on which software will execute, the Mac ID of the hardware (if required), the registry references etc.

Allurfine/Chimel:

The "show desktop" icon is redundant, good stuff... Ok then, let's put all this together:

1) How to create "show desktop" icon- and place it on start bar (Or for that matter in any folder you want that you have write access to on the drive):

Click Start, click Run, type notepad in the Open box, and then click OK.
Carefully copy and then paste the following text into the Notepad window:
[Shell]

IconFile=explorer.exe,3

[Taskbar]
