frederik long say:

It is an old computer. Could that mean it's time to buy a new computer?

Not really. If the problem was component-related then it would not even get this far. This is almost certainly a software problem. Software problems can be fixed. Of course, if the computer no longer meets your requirements (speed, capacity) then this would be a good time to replace it.

To fix the problem you can do this:

  1. Keep tapping F8 at boot time to get into the Windows Repair Environment.
  2. Select System Restore.
  3. Restore Windows to a point about one week ago. You won't lose any personal files.
