pauli o say:

Virtual Machine environment like Parallels and Mac (with Boot Camp) are separate "computers" and those can't use same license.

Your Windows license is linked to Parallels "hardware ID" and can't be transferred to mac unless you have Retail license. You can check your license type in Command Prompt (CMD) with command: slmgr /dli

That is exactly same situation as using Hyper-V or VMWare on Windows PC. Those virtual machines need their own licenses.
