robert aldwinckle on forums say:
                 list of favorites folders I could access down the left hand side of every folder I opened. 

Favorites is an alias for Links.  Same screwy naming that IE uses.   So, if you have them they would be in  %UserProfile\Links.   Then in a File Explorer window you would want to turn on the Navigation pane.  Apparently we don't get to see that from the This PC node.  So open a "folder" and turn on the Navigation pane.  In addition you will have to customize the Navigation pane to show Favorites.  To do all this I have a QAT button at the top of the File Explorer window.  I don't know if that is a default.  Otherwise the standard keyboard shortcut is (apparently)  Alt-V N.  Who knew?  Thanks for making me see this.  I think I'll stick to my QAT.  If necessary I can use  Alt-3 (i.e. its position in my QAT) to open it without resorting to a pointer device.

a. user say:

Hi,

 

Since you're using Windows Live Mail, please try to recover it by checking your folders.

 

You may be able to selectively recover the deleted folders using the Previous Versions component. You will need to know the name of the folders that were deleted. Here is the step by step procedure.

 

1. You will first need to select the Show hidden files option.

    a. Open Windows Explorer and click Organize / Folder and search options. Click the View Tab.

    b. Place a check mark in the Show Hidden files, folders and drives option. Click Apply/OK.

    c. Close and restart Windows Explorer.

 

2. Open the WLM Store folder.

    a. Navigate to the following folder.

    C:\Users\your name\AppData\Local\Microsoft\Windows Live Mail

    (Where 'your name' is the name of the account that you use to logon to Windows 7)

    b. In this folder you will see all of the WLM storage folders. Each email account will have its own group of folders.

    c. Any general storage folders that you created will under the main Storage Folder item.

 

3. Make a copy of the Windows Live Mail folders previous version.

    a. Right click the Windows Live Mail folder and select Properties / Previous Versions Tab.

    b. After the list populates, select a version of the folder with a date and time prior to when the folders were deleted.

    c. Click the Copy Button. In the Copy dialog box, select the Desktop as the location to place the copied folder.

 

4. Move the previous versions of the deleted folders to the WLM store folder.

    a. Go to the Desktop and open the copied WLM folder. Locate the folders that were deleted.

    b. Right click the folder and select Copy. Go to the current WLM storage folders and paste the folder into the same location where it was deleted from.

 

After you have retrieved all of the deleted folders, you can delete the copied WLM previous version folder from the Desktop.

 

If you are using POP3 and you are leaving a copy of the email on the server are not set to delete email from the server then there should still be a copy of the email available on the server.

 

If the account is a Hotmail account the email may be recovered via MS… Refer to Hotmail article: http://windows.microsoft.com/en-US/hotmail/how-recover-deleted-messages

 

Thanks!
