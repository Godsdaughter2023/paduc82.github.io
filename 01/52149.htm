saltgrass say:

Just some things I noticed about your system.

You have Disk 0 with drive letter A which is normally reserved for floppy drives..

Disk 0 contains your boot files in the System Reserved partition.  That happened because on a Legacy system the install looks for the primary hard drive and puts the boot files there and then puts the OS where you tell it to.  Hopefully when you reinstalled the problem was corrected.  The Disk numbers are determined by SATA port.  The lower SATA number the lower the Disk Number.  If you were to change connectors you would probably have to reset the bios boot options.

tim de baets say:

Such 'ghost' music files are a common problem in WMP, and can only be removed by resetting the library. First note that resetting the library will make you lose all current play counts, and possibly ratings too. Then, follow these steps:

- Reboot the PC.
- Go to Start, type  services.msc in the Search field, and press Enter. In Services, find the Windows Media Player Network Sharing Service . If its status is Started , double-click the service and click  Stop .
- Go to Start, type  %LOCALAPPDATA%\Microsoft in the Search field, and press Enter. This folder will open up in Windows Explorer.
- In this folder, rename the  Media Player folder to something like Media Player old . If you can't see the folder, then it's probably hidden. Set Explorer to show hidden files and folders, and to show protected system files.
- Start WMP, your library will have been reset and will be rebuilt.
