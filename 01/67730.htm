A power cycle of the console for a brief amount of time will not refresh the wifi connection; however, if you cycle the console AND the network hardware simultaneously for a minimum 5 minutes, then turn on the network hardware (wait until the lights all come back on) and then power up the console, it will strengthen your wifi connection.     considerate_guy say:

    In general a transition (animation) happens between two clips and an effect (visual effect) applies to the full duration of a single clip.

    So a Fade In or Fade Out really doesn't fit cleanly in either category. There are a few transitions that were made to work on the first clip and then they transition between an implied black screen and the clip itself. So a "Blur from Black" works here. But you can't apply it to the last clip. Any transition applied here is between that clip and the one before.

    Instead the developers implemented a Fade In or Fade Out as a special type of effect but one that only applies to the beginning or ending few seconds of a clip rather than the whole clip. 

    It's really confusing so I can see how it can be hard to figure out.

    Unfortunately, we are unable to access the screenshot from the link that you have posted on this thread. I would suggest you to upload the screen shots in the SkyDrive and then mention the link of its location in your post. You may refer to this link for more information:

     

    http://social.technet.microsoft.com/Forums/en-US/w7itproui/thread/4fc10639-02db-4665-993a-08d865088d65

     

    By default, Windows 8 does not show all the user accounts of the computer on the login screen as we had in the previous version of Windows. It only shows the last user account from which the user had logged out.

     

    In order to view the other accounts of the computer, you need to click on the “Arrow” mark pointing towards the left side, which is located on the left side of the user account picture on the login screen.

     

    If you are unable to find the “Arrow” mark on the login screen, then you may need to check if all other user accounts of the computer are enabled or not. If they are disabled, then you may have to enable them and check if that resolves the issue.

     

    We can check this from “Computer Management” utility. You may follow these steps to try this.

     

    a) Press “Windows Logo” + “X” keys on the keyboard and then select “Computer Management” from the menu to open it.

    b) In the Computer Management window, on its left pane, click on the option “Local Users and Groups” to expand it.

    c) Click on the sub option “Users” and you will be able to see all the user accounts of the computer listed in the right pane of the same window.

    d) Check if there is an inverted arrow mark on the user account icons shown there. (The inverted arrow mark shows that the account is disabled)

    e) If you see the inverted arrow mark on the user account which you would like to have on the login screen, then you may double click on it and then uncheck the option “Account is disabled” on the “User” Properties window.

    f) Click on “Apply” and then on “Ok”.

    g) You may now close the Computer Management utility and then restart the computer and check if you are able to find the “Arrow” mark pointing towards the left side on the login screen.

     

    Hope this resolves the issue. If you still face the same issue, please do get back to us with all the additional information about the issue.
