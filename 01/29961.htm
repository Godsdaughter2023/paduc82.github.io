Where are the downloaded apps from the store located in Windows 10?

I would like to find them and be sure they are their.Also,where can I download Windows Server 2003 OS iso?

a. user say:

Hi,

Thank you for posting in Microsoft Community.

By default Windows stores all apps in a folder "WindowsApps" present in "Program Files" folder. To access this folder follow these simple steps:

1. Open My Computer and go to C: drive. Now double-click on "Program Files" folder.

2. This folder contains "WindowsApps" folder but you'll not be able to see it because its a hidden folder. 

3. You'll need to enable the option to show hidden files and folders. You can enable the option by clicking on "View" menu in ribbon and enable "Hidden items" checkbox.

4. If you try to open this folder, you'll get following error message:

You don't currently have permission to access this folder. Click Continue to permanently get access to this folder.

And if you click on continue button, you'll get another error message:

You have been denied permission to access this folder. To gain access to this folder you will need to use the security tab.

It happens because you don't have sufficient permissions to open the folder.

6. You'll need to take ownership of the folder. Just follow the steps given in following tutorial:

https://technet.microsoft.com/en-us/library/cc753659.aspx?f=255&MSPPError=-2147217396

7. Once you take ownership, you'll be able to view the content of "WindowsApps" folder. You'll find various folders containing all installed apps.

 

Refer the below link with regards to Windows 2003 server ISO.

http://www.microsoft.com/en-in/download/details.aspx?id=21700

    a. user say:

    The graphics resolution of the images on your phone's screen seems to be lower than the resolution of your computer's monitor. As a result, what looks fine on your phone will look pixelated on your monitor.

    The product key is usually located on the COA sticker attached to the bottom of your laptop or in the battery compartment. In some rare cases it can also be found in the memory compartment. For tower systems, look at the top or the side of the system unit. It looks like the following:

    If the Certificate of Authenticity (COA) sticker is damaged or letters or not legible, you will have to consider the following options:

    • Use a digital camera to capture a picture of the COA then enlarge it on your computer.

    • If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee. This will exempt the need to use the product key also.

    Hi Gerald,

     

    If Spirit's posts ae beyond you, that's OK.  BSOD problems are tough for even the very best of us and sometimes though we do all that's there and even more, we still don't resolve the problem.

     

    I would offer the following choices at this point (we'll talk about the order of them later):

     

    1.  Try a repair install as explained here: http://www.sevenforums.com/tutorials/3413-repair-install.html.

     

    2. Try a clean install of Windows 7 (start from scratch in essence).  If you moved from Vista to Windows 7 with an Upgrade Disk, use this tutorial: http://www.sevenforums.com/tutorials/31402-clean-install-upgrade-windows-7-version.html.  If you moved from Vista to Windows 7 with a full Installation Disk, use this tutorial: http://www.sevenforums.com/tutorials/1649-clean-install-windows-7-a.html.  If you want to do a custom install of Windows 7 instead of a clean install to preserve some of the old information on the system (which may be good or may retain the cause of the problem though that's rare), use this tutorial: http://www.sevenforums.com/tutorials/20146-custom-install-windows-7-a.html.

     

    3.  Do a more thorough test of your hard drive to confirm it isn't failing (as the loss of information like this plus the earlier BSOD issues makes be think bad hard drive).  To test your hard drive, check the manufacturer and try HD Diagnostic (http://www.carrona.org/hddiag.html) or this test: Hard Drive Diagnostics Tools and Utilities (Storage) - TACKtech Corp  (http://www.tacktech.com/display.cfm?ttid=287) or see if the computer manufacturer or the hard drive manufacturer have their own diagnostic utility you can download and run (which might be the best as it's very specific for that drive and sure to be current).  If it fails the tests, replace it and then do option 2 above making sure to add the appropriate drivers during installation.  If it's bad, this may fix everything once you reinstall everything.  On the other hand, even if it's bad, while replacing it will help and may be needed anyway (or at least should be done if you plan to keep the computer), there's no 100% guarantee that it was the only problem or that replacing it will resolve everything.  It's likely, but not a guarantee.  You need to decide whether the cost is worth it to test a probable but not proven solution to a known problem but not necessarily the one or only one behind the problems you're experiencing.  Of course, if it comes with a money-back guarantee, that makes the choice a little easier to make since if it doesn't work, you can always return it at little or no cost.  But given your level of understanding, while the tests may be possible, I suspect you'll need help if it is found to be bad and you decide to replace it.

     

    4. Find a reputable computer repair shop (not Geek Squad but one that's has true professionals, a long history, and wonderful reviews from trusted sources) or see if the computer manufacturer will repair it if you ship it to them (and compare the costs, timing, warrantees, and other factors - just like shopping for a place to repair you television or car).  A relative, friend, neighbor who will do the work for free with you only paying for parts may save money, but be sure this person REALLY knows what he/she is doing and you're not selecting on the basis of your social interactions or dealing with someone who says he/she knows a lot more than is actually true (and like all such choices, this can turn out to be a big win or a big regret - it may end up a lot cheaper or may end up a lot more expensive along with risks to your personal information or making things that are only slightly bad much worse - but that's a risk in a way with whoever you choose).

     

    5. Prepare to move this computer to the basement and start over with brand new computer and transfer only your important data to make sure you don't potentially move the cause of the old problems to the new computer.

     

    In some cases you can try several of these options or, depending on what you choose, only one may be required. 

     

    Given what you've said, I'd say try option 4 if it can be done reasonable inexpensively or if it's going to cost almost as much as a new computer anyway, then go with option 5 (assuming you have the money to do either of these - a big assumption on my part that only you can decide about). 

     

    If you don't have that kind of money on hand (I sure know that feeling - LOL!), then I'd say try just the testing part of option 3 to verify the hard drive is OK (and hope it passes) and, if so, first give option one a try and if it doesn't work, then give option 2 a try.  Of course if even these are beyond you, then options 4 or 5 are really your only choices.
