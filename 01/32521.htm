You can use an Upgrade or Full Windows 7 disk to do a custom clean install. This will remove all your programs, but you can save your files and settings to external storage before doing so. You will have to reinstall all your programs after installing Windows 7. Here are the steps:


1. Run Windows Upgrade Advisor to see if there are any known issues that might affect the installation and whether you can install the 32 or 64 bit version of Windows 7.

http://windows.microsoft.com/en-US/windows/downloads/upgrade-advisor

2. Backup all your files and settings to avoid losing docs, photos, and other information. Windows Easy Transfer is a free tool to copy files (not programs) from your PC, and then transfer them back after you install Windows 7. If you don't want to use Windows Easy Transfer, you can copy your files to a USB drive, or CD/DVD. Note: Easy Transfer will not work when changing languages, or when going from 64 bit to 32 bit (it will work going from 32 bit to 64 bit).

http://windows.microsoft.com/windows-easy-transfer

3. Locate the installation disks and any associated product/license keys for all your programs because you'll need to manually reinstall all programs. If you downloaded some programs from the Internet you can redownload them.

4. Put the Windows 7 disk into your DVD drive. A window should pop up starting the install process, when asked the type of installation do you want click Custom (advanced). Note: 32 to 64 bit requires you to restart and boot from the DVD.

5. After Windows 7 is installed you can use Windows Easy Transfer (or another backup method you used) to restore your files & settings, and use your installation disks to reinstall all your programs. Then you're done - enjoy Windows 7.

shenan stanley say:
   Is Outlook included in Windows 8.1 or do you have to purchase separately  
This all depends on what *you* are really asking as an end-user/consumer.
 
There is some misunderstanding/confusion about the different Windows 8/8.1 offerings out there.
 
Know there is two different platforms.
  1. x86 (your standard desktop and laptop like Windows 7 and prior ran/run on)
  2. ARM (systems like your phone and tablets.)

    The former is usually more expensive (in compact form - like laptop/tablet format) than the latter for various reasons - depth of functionality being one of them.
For end-user (home consumers) there are technically three 'editions' of Windows 8/8.1. 
  1. Windows 8/8.1 STD (Standard - x86 system) (32-bit and 64-bit versions)
  2. Windows 8/8.1 PRO (Professional - x86 system) (32-bit and 64-bit versions)
  3. Windows 8/8.1 RT (ARM processor)
Microsoft has not, until Windows RT, included any version of their Office suite with their operating system.  OEM (original equipment manufacturer) assemblers like Dell, HP, Gateway, Packard Bell, Lenovo, etc. may have installed full or trial versions of the various Microsoft Office products in the past - but that was *their* doing, not Microsoft's.
 
With Windows RT, Microsoft included the RT version of the Microsoft Office product suite.  At first (Windows RT v8.0) that did not include "Outlook" and the reason was something about it not being able to work with the way Windows RT conserved power or something like that.  Somehow either that got resolved or it never existed and with Windows RT v8.1 - Outlook was included with the Microsoft Office RT suite.
 
So - if you are looking at a true Windows 8 system on an actual x86 system (a system that might run other versions of Windows and applications you currently run on top of other versions of Windows) - then you do not get any Microsoft Office suites with your purchase *provided by Microsoft*.  If you are, instead, looking at a new platform (do not care about running the applications you may already run on a current Windows system you have) and get a Windows RT system (8 or 8.1 - it's a free upgrade to 8.1 anyway) - then you will have access to Microsoft Office Outlook RT once the system is fully upgraded/patched.
