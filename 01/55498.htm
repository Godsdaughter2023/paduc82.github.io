Refer to these steps: 

1. Press Windows key + X, select task manager.

2. Click on the ‘Processes’ tab and it will show you show you a list of programs and services running and their corresponding memory usage and disk usage.

3. Check if any of the item listed is consuming more memory or disk usage than normal.

 2. Click on Advanced system settings.
 3. Under Performance, click on settings
 4. Go to Advanced tab
 5. Under Virtual memory, click on Change
 6. Un-check "Automatically manage paging file size for all drives" check box
 7. Select your windows drive (where OS is installed)
 8. Click on Custom size radio button
 9. Set Initial size equivalent to your RAM size in MB
 10. Set Maximum size to twice of your RAM size in MB
 11. Click on Set
 12. Click on OK thrice
 13. Restart your system

You want the chipset driver (you may need additional drivers as well, but that's a different problem).\

Normally, you would get the chipset driver from the motherboard manufacturer, but Giada doesn't seem to provide any support whatsoever.  This board uses the Intel® H61 Express Chipset, so download the driver directly from Intel:  https://downloadcenter.intel.com/download/22153/Intel-Chipset-Device-Software-for-6-Series-Chipset-Based-Desktop-Boards?product=70402

If you have Internet connectivity with this computer, you might want to run the Intel® Driver & Support Assistant to make sure that you get the correct chipset driver and any other Intel driver that may be needed:  https://www.intel.com/content/www/us/en/support/detect.html
