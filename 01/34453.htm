andre da costa say:

Installing to GPT disks requires EFI hardware.
If your system support EFI firmware, Windows 7 will install and boot from an EFI disk.
There are not that many EFI systems available yet.
On MBR hardware GPT disks can only be data disks.
http://www.microsoft.com/whdc/device/storage/GPT_FAQ.mspx
http://technet.microsoft.com/en-us/library/dd799232(WS.10).aspx

Booting is only supported for 64-bit editions on UEFI-based systems. (UEFI – Unified Extensible Firmware Interface).
