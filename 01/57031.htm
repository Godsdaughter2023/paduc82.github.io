frederik long say:
How do I recover a lost administrator password? The password reset disk isn't working.
There are numerous options built into Windows to help you get out of this predicament:
  • Reboot the machine with your Windows Repair CD, then use System Restore to set Windows back by a few days (works only if you changed the password recently).
  • Log on under your spare admin account, then reset the password to your usual account (works only if you set up a spare admin account).
  • Log on as Administrator in Safe Mode, then reset the password to your usual account (the password is usually blank).
  • Use the password hint (works only if you created a hint and if it is good enough to let you guess the password)
  • Check your paper records (works only if you jotted down the password somewhere)
  • Ask your friendly computer dealer to reset the password for you.

heytae say: OMG, I finally, finally found the solution!  It only took 4 days!

After much looking at the registry and all, I noticed that I got an "Music .library-ms is no longer working" error when I tried to open my Music library folder.  That was a huge clue.  So I googled that error message learned that that error occurs when the library file is corrupt or missing.  Mine was there, so I deleted it.  Then right-clicked the "Libraries" folder name and did a "restore default libraries."  That re-created the Music library folder. 

Voila, now Windows Media player and Windows Media Center (which is what I really wanted to play my music in) has the Music Library working as designed.  And all is once again peaceful in my house!  :-)  Hope this might help others who have reported having similar symptoms.  Good luck!

Gotta love it when you persevere and find solutions on your own!

arnymars say:

Switching HDMI Audio Output port in Windows 7 Display & Sound Panels

Most GPUs are now offered with 2 DVI (HDMI) ports. Those AMD GPUs with DVI ports can output sound via HDMI, if a special DVI-to-HDMI Adapter is used. The common scenario of using 2 GPU ports is extending or duplicating a desktop on each Monitor connected. Extending the desktop usually needed to create more convenient work environment. Duplicating desktop is more suited to setting monitors in different rooms & locations.

There is a long standing deficiency in Windows Display and Sound Control Panels: they don't allow to select & change GPU HDMI Audio Output Port. In practice the only way to switch HDMI output to 2-d Monitor for a user is to Disconnect 1-st Monitor in Display CP. This contradicts the idea of extending or duplicating displays. Its not currently possible to redirect HDMI Audio output without reboot to a 2-nd display even after changing it to Main, powering OFF 1-st display, or taking its HDMI cable out, which is absurd.

Please add the following functionality to Windows Display and / or Sound Control Panel:

- Allow a user to freely select and change any time Display & GPU Port that outputs HDMI Audio

- Allow to dynamically switch without reboot HDMI Audio output to the display selected as Main Display as soon as its selected

- Allow to direct HDMI Audio output to both GPU ports in parallel by user's choice, if Duplicate Desktop option is selected

This issue of Microsoft ignoring users needs in switching HDMI Audio Output was discussed many times on AVS and other major forums, while never commented on by MS. Resulting in users dropping interest in using Windows for Home Theater or work tasks requiring Monitor Audio output control.

I'll give you one typical example: my PC is connected to a 24" display, used to work with the PC, and to 38" Display used to watch movies. Its a daily struggle to redirect HDMI Audio output to the GPU port currently being used without Switching displays OFF in Control Panel before and after each change of display currently used. Instead, when 2 monitors are connected to a GPU, Win7 Sound - Playback Devices Panel should show HDMI Output for each GPU port as a separate entry, so that a user can select & fast switch the GPU port which outputs HDMI Audio. 

PLEASE JUST FIX IT!

Its a simple issue to fix but how much more convenience you'll offer your customers, including competitive advantage over 3-d party Media Centers. Or, may be someone can suggest an easy workaround that doesn't involve switching OFF one display or rebooting the PC?
ralf_g say:

Two HDMI output ports does not equate to two sound chips on the card. If it did there would be 2 HDMI audio devices in the Sound panel. In that case switching between the audio devices wouldn't be an issue because you can already do that. Dual video cards would provide that functionality. As it is the HDMI audio signal follows the Primary monitor. When you disable output on one monitor in Windows, similar to physically disconnecting it, the remaining display automatically becomes primary. That's not to say that the circuitry involved in changing the audio connection on any given video card model supports being flipped between ports manually.

Counter example, HDMI audio port switching, in the driver, Nvidia product:

http://www.evga.com/support/faq/afmviewfaq.aspx?topicid=26&faqid=58741

You can skip down to step 10 for the relevent bits.

Driver certification is irrelevent if there is nothing in the standards about manually switching the HDMI audio output. Its up to the hardware vendors what features they include in their products and its up to them as well to make those features functional in whatever OS they intend the product to be used with.
