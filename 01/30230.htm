a. user say:

All was fine until the update from v1709 to 1803.

Have you done a clean install? On supported hardware, a clean install succeeds where upgrades fail.

Windows 10's telemetry provides many benefits, which include stopping further updates when the telemetry concludes that further updates will not succeed, either due to unsupported hardware or a problem with your configuration. Features updates are supported for 18 months (which may have increased to 24 months) after which time you'll start seeing "can't install updates" or "we couldn't finish installing updates" messages. If this continues you'll eventually stop getting quality updates as well, which ain't good.

The only way forward - if you want to stay with Windows 10, and a clean install fails - is to buy a new computer. That gives you new, supported hardware and a fresh copy of Windows 10.

There may be more than just the Windows Partitions on your current hard drive.  There are probably Recovery Partitions, and also special, but small, System Reserve Partitions (which might need to be cloned onto the SSD too if they don't get created during any cloning or imaging.

Have a look at some of these to get a good picture of what you have to do.

How to Upgrade Your Laptop’s Hard Drive to an SSD
http://blog.laptopmag.com/ssd-upgrade-tutorial


Moving Windows 8.1 from HDD to SSD
http://answers.microsoft.com/en-us/windows/forum/windows8_1-windows_install/moving-windows-81-from-hdd-to-ssd/bb57db4a-3f64-48aa-8350-730914301c55

As to the Recovery Partition, this can be copied on to a USB drive should you ever need it and don't want to waste space by storing it on a SSD.

On a working Windows 8.1 machine, see  Control Panel\All Control Panel Items\Recovery\Create a Recovery Drive.

Create a Recovery Drive and use that to boot from in a faulty machine and try accessing the advanced startup options/safe mode.

When you create a Recovery Drive you are given the option (via a check box) to copy across the hidden recovery partition from your machine.  If you choose that option then the size of USB required will be many times larger than if you don't choose the option.
