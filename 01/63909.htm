rmariboe say:

It's a key for MSI. I'd put it back where I found it if I were you:

[HKEY_CLASSES_ROOT\CLSID\{000C101C-0000-0000-C000-000000000046}]
@="Msi install server"
"AppId"="{000C101C-0000-0000-C000-000000000046}"

[HKEY_CLASSES_ROOT\CLSID\{000C101C-0000-0000-C000-000000000046}\ProgId]
@="IMsiServer"


You never need to or should manually edit the registry, unless you are 100% certain what you are doing :)
