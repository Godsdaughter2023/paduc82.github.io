tryx3 say:

Are you referring to the dialog box headed "Do you want to allow the following program to make changes to this computer?"

 

That's is seeking your confirmation that the program be allowed to run & does not necessarily indicate that there is any problem with your admin account, your computer or any application you are trying to install.  It is seeking confirmation for that process rather than forcing you to repeat a login.

 

These confirmations are required even if you are already logged in as an Admin and will occur every time that the operation you are trying to do is one that only an Admin can do.

 

- - - - - - - -

 

If you only have that one account on the computer then, if the account's user profile ever became corrupted, you would have a serious problem.  In addition, using an Admin for routine operations lays you open to particular types of malware attack - see  Why use a standard user account instead of an administrator account?  [you can also see the information in the Help file of the Computer management dialog if you didn't want to use the internet link]

 

Create two new password-protected Admin accounts.  Test the accounts by logging into each of them & create a Password reset disk for each of them.  Demote your existing account to a standard user account for all day-to-day operations and make a Password reset disk for it as well.  You do all this using the Control panel, User accounts dialog.
