lemp say:
You have two issues.  One is assigning the correct port to the printer and the other is fixing whatever is causing the Print Spooler service to fail to run.

Although I can give you directions on how to fix the second problem, it will be far easier for you to uninstall and reinstall the printer (once the Print Spooler is working again) than to create a "virtual USB port" which is what will need to be done unless the computer once had a USB-connected printer installed.

Often, but not always, a stopped Print Spooler is the result of a corrupt print job stuck in the queue or a corrupt printer driver.  However before cleaning things up, on general principles you might want to download, install, update, and run full scans with each of these two free programs:

MalwareBytes AntiMailware
SUPERAntiSpyware

Don't run both scans simultaneously.  Each will take some time, so start it and then go do something else for a while.

Clean up printers

NOTE:  If you have ever had (but removed) a Lexmark or Dell-badged Lexmark printer installed, the "net start spooler" command in Step A below may fail.  In that case, type the following line in the black Command Prompt window and then type the "net start spooler" command again:

         sc config spooler depend= RPCSS

note that there is no space before the =

If still no joy, use the alternative method linked below.

A.  Clean out the pending print jobs

  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Type the following in the black Command Prompt window and press Enter after each line

                           net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

B.  Clean out the old printer drivers and install the latest drivers using the directions in Article One. 


Alternative to Article One's method: http://members.shaw.ca/bsanders/CleanPrinterDrivers.htm 


Once the Print Spooler has been started, go to Start > Printers and Faxes, right click on the icon for the Lanier printer and select Properties and select the Ports tab.  If you have ever had a USB-connected printer installed on this computer, there should be a port named USBnnn.  If there is, check its box and OK your way out and your printer should now print when connected by USB.  If there is no USB port listed, then do the reinstall as explained in Article One.  The driver install package available here http://www.lanier.com/downloads/driverFrame.aspx?path=http://support.aficio.com/bb/html/dr_ut_e/lan_u/model/c30e/c30een.htm
