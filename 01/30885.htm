Das liegt daran, dass sobald Du das Land wechselst (in Deinem Fall von Österreich auf Deutschland), wird Dein Österreichisches Konto auf dem Du die 25€ hast "eingefroren", und ein neues (leeres) Deutsches Konto erstellt.

Daher kannst Du mit dem deutschen Konto kein Geld aus dem Österreichischen Konto auszahlen.

Hintergrund der Geschichte: Sobald Du ein Land wechselst, könnte es ja eine andere Währung in dem neuen Land geben, und durch wechselnde Geldkurse müsste Xbox ständig aktuelle Währungskurse ziehen und umrechnen. Daher wurde einfach gesagt: Beim Länderwechsel bleibt das Geld im alten Land, und wenn Du zurück kommst, hast Du Dein Geld wieder.

Verhindert auch, dass Leute ständig hin und herspringen und sich Krempel aus anderen Ländern kaufen, auf die sie normalerweise keinen Zugriff hätten.

Trim/Edit down a MP4 to reduce file size - Microsoft Community

greg_3188 say:

Trim/Edit down a MP4 to reduce file size

I've got a MP4 video file that's around 1.5GB.  It was shot in 1080HD.  I want to trim it down so I can burn it on a CD that will only handle around 700MBs.  I don't have a problem using the editing tools to get it to about half it's length, but when I go to save the project it ends up being just as large, or larger.  How can I just trim it down without increasing it's file size?

considerate_guy say:

Your choices for reducing the file size are:

  1. Use a more efficient compression algorithm. The two choices in Movie Maker are the default of .mp4 or the optional choice of .wmv. However, .mp4 is probably more efficient and already the default choice in Movie Maker 2012, so changing that won't help. (If you are on Windows Live Movie Maker 2011, updating to have the .mp4 choice would be a good idea.)
  2. Reduce the overall length of your movie. This too is something you've already done. But if you have any choice, pick scenes that have lots of action to remove over ones that are static.
  3. Pick smaller dimensions. Instead of 1920 x 1080, you could choose a smaller resolution.
  4. Pick a lower bitrate. Instead of the default of 24 Mb/s, try a lower rate.

You didn't mention the current length of your movie. 1.5GB already sounds like it was pretty well compressed. Perhaps you have a more efficient codec within the .mp4 file and when Movie Maker makes its edits for length, it ends up compressing it less than before so that you end up with a file size that isn't much different.

Another option would be to put the movie on to a DVD disc instead. A single-sided DVD can handle about 5GB. Or you could also put the movie onto a USB drive (say a 2GB drive).
