Hi,

 

Just managed to solve this problem. It seems that all problems are with drivers.

All you need is to:

1. Login to safe mode:

a. Windows + R

b. msconfig

c. Boot -> Safe mode -> minimal -> OK -> Restart

 

2. Uuninst all drivers

a. Right click on My Computer

b. Devices

c. Uninstall  Graphic and Network drivers - it can be one of them

Try to uninstall all possible drivers

d. Restart PC

 

3. Return from safe mode

a. Windows + R

b. msconfig

c. Boot -> unclick Safe mode -> OK -> Restart

Hi I-Gee,

I also had your problem for the longest time. No matter what I did, I was unable to change anything due to the constant restarts immediately after login.

What I did was after one of the Blue Screens, I pressed the power button in the lower right-hand corner of the login screen and then pressed shut down, not restart. After I left the PC off for about thirty seconds, I turned it back on and logged in. I had to do this several times because sometimes it did not work. However, restarting never worked for me, and I couldn't boot into safe mode, so this was the only option for me.

After a black screen that lasted for about two minutes, eventually the Windows desktop came up. Now immediately, right click on the Windows taskbar and open Task Manager. You have a very limited time before another Blue Screen, so quickly kill any AMD services because their GPU drivers seem to be causing the issue. There are two AMD services that you have to kill, and one Catalyst Control Center to kill also. Be careful though because these might pop back up again or only one of the three might show up first before the rest do. Essentially, keep a constant eye on Task Manager. I had to try this several times because I was to slow, so I recommend to keep trying.

Once you have those services killed, open Device Manager and manually update your AMD GPU to the latest driver by right clicking the item in Device Manager.

Since I changed the GPU driver, I haven't had another Blue Screen. This worked for me after lots of frustration, so I would give it a try.

I hope that it works for you because I know the frustration you're experiencing.

Jack

alex t jr say:

Hello,

While you can convert your copy of XP to a Virtual Machine, the one that is used for XP Mode will be different.  When XP Mode installs, it installs a fresh clean copy of WinXP.  Therefor you would have to go into Virtual Machine and select your now VM of XP to be used seperately from the one created by XP Mode.  The License used for the XP Mode cant be transferred to another VM of XP.

Since it is all on the same machine you are not transferring the license.  At least that is my understanding.  You might find it more useful to just dual boot between XP and Win7 to accomplish the same task, unless you have a specific reason that you want to run XP in a VM.

Try this process to make sure that the Wireles is configured correctly.
