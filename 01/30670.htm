That isn't true that HDMI 1.3 only supports up to 1080i. HDMI 1.0 supported 1080p at 60Hz. a. user say:

Hi,

I suggest you to perform the methods given below before you go for new battery and check if it helps.

Method 1: I suggest you to Run the Power troubleshooter and check the issue.
Refer to these steps to run this troubleshooter:
1. Type Troubleshooting in the search box.
2. Click on Troubleshooting and select the view all option on the left pane.
3. Click on Power troubleshooter and follow the prompts.

Method 2: Reset and restore the power plans to default.
1. Click Windows Key + X, and then click on command prompt (admin).
2. In the command prompt, type powercfg –restoredefaultschemes and hit Enter.
3. Exit command prompt.

Note: This would reset the power plan settings to default. Any previously customized power plans would be removed.

Method 3: Create a new power plan and check.

1. Right click on Start button and select Power Options option from the list.

2. Click on Create a power plan on the left navigation pane.

3. Enter your new power plan name in the Plan name text box and click on Next.

4. Follow the on-screen instructions.

Hope it helps. Revert to us for further assistance with Windows, will be glad to assist you.
