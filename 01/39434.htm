What troubleshooting steps have you tried so far?

 

From what you have described, it appears that the issue is caused by an incompatible or corrupt driver. I would suggest you to try these steps and check:

 

If an incompatible driver is causing the issue, rolling back to the previous driver might resolve the issue. Refer to these steps:

 

  1. Press Windows key + X, select Device manager.
  2. Locate the Keyboard device and right click to select properties.
  3. In the driver tab, click on ‘Roll back driver’ option.

 

Now check if the issue persists. If the issue persists or if the option to roll back is greyed out, then I would suggest you to reinstall the driver and check:

 

Note: Before you uninstall the driver, download the Windows 7 driver for this device from here:

 

http://www.microsoft.com/hardware/en-gb/d/wired-keyboard-600

 

  1. Press Windows key + X, select Device manager.
  2. Expand the “Keyboards” section and locate the Keyboard.
  3. Right click the device and select “Properties”.
  4. Click on the driver tab to check the current driver installed for that device.
  5. Click on the uninstall button to uninstall that driver.

 

Now, restart the computer. Windows will load the device with inbox drivers. Check if the issue persists with the inbox drivers. If yes, install the drivers that you downloaded from the Microsoft website and check. Since the latest driver available is for Windows 7, install the driver in compatibility mode as explained in this article:

 

http://windows.microsoft.com/en-us/windows-8/older-programs-compatible-version-windows
