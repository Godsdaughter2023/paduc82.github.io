3. Open Network & Sharing Center

4. Open the Properties of the network you what your IP to change. In my case it is an EVDO modem. 

5. On the Networking tab double-click TCP/IPv4 which will open a window to edit IP and DNS IP address.

9. Disconnect from that network.

10. Reconnect to that network and if it goes as planned the ISP should force a Dynamic IP.

cmdrkeene say:

If you were using your own administrator account (as you are supposed to), then the built-in one should still be available for you to use.  Just boot into safe mode by rebooting while pressing F8.  Once inside Safe Mode, the built-in administrator account should automatically become available for use (it has no password by default).  Use it to reset the password of your own account, then reboot again to resume normal operation.

If you were already using the built-in administrator account, and you have no other accounts (bad idea!), then you have to erase the computer and re-install Windows.  In that case:

If your computer came with Windows 7, then your computer manufacturer may have shipped it with a built-in recovery mode.  How you access this varies from maker to maker, but usually involves rebooting while pressing a special key like Delete or Escape.
