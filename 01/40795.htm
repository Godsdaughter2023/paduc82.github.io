Having this problem myself, the only solution I found was to force an install of windows 7 usb 3.0 drivers.  I see people all over trying all kinds of crazy things, even trying to get RMA's for hardware....

What I am seeing though is a big potential problem with Windows 8/8.1.

As of Windows 8, it is a fact that Intel would no longer assume responsibility for usb 3.0 drivers for their products. Microsoft took over. Then people start having troubles. Something that seems to be confusing the issue is that a lot of people cannot pinpoint the source of their difficulties. When you start returning merchandise because the drivers are bad you are clearly barking up the wrong tree.

I still haven't seen 1 reply to this issue that actually solved the problem within Windows 8/8.1.

There are also reported problems with usb 3.0 devices disconnecting because of power-saving issues , but at least this problem seems to be addressed rather properly.

I would raise the question, to start a discussion:
