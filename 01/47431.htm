sumit (volunteer moderator) say:

Not an MVP but...

The OS which come preinstalled with the OS are OEM licenses. Microsoft on the other hand sells Retail licenses. The two main differences between OEM and retail licenses are:

  • OEM licenses cannot be transferred to another computer in case if the device is dead.
  • OEM licenses do not offer direct support from MS- The OEM provides the support.

andre da costa say:

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

The $99 Pro Pack license sold through the Microsoft Store is a retail license and includes all the benefits of a full license, such as the ability to transfer and reactivate on another computer. When you upgrade say a Windows 10 Home OEM license to Windows 10 Pro Pack, it becomes a retail full license.
