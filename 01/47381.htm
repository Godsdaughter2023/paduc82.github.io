Hi

You can use the Alt+Tab key combination to switch between windows. The thumbnails of all open windows will be displayed in a rectangular box, in the middle of the screen.

Press and hold Alt and start presssing Tab. Each time you press Tab the next window will open. When you release Alt, the current window will remain on the screen.

The second option is to use Windows Logo Key+Tab to switch between windows. This is the Flip 3D effect. When the target window is at the front of the stack, release the Windows Logo Key.

More Information:

You can select any of the program icons that are pinned to the Taskbar by pressing the Windows Logo Key+#.

( Where # is the number of the target icon, counting from Left to Right, NOT including any Toolbars that are pinned. The numbers are 1-0 for the first 10 Taskbar Icons, if you have more than 10, you can use Windows Logo Key+T to cycle through the icons )

1. Press and hold the Windows Logo Key+#. This will select the target icon and show the Thumbnails for that program.

2. Keep the Windows Logo Key pressed, move the Mouse over the Thumbnails and click the X to close the target Thumbnail.

3. Release the Windows Logo Key.

To make it more comfortable, you can Drag the icon for the target program to a different position on the Taskbar.

onyx is here say:
Error 8007-045d occurs when you try to download an Xbox 360 update
Problem

You see the following error code and message when you try to download an update to your Xbox 360 console:

8007-045d

Can't download the update. To test your connection, open System Settings and select Network Settings, Test Xbox Live Connection.

If the problem persists, go to www.xbox.com/support

This may mean that a storage device was not found during the software update.

Solution

To resolve this problem, try the following:

  1. Turn off your console.
  2. Remove all console storage devices, including any USB flash drives, any external hard drives, or the internal hard drive on the Xbox 360 console. Learn how to remove the hard drive.
  3. Add your primary storage device; for example, the hard drive.
  4. Turn on the console.
  5. Try downloading the update again.

If you keep experiencing the problem, contact Xbox Support.

Give the following a try:

Log out and go to the sign in screen.
Hold down the shift key on your keyboard while clicking the Power button on the screen.

Continue to hold down the shift key while clicking Restart.
Continue to hold down the shift key until the Advanced Recovery Options menu appears.

Click Troubleshoot
Click Advanced options
Click Startup Settings
Click Restart
Press (4) to Enable Safe Mode
Wait while Windows 10 starts in safe mode.
Log in

Press Windows key + X
Click Device Manager
Expand Display Adapter
Right click your Display adapter and click Uninstall
Exit Device Manager

If you have an older Antivirus utility also uninstall it.
Restart

You might need to go to a working computer, download the latest Video Driver for your computer, save it to a thumbdrive. Copy it to the affected computer and install it. If there is no native Windows 10 driver, download the latest Windows 7 or Windows 8.1 video driver and install in compatibility mode.

Learn more:
How to: Install and Update drivers in Windows 10
http://answers.microsoft.com/en-us/insider/wiki...

What is the make and model of your Dell Inspiron?

Scroll down, select Playing audio
Click Run the troubleshooter

When complete, restart your computer then check if audio is working again.

Click Device Manager
Expand Sound, video and game controllers

Right click your audio adapter
Click Properties
Click the Driver tab

Is there an option rollback?

Select the Playback
Right click your Speakers
Click Set as Default Device if it isn't already

Select your speakers
Click Properties
Click the Levels tab
Use the slider to adjust level
