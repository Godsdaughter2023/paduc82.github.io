    steve cochran say:

    If you want the Default to be Num Lock = on prior to sign-in, then the correct method is to set the  number under that registry key (HKEY_USERS\.DEFAULT\Control Panel\Keyboard) "InitialKeyboardIndicators" to a value 2 above the current value (from 2147483648 to 2147483650).

    After login, the Num Lock status is determined by the Num Lock status of the key when the user logged off the last time.  So if you log off with the Num Lock on, then you will log on and the Num Lock will come on after you log on.  The Registry key change is only for prior to log on.  When the user signs off, then Windows sets the value of ""InitialKeyboardIndicators" under HKEY_CURRENT_USER\Control Panel\Keyboard to 2 if the Num Lock is on and 0 if off.  It then reads that value when the user logs in and sets the status accordingly.

    See also here:  Just installed windows 10. I want the number lock to come on automatically on startup. How do I do it?
