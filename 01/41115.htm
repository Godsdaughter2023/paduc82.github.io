Re-install/Re-register Cortana App
Right click on Start > Select PowerShell Admin
Copy/paste the following into the window then Press Enter

Get-AppxPackage -AllUsers Microsoft.Windows.Cortana | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

When the command finishes, reboot and retry Cortana. This registers the app for all Users on the system.

Upgraded to Windows 10 lost a hard drive - Microsoft Community

davyseal say:

Upgraded to Windows 10 lost a hard drive

I have three SATA back drives in my PC, when windows 10 was installed only 2 are showing up in file manager now.   I do not see it listed in disk management either, how do I get it back?

a. user say:

Hi Davyseal, 

Thank you for your interest in Windows 10.

I suggest you to use Diskpart tool to assign a drive letter to the drive and check if it helps.

Follow the steps:

  1. Press Windows key + X.

  2. Click on Command Prompt (Admin). Type the following command and hit Enter: diskpart

  3. At the DISKPART prompt, type: ‘list volume’ and make note of the number of the simple volume whose drive letter you want to assign.

  4. At the DISKPART prompt, type: ‘select volume n’ and select the volume, where n is the volume's number, whose drive letter you want to assign, change, or remove.

  5. At the DISKPART prompt, type one of the following: ‘assign letter=L’ (Where L is the drive letter you want to assign or change).

  6. See if this helps in restoring the drive in File Explorer.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.

nlp22 say:

 I had a 3T disk that wasn't showing in Windows 10. 

I went back to Windows 7

Made a Backup of the drive using Macrium Reflect (the drive worked in Windows 7)

Deleted the data off the drive (which may or may not have been necessary)

Upgraded to Windows 10

I could not see the drive in File Explorer, but could in Disk Management. (it showed up as GPT protective.)

Changed drive from GPT to MBR using 

     http://www.aomeitech.com/aomei-partition-assistant.html

The disk then showed up in Disk manager, but as 2 separate partitions. I used Aoemi Partition Assistant to change drive back to GPT. It now showed up in Disk management with the same 2 partitions. I created one large 2.7T partition using Disk Manager.

Put my backup on the drive. Using Macrium, it creates a virtual drive with each folder visible.

Case solved.
