About MSDN or TechNet Product Keys
They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. The main purpose is for evaluation purposes. The great thing about them, unlike trial Microsoft software, MSDN or TechNet keys don't expire. Because the agreement under which the subscription is provided is a single license, none of the software should be distributed outside of it. Even though its $50,000 worth of licenses, it is for one person only to use and no one else. Unfortunately, regardless of the licensing terms, persons still abuse the program, either giveaway product keys or resell it on auction sites. Microsoft licenses it in good faith that customers won't do so, but I guess human nature wins out.

In your case, what probably happened is, you bought an MSDN licensed key, which carries up to 10 activations unlike full packaged retail licenses which only carry 1 activation. The person who sold it to you probably sold it to 10 other persons. The setting Shilohbob points out for changing 'all' text sizes is accessed by:

Right-click on Desktop > "Display settings"

This setting has been available since at least as far back as W8, but in the past it did not allow 'finer' adjustments in-between, for example, 100% and 125% and 150%.

At this initial 'Display' window, you will only be able to change from the default of "100% (Recommended)", to "125%", or to "150%".

For other text sizes between 100% and 500%, click the text-link "Custom scaling".

Happily, typing a custom setting of, for example, "110", or "120", does in fact make those incremental adjustments to everything, including the Taskbar, and the 'Start menu' I never ever use, as side-by-side screen shots have confirmed for me.

After typing in a number between '100' and '500', click "Apply".

You will then have a new text-link appear slightly above: Click "Sign out now".

When you click 'Sign out now', you will be logged off of Windows 10 as a user, but you will Not have to wait for your computer to power down and restart.

When you are then presented with a new image across your screen, mouse-click anywhere one time to display the button "Sign in"-- Click the button "Sign in".

Note 1 of 2: Regardless of the 'Custom scaling' value you specify, and which is actually being used, you will always see a value of '150%' displayed on the initial 'Display' window under the heading 'Change the size of text, apps, and other items'.

Note 2 of 2: After you make a "Custom scaling" adjustment, if you want to return to the original default condition, which allows you to access the initial 'Display' window's drop-down menu, etc., click the link: "Turn off custom scaling and sign out".

frederik long say: Save the code below as "WindowsUpdate.vbs", then invoke it like so:

cscript.exe //nologo d:\WindowsUpdate.vbs > "%UserProfile%\Desktop\WindowsUpdate.txt"

You will find the complete update history in the desktop file WindowsUpdate.txt.

'--------------------------------------
'Compile a list of all Windows Updates.
'Core code downloaded from
'http://gallery.technet.microsoft.com/scriptcenter/b9ba9b9d-cfc7-4bc9-980b-d914e81b3a84
'19.11.2011 FNL
'--------------------------------------
Set oSession  = CreateObject("Microsoft.Update.Session")
Set oSearcher = oSession.CreateUpdateSearcher
Set cHistory  = oSearcher.QueryHistory(1, oSearcher.GetTotalHistoryCount)
 
For Each oEntry In cHistory
    WScript.Echo "Operation:             " & oEntry.Operation
    WScript.Echo "Result code:           " & oEntry.ResultCode
'    Wscript.Echo "Exception:             " & oEntry.Exception     'Does not work on my Windows 7 installation
    WScript.Echo "Date:                  " & oEntry.Date
    WScript.Echo Format("Title:", oEntry.Title)
    WScript.Echo Format("Description:", oEntry.Description)
'    Wscript.Echo "Unmapped exception:    " & oEntry.UnmappedException    'Does not work on my Windows 7 installation
    WScript.Echo "Client application ID: " & oEntry.ClientApplicationID
    WScript.Echo "Server selection:      " & oEntry.ServerSelection
    WScript.Echo "Service ID:            " & oEntry.ServiceID
    i = 1
    For Each sStep In oEntry.UninstallationSteps
        WScript.Echo i & " -- " & sStep
        i = i + 1
    Next
    WScript.Echo Format("Uninstallation notes:", oEntry.UninstallationNotes)
    WScript.Echo Format("Support URL:", oEntry.SupportUrl)
    WScript.Echo String(23, "=")
Next

'--------------------------------------
'Format the output into aligned columns
'--------------------------------------
Function Format(sItem, sLine)
    sSpace = Space(23)
    sLine = Trim(sLine)
    sResult = Left(sItem & Space(24), 23)
    Do
        sTemp = Left(sLine, 80 - 24)
        p = InStrRev(sTemp, " ")
        if p = 0 then p = 80 - 24
        If Len(sline) <= 80-24 Then p = Len(sline)
        sResult = sResult & RTrim(Left(sLine, p))
        If Len(sLine) > p Then
            sLine = Trim(Mid(sLine, p + 1))
        Else
            Exit Do
        End If
        sResult = sResult & VbCrLf & sSpace
    Loop
    Format = sResult
End Function
