jeremy oestreich say: Hello Jim, I realize you may have already solved this problem... since it is now August 2012.  However, I thought my reply might help others that have this problem.

A very easy fix... 

The problem is in the Windows Easy Transfer for XP that you downloaded.  

If the Windows Easy Transfer program on your old computer does not generate a 6 digit number with a dash in the middle, then it is not the right code for your new Windows 7 computer.  I.e. the code needs to look like this:  123-456.  It does not matter, 32-bit to 64-bit or 32 to 32...so don't worry about it.

Rather than scouring the internet for the correct version, use your new computer to install the correct version onto a USB stick and go about your business. (Regardless of whether you will use a Easy Transfer cable or Network to finish the transfer).

Insert a USB drive into your NEW windows 7 computer, any stick will do. 

Open Windows Easy Transfer on your new Windows 7 computer and click "This is my new computer" and then "I need to install Windows Easy Transfer" as the answer to whether or not Windows Easy transfer is installed on your old computer.

Click the USB option...

You will see an icon that says computer and underneath that an icon offset to the right with the name of your USB stick.  Select your USB stick and hit next.

After installing onto the USB drive, Windows Easy Transfer will ask you to remove the USB and insert it into your old XP computer.

Depending on the age of your XP computer, you may need to restart after the computer installs drivers to use the USB disk.  Your computer might autorun the file, as your windows 7 computer claimed it would, but don't worry if it does not.  Just open the USB stick folder and run the program alias "Windows Easy Transfer" that is conveniently placed in the main window of your USB stick.  Select "This is my old computer". Then select the transfer method that you wish to use.

**** WAIT *****

Now, here is where you will have the option to use the Easy Transfer Cable, your network or an external disk or network disk.

You may need to close and reopen Windows Easy Transfer on your WIndows 7 computer and select the same transfer method that you just picked on your old XP computer.

This method will ensure that the versions of Windows Easy Transfer will talk to each other.

Happy transferring....

vijin v say:

Hi Tech,


This issue might occur due to user account settings.


Let’s try the following methods and check if it helps.


Method 1:

I would suggest you to check the auto login settings.

Press Windows Key + R and type Regedit.exe box.

Scroll to: HKEY_LOCAL_Machine\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\Winlogon

Add a value for: DontDisplayLastUserName

Type: REG_SZ

Value: 0

The value DontDisplayLastUserName determines whether the logon dialog box displays the username of the last user that logged onto the PC.

The value does not exist by default. If it exists, you must set it to 0 or the value of DefaultUser will be wiped and autologon will fail.



Registry Note: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry.



Method 2:

Let’s try to change the local security settings.

a.    Press the Windows + R keys to open the Run dialog, type secpol.msc, and press Enter.

b.    In the left pane, double click/tap on Local Policies to expand it, and click/tap on Security Options to open it.

c.    In the right pane, double click/tap on Interactive logon: Do not display last user name.

d.    To Sign in to Windows 8 with Password Only, Select (dot) Disabled, click/tap OK.

e.    Close the Local Security Policy Window, restart the computer and check if it helps.
