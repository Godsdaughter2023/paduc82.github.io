This add-in lets you create (your choice) ONE of three possible sync relationships (either one way from Outlook to Icloud, one way from icloud to Outlook, or a bi-directional sync between icloud and Outlook, for THESE THREE FOLDERS: the Reminders folder (for icloud and Outlook), the contacts folder (for icloud and Outlook) and ONE calendar folder of your choice on the icloud side, to sync with the Outlook Calender folder.

On the outlook side, the Outlook folders would be the folders in the Outlook folder hierarchy for the CURRENTLY SELECTED default data file.  The idea is that you install office (but don't run Outlook yet), then install icloud for windows (telling it to configure Outlook for icloud), THEN run your mail app in control panel and MANUALLY CONFIGURE YOUR EMAIL APPS AS ALL IMAP ACCOUNTS THERE (you have more configurability using the control panel's mail app then you do IN Outlook).  You don't have to manually configure your icloud email account.  Icloud for windows already did that, if you have one.

Step 7: Turn on UPnP to refresh your NAT table

UPnP is a standard that helps routers communicate effectively. If your router or gateway supports UPnP, it is likely enabled by default.

In some scenarios, you may be able to resolve connectivity issues by refreshing the Network Address Translation (NAT) table. To refresh the NAT table on a router where UPnP is already enabled, you need to disable UPnP, save your changes, and then power cycle your router. After that, you will need to turn UPnP back on and save your changes.

Important Do not use a combination of port forwarding, UPnP, and perimeter network (also known as DMZ) settings. If you previously enabled perimeter network functionality on your router, you must disable the perimeter network before you try this procedure.

Follow these steps to turn UPnP off and on:

   Log in to your router's web-based configuration page to and make sure UPnP is turned on.

   Note Refer to your router's documentation or support website for help turning UPnP on for the first time, or off and then back on.

   Note Make sure that the router is restarted each time you update any of the router settings. If the router does not restart automatically, manually restart the router.

   Turn the UPnP setting off, and then save your changes.

   Restart your console and all your network hardware (your modem and router).

   Log in to your router's web-based configuration page to and make sure UPnP is turned off.

   Turn the UPnP setting back on, and then save your changes again.

   Restart your console and all network hardware (your modem and router)."

Windows Mail is the default mail client included with Windows Vista. Windows Live Mail is the email client bundled with Windows Live Essentials. In order to export messages from Windows Mail or Windows Live Mail to Thunderbird, you must export messages from the Windows application and import them into Thunderbird. malkeleah say:

OK, that's what I thought you meant. You have Computer A (Intel) and you want that Windows installation to be on Computer B (AMD). You can't do this the way you're trying. If you don't want to use Computer A's hard drive in Computer B (I interpret your post as saying you have a new SATA hard drive in Computer B) then you have to image Computer A using enterprise imaging software (I used Acronis True Image Echo with Universal Restore) and then restore that image to Computer B using the Universal Restore feature. Even then it will be tricksy because of the different motherboards, different processors, different hard drives (IDE and SATA). And then of course you'll have an activation problem with Windows unless the version installed on Computer A was a retail version and not OEM. If you're keeping Computer A up and running, you need a second Product Key for Computer B and you'll have to change to that new Product Key using a special utility.

You can't just copy files like you're doing. All in all, unless you have a *very* pressing need to do this, you will be far better off to just do a clean install on Computer B, install your programs from CDs or installers downloaded from the Internet, restore your data from backups. If you feel you *must* do this, then you can purchase imaging software (make sure it has a Universal Restore feature which means you can install on bare metal with disparate hardware). It will take a fair amount of computer skills to do this, too. Or you can have a reputable local professional do this for you. Certainly a BigComputerStore/GeekSquad type of place will not be capable and it won't be cheap, particularly if you *are* going to keep Windows running on Computer A because you'll need to buy a second copy of the operating system. I can't address how much it would cost to have a professional do it because I don't know where you live or the prices in your area.
