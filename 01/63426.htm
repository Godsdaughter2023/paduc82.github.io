jack mvp say:

Hi

It cannot work this way.

Two Network cards (does not matter Wire or Wireless) in the same computer cannot work on the same Network at the same time when using Windows client OS, as oppose to real server like Windows 2008).

You can configure the second Wire card on each computer as a different Network with a Static IP.

Then unbind the sharing from the Wireless and bind only the wire card to the File Sharing (check/uncheck in the TCP/IP properties.

Second network means different subnet.

On Subnet

Classical IP looks like this (example), 192.168.1 .x  The first three groups are also called Subnet , and they identify the specific Network, x can be a number from 0 to 255 and it identifies each unique device on the Network.

An IP of 192.168.2 .x  would be a Subnet of another Network.

Subnet and Firewall.

Most Software Firewall can set a Trusted Zone (  http://www.ezlan.net/faq#trusted   ).  I.e., if your subnet at home is 192.168.1 .x any device that has different subnet will not be allowed to pass. serious-sound-studios say:

This is likely the reason right here. Win 8 uses GPT and not MBR to organize the disk.

There are additional partitions you do not see and those partitions use up some disk space.

GPT has both goo and bad qualities. On the good side, you can have more than 4 primary partitions on the drive. On the bad side, it is much more difficult to create a true ghost or clone of your partition or drive.  The standard tools such as Partition Commander do not yet work with GPT partitioning.  Also, even when installing Win 8 or 8.1 from a DVD a "Recovery" partition will be created on the drive in addition to the boot partition and OS partition resulting in less disk space available for other apps and data.

The Windows 7 Product Key (License) is perpetual, it never expires. You can reuse the key as many times as you wish, as long as the operating system is only installed on one computer at a time.
