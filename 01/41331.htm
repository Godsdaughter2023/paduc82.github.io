  • net start msiserver

malkeleah say: Problems sharing files between computers on a network are generally caused by 1) a misconfigured firewall or overlooked firewall (including a stateful firewall in a VPN); or 2) inadvertently running two firewalls such as the built-in Windows Firewall and a third-party firewall; and/or 3) not having identical user accounts and passwords on all Workgroup machines; 4) trying to create shares where the operating system does not permit it.

In Windows 7, go to Control Panel>All Control Panel Items>Network and Sharing Center. Click on "Change advanced sharing settings". You don't want to use Homegroup unless you have all Windows 7 machines. If you do and you want to use Homegroup, see Windows 7's Help & Support. Otherwise, in the Advanced Sharing:

Turn ON network discovery
Turn ON file and printer sharing
Turn ON sharing in the Public folder sharing section
Turn ON password protected sharing

A. Configure firewalls on all machines to allow the Local Area Network (LAN) traffic as trusted. With Windows Firewall, turning on Windows 7's File and Printer Sharing as above will take care of this for you. If you aren't running a third-party firewall or have an antivirus/security program with its own firewall component, then you're fine.  With third-party firewalls, I usually configure the LAN allowance with an IP range. Ex. would be 192.168.1.0-192.168.1.254. Obviously you would substitute your correct subnet. Refer to any third party security program's Help or user forums for how to properly configure its firewall. Do not run more than one firewall.

B. For ease of organization, put all computers in the same Workgroup. This is done from the System applet in Control Panel, Computer Name tab.

C. Create matching user accounts and passwords on all machines. You do not need to be logged into the same account on all machines and the passwords assigned to each user account can be different; the accounts/passwords just need to exist and match on all machines.
Start>Search box>type: netplwiz [enter]
Click on Continue (or supply an administrator's password) when prompted by UAC

Uncheck the option "Users must enter a user name and password to use this computer". Select a user account to automatically log on by clicking on the desired account to highlight it and then hit OK. Enter the correct password for that user account (if there is one) when prompted. Leave it blank if there is no password (null).

D. If one or more of the computers on your network is XP Pro or Media Center, turn off Simple File Sharing (Folder Options>View tab).

E. Create shares as desired. In Windows 7 I usually share out the user's Desktop and the Public directory.

brad fish say:

I think I have come up with the answer. If you are needing to use Easy Transfer to transfer files and settings to a new Windows 8 computer follow these steps:

 

  1. Complete the initial setup of the new Windows 8 computer and use a LOCAL ACCOUNT to set it up.
  2. Connect the old and new computers with a USB transfer cable or network. If you cannot connect them together us an external hard drive to store the files and settings.
  3. Run Easy Transfer on the new Windows 8 computer
  4. If needed create a copy of Easy Transfer on the thumb drive to use on your old computer (Windows XP)
  5. Run Easy Transfer on the old computer and follow the on screen instructions
  6. After Easy Transfer has selected the possible files and settings to transfer choose the account(s) you wish to transfer and then use the advanced settings to choose the LOCAL ACCOUNT that you wish to transfer the old computer's account to.
  7. After the transfer is complete reboot the new computer and login with the LOCAL ACCOUNT
  8. Open the Settings Charm and select Change PC Settings.
  9. Select Users and select Change to Microsoft Account and enter or create the Microsoft
  10. Logout of your LOCAL ACCOUNT
  11. Login with your Microsoft Account

You are now using your Microsoft Account which is linked with your LOCAL ACCOUNT and should have all of your files and settings that you transferred from your old computer. You will also need to install any additional software that you had on the old computer that is not already on the new computer.

 

I hope this is helpful to someone.
