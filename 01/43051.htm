davem121 say:
Hi catfishscatfish, yes this seems to be a setting that has been forgotten in Windows 10 . . .

Click your Start Button, type regedit and hit Enter to open the Registry Editor

In the left pane, navigate to:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkList\Profiles

Expand the Profiles Key (folder)

That will contain sub-keys with different GUID

Click each Sub-key

On the right, check the ProfileName value

When you find the sub-key with the correct ProfileName value

At the top you will see 'Category' - double click that

Change its value from 0 to 1

(Note 0 = Public, 1 = Private)

Click OK

Close the Registry Editor and restart your PC

Best way to do this is via powershell.

Get-NetAdapter

Set-NetConnectionProfile -InterfaceAlias Ethernet -NetworkCategory Private

That's it.

To change it now, you either have to delete the network profile and let it recreate - make sure you choose the right setting! Or you can use PowerShell:

Type: powershell
Right click powershell
Click Run as administrator

Get-NetConnectionProfile
