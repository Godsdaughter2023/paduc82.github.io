1. Type cmd in the search bar to open command prompt

2. Right click and select "run as administrator"

3. Type the following command and press enter : cd /d %LocalAppData%\Microsoft\Windows\

Note: Do not close the Command prompt

4. Press Esc + Shift + Ctrl key to open task manager

5. Right click on "windows explorer" under processes tab and select "end task".

6. Open the Command prompt window and type the following command and press enter:

copy appsFolder.itemdata-ms c:\backup\*.*

Note: Replace "c:\backup" with the actual path on your PC.

7. Open task manager by following step 4 and click on "file"

8. Select "run new task" and type explorer.exe and press enter

1. Run command prompt as administrator

2. Press Esc + Shift + Ctrl key to open task manager

copy /y c:\backup\appsFolder.itemdata-ms "%LocalAppData%\Microsoft\Windows\appsFolder.itemdata-ms"

5. Open task manager, click "file" and select "Run new task"

6. Type explorer.exe and press enter 

You can also sync your system with another system through the same Microsoft account so that the next time you decide to re-install Windows 10, once you login through the same Microsoft account, the same screen layout and settings from the other system will be sync to your new system.

This policy applies to the Free Windows 10 Upgrade because it is activated by digital entitlement based on the mobo's unique serial number.  Same make and model is not good enough for transferring the free upgrade.

If you purchased Windows 10 then it would be transferable.  If Windows 10 came preinstalled on the computer then Windows 10 would activate on a repair replacement mobo because OEM (preinstalled) Windows 10 is product key activated.  You might need assistance from Microsoft Support to complete activation, but you would be entitled to do it.
