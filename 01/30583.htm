Windows 10 Flashing Startup Screen Problem/Solution - Microsoft Community

soudal-qasem say:

Windows 10 Flashing Startup Screen Problem/Solution

Okay. First off, I would like to address I problem I have faced when first downloading/installing Windows 10 on one of my PC's. This problem involved my computer screen flashing constantly while loading the login screen. I'd like to point out that this is not the same as the problem faced by several others which involved the DESKTOP screen flashing because you would be able to actually get to the desktop, which in my case was impossible unless you started Windows in safe mode [This can be achieved by pressing F8 and holding it right after your boot screen three times consecutively. In other words, you need to start up your PC. Press and hold F8 until you hear a beeping noise. Wait for the Windows Logo to pop up. Turn of your PC. Then repeat another 2 times whilst stopping at your third attempt. As for why it's made this way, my guess is as good as yours].

As you can imagine, it was quite the frightening experience.

However, after a couple of days of endless pondering and web-surfing, I managed to identify the problem, and hence solve it. So I thought I would share it with my fellow Windows 10 users! Yay!

Ahem. Now, ladies & gents, it turns out that the Windows 10 login screen does not support 3rd party software. And by that I mean facial recognition software, voice recognition software, ...etc. Now, I am not sure if this applies to all forms of such software, but I can with full certainty say that it does not support versions of software of the likes of KeyLemon TM that were made for Windows 7 (or 8 I believe).

Now, for the solution part, I did the following:

1)Started my PC in Safe Mode

2)Opened up the software's directory (e.g C:\Program Files\KeyLemon)

3)Moved all the files with the .dll file extension from there to a New Folder on my desktop

4)Restarted my PC

5)Profit

Now, there's always the chance you might not have faced that problem, but I find that highly doubtful since the Windows 10 login screen is fairly new. So if you still wanted to use a facial recognition software, you might need to reinstall a Windows 10 compatible version/update if one does get released.

I hoped this helped the poor souls who are facing this problem and have yet to find a solution out there.
