As long as you have a password set, it should require you to provide it after a full shutdown or restart.

You can check if you have a password set by going to Settings > Accounts > Sign-In Options.  There you can set a password, and at the top of the screen you can make sure it's set to require a password after you've been away (such as when the computer goes to sleep mode).

Finally, you can set it to lock after X minutes, if you want.  This can be useful so it requires a password anytime you walk away for more than a set time.  To do this, go to Settings > Personalization > Lock Screen.  Scroll down to find and click the Screen saver settings link.  Inside there, check the box that says "on resume, display logon screen", and set the wait timer however you want.  You don't have to choose a screensaver, the timer will still lock the screen after the minutes you choose, even if the screensaver says "none".
