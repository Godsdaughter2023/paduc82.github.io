rolando vil say:

Hi,

Unknown devices usually show up in the Device Manager when Windows is unable to identify a piece of hardware and attempts to provide a driver for it. An unknown device isn’t just unknown — it’s not functioning until you install the right driver. Most of the time Windows is able to identify and download drivers automatically. In cases where it is unable to do so, you'll have to identify and look for the driver yourself.

Follow these steps to locate the Unknown Device:
1. Press Windows Key + X and select Device Manager.
2. You’ll find unknown devices and other non-functioning devices under Other devices. Each device with a problem has a little yellow exclamation mark over its icon.

To check the Hardware ID of a device, follow these steps:
1. Open the Device Manager.
2. In the Device Manager, right-click the device, and choose Properties in the popup menu.
3. Tap the Details tab.
4. Select the Hardware IDs in the drop-down list.
5. Search the web for the Hardware ID using your browser. You should be able to get the name of the piece of hardware associated with the unknown device.
6. Get the driver for the device and install it normally.
