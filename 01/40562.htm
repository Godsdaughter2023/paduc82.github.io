How To Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Enter http://192.168.1.1 into the address bar. (Or click the link)
  3. Enter “admin” for both the username and password.
  4. Select Advanced Setting from the menu on the left.

  6. From this point forward there are no screen shots or reference material to go off of for this device.

  1. Open a web browser.
  2. Enter http://192.168.1.1 into the address bar. (Or click the link)

  4. Select Advanced Setting from the menu on the left.

  8. Click the Save button at the bottom of the screen.
