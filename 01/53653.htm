There's something named ISUSPM(Install Shield Update Service Scheduler) which is causing the problem and runs on startup. If you are facing problem then after restarting your PC if you swipe three fingers up(Windows 10) or press Win + Tab, it will show this application open though it won't be shown open in task bar.

So to solve the problem just end this this running from Task Manager(Ctrl + Shift + Esc), but this will start again after restarting. So to solve the problem, before ending this in Task Manager, right click it and go to file location, it will take you to the location it is saved, then minimize the open window and end it running from task manager and then delete it. Deleting it doesn't cause any problem to your PC as I observed but to be safe you can first just delete it simply to send it to recycle bin and then use your PC normally and if no problem is observed, you can delete it permanently.
