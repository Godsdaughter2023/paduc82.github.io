Description of Windows Easy Transfer for Windows Vista
http://support.microsoft.com/kb/928635/en-us

You are pretending that your current computer is your "old" computer, and that it is also the "new" computer as far as Windows Easy Transfer wizard is concerned.  Save your created file to C: drive.

Run the Windows Easy Transfer from Anita and transfer your settings that you created earlier into the new account.

linney say:

You are not really deleting your accounts you are recreating them in case  a corrupted profile is the cause of your problems.  The problem of deleting Anita from Anita.TheII or vice versa, is maybe cross corruption between the two similarly named accounts?  Besides containing your user files and data, accounts also include hidden settings and files identifying that particular account to Windows.  Wherever those hidden files end up so does that user account including any corruption within them.  When you copy a user account to a new user because of a corruption problem, those files (see links) are NOT to be copied.

What I outlined above is  possibly one way out of the mess.  You can look at these alternatives.

Fix a corrupted user profile
http://windows.microsoft.com/en-US/Windows7/Fix-a-corrupted-user-profile

Fix a corrupted user profile
http://windowshelp.microsoft.com/Windows/en-US/help/769495bf-035c-4764-a538-c9b05c22001e1033.mspx

811151 - How to Copy User Data to a New User Profile
http://support.microsoft.com/?kbid=811151

What is the "Standard" Account called?  The terms you use are likely to be misinterpreted by some.  There are two "types" of User Accounts most commonly used in  Vista, one is an Administrator user account the other is a "Standard" user account.  If there are no problems with any Standard account then that can be left alone.
