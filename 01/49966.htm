a. user say:

Has your system ever worked properly when using Sleep mode?

How do you think it is supposed to work - what do you think you need to do to get your system to wake up?

Sleep mode is inherently dangerous since if you have something like a document or spreadsheet open and enter sleep mode, those files are still "open" and the data in them is still in RAM so if your system loses power while in sleep mode, the files run the risk of corruption (not being closed properly) or you could suffer data loss.  Sleep mode was not designed to be used for long periods of time.

The fans are supposed to still be running since in sleep mode, there still has to be power to the motherboard to keep the RAM and other things "alive".  For me, the only thing that actually gets power cut off is the hard drive - everything else still has power so you are not doing yourself much power saving in sleep mode - that is not why it was invented.

If you think you want to do something to save power, you might think about using Hibernate instead since that is why it was invented.

Unless you want to waste your time when solving problems, approach with extreme skepticism any replies or suggestions that begin with the word "try".

Running sfc /scannow will not help you - it is a waste of time and a wild goose chase and when some "expert" suggest that, I interpret it to mean "I have no idea what the problem is or how to solve it".

This is also a terrible idea:

Note: Refer "How to reset the computer to start normally after clean boot troubleshooting" to reset the computer to start as normal after troubleshooting.

That is the absolute worst possible configuration for your system (unless you want it to run slow).

If you do that, your system will load every single driver and startup item on your system and you will pay for that with poor performance.  Most systems are not set up that way since there are many drivers and start up items that you probably don't want to be loading every time your system starts - unless you want it to run like yak squeeze.

In order for your system to come out of sleep mode by using the mouse or keyboard you have to configure your system to allow those devices (keyboard and mouse) to allow bring the system out of sleep mode and you do that from the Power Management settings in Control Panel or Device Manager.

I prefer Device Manager, but somehow you need to get to the Properties, Power Management tabs of those devices and enable them to bring the system out of sleep mode.

Here are some screen shots showing keyboard and mouse Power Management set up properly.  After yo make any adjustments, restart your system and a mouse click or something like space bar should bring your system out of sleep mode - if it is not buggered up with all these other "suggestions":

ipconfig /flushdns
ipconfig /release
ipconfig /renew
exit
