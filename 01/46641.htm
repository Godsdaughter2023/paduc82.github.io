Change Wireless Mode

Search for updates
Press the Windows + R logo key, type the following command line in the Open box, and then select OK to open Settings:

ms configuration:

In Settings, choose Update and Security> Windows Update and then select Check for updates. Install the available updates and restart your PC if necessary.
Restart your device
If you did not have an update that required a restart, it is a good idea to restart your device to ensure that all pending updates are installed.

Update the drivers
While you may have the latest drivers, if you still have problems with the Start menu, you can try to fix it by manually updating your graphics or audio drivers. See Update drivers in Windows 10 for more information.

Create a new local administrator account
If you still can not solve your problem with the Start or Cortana menu, try creating a new local administrator account.

If you are logged in with your Microsoft account, first remove the link to that account by doing the following (otherwise, see "Create the new administrator account"):

Press the Windows + R logo key, type the following command line in the Open box, and then select OK to open Settings:

ms configuration:

Under Settings, select Accounts> Sign in with a local account instead.
Enter your Microsoft account password and select Next.
Choose a new account name, password and password hint, and then select End and log out.
Create the new administrator account:

Press the Windows + R logo key, type the following command line in the Open box, and then select OK to open Settings:

ms configuration:

In Settings, select Accounts> Family and other people (or Other people, if you are using Windows 10 Enterprise).
Under Other people, select Add another person to this PC.
In Windows 10 Home and Windows 10 Professional, provide a name for the user and a password, and then select Next. In Windows 10 Enterprise, select I do not have this person's login information and then select Add a user without a Microsoft account.
Establish a username, password and password suggestion. Then select Next> Finish.
Next, make the new account an administrator account.

In Family and other people (or Other people, if you are using Windows 10 Enterprise), choose the account you created and then select Change account type.
In Account Type, select Administrator> OK.

Log out of your account and then log in to your new account. If everything is fixed, move your personal data and files to the new account.

To move your personal files from your old account to your new account, try opening the File Explorer and look in the Users folder, where you should see the folders of both accounts. Then, you can copy and paste your files from the previous account's folder to similar locations in the new account's folder. Or if you prefer, you can upload your files from your old account to OneDrive, and then log in to the new account and download the OneDrive files.

If the problem has not yet been fixed, try deleting the previous administrator account:

Under Other users, select the previous administrator account, then Delete> Delete account and data.
After deleting the previous account, restart your device and log in with the new account again.
If you were using a Microsoft account to sign in earlier, associate the Microsoft account with the new administrator account. Under Settings> Accounts, select Sign in with a Microsoft account and enter your account information.

Reset your PC
If none of the above options work, try restarting your PC with Windows 10 recovery options.
