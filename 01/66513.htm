sdct989 say:

For anyone that comes along later with this same problem, what finally worked for me was changing the Windows Task settings. Each time I got the 12/30/1899 problem, I would try tweaking another setting on the task, if that didn't work I would reset what I changed and then try tweaking something else.

The change that finally worked for me was changing the "Configure for" setting on the task. The Task Scheduler can be found by searching for tasks in the Windows Menu, then navigating Task Scheduler Library > Microsoft > Windows > WindowsBackup. Once there, double-click on the "AutomaticBackup" task. At the bottom of the "General" tab change the "Configure for" option from the default value of "Windows Vista, Windows Server 2008" to "Windows 7, Windows Server 2008 R2". Obviously, this will only work if you are, in fact, on Windows 7 or Windows Server 2008 R2.

After changing this option, I have been running for weeks and haven't had it reset to 12/30/1899 again. Previously, it had reverted almost every day.

Move most of your messages out of the Inbox and then create new Outbox and Sent Items folders after you move any messages you wish to save to a local folder you create.

In WinXP, Win2K & Win2K3, the OE user files (DBX and WAB) are by default marked as hidden. To view these files in Windows Explorer, you must enable Show Hidden Files and Folders under Start | Control Panel | Folder Options Icon | View, or in Windows Explorer | Tools | Folder Options | View.

With OE closed, find the DBX files for the Outbox & Sent Items and delete them.  New ones will be created automatically when you open OE.

After you are done, follow up by compacting your folders manually while working *offline* and do it often.

General precautions for Outlook Express:

Do not archive mail in the Inbox or Sent Items. Create your own user defined folders and move the messages you wish to save to them. Empty Deleted Items folder daily. Although dbx files have a theoretical capacity of 2GB, I recommend about a 300MB max for less chance of corruption.

Compact often as specified above.
