kaushik chakrabarti say:

I know it is a old thread but some people still face this problem. So I am going to clear some misconception.

I re-installed my drivers directly from the Samsung website, and now Windows 10 regards my monitor as "other".

The problem is not the monitor or Windows 10 but it the graphics driver. You don't need to install monitor (Samsung or any other brand) driver but you must install GPU (Nvidia/AMD/Intel HD graphics) driver based on what you have. Because it is the GPU which is responsible for processing any visual content shown on display. Without it you can't run graphic intensive programs, play videos or run games and your monitor is recognized as "generic" or "other".

I then proceeded to see if it's a problem with my monitor itself, it claims its running normally. It recognizes that my resolution is at 1024x768, and recommends the optimal mode be set to 1600x900. Windows10 will not allow me to do that. 

Without the graphics driver older Windows show the resolution at 800x600 and for newer Windows (8 and later) it is 1024x768. After you install the driver it will support almost any resolution more than 1024x768. Find out what hardware you have and download drivers for it. If your hardware vendor don't provide Windows 10 drivers then you probably have to stick with older version of Windows or need to buy new graphics hardware.

I have Samsung SMB2030 (1600x900) and AMD Radeon graphics and it worked like a charm.
