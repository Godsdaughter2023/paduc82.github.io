This problem is caused by the discrepancy between the ADMX and ADML file versions which must match.

The admx file uses variables like the above WU_Supported*, the language file ADML then holds the appropriate texts. If you are using other language then english, the ADML language file of the appropriate language must match too. Example:

C:\Windows\PolicyDefinitions\Windows.admx vs C:\Windows\PolicyDefinitions\en-US\Windows.adml

or

C:\Windows\PolicyDefinitions\Windows.admx vs C:\Windows\PolicyDefinitions\de\Windows.adml
