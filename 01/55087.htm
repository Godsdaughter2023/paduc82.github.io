a. user say:

Hello,

I suggest you to try the below steps and check if this helps.

Initially, start your Computer in safe mode from sign-in screen. Refer the below link:

http://windows.microsoft.com/en-in/windows-10/start-your-pc-in-safe-mode

Once you login with safe mode, enable the built in admin account.

  • Right click on Start and click on Command Prompt(admin).
  • On the run command type the below command to enable built in admin account.

              net user administrator /active: yes

Now restart your system and login with built in admin account. From here you can change your earlier account password or you may create a new account and check if this helps.

To change password from admin account:

  • Right click on Start and click on Control Panel.
  • Select User account.
  • Click on Manage another account.
  • Select your local account and click on Change the password and follow the instructions.  

If this doesn't work, try to create a new account:

1.   Go to Settings.

2.   Choose Accounts and then select Family and other users.

3.   Select add someone else on this PC.

4.   Enter a user name and hit next.

5.   Click on Finish.

6.   Sign out from the current Account and Log into the new account.

 

Note: To disable built in admin account, run the below command and hit enter.

net user administrator /active: no

Hope this helps. Kindly get back to us with the status of the issue to assist you further.
