I don't know if I can explain this so that it makes sense
but maybe you will get the idea.

If the expected final result is three separate video clips
you mist remove the unwanted portions from the project
before saving.

Split the clip at approximately the 1/3 position...remove
the remaining part. now with only 1/3 in the project...save.

Begin again with the full clip and split at the 1/3 and 2/3
position and remove the first and last portion leaving only
the center and save.

On final time with the last portion and your result should
be three clips.

FWIW...the freeware Freemake Video Converter has a
split (Cut) option that is easy to use:

Freemake - How to cut video quickly
http://www.freemake.com/how_to/how_to_cut_video_quickly

Here again...you have to remove the unwanted portion
and then save.

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Freemake Free Video Converter (freeware)
http://www.freemake.com/free_video_converter/
(Windows XP / Vista / 7 / 8)
(***Choose the Custom installation and carefully
read *all* the install screens...there may be some
items you want to uncheck***)

If a green Install button appears when you launch
Freemake...you can ignore it and it should disappear
when you import files.

To download the full version instead of using the
web installer...see the following:

Freemake Apps - Offline version
http://www.freemake.com/offline/offline_all/
(If you want to install Freemake freeware
to your PC without the Internet connection,
download these files)

When you have Freemake installed...open Freemake
and drag/drop your video files on the main screen.

    mecalith say:

    It sounds like the space for the program tabs on your Taskbar is too narrow.

    Try reducing the width of your toolbars:

    Uncheck: [ ] Lock the Taskbar

    (<Taskbar> right click > Properties > Taskbar: Taskbar appearance)

    Adjust the width of the toolbars:

    1. Move the mouse pointer over a toolbar handle (column of dots).

    2. When the mouse pointer changes to a double arrow, press the mouse button.

    3. Drag the mouse to reduce the width of the toolbar.

    (for example: drag the handle of the quick launch toolbar to the left)

    The program tabs should display side by side on the taskbar.
