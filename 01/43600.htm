Trying to Add a Logitech speaker to my computer - Microsoft Community

jlobahamas say:

Trying to Add a Logitech speaker to my computer

I am trying to add a speaker to my computer but there is no type of volume
Now it is saying there is no mixer devices available
How can i get my Volume on

ralf_g say:

Check in Control Panel for the "Sound" icon. Clicking that should give you configuration settings for your sound devices. The mixer and sound controls install automatically when Windows detects a supported sound card. If you're not seeing them or the settings in Sound are greyed out, chances are your sound card did not get its drivers installed properly or the drivers aren't supported in Windows 7. If yours is an upgrade installation of Windows 7 check that your sound card or on-board audio is compatible with Windows 7. The old drivers might not be and could interfere with the automatic installation of working drivers. Identify what type of sound device is in the computer and go to the manufacturer's web site to look for Windows 7 drivers. If there are no Windows 7 drivers you might have to to try Vista drivers or even Windows XP drivers. Look for the sound device entry in Device Manager under "Sound, Video and Game Controllers". If the device is already listed there try clicking Update driver from its properties page. If there is no sound device listed or the device is there but marked disabled then install the driver that you previously downloaded. Try updating the driver again. If still no luck click uninstall the driver and then right-click "Scan for Hardware changes" at the top of the devices list. You might have to reboot at this point to finalize the driver installation and have the volume control appear in the notification area.  

 If your sound card is supposed to be a high definition sound device then there should also be an MS driver called "High Definition Audio Controller" showing up in the System Devices section. That driver typically installs automatically after the sound card itself gets installed. 

Some drivers install their own audio controls that you can use to configure your speakers, microphones etc.. For example Realtek puts a brown coloured speaker icon in the system tray. The main controls are duplicated in the Windows mixer but occasionally there is a mis-match of the settings. That's something to look for if you still have problems after the sound card driver has been installed.

malkeleah say: You inadvertently installed drivers for your network adapter from Windows Update. Set your Windows Update to download and let you know when updates are ready instead of automatically downloading/installing. Then look at the updates and don't select any for drivers.

Rollback the drivers or, if that isn't successful, reinstall the correct network adapter drivers now.

Never get drivers from Windows Update. Get them from:

1. The device mftr.'s website; OR
2. The motherboard mftr.'s website if hardware is onboard; OR
3. The OEM's website for your specific machine if you have an OEM computer (HP, Dell, Sony, etc.).

Read the installation instructions on the website where you get the drivers.

To find out what hardware is in your computer:

1. Read any documentation you got when you bought the computer.
2. If the computer is OEM, go to the OEM's website for your specific model machine and look at the specs (you'll be there to get the drivers anyway)
3. Download, install and run a free system inventory program like Belarc Advisor or System Information for Windows.

http://www.belarc.com/free_download.html - Belarc Advisor
http://www.gtopala.com/ - System Information for Windows

If you have installed drivers from Windows Update, you can roll them back:

http://pcsupport.about.com/od/windows7/ht/roll-back-driver-windows-7.htm
