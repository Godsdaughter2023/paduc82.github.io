What is a protection fault? - Microsoft Community

bristolsue say:

What is a protection fault?

Since big W 10 upgrade I can't now use my printer properties and keep getting a message "function address 0xdf3c8e08 caused a protection fault (exception code 0xc0000005). Some or all property pages may not be displayed. "  This message is under the title C:/WINDOWS /system32/rundll 32.exe.   z This is all gibberish to me !  Any help out there please? 

***Post moved by the moderator to the appropriate forum category.***

a. user say:

TL;dr version: You need to install the Windows 10 driver for your printer, if there is one.

0xc ... 5 is a broad error that means simply that the program ('app' in modern parlance) can't continue because it's trying to do something potentially damaging to your computer.

The component of the program ('function') that is causing the problem is located in RAM at memory address 0xdf3c8e08, which is useful to know if you want to send it a Christmas card, I suppose.

The particular problem that the naughty function is causing is a 'protection fault' which tells you that the function is attempting to do something in a memory location where it's not supposed to be doing that, like smoking in the bathroom.

If the manufacturer of your printer is supporting that model on Windows 10, they will either have a Windows 10 driver for you to install or instructions on which earlier version of the driver to use. You'll find the information on the manufacturer's website.

OTOH, if the manufacturer has decided not to support your printer on Windows 10 you are SOL ('sadly out of luck') because it will eventually stop working.
