Hello,

To back up and restore notes from Sticky Notes app in Windows 10, we'll have to specify which version of Windows are you using. Here's a simple guide on how to check the build and version of your PC:

  1. Press the Windows button + R.
  2. Type winver.

Once we're able to determine your PC's build and version, follow the steps below to locate the storage files of Sticky Notes and create a backup:

  1. Open File Explorer, click the View tab.
  2. Select Show/hide option then check the Hidden items box.
  3. To find the Sticky Notes storage folder, copy and paste the location below into your File Explorer's address bar. username is the name of your actual user account.
  • For build 1607 or later: C:\Users\username\AppData\Local\Package\Microsoft.MicrosoftStickyNotes_8wekyb3d8bbwe\

From there, you can copy everything and paste it to the backup folder.

  • For anything lower than build 1607: C:\Users\username\AppData\Roaming\Microsoft\Sticky Notes\

You will only see a single file: StickyNotes.snt. You can copy and paste it to the backup folder.

To restore the files to Sticky Notes, make sure the Sticky Notes app is closed. Locate the same folder we pointed you to above and copy all your backed up files there, overwriting whatever’s there currently. When you launch Sticky Notes again, the notes you backed up previously should pop up.

Please be reminded that notes in the desktop and Store app versions of Sticky Notes aren’t compatible. You won’t be able to copy notes from a PC running Windows 7 to a PC running Windows 10 Anniversary Update.

This can be caused if test-sign mode is started. 

To disable test-signing, use the following BCDEdit command using the command prompt:


Bcdedit.exe -set TESTSIGNING OFF


Note: After you change the TESTSIGNING boot configuration option, restart the computer for the change to take effect.

To enable test-signing, use the following BCDEdit command:

Bcdedit.exe -set TESTSIGNING ON

Once started, it may take 10 minutes or so for the appraiser to complete. Once complete, re-open the Get Windows 10 app and it will have the updated compatibility information.
