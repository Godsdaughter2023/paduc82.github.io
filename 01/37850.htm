ralf_g say:

Things to double check:

WPA-PSK(personal) versus WPA(enterprise/radius) - the first is more typical for non-business networks, using a passkey for access.

TKIP encryption versus AES encryption - set for the type being used by the school

128 bit versus 56 bit encryption - probably the higher is being used but you never know

After having been added to the school domain you now have 2 logon levels to contend with, the first being the basic WiFi (hardware) logon to access the physical network. The second logon accesses the school domain and shares on the network. These seem related but are independant of each other. It would be similar to a home WiFi network where the computer WiFi adapter logs in only to the router. Individual shared resources (folders, printers etc) can be made available to anyone who connects to the LAN (router) or can be set to be accessed only if you have a correct User password (like your domain logon). 

The best person for getting you connected should be the school's network administrator. If other Windows 7 users are connecting successfully and all of your basic WiFi settings are correct then you should look for something in your 3rd party software being the source of your problem. Security suites and non-MS firewalls would be the first suspects.

Hi,

References to Vista also apply to Windows 7.

Updating drivers, BIOS, and control software (if any) is a good idea.

This is my generic how to for proper driver updates :

For Drivers check System Maker as fallbacks and Device Maker's which are the most current.
Control Panel - Device Manager - Display Adapter - write down the make and complete model
of your video adapter - double click - Driver's tab - write down the version info. Now click UPdate
Driver (this may not do anything as MS is far behind certifying drivers) - then Right Click -
Uninstall - REBOOT this will refresh the driver stack.

Repeat that for Network - Network Card (NIC), Wifi, Sound, Mouse and Keyboard if 3rd party
with their own software and drivers and any other major device drivers you have.

Now go to System Maker's site (Dell, HP, Toshiba as examples) (as rollback) and then Device
Maker's site (Realtek, Intel, Nvidia, ATI as examples) and get their latest versions. (Look for
BIOS, Chipset and software updates at System Maker's site while there.)

Download - SAVE - go to where you put them - Right Click - RUN AD ADMIN - REBOOT after each
installation.

Always check in Device Manager - Drivers tab to be sure the version you are installing actually
shows up. This is because some drivers rollback before the latest is installed (sound drivers
particularly do this) so install a driver - reboot - check to be sure it is installed and repeat as
needed.

Repeat at Device Makers - BTW at Device Makers DO NOT RUN THEIR SCANNER - check manually
by model.

Manually look at manufacturer's sites for drivers - and Device Maker's sites.
http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm

Installing and updating drivers in 7 (updating drivers manually using the methods above is
preferred to ensure the latest drivers from System maker and Device makers are found)
http://www.sevenforums.com/tutorials/43216-installing-updating-drivers-7-a.html

If you update drivers manually then it is a good idea to disable Driver Installations in Windows
Updates, this leaves Windows Updates ON however it will not install drivers which will usually be
older and cause issues. If Updates suggests a new driver then HIDE it (Right Click on it) and
then go look for new ones manually if you wish.

phillfri say:

Right click on the desktop and select 'Display Settings'.

Scroll down to the bottom of the window that pops up and select 'Advanced Display Settings".

Scroll down to the bottom of the window that pops up and select 'Advanced sizing of text and other items'.

Select the underlined text 'set a custom scaling level'.

I use 150% on my 8" Asus Vivotab Note 8.

The slider in the screen display settings only goes up to 125% by default. You have to do the above to get beyond that limit.
