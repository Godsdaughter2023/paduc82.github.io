Hi Richard,

We understand that you're having issues with the default file app association on your computer. To assist you with your concern, you may refer to the steps below on how to change the default setting of your application:

  1. In Search Windows textbox, search for choose a default app for each type of file and set .PDF and any other file extensions to point to your choice for default programs. Do not use Adobe Reader to set the association.
  2. Navigate to %systemroot%\system32 and locate the OEMDefaultAssociations.xml file. Make a backup of it just in case.
  3. Open the OEMDefaultAssociations.xml file and keep it open using Notepad.exe.
  4. Open PowerShell or CMD and run it as Administrator.
  5. Enter the following: Dism /Online /Export-DefaultAppAssociations:C:\AppAssoc.xml
  6. Open C:\AppAssoc.xml with Notepad.exe
  7. Copy the entire contents of AppAssoc.xml and replace the entire content of OEMDefaultAssociations.xml.
  8. Save OEMDefaultAssociations.xml.
  9. Delete C:\AppAssoc.xml (optional).

After doing the steps, you may now change the default file app association for opening .pdf.

Do update us with the outcome and if you need further assistance.

Hi Mansoor,

Follow the Methods below and check if that helps.

Method 1: I suggest you to run a SFC scan and check if it helps.

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.


Follow the below steps:

1. Right click on the Start button, click on Command Prompt (Admin).
2. Type “sfc /scannow” without the quotes and hit Enter.
3. After the scan, restart the computer and check for the issue.

Method 2: Unregister and re-register the MSWB7.dll and then check if you come across the issue.


Step 1: Steps to un-register MSWB7.dll

1. Press Windows + R from the keyboard to open Run command.
2. Type regsvr32 /u MSWB7.dll

3. Click on OK.
4. Close the Run command window.

Step 2: Steps to re-register MSWB7.dll

1. Press Windows + R from the keyboard to open Run command.
2. Type regsvr32 MSWB7.dll

3. Click on OK.
4. Close the Run command window and check if the issue persists.
