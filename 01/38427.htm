c. Under Storage Volumes, you should notice a volume listed as "Unknown device." Right click on this device and choose "Uninstall." When asked to confirm, choose OK.

Note: If the power loss occurs within a few seconds of volume creation, in rare circumstances you may be prompted to format the volume after you follow the above steps. Method 2: Now try using diskpart to assign a drive letter to Partitions and check.

a) Click Start Orb > type diskpart

b) Right-click diskpart in results above > Right click Run as administrator

c) Once diskpart loads type list volume, press enter

You should notice that you will see your partition that you use for your data, it should not show a drive letter assigned to it.

d) Type select volume 6 (replace 6 with the volume number assigned to the partition in question)

e) Type assign letter Z (replace Z with the letter that you want to assign to it)

 

When you look in Computer, you should be able to see your partition.

Disable Windows Media Player:

 

a.     Press the Windows + R key on the keyboard.

b.     In the ‘Run’ windows type appwiz.cpl and click ‘Ok’.

e.     Restart the computer.

 

Enable Windows Media Player:

a.     Press the Windows + R key on the keyboard.

b.     In the ‘Run’ windows type appwiz.cpl and click ‘Ok’.

c.     Select the option Turn Windows features on or off.

d.     Check the option “Windows Media Player”.

e.     Restart the computer.
