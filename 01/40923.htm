In case of any incompatible drivers or application creating conflicts at the time of System Startup, then system fails to load all basic Widows features. End result, it will throw an error message on to the screen stating, Critical Error: Start menu and Cortana aren't working. We'll try to fix it the next time you sign in. In this situation, it won't allow you to access your desktop nor any Windows 10 features, except Task Manger.

As per the reports received herewith, that some third party software's which includes Antivirus software is causing the issue. I recommend you to remove any suspicious software through Control Panel that are installed on your PC after your PC comes up and running. 

To fix this critical error, Shutdown your PC completely and then restart it after a while. If you see the error message still appears, then follow the below steps to see if it fix the issue.

  1. Press CTRL+Shift+ESC to open Task Manager

  2. Click File > Run New Task

  3. Make sure you have a check mark beside “Create this task with administrative privileges”. If the User Account Control dialog box appears, confirm that the action it displays is what you want, and then click Continue. Type CMD and hit Enter key.

  4. Once you're on Command Prompt, type or copy paste the below command lines into it and hit Enter key:

DISM.exe /Online /Cleanup-image /Scanhealth

DISM.exe /Online /Cleanup-image /Restorehealth

sfc /scannow

To close the Administrator: Command prompt window, type Exit, and then press Enter. Reboot your system to see if problem still occurs.

Another way by booting your PC into safe mode with the same User Profile will fix the issue sometimes. To boot your system in Safe Mode,

  1. Restart your PC. When you reach the sign-in screen, hold the Shift key and select the Power button, and then select Restart.

  2. After your PC restarts, select Troubleshoot > Advanced options >Startup settings > Restart. After your PC restarts, you should see a number of options. Press 5 or F5 for Safe Mode with networking.

  3. When you're on to the Safe Mode, check whether basic features of Windows work fine there. Like Start Menu, Notification bar (I.e. in your system tray) and some other things. If the error message doesn't appear there, then it's an indication that you can login to your profile in normal mode without any issues.

This alert has something to do with corrupt or incompatible graphics card drivers. The first thing to do when receiving a blue screen along with the VIDEO_TDR_FAILURE error is to update your display and graphics card drivers. Here's how:

  1. Select the Start button, type Device Manager, and select it from the list of results.
  2. Locate the Display adapters entry and expand it.
  3. Right-click on each graphics card and select Update Driver.
  4. Select Search automatically for updated driver software.

If Windows doesn't find a new driver, you can try looking for one on the device manufacturer's website and follow their instructions.
