How to add NAS shared folders to Win10 Library

boundangle say:

Thanks for your persistent attempts to solve this.  Today I figured it out.

Apparently, The Windows File Explorer Library is directly linked to Window's Media Player.  It is possible to map NAS and other network shares if you use WMP to add the locations.  Once you add photo, music, or other network folders to the WMP Library, then they appear in the File Explorer Library.

We've heard that WMP is being phased out of the Windows hierarchy.  Perhaps this explains the disconnect.
