For powering off:

Your PC does not fully power off when it is shut down. There is still a small trickle of electricity. You may even be able to detect the light from a glowing LED on your motherboard.

This is by design and it's very safe. PCs are designed with a feature called Wake-On-LAN which enables a network administrator to 'wake' a computer that is turned off. The PC needs to have a tiny amount of power to be able to respond to commands coming in from the network.

Wake-On-LAN can be useful in other situations too, so I would recommend leaving it alone. If you really need the power to be fully off - if you're going to be working inside your computer: Shut down your PC as you normally would, count to 10, then disconnect the electric wire.

frederik long say:

I have gone over task scheduler many times with a fine tooth comb and can not find a task that does this.

-> The average Windows PC contains up to 200 scheduled tasks. That's a lot of tasks to search! I would use a comb with finer teeth to search them, e.g. the batch file below. It will generate the Desktop file Shutdown.txt, containing the name of every task that invokes shutdown.exe. You must run it under an elevated command prompt in order to gain access to the tasks folder.

Do not retype the batch file - use copy & paste instead.

Please post again if you need more detailed instructions.

@echo off
set Folder=c:\Windows\System32\Tasks
set Log="%UserProfile%\Desktop\Shutdown.txt"
echo Files containing "shutdown": > %Log%
dir /b /s /a-d %Folder% > "%temp%\Tasks.txt"
for /F "delims=" %%a in ('type "%temp%\Tasks.txt"') do (
  type "%%a" | find /i "shutdown" && echo %%a >> %Log%
)
