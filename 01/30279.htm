When you create a restore point it does not write over other existing files. An area of the disk in each monitored partition is set aside and reserved for the system volume information folder and  the restore points are written there, If the area becomes full the oldest restore point is removed to enable a new restore point to be written. ¡firedog say:

When viewing your message list, press Ctrl-Shift-O for Options. On the General tab, deselect Check for new messages every ... and click OK.

Right-click on each account name in the folder pane and select Properties. On the General tab, deselect Include this account when receiving mail or synchronizing. If you want to make absolutely certain that you don't start downloading messages by accident, on the Servers tab change the server name for incoming mail (POP3) to a.b.c , then click Apply and OK.
