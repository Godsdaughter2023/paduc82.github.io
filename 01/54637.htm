Hi Pixelme

Using the on-screen keyboard is no more secure than using a physical keyboard

The underlying process is the same, regardless of the input method and a keylogger will pick up input from the on-screen keyboard just as easily as from a physical keyboard

The hacker can still read your input from the Windows I/O System.
