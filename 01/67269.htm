I recorded a video on a camcorder in mts format. How can I convert it - Microsoft Community

needhelp1290 say:

I recorded a video on a camcorder in mts format. How can I convert it to a more commonly used format?

I can download and make a DVD, but I can't read it in any older computers or DVD players. Need help and planning on sending copy ASAP!
considerate_guy say:

Most likely your .mts file is an HD (high-definition) video file.  What lots of people forget, the DVD format is actually an SD (standard-definition) format.  When you burn that .mts file onto a DVD, you are making a data DVD that will only be playable on other PCs that are capable of playing HD .mts files.  As you have discovered, it won't play on older computers and definitely won't play on a standard DVD player.

You can create a video DVD if you follow the correct DVD format (MPEG-2 in a .vob file) along with the requisite files for the menus, etc.  This isn't easy to do manually but can be done with a program.  One option, if your computer has it, is to use Windows DVD Maker.  It will do the transcoding and burning to make a video DVD (as opposed to a data DVD).  Obviously it will be lower SD resolution, but at least it will be playable.

If your goal is to play it on other computers, you can try a more standard format like .wmv.  That can handle HD video.  One choice is to do it through Movie Maker just by adding the .mts file to the timeline then saving the movie (File > Save Movie) and choosing a .wmv format.  Or if you don't plan to do any editing, you can just do a straight conversion with something like Freemake Video Converter (freemake.com).  Since FVC is freeware, it offers other add-ons during installation, so I recommend unchecking all those and only installing the program.

Bug Check 0x3B: SYSTEM_SERVICE_EXCEPTION

The SYSTEM_SERVICE_EXCEPTION bug check has a value of 0x0000003B. This indicates that an exception happened while executing a routine that transitions from non-privileged code to privileged code.

Important Info If You Have Received a STOP Code

If you received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss. A hardware device, its driver, or related software might have caused this error. If your copy of Windows came with your computer, call the manufacturer of your computer. If you purchased Windows separately from your computer, Microsoft provides support. To find contact info for Microsoft or your computer manufacturer, Contact Support.

If you have experience with computers and want to try to recover from this error, follow the steps provided in the Microsoft article Resolving STOP (Blue Screen) Errors in Windows.

These actions might prevent an error like this from happening again:

  1. Download and install updates and device drivers for your computer from Windows Update.
  2. Scan your computer for computer viruses.
  3. Check your hard disk for errors.

SYSTEM_SERVICE_EXCEPTION Parameters

The following parameters are displayed on the blue screen.

Parameter Description

The exception that caused the bug check

2

The address of the exception record for the exception that caused the bug check

3

The address of the context record for the exception that caused the bug check

4

0

This error has been linked to excessive paged pool usage and may occur due to user-mode graphics drivers crossing over and passing bad data to the kernel code.
