DNS server not responding error may occur due to an issue with TCP/IP settings which may not be configured properly on the computer or due to any sort of malware or virus on the computer.

Let’s try the following steps and check.

Method 1:

Use the IPconfig command-line tool to re-register DNS.

a. Press Windows and x key together on your desktop screen.
b. Click on Command Prompt (admin).

c. At the command prompt, type the following commands. Press Enter after each command.

ipconfig /flushdns
ipconfig /registerdns
ipconfig /release
ipconfig /renew

For further information, refer to the article:

Why can’t I connect to a network?

 

Method 2:

I would suggest you to reset TCP/IP settings on the computer and check if it helps.

You can follow the steps:

a. Press Windows and x key together on your desktop screen.
b. Click on Command Prompt (admin).

c. At the command prompt, type the following command and then press ENTER:

netsh int ip reset c:\resetlog.txt

d. Restart the computer.

 

Method 3:

You may also refer to the article and perform a full scan on your computer using Microsoft Safety Scanner.

Microsoft Safety Scanner

Note: Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.
