myronh say:

Hate to be flippant about the issue, but the error is already telling you the problem. Not having any other specifics about which printer you have it's hard to even look into if there are any options. However, there probably aren't many, if any. You can check HP Support to see if they have a Windows 7 driver for your printer. If they don't you won't be able to install it on Windows 7.

There is one other option that might be of use to you, but if there are costs incurred you will have to balance that with how much it would cost to get a new printer that is compatible. If you have Professional or Ultimate and your hardware supports Hardware Assisted Virtualization (HAV) you can try XP-mode...creating a virtual machine running a full XP SP3. You will probably have to download XP drivers, but that shouldn't be a problem since your printer is definately compatible with Windows XP. The Windows Virtual PC will provide instructions, a HAV test application, and all other downloads necessary to install XP mode. Once XP mode is installed and running on your system you simply have to install your printer under XP. This link will take you to a tutorial on loading USB devices and running them in XP mode to include sharing with the host Windows 7 system.

If you don't have the proper version of Windows, anytime upgrade to Professional will run $90 (everything else is free). However, before you spend on the upgrade, run the test of your HW to make sure you support HAV. That is step two of the Windows Virtual PC instructions. No sense spending $90 if you can't run XP-mode anyway. If your HW doesn't support you would have to buy a new PC to get there...much more expensive than a new printer.

If you don't support HAV but really want to keep the printer, you can use VirtualBox.org for virtualization (free) and then load a full version of either XP or Vista to create a virtual environment to load your printer. The cost here would be about as much as the cost of a new printer.

If you are not overly attached to the printer, just buy a new one. HP has some photosmart printers that include wireless conectivity and even use separate color cartridges. That last is very nice as you rarely run out of all colors simultaneously.
