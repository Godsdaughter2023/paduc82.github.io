Even after you have enabled language bar in control panel, it disappears again.
You can see the language bar only when UAC prompts for administrator elevation.

Open cmd and issue the following command:

ctfmon

Language bar appears. But this is not enough.
We have to run ctfmon everytime we start Windows 7.

Follow the instructions below to run ctfmon automatically on Windows startup process.

1. Click Windows 7 start button.
2. Type: regedit
3. Open it with administrative rights.
4. Goto HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run
5. Create a new string value
6. Name it as you wish
7. Open it for edit
8. Type “ctfmon”=”CTFMON.EXE” in Value data field
9. Press OK
10. Restart your computer

I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

Your Microsoft account is able to only sync some settings on the PC which you choose yourself at Settings > Accounts > Sync Your Settings. Any files you drag into OneDrive or are having OneDrive sync from a device are also associated with your account but if you change the account on your PC to a Local Account or another Microsoft account you can always sign into that OneDrive account separately. So the only thing that the Microsoft account carries with it alone is the Sync Settings you choose.

If you want to change Accounts on your PC you can change to a Local Account to leave your MS account: http://www.zdnet.com/article/windows-10-tip-swi...

Once you're in the Local Account it will be the same exact account. If you're wanting your OneDrive files that are associated with the old MS Account, just sign into OneDrive. Same with Xbox or any other Microsoft Account feature except sync which only comes with the actual MS Account.
