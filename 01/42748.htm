    Hi again Krista -

    If you know for sure that you had the RTM permanently (as opposed to just the build) Activate then you should indeed be able to change hardware and it should reactivate. 

    This is further explained in How to Use Your Free Windows 10 License After Changing Your PC’s Hardware although that pertains more to changing the motherboard which is the only single piece if changed that should force reactivation.  So if you know for sure it was permanently activated, I'd try the suggestions in the tutorial which are already working for many. 

    I'm afraid that if it will not reactivate after going online once the Hard Drive is changed, then you might not have had it permanently Activated.  To confirm permanent activation, one must right click Start button to choose Admin Command Prompt, type slmgr /xpr.  This will tell you if its permanently activated, which is the state it needs to be after free Upgrade before doing the Clean reinstall. It might have only had the latest build activated as an Insider.

    As a last resort you can try  one of the workarounds which was used by many in a huge thread from August Can I clean re-install Windows 10 after upgrading? - Microsoft Community  for those who  could not initially get activated after Clean Reinstall even though they'd confirmed Activation.  You can reinstall Windows 7 to reactivate and then Upgrade again, or  shrink C to install and Activate 7, upgrade, then activate Upgrade which should activate your present install so you can delete the crutch partition.  If  Windows 7 will activate and is not blocked, then you'll know it was never used to officially Activate.  There is another workaround there which doesn't require reinstalling the qualifying OS that is tacitly being allowed by developers since it mimics what the Upgrade does  but won't work after free Upgrade period.  However by then you should have your activated RTM preserved in a backup image so that it can be recovered at any time in 20 minutes.
