matteo_giovanetti say:

I had this same issue as well in Windows 7, and I was able to resolve it without disabling the autoplay function for all devices as suggested in some other replies.

Go to regedit and find the following keys:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\AutoplayHandlers\EventHandlersDefaultSelection\UnknownContentOnArrival

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\AutoplayHandlers\UserChosenExecuteHandlers\UnknownContentOnArrival

Change the (Default) entry within both of these keys to be "MSTakeNoAction" (without the quotes).  

After this, when you plug in your iPod, you will no longer get the autoplay prompt at all for this device.  All of the other known devices will maintain whatever autoplay action you specified.
