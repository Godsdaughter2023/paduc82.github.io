How can I use Movie Maker to burn 1080p Blu-Ray discs? - Microsoft Community

capndoc say:

How can I use Movie Maker to burn 1080p Blu-Ray discs?

Movie Maker allows "Burn a DVD" but it is only 720 x 480.  Any way to get 1080p HD resolution burned to my Blu-Ray burner?

 

considerate_guy say:

Movie Maker does not have that ability.  The "Burn a DVD" option calls Windows DVD Maker to make a standard-definition DVD, as you have discovered; there isn't a Blu-ray option.  However, the Blu-ray standard includes WMV video using the VC-1 codec, so I've been able to workaround this as follows:

  1. Save my movie computer in the HD format (1920 x 1080) as a .wmv file.
  2. Burn that just as a data file onto a DVD.
  3. Put that into my Blu-ray player (PS3) and it works fine.

While it doesn't have any fancy features like a menu, it does play correctly.  It also only uses a DVD disc, rather than an expensive writeable Blu-ray disc which makes it inexpensive to create and test.  The only drawback is you are limited in the amount of HD content you can fit on a DVD this way.  But I'm not create feature-length movies, just archival copies of our home videos which are usually around 30-45 minutes.

Other than that, you would have to look into software that might have come with your Blu-ray burner and/or PC.

considerate_guy say:

Typically I just stream my videos to the PS3.  But if I burn the HD .wmv file onto a DVD, it plays correctly in the higher resolution.  Are you just burning the .wmv file onto the DVD, or are you trying to make a video DVD? The latter will have to down-res to the DVD standard resolution.
