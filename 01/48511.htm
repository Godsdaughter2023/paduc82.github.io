It seems that you would need to take ownership of the folder in which you are trying to copy the file.

Use the following syntax to take the ownership:

1. Click Start and then type cmd in the Start Search box.

2. In the results area, right-click cmd.exe, and then click âRun as administratorâ. You will be prompted to type the password for an administrator account. Click Continue if you are the administrator or type the administrator password. Then, click Continue.
