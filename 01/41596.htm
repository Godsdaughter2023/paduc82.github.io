nikhar_k say:

Hi,

Thank you for writing to Microsoft Community Forums.

This issue could be due the corruption of system files related to Windows Mail app. I would suggest you to follow the steps mentioned below:

Method 1: Running In-built troubleshooter

You can run the In-built app troubleshooter to fix the issues with the app crash. Follow the steps mentioned below to run the app troubleshooter.

  1. Press and hold Windows key + I, to open Settings.

  2. Click on Update & Security.

  3. Click on Troubleshoot.

  4. Scroll down, click on Windows Store Apps and then click on Run the troubleshooter.

  5. Follow the on-screen instructions.

Method 2: Uninstalling and Reinstalling the Mail App

If Method 1 didnât work, then we can uninstall the Mail App and reinstall it using Windows PowerShell. Follow the steps mentioned below to uninstall and Reinstall the App.

  1. Type PowerShell in the search bar on the Taskbar.

  2. Right click on PowerShell and select Run as administrator.

  3. Now, copy paste the command and hit Enter key: Get-AppxPackage *windowscommunicationsapps* | Remove-AppxPackage

  4. Once the command is executed, open Microsoft Store application, search for Mail and Calendar app with the help of search bar and install it.
