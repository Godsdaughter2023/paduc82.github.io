I jus now wanted to do the same thing, this worked for me on Windows 7.

shane-- say:  

Hi Bart,

  If you have Outlook Express 6 on the Windows XP PC you can import the messages into it directly from Microsoft Outlook and then copy the Outlook Express data files to the Windows Vista PC.  You can then use the import tool in Windows Mail and choose the Outlook Express 6 option.   

Set up a temporary account in Outlook Express on the Windows XP PC and select the File menu, choose Import, then Messages, and Outlook (Outlook must be configured and will have to contain the messages you are wanting); then select what folders you want to import.  Once the import is complete and you see all of your messages you will need to find out where the data files are stored. 

Within Outlook Express 6:
Click the Tools menu, and select Options

Click the Maintenance tab

Click the Store Folder button

This will be the path to the files you need

Browse to that path and copy any files in that directory to the Windows Vista PC. 

On the Windows Vista PC:
Within Windows Mail, Go to File, then Import, then select Messages
Choose Microsoft Outlook Express 6
Click Next
Choose Import mail from an OE6 store directory, and click OK
Click the Browse button
Browse to the folder that you copied the data stores to.
The files will not show up and you will see a message saying "No items match your search"  This is ok.  Just click the Select Folder button.
The path should now show up to the left of the Browse button
Click Next
Select which folders you want to import
Click Next

This will begin the import process.  Within Windows Mail there should now be a folder named Imported Folder that will contain whatever you imported over.  If you have any other questions, please let us know.  Thank you.
