bossdweeb say:

"Would changing the front audio setting to AC'97 in the bios work? "

- You do want your AC97 audio enabled in BIOS.

- You also need the Realtek HD audio drivers that are supplied by the mfg of your motherboard

(or PC if a brand name such as DELL/HP).

- There are several ways to configure audio for using either the speakers or headphones, or both,
with Realtek AC97 audio, and the terminology is a bit... confusing ?

The following assumes you have connected the front panel to the correct header on the motherboard

(some motherboards have two options, but I don't recall the difference at the moment).


- (Depending on which version AC97 and the software you are using)....
One option is - Open the Realtek HD Audio Manager (Realtek HD  should be in Control Panel, but you can also add a short-cut to the Task Bar Notification area).

- Click on the folder icon in the upper right of the window and make sure the tick box
next to  Disable front panel jack detection  is clear.

- In the Device advanced settings option if you select:
'Make front and rear output... simultaneously' (yes sounds wrong. I know),
the speakers will be disabled when the headphones are plugged in, and the speakers
will function when the headphones are unplugged.
- This option also allows you to set-up the configuration (Sound Effects, etc.).  for the headphones  to be different from the speakers.
(The option  'Mute the rear...plugged in'  does not offer an option to set-up a different configuration
for the headphones from the speakers, but works the same otherwise).

Alternatively -  If you tick the box next to  Disable front panel jack detection and select -
Make front and rear output... simultaneously the speakers will not work if you unplug the headphones.
- If you select - 'Mute the rear...plugged in',  both speakers and headphones will

operate simultaneously, and when the headphones are unplugged the speakers will still function.
[ If that is clear as mud you at least know which settings to experiment with ]

- To allow the headphones to always show in Windows Control Panels  'Sound > Playback Devices',

R/click on any of the Devices listed and tick Show Disabled Devices and Show Disconnected Devices.

NOTE: If playing games you need to plug the headphones in before launching the game or the
audio in the game will not work (if using the first option. Not sure about the others).
