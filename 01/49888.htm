syed n say:  

Hi JayOttawa,

 

There's a limitation of the file system. It can only handle file names of a certain length. Path limit is 248 characters.Path+Filename limit is 260 characters. Exceeding either will result in the error mentioned. Simplify your folders and titles.

 

A simple fix is to copy or move the files to different location which shortens them quite a lot, then remove/delete the original. For this, you will need to go to the actual file location and then proceed. This will help you backup these files as well.

 

As an alternative, you can skip these files when backing up. Windows 7 gives the ability to choose individual folders that you want to be included in the backup. While setting up the backup, there is an option provided to select the location and the files that you want to backup. You may select “Let me choose”and select the location for the backup and the files that you want to backup.

 

Hope this Helps!
