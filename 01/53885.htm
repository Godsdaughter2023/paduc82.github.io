It's not clear what you want to do.

If you have a computer that already has Windows XP installed and you want to update it to service pack 3 but the computer is not connected to the Internet, all you need do is download the appropriate SP2 and/or SP3 installation files as described below.  Because the machine is not connected to the Internet, ignore the instruction to connect to Windows Update for post-sp3 updates and instead install SP3 together with those updates as described at the web site linked at the very bottom of this post.

If the computer that is not connected to the Internet does not have any version of Windows XP installed, that's a completely different problem that will NOT be solved by the instructions below.

All of the Windows XP updates released prior to April 8, 2014, are available through Windows Update -- once service pack 3 has been installed (Windows Update will not work for XP pre-sp3).

Service Pack 3 can only be installed AFTER you have installed service pack 1, service pack 1a, OR service pack 2.

Disconnect from the Internet.  Disable (or preferably uninstall) your antivirus program.

hp link --> http://h20564.www2.hp.com/hpsc/swd/public/detail?swItemId=pv_60484_2
