    hairyfool say:

    Although it is possible to recover deleted files in the right circumstances most if not all tools work at a sector level on the physical data on the disk. Because the data is encrypted you are not going to be able to read the drive contents because you would need to run "through" bitlocker in order to see any readable data.

    This is the entire point of bitlocker, it makes it impossible to read the drive contents without proper decryption. Such encryption tools should be used with extreme caution and are not targeted towards home users. Intended for use in a business environment where the data stored can be taken off site the ultimate protection against failure is that unencrypted backup held within a secure corporate network.

    I would NEVER recommend users employing encryption unless there is a specific need and that they a/ understand the inherent risk and b/ have that unencrypted backup store.
