e) Restart the computer and let windows install this software automatically and then verify if it works.

wpak say:

Hi there,

usually it's not possible to see any Trial versions as a subscription, which is likely the cause why you can't see it.

It's not needed to create a seperate account for Live, if you can login at xbox.com (with that hotmail account) then you already got an Xbox Live Account to which you can connect Xbox Live Gold subscriptions. Be sure to check if that Xbox Live Account shows the same gamertag and gamescore to what it shows on the Xbox console. That needs to be the same, if a different one is shown then you are either using a different email address or for an unknown reason you have 2 accounts connected, which shouldn't be possible. (In such case contact Xbox Support and ask for steps on how to remove one of the accounts)

I don't think Windows 7  should be a cause, but if possible try to go to http://account.microsoft.com/services on any other device to see the subscriptions.

Side note: keep in mind that you can't add multiple trial subscriptions on top of each other.
