Is there any way you could try using a different HDMI cable with your current television?

Try unplugging the HDMI cable before plugging in the power block. Once the HDMI cable is unplugged, plug in the power cable and then try turning on the console. If there is no problem when the HDMI cable is in, it is likely to be a problem with either the cable or the television.

If it works fine when using other cables and a different television, try with the composite cable (3 colour cable) on the original television.

If it works with the composite cable on the original television, it is likely to be a problem with the HDMI cable.

If this is still a problem, try using the HDMI cable with a different television.

If the HDMI cable causes no problem when using it with a different television, it is more likely to be a problem concerning the television.

(I have used successfully used the manual Regedit method shown before, I have not tried the Fix it method).

 

Now you may restart your PC, enter BIOS and change to AHCI and start your PC normally.

  

You should make sure that you have backups on separate drives before starting.
