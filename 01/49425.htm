Hi Long Johann,

This issue might have occurred when the boot files are missing or corrupted.


I suggest you re-built the boot record by following the below commands and check if it helps.


You will have to use Windows 10 installation media to follow the steps to boot into Recovery Environment and try to rebuild BCD.

Step 1: I suggest you to create an installation media for Windows 10 using the media creation tool:

Installing Windows 10 using the media creation tool

Step 2: After preparing the installation media, insert it hard reboot the device and follow the below:

  1. Boot into Windows Recovery Environment using the Windows 10 installation media.
  2. Click on Repair your computer.
  3. Click on Troubleshoot and in the Advanced options.
  4. Select Command Prompt. Type the following commands into the prompt and press Enter key after each command.

  • bootrec /fixboot

  • bootrec /scanos

  • bootrec /rebuildbcd 

    5.  Restart the computer.

carlosbk-- say:

Larry E,
 Is there any more details on the error that you got when you tried to install the monitor? And you didn't mention if there was an error at all when you tried to install your monitor manually? You can usually find more information about an error in the "Event Viewer" you will need to know the exact time that you got the error so it can be easier for you to find. To look in the "Event Viewer" please follow the steps below.

1. Click on "Start"
2. Right click on "My Computer"
3. Select "Manage"
    The "Computer Management" window will now appear.
4. On the left side of the window you will see "Event Viewer". Click on the arrow beside it to expand the option.
5. Click on the arrow beside "Windows Logs"
6. Search through the categories and see if you can find the error that you are having. (It will help if you remember the time that it happened)

It is possible to find out if the monitor would work properly with Generic PnP drivers by going to the device manager and looking at the properties of the device under status. To check the status of a driver in the "Device Manager" please follow the steps below.

1. Click on "Start"
2. Right click on "My Computer"
3. Select "Manage"
    The "Computer Management" window will now appear.    
4. On the left side of the window select "Device Manager".
5. Now in the middle section of the windows double click on "Display Adapters"
6. Right click on the device that appeared under "Display Adapters" and select "Properties"
7. In the "General" tab under the "Device status" will give you the status of how the monitor is operating.

You also might want to try to update the driver while you are in that properties window.

8. Click on "Driver" tab
9. Select "Update Driver"

Let us know on your results.
