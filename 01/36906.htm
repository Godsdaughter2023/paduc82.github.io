The simple answer/solution is to Change the Transition Effects for Menus and Tooltips

Follow these "no-brainer" steps:

1. Start

2. Right Click Computer

3. Select Properties

4. Click the Advanced System Settings

5. Advanced Tab

6. Under Performance, Click Settings

7. Under Custom: tick (uncheck) the check-box FADE OR SLIDE MENUS INTO VIEW

8. Apply & OK.

If this virtual network interface is enabled, then MBSA fails with error code 0x00000001. If this interface is disabled using for instance the control panel, then MBSA no longer fails during the test of the firewall.
