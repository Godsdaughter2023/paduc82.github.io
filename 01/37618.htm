Hi,

To better assist you, here are the steps on how to set up your Roadrunner email account to Windows Live Mail:

1. Open Windows Live Mail and click the Accounts tab.

2. Click the Email icon.

3. Enter you email address, password and the name you would like to display with your sent messages, then click Next.

4. Select POP or IMAP in the Server type dropdown. Note:  IMAP leaves messages on the server so multiple devices and email clients can access the messages and the messages still remain on the server. POP downloads the message onto a client.

5. Enter your server settings:

6.POP Settings:

    Account Type: POP or POP3

    Incoming Mail Server: mail.twc.com

    Incoming Server –POP3 port: 110

    Outgoing Mail Server (SMTP): mail.twc.com

    Outgoing Server –SMTP port: 587

    Inbound & Outbound User Name: Enter full email address

    example@[division].rr.com

    *** Email address is removed for privacy ***

    *** Email address is removed for privacy ***

    Inbound & Outbound password: Email Password (same as Web mail)    SMTP Server Requires Authentication: Checked

7. IMAP Settings:

   Account Type: IMAP

   Incoming Mail Server: mail.twc.com

   Incoming Server –IMAP port: 143

   Outgoing Mail Server (SMTP): mail.twc.com

   Outgoing Server –SMTP port: 587

   Inbound & Outbound User Name: Enter full email address

   example@[division].rr.com  

   Inbound & Outbound password: Email Password (same as Web mail)   SMTP Server Requires Authentication: Checked

8. Click Next.

9. Your TWC Mail is added to Windows Live Mail.

10. To make changes to your account, choose your email account in the left column and click the Properties icon.

11. Click the Servers tab to edit your settings.

12. To check for mail, select your TWC Mail account in the Send/Receive drop-down.

For more information regarding this process, visit this link.

Let us know if you need further help.
