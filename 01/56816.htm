a. user say:

Hi,

Thank you for contacting Microsoft Community.

I appreciate your time and effort.

I understand that you are facing the issue suspected file. I will certainly help you with this.

Most antivirus programs identify nbt.exe as malware. The nbt.exe file is a software component of Win32 Backdoor Trojan. A file known as "nbt.exe" is a file launched by explorer.exe in Windows. This particular file modifies certain system files to give it administrative control. It opens hidden operational functionality to allow remote hacking control.

Nbt.exe is not essential for Windows and will often cause problems. The nbt.exe file is located in a subfolder of the user's profile folder (C:\Users).

Try to remove this program from programs and features:

a. Press WIN+R.
b. Type appwiz.cpl.
c. From the list, check for nbt program.
d. Right click on it and select uninstall.
e. Reboot the computer.

However, it is not possible as each wireless dongle and mouse/keyboard set are matched to the radio frequency that they have been configured to use, this is set at the manufacturing plant. No two wireless sets have the same frequency to avoid Crossover when two sets are being used in close proximity. We would suggest you to buy a new wireless set complete with matching transceiver.
