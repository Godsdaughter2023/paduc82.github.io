lemp say:

Please try to be a little less cryptic.  I'm not sure what you mean by "move to other users startup."

There are several ways you can share files between users on a single Windows 7 computer.

1.  There are two types of users on a Windows 7 computer:  Standard users and Administrators:

Although "security best practices" says that you should use a "Standard user" account for everyday use, many people use "Administrator" accounts, especially if there is only one user account on the computer.  If you do use an "Administrator" account on a regular basis, you should create a new "Administrator" account, give it a password (that you will write down in a place that you won't forget) and keep that as a spare in case of emergencies, much like a spare front door key to your house.

Administrator accounts can access everyone's files on the computer, whether they have been explicitly shared or not.

2.  Administrator accounts aside, you can explicitly share files by right-clicking a file (or a folder) and selecting "Share with" (there is also a "Share with" button in the menu bar of Windows Explorer):

Select "Specific people"

Click the Down arrow in the box next to the "Add" button to see a list of the other user accounts on the computer.  The "Everyone" group and probably also "Homegroup" will be included in the list.

Select which user you want to permit access and click the "Add" button.

You can click the arrow under "Permission Level" to select Read, Read/Write, or Remove.

Click the "Share" button when you're done.
