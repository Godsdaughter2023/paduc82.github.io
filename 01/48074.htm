determinator say:

I solved this problem by removing the archive bit from the files that I brought across from my old PC.  I believe that it gets set once a file has already been indexed on the old machine and then isn't reset when it is copied (unless you use the new robocopy command with this option on Windows 7).

I was able to fix it by going to the command prompt and changing to the directory in question and then issuing an "attrib -A *.* /s" (clears the attribute bit on all files, recursively through all directories).

My blog has more details of how I found this solution.

http://mikekirby.spaces.live.com/blog/cns!E008D856FA702CAC!758.entry
