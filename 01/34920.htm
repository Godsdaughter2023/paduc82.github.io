    sonic_spike say:

    When you upgrade to Windows 8.1, parts of your machine behave like a new install.  So try the following:


    1) Go to the option "Allow a Program Through Windows Firewall".  This is as easy as typing "Allow" from the Windows 8.1 "Modern" Start menu...when it opens, there is a list of programs controlled by the firewall.  Click the "Change Settings" button above the list to put the list in a mode you can edit.  Locate any and all instances of "Steam" in the list and check the boxes to allow them through.  Apply your changes.


    2) Open a browser, go to http://www.steampowered.com/ and attempt to log in.  You may get a message saying that this is an unrecognized location you are accessing Steam from for the first time, and what Steam will do is send you an e-mail to the address you use for your Steam account.  The e-mail will contain a code that you need to type in, click the button acknowledging that you got the code and have typed it in above, and it should log you in.


    3) Regardless of whether this allows you to log into your account from the Windows 8.1 machine, at the steampowered page, download the Steam installer.  When you run the installer, choose the option to Repair your Steam installation.


    You might now need to reboot to make it stick.  Good luck!
