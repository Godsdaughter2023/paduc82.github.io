Windows 10 downgraded to windows 7 - Microsoft Community

a. user say:

Windows 10 downgraded to windows 7

Hi,

Can a Dell OptiPlex 7060 which comes with Window 10, be downgraded to Window 7 Professional?

canadian tech say:

The answer is probably, but with difficulty and no guarantees.  You will need a usable Microsoft Product Key for Windows 7.  If you previously bought a Windows 7 upgrade product at retail, that licence does not expire and can be installed on any number of systems -- one at a time.

First if your  system is still under warranty, you would void the warranty by doing this.

Second, you will have to find Windows 7 drivers and Dell does  not list them on their support page.

I have done this before on a few different machines, although not this specific one.

You need to get a Windows 7 installer ISO with USB 3 drivers embedded in it.  You can find the instructions to do this here:

https://dellwindowsreinstallationguide.com/download-dell-skylake-windows-7/

When the installer finishes, it  is  likely it will have found most, if not all the drivers it needs in its own install library.  As long as the missing driver is not a network driver, you start the Windows Update process and it is likely it will find and install the rest.  Strongly recommend you read the following before you do that:

https://answers.microsoft.com/en-us/windows/forum/windows_7-update/windows-7-updating-in-2018/291a9582-17a7-4942-8145-8d9f68265189?messageId=2f46cef5-da27-431d-8c6f-b5e7b2b89246

When this is done, you could have a device without a driver.   That's the hard part. I have done this before, but it is technically quite challenging.  

 

Do not even think about using one of those hundreds of driver download sites.  They are full of malware and very few have valid or usable drivers.

 

The technical trick is to use Device manager to identify the "hardware ID"  Then Google  search for the driver for it.  Very few of these devices are really unique to Windows 10.  The OEMs are just to lazy, beholden to support of Windows 10 and cheap to put the device drivers on their web sites.  So, the devices you need the drivers for are likely commonly used in both operating systems.  It is not very probable that a new device was invented just for Windows 10.  You probably will find the driver on another product from the same OEM, or another OEM.

  1. Device Manager
  2. Click the arrow next to the category
  3. Right click on the device
  4. Choose Properties
  5. Details tab
  6. Find Hardware ID's in the  drop down

Now Google the hardware ID along with the word driver.  It may take a lot of searching, but in each case, I have attempted this, I was successful.

 

Note well, you probably would  be violating the terms of your warranty by doing this.

I  strongly advise you to create an image of your hard drive BEFORE you try this.  If it does not go well, you want to be able to get back to safety.
