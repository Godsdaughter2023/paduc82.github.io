Port Forwarding

  1. Plug the computer into the LAN port of the device via an ethernet cable.
  2. Open Web Browser.
  3. In the address bar of the browser enter in "gateway.2wire.net" (if this does not work, try one of the following IP addresses: (192.168.0.1/192.168.1.254/192.168.254.254)
  4. Click on Settings.
  5. Click on Firewall.
  6. Click on Applications, Pinholes and DMZ.
  7. If it asks for a password check the side of the device.
  8. Click on the device from the device list or specify an IP address.
  9. Click on Allow Individual Applications.
  10. Click add a new user-defined application.
  11. Fill in the information for port forwarding.
  12. Click Done.

 

Disable Port Forwarding

  1. Open Web Browser.
  2. In the address bar of the browser enter in "gateway.2wire.net" (if this does not work, try one of the following IP addresses: (192.168.0.1/192.168.1.254/192.168.254.254)
  3. Click on Settings.
  4. Click on Firewall.
  5. Click on Applications, Pinholes and DMZ.
  6. If it asks for a password check the side of the device.
  7. Select Reset All Firewall Settings in the top right corner.
  8. Select Continue.
