Hi,

Thank you for posting your query on Microsoft Community.

If you are using a Microsoft Account you need to enter your password every time you reboot. However, if you are using a local account you can disable password.

Make sure your account has Administrator privileges to make changes in the local account. 

1. Right click on the start menu and select Computer Management. 

2. Go to Local Users and Group > Users. 

3. Right-click on the user-account you wish to have no password for and select Set Password. 

4. If a prompt appears, click on Proceed. Leave both the textboxes to enter password and click Ok.

NOTE: It is always recommended to keep a password on your User Account even if you are the only person using the system.

Sometimes your computers firmware, which deals with things like power management, which is related to sleep needs to be updated. Second option
uninstall your display driver then restart
Boot to the Windows 10 desktop

Press Windows key + X
Click Device Manager
Expand Display adapters
Right click the current display adapter
Click Uninstall
Exit Device Manager
Restart
