malkeleah say:

The message about using Broadcom instead of Windows for wireless management has nothing to do with the router. It means that your Broadcom Wireless Network Adapter (hardware inside your computer) has its own wireless management program and you are using it instead of the Windows one. This is OK. Are you sure you don't have a Comcast combination modem/wireless router? If you do, then you don't need the Linksys. If you definitely have only a Comcast modem, then you do want the Linksys router and either it isn't set up correctly or it isn't working. How old is the router and did it ever work?

A pure modem (not a combination modem/router) will be a small box that has connections on the back for:

1. The coax cable (biggish, black, with a screw-on connector) that runs from the modem to the cable box outside your house.
2. The power connector (electricity) which is usually black.
3. Sometimes a USB port which should really not be used.
4. An ethernet port.

Assuming you do have just a modem, connect the ethernet cable directly from the modem to the ethernet port on your computer. There is only one place this can go. Although it will look similar to a dialup modem port (if you have one), it is bigger and will not fit. Remove the power cord from the back of the modem and wait about 30 seconds. Replace the power cord and wait until all the modem's lights are back on. Now on the computer, open up a browser (Internet Explorer or Firefox). Are you online? If you aren't, it's best to test the modem by connecting a different computer to it if you have one. If the different computer can't get online when directly connected to the modem then you need to call Comcast because there is a problem with the modem and/or their service.

If you can get online when connected directly to the modem, the router is your culprit.

1. Turn off the power to your cable modem. The router should also be unplugged at this point.
2. Attach a cat5e ethernet cable (usually provided with the router) going from the router's Internet/WAN port to the cable modem's ethernet port.
3. Attach the cat5e ethernet cable going from your computer's network adapter to one of the ports on the router.
4. Turn on the cable modem. After all the lights are on, turn on the router.

Now open up a browser on the computer and see if you are online. Note that we are still not worrying about the wireless part yet. We need to make sure you have Internet connectivity wired first. Let me know the results of this troubleshooting and we can go forward with the next steps.

malkeleah say:

Yes, that's exactly what I was telling you to do. One of the most important things about troubleshooting is that you are essentially using the process of elimination. Therefore if you have good Internet connectivity when connected directly to the cable modem by ethernet and you don't with the router, you know the router is probably at fault. And it sounds like it's time to replace the router. They don't last forever.

And just in case you need it, here are some notes about setting up your wireless network securely:

Have a computer connected to the router with an ethernet cable. Examples given are for a Linksys router. Refer to your router manual or the router mftr.'s website for default settings if you don't have a Linksys. Open a browser such as Internet Explorer or Firefox and in the addressbar type:

http://192.168.1.1 [enter] (this is the router's default IP address, which varies from router to router so check your manual)

This will bring you to router's login screen. The default username is left blank and the Linksys default password is "admin" without the quotes. Enter that information. You are now in the router's configuration utility. Your configuration utility may differ slightly from mine. The first thing to do is change the default password because *everyone* knows the default passwords for various routers.

Click on the Administration link at the top of the page. Enter your new password. username blank and put in your new password to get back into the configuration utility.

Now click on the Wireless link at the top of the page. Change the Wireless Network Name (SSID) from the default to something you will recognize. I suggest that my clients not use their family name as the SSID. For example, you might wish to name your wireless network "CastleAnthrax" or the like. ;-)

Click the Save Settings and when you get the prompt that your changes were successful, click on the Wireless Security link which is right next to the Basic Wireless Settings link (where you changed your SSID). Most computers purchased within the last 4 years have wireless hardware that will support WPA2-Personal (also called WPA2-PSK). This is the encryption level you want. If your wireless hardware is older, use WPA. Do not use WEP as that is easily cracked within minutes. So go ahead and set the Security Mode to WPA2-Personal. Do that and enter a passphrase. For example, you might use the passphrase, "Here be dragons, beware you scurvy dogs!". The passphrase is what you will enter on any computers that are allowed to connect to the wireless network.
At this point, your router is configured and if the computer you were using to configure the router is normally going to connect wirelessly, disconnect the ethernet cable and the computer's wireless feature should see your new network. Enter the passphrase you created (exactly as you wrote it with all capitalization and punctuation) to join the network and start surfing.

Excellent guide to setting up a wireless network, with pictures - http://www.microsoft.com/windowsxp/using/networking/setup/wireless.mspx

For now, I suggest that you follow the steps below to activate the one finger scroll.

1: Update your Synaptic driver to the latest version. If the Synaptic touchpad driver isn’t installed, please the manufacturer website and download the drive and install it. You might be asked to restart your PC to complete the driver installation.

2: Once installed, type Change Mouse Settings on Search bar, change the Search filter to Settings to see the Mouse settings entry in search result, and then press enter key to open Mouse Properties dialog.

3: Here, switch to the Device Settings tab. Please note that Device Settings tab appears only if the touchpad driver is installed on your PC.

4: Under Devices, select Synaptic Touchpad and then click Settings to open Properties for Synaptic Touchpad.

5: On the left-side, expand MultiFinger Gestures, check Two-Finger Scrolling option and then click Apply button. That’s it!

NOTE: To change the default behavior of two-finger scrolling, select Two-Finger Scrolling option and then click the Settings icon next to it.

Hope this helps
