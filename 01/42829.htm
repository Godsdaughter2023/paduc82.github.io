wunders say:

To answer your question, this behavior can happen if Windows detects a corrupt user profile. When the user goes to logon to the computer and a corrupted profile is detected, one of the things that Windows sometimes does is simply create another user profile that is based on the current user name (similar to "111-pc-owner").

 

I don't know how you tried to delete it, but the generally accepted method is the following:

  • As Administrator user, Right-click "My Computer" and then select "Properties"
  • Select the "Advanced" tab
  • Click on the Settings button in the User Profiles section (may take a while)
  • Select the profile you want to delete (all these profiles should show up in that screen) and click on the "Delete" button

You should confirm which profile is the current (active) profile first so that you delete the correct one.

 

BTW: Corrupt user profiles are almost always accompanied by, if not caused by, file system corruption.  Seeing this is an indication that you should perform a disk check (chkdsk) with the /R or "Repair / Recover" option...  See:

 

"How to perform disk error checking in Windows XP"

  < http://support.microsoft.com/kb/315265 >

 

HTH,
  JW
