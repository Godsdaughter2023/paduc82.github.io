Computer-> Manage -> Device Manager -> Network Adapter ->NVidia -> Power Management, then unclick all boxes.

I think when the computer went to sleep, it turned off the adapter, and when the computer awakened, the adapter stayed sleeping...just a hypothesis.

Anyways, since I made the change, I've had no problems.
