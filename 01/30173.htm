quilnux say:

Managed to figure this one out. My Video's folder resides on my V:\ drive (space reasons). It was moved properly (using the location tab on the video folder in my profile). However, it was originally moved to the Z:\ drive which was later renamed to the V:\ drive in my storage pool. I didn't "move" this special folder to the V:\ drive when it was renamed so every time I tried to download a video in the Xbox Video app it was trying to download to the non-existent Z:\ drive. By updating the registry using the article here: http://www.winhelponline.com/blog/do-not-move-special-folders-to-the-root-drive/ I was able to resolve this problem.

¡firedog say:

Well, I doubt the Charter bloke really knew what he was doing. The error message shows that you're trying to connect to an IMAP server (that's what port 993 means) using the POP3 protocol. That won't ever work. IMAP servers use the IMAP protocol, and when the connection is secured, the port has to be 993. 

Windows Live Mail always tries to download messages from the server before sending any waiting messages, so now you're a bit stuck. It fails to connect because of the protocol fault, so it never got to test sending a message.

Once you've set up an account to use POP3, you can't change it to using IMAP and vice versa. You have to start from scratch setting up a new account. Please make sure when you do that you get the settings absolutely right. We still haven't established whether you can send a message using mobile.charter.net over port 587. 

You mention your phone; Windows Live Mail doesn't work on phones! Charter set up these servers for mobile devices, but they should work for static computers as well, even when you're not connecting to the Internet via Charter.

What is the file name for WLM files and where is it stored? - Microsoft Community

paul r.soucy say:

What is the file name for WLM files and where is it stored?

I'm trying to backup my WLM messages to an external drive. WLM program wants to backup to a folder, I want to backup to an external hard drive. Is it possible?
¡firedog say:
Each mail message is stored as an individual text file with the extension .eml. The recommended way of backing up mail messages is to use the program's export function, but it depends what you want to use the backup for.

For a successful export, follow these guidelines:
  1. Set Windows Live Mail to Work offline before starting.
  2. Create a temporary folder on your desktop to contain the exported messages.
  3. In the export wizard (Alt-F E M), select the temporary folder and select All folders.*
  4. When it's finished, check that the number of files in each folder within the temporary folder is one more than the number of messages in the corresponding WLMail folder.
  5. Copy the temporary folder to its final destination.
    If you use a non-NTFS drive to do this, you will see a warning that some file properties will not be copied with the files. This can be ignored.

*  If you want to export selected folders only, select each one that contains messages. Sub-folders are not automatically selected by default.

Hi,

Thank you for posting your query on Microsoft Community.

I understand that you are unable to use the keyboard after installing Windows 10. We will help you to resolve the issue.

I suggest you to plug in an external keyboard and check whether you are able to access all the keys.

I also suggest you to uninstall and reinstall driver.

  2. In the list of devices, you should see the keyboard drivers.
