a. user say:

Microsoft offers academic pricing on many of its products here, but not on Windows.

Replacing the processor on a Windows 10 computer doesn't deactivate your Microsoft account. It means that Windows 10 is no longer activated on the computer. Since you can't find your Windows 8 license key, you lost your ability to upgrade to Windows 10 for free. And you probably don't have a backup of your Windows 8 installation, or you wouldn't be asking the question. Looks like you painted yourself into a corner.

I would try this before buying a Windows 10 license: Call Microsoft product activation and ask them to cut you a break. Microsoft is really desperate for Windows 10 to succeed, and how would it look if they treated a poor student badly?

Most people who replace a processor also replace their motherboard. Not a rule of course, but if you're going to the expense of a new CPU and cooler, you want the newest generation, which typically uses a different socket. When you add up the cost of a new motherboard, processor and a license for Windows, it's often a better deal to just buy a new computer.

Check if you need to update drivers for your Bluetooth adapter from Device Manager.

 

a)      Click on Start, type Device Manager and enter.

b)      Click on Other Devices and find your Bluetooth device and see if there are any warning symbols. If yes update the drivers.

c)      Right click and click on Properties, and go to Driver tab and click on Update driver.

You may also try below steps to ensure drivers for the device has been removed from Device Manager:

a)      Click Start -> type ‘cmd’ without quotes in the search box.

b)      Right-click Command Prompt, select ‘Run as administrator’. If you are prompted for an administrator password or confirmation, type your password or click Continue.

c)      At the command prompt, type the following commands (press ENTER after each):

 

SET DEVMGR_SHOW_NONPRESENT_DEVICES=1

g)      Click the Driver tab, and then click Uninstall.

h)      If there is a Delete the driver software for this device box, check it (make sure you have internet connectivity so that you may reacquire these drivers). Read and Confirm Notification, click OK.

i)        Exit Device Manager, and restart your computer.

lemp say:

I don't know of any registry value, but you can use the following sequence of key presses to "Save as JPG."

ALT  

F

V

J

The F key may be pressed simultaneously with the ALT key or after it.

You can reduce the number of keystrokes to just ALT plus a number key using the following method, adapted from the one described HERE.   The setup is much easier using a mouse, but can be done using the keyboard alone.  I've shown both mouse commands and keyboard commands to add a "Save as JPG" icon to the Quick Access Toolbar.

Open Paint

Mouse Keyboard Equivalent
Click File Menu (upper left corner) Press ALT+F to open File Menu
Hover mouse over "Save as" to open list of file types Press V to show list of file types
Right-click on JPG to open context menu Use TAB or Down Arrow to highlight JPG and then Press SHIFT+F10 to open context menu
Click "Add to Quick Access Toolbar" Press A to "Add to Quick Access Toolbar"

At this point, your setup is complete. 

With Paint open, press the ALT key.  This will show the keys associated with various actions.  In my case, the "Save as JPG" icon that was added to the Quick Access Toolbar was the fifth icon (after Save, Undo, Redo, and Print).  Thus, to "Save as JPG," I would press ALT+5 (again, you can press the 5 key either with or after you press the ALT key).
