    7e4dca80246863e3.automaticDestinations-ms = Media Player (Frequent, and Pinned)

    These last 2 I have not seen before, so one of these should be the file that contains the data for the Solid Edge program. Determining which file is a 2 step process.

     


    Step 1. Identify    :

    a. Open windows explorer to the automaticDestinations folder.

     


    b. Open an instance of Notepad, Drag one of these unknown files and drop it in the open Notepad window. This will show the contents of that file.

     


    c. Most of the text in the file will be raw code and not readable. Scan the file carefully and you should see paths to files that have been opened with the program that the file is associated with. Once you have determined the program, close the Notepad window.

     


    Step 2 Confirm    :

    a. To confirm that this is the correct file for that program, drag that file out of the AutomaticDestinations folder to another folder, temporarily.

     


    b. After the file is moved, right click the pinned taskbar icon for that program and you should see that the JumpList has completely disappeared, except for the default items at the bottom of the right click menu (program name and unpin option).

     


    c. Go back to the folder where you moved the file and delete it.

     


    You should now be able to rebuild the JumpList for that pinned program using the previous steps.

     


    Let us know if this works for you. Please post back and let us know which programs those 2 unknown files are associated with so that we can add that info to our database.
