   - SSDP Discovery

   - UPnP Device Host

If the hard drive is failing, it may be too far gone for cloning, but you won't know this until you try. At this point you should do hardware troubleshooting, starting with the hard drive.

If you didn't receive XP installation CDs, contact Lenovo tech support and purchase them now. Otherwise yes, you can use a generic OEM XP installation disc that matches the version of XP originally installed and use the Product Key on your laptop. Or you can purchase a full retail copy of XP which will come with its own Product Key. You must then download/install all the Lenovo drivers and laptop utilities from their website. There are a lot of them.

considerate_guy say:

Unfortunately it appears that WLMM does not retain the audio when slowing down or speeding up your video.  Prior versions of Windows Movie Maker *did* keep the audio. In WMM 2.1 under XP, you would get the "chipmunk" effect in your sound.  In WMM 6.0 under Vista, it would try to keep the same pitch but speed it up.  This would work best when doubling the speed, not so well for higher speeds.

If you want, you could install WMM 6.0 on Windows 7.  It can coexist on the same PC so you could use it to speed up your video, save it as a movie, then use that file for further editing in WLMM.

Alternatively, you could use software like Audacity to speed up the audio (assuming you could extract it from your video into a separate file with another program) then bring it back into WLMM as your added music.  You'll have to try to synch them up which might be hard.

The last option is to suggest this as feedback to the WLMM team as a desired improvement to the program.

How do I Calibrate Touch?

1.Open Search by swiping in from the right edge of the screen (or if you're using a mouse, pointing to the upper-right corner of the screen and moving the mouse pointer down), and then tapping or clicking Search  

2. Type Control Panel in the Search text box, and then select Control Panel from the returned list.

3. In Control Panel and select Tablet PC Settings. 

4. From Tablet PC Settings, select Calibrate.

5. Follow the on screen calibration for Pen input or Touch input settings.

How do I adjust Touch settings?

1. Open Search by swiping in from the right edge of the screen (or if you're using a mouse, pointing to the upper-right corner of the screen and moving the mouse pointer down), and then tapping or clicking Search.

2. Type Control Panel in the Search text box, and then select Control Panel from the returned list.

3. In Control Panel select Pen and Touch.

4. From the Pen and Touch menu, select Touch.

5. Select the touch action that you would like to change and click Settings.

If the issue still persist, then try method 2.

Note: Make sure that you have downloaded the latest driver before uninstalling the current version.

1. Press Windows key + X key, task list will come up and select Device Manager.

2. Device Manger window will be open.

3. Locate the Display adapter icon and expand it.

5. Follow the onscreen instructions to uninstall the current version of the driver.

6. Restart the computer.

7. Open the download folder which contains the setup file of the latest driver that you have downloaded.

8. Double click on the setup file and follow the onscreen instructions to re-install the latest display driver.
