I see that this is your first pot in this forum, welcome to the Microsoft Community.

This issue could be caused due to some incorrect settings or due to a driver issue. I would suggest you to try these methods and check if it helps:

First, I would suggest you to make sure that Bluetooth feature is enabled in the laptop. If the Bluetooth is enabled, you will be able to see the Bluetooth indicator below your touchpad.

If the Bluetooth indicator is on, then I would suggest you to check if you are connecting the Bluetooth devices as mentioned in the 68th page of the user manual:

http://dlcdnet.asus.com/pub/ASUS/nb/N550JV/E_eManual_N550JA_JV_LF_VER7914.pdf

Method 1:

If the issue persists, I would suggest you to run the hardware and devices troubleshooter which can automatically detect and fix most of the incorrect settings which could cause issues with hardware devices. Refer to these steps:

  1. Press Windows key +F to bring up the search Charms bar.
  2. Select settings in the drop down menu below search and type Troubleshooting.
  3. Click on Troubleshooting and select the view all option on the left pane.
  4. Click on Hardware and devices troubleshooter and follow the prompts.

Method 2:

If the troubleshooter didn’t detect any incorrect settings, then it is possible that the driver wasn’t installed correctly. I would suggest you to reinstall the Bluetooth driver and check. Refer to these steps:

  1. Press Windows key + X, select Device manager.
  2. Expand the “Audio input and output” section and locate the audio adapter.
  3. Right click the device and select “Properties”.
  4. Click on the driver tab to check the current driver installed for that device.
  5. Click on the uninstall button to uninstall that driver.
  6. Now, restart the computer. Download and install the driver from the following link:

https://www.asus.com/Notebooks_Ultrabooks/N550JV/#support_Download

If the above steps didn’t help, I would suggest you to check if the Bluetooth service is enabled and running. Refer to these steps:

  2. In the services window, locate the “Bluetooth support service’ and check if it is running.
  3. If not, right click on the service and select ‘Start’.
  4. Check if the issue persists.

If the issue persists, I would suggest you to boot the computer in a clean boot state and check. Booting the computer in a clean boot state can reduce software conflicts and help identify the root cause of the issue. You can refer to this article for more information:

 

http://support.microsoft.com/kb/929135

 

Note: After troubleshooting, reset the computer to start as usual. Refer to the step How to set the computer to start as usual after clean boot troubleshooting under more information.
