steve winograd say:

Some computer makers put a Factory Image or Recovery partition on the hard disk of new computers.  By pressing a special key combination when the computer starts, you can use it to restore the computer to the way it was when it was new.

I recommend that you remove the drive letter from the Factory Image partition to prevent Windows 7 from accidentally damaging it:

1. Click the Start button.
2. Type diskmgmt.msc in the Search box and press Enter .
3. In the Disk Management window, right-click the Factory Image partition.
4. Click Change Drive Letter and Paths .
5. Click Remove .

About MSDN or TechNet Product Keys

They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. The main purpose is for evaluation purposes. The great thing about them, unlike trial Microsoft software, MSDN or TechNet keys don't expire. Because the agreement under which the subscription is provided is a single license, none of the software should be distributed outside of it. Even though its $50,000 worth of licenses, it is for one person only to use and no one else. Unfortunately, regardless of the licensing terms, persons still abuse the program, either giveaway product keys or resell it on auction sites. Microsoft licenses it in good faith that customers won't do so, but I guess human nature wins out.

 

In your case, what probably happened is, you bought an MSDN licensed key, which carries up to 10 activations unlike full packaged retail licenses which only carry 1 activation. The person who sold it to you probably sold it to 10 other persons. Somewhere along the way, one of those persons might have installed it on a second system, activated it, because it went past 10 activation threshold, Microsoft detected it that it was being abused and blocked the key from further use.

    john rubdy say:

    Hi,

     

    Parallel and Serial ports are not supported in Windows 8 or Windows 8.1. A USB converter or optional network accessory is needed.

     

    Let's analyze and identify the cause of the issue. Perform the steps provided and check.

     

    Method 1: Run the Hardware and Device Troubleshooter and check if it helps.

     

    a. Press “Windows key + F”

    b. Type Hardware and device then click on Settings

    c. Click “Find and fix problems with the device”

    d. Click Next and follow the instructions.

     

    Method 2:

     

    Step 1: Completely uninstall the parallel port drivers from the computer and then try installing the printer drivers and check. Follow these steps:

     

    a. Press Windows key + R

    b. In run window type “devmgmt.msc” and press Enter

    c. Locate the parallel from device manger

    d. Right click on it and select uninstall

    e. Reinstall the latest printer drivers and check if it helps

     

    Step 2: Run the pager programming applications in “Compatibility mode”

    Make older programs compatible with this version of Windows

    http://windows.microsoft.com/en-US/windows-8/older-programs-compatible-version-windows

     

    Hope this information helps. Reply with the status so that we could help you further.

    eliyas yakub [msft] say:

    That's not true. Drivers for parallel and serial ports are still included in the Windows 8.1 for compatibility reasons.

    parport.sys

    serenum.sys

    serial.sys

    drivers are still in Windows 8.1. We haven't explicitly removed any software support. So if you have a device and application working on Windows 7 and 8, it should work on 8.1.

    -Eliyas
