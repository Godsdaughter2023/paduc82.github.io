The Control panel item Backup and Restore (Windows 7) is, as you thought, the correct place for you to select the link to Create a system image [on the left-hand side of the window]. 

  • The poorly chosen name was introduced late in the Windows 10 development process and reflected the fact that this item did exactly what its Windows 7 predecessor had done. 
  • In earlier development versions, backups had been run by different items within Control panel and I think people complained that there was no central point to go to.

The system image you create is a system image of your Windows 10 installation and it can be restored correctly.  I have made & restored system images using this utility several times on Windows 10 and it all works.

The only difference you might find with Windows 10 system images is that not all third party imaging facilities have been updated to be able to work with Windows 10 system images.  For example, my Acronis True Image 2015 utility can restore Windows system images I made using this Control panel facility in Windows 7 as well as system images made using Acronis TI itself.  But it cannot restore their Windows 10 equivalents, such as the ones you have just made, because the files at the core of the image have different file extensions and presumably slightly different internal data structures.
