a. user say:

Hello,

LanguageComponentsInstaller is associated with Windows 10 to support different languages. When making it available in other languages, Windows will create a different set of program files and a task in order to display user interface elements in another language.

If it is preventing your system from shutting down, it is possible that a recently installed language pack is incompatible. May we know if a recent language pack is installed?

If it happened after an update, you may restore your PC from a system restore point. Restoring from a restore point won't affect your personal files, but it will remove apps, drivers, and updates installed. Here's how to restore from a system restore point:

  1. Select the Start button, then type control panel, then select Control Panel from the search results.
  2. Search Control Panel for Recovery.
  3. Select Recovery > Open System Restore > Next.
  4. Choose the restore point related to the app, driver, or update, and then select Next > Finish.

Note: If you are not seeing any restore points, it means that system protection isn't turned on. To check, go to Control Panel, search for Recovery, and then select Recovery > Configure System Restore > Configure, and make sure Turn on system protection is selected.

If restoring from a system restore point does not help, you may disable the service or task by using the System Configuration utility. Here's how:

  1. Press the Start button, then type msconfig, and select System Configuration from the search results.
  2. On the Services tab of the System Configuration dialog box, search for the task, then click to clear the check box for the problem item.

I've managed to find a fix after hours of searching!

You'll need to update an Internet Explorer record in the Windows Registry in order to run Microsoft Money 2003.

I'll assume that you're a novice (No Offence Intended) and guide you through the steps, one by one.

1. We'll need to open the registry editor, to do this you'll need to search for 'regedit', this can be achieved in the Windows 10 search.

2. Once open, you'll need to navigate your way through the registry, following the below path.

[Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer]

3. Click once on the Internet Explorer folder and you'll see a list of items on the right.

4. Change 'Version' so that the last digit is a zero.

For example, my current version number is [9.11.10240.16384], I simply changed it to [9.11.10240.0].

5. Once saved, you'll need to restart your PC.

From this point on you should be able to run Microsoft Money 2003 with no problems.
