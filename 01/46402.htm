lemp say:
This is a feature, not a bug.  Although 32-bit versions of Windows have a 4 GB address space for memory, some of those addresses are used for various hardware devices installed in your computer.  Any "memory address" used for a piece of hardware is not available as an address for your RAM.  Thus, although your system can install 4 GB of RAM, the amount of RAM actually addressable (i.e., usable) by Windows will be somewhat less. 

With 4 GB of physical RAM installed, the usable RAM often is around 3.2 GB, but perhaps you have a video card that has a lot of built-in RAM (which needs the address space, even if it has its own physical memory chips) or perhaps you have several PCI or PCIe cards that use memory address space.

If you want to have more RAM available for Windows, you'll need to switch to the 64-bit version of Windows 7.

--------------------------------------------------------------- How To Lock a Folder With a Password in Windows 10

1. Right-click inside the folder where the files you want to protect are located. The folder you want to hide can even be on your desktop.

MORE: How to Change Your Password in Windows 10

2. Select "New" from the contextual menu. 

3. Click on "Text Document."

4. Hit Enter. It doesn't matter what the file will be named. You can delete this text file once the lockable folder has been made.

5. Double-click the text file to open it.

6. Paste the below text into the new document:

cls

@ECHO OFF

title Folder Locker

if EXIST "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}" goto UNLOCK

if NOT EXIST Locker goto MDLOCKER

:CONFIRM

echo Are you sure u want to Lock the folder(Y/N)

set/p "cho=>"

if %cho%==Y goto LOCK

if %cho%==y goto LOCK

if %cho%==n goto END

if %cho%==N goto END

echo Invalid choice.

goto CONFIRM

:LOCK

ren Locker "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}"

attrib +h +s "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}"

echo Folder locked

goto End

:UNLOCK

echo Enter password to Unlock folder

set/p "pass=>"

if NOT %pass%==Your-Password-Here goto FAIL

attrib -h -s "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}"

ren "Control Panel.{21EC2020-3AEA-1069-A2DD-08002B30309D}" Locker

echo Folder Unlocked successfully

goto End

:FAIL

echo Invalid password

goto end

:MDLOCKER

md Locker

echo Locker created successfully

goto End
