gerry c j cornell say:

BootTime   83115

A boot time of 83 seconds is much to be expected if Windows 7 is installed on a conventional hard drive. You need a solid state drive to get a significantly lower boot time.

It’s not an error as such. All Event Viewer reports are given a level, which can be:
Information
Warning
Error
Critical
or Verbose.

Event ID: 100 reports are no exception. Warning changes to Error at about 40 seconds. Error changes to Critical at something like 110 / 120 seconds.

All Event Viewer reports are subject to system blips. When you look examine Event 100 reports look at the pattern. Do not be over concerned if you have the odd report of a much longer time as these can be caused by programmes and the system updating.

General notes on Event Viewer
http://www.gerryscomputertips.co.uk/syserrors5.htm

If you plan to try to reduce the boot time concentrate on changing programmes to load on demand instead of loading automatically when the computer is booted. Leave all Service startup types at their default setting as changes tend to lead to system error.
