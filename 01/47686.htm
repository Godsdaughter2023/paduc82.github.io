hula_baloo say:

 While your PC is powered on, the clock shouldn't have to rely on the CMOS battery, as you point out.  However I believe it's the motherboard that does the real time updating.  Since you've properly checked for malware, it would seem that the remaining possibilities are a corrupt file somewhere or some problem with your motherboard.  Dell makes great hardware in my experience, so I guess I'd suggest you run SFC. 

 

 To run SFC, go to the Start menu and type in CMD in the search box.  When you see CMD.EXE in the programs list, right-click it and click Run as Administrator.  Then from the command prompt type sfc /scannow.  If that finds a corrupt Windows file, it will correct it.  If it doesn't, you've got two options as far as I can see.  (1) Restore to factory settings (which won't help if it's a hardware problem) or (2) call Dell's support folks and see what they have to say.

 

 Good luck!
