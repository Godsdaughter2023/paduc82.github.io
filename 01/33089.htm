petercon [msft] say:

Hi. Many Windows fonts were moved into optional features in Windows 10 because they are primarily intended for particular languages, and don't provide real benefits to all users. For example, most Japanese users don't need to have dozens of Thai fonts taking up space on their system.

These optional features are associated with particular languages and get installed automatically if you add the associated language in your user profile (in Settings). For instance, if you enable a Thai keyboard, the optional Thai fonts feature will be automatically installed from Windows Update. (You need to be online for this to work, of course.)

But you can also install any of the optional features manually without changing any language settings. So, for instance, if you happened to need one of the Thai fonts, you can install the optional Thai fonts feature without needing to have a Thai keyboard enabled and have Thai in your language preferences.

To install the optional features manually, do the following:

  • Click the Start button.
  • Click Settings.
  • Click on System.
  • Click on Apps & features.
  • Click on the link, Manage optional features.

The view that appears will list the optional features that are already installed. To install any other optional features.,

  • Click on the "+" icon next to Add a feature.
  • Scroll through the list to find the optional feature you'd like to install. For the optional font features, if you hover over the item, you'll see additional details about which fonts are included in that feature.
  • When you find a feature you'd like to install, click on that item, and then click on Install.
  • Repeat this for as many optional features you'd like to install. When done, click on the back arrow in the upper left corner (upper right corner for Arabic or Hebrew Windows).

You should see the features you installed in the list of installed optional features, or they may appear as in progress of installing.

I hope that helps.

Peter

petercon [msft] say:

Hi, Hareesh.

I understand that unexpected changes can be disconcerting. The reason for the change has to do with newer device form factors, which include devices with very limited storage. Just as you don't like having fonts missing, you also wouldn't want to buy a tablet and discover that you can only use a fraction of the advertised storage. So, rather than pre-installing all the Windows fonts on every device (around 500 MB), most of which a given customer will never use, we're trying to be a little smarter about how much of the available storage we take up. Specifically, we try to automatically give customers the fonts that they are likely to use, while also giving them manual access to all of the fonts. We can't predict which fonts every customer wants with 100% accuracy, though, which is why enabling them to choose manually is important.

We use language settings as an indicator of which optional fonts you are likely to use and want. Since people select fonts for formatting text while editing documents, the keyboards you enable are the key indicator. If you have Hindi, Telugu or other languages of India in your language profile, which you would do to enable keyboard input in those languages, then you should automatically get the optional fonts for those languages. But if you don't want to enable keyboards for these languages and have the languages in your language profile, you can always install the optional components manually, independent of your language settings.

Also, Windows 10 still includes fonts for every language in the base image, so you should not ever have to see boxes.

I hope that helps clarify things.
