a. user say:

Hi Elias,

 

Thank you for posting your query on Microsoft Community.

 

Software Distribution Folder contains the history of downloaded Windows Updates. Deleting files in these folders will cause no harm but will erase the history of your updates. Hence no data about the previous Windows Update will be available. It is always recommended you first rename this folder to “Software Distribution.old” and then restart. A new folder named “Software Distribution” should be created automatically. Once its checked you can delete the old folder.
