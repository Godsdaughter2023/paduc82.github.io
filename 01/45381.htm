¡firedog say:

I can see the subjects and new emails seem to come in. Attachments seem to be there. I'm not easily able to go to different accounts.

If I understand you correctly, you see the header bar with the subject, date sender and so on, and attachments, but the reading pane is blank. The program is sluggish when you choose a different message to read, whether it's in the same folder or a different one.

These are the classic symptoms of having too many Temporary Internet Files. Each time you select a message for viewing, Windows Live Mail creates two temporary files for Internet Explorer to render. If there are too many files in the folder it uses to store these files, they can't be created and there's nothing for Internet Explorer to render, so you see a blank preview pane.

The solution is to delete your Temporary Internet Files. Close down Windows Live Mail and all your browser windows. In Control Panel open Internet Options. On the General tab, click Delete. Then select the top two boxes and click Delete. When it's finished, re-launch Windows Live Mail.
