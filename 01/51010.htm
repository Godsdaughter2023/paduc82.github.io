Hey, I have a VHD file on my D drive that is taking up 188Gbs. What - Microsoft Community

anonymous_24 say:

Hey, I have a VHD file on my D drive that is taking up 188Gbs. What would happen if I deleted it?

All of the sudden, I get a notification that there is little to no space left on my hard drive (D) and I haven't installed anything on this computer in a very long time.  Under the system image file is where this VHD file is and it's 188Gbs.  Someone told me exactly what it is (technically) but I really don't understand it.  They also said that that's an outrageous amount of space for a VHD to take up.  Can I delete it?
hairyfool say:

VHD is a Vitual Hard Disk and is normally generated by a virtualisation program such as Virtual PC, VMWare etc. These allow you another PC as a program on your main PC.

 

When a Virtual PC is set up you have to create a single file which contains all the files in a PC. However even though you might want the disk to be similar in size to a real PC for most the actual space taken up is much smaller. Normally when you define a virtual disk you may set it to be 100Gb but you normally specify the actual file to only be as big as the information in it.

 

As to can you delete it, if you have no idea where it has come from probably yes. However to be sure it is possible to mount a VHD as if it is another disk in your computer. That way you can check and see what the contents are.

 

Right click on Computer and select Manage.

 

Click on Disk Management and wait for the utility to load.

 

After it has loaded right click on Disk Management and Attach VHD. Browse to where the file is and select it.

 

Now in Computer you should have an extra drive which is the VHD file. Navigate down and see if you can work it out where it is from.

 

Following the same sequence you can also Create VHD the defaults for which are Fixed Size and all free space on the disk, sounds a bit like what you have.
