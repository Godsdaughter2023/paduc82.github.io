basith m say:

Hi Ivan,

Windows Audio Device Graph Isolation is an audio file for recent versions of  Windows that prevents other applications from modifying audio-related content  and plug-ins. The application also serves as the audio engine for Windows. The  process typically launches when you start your computer and is nothing to worry  about if you see it pop up in the Task Manager.

Windows Audio Device Graph Isolation runs under the process name "audiodg.exe"  and is a component of Windows 7 and Windows 8.1 operating  systems. The file runs the audio engine for Windows, which allows third party  code from vendor audio applications to run sound in the computer system. The  process is not crucial to basic computer functions.

If this is the case, you can at this point go to websites and get drivers, or take it back to Best Buy and make them do it for you for free.
