burrwalnut say:

Yes, of course, but firstly you need a) to back up all your personal data to an external device and b) the Vista DVD.

Now you can reinstall Vista, remove all external devices, boot from the DVD. You may have to change the first boot device to DVD/CD in the BIOS. To do this, when you switch on look for a Setup message on the screen indicating which key to start tapping. Look for a section in the BIOS that contains the boot sequence and change it to DVD/CD.

The install starts with a message asking you to ‘Press any key to boot from CD/DVD’ and then it loads files, you select the Language, Time/Currency, Keyboard. Click Install now, type the Product key but don't allow Activation, click to accept the License and click Custom for a clean install. If you want to delete/create/format partitions, click 'Drive Options Advanced' and set it up accordingly.

Then the installation starts and will reboot several times, so ignore the message to Press any key to boot from DVD/CD. Lastly, download all the missing Microsoft updates to fully secure the system.

Good luck.

Switch to a Local account, the switch back to a Microsoft Account:

You can choose between whether it uses your Microsoft Account or a normal user account by going to all PC settings, to the Users section.:
1) Just move your mouse to the bottom-right corner of the screen till the side menu pops up, and click on Settings (gear icon).
2) From there, click on the "Change PC settings" link on the bottom-right.
3) Now you should be on the PC settings full-screen menu. On the left, go to the "Users" section.
4) On the right side, it will show your current account with a button that says "Switch to a local account". Clicking on this button and following the steps will enable you to be able to log into Windows as you do with all other versions of Windows like Windows 7, Vista, XP, etc.

greg carmack - mvp 2010-2020 say:

Create fresh installation media using the Media Creation Tool

choice to Create Media for another PC. 

With media inserted, power up PC while pressing the HP BIOS Boot Menu Key

which is ESC, choose UEFI verison of media if offered, boot into second screen to choose Repair Your Computer. 

Failure to boot the media is almost always a user failure due to bad timing, but if it persists you can also try booting into BIOS setup/UEFI firmware also via the ESC, F10 or F2 keys.  Browse through the tabs to find Reset to Default, do this and then Save Settings before exiting, reboot and try again.  You may need to also adjust the SATA controller which should be optimally at AHCI but might be installed to less optimal SATA setting.

From Advanced Troubleshoot options in Advanced Startup Options in Windows 8 or 10 try Startup Repair or a System Restore.  If these fail see if you can get into Safe Mode with Networking via Startup Settings, to complete the Windows 10 Install Integrity Checklist. If not try the repair commands via the Command Prompt also in Advanced Troubleshoot options. 

There are other fixes to attempt in Troubleshooting Windows 8 and 10 Failure to Start - Microsoft Community including the bootrec commands.  If it becomes apparent the hardware is not working properly you can also do the hardware tests listed in the beginning. 

If all fails then you can try to Reset Your Windows 10 PC which will be a much better install since it will leave out the HP bloatware which cripples Windows and makes HP the worst install in the industry. Or since you already have the media do the vastly superior Clean Install Windows 10 .

At any time you can Rescue Files when Windows Won't Start.

Feel free to ask back any questions and report back results.  These situations are always resolved where there is close cooperation and complete thoroughness in following the steps.  We will help you as much as you need.  The worst case might be that you need to reinstall but the reward for sticking it out is the best install of Windows one can possibly have if you choose the  Clean Install Windows 10.
