shinmila h say:

Hey charlotte1993,

Please follow these steps that may help in bringing back the battery meter icon.

1. Shut down your computer.
2. Detach the battery and reattach it.
3. Boot the computer.
4. Once booted completely, see if the battery icon appears.
5. If it doesn’t, right click the system tray and select properties.
6. Select the "Notification Area" tab.
7. Place a checkmark at "Power".
8. Click on the Apply or OK button.
9. The Battery Meter should show up in the tray.

If it still doesn’t, run the following command from command prompt.

1. Go to start and type cmd.
2. Right-Click on cmd and select “Run as Administrator”.
3. Type powercfg.exe /globalpowerflag off /option batteryicon and press Enter.
4. Type powercfg.exe /globalpowerflag on/option battery icon and press Enter.

Check if the battery icon comes back.
