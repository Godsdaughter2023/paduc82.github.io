trekdozer say: If you don't have a DVD drive you can use an external DVD drive, or use a USB flash drive to transfer and install programs.


For a flash drive:

Transfer File From CD To Flash Drive Using a Different Computer

1. Insert the CD/DVDand also connect a USB drive

2. Open Windows Explorer - click Start then click Computer.

3. Within the Windows Explorer window, open the drive that contains the files. For example, if they're on a CD or DVD open that drive or the letter of the disc drive.

4. Within the drive that contains your files select them all, then right click the files and click Copy.

5. Navigate to a folder on the flash drive, right click on it and paste the files to that folder.

6. You can then use the flash drive to install your program by connecting it to the computer without a DVD drive, see below.


More info, Copy and paste a file:

http://windows.microsoft.com/en-US/windows7/Copy-and-paste-a-file





How can I install software from a USB Flash Drive?

1. Insert the USB Flash Drive

2. Open Windows Explorer - click Start then click Computer.

3. Within the Windows Explorer window, open the flash drive that contains the installation files.

4. Within the drive that contains your files, locate either a setup or install file. Double-clicking on this file should start the installation for the program, game, or utility. If you see multiple setups or install files, try to locate the Application file or double-click each of setup or install files until you find the file that starts the installation.

steve winograd say: If I understand your question, you're referring to the drive letter that Windows has assigned to your USB external hard drive.  USB ports don't have driver letters assigned to them, but hard drives do.

To change the drive letter from F to E:

1. Press WindowsKey+R .
2. Type diskmgmt.msc in the box and press Enter .
3. In the bottom part of the Disk Management window, right-click the USB hard drive.
4. Click Change Drive Letter and Paths .
5. Click Change .
6. Select E in the menu and click OK .
