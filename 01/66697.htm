ggabunia say:

Recovery Partitions

Hello. bought a Laptop with windows 8.1 and upgraded it to windows 10.
Windows 8.1 had 2 recovery partitions  (600mb and 15gb). I've read somewhere that windows 10 doesn't require recovery partitions. So, after the upgrade, are those recovery partitions still of windows 8.1? If I were to remove them, would I still be able to use Windows 10 recovery options, or does Windows 10 actually use those partitions? 

marks1964 say:

Hi,

Think about what you may want to do in the future BEFORE deleting these partition as it can cause you problems.

The two hidden partitions you are referring to are for restoring the laptop to it factory (Windows 8.1) condition.

The 600 MB partition contains the SYSTEM files (for both Win 8.1 & Win 10) and this is necessary for computer to boot. If deleted the computer will not boot.

The 600 MB partition ALSO contains the tools to reset the laptop to its factory condition - Windows 8.1.

The 15 GB partition contains the backup image file for Windows 8.1.

Now if you wanted to go back to Windows 8.1 for any reason, you would need to have both of the hidden partitions still intact.

Kind regards,

Mark

ggabunia say: Thank you. But I already made a recovery drive on a flash drive, and I think it copies the content of those partitions, doesn't it? If I need to  revert to windows 8.1, I'll use the flash drive.
The thing is, I made the recovery drive AFTER I updated to windows 10. The manual said that after I made the recovery drive, the application would ask me whether I wanted to keep the recovery partition or remove it. But on windows 10, it did not. That's why I'm asking whether those partitions are necessary  for windows 10 in some way, besides reverting to windows 8.

And are you sure the 600mb recovery partition is for booting? There is another, 300mb Partition called Healthy (EFI System Partition). I thought that one was for booting.

marks1964 say:

Hi,

To answer your questions in order.

1) The USB flash drive is for Windows 10 only, not for Windows 8.1 and if you attempt to reset the computer using the USB flash drive you may wipe off the hidden Windows 8.1 recovery partitions.

Windows 10 does not have or make the recovery partitions, they were made by the computer manufacturer and as such  Windows 10 is not able to read those hidden partitions.

2) The manual refers to the computer when it was a Windows 8.1 and that has nothing to do with Windows 10.

3) The 300MB EFI System Partition is to do with the UEFI and that replaced the Basic Input/Output System (BIOS). Your laptop has is the newer GPT type of partitioning as apposed to the old type of MBR (Google it for more info).

When you did the upgrade; if you booted into Windows 8.1 and then did the upgrade from within Windows 8.1, your hidden recovery partitions should still be intact on the internal hard drive.

However, you cannot access them from within Windows 10 and you will need to follow the computer manufacturer's special instructions on how to return it to Windows 8.1.

Kind regards,

Mark

FOR OTHER READERS.

From within Windows 10, here is how to see if the partitions are still there:

1) RIGHT click on the Start button.
2) LEFT click on Disk Management.

You will now be able to see all of the partitions, including the hidden ones.
