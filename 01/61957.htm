Here are other options you can perform to remove a printer in Windows 10:

tom a - windows virtual desktop team [msft] say:

Hello

Have you checked your storage location settings? You could try updating that to another drive, making sure the current user you are logged in with has permission to write to that drive. Here's how you check storage permissions:

To check current picture save location:

1) Click Start button

2) Click Settings

3) Click System

4) Click Storage

5) Under "Save locations" note the drive where the "New pictures will save to:"

6) Try changing to your default system drive (usually c:\)

You should be able to find the Camera roll in the <username>\Pictures folder from the drive location from step 5, if it's not on your default OS drive (maybe changed to another location like D:\) then it would be something like D:\<username>\Pictures.  You'll want to locate that Pictures folder and make sure the currently logged in user has permission to write to that folder. Else, if it's not on default C:\ drive you might try changing back to that drive. You can read up more about file permissions and how to check them here:

http://windows.microsoft.com/en-us/windows/what-are-permissions

Tom
