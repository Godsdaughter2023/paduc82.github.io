jollyfred57 say:

Do files deleted on an external drive ever get copied to the root drive via the recycle bin?

I've read several threads (and now understand) about how files are deleted from various drives in Windows - but want to know the following: if a deleted file from an external drive shows up in the root drive recycle bin (which it does for me), does this mean that the deleted file has been transfered to the root drive in the same way as copying or saving it to the root drive?

Also, some of the threads mention that you can see 'recycle bins' on all drives if you unhide them in Windows Explorer - my folders are set to unhide and yet I can't see a recycle bin/drive/file anywhere but the root drive.

I'm using Vista 32bit Home Premium.

david_o - say:

Hello Jollyfred57,

Thank you for using the Microsoft Vista Answers Forum!

It appears that when you delete a file that you have moved the file to the Recycle Bin, but the file has not moved. What actually moves to the recycle bin is the pointer (shortcut) to the actual file so the file appears to have been deleted. The deleted file remains on that particular HDD in a hidden folder called RECYCLER. The space on the drive where the data was stored is tagged "ready for use" and when you store some other data on this drive then the old data is "deleted" by having the new data over write the space where the old data was. The Recycle bin serves as a buffer for the delete command and holds that command in the Recycle bin until confirmed.

 

Thus, when you delete a file on a drive or external HD and send that "deleted" file to the Recycle bin; the file has not been deleted and the space that the file used has not been freed. To actually free up that space on that drive you must go to the Recycle bin (or open the RECYCLER folder on that particular hard disk, which has a folder with a really long numerical name and open this folder) and delete the file once more.

True, other drives have a folder called RECYCLER which is connected to the main Recycle bin. The RECYCLER folder is a hidden folder and Explorer must be set to "show hidden folders" in order for you to see this file on each drive.
Hope this information is helpful.
