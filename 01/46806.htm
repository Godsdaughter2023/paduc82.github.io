What are "Compatability files" and how can i see them in - Microsoft Community

paul kreitz say:

What are "Compatability files" and how can i see them in programs such as Notepad?

When I save .txt files from IE9 to a folder (accessed a web site that provides a link to certain .txt files, click on link, do file > save as > filename.txt) then look for them in an app under Windows 7 Home Premium, such as Notepad, they don't show up.  Go to Windows Explorer, look for the files, they still don't show.  Click "Compatability files" on tool bar and there they are!  How do I save them without getting this runaround, or how do I get all applications to see them?  What are "Compatability files"?
cmdrkeene say:
Compatibility Files are files that were re-directed to when a program tries to save to either (A) a secure location such as the System folder, or (B) when a program tries to save to a location that doesn't actually exist (such as C:\Documents and Settings\Paul\Desktop -- which doesn't technically exist in Windows 7, but bad programs that would otherwise break get silently redirected so that they don't crash.

When the redirected program tries to look back into the protected location, it's quite literally fooled into seeing the files that aren't there -- for compatibility.  Since the files aren't actually there, other programs (or even the user) won't see them unless you look in the place the file really is.

The best way to work around this is to not let programs save to security protected system locations.  I would never recommend trying to circumvent this protection.  For example, it's a good thing your Internet Browser (or sites it loads) can't start adding/replacing files into your System folder.

You can contact your Computer Manufacturer and ask them to send you a set of Recovery Disks.

It may be F10, F11, Alt + F10, etc depending on the Manufacturer.

Ask them for the correct Key sequence.

 

And if you do have a Manufacturer's Recovery Partition on your Hard Drive, you should be able to make your own Recovery Disks from it to reinstall the Operating System.

If you can not find any reference to it, contact the Manufacturer for advice on how to make those Recovery Disks.

 

Save all Data before doing a Recovery to Factory Settings as the Hard Drive will be wiped clean during the Recovery process.
