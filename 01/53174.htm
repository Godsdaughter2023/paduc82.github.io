tryx3 say:

WMP cannot display the information but you can see it in Windows explorer by adding the Comments field to the fields you display in a Details folder view.

Windows Help explains folder views at the bottom of the section for Change folder options.  You will want to add the field to your default views for Music type folders and the same Windows Help section explains that as well.  There are several threads in the forum on this subject so if you need further advice you should be able to search the forum for some.  I have copied my own notes in the subject after the - - - - - line below.

Once you have added the field to Windows explorer then, if you are in WMP and you want to see the Comment field for a track, right-click on the track and select Open file location. 

That's as good as it gets given the absence of any direct method within WMP itself [and this is not going to change as there is no development effort going into WMP anymore].

- - - - - - - - - - -

If you access your folders through C:\Users\YourUserName\... then you can set each folder up as you wish.  Libraries are different - they get set once for each folder type but individual folders cannot be different & actually change all of that type if you alter them. 

You can set up a default layout for each of the folder types [Documents, Music, Pictures, Videos, General] then change any individual folder afterwards if you want [this can be useful as it sets up what you normally want everywhere but still lets you change individual folders to suit your needs].

1  Start off by settings folder types overall.

1.1  In Windows explorer, go to C:\Users\YourUserName then right-click on the Documents folder, select Customise, set it to Documents & set the checkbox for applying it to subfolders as well.

1.2  Do this for videos, pictures & music folders at this top level as well.

1.3  The Links & Favorites folders can usefully be left as the General folder type.

1.4  You now have a folder type set for every folder in the C:\Users\YourUserName folder set.

2.1  Next, find a folder with music in it in Windows explorer.

2.2  Check that it really is set up as a music folder [right-click on the folder name then select Customise].  

2.3  Set up the view how you want it [Details or icons or whatever, sort order, relevant fields visible].

2.4  Then go to the Tools menu [alt, t], select Folder options, View tab & click on Apply to folders.

2.5  Repeat this procedure for each of the other folder types.

3  You now have folder views set for every folder type.  Check that this has worked  by looking at some other folders where you should see that all other folders of each type have changed to match what you have just been setting up.

4  If you want an individual folder to be differently laid out than others of that folder type then, after you have completed the above settings, you can just set any individual folders up how you want. If you change an individual folder's layout but later change the layout for that folder type then that individual folder’s layout will be overwritten by the new settings.

5  If you want an individual folder to be a different folder type from those around it then you can also set this up.  Right-click on the folder name then select Customise and set it to be whichever folder type you want.
