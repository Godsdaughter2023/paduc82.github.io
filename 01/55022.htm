The factory restore process is normally invoked by pressing a Function key (like F11) at computer startup. You can also refer to your computer manual, or the computer mftr.'s website, or contact its tech support to see how to restore your computer to factory condition since each computer mftr. does it differently - sometimes even between models.

These machines also usually have a utility with which to create physical recovery discs in case the hard drive needs to be replaced or the factory restore image is somehow damaged. If you did not create the physical recovery discs and/or damaged the recovery image, contact the computer mftr.'s tech support to order a recovery disc set. deana1988 say: Ok nevermind. 

I found the answer thanks to Ronnie Vernon...

If anyone else wants to know how to fix this, read what he wrote:

The libraries should only be showing the files that are contained in the folders that you add to that library.

Example:

In Windows Explorer, select the Documents Library.

In the right side View window, click the x Locations link at the top/left of the window.

This will open a dialog box showing all of the folders that have been added to that library.

By default, the Documents Library should show the My Documents and Public Documents folders.

If there are any other folders showing that you do not want in that library, select that folder and click the Remove Button.

You can also Right Click the Documents Library, in the Navigation Pane and select Properties. At the bottom of the Properties dialog box, click the Restore Defaults Button to remove any extra folders that have been added.
