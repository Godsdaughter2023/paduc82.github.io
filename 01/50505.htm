I recently encountered this error and did not find a solution that worked for me despite several hours of reading and tinkering, but I did eventually stumble upon a solution.  Try this:

What worked for me

 - Enter your login information

 - Spam Ctrl + Alt + Del when you see the "Welcome" message

 - Select Task Manager

I had to do this repeatedly until, by chance, the system did not immediately sign me out.  Once you reach this point do the following:

 - In Task Manager, select File -> Run new Task -> enter explorer.exe

 - If a File Explorer window opens, click "System Properties" in the bar below the menu bar

 - System Properties -> System Protection -> System Restore

This will load a restore point and restart your computer.  This allowed me full access to my system with no further issues.


Some details

I came across this solution by accident.  I had tried the other suggestions I found through a search, including registry edits to Userinit and copying userinit.exe from a working PC.

One person stated that he resolved the problem simply by repeatedly trying to login and eventually having success.  I did that with no real success until, on one attempt, a dialog window popped up telling me that a program was unresponsive and would not let me sign out.  I pressed Ctrl+Alt+Del and got the Task Manager open, saw all of my start up programs were running, and began attempting solutions.  The desktop picture and task bar would load if I launched explorer.exe through Task Manager but none of my desktop icons appeared nor did anything on the task bar, including the items in the system tray.  From there I attempted various solutions that I had found online, including editing the value of Userinit in the registry and copying userinit.exe from a working PC to Windwows\System32 on the compromised one.  None of those helped and when I shut down I was returned to the login loop.  (I had to use the power button rather because the OS shutdown feature did not work.)

I got the idea that some of the guts of the system were working so I tried hitting Ctrl+Alt+Del as soon as I entered my password and saw that the selection menu came up, though I was logged off several times.  Perhaps it was luck, but I got Task Manager to stay open one time and loaded the system restore point.


I hope this saves someone the nerve-wracking hours I spent resolving the issue.

System restore only has a certain amount of space allocated to it. When that space becomes fully used, it overwrites the oldest restore point with the new restore point. Therefore, unless you only recently purchased the PC, using system restore to a "like new" point will not be available.

These are two VERY different, ENTIRELY different things!

ElderL kindly provided you a link to your Owner's Manual. Both these things are discussed on pp. 75-78.

First of all, Windows XP System Restore will not help you. It's not worth getting into now, but trust us, it will NOT help you at all.

3. Do you want to attempt to first identify/solve your specific program or would you rather "nuke it" and run Dell PC Restore by Symantec (knowing that this will most likely be the most time-consuming activity)?
