Automatic updates

How do i turn on automatice updates

ernestoacosta labrada say:
Hi, 
My name is Rogelio. I am an Independent Advisor. Thanks for contacting our support.

Configuring automatic update options
Alternatively, if disabling the policy isn't working, you can also use the Group Policy Editor to configure the Windows Update options to prevent updates from installing automatically on your computer.

Use the Windows key + R keyboard shortcut to open the Run command.
Type gpedit.msc and click OK to open the Local Group Policy Editor.
Browse the following path:

Computer Configuration\Administrative Templates\Windows Components\Windows Update

On the right side, double-click the Configure Automatic Updates policy.
Check the Enabled option to turn on the policy.
Under the "Options" section, you'll find a number of options to configure automatic updates, including:

2 - Notify for download and auto install.
3 - Auto download and notify for install.
4 - Auto download and schedule the install.
5 - Allow local admin to choose setting.
You should select the option that best fits your preference, but option 2 - Notify for download and auto install is the closest to permanently disable automatic updates.

Click the Apply button.
Click the OK button.
Once you've completed the steps, Windows 10 will no longer download updates automatically, and once new updates become available, you'll get a notification to install them manually from Settings > Update & Security > Windows Update.

At any time, you can revert the changes using the same steps, but on step No. 5, make sure to select the Not Configured option. Then reboot your device to apply the changes.

azeez n say:
Hi,
 
First you need to add the songs to the queue and then save it.
 
Here is how you do it.
 
In Windows Media Center
 
a. On the start screen, scroll to Music, and then click music library.

b. Click album artists, albums, artists, genres, songs, playlists, composers, or years, and then navigate to the music you want to add to the queue.

c. Click either a title or a name, and then click Add to Queue.
 
Now, do this to save the queue as a playlist
 
a. After adding music to your song list, on the Windows Media Center start screen, scroll to Now Playing, and then click the icon.

b. Click View List, and then click Save As Playlist.

c. Type a name for the playlist, and then click Save.
 
When you save the song list (queue) as a playlist, it's added to your permanent set of playlists. You can access your playlists in the Music library under Playlists.

cbarnhorst say:

You have to type the "slmgr /xpr" command in a Command Prompt.  If you type in the Search box all you will get is websites.  To get a command prompt window, right click on the Start button to get the context menu and choose Command Prompt.  Enter the "slmgr /xpr" string in the black window that appears after a few moments.  This is the command that will confirm that Windows 10 is "permanently activated" and therefore can be reinstalled any time without needing a product key.  There is a long pause while the command executes.

The "winver" command I also mentioned does work in the Search box.  That's the one that will tell you if the software can be transferred to another computer or not.  What you are looking for is who owns the rights granted by the license, the computer's manufacturer or you.  If it is you, you have the right to transfer Windows 10.  If it is the computer's manufacturer, then you do not.

Concerning product keys:

No product key is needed to reinstall Windows 10 on the same device once it is permanently activated.  

(The key you see on all the Windows 10 machines is a dummy key and blocked on Microsoft's activation servers.  That's why it cannot be used even if a screen appears with a place to enter a key.  Instead such a screen must be "skipped."  Or, if you like dreadful modern yakkity yak, "Nexted.") 

The special Get Windows 10 program is a unique upgrade program for Windows 7 and 8 owners and the dummy key that is inserted into the activation files by the installer is a special case.  It insures that the free copy of Windows cannot be installed on any computer that is not running Windows 7 or 8.

(If you buy a retail copy of Windows you do get a unique Windows 10 product key that works the same way Windows 7 and 8 product keys have always worked.)

cbarnhorst say:

Remember the old Repair Install option in XP?  It went away with Vista, 7 and 8.  It's BACK.  That's great news for you.  You clean install Windows without losing your apps, files, and settings.  Not a one.

Unlike XP's Repair Install, the Windows 10 version has to run from the desktop.  The big difference between the new Repair Install and the Reset this PC option in Settings is that Reset this PC uses the same files that were used to install the Windows 10 you have.  That is OK if a file corrupts after Windows is set up but not so good if the original installation file was bad.  If you just Reset this PC, it will use the same bad file.

Repair Install comes to the rescue because you download a fresh media creation tool and make an .iso with fresh Windows files.  You won't get the same bad file.

The procedure is to start Windows setup from the desktop and proceed as normal to the last page of questions, which begins "Choose what you want to keep."  Choose the top button, which is everything.  Sounds like you are doing the same thing over again, and you are...except you are not doing it with same copy of Windows.  And you won't have to reinstall any apps like you do with a clean install and with Reset this PC.
