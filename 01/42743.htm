¡firedog say:

Windows Mail is the mail client program that is included with and part of Windows Vista. Windows Live Mail is a program available for free download; it is a mail client, calendar application, contacts manager, feed aggregator and news reader all in one program.

Until a few years ago it was possible to synchronize Hotmail accounts with the mail clients on Windows XP and Vista (Outlook Express and Windows Mail respectively) using the WebDAV protocol over HTTP. This method was deprecated after a long lead-in in September 2009. Before that time, POP3 access had been made available for all Hotmail accounts. At no stage was it not possible to access Hotmail accounts from mail clients. There is no bug involved here, neither in HTTP format nor anything else.

Windows Live Mail is constructed to use a protocol different from the old WebDAV; the DeltaSync protocol allows for synchronization of email folders, contacts and calendars between client and server.

You may be able to download the installer for Windows Live Mail 2011 from this page: How do I install Windows Essentials while offline? Don't attempt to include Windows Live Writer or Messenger in the installation; neither of them will work and you may foul up the entire Essentials installation if you do. It may be best to stick to Mail alone in the first instance. Windows Live Mail 2011 is the last version that will run on Vista.
