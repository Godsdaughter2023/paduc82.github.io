You can't really password protect folders with Vista (though you can protect zipped files/folders  http://techblissonline.com/how-to-password-protect-zip-files-in-windows-xpwindows-vista/) - that's not one of its features (at least it isn't in 32-bit Vista Business SP2).

Perhaps you don't want a program that hides folders to password protect them.  Here are some other options I've found.  Some work in varous ways (not always password protection directly) - but they are all designed to protect your files and folders from being viewed by other people.

 

Here's a free program that protects folders by encrypting them so only the people you want to be able to access them can do so without seeing just gibberish: http://www.newsoftwares.net/folderlock/.  It doesn't use a password (just your username login rights) but it accomplishes the same objective.  There's also a paid version of this program that I think does password protect folders (or the free version may as well - the information isn't entirely clear).

 

This isn't exactly what you want, but it may serve your purposes - you can check it out and see for yourself: http://www.bestvistadownloads.com/software/k-folder-lock-hide-password-protect-private-file-t-free-easy-file-locker-download-thfxsbeu.html.

 

This also isn't exactly what you want, but it does set up password protection for folders (in an unusual way): http://www.nirmaltv.com/2008/07/25/password-protect-your-folders-in-windows-with-my-lockbox/.

 

Here's one that seems to have what you want but I can't tell if it can be isolated from the other features that you may not want: http://www.easyfreeware.com/folderdefence_free-38583-freeware.html.

 

I don't know if you host a web page, but if you do you can get free password protection sofware in exchange for a permanent link of the company being placed on your web page: http://www.protect-folders.com/free.html.  It can also be purchased - I didn't check the price.

 

Here's one for $20 which does pretty much everything you would want from a password protected folder software: http://www.filebuzz.com/fileinfo/5682/Protect_Folder_98.html - but I normally don't recommend software that costs money because there are options (such as the others listed here) that are free and often work just as good if not better.  I added this just so you'd have a definite option if you don't like the others.

That should be enough options to check them out and make a choice (if one meets your needs).  To do exactly what you want (and only that) probably requires commercial software (like the above) - though I may be wrong and one of these may do the job quite well.

I hope this helps.

Good luck!

Device that can be disabled for doing a Hardware clean boot are:
 
 
a. Video Cards.
b. Sound Card.
c. Network Card.
d. USB Devices and controller (Please don’t do this if you are using USB keyboard or Wireless Keyboard or mouse.
e. CD and DVD.
f. Modems.
g. Ports (serial and parallel).
h. After doing this if you are able to Boot in Normal Mode try to enable the device one by one and check out which device is causing the issue .Try to install the latest driver for that device or contact the hardware vendor.

  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.

  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.
