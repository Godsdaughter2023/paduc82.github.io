1) What is the Make and Model of the computer?
2) What are the particular devices that cannot be connected to the computer?
 
I would suggest you to follow the methods below to try and resolve the issue:
 
Method 1:
 
I would suggest you to update the drivers.
 
Follow the steps:
 
a) Press Windows and d key together to go to the desktop.
b) Press Windows and x key and select Device Manager.
c) Locate the respective driver which is not working.
d) Right click on the driver icon and select Update driver software.
e) Click on Search automatically for the updated driver.
f) Follow the onscreen instructions to update.
 
Method 2:
 
I will suggest you to uninstall the drivers of the devices that are currently installed.
Note: Download the latest drivers for the devices form the manufacturer website.

Follow the steps:
 
g) Press Windows and d key together to go to the desktop.
h) Press Windows and x key and select Device Manager.
i) Locate the respective driver which is not working.
j) Right click on the driver and select Uninstall.
k) Right click on the Setup file which you have downloaded.
l) Select Properties.
m) Click on Compatibility tab.
n) Under Compatibility Mode, check the box which states “Run this program in compatibility Mode for”:
o) Select Windows 7 from the list.
p) Click on OK.
q) Follow the onscreen instructions to install the driver in Windows 7 compatibility mode.
