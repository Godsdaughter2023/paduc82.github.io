That is a memory management error.  Several things can cause that.


Bad memory, not enough memory, memory not properly seated or with dust in the slots, problem filesystem on the hard drive, thermal conditions, bad or deteriorating PSU, or just a PSU that does not supply the proper amount of wattage during high stress periods, malware, or even a problem driver.


Make and model of hardware, processor, amount of RAM, etc., would be helpful to know.


You will want to check your memory, and set the memory checker to put stress on the memory using advanced settings while running it for more than five to ten passes.


You may want to try removing and reseating memory back into the slots on the mainboard.


Make sure you are not overclocking.  Check the filesystem for errors using chkdsk /f /r (takes a while).


Check for the presence of malware.


Check your PSU wattage and make sure it is enough to handle all your hardware.  Check your PSU for any faults.


Also try installing while in clean boot mode.
