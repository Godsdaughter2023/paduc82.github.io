lucas módena say: I had the same problem with my Dell Inspiron 15R SE. It took me WEEKS to find out what was happening to me. I'm writing here hoping people with se same problem find a solution.

First, while Windows is stuck at 100% disc reading after boot, try disabling the wifi via fn key. If Windows "calms down" after that, then it's probably the bluetooth/WLAN driver. Uninstall it, leave the drivers up to Windows and be happy. I almost couldn't believe it was so simple.

I had to do a cleand Windows 8 install (had 8.1 before) to realize what the problem was, as the problem returned after I install this driver.

I hope it is the same problem and this solution works for you.
