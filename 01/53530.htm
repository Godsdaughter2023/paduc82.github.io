Hello Mike,

To tackle such situations Windows has a hidden built-in administrator account, which can be enabled and accessed to assign permissions to other users on the computer.

Let's try to enable the built-in administrator account on your computer and through that we can change back your user profile type from guest to admin.

Enable default administrator account

  1. Type cmd on start screen, Right Click CMD from the available list and open as administrator.
  2. Enter the following command to enable the hidden administrator account:
    net user administrator /active:yes
  3. Restart the computer.

After the account is enabled, a new user account will appear on the signin page. Note that the account has not assigned a password to it, and that you should consider setting one to improve account security.

Once you are able to get into the computer by logging into default admin account, you can follow the steps below to give administrator privilege to your account.

  1. Type Control Panel on start screen, open Control Panel, click User Accounts in control panel, and then click Manage another account.

  2. Click the account you want to change, and then click Change the account type.

  3. Select the account type to Administrator, and then click Change Account Type.

After successfully changing your account type to Administrator, you can disable the Default admin account.

To disable, type following command in Command prompt

net user administrator /active:yes

For more information on How to enable and disable the built-in Administrator account please refer following Microsoft Knowledge Base article.

Enable and Disable the Built-in Administrator Account

http://technet.microsoft.com/en-us/library/hh825104.aspx

User accounts: FAQ
