ronnie vernon say:

when I open file explorer and press the file tab it displays a list of frequent places invading my privacy


Hi

 

I'm not sure what you mean by "invading my privacy"?

 

The Frequent Places only show the folders and files that you have opened in File Explorer. Each user account will only show the frequent places for that specific account. No other account can see the frequent places for your user account.

 

The Frequent Places are associated with the Taskbar Jump Lists for apps that are pinned to the Taskbar.

 

You can right click the Taskbar and select Properties.

 

Select the Jump Lists Tab and remove the check marks from the 2 options you see there.

 

This will disable the Frequent Places list in File Explorer and the Jump Lists on the Taskbar.

Hello Nourelyan,

Thank you for posting your query in Microsoft Community Forum.

  1. Is the issue specific to a particular game or with every other audio application? 
  2. Did you make any significant hardware or software changes to the computer prior to this issue?
  3. Do you get any error message? If yes, what is the complete and exact error message that you get?

I understand that the sound is not played in games but while playing movies. In that case, you may check with the audio and volume options in the game and check if the sound is muted. In some cases, you can work around the problem by disabling optional speaker effects that are collectively known as enhancements. If disabling speaker effects solves the problem, you might need to update your audio driver. Refer to the following methods.

Method 1:

To disable speaker audio enhancements. 
a. Open Sound by clicking the Start button, and then clicking Control Panel. In the search box, type sound, and then click Sound.
b. On the Playback tab, click Speakers, and then click Properties.
c. Click the Enhancements tab, and then select the Disable all enhancements check box.


If you do not see an Enhancements tab in the Speakers Properties dialog box, those features are not available for your speakers and this procedure is not applicable to your computer.

If disabling audio enhancements solves the audio problem you are experiencing, this typically indicates that your audio driver needs to be updated.
If disabling audio enhancements solves the audio problem you are experiencing, check to see if an updated driver is available for your computer from the manufacturer’s website.

If the issue still exists, try Method 2.

Method 2:

Perform the following steps and check if that helps.
a. Click the Start Button, type "MMSYS.CPL" (without quotation marks) in the Start Search box and press Enter.
b. On Playback tab, click the Speaker which is used currently and click Properties button.
c. Switch to Advanced tab, uncheck the "Allow applications to take exclusive control of this device" option.
d. If the Enhancements tab is available, click it. Check 'Disable all sound effects".
e. In Setting section, Select None.
f. Click Apply and click OK.

Additional information:

  • Tips for fixing common sound problems
  • No sound in Windows

Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.
