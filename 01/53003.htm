When faced with seemingly unexplainable phenomena it often pays to gather some evidence. In your case it would be interesting to know when exactly the lock-out occurs. The batch file below would register it. Here is how to implement it:

  1. Save the file as Log.bat in a suitable location. Make sure it has a .bat extension rather than .txt.

  4. Wait 70 seconds.
  5. Unlock the machine.
  6. Check the file Log.txt on your desktop. It should report the lockout.

set msg=%date% %time% Logonui.exe is active

if %ErrorLevel% EQU 0 echo %msg%  >> "%UserProfile%\Desktop\Log.txt"
timout 60
goto Loop
