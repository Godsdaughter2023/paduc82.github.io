shenan stanley say:

No "seems to be gone", it's gone.  :-)

  

The clipboard is stored in volatile memory space - meaning that when you restarted the compute: you cleared it.

  

If you wanted to SAVE that text, you should have pasted it before restarting or filling said volatile memory with other information.  (for example: Open Notepad/Wordpad and paste it there and SAVE that document to disk - the non-volatile memory in your system.)

  

http://theydiffer.com/difference-between-volatile-and-non-volatile-memory/

  

Now, you might argue that restarting Windows 10 isn't 'losing power', and in a strict sense, you are correct; however, the clipboard you copied the data into (the application that is the clipboard) ceased to be running when Windows 10 restarted (since Windows 10 itself wasn't running for the clipboard to be running on top of it - so...)
