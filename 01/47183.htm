Hello,

If you see that you have 0 bytes in your hard drive, that volume may have been corrupted. You may have to format that volume to NTFS/FAT using the Disk Management tool in Windows 10. Before formatting the drive/volume, kindly extract the files on that drive/volume to avoid permanent data loss. Here's how to format a volume in Windows 10:

  1. Open Computer Management by selecting the Start button, then open Control Panel > System and Security > Administrative Tools, and then double-click Computer Management.
    • Note: View by must be in Category to see System and Security.
  2. In the left pane, under Storage, select Disk Management.
  3. Right-click the volume that you want to format, and then select Format.
  4. To format the volume with the default settings, in the Format dialog box, select OK, and then select OK again.

Note: You can not format a disk or partition that is currently in use, including the partition that contains Windows.

Tip: The Perform a quick format option will create a new file table, but will not fully overwrite or erase the volume. A quick format is much faster than a normal format, which fully erases any existing data on the volume.

Documents and Settings is not a real folder in Windows 7. It is a Junction that is set up to allow legacy programs to interface with Windows 7 's folder structure.

Windows 7 uses these folders instead:

C:\Users\*your User Name*\My Documents

and

C:\Users\*your User Name*\Appdata

(appdata is a hidden folder)

Look into C:\Users to see what accounts exist.

Also you can access the Control Panel>User Accounts to alter them.

Reinstall Windows Preinstalled On Netbook

OEM computer manufacturers are required to provide at least one way for you to reinstall Windows if you buy a computer with Windows preinstalled - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks (you would need an external DVD drive﻿ if you have a netbook).

If the restore partition is still there and was not removed, it will reinstall Windows 7. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to. This will remove any personal files on the computer, back them up first.

If you don't have the restore partition or install media:

Contact the OEM computer manufacturer and ask them how to reinstall Windows. If they agree to send you something, DVD or USB, they may charge a small fee.

cmdrkeene say:

It is a not really a "Windows" issue, but it is an issue with any app that doesn't flag and broadcast which button is default in a given interface in a standard way, or at least not broadcast it in a way that the Logitech software is listening for.

You might try the built-in "Snap To" option, which comes as part of Windows, but it may suffer the same issue depending on the app being used and how it broadcasts its default buttons.
