Hi WillisPr:

What is the model of your Seagate 2TB external hard drive and what backup software are you using?  When you say "a full backup" do you mean a full system image, and if so have you been able to create a full system image of your Vista computer on this Seagate backup drive in the past without generating an error?

See the MS support article Microsoft Support Policy for 4K Sector Hard Drives in Windows. Many hard drive manufacturers are changing the physical format of their larger hard disk drives from 512-byte sectors to 4,096-byte sectors [4K native or Advanced Format (512E)] to improve storage density and reliability and you might not be able to be able to back up personal documents and/or create full system image backups to these types of external drives with backup utilities like the built-in Windows Backup utility that comes with Vista and Win 7.

See FrancisNicholson's 2013 thread Seagate backup drive error "0x8078002A" for further information.  One Win 7 user posting in that thread noted that they were able to back up personal documents and other data but would see an error message if they tried creating a full system image backup.  Seagate customer support suggested that they use Seagate's own DiscWizard imaging software which can "read the 4K sectors, and create full image backup to save to the Backup Plus drive".    If you contact Seagate customer support directly (see https://www.seagate.com/ca/en/contacts/) they might have other suggestions.

JaredM also made this comment about 2TB Seagate external hard drives in the Tom's Hardare forum thread 512K Sector 2TB External HDD : "Just avoid Seagate and you'll probably be fine. Most hard drives will emulate a 512byte sector size even if the drive is a native 4K sector size. Even Seagates do this, but for some stupid reason, they re-emulate it back to 4K via the USB bridge board..." so your problem could be specific to the Seagate brand.

Step 3: Determine whether the problem returns
If the problem still occurs, repeat step 1 and step 2. In step 2, click to clear half of the check boxes that you originally selected in the Service list.
