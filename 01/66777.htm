a. user say:

***removed***

SOLUTION

It is a problem with Windows Explorer in Windows 7 not resolving the target computer name to its IP address. It is not a problem with firewalls or virus protection.

There is a simple work-around solution.  Use the IP address of the target computer, not the computer name.

1) Find out what the target computer's IP address is.  It is desirable to give the target computer a static private IP address, for example, 192.168.1.10, so it does not change.  Use the local windows help when sitting in front of the target computer to do this if you don't know how or search online for that info.

2) When sitting in front of the Windows 7 computer you want to access the target computer from, open Windows Explorer and in the address bar type \\ followed by the target computer's static address, for example,\\192.168.1.10 and hit enter

If the target computer uses username/password sharing, after a few seconds you will be prompted for credentials.  Enter the same username and password employed to log on to the target computer when you are sitting in front of it.

The target computer will be added to the Windows 7 computer's Network list, appearing as an IP address instead of a computer name.  Then you can access its shares just as if it were listed by name instead of by number.

That's it.

a. user say:

The updates are not 'forced.' Windows 10 is a continuous series of updates. Microsoft made that very clear right from the beginning. Windows 10 is behaving exactly as advertised.

Since updates are the crux of Windows 10 - and what keeps Microsoft competitive in the marketplace - Microsoft goes to great lengths to ensure that updates are delivered, while offering users both the ability to delay receiving updates for extended periods of time, and to schedule when computers are restarted to finish installing updates.

Only by interfering with the normal operation of Windows 10, or by breaking it altogether, are you able to stop receiving updates. However, since future development of Windows, and the application software that runs on Windows, depends on having an up-to-date operating system, you will find yourself increasingly falling behind as time goes by, and increasingly exposed to security risks.

In short: If you didn't want a computer that receives regular updates, a Windows 10 PC was the wrong computer for you.

a. user say:

If you really want to switch to a different operating system, that's up to you. This is a community forum, so nobody really cares whether you switch or not.

The problem you're going to run into is that there are few modern operating system that don't update regularly. Forget about a Chromebook - they get more updates than Windows, and Google doesn't even give you the opportunity to defer the updates. Apple has a different way of enforcing updates, but it does. And if you think I'm sympathetic, try telling Apple technical support that you only want to install the updates that you want, when you want to.

Android tablets are gradually going away, now that Google has decided to stop making them. The iPad makes a good computer, but you'll have to deal with Apple for updates.

Linux is different. You can update Linux or not, whenever it suits you, although if you don't update for a long time you may run into compatibility problems with applications. I've used Linux Mint, and it works well for me.

The other problem you'll run into is that software applications also update on their own, usually without telling you. For example, that's what Microsoft Office does, and all of Google's software does that too.

Refusing to update your software works against your own best interests. And the more you fall behind, the harder it is to catch up. Ultimately, people have to learn how to use and maintain their computers correctly and wisely, so that they know how to deal with regular updates. Because updates ain't going away.
