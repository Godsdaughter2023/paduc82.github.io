srimadhwa b say:

Hi,

Please answer these questions to get more clarity on this issue.

  • Did you make any changes to the computer prior to this issue?

  • What is the issue you are facing on the computer for which you are performing check disk?

This issue might occur due to reasons like bad sectors on the hard drive, virus infection and some broken or corrupt system files.

Let us follow these methods and check if this helps.

Method 1.

As check disk scan window is stuck for about 2 hours, you may hold the power button for about 10 seconds and ensure that the computer is completely shut down. Once this is done, wait for about 3 minutes and then restart the computer to check if you are able to get to the desktop screen.

You may perform SFC scan and check if that helps. The SFC /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions. Once the SFC scan is done, restart the computer and check if the issue persists.

For more information how to do SFC/ scan on Windows Vista and Windows 7, please follow this link: http://support.microsoft.com/kb/929833

Method 2.

You may run a full computer scan with Microsoft Safety Scanner to make sure that the computer is free from virus.

Microsoft Safety Scanner: http://www.microsoft.com/security/scanner/en-us/default.aspx

Safety Scanner Disclaimer: There could be a loss of data while performing a scan using Microsoft safety scanner to eliminate virus if any.

Let us know if you need further help with any Windows issue. We will be glad to help.
