hairyfool say:

There is nothing within Windows that can prevent the Windows 7 operating system from booting unless you modify it in some way that it will no longer load at all which would then prevent it loading even when required. The key flag is is the partition "Active", remove the Active flag and it should not boot.

It is the BIOS that passes control to the 1st bootable device it finds in the sequence, if the SSD is the 1st device in the sequence then that is the one that boots. The only way it could get to any following device would be if the BIOS fails to find the SSD in the 1st place, if that happens then you have a fault.

Some BIOS have the ability to limit the number of devices in the sequence so for example if three can be allocated setting USB then Optical then SSD it should not go to the HDD. More common would be that the 3rd slot is just Hard Disk and the drives themselves have a sequence and all drives will be listed.

Hiding the Win7 partition in Windows 10 could be as simple as removing its drive letter. The booted operating system sets the drive letter of the other partitions and these can be changed including "none", a drive that has no letter will not appear in Explorer.

Check the settings on the Printer itself to verify that it isn't changed to black-and-white only there (if there are any such settings on the printer) - that would probably override your computer settings.  BTW, what is the exact printer?  Is it this one: http://support.lexmark.com/index?page=answers&startover=y&question=lexmark+7600+series&locale=en&productCode=LEXMARK_X7675&segment=SUPPORT&userlocale=EN_US#1?  If not, what one is it?

You may be having troubles with your device drivers, firmware, and/or software.  Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (and you shouldget them all for at least the printer (http://support.lexmark.com/index?page=content&productCode=LEXMARK_X7675&actp=PRODUCT&id=DR11032&segment=SUPPORT&userlocale=EN_US&locale=en (if I was right about the printer above and you're using 32-bit Vista) and any you have time to do - they may help in ways beyond just resolving this problem), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows: http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows:http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html. You may also want to try the Driver Verifier Manager to see if you can identify the driver or device responsible (assuming that is the cause):http://support.microsoft.com/kb/244617.

Click on Add a Printer and start from scratch.  Then see if the new printer connection/setup works any better than the old one.

If that doesn't work, then contact Lexmark Technical Supporthttp://www1.lexmark.com/content/en_us/about_us/contact_us.shtml and see if they can assist you as I don't believe it would be a Vista problem if you reinstalled everything.  It may be a problem with the setup but it may also be a hardware problem - either way, they should be able to assist you.

I hope this helps.
