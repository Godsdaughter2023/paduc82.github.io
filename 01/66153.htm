If 'factory defaults' means that you want your computer to be exactly as it was when it left the factory, i.e., the way it was when you powered on for the very first time, that's actually quite easy to do.

Instructions should have come in the box with the computer, but not to worry if you don't have them. The very same instructions are on the manufacturer's website, with full details. You only have to remember to follow the instructions for your model.

You won't need discs - computers haven't shipped with recovery discs for years - and you won't need license keys or anything like that. It's all been taken care of for you.

Resetting your computer this way has three great advantages over the reset methods that Microsoft offers:

1- You'll get back all the additional software that was originally installed on your computer. Not so with a Microsoft reset.

2- All drivers and other accessory software are included in the installation, so there's nothing that you have to add.

3- The installation has been customized for your computer, so all you have to do is answer a few simple questions. Then, sit back and your computer takes care of everything.

a. user say:

Hello Hassan,

We understand the inconvenience you have experienced.

  1. Did you make any changes to your laptop?
  2. During which application session will this crash happens?
  3. Which Windows 10 version installed on your computer? Press Windows logo key + R, type winver and hit Enter key. Now see the version and build details.

The DRIVER_IRQL_NOT_LESS_OR_EQUAL bug check has a value of 0x000000D1. This indicates that a kernel-mode driver attempted to access pageable memory at a process IRQL that was too high. A driver tried to access an address that is pageable (or that is completely invalid) while the IRQL was too high. This bug check is usually caused by drivers that have used improper addresses.

I guess the problem might be with the Intel Wireless WiFi Link Adapter from Intel Corporation. Suggest you to remove the existing Intel Wireless WiFi Link Adapter driver and then try to install the latest driver to see if it makes any difference. You can search them on device manufacturer website or on Intel website. Most probably this will fix the issue.

http://www.intel.com/content/www/us/en/support/network-and-i-o/wireless-networking/000005600.html

For more help, see Troubleshoot blue screen errors

You can also update the Network drivers through Device Manager manually and check if this helps.

a) Open run command by pressing Windows key + R and type “devmgmt.msc” (without quotes) and hit Enter.

b) Spot and expand the “Network Adaptor”.

c). Right click on the network driver and select “Update Driver Software”. When it prompts to choose an option How do you want to search driver, then go with Search automatically for updated driver software. If you have a driver available on your PC, then select Browse my computer for driver software, then select the appropriate location, where the driver located on your PC for updating it. (Ensure that PC is connected to Internet while performing this step)

d). Once this is done, restart the computer to check the problem fixed. If it doesn't fix the issue, then Right click on the same driver and select “Uninstall” then restart the system and let the system install the driver automatically on Start. Check and see if it resolve the issue. (Recommended step)

Keep posted if you require further assistance.
