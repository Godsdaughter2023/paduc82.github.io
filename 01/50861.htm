ralf_g say:

How do I put a shortcut icon on the desktop, so that when I click it, it just opens Windows Explorer at the "My Documents" level, avoiding the multiple clicks and scrolls it takes to get there in Win7?


Unless you're using a different folder than the default Documents folder, that functionality already exists when clicking the "Documents" link on the start menu. In case you don't have the Documents link enabled, right click on the Start button and select Properties. On the Start Menu tab click Customize and then look down the list of items for Documents and select "Display as a link". Clicking Documents from the Start menu opens the Documents folder directly.

If you still want a desktop shortcut to the Documents folder you can right-click the Documents link in the Start menu and then select "Send to - Desktop (create shortcut)". 

If that is still not quite what you want, you should be able to create a desktop shortcut to any desired folder by right-clicking on the folder in the left-hand pane of Windows Explorer and select "Send to - Desktop".

arminatorx say:

I have been using the regular reachargable AA batteries that come with the wireless x1 elite controller. It isnt the play and charge kit.[...]

The AA Batteries that came with my Elite controller are NON-rechargeable Duracell batteries. 

If you are using rechargeable AA batteries, they will also NOT charge through the USB cable. 

Only the specific battery pack of the Play and Charge kit can be charged via USB. If you look into the battery compartment, between the battery contacts, you should see three "golden" contacts. These are for the Play and Charge battery pack, and only those contacts provide the charging capacity for the Play and Charge battery pack. No charging current is sent to the AA battery contacts, as the controller can't know if you inserted standard non-rechargeable AA batteries (which might explode if charged), or if you are using rechargeable AA batteries. 

Therefore it's not possible to charge AA batteries inside any Xbox One controller.
