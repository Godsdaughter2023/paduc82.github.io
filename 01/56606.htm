3. Are you able to use other features like Quick scan using Windows Defender? (Settings > Update & Security > Windows Security > Virus & threat protection > Quick scan)

4. Do you have any third party anti-virus software installed on your computer?

5. Is your computer connected to domain network?

If you're using Windows 10 Pro or Education, and if the system is not connected to any domain network, you can reset Local Group Policy Editor settings to default and check if it helps. To do that:

1. Click on Start icon.

2. Type Command Prompt.

3. Right click on it and select Run as administrator.

4. Enter the below commands one by one and press Enter after each command:

RD /S /Q "%windir%\System32\GroupPolicyUsers"

RD /S /Q "%windir%\System32\GroupPolicy"

gpupdate /force

5. Restart the system and check.

Got a better solution from another thread :

Had exactly the same problem here and tried all the usual answers (wsreset, using powershell to re-register the store) but nothing worked.

Finally found this knowledge base article: KB2798317 which pointed me in the right direction.  For some reason "ALL APPLICATION PACKAGES" did not have read permissions on HKEY_CLASSES_ROOT in the registry.  Run regedit, right click HKEY_CLASSES_ROOT and choose "Permissions" from the menu.  If "ALL APPLICATIONS PACKAGES" is missing then click add, enter ALL APPLICATIONS PACKAGES and press OK.  Make sure it has "Read" checked in the Allow column and press Apply.

According to the KB article there are other registry keys and file folder permissions worth checking but this did the trick for me.

This solved my problem. I can now postpone reinstalling my pc.

Hello Walter,

Welcome to Microsoft Community forum.

Automatic start up of computer is not easy and simple. It can be possible if we do some changes in BIOS settings.

Important: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk. 

To automatically start your computer up at a specific time of day, you'll actually need to edit your BIOS settings. To do this you may try the steps provided below.

  • Press delete when you get to the screen that says "Press DEL, F1 or F2 key to enter SETUP or BIOS"

 

  • Use the arrow keys to find the "Power Management" tab. Scroll to "Power Up Control. " Press enter.

 

 

  • Scroll to "Resume by alarm / Power up by alarm. Enable it.

 

 

  • Select wake up Day/Hour/Minute options. (Select Daily in wake up Day option, if you want PC to start daily automatically at given time).

 

 

  • Press "Esc" on your keyboard when you have reached the desired power-up time. Say yes when it asks for confirmation of the changes.

You probably shut down your computer when you're done using it at the end of the day, but if not, you can set it to shut itself down on a schedule. This is easy to do with Windows Task Scheduler:

  1. Hit the Start menu and type in "task scheduler" in search box. Open up Task Scheduler from your results.
  2. In the right pane, hit Create Task. Give it a name, and under the General tab, check "Run with highest privileges". Also check "Run whether user is logged on or not", if you ever leave your computer logged out.
  3. Now click on Settings tab and check "Stop the task if it runs longer than" and set it to "1 hour". This won't stop your computer from sleeping, but will stop your computer from thinking a task is still running.
  4. Head to the Actions tab, hit New, and choose "Start a Program" as your action. Set the Program to shutdown and the arguments to -s.
  5. Lastly, move to the Triggers tab and click New. Change the schedule to fit whatever you want (say, Daily at 12:00AM), and hit OK. Hit OK again at the next window and your task should be saved in Task Scheduler.
