loraine mon say:

To resolve the issue with Brightness control after the Anniversary update, you can use the inbox Deployment Image Servicing and Management (DISM) tool by following the steps below:

  1. Swipe in from the right edge of the screen, and then tap Search.
  2. Type Command Prompt in the Search box, right-click Command Prompt, and then click Run as administrator.
  3. Type the following command, and then press Enter.

DISM.exe /Online /Cleanup-image /Restorehealth

Then uninstall and re-install the latest display drivers on your computer again.
