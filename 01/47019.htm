Hi Rich,

I understand that you want to use the Canoscan FS2710 scanner on your Windows 8.1 computer, unfortunately you are not able to use it as the scanner supports SCSI (Small Computer System Interface) and not USB.

Canon suggest that the scanner is not compatible with Windows 8. You may refer to the following link for more information:

http://www.usa.canon.com/cusa/consumer/standard_display/windows_xp_upgrade

 

http://www.microsoft.com/en-us/windows/compatibility/CompatCenter/ProductDetailsViewer?Name=Canon%20CanoScan%20FS2710%20Scanner&vendor=Canon&Locale=1033%2C2057%2C3081%2C4105%2C16393&BreadCrumbPath=canon%20canoscan%20fs2710%20scanner&LastSearchTerm=canon%2Bcanoscan%2Bfs2710%2Bscanner&Type=Hardware&tempOsid=Windows%208.1

I would be better if you get in touch with the manufacturer of the scanner to check for scanner compatibility.

However, I would like to inform you that SCSI connection is no longer used and has been replaced by faster USB and firewire ports. The SCSI connection is roughly two inches long and is used by older printers not using USB or even a serial connection. As you are looking to hook up a SCSI device (Canoscan FS2710 scanner) to your computer it is possible to do so with the aid of a USB adapter. Follow the instructions:

1. Insert the USB end of the USB to SCSI adapter into one of the USB ports on your computer (it does not matter what USB connection port you use).

2. Connect the SCSI end of the adapter into one of the ends of the SCSI cable. Tighten the metal bands around the side of the cable to secure the connection.

3. Insert the opposite end of the SCSI cable into the SCSI port on the hardware (Scanner) you want to connect to the computer.

4. Power on the newly connected hardware (Scanner), then install the drivers for

Now go ahead and install compatible Windows 8.1 drivers for Canoscan FS2710 scanner from the following website and check:

http://www.usa.canon.com/cusa/support/consumer/scanners/canoscan_series/canoscan_fs2710#DriversAndSoftware

Hi Patricia,

 

Thank you for posting your question in the Microsoft Community Forum.

 

Based on the information, you continue to receive the error message “could not start the print spooler service on local computer with error 2” when trying to restart Print spooler service.

It would be helpful if you could reply to these questions to assist you further.

1. What is the make and model of the printer?

2. Did the issue start after installing Windows XP SP3?

3. What steps did you try to fix the issue? Did you try to update the printer drivers?

 

Method 1: Clear Printer Spooler files and enable the Spooler service.

a) Click Start, select run and then type services.msc in the search box and press Enter.

b) Double-click ”Printer Spooler" in the Services list.

c) Click Stop and click OK.

d) Click Start, select run and type %WINDIR%\system32\spool\printers in the search box and press Enter. In the open window, and delete all files in this folder.

e) Now, go to “Services” again. Double-click “Printer Spooler."

f) On the General tab, next to Startup type, make sure that Automatic is selected.

g) If the service is not already running, under Service status, click Start, and then click OK.

 

Method 2: Start the print spooler dependency services manually.
a. Click on start and type in services.msc in the search bar and hit enter.
b. Locate Print spooler service.
c. Right click on it and select properties.
d. Under dependencies tab, check the dependency services and start those services.

 

Method 3:

If the issue persists, you may run the SFC scan and check if it helps:

a. Click Start, select All programs and then Accessories.

b. Click Command Prompt, type “sfc /scannow” (without quotes) and press Enter.

Note: If information from the installation CD is needed to repair the problem, you may be prompted to insert your Windows XP installation disk.

c. After the scan has completed, close the command prompt window, restart the computer and check for the issue.

For addition information, you may refer the following link:

Description of Windows XP and Windows Server 2003 System File Checker (Sfc.exe)

 

Method 4:

You may also refer to the Printer manufacturer website and try installing the latest version of printer drivers.
