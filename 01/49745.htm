a. user say:

Hi Lucas,

Thank you for posting your query on Microsoft Community.

I understand that you are unable to setup Internet connection to Surface Pro 10 from Windows phone. I will be glad to help you.

This issue can be due to misconfiguration in network adapter settings and can occur if the "Allow the computer to turn off the device to save power" power option for the wireless network adapter is cleared.

Enable the "Allow the computer to turn off the device to save power" power management option for the wireless network adapter using the following steps:
1. Right click on the Start Button and select Device Manager.
2. Select and expand the Network adapters from the list of devices.
3. Find the wireless network adapter and right-click on it and select Properties.
4. Select the Power Management tab.
5. Under the Power Management tab, make sure the following option is checked (enabled): “Allow the computer to

    turn off this device to save power".

You should now be able to restart your hosted network using the classical wlan command.
netsh wlan start hosted

Note: Sharing your data connection through your phone may go over your data limit and could result in extra charges from your Internet provider.

Tethering: Use your phone’s data connection

Perform the following methods to resolve the issue: 

Method 1: Startup Repair from the Windows Recovery Environment (WinRE)

============================================

1. Insert the Windows 7 installation disc into the disc drive, and then start the computer. 

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

3. Select a language, a time and currency, and a keyboard or input method, and then click Next.

Method 2: Rebuild BCD using the Bootrec.exe tool

============================================

To run the Bootrec.exe tool, you must start Windows RE. To do this, follow these steps:

1. Put the Windows Vista or Windows 7 installation disc in the disc drive, and then start the computer.

2. Press a key when you are prompted. Select a language, a time, a currency, a keyboard or an input method, and then click Next.

3. Click Repair your computer.

4. Click the operating system that you want to repair, and then click Next.

5. In the System Recovery Options dialog box, click Command Prompt.

6. Type Bootrec.exe, and then press ENTER.


Refer the following link for more information:

How to use the Bootrec.exe tool in the Windows Recovery Environment to troubleshoot and repair startup issues in Windows

Method 3: Use the Windows Recovery Environment (WinRE) to run System Restore

============================================

1. Insert the Windows 7 installation disc into the disc drive, and then start the computer. 

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

3. Select a language, a time and currency, and a keyboard or input method, and then click Next.

8. Click Finish to restore the system.
