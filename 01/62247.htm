  1. In the main menu (Skype application), select Tools > Options.
  2. In the Skype Options box, select Privacy > Blocked Contacts. A list of blocked contacts is displayed.
  3. Select the contact you'd like to unblock and click Unblock this person.

Windows 10:

  1. Open Profile window by pressing the Profile icon shown at the top of the Recent list.
  2. In the Profile window select Settings.
  3. In the Settings window scroll down to the Contacts section and select the option Manage blocked contacts.

Mac:

  1. In the main menu, go to Skype > Preferences....
  2. Click Privacy. In the Privacy panel, click Manage Blocked Users. A list of blocked users is displayed.
  3. Click the person you’d like to unblock, and then click Unblock.

I hope this information helps, but if you have questions and clarifications, please do not hesitate to reply to this post.

I wish to transcribe lectures and interviews available on the Internet in MP3 format using my desktop PC.  After a few seconds of play I would like to be able to press a key such as Pause/Break, type what I have heard, then press the same key again to resume play;  then repeat every few seconds until the entire file has played.

a. user say:

The media player you use to play the lectures likely has a keyboard shortcut to pause the playback. For example: In Windows Media Player you can press CTRL+P to pause and resume playback.

These keyboard shortcuts only work if the audio file is on your hard drive. If the audio streams from the internet and the web page doesn't provide a control to pause the playback, there's nothing you can do to force it.
