While I don't doubt the error message your are receiving, it sounds like right now you're looking in the wrong place.

If you are looking at a folder's Properties dialog, there is a checkbox labeled Read-Only with a gray square inside it. That gray square is not a status indicator for either the folder or its contents!! It can be used to toggle the read-only attributes of the files contained in the folder.

But the error you are dealing with is most likely a permissions issue, and these are addressed via the Security tab in the Properties dialog.
