You get error code 43 when one of the drivers controlling the device notified the operating system that the device failed in some manner.
To resolve error code 43, I would suggest you  to  try following methods:

Error codes in Device Manager in Windows

Use Device Manager to uninstall and re-install the USB controller. This lets the controllers recover the USB port from its unresponsive condition. The USB controllers represent the USB ports in Device Manager.

To uninstall and re-install the USB controller, follow these steps:

1. Click Start, type device manager in the Start search box and hit Enter.

2. Expand Universal Serial Bus controllers.

Note: You might have to scroll down the list to find this item.

3. Right-click the first USB controller underUniversal Serial Bus controllers, and then clickUninstall to remove it.

4. Repeat the above step for each USB controller that is listed underUniversal Serial Bus controllers.

5. Restart the computer. After the computer starts, Windows will automatically scan for hardware changes and reinstall all the USB controllers that you uninstalled.

6. Check the USB device to see whether it is working.

 

Also update the latest chipset drivers from the computer manufactuere website.

Time stamp on outgoing emails - Microsoft Community

michele r say:

Time stamp on outgoing emails

I am running Windows Home Premium 64-Bit and Microsoft Office Home and Student 2007.  I use Windows Mail for email.  The date and time on my computer are correct but when I send emails the date stamp on my emails is off by 2 1/2 hours.  I don't know how to fix this.  Please help.  Thank you.
lava k say:

Hi Michele R,

 

Welcome to Microsoft Answers Community Site. I would suggest you to follow the steps below;

 

Step 1: Set the Time Zone and Internet Time Server on the Computer.

 

I) Click on the clock on the taskbar.

II) Click on Adjust Date/Time.

III) Click the Change time zone button

IV) Change the Time zone to the correct one for your region and select OK

V) Click the Internet Time tab, and then click Change settings.

VI) Click Automatically synchronize with an Internet time server, select a time server, and then click OK.

 

For more information check this Microsoft Help & How-To link: Set the clock. I would also suggest you to refer Change the display of dates, times, currency, and measurements and Change the country or region setting

 

Step 2: Ensure that you have set the correct time zone in the Webmail email account.

 

In Hotmail, click Options > More Options > View and edit your personal information > Registered information and specify the correct time zone.

 

Email services other than Hotmail have similar settings in profile, user account, or personal information settings. Regardless of which service you use, the time zone must be set properly for your email to be time stamped correctly.

 

If the issue persists it could be due to the mail server(s) time/date is/are set incorrectly. In this case, you will have to take a look at the mail headers to determine if this is the case. If it is determined that the mail server is configured incorrectly, you need to contact the owner of the mail server to have it configured correctly.

 

 

Let me know if that work. Good luck!

Hope this information is helpful.

 

Thanks and Regards,
