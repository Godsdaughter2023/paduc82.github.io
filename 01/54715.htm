To get rid of a ghosted device is a little convoluted... Microsoft doesn't make it easy, and it's even harder in Win10 since some of the interfaces required aren't accessible (at least that I've been able to find) from the new GUI.

e.g.

In Windows Key + R, start sysdm.cpl

on its Advanced tab, click the Environment Variables... button

in the bottom pane (System variables), click New...

enter DEVMGR_SHOW_NONPRESENT_DEVICES in the Variable Name field; make the Variable Value 1

OK, OK, OK back out.

From Windows Key + R again, run devmgmt.msc

in its View menu, select Show hidden devices

find your USB device[s] (probably in the Universal Serial Bus controllers category)

right-click the ghosted device[s] and choose Uninstall device

That should get rid of it for you.
