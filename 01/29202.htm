For your general information: The BIOS is a ROM (=Read-Only Memory) that you cannot modify unless you do a BIOS upgrade. lemp say:
How can I save a scanned document as a PDF file??

Even if your scanner didn't come with software that permits pdf creation, you're not out of luck.

  1. Download and install free CutePDF Writer
  2. Scan and save the document as its default file type (probably a *.tif graphic file).
  3. Open the saved document in your default graphics viewer application (e.g., Windows Picture and Fax Viewer).  If you don't already have IrfanView (a free but extremely versatile graphics viewer), I highly recommend it.
  4. In the graphics viewer application, open the Print dialog (usually File > Print) and select CutePDF Writer from the drop-down list of available printers. Click "Print," and enter a file name and location when prompted.
  5. Optionally, delete the original saved graphics file.
