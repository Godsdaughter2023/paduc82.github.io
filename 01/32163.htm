List of games with Game Bar support for full-screen mode

In the Windows 10 Creators Update, we now support Windows Game bar in full screen mode with over 80 games, listed below. This feature is controlled with the new Gaming settings page in Windows Settings. Look for “Game bar” settings, and make sure that “Show Game bar when I play full screen games Microsoft has verified” setting is checked.


If you have installed Windows 10 Creators Update, and are playing one of the games below and Win+G doesn’t bring up the Windows Game bar when you’re playing in full screen mode, please note that your graphics drivers need to be WDDM 2.0 or greater for this feature to be enabled. To confirm that you have a WDDM2.0 driver, hit Win+R, type “dxdiag”, and then on the Display tab look for the Driver Model listed on the right.

 

If you experience any issues playing full screen games with this setting enabled, please use the Feedback Hub to report them. Here’s how to do it:


  1. Launch the Feedback Hub, type a brief description of your problem into the search box and press enter.
  2. Select Add new feedback, mark it as a Problem (important).
  3. Provide any further information about the issue in the details section.
  4. Select Apps & Games category, Game Performance and Compatibility subcategory (important).
  5. Launch your game and get to the issue area.
  6. Click the "Start capture" button.
  7. Reproduce the problem for 10-15 seconds.
  8. Click the "Stop capture" button.
  9. Click the "Submit" button to send the bug to us.
