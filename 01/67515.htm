There is no "Documents and Settings" folder native to Windows 7.
Either you created it yourself (unlikely), or it was created by some
application incompatible with Windows 7. What you're "seeing" is called
a "Junction Point," whose sole purpose is to direct legacy applications
to the correct locations of user settings and data files; it's not a
folder that can be opened, nor does it have any contents.
