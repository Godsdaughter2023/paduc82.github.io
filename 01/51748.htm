Start Menu and Cortana Aren't Working - Restore Previous Version of Windows

There are numerous threads on this topic, so I started a new one with a fix to restore your previous version of Windows while they work on Windows 10. After installing Windows 10, I got the message, " * Critical Error * Start Menu and Cortana aren't working. We'll try to fix it the next time you sign in. "  As directed, I logged out and back in, shut off and restarted several times, but nothing worked. I tried F8 and other options. Web search showed you could revert to the previous version of Windows, but the critical error message wouldn’t let me use any Windows features as directed on Microsoft help. Even installed programs compatible with Win10 like Adobe InDesign weren’t working properly.

I tried a lot of the fixes in this thread and finally gave up and called Microsoft. After 2 hours on the phone with a very patient tech, I was able to roll back to my previous version, 8.1.  Try this:

  1. In the critical error message, click to log out.
  2. Don’t log in. At the bottom right, click the power symbol, then click Restart and hold down the Shift key. Keep holding Shift down while the computer goes down and restarts.  This should take you to a Windows menu page with a number of options. The last option will allow you to revert to your previous version of Windows "Go back to previous build" Enter your computer name and system password.
  3. If your computer comes back to the login window, repeat hold down Shift and Restart. I had to do it 3 times before it gave me the menu.

My computer hung up during the reversion process. It looked like it was working since the cursor was spinning, but I was getting no action on my disk drive light for about 5 minutes. The tech said shut it down, so I held down the power key for a hard shutdown. Then I turned the power back on and the reversion restarted. This time I was back to my old version and everything looks fine.  Be patient; some of these steps take 5 to 10 minutes.

I’m committed to going to Windows 10.  At least now I know I can go back if the upgrade doesn’t work next time.

How To Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.

  5. Select Encryption from the sub-menu.
  6. Verify the wireless passphrase or security key.

  1. Open a web browser.
