considerate_guy say:

DV-AVI is a standard definition digital video format that was used by tape-based camcorders. Since most everyone is using memory based camcorders these days, the support for DV-AVI was dropped after Windows Movie Maker 6.0. So any outdated instructions you are seeing about saving in that format must apply to that older version.

The green line seems to indicate an issue with Movie Maker being able to correctly decode the video, or that the video is using a non-standard codec. Do you see the green line on all video clip types? Do you see it on still photos too?

If it only appears on certain video clips and not photos, then I would suggest trying to convert the video to a more compatible format like .wmv. Then use that video inside of Movie Maker instead.

If it appears everywhere (even on photos), then I would suggest updating your video graphics driver.
