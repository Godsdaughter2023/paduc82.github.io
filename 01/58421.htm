Geethika,

It's possible that your over simplifying the abilities of Windows Movie Maker here. Let me explain.

Hardware

While your hardware may support DirectX 11, you need to make sure that you're running the most current version. You have not stated previously that you do have DirectX 11 installed. In addition to having the latest software installed, this also combats situations where the installations of DirectX could've been compromised/corrupted.

Windows Live Movie Maker uses several components; one of which is DirectX. You need to make sure that you're using the latest version of DirectX when working with Windows Live Movie Maker. You can also use this link to install/reinstall DirectX if you're having issues.

Link:
     http://support.microsoft.com/kb/179113 

Codecs

Windows Movie Maker does not support video formats unless the appropriate video codec is loaded. This information is noted in the frequently asked questions section on Microsoft's website. While the application is compatible with several different video formats, the video codecs have to be loaded; Microsoft does not provide all of the supported video codecs when you download the application. Microsoft even mentions that if required, you would have to purchase the required codecs for the video formats that you want to work with.

Reference:

http://windows.microsoft.com/en-us/windows/codecs-frequently-asked-questions#codecs-frequently-asked-questions=windows-7

Which codecs are included with the Player?

There are hundreds of audio and video codecs in use today. Some have been created by Microsoft, but the vast majority of codecs have been created by other companies, organizations, or individuals. By default, the Windows operating system and the Player include a number of the most popular codecs, such as Windows Media Audio, Windows Media Video, and MP3.

There might be times, however, when you want to play content that was compressed with a codec that Windows or the Player doesn't include by default. In many cases, you can download the necessary codec from the web for free or for a fee. And, in some cases, the Player can automatically use the codecs installed by other digital media playback and creation programs on your computer.

MP3 and AVI 

The AVI extension for video can be a canister for a wide array of video formats. Formats that perhaps not be supported by Windows Live Movie Maker. And if the AVI file isn't a pure AVI format, you'll need a matching codec to be able to properly read the AVI format you're using.  Converting the AVI to WMV is the safest bet before working with Windows Movie Maker. Personally, I would never use an AVI format; 90% of the time you're asking for trouble.

Also, MP3 files can have problems. They could have DRM injected into them or the MP3 files can be corrupted in some way.  

As mentioned previously, one of the best ways to approach these types of problems where you're having problems with video editing is to convert those video formats to make sure that the media [audio/video] structures are correct.

Generally speaking, these are typically the things you're going to run into when doing video editing; it really doesn't matter which video editing package you talking about. These types of issues always exist. They exist because there is no universal standard for A/V capture. when all the companies of the world agree on an A/V standard for recording, these issues will go away [most of them].

Again, Windows Movie Maker is not a high-end video editing package; it's free from Microsoft. it's existence goes from hero to zero every day.  

I hope this helps...

Larry
