To format or delete the Windows 7 partition, you need to start (boot) your computer using the earlier version of Windows.

Formatting a partition deletes the data on the partition but keeps the partition on your computer's hard disk. Deleting a partition results in unallocated disk space on the hard disk. You can create additional partitions from this space, or expand an existing partition so that it can make use of the space.

  1.   1.   3. Click Delete, click OK, and then click Restart.‍‍‍

Ideally, the percentage (70% as you mentioned) does not only account for the processes run by your particular user account. It consists of other background processes including the processes run by your user account out of the total RAM installed in the system.

Let me explain you the complete configuration of the total RAM installed in your system with the help of Resource Monitor image displayed below:

Image 1: Resource Monitor

Image 2: Task Manager (Users tab)

Image 1 shows a snap of Resource Monitor (Task Manager > Performance > Open Resource Monitor) that gives you the complete configuration of the total physical memory (4 GB RAM, in this case) including its usage and free memory available. The RAM is ideally divided into the following sections:

  • Hardware Reserved Memory
  • In Use Memory
  • Modified Memory
  • Standby Memory
  • Free Memory

If you hover the mouse over these sections in the Resource Monitor, you will get to know the information about each of these items.

Now, as highlighted in Image 1, the In Use memory is given as 1921 MB and the total RAM installed is equal to 4096 MB (4GB RAM). If you calculate the percentage, it comes out be 47% approximately(~46.8%).

However, as described in Image 2, the Task Manager shows only 710.9 MB of data being used by the user account.

So, a question arises, where is the rest of data being used when it displays 1921 MB of the In Use section?

The rest of the data is being used by the background Windows processes to run the complete Operating System and to operate various services required to run different hardware components connected to the system and other miscellaneous reserved processes. If you closely observe the Resource Monitor (Image 1), it will display the list of all the processes including those that are not mentioned in Image 2 and serve as background processes to operate the computer system completely.

By USB adapter, you're talking about the one designed for Windows, right? The little USB dongle that plugs into a PC and allows controllers to connect wirelessly.

I don't *think* this works on the console itself, but I cannot be sure. I would advise against spending money on that as a solution in the way you describe.

However, there are a couple alternatives:

-- If you don't mind using the controller as a wired controller, you can run a USB from console to where you play and plug the controller in.

-- Alternatively, using a Windows 10 PC in the location you actually play, you can stream the Xbox One's video output to the PC, but, more importantly for you, you can stream controller inputs back from PC to Xbox One.

Elaborating on the Windows 10 example, even if you plan to play on a TV (directly connected via HDMI to the console, or however the setup is currently), you can have a Windows 10 PC in the room to which you stream the video output across the local network (PC can be connected by wire to router for minimal latency), and to the PC you can either connect the controller by USB cable OR use the Wireless Adapter and have the controller connected to the PC wirelessly. In both cases, controller inputs go from Windows 10 PC back to console, and for video you could be looking at either your existing monitor or Windows 10 PC.

With the Windows 10 setup, with controller inputs and video output now being transmitted across the local network, you may introduce some latency to your gaming, but there's not really any good way to know before you try. If the PC is wired to the router, the latency should be minimal, but I cannot say for certain. What I do know is that I have streamed the Xbox One video output to my own PC (both connected wirelessly to router) on the highest settings, with a controller connected via USB to console, and I have not had any significant performance issues.
