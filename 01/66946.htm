Where did Processor Power Management go? (update 1703) - Microsoft Community

nika kvir say:

Where did Processor Power Management go? (update 1703)

So i just installed the new update and processor power management disappeared from my power options. Can anyone tell me where Microsoft moved it? 

marilou ser say:

Hi Nika,

Power options are unavailable on the Creators Update. You can try these workarounds and see if it helps:

Method 1- Run a series of scans in Command Prompt

  1. Go to Start
  2. Type cmd
  3. Launch Command Prompt as an Admin
  4. Type the following commands and press Enter after each one:
  • sfc /scannow
  • powercfg –restoredefaultschemes
  • DISM /Online /Cleanup-Image /RestoreHealth

Method 2- Tweak the Registry

If you're still not getting any Power options in the Start Menu, you can try the following steps to fix the issue:

  1. Open the Group Policy Editor
  2. Type gpedit.msc in Search
  3. Click Enter.
  4. Go to settings and double-click it.
  5. Then open the configuration box: User Configuration > Administrative Templates > Start Menu and Taskbar
  6. Select “Not Configured” or “Disabled”
  7. Click Apply.
  8. Exit after completing the steps above.

But, if you are using Windows 10 Home, the Group Policy Editor will not be available. But you can still run regedit to open the Registry Editor and go to the following registry key:

  1. HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
  2. Ensure that a value named NoClose exists with a value of 0.
  3. Reboot your PC 
  4. Then check whether the Power Options are back.

Let us know if the troubleshooting steps worked.

greg carmack - mvp 2010-2020 say:

The Control Panel is being merged into Settings gradually in Windows 10. 

Right click Start button to open Power Options in it's new Settings home.  To the right you'll see Additional Power Settings that will open up the old Control Panel Power Options, so you can choose Change Plan Settings next to your Power Plan - I suggest Balanced unless you're a gamer. 

In the Plan Settings, open Change Advanced Power Settings to see the familiar detailed settings including Processor Power Management.
