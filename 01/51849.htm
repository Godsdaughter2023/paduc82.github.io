Hello,

here is a temporary workaround that I found elsewhere on the net:

1. Update your printer's driver (let Windows and Device Manager search for updates). There will hardly be anything for the Laserjet 6P but you may never know :)

2. Go to Device Manager, open your parallel port's properties, choose port properties and check "Try not to use an interrupt" (it is the first option) and tick "Enable legacy Plug and Play detection".

3. Get DevCon.EXE from https://support.microsoft.com/EN-US/kb/311272

4. Write a small script with the following contents:

devcon enable *PNP0401

5. Put this script together with DevCon.EXE in a directory of your choice.

6. Create a scheduled task which executes this script on system start and/or user login. Let the script be executed under the SYSTEM account there.

That's it! At least my good old LaserJet 6P now works again like a charm :)

1) turn off the Xbox then

2) unplug the power from the router (Some folks have a combination modem-router AKA 'gateway'. If so, skip step 3).

3) unplug the power from the modem.

4) Wait at least 30 secs then replug the modem (or gateway) power.

5) Wait 'til the light on the front of the modem (or gateway) which says ONLINE, READY, CABLE or DSL is on solid (not flashing), then plug in the router. If you have a gateway instead of a modem and router, then when that light says it's ready, skip step 6.
