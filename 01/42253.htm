Hi,

What you read is correct.

Normal reasons for that:

1. Trying to Boot to Network Boot

2. Failed Hard Drive.

3. Loose connections between Motherboard and Hard Drive.

To see what is first in Boot Order and to see if Hard Drive is seen in Bios, you would normally press F2 or DELETE key when you first see the Splash Screen (Dell, HP Logo, etc).

If you do not see it, ask the Computer Manufactuer what Key to press to enter Setup/BIOS (it can be called by both names)

Cheers.
