SOLVED, you all can relax now!

The solution is so simple, I gave READ ONLY Permission to every entity listed in the RUN registry under both "HKey Current User" and "HKey Local Machine", my original mistake, I only dealt with the "HKey Current User" and forgot all about the most important one "HKey Local Machine". From now on, no program can write itself into the Run Registry Windows Startup, only the ones listed have that permission.
