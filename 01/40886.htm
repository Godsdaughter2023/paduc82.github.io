If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

net stop wuauserv
net stop BITS
net stop CryptSvc
cd c:\windows (Note: C: refers to the drive where Windows is installed)
ren SoftwareDistribution SoftwareDistribution.bak
net start wuauserv
net start BITS
net start CryptSvc

4. Try installing updates again and check if the issue is fixed.

    Important:     This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly.     c. Double-click on the key on the right-hand side for SpecialPollInterval.
    d.     Enter the value data as 604800 and the Base should be Decimal and not Hexadecimal.
