There are lots of options built into Windows to help you get out of this predicament:

Hi Shimshon79,

By default, most items in your Player Library are automatically given a rating. The auto rating is controlled by non-Microsoft data providers, such as AMG and some online stores. You can override an auto rating by explicitly choosing a new user rating. You can distinguish between these two types of ratings by the appearance of the stars, which are blue for auto ratings and bright yellow for user ratings.

There is no function in windows media player to disable auto rating. This is by design.
