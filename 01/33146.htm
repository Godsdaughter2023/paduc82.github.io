Here's how I solved message "unable to open comm port". - Microsoft Community

joeg314 say:

Here's how I solved message "unable to open comm port".

2011Oct12

(gervais.joe AT gmail DOT com)

I'm running Windows 7 on a new PC with a Dual Core processor, etc.

My COM3 port is used with my internal modem and phone line.

I was successfully running my NSC program; which was accessing

my phone line via the modem on COM3 port. However, when

I ran WFS (Windows Fax and Scan) for the first time, it set the

FAX service (FXSSVC.EXE) to AUTOMATICALLY start

whenever I restart my PC -- however, with that FAX service running,

my NSC program would produce the error message "unable to open comm port"

and would not run. So, in order to run my NSC program, I had

to use Task Manager to:

1) Find the FAX service (FXSSVC.exe) program;

2) "stop" that service.

3) Set that service to "manual" startup.

So now, if/whenever I run WFS and it causes my NSC program

to say "unable to open comm port", I just re-do those 3 steps

above to "stop" the FAX service (using Task Manager)

and then re-start my NSC program.

Hello!

To rename the file using Command Prompt ( easier ):

Open Windows Explorer and locate the folder that contains the file you want to rename.

Press CTRL+SHIFT on the keyboard and at the same time right-click the folder containing the file and choose "Open command window here".

In command prompt type:

ren mydocument.txt mydocument.doc [ENTER]

( to rename mydocument.txt to mydocument.doc"

exit [ENTER] -> to close Command Prompt

To rename the file using Windows Explorer:

Open Windows Explorer. Go to "View" section on the ribbon, click "Options" button.

In the new opened window, go to "View" tab, uncheck "Hide extensions for known file types". Click OK. Go to the file you want to rename, right-click it and choose "Rename" or press F2 key and type the new filename with desired extension.
