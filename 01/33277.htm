mac1-131 say:

Chances are you had a OEM SLP (system locked) Windows 7 key, which means you need the original install media from the PC manufacture to match the BIOS hash in your PC.  Then it goes just installs and no activation is needed.

When you reinstall Windows 7 from a different DVD (say an OEM version someone had) you don't have the system locked version from the manufacturer and it wants a product key now and it will have to pass validation.

So my guess is that is the situation you are in, and you don't have the correct restore media for your PC.

Sometimes PCs come with a backup product key and COA for this situation.  If so, you could activate your x64 Windows 7 with the backup COA on your computer and then proceed.  But if you don't have one of those either, then that is not an option.

Bottom line is you are going to have to get a clean Windows 7 activation before you can proceed now.  If would have been easier to fix this if you had done an image backup first, or had the correct restore media for your Windows 7.  I doubt MS is going to provide you with a complimentary Windows 10 key but who knows.

As for your USB Windows 10 install media being x32, it is easy to use the media creation tool to re-do your USB stick to Windows 10 x64.  You can do that on any PC.

good luck!

But apparently, that only affects the current user.

cmdrkeene say:

Sure thing, happy to help.  Just a quick disclaimer: registry edits can break your PC, so proceed with caution. That being said, this should be a very painless process.  Here's what I suggest.

  1. Create a system restore checkpoint.
    Better safe than sorry, and this only takes a second.  Type "restore point" into your Start menu and choose the 'create a restore point' option. In the box that opens, go ahead and click Create and name it something like "before making registry edits" and save it.


  2. Open the Registry Editor.
    Hold Windows Logo key  and type R to open the Run box, then run regedit to open the registry editor. Accept the UAC security prompt if you receive one.


  3. Navigate to your mouse settings.
    Navigate to your own user profile's mouse settings in the registry database. It works much like file folders, so navigate to this path:

    HKEY_Current_User\Control Panel\Mouse

    Once you open this key ("folder"), you'll see all your mouse settings on the right side of the window.


  4. Export your own mouse settings to a file
    Right-click the mouse folder, choose Export, and save it to your desktop.  Close the Registry Editor at this point, we're done with it.


  5. Modify the saved file.
    We're using that file as a template. Go to your desktop, right-click the file you saved, and choose Edit. It should open in Notepad.  We're only going to change the 3rd line in the file.  Here's the change you'll make:

    Old:  [HKEY_CURRENT_USER\Control Panel\Mouse]
    New:  [HKEY_USERS\.DEFAULT\Control Panel\Mouse]

    After making the change to that line, close and save the file.


  6. Import the modified settings into the registry.
    This one is the easy part. Just double-click that settings file, and allow it to import to the registry when prompted.

What this is doing is importing the settings from your account, and applying them to the .DEFAULT account, which is the account used on the logon screen.  So after applying these, restart the computer and you should find that the settings are now on the logon screen.

Please let me know how it goes.
