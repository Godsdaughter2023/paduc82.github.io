a. user say:

Hi,

 

Welcome to the Microsoft community forum.

 

When Windows installs a new update for a particular software, it generally removes any old updates for that software that is not needed anymore. Sometimes, some of these old updates might be still saved to be used by the rollback features. In case you face a problem after installing a new update, for example a driver, you can use the roll back driver feature to restore the old driver. These old updates are also needed even when you restore your computer to an earlier date.

 

Hence, it’s not recommended to remove Windows updates manually. If you have very less disk space, you can use the disk clean up utility to remove unwanted temporary files and recover disk space. Refer to this article for more information:
