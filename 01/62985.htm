Another method is available via the command line with PowerShell.

You run chkdsk by typing the following into the Start > Run box

     chkdsk /f

Click OK

A black Command Prompt window will open with the following message:

The type of the file system is NTFS.
Cannot lock current drive.

Chkdsk cannot run because the volume is in use by another
process.  Would you like to schedule this volume to be
checked the next time the system restarts? (Y/N)


Type Y (for Yes) and restart the computer.  Chkdsk will now run and attempt to repair disk errors.

However, unless you have an urgent need to use System Restore, you should first delete all System Restore points as follows:
  • Right-click My Computer > Properties > System Restore
  • Check the box to "Turn off System Restore on all drives"  and click OK.
Then start chkdsk as described above.

Don't forget to turn System Restore back on once chkdsk has completed.
