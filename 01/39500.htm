hrlyqn say:

I had the exact same issue on a pretty similar rig. My issue was I had programs set to install on a separate D: drive, instead of the C: drive windows was installed on.

You will need to change the program directory back to C: and problems solved (for me anyways.

type - %systemroot%\syswow64\regedit into the search bar
Go to: HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows \CurrentVersion
Look for ProgramFilesDir and ProgramFilesDir (x86) and change it back to C:Program Files(x86)

This fixed the issue for me and windows 10 allowed me to keep most of my programs and all my files.

Hope that helps.

Create a New Windows Address Book (WAB) File

The WAB file may have become damaged or corrupted causing one or both of the above error messages. You will need to rename the existing .wab file and restart Outlook Express(OE).   3. In the Search results window, Right-click the file(s) with an extension of ".wab" and then clickRename.
  4. Rename the file extension (the three left-most characters in the name) to "old" and press ENTER.
