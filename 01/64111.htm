How to create a custom refresh image in Windows 8

A program called recimg.exe allows you to create a custom refresh image that you can set to be used when you perform a refresh in Windows 8. This image will be a snapshot of the current state of Windows 8 and will include all applications that are currently installed on your computer. This allows you to create snapshots of Windows 8 over time and store them so that you can easily restore Windows 8 to various states if you run into an issue. The best part is that all applications that are installed when you create the custom image will still be there after you perform a refresh.
