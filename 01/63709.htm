Hey Bones,

 

TVicPort is set of general purpose device drivers. Tvicport.dll error notifications may appear while using or installing certain programs, when Windows startup or even shutdown. Tvicport.dll errors are caused by situations that lead to the removal or corruption of the tvicport.dll. In some cases, those errors could indicate mostly a registry problem, a virus or malware issue.

 

Here are two things you can try.

 

Method 1:

 

One of the installed software on the computer could be using this file. Check which application by performing clean boot. Clean boot should help you in tracing down the culprit application.

 

Refer:

 

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

 

Restart the computer once the system is in clean boot configuration. Follow the elimination methods provided in the article above to find out which application is causing this issue. Once you determine the application, either check for its updates from the vendor's website or you can post the application details here for further investigation.

 

Note: Ensure that you put your computer back to normal startup mode after completing the troubleshooting steps. Steps are available in the article given above.

 

Method 2:

 

Run System File Checker (SFC) scan to fix tvicport.dll error.This command will immediately initiate the Windows File Protection service to scan all protected files and verify their integrity, replacing any files with which it finds a problem.

 

Refer this for details.

 

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7

http://support.microsoft.com/kb/929833

 

In case SFC scan doesn’t repair the files, follow the steps given in the link above on how to repair those corrupted files.

Restore Factory Defaults

Login Information

  • Gateway - 192.168.1.1 or 192.168.0.1
  • Username - admin
  • Password - password

Restore Factory Defaults
*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)

  4. If prompted, enter "admin" for the user name and "password" for the password.

  5. Click Utilities from the menu at the top.

  6. Click Restore Default Settings in the menu on the left.
  7. Click the button labeled Restore Default Settings in the center of the page.
