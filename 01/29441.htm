Hi James,

I don't think this is an actual problem. It's the default behaviour of File Explorer. Once there's visual media content such as pictures or videos in a folder, Windows will create the 'thumbs.db' file and create a thumbnail for each item. The more videos in each folder and sub folder means that it's going to take a moment to do this. This is directly related with the power or speed of your CPU and Hard Disk Drive as the HEVC codec has to be engaged to create these thumbnails so disk and CPU are taxed heavily.
