Step 1

Let’s make sure that the Volume Shadow Copy service which is responsible for performing system restore is running properly.

1. Type services.msc in start search box, press Enter.
2. In services window, locate Volume Shadow Copy service in the services list.
3. Verify that the Status is Started. If the Status column is blank, right click on Volume Shadow Copy service and select Start.
4. If you get error trying to force start the Volume Shadow Copy service, try to change the Startup Type toAutomatic, and then restart the computer.

Step 2

You may check for the exact error code in event viewer.

1. Open Event Viewer by clicking the Start button > Control Panel > System and Maintenance > Administrative Tools, and then double-clicking Event Viewer > Windows logs. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

2. Click on System in the left pane.

3. Double-click an event to view the details of the event.

4. Post back the exact error code.

Open Event Viewer

http://windows.microsoft.com/en-US/windows7/Open-Event-Viewer

1.Click Start, right-click Computer, and then click Properties.

2.In the left pane, click on System protection.

3. Click on system protection tab.

4. Turn on system restore on the drive where you want to perform system restore. Under Protection Settings, click the disk,   and then click Configure.

5. Select the option” Restore system settings and previous versions of files”.

6.Click OK, and then click OK again.

Turn System Restore on or off

http://windows.microsoft.com/en-US/windows7/Turn-System-Restore-on-or-off

Additionally, check if you have sufficient disk space on all the drives where System Restore is enabled. To store restore points, you need at least 300 megabytes (MB) of free space on each disk that is 500 MB or larger. For more information refer this link.

    Hello,

    Thank you for posting your query in Microsoft Community.

    To diagnose the issue, I would suggest you to try the following methods and check if the issue persists.

    Method 1:

    Run the Power troubleshooter.

    a) Press the ‘Windows + W’ key on the keyboard.

    b) Type Troubleshooter in the Search box and then press Enter.

    c) Select Troubleshooting. Click on View All and select Power.

    d) Click Next and follow the on-screen instructions.

    2. In the Command Prompt, type the following command, and then press ENTER:
      sfc /scannow
