Wireless Security

 Login Information

  • Gateway - 192.168.2.1
  • Username - admin
  • Password - admin / admin1

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Enter http://192.168.2.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/admin or admin/admin1)
  3. Select Wireless Setup from the top menu.
  4. Select Which type of security you have from the left (WEP,WPA or WPA2)
  5. The password is displayed as passphrase.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Enter http://192.168.2.1 into the address bar. (Or click the link) Enter the username and password if the gateway asks. (admin/admin or admin/admin1)
  3. Select Wireless Setup from the top menu.
  4. Select Which type of security you have from the left (WEP,WPA or WPA2)
  5. Select the Disable radio button and click Apply. (To remove the security stop after this step)
  6. Select the other type of security on the left. (WEP,WPA or WPA2)
  7. Select the Enable radio button and type a desired passphrase.
  8. Click Apply.

Here is the advice I usually offer for that situation:

If you are using/relying on the built in NT Backup to backup your system, that is primitive technology (obviously - since it will ask you for a floppy when it is done).  If you have a disaster (even a minor one), you are going to have a big problem trying to recover your stuff if you don't have a floppy drive - and maybe even if you do have a floppy drive you will still have a big problem.

If you want to make a backup, an image or some kind of clone of your system, it makes sense to use modern technology - and that technology is not built in to XP.

There are lots of popular options for third party XP backup and imaging software(s).

A popular one folks seem to like is from Acronis.

With that (free 30 day trial) you backup, you can make a bootable CD or bootable USB (that's modern, huh?) and use that in the event of a disaster.

They used to have a totally free version - maybe not anymore - just the 30 day trial, then you have to buy it, but it is well respected.

It is so much more robust than the XP backup, it is laughable.

There are many other options if you do a little Googling...

Acronis

Whatever method you choose, it is prudent to test your method from end to end to be sure it works.

The day you need your backup/restore process to work is not the day to find out it doesn't work.
