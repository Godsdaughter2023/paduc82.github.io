It's probably one update in particular that the cover needs to be attached for, but I don't know which one.

Which files will be affected?

When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset.

What if I have multiple drives?

You can choose whether to reset all hard drives or just the one with Windows 8 installed on it. Any drive that Windows 8 doesn't recognize, such as those with non-Windows operating systems, will not be affected.

Hello,

Thank you for posting your query on Microsoft Community.

As per the description, I understand that you need assistance in converting basic disk to dynamic disk. I will certainly help you with this issue.

Answering to your questions respectively:

Question 1. Does the drive have to be changed to dynamic in order to use mirroring?

Answer: Yes, the drive needs to be change into dynamic volumes.

Question 2. Can I convert my bootable drive with the OS and all of my data to dynamic without any data loss, and will it retain the recovery partition that has been provided for recovery purposes?

Answer: When you convert the basic disk to a dynamic disk you must delete all disk partition from the disk and then use the “Convert To Dynamic Disk” command. If you want to keep your data, you must first back it up or move it to another volume.

Question 3. Is there any downside to having the disk as a dynamic instead of a basic, and if not why are all disks simply not dynamic to start with?

Question 4. Is using this type of mirroring the same as a hardware RAID 1 where if one drive fails the other can be used without any interruption and will be bootable?

Basic and Dynamic Disks

Upgrade Assistant: FAQ

I would also like to inform you that after converting the basic disk to a dynamic disk, you cannot change the dynamic volumes back to partitions. Instead, you must delete all dynamic volumes on the disk and then use the “Convert To Basic Disk” command. If you want to keep your data, you must first back it up or move it to another volume.

Important: Once you change the basic disk to dynamic disk, it cannot be changed back to basic disk without deleting the partitions and rebuilding the disk and subsequently resulting in data loss. Hence before proceeding, please backup all your data.
