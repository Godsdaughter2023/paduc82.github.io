Disk usage 100% - FIXED - Defender is the Problem

(NOTE: Read also my second post with Update)

My laptop was purchased with Windows 8 which has since been upgraded to Windows 10. I've been using Windows Defender and performance in general has always been good. Recently, I decided to run Defender's "Full" security scan simply to see if there was anything left undetected by the "Basic" scan. Sure enough, the Full Scan detected the following threat: (Trojan:Win32/Dynamer!ac). Defender labeled this as a "Severe" threat with the recommended action of "Remove this software immediately."

NOTE: Additional info --- containerfile:D\preload\install.wim --- file:D\preload\install.wim->(Image32408)ProgramFiles (x86)\HP Games\FATE The Cursed King\Fate=WT.exe->(EXEEmb)->(EXEEmb)

Clicking on "Additional Information" = Microsoft further declares this to be a severe threat.

So, after numerous attempts to have Defender remove this threat, it never did. It runs and runs, finally declaring that it's unable to remove it. In researching this issue, I find that it's being falsely tagged as a threat by Defender. Hmmm.....

OK, I can live with that, EXCEPT that, Defender now continually wants me to remove it. Defender no longer will run to a "Clean" ending. Perhaps the biggest problem now is that my Disk usage sits at 100% (per Task Manager: Windows Antimalware Executable), almost all the time. The fan is running hard and the computer is hot as a result. Function is extremely slow due to this condition and I'm concerned that the hard drive will self-destruct if I allow it to continue. "Research" has found many other people to be encountering this problem with many "work-arounds." Some I've tried to no avail.

Turning off Defender's on/off Switches improves nothing.

I read that if I install another program similar to Defender, that Defender will turn off. Thinking this would be a good way to test the issue, I installed Spybot Search & Destroy. Sure enough, Defender was turned off (switches become "greyed-out") in the process of installation. So, for the last couple days......my Disk usage has been "normal" at 0 to 10%.  Computer fan is slow and temperature is normal.

Clearly (to me) Defender is the problem here. Perhaps it's trying constantly to remove the "False" Trojan. If it's indeed "False," then an update to Defender is needed to ignore it (or) if it's indeed a "Threat," then an update is need to definitively remove it.

I've always liked using Defender and (in the past) MSE and would prefer to go back to it as my #1 Security Program, but I just can't tolerate what it's doing to my computer.
