Hi tigertail02,

This might be due to one of the following causes:

-Older Windows Live Movie Maker version
-System resources
-Unsupported media files
-Missing or incorrect Video/Audio codecs
-Outdated VGA card driver

Please refer to the steps below for further assistance.

1. Older Windows Live Movie Maker version - Ensure you have installed the latest version under: http://explore.live.com/windows-live-movie-maker.

-The current 2011 version: 15.4.3538.513
-The current 2009 version recommended for Windows XP users only: 14.0.8117.416

2. System resources - The more you add to Windows Live Movie Maker the more memory the application uses especially if you have a large video.

To check how much Windows Live Movie Maker is using from your system, please go to your Task Manager, click on Processes tab and from there you can see if it's using more memory. 

If your virtual memory is low, there are two things that you can do. Increase the minimum size of your paging file and Split the video in two parts.

A. Increase the minimum size of your paging file. Below are the steps on how to do so:

a. Open System by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking System.
b. In the left pane, click Advanced system settings.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
c. On the Advanced tab, under Performance, click Settings.
d. Click the Advanced tab, and then, under Virtual memory, click Change.
e. Clear the Automatically manage paging file size for all drives check box.
f. Under Drive [Volume Label], click the drive that contains the paging file you want to change.
g. Click Custom size, type a new size in megabytes in the Initial size (MB) or Maximum size (MB) box, click Set, and then click OK.

Increases in size usually don't require a restart, but if you decrease the size, you'll need to restart your computer for the changes to take effect. We recommend that you don't disable or delete the paging file.

B. SPLIT THE VIDEO - If you have a large video the best thing to do is to split the Video into smaller parts. This will help Windows Live Movie Maker deal with resources.

a. Click the video, and then drag the playback indicator to the point where you want to split the video.
b. Under Video Tools, on the Edit tab, in the Editing group, click Split.

3. Unsupported media files – Windows Live Movie Maker isn't compatible with a lot of formats it. Check if file types are supported by Windows Live Movie Maker http://explore.live.com/windows-live-movie-maker-file-types-faq

4. Missing or incorrect Video/Audio codecs - Audio and Video require codecs to play the file and it’s important to have the right codecs installed or updated in your PC.

Media info will be able to identify any issues with missing or faulty codecs and whether it’s already installed in your computer or not. You can get it at  http://mediainfo.sourceforge.net/en

Note: If can't find the codec you need you may convert the video to AVI and then import it in to Windows Live Movie Maker.

5. Outdated VGA card driver - Movie Maker makes extensive use of the video card GPU (graphics processing unit). Computers with video cards that are not intended for high-end graphics usage or older cards may encounter issues in rendering video and effects in Movie Maker.  The display driver affects the reliability, performance, and quality.

There are 2 ways how to determine the version and VGA card driver date to determine if it should be updated.

Control Panel

1. Click on Start button and type Device Manager in Search programs and files box.
2. Click Device Manager on top of the results list.
3. Look for Display Adapter and click the triangle/chevron next to it.
4. It should show the name of you display adapter/video card. Right click on it and left click on properties.
5. Click the Driver tab and you will see the driver details including the driver version.

Note: You may also run the update from here. On the Driver tab, click the Update Driver button and on the new window click Search automatically for updated driver software.

MS Diag/DirectX Diagnostic Tool.

1. Click on Start button and type dxdiag in the Search programs and files box and press Enter key on the keyboard.
2. Click on the Display tab and you will see the driver details including the driver version.

jimwynne say: What you are referring to are not "bugs."  Bugs are coding errors that are generally reproducible in different machines.  You have a local problem which will probably best remedied by backing up your data and doing a clean installation of Windows 10 using the Media Creation Tool.   Note that this will have no effect on program compatibility issues.  You also need to make sure that your motherboard drivers are compatible with Windows 10.  The best source of information for this is the PC or motherboard manufacturer.
