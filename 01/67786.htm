considerate_guy say:

The first thing to check is whether the files still exist on your hard disk in the original locations.  A project file in Windows Live Movie Maker does not store any video, photos or music.  It just stores a reference (the file name) to where the file is on your computer (the hard disk, a temporary folder, an external hard drive, etc.).  If you've done anything in the meantime to rename, move or delete those source files, WLMM will show you an exclamation point instead.

If you have just renamed or moved the files to another folder, simply double-click on the yellow triangle and choose 'Fix' then find the new location of the file.

As for the video shown inside Movie Maker, it only shows you a thumbnail view of the video clip where the first frame is shown.  But if you click play, you should see it move through the whole movie.  You can zoom into the storyboard/timeline by changing the zoom scroller in the bottom right.  That will give you an expanded view of the timeline.  You can also make splits to break up a long video into separate scenes.  Each split will cause Movie Maker to show a separate new thumbnail for that split off portion.

rohit chettri say:

Hi Ronia64,

 

1.       Are you logged in as the administrator?

2.       Are you able to perform other administrative level functions such as installing applications, creating accounts, making changes on the system etc?

3.       Have you made any recent software or hardware changes on the computer?

 

I would suggest you to start your computer with default/inbuilt administrator account and check for the issue.


To start the computer with default administrator account follow the below provided steps.

1. Open the Microsoft Management Console console and select Local Users and Groups. Click Start, type mmc and hit enter to open MMC console.
2. Right-click the Administrator account and select Properties.
The Administrator Properties Window appears.
3. On the General tab, clear the Account is Disabled check box.
4. Close the MMC console.
5. Log off, and you will now see the built-in Administrator account log on icon added (enabled) from the log on screen.
