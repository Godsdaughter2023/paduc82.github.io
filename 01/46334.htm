It sounds like you Start Menu is corrupted, a Work Around is:

When on your Desktop, Press Alt + F4 to bring up the old Shut Down dialog

With regards to your Start Menu, there is a registry fix that used to work on Windows 10, depending on the version you are running:

  1. Press Windows + R keys together to open the run dialog
  2. Type regedit and hit enter
  3. In Regedit navigate to: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
  4. In the right pane, if this key (NoClose) is set to 1, double clock it and set it to 0

Close Regedit and Reboot . . .
