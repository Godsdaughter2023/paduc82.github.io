jazzette7 say:

comrepl.exe

Does anyone know what this file is. I looked it up and some comments concerned me. Is it a virus?

lemp say:

It may be a virus.  That's why you should have a good antivirus program.

If the copy of this file on your computer is a virus, it's like many malware programs -- disguised by using the name of a legitimate Windows executable program but located in a different directory. 

Comrepl.exe should NOT be located in any of the following places:

  • C:\Windows\System32
  • C:\Windows\System32\drivers
  • C:\Windows
  • C:\Program Files (or subfolders of this)

See, e.g., http://www.file.net/process/comrepl.exe.html

On the other hand, it's probably not malware if it's located in:

  • C:\Windows\System32\com
  • C:\Windows\SysWOW64\com
  • or a few other places

If you're not confident of your antivirus program, upload the file to https://virustotal.com/

If Virus Total tells you that it IS malware, delete the file and replace your antivirus program with a better one.

Try resetting it. Here's how to reset Skype for Windows 10:

1) Click on Windows Start
2) Click on Settings
3) Click on Apps
4) Click on Skype
5) Click Advanced Options
6) Click on Reset
7) Restart Computer
8) Check Windows Store for Updates

If that doesn't work, I’d suggest downloading classic Skype and use it to see if that corrects your problem.
