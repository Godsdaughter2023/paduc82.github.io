    Hi Danya,

    1. Did you try connecting other bluetooth device to the computer?

    2. Is the bluetooth speaker working fine with other computer?

    3. Which Scan did you perform?

    A Code 10 error is generated in Device Manager in one of the following situations:

    • Device Manager can't start the device.
    • One of the drivers that the device needs does not start.
    • Device Manager has insufficient information to recognize the error that is bubbled up by the device driver.

    Error code 31 means this device is not working properly because Windows cannot load the drivers required for this device.

    The Bluetooth drivers usually are released by the computer or the Bluetooth device manufacturer. Microsoft only provides the generic drivers for most of the devices via Windows Update. So, you may check if there are any driver updates via Windows Update with which your Bluetooth device might work. If that does not work, then you may have to search the latest Windows 8.1 compatible drivers from the computer or the Bluetooth device manufacturer’s website and if the drivers are released, then you may install them and check if that resolves the issue.

    If the computer or the Bluetooth device manufacturer provides only the legacy drivers and does not have the latest Windows 8.1 compatible drivers, then we can still make these legacy drivers to work on Windows 8.1. This is done by installing them in their compatibility mode. To install any drivers/apps in its compatibility mode, you may refer to this article:

    Get your apps and devices working in Windows 8
