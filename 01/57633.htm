What you see in System Properties for RAM is going to be (approximately), the total RAM you have installed MINUS the amount of RAM that your other components are consuming.

For example, on my system I have 4GB of RAM installed.

System Properties says I have 2.68GB of RAM - how can this be and where is the rest of it!?

I have two video cards installed and one takes up 1GB of RAM and the other takes up 256MB of RAM, so that equals ~1.256GB of RAM (about).

So System Properties is showing me what XP has available after the RAM consumed my video cards has been consumed and subtracted from my 4GB, so it works out to be pretty close:

2.68GB from System Properties + the 1.256GB consumed by my video cards = about 3.936GB (pretty close to the 4GB I have installed), so that all makes perfect sense (the rest may be consumed by other smaller devices).  It may never quite all add up perfectly, but that is pretty darn close so it is acceptable to me.

                              ren C:\Windows\System32\catroot2 Catroot2.old

    4. Now, let’s restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt for this. Press the ENTER key after you type each command.

                            net start wuauserv

                            net start cryptSvc

                            net start bits

                            net start msiserver

    5. Type Exit in the Command Prompt to close it and then restart the computer.
