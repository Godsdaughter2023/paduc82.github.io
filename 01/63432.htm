a. user say:

Like Mike P says, once a computer is activated with Windows 10 it remains activated for life, no matter who owns it. That's because Windows 10 is registered to a computer not to a person.

The only exception is if you change the motherboard, because a computer with a new motherboard is considered a new computer.

The above is true for 98% of consumers. For the 2% of consumers, more or less, who buy a retail license for Windows 10, that type of license can be moved to a different computer, so when the license moves from computer 'A' to computer 'B' then computer 'A' loses its activation.
