sharief k say:

Hi Kathleen,

 

How can I prevent duplicate or invalid entries from being added to my library when I play music files?

When you move digital media files on your computer, the file name and file path information remain unchanged in your library. Then when you select a file to play from its new location, a new entry is created in your library if the option to automatically add it to your library when it is played is selected. As a result of this, your library may quickly contain a large number of duplicate or invalid entries.

To prevent music files from being added automatically to your library:

1. In Windows Media Player, on the Tools menu, click Options.

2. On the Player tab, clear the Add music files to library when played check box.

Now, when you play any music files on your computer or from the Internet, the file will not be added automatically to your library.

 

If you face any issues, try these methods and check if it helps.

 

Method 1: Clear the Windows Media Player database cache files.

 

a. Exit Windows Media Player.
b. Press Windows Key + R, type %LOCALAPPDATA%\Microsoft, and then click OK.

c. Select the Media Player folder, and then Delete.

d. Restart Windows Media Player.

 

Method 2: Clear the Windows Media Player database and check.


a. Exit Windows Media Player.
b. Press Windows Key + R, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.
c. Select all the files in the folder, and then click Delete on the File menu.
Note: You do not have to delete the folders that are in this folder.
d. Restart Windows Media Player.
Note: Windows Media Player automatically rebuilds the database.

keith a. miller say:

Yes. The default drop action for any file system folder can be modified by an entry in the desktop.ini file for that folder.  The desktop.ini file controls folder customization, most commonly  a custom icon and display name. But an entry like the following:

[.ShellClassInfo]
DefaultDropEffect=1

will, as the name implies, modify the drop action.

1 = Copy

2 = Move

4 = Create Shortcut

If the folder doesn't already have a desktop.ini file, the easiest way to create it & set the appropriate attributes is to use the Customize tab of thefolder's Properties dialog to customize the folder icon. You can always delete the icon erference when editing the file.

Craeting a shadow copy on FAT 32 external backup - Microsoft Community

goldcy say:

Craeting a shadow copy on FAT 32 external backup

Is there a way to save to the backup formatted in FAT 32. Windows 7 notice comes up saying that it has to save in NTFS
gaurav prakash say:

Hi GoldCy,

System Restore doesn't protect FAT32 and other FAT disks because FAT disks don't support the use of shadow copies. Shadow copies contain information about changes to documents and system files. Shadow copies require the NTFS file system. In this version of Windows, System Restore uses shadow copies to create restore points. If you store system files on a FAT disk, you cannot use System Restore to undo changes on that disk.

 

I would suggest you to convert a hard disk or partition to NTFS format.

The NTFS file system provides better performance and security for data on hard disks and partitions or volumes than the FAT file system used in some earlier versions of Windows.
