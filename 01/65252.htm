When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. To extract files from a .msi file at the command line, type:

msiexec /a PathToMSIFile /qb TARGETDIR=DirectoryToExtractTo

For example, to extract files from f:\zenworks\zfdagent.msi into c:\zfd701 you would type:

msiexec /a f:\zenworks\zfdagent.msi /qb TARGETDIR=c:\zfd701

andre da costa say:

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.

Autoruns can be a bit overwhelming.  You should be looking only in the "Logon" tab, not elsewhere.

Interestingly, a Google search for hp3170w.dll turned up no hits.  Did you spell this correctly?

On the other hand, hpqtra08.exe seems to be the HP Digital Imaging Monitor, and it is a very frequent cause of complaints.  It is my understanding that this application is needed if you want to use the buttons on the front of your all-in-one printer to manually scan documents or transfer photos from a camera. If you have uninstalled the printer, you certainly don't need to have a tray app to monitor whether the buttons are being pressed.

Go to Control Panel > Add or Remove Programs and look for "HP Photo Imaging" or "HP Image Zone" or similar entries.  Uninstall them.

Reboot the computer.

Click Start > All Programs > Startup.

If there is an entry under Startup that looks like "HP Digital Imaging Monitor" or something similar, right click and delete it.

Click Start > Printers and Faxes and then click File > Server Properties > Drivers tab.

If there is an entry for your old HP 3170, select it and click Remove.  Be careful not to remove a driver for any printer that you are still using.  If you are not sure, do not remove it.
