greg carmack - windows mvp 2010-2020 say:
Hi Manerr. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

steve winograd say:

Windows 7 doesn't encrypt files unless you tell it to.  If you have Windows 7 Professional or Ultimate, see Encrypt or decrypt a folder or file for details.  Home Editions of Windows 7 don't have file encryption.

When you encrypt files, Windows 7 creates a file encryption certificate and stores it on your computer.  If you want to use your encrypted files on another computer, you must transfer that certificate to the other computer.  If you lose the certificate, which can happen if your computer's hard disk fails, you won't be able to decrypt and open the files.  That's why it's important to back up the encryption certificate.

bossdweeb say:

- This is one of the rare games that using Compatibility settings helps with.

 Selecting  XP sp2 or Vista sp1 and  the 'run as admin' option should help get the game running. 

(R/click on the  LaunchGTA4.exe  in the games install folder and go to  Properties > Compatibility.)

- This issue can also come from a corrupted C++ reinstall (from when the game reinstalls C++ ).

If using the disk based version of the game you can find an installer for the C++ version the game uses

on the disk in - GTAIV > Redistributable > vcredist_x86.exe.

- If using the Steam version of the game the Redistributable folder should be in the games install folder.

- That's just two of several possible issues with this game.

Another issue you may face is with Games For Windows Live.

The game installs with an out of date GFWL. Installing a more recent version of GFWL may also help.

Download: Games for Windows Software Setup 3.5 - Microsoft Download Center - Download Details


- If you have problems with GFWL after installing version 3.5 you will need to download what is called an xliveless fix

(a replacement for the xlive.dll file in the games install folder). If you use this fix you can only play single player though. 

http://www.gta4-mods.com/tools/xliveless-097b-f2408


For more information,  visit one of the many GTA4 mod sites and forums.

- You will also need to patch the game . There are several patches, but I stopped at patch 1.4.

Rockstar's GTA site only has the 1.7 patch at the moment, but I don't know if this patch is cumulative,

or if you will need to use some of the earlier patches first.

Note that patch 1.7 improves shadows so if your machine is not powerful eno ugh to use the new

shadows you will need to turn the shadow setting in the games setup down. 

https://support.rockstargames.com/hc/en-us/sections/200034196-GTA-IV-Patches-and-Title-Updates

Patch 1.0 - 1.4 (other GTA sites may also have patch 1.5 and 1.6. )

http://www.thegtaplace.com/downloads/c76-gta-iv-patches
