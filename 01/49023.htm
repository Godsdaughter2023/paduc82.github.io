balaji kundalam say:
The issue is caused mainly due to the following reasons:
1. Wireless profile with the same name is already present in the PC.
2. Channel interference with other SSID.
3. WLAN AutoConfig and the dependent services are not running.
4. The network is protected (with MAC ID).
5. The network has number of connections restrictions.
6. Issue with the antennae connection to the WLAN card.
7. Issue with the WLAN card/drivers. 

Explanation:
1.Right click on the network icon on the system tray/notification area->click Open Network and Sharing Center. On the left pane, click Manage wireless networks. Here, delete any of the wireless profile saved with the same SSID with which you are having issues. Once done, check the connectivity.

2. This can occur with more than one SSID with same name (very likely, since the SSIDs are not case sensitive) or even with different names. 

3. When WLAN AutoConfig service or the dependent services are not running connectivity issue raise. But in your case, this is not possible, since you can connect to other networks.

4. & 5. Contact the network administrator for this.

6. When there is any issue with the antennae (loose connection or disconnection), the networks will be displayed, but can't be connected (to any of the networks). In your case, you mentioned you can connect to other networks.

7. As you mentioned, the device and the drivers are working properly.

Therefore, check with options 1, 2, 4 and 5.
