As you've seen, since your options and preferences are stored in your user account, the typing sound preference you choose doesn't take effect until you've logged in.  So at the login screen, the default setting is used (plays the sound).

 

I want to point out that you can change these defaults for the "default user" (used before no user has logged in), but it must be changed in the registry directly under "HKEY_Users\.default\Control Panel".  This isn't supported or recommended for users to edit, but it is possible.  If you decide to do so, just be sure to make a system restore checkpoint before you do it so you can undo the process if any unintended effects come up.

To know how To Reset Icon Cache In Windows 10
1. In Windows 10, right click on Start Button and select Command Prompt (Admin). On older Windows editions such as Windows 8/7, you can open administrative Command Prompt using search.

cd %homepath%\AppData\Local\Microsoft\Windows\Explorer
Then type following command and press Enter:

dir iconcache*


This should list all the icon cache files. There are total 15 files in the icon cache database.

3. Now execute following command:

taskkill /f /im explorer.exe
This will kill the Explorer process and your desktop background will disappear. Don’t worry, its completely safe and not permanent.

Then type following command and press Enter:

del iconcache*




4. The icon cache has been deleted by last command we used. To cross check, run dir iconcache* command and this time you’ll find File Not Found output with this command, which means icon cache was deleted successfully.



Now close Command Prompt and reboot the machine. After reboot Windows 10 will automatically generate new icon cache.

Hence, in this way, you’ve rebuild icon cache for your Windows. Note that this method doesn’t affect thumbnails.
