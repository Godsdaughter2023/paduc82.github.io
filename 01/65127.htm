rgharper911 say:

First off, there is no valid reason for CHKDSK to run when the computer is re-started.  It runs only when disk damage is detected, or when the system has not shut down cleanly, or if for some other reason the hard drive has recorded that the system was shut down improperly. You might see it run once and not worry about it, but if it runs repeatedly there is a problem you need to resolve - CHKDSK is a symptom, not the problem.  So find out why CHKDSK keeps running and resolve that problem and the CHKDSK issue will go away.

ken blake say:

Several points:

1. You can upgrade to 7, but not to 8 or 10. Going to 8 or 10 requires a clean installation.

2. The fact that Vista is 32-bit is irrelevant. Windows 10 comes in both 32-bit and 64-bit versions. Is Vista running on a 32-bit machine or a 64-bit one?

3. You don't say anything about the hardware configuration of the computer, but there's a good chance that such an old computer's hardware will be inadequate to run newer version with acceptable performance.

4. There is also a good chance that a newer version of Windows, especially 8 or 10, will need drivers that are not available for it.

5. To replace the current office suite, you can buy a newer version of Office. But first decide what version of Windows you are going to install, and then check what Microsoft Office version are compatible with it.

6. Instead of buying a newer version of Microsoft Office, you may want to download the free LibreOffice or OpenOffice. They are not completely compatible with Microsoft Office, but they are very close.

7. I recommend going to Windows 10. As far as I'm concerned, it makes no sense to install a older version that will go obsolete at most in a few years

8. Having said all the above, my recommendation is that you do not install Windows 7, 8, or 10 on it, but instead by a new computer that comes with Windows 10. I say that because if you need to upgrade the hardware (see points 3 and 4 above), and also buy a copy of Windows, just buying a new computer may not cost any more.

Several points:

1. You can upgrade to 7, but not to 8 or 10. Going to 8 or 10 requires a clean installation.

2. The fact that Vista is 32-bit is irrelevant. Windows 10 comes in both 32-bit and 64-bit versions. Is Vista running on a 32-bit machine or a 64-bit one?

3. You don't say anything about the hardware configuration of the computer, but there's a good chance that such an old computer's hardware will be inadequate to run newer version with acceptable performance.

4. There is also a good chance that a newer version of Windows, especially 8 or 10, will need drivers that are not available for it.

5. To replace the current office suite, you can buy a newer version of Office. But first decide what version of Windows you are going to install, and then check what Microsoft Office version are compatible with it.

6. Instead of buying a newer version of Microsoft Office, you may want to download the free LibreOffice or OpenOffice. They are not completely compatible with Microsoft Office, but they are very close.

7. I recommend going to Windows 10. As far as I'm concerned, it makes no sense to install a older version that will go obsolete at most in a few years

8. Having said all the above, my recommendation is that you do not install Windows 7, 8, or 10 on it, but instead by a new computer that comes with Windows 10. I say that because if you need to upgrade the hardware (see points 3 and 4 above), and also buy a copy of Windows, just buying a new computer may not cost any more.
