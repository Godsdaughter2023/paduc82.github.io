System requirements

If you want to upgrade to Windows 10 on your PC or tablet, here’s what it takes.

Latest OS:  Make sure you are running the latest version either Windows 7 SP1 or Windows 8.1 Update.

Display:  1024x600

Windows 10 uses approximately 10 GBs of disk space. The minimum disk space available is to facilitate the installation process. Some install files are compressed and need to be expanded during setup. To show hidden files type Folder Options in the search box above the Start button and select View, Advanced Settings and verify that the box before "Show hidden files and folders" is checked and "Hide protected operating system files" is unchecked. You may need to scroll down to see the second item. You should also make certain that the box before "Hide extensions for known file types" is not checked.
