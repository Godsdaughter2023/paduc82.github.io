There is no such thing as UEFI firmware settings. You have only BIOS settings present in a PC that is accessible by the user. UEFI is a part of the new BIOS standard. lemp say:

Most applications do not add a service.  What programs are you concerned about?  If you are referring to antivirus or other "security" programs, you should use the removal tool provided by the software's publisher, e.g., McAfee Consumer Products Removal tool or Norton Removal tool.

It's generally not a good idea to delete services unless you are certain you know what you're doing.  If you are, then delete this way (following assumes XP; later versions will need to use an elevated command prompt):

  1. Open the Services applet (Start > Run > services.msc)
  2. Stop the service to be deleted
  3. Double click the service to open its Properties
  4. On the General tab, make a note the Service name; this often will be different from the "Display Name", which is what is shown in the list of services; in XP, you can't copy the Service name, although you can in Windows 7.
  5. Close the Properties window and exit the Services applet
  6. Open a Command Prompt window (Start > Run > cmd > OK)
  7. Type the following in the black Command Prompt window and press Enter; "servicename" is the name that you noted in step 4; if there are spaces in servicename (most servicenames don't have spaces), enclose the name in double quotes
            sc delete servicename
