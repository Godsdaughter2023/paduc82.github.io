a.     Insert the CD from which you are trying to install the program to computer B.

f.       Look for the file Setup.exe or the installation file to install the program to computer A.

1. Click Start, type services.msc and hit enter.
2. From the list of services, double click on Windows Backup and Volume shadow copy services.
3. In the properties window, select Automatic as the startup type and click Start.
4. Apply the changes.
5. Try to open system restore and check the result.

Method 2: If the issue persists, check if system protection is turned off. To check this, follow these steps.
1. Click Start, right click on Computer, and click Properties.
2. Click System Protection under Tasks.
3. Under System Protection tab, make sure to place a check mark under all the available disks to enable the system restore or turn it on.
4. Apply the settings and check the result.
For more information, check this link:
http://windows.microsoft.com/en-us/windows7/System-Restore-frequently-asked-questions
Method 3: System File Checker (SFC)
SFC tool scans system files and replaces incorrect versions of the system files by using the correct versions.
To run the System File Checker tool, follow these steps:
a) Click Start, and then type cmd in the Start Search box.
b) Right-click cmd in the Programs list, and then click Run as administrator.
If you are prompted for an administrator password or confirmation, type your password or click Continue
c) At the command prompt, the following line, and then press ENTER:
sfc /scannow
How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7?
http://support.microsoft.com/kb/929833
