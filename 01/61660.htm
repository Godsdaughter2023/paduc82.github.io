bossdweeb say: While Win7 does install DX 9, 10 and 11 it does not install with all the files.
d3dx9_43.dll  is the most recent of the DX 9 files.
The best method when you have not previously updated DX is to use the DX Web Installer,
so you may want to try using that.
Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

If you have the same problem with the DX Web Installer try booting into Safe Mode with Networking
and try running the DX Web Installer from there.

- The Web Installer does not overwrite DX files, it only installs 'missing' DX files, so when there
is a corrupt DX files and you don't get an error with the name you will need to use the full
DirectX Redist (2010), which I assume is what you are trying to use, as this will overwrite all the
DX files.
- Similarly, you can use Safe Mode (networking not required as you have all the files) can
be used when there is a problem installing DX with the full installer.

- When there is a corrupt DX file and you get the name of that file in an error, such as with
your d3dx9_43.dll , you can delete that file from System32 (and sysWOW64 when using 64bit
Windows) and when you run the Web Installer it will replace the now 'missing' file with a new
copy.

Failing the above it may be your problem is more to do with not having permissions to
install DX.
This could be due to too high UAC settings, not running as Administrator, or some other
restrictive setting, maybe even your Anti-Virus (disable it temporarily when trying to install DX).
Try - R/click on the  DXSETUP.exe . Then go to Properties \ Compatibility and tick the box
next to Run... as Administrator.

Questions About Backup And Restore

The estimated size of this disc is 11.95 GB and it says it requires 4 DVDs. If a DVD holds 4.7 GB, wouldn't I need three? The reason I ask is that I only bought a 3-pack.
  4. Is the system repair disc factory drivers and applications as they come on the computer or a disc of current drivers and applications at the time the disc is made? It says it is 1.79 GB and requires 1 DVD.

  7. Does it matter that I've used my computer for a few days, customizing and tweaking it to my preference, before I make these system image and system repair discs?
steve winograd say:

Good questions!  Windows 7 has four backup/restore/repair options, each with a different purpose:

  • System Restore lets you return the system's programs and settings to the way they were at an earlier time, using restore points created by the System Protection feature.
  • The system repair disk fixes problems that can prevent Windows 7 from starting or running successfully.
  • The system image disk and manufacturer-provided recovery partition (like Acer's PQSERVICE) restore the computer to factory condition.
  • The Backup and Restore feature in Control Panel backs up the current state of the computer.  You can back up files, create an image of the entire disk, or both.

To answer your questions directly:

1. You're correct.

2. You don't need to set up System Protection on the factory PQSERVICE partition.  That partition lets you restore the computer to factory condition, which can be helpful if Windows 7 gets corrupted but the hard drive is still working.  It won't help if the hard drive crashes.  That's why you need to make the system image disks.

3. The system image disks restore the system to factory condition.  Why would 11.95 GB require four DVDs?  Perhaps it doesn't use the full capacity of each disk.

4. The system repair disk lets you repair problems that can prevent Windows 7 from starting or running successfully.  It loads a self-contained Windows recovery environment that has its own applications and drivers.  It doesn't use the applications and drivers from the running Windows 7 system.

5. The Backup and Restore feature in Control Panel can use an external hard drive.

6. The Backup and Restore feature in Control Panel backs up the current state of the computer.  That's a different function than creating system image disks.

7. It doesn't matter.

Troubleshooting log in issues in Windows 10 – Incorrect Password Errors

Technical Level : Intermediate

Summary

If you just upgraded to Windows 10, but you are unable to log in because of an incorrect password error, there might be a combination of factors why. In this article, we show you how you can resolve the problem.


Details

First, if believe you have forgotten your password and you were running a previous version of Windows such as Windows 8.0 or Windows 8.1, then review the following article. Also, this pertains to configurations that used a Microsoft Account, such as a Hotmail, Outlook, Live or MSN.com domains. If you were previously running Windows 7 or earlier or a local account on Windows 8.0 or Windows 8.1, then password reset might not work:

Keeping Passwords Secure- Microsoft Policy on Getting Around Passwords

If you upgraded from Windows 7 or Windows 8.0/8.1 with a local account, there might be a number of reasons why you are unable to log in.

Were you using a password at all in your previous version of Windows? If you were not using a password and you are prompted, leave the password box blank then hit Enter.

Check to make sure your language option on the Sign in screen is correct. If you are English speaker, check to make sure it is set to that language.

 

Were you using a PIN or a password? A password contains alphanumeric characters and possibly symbols, while a PIN consist of four numbers. Check to make sure you are choosing the right option. To do so, click the Sign-in options link, then choose the appropriate button.

 

Is caps lock on or is it possibly interchanging because of an issue with your keyboard? For instance, caps lock might not be working properly on your keyboard, when you are supposed to be entering capital letters, it is registering common letters. Use the password preview to check your password. You can also try using the on screen keyboard to log in.

  

If you have multiple user accounts setup on your computer, make sure you are logging into the right one.

The System Volume Information folder is a hidden system folder that keeps information about the drive and the drive manufacturer. Since it is a hidden folder, I would suggest you to enable the settings of show hidden files and folders and check if it is still there.

  1. Press Windows + X key on the keyboard
  2. Click on file explorer
  3. Click on view
  4. Options click change folder and search folders

  8. Click on Apply and Ok

You can prevent the system from creating restore points in your external hard drive by following these steps:

b) Click on “Create a restore point” in the search results
c) Now, in the new window, under “system protection” tab, select your External hard drive and click “Configure”
d) Click on “Disable System protection” and click Ok to save settings.
