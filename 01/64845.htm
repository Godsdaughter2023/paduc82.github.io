greg carmack - windows mvp 2010-2020 say:
Hi Richard. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

IT sounds like you were using the Media Creation Tool to create a bootable USB installation drive. It warns clearly that the drive will be formatted. Then if it's given the go ahead normally it formats the first 32gb of the drive since it only requires a drive up to 32gb and doesn't expect one any larger.

Perhaps it's because you used an external USB or much larger flash stick than expected that it jumped the gun on you. I don't know. I know that we do hear from others who have had this happen. What has worked for a few of them to recover files is to run Recuva: http://www.piriform.com/recuva/features

The reason it might work when it normally doesn't once a drive is formatted, is because it doesn't format any more than 32gb of the drive. The problem is that in that first 32gb is the Partition table that holds all the files together and it may have been destroyed by formatting.

In that case what has worked for some is running the Partition Recovery WIzard from a Partition manager like free Partition Wizard: http://partitionwizard.com/video-help/partition...

This pieces back together the Partition Table so the files can show up again and be copied out. It is worth a try. If PW doesn't work try Aoemi or another.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
