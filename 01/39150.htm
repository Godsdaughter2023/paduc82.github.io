what is formating disk f

I'm trying to upload pictures from an sd card a msg comes up that says I have to format the disk in drive f What disk?  the first time I put my sd cd in the pictures uploaded with no problem HELP what am I doing wrong
irfan h say:

Hello computor dunce,

·         Have you tried to format your SD card when you are prompted to format it?

If there are bad clusters on the file system of your Secure Data card, windows prompts you to format the physical drive.

If this is happening with your computer I suggest that you backup all the content (pictures) on a different computer and format the SD card by connecting to your computer.
Once you backup data on a different computer, run a Virus scan on the computer.

http://onecare.live.com/site/en-my/default.htm?mkt=en-my  

You can read the below post that will give you more information on ‘How to get rid of malware’

http://social.answers.microsoft.com/Forums/en-US/vistasecurity/thread/ba80504b-61f1-4d71-960f-b561798b7b42

If you have backed up the pictures you may format the Secure Data card as this would be helpful to use the card for the next use and would not prompt you to format the card.

Note: Formatting the Secure data card would delete all the content of the card.

To format the SD card follow the below steps:

1.      Right click on the Secure Data card and click on format.

2.      From the file system drop down, select NTFS and click on start.
