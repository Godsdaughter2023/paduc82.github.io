Now Click Ok

The names from within the Group will show up in the To field of the Email and all will work.

The To field in an Email cannot.

If it is your goal to replace the c:\windows\system32\msvcrt.dll file, there should already be a backup copy of the file in other places on your system.

If you boot into the XP Recovery Console, you should be in the C:\WINDOWS folder (that is what the prompt should say).

Does your Recovery Console prompt say C:\WINDOWS or something else?

Since we know that XP keeps backup copies of critical system files here:

c:\windows\system32\dllcache

Then from the Recovery Console, you would enter this command:

copy  c:\windows\system32\dllcache\msvcrt.dll    c:\windows\system32

Respond in the affirmative if asked to overwrite the existing file.

Depending on your installation, there might also be a copy of the file here:

c:\windows\ServicePackFiles\i386

So if the first copy attempt fails for some reason, you could enter this command:

copy  c:\windows\ServicePackFiles\i386\msvcrt.dll    c:\windows\system32

Also note that if you have Service Pack 3 installed on your system and the XP installation CD is not also Service Pack 3 (most people don't have this), if you expand the msvcrt.dll file from an older installation CD, you are going to have the wrong version.
