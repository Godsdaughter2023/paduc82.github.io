Doc

 

Change (or create) the associated "print" action for the file type:

Open Windows Explorer and click Tools > Folder Options > File Types.

Find the extension you're interested in, e.g., TIF.  If it's not in the list, click New and add it.

Click the Advanced button.  If there is no Advanced button, click the Restore button to restore to default.

When the Edit File Type dialog opens, you'll see a list of the actions that will appear when you right click a file with this extension.  In my XP Pro system, there is a "printto" action (which starts the Photo Printing Wizard).  If I click the Edit button, the Edit Action dialog opens and reveals the command line for the application used to perform the action, in this case

rundll32.exe C:\WINNT\system32\shimgvw.dll,ImageView_PrintTo /pt "%1" "%2" "%3" "%4"

So, you can either change the application command line for the printto action or (and this is probably a more conservative approach to start with) create a new print action with an appropriate command line.  To test, I created printIV which is intended to use IrfanView (a very versatile free file viewer) to print the file. The appropriate command line for IrfanView is

"C:\Program Files\Irfanview\i_view32.exe" "%1" /print

Once you have created the action, you can print the file by right-clicking and selecting your new action from the context menu.  If you want, you can make your new action the default action, which will then print the file when you double-click it (rather than opening the file).

The only catch with this method is that the application you select to do the printing must be capable of being invoked from the command line -- and you must know the correct switch and syntax to cause printing to happen.  I don't know that information for any of the apps you list (except, of course, Picture & Fax Viewer, which is the current default).  IrfanView can be obtainedhere .

One final caveat:  if you use XP Fax Services, you might not want to change the defaults for TIF files.  Adding a new option should be OK, however.

Copy the contents of "application used to perform action" for the printto command.

For reference, this is

     rundll32.exe C:\Windows\system32\shimgvw.dll,ImageView_PrintTo /pt "%1" "%2" "%3" "%4"

Leave the first argument as is.  Replace the second argument (%2) with the name of your default printer.  As shown, this needs to be surrounded by double quotes.  Replace the third and fourth argument with pairs of double quotes.  For example, if your default printer is HP LaserJet you would enter the following for your new right-click command:

    rundll32.exe C:\Windows\system32\shimgvw.dll,ImageView_PrintTo /pt "%1" "HP LaserJet" "" ""

Note that if you want to print to a shared network printer, use the form "\\computername\sharename"

genehart2832 say:

How to recover files after windows 10 update

We currently update to Windows 10 and have lost all our documents, pictures, videos etc. Since then, we tried going back to Windows 8.1 in hopes that our files would be back. They haven't and are nowhere to be found. How do I get them back? Also can't find the windows.old file. Please help!

a. user say:

Hello,

Thank you for posting your query in Microsoft Community.

I can imagine the inconvenience you have experienced. I will definitely help you with this.

  • How did you upgrade to Windows 10? Is it through Windows Update or an Installation Media?
  • Did you do a clean installation of Windows 10 using an media creation tool?

If you have performed clean installation, you would need to install all the application or software. As the clean installation erases the contents of the computer hard drive, your files, settings, and programs will not be saved any more.

Upgrading will not result in any loss of files or photos, I suggest you to check for windows.old folder on the drive where Windows 10 is installed. The Windows.old folder contains a copy of the previous Windows installation, and can be quite large in size.

Usually the path is C:\windows.old where is 'C' is the drive on which the previous Windows was installed and you upgraded it to Windows 10.

If you failed to find your files in windows.old folder, I suggest you to use your favorite search engine and search how to recover lost files from Windows. 

I hope this information helps. Please do let us know if you need any further assistance.
