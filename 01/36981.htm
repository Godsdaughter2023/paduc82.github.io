The Windows Media Player C00D1199 occurs due to various listed in the following article:

Windows Media Player C00D1199

In this case, I would suggest you to try the steps from the following link and check if the issue persists.

Windows Media Player Error Message Help

If the issue persists, refer to the following methods:

Method 1:

Run the Windows Media Player troubleshooter and check if it helps. The troubleshooter checks the configuration and network settings for Windows Media Player to make sure that you can play content you've downloaded and saved to your computer, or content streaming over a network or the Internet.

Steps to run Windows Media Player Settings troubleshooter are as follows:

a. Press Windows key + X and select Control Panel.
b. In the search box, type troubleshooter, and then click Troubleshooting.
c. Click View all, and then click Windows Media Player Settings.

If the issue still exists, try the next method.

Method 2:

Try turning WMP on and off from the Media features and check if the issue is reproduced.

  1. Press Windows key + X.
  2. Click Programs and Features.
  3. Click on Turn Windows features on or off.
  4. Expand Media Features.
  5. Clear the box for Windows Media Player.
  6. Click Yes on the warning message.
  7. Reboot your system.
  8. Repeat steps 1 to 4.
  9. Check mark the box for Windows Media Player.
  10. Click Yes on the warning message.
  11. Reboot your system.

Check if the issue persists. If so, follow the next method thoroughly.

Method 3:

Rebuild windows media player.

  1. Exit Windows Media Player.

  2. Press Windows key +R, type %LOCALAPPDATA%\Microsoft\Media Player in the Run dialog and then click OK.

  3. Select all the files in the folder, and then click Delete on the File menu.

    Note: You do not have to delete the folders that are in this folder.

  4. Restart Windows Media Player.
    Note Windows Media Player automatically rebuilds the database.

If this does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:

  1. Exit Windows Media Player.
  2. Press Windows key +R, type %LOCALAPPDATA%\Microsoft\Media Player in the Run dialog and then click OK.
  3. Select the Media Player folder, and then click Delete on the File menu.
  4. Restart Windows Media Player.
Note: Windows Media Player automatically rebuilds the database.

cmdrkeene say:

If you want to remove an account:

  1. Click Settings (the gear icon)
  2. Click accounts
  3. Click the one you want to delete, then click delete account

If the account is associated with your Microsoft Account (sync/backup/app store/purchases/etc.), then you can't delete it outright from here.  But you can tell it that you don't have a mailbox there and not to show it as a mail account.  To do this:

  1. Settings > Accounts > choose account again (same as above)
  2. Click "change mailbox sync settings"
  3. Scroll down and turn off the email option

Whichever kind of account you need to nix out of the mail app, that's how to do it.  Hope this helps!
