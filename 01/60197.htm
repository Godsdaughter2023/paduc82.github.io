Because Windows 7 is in limited stock because Microsoft ended sales in October 2013, this has created demand because of its scarcity. This means retail boxed copies demand a premium, while OEM System Builder copies remain affordable.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

There is nothing wrong with using it too, especially since all future releases of Windows will be full versions anyway.

frederik long say:

I have not experienced the problem myself but it appears to be caused by a live hibernation file. Here is way to deal with it:

  1. Set your BIOS so that it uses the USB flash drive as its primary boot device.
  2. Boot the machine with your System Repair flash disk.
  3. Open a Command Prompt.
  4. Type these commands until you find the hibernation file:
    dir  /a  C:\hiber*.*
    dir  /a  D:\hiber*.*
    dir  /a  E:\hiber*.*
    Make a note of the drive letter where the file was found. I shall call it Q:.
  5. Type the following commands exactly and press Enter after each:
    takeown  /f  Q:\hiber*.*
    cacls  Q:\hiber*.*  /g  everyone:F
    attrib  -h -r -s Q:\hiber*.*
    del  Q:\hiber*.*
    exit
  6. Retry the System Restore operation.

That's a distinct difference.[quote]When I go to friend's house an play with him, I don't get any reward or something because I don't play with my profile. So I thought that I could have a copy of my profile on a USB without removing it from my console.[/quote]From System Settings, Memory, I believe your profiles are stored under Gamer Profiles. From there you may Move the profile onto a USB Memory Unit. Once at your friend's house, you'll need to select your profile for your player so that the game applies the correct updates to your profile for achievement tracking.


The master record of your gamertag is stored on XBOX Live servers. The local copy you have is just a copy. When in doubt, Live will use the information stored on the Live servers over your local copy. If you play offline with your friend using your profile, remember to log into Live when you get home using your USB Memory Unit copy. If you recover your gamertag before syncing, you will lose any achievements gained.
