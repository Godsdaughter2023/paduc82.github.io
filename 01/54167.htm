If the OEM license is tied to the BIOS, could the BIOS be updated - Microsoft Community

craig finney say:

If the OEM license is tied to the BIOS, could the BIOS be updated without issues?

I am thinking of updating my Vista 64 bit to windows 7, laptop.  I was reading the fourm on the differences between OEM and Full and in one post it mentioned that the OEM license is tied to the BIOS serial #.  Is that a hardware serial number of a firmware #.  What would happend if you fast a new BIOS revison?

 

Thank you

george1009 say: a preinstalled OEM SLP key must interact with the slic table in the bios in order to self activate. You can update the bios with no ill effect as long as you get the proper bios for your machine.

A system builder OEM license does not use the slic table.

Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

 

A)   OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the special instructions found only on that Manufacturer's computer hardware. So when Windows was installed using the OEM SLP key (at the factory) Windows 7 looks at the motherboard and sees the special instructions and Self-Activates. (that's why you did not need to Activate your computer after you brought it home)

 

B)    COA SLP: This is the Product key that you see on the sticker on the side (or bottom,[ or inside the battery compartment]) of your computer. It is a valid product key, but should only be used in limited situations (such as if the OEM SLP key stops self-activating for whatever reason). The key must be activated by Phone. (Note: All manufacturers that use the OEM SLP system are required by contract to include a Certificate of Authenticity (COA) sticker, that has a COA SLP key, on the computer)

andre da costa say:

Windows 10 Education is an academic edition of Windows 10 used in schools. Features like Cortana are removed because of the environment it is used in.

Windows 10 Pro Education builds on the commercial version of Windows 10 Pro and provides important management controls needed in schools. Windows 10 Pro Education is effectively a variant of Windows 10 Pro that provides education-specific default settings, including the removal of Cortana1. These default settings disable tips, tricks and suggestions & Windows Store suggestions.

https://technet.microsoft.com/en-us/edu/windows/windows-editions-for-education-customers?f=255&MSPPError=-2147217396#windows-10-education

If you want to downgrade to the Windows 10 Pro edition, you will need to perform a clean install and you will need a unique product key to install it. The Windows 10 Pro Edu or Windows 10 Education key will not activate Windows 10 Pro.

If you don't have a license for Windows 10 Pro, you will either need to request it from the institution you got Windows 10 Education; or you will have to purchase a license for Pro, then perform a clean install.

maritzacapiro say:
Hi.

Windows 10 has a new, easy-to-use tool for freeing up disk space on your computer. It removes temporary files, system logs, previous Windows installations, and other files you probably don’t need.
To find this new tool, head to Settings > System > Storage. Click the “Free Up Space Now”
Windows automatically scans your PC for unnecessary data that it can removed to free up space. Unlike the old Disk Cleanup tool, this screen only shows data you can actually remove, and it scans both user files like your Recycle Bin and system data like old Windows installations at the same time.

floridasunshinegirl say:

HP Laptop with Windows 10

Laptop has power but Windows does not load.  I get HP logo with message attempting to repair, then black screen with nothing.  How can I fix this?

davem121 say:
Hi floridasunshinegirl, please try this:

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

Please try each option one at a time, then see of Windows will boot . . .

Go to Troubleshoot - Advanced Option - Startup Repair
See if Windows startup can repair itself

Go to Troubleshoot - Advanced Option - System Restore
Check to see if you have a restore point you can go back to

If you do not have a Restore Point, go to Troubleshoot - Advanced Option - Startup Settings and click Restart
Upon restart, press 4 to enter Safe Mode
Once in Safe Mode, backup your data if you do not already have a backup
Open Device Manager and check to make sure all your Drivers are up to date
Reboot to see if Windows will start Normally

If the above fails, go to Troubleshoot - Advanced Options - Reset this PC
Choose to keep your files and apps, just your files or to keep nothing
Then start the Reset

vijay a. verma say:
A. Try to boot into safe mode

Power ON your computer and as Windows / manufacturer's Logo appears, Power OFF - Repeat 3 times
Now your computer will go into Advanced Recovery Environment
Advanced Options > Troubleshoot > Advanced Options > Startup Settings > Restart
Press 4 or F4 to enable Safe Mode (5 or F5 to enable Safe Mode with Networking)
Log in once Windows starts in safe mode

B. Another option when you are in Advanced Options in previous step is to choose (Note - You won't lose any files in this step)
1. System Restore - This will restore your system to a point when things were working fine.
OR
2. Startup Repair - Startup Repair upgrade fixes all Windows errors and retains all files, applications and settings.

C. Penultimate option could be Reset this PC.
In Step A > Troubleshoot > Reset this PC - Keep my files and Remove everything. Choose this carefully as most likely you would like to keep your files.
All apps will be removed and windows will be reinstalled.

D. Final option could be Clean Reinstall -
You can use below guide by Greg Carmack, MVP for this purpose. This is the best method to perform a clean install of Windows 10. (If you want to rescue your files, there is a section Rescue Files when Windows Won't Start)

https://answers.microsoft.com/en-us/windows/for...

questions_ say:

1) Power on the computer and repetitively click the F2 key to boot to the BIOS UEFI diagnostics.

There are different versions that perform different tests.

The default version should test RAM and drive.

The installed versions should have extensive, loop until failure, and component testing.

Perform the available tests.

Indicate which tests were performed and report the results into the thread.

2) Open the HP website > software and drivers > enter the product or serial number or model > select the operating system > view drivers > post a URL or hyperlink into the thread

3) Find a flash drive that you can format (> or = 8 GB)

4) Create a bootable Windows 10 iso:

https://www.microsoft.com/en-us/software-download/windows10

https://www.tenforums.com/tutorials/9230-download-windows-10-iso-file.html

5) Remove the computer battery.

6) Turn the computer off and remove the power cord > hold down the power button for 30 seconds > insert the power cord > power on the computer > report into the thread what is seen and heard > reinstall the battery

7) If the computer was unable to boot perform the hardware method to open the computer's Windows advanced troubleshooting menu:

a) Power on the computer and as soon as you see the Microsoft icon > power off

b) Immediately power on and again as soon as you see the Microsoft icon power off

c) Power on the third time and allow the boot to progress as far as possible.

8) Report into the thread whether it was able or not able to open the Windows advanced troubleshooting menu on the computer.

9) If you were not able to boot to the computer's windows advanced troubleshooting menu then perform the following steps:

10) Insert the Windows 10 iso into any USB port

11) Power on the computer and repetitively click the F9 key to boot to the BIOS boot menu

12) Click external USB HD

13) It may take 5 - 10 minutes for the iso to load while viewing the Microsoft Windows icon

14) Select language, time, currency, and keyboard or click next

15) In the left lower corner click repair your computer (do not click install)

16) Click troubleshoot

17) Click startup repair

If this works it will complete the troubleshooting.

If this fails there may be a pop up for srttrail.txt > if you see the pop up > report into the thread

If startup repair fails > advance to the next step

18) Click System restore

Start with the oldest system restore point

Keep repeating system restore as necessary to the most recent restore point

If system restore fails or if there are no system restore points then advance to the next step

19) Click Command prompt > this should open a black box with title bar:  Administrator: X:\Windows\System32\cmd.exe and prompt:  X:\Sources>

20) Find a camera or smartphone camera to take pictures and post images into the thread

21) Use share links:  one drive, drop box, or google drive for posting images
