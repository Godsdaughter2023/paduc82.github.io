mac41two say:

How can I remove a duplicate "Add or remove programs" icon from the control panel?

Every time I open the control panel on my computer, I see two  "Add or Remove Programs" icons! How can I correct this problem?

a. user say:

Some of what you see in Control Panel comes from Control Panel Extension files that are found in the  c:\windows\system32 folder (assuming XP is installed on your C drive).  if you look our there you will find there are lots of .CPL files and those make up alot of what you see in Control Panel.

The Add or Remove Programs applet you see displayed in Control Panel shows up because there is a file out there called:

appwiz.cpl  

If you go to c:\windows\system32 and copy the appwiz.cpl file and call the copy something like jose.cpl then look in Control Panel, you will then see two Add or Remove Programs listed (and they both will work).  Non believers, try it.

If you see two copies of Add or Remove programs in Control Panel, do they both work?

I don't know what your problem is exactly or how it got that way, but if you use Windows Explorer and navigate to the c:\windows\system32 folder and then click Search for *.cpl, you will get a list of all the .cpl files in that folder (and maybe also see the backup copies in the dllcache folder).

Click the Size column heading to sort the .CPL files by size, then find appwiz.cpl in the list and then look for another .CPL file above or below that which has the exact same size and date as the appwiz.cpl file and see what the name of that file is (like jose.cpl).  If there is another file that is the exact same size and date as appwiz.cpl, that is your problem.

If you find it are not sure what it is, you can just rename the suspicious file to something that does not end with a .cpl extension (like jose.xxx) and see how things look.  If things get worse, just rename the file back to the original name and look for some other idee-ers.

If you find the file that looks just like appwiz.cpl and are not sure what it is, tell us what you see and somebody can help you figure it out.
