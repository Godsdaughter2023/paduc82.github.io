I am an independent advisor.

If your camera is not working in Windows 10, it is possible that the drivers are missing after a recent update, your antivirus may be blocking the camera, or your privacy settings may be set to not allow access to the camera.
Here's how you can check if your camera is not working and how to fix it. Try the following steps in order.

1) Check your privacy settings:

If you recently upgraded Windows 10, you might need to provide permissions to apps to use the camera. This is because, in newer versions of Windows 10, some apps do not have standard access to the camera.

To allow apps to access the camera, select the Start button and select Settings > Privacy > Camera. Then, enable Allow apps to access your camera. If this control is grayed out, enable Allow camera access on this device. If you do not see this option, you will need to ask your device administrator to enable this setting for you.

Once you allow access to the camera, you can choose which apps can access it from Choose which apps can access your camera.

Select the apps that can access the camera

2) Check your antivirus software:

In your antivirus software settings, look for settings that block access to the camera or permission to use your webcam. For help, visit the manufacturer's website for your antivirus software.

3) Check if there is an out-of-date webcam driver:

Another possible cause may be an outdated webcam driver.

1.Select the Start button, enter Device Manager, and then select Device Manager in the search results.
2. Find your webcam under Imaging Devices or Sound, Video, and Game Controllers.
3.Press and hold the webcam name (or right-click on it) and select Properties.
4. Select the Driver tab, select the Driver Details button, and search for a file name that includes stream.sys. If it's there, your webcam was designed before Windows 7, and you'll need to replace it with a newer model.
5. If you can not find a file name that includes stream.sys, try to revert the webcam driver.

4) Revert the webcam driver:

1. In the Device Manager, press and hold the webcam (or right-click on it) and select Properties.
2. Select the Driver tab, select Reverse Driver, and choose Yes. Note that some drivers do not offer a rollback option.
3.After the rollback is complete, restart your computer and try opening the Camera app again.
4.If the rollback did not work or is not available, uninstall the webcam driver and check for any hardware changes.

5) Uninstall the webcam driver and check for hardware changes:

1. In the Device Manager, press and hold the webcam (or right-click on it) and select Properties.
2.Select the Driver tab, select Uninstall> Delete the driver for this device, and select OK.
3. In Device Manager, from the Action menu, select Check for hardware changes. Wait for the scan to finish and reinstall the updated drivers, restart the computer, and try opening the Camera application again.
4. If the webcam still does not work, get help on the webcam company website.

6) My camera could not be found in Device Manager:

If your camera is not found in Device Manager, first try using the camera with a different port on your device. If that does not work, try these steps:

1. In the Device Manager, from the Action menu, select Check for hardware changes. Wait for the scan to finish and reinstall the updated drivers, restart your computer, and try opening the Camera app again.
2. If the webcam still does not work, visit the website of the webcam manufacturer.


7. Find answers in forums:

If the previous workarounds do not work, try searching for similar issues in the Microsoft Community Forum for the Windows Camera. If you can not find a solution, share the details of your experience there and come back later to check the answers.

8. Send feedback:

If you have tried the steps in this article and have not been able to find a solution that works, use the Windows Feedback Hub to record a bug. This helps Microsoft find the root cause of this problem.

Include details about what you were doing or the specific steps that were being taken when the problem occurred. Use Re-create my issue when submitting your feedback in the Feedback Hub.

In Windows 10, right click anywhere on open desktop. Then click "Display Settings". Next, click "Tablet mode". Set "Make Windows more touch-friendly" to OFF. "When I sign in" to "Go to the desktop" and "When this device automatically switches" to Don't ask me and don't switch".
