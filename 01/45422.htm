How to save my movie in a DVD or stick? - Microsoft Community

smithsfalls say:

How to save my movie in a DVD or stick?

movie maker

 

I have made my movie and now want to save it on a dvd or stick.  How can I do that?

considerate_guy say:

Assuming you have already saved your movie using File and Save movie you should have a .wmv or .mp4 file saved on your hard disk.  To put it onto a USB drive is just a simple matter or using Windows Explorer to copy the file from where it is, to the drive representing the USB stick.  Alternatively, you could use File > Save movie and save it directly onto the USB drive.

 

For a DVD, it is a little more complicated.  If you just want a data DVD for archival purposes, you can burn the file onto a DVD directly.  That will be playable in another Windows PC, but not in a DVD player.  If you want to make a playable DVD, you need a program for making playable video DVDs.  Windows Vista and Windows 7 came with Windows DVD Maker, so you could use that.  If you have it, Movie Maker will show the option under the File menu for Burn a DVD.  If you don't you'll need either freeware or 3rd party software for making a video DVD.
