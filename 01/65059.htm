a. user say:

It means that one of the modules is not downloading properly (so it's failing the test to see if it's good) and it has failed to download properly multiple times. If your computer is over 4 years old, I'd check the hard drive for any problems:

  1. open the start menu
  2. in the search bar, type in cmd
  3. right click the result and select Run as administrator
  4. type in this next line exactly:
  5. chkdsk /r
  6. it will probably say you need to restart the computer, do so
  7. DO NOT PRESS ANY KEY (since that skips what we're trying to run)
  8. this can take several hours to run

Basically, this checks your HDD for bad sectors and makes them off limits for data. Any data that's written to these bad sectors is automatically corrupted. So, if nothing else, this will rule out that possibility, and at worse fix the corruption problem.

 

After that has run, you're back to the earlier step where you have to trigger this to download again.

  1. open up the start menu
  2. type in cmd into the search bar
  3. right click and run as an admin
  4. type in this next line exactly as is
  5. bitsadmin /reset /allusers
  6. you should see cancelled jobs
  7. close out of the command prompt
  8. double click on the upgrade icon on the desktop
