How to Activate Windows 7 (the right way) - Microsoft Community

-jamal say:

How to Activate Windows 7 (the right way)

You know what I really hate? Purchasing a computer only to find out that the OS is preloaded with a bunch of bloatware. I usually wipe the hard drive and perform a clean install.

Now I will explain what a lot of users do wrong when trying this. When all is done and the product key is inserted from the back of the laptop or on top of a desktop, the user might receive a message saying that his/her product key is invalid. Now how could that be? They bought the laptop from a legit source so how can it say that it is invalid? Invalid doesn't have to mean that the product key is non-genuine as what most people assume. It might not be activated as yet and you may be required to call an automated phone system to do so. I will run through the steps.

Scenario: The user is disappointed that Windows 7 is preloaded with junk from the Computer Manufacturer. The DVD that came with the system restores the same annoying programs. What to do?

1. Look for your exact Windows 7 version on this website to download a retail copy ISO.
2. Burn it to a DVD using imgburn or the ISO burner that came with Windows 7.
3. Write down the product key located physically behind your laptop or on your system unit.
4. Set the first boot device in the BIOS to be CD/DVD or use boot manager to boot from the DVD drive.
5. Install Windows 7 (Enter the product key and uncheck the description that says to automatically activate).
6. When you're finally on the desktop. Right click Computer > Properties and scroll down. Click the link to activate windows.
7. If it says that it is invalid, there should be an option saying Use automated phone system to activate.
8. Select the nearest location to you and you will be given a phone number.
9. Call the number and a robot will ask you to call out a certain amount of numbers and then tell you a bunch of numbers to insert into a field to activate your product. When you're finished, select OK and you should receive the Genuine Windows stamp.

Note: Speak clearly when calling out the numbers, if the robot cannot understand what you are saying for a certain amount of times then you will be transferred to an agent to perform the process.

Suggestion: Never give out your product key to anyone! Keep it in a safe place.

spiritx say:

Hi,

Some of your ports are powered, in pairs so you have 2 or 4, and those could be on the back.
If you have a device that really needs a powered hub you might get the same message. It is
also possible there is some weakness in the USB system or even the cabling to the front ports.
Maybe a USB device you are using makes the machine think it is a 1.1 type.

Another thing you can do is Control Panel - Device Manager - USB Controllers - check the chipset
mine say Intel(R) N10/ICH7 - double click each - Advanced Tab - uncheck the "Tell me if my
device can perform faster". And it may not be set for all of them.

On a PC the on-board USB can be disabled in the BIOS and an add-in USB card used which is
not expensive. So if there is something wrong with the USB that affects performance it can be
fixed easy enough.
