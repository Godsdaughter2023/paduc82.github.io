New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

 

This is how the recovery partition is accessed for the most popular makes

For Dell, press CTRL + F11 directly after powering up the unit

For HP, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

medflex say:

ICC profiles are not a program to be compatible with, but the way color management is handled.

Some applications in Windows 7 like Microsoft Photo Viewer supported V4 profiles in slide show mode but only V2 profiles (the older ICC standard) when not in slide show mode.  This was annoyingly inconsistent.

So another way to ask the question might be how well implemented are the Microsoft Apps and Windows 10 in general to the latest ICC color profile standard or are their still weird inconsistencies like in prior operating systems.

A related question would be does the Edge browser properly support Version 4 ICC profiles.  Prior browser support was another problem area where browsers are concerned.

For instance the latest version Chrome supports only v2 icc profiles (at least in my test).

While IE 11 supports V4 and 2 ICC profiles which makes IE 11 a much better browser if you have profiled your monitor with an up to date tool and want to view accurate colors while browsing as compared to the Chrome browser.

Here's an interesting link one can use to see what level of ICC support applications have:

http://www.color.org/version4html.xalter

The box at the top of the page is broken up into 4 quadrants with different color standards.  How this page looks on your monitor in different apps will answer the questions about which versions of ICC support exist or if there is no ICC support.

I've upgraded a couple of my computers to Win10 but have not profiled them yet.  My profiling app is the latest i1 toolset can calibrator from xrite.  This creates Version 4 ICC profiles which can be problematic.  The older Version 2 profiles are generally supported.  I can check this out.  Of course, an official response about the level of ICC V4 color management support by the major Microsoft apps in Win10 would be nice to know without having to do the basic research.  And clearly, the Windows Compatibility Center, since we are not talking about application compatibility, not going to help answer the original question.

Hopefully that clarifies the Original Poster's question.

I would imagine since IE11 had V4 support that Edge still does, but since Photo Viewer had mixed support within the same app, no way to know without testing.
