considerate_guy say:

There are two files to worry about:

1) The .wlmp (project) file - this is like the recipe listing all the ingredients, how they go together, what added effects and transitions are applied, what captions are added, etc.  This is file you need if you are going to directly change any of the parts.

2) The .wmv (movie) file - this is like the final baked cake.  This is generally the last step in your process and creates a single video/audio file that can be viewed, shared, uploaded, etc.

In essence you are asking to turn a cake (movie) into a recipe (project) and that is going to be rather hard to do.  The movie doesn't have any knowledge of the individual files, the individual captions, the individual effects & transitions, etc.  It's been all blended and baked together into a new format.

An equivalent analogy would be trying to take a physical printout and turning it back into a Word document.

The first thing I would do is verify that you don't have the actual project file.  Movie Maker prompts you always to save your project before exiting.  So unless you explicitly said "no", you have one saved. Open Movie Maker and under the File menu (Alt+F), check Recent Projects.

Assuming you don't have that, then you might be able to make some edits to your new movie. You would start a new Movie Maker project and add this video.  You won't find any splits and individual parts, but you can perhaps manually make splits, delete spots you don't like, add new photos and video, add new captions (though you can't change the ones that are already "baked in"), mute the audio and add different audio, etc.  Again using the cake analogy, you might be able to cover over the cake in a different frosting or perhaps add another layer, but your editing will be limited.

Hello CatieWaite,
