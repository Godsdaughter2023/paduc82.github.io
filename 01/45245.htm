a. user say:

Hi Anthony,

Thank you for posting your query in Microsoft Community.

I understand you wish to transfer your contacts from Windows Live Mail to Gmail account and, so would suggest you to follow the below provided steps:

  1. Open Windows Mail
  2. Select ‘File’, then ‘Export’, and finally ‘Windows Contacts…’
  3. Select ‘CSV (Comma Separated Values)’ and click ‘Export’ when the ‘Export Windows Contacts’ window pops up.
  4. Click ‘Browse’ on the next pop up window and select a name (e.g. “contacts”) and location (e.g. desktop) to save your contacts.csv file to.

Now, follow the steps to import contacts in Gmail account.

  1. Open Gmail in your browser.
  2. Click on the ‘Contacts’ link on the left sidebar.
  3. Click the ‘Import’ link in the upper right of the next page that appears.
  4. Click the ‘Choose File’ button.
  5. Select your contacts.csv file from where you saved it on your computer in the first part.
  6. If you want to add these contacts to a specific group, check the box and select the group.
  7. Click the ‘Import’ button and you are done.

One problem with this transfer is that if your Windows Contacts have more than one email address per contact, then extra Gmail contacts will be made and you have to manually transfer those addresses into one contact. If you are running into errors, formatting of the .csv file is probably to blame, and help to correct this can be found on the Gmail help forums.

Hope this information is helpful. Do let us know if you need any further assistance, we will be glad to assist you.
