Here you go:

1. Click the Start globe.
2. Type the letters   cmd   and press Enter.
3. Type this command and press Enter:
   control  userpasswords2
4. Single-click the new account name.
5. Untick the box that requires users to enter a password.
6. Click OK.
7. Enter the password when prompted and click OK.
8. Make a written note of this password (because you will soon forget it!)
