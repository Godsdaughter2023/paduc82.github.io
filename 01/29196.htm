    Hi Pete,

    This watermark at the bottom right of your desktop usually appears if you installed an application whose drivers aren't digitally signed by Microsoft & if they are still in the testing phase. Please note that this cannot be removed as long as you are in test mode.

    To disable test-signing, kindly follow these steps:

    1. Press Windows key + X and select Command Prompt (Admin)
    2. In the Command Prompt window, type the following command, and then press Enter:

             bcdedit -set TESTSIGNING OFF

         3. After you see the confirmation, close the Command Prompt window.

         4. Save any unsaved work, and then restart the computer.
