Ome motherboards will have a setting in the bios to make the device work automatically , set to on or set to off.

If the bios detects a physical device sound or video card it will shut off the  onboard.

Some  bios give an option to run both onboard and physical devices at the same time.

Most OEM like hp, dell, toshiba, etc will set this this way,especially on older  systems.

Even though there can be multiple output and input devices, the one you want to use needs to set as the default each time you set it.

I know on win7 and 8 adding or removing a some input-output devices would make it change on the fly, but win10 audio is a different beast.  Does the system board have an audio header for hdmi hookup?

Is there a audio header spfid (2 pin) connector ) on the motherboard, was designed to port the onboard

from the system board through the display card (gpu) than out through the hdmi, but newer gpu's and audio devices

do not use spfid any longer (removed from microsoft). 

How many gpus, audio devices are in the system; cpu based video (intel), amd apu' ( on die chip), physical cards,

onboard audio, need the device name also.
