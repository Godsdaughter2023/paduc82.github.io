afzal taher say:

Hi,

If you are working offline and make changes to offline files from a network folder, Windows automatically syncs any changes you made to the files the next time you connect
to that network folder. If you are working offline while someone else changes a file in a shared network folder, Windows syncs those changes with the offline file on your computer
the next time you connect to that network folder.

 
If you have also changed the file since you last connected to the network folder, a sync conflict will occur and Windows will ask you which version you want to keep. Otherwise, there
will not be any conflicts and you can use the offline files normally.


Refer these links for more information about offline files:
http://windows.microsoft.com/en-US/windows7/Understanding-offline-files

paulmcilroy say:

IPhone 4s / PC Bluetooth wont link

IPhone 4s wont bluetooth link with PC, Unavailable driver?? Any ideas?

foxfifth say: By Apple's design the iPhone connectivity to a computer (Apple or Windows) via bluetooth is extremely limited.  The iPhone does not currently support syncing or file or picture transfer to a computer via bluetooth.  It only supports sharing the phone's cellular internet data connection with the pc and that is only in some countries with some carriers. 

If you are trying to connect via bluetooth to sync or to transfer pictures or files, that is not supported by Apple.  Syncing and picture transfer is only via iCloud, USB cable, or with iOS 5 and later via wifi if both the pc and iPhone are on the same router.


Apple doesn't  brag about the lack of Bluetooth connectivity to computers and you really have to dig to find that it basically isn't there.  If you look at the user guide's instructions for synching, they tell you how to do it via connecting the USB cable and via iCloud and gloss right over that you can't do it via bluetooth.  Similarly, the following is a direct quote from the iPhone User Guide (iOS 5) which is one of the places where they say what it does do but gloss right over what it doesn't do:

"You can use iPhone with the Apple Wireless Keyboard and other Bluetooth devices, such as Bluetooth headsets, car kits, and stereo headphones. Third-party Bluetooth headphones may support volume and playback controls. See the documentation that came with your Bluetooth device. For supported Bluetooth profiles, go to support.apple.com/kb/HT3647"
