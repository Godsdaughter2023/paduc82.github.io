wunders say:

Another thing to consider is that if you have a quad or dual-core CPU, a single task tends to run on a single core only.  Therefore, if it takes 100% of the CPU on one of four cores, the CPU meter will result with a reading of 25% assuming the other cores are idle.

Instead of Windows Task Manager, you might want to try [freeware] Process Explorer.  It gives you a lot more information about what is going on in your computer.  Once you use it, you'll never go back to Task Manager.

Process Explorer: < http://technet.microsoft.com/en-us/sysinternals/bb896653.aspx >
