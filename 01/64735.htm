When trying to start the Print Spooler Service, Error 1068 means "The dependency service does not exist. or the dependency service or group failed to start." 

Error 126 usually means that some module, named in the error message, can't be found.

In general, it's much more helpful if you quote the entire text of error messages, without paraphrasing.

Without more information from you, my guess is that you used to have a Lexmark printer (or a Dell-badged Lexmark printer) and then uninstalled it. 

So ...
What are the makes and models of all printers currently installed?
Did you ever have a Lexmark or Dell printer installed?
What happened to your system prior to the problem occurring?

To return your printing system to its "normal" configuration,
Click Start, then click Run.
Type the following in the "Open" box and click OK
    sc config spooler depend= RPCSS

make sure that there is a space after the = sign but not before it

You should be able to print without those errors now.

How to Grant Permissions in Windows 7

1. Locate the file or folder in which you want to take ownership in Windows Explorer.
2. Right click on file or folder and select Properties from the Menu
3. Click on the Security tab, then click the Edit button. Click OK to confirm UAC elevation request if needed.
4. Select user/group from permission windows or click add to add other user or group (you may need to do this for multiple users - ex: administrator and your own user account).
5. Now under the Permission section check the rights which you want for the user i.e check "Full Control" under the "Allow" column to assign full access rights control permissions to the user.
6. Click OK for changes to take effect, and click OK again to exit from the Properties window.

Disable User Account Control (UAC) on Windows 7

1. Type UAC into the Start Menu or Control Panel search box.
2. Click Change User Account Control Settings.
3. In the windows that pops up drag the slider up or down, depending on the setting you want.
    -If you drag it all the way to the bottom, you will disabled it entirely.

Please note that disabling UAC completely is not recommended.
