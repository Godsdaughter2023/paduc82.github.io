yolvi ell say:

Hi Bob,

The Windows Boot Manager would normally be the first thing to show up at startup once you have two operating systems installed on your computer. This is where you can choose which operating system to use. This uses your Boot Configuration Data (BCD).

Since you first installed Windows 7 on your C drive, the BCD is stored there, even for the G drive. If you want to boot the G drive while your C drive is not plugged in, reinstall Windows 7 on your G drive while C drive is not plugged in as well. Don't haphazardly change the drive letter as it could corrupt everything on that drive. The G drive will effectively be reformatted as a second C drive. After doing this, you can plug in your original C drive and simply change the boot order through your BIOS.

Let us know if you have any other concerns about your computer.

mohammad _ imran say:

Hi Rickshaw,

 

Thank you for posting your query in the Microsoft Community.

 

From the issue description, you are facing issue with changing Windows XP Operating System language from Dutch to English.

 

You need to install English Language pack.

 

I would advise you to review the eligibility requirements from the below mentioned link:

 

http://windows.microsoft.com/en-US/windows/language-packs#lptabs=xp

 

Windows XP has Language Interface Pack provides a translated version of the most widely used dialog boxes, menu items, and help content. You might be eligible for the same depending on the XP license.

 

After installing the language pack you need to change the below settings on your computer:

 

a.       In the Windows XP standard Start menu, click Start, and then click Control Panel.

In the Windows XP classic Start menu, click Start, click Settings, and then click Control Panel.

b.      Double-click Regional and Language Options.

c.       Click the Languages tab, and then click Details under "Text Services and Input Languages".

d.      Click Add under "Installed Services", and then click the language you want to add and the keyboard layout you want to use for that language.

e.      To configure the settings for the Language bar, click Language Bar under "Preferences".

 

You may also refer the following article for more information:

 

Regional and Language Options overview

http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/int_pr_custom_workspace_regionalsettings.mspx?mfr=true

 

Hope this information helps. Please revert for any clarification on this or any Windows issue. We will be glad to help you.
