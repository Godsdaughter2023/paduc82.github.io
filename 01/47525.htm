Windows DVD Maker and Movie Maker store files with different file extensions.  This means that if you named the Movie Maker file "file" and the DVD Maker file "file" the true file names would actually be file.MSWMM and file.MSDVD

To view extensions on all files perform the following:
-Click the Start Orb and Control Panel
-In Control Panel click Classic View on the upper Left
-Click on Folder Options and click View
-Scroll down to "Hide Extensions for Known File Types" and uncheck this box

Looking in the folder now where you had saved these files or using the search feature you should see the 2 different files.


What version of Windows are you running?  If Windows Vista Ultimate and the file was somehow deleted you may be in luck.

Vista Ultimate has a feature called previous versions.  To revert the file back to it's previous state you would want to right click on the folder where the document was stored (If Documents then Start, Right Click on Documents) and then go to Properties

-Click on the previous versions tab and you will see dates corresponding to previous versions of the folder you are looking at.
-Select a date and time that matches when the file within that folder was in an operational state and double click the folder to open it

Locate the file in the newly opened folder and drag the file to your desktop (be sure not to drag it to the exact same location where the file was initially as to not overwrite the file).

Hi Peter,

The message that you've encountered when deleting the USERS folder can be due to a permission set on it. To change the permission of the affected folder, follow these steps:

  1. Right click the the parent folder.
  2. Select Properties.
  3. Select the Security tab.
  4. Click Edit to change the permission.
  5. Select the new owner, and then put a check mark in the Full control box.
  6. Click Apply, then OK.
  7. Once you're done, the folder can now be deleted.
