  1. Press Windows key + R to open the Run dialog box.

  4. Click the Power management tab. If the Power Management tab is not available, click the Change Settings button. The same Keyboard Properties window opens with the Power Management tab available.

  5. Select the box next to Allow this device to wake the computer.
  6. Click OK.

  1. Go to Settings > System

    When I press the power button ~ Hibernate.
