    Note: There could be a loss of data while performing a scan using Microsoft safety scanner to eliminate viruses if any.

     

    Method 2: Temporarily disable the third party security program. (If you have installed any)

If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

    e) Check if the issue persists.

    a. On the Windows 8 User Interface Start screen select control panel or right click on the bottom left corner on the desktop to select control panel.
    b. Click Windows Firewall, In the left pane, click Turn Windows Firewall on or off.
    c. Click Turn on Windows Firewall under each network location that you want to help protect, and then click OK.

     

    Windows Defender:

    a. Press the ‘Windows + Q key on the keyboard.
    b. Type “Windows Defender” in the search box and then press enter.
    c. Click Settings and make sure there is a check mark on Turn on real time protection recommend.
