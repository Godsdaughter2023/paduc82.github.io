markfilipak.windows say:

Sarah,

It has nothing to do with Home Edition.

I think I've figured it out... First, some facts, eh?

Fact: Changing registry keyboard settings doesn't take effect until the next start-up.

Fact: Winlogon reads the registry keyboard settings when it starts up and then writes the registry keyboard settings when it shuts down.

Fact: Any script that attempts to change registry keyboard settings before Winlogon has established HKCU is bound to fail (0x80070001).

Fact: Any script that changes registry keyboard settings after Winlogin has established HKCU succeeds, but those setttings are ignored.

Fact: During a session, Windows stores current keyboard settings (e.g., the current NumLock state) somewhere, but it won't tell us where and it won't let us diddle those current settings.

Well, that puts us (users) between a rock and a hard place. If we want to programmatically freeze keyboard settings (NumLock, for example), we can't.

This is what we hardware engineers call a "State Machine Tragedy" -- a deadlock.

Windows handles keyboard input (as a service to applications) but doesn't provide an API to change it, and until the Windows developers give us a way (i.e., an API) to diddle the current keyboard settings via a script, we're screwed

Another way to succeed would be to write the keyboard settings in the registry, then force Winlogon to read them. But I don't know how to do that 2nd part.

sumit (independent advisor) say:
Hi,
Digital license is just a method of activation. License is the real thing :)
If you have a retail license, skip the OEM rant.

Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

However, this is not same for retail licenses i.e. you purchased the license through Microsoft store. That does carry transfer rights. If that is the case, please see the below steps:

Run the activation troubleshooter on the new device using the below steps:
Go to Settings>Update and security>Activation and run the troubleshooter.
Select I recently changed hardware on this device.
Please note that Activation troubleshooter requires you to have digital license linked to Microsoft Account.

Alternatively, if that does not help, you would have to call Microsoft Support.
Contact Microsoft using the phone numbers below or use the chat support link(weekdays only)
https://support.microsoft.com/en-in/help/405170...

https://partner.support.services.microsoft.com/...


Please let us know if you need any assistance.
