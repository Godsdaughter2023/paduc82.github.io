If you don’t see the Get Windows 10  app on your PC, it might be because:

  1. Your device isn’t up-to-date with at least Windows 7 SP1 or Windows 8.1 Update.
  2. Windows Update is turned off or is not set to received updates automatically.

  5. Your Windows License is from a MSDN or TechNet subscription. If it is, you will be able to get Windows 10 through your subscription if it is still active.

Don't know who to give credit to but try this...

Hope this helps someone, but I found my problem with my Homegroup. It was about the "Peer Name Resolution Protocol service". When I checked my services, it wasn't starting. It was giving this error: "The Peer Name Resolution Protocol service terminated with the following error: Unable to access a key." I found out that the key it is talking about is C:\ProgramData\Microsoft\Crypto\RSA\MachineKeys. So I fixed it by going to the MachineKeys folder and adding to the permissions the user "Local Service" (the owner of the Peer service above), with permissions of Modify, Read, Right (everything except full control). Now can start Peer services and Homegroup stuff.
