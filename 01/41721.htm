cherieiam say:

Computer had xp, now it just says media test failure, check cable= exiting broadcom pxe rom

rgharper911 say:

This message usually means that the laptop cannot boot from any of its usual boot devices (like hard drive or CD) and is performing a PxE (network boot) attempt to find an operating system to load.  It is not good news.  It likely means that your hard drive has failed, or that it has been wiped, or that there is a problem with the motherboard in your laptop.  If your laptop manufacturer doesn't have a diagnostic CD or utility you can run on the laptop I'd recommend taking it to a reputable repair shop for diagnosis and repair.

a. user say:

Here is a little more information for you:

Take a look at what the message says...

Maybe something like this with a Broadcom network adpater:

PXE-E61: Media test failure, check cable
PXE-M0F: Exiting Broadcom PXE ROM
Operating System not found

or something like this with an Intel network adapter:

Intel Boot Agent GE V1.2.45
Initiating and Establishing Link
PXE-E61 Media Test Failure
PXE-MOF Exiting Intel Boot Agent

PXE stands for "Preboot eXecution Environment" sometimes known as a network boot.

If your system and environment are capable, you can configure your system to look out on the network and load the operating system from the network.  Such a setup is usually found in a corporate environment or when workstations are deployed without hard drives.  Such a setup is generally not found in a home environment.

What is your system make and model?

What do you think happened since the last time the system booted properly?  Have you made any BIOS adjustments lately?  Experienced any power interruptions?  Did XP crash, etc.

If you wanted your computer to boot off the network instead of the HDD, in the system BIOS, you would choose the PXE boot option (make it first), make the HDD second, etc. or perhaps you could just disable PXE all together if it does not apply to your environment. 

Your system thinks it is supposed to be booting on the network or, it tried to boot off the Hard drive and couldn't, so it is trying to boot off the network next and not finding a place to boot off the network it is asking you to check the network cable.  Are you trying to boot off a network?

For example, on some Dell computers, Pressing F2 gets you into the System Setup program and choosing the integrated network controller, you can choose to enable or disables the system's integrated NICs. The options are ON without PXE, On with PXE, and Off. The option On with PXE allows the system to boot from the network during every boot. Changes take effect after system reboot.

It sounds like either your system BIOS is set to try to use PXE first (and not finding any network to boot on), or it tried the HDD first and can't boot from the HDD for some reason (like your hard drive is failing or has failed) then the BIOS may try the PXE option next.  If it used to work okay and you did not change the BIOS to PXE, it is unlikely to be that, but you should certainly check it.

You find out by pressing whatever key(s) you need to press when your system is starting up to get into the setup BIOS configuration.  If you want us to look, you need to tell us what kind of computer it is (probably a Dell which means you would press F2).

If it is set to boot from the HDD first and not booting (or even trying the HDD), then you should boot into the XP Recovery Console and see what the condition of your HDD is by running:

chkdsk /r

and maybe a few other things depending on how that turns out.

It would seem you would get some other message though if it was trying the HDD, so check the BIOS, cables, etc. and be sure that your HDD is all plugged in nice and tight and that the BIOS is configured to look to boot from the hard drive as the first boot option.
