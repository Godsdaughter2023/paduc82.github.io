What you can do before attempting the upgrade?

You can do some pre-requisite tasks to ensure a smooth migration from Windows 7 to Windows 8. If you are using a name branded computer such as a Dell or HP, go to the manufacturers website for the model computer you are using and download the latest available drivers for that computer. Store them on a disc or USB thumb drive. Important drivers you should try to obtain include Network and Video Drivers. Its possible that they might just have Windows Vista or Windows 7 drivers, those will work with Windows 8.

Other things you can do:

  1. Uninstall any security software before attempting to upgrade.
  2. Disable any encryption software you might have installed.
  3. Disable/uninstall disk utility software such as DVD/CD burning utilities or third party defragment programs such as Perfect Disk.
  4. Make sure your computer is updated (devices and applications).
  5. Disconnect any external devices before installing.
  6. Check your hard disk for any errors:
    - Click Start
    - Type: CMD
    - Right click CMD then click Run as administrator
    At the Command Prompt, type: chkdsk /r /f
    Exit the command prompt.
    When you restart your system, your computer will be scanned for errors and attempts will be made to correct them.
  7. Another thing you can do is disable Start items:
  • Click Start
  • Type: MSCONFIG.
  • Hit Enter on your keyboard
  • On the General tab, click Selective Startup.
  • Under Selective Startup, click to clear the Load Startup Items check box.
  • Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.
  • Click OK.
  • When you are prompted, click Restart.
  • After the computer starts, check whether the problem is resolved.

printing information - Microsoft Community

maruca123 say:

printing information

I want to print out basic information about this new laptop -- serial number, etc. How do I do that??

lemp say:

Printing out most "basic information" is relatively easy.  Including your serial number is a bit more difficult.

Start this way.

Click Start, type msinfo32 into the Search box, and press Enter

You'll get the following.  The "System Summary" view (which is what displays when the utility first opens) will have most of the relevant information, but click on some of the other items to see what's there.

You have two options:

  1. Click File > Export and type a file name and select a save location.  You'll get a *.txt file that includes EVERYTHING in all of the categories shown in System Information.  There's probably more information in this file than you want, but you could always open the file in Notepad and edit out portions that you don't want to keep.

  2. With the System Summary showing, click Edit > Select All (or press CTRL+A) and then Edit > Copy (or press CTRL+C).  Then open Notepad and click Edit > Paste (or press CTRL+V or right-click > Paste) to paste the data into Notepad.  Then click File > Save, give it a filename and location, and click Save.

In order to get the Serial Number, the easiest way would be to just read the label on the machine, where you can also find the computer's "model designation" (or whatever the manufacturer calls it).  Then open the Notepad file where you saved the data from msinfo32 and manually type in both the serial number and the model designation.

If you want to get the serial number programmatically, you can do the following.  In this example, I'll assume that you've already saved the msinfo32 data into a file on your Desktop named Info.txt

  1. Open an Elevated Command Prompt (click Start, type Command Prompt in the Search box, right-click on the link to Command Prompt and select "Run as administrator" then respond as necessary to the UAC prompt 
  2. Type the following command in the black Command Prompt window and press Enter

    wmic bios get serialnumber >> "%userprofile%\Desktop\Info.txt"

Using the two "greater than" characters ( >> ) appends the output from the command to the existing Info.txt file.  You can enter just the command and see the output in the Command Prompt window.

Instead of using msinfo32 -- which has a graphical user interface -- you can use the following command in an Elevated Command Prompt.  It will produce most of the information that's on the "System Summary" screen of msinfo32 plus a few more items and a list of Windows "Hotfixes" (patches) that have been installed.

        systeminformation
