thomas ferguson say: This is usually one of two things:

1) In error or by accident, the printer driver setting to print on black only has been set on or true. To check on this, go to printer properties and check driver settings. Printer drivers are written by the printer makers and they differ in the location of the setting.

2) The printer, itself has a fault; e.g clogged ink delivery system. Many drivers provide a facility to clear clogs. Check the tabs in the driver for you printer. Also, check for some information in the printer manual or the printer maker's web site for information.

It is less common but sometimes even a new ink cartridge is faulty. It has even been known that a user has carefully prepared the new cartrisge for use and then actually put the old back in. But you did write that it worked for a short while.

I would start by removing the cartridge and reseating it. Next, run the printer self-test. (See the manual or web site for instructions). Next run the cleaning routine.

Tom
