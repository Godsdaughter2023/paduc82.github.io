Hidden Attribute Removal Help Needed

Need to turn off the hidden attribute on a large group of files, some of which are in subfolders. I tried to reset it with all files selected from the highest level folder but that did not work. I am not sure what freeware I might try with Win7. I'm no expert so any help would be appreciated.
steve winograd say: You can do that from a command prompt window (Start > All Programs > Accessories > Command Prompt) by specifying a wildcard pattern that matches all of the files.  For example, to remove the hidden attribute from all text files in a folder and its subfolders, enter these commands:

   cd /d path
      attrib -h *.txt /s

where path is the full path to the folder
