Figured it out. Just in case anyone else has this problem, this is what I did:

Had to go buy an Ethernet cable as well as a USB to Ethernet adapter. Adapter came with a disc (my laptop doesn't have a disc drive), so I emailed D-Link (it was their product) and they sent me a link to install the required driver.

Maximum usable random access memory (RAM) on 32-bit Windows 7 is limited to 4 GB.

Physical Memory Limits: Windows 7

 

Since you have a memory card of 1 GB installed on your computer, the maximum usable memory (RAM) from the memory sticks would be limited to 3 GB. In totality, a 32-bit operating system does not support greater than 4 GB random access memory.

mike hall... always say:

Microsoft Windows 32-bit desktop operating systems were limited to 4gb addressable resources in order to preserve stability when installed and run over a great many hardware variations.

The 4gb addressable includes the motherboard and whatever resources are on it. So for example, if your motherboard is worth 750mb of addressable resource, the system will only pick up on and use 3.25gb, regardless of how much RAM is installed over 3gb.

 

If you want to use more, and for the system to be able to allocate more than 2gb to any running application, you will have to install the 64-bit OS variant. 

 

There is no direct upgrade path as 32 and 64-bit are different platforms and you can't upgrade form one platform to another. It is a clean install only, which requires that you back up or plain save all of your important data first.

Hello JacobCarpent,

You may try the basic troubleshooting steps below to fix the issue.

To delete the shared.xml file:

1. On your keyboard, press the Windows key and R at the same time. The Run window opens.

2. Type %appdata%\skype and click OK.

3. Locate and delete the shared.xml file.

4. Open your Skype Name folder and look for config.xml. Delete this file as well.

To Re-name Skype Folder:

1. On your keyboard, press the Windows key and R at the same time. The Run window opens.

2. Type %appdata% and click OK.

3. Locate Skype and Rename it to SkypeOld

If the issue persists, kindly uninstall and reinstall your Skype Application. You may follow the steps on this FAQ.

Hope this helps. Thank you!
