Hello lkmack,

If you have pressed ‘Shift’ and deleted the files, it would not go to the Recycle Bin. However, if there were two copies of files, and if you deleted just one, you should still have the other one.

Also if you are playing the recording with Media Player, if you make an attempt to delete the files, you would have got a prompt if you like to delete it from the library or the computer. If you have selected the other option, it would delete the file from your computer.

I am not sure where you are looking for the recordings. My take on this would be that you should look for the recordings in the default recording folder.

The default recording folder is ‘C:\Users\Public\Recordings’. You can also look for the location in the Media Player settings.

You can also find the location for the recordings by follow the below steps:

a.     Open Windows Media Center.

b.      Click on Settings.

c.      Scroll down to TV and click on it.

d.     Click Recorder.

e.     Now click on Recorder Storage.

Here you can find the location where the Recordings are saved. Browse through this location and see if you can still find your movie.

Also check if the file is hidden. Here is what you need to do to show hidden files and folders.

http://windows.microsoft.com/en-US/windows7/Show-hidden-files
