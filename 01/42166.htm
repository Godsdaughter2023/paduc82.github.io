andre da costa say:
Microsoft is aware that previously-hidden partitions are improperly being assigned drive letters after installing the April 2018 Update. As a result, they now appear in Disk Management and File Explorer. These partitions are typically recovery partitions used by the Windows Recovery Environment or by the Original Equipment Manufacturer. It is not recommended that you attempt to delete or reformat these partitions as this may have adverse effects on your ability to recover Windows. Microsoft is currently working to address the issue in a future Windows Update, and we’ll update this thread when a solution becomes available.

(Update 5/14/18: The solution that will soon be released via Windows Update will only prevent this issue from occurring for devices that install the April 2018 Update after the solution is released. If you've already installed the April 2018 Update and are experiencing this issue, you'll need to follow the steps below if you want to hide these partitions.)

Although you can safely ignore these partitions, if you prefer do away with the low disk space warnings and hide the partitions so they no longer appear in File Explorer, you can do so by manually removing the partition’s assigned drive letter. To do this:

1. Select the Start button, type cmd and then right-click Command Prompt in the search results and select Run as administrator.

2. In the Administrator: Command Prompt window, type diskpart and press Enter.

3. Type list volume and press Enter.

4. Make note of the volume number for the drive letter you want to remove. For example, if the drive letter you want to remove is Drive “X”, note the volume number for Drive X.

5. Type the following command, and press Enter.

select volume <volume number>

When typing this command, in place of <volume number>, substitute the volume number you made note of in Step 4. In the example above, if the volume number for drive X is 5, you would type: select volume 5

6. Type the following command and press Enter.

remove letter=<drive letter>

When typing this command, in place of <drive letter>, substitute the drive letter you want to remove. In this example, you would type: remove letter=X

7. Close the Command Prompt window.

jay974 say: I was having a similar issue myself and I found this post. The easiest way I found was using a command window (start key +R)

In here, navigate to the folder which contains the files you want to rename using the change directory switch (CD). In the example below, I will switch to the C:\Users\Audio folder

C:\>CD C:\Users\Audio
C:\Users\Audio>

To view a list of files in this folder, type dir and hit enter, which will show something like this...

C:\Users\Audio>dir
 Volume in drive C is HardDrive
 Volume Serial Number is DC91-461A

 Directory of C:\Users\Audio

29/11/2009  17:17    <DIR>          .
29/11/2009  17:17    <DIR>          ..
24/10/2009  19:22         9,952,510 Track1.wav
24/10/2009  19:22         9,952,510 Track2.wav
24/10/2009  19:22         9,952,510 Track3.wav
               3 File(s)     29,857,530 bytes
               4 Dir(s)  51,472,916,480 bytes free

To rename the files individually, you can do so using the ren command like...

C:\Users\Audio>ren Track1.wav Track1.mp3
C:\Users\Audio>

This changes Track1 from a wav filetype to an mp3 filetype. Of course, this can be a long process if you have alot of files you need to change. Instead of doing each individually, you can use the wildcard character like this...

C:\Users\Audio>ren *.wav *.mp3
C:\Users\Audio>

The * character just takes the place of the filename (TrackX). This command will affect every file which has a wav extention and changes it to an mp3 extention. Now, when you run the dir command, you will get this...

C:\Users\Audio>dir
 Volume in drive C is HardDrive
 Volume Serial Number is DC91-461A

 Directory of C:\Users\Audio

29/11/2009  17:17    <DIR>          .

24/10/2009  19:22         9,952,510 Track3.mp3
               3 File(s)     29,857,530 bytes
               4 Dir(s)  51,472,916,480 bytes free

Of course, this only allows you to work around a problem that you shouldn't be having in the first place
