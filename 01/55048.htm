frederik long say:

You could do it like so:

  1. Select a screen saver (e.g. Bubbles) to start after x minutes of non-use.
  2. Create a monitoring task to run once every 3 minutes. It will check if bubbles.scr is active. If it is then it will log you off.

There are several problems with this solution:

  • Experience says that few users are able to create the .vbs file that is required, even when given the code.

  • If you happen to have any unsaved files at logoff time then their contents will be lost.

Setting a screen saver password would be much easier and would give you the same level or protection.

You should see a file named uninst.exe, run that file

If the file does not exist:

Either reinstall the program and then uninstall it - reinstalling generally fixes the uninstall routine being missing

Otherwise, just delete the program folder from the C"\Program Files or C:\Program Files (X86) and also from the ProgramData folder on your C drive.

This will probably only leave a few registry keys, but this is rarely an issue . . .
