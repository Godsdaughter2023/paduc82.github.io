I need to change administrator name & password over to my name - Microsoft Community

lmstater say:

I need to change administrator name & password over to my name

My husband died last year and I have entered myself as a family member on his pc, but how do I delete his name as administrator and add me as such ?

hairyfool say:

You have my sympathy, not always the easiest thing to do.

The pedantically correct answer would be to Reset Windows completely so remove all accounts and start off as new. We make this recommendation whenever transferring a computer from one ownership to another. That way you know that absolutely everything from its previous use has been completely removed and have a known starting point.

In this case obviously you are familiar with the computer from when your husband owned it so the need for a clean system is somewhat reduced.

There is however a twofold consideration to allow for. Are you talking about who owns the computer or who's account runs the system. In terms of the former earlier windows versions had two values, Registered Owner and Registered Company and they would be seen on the system Properties page. The only way to change this would have been to modify the registry itself. Those values still exist under HKLM\Software\Microsoft\WindowsN\CurrentVersion so in theory could be referred to verify who owns the computer. However the System Properties page no longer shows these items so may not be an issue

Otherwise it is simply a matter of removing your Microsoft Account from the Family Safety group held on your husbands Microsoft Account by logging in as him. Then you can make your account have administrator level control on the computer itself. Finally back up your husbands data then remove his account including any files. As yours is then the only Administrator you are the de facto owner of the system.

The 1st option is the more complete process to use but probably involves more work.

Entering PPPoE

Netgear DGN2200

Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - password

Netgear SmartWizard

Entering PPPoE

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Login to router by entering http://routerlogin.net  in to the address bar or by entering the default gateway IP address.
  3. The router will then prompt you to enter the user name and password. the default user name and password are: "admin", "password".
  4. From the left hand menu select Basic Settings.
  5. Where it says Does your internet connection require a login select yes.
  6. Enter the user name and password for your pppoe in their respective fields.
  7. Select apply.

 

Netgear Genie

Entering PPPoE

  1. Login to router by entering http://routerlogin.net  in to the address bar or by entering the default gateway IP address.
  2. The router will then prompt you to enter the username and password. The default user name and password are: "admin", "password".
  3. Select the Basic tab from the tabs at the top.
  4. Select Internet from the menu on the left
  5. Where it asks if your internet connection requires a login select yes.
  6. Input in your PPPOE user name and password.
  7. Select Apply.
