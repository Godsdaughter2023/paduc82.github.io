You can make it like new. That's what the "Reset" option does (not "Refresh", which leaves stuff behind).


Reset: Totally wipe the drive and return to factory condition. Nothing remains, it even scrubs the drive so nothing can be recovered.


Refresh: Just erases the \windows and \programs folders and then replaces them with default copies. All user files/registry settings remain.


Restore: The old "go back in time to a checkpoint' system that started coming with Windows XP.

Have to be logged in as Administrator to remove startup programs

lemp say:
1.  Msconfig is a diagnostic tool, not a startup manager (although it can be used in that manner).

2.  A better tool for managing startups is Autoruns.  Download and install it.  Run it and let the scan complete.  Click the "Logon" tab.  UNcheck any item that you want to disable.  When you want the item to run at startup again, just re-run Autoruns and replace the check.  If you are really certain that you never want an item to run again, you can Delete it.  Right-clicking an item in Autoruns and then selecting "Search online" will perform a search using your default search engine that will help identify what the item is.

2.  If you can't even start msconfig, you need to log in using a "computer administrator" account.  However, even if you are logged in as a "computer administrator," attempting to disable certain items may also generate a similar warning.

3.  Click Start > Control Panel > User Accounts (if you are using the "Category view" for Control Panel, click User Accounts again).  Under the heading "Pick an account to change" you'll see all of the normal user accounts on the system.  Does the one that you use on a daily basis say "Computer administrator" or "Limited user"?

4.  If your account is not a "computer administrator," you can log into the built-in account named "Administrator" (I hope that you are NOT using this on a daily basis).  How you get to "Administrator" depends on whether you have XP Home or XP Pro.
  • If you have XP Pro, then while at the "Welcome" screen and NOT logged in to any account press CTRL+ALT+DEL (press all 3 keys simultaneously) startup sequence.  If you see the Windows logo, you waited too long and will have to try again.  As with the "classic" logon window, leave the password for "Administrator" blank.

5.  If you are logged in as a "computer administrator" and are still getting a warning about "access," the problem may be caused by (a) a service installed by some HP printers, (b) some security applications (e.g. Norton), or (c) malware.  Except for malware, you generally can ignore this warning.  If this is your situation, please describe exactly what you are attempting to do and report the complete text of any error message, without paraphrasing.
