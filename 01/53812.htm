andre da costa say:

You cannot do an in place upgrade from Windows Vista Ultimate to Windows 7 Home Premium, that is not a logical upgrade path, it would be a downgrade. Your only option is to backup your personal files and settings and do a custom clean install. Your applications will and drivers will need to be reinstalled.

When making significant changes to your computer such as an upgrade, it is always recommended you backup your system prior to installing a new version of Windows. In your case, you can upgrade to Windows 7 depending on edition and architecture of Windows Vista you have installed. If you are running a 32 bit version of Vista and want to upgrade to a 64 bit version (edition of Windows 7), you will have to a do clean install, this applies vice-versa.

If you are running consumer editions of Windows Vista such as Home Basic or Premium, you can only upgrade to either Windows 7 Home Premium or Ultimate. Also, you cannot do downgrades, which means, if you are running Windows Vista Ultimate now, you cannot downgrade to Windows 7 Home Premium. You must do logical upgrades:

Windows Vista Home Basic > Windows 7 Home Basic, Premium or Ultimate

Windows Vista Home Premium > Windows 7 Home Premium or Ultimate

Windows Vista Business > Windows 7 Professional, Enterprise or Ultimate

Windows Vista Enterprise > Windows 7 Enterprise

Windows Vista Ultimate > Windows 7 Ultimate.

Possible Workaround:

http://stevejenkins.com/blog/2010/02/upgrade-or-downgrade-windows-7-ultimate-or-vista-ultimate-to-windows-7-professional-or-windows-7-home/

What that trick will let you do is downgrade to Windows Vista Home Premium first, then you insert the Windows 7 Home Premium DVD (make sure that you don't restart the computer after making the registry change), then you can do the in place upgrade to Home Premium.

Make sure you backup your registry before making any changes.

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

OEM licenses are to be installed by professional system manufacturers only. Under Microsoft's OEM License Agreement, they are not to be sold to end-users under any circumstance, and are to be preinstalled on a computer using the OEM Preinstallation Kit (OPK) before shipment to the customer, and must include at the very least the manufacturer's support contact information. They are, therefore, designed for installation only on a single computer and are not transferable, even if the original computer is no longer in use. This is not usually an issue for users who purchase new computer systems, because most pre-assembled systems ship with a preinstalled operating system. There are few circumstances where Microsoft will allow the transfer of an OEM license from one non-functioning system to another, but the OEM System Builder License Agreement (SBLA), as well as the OEM End User License Agreement (EULA) do not contain any allowance for this, so it is entirely up to Microsoft's discretion, depending on the situation

http://www.microsoft.com/oem/en/licensing/sblicensing/pages/licensing_for_hobbyists.aspx

There are two types of OEM vendors. Royalty OEM vendors can distribute Windows products by using System-Locked Preinstallation (SLP) Product Keys so that end users can bypass product activations. All other OEM vendors distribute Windows products by using the standard Product Keys that require the user to activate the Windows products.

Check Bluetooth Settings
1. Go to Settings (Go to Start Menu > select the Settings icon above restart > select Devices). Are your devices there? Do they show any driver errors?

3. Tap on Surface Keyboard then Remove Device. Confirm you want to remove at the next prompt. Then do this for the Mouse and any other Bluetooth devices.

a. On your Studio Keyboard press & hold the Bluetooth icon. 
b. Find the Bluetooth button by flipping your Keyboard upside down.
c. To the right of the battery holder is the Bluetooth button/icon.
d. Press & hold this button for about 30 seconds.
e. On the Surface you should see Surface Keyboard appear and after tapping on screen, you will have an option to Pair.
f. Select Pair and let it install the drivers.  Sometimes this can take a few minutes, even after you notice it has paired & working there may still be a driver download indicator on the taskbar.
g. Repeat for other Bluetooth Devices

Run the Bluetooth Troubleshooter (If the above has not fixed your issue)
a. Go to Cortana (the Search Box on your taskbar, sometimes just a circle next to your Start Menu icon that will say “Ask Me Anything,” Type here to search,” or “search the web & Windows”) and type in the word Troubleshoot

c. Tap or select Bluetooth
d. Tap or select Run the Troubleshooter. If a prompt pops up to enter your Admin password, please allow.
e. Once complete, select “Done.” 

When completed it will provide any settings it has adjusted or any problems still not solved. Please make a note of that for Surface Support.

f. Restart the Surface through Windows, sign in, and test.

Anytime a restart is needed it is best to restart through Windows instead of simply holding down the power button.  This allows Windows to install any updates that were pending and can resolve any issues that have popped up after an update and we know it is not just going into Sleep Mode.

Uninstall & Reinstall the Bluetooth Driver
1. Go to Settings > Devices > Bluetooth
a. Tap on each Bluetooth connected device and select Remove

2. Open Device Manager
a. Right click on the Windows Start Menu icon and select Device Manager, or type Device Manager into Cortana and select it.
b. Expand Bluetooth
c. Find Marvell AVASTAR Bluetooth Radio Adapter
d. Right click on Marvell
e. Select Uninstall Device.
For the pop up warning: if a check box appears and says “delete driver software” do not check the box. Leave it empty.  Just select “Uninstall” at the bottom of the pop up.
g. Restart Surface through start menu. Sometimes after selecting Uninstall it will automatically restart so do not be alarmed but usually we have to initiate the restart.
h. Go back to Settings > Devices > Add a Bluetooth Device and repair your Surface Keyboard & Mouse and test.
