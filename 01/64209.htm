In anticipation to Windows 8 being available to the masses, I would like to know what are the possibilities of installing Windows 8 64bit when Windows 7 32bit is already installed.

Would Windows 7 32bit and all 32bit drivers have to be reinstalled with 64bit versions?

If you are dual booting Windows 7 32 bit and Windows 8 64 bit, you will need to install 64 bit native drivers. Windows 8 and 7 64 bit do not support 32 bit drivers.

Meaning a complete reinstallation of all existing/installed applications and settings/data be retained or partitioning the hard drive and start from new again is the only solution?

Yes, you will need to reinstall applications too. You will be able to migrate your personal files and settings using the Windows Easy Transfer utility.

Or would installing another hard drive be more appropriate?

You can install it on another partition or hard disk. If you are installing on another partition or hard disk, you will need a full version of Windows 8.

I read about performance tips concerning SSDs and disabling hibernation to release the space occupied by hiberfil.sys.

However, in my case it is less about getting more space, but about avoiding writes to the SSD to increase its lifetime.

I use hibernation just sometimes, so I wondered when exactly the contents of  hiberfil.sys are really written to disk:

1. Only when I hibernate the PC

2. or is it written all the time in the background?

If 1. is true, then I could keep hibernate feature enabled. However, if 2. is true, I would disable it.

Most of the time I use Windows 10, but it would also be good to know, how it works in Windows 7.

zigzag3143 (ms -mvp) say:
Since the contents of memory change dynamically HF only writes to disk when going into hibernation.  Otherwise on waking the contents would be different.  Most SSD have a mean time between failure of ~20,000 hours and a fixed amount of writes both of which are in most cases longer than the actual life of the SSD.  The one caveat is that SSD'S are assumed to have a duty cycle of 10%.  If you game excessively you are probably writing to the HD more often than 10%.  Otherwise your fine
