Hi,

If you're able to move messages from your inbox but haven't moved your storage folders yet, please follow the steps below.

    You can first try to do the steps in the following solution article:

        How to import messages from Outlook Express to Windows Live Mail


    If you were not able to copy your emails to Live Mail's Storage folder, you can try this workaround. In the following workaround, assuming you've copied your accounts from OE to Live Mail, you will have to do two things:

        • Backup your folders in Outlook Express email messages.
        • Import the backup file of your account and email folders to Windows Live Mail.

    To back up your folders in Outlook Express:

  1. Open Outlook Express.
  2. Please go to Tools > Options...
In the Maintenance tab, please click on Store Folder... button.
  • The store location will be displayed, Copy the store location.
  • Click on Start > Run to open the Run tool of Windows XP then Paste the copied store location of your emails in Outlook Express.
  • The folder where Outlook Express stores your emails will be displayed. All email folders will be displayed as *.dbx file
  • Copy the files to a temporary folder in a flash drive or to any portable storage.

    •     To import your email folders to Windows Live Mail:

    1. Open Windows Live Mail.
    2. Please go to File > Import messages or press Alt + F, then I.
    In Windows Live Import, please click on Next. Outlook Express 6 is selected by default.
