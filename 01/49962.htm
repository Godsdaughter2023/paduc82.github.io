New features such as Cortana, Windows Hello, and PDF viewing in the new Edge browser rely on Windows Media files that aren't included in N editions. And some websites and software, such as Windows Store apps or Microsoft Office, use Windows Media-related files that aren't included in N editions.

The Media Feature Pack for N versions of Windows 10 will install Media Player and related technologies on a computer running Windows 10 N editions.

https://www.microsoft.com/en-us/software-download/mediafeaturepack

Media Feature Pack for Windows 10 N (October 2017):

Once the Media Feature pack is installed, follow these steps to make Windows Media Player as default:

  1. Open the Settings screen.

  2. Click System.

  3. Click Default apps. Scroll down to see Music Player section.

  4. Click the download arrow to see the list of options and you should see Windows Media Player in the list.

  5. Select Windows Media Player to make it default.

Try the steps and let us know how it goes.
