MAC Filtering - Microsoft Community

xb1r athena say:

MAC Filtering

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

 

 

MAC Filtering

 

  1. Open a web browser on a computer.
  2. Type the address “192.168.1.1” into the web address bar.
  3. When prompted for a username and password type in “admin” for both the username and the password.
  4. On the left hand side of the web page select the “Firewall” option.
  5. Select the “MAC Control” subcategory below.
  6. On the main page select the “MAC Address Control” drop down box and select the “Allow Internet Access for the following MAC addresses”
  7. Enter in the MAC address for the device you would like to add.
  8. Click on the “Add” button.
  9. Continue to repeat steps 6-8 untill you have added all the device from your home network.
  10. Click on the “Ok” button at the bottom of the page.
