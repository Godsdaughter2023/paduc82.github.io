rarson say:

Jessen's suggestions are not only unnecessary, but pointless as well. You don't need to go into the BIOS or change the boot order, or use any recovery media, you simply need to click the Notifications icon in the system tray, go to All Settings and click on Update & Security. On the left side, there's an option for Recovery, and clicking on that will give you the ability to choose an Advanced startup. This will take you to advanced options including repairing your computer using Startup Repair.

However, as I said before, this suggestion is pointless because I can tell you right now that this is not an issue that Startup Repair will solve. Startup Repair, as its name suggests, repairs startup problems. Microsoft has changed the name of it to "Automatic Repair," but it's still doing the same function. You would think a Microsoft "support engineer" might actually know that, or offer some kind of helpful advice, but they rarely do.

What you (or anyone else with this issue) might try doing is Resetting the PC, which reinstalls the operating system fresh, or rolling back to a previous build if that is an option. Note that while Windows may save your personal files, your programs will be gone and have to be reinstalled. You can avoid this by obtaining a copy of the Windows 10 iso, using a tool like Rufus to install it to a USB thumb drive, and then run the setup from within Windows. This will give you the option of reinstalling the operating system while keeping all of your programs and files intact.

But before you do that, I recommend opening an elevated command prompt (right-click the Start button and choose "Command Prompt (Admin)" and running chkdsk /f. Once that is complete, you'll want to open another elevated command prompt and run the following command: dism /online /cleanup-image /restorehealth. (Don't include the period at the end of my sentence, obviously). Once that completes, the last step is to run sfc /scannow. I doubt that this will solve your problem, but it's a much better suggestion than running Startup Repair.
