 This info is in your router.  Usually you bring up Internet Explorer and in the address bar you type in something like 192.168.xxx.yyy where the xxx and yyy part are in your router's documentation.  Then you have to sign in to the router.  The default sign in is also documented in the router's documentation, although they usually recommend that you change it to prevent hackers from getting into it.

    That button on your computer is an ON button, not off.  The only way you should shut down Windows is using the start globe (bottom left) and the shut down button on your screen.

    Each time you shut down using the ON button, you risk corrupting your system.  It is likely that you did just that.  You now need to try some repairs.  If those repairs do not work, you will need to re-install your Windows system.

    Windows corruption repairs:

    1. Start globe (bottom left)
    2. All programs
    3. Accessories
    4. RIGHT click on Command prompt and choose Run as administrator
    5. Type chkdsk c: /r into the black box, then Y for yes
    6. Restart

    Process takes from 1 to 3 hours

    After that run SFC to fix corrupted Windows files (not data)

    1. Start globe (bottom left)
    2. All programs
    3. Accessories
    4. RIGHT click on Command prompt and choose Run as administrator
    5. Type sfc /scannow into the black box

    Check health of your hard drive hardware by running test software for your particular make of drive

    1. Start globe (bottom left)
    2. Type device into the text box that pops up
    3. Click on Device Manager
    4. Click the triangle next to disks
    5. Read the model number of the drive installed in your computer
    6. Reply with that model number and I will reply with a link to the appropriate software for your drive.
