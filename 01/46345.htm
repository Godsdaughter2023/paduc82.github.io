patrickmondor say:

Jaunice,


Windows 8.1 is a departure from previous OS's, so doing things differently takes some learning.  Finding things in Windows 8.1 can be quite easy once you know where to start.  First, while on the start screen...simply start typing.  For instance type in Word.  The charm bar will fly out on the right side and you will see search results based on what you typed.  In this case you should see the Word icon.  You can launch from there....or you can place the icon on the start screen.  In Windows 8 icons populated the start screen.  In Windows 8 all installs placed icons on the start screen.  Many users didn't like the "mess" created with all the icons on the screen.  So Windows 8.1 by default will NOT install icons on the screen.  You must click on the small arrow pointing down on the left side to bring up the list of apps.  You should see the Office suite icon somewhere in the list.  Simply right click on the icon you want to see on the start screen and select "Pin to Start".  With the new windows you don't have the "old style" start menu, where Office normally would be installed under the Microsoft Office program group.  The start screen is the "new" start menu.  Finally, the download most likely is hiding out under you downloads directory.  Go to the start screen and type File Explorer..select the file explorer icon on the Charms bar....and then you will see the old style desktop launch with file explorer now showing you drives, libraries...everything.  Look for Downloads under the Favorites group near the top of File Explorer.  Hope the above is helpful.  Good luck.
