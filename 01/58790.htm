This solved the problem: To help Windows Speech Recognition sort out any problems it might be having figuring out which microphone to accept input from, you might want to try disabling any other recording devices except for the one you want to use. Try this: 1. Open the Advanced Speech Options/Speech Properties control panel window. 2. Click the Audio Input button, and then click the Recording tab in the dialog box that appears. 3. If there are multiple recording devices listed in the dialog box, right-click any of the ones you don't want to use, and choose Disable from the menu that appears. 4. If there is only one recording device listed, set it as the default (so that it has a green checkmark on it). 5. Click OK to confirm your settings, and try your microphone with WSR to see if it works. copied from http://www.knowbrainer.com/PubForum/...c&topicId=816056

a. Insert the Windows 7 installation disc into the disc drive, and then start the computer.
