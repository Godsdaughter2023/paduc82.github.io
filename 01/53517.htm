    SOLVED: No WiFi after latest windows update - Microsoft Community

 olu odeniyi  say:


            SOLVED: No WiFi after latest windows update

                    
I fixed this problem and thought I'd share with others in case you have the same issue.




After the latest Windows 10 Update  ver: 10.0.15063, wifi would not connect to any of the available trusted networks. This was not an issue prior to the update.




Initially, I checked the normal areas, such as the router (restarted it), forgot the networks and re authenticated, etc still no resolution.




I then checked the network connections and realised the wifi adapter was connected to a virtual bridge - I had a couple of VMs setup and a virtual Bridge. To resolve, :-




1. Right click the bridge and select properties


2. Uncheck the Wifi box and close.




The wifi adapter was no longer connected to the bridge (which disappeared) and wifi was working as normal.




Somehow, the windows 10 update had mixed adapter connections over the update.




Hope this helps someone - wouldn't want anyone swapping out h/w to fix this!




Enjoy!
