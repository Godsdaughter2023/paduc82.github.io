a. user say:

Hi Shayaan,

Thank you for contacting Microsoft community. I appreciate your efforts in trying to resolve the issue.

I understand that you are not able to boot your computer to the desktop and it falls continuously into restart loop. This issue could occur if there are any incompatible drivers installed or due to any third party software conflict.

  1. Did you make any changes on your computer prior to the issue?
  2. Did you try to Reset or Refresh your computer?
  3. Are you able to boot to recovery console?
  4. Did you try to boot in safe mode?

Try to boot with "Disable Driver Signature Enforcement"

This method is used to forcefully disable the signature of drivers. Try to launch the Windows 8 ‘recovery mode’, follow the steps below:

  1. Past the BIOS splash screen, hold down the Shift key and repeatedly hit the F8 key. Doing so should get you to the recovery mode, but it might take a few tries before you achieve that.
  2. Once the Recovery screen is displayed, click the See advanced repair options button. In the next screen, hit the Troubleshoot button.
  3. In the troubleshoot menu, click Advanced options. In the advance options, hit the Windows Startup Settings.
  4. The next screen will ask you to restart your PC in various modes. Choose Disable Driver Signature Enforcement and then hit Restart.

If you are able to boot the computer with driver signature disabled, place the computer in a Clean boot state to isolate the issue. A clean boot is a troubleshooting technique that allows you to get the computer up and running so that you can determine which elements of the normal boot process are causing problems.

How to perform a clean boot in Windows

http://support.microsoft.com/kb/929135

Note: Once you are done with clean boot, please follow the section How to reset the computer to start normally after clean boot troubleshooting the article to put your computer back to normal startup.

You may also try installing the latest version of graphic card drivers from the manufacturer’s website and check if it helps.

I also suggest you to create a Windows 8 installation disc to repair your computer.

Upgrade Windows with only a product key
