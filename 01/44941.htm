Hello,

Thank you for posting the query on Microsoft Community. I am glad to assist you on this.

Please be informed that, we can Rollback to previous version of Windows only before 30 days of Windows 10 installation. After 30 days, the Rollback option will be grade out and we will not be able to go back to previous Windows using this option.

Note:

Please keep in mind the following actions will disable Go Back:

  • Refresh
  • Reset
  • Disk Cleanup (if it touched Windows.old or $windows.~bt)
  • Manually deleting files within Windows.old or $windows.~bt
  • Running on battery (must be plugged in)  

After 30 days we can only perform a Custom installation or a Clean installation in order to go back to previous version of Windows. I would suggest you to use previous Windows installation disk and perform a Custom or Clean installation in your PC to go back. 

Note:

I highly recommend backing up all of your important files and folders onto an external storage device before attempting any type of OS repair or upgrade.

Hi Digren,

You cannot split an unallocated partition on your computer. First you need to create a partition out of it and then try to split it.
