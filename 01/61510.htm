Login Information



  • Gateway: 192.168.0.1
  • Username: admin
  • Password: admin





Restore Factory Defaults



Through Firmware



  1. Connect the computer to one of the LAN ports on the router via an Ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. When prompted, enter "admin" for the user name and "admin" for the password.
  5. Select "System Tools" from the menu on the left.
  6. Select "Restore to Factory Default Settings" from the sub menu.
  7. Select Restore.



Through Hardware　



  1. Connect the computer to one of the LAN ports on the router via an Ethernet cable.
  2. Locate the "Reset" button on the device.
  3. Using a toothpick/pen/paper clip, press and hold the button for 30 seconds.

dgtexan say:

It allows you to "hide" what buttons you press.  When not "hidden", the button you press shows up above your finger so you could verify that you are typing the letters you want to type.  When the letters are "hidden", there is no visual feedback of the letters you type.  This way, if someone is looking over your shoulder, it's harder for them to watch what you are typing as your password.

If you performed a Reset and are encountering this error, you should perform a clean install to restore your device.   To do this: 1.

 

From a different PC, visit the Software download page and run the tool to create Windows 10 installation media.  When choosing whether to download the 32-bit or 64-bit version of the tool, you want to choose the version that matches the PC that you are using to create the media – not the version of the PC that is experiencing the bluescreen. 2.

Once setup starts, choose a Custom install. 4.

 

If you are prompted for a product key, you can choose to skip that step.  If you were activated previously after first upgrading to Windows 10, you will be activated again once you connect online.
