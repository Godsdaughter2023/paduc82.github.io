The OEM, System Builder 3 Pack comes with 3 separate Product Keys, one for each of the 3 computers. System Builder versions of Windows 7 can only be installed on computers that are to be resold by the System Builder.

 

 

For Windows 7, the only license that is transferable is a Retail license version that is bought separately. This license can be transferred to another computer, IF it is removed from the previous computer.

 

If Windows is pre-installed when you purchase a computer, it cannot be transferred to another computer. If you want to sell or give away the version of Windows that came pre-installed, it must include the complete computer, including the Windows installation and product key.
