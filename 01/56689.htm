davem121 say:

Hi David, it is not easy to remove a file association in Windows 10,

Open Windows File Explorer

On the view Ribbon (tab) check the box marked 'File Name Extensions'

Close File Explorer

Right click your Desktop and choose New - Text File

Change the file extension on that new file from .txt to .exe

Confirm the change

Right click a .VDI file, choose open with - Choose another App

Check the box marked 'Always use this app to open .VDI files' - IMPORTANT!

Scroll to the bottom and click 'More Apps'

Again scroll to the bottom and choose 'Find another app on this PC'

Navigate to the dummy file exe on your Desktop and click OK

The .VDI file will not open

Delete the dummy exe file on your Desktop and restart your PC

The file association will then be broken!

Windows 10 fails to start up first time - Microsoft Community

gavinbevan say:

Windows 10 fails to start up first time

Hi,

 Since I've upgraded to Windows 10 it fails to start/boot up first time, I've always got to press the power button a second time for it to come on, any ideas why?

david doherty say:

Hi, try this it worked for me.

Turn off the fast restart option as follows;

1. Right Click on the start Button and choose 'Power Options' (second top menu choice),

2. Select 'Choose what the power button does' in the left hand menu options,

3. Select 'Change settings that are currently unavailable' because the option you want to adjust is currently greyed out further down this page in the Shutdown settings.

4. All the check boxes in the Shutdown settings should now be available to edit,

5. Uncheck the first box 'Turn on fast Startup',

6 Save your changes, and

7. Shutdown.

Next Start should be first time. Hope it works for you, hasn't failed for me yet after about a week. :)

Good Luck
