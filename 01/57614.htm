lemp say:

The Trusted Installer is likely installing updates that have been downloaded from Windows Update.  High CPU usage from this process ought to calm down in a "reasonable" amount of time after Windows starts.

Do you leave your computer on at night or do you turn it off when you're not using it? 

If you have a desktop/tower (i.e., not a laptop), click the Start Orb and then click Control Panel.  In the search box at the upper right corner of the Control Panel window, type "power" (without quotes) and press Enter.  Then click the link to "Change when computer sleeps" (actually, you get to exactly the same place if you click "Change when to turn off display" or "Edit power plan").

Set the times for the display to turn off and for the computer to turn off to whatever suits you.  If you make any changes to the values already shown, don't forget to click the "Save changes" button.  Then close Control Panel.

When you are finished working on your computer, just walk away.  The computer will go into a low-power mode after the time you set.  When you want to use the computer again, just tap a key on the keyboard.  There's no need to press the main power button.

If you have left Windows Update at its default settings, the computer will "wake up" at 3:00 a.m., check for updates, and download and install "important" updates (and then go back to sleep).

Allowing updates to occur in the middle of the night (when you presumably are not using the computer) should alleviate the problem with Trusted Installer.

Unless you have a local network and really want to share your media files among several network-connected devices, you can safely disable the Windows Media Player Network Sharing Service as follows.  Disabling this service will alleviate any problems you are having with it taking CPU resources.

  • Click the Start Orb, type "services.msc" (without quotes) into the Search box and press Enter
  • Scroll down until you find Windows Media Player Network Sharing Service (the entries should be in alphabetical order by service name; if not, click "Name" in the header row to sort; click "Name" a second time to sort in the reverse order).  Be sure you have the correct entry:  there are 3 different services relating to "Windows Media Player"


  • Double-click Windows Media Player Network Sharing Service to open its "Properties"
  • On the "General" tab of the Properties dialog, first click the "Stop" button under "Service status" and then select "Disabled" from the drop-down list for "Startup type"


  • Click the OK button and close the Services applet.

You should understand that I and most of the experts here at the forum are not Microsoft people.  We are just people with a lot of experience who volunteer our advice.

My first concern is the hard drive.  Hard drives in a laptop are one of the first failure points.  If your hard drive is beginning to fail, it could be the cause of a lot of those symptoms.  Plus, I would hate to spend a lot of effort on fixing a machine with a hard drive that is about to fail.  There is good software to test them, but you need to have windows running to use it.

Do you have an external drive housing that you could temporarily use to put the hard drive into to connect to another computer to run the test software?

Accessing Toshiba factory restore partition is done by pressing the 0 (zero) key repeatedly about once per second at start up.  If that does not work, there may not be a partition to restore.

There is software around that will give you the installed product key, but you need to have windows running to use it.

If you have no restore points, it could well be that the virus removed them.

Print Spooler service, spools print service and handles interactions with the printer. If you turn off this service you won’t be able to print or see your printer.

  1. Press Windows + R keys from the keyboard to open Run command.
  2. Type services.msc in the search box and press Enter.
  3. Search for the Remote Procedure Call locator service
  4. Make sure the Start-up type is Manual.

Steps to set the services to Automatic

  1. Right click on the service
  2. If the service is stopped then Start the service
  3. Select Properties
  4. And in the Start-up type, select Manual.
