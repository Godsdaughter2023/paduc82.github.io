You are logging into Windows 8.1 with your "Microsoft Account", instead of a regular "Local Account". It's not your fault. When you use the new, special SkyDrive client that is built-in to Windows 8.1 (which you cannot uninstall or disable) it will only let you connect to your SkyDrive if  you're using an "Microsoft Account" to log-in to Windows, and it has to be the same Microsoft Account where your SkyDrive files are located. As far as I know you cannot create a SkyDrive-specific password to that is different than your Microsoft Account password. This predicament exists only in Windows 8.1, you wouldn't have this issue on Windows 8.0. You cannot even download and install the Windows 8.0 SkyDrive client that's available on Microsoft website (that would have allowed you to specify your MS-account when you run it, and let you keep using a Local account to log-in to Windows)

If you want to keep using the (official) SkyDrive client in Windows 8.1, and you want to keep using a Sophisticated Password to control access to your SkyDrive, then you must use the same sophisticated password to log-in to Windows. (But that is not as necessary as you think, because SkyDrive does not encrypt your files with your password. If securing your SkyDrive files is a concern, then consider using client-side encryption for could storage - search the web for such solutions)

If you want to switch to using a Local Account for Windows and still get some sort of SkyDrive functionality, there are some solution:

1) Map a Network Drive to your SkyDrive account (video instructions

, text instructions). This uses SMB or DAV technologies and may be slow, sluggish or generally unreliable. Any hiccup on your connection and you need to upload the files all over again. This solution only allows Upload, Download, Create, Delete.

2) Use a program called GoodSync which uses SkyDrive API, same same API that the Windows 8.0 SkyDrive client uses, which adds the ability to Sync and Update, in addition to Upload/Download. This solution does not provide you with a Drive or a Library that you can use in Explorer or Save-To in applications.

3) One more solution which simulates the regular SkyDrive client, but I fear it includes spyware so I won't link to it.

4) If you use SkyDrive only for Office documents - use Office 2013, which has a SkyDrive implementation similar to that of Windows 8.0

P.S.
