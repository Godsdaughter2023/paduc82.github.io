    a. user say:

    Needless to say, cutting power during the update process can cause all sorts of problems - many not even related to the update itself.  This is a common ramification.

    Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  I don't know if this will resolve the problem, but it might.  It's worth a try (and it may help if there are other problems that you haven't yet detected).  Whether or not it works, you will need to re-install the update (and this time don't turn off the power - LOL).

    If you still have the validation problem after system restore, then go to Start / Computer / System Properties and check at the bottom for the Activation section.  Does your system still show it is activated?  If not, we will need to re-activate it.  To manually activate, go to start, search box and type in slui 4 and enter.  Enter your country and press next.  You'll be on a screen with a phone number and a bunch of codes at the bottom.  Call the number and wade through the automated prompts until you reach a human.  Have your product code handy as it will be requested.  Explain the situation but it's the same PC.  They should give you a code to enter on that screen to activate Vista.

    If that doesn't resolve the "not genuine" message, please post your problem in the Windows Genuine Advantage Forum at: http://social.microsoft.com/Forums/en-US/genuinevista/threads where experts on this sort of problem will be more than happy to assist you with your problem.  They will probably ask you to run a program and download the results so they can review them to find the problem - just follow their instructions.

    I hope this helps.

    Good luck!
