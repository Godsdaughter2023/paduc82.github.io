    [david} say:

    Since you are able to access the command line, try the following steps to resolve the recovery mode loop (I found this from https://support.microsoft.com/en-us/kb/927392):

    bcdedit /export C:\BCD_Backup
    c:
    cd boot
    attrib bcd -s -h -r
    ren c:\boot\bcd bcd.old
    bootrec /RebuildBcd

    After doing this and restarting, Windows 8.1 will load, and at the login screen, you will see "Configuring Windows Update" and "Cleaning up".

    I hope this helps!
