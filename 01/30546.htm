michael van g say:

Workaround for Windows 8 Pro x64

 

Go to your Control panel -> Installed Programs.

Then uninstall both versions of the Microsoft Visual C++ 2010 Redistributable Package.

That means both the (x86) and (x64) version of the package.

 

After the uninstall of the redistributable packages start the AutoRoute 2011 installer.

It will install a new (*old*) redistributable package for x86.

The installation should complete now without errors.

 

When the install of AutoRoute 2011 is done, download and install the x64 redistributable package from here (Microsoft site): Microsoft Visual C++ 2010 Redistributable Package (x64)

 

AutoRoute 2011 should now run.

 

After this you can safely update the x86 Redistributable Package trough Windows update (will be a 2 stage update, first the new package, and then the servicepack 1 update).

The updates should not prevent AutoRoute 2011 from running.

 

Hope this solves the issue for you.

It's the only way I managed to get it working on my end.

 

Regards, Mike.

The "thumbnail preview" of which you speak is called an Aero Live Preview.  A search for "Live Preview" may turn up more help.

If Aero is turned off and you are using a basic theme, you will see titles instead of the Live Preview graphic.

However, there is a way to get both the window title AND a Live Preview.  Try this:

1.  Right-click the Start orb.

2.  Choose Properties.

3.  Click the Taskbar tab at the top.

4.  In the "Taskbar buttons:" selector, choose Combine when taskbar is full.
