Hi StephenfromChch

Let's take a look at your backup settings by doing the following:
1) Click the Start button
2) Click the Settings icon
3) Select the Update & security option
4) Click the Backup menu option on the left

Do you see an option called 'Automatically backup my files'?
If yes:
a) Ensure it's in the 'On' position and not 'Off'. If it's off, you will need to click the slider to turn it on.
b) With the backup option enabled (turned on), click the 'More options' button to see your backup options.
c) Does it say 'Backup is disabled' or 'Backup enabled'?


If you do not see 'Automatically backup my files', and instead you see an 'Add a drive' button, then it would appear that File History either wasn't setup properly or that specific configuration was lost.
To configure File History follow these steps:
1) Click the Start button
2) Click the Settings icon
3) Select the Update & security option
4) Click the Backup menu option on the left
5) Click 'Add a drive' and select either a local disk or a network location by clicking 'Show all network locations;
6) As soon as you choose a location, you will see an 'Automatically back up my files' option appear.
7) Click the 'More options' link
8) On the 'Backup options' screen click the 'Back up now' button

a. user say:

To backup you'll need third-party backup software. File History was developed to backup individual files; it's not well-suited to backing up all your files.

There are two types of backup: 

1. an image backup backs up your entire hard drive; this is ideal for backup up your Windows installation;

2. a file and folder backup backs up individual files and folders; this is a great way to backup your personal files (documents, pictures, etc.)

You can find applications for both types of a backup on the internet. A great way to find them is to search for:   windows 10 backup software 2018

Your search results will include professional reviews of popular backup software. An example of image backup software is Macrium Reflect; an example of file-and-folder backup is Backup4All. A file compression program like 7-zip may be used in place of a file-and-folder backup, but it won't be as convenient.
