Are you experiencing “Error Code 0xC0000005″ while trying to install Windows 7 on your hard drive? Although this error can also be associated with corrupt or missing graphic card drivers, it is usually due some temporary hardware issues.

1) First, Check Windows 7 System Requirements.

2) You should run a memory diagnostic test to make sure all your ram is stable

3) If that still does not solve error code 0xC0000005 during install, you most likely have a bad hard drive that needs to be replaced.

a. user say:

Hi, FredVDW,

I am an Independent Advisor

Right Click on Start > Select PowerShell (Admin)
Type the following command to list the languages currently installed on your device and press Enter:

Get-WinUserLanguageList

Make a note of the LanguageTag you want to remove.
Type the following commands (one at a time) to remove the language that you don't need and press Enter:

$LangList = Get-WinUserLanguageList
$MarkedLang = $LangList | where LanguageTag -eq "LANGUAGETAG"
$LangList.Remove($MarkedLang)
Set-WinUserLanguageList $LangList -Force

In the second command, make sure to change LANGUAGETAG with the code of the language you want to remove. For example, es-US, if your primary language is not the English US, and that's the pack you want to delete.
