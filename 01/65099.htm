suat.cini say:

Hi.

You can back up all the drivers in your working OS to a folder and then integrate all your working drivers to any Windows 10 build.

Just give the commands below in a command prompt with admin rights or copy the commands below ( between dashes ) to a text file ( ex: drivers.txt ) and rename the text file to drivers.cmd. You must change the .txt extension to .cmd. In the following commands, G: is the drive where you wish to copy the drivers. You must change it to a suitable drive letter by your PC. If you use a USB flash drive and your PC assigns drive letter E: to it, then change G: to E:. Now run the drivers.cmd with admin rights. All your drivers will be copied or backed up to folder G:\Drivers.

--------------------------------------------------------

@echo on

@setlocal

@md G:\Drivers

@set pathtodrivers=G:\Drivers

@set path=%pathtodrivers%;%path%

@dism /online /export-driver /destination:%pathtodrivers%

@endlocal

@exit

----------------------------------------------------------

After you copy all the working drivers, keep them in a safe place.

The drivers folder will look like this:

When you wish to clean-install a new Windows 10 Build, you can integrate these drivers into the new Windows 10 build. Again copy the following lines ( between dashes ) to a text file ( ex: integrate.txt ) and rename the integrate.txt to integrate.cmd. You must run this command file with admin rights.

Please note: You must make a new folder in G: drive G:\Win10 and copy the contents of the new Windows 10 build into folder Win10. If you are using a USB flash disk with drive letter E:, then change G: to E: ( USB flash disk size must be 8 GB or larger, depending on size of drivers plus about 4 GB for Windows 10. ). You must also create another folder for mounting of Windows 10 wim file ( install.wim ). This folder must preferably be on your HDD/SSD. In the commands below, it is called G:\Offline. Change it to, say, C:\Offline

Please note: If you have Windows 10 install media as Windows.iso, you can directly mount Windows.iso and copy the contents of mounted iso into G:\Win10 folder.

Mounted iso will look like this:

-----------------------------------------------------------------------

@echo on

@setlocal

@md G:\Offline

@set pathtodrivers=G:\Drivers

@set pathtowimfile=G:\Win10\sources

@set imagemountdir=G:\offline

@dism /mount-image /imagefile:%pathtowimfile%\install.wim /index:1 /mountdir:%imagemountdir%

@dism /image:%imagemountdir% /Add-Driver /Driver:%pathtodrivers% /Recurse

@dism /unmount-image /mountdir:%imagemountdir% /Commit

@endlocal

@exit

--------------------------------------------------------------

After the drivers are integrated into install.wim file, you must create a bootable USB flash disk, using Windows.iso. This USB flash disk must be a different flash disk. After the bootable USB flash disk is created, you can delete the contents of this USB flash disk because it does not contain your drivers and copy the contents of G:\Win10 folder on to the USB flash disk. G:\Win10 folder contains Windows 10 with your drivers integrated. This operation does not alter the bootable property of your USB flash disk.

Hope this helps.

Please ask if something above is not clear enough.

Please also don't forget to change drive letters to suit your PC drive structure.

suat.cini say:

Further to my post above, the Windows 10 installation media you download from Microsoft contains several editions with different index numbers such as the ones below:

Therefore, before integrating the drivers, you have to find the index number of your edition of Windows 10 and change the index number in your integrate.cmd file. For ex: If your edition is Windows 10 Pro, then your index number is 4.

To find the index number, you must create an index.cmd script and use it to find the index number. Again you have to change the drive from G: to the drive letter where your Win10 folder resides.

In some instances Windows installation files may contain install.esd instead of install.wim. Then you have to convert install.esd to install.wim.

Copy the lines between dashed lines and paste them into index.txt file. Change the extension to cmd, e.g. index.cmd. Run this file in a command prompt with admin rights.

-----------------------------------------------------------

setlocal

set pathtowimfile=G:\Win10\sources

set path=%pathtowimfile%;%path% 
set userinp=
echo ================================================
echo      Press 1 to get image info from WiMFiLE
echo      Press 2 to get image info from ESDFiLE
echo      Press 3 to abort all operations

echo ================================================

set /p userinp= ^> Enter your option and press "Enter": 

if "%userinp%"=="" goto :QUIT

set userinp=%userinp:~0,1%

if %userinp%==3 goto :QUIT

if %userinp%==2 goto :ESDFiLE

if %userinp%==1 goto :WiMFiLE


:WiMFiLE
dism /get-imageinfo /imagefile:%pathtowimfile%\install.wim
goto :QUIT
:ESDFiLE
dism /get-imageinfo /imagefile:%pathtowimfile%\install.esd

:QUIT

endlocal

pause

exit
