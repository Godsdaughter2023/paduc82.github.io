To troubleshoot this issue and to help you further, I need some more information:

Did you make any changes on your computer prior to this issue?

 

Please follow the methods below and check the issue.

Note: Before trying the methods below please check in the device manager if the Network Adapter is working properly or not.


a. On the desktop, press Windows Key + R and type devmgmt.msc in the Run Dialog box and press Enter.
b. In the Device Manager window, expand Network Adapter.
c. Right-click on the Adapter and select Properties.
d. In the General tab, under Device Status, check if the device is working properly.
e. You may also check if there is any error message or yellow exclamation mark.

 

Method 1: Run the in-built Internet Connection Troubleshooter and Network Adapter Troubleshooter.

Step 1: Run the in-built Internet Connection Troubleshooter.

This troubleshooter is an automated Windows 7 inbuilt tool which finds and fixes the common issues with the Internet Connections on the computer. Perform these steps to run this tool and check if that helps.

  1. Press Windows key + R key.
  2. Type Control and click OK.
  3. In the Control Panel, go to Network and Internet then click on Network and Sharing Center.
  4. Click on Troubleshoot Problems then on Internet Connections.
  5. Open the troubleshooter, click on Next and check the solution.

Step 2: Run the Network Adapter Troubleshooter.

This troubleshooter is an automated Windows 7 inbuilt tool which finds and fixes the common issues with the Network Adapter Settings on the computer. It also checks if the network adapter drivers are up to date or not. It may also provide you some clue about the hardware failures happening to the Network Adapter.

Perform these steps to run this tool and check if that helps.

Follow the article below to troubleshoot network Adapter issues.

http://windows.microsoft.com/en-us/windows/fix-network-adapter-problems#1TC=windows-7

Once after running the troubleshooters, check troubleshooting reports and let’s know the details. You may also try out the suggestions given by the troubleshooter and check if that helps.

You may also try uninstalling and reinstalling the latest Network Card Drivers from the computer/network card manufacturer’s website and check the status of the issue.

 

Try method 2 if method 1 doesn't fix the issue.

Method 2: Uninstall and Reinstall the Network Card Drivers.

  1. Press Windows key + R key type devmgmt.msc and click on ok.
  2.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  3. Right-click Network adapters, and then click on Uninstall your network adapter.
  4. On the un-installation window, if you have an option: “Delete the driver software for this device” you may select that and then remove the corrupted drivers from the computer.
  5. Follow the on-screen instructions to complete it and the restart the computer if prompted.
  6. After the restart, please go ahead with the next steps to install the latest drivers and check if that resolves the issue.

 

Step 2: Check the Windows updates for any latest driver updates.

Check in Windows Updates if any driver updates are available. If so, install the same and check if it helps.

Refer: http://windows.microsoft.com/en-US/windows7/Install-Windows-updates

If the latest network adapter driver are not available via Windows Updates, then download and install it from the computer manufacturer’s website.

Step 3: Update Driver Software from the Manufacturer’s website:

Go to the Computer or Network Card manufacturer’s website and search for the latest Windows 7 drivers available for the Network Card model and then install it by following the instructions given in the website.

http://us.acer.com/ac/en/US/content/drivers

If method 2 doesn’t help, try method 3.

Method 3: Flush your computer's DNS cache.


a) Click on Start button, type command prompt in search programs option.
b) Right-click on Command Prompt and choose Run as Administrator.
c) At the command prompt, type the following commands. Press Enter after each command.

ipconfig /flushdns

ipconfig /registerdns

ipconfig /release

ipconfig /renew

Note: If still the issue persists, login to the computer in normal mode and then check if there are events created under Event Viewer. Follow the steps to collect the logs under Event Viewer:

http://windows.microsoft.com/en-us/windows/open-event-viewer#1TC=windows-7

Most of your event will be Information. You may see Yellow Warnings or Red Errors. If you see any red error, you can double click on it to bring it up and copy the contents to a document. You can post events in the next reply.
