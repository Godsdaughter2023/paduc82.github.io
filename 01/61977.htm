Upgrading a Dell Inspiron 1545 from Windows XP to Windows 7 - part 1

I have a Dell Inspiron 1545 [supplied with Windows 7 pre-installed in January 2010] and it works okay.  I have also added Windows 10 in a dual-boot arrangement and that works okay as well.

Because I expect my 1545 to last beyond the Windows 7 14th January 2020 end-of-extended-support date, I am going to take advantage of the free upgrade to Windows 10 by upgrading my Windows 7 installation to Windows 10 before 29th July 2016.  I have added several notes about Windows 10 below in case you are considering this - given the cost & effort of upgrading at all then it would seem worthwhile getting the whole chore out of the way now.

Buying either Windows 7 or Windows 10 is an act of faith.  You cannot know if the computer will last long enough for it to be worth the cost.

10                   Using System images in Windows 7 & 10

Both Windows 7 & Windows 10 include tools for making System images.  In both cases, the opening dialog of Windows backup contains a direct link to making a System image so you can make a System image without having to bother with Windows backup itself.  In both cases, Windows backup can be set up to include making a System image if that’s what you prefer.  Whilst you need to do something to make backups of your own files, there are lots of ways to do this and Windows backup might not be your preferred solution.  In all cases, making System images or file backups on a drive within your computer is only slightly more reliable than a chocolate teapot - external drives are the reliable choice.

I have the WD Edition of Acronis TI on my Inspiron 1545 and Seagate DiscWizard’s version of Acronis TI on my other computer. 

I make a Windows-based System image for each computer every other week.  I make an Acronis TI-based System image for each computer every other week. 

If I have a problem with either computer that cannot be rectified with straightforward troubleshooting, I restore the latest System image from before the problem’s appearance.  So I might waste an hour not fixing a problem and then I let the computer spend 25 minutes restoring a System image so the problem is bypassed.  Even if the problem had been caused by a Windows or application update not installing correctly, I find that I can then repeat the update and it is all fine [in other words, I find that most problems are caused by unknown errors during updates, because update installation procedures are less than perfect, but the faults do not reoccur when the update is repeated].  Making and later restoring a System image can also allow me to experiment safely - when I worked through choosing all my Windows settings in Control panel & elsewhere and all my applications’ options, I had a System image ready to restore in case I made a pig’s ear of the whole thing.

I keep at least two Windows images and at least two Acronis images so the space needed does add up but this habit ensures that I have a month’s worth of images to fall back on.  I also keep important System images for longer so I need even more space.  I still have the System image I made straight after I reinstalled Windows onto a new [WD] HDD at the end of 2011.  I still have the System image I made straight after I fitted an SSD in place of an HDD in mid-2015.  My external HDDs need to be big enough to hold all these images as well as the latest file backups of my own files, installation files etc; that’s why both my 500GB external HDDs are full and why I would not recommend anything smaller than 1TB now. 

Windows System images are just as easy to make as Acronis TI-based System images but Acronis TI has a better reputation for being able to restore System images onto different hardware.  For example, when I decided to get an SSD, only Acronis TI WD Edition could restore my System image onto the new drive.  The Windows System image turned out to be useless in this situation so I would have had to reinstall everything from scratch if that was all I had.  I only continue to alternate between the two types of System image because I feel I am hedging my bets that way.

My use of the free utilities Seagate DiscWizard and Acronis TI WD Edition mean that I can only replace my existing HDDs with Seagate or WD ones unless I decide to pay to get the full Acronis TI version.  The only flexibility the free ones have is that they work however the qualifying make of HDD is connected [so it could be in a USB enclosure, for example]. 

Some descriptions of these free TI versions state that the qualifying make of HDD is only needed to make System images but is not needed to restore them.  This is false, they will not restore without being able to find the qualifying make of HDD connected.  I tested this a year ago.

11                   Additional Windows 7 & 10 utilities11.1            HDD Smart disk testers

You can get disk utilities that run “Smart” tests on HDDs.  There are lots of them about.  You can search this forum for better explanations of Smart tests but, in short, they test HDD behaviour in great detail and are able to identify developing problems before you start to get symptoms developing.  That gives you time to make backups, make system images & buy a replacement HDD before your computer starts to fail.

11.2            Measuring HDD space usage

Windows XP only had a few instances of system files or folders reporting misleading sizes.  In both Windows 7 & Windows 10, this has increased substantially.  For example, the folder C:\Windows\WinSxS might be reported as being 15GB.  It is not.  Many of its contents are a special type of hard-wired shortcut to other system locations understood only by the OS but not by Windows explorer [or its Windows 10-equivalent File explorer].  So the folder is smaller than it is reported [and some files incorrectly appear to be unnecessarily duplicated].

Less misleading reports of file & folder sizes can be obtained using TreeSize Freeor equivalents.  I have never seen any proof that TreeSize reports correctly in all cases but it is definitely better then Windows explorer / File explorer.  I have found this utility to be useful when trying to understand why the OS drive has less free space than it had a month ago - it enabled me to track down, for example, a setting in my phone syncing software that was saving cached copies of every version of every file that it had ever synced to or from the phone [because I had ticked the relevant setting during initial testing then forgot all about it].

11.5            Windows update in Windows 7 & Windows 10

Windows updates give rise to a security vulnerability because hackers review the latest updates to see which holes they have plugged and then try to develop exploits targeting those who have not yet run the updates concerned.

Windows 7 updates are easily controlled.  The user can choose to let them all happen automatically or to check for them then download & install them only when desired.  Individual updates can be rejected but I have never seen any analysis of the effect of rejecting an update on subsequent updates affecting the same Windows components.

Windows 10 wants to update itself using its own rules; it cannot be constrained to download updates only after user permission has been granted.  Many people dislike this intensely.  Individual updates can be rejected [albeit not as conveniently as with Windows 7] but I have never seen any analysis of the effect of rejecting an update on subsequent updates affecting the same Windows components.

It is possible to tame Windows 10 updates by setting a particular property on the network connection but users then have to run updates manually when they get the chance.  Windows 10 updates cannot be avoided entirely as new updates are designed on the assumption that all previous updates have been installed. 

I have disabled the ability to search for, download & install Windows 10 updates on my day-to-day limited data mobile internet connection.  Every month or more frequently, I connect to a local free WiFi and get all my updates done.

11.6            ReadyBoost

Unless you are using an SSD or have stacks of RAM, you can improve some aspects of performance by setting up an SD card or USB stick to assist the system’s efforts in managing page files.  I can see the difference on my older computer when video editing, making System images and [sometimes, I think] when moving large files around.  I had an SD card slot sitting there unused on my older computer so I thought it worth a go.

There can be a benefit to fitting an SD card or USB stick with up to 2 ½ times the size of the system RAM [anything greater is wasted] but the benefit can never be as great as that achievable by fitting more RAM.

Vista has been around since Vista.  See Understand ReadyBoost and whether it will speed up your system - TechNetfor a discussion of it.  The ReadyBoost device must have an access time of 1 ms or less. The device must be capable of 2.5 MB/s read speeds for 4 KB random reads spread uniformly across the entire device and 1.75 MB/s write speeds for 512 KB random writes spread uniformly across the device.  ReadyBoost cannot be greater than 4 GB on a FAT32 file system or greater than 32 GB on an NTFS file system.  [I just bought a cheap SD card and tried it to see if it worked; it did.]
