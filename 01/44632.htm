rick rogers say: Hi,

What it means depends on where it's coming from. If it's from the speakers, then it's coming from some installed software on the system, possibily running the in the background and could indicate a warning or alert (antivirus?). Resolving it would involve using Task Manager to shut down (kill) processes until the sound stops.

If it sounds like it is coming direct from the tower or vents (laptop), then it's a motherboard alarm telling you that there is either a faulty component or overheating condition exists. The latter is usually resolved by clearing vents of dust/debris and ensuring all fans are in good working order and are not blocked. A clogged fan can emit this type of squeal as well.

Hi
Thanks for contacting Microsoft Community.

Safely remove hardware is not needed when you stop all the software services that is using the modem. Moreover, safely remove is mainly for storage devices like pendrives and external hard drives, but that too is not needed as Windows optimizes each removable device by disabling write-cache in most devices. If it is enabled by user, safely remove hardware option must be used. But note that it doesn't mean you can simply remove the device during a file transfer. You can find the notification light in pendrive blinking during data transfer and only it is done, the light is idle (no blinking), at which state, you can remove the device instantly, no need for safely remove hardware.

When any removable device, like pendrive, is connected, you can got start -> type devmgmt.msc -> press enter. Expand disk drives, double click at the pendrive/removable device. In the properties dialog, go to Policies tab. You can find two options: Quick removal (default) and Better performance. If you select Better Performance, you should use safely remove hardware option. But for Quick removal, you need not use safely remove hardware option.
