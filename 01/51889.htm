¡firedog say:

... what is the limit of BCC og CC in a email?

It seems that the support staff don't know the answer.

With most mail clients, there is no limit at all. Any limit you encounter is usually set by the Email Service Provider (ESP). Some have very restrictive practices - Outlook.com, for example, may set the limit at 10 for new accounts, but it can increase to many more with unabusive use over a period of days, weeks or months. Others have a blanket rule like 25 or 50. Check with your ESP.

That said, there may well be a built-in constraint in the Mail app, but unless the support staff can tell us what it is, we're reduced to testing ourselves - and I don't have 50 or 100 contacts who wouldn't mind receiving a series of test messages from me just to see whether they arrived or not.

greg carmack - mvp 2010-2020 say:

What the labels you posted tell us is that Windows 10 is Booted, but using the System boot files from Windows 7.  When you install another OS with the old one plugged in then it merely updates the boot files on the old OS to create the Dual Boot, a reason why it's good to unplug other OS drives during install and then boot afterwards via the BIOS so that they remain independent. 

To move the Bootmgr to Windows 10 C, download and install EasyBCD (click Download - no Name or Email required). 

1) Choose BCD Backup/Repair Tab.

2) Select Change boot drive

3) Click button to Perform Action

In the popup confirm the New Boot Drive says C, click OK.

Reboot PC, change the boot drive in BIOS to Windows 10, in Disk Mgmt confirm C is now marked System Active Boot.   You can now unplug the Windows 7 drive. 

If this fails a method that also works is to unplug all other drives except Windows 10 which is marked Active, boot into Recovery drive or  Windows 10 Installation Media to run Startup Repair  up to 3 separate times with reboots.  It will discover there are no boot files, write them to C, make it bootable. 

Let us know how it goes.  For further help please Post a screenshot of Disk Mgmt

I've never seen any ill effects at all from using EasyBCD.  To the contrary it easily patches back up a Multi-Boot if it loses one OS by just removing the old listing on Edit OS Menu and adding a new one for it.  It also can refresh the boot files on its BCD Backup/Repair tab, even move the boot files to a different partition. 

You'll also then have the OS's independently bootable via the BIOS if you want to keep or add back later the WIndows 7 drive.  This avoids problems that sometimes arise on Dual Boots.

Just follow the illustrated steps I gave you and it should work fine.  Remember to install Easy to WIndows 10 to do this. 

But if you have any sluggishness at all then it's always a good thing to delete all partitions down to Unalllocated Space with only the target drive plugged in while doing a Clean Install Windows 10 Upgrade. No one who has followed the steps in that link has had a single complaint.
