lemp say:

Windows 7 Home Premium is one of the editions of Windows offered by Microsoft.  You can check to see what you have by doing either of the following:

  • Click Start (the Microsoft "globe"), right-click "Computer" and select "Properties."
  • Press the "Windows Logo" key and the "Pause/Break" key simultaneously

You should see a screen that looks like the following:

  • What do you see under "Windows Edition"?  Does it mention a "Service Pack"?
  • What do you see as the "System type"?
  • For your "Product ID," what are the 3 characters in the second grouping (corresponding to "068" in the image below)?

"Drivers" are small programs that act as the interface between your computer's "operating system" -- Windows 7 -- and the various hardware components in your computer.  For example, there is a driver for:

  • your video card (also called the "display adapter"), which creates the output to your display device
  • your sound card, which creates the output to your speakers or headphones
  • your mouse
  • your keyboard
  • etc.

Windows 7, like all versions of Microsoft Windows, incorporates a collection of drivers for many -- but not all -- hardware devices.  When Windows is installed, it detects the various hardware devices and looks to see if it already has a driver for each device.  If it does, it installs the driver.  Windows also checks for new hardware devices when the computer starts.

If Windows does not have a "built-in" driver for a device, you need to obtain that driver elsewhere.  For "brand name" computers (e.g., HP, Lenovo, Dell, etc.) the first place to look for drivers is the computer manufacturer's support site.  The next place to look for drivers is the web site of the manufacturer of the specific hardware device.  For example, if you have an Nvidia graphics adapter, you would go to the Nvidia site (http://www.nvidia.com/Download/index.aspx?lang=en-us).

The reason to go to the computer manufacturer site first is that the big computer manufacturers will often customize hardware that they incorporate in their products.  Thus, for example, an Nvidia graphics adapter in a Dell computer might require a Dell-specific driver rather than the "generic" driver from Nvidia.

Sometimes, Windows will have a basic driver that gets a hardware device functioning but doesn't enable all of the special features that you get when you install the specific driver from the computer manufacturer or the hardware device manufacturer.

In all cases, whether the driver comes from Microsoft, the computer manufacturer, or the hardware device manufacturer, drivers are free.  You should never pay for drivers.

Except for a few hardware devices -- notably printers -- you can find information about device drivers in the Windows Device Manager.  Drivers are not "located in" Device Manager, but you can get information about your hardware devices and their drivers using this tool

The following is what you could see if you opened Device Manager and clicked on the "Help" menu:

Device Manager provides you with a graphical view of the hardware that is installed on your computer. All devices communicate with Windows through a piece of software called a device driver. You can use Device Manager to install and update the drivers for your hardware devices, modify hardware settings for those devices, and troubleshoot problems. 

Uses for Device Manager

You can use Device Manager to:

  • Determine whether the hardware on your computer is working properly.
  • Change hardware configuration settings.
  • Identify the device drivers that are loaded for each device, and obtain information about each device driver.
  • Change advanced settings and properties for devices. Install updated device drivers.
  • Enable, disable, and uninstall devices.
  • Roll back to the previous version of a driver.
  • View the devices based on their type, by their connection to the computer, or by the resources they use.
  • Show or hide hidden devices that are not critical to view, but might be necessary for advanced troubleshooting.

You will typically use Device Manager to check the status of your hardware and update device drivers on your computer. Advanced users who have a thorough understanding of computer hardware might also use Device Manager's diagnostic features to resolve device conflicts and change resource settings.

Ordinarily, you will not need to use Device Manager to change resource settings because resources are allocated automatically by the system during hardware setup.

You can use Device Manager to manage devices only on a local computer. On a remote computer, Device Manager will work only in read-only mode, allowing you to view, but not change the hardware configuration of that computer.

frederik long say:

I read somewhere that it is impossible to change the user folder name without re-installing the OS.

-> This is correct. The reason is simple: The registry contains countless references to the name of user profile. Some of them were created by Microsoft software, some by third-party software. If you changed the name of the profile folder (which is fairly easy) then these references would be invalid and your account would be crippled.

If I felt strongly about the name of my profile folder then I would do this:

  1. Create a spare admin account.
  2. Reboot and log into this account.
  3. Delete the problem account (but keep the user data!).
  4. Create a new local account with the desired name.

You will get a profile folder with the correct name. If desired then you can copy the personal files from the old to the new profile folder.
