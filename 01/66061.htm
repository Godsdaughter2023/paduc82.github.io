Under the Auditing Entry window, ensure to tick the checkbox of Full control under Basic permissions. Then click OK. Under Auditing entry, Everyone is now part of the settings.

2. Make sure that you've enabled file and printer sharing on your network.

Refer:

Share a printer
http://windows.microsoft.com/en-US/windows7/Share-a-printer

3. Turn off all your firewalls and try connecting. It if doesn’t appear offline, it indicates that the firewall needs to be reconfigured to allow file and printer sharing.
If you’re running third party firewall, please refer the first article above on how to open the ports required for the sharing to be  possible.
4. Go to Start > Devices and Printers. Right-Click on the printer and uncheck use printer offline".
5. Again right click on the printer and set it to default.
6. Power cycle both the printer and the computer. The best way to power cycle a printer is as follows:
           a) Turn the computer off.
           b) If the printer has an external Ethernet adapter, unplug its AC adapter from the power source.
           c) Save your work and shut down your computer.
           d) Wait a few seconds, plug the Ethernet adapter back in first, and then turn the printer back on. 
           e) Turn the computer back on
7. Check if the spooler service is started.

Refer: Why can't I print?
http://windows.microsoft.com/en-US/windows7/Why-cant-I-print

8. Try pinging to it from other computers. The PING command is used to test the connection and latency between two network connections. See if there’s any response.
Go to start>run and type cmd.
Type: ping ipaddress
9. On the desktop which is not printing try installing only the network printer driver.While installing driver, you should get the option to choose network driver and not the entire installation.

Update  a driver for hardware that isn't working properly
