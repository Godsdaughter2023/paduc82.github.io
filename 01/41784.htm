Hi Jey,

1. Use this Wiki to check System Integrity =- Steps 4, 7, & 11

2. Start Windows in Safe Mode with Networking
Start > Type Msconfig > Press Enter
Select Boot Tab
Select Safe Boot
Select Networking
Click Apply, Ok
Reboot and you'll load into Safe Mode with Networking.

Retry the System Restore Point

To Load Normally go back to Msconfig > Uncheck Safe Boot.
Click Apply, Ok
Reboot and you'll boot normally into Windows

3. If the above fails, perform the following scans.
https://answers.microsoft.com/en-us/windows/for...?

Retry Restore and verify if it still gives the error

4. If you still cannot get System Restore to run properly, try adjusting the hard disk space allocation. It might have run out without informing you. Recommended to allocate 4 GB.
