Hi Bob,

I have the same issue. It shouldn't be a big problem for MS to supply a nice "tile-customization-tool". OblyTile didn't work till now, either.

My Workaround is simple but takes some preperation.

I use Visual Studio Community (VS) to create a small EXE file to start the link. Any other Compiler also works.

Just paste the following Code to an C# project:

===

class Program
    {
        static void Main(string[] args)
        {
            System.Diagnostics.Process.Start("http://www.heise.de");
        } 
    }

===

In VS is a possibility to place an individual .ico-grapic. The filename will be shown in the starmenu (in this exampel heise.exe)

after building the EXE you are able to add this to your starmenu. (right-click)

And voila: it starts the link (sometimes it takes a second)

As said: just a Workaround.
