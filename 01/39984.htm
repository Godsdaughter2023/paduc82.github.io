martmcd say:

its the one at the top of the list, if you expand the status column you will see it says healthy (EFI System Partition)

more importantly it is not showing a drive letter.

DO NOT ATTEMPT TO DELETE THE PARTION FROM THE HDD

First check for updates, in case there is now an update to fix it, if a restart is required allow it to do it.

if not fixed.

open an elevated command prompt (admin)

type diskpart and press enter

type list volume and press enter

if the partition does not have a drive letter

type exit and press enter, close the command prompt and try to ignore the drive letter for a week, keep checking for updates then come back here and let us know.

if it does show a drive letter and you don't want to wait to see it will be resolved with an update

make a note of the Volume no (if in any doubt, paste the results of list volume here)

type select volume # (where # is the number you made a note of earlier) and press enter

type remove letter=F (where F is the drive letter shown in Volume list) and press enter.

type exit and press enter, close command prompt.

check file explorer if it still shows a drive letter, restart the pc.
