Hi,

Thank you for posting your query in Microsoft Community.

I understand your concern regarding the issue that you are facing.

I suggest you to follow the steps and check if it helps

Method 1: Add input language

  1. Press Windows Button and start typing Control panel
  2. Click on Clock, language and Region
  3. Click on Language and click on Keyboard Layout
  4. And you can add the input Language for keyboard and check if it helps  

 

Method 2: Troubleshoot

Windows 10 has a built-in troubleshooter to check and fix issues with hardware and devices. I would suggest you to run this troubleshooter to check if the issue is with your touch Screen. Refer these steps:

  1. Press Windows key +X, select Control panel
  2. Change the view by option on the top right to Large icons
  3. Click on troubleshooting and click on the view all option on the left panel
  4. Run the Hardware and devices troubleshooter

 

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.
