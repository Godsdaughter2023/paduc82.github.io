how to convert wlmp file - Microsoft Community

patricialily say:

how to convert wlmp file

I had made a keepsake video a few years ago and thought I was doing it right, but now that I am on windows 10 my video won't play ..How can I convert this video to a different file so I can open it ?

considerate_guy say:

The .wlmp file is the project file. The project file is actually just a relatively small text file that contains the following:

  • A list of all your source media files (file path and name)
  • Data relating to the order of each clip, where you trimmed the clip to start and stop, what effects you applied, etc.
  • Data relating to your overall project such as the aspect ratio and audio balance settings.

The project is essentially useless on its own. It contains no actual media since it is just referencing their location on the computer.

Before you switched computers to your Windows 10 PC, did you have a backup of the source files (photos, video, audio)? If you did, then you might have a chance if you put those in the same location and also reloaded the same version of Movie Maker to get the project file to reopen. But if you don't have the original media files, there is no way to magically resurrect them from nothing but the file name.

In the future, if you are working on a project, be sure to always finish by rendering your work to a movie file. The movie file can be saved and used on another PC. The project file isn't as transportable, for all the reasons I stated above.
