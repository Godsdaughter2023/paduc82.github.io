martmcd say:

How come the update does this?

the major feature updates (1511 and 1607) are effectively a reinstall/upgrade. a number of things will have been reset to the Microsoft default

Will MS fix this? Do they even know about it?

the good news is that they do know about it and have recognised that users don't like uninstalled apps being put back and all the settings changed, and have said, in an Insiders announcement 14/9/2016, that future builds will not do that.

Is it all intentional ? 

not really, but if any particular feature has many major updates (or a complete rewrite) it is logical to think that the settings for that feature will be reset to the defaults.

so although Microsoft have said it wont happen in future builds what I think they mean is 'it wont happen as much'

  2. Select "Properties"
  3. Select the Compatibility tab
  4. Check "Run this program in compatibility mode for" and select "Windows XP (Service Pack 3)"
  5. At the bottom of the screen check "Run this program as an administrator"
  6. Click OK

It should finish OK.  You'll get security warnings when you start the game, but it will play OK (don't forget to apply the latest SOM patch file).
