Just rename Pictures.pd6 to Pictures.OLD. This may cause
you to lose some tags and metadata that were not written back
to the photo files.

The next time Photo Gallery is launched...a new Pictures.pd6
file will be created and the Gallery will be as it was the first time
you opened it.

The path to the file (in Windows 7) is :

Windows\Users\%Username%\AppData\Local\Microsoft\
Windows Live Photo Gallery\Pictures.pd6

This is a Hidden file so you may need to change the following
setting before you can find it:

Try the following steps to 'Show Hidden Files' in Windows 7:

Go to.....Start button / Control Panel / Appearance
and Personalization / Folder Options / View tab
Advanced settings / Show hidden files, folders, and
drives / OK

If you are unhappy with the result...delete the new Pictures.pd6
file and rename Pictures.OLD back to Pictures.pd6

Good luck...

Blocking the upgrade prompt and blocking the upgrade itself have the same steps for you not see it in the future. Follow these steps to block the upgrade by using Computer Configuration:

  1. Click Computer Configuration.
  2. Click Policies.
  3. Click Administrative Templates.
  4. Click Windows Components.
  5. Click Windows Update.
  6. Double-click Turn off the upgrade to the latest version of Windows through Windows Update.
  7. Click Enable.

If you are unable to block it using Computer Configuration, we might need to modify the registry. Before you modify it, make sure to back up the registry first in case problem occurs. To block the upgrade to Windows 10 through Windows Update, run “regedit” to open the Registry Editor, then specify the following registry value:

Subkey: HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate
DWORD value: DisableOSUpgrade = 1
