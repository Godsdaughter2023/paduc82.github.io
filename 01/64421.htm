ADS is actually used in the latest Solaris (SMS) versions as well as in all versions of Windows who use the NTFS file system. ryen kia zhi tang say:
Hi,

I believe you are trying to upload or download a file using your FTP server? Can you remote user apply a PUT command to upload to your FTP Server or GET command to download from your FTP Server?

If you are referring to the FTP folder permission, use the CACLS command.

1.) Open the command prompt as administrator
2.) Type the command to allow users to READ,WRITE,MODIFY and FULL CONTROL the folder call "ftpfolder" for an example.
         cacls c:\ftpfolder /g username:R
         cacls c:\ftpfolder /g username:W
         cacls c:\ftpfolder /g username:C
         cacls c:\ftpfolder /g username:F

Read the appendix for more detail about CACLS

Please clarify to be able to assist you further. Hope it helps.



Appendix
http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/cacls.mspx?mfr=true

I believe the access is denied is possible if,
a.) the right is already assigned
b.) you are not using administrator rights

Anyway, you can avoid the (Y/N) prompt by having echo on the batch file
   echo y| cacls <var>filename</var> /g <var>username</var> :<var>permission

Refer to Microsoft KB on How to Use CACLS.EXE in a Batch File .

Hi Thomas. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

It could also be the sound controller hardware is failing. So you can absolve the drivers by trying older ones too, from the PC or device maker's Support Downloads web page, and also from Device Manager > Sound Controller > Driver tab > Update Driver > Browse > Let Me Pick to try all previously installed drivers working backwards.

Update the sound driver from the PC maker's Support Downloads webpage for your exact model, if it is newer than the one presently installed in Device Manager reached by right clicking the Start button. While there compare to make sure you also have the latest BIOS, Chipset, Display, Network and all other drivers.

Right click the volume icon in System Tray at right end of task bar, to Troubleshoot Problems.

Run the Playing Sound troubleshooter at Start button > Settings>Update & Security>Troubleshoot.

Try rolling back the sound driver in Device Manager reached by right clicking the Start button. Choose Sound controller device, then Driver tab, then Roll back if available.

While there make sure you also have the latest Chipset, Display, Network, USB and all other drivers, comparing with the version presently installed in Device Manager.

If no newer driver then uninstall the one presently installed in Device Manager, reached by right clicking the Start Button. Restart PC to reinstall.

Try other and older drivers for your model Sound controller, too. For example, try the next oldest driver in the model's online Support Downloads list. To try all previous drivers installed on your system, in Device Manager choose Sound controller's Driver tab, then Update Driver, then Browse, then Let Me Pick.
