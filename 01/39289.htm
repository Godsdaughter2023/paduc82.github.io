If it mentions OEM <method>, you’ve OEM license installed.

df-458 say:

Same problem affected my wife's computer, although I didn't realize there was a problem until she attempted to save a Word document and received the message that the file could not be saved - access denied.

As I checked other Office programs they all had the same problem including Outlook.

I checked online where there were a number of ideas presented, but none worked.  After carefully going through a number of steps trying to resolve the "access denied" message, I finally resolved the issue once and for all.

Note that out of frustration, I restored the root drive (C) where the OS is loaded and restored Win7.  I did this so I could hopefully copy the files needed as I will be loading onto a new system.

Evidently the upgrade process takes ownership of the folders and files - not sure why, especially when in my case all of the files such as documents, outlook files, spreadsheets, etc., are kept on a different drive.  What the upgrade process fails to do is return the ownership to the original owner.  As such, you no longer have permission to access the files.

Here's what to do.

If you don't have the administrator account enabled, enable it (be sure to give it a strong password if you leave the account enable).

Login as the administrator (this is what I did, it might just work if your normal user login has admin rights, but I did it using the admin account).  Note, again the steps listed here were done under Win7, but Win10 is similar and you'll be able to navigate through it.

  • Right click on the highest level folder
  • Select properties tab
  • Select security tab
  • Select advanced tab
  • Select owner tab
  • Select edit owner (note that you'll see a long string name that doesn't represent any normal user on your system which is why you don't have "permission")
  • Select the appropriate user from the list
  • Select the check box to replace owner on sub-containers and objects
  • Take ownership
  • You'll get a message box that you'll click through, and select OK on the other windows to get you back.

That should resolve the issue.
