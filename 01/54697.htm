considerate_guy say:

The most common causes of not being able to save your movie are:

1) Incompatible file types. Even types that are considered "supported" can cause problems, so when in doubt, I always work exclusively with .wma for audio and .wmv for video.  The fix for this is a file converter like Freemake Video Converter, Prism Video Converter or Microsoft Expression Encoder 4.

2) Too complex a project.  Sometimes if a project has lots of pictures, lots of clips, lots of transitions, lots of effects it can cause problems during saving.  In this case make several copies of the project and pare down each one so that it has only part of the original project.  You can always save a movie from each project part and then join the resulting movies back together in one final project.
