Method 1: Run an SFC Scan

The “sfc /scannow” command scans all protected system files and replaces incorrect versions with correct Microsoft versions.

a) Click on “Start” button and type “cmd” in the search box.

b) Right click on “Command Prompt” icon from the search results and then select “Run as administrator”

Use the System File Checker tool to repair missing or corrupted system files

b) Right click on “Command Prompt” icon from the search results and then select “Run as administrator”

c) Run the following commands on the command prompt window one by one and press “ENTER” after each command.

net stop w32time

w32tm /unregister

w32tm /register

net start w32time

w32tm /resync
