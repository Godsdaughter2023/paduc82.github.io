balaji kundalam say:
Windows 7 has a feature that records errors (and warnings) and finds solutions to them from internet and inform the user about the solution. This is done through Action Center and Reliability Monitor is a part of the tool that provided the detailed list of issues, information and warnings, recorded with date and displayed by date. When you open any error report present here, you have two options: 1. Manually click "Search for solution" and 2. Download/perform the solution steps available. If no solution is available, the message "No new solution found. When solutions are available, they will appear in Action Center".

Information includes starting a service, installation/updation of a hardware/software, uninstallation of a hardware/software, etc.

Application failures includes application crashes and error reported by applications.

Windows failures includes any system error like BSOD.

Miscellaneous failures includes issues like if the PC was not shutdown properly.
