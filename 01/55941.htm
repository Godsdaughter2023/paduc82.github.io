Did you leave your user name out of the path to that folder?  There shouldn't be an AppData folder directly under C:\Users (it should be C:\Users\<yourusername>\AppData)

You didn't see what's taking up the space in the folder because some of the files and folders there have "Hidden" and/or "System" attributes.  Although you can make such objects visible in Windows Explorer (from the Tools menu > Folder Options > View), you should instead run TreeSize Free to determine what specifically is taking up the space.  Be sure to run TreeSize Free "as administrator." 

https://www.jam-software.com/treesize_free/

The subfolder that usually takes up the most space in C:\Users\<username>\AppData\Local\Microsoft\Windows is the Internet Explorer cache -- Temporary Internet Files.  That, however, should never be as large as 176 GB.

If you report what is taking up the space, we can probably tell you how to deal with it.
