Windows 7 can't display all items at that resolution.  Plus it will (in most monitors) cause blurry text and stretched graphics.

Switch your gamertag to a different Windows LIVE ID

You can change the Windows LIVE ID associated with your gamertag. Here’s how:

  1. Sign in using your Xbox LIVE gamertag.
  2. Go to Settings and then select Account.
  3. Go to Your Information and then select Windows Live ID.
  4. Select Change Windows Live ID.
  5. Enter the password for the Windows Live ID that is currently associated with your gamertag and then select Sign in.
  6. Enter the email address and the password for the new Windows Live ID that you want to associate with your gamertag and then select Sign in.
  7. Select Yes, change to change the Windows Live ID associated with your Xbox LIVE account.
  8. Select Update Contact Information to update your contact information or Done to finish.
