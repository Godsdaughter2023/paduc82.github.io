Hi there,

I had the same problem as described above and  was able to resolve.

These are the steps I took.

I manually searched for the file named ASUS smart gesture and deleted it from the computer. Go to File Explorer, This PC, and search for "Asus Smart Gesture" using search function located on the top right corner. Delete every file that pops up, and do it just once. Files will not disappear until you restart computer.

Next, go to ASUS support website and download the same version of the touchpad driver for your notebook (for the same operating system you had initially installed, in my case that was Windows 8).

Once you have downloaded driver, leave the ASUS webpage. Go back to File Explorer, open Downloads folder, click on Disable3fun.exe folder and extract all folders. Once extracted, open the folder SetupTPdriver.msi and follow the next few steps and the driver should be successfully installed. Restart you computer.

Go to device manager and look under "Mice and other pointing devices", select ASUS touchpad, click od Driver, and update driver. This should automatically give you the newest version for the Windows 10.

Restart computer and you should be able to see the "Smart Gesture" icon in notification area. From here, everything should work just fine.
