In general, it should be enough to right-click on the game installer, "properties", select "compatibility" tab, choose "Compatibilty mode: Windows XP SP3" and "run as administrator". Create an own \Program Files or \Games folder on another partition/disk and point the game installer to that location (usually the installer has a "browse" button to select another installation target than the default one).

With end of support Microsoft is not telling you that your Windows 7 operating systems won't work.

Whether they still work or not with particular programs is up to the particular program manufacturers.

Main tasks for task manager. - Microsoft Community

dude_20_80 say:

Main tasks for task manager.

What are the main task that should be running in the task manager?  I need a list of them.  The ones you have to have for the computer to run.  Just the basic task that are in the programming.  Nothing extra. 
john john __ say:

The ones that you must have and the ones that you should   have are two different stories altogether!
There are six critical system services, Windows XP needs these services to start and run porperly.  These critical services cannot be killed, killing these processes would end the Windows session:

- Csrss.exe (Client/Server Runtime Server Subsystem)
- Lsass.exe (Local Security Authority Subsystem Service)
- Smss.exe (Session Manager Subsystem)
- RpcSs (Remote Procedure Call Server Service)**
- Winlogon.exe (Windows logon process)
- services.exe (Windows Service Controller)

** RpcSs runs inside one of the SvcHost.exe.  RpcSs is not critical in its own right and strictly speaking Windows XP can run without it but hardly anything runs without it.

These are the 6 critical NT processes, without these 6 items things don't work too well!  Along with that the Task Manager would show:

- System (the kernel or kernel-mode threads)
- System Idle Process (Not a process or service but a single thread that runs on each processor, its sole task is to account for processor idle time or time spent doing nothing.)

There you have it, the minimum 8 items that will or should always show in the Task Manager, add the Task Manager itself to the list and it will give you 9 processes.
