vasallo say:

Figured it out!

1. Go to XP Mode then "Printers and Faxes" control panel
2. click "Add a Printer" to start the printer-installation wizard
3. select "Local Printer attached to this computer" (and clear the checkbox next to "Automatically detect and install my Plug and Play Printer").
4. select one of the TS number ports. (This brings up a list of ports labeled "TS0001" and higher). 

One of these numbered ports leads to your Windows 7 printer—but you have no way of knowing which one. The only thing to do is choose one of the TS ports, continue with the wizard, install the drivers for your printer when prompted, and then, when you reach the last page of the wizard, click the button that prints a test page. If the test page prints, you've chosen the right TS number. If the test page doesn't print, you chose the wrong number, and you need to go back and repeat the process until you find the one that works.

a. user say:

Hi,

Transferring files, data, or information to other user profile in Windows 10 can be done as follows:

  1. First, click Start, type and select Control Panel.
  2. Select System and Security and then System.
  3. On the left side, click Advanced System Settings.
  4. Under User Profiles, click Settings.
  5. Select the profile you want to copy.
  6. Click Copy to, and then enter the name of, or browse to, the profile you want to overwrite. This will usually be in C:\Users\username, where username is the name of the profile to which you are copying.
  7. Click Permitted to use, and then add all user accounts to which you want to give full profile control.

Code et message d'erreur

Le code d’erreur 80190821 est précédé du message suivant :

Impossible d’extraire les informations depuis Xbox LIVE.

Code d’état : 80190821

Problème

Si le code d’erreur 80190821 s’affiche lorsque vous tentez d’effectuer un achat sur Xbox LIVE, cela signifie qu’un problème lié au moyen de paiement est survenu. Par exemple, cette erreur s’affiche lorsqu’un paiement par carte de crédit est refusé.

Ce problème peut se produire si la société émettrice de votre carte de crédit effectue des vérifications de sécurité sur votre compte avant d’autoriser le paiement.

Solutions

Pour résoudre ce problème, essayez les solutions suivantes :

Solution 1 : vérifier l’état de Xbox LIVE

Au cours des maintenances système planifiées ou lorsque des problèmes Xbox LIVE imprévus se produisent, certaines fonctionnalités Xbox LIVE peuvent être indisponibles. Pour connaître le statut de Xbox LIVE, consultez État du service.

Solution 2 : tester la connexion Xbox LIVE

Pour tester la connexion à Xbox LIVE, procédez comme suit :

  1. Accédez à Paramètres, puis sélectionnez Système.
  2. Sélectionnez Paramètres réseau.
  3. Sélectionnez Réseau câblé ou le nom de votre réseau sans fil, si vous y êtes invité.
  4. Sélectionnez Tester la connexion Xbox LIVE.

Solution 3 : vérifier votre adresse de facturation

Assurez-vous que l’adresse de facturation de votre compte Xbox LIVE correspond à celle utilisée par la société émettrice de votre carte de crédit. Vérifiez également que les informations de paiement de votre compte Xbox LIVE, y compris la casse, correspondent à celles utilisées par la société émettrice de votre carte de crédit.

Pour changer l’adresse de facturation de votre compte Xbox LIVE, consultez Modifier les informations de votre compte Xbox LIVE.

Solution 4 : contacter votre banque ou institution financière

L’achat Xbox LIVE peut avoir échoué pour l’une des raisons suivantes :

  • La carte de crédit n’a pas été activée.
  • La carte de crédit a été refusée à cause de fonds insuffisants.
  • La carte de crédit n'est pas autorisée pour les achats en ligne ou la facturation automatique.
  • La société émettrice de carte de crédit bloque la transaction.

Contactez la société émettrice de votre carte de crédit ou votre banque pour vous assurer que la carte que vous utilisez a été activée, qu’elle dispose des fonds suffisants et que la transaction n’est pas bloquée pour une raison quelconque.

Solution 5 : réessayer d’effectuer votre achat plus tard

L’achat Xbox LIVE peut être bloqué pour l’une des raisons suivantes :

  • Trop de tentatives d’achat ont été refusées récemment.
  • L’option de paiement ou le compte Xbox LIVE est utilisé à partir d’un endroit suspect.
  • L'option de paiement fournie présente un défaut de paiement qui empêche son utilisation pour des achats supplémentaires.

Attendez un peu et réessayez d'effectuer la transaction dans un délai de 24 à 48 heures. Un nombre important d’échecs lors de l’achat peut empêcher le traitement des transactions pendant une période donnée au cours de laquelle aucune tentative d’achat ne doit être effectuée (généralement entre 24 et 48 heures).

Solution 6 : utiliser une carte Microsoft Points ou une carte d’abonnement prépayée

Au lieu d’utiliser une carte de crédit, vous pouvez acheter une carte Microsoft Points prépayée dans un point de vente et vous en servir pour réaliser vos achats. Notez que certains achats ne peuvent pas être réglés à l’aide d’une carte Microsoft Points ou d’une option prépayée. Avant d’acheter des Microsoft Points ou une carte prépayée, assurez-vous que ce moyen de paiement vous permettra de régler l’élément que vous souhaitez acheter.

Solution 7 : vérifier que vous utilisez un moyen de paiement approuvé

Le Marché Windows Phone permet d’utiliser les cartes Visa, MasterCard ou American Express si celles-ci prennent en charge les achats internationaux. Même si votre carte porte un logo Visa ou MasterCard, il est possible qu’elle ne soit pas autorisée par votre banque ou société émettrice à effectuer un achat international.

Remarque Les cartes prépayées, les cartes de débit et les cartes cadeau (même celles portant un logo Visa ou MasterCard) ne sont pas considérées comme des cartes internationales et ne sont pas acceptées actuellement par le Marché Windows Phone.

Si votre carte de crédit constitue un mode de paiement approuvé, essayez la solution suivante.

Solution 8 : vérifier le numéro, la date d’expiration, le code de sécurité et le nom de la carte

Entrez à nouveau le numéro de la carte sans espaces, virgules ou autres caractères non numériques. Vérifiez que la date d'expiration, le code de sécurité et le nom de la carte que vous avez entrés sont corrects et correspondent aux informations figurant sur la carte.

Windows has a number of recovery features to get "back up and running" quickly after a problem occurs. Some of these recovery features are more impactful than others, to be sure to read the impacts carefully.

  • Option One: System Restore (Safest, try this first)
    If the problem just began recently, you can restore Windows back to an earlier point in time, called a restore point. System Restore doesn’t change your personal files. It only takes a couple minutes, and you can undo the process or try another date if it doesn't fix the problem on the first try. When you aren't sure where to go, this is by far the safest, fastest, and easiest option to try first.
    1. Press +R to show the Run box, type RSTRUI and click OK.
    2. Choose a date when the computer was working correctly, and click Next to restore the system files from that date.
    3. Check to see if the problem is fixed. If not, you can go back to step 1 to either undo the restore, or try another date.
  • Option 2: Create a new profile
    Because profiles are separated from each other and the system, creating a new profile on your computer can help troubleshoot and resolve issues. Many times an issue in a profile can be resolved by creating a new one. It takes a little time to set up your preferences and drag your files to the new profile, but it can save many hours of headache. As with the System Restore, this is a "safe option", because no files will be lost if you try this.
    1. Open the Control Panel, go to Accounts, Other Accounts, and then Add An Account.
    2. Login to the new account and see if the issue is fixed. If so, you can open the C:\Users folder and drag your files into the new profile.
