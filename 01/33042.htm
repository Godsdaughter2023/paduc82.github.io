cmdrkeene say:

Google Chrome is using a HUGE amount of your RAM in that picture. If you don't want that hogging your memory, you can go into Chrome settings and turn off the "run in the background" option so it only runs when YOU want to use it.  This will save a lot of startup RAM.

But you don't necessary have to worry: that is a very normal amount of RAM to have available. Remember, empty RAM is wasted RAM.  If your RAM was empty, it means you're forcing your computer to run slower -- because every time you want to use something, it would have to load from a slower source than RAM (like a hard drive, which is hundreds of times slower).

You also can't look at used RAM and think it makes the computer slow.  Windows is a very intelligent system, it will pre-load things into RAM but still mark it as "available". If you ever load something that needs that pre-used RAM, it will instantly (in one millisecond) clear it out and use it for something else.  This pre-used RAM technology is called PreFetch and SuperFetch, and it makes computers much more responsive.

vijay a. verma say:
Hi SakshiU
Greetings! I am Vijay, an Independent Advisor. I am here to work with you on this problem.

As first step of troubleshooting let's try to rule out Skype client as the source of trouble. Let me know the results after below steps to advise you further on this.

You may need to go for Skype for Desktop, if you are using Store version of Skype. Use below method to uninstall and install Skype to solve this problem.

CTRL+SHIFT+ESC to invoke Task Manager > Kill any instance of Skype.exe

A. If this is UWP (Microsoft Store) Skype app of Windows - Windows Key+i > Apps > Click Skype and Uninstall
Install Skype again from Microsoft Store. You can install from https://www.microsoft.com/store/productId/9WZDN...

If above doesn't work -

Choose to install desktop client of Skype from and use Skype for Desktop https://go.skype.com/windows.desktop.download

B. If this is Skype for Desktop client
Copy the below line and Windows Key+R and paste into Run window by CTRL+V and Enter
%appdata%\microsoft\Skype for Desktop
Rename this folder also to something say Skype for Desktop_Old
Reinstall Skype again from https://go.skype.com/windows.desktop.download

Do let me know if you have any more question or require further help.

ruwim.b say:

Reset all local Skype settings.

Quit Skype or use Windows Task Manager to kill any Skype.exe process. Go to Windows Start and in the Search/Run box type %appdata%\microsoft and then press Enter or click the OK button. The Windows File Explorer will pop up. There locate a folder named “Skype for Desktop”. Rename this folder to something different, e.g. Skype for Desktop_old.

Reboot your computer and try again to start Skype.
