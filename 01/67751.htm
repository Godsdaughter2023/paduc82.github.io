How to temporarily prevent a Windows or driver update from reinstalling in Windows 10

To remove an unwanted Windows 10 update and to prevent its reinstallation, Microsoft produced this utility  How to temporarily prevent a Windows or driver update from reinstalling in Windows 10

This utility has been around for several  months but I have not found any reports in this forum about it so, if you use it, please do post your experiences of it for the benefit of other users.

This is an alternative to turning off all hardware-related updates -

     right-click on the Start button,

     System,

     Advanced system settings -

          select

               No, let me choose ...,

               and

               Never install ... 

If you use the utility or if you turn off all hardware-related updates, you should consider how you would check for any useful updates manually [possibly by checking with hardware makers' support sites every so often].  You might find this guidance useful - Where to get hardware drivers.
