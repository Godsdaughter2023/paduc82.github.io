The Microsoft virtual wifi miniport adapter is a new feature in Windows 7, Wireless Hosted Network:

See: About the Wireless Hosted Network

This issue cannot be interrelated with installing Windows Update. This device is just a virtual device and it should not affect connecting the wireless network with the physical wireless adapter.
