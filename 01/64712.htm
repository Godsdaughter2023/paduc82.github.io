Since the sndvol32.exe is one of the 3498 files that Windows File Protection looks after, it should never be missing.  For it to be missing would require some extra effort even to delete it (or rename it) on purpose - try it!  If one of the 3498 protected fils sfiles ever comes up missing from the system32 folder, Windows File Protection will want to quickly and silently replace it if it can using the backup copy from the dllcaches folder (that is what the dllcache folder is for).

Protected Files can sometimes come up missing if your antivirus/antimalware application thinks the file is suspicious and quarantines (effectively deletes) the file.  You can replace the missing file with a backup copy of the file that is on your system in another folder if you can find one.  Sometimes even the backup copy gets quarantined (effectively deleted).

c:\windows\system32\dllcache

Using Explorer, browse or navigate to the following folder (or click Start, Run and in the box enter):

c:\windows\system32

Respond in the affirmative if asked to overwrite the existing file and then you should see a message that 1 file(s) was copied.

andre da costa say:

I am not familiar with Apple's repair tools, so it would probably best if you go to Apples support site for issues and support for Boot Camp users:

Apple - Support -Boot Camp Boot Camp : Windows 7 installation frequently asked questions

Here are some solutions you could try based on my memory using Power PC Mac's, they might help:

- Inset your OS X installation disk, and restart your Mac while holding down the C key.

- This should boot into the OS X installer.

- On the left side of Disk Utility window, click on the drive that contains your Mac OS X volume. This might be different, if you don't see Disk Utility, click Utility on the menu bar and click Disk Utility.

- Click on the verify button to check your drive for problems.

-If First Aid reports problems, click on the Repair button.

- Quit Disk utility by choose Quit from Disk Utility menu

- Quit the installer application

- When the Quit sheet appears, click on the Restart button.
