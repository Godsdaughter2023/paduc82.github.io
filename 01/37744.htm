  • active

How to fix this problem?

In Windows, the little arrows remind you that you're at a shortcut. To remove the arrows, please follow the steps below:

  • Press Windows key+R. Type regedit, then hit Enter.
  • Navigate HKEY_LOCAL_MACHINE – SOFTWARE – Microsoft – Windows – CurrentVersion – Explorer – Shell Icons.

Note:  You may not have the Shell Icons key under Explorer. Right-click on Explorer in the left pane and choose New - Key, then name it Shell Icons.

  • Right-click on Shell Icons folder, then choose New - String Value.
  • Change the name to number 29.

  • Double-click the new 29 value then copy and paste %windir%\System32\shell32.dll,-50 into the Value data.

  • Click OK.
  • Once done, close Registry Editor, then restart you computer to see the changes.
