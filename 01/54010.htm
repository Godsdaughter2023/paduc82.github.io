You cannot recover the actual photos from the .wmv

at the bottom of the Display Screen. You will be prompted
to save your capture.

pa bear - ms mvp say:
I have installed Silverlight and MS security essentials.


Installing Silverlight or Microsoft Security Essentials (or MS Office or one or more Windows Live applications) can have several consequences:

  • If Windows Update was your default update source, it will be "flipped" to Microsoft Update.

  • If Automatic Updates was disabled (turned off), it will be enabled (turned on) again.

  • If Automatic Updates was set to the Download updates but let me choose... option or the Check for updates but let me choose... option, AU will be reset to the default Automatic setting.

One or more of the above would explain the behavior you saw after installing Silverlight and MSE.

 

PS: The above is documented in the EULA you must accept before installing either application - but hardly anyone reads those.

pa bear - ms mvp say:

While Windows Update provides you with updates specifically for Windows, Microsoft Update expands the service to download and install updates for other Microsoft software (i.e., MS Office, Windows [Live] Essentials, Silverlight & Microsoft Security Essentials).

 

If you "flip" back to Windows Update as your default, you will no longer receive any security updates, program updates or performance updates for Silverlight & MSE - and you really don't want to do that!

 

PS: You can change your Automatic Updates setting back to the Download updates but let me choose... or the Check for updates but let me choose... option but the change will NOT be applied to any updates already in the queue. As far as AU's concerned, you've already approved any such updates.
