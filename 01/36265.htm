shinmila h say:

Hey ChicGeek02,

In this scenario, since Windows is not getting past the screen “Installing 1 of 2 updates..”, there’s no point in waiting for it to complete. Instead, restart the computer and get into Windows Recovery Options so that we can use the various tools available to troubleshoot.

To open the System Recovery Options menu on your computer

1. Remove all floppy disks, CDs, and DVDs from your computer, and then restart your computer using the computer's power button.
2. Do one of the following: 

        o If your computer has a single operating system installed, press and hold the F8 key as your computer restarts. You need to press F8 before the Windows logo appears. If the Windows logo appears, you need to try again by waiting until the Windows logon prompt appears, and then shutting down and restarting your computer.

        o If your computer has more than one operating system, use the arrow keys to highlight the operating system you want to repair, and then press and hold F8.

3. On the Advanced Boot Options screen, use the arrow keys to highlight Repair your computer, and then press Enter. (If Repair your computer isn't listed as an option, then your computer doesn't include preinstalled recovery options, or your network administrator has turned them off).

4. Select a keyboard layout, and then click Next.

5. On the System Recovery Options menu, click on System Restore to open it. This restores your computer's system files to an earlier point in time without affecting your files, such as e mail, documents, or photos.

Note: If you use System Restore from the System Recovery Options menu, you cannot undo the restore operation. However, you can run System Restore again and choose a different restore point, if one exists.

For other ways of accessing System Recovery Options and for more information on the tools available, check this article.

What are the system recovery options in Windows 7?
http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7

Now when you install the updates, please turn off Automatic Updates. Instead, install one update at a time. This will help in figuring out which updating caused this.

1. Open Windows Update by clicking the Start button. In the search box, type Update, and then, in the list of results, click Windows Update.
2. In the left pane, click Change settings.
3. Select “Check for updates but let me choose whether to download and install them”.
4. Click OK.
Now from the list of updates, check one update at a time to download and install. Note down the KB article number so that we know for sure which update is causing your PC to crash.
