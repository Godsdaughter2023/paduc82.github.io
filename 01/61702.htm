daniellefergus say: Hello osasapers,

Typically when you go through the installer wizard, there is an option to install for all users on computer or just 'this user'. You will need to make sure 'this user' user is selected, to prevent it from installing on other users profiles on the machine.

if you dont see that option when you are installing the program, there are these steps to try as well:

1. Navigate to your program files and locate the .exe for the program being installed.
2. Right click on the .exe and go to properties
3. click the compatibility tab
4. at the bottom of that box, you should see change settings for all users
5. in the privilege level section, you can select run this program as administrator.
6. click apply then ok.

Please let me know if that helps!

REG ADD "HKLM\SYSTEM\CurrentControlSet\Control\SafeBoot\Minimal\MSIServer" /VE /T REG_SZ /F /D "Service"

net start msiserver

'mike p say:

get out of safe mode maybe;

If you can use another computer .. make yourself a bootable win10 installer on a usb drive.

Boot off the usb drive as if you were going to fresh install win10 & on its second page is a link to its repair environment.  Use its command prompt;

cd /d c:

bcdedit /deletevalue {default} safeboot

then exit and continue to windows.

sometimes when booted off a usb thumb drive the drive letters will not be normal - you may need to figure it out which is your windows drive... which you can trial and error ..  example;

cd /d f:

dir

the dir command will list the file structure so you should be able to tell which is your windows partition.

In the future you may want to consider asking cortana for Settings, Advanced Startup and use it to enter your safe mode rather than msconfig.
