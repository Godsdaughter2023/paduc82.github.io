Try selecting the key for Boot Options when the SONY logo appears...

Since you noted the EFI Error, it might be that your machine is trying to boot to the wrong device...

Not device actually, but a file on your HDD...

When using EFI\UEFI Bios, the machine starts out by booting to the EFI file, and then is redirected to Windows... Try setting the Boot Option to EFI\UEFI type of selection... The actual name varies a little by manufacturer.

There are two settings in your BIOS that you may have to disable to get the machine to boot from the CD\DVD drive....

Start with FAST BOOT, and then SECURE BOOT...
