hairyfool say:

You can't do a clean install using the original key.

This key is issued to the manufacturer to pre-install Windows in the factory, each manufacturer has its own key for each version of Windows but all installations of one particular version from a manufacturer will have the same key. The factory install is always preactivated which is why you do not have to go through the activation process on a new computer.

DVD/USB key installs must be activated on line after setup completes but the Microsoft servers know that the manufacturer keys do not activate and will reject any attempt to do so which is why you get the error.

Do the factory recovery but make sure the  computer is not connected to the internet. Run Programs and Features and uninstall everything you do not want. Most Antivirus program developers provide an uninstaller from their website which does a better job of scrubbing the system of the leftovers following the normal uninstall. Run msconfig and check that all entries for the programs that you have removed have been cleaned. Check the Program files directory(s) for anything left by the uninstallers

The majority of "bloatware" are just files on the disk and have no implications of the running condition of Windows. Only those that pre-load have a potential to be an issue which is why you check with msconfig. If you clean up before any internet connection you prevent any updaters trying to upgrade the software you don't want anyway. Even the odd rogue entry in the registry will have no effect if the file it refers to does not exist.

Yes a clean install is preferable but the factory image will have working drivers already installed and provided a basic cleanup is done as the 1st step the results are a reasonable alternative. Otherwise you will need a new key at a cost, there is no alternative.

Hello,

In File Explorer, under This PC, locate your CD/DVD drive, right click and select Eject.

If that doesn't works, the CD-ROM, CD-RW, or DVD drive can be opened using the emergency eject pinhole. Use the following steps:


  1. Close any applications using the drive and then shut down the PC.
  2. Find the pinhole on the drive door.
  3. Bend part of the paperclip to a point. Insert the paperclip slowly until there is resistance, then gently push until the drive door is opened.
  4. Pull the drive tray out and remove the disk. Close the tray.
