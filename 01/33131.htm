paul a. smith say:

Hello ITknowledge's answer is correct assuming you're using Professional or up.  If you're using one of the Home Premium you'll need to edit via the registry instead.

You'll need to open regedit which you can do from the Start Search.

And then make your way to HKEY_Local_Machine\Software\Policies\Microsoft\Internet Explorer\Privacy.

You'll then need to create a new Dword value, called EnableInPrivateBrowsing and you'll want to give it a value of 0.

If you're a bit overwhelmed I've created some registry files which can make the changes.  The first will disable it, the second will enable it.

http://www.windowsresource.net/downloads/inprivate/disableinprivate.reg and http://www.windowsresource.net/downloads/inprivate/enableinprivate.reg

Download to your desktop, or another location of your choice, double click them.  After which you'll be promoted if you want to run them, select Run.  Then the regedit program will want to request administrator rights, click Yes.  And finally it'll ask if you want to merge these changes into the registry.  Click yes.  InPrivate should then be disable for all user accounts.
