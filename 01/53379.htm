frederik long say:

Folders do not get "locked". If they have a lock symbol then this simply means that only accounts with admin privileges can access them.

"Renaming" an account is pointless with respect to access permissions. The underlying account name remains - all you change is the display name. If you could do this then you would have a huge security hole.

Creating a new account under the old name is equally pointless. Windows will not grant access to a resource created previously under an account of the same name. If you could do this then you would have yet another security hole.

You now need to seize ownership of your folders - see here:
