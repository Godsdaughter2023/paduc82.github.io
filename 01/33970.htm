xbf eladio m say:

Hey Gabriel_297

Thank you so much for taking the time to post this on the Xbox Forums.

On the consoles, you will be unable to use any user-created content, due to not having anyway to modify your game. You can however get texture packs, skins and map via the in-game Minecraft Marketplace.

If you are talking about Minecraft Windows 10 edition, The Minecraft: Windows 10 Edition is actually just Pocket Edition played with a mouse and keyboard with a PC-ish UI. So PC mods most likely will not work on Windows 10 Edition unless it is designed to do so.

The Minecraft: Windows 10 Edition is completely different from the Java Edition (also known as the PC Edition). As such, mods and saves from the Java Edition will not work on the Windows 10 Edition. You can get add-ons that are specifically created for Windows 10 Edition in-game though.

That being said, I don't think that there are any restrictions to using custom maps and texture packs. The game was created with consideration of the Minecraft mod community so there shouldn't be any enforcement actions if this is done on a private world.

As long as you can find content that is compatible with your platform you shouldn't run into any issues.

I understand you are trying to remove everything and re-install Windows 8 however you are getting error. I will be glad to assist you with your query.

There may be possibility of any third party software conflict. I would suggest you to perform clean boot and check.

Putting your system in Clean Boot state helps in identifying if any third party applications or startup items are causing the issue. You need to follow the steps from the article mentioned below to perform a Clean Boot.

Refer to the link to perform clean boot.

How to perform a clean boot in Windows

http://support.microsoft.com/kb/929135/en-us

Note: Refer "How to reset the computer to start normally after clean boot troubleshooting" to reset the computer to start as normal after troubleshooting.

In most cases, once you start to refresh or reset your PC, it’ll finish on its own. However, if Windows needs missing files, you’ll be asked to insert recovery media, which is typically on a DVD disc or thumb drive. If that happens, what you’ll need depends on your PC.

  • Are you performing Reset PC option with an installation media?

If you need to reinstall Windows 8 or you want to install it on a partition and you don't have installation media, you can create it using the Windows 8 Setup program.

Follow the instructions below to create an installation media.

Click here to download Windows 8.

When prompted to run or save the file, choose Run

When prompted to enter a product key, enter the 25-digit product key you received when you bought Windows 8.

Select the type of media you want to use: either a USB flash drive or a DVD.

To create a USB flash drive

  1. Click USB flash drive.

  2. Plug the USB flash drive into a USB port on your PC.

  3. When prompted, select the USB drive you want to use and then click Next.

    If you have files on the USB flash drive, they'll be deleted.

To create a DVD

  1. Click ISO file and then select the location to save the ISO file.

  2. Insert a blank, recordable  DVD into your DVD drive.

  3. Burn the ISO file to the DVD using Windows Disc Image Burner, or another DVD burning program.

Note: Make sure the USB flash drive or DVD you're using has nothing else stored on it, and has enough free space to download the Windows 8 Setup program: about 1.8 GB for the 32-bit version and about 2.3 GB for the 64-bit version.

Once you have the installation media created, then refer to the article: How to  refresh, reset, or restore your PC

http://windows.microsoft.com/en-US/windows-8/restore-refresh-reset-pc

Warning (when you perform Remove everything and re-install Windows): All of your personal files will be deleted and your settings will be reset. All apps that you installed will be removed. Only apps that came with your PC will be reinstalled.

Hope this information is helpful. Please do let us know the result, we’ll be glad to assist you.
