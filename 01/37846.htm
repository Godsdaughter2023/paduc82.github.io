The error message: An unexpected I/O error has occurred. Status: (0Xc00000e9) is a hardware error pertaining to your system’s BIOS.  Basically the error message along with the code is telling you the computer is not able to reliably communicate with the hard drive. Your DVD/CD drive could be failing somehow or there could be issue with cable (IDE cable) connected between DVD/CD drive and motherboard.

 

Perform Automatic repair by downloading Windows 10 ISO image from the Media Creation Tool and check if the issue is resolved.

  4. Select the correct time and Keyboard type.
  5. Click Repair your computer in the lower left corner.
