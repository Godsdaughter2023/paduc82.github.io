hairyfool say:

If fully off Windows has been completely unloaded and has no method of turning it back on, the memory is empty and the CPU stopped altogether and apart from a 5v Standby supply the PSU is off.

That 5v is needed in order give the Power button something to signal the system to start.

Otherwise specific external devices can be made to switch it on according to BIOS settings. A network administrator can send "magic packets" to the network card to switch the computer on, this is how they can send updates out overnight even if the computer is off. Alternatively a modem would wake up a computer in order for information to be collected, often used in shop chains. Both these use the 5v standby to power the card only, that is why you see the network card lights active even if the computer is off.

Finally the inbuilt clock can work essentially as an alarm clock to power the PC up at a programmed time. Somewhere in the BIOS setting would be a screen to enter a time and that would be the time to wake up each day. As the previous 2 are only seen is special environments these are unlikely so that leaves the clock.

Working on that basis and the reasonable assumption your BIOS has that feature (all the ones I have worked on do) find the setting as it must be in one of the menus. If it is zeros then I am wrong but at least it is eliminated.
