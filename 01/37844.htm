geethu b say:
Hi,
As I understand the issue, you want information about coinlog.log text document. And want to know if it is a malware. Please let me know if this is not correct.
To let you know coinlog.log is the co-installer log file. A co-installer is a Microsoft Win32 DLL that assists in device installation. Co-installers are called by SetupAPI as "helpers" for Class Installers.
For example, a vendor can provide a co-installer to write device-specific information to the registry that cannot be handled by the INF file.
A DIF_DESTROYPRIVATEDATA request directs a class installer to free any memory or resources it allocated and stored in the ClassInstallReserved field of the SP_DEVINSTALL_PARAMS structure.
For additional information refer to the link:
http://msdn.microsoft.com/en-us/library/ff543669.aspx
