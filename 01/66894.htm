lemp say:

"CMOS Checksum Bad" means that something is wrong with memory that the computer uses to boot up.  Although there are other possibilities, the odds are very high that replacing the "CMOS battery" will fix the problem.

Because such batteries are generally inexpensive and easy to replace (except for some laptops), you should start there.  After you replace the battery, you may have to "press F1 to load the default values and continue" one more time.

What is the make and model of your computer?

lemp say:

If it's the CMOS battery (and it does seem as if that's the problem), then no, you won't damage anything by waiting ... you'll just have to keep pressing F1 every time you start up.  You should fix it soon, however, because sometimes Windows activation problems (as well as other software problems) can be caused by an incorrect clock time/date.

I assume that if you've been looking for a replacement battery that you've already opened up the computer and determined what battery you need.  If it's a CR2032 (picture below), which is the most commonly-used CMOS battery, I'm very surprised that you can't find it.  If you live in the US, any chain drugstore (CVS, Walgreens, etc.) should have it.  If you can't find a convenient local store, these batteries are readily available online.

duncan graham say:

Hi,

Finally found the answer! Don't know what search term I used or what I did differently but have now found a different HP support page with a patch. See below:

http://h20000.www2.hp.com/bizsupport/TechSupport/Document.jsp?lang=en&cc=us&objectID=c01390447&jumpid=reg_R1002_USEN

When printing to the HP Color LaserJet 2600n connected to the network the following error may appear Spooler “SubSystem App has encountered and problem and needs to close. We are sorry for the inconvenience.”

 

SOLUTION:

 

Download and install the "HP Port Monitor 7.1.3.31 Patch" available atwww.hp.com/support/clj2600n . SelectDownload Drivers and Software then select your operating system.

A Microsoft Fix it wizard is available now which will resolve several spooler related errors:http://support.microsoft.com/kb/2000007 .

This restores the print spooler to default settings.

Resolves errors:

·    System error 1068 has occurred

·    The dependency service or group failed to start

·    Spooler SubSystem App has encountered a problem and needs to close

·    We are sorry for the inconvenience. Operation could not be completed

·    The print spooler service is not running

 

CAUSE:

This error occurs when a newer version of HP’s Standard TCP/IP Port monitor (hptcpmon.dll), that is not compatible with the HP Color LaserJet 2600n, has been installed by another HP product. Both the HP Color LaserJet 2600n and the other product must both have performed network installations on the same system to encounter this error. To verify the issue browse to the C:\windows\system32 directory and find the filehptcpmon.dll . Right click on the file name and choose Properties . In the Properties dialog choose theversion tab. If the version of hptcpmon.dll is previous to 7.1.3.31 use the patch to update the file.
