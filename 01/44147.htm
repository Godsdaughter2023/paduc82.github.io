Computer\HKEY_USERS\S-1-5-18\Control Panel\International\User Profile System Backup (Languages Key)

Computer\HKEY_USERS\S-1-5-18\Keyboard Layout\Preload

Computer\HKEY_USERS\S-1-5-18\Keyboard Layout\Substitutes

I made a registry backup, deleted them all, and restarted. So far, it didn't happen again, so hopefully it is gone. I'll update this if it appeared again.

I'm Independent Advisor and Microsoft MVP, here to help you with your question.

First, check the type of license installed on your system.

1. Open administrative Command Prompt.

2. Type slmgr /dlv and press Enter key.

3. Note the Product Key Channel section Windows Script Host box:

If it mentions OEM <method>, you’ve OEM license installed.

If it says Retail, you’ve retail license installed.

If it mentions GVLK, VOLUME_<method> then you’ve volume license installed. Note that GVLK stands for Generic Volume License Key.

Retail license have the transfer rights, while OEM license is tied to motherboard and can't be transferred to other machine.

richard urban say:

Battery Drain Problem = Solved

Like so many others, I have been seeing the condition where a fully charged battery in my Surface Pro 4 (M3) goes from 100% to about 20% after being asleep overnight.

I tried everything that was mentioned in these forums and nothing would alleviate the problem. I ran the battery test and sleep study test using powercfg in the command prompt and found what so may others seemed to find. But no real answers!

Then I hit gold, at least for my computer. I ran powercfg /energy. This indicated that a USB device was causing the problem. See the image below.

You can see that there is one error posted under Energy efficiency problems were found.

I powered off the Surface 4, removed the Micro SD card (Sandisk 128 gig ) and started the computer again. After waiting a couple of minutes for it to stabilize, I ran powercfg /energy again. This time there were no errors reported. See attached below.

When you run this test you must open the results using Edge. Internet Explorer will not do it.

Anyway, I charged the computer to 100% and put it to sleep for 48 hours. When I woke the computer I still showed 92% battery capacity remaining.

So people, run this test and see what shows up for your computer. It may be just the Sandisk SD card or it may be all SD cards. I am not in the position right now to go out and spend yet more money to try a different brand.

As for me, I am retiring the SD card and will just plug in an external hard drive when I need to store larger files. I also plan to run the above test with my external USB drive plugged into the USB port to see what the results will show.

I hope this helps some people.

Richard Urban
