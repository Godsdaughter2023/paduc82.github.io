Solution 2: Uninstall and reinstall the Teredo adapter

To uninstall the Teredo adapter:

Right-click the Start icon, and then select Command Prompt (Admin).
Run the following command from the Admin command prompt:

netsh interface Teredo set state disable
Right-click the Start icon, and then select Device Manager.
Select View > Show Hidden Devices.
Select Network Adapters.
Identify any adapters that contain "Teredo," such as Teredo Tunneling Pseudo-Interface or Microsoft Teredo Tunneling Adapter. Right-click the adapter name, and then select Uninstall.
Restart your PC.

>netsh
netsh>int teredo
netsh interface teredo>set state disable

While still in the command prompt i also executed this command and i was able to play online without restarting my PC, you can actually complete this while the game is running and be able to connect right away:

netsh interface teredo>int ipv6

    a. user say:

    Hello,
     
    To answer your query, you need to enter the Administrator password to access the batch file.
    Windows 7 Instructions
    In order to run as an Administrator, create a shortcut for the batch file.
    1.Right click the batch file and click copy
    2.Navigate to where you want the shortcut
    3.Right click the background of the directory
    4.Select Paste Shortcut
     
    Then you can set the shortcut to run as administrator:
    1.Right click the shortcut
    2.Choose Properties
    3.In the Shortcut tab, click Advanced
    4.Select the checkbox "Run as administrator"
    5.Click OK, OK
    Now when you double click the shortcut it will prompt you for UAC confirmation and then Run as administrator (which as I said above is different than running under an account in the Administrator Group)
     
    Hope this helps.
