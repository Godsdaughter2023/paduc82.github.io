How to retrieve and use a previous File History - Microsoft Community

aldago say:

How to retrieve and use a previous File History

My computer crashed, I installed a fresh copy of windows 10 Pro, turned on File History and ran a backup. Now, on my side drive I have a File History folder containing folders of previous backups and one folder with the new backup I just ran. I'd like to restore one of the older backups but I can't get to them. When I go to Control Panel/File History/Restore Personal Files the only backup that shows up is the one I just made. How can I access the other folders so that I can restore the files I want?

cmdrkeene say:

When you turned on File History again following the fresh install, it should have asked you where you want to create the file history -- as long as you selected the same drive and folder as last time, it should have said "you have a previous File History already here, do you want to keep using that one?"  And if you say yes, then it continues on using it (adding new backups to it) without skipping a beat.

But since you have two folders now, one for the old set and one for the new one, you must have specified a different folder the new time.  So you can't so easily pick up where you left off.

So at this point, you have two options:

1) Change your file history settings to specify the old location, which will give you the "keep using this?" prompt I mentioned above and let you have a more seamless experience.

or

2) Manually go into the old folder and find the file(s) you want to get back and make your own copy of them (for example, copy and paste them to your desktop).
