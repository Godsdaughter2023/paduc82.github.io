What happens during a check disk? - Microsoft Community

secretgamer96 say:

What happens during a check disk?

I have recently be getting messages to run a check disk at the start up so i have been wondering what exactly happens during a check disk,how long it takes and what do you do in order to end it in case of an emergency.I also want to know the consequences of not doing this soon enough and what it does exactly.
cmdrkeene say:

It just checks that every file is accurately written to the drive by making sure the file system's record matches the actual file (for example, that the logged file size and the actual size are the same).  This takes a lot of time to read every file, which is why a CHKDSK process can take several hours (or more).

 

CHKDSK scans are not necessary and should not be run unless your computer has a problem.  The NTFS file system is a jounaling file system that is very resilient to errors and self-corrects as needed.  It can handle delayed write failures and power outages and will schedule checks when necessary. In most cases, running a check on your own is just a waste of time (and unnecessary disk usage, which will shorten the life span of your hard drive).

 

You can end it safely if you want, just press Ctrl-C together.
