Hi Tan,

Before performing the power shell commands you need to try these steps first:

  1. Open an Administrator CMD window.  If you can’t find a way, use these steps:
  2. Press CTRL+Shift+ESC to open Task Manager.
  3. Click File > Run New Task
  4. Make sure you have a check mark beside “Create this task with administrative privileges”
  5. Type CMD
  6. Type the following commands at the CMD prompt:
        dism /online /cleanup-image /restorehealth
        sfc /scannnow
  7. If EITHER command does not show a good outcome, we need to stop and repair Windows. We have 3 options:
  8. Reset: If possible, remove Antivirus/Sec Software and perform a Reset (keeping data and settings). Customers will need to reinstall apps with this option.
  9. If the above commands both completed and show no remaining issues, run these commands:
    powershell
    Get-AppXPackage -AllUsers |Where-Object {$_.InstallLocation -like "*SystemApps*"} | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
  10. Check to be sure this command ran, showed a blue progress indicator and then returns to the prompt.
  11. If you see a red error, then the command may have been typed or pasted incorrectly, or there may be an larger problem with Windows 10.  Try again.
  12. Check to see if the issue is resolved.
  13. If not, restart the PC.In this case try as a new administrator user account.
