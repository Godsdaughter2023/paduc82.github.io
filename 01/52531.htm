Your store folder is corrupted. I don't know why this happens (only a few generic causes that may or may not apply in particular cases), nor do I know an easy way to repair the damage. Here is a procedure to create a new store folder and populate it again:

 

  

  1. Close down Windows Live Mail and wait a few minutes (the program will continue to run in the background while it does its housekeeping).
               
  2. Open an Explorer window and navigate to your store folder.
     
  3. Right-click on the folder name in the navigation pane and select Rename. Call it Live Mail Backup.
     
  4. A repair should check that all the program files are present and correct and properly registered where necessary. To do this, close all running programs, then press Windows key + R, type wlarp and press Enter. In the window that opens, select Repair all Windows Essentials programs:



    When it's finished, reboot the computer.
     
  1. Launch Windows Live Mail and wait until the program settles down before doing anything. Then press Ctrl-Shift-T to start the Add your email accounts wizard and follow the prompts to add your email accounts again.
               
  2. Click once on Working online on the status bar and disconnect from the Internet to prevent interference from email downloads, then use File > Import messages (Alt-F I) and point the wizard at the Live Mail Backup folder. Don't interfere while the import is going on.
     
  3. Leave the program alone for some time (an hour or two, or even overnight) to allow it to re-index all your messages, then close it down and wait again for half an hour for the program to do its housekeeping.
     
  4. Re-launch the program and look for your messages under Imported folder in Storage folders. Taking it gently, drag folders out of Imported folder and drop them in the location you want them. Again, give the program plenty of time to re-index messages in their new location.

 

 

Note that you can't drag folders out of Storage folders. If you want to transfer messages to an account folder higher up the folder pane, you must select the messages to be transferred, then use right-click >

Good luck!
