Win7 when configured on peer-to-peer Network has three types of Sharing configurations.

HomeGroup Network = Works only between Win 7 computers. This type of configuration makes it very easy to Entry Level Users to start Network sharing.

 Public Sharing = Public Network (like Internet cafe) to reduce security risks.

http://www.ezlan.net/Win7/net_name.jpg

If you use 3rd party Firewall On, Vista/XP Native Firewall should be Off, and the active Firewall has to adjusted to your Network IP numbers on what is some time called the Trusted Zone (consult your 3rd Party Firewall instructions

General example,  http://www.ezlan.net/faq#trusted

If possible configure the Firewall correctly, or totally uninstall it to allow clean flow of local network traffic. Hi Adam,

 

 

Windows Customer Experience Improvement Program (CEIP) upload occurs once every 19 hours. You can verify that CEIP upload completed successfully by viewing details about the CEIP uploader task in Task Scheduler.

To perform these procedures, you must be a member of the Administrators group on the local computer, or you must have been delegated the appropriate authority. You can only verify CEIP upload on a computer that is enrolled in CEIP.

To view upload details in Task Scheduler:

  1. Go to Start screen type Task Scheduler, click on create task from the search results.
  2. In the tree pane, open Task Scheduler Library, open Microsoft, open Windows, and then select Customer Experience Improvement Program.
  3. In the results pane, select the Uploader task.
  4. On the History tab of the results pane, look in the Task Category column for the most recent occurrences of the following two events, in chronological order:
  5. Task Started
  6. Task Completed

  7. On the General tab of the Event Viewer area of the results pane, view the description text for the Task Completed event. To verify that CEIP upload completed successfully, look for description text similar to the following example:

    • Task Scheduler successfully finished GUID_number instance of the "\Microsoft\Windows\Customer Experience Improvement Program\Uploader" task for user user_name.
  8. When you have finished verifying that the CEIP upload task completed successfully, close Task Scheduler.

 

Now verify the issue status.
