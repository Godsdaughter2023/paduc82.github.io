mike hall... always say:

If the above is a normal 'day in the life' for you, the OEM system builder version is not what you should be using.

 

An OEM version of any Windows operating system is the cheapest way to get Windows onto a computer but there is one major restriction. It can only be installed once on a computer and its stays with that computer.

 

What constitutes a computer?

Essentially it is the motherboard and BIOS combination. It doesn't matter what name is on the outside of the case or what peripheral internal and external devices are attached.

My computer is a Gigabyte GA 880GM-UD2H (the motherboard), and if I elect to change it for something newer or older, it is no longer the same computer. It may seem obvious but many do not understand the principle at all.

 

Whether OEM or full retail, Windows can only be installed and operational on one computer at any one time, and that precludes installing it twice on one computer too. If you want a backup, you have to make a backup, not keep multiple installations on separate drives.

 

The advantage of full retail is that it can be transferred from one computer to another, assuming that the previous installation is not maintained or used. Scrapping the old installation is best as it prevents 'accidental' use and you will not be caught out by the licensing and activation issue.

 

In summary, taking into account your M.O. or maybe just plain bad luck, you now need to buy a full retail version of Windows 7. The bad news is that it will not be cheap. The good news is that it will last you way longer than an OEM version.. :-)

Items pinned to the Taskbar (on the left side) are not running, they are there to enable faster access. To further speed up accessing, one (you) can create jump lists of documents/files, as follows:

Click the Windows Orb (Start) > All Programs, navigate to the program, right-click it, and then click Pin to Taskbar. Folders and files can be pinned to their respective programs by dragging them to the pinned program then clicking the pushpin. To open them, right-click the pinned program and select the one you want, e.g. a spreadsheet.

To unpin an item, point to it, click the pushpin icon and then click Unpin from this list. To remove an item from the list, right-click it and then click Remove from this list.

To see which applications are actually running, press Ctrl+Shift and Esc to start the Task Manager.

_____________________

Three attempted methods I followed to resolve:

Method 1: Update your display driver.

  1. Type device manager on Search/Cortana and select Device Manager from the results.
  2. Once the Device Manager is open, expand the Display adapters section, then right-click the listed adapter and select Update Driver Software...
  3. Select Search automatically for updated driver software.
  4. The tool will let you know if a driver update needs to be installed or if the display driver is up to date.
  5. If the driver is updated, please proceed to Method 2.

Method 2: Uninstall and reinstall the display driver.

  1. Type device manager on Search/Cortana and select Device Manager from the results.
  2. On the Device Manager, expand the Display adapters section, then right-click the listed adapter and select Uninstall.
  3. Do not check the Delete the driver software for this device.
  4. Click OK.
  5. The system will prompt you to restart the device, click on Yes.
  6. Once restarted, the system will automatically reinstall the display driver. Once reinstalled, check if the flickering persists.

Method 3: Disable Hyper-V.

If updating or reinstalling the display driver did not work, the issue may be caused by the Hyper-V being enabled. You can disable it by following these steps:

  1. Type Windows features on Search/Cortana.
  2. Select Turn off Windows features on or off from the results.
  3. Uncheck Hyper-V and click OK.
