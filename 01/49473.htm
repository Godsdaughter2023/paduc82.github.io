canadian tech say:

Please, take it from me, This is the wrong conclusion. 

I look after 160 Windows 7 computers and have been for 13 years now.  I have done this many, many times.  It always results in a dramatically better performing computer.  It takes a lot of elapsed time, but the difference will be very gratifying.

You can use any Windows 7 disk that matches the type of your activation code.  For example Windows 7 Home Premium 64 bit.   I almost always use a Dell Windows 7 re-install disk, no matter the brand of computer.  A version that has SP1 will save you a lot of time.  In many cases, the activation process will be automatic.  If not, type in the activation code on the sticker.  If it is not accepted, call Microsoft and ask for their help in activation.  They will do that.

There is a benefit to not using the OEMs restore disk in that it does not load all the bloat stuff that serves to slow your computer.

This is not that difficult, in fact you may be amazed at how easy it is. 

In most cases, the Windows 7 install process will install all the drivers you need, and if you set Windows Update to get other Microsoft Updates, Windows Update itself will update your drivers and even provide drivers that were not installed by the Windows 7 install process itself. 

There is a chance that you will need to get a driver from the computer manufacturer, but that is usually not that hard to find.  Worst case scenario is when the network driver is not installed automatically by Windows 7 install.  In that case, you will need another computer to get the driver.  You could play it safe and download that driver before you begin the re-build, just in case.  Without that driver, you can not get online to get Windows Updates.

Use the computer manufacturer's web site to get these drivers, not the plethora of driver download sites.  It has been my experience that the drivers provided by Windows and Windows update are better than any other source.

If you need more assistance in this process, come back to this thread and ask.  I will be happy to help.

Do yourself a service and do a lot of diagnostics work on the hardware of your PC before you invest this time. Use the Windows 7 memory tester.  Run the drive tester software provided by the hard drive manufacturer.
