spiritx say:

Hi,

As you said you have 3 licenses for Windows 7. If you upgrade any of those using

the free upgrade currently available you still only have 3 licenses. So you can have
Windows 7 or the free Windows 10 upgrade on a total of just 3 computers. If you
wish to use Windows 7 or Windows 10 on a 4th computer then you would need
another license.

The Windows 10 free upgrades are only extensions of the Windows 7 licenses
and not 3 new licenses.
