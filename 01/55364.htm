Hi Kendell,

Before removing an account from your device you should be sure that you set up or have different signing option and all date from this profile is saved.

If you have a user account on your PC that is not being used, you can permanently remove it by deleting it.

! You must be signed in as an administrator to be able to delete a user account.

! You will not be able to delete a user account that is currently signed in. You will need to sign out the user first.

! Be sure to always have at least one enabled administrator account on the PC to avoid not being able to perform actions that require administrator rights.

1) Remove a User Account in Settings
- Open Settings, and click/tap on the Accounts icon.
- Click/tap on Family & other people on the left side, click/tap on an account you want to delete on the right side, and click/tap on Remove.
- Click/tap on Delete account and data to confirm.
- When finished, you can close Settings if you like.

2) Delete a User Account in Control Panel
- Open the Control Panel, and click/tap on the User Accounts icon.
- Click/tap on the Manage another account link.
- If prompted by UAC, click/tap on Yes.
- Click/tap on a user you want to delete.
- Click/tap on the Delete the account link.
- Click/tap on Delete Files or Keep Files for what you want to do with the user account's personal files.
- Click/tap on Delete Account to confirm.
- When finished, you can close the Control Panel if you like.

3) Delete a User Account in Local Users and Groups
- Open Local Users and Groups, and click/tap on the Users folder in the left pane to open it.
- Click/tap on a user you want to delete in the middle pane to select it, click/tap on More Actions under the same user's name in the right "Actions" pane, and click/tap on Delete.
- Click/tap on Yes to confirm.
- When finished, you can close the Local Users and Groups if you like.

4) Remove a User Account in Netplwiz
- Type netplwiz in the search box in your Start menu or on your taskbar, and press Enter.
- If prompted by UAC, click/tap on Yes.
- Check the Users must enter a user name and password to use this computer box.
- Click/tap on a user you want to delete, and click/tap on Remove.
- Click/tap on Yes to confirm.
- If you set Windows to automatically sign in to a user account at startup, then you will need to set this again when finished.
- When finished, you can close Netplwiz if you like.

5) Delete a User Account in Command Prompt
- Open an elevated command prompt.
- In the elevated command prompt, copy and paste the command below

net users

and press Enter to see a list of all user account names on the PC. Make note of the account name you want to delete.
- In the elevated command prompt, type the command below

net user "UserName" /delete

press Enter
- When finished, you can close the elevated command prompt if you like.

6) Delete a User Account in PowerShell
- Open an elevated PowerShell.
- In the elevated PowerShell, copy and paste the command below

Get-LocalUser

and press Enter to see a list of all user account names on the PC. Make note of the account name you want to delete.and press Enter to see a list of all user account names on the PC. Make note of the account name you want to delete.
- Type the command below

Remove-LocalUser -Name "UserName"

into the elevated PowerShell, and press Enter.
- When finished, you can close the elevated PowerShell if you like.

Let me know if you need more help
