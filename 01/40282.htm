3. NAS - 10.0.0.61

frederik long say:

You invoke mstsc.exe with three separate port numbers, e.g.

  • 50179
  • 50180
  • 50181

I would not use port 3389. Every hacker tries this port while for a potential target.

You must then create port forwarding rules through your router, one for each port.

Each PC must have a fixed IP address that you use for the above rules.

On each PC you must set the correct port number in this location:

HKLM\SYSTEM\CurrentControlSet\Control\Terminal Server\WinStations\RDP-Tcp/Port Number.

Alternatively you could set up a VPN. In this case you can reach each PC by its NetBIOS name.

Each motherboard has its own set of different settings, below is a general guide of that changes need to be made . . .

Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot

archangle11 say:

hello peter,

 I understand that you need clarification about the difference between media creation tool and USB recovery drive.

A USB recovery drive stores  recovery data onto it . By recovery data I mean to say the recovery system Image of the current operating system along with your backup apps, data etc.(only if you choose this option while creating, because it will ask for more stotage space) like they were at a previous point of time. so that when system goes into problems you can restore it onto a previous image.  I am sure You must be well aware of the backup and restore function in windows that creates a system image and backup the recovery partition on PC i.e. 16 GB (normally). And as per your query a recovery USB drive can be used to refresh or restore your PC based on a system image of the current operating system only if you have created the USB recovery drive with this option. For more information you can visit this link create a USB recovery drive.

And coming to USB installation media created using windows media creation tool, It is a clean installation media and does what a clean installation media does. It installs windows 10 from scratch.

COMPARISION (which is better):

If you want to have faster PC that works as good as new, well then media creation tool is the best option as it installs clean OS on to the device.

If you are facing troubles with your current system and want to restore a previous point of system image on to your system without loosing your data then system restore is the option. But restoring a system using system image DOES NOT guarantee a faster or much better system than a clean install .

By the way, If you want to install windows 10 on a new SSD or HDD after free upgrade process you can do so as long as you are doing it on the same PC (mother board). windows recognizes your mother board ID to verify that you are eligible for your copy of windows. If you do it on another computer or if you change mother board then the license won't activate . changing hard drive should be OK though too many hard ware changes are not advisable. so you won't run into trouble by doing a clean install on the same PC that you upgraded to windows 10 for free. Just keep in mind that when clean installing, you will be asked twice to enter Product key or license. First when on initial start up choose "SKIP THIS" option and when asked after installation but just before opening choose "DO THIS LATER" option.

P.S : having a Microsoft account connected to your current free copy of windows 10 makes your OS validity more safe and secure
