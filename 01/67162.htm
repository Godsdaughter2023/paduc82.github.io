cmdrkeene say:
That's fine.  Memory is there to be used.  If memory begins to be used too much, Windows will automatically take care of it for you by caching unused data.  If your memory usage was zero (or as close to zero as possible), your computer would run VERY slow while it reads everything from the disk (which is the absolute slowest part of any computer).  You want things to be placed into memory instead.

So much so, in fact, that Windows actually fills up the memory automatically with stuff you might use, just so it's faster when you do.  This is called SuperFetch.  Windows learns which programs you use at specific times, and loads the libraries into RAM before hand.

Don't worry about performance: like I said, if memory is low, it will clear the cache and concentrate on your actual running programs instead.

Solution 2. Reset the game from the Settings menu. This process can help restore corrupted game installation files.     

  1. Press the Windows key on your keyboard, and then click the Settings icon.    
  2. Select System, and then select Apps & features.     
  3. Find the game, and then click on Advanced options.    
  4. Click Reset.    
  5. Restart your computer for the changes to take effect.    

Solution 3. Remove temporary files.    

Temporary files are automatically deleted after exiting an application or a game. However, restarting your computer without exiting an active Windows session won't delete temporary files. These files may cause issues running games on Windows 10.      

  1. Press the Windows key on your keyboard, and then click the Settings icon.    
  2. Select System, and then select Storage.     
  3. Select This PC and click on Temporary files.     
  4. Select Temporary files.     
  5. Click Remove files.     
  6. Restart your computer for the changes to take effect.   

Solution 4. If the issue persists, rollback your graphics card drivers to the previous version.

  1. Press the Windows key on your keyboard, and then type Device Manager in the search box.  
  2. Hit Enter.  
  3. Right-click the device for which you want to roll back to the previous version of the driver, and then click Properties.  
  4. Click the Drivers tab.  
  5. Click Roll Back Driver.  
  6. In the Driver Package rollback dialog box, click Yes.

Your best option is to purchase a genuine Windows 7 license, because it looks like your friend has provided illegal and non-genuine means of installing Windows on your system.

Because Windows 7 is in limited stock because Microsoft ended sales in October 2013, this has created demand because of its scarcity. This means retail boxed copies demand a premium, while OEM System Builder copies remain affordable.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

There is nothing wrong with using it too, especially since all future releases of Windows will be full versions anyway.

You can find OEM System Builder software from dozens of online merchants. The current price for OEM Windows 7 Professional at cost only

There are no technical limitations to prevent you from using OEM software on your own PC, although this software will work only for a clean installation and not for an upgrade.
