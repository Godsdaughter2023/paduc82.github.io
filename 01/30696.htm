There is an established procedure for importing; you point the wizard at the folder containing the folders you want to import and let it get on with it. The error message you saw will arise if you point the wizard at a folder containing messages.

I would Work offline, disconnect from the network and disable anti-virus before doing this to avoid unnecessary interference.
