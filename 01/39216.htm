I am not a developer. Do I need Net Framework - Microsoft Community

donnahadley say:

I am not a developer. Do I need Net Framework

I am running out of disk space.  Net Framework was installed several years ago and I cannot determine if anything else I use needs this program.
lemp say:
The various versions of the .NET Framework (and no, they are not "versions" in the sense that you can necessarily get rid of the earlier ones) are installed when you install an application that relies on them.  If you uninstall any of the .NET Frameworks, whatever application relies on it will stop working and complain the next time you try to use that application.  Of course, you may have uninstalled the application, so you won't know if you have a problem until you actually have a problem.

In any case, here is my canned answer about low disk space:

There are a number of things you can do to -- temporarily -- free up disk space.  The only real solution, however, is to get a larger internal hard drive.   You can move data to an external drive, but if it is data that you use frequently, the slower access times may annoy you.  Therefore, I don't recommend that you move your My Documents folder in its entirety to an external drive, although you certainly could do that by following the linked instructions. 

  • You can run Disk Cleanup (Start > Run > cleanmgr > OK)
  • You can reduce the size allocated to System Restore to about 1 GB (right-click My Computer > Properties > System Restore >Settings)
  • You can turn off Hibernation (if you don't use it) (right-click empty spot on desktop > Properties > Screen Saver > Power > Hibernate)
  • You can disable Disk Indexing, which will also speed up your computer a bit (http://lifehacker.com/031440/turn-off-indexing-and-speed-up-windows-xp)
  • You can reduce the size of your Internet browser cache (varies depending on which browser you use)
  • You can remove most of the $NTUninstallKBxxxxxx$ folders following the directions here:  http://windowsxp.mvps.org/Hotfix_backup.htm  (read the caveats in the grey box first)
  • Also see (which repeat some of the above):

«     How to reclaim disk space on Windows XP and Windows Server 2008-based computers?

«     Forty Ways To Free Up Disk Space


But none of the above will really gain you significant amounts of space for very long.  The only long term solution is a bigger hard drive.

In the meantime, download and run JDiskReport, which will graphically show you what files are taking up the most space and thus suggest what files you may delete or move to an external drive.  If you want to move an application, you'll have to uninstall it and then reinstall it specifying your external drive as the install location.  You can't simply move the files.
