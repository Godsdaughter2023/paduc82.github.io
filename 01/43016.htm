tryx3 say:

Go to the Start menu search box, enter management then right-click on the Computer management shortcut that is offered to you & select Run as Admin.  When the dialog opens [it takes several seconds], select Disk management.

 

Be careful to identify the correct disk as your remedial steps would mess up your PC if you accidentally identified the wrong disk to work on.  The misbehaving USB probably has no drive letter assigned but special partitions inside the PC may also not need them.  Use the info in the list of drives & in the illustrations below them to be quite sure you are selecting the correct drive.

 

Does the Status column of the list show that it is formatted [probably NTFS or FAT32]?  Is it's size shown correctly?  If so then right-click on the drive's entry in the list & select Change drive letter & paths.  Give it a permanent drive letter [this will continue to apply after reboots & when you plug it in next].  There will be a warning that any processes using the drive will be disrupted by the "change" of drive letter.

 

If this does not fix it & normal procedures will not then format the drive, return to this dialog & format it there.

 

If that does not fix it then you need to examine the disk with a Partition Manager.  There are several free ones available but now is probably not the time to sit down & learn something new.  If none of your friends or colleagues are familiar with these utilities then try the free versions of EaseUS® Partition Master or Partition Manager 12 Free Edition [Paragon seem to have become obsessed with Facebook - the software is also available for download at CNET.
