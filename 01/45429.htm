cmdrkeene say:

If you are trying to access the "Documents and Settings" folder that is part of your Windows 7 installation: that folder doesn't actually exist.  It's called a junction or a 'hard link' that points to the real folder called "Users".  However, I don't think that's what you're talking about because you mention being able to access other specific users folders in a backup from your previous Windows installation.

This leads me to believe that the user account for whom's folder your trying to access was encrypted, an option users can choose when creating accounts in Windows XP.  If this was the case, the contents of the folder cannot be retrieved (to my knowledge there is no way to break NTFS encryption).
