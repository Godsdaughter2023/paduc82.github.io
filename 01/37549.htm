How to find the 25-character product key on windows 8 - Microsoft Community

sodavyou say:

How to find the 25-character product key on windows 8

My computer was refreshed and I have to reactivate everything, including Microsoft office. When I bought my computer, it came with it already installed in my computer. Now, it asking me for the product key, which I don't have. How do I access my 25-character product key on my laptop?
andre da costa say:

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.

If you are looking for the product key for Microsoft Office, you will need to buy one if a trial copy came with your computer.

I've just bought a Microsoft Surface RT and Samsung ML-2165W wireless printer. While the Microsoft Surface is excellent, Samsung haven't yet provided any drivers for Windows RT so the Samsung is just an expensive paper weight!!!

 

On connecting the ML2165W to the USB port on the Surface, ML-2160 is shown in the Windows RT>PC Settings>Device dialog. However, it is not recognised as a printer as it is not shown when trying to print something from an app using, for example, the procedure to print described in http://www.microsoft.com/Surface/en-GB/support/surface-with-windows-RT/hardware-and-drivers/print-from-surface

Using the Windows RT "Add device" option results in the Samsung ML-2160 Series(192.168.3.1) being shown as "Not compatible with this version of Windows"

Using the Windows RT Advanced Printer Setup I can find a list of installed drivers for printers including Samsung class drivers for PCL6, PS and XPS and quite a long list of Samsung printers, eg CLP-320 and ML-2580 but not anything like the ML-2165. I thought the Samsung XPS Class Driver would be the best alternative and when set up in this way the printer is then available for printing from programs but nothing actually prints either from the program or when trying to print a test page. I've also tried a number of other drivers from the list with no success, in some cases after trying to print something an error is then shown in the device list by the printer.

 

During these experiments I have, of course, tried restarting the computer and turning off and reconnecting the printer, etc.

I've exchanged a number of emails with Samsung's support but the latest reply I've received contains the statement:

 

"The drivers provide for this tablet come directly from Microsoft itself, so when we plug the printer into the device it will go through the download centre looking for the correct driver. If the driver cannot be found then there is no driver available. I would advise you to keep checking on the tablet to see if the driver will become available. The driver for this model will become available this model but we are unable to provide any date when it will be released."

 

If they had at least given an indicated date I would have felt a whole lot happier. Moreover the implication that it is Microsoft rather than Samsung seems rather misleading because as I understand from: http://blogs.msdn.com/b/b8/archive/2012/07/25/simplifying-printing-in-windows-8.aspx while Microsoft provide the class drivers it is the printer manufacturers who are responsible for developing drivers which are then deployed via the Windows Update service or the Windows Store.

 

I notice there is now a "jail break" availabe for Windows RT: http://forum.xda-developers.com/showthread.php?t=2092158 Does anyone know if this would provide a possible short-term fix as a means of getting the Windows 8 Samsung 2165W driver installed on Windows RT? However, I'm guessing the Windows 8 driver installation program will be an x86 .exe and won't run on the ARM chip fitted in the Microsoft Surface??

 

If anyone has any other ideas, thoughts or comments I'd be very grateful.

 

Many thanks all

Roger

davem121 say:

You do not need to worry about a product key or activation of your Windows 10

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Servers

If you do not have Windows 10 Installation Media Click HERE to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 4GB) using that tool or create the ISO file which you can later burn to DVD
