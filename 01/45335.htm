calculon2020 say:

Windows Media Player 11 Pop Windows

I have a ThinkPad tablet running Vista Business.  I use Windows Media Player as my music player.  When it's docked in the taskbar and it goes to the next song, a pop or OSD if you will pops up telling you what the next song is.  I don't like this.  Is there a way to get rid of it?  Thanks for any help.

tim de baets say: Open the Start Menu, type  regedit in the Search field, and press Enter. In Regedit, navigate to HKEY_CURRENT_USER\Software\Microsoft\MediaPlayer\Preferences , create a DWORD value named  DeskbandFlyoutTimeout , and set it to 0x0 . That should prevent the 'toast' from popping up again.
