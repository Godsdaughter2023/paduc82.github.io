carey frisch say:
Computer manufacturers elected to provide the Windows 7 Upgrade disc for the same operating system architechure as the one originally shipped with the computer.  For example, if your Fijitsu computer originally came with Windows Vista 32-bit preinstalled, Fujitsu will only provide you with a Windows 7 Upgrade 32-bit DVD.  If you want a Windows 7 64-bit installation DVD, then you'll have to purchase one.  Example: Microsoft Windows 7 Home Premium 64-bit 1-Pack for System Builders - OEM

Please bear in mind that one cannot perform a direct upgrade from a 32-bit O/S to a 64-bit O/S.  A "clean install" is required.
