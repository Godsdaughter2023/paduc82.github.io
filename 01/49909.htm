lemp say:

A large capacity USB flash drive is actually the easiest method and is actually not very laborious unless you have so much data that you have to reuse the flash drive multiple times.

If you have a router (and many people have routers even with only a single computer), the easiest method is to connect both computers to the router and configure file and printer sharing.

If you don't have a router, you can use an Ethernet cable.  In the past, you had to be sure to get a special "cross-over" Ethernet cable to avoid burning out the network adapters, but most modern computers have "auto-sense" network adapters that can be connected using a regular Ethernet cable.  Check the specs for your two computers.  Here are some directions:  http://www.wikihow.com/Connect-Two-Computers-Together-with-an-Ethernet-Cable

Yes there is a cable to do this directly, but you will have to buy it and it has no other uses.  External storage is easier.
