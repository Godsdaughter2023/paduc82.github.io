ken blake - microsoft mvp say:

On Tue, 6 Mar 2012 12:11:39 +0000, GrahamDurrant wrote:

What does DNS not responding mean and how can i rectify the situation please

We all tend to think of a web site's URL as its address, but that's
actually not correct. The real address of a web site is its IP
address. But because a URL is much easier to remember than an IP
address, we normally use URLs instead. When you type a URL into your
browser, it goes to a computer called a DNS (Domain Name System)
server to get the equivalent IP address for it and then uses that IP
address to get to the web site.
So, for example if you type the URL www.microsoft.com it goes to the
DNS server and finds the IP address 207.46.19.254 which it uses to go
to the Microsoft site,

The DNS server you use is normally one provided by your ISP. If you
are having a problem with it not responding, you should contact your
ISP to find out why.

Ken Blake, Microsoft
