xircal say:

Good. But disabling UAC means you won't get a warning when you try to install something.

As a precaution, right click whatever it is you want to install and choose to scan it with your AV in future. Keepass is OK though and I use that myself (saw it in your screenshot).

It's also a good idea to create a manual System Restore point before installing anything. That allows you to wind back the clock to the time prior to installing it if the app causes system instability. To create a manual Restore point, do the following.

  1. Hit Windows key + X and then click System.
  2. Click the link on the left to System Protection and then click the "Create" button at the bottom of the menu.
  3. Give it a name, something like, "Before installing program X" and then click OK.

Windows will subsequently create a Restore point which takes a couple of minutes.

If you want to run System Restore, repeat steps 1 & 2, but instead of clicking the "Create" button, click "System Restore" button and then click Next. Your manual Restore point will appear in the list with whatever name you gave it. Select that. click Next and follow the prompts.

When System Restore runs, the computer will reboot and for a time all you'll see is a plain black screen where nothing appears to happening. But eventually, the usual lock screen will appear and you'll be able to login again.
