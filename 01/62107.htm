robinblackdiamond say: Your system appears to have been joined to a domain.  After this is done, it requires a reboot to take effect.  So once you rebooted is when it prompted you to logon to the domain. 

There should still be local accounts on your machine that you can use to logon, however they'll have a password for all accounts if your system is joined to a domain (as it's a requirement).  To logon using a local account, pick your machine name from the "Logon To:" drop-down at the logon prompt.  You'll need to know a valid user name and password for this computer.  If you or someone you know has an account in the domain that your system is joined to, they can help you logon to the domain by providing valid domain user name and password.

Once logged on you can disjoin your computer from the domain (Right-click Computer from start menu, properties, Advanced System Settings, Computer Name, Change, select workgroup and type work group name.  Click Ok.)  You'll need permissions to perform this task.

To be able to make customized settings with date and time, you need to disable Windows time service and unsynchronized the internet time.

Lets follow the below methods and see if this helps. 

Method 1 : Disable Windows time service. 

  1. Press Win key + R key and type services.msc in the run command.

  2. In services window select “Windows time”.

  3. Right click on the service and from the drop down menu select stop and close the Window. 

Method 2 : Unsync internet time . 

  1. Press Win key + R key and type timedate.cpl in the run the run command

     and press enter.

  2. In date and time window click on Internet time tab.

  3. Click on the change settings.

  4. Here Uncheck the box next to synchronize with a internet time server.

  5. Click on ok and than apply and ok in the later windows.
