xbr dionysus say:

DMZ

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin or a custom password was created during setup.

DMZ

 

  1. Open a web browser on a computer.
  2. Type the address 192.168.1.1 into the web address bar.
  3. When prompted for the username type in admin, use the password created during the setup process. If you did not setup one then the use the default password admin.
  4. Once logged in click on the Applications & Gaming tab on the top of the page.
  5. Click on the DMZ option in the subcategory.
  6. Click the Enable option.
  7. Under the Destination section select the MAC Address option.
  8. You can type in the Xbox’s MAC address and then click on Save Settings.
  9. OR click on the DHCP Client Table.
  10. Select the Xbox; it usually does not have a client name.
  11. Click the Save Settings button.

 

 

Disable DMZ

 

  1. Open a web browser on a computer.
  2. Type the address 192.168.1.1 into the web address bar.
  3. When prompted for the username type in admin, use the password created during the setup process. If you did not setup one then the use the default password admin.
  4. Once logged in click on the Applications & Gaming tab on the top of the page.
  5. Click on the DMZ option in the subcategory.
  6. Click the Disable option.
  7. Click the Save Settings button.
