Hi, BALAJISETHURAMAN.

The suggestion that Perumal Kumar gave, to add Tamil to your language settings, actually does more: in addition to enabling a Tamil keyboard, it also install certain optional language-enablement features for Tamil.

One of these optional features is the Tamil Supplemental Fonts feature. This optional feature includes the Latha and Vijaya fonts. In the past, they would have been included in every Windows system, whether users actually used Tamil or not. In Windows 10, they were moved into an optional feature so that users that don't need or want these fonts don't have them taking up resources on their system.

This had a side effect for some existing apps, such as Chrome: it has custom font handling that expected these fonts to be present. Without them, your symptoms will result.

The optional feature is associated with language settings, and in an upgrade from Windows 8/8.1, existing language settings will be detected to apply the optional feature during the upgrade. Unfortunately, this isn't working for upgrades from Windows 7.
