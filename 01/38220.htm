  • In Command prompt type sfc/scannow and press enter.

  • Double-click on NoLowDiskSpaceChecks, enter 1 for the value

Which version of Operating System are you using?

This error prevents iTunes from launching. The library file was modified to work with the newer iTunes version, so it is no longer compatible with the previous version. The good news, however, is that Apple does backup the iTunes library file into a subdirectory that you can use to revert back to the previous version.


Method 1:
You may try the steps to recover a previous version of the iTunes Library.itl file.
a. Open up the My Documents\My Music\ (iTunes) folder.
b. We will first backup the current version. Rename iTunes Library.itl to something like iTunes Library.itl.bak.
c. Go into the My Documents\My Music\Previous iTunes Libraries folder.
d. You may see several files that start with iTunes Library and have a date included in the name. Copy the newest file in the folder.
f. Go back to the My Documents\My Music and paste the file from the previous step.(The folder in Step 1)
g. Rename the new copied file to iTunes Library.itl.
The library should now be restored to the previous version of iTunes, and the issue gets resolved.

Method 2:
If the issue persists then you may also uninstall and reinstall iTunes and check if it helps:
Removing and reinstalling iTunes, QuickTime, and other software components for Windows Vista or Windows 7
http://support.apple.com/kb/HT1923
