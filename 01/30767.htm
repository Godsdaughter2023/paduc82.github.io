You can try the following steps to start your PC from USB.

  • Start the machine and hit F2 as soon as you see manufacturer's logo to get to BIOS.
  • Disable Secure boot in the boot options screen
  • Enable load legacy option ROM
  • Keep boot list option set to UEFI

  • F12 on boot to get legacy boot option
  • Pick USB from the list and it should start from USB

greg carmack - mvp 2010-2020 say:

The   Boot Priority Order in BIOS or the modern UEFI firmware should not used to boot a flash stick as  you'll be caught in a never-ending loop of starting install over. Instead use the Dell F12 boot menu key to trigger the stick.

First you must format the stick to boot as a UEFI device using the Media Creation Tool for either WIndows 8.1 or 10 from Create installation media for Windows 8.1  or Create Windows 10 Installation Media.  If these are not what you want to boot media for, then please tell us what it is as should have been done in the OP.

Once you have UEFI-capable boot media, then power up with stick inserted while pressing the Dell F12 BIOS Boot Menu Key, choose the UEFI flash stick option only.  If the F12 menu doesn't appear it may be locked down by the UEFI firmware so then follow How To Access Advanced Startup Options in Windows 8 or 10 to choose Use A Device:



Choose Custom install, delete all partitions down to Unallocated Space, click next to begin install. 

 I would not enable boot the stick as a legacy drive as you'll lose all modern features of the UEFI replacement for BIOS'.  This was mainly necessary when UEFI was new and often could not install Windows, but now its firmware has evolved beyond this. In addition, you'll always need to boot Windows Repair or Recovery media as a UEFI device if you already have a UEFI install, proven by the appearance in the drive list of an EFI System partition or when you Search MSINFO for System Information.
