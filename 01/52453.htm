hairyfool say:

OEM licensing always has had that restriction it that it is not intended for the end user to build their own computer but for a system manufacturer to install prior to selling the computer. This is because the manufacturer has the responsibility to provide support, not Microsoft. If you build it yourself using this license you have no support entitlement.

However computer component suppliers have been selling OEM licenses to home builders for some time and although never officially sanctioned by Microsoft they have not actively prevented the practice.

In part this is because if you purchased a computer with Windows pre-installed you have the right to reload the computer from scratch at any time even before 1st use, this makes it impossible to identify who actually installed Windows on the computer in the 1st place.

There is no guarantee that this clause won't be enforced more strictly in the future but at the moment the same situation applies as with earlier versions of Windows. There was a slight change when Windows 8 was released but Microsoft reverted back to the original licensing model with 8.1
