a. user say:

Every router comes with a way to be reset to its defaults. The manufacturer of your router will have that information on its website. Typically, you'll do something like pushing in a tiny button on the router with the end of a paper clip and counting to 10 (or saying The Lord's Prayer if you're religious.)

Resetting your router, among other things, removes the passwords from your network and your router's administration console. Now you can setup your network again from scratch.

Resetting a router does not uninstall the router's firmware, so you won't need to re-install any firmware.

Tip: The administration console of a router includes a way to backup all the router's settings to your computer. In the event that you have to reset your router, after the reset is done you can easily restore your backed up settings. That's faster than setting up everything from scratch.

Since you see the picture as a big X, this can be caused by insufficient codecs on your computer. To troubleshoot the issue, please follow the following steps below:

1. Ensure you have installed the latest version of Windows Movie Maker

Note: Please read the system requirements for Windows Movie Maker 2012 before installing this version.

If you didn’t meet the system requirements, please install Windows Movie Maker 2011 instead.

2. Try making movies of a shorter duration - If you have a large movie project, splitting the video into smaller projects may help.

 

3. Check that the files you are using are of a supported format – Some file formats are not supported by Windows Live Movie Maker. A list of supported formats can be found here: Movie Maker file types

 

4. Ensure that you have the proper codecs to play your videos and audio files – audio and video require codecs to play the file and it’s important to have the right codecs installed or updated in your PC.

  - Try running the Microsoft FixIt tool for codecs: FixIT

  -  Media info will be able to identify any issue with missing of faulty codecs and whether it's already in your computer or not: Media Info

 

Note: If you can't find the codec you need try converting the video to a supported format (eg. wmv) and then add it to your Windows Live Movie Maker project.

 

5. Ensure that your video card drivers are up to date - Movie Maker makes extensive use of the video card GPU (graphics processing unit). The display driver affects the reliability, performance, and compatibility of the video card with different software.

To determine the version and video card driver date, run the DirectX Diagnostic Tool (dxdiag):

 

DirectX Diagnostic Tool

1. Click on Start button and type “dxdiag” in the Search programs and files box and press Enter key on the keyboard.

2. Click on the Display tab and you will see the driver details including the driver version.

3. If your video card driver seems outdated please go to the video card driver manufacturer and see if a newer version is available.

Example:

http://support.amd.com/us/gpudownload/Pages/index.aspx

Wuauclt.exe is the AutoUpdate Client of Windows Update and is used to check for available updates (for the various versions of the MS Windows platform) from Microsoft Update. The wuauclt.exe file is included in the Task Manager’s list of active processes when it is waiting for a response or an action to be performed by the user.

Aside from updating a user’s MS Windows-based system, the wuauclt.exe file also looks for MS Windows-based software and hardware updates. These updates can help a user avoid stability issues and even security risks or system vulnerabilities.
