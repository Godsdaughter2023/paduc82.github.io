How to increase the VRAM on a HP Envy m6 - Microsoft Community

jacobsawyer_24 say:

How to increase the VRAM on a HP Envy m6

How do u increase the vram of a HP Envy?

My processor is a AMD FX-8800P Radeon R7, 12 Compute Cores 4C+8G 2.10 ghz with a 64 bit operating system. Any ideas??? This is for gaming purposes btw. 

bossdweeb say:

HP Envy is too generic a term. You need to use the full model number.

When you find the full model number go to the HP web site where the specs for your model

will show what the maximum RAM for that machine is.

- If the machine has two empty slots you could add two more sticks of the exact make and model RAM

that is now in the machine (assuming 8GB is not the limit).

If it has only two RAM slots you will need to buy a matched kit of 2 RAM sticks ( 2X8GB ) that have exactly the

same specifications ( speed, latency, voltage ) as the current RAM.

I suggest you monitor the RAM use under full gaming load before splashing out on RAM you may never use.

You are very unlikely to require more than 8GB RAM.

I am yet to use much more than 4GB on any game with a 1920x1080 monitor resolution, even

with max settings (GTX980 ).[ Watch Dogs on ultra w/ 4xTXAA = just over 5GB - FC4 and BF4 = just over 4GB ]

- Of course some people like to have a large number of browser windows with lots of images, and a bunch of

32 bit video and system ram - Microsoft Community

pete_826 say:

32 bit video and system ram

I know that an 32 bit OS can handle only up to 4gb, but I heard that this amount includes video memory as well. As in it can handle a total of 4 gb of ram, whether its from the system ram or video ram?
j w stuart say:

The 32Bit address space (4GB) includes any memory your graphics card has plus some additional hardware resources. So a graphics card with 512MB of memory would leave you with about 3.2GB of usuable memory on a computer with 4GB of DRAM installed.

 

In round numbers XP, Vista and Windows 7 32-Bit Versions of the Operating System can only address 4GB. Does not matter how much memory you have installed 4GB is the max. From that max your video card has XXXMB of ram on the card so that limits the operating system to 4GB minus the cards XXXMB of memory.

In addition other hardware also takes away additional amounts of memory (usually a small when compared to the video card) and the remaining memory (address space) is available for Windows XP, Vista or Windows 7 OS and your applications.

So that 4th GB thru 16 GB of ram you installed or will install goes unused on a 32-Bit OS. Does not matter what the BIOS reports or what the OS shows as installed memory it still limited to that 4GB address barrier.

Memory Inc. provides a good clear explanation of how a video card effects
the amount of available memory in your PC.
