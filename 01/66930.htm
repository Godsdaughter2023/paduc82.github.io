andre da costa say:

Previously called Digital Entitlement, when a computer is upgraded from a previous version of Windows; it receives a unique signature of the computer, which is stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license your system was previously running is exchanged for a diagnostics key.

With Digital License, there is no need to know your product key or purchase another product key if you lose it. Anytime you need to reinstall Windows 10 on the same computer, just proceed to do so using any of the available options. Windows 10 will automatically reactivate without asking questions. The only exceptions where the license will be invalidated is dependent on the product channel. If you were previously running an OEM Windows 10, Windows 7 or Windows 8 license; then changing the motherboard will cause the Digital License to no longer work. This will require purchasing a new license. Transferring licenses upgraded from Windows 8 or Windows 7 under the Windows 10 free upgrade is also not allowed under Digital License.

bossdweeb say: [ EDIT: For the short version read  Balaji's  comments below - ]

There is no need to 'revert' to DX 10 or uninstall DX11 (which cannot be done in any event) because of perceived  DX 11 issues.
Win 7 installs with three versions of DX - 9, 10 and 11.
You do not choose which version to use, your software does that (or offers an option is some cases, ie. - games).
You have mentioned your 9800GTX gpu cannot use DX 11 . True. It can use DX9 and DX10, but what is the issue you are facing regarding Direct X ?
What is, or is not happening ? Does the problem occur with a particular software/game ?
I assume you are getting an error message regarding DX.  What is that error message exactly ?
There are maybe a handful of games that do not use DX9, but I am unaware of any that won't run on DX10, even when they have a DX11 option.

There are two options for DX issues. The first, as you apparently already know, is to use the
Direct X Web installer. This will update DX, but will not repair any corrupt DX files.
The message - "a more up to date version of DirectX is already installed on the PC."
would indicate just what it says, but does not take into account the possibility of corrupted
DX files.
The Web Installer is most useful with new Win7 installs as there have only been a couple of
new files since the release of Win7, and Win7 also installs with a couple of missing DX9 files,
so an update straight after installing is important.

The second option is the Direct X Redistributable from June 2010.
This installer will overwrite your Direct X files thereby fixing any corrupt files (except for the couple of DX10 and 11 files that were released after June 2010, but if these are deleted from System32 and SysWOW64  the Web installer will replace them with new files).

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

However, you can also get error messages that mention DX when there is something wrong with your gpu, as it cannot process DX properly if it is having problems. Like many error messages, one that points toward DX can be misleading in this regard.

Yes, there would be hundreds of threads on the topic. Some because most people start a new thread without reading older ones, and some because of hardware issues.

EDIT :
The following is a list of the Direct X .dll's you will have when the files are up to date (last time I looked).
Go to the Windows / System 32 folder, (and They are in alphabetical order and will start with d3dx9 - 24 > 43 . Then d3dx10 - 33 > 43 & finally d3dx11 - 42 > 43.
There should also be - d3d9, d3d10 and d3d11 these come before the D3Dcompilers - 33>43.There are also three new XInput1 - 1>3.
There are more DX files, but these are most of the graphics related Direct X  .dll's
