ang3l3yes say: Hi Choggner,

When it comes to flashing the BIOS you need to make sure you know what your doing or you could end up with a totally dead motherboard, you may want to try googling something like "BIOS Flashing guide" and reading a few articles first so that you have it all down, and maybe try the HP forums in case there's any proprietary methods they employ.

With that being said, its a fairly straight forward process usually, though OEM Manufacturers (Acer, Dell, HP etc) tend to make things a bit more difficult as they are usually pretty slow in releasing updated BIOS'.

Step-by-step for BIOS flashing varies from manufactuer to manufacturer..  some will have a built in BIOS Flashing tool in the BIOS itself, which makes things easy, others use an online BIOS update method (which I don't really trust myself) others require a Floppy Disk Drive, and flashing utilities ( i.e AWDFLASH) which are run from DOS, needing a startup diskette to boot to DOS first before then running the AWDFALSH utility.

I've avoided buying OEM PC's for some years now since I had difficulty in getting a motherboard BIOS update when I wanted to upgrade my CPU, and it needed a BIOS update to recognise it, which took a lot of digging to find out the origins of the motherboard used by (Acer in my case).  These OEM manufacturers usually use a branded motherbaord which they then give a model no. of their own, but it is possible sometimes to identify the original manufacturer of the motherboard and obtain a BIOS update direct from the original motherboard manufacturer.

However if HP have provided an updated BIOS then you should be able to avoid those steps.

From your post above it sounds as if one of HP's methods for updating the BIOS is done using an online BIOS update tool, I don't like these personally as they can often prove to be unreliable.  However its usually possible to obtain the BIOS file and Flash it manually.

THe first thing you need to do is check to see what version BIOS your currently using, an easy method of doing this is to download CPU-Z and open it up and click on the mainboard tab, this will display your BIOS version, another method is to watch carefully when booting the PC to read the BIOS version number on the POST pages.  Though sometimes these aren't displayed as many OEM PC's display a branding page during POST, though this can usually be disabled in the BIOS.

Once you establish what your current BIOS version is, next go to the HP website and check to see what the latest available BIOS version for your PC is and see if there is actually a newer version.

If so try to download the BIOS file, and save it to a USB stick or floppy disk if your PC has a floppy drive. You may need to extract the file first if its a .zip or .rar file.

If you have a BIOS update tool incorporated in your BIOS, then you can just re-boot the PC to the BIOS, with the USB stick plugged in, entering the BIOS (Usually by pressing the DEL key repeatedly during POST)
Navigate to the BIOS update utility in the BIOS and starting it.
Once the BIOS Update utility is running it should give you a screen with details of your exisitng BIOS and a section to choose a new BIOS file. (this would be the file saved on the UBS stick)
You may need to switch between available drives before it will find the drive with the new BIOS file, usually using the TAB key to tab between the available drives, you will have a split section of 2 panes, 1 with a list of drives on one side and the available directories for the selected drive in the other.
TAB through the available drives until you see the BIOS file in the other pane.
Once you see the BIOS file listed using the cursor keys to navigate the cursor to highlight the BIOS file and hit enter
You will then usually be prompted asking if you are sure you want to update the BIOS, hit 'Y' and the BIOS flash will start.
It's imperative that you do not switch off the PC while the BIOS is being flashed, or until you get a message that "BIOS succesfully flashed", if for any reason it says the BIOS flash failed, DO NOT TURN OFF the PC until you get a succesful flash message.
It's normal when the BIOS successfully flashed message appears for it to say that the PC will re-start within x seconds, this means that all is well, and the new BIOS has been flashed.
When the PC re-boots, I usually enter the BIOS again before letting it boot to Windows, to manually check all the settings, and enable/disable those settings that are necessary/unecessary....

Method 2
======

If there is no built in tool in the BIOS then you may need to to use a Flashing utility from DOS in which case you will need:

Startup disk to boot to DOS or a bootable USB key (a bootable USB key is simple to do, and I beleive that HP even have a utility that creates one automatically)
If you have or make a bootable USB Keyyou will need the new BIOS file saved to it
If you are using a floppy disk method, then you will need 2 floppy disks, a startup disk and a seperate disk with the awdflash utility and the BIOS file saved on it. (Depending on the size of the BIOS file you may need another disk for this alone, so 3 disks in total, the startup disk, a disk with awdflash, and a disk with the BIOS file.

OK so I'll go through the USB key method first..

  1. Boot the PC with USB Plugged in and enter the BIOS
  2. Navigate to the BOOT Priority page and change the order so that your USB Drive is the 1st BOOT Device
  3. Save and exit the BIOS at which point the PC will re-boot and should now boot to the USB Drive
  4. Provided you have configured the USB Key Drive correctly to be a bootable drive it should start loading the files necessary to boot to DOS and you will be presented with a "A:\>" prompt, or other Drive designator for your USB Key Drive i.e. "C:\> "  "D:\>"
  5. At this prompt you need to type "AWDFLASH.EXE" without the quotation marks and hit enter, this will load the flashing utility
  6. When it's loaded you will see the the screen that lists your current BIOS and below at the bottom it will give you the choices available for navigating etc...  TAB, Left, Right, Up and Down Cursor keys, Enter, F10, Esc etc....etc and their functions.
  7. Use these keys to navigate to and select the New BIOS and follow the onscreen instructions to proceed with the BIOS flash.
  8. It will usually check the BIOS file integrity first, then will delete the old BIOS and load the new BIOS
  9. Once completed you will be given a message saying so and you can then exit the program at which point you will be returned to the DOS prompt.
  10. Remove the USB key and re-start the PC and enter the BIOS again, and change the boot priority back to the original order, save and exit the BIOS and that's it your all done.
  11. The PC will re-boot and on starting, Windows (all being well) should detect the Wireless Network Adapter, i.e New Hardware Found,  and will load the necessary drivers, and you should be able to enable the wifi adapter again in the network connections screen.

The Floppy Disk Method is only slightly different.

  1. Start the PC and go to the Boot Priority Settings page, and set it so that the floppy disk is your 1st Boot Device, insert the startup floppy disk, then save and exit the BIOS, the PC will re-start.
  2. On re-starting it should boot from the floppy drive and the startup disk, booting you to a DOS prompt A:\>
  3. At this point you can now remove the startup disk and insert the disk with the AWDFLASH utility saved on
  4. Once the disk is inserted, at the A:\> prompt type AWDFLASH.EXE and hit enter to load the Flashing Utility, once it has ful;ly loaded, and you need to navigate to the new BIOS file, you can again swap floppy disks and insert the floppy with the BIOS file saved to it.
  5. Select the BIOS file and follow the same steps as above for the USB method.

Those are the main ways of manually flashing the BIOS, and IMO the safest methods.  I dislike the online updating methods but thats just a personal thing.

The online methods I've come across usually involve the BIOS file being downloaded to the PC, by the updater software, which then runs the BIOS update within Windows environment, it's possible that the first time you tried this that the incompatible message you got was just due to a corrupted download of the BIOS file, trying it again, may see it work, and it is a lot easier if your not familiar with DOS etc...

a. user say:

OMG

The blur on the lock screen is intentional. It's one of Windows 10's new design elements, called 'Acrylic.' It was implemented because so many people complained that the existing design was flat and boring. They wanted a return to something more like Windows Vista's Aero.

Microsoft knew that many of its users, perhaps most, were fine with flat and boring, because eye candy doesn't improve Windows' performance but it does place a burden of the graphics subsystem.

And so Microsoft made it possible to turn off the Acrylic feature:

Start > Settings > Personalization > Colors > Transparency effects > Off

This works. If it doesn't, then you've got a problem that's not caused by Acrylic. There's no need to edit the registry to make this happen.

Hello,

Thank you for keeping us updated on this issue.

I would suggest you to try the following methods:

Method 1:


For DNS server follow the below steps and check for any errors listed in event viewer–

1. Click Start. Type cmd in the Start Search box, and then press ENTER. 

2. At the command prompt, type the following command, and then press ENTER:
ipconfig /flushdns (there is a space between ipconfig and / ).

It will show the message successfully flushed the DNS resolver cache.

3. At the command prompt, type the following command, and then press ENTER:
ipconfig /registerdns (there is a space between ipconfig and / ).

4. At the command prompt, type the following command, and then press ENTER:

ipconfig /all (there is a space between ipconfig and /).


The ipconfig /all command displays Windows TCP/IP settings for all your network adapters.

If your IP address for your Local Area Connection is either 0.0.0.0 or 169.x.x.x (where x is any number), then your computer is not getting an IP address from the router. 

If that is the case, try this:

. ipconfig /release in the command prompt and press Enter.

· ipconfig /renew in the command prompt and press Enter.

Test it again by typing ipconfig /all to see what the address is?  (If you are on a home router, it should start with either 192.168.1.x or 192.168.0.x or 10.x.x.x).

Method 2:


If the issue persists you may refer and use the steps provided below and as mentioned in the provided link -

1. Click on Start and then Control Panel.

2. Go to Networking and sharing center and then click on Change adapter settings.

3. Right click on Local Area Connection and choose properties.

4. Highlight Internet Protocol Version 6 and click on Properties.

5. Choose Obtain DNS servers address automatically and press Ok.

6. Choose Obtain IP address automatically.

7. Repeat the steps for Internet Protocol version 4 as well.
