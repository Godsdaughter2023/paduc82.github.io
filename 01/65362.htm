Hi onesun1,

We actually did figure this one out. Apologies for not updating this ticket. I got an email notification when you added your reply, so here's our fix:

You'll need to get hold of a copy of the wme9 64-bit installer. Once downloaded, what you'll need to do is extract it using something like 7-zip and launch the unpacked  "WMEncoder64.msi" installer from there.

From there you'll be able to successfully run wme9 on windows 10 and the interface you're attempting to use *should* start behaving again.
