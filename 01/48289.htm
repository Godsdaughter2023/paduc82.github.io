Hi Ralph


1. If you are using Windows 7 Professional or ultimate, you can use the group policy editor
to remove the Shut Down, Restart, Sleep, and Hibernate commands from the Start Menu.

Note: If you enable this policy setting, the Power button and the Shut Down, Restart, Sleep, and Hibernate commands are removed from the Start menu. The Power button is also removed from the Windows Security screen, which appears when you press CTRL+ALT+DELETE. The only commands remaining on the Start Menu will be, Log Off and Switch User / Lock, on the expanded menu.

Press the WINKEY+R and type gpedit.msc in the Run Box. Press the OK button.

In the Group Policy Editor, navigate to the following location:
User Configuration / Administrative Templates / Start Menu and Taskbar.

On the right side, double click the following setting.
Remove and prevent access to the Shut Down, Restart, Sleep, and Hibernate commands

Click the Enable radio button. Apply/OK.

In the registry, navigate to the following location.

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

In the right side, right click and select New / DWORD(32-bit)Value.

Name this value NoClose  (Case sensitive)

Double click the new value and type a  1 (One) in the Value Data Box.

Apply/OK.

Exit the registry editor.

john ado say:

Hi Janvi,

The changing lock screen photos you are seeing might be the Windows Spotlight or a slideshow of your own pictures. You may double-check it by going to Settings > Personalization > Lock screen > Background. Switch it to Picture to display a single lock screen photo.

With regard to the language issue, you just have to make sure that correct region and language was set on your device. From Settings, go to Time & language > Set the appropriate time zone in Date & time, and select the appropriate country or region in Region & language. While we're at it make sure that there are no other language installed. If you are seeing languages other that your desired language, click to Remove it.
