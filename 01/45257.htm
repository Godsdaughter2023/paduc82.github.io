It's ALWAYS a good idea to backup before upgrading an operating system. That should be item #1 on everyone's checklist before upgrading an operating system. Why take a chance?

You should be backing up frequently in any case. It's easy to do and it doesn't take long. Backing up is a total no-brainer. No matter what happens to your computer, you can always restore from a backup.

thorntonderevoires say:

If memory serves, the 'generic file backup' option within Windows will back up specific file types only. I don't think it even lets you specify which files and folders to back up but just allows you select types of files, e.g. pictures, documents videos etc. I do not believe this will not neccessarily let you back up ALL your files let alone the entire system.

Some installations appear to have a menu option that lets you create a system restore DVD. This creates an operating system restore DVD that can be used to restore your system to 'factory default' but it does not usually back up your data.

I partition my hard disk to keep the OS and data on seperate partitions. This allows me to switch between operating systems without affecting the data. I periodically back up the system partition using Paragon Recovery software, but use a more traditional file backup tool to back up my data partition.

Most Windows users, however, will have everything on one partition. I believe the quickest way to back up everything is to use something like Paragon Recovery disk or alternatively Gparted which you can download from the Internet and burn your own boot CD. These will back up the entire partition and restore it back exaclty as it was in the event that it becomes neccessary. Of course, whatever you back up on to would need to have enough space to contain all your data. The tools in question do compress the backup to a certain degree and back up only the used portions of the disk.

There will be other tools out there and this is a matter of choice, but just to second the point made by Bulldog, it is always a good idea to back up your entire system before attempting an upgrade.
