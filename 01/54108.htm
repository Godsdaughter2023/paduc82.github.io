arminatorx say:

Well, if you still have a real Xbox 360 and an active Gold membership, you can do it like this:

Save your backwards compatible game on the Xbox One, then make sure to quit the game by going to the Home screen, pressing the Menu button on the main game tile with the 360 game running, and choose "Quit" from the menu. 

This forces the Xbox One to synchronize the savegame with the Cloud. 

After the sync is complete, go to your Xbox 360 console and make sure you're signed in with the same Microsoft Account email. 

Then go to the Settings menu on the 360, and under storage, insert your USB drive, and let it be prepared for storage, if it isn't already. 

Then open your cloud storage, and copy the desired savegame to your USB drive.

Windows 10 single lanaguage - Microsoft Community

anishmahato say:

Windows 10 single lanaguage

I Don't understand clearly what does Windows 10 single language means? WIll I be abe to use only single language?

Can I change to windows 10 pro?

What is the significance of windows single language..  I have lot of queries about it. Please help me about it.

a. user say:

HI Anish,

Thank you for posting your query on Microsoft Community.

Windows 10 single language means that the you are not eligible to download any other language pack apart from the language pack provided in the system. Therefore, you cannot install any other regional language pack.

Therefore, if you wish to upgrade from Windows 7 or Windows 8.1 Single language you will only get a Windows 10 Single Language.

Note: If you have Windows 8.1 Pro Single language you can upgrade to Windows 10 Pro Single language and if you have Windows 8.1 Home Single language you can upgrade to Windows 10 home Single language (The same goes for Windows 7 editions).

Please go through the following link for more information:

https://www.microsoft.com/en-us/windows/Windows-10-specifications
