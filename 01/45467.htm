eg x thee elf say:

Mazda,

 Sorry to hear you are having problems. Keep in mind that when connected to the internet, a game update is ALWAYS going to take precedence over the game install. Unfortunately, you have no way to monitor the progress when this is updating so the install process just sits at 0% with no progress bar available for how far along the update is. Depending on the size of the update and your connection speed will depend on how long the update takes. By disconnecting the internet, the system has no way to check for the update so it immediately runs into the install process. However, before you can play it the game update still takes place. So it's not really a work around but just change the order of how things are done.
 

 As far as a possible solution, first try to power cycle your console, by holding the Xbox Console touch button for 8 seconds until the console turns off, wait 30 seconds, then turn the console back on and wait for the game update to complete. Doing a power cycle will clear the cache and reset the network card. If that doesn't work, try uninstalling the game by going to My Games & App, highlight the game, click the Menu button (the old start button), and select Uninstall. Now power cycle your console, leave it connected to the internet, turn your console back on, and try installing the game. The update should take place first but please wait it out. Please let us know if this helps in resolving your issue. If not please provide us a status and we will be more than glad to continue troubleshooting your issue.
 

 In regards to a possible server crash on December 25th, it is highly unlikely that Christmas day will have a higher server demand than November 22nd. I could go on to reasons as to why the game install process has changed with the DRM and always online discussions, but I would prefer to keep this conversation focus on troubleshooting your issue. Please keep us posted. Thank you,
