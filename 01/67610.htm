You showed in a later response that this account is using POP3. This protocol has no concept of read and unread, because it's one-way traffic only (from server to client). Whether a message has been read or not will usually set a flag against the message in the client; that flag can only be transmitted back to the server where the mailbox is hosted if there's some synchronization going on, e.g. by using IMAP instead of POP3. Many IMAP implementations will flag messages as read at the server once the folder concerned is synchronized. POP3 ignores all folders in the mailbox other than the inbox.

I don't understand your reference to "all mail." Where did you check this option? Gmail accounts (and others powered by Google) do have a label All mail which will translate to a folder in Windows Live Mail when the account is set up to use IMAP. This should normally be avoided, though, because All mail is just that - a label applied to all messages in all folders. It would simply mean duplication of all messages in the client.

I would request you to answer the questions mentioned below, before we start with the troubleshooting steps.

  1. Are you able to work with other USB devices without any issues?
  2. What is the make and model number of the computer?

There are multiple reasons as to why a device is not detected by the computer. Below, I have listed few of them for you.

  • The device which you are trying to connect is faulty.
  • The USB port is not working or is faulty.
  • The USB device drivers are corrupted or outdated.
  • The USB cable using which you are connecting the device is faulty.

As the first steps of troubleshooting, I would recommend that you run the Hardware and Device Troubleshooter.

Open the Hardware and Devices troubleshooter

If the Hardware and Device troubleshooter fails to resolve the issue, you can try to uninstall and reinstall the USB Serial Bus Host Controllers.

Follow the steps provided below to uninstall and reinstall the USB Serial Bus Host Controllers.

a)     Open Device Manager by clicking the Start button, then Device Manager in the search box.‌

b)     If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

c)      Look for the device under USB [Universal Serial bus] host controller, right click on it and select Uninstall.

d)     Reboot the computer.

e)     On reboot, Windows 7 should re-install the drivers automatically.

Feel free to post on Microsoft Community in case you need further assistance.
