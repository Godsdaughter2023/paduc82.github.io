oddessa say:

Hi, i recently had similar issues with scheduled recordings, eventually i had as you mentioned total failure of scheduled recordings. It got to a point where i could not schedule a program if it was due to start in anything over 15mins as i knew it would fail.

After trying various fixes ie, remove WMC and reinstall rerun tvsetup, the problem persisted, i was on the verge of reinstalling windows, which would have been a pain and should not be neccassary.

But before i did i decided to ferret around in the registry to find a possible cause to the problem, and eventually i found the offending entries.

Uninstalling WMC and reinstalling should fix the issue but there are 3 keys left which are not removed from the registry, these were the cause of the problem.

These entries were:

mcupdate_scheduled

StartRecording

PvrScheduleTask

So to fix it, i did the following:

Go to C:\ProgramData\Microsoft and delete the ehome folder, you may need to stop, ehrcvr and mcglidhost.

Run "Turn Windows Features On or Off" and uncheck "Media Features" click OK and reboot.

Once rebooted open regedit and go to:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Schedule\TaskCache\Tree\Microsoft\Windows, find the key "Media Center" and delete. Now run "Turn Windows Features On or Off" and check the box "Media Features" to reinstall WMC. Open WMC and rerun TVSetup, this will generate the new keys for the registry.

Once complete all scheduled recordings should return to normal operation. Since doing this i have had no isuues with scheduled recordings.

Hope this helps.

Remember to always back up the registry or create a restore point, before attempting to edit the registry.
