To find the source, you can ask yourself....

Is the computer running through the full shutdown routine (such as showing "Shutdown in progress...." for a few seconds as it performs the restart)?

If yes, it means something started a reboot (probably an update, configuration change, or a script).

If no, it means the system is rebooting because of a sudden failure or crash.  You can "disable automatic reboot on system failure" so that it doesn't reboot but instead just stops on the error page, waiting for you to reboot yourself. That will help find out what is causing the crash.
