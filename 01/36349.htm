Some fonts may appear as gibberish in some apps on systems with Greek, - Microsoft Community

a. user say:

Some fonts may appear as gibberish in some apps on systems with Greek, Hebrew, Thai, or Baltic languages

We are aware of an issue where some applications are no longer displaying all of their text properly after installing the Windows 10 Creators Update.   Our investigation shows an issue with GDI (Graphic Device Interface) that affects the rendering of bitmap fonts on the non-Latin, single-byte system locales of Hebrew, Greek, Baltic, and Thai.   The fonts impacted are Courier, FixedSys, MS Serif, MS Sans Serif, Small Fonts, System, Terminal, Modern, Roman, and Script.   

 

We are addressing the issue in a future Windows update.  We will update this forum thread when the update is available.

 

As a temporary workaround, you can try using a different font in TrueType format by changing the font manually within the impacted app.  (Some apps have a setting to change fonts.)  For example, in Windows Powershell, right click the title bar of the application, select Properties, select the “Font” tab, and then change the font to a TrueType font like Times Roman or Helvetica.
