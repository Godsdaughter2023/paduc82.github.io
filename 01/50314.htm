Junction point not updated w moving of folder

Why is it that when one moves My Documents, My Videos, My Pictures and My Music via the Location tab option and move the fils, the junction points for these folders do not get updated? Instead they get deleted because their old destination folder is gone after the move!

ronnie vernon say:

Hi

The Junction Points in Windows 7 are there for backward compatibility purposes. In Windows 7, many of the default folders names and locations were changed. If an older program is installed and it is hard coded to look for a folder in a certain location, the Junction will redirect it to the new location for that folder.

When you move one of the default user folders to a new location, it is not necessary to update the junction that points to that folder. The junction will still redirect programs to the original location and from there the system will take over and direct it to the location where the user moved the folder just as it would with any other operation after a folder is moved.
