If other information is relevant, please let me know.  Thanks!
cari [former ms-mvp] say:

It sounds to me like you have changed the USB port a couple of times as well as re-installing it.  Some printers contain a unique ID within their hardware so it you disconnect/reconnect it, Windows will know immediately it is the same printer, whichever USB port you have it on, and reinstall the correct software automatically.  Many do not have this unique ID so whenever you disconnect and reconnect it, Windows will ask you to reinstall the software all over again since you may indeed have two printers of the exact same brand and model.  Your printer sounds like it doesn't have the unique ID, so poor Windows has to ask you each time you change port to install the software. 

 

I would disconnect it from the PC, and uninstall the software from the Control Panel AND from Device Manager, so that you end up with just one printer installed.
