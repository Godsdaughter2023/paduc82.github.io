After some investigation I found a solution for eGalax USB touchscreens used in many touch devices.


Details

The reason for this issue: The eGalax USB touchscreen driver installs in branch Mice and other pointing devices. So Windows 10 is using the touchscreen as a mouse. 

To avoid this, the Galax USB touchscreen  driver needs to be registered within Human Interface Devices in device manager. Gladly eGalax and other vendors has provided a fix for that issue. Further details may be found within this blog post: 

Windows 10 upgrade: On-Screen-Keyboard/Touchscreen fix

ethan b. say:
Hey Wallyjoe2009,

You can deal with this one of two ways:

Without installation media on hand, you could open an administrator session of Command Prompt (go to Start, look up "cmd" without quotes, then right-click Command Prompt, and Run as administrator) and enter the following: Dism /online /enable-feature /featurename:NetFX3 /All

You'll need a Windows 10 ISO for this next solution given the one above does not cut it: https://www.microsoft.com/en-us/software-downlo...

Choose Create installation media at the prompt, instead of Upgrade this computer now. Select your Windows 10 version, edition, and language. Select ISO file and let it download.

Double-click the ISO file and it should make a new drive with the ISO in This PC.

Make another administrator session in Command Prompt, and enter the following:

Dism /online /enable-feature /featurename:NetFX3 /All /Source:<insert_my_pathname_here>\sxs /LimitAccess

The path includes the new drive letter you made, and find the sxs directory in the new drive.

Otherwise, someone in the Reddit thread below suggested you turn off .NET Framework 4.7 in order to turn 3.5 on.

I hope this helps, please do let me know if it doesn't so we can pursue other solutions. Source, if you want to read more about this: https://www.reddit.com/r/Windows10/comments/7ld...

a. user say:

HI,

Thanks for posting your query to Microsoft forum, I will certainly help you with this.

Please try these basic steps and check.

  • I would suggest you to remove the battery and reinsert it, to make sure it’s connected firmly.
  • Make sure to use the original cord and adapter that came with your Cyclone Voyager 2 Tablet.
  • Check for any damage or loose connection on the charging port of the tablet.
  • Connect the charger to the wall socket directly instead of a surge protector while charging.
  • Check with another original charger

Follow the methods if above steps doesn't help.

Method 1: Use the hardware troubleshooter to fix the issue

The Hardware Troubleshooter is an automated tool which will check the hardware connected to the computer for any known issues with it and provides the details and on how to fix them. Follow these steps and run the troubleshooter.

Follow these steps:

  1. Press Windows key to get the Start menu.
  2. Type Troubleshooting in the search box right up corner and hit enter.
  3. Click on Troubleshooting tab.
  4. Click on view all option on the upper left corner.
  5. Select the Hardware and Device options from the list.
  6. Click Next to run the Hardware and Device Troubleshooter.

If the issue persist, follow method 2.

Method 2: Disable USB selective suspend setting.

This method prevents your USB device from powering down.

  1. Click the Start button, type power plan in the Search box, then click Choose a power plan.
  2. Next to your currently selected plan, click Change Plan Settings.
  3. Click Change advanced power settings.
  4. Click the box to expand USB Settings, then expand USB selective suspend settings.
  5. Click plugged in, then click the drop down menu and select disabled.
  6. If you're using a laptop, click Battery, then click the drop down menu and select disabled. Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.
