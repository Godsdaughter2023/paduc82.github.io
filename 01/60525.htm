flowerchilddk say:

Hi bot47

I had the same problem but I've figured out a workaround. Basically we're going to open the folder where Windows stores the library files (not the library's files) and trick Windows into adding space at the end of the stubborn library's name:

1.  Open the Command Prompt (Typecmd in the Start menu).

2.  Paste the following text into the command line (Right click and choose paste):

%userprofile%\AppData\Roaming\Microsoft\Windows\Libraries

3. Type dir to see a list of the libraries you have. Your stubborn "Clips"-library is probably named something like this:

Clips.library-ms

4. Now just type (Or paste):

ren "Clips.library-ms" "Clips .library-ms"

Remember the quotation marks and don't forget the the space that has been added between the file name and the ".library-ms"!

You may need to restart your computer for the changes to take effect.
