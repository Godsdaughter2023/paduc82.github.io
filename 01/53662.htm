another rob hawkins say:

How to get Deltek Advantage Software to run on Windows 7

Hey all!
So here's the problem and the solution. I hope this will help someone else. Not sure If I should post question and answer here... but thought it would help... Please let me know if it did. :)

A software package from the company Deltek, called Advantage and Timekeeper, keeps track of complex jobs and time tracking. Their current version isn't Windows 7 compatible, and Deltek will not offer any W7 support. They changed their product lines to be web based.

Solution for Windows 7.
I loaded and setup the FREE Virtual PC Windows XP, installed all the OS updates, Installed the Deltek Advantage and Timekeeper software. It installed with no errors.

The next problem I had was UNC/DNS resolution with the Deltek software.

So to fix that, within the Virtual PC Windows XP session, I edited the virtual c:\windows\system32\drivers\etc\hosts file to add:
192.168.69.21    Server21

As the last entry in the file. Where Server21 for the rest of the newtork is the Deltek MSQL server used by the software, and needs to be referenced by UNC/Netbios, and not IP.

I then rebooted the system, and the software within the Virtual PC Windows XP mode saw and correctly addressed Server21.

Hope this helps you!

DELTEKEXTRACT\MDAC\JetSetup

The issue could be related to Display drivers then please try the steps provided to uninstall the Display Adapter from Device Manager, uninstall the device drivers from program list and then download the latest drivers from the manufacturer's website.

 

Step 1:

Uninstall the Device from Device Manager

  • Open Device Manager
  • Expand the Display Adapter section.
  • Find the Display card Driver.
  • Right click Display card Driver and select Uninstall.
  • Select the check box delete the driver software for this device.
  • Reboot the computer after uninstall process has finished.

 

Step 2:

Uninstall the Device Drivers from Program List in Control Panel

  • Click on start, select Control Panel
  • Now click on Programs and select Programs and Features.
  • Find the Display card Driver.
  • Right click Display card Driver and select Uninstall.
  • Reboot the computer after the uninstall process has finished.
