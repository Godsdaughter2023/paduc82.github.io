Let’s change the boot sequence in BIOS by giving boot from CD/DVD as the first priority and then remove the Updates using the DISM Tool by following the steps given below:

1.Boot to System Recovery Options by following the steps given in this link and select Command Prompt.

 What are the system recovery options in Windows 7?

http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7

 

2.      From the command prompt run the following command: Dism /online /get-packages

3.      This will produce a list of installed updates and packages.  They will look like this:

4.      Package Identity: WUClient-SelfUpdate-Core-opLevel~31bf3856ad364e35~x86~~7.4.7600.226

5.      State : Installed

6.      Release Type : Feature Pack

7.      Install Time : 9/25/2009 12:02 AM

8.       Find the most recent one and then make a note of the package name.

9.      Then run: dism /online /remove-package /PackageName:WUClient-SelfUpdate-Core-TopLevel~31bf3856ad364e35~x86~~7.4.7600.226(name of update from previous step)

10.   Then reboot your PC.

11.  You will want to start with the most recent and remove them one at a time to find the problem update.
