george222(2) say:

Are these older games that require a CD/DVD to be inserted into the PCs drive to play them? If so there is a problem with the disc security system they used. 

For Windows 7 and Windows 8.1 you need to remove an update, or create a workaround to get RTW running, I'm not sure about Caesar 3? See https://answers.microsoft.com/en-us/windows/forum/windows_7-gaming/kb3086255-is-this-a-half-baked-solution-or-am-i/13d4c03e-f04f-40b9-b438-e903eda3cab9 for details. The offending update is https://support.microsoft.com/en-us/help/3086255/ms15-097-description-of-the-security-update-for-the-graphics-component

Unfortunately, in W10 the update is built in, and as far as I'm aware there is no workaround and the update cannot be removed.

So you either have to play the games on W7 or W8.1 with the update removed, or using the workaround mentioned in the first post. Or for W10 ( and W7/W8.1 ) you'll have to re-purchase RTW from Steam or similar to get a compatible version. Steam itself is free, the RTW game is about £7-£9(£ UK) - http://store.steampowered.com/app/4760/Rome_Total_War__Collection/ at the moment.

That assumes you have the old original disc version of RTW.

For Caesar 3, I'm not certain as I haven't got it to try, but if it uses the same disc protection it'll be the same problem. If not, you could try installing it and running it in Compatibility mode just to see if it helps. You'd need to select a version of windows that the game was intended to run on, say W7, W-XP or W-98 whichever is applicable.
