Hi Christopher,

There are different available methods on how you can set a scheduled restart on your Windows device. This can be done using third-party apps or using the task scheduler that comes with Windows. To resolve your concern, you can follow the steps below to set a scheduled restart in Task Scheduler:

  1. Press the Windows key+S to open Cortana or search (Windows 10 shortcut) then type taskschd.msc and press Enter. (This will launch the Task Scheduler) If you're not using Windows 10 you can use the key combination Windows key + R to open the run box then type the command.
  2. Left click on the folder labeled Task Scheduler Library on the left-hand panel then select Action from the top menu.
  3. Inside the Action menu, select the option New Folder... and then enter a personalized name you want then click Ok.
  4. Left click on the folder that we created then select on Action from top menu then Create Basic Task.... (This will open the task wizard)
  5. You can now enter the name of the task, for example, "Restart", and press Next.
  6. In the Trigger section you can specify when you want to run your task, for example:
    Select Daily, and press Next.
    Now you can specify the day, the hour and set to recur every day, after that press Next.
  7. In the Action section you can choose what you want to run, for example, to restart your system:
    Start a program, and press Next.
    In the Program/script field type shutdown /r, and press Next.
  8. Select Finish.

Take note that there are other methods that you can utilize for more specific options on how a certain task will be executed.

allan mej say:

Sorry for the late. You can actually schedule or delay PC shutdown in Windows 10 by running some command on the Command Prompt. To do so, follow these steps:

  1. Search and open the Command Prompt.
  2. Run this command to create a shutdown timer (XXXX is the time in seconds):
    shutdown -s -t XXXX
    To create a restart timer:
    shutdown -r -t XXXX
    To cancel a shutdown:
    shutdown -a

Also, you can create a shortcut in order to run a shutdown/restart timer easily. To do so, follow these steps:

  1. Right-click on any blank space on the desktop and hover to New>Shortcut.
  2. In Type the location of the item, enter the shutdown or the restart command.
  3. Enter your desired shortcut name and click Finish.

We also recommend creating a second shortcut which cancels the shutdown/restart timer. Simply follow the steps above and use the shutdown command in the Type the location of the item.
