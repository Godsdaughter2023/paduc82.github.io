SD media cards, and all media cards, seem to all work just like the older floppy diskettes.  The card must be in the correct drive / slot before you are able to write to it.

SD cards have a write protect tab, just like the older floppy diskettes.  You need to make sure that the tab is not set to Write Protect.  And it is part of the SD card "specification" for this write protect tab to be on the card but several media card "drives" appear to not follow the specifications.

You must have a SD media card inserted into the SD card reader slot before you can write to it.

You would also need to know that most digital cameras manufacturer will probably only recommend that you format the SD card from the camera's own format option and not from Windows.

ken blake say:

See DaveM121's reply, and let me add several points:

1, Your fears are justified.  Formatting the drive *will* erase its contents. By definition, that's what formatting means.

2. You've misinterpreted what the articles you've read said. There is no need to  format the drive.

3. As Dave said, since you apparently want the data on the drive, rather than add it to the new computer, it would be better to copy the data to the new computer. You can do that by installing the old drive in the new computer, or by buying an external enclosure fr it (around $20), installing the old drive in it (very easy to do), and then copying its data to the new computer's drive.  That's easier than installing it in the computer.

4. Note whether you install the drive and use it, install it and copy it, or install it in an external enclosure and copy from there, you can use the data on it but *not* the programs there. The programs need to be reinstalled on the new computer from their original media, not copied to the new computer.

5. If you do install the old drive in the new computer, after copying its data there, you can then format it (erasing everything there) and use it as additional storage space.

6. Assuming that both your old drive and new drive are of adequate size, in my view it would be much better to buy an external enclosure for the old drive and use it to do regular backups of the critical data on the new drive. An external hard drive, rather than a second internal dive,  is an ideal medium for backup. Using an internal drive for backup leaves you susceptible to simultaneous loss of the original and backup to many of the most common dangers: severe power glitches, nearby lightning strikes, virus attacks, even theft of the computer.
