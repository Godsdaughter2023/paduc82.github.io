You are getting audio in at least one game, or not ?

- Not knowing what you have done I'll start from the top. There are two main places to go for your audio setup.

Assuming you are using PC speakers connected to the rear panel outputs (also verify they are all connected to the correct jacks) ...

In - Control Panel > Sound > Playback tab (or Settings > System > Sound > Manage Sound Devices > Sound Control Panel)

verify Speakers are set to Default

- Highlight Speakers and click on Properties.

In the Levels tab, verify all the Speaker Levels are up to 100% (not the main Speaker level, which you use to adjust volume as needed).

cannot use higher Hz settings. You can set a higher level, but remember this option if you get a game that does not reproduce audio.

- You are likely to have other audio options such as Intel/AMD/NVidia. These should be disabled unless you are using HDMI for

audio via speakers built into a monitor/TV.

Note that if an HDMI option is enabled you may be able to hear music from your PC speakers, but games may run through the

Monitor/TV speakers and not be audible unless you crank up the volume as these speakers are generally very weak.

- VR Chat...

If you are using a Headset that connects via

- When using headphones, if you setup so the speakers switch off you can also select a different EQ, etc for the headphones in

the Realtek HD Audio Manager, which should enable automatically when you plug the headphones into the front panel.

steve winograd say:

There's no program by that name in Windows, and no programs should be loading from the AppData folder.

That program was probably part of a virus or spyware infection that was partially removed. Run a full virus/spyware scan.  If the problem still exists, use the free Autoruns  tool from  Windows Sysinternals  to remove the reference to the program from the indicated registry location.

debleena s say:

Hi Vinbug,

·         Are you using a wired or wireless mouse?

·         What is the make & model of the mouse? 

Let’s follow these steps and check if it helps.

Method I

A) You may uninstall the mouse drivers and reinstall it and check if the issue persists.

1. Type devmgmt.msc in start search box and hit Enter.

2. In device manager, expand mice and other pointing devices.

3. Right click on it, click on Uninstall.

4. Reboot the computer. The default drivers for mouse will get installed automatically.

B) You may also update the drivers of mouse.

1. Right click on mice and other pointing devices.

2. Click on update software driver.

3. Select the option ‘Browse my computer for driver software’ and browse to the location where you have saved the downloaded software.

Update drivers: recommended links

Method II

You may also create a new user account with administrative privilege and check if the mouse is working fine. If it works fine, it means that the other user account is corrupted.

Create a user account

http://windows.microsoft.com/en-us/Windows7/Create-a-user-account

Access the link below and follow the steps to fix a corrupted user profile:

http://windows.microsoft.com/en-US/Windows7/Fix-a-corrupted-user-profile
