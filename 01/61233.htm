does installing OS in a disk which has allocation unit size of 64KB, remains after it has been formated for OS installation by the windows installation disk? or will it reformat it into default allocation size (4KB)?

a. user say:

Hi ManishCharan,

Thank you contacting Microsoft Community.

 
By default, the maximum cluster size for NTFS under versions of Windows is 4 kilobytes (KB). The format won't use clusters larger than 4 KB unless the user specifically overrides the default settings.
 
If the FAT cluster size is greater than 4 KB, then the clusters are converted down to 4 KB in NTFS.
 
For more information please refer the article below:
Refer Section (Default cluster sizes for NTFS)

http://support.microsoft.com/kb/140365/en-us

Note: Above Article applies to Windows 8.1.

 
Conclusion
If you Re-Install Windows using Windows Installation Disk it will Restore to default Allocation Size to 4kb.

Understanding Disk Partitions
