Wireless Security - Microsoft Community

xbr dionysus say:

Wireless Security

Login Information

  • Gateway - 192.168.0.1 or 192.168.100.1
  • Login Username - admin
  • Login Password - motorola

 

How to Find Wireless Passphrase(WPA/WEP)

  1. Open a web browser.
  2. Enter 192.168.0.1 or 192.168.100.1 in the address bar. Enter the username and password if the gateway asks (admin/motorola).
  3. Select Wireless on the across the top.
  4. Select Primary Network on the left.
  5. The password is on this page in the box labeled “Pre Shared Key”.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Enter 192.168.0.1 or 192.168.100.1 in the address bar. Enter the username and password if the gateway asks (admin/motorola).
  3. Select Wireless on the across the top.
  4. Select Primary Network on the left.
  5. The security is on this page. Select Enabled on desired security type. (To turn off the security, select Disabled on all the security types.)
  6. Click Apply.

ddeus say:

Reviving an old post, but wanted to put in my $0.02...

I just set up an 8 monitor display for the CEO of my company.  Proved to be a headache, but here's what I did:

Hardware:

2x Jaton Corp ViDEO-PX309-QUAD

Thermaltake 850W Black Widow PSU

Core i7-2600k

Asus P8P67 Deluxe

OCZ Vertex SSD

 

Originally, the cards were bought to be put in a Dell Precision 3400.  This machine came with a 375W PSU, but the cards needed 400W.  Sadly, the mobo on the Dell only supports up to a 535W PSU, so a full rebuild was in order.  Once the build was done, troubleshooting wasn't too bad.  Make sure you have updated your BIOS and drivers on the cards.  Also, be sure you're meeting the power requirements.  The thing with these cards is that they don't take a dedicated power plug, it's just pulled through the PCI-E slot.  I am running into intermittant freezing here and there, but I'm not sure if that's an issue with Outlook or XP not wanting to support the 8 screens.

 

Anyways, hope this helps whoever else looks, and to the OP, if you found a fix for your show, please share!
