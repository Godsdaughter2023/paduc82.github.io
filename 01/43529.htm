lemp say:

I don't have a Surface keyboard or Windows 10, so take the following comments with that in mind.

You do not need a driver for normal keyboard functionality.  Some keyboards have special keys or other non-standard features.  In order to get the functionality for these additional features, you have to install a driver or other software.  For older Microsoft keyboards, for example, you had to install the Microsoft Mouse and Keyboard Center (I don't believe that applies to the Surface keyboard).

It's my understanding that the Surface keyboard communicates to the computer using Bluetooth 4.0, also known as Bluetooth Smart/ Bluetooth Low Energy (Bluetooth LE or BLE).  The problem is that Windows 7 does not include drivers for Bluetooth 4.x, and typically you see the statement that "Windows 7 is not compatible with BLE."

Depending on the hardware in your computer, however, you may be able to find drivers that will support BLE.  See the discussion here: https://stackoverflow.com/questions/40269110/bluetooth-low-energy-on-windows-7?rq=1

There are Bluetooth 4.x USB adapters that claim to work with Windows 7.  If you've already bought a Surface keyboard, you might want to invest a few more dollars and see if one of them will let you use it:  https://www.amazon.com/s/ref=nb_sb_noss_1?url=search-alias%3Daps&field-keywords=bluetooth+4.0+adapter (Read the user reviews and user questions, to see if they will really give you BLE on Windows 7 or if they will just function as "backwards compatible" normal Bluetooth adapters).  Note that if your computer already has Bluetooth built in, you'll probably have to disable it before installing one of these "dongles."

The date on which Microsoft releases the next feature update (build 1803) may be of academic interest, but your computer won't receive it on that date. It takes several months to send an update to all of the more than 600 million devices that are currently running Windows 10.

Updates are released in waves. Each wave targets different groups of users, and Microsoft waits between waves to see if users are discovering bugs in the update. If so, Microsoft will try to avoid sending updates to computers that may be affected by the bugs, until they are fixed.

The success of all this depends partly on how much telemetry Microsoft can gather from users' computers. Users who minimize or outright sabotage Microsoft's ability to collect information about how they use their computers hurt themselves as they may receive an update that is not appropriate for their systems.

Aside from all this, the conventional wisdom has always been to wait before installing major upgrades ... to wait months, if possible ... to allow time for early adopters and other software testers to test the upgrade.

louismark say: Steps to find a WEP key:

1. Open a browser (the program used to access the internet). The following process worked for myself when using the

Internet Explorer, Firefox and Opera browsers, although I am unsure whether it will work for other browsers (eg

Chrome, Safari etc).  

2. Type the following into the address bar of the browser. If using a Linksys router type in "192.168.1.1" and if

using a Belkin router type "192.168.2.1". The address bar is the area where webpages are typed. It is important

when typing in that there is NO http://www. before the numbers.

3. If all has gone well this should open a page that either says Linksys or Belkin.

3. There may be the need to login. When using a Linksys if you don't know what the username and password are it may

still be the factory settings, and so enter in "admin" for the username and "password" for the password. If using a

Belkin router, the factory settings often come with no password, so just click submit without typing in a password.

4. Click on a) "Wireless" or b) "settings".

5. Click on a) "Wireless security" or b) "wireless settings".

6. There should be a page with a WEP key listed.

7. Write it down/copy and use as you wish!
