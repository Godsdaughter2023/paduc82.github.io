Welcome to the Skype Community forum. I do understand the inconvenience you have faced. Let me assist you with this.

It is possible that your (or the person you're calling) audio driver is outdated. Let's try updating your audio driver.

First we need to identify your default device driver. Go to the Sound Control Panel via one of the following ways:

  1. Navigate to the Control Panel, and click the “Sound” link

  2. Run “mmsys.cpl” in your search box or command prompt

  3. Right-click on the sound icon in your system tray and select “Playback Devices”

  4. In the Sound Control Panel, note which device is your system default

  1. Return to your desktop.

  2. Type “Device Manager” in your search box and click on the Device Manager app

  3. Expand the “Sound, video, and game controllers” tab

  4. Right click your default audio device driver, and click Update driver.

  5. Reboot your device.

Once done, completely uninstall and reinstall Skype application to detect the updated driver. Carefully follow the instructions here. In the event that issue persists, do the following:

  1. no longer running.
  2. Press the Windows   and R keys on your keyboard at the same time.
  1. Type "%appdata%\skype" into the Run window and click OK.
  2. Find and delete the shared.xml file.
  3. Open the folder that has your Skype name.
  4. Locate and delete config or Go back to the appdata folder and rename Skype e.g. Skype_old
  5. Launch Skype.

srimadhwa b say:

Hi Ray,

 

You need a DVD and not a CD to save the ISO image.

 

Follow these steps to create Windows System Image.

 

1. Press Windows (logo) + X.

 

2. Go to Command Prompt (Admin).

 

3. Then type mkdir C:\RefreshImage click Enter.

 

4. Then type recimg -CreateImage C:\RefreshImage click Enter.

 

5. Waiting for the ISO Image of Windows 8.1 to create until finished.

 

6. After that restart your computer.

 

7. Then go to Computer>Local Disc(C :)> RefreshImage>CustomRefresh.wim>right click to copy that ISO image of Windows 8.1 to the preferred devices that you would like to keep that ISO image.

 

Article for Reference.

How to create a system image to refresh your Windows 8 PC: http://support.microsoft.com/kb/2748351/en-us
