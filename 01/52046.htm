shadowsports say:

Greetings,

WoL typically needs 3 things to function properly, depending on where you will be accessing the device from, and only 2  if you will be accessing locally from inside your LAN.  

First step.  You typically need to enable WoL or Power On by "device type" in BIOS.  This is not possible with the SP2 as you are not able to access or enter the BIOS, instead we are only given access to an extremely limited number of functions.

Second, is the settings or Properties of the Ethernet adapter being used.  The device must support WoL and if a 3rd party device is installed such as a USB to gigabit adapter, you may need to install the manufacturers device drivers in addition to the ones windows may already supply for basic function.  The driver must support WoL and Power Management, otherwise you cannot set the wake capability or ability to keep the Ethernet adapter powered and "listening" for a wake packet when the device is sleeping.    

These additions may or may not be available depending the network device.  I know the MS adapter does not support this since I have read that the Power Management Tab is grayed out, and you confirmed this regarding the Dock as well.  I have not tested these functions with my SP2 as I only ever intended to use them as mobile only devices.  You may want to investigate a 3rd party app or program

MS Rempte Desktop App (download in store)

VNC, Real VNC or Tight VNC as alternatives

I suspect however, that you will need hardware that allows Power Management to be tweaked in order to gain this functionality.

CPU Not Compatible

To install Windows 10 on your PC, the processor (CPU) must support the following features: Physical Address Extension (PAE), NX, and SSE2. Most CPUs have support for these features, so if you receive this error, it is likely because the NX feature is not enabled on your system.

If your computer does support these extensions, you could still try using ISO media. In some instances, you might have to switch from your current architecture,  64 to 32 bit since these requirements mostly affect installation of 64 bit Windows 10:
