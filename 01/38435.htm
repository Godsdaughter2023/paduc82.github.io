"I've tried going to the file in question and replacing it with a freshly downloaded file to remove any possible corruptions, however it won't let me delete or replace it and says "you require permission from TrustedInstaller to make changes to this file."

- Just as well. These .dll download sites are notorious for including a bit of spyware or malware with their downloads.

I also avoid 'optimising' software such as tweakbit PCRepairkit, particularly any Registry 'cleaning' tool that may come

with these programs as they have been known to create serious issues at times.

- The d3d11.dll is the main DirectX 11 file. If d3d11.dll, or any DX file is in fact corrupt the best option is to download

the the DirectX Redist (June 2010) installer and reinstall DX.

This will overwrite all DirectX files (except the couple of DX12), replacing any corrupt files.

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

(note that the download first unpacks the files to a location of your choice - a new folder on the desktop is good as

you can deleted the folder after you run the DXSETUP.exe file from within that folder to finalise the DX reinstall ).

- However, DirectX errors are not always accurate. They are often due to something else such as using a graphics processor

that is not adequate for the games system requirements, or a corrupted graphics driver, among other things.

- DX errors can also be generated if the correct version of C++ is not installed, or has been corrupted. Games will generally

install\reinstall the version of C++ they require (which, as with DX, can corrupt an already good DX or C++ install),
