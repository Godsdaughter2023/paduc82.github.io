Windows 10 is notorious for creating archived files you don‚Äôt need. Two features in Windows 10 that can help you with that are Disk Clean up and Storage Sense.

Open Start, type: disk cleanup then hit Enter.

Once Disk Cleanup opens, select the drive where Windows is installed if it‚Äôs not already. Windows will scan your hard disk for files it can clean up.

After Disk Cleanup completes its scan, click the button Clean up system files.

Click Reset which will delete all the data stored by the app.

The recovery partition used to reinstall Windows 10, can eat up a significant amount of disk space. It‚Äôs used rarely, if ever, so there is no need to keep it.
