Windows Crash. Recover data on seond HD? - Microsoft Community

ponyboyonline say:

Windows Crash. Recover data on seond HD?

So last night it appears that Windows has completely crashed.  I used the Windows installation disc to try and run a system recovery but that failed.  I was also unable to restore from a backup saved on an external hard drive as it said it was corrupt.  My assumption is that I have to completely reinstall Windows.  I will have to figure out what caused this weather it be hardware or software issues, but my first concern is my data.  All my important data that I really care about is located on a secondary SSD (Edit:  Primary drive is SSD, secondary is spinning HD).  Is there a way to recover that while Windows is reinstalled on my primary drive?  Any advice or support would be greatly, greatly appreciated. 

canadian tech say:

Remove the secondary drive.  Mount it temporarily in an external drive housing.  Hook the external drive housing up to any computer and you should be able to find your data and copy it off to what ever safe place you choose.  You can find these external drive housings in lots of places and they are not that expensive.  After you finish with this recovery, consider buying a new high capacity hard drive to install in that housing to make your own backup drive.  If you buy a good quality drive housing with the ability to cool itself and protect the drive inside, it will prove to be far more reliable.

You might want to reconsider using SSD drives.  Rotating drives are far bigger, last longer and are cheaper.  Most importantly, rotating drives typically give far more warning time of an impending failure.  SSDs tend to fail rather suddenly.  If you are going to replace a hard drive, do not settle for a 5400rpm.  Insist on a 7200rpm drive.  Makes a big difference.

My experience is that SSDs are over-sold.  They do speed up computing, particularly on boot up, but when you take into consideration the overall impact in time saved, there is limited value to this.  The vast majority of computers are started once per day.  SSDs will cut that boot up time by maybe 3 or 4 minutes.  In a well running computer, an SSD will have little noticeable affect on performance after it is started.  Your Internet service and the performance level of your Windows installation will have far greater impacts.

In most cases when a computer has slow startup, the primary cause is a lot of stuff loaded on your computer that is pigging up the resources and when you take a considered review of them, you will likely remove many of them, or at least use autoruns to prevent all the stuff that is loaded at startup that you do not need.

If your computer has had its OS installed for more than about 3 or 4 years, test your hard drive to make sure it is sound using the hard drive's manufacturer's drive testing software.  If sound, re-install the OS, apps (the ones you really use) and data.  You will discover a vastly better performing computer.

how do i connect a scanner to windows 10? - Microsoft Community

jace kaymeyer say:

how do i connect a scanner to windows 10?

My computer recognizes my scanner but will not connect. What do I do?

greg carmack - mvp 2010-2020 say:
Hey Jace. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

Install the scanner's software from the manufacturer's Support Downloads webpage which is the only way to get bi-directional scanning to/from the PC.

If any problems then follow these steps to Clean install the scanner which always works, lasts longer and which are the same for scanners:

1) Power up the printer, if wired unplug it's USB cable from PC.

2) Go to Settings>Apps>Apps and Features to find and Uninstall the printer software.

3) Next copy and paste in Start search box: printui /s /t2 (Note that there is one space before each forward slash).

4) Right click result to Run as Administrator.

5) Open the Print Server properties, Remove the printer’s drivers for this printer and for any other printers you don't use presently. If you get a message about access denied or the printer being in use and you cannot find the driver or any installed program that would account for this, you may restart the computer in a diagnostic startup and then use the printui command to remove the driver package(s): https://www.isunshare.com/windows-10/enable-dia...
You must restart back to normal mode before you will be able to install the printer again.

6) Now type Control Panel in Start Search, open CP, choose Icon View. In Control Panel open Devices and Printers, right click the printer to delete the Printer and any scanner or fax of the same name. Restart PC.

7) Go to the printer model’s Support Downloads webpage to download and install the latest full software package. Now is also the time to check for any newer printer firmware to install which matters a lot.

Older software may need to be installed in Compatibility Mode to run correctly: https://www.groovypost.com/howto/run-old-softwa...

8) Extract Printer software, right click Setup file to Run as Administrator. Follow the setup screens closely as you install the software. For USB it should prompt when to plug in the printer.

If you have further problems reply back with results of each step.

How do I reset Windows Updates components ?

<<http://support.microsoft.com/kb/971058>>

Download and save the Fix it 50202 from the above page.

Close all open programs and browsers prior to running the Fix it.

Since the downloads are failing, when you run the Fix it choose it's Default mode when you run it the first time.

When it has finished running, check to see if updates download now.

If updates still fail to download after running it, then run it again and choose it's Aggressive mode.

Restart afterwards and see if the updates download now.
