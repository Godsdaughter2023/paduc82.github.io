How to troubleshoot Windows store apps on Windows 10, using the Store App troublshooter

Technical Level : Basic

Summary
Throughout this article, I will go though how you can run the App troubleshooter on Windows 10. The App troubleshooter will scan through the Apps, and Applications, on the computer/device to find any problems. Windows, once it has found problems with the Applications, it  should then try and fix these problems.
Details

To launch the Windows 10, app, troubleshooter, please follow these steps:

You will need to click into Cortana, on your Task bar. Once you have clicked into Cortana's search bar, you should see the curser flashing in the box, like in the picture below:

Once your curser is flashing at you, you can then type Troubleshooter. This then allows Cortana to search for "Troubleshooter" on the device and the website.

It will display results (As seen in the image above) from Control Panel, folders and the web. Once the results have been displayed, please click onto Troubleshooting, which should have control panel underneath.

After you have clicked onto Troubleshooting, then a windows will appear on your screen like this:

When the windows opens, you will need to click onto Hardware and Sound. This will then direct you to another page like this:

From this page, you will need to click onto Windows Store Apps. You may need to scroll down until you can see it. Once you click onto the page, Windows will begin scanning your Computer/device for any Apps that are not working correctly, or problems. Windows will also try and fix these problems for you, if it finds them. This will only scan Windows Store Apps that have been installed on your machine.

Windows 10 has been designed to make troubleshooting easier for people, as Cortana can search for solutions for problems you have on your computer/device as well.

On an OEM computer when you install Windows 8 using the ISO the product key that you are using is the same product key which is activate on the tablet. Hence you will not be able to activate Windows 8 on the OEM computer.

Alternatively, you may put the OEM computer back to factory setting and then upgrade to Windows 8 Pro with a legitimate product key. As the base version of Windows is not activated, you will not be able to upgrade to Windows 8 Pro with WMC.

 

Activation consists of mainly 2 components. They are software hash and hardware hash of a computer. Whenever there is a change in any of these 2 hash components, the activation is triggered.

If you have one copy of Windows and installed it on more than one PC, activation might not work because the product key has already been used on another PC, or it's being used on more PCs than the Microsoft Software License Terms allow. Normally, the rule to follow for most copies of Windows is that one copy can be installed only on one PC.
