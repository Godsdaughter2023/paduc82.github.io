"Special Permissions" simply means that the permission panel shows more options than are normally visible. To fix the permissions for a particular folder, do this:

  1. Log on under an admin account.
  2. Seize ownership of the folder.

Go to the order summary page, there you will find your product key:

https://www.mswos.com/

 

If Windows 8 came preininstalled on your computer:

 

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.

gerald vit say:

Apologies for the delay in response. You can use the Advance Sharing option to provide access to specific people on the network. To do this, follow the instructions below:

  1. Right-click on the drive or folder that you wish to share.
  2. Click Properties.
  3. Click on the Sharing tab.
  4. On the Sharing tab, click on Advanced Sharing.
  5. Tick on the check box beside Share this folder.
  6. Click on Permissions.
  7. A window will pop out with the name Permissions for <insert folder or drive name here>.
  8. Under Group or user names, click on Add.
  9. From here you can type-in the name of the user under Enter the object names to select.
  10. By default, the system will list the available profiles within the system.
  11. If the user is not available in your computer, click on Advanced.
  12. Click on Locations and select the location on where the system is going to look for the profile in another computer in the network. (e.g. desktop or laptop)
  13. Click Find Now and a list below is going to be populated.
  14. Under Search results, look for the user profile that you wish to provide access.
  15. Once you have specified a user to provide access, click on OK.
  16. Click OK again.
  17. Back at the window with the name Permissions for <insert folder or drive name here>, you will see the newly added account. Full Control, Change/Edit, or Read only priviledges for that user account.

If you have additional questions, or if the issue persists after the troubleshooting steps above, please let me know by replying to this thread. This way, I'll get notified and respond back to you as soon as I can.
