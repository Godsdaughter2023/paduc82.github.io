Fix 1 - Run Automatic Repair

1) In the menu, choose Troubleshoot > Advanced Options > Startup Repair.
2) Next, select the target operating system to fix. Windows will initiate the repair and then restart.
Fix 2 - Use SFC /SCANNOW to Repair System Files

1) In the menu, click Troubleshoot > Advanced Options > Command Prompt.
2) In the command line, type sfc /scannow and press Enter button.
3) Restart your computer when the process is finished.
Fix 3 - Run CHKDSK to Repair Boot Drive

1) Open Command Prompt using the steps in fix 2, type chkdsk /r c: and press Enter.
2) Type Y and hit Enter when you are prompted to schedule the check after a reboot.
3) Reboot and let the check proceed. It might take a while.
Fix 4 - Fix MBR

Open Command Prompt, type the following commands and press Enter after each move.

bootrec /rebuildbcd
bootrec /fixmbr
bootrec /fixboot
exit

clgreenwald say:

I was also able to achieve two Dell P2415Q monitors at full resolution (3840x2160), at 30Hz refresh rate with a Surface Book Performance Base and Docking Station. This is on Windows 10 Pro version 1607 OS Build 14393.693.

Both monitors are connected to the docking station using the Dell supplied Mini-DP to DisplayPort cables.

Both monitors had to be set to 'MST Secondary' as outlined in Dell's support document http://www.dell.com/support/article/us/en/19/SLN296015/EN

The setup was complicated by the fact that Dell does not expose the monitor menu without an input signal present.

Steps I took to get this working:

  1. Attached first external monitor miniDP to dock.
  2. Power on monitor.
  3. Booted docked Surface Book.
  4. Using P2415Q On-Screen-Menu, set MST to Secondary.
  5. On the Monitor tab of the display adapter's advanced dialog box, set screen resolution to maximum, refresh to 30Hz.
  6. Shut Down Surface Book.
  7. Removed Monitor 1 MiniDP connector from Dock.
  8. Added Monitor 2 MiniDP connector to Dock.
  9. Repeated steps as for Monitor 1.
  10. Shut down Surface Book.
  11. Plugged Monitor 1 and Monitor 2 MiniDP connectors to Dock.
  12. Powered on Surface Book.
  13. Three screens function. I do not notice any flicker.
  14. Adjust screens as desired using Display Settings. I have one Landscape, one Portrait.

One note: I have had issues with Surface Book re-enabling a monitor which has been powered off. The systems beeps like it sees it connect, but does not detect the power-cycled monitor when Display Settings 'Detect' is pressed. Sleep/wake or reboot cures this. I disabled monitor power-off when on AC power just in case.
