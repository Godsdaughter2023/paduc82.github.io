Hi,

There is no way to recover lost password for local user accounts. However, if you are using a Microsoft account, you can follow the steps on this link to reset your password. If you are using a local account, what you can do is to use a Windows 10 bootable installation media to enable the built-in administrator account. To create a bootable ISO file, kindly download the media creation tool on this link and follow the steps provided in the using the tool to tool to create installation media (USB flash drive, DVD, or ISO file) to install Windows 10 on a different PC section.

Once you have a bootable installation media, you can follow the steps below to enable the built-in administrator account:

  1. Boot from Windows 10 bootable CD/DVD or USB.
  2. When the install now screen shows up click on Repair your computer > Advanced Options >Troubleshoot > Command Prompt.
  3.  Once you can boot into the command prompt, type net user administrator /active:yes and click Enter. This will activate hidden administrator account.
  4. Close command prompt and remove installation media. Boot back to Windows 10 normal mode and sign in to Administrator account. You do not need a password for the hidden admin account.
  5. Create a new administrator account from the admin account.
  6. Once you have created a new administrator account, sign out of the hidden admin account and then boot into the newly created account.
  7. Now, you can disable the hidden admin account and use the new account.
  8. To disable hidden admin account press Windows key + X and click on Command Prompt (Admin).
  9. In the Elevated Command prompt, type net user administrator /active:no and press Enter. This will disable hidden administrator account.

You can now use the newly created admin account to login to your computer.
