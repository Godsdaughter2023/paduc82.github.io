Port Forwarding - Microsoft Community

xbr poseidon say:

Port Forwarding

Zhone 6381-A2-GB23-10

Login Information:

Default Gateway – 192.168.1.1

Username – Admin

Password – Admin

 

Port Forwarding

  1. Open a web browser and type http://192.168.1.1 .
  2. Enter admin for both the Username and Password.
  3. Click the Login button found in the lower right corner of the screen.
  4. Select Advanced from the menu across the top.
  5. Select Port Forwarding from the menu on the left.
  6. Select Custom Port Forwarding.
  7. Enter the IP Address of your Xbox console in the Server IP Address field.
  8. For all 4 port fields enter 3074.
  9. Set the protocol to both if available, if not create two rules one with UDP selected and another with TCP.
  10. Click the Save/Apply button.
  11. Select Tools from the tabs across the top.
  12. Select System Commands from the menu on the left.
  13. Click the Save All button.

 

Disable Port Forwarding

  1. Open a web browser and type http://192.168.1.1 .
  2. Enter admin for both the Username and Password.
  3. Click the Login button found in the lower right corner of the screen.
  4. Select Advanced from the menu across the top.
  5. Select Port Forwarding from the menu on the left.
  6. Select Custom Port Forwarding.
  7. Select any entries regarding port 3074.
  8. Click the Remove button.
  9. Select Tools from the tabs across the top.
  10. Select System Commands from the menu on the left.
  11. Click the Save All button.
