considerate_guy say:

A Movie Maker project will list the explicit path to each of your source media files. The next time you open the project file (.wlmp) Movie Maker will double-check that that path still exists and will then reload the images, videos and music files.

However, in your case, it seems like you have got one or more items that have been moved, renamed or deleted. In this case you'll see a yellow triangle with an exclamation point. When you double-click on that icon, you'll be allowed to navigate to the correct location to fix the reference. It might start in the folder where it thought the media was, or if it can't find that folder, it may just start you in the Pictures folder. In any case, it is up to you to then find the file again.

Once the references are fixed, you should immediately resave the project file. The next time you reopen the project, it should not have the yellow triangles and you shouldn't have to "fix" the files. If you find this happening frequently, then you need to figure out a better way to organize your media files so you aren't moving them around before a project is complete. The Movie Maker project just contains a *reference* to the files, not the actual media themselves. So it is up to you to not disturb the files if you are still using them in a project.
