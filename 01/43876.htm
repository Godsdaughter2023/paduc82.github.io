Presumably you don't have a Recovery Drive for Windows 7?

If you do, then boot from that.

If you don't, then there are two possibilities.

Go into the BIOS on start-up and check that Windows Boot Manager is showing as the default Boot Device.

If it's not there, then AFAIK there's only two methods of recovering data (if in fact the data is still there)

The first is probably the easiest.

You need to burn a Live Linux CD on another computer, and boot from it. When it's fully booted you can then use the file explorer to copy all or any of your data to an external drive.

Full instructions here.

The second way is to remove the HDD and either connect it to another computer via a Caddy (easier) or install it as a slave in the second machine (harder and only works if there's a spare bay).

Once you have removed any data you require, re-install Windows.
