a. user say:

Without a doubt, the safest and best way to install an operating system is with a clean install. That means erasing your hard drive and installing everything from scratch. It's like the difference between making brownies from a mix and from scratch. There's no comparison.

On the other hand, a clean install is one of most technically advanced things you can do with a computer, so you've got to have the knowledge and experience to do it.

An upgrade also requires a lot of technical expertise - people don't realize that - but it's not quite as difficult as a clean install. On the other hand, an upgrade is a crapshoot while a clean install on supported hardware is a sure thing.

That brings me to my last point: If your hardware isn't supported on Windows 10, an upgrade and a clean install will both fail, either immediately or in the future.
