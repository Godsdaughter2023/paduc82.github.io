1sweber say:

Win 7 Pro Drive RePartition Suggestion

I'm posting this as potential additional options with Win 7 Pro users to repartition drives.

I had a similar issue. Using the Windows 7 tools, I found the option to delete the logical D: drive grey'ed out.  I had to:

1. Set the default PAGEFILE to the C: drive

2. Set the PAGEFILE on D: to nothing/remove

3. Restart the computer

4. The option to Delete the logical D: volume is available.

5. The option to Resize/Extend the logical C: drive then becomes available.  Note: you may want to leave approx. 30Gb to create a new logical D: partition/drive during the resize/extend config.

6.  Create a new logical D: partition/drive with the remaining space

7. Re configure the PAGEFILE to the logical D: partition/drive to be managed by the system

Note: copy/backup any data on D: - pagefile doesn't require backup. And backup C: drive for recourse unless you're comfortable navigating task.  I didn't get into the specifics steps/tasks and just abbreviated/summarized above.

Hope this is of use to some.
