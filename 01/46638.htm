Hi Mark,

Code 19 indicates that there is an error in the registry of your operating system and it causes problems with hardware devices connected to your system.   2. Select Scan for hardware changes to reinstall the device driver.

If reinstalling the driver did not resolve the issue, we may try to simply roll it back to the last good configuration. To do that, you may follow these steps:

  1. Right-click the Start button and select Device Manager.
  2. Right-click your sound driver, select Properties, and browse to the Driver tab.
  3. Press the Roll Back Driver option if available, and Windows 10 will begin the process.

If the issue persists, try to install the driver in compatibility mode. You may refer to the steps below:

  1. Right-click on the driver and select Properties.    
  2. Select Compatibility tab.    
  3. Place a check mark next to Run this program in Compatibility mode and select the operating system accordingly from the drop-down list.
  4. Let the driver install, and then check the functionality.

•Ntuser.dat.log
•Ntuser.ini

j) Right click inside the folder and select Paste.
