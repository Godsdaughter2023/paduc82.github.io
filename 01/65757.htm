wunders say:

Keep in mind that an XP Home machine forces "Simple File Sharing" to always be enabled.  This means thatany connection to this machine from any other machine on the network will automatically authenticate as the "Guest" user -- no matter what user they really are.  The Guest user account *must* be "active" (but not necessarily "enabled") to connect.  Make sure your guest account is active by entering the following command from a command window:

   net user guest /active:yes

For more info, see the following Microsoft article:

"How to configure file sharing in Windows XP"
  <http://support.microsoft.com/kb/304040>

Read the "Known Issues" section near the end.

15. UPGRADES. To use upgrade software, you must first be licensed for the software that is eligible for the upgrade. Upon upgrade, this agreement takes the place of the agreement for the software you upgraded from. After you upgrade, you may no longer use the software you upgraded from.

So, if you are still denied, you will have to simply reinstall Windows XP or Vista and let it do the eligibility check.

 

Note: You will not need to activate the previous license, just install it.

Hi Avipan,

 

Welcome to Microsoft Community Forums.

 

Based on the description, it appears that you are getting an error message "Server execution failed” while trying to play videos.

 

1. How are you trying to play videos? Are you using Windows Media player or any third party media player?

2. Have you made any changes to the computer prior to this issue?

 

If you are using Windows Media player , you may follow the methods mentioned below.

 

Method 1:
Run the Windows Media Player settings troubleshooter and check if it helps.
Open the Windows Media Player Settings troubleshooter:
http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter
 

Method 2:

Add Local Service to Administrators group using commands

a) Click Start Button, type "cmd" in the search bar. The CMD icon will appear in the Start Menu.

b) Right click the CMD icon and click "Run as administrator".

c) In the popup window, type the following command line and press Enter.

NET LOCALGROUP administrators "NT Authority\Local Service" /ADD

You should receive a message “The command completed successfully”.

 

Method 3:

Restart Windows Media Player Network Sharing Service

a) Click the Start Button, type "services.msc" (without quotes) in Start Search box, and press Enter.

b) Double click the service "Windows Media Player Network Sharing Service".

c) Click on the "General" tab; make sure the "Startup Type" is "Automatic" or "Manual". Then please click the "Start" button under "Service Status" to start the service.

d) Click OK.

If it has been started, please stop it, restart the computer and then restart the service.

For reference :

http://social.technet.microsoft.com/Forums/en-US/w7itpromedia/thread/f690ff23-c486-47e7-acd4-ee6187613ae8/

 

Method 4:
Try Uninstalling and Reinstalling the Windows Media Player and check if it helps.
Step 1:
Uninstalling the Windows Media Player:
a. Go to Start and in the search type "Turn Windows features On or Off".
b. Click on "Turn Windows features On or Off".
c. Browse to the Media Features and uncheck the mark in front of Windows Media Player.
d. Restart the computer
Step 2:
Reinstalling the Windows Media Player:
a. Go to Start and in the search type "Turn Windows features On or Off".
b. Click on "Turn Windows features On or Off".
c. Browse to the Media Features and place a check mark in front of Windows Media Player.
d. Restart the Computer.


Give this a go; I’m confident you will find what you need here. But if you face any further issues regarding Windows, please reply back to this post and we'll be happy to assist you further.
