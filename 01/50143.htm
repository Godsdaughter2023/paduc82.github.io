alan russell say:

I had this problem under Windows 8.1, but I solved it and the solution appears to have carried over to my Windows 10 upgrade. Please bear in mind I am using Seagate external hard drives and not Western Digital, so there might be differences.

1. I used the Seagate included software to set the time for the disk to power down. I don't need the disk to be permanently spinning and wearing out bearings.

2. I went through Device Manager/Universal Serial Bus controllers. On every entry for Generic USB Hub and USB Root Hub I de-selected, on the Power Management tab, Allow the computer to turn off this device to save power. In my case this was two Generic USB Hubs and two USB Root Hubs.

3. I went into the Power Plan I was using and into Advanced power settings. Under Hard disk, I set Turn off had disk after to never. Then I went to USB settings and set USB selective suspend setting to disabled.

After I had done all this (and it might be over the top) I had no more trouble with my back-ups. Again it might be different for you as I am using Macrium Reflect.
