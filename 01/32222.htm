That is not a problem.

That is what an event 7026 looks like when you boot into Safe Mode with networking.

See:

Event Type:    Error
Event Source:    Service Control Manager
Event Category:    None

Description:
The following boot-start or system-start driver(s) failed to load:
AFD

MRxSmb

NetBT

For more information, see Help and Support Center at http://go.microsoft.com/fwlink/events.asp.

This is followed by more "errors" since you have chose to boot in Safe Mode as the drivers are not being loaded in Safe Mode.

Your log says you also have some kind of Toshiba system with Avira and Zone Alarm installed - that is probably your real problem!
