Hi,

I understand that you are facing issues with audio after upgrading to Windows 10. Sorry for the inconvenience caused.

Few more details would help us to provide a better response.

1. "I tried deleting the Speakers and Digital Audio", did you re-install the audio drivers?

2. "I tried going into Control Panel -- then Troubleshooting -- then All. This screen froze and then eventually crashed", do you get any error message?

3. "My computer has no audio output.", Is this the error message that you receive? are you using headphones?

You may follow the below methods and check.

Method 1: You may run the SFC and DISM by following the below link and check if it help to fix the crashing.

SFC: System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

DISM: It detects incorrect manifests, cabinets or registry data, it may replace the incorrect data with corrected version available.

http://answers.microsoft.com/en-us/windows/wiki/windows_10-update/system-file-check-sfc-scan-and-repair-system-files/bc609315-da1f-4775-812c-695b60477a93

Method 2: After performing Method 1, I suggest you to run “Playing audio” troubleshooter to diagnose and fix the issue.

Follow these Steps to run Playing Audio Troubleshooter in Windows 10.

a. Press “Windows Key”.

b. Type “Troubleshooter” in the Search box and hit enter.

c. In the Troubleshooting window, tap or click View all.

d. Select “Playing Audio”.

e. Follow the On Screen instructions.

Method 3: If the issue still persist, let us restart the Windows Audio and Windows Audio Endpoint Builder service and check if it helps.

1.Press Windows key + R on the keyboard, and type Services.msc and press Enter.

2. Double click on the “Windows Audio” and click on stop.

3. Once the service is stopped, you may click on start.

4. You may perform the same steps for Windows Audio Endpoint Builder.

Reference:

http://windows.microsoft.com/en-us/windows-10/fix-sound-problems#v1h=tab01

You may also install all the pending updates and check if it helps.
