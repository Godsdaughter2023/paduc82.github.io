j w stuart say:

By default, if you create a 'System Image' all the folders and files contained in the Windows partition are backed up. However if you use the 'Restore my files' or 'Restore all user's files' option you will not be able to access anything contained in the 'default' folder. To restore that folder you must use the 'System Restore disc' and perform a complete restore using one of the Image Backup you created.

You can change what is backed up and available for file and folder restoration by using the 'Change settings' option to include the Users/default folder. Now when you click on the 'Backup now' button that folder will be included.

pa bear - ms mvp say:

Note: This nearly three-year-old thread did not concern a clean install of Windows.

The "scenario" will repeat"weekly or every few days" unless you reset Windows Update components (which effectively removes all pending updates from the installation queue), reboot, and then immediately change your AU configuration to e.g, the "Notify Only" setting.

TIP => Most Windows computers come with an OEM-preinstalled Norton or McAfee free-trial. When you do a clean install of Windows using the hidden Recovery partition or the OEM-supplied disks, the free-trialware will be installed again (but invalid now). You MUST (a) uninstall the Norton  or McAfee application AND THEN (b) download/run a fresh copy of the Norton Removal Tool or the McAfee Consumer Products Removal Tool & reboot (c) BEFORE you install your own, third-party anti-virus application or security suite (e.g., Microsoft Security Essentials) AND BEFORE you attempt to install any updates (including SP1 & IE11).

Hey there Kneel thanks for keeping in touch.

We apologize for the inconvenience but rest assured we are here to help. We suggest a reset of the console since the problem might be on the actual OS of the console.

  1. Unplug the network cable if you're using a wired network connection.
  2. Power off your console, and then unplug the power cord to ensure that the console is completely powered off.
  3. Wait 30 seconds, and then plug the power cord back in.
  4. Press and hold the BIND button (located on the left side of the console) and the EJECT button (located on the front of the console), and then press the Xbox button on the console.
  5. Continue holding the BIND and EJECT buttons for 10-15 seconds.
  6. Listen for two “power-up” tones a couple of seconds apart. You can release the BIND and EJECT buttons after the second power-up tone.

From there you can reset your console. Give it a shot and let us know how it goes!
