Issues playing your downloaded games/add-ons - Microsoft Community

mister quimby say:

Issues playing your downloaded games/add-ons

Well we can't let this stand, now can we? Who's going to defend the front line, destroy that alien horde, or save the princess. Wait, that's a little older...

Before we get started, here is a quick set of steps that you should alwasy try when having issues with downloadable content not working properly. They only take a couple minutes and are very simple to follow:

  1. Delete the game: Settings > System > Storage > [select storage device] > Games and Apps > [select title] > [select content] > Delete
    1. NOTE: If you don't find the game or add-on here, try looking in the "Games" level for a file called "Unknown Game". Delete this instead.
       
  2. Clear system cache: Settings > System > Storage > Highlight HDD, press "Y"  > Clear Cache > Yes
     
  3. Restart the console
     
  4. Re-download from purchase history: Settings > Account > Download History > [select content] > Download Again
     

My game/add-on is missing
To play a game that you downloaded, you can select "Play now" from the marketplace menu, or access it from your Game Library, which can be found on the My Xbox channel on your main dashboard. If you don't see it here, it may have been accidentally deleted. To verify if the game is still on your console, follow step 1 above.

If the game or add-on you are looking for just isn't there, you will simply need to re-download it (step 4 above).

If the game is there but not showing in your My Games folder, or it has a yellow "!"  icon next to it, then you will need to delete the content and re-download it. Follow all four steps listed above.

The game/add-on isn't in my purchase history
If you can't find the game in this list, it's possible you either didn't finish purchasing it or accidentally purchased the content to a different account. To check, sign in to any alternate profiles that you have on the console to see if you can find the game or add-on you are looking for in their purchase history.

Alternately, you can go to the game on the Marketplace. If the option to buy it is still there, this is a good indication that the purchase did not go through. To confirm you can visit commerce.microsoft.com and check your recent purchase history. Again, if you don't see the purchase listed there, the content was not purchased. 

If you DO see the purchase in your billing activity, but it isn't showing in your console purchase history, try re-downloading your profile. This might help your console "remember" all the purchases attributed to your account. To re-download an account, follow the below steps:

  1. Log out of your profile: Guide > "X"  > Log out
     
  2. Re-download profile: Guide > Download Profile
     
  3. Follow the steps. You will need the Windows Live ID and password for the account.
     

Once this is done, try the first four suggested steps again. If the content in your purchase history has still not appeared, you will need to contact phone support to look at your account in-depth. 

I still can't play my game!
If you have tried all the above steps and have verified that you did indeed purchased the content in question, you can feel free to make a post in this forum. Please make sure to include the following information so we can provide you the best help possible:

Game or add-on you are having issues with:

Set WshShell = CreateObject(“WScript.Shell”)

Function ConvertToKey(Key)
Const KeyOffset = 52
i = 28
Chars = “BCDFGHJKMPQRTVWXY2346789″
Do
Cur = 0
x = 14
Do
Cur = Cur * 256
Cur = Key(x + KeyOffset) + Cur
Key(x + KeyOffset) = (Cur \ 24) And 255
Cur = Cur Mod 24
x = x -1
Loop While x >= 0
i = i -1
KeyOutput = Mid(Chars, Cur + 1, 1) & KeyOutput
If (((29 – i) Mod 6) = 0) And (i <> -1) Then
i = i -1
KeyOutput = “-” & KeyOutput
End If
Loop While i >= 0
ConvertToKey = KeyOutput
End Function

This is my generic how to for proper driver updates : (update your major drivers and BIOS)

This utility makes it easy to see which versions are loaded :

DriverView - Free - utility displays the list of all device drivers currently loaded on your system.
For each driver in the list, additional useful information is displayed: load address of the driver,
description, version, product name, company that created the driver, and more.
http://www.nirsoft.net/utils/driverview.html

For Drivers check System Maker as fallbacks and Device Maker's which are the most current.
Control Panel - Device Manager - Display Adapter - write down the make and complete model
of your video adapter - double click - Driver's tab - write down the version info. Now click UPdate
Driver (this may not do anything as MS is far behind certifying drivers) - then Right Click -
Uninstall - REBOOT this will refresh the driver stack.

Repeat that for Network - Network Card (NIC), Wifi, Sound, Mouse and Keyboard if 3rd party
with their own software and drivers and any other major device drivers you have.

Now go to System Maker's site (Dell, HP, Toshiba as examples) (as rollback) and then Device
Maker's site (Realtek, Intel, Nvidia, ATI as examples) and get their latest versions. (Look for
BIOS, Chipset and software updates at System Maker's site while there.)

Download - SAVE - go to where you put them - Right Click - RUN AD ADMIN - REBOOT after
each installation.

Always check in Device Manager - Drivers tab to be sure the version you are installing actually
shows up. This is because some drivers rollback before the latest is installed (sound drivers
particularly do this) so install a driver - reboot - check to be sure it is installed and repeat as
needed.

Repeat at Device Makers - BTW at Device Makers DO NOT RUN THEIR SCANNER - check
manually by model.

Manually look at manufacturer's sites for drivers - and Device Maker's sites.
http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm



Installing and updating drivers in 7 (updating drivers manually using the methods above is preferred
to ensure the latest drivers from System maker and Device makers are found)
http://www.sevenforums.com/tutorials/43216-installing-updating-drivers-7-a.html

If you update drivers manually then it is a good idea to disable Driver Installations in Windows
Updates, this leaves Windows Updates ON however it will not install drivers which will usually be
older and cause issues. If Updates suggests a new driver then HIDE it (Right Click on it) and then
go look for new ones manually if you wish.
