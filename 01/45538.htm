You can create a Windows System Image on a set of DVDs.

About a system image:

From Microsoft: http://windows.microsoft.com/en-US/windows7/What-is-a-system-image

"A system image is an exact copy of a drive. By default, a system image includes the drives required for Windows to run. It also includes Windows and your system settings, programs, and files. You can use a system image to restore the contents of your computer if your hard disk or computer ever stops working. When you restore your computer from a system image, it's a complete restoration—you can't choose individual items to restore, and all of your current programs, system settings, and files are replaced with the contents of the system image."


That means even if your computer gets fully corrupted, you can restore your computer from the system image as it was before - without worrying about anything such as activation issues.


Creating a system image in Windows 7-

If you are creating a system image on a set of DVDs, please insert one blank DVD before doing the following-

Click Start Orb (previously called Start button) > Control Panel. Select System and Security > Backup and Restore. From the left pane, click Create a system image.

Select the desired option, however since you say "your hard drive is starting to die out", I would recommend 'On one or more DVDs' option.

After selecting the desired option, click Next and follow the instructions on-screen.

Hi,

There are a few options that you can try to reinstall Microsoft Photos on your computer, such as updating Windows or updating photos app. You can check out the following steps for assistance. 

To update Windows

  1. Go to the Settings app.
  2. Updates & security, and check for updates.
  3. If the fixing patch is available, it will be installed on your computer.

Windows updates should be installed automatically, but you can also check manually if you don’t want to wait for the system to do it.

Update the Photos app

The same thing applies to the app itself. If there’s some component of the Photos app that conflicts with your system, developers will probably find a solution for it. But instead of Windows Update, the fix might be delivered to you through the app update.

To update Windows 10 Photos app, simply go to the Windows Store, open the Store Page of Windows 10 Photos, and check for updates. If there are new updates available, they’ll get installed on your computer automatically.
