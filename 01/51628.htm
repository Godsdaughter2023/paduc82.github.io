Hi Ravi,

 

Welcome to Microsoft Community and we are glad to assist you.

 

As per the issue, you get a blue screen error when the computer frequently shuts down prompting “UNEXPECTED KERNEL MODE TRAP”. After removing AVG Antivirus from the computer you get a bug check error code “0x0000007f (0x0000000d, 0x00000000, 0x00000000, 0x00000000”.

 

Bug check error code 0x0000007f comes up for the following reasons:

 

•Your computer has hardware or software problems (hardware failure is the most common cause).
•You try to over clock the speed of your computer's processor (for example, you set a 150 MhZ processor to run at 187 MhZ).
The above STOP error means a trap occurred in kernel mode and the trap is either one the kernel is not allowed to have or is always fatal. The most common causes of a STOP 0x7F are:•Low-level hardware corruption, such as corrupt memory (RAM)
•Mismatched memory modules
•A malfunctioning motherboard
