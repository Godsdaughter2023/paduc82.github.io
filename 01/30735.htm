considerate_guy say:

Are you trying to import the *movie* or the *project*?

The project file (.wlmp) is not a movie. It's just a text file that contains the editing state of your project. It contains a *list* of all the media elements in your project along with the trim points, effects applied, etc. But it has no media within it. Hence it is NOT a movie.

In order to create a movie, you must save the movie using File > Save movie. This will create a .mp4 or .wmv file. I believe Expression Encoder 4 prefers .wmv files, so you might try changing the output format to "Windows Media Video (.wmv) when saving the movie.

The *movie* should import fine into Expression Encoder 4.
