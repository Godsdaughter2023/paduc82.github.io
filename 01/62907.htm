    Start > Run > regedit
    Navigate to HKLM\SOFTWARE\(Wow6432Node\ on x64 systems)MICROSOFT\RPC\INTERNET
    Export this key to back it up first, then delete the whole Internet key (and it's sub-keys).
    Reboot your system
    Task Scheduler service and Task Scheduler application worked just fine after reboot.

    Reference: http://social.technet.microsoft.com/Forums/en-US/itprovistasp/thread/0c438376-1486-4ae4-9847-2de7a8767f27/
