vfoxtech say:

I just ran into the same issue troubleshooting the same model of printer and was able to find a solution.

When using the Kodak All-in-One Printer Home Center Software (the only driver download that Kodak offers) it would install the drivers for everything but the printer and fail out after a while saying it couldn't communicate with the printer.

Looking behind the scenes it seemed like it was trying to get a driver from Windows update but wasn't able to.

Here's what I did to resolve the issue:

Went into the Device Manager in Windows (here's a guide for that: http://pcsupport.about.com/od/windows7/ht/device-manager-cp-windows-7.htm)

Found the Eastman Kodak Hero AiO 7.1 under Other Devices

Right clicked on the printer and clicked on Update Driver Software

Then clicked on 'Search automatically for updated driver software'

Took a little while to find the driver through Windows Update but eventually it found the driver and installed it.

One other thing I did that may have helped was to go into Add Printer through Devices and Printers, clicked Add a local Printer, chose USB from the Use an existing port option, then clicked the Windows Update button on the Install the printer driver page. This goes through and grabs all of the printer drivers from Windows Update (takes some time) and after it finished it had the option for the Hero AiO 7.1 XPS under Eastman Kodak.

Hope this helps anyone that's run into the same issue.
