To reset your Surface from a USB recovery drive:
Step 1:    Make sure your Surface is turned off and plugged in (hold down the power
button for 30 seconds).
Step 2:     Insert the USB recovery drive into the USB port on your Surface.
Step 3:    Press and hold the volume-down button while you press and release the
power button on your Surface.
Step 4:     When the Surface logo appears, release the volume button.
Step 5:    When prompted, select the language and keyboard layout you want.
Step 6:     Select Troubleshoot > Reset your PC.
    If prompted for a recovery key, select Skip this drive at the bottom of
the screen.
Step 7:     Select the target operating system you wish to reset. This
refers to the current operating system installed on your Surface.
Step 8:    Select Yes, repartition the drives, and on the next page, select Next.
Step 9:     Choose Just remove my files.
Step 10:    Select Reset.
    Surface restarts and the Surface logo displays while the reset process
continues (this can take a while).

Windows 7 Add a device wizard fails with code 0x80070643

When I use the Add a Device Wizard to add a Wi-Fi network device, I get an error with code 0x80070643. The device cannot be added to the computer. However, the device does When I use the Add a Device Wizard to add a Wi-Fi network device, I get an error with code 0x80070643. The device cannot be added to the computer. However, the device does appear to be added to the Wi-Fi network. What should I do?
anannya podder say:

To resolve this problem, follow these steps:

1.

Run the Add a Device Wizard and try pairing the device again.

2.

Make sure that the device is connected to the network. To do this, follow these steps:

a.

Click the  Start button, and then click Computer.

b.

In the navigation pane, click Network .

If you can see an icon for the device, it is connected to the network. Try pairing the device again by running the Add a Device Wizard.

If you cannot see an icon for the device, then it is not connected to the network. Try one or more of the following methods:

•

If the device is configured for the wrong network, it will not connect, and this error may occur. Put the device back into Wi-Fi pairing mode (also known as Wireless Protected Setup [WPS] mode), and then run the Add a Device Wizard again. Make sure that you specify the correct network profile in the wizard.

•

Check firewall settings on the computer and on the Wi-Fi router or the access point. Make sure that the SSDP (UDP 1900) and WS-Discovery (TCP/UDP 3702) protocols are not blocked.

•

Manually configure the device's Wi-Fi settings. For information about how to configure the settings, see the documentation for the device.

•

Make sure that the Wi-Fi router or the access point is running and functional.

•

Turn off the device, and then turn on the device.

•

Connect the device to the network by using a network cable. You may have to turn off Wi-Fi support on the device. If you now see an icon for the device, then there is a problem with the device's Wi-Fi configuration. Put the device into Wi-Fi pairing mode, and then run the wizard again. For information about how to turn off Wi-Fi support on the device, refer to the device's documentation.
