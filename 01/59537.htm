Here are some troubleshooting methods that can help you resolve the issue:

Run the Bluetooth troubleshooterYou will need administrative permission to run this tool. This tool will scan and fix your PC for any issues found with your device drivers.

  1. Right-click the Start button, select Settings > Troubleshoot.
  2. Select Bluetoothunder Find and fix other problems.
  3. Click Run the troubleshooter.
  4. If prompted, click Yes or enter the administrator password.
  5. Follow the on-screen instructions.

Reinstall the Bluetooth adapter in Device Manager

  1. Click the Start button, type Device Manager, and then hit Enter.
  2. In Device Manager, select Bluetooth. Right-click, the Bluetooth adapter name, which may include the word “radio”, and select Uninstall device.
  3. Shut down your computer.
  4. Wait for at least 30 seconds, and then turn the device back on. Windows will try to reinstall the driver.
  5. If Windows doesn't reinstall the driver automatically, open Device Manager and select Action > Scan for hardware changes.
