  1. Use Backup and Restore in the Control Panel to create an image file.
  2. Park it on an external hard disk.
  3. Use Backup and Restore in the Control Panel to burn a Repair CD.
  4. Boot the machine with this CD.
  5. Check if it can discover the image you made in Step 1.
