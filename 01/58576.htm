androidbeat say:
I figured out myself no thanks to the Microsoft Answers community. For those who also need help, download a MSWINSCK.OCX file. Then drag it into SysWOW64. Then go to command prompt and run as administrator and type in regsvr32 c:\windows\syswow64\MSWINSCK.OCX. It is caps sensitive so if your MSWINSCK.OCX is lowercase like this: mswinsck.ocx then you have to put mswinsck.ocx with lowercase when typing into the command prompt. This works for registering all .ocx files.
