I hate to tell you this, but the first thing you should do when you get somebody else's computer is a clean install. That's the only way you can know 100% that there are no errors or problems. That's extremely important if you're thinking of upgrading to Windows 10.

If you're thinking of installing Windows 10 on a computer that didn't originally come with Windows 10, and particularly a laptop since they frequently use proprietary hardware, you need to check whether the hardware is being supported on Windows 10. If the hardware manufacturer has decided not to support the laptop's hardware, it's only a matter of time before trouble comes.

Here's what I would do in your shoes:

2b - If it's not supported, you need to ask yourself that question from Dirty Harry: "Do I feel lucky?" Otherwise, do a factory restore, which will return the computer to the way it was when it left the factory.

You need to make sure that the address used in the link actually points to the correct place.

What program (word processor) are you using to create your family journal?  Please include the version (e.g., Microsoft Word 2016).

Assuming that you are using a relatively recent version of Microsoft Word, open the copy of the document that has been saved on the USB flash drive, right-click on the link to the video and select Edit Hyperlink.

Edit the link so that it points to the copy of the video file on the USB flash driver rather than to C:\Users\<yourusername>\Videos\familyvideo.mp4 or the like.

If you have a lot of these links in the document, see (again, supposing you have some version of Word) https://wordribbon.tips.net/T010022_Changing_Many_Link_Locations.html

lemp say:

Did you do steps 9, 10, and 11?

Pressing ALT+F9 toggles between displaying the field codes and not displaying them.

For example, assume that your user name is Bob and you have 3 video files saved in your "My Videos" folder.  The paths to the files are:

C:\Users\Bob\Videos\movie1.mp4

C:\Users\Bob\Videos\movie2.mp4

C:\Users\Bob\Videos\movie3.mp4

After you insert these links into your document, when you press ALT+F9, you see:

{ HYPERLINK "C:\\Users\\BOB\\Videos\\MOVIE1.MP4" }
{ HYPERLINK "C:\\Users\\BOB\\Videos\\MOVIE2.MP4" }

{ HYPERLINK "C:\\Users\\BOB\\Videos\\MOVIE3.MP4" }

Now move the document and the 3 movie files to the USB flash drive which is drive F:

Open the copy of the document on the flash drive and press ALT+F9 (if the field codes are not already visible)

Open the Find/Replace dialog and replace

    C:\\Users\\BOB\\Videos

with

     F:

You should now see:

{ HYPERLINK "F:\\MOVIE1.MP4" }
{ HYPERLINK "F:\\MOVIE2.MP4" }
{ HYPERLINK "F:\\MOVIE3.MP4" }

Or you could simply replace C:\\Users\\BOB\\Videos with nothing, to get "relative" links, which also should work:

{ HYPERLINK "MOVIE1.MP4" }
{ HYPERLINK "MOVIE2.MP4" }
{ HYPERLINK "MOVIE3.MP4" }

With the changed field codes visible, select all of them (CTRL+A) and press F9 to update them.  Then press ALT+F9 to no longer show the field codes and save the document.
