Please confirm that you used the Remove account ... option to remove an account from the folder pane of Windows Live Mail.

If you did so, you would have seen a warning similar to this one:

... so you have to confirm that you want to delete permanently all the messages in the account folders.

You may be able to recover some of what was 'permanently' deleted. Your store folder is by default this one:
 

"%localappdata%\Microsoft\Windows Live Mail"
 

You can copy that black line, then press Windows key + R to open a Run box, press Ctrl-V to paste into the box and then press Enter to open Explorer at your store folder. Right-click on the folder name in the navigation pane and select Restore previous versions. Do you have any previous versions?

  1. If you can locate a previous version that predates the removal of the account, Open it and then locate the missing folders.
  2. Copy the folders into a temporary folder on your desktop.
  3. Use the File > Import messages function (Alt-F I) in Windows Live Mail to import the temporary folder.

Otherwise, you're left with having to use an 'undelete' utility like Recuva to search for and retrieve deleted .eml files from your hard disk. The quicker you do this, the better.
