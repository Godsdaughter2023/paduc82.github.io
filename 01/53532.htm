Import photos in Windows 8

Import photos and video from your   device to your PC:

  1. Connect your iOS device to your computer.
  2. Move the cursor to the upper-right corner and click the magnifying glass icon.
  3. Type Computer and press Return.
  4. Right-click the logo for your device and click "Import pictures and videos."
  5. Click More Options.
  6. Click the Browse button next to the "Import images to" field and choose where to import your Camera Roll's photos. By default, these will import to the My Pictures folder.
  7. Select "Delete files from device after importing."
    • If you delete photos after you import them, you can reduce the size of your Camera Roll and improve backup and restore times.
    • If you wish  to import the photos to other computers, don't select this option.
    • If you don't want Windows to delete photos from your camera on subsequent imports, deselect this option when you're finished.
  8. After setting your import options, click OK to dismiss the window.
  9. Click Import.
  10. After the import is complete, Windows will display your photos.

When you see message like this in the log that begin with (f):

(f)    CSI Payload File Missing    0x00000000    dmocx.dll    amd64_microsoft-windows-pnpdevicemanager_31bf3856ad364e35_6.1.7600.16385_none_6fcb8c193931bf54   

If they are not followed by another like that says (fix) for example:

(fix)    CSI Payload File Missing    0x00000000    dmocx.dll    amd64_microsoft-windows-pnpdevicemanager_31bf3856ad364e35_6.1.7600.16385_none_6fcb8c193931bf54   

The first part of the message dmocx.dll is the missing file and the rest after that is the folder where file should be.

Assuming Windows is installed on your C drive in the Windows folder under that is another folder called winsxs (Windows Side by Side) and under that are lots of other folders with very long and curious names.

There are lots and lots of folders under the winsxs folder on your system and most only contain a few files.  When a file is missing, the SURT is going to complain about it and put it in the report.  The missing file is probably not the cause of whatever your issue is but you can replace them to keep SURT happy.

In the case of your first error, the message is telling you that the dmocx.dll file is missing from this folder:

c:\windows\winsxs\amd64_microsoft-windows-pnpdevicemanager_31bf3856ad364e35_6.1.7600.16385_none_6fcb8c193931bf54

You can navigate to the folder and check if the file is really missing using Windows explorer.

You should navigate to this folder on your system and see if the dmocx.dll file is missing:
