Hi Steve,

The 100 Kbps that is inline with the Throughput is a default settings for the network virtualization view of a user. It does not mean that your connection can only be limited on that speed. Your speed still depends from your network provider, as you've mentioned, you have 1 Gbps internet speed. This view only shows you how your internet connection is being utilized and successful delivery of your provider to your computer. You can visit a streaming website and watch it for a moment while observing the Ethernet Throughput. You will notice that that graph and the Throughput will change as you are utilizing your internet connection.

a. user say:

Hi Joe,

Error 1603 is generally displayed when the user account from which you are trying to install doesn’t have full permissions or if the location where you are trying to install is encrypted. The 1722 error message is a general installer error message that states that a system level failure has occurred. I would suggest you to log into an administrator user account and try these methods and check.

Method 1:

Try the steps suggested in this article to resolve the error 1722:

http://support.microsoft.com/kb/928080/en-us

Now, try installing the FSX and check if you still get the error 1603. If yes, then try method 2.

Method 2:

As explained earlier, error 1603 is generally displayed when the user account from which you are trying to install doesn’t have full permissions or if the location where you are trying to install is encrypted. In this case, I would suggest you to change the location where you are trying to install FSX and check if the issue persists.

If yes, then try the following steps and check:

  1. In File explorer, right click on the drive where you are trying to install FSX and select “properties’.
  2. In the properties window, click on the security tab and click on the ‘System’ option.
  3. Check if the system account have full permissions in the below list of permissions.
  4. If not, click on the ‘Edit’ button above and select the check box against ‘Full permissions’ to allow full permissions to that account.
  5. Save changes and try installing FSX now.

Try these steps and let us know the results for further assistance.
