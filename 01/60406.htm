Since it is a critical drive, it is not listed as a backup location because it will be included during an image recovery.

If you want to reset the System to the C: drive, you just need to make the partition active, and set that drive to first priority in the bios.  Disconnecting the other drive completely is the best way to make sure the boot files are set up on the correct partition.  Then run the Startup Repair, which is an option in the Advanced, inside the Advanced Repair options.  Note:  If the system goes into Automatic Repair, it may take three tries, but if it shows as needing a Restore, cancel and try booting normally.

Another option would be to run the command below, after making the partition active, to place the boot files on the C: partition.  This procedure would keep the system from entering Automatic Repair.

bcdboot C:\Windows /s C:

http://technet.microsoft.com/en-us/library/dd744347(v=WS.10).aspx

Using this command will probably require you to re-register the Recovery Options using Reagentc.exe
