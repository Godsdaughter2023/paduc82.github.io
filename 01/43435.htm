    Hi,

    The error "This Parameter is incorrect" usually happens because of a system or disk errors. To isolate this issue, we suggest running CHKDSK using the command prompt. Running CHKDSK can check the integrity of a hard disk and can repair various file system errors. To run CHKDSK follow these steps:

    1. Plug in external hard drive, USB drive or memory card to the computer.
    2. Press Windows Key + X and then select Command Prompt (Admin).
    3. In the Command Prompt window, type chkdsk E: /f /r. You should replace E with the drive letter of your external hard drive or external device.
    4. Restart your PC once the process is done.

    - It's a common misconception that Win7 has only DX 11. Win7 has three versions of DX - 9,10 and 11.

    They are separate entities and do not conflict with one another.

    However, a new Win7 install does not have all of the DX files, so it is important to run the
    DirectX Web Installer to bring the files up to date -

    Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

    - The following is a list of the Direct X .dll's you will have when the files are up to date.
    Go to the Windows / System 32 folder.  
    They are in alphabetical order and will start with d3dx9 - 24 > 43 . Then d3dx10 - 33 > 43 & finally d3dx11 - 42 > 43.
    There should also be - d3d9, d3d10 and d3d11 these come before the D3Dcompilers - 33>43.
    - There are also three new XInput1 - 1>3.
    There are more DX files, but these are most of the graphics related Direct X  .dll's

    If your Windows has all the files above it may be the files used by the games are corrupt. In this case
    it is best to use the full DirectX Redis (June 2010) installer package as this will overwrite all the DX files.
    The Web Installer does not overwrite files, it only installs 'missing' files.

    Download Details - Microsoft Download Center - DirectX Redist (June 2010)

    - Another reason for DirectX errors is if your graphics accelerator (either Integrated graphics or graphics card) does not meet the specifications of the game.
    - Outdated audio drivers can also cause DirectX errors in some cases.

    - A forum user a while back had a problem reinstalling DX using the Redist (June 2010) and did the

    following -

    "I had to run services.msc  turn off windows audio and windows audio manager and active virus protection. Then directx installed fine."
