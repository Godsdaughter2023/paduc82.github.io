Please follow the steps mentioned below to resolve the issue:

1.     Launch Windows Media Player.

2.     On the “Right Pane”, right click on “DirectTV2PC Media Server” and select “Manage Library”

3.     In the new window, highlight whatever you don’t want and select “Remove”

4.     Then rebuild the media library by following the steps mentioned below:

a.     Exit Windows Media Player.

b.     Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

c.      Select all the files in the folder, and then click Delete on the File menu.

d.     Note: You do not have to delete the folders that are in this folder.

e.     Restart Windows Media Player.

              Note: Windows Media Player automatically rebuilds the database.

 

If the above does not fix your issue then clear the Windows Media Player database cache file.

To do this, follow these steps:

a.     Exit Windows Media Player.

b.     Click Start, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.

c.      Select the Media Player folder, and then click Delete on the File menu.

d.     Restart Windows Media Player.
