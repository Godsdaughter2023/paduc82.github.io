smiling cat-fab725ba-e75f-400c-9250-ecd9955c9c75 say:

What you're getting is probably inductor coil whine/noise. When the power supply is plugged into an outlet and the 360 it sends a small amount of power to the console all the time. This is so it can wait for a controller to turn it on or charge devices connected to it. Thing is most inductors are meant for when a power supply is supplying the max amount of current the power supply is meant to output. So in low power it can start to resonate and you get a high pitch noise.

Anyways about the only thing I can think of is find a replacement or fix it yourself since it should still be fine for use probably longer than the console will. Simplest methods for dampening an inductor coil is usually non-conductive epoxy. Hot glue or a small piece of foam can also work if it doesn't heat up to much. If you do decide to try and fix it be really careful and give the unit plenty of time to discharge.

Could also just unplug the power supply or plug it into a surge protector you can turn off when it's not being used. You'll lose a couple features I mentioned, but won't hurt the console or power supply.

                To fix MSI software update registration corruption issues automatically, click the Fix it button or link. Click Run in the View Download dialog box, and then follow the steps in the Fix it wizard.

                 http://go.microsoft.com/?linkid=9666880

                Restart and install update.

Step 5: Start Windows in Selective Startup mode (Note: Do not perform this step if you are using a wireless internet connection. Instead just turn off your antivirus or antispyware program)

b.    On the General tab, click Selective Startup, and then clear the Load Startup Items check box.

Warning : The above steps will temporarily disable applications including your security software. Disabling antivirus or antispyware software might increase your security risk. Use the System File Checker tool (SFC.exe) to determine which file is causing the issue, and then replace the file. To do this, follow these steps:

a.     Open an elevated command prompt. To do this, click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

sfc /scannow
