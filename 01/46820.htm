How to Delete a Wifi Network Profile - Windows 7 or Vista - Microsoft Community

lucerova say:

How to Delete a Wifi Network Profile - Windows 7 or Vista

*These steps are applicable if you are having trouble connecting to a wifi network that you have previously connected to (such as a hotspot).

Please use these steps only if you are comfortable performing them and ONLY if you know the exact purpose for which you would delete a wifi profile. 

1. Locate and click on the Wifi Networks icon located in the lower right corner of the task bar.

2. Click on Open Network and Sharing Center.

3. Click on Manage Wireless Networks.

4. Click on the desired Wifi Network Profile.

5. Click Remove. (Remove is located immediately above the Wifi Network Profile List.)


6. Click Yes. (This will confirm removal of the wifi network profile.)

7. After the network profile has been removed just simply reconnect to wifi to the desired network. This will create an entirely new Network Profile

(In the Network and Sharing Center you can also change the Wifi Network Priority by selecting the network and selecting "Move Up" or "Move Down".)
