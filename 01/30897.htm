mark ole say:

Hi,

To address your concern, we suggest that you try the following methods to resolve this issue.

Method 1: Run the chkdsk tool

  1. Type cmd in the search box. Right click on Command Prompt and choose Run as administrator.
  2. Type chkdsk /f /r. This will check the drive for errors.
  3. Once prompted to schedule the check after a reboot. Type Y and hit Enter.
  4. Reboot and let the check proceed. It might take a while but will eventually tell you if the drive has errors or whether it is operating normally.

Method 2: Run the System File Checker tool.

  1. Type cmd in the search box. Right click on Command Prompt and choose Run as administrator.
  2. Type or paste ‘sfc /scannow’.
  3. Let the process complete and follow the on-screen instructions should it find any errors.

If the issue still persists, try fixing the master boot record. To do that, follow these steps:

  1. Boot from your Windows 10 installation media. (If you don't have one, refer to this link)
  2. Select Repair my computer  when prompted and then choose Troubleshoot.
  3. Select Command Prompt.
  4. Type bootrec /RebuildBcd and press Enter.
  5. Type bootrec /fixMbr and press Enter.
  6. Type bootrec /fixboot and press Enter.
  7. Type Exit and press Enter


Let us know how it goes after performing the recommended methods.
