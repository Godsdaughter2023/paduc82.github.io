How do you copy and paste? what is a clipboard? - Microsoft Community

jessicalamartz say:

How do you copy and paste? what is a clipboard?

I have no idea how to copy and paste. I can get the copy part but after that is what confuses me. what is a clipboard?

ramesh srinivasan say:

Clipboard is a temporary storage area in the memory (instead of disk drive), which can store images as well as text. To paste data from clipboard, use the Paste command (CTRL+V) which is supported by almost all applications and edit control boxes.

If the clipboard object is in text format, then open Notepad or any other text editor, and choose Paste. Note that to keep the source formatting intact (suppose you copy text from a website), you'll need to paste it in an editor that supports rich text capabilities.. like Microsoft Word. Pasting in Notepad strips off all the special text formatting.

If the clipboard object is an image, then open Paint (mspaint.exe), paste it and save the file.
