a. user say:

Hi,

 

App publishers sometimes update their apps to add new features and fix problems. The Windows Store can automatically install app updates when they become available. If the “Automatically update my apps” feature is already enabled, then, it could be an issue with the Windows Update services.

 

You can turn off the automatic app update feature and then restart it to troubleshoot the issue. Reefr to the link:

http://windows.microsoft.com/en-us/windows-8/automatic-app-updates-help

Note: After you turn off the feature, restart the computer and then turn on the feature.

 

If that does not work, let’s try stopping and restarting the services to try and fix the issue. Follow the steps:

a) Press Windows+R key to open run and type “services.msc”. Hit Enter.

b) Locate “Windows Update”, “Background Intelligent Transfer Service (BITS)” and “Cryptographic” services. Right Click on each service and click on Stop.

c) Restart the computer and then open “services.msc”. 

d) Right Click on each service and then click on Start.

e) Restart the computer and check if the update is getting installed.

 

If not, you can manually install the update as well. Right click on the update that is available and select install to do so.
