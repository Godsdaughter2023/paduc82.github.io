Actually, schtasks.exe is available on XP Pro, but not on XP Home. "sc.exe" associated with "Services", which is not the same. The equivalent command for XP Home is "at.exe". (see http://support.microsoft.com/kb/313289/en-us). Both "schtasks.exe" and "at.exe" create a scheduled task that "Run as: "NT AUTHORITY\SYSTEM". Suggest you use a batch file to create your "at" (and shctasks) jobs as this saves a lot of typing, and allows easy editing. The available documentation is a bit sparse. Note: there are caveats and limitations to these scheduled tasks. I have been unable to make it work to access across a network., even if it executes another batch file that will otherwise  access a network. Another thing is they can not be modified. Editing an "NT ATHORITY\SYSTEM" task may (or will) render it non functional.
