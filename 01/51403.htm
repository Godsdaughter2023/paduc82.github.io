slance310 say:

Hi,

Exactly what is happening... and when ?

I don't know if you are aware of the fact that newer motherboards can use two different types of BIOS... UEFI and Legacy...

They require different installation media and\or boot option selections...

It is a VERY bad idea to mix different memory of different brands and speeds...

I'm guessing you have 4 memory slots... Are you using slot #1 with a single stick ?

If that doesn't work, try slot #3...

You might try opening Boot Options at the manufacturer's screen with F9, F11, F12, or whatever your machines uses and select your flash drive from there... You should be able to choose from a "USB Flash Drive" or a "UEFI USB Flash Drive"...

If you created the flash drive with the Win 7 USB\DVD Tool, it will not work with your BIOS set to UEFI... It uses NTFS for the file type, and UEFI requires FAT32...

It is possible that your machine will automatically switch from UEFI to Legacy, or vice versa, depending on the selection of the installation media, but maybe not...

It is also possible that your CPU or motherboard has a bent pin or two causing a problem... That has happened to me before more than once... Even with new stuff..
