Filter Keys is a Windows feature that ignores keyboard input unless the key is held down for a significant period of time. It’s also triggered when you hold down a button such as SHIFT for 8 seconds.

Follow these steps to disable filter key

3. Click on change how your keyboard works.
4. Uncheck the option "Turn on filter keys."
5. Click Ok to save changes.
