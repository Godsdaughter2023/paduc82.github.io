Create the folder (with the same name) in the same location in the C:\ drive where it was before. And see if you can unpin it. Then you may delete that folder. That would be simple I guess.

Another option is to clear an .AutomaticDestinations-ms file located here:

%AppData%\Microsoft\Windows\Recent\AutomaticDestinations\

The file name you want to delete would be f01b4d95cf55d32a.automaticDestinations-ms.

This clears all the pinned items though.
