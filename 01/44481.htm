Skype is trying to connect to a landline number abut won't let me - Microsoft Community

clarejefferis say:

Skype is trying to connect to a landline number abut won't let me hang up. Been like this for over 5 mins...

It says connecting. But it obviously can't and I don't seem to be able to hang up. What can I try?

vijay a. verma say:
Try these basic troubleshooting steps first and check after each step and please update me. (Please note your data is stored on the cloud, hence you won't be losing data)

1. This issue may be due to the fact that caller ID is not setup. . If this is not setup, set it up.

What is caller identification and how do I set it up? - https://support.skype.com/en/faq/FA1248/what-is...

2. Reset the Skype
Windows Key+I > Apps > Locate Skype > Advanced Options > Reset
Go to Microsoft Store > Click 3 horizontal dots on top right > Downloads and Updates > Check if there is any update for this app > If yes, update it.

3. Uninstall and reinstall Skype app (Note - You will not lose any Skype data because of this)
Uninstall
Windows Key+X > Windows Powershell (Admin) > Copy the below, paste into Powershell > Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Remove-AppxPackage

Reinstall
Copy and paste below and Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

4. To isolate the issue, can you make call using Classic Skype (Download)

Download and install Classic Skype - https://go.skype.com/classic.skype
