Are you trying to tell me that there are no Restore Points on your machine? If so then a manual Registry Restore might help. The process is involved and requires some careful typing. Ask a computer-savvy friend to assist you if you lack experience in this area. Note that it may reset your password to an earlier value.
1. Keep tapping F8 during the early boot phase.
2. Select Repair from the menu that looks like this one:
   http://windows.microsoft.com/en-US/windows-vista/Advanced-startup-options-including-safe-mode
3. Log on as Administrator if prompted.
4. Select Repair when prompted.
5. Select the Command Prompt from the menu that looks like this one:
   http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7
6. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands
    dir C:\Win*
    dir D:\Win*
    dir E:\Win*
    until you find the Windows folder.
    I will call that drive "Q:".
7. Type these commands:
    cd /d Q:\windows\System32\config
    xcopy *.* Q:\MyRegBack\
    cd  RegBack
    dir
8. Examine the dates of the Software, System and SAM files. Were the files created before or after your problem became apparent?
9. If they were made before the problem became apparent then you can type these commands:
    copy /y software  ..
    copy /y System  ..
    copy /y Sam ..
    (the two dots are part of each command)
10. Reboot normally. If things do not work out then you can backtrack by copying the original registry files from Q:\MyRegBack to Q:\Windows\System32\config.

Webcam not working after update - Microsoft Community

carlos zanella say:

Webcam not working after update

After a recent update on my Windows 7, no webcams work any longer. The system can´t locate them, as if they weren´t connected.

I can´t reinstall the camera drivers, because I have a Microsoft LifeCam VX-500, and its drivers are supposedly a native part of Windows 7. I also tried with other cameras and none of them work either.
ralf_g say:
After a recent update on my Windows 7, no webcams work any longer. The system can´t locate them, as if they weren´t connected.

I can´t reinstall the camera drivers, because I have a Microsoft LifeCam VX-500, and its drivers are supposedly a native part of Windows 7. I also tried with other cameras and none of them work either.


You can always re-install drivers even if they are native in Windows. Go to Control Panel and open Device Manager. Look down the list for all references to your webcam (video, microphone, etc). They may show up with a yellow ! mark indicating there is a problem with the driver or they might not. If none show up at all try enabling "Show Hidden Devices" from the View menu. For each webcam entry that you find, right click on it, select Properties then Driver. Click on the button that says "Roll back driver". The previously working driver should then re-install.


If the Roll Back driver option is disabled that means there was no new driver installed. You can still reinstall the driver however. Right click on each webcam entry and select Uninstall driver. When that is done unplug your webcam. Reboot is optional, otherwise plug your webcam back in again (into a port on the computer NOT into a hub) and Windows should re-detect it and install the drivers again.


In case it was a driver update that caused your problem, when you have rolled back each of the webcam drivers and tested that the camera works again, go into Windows Update and search for updates. The problem driver should show up again. From the view updates screen (the one with the details about the updates) right click on the webcam driver update and select "Hide update". After that it shouldn't bother you again.
