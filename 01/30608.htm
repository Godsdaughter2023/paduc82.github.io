This can happen for a number of reasons:

¡firedog say:

One or more of the pictures in this message could not be found ...

This is a distressing bug that has affected some users of Windows Live Mail since the release of the 2011 version in 2010. It happens because the program stops being able to make the connection between the image placeholder in the body of the message and the actual attached file. It only happens to some incoming messages, including those that have at some stage passed through an AOL server, some that were composed in Word, some that originated on an Apple computer and so on. Despite having asked for and receiving thousands of examples, the support staff have never come up with an explanation, never mind a cure.

A search for your error message produced something like 200 threads about this problem in the Windows Live Mail forum. The only reliable solution is to click on the bottom half of the Forward button and choose Forward as attachment. This will send the message onwards exactly as you received it. It requires that the recipient can open .eml files.

Otherwise, right-click on each picture in the incoming message and select Save as to save the picture on your computer. Then, when you're composing the forwarded message, remove each image and use the Single photo option on the Insert tab to insert the picture you saved.
