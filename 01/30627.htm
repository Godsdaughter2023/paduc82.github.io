a. user say:

Code 22: This code means the device is disabled (even if you say it is not disabled).  Do you  know what device the error refers to - what is it (type of device, manufacturer, make, and model)?  What is the EXACT text of the message (including any codes)?

Do you know when this problem began?  Try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.  Use the recovery disk if the system prompt doesn’t work. The recovery disk works a bit different from the above procedures but if you follow the prompts from the System Restore menu option with the above information you should be able to restore with no problems.

To resolve this error code, enable the device (or in this case, re-enable the device). To do so, follow these steps:

  1. Select Start >  Control Pane > Device Manager.
  2. Double-click the category for the device you want to enable, and then double-click the device
  3. Disable it and the re-enable it again.
  4. In the Device Usage box, select the check box for the configuration in which you want to enable the device
  5. Click OK, and then click Close
  6. Shut down Windows, and restart your computer

If this doesn't work, try removing the device in Device Manager, then re-detecting it using the Add New Hardware wizard (Start / Control Panel / Add hardware) and then add it again using the drivers you already have.

If that doesn't work, try a clean boot  http://support.microsoft.com/kb/929135.  If the problem goes away then it's just a matter of tracking down the culprit causing the problem.   Follow the procedures in the article.  Once found, delete, remove, deactivate, or uninstall it.  Once done be sure to reset Vista back to normal status as explained in the procedures. If the problem occurs in clean mode then just restore the system to normal status and reboot - this solution is not going to work.

Boot into Safe mode with networking and see if the problem occurs there.  Either result helps us narrow down the search for the cause,

If none of this works, post back with the results of what you tried and we'll try to come up with some other options.

I hope this helps.

Good  luck!

greg carmack - mvp 2010-2020 say:

A System Repair Disk or Windows 10 Recovery Drive   is bootable media that will allow you a way to access the Advanced Startup Options to repair Windows.  The same tools are built into Windows and available via Recovery Settings, but the bootable methods are more stable and necessary if Windows can't start. 

Windows 10 Image Backup and Restore can save externally or to another hard drive an image of Windows partition so that if it ever becomes irreparable you can boot the Repair/Recovery media to Recovery using the stored image.  Many prefer a more flexible third party imaging app like free Macrium Reflect is a Free and Easy To Use Backup Utility which can generate its own rescue media.

The OS image backup does not replace the need to also have your files backed up.  For that see How to use Windows 10's File History backup feature to do a backup to an external or another hard drive.  You can also drag up to 5gb of your files free into built-in OneDrive for safe storage in the Cloud out of reach of fire or theft;  Google Drive offers 15gb free storage and can back up your User folders you choose like Documents, Pictures, etc in real time.
