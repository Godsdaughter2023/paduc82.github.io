Verizon Firmware (Red/Black)

 

MAC Filter

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. Select Advanced Security Settings.
  7. Select Wireless MAC Authentication.
  8. Un-Check Enable Access List.
  9. Click the Apply button.

 

Westell Firmware (Blue/White)

 

MAC Filtering

  1. Open a web browser.
  2. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  3. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  4. Hover your mouse over Configuration at the top.
  5. Select Wireless from the resulting drop down menu.
  6. From the MAC Address Filtering drop down select Disabled.
