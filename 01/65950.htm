a. user say:

Hi Ethan,

Do the following to reset the layout of the start menu in Windows 10 so that the default layout is used:

1. Digit "cmd" in windows search bar
2. Right click on "Command Prompt"
3. Select "Execute as Administrator"
4. Type cd /d %LocalAppData%\Microsoft\Windows\ and hit enter to switch to that directory.
5. Exit Explorer. To do so hold down Shift and Ctrl, then right-click on the Windows taskbar, and select the option from the context menu.
6. Run the following two commands afterwards:
6.a "del appsfolder.menu.itemdata-ms"
6.b "del appsfolder.menu.itemdata-ms.bak"
7. Restart the Explorer process

To restart Explorer process you have to:

1. Press windows+r
2. Digit "explorer"
3. Select "Ok"
