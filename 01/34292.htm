Those would be Explorer extensions that do not belong to Microsoft.  That means extensions that you added.  Explorer extensions are usually okay and installing some applications will install Explorer extensions for you, give you a choice and sometimes they can be added without your knowledge when you install new software.

Explorer extensions are sometimes added as a new right click option you see on folders and files (like scan this file, open this file, play this song...).

If there is a particular thing you do when Exploring that you know will cause the problem, that will help zero in on the problem and help you know for sure when you have found and fixed it.  If you can make it happen anytime you want, make an adjustment and then there is no message the next time you do whatever it is you do, you have found and fixed it.

When I upgrade a preinstalled (OEM) or retail version of Windows 7 or Windows 8/8.1 license to Windows 10, does that license remain OEM or become a retail license?

If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.

If you upgrade from a retail version, it carries the rights of a retail version.

If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

¡firedog say:

... they are named such as "76047268-00000054," which means nothing to me. Will it cause problems to WLM if I rename the file ...

Yes. You will almost certainly have problems if you perform any sort of file operation on files in the store folder - that's why it's hidden by default.

If you simply want to view those files in Windows Explorer rather than within Windows Live Mail, select Details view on the View menu, then right-click on a column heading and then More to select the columns to view. You should be able to see Subject, Date, From and To just as you would in Windows Live Mail. If you enable the preview pane (Alt-P), you will be able to read the messages as well, although images will probably be suppressed.

An alternative to viewing the messages in the store folder would be to create a new folder, e.g. on your desktop, then select messages in the message list within Windows Live Mail, drag them out to the desktop and drop them on the new folder. Each message will become a new file, now with its subject as the file name.  

Another little tip: in Explorer, select Folder options on the Tools menu. On the View tab, deselect Hide extensions for known file types. You'll then see that mail messages actually have names like 76047268-00000054.eml. There are very good security reasons for always being able to see what a file's extension is and I've never understood the logic behind hiding them by default.

james vinyard say:

Printers can be a huge problem with the RT class tablets (Surface RT and Surface 2).


As you know, you cannot install the driver/software suite from Brother into the tablet.  It will not install "ANY" regular Windows software and drivers.


Normally, the tablet will automatically configure a printer if recognized and install a Class 3 driver.  Your particular printer model should have configured to include scanning.


What you can try to do is first, shut off the printer and go to the Control Panel and select "Hardware and Sound". and select "View devices and printers".  You need to remove your current Brother printer setup if it is displayed.


Now, reboot the tablet and at the same time, turn on the printer.  Once you are at the START screen, wait a minute or two and check to see if the printer was added.  If so, do a test print to make sure it is working.


If that does not work and no printer icon is displayed on the panel, click on "Add a printer".


At the panel, close to the bottom, click on "The printer that I want isn't listed" and go through the Brother list of printers and find a model that is close to the model number you have.  Also, check that the driver says "Class 3" along with it.


You may have to try that with several printer drivers and hopefully it will work.  There is no guarantee that it will.  If it absolutely will not work, there is no other real work-a-round.


I personally have three printers that will work with the RT class tablets: Lexmark S315, HP Deskjet 3000 j310, and HP Officejet Pro 8100.  I got the Lexmark for $48 when on sale and the other two are much more expensive.


Your Brother is a good printer and don't get rid of it.  Also note that with Brother printers, "NEVER" get refilled cartridges for it or you will burn the print head mechanism out.  That is from experience and two dead Brother MFCs.
