    Each time I open Windows Live Mail 2011 I am prompted to sign in to windows Live Mail ... each time it says 'The email or password is incorrect

    The usual reason for this error is that two-step verification has been enabled for the account. If it has, you must generate a unique app password to use in programs like Windows Live Mail.

    I would like to be able to access the Hotmail account using Windows Live Mail. 

    Signing in has nothing to do with accessing your Hotmail account. All it does is load the contacts list and calendar for the Hotmail account. To read, send and receive mail for this account, you must set up the account so that you see it in the left-hand folder pane. If you haven't yet done this, press Ctrl-Shift-T to start the Add your email accounts wizard. If the account is already set up, are you also seeing an 'incorrect password' error message when you try to synchronize it?

    Is this problem linked with the fact that I can't open Internet Explorer?

    I doubt it, although not being able to open Internet Explorer is a serious fault. Are you sure it's enabled? To see, open Programs and Features in Control Panel, then select Turn Windows features on or off.
