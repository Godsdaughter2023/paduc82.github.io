george222(2) say:

If I've understood correctly, one "cludge" way is to use a command prompt.


Open a command prompt at the folder where you want the listing to start, i.e. Use explorer to navigate to where the folder is, press <shift> and right-click the folder. Click "open a command window here".


Then enter:

tree /a /f > folderlist.txt


If all goes well a file called 'folderlist.txt' should be created with all the visible files and folders listed, starting at your start folder.


As I say it isn't perfect but it sort of works, and is quite easy to do.

Hi Rdawg,

 

The User Account Control (UAC) message is displayed to request consent or credentials to allow an application to use the full administrator access token in many circumstances such as while running a setup application. You cannot set UAC to automatically run a program as an administrator without the user being prompted.

 

However you can grant full administrator access token to those application while starting up so that you don’t get the UAC prompt.

 

Note A security risk may result by using administrative credentials to run an unknown program. Only run programs that you trust as an administrator.

cmdrkeene say:
Typically if the data on the network drive is stored on a server, the server handles all the indexing. When you search, your PC sends the query terms to the server, it runs the fast instant index-search and you see the results.   This way, only one index needs to be built and maintained (on the server) and all the client access computers don't need to continuously re-scan the network drive for their own index (which kills performance because every client PC is trying to scan the remote drive).

Blu Ray player not playing Ultra HD Blu Ray discs

I have a Sharp Aquos LC-70UD27U TV which supports 4K 4:4:4 10-bit color, but does not support HDR.  My Xbox One X Blu Ray Player is not playing Ultra HD Blu Ray discs.  I have my One X connected to the correct HDMI input that supports 4:4:4 10-bit color.  I am also using a proper HDMI 2.0 cable (which honestly does not even matter).  Still, when I put in an Ultra HD Blu Ray disc, the disc will not play.  The TV perpetually seems like it is trying to display something and cycles the HDMI input.  I would assume I should still be able to play the Ultra HD Blu Ray disc in spite of my TV not supporting HDR.  I simply would not see the HDR benefit.  But I should still be able to play the Disc and get the resolution benefit.  What is the problem?  Why won't my Xbox One X play my Ultra HD Blu Ray discs.  The two discs I have tried are Warcraft and Wonder Woman Ultra HD Blu Rays.  I am using the Xbox Blu Ray App from the Microsoft Store

perfectcell jrh say:

Just to answer my own question and provide the results, I was not using the TV HDMI input that supported HDCP 2.2.  Since HDCP 2.2 is used for 4K copy protection you must be connected via HDMI to an input that supports HDCP 2.2 when watching protected 4K content.  Once I changed my TV input from the 4:4:4 supported input to the HDCP 2.2 input the Ultra HD Blu Rays played.  It's too bad my TV has 4:4:4 and HDCP 2.2 on separate inputs.  Also, I will need to upgrade my AVR to one that supports HDCP 2.2.  Once I add my AVR back into the middle between the TV and Xbox One X, the discs will not play again since my current AVR does not support HDCP 2.2.
