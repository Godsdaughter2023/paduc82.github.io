Ok,

First of all, I would like to potentially clear up a possible misconception (or two?). The Create a Recovery Drive function/option is meant to be made on/used with a USB flash drive (thumb drive) stick, this is/would/should not be confused with a full system disc image, (the recovery drive could/would be used to access the system disc image, on an external HDD/SDD by enabling it's use from the Windows recovery environment when need be), but is not, in and of itself, a full system disc image. The same function of entering the recovery environment can also be achieved by creating the installation media (ISO, DVD, or dedicated USB stick (recommended 16GB minimum).

richardeiler say:

A full copy, as you put it, would be a full system disk image. The links provided in my last reply contain the detailed information needed to create said (for yourself). There are also 3rd party software products that can perform this as well (such as Macrium Reflect or Acronis True Image ---at least, the paid versions—do offer some advanced features you won’t find in the Windows system image backup tool) if you prefer.

Do note that "in a few years" the Windows 10 OS version build that is now installed on your device will be not only badly outdated but potentially no longer supported, it would then have to be upgraded (Feature Update) to the new most recent version of the OS anyway. A full system disc image should be updated regularly while the system is performing well, it can then be used to restore the system to a recent state from virtually any catastrophic event, even a HDD failure as it can be used to image a brand new hard drive if the need be.

I suggest that you read through the links provided to learn more about the subject. Conversely, you could have a professional set this up for you and explain the details involved that are needed to use this/these function(s) successfully.

pedro v flores say:
HI Dawn

Connect the printer to your wireless network. The process for this will vary depending on the printer. You'll need to know your Wi-Fi network name (SSID) and password.[1]

Many printers will allow you to connect to the wireless network using the built-in menu system. Refer to the printer's documentation for the exact location. If you can't find your manual, you can almost always download a PDF copy from the manufacturer's Support site.
If your printer and router both support WPS push-to-connect, simply push the WPS button on your printer, then press the WPS button on your router within two minutes. The connection will be made automatically.
Some older wireless printers may require you to connect to a computer first to set up the wireless connection. This is common if the printer doesn't have a built-in menu, but does support wireless. Connect the printer to your computer via USB, and use the included software to set up the printer's wireless connection. After configuring its wireless, you can disconnect it from your computer and place it where you want.
