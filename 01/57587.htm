So, try to find shortcuts which launch cmd.exe and see what their keyboard shortcut is set to.  E.g. via right-click Properties.

    If not then, click on control panel. Click on Sound and then click on recording tab. Check if the microphone is enabled.

     

    I would suggest you to download and install the sound drivers for windows vista in windows Vista compatibility mode.

    1.       Download the Windows Vista drivers for your sound card.

    2.       Save the driver installation file on your desktop or your preferred location.

    3.       Right click on the installation file and then click on properties.

    4.       Click on the Compatibility tab on top.

    5.       Under Compatibility mode, select the check box which reads as “Run this program in compatibility mode for” and select Windows Vista in the drop down bar.

    6.       Click apply, and then click ok.

    7.       Now, double click on the driver installation file and install the drivers.
