Option 1: System Restore.    This option will allow you to roll your PC back to a working restore point.  Any system changes made since that restore point will be lost.

  1. From the System Recovery screen, select Advanced Options.
  2. Select System Restore. Your PC will restart and prepare the System Restore experience.
  3. Choose an account name to continue and enter the last good password used on the PC.
  4. In the System Restore window, click Next and select the Show more restore points check box.
  5. If one is shown, select a restore point at or before the Webroot installation and follow the prompts to complete the system restore.
  6. When returned to the sign-in screen, if you still can’t sign in, perform the same steps and select an earlier restore point until you are able to sign in.
  7. Once signed in, we recommend you uninstall Webroot until Webroot has shipped an update to address this problem.

 

If you do not have any system restore points, you will need to do a system Refresh.  This will restore all Windows 10 operating system files, and also backup your files and folders.  Any programs installed will need to be reinstalled.  This process requires you provide the original Windows 10 media that came with your computer, as well as any media you used to install other programs.

Option 2: System Refresh.  This option will restore key system files.  It will uninstall any third- party applications, but won’t affect personal settings and files.

1.       From the System Recovery screen, select Refresh. You’ll see a screen that explains what actions will occur.

Your PC will restart and a refresh will occur. When this is completed, there will be a file saved on your desktop with a list of programs that were uninstalled. Don’t reinstall Webroot until an update to fix this problem is released, but feel free to reinstall any other programs on the list.

2.       Sign in with the same password you used to start the refresh process. Once you’ve signed in, if you have changed your password online you’ll be prompted to enter that new password once Windows 10 fully loads.

Or you can drag or copy them to a USB external or flash drive.

There is an easy way to mount the System Image to extract files if needed. Some prefer to move their User folders off of C to make the image smaller so they stay current if you need to reimage C. Here's how: http://www.dummies.com/computers/operating-syst...

a. user say:

What is the official Microsoft procedure for how to protect my computer against a complete loss such as a hard drive failure?

The official Microsoft policy is to use a third-party image backup application.

This policy was adopted with the release of the Fall Creators Update (v.1709) during the fall of 2017. This update deprecated (meaning 'abandoned') System Image Builder which, until that time, was the only Windows 10 utility that was supported for making image backups.

Microsoft's official pronouncement is here:

Features that are removed or deprecated in Windows 10 Fall Creators Update

where Microsoft states:

"We recommend that users use full-disk backup solutions from other vendors."

Fortunately, there is no shortage of excellent full-disk backup software from well-established developers. If you are interested in learning about full-disk backup software, which is also known as image software, image backup software and system image software, I'll be glad to help as will others in this group.

Image backups have been a staple of computer users for many years, so there's nothing new about this. If interested I'll gladly explain.

3.      In the Accessories menu, Right Click on the Command Prompt option

4.      From the drop down menu that appears, Click on the 'Run as administrator' option

5.      If you have the User Account Control (UAC) enabled, you will be asked for authorization prior to the command prompt opening. You may simply need to press the Continue button if you are the administrator or insert the administrator password etc.

6.      In the Command Prompt window, type: sfc /scannow and then press Enter

7.      A message will appear stating that 'The system scan will begin'.

8.      Be patient because the scan may take some time

9.      If any files require replacing SFC will replace them. You may be asked to insert your Vista DVD for this process to continue

10.  If everything is okay you should, after the scan, see the following message "Windows resource protection did not find any integrity violations"

11.  After the scan has completed, close the command prompt window, restart the computer and check.

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7

jebjr say: Well, finally solved the issue.  Turns out that the lower level nVidia graphics cards I tried (the 210 and GT220) couldn't handle Internet TV through Windows Media Center at the high resolution I was trying to use (1920x1080, in my case).  I pulled a higher end graphics card from one of my other computers (an nVidia 9800GT) and Internet TV played without incident for a couple of hours.  I don''t know where the dividing line is between inadequate and adequate in terms of GPU performance, but the GT220 (which I think is about equivalent to a 9400GT) did not work and the 9800GT did.

So, if you're having a skipping/stuttering problem, either try a lower resolution or a more powerful card.
