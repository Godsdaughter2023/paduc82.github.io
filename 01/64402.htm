6. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands

    dir E:\Win*
    until you find the Windows folder.

    cd  RegBack
    dir

    copy /y System  ..
    copy /y Sam ..

10. Reboot normally. If things do not work out then you can backtrack by copying the original registry files from Q:\MyRegBack to Q:\Windows\System32\config.

    a. user say:

    The folder hierachy doesn't exist -- MyPDF's gone, MyWord, MyExcel etc aren't even there.

    Is that a folder hierarchy that you created? Neither Windows nor Office creates folders by that name.

    In Windows, your personal data files are supposed to be kept in your user profile, i.e.,

    C:\Users\<your user account name>\ Documents

    C:\Users\<your user account name>\ Music

    etc.

    That's how Windows was designed. Since that's where Microsoft expects your personal data to be, those folders are never touched by updates to Windows. If you store your personal files in any other place they might possibly get erased during the course of an update.

    Until the most recent update - v.1809 - there were no widespread reports of user files being erased if they were stored in the user's profile. What happened with v.1809, which was totally Microsoft's fault, applied only in specific situations where people were using OneDrive, and that problem was fixed months ago.

    Sign in to a user account with Administrators permissions. (Note that this is not the same system as the Administrator account, which is a hidden system account.) A user account with Administrators permissions is able to obtain access to all other user accounts on the computer.

    From an Administrators account, navigate to the C:\Users\<your user account name> folder. You will likely be asked if you want to have access to that folder - answer yes. Within that folder there should be sub-folders for Documents, Pictures, etc. That's where you'll find your personal files, if that's where you stored them. If you stored them outside  the C:\Users\<your user account name> folder, they may be gone.

    There's one other place where your files might be if they weren't stored in your user profile. After every feature update, Windows creates a folder called windows.old which is used to rollback your computer after an update. Your missing files might be inside the windows.old folder. However, that folder only stays on your computer for 10 days after an update has been installed, after which it's deleted.

    Are you signed in to a temporary profile, or where you previously using a temporary file? That's something that Windows 10 creates when your regular user account has been damaged to the point where it's no longer usable. Nothing that you store in a temporary profile is saved once your computer has restarted, which it will do after receiving an update, and you cannot access your previous user account from a temporary profile.

    Are you using the built-in Administrator system account? That's not a user account, so it doesn't work like a regular user account. I honestly don't know what happens to personal files stored in Administrator after an update to Windows.

    System Restore is not a backup. Only a backup is a backup.
