Do you have any issues in printing documents? If yes what is the complete error message do you get?

The print spooler is an executable file that manages the printing process. Management of printing involves retrieving the location of the correct printer driver, loading that driver, spooling high-level function calls into a print job, scheduling the print job for printing, and so on. The spooler is loaded at system startup and continues to run until the operating system is shut down

If you need to enable print spooler follow steps provided below:

  1. Click on start.
  2. Type services.msc in search box, click on it to open it.
  3. Search for Print Spooler and double click on it to open it.
  4. If the status of service is stopped, then start the service and check if you are able to print.
If you are facing any issues while printing , you may run this fixit:
http://support.microsoft.com/mats/printing_problems

Try splitting your project into two parts. Make a second copy of your project (File > Save as). In the first project copy, only save the first 50% of the project as a movie. Alternatively, if that still doesn't help, look for the clips that are at the 89% point in your project. Perhaps they aren't 100% compatible and that's causing an issue for Movie Maker. Try converting the problem file(s) and then see if you can't save the movie.
