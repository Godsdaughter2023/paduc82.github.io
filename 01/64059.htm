To add or remove columns in Detail view, right-click one of the column headings, from this context menu, select More, then you can select Attributes as an additional column. You can also use the Move Up/Down buttons to modify the order of the columns.

I say the Read-Only checkbox is meaningless because it does not indicate the state of the folder's Read-only attribute, but that is how most casual users interpret it. What it can do is toggle the Read-only attribute on files contained within the folder.

gh_st say: In reply to deleted message

This will most likely remedy your issue as it did for me. If not I wish you all the best.

1) Open "Control Panel"

2) Open "Network and Internet"

3( Open "Network and Sharing Center"

4) Open "Advanced sharing settings"

In your current network "profile" look for "Turn On Automatic Setup Of Network Connected Devices" and REMOVE the check from it.

5) Save the change and disable the network adapter and re-enable it.

Look for the change in File Manager.

This is what worked for me.
