The Recent Items is a special system folder and cannot be used as a normal toolbar.

There are 3 other ways provided to access Recent Items.

You can place the Recent Items on the right side of the Start Menu that will display a flyout menu.

You can also place a shortcut to that folder on the Desktop or pin that folder to the Windows Explorer Jump List on the Taskbar.
