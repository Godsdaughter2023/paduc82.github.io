Windows 10, once activated and a Digital Entitlement obtained, no longer uses the normal License Key you have been used to.  You can clean install on that machine whenever.

The recovery drive can be used to boot to and load the Recovery Tools to repair your system.  If you check the box (default) to copy over System files, it will copy an image of your current Win 10 install but I believe it only contains the basic Win 10 install.  It may not contain other items.  You will need your password to use the Recovery Drive and I don't think it will use the PIN if you have one set up.

If you want to put your system back where it was in case of a hard drive failure you may still want to have a System Image you can restore.

Couldn't Update System Reserved Partition

You get this error if the System Reserved partition in Disk Management is too small. The system reserve partition needs to be at least 450 to 500 MBs.

Press Windows key + R

Type: diskmgmt.msc

Hit Enter

Click the system drive to view the size.

Resolution:

  • Createa system repair disc
  • Download Ease us partition manager and resize it to 300 to 450 MBs.
  • Please ensure you create a system repair disc in advance before doing this.

Windows 7 - windows.microsoft.com

  • After installing Easeus partition manager, launch the program.
  • Select the system partition where Windows is installed.
  • Click Resize/Move button on the toolbar

Under Decide size and position, use the knob and drag to the left until a desired amount of space is allocated. Look in the Unallocated Space Before: field to see how much. Once that is done, release.

Click Apply and OK to confirm changes. Your computer will be restarted to apply the changes.

Select the System Reserverd partition then right click it and click Extend Volume

Select the Unallocated space we created earlier then click Next then click Finish
