In Windows 10, Windows update behaves differently [and is accessed differently, it is not in Control panel].

Updates take place automatically unless you have set your internet connection to be a "metered" connection [which can be useful for those with a limited data allowance such as a mobile phone connection or a satellite connection].

You can stop Windows update from updating all hardware drivers & associated software.  See System, Advanced system settings, Hardware, Device installation settings. 

You cannot pick & choose between Windows updates, they are all installed.  You can subsequently uninstall chosen updates & stop them from downloading & installing again - that is the utility that the first reply gave you a link to.

You can run update checks manually as well if you wish - see

Start button,

Settings,

Update & security.
