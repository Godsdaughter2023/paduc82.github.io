Defender compatibility with Malwarebytes - Microsoft Community

lois r hills say:

Defender compatibility with Malwarebytes

I simply want to know ONLY about the Compatibility between 'WINDOWS10 DEFENDER 2017' and Malwarebytes PAID version.

a. user say:

When you install the paid version of Malwarebytes, which runs automatically in the background, it disables Windows Defender from also running in the background. In that way they are compatible.

When you install a third-party anti-malware solution - like Malwarebytes' paid version - Windows Defender occasionally runs a scan. This is something that you want to happen. Since Windows Defender is not running in the background, its scan does not interfere with the third-party solution. In that way, they are compatible.
