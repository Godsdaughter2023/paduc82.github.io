balaji kundalam say:
Sorry for the delay in reply.


The WWAN connection depends on the signal strength, technology in use, speed of the network, and the power requirements of the adapter and also the priority/order in which the connections are created. Therefore, out of all these criteria, one of the connection that suits best is chosen. Sometimes, more than one connection may satisfy the criteria, but note that only one connection is used to save power.


However, you can use more than one internet connection in a PC if you are running virtual machines (like Windows XP Mode, Hyper-V, VMware, etc.). You can assign one internet connection to one virtual machine, one connection to other virtual machine and one connection to the host (the physical machine). Note that even in this case, a single machine/PC will not use more than one internet connection at the time.
