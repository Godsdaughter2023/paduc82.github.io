a_user_here say:

It is not clear what you mean by "close all my tabs".

Are you talking about closing apps or windows?

There are two reasons you might see the Start Screen instead of the Start Menu (when you are seeing the Start Screen you can't see the desktop).

One is that you have elected to use the Start Screen instead of the Start Menu, the other is that you are in Tablet Mode.

For selecting it as an option:

Click on the Action Center icon near the clock -> All Settings -> Personalization -> Start -> turn off Use Start full screen

You can get into the Tablet Mode for a few reasons.  One is that you selected it in the Action Center.  Another is that you have a device that Windows sees as a Tablet.  Which could be a tablet or a hybrid computer with the keyboard detached.  When it see that it goes into Tablet mode automatically.

You can control the Tablet mode settings in All Settings -> System -> Tablet Mode
