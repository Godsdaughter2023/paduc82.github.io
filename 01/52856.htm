The On Screen Keyboard program has an executable file "osk.exe". Presumably something is causing osk.exe to run, every time you log in. Follow either Method 1 or Method 2 to fix the issue.

 

Method 1:

 

To verify if osk.exe is triggered to run during startup, follow these steps.

 

a)   Go to Start type “msconfig” without the quotes and press Enter.

b)   Go to “Startup” tab.

c)    Look for any entry which appears to run osk.exe.

d)   If there is any entry, uncheck the entry and then press Apply.

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Back up the registry

http://windows.microsoft.com/en-us/windows7/Back-up-the-registry

 

a)   Go to Start, type regedit and press Enter.

b)   Go to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run
