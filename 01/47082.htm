wunders say: Even though the file itself may have a filename of 9 characters plus extension, it could be buried inside of multiple folders.  So when you mount your external USB drive and it shows up as, say, Drive M:, The number of characters in the complete path to the file starting at "M:\" and including all directories in the path to the file plus the number of characters in the filename itself cannot exceed 255 or Windows refuses to handle it.

Robocopy is a command line program.  You must type in the command from a command prompt window  (Start -> Run -> "cmd").  It works by accessing the file using a different set of library copy routines that does not have the 255 character limitation.  A documentation file should accompany Robocopy explaining how to use it.

canadian tech say:

Those "removable disks" are physically present.  You cannot remove them from the listing unless you physically disconnect or remove them.  You likely have several different card readers built-in to your computer -- those are the removable disks.

As far as your camera is concerned, I recommend you do NOT connect your camera to your computer directly.  

Instead, remove the memory card from the camera and plug it into the memory card reader on your computer that is appropriate for it -- chances are that is the SD port.

Once you plug in your memory card, you use it much like you use any USB thumb drive and move or copy files from it to your computer.  It is much faster and requires no special camera software.  It allow you to do that on any computer.  Deleting pictures from the card is dramatically faster and easier.
