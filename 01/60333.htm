somasundaram pandiaraj say:  

To enable Microsoft Office Document Scanning in Microsoft Office 2007 follow the steps given below:

 

1.       Quit all Office applications

2.       Click on Start > Control Panel > Add or Remove Programs

3.       Select the Microsoft Office (Edition) 2007 available in the list and then click on Change

4.       Select the Add or Remove Features option and click on Continue

5.       Select ‘Run all from my computer ‘using the drop-down arrow for Microsoft Office and then click on Continue.

 

Once the configuration is complete, click on Start > All Programs > Microsoft Office > Microsoft Office Tools to find Microsoft Office Document Scanning

 

For more help on how to add Office components in Office 2007 check the link below:

This problem goes back through Windows 8..

If you locate and open the CBS.Log file and search for "corrupt" you may find that the corrupted files are associated with IE or Edge.  This has been going on for years, on my systems. Rarely does stc complete with no errors.
