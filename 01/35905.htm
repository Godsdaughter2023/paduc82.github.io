¡firedog say:

Do you already have your EarthLink account set up to use POP3 in Windows Live Mail? If so, right-click on the account name and select Properties. On the General tab, deselect Include this account when receiving mail or synchronizing.

Press Ctrl-Shift-T to start the Add your email accounts wizard, enter your EarthLink email addrsss, password and display name for outgoing messages and select Manually configure server settings before clicking Next.

On the next page of the wizard, enter these details:

  • Incoming Mail Server: imap.earthlink.net 
  • Incoming Port Number: 143
  • Outgoing Mail Server: smtpauth.earthlink.net
  • Outgoing Port Number: 587

You should also select requires authentication for the outgoing mail server.

When you complete the wizard, the program should download all your folders, so give it plenty of time to finish doing this. When it's done so, review your settings. Here are some suggestions - you may find on experimentation that different settings work better for you: 

  1. Select IMAP folders:
    Right-click on the account name in the folder pane and select Show or hide folders to open the Show/hide IMAP folders dialogue. Use the buttons to show only the folders you want to see in the folder pane. Note the exact names of the 'special' folders.
  2. Map the local special folders to the IMAP folders on the server:
    Right-click on the account name in the folder pane and select Properties.
    On the IMAP tab, select Store special folders on IMAP server,  then enter the server's names for the special folders in the boxes. If the folder for junk email at the server is called Spam, then enter Spam (not spam or SPAM) into the box alongside Junk email path and so on.
  3. Make Deleted items folders work the same way for all accounts:
    When viewing your message list, press Ctrl-Shift-O for Options.
    On the Advanced tab, select Use the 'Deleted Items' folder for IMAP accounts.
  4. Review the synchronization settings for each folder.
    Right-click on each folder in turn and select Synchronization settings, then select suitable ones. For example, if some of the folders are only for archive purposes, it may not be necessary to synchronize them each time - perhaps once in a while just to make sure the local copies are up to date. 
EarthLink IMAP Email Setup

When the account is working properly, you can transfer messages from the old POP3 account (e.g. ones in the Sent items folder) to the new account, then remove the old one.
