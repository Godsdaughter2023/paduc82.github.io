jetta48 say:

One method:

 

1. Insert thumb drive and wait for it to be recognized in File Explorer.

2. Note the drive letter of the thumb drive.   (I'll call it E:\  in this example)

3. Press Windows logo key + R   (at the same time).

4. Type CMD in the Run box then click OK.

5. In the Command Prompt box type:

dir E:\>filename.txt

(where filename can be anything, but make sure it has the extension .txt)

6. Close Command Prompt box (X in top-right corner).

7. In File Explorer, navigate to folder:

C:\Users\Username\

(where Username is whatever your username is)

8. Locate the file filename.txt then double-click on it to open it in Notepad.

9. Edit if wanted then print from Notepad as if it were a normal text file (which it is) by clicking on File then Print.
