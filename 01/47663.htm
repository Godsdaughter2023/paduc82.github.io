Hi Regina,

Thank you for posting your query in Microsoft Community.

  • What type of installation did you perform to upgrade to Windows 10?
  • How did you install Windows 10, through Windows updates or Media Creation Tool?

The option to go back to previous version will be available only if you upgrade to Windows 10 through Windows updates or ISO (only if you choose "Upgrade" or "Keep everything and install" or "Keep files and settings and install".

If you choose "Keep nothing and install" which is also called clean installation, you may not get the option to Go back to previous version.

Please keep in mind that the following actions also will disable Go Back:

  • Refresh
  • Reset
  • Disk Cleanup (if it touched Windows.old or $windows.~bt)
  • Manually deleting files within Windows.old or $windows.~bt
  • Running on battery (must be plugged in)

If external device was used during upgrade to store Go Back data – you must have it to perform a Go Back

Hope this information is helpful. Please feel free to reply in case you face any other issues with Windows in future.
