cmdrkeene say:

Officially, the only way to tell is with the "info" bubble on the picture.  But those aren't always that informative.

A "power user trick" is to copy those pictures to your desktop so you can right-click and view properties and see where the picture was taken (which is in most of those files).  Here's what I'd do if you want to try it out:

  1. Create a new folder on your desktop named "Pics"

  2. Use your Start menu search box to find and open "PowerShell"

  3. Copy and paste this block of code into the PowerShell window, then press Enter.  It will copy all the "Spotlight" pictures to the Pics folder on your desktop.

    Get-ChildItem -Path $env:localappdata\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets | Copy-Item -dest {"$home\desktop\pics\" + $_.BaseName + ($i++) +".jpg" }


  4. Close PowerShell and open that Pics folder.  There will be some extra pictures in there, but you can find the Spotlight picture, right-click and go to Properties for it, and see if the details shows where it is from.

Example:

Now not all pictures will have Details shown there like this one.  So here's one more trick.  Go to http://images.google.com, click the little "camera" button on the right side of the search box, and choose one of those pictures.  Google will reverse-lookup and tell you where that picture came from (if it has found that picture anywhere on the web before).
