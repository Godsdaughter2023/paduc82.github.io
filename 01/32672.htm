silverhead123 say:

How does Google Chrome know my Windows password?

Hello,

I needed to change a couple of 'remembered passwords' in Google Chrome. When I tried to delete/change them Google asked me to enter my Windows password (the one I use to log into computer). How does it know this log in password please?

cmdrkeene say:

It doesn't know it -- it is using the Microsoft API to authenticate you.  Google never sees, checks, or authenticates that password.  It simply calls the standard elevation prompt.

For example, it's the same thing as when you right-click a program and choose "Run as administrator" and get prompted for password.  The program you're running never knows or sees your password.

cmdrkeene say:

Press Ctrl+Shift+Escape to show the Taskbar Manager and you can see this one:  Chrome runs every extension, tab, and some other functions as fully separate processes, so that it can "restart a chrome.exe process and only elevate that one, without giving all Chrome-hosted tabs any elevation. This keeps things sandboxed and secure.

Internet Explorer does the same with "Protected Mode", as does Project Spartan/Edge browser.  I don't know about Firefox/Safari/Opera, but I assume they are pretty close as well in this regard. Separating processes like that is great for security (but it's also why Chrome uses so much RAM).

10. Restart the computer. On restart, Windows will re-install the generic drivers automatically.
