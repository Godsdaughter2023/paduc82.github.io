preciousleb say:

Hi Ben,

This occurs when Windows finds no device or program that matches the name described in the error code. With that being said, we suggest that you check this article to get to know more about the Uninstall the following programs: Storage Manager Driver error message. You may also find the troubleshooting steps for the said error.

We also suggest that you perform a clean boot. By using clean boot, make sure that you turn each service enabled one by one after each restart to determine if there's any program that is causing the issue. Once determined, uninstall the program. Here are the steps:

  1. From Start, search for msconfig.
  2. Select System Configuration from the search results.
  3. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then tap or click Disable all.
  4. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.
  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.
  6. Close Task Manager.
  7. On the Startup tab of the System Configuration dialog box, tap or click How to reset the computer to start normally after troubleshooting with clean boot of the KB article to boot the computer in normal startup after fixing the issue.

    Let us know how it goes and feel free to get back to us if you need further assistance.

    If you think this was useful, do not hesitate to "mark it as answer" to help those who are facing the same problem.
