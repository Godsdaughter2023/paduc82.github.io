tryx3 say:

There are both technical and contractual considerations.  I also use my phone for my PC's internet connection, by the way.

Tethering mobile phone to computer

Technical

1  USB connection [what you are doing already]

1.1  To connect the phone to the PC using the phone's MicroUSB-USB cable, you would normally have to install the phone's drivers on the PC.  All phone makers supply drivers but they might be integrated within their sync software rather than as separate files so you would have to follow the instructions for using that even though you might never want to use the sync software itself.  Your phone's handbook, your phone maker's support site & its support forums are bound to give you suitable advice for doing this task.  If the drivers exist as separate downloads from the support website then you can keep things simple by only installing them. 

1.2  Go to the phone and into its settings.  Somewhere in there will be a set of entries for mobile network sharing ["tethering"].  Make sure you have selected the entry for sharing the mobile phone's network connection [rather than the other way round - which would normally be called "Internet pass-through"].  You might find that your phone has a setting for automatically deciding which of these it needs so you might prefer that if you intend to work the other way round on occasion.

2  WiFi hotspot connection

2.1  If you have a "WiFi hotspot" capability on your phone then switching that on creates a "WiFi Access point" that a WiFi-enabled computer can detect and connect to.  If you then switch on the phone's data connection [the same one you'd use for the phone to use the internet] then that WiFi Access point will give other devices the ability to connect through the phone to the internet.

2.2  Look in the settings for your phone to find the WiFi hotspot connection details - the hotspot will have an SSID [a name] and a password.  

2.3  Click on your computer TaskBar's network icon then select the network that has that SSID [name], enter the password and you should connect to it.

2.4  You would not normally have to alter the settings [see para 1.2 above] for the computer to be connected straight through to the internet this way as it's part of the way WiFi hotspots are set up.  It would not hurt to look in settings but I don't think you will find that you need to do anything else.

2.5  As soon as you are confident with the setup, go back to the phone and change both the SSID [name] and password because hackers are able to exploit the default ones that phone makers put in.  Then repeat the 2.1 - 2.4 procedure with these new parameters.  This time, you can set the checkbox you'll see on the computer for "Connect automatically" so you can, in the future, just turn things on and let them sort themselves out.

3  Connecting through your local network

3.1  If you have a network router then you might be able to use the phone's USB cable and / or its WiFi hotspot and / or its own WiFi capability to connect your router's network to the phone for internet use.

3.2  I know nothing about routers and have never tried any of these methods so am not fit to advise you about them.  All I know is that some other people have things set up this way.

Contractual

4.1  Some ISP's only allow the mobile device itself to connect to the internet.  Some of them have payment plans that allow computers to use the data connection as well - ISPs use the term "tethering" to mean both USB cable connections and WiFi hotspot connections. 

4.2  You need to find out if your ISP allows your computer to use the connection.  So you need to check the terms & conditions to make sure you are allowed to tether - you might find the info easily or it might be buried in a "fair use" policy statement or a "network traffic management policy" statement.  If your ISP does not allow tethering then you might find that the connection gets slowed down or blocked - you might even find that your account gets suspended.  It's also possible that an ISP states that it does not allow tethering but then does nothing about it so you could abuse the contract to use tethering without incident [but you'd only find that out by trying it and seeing what happens - if they monitor for tethering then you might be penalised]. 

4.3  If you have a "fair use" policy on your ISP's internet connection then you might find that tethering your computer to the phone eats through your data allowance quicker than you could imagine.  You need to consider both the internet use you are aware of and the other things that happen automatically without your direct knowledge - Windows & application updates, subscriptions to customer feedback programmes etc.  There are several examples in WMP alone so look in WMP, Options, Privacy tab for the setting "WMP customer experience improvement program" [look around the other WMP options and you'll see several more items that involve background internet data usage. 

4.4  Mobile data connection price plan prices vary enormously.  So do warning systems for exceeding the data allowance.  I was with one ISP that sent warning texts at 80% then at 100% of data usage & actually interrupted my web browsing with a webpage explaining the situation so I would not incur excess use charges without deliberately deciding to do so.  I later switched price plans with that same ISP without checking the details and I found that I was charged extra because the new price plan did not include these warnings.  I've changed to another one that does issue warnings and I intend to stay. 

I hope this has answered your question. 

5  Syncing with your phone [I realise you did not ask about this] - if your phone is an Android then you might be interested in reading my Android syncing ditty.

lemp say:

Yes, if your computer has a wireless communication device that's compatible with whatever wireless technology is used by your phone.

For example, Apple permits you to tether an iPad to an iPhone using either or both wi-fi or Bluetooth, both of which are wireless communication technologies.

Assuming that your mobile phone permits tethering using wi-fi, then your computer must have a wireless network adapter.  Almost all laptops sold today have one of these built in.  Most desktop/tower systems do not, but such an adapter can be purchased relatively inexpensively.  It can either be external (some USB wi-fi adapters look very much like typical USB flash drives) or it can be an internal card (like a video or audio card).  The latter, of course, can only be used if the computer has an available internal connector (usually called a "slot").

If you want to use Bluetooth (a short-range wireless technology), your computer needs a Bluetooth adapter.  Again, most laptops will have this and most desktop/towers will require the purchase of such an adapter.

What does Windows as a Service entail?

Since the introduction of Windows Update as part of Windows, Microsoft has delivered updates as way to keep Windows updated and secure. This has always been the way the company keeps commercial versions of Windows on the market current.

With Windows 10, Microsoft is making significant changes to this model by delivering new methods of servicing Windows. Taking into account we now live in a mobile, constantly connected world, where threats happen all the time. Part of the solution to this is to keep users current by providing the latest updates as soon as they are available in addition to delivering features as soon as they ready too.

Microsoft will do this using three types of service branches for Windows 10:

  • Security Updates, Features and Fixes are automatically applied.
  • There is no option to delay or customize these updates.
  • Windows 10 Home
  • CBB includes the requirements of the Current Branch, but also provides the option of customizing when and which Security Updates, Features and Fixes are applied, similar to how Windows Update works today in current versions of Windows. 
  • Updates cannot be deferred indefinitely.
  • Windows Updates can be managed using enterprise management tools such as Windows Update for Business and/ WSUS.
  • Windows 10 Pro
  • Windows 10 Enterprise
  • Windows 10 Education
  • Only available to volume license customers running Windows 10 Enterprise.
  • Flexible options for managing Windows Updates.
  • Windows Updates can be customized to only accept Security Fixes.
  • Windows Updates can be managed using enterprise management tools such as Windows Update for Business and/ WSUS.
  • Windows 10 Enterprise

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device – at no additional charge. With Windows 10, the experience will evolve and get even better over time. We’ll deliver new features when they’re ready, not waiting for the next major release. We think of Windows as a Service – in fact, one could reasonably think of Windows in the next couple of years as one of the largest Internet services on the planet.
