There is an easy way to mount the System Image to extract files if needed. Some prefer to move their User folders off of C to make the image smaller so they stay current if you need to reimage C. Here's how: http://www.dummies.com/computers/operating-syst...

ajay k say:

Hi Duck Butler,

Welcome to Microsoft answers forums!

Who is the computer manufacturer?

This functionality is specific to the computer and not a part of the operating system.

It’s a program “On-Screen Cap/Num/Scroll Lock Indicator” installed by the computer manufacturer.

Step 1:  Disable the On-Screen Cap/Num/Scroll Lock Indicator service and check

Log on to the computer by using an account that has administrator rights.

1.    Click the start button, type msconfig.exe in the Start Search box, and then press ENTER to start the System Configuration Utility.

If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

2.    Click On the services tab, look for the service On-Screen Cap/Num/Scroll Lock Indicator and then click disable.

3.    Click OK, and then click Restart

 

Note: You need to follow the steps to enable the service back.

 

If this doesn’t help,

Step 2:  You may uninstall the program “On-Screen Cap/Num/Scroll Lock Indicator” From programs and features

http://windows.microsoft.com/en-US/windows-vista/Uninstall-or-change-a-program

Important:  Please check with the computer manufacturer before uninstalling the program

c:\windows\fonts\vgaoem.fon

c:\window\system32

c:\windows\fonts
