The mail app only leaves the messages on the server, there's no way to change it.  You'll need to go to your email server options and set it the there.  Such as this example from Gmail to tell it what happens after an email client downloads a message.

This could also be the source of your replication if you're using Gmail, which uses "tags" instead of folders, which means that a message being marked as read is really a tag change which IMAP reads as a folder move.

  • POP (Post Office Protocol)
