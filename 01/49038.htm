denferg say:

I was able to get Remote Desktop to span two Samsung Syncmaster P2450 dual monitors by doing the following.

  • Created a shortcut to remote desktop on my desktop and edited the Remote desktop properties ( Target field) to include /span

example:  C:\Windows\System32\mstsc.exe /span /v:"<hostname>"

  • Ensured both displays had same display resolution. Originally I had one set to 1920 x 1080 and the other was set to 1776 x 1000.
Once I changed both to  1920 x 1080, spanning now works across both monitors.

Hope this helps. This was after upgrading from Windows 8 to Windows 10
