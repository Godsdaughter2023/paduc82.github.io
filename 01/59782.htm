I would like to inform you that .dll (Dynamic Link Library) files are present in every application folder, and are used to store common pieces of application logic that can be accessed from multiple applications.
Since there’s no way to directly launch a DLL file, the rundll32.exe application is simply used to launch functionality stored in shared .dll files.

 

Please back up registry before performing method 1

To back up registry follow the steps below
Go to start screen->Type Run in Search box and open Run window->type regedit and click on ok->Right click on computer in registry window and export it to desktop->provide a name as back up and save it on desktop.

To do so: Important this section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. 

CTMasterOnOffMonitor is a part of Creative Technology software.

Method 1: Open a command prompt. (Run a command prompt as Administrator in Windows 7
Copy command line and Paste.

echo Start
echo #
echo ######################## Default dirctory for x86 x64 ########################
echo #
echo this command is default system32 directory for x86 OS or x64 OS
cd %windir% & cd system32
reg delete "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v "CTMasterOnOffMonitor" /f
reg delete "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v "CTMasterOnOffMonitor" /f
echo #
echo ######################## Change the dirctory for x64 ########################
echo #
echo this command is x86 application's registry for x64 OS
cd %windir% & cd syswow64
reg delete "HKCU\Software\Microsoft\Windows\CurrentVersion\Run" /v "CTMasterOnOffMonitor" /f
reg delete "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v "CTMasterOnOffMonitor" /f
echo End


Close the command prompt.

 

Method 2: Run SFC /scan.

The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions. When the scan is complete, the System File Checker tool indicates whether integrity violations were found. If integrity violations are found, the System File Checker tool lets you fix the integrity violations.
Follow these steps to run SFC scan:
a. Click on the Start button and type Command Prompt on the Search Field.
b. Click/tap on Command Prompt (Admin).
c. In the elevated command prompt, type sfc /scannow and press Enter.
Note: This may take a while to finish running.

bob dahlin say:

Isn't this typical... I try to fix it for a week, but five minutes after posting the question here I find the answer...

It seems ASUS Smart Gesture is altered/removed by one of the later Windows 10 updates.

OK, if you have an ASUS computer and this happens here's what you do (thank you BillD116 for the solution):

-Go to Settings

-Control panel

-Uninstall/Change program

-ASUS Smart Gesture

-Repair.

-Restart Computer

THEN

-Start (bottom left)

-Control Panel (type it in the search box)

-ASUS Smart Gesture

-Mouse Detection

-Disable touchpad when mouse is plugged in.

If you don't have ASUS Smart Gesture in your program list at all, you can google it and download it, and then go through the steps above. I even had to do a restart before I was able to install as the installer said another install was ongoing... weird... 

But now the auto-switch off function works again, and the fn+F9 button quick switch as well.

1) WinKey-R   (to get the Run dialog box)
2) devmgmt.msc  (and hit the OK button)

3) Display adapters

4) Right-Click on the adapter listed and choose Properties.  Mine was ATI Radeon HD 4200 
5) Under the "Driver" tab, note the Driver Date.  Mine as July 2017

6) Click on the Roll Back Driver button.

7) You'll be asked to fill out a short survey

8) I think I clicked on Okay, but not sure.  Screen corrected almost immediately.

9) Note the Driver Date again.  Now mine says Jan 2015 and things are working much better.
