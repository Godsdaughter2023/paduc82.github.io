so a repair installation is more thorough but also requires a disk.

 

you might be able to borrow a xp cd to execute a repair installation, but it has to be the same sp version as the o.s. on the machine, i.e. xp cd sp3 = xp o.s. sp3

A popular reason things stop working after Windows Update is because hardware driver updates were included in the updates and most folks just install all the offered updates without looking through them believing that Microsoft knows what updates are best for their system.

Here are some general ideas that might prevent the problem in the future:

It is generally not a good idea to accept hardware driver updates (network, audio, video) offered by Windows Update and if you ever see them it is best to hide them so you never see them again.  Only get hardware driver updates from your hardware manufacturers WWW site.
