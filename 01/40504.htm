A few points:

If you actually convert a document that was created using one word processor (e.g., MS Works) to the format of another word processor (e.g., WordPerfect), it is not at all uncommon to find that not all the formatting survives.  This is less likely going from Works to WordPerfect because Works is such a simple word processing application.

Merely opening a document that was created using one word processor (e.g., MS Works) in another word processor (e.g., WordPerfect) doesnot "convert" the document (at least, not permanently).  To make the change permanent, you have to "save" the file from the new word processor.  If you choose not to "save changes" (because, e.g., you didn't make any changes), nothing happens to the original file.

Changing the file extension of a file (from, e.g., wps to wpd) does not convert the file at all.  It will, however, fool Windows into opening the file with the application associated with the extension.

It may be that when you reinstalled WordPerfect you didn't customize it the way it had been.  If you can describe what it is that "looks entirely different than what it used to look like," perhaps someone can help you change things.  Although it's possible that someone here can do that, you're more likely to get help in a WordPerfect-specific forum or newsgroup:

Windows 10 update version 1511 to version 1803

I have a Dell D620. Originally it was on XP then updated to Windows 7. It would only update to Win 10 version 1511. I lost the hours of attempting to update, always failed. Today I got it to Windows 10 Version 1803. This is what I did.

1. Formatted the drive and re-installed XP

2. Updated the BIOS from Dell to A10.

3. Installed Win 7 Professional, did not format the hardrive.

4. Installed SP 1

5. Installed Microsoft update package about 250 mb.

6. Used an updater for drivers and updated them all.

7. Installed netframework 4.7

8. Disabled all Broadcom Drivers including BIOS.

9.  Loaded Win 10 1511 from disk

10. After loading used Windows Media Creators tool for Widows 10 version 1607.

I could not believe it took 1607 then automatically updated to 1803. It took time and patience and one switch off but it worked. Its something to do with the drivers and the Windows 7 update package. It took 32 hours including monitoring to do it!
