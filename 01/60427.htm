vijay b say:

Hi,

There is no way that you can delete guest account on your computer. You can rather turn it off or on.

Deleting guest account is a dangerous practice. I suggest that you turn off the Guest account following the below steps:

a. Click the Start button in the lower left corner of the desktop.

b. Click Control Panel.

c. Click User Accounts.

Click Add or Remove User Accounts.

You should see a line under the guest account which states, “Guest account is off”

If it is not, then:

a. Click Guest.

b. Click Turn off the guest account.

You can refer this article for more information on this:

http://windows.microsoft.com/en-US/windows7/Turn-the-guest-account-on-or-off


If you have two accounts with Administrative privileges, you can go ahead and delete one of the Administrator account.

To create a delete a profile, you must be logged in to the account that you would like to keep:

a. Open User Accounts by clicking the Start button , clicking Control Panel, clicking User Accounts and Family Safety (or clicking User Accounts, if you are connected to a network domain), and then clicking User Accounts.

b. Click Manage another account. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

c. Select the user account you would like to delete and delete it.
