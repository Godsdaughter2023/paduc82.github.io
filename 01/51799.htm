Problems with file or printer sharing using HomeGroup following a Windows 10 upgrade

ruizell rey say:

Hi Harold,

A PC may depart from a HomeGroup if it’s upgraded while disconnected from the home network, or if it’s the only HomeGroup device active on the home network at the time of the upgrade.  This can result in file and printer sharing issues on your home network, including your Xbox 360 not playing media files. 

If the upgrade was performed on the HomeGroup network and there is a HomeGroup error, the HomeGroup will need to be reset by following these steps:

  1. Make sure the PC is connected to a home network, and that no other machines running HomeGroup are connected.
  2. Stop HomeGroup using Command Prompt:
    • Search for “command prompt” using Cortana, then launch Command Prompt.
    • Copy and paste “net stop homegroupprovider” (without the quotes) in the Command Prompt window.  Then press Enter to run the command.
  3. Search for “regedit” using Cortana, then run the regedit command to complete these steps:
    • Navigate to the following registry folder: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\HomeGroup\SharingPreferences
      • If present, right-click on this Value: “UsersAndPublicShared”=1 and select “Delete”
    • Navigate to the following registry folder: Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Function Discovery\RegistryStore\Publication\Homegroup
      • NOTE: Do not remove this folder, it has special permissions
      • Right-click and delete the values in this folder.
  4. Restart HomeGroup using Command Prompt:
    • Search for “command prompt” using Cortana, then launch Command Prompt
    • Copy and paste “net start homegroupprovider” (without the quotes) in the Command Prompt window, then press Enter to run the command.
  5. Open HomeGroup in the Control Panel and create a new HomeGroup.

Note: If you see “join” then another PC has already been joined. Get the password from the other PC and join.

If you have additional questions on setting up a HomeGroup, please go here:

https://support.microsoft.com/en-us/help/17145/windows-homegroup-from-start-to-finish

Understanding downgrade rights

Consumer versions of Windows 8 Pro that come preinstalled on PCs have downgrade rights to a previous version of Windows (example: Windows 7, Windows Vista).

If your copy came pre-installed on your system, you may downgrade your copy to Windows 7.

If not you need purchase a copy of Windows 7 and install the same on your computer.

• Downgrade rights are only available for licenses obtained through OEM or Volume Licensing (VL) and not apply to Full version (FPP).

• OEM downgrade rights apply to Windows 8 Pro and allows for downgrades up to two prior version

• Software Assurance through VL (Volume licensing) provides the greatest flexibility, allowing for downgrades to all prior versions and editions besides Windows 7 Ultimate and Vista Ultimate editions.Without Software Assurance coverage, VL has downgrade rights to prior versions of Windows with the exception of Enterprise.

• It is not possible for a Windows 8 copy with multi-language function to downgrade to Windows 7 Pro or Windows XP Pro as multi language function is exclusive to Windows 7 Enterprise edition and not available for Professional edition.

It was a broken font called Arial.

petercon [msft] say:

Hi, Valvius.

We think we've figured out what's going on with the font issue. There are a lot of threads dealing with variations of the same issue, so instead of explaining here, here's a link to a thread with the details:

Wrong, "grafitti" font in apps and Web pages after upgrade to Windows 10

Your particular workaround was to install the European Union Expansion Font Update. As indicated by the system requirements for that update, it was intended for Windows XP and Windows Vista. Unfortunately, by installing that, you've installed much earlier versions of certain Windows fonts than the ones you had on Windows 10. The problem you were experiencing was not likely due to the Arial font on your system itself being broken, but rather due to some third-party font you had installed that was taking the place of Arial. (See the link above for more details.)

If you are not experiencing any problems now, then there's no particular concern. At some point, there will probably be a Windows 10 upgrade with a newer version of the Arial fonts, and at that point the fonts on your system will be current again.
