Simple Fix for Start button not working - Microsoft Community

twwinva say:

Simple Fix for Start button not working

Several days ago my Start button stopped working. After seeing the myriad numbers of posts on this topic, and trying several of the recommended fixes (restart, system restore, driver reinstalls, etc.) without luck, I discovered a simple fix from Settings that enabled the Start button to work again.  

Go to Action Center (click icon on far right of Taskbar)

Click on "Tablet Mode" to switch to Tablet Mode. (In tablet mode, when I hit the Start button, it worked fine)

Go back to Action Center and hit "Tablet Mode" again to toggle Tablet Mode off.

VOILA!  My Start button was back working again!

I hope this works for many others who have been vexed by this issue!

Terry
