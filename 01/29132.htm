Finally got it. Almost exactly as Crobuzon said above, with some additional detail.

  1. Go to the Win8 Start Screen
  2. Type "Device Manager"; a menu of search sources slides in from the right. Tap Settings in that menu. In the resulting menu of search results, tap Device Manager. A window will open in the Desktop.
  3. At the top of the new window, there is a View menu - open it and click/tap Show Hidden Devices.
  4. There is a tree of different 'devices'.
  5. First, open the Bluetooth tree with devices.
    ... Right click on each, choose Properties, and look for a Power Management tab.
    ... If there is a Power Management tab, go to it and uncheck 'Allow the computer to turn off this device to save power'.
    ... If there is no Power Management tab, repeat for all the Bluetooth devices in that tree.
  6. Next, open the Network Adapters tree.
    ... Repeat the same steps as above.
    ... The setting that I think did the trick for me was on the Power Management tab of the Qualcomm Atheros QCA9565 802.11b/g/n WiFi Adapter (I am on a HP ENVY Touchsmart Ultrabook).


  7. As a just-in-case step, I deleted the mouse & keyboard from Control Panel\Devices via the Desktop. I also uninstalled the Microsoft Mouse and Keyboard Center Software. Keyboard.
  8. Then I shut down, powered up, and reinstalled/paired using the Win8 Add a device option in the Win8 PC Settings\Devices Menu.

Just like before, I will return if this doesn't help. But I think, finally, I got it.

davem121 say:
Hi Jaime,

Start your PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

Please try each option one at a time, then see of Windows will boot . . .

Go to Troubleshoot - Advanced Option - Startup Repair
See if Windows startup can repair itself

Go to Troubleshoot - Advanced Option - System Restore
Check to see if you have a restore point you can go back to

If you do not have a Restore Point, go to Troubleshoot - Advanced Option - Startup Settings and click Restart
Upon restart, press 4 to enter Safe Mode
Once in Safe Mode, backup your data if you do not already have a backup
Open Device Manager and check to make sure all your Drivers are up to date
Reboot to see if Windows will start Normally

If the above fails, go to Troubleshoot - Advanced Options - Reset this PC
Choose to keep your files and apps, just your files or to keep nothing
Then start the Reset
