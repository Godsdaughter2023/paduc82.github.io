It is important to know that there are few independent entries in the Network system and various names can be assigned to them.

Having few names is not an indication that anything is functionally wrong with the Network.

The Network name as appeared in the System Window of the OS must be the same for all computers since it is what counts to the actual Network working between the Computers (File, and Printer Sharing).

http://www.ezlan.net/Win7/net_name.jpg

The Wireless SSID (Wireless Network name) is important too, but it can be different from the Network name since technically the two are unrelated.

The SSID can be changed in the Router's Wireless menu (see the Router's manual).

Some other Network’s name entries are just cosmetic and do not necessarily affects the Network working.

There is No functional importance to the Network number (like network 5).

It is an Id number, and Windows advance it every time that the Network adapters are reconfigured. If it shows Network Number 9, it does mean that you have 9 active Networks. It means that the Network core was thus far reconfigured 9 times.

Yes, you are correct that the Reset option and Startup Settings leading to Safe Mode are no longer a part of Advanced Options when

The Reset option and Safe Mode options can only be found within Windows 10 Advanced Startup Options. Found at Settings > Update Security > Recovery > Advanced Startup.    

So what is a person to do when stuck in an infinite repair loop trying to recover their system?  The only options left are reverting to old restore points, or trying Startup Repair (which never seems to work anyway) 

You might want to try booting from ISO installation media while tapping your boot options menu key # and choosing to boot from your normal HDD that is offered there in the boot options menu.  Sometimes this will jettison you to the Windows desktop, especially after successfully reverting your system to an earlier restore point.  Be sure to try the earliest and the oldest restore points.  Don't give up after one failure notice with restore points...try again.  Reverting can take 50 minutes to complete.  Be patient.  This morning I was able to revert to my oldest restore point but not my newest....again don't ask me why.

The other option is to Clean Install Windows 10  booting from installation media and choosing the custom installation option, wiping the HDD, deleting all partitions and reformatting afterward.  That may be your only option if other strategies are unsuccessful.
