sureshgarhwal say:

shipped with 3GB RAM and 320GB hard drive with windows 7 home premium.

C drive space decrease continuously 6 to 8GB with increasing virtual memory about 8GB and after this the screen goes blank. nothing to work. I have to directly power off to restart

cbarnhorst say:

Please answer my other question.  How is the HDD partitioned?  What are you using the rest of the HDD for?

My main points are to add ram and redirect paging.

The 1545 will accept 8GB of ram and that ram is inexpensive.  8GB of ram will improve performance dramatically because the only reason the paging file is being used heavily is that memory pages are being written to and read from the HDD constantly with only 3GB of ram.  With 8GB very few memory pages will need to be moved to make room for new memory requests.  If you are running a 32-bit version of Windows you should consider installing the 64-bit version (your present Windows license covers that) so that you can use the full amount of ram available.

Whether you can redirect the paging file as a short-term solution until you get more ram installed depends on what you are using the rest of the HDD for.  In modern Windows partitioning a drive leads to some inefficiencies (the read/write heads have to travel greater distances for no good reason).  Consider not partitioning your HDD.  Partitioning was fashionable back in the days of FAT32 file systems because partitions greater than 32GB produced inefficiencies in use of larger HDDs.  However you are not using FAT32.  You are using NTFS and no such issue exists anymore.

cbarnhorst say:

If the partitioning scheme is something you have always used, even back in Windows XP days, then consider abandoning it for a single partition drive.  The layout you describe is unnecessary with NTFS.  It leads to the sort of things you are experiencing and overworks the read/write heads of the drive, reducing its longevity.

I would backup everything and clean reinstall Windows.  During the installation process you will have an opportunity to use the Drive Tools.  Remove all partitions so that the entire drive is unallocated space.  Then install Windows.  The drive does not need to be partitioned or formatted first.  Just use it unallocated.  Setup.exe will provide the optimal layout which is a small System Reserved partition and one main partition.  Then reinstall your software and your files.  You will see the Product Key page during installation.  That is for new installations.  You are reinstalling.  Don't enter a product key (you don't have one anyway) but click the metalink at the bottom of the page that says "I don't have a product key" and let the installation complete.  Since your 1545 is already a registered Windows 10 device Windows will activate automatically.

You can backup your files to an external storage device, a network drive, or the cloud.
