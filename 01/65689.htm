irfan h say:

Hello Fokker99,

Welcome to Microsoft Answers Forum!

I understand that the Internet Explorer 8 is automatically downloaded on your computer from Windows update and you would like to remove it and want to roll back to Internet Explorer 7.

Internet Explorer 8 is an updated version of the browser that comes with many added features. Here is a link that will give you information on the features included in Internet Explorer 8:
http://www.microsoft.com/windows/internet-explorer/features/faster.aspx

It is recommeded that you have this installed to get your browser updated.

However, if you still would not like to install this, there are 2 ways that you can try to delete the downloaded update.

All the downloaded updated would be saved in the 'Software Distrubution folder'. Here are the steps to delete the downloaded updated:

1. Stop the Automatic Updates service:

Start > Search> (type in) services.msc > [OK]
Double-click Automatic Updates > Click on Stop
(Stopping the service will take a moment)

2. Delete the contents of the Download folder:

Start > Search> (type in) %windir%\SoftwareDistribution > [OK]
Open the Download folder and delete its contents
Close the window.

3. Go to Windows Update website > Select CUSTOM and allow the scan to complete > Uncheck/deselect the driver update, then "hide" it ("Don't show me this update again").

4. Start the Automatic Updates service:

Start > Run > (type in) services.msc > [OK]
Double-click Automatic Updates > Click on Start
(Starting the service will take a moment)

The second method would be to hide the updates.

Open control panel
Open Security center
Clcik on automatic updates
On the screen that shows put the dot in download but let me choose when installed.
Press apply or OK and exit out

I hope that this information helps.

Thanks.

1 Check the folder settings in Organise, Manage libraries, Music [I think this is what you have already done]

2 Check the whole set of WMP troubleshooters.

2.1 Control panel, Troubleshooting, Hardware & sound

then the online versions [this might be a complete duplication of effort but I've never been able to confirm that]

2.2 Media player library troubleshooter

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Library-troubleshooter

2.3 Media player settings troubleshooter

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter

3 Then, if necessary, run Tools [alt-t], Advanced, restore media library

If you have done all these things and the library will still not work correctly then do the following steps

4 Delete the WMP library database to force a complete rebuild - Close WMP, in the Start menu search box type%LOCALAPPDATA%\Microsoft\Media Player In the Windows explorer window that appears delete all the files [butnot sub-folders]. See if that has fixed the problem.

5 Delete the WMP database cache - Close WMP, in the Start menu search box type%LOCALAPPDATA%\Microsoft\Media Player In the Windows explorer window that appears, select the Media Player folder in the left-hand pane & delete it together with its sub-folders. See if that has fixed the problem.

6 If that hasn't fixed it then do both these steps [they are the closest thing possible to uninstalling then reinstalling WMP]

6.1 Unload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, clear Windows Media Player checkbox, Yes, OK, Restart the PC.

6.2 Reload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, set the Windows Media Player checkbox, Yes, OK, Restart the PC.

At the end of 6.2 you will effectively have a new installation of WMP.
