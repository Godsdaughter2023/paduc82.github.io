A yellow exclamation point next to a device in Device Manager means that Windows has identified a problem of some kind with that device is detected but not installed.

 

There are several causes of Code 39 errors:

 

·         A required device driver is missing.

·         There is a problem with the file I/O process.

  1. Unplug power connector from side of device.
  2. Unplug power supply unit from wall outlet.
  3. Unplug any USB devices plugged into USB port on power supply unit.
  4. Reseat cord from wall to power supply unit.
  5. Plug power supply unit back into wall outlet. If possible, try different wall outlet.
  6. Plug power connector back into side of device.

  • Check if there are any damage on the power supply unit. Make sure that you're using the power supply that came with your Surface.
