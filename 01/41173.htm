When I see a STOP error 7B and the second parameter ends in 00034 and the last two are all zeros I think that somehow the system is not properly communicating with the hard drive.  These kinds of problems are usually not problems with the Windows installation which is why none of the things you have tried so far have worked.

In the system BIOS there are settings that define the type of interface to the HDD and if they get changed or somehow get set back to a default/wrong setting you might encounter this error.

This can happen if the CMOS battery on the motherboard failed or is failing and then the system loses AC or battery power the settings can get into a state that is incompatible.

Since I don't have your laptop I can't tell you what the right setting, what the setting is right now or what adjustments to make so you will have to look for and find it yourself.

Looking at the Toshiba support pages it looks like to get into the BIOS after powering up you press the F2 key to get into the BIOS Settings, then look for something about Advanced settings (they have no images).

Before making an adjustment you should write down the setting the way you find it so if things get worse you can put things back like they were.

You can make an adjustment, save the settings and exit the BIOS setup and see how things look.  If it doesn't work make another adjustment knowing that you can always go back to the settings you wrote down before making any changes.

There may be an option to choose Default Settings which may work.

That's all I can think of at the moment but others may have some other ideas.
