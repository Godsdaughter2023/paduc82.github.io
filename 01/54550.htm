This error usually means that you have a few failed sends under your belt. Remove any unsent messages from the outbox (below the account folders in the folder pane).

Protocol SMTP

Port 587

Secure SSL: Yes

Roadrunner (TWC) doesn't support SSL for SMTP. Right-click on the account name and select Properties. On the Servers tab, ensure that My server requires authentication is selected. On the Advanced tab, deselect This server requires a secure connection (SSL).

When you've made these changes, close down Windows Live Mail and wait a few minutes for the program to do its housekeeping and store the new settings. Re-launch the program, then compose a new test message and send it to yourself.
