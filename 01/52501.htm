How do a convert a pdf attachment to a word document?

I received proofs for an article I wrote to make final corrections but when I open the file it is a set of symbols I cannot read.  Please help, I am on a deadline.
lemp say:
If you mean that you are opening a pdf document using Adobe Reader and the document appears to use a symbol font instead of something recognizable ... the file is damaged and you'll have to get the sender to re-send it.

If you mean that you are attempting to edit a pdf document by opening it Word, you can't do that.

You can edit (some) pdf documents if you have Adobe Acrobat (which, unlike Adobe Reader, is NOT free).  The only way you can edit a pdf document without Acrobat is to employ Optical Character Recognition (OCR) software.  If you have a scanner, such software might have been bundled with it.  Or you could see if printing the pdf and then scanning it would result in an editable document -- again, only if there is OCR software available to you.

You can request the sender to send you the document in Word form (or in Rich Text Format (rtf) which is openable by Word).

Or you can just print the pdf, mark it up the old fashioned way (with a red pencil), take it to Kinko's, Staples, or a similar office supply place and get it scanned into pdf and then send that pdf back.

thomas ferguson say: The indication that the file system is RAW usually means that there has been corruption of file system structures. How you go about recovery depends on the value you place on the data on the drive. If your budget of time or money is finite and the data of low value or is backed up-

Navigate to   Start   All Programs    Accessories    and click on the command prompt. When the window opens, type

cmd   <press Enter>
chkdsk D: /f /r    <press Enter>   Note: D: is the letter of the drive to be checked. Change it to the appropriate letter.

When the process completes, type
Exit

Go to    My Computer     to see if it has been repaired enough to at least get a directory listing.

If no joy, go to the web site of the disk's maker. Check for a utility to check the disk and do a "low level" format. Follow their instructions. After that, do a format to NTFS.

If that fails, it is likely that the disk will have to be replaced.

Mind you, there are tools (disk sector editors and the like) that you can use to read and possibly recover at least some data. It does require appropriate knowledge, skill and considerable patience. Alternately there are data recovery services-quite expensive.
