DOS command: search for the exact extension .doc - Microsoft Community

falcios say:

DOS command: search for the exact extension .doc

What is the DOS dir command for finding only .doc extensions? What is the DOS dir command for finding only .doc extensions?

Currently, it's finding .doc and .docx.

In Windows 7, you can search for ~".doc" but this doesn't work in DOS.

Thanks in advance.

the larch say:

I read some information and came up with a couple of alternatives:

dir "*.doc" | findstr /e /i ".doc"

The output of the dir command is piped to the findstr command.  The /e switch ensures that only lines ending in ".doc" will be shown.  The /i switch ensures that the searching is case-insensitive, so uppercase DOC and lowercase doc are both included.

dir "*.doc" | find /v /i ".docx"

Unlike findstr, the find command does not have a /e switch, but it does have the /v switch that allows filtering out lines containing ".docx" from the results of the dir command.
