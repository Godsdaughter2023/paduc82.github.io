greg carmack - mvp 2010-2020 say:

It means your User Profile is corrupted.  You can try to repair it by working through the Windows 10 Install Integrity Checklist completing every step including repairs. 

If you need an Admin Account to do these then Enable the (Hidden) Administrator Account on Windows 7, 8, or 10.

If these fail to restart into your User account at reboot, then you can create a new account to move into:  How to Create a New Local User Account in Windows 10.

Be sure to assign Administrator status to it. 

Then move your files from old User account over, test that all programs work from new User account or reinstall them.  When everything is working correctly, delete the old User account and hide the Hidden Admin account again to save for emergencies only.
