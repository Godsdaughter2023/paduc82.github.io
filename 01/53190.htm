paulfhyk say:

Daylight Time saving update during sleep mode

My UK-based Community Hall recently installed a door entry system which uses software stored on an on-site Windows 10 laptop. The laptop is normally left in sleep mode, but actively registers all door use while in that mode. It is only switched on when entry fobs need reprogramming, or for access audit purposes. 

Last weekend the clocks changed from winter to summer time (Daylight Saving time). A hirer could not gain entry on the following Monday. The laptop had registered her entry attempt as being an hour earlier (ie winter time) , but had updated to the correct summer time as soon as I woke it from sleep mode. Thereafter hirers have accessed with no problem.   

I do not wish to have to visit the Hall on the morning immediately following each six monthly clock change , merely to turn the laptop on and let it update the time.

 
Is there a fix which would allow the update to occur while the laptop is in sleep mode. 

Grateful for any thoughts  

bill.p. say: Hi Paul, I'm Graeme, an independent adviser here to help.

Windows isn't able to update the time settings while in sleep mode, since the network service isn't actively checking the time while in sleep mode.

The simplest solution for you to fix this issue would be to simply leave the laptop switched on when the time is expected to change to daylight savings or vice versa. This would allow the laptop to update the clock overnight, so you'd just need to leave it on for two nights per year.

Hopefully this helps! Please let me know if you need anything else. Thanks!
