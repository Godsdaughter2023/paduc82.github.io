Being a Steam and Windows 10 user personally, I'm quite familiar with some troubleshooting steps that might help resolve this error. Here are some things to try:

Method 1: Install the latest Windows updates

Method 2: Delete libswscale-3.dll and steamui.dll files

  1. Make sure that Steam is closed and not running in the background.
  2. Right-click on your Steam desktop shortcut, click Properties, and select Open File Location.
  3. Delete libswscale-3.dll or libwscale-4.dll and steamui.dll files.
  4. Restart Steam.

Method 3: Change your Steam to public release version

  1. Make sure that Steam is closed and not running in the background.
  2. Right-click on your Steam desktop shortcut, click Properties, and select Open File Location.
  3. Open the package folder and delete the file called beta.
  4. Restart Steam.

Method 4: Get past through the error message

  1. Open the Steam app, and wait for the error message to appear.
  2. Launch Steam again without closing the the error message dialog box. It should start updating the program and run normally.

Method 5: Using steam://flushconfig command in Windows

  1. Make sure that Steam is closed and not running in the background.
  2. Right-click on the Start menu button, and select Run.
  3. Type (or copy and paste) steam://flushconfig then press Enter.
  4. Wait for some time and restart your PC.
  5. Launch Steam from its installation folder (not from the desktop or Start menu shortcut), usually located in C:\Program Files (x86)\Steam\.

Hi EKahn,

When you remove a folder from Photo Gallery, you do not delete it from your computer. If you remove a folder, Photo Gallery will no longer display the pictures and videos that were in that folder, though the folder will remain on your computer.

You can remove folders that you have added to Photo Gallery at any time, but you can't remove the folders that are in Photo Gallery by default.
