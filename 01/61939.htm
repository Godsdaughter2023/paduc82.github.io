gclowney say:

The real problem is that even if the Caps Lock key is disabled, Windows can still get confused and toggle the key's state if you accidentally press it along with other keys.

 

The most common case if when using ALT+TAB; if your finger presses the Caps Lock key along with the Tab key, and the timing is just right, you end up with the Caps Lock toggled, even if the key was disabled. Because normal Caps Lock key presses are still ignored, you can't undo the damage by pressing the key, or using the SendKeys function or the On-Screen Keyboard utility to simulate a press. The only reliable way to turn off the caps lock status is to start the keyboard applet in Control Panel, enable Caps Lock, press Caps Lock to turn the status off, then re-disable the Caps Lock key again. This is inconvenient enough to be very annoying, if you end up having to do it once a week or so. This bug in Windows has been around for several major versions, and I really wish Microsoft would fix it

 

(It's possible that the original poster has a different problem, but I'd be surprised if that was the case.)

 

(The temporary "fix" is particularly annoying because the Microsoft Mouse and Keyboard Center is a non-standard application that is difficult to use with the keyboard alone.)

transfer files from Macbook Pro to PC windows 10 - Microsoft Community

margyquale say:

transfer files from Macbook Pro to PC windows 10

I have  given up on the apple and want  to transfer documents and photos to new PC that has windows 10.....any suggestions?
hairyfool say:

Although it should be possible to transfer over a network setting up the two devices so that one can see the others storage can be slightly problematic

The simplest option would be the use of a USB device but which type is best will depend on how much there is. Both operating systems support FAT32 so the Mac can write to the device and the Windows PC can read it to copy the files in.

The reason why the amount matters is FAT32 is not particularly efficient in terms of using space when there are a large number of files, the largest single file size supported is 4GB and Windows will only format a 32GB FAT32 partition.

The 1st limitation is academic, once you have done the transfer you won't need it again, the second won't matter if none of the files are that big and the third can be solved if you format the drive on the Mac as OS-X can format the disk larger and Windows will still read it.

Of course if the amount to transfer is smaller then a 32GB USB memory key  will work natively on both systems using FAT32. If just over then transfer the documents, clean the disk then transfer the photos. If you have only photos on the key during a session the Photo import tool will pick them up and organise them by date into the library.
