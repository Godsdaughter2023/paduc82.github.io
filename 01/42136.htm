"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"

bossdweeb say:

There are several versions of  Mobile Intel (R) Series Express Chipset Family Integrated graphics.
If the number on yours is lower than the one reviewed below it would have lower performance,
if higher it would perform somewhat better. It may also be another series.

notebookcheck.net/Intel-Graphics-Media-Accelerator-  4500MH

As you can see in the benchmarks in the review, you may be lucky that you are able to run the other games well. It may be that Spiderman 3 using a game engine that is not well optimised for a low end integrated graphics.

Some games have a wide spread in the settings from low to high settings that allows for lower end graphics processors to still run reasonably well with low settings, other don't.- Two examples of games that are well optimised for low end graphics processors  that I can think of off hand is the FIFA series and COD-MW (the first one).

- Some games also require more RAM than others, and some games don't run well on 'shared' RAM, which is when the graphics processor uses part of the system RAM as do all Integrated Graphics.

Some games require dedicated Video RAM (VRAM) to run well and others will not run at all without
VRAM. VRAM is only available on a dedicated graphics card.

- If your system can't be setup to allow adequate RAM the game would lag when loading the opening scene due to not being able to swap data quickly. Even if it were to manage to load the first level and run that smoothly the game may start to lag when loading subsequent scenes, or if more things start happening on screen, such as a larger number of enemies appearing.

    1. You press Windows Key +R and type in regedit

    5. You double click it, switch to Decimal (Hexadecimal by default), and type in after how many seconds you want the keyboard backlight to turn off (60 seconds by default)
    6. Restart your laptop for changes to apply
