dru - support engineer say: Hello, ebonyxxx and thanks for posting!

If you're getting the "your system has not been modified" error message, there are probably the corrupted remains of the prior iTunes installation still on your computer.  These can keep you from properly installing iTunes again.  I would recommend trying to find iTunes in your "Programs and Features" and performing an uninstall from there. 

To do this, just click on the "Start" orb and then click on "Control Panel".  Once in there, look for "Programs and Features" and click to launch it.  If you find iTunes in the list, select it and then choose "Uninstall" on the bar above.  Then try to reboot your system and install iTunes again.

If it doesn't show up in the list, try going to: http://support.microsoft.com/kb/290301/ and downloading the Installer Cleanup Utility.  Once downloaded and installed, run the program and try to find iTunes in the list.  If you see it, select it and choose "Remove".  Then close the utility, and restart your system.  Now try to install your iTunes again by right-clicking the iTunes installer, and left-clicking on "Run as Administrator".

Hello,

In setting up two display monitors, options like Duplicate these displays, Extend these displays, Show only on 1, and Show only on 2 are provided to customize the display settings. In choosing Extend these displays, which lets you extend your actual working area to the second monitor, a primary or main monitor is set up as well. The primary monitor is generally the one that displays the majority of details and hosts the primary desktop. Although the Taskbar is available on both the monitors, but the primary one is more functional and accessible. The icons you see in the main monitor cannot be displayed on the second monitor as well, this is the distinction between the main display monitor and the extended or second monitor, which is a software design.
