The problem is that windows does not have a driver for your hard drive controller.  (ULi SATA Controller)

I did a quick look online and it does not appear that ULi makes a windows 7 driver, you have a couple of options:

If you are unable to boot your device in to Windows, we suggest that you download and use an installation media to restore from a system restore point. Here's how:

  1. On a working PC, go to the Microsoft software download website.
  2. Download the media creation tool and then run it.
  3. Select Create installation media for another PC.
  4. Choose a language, edition, and architecture (64-bit or 32-bit).
  5. Follow the steps to create the installation media, and then select Finish.
  6. Connect the installation media you created to your nonfunctional PC, and then turn it on.

Let us try the following methods and check.

Method 1. Let us reset the TCP/IP and verify.

To reset the TCP/IP stack go to this article and follow the instructions.
http://support.microsoft.com/kb/299357


Method 2.
Step 1.
1. Connect to your router (by typing 192.168.0.1 or 192.168.1.1 in the address bar) and log in. Check your user manual or documentation if you do not know the log-in credentials.

Hi xXRaineXx,

 

It seems that, you need more information to enable Wireless network in System configuration Window.

 

Refer to the following link to enable Wireless network in System configuration Window. You have to enable only WLAN service. The following steps help you to enable the Wireless network to use Wireless Internet connectivity.

 

a.      Swipe in from the right edge of the screen, and then tap Search. Or, if you are using a mouse, point to the lower-right corner of the screen, and then click Search.

b.      Type msconfig in the search box, and then tap or click msconfig.

c.       On the Services tab of the System Configuration dialog box, check the WLAN service.

d.      On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.

e.      On the Startup tab in Task Manager, Check the WLAN.

f.        Save the changes by clicking on Apply and ok.
