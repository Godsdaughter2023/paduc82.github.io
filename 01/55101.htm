a. user say:

I cannot open my movie in Movie Maker

Original title: Cant Open My Files!

 

When I try to open my movie named "My Movie.wlmp" in Windows Movie Maker 2.6 it says "My Movie.wlmp is not a valid Windows Movie Maker project file."



Then when I try to open the same file on Windows Live Movie Maker it says "Sorry, Movie Maker can't open this project file. The file may not be compatible with Windows Live Movie Maker or be corrupted. Please select a different project file."


If you could help that would be great! Thank you staff and community!

considerate_guy say:

First, the .wlmp is a Windows Live Movie Maker project file.  It's the set of instructions for building a movie, but isn't a movie at all.  Additionally, that project file will only open in the correct version of Movie Maker.  If you open it in an earlier version (e.g. Windows Movie Maker 2.6, or even an earlier version of Windows Live Movie Maker from which it was created), the project file won't be able to open.

 

Furthermore, since it's a project, it doesn't contain any actual video, photos or music.  Instead it contains the names of these files on the computer where the project file was originally created.

 

If you are really trying to edit a project on another computer from the one on which it was created, you need both the project file and the source files.  You also need a matching (or newer) version of Movie Maker.  The latest version is Movie Maker 2012.

 

Alternatively, if all you want to do is view the movie (not edit it), then you could go to the original computer where it was created and in Movie Maker choose the 'Save as movie' option instead.  That will create a .wmv or .mp4 file that is playable on other computers with Windows Media Player.
