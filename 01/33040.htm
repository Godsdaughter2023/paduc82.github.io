In addition to the information that Shinmila H gave you, if you only have the one user account on your computer it isn't set up optimally and this may cause you issues down the line. The Shared Documents, etc. folders are system folders. You don't need to use them but they need to be left alone. If you need more information, please post back with a better explanation of what you mean by "share stuff".

User accounts - Recommended Setup (Vista and
You absolutely do not want to have only one user account. Like XP and all other modern operating systems, Vista and Windows 7 are multi-user operating systems with built-in system accounts such as Administrator, Default, and Guest. These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges on Vista and Windows 7 because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult. It isn't impossible to activate the built-in Administrator to rescue things, but it may require more work than you want to do. Best not to get into a bad situation to begin with.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. Running as a Standard user is best practice for security purposes and will help protect your computer from infection. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

If you want to go directly to the Desktop and skip the Welcome Screen with the icons of user accounts, you can do this:

Start Orb>Search box>type: netplwiz [enter]

unlink microsoft skype account - Microsoft Community

kathleensheffield say:

unlink microsoft skype account

How do I unlink my microsoft Skype account from my original skype on macbook air?
_elaine_ say:

Hello, KathleenSheffield

Hello, blaford

Unlinking a Skype account from a Microsoft account is not available. However, we can now transfer - one time only - a Skype account from one Microsoft account to another.  Here is how to do this:

Microsoft account if it's already been linked?

If I were to delete my account could I create a whole new Skype account using the same email address?

This simple question is complicated to answer, unfortunately.

If your Skype account has been linked to an email address that is a Microsoft email such as hotmail or outlook, then no, or not immediately.  This is because closing the Skype account also closes its associated Microsoft account. What is actually occurring is we are closing the Microsoft account, not the Skype account. In this scenario, relinking the Skype account you wish to close from this email address would be the best option - create a new Microsoft account as a "throw-away" account, carefully review how to re-link the Skype account you wish to close to it, and proceed with closing that new Microsoft account. 

If the email address is gmail, yahoo, or something else, that email address itself is not effected although the Microsoft account that the email address was used for, to set up the Microsoft account, can be closed.  After the account closure waiting period has passed, that same email address could be used to create a new Microsoft account.  Here's the rule to remember: one Microsoft account per email address.

Please remember that creating custom Skype account names was ended when all Skype accounts were transitioned to be basic Microsoft accounts in September/October 2016.
