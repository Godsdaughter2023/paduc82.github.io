If you want to stop check disc at start up you may use CHKNTFS utility. CHKNTFS is a utility that has been developed by Microsoft to disable disk checking on certain drives at Windows startup.

a. To use this utility go to Start –> Run –> type "cmd".

b. If you want to disable check disk on C: drive, use the following command on command prompt.

chkntfs /X C:

c. If you want to disable check disk on two or more drives, just keep on adding the drives:

chkntfs /X C: D: E:

This will disable check disk on C, D and E drives.
