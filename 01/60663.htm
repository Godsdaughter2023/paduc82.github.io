Sounds like you need to clarify a few things

Changing the WLID on the Xbox console is actually quite easy, You only need three things. a) the password to your existing WLID, b) a new WLID and c) the password to that new WLID

So you say that you have changed the password to your existing WLID several times? When you changed the password did you actually recover your profile back down onto the Xbox? Sometimes if you make significant changes to your profile you need to recover your account again to help it update.

To avoid speculation I will include instructions on how to changeyour WLID(apologies to anyone that already knows)

1) Sign in as the profile in question,press the guide button on the controller, move to the settings menu and select account management

2) Move two spaces to the right and select Windows LIVE ID | select the option to change WLID

3) Follow the on screen prompts and first enter the password to your existing WLID, then the new WLID then the new WLID's password

If in step 3 when you enter the password for the WLID and you are getting a message saying your password is wrong, that's pretty much what it means.

If you want to make sure the password you are using is correct, go to http://login.live.com enter your WLID and what you believe the password to be, if it logs you in, you're all good. If you do that and you still cannot change the WLID, it means you have to recover your profile first

lemp say:

If you actually have an account named "System Administrator" that one thing. 

On the other hand, all versions of Microsoft Windows since Windows XP have a built-in account named "Administrator" that can not be removed.  In Windows 7, the "Administrator" account is disabled by default.

However, it sounds like you're referring to error messages like this:

The reason you get messages like that is because in Windows 7, even if your account is an "administrator" account, it normally runs in "standard user mode" in order to protect your system from malware that may be running in your account.  In order to take actions that potentially might harm the system, Windows requires that you take a deliberate action to "elevate" programs that attempt to do those actions.

In the example above, I was attempting to save a file from Notepad into the directory where application programs are typically saved.  In order to do that, I would have had to right-click on the shortcut to Notepad and select "Run as administrator."  Having done that, I could save the file in the C:\Program Files (x 86) directory.

Sometimes you get a slightly different message.  For example, this happens when trying to delete that file that I just saved:

Because I'm logged in as an "administrator," all I have to do is click "Continue."  If I had been logged in as a "Standard User," I would have been asked to provide the user name and password for an administrator account.

This is all handled by what Microsoft refers to as User Account Control (UAC).  You can disable UAC and not get the types of messages discussed above, but that's not a good idea.  For more information, see https://www.digitalcitizen.life/uac-why-you-should-never-turn-it-off

bossdweeb say:

 Using a RAM Drive does not improve frame rates. It only loads areas faster.

So if all you want to achieve is quicker loading times RAM Drive may help some.

[EDIT: There is a suggestion that with some games using a 5400rpm HDD, can also cause stutter due

to the slow loading times. ]


- Most recent open world games load files into RAM preemptively according to what the player is doing at the time,

which helps make a more seamless transfer between areas/cells than in the past,

but even though a RAM Drive can  speed up loading areas, you still need to load

the whole game onto the  RAM Disk  (otherwise the game does not know where to look for the files).

- Filling a large RAM Disk can also be a bit tedious, as it slows Windows boot,  and shutdown times.


- If you cannot achieve adequate frame rates with the graphics settings you wish to use there is

no cheap way out. A higher end graphics card with 2-4+GB VRAM  would impact on gaming performance far

more than using a RAM Drive (the games texture resolution, level of Anti-aliasing, and screen resolution are major

modifiers).

I believe some flight sims  are also somewhat cpu bound, so a good cpu is also beneficial.

- Using a RAM Drive would be more beneficial when frequently opening large files such as high res images,

and possibly (depending on cpu power) saving large files, such as video.
