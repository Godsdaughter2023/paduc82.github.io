You probably can't stop it.  Is the email to her / him?  If the To is not to her, then set up two message rules.  The first rule is if the messages are to her email address, then stop processing more rules.  Then set up a second rule subsequent to the first rule to move all messages to a junk folder.  That way any message to her will wind up in the inbox and any messages not to her (most spam uses the bcc field) will wind up in a junk folder that you can then screen before deleting.
