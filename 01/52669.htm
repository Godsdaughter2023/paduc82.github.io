nicholas_w_ say:
Hello Vickibays

There are a variety of steps you can at least try before you go to your ISP. 

First, are you connecting to the internet through a wired connection or are you connecting wirelessly? 
If wireless and the system is a laptop you may have a wireless switch on the system that is turned off. If this is the case then turn it on. 
You may also want to boot to the bios and verify that the network card is enabled. 

Check to see if you are getting a valid IP address. 
Click on the start pearl. In start search type in CMD. Right click on cmd.exe and select the choice to run as administrator. You may receive a pop up window from user account control asking you if you like the application to make changes. Select yes. You will be presented with the command prompt box. Type in ipconfig /all. If your connect to the internet is wired then you will be looking for Ethernet adapter Local Area Connect. Find IPv4 Address. Write it down. 
If your connect is wireless you will be looking for Wireless LAN adapter Wireless Network Connection. Again find IPv4 Address and write it down. 
Now if you are connected to network then you should be getting a valid ip address. Generally ip address that are assigned by your router will start out with the numbers 192.168. If you see this then your connection to the network should be fine and the issue could possible be with the ISP. If you are getting an ip address of 169.254 then you most likely not connected to the network as that is a default ip assigned by Windows. Either way, try power cycling the network. 
Power-cycle basically means to power down everything such as your computer, router and modem. Leave them turned off / no power for at least 3 to 5 minutes. Once you have done that I want you to power on the modem first. Once it is up and completely running then I want you to power on the router. After verifying that the router is completely powered on and you have indication that it is receiving an internet connection (normally a light on the front or top of the router that is either marked or looks like a globe and is flashing) turn on the computer. Test to see if the issue is resolved. 

Try these steps and let me know the results. Thanks.

Transfer pics to Flash Drive on Windows 10 - Microsoft Community

karenvks say:

Transfer pics to Flash Drive on Windows 10

I recently bought a new computer.  I want to transfer my photos on old computer to new one.  How do I do that?  I insert the flash drive, and it only gives me an import option, not export or copy.

a. user say:

Hi,

Thank you for posting your query on Microsoft Community.

Please follow the steps to copy Photos to flash drive.

  • First, insert the USB memory stick into an available USB port.

  • On Windows, this will bring up a window showing the memory stick is now seen as a drive on your computer (e.g., "Removable Disk" E: drive). You'll also get a menu of options for using the device. Select "Open folder to view files" at the bottom.

    Alternatively on Windows, you can also get to the drive by going to File Explorer by pressing Windows and E key, then click on This PC.

  • To copy files to the USB memory stick, drag them from the Windows Explorer folder to the drive or right click on file and select Copy then open Flash drive, right click Paste.

  • When you're done, right-click on the drive and select "Eject" so you can safely remove the drive.

Hope this helps. Please reply if you need further assistance.

Thank you.
