    By chance are you running your display at either 125% or 150% (Control Panel, select Display). If so, your menus are garbled because applications, including Windows, are not DPI aware. In other words they don't deal well with large magnifications. Two ways to deal with this:  1) reduce your Display setting (go to Control Panel, select Display, then go to the left-hand menu and select Set custom text size (DPI) and adjust the slider bar);  2) select Compatibility settings as follows.

     

    For every program that is a problem, right-click on it and select the Compatibility Tab from across the top row, select Run in compatibility mode (first dialog box) and select Windows 7, then go to the last check box in the listing underneath check the box by the setting which reads, "disable display scaling on high DPI settings", select Apply and then OK to back out. This last approach doesn't work all the time or with every program, but for those it does work with its the proverbial saving grace.
