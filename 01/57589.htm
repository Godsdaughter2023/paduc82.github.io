  • Characters that might interfere with the conversion to CSV, like " , ;

  3. Open Windows Contacts (type contacts into the start search box and press Enter), click Import and select vCard.
  4. Point at the temporary folder and select all the VCF files (Ctrl-A) for import. This will create .contact files for each contact in Windows Contacts.

a. user say:

That can be a sign that the coin cell battery on your motherboard needs to be changed. That battery powers the motherboard's clock - which is where Windows gets the time - so that the clock keeps running even when the computer is powered off.

However, those batteries last for years, so your computer would have to be fairly old for the battery to be the culprit here.

The other possibility is that Windows is not syncing with an online time server. Windows does this to keep the time accurate. You can check this by signing in to an Administrators account, and then right-clicking on the date/time in the notification area. Make sure that 'Set time automatically' is turned on.

How to Remote Desktop connect from a MacBook Pro to a PC? __Remote Desktop Mac to PC?

What are the steps that I need to take to set this up?

What are the steps that the PC needs to take to set this up?

 

sandeep ghatuary say:

Hi NoNamePro24,

Try the steps mentioned below:

Step 1:  Start up your Mac and PC. Make sure they are both connected to the network before attempting to access the PC's files.

Step 2:  On the PC, go to Control Panel>System. Check the name of the workgroup on the network. If you are having trouble, the default for Windows XP is MSHOME and for Vista is WORKGROUP.

Step 3:  On your Mac, open up the Directory Access application in Utilities. You might need to click the lock in the lower left-hand corner in order to make changes. Double-click on SMB/CIFS, and change the Workgroup name to the same name as the workgroup on Windows XP.

Step 4: In Finder, click on Go>Network. Locate the name of your Windows PC. Login using your Windows login.

You may also contact the  Mac support  for more assistance and information.
