Its 'Manual'

If your updates are not installing automatically, see if the following below helps.

Before carrying out this procedure, disconnect from the Internet. That means turning off Wi-Fi and disconnecting from a wired connection. If you don't, Windows 10 will indicate some files are in use and cannot be modified or the command will be non-responsive. Also, you should restart your computer after disconnecting from the Internet. The easiest way to do it is to enable Airplane mode from the Action Center - Windows key + A.

Open Start, type: CMD
Right click CMD
Click Run as administrator

Type each of the following then hit Enter

taskkill /f /fi "SERVICES eq wuauserv" (Do this multiple times)
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
rmdir C:\Windows\SoftwareDistribution\DataStore
rmdir C:\Windows\SoftwareDistribution\Download

When complete, hit Enter, then restart, then try updating again.

Performance issue regarding memory - Microsoft Community

ryan redfield say:

Performance issue regarding memory

I recently noticed my laptop lags and a bit slow to open folders and stuff, so i Opened task manager

in performance tap in memory section i noticed Cached is taking 1.3 GB compared that to my other PC

and it was. I always thought Cached memory was meant to speed things up lol but why is it Caching everything?

screenshots were taken after restart 

cmdrkeene say:

Cached Memory is "free memory" that hasn't been erased and used by something else already yet.  You can consider it essentially open/empty memory, because it can be instantly cleared in under a millisecond if the need arises.  It's the same as hard drive space after you delete a file:  technically the data is still there, but that location is free to be overwritten with something new later, so it's the same as being "free" space.

As for why that computer has more: easy. You have more memory on that computer, so more of what was loaded and now free is still cached. The other computer doesn't have as much memory, so when RAM is needed, it more quickly overwrites the "free" memory with what comes next.  You can learn more about this under Tip: Make Sense of Memory Management and Key Memory Measurements

It doesn't contribute to your speed/performance at all.  To find out why your computer is sluggish, you'd need to look at what is actually using the CPU (your RAM is half empty and is not contributing to your slowness).  Learn more under Fix a Sluggish, Slow PC.

Hi,

1.       What you mean when you say that “can't get my work computer to work at my house”?

2.       When do you get this error message?

3.       Did you make any changes to the computer before the issue occurred?

 

This could be an issue with the connection between the cable and the port.

· The network cable is defective or damaged. Examine the cable for breaks, for tears in the outer covering, or for damaged connectors. Test the network connection by using a network cable that works. If another network cable resolves the issue, replace the defective network cable.

· The network cable is plugged in securely at both ends, but the network device may not be turned on. Verify that all network hardware is plugged in to a power supply if it is required and is turned on. To resolve this issue, you may have to reset the network device. Some network devices have a reset button. Other network devices require that you turn off the power, wait for several minutes, and then turn the power back on.
