how do i fix this error code ox80041321 for windows 7 - Microsoft Community

dennishatley say:

how do i fix this error code ox80041321 for windows 7

how do I fix this 0x80041321

a. user say:

Hi Dennis,

 

You will get this error message when you create backup using Windows backup.

 

I suggest you to try the following steps and check if it helps.

 

You may try running the following steps in elevated command prompt and check.

 

Step 1:

 

a. Open command prompt by clicking Start and typing command prompt in the start search box.

b. Right click on command prompt and select Run as Administrator.

c. Type cd %windir%\system32\tasks\Microsoft\Windows\WindowsBackup

d. del AutomaticBackup

e. del "Windows Backup Monitor"

f. Restart the backup again by opening “Backup and restore center” from the Control Panel”.

 

Step 2:

 

Delete the task from the task scheduler

 

a. Click Start, type task scheduler and hit enter.

b. In the left pane, Navigate under Task Scheduler Library->Microsoft->Windows->WindowsBackup folder

c. Delete Automatic Backup task

d. Delete Windows Backup Monitor task

e. Try to Restart the backup again by opening “Backup and restore center” from the Control Panel.

 

Hope this helps.
