Change .DNG photo files to .JPEG files? - Microsoft Community

peterimitchell say:

Change .DNG photo files to .JPEG files?

How do I change multiple .DNG photo files to .JPEG files? I can do one at a time but need to do many at once

a. user say:

Hi Peter,

Thank you for posting your query in Microsoft Community.

You will need to download the latest Microsoft camera codec in order to open raw .DNG files and install it in your computer. Here is the link on where to download this codec.

 

Microsoft Camera Codec Pack (16.4.1734.1104)

 

After installing this codec, open the .dng (RAW) file in Photo Gallery.

 

If you want to convert the .dng (RAW) to JPEG, you will need to use the option of Photo Gallery to make a copy of that RAW picture. To do this, please follow these steps:

 

1. Launch Photo Gallery.

2. Double click the RAW photo and click the option Make a copy under Manage category.

3. Rename the file and change the file format as JPEG and click the Save button.

 

NOTE: You can only apply the steps above to your raw photos one at a time. If you want to convert the raw photos in bulk, you will need to use a third party application to do so. You may use www.bing.com or any of your preferred search engines to search for that converter.

Hope this information is helpful. Please feel free to post in case you get any other issues related to Windows, we'll be glad to assist you.

neilpzz say:

In that case I would consider creating another account. You need to create an administrator account, by default windows will create it as a standard account so be sure to make it administrator. (and local)

There could be two routes to try once you have the account setup. Login to the account and try system restore again. If that finishes successfully then try the old account name again.

If the restore still fails then try restarting the PC a few times with the new account name to make sure new account is working properly. If you don't get the error message using the new account then you can look at copying the files from the old user folder to the new one. Files should be located in C:\Users\[oldname]\Documents, Pictures etc.

I can't offer any explanation as to why the problem started. Searching suggests that some think it was caused by 3rd party virus scanners. (you could try disabling a 3rd party scanner, if you use one) I had similar in Windows 7 many years ago and created the new account. It actually also fixed another problem I'd had at the time so I assumed the profile had got corrupted.
