Intel 4400 Drivers as of 2/25

saltgrass say:

The original Windows 7 installs did not have a Recovery Partition, just the 100 MB system partition.  The recovery options were kept on the OS partition.  Windows 8 and maybe later versions of 7 started using a Recovery partition in order to make your system bootable to the Recovery options even if your OS partition was corrupted.

Be aware, there are two types of Recovery partitions. One used for the recovery options and one to hold the Factory Image, usually around 15 GB and the last partition on the drive.  None of my systems have an OEM provided Recovery Image partition since I don't have any OEM systems.

I put my image in the C: \Image folder so I was on other than the Root partition.  If you use the Network Location option, you should be able to find such a folder on the drive.

I will point out, making an image backup on a network is much slower than attaching an external USB 3.0 drive directly to the computer.
