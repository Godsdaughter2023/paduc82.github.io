On Windows 8 the Microsoft Camera Codec Pack is installed automatically via Windows Update.  If you run Windows Update it should install the latest version of the codec pack.  In many cases it might have already been installed.  You will not see it listed in the list of Programs and Features like the codec pack was for Windows 7 but under the Installed updates link.

¡firedog say:

If I add an item to S Planner on my Galaxy S3, it does not appear on my WLM calendar.

The item you add on your S3 must first be synchronized with the calendar at http://calendar.live.com. This may or may not happen immediately, depending on how synchronization is configured on the device. Once it's entered there, it will be available for synchronization to the Windows Live Mail calendar. This synchronization does not happen automatically - it has to be triggered.

... when I manually add that item to WLM, it copies to the phone

Yes, because adding an item triggers a sync. You may also be able to force a sync by pressing Ctrl-1 and then Ctrl-2 to switch windows from calendar to mail and back again, by clicking Work offline then Go online, or by using the right-most button on the Home tab of the ribbon to Sign out... followed by Sign in after the default calendar has loaded. If a sync takes place, you will see the message Your calendar has been updated at the right-hand end of the status bar.
