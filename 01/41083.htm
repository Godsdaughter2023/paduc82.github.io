I don't understand your question.  Why are you using the on-screen keyboard?  Is your regular keyboard broken?

What "keyboard picture" are you trying to remove?

When you use the Print Screen key (whether on the normal keyboard or the onscreen keyboard), an image of the entire desktop is captured to the clipboard.  If you are using the onscreen keyboard, it will be part of the image.

If you use the ALT key in combination with the Print Screen keyboard, only an image of the active window (the window with the Title Bar "lit up") is captured. 

The onscreen keyboard itself doesn't appear to "count" as an active window.  That is, I don't think you can get an image of just the onscreen keyboard without the rest of the Desktop showing.
