    If you're using the volume & mute buttons on the remote, that behavior is By Design.  MC is meant as a '10-foot' interface; the remote behavior reflects this.

    Workaround for you:  I'd imagine that you're running MC in 'windowed' mode (rather than full-screen), yes?  If so, open the Volume Mixer (right-click the volume icon in the Notification Area, or type volume mixer at the Start menu), and mute MC there.
    Here, too, lowering MC's volume will not lower the system volume (unlike using the remote); but, raising the MC volume will raise the system volume.  FYI.
