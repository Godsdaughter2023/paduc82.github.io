THIS IS THE FIX FOR WINDOWS MEDIA CENTER NOT DOWNLOADING THE GUIDE!

This is also helpful if you reinstall your operating system and have this problem as I did.
I spent about 24 hours and multiple web pages trying EVERY possible solution more than once and in
various combinations.
If you want to fix this issue, FOLLOW this format. It is a little tedious, but it WILL work.


FIRST-
-Click Start.
-Select Programs.
-Select Accessories and locate the Command Prompt shortcut.
-Right click the Command Prompt shortcut and select Run as Administrator from the drop down menu.
-Click yes.
-Type ipconfig /release in the command prompt.
-Wait a few seconds for a reply that the IP address has been released.
-Type ipconfig /renew in the command prompt.
-Wait a few seconds for a reply that the IP address has been re-established.
-Type ipconfig /flushdns in the command prompt.
-Close the command prompt and reboot computer.


SECOND-

Click start button. Click on help and support. In search box type: show hidden folders. On # 1 click to
open folder options. When that window opens, click on the view tab at the top. Under files and folders, click on show hidden files, folders and drives. If the show hidden files, folders and drives happens to be already filled in on the circle, just close the window; Otherwise, click apply, then OK.


THIRD-

Click the start button and then click COMPUTER or my computer, (depending on your operating
system). Click on C: DRIVE and then you should see a folder that is slightly faded out that is labeled as program data; double click it to open it. Then double click to open a folder labeled Microsoft, then double click to open a folder labeled ehome.
Delete everything that is within the folder "ehome". If some of them won't delete because they say they
are being used in another program, just delete whatever you can then reboot your computer and then repeat everything that you just did within "THIRD" and finish deleting all of the folders and files in "ehome". And don't worry, the system will recreate the folders and files that are needed when you access windows media center.


FOURTH-

Go to Control Panel . If you're using the icon view, find Programs and Features in the list and click on it. If you're using category view, then under Programs it shows uninstall a program. Click on that and in the left margin, you'll see Turn Windows Features on/off, click on it. When that window opens, you'll see down on the list "media features". You'll see to the left of this a "+" symbol inside of a box. Click on the "+" symbol and uncheck Windows Media Center. A window will open and ask if you want to continue. Click yes. Then click OK. After it is through configuring, it will want to reboot. Go ahead and click yes to reboot.
After the reboot, go and repeat the above process in FOURTH. This time you will replace the check-mark next to Windows Media Center, then press OK. It will configure again and then finish. Go ahead and close out of everything at this point.

FIFTH-

This is where you will be introducing your own EPG guide information. This is the heart of everyone's

problem when Windows dropped their old EPG guide client. This is very easy, so don't panic.
Click the start button and click 'all programs'. Then click on the 'accessories' folder and "right click"
on "notepad". Then click on "run as administrator". Then click yes when the window opens. When notepad opens, click on file then "open". In the file name field that opens, type: C:\\windows\system32\drivers\etc\hosts
Click open. You'll then see most likely, two ip addresses under where it says "for example". This will be under all of the typed words explaining about this being a sample host file used by Microsoft windows, etc. blah, blah, blah. All of the beginning two or three paragraphs don't matter to you.
Now where the most likely two ip addresses are: delete how ever many there are, but wherever the first
ip address starts, this is where you'll be typing in "your new ip address". Do not mess with the local host name resolution ip address that is on the bottom of the window content.
You'll be using TITANTV.com as a source for your new ip address for the guide information that is to be
installed. FYI, even though this is the entity that I used, (because it is fairly universal for anyone to use, just by using their zip-code), Rovi ended up as the guide source that showed in the "about the guide" in media center, under guide, under tv, under tasks. But Titantv seemed to trigger the OTA guide information. I say this because for the first time since I have ever used media center, it actually showed my city and OTA digital signal, when I went to set up the tv signal and went through this process of it being able to download guide information and not going through the beginning of media center setup.

SIXTH-

Make sure you use MY SETUP verses starting at the beginning of the Windows Media Center setup.
It is possible you might have to initially put your zip-code in the Windows Media Center before you
start "my setup", BUT DO NOT proceed beyond that point. Back out of the setup and proceed with "my
setup". When I was having all of my trouble, I still had my zip-code in place before I started "my setup". BUT try "my setup" first without touching the zip-code and see how it does. Otherwise go to SEVENTH.


SEVENTH-

Now go to medic center and open it and under "tasks" click on settings. Click on tv and then on tv
signal. Follow the directions and let it download the tv signals and then it should download the guide listings.

This is the only way I was able to get this back.
And when it was downloading the tv signals, it made a point about it downloading the guide listings. I knew then it was going to work with all of this format.


EIGHTH-

steadfaststorm say:

Hi there. I'm sorry that you're having trouble with your Xbox 360 console.

The E68 error code indicates that the Xbox 360 console has a hardware problem. A possible solution is to remove and reattach the hard drive using the following instructions:

1. Turn off the console.

2. Unplug all accessories connected to the front or back of the console, including video cables, network cables, and USB devices.

3. Position the console horizontally.

4. On the right side of the console, locate the hard drive cover release close to the back of the console.

5. Slide the release toward the front of the console while pulling the hard drive door open.

6. Pull the tab to remove the hard drive.

7. Reconnect each accessory. Turn on the console (with the hard drive still out).

If the problem persists with the hard drive removed, you will have to get the console repaired. Request a repair. If there is no red light around the power button, reattach your hard drive. If the problem persists only with the hard drive attached, order a new Xbox 360 Hard Drive from the Xbox Online Service Center (sign in with your Microsoft account). Console registration is required to replace a hard drive under warranty. You can request a repair here: myservice.xbox.com/.../ConsoleManager.aspx

Depending on how the repair process goes, the data on your hard drive may or may not be recovered.

Please let me know how it goes, or if you have any further questions. Happy Holidays!
