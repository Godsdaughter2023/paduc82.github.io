NTFS is the recommended file system for Windows 7. While FAT32 is an older file system it is more compatible between different versions of Windows (even Apple Macintosh?) so it is useful for external drives to be formatted that way. Windows 7 can read a FAT32 drive.

If you haven't done so already, download and run the Windows 7 upgrade advisor software on the computer you want to upgrade. It will identify most if not all of the potential software or hardware issues you might get with Windows 7. You will almost certainly have to find drivers for the motherboard chipset and some of the other internal hardware.

http://windows.microsoft.com/en-ca/windows/downloads/upgrade-advisor

When you buy a retail copy of Windows 7 upgrade it comes with 2 DVDs. One DVD is the complete Windows 7 in 32bit version and the  other is the same thing in a 64bit version. You could install either one but for compatibility with the computer hardware, unless your XP is a 64bit version you'd be best off to install the 32bit version of Windows 7.  

The 3 discs you got with your laptop are likely a System Recovery disc set from the laptop maker. They would be used to reinstall Windows and all of the drivers and software that originally came with the laptop.

In case you were thinking that way, System Recovery discs are locked to the computer model they came with and won't install on different computers.
