volunteer j say:
I keep all of my files on a thumb drive.  I would like to make a copy or a back-up of that thumb drive using another thumb drive.
=========================
If you have two USB ports...
place one thumb drive in each one.

If any windows open...close them.

Go to...Start / Computer...

Each thumb drive should be recognized
with a drive letter...

Example:

Removable Disk (E:)
Removable Disk (F:)

Left click the drive letters to display a
directory of the drives.

Now you can Copy/Paste or Drag/Drop
the files from one drive to the other.

If you only have one USB port...copy all
the files to a folder on your desktop and
then copy them back to the other thumb
drive.
