ruwim.b say:

There are currently two different Skype applications that can be installed and used on a Windows 10 computer.

 

1. The Microsoft Store Skype for Windows 10 App (Application version 14.34.81.0/Skype version 8.34.081).

 

2. Skype for Desktop (Skype version 8.34.0.78). Skype for Desktop is a standard Win32 application, and has nothing to do with the Microsoft Store App.

 

These are two different applications and its not correct that the 8.34.0.78 version will be updated to the Windows Store version.

 

Skype for Windows 10 (the Store App) has now the Split View Mode available. Skype for Desktop has not this feature available yet.

 

Skype for Desktop has the system tray icon, Skype for Windows 10 has not yet this option available. There are several other differences between these two different applications.

 

Note that you can have both applications installed on your computer at the same time, and they will not conflict with each other. You can run and test both applications and see which one suits you best.

If you open skype.com/en/get-skype link in the Edge browser, you will see two options available to download Skype on a Windows 10 computer.

One download option is for Skype for Windows 10, which will download Skype App form Microsoft Store.

 

The other option is for Skype for Desktop application, and this link will results in downloading Skype from this location:

https://endpoint920510.azureedge.net/s4l/s4l/download/win/Skype-8.34.0.78.exe
