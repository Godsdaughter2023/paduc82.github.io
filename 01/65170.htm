Hi Swaroop,

Thank you for posting your query in Microsoft Community.

From description I understand that you need to activate Product key to your system. I would like you inform that, you can activate your Product ID through phone, command prompt etc.

First of all, I suggest you to check if the Windows 8 sticker is still present on backside of your laptop.

If present, proceed with the below steps. If not, please contact the OEM (Original Equipment Manufacturer) and check if it helps.

Before activating, please download an ISO file of Windows 8 from the given link below:

http://windows.microsoft.com/en-GB/windows-8/upgrade-product-key-only

To activate the product key, please follow any of the below steps:

Step 1: Activation from Start screen.

  1. Press Windows + C keys from keyboard on desktop (opens Charms menu to right side of the screen).
  2. Click on Settings, and on Change PC Settings.
  3. Select Activate Windows and click on Activate.
  4. Here, a window pops up to enter your Product/License key. Enter the digits.
  5. Windows will now automatically detects whether the product key is valid. If your product key works, click Activate.
  6. If Windows Activation is successful, click Close.
  7. If activation was not successful, check the key and try entering it again.

Step 2: Activation using Phone.
If you can’t activate Window over Internet, then try to contact Microsoft by phone and activate your product.

  1. Press Windows + C keys from keyboard on desktop (opens Charms menu to right side of the screen).
  2. Click on Settings, and on Change PC Settings.
  3. Select Activate Windows and click on Activate by phone.
  4. Here, select the country/region from drop down menu, and click Next.
  5. Call the toll free number provided.
  6. Follow the instructions from the telephone system and write down the confirmation ID.
  7. Enter the Confirmation ID numbers into fields A through H, in the same sequence as the telephone system provided them. Later, click on Activate.

 If product activation was successful, a success screen is shown and you are done.

Step 3: Activation through Command Prompt.

  1. Press Windows + C keys from keyboard on desktop (opens Charms menu to right side of the screen).
  2. Select Search icon and type ‘cmd’ in the box. Click on Command Prompt to open Command Prompt window.
  3. Here, type “slui 3” (there is a space in between) and press Enter.
  4. Now, a Windows Activation panel appears and asks to enter the product key.
  5. Enter your Product/License key and click on Activate.

Note: If you are in trouble, then try checking your product key once again. If still not working, most probably your product key will be wrong or useless. You can always go for the second method and activate by downloading the Windows 8 activator.

Note: These steps will work out successfully only with an original product/license key that has been given to you in the time of purchase/download.
