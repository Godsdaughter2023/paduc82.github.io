Solution finally! "Size all Columns to Fit" as default - Microsoft Community

bob is interested say:

Solution finally! "Size all Columns to Fit" as default

This is for Details View.

The key is to drag the columns so the columns that do NOT change size much (like Size, Type, Date Created or Modified, etc) to the left of the Name column because the width of one column affects the others to its right. Now drag the Name column to the far right of any others because that column's size changes the most. Drag the width marker of the Name column to the far right of your screen. And of course one needs to set "Apply to all Folders" in the View settings so its always displayed this way. I find this principle of always placing columns that are narrow, or which do not change size, on the left side of the name column. This works well in sorting anything. For further sorting efficiency, always show the date as year, month date so it always sorts properly. Maybe one day this will be a standard. Set this in Region or Regional Settings in Control Panel.

Hi Jdh,

 

If you are running out of room on your Remote Desktop Connection window and want to increase the screen space of the remote computer, you can do so by increasing the size of the Remote Desktop window. Making this window wider increases the virtual "desktop" of the remote computer so that programs you open through Remote Desktop have a larger space in which to operate. Your Remote Desktop Connection client window must be closed before you can adjust its settings.

 

Here are steps to resize the Remote Desktop Window, follow these steps:

a.       Press Windows + Q key and type Remote Desktop Connection

b.      Click on "Options"

c.       Click on the "Display" tab

d.      On "Display configuation" settings, you can change the "Remote Desktop Connection” display by moving the slider from "Small to Large".

e.      By moving the "Slider" all the way to large, the display settings will automatically set to "Full Screen".

 

You may also refer to the following article for more information:

 

Remote Desktop: Frequently asked questions
