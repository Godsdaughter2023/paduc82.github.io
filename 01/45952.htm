Ecco qua una guida per aprire le porte

4: Aprire le porte di rete.

È possibile che l'hardware di rete o il firewall blocchi la comunicazione con i server Xbox LIVE.

Se si è connessi a una rete dal luogo di lavoro o dall'università, chiedere all'amministratore di rete di aprire le porte elencate di seguito:

  • Porta 88 (UDP)
  • Porta 3074 (UDP e TCP)
  • Porta 53 (UDP e TCP)
  • Porta 80 (TCP)

Se non si è connessi a una rete dal luogo di lavoro o dall'università, è necessario aprire le porte di rete sull'hardware di rete. È possibile aprire le porte di rete in due modi.

Metodo 1: port triggering
Metodo 2: port forwarding

La tecnica detta port triggering rappresenta il modo più rapido per aprire le porte di rete, ma non è supportata da tutti i router. Verificare la documentazione del router per sapere se il port triggering è supportato. In caso contrario, provare il Metodo 2: port forwarding.

Metodo 1: port triggering

Quando si imposta il port triggering sul router, è necessario attivare le porte seguenti:

  • Porta 3074 (UDP e TCP)
  • Porta 88 (UDP)

In genere, per ogni porta che richiede l'attivazione del port triggering è necessario impostare quanto segue:

"Nome applicazione": Xbox LIVE "trigger port" su UDP 88 e "forwarded port" su UDP 88

Per informazioni su come impostare il port triggering, vedere la documentazione del router.

Metodo 2: port forwarding PARTE 1: PASSARE ALLE IMPOSTAZIONI IP MANUALI

Prima di aprire le porte sull'hardware di rete, è necessario recuperare le impostazioni IP del computer e utilizzarle sulla console Xbox 360.

    1. Sul computer fare clic sul pulsante Start, digitare cmd, quindi premere INVIO. 

      Nota: se si utilizza Windows XP, fare clic sul pulsante Start, scegliere Esegui, digitare cmd, quindi premere INVIO.
  1. Al prompt digitare ipconfig, quindi premere INVIO.
  2. Individuare la connessione di rete, ad esempio cercare una voce denominata "Connessione alla rete locale (LAN)" o "Connessione rete senza fili". Prendere nota dei numeri dei seguenti elementi:
    • Indirizzo IPv4 (o indirizzo IP)
    • Subnet mask
    • Gateway predefinito
  3. Creare un indirizzo IP univoco per la console Xbox 360 e prenderne nota. A questo scopo, un modo semplice consiste nell'aggiungere 10 all'indirizzo del gateway predefinito del computer in uso. Se l'indirizzo del computer è ad esempio 192.168.1.1, utilizzare 192.168.1.11 come indirizzo IP univoco per la console.
Nota: tutti gli indirizzi IP della rete devono essere univoci.

A questo punto è possibile configurare le impostazioni IP della console.

  1. Sulla console andare a Xbox personale, quindi selezionare Impostazioni di sistema.
  2. Selezionare Impostazioni di rete.
  3. Se viene richiesto, selezionare Rete cablata o il nome della rete wireless.
  4. Selezionare Configura rete.
  5. Nella scheda Impostazioni di base selezionare Impostazioni IP.
  6. Selezionare Manuale.
  7. Selezionare Indirizzo IP.
  8. Immettere l'indirizzo IP univoco creato in precedenza, quindi selezionare Fatto.
  9. Selezionare Subnet mask, immettere la subnet mask del computer, quindi selezionare Fatto.
  10. Selezionare Gateway, immettere il gateway predefinito del computer, quindi selezionare Fatto.
  11. Selezionare Fatto.
  12. Premere B sul controller, quindi verificare di nuovo la connessione.
PARTE 2: APRIRE LE PORTE SUL ROUTER O SUL GATEWAY

Aprire quindi le porte sul router o sul gateway in modo da consentire la comunicazione dell'hardware di rete con i server Xbox LIVE.

    1. Avviare il Web browser.
    2. Nella barra degli indirizzi digitare l'indirizzo IP predefinito del router o del gateway, quindi premere INVIO.

  2. Dopo aver effettuato l'accesso, verrà visualizzata la pagina di configurazione del router o del gateway.

This seems like there is some unknown problem with the windows media player. That is why the troubleshooter is trying to fix it every now and then.

Method 1:  I would suggest you to please disable and enable the windows media player on your computer.

You cannot delete Windows media player 12 from the computer as it is default in Windows 7 but it can be enabled or disabled using the Windows features.

Enabling Windows Media Player on Windows 7:
a) Go to Start and in the search type "Turn Windows features On or Off".
b) Click on "Turn Windows features On or Off".
c) Browse to the Media Features and place a check mark in front of Windows Media Player.
d) Restart the Computer.
