steve winograd say:

Some computer makers, including Dell and HP, put a recovery partition on the hard disk of new computers.  By pressing a special key combination when the computer starts, you can use the recovery partition to restore the computer to the way it was when it was new.

If that's what your computer has, there's nothing wrong, and you don't have to do anything.  Windows has assigned a drive letter to the recovery partition, but nothing in the system will access it, and it doesn't matter that it's almost full.

If your D: drive is a recovery partition, I recommend that you remove the drive letter from it:


1. Click the Start button.
2. Type diskmgmt.msc in the Search box and pressEnter .
3. In the Disk Management window, right-click the recovery partition.
4. Click Change Drive Letter and Paths .
5. Click Remove .

hairyfool say:

Although targeted to transfer your user profile from an old computer to a new one there is no reason why Easy Transfer can't backup the profile to be restored to the same PC after a rebuild. This would pick up your user settings, desktop backgrounds and shortcuts, files etc. into a single backup, it does not backup the actual applications.

 

Once rebuilt you would then rerun the Easy Transfer utility after re-installing the applications to restore your files as the "new" computer.

 

Easy Transfer is part of Windows. All Programs -> Accessories -> System Tools -> Windows Easy Transfer.

 

If you Google "driver backup" you will find a number of free utilities to backup the current drivers although personally I would use a rebuild to make sure all the drivers were current and up to date.

 

Even when you use Easy Transfer it does not include the Outlook account profiles used to connect to mail hosts. If you use Outlook export the following key from the old PC and merge it on to the new one

 

HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows Messaging Subsystem\Profiles\Outlook

 

Being of the nervous type I very rarely actually wipe a hard disk to do a rebuild but use the process to install a new/bigger/faster drive so I always have the old one to fall back to if anything goes wrong. If you did decide to "upgrade" to Windows 8 you would need the Easy Transfer from Windows 8 to export the profile from Windows 7.
