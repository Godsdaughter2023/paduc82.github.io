Hi WIly, below is the advice from Lenovo Support:

Press Windows + X
Select Device Manager

Find the Touch screen driver under Mice and Other Pointing Devices - USB Touchscreen Controller(A111)

You're going to uninstall this and check the box that says "Delete the driver software for this device". Restart your computer.

If the feature is still not back, open Device Manager -> Human Interface Devices. Right-click HID compliant touch screen, then uninstall. When you restart the PC, it will reinstall.

Right-click on it then select enable. That may bring the touchscreen back.
