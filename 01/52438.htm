If you have an SSD, the problem may be an old storage driver.

Here's how you can manually update your storage controller driver:

  1. Go to Control Panel -> Device Manager
  2. Expand Storage controller
  3. Verify if your storage controller is compatible. C600+ series chipsets should be compatible.
  4. Access https://downloadcenter.intel.com/download/25358/SAS-Hardware-RAID-Driver-for-Windows-x86-x64-version-
  5. Extract the downloaded file to a known location
  6. Right click  Intel(R) C600 Series Chipset SAS RAID (SATA Mode)->Update Driver
  7. Click Browse my computer for driver software
  8. Find the extracted folder
  9. Then click Let me pick from a list of device drivers on my computer
  10. Select the newest driver and install
  11. Repeat the steps from 6 to 10 for Intel(R) C600+/C220+ series chipset SATA RAID Controller

Hope this works for you!

"What can I do if there is not enough memory on the system to install updates or install more apps?"

"Applications can only be installed on the internal main memory of the device. Therefore we recommend to use this storage mainly for the applications and to use an external SD card to store personal files such as pictures or movies.

If the internal memory is full, you can go in the Settings / Storage and check what can eventually be deleted to free up some space. Applications can store a lot of data in the cache so you may have to clear the applications cache too. This can be done by clicking on the line called Cached data. Ultimately, you might need to uninstall some applications.
Updates can be installed from an external SD card if needed. If possible it is safer to do it using the internal memory. If you need to use the SD card, make sure that the SD card will not be removed during the update process as it may corrupt the system."

hugo_v say: Hello Josh,

Apps you install from the Windows Store don’t slow down your Surface, so you don’t need to close them. When you’re not using an app, Windows will leave it running in the background and then close it eventually if you don’t use it. If you want to remove an app from view, drag the app to the bottom of the screen. To stop all processes associated with an app, drag the app to the bottom of the screen, and hold it there until the app flips over.


Note It’s still a good idea to close desktop apps when you're done using them, particularly before shutting down your Surface.

For more info, see Start and find apps on Surface.
