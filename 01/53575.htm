tryx3 say:

Daniel,

Your symptoms indicate clearly & unambiguously that your Admin-level account has suffered from user profile corruption.

What ought to happen in these conditions is that you can boot into Safe mode and your login screen will show a user account that does not normally appear -  a user account with the username Administrator.   Administrator is not an account that you have created, it is a Built-In Admin account that exists for precisely this situation.

So, in brief, you need to

  1. Boot into Safe mode,
  2. Log into Administrator [it has no password so you just select that account and press the Return key],
  3. Use Administrator to create a new password-protected Admin-level user account [actually I always have two of them to ensure that I will be okay even if one of them does get afflicted with user profile corruption]
  4. Log out of Administrator & log into the new Admin-level accounts you have created just to check that they work correctly.
  5. Reboot [and note afterwards that there is no longer any sign of Administrator - it gets disabled again automatically when the fault condition that enabled it no longer applies].

Do you need to be given details of each step?  I recently helped another user in this situation so I will dig out a link to that thread and post it here for you in a couple of minutes.
