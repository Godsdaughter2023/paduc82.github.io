How to restore bookmarks using windows.old file

rajithr say:

Hi,

You can just copy the bookmark to the Firefox folder located in a folder called Profiles.

To do this, follow these steps:

  1. Click the Windows Start button. The Start menu will open.
  2. In the Search box at the bottom of the Start menu, type:
    %APPDATA%\Mozilla\Firefox\Profiles\
    without pressing Enter. A list of profiles will appear at the top of the Start menu.
  3. Click on the profile with “default” in the name to open it in a window.

Hope this helps. If you have further queries, we will be happy to help.
