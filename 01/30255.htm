internet expert say:

The Xbox One S uses a built-in power supply, so there is no external power brick.


All Xbox One S consoles are (internally) the same worldwide, and they will handle the voltages of all countries they are sold in.


The Xbox One S uses the standardized IEC-60320 "C7" appliance coupler to connect to power. Basically, this boils down to just finding a C7 cord with the appropriate mains connector for your country.


You can contact Microsoft (via chat or phone call) and request a power cord for your Xbox One S... they might just send you one. If that doesn't work, you can just order them elsewhere online; these appliance coupler cords are very commonplace, cheap, and used by many other devices. 10.  Accept the Xbox LIVE Terms of Use and the Privacy Statement.

Desktop background themes - Microsoft Community

richard tlh say:

Desktop background themes

The desktop background themes are a great addition with Windows 7. Is it possible to create a custom theme combining 30 to 40 favorite images from

existing downloaded themes?

sandeep dsouza say:

Hi,

Yes, we can create our own themes; to do that we need to Create new folder and we need to copy all the different desktop background images in that folder (you may copy the images of your own and please make sure that which location you have saved that images).

Please follow the steps to copy Different in built desktop background images:

1.    Click on Start button.

2.    Click on Computer

3.    Double click on C:\ drive(where your Operating System is installed)

4.    Search folder Windows and open it.

5.    Now in Folder Windows Search folder named Web and open it, now open File named Wallpaper now you can see number of folder open the each folder and it contains picture.

6.    Copy whichever images you like in to the newly created folder and save it your own location.

Now follow the steps given below to create it:

1.    Click on Start button.

2.    In search box type Personalization and hit enter.

3.    Now click on Desktop Background link, you will get Choose your desktop background screen.

4.    Now click on Browse and select that folder which we just now created and saved.

5.    Click Ok and click on Save Changes

6.    Automatically theme background will get changes.

If you want to change the Windows Background Color as well as Screen saver to newly created desktop background you can do that using these methods:

Method 1:  To change the Widows Background Color

a. Click on Start button.

b. In search box type Personalization.

c.    Now click on Windows Color link, you will get Change the color of your window borders, start menu and task bar screen.

d.    Now select the current colors of your own, click on Save Changes.

Method 2: To change the Screen saver

a.    Click on Start button.

b.    In search box type Personalization.

c.    Now click on Screen Saver link, you will get Screen Saver setting screen.

d.    Now select the different screen saver of your own, click on Apply and Ok.

After doing all these steps you can create your own Theme and you can save this theme if you want to.

 Hope this information is helpful!!

techfreak_ say:

You can easily determine what is going on by checking your actual minute consumption found in your usage history.  Sometimes people find out someone else has access to their account making unauthorized calls.  Other times they are either placing calls outside their subscription or their prior monthly cycle has not completed.  If there is any discrepancy you should see it. 

Be aware that any subscription covers the whole month.  If you purchase additional monthly subscriptions of the same amount in the same ~30 day period, no additional minutes will be awarded until the prior subscription's ~30 days is up.  Purchasing a different minute subscription is the only way that you could get additional minutes in the same month.  Any other subscriptions will have their own ~30 day cycle. 

If you are unable to determine what status your subscription is in or how everything was deducted, then you would need to contact Skype Customer Service.

Simply visit the support request page and choose the following options to reach live text chat support.

Payments & Orders - Help with a subscription then click Next.
