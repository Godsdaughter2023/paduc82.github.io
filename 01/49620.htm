geordie flash say:

Hi there,

 

First of all, you are not exactly copying the game to a USB stick, you are just installing it to a USB stick instead of the Hard Drive.
 

  1. Find a USB stick with at least 8GB of memory on, it's recommended you use 16GB for best results.
  2. Insert the Game disc (not the install disc) of GTA V
  3. Go the the game disc on your dashboard as if you were going to play it and press X.
  4. Choose install and select the USB stick as the memory device.
  5. Play the game and see the benefits.

     

Hopefully this makes sense.
