steve cochran say:

HKCU is per user.  So if you change it for the Administrator, it won't change it for any other user.  You have to change it for each user.  Or add the entry under HKLM

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Explorer

create the NoDriveTypeAutoRun DWORD value and give it a Hex value of FF.

and that should apply to the entire machine.
