a. user say:

what can I do to unnistall or remove a program without accessing "control panel" ?

Hi Eghy:

You can run its uninstaller from its directory inC:\Program Files
or C:\Program Files (x86). Many programs have an uninstaller exe.
I mean it’s NOT exactly Windows® that removes the program from the
machine when you remove it from the Control Panel.

In the Windows® Registry under the following registry entry path:
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall.  

Lets give you an example:
If you have Mozilla® Firefox® installed then you will find it under that
registry entry (the uninstall path), some programs like Microsoft® Office
use a Class Identifier (CLSID), here is an example of such:

{22B775E7-6C42-4FC5-8E10-9A5E3257BD94}

While Mozilla® Firefox® doesn’t use such. Anyway, if you go to that
registry entry and you find the Mozilla® entry, select it and on the
right-side you’ll see a list of values:

Example:
The String value “DisplayIcon” represents the icon of the program, that
icon you’ll find in the list of program in the Programs and Features in
your Control Panel.

Now each program has a Standard Value which is called “UninstallString”,
the Program and Features in the Control Panel will check for such string
value when it is executed by the user (when you open the Program and
Features).

The “UninstallString” represents the exe file made by the manufacturer of
that program, the value in that string is then executed by the context-menu
in Programs and Features, so when you right-click and choose “Uninstall”
the Program and Features will execute the manufacturers exe.

The Mozilla® Firefox® uninstall string holds the following value:
C:\Program Files\Mozilla Firefox\uninstall\helper.exe.

I can keep on going and I can explain the entire connection and
how the Program and Features in the Windows® Control Panel
really works.

*It is recommend to use the Programs and Features from

the Control Panel. I would like to give you a suggestion, I suggest
you make a shortcut of the Programs and Features to your
Desktop, thereby you do NOT need to open Windows Control
Panel, just open the shortcut.

I hope you find this information useful and if you need any further assistance,
please feel free to contact me and let me know.

I hope this information was helpful…

The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.
