allan mej say:

Try doing a startup repair. A startup repair could possibly resolve PC booting issues by repairing corrupted files that are essential for your PC to boot. To do a startup repair:

  1. Upon startup, when the Windows logo appears, press and hold the Power button to force shutdown your PC. Do this for 3 times.
  2. Windows Recovery Environment will appear, click Troubleshoot.
  3. Go to Advanced options>Advanced options.
  4. Click Startup Repair and let your PC run the necessary repairs.

If that didn't resolve your issue, try using an installation media to fix your startup issues. To create an installation media, kindly follow the steps on this link under Using the tool to create installation media. After creating an installation media, set up your BIOS to boot from the USB drive. After doing so, proceed on doing these steps:

  1. Upon booting into set up, go to Repair Computer>Troubleshoot>Advanced Options>Command Prompt.
  2. Type BOOTREC /FIXMBR and press Enter. You should see The operation completed successfully if all goes well.
  3. Type BOOTREC /FIXBOOT and press Enter.
    Note: If you see Boot Manager is Missing, type BOOTREC /RebuildBcd and press Enter.
  4. If all goes well, reboot your PC.

Let us know how things go.

If you think this was useful, you may "mark it as an answer" to help those who are facing the same problem.
