xb1r athena say:

Wireless Security

Login Info

 

  1. Gateway Address: "192.168.2.1" or try "172.16.0.1" or try "http://gateway.2wire.net/management"***
  2. Username: Try "admin" otherwise there is no default username.
  3. Password: Try "admin" or try “2wire” otherwise there is no default password.

 

***Note: The Management and Diagnostic Console (MDC) settings are dependent on the 2Wire gateway software release. The MDC are for 2Wire gateways running software release 4.21.x If your gateway is running a software release earlier than 4.21.x, some of these settings may not be available.

 

 

How To Find Wireless Passphrase (WPA/WEP)

 

On the Firmware:

  1. Connect a computer to your gateway with an Ethernet cable.
  2. Open a web browser.
  3. Type the gateway address: "192.168.2.1" into the web address.
  4. When prompted for a password, type in "admin" for both the username and password. If not, check the label on the gateway or ISP documentation for a username and password if you are prompted for a login.
  5. Once logged into the gateway, click on "Home Networks" on the top.
  6. Click on the "Advance" subcategory.
  7. Under the "wireless security" section you will see 2 options. One for "Use default encryption key" or "Use custom encryption key"
  8. If the "default" is selected, check for a label on the 2wire gateway for a 10 digit number.
  9. If the "custom" is selected, then it should show the current password.


 

On the Hardware:


*If you're using the default wireless password, you can try the following.*

 

  1. Check for a label on the 2wire gateway.
  2. Look for a 10 digit number on the label; it might be labeled as "WEP key" or "default encryption key".

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Connect a computer to your gateway with an Ethernet cable.
  2. Open a web browser.
  3. Type the gateway address: "192.168.2.1" into the web address.
  4. When prompted for a password, type in "admin" for both the username and password. If not, check the label on the gateway or ISP documentation for a username and password if you are prompted for a login.
  5. Once logged into the gateway, click on "Home Networks" on the top.
  6. Click on the "Advance" subcategory.
  7. On the main page, look for the "Wireless security". There will be a drop down list for "Authentication". From there you can choose your wireless security type.
  8. Also you can disable the wireless security.
  9. Click “Save” at the bottom of the page.
