Iomega Zip Drive not working with Windows 10 - a possible solution. - Microsoft Community

simon(s) say:

Iomega Zip Drive not working with Windows 10 - a possible solution.

I had a similar problem that others have had with Iomega Zip drives not working with Windows 10 and threads to user issues are locked so I thought I would add a solution that worked for me here.

My device is an Iomega USB Zip 250 with its own power supply.

It was working on an XP desktop but not on a Windows 10 laptop.

The device would appear on 'control panel - devices and printers' and in 'device manager' as a disk drive.

The device properties indicated that it was working properly.

However, in 'file explorer' it would either not exist or be inaccessible.

This is what worked for me:-

Open 'Device Manager' (Cortana search).

Expand 'Disk drives' (click the > symbol).

Open properties for the Iomega device (double click).

Click the 'Policies' tab.

Under 'Removal policy' check 'Better performance'.

Under 'Write-caching policy' check 'Enable write caching'.

Under 'Write-caching policy' check 'Turn off Windows write-cache buffer'.

Click the 'OK' button.

After the above steps the Zip drive appeared in file explorer as a normal USB storage device.

My thoughts are that the Zip drive itself was powering down to an inaccessible hibernation mode after initialisation but with caching switched on the zip disk and its files become available in this mode.

We come across this issue when you have files in your print queue or if the hidden drivers are installed in the system. To resolve this issue, let us follow these methods and check if that helps.

Steps to remove Printer.

Method 1: Stop the service and delete files.

Step 1: Stop the Printer Service.

  1. Press Windows & R key at the same time on the computer.

  2. Type: services.msc.

  3. Search for Printer Spooler service, right click on it and select Stop.

Step 2: Delete the files.

  1. Navigate to: C:\Windows\System32\Spool\Printers folder.
  2. Select all files and press delete key to delete all files. This will clear the print queue. Click Yes button if you see the confirmation dialog.
  3. Now go to control panel, select the printer and remove it.

Method 2: Remove Printer through registry.

Note: Serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. To backup and restore registry key, refer Microsoft article on How to back up and restore the registry in Windows.

  1. Press Windows & R key at the same time on the computer.
  2. Type: regedit
  3. Navigate to:

HKLM\System\CurrentControlSet\Control\Print\Environments\Windows x64\Drivers\version-3

HKLM\System\CurrentControlSet\Control\Print\Environments\Windows x64\Drivers\version-4

HKLM\System\CurrentControlSet\Control\Print\Printers

4. Delete all the entries of the printer which you don’t need from the above location.

Steps to remove Bluetooth device.

Method 1: Remove Bluetooth devices through device manager.

  1. Disconnect the system from the internet access and disconnect all the Bluetooth devices. 

  2. Press Windows & R key at the same time on the computer.

  3. Type: devmgmt.msc

  4. Click on View and select Hidden Devices.

  5.  Expand the Bluetooth drivers. Right click on the Bluetooth device and click Uninstall.

  6. Now, try to remove the Bluetooth devices from the setting page and check for the results.

  7. Restart the system and connect the internet.

Method 2: Remove Bluetooth devices through registry.

Note: Serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. To backup and restore registry key, refer Microsoft article on How to back up and restore the registry in Windows.

  1. Press Windows & R keyat the same time on the computer.
  2. Type: regedit
  3. Navigate to: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\BTHPORT\Parameters\Devices
  4. Under this, you have one to several 12-digit device keys which represent the devices, like: “00024c02b458” (which is a Bluetooth headset).
  5. Right click and delete the device key which is not required. Now, reboot your computer, after this the paired device will be gone. You can find the correct device by double-clicking the “Name” value associated with each device, which shows the (binary!) device name in plain text.
