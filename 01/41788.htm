steve winograd say: An icon that you add to your desktop will only appear on your desktop.  When you install a program, look for an option to install it for the current user or for all users.

ramesh srinivasan say: The Desktop is comprised of files/icons from two different locations:

Public Desktop: Icons present in this folder will be displayed for every user profile. The Public Desktop folder is located here: %PUBLIC%\Desktop (Type the path in the Explorer Address bar or Run command.)

User Desktop: Icons present in this folder will be shown for that particular user account. The desktop folder for the currently logged on user is located at %USERPROFILE%\Desktop, which is usually C:\Users\{Username}\Desktop

You may simply move the specific desktop icons from Location #1 to Location #2, if you don't want the icon to be shown for all users.

joseph ban say:

Hi Debo,

You can copy your files from the computer to your USB flash drive or other storage devices through the File Explorer. To sort out your concern, we suggest that you follow the steps below:

To copy the files from your PC to USB flash drive

  1. Click Start, type File Explorer, and then press Enter.
  2. Insert the external storage device to your PC.
  3. On the File Explorer window, select the files that you want to copy.
  4. Right-click on the selected files and choose Copy.
  5. Click This PC on the left-side pane of the File Explorer window, open your device under the Devices and drives section.
  6. Press Ctrl + V key on your keyboard to paste the documents.

To copy the files from the USB flash drive to your computer

  1. Select the files from your external device.
  2. Press Ctrl + C key on your keyboard to copy the files.
  3. Select which location you want to paste the file from the left-side pane of the File Explorer window.
  4. You may choose a location such as Desktop, Documents, or This PC.
  5. Once you’ve selected the location, press Ctrl + V key to paste the files.

Let us know if you need additional assistance.
