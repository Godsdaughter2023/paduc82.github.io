Do you have an Ethernet cable connected to this computer?

Windows 7 has a quirk when both cable and wireless are connected.  It will show the wireless connection in the system tray as connected, and may actually use the Ethernet connection instead.  If Win7 sees both connections, it will prefer the wireless one.

If you are not connected by Ethernet cable, then try it and see if you get a connection.  If you do, then you can be confident there is nothing wrong with the modem, but the problem is a wireless one. 

If it is a wireless problem only:

  1. You have the wrong network password entered
  2. The driver in your computer for the wireless facility needs to be updated or just re-installed
  3. The wireless part of your router is not functioning correctly

Write back here with the answers and if you need instructions on how to any one of these things.
