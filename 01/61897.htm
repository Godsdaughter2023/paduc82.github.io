Hi,

BootSmssInitTime is the time it takes the Session Manager Subsystem (smss.exe) to run at startup. This process starts the system thread and other vital tasks along with winlogon.exe, csrss.exe and setting system variables for the user session. If there is an error with either of the two executable files, the smss.exe process will cause the computer to hang and not respond.
