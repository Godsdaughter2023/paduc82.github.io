Step 1: Try to enable built-in windows administrator account.

3. Type the following command to enable built-in admin account:

 

net user administrator /active: yes

Note: Please follow the same steps and type net user administrator /active: no to disable admin account after the troubleshooting is done.

Step 2: New Admin User Account

Please follow the below mentioned steps to create a New User Account:

1. Open Control Panel by searching in the search box, clicking User Accounts, and then clicking User Accounts.

2. Click Manage another account. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

3. Click Create a new account.

4. Type the name you want to give the user account, click an account type, and then click Create Account.

5. Restart the PC.

Hi Chris,

Since the system file checker tool didn't find any error when you ran the scan, we recommend that you manually delete the icon cache by following these steps:

  1. Click Start, type Command Prompt, right-click Command Prompt on the search results and select Run as Administrator

  2. Type the following command to navigate to the icon cache database location and press Enter:
    cd %homepath%\AppData\Local\Microsoft\Windows\Explorer

  3. Type the following command to verify the icon cache database files are at the location and press Enter:
    dir iconcache*

    Note:     The output of the dir command should list these database files:
    concache_1280.db
    iconcache_16.db
    iconcache_1920.db
    iconcache_256.db
    iconcache_2560.db
    iconcache_32.db
    iconcache_48.db
    iconcache_768.db
    iconcache_96.db
    iconcache_custom_stream.db
    iconcache_exif.db
    iconcache_idx.db
    iconcache_sr.db
    iconcache_wide.db
    iconcache_wide_alternate.db

  4. Type the following command to stop File Explorer and press Enter: taskkill /f /im explorer.exe

    Important:     Once you terminate File Explorer, your desktop background and taskbar will disappear, leaving a black screen — but it's OK, it's only temporarily. You need to close File Explorer to be able to delete the icon cache files. Otherwise, you won't be able to do it

    Note:     If you get access denied trying to remove the files, you're probably not running Command Prompt with as an administrator. Repeat these steps, but making sure Command Prompt starts with administrative privileges

    dir iconcache*

    Note: If you still see some iconcache files, then it's likely that some apps are still running, which will prevent you from deleting the database files. Make sure to close all open programs on the taskbar and repeat the steps again.

  7. Type the following command to start File Explorer and press Enter: explorer.exe

  8. Restart your computer and check if the issue persists
