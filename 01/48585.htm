doubleclick explorer right pane always opens a new window instead of - Microsoft Community

setsunaaa say:

doubleclick explorer right pane always opens a new window instead of in existing window + possible fix !

The annoyance:
In the right folder pane, if you doubleclick a folder it opens in a new window instead within that window.
It appeared after installing IE8 and / or playing a bit around with my startmenu options, but I cannot tell whether they are really the reason.
All those many hints about "folder options" fail miserably in my case.

Here is the real fix for my case:
open regedit.
Navigate to:
[HKEY_CLASSES_ROOT\Folder\shell\explore\command]
The default value was set to:
%SystemRoot%\Explorer.exe /separate,/e,/idlist,%I,%L

I changed it to:
%SystemRoot%\Explorer.exe /e,/idlist,%I,%L

and then explorer.exe behaved as expected.

Don't mix it up with the Folder option "always open in a separate process", that somewhere else in the registry.

What makes it even weirder, in some cases you can change it back to:
%SystemRoot%\Explorer.exe /separate,/e,/idlist,%I,%L

and the behavior stays the way it should.

mersonary say:

Recent Windows 10 update broke usb mic

Recently downloaded the newest version of Windows 10 and now my snowball blue doesn't pick up any sound input. I've gone through all the fix guides on these forums and others, but still no luck. Wondering if anyone has had the same issue from the most recent windows patch and what they did to fix it.
weswoodell say:

Found a fix ...

Click to open the Windows menu in the bottom left, click the gearbox to open Windows settings

Settings > Privacy > Microphone - make sure your microphone is turned on in the privacy menu giving Windows access.

The mic used to work in applications with that setting turned off, but now it must be turned on for my USB mic to work in games or other applications, even if they aren't exclusively "Windows" applications (like apps you'd get from the Windows Store).

Hope this helps
