Skype for Windows 10's interface is different from classic Skype. For information on how to navigate Skype for Windows 10, please visit this support article: Finding your way around Skype for Windows 10 

Adding phone number as contact in Skype for Windows 10 works differently. To do this, please follow the below steps:

  1. Dial the number and call it. (There's no need for the call to go through)
  2. After dialing the number, the number will appear on your recent chats list.
  3. Right-click the Phone number.
  4. Choose View Profile.
  5. Under it, click Save to local address book.
  6. Then make the changes you want.
