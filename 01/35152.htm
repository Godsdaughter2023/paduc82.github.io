a. user say:

Hi Karl,

I suggest you to follow the steps below and then check if it helps in resolving your issue.

Method 1: Update Graphics Card

Follow the steps below.

  1. Press Windows key + X and click on Device Manager.
  2. Expand the Display Adapter.
  3. Right-click on the Graphics card and select Update Driver Software.
  4. Restart your system

Method 2: Uninstall and then re-install Graphics Card 

Follow the steps below.

  1. Press Windows key + X and click on Device Manager.
  2. Expand the Display Adapter.
  3. Right-click on the Graphics card and select Uninstall.
  4. Once, you have uninstalled the driver do not restart, go to the Manufacturer's website and download the latest Graphics Card Driver.
  5. Double-click (run) and install the driver setup file.
  6. Restart your system.

Hope this helps. Please respond if you have further related queries.
