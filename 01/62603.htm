a. user say:

First things first: You can't backup an entire user profile. You can (and should) backup users' data folders, and Microsoft supports that. I'm referring to each user's Contacts, Desktop, Documents, Downloads, Music, Pictures and Videos folders. Also the Favorites folder, if you're still using Internet Explorer. The other folders in a user profile are part of the operating system.

If you want to reinstall Windows from scratch, the most effective way to do that is with a clean install (which Microsoft persists in calling a custom install.) The easiest way to do a clean install is with a reset. After a half hour or so of grinding noises and smoke you've got a squeaky clean fresh copy of Windows 10. Now you can set up your kids' user accounts and restore their data to the corresponding folders.

Each user account in a computer needs to be unique. That means your kid's account on the old copy of Windows is separate from their account on the new copy of Windows. Even though they belong to the same person. That's why you get warnings about file permissions. But it's no problem. Sign in to each child's account and double-click on a previously backed up folder. You'll be stopped and asked if you want to take ownership of the folder - just say yes. I think you might need to temporarily promote the child's account to Administrator in order to do this. Temporarily, that is.

You can use Acronis for the backup but in my opinion it would be faster to use file-and-folder backup software, or a compression program, or just plain old copy and paste.

As long as you're thinking of a clean install, and you know your way around making images, mind if I make an unsolicited suggestion? After Windows is reinstalled, partition your disk so you have a separate partition just for data, and move the kids' data folders to the new data partition. With three kids on the same computer, who knows, you just might need to reinstall Windows again in the future, maybe more than once, and with a separate data partition you won't have to backup their data every time.

As long as you're good with making images: After Windows is reinstalled and you've run Windows Update to completion, make an image of the system partition. Because it's faster to restore an image than to do a reinstall every time.

Hug those kids because they'll be off to college before you know it.
