So what do you do:

  •

    2. Highlight Microsoft Silverlight, then select remove.
    3. if needed, type the following in a command prompt:

    reg delete HKLMSoftwareMicrosoftSilverlight /f

  •  
    1. Then delete the “Microsoft Silverlight” directory under program files and then try to install Silverlight.

Follow the instructions below to start the computer in Clean boot state.

 

  1. Click Start, type msconfig in the Start Search box, and then press ENTER.

If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

  1. On the General tab, click Selective Startup.
  2. Under Selective Startup, click to clear the Load Startup Items check box.
  3. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.
  4. Click OK.
  5. When you are prompted, click Restart.
  6. After the computer starts, start the download and check whether the problem is resolved.
