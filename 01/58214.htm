By itself, the number of running processes has no meaning. Many or most could be occupying virtually no resources.

What matters is: What are those processes doing, and in particular, which process or processes are using so much of your computer's resources that your computer is actually slower because of them? Next you would have to decide whether those processes are performing legitimate activities.

These are not easy questions to solve. It requires a detailed understanding of how your computer works, if you want to be sure of doing the right thing.

In my opinion, you should avoid the rush to judgement found in YouTube videos and elsewhere from self-styled 'experts' who don't really understand what they're looking at. That said, there's one widely used technique that can help you.

Restart your computer in Safe Mode, which runs a bare-bones version of Windows and nothing else. If your computer is still slow, the reason is something having to do with Windows. If your computer is faster in Safe Mode, your slowness is being caused by something else that starts together with Windows.

Restart your computer normally and open Task Manager [CTRL+SHIFT+ESC] then open the Processes tab. Sort your processes by CPU usage and also by RAM usage. That will tell you what's occupying most of your computer;s resources. Do you recognize those processes, and do you need to have them running?

When I upgrade a preinstalled (OEM) or retail version of Windows 7 or Windows 8/8.1 license to Windows 10, does that license remain OEM or become a retail license?

If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.

If you upgrade from a retail version, it carries the rights of a retail version.

If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

What happens if I change my motherboard?

As it pertains to the OEM licenses this will invalidate the Windows 10 upgrade license because it will no longer have a previous base qualifying license which is required for the free upgrade. You will then have to purchase a full retail Windows 10 license. If the base qualifying license (Windows 7 or Windows 8.1) was a full retail version, then yes, you can transfer it.

15. UPGRADES. To use upgrade software, you must first be licensed for the software that is eligible for the upgrade. Upon upgrade, this agreement takes the place of the agreement for the software you upgraded from. After you upgrade, you may no longer use the software you upgraded from.

17. TRANSFER TO ANOTHER COMPUTER. a. Software Other than Windows Anytime Upgrade. You may transfer the software and install it on another computer for your use. That computer becomes the licensed computer. You may not do so to share this license between computers.

This was fixed by an "undelete" operation with a Partition Management application.  Somehow, the active partition, which was also the only partition, had been deleted or some equivalent.  When Windows viewed the disk there was no file structure.  To Windows it was "as if" the disk had never been formatted, which was why it appeared as uninitialized.  Fortunately there are applications designed to fix accidental deletion of partitions, and they are cheap, when purchased online.  I bought a Windows 7 version that was a few years old and it worked perfectly.  There was no need to do a deep search to find all of the partitions that had ever been created on the disk (I did this with the trial version and it took over 36 hours!).  The fast version found the newly deleted partition in moments, and undeleted it similarly.

What is NOT required is Data Recovery.  This is what the hard disk manufacturer broadly recommended, and is probably an industry catch phrase that includes Partition Recovery.  With a Data or File Recovery application, the file structure is lost and depending upon the application, what is generated can be anything from a nice collection of file folders similar to the original file structure, or a complete jumble of files not in folders at all.  So if you have Terabytes of information as tens of thousands of files in dozens or hundreds of file folders, it could take days to reassemble the original folder structure.  Doing a whole disk Data Recovery can also take multiple hours on a big disk, since its looking for absolutely everything even remotely like a file.  Also you could need another disk to transfer all of the found files onto.

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.
