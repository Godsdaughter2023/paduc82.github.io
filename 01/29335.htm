So the other answers cover allowing Home console sharing, which will allow him to play all the games owned by the account designated as "Home" on that console, while that account can still play all those games on another console.

If you want to move games from one console to another, there is a network transfer feature in Settings > Network > Network Transfer, as well as options to simply copy the games to an external drive and then move that drive to the new console.  

But I think the question you have is if you can share save games?  The files are all account specific, and you cannot share, trade, or transfer those to another player.  They will be able to play the games, but they will only be able to access the save data associated with their account, and there is no current solution to allow otherwise.

Please let us know if you have any other questions and we'll be happy to help answer!

Edit: Also, want to clarify that you cannot transfer titles between accounts either, but simply share them via the Home console sharing features.  Transfers to other accounts are not possible currently.

How to Find SSID (Network Name) - Microsoft Community

xb1r athena say:

How to Find SSID (Network Name)

Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - admin

 How to Find SSID (Network Name)

  1. Open a web browser.
  2. Enter "192.168.0.1" in the address bar.
  3. If prompted for a username/password, type in the username and password.
  4. Click on the "log in" button.
  5. Click on the "Wireless" category located on the top of the page.
  6. Click on the "Basic Settings" subcategory located just below the main category.
  7. Your Network name (SSID) will be located on the bottom of the main page.
