Hello Kavi,

Thank you for posting your query in Microsoft Community.

a. Do you get the error message, "The system could not find the phone book entry for this connection"?

b. Did you make ay changes to the computer prior to this issue?

If you get "error 623: The system could not find the phone book entry for this connection", then this error arises if the connection is configured to use a dial-up connection that has been deleted.

To work around this issue:

  1. Open Control Panel, and then click Network and Internet Connections.
  2. Click Network Connections.
  3. Right-click the modem connection, and then click Properties.
  4. Click to select the Dial another connection first check box, and then click the appropriate dial-up connection. If there are no dial-up connections to select, you may have to create a dial-up connection and repeat these steps.

Note: If your connection does not require a dial-up connection, click to clear the Dial another connection first check box, and then click OK.

If the issue still persists, then follow these steps, uninstall and re-install the Data card drivers in compatibility mode. Check if it helps.

Note: Download the drivers from the Manufacturer’s website and save it on to the Desktop.

a)  Press Windows Key + R, type devmgmt.msc.

b)  Expand the Universal Serial Bus Controller (for the Data Card) click on the drivers.

c)  Click on properties, click on uninstall tab.

d)  Click OK.

e)  Right-click on the driver (Downloaded from the manufacturer’s website) file.

f)  Click on Properties and select the Compatibility tab.

g)  Click the checkbox before Run this program in Compatibility mode for option and select Windows 7.

h)  In the list-down box; then try installing the device driver via Run as administrator.

debbie - say: Hi Tom_g35,

Thanks for posting to "Getting Ready for Windows 7" community forum.

Does the Belkin N card have a Windows 7 driver listed on the manufacturers website?  If it is not a Windows 7 driver you may try to reinstall it using the Troubleshoot Compatibility tool from Windows 7.  This usually will install in an XP SP2 mode if it has the option available.

Here is how to use the tool.

1.  Download the latest install package for the card into a folder on the desktop of the Windows 7 PC.
2.  Uninstall all instances of the driver from the system.  Start / Control Panel / Programs / Uninstall Program  - look for any software that may be applicable to the card (Be certain to have the card out of the system at this time.)
3.  After all software is uninstalled and the card is no longer viewed as being listed on the system - reboot the system.
4.  When back at the desktop, open the folder that contains the latest revision of the driver.  Locate the SETUP.EXE file or the file the manufacturer uses for setup and RIGHT Click the file. 
5.  Select TROUBLESHOOT COMPATIBILITY
6.  Windows will troubleshoot the file and check if it is compatibile with Windows 7.  If it does not find it is compatible and available it will offer a WINDOWS RECOMMENDED SETTING solution.  It will also offer a CUSTOMIZE YOUR SETTINGS solution to allow you to choose the install.  I suggest using WINDOWS RECOMMENDED SETTINGS solution.
7.  Follow the Wizard to complete the install of the device.

Let us know if this assists you in resolving this issue or if further assistance is needed.
