technitrevor say:

Static IP addresses are not as necessary anymore because of UPnP or Universal Plug and Play. UPnP is a set of networking protocols that permits networked devices, such as personal computers, printers, internet gateways, Wi-Fi access points and mobile devices to seamlessly discover each other's presence on the network and establish functional network services for data sharing, communications, and entertainment (source: en.wikipedia.org/wiki/Universal_Plug_and_play). However, devices must support UPnP, and almost all recent hardware does--like routers and printers. 

If you'd really like to setup a static IP, you can do so by configuring the router to assign the specific device a static IP. portforward.com is a great resource to reference in regards to setting up a static IP and opening ports to create a port forward. Basically, you can follow the steps below to find your router's IP address and access it's webpage to configure your router's settings:

1. Right click on the start icon, orb, or lower left corner and display the shortcut menu for administrator tasks.

2. click on *run*

3. In the run dialog box, enter "cmd" and press *enter*.

4. Now type "ipconfig" and press *enter*

5. This will run a quick command relating to how your computer is connected to the internet, if it is connected at all. It will provide you with information as to how your computer is connected to the internet, such as via ethernet or Wireless LAN adapter Wi-Fi and will provide you with your computer's IP address alongside your routers IP address. Your Router's IP address is listed as the "Default Gateway"

You can now open a web browser and enter the Default Gateway's IP address and you'll be viewing your routers settings via a website. You may need to know your admin username and password. All routers come pre-installed with a default username and password, but some users change these settings. Consult your router manufacturer for information on how to assign a specific device a static IP and open ports to forward ports (or reference portforward.com).

TL;DR

go to portforward.com
