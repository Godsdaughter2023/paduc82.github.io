jaimelobo say:

The exact specs will dependent upon how you receive the TV signal. For over-the-air digital (using an antenna), you need one that receives ATSC signals, for digital unscrambled cable channels, it should be one that receives Clear QAM. If you have a set top box from you cable company, then you need to make sure the tuner has the correct connection to hook up to that (lik s-video). Fortunately most tuners these days have all the needed options. Some also have FM radio capability.

If you have a laptop, you probably want a USB tuner (although there are some that will connect the the laptop via the card slot on the side), just make sure it is Windows 7 compatible. Many folks like the Hauppauge or Avermedia ones, but some folks have had good luck cheap no-name brands as well. Here is a couple examples of some at newegg:

http://www.newegg.com/Product/Product.aspx?Item=N82E16815100048

This one includes a remote, if needed

http://www.newegg.com/Product/Product.aspx?Item=N82E16815116034

Here is one that slides into a slot in the side of the computer (there are various types of card slots)

http://www.newegg.com/Product/Product.aspx?Item=N82E16815100045

You can see many of these are $50-$75, but if you look around or watch the deal web sites, some can be found for <$30 at times and as I mentioned, the cheaper one may work fine. Like anything, you need to read some reviews.

Installing is usually pretty easy, plug it in and follow the on screen instructions. Once the tuner is installed, there is a setup wizard in Windows that walks you through the process of connecting and setting up the channels for your location. Here's a little video with an overview of the process.

http://www.microsoft.com/windows/windows-media-center/learn-more/record-tv/default.aspx

Retrieve your product key:

With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Eliminating the product key sticker also removes any worry that the sticker might get damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If the user has to reinstall the operating system on a machine that came with Windows 8, the installation process automatically grabs the software product key from the motherboard BIOS with no input from the user. This means that those familiar Windows product key stickers will no longer appear on the Windows 8 computers.

If Windows 8 setup did not retrieve your key and you had to use a generic key from the link provided, here is how to retrieve the product key from the BIOS.
