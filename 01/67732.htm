lemp say:

If you are seeing a program or a pop-up message that identifies itself as "Windows Command Processor" then that is malware and needs to be removed.  You may need to request expert guidance in removing this.

If you're just asking a general question, then that phrase may refer to cmd.exe which is the program built into modern versions of Windows that opens the black "Command Prompt" window that many people mistakenly call the "DOS" window.  This program "processes" commands that are typed into the window in a manner similar to the way personal computers were used prior to the development of graphical user interfaces like Microsoft Windows.
