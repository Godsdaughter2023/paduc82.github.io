WindowsSearch.exe missing uncdms.dll error - fixed on my Win7 machine - Microsoft Community

mkrw14 say:

WindowsSearch.exe missing uncdms.dll error - fixed on my Win7 machine

I'm starting a new discussion on this because for two days I've been searching for a solution and there were tons of opinions offered and nothing seemed to work until I stumbled upon this solution for myself. With luck, this will work for others in my situation and they won't have to slog through so many different offerings that, for me at least, did not work.

The situation:

(1) I did a clean install of Window 7 on a new virtual machine. All went well and on start-up there was no "WindowsSearch.exe could not start because uncdms.dll is missing" error.

(2) I used Windows Easy Transfer to transfer everything from an XP machine to the new Windows 7 machine.

(3) After the transfer, the "WindowsSearch.exe" error started appearing with every start up.

This course of events made me think that during file transfer some version of some file having to do with Windows Search had been replaced on the new machine by a file from the old XP machine. This made me think to look for a compatibility problem in the replaced file, so I:

(1) Start > Control Panel > Programs

(2) Under "Programs and Features" there is a link called "Run programs made for previous versions of Windows". I clicked it.

(3) In the "Program Compatibility" window, I clicked "Advanced" then clicked "Run as administrator" then cliicked "Next".

(4) I was presented with a list: "Select the program you are having problems with". I scrolled down to "Windows Search", selected it, then clicked "Next".

(5) In the "Select troubleshooting option" window, I selected "Try recommended settings".

(6) In the next screen, I clicked "Start the program..."

(7) When asked if I wanted "WindowsSearch.exe" to make changes to my disk, I said yes.

(8) When I was returned to "Test compatibility..." screen, I clicked "Next".

(9) On the next screen even though I couldn't test if the problem was fixed (would require restart), I selected "Yes, save these settings for this program".

(10) I closed the Troubleshooter, restarted my machine.

The WindowsSearch.exe start-up error message was gone. So far it has not returned.
