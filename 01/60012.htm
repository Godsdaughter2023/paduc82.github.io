a. user say:

 It just assumes it's a Public network and assigns an incorrect IP Address.

No such thing as an 'incorrect IP' address - the IP address doesn't make a network private or public.

A public network is simply a network that has no password - aka an 'open' network - or has a password that is published. 

Mobile devices often have built-in utilities or available apps that will try to determine whether a network is public, in which case the utility or app automatically assigns settings for the type of network it determines. Perhaps you should be looking at the settings for the utility or app.
