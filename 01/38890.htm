arupratan say:

Crossover or Straight-through cables are no more of concern because in today's computers these cables are treated as the same. You can use either crossover or straightthrough in any topology and connecting any device without interrupting the functionality because today's machines use a technique of "Electronically Swapping" which converts the crossover or straightthrough signals and makes usable. Crossover straightthrough problem was present in the early computers and doesn't exist in today's computers. It means your problem is not for crossover cable.

It is common in Windows 7 when you connect two PCs with an Ethernet cable (crossover/straighttthrough), it will show "Unidentified Network". It's ok, "Unidentified Network" is not a problem. This thing is shown because of security reasons and nothing else. The problem you described is neither happening because of a crossover cable nor for the "unidentified network".

Step1: First make sure in the  Advanced sharing settings in Network and Sharing Center , "Network Discovery" is enabled for Home profile in both your Laptop and Desktop. You can't make a Homegroup using Public profile. If your Network and Sharing Center shows that the network profile is Public then make it Home. To make a Public profile to a Home profile, click Homegroup an click "What is a network location".

Remember, whatever your network location is, you can't view the computers in the network until the Network Discovery is enabled. So first make sure this.

If you have Network Discovery enabled then refresh the Network and wait. Sometimes it takes several minutes to view the computers connected in the network. So be patient.

*** Network discovery always remains disabled by default, so everytime you disconnect the network, Network discovery also becomes disabled. So, check this again and again.

To open  Network and Sharing Center open  Control Panel , click Network and Internet , and click Network and Sharing Center .

Step2. If you still can't view the computer, check whether both your Laptop and Desktop can ping each other. To ping each other open Command Prompt by typing "cmd" in the Run , then type "ping ip address ". If your computers can't ping each other then check the cable connection and verify if the cable is good or not.

Hope this will solve your problem and if not ask again. Also contact me with any issues related to computers here .
