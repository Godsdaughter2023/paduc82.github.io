a. user say:

Is it possible that your computer can no longer be updated?

Windows 10 has built-in telemetry that - when it works - detects conditions that would make it unwise to install further updates, and tells Microsoft Update to stop updating the computer. There are essentially three conditions that would cause this to happen: 1) hardware known to cause a problem for Windows 10; 2) otherwise valid configuration settings that can cause problems, due to known bugs in the update; and 3) user errors that make it impossible to install an update.

The telemetry is a work-in-progress, but when it works right, it saves lots of headaches by cancelling an update that would otherwise fail and leaving the computer untouched. If this is what happened in your case then thank your lucky stars. I hope to be so lucky because my 9-year old computer is a candidate for condition #1 above.

If you look in Event Viewer at the exact time you last tried to install an update, you will likely see an entry with a description (written in geek-ese) that describes what happened and why.

If your computer is unable to receive further updates you should not try to force them. PCs respond poorly to enemas.

All the above presumes that you haven't in the past tried to interfere with Microsoft Update's ability to deliver updates. Unfortunately, forcing updates and interfering with Microsoft Update are both strategies favored by Directly's 'independent advisors.'
