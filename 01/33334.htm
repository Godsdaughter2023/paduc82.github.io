It sounds like you have two problems.  I'll try to tackle each.

First, The email option you are using from your OpenOffice program is only available if you have a desktop email program installed (such as Microsoft Office Outlook).  This is because this type of command uses MAPI (Messaging Application Programming Interfaces) to send the email.  Possible solutions:

  1. Install a MAPI-compatible email program such as Office Outlook (or whatever email program you prefer).


    or

  2. Follow your OpenOffice suggestion:  save the file locally and then attach it to a new email normally (that varies depending on what email app or website you use).


Next, you cannot add CenturyLink to the Windows 8 built-in email app.  Please note that the mail app only works with IMAP email services.  CenturyLink might be using a POP type of service, which will not work.  Possible solutions:

  1. Setup your CenturyLink mail to be retrieved into Outlook.com (the Hotmail successor, not to be confused with Office Outlook).  Then the Windows 8 mail app can show that mail and send outgoing email to and from your CenturyLink email.


    or

  2. Install a different email program.  Whichever one you used on your computer before Windows 8 will work fine now too.

Well the only thing I can really think of is that the computer is going to be a lot more lenient about disconnects, most computers will attempt to reconnect for up to a minute before they actually tell you there's a problem. The streaming of TV would also be a little different than the way the game works because it's a 1 way transaction with a buffer, so if it loses connection it will just read from the buffer until the connection is established again.

The xbox however isn't going to be able to buffer gaming. If it's losing connection it will have to disconnect because most everything it does is real time, I'd only be able to suggest hooking up another xbox to the connection and seein how that works since that's the only real device that will operate on the same principles. It's not a computer, it uses those components but ultimatly it's whole operating system fits on a flash rom chip on the motherboard, so it can't be as complex and forgiving as any computer.
