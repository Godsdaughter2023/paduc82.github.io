Do you have a C:/Windows.old folder?  If so the old install is archived there so you can browse into your User folders to rescue them to the new install. 

There isn't really anyplace else the pictures could have gone because Windows Update doesn't delete files but in rare cases when it has problems it can channel files into that archive created during an Upgrade install (which version 1607 is).
