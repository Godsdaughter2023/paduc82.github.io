frederik long say:

Is there a way that I can make this happen so that I can install programs for myself only?


Thanks!

Not really. When you install a program under Windows then several things happen:
  • Program code usually goes into one of the Program Files folders. Everyone has at least read-access to these folders.
  • Registry data goes into the registry. Again everyone has read-access.
  • Shortcuts go to the common desktop or the common Startup folder. They are visible to everyone.
Some programs give you the option of placing the shortcut on the current user's desktop only. Alternatively you could manually move them from the common desktop to your own desktop. This is sufficient protection against non-inquisitive users.

You can get stronger protection by altering the permissions for the relevant subfolder in the Program Files folder. Warning: When setting permissions, give full access to admins and the System account, read-only access to selected users. Do NOT use the "deny" box. It often has unexpected and disastrous consequences.
