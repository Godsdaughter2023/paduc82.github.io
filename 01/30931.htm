cmdrkeene say:
That's up to the program that is currently in-focus.  For example, the F5 key does something different in every program.  Windows itself has no control over this.

Internet Explorer (and most browsers) use F5 to refresh.
Windows Notepad uses F5 to insert the current date and time.
Your database program uses F5 to clear all your work (or whatever it does)

Windows can't block programs from using a key.  Your best bet at disabling the key is to snap it off your keyboard.
