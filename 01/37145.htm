Hello, my name is Bradford and I'm an Independent Advisor for the Microsoft Community and I'll help to answer your question.

The portion of RAM not available to Windows is being used by the CPU as Graphics Memory. Graphics Memory meaning memory meant to hold images, data for making what you see on your screen, and so on.

You can adjust in your computer's Bios the amount of RAM the CPU uses as Graphics Memory or Disable onboard graphics ONLY IF you have a graphics card installed.

If your system begins to run low on memory the CPU will then free up some of the memory it's reserved for Graphics.

This is behavior is normal so no need to be concerned.
