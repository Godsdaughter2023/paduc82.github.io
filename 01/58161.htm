balaji kundalam say:

Hi

Thanks for contacting Microsoft Community.

 

Fan in laptop/desktop runs all the time but only the speed varies based on the requirements. If the processor/graphics processor usage is high, fan speed will be high and if the processor/graphics processor usage is low, fan speed will be less. On an average, fan speed will be 20% - 30% of the maximum speed, at idle condition or when processor/graphics processor usage is less. Note that high usage increases heat and so fan usage will be high. It is absolutely normal and no need to worry of this.

 

You can use the utilities application came with your laptop, for eg, Dell Support Center in case of Dell, for viewing the processor temperature as well as fan speed in RPM.

 

You can also use any trusted third party utility for this purpose. You can google/bing it.

 

Kindly DO NOT TRY TO CHANGE THE FAN SPEED USING ANY THIRD PARTY TOOL. IT WILL CAUSE STABILITY ISSUES IN THE LAPTOP WHICH MAY REQUIRE SERVICING BY THE MANUFACTURER.
