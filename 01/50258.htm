martystanley say:

I've lost my DRM rights and tried to do a manuel restore, didn't work If I upgrade to windows 7, will that fix the problem? __

Media Center will not work.
mouneshwar r say:

Hi ,

Welcome to Microsoft Answers!

Many DRM licenses maintain information based on the hardware configuration of the computer they are installed on, making major hardware changes could make your DRM-protected media unplayable. These steps apply to Windows Media Player 10 and lower. Windows Media Player 11 no longer allows the manual backup or restoration of DRM licenses. Windows Media Player 11 has an automated DRM recovery system

I would suggest you to search for your DRM files, back it up and then restore it from the windows media player.

Note: Your Internet connection has to be active when you’re restoring the DRM files.

1. Search for your backed-up DRM licenses by navigating to "Start"->"Search" and type "DRM*.bak" to locate the file. The .bak file is a backup of your DRM license.

2. Copy and paste the DRM backup file to your desktop. While this step is unnecessary, it will make the backup file easier to find

3. Start Windows Media Player and navigate to "Tools"->"Manage Licenses" (or "License Management" depending on the version of Windows Media Player).

4. Select the DRM backup file you placed on your desktop from the file browse dialogue box and click "OK."

5. Ensure that your Internet connection is active and click "Restore Now" to restore your DRM licenses.

If you’ve tried the same method in order to restore, then I would suggest you to ignore it.

Check out the link below which talks about You may be unable to play protected content after your computer hardware changes

http://support.microsoft.com/kb/891664

I don’t think upgrading your Windows Vista to Windows 7 will not get you the DRM Licenses back.

Hope this information is helpful.

Regards,

Mouneshwar R – Microsoft Support
