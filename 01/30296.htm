Options for freeing up space:
Windows over time will accumulate data outside of the normal types of data we store. This can include things like copies of Windows Updates, memory dump files used to diagnose and troubleshoot error messages, temporary internet files for web pages we often visit and Windows Upgrade files, commonly used to assist with installing updates for Windows. Sometimes we don't need this type of information and removing it can free up precious disk space inaddition to improving overall system performace.
Having enough disk space is especially important if you are performing task such as upgrading to the new Windows 10. It is one of the the first things Windows setup checks before it starts copying files. Large programs too will often require that you have enough disk space available.

Launch Disk Cleanup, the quickest way is this:
Press Windows key + R then type: cleanmgr.exe
Then hit Enter
Windows 10 > Start > All Apps > Windows Accessories > System Tools > Disk Cleanup
Once Disk cleanup opens, select the drive where Windows is installed if its not already.
Windows will scan your hard disk for files it can clean up.
Note some of these files too can include older installations of Windows. For example, if you upgraded from a previous version of Windows 10, a copy will be kept at the root of the hard disk in a folder called Windows.old. Please becareful though, this folder can actually be useful if you need to go back to your previous version of Windows for whatever reason. So be mindful before deleting it. Removing it though can significantly free up disk space.
After Disk Cleanup is finished scanning, you will see the Disk Cleanup window. This window provides a list of areas of the system you can clean up safely. Before we start cleaning up the system, you will notice appears on the window called Clean up system files. Clean it and the Disk Cleanup will run a thorough check of the system again to include every file that can be removed.

chihamie say:

Hard disk space missing

Iam having problems with my hard disk. All unused space has vanished from my hard disk and i have 0 kb of memory space. Please help
cager say:

As you are in the Windows XP forum, I'll assume that your pc is running XP (not Vista or Windows 7).

Go into Control Panel & open the "Folder options" icon.

When the Folder Options window opens click on the "View" tab and tou will see a scroll bow labelled "Advanced settings" in the bottom part of the window.

Part way down the scroll box, under "Hidden files and folders" there are 2 options - you should click the "Show hidden files and folders" option.

Then open Windows Explorer and go to your pc's system drive (usually C:). Click on the Windows folder and you will see a number of sub-folders labelled "$NtUninstallKBxxxx$. These are the uninstall files for the various Windows updates that you or your pc have downloaded and installed. If you have never deleted any of these folders & had automatic updating enabled, you will find quite a number of them.

It is unlikely that you will need to use them because your pc is presumably running OK after they have all been installed - so delete the uninstall folders & release some space on your system drive.

Secondly, while in Windows Explorer, right click on the system drive name and select "properties" from the drop down box that opens. You will see a graphic showing the drive capacity & free/used space. Click the Disk Cleanup button & Windows calculates how much space you could free-up on the drive. A Disk Cleanup window then appears with a scroll box showing tick boxes against a variety of file types. The ones you should tick are the "Downloaded Program Files", "Temporary Internet Files", "Recycle Bin", "Temporary Files", "WebClient/Publisher Temporary Files" and "Temporary Offline Files". Then click the OK button & Windows will remove the files in the categories you chose.

It will also offer to free up space by Compressing old files - but you don't want to do that until other avenues have been exhausted.

Thirdly, do you have another drive (D:, say)? - this is usually a separate partition set-up on your hard disc for data storage, etc - but people tend not to use it. If you have one, you will see it in Windows Explorer. If you right click on the drive name and select "properties" in the drop down box that opens, you will be shown information on the capacity & used/free space on the drive. If you were not aware of it or haven't used it, you will find quite a bit of space available.

Under Windows XP, applications don't have to be installed on the system (C:) drive - although do so normally by default. Whenever you install a new application you are given the choice to "browse" to a location where the software is to be installed. If you have installed a number of applications without telling the pc to install them on a drive other than the system drive, this software & related data will have contributed to filling-up the system drive.

Fourthly, Windows uses a "Swap file" - usually an area of the system drive to which it can copy portions of the operating system or other software when it starts to run out of system memory. By default the swap file is managed by Windows on the system drive & takes an area of the system drive varying from the system memory size up to 1.5 times the system memory of your pc - so if you had 512Mb system memory, the swap file size could be around 1.5Gb - however, because Windows manages it, when your system drive free space gets a bit low, you start getting messages from Windows saying that you have no available memory space. This is because Windows treats the swap file as part of your system memory.

There are several ways to get over this - the first, of course, being to get rid of some of the rubbish that has accumulated on the system drive (see above). The second, if you have another drive (D:, say), is to move the swap file off the system drive and onto that other drive.

This can be done by going into the Control Panel and clicking on the "System" icon. A System Properties window opens - click on the "Advanced" tab and then on the "Settings" button in the "Performance" box. Note the comment at the top saying that you need to be logged on as an Administrator to make most of the changes.

When you click the settings button a "Performance Options" window will open - click on the "Advanced" tab and at the bottom will be a box labelled "Virtual memory" giving you information about the paging file (the swap file by another name). Press the button labelled "Change" and a window labelled "Virtual Memory" opens with a scoll box showing the paging file size set on each of the drives in your pc. It will probably just show a paging file set on your system drive (C:). At the bottom of the window is a box labelled "Total paging file size for all drives", which gives the minimum size allowed, the recommended size and the size currentlu allocated.

What you do now is to select your D: drive in the scroll box and then in the box below labelled "Paging file size for selected drive", select the "Custom Size" option and  type into the 2 boxes (Initial size) and (Max Size) the same figures - which should be just above the "recommended size" given in the box at the bottom of the window. Then press the "Set" button and the paging file size will appear in the scroll box above. By having the same initial and max sizes, Windows will set aside that specific area of the drive & won't manage it up & down - thereby reducing fragmentation on the drive.

Next you select your system drive (C:) in the scroll box and in the "Paging file size for the selected drive" box select the "No paging file" option. Click the "Set" button and you will have told Windows to move your swap file from the system drive to your other drive. Press the OK buttons to get you out of all the windows & you will need to reboot your pc to start the new configuration.
