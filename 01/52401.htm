Apply to Folders works on a per-type basis. Folder types are: General/Generic, Documents, Photos, Music, Video, Contacts, and a few others. So you have to use it once for each folder type.

You can determine the type of folder by viewing the Customize tab in the folder's Properties dialog:

For each type, Windows creates an entry in the Registry under:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Streams\Defaults

Here, there is only a template set for Documents, represented by the entry named '{7D49D726-3C21-4F05-99AA-FDC2C9474656}'

You can check here to see if you have a template set. The common folder types are:

-----------                            ----

{5fa96407-7e77-483c-ac93-691d05850de8} Videos
