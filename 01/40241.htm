Hi Vignes,

When you decided to use that drive as Windows Installation Media, you would have been told that all files would have been deleted . .

However, there is still a chance you can get back most of your Files!

When the Media Creation tool formatted the drive, it created a 32GB partition for the Windows Installation Media - any files in that partition are gone!

The rest of the drive should marked as unallocated in Disk Management - check this!

You will need to download 3rd party software that can recover files from Unallocated space. EaseUS and AOMEI have Free editions of their software that may be able to do this - try the Free versions only !
