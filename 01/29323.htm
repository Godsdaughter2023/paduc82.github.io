lemp say:

If the Command Prompt window opens when you boot up, one of the drivers either didn't install properly or didn't clean up after itself properly.


If you want to fix this, download and install (or run online) Autoruns, from here --> http://technet.microsoft.com/en-us/sysinternals/bb963902.aspx

After Autoruns completes its scan, select the "Logon" tab and look for an entry that includes cmd.exe in the "image path" column.  If you can't find it, click the "Everything" tab, then click File > Find and search for cmd.exe.

If you find a line that includes cmd.exe, uncheck the box at the left end to disable the entry.  If when you next reboot, the Command Prompt window doesn't pop up and no other ill effects appear, you can re-run Autoruns and delete the line.  Unchecking is safer than deleting because you can re-check the box for any given entry and things will be restored as they were before you made the change.

If you can't find any line that includes cmd.exe, run Autoruns, select the Everything tab, and click File > Save.  Save the file (autoruns.arn) someplace that you'll be able to find it.

Go to OneDrive (https://onedrive.live.com/), sign in if necessary using the same userID and password you use to sign in to this forum, and upload the autoruns.arn file.  After the file is uploaded, right-click on it, select Share, and "get a link."  Copy and then post the link in your reply here.  For more details on how to use OneDrive, see http://windows.microsoft.com/en-us/onedrive/share-file-folder

Before proceeding with the troubleshooting steps, Connect your Printer to the computer and Switch it ON. In case of Wireless printer, Switch it ON and connect it to the Wifi network.

Download Link - http://ftp.hp.com/pub/printers/hppsdr/HPPSdr.exe

2. Update Printer Driver
Download printer driver from printer manufacturer's website and reinstall.

3. Uninstall and reinstall the Printer (Credit to Andre Da Costa, MVP for providing steps)
A. Uninstall Printer
Click on Windows Key+X > Apps and Features > Scroll down and click Programs and Features > Select your printer > Select Uninstall
Type Printer in Windows Search box > Printers & Scanners > Select your printer > Remove device
Type Printer in Windows Search box > Click Printers & Scanners > Scroll down and click Print server properties under Related settings > Under Driver tab, look for your printer > Select it and click Remove button at the bottom

B. Install Printer
Go to the Printer manufacturer’s website and search for the latest Windows 10 drivers > Reinstall the Printer driver > Reboot the computer
Type Printer in Windows Start Search box > Click Printers & Scanners > On Right side, Click Add a printer or scanner > If Windows detects your printer, it will be listed > Select the printer and follow on screen directions to set it up (In case of Wifi printer, your computer should also be logged into Wifi network)
If Windows doesn't detect your printer, then you will get a blue message - Click The printer that I want isn't listed
If you are using Bluetooth / Wirelss printer > Select Add a Bluetooth, wireless or network discoverable printer > Choose the printer > Select your Printer and follow on screen directions
If you are using wired printer > Select Add a local printer or network printer with manual settings > Select Use an existing port > Select your Printer and follow on screen directions

andre for directly say:
Hi Debora

My name is Andre Da Costa; an Independent Consultant, Windows Insider MVP and Windows & Devices for IT MVP. I'm here to help you with your problem.

Open Start > Settings > Update & security > Recovery > under Go back to my previous version of Windows 10, click Get started.
or
Press Control + Alt + Delete
Click Sign out

At the Sign In screen, hold down the shift key on your keyboard while clicking the Power button on the screen
Continue to hold down the shift key while clicking Restart
Continue to hold down the shift key until the Advanced Recovery Options menu appears

Click Troubleshoot

Click Advanced options

Click Go back to the previous version of Windows
