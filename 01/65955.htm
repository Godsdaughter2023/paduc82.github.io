Arial Unicode MS Font Missing in Windows 7 Notepad (Asked and Answered)

In previous Windows versions, I could select the Arial Unicode MS font for the Notepad application, but in Windows 7, with Microsoft Office installed, the font is missing from the Notepad font list. The font does appear in the Microsoft Word font list. After some investigation, I found the following answer.
In Notepad, select Format > Font.
In the Font dialog box, click the Show more fonts link.
In the Fonts window, select the Arial Unicode MS font file, and click the Show button on the top-line menu.
In Notepad, close and reopen the Font dialog box, and Arial Unicode MS should now be visible.
