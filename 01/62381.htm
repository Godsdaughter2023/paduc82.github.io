Step 1:

 

To resolve this problem, clear the Windows Media Player database. To do this, follow these steps:

 

    a. Exit Windows Media Player.

    b. Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

    c. Select all the files in the folder, and then click Delete on the File menu.

    d. Restart Windows Media Player.

   a. Exit Windows Media Player.

OK - that's a different sort of error message that I've never seen before, but the screenshot gives rise to a couple of questions.

In the message compose window, if there is more than one account available to send with, there will normally be a From box to the right of the To... box:

You clearly have two accounts, but I wonder if they're in fact two instances of the same account. This might be the reason why Windows Live Mail can't send; it can't work out which of the two accounts you want to use.

If they are in fact the same account, you should remove one of them. Right-click on the account name in the folder pane and select Remove account. As long as you did not call and give them remote access to your computer, you have nothing to worry about.
