a. user say:

YES - and the key is to immediately stop using the computer.

There are two types of formatting:

A high-level format is what most people mean when they talk about formatting a hard drive. In a high-level, format, partition data is erased so the files can no longer be found - but those files are still on the hard drive. You can retrieve these files with special file recovery software. The reason for immediately stopping to use the computer is to not over-write those files.

A low-level format, aka a 'bare metal' format, is when the entire hard drive is overwritten with zeroes. In a low-level format, all files on the hard drive are gone for good. If that's the case you'll need sophisticated (and expensive!) software and techniques to measure the magneric charges on the surface of the disk to have any chance of recovering files.

For a high-level format you can find plenty of file recovery software on the internet. Recuva is popular. On a different computer, read about file recovery programs and download one that sounds right to you. Then you can get to work.

If you're not totally confident in your technical skills - you're a member of a big club - of if you've lost really critical or precious data, it might be worthwhile to have a professional do this.

Hi,

This issue might occur if the system files has been corrupted.

I would have you run the system file checker (SFC) scan in Windows Recovery Environment and see if it fixes the issue.

1. Use the power button to restart your computer during the boot process three times (do not let it get to the end of the boot process). This should bring up the Windows Recovery Environment.

2. In the Windows Recovery Environment, select command prompt.

sfc /scannow

Note: If any files require a replace SFC will replace them. You may be asked to insert your Win8 DVD for this process to continue.

If the issue persists, you may restore your computer to an earlier working condition and check if it helps.
