I have just written this article about customising the Desktop and the Taskbar. It could help.

The new Desktop and the changes to the Start Menu in Windows 10 can be quite a challenge for a user migrating from Windows 7. It is important therefore to customise the displays to suit the way you wish to work and to give easy access to the programmes you use regularly.

 

If you use two or three programmes much of the time you may find it convenient to pin programmes to the Taskbar and use Jump Lists. In the first image below you will see a total of 15 programme icons including Excel, Word, Internet Explorer, and other less known programmes pinned to the Taskbar. It is for you to choose what appears there. What follows next is one way how to do it.

 

The program files for the principal Office 2010 components are:
Excel = excel.exe
Outlook = outlook.exe

Word = winword.exe

 

If you need other exe file details please ask.

 

You can open or run any program e.g., Excel, by selecting the keyboard shortcut Win+R, type excel.exe and press the ENTER key. An Excel work sheet opens and an icon is temporarily added to the taskbar, usually at the bottom of the screen. Right click on the icon and select Pin to taskbar. The next time you want to open Excel you click the icon. If you right click the icon you will see recent Excel files you have opened. You can click any recent file to open the file. If you want to be able to access a particular Excel file regularly right click the file and select Pin to this list. The right click options linked to the taskbar icons are known as Jump lists. You can use these procedures with Word, Outlook and many other programmes.

 

You can customise the Desktop by deciding which panels appear on your Desktop and their position. Click the three lines 1 icon in the top left corner of the desktop and then click All apps 2 at the bottom of the left pane.

 

 

Click A 3 at the top of the list of programmes beginning with the letter A to get an A to Z index. Click M to get a list of programmes beginning with the letter M.

 

 

Click Microsoft Office 4 to expand it to see Microsoft Excel 2010. Right click Microsoft Excel 2010 5 and click Pin to Start (the drop down menu also offers another way to Pin to Taskbar). An Excel panel appears on your Desktop. If you do not immediately see it, scroll down the screen as far as you can go. It will be there. You can then drag any icon to an empty space on your screen. If you want to see Excel in the top left of the Desktop. You will need to move the existing item occupying the place to another position on the screen so that the space is vacant. You can then move Excel there.

Method 1: Reset Windows update components.

Resetting Windows Update Components will fix corrupt Windows Update Components and help you to install the Windows Updates. Please follow the below steps to reset the Windows Updates Components manually:

  1. Press Windows Key + X on the keyboard and then select “Command Prompt (Admin)” from the menu.
  2. Stop the BITS, Cryptographic, MSI Installer and the Windows Update Services. To do this, type the following commands at a command prompt. Press the “ENTER” key after you type each command.
    • net stop wuauserv
    • net stop cryptSvc
    • net stop bits
    • net stop msiserver
  3. Now rename the SoftwareDistribution and Catroot2 folder. You can do this by typing the following commands in the Command Prompt. Press the “ENTER” key after you type each command.
    • ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
    • ren C:\Windows\System32\catroot2 Catroot2.old
  4. Now, let’s restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt for this. Press the ENTER key after you type each command.
    • net start wuauserv
    • net start cryptSvc
    • net start bits
    • net start msiserver

5. Type Exit in the Command Prompt to close it.

Now you may try running the Windows Updates and check if the above steps resolve the issue.

Click Start, type Anytime Upgrade, click the option to enter a key, enter the Windows 7 Ultimate key when requested, click Next, wait while the key is verified, accept the license agreement, click upgrade, wait while the software upgrades, (it might take 10 minutes or more depending on if updates are needed), your computer will automatically restart, after restarting, Windows 7 will notifiy you its updating the computer, the computer will restart one more time automatically and the upgrade will be completed, a windows will pop up notifying the upgrade was successful and your computer is ready to use, click close, you should be upgraded to Windows 7 Ultimate your files, programs and settings preserved.
