burrwalnut say:

To delete a small number of searches, click inside the Explorer search box, scroll down to the item you want to remove and press the Delete key to remove it from the search history. Continue pressing Delete to remove other entries.

 

If you have a large number of searches and want to remove them all, it is quicker to delete the registry key where they are stored. To do so, open a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery. Right-click on WordWheelQuery and click Delete. The key will be recreated next time you run a search.

 

It is prudent to make a backup of the registry or create a system restore point before making any registry changes.

 

To identify an individual search in the registry, double-click on its Name in the right pane to see the translation.
