I too was having the “…login with administrator privileges…” error message, and was disheartened to see it was a relatively new problem without a verified fix.
 
I tried all the obvious resolutions: uninstall/reinstall ; enabling the admin account, as described ; clearing the leftover registry entries between uninstall/reinstall ; disabling the firewall ; disabling AV ; disabling windows defender … all to no avail.

Finally, I realized it was such a recent problem it may be related to updates – and since windows 10 controls updates in a new, overbearing way, it was wrongfully being blamed for something I was encountering on my Windows 7 machine.

So I removed all the recent updates, and sure enough RON now launches.  So now it was a matter of identifying the specific culprit.

…Well I’ve found it.

Windows Security Update for x64 Systems KB3086255

Remove the update, and you’ll be able to play again – regardless of your OS.

Once I reinstalled it, sure enough RON wouldn’t start again – and then tried it twice more to verify.


**TO UNINSTALL**

Either, search for “Installed Updates” and then identify the described update

...OR...
 
Simply copy and paste the below address into the windows search bar

C:\Windows\explorer.exe shell:::{d450a8a1-9568-45c7-9c0e-b4f9fb4537bd}

**NOTE**

This is for Windows 7 & 8.1 ... you will need to reboot the machine for the changes to take effect.
