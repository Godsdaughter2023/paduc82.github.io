shenan stanley say:

Manual copy only what you need/what YOU created.  Your pictures, music, documents, Internet Explorer favorites, etc.  You should need nothing from AppData (with rare exception.)

My suggestion: copy/paste everything YOU CREATED (see above) over to the new profile.

Restart the computer.

Rename the entire OLD user profile directory (C:\USERS\OLDNAME to C:\USER\TEMP-OLDNAME-TEMP).

Restart the computer.

Use it for a while, see how things work.  If you are not missing everything, you could copy the old directory to an external drive to be sure and/or just delete it at that point.
