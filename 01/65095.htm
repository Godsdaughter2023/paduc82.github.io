Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - admin
  • Password - 1234 or telus

Restore Factory Defaults

Through firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 in the address bar. (Or click one of the links) Enter the username and password if the gateway asks. (admin/1234, admin/telus).
  4. Select Maintenance from the menu at the bottom.
  5. Select Configuration from the submenu.
  6. Select the Reset button.

Through Hardware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the reset button on the device.
  3. With paperclip/pen/toothpick hold down the reset button down for 30 seconds.
  4. Wait for lights to turn off then return to normal.
