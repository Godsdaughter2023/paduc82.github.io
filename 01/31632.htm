Why is the default for System Restore/Protection - Microsoft Community

jdelaney77 say:

Why is the default for System Restore/Protection Settings/(C:) set to off?

Hi,

Why is the default for System Restore/Protection Settings/(C:) set to off? Pleas advise.

Thanks,

Joe

a. user say:

When System Restore was first introduced (in Windows ME!) Windows didn't have a good way to rollback the operating system if it became messed up.

Windows 10 has several ways to rollback, and they make more sense than System Restore. You can Reset, Refresh or restore a System Image. There are more third-party applications for restoring your computer (and they're easier to use than Windows' own tools, in my not-so-humble opinion.)

Add to that the fact that restore points have limited usefulness. Windows 10 has been undergoing substantial changes since it was first released, and with every major update, previous restore points get erased. You definitely don't want to restore a pre-update restore point once a major update has been applied.

Finally: System Restore was always widely misunderstood. As a result it got lots of complaints from people who thought it was a form of backup. Not so.

For all these reasons, System Restore is pretty useless, and that may be why it's off by default. I would not be surprised to see it abandoned over time.

a. user say:

System Restore was originally created to restore the Windows user interface so users could access Windows' own tools to help fix a problem. If a restore point achieved anything more than that, it was a happy coincidence. Restore points were not designed to fix problems - they were designed to give users the opportunity to try to fix a problem with Windows' own tools.

Microsoft never did a good job of explaining this, and many users don't bother learning about the features of their computers anyway. They only see 'System Restore' and think it will restore their computers from whatever problem they're experiencing. That's not what System Restore does.

Like many Windows utilities, System Restore was created for people who don't know how or are too afraid to install third-party software. People who are comfortable with installing third-party software disabled System Restore - it's a waste of time and disk space - and instead made regular backups.
