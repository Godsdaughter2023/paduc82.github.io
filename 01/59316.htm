How can I change the file format of a group of files? - Microsoft Community

maxgf say:

How can I change the file format of a group of files?

Original title: Extension .jpg

 

I have been working with html for 17 years and kept jpeg images in lower cursive .jpg. Windows 8 win explorer uses upper cursive .JPG. How can I batch change a subdir of 700 images from .JPG to .jpg

Thanks.

a. user say:

Hi Max,

 

 

Please follow these steps to batch rename file extensions.

 

a.      Put all the files in a single folder for those you want to change the file format.

b.      Press Shift and right-click on the folder and select Open Command Window Here.

c.       You will now see a Command Window with the current folder directory open. Now type the following command line.

ren *.(current extension name) *.(extension you want)

d.      Now all the files inside the folder will change to the extension you want.

 

 

Let us know if you have any further queries related to the issue. We would be happy to help you.
