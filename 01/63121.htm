  • prompts to select which Win Installation to use
  • prompts for administrator password
  • The C:\WINDOWS> prompt is displayed and its files can be listed
  • MAP cmd shows A: Floppy, C: System Partition, D: Recovery Partition + 2 x CDs

lemp say:
I won't ask why you want to do this or comment on the possible dangers of deleting all image files, but the following should do what you want:

  • Do your search.
  • In the Search Results window, click View > Details (this just makes a one-column list, which is easier to work with)
  • Delete the files in groups:  click once on the file at the top of the list, then scroll down a ways, hold down the SHIFT key and click on another file.  All of the files between the two files you clicked on will be selected (including the start and end files).  Press the Delete key.
  • Keep repeating this in batches.  If you get an error message about a particular file, press the CTRL key and click on that file.  This will deselect it from the group.  You shouldn't have to deselect more than a few files in each group before the group will delete.
  • When you are done, you may have a few files left over that can't be deleted.  If so, post the file names here and someone will advise.
