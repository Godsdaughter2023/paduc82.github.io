a. user say:

For image backups I use Macrium Reflect Home Edition. This is truly excellent software with a comprehensive knowledgebase.

For file-and-folder backups I use 7-zip, which is actually a file compression program. By adding command line switches I can use it to make backups. If command lines are not your cup of tea, I previously used Backup4All, which is also excellent software.

All of these programs are rock-solid reliable, which is what you want in a backup program. They do have a learning curve, but your invested time will be amply rewarded.

ShadowProtect Desktop is another rock-solid reliable image backup program that I used for two years, but it's really meant for large scale business environments so it lacks some convenience features that home users appreciate. It's also expensive.

I've also used Norton Ghost and Acronis True Image for image backups. Ghost no longer exists and True Image suffered from half-baked (in my opinion) features and inadequate support.

For file-and-folder backup I used to use Winzip Pro, but Corel (who bought it from the developers) made it bloated with extraneous features and increased the price.

time lady say:
I don’t know if this will work in your case but I had the same problem - upgraded to Win 10 Pro from home via a win 8 key - when I clean installed it picked up the home key rather than the pro upgrade, I found this information & it worked for me:
Create a PID.txt file containing the following text:
[PID]
Value=XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

The Xs get replaced with your Windows 10 Pro product key. If your Windows 10 Pro activation is with a digital license, then insert the generic product key: VK7JG-NPHTM-C97JM-9MPGT-3V66T

Save the PID.txt file to the \Sources folder on your Windows 10 installation media (USB flash drive).

Windows 10 decides which version (Home or Pro) to install in the following order:

1. Finding a product key in the PID.txt file
2. Finding a product key in UEFI/BIOS
3. The user manually enters a product key when asked for one
4. No product key is found or entered - then the user is asked for Home or Pro
