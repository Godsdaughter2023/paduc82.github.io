outis metathemenos say: I was having this problem since I upgraded, turning off DPTF in the BIOS fixed it for me.  DPTF throttles the system whenever it detects the laptop's in a configuration other than standard, which, with Windows 10 Tablet Mode, can be always.  

bofoba say: How do you do that ?

outis metathemenos say:

Restart Windows while holding the Shift key, it should show a blue screen with options.  Select Troubleshoot -> Advanced Options -> UEFI Firmware Settings ->Restart.  

You should now boot into the computer's BIOS (technically a UEFI).  Use the arrow keys to move to the Configuration tab, then move down to DPTF and set it to Disabled.  Now move to the far right tab and select Exit Saving Changes and confirm.  

The computer will now restart to Windows and all should be well.  

This will probably lower your battery life, as your CPU is no longer throttled automatically in non-standard configurations, but it won't damage anything in and of itself.  You can still configure CPU power-saving features through Advanced Power Settings if you really want to.
