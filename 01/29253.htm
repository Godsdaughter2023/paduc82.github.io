@RedRobin183: The OP already has two external devices configured for Xbox storage already.

This problem is simply a limitation of the Xbox OS which hasn't been addressed.  However, there is a way to download to an external hard drive or Flash memory, even if it is a little convoluted.

1. Go to the Marketplace and download something, it doesn't matter what you choose, as long as it doesn't finish before you get to the next stage.
2. While the first item is still downloading, select the content that you really want to download, be that a full game, demo or DLC. This will be added to the download queue.
3. Press the X button on your console to bring up the mini guide.  Go to downloads.  You will see two items. Go to the second item in the queue (the content you actually want).  This will be 'Pending'. 
4.  Select it and press A on your controller.  Choose 'Change Storage Device'.
5. Cancel the first download.  The item that was pending will now download to your chosen storage device.


Hope that helps!

achillesxiong say:

Error 80070643 stops me from installing any programs on Windows 10

I just got my new 2-in-1 and went through all the usual procedures at set up. I tried to install Google Chrome but showed error and it failed.  Then I tried to install Gopro Studio and error 80070643 happened. I need know what is going on and know what I can do to fix this. Help, please.

a. user say:

Hello,

I realize the inconvenience you are experiencing with Windows 10. I will certainly help you.

The error code 80070643 is a generic error that basically state that an error was encountered by Windows Installer.

I suggest you to follow the methods below and check if it helps:

Method 1:

I suggest you to start the Windows Installer services.

1. Press Windows Key + R Key. 

2. Type "services.msc" without the quotes and hit on Enter Key.

3. Search for Windows installer.

4. Double click on it and under Startup type, select Manually.

5. Click onApply and click on Ok. 

Method 2:

If the service is started you might need to unregister and reregister the Windows Installer. To do so, follow the steps given below.

  1. Type "cmd" in the Start Search box.

  2. Right-click on the "cmd" program and choose "Run as Administrator"

  3.  In the command prompt window type:
    msiexec /unregister and hit "Enter"

  4. Then type:
    msiexec /register and hit "Enter"

  5.  Then type "exit" and hit "Enter"

Now try installing program and check what happens.

Hope this helps. If the issue remains unresolved, please get back to us and we would be happy to help.

Hi CHarles, here is how to revert to Windows 7 . . .

Insert the bootable media into the PC

Start your PC, on the manufacturers logo screen press the appropriate button to get into BIOS Setup (Usually F2 or Del key)

Move the drive with the bootable media to the to of the Boot order list

Press F10 to save settings and restart

Windows 7 will begin installing

When your system reboots, go into BIOS and set your hard drive as top of the Boot Order

Do not remove the Installation media until you reach the Desktop
