Hello DerekDiCiacca,

Error codes are difficult to pin down and somewhat generic.  A lot of information is available by performing a search of "Error Oxc0000022" ....how good that information is another matter.


To rule out file corruptions I would perform the following first.

1.  SFC /scannow in elevated command prompt. 

2.  DISM.exe /online /cleanup-image /restorehealth  in same      command prompt...being cautious about spacings.

3.  CHKDSK /f  in same command prompt.....answer yes, and reboot

Try those first.

I wonder what version of Windows 10 are you using?  Sometimes when all else fails, an in place repair of Windows 10 with the Media Creation Tool (MCT) down loaded to an USB thumb drive and then run as "setup.exe" within Windows can be most effective as a repair of Windows 10 problems.  This is an in place repair that allows you to retain your apps, files, and programs.... if you are on the same version 1703.  That is the stipulation and caveat!  The repair will fail if you are a different version of Windows and could force you to install 1703 without your files!

Hoping this helps with your problem.

gerry c j cornell say:

A lot of problems arise from setting Windows Update to automatically update. You may prefer to change this setting. Select Start, All Programs, Windows Update, Change Settings. Under Important Updates is a box showing the current setting. Click on the Arrow down to the right and change the selection to “Check for Updates but let me choose whether to download and install them” and click on OK. Restart your computer.

 

Select Start, All Programs, Windows Update, Check for Updates. If optional updates are available click optional updates and make sure that the box before Upgrade to Windows 10 is unchecked. Right click on Upgrade to Windows 10 and select Hide this Update. Check the boxes before the remaining updates. Click on Install and click again on Install in the next window and the process of installing starts. You should let the process proceed without interruption. Multi-tasking at the time updating is in progress is unwise. The progress window will tell you what is going on. You need to be patient. Restart the computer when prompted to do so.

 

You may find that Upgrade to Windows 10 does not stay hidden. If that happens repeat the exercise as necessary.

 

Right click on the Start button and select Properties, Taskbar tab. Click Customise (under Notification area). In the next window place the cursor on GWX  1  and use the scroll  2  to change the behaviour to Hide icon and notifications  3  and click OK  4  . You may need to uncheck the box before Always show all icons and notifications on the taskbar to be able to change the GWX icon behaviour.
