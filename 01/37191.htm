ipconfig /flushdns

 

ipconfig /registerdns

 

ipconfig /release

 

ipconfig /renew

 

Check for the issue while browsing on Internet.

Hi Linda

Reading the CBS.Log can be a daunting task.

There is an easier way to read the results of the sfc /scannow command.

1. Click Start and type cmd.exe in the Search box.

2.In the Results, right click cmd.exe and select the Run as Administrator option.

3.Type the following command.

cd\users\xxxxx\desktop

(Where xxxxx is your user name)

Press Enter.

4.Type the following command.


findstr /c:"[SR]" %windir%\logs\cbs\cbs.log >sfcdetails.txt

Press Enter.


5. After the command completes, close the command window, go to your Desktop and open the

This file will only contain the results of the sfc /scannow operation.

Let us know if this works for you.
