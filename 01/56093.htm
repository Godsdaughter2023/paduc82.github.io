aarontomilson say:

I use Windows 7 task scheduler to schedule vlc (http://www.videolan.org/vlc/download-windows.html) to run a playlist that plays an internet radio station I like to listen to when I wake up and get ready for work.  

Then after an hour (could be any length of time) I have the task scheduler run a command file that kills the process that is vlc.exe playing the internet radio station.  The command file is just a text file with a .cmd extension and the following content:

taskkill /IM vlc.exe /T /F

You could use whatever player you like.  Find out how to run it from a command file that gets run by the task scheduler.  Have it play whatever radio station or other source of music you like. The only way I could find to make my player play an internet radio station from command line was to have a playlist that pointed to the internet radio station. Then find the name of your player's process and have task manager run a command file with taskkill to kill it by name or PID when you want it to stop playing.  

I went through many little alarm clock apps which all worked sporadically or not at all before I came up with this way.  It's been working flawlessly 5 days a week for years.
