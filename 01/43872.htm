[SOLUTION] Windows GameBar File Error - Microsoft Community

alden bansemer say:

[SOLUTION] Windows GameBar File Error

The Error:

Can't save your recording. Make sure your captures folder allows files to be saved.

My Solution:

I had this issue on my desktop running Windows 10 Pro. I referred to this thread to try and get a solution. I'd post this as a reply if Microsoft hadn't locked the thread when they couldn't fix the problem. Basically, a recent update to Windows changed the way the Game Bar save directory works. Whenever you move the Captures folder, Microsoft decides to use your new location as the save directory for screenshots/video. So the error for me appeared after updating Windows and performing my weekly manual backup to an external drive. After copying the captures folder to my external drive, Microsoft decided to save this as the new save directory. When I unplugged the external drive, poof. Everything broke... and I wouldn't realize this for another couple days.

So I unknowingly solved my problem when I went to backup my computer before resetting Windows. The moment I plugged in my external drive, everything worked just fine. So I simply looked up in my settings where Microsoft was saving videos/captures for the Game Bar and I cut/pasted the Captures folder back to my computer. Everything works as it should now.

A note to Microsoft...

Poor programming, come on. Not all people use fancy backup programs for their computers. I prefer to manually move files so the fact that this can even happen is frankly sad. I'd highly recommend adding a patch for this issue as I can't be the only one who had this problem.

bill smithers say:

Hi,

It is predetermined what gets backup.

You can exclude, but not add to the set list.

Read this information in bolded Tutorial link:

File History has a predefined set of items that will be backed up automatically: all your libraries (both default libraries and custom libraries you created), the Desktop, your Contacts, Internet Explorer and Edge favorites and, finally, OneDrive. The idea behind File History is that it allows you to retrieve earlier versions of a particular file. This means that even if you make changes to a document or any other type of file, you can always revert in time and rollback to any of its earlier versions.

Unfortunately, in Windows 8.1, you cannot set it to backup specific folders or libraries. You can only set it to exclude items from your backup, using the procedure detailed in the next section of this tutorial. So, if you want File History to backup a special folder in Windows 8.1, you have to create a new library for it or include it in an existing library.
