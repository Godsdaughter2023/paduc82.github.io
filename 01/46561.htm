5.   On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.

6.   Close Task Manager.

7.   On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

pc-learner say:

On both configurations, go into Device Manager, expand Disk drives, open the SSD's Properties window, then go to the Policies tab. Make sure the settings are the same under both configurations. Then go down and expand Storage controllers, open the Properties windows for the controller being used by the SSD and make sure the same driver is being used in both configurations. If they aren't, fix that if possible, or make sure both are using the standard Microsoft Windows driver for that OS.

Finally, if the SSD maker provides a utility program for firmware updates and optimization, like Samsung Magician, run it under both configurations and make sure the same settings and OS optimizations have been done on each.
