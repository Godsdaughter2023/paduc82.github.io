shenan stanley say:

If you format and leave that PC that you did all that on (the motherboard) with NO Windows OS at all and transfer the Windows 7 license to the new computer and then upgrade directly to Windows 10 (if that is your end-desire) - you will have a new license for Windows 10 tied to that RETAIL license of Windows 7 based off the hardware in that new machine (and that Windows 10 license cannot move if it is the FREE upgrade.)

This does not FREE up the Windows 7 license.  As long as Windows 10 (free upgrade) is running on that machine that you upgraded (even eventually) from that Windows 7 retail license - that Windows 7 retail license is tied up.  If you build another machine before July 29, 2016 and want to do it again - you'd remove the Windows OS from the current machine and install it on the new one and then upgrade to Windows 10 for free there and you'd have another license specific to that hardware.

However, the free Windows 10 license tied to the HARDWARE and the original FULL license you had.  (The Windows 8/8.1 license is tied to the Windows 7 license alone.  You could move it in a similar fashion (with the Windows 7 license), and it is not 'tied' to the hardware much like the Windows 7 retail license is not.)

  

So - you would really be transferring the Windows 7 license and getting the free upgrade to Windows 10 - leaving the original system with NO Windows OS at all.  You can easily skip the Windows 8/8.1 path to get to Windows 10.  You WILL have to perform the UPGRADE installation however, to make it work.
