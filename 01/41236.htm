  • Press the Windows key + C to access the Charms bar.
  • Select Settings
  • Select Control Panel
  • Select Troubleshooting. Under the System and Security heading, select Fix problems with Windows Update.

Suggestion 2:

Perform a clean boot and try Windows updates - http://support2.microsoft.com/kb/929135

  • Press the ‘Windows + R’ key on the keyboard.
  • In the ‘Run’ window type ’MSCONFIG’ and click ‘Ok’.
  • Click the ‘Boot’ tab and uncheck ‘Safe Boot’ option.
  • On the General tab, click to select the option ‘Selective startup’, and then click to clear the option Load startup items check box.
  • On the ‘Services tab’, click to select the ‘Hide all Microsoft services’ check box, and then click Disable all.
  • On the Startup tab click the link to Open Task Manager.
  • Select each application and click on the Disable button until all of the applications are disabled.
  • Close Task Manager
  • On the System Configuration, click OK, and then click Restart.

Note: After performing the troubleshooting steps in clean boot, follow these steps to return the computer to Normal startup mode.

  • Press the ‘Windows + R’ key on the keyboard.
  • In the ‘Run’ windows type ’MSCONFIG’ and click ‘Ok’.
  • On the General tab, click the Normal Startup option
  • On the Startup tab, select each application and enable each one until all of the applications are enabled.
  • Close Task Manager and click Apply and OK on the System Configuration window.
  • When you are prompted to restart the computer, click Restart.
