Hi Jack,

There are some factors to consider why BITS is behaving unexpectedly. Let's try to manage it before we restart services:

- Press Windows key + R.

- Type services.msc.

- Right Click on Background Intelligent Transfer Service and click Properties.

- Under Startup type, chose Automatic.

- Reboot your PC.

Check the startup type under BITS properties after rebooting and see if it returned to Automatic (Delayed start)

If it returned, go ahead and restart BITS:

- Press Windows key + R.

- Type services.msc.

- Right Click on Background Intelligent Transfer Service and click Restart.

Don't hesitate to update us if you need further assistance.

Best regards.

Method 1:

Since you are not able to add the items to the Windows Media Player Library  you can follow the below provided steps, because this issue could occur due to the corrupted Windows Media Player library database.
