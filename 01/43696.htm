thrashzone say:

As the first step, Make a tag on the tags and caption pane on the right side column, by selecting add a descriptive tag, type the name you want to use, left click a blank space to save that named tag, if you can't see the right side column, left click the view tab located above the toolbar and click on the tags and captions pane listing on the far right end until it appears,

Push and hold the (ctrl) key as you click on the photos and they will all turn blue, this highlights them as you click on them, try to do about 20 at a time or so, and then just click the add descriptive tag again and add that tag you made, you should only have type the first letter and it will list it, repeat to add more selections.

Once you get all of the photos you want in this new folder with a common tag associated to them,

*You can at this point click on the (home) tab located above the toolbar and select (new folder) listed next to the dark blue button, name it and click a blank spot to save it. And skip the my pictures and desktop parts.

*Photo gallery should still be open, left click on the (find) tab located above the toolbar, left click the (tags) listing in the toolbar, a drop down list of all of your tags will be listed, click on the one you made earlier,

All of the photos should be on this page you selected and tagged,

Left click one photo on that page, now hold the (ctrl) key and press the (A) key, this will select all of those photos on this page,

Left click and hold and drag to the folder listing on the file folders tree till it lists the new folders name and drop.

Dial up and LAN simultaneously - LAN Drops connection - Microsoft Community

rwaldron say:

Dial up and LAN simultaneously - LAN Drops connection

Hiya all,


I have windows 7 64 bit PC that monitors devices on our LAN.

We also use a dialup connection to dialup to the internet and send off errors via email.
The LAN doesn't need internet conenction.

(It's a long story but our customer doesn't want us to use the LAN to send emails,
they require us to use the dialup)


My problem is that when Dialup makes a connection our LAN Connection Drops ?

I need the LAN to stay active..

I have made some progress.

In windows 7 if untick the option "use default gateway on remote network" then The LAN no longer drops when dialup
connection is made. However dialup connection then still fails to send email.
If I tick this option again then dialup email does send correctly but LAN Drops

The option can be found
in windows 7 goto your dialup setting properites -> networking -> select TCP/IP settings (IPv4) -> click properties-> in general tab, goto advanced tcp/ip settings

Any Help Please ?

Ray..

 

rwaldron say:

Finally got it sorted.

For anyone interested. creating a static route on the LAN prevented the LAN from disconnecting when Dialup connected.

“use default gateway on remote network”  set to default (ticked)

the command to run is

route -p add 10.0.0.0 255.0.0.0 x.x.x.x metric 2

where
10.0.0.0 is the destination LAN
255.0.0.0 subnet mask
x.x.x.x Gateway of the LAN

in windows7 the command window needs to be run as administrator.

ie:start\run and type command but don't press enter
Right click on the Command prompt icon that windows displays and "run as administrator"


Hope this helps someone else.

Ray..
