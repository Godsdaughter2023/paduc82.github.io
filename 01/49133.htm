  1. Press Windows + X key on the keyboard.

As a initial troubleshooting, try to reset Microsoft Edge and check if the issue persists.

Please know that resetting the Microsoft edge will remove your bookmarks and History. Follow the instruction provided below and check.

a. Navigate to the location:

C:\Users\%username%\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe

b. Delete everything in this folder. 

c. Type Windows Powershell in search box.

d. Right click on Windows Powershell and select Run as administrator.

e. Copy and paste the following command.

Get-AppXPackage -AllUsers -Name Microsoft.MicrosoftEdge | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml" –Verbose}
