¡firedog say:

I can't think of any difference between editions of Windows 7 that would come into play here.

With any luck, you'll have a Previous version of your store folder with the account intact.

  1. In Windows Live Mail, press Ctrl-Shift-O for Options. On the Advanced tab, click Maintenance and then Store folder. Copy the path shown, paste it into the start search box and press Enter.
  2. Right-click on the active folder in the left-hand navigation pane and select Restore previous versions. Find one that just pre-dates the deletion, then Open it and locate the folder for the account you deleted. If you didn't change its name, the folder will be called Domain (username). probably truncated and with some random characters at the end of the folder name. 
  3. When you find it, copy the entire folder to your desktop. 
  4. Back in Windows Live Mail, use File > Import messages (Alt-F I) to import the folder on your desktop.
    When the import's finished, you'll find the imported folders and messages in a folder called Imported folders under Storage folders. You can copy messages from these folders back to the ones they came from under the account.
