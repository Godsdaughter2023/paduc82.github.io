As a workaround we can try to fix this issue by fixing your user profile.
 

Your user profile is a collection of settings that make the computer look and work the way you want it to. It contains your settings for desktop backgrounds, screen savers, pointer preferences, sound settings, and other features. User profiles ensure that your personal preferences are used whenever you log on to Windows. If you tried to log on to Windows and received an error message telling you that your user profile might be corrupted, you can try to repair it. You will need to create a new profile, and then copy the files from the existing profile to the new one.
 

Before you create a new profile
Occasionally, Windows might not read your user profile correctly—for example, if your antivirus software is scanning your computer while you try to log on. Before you create a new user profile, try restarting your computer and logging on with your user account again.
 

To create a new user profile
To create a new user profile, you must first create a new user account. When the account is created, a profile is also created.
 

The steps that you should follow will vary, depending on whether your computer is on a domain or a workgroup.
 

To copy files to the new user profile.
 

After you create the profile, you can copy the files from the existing profile. You must have at least three user accounts on the computer to complete these steps, including the new account you just created.
 

a. Log on as a user other than the new user you just created or the user that you want to copy files from.
b. Press Windows Key + C and now click on search and type computer, now click Computer. Double-click the hard disk drive that Windows is installed on (it's usually your C: drive), double-click Users, double-click the folder with the name of your account, and then double-click My Documents.
c. Click the Tools menu, and then click Folder Options.
If you don't see the Tools menu, press Alt.
d. Click the View tab, and then click Show hidden files, folders, and drives.
e. Clear the Hide protected operating system files check box, click Yes to confirm, and then click OK.
f. Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed on, and Old_Username is the name of the profile you want to copy files from.
g. Select all of the files and folders in this folder, except the following files:
• Ntuser.dat
• Ntuser.dat.log
• Ntuser.ini
h. Click the Edit menu, and then click Copy.
If you don't see the Edit menu, press Alt.
i. Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New_Username is the name of the new user profile you created.
j. Click the Edit menu, and then click Paste.
If you don't see the Edit menu, press Alt.
k. Log off, and then log back on as the new user.
 
Now try accessing the other user account and check.
 

If you have e mail messages in an e mail program, you must import your e mail messages and addresses to the new user profile before you delete the old profile. If everything is working properly, you can delete the old profile.
 

Hope the information helps.

Hi mstrblstr,

Let's try and clean out your Xbox without losing any data as best we can :-)

Delete your System Profile

Head over to Settings --> System --> Storage --> All Devices --> Gamer Profile
Select your profile, and select "Delete Profile Only". Do not delete items!

Delete your System Cache

Head over to Settings --> System --> Storage
Highlight any storage device and press "Y" on the controller
Select "Clear System Cache"
Repeat this 3 times to ensure the cache is 100% clear

Delete System Updates

Head over to Settings --> System --> Storage
Highlight your storage device, and press in order:
Left bumper, right bumper, X, Left bumper, right bumper, X.
It should prompt you to delete system updates. Select Yes

Restore your Profile

When you restart your console after deleting the system updates, download any updates your prompted for.
Then Press the Xbox Guide button on your controller, and go to "Download Profile" recover your profile.
Download any updates you are prompted for for games.

Once these are done, give it a try and see if you have any further problems. Thanks! :-)
