Should RAID 5 be used as a Windows OS drive? - Microsoft Community

a. user say:

Should RAID 5 be used as a Windows OS drive?

Should a 7200RPM SATA-6G RAID 5 be used as a Windows 8.1 and/or Windows 10 OS drive?  Are there disadvantages? Will there be problems with defrag or system images, etc?  At least one manufacturer is selling several systems with RAID OS drive options, which all sound good, but the more I read about it, the more confusing it becomes.   The other options are SSDs or conventional drives.  Thank you.
gadgetster say:

RAID5 is good on the capactity efficiency and also good on read performance, but very poor in re-writing performance.

The system volume is to have page files and temporary working files which are rather re-writing intensive. Therefore, RAID5 is not a good choice for the system volume in its nature. The archiving is the best application.

For the system volume, a mirror volume would be the best balance between the performance and availability.

I hope that this makes sense for you.

Have fun!

gadgetster say:

It would be correct to estimate the life of SSD shorter than HDD, but it is not due to the power.

Writing activity shorten the life of SSD.

An unexpected battery failure used to be one of the cause of a crash for HDD, so we would be able to say SSD is a better choice for such a power failure. However, I do not worry about it for recent HDD for a couple of these decades.

If you are considering a use for backup, it should be write intensive, and disk-capacity demanding.

Therefore, I recommend you HDD than SSD.

If you want to protect your system from an unexpected power failure,

be aware that RAID is effective only when we know which 1 specific drive has been failed.

When no actual falilure but the disk array went inconsistent, typically caused by a power failure during writing, the system may not be able to detect the error and data can be corrupted.

(Therefore, an enterprise-class RAID controllers have battery-protected buffer memory)

As a summary I would recommend you:

  • HDD for backup storage
  • UPS against an unexpected power failure
  • optionally SSD in mirror for your system volume

I hope these tips can help.

Good luck,
