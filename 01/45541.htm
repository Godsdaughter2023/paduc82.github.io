a. user say:

Does a "white screen" mean an empty page where the System Restore calendar of dates should be?

Sounds like a malicious software infection, but you may be able to get your System Restore to behave in spite of the potential infection.

If your System Restore page only show the "System Restore" title bar and/or your User Accounts settings page only shows the "User Accounts" title bar and the display where the information should be is an empty white box, and/or your Windows Search function just shows Rover (the animated search dog)...

System Restore is a frequent target for malicious software attacks.  If SR was working before and is now afflicted, run some scans and then resolve any remaining issues.

Keep in mind that System Restore is not a time machine.

It is likely that your system is now or has been afflicted with malicious software.

You can fix the symptoms now and scan for malicious software later, or you can scan for malicious software now and fix the leftovers later. 

I would not recommend just fixing the symptom of the empty page and forget to do the scans.  Fixing the symptom of the problem and fixing the problem are two different things.

Download, install, update and do a full scan with these free malware detection programs:

Malwarebytes (MBAM):  http://malwarebytes.org/
SUPERAntiSpyware: (SAS):  http://www.superantispyware.com/

They can be uninstalled later if desired.

When the scans run clean it may also be necessary to reregister some DLLs (or you can do this part first, but you still need to do the scans).

To just fix the immediate issue:

It sounds like your jscript.dll file has somehow become unregistered, so you can reregister it.

There are probably a couple other things that don't work that you don't know about yet, so fix them all at once.

Reregistering these DLLs is okay even if they do not need reregistering.  You can also reregister the DLLs before the scans which may fix the issue, but the scans are still recommended.

Click Start, Run and in the box enter the following commands one at a time.  Click the OK button to execute each command one at a time.

regsvr32  jscript.dll
regsvr32  vbscript.dll
regsvr32  /i mshtml.dll   (that's a slash eye)

The last command for mshtml will load the DLL but may also contain a harmless warning message depending on your environment.

Check your System Restore calendar and/or User Account settings page now.
