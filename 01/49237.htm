ken blake - microsoft mvp say:

On Sat, 13 Nov 2010 12:14:29 +0000, Shadowcu wrote:

One question before I wrap up: there are several arguments as to the RAM usage of 32 and 64 bit OS' es. Is it true that a 32 bit installation of Win 7 would not use the remaining 2 gigs of my RAm after the 4 ????

It won't even use the full 4GB.

All 32-bit client versions of Windows (not just Vista/XP/7) have a 4GB
address space (64-bit versions can use much more). That's the
theoretical upper limit beyond which you can not go.
But you can't use the entire  address space. Even though you have a
4GB address space, you can only use around 3.1GB of RAM. That's
because some of that space is used by hardware and is not available to
the operating system and applications. The amount you can use varies, depending on what hardware you have installed, but can
range from as little as 2GB to as much as 3.5GB. It's usually around
3.1GB.
Note that the hardware is using the address space, not the actual
RAM itself. If you have a greater amount of RAM, the rest of the RAM
goes unused because there is no address space to map it to.

Ken Blake (MS MVP)

rgharper911 say:

"AMD64" is a generic identifier for 64-bit code in Windows.  There is the AMD64 code branch which includes both AMD and Intel processors, and there is the IA64 code branch which includes only Intel Itanium 64-bit processors.

Intel processors use AMD64 branch code because they use features and microcode licensed from AMD.  Intel's own IA64 processors were never very popular and are for the most part abandoned.

Software Distribution folders are those containing the Windows Update installation files. These folders, along with the folder contents, may be deleted using Windows Explorer. However, these files may be saved to a Flash or external drive and used to update another system. Deleting the Software Distribution folders will not affect your computer’s functionality. When new Windows Updates are ready for download, a new Software Distribution folder will be created. Be sure to set Windows Explorer folder options to view system folders and files. Windows does not let you delete your software-distribution folder by simply right clicking it. You must use the command line to delete this folder.
