Why are you concerned about the 100MB System Reserved partition being created anyway?

 

About repairing the system.  You can get to the "System recovery options menu"* in Windows 7 by pressing F8 then selecting Repair your computer [only if you have the 100MB System Reserved partition installed], by booting from a "System repair disk"* that you create from the opening screen of Windows backup or by booting from a Windows 7 installation disk*.  The menu contains "Startup repair", "System restore", "System image recovery" & "Windows memory diagnostic" & "Command prompt"*.
