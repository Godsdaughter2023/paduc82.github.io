darleen pen say:

Hi Philip,

Windows 10 is capable of recording live streaming videos using the Game DVR tool available in the Xbox app. To set your proper expectations, for you to use the Game DVR tool, you'll need to download the Xbox application from the Windows Store. Follow these steps to be able to record stream videos in Windows 10:

  1. Open Windows Store.
  2. Using the Search box at the upper right corner of the Windows Store page, type Xbox and then hit Enter.
  3. Choose Xbox app from the search results.
  4. Install the Xbox app.
  5. Once the installation is finished, launch the Xbox app.
  6. Sign in using your Microsoft Account in the Xbox app.
  7. Press Windows key + G to pull up the Game bar.
  8. Check the checkbox on the pop-up message dialog.
  9. Click on the red circle to start recording. Take note that Game DVR will let you record videos with time duration of two hours.
  10. Once you are done recording, a pop-up notification will appear on the lower right part of your screen, click on it to view the recorded video.
  11. To save the video on other location, click the Open folder button.
  12. Right-click the video and choose the location where you want to save the file.

Should you need further assistance, don't hesitate to reply on this thread.
