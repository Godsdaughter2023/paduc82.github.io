Hi,

 

It is not possible to change a Skype username. Your Skype contacts should sync on your Skype app whenever you sign in to it. Most likely, you’re signed in to a different Skype account. Probably an account linked to your Microsoft email address.

 

It happens most of the time when Skype users are mistakenly signing in to Skype using the Microsoft account email address. After the migration of Skype to Microsoft, if you sign in to Skype using the Microsoft email address it will route you to a Microsoft Skype account.

 

Technically, this account is automatically created using a Microsoft account and has a Skype Name pattern like “live:SkypeName123”. A totally new account with no contacts and conversations on it.

 

Here’s what you’re going to do, Sign out the account that is currently logged on your Skype app. I strongly advise you to check the correct Skype Name that you’re using and sign in using the Skype Name and not the Microsoft email address. rohit siddegowda say:

Hi,

 

I am sorry that Windows 7 cannot, by default, read a disk formatted with Apple's HFS+ file system. Microsoft does not provide any drivers or other software that enables Windows 7 to read HFS+ disks. The freeware/open source/GPL software for the purpose is Apple's Bootcamp drivers.

 

Apple provides an application called “Bootcamp” to support such an installation, and Bootcamp includes drivers that enable Windows running on an Apple computer to read (but not write) an HFS+ formatted disk.
