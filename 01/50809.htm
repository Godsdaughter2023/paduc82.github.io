Because the Keyboard is integrated into the Laptop Case, something as simple as the slight vibration when typing can cause the touchpad to move the cursor or initiate a click, depending on the touchpad sensitivity settings.

 

You need to disable the touchpad when typing. Most modern laptops will provide an option or keyboard shortcut to accomplish this.

 

The following Keyboard shortcuts have been known to disable the touchpad on various systems. Press and hold the Fn key and then press either or F12 to disable the Touchpad.

 

 

1.    Press the Windows Key + X to display the system menu.

2.    Select Control Panel.

3.    Switch the "View by" from "Category to Large Icons or Small icons on the top right hand corner.

4.    Double click on "Mouse".

5.     Click on "Device Settings" tab (Tab may also be named 'Touchpad, Synaptics, etc).

6.    Click on "Settings."

 

Read the manual for your system specific instructions to disable the touchpad or contact the laptop manufacturers support.

Three posts to get all the information in.  Take a deep breath, relax. ;-)
 
"Dell Laptop (1525 or something)" <-- really?  You cannot take the time to figure out what model the laptop is?  *grin*
 
You "... took to someone who worked on computers and requested Xp for her."  <-- what does that mean?  Did you purchase Windows XP from them?
 
"It says its a bad VLC key" <-- You got taken and/or you knew what you did.  That is a Volume License Key (VLK, not VLC) - most likely not gotten in any legitimate fashion.
 
What you need now is to purchase a legitimate license for Windows XP if you plan on keeping Windows XP on the laptop.  Personally - I would think you better off (as would your daughter be) to purchase a WIndows 7 license and clean-install Windows 7.
 
You should not be concerned about 'keeping all the programs she has on it' as you should have the installation media for all of that and her data should be (or could be) backed up first.  Your main concern should be to get the illegitimate stuff off this system (all of it) if for no other reason than you don't know what all is there.  This "someone who worked on computers" could have installed whatever they needed to spy on everything you/your daughter did on this computer.
 
For your direct question - yes you can use the Microsoft Key Update utility to go from a VLK to a valid product key for whatever product you have installed - but you need a valid product key.
 
I don't believe this solves the actual problem - which is that you still do not know what all is installed on this system and/or you do not have proper backups/media for it.
 
My suggestion:
 
1) Backup everything important - the documents, emails, contact, bookmarks/favorites, music, pictures, etc.  If you/your daughter do not backup - that's just unwise.  (For now: manually copy the important stuff elsewhere.  For later: Get a Seagate Replica 500GB drive, plug it in and follow the directions.  First backup will take a while, after that things will not take as long.)
2) Collect together all installation media.  Microsoft Office if you have it installed, etc.  Make sure you have the product keys/serial numbers handy as well.
3) Purchase Windows 7 - Home Premium is probably fine.  32-bit will better ensure compatibility with more things.
4) Clean install Windows 7 onto the machine.
5) Install an AntiVirus software (Avira AntiVir is good for Free if you don't already have one.)
6) Update Windows 7.
7) Install the other applications.
8) Copy back the files you backed up in step 1 (documents, emails, contact, bookmarks/favorites, music, pictures, etc.)
 
Looks simple because it is.  You don't have to be a technical wizard to do any of it.  Patient, maybe.

If you get a legitimate copy/license of Windows XP Professional - you can perform a *repair* installation.  That will ask you - later in the installation - for your key.  This will keep all of your files/programs intact (although there will be some things to fix up afterwards.)
 
Before you do a repair installation - I receommend rolling back to Internet Explorer 6.0 (uninstall the updates through the Control Panel --> Add or Remove Programs) and making a complete backup of the system (make an image of the drive usig something like Symantec/Norton Ghost, Acronic TrueImage, etc.  That way you know you can go back to that if you have to (An image would be a bit-for-bit copy of the system.  You would need the application to do this and a drive (like an external USB) with enough space to put the image on.
 
Once your backups are in order - you would perform the repair installation with your Windows XP Professional (hopefully with SP3 integrated) CD...
http://michaelstevenstech.com/XPrepairinstall.htm
 
It is possible your TECHNET obtained license is not working because that is a license for a copy of Windows XP you plan on using for testing only.  Not for your own personal use. ;-)
 
http://technet.microsoft.com/en-us/subscriptions/ms772427.aspx
"Can I use evaluation software received in my TechNet Subscription on my devices in my home?
The TechNet Subscription license terms grant one user the right to install the program software on any devices, including those located at his or her home, but the user must fully comply with all the license terms no matter where the device is located.  In other words, one user may install and use the evaluation software, only to evaluate it. You may not use it in a live operating environment, a staging or production environment, or with data that has not been sufficiently backed up. You may not use the evaluation software for software development or in an application development environment."
 
But yes - it could be the blocked key might be preventing things all around.  You might try using the Microsoft Product Key Update Tool... http://www.microsoft.com/genuine/selfhelp/pkuinstructions.aspx
 
It might all mean - if that fails too - that you will need to locate an OEM licensed copy of Windows XP out there someplace and obtain it.  Not your TechNet Licensed copy.  ;-)
 
BTW - in the list of applications you listed...  Microsoft Office is the only one that you really listed of any 'value' and that would have a key.  Limewire and Bearshare are file sharing utilities - usually used to obtain music/videos that were not purchased.  Nero is a CD/DVD burning utility - there are free alternatives that most report as better.  RegCure should never be used, Spyhunter... blah, TuneUp - no use and Avast is free.  All of the others (except Office) are downloads too.  If you have Office - you have everything you will likely need.  The rest is free/easily downloaded.
 
So - I would still like to push the whole Windows 7 thing.  It'd be a clean install, with all of that stuff on it - the system obviously had its fair share of issues anyway.  But - if you can find a valid OEM/retail license key and/or the Product Key Update Tool works for you - if that's good for you, who am I to argue? ;-)
