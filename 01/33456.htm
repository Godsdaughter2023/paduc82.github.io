Hi Adam,

 

Welcome to the Microsoft Community forum, we’re here to help you with this issue.

 

The error ‘The drivers for this device are not installed. (Code 28)’ is displayed when the driver is not installed or not configured correctly. You have mentioned that you have already tried reinstalling the driver and yet the issue persists. In this case, I would suggest you to try the followings steps and check:

 

Method 1: Hardware and devices troubleshooter

 

First, I would suggest you to run the hardware and devices troubleshooter which can automatically detect and fix any incorrect settings or errors that could cause issues with the computer hardware. I would suggest you to run this troubleshooter to check if there is any issues with the network adapter. Refer to these steps to run this troubleshooter:

 

a)      Press Windows key +F to bring up the search Charms bar.

b)      Select settings in the drop down menu below search and type Troubleshooting.

c)       Click on Troubleshooting and select the view all option on the left pane.

d)      Click on Hardware and devices troubleshooter and follow the prompts.

 

Method 2:

 

If no issues were detected with the hardware and devices, then most probably it is driver issue. In this case, some other program might be conflicting with the network adapter driver. I would suggest you to reinstall the driver in a clean boot state and check. Refer to these steps:

http://www.samsung.com/us/function/search/espsearchResult.do?input_keyword=Samsung+QX411&keywords=Samsung+QX411

 

Now, boot the computer in a clean boot state to make sure that no other program is conflicting with the network adapter driver. Refer to this article:

 

http://support.microsoft.com/kb/929135

 

Once you have booted the computer in a clean boot state, follow these steps:

 

a)      Press Windows key + X, select Device manager.

b)      Expand the “Network adapters” section and locate the Network adapter.

d)      Click on the driver tab, click on the uninstall button to uninstall that driver.

e)      Restart the computer, windows will load the device with generic driver.

f)       Check if you face the issue with the generic driver.

g)      If yes, click on the update button in the driver tab and browse to open the driver downloaded from the manufacturer’s website.

h)      Now, right click on the device and select “Scan for hardware changes”.

i)        Now, check if the same error is displayed for the device.

 

Try these steps and let us know the results. We’d be happy to help if you need any further assistance.
