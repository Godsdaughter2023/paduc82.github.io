how to cancel "read only" on a file?

lemp say:

Is this problem with all Excel files or only one particular file?

Does the file not open or does it open as read-only?  These are two different behaviors.

You should be able to make any changes you want, even if the file is marked "read-only."  When you go to save it, you'll have to use Save as (in the "File" menu).  When you click OK on the pop-up warning about "read-only," the "Save as" dialog should automatically open.

If you want to remove the "read-only" attribute, go to wherever you've saved the file (e.g., in My Documents), right-click on the file, select "Properties" and then clear the "Read-only" box on the "General" tab.  Click OK.
