benjaminhash say:

This worked for the ICD-PX720 in Windows 10. Might work for you.

1. Install the Voice Editor from the original CD.

2. On that CD there is a Folder called Setup. Inside that folder is win2kdriver folder.

3. Go into that folder and right click Sony Recorder_driver. Go to properties.

4. Click the Compatibility Tab and set it for Windows XP (Service Pack 2)

5. Hit OK. Now double click the Sony Recorder_driver setup file. (it'll take a bit to install)

6. Go to you Device Manager and to your voice recorder.

7. Right Click and go to update/install Driver.

8. Click Browse Computer for driver. In the Field type C:

9. BAM! It works now!

If you don't have the original CD message me. We'll figure it out.

Might work if you download the update file from Sony and do the compatibility change to Windows XP (Service Pack 2) as well, haven't tried that.

0xc0:ff:ee say:

All the Skype for Linux-specific settings are stored in "~/.config/skypeforlinux". Deleting this directory should reset all the SfL-specific settings. However, some settings (e.g. who can send you messages, who can call you, chat appearance) are stored remotely so that they can be are nicely synchronised across your devices and, to the best of my knowledge, there's no way to reset these unless Microsoft provide a way for wiping all stored Skype data from their servers.

As for the login, your credentials are stored using Gnome Keyring and unless you delete them from there manually, Skype will automatically pick them up. You can change or delete them using Seahorse or other gnome-keyring management application. If you use Gnome, it should have one provided for you somewhere in the DE settings.

You can also double check if you have any other user-specific files stored somewhere else by running:

$ find ~/ -iname '*skype*'

Make sure you double check the list of files (if any) before you delete any as it will show everything that matches the given pattern.
