Hey AlexakaAJ,

It's a recovery partition brought about since the April 2018 Update (1803) that is intended to be hidden. You will need to remove the drive letter and mount point in DiskPart in order to hide it.

Hit Windows key + R together to go into Run. Type in the following: diskpart

select volume E
remove letter=E

Does not work after a reboot? Try entering this instead in an administrator session of Command Prompt: mountvol E: /D
