g)       Follow the on screen instructions to restore the computer to an earlier state when the computer was working fine.

Windows Media Player streaming audio sound hiccups - Microsoft Community

lmartist say:

Windows Media Player streaming audio sound hiccups

All my Windows Media Player streaming audio is having sound hiccups today. How can I fix it?
shinmila h say:

Hi lmartist,

 

Are you streaming the audio files over internet or over home network?

 

Since you mentioned that the problem started today, it will be a good idea to follow these suggestions that can point to the root cause of the problem.

 

1.    Restart the computer.

2.    Verify whether files stored on the computer behave the same way.

3.    If the problem is only with the audio files streamed from Windows Media Player, check the network connection. See that the connection is consistent.

4.    Configure Windows Media Player to buffer up to a minute of content before playback. While this requires extra time before streaming media begins to play, it should reduce, and in some cases for smaller length streams eliminate, re-buffering during stream playback.

 

Here’s how:

 

a)   Open Windows Media Player.

b)   In the "Library" view, click the "Organize" button, selecting "Options".

c)    Click the "Performance" tab.

d)   Underneath "Network buffering", click the "Buffer" radio button.

e)   Next to "Seconds of content", enter a number from 1-60 seconds.

f)    Click "OK" to close the dialog box.

 

See how it goes.

 
For more information on media streaming in Windows Media Player, see these articles.
 
Share your media in Windows Media Player with other people or devices
http://windows.microsoft.com/en-US/windows-vista/Share-your-media-in-Windows-Media-Player-with-other-people-or-devices

Sharing media on a network using Windows Media Player
http://windows.microsoft.com/en-US/windows-vista/Sharing-media-on-a-network-using-Windows-Media-Player
Regards,

Shinmila H - Microsoft Support
