buying and selling Microsoft COA s - Microsoft Community

jon1245 say:

buying and selling Microsoft COA s

is it legal to sell and buy Microsoft COA s for software from scrapped PC s, on places like EBAY ? 

and will Microsoft activate it if you do buy ?

thanks--john

bill smithers say:

is it legal to sell and buy Microsoft COA s for software from scrapped PC s, on places like EBAY ? 

and will Microsoft activate it if you do buy ?

thanks--john

Hi,

You do not tell us what type of License you have.

You can not resell Volume Licenses, which a lot of companies use..

If the computers are scrapped computers from Dell, HP, etc., with preinstalled operating system Licenses, NO. You can only sell them as the complete original package; as in the computer with the license/operating system still installed on it.

I suggest that you read this:

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

  7. Restart your computer and check it again.
