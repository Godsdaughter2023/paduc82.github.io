On Sat, 8 Jun 2013 18:10:23 +0000, Andre0111 wrote:

If you are talking about files,
    Delete removes the file entirely. It goes nowhere except to
    the recycle bin.

    Cut puts the file in the clipboard. It does that so you can
    then paste it elsewhere. Cut and paste is a way of moving
    a file to a different folder or drive.

If you are talking about something other than a file (for example a
word or sentence in a word processing document), it's almost the same.
The only difference is that Delete doesn't put it in the recycle bin
and cut and paste is used to move the word or sentence either to a
different place in the document or to another document.

In both cases (files or words) use Delete to get rid of something; use
Cut as the first step in moving it elsew

considerate_guy say:

The latest version of Movie Maker is version 2012 (build 16.4.3528.0331) and is part of Windows Essentials 2012.

None of the versions of Movie Maker (since Windows ME/XP) have had the ability to overlay a logo. If you need to do that, you might want to explore Microsoft Expression Encoder 4. It has an ability to do a video or photo overlay. There are tutorials on YouTube that community members have created that should explain the steps.

As for the font for the captions, that's already a feature and you just choose a different font name in the dropdown box.

Finally, regarding your query given in your title, you would like to remove the black bars on your photos. To fix this, you need to crop your photos to match the aspect ratio of your project. The aspect ratio of your project is either 4:3 (full screen) or 16:9 (wide screen). You can use Windows Photo Gallery to crop photos to exactly this aspect ratio and then you'll not have any black bars. But if you have a photo that doesn't fit in that exact size rectangle, you'll have black on the top or sides. This is particularly true of portrait (vertical) photos that don't perfect fit in the horizontal rectangle of a video project.
