Saving Videos on Windows Movie Maker 2.6 - Microsoft Community

sarahnull say:

Saving Videos on Windows Movie Maker 2.6

Saving Videos on Windows Movie Maker 2.6 I have Windows Movie Maker 2.6 I want to know about saving more the one video at a time. What I mean is if like I spilt in a video into two videos and I want to save them separately but not wanting to take the time and doing undo to delete a video to save the undo do the delete and save the other video. Like also if I title something lets say Untitled as the video name. Can I save more then one video and I will save like Untitled.1 then Untitled.2 and so on.

considerate_guy say:

None of the versions of Movie Maker let you save all the parts of your timeline/storyboard as individual movie files.

But there is a workaround that will work in any of the versions.

Step 1 - Make a backup copy of your project file (in 2.6 it has an extension of .mswmm)

Step 2 - Working in a copy of the project, delete all but the first clip.

Step 3 - Save that as a movie with a name like "Clip 1"

Step 4 - Reload the project and delete all but the second clip.

Step 5 - Save that as a movie with a name like "Clip 2"

Repeat the last two steps as many times as necessary.

P.S. Since you are using WMM 2.6, you might want to switch to using WMM 6.0. It still reads the same project files as WMM 2.6 and has the same traditional timeline, but it uses the GPU for rendering rather than the CPU. It also has a 64-bit version which can take advantage of modern hardware. It has slightly better compatibility with video file types too. Read: Installing Movie Maker 6.0 on Windows 7, 8, 8.1 or 10
