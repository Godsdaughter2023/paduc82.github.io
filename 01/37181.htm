    cmdrkeene say:
    What drive letter is your File History drive stored on?

    hef2k say: The File History drive is an external USB drive, and it comes up as Drive E.

    cmdrkeene say:

    Thanks, I want to make sure I keep the drives straight.

    I think you may have more success using the old Control Panel instead of the Settings app to configure this (the Windows 7 style Control Panel).

    Also as a little background, the File History uses the Libraries as a search index to find files to backup.  And Libraries (and also in turn, File History) do require that the location you want to include be in the search indexer's scope of places to index (it uses the index to determine when a file has changed since the previous copy was made).

    So I'd like to suggest creating a library that includes folders on your D: drive.  

    1. Open File Explorer
    2. In the left navigation panel, right-click any blank space and choose "Show Libraries" (if it's not already visible)
    3. Right-click the Libraries heading and choose "New Library" and give it a name like "SD Card" (that name doesn't really matter)
    4. Right-click that newly created library and choose Properties.
    5. Add the SD card to the list of locations.

    Or alternatively instead of the above, add the SD card to the existing Pictures library's search scope (perform steps 4 and 5 on the existing Pictures library).
