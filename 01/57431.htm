I've only had the monitor 2 days - just a few minutes ago I discovered that setting your monitor to DP1.2 doesn't automatically enable 60Hz refresh, you need to set it under DISPLAY SETTINGS -> Advanced Display Settings -> Display adapter properties, then under the "Monitor" Tab, select 60 Hz under the Screen Refresh Rate drop down list box and Apply.  What a difference, the cursor motion is smooth as butter now.

Note: You have to be using a DP 1.2 cable to enable 60 Hz refresh at 3840 x 2160 resolution.  I'd recommend turning down the monitors "Smart Response" setting to None or "Fast" at the highest, the "Faster" and "Fastest" settings seem to cause windows to blur when dragging them around.
