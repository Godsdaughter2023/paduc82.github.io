james vinyard say:

Okay, you are talking about the one that is not assigned to any account.  However, that lock screen is configured by the primary administrator account.

Since you cannot change it in a normal fashion, do a search for that image file name (complete with extension) and it should be within the Windows folder on Drive C:.

If you locate it, write down the location of that image and then copy the image you want to it but make sure you use the exact same name (also same JPG format).

Windows can be particular when making those types of changes and the image has to be the exact same resolution and screen aspect ratio.  If you deviate from that, the system can come up with just a black screen and no background image.

ken blake say:

On Tue, 10 Aug 2010 21:00:05 +0000, bodybylaw wrote:

I'm selling a laptop with Window XP Professional on it. I'm passing the OS disc on to the buyer with the sale. It's registered in my name. Can I/Do I unregister the software so the buyer can then register it in his name? Thanks

You don't need to do anything.

Bear in mind that registration (for which you need to use your name)
is completely optional, and does nothing for you. It's used by
Microsoft for marketing purposes. You didn't need to do it and he
doesn't need to do it.

What is required is activation, not registration. And unlike
registration, activation is completely anonymous and has no name
attached to it. You activate the computer, not the owner. Registration
registers the owner.

But there is no such thing as "unregistration" or "unactivation." As I
said, nothing needs to be done.
