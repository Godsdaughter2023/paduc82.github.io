    It depends on the game pad in question. While the XBOX controller has become the default game pad on

    many games over the years, it will not work on games that don't specifically support it.

    This is because XBOX controllers are limited to using Xinput.

    However, if you use a Logitech (eg- Logitech F310) or some of the other brand game pads that have profiling

    software from the mfg of the game pad, and which can use both Xinput and the older Dinput  you can make

    any game  work on the game pad by creating a profile for the game by assigning keystrokes

    to the buttons on the game pad and using the Dinput option (a switch on the controller that selects either

    X or D ).

    - Game pads that only use Dinput will also work on games that default to XBOX because you can still create

    a profile using the keyboard commands.
