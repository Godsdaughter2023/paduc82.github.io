If you receive Windows Update error 8024000b, it means that Windows Update can't read a file (Update Manifest) that is required for installation.

 

To fix this problem, follow these steps:

 

Press Windows key + R, to open Run box. Copy the following text, and then paste it into the Run text box:

 

Cmd /c ren %systemroot%\System32\Spupdsvc.exe Spupdsvc.old

 

Click OK. Run Windows Update again.
