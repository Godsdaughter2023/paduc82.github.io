How do I get my data to go to the D drive instead of filling up the C - Microsoft Community

pablo_tzx say:

How do I get my data to go to the D drive instead of filling up the C drive?

As I transfer my data (music and photos) to my  new computer they seem to move to the C (windows) drive by default rather than the D (drive). Consequently my C drive is full. What is the best way to make sure the appropriate files go to D? I understand that the hard drive is partitioned with one drive designated for optimizing the OS and the other is for data storage.  I am assuming that the C drive should contain only the windows folders and the rest should go to the D. Is that correct?
cmdrkeene say:

A recommended practice is to install programs files (executable code) on the C drive, but store all your data on the D drive.

Example 1:  Microsoft Word program is installed on C: but all your document files (Word docs) are on D.

or

Example 2:  Your favorite movie player program installed on C, but all your video files on D.

Over time, the C drive might accumulate some junk that isn't important.  Temporary copies of installers, updates, and more.  To clean that up, run the built-in Disk Cleanup program:

  1. Press  and type cleanup
  2. Choose the Disk Cleanup option that appears in the search results
  3. After you use it to clean up files, run it again.  This time, choose "clean up system files" to get the rest.  You can't do both at once, so be sure to run this program both ways to get a full cleanup.

--

In addition, you can set your folders to store on the other drive instead.  Consider doing this:

  1. Create folders on the D drive for "Documents", "Pictures", "Music", and such.
  2. Go to your C:\Users\Pablo\ folder.  Right-click your Music folder, choose Properties.  On the location tab, set it to store your music under D:\Music instead, then click OK.
  3. Repeat with your other folders to store them on the Data drive instead.

You can do this with almost any folder (like OneDrive, Desktop, and more).

I too am experimenting with switching to Edge after their latest update, and ran into this problem.

Just thought I'd point out that the answers so far are unsatisfactory, and in some cases miss the point of the original question.

Of course you can click the X at the upper right to close the window (as you can do in any other browser) -- the problem is, that doesn't constitute ending a full browsing session all at once, rather it only closes the tabs in that window.

So you have to do it separately for each window, and then upon restarting Edge, it only restores the last window you'd closed.

If you right click Edge in the taskbar and select Close all windows, same result, it will thereafter only restore the last window that it happens to close (presumably the first in the list).

At the moment, the only apparent way to exit Edge in such a way that multiple windows and their tabs will be restored upon next launching it, is to kill the process (from task mgr or cmd line, etc.).

That is obviously a far from ideal part of a regular workflow, to put it mildly.

Working in, exiting, and restoring a session with multiple windows and tabs grouped within them is a standard way of working in Chrome, FF etc. -- I'm pretty disappointed that Edge is missing something so simple as a general Exit menu option to allow this to function in a typical fashion. (No, the new save/restore tabs feature, nice enough for what it is, is not a substitute.)
