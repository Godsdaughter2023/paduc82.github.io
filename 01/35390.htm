The word "download" means to copy a file (perhaps a program file, but
not necessarily) from a server someplace on the internet to your own
computer. Please do not mix up that word with "install," which means
to load a program on your hard drive in a form from which it can be
executed

You typically install a program from a CD, from a diskette, or from
the internet. Only in the last case, the internet, is there any
downloading involved, but even there, it's a two-step process: first
you download it from the internet, then you install it.

you post the entire message verbatim, not just a description of it.

Hi mario savard,

• Is the computer on domain?

This issue may occur due to insufficient permissions to access the Drive Letter:\Users\All Users\Application Data\Microsoft\Crypto\RSA\Machine Keys folder on the computer.

To resolve the issue set the correct permissions for the Machine Keys folder.

1.Right-click on the folder (Machine keys) and go toProperties. Then click on the Security tab.

2. Now click on the Advanced button at the bottom and click on the Owner tab.

3. Click on the Edit button, click Other users and groups and type in your user account name and click Check Names.

4. Click OK and on the Owner tab make sure to check the “Replace owner on subcontainers and objects” box.

5. Now select your user account from the list on the Security tab and click  Edit. Click Allow next to Full Control and click OK.
