max_zq say:

Update:

Resetting faulty disk did help to re-attach fixed disk.

I'll remind - I've removed 1 disk from RAID5 array, fixed it (hopefully), but could not re-attach it back.

Necessary steps to re-attach:

1) Run CMD as Admin

2) run "powershell" command

3) In the powershell get list of your disks with names/ids:

Get-PhysicalDisk | ft FriendlyName, SerialNumber, UniqueId -auto

You'll get a list like this:

PS C:\Users\Admin> Get-PhysicalDisk | ft FriendlyName, SerialNumber, UniqueId -auto

FriendlyName              SerialNumber         UniqueId
------------              ------------         --------
Samsung SSD 960 EVO 250GB    0025_3851_71B0_F458. eui.0025385171B0F458
ST2000LM015-2E8174                 WDZ24LXS             {90dbac25-f5dd-ea00-8e51-4ca8414aa21f}
ST2000LM015-2E8174                 WDZ27TPY             {b7f23aa9-b6bb-c5b0-3ab7-0719def510e8}
ST2000LM015-2E8174                 WDZ28NJK             {b9bea270-c3b6-0f9c-2759-3d6d0631e40b}
ST2000LM015-2E8174                 WDZ27943             {e0996a91-1e6a-f88d-c339-83963c369e1d}

4) Reset problematic disk by FriendlyName or by UniqueId (only one command is needed as they do the same):

Reset-PhysicalDisk -UniqueId "90dbac25-f5dd-ea00-8e51-4ca8414aa21f"

In my case FriendlyName was the same for all disks, so I could not reset it by FriendlyName (it would break the RAID Storage), but you can rename disks in Storage Spaces one by one (e.g. "ST2000LM015-2E8174" to "Disk1", "Disk2"...), and then run:

Reset-PhysicalDisk -FriendlyName "Disk1"

5) Re-attach the disk to the Storage Space.

Hi NICOLEHEART

Thank you for using Microsoft Answers Forums!

You can follow these steps to use your iTunes in the Windows Movie Maker.

By default, iTunes rips CDs to the AAC format, which is a great format for storing music, but is also incompatible with Windows Movie Maker. There are two ways to resolve your problem:

1) Convert the AAC file to MP3 with iTunes

By default, iTunes rips all CDs to AAC format. You can use iTunes to convert AAC files to MP3, but you need to make a few changes in iTunes first. To convert an AAC file to MP3 with iTunes follow these steps:

1.     Open iTunes Preferences from Edit > Preferences. Click the Advanced tab and choose the Importing sub-tab. On the Import Using dropdown list, choose MP3 Encoder and click OK.

2.     Then add the column called KIND to the list you can arrange your music. (song, artist, genre, etc right click and add kind)

3.     Click this column and it should arrange your music by format. Any songs you imported from CD’s using iTunes will be AAC. Any songs you purchased through iTunes will be protected AAC. Any songs that iTunes converted from windows media player will also be AAC.

4.     Highlight all the AAC songs (not protected) and right click and then click Convert to mp3.

5.     For the protected songs you will have to burn them to a CD and then re-import them onto your computer. Since you changed the default from AAC to mp3 they will go onto your computer now in the mp3 format.

6.     Once you have done that you will have duplicates of all the songs you converted in both aac and mp3. You can keep them both or just delete the AAC ones.

7.     The files will go into your default iTunes folder which should be in the C drive under My Documents\My Music\iTunes\iTunes Music.

8.     Since these songs are now mp3 format you can use them in windows movie maker.

9.     When you want to add a song to windows movie maker just browse your computer and go to that folder location C:\ My Documents\My Music\iTunes\iTunes Music. All the songs should be there for you to pick that are in the mp3 format.
