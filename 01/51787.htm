mouneshwar r say:

Hi iHate Apple,

 

Could you please provide the boot device order set in the BIOS (Basic Input Output System)?

 

This issue happens if your computer is using the BIOS Setup Boot Order and not the Network Boot Protocol. PXE (Preboot Execution Environment) is a Network Boot Protocol. In the BIOS, boot device order should be followed after PXE. This makes PXE to get executed when you start your computer.

 

Note: Modifying BIOS/ chipset incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/chipset can be solved. Modifications of the settings are at your own risk.
