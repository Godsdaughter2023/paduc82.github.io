cwl-inc say: lscott35020,

Yes, if you are using VMware and made a Ghost image of your drive and then restored that Ghost image inside of your VM - it will work. Well, it can work - the drivers are always a sticky point - so give it a try and let Windows boot up an redetect drivers, etc.  I have done this in the past with some success on Windows 95 and Windows 98.

With regards to your dual-boot, copy the currently in-use drive with Windows 95 (so you have a drive you can mess with) and then used that copied drive to install Windows XP on the same partition as Windows 95. As long as you're using FAT32 on the drive, you'll see dual boot come together. If not, then you can try something different. Here's a little more detail from Microsoft on the subject.

Regardless, I would say the first thing you want to do is get the ghost image of the drive before it fails, at least then you will have options.
