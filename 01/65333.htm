bobbymi say:

Hello LD,

 

If your system fails to enter hibernation, a [Hibernate file Cleaner] option will be available under Disk Cleanup.  Going to Disk Cleanup and cleaning the hibernation file will basically reset the hibernation file. To do this, follow these steps:

 

1.       Click Start and then click Computer

2.       Click your system hard disk (Usually C:/) then click Properties

3.       Click Disk Cleanup

4.       Click Hibernate file Cleaner

5.       Click OK

6.       Restart your computer

 

If you don’t have a [Hibernate file Cleaner] option under Disk Cleanup, and you want to disable Hibernation, follow these steps:

 

1.       Click Start and type cmd in the Search box

2.       Right-click on cmd.exe and click Run as administrator

3.       Type the following command  and press enter:

powercfg.exe -h off

4.       Type Exit and press Enter

5.       Reboot your computer

If you want to turn Hibernation back on, follow the steps above but type the following command:

powercfg.exe -h on

 

Hope this helps.
