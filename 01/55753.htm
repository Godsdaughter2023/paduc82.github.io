Hi Emanate Presence,

The background color feature will only be available for Titles and Credits. The background color will show up on the whole screen, and not on the text part.

As for the most readable font type, you can use standard fonts like Arial or Verdana. Avoid using cursive or stylized fonts.

Also, you can also add Outlines to your caption so it can be readable on any part of the video. You can do this by following these steps:

1. Add the caption on your project, and edit it.
2. Set the Font color to White.
3. On the right-most part of the Text tools, click Outline Size, and select your preferred size.
4. Click Outline Color, then select the color as Black.
5. Edit the other font options as per your preference.

This would make your caption white with black borders around it. You can also try experimenting with different contrasting colors.

    Saved passwords and defaults no longer being saved - Microsoft Community

    sandiparker say:

    Saved passwords and defaults no longer being saved

    I keep changing default files to open with a certain program, and all of a sudden, every time I shut down the computer, the next time I boot up, it has reverted back to all the old default programs. Also all my saved passwords are gone. Why is this happening?

    cmdrkeene say:
    This can happen if your profile has become damaged, or can also arise if there is a virus infection or file corruption.  In short, it's very hard to say right now exactly what may have caused it.  However, the first thing I would do is press WindowsLogo+R to show the Run box, run the command RSTRUI and then click OK.  This will let you revert back the system data files and registry entries to before they became corrupted.  It won't hurt your own files, and you can always undo it if you don't like the results so this is by far the safest easiest option to try first.
