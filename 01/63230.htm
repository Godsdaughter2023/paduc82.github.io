captain_wd say: Hey there DaShek :)

WD Elements is an external storage drive with a HDD, not a SSD. The drive doesn't feature hardware encryption (unlike WD My Book and My Passport devices) so any encryption done to it must have been through a software. Moreover, if you have swapped the HDD inside the enclosure with a SSD that has voided any warranty on the WD Elements drive.
Are you referring to the WD Elements Portable or the WD Elements Desktop drive?

I would run WD Data Lifeguard Diagnostic tool on the drive and see if it passes both the quick and the extended tests in order to verify that the drive is at full health. 

This particular drive does not have hardware nor software encryption unless you have used an application that has encrypted your data.

andre da costa say:
It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

Learn more about transferring your license: 
http://www.groovypost.com/howto/transfer-windows-10-license-new-pc/

If its a retail license, learn how to reactivate:
http://www.groovypost.com/howto/activate-windows-10-license-microsoft-support/
