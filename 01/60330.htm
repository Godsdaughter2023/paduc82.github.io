Run the command (folder path is an example only, modify the folder path as needed)...

netsh wlan export profile folder=c:\users\al\documents\misc\wifi key=clear

...as an administrator. The resulting profile .xml files will be saved in the c:\users\al\documents\misc\wifi folder.

In order to run the netsh command as an admin user you need to go to the Start button and type cmd in the search window.

Right click on cmd.exe and select Run as administrator.

At the command line type your netsh command string...

Save the resulting .xml files on a flash drive or CD or DVD or whatever medium you choose.
