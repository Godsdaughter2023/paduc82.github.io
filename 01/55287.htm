New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

 

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee.

 

For HP, press F11 directly after powering up the unit

geethu b say:

Hi,

Thank you for contacting Microsoft Community. I understand that you are unable to install and uninstall programs and have an issue with Windows Installer.

To understand the issue better, I would need more information from your end.

  1. Do you get any error messages while installing or uninstalling programs?
  2. What is the knowledge Base article numbers of the Windows Updates that were installed recently? 

To find Knowledge Base Article number of the Windows Updates, view your Windows Update history:

1.  Click on Start and click on Settings.

2.  Open Update and Recovery.  

3.  Click on Windows Update, and then click on Advanced Options.

4.  Click View your update history.     

This issue could have occurred due to these reasons:

1. Windows Installer service is not started. 

2. Windows Installer is corrupt.

Method 1:

I suggest you to make sure if the windows installer service is started and automatic.

a. Press Windows key + R and type "services.msc" in the run box and press Enter.

b. In services Window check for Windows installer service status.

c. If it is stopped, click on the drop down menu and click on automatic and click on start.

d. Check the status of the issue.

Method 2:
If the issue persists, I would suggest you to unregister and re-register the Windows Installer.

a) Click Windows Key + R on the keyboard.
b) In the "Open" box, type "msiexec /unreg", and then press ENTER.
c) Click Windows Key + R on the keyboard.

d) In the "Open" box, type "msiexec /regserver", and then press ENTER.

Check if the issue is resolved or not.

Hope this information helps. Reply to the post with updated status of the issue so that we can assist you further.
