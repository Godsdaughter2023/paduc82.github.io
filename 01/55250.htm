a. user say:

One problem appears to be that your system is not running Service Pack 1 so why is that?

Have you reinstalled Windows lately and not installed SP1 or any Windows updates?

I would not spend much time (zero time) troubleshooting a system that is not up to date but here are some other things you can think about regarding system crashes and hangs:

When Windows starts it checks to see if the system was shut down gracefully the last time it was running.

If Windows was not shutdown gracefully the last time it was running an Event 41 will be logged:

The system has rebooted without cleanly shutting down first. This error could be caused if the system stopped responding, crashed, or lost power unexpectedly.

The problem is there is usually nothing in the event data that tells you what happened so you are not going to get much information from that event (unless the system experienced a Blue Screen of Death (BSOD)) and even then you are probably not going to be able to diagnose the problem by looking at any Event 41.

So an event 41 is going to inform you that the system restarted ungracefully but is probably not going to tell you

In your example the BugcheckCode and other Parameters are all zero so in that case the system did not crash with a BSOD but may have locked up (or appeared to be) instead and you pressed the reset button, cycled the power to get going again or you had a power failure.

The next time the system locks up see if the Caps Lock key will toggle the caps lock indicator on your system.  If that doesn't work, the system is in a hard hang.  If it does work (it toggles) Windows is still running so something else is wrong and you may still be able to get into Task Manager (Ctrl-Alt-Del) to see what is going on.

If your system is crashing you need to adjust the system to generate a crash dump file when it does crash so you will have something to look at after the fact (and you will still see an Event 41).

Here's how to do that:

On your Desktop right click Computer, Properties, Advanced system settings and make it look like this:

There is no need to create the Minidump folder as Window will create one if it needs to.

Then click OK enough times to close all the dialog boxes and restart your system for the changes to take effect.  After rebooting you can go back and check the settings again to be sure the adjustments "stick".
