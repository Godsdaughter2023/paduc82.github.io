cmdrkeene say:
You should be able to do this, provided your E drive is formatted as NTFS.  Also, any user with Administrator access will always be able to override your security and access the folder, so be sure that all users are standard users.

Right-click the folder you want to restrict, and choose Properties.  On the Security tab, click Advanced, then click Change Permissions.  You'll need to take these steps:

1 - Turn off (uncheck) the Include Inheritable Permissions from this object's parent option.

2 - Remove all users listed in the Permissions entries box.

3 - Add back only the single user that should have access.  You'll be prompted to choose their access.  Give them allow access to the following items only:
  • Traverse folder/execute file
  • List folder / read data
  • Read attributes
  • Read extended attributes
  • Create files/write data
  • Create folders/append data
  • Write attributes
  • Write extended attributes
  • Delete subfolders and files
  • Read permissions (optional)

Most importantly, you are not going to allow changing permissions, taking ownership, or deleting the folder itself.

4 - After you add the user, click OK.  Before closing the Advanced Security options, make sure the Owner tab says that you (or Administrators) are the owner of the folder.  Click OK to close all dialogs.

Repeat those steps from the other Operating System.


Special Notes:
  • I would recommend against setting  the Deny options, they aren't necessary to accomplish what you need, and doing so can introduce some disgustingly painful issues -- such as preventing you from changing settings later even from your account/admin account.


  • If you (the admin, owner) ever need to change your computer password, only do so from the Ctrl-Alt-Delete screen.  If you do it from the Control Panel, it renders your security settings obsolete.  The folder will still be secure, but you will no longer be able to access it because the owner is now a non-existent user.

You have to boot from the recovery disc 1.

 

Put it in the optical drive and restart the PC. As the PC starts up do one of two things...

 

1. Look for a prompt to invoke the boot device menu (normally by means of an F-key). Hit that F-Key and choose to boot from the optical drive with the disc in it.

 

2. Look for a prompt to invoke BIOS setup (normally DEL or F2 but might be something else). Once in BIOS, change the boot device order and promote the optical drive to be the first boot device. Exit BIOS saving changes and the PC should boot from recovery disc 1.
