The first way to install Windows 7 on SSD is to use a Windows 7 installing disk. It's just the same as installing it on your common hard drive except one thing that you need to go into your PC's BIOS and change the hard drive setting.

Note: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk. 

Before you are going to install Windows 7 on your computer, please go into your PC's BIOS then change the hard drive setting of it. Computers are typically set up to use Serial Advanced Technology Attachment to attach hard drives. But with SSD, you need to change SATA to Advanced HostController Interface. After the setting, you can install Windows 7 on SSD as usual now.

wunders say: Corrupted profiles are not uncommon and are not too difficult to recover from.

If this is a recent development, you may be able to recover by performing a System Restore operation:

" Windows XP Problems if Your Profile is Damaged"
  < http://support.microsoft.com/kb/326688 >

If this doesn't work, one of the following should get you going again:

"How to recover the damaged user profile in Windows XP"
  < http://support.microsoft.com/kb/555473 >
""Windows cannot load your profile because it may be corrupted" error message when you try to log on to Windows XP"
  < http://support.microsoft.com/kb/318011 >

Almost always, a corrupted profile is accompanied by file system errors.  It would be prudent to perform a Disk Check (chkdsk) as well with the /F or /R option (or with both options checked if using the GUI version) either before or after you recover.  In some cases, a disk check will fix the problem -- although it can take a couple of hours to perform.

"How to perform disk error checking in Windows XP"
  < http://support.microsoft.com/kb/315265 >

Whatever else you do, do not save or move any important files in your temporary profile because they will be deleted (along with your profile) when you log out.
