cmdrkeene say:

Yeah it unfortunately sounds like it's gone.  But since you have Office 365, you can actually just login to http://portal.office.com and download a new copy of Office for free, so that's the good news.

For the similiar names, here's what they mean:

  1. System Restore:  makes periodic backup copies of key system/registry files (usually daily plus when new things are installed), and lets you restore back a set of those backups. It's like a "global undo" and is useful to undo recent changes made to the computer. Unless the restore was performed in safe mode, you can also "undo the restore", make it a very safe and versatile option with little risk. It doesn't touch personal files (it only cares about key specific system files).  System Restore is available in Windows Me/XP onward, but is not turned on by default on all computers (to save drive space and such), so it's best to make sure it's on when you setup a new computer so it's there when you need it later.


  2. System Refresh:  makes a copy of user folder files (My Documents/Pictures/Music/etc.), then erases the drive back to factory defaults, and then places the copied user files back into place.  Though this shouldn't risk losing files, users should always have a backup before performing any operation on the computer.  After a refresh, files and settings remain, but any apps that didn't come factory pre-installed must be installed again. Refresh is available in Windows 8.x only (but is still possible in Windows 10, under a new name described below).


  3. System Reset:  same as (2), except it doesn't make a copy of user files. It just flat-out erases the drive back to factory conditions. As above, any apps that didn't come factory pre-installed must be installed again.  This is useful if you're going to sell or give away your computer and want to make it "just like new."  Reset is available in Windows 8 onward.

In Windows 10, the option for #2 Refresh was removed, but the "Reset+Keep My Files" does the same thing (basically they changed how the options are presented, but the actual capabilities didn't change).
