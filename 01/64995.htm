Codecs: frequently asked questions

COM surrogate error mostly occurs because of codecs and other COM components installed by various softwares that aren’t fully Windows 7-compatible.

Since the issue is mostly related to third party software, check and see if you can find out the culprit application.

To do so, put your system in clean boot state and restart the computer. See if the error shows up.

Here’s how:

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7
