Resolution:
Check Dynamic Host Configuration Host Protocol (DHCP) settings on the router.
Check each IPv4 address on each PC connected to your network.

Follow the methods.


Method 1

1. Click Start Menu and in the Instant Search type “Command Prompt” then
right-click and left-click Run as administrator.
2. Type the first command ipconfig /release and press Enter.
3. Type then the second command ipconfig /renew and press Enter.


Explanation:
The fist command will send a DHCPRELEASE message to the DHCP server to release the current
DHCP configuration and discard the IP address configuration for either all adapters, the second
command Renews DHCP configuration for all adapters.
