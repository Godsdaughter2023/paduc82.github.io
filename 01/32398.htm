noel d paton say:

What type of WIndows 7 license do you have?

What type of Windows 8 license are you proposing to get?

 

If you purchase an Upgrade license for Windows 8, it subsumes the license for Windows 7 - so you can't use the Windows 7 license elsewhere, but you could still reinstall it in the same machine so long as you uninstall Windows 8.

 

If you purchase a Full license for Windows 8 AND your copy of Windows 7 is a Full Retail license, then you can use the Widows 7 license elsewhere, or in a Dual Boot scenario.

 

If you purchase a Full OR Upgrade Windows 8 license and your Windows 7 is an OEM license, then the Windows 7 cannot be moved to another machine under any circumstances - but could still be used to reinstall the original machine should you wish to remove Windows 8.

jeeped say:
Depending upon the video format, the original video file has already received compression to some degree. You are essentially ZIPping a ZIPped file.

If it is mission-critical to get the video file smaller, you can a) lower the pixel resolution and/or frames/second, b) lower the colors used in the video's color palette or c) change the video format to one with a higher, more efficient compression algorithm. In regard to this latter choice, MP4 is typically more compressed than other formats such as AVI.
