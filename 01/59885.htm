A Windows revocation and renewal message may appear when using Microsoft Media Player to play a protected file. Microsoft Media Player can play many types of files, including those that have DRM protection stored in the file. When there is a problem with the security components of the media player, a revocation and renewal message will appear. You can correct this problem by upgrading a security component for the Media Player or configuring registry key on your system

Hi Andrzej,

If the Hotkeys on your Microsoft Designer Keyboard don't work, it is possible that the Human Interface Device Access service on your Surface was stopped. Make sure that the service is set to automatic. To do that, follow these steps:

  1. Type Services in the search box. Click on Services
  2. In the list of services in the Details pane, double-click HID Human Interface Service, and then make sure that the Startup type is set to Automatic.

If the Hotkeys still won't work, try to create a new local user account and test if the "special keys" will work. If you're a Windows 10 user, refer to this link on how to create a local user.  

You may also test the keyboard with a different computer. If the keyboard does not work correctly with the other computer, this could be a possible hardware issue.

Let us know if you need further assistance.
