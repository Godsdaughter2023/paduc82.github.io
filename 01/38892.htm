Set your Internet connection as metered:

You can create a stop and start script for Windows Update.

Create a Notepad file with each of the following and save them with the .bat extension.

stop-updates.bat:

sc config wuauserv start= disabled
net stop wuauserv

sc config bits start= disabled
net stop bits

sc config dosvc start= disabled
net stop dosvc

pause

start-updates.bat:

sc config wuauserv start= auto
net start wuauserv

sc config bits start= auto
net start bits

sc config dosvc start= auto
net start dosvc

pause

Now you can simply right click on stop-updates.bat and select Run as Administrator

a. user say:

Hi, I don't pretend to be any kind of bigshot.

The idea of Active Hours is that your computer won't restart to install an update during Active Hours. It's the restart that disrupts your work. Downloading the bits in the backgroud is managed so as not the interfere with your other uses of internet bandwidth.

Updates aren't the only thing that gets downloaded in the background. Your applications can also download in the background, particularly antivirus definitions. Your computer can also upload in the background, to cloud storage, for example. All this down- and uploading in the background doesn't affect your computer's performance. It can affect your internet speed, which is why it's managed so as not use much (or any) bandwidth while you're using it for other things.
