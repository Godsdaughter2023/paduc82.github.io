vlad zaharia say:

No, that's not true at all. We're talking about two completely different things, and you have your terminology completely off. There is no such thing as a "Games for Windows account" or a "Xbox account". It's one account, a Microsoft Account (previously Windows Live ID). All your services, Xbox, Games for Windows/Xbox for Windows, Windows Phone Store Account, Windows Store Account, Hotmail, etc. They're all in one single account. Yes, you have an Xbox Gamertag, but even that is shared across your different gaming services, and is tied to your Microsoft Account. When you login to your Xbox, you're logging in to a Windows Live ID, not an "Xbox account"

This all traces back to the old Zune Players. Because of the tight integration in billing information between Zune and Xbox back then, it caused a single billing account for your entire MS Account. If you go to billing.microsoft.com, you will see you have a Personal billing account, which will have your Xbox transactions (both Live, and content purchases), as well as any apps that you bought (either free or paid) in the Windows or Windows Phone stores. This account has a single billing region, which all services that do billing use. So Xbox, Windows Store, Windows Phone Store, all of them. They all use this billing information, which is tied to your MS Account. That's why, when you connect to the Windows Store, you will only see items in the region associated with your account. Same with Xbox content, or Windows Phone content.

This billing region is essentially created when you first use a billing service. So if you used Xbox first, and chose the Canada region, then when you sign into Windows Store, you'll also be on the Canada region. If the region isn't available in the service, then you're denied access (ie. Latvia not being an Xbox Live region. It could be your MS Billing region, but you won't be able to use Xbox Live services with that account).

Because they're all linked together, changing the region on one service will change it on all your services. Until recently, you haven't been able to do that. However, Xbox Support can now change the region on the MS Account, therefore changing your Xbox's Billing Region, as well as the regions on the Windows and Windows Phone store.

TL;DR: All billing regions are linked together through a single source of billing information, associated to a Microsoft Account. Changing it one one changes it on all of them.

Programs usually have the option to install on the drive location you want. They all vary but are often in a Customise or Advanced option during install. Changing the setting for Apps only affects apps installed from the Store.

Having said that some programs will only install to the system drive, it depends what they are and you cannot change them.

If you want to change your personal folder location you should right click the folder in File Explorer (say Documents for instance) select Properties and then the Location tab and Move the folder. Repeat for other folders.

Perhaps if you reference a program that you want to have installed on the HDD but installed to the SSD by default it might help. There may be some program writer that don't include the option but that's down to them and not Windows 10.

Programs that are installed cannot be moved to another drive. They would have to be uninstalled using Apps and Features and then installed again making sure they are installed to the HDD.
