I suggest you to run hardware troubleshooter to diagnose the exact issue. It will automatically detect the problem and fix it.

  1.  Press Windows + X keys on the keyboard and select Control Panel.

  2. Type Troubleshooting in the search bar on the Control Panel.

  3. Click on Small icon on the left navigation pane. 

  4. Click on Troubleshooting and click on Hardware and Device. 

  5. Click on Advanced on the Hardware troubleshooter dialog box and click on Run as administrator (Enter the credentials if required).

  6. Click on Next and follow onscreen instruction complete the Hardware troubleshooter. 

If the issue still persist, try the below method.

 

Method 2.

Try uninstalling and reinstalling generic driver and check.

  1. Press Windows + R and type devmgmt.msc and hit the Enter.

  2. Go into Device Manager, click and expand Bluetooth.

  3. Right click on the device and click on uninstall.

  4. Once uninstall is done, restart the computer. It will automatically install generic display driver.

If the issue still persist, try installing the driver in compatibility mode.

  1. Right-click on the setup file of the driver and select Properties.

  2. Select Compatibility tab.

  3. Check Run this program in Compatibility mode.

  4. Let the driver install and then check the functionality.

Method 3.

I also would recommend you to update the keyboard driver and check if the calculator pops up by pressing calculator shortcut key.

  1. Press Windows Key + X and click on Device Manager.

  2. Locate for the Keyboard, click on keyboard to expand.

  3. Right click on the keyboard and click on updates.
