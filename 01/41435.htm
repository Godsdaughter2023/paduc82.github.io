Welcome to Microsoft Community. I will certainly help you with the issue that you are facing. Ideally when an administrator account is deleted, there is not much that can be done apart from reinstalling operating system by booting from installation media. However, we can try to enable system default administrator account in this scenario and check if it helps.

To enable default administrator account, follow the steps below:

  1. Open Start page, and then type cmd.
  2. In the search results list, right-click Command Prompt, and then click Run as Administrator.
  3. When you are prompted by User Account Control, click Continue.
  4. At the command prompt, type net user administrator /active:yes and then press ENTER.
  5. Type exit and then press ENTER.
  6. Log off the current user account.

(Note: If you are prompted to type in the password while opening administrator command prompt, try the password for the old user account)

Once you log off, you will see the option to log in to Administrator account. Log in to the administrator account and follow the article below to add a new administrator user.
