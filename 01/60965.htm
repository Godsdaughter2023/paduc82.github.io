alexseliv say:

In my case there was not Chrome browser available in the Startup tab,

also reinstalling Chrome didn't help me, as the link started opening in another browser, in Edge.

 

I had to delete one entry from Windows registry (please do it very carefully, as it influences Windows).

 

1. Go to Start > Run (type "Run" after pressing start, or just without pressing on Start press Windows logo button on the keyboard plus R button).

2. Type "regedit" > press OK > press Yes

3. Go to HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Run

4. In my case there was odd registry entry:

Name = <my PC user name> (e.g. johnb)

Data = "C:\explorer.exe xx-abc.info" (I don't remember exactly the link, and likely you will have another link)

 

This registry entry told Windows to run link xx-abc.info on startup.

 

I've deleted the entry, but if you don't want to do it, you can just modify the entry by deleting the "xx-abc.info".

I assume, in such case Windows will open My Computer on startup.

 

Hope it helps.
