aljon_t say:

Hi

Question: Someone asked my what my skype id is. I can't seem to discover what my skype id is. What is it? 

Answer: The Skype ID or Skype name is the username created or can be system generated when first joined Skype, other than the email address or phone number. For a system generated Skype ID by email address, it starts with "live:" as a prefix, then followed by the word before "@". On the other hand, for account created using Phone number, after the "live:", it follows by random alphanumeric characters.

You may check this link for more information. What's my Skype Name?

Hope this information helps! Feel free to post back if you need further assistance.
