How to change account configuration from POP3 to IMAP - Microsoft Community

rodtempleton say:

How to change account configuration from POP3 to IMAP

I cannot change the configuration of my incoming mail server type from POP3 to IMAP.  How do I delete the POP3 and replace it with IMAP?--I can't seem to change this field.

* Moved from Windows Vista | Mail

[Original subject: Windows Live Mail]

¡firedog say:

In Windows Live Mail (and, I suspect, Windows Mail too), you can't change the type of account once it's been set up. Once a POP3 account, always a POP3 account. 

In order to start using IMAP instead of POP3, you have to set up a new account from scratch. First, disable the POP3 account by deselecting Include this account ... on the General tab of the account's properties dialogue. Then use the Add an email account wizard to set up a new IMAP account; be sure to select Manually configure server settings on the first page of the wizard, then enter the server settings as advised by your email service provider.

If all goes well, the account folders will be downloaded from the IMAP server and eventually be filled with the same messages as their server equivalents have. Once this process is complete, give the program plenty of time to index the new account before doing anything else.

Once all has settled down, you can drag messages from the POP3 account into folders under the IMAP account. This will be helpful for drafts and sent messages, which will in due course be uploaded to the IMAP server.

When you're sure that the new account is working properly, and that there are no messages you want to keep left in the POP3 folders, you can remove the POP3 account. Removing it will permanently delete its folders and their contents.

Should you have further questions about this IMAP account in Windows Live Mail, please start a new thread in the Windows Live Mail forum.
