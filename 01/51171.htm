¡firedog say:

There are three types of mail account in Windows Live Mail: IMAP, POP3 and HTTP. Right-click on an account name and select Properties. An IMAP account's properties dialogue has five tabs, a POP3 account four and an HTTP account three. HTTP is only used for outlook.com accounts using DeltaSync.

 

 

burce999 say: How about Exchange, STP...etc? what they are in group of e mail account type? 
if not what diferance of Exchange, STP, POP3..etc? 
Regards,

¡firedog say:

Exchange usually refers to accounts hosted at enterprise level by Exchange Server. You would normally need to use either Exchange Client or Microsoft Outlook to access a private account like that. It's up to the server's administrator whether an Exchange account can also be accessed via POP3 or IMAP.

I don't know of a mail protocol or account type named STP. SMTP is the name of the protocol normally used for sending messages.

POP3 is a method of downloading messages from a mailbox. This will normally be the inbox of an account. IMAP varies in its implementation; some clients will show you the contents of folders on the IMAP server, while others will also download them so you can read them offline.

Windows 10 came preinstalled on my computer, how do I find the product key?

Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

The product key will be revealed, copy the product key then enter it

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support
