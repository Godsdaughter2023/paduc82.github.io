balaji kundalam say:

The system reserved partition is a small partition in the hard drive that contains the boot configuration data and BitLocker data. Usually, backing up this partition is not needed as the data is completely related to the operating system and that no user

Since the partition contains important boot files, converting the filesystem is not possible when logged onto Windows. You can try converting the filesystem via installation media, however, it's not recommended as it will cause boot issues.
