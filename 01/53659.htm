rajesh govind say:

 

Hi,

 

Are you looking for the feature in Windows RT by which you can record the screen and output a video file?

 

If yes, there are no built-in feature or App in Windows RT which can record the screen and output a video file. But, we do have a feature called “Problem Steps Recorder” in Windows RT which will capture the screen shots and all the clicks that the user make on the screen and outputs to an MHTML file. This also includes a text description of where you clicked and a picture of the screen during each click (called a screen shot). However, we do have multiple third party apps available in the Windows Store which can record the screen and then can output to a video file.

 

To search for the screen recording apps in Windows Store, you may follow these steps:

 

a)       Tap on Windows Store App icon from the Start Screen to open it.

b)       Swipe in from the right edge of the screen and then tap on “Search” option from the Charms bar.

c)       Type “Screen recording” in the search box and search for an app which can record the screen and output to a video file.

 

You may also refer to the below given thread link:

 

http://answers.microsoft.com/en-us/surface/forum/surfwinrt-surfgetstart/how-does-one-do-a-screen-capture-from-a-ms-surface/0e18a04a-ffcc-4c62-9c07-1ff668a8f591
