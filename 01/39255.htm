Thank you for posting your question in Microsoft Community. We are glad to help you.

 

"BootUserProfileProcessingTime” is user profile specific and is proportional to the number of applications installed in the computer. This should be the primary reason that your computer takes more time to boot in normal mode than starting the computer in clean boot.

 

To check if the issue is specific to user account, I would suggest you to create another account and then compare the “BootUserProfileProcessingTime”. This will give us a clear picture and we will be in a better position to access and evaluate the performance of your computer.
