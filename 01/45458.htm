Any one can help me?????

rohit siddegowda say:

Hi Yashwanth,

 

This issue might occur if the adapter is in sleep mode.

 

Method 1:

First, I would suggest you to check if you have configured the settings properly:

Step 1:

a.     Press Windows key + R on the keyboard.

b.    Type “ncpa.cpl” without quotes and press Enter.

c.     When your network connections open right-click on your Wireless network adapter and select Properties from the context menu.

d.    Under the Sharing tab, check the option “Allow other network users to connect through this computer’s internet connection”.

e.    Click on OK.

 

Step 2:

Set up the wireless network:

a.     Swipe in from the right edge of the screen, and then tap Search.

(If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, and then click Search.)

b.    Enter “CMD” without quotes in the search box, tap or click Apps

c.     Touch and hold the Command prompt tile in the search results. Then drag your finger down until the tile is outlined by a solid line. Once it happens release your finger from the screen.

d.    Tap or Click Run as administrator.

e.    Type the following command and press Enter:
netsh wlan set hostednetwork mode=allow ssid=”name of your network” key=”Password”

f.      Then type the following command to start broadcasting:
netsh wlan start hostednetwork

 

Method 2:

If you are still getting the error, follow these steps and check if it helps:

a.     Press Windows key + R on the keyboard.

b.    Type “devmgmt.msc” without quotes and press Enter.

c.     Under System, select and expand the Network adapters from the list of devices.

d.    Find the wireless network adapter and right-click on it and select Properties.

e.    Select the Power Management tab.

f.      Under the Power Management tab, make sure the following option is checked (enabled): “Allow the computer to turn off this device to save power".

 

Hope it helps. If you have any further issues on the computer, please post your question regarding Windows and we will be happy to help you.
