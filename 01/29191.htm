    1.    Do you get any error message when you are trying to create a playlist or rip the CD?

    2.    Was the Windows Media Player working fine before?

     

    Information that is stored in the database or the database itself can become corrupted. This can lead to Media Player finding problems creating or saving playlist.

     

    Ripping music lets you put tracks from your audio CDs onto your PC. When you rip the tracks, they become digital files on your PC.

    There are two places in the Player Library where you can customize your settings and make ripping work better for you:

     

    ·         In the Rip settings menu (with fewer customization choices).

    ·         In the Options dialog box (with more customization choices).

     

    Method 1: Run the Windows Media Player library and Windows Media Player Settings troubleshooter and check if Ripping is possible.

     

    Step 1:

     

    a: Press Windows logo + W to open the search box.
    b: Type troubleshooting, hit enter.

    c: Now click Programs.

    d: Then click Windows Media Player  Library.

     

    Step 2:

     

    a: Press Windows logo + W to open the search box.
    b: Type troubleshooting, hit enter.

    c: Now click Programs.

    d: Then click Windows Media Settings.

     

    Method 2: Perform steps from the link mentioned below and see if it helps.

     

    http://windows.microsoft.com/en-us/windows-8/change-settings-for-ripping-music

     

    Method 3:

     

    Step 1:

     

    1.     Exit Windows Media Player.

    2.    Press Windows key + R, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

     

    3.     Select all the files in the folder, and then click Delete on the File menu.

    Note     You do not have to delete the folders that are in this folder.

    4.     Restart Windows Media Player.

    Note Windows Media Player automatically rebuilds the database.

    5.    Run Windows Media Player and verify.

     

    Step 2:

     

    If Step 1 does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:

     

    .     Exit Windows Media Player.

     

    2.      Press Windows key + R, type %LOCALAPPDATA%\Microsoft, and then click OK.

    3.     Select the Media Player folder, and then click Delete on the File menu.

    4.     Restart Windows Media Player.


    Note Windows Media Player automatically rebuilds the database.

     

    This should fix the issue.

    So basically, how can I lock my orientation to horizontal while still using the keyboard attachment (if I want to lay on my side or something)?

    I have seen this (the keyboard going unresponsive when it is attached and the Surface is rotated into Portrait) but in my case it is locked in Portrait so I was guessing that that was what was doing it.   So, if you can't use your keyboard but you want to rotate your screen you can try using the Mobility Center's Rotate button.  Open the Mobility Center (Win-x b) before it is needed if necessary.  Then when needed switch to it (e.g. swipe into Task View or tap on the Taskbar button)  and push the Rotate button until your screen is in the orientation you want it.  Does that unlock your keyboard?

    david darroch say:

    I think I've got this figured out.  Seems to have been a problem with corrupted system files.

    I opened the command prompt as administrator and ran SFC /scannow.  Took a while and the log showed a whole host of fixes were made.  Restarted the system and Windows identified the problem devices and installed the drivers fairly easily.

    So if you have a similar problem try typing "SFC /scannow" (without the ") in the command prompt window, then re-boot your system
