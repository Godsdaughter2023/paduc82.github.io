An "image" is a snapshot of your hard disk. In includes everything: documents, pictures, music, programs, system files. This is why it takes such a long time to run. Since you're giving the old machine away, it is totally useless to you. In your case I would use this simple solution:

  5. Use the new machine for a week or two before giving the old machine away.

I have no problem with regularly updating Windows, but being a developer I much prefer the Windows 7 method, where a notification is displayed when updates are available, and I am given the choice of what, and when to install. Apart from taking up bandwidth, and write cycles on the hard drive, not all Windows updates are stable, and sometimes have security issues. Not only that, one update can occasionally interfere with other software... especially if I'm intentionally working using older software, and Windows Update decides that it's going to automatically keep switching to newer drivers etc. With this in mind, I'm actually incredibly disappointed that Microsoft has decided to take this more aggressive stance on updates.

Is there any work around to choose, or even just to see which updates are being installed?

Update:

So I've found out that apparently Microsoft backtracked on this a little, by providing a tool that allows you to block specific Windows updates (can be downloaded from https://support.microsoft.com/en-us/kb/3073930). If the update causes a security vulnerability, the chances are you'll only have blocked it after the damage has been done.

Compared to Windows 7 where you could sit back and let updates pile up, you don't even have the time to read up on any problematic drivers etc. I honestly don't understand why Microsoft insists on forcing the automatic updates...
