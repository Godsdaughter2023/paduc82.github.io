¡firedog say:

How can I configure windows live mail to check for new messages every one or two minutes?               

That is probably not a very good idea. I see you found out how to do it, but you should be aware that it could easily take more than a minute or two to complete a poll for new messages. In that situation, the program will try to start a new session and fail because the previous session is still open, so you'll just see an error and you won't get any new mail. Some mail servers apply a lock at the beginning of each session which isn't released until some time later, in some cases 30 minutes. Hotmail used to recommend a polling interval of at least 20 minutes.

If everyone polled for new messages every one or two minutes, the servers would soon collapse under the pressure. Set your polling interval to 10 minutes and see how it goes.
