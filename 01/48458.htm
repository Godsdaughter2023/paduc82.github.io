mzig say:

Subsequent Windows updates have broken the process I outlined above.   After several weeks of functioning properly,  my start menu tiles for webpage shortcuts reverted to the default Chrome icon once again after subsequent updates were installed.   After searching the web once more, found another solution that, when combined with the original technique, seems to be working (when using Chrome as your Browser), at least for now:

  1. Open Chrome and go to the webpage for which you want to create a Start Menu tile.   Drag the upper left corner of the url onto your desktop to create a shortcut.  This is the shortcut you will be using for your Start Menu.
  2. To get Chrome to create a folder with the appropriate icon, create another shortcut using the Chrome menu:  From the Chrome customization menu on the same webpage (the 3 vertical dots in upper right hand corner), click on “More Tools” then “Add to Desktop” to create a second shortcut.   This step will create a folder with the correct icon in the following location:  %UserProfile%\AppData\Local\Google\Chrome\User Data\Default\Web Applications\%shortcutfolder%.    The name of the %shortcutfolder%  may be a long string of letters with nothing identifiable to the shortcut.  Just open the most recently created folder and you should see the icon for the shortcut you just created.  Copy the path to this folder - you will need it later. 
  3. You should now have two shortcuts on your desktop.  The first one (from step 1 above) will probably have the default Chrome icon and the second  (from step 2 above) will have the correct icon.  Delete the second shortcut even though it looks right.  If you try to pin this one to the Start Menu, it will NOT display properly as a tile!  Even though everything looks good and the icon is pointing to the right file, it will not work after you pin it to the Start Menu tile.  
  4. Take the first shortcut you created in step 1 (with the incorrect icon) and drag it (or copy) to:  %UserProfile%\AppData\Roaming\Microsoft\Windows\Start Menu\Programs
  5. Right click on the shortcut you just copied and go to “Properties”.  Click on “Change Icon” and then “Browse”.  Browse to the folder Chrome created for you in step 2 and click on the appropriate icon file:   %UserProfile%\AppData\Local\Google\Chrome\User Data\Default\Web Applications\%shortcutFolder%.   Click on “Apply” or “OK”.
  6. You should now see your new shortcut listed under your Start Menu.  Right click on it and pin it to the Start Menu.  Your tile should now display with the correct icon.

If the issue is resolved by placing the machine in “clean boot” then we have some sort of third party application and or service running that is possibly interfering.
