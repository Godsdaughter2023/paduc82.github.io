leonlaude say:

Hi Tenayie,

This error code indicates that the boot configuration is either broken or corrupted.

I suggest that you download the Windows 10 ISO from a working computer and create a Media Creation Tool. 

To download and learn how to create the tool click on the link below
http://www.microsoft.com/en-us/software-download/windows10   

Once you have the installation media, follow the steps: 

  1. Start your computer from the installation media. Insert the disk or USB stick and restart your computer. If you see a message asking you to Press any key to boot from DVD, do so. If you don’t see the message, you might need to change the boot order in your computer's BIOS settings so that it first starts from the disk or USB. 

  2. When you see the Install Windows page, tap or click Repair your computer link to start the Windows Recovery Environment.

  3. Here, on the Choose an option screen, click Troubleshoot > Advance Options > Command Prompt.

  4. In  this window, type the commands below

bootrec /scanos 

bootrec /rebuildbcd

bootrec /fixmbr

bootrec /fixboot

Once the command runs successfully, reboot the computer and check if the issue persist.
