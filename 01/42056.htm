Visual C++

a. user say:

I have 9 versions, but I have not installed or reinstalled XP on this current friendly system in over 6 years.

If you only go up to 2008, then you are behind, but you may not be using applications that require the latest C++ runtime redistributable files (and that is fine).

It is important to understand that it doesn't mean you have 6 versions of C++ development environment installed or anything to do with C++ programming.

It does mean you have 6 flavors of the C++ run time distribution files installed.  Mostly it is just runtime libraries and stuff that you don't need to worry about - it takes care of itself using Windows Side by Side Technology (which I will be happy to explain if you are interested).

You may or may not need them all and it depends on what third party programs you have installed and what they need. 

Some applications may use old versions of the C++ redistributables, some will use the latest versions.  I cannot think of a good way to figure out which ones you might not need, but they don't consume any resources until the third party program pulls the files in (like DLLs to use them), they use what they need while they are running and that is that.

If you were to start uninstalling them from the bottom up starting with the oldest, pretty soon you would likely run into some application that is going to complain about missing runtime libraries and/or DLL files that are missing.   Then the Microsoft Support Engineers will suggest you run sfc /scannow which will not help you at all.

Since most systems are different in what they have installed and need, there is really no way to give you any good advice about what to uninstall (unless some application is complaining).

To begin with, you will need to shop carefully and look for an active converter Ii.e. powered either by USB or AC) and steer clear of just cables with the proper connections. Most (if not all) if those cables are meant to go INTO HDMI and not the converse - in other words they don't work both ways. This mostly applies to component and VGA which are analog HD, DVI may not have this same issue as it is still digital and doesn't need converting. Make sure you research them properly before buying. Most streaming services won't be an issue (like Amazon), but I have heard some of NetFlix content has HDCP and cannot be watched if you use a setup like this.

  • Type “osk” and hit enter to launch the On-screen keyboard and check if this helps.

You may want to open a notepad and type using that key to check if that key works fine.

For a quick cleaning, turn the keyboard upside down and use a can of compressed air to blow out any foreign matter. Be sure to do this in a location where falling debris can be cleaned up easily. Turn the keyboard upside down and tap on it a few times. You should see a bit of dirt fall out. Change the angle and tap harder to make most of it get out.

Clean the sides of the keys with a clean cotton swab dipped in isopropyl alcohol.
