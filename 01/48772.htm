considerate_guy say:

First, video stabilization is only available if you are running Movie Maker 2012 on Windows 8 or 8.1.  You can confirm your version by going to File | About Movie Maker.   

The basic steps of editing are:

  1. Add photos or videos to your project area.  This is done from the Home tab by clicking Add videos or photos.
  2. Add music to your project area.  This is also done from the Home tab by clicking Add music.
  3. Double-click your video in the project area to the right.  This will take you to the Video Tools | Edit tab.  Choose your settings for Video Stabilization.  You may also want to reduce the volume on the video clip(s), if you just want
  4. Save your project so you could come back to make more edits.  This is done from the first File tab then choosing Save project.
  5. Save your movie in the final desired format and resolution.  Again, go to the File tab, but choose Save movie.  You can hover over each option and see an estimated size per minute of video that should help you decide on settings that will make a smaller file size.

criter say:

It's a quick fix. Your original msnmsn.inf was delted or corrupted. The easiest solve is to replace it.

Go to this url http://www.infdump.com/download-inf-files.php/inffiles/M/MSNMSN.INF/download.html

Download MSNMSN.INF. Save to your desktop, or where ever  you can get to it.

Copy it onto the clipboard.

Navigate Windows Explorer (Windows Start key +e) to C:\Windows\INF

You will know you are in the right spot because all thats in this folder are *.inf and *.pnf files.

Paste in the MSNMSN.INF file, right click on it and choose install.

Now you can access Windows Components in ADD/REMOVE Programs.
