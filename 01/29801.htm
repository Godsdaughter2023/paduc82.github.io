    battle predator say:

    Working Solution To Obtain an Xbox One Live Open Nat

    This is specific instructions to obtain an Xbox Live open NAT. The instructions work well if your internet provider issues you an IPv4 IP address. Unknown if this will work for IPv6 addresses. Please search for a video on youtube or lookup a tutorial on how to setup a static IP for your Xbox One.

    1. Setup a static IP address for your Xbox One

    2. Important. You need to port forward port 3544 to destination port 3074 to your Xbox One's IP address. Do not port forward port 3544 to port 3544.

    3. Forget about all the other ports Microsoft indicated needed to be port forwarded for Xbox Live at the moment.

    4. Check to make sure your router does not do source port randomization. If it does, set up your router to allow all Xbox One outbound traffic to use static ports,

    5. Unless you've made modifications to your consumer router's rules set to restrict outbound traffic then the above will work.

    The information I've provided above is specifically to obtain an Xbox One Live open NAT. To obtain an open NAT for specific games you may need to forward specific ports. For instance Black Ops III requires you to forward port 3075 to destination port 3075 to obtain and open NAT. I do not recommend using UPnP or DMZ as it could potentially cause security issues with your network. Let me know if this helps you out.
