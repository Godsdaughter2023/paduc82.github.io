Follow these steps to try and resolve the issue.

1. Open a File Explorer window.

3. Go to C:\Users\(User Name)\AppData\Local

Hi,

 

Welcome to the Microsoft Community. As per the problem description wisptis.exe is making the hardware work incorrectly.

 

I will help you with this issue. Wisptis.exe is a product installed by Microsoft Office or packaged with Windows updates. It is a high memory consumption program that automatically launches at computer start-up. It deals with "Windows ink Services" and tends to work together with Adobe Acrobat Reader. It is not a malicious tool, but can be very annoying because it runs anytime even when it is not being used. Remove this nonessential program quickly and easily without affecting system processes and programs. I would suggest you to follow these steps:

 

a)    Press Ctrl, Alt, and Delete together to open the Windows Task Manager. Click Processes.

b)    Scroll through and search for Wisptis.exe. Click it and select End Process. If it is not a currently running program, move to the next step.

c)    Click the Windows Start button and select Run. Type in "%WinDir%" and press Enter to open the Windows folder.

d)    Double-click System32 and locate Wisptis.exe. Right-click it, and select Delete. Empty the recycle bin.

 

Hope this helps. Let us know if you need further assistance with windows.
