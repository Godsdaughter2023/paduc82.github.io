The following are the reasons that cause this error:

  • Device infected with a virus.
  • The allocated virtual memory needs increase.
  • Programs opened in the background are consuming too much memory.

Let's start troubleshooting your concern by following these methods to correct each case:

Method 1: Run a full scan on your device by running your antivirus.

Method 2: Open your Task Manager by right-clicking your taskbar and select Task Manager.  Click the Details tab to see all running programs, then click Memory (private working set) to sort by memory usage. Right-click a program, then select End process that uses too much memory.

Method 3: Check your system using Windows Troubleshooter. Type in Troubleshooting in Cortana search and press Enter. Click View All and select System Maintenance in the list. Run the troubleshooter.


Method 4: Manually change or increase your Virtual Memory. Right-click the Start menu and select System.
