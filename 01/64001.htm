hula_baloo say:

 Did you run your antivirus program in Safe Mode?  Some viruses can hide in normal mode, so it's really necessary to run a full system scan in Safe Mode with a fully up to date virus checker.  Get Defender up to date and run that in Safe Mode too.  If that doesn't help, try System Restore back to prior to the start of the problem.  I'd also run CHKDSK to make sure you don't have a bad sector on your hard drive to cause this problem.

 

  Forum moderator Keith has some suggestions at

http://answers.microsoft.com/en-us/windows/forum/windows_7-performance/windows-explorer-has-stopped-working/6ab02526-5071-4dcc-895f-d90202bad8b3

 

 Regarding your question about copying the file, if your laptop is at the same maintenance level as your main system, you should be able to copy that file, but that may not fix the problem since other files could be deleted or corrupted as well.

 

 If you're behind a router, you're protected by a hardware firewall.  If not you sure want to get some kind of firewall protection as soon as you can to keep hackers out of your machines.

 

  I hope this helps. Good luck.
