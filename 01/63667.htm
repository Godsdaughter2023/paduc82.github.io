saving a movie maker project - Microsoft Community

ctd3 say:

saving a movie maker project

I want to save a movie maker project in a mp4 format not a wlmp which seems to be automatic...please help!

considerate_guy say:

There are two ways to save your work from Movie Maker. The first is as a project. You do this with File > Save project.

This saves the current editing state of your project where it remembers all your individual media files used in the project along with all the trim points, effects and transitions, titles, etc. But the project is not a playable media file. It's more like the save state in a video game allowing you to go back to where you were the last time you were editing.

If you try to close Movie Maker, it will always try to remind you to save the project (.wlmp) file so you don't lose your progress.

After you have made a few edits and you want to see what they will look like in an actual movie, you should also save your movie. The option to do this is File > Save movie. It's slightly lower in the menu, so be sure you are choosing the correct option.

That will try to follow all the edits you have made in your project to make a single video/audio file that you can then view elsewhere. The resulting file will be a .mp4 file by default but you can also choose the .wmv format if you need that instead.

So are you having trouble finding the option (File > Save movie) or are you getting an error if you try to save as a movie?

First of all you need to locate your Startup. But before that make a small note that each user on a computer has totally another Startup folder. So if you add a shortcut to any program to the Startup folder from the user account that you are logged into then it will run when that particular account is logged into.



So bring the Run command dialog box into your computer screen by pressing Windows and R
Type shell:startup on the command field and click Ok.

The Startup folder for the logged in user account will come on the screen.
For the programs that you wish to run automatically at boot add their shortcuts to the Startup
To create a program shortcut right click on an empty area of the Startup folder and select New followed by Shortcut.

On the newly opened Create Shortcut window click on the Browse link and find the file that you wish to run automatically.

Provide a name to the shortcut created.
And click Finish.
For the programs whose shortcuts are already created, you can shift them to the Startup
In case you want to disable the created shortcuts than you can do from Task Manager.
Or you can easily remove the shortcut from the Startup.
