You should be able to create an account using Command Line with Elevated Privileges.

Click on the Start Button, and type in CMD. In the list you should see the CMD option.

Right click it and select Run As Administrator

Select YES to the prompt and CMD will open

In this example I am going to create an account called Test with a Password of Workout1

In the CMD window type the following, with the details that you want for your wife

net user Test Workout1 /add

you should then see command completed successfully.

At this point the account is created but is just a basic user. If you want to now add her to the local administrators group then you will need to enter the next command:

net localgroup administrators Test /add

The Test account should now be added to the Administrators group with the password Workout1

Remember to specify the account name you want instead of Test and the password that you want as well.

Please follow the below methods and check the issue.

 

Method 1:

 

Disable antivirus software:

If you have installed any security software, then try to disable the Security Software temporarily and check if it helps.

 

Important Note: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you have to temporarily disable it to install other software, you should re-enable it as soon as you're done. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

Enable the security software back after resolving the issue. If the issue resolves then you might need to uninstall the anti-virus software and install it back or change the settings of the same.

 

 

Method 2:

 

Check whether Volume Shadow Copy service is started.

 

a. Press Windows symbol + R key simultaneously, type in services.msc and then press Enter.

b. In the Service list, right click on Volume Shadow Copy service , and then select Properties.

c. Make sure that the service is started and running automatically.

 

Method 3:

 

You may boot in Safe mode and then try to perform a system restore. This will help check if any non-Microsoft service is interfering with the system restore.

 

In Safe mode; you have access to only basic files and drivers. Refer to the link to start the computer in safe mode and then check –

 

Windows Startup Settings (including safe mode):

http://windows.microsoft.com/en-US/windows-8/windows-startup-settings-including-safe-mode

c) Following the steps as mentioned in the interface to restore the computer to a previous point.

Note: When you perform System Restore to restore the computer to previous state, programs and updates you installed are removed.

 

Method 4:

 

If you can perform System Restore in Safe Mode, I would suggest you to put your system in clean boot state to prevent any conflicting third party applications from interfering with your computer.

How to perform a clean boot to troubleshoot a problem in Windows Vista, Windows 7, or Windows 8:

c.       Tap or click the Services tab and clear the check box beside Hide all Microsoft services, and then tap or click Enable all.

d.       Next, tap or click the Startup tab, and then tap or click Open Task Manager.

e.       In task manager, enable all of your startup programs, and then tap or click OK.

f.       When you are prompted to restart the computer, tap or click Restart.

 

Method 5:

 

 

If issue persists, then I would also suggest you to try performing SFC (System File Checker) scan on your computer.

 

 

Run the System File Checker (SFC) scan to fix any corrupted files. Follow these steps:

sfc /scannow

 

The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions.
