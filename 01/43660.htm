The configuration that you want to achieve can be done by changing the corresponding registry entry via the Registry Editor. Follow the steps below to do so:

  1. Click the Start menu, type regedit and press Enter to open Registry Editor.

  2. Navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\System.



  3. In the right pane, double-click on Wallpaper string value labeled 2 on the screenshot and enter the full path of the wallpaper you want to use (.jpg and .bmp). Check the screenshot below for example.

  5. Close the Registry Editor and restart the Explorer process or simply sign out and sign in to apply the changes.

patricia2002 say:

I was going crazy with exactly the same issues as you and finally figured it out. 

Explorer and Google use the same settings. So when you change one, the other responds as well.  The easiest way to fix this is in Explorer, but it you don't have that, you can do the same thing by going to the Google Chrome icon, right click and open it with "Run as Administrator."   Then do the same things I did below from Chrome Settings  If you don't know where to find this, the bar just below the  top bar has see 3 horizontal lines. Click with left button, down to settings, then go to Advanced Settings towards the bottom to CHANGE PROXY SETTINGS.  This brings up the same Setting Box as in Internet Explorer.  Then follow instructions on "D".

A. If you have Internet Explorer, go to your icon for Internet Explorer on the Start Menu.  Click on your right right mouse button and on the drop down menu, open it with "Run as Administrator." 

B,. When you do this, then a box opens and it asks if you want this program to make changes to your computer.  Click Yes.  Then it opens Explorer.

C. Go up to the menu (If you don't see one, then click on gray bar just under the dark blue Internet Explorer Bar with your right mouse and check menu).  On the Menu bar, go to Tools and then at the drop down menu, click on Internet Options. 

D. Then select the tab, Connections, then LAN settings and REMOVE THE CHECK from  USE PROXY SERVER and now CHECK AUTOMATICALLY DETECT SETTINGS.  CLICK OK in LAN Setting Box and then OK in the final window.  NOW IT WILL STAY since it now recognizes you as the Administrator.

All the best!

Patricia

sharyl_323 say:

I was dealing with the same problem until just yesterday (August 31, 2014).  I had tried a variety of "fixes" mentioned online with no success, including the "Run as Administrator" suggestion mentioned here; initially it seemed to work, but the LAN settings kept reverting back to being checked.  I ended up finding a program that had installed itself on my computer to be the source of my frustrations.  I went into Remove Programs within Control Panel and scrolled through the installed programs looking for anything suspicious. I found one called "Browser Security" that somehow became installed back in January, 2014.  I recognized that it was not associated with either McAfee or with Windows Defender, so I removed it.  My connection to the internet has worked well since then; the LAN proxy server box has remained unchecked and the Automatic Detection box is checked, as it should be.

If you find a similar program on your computer, just know that when you go to delete it a message box will come up with a rather official looking "warning" stating that you are about to delete vital security software. This is no doubt intended to frighten the timid in order to help insure that the program remains in place. If you are using Windows Defender, McAfee or some other trust worthy internet security software, then you needn't be concerned.

Hopefully this will work for anyone still experiencing frustrations.  Good luck.
