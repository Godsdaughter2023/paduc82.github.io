The Facebook Connect function that the old instructions relied on was discontinued in June last year, so it's no longer possible to break the link between People and Facebook even though there's no connection between them. However, if you're satisfied that your contacts list at people.live.com is the one you want, you can refresh it in Windows Live Mail like this:

  1. Sign out of Windows Live Mail using the right-most button on the Home tab of the ribbon and wait for the default contacts list to load.
     
  2. Close down the program and any other Windows Live programs that may be running. 
               
  3. In Explorer, navigate to
    C:\Users\{user}\AppData\Local\Microsoft\Windows Live\Contacts
    where {user} is the name of your Windows user account. You will have to enable Show hidden files, folders and drives on the View tab of Folder Options in Control Panel to be able to see the folder.
     
  4. Right-click on the folder named with the email address of your Outlook.com account and rename it, e.g. by removing the @ symbol.
     
  5. Re-launch Windows Live Mail and switch to the contacts window (Ctrl-3). When there is no longer any status message, Sign in with the Outlook.com account.

The system locale determines the default character set (letters, symbols, and numbers) and font used to enter and display information in programs that don't use Unicode. This allows non-Unicode programs to run on your computer using the specified language. You might need to change the default system locale when you install additional display languages on your computer. Selecting a different language for the system locale doesn't affect the language in menus and dialog boxes for Windows or other programs that do use Unicode.

1: Open Region and Language by clicking the Start button , clicking Control Panel, clicking

   Clock, Language, and Region, and then clicking Region and Language.

2: Click the Administrative tab, and then, under Language for non-Unicode programs, click

  Change system locale.  If you're prompted for an administrator password or confirmation,type the password or provide confirmation.

3: Select the language, and then click OK.

  To restart your computer, click Restart now.
