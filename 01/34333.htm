Hey mate, that code can have a range of solutions, but rest assured it does not mean anything is banned.

The first thing to try is switching the Xbox off and unplugging the router from the power. Let it sit for at least 60 seconds before switching it back on as well as the Xbox. princess mat say:

Hi,

Store App icons are saved in an icon cache. The Icon Cache is a database file used to keep copies of every icon. Sometimes, Icon cache can be corrupted. That could be the reason why you're having problems with the app icons.

We suggest rebuilding the Icon cache to resolve the issue. Follow these steps:

  1. Close all the open programs.
  2. Go to Task Manager.
  3. Locate Windows Explorer, right-click it and select End task.
  4. Select Run new task from the File menu.
  5. Type cmd.exe.
  6. Put a check on Create this task with administrative privileges and click OK.
  7. Once an elevated command prompt open, type the following and hit Enter.

cd /d %userprofile%\AppData\Local\Microsoft\Windows\Explorer attrib –h iconcache_*.db del iconcache_*.db start explorer

Your Icon cache will be rebuild.
