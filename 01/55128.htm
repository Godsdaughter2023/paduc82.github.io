Hi Butler,

 

This issue could occur if Bing Search has been set as the Default Search Engine and the check box for “Prevent programs from suggesting changes to my default search provider“has been selected.

 

 

Method 1: Follow these steps to uncheck the same:

a: Open Internet Explorer by clicking the Start button. In the search box, type Internet Explorer, and then, in the list of results, click Internet Explorer.

b: Click the Tools button, and then click Manage Add-ons.

c: In Manage Add-ons, under Add-on types, click Search Providers.

d: Toward the bottom of the screen, clear the check box for “Prevent programs from suggesting changes to my default search provider” and then click Close.

 

 

Refer this link for more information:

How do I prevent programs from changing my default search provider?

http://windows.microsoft.com/en-us/Windows7/How-do-I-prevent-programs-from-changing-my-default-search-provider

 

 

Method 2: Try removing all the Search Providers that you have currently installed. Follow the steps given under the section “To remove a search provider” from the following link:

Change or choose a search provider in Internet Explorer

http://windows.microsoft.com/en-US/Windows7/Change-or-choose-a-search-provider-in-Internet-Explorer

 

 

Method 3: Once you are done with it try adding the new Search Provider by following these steps:

To add new search providers

http://windows.microsoft.com/en-US/Windows7/Change-or-choose-a-search-provider-in-Internet-Explorer

a. Open Internet Explorer by clicking the Start button. In the search box, type Internet Explorer, and then, in the list of results, click Internet Explorer.

b. Click the arrow to the right of the search box.

The Internet Explorer search box

c. Click Find More Providers.

d. Click the search provider you would like to add. This opens the Add Search Provider dialog box.

e. If you want the provider that you just added to be used by default when you search from the Address bar or search box, select the Make this default search provider check box.

f. If the search provider offers search suggestions, select the Use search suggestions from this provider check box to receive search suggestions.

g. Click Add.

h. Repeat steps d through g for each provider you want to add.

Method 3: Change the touchpad settings.

1. Attach the Cover to your Surface. (If you have a Wireless Adapter for Typing Covers, attach the Cover directly to your Surface).

2. Go to Start, and select Settings > Devices > Touchpad.

3. Adjust your settings under Touchpad.
