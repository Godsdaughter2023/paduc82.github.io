considerate_guy say:

It depends on whether the bells are isolated or if they are playing over the talking.  If they were happening in between the pauses in the speaking, then you could perhaps trim out that part of the audio and replace it with similar background noise from another part in the interview.

 

However, if the bells are on top of the speaking, there isn't much you can do in Movie Maker apart from lowering the overall volume of just that section.  To compensate, you might add captions, so even if you can't clearly understand what the person is saying with the bells, it could be read.

How can I resolve software configuration issues in Windows Live - Microsoft Community

a. user say:

How can I resolve software configuration issues in Windows Live Mail?

How can I resolve software configuration issues in Windows Live Mail? How can I resolve software configuration issues in Windows Live Mail?

a. user say:

The software configuration issues in Windows Live Mail may cause synchronization, send and receive errors such as 0x80048820, 0x80070057, 0x80072EFF or 0x80048849. These issues may occur due to one of the following reasons:

 

1.       An email is stuck in the Outbox folder of Windows Live Mail

2.       You may not be using the latest version of Windows Live Mail

3.       Your computer needs Microsoft’s latest updates

4.       Your mail account may not be properly configured in Windows Live Mail

5.       Internet Explorer settings may be causing connectivity problems

         A.      SSL settings are incorrect

         B.      Incorrect Proxy settings

         C.      Internet Explorer may be set to work offline

 

1.       An email is stuck in the Outbox folder of Windows Live Mail

 

i.  Check the Outbox folder and delete all of its contents before sending a new e-mail.

ii. If you’re unable to delete the email from the outbox folder, you may have to first go offline, delete the email and then go back online.

 

2.       You may not be using the latest version of  Windows Live Mail

Ensure that you are running the latest version of Windows Live Mail. To determine the version of Windows Live Mail installed on your computer:

 

i.    Open Windows Live Mail.

ii.   Click on the file Menu drop down in the upper left corner.

iii.  Click About.

iv.  The version of Windows Live Mail will be displayed.

 

If you are not running the latest version of Windows Live Mail you may need to uninstall and install the latest version.

 

i.  Windows Live Mail is installed with the Essentials package and may be downloaded here.

ii.  In some cases you may need to uninstall Windows Live Mail or Essentials before you are able to install the latest version(s).

     a. Uninstall one or more Essentials Programs  

     b. In some cases you may need to install Windows Live Mail / Essentials on a system without internet access. Click here for Instructions.

 

3.       Your computer needs Microsoft’s latest updates

 

i.      Visit http://update.microsoft.com/microsoftupdate/v6/default.aspx and follow the instructions to install the updates.

ii.      Restart the computer

 

4.       Your mail account may not be properly configured in Windows Live Mail

If you are running the latest version of Windows Live Mail and your email provider has confirmed that the email servers are running, you may need to remove and re-configure your email account in Windows Live Mail. Please follow the steps given in the article ‘How do I add an account to Windows Live Mail?’ to re-configure/ add an email account to Windows Live Mail. You may need to remove an account before you re-configure it.

 

5.       Internet Explorer settings may be causing Connectivity Problems

 

A.   SSL settings are incorrect
Incorrect SSL settings in Internet Explorer may prevent email from being sent or received

 

i.    Start Internet Explorer. 

ii.   On the Tools menu, click Internet Options, and then click the Advanced tab. 

iii.  In the Security section, make sure that the following check boxes are set correctly:

      a.       Clear the Check for server certificate revocation check box. 

      b.      Select the Use SSL 2.0 check box. 

      c.       Select the Use SSL 3.0 check box. 

iv.  Click OK to close the window. 

 

B.   Incorrect Proxy settings
Clear the proxy settings in Windows to establish a direct connection to the Windows Live service. To do this, follow these steps:

i.    Start Windows Internet Explorer. 

ii.   On the Tools menu, click Internet Options. 

iii.  Click the Connections tab, and then click LAN settings. 

iv.  Click to clear the automatically detect settings check box. 

v.   Click OK two times. 

 

C.      Internet Explorer may be set to work offline

To enable Internet Explorer to work online:

i.      Open Internet Explorer.

ii.      Click to open the file menu.

iii.      If there is a check next to Work Offline, click Work Offline to clear the option.
