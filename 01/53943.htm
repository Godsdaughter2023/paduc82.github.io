    cmdrkeene say:

    Hi Frank,

    You don't want to remove those.  It's possible from the command line with some advanced commands, but it will break.  Many programs, even parts of Windows itself, look for those folders in that place.  These placeholders function as junctions and will redirect any programs to the new location if they try to use them.  But if you delete the placeholders, those programs won't get redirected properly and they will fail.
