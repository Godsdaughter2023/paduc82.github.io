lemp say:

You're looking at the BIOS boot order selection screen.  This is not what most people refer to as a "Boot menu."

The BIOS shows that you have two CD drives, an internal one and a USB one.  Pick one and move it to the top of the list (probably by selecting it an using the arrow keys, but you should see directions somewhere else on that same screen).  Once a CD drive is at the top of that list, exit saving your changes (again, there will be directions on the screen for which key to press to do this -- probably F10).  The next time you boot up the computer, the BIOS will check the CD drive first and if there is a bootable CD in it, you'll boot from that CD.  Or perhaps you'll get the message "Press any key to boot from CD."  This boot device selection order remains how your computer will work until you go back into the BIOS and change it.

When a computer has a "boot menu" accessed by one of the F keys during the boot up process, that allows you to make a temporary change in the device to be used to boot from.  If you make the change this way, your normal boot sequence -- booting from the hard drive -- will occur on subsequent boots without needing to check the CD drive first.
