    To know that your (admin) profile is corrupt, go to Task Manager then click on tab details, look for dwm.exe (by default) it will show running DWM-1 (admin/main). Then shutdown or restart. go back to Task Manager - details see what is running now. If it shows DWM-2 or others. notify Microsoft.
