al jarvi say:

If you do do a hard reset or simply reconfigure the router then I highly suggest...

  • Change the wireless routers admin password to something other than the default. Use a strong password.
  • Disable the ability to configure the wireless router over the public internet. Do all administration tasks via a wired connection from a laptop or a desktop PC on your local network.
  • Change the default wireless network SSID to an easily identifiable and unique SSID
  • Use encryption on your wireless network using the strongest encryption scheme supported by your wireless router and client computers. Use WPA2-Personal if possible. At a minimum use WPA-Personal. Do not use WEP.
  • Use a long random ASCII key longer than 25-characters. I recommend the use of an online key generator to generate a long random ASCII key.

dipandocat say:

Where can i download Windows 7 to reinstall it?

Hi, I dont have my Windows 7 installation DVD, but i have the license key that comes with my laptop, where i can download Windows 7 if i want to reinstall it?
andre da costa say:

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

　

- Samsung uses ESC on most of its systems

- ASUS F9

- HP uses F9 on most of its systems

- Fujitsu uses F12 on most of its systems

- Acer uses F12 on most of its systems, but you often need to go into the BIOS to enable it

- Dell uses F12

　

In general, look at the screen during POST for a hint about what is possible.

　

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee.

trekdozer say: When the Windows (and the Windows license) comes with your laptop you usually don't need install disks to reinstall Windows 7 - there will be a recovery partition you can use to reinstall.


When Windows comes installed with the computer, OEM computer manufacturers are required to provide at least one way for you to reinstall Windows - your computer should have included either a restore partition on your hard drive, or Windows install/restore disks. ﻿Most computer manufacturers also have a feature to make your own install disks.
If the restore partition is still there and was not removed, it will reinstall Windows. There is usually a prompt when you first start the computer telling you what key to press to start the restore﻿ - check your manual if you're not sure how to. Or if you have install disks - insert them, restart and boot from them to start the reinstall process.
If you restore it to factory condition, it will be like it was when you first bought it - all preinstalled programs will be installed. Anything you installed after buying it must be reinstalled. This will remove any personal files on the computer, back them up first.
 
Contact the OEM computer manufacturer and ask them to send you an install disk for your computer - they may charge a small fee. Contact information for the manufacturer of your computer can be found here:
http://support.microsoft.com/gp/oemphone/
