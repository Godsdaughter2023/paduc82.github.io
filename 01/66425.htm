syriel say:

A US console will have the NTSC-U/C region by default. It will not have the PAL region code unless it has been modded by a third party (and thus any warranty and support voided).

That said, Xbox 360 games are not really NTSC or PAL native. Instead, the game is rendered internally (usually at 720p, though it does vary) and the output is determined by the console settings. If it is HD, it will be the same the world over. If SD, then the various NTSC and PAL options (as well as VGA) are exposed based on the console region.

Because of this, PAL games which are region free WILL PLAY without issue in a NTSC console. However, region locked games will get a region mismatch error. This is why you are getting the error with Kinect Star Wars. As for Nike+, you are probably getting the mismatch for the same reason.

If you purchase a local console rather than an import you shouldn't see these issues.

andre da costa say:

You upgraded to Windows 8, but didn't have a previous version of Windows installed (error 0xC004F061)

If you see error 0xC004F061 when you try to activate Windows 8, it means that you're using a product key for an upgrade version of Windows 8 and a previous version of Windows wasn't on your PC when Windows 8 was installed. To install an upgrade version of Windows 8, you must already have Windows 7, Windows Vista, or Windows XP installed on your PC.

If you formatted the drive before the upgrade version of Windows 8 was installed, you won't be able to use your upgrade product key to activate Windows 8. To activate Windows 8, you'll need to install your previous version of Windows, and then reinstall Windows 8. For help with the activation process, contact support.
