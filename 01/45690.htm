billmcgrathwriter say:

I have the same problem, but first a SIMPLE possible solution to the OP teacher's problem. 

Although this will not actually make one long movie, it will be an easy fix to achieve the same goal.

Every DVD maker software I have ever tried allows you to burn more than one movie file onto the same DVD.

Almost every DVD player software allows you to start the first movie on a DVD and when it is done it will automatically launch the next.  So burn one DVD with multiple movies on it (I suggest titling them by students last name.) Then carry around just 1 DVD, and be able to play all of the individual movies back-2-back.

Don't be intimidated by the DVD maker software. Some call the individual movies "Chapters" but they are simply individual WMV files (The output of Windows Live Movie Maker). Also, unless you want to do a bunch of hard work, put the individual movie files all in the same folder and rename them in the sequence you want them to appear on the DVD. Unless you tell it to do something different, The DVD maker software is going to load the movies onto the DVD in alphabetical order using the file name.  So if you give it three files called

 "Smith, Jones, and Adams" then it will burn them as Adams first followed by Jones and last will come Smith. If you want them to play in the original order then name the files something like "AAA-Smith, BBB-Jones, and CCC-Adams."

Good luck.
