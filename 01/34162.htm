"Edit in Microsoft Word" from other applications truncates - Microsoft Community

john rehmert say:

"Edit in Microsoft Word" from other applications truncates source location...

This is an interesting issue and I've boiled it down to some command line or function limitation but want to report it, nevertheless...

PROBLEM: Essentially, when one chooses the "Edit in Microsoft Office Word" option under the "File" menu in Internet Explorer (or other Microsoft programs with that option) AND Microsoft Word is NOT already open (a running WINWORD.EXE process in Task Manager), then the location for the source document (a URL in IE) is passed via the call to start Microsoft Word (via the cmd processor or shellexecute, maybe?). The fact that the location is passed in that manner makes it susceptible to a character limitation of around 255 characters and truncates the location. That truncation makes the resulting converted document different than the source in Internet Explorer, because parameters that existed in the URL used by IE are truncated when passed to Word so, in essence, Word is requesting a different page.

WORKAROUND (not really): If one already has Word open (WINWORD.EXE process in Task Manager) and chooses the "Edit in Microsoft Office Word" option under the "File" menu in Internet Explorer, the source for the document conversion must be passed via an object to the running WINWORD.EXE process, rather than via a call to open Word and the resulting document is an accurate conversion of the page represented by the entire URL (no truncation occurs).

QUESTIONS:
1. Might Microsoft consider re-wiring the "Edit in Microsoft Office Word" function to first spawn the winword.exe process and then pass the location to Word using the object used when winword.exe is already running?
2. Might there be an easier workaround/fix that can be put in place (reasonably) in the meantime - or forever?
3. Does anybody know exactly what is limiting the character length for the "Edit in Microsoft Office Word" function?

Any answers are greatly appreciated!
- John
