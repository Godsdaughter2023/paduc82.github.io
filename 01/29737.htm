Hi,
 
Thank you posting on Microsoft Community.
 
We understand that you are facing an issue adding the printing device.
 
Please follow the steps and try resolving this issue:
 
Method 1: Run Printer troubleshooter after connecting the printer.
1. Press the ‘Start’ Button.
2. Type Troubleshooting
3. Select Troubleshooting and press Enter.
4. Click on view all on the left side and select Printer.
5. Click next and follow the on-screen instructions.
 
Method 2: Change the Print Spooler service type.
1. Press Windows +R keys.
2. Type services.msc in the box and press enter.
3. Now locate services called Print Spooler, make sure the services are Started.
4. Right click on the service and click Stop.
5. Click Apply and then click ok.
 
Note: After installing the printer make sure you Turn On this Print Spooler Service again by following the same steps.
 
Method 3: Turn off the Windows Firewall and check adding the Printing device and Turn On after installing.
1. Open Windows Firewall by clicking the Start button, and then clicking Control Panel. In the search box, type Windows Firewall, and then click Windows Firewall.
2. Click Turn Windows Firewall on or off on the left side. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
3. Click Turn off Windows Firewall (not recommended) under each network location that you want to stop trying to protect, and then click OK.
 

Warning: You should not turn off Windows Firewall unless you have another firewall enabled. Turning off Windows Firewall might make your computer (and your network, if you have one) more vulnerable to damage from worms or hackers.

Reference: Click on the link to Fix printer problems
http://windows.microsoft.com/en-us/windows-10/fix-printer-problems
