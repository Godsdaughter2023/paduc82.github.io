a. user say:

Hello,

I understand you concern in this regard.

Usually safe mode starts Windows with a limited set of files and drivers. Startup programs don't run in safe mode, and only the basic drivers needed to start Windows are installed.

So this behavior can occur because the Safe Mode with Networking option does not start the services for the Zero Configuration utility (a built-in utility) that is used by the Institute of Electrical and Electronics Engineers, Inc. (IEEE) 802.11 wireless local area network (LAN) standard network adapters nor does it start the service for IEEE 802.1x wireless LAN standard authentication.  So you are unable to use Wireless connection in Safe Mode.

To work around this behavior, attach or use a standard-wired network adapter in Safe Mode with Networking.

davem121 say:
Hi JerryG, you can reinstall Windows 10 at any time and it will not cost you anything !

Create bootable installation media on a USB flash drive, rather than a DVD, it is far more reliable and faster to install

Disconnect any drives other than the drive you are installing Windows onto

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

At the Start of the installation is a window with 2 large buttons, choose the bottom one (Custom: Install Windows only) then on the next window, select each partition on that drive and delete until you are left with just one unallocated space. Select the unallocated space and click 'Net' to begin the installation . ..

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers


If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

slance310 say:

Hi,

It is probably best to set the CSM to AUTO....

This way if you happen to boot from something not configured for UEFI, the Bios will automatically switch to Legacy...

The UEFI Bios is supposed to be faster because it has eliminated a lot of the older stuff that is obsolete from what it looks for everytime you boot...

It also makes possible in Windows 8, 8.1, and 10 called Fast Startup.. This is a type of hybrid Shutdown\Hibernation that also speeds things up by storing settings in effect when you shutdown, instead of reloading them...

One drawback to the Fast Startup is that if you remove power from your machine after the shutdown, it can corrupt your HDD\SSD requiring a chkdsk to run, or worse...

Some of that other stuff I am unfamiliar with as well.. There should be detailed description in the manual, but maybe not...

The interaction is touchy, especially a installation or repair of windows with boot media...

Not all machines have the "UEFI DVD" option in Bios, and trying to use one can be quite a problem...

Same with flash drives... To do a installation of Windows 10, you need a flash drive created specifically for UEFI.. A flash drive needs to be a GPT\FAT32 type drive, and the older Legacy Bios needs to be MBR\NTFS type drive...

Glad your problem is resolved after all that work.. I've been there at times myself...
