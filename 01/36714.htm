What is the most efficient way to get a computer to start - Microsoft Community

w. carter say:

What is the most efficient way to get a computer to start (Boot) w/ hard-drive when USB / CD is the "Start-up" selection (Bio's order)?

Please help,

I just need for my computer to start normally, from the hard-drive, but it's stuck on boot from external (USB / DVD) in "Start", or "Bio's". Anybody? So simple I don't know. Thanks! Help. Win $$ :)

a. user say:

If the boot order in the BIOS is ...

first from an optical drive,

then from a hard drive

and there's no disk in the optical drive, the BIOS will skip over the optical drive and try to boot from the hard drive.

You can actually hear the BIOS checking the optical drive, because it will start to spin until the BIOS realizes there's no disk there.

If there's no bootable operating system on the hard drive, the BIOS will try to boot from the next drive, if there is one, and onward until the BIOS finds a bootable operating system.

That's how booting from a BIOS works, and it happens automatically. If your computer doesn't work that way, then something else is going on.

Of course, it's also possible to boot from an external drive - a USB flash drive, for example - but in that case you have to manually interrupt the BIOS and tell it to boot from the external drive.
