burrwalnut say:

If the error is triggered by unplugging a device too soon, sometimes plugging it back in corrects the problem.

Otherwise a registry key can be changed but create a system restore point before making the change. Also, make a note of the existing value, which, if it hasn’t been altered previously, is the exact amount of RAM that is installed. In a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management

In the right pane, right-click SystemPages then click Modify and type FFFFFFFF. If it doesn’t solve the problem, change the registry key back to its original value.
