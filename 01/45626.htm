foolsfortune say:

Transferring Outlook Express messages to Outlook after Windows 7 "upgrade"

I had backed up my Outlook express messages and .iaf file thinking I'd be diligent and be able to restore after formating and erasing XP and installing Windows 7 only to find out Outlook exppress was history and Outlook won't read .dbx files.

greg schuck say: Well, you are in a little bit of a pickle.  You are correct that Outlook cannot read .dbx files, and it can only import them from a working Outlook Express environment.  Your best bet at this point, since you formatted the XP environment, is to copy the .dbx files to a working Windows XP computer, then import them into that computer's Outlook Express.  When doing the Import, you must specify that you want to import from an "OE6 Direcotry Store" and then point it to your saved OE6 data.  You may want to create a new Windows user login for yourself before doing this.

With the e-mail imported into that Outlook Express environment, if your e-mail takes up less than 2GB you can use Outlook Express' Export function to dump all your e-mail into an Outlook .pst file.  You can then copy that .pst file to the Windows 7 PC and start Outlook, the do File-Open-Outlook Data File and open it.  Do NOT do an Import with Outlook.

If your Outlook Express e-mail is over 2GB, you will not be able to export to a .pst from Outlook Express because it can only do so using Outlook's older .pst format, which has a 2GB file size limit.  In that case, you will need to install Outlook 2003 or 2007 on the computer and tell it to Import the Outlook Express data.  It will pull it all into a .pst file that can grow to over 25GB.  When it finishes, copy that .pst file to your Windows 7 PC, then start Outlook and do the File-Open-Outlook Data File thing.  Do NOT import the .pst file.
