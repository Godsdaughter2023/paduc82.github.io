Hi Kingston Fischer,

Welcome to Microsoft answers!

This behavior is the expected result of certain hardware and software factors.

Various devices in a typical computer require memory-mapped access. This is known as memory-mapped input-output (MMIO). For the MMIO space to be available to 32-bit operating systems, the MMIO space must reside within the first 4 GB of address space.

For example, if you have a video card that has 256 MB of onboard memory, that memory must be mapped within the first 4 GB of address space. If 4 GB of system memory is already installed, part of that address space must be reserved by the graphics memory mapping. Graphics memory mapping overwrites a part of the system memory. These conditions reduce the total amount of system memory that is available to the operating system.

The reduction in available system memory depends on the devices that are installed in the computer. However, to avoid potential driver compatibility issues, the 32-bit versions of Windows Vista limit the total available memory to 3.12 GB.


If a computer has many installed devices, the available memory may be reduced to 3 GB or less. However, the maximum memory available in 32-bit versions of Windows Vista is typically 3.12 GB

To clarify your doubt, I would like to list out the differences as well as the similarities between File history and Backup and Restore feature.

File History:-

1. This feature creates a backup of your personal files and documents as well as your offline onedrive files provided if stored in Library.

2. It creates a timely/periodic backup of your files.

3. You can exclude files that you do not want to backup.

Backup and Restore:-

1. Can backup your personal files as well as enables you to create a system image.

2. Freedom to backup the files that you wish to.

3. Create automatic backup of your files and folders. 

 

Having stated the above facts, I would like to add that the features of file history as compared to backup and restore are more or less the same. The difference being that the features of file history is limited compared to the latter. For more information refer to the article What happened to Backup and Restore? provided in the link below

http://windows.microsoft.com/en-us/windows-8/what-happened-to-backup-restore

How does Windows XP Mode on work in Windows 7 ?

Windows XP Mode works in two ways—both as a virtual operating system and as a way to open programs within Windows 7. It runs in a separate window on the Windows 7 desktop, much like a program, except it's a fully-functional, fully-licensed version of Windows XP.
