Follow the below steps:

a.     When you move digital media files on your computer, the file name and file path information remain unchanged in your library. As a result of this, your library may quickly contain a large number of duplicate or invalid entries.

b.     To prevent music files from being added automatically to your library

c.     In Windows Media Player, on the Tools menu, click Options.

d.     On the Player tab, clear the Add music files to library when played check box.

I think the default settings of the File Save dialog are determined by the application that calls it. The change in behavior may have been caused by an update from Adobe.

However, if you've been typing extensions only to have them appended with the Adobe extension, i.e :

becomes:

myfile.myext.psd
