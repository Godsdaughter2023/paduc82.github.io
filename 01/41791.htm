Hi Jacques,

The setting you want to make in Windows 10 version 1607 (that's the WAU) has changed.  Here's what to do.

Go to: All Settings>Network & Internet>Wi-Fi>Manage known networks> Then select the Wi-Fi network your going to use from the list> select Properties> Then switch to on "Set as metered connection". Then close settings and connect to the network. Windows should remember this network connections settings the next time you use it as well. Updates now should not auto download when using this network connection.

With this setting turned on you can still check for updates if you want to by doing it manually at: All Settings>Update and Security>Check for Updates.
