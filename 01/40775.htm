Confirmed here too on LTSB 2016.

Here is a workaround

  1. Backup your tiledatalayer folder if you've put a lot of work into the start screen
  2. Go to HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders in regedit
  3. Copy the path of the "Start Menu" string and browse to that location.
  4. Copy the Start Menu folder to C:\Users\username
  5. Edit the registry value of the "Start Menu" string to point to this local folder
  6. At this point you should see some items in the start screen disappear (don't panic just logout)
  7. Login and you will be able to export the start screen to xml
