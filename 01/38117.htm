Hi,

This can get a bit complicated because of the different BIOS types in current use... I suspect your new motherboard is using UEFI Bios, but that is just a guess...

If this is the case, and you are not using compatible Installation Media, you will have problems... Windows will not normally install to UEFI Bios using a CD\DVD for starters... But it can sometimes be done... Same with a Flash Drive that is not created specifically for UEFI Bios...

First... When you power on your computer hit the key for "BOOT OPTIONS"...

Usually something like F9, F11, F12, ESC, Delete... This is just a one time boot selection and does not effect other Bios settings...

Look for options in Boot Devices involving "UEFI"...

"UEFI Optical Drive" as opposed to just the standard "Optical Drive" as an example... You may see something similar for the Flash Drive...

If available, try the "UEFI" device... Preferably the Windows 10 Flash Drive if that is what you hope to end up with...

If and when you get to the Installation Screen with "INSTALL", you will want to select "Custom Install" either at that screen, or the next...

You should be able to then delete everything on the HDD\SSD with the tools provided during the Custom Install... Do not Format anything...

Select the Unallocated Space for the installation...

if you do not want to mess with all of the UEFI stuff, you might try switching your Bios to "Legacy" or "UEFI CSM"... which ever option is available...

I have my machine running with the "UEFI (CSM)" which causes it to switch between Legacy and UEFI depending on the type of installation media I use...

I basically run Legacy... It is much easier to deal with... The only real advantage to UEFI Bios is that it allows more than 4 partitions on the OS drive, and MAYBE a slightly faster boot...

Audio Edit in Muvie Maker - Microsoft Community

ca anirudhapusalkar say:

Audio Edit in Muvie Maker

I am having file of video with sound and now want to edit the sound by giving beep on objectionable sound.How can I do?
considerate_guy say:

The first thing you want to do is isolate the part of the clip with the objectionable sound. To do this, play until you get to the start of it and pause. (Note: if you are on Movie Maker 2012, you can view the waveforms to better position the playback marker at the exact start of the sound).

 

Now use the 'Split' button on the Video Tools | Edit tab, or just press 'M' on the keyboard.  Go to the end of the objectionable sound and make another split.

 

If you have done this correctly, your video will be in 3 parts now: good audio, objectionable audio, good audio.  Choose the objectionable video part in the middle and mute its volume (go to the Video Tools | Edit tab and set Video Volume to the far left = zero, mute).

 

The last thing you need to do is add an audio clip of a beep as the "music" in its place.  You just need an audio file of the beep, place it in the empty space and trim it to the right length.

Hi Robert. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

Check for a newer SD Card driver on the PC or Display adapter Support Downloads webpage. While there check if there are other newer drivers to Update including BIOS, Chipset, Display, Sound, USB3, Bluetooth etc. You can take your time until they are all updated.

There is a known issue of incorrect drivers being pushed by Windows Update which requires rolling back the driver. Right click Start to open Device Manager, choose SD card device, then Driver tab, then Roll back. (You may need to unhide drivers from View tab)

If there is no Roll Back or that doesn't help choose Update Driver on device Driver tab, then Browse Computer, then Let Me Pick to see a list of all previously installed drivers, work backwards trying each to find one that works.

If necessary you can also go to Settings>Update & Security>Windows Update to look for the driver under Installed Updates and Uninstall it form link there. Then you can hide the driver so it isn't pushed again using the Hide Updates tool downloaded from here: http://download.microsoft.com/download/f/2/2/f2...

If none of those work try to System Restore to a point before the problem began: http://home.bt.com/tech-gadgets/computing/how-t.... Make sure System Restore is always enabled for C with plenty of disk space apportioned (5-10%) in its Configuration settings.

As a last resort I'd install Media Creation Tool and choose Upgrade Now to do a Repair Install, which will also advance you to the latest version (which you need anyway) if necessry, by the most stable method: http://windows.microsoft.com/en-us/windows-10/m...


I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
