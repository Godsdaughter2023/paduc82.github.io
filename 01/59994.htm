lemp say:

Consider that this is a peer-to-peer technical support forum and thus any legal advice you get here is worth what you pay for it, the answer to your question is that it's probably within the terms of the OEM EULA, but you'll have to convince someone at Microsoft of that when you call to activate the virtual machine.

See the discussion here: 

https://community.spiceworks.com/topic/1978730-can-i-use-windows-oem-license-on-vm-on-the-same-machine

It's not easy to find a copy of the Windows 7 OEM EULA online.  I think this is it, but I'm not sure:  http://tinyurl.com/yblokllh (pdf)

The argument is as follows:

Section 2a defines the term "licensed computer": 

2a. One Copy per Computer. The software license is permanently assigned to the computer with which the software is distributed. That computer is the “licensed computer.”

Section 2d is as quoted in the spiceworks discussion linked above:

2d. Use with Virtualization Technologies. Instead of using the software directly on the licensed computer, you may install and use the software within only one virtual (or otherwise emulated) hardware system on the licensed computer.

The confusion appears to come from the common understanding that OEM Windows software is limited to the computer on which it is first installed.  If that was correct, then the license would be limited to the hardware of the computer you purchased and you couldn't transfer it to the "new computer" which would be the virtual machine, regardless of the hardware on which the vm is hosted.

Good luck.

How to activate Windows by phone:

If you have trouble activating Windows 7 you can call Microsoft to activate by doing the following:
1. With Windows running click Start, then in the Search box type:  slui.exe 4
2. Press Enter on your Keyboard
3. Select your Country.
4. Select the Phone Activation option, then call the number given and hold for a real person.

For more details: http://www.sevenforums.com/tutorials/18715-activate-windows-7-phone.html
