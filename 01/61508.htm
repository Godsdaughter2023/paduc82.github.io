How do I reset Windows Update components?

Let's try troubleshooting them. Kindly note that upon running these games, maintain low to medium graphics setting to prevent crashes. First, check the dxdiag if your PC is currently using your dedicated GPU. If you confirmed that it is currently utilizing your GPU, try these following methods:

Methods 1: Run Windows 10 System Maintenance

Running the maintenance resolves possible hardware and software conflicts. To do so, follow these steps:

  1. On the search box, type system maintenance and press Enter.
  2. Expand Maintenance and on the drop-down menu, click Start maintenance.
  3. Follow the on-screen instructions and upon finishing, restart your PC and check if your issue has been fixed.

Method 2: Run the Hardware and Device troubleshooter

Running the troubleshooting repairs issues that are created by hardware misconfiguration, driver incompatibility, etc. To run the troubleshooter, follow these steps:

  1. On the search box, search and open Troubleshoot.
  2. Under Find and fix other problems, select Hardware and Devices.
  3. Click Run the troubleshooter.
  4. Follow the on-screen instructions and upon finishing, check if your issue has been fixed.

Method 3: Reinstall the Graphics Driver

Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall. Restart your PC and Windows will attempt to reinstall the driver.
