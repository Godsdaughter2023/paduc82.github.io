Overview - Microsoft Community

xbr poseidon say:

Overview

SMC8014WG

Login Information

  • Gateway - 192.168.0.1
  • Username - cusadmin
  • Password - password

 

How to Find SSID (Network Name)


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select Channel and SSID from the submenu.
  6. The Network Name (SSID) will be listed to the right of SSID.

 

How To Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select Encryption from the sub-menu.
  6. Verify the wireless passphrase or security key.

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select Encryption from the sub-menu.
  6. Select None from the Security drop.
  7. Click the APPLY button in the lower right.

 

MAC Filtering


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select MAC Filtering.
  6. Select ALL Wireless stations.
  7. Click the APPLY button in the lower right.

 

Change Wireless Channel

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Select Channel and SSID.
  6. From the drop down menu labeled Channel select the desired channel. (Channels 1,3,9,11)
  7. Click the Apply button.

 

Change Wireless Mode

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Wireless from the menu on the left.
  5. Set the mode to 11G only.
  6. Click the Apply button.

 

Change Wireless Band (width)

 

The default wireless band is 20MHz and cannot be changed.

 

DHCP Reservation


*DHCP is not available on this device.*

 

UPnP


*UPnP is not available on this device.*

 

DMZ


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select Firewall.
  5. From within the Firewall submenu select DMZ.
  6. Check the Enable DMZ Host box.
  7. Enter the consoles IP Address in the boxes to the right.
  8. Click the Apply button at the bottom of the page.

 

Disable DMZ


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select Firewall.
  5. From within the Firewall submenu select DMZ.
  6. Uncheck the Enable DMZ Host box.
  7. Click the APPLY button in the lower right.

 

Port Forwarding


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select NAT.
  5. From the NAT submenu select Port Forwarding.
  6. Click the Add button under the Customer Defined Service Table.
  7. In the Name field enter XBL or something that indicates Xbox Live.
  8. In the type drop down select TCP/UDP.
  9. In the LAN Server IP enter the IP address of the console.
  10. For the Remote IPs select Any.
  11. From Public IP Ports drop down select Single Port.
  12. In the Start Public Port box enter 3074.
  13. Leave End Public Port blank.
  14. In the Private Ports box enter 3074.
  15. Click the Apply button.

 

Disable Port Forwarding


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select NAT.
  5. From the NAT submenu select Port Forwarding.
  6. Check the box next to the Xbox entry.
  7. Click the Delete button.

 

Firewall


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. From the menu on the left select Firewall.
  5. On this screen set the Firewall to Disable.
  6. Click the Apply button.

 

How to Find Firmware Version


  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Status from the menu on the left.
  5. The Firmware Version will be listed under the INFORMATION column next to Software Version.

 

Firmware Update


*Firmware update is not available in this device.*



Restore Factory Defaults


Through Hardware

 

  1. Locate the Reset button.
  2. Using the tip of a ballpoint pen or toothpick press and hold the reset button for 15 seconds.
  3. Release the reset button.
  4. Wait for the device to restart.

 

Through Firmware

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted to login enter “cusadmin” as the username and “password” for password.
  4. Select Tools from the menu on the left.
  5. Select Configuration Tools from the submenu on the left.
  6. Click the Factory Reset button.

 

Bridging Guide

 

*Bridging is not available in this device.*

 

Multiple Console Support


We have confirmed that this network device is unable to support more than one console with an Open NAT. 

 

Known Issues

 

The firmware for this device may be partially or completely blocked for use by consumers. You may need to contact your ISP to have any changes done within the firmware.

imran chand say:

Hi,

In order to perform the task you have asked in the question we need to edit the registry settings. However, we can either disable the right click context menu or can enable it but we cannot disable some functionality of it.

 

a.     Type regedit.exe in the start search box.

b.     Locate the following key:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer

c.      If key does not exist then create a key by right clicking on the registry editor and select DWPRD 32 bit.

d.     Create a new 32-bit DWORD value on the right-hand side named NoViewContextMenu with one of these values:

•Enable Context Menus: 0

•Disable Context Menus: 1

e.     The change should be instant, whether enabling or disabling the context menus.

Note: This setting could also be set under HKEY_LOCAL_MACHINE as well, so you should check both places if this doesn’t work.
