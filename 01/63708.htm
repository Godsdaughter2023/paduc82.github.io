marymcgowan say:

AM_Delta_Patch1.exe

My XP SP3 just froze one me.  Task Manager listed first item running as "AM_Delta_Patch1.exe".  Cannot find this in Help Index.  Do you know what it is?

 

rhabdomantist say:

This is part of the Microsoft Antimalware definition update process for Microsoft Security Essentials or Windows Defender .

During the Automatic definition update you will experience a spike in resource usage particularly MsMpEng.exe and wuauclt.exe as well. Older XP  machines with low RAM (<1GB) will freeze or slow down during the update for a short period of time (<2 min.)

For reference: http://systemexplorer.net/db/am_delta_patch1.exe.html

and http://answers.microsoft.com/en-us/Search/Search?SearchTerm=AM_Delta_Patch1.exe&CurrentScope.ForumName=Protect&CurrentScope.Filter=&askingquestion=false

Which product are you using? (note: if using MSE then Windows Defender should have been uninstalled during the  installation of MSE-Windows Defender and Microsoft Security Essentials )

For a better observation of what occurs during the update consider downloading and using Process Explorer to monitior the occurence.

Process Explorer v15.12
http://technet.microsoft.com/en-us/sysinternals/bb896653

You can replace Task Manager with Process Explorer  for more comprehensive view of system functions and aid in diagnosing issues including malware infections.

Some additional informative resouces are:

Windows Sysinternals Primer: Process Explorer, Process Monitor, and More
http://channel9.msdn.com/Events/TechEd/NorthAmerica/2010/WCL314

Case of the Unexplained
http://technet.microsoft.com/en-us/sysinternals/bb963887

More direct to your question, the process you inquired about is  a legit one and the slow down/ freeze of your computer is most likely due to low RAM and/or having a number of other programs/applications running during the Automatic Update process which typically occurs once every 24 hrs.

Do you have any other antivirus/antimalware (such as Norton, Avast, McAfee, AVG) installed? If so conflicts may arise and contribute to the problem. If you have MSE installed refer to the following:

Microsoft Security Essentials - Installation Checklist and Frequently Asked Questions
http://experts.windows.com/w/experts_wiki/89.aspx

 

Hope this helps.
