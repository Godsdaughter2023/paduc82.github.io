vijay b say:

Hi,

Do you have any proxy settings enabled on the computer?

If you are seeing the apps showing down arrow, then it tells that the apps are downloading.

First, I would suggest you to make sure the apps get downloaded completely. If the apps are not getting downloaded, then I would suggest you to reset store cache and see if that helps.

Method 1: Reset Windows Store cache
a) Press the ‘Windows + R’ key on the keyboard.
b) Type "Wsreset.exe" and press Enter.
Check if the issue persists, if yes, please perform a clean boot.

Method 2: If you have any issues in downloading the apps, then I would suggest you to check if you have proxy settings enabled.

a. Click the Gear box at the top right corner in internet explorer.

b. Click the Tools button, and then click Internet Options.

c. Click the Connections tab, and then click LAN settings.

d. Uncheck the option “Use a Proxy Server for your LAN”.

e. Click Apply and Ok to cave changes.


Method 3: Once the apps are downloaded, then I would suggest you to check if you are able to pin the apps.

Method 4: If you are not able to pin the apps after downloading completely also, then I would suggest you to create new user account.

a. Press the “Windows Key + i”
b. Click More PC Settings
c. On the left, click Users
d. Click “Add a User”, and follow the onscreen instructions to create a new user account.
http://windows.microsoft.com/en-US/windows-8/create-user-account

Copy old files to new profile:
a. Navigate to C:\Users\Corrupted_Profile_Name\. The profile name will be the corrupted profile name.
b. Copy all the files.
c. Open Control panel and Folder View options uncheck Hide protected operating system files.
d. Navigate to C:\Users\Corrupted_Profile_Name and copy Appdata folder.

Note: Do not copy these three files. NtUser.dat, NtUser.ini, NtUser.log.
e. Navigate to C:\Users\New_Profile_Name\.

Hope this helps! Let us know if you have any other issues with Windows in future.
