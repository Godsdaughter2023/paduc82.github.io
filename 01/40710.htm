grahamcennon say:

Scan to Network broken after windows update - 2019 Jan 11

Hey guys, We figured out a solution.

So we had multiple clients calling saying the scan to network no longer works. This applies to all printers models, Xerox, Richo , Samsung and all versions of windows.

It appears that the windows update actually broke the administrator function from being able to handle the transaction. If the user created is part of the administrators group, they will not be able to scan to network anymore.

here’s how to fix.

Create a new user on the machine in question - But leave them a member of just users/and add guests. Set a user name and password this new user.

Log into the scanner (scan to network settings) and set the new user credentials and the scan to network will complete again.

I am 150% positive that this will fix your issue. I have fixed 7 locations already all with different printer makes and models.

The windows update basically won’t allow administrators to handle the scans anymore. WHY? because Microsoft that’s why.

hairyfool say:

MBR drives cannot have 12 primary partitions, the only way you can do what you require under MBR is 3 primary, 1 extended and then create 9 logical partitions within the extended or create a single extended with 12 logical

GPT (GUID Partition Table) allows more partitions.

How you format those partitions (NTFS, exFat, HPFS etc.) is what you do IN those partitions.

Disk manager has limits to what it can do, DiskPart can do more but is quite technical.
