a. user say:

Hello Tamisha,

We realize the inconvenience you have experienced.

Where exactly you're looking for web cam drivers in Device Manager?

If your webcam or Camera app in Windows 10 won’t open and you get an error message saying that we can’t find or can’t start your camera (including error codes 0xA00F4244 or 0x200F4244), the cause might be antivirus software that’s blocking the webcam or camera, or it could be an outdated webcam driver. For more help, see also

Check your antivirus software:

In your antivirus app, look for settings related to blocking access or permission to use your webcam. For help, visit your antivirus software company’s website.

Check for an outdated webcam driver:

  1. Select the Startbutton, type Device Manager, and then select Device Manager from the search results.

  2. Find your webcam under Imaging devices or Sound, video and game controllers.

  3. Press and hold (or right-click) the name of your webcam, and then select Properties.

  4. Select the Driver tab, select the Driver Details button, and look for a file name that includes stream.sys. If it’s there, your webcam was designed before Windows 7 and you'll have to replace it with a newer webcam.

  5. If you don't find a file name that includes stream.sys, try rolling back your webcam driver.

Roll back your webcam driver:

  1. In Device Manager, press and hold (or right-click) your webcam, and then select Properties.

  2. Select the Driver tab, select Roll Back Driver, and then choose Yes. (Note that some drivers don’t provide a rollback option.)

  3. After the rollback is complete, restart your PC, and then try opening the Camera app again.

  4. If rolling back didn’t work or isn’t available, uninstall your webcam driver and scan for hardware changes.

Uninstall your webcam driver and scan for hardware changes:

  1. In Device Manager, press and hold (or right-click) your webcam, and then select Properties.

  2. Select the Driver tab, select Uninstall > Delete the driver software for this device, and then select OK.

  3. In Device Manager, on the Action menu, select Scan for hardware changes. Wait for it to scan and reinstall updated drivers, restart your PC, and then try opening the Camera app again.

  4. If your webcam still doesn’t work, try looking for help on the webcam manufacturer’s website.

Check and install latest Windows 10 Camera drivers for your laptop on Device Manufacturer website. You can search for them using your device model number. Most probably this will fix the issue.

I have read your post very carefully. There seems to be some confusion on your part. Let me arrempt to clarify the information so that a good solution might be found for your needs.

The Epson LQ-680 is a dot-matrix printer with a 24-pin print head. It is engineered to use a parallel port. This is to say that its computer interface conforms to IEEE 1284 specification. This is often referred to as a parallel Centronics port. Not serial.

http://www.epson.com/cgi-bin/Store/support/supDetail.jsp?BV_UseBVCookie=yes&oid=14331&infoType=Overview

Thus, since your computer lacks a suitable port for a direct connection, you must be using a parallel to USB device.

Not all of these devices support printers. Please check the specifications of the device you are using to connect your printer to USB and confirm that it does support printers. It is best that it support bi-directional printing through the device. If it does not, getting the device set up for this printer might be a problem.

First of all, iTunes doesn't have anything to do with Firefox or any other browser.  It's a completely separate program.  QuickTime, however, may be used with Firefox in order to view certain types of media files.  There are other programs that do the same thing, so you don't absolutely need to install QuickTime.

Please answer the following questions:

1. What do you mean by using iTunes with Firefox?  Is Firefox open or closed when you attempt to start iTunes?  What happens if you close Firefox and then attempt to open iTunes?

2. What do you mean by using QuickTime with Firefox?  Are you attempting to view a file on a website that needs QuickTime to view or are you attempting to open QuickTime manually, either from the Start > All Programs menu, a desktop shortcut, or by double clicking on a media file saved to your computer?  If you're not attempting to view a file on a web site, what happens if you close Firefox and then attempt to open QuickTime?

3. What version and service pack of XP are you using? Right click My Computer and select "Properties."

4. What version of iTunes are you using?  With Firefox (and all other web browsers) closed, start iTunes.  If it starts and stays open, click Help > About iTunes.  If it doesn't start or starts and immediately closes, please report that fact.

5. What version of QuickTime are you using?  With Firefox (and all other web browsers) closed, start QuickTime.  If it starts and stays open, click Help > About QuickTime.  If it doesn't start or starts and immediately closes, please report that fact.

If you don't want to answer the previous questions, the simplest way to deal with issues of this sort is to uninstall things and then download fresh copies and reinstall.

To perform a "clean installation" of iTunes use this suggested technique, taken from Apple Support article HT1926.

More recently, Microsoft has made available its Program Install and Uninstall troubleshooter which is supposed to "work better" than the old Installer Cleanup Utility.

1. iPod

3. QuickTime

Empty your temp directory:
Open My Computer > C: > Documents & Settings > your user name > Local Settings and delete the contents of the folder called "Temp"
