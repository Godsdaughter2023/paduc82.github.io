ralf_g say:

At a guess your HD 4550 card is resetting itself after a driver crash or card lockup. That leaves only your HD4850 recognised by the system which reverts it to the standard orientation. Subsequently when the HD 4550 is re-established in the system it is placed in the default secondary display position. Solve the driver problem and your chosen orientation should stick. IMO.

Your side issue could be related to the primary issue.  A compatibility problem with the TV perhaps. Or possibly a resource sharing issue in your computer. Having 3 high definition audio devices for instance might be too much (I'm assuming audio chips on the mobo). Run System Information and look at the Conflicts and Sharing section under Hardware. To try to instigate a change in resource allocations, check in your system BIOS if you can change the setting to have the motherboard allocate system resources instead of leaving it all to the operating system. Might not help at this point because Windows doesn't like to change its resource allocations but worth a try. I have one mobo that invariably shares the same IRQ with several key system devices, regardless of what OS version I run. Almost everything works regardless, but Windows is never able to allocate resources to the HDMI audio in any of the video card's I've had in that system. Was well out of warranty before I discovered that little problem though.

a. user say:

If it was my system I would perform a System Restore to the date/time prior to the updates - Windows Update always creates a Restore Point before installing updates and that is for a good reason.

If you have lots of time you can try lots of things but doing things is usually better than trying things.

LKGC is unlikey to work unless the problem is a driver or a registry problem (unlikely with Windows Update).

Startup Repair is unlikely to work since the system sounds like it might at least be attempting to boot, you you can sure give that a try if you have some Windows 7 bootable media (most people don't).

From the F8 Advanced Boot Options menu you can choose Repair Your Computer and perhaps get to the Windows System Recovery Options and choose System Restore.

If you don't have any bootable Windows 7 media (CD/DVD or USB) now would be a good time to make some - perhaps you will be able to do that with the ideas from Bill Smithers.  Think about what you are going to do the next time this happens and how are you going to fix it?

Plus you have not described any symptoms of the non boot scenario...  Do you get a Blue Screen of Death, does the system just restart, do you get some other error message, etc.
