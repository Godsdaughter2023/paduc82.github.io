  1. Do you have a Windows 7 installation disc?

If you have a Windows 7 repair/installation disc, you can forcefully shut down the computer, try to
boot from a Windows 7 installation disc to the Windows Recovery Environment mode, and then try to run Startup Repair and check if you are able to boot normally.

In addition to that, I would suggest you to run a chkdsk as detailed below and then try booting the computer.

  1. Open command prompt tool in the Windows Recovery Environment mode.
  2. You'll see X:\Sources
  3. Type "chkdsk c:\" at the flashing prompt to begin the Chkdsk utility. Replace "c:\" with the letter of the drive for which you want to run the Chkdsk utility. If the hard drive is "d:\", then type that instead.
  4. After the "chkdsk c:\" command insert "\f" to fix errors on the disc volume or "\r" to recover bad sectors and then press Enter.
  5. Restart the computer once the chkdsk is completed. 

Note: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive, it tries to repair that sector, if any data available on that might be lost.

Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.
