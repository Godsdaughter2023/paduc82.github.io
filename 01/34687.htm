From the Windows 8.x Mail app:

  • Click on the More button (a circle with 3 dots inside) at the bottom right of the toolbar
  • Select Print
  • Chose the Microsoft XPS Document Writer "printer"

As far as I know email attachments are not embedded into the XPS file so there no way to open them from within the XPS Viewer.

mister wizard say:

Hello Dale,

You don't need to convert your files.  The Mp3 properties that you are asking about are called tags (short for "meta tags").  Unfortunately, Vista has limited support for editing them.  It sounds like you are using Windows Explorer to edit tags, which is not a good choice.  The preferred way is the Advanced Tag Editor in Windows Media Player.  Right-click on an Mp3 file in the Library pane and choose Advanced Tag Editor.

There are far better free tag editors available.  My favorite is Mp3Tag.  Google will find that download and others for you.

Doug M. in NJ
