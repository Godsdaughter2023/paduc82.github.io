You can try the following procedures below explain how troubleshot missing network or wireless icon in Windows 10:

Windows Key+I > Update & Security > Troubleshoot > Click Internet Connections > Run the Troubleshooter

1. Click the small up arrow icon on the taskbar to view hidden icons.
2. If the network or wireless icon is appearing there, simply drag and drop it to the taskbar area.

Turn on the Network or wireless icon in Settings
1. Open Start menu, click Settings icon to open the Settings app. The Windows logo + I keyboard shortcut can also be used to open Settings.
2. Click System icon. Click Notifications & actions.
3. Click Turn system icons on or off.
4. Check the status of the Network icon, and turn it on.
5. Click the Back arrow to go back to Notifications & actions. Click Select which icons appear on the taskbar.
6. Check if the Network is turned off, and turn on the same if it’s off.

Restart Windows Explorer
This solution is ideal if the network or wireless icon is not automatically updating to show the status of the internet connection.
1. Open the Task Manager by right-clicking on the empty area of the taskbar and then clicking Task Manager option.
2.If you see the Task Manager with restricted view, click More details.
3. Under the Processes tab, right-click on Windows Explorer entry, and then click Restart button.

And if there is no Windows Explorer entry, it’s because Windows Explorer process is not running. Open up This PC or any folder to see Windows Explorer entry in Task Manager.

3. On the right-side, look for the entry named Remove the networking icon. Double-click on the same to open its properties.
4. Select Disabled, and then click Apply button to see network or wireless icon in the system notification area of taskbar.
5. Sign out and sign in again to see the change. You can also restart Windows Explorer to apply the new setting.
