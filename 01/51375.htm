burrwalnut say:

You will not have to reinstall Windows because your browser has been hijacked. Work your way through this list:


1. Perhaps the Hosts file has been infected. Check its contents and/or change it. It is normally located at C:\Windows\System32\Drivers\etc\ If it is not there, look for its location in registry key HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Tcpip\Parameters\DataBasePath.

If the file is Read-only, right-click it, click Properties and remove the check mark. Make a backup copy of it, then using Notepad change its contents and save it (not as a .txt file, i.e. it has no extension). If you are unable to access the file, browse the Windows folder, right-click notepad.exe and choose Run as Administrator. The file contains a number of comment lines beginning with # followed by IP Address entries, e.g. 127.0.0.1 localhost, which refers to the IPv4 address of this, your computer. Entries beginning with ::1 localhost are the newer IPv6 equivalent of IPv4.

A line that begins with 127.0.0.1 stops access by this computer, so don’t remove entries like 127.0.0.1 MalwareWebSite.com as they help to protect the system from accessing malware sites. However, if you see entries with legitimate names like 127.0.0.1 microsoft.com or 127.0.0.1 ibm.com, you should remove them, as malware is trying to block access to them (using 127.0.0.1) or, more normally, redirecting them to another website (using n.n.n.n NastyVirus.com). Note that the immunisation feature of programs like Spybot can add many malicious websites to the Hosts file, making it quite lengthy.

2. There are a number of registry keys that can be infected and cause redirection. Here are a few of the major ones that should be checked for accuracy. Open a Run window (Windows Logo key+R), type regedit and press Enter. In the left pane navigate to HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\URL\DefaultPrefix. The (Default) field in the right pane should contain http://  It may be suffixed with something like ehttp.cc/, if so just change it and restart the computer.
 
Next, look in the right pane of HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Internet Explorer\MAIN for four keys named Search Page, Start Page, Default_Page_URL and Default_Search_URL. These normally contain a Microsoft address and must be considered suspicious if a different or unknown address is present.
 
Also, an equivalent HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main key may be present for Search Page and Start Page, ‘Secondary’ keys may be present as well. All these keys should be checked for correctness, i.e. legitimate website addresses that the user has created.

3. If you are still being redirected, download/run HijackThis, from here http://download.cnet.com/Trend-Micro-HijackThis/3000-8022_4-10227353.html  and delete any redirection entries beginning with 01, 13 or 17.
 
4. If none of the above resolve the issue, go to the Malware section of this site but thoroughly read the posting instructions first http://aumha.net/

Yeah the life span for external ones are actually lower because they get moved around.  Inside the drive the plates are spinning at over 7200 RPM, and they wear out over time.  Now some last longer, some die younger, but the average is 4 years.  You can read the details in the links (or even on the website of the company that built the drive: they are required by law to publish their statistics).  But on average, 80% of drives last 4 years.  That means 20% of drives die in that time, and it only goes up from there.  That's why you should always, always have backups of anything you care about.

don cuthbert say:

Boot with your install media or Windows PE. Get a Command Prompt (SHIFT-F10 from first screen of install media setup).

Before you run your export from WIM to ESD, create the folders \Recycler\Scratch on the partition that has 900GB empty. When you run the export command, add /ScratchDir=C:\Recycler\Scratch or use whatever drive letter has the 900GB space. You need a drive with minimum 1024MB for a Scratch directory.  Windows PE does not have 1024 room in its RAM disk (Drive X:).

 System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files. This article describes how to run the System File Checker tool (SFC.exe) to scan your system files and to repair missing or corrupted system files in Windows.
