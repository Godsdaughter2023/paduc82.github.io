Hi Jonathan,
Thank you for posting your query in Microsoft Community.

 

From your issue description I understand that you found taskhostex.exe in task manager and you want to know what it is.

In order to help you better please provide us the following information:

1)   Are referring to Taskhost.exe or taskhostex.exe?


Taskhostex.exe is a Part of Host Process for Windows Tasks by Microsoft
Host Process for Windows Tasks (TASKHOST) is a generic process which acts as a host for processes that run from DLLs rather than EXEs.

 

At startup TASKHOST checks the Services portion of the Registry to construct a list of DLL-based services that it needs to load, and then loads them.

 

Taskhostex.exe is an instance of a running program. This 32-bit program executes with the privileges as the currently logged in user account.

 

This file is part of Microsoft Windows Operating System. Taskhostex.exe is developed by Microsoft Corporation. It’s a system and hidden file.
Taskhostex.exe is usually located in the %SYSTEM% folder and its usual size is 54,272 bytes.


The taskhostex.exe process is safe and disabling it can be dangerous, because programs on your computer need it to work correctly.
