How To Pin Spreadsheets to Program Icons In Windows 10

I've seen a million questions about how to pin a file to taskbar or start menu in Windows 10. I wanted to share something that I found. Did you know you can pin individual spreadsheets to the Excel Program icon context menu so that wherever the program icon appears you have access to  your spreadsheet, including in the Start Menu or on the taskbar?

  1. Drop a shortcut of a spreadsheet on your desktop.
  2. Rename it to what you want to have appear within the program context menu. This is the menu that appear when you right click a program icon in Start Menu or on Taskbar.
  3. Make sure you either have the Excel open or already pinned to your taskbar. As long as Excel is currently showing up on your taskbar.
  4. Click and drag the shortcut from your desktop onto the top of the Excel icon on the taskbar until you are asked if you want to pin it. Say yes.
  5. To test, right click any Excel program icon in order to get the context menu to appear. You should see your spreadsheet shortcut in the menu now.
