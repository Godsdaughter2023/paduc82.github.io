lead3 say:

 

Windows 7 actually has two desktops.

One for the user and one for all users (public) and they are displayed as one desktop to each user.

You need to put the folders you want all users to see in the Public desktop.

This is a hidden folder in C:\Users\Public

Go to the Control panel and select Folder Options.

Select View.

Select Show hidden files, folder and drives.

 

lead3 say:

Windows 7 users two folders to make up each Desktop (what you see)

One folder is called Public Desktop, the other is the users Desktop folder.

Windows then takes the contents of both those folders and creates the Destop that you see.

Therefore, what ever you put in the Public desktop folder will appear on each users desktop.
