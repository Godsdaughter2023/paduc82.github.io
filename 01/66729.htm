¡firedog say:

Each type of folder has its own layout, so there is one layout for incoming POP3 mail, one for incoming IMAP mail, one for outgoing mail and so on. All storage folders except Sent items adopt the Incoming POP3 layout, so you see the uninteresting From: name instead of the interesting To: name for an outgoing message. Two workarounds:
(a) on the View tab, select One-line view under Message list. You then have control over which message attributes to display in which order, so you could select both From: and To:
(b) keep sent messages in the Sent items folder under storage folders. This is admittedly not particularly helpful if you're trying to consolidate incoming and outgoing messages by topic, say.
