Windows 8 : Installing Samsung ML-2010 Printer connected to a - Microsoft Community

vgzc say:

Windows 8 : Installing Samsung ML-2010 Printer connected to a print server

(Information dated 2013/01/27)

As of now, Samsung does not appear to have a downloadable Windows 8 driver for this printer (they do for Windows 7/Vista).  Their Universal Printer Driver could possibly work, but its search function does not seem to find printers installed on a print server.

Here are the steps I followed to get this printer installed - I thought I will post it in case it helps others:

1.  Connect the ML-2010 DIRECTLY to the PC running Windows 8 via USB cable  (you may need to be logged in as administrator)

2.  Windows 8 should automatically install the printer in a couple of minutes.   Look in your "Devices and Printers" to verify that "Samsung ML-2010 Series" appears.

3.  Once step 3 is completed, the ML-2010 driver has been installed.

4.  Now if you want, you can move the printer to another PC and use it with "printer sharing", or connect to it via a Network Print Server.  Then install that indirectly connected "new" printer.    Do not remove the original printer you installed via USB while you are doing this.  If/when you are asked for a driver for the "new" printer, you can pick "Samsung" ans "ML-2010 Series" in the manually select dialog box.  After this has been successfully installed and working, you can optionally remove the original "USB" printer.

5.  Just remember that at least one installed printer must be using this driver  - else it will get de-installed aand you will have to go through the whole process again.

elmedi say:

Never mind, with a little more diggin' I found the solution for my problem!

 

Detailed instructions:

Open a window, and in the address bar / location bar type in the following:

%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Themes\

Then delete TranscodedWallpaper.jpg , and try changing the theme again!

If this didn't work, you might have to delete the (hidden file) slideshow.ini as well,  but then you'll have to restart your PC.

After you restart you're probably free to set a new wallpaper/slideshow/theme.

Good luck!

PS: to enable your PC to view the hidden file slideshow.ini

-> press Alt + T
-> select Folder Options 
-> View
-> Click the radio box "Show hidden files, folders or drives"
-> Uncheck "Hide protected operating system files (Recommended)"!
