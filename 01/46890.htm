edjay rust say:

Thank you for keeping us updated, Louis. 

Just for clarity, when Windows 10 executed an update (like what happened to your Surface recently), it will create a folder named as Windows.old.

Windows.old folder is created when you upgrade from one version of  Windows to another. This folder stores all the files and data you may need in case you need to rollback to the previous version of Windows using the recovery options.

This makes perfect sense in case you encountered a problem with your current Windows version, you'll still have an option to go back to a point in time when your device is working properly. Windows will automatically delete the Windows.old folder to free up space after a month.

kevin-- say:

How you add a program depends on where the installation files for the program are located. Typically, programs are installed from a CD or DVD, from the Internet, or from a network

 

To install a program from a CD or DVD:

·        Insert the disc into your computer and follow the instructions on your screen.

 

 To install a program from the Internet:

1.      In your web browser, click the link to the program.

2.      Do one of the following:

o   To install the program immediately, click Open or Run and follow the instructions on your screen. 

o   To install the program later, click Save and download the installation file to your computer. When you are ready to install the program, double-click the file and follow the instructions on your screen.

 

 To install a program from a network:

1.      Click the Start button, click Control Panel, click Programs, click Programs and Features, and then, in the left pane, click Install a program from the network.  

2.      Select a program from the list, and then click Install.

3.      Follow the instructions on your screen.
