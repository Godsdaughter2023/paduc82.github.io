Method 1:

Check whether the Windows Installer service is set to a value of Disable.

To verify the Windows Installer service properties, follow these steps:

a) Open the Service control panel by typing “services.msc” without the quotes on the Start Menu.

b) Right-click on Windows Installer and then select Properties.

c) Verify that the service is set to Manual Startup type and that the path to the executable is

c:\Windows\system32\msiexec /V. (Where C is your system drive)

d) Verify that the service is set to Log on as the Local System account on the Log On tab.

 

Method 2: Reinstall the Windows Installer

To reinstall the Windows Installer, rename the damaged Windows Installer files, and then reinstall the Windows Installer. To do this, follow these steps:

a) Click on Start and type “cmd” without the quotes.

b) Click on “cmd”.

c) Type each of these commands and press "Enter" after every one:
cd %windir%\system32

attrib -r -s -h dllcache

ren msi.dll msi.old

ren msiexec.exe msiexec.old

ren msihnd.dll msihnd.old

exit

d) Type "exit" at the command prompt to close the Command utility.

e) Restart the computer.

 

Method 3:

SFC Scan

a) Click Start, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow

b) Type the following command, and then press ENTER: sfc /scannow
[The sfc /scannow command scans all protected system files and replaces incorrect versions with correct Microsoft versions.]

c) A message will appear stating that 'The system scan will begin'. Be patient because the scan may take some time

d) If any files require a replace SFC will replace them. You may be asked to insert your Win7 DVD for this process to continue

e) If everything is okay you should, after the scan, see the following message "Windows resource protection did not find any integrity violations"

f) After the scan has completed, close the command prompt window, restart the computer and check.

g) Check the below mentioned link for further guidance on SFC scan –

http://support.microsoft.com/kb/929833

Hello, bru62

Under the default admin settings that STV0726 described, with UAC enabled, you are essentially running as a Standard user and the prompt is for running as administrator so for the most part, there is little difference under normal usage.

That being said, to clarify a few points, most programs do NOT need to be ran as an administrator and for those that do, you will see a prompt automatically.     I FOUND A FIX FOR OLD GAMES ON SURFACE PRO WITH SMALL SCREEN - Microsoft Community

    davewright1234 say:

    I FOUND A FIX FOR OLD GAMES ON SURFACE PRO WITH SMALL SCREEN

    I installed a couple games on my surface pro. The games worked fine except the game opened up in a small box on my screen when they opened up. I tried all the settings to make the games go full screen. Nothing worked. I got an agent at Microsoft tech support on the phone and we spend several hours trying to figure anything out to make the games work full screen. nothing worked.

     

    While on the phone with Microsoft, I accident I figured it out. Its fairly simple but it works. I decided to post this so if anyone else has this problem they will know what to do.  I tried it on a couple different games and it works. Star craft, Heroes 3, etc should work just fine full screen on the Surface Pro.

     

    Follow these steps:

     

    1st, make sure you are in desktop mode and that a shortcut of the game is on the desktop to start out. Right click on the game and go to properties. Go to the compatibility tab and make sure all boxes are unchecked EXCEPT the "disable display scaling on high DPI settings". Click Apply/OK and then go back to desktop.

     

    2nd, open up the program on your computer. At this point, the game should open up on a small box surrounded by black.  Once you are on the menu or in the game, press the following buttons: 

     

    CONTROL+ALT+FN+F11 press this 2 times or until the game is full screen. When I first did this, most of my screen went black. Its ok. If it looks like there is a glitch, its ok. Once you have it full screen, then you can press the escape or menu button and the graphics should clear and the game should be working full screen. You are half way done.

     

    At this point, you have your game full screen. Then you exit the game. Don't press the windows key to exit. You actually need to close the game from the game menu. When you do, it takes you back to your desktop. But, your desktop is large and does not have full screen resolution. IT is ok. Next, right click on the desktop and go to screen resolution. Click on the resolution and take it up to the top again and click apply.

     

    After you change this and you exit the screen resolution, you should be on your desktop with normal resolution. You are now done with this game. Now when you go into the game, it is full screen. When you exit the game, the desktop also is normal resolution too.

     

    If you have multiple games that do this, you must follow this process with all the games. I only tried 2 separate games after I learned this and I had to do this process on both. Now both off them work great without the problem occurring.

     

    Hopefully this helps. If you have questions, please do not hesitate to post and ask them. Again I am posting this so it can maybe save you some time.

     

    Good luck.
