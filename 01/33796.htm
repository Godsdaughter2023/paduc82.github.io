can i add a image to my email signature - Microsoft Community

ray luca say:

can i add a image to my email signature

I CREATED A EMAIL SIGNATURE FOR ALL MY EMAILS , HOW DO I ADD A IMAGE WITH THE EMAIL SIGNATURE THAT EVERY ONE WILL SEE? THANKS.  (removed for privacy)

a. user say:

Hi,

 

If you are using Windows Live Mail as your email client, you need to create a HTML Signature Source File first. Here's how to do it:

 

1. Open up your email program.

2. From the File menu, click New and then Mail Message. The New Message window opens.

3. From the Insert menu, select Picture… (Live Mail users select Insert | Image | Inline…)

4. Navigate to the image file you wish to insert.

5. Click the Open button. Your image should show in the New Message window.

6. Optionally, you may add text.

7. From the File menu, select Save As…

8. Type your File name and select HTML as the file type.

9. Close the open window.

 

Afterwards, you need to link the HTML file to your Email Signature. Now that the file with your image and text is saved, you need to point your email program to it. Here's how to do it:

 

1. From the Tools menu, select Options.

2. Click the Signatures tab.

3. In the Signatures section, click the New button.

4. In the Edit Signature section, select the radio button for File.

5. Click the Browse… button.

6. Navigate to the HTML file you just created. You may need to change the file types list to HTML Files.

7. Highlight your file and click Open.

8. Check your desired Signature settings.

9. Click OK.

 

If you are accessing your account using Hotmail.com website, here's how to add a photo in your signature in Hotmail:

 

1. Log in to your Hotmail account and click "Options" in the top-right corner of the page.

2. Select "More Options" from the drop-down menu.

3. Click the "Message font and signature" link, under the Writing email category.

4. Open a new browser tab or window and navigate to the page that contains the image you want to add to your signature.

5. Right-click the image and select "Copy image."

6. Return to the tab with your Hotmail signature page. Place the cursor where you want the image to appear in your signature, right-click and select "Paste."

7. Click "Save" to exit the signature editing window.

 

Best regards.

1. Right-click "Speakers" icon in the taskbar and then select "Playback devices"

2 . Under "Playback" tab, right-click "Speakers", and select "Properties".

3. Click on " Enhancements" tab under "Speakers Properties". Check " Disable all enhancements " check-box .

By the above steps, you could resolve the problem that " audiodg .exe" process hosts high CPU usage and prevent your machine from slow running and fake crash.
