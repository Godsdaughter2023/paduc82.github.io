The Internet Archive site: https://archive.org/

Technical Level : Basic

Summary

A frustrating fact of life when using the internet is that web pages and sites die, for various regions.  The owner may no longer want to pay for the site or they site may be re-organized so your old link no longer works.

What do you do if you have a link to a page that no longer works.

The "Internet Archive" website, aka the "Wayback Machine" (a 1960's cartoon reference) is one tool you can try. This site uses "spider" tools to "crawl" the internet and take snapshots of sites which it saves, and gives you access to later.


Details

You are given a link to an article, but when you try it you get any one of a number of messages saying the page cannot be retrieved. What do you do now?

First thing you can try is using google or any other search engine to see if you can find the new location of the page that way.

If that does not work, you can try the Internet Archive site: https://archive.org/

Near the top of the page is title, "Internet Archive Wayback Machine" and an input box.  Paste your "bad" web address into the input box and click on the "Browse History" button.

Note the green arrow pointing to the input box where I pasted the address

You will get a result page that looks something like this:

You can click on any of the dates with the blue circle to see what information was archived.  If you are lucky, it is the missing page.

    Start your system, then just after the Manufacturers logo disappears and Windows 10 tries to boot, press and hold the power button down for 5 - 10 seconds to perform a hard shut down.

    When you are at the command prompt run each of these commands and press return after each one :

    Bootrec /fixMBR
    bootrec /fixBoot
    bootrec /rebuildBCD

    When the commands have run, type exit

    Reboot and hopefully you will be good to go . . .
