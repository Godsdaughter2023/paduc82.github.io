Yes, it is something that you should worry about.  If the volume is dirty it means that there are inconsistencies or errors in the file system and a proper chkdsk needs to be run on the drive.

When the volume is dirty by default it should be checked the next time Windows is booted.  If autochk doesn't run when Windows is booted maybe 'someone' changed the default behaviour of the operating system.  To return the system to the defaults enter the following command at a command prompt:

CHKNTFS /D

You can also schedule a boot time check by running this command:

chkdsk c: /f

Hi John,

No need, I found a solution. I'll post it below in-case anyone else stumbles across this thread.

Just in case anyone else ever has the same issue. You need to delete the affected User Profile from C:\Users. In this case; Delete: C:\Users\Dominic

Then clear the registry entries for the User Account under ProfileList and ProfileGuid. The Registry entries will be found at:

HKLM\Software\Microsoft\Windows NT\Current Version\ProfileList

Look for the affected User Account and Delete the S-1-5-21-XYZ Folder (Where XYZ is the menu as per the ProfileImagePath marked by the Red Arrow)

The corresponding Key to Delete under ProfileGuid is the same as the Guid Registry Entry under the ProfileList for the affected User Account.

Once that is done, Reboot the machine and setup Work Folders again and the problem will disappear.

Laptop vs Desktop- what are the pro's and con's? - Microsoft Community

jamesjoey say:

Laptop vs Desktop- what are the pro's and con's?

I'm considering going with a laptop for the first time. I'm tired of the clutter of wires and the non-mobility.

I understand I can access the internet with a router for wireless connection but other that that I cant seem to find any other differences.

ken blake - microsoft mvp say:

I'm considering going with a laptop for the first time. I'm tired of the clutter of wires and the non-mobility.
I'd appreciate if someone can give be pros and cons of Laptops form personal experience.
I understand I can access the internet with a router for wireless connection but other that that I cant seem to find any other differences.

Thanks,

James


First, note that you can have a wireless internet connection with *either* a laptop or a desktop. That's no difference at all.

 

Here are the major differences, in no particular order:

 

1. Laptops are considerably more expensive.

 

2. Laptops are harder to fix if something goes wrong, and service on them is more expensive.

 

3. Laptops break and have hardware problems more than desktops do.

 

4. Replacement parts for laptops are more expensive, and service often costs more..

 

5. Your ability to upgrade laptop hardware is much more limited than with a desktop.

 

6. A laptop screen is smaller that monitors that are available for desktops.

 

7.. Laptop keyboards are sometimes smaller than desktop keyboards.

 

8.  It may nnot be possible to upgrade a laptop to a new version of Windows. because drivers for that version may not exist.

 

9. Laptops are stolen more than desktops are.

 

10. Regarding the clutter of wires, almost the only wires you need on a desktop and not on a laptop are those for a monitor. A desktop can have a wiresless keyboard and a wireless mouse.

 

As you might guess from the above, for almost everyone I recommend that you have a desktop as your main computer, not a laptop. Almost the only advantage of a laptop is its portability, so it's good for use when traveling. So, for example, I have a desktop for my main use at home, but also have a laptop that I take with me when traveling.

liza.z say:

Hi,

 

If you start Windows and see a blank or black desktop with no icons, video adapter problems are the most common cause of this issue.

 

Try the following troubleshooting steps:

 

Method 1: Gain access to your computer and reinstall your current video driver

 

Step 1: Launch Task Manager and restart Explorer.exe:

 

1. Press the Ctrl + Alt and Delete keys on the keyboard at the same time

2. Click the Start Task Manager

3. Click the Processes tab, right-clickexplorer.exe from the list of running processes

4. Click End Process to stop the Explorer.exe process

5. Click File from the menu and click New Task (Run)

6. Type Explorer.exe in the Open: window and click OK

 

Step 2: Uninstall your video driver and allow Windows to re-install it:

 

7. Click Start and type Device Manager in the Search box

8. Click Device Manager under Programs

9. Restart your computer to allow Windows to reinstall the driver

 

Note: Once the driver is reinstalled you may need to reboot your computer again to complete the installation. Additionally, you may also need to adjust the screen resolution to its previous setting.

 

Method 2: Check for registry issues

 

If reinstalling your video driver does not work, follow the steps in method 1 again and then perform the following troubleshooting steps to check for registry issues:

 

1. Click Start and type regedit.exe in the Search box

2. Click Regedit.exe under programs

3. Browse to the following registry key

4. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

5. Click to select the \WinLogon key

6. Browse through the list to\Shell

7. The value of Shell should be explorer.exe

 

Note: If the value of Shell is not Shell, double-click Shell and type explorer.exe in the Value data: text area and clicklock and then close the registry editor and restart your computer

 

This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:

322756 (http://support.microsoft.com/kb/322756/ )

How to back up and restore the registry in Windows

 

 

Method 3: Enable low-resolution video (640x480), and then update your video adapter driver

 

If resolving registry issues does not work, perform the following steps to enable low-resolution video:

 

1.       Click the Start button, click the arrow next to Shut Down, and then click Restart.

2.       Press F8 before the Windows logo appears.

3.       On the Advanced Boot Options screen, use the arrow keys to highlight Enable low-resolution video (640×480), and then press Enter.

4.       Log on to your computer with a user account that has administrator rights.

 

If you can see your desktop, there's a problem with your video adapter. Try updating your video adapter driver. For more information, see Update a driver for hardware that isn't working properly.

 

Method 4: Start your computer in Safe Mode and disable your video adapter

 

If enabling low resolution and then updating your video adapter driver did not work, perform the following troubleshooting steps to disable your video adapter driver:

 

Start your computer in safe mode

http://windows.microsoft.com/en-US/windows7/Start-your-computer-in-safe-mode

 

Note:  Safe mode starts Windows with a limited set of files and drivers. Startup programs don't run in safe mode, and only the basic drivers needed to start Windows are installed

 

When your computer restarts, if you can see your desktop, there's a problem with your video adapter. Try updating your video adapter driver. For more information, see Update

It can also make changes to scripts, batch files, and other types of executable files created under any user account on your computer. System Restore does not affect personal files, such as e-mail, documents, or photos, so it cannot help you restore a deleted file.
