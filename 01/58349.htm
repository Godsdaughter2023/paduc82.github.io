The TCP/IP stack Winsock settings in Windows 8 may get corrupted, causing errors and problems with Internet connectivity. Let us follow these methods.

 

Method 1.

1. Press (Windows key + X) and select “Command Prompt (Admin)”. This will open command prompt in elevated mode.

Method 2.

1. Press (Windows key + X) and select “Command Prompt (Admin)”. This will open command prompt in elevated mode.

2. Type netsh winsock reset in the Command Prompt, and then press the Enter key.

3. Restart the computer to check if this helps.
