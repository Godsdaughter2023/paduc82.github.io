    How do I schedule a weekly backup to Microsoft OneDrive??? - Microsoft Community

    jasonjohnson5150 say:

    How do I schedule a weekly backup to Microsoft OneDrive???

    How do I schedule a weekly backup to Microsoft OneDrive???

    a. user say:

    The idea of using OneDrive is to sync with it. You don't want to use OneDrive for once-weekly backups.

    In the first place, you only get 5 GB of OneDrive space for free. That will fill up quickly with weekly backups. But let's say you have an Office 365 subscription or you're paying for extra space on your own.

    Most people's upload speed is a small fraction of their download speed. (Verizon FIOS is a notable exception.) It will take many hours to backup to OneDrive. That makes no sense. And what happens to your backup if your internet connection gets cut off?

    Finally, what good is your backup when you really need it but you're in a place with slow or no internet?

    Yes, your backups should be stored somewhere that's not in your computer, but cloud storage ain't the place. An external USB hard drive is perfect for the purpose: fast, cheap and easily portable. NAS is also good, if you have one, for the same reasons.

    Having said all this, cloud storage is a good place to archive your most critical or precious files. For example, my wedding video is backed up several ways and one copy is on Google Drive. (Sorry Microsoft!)

    1. Apart from Windows Update issue, is the computer working fine?
    2. It shows “Updates were installed: Never”, are you getting this issue ever since you got this computer or it started after some resent significant changes?

    There are several reasons why the installation or finalization of one or more Windows updates hang. Most often, these types of problems are due to a software conflict or a preexisting interrelated issue that simply wasn't brought to light until the Windows updates started installing.

    Let’s try these methods and check if it helps in fixing the issue.

    Method 1:

    First, I would suggest you to run Windows Update troubleshooter to get the problem automatically detected and fixed.

    Refer the section “My PC stopped responding while checking for or installing updates” and click the hyperlink “Windows Update troubleshooter” to download the troubleshooter and run.

    Troubleshoot Problems with Installing Updates.

    http://windows.microsoft.com/en-us/windows/troubleshoot-problems-installing-updates#1TC=windows-8

    Proceed with the next step if issue persists after running Windows Update troubleshooter.

    Method 2:

    Let’s rename Windows folders pertaining to Windows update to clear the previous update data and start the update afresh.

    Follow the steps below to do so:

    Step 1: Stop services responsible for Windows Update

    1. Press Windows Key + X.
    2. Click on ‘Command Prompt (Admin) to bring up elevated command prompt. Note: Click ‘Yes’ if it prompts for permission.
    3. At the command prompt, type the following, commands and then press ENTER after each command:

      net stop wuauserv
      net stop bits
      net stop cryptsvc

    4. Please do not close the Command Prompt window.

    Step 2: Rename folders pertaining to Windows Update.

    At the command prompt, type the following commands, and then press Enter after each command:

    ren %systemroot%\System32\Catroot2 Catroot2.old

    ren %systemroot%\SoftwareDistribution SoftwareDistribution.old

    Step 3: Restart the services responsible for Windows Update

    At the command prompt, type the following, commands and then press ENTER after each command:

    net start wuauserv

    net start bits

    net start cryptsvc

    Close the Command Prompt window and check if you can install the updates.

     

    Method 3: Run the DISM tool and check if it helps.

    1. Swipe in from the right edge of the screen, and then tap Search.
      (If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, and then click Search.)
    2. Enter command prompt in the search box.
    3. In the list of results, swipe down on or right-click Command Prompt, and then tap or click Run as administrator.
    4. In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

      DISM.exe /Online /Cleanup-image /Scanhealth
      DISM.exe /Online /Cleanup-image /Restorehealth

    5. To close the Administrator: Command prompt window, type Exit, and then press Enter.
    6. Run Windows Update again and check.

    Hope this helps. Please let us know the results, feel free to write us back for any further assistance, we’ll be glad to assist you.
