Windows XP Pro is a 32-bit operating system and cannot see anymore that 3 (and a bit) Gbs of RAM. The reason I'm stating and a bit is because hardware system overhead (e.g. on-board video, etc) uses some RAM and this number fluctuates.

transfer window key/license from one pc to another

i have own a laptop with windows 10 pro upgraded from win 7, is it possible to transfer the license of this PC to another which have different hardware configuration.

rdrewd2 say:

If it is an "upgrade license", as in a free upgrade from another version of Windows, then the answer is no. That Windows 10 license is only valid on that machine and can't be transfered to another PC. Your license is tied to motherboard on the machine that was upgraded.

andre da costa say:

It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.
