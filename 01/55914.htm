cmdrkeene say:

The fan's bearings may be coming loose, causing a loud sound.  Or the fan vent holes might be dirty and causing the computer to get too hot and cause the fan to run faster. 

Your Windows system doesn't control the fan -- it's entirely based on a thermometer sensor inside the computer.  So if the fan is fast, it's likely that the computer is hot.

If you are already sure that the fan vent holes are clean, here's how you can find out if a running program is causing the computer to get hotter than normal:  Press Ctrl-Shift-Escape, then look at the CPU column and see what program is using a lot of CPU.  To run cool, your CPU should be under 2% used.  For example, as I write this post my CPU is only 1% used even though I have hundreds of processes running.  That means my CPU is "idle" and cooling off 99% of the time, and only heated/working 1% of the time.
