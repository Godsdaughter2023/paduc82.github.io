bhushan timilsina say:

The files can't be opened.

 

I have found solution to this problem on my own. Actually, I was frustated waiting Microsoft guys find solution for me. I have elaborated my problem, scenario and solution to help you in case you have exactly same problem as of mine. Read carefully, whether your problem is exactly same or similar. Also see previous posts for better overview of problem that I was facing earlier.

 

PROBLEM DESCRIPTION


When users of Windows 7 tries to open executable file(.exe) from some programs that opens .exe file from its UI, Windows stops execution and shows error: "The files can't be opened" Your Internet Security settings prevented one or more files from being opened. After this you can not continue to execute your program. It seems as if any virus or malware is persisting on user's computer and preventing execution of required file. Though, this kind of error might have been caused by overwriting of Internet Security key in the registry by some malicious code or any 3rd-party software installation.

 

SCENARIOS

  •  Open "explorer.exe" from Windows Task Manager(WTM) > Run.
  •  Click [Show all processess from all users] button in [Processes] tab of WTM and nothing happens.
  •  Right-click a process in Resource Monitor and click [Search online...] 
  •  You see added security zones (My computer, or so) in Internet Security Options [Internet Options-Security tab]. Default zones are Internet, Local intranet, Trusted sites, Restricted sites
  • Type 'mrt.exe' on Start menu search box and launch it.
  • Launch a downloaded program from any download manager program's UI directly.

 

SOLUTION

 

  • Create a new user account with Administrator privilege.
        * Open Control Panel -> User Accounts -> Manage another account -> Create a new account -  Provide Name of new account (eg. FIXME) and select [Administrator] option -> click [Create  account] button.
  • Switch user to FIXME.
        * Press [WinKey]+[L] ->  click FIXME account to login. It might take few minutes to prepare desktop for first time login.
  • Launch regedit.
        * Type regedit on Start menu's search box then select regedit.exe and press [Enter].
  • Navigate to: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings
  • Export Internet Settings key and subkeys to a file (eg. Exported.reg)
        * Right-click Internet Settings -> click [Export] -> select public folder (%systemdrive%\users\public) -> Give it name 'Exported.reg'and click [Save].  
  • Switch back to your original account.
        * Log off from FIXME user and login to your previous user account.
  • Remove the the Internet Security keys.
        * Navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings -> select Internet Settings, right-click it and click [Delete].
  • Run the exported.reg form saved location.
        * Type exported.reg on Start menu search box. You should see the file then click it to launch. Press [Yes] when prompted.
  • Check problem is fixed by trying scenarios above.

 

  • Thanks for everyone who helped me to find solution to this problem.

difference between WPS and WPA

i want to know these type of information ,which is use ful to operate  my machine.

lemp say:
Questions of this type are best answered by using your favorite search engine (e.g., Google or Bing).

Briefly, WPS or WiFi Protected Setup is a system for more-or-less automatically configuring security for a wireless network.  Although promoted by manufacturers of wireless equipment, this method is considered not secure.  See http://en.wikipedia.org/wiki/Wi-Fi_Protected_Setup

WPA or WiFi Protected Access is a wireless encryption standard.  It is no longer the most secure method of securing a home wireless network.  If your equipment supports it, use WPA2 instead.  WPA is, however, considerably more secure than the older and outdated WEP standard.  See http://en.wikipedia.org/wiki/Wi-Fi_Protected_Access

  1. Press Windows Key + X and select Command Prompt (Admin) from the menu.
  2. Once Command Prompt window opens, enter the following command: powercfg -devicequery wake_armed
  3. You should be able to see the list of all devices that are allowed to wake up your computer. Enter powercfg -devicedisablewake “device name” into Command Prompt to stop a device from waking up your PC. Remember to replace “device name” with a name of the actual device. It’s also important to keep the quotation marks, so do not delete them.
  4. You can also use powercfg -deviceenablewake “device name” command to enable certain device to wake up your PC, but it is optional. You can find the name of the specific device by using the Device Manager.

Keep us in the loop with the result.
