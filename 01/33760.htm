I began with the simplest of disabling my AV software,deleting it and reinstalling both the anti virus and windows live mail.... No dice.

I was messing around in files i shouldnt have been trying to locate the option to unhide the C:user\appdata\roaming file so i could look at a log of a computer game to figure out why it wasnt running correctly.  "C: user\%username%\appdata" properties, it some how didnt get to the windows live mail folder and resolve that.

Long story short
  1. C:\Users\&username%\AppData\Local\Microsoft
  2. right click to bring up properties for windows live mail
  3. uncheck the read only box in the general tab
  4. apply
  5. yes
  6. ok

hope this works for you
