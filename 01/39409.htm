Hi there!

Please do the following (one after one before trying to check if the problem is fixed):

 

1. Run your computer in Clean Boot Mode.

 - Click your start button.

 - On the search field type: msconfig

 - Hit the Enter Key to open it right away.

 - On the General Tab, select "Diagnostic Startup" and be sure ONLY "Load System Services" is checked. (Uncheck any other, the third one can't be unchecked this is normal)

 - Go to the Service Tab.

 - Look for a checkbox called "Hide all Microsoft services" and check it.

 - Hit the button "Disable all"

 - Hit the button "OK" it will ask for a restart.

 - Reboot your computer.

 

 2. Delete temporal files.

Copy and paste this in your Windows Explorer address field (Change YOUR USER with the name of your user account):

 

C:\Users\YOUR USER\AppData\Local\Temp

 

a) Use this keyboard combination to select all the items of the folder: Ctrl + A

b) Use this keyboard combination to delete all the items of the folder: Ctrl + Alt + Delete

(It can say you need admin rights to delete a file there, check "do the same for all conflicts" and select yes, if it says a file can't be deleted because it's being used, check "do the same for all conflicts" and select "skip")

 

Copy and paste this in your Windows Explorer address field and repeat the same steps we just did for the other folder:

 

C:\Windows\Temp

 

4. Stop Windows Update Services and rename the SoftwareDistribution Folder then start the services again.

 - Press your Windows key in your keyboard to open the Start Screen (it looks like a flag between Ctrl and Alt)

 - Type the following using your keyboard: cmd

 - Right click the result and select at the bottom "Run As Administrator"

 - On the black screen type: net stop wuauserv

 - DO NOT close the black Window, just minimize it.

 - Now go to C:/Windows and look for a folder called "SoftwareDistribution" (if you can't find it try to look for a guide on how to show hidden files and folders).

 - Right Click SoftwareDistribution folder and select "Rename", add a ".old" at the end of the name. (It should look like this: SoftwareDistribution.old ).

 - Bring back your cmd black window again, and type: net start wuauserv

 

3. Run "CheckSUR" commands in your computer.

- Press your Windows key in your keyboard to open the Start Screen (it looks like a flag between Ctrl and Alt)

- Type the following using your keyboard: cmd

- Right click the result and select at the bottom "Run As Administrator"

- This command has two parts, each can take up to 20 minutes and will seem to be stuck at 20% but they are actually working, in the black window type the following:

DISM.exe /Online /Cleanup-image /Scanhealth

And when that's done, this one:

DISM.exe /Online /Cleanup-image /Restorehealth

 

If after doing all of this the problem persists please refer yourself to this article, you might want to Refresh or Reset Windows 8 to an out-of-box state. I warn you this means you will lose all your applications.

Refresh or Reset your Windows 8 PC

 

Hope I've been helpful. Let us know if you managed to fix your issue.
