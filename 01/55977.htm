First, find the identifier for the computer as follows, which brings up all the boot parameters:

bcdedit /enum

Then taking the Boot Loader idenitifier, use it in the following command:

bcdedit /deletevalue "{default}" safeboot

The {default} parameter is the identifier found by first step. Also note that the " " around the identifier are required because of the non-standard characters { }.

Because this is a desktop and I am not as power conscious, I unchecked the first box on every device I found that had it.
Generally I found them on Generic USB Hub, USB Root Hub, and on the LAN.

Maybe someone knows a different way around this, but since I have done this, the game is now playable with Windows 8.1.
