lemp say:
If I purchase a bare bone computer, and I want to install Windows XP OS in this computer.             How can I get the official license for Windows XP?
That's the problem.  You can't.  Microsoft hasn't sold Windows XP for several years.

If you buy XP online, I suggest that you be certain of what license type you are buying (e.g., upgrade, OEM, System Builder, etc.).  There are different restrictions for the various license types.

The problem with returning the software bought online -- even from vendors who permit returns -- is that you may not become aware that you have a counterfeit copy for quite some time.  I have seen many posts from people who successfully installed Windows, had it activated, and even successfully downloaded updates from Windows Update ... only to get, many months later, the "you have counterfeit software" message. 

Microsoft is constantly adding pirated product keys to its database.  Thus, although a key may work for a while, it may suddenly become invalid.

Microsoft support for Windows XP will end next April.  Several years ago, I read somewhere that Microsoft was going to stop checking for valid product keys after the end of support.  I haven't seen anything like that recently, so I don't know if that will be the case.  Perhaps you are willing to take a chance that whatever you buy will remain valid until next April and that there will be no more checking after that.  It's your money.
