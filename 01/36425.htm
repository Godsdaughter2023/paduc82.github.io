Assuming that you have authorized access to the router/wireless access point, the best way to troubleshoot wireless network authentication problems is to access the router's configuration utility and disable all wireless security. 

Then go to the problem computer, get to the Wireless Network Connections Properties screen and select the "Wireless Networks" tab.  (screenshot ).  "Remove" at least the entry for your wireless network (if not all entries) and OK your way out. 

Now try to connect. 

If connection is successful, then you can reconfigure security on the router.  When you go back to the router, use the same encryption password as before (assuming that the router doesn't automatically remember it when you re-enable wireless encryption) so that the other wireless networked computers will still work.

If the connection isn't successful, then the issue isn't wireless security, so post back.
