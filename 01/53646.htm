tryx3 say:

Daniella,

It is possible that the WiFi networks are broadcasting on both 2.4GHz & 5GHZ wavebands & that all your other devices have "dual-band" capabilities but your laptop WiFi adapter does not and is therefore only able to see the slower 2.4GHz signal. 

Just for example -

  • My router & my other computers have dual-band 802.11ac WiFi,
  • My older computer has an older 802.11n WiFi card that is only single band,
  • So the older one can only connect to the slower 2.4GHz signal from the router,
  • But the others can choose either signal.

- - - - - - - - - - - - - -

Note that it is possible to hide individual networks using specialised commands but it is not really possible to do it by accident.  I do not think that you could have forgotten doing it deliberately.  The command you would have had to have issued [in an elevated command prompt] would have been

netsh wlan add filter permission=block ssid="NameOfTheWiFiNetworkToHide" networktype=infrastructure

You can, if you wish, confirm this -

Right-click on the Start button, select Windows PowerShell [Admin] / Command prompt [Admin], give the necessary Admin permission to proceed then enter

netsh wlan show filters

If you had blocked any then you would be shown a list of them but if you have never blocked any then each heading in the results will just have this under it

<None>

The Windows 7 Upgrade Option Program debuted on June 26th, it provides anyone who buys a PC from a participating OEM or retailer with Windows Vista Home Premium, Business or Ultimate to receive an upgrade to the corresponding version of Windows 7 at little or no cost. The Windows 7 Upgrade Option Program will be available until January 31st, 2010 and is global.

You would need to contact the OEM that you bought the systems from to find out if the option is still available to you.

As for doing an upgrade from Windows Vista Business to Windows 7 Professional, yes you can.

When making significant changes to your computer such as an upgrade, it is always recommended you backup your system prior to installing a new version of Windows. In your case, you can upgrade to Windows 7 depending on edition and architecture of Windows Vista you have installed. If you are running a 32 bit version of Vista and want to upgrade to a 64 bit version (edition of Windows 7), you will have to a do clean install, this applies vice-versa.

If you are running consumer editions of Windows Vista such as Home Basic or Premium, you can only upgrade to either Windows 7 Home Premium or Ultimate. Also, you cannot do downgrades, which means, if you are running Windows Vista Ultimate now, you cannot downgrade to Windows 7 Home Premium. You must do logical upgrades:

Windows Vista Home Basic > Windows 7 Home Basic, Premium or Ultimate
Windows Vista Home Premium > Windows 7 Home Premium or Ultimate
Windows Vista Business > Windows 7 Professional, Enterprise or Ultimate
Windows Vista Enterprise > Windows 7 Enterprise
Windows Vista Ultimate > Windows 7 Ultimate.
