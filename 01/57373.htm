  Movie Maker - How To Save As .WMV Movie File
======================================

Originally Posted by  Leonard_Su

1. Open Movie Maker and open the project file you
have (.wlmp)

2. In the program window, press these keys on your
keyboard ALT + F + select Save Movie and choose
Recommended for this project.

3. After that choose your preferred destination where
you want to save the movie.

4. Edit the movie file name and the file format of the
movie. (You can choose .WMV or MPEG-4 if you
are using the latest version of Movie Maker 2012. If
not you can only choose .WMV)

5. Click Save button.
