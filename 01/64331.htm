thefaz50 say:

I had the same problem, but I FIGURED IT OUT! I went from windows vista home premium to windows 7 ultimate, and my internet kept disconnecting. I had updated the drivers to the windows 7 drivers from realtek website, but it made no difference. Here's what I did to fix this problem with realtek rtl8187b wifi:

1. Download the windows vista drivers for realtek rtl8187b

2. Go to Control Panel-> Hardware and Sound-> Devices and Printers

3. Right click on the name of your computer and click "device installation settings"

4. Turn off auto driver updates from windows. Otherwise, windows will automatically reinstall the windows 7 drivers for realtek rtl8187b, which would cause it to have the same problem again.

5. Go to Control Panel-> hardware and sound-> device manager and find realtek rtl8187b under network adapters and uninstall the drivers.

6. Go to the location where you saved the vista drivers for realtek rtl8187b. Install the visa drivers for realtek rtl8187b. Driver date should be: 3/23/2009 and the driver version should be: 6.1159.323.2009

 

Now hopefully your internet is working properly again!

I've found the solution...

 

I use registry editor to remove the unnecessary folders, but...

BE CAREFUL If you delete something else with the Registry Editor, you may damage your computer.

 

Anyway, open the Registry editor and go to this path:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace

 

And delete every folder inside 'NameSpace'-folder

 

If you are not sure how to do all that stuff:

  1. Press 'Windows Key' + Q
  2. Type: regedit
  3. Open regedit as admin.
  4. Go to this folder: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\NameSpace
  5. Remove every folder inside NameSpace folder

To turn on network discovery, file and printer sharing, and public folder sharing in Windows 7

1.     Open Advanced sharing settings by clicking the Start button, and then clicking Control Panel. In the search box, type network, click Network and Sharing Center, and then, in the left pane, click Change advanced sharing settings.

2.     Click the chevron to expand the Home or Work network profile.

3.     Select the options to turn on network discovery and file sharing.

4.     Under Public folder sharing, do one of the following:

o    To share your Public folders so that people on other computers on the network can open files in them but can't create or change files, click Turn on sharing so anyone with network access can open files.

o    To share your Public folders so that people on other computers on the network can open files in them and also create or change files, click Turn on sharing so anyone with network access can open, change, and create files.

Click Save changes. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
