You cannot, and you do not, want to delete these files.  Please refer to this information I posted earlier:

***
"AMD64" is a generic identifier for 64-bit code in Windows.  There is the AMD64 code branch which includes both AMD and Intel processors, and there is the IA64 code branch which includes only Intel Itanium 64-bit processors.

Intel processors use AMD64 branch code because they use features and microcode licensed from AMD.  Intel's own IA64 processors were never very popular and are for the most part abandoned.
