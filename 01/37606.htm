Greetings,

Sorry you're having problems getting your patches installed. I have a number of things for you to look at.  First off, are your Windows Update and Background Intelligent Transfer Services set to Started and Automatic? You can check these settings by typing services.msc in the Search field on the Start menu.  

Since they are all Office updates, you might want to do a Repair function for Office.  To do this click the Start button, type Programs and Features and click the Enter key. Right-click on Microsoft Office 2007 and choose Change.  Choose Repair and Continue.  Microsoft Office will attempt to repair any issues it has (this will take a few minutes to run).

Sometimes in order to get patches to install properly, you need to do them manually:

1) Visit Microsoft's Download Center: http://www.microsoft.com/downloads/en/default.aspx.
2) Search for each update code which starts with the letters "KB" and followed by a set of numbers (i.e, KB979309).
3) Click the link for the download and then click Download and Run.  You'll need to accept the agreement and then click Continue.
4) Repeat this process for each download that Windows Updates was unable to install.
5) Run Microsoft update again, click Check for updates.  Hopefully it will show that you are completely patched.

If that doesn't work, try running Microsoft's OneCare safety scanner.  Sometimes it will resolve hidden registry issues, or can clear up viruses which can prevent installation of patches: http://onecare.live.com/site/en-us/center/whatsnew.htm. Click Full Service Scan, then Quick scan (complete scan takes a very long time to complete). Try the patches again.

Hope this helps get your patches installed.
