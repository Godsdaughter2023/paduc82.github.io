Step 1:

Change your Windows display language


To change your Windows display language (that's the language you see most often in Windows and apps), first add a language, download and install the language pack, and then follow these steps.

 

1. Swipe in from the right edge of the screen, tap Settings, then tap Change PC settings.
(If you're using a mouse, point to the bottom-right corner of the screen, move the mouse pointer up, click Settings, then click Change PC settings.)

2. Tap or click Time and language, and then tap or click Region and language.

3. Tap or click the language that you want to set as your display language.

4. Tap or click Set as primary to move the language to the top of the list. If the language can become your Windows display language, you'll see Will be display language after next sign-in appear under the language.

5. Sign out and then sign back in to finish.

 

Step 2:

Add a keyboard layout for a language

 

You can add a keyboard layout to a language if you want to use a different keyboard than the one that comes with the language.

To add a keyboard layout for a language

 

1. Swipe in from the right edge of the screen, tap Settings, then tap Change PC settings.
(If you're using a mouse, point to the bottom-right corner of the screen, move the mouse pointer up, click Settings, then click Change PC settings.)

2. Tap or click Time and language, and then tap or click Region and language.

3. Tap or click the language you want to add a keyboard to, and then tap or click Options.

4. Tap or click Add a keyboard, browse the input method list for the keyboard layout you want to use, and then tap or click it.

 

Make a keyboard your default keyboard

 

To set a keyboard as your default keyboard so you don't have to switch input methods to enter text with it, follow these steps:

 

1. Open Language in Control Panel by swiping in from the right edge of the screen, tapping Search (or if you're using a mouse, pointing to the upper-right corner of the screen, moving the mouse pointer down, and then clicking Search), entering Add a language in the search box, and then selecting Add a language.

2. Tap or click Advanced settings.

3. Under Override for default input method, choose the keyboard layout you'd like to be your default keyboard, and then tap or click Save.

I had the error "a required device is not connected, or cannot be accessed" just a bit ago.  After frantically using my phone to find a solution, -none of which worked -, I thought about it logically.. I was able to access the operating system in safe mode with command prompt.  I did this following someone's advice about using bootrec /fixmbr to solve the issue.  I thought about it more, and being as safe mode should be compatible with ANY hardware, I realized what was causing this issue, and EXACTLY how to fix it!

I recently rearranged my apartment, this took several days, but I still needed to use my computer, so I was using my laptop.  Not an issue until I had a head crash (the part of a hard drive that reads data slams into the disk inside and corrupts everything) This was a heartbreaking sound! I hope no one has to listen to this as the computer locks up and blue-screens.  I popped the solid state drive from my desktop in, and after a long boot (2-3 minutes), it started, and all my hardware was functioning perfectly.  I finished rearranging, and popped the drive back into my desktop, and was getting this error.  I didn't understand why this could happen when the OS was working perfectly less than 2 minutes prior.

After fiddling around with other people's suggestions, and them failing to work, I sat back and thought.  A laptop has unique hardware, and it probably has some sort of firmware capable of communicating with the OS to get the basic systems working.  A desktop on the other-hand, is different, you can swap motherboards, processors, RAM, PCI devices, ETC with great ease, and they all come with drivers specific for them.  Safe mode can use these devices on their most primitive level, but trying to boot normally, and all goes arry.  I decided that this error must be the fact that the OS didn't know how to use my WIFI card to download the drivers for my system.  I strung a 50 foot ethernet cable across my apartment, and after a 5 minute boot time, everything was good as new.  

The point of this, is that if you are changing hardware, or swapping a drive, and you have an issue like this, odds are, it is a driver issue.  If the OS doesn't know how to use the hardware, it can't, and it will simply fail to load.  Try to connect your system with the most legacy devices as possible.  This means no bluetooth keyboard, no WIFI, avoid HDMI, no non-stock SD card readers, get the system on USB keyboard(or preferably PS/2), Ethernet, VGA, and USB ports built into the motherboard.  These systems should get your computer to access the internet, with you being able to control the options via keyboard, and it will download the drivers needed to run the system as it should.

A computer can be smart, it can do phenomenal things, but it can't do what it isn't taught to do.  You have to enable it to use what it was already taught, to teach it to do what you need.  This is the basic premise of programs and applications.  The computer can run the latest games and render the latest graphics, but it needs to be taught how, with a program - drivers are programs that allow it to use the hardware that is installed.

Thanks for reading, and I hope this helps some people.  Keep in mind, this is only relavent if you have changed hardware, and usually if it is a big change, adding a HDD or some RAM probably isn't relevant, Replacing a motherboard, a CPU, or a graphics card is what can cause this.
