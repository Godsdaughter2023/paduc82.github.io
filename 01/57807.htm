The Boot Camp partition's operating system must be activated twice. Once while the Mac is booted into the partition natively, and once while it is running within a virtual machine. Refer to your operating system's documentation for the proper steps to accomplish this.

If you are still having issues with activation:
  1. Quit VMware Fusion
  2. Browse to Users/<user name>/Library/Application Support /VMware Fusion/Virtual Machines/Boot Camp/<your disk ID>/Boot Camp partition/.

    SMBIOS.reflectHost = "TRUE"

  9. Delete the VMware folder from C:\Program Files.
  10. Shutdown Windows Vista virtual machine and quit VMware Fusion.
  11. Launch the Windows Vista virtual machine.
  12. Click Virtual Machine > Install VMware Tools.
  13. After VMware Tools installation, reboot the Windows Vista virtual machine and try activating the operating system.

a. user say:

Hi Mark,

Thank you for your interest in Windows 10!

Yes, SFC scan and DISM work and log in the same way as in Windows 8.1. To validate the commands please refer to the following methods:

Method 1:

Run a System File Checker (SFC) scan to check for any file corruption.

  1. Press Windows key + X, click Command Prompt (Admin).
  2. In the Command Prompt, type the following command, and then press ENTER:
    sfc /scannow

Method 2:

Run the Deployment Image Servicing and Management (DISM) tool as follows.
  1. Open an elevated command prompt. Press WinKey + X, click Command Prompt (Admin).
  2. Type the following command, and then press Enter. It may take several minutes for the command operation to be completed.
    dism.exe /online /cleanup-image /scanhealth
  3. After you get message “The operation completed successfully” type second command and press enter. It will take some time to finish.
    dism.exe /online /cleanup-image /restorehealth
Important: When you run this command, DISM uses Windows Update to provide the files that are required to fix corruptions.
