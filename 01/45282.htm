 What makes you think you deleted a System partition?  Unplug the new drive and any flash media to see if it will boot.  On a UEFI install the Windows Boot Manager must always be set to boot first in BIOS setup.  Your cloning or imaging solution must support UEFI too.  The clone or image must contain EFI System partition, hidden MSR partition and C. 

If you want to repair it then  make the Easeus boot disk to and attach back a picture of the  Easeus drive map and listings.  Once we see what's missing we can tell you how to restore it.  If it's the 100mb  EFI System partition, you can recreate it FAT32 using Easeus' tools..  Then boot into the  Windows 10 Installation Media or Recovery drive to run Startup Repair to  make it bootable.  If the partitions are not missing then you might also be able to get it booting with Startup Repair.  Unplug the other drive.

It sounds like your media may be bad, so start over to Create Windows 10 Installation Media.

If you want to skip this and you know for sure the Upgrade was activated, then you can unplug the old drive and plug in the new one, boot the installation media to do a Clean Install Windows 10 .  Be sure to delete any partitions already on the drive using the Drive Options at the "Where do you want to install Windows?" screen. 

But if the Windows 10 Upgrade was not Activated then it will not activate a Clean Install.  You'll need buy a key to insert or do the Upgrade over after running Factory Recovery to 8.1 or doing a Clean Install - Windows 8 using the media provided at Create installation media for Windows 8.1 - Windows Help.  Again delete all partitions during the booted install to get it cleanest.  After install, with only Defender and your network LAN installed, fully Update, Activate, save a System Image Backup  then Install  Windows 10 using the media creation tool - Windows Help. 

If asked do not insert a Product Key at any time, just click Skip or Next. 

If you want to really maximize chances for a successful Upgrade, then choose to Save Nothing so the Upgrade has much less to choke on.  

To improve the Upgrade's chances turn off everything at Startup:
