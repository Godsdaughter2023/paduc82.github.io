Hi,

Before I give you the solution I am assuming that it is a 'Registry Problem'.

Solution:

Windows 10

1) You need a Bootable USB/DVD Windows 10. If you don't have the bootable Windows 10 USB/DVD please follow this guide to make one.

2) Insert Bootable Windows 10 USB/DVD, Start your PC and Press F9 or the key for the "Boot Device Order" which varies PC to PCs.

3) Select your bootable device and wait for the windows setup to start from the bootable device.

4) Select 'Repair System/Windows' from the window.

5) Go to "Advance Option"

6) Select "Command Prompt"

7) Navigae to C:\Windows\System32\config   by using command "cd /d c:\Windows\System32\config"  without "".

8) Now you have to rename 5 files (DEFAULT, SAM, SECURITY, SOFTWARE, SYSTEM) to *.old by using the command given below.

    ren DEFAULT DEFAULT.old

    ren SAM SAM.old

    ren SECURITY SECURITY.old

    ren SOFTWARE SOFTWARE.old

    ren SYSTEM SYSTEM.old

9) Now you have to copy the registry hive from "c:\Windows\System32\config\RegBack" to "c:\Windows\System32\config". To do that please type the command given below:

     xcopy c:\Windows\System32\config\RegBack c:\Windows\System32\config

       With this command, those 5 files will be copied to the location "c:\Windows\System32\config".

10) Close Command prompt and Shut Down properly and Start your PC.

smiling cat-fab725ba-e75f-400c-9250-ecd9955c9c75 say:

Well if you have a the newer controller with the built in 3.5mm audio port all you need is a male 3.5mm four pole TRRS to 2x 3.5mm 3 pole TRS female jacks. If you don't have that port you need the headset adapter. Quick search and found one on Amazon.com.  You can see in the customer galleries of how people are using it with the XBox One. With this setup you can just plug in your headphones into one port and whatever 3.5mm TRS standard microphone you want into the other. If you look around more you might find a cheaper cable or something similar you can pick up at a local retailer. The thing you need to make sure of is if the adapter cable is CTIA standard and not the OMTP standard. Sounds complicated, but all it means is CTIA uses ring 2 for ground and the sleeve for Microphone and it's reversed for OMTP. Here's the wiki page if that doesn't make enough sense it's under section 7.1.
