    I have noticed this as "info" in my Norton Security History. Is this a security concern, or a necessity for my OS to operate? If a concern, can it be disabled or uninstalled?

    Original title : "Connected to a public network, (Teredo tunnel adapter (::0))"

    a. user say:

    Hi Chas,

    Thank you for keeping us updated.

    Teredo is an IPv6 transition technology that facilitates IPv6 end-to-end connectivity between any IPv6/IPv4 hosts that are located behind one or multiple IPv4 NAT devices. When Teredo is used, the IPv6 data packets are sent as IPv4-based User Datagram Protocol (UDP) messages between the hosts. Norton Internet Security provides the Automatic Learn IPv6 NAT Traversal Traffic option to control the network traffic that use Teredo to communicate with your computer. By default, Automatic Learn IPv6 NAT Traversal Traffic is turned off. In this case, Norton Internet Security blocks all IPv6 NAT Traversal traffic. The Automatic Learn IPv6 NAT Traversal Traffic option is available only when Automatic Program Control is on. When you turn on Automatic Learn IPv6 NAT Traversal Traffic, Norton Internet Security allows IPv6 NAT Traversal traffic.

    I would recommend checking with Norton as suggested earlier in this thread to help you disable it if you want to.

    Hope this information is helpful. Please feel free to reply in case you face any other issues with Windows in future.
