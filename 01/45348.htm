Please uninstall Webcam driver by following steps below:

 

1. Click "Start" and then "Control Panel". After that Double-click "Device Manager."

2. Click the plus sign or triangular icon next to "Imaging devices." This expands the category, revealing the webcam.

3. Right-click on the webcam, and click "Uninstall."

4. Place a check in the box labeled "Delete the driver software for this device," and click "OK." This deletes the driver from the system. Close the Device Manager when the process is complete.

5. Re-open the Control Panel, and double-click "Programs and Features".

6. Check the list of installed programs to see if any programs are associated with the webcam. If webcam software is present, click it and then click "Uninstall" button.

 

After this, please install Webcam driver by using steps below:

 

Method 1:

If you have got a disk with that, Please insert disk in the disk drive and try to reinstall webcam driver by following on-screen prompts.

 

Method 2:

If you do not have disk, Please follow steps below:

1. Shut down all applications on your computer. This will help facilitate the installation of the webcam. Follow the manufacturer's instructions to reinstall the hardware.

2. Go to the manufacturer's website and find the driver software for your webcam. Click on the link and click the "Save" button to download the software.

3. Find the downloaded software on your computer and double-click the icon, if the software does not begin the installation process automatically.

4. Agree to the software license and designate a location for the software on your hard drive. Click the button to finish the installation and restart your computer.


I hope this fixes the webcam.

a. user say:

Hi,

 

Thank you for posting here in Microsoft Community.

 

I'm happy to help you with your concern.

 

If you're currently receiving the error "A problem occurred when opening the message”, it is possible the Mail.MsMessageStore file of Windows Live Mail is corrupted. Kindly follow these steps to repair it.

 

1. Restart the computer (this is to ensure that Live Mail is fully shut down and that none of its components are in use when the computer starts).

 

2. Click Start, type CMD but do not press enter.

3. You will see cmd.exe in the search results, right click on it and select Run as Administrator - type your admin password if asked, and if UAC pops up asking permission, allow it.

 

4. You should now see the command prompt, with the following displayed:

 

Microsoft Windows [Version 6.1.7600]

Copyright (c) 2009 Microsoft Corporation. All rights reserved.

C:\Windows\system32>

 

5. Copy the following (Ctrl+C) esentutl.exe -d "%userprofile%\AppData\Local\Microsoft\Windows Live Mail\Mail.MSMessageStore"

 

Note: the "" are necessary.

 

6. Right click in the command prompt Window, and select Paste, so you now see the following:

 

Microsoft Windows [Version 6.1.7600]

Copyright (c) 2009 Microsoft Corporation. All rights reserved.

C:\Windows\system32>esentutl.exe -d "%userprofile%\AppData\Local\Microsoft\Windows Live Mail\Mail.MSMessageStore"

 

7. Press enter. esentutl will now defragment the Live Mail database file.

 

8. Once done, you can close the command prompt, and restart the PC again.

 

If the issue still persists, please let us know.

 

Thank you.
