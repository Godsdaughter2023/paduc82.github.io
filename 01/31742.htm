If you use the version of IE in the Metro interface you cannot edit a SharePoint 2010 document.  You get the following message

The temporary workaround

 

In Metro

Start up the Desktop application

Start IE from within the Desktop Application

Now (within the Desktop version of IE) you can edit SharePoint documents.

 

Makes no sense, but for now it is a workaround.
