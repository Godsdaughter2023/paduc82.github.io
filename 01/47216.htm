cmdrkeene say:

Every folder remembers it's on specific icon layout, display, and icon size.

 

However (and this is a big however): libraries are not folders.  Libraries are basically live search results that combine files from many locations into one view for easy viewing and searching. For example, I have video files and movies on several drives (too many to fit on one).  But my Video library is set to include all of those, so I only have to open one place to get to all movies, regardless of where they are stored.

What's that have to do with views?  Well, the views in a library apply to the entire library. Usually this is no problem, you probably want all your video folders to have the same layout and size, and all your documents folders from the documents library to have their own layout.

If you access folders via a library, the library's view will be used.  The folders still remember their settings, but those settings aren't being used because you're actually opening a set of search results instead of the real folder.

 

Solution?  Open the folder directly by going to Start > Jeff (or your username, at the top right), the navigate into a folder directly.
