How do I export files (messages and contacts) back to Outlook Express?


Tricky as OE predates both WLM AND Windows Mail.

If you have access to an IMAP email account then that is by far and away the easiest option.

If not then the second easiest way as far as I can see is to download a Trial version of Office onto your Windows 7 machine.

EXPORT messages from WLM to Exchange (you get the option for Outlook on the second screen AFAIK).

Save the resultant pst file to external medium.

Also Export your Contacts into a csv file. Save that to the same external medium.

On the XP machine download and install a trial Office suite.

Open Outlook and attach the pst file (which you have copied from the external medium to your My Documents Folder). Then in OE import messages.

You can import the contacts csv file directly.
