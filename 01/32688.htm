cmdrkeene say:

The view will always be remembered if you go to a real folder, like one of these examples

 

  • Start Menu > Your Name (at the top-right)
  • Computer > C > Users > Name > Pictures
  • C:\Program Files\Microsoft Office
Any view you make to a folder like those will be retained every time you go to the folder -- no need to even save.

BUT, if you go to a library location, such as one of these, your view choice will not be remembered.
  • Start Menu > Pictures
  • Libraries > Pictures
  • Libraries > Music
Every time you open a library, the view will be reset to the default view that you last picked from the blue "Arrange By" drop-down menu at the top-right of the window and the last icon size/layout you chose while in that library.

Why?  Libraries are not places, they are actually saved searches.  In fact, a library is just a text file (you can open it in Notepad to see how it works) that says "show me all files from x/y/z places, arranged in some specific style.  It's literally defined in that file.  When you make changes to how a library view is displayed, you're just changing that library, and the change cannot be applied to the direct folder (because you're not in the folder, you're looking at an auto-search that happens to include results from the folder).


So... If you don't use libraries, you can just start your file browsing by clicking your Name at the top of the Start menu, which will preserve your choices.
