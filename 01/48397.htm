wunders say:

According to Microsoft:    "This is not a problem....  it is a feature"...

 

The normal behavior for Windows XP is to hide the built-in administrator account once another user account is created.  Do not rebuild your machine again because this will happen again.  One way to get around this is to create another administrator-capable user as well as an unprivileged user.  Then, when the built-in administrator account is hidden, you will then have a visible admin user to login as.

 

As it is now, your solution is simple.  At the normal-boot login screen, hit the Alt-Ctrl-Del key combo twice.  This will bring up a standard login screen where you can enter the username and password (if any) of the built-in administrator and log into your computer in the normal boot mode and make any changes that you desire.  (I recommend you create a 2nd Admin user with the Users Control Panel)

rgharper911 say:

No, you should be afraid to delete it because it is a Windows Image Backup of your computer, and if anything happened to your computer you would no longer be able to restore from the backup.

foxfifth say: The following link gives information on how to "Manage Space" using Windows 7 Backup and Restore:  http://windows.microsoft.com/en-US/windows7/What-backup-settings-should-I-use-to-maximize-my-disk-space  Deleting will not damage your computer but in order to have a complete backup it would be a good idea to create another system image backup in another location.

Your post mentions that the backup is on Drive D.  That drive letter is frequently (but not always) a partition of your main drive.  If that is true in your case, it is not a great place to store a backup because if your main drive has a hardware failure both your data and your backup of the data stored on the same drive will be lost.  Also consider the possibility of theft of the computer or a fire.  External USB drives are relatively inexpensive and resolve the possibility of loss due to hardware failure of the main drive and can resolve the other issues if kept in a separate location.
