a. user say:

So after reaching out to MSFT, the HD housing vendor and the eSATA card vendor and reviewing all open configuration, setting, driver update and connectivity issues I found the problem and the solution. 

I now have a fully working Win10 Pro (AMD based) pc/server hosting 2 each xGB drive arrays (1 via USB3 and 1via eSATA II) running WEB and DB services.  I've also tested this device on a Mac and it behaved the same on both systems.

The solution is not so technical that MSFT or the card vendor needed to provide a solution.  It was however one the HD housing vendor should have provided better support for.  The problem was caused by the hardware and or the software configuration tools of the HDD housing device.  

This issue was an American Media Device, VenusT5 mini (5 bay external HD enclosure max capacity eSATA 3GB) device software configuration.  

The device is delivered with software on the backplane of the HD enclosure that can use hardware switch based configuration to manage the drives as RAID or non-RAID devices.  In my case I had 5 1TB devices (I didn't need bigger) and until resolving the issue only 4 of the 5 would be available under every configuration I tried.

The AMD enclosure vendor supplies their device with a CD also for using software based configuration, as well as, the hardware switches without instruction on using them together or separately.  So I first configured the system with both the HW and SW configuration for 4 drives as a RAID10 array and 1 drive as a standard device.  No joy.

I then (after updating drivers and settings in both Win10Pro and the eSATA card) tried every combination I could using their HW and SW.  No joy!

So I finally tried resetting the device to no HW or SW based RAID setting (all individual drives).

Presto joy to be had!

So the answer is:

If you want to use 2.5" HDD in a portable drive array container that connects via an eSATA interface and can deliver up to 3GB transfer to the devices (up to 3TB per drive) and have the device be "hot swappable" you can use the VenusT5 mini.  Just don't pay attention to the installation and configuration instructions for using their RAID-ware unless you want to disable one of the drives.
