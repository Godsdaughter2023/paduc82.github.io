All data should be backed up to at least two external media destinations.

The windows.old folder doesn't help you with the Program/Apps reinstallation that are removed when you do a Refresh.

Retrieve files from the Windows.old folder

https://support.microsoft.com/en-au/help/17125/windows-8-restore-files-old-folder-upgrade

You have only 28 days to retrieve most files from the Windows.old folder, beginning from the time you install, upgrade to, refresh, or reset Windows 8.1. After that time, most of the content of Windows.old is automatically deleted.

If you choose to “Keep nothing” when you upgrade to Windows 8.1, or if you reset, refresh, or reinstall Windows, your personal files are temporarily saved to the Windows.old folder for 28 days (unless you formatted your hard drive before installing). If you decide you want some or all of these files back, you can usually retrieve them from this folder.

Which Settings will be Preserved

slance310 say:

Hi,

The answer at the link you provided is concerning a MBR type Hard Drive...

The machine you are working on may be a GPT Drive...

If the machine is using UEFI BIOS, it is a GPT Drive...

Try just the command BootRec /RebuildBCD...

The X:\ you are seeing is normal... that is what the windows repair media calls the location it has stored the files it is using to run...

BTW... There are some free 3rd party bootable tools that will make it possible to copy the contents of the drive to a external drive should that become necessary...
