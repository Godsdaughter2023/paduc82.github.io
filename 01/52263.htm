Hello, my name is Wendy Jones, I am an Independent Advisor.


Are you trying to transfer your Windows license to another machine right?
For that you need: You can make the transfer of the license taking the 25 digits of the activation key whenever your license is retail.

So the first step is to check what type of license you have, for this open a cmd as administrator and then you will execute: slmgr / dli

If in the answer you get something like this: Description Windows (R), Operating system, RETAIL channel, your license is transferable to a new PC.
However, if you get this answer: Description Windows (R), Operating system, OEM_DM channel your license is the one that appears by default when you buy a machine and in that case it is not transferred.

In the case that you have a retail license, you must unlink the license of the machine that you are not going to use at this time, for this: open a Command Prompt as administrator and execute these two commands:

slmgr / upk

slgmr / cpky
The first uninstalls the license, and the second deletes it from the registry. After restarting the computer, Windows will not have a product key and will be in an inactive and unlicensed state.

Then on the new computer once you have installed Windows, to activate it:
Open a system symbol and execute: slmgr / ipk [product key]

and then: slmgr / ato

Finally, link your new license with your Microsoft account.

Reference: https://answers.microsoft.com/en-us/windows/for...

If you do not know which is the activation key of Windows you can obtain it before unlinking it from the machine that you have active at this time as indicated below:

Check if in Downloads you have a zip folder with the name: "ShowKeyPlus"
Extract the file and you can find the key of the original product.

https://support.microsoft.com/en-us/help/20530/...

Original title: Fonts uninstalled

a. user say:

Hi Reba,

Do not worry we will help you with this issue.

Let us know if this issue confined to any specific font on your computer.

I suggest you to follow the below to install the fonts and check if it helps:

  1. Press Windows + R keys to open Run command.
  2. Type the following command, and then click OK:
    %windir%\fonts
  3. On the File menu, click Install New Font.
  4. In the Drives box, click the drive that contains the font that you want to add.
    Note: The floppy disk drive is typically drive A or drive B. The CD drive is typically drive D.
  5. In the Folders box, click the folder that contains the font that you want to add, and then click OK.
  6. In the List of fonts box, click the font that you want to add. To select more than one font at a time, press and hold the CTRL key while you select each font.
  7. Click to select the Copy Fonts To Fonts Folder check box. The new font is saved in the Windows\Fonts folder.
  8. Click OK.

Note: Windows supports TrueType fonts or fonts that are designed especially for Windows which can be purchased separately. Some programs also include special fonts that are installed as part of the program installation. Additionally, TrueType or special Windows fonts are frequently included with printers. Follow the directions that are included with these products to install these fonts.

Hope it helps. Reply to the post with an updated status of this issue for further assistance.
