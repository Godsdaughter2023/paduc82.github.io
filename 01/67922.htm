Hi there,

 

Thank you for writing to Microsoft Community Forums.

  1.  What is the make and model of the computer?
  2.  Are you trying to install this game on Windows PC or on XBOX? 

 

I would suggest you to follow the methods given below. 

Method 1:

Try running the troubleshooter for Microsoft Store apps.

Select the Start Windows logo Start button > Settings Gear-shaped Settings icon > Update & Security > Troubleshoot > Windows Store Apps. 

 

Method 2:

1: Right click the Start button (down left hand corner of your screen).

2: Click "App and Features".

3: In the list, search for "Forza Horizon 4"

4: Left click it and press "Advanced Options"

5: Left click "Reset"

This will delete all the LOCAL settings and save files.

 Note: You will not lose progress as the game will re-sync your saved files.

 

Method 3:

Most of the time these type of issues are caused due to display adapters, which requires the latest display drivers. I would suggest you to try Updating the display drivers with the below mentioned steps.

 

To update display card driver.


1. Click on Start.
2. In start search box type Devmgmt.msc and press Enter.
3. In the Device manager window, expand Display adapter.
4. Right click on the Display adapter installed.
5. Click on Update driver software and select the option to automatically search for the updated driver software.
6. Restart the computer after its updated and verify if you are able to launch the game.

 

Method 4:

If the issue still persists, try to put the computer in clean boot state to make sure that third party applications are not interfering

 

To perform a clean boot, follow the steps below:

1. Click Start, type msconfig in the Start Search box, and then press ENTER.

 If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

2. On the General tab, click Selective Startup.

3. Under Selective Startup, click to clear the Load Startup Items check box.

4. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

5. Click OK.

6. When you are prompted, click Restart.

 

If your issue is resolved after performing Clean Boot, then follow the steps mentioned in the following KB article to narrow down the exact source:

http://support.microsoft.com/kb/929135

 

Also, see the section on how return your computer to a Normal startup mode by following the steps under “Reset the computer to start as usual”.

 

Hope it works,


Muralidhar
