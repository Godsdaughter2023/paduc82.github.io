The short answer is no - that information isn't available through Microsoft.  Whether you can find out using the computer depends - read on.

 

Is this the product and product key that came with the machine (OEM) or a retail copy you installed later?  Or if it is the product that came with the computer, did you ever do a clean install of the system in the past using the product key on the machine label to activate it?  If either of the last two are true and the computer is working well enough to do this, one option would be trying Belarc Advisor http://www.belarc.com/free_download.html.

 

(NOTE:  If you are dealing with the original program installed by the manufacturer or you used a restore to factory conditions vs. a clean install in the past, the key it shows will be the manufacturer's special key and not your unique key and will not work to manually activate the computer if you re-install).  Also, if you have an upgrade key and need to start over by first installing the original full product, only your original key will work for that first installation and the key to the upgrade will not work with that but only with the upgrade.) 

 

For further information, see http://windows.microsoft.com/en-US/windows7/Where-do-I-find-my-Windows-7-product-key.

 

I hope this helps.

 

Good luck!
