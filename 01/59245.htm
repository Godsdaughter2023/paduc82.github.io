This function does not exist natively (for quite some time) on a local file system.  If this is a network share and you mean 'access from another computer', that is a different story.

You control access to a local folder/file by file & folder permissions with the NT File System (NTFS.)  Some people have used the built-in ZIP handling features to accomplish something like you present - but the password-protected local folder went out (in-as-far-as native Windows) a decade ago.

If you truly want to password protect a local folder like that, you *will* have to resort to third party tools.

davem121 say:

Yep, Office 2016 can open/convert some types of PDF to an editable format,

Here is another tip that will work every time and that most users are unaware of:

  1. Make sure file extensions are showing (In File Explorer, on the View Tab, check the box 'File Name Extensions'
  2. From any application, click Print
  3. Print to the Virtual XPS Document Writer which is in most versions of Windows
  4. You will then end up with a document with either a .xps or .oxps extension
  5. Change the extension to .zip
  6. Extract the the document (now a zip file) to a folder
  7. Inside that folder, navigate to: Documents\1\Resources\Images
  8. In that folder will be all the original images contained in the PDF

Glad that worked for you, if you have any more questions, please contact us at MS Community!
