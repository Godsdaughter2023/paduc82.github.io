rmdeq say:

My very last sentence was key. The problem is solved. The hidden 452MB partition has to be the active partition, at least when your Windows partition is BitLocker.  More precisely:

1. Boot to Windows RE

2. Open a command line

3. Start diskpart.

4. List disk

5. (Find which number (d) has the System partition, ie the small partition NOT your Windows partition)

6. select disk <d>

7. list volume

8. (Find out which number (v) is the System partition)

9. select volume <v>

10. active

11. exit

Now. reboot your system and make sure your UEFI/BIOS boots to the disk with the System partition on it, either because your Boot Order is set that way or use something like a Boot Override.

I don't remember what I did to make the wrong partition the active partition, but that was it.

irfan h say:

Hello jrcro96,

You can follow the below steps to rename the Boot menu entry.

To change the description of the entry that is in the boot list:

a.     Boot into the Windows entry that you would like to say “Windows 7” instead of “Windows 7 Recovered”.

b.     Click on Start and type ‘cmd’ in the search field.

c.      Right click Command Prompt and click Run as administrator.

d.     Carefully type this command: bcdedit /set {current} description “Windows 7”

e.     Press Enter on your keyboard.

f.       Exit the Command Prompt.

g.     Restart your computer.
