john_barnett say:

Hello ColinBoyce,

 

You cannot uninstall Windows Vista (or any other version of Windows) from within itself. To remove it you have to install another operating system over the top or reformat the drive outside of the windows environment, i.e., the pre-windows environment or old DOS.

If you are running a dual boot system, i.e., two versions of windows on different partitions, then you can simply delete the partition which houses Windows Vista. If you only have one version of Windows on your system you will have to install a newer version over the top or, as I have already pointed out, use partitioning software to enable you to format the C: drive from the pre-windows environment.

 

If you simply want to re-install Windows Vista then insert the Windows Vista DVD into your DVD drive and follow the instructions on screen to Install the Windows operating system. This will then over-write your current version of Windows and replace it with a new one.
