If "I updated to Windows 7 64 bit" means that you used to have a 32-bit installation of Windows 7 and decided to change to the 64-bit version (perhaps when you did the "hard disk transfer" in April 2016), then that is the problem:  drivers have to be written for the architecture of the system; your original 32-bit driver isn't working for the current 64-bit system.

To confirm what you have, press the "Windows Logo" key and the "Pause/Break" key simultaneously to open Control Panel/System and "View basic information about your computer."  Look in the middle of the screen to see what type of system you have:

Based on the last post at the BleepingComputer forum link that you cited, it seems to me that despite the names, the files in the Microsoft Update Catalog are, in fact, the compete driver packages. 

Because it's a bit complicated to use Microsoft Update Catalog to manually install printer drivers, see if the following works:

  1. Disconnect the printer from the computer
  2. Go to Devices and Printer and click any icon in the "Printers and Faxes" section
  3. Click the "Print server properties" button at the top of the window
  4. Select the "Drivers" tab
  5. If there is a driver listed for your Canon i560 but it is for the wrong version (e.g., 32-bit instead of 64-bit), select it and click "Remove"; when prompted, remove both the driver and driver package
  6. Click "Add" and step through the "Add Printer Driver Wizard"
  7. Select the correct architecture for your system
  8. On the "Printer Driver Selection" screen, click "Canon" and confirm that there is no entry for the i560; then click "Windows Update"
  9. Go get a cup of coffee or lunch; the update will take some time
  10. A driver for the Canon i560 should appear.  That happened for me:


  11. If you did get the driver, select it, click "Next" and complete the wizard.  You should now be able to connect the printer and it should automatically install.

Assuming that no driver for your printer appeared, you will have to use the Microsoft Update Catalog manually.  Let me know if this happened and I'll give you detailed instructions.
