I cant permanently rename a Custom View in Event Viewer. - Microsoft Community

allen hall say:

I cant permanently rename a Custom View in Event Viewer.

I can’t permanently rename a Custom View in Event Viewer. When I close the Computer Management console and reopen it the Custom View has its original name.

mouneshwar r say: Hi Allen Hall

Reusable Custom Views

When you work with Event Logs, your primary challenge is to narrow the set of events to just those that you are interested in. Sometimes this is easy. Other times this involves a great deal of effort; effort that is lost if you do not have some way to save the view of the logs that you worked so hard to create. Event Viewer supports the idea of custom views. Once you have queried and sorted your way to just the events you wanted to analyze, you can save that work as a named view and it will be available for you to reuse in the future. You can even export the view and use it on other computers or share it with other people.

To create a custom view

1.      Start Event Viewer.

2.      On the Action menu, click Create Custom View.

3.      To filter events based upon when they occurred, select the corresponding time period from the Logged drop-down list.


Note : If none of the options are acceptable, choose Custom range. In the Custom range dialog box, specify the earliest date and time from which you want events and the latest date and time from which you want events. Click OK.

4.      In Event level, select the check boxes next to the event levels that you want included in the custom view.

5.      You can either specify the event logs or the event sources of the events that will appear in the custom view.

6.      To specify the event logs: Select the Event Log option and, in the Event log drop-down list, select the check boxes next to the event logs from which you want to
         include events.

7.      To specify the event sources: Select the Event Source option and, in the Event source drop-down list, select the check boxes next to the event sources in the
         drop-down list that you want to include in the custom view.

8.      In Event IDs, type the event IDs that you want your custom view to display. Separate multiple event IDs by commas. If you want to include a range of IDs, say
         4624 through 4634 inclusive, type 4624-4634. If you want your filter to display events with all IDs except certain ones, type the IDs of those exceptions, preceded
         by a minus sign. For example, to include all IDs between 4624 and 4634 except for 4630, type 4624-4634,-4630.

9.      In Task Category, select the check boxes next to the task categories in the drop-down list that you want included in the custom view.

10.    In Keywords, select the check boxes next to the keywords in the drop-down list that you want included in the custom view.

11.    In User, enter the name of the user accounts you want to display. Enter multiple users by separating them with a comma (,).

12.    In Computer(s), enter the name of the computers that you want your custom view to display. Enter multiple computers by separating them with a comma (,).

13.    Click OK.

14.    On the Save Filter to Custom View dialog box, in Name, type a name for the custom view.

15.    In Description, type an optional description of the custom view.

16.    Select the folder in which you want to store the custom view.

17.    To save the custom view and make it accessible to anyone using the computer: Ensure the All Users check box is selected and click OK.

18.    To save the custom view and make it accessible only to someone logged on to your current account: Ensure that the All Users check box is not selected and click
        OK.

Check out all the sections below in the following link

a.      Custom Views
b.      Create a Custom View
c.      Export a Custom View
d.      Import a Custom View

http://technet.microsoft.com/en-us/library/cc766042.aspx

Regards,
Mouneshwar R – Microsoft Support

hairyfool say:

Was the Windows 10  on the old computer an upgrade during the 1 year free upgrade program? If so did the original O/S (Win7/8) come with the computer?

If it did then the Windows 10 upgrade would take on the license terms of the qualifying Windows license which was most likely OEM. If so OEM licenses cannot be transferred to another computer so you could not install your old drive into another PC and the license be valid. In that case keeping the Windows installation on the old drive would be pointless.

When the old drive was the OS drive that installation of Windows would have identified System reserved as a hidden partition and not display the drive. The new OS doesn't have that status so shows them. Removing the drive letter will hide them.

gerymolnar say:

Can I transfer my copy of Windows 10 to a USB or the like? And if I format the disk, it will remove the system reserved disk right? (I'm also assuming I won't have to delete everything on it before formatting)

hairyfool say:

If all you want to do is read the files on it then yes you can, you haven't really explained what you want to do with it.

If you wanted to return it to another drive in the future then no, the only real way to do that would be to boot from it and then create a System Image. To your current O/S the disk is just two partitions with no specific relationship, when it was the booted drive the two partitions are needed together because the System Reserved is really the Boot environment to load the O/S in the second partition.

Formatting is done at a partition level, if you just formatted then you would be left with two partitions. In order to make it a single partition (volume) you would need to delete both of the existing partitions, create a new partition and then that is what you format.

Deleting volumes and formatting volumes removes the contents of the disk, you do not need to remove the files 1st.

vijay a. verma say:
The setting to enable pop up is different in different browsers

1. Firefox
Open Firefox > Click 3 vertical bars on right top side > Options > Privacy & security > Scroll down and uncheck Block pop-up Windows

2. Chrome
Open Chrome > Click 3 vertical dots on top right side > Settings > Scroll down and click Advanced > Content Settings > Uncheck Popups

3. Edge
Open Edge > Click 3 horizontal dots at the top > Settings > Advanced Settings > Toggle OFF Block pop ups
