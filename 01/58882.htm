j w stuart say:

Yes that's correct, Windows 7 is stored inside the Windows.old folder so you will loss space, exactly how much depends on what you have installed and the amount / size of your data files.

If you have no plans to restore Windows 7 using the roll back option due to drive space issues (and you have created an Image Backup of Windows 7 just before you upgrade) you can delete the Windows.old folder by using Disk Cleanup and select the Cleanup System files option.

Just keep in mind that any new data files created in Windows 10 will need to be saved to external media prior to restoring 7 from an Image Backup.
