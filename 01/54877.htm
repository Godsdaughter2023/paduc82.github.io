bossdweeb say:

Even when using appropriate active hours Windows Updates will download updates in the background.

All active hours does is prevent the update from rebooting your machine without asking, which should only happen

when you shut the machine down (install and shut down or install and reboot).

- Generally, if you have a good internet connection the cumulative/security updates shouldn't cause problems, however if

you are allowing Windows Updates to do your bi-annual version updates eg- 1809 etc.,  in the background they can definitely

slow your connection (effectively stealing focus) due to the large size of the download.

Downloading a version update in the background is also more likely to result in corrupted files and possibly a corrupted

version update as well.

The safest way to do the version updates is to download the  --  Update Assistant  -- and run it when you are not gaming.

Allow it to use as much of you internet bandwidth as it needs without slowing the download. When the download is done install

the update and it should not cause any issues.

- Similarly, if the cumulative/security updates are stealing focus, run Windows Updates manually when you log on so you can be

sure your Win10 is up to date before starting a online gaming session.

- Of course there is potential for your Anti-Virus, Browser, or other software you may have running while games to steal focus as well.
