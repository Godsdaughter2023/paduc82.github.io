Hello, I’m Virginia an independent advisor here.
For safe eject notification missing:
• Press keyboard "win + R"
• Type "regedit"
• Search the value "HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Notifications\Settings\Microsoft.Explorer.Notification.{...}"
• If you REG_DWORD 0x00000001(0) set "0" you can double click on value. Set the value data to "1"
• Click "OK"
I hope this helps.

I’m seeing several people with the same problem but there’s no entry within the registry.

What version of Windows are you using?  Windows Vista, Windows 7 or 8/8.1?

How to uninstall software:

  1. From the Start Screen
  2. Type in Control Panel
  3. Click Control Panel (Make sure Apps is selected)
  4. Click Uninstall a program under Programs category.
  5. Find TOSHIBA Web Camera Application and double click it.
  6. Finish the uninstall following on screen instruction.


Access TOSHIBA support website to get the latest component to install.

  1. Go to http://support.toshiba.com/drivers
  2. Enter you product serial number
  3. Press GO
  4. Select Camera under Filter By
  5. Select Windows 8.1 (64bit) under Filter By
  6. Click on the WEB camera application at the top of the list
  7. Click Download
  8. Install the application by following the on screen prompts
