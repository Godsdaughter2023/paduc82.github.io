jay freedman say:

The path to the Startup folder for your login profile is

%appdata%\Microsoft\Windows\Start Menu\Programs\Startup

The %appdata% is an environment variable that evaluates to the folder
C:\Users\<your login name>\AppData\Roaming. But because the AppData
folder is hidden, you need to use this variable or change the Windows
Explorer option to allow it to show hidden folders.

There's a similar location used for all profiles at

C:\ProgramData\Microsoft\Windows\Start Menu\Programs\Startup

BUT... The programs you want to stop from running may or may not be
called from the Startup folders; there are several other ways that
programs can make themselves run at startup, including a couple of
different registry entries.

An easy way to disable any program's startup behavior is to type

msconfig

in the search box on the Start menu and then click the msconfig.exe
link that appears. In this program, click the Startup tab. Uncheck
each program that you want to prevent from running at startup. (The
entry in the Location column shows you what's causing the program to
run.) After clicking OK to close the program, reboot.
