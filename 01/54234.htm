tricky300 say:

Hi,

Admittedly I'm making an educated guess here, but I think that what is happening is a result of originally having Vista on the laptop rather than XP.

Vista has in built drivers to cope with SATA disk drives in AHCI mode, consequently I suspect that your BIOS is set to treat all SATA disk drives as AHCI.

Windows XP does not have built in drivers to cope with SATA disk drives in AHCI, they have to be loaded by means of an F6 driver floppy diskette (or another much more complex process which hopefully we won't have to worry about).

Your laptop almost certainly did not come with such a diskette and, even if it did, I doubt that your laptop has a floppy drive anyway.

So, the solution is to get into your laptop's BIOS and change the disks to be used in IDE mode rather than AHCI. BIOS is entered by a key depression at boot time - look for the message telling you what key but it is normally F2 or DEL. Once in BIOS you must look for this AHCI setting and change it to IDE. Make a note of any changes you make so that they can be re-instated if necessary. There will be a mechanism to save any changes that you have made to BIOS settings - you must save changes or the result will be the same when you try to install.

Then try installing XP again.

If you have a recent model laptop with a fingerprint reader, you can setup Windows Hello to log in to Windows using your fingerprint.

To do this, you’ll first need to set up a PIN to logon.

Here are the steps:

1. Go to Settings, then to Accounts, click on Sign-in options. If you have a supported device, you’ll see the Windows Hello option. As you can see, It is required to first setup a PIN before the Setup button is activated.

2. To setup a PIN, you will need to login to Windows 10 with a Microsoft Account and add your PIN to your MSA profile. Enter your password in the field provided and then select Sign In.

3. Next, you need to set up a PIN and confirm it. Then select OK.

4. Next, select Set up under Windows Hello.

5. Select Get started.

6. Run your finger over the fingerprint scanner until the setup app reports completed. You can now optional add more/different fingers.

7. When you restart, you will be able to select Sign in Options and then sign in using your fingerprint.
