joey.coyle say:

Check your System Reserve Partition and make sure you have free space.

I was having this issue and could not figure out why the update would not install.  I kept getting the message that the update could not be installed and it was undoing the changes.  It was by chance after using the Media Creation tool to attempt to do an in-place upgrade (hoping this was going to get around the update issue) that I received the message that the upgrade could not be done because the System Reserved Partition could not be updated.

Sure enough, I had installed Kaspersky and it had taken up the remaining free space on the SRP.   I was able to assign a drive letter to this partition and delete the Kaspersky directory.  I also read that you can delete the MS language packs that you do not need to free up a little extra space.  If this does not free up enough space then you would need to use some type of partition tool to increase the size of SRP.

I hope this helps.

Joey
