My Challenge to Microsoft

 
Over the past few months I have invested in three beautiful new computers - one laptop and two desktops.  Each system came with Windows 10 pre-installed and worked great out of the box.  Fast forward to Windows 10 automatic updates and all three computers lost some or all functionality ranging from no screen fonts to CRITICAL_PROBLEM.DIED blue screens.

The root of the problem appears to be that continuous Windows 10 updates vs the original manufacture's drivers and software applications are resulting in failed systems.




Each computer required a different fix.  On my Dell desktop when a clean Windows install failed to get my machine back online I recovered the factory image, removed all manufacturer's utility software and select drivers then, over time let Windows install updates at its own pace. 

My hp laptop required a factory re-install then disabled Windows 10 updates to block versions 1607 and 1703.  It's running fine on the original version 1511.  

In each case once the system settled down and returned to normal I turned off Window's Updates.  Security updates on all machines are now download manually from my anti-virus software and the Microsoft Update Catalog online.

http://www.catalog.update.microsoft.com/Search.aspx?q=windows%2010

Summary:

Windows Service Updates vs Original Manufactures Drivers & Software = CRITICAL ERRORS

Remember the days when a new version of Windows was incompatible with select hardware or software applications on your existing machine?  Are we seeing this play out again in Windows 10?  Updated versions are forced upon our computer without the option of NOT installing them.  End result - repeated system failures.

This "one OS fits all" approach is causing havoc with our beautiful machines!  Do an online search of the topic and you'll see problems with the Windows 10 "Service" OS updates are numerous.  By comparison, although not bug free, previous versions of Windows such as XP and 7 were much more stable.  Why?  Because once we had our hardware drivers and software applications installed that was it.  We didn't need to constantly update or replace versions to keep up with changes from the OS.

Windows System Image, Factory Restore and/or Third Party System's Images applications are priceless and have saved me on numerous occasions, however having to rebuild all three new computers is not what I hoped for. 

My Challenge to Microsoft: 

Go down to any retail computer store or outlet and purchases a number of new laptop and desktop PC's.  Run them with the original manufacture's setup then let Windows perform updates overtime.  Observe the stability of each system.  Microsoft must return to placing stability of existing systems as a first priority when developing new version upgrades.  Progress is great but not at the expense of an already working machine.

The scrolling text can also be mirrored for use with teleprompter hardware. You can produce teleprompter hardware very cheaply or just put your display running Teleprompter Pro next to your camera to enable you to deliver professional quality speeches.
