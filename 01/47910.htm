bobbymi say:

Hello LD,

 

Method 1: Change the user account picture to one of the default pictures, and then try to browse for more images.

 

1.       Open User Accounts by clicking the Start button , clicking Control Panel, clicking User Accounts and Family Safety (or clicking User Accounts, if you are connected to a network domain), and then clicking User Accounts.

2.       Click Change your picture.

3.       Click a picture in the default pictures, and then click Change Picture.

4.       Log off of the account and then log back on to the account.

5.       Go back to steps 1, 2, and 3 Browse for more pictures...

6.       Browse to a picture and click Open

 

Note: The picture can be of any size, but it must be in a .bmp, .gif, .jpg, or .png file format. Your new user account picture will be used in all locations, including the Start Menu and Log On screen

 

If this doesn’t resolve the issue go to the next method:

 

Method 2: Do some basic steps to clear any issues with the system.

 

  • Restart your computer to clear any possible cache or other issues
  • Visit Windows Update and install any necessary updates
  • Scan your computer for Viruses and Malware

 

Method 3: Replace the default user account image.

Try the following steps to set the default pictures for the user accounts and see if it resolves the problem. (This method may be a little more difficult for standard users)

 

Name the image that you want to use for your user account image to user.bmp and/or guest.bmp for the account type you’re setting the default for.

 

1.       Make sure that you have Show hidden files and folders selected in Folder Options.

2.       Browse to  C:\ProgramData\Microsoft\User Account Pictures

3.       Select the user account image to be changed:

     a.        For the default logon picture you want to change, right click on user.bmp or guest.bmp and click Rename.

     b.      Rename them to user.bmp.bak and/or guest.bmp.bak and press Enter, then click Yes, Continue, and Continue.

      c.       Right-click on the image from step 1 and click Copy, then right-click on an empty area in this window and click Paste.


Note: To undo this, delete the new image from step 4C and remove the .bak off of the name of user.bmp.bak and/or guest.bmp.bak to restore the default images.

 

4.       Close Windows Explorer.

5.       Select Do not show hidden files and folders in Folder Options.

6.       Log off of the account and then log back on to the account to set and check the picture.

 

Method 4: Create a new user profile.

 

If none of the other methods resolve the issue you may have a corrupted user profile. This is the worst case solution because you will need to move all your personal files and folders to a new user account, but this will resolve the issue. Follow the steps in the article below to create a new account and move your files.

 

Fix a corrupted user profile

 

Hope this helps
