cmdrkeene say:

File History would probably meet your needs.  It won't make a duplicate versioned copy of a file if that file hasn't changed, so it would indeed give you a single copy of that file.  Of course if you changed the source file, it would keep both so you can "go back in time".

 

If you don't want that time machine stuff, you can actually do what you want with a scheduled command.  Some folks use the robocopy command, but I'm not as familiar with it and I prefer the simpler xcopy command.  For example, this command would copy all files from your Documents folder to the E: hard drive, overwriting any files already on E: (so replacing it with the most current version).  The other options I used are described in the screenshot below:

 

xcopy C:\Users\Shawn\Documents /s /v /c /i /q /y /z /j

 

You can put this into Task Scheduler and tell it to repeat every night at 4:00 AM or something so your backup is never more than a day old.

rooster6975 say:

Hi, a 32GB system is definitely tight for Win10.  For the biggest file (temp file), I assume that is your pagefile.sys?  If it is, you can move that to your USB3 stick.  In the bar at the bottom, type System, then click on Advanced System Properties.  On the Advanced tab, section Performance, click Settings.  That brings up Performance Options.  Again select the Advanced tab and where it says Virtual Memory, click Change.  Uncheck the option to "automatically manage pagefile for all drives"  When the dialog box with your drives is no longer greyed out, select your USB stick.  Then OK, OK, OK... until you are back out.  Reboot.  Now you can safely delete the pagefile.sys located on the root of C.

That will save you at least several GB of space, likely more.  Another thing you can do is max out the RAM on your system to whatever the max is.  If, for instance, that computer allows 32GB of RAM, you can set your paging file to be very small without worrying about it (say 2 GB).  But that is the expensive option.....

For a cheaper third alternative (free), download and run ccleaner.  Once installed, go through the advanced options to have it clear out Hotfix uninstallers amongst all the other default stuff it clears out.  Works like a charm, you will likely need to run it fairly often to recover your disk space.
