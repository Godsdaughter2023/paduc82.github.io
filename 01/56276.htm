brian-- say: From: Rick Rogers


Hi,

The upgrade license is very specific in that it can only be used to upgrade an existing license you own. That license becomes part of the upgrade and can no longer be used for the original operating system. To use the upgrade to create a dual boot such as you plan is not in keeping with the license terms. From a technical standpoint though, you can make it work. An alternative solution would be to install XP into a virtual machine under Windows Vista using Virtual PC and run your application there.

Virtual PC runs fine under Windows Vista Home Premium, even though it's not listed as supported:

Hello, when select 5.1 configuration, i finish the configuration and then i when i play movie in any video player, i only hear the background music but not the voices of the actors or people in the movie. I have the Logitech z506 5.1 surround sound speakers and i would like to use them to their full capability.

SOLUTION: I solved this myself, i found that you need to use the black and yellow with the green 3.5mm cable to get 5.1 surround sound. You have connect green  cable to the line out(green port),yellow cable to the mic in(pink port), and the black cable to the line in(blue port) and then connect it to the black and yellow port in subwoofer. Make sure you configure the line in (side speakers out)and mic in (center/subwoofer out). Then just configure as 5.1 surround sound and your good and gravy.
