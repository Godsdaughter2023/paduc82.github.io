shreyas p v say:

Updates from Microsoft are first downloaded and then installed automatically. If you have changed the option to "Check for Updates but let me choose when to install them" then you haven't faced this problem now.

The problem is that the Microsoft Update has partially downloaded the update files and tried to install those , but when it didn't get the complete file it tried to rollback the changes.The rollback time is usually longer because it would have over written the older files with the partially new ones and when its partial it fails to update completely and the new ones have to be removed first then the older one has to be restored which consumes time.

I think in your case you have downloaded larger update files and the update has been interrupted.

Now the only way is to have patience and wait for it to complete so that it restores your OS to the previous state before updating.

I had also experienced the same problem while installing the KB2919355 update which is around 800-850 MB

I let the rollback to continue and within 2-3 hrs it rolled back , restoring my OS safely.

If you open File Explorer in WIndows 8.1 and search for files you are not allowed via details pane to change any file properties.
