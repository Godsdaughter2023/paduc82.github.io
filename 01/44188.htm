Follow the steps below.  If they don't work, please provide the complete text of the error message that occurs when you attempt to install the driver.  Also see the following, which sounds suspiciously like your situation but for the fact that you haven't mentioned that your printer is connected over a network -->  https://patrickhoban.wordpress.com/2013/10/30/0800-9/

1.  Disconnect the printer from the computer (I assume that the printer is connected directly to the computer using an USB cable).

2.  Go here --> https://support.microsoft.com/en-us/kb/2793718 and download both of the Fix it routines.  That is, if you are using Internet Explorer, choose "Save" rather than "Run."

3.  Download a fresh copy of the Vista 64 bit driver (cjs8300EN.exe).  Temporarily disable your antivirus program before you click the "Download" button.

4.  After downloading both the driver file and the two Fix it routines, disconnect your computer from the Internet so you can leave the antivirus disabled.

5.  Uninstall all Lexmark software using Control Panel > Programs and Features.

6.  You'll probably need to run the "Full mode" Fix it routine, but be aware that doing so will remove ALL of the printers currently installed.  The Full mode Fix it essentially puts your Print Environment back to the way it was when the operating system was installed.  You must reboot the computer after running the Full mode Fix it.

If this Lexmark is your only printer, then use the Full mode Fix it.  If you have other printers, you can see if the "Light mode" Fi xit solves the problem, but I suspect it won't.

7.  After running whichever Fix it you selected, and rebooting if you selected the Full mode, run the driver installer.  Connect the printer to the computer either when prompted during the installation of the driver or when the installation is complete.

If you used the Light mode Fix it and the driver installation wasn't successful, start back at step 5 and use the Full mode Fix it.

It can't be done from the spooler (that's too late in the process without 3rd party sofware like http://www.sharewareconnection.com/ok-printer-viewer.htm) but it can easily be setup so that the print jobs themselves are setup as .PRN files which can then be printed on another printer (with some limitations as to comptatability).  Here's the procedure and the guidelines: http://filext.com/faq/print_from_prn_file.php.  The latter is the standard way of doing this (it's how I work when I am away from my network); however, the 3rd party solution may work better with a wider variety of printers once you do try to connect  (but if it is to your own printer, I don't think that will be a problem).  Just save the .PRN files in a single place and print them when you're back with your printer.

I'd start by clearing the system cache (it's just updates plus other temporary files, nothing that's at all important to anyone) by:

1) Tap the silver Xbox Guide button on the controller, go right 2 clicks to Settings, pick System Settings.

2) Select Storage (or Memory).

3) Move over any hard drive or memory unit (not the Cloud) & press (Y) on your controller & pick Clear System Cache.

4) When asked to confirm storage device maintenance, choose Yes.

Delete profile from your console

1) go to www.xbox.com/accounts & click on Sign in at upper right - if you can't sign in then STOP: DON'T delete your profile. Please call Xbox Support support.xbox.com/contact-us for help with your account. If you can sign in, delete & redownload your profile on your Xbox this way:

2) Go to Settings

3) pick System

4) choose Storage

5) pick All Devices

6) choose Gamer Profiles

7) pick the gamertag

8) choose Delete, and

9) pick Delete Profile Only. This deletes the profile but leaves saved games and achievements. Now, re-download your profile.

Re-download

1) Tap the silver Xbox Guide button

2) Select Download Profile (and if you don't see it, press the blue X button to make sure you're not signed in) and proceed with Profile Download.

3) Enter the Windows Live ID for your Xbox LIVE gamertag when asked.

4) Enter the password for the WLID.

 4A) If your account is an Xbox LIVE child account, the parent’s Windows Live ID & password is required as well to download your profile.

5) Pick the storage device for your profile (choose Hard Drive if you have one, or Memory Unit if you don't have a Hard Drive).

6) When the download completes you will be asked if you want the console to remember your password. If it's your home console, I'd say yes, otherwise I'd think carefully about leaving my profile unprotected on somebody else's Xbox!

7) Sign in, and retry.

arminatorx say:

Ein paar Ergänzungen zu Kolea:

Wenn Du bei der Xbox One die Spracheinstellungen auf Deutsch Land Österreich stellst, wird ein Österreichisches Abrechnungskonto erstellt. Diese Konten sind absolut getrennt, und Geld aus einem Länderkonto kann nicht von dem anderen Länderkonto verwendet werden.

Beispiel:

Du kaufst eine Guthabenkarte hier in Deutschland, und hast Deine Konsole auf "Deutsch / Deutschland" gestellt. Du lädst das Guthaben auf Deinen Gamertag und kannst damit im Deutschen Xbox Marktplatz Spiele kaufen.

Nun wechselst Du die Konsolensprache und Region auf "Deutsch / Österreich", dabei wird dein Deutsches Abrechnungskonto "eingefroren" und ein Österreichisches "leeres" Konto dem Account zugewiesen.

Da hast Du dann noch kein Geld drauf.

Wechselst Du die Konsole wieder zurück auf Deutsch / Deutschland, wird das leere Österreichkonto abgeklemmt und eingefroren, und das Deutsche Konto mit Deinem Guthaben ist wieder verfügbar.

Und 360 Spiele kannst Du mit Guthaben bezahlen, wenn Du ZUSÄTZLICH eine gültige Zahlungsmethode wie Kreditkarte oder PayPal an Deinem Konto hast.

Zumindest kann ich so einen Teil mit Guthaben aus meinen Rewards-Zahlungen und den Rest mit meiner Kreditkarte zahlen.
