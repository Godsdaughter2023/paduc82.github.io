ann marie scrimshaw say:

Hi groover000.  It's possible that there may be an issue with your file system and/or hard drive.  Try running chkdsk from a command prompt and see if it helps.  Backup your files first though (see instructions below).

Plug in your USB drive and  boot using the DVD.  See the Vista tutorial (W7 is the same) -> http://www.bleepingcomputer.com/tutorials/tutorial147.html for more help.  Open a command prompt and type:

C:\windows\notepad.exe

Notepad will open.  Go to File Open and where it says Look In, if Windows 7 is on your C drive, change the directory to OS (C: ) or to whatever drive letter W7 is installed on.  Next, change the Files of Type to "All Files".  You will now be able to navigate through your drive and save files to the USB drive.  Rightclick on the file you want to save and choose Copy.  Navigate to your USB drive, rightclick again and choose Paste.

chkdsk c: /r

and hit Enter.  It will take a while to run but keep an eye on it. It should fix any errors on the volume if this is the issue and allow you to boot into Windows when completed.
