    This can happen if the notification area's icon settings get corrupted.  You can delete the bad settings from the registry to repair the error.  Here's how.

    1. Right-click your Start button and choose "Command Prompt"
    2. Highlight this following command (bold text below), and copy it.

      REG DELETE "HKCU\Software\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\TrayNotify" /v IconStreams /f
    3. Right-click inside the Command Prompt and choose paste.  Press Enter to run the command.

    4. Do the same with this next command (copy and paste this into your Command Prompt, then press Enter).

      REG DELETE "HKCU\Software\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\TrayNotify" /v PastIconStream /f

    5. After both these commands have been run, try turning on your volume control from the "turn system icons on or off" again.

    4. Click on Boot Tab.

    5. If you see safe mode box has a check mark in it, uncheck the safe mode box and hit apply or ok.

    6. You will then see a restart button. Click it to restart your computer.
