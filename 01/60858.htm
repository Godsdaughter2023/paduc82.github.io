Elsewhere online I have seen reference to an option to launch each new window in Explorer (not Internet Explorer) as a separate process. This option is accessible through these steps:

1. Open a folder - any folder - so that you have a window open with the usual File Edit View Favorites Tools Help strip at the top

2. Click on Tools and choose Folder Options

3. Click on the View tab

4. Scroll down until you see an entry labeled "Launch folder windows in a separate process". It will almost certainly be checked. Uncheck it and click the Apply button. If it is NOT checked then this proposed solution probably won't apply in your case, so continue looking for solutions.

5. Close the window

If you have run Task Manager in the past and seen multiple entries for Explorer under the Processes tab - usually one for each folder you currently have open, plus one - unchecking the box will stop that happening in future, and should prevent the Proxy Desktop dialog from recurring.
