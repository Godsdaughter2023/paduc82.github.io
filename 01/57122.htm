Public vs Home - Microsoft Community

billral say:

Public vs Home

What is difference in access? 

If I have only 2 computers on my network at home & add no more, is the Home Network of Win 7 more secure than the Public?

My XP laptop can access some Public folders on the Win 7 desktop,but does not see or recognize the Home Network folders

arupratan say:

You have to configure both Desktop and Laptop for file sharing.

Step 1. Enable file sharing and network discover and other settings on Desktop (Windows 7):

Connect the Ethernet UTP cable and wait for the Windows to identify. You'll see the network location is automatically set to Public and you can't change it to Home . Besides you can keep it Public if you want and don't need to create Homegroup. It's safe if you don't frequently share files. But if you frequently share files and you have Internet as well then it's safe to create separate locations. Let's create a  Homegroup . Open   Network and Sharing Center by clicking  Control Panel , then click  Homegroup . then click  "What is a network location ", you will get the 3 options Home/Work/Public . Select Home . Now you will see that the network is denoted as Home but it will show as "Unidentified Network". Don't worry, it's just a name and leave it like that. Now in the Homegroup enable all types of Sharing i.e. Videos, Music etc.

After that click  Change advanced sharing settings . Now in the Home profile turn on everything. If you don't want password protection then turn off it. If your home network is not connected/shared with the Internet then you may turn off password protection. Enabling password protection will deny the access to this computer from any other computer. So if you turn on it and then if you try to enter your desktop from the laptop, every time it will ask for the password which may be a boring thing so I prefer turning off it.

Now click the network and see if in the list of computers whether your desktop is visible or not. Your laptop will not be visible now because you haven't configured it yet. You may also go to network from My Computer. Go to My Computer and from the left click  Network and the Network window will show the visible computers and devices.

N.B. If your are connected to multiple networks while you are configuring a Homegroup, you can't create a Homegroup. So make sure there is only one network available when you create this Homegroup and contain only your desktop and laptop. You must disconnect the Internet while you configure the above configurations.

Step 2: Configure Network, file sharing in the Laptop (Windows XP)

Go to  My Network Places , click  Setup a home or small office network , click Next . Again click Next . If you have multiple network adapters, such as a LAN adapter, a Bluetooth adapter, Wireless adapter, then it show the list of existing adapters. Select your LAN adapter and click to select the check-box  "Ignore disconnected network hardware ". I recommend you disable all other adapters before this configuration for avoiding complexities. You can enable then back again anytime.

Click  Next , now select any of the first two options, you may select the second one (This computer connects to the Internet through a residential gateway or through another computer on my network) since your laptop will connect to the Internet through the wired LAN shared with the desktop. Click  Next . Give the name and description and click  Next .

Now this is the critical step. The workgroup name must be the same as the desktop. Give the name same as the workgroup name of your desktop. In Windows 7 the workgroup name is set by default as "WORKGROUP", you can see the name in your System Properties, just right-click  My Computer and view the name of the Workgroup. In Windows XP the default Workgroup name is set to  "MSHOME". Rename it according to your desktop. Click Next and wait until Windows configures. Now click to select the radio button  "Turn on file and printer sharing ". Click Next and again click Next and then wait for Windows to configure. Now select  "Use my Windows XP CD ". Click Next , again  click Next . Click Finish . Restart.

Step 3:

After Restarting your laptop wait for both the Windows for identifying. After that make sure in the Network and Sharing Center everything is enabled in the Home profile, i.e., network discovery, file sharing etc. It's because since you've restarted the laptop, the network goes down for a while and Windows 7 will set back the Home profile to default configurations. If so, it'll again make the network location  Public. So, you just have to do the same thing again to get back the  Home profile and create the  Homegroup. So, beware of restarting and disconnecting your network frequently. Everytime you disconnect, you have to reconfigure only the Windows 7's network location, not in the Windows XP.

Now go to  My Computer, click  Network and since everything is turned on (network discovery, file sharing), you will see both computers within few seconds. Sometimes it takes more times, just refresh the window and wait and be patient. If everything is ok you will definitely see both computers.

Step 4: Sharing the Internet Connection:

Now your home network is working properly, you want to use the Internet in both computers. All you have to do is to share the Internet connection in the desktop. To do this go to Network and Sharing Center in your desktop. Click Change adapter settings . Now you will see all the network adapters including the adapter through which you are connecting to the Internet (in this case the adapter that is directly connected to the router).

Right-click the adapter and  click  Properties . Click  the Sharing tab. Select  "Allow other network users to connect through this computer's Internet connection ". Then from the Home networking connection drop-down list  select Local Area Connection ,   click   OK . Click OK again and again. Now within a few seconds you can browse the Internet from your laptop.

*** Before enabling this Internet sharing, in your Windows XP laptop you will see in the taskbar the network connection icon will show "Limited or no connectivity". This is nothing and not a harmful message, just forget it.

Hope this will help. If you face any problem ask again and if you'd like to receive further help from me on any computer/Internet/networking related issues, contact me  here. Thank you.
