Language pack for Windows 7 Home Edition - Microsoft Community

rph888 say:

Language pack for Windows 7 Home Edition

I just bought a new computer with Windows 7 Home Edition (Simplified Chinese). I want to install an English language pack but can't find it anywhere. Please advise.

ronnie vernon say:

I just bought a new computer with Windows 7 Home Edition (Simplified Chinese). I want to install an English language pack but can't find it anywhere. Please advise.


Hi

There are two types of language packs, MUI (Multi-language User Interface) and LIP (Language Interface Pack).

The MUI changes 100% of the interface to another language and the LIP changes 80% of the most commonly used parts of the interface.

The MUI language packs can change any of the 35 Parent languages to another Parent language. The MUI language packs are only available in the Ultimate or Enterprise editions of Windows 7. The LIP language packs can be installed on any edition of Windows 7. Both the MUI and LIP are freely available.

The LIP language packs cannot change a parent language to another parent language. These are only available for localized languages in countries where there is a large population that speaks a different language from the parent language of that country. An example is Spain where there is a LIP available in the Basque or Venetian language dialect.

Since Simplified Chinese and English are both parent languages, you would need to upgrade to the Windows 7 Ultimate edition to be able to change the language to English.

Hope this helps.

Thank You for using Windows 7

Ronnie Vernon MVP
