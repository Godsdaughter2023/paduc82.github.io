OK big question. 
There is a built in backup feature in Windows, but I have heard that MS is planning to "improve" it out in the not too distant future.  So it is probably best to go with a third party backup too. There are several free ones that are perfectly adequate. 
In general, what you do is install the backup program on your computer. 
Then you use the backup tool to create a "bootable" backup disk or USB drive.  This is so that if your computer is un-runable, you can boot and run the backup program  from the DVD/USB to do the required restore.
Then you use the backup program to create backups of your HD.  There are 3 general types of backups, "Image copy", "Incremental copy" and "simple copy".  Image copies are the digital equivalent of a photocopy of your drive.  When you restore from an image copy you have an exact copy of your system from that date. Windows and all programs are in full running condition.  
Incremental copies are used in conjunction with image copies.  An incremental copy identifies files that have changed since the last image or incremental copy.  This makes the backup much smaller and therefore faster.  When you do a restore you first restore the image copy followed by any subsequent incremental copies.
There are also some 'backup' programs that simply copy files. They do not allow you to restore your computer to working condition. 
Most backup programs give you the option of where you backup to.  You don't want to use the built in drive, in case it dies. So that leave some sort of external storage. In the not too distant past it was reasonable to backup to floppy disks, to writeable CD and DVD drives. But these days the only reasonable choices are large USB connected Hard Drives or to the internet / cloud because typical HDs are just too big for other smaller media.  You can buy TB sized USB drives (and larger) for about $50-200. That is enough to hold several backups.
