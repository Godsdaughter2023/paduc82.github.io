My initial download of Windows 10 seemed to work but I backed out of it as soon as I discovered that MS Money didn’t work with it at the time, (it does now though).

My second download failed due to an unknown error.  This turned out to be because I had failed to uninstall my Dataplex, (an SSD cache for my hard disk).  Learning Point if you have a Dataplex or similar uninstall it before attempting the upgrade.

My third attempt completed the upgrade to Windows 10 but a number of applications I use did not work.  The Support website suggested reinstalling from the original sources and this was when I discovered that my machine had "lost” the DVD drive.  The bios could see it but not the OS.  Following some advice from the Community I logged onto the Answer Desk (support.microsoft.com) who carried out remote diagnostics on my machine without success.  As time was running on we parked the issue overnight and returned to it the following day.  This was when we discovered that the Motherboard in my computer did not actually support Windows 10.  Learning Point if you are suffering problems that seem to defy logic check that your hardware configuration actually supports Windows 10.  To do this log onto the Motherboard supplier’s website and enter your motherboard’s details. This will generally bring up a page with the specifications on it where you will find a link to their support / downloads page.  If you take the download drivers link it will have a drop down listing the supported operating systems and if Windows 10 is not on the list it isn’t supported.

I upgraded my motherboard and CPU configuration to one that I knew supported Windows 10 and tried the upgrade again.  This time it worked but a number of drivers were absent including the LAN driver which meant I had no internet access; so back to Windows 7 again where I downloaded all the Windows 10 drivers for my motherboard and CPU configuration onto a USB drive.  I then ran the Upgrade process again which completed with the same missing drivers but this time Windows could find them because they were physically on my machine and install them.  Learning point downloading Windows 10 drivers from your hardware supplier’s support site before you run the upgrade process can help avoid problems.

I now had Windows 10 on my computer and all my software worked but there was one final hurdle to overcome.  Windows 10 refused to Activate because I had changed my motherboard and CPU.  On advice from the Community I went back to the Microsoft Answer Desk who again ran remote diagnostics on my machine to confirm:

  1. Roll back my computer to Windows 7 and ensure that my Windows 7 O/S was activated.  [Control Panel, then System, then Change Product Key where you enter your Windows 7 product key when requested which should return a Genuine Microsoft logo.]

  2. Once you have done this you run the Windows 10 Update from the Windows Update panel and you should wind up with a version of Windows 10 on your computer that will activate once it has connected to the internet.  This can take some hours.
