lemp say:

Windows Photo Viewer is not a "program" as that term is commonly used.  It is a DLL file.  The package of files that comprise Windows Photo Viewer is not available as a separate download.

You may find that free IrfanView is a far better and more versatile image viewer (https://www.irfanview.com/).

Please do the following to test for the presence and proper operation of WPV:

Open Windows Explorer (the file manager) and navigate to C:\Program Files\Windows Photo Viewer (if you have a 64-bit system, there also is C:\Program Files (x86)\Windows Photo Viewer).  You should see the following:

Choose one of your image files and obtain its complete path.  For example:

        C:\Users\ROBRCR\Pictures\IMG_01234.jpg

Open a Command Prompt (click Start, type Command Prompt in the Search box, and press Enter)

Type (preferably, copy and right-click>Paste) the following command into the black Command Prompt window; do NOT press Enter just yet (the command should be typed/pasted as one line -- it may wrap here in the forum)

 %SystemRoot%\System32\rundll32.exe "%ProgramFiles%\Windows Photo Viewer\PhotoViewer.dll", ImageView_Fullscreen

Type a space after "Fullscreen" and then copy or paste the full path to your image file.  Do NOT use double quotes around the path.  Press Enter at the end.  The complete command, including the image file, should look like this:

 %SystemRoot%\System32\rundll32.exe "%ProgramFiles%\Windows Photo Viewer\PhotoViewer.dll", ImageView_Fullscreen C:\Users\ROBRCR\Pictures\IMG_01234.jpg

Did WPV open displaying your image when you pressed Enter?

What is a Nintendo Video Game Emulator?

A Nintendo emulator is a software program that is designed to allow game play on a platform that it was not created for. The video games are obtained by downloading illegally copied software, i.e. Nintendo ROMs, from Internet distributors. Can I Download a Nintendo ROM from the Internet if I Already Own the Authentic Game?

There is a good deal of misinformation on the Internet regarding the backup/archival copy exception. It is not a "second copy" rule and is often mistakenly cited for the proposition that if you have one lawful copy of a copyrighted work, you are entitled to have a second copy of the copyrighted work even if that second copy is an infringing copy. The backup/archival copy exception is a very narrow limitation relating to a copy being made by the rightful owner of an authentic game to ensure he or she has one in the event of damage or destruction of the authentic.
