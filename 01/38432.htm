tryx3 say:

Joshua,

The latest update seems to have overwritten your display driver with an inappropriate one [Windows 10 does make mistakes when it comes to identifying the correct driver for particular hardware].

1  Check the correct name for your display card in your own notes or old records of what it was called in Device manager before the current problem happened.

2  Go to Device manager, Display adapters. 

  • Double-click on the entry for your adapter to open its Properties & select the Drivers tab. 
  • Make a note of the card name, driver version & date.  Click on Roll back driver unless it is greyed out [which I do not expect judging from your description]. 
  • Make a note of the replacement card name, driver version & date. 
  • Test it.

3  Even if step 1 fixes things, go to the display card maker's website and get a copy of their latest driver for your display card {hence checking its name carefully} and, if it is different, a copy of the rolled-back one above if it worked.  MS support staff will not necessarily have tried the drivers here as they will have incorrectly assumed that the display card maker will have passed them the most suitable driver anyway .

4  If you do not get any joy then you can, as a workaround, restart the computer and go into the BIOS/UEFI and manually tone down the default brightness setting to some sort of compromise level so you can at least get on with your job while you search for a better solution.  I cannot tell you where in the BIOS/UEFI this will be because they are all different.

5  If you have still not fixed things permanently, go to Device manager, Display adapters, {your card}, Properties, Details tab.  Scroll down the property list and show the Hardware IDs [specific identification codes for your hardware].  Right-click on the list and copy it into a text file.  Search online for these Hardware IDs starting in the support site & any user forums for your display card.

6  To avoid these problems being caused by hardware driver updates in the future, you can go to Control panel, System, Advanced system settings, Hardware tab, Device installation settings and set it to No. 

  • This is a very broad switch, it will prevent all hardware updates not just those for the current problem so you would want to make a note to check for updates to your hardware drivers every so often. 
  • In Windows 10, hardware is much more closely integrated with the OS in an attempt to make the computer more secure [against malware] so supported hardware does get much more frequent updates than it used to because it needs to maintain compatibility with Windows updates as they are released.

Best of luck,

Denis

Is there another user account, such as Administrator, within the active profiles? If things go south again, try login with a secondary user account.

If it works with an Administrator or Guest account, then we have a corrupted user profile.

If it works from another computing device, with the same User ID and password combo, then it appears likely again that the user profile is damaged. Rebuilding it is an option.

  • Username: admin

  1. Open a web browser.

  3. When prompted, enter "admin" for the user name and leave the password blank.

  5. Select "Wireless Settings" from the menu to the left.

Audio notifications can come from any running program or process, and most applications will have an option to turn sound off. Click Start or tap the Windows key and type (or paste) in mmsys.cpl and hit <enter>.


Figure 1.0 - Sounds dialog

 With the Sound dialog open, click on the Sounds property page (e.g. tab). Select an item from the list of Program Events that has a small speaker next to it and click Test. Move down the list until you identify one or more sounds that you want to disable. To disable any event's sound, simply change the sound clip listed in the Sounds: dropbox to (None).
