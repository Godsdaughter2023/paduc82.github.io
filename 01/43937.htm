"Otshot unable to load skin".


Procedure

1. GO TO " Task Manager"; Click Processes; select OTSHOT; Select End Process.

2. Click "MY COMPUTER ICON"; enter in Search box enter OTSHOP; click GO.

There may be 4 to 6 files of OTSHOT found.

3. Right Click on each file found {of OTSHOT}; and Rename each file to a dummy name such as [for example]  xxxOTSHOT.  [you may need to run as administrator when renaming files].

4. Close down the computer and RESTART Windows.

  The problem should now be gone.


5. CLEANUP PHASE: 

Click "MY COMPUTER ICON"; enter in Search box enter xxxOTSHOP; click GO.

The files found  of xxxOTSHOT will be displayed.

6. Right Click on each file found {of xxxOTSHOT};  Delete each dummy file [xxxOTSHOT].


The problem is gone.  Note: that this probably does not take care of the registry problems.  But you will no longer be annoyed.  Editing the registry [is risky] should be done by people that have a very good knowledge of Windows.
