How to delete System Volume Information

Okay so I accidentally put a System Volume Information file on my friend's USB, and I also accidentally moved 8 GB of stuff into it.... cmdrkeene say:

Well to get the files moved somewhere else, you can just drag-and-drop.

You can format the USB drive (erase everything) by right-clicking it in your File Explorer.

lalalalauren say: Okay, so I tried to move the files inside of the SVI to a different folder in the USB, but it's not working. Nothing happens. And I don't want to delete the entire USB, just that one folder.

cmdrkeene say:

Let me add some clarification:

1). The System Volume Information folder is present on all drives that use the NTFS file system.

2). This folder cannot be removed unless you choose another kind of file system.

3). The only way to choose another kind of file system is to format (erase) the drive and pick a new file system at that time.

Since you probably did not format the drive yourself, that folder was probably always there and you just didn't notice it.

In fact, it was probably hidden from view entirely.  You can set your folder options to hide operating system files like this by choosing Folder Options > View > Hide protected operating system files and "hide hidden files and folders".

lalalalauren say: Thank you for the information, but the real problem is how do I erase JUST what's INSIDE that folder without erasing the entire USB? Let me clarify: I don't necessarily want to erase the SVI, but I put outside files into the SVI so I wpuld just like to erase THOSE files.

cmdrkeene say:

You can't.  If you try to erase the contents inside the System Volume Information folder, you will destroy the drive and all files on it.  It's required and part of the system of the drive.

You can hide it from view with the options, but that's it.  It's absolutely part of the flash drive's file system (and the same folder exists on all hard drives too).
