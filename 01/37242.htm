daft badger say:

Error 8007-0070 occurs when installing an update or buying content on Xbox 360

Problem

You see the following error code and message when you try to update your Xbox 360 console or purchase content:

xxxx-xxxx-xxxx-xxxx-8007-0070

Can't download the update. To test your connection, open System Settings and select Network Settings, Test Xbox Live Connection.

Note The “x” characters in the error code vary.

This may mean that there's not enough space on your storage device to install the update or the content.

Solutions

To resolve this problem, try the following solutions.

Solution 1: Free up space on your storage device

You need sufficient free space on your storage device to accommodate a console update or a content download. If you don't have enough free space, you need to move or delete some items. For information about how to do this, see Copy, move or delete Xbox saved games, profiles and avatar items.

Note You may also be able to resolve this problem by obtaining a larger-capacity hard disk. For example, the Xbox 360 320 GB Media Hard Drive is available from the Microsoft Store.

Solution 2: Delete previously installed updates

1.On your console, go to Settings, and then select System.

2.Select Storage.

3.Highlight the memory unit.

4.On the controller, press the following series of buttons: LB, RB, X, LB, RB, X. This opens the Delete System Update window.

5.Select Yes. This restarts the Xbox 360 console.

6.When the console restarts, you are prompted to install the latest updates. Follow the instructions to install the updates.

If you keep experiencing the problem, contact Xbox Support.
