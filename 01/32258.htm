There are some misunderstandings regarding the licensing for Windows 8.

First of all, the Windows 8 Pro license you originally bought, is not a full version if it was bought at retail. It is a Upgrade version.

To install an upgrade version of Windows 8, you must already have Windows 7, Windows Vista, or Windows XP installed on your PC.

If you formatted the drive before the upgrade version of Windows 8 was installed, you won't be able to use your upgrade product key to activate Windows 8. To activate Windows 8, you'll need to install your previous version of Windows, and then reinstall Windows 8.

The free Media Center product key that was offered during the launch of Windows 8 in late 2012, required that you first had an installed and activated copy of Windows 8. Once you have done that, you can apply the product key for Media Center through Add Features, then you will have to reactivate again.

So, two instances of phone reactivation would be required:

- Windows 8 Pro

- Media Center key

There was no free product key for Windows 8.1, Windows 8.1 was provided as a free upgrade to existing Windows 8.1 users through the Store.

Presently you can download updated Windows 8.1 with Update 1 media to avoid going through all processes of downloading 8.1 then Update 1. Learn more at the following link:

http://dellwindowsreinstallationguide.com/download-microsoft-windows-and-office/download-microsoft-windows/download-windows-8-1-retail-and-oem-iso/

frederik long say:

The command

net user administrator /active:yes

will make the Administrator account visible at the logon screen.
_________________________

If this response resolves your problem, please click "Propose as answer" so that the thread can be closed. Thanks.

frederik long say: OK, I will be more detailed:
1. Create an administrative account.
2. Log on under this account.
3. Click Start.
4. Type the three letters cmd into the Search box.
5. Press Ctrl+Shift+Enter
6. Click "Run as Administrator".
7. Type the following command:
   net user administrator /active:ye
