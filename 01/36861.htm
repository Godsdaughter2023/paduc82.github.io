Select Start, All Programs, Accessories, right click Command Prompt, and click Run as administrator from the drop down menu. Type or copy and paste:
findstr /c:"[SR]" %windir%\Logs\CBS\CBS.log >"%userprofile%\Desktop\sfcdetails.txt"
and press the ENTER key. You will find an sfcdetails.txt file on your Desktop.

 

Please note there are three spaces in the command:
Between findstr and /c,
Between [SR]" and%windir
Between CBS.log and >"%userprofile

 

Please upload a copy of your sfcdetails.txt file to your OneDrive as a shared file and post a link here. Enter sfcdetails.txt in the Search box above the Start button and press the ENTER key to locate the file created.

 

Note The Sfcdetails.txt file contains details from every time that the System File Checker tool has been run on the computer. Verify the date and time entries to determine the problem files that were found the last time that you ran the System File Checker tool.

Error code 8007000B occurs when you try to sign in to Xbox LIVE

Overview

If you see error code 8007000B when you’re trying to sign in to Xbox LIVE, it could mean one of several things:

   Certain Xbox LIVE features aren’t available because of system maintenance or because of an unexpected Xbox LIVE issue.

   Your console might not be connected to the Xbox LIVE service.

   You might not have the necessary ports open on your network.

Read this page to learn more about 8007000B and what you can do about it.

Error code and message

Error code 8007000B is preceded by the following message:

   Sorry, there's a problem with the Xbox service. Try again later.

   Status code: 8007000B

Problem

If you see error code 8007000B when you’re trying to sign in to Xbox LIVE, it could mean one of several things:

   Certain Xbox LIVE features aren’t available because of system maintenance or because of an unexpected Xbox LIVE issue.

   Your console might not be connected to the Xbox LIVE service.

   You might not have the necessary ports open on your network.

Solutions

Solution 1: Check the status of the Xbox LIVE service

Certain Xbox LIVE features may not be available at times because of scheduled maintenance or because of unexpected Xbox LIVE issues. To determine the Xbox LIVE service status, go to the Xbox LIVE Status page.

If there is an Xbox LIVE service alert, please try again later when the Xbox LIVE service is available. If the Xbox LIVE service is available and you keep getting this error message, try the next solution.

Solution 2: Verify that your console is connected to the Xbox LIVE service

To test your Xbox 360 console’s Xbox LIVE connectivity, follow these steps:

   On your console, go to Settings and select System.

   Select Network Settings.

   Select your wireless network on the list that appears. If you do not use a wireless network, select Wired Network.

   Select Test Xbox LIVE Connection.

   Read any on-screen service messages that appear.

If you get a message saying that your console is not connected to Xbox LIVE, see Xbox LIVE connection problems for more information.

If the Xbox LIVE service is available but your connection is not working, try the next solution.

Solution 3: Open network ports

To access Xbox LIVE, several ports need to be open on your network hardware or firewall. For information about which ports must be open for Xbox LIVE, see Network ports used by Xbox LIVE.

If you keep getting this error message, try the next solution.

Solution 4: Contact Xbox support

If the previous solutions don’t solve the problem, contact Xbox support.

If no other USB devices are connected it could be that the motherboard does not support NX bit or No-eXecute. It is a setting the motherboard must support (and must be enabled) for Windows 10. That could be why the install fails.

Go into the bios and see if you can find the setting.

e. Then Uncheck the option which reads” Use a proxy server for your LAN”
f. Click apply and then ok.
