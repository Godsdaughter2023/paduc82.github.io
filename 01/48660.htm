If the laptop came preinstalled with Windows 7 Professional, that means its a OEM license. OEM licenses are not transferrable and they are tied to the first computer they are installed.

If its a retail store bought license, it is transferrable to another computer. Retail licenses include documentation, certificate of authentication and reinstallation media, which you should have if you want to transfer it to another computer. You don't seem to have these things, which would make it impossible to transfer the license to another computer.

 

If its a Windows 7 Professional upgrade license, the computer you are transferring it must already have a preinstalled Windows XP, Vista or Windows 7 license.

The requirements for upgrade media though is that you have a previously qualifying operating system such as Windows XP or Vista installed to use it. From the Windows 7 End User License Agreement.

15. UPGRADES. To use upgrade software, you must first be licensed for the software that is eligible for the upgrade. Upon upgrade, this agreement takes the place of the agreement for the software you upgraded from. After you upgrade, you may no longer use the software you upgraded from.

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee.

Hi ThaibinhL,

This issue will require us to check different settings and perform a series of steps. Are there any apps that you use with OneNote exclusively when you are at your university? Were you able to use the Surface at the university before without experiencing this issue?

Let's see if an update was recently installed on your Surface. Go to Settings (Windows key + I) > Update & Security > View update history.

When the problem occurs while you're at the university, verify that the settings needed are applied for the palm rejection feature to work. In Settings, go to Devices > Pen & Windows Ink > make sure Ignore touch input when I'm using my pen is On. If it's already enabled, try changing the hand option under Choose which hand you write with, then check if palm rejection will work. Apart from that, try using a different app where you can write or draw with the Surface Pen, and see if palm rejection will also not work.

If the feature still does not work, download and run the Surface Diagnostic Toolkit. Afterwards, perform a forced shutdown. While the device is on, press the power button for about 10 seconds or until the screen goes off. Once the device is up, revisit the Pen & Windows Ink settings to make sure the settings are correct, then test if the palm rejection will work.

In case the steps are not successful in resolving the issue, we would have to perform further troubleshooting. Kindly update the post with your findings. I will be notified when you reply and I'll get back to you the soonest I can.

Regards.
