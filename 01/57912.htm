Hi Alex,

-What is the make and model number of the computer?

The fans purpose is to cool down the interior of the PC or, in the case of the CPU fan, to keep the processor cool. Most motherboards have temperature sensors which regulate when and at what speed the fan works. Initially the fan spins at full speed until the desired reduction is temperature is achieved and then the fan either slows down or, in some cases, stops completely until the temperature builds up again. Once the temperature hits its top limit the fan is initiated again and the cooling process starts all over again.

 

If the fan is on continuously then there is a possibility that the inside temperature of your PC is hotter than it should be. Also make sure that you are not using the laptop on a couch, bed or on a place where it block the fan vents.

Press Ctrl + Alt + Del to bring up Task Manager and find out which processes are running - sort list by memory usage and check which process produces higher CPU (Central Processing Unit) usage.

It may also be a virus/spyware/malware hogging resources and pushing the CPU so much that it needs to cool the case down.

Run virus scan using Microsoft Safety Scanner, which would help us to get rid of virus, spyware, and other malicious software.

Note: Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.

You may follow these methods:

Method 1:

SFC /scannow is a super-useful command you can use in any Windows version. When the SFC (System File Checker) command is used with the /scannow switch, the tool will scan all of the important Windows files on your computer and replace them as necessary.

Missing and corrupt operating system files (like many DLL files) are arguably the biggest cause of major Windows issues. Considering that, plus the fact that SFC /scannow is completely automatic and very easy to use, the tool should usually be one of your first troubleshooting steps.

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7
http://support.microsoft.com/kb/929833
