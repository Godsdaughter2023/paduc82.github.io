Step 1:

  • netsh winsock reset catalog (reset winsock entries)

Follow the steps to create a hotspot connection the laptop:

  • Press the Windows + R key to bring up a run box, type ncpa.cpl and hit Enter.
  • When your network connections open, right-click on your Wireless network adapter and select Properties from the context menu.
  • Switch over to the Sharing tab and allow other devices to use your machines internet by selecting the first checkbox “Allow other network users to connect through this computer’s internet connection” and then unchecking the second box before clicking on the OK button.
  • Press Windows+X and select Command Prompt (admin). Type the following command and hit Enter to setup hotspot connection:

netsh wlan set hostednetwork mode=allow ssid=”wireless network name” key=”Pa$$w0rd”

Where, ssid is the name of your network and key is the password you want users to connect with. Also make sure that the access point is created with WPA2-PSK (AES) encryption.

  • We are ready to start broadcasting our network:

netsh wlan start hostednetwork

  • The last command you will need to know is the show command, it will tell you things like the channel you are on as well as the number of clients connected to your network:

netsh wlan show hostednetwork

Check if you are able to connect to the hotspot network hosted on the laptop running Windows 8.

  3. Type Dism /Online /Cleanup-Image /CheckHealth and click Enter.
