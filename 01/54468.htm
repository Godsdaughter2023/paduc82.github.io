It appears that neither Microsoft engaged Support Engineer "expert" Mahesh B G or Microsoft engaged Support Engineer "expert" Babu V still do not know that it is not necessary to edit the registry to "reset" the Session Manager to its default value.

Let's try one more time to clear that up as I introduce them to the chkntfs and fsutil commands:

Chkdsk runs when you reboot either because you scheduled it to run on the next reboot and/or because your hard drive has been marked "dirty".

Perhaps the Microsoft engaged Support Engineer "experts" do not know that you do not have to edit the registry to clear a pending scheduled chkdsk.

To clear any pending scheduled chkdsk command (if there is one), click Start, Run and in the box enter:

chkntfs  /d

Click OK and you will not see any message, but that will clear any pending scheduled chkdsk commands and you will know that is not the problem.  If you reboot and chkdsk still runs, you have another problem.

XP automatically checks the "dirty bit" on all your volumes when it restarts.  If XP finds a volume that has been marked "dirty" it will run a chkdsk with error correction on the afflicted volume to try and clear the dirty bit and you will see a message stating so when XP starts.

A volume could be marked "dirty" after something like a power interruption, power failure, aborted restart or ungraceful shutdown.  Have any of those events occurred?

You can query the dirty bit on a volume from the command prompt window.

Click Start, Run and in the box enter:

cmd

Click OK to open the command prompt window.

For example, to query the dirty bit on drive C, type:

fsutil  dirty  query  C:

Sample output:
Volume C: is dirty
Volume C: is not dirty

Is your volume dirty or is it not dirty?

The only thing that can clear the dirty bit is a successful run of chkdsk with error correction, sometimes you need to run it more than once and sometimes that doesn't work at all and you will need to see if running chkdsk with error correction from the XP Recovery Console can clear the dirty bit.
