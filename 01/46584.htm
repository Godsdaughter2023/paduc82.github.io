Validate the installed copy of Windows

Validation is an online process that verifies that your copy of Windows is genuine and that critical Windows licensing files have not been damaged, deleted, or removed. Access the following Website and validate the installed copy of Windows 7.

Genuine Microsoft Software validation

http://www.microsoft.com/genuine/validate/

The first time you validate Windows 7 on your PC, you’ll need to download additional software components for validation to run properly.

If Windows on your PC is found to be non-genuine, these downloaded components can restore critical Windows 7 licensing files that might have been damaged, moved, or deleted.
