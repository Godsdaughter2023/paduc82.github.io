Hi

For the App Switcher, there is a new option to disable the top-left corner. Check the following setting to make sure that the App Switcher is not disabled.

On the Desktop, right click the Taskbar and select Properties.

Click the Navigation tab. In the Corner Navigation section, make sure that there is a check mark in the When I click the upper-left corner, switch between my recent apps option.

As for your concern about resource problems with the apps not being completely closed, this is not a problem with the new Memory Manager that is a part of Windows 8x.

From the user metrics feedback Microsoft has found that most users switch away from most apps, rather than exiting the apps and losing any unsaved information.

Prior to the new 8.1 Update, if you simply 'switch away' (start another app) after 5 seconds the Memory Manager will Suspend the app. The app will remain in Memory.

This saves the current state of the app allowing you to return to the app without losing your place or any unsaved information.

This allows you to have the app remain in the same state if you plan on returning to that app.

When the app enters the Suspended state, the memory it is using is reduced to a minimum.

If the new Memory Manager does detect a Low Memory state, it will immediately start ending task on all of the
