pauli o say:

Sometimes change between Private and Public network profile is missing from Settings / Networks page. In that case profile must change by PowerShell.

Right Click Windows Icon (Start button) and open PowerShell (Admin) and use following commands to get interface index number:

Get-NetConnectionProfile

Use that index number and give following command to change profile to Private (or Public):


Set-NetConnectionProfile -InterfaceIndex <index number> -NetworkCategory Private

kapil arya mvp say:

Hello,

To change network connection from public to private, try these steps:

1. Open administrative Windows PowerShell.

2. Type Get-NetConnectionProfile and press Enter key. Note the Network name value appeared now.

3. Type Set-NetConnectionProfile -Name "<Network-name>" -NetworkCategory Private, after substituting <Network-name> with value we got in step 2. Press Enter key.

Once command execution is complete, the network connection should be set as private now. 

You can also change network connection from public to private using registry manipulation. Refer steps mentioned here:

https://www.kapilarya.com/switch-network-from-public-to-private-and-domain-windows-10

Hope this helps!

Note  If you want to perform a detailed test of the hard drive, you can select the Scan for and attempt recovery of bad sectors check box.
