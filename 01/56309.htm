2 reasons this happens.

Antivirus software you are running or existing corrupt files.

First disable your antivirus while you are doing the update.
Second do the steps below in order.

Go to search and type command prompt and right click and select to Run as Administrator.

Type each of the following then hit Enter

taskkill /f /fi "SERVICES eq wuauserv" (do this command at least twice in a row)
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
rmdir C:\Windows\SoftwareDistribution\DataStore
rmdir C:\Windows\SoftwareDistribution\Download

REBOOT

Still doesn't work then Go to search and type command prompt and right click and select to Run as Administrator.
Type one after another

Net Stop bits
Net Stop wuauserv
Net Stop appidsvc
Net Stop cryptsvc
Ren %systemroot%\SoftwareDistribution SoftwareDistribution.bak
Ren %systemroot%\system32\catroot2 catroot2.bak
Net Start bits
Net Start wuauserv
Net Start appidsvc
Net Start cryptsvc

Try update again.
