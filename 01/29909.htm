^^^ You can, it just depends on the game and if they allow it. On top of that, whatever game file you do this with will prevent you from unlocking achievements while using that file.

Example.

Lets say your first account is named "I Heart Games"
then you created a new gamertag on the same console/HDD...lets call that one "Video Games Rule"

So naturally, you wont have any progress on any of your games while using this new gamertag and you will also have 0 achievements too.

Let's say you are one of those rare few who don't care about the achievements...and you simply want to get your hands on your save file that you have played for so long and invested some serious time into.

What you can do is.
Sign into your new gamertag "Video Games Rule"
scroll over to your storage section, games and look at the saves.
We can use Skyrim for an example. You would scroll down to Skyrim, press "A" to look at your save files.
When you see them, you will notice all of your saves say "Owner I Heart Games"

Pretty much telling you that in order to access that save, you need to be on the profile.

Now you can select a desired save file and then you will see 3 options. Move Copy and Delete.

What you can do is Select Copy. If Skyrim allows you to do this, you will now receive a pop up message that says something along the lines of...

"You are not the original owner of this save file, You will not be able to unlock any achievements if you continue to copy this save file onto your gamer profile"

After you select yes, you will now notice that a new save file has appeared on your list but now it will say "Owner Video Games Rule"
