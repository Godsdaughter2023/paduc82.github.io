I got a temporary solution to get the gestures back, without the trouble of reboot.

1. open "Task Manager"

2. In the "processes" tab, find the process "synaptics touchpad 64-bit enhancements".  For a different driver version, the process may get different names.  

3. right-click and select "open file location". Remember its location, just in case the program won't restart automatically.

4. in "task manager", end the process  "synaptics touchpad 64-bit enhancements".

The process should restart instantly as if it had not been ended at all.  If it do not restart, run it manually.

Then gestures are back.

It works on my Inspiron 14 5447.  Touchpad driver version is 19.0.15.2.  Other version might get a different name for that process. Its favicon in "task manager" is the same as the touchpad icon on the down-right corner of desktop.

lemp say:
Although you could follow those instructions, usually the place to create new folders and documents is in My Documents (just so that you have all of your user-created material in the same place).

I suspect that the reason the instructions have you place My Website in the root of the drive is so that it's easier for the person writing the directions when they later have you save things there.

Click Start and then click My Documents.  This opens a Windows Explorer window showing the contents of your My Documents folder. 

Depending on how your computer is set up (and which version of Windows you're using), you may see a blue column on the left side of the window with the heading "File and Folder tasks."  See this image: http://onlineitpro.com/wp-content/uploads/2007/12/mydocs.png

If you have the blue column, just click on the link that says "Make a new folder."  A new folder will appear at the bottom of the list in the right pane with the words "New Folder" highlighted.  Just type My Website and press Enter and the new folder will be renamed My Website. 

If the blue column does not appear, just click File (at the top left) and then New and then Folder.  This is exactly the same as clicking the "New Folder" link in the File and Folder tasks column.

If you want to follow the directions exactly, after you click Start > My Documents, just type C:\ in the address bar and click the green Go button.  This will change the display from your My Documents folder to the location where the instructions want you to put the folder.  Then follow the rest of the directions above.

winston m say:

Hi,

 

1. Have you connected any external devices to the computer?

2. Are you able to boot the computer to safe mode?

 

Let's analyze and identify the cause of the issue. Perform the steps provided here and check.

I would suggest you to unplug all the external devices to the computer except mouse and keyboard to see if this helps.

 

Automatic Repair

 

Note: To perform this step, you need to have a bootable Windows 8.1 installation media such as disc or USB.

I would suggest you to create a bootable Windows 8 USB drive and then perform Automatic repair on your computer so that it starts Windows normally.

 

Note: You can create installation media using a different computer.

 

Create a Bootable Windows 8 USB drive by following the information provided in this link.

http://windows.microsoft.com/en-IN/windows-8/create-reset-refresh-media

 

Startup Repair is a Windows 8.1 system recovery tool that can fix certain problems, such as missing or damaged system files.

 

To perform Automatic repair on Windows 8.1 computer follow these steps:

 

a. Insert the media such as (USB or DVD) and restart your computer.

b. Press F12 key (typically this is F12, but it can differ between computer manufacturers) and choose the drive that you inserted the installation media into.

c. Once the Windows Setup window appears, follow these steps:

d. Click next and selectRepair your computer.

e. You will then see a blue screen and an option to choose. Choose the option Troubleshoot and select advanced options.

f. You may choose Automatic Repair from Advanced boot option.

 

Hope this information helps. Reply with the status so that we could help you further.
