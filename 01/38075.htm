Hi,

Welcome to Microsoft Windows Vista Forums.

·         This problem may occur for the following reasons:

·         You assume that you are performing a single physical click, when in fact, you are performing multiple clicks.

·         The mouse settings have been modified, either manually or by another program, to send a multiple-click command when you perform a single physical click.

·         The mouse drivers are corrupted.

·         The mouse hardware is defective.

Try these steps:

Method 1:Verify your clicking method

Make sure that you are performing a true single physical click. A single-click occurs when you press and release a mouse button one time.

 

Method 2: Test the mouse in multiple programs

Close all programs and test for the problem on the desktop and in other components of Microsoft Windows, such as Control Panel or Microsoft Windows Explorer.

 

If the problem only occurs in a particular program, troubleshoot that program. The program is most likely customizing the mouse commands.

Method 3: Test the mouse in safe mode

·         Click Start, click the arrow next to the Lock button, and then click Restart.

·         Use one of the following methods:

1.      If the computer has a single operating system installed, repeatedly Tap the F8 key as the computer restarts. You must press F8 before the Windows logo appears. If the Windows logo appears, you must try again.

2.      If the computer has more than one operating system, use the arrow keys to highlight the operating system that you want to start in safe mode, and then press F8.

·         On the Advanced Boot Options screen, use the arrow keys to highlight the safe mode option you want, and then press ENTER.

·         Test the mouse.

·         To leave safe mode, restart the computer and let it start in Normal mode.

             Method 4: Troubleshoot your software

Windows Vista

·         Click this image, type Appearance and Personalization in the Start Search box, and then click Appearance and Personalization in the Programs list.

 

·         If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

·         In Appearance and Personalization, click Folder Options.

·         On the General tab, click Double-click to open an item

Method 5:Remove and then reinstall the mouse drivers

Windows Vista

·         Click Start type Add or Remove Programs in the Start Search box, and then click Add or Remove Programs in the Programs list.

 

·         If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

·         Click The <Mouse name>, click Change, and then click Remove.

·         Restart the computer

·         Let the Vista itself search for the Drivers and install it automatically or download the  latest drivers form the manufacturer’s website.
