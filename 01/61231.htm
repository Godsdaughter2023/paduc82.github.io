We simulated your issue and we encountered the same thing. It seems that the Date Created column is the date where you have moved or changed the location of the file and the Date Modified column is the actual date when the clip was recorded. In the current design of Windows  7 operating system the Date Created will always change as long as you move or change the location of the file and the Date Modified will be the original date of the file when it was created and it has nothing to do with Windows Live Photo Gallery. How To Fix Windows Update Error 80070003

Method 1: Stop the Automatic Updates service.

    Click the "Start" button, type "services.msc" in the search box.
    Click "services.msc" on the pop-up "Programs" menu.
    On the list of "Services" windows, right-click "Automatic Updates" and the click "Stop".

Method 2: Rename the SoftwareDistribution directory.

    Click the "Start" button, type "cmd" in the search box.
    Click "cmd" on the pop-up "Programs" menu.
    In the "C\Windows\system32\cmd.exe" windows, type "cd %windir%" and press Enter, type "ren SoftwareDistribution SDTemp" and press Enter, type "exit" and then press Enter.

Method 3 :Restart the Automatic Updates service.

    Back to the "Services" windows, right-click "Automatic Updates" and the click "Start".
    Close the "Services" windows.

Method 4 :Download Updates from Windows Update Web site again and reinstall Windows Update.
