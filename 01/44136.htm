Hello,

Be informed that you cannot delete an administrator account being logged into that account.

I suggest you to enable the hidden administrator account and try deleting the account with hidden administrator account.

Follow the steps to enable the Built-in Administrator Account and try deleting the account from this administrator account.

Steps to Enable the Built-in Administrator Account.

a)      Press Windows + X, Select Run.

b)      Type   net user administrator /active:yes

c)       Close the elevated command prompt.

d)      The built-in Administrator will now be available to select to sign into.

e)      Restart the system.

Then enter into the administrator account and try to delete or remove the account and see if it helps.

Once you create the new administrator user account try to disable Built-in Administrator Account.

Steps to disable the Built-in Administrator Account.

a)      Press Windows + X, Select Run.

b)      Type   net user administrator /active:no

c)       Close the elevated command prompt.

d)      The built-in Administrator will now be available to select to sign into.

e)      Restart the system.
