lemp say:

DAT files attached to emails are almost always superfluous.

Several versions of Microsoft Office ago, there was an issue that caused some email to have a winmail.dat attachment.  This was caused because that version of Outlook and/or Microsoft Exchange Server sent "Rich Text formatted" messages using a proprietary format that no other email client could understand.  See, e.g., https://support.microsoft.com/en-us/kb/278061 and https://support.mozilla.org/en-US/kb/what-winmaildat-attachment

I don't think a file named ATT000nn.DAT is the same thing, but it may be some sort of similar formatting information.  If you can read the rest of the message, I would just ignore this attachment (I do).

If you're really curious, when you right-click on the file, select Notepad as the program to use to open it and see if there is anything intelligible there.
