If this popup is happening when your Windows Starts up, then you have to find the process that refers to trying to open a media file:

  1. Press Ctrl + Shift + Esc keys together to open Task Manager
  2. On the Startup Tab look for anything relating to this file
  3. If you find something related, select it and click 'Disable'

  1. Press Windows + R keys together to open the Run Dialog
  2. Type msconfig in the run dialog and hit enter
  3. Click on the Services Tab of the resulting dialog
  4. Check the box marked 'Hide all Microsoft Services'
  5. Look for anything relating to this file
  6. If you find it, uncheck it
  7. Click Apply and Reboot
