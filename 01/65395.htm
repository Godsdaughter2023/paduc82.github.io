Hello Collin,

Thank you for posting you concern in Microsoft community and welcome to the Windows 10 Family.

To be able to assist you better, provide us a few more details on the below questions.

  • Do you save the back up on a hard drive locally or onto any network drive?
  • Are you connected to any domain network?

Now, certain computer manufacturers support this feature inbuilt into BIOS settings which you can access through BIOS setup.

In Windows, you can use the Windows Task Scheduler to wake up your Windows computer wake from sleep mode, automatically at a pre-set time.

Follow the below steps to wake up your computer from Sleep at a particular time (midnight, in your case).

  1. Type Task Scheduler in Start Search and hit Enter.
  2. In the right side, select Create Task. A new window will open. Here under the General tab, fill in the Name and Description. Also check, Run with highest privileges. Also check "Run whether user is logged on or not"
  3. Under the Triggers tab, click New. Another window will open. here. Select One time (or Daily if you want it repeated every day at a particular time).
  4. Next under the Actions tab, you have to mention a task. Click New. You can run a simple task, like say, first launching & then closing the command prompt window. This will wake up Windows at the pre-set time to carry out the task.
  5. Select Action as : Start a program. To schedule a task that executes the command cmd.exe with command line arguments, copy-paste under Program/script.
  6. Under Conditions tab, check on the Wake the computer to run this task check-box.
  7. Click OK and Exit Task Scheduler.

Hope the above information is helpful. If you need further assistance, feel free to write to us and we would be happy to help you.

Thank you.
