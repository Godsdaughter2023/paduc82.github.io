skype voicemail

josephv_a say:

Hello

If you are trying to set up voice messaging, please follow the following steps:

  1. Sign in to your account.
  2. In the Manage Features section, select Call forwarding.
  3. Select the button to enable call forwarding and then select how many seconds to forward calls after.
  4. Select Voice messaging, then Confirm. 
  5. If you buy an unlimited subscription or a Skype Number, voice messaging is activated automatically.

Please be advised that voice messages that are not listened to are stored on our servers for 60 days before they are deleted.

However, if you've listen to a voice message, it is transferred from our servers and stored locally* on your computer. This means that you won’t be able to listen to it if you sign in to Skype on a different computer.

*Messages are stored locally as .DAT files in the following locations:

Skype for Windows desktop:
C:/Documents and Settings/username/Application Data/SkypeName/voicemail.

Skype for Mac OS X:
Library > Application Support > Skype > Skype_username > Voicemail.
