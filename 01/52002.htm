Hi Daniel,

 

There is no option to rename Apps on the Start menu, you can change the size and pattern of the tiles but not the name. Let’s try to unpin the icons from start menu and Pin them again to start menu.

 

Method 1:

To unpin app from the start menu. To unpin follow the steps below.

  • Open Start screen, Select the My computer.
  • Right click on the My computer icon.
  • From the options, select Unpin from start.
  • Once you have unpinned my computer and control panel from start menu, pin them again to start menu again and check if it helps. To pin the icon to start menu follow the steps below.
  • Right click on My computer icon.
  • From the options, select pin to start.

Note: Repeat the above steps for control panel icon.

  • Right click on the control panel Icon on start menu, select more and click on open file location.
  • Select control panel icon, right click on it and select rename and change the name.
  • Once the name is changed right click on control panel icon and select pin to start option.

Note: Repeat the above steps for My PC or My computer icon.
