afzal taher say:

Hi,

There is no option to disable restarting the applications. But you can disable automatic restarts after automatic updates. Automatic reboot is required
because some updates require access to files currently in use by your system.  Windows Update does not know if or when those files will be available to
use, so your system requires a reboot to avoid any conflicts. For security-related updates, your computer is unsafe before that reboot occurs: it is
vulnerable to attacks which exploit the issue(s) fixed by the update requiring a reboot. 

You can configure Windows update to suit your needs, for example, download the update and then let you choose when to install the same. That way
you still get the automatic updates but you chose when and where and how to install the updates at a time that is convenient.

You may follow these steps to turn off automatic restart after installing automatic updates:

1. Go to start and type in gpedit.msc in the search box and press enter.
2. Once inside the Group Policy Editor, expand Local Computer Policy.
3. Then expand Computer Configuration and click on Administrative Templates.
4. Expand Windows Components and then select Windows Update.
5. Check for the option “No auto-restart with logged on users for scheduled automatic updates installations”.
6. Enable the option.
7. Exit Group Policy editor.
