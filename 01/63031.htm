a. user say:

Hi Henry,

 

The main difference between an Administrator account and a Standard account is that the Standard account has limited privilege when compared to an Administrator account. You will need an Administrator account to make any critical changes to your computer. If you are the only one who uses the computer, then it’s recommended to use the Administrator account.

 

A Standard account is useful when others access your computer frequently. If you have children who access your computer frequently, you can activate “Family Safety” with which you can restrict and monitor their usage. A standard account also comes handy when you want to make sure that others who access your computer can’t make any critical changes to the computer.

 

Coming to your next question, each user account have a different folder where all the user data like downloaded files, user settings and other data are stored. So when you download a document, it gets stored in the user folder for that particular folder and it doesn’t get transferred to other user folders. You can view the files in other user folders from an Administrator account but you can’t view the Administrator account files from a standard account. Hence, if you are downloading any confidential documents that you don’t want to be accessed by others, it’s recommended to use an Administrator account.

 

Hope this answers your questions. Feel free to get back to us if you any Windows related queries in the future; we’d be happy to help.

Download and run JDiskReport.  That should show you graphically which files are taking up the space.

Most normal Windows installations need only 1 pagefile, which by default is on the system drive (the one where Windows is located).  To check pagefile configuration:
  • Control Panel > System
  • Click "Advanced system settings"
  • On the "Advanced" tab, click "Settings" in the "Performance" section
  • Click the "Advanced" tab in "Performance Options"
  • Click "Change" in the "Virtual memory" section
Default should be: check the box to "Automatically manage paging file size for all drives" and the table below that should show "System managed" for the system drive (usually C:) and "none" for all of the other drives.
