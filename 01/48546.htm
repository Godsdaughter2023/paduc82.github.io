Neither product has the ability to stop adware that you have voluntarily installed on your computer. You need to find out what the adware program is and uninstall it.

When you install a program from the internet without using the 'custom install' option you will often install adware. That's because lots of programs that you download from the internet will install additional software without asking you, and much of that additional software is adware.

Your anti-virus program won't stop this from happening unless the adware program is malicious. While most people find adware annoying, most adware is not harmful to your computer.

That's why you always need to choose the 'custom install' option, which gives you the opportunity to stop the program from installing additional software.

Find the list of installed software: Start > Settings > System >Apps and features > select the adware program > click the Uninstall button that appears.

If you don't see the adware program or you don't know the adware program's name, then uninstall the last program you downloaded from the internet. If that makes the adware stop, you can re-install the program you downloaded, but this time use the 'custom install' option and watch carefully for your opportunity to prevent any additional software from installing.

burrwalnut say:

To change the interval between system restore points in Win XP, the registry key RPGlobalInterval is changed. This key doesn’t exist in Windows 7 (not in my version anyway), but it will not cause any harm if you create the key and it is not accessed by system restore as you can simply delete it.

 

Open a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\SystemRestore.

 

Right-click ‘white’ space in the right pane then select New > DWORD, name it RPGlobalInterval and set it to the value you want (entering 86400 will set the interval to 24 hours in seconds).

 

To test it out, set it initially to a low value, say 21600 (6 hours) and check the results over a period of several days.

 

If you’re apprehensive about making registry changes, backup the registry (or create a system restore point) before making any changes.

frederik long say: You're probably using the wrong tool if you intend to keep restore points for months or years. They are designed so that you can back-track in case something goes wrong. Going back weeks or months is unlikely to work reliably.

If you wish to have an ability to go back a long way into the past then you should create system images, either with the native image creation tool (which I find rather marginal) or with a third-party imaging tool such as Acronis True Image. To use them effectively you need to split the hard disk: Use drive C: for your system and program files, drive D: for your data.
