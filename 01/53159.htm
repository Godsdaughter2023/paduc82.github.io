what's the difference between sleep or hibernate - Microsoft Community

francislittleton say:

what's the difference between sleep or hibernate

When I close the lid on my laptop, does it have to be set to sleep or hibernate, to allow it to continue downloading?

pranavmishra5582 say:

Hi Francis,


Thank you for posting the query on Microsoft Community.

 

A sleeping computer awakens quicker than if you set the same machine set to hibernate. The price to pay is that a sleeping computer could lose unsaved data if someone turns it off, this is because the information is stored in RAM, which is cleared if the operating system powers down.

 

Although a hibernating machine takes longer to return to its previous state, the data is safe because it reloads the contents of a physical file called hiberfil.sys into memory. Consequently, if someone accidentally turned off your machine while it was hibernating your data would still be just as you left it. Another benefit of hibernation is that it consumes less power than sleep mode.

 

You cannot download anything while your computer is in Sleep or Hibernate state.

Both of these HDMI ports will be outputs to connect an external monitor/TV to. Neither can accept a signal in.

 

It might be possible access the TataSky box via Ethernet as a network device, you cannot connect them together with USB as again they are both master devices. Network devices can only be connected together with a crossover cable, you would normally interconnect them with a network switch which does the crossover for you.
