a. user say:

"So I "refreshed" the system which should have removed only the programs I downloaded subsequent to the Windows 10 install."

No, that's not what a Refresh does.

A Refresh uninstalls Windows and all your software while leaving your user folders (Documents, Music, etc.) untouched and saving certain account preferences in the cloud.

Then Windows is reinstalled from scratch and your Windows Store apps are re-installed. Applications that did not come from Windows Store are not re-installed. Instead, Windows leaves an HTML file on your desktop with a list of the software it removed.

Following this, Windows sets up your user account(s) again, using the account preferences it saved in the cloud.

For more details, search the web: what is a windows refresh?

To learn how to install your software, go to the developers' websites (Adobe, Cyberlink, etc.) where you will find instructions for downloading and installing. To reinstall software that came from Microsoft (e.g., C++ redistributables) run Windows Update.

Hello Van,

We understand that your Surface Type Cover of your Surface Pro 3 stops working after every sleep or shut down. Since you've tried most of the troubleshooting steps to get it working again, try to download and run the Surface Diagnostic Toolkit to update both Windows and Surface driver and firmware.

After running the toolkit, force your Surface to shut down and restart to ensure that all updates are applied. To do this, press and hold the Power button for a full 10 seconds, until the screen turns off, then release the Power button. Press and release the power button to turn your Surface back on. You should see the Surface logo. If that doesn't work, use the two-button shutdown process.

If the Surface Diagnostic Toolkit didn’t help, it may be best that you visit a Microsoft Store near you to have it checked by one of our Store personnel. If you want to talk to someone about this, please contact us here.
