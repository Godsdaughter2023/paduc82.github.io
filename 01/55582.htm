starvinmarvin say:

Thanks for your suggestions. I should have mentioned that i did a clean boot, yet it didn't resolve the problem. My preferred method was using the Tools/Options of CCleaner to disable all unnecessary items at startup including anti-virus, etc.

Anyway, i'm happy to report the problem is solved by uninstalling and reinstalling all the programs involved.

The procedure was to go to Start/Control panel/Programs and features.

On the left-hand menu select Turn Windows features on or off.

Select Media features and un-check the boxes next to Media Center and Media Player.

Click OK.

Restart the computer.

Run CCleaner's regular cleaner, then run CCleaner's Registry cleaner following the usual steps to save a copy of the Registry in case you want to undo changes later on (that's never happened in 10 years of using CCleaner).

Uninstall HD Homerun Prime software.

Run CCleaner again (same as above).

Reinstall HD Homerun Prime software.

Reinstall Media Center and Media Player (under Stat/Control panel/Programs and features/Turn Windows features on or off).

Run Media Center setup.

Everything now works as it should.

Total time required: about 12 minutes.
