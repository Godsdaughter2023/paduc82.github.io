considerate_guy say:

You don't need Windows Pro, but you would need the Media Center upgrade.  Unlike prior versions of WIndows Vista and Windows 7, MPEG-2 encoding/decoding is not included.  Most often, computer manufacturers will include software for playing DVDs and the like.


Since DVD decoding requires a licensed codec, it is an additional cost to include such decoding in each and every copy of Windows.  It appears that Microsoft has decided to reduce costs by not including that as standard.


I would be careful downloading codec packs.  They are often fraught with peril in the form of malware, viruses, etc.  Either get a DVD codec from a reputable source, buy software for playing/burning DVDs from a reputable 3rd party, or forego the use of .mpg video.


Another thing you could try, is to convert the .mpg video to another supported format.  For example, I've used Freemake Video Converter to turn .mpg video into .wmv.  This is especially helpful if you plan to edit the video in something like Movie Maker.  If you do install a freeware application like Freemake, be sure to choose the "Custom Install" option so you can uncheck any add-ons you don't want.

a. user say:

Hi,

I  suggest you to create guest account. Guest account is designed for users who require temporary access to a computer and don’t need to store their user-specific profile settings permanently. Users who log on as a guest have a very limited set of permissions. For example, they cannot access other users’ files or perform system wide tasks such as installing software or hardware.

For security reasons, the built-in Guest account is disabled by default. This prevents users from having an option to log on to the system as Guest.

Files on the root of C: are available to the guest account in order to allow it to run properly, but files in the "Users" folder are not unless someone gave the guest user folder permissions.

To enable the guest account I would suggest you to try the following steps.
a. Type This PC in the search bar.
b. Right-click This PC , and then click Manage.
c. Click Local Users and Groups.
d. Click Users.
f. Right-click the Guest account, choose Properties.
g. On the General tab, select Account is Enabled.
h. Click Ok.

Hope this information is helpful and do let us know if you need further assistance. We will be glad to assist.
