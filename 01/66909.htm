Internet explorer missing from Windows 10

If you've recently made some hardware changes to the computer or had reinstalled Windows 10 on the computer and not being able to find Internet explorer on your computer, then this article might help you!

1. Search for Internet explorer through cortana or search bar on the desktop

2. Try to run the command iexplore.exe and check if you are able to open Internet explorer

3. If the above steps doesnt find Internet explorer, go to Control Panel > Programs and features > Turn windows features on or off and look for Internet explorer 11 on the list and enable it

4. If you are not able to see Internet explorer option by following the above point, make sure you have all the latest updates are installed on the system and then go to Settings > App > Manage options features > Add and enable Internet explorer from the list and reboot the computer once done.

Once the system restarts, you will be able to find and open Internet explorer on the computer.

technutcanada say:

I think this is related: KB4493509 causes 6 second IP drop at DHCP lease expiry

After KB4493509 is installed, the computer is failing to renew / extend the existing IP address lease.

Instead it resorts to requesting a new lease, which means you could end up with a new IP address each time the current DHCP lease expires (like in your case).

In my case the DHCP lease is good for 24 hours (a value that is set by my router, not the computer) and I'm using a router-assigned static IP address, so I get the same IP address every time and the only symptom I see is a short dropout every 24 hours.

But in your case the DHCP lease from the ISP's router is only good for 1 hour and then you are getting assigned a new IP address. That is going to cause more significant connection problems.

I have blocked KB4493509. I hope Microsoft is paying attention to this issue and fixes it soon.
