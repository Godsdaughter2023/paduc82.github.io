Saving Desktop Shortcuts

a. user say:

Another thing to consider, is that if you are doing a clean install, you will have to re-intall your programs so you may need new shortcuts anyway because the new installations may install in different paths than the old installations did in Vista (in effect, the shortcuts may not work and that also applies to the system shortcuts which you shouldn't include but if you do, make sure you know which are the new and which are the old when you delete the duplicates because I don't think the old ones will work correctly). 

In any event, if you still want to do it, you can create a folder in Documents called "Old Desktop" then go to your desktop and click on the icon in the upper left corner holding down the Shift Key and still holding down the shift key click on the icon in the lower right hand corner and that should select all the icons at once.  Then right click and click on Copy.  Then go to the folder you made in Documents and click on an open space and right click and click paste and all the shortcuts and files and folders and whatever was there will be moved to that new folder.  Then if/when you need to backup your system (probably a good idea even if just doing an upgrade just to be on the safe side), backing up Documents will also backup this folder.

Also, what Ken said is correct - if you do it as an upgrade the data, programs, and settings will be preserved and this won't be necessary as your current desktop will still be there after the upgrade (though some icons may not work because of compatibility or other issues - but for the most part it will work fine and you will be able to delete that folder in Documents if you bothered).  Keep in mind we're talking about an Official Upgrade (the mere fact that you are going from Vista to Windows7 doesn't always mean it is an official upgrade - there are some situations where a clean install is required because of the versions involved).  But you should know soon enough as it will either present you with an active upgrade button or it will have no button or the button will be greyed out and inactive meaning it can't be done that way.

I hope this helps.

Good luck!

It simply selects a one time boot device... You do want to select your Flash Drive as a "UEFI USB Device" though... Not just a "USB Device"...

Your Boot Device to boot the machine from the SSD will be "WindowsBootManager"...

You will not even select a specific drive... The boot files will be located on a small 100MB "EFI" partition on the SSD...

That can happen sometimes... Especially if created on a different machine..

You might try downloading a ISO instead of creating a flash drive with the Create Media Tool, and then burn the ISO to the flash drive with a free tool called RUFUS...

It must be specifically created for "GPT" and UEFI Bios, as well as being FAT32 file format..

If any of these are not correct, it can be a problem..

Here is what your RUFUS screen should look like... The USB drive and Image name will be different, but the settings are what you need...

BTW... Be sure to delete everything on the SSD and select the unallocated space as Windows 10 will need to create 4 partitions... You could create a data partition on the end of the SSD, but leave the OS partition unallocated...

If your flash drive is detected properly as a UEFI Installation, your SSD should be properly "Initialized" as a "GPT" type drive, and the needed partitions created automatically...

You should not need to mess with any of your Bios settings if your machine is already set up for UEFI Bios...

I would suggest you to check the Power Management settings for the mouse device.

a)       Click Start, type device in the Start Search box, and then click Device Manager in the Programs list.

c)       In Device Manager, expand Mouse.

f)        Click to uncheck the Allow the computer to turn off this device to save power check box, and then click OK. If the option is checked, then uncheck the option and apply the settings.
