Here's a few others places that can be storing the iTunes startup option.  These are the places that the Task Manager startup tab is checking, but it can't hurt to check yourself too.

  1. Start menu under the "Startup" folder
    Although the Windows 8 Start screen looks different, it's really the same as it's been since Windows 95 (just larger and full screen now).  So if you look at your Startup folder, maybe iTunes is listed.  To browse it as if it was a file folder, press +R and run the command shell:startup to pop it open.


  2. The "All Users" Startup folder
    Another legacy dating back to Windows 95, the "all users" startup folder is the shared folder. Icons placed here will run for any user that logs in. Use +R again and run this path to see if that folder has the iTunes icon:

    C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp


  3. The Registry
    Under either of these locations might be an iTunes entry.

    • HKEY_Current_User\Software\Microsoft\Windows\CurrentVersion\Run
    or
    • HKEY_Local_Machine\Software\Microsoft\Windows\CurrentVersion\Run
