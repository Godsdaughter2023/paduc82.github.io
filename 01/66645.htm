    If, in Character Map, you select a standard character font like "Ariel" and click on the [ ü ], the window in the bottom tells you that it is Character Code 0x00FC.  If you now switch the font to Wingdings and click on the checkmark, you will see that the character code for the checkmark in the Wingdings font is also 0xFC.  Therefore, both symbols have the same character code and the only thing that differentiates them is the Font.

    In a "plain text" window, I surmise that all characters must be shown in the same font.  Therefore when you copy a character from the Wingdings font and paste it in a window that has been set up with, for example, an Ariel font... then that character code will take on a the appearance of the character in the Ariel font with that same character code.
