a. user say:

The first link is regarding backing up the registry and was written for Windows 7


For Windows 8


Press Windows key + Q

In the search Bar type regedit and press Enter to open Registry Editor

In the Menu click on File > Select Export  This will back up the entire registry

The computer will now ask you to browse the path where you want to save the backup registry file. Select path, give a file name and click on Save button

http://mywindows8.org/backup-registry-in-windows-8/


If you navigate to the registry key


HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\ImmersiveShell


Click on the key to select it and again you can select File > Export to backup the key

This is the original key in the registry.


Next, you right click on the above key and select New Key

You'll see the menu open for a new key (looks like a folder) to be added

Type in EdgeUI

Click on EdgeUI to select

Right click on EdgeUI and select New DWord (32-bit) Value

Type DisableCharmsHint

Right click on DisableCharmsHint and select Modify

Type 1 for the value

Click Ok

Close Regedit


Usually if you edit the registry you will need to reboot your computer for the changes to take effect.  According to the document it is not necessary.  If you find the change isn't working, reboot



Those members with Windows 8.1 Preview will only have to add DisableCharmsHint and give it a value of 1.

sarah bee 69 say: Your computer is only giving you the option to allow these programs, with unknown to publishers, to make changes to your computer because the changes they can make to your computer could affect some of the files on your computer, and windows 8.1 is more secure and more foolproof than windows 8.  You get the option to stop and think before you accidentally allow programs to run that may affect other files on your computer than just the ones you're working on at the time.  

There are things you can do!!  You can got into your add/remove programs and CHANGE rather than UNINSTALL windows 8.1 so that you uninstall the upgrade to 8.1 and you're back to windows 8, or you can allow exceptions in whichever security or firewall you use so that when you use these programs you will not see this popup again - hopefully!

As for  "CreateProcess failed with Error 740", this occurs when Windows OS is unable to load the proper permissions necessary for a computer to run properly.  This means the user - you - doesn't have the rights to perform the operation during which this error message is displayed.  This can mean that you are not using your computer as the 'Administrator', or that the 'User Account Control' may be corrupt or damaged.  First thing you should try is logging onto your computer as 'Administrator', if you still get the error message you can also try turning off 'UAC',  open your Control Panel and in the 'search' box type "uac".  There you should see a link to turn on or off your User Account Control,  you want turn it off then click 'next' the next screen you should see a screen which gives you the option to use the 'UAC' (User Account Control) and you should un-check this box and click 'OK'.   

There is a chance you could have a virus as there are now viruses that attack or infiltrate the user accounts.  If you are using an anti-virus you may need to update it and run a complete system scan, as the latest security threats are viruses capable of damaging your UAC, disabling the UAC and damaging Registry Entries that are related to the UAC, and hijacking your system.  For this you may need more than a simple anti-virus alone, you may need an anti-virus with registry tools to clean your registry or you can purchase a registry cleaner online at quite reasonable prices which will clean this up and help with the error code message that you keep getting!!

I hope this was not too long-winded for you and I hope this will have helped with your problems.
