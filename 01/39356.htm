considerate_guy say:

The .wlmp file is a project file.  The project file remembers the current state of editing your project.  But a project file is not a playable movie file.

First, if you need to reopen your project file so you can do more editing, the easiest way is to open up Movie Maker than go to the File menu (Alt+F) and look to the right under Recent projects.  You can find the last few projects listed there.  And if you hover your mouse over an item in the list, you'll be shown the full path where that project (.wlmp) is saved.

On the other hand, if you want to have playable movies, you need to save the movies separately.  This is done from the File menu, choose Save movie.  The result will be a .mp4 or .wmv file that can be played in Windows Media Player or other media player.  It can also be manually uploaded to YouTube, Vimeo, etc.  When you are saving your movie, pay attention to the folder you are placing it in so that you can find it later.
