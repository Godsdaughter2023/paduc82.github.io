Hello KatzAtNine, 

Some users have reported the same behavior where the Wi-Fi key toggles airplane mode as opposed to the Wi-Fi radio.

We are still investigating the cause of the issue as it seems it only affects certain third party vendors. It's not yet clear if there is a compatibility problem between third party special key function drivers or if it is possible BIOS settings are being misconfigured on certain devices during the upgrade.

Since this is a third party vendor specific issue could you please provide us with the make and model of your device, so that we can follow up with the vendor.

In the mean time here are a few ways to work around the issue that you can try:

Method 1: Enable Wi-Fi in BIOS settings

  1. Completely shut down your computer.
  2. Power on the system. As soon as the first logo screen appears get into the BIOS

                Getting into the BIOS varies depending on the model, usually it's done by pressing and holding the Del (Delete), F2, F10 or some other key as soon as the booting process is near completion (if you see the Starting Windows screen, you've waited too long).

      3. Once into the BIOS, look for the network settings page and click on that, or scroll through all the pages till you get to the networking page. Make sure Wireless Radio is enabled. If there is any other settings related to Wi-Fi or Airplane mode make sure they seem correct.

Method 2: Reset default BIOS settings and perform a hard reset


1. Remove the battery and unplug your power supply

2. Press and hold power on button for 30 seconds

3. Leave battery out but plug power supply back in

4. Power on the system. As soon as the first logo screen appears get into the BIOS

Getting into the BIOS varies depending on the model, usually it's done by pressing and holding the Del (Delete), F2, F10 or some other key as soon as the booting process is near completion (if you see the Starting Windows screen, you've waited too long

 5. Load Setup Defaults (press F9) and press Enter.

 6. Save the changes (press F10) and restart the system.

Method 3: Reinstall Hotkey Keyboard driver from the manufacturer website.

1. Go to the manufacturer support website and look for the appropriate driver that handles special keyboard functions. The driver might be name in several ways (Wi-Fi Key/Wi-Fi Switch/HotKey/Special Fn/Quick launch/Keyboard etc.)

2. Download and install the driver for your particular device model.

3. Reboot.

If any of these methods solves the problem for you let us know and please provide us with the make and model of your device, also mark the post as answer so other users can quickly find it and benefit.
