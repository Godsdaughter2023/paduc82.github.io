Since the issue does not persist on a new user account it could be possible that the User account is corrupt.

To resolve the issue with corrupt user account we will need to create a new user account and transfer data from old account to new account and delete the old account.

Refer the link below on creating a new user account.

http://windows.microsoft.com/en-us/windows/create-user-account#create-user-account=windows-8

Once you create a new account I would suggest you to make that account as an Administrator account. Please follow the steps below to do so:

  1. Press Windows + X
  2. Select Control Panel
  3. Go to User account
  4. Under then clicking Manage another account.
  5. Click the account you want to change, and then click Change the account type.
  6. Select the account type as Administrator, and then click Change Account Type.

Once you are done changing account type follow the steps below to transfer user data to new account from old account.

  1. Log on as a user other than the new user you just created or the user that you want to copy files from.

  2. Open the My Documents folder  by Pressing Windows logo + Q and typing computer while highlighting settings Double-click the hard disk drive that Windows is installed on (it's usually your C: drive), double-click Users, double-click the folder with the name of your account, and then double-click My Documents.

  3. Click the Tools menu, and then click Folder Options. If you don't see the Tools menu, press Alt.

  4. Click the View tab, and then click Show hidden files, folders, and drives.

  5. Clear the Hide protected operating system files check box, click yes to confirm, and then click OK.

  6. Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed on, and Old_Username is the name of the profile you want to copy files from.

  7. Select all of the files and folders in this folder, except the following files:                       Ntuser.dat                                                                                                                            Ntuser.dat.log                                                          Ntuser.ini                                                                               

  8. Click the Edit menu, and then click Copy. If you don't see the Edit menu, press Alt.

  9. Locate the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New_Username is the name of the new user profile you created.

  10. Click the Edit menu, and then click Paste. If you don't see the Edit menu, press Alt.

  11. Log off, and then log back on as the new user.

Blu ray discs labeling - Microsoft Community

sabre444 say:

Blu ray discs labeling

s.sengupta say:

blu ray printable disc

considerate_guy say:

I have seen labels advertised for use on Blu-ray media, but there is some debate on whether this is a good idea.  Some say the added weight of a label can unbalance the drive or could come off the disc and attach to the innards of your player.  Others say the adhesive can breakdown the polycarbonate layer of the disc and cause data errors.

 

Again, I don't know if this is true or just anecdotal, but if you are concerned then you should avoid sticky labels.  A couple other options are writing on the disc with a pen designed for that task.  Again, make sure it is one safe for writing on an optical disc.  Certainly don't use a standard permanent marker.  Of course, this doesn't get you a very pretty label, but it does get the disc labeled.

 

Another option is to use printable media that is white on the top.  You'll need to make sure your ink jet printer comes with an accessory tray that will hold/feed a disc into the printer and use the software that is provided for doing this.

 

The last choice I can think of is to just label the case and not the disc.  But then you have to be very careful to keep the two together.

...(QUESTION: IF IT WAS NEVER MEANT TO STREAM OUTSIDE OF YOUR HOME NETWORK, WHY WOULD YOU BE PROMPTED TO PUT IN THE IP ADDRESS.) ...

Due to latency issues it was not meant to be streamed through the Internet, as "normal" consumer internet connections have a smaller upstream bandwidth, which usually results in poor video quality. 

True, some people do have fibre internet with gigabit upload speeds, but the majority of people are stuck with regular cable or DSL connections that have upload speeds of a few Mbit/s. 

And the "feature" to put in an IP address was there, because some routers have wonky discovery methods for their LAN, and instead of telling people "Oh, I just can't find your console. Too bad. No streaming for you in your own local network", you can tell the App: "I know my home network better than you, my console can be reached at 192.168.XXX. YYY" and you could stream away. 

From the very beginning it was always advertised to be able to stream locally only.
