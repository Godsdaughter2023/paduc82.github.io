Did you Clean Install Windows 10 Upgrade with the correct Pro version?

Double check at Settings>System>About if it's Pro version and that you have Version 1511.  If so you should be able to insert the WIndows 7 Product Key at the link there as long as the correct version was installed. 

If no Version 1511 then expedite Updates until it installs. 

If key still fails then the main reason is incorrect characters being inputted.  What is the exact error message when you triple check each character and apply it? 

If you tried inserting the Windows 7 Product Key during install this may also have messed it up because this is only done afterwards per the Version 1511 FAQ at the end of Activation in Windows 10.
