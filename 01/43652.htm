Hi Stephen, no need to worry, you can reinstall Windows 10 at any time, you will not need a product key, and it will not cost you anything !

6. Save BIOS changes, restart your System and it should boot from the Installation Media

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.

Remove files and clean the drive - if you plan to giveaway or resell your computer, this option ensures the computer is cleaned properly. Its takes a bit longer, but it is recommended if you plan to no longer use this computer.

HP Photosmart 7150 Driver - Microsoft Community

anthony bernice say:

HP Photosmart 7150 Driver

I need the print driver for a HP Photosmart 7150. No longer supported by HP.
lemp say:

On the HP support page for the Photosmart 7150 (http://support.hp.com/us-en/drivers/selfservice/hp-photosmart-7150-printer-series/72880), HP says that a driver for this printer is available from "the Windows built-in solution."

Although the "Learn more" link following that message is broken, what HP means is that you should connect your printer to your computer and when the computer detects it, you should get a pop-up asking for permission to connect to Windows Update.  You should allow this and make sure that you have a working Internet connection.

If you don't get the pop-up message, there are two more methods to use.

Method 1

  1. Disconnect the printer
  2. Click the Start Orb and then click Devices and Printers
  3. Click "Add a printer"
  4. Click "Add a local printer"
  5. Select "Using an existing port" and leave the default "LPT1: (Printer Port)" and click Next
  6. Select "HP" in the left section and scroll through the entries on the right to confirm that you do not have "Photosmart 7150" listed (the "Photosmart C7100 Series" and "Photosmart D7100" Series are not the same as your printer).
  7. Click the "Windows Update" button
  8. Wait (this may take quite some time)
  9. When the update is complete, look through the list again.
  10. Click "Cancel" to close the Wizard

If in step 9 you find that the "Photosmart 7150" is now included in the list, connect your printer to your computer, turn on the printer, and it should install.

If in step 9 you find that there is still no entry for the "Photosmart 7150," you'll have to do things the hard way, which is Method 2.

Method 2

  1. Using Internet Explorer (only), go to http://catalog.update.microsoft.com/v7/site/Search.aspx?q=photosmart%207150
  2. You will see that there are 51 entries for this printer.  I cannot tell you what the differences may be between different downloads.  Here's what I would do if I were you, but this is only a guess:
    • select one of the items dated 3/21/2007
    • click on the item name to see some detail.  If the "Architecture" line says X86, the item is for a 32-bit version of Windows 7; if the "Architecture" line says AMD64, the item is for a 64-bit version of Windows 7
    • Make a note of the item that is intended for your version of Windows 7; pick the one that says "(DOT4USB)"
    • Click the "Add" button for the item you selected
    • Click "View basket" (just below the search box at the upper right)
    • Click "Download" -- a pop-up window will open
    • Click the "Browse" button in the pop-up and respond to the UAC prompt as necessary
    • Select the location to save the item (I suggest your Desktop) and click OK; click Continue
    • After the file has downloaded, close all of the windows
  3. The item you downloaded will be a folder named something like "Hewlett-Packard  - Printing - HP Photosmart 7150 Series (DOT4USB)" -- this folder contains a *.cab file, which is a compressed file that needs to be expanded before you can use it.  There are several methods that you can use to do this, but I suggest that you download and install free 7-zip (http://7-zip.org/)
    • Create a new, empty folder in the same location where you saved the downloaded folder (e.g., your Desktop); call this something such as "HP 7150 Driver"
    • Start 7-zip and click File > Open
    • Browse to the folder you downloaded and then double-click on that folder so that the cab file (which probably has a very long name indicating all of the languages included) is showing in the 7-zip window
    • Click "Extract" and then click the "..." button to the right of the "Extract to" box; select the new empty folder you created and click OK
    • click OK and when the extraction is complete, close 7-zip
  4. The contents of the new folder should look something like the following (you may not see the filename extensions (cab, cat, inf) unless you have configured Windows to show these):


  5. Click the Start Orb and then click Devices and Printers
  6. Select any item in the "Printers and Faxes" section
  7. Click "Print server properties" at the top of the Devices and Printers window
  8. Click the "Drivers" tab
  9. Click "Add" to open the "Add Printer Driver Wizard"
  10. Select the proper processor (32-bit or 64-bit)  Do not select "Itanium"
  11. In the "Printer Driver Selection" window, click "Have disk"
  12. Click "Browse" and go to the folder where you had extracted the contents of the downloaded cab file (e.g., "HP 7150 Driver")
  13. Select the file.  If more than one file is shown, select "hpp7150s.inf" (or hpp7150s if file extensions are not shown).  Click Open and then click OK
  14. You should now be at the following window:


  15. Click "Next" and then "Finish"
  16. You should now be back at the list of printer drivers in the "Print server properties" dialog, and the list should now include a driver for your HP 7150.
  17. Connect your printer and turn it on.
  18. If all goes well, your printer should install now.
