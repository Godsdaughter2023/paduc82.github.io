dlchief58 say:

There is no defrag utility on the 360 and generally it is not needed. You can clear the System Cache which clears up most issues (which does perform some hard drive maintenance). If you feel your hard drive has become fragmented to such a degree that it is affecting performance and clearing the system cache does not help, then you can format your hard drive and reload your games, apps, and DLC. Try re-registering Store app. Perform these steps:

1. Search for Windows PowerShell using Cortana or Windows Search.

The Error Status 0xc0000020 error may be caused by Windows system files damage.

There can be many events which may have resulted in the system files errors.

  • An incomplete installation or an incomplete uninstall.
  • Improper deletion of applications or hardware.
  • Computer has recovered from a virus or adware/spyware attack.
  • An improper shutdown of the computer.

All the above actives may result in the deletion or corruption of the entries in the windows system files.

Now, please follow the below suggested methods and check if it helps to resolve the issue.

Method 1:

I suggest you to deploy the System file checker. System File checker (SFC) scan is done to check if there are any corrupted system files that could be causing this issue. Kindly follow the below given steps: 

  • Press Windows key + X, select Command prompt (Admin) to bring up elevated Command prompt.

  • In Command prompt type sfc/scannow and press enter.

  • Restart the computer.

Method 2:

If no corrupted system files were found in the previous state, then I suggest you to try the following steps to employ the DISM command lines: 

  • Open command prompt by pressing Windows key + X and selecting “Command prompt admin”.

  • In command prompt, type the following commands and press Enter after each command:

                DISM.exe /Online /Cleanup-image /Scanhealth

    DISM.exe /Online /Cleanup-image /Cleanupimage

                DISM.exe /Online /Cleanup-image /Restorehealth

  • Close the command prompt and restart the PC and check if it works.
