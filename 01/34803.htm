Thanks for any help you can offer

David

jomel_g say:

Hi David,

Question 1: How do I minimize to try when I close using the X?

- Hitting the X button will close the whole application. As a suggestion, please use the minimize button ( _ ) from the upper right side of the app to make the app visible in the taskbar.

Question 2: How do I clear all history?

- Unlike classic Skype, Skype for Windows 10 does not have the function to delete your chat history at once. You will need to delete your conversations one at a time.

  1. From the Chats list, right-click on a conversation.
  2. Select Delete conversation.

lemp say:
The System Volume Information folder is a "super hidden" folder to which you normally do not have access. Among other things, it contains System Restore information. If I recall correctly, it also contains file indexing information. There is a System Volume Information on every "volume" (partition).

You shouldn't delete it.

Even if you have enabled viewing of "Protected operating system files," you will not have permission to view or open the System Volume Information folder. That's why it shows as having "nothing in it" when actually it often occupies a fair amount of space.
