How to view information about a device driver
Open Device Manager
Select the device category 
right-mouse click the specific device you want, and then click Properties
On either the Driver tab, or General tab, information about the installed driver is displayed.

Both Host Controller and Root Hub are part of the topology in the USB system architecture. 
- USB Root Hub monitors each port and manages power.
- eXtensible host controller  arbitrates the USB bus communication. 
The eXtensible host controller is capable of interfacing with USB 3.x, 2.0 and 1.x and so shouldn't place restrictions on your peripherals.
