WMPInfo.XML is an ‘Extensible Markup Language file’. norrad say:

I am having the same problem trying to create a system image on DVD+R with my new Sony Vaio VPCF1190X. Just keeps asking to label & insert media >1G.

I find this is a common problem in Windows 7, when writing the "system image" to an external DVD:

  1. Creating a small Restore Disk usually works.

  2. Creating the multi-DVD "System Image" fails by:

        a. Asking if you want to format the DVD

        b. When you say "Yes", it formats it, then attempts to write.  After either a short or long period of time, you are asked to insert another DVD into your external drive, and label it with the SAME label, and always #1.

I wrote successfully to an external Memorex 24X DVD drive, on the USB port, but only after getting DVD+RW (4x) media:

  a. DVD-R 16x (Sony) failed in the above manner.

  b. DVD+R RW 16x (Memorex) failed also.

  c. DVD+RW 4x worked flawlessly.

From the experience of others (various systems and various external DVD drives), the only common factor I see is that Windows 7 can ONLY write the system image if the external device writes DVD+RW and does it with a slow medium (4X in my case.)  I confirmed this problem with a Tech at Staples, who said to try various media until one works!

I Hope this is not too late to help someone else (and to alert Microsoft that they really have a problem.
