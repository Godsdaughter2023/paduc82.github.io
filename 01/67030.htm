All of your folders and messages are in the Windows Live Mail store folder. Its location is shown in the program: when viewing your message list, press Ctrl-Shift-O for Options. On the Advanced tab, click Maintenance and then Store folder. The store folder is hidden by default to discourage users from 'messing' with it, which could cause the program to malfunction. Perhaps a better way of backing up your folders and messages is to use the Export email > Email messages function on the File menu. It's not a one-click procedure as you'll discover, but it's quite manageable as a, say, weekly exercise. You'll find some tips for a successful export in this article: Export and import messages.

Even easier is to use Windows 10's File History process, but you can't use DVD for this. After setting up the drive to be used, you can ensure that the store folder is included in File History's backup set by creating a library for it: right-click on the folder name in File Explorer and select Include in library > Create new library. Library contents are backed up by default in File History.
