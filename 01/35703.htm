This is a long-standing problem with older Windows computers that run Broadcom Bluetooth driver and control software.

Broadcom supplied various versions of Bluetooth hardware for a number of PC brands in the 2000-2010 period. 

BTTray is the Broadcom control app that, when properly installed, appears in the bottom task-tray (I think).

My problem is with a Dell Inspiraon N7010 that had its Bluetooth driver erased when I installed Win 10.  The only Broadcom BT drivers and BTTray apps that Dell offers are for Window 7.  Getting them to install under Win 10 is a tough under-taking which involves using the Compatibility settings to fool the drivers software into believing it is running under Win 7.

But once BTTray (WIDCOMM) is successfully installed it begins to complain after booting that it cannot run because it cannot locate the Broadcom BTBalloon.dll.  For years now people have asked Microsoft support how they can replace BTBalloon.dll.

Here is what I believe that I have discovered:

Each Broadcom installation package comes with a file BBalloon.dll.  It comes in 32 and 64-bit versions.  During a successfull installation of the Broadcom package its Setup.exe installs BBalloon.dll somewere and renames it BTBalloon.dll.

I was able to stop the boot error by manually renaming the file and by adding the new Broadcom installation folder path to the system environment path variable.

My only problem now is successfully re-installing the BT device drivers.  I accomplished this once before before I performed a clean installation of Win 10.  But now the device manager will not recognize any of the BT driver files and the driver installation software will not function under Win 10.
