j w stuart say:

If you install Windows Vista on one drive (call it drive #1) and Windows 7 on another drive (Drive 2) with drive #1 removed then you will not have a Dual Boot system. You will need to pick which drive / OS you want to load (select a specific drive) from the BIOS menu when you first power on the computer. The same is true with GPT.

If your drive is running out of space because of two versions of Windows installed then there is a means of moving Windows 7 to another drive and not be forced to perform a fresh install.
