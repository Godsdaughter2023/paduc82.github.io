File History doesn't automatically delete versions of files older than the time interval you choose unless your drive is getting full and space is needed to back up more recent versions.

Hiperfil.sys is created by Windows 7 when hibernate is enabled.

Type Cmd into the start search box.

Right click on  Cmd.exe  at the top of the list and select Run as administrator.

Type Exit

Restart your PC

If you want to enable hibernate again then do the following:

Type Cmd into the start search box.

Right click on Cmd.exe and select Run as administrator.

Type powercfg.exe -h on

ENTER

Type Exit

Restart your PC
