The image preview you are seeing in Skype chat window is just a link (with graphical representation) to the image file saved in Skype internal image folder. The path to this folder is as follows:

 

C:\Users\<winuser>\AppData\Roaming\Skype\<skypeuser>\media_messaging\media_cache_v3\

 

If you open this folder, you should see there all the sent and received image files with somewhat cryptic names but with the extensions .jpg or .png.  Other media files, such as sent/received video messages (.mp4 extension) are saved in this folder too.

 

Clicking on the image preview in the chat window should just open the image file from this folder using the associated image view program.

 

You can try to see if the images open properly if you manually navigate to this \media_cache_v3 folder and click on one of the saved image files.

Change Windows fonts

You can change the font in any part of Windows (for example, menus). Changing a Windows font does not change your program fonts.

  1. Right Click on an open space on the desktop
  2. Select Personalize at the bottom
  3. Select Window Color and Appearance
  4. Open Appearance Settings.
  5. Click Advanced.
  6. In the Item list, click the part of Windows where you want to change the font. For example, if you want to change the menu font, click Menu in the list.

  8. In the Size list, click the font size that you want.
  9. In the Color list, click the font color you want.
  10. Repeat steps 3 through 6 for each part of Windows where you want to change the font, its size, and its color, and then click OK.
