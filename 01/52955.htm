considerate_guy say:

As noted by skier-hughes, a transition is just that, a transition from one clip into another.  During the time of the transition, clip A is transitioning out as clip B is transitioning in.  That transition time has to come from both clips so make sure that the clips on either side are long enough to accommodate the desired transition length.

considerate_guy say:

To change a transition duration, select the clip with the transition (the second one) and then go to the Animations tab.   Next to the list of animations (on the right) you should see a box showing the current duration (generally 1.5 seconds). Use the drop-down to pick a new value.

There are several things that are important:

  1. The clips on either side must be at least as long as the transition you are asking for.  For example, if you have two pictures that are 7 seconds long each, you can't have a transition that is 10 seconds long, obviously.  If that is what you want to do, extend the duration of your pictures first.  If this is video, use a longer clip.
  2. If you don't see the transition time you want in the drop-down list, type over it and press Enter.

If you are still having problems, can you explain exactly what steps you are trying and where it doesn't behave as expected?

a. user say:

jetta48 say:

The most common external hard drives are USB ones and you should find that most current product lines will be compatible. However, whenever you buy new hardware or software it is always advisable to check the hardware manufacturer's websites (e.g. Western Digital, Seagate, Buffalo) for specific compatibility information - email the company's tech support if you're not clear.

 

The current external USB hard drives can come in two variants of USB - USB2 (aka Hi-Speed) or USB3 (aka Super Speed). If your computer has USB3 ports then you can benefit from the higher data transfer speeds of USB3. If you try to use a USB3 hard drive connected to a USB2 port it should still work, but at only USB2 speeds.

 

If you are not too worried about portability/physical size then I would recommend getting a drive that has its own power supply as some USB ports can struggle to supply sufficient power.

 

Some external drives come packaged with backup and recovery software, so you may wish to consider that as an option.

afzal taher say:

Hi,

Try these steps and check if it fixes the problem:

Step 1: Delete unnecessary files in the themes folder.

Check the contents of the themes folder.

The folder is located in the location shown below. Access it by typing the following text in the start search field

%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Themes\

Deleting the unnecessary files present in the folder seems to have fixed the desktop background issue for many other users facing the same problem.

Step 2: Disable remove background images option.

a. Open Start and then clickControl Panel.
b. Click on Ease of Access Center.
c. Now click Make the computer easier to see.
d. Ensure that Remove background images (where available) is unchecked.
e. Click Save.
f. Click Apply.
g. Close the window.

Step 3: Create a new user account.

This issue can also occur due to a corrupt profile, so create a new user profile.

Steps to create a new user profile:
a. Open User Accounts by clicking the Start button  , clicking Control Panel, clicking User Accounts and Family Safety, and then clicking User Accounts.
b. Click Manage another account.   If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
c. Click Create a new account.
d. Type the name you want to give the user account, click an account type,(Administrator) and then click Create Account.

Refer this link for help:
http://windows.microsoft.com/en-US/windows7/Create-a-user-account

Thanks in advance

sharief k say:

Hi,

We apologize for the inconvenience caused to you.

Please be assured that I will do my best to help you fix this issue.

The issue may occur if the service is started by the Local System account instead of by the Local Service account (NT AUTHORITY\LocalService). The Local System account may have insufficient permissions to start the service. 

To fix the issue, please follow these steps:

1. Click Start, type Services.msc and then press ENTER,
2. Locate and double click the Bluetooth support service.
3. Click the Log On tab,
4. In the This account text box, type NT AUTHORITY\LocalService.
5. Set both Password fields blank.
6. Click Apply and then OK,
7. Restart the service and check the issue.

If the issue persists, please answer to the following questions to assist you better:

  1. Why are you trying to start the service?

  2. Do you have any issues with the Bluetooth?

  3. Were there any recent changes made on the computer prior to the issue?

Keep us posted on the status of the issue. We will be glad to assist you.
