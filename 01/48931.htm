arminatorx say:

First of all, cdkeys.com is NOT a reputable seller. They are a marketplace provider similar to eBay or Amazon Marketplace, where complete strangers can offer their codes to sell to others. 

So it might be, that you bought your code from a scammer, or maybe the seller checked if you redeemed the code, and since you took some time to redeem it, sold it again. 

Unfortunately, Microsoft can't help in any way, besides the chat record you have that you did not redeem the code for you. 

With that you need to contact the cdkeys support, and have them deal with the issue. 

The next time you buy a code, make sure to redeem it right away. Codes you redeem "stack", so you don't lose any subscription time. 

For example, Live Gold memberships can be stacked up to 2 years at least. So if you have 6 months left on your Gold membership, simply redeem your 12 month code, and you will have 18 months of gold. 

NEVER wait to redeem your Gold codes until your gold is running out!

That way you prevent scammers from re-selling the code again. 

Unfortunately, the only one to help you with that issue is the cdkeys support. If they are unable to help you, I'd recommend to NOT make ANY purchases from key sellers again. 

cdkeys, G4A, Kinguin and similar key sellers often appear in this forum with complaints from customers. 

Please buy from really reputable sellers like Amazon (the actual Amazon, not Amazon Marketplace sellers), or local shops that sell you actual code cards where you have to rub off a protective film from the code to make it visible. 

That way you can be sure that you get an unused code that only you can redeem.
