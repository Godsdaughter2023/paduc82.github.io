I suggest that you check the status of the device in the Device Manager after you reboot your computer.

a.     Click Start. In the Start Search box type "devmgmt.msc" and press Enter.

b.     Open Device Manager and look for the device.

c.      Locate the device, right click on the device and check the status.

I also suggest that you check if there are any updated drivers installed for the device from the device manager.

a.     Click on Start and type ‘Windows update’ in the search field.

b.     Open Windows update and click on ‘View update history’.

c.      Check if there are any drivers installed for the device from Windows update.

If a driver is being installed, it may not be working correctly and causing this issue. I suggest that you hide the update and make sure that it does not get installed.
