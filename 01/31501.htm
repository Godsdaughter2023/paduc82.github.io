To be able to use Windows XP Mode and Virtual PC, you have to have Windows 7 Professional or Windows 7 Ultimate. They will not work in Windows 7 Home Premium. You would have to use a 3rd party virtualization program.

Running Windows XP Mode requires:
 
1. Windows 7 Professional, Enterprise, or Ultimate editions.
2. A computer capable of hardware virtualization. This means your computer has a central processing unit (CPU) with either Intel-VT or AMD-V virtualization features. (This option is no longer necessary)
3. Virtualization features turned on in your computer’s basic input/output system (BIOS).

Please look at the following website for information:
http://windows.microsoft.com/en-us/windows7/install-and-use-windows-xp-mode-in-windows-7#section_1

http://www.microsoft.com/windows/virtual-pc/support/configure-bios.aspx
 
To find your brand of processor so you can download the appropriate utility, click the Start button , right-click Computer, and then click Properties. Under System, next to Processor, you can see what type of CPU you have.
 
Even if your CPU can run Windows XP Mode, you might also need to turn on virtualization settings in your computer’s BIOS. Procedures vary depending on the BIOS manufacturer. For more information, see BIOS: frequently asked questions.

http://windows.microsoft.com/en-us/windows7/BIOS-frequently-asked-questions

It's recently been found if you had either AVG or Avast antivirus suites installed on your computer, it causes an issue updating to the new April 2018 (1803) update, so this is what you may be experiencing, though there's a possibility that it couldn't be that.

You can prepare Windows installation media by using the Media Creation Tool, here: https://www.microsoft.com/en-us/software-downlo...

When you're done preparing your USB flash drive, plug it into your Samsung RC520, be sure to go into the boot menu (typically you hit F12 repeatedly when you're starting it up), or a drive selection menu where you can select what device to boot from, in this case it should be your flash drive.
