Media Player Library Issue After Drive Letters Changed - Solution - Microsoft Community

sjmuk01 say:

Media Player Library Issue After Drive Letters Changed - Solution Found

I wanted to share a solution to an issue I found and after Googling I could not find any answer.  However after some perseverance I managed to fix this myself.

I installed Win 7 & got it to read all my media files into WMP12, no problems.  However, I changed the drive letters on the drives where my video, music & picture files were held and then lost all library details.

I thought I could rebuild the media player library by updating the folders included in the relevant libraries, but this did not work.  I tried various other fixes posted (primarily deleting all the WMP library files to get it to rebuild), but with no success.  What I did find worked was:

Close WMP and Media Centre
Remove all folder paths from libraries (Music, Pictures, Video) so no files were included.
Run Regedit
Search for all references to old drive letter paths & modify registry data to show new drive path (EG my music was on E:\My Music, after assigning new drive letter it is on H:\My Music, so I searched for all instances of E:\My Music to replace E: with H:)
Rebooted
Ran WMP
Added the new folder path back into the library
Hey presto, it began to rebuild the music library!

Hope this helps someone else as I spent several frustrating hours trying to figure this one out!

Is Windows 10 or Microsoft Office asking you to activate?

Windows 10 and Office are two different things. If Windows 10 came preinstalled, it should not be asking you to activate.

Windows 10 came preinstalled on my computer, how do I find the product key?

Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

The product key will be revealed, copy the product key then enter it

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate

If you are unable to activate Windows 10 after all the above and you are certain you have the correct edition installed, then please attempt phone activation:

Activate Your Windows 10 License via Microsoft Chat Support
