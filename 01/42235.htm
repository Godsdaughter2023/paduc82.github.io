Hi

What you are seeing with multiple versions of the Microsoft Visual C++ redistributable installed is normal.

 

When a developer builds a program using Visual C++, the version that they use may be the current version or an older version. The version that they use will be installed on the computer, when you install that program. A later version will not work with that program.

Occasionally, these versions may be updated to fix newly discovered problems or vulnerabilities, or to add enhancements.

When Windows Update distributes an update for an older version of C++ and it detects that version is installed on your computer, it will apply that update.

Uninstalling any of these versions will result in some installed programs no longer working properly.
