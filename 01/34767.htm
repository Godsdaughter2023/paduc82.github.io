How To Find SSID (Network Name)

Login Information



  • Gateway: 192.168.0.1
  • Username: admin
  • Password: admin





How To Find SSID (Network Name)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Wireless Settings" from the menu on the left.
  5. Select "Basic Settings" from the sub menu.
  6. The wireless network name will be labeled as "Main SSID".
