FAQ: GamerScore, Microsoft Points &amp; CUL - Microsoft Community

magic mart say:

FAQ: GamerScore, Microsoft Points &amp; CUL

FAQ: GamerScore, Microsoft Points & CUL

In deze thread gaan we het hebben over 3 onderwerpen:
GamerScore (G)
Microsoft Points (MP)
Community User Level (CUL)



GamerScore
GamerScore wordt behaald door bepaalde achievements (prestaties) te behalen in games.

Games op Disc leveren je in totaal G1000 op, mocht je alle achievements behalen.
Arcade Games leveren je in totaal G200 op, mocht je alle achievements behalen.

Nu kunnen uitbreidingspakketten voor games het maximaal te behalen GamerScore (zie bovenstaande) nog eens extra verhogen.
De behaalde GamerScore van alle games die je bezit wordt bij elkaar opgeteld en in/bij je profiel weergeven.

Je kan aan een GamerScore dus zien in hoeverre je tegenstander/medespeler zijn games uitspeelt.
Verder kan je niets met je GamerScore, je kan deze niet omwisselen voor Microsoft Points. (zie verderop in FAQ)
GamerScore is enkel de status van jouw game completion!

Het resetten van de GamerScore gebeurd enkel in gevallen als er op jouw Xbox op illegale wijze aan Achievements is gekomen.
Is dit onverhoopt gebeurd dan kan er altijd contact worden opgenomen met de Microsoft Klantenservice.

 

Microsoft Points
Microsoft Points koop je via Redeem Cards in de winkel of door middel van je Credit Card op je Xbox.
Microsoft Points is het "geld" waarmee je artikelen van Xbox Live Marketplace betaald.

1 Microsoft Point = 0,012(5) Euro
800 Microsoft Points = 10 Euro
5000 Microsoft Points = 60 Euro

Meer punten kopen, levert je dus €2,50 winst op in het geval van het bovenstaande voorbeeld.

Enkel het onderdeel 'Games on Demand' op Xbox Live Marketplace laat voor Credit Card bezitters de keuze om on Euro's of in Microsoft Points af te rekenen. Verder dient alles in Microsoft Points betaald te worden.


Microsoft Points opwaarderen
Om Microsoft Points op te vragen door middel van de Redeem Code druk je na het starten van de Xbox op de:
Guide Button--> 2x naar Links --> Klik op 'Redeem Code' --> Vul de code van 25 tekens in.



Community User Level (CUL)
Wij onderscheiden 2 soorten Community User Levels.

Xbox.com Community User Level
Xbox Live Community User Level


Xbox.com CUL
Zodra je iets post op het forum van xbox.com zie je bij je naam een cijfer staan. Dit is als je hier net bent een 0, dit groeit door naar gelang je actief en ontopic posts plaatst op het forum. Zodra dit niet het geval is kan een moderator besluiten je CUL terug te schroeven, of in het positieve geval een 'boost' te geven. De CUL op het Xbox.com Forum laat dus ook goed zien hoe lang en actief iemand op een forum aanwezig is, alsmede ook de manier waarop hij dit is.
Naast een CUL kunnen er ook icoontjes worden weergeven, check voor een overzicht de Handleiding Xbox.com Forums.


Xbox Live CUL
Als je op je Xbox kijkt zie je bij je Gamertag mits je een Xbox Live Gold Abonnement hebt een cijfer staan.
Dit is het eerste jaar nog niet het geval. Zodra je 1 jaar Xbox Live Gold hebt ontvag je een 1 achter je Gamertag. Per jaar dat je onafgebroken Xbox Live Gold hebt groei je door. Zo heeft iemand met 5 jaar Xbox Live Gold een 5 achter zijn gamertag.
Een verdere functie dan status heeft dit cijfer achter je gamertag niet, het laat enkel zien hoelang je al betalend lid bent van de community.



Mochten er verder nog vragen zijn, dan kunnen deze worden gesteld in deze thread.
