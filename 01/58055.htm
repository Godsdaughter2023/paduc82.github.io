You can try following steps:

Step 1: Stop the disk defragmenter service completely and check if the issue persists

1)  Click windows key + R. Run box,

2) Type services.msc and Click OK.

3) Right-click the Disk Defragmenter service. Click Stop.

Stopping the service will take a moment.

Step 2: Temporarily disable any of the security software running on your computer and check if the disk defragmentation still appears. Some security programs have a defragmentation program which runs on the boot volume and cause this issue.

Note: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you have to temporarily disable it to install other software, you should re-enable it as soon as you're done. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

Step 3:

I would suggest you to perform an SFC scan and then check if it lists any issues -
Perform a SFC Scan

1. Click Start, click All Programs, click Accessories, right-clickCommand Prompt, and then click Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or clickAllow

2. Type the following command, and then press ENTER:

sfc /scannow

A message will appear stating that 'The system scan will begin'. Be patient because the scan may take some time

3. If any files require a replace SFC will replace them. You may be asked to insert your Win7 DVD for this process to continue

4. If everything is okay you should, after the scan, see the following message "Windows resource protection did not find any integrity violations"

5. After the scan has completed, close the command prompt window, restart the computer and check.

slance310 say:

Hi,

The difference is that you upgraded from Win 7, and had a version of Windows Media Player that would play .mpeg files, and Media Center, which also provided additional Codecs to play mpeg files... Some of that capability was retained as part of the upgrade...

Win 10 will not play .mpeg files on it's own... First because Microsoft discontinued the playback of DVD type movie files for licensing reasons, and second, because of the loss of the codecs that were available from Media Center...

MPEG files are very similar to the DVD Video files, so they don't play...

About the only solution for your sister would be to use a 3rd party player, or 3rd party Codec Package...

There are several free players, but the most popular seem to be VLC Media Player, and Media Player Classic_Home Cinema...

As far as a codec package... I use K-Lite Codec Pack_Basic... If you use this, be sure to watch closely as it will ask if you want to install some type of extra app.. If you don't check closely, you may inadvertently install something you don't want...

BTW... WMP 12 is available with your sister's machine... It's just that the Movies + TV is set as the default player... You should be able to change that by the "Default Programs" settings, or by right clicking on the file and selecting "Open With"...

If WMP is not on the list, you can browse to it...
