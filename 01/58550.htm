Network and Internet Speed is measured by the number of basic info that can be transferred in a sec.


Your measure indicates that you have a n Internet connection that transfers from the Internet to your computer (download) info at the rate of about 1565,000  bits per second (1565K/b/sec.)  and it sends info at the rate of 530,000 bits per sec. (530K/b/sec.)

lead3 say:

Download Speed = the speed in kbps (kilobits per second) and KB/sec (Kilobytes (byte = 8 bits) per second)  that data is sent to your computer

Upload Speed = speed  in kbps (kilobits per second) that data is sent from your computer

Latency = time in milliseconds (ms) of a delay in processing network information. Should be less than 100ms.

 

Up load is always a slower by design, since we typically don't send as much information as we receive.

hand nimet say:

Unmountable boot volume what does it mean?

Hi

I have tried holding power key for 5 seconds hopin it will give me a menu of options. This doesn't happen.  I can't get it safe mode. none of the fn keys operate. I have an acer aspire 5742 with windows 10.

Can someone please help. All the advice assumes that a safe mode or a page with help menu shows but I just get stuck in the repair loop.

Many thanks

a. user say:

'Volume' is the term for a unit of disk storage. For example, a partition is a volume, and the two terms are often used interchangeably.

The 'boot volume' is the partition that contains a bootable operating system. For Windows users, the boot volume is typically C:

'Mounting' a volume means making it accessible ... making it possible to find and use the folders and files in the volume.

Therefore, an unmountable boot volume (in a Windows system) is a C: drive that is inaccessible. Since the C: drive is used to boot the computer, if the C: drive is inaccessible your computer can't boot.

The C: drive is inaccessible because it has become corrupted. This can happen for hundreds of different reasons, including problems with upgrades, malware, inappropriate configuration changes and misbehaving software applications.

To fix the problem correctly you have to know what caused it in the first place. These are highly technical things that confound most people, so many people end up re-installing Windows. Before you do that, it could be worthwhile to show your computer to a technician. Pick someone who really knows their stuff, not that Up The Creek Squad or whatever they're calling themselves these days.

Hi Blancho. I'm Greg, an installation specialist and 9 year Windows MVP here to help you.

If it's a school PC it should have a Help desk behind it that you can call on for things like this. Most schools do.

If not I will give you all possible steps to troubleshoot installation failure which work in about 90% of cases. If not we'll go from there:

Follow the illustrated Clean Install steps in this tutorial which compiles the best possible install that will stay that way as long as you stick with the tools and methods given: http://answers.microsoft.com/en-us/windows/wiki...

Make sure all other drives and peripherals are unplugged. Update the BIOS or UEFI firmware to latest, or reset to Defaults, making sure SATA controller is set to AHCI, UEFI is enabled, CSM disabled, Windows Boot Manager set first to boot, if necessary disable Secure and Fast Boot until after install: https://www.mydigitallife.net/comprehensive-lis...
https://www.groovypost.com/howto/reset-pc-bios-...

Create the latest new Windows 10 Installation Media by installing Media Creation Tool and following the directions here: http://windows.microsoft.com/en-us/windows-10/m.... If you're using another PC to do this then uncheck the box for Recommended Settings to choose the exact version and bit rate for the target PC only.

Insert media, boot it by powering up PC while pressing the BIOS Boot Menu Key: https://www.sysnative.com/forums/hardware-tutor...

If the media won't boot you may need to enter BIOS/UEFI Setup (pressing key given in chart in link above) to turn off Fast Boot or Fast Startup first. If you can start Windows enter UEFI firmware here: http://www.isunshare.com/windows-password/four-...

Choose the boot device as a UEFI device if offered, on first screen press Shift + F10 to open a Command Prompt, type the following commands, each followed by Enter key:
DISKPART
LIST DISK
SEL DIS # (replace # with the disk # of the Windows disk from list in previous command)
CLEAN
EXIT

Close Command Box, on second screen choose Install Now, then Custom Install, then at the drive selection screen choose Unallocated Space, click Next to let it create needed partitions and start install - this makes it foolproof.

Make sure to follow the setup steps in the first link tutorial to complete your install. Drivers are important enough that those steps are printed in red.
