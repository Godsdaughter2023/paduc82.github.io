What happens when I run the recovery discs in the computer? - Microsoft Community

minkycat say:

What happens when I run the recovery discs in the computer?

Original Title : .vista

i want to run my vista recovery disc, what will actually happen when i do this.

bill smithers say:

Hi,

When you run the Manufacturer's Recovery Disk, it puts your Computer / Laptop back to Factory Settings; the new condition that you bought it in.

 

All your Data: Pictures, Music, Files, etc should be backed up to an external Hard Drive before doing a Recovery, as they will normally be lost.

 

All Windows Updates, 3rd Party Programs, Data that you have installed since your Computer purchase will no longer be on the Hard Drive as the Hard Drive is normally wiped clean of everything during the Recovery Process.

 

The above is called a destructive Recovery.

Some Computer Manufacturer's also have the option of a non-destructive Recovery, which keeps Programs and Data in place.

But, Data should be backed up, IN CASE.

If your problem is so bad that you have to reinstall, a non-destructive Recovery does not normally fix the problem and a further destructive Recovery has to be done

 

Cheers.
