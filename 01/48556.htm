Hi Sandra,

If it is the serial number of the actual computer then this should be on a sticker on your PC case. The sticker usually states the model number of the PC along with the Serial number. tryx3 say:

Do both these steps [they are the closest thing possible to uninstalling then reinstalling WMP & will effectively provide a new installation of WMP]

1  Unload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, clear Windows Media Player checkbox, Yes, OK, Restart the PC.

2  Reload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, set the Windows Media Player checkbox, Yes, OK, Restart the PC.
