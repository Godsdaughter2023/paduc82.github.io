vanessa yar say:

Hi Stephane,

The error 0xc00d4e85 appears when an application wants to take sole control over the sound device. This usually happens when the Windows Audio services are not running or have not set to start automatically that's why it's getting an error when switching between audio applications.

To work around this problem, please ensure that the Windows Audio and Audiosrv service is set to start and to run automatically. Follow these steps:

  1. In the Cortana search, type in Services, and click it from the search results.
  2. Under Services (Local), scroll down to Windows Audio, right-click it, and click Properties.
  3. Under the General tab, check if the Service status is running. If it's not, click Start to start the service.
  4. Look also for Startup type. If it's not set to Automatic, then click the drop down arrow, and select Automatic.
  5. Click Apply, then OK.

The same error may also appear if Windows has been set to allow devices to take exclusive control over the audio device. To disable this setting, follow the steps below:

  1. Right-click the speaker icon on the system tray, then click Properties.
  2. Right-click on the playback device within the list of devices, and select Properties.
  3. Go to the Advanced tab and uncheck Allow applications to take exclusive control of this device from the Exclusive mode.
  4. Click Apply, then OK.

Once done, check if you can now switch between Dolby and DTS. If you still can't, restart your PC and try switching again.

Let us know how it goes.

Regards.

bruce hagen ~ mvp say:

From a Google search I found:

https://www.rogers.com/customer/support/article/email-setup

Add an account using POP

  1. Click on Accounts.
  2. Click Email.
  3. In the Add your email accounts window, enter in the following:
    • Email address: Rogers Yahoo! email address.
    • Password: the App Password you created.
      • Selecting Remember password is optional, but if it isn’t selected, you will have to enter your password every time you send or receive an email.
    • Click on the Manually configure server settings button and click Next.
  4. In the Internet Email Settings window, enter the following information:
    • Incoming Server information
      • Server Type: POP3
      • Server address: pop.broadband.rogers.com
      • Port: 995
      • Authenticate using: Clear Text
      • Logon user name: full Rogers email address.
    • Outgoing server information
      • Server address: smtp.broadband.rogers.com
      • Port: 465
      • Requires authentication needs to be checked.
    • Requires a secure connection (SSL) should be clicked.
  5. Click Next, and then Finish.
  6. Send a test email to make sure the settings are correct and the email is working.

Modify existing POP settings to use Rogers Yahoo! servers

  1. Click Accounts.
  2. Click Properties.
  3. Click Servers.
  4. In the Servers window, check the following:
    • Server Information
      • My incoming mail server is a: POP3
      • Incoming Mail Server: pop.broadband.rogers.com
      • Outgoing Mail Server: smtp.broadband.rogers.com
    • Incoming Mail Server
      • Email username: enter in the full email address.
      • Password: the App Password you created.
        • Selecting Remember password is optional, but if it’s not selected you will have to enter your password every time you send or receive an email.
    • Log on using clear text authentication needs to be checked.
    • Outgoing Mail Server
      • My server requires authentication needs to be checked.
  5. Click Settings.
  6. Make sure Use same settings as my incoming server is selected, and then click OK.
  7. Click Advanced.
  8. In the Advanced window, check the following:
    • Server Port Numbers
      • Outgoing server (SMTP): 465
      • Incoming server (POP3): 995
      • This server requires an encrypted connection (SSL) box is checked
    • Delivery
      • If Leave a copy of messages on server is selected, the emails will be backed up on webmail that you can access through the Rogers Yahoo! website.
  9. Click Apply, OK, and then Finish.
  10. Send a test email to make sure the settings are correct and the email is working.

What will be my product key of windows 10 after upgrading from windows 8.1 ?
By using a software i see my windows 10 key which is different from my windows 8.1 key. For how much time this key will work?

zigzag3143 (ms -mvp) say:

Windows 10 handles keys differently.  When you upgrade to Windows 10 via Windows Update from Windows 8.1(or win 7 SP-1) registers your computers hardware, and qualifying OS with a Microsoft Product Activation Server. The device is then considered a Windows 10 device.


Subsequently anytime you re-install (or clean install) the installation checks with those servers, finds the unique installation ID and produces the validation

There is no "key" as there was in the past.  Everyone gets a generic key but the unique thing is the activation ID which is stored on the MS servers.

This "key" works for the supported lifetime of the device (~5years)
