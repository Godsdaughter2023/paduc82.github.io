Hi Lisa IDEX, 

Thank you for posting in Microsoft Community.

The "Plugged In, Not Charging' status you see when you tap over the battery icon in the windows taskbar indicates that the AC adapter is plugged in to run your Surface, but the battery isn't charging.

If charging doesn't return to normal, remove and reinstall battery driver.

  • Go to Device Manager > expand the Batteries > Double-tap or double-click Microsoft Surface ACPI-Compliant Control Method Battery > select the Driver tab, and select Uninstall > OK.
  • From the Action menu, select Scan for hardware changes. 
  • Restart your Surface device.

Download and run the Surface Diagnostic Toolkit as it helps fix common Surface problems and makes sure that you have the latest updates installed on your Surface Pro 6.

Performing a Forced Shutdown is like removing the battery from a laptop or unplugging a desktop PC. It resets the hardware and clears all the cache and memory for your hardware.

  • Hold the power button for 10 seconds to force Windows to quit. Press once again start Windows. If that does not work, Press and hold the power button for 20 seconds.
