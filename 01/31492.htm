zacabeb say:

PC resolutions like 1360x768 are a bit of a pain when it comes to sound since there is no official standard on how to embed it there - hence why DVI (from which HDMI is derived) doesn't normally support sound.

The HDMI specs detail how the signal should be formed to make enough room for sound in the data island sent after the video in each frame, including pixel repetition if the data rate needs to be raised. However, this doesn't cover PC resolutions which are limited the same way as in DVI by legacy. The Xbox 360 often tries to embed sound with the same principles in those resolutions (2-channel LPCM or Dolby Digital don't need much space), but not all decoders expect sound in PC resolutions or are able to detect it properly.

I'd blame developer laziness and Microsoft. It's common for flat panel HDTVs to crop about 2.5 - 3.5% off 720p signals unless they have a mode for disabling overscan. Broadcasters even want it this way because the extreme edges of the picture may contain garbage. Game developers used to be forced to take the overscan into account, with HUD elements and text being at least 7.5% in from each edge by default.

Unfortunately I can't think of any solution, unless the game has an internal scaling feature to compensate (I haven't played the XBLA version).

¡firedog say:

How do I get to the options for win live mail 12

Windows Live Mail v.12 is very old and obsolete. The oldest current version is v.14, the latest that will run on Windows XP. The latest version is v.16, which requires Windows 7 or 8. It is known as Windows Live Mail 2012. 

In v.12, Options are on the Tools menu. On later versions, program options have been well hidden.

There are several ways of accessing them:

The nondescript button giving access to some functions. The Quick Access Toolbar is above it.


  1. Click the nondescript* button to the left of the Home tab. On the menu that opens, select Options and then Mail.
  2. Click the nondescript button to the left of the Home tab. On the menu that opens, select Options. Right-click on Mail and select Add to Quick Access Toolbar (QAT). This will put a button for one-click access to the options sheet on the QAT in the top left-hand corner of the window. Hover over it; it’s called Mail.
  3. Use accelerator keys: press Alt, then F O M. Or, if you did (2), press Alt, then n if the Mail button is the nth from the left on the QAT.
  4. My favourite: the keyboard shortcut Ctrl-Shift-O for Options.

* This button is labelled File in W8.

how do u enable keyboard shortcuts ??

Keyboard shortcuts are enabled automatically. There is no user option to enable or disable them. KBSCs are not officially documented anywhere; you'll find a partial list of those available in Windows Live Mail 2011 and 2012 here: Keyboard shortcuts 

need to use control P to print message...

Ctrl-P will work if a message is being viewed, either in the reading pane or in its own window. It should invoke the default printer.
