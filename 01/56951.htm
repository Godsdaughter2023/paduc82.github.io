johnny-7 say: This is a well-known problem, which as it turns out is actually a kind of feature of Windows 7.

Basically, there's a script that Windows 7 runs that regularly checks your desktop shortcuts and if it finds more than 4 of them "broken" (i.e., pointing to something that's not available at the moment), it removes all the "broken" ones!

I guess Microsoft feels somehow responsible for helping us keep our desktop clean. Yeah, right.


The script that generates this activity is in:

C:\Windows\Diagnostics\Scheduled\Maintenance\TS_BrokenShortcuts.ps1

Open it, (you may have to change your security settings to access it), and not too far from the end, about the 3rd paragraph from the end is a statement that compares the length of the list of broken shortcuts to 4.  It's the only numeral 4 in the entire script.

If you replace the 4 with a very large number, the problem should be solved.

It's easy and quick and you don't have to turn off the other Maintenance items that may actually be useful!

superfetch - Microsoft Community

harshal&8558 say:

superfetch

what work superfetch in laptop and what effect when stop and desable this servise.

a. user say:

Hello,

Thank you for posting your query on Microsoft Community.

Superfetch is a service in Windows that's designed to speed up app launching by preloading certain apps based on your usage patterns.

Every time  you run an application in your PC, a Prefetch file which contains information about the files loaded by the application is created by Windows operating system. The information in the Prefetch file is used for optimizing the loading time of the application in the next time that you run it. SuperFetch attempts to predict which applications you will launch next and preloads all of the necessary data into memory. Its prediction algorithm is superior and can predict which next 3 applications you will launch by what time in a day.

However, it should not be maxing out your processor all the time. If it is slowing down your computer, you can try to disable it.

To manually disable Superfetch in Windows 8, launch the Windows Services manager by right-clicking on the Desktop Start Button, choosing Run, and typing services.msc. Alternatively, you can search for services.msc from the Start Screen.

In the Services Manager, scroll down to find Superfetch, which is controlled by the Windows service called SysMain. Double-click Superfetch to launch its Properties window and click on Stop to stop it.

Under the “Startup Type” drop down menu, select Disabled. Click Apply and then OK to save your changes. Close the Services Manager and reboot to have the change take effect.

Hope this information is helpful. Please do let us know if you need further assistance with Windows.
