Why do I have a black screen when I start Windows 7? - Microsoft Community

pollysyllabicprincess say:

Why do I have a black screen when I start Windows 7?

courtneyj say:

If you start Windows and see a blank or black desktop with no icons, video adapter problems are the most common cause of this issue.

 

Try the following troubleshooting steps:

Method 1: Gain access to your computer and reinstall your current video driver

Step 1: Launch Task Manager and restart Explorer.exe

1.       Press the Ctrl + Alt and Delete keys on the keyboard at the same time

2.       Click the Start Task Manager

3.       Click the Processes tab, right-click explorer.exe from the list of running processes

4.       Click End Process to stop the Explorer.exe process

5.       Click File from the menu and click New Task (Run)

6.       Type Explorer.exe in the Open: window and click OK

 

Step 2: Uninstall your video driver and allow Windows to re-install it

7.       Click Start and type Device Manager in the Search box

8.       Click Device Manager under Programs

9.       Restart your computer to allow Windows to reinstall the driver

 

Note: Once the driver is reinstalled you may need to reboot your computer again to complete the installation. Additionally, you may also need to adjust the screen resolution to its previous setting.

Method 2: Check for registry issues

 

If reinstalling your video driver does not work, follow the steps in method 1 again and then perform the following troubleshooting steps to check for registry issues:

1.       Click Start and type regedit.exe in the Search box

2.       Click Regedit.exe under programs

3.       Browse to the following registry key

4.       HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon

5.       Click to select the \WinLogon key

6.       Browse through the list to \Shell

7.      The value of Shell should be explorer.exe

 

Note: If the value of Shell is not Shell, double-click Shell and type explorer.exe in the Value data: text area and click OK and then close the registry editor and restart your computer

 

Method 3: Enable low-resolution video (640x480), and then update your video adapter driver

If resolving registry issues does not work, perform the following steps to enable low-resolution video:

  1. Click the Start button , click the arrow next to Shut Down, and then click Restart.
  2. Press F8 before the Windows logo appears.
  3. On the Advanced Boot Options screen, use the arrow keys to highlight Enable low-resolution video (640×480), and then press Enter.
  4. Log on to your computer with a user account that has administrator rights.

 

If you can see your desktop, there's a problem with your video adapter. Try updating your video adapter driver. For more information, see Update a driver for hardware that isn't working properly.

 

Method 4: Start your computer in Safe Mode and disable your video adapter

If enabling low resolution and then updating your video adapter driver did not work, perform the following troubleshooting steps to disable your video adapter driver:

 

  1. Click the Start button , click the arrow next to Shut Down, and then click Restart.
  2. Press F8 before the Windows logo appears.
  3. On the Advanced Boot Options screen, use the arrow keys to highlight Safe Mode, and then press Enter.
  4. Log on to your computer with a user account that has administrator rights. When your computer is in safe mode, you'll see the words Safe Mode in the corners of your monitor.
  5. Open Device Manager by clicking the Start button , clicking Control Panel, clicking System and Security, and then, under System, clicking Device Manager. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  6. In the list of hardware categories, expand the Display adapters category. Make a note of your display adapter.
  7. Right-click your display adapter, and then click Disable. When prompted for confirmation, click Yes.
  8. Close all open windows and restart your computer normally.
  9. When your computer restarts, if you can see your desktop, there's a problem with your video adapter. Try updating your video adapter driver. For more information, see Update a driver for hardware that isn't working properly

 

If these steps did not solve your problem, try booting into Safe Mode again and follow the steps in System Restore: frequently asked questions to restore your computer to an earlier point in time. If that still does not solve the problem, contact your computer manufacturer or technical support for further assistance.

Since you have mentioned that you face this issue only when you enter an incorrect password, I was wondering if this could be a network issue. When you enter a password to login to the computer, Windows initially checks the password with a copy of the password stored locally. Then it verifies with the Microsoft servers to check if you have recently changed your password. If the online servers confirm that the password is correct, then you are logged into the desktop.

 

I believe something is interfering with this verification process and hence the delay. Either something is interfering with the process where Windows verifies your password with the local copy or there is a delay in Windows reporting that the password you have entered didn’t match with the local copy of the password and then tries to verify it with the online servers and get a finalized result that it’s an incorrect password.

Before you do this, check the network connection in the lock screen. Check if the PC is connected to the network and then try.

 

If you couldn’t find anything wrong with the network, then you can try disabling any third party security software that you have installed from starting up automatically during the startup. If there is a delay in the security software startup, it could also interfere with the internet connection resulting in such issues.

 

If the issue still persists, boot the computer in safe mode to check if any third party service is conflicting with the password verification process. Booting in Safe mode boots only the primary services that is needed for the computer to function properly and disables all other third party services. Refer to this article for more information:
