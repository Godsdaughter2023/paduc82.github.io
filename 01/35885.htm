Method 1: Run System File Checker.

  3. Type the command below:
    SFC /Scannow
  4. Hit Enter to execute the command. It may take a few minutes to complete. If you find Verification 100% complete, click X to close the Command Prompt.
  5. Reboot the PC for the changes to take effect.

Method 2: Use the DISM tool.

You can use the Deployment Imaging and Servicing Management tool to update the files and correct the problems in Windows.

  1. Press Windows + X keys.
  2. Click on Command Prompt (Admin).
  3. Type the command below:
    DISM /online /cleanup-image /restorehealth
  4. Hit Enter to execute the command. It may also take a few minutes to complete. If you find The operation completed successfully, click X to close the Command Prompt.
  5. Reboot the PC for the changes to take effect.

Method 3: Windows PowerShell

This troubleshooter will re-register the taskbar and other app.

  1. Press Windows key.
  2. Search for Windows PowerShell.
  3. Right-click on Windows PowerShell and select Run as administrator.
  4. Copy and paste the command below:
    Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
  5. Hit Enter to execute the command.
  6. It may take a while to complete.
  7. Once it's done, reboot the PC for the changes to take effect.
