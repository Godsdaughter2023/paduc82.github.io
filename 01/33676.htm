Sorry I missed this post. As you can see, this is a server system message about a temporary server problem, about a message that couldn't be retrieved. This is not unusual with AT&T Yahoo accounts, which are notoriously sensitive to minor corruptions in messages in the mailbox.

Now move the oldest half of the messages in the temporary folder that haven't yet been downloaded back into the inbox and again send/receive in Windows Live Mail.
