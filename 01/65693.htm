hallie s say: The US international keyboard will put in an accent by default if you type an accent key (e.g apostrophe, double quote, caret, `, ~) followed by a letter that takes that accent (a vowel or c or n). For example if you type and apostrophe (') and then type the letter e, you get é. (Nothing shows up on the screen after you type the apostrophe; the keyboard is waiting to see if the next letter you take accepts an accent.  If you type double quote, e you get ë by default.  If you type double quote, R, you get "R, because the letter R never takes an accent.)
To suppress this feature, you have to hit the space bar after you type in the accent character.  Example: to type the French word, "j'espère", you have to type;
 j, apostrophe ('), SPACE BAR, e, s, p, accent grave (under the tilde key), e, r, e.
If you just type j, apostrophe, e, you get jé because the letter e can take an acute accent.
There's a knowledge base article on how to use the US international keyboard in the Microsoft TechNet:
http://support.microsoft.com/kb/97738/en-us

frederik long say:

A padlock next to a folder name means that only users with administrative privileges have access to this folder.

Under Windows 7, user profiles are kept in the folder C:\Users. Windows does not store any essential files in user profile folders.

To get rid of an unwanted user profile folder, you can do this:
1. Reboot the machine.
2. Log on under an account with administrative privileges.
3. Click Start.
4. Type the three letters cmd into the Search box.
5. Press Ctrl+Shift+Enter
6. Click "Run as Administrator".
7. Type the following command:
    rd  /s  /q  "c:\Users\xxx" {Enter}

Replace xxx with the name of the profile folder you wish to delete. Note that this action is irreversible. The files and folders you delete willnot go into the Recycle Bin.

How do I switch parental controls off?

maxim z. say:

Greetings!

After a little while of research, I found out how (since I don't use windows 7, I generally need more research). So this is how you disable parental controls:

Hold windows key and push the 'r' key. By doing so a console called "Run" should appear. In case it does not appear, right-click onto the windows key on your task bar, then push "Run". Once the console appears, type in "msconfig". Once you have done that, another menu will appear. At the top, open the services tab and scroll down until you find parental controls, then uncheck the box. Push 'Apply', then 'Ok'. Your parental controls should be disabled. If they are still enabled, try restarting your computer.

Hope this helps you, and reply is it didn't!
