jeeped say:

By default, Windows will use Notepad to open any pure text file (e.g. .TXT) that holds no paragraph or font formatting or other concerns. If you wish to use some other application (e.g. disable Notepad) to view these files, you need to change the File Association.

Click Start, and type in default programs, then hit <enter>. Click Associate a file type or protocol with a program from the list on the right had side. Once the full list has loaded in the new window, scroll down the list for .txt and select it. Click the Change Program... button and make a selection from the Recommended Programs or click Browse to locate the program of your choice. Word or WordPad are two likely choices. Save your changes by clicking OK as you exit each window.

There may be other Text Document entries with Notepad as the opening application, so you will likely want to review the list.

Alternately, you can access Default Programs from the Control Panel. The rest of the procedure should be the same.
