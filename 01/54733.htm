shinmila h say:

Hi brit007ca,

 

This error message can occur if you chose to run the MSI file during a download and later attempt to repair or uninstall the software using an MSI file with a different filename.

 

For example, if you download the Ask Toolbar.msi file and choose to run the file (rather than save it), Internet Explorer will add a number to the filename in brackets (such as [1]). In this case ask toolbar would actually be installing from a file named Ask Toolbar[1].msi. Windows Installer saves this filename in the registry and expects the same filename if you run a Repair or Uninstall from a double clicked file. If the filename does not match, an Error 1316 will occur.

 

Here are 2 things you can try.

 

1.    One solution would be to rename the saved MSI file to match the name indicated in the Error 1316 dialog.

Once you rename the saved file to include [1] the Uninstall should complete normally.

2.    Run a third party uninstaller or clean up utility to remove registry entries of Ask Toolbar.

 

Note: Microsoft cannot guarantee that any problems resulting from the use of Third Party Software can be solved. Using Third Party Software is at your own risk.

I'm Independent Advisor, here to help you with your question.

First, check the type of license installed on your system.

1. Open administrative Command Prompt.

2. Type slmgr /dlv and press Enter key.

3. Note the Product Key Channel section Windows Script Host box:

If it mentions OEM <method>, you’ve OEM license installed.

If it says Retail, you’ve retail license installed.

If it mentions GVLK, VOLUME_<method> then you’ve volume license installed. Note that GVLK stands for Generic Volume License Key.

So if you've OEM license, run please run Activation Troubleshooter:

1. Open Settings app, navigate to Update & security > Activation.

2. Click on Troubleshoot link to launch Activation troubleshooter.

3. The troubleshooter will automatically detect the issue and fix it.

Most fax machines, printers, and scanners require driver software to work properly. If you recently updated your version of Windows to the latest version, it's possible that the current fax machine driver is for the previous version of Windows and may not work (or work well) with your version of Windows 10.

While it's likely you already have the most recent driver for your fax machine after a Windows 10 update, if you're having trouble with a device, you can try fixing it by updating the drivers or reinstalling them. Here's how:

  1. Select the Start button
  2. Type Device Manager, and select it from the list of results.
  3. Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver Software.

1. Open Device Manager,

2. Right-click (or tap and hold) the name of the device, and select Uninstall.

3. Restart your PC and Windows will attempt to reinstall the driver.
