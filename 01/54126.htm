xb1r athena say:

DMZ

Login Information

 

  • Gateway: 192.168.2.1
  • Username: admin
  • Password: admin

 

 

 

DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "DMZ & UPnP" from the menu to the left, located below "Advanced Setup".
  5. Next to the "DMZ Host IP Address" select the radio button next to the drop down menu.
  6. Select the IP address of the Xbox from the drop down menu.
  7. Select Apply Changes.

 

Disable DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "DMZ & UPnP" from the menu to the left, located below "Advanced Setup".
  5. Next to the "DMZ Host IP Address" select the radio button next to text box, below the drop down menu.
  6. Delete anything that might be typed there.
  7. Select Apply Changes.
