andre da costa say:

Your title says dual boot, so I guess that assumes you want to dual boot the old drive running Vista with the new drive running Windows 7:

You will need to install the old drive then do the following:


1. Put the Windows Windows Vista installation disc into the disc drive, and then start the computer.

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

4. Click Repair your computer.

5. Click the operating system that you want to repair, and then click Next.

6. In the System Recovery Options dialog box, click Command Prompt.

7. Type BOOTREC /FIXMBR, and then press ENTER.

8. Type BOOTREC /FIXBOOT, and then press ENTER.

9. Type Drive:\boot\Bootsect.exe /NT60 All, and then press ENTER.

 

Note: In this command, Drive is the drive where the Windows 7 installation media is located.

Repeat this step with the Windows 7 disc.

Please note, if its a Windows Vista installation that came preloaded on a OEM branded computer such as a DELL or HP, you will not be able to boot or use it. Might as well just attach it as a data drive and take ownership of the files and folders:

Lync 2010/Earlier clients - %userprofile%\tracing
