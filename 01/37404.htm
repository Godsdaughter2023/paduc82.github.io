lemp say:

The "Send in email" feature works by "associating" a "default mail program" with the commands used in applications such as WordPad and many others.  The "default mail program" must be installed on your computer. 

That means that you will ONLY be able to use the "Send in email" feature if you are using an email application that is installed on your computer. The two most common such applications these days are Microsoft Outlook (but see below) and Mozilla Thunderbird, although there are others.

The problem is that if you use something like Gmail or any other "web mail" service that works by going to a website with your browser (Internet Explorer, Firefox, Chrome, etc.), you don't have a "default email program" and thus can't use the "Send in email" feature. 

Over the years, Microsoft has used the word "Outlook" to describe many different programs, some of which are installed on your computer and can be used as the default mail program and some of which are web-access only and thus can not be used as the default mail program.  If you use a program with "Outlook" in its name, you may need to ask for help in the part of these forums where the Microsoft Office experts hang out:  https://answers.microsoft.com/en-us/msoffice/forum

For a general discussion about this, including the possibility of using third-party applications that enable web mail applications to be used as the default mail program, see http://montgomeryminds.com/blog/windows-desktop-applications-sending-mail-mapi-via-webmail-like-gmail-yahoo-hotmail-aol/
