I would like to have more information regarding this issue. It seems that you have not tried the complete troubleshooting steps on the computer. Please do answer to these questions for a better understanding.

I would suggest you to try the following methods and check if that resolves the issue.

 

Method 1: Update the Firmware of your Router

 

I would first suggest you to back up your Router settings and update the Router Firmware. Check the Router manual for more information or contact the Router support for the same.

 

If that doesn’t help, then try the next method to flush the DNS in the elevated command prompt and check if that helps.

 

Method 2: Flush the DNS using elevated Command Prompt

 

a)       Click on “Start” button and type “cmd” in the search box.

b)       Right click on “Command Prompt” icon from the search results and then select “Run as administrator”

c)       On the command prompt window, type the following commands, and then press “Enter”

 

ipconfig /flushdns

ipconfig /registerdns

ipconfig /all

 

Note: There is a space between ipconfig and / in the above commands.

 

The ipconfig /all command displays Windows TCP/IP settings for all your network adapters. If your IP address for your Local Area Connection is either 0.0.0.0 or 169.x.x.x (where x is any number), then your computer is not getting an IP address from the router. If that is the case, then type the following commands in the command prompt window and press “Enter” after each commands.

ipconfig /release
ipconfig /renew

 

Test it again by typing ipconfig /all to see what the address is? (If you are on a home router, it should start with either 192.168.1.x or 192.168.0.x or 10.x.x.x)

 

You may also try resetting the TCP/IP settings and check if that helps.

 

Method 3: Reset the TCP/IP settings

 

You can also try to reset the TCP/IP on the system and check if that helps.

 

Change TCP/IP settings
