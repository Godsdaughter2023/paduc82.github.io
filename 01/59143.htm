    a. user say:

    To send a message, I type in the first letter of the persons name.  There are several other names that are no longer valid.  How do I delete those unwanted names?

    Windows Mail works in a strange way.
    When You type in a few letters, a list of names will show up for you to choose. That is called auto-fill.
    What it actually does is to show you a list of LAST 29 names you have responded to. It does not come from your address book.
    As you e-mail to new names that are not within the list of 29, the oldest names in that list will be deleted, and so on.
    You can't delete names from that list, but they will go away eventually because you no longer use them.

    :: ---------------------------------------------------------------------------

    if %errorlevel% equ 0 (

      goto :BypassVersionError
