    spiritx say:
    Hi,

    To add to the other excellent advice :

    If you format a drive that wipes everything off the drive including the recovery partition which means
    you would need a Windows Installation media from the System Maker to re-install Windows.

    Restore to factory out of the box settings - wipes the programs and data off the system and reverts
    it back to the state it was in when you opened the box.

    Format - wipes everything off the drive including any recovery  (there is malware that can survive a format)
    (Normally the recovery is on a separate partition which would have to be formatted separately to wipe it
    however it might not be accessible if you format the C: or main partition.)
