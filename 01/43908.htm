sarahpaul say: .msdvd file is the project file of windows DVD Maker. It doesn't have the DVD content in it. Is the computer you use to made the .msdvd file the same computer with the one you use to burn to DVD with your DVD maker? If not, only the .msdvd file can't be burned with DVD maker. To burn to DVD, you still need the DVD content in the computer where you made the .msdvd file.

You said that "I uploaded a video that I recorded and added some callouts, dvd menu and other things using vista movie maker. I published it and then burned a dvd." Did you burn the DVD by only copying your published video onto the DVD? If yes, it's sure can't be played on dvd players. If you want to burn a DVD playable on DVD players, You must go to the computer on which you made the .msdvd file, and then burn the DVD with DVD maker.

If you want to make AVI video, you just need to use Windows Movie Maker by selecting "publish to this computer". Then choose the output file extension as AVI. Attention, the output AVI file is dv-avi. If you want AVI video in other format, then you need a third party converter software to convert your video to what you want.
