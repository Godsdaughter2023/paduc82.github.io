a. user say:

Hi Joseph,

Thank you for posting  your query on Microsoft Community, I can understand that you are stuck in a boot loop that says “bootmgr is compressed, press ctrl+alt+del to restart”.

Though disk compression saves disk space, still if the compression is not done in a proper way, then it may cause serious issues like this one. BOOTMGR or Boot Manager allows different operating systems run on the same computer, it is also called a boot loader, it is responsible to initiate and load the OS into the memory. Bootmgr file remains hidden, so during the disk compression it got compressed and now your system is unable to boot but don’t worry let me provide you with some instructions to redeem your computer from this situation.

Method 1:
Startup repair: This one is the most basic way, see if that helps
Refer to the link below:
What are the system recovery options in Windows?
http://windows.microsoft.com/is-is/windows/what-are-system-recovery-options#what-are-system-recovery-options=windows-7
Note: You may need to perform the above operation 2-3 times, since may more than one boot files got compressed. If that doesn’t solve the issue try,
Method 2:
Repair bootmgr using command prompt manually:

Steps:
a. Insert your Windows 7 disc, restart your computer and keep pressing F8.
b. Select Command Prompt from the list of System Recovery Options.
c. Type in bootrec /FixMbr & hit Enter.
d. Then type in bootrec /fixboot & hit enter.
e. Then type in bootrec /rebuildbcd & hit Enter.
f. Then type in exit & hit Enter.
g. Restart your computer and check with the issue.
For further references please go through these Microsoft Articles below:
1.Windows error message “Bootmgr is missing Press Ctrl+Alt+Del to restart”
http://support.microsoft.com/kb/2622803
2.Use Bootrec.exe in the Windows RE to troubleshoot Start Up issues.
http://support.microsoft.com/kb/927392
