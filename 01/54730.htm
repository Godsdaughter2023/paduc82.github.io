The error code 0x80070005 is also described as ACCESS DENIED (you do not have permission to install updates.

Please check that you have logged on an account with administrative privileges.

First, make sure that your anti virus software isn't locking the registry and preventing the installer from modifying the same.

 

Please follow these instructions if you are encountering this error code when checking or installing updates via Windows Udpate.  The following is only applicable to Windows Vista.

1. Download AccessChk (Sysinternals).  This tool allows you to evaluate the access level of specific users or groups of resources including files, directories, Registry keys, global objects and Windows services.  Here's the link to download the tool: http://technet.microsoft.com/en-us/sysinternals/bb664922.aspx

2. Save the zip file on your desktop, and extract the file:
- Right click on the file, and select Extract All...
- Click Next when prompted for the Destination.

As a result, you should see a folder called AccessChk  on your desktop.

3. Open the folder AccessChk 

4. Hold the shift-key and right click in the window.  Select "Open Command Prompt Here"

5. A command prompt window should open with a similar prompt:
C:\Users\<USERNAME>\Desktop\accesschk>

6.  Type the following command, and press Enter:
accesschk.exe -s -n "nt service\trustedinstaller" -k "hklm\software\Microsoft\Windows\CurrentVersion\Component Based Servicing" >accesskchk.txt
Tip: You should be able to copy and paste the command into the command prompt.

7. Close the Command Prompt Window.  Open the Accesschk folder on yoru desktop (if it has been closed).

8. Double click on Accesschk.txt (the text file), this should open Notepad.

If this does not help,

1. Please download the subinacl.msi file from the following link and save the installation patch onto the Desktop:
 
http://www.microsoft.com/downloads/details.aspx?FamilyID=e8ba3e56-d8fe-4a91-93cf-ed6985e3927b&displaylang=en#AffinityDownloads
SubInACL (SubInACL.exe)
 
2. Please go to the Desktop and double click the downloaded file.
3. Please select the C:\Windows\System32 folder as the Destination Folder during the installation. Later we will use this tool to reset the permission settings on the current machine.
4. Click the "Start" Button, in the "Start Search" bar, type: "notepad" (without quotes) and press Enter.
5. Copy the following commands and then paste them into the opened Notepad window:
 
@echo off
 
subinacl /subkeyreg HKEY_LOCAL_MACHINE /grant=administrators=f
subinacl /subkeyreg HKEY_CURRENT_USER /grant=administrators=f
subinacl /subkeyreg HKEY_CLASSES_ROOT /grant=administrators=f
subinacl /subdirectories %SystemDrive% /grant=administrators=f
subinacl /subkeyreg HKEY_LOCAL_MACHINE /grant=system=f
subinacl /subkeyreg HKEY_CURRENT_USER /grant=system=f
subinacl /subkeyreg HKEY_CLASSES_ROOT /grant=system=f
subinacl /subdirectories %SystemDrive% /grant=system=f
 
@Echo =========================
@Echo Finished.
@Echo =========================
@pause

7. Refer to the Desktop and right click the reset.bat file, then choose "Run as administrator."

8. You will see a DOS-like window processing. 
 
NOTE: It may take several minutes, please be patient. When it is finished, you will be prompted with the message: "Finished, press any key to continue".

Says you can play single player offline yet still requires you to need internet to even launch the game and I have a physical copy of the game and I am starting to regret my purchase

What exactly is the wording of the internet connection message?

Does the game itself show you the message, or do you get a system message like "Do you own this game or app?"

If you get a system message, then it might be that the console is not correctly detecting the disc. 

To resolve that, eject the disc and hold the power button on the console for about 10 seconds. Then start the console again, and make sure it's showing you a green boot screen with an Xbox One logo. 

If the boot screen does NOT show up, and the console immediately goes to the home screen, hold the power button again for 10 seconds. 

While the boot screen is showing, take your Forza Horizon 4 disc to the kitchen. There, apply a small drop of mild dishwasher soap to the silver side of the disc. Then wet your fingers, and carefully "massage" the soap around the silver side in a spiraling motion. When the whole silver side is covered in soap, rinse the disc under running water. 


Now only hold the disc with your index finger in the center hole and your thumb on the outer edge of the disc, with the silver side "outwards". Shake the disc several times to get the water off, and wipe the remaining drops with a soft, lint-free cloth from the center of the disc to the edge. 

After the disc is cleaned and dried, return to your console. Now try launching the game again.

If the error message appears after the game is started, be aware that Forza Horizon relies heavily on an online connection for your drivatar progress, rival events, Forzathon Live events and all that. 

So it might be necessary that the game at least connects online once, before it can be used offline (temporarily).

shekhar s say:
Hi Hi EMGoody,
 
 
Let us perform the below steps and check if that helps us in resolving this issue.
 
Step 1.
1. Open the Task Scheduler.
2. On the View Menu, select the 'Show Hidden Tasks' option.
3. In the left side menu, select 'Task Scheduler Library'.
4. In the Center section, right click the   'User_Feed_Synchronization'   task and select Delete.
5. Exit the Task Scheduler.
 
Step 2.
1. Next, select Start / Accessories. Right click 'Command Prompt' and select the 'Run As Administrator' option.
2. Click 'Continue' on the UAC prompt.
3. In the command window, type the following command.
 
msfeedssync enable
 
4. Press ENTER.
 
This command will replace and refresh the 'User_Feed_Synchronization' task.
 
This will not affect the RSS Feeds you are subscribed to.
