lemp say:

Generally, anything in a temp folder can safely be deleted.  Run Disk Cleanup (Start Orb > cleanmgr.exe > press Enter) or just click in the folder, press CTRL+A, and then press Delete.  If Windows indicates that a file is in use and can't be deleted, just press the "Skip" button.

Pagefile.sys should NOT be deleted.  This file is required by Windows and you should let Windows manage its size:

  • Click the Start Orb, type Advanced System Settings into the Search box, and press Enter
  • Click the "Settings" button in the top section ("Performance")
  • Select the "Advanced" tab
  • Click the "Change" button

If the box to "Automatically manage paging file size ..." is NOT checked, check it and OK your way out.  If the box was already checked, CANCEL your way out.

Hiberfil.sys is used if you "Hibernate" your computer and it is equivalent in size to the amount of RAM you have.  Hibernate is usually only used for laptop computers.  If you want to disable Hibernation (which will automatically eliminate hiberfil.sys), see https://support.microsoft.com/en-us/help/920730/how-to-disable-and-re-enable-hibernation-on-a-computer-that-is-running-windows

In general, you can safely delete *.log files.

dries geris say:
Hi, this happened to me too on my Pro 300, but it has nothing to do with Windows 8.
Probably the IP address from your printer/scanner changed.

I was also still able to print, but there was no communication with the scanner possible anymore.
I found out that the IP address of the hardware was changed.
The printer settings on my PC changed the IP address of the printer automatically to the new one, that's why I still was able to print, but the scanner setting still had the old IP in it's memory, therefore I was not able to connect to the scanner anymore.
It's strange, it's the same hardware, but you still have to configure the printer and scanner separately on your PC.
You can easily change the IP adres in the printer settings, but not in the scanner settings.

You can solve this by uninstalling and reinstalling the complete software, but here's how I solved it.
1) download HP Print and scan doctor.
2) Run the print and scan doctor, you will see that it will find the printer with an IP address and the scanner with another IP address.
3) write the scanner IP address down, close the scan doctor (don't follow the next steps, they won't work)
4) Open the printer webpage control panel (open windows explorer, go to network and double click on the printer, or enter the printer's IP address in your browser).
5) in the control panel, go to the tab "network", then choose in the left menu "IP-4 configuration", then manually change the existing IP address to the IP address from the scanner you just wrote down. 
6) Save it, Accept the warning you'll get.. The hardware itself now has again the correct (old) IP address, the same as the scanner has, so the scanner software will connect again. Problem solved.
7) But, of course you lost the connection to your printer now, as the printer settings on your PC still refer to the new IP address. So open your control panel, go to printers/hardware, right click on the printer and select "printer properties" (not just properties). Go to tab "ports", the port from your printer should be selected already, if not, select the port from your printer and click on configure port. Here you can change the IP address to the old one again, so the printer settings also refer to the same old IP address as the scanner and hardware.

This seems a long list of to-do's, but it's quicker then uninstalling and reinstalling the complete software.

On your desktop you will find the shortcut. Right click on it, click on Properties and then go to the second tab sheet. You will see there under Target "C:\Program Files (x86)\HP\csiInstaller\9D1DE902-8058-4555-A16A-FBFAA49587DB\Setup.exe".

At the end add  /ReconfigWireless.

So it looks like this: "C:\Program Files (x86)\HP\csiInstaller\9D1DE902-8058-4555-A16A-FBFAA49587DB\Setup.exe" /ReconfigWireless

Mind the white space between \Setup.exe" /ReconfigWireless

Save the changes and double click on the icon. The setup wizard will start, but this time, there's no option to delete the software, you'll see the option to update the wireless settings. Follow the steps and you're done. You can keep the shortcut on your desktop in case the IP address changes again in the future.

This way you don't need to delete and install the software again as it takes too long.
