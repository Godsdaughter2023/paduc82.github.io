How to share files running different versions windows on wireless - Microsoft Community

debbie3921 say:

How to share files running different versions windows on wireless network

I have a computer running windows 7 another windows vista.  They both r on the same workgroup & conncected on the same network.  They both can c each other.  When I try to open any files nothing can b accessed.  I thought I had shared the whole drive C on both computers but it won't let me access anything.  Can anyone help me?
jack mvp say:

Hi

Assuming that all your computers can get access to the Internet through the Router.

Win7 when configured on peer-to-peer Network has three types of Sharing configurations.

HomeGroup Network = Works only between Win 7 computers. This type of configuration makes it very easy to Entry Level Users to start Network sharing.

 Work Network = Basically similar to the previous methods of sharing that let you control what, how, and to whom folders would be shared with.

 Public Sharing = Public Network (like Internet cafe) to reduce security risks.

 For best results log to each computer's System Screen and set all the computers to be on a Network with the same name while each computer has its own unique name.

http://www.ezlan.net/Win7/net_name.jpg


Make sure that the Software Firewall on each computer allows free local traffic. If you use 3rd party Firewall On, Vista/XP Native Firewall should be Off, and the active Firewall has to adjusted to your Network IP numbers on what is some time called the Trusted Zone (consult your 3rd Party Firewall instructions

General example,  http://www.ezlan.net/faq#trusted

Please Note that some 3rd party Software Firewalls keep blocking aspects of Local Traffic even it they are turned Off (disabled). If possible configure the Firewall correctly, or totally uninstall it to allow clean flow of local network traffic.

Since you have a Network with Win 7 and other version of Windows you need to configure Win 7 as Work Network.

In the Network Center, clicking on the Network type opens the window to the right.

Choose you type of Network. Notice the check mark at the bottom and check/uncheck it according to your needs.

http://www.ezlan.net/Win7/net_type.jpg

Win7 - http://windows.microsoft.com/en-us/windows7/Networking-home-computers-running-different-versions-of-Windows

Win7  Work Network specific folder sharing - http://www.onecomputerguy.com/windows7/windows7_sharing.htm

Vista File and Printer Sharing- http://technet.microsoft.com/en-us/library/bb727037.aspx

Windows XP File Sharing - http://support.microsoft.com/default.aspx?scid=kb;en-us;304040

 Printer Sharing XP - http://www.microsoft.com/windowsxp/using/networking/expert/honeycutt_july2.mspx

 Windows Native Firewall setting for Sharing XP - http://support.microsoft.com/kb/875357

 Windows XP patch for Sharing with Vista (Not need for XP-SP3) - http://support.microsoft.com/kb/922120

 When finished with the setting of the system it is advisable to Reboot all the hardware including the Router and all computers involved.

*Note. Some of the processes described above are done Not for Windows sake but to compensate for different Routers and the way their firmware works and stores the info about the computers that are Networked.

 

Jack, MVP-Networking. EZLAN.NET

bossdweeb say:

Trainz 2009, yes ? Not obsolete.

If the game is a download from Steam or similar site there should be no need to install a patch as is will have

been pre-patched by Steam.

- Note the Trainz '09 Service Packs page suggest you clear out the Temp folder before trying to install the Service Pack

http://www.auran.com/servicepacks/#TS2009

If you have not done so, run the DirectX Web Installer to verify all DX files are installed.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

- The DX redist installer you linked to is outdated.

In the event your Win10 has all DX files and there is a corrupt file or files use the DX Redist (June 2010).

This installer is cumulative and includes the most recent DX9,10 and 11 files (Recent ? ).

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

Note that you first unpack the files to a folder of your choice. The desktop is a good place for this

as this folder can be deleted once the reinstall of DX if finished.

Step 2 - Use theDXSETUP.exe that you will find in this folder to finalise the DX reinstall.

As you seem to have previously unpacked the DX installer into your Temp Folder, you should clear the

Temp folder as well. You can do this by running Windows Disk Cleanup,  or by manually deleting the files.

- Some games install a copy of the DX file/s they use in the games main install folder. In this case if the file in question is

corrupt you can copy the file with the same name, eg- d3dx9_39.dll,  from System32 to the games main folder

(or sub-folder with the games .exe launcher ).

- Many games also install a sub-folder with a name such as Redist . This folder may have a DX installer in it as well

as Visual C++ (vcredistx86) that you could use to reinstall these features again.

- It probably does not apply to this game, but in the past some games had permissions issues if you installed to

the default location - Program Files(x86). Try installing to a new folder in another partition or HDD.

If you have only a C:\ drive/partition make a new folder there and try installing the game in that folder.

- For fixes to peculiarities with the game you would do better to post on a dedicated Trainz forum such as....

https://www.trainzportal.com/
