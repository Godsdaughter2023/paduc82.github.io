This is a Microsoft Management Console (mmc) snap-in.  To use it:
    Start -> Run -> "mmc"
When the console windows appears:
    File -> Add/Remove Snap-in -> Standalone (Tab) -> Add (button)

    "Add" -> Close ->  OK

xbr poseidon say:

MAC Filtering

Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - password

MAC Filtering

  1. Open web browser.
  2. Login to router by entering http://routerlogin.net  in to the address bar or by enter the default gateway IP address.
  3. The router will then prompt you to enter the user name and password. The default user name and password are: "admin", "password".
