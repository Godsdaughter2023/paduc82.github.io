[quote]

3) Scroll right to Your Information and select Account Security.

4) Select Microsoft account

5)Do one of the following:

  • If you already have another Microsoft (email) account, select Yes.
  • If you don't have another Microsoft account, go to http://accounts.live.com/ and click Sign up. After you have another Microsoft account set up, select Yes and continue with the next step.

7) Enter the email address and the password for the new Microsoft account that you want to associate with your gamertag, and then select Sign in.

8) Select Yes, change to change the Microsoft account associated with your Xbox LIVE account.

9) Select Update Contact Information to update your contact information or Done to finish.[/quote]

Hi,

If a Windows application or service uses lots of CPU resources for an extended period, the processor will get warm. When an i7 processor is under high utilization, it will become warmer compared to other processors. Here are some methods to resolve your issue:

Method 1: You may need to use performance monitoring tools to determine what is causing the high CPU utilization. Here's how:

  1. Use the Windows key + X keyboard shortcut to open the Power User menu
  2. Select Computer Management, 
  3. Click on Performance.

Method 2: You may also check Task Manager for any processes using excessive CPU, which can cause heating. Terminate any processes using excessive CPU to see if the heat decreases.

Method 3: Perform clean boot troubleshooting (to prevent unwanted processes from starting):

  1. Open Task Manager
  2. Click Startup tab
  3. Disable each item by clicking to highlight the item and then click Disable
  4. After disabling all items, restart the device
  5. Test to see if issue is still occurring
  6. If it is, re-enable the Startup items and restart the device to put it back into its original state.
  7. If the issue is not occurring after disabling all the startup items, one or more of those items is likely causing the issue. Follow the step by step procedure to isolate the cause of the problem:
    1. Re-enable the first half of the items on the Startup tab.
    2. Restart the device and test to see if the issue occurs.
    3. If the issue is still not occurring, disable the first half of the items then re-enable the last half of the items.
    4. Restart the device and test to see if the issue occurs.
    5. Continue disabling and re-enabling by "halves" until you isolate the issue to one or more of the Startup items.

Method 3: Install Surface and Windows updates.

Please keep us updated with the results.
