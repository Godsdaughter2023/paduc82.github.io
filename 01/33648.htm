linney say:

The two commands to enable or disable the Built-in Administrator are these.

 

net user administrator /active:yes

 

net user administrator /active:no

 

They are run from an Elevated Command Prompt.

 

 

How to start an Elevated Command Prompt in Windows 7 and Vista
http://www.bleepingcomputer.com/tutorials/windows-elevated-command-prompt/

 

 

 

The Administrator that those commands refer to is the Built-in Administrator, this is different (and more powerful) that the usual Admin user.   It is disabled by default for security reasons and should not be running all the time.  Basically it is a troubleshooting step.
