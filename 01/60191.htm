Standard OpenHCD USB Host Controller

For audio in/out, I know that having dedicated USB controller for that task alone is preferable to having many USB devices connected and sharing the resources. nithyananda j say:

Hi Frank3si,

USB2 is enhanced and the other is USB1.

The difference between front and back ports is that the front ports interface with USB 1.1 devices and transfer data to the back ports.

If you have an enhanced USB Host Controller then you have High Speed USB (USB 2.0)
