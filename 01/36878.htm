Here is what I do to accomplish this on my machine.  The Micro SD card is mounted as drive letter Y, and it must stay as drive Y for this to work as intended (you may pick your own letter but it must stay stable).  Then I edit two registry keys, one to set the icon for the drive and one to set the label for the drive.  The second part may be redundant but it supplies a nice label if the actual SD card has none.  These keys are

HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons\Y\DefaultIcon, for which I set the default (unnamed) value to "C:\Windows\System32\Imageres.dll,27", and HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\DriveIcons\Y\DefaultLabel, for which I set the default (unnamed) value to "Y-MEM" (my preferred name for the SD card).  These set the information for Drive Y (the \Y\ within the key).

Note that you must be running an administratively enabled registry editor to do this.

These values are not sticky, and seem to revert to their original values at times (at boot time especially), so I have also added a scheduled task that runs at system startup, at logon of any user, and on workstation unlock of any user; the task automatically does the registry edits described above (using regedit.exe and a .reg file).  This, too, involves creating tasks that will run a program with administrative priveleges, and there are solutions for this described in various places on the net.

If you are not completely competent with registry editing and system backup and restore then I wold not do any of this.

After all of the above, the drive shows up with the normal disk icon most of the time (but not all of the time) and I've yet to find the permanent solution.

danielmus2 say:

Movie Maker: Can I crop several short clips out of a long video and save them separately.

Hi

I have a one hour video of my opera.

I wish to select/clip several - say 10-   4 mins long  arias out of it.I know how to split those 10 arias, but I don't know if I can save them separately, so  as to make 10 short films out of it.

An obvious, but long process would be

Load the whole opera

split, one aria

delete everything else but the aria

save the aria as a film

Start again the process for the next aria.

help appreciated

Daniel

considerate_guy say:

Movie Maker doesn't have a way to export individual clips as movies. So about all you can do is save the clips individually. You can use Undo (Ctrl+Z) however, to speed up the process slightly to avoid having to reload the project each time.

Step 1 - Save a second copy of your project with another name (File > Save project as) in case you mess up.

Step 2 - In your project with the 10 arias already split separately, click on the *second* clip. Hold down Shift and select the last clip. Right-click on one of the selected clips in the block and choose "Remove". It should remove all 9 clips.

Step 3 - Now you should only have clip 1 in the storyboard. Choose File > Save movie and give it a name like "Aria #1".

Step 4 - Press Ctrl+Z on the storyboard to undo the deletes. If that doesn't work, reload the project.

Step 5 - Select the first clip and right-click, Remove. Select the 3rd through 10th clip, right-click, Remove.

Step 6 - You should have just clip 2 on the storyboard. File > Save movie and give it a name like "Aria #2".

Step 7 - Repeat the undo process until you have 10 clips again, then repeat the steps to save clips 3 though 10.
