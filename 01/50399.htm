petercon [msft] say:

Hi, venugopalLV.

As rI03 points out, there are language interface packs available for languages of India, including Assamese, Bangla, Gujarati, Hindi, Kannada, Konkani, Malayalam, Marathi, Odia, Punjabi, Sindhi, Tamil, Telugu, and Urdu. These will allow you to display the Windows shell user interface in these languages.

But even before that, you probably want to view and edit content using these languages. There is that level of support for all the above languages and more in every Windows system. In Windows 10, the best place to start is to set up your language profile in the Settings app:

  • Click Settings.
  • Click the Start button.
  • In Settings, click Time & language.
  • Click Region & language.

If the language you want to use is not included in the list of your languages that appears on that page, then you want to add it:

  • Click the "+" icon next to Add a language.
  • Scroll to find the language you want, then click on it to add it to your language list. In some cases, when you click on the language, you'll be take to a list of regional variants; in that case, click on the particular regional variant you want.

Once you have added a language to your language list, a few things will happen.

The first thing you might notice is that the system may automatically install some optional features associated with that language. These optional features may include additional fonts, or capabilities for text entry and editing, such as text prediction. These are installed via Windows Update, so you need to be online. If you are online, then this should only take a few minutes.

The next thing you might notice is that an additional icon may appear in the desktop system tray. Here's how it appears on my system:

(I added the red border to the screenshot.)

When you click on this, it will present you with the list of languages you have added to your profile, as in this example:

You can use this with a mouse to change the language for keyboard input. There are also keyboard shortcuts for this:

  • Windows key + Space will cycle forward through the list.
  • Windows key + Shift + Space will cycle backward through the list.
  • Windows key + Space + 1 will select the first language in the list.
  • Windows key + Space + 2 will select the second language in the list.
And so on for up to nine languages.

Another more subtle change is that, when you browse on the Web, the browser will report your language preferences to the site so that the site can provide content in the language most appropriate to you. (This has always been supported in IE, but you had to set the preference in Internet Options and most users never found that. This consolidated language settings behaviour was added in Windows 8.) Now, many Web sites don't follow the language preferences reported that way, but the language experience should be better in sites that do.

Adding a language to your profile can also affect the way that Windows Store apps behave. The modern Windows platform was designed to have apps automatically follow your language preferences as best as possible, if the app developer has localized the app's user interface for different languages. The order of languages in your preferences is important for this, since apps will run in the language that the app supports that is the highest preference in your list. Back in the language area of Settings, if you have added a language to your profile and click on that language, you'll see "Set as default" as an option. That will set that language to be your highest preferences, so any apps that support that language will run in that language.

(Note: when you change your language preferences, apps that support different languages may need to install an update before they can start running in a different language. So, it might take a little time before you see an app change. You can speed that along by running the Store app and manually checking for updates.)

If you also want to have the Windows shell display in your language, then you can take an additional step to enable that after you have added the language to your profile. Back in the language area of Settings, click on the language in your language list and select Options. If a language interface pack is available, then you'll be able to add it there.

I hope this helps.

Load Windows XP from external DVD drive - Microsoft Community

jummers say:

Load Windows XP from external DVD drive

Replaced hard drive on laptop. DVD drive does not work on laptop. How do I install windows XP from an external DVD Drive to the new hard drive?
darrennyq say:

You need to change the boot order in your computer BIOS.

 

The boot order is the order that computer tries to boot from, for example, it might check the internal DVD first, then if there is nothing in there, move on to the hard disk.

You need to plug in your external DVD drive (presumably it's USB).

 

Then go into the BIOS setup - look for boot order and move the external DVD drive to the top of the boot list, so it is the first device to be "booted" from.

Next restart your laptop and it should then boot from the external DVD drive, which presumably contains Windows XP installation media (DVD/CD).
