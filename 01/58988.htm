    There are a couple of types of "system images".

    The first one is the default configuration shipped from the factory. These days most computers ship with a "reset partition" that contains this info. It is cheaper than providing Windows install disks. If you insist on a factory install disk, then yes, they will charge you extra fro them

    USE RESET TO RESTORE YOUR WINDOWS 10 PC - FREE DOCX/ EBOOK DOWNLOAD
    http://ligman.me/29yWAAO



    The other is an image copy backup. Contrary to the general flow of suggestions already made I will say you should make an image copy backup shortly after buying a new computer. BUT don't do it immediately. First, do:
    . * Windows update more than once to get all outstanding updates
    . * Install personal applications, ie Office, browser
    . * Customize Windows and Applications
    . * "Decrapify" your computer, uninstall unneeded apps, clean disk etc
    . * manually trigger HD defrag

    Now you can do an image copy backup using the built in "Windows 7" Backup tool or one of the free alternatives. By having this image copy, you can use it to restore the computer to a "clean install" but without having to redo all of the above steps. This gives you one more disaster recovery options, each subsequent one requiring more work to get your computer fully functional again ie:

    . * Restore using this configured image copy
    . * Restore using factory reset option
    . * Clean install of most current Windows update version.
