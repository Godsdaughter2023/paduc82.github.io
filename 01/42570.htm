'rickt say:

Hi Randy,

Using a new hard drive shouldn't be a problem for Activation, but you have to do a fresh install if you're going from 32 to 64.

With that in mind, assuming you're going to keep the 500GB drive for other things, I'd suggest the following:

- download the 64 bit install (of the exact same type as the old one, of course: OEM, Pro, Home, Retail, whatever)

- install the HDD, change the boot order in the BIOS to look at the 1TB drive first

- install the OS on the 1TB from the USB stick or DVD you made earlier; it might put the 500GB's OS into the boot menu as well: no worries, you can just take it out later on if you want. 

- go on the 'net and let it Activate itself and install updates

- migrate the personal files over to the new drive, install programs (64 bit of course, though 32 will work)

- play with it a bit to make sure everything's working right

- take the OS off the old drive, if you want.
