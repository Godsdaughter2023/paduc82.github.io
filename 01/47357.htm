greg carmack - mvp 2010-2020 say:

Restart and press the HP ESC button to get menu, choose Diagnostics. 

That only 32gb of the 1tb hard drive remains suggests it's a hybrid drive with a separate 32gb caching chip to speed it up and possibly contain the boot files. 

You can check this by creating bootable media from Media Creation Tool, power up while pressing the ESC or F9 key to get a boot media, boot media as UEFI device if offered, then choose Custom Install, see what drives are offered.  If it's a 1tb drive with 32gb caching chip which is now unformatted you'll need to reinstall if the hard drive passes diagnostics. 

You can attempt to recover files using File Recovery software, even attempt a partition undelete using Partition Wizard Partition Recovery Wizard - Video Help in case it's recoverable as a partition.  But if nothing was done to delete it then it's likely a failing drive. 

If the drive fails diagnostics then I'd replace it with an SSD which is the best upgrade you can get, lightning fast, even faster than a new higher end PC without one.  Then do the gold-standard Clean Install Windows 10  which compiles the best possible install of Windows 10 that will stay that way as long as you stick with the tools and methods given.
