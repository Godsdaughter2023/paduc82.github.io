lemp say:
Angela, that's not an "alert."  It's just an icon showing that Task Manager is running (just like the other icons in that area show that other programs are running).  Normally, you shouldn't be able to have multiple instances of Task Manager running (which would show multiple icons), but some malware may cause that to happen.

The bright green color goes up and down as the percentage of CPU cycles rises and falls.  If it's all bright green, that means that you're using 100% and if it's all dark green, that means that it's close to 0% -- your computer is idle. 

As you have seen, if you "hover" your mouse of the Task Manager icon, it shows a pop-up with the percentage of CPU cycles being used; if you right-click that icon, you should see two choices:  "Close" and "Always on top."  If the latter is checked, the Task Manager window won't get hidden when you open other windows.
