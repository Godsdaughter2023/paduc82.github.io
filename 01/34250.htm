andreasfalco say: I've found the reason the install is so slow! Well, sort of. The immediate cause of the problem in my case is a problem with the Volume Shadow Copy Service (VSS) which for some unfathomable reason stopped working. This service is needed for System Restore and most installer starts with a restore point creation. The install waits for this restore point creation for a long time and only continues with the install after the restore point creation fails. Unfortunately the installer does not give an error message about this failure.

You may also check whether this happens in your case too by trying to create a restore point manually. (Control panel / System, System protection menu on the left, Create button on the dialog window)

There are separate threads on this site about the error ("The creation of a shadow copy has timed out" http://answers.microsoft.com/en-us/windows/forum/windows_7-system/what-is-causing-the-creation-of-a-shadow-copy-has/44468e50-fbca-4aef-a012-f02a5ad2a8c8)
