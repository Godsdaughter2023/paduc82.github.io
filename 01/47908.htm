Switch your Xbox One back to DHCP.  Then configure your router to reserve an IP for your Xbox One (see the "add device with reserved IP" button in your first screenshot).  When you reserve an IP on your router, it will always hand out that same reserved IP to the Xbox via DHCP.  There is no need to configure a static IP on the Xbox (as you have found, there appears to be a bug with that).  Forward  the following ports to the reserved IP (assuming your router does not have a good, working UPnP implementation) :

Is resolution depend upon the video memory in graphic card? - Microsoft Community

ahsannajam say:

Is resolution depend upon the video memory in graphic card?

Is resolution depend upon the video memory in graphic card? if it is true so how much resolution a 2GB card can support? i mean on which card i will be able to run a game like "NFS RUN" on higher resolution if i have 2 cards with same specification but difference is 1Gb and 2Gb, i am talking about Nvdia GT 440 128 bits GDDR3.?
sc tom say:

Resolution is dependent on the GPU, not on the memory installed. The memory will help the screen refresh faster, and helps video run smoother.

According to the specs here:

http://www.geforce.com/hardware/desktop-gpus/geforce-gt-440-channel/specifications

you should be able to get a max digital resolution of 2560x1600 and an analog (VGA) resolution of 2048x1536. This is of course assuming that your monitor will go that high. The highest a card will go is limited by the monitor's capabilities also.

So, the more video card memory you have, the smoother the graphics should run, but it won't have any effect on the resolution.

SC Tom
