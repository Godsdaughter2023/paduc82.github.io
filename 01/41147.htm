Hi,

As per the issue description you are able to backup files but all of the files are not getting backed up. So, it means the service responsible for backup is running. However, for a workaround, I would suggest you to stop the service first and then restart it and see if it brings any difference to the issue.

To stop the service.

You must be logged on as an administrator to perform these steps.

  1. Press Windows key + R and type services.msc in the search field or runwindow and press ENTER.

  2. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

  3. In the Services Windows search for Volume Shadow Copy in the list, right-click on it, and then click Stop.
  4. Now repeat the same for Windows Backup service.

To Start the service.

You must be logged on as an administrator to perform these steps.

  1. Press Windows key + R and type services.msc in the search field or runwindow and press ENTER.

  2. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

  3. In the Services Windows search for Volume Shadow Copy in the list, right-click on it, and then click Start.
  4. Now repeat the same for Windows Backup service.

If issue persist, I would request you to run chkdsk command to scan your hard disk if there is any corruption with disk or not. To check if your hard drive is causing the issue we can try running chkdsk command.

Steps to run CHKDSK command.

  • Press Windows key + X.
  • Select Command prompt (Admin) from the list.
  • Press OK the UAC prompt if shown. That will open an elevated command prompt. (It will show Administrator in the caption.)  
  • Then enter the chkdsk command
    chkdsk c: /f (or other drive letter if applicable).
  • Confirm to run on next startup and reboot.
