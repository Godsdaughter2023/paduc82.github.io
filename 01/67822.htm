Hi Evelynne Marie:

Keyloggers are simple programs designed to stay in the background
and run hidden. As they run they invoke Windows® API to get access
to physical keyboard and to log each pressed key they listen. The following
API is used, GetAsyncKeyState();, FindWindow(); andShowWindow();. The
FindWindow() is often used to locate itself, and the ShowWindow() is used to
hide itself. The GetAsyncKeyState() is used to listen to keypresses. The
GetAsyncKeyState(parameter in form of Integer) and then the GetAsyncKeyState()
function will return a number which represents the keypressed. There
appear up-to 255 keys (the 255 is based on an old calculation).

To defend yourself against such use good antivirus like Kaspersky AV.
Once you find a Keylogger file kill its process and delete the file.

Method 1

Reregister the installer engine and later try to install the game.

a)   Click Start, click  Run then type cmd in the dialog, and then click  Command Prompt.

c)    Try to install the game again.

 

Method 2

Make sure that the Windows Installer service is not set to Disabled.

 

a)   Click Start Collapse this imageExpand this image, type services.msc in the  Search box or click Run then type  services.msc in the dialog (Windows XP or Windows Server 2003), and then press Enter to open  Services.

b)   Right-click the Windows Installer service, and then click Properties.

c)    If the Startup type box is set to  Disabled, change it to  Manual.

d)   Click OK to close the Properties window.

e)   Right-click the Windows Installer service, and then click Start. The service should start without errors.

f)    Click the File menu, and then click  Exit.

g)   Try to install the game again.
