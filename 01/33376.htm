demonnus say:
I've managed to find the solution to this annoying issue and I hope this post can help others in the community if they find themselves in a similar situation.

What I found it came down to was the public IP address I'd been issued by my ISP. I'm based in the United Kingdom, using Plusnet. The IP address I'd been issued was within the range of 146.x.x.x. Checking this IP address online showed it to be a UK address but it thought the ISP was based in the United States. Therefore, when attempting to download the problematic games, because they're region locked, it would see that my Xbox profile / account was all set to UK, but it deemed the IP address / ISP to be within the United States, hence, corrupting the download.

Googling this issue shows it to be quite common (geolocation data providers). For Plusnet at least, a solution was to turn off the router for 65 minutes and then turn it back on, and hopefully you should get a different IP address assigned (this time for me it was within the range of 88.x.x.x). After I'd done this, all games that previously corrupted on download worked fine. Granted, this may not be a permanent solution, if my router resets / is turned off / on again, there's a chance I may be given a 146.x.x.x address again. At least there's a work around for now.

This solution may not work for everyone, i.e. those in different countries to myself or using different ISPs, but it may help a few others for this annoying issue.

Cheers

In any event this is what wrked for me........


To add a contact into People, from an e-mail in the Mail app.

Zero:

Open the Mail app.

One:

Left click on the email address of someone that you want to add to People.

Two:

Left click on "Open Contact" on the right side of the little dialog that pops up after executing step One:

Three:

The People app should open and you will be looking at the contact information associated with the email address you just left clicked on.

Four:

Left click on the little "save" icon at the top. (It looks like a little diskette.)

Five:

A "Choose a contact" dialog will appear.

Here's what was throwing me for a loop ....

Six:

At the bottom of the "Choose a contact" dialog, left click on the "+". (Plus sign.)

Seven:

A "NEW GMAIL CONTACT" dialog appears.  <== I have a GMail account so that's why is reads "GMAIL".

At this point you may choose to edit the contact information as you see fit.

Eight:

When done editing, (or if you choose to simply accept the contact information as it is.), left click the little save icon (looks like a diskette) in the upper right of the "NEW GMAIL CONTACT" dialog.

Yes, it's a different save icon than referenced in step Four (above.).

Nine:

You are returned to the same screen you saw in step three (above.).

Done !

If you close the above screen, and reopen the People app, it will open with your contact (or contacts if you have more than one) listed.

Or you can start typing the name of the contact you are interested in finding in the "Search" box at the top left of the app.

xbr poseidon say:

Overview

Netgear DGND3700

Login Information:

  • Default Gateway - 192.168.0.1
  • Username – admin
  • Password – password

 

How to Find SSID (Network Name)

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings from the menu on the left.
  4. The network name for your 2.4Ghz network can be found in the top section (2.4Ghz b/g/n) next to Name (SSID).
  5. The network name for your 5Ghz network can be found in the bottom section (5Ghz a/n) next to Name (SSID).

 

How To Find Wireless Passphrase (WPA/WEP)

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings from the menu on the left.
  4. The Wireless Passphrase for your 2.4Ghz network can be found in the lower half of the top section (2.4Ghz b/g/n) next to Passphrase.
  5. The Wireless Passphrase for your 5Ghz network can be found in the lower half of the bottom section (5Ghz a/n) next to Passphrase.

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings from the menu on the left.
  4. From the Security Options section Select the desired security type, None if needed.
  5. Click the Apply button if any changes were made.

 

MAC Filtering

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless Settings under the Advanced section of the menu on the left.
  4. Click the Setup Access List button.
  5. Uncheck the Turn Access Control On box.
  6. Click the Apply button at the bottom of the page.

 

Change Wireless Channel

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless from the menu on the left.
  4. From the drop down menu labeled Channel select the desired channel. (Channels 1,3,9,11)
  5. Click Apply.

 

Change Wireless Mode

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Login to the router by entering 192.168.0.1 in to the address bar.
  3. Enter the username “admin” and the password “password”.
  4. Select Wireless from the menu on the left.
  5. From the Mode drop down select up to “Up to 54 Mbps”.
  6. Click the apply button at the bottom of the screen.

 

Change Wireless Band (Width)

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Wireless from the menu on the left.
  4. From the Mode drop down select “Up to 145Mbps”
  5. Click Apply.

 

DHCP Reservation

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Lan Setup from the menu under the Advanced section on the left side.
  4. Click the Add button under the Address Reservation section.
  5. Enter the Desired IP, the MAC Address of you console and a name for the device in the provided fields.
  6. Click the Add button to save your DHCP Reservation.

 

 UPnP

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select UPnP from under the Advanced section on the left side.
  4. Toggle UPnP on or off by checking or unchecking the Turn UPnP On box.
  5. Click the Apply button at the bottom of the screen.

 

DMZ

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select WAN Setup from under the Advanced section on the left side.
  4. Check the Default DMZ Server box and enter the IP Address of your Xbox console.
  5. Click the Apply button at the bottom of the page.

 

Disable DMZ

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select WAN Setup from under the Advanced section on the left side.
  4. Uncheck the Default DMZ Server box.
  5. Click the Apply button at the bottom of the page.

 

Port Forwarding

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Port Forwarding/Port Triggering under the Content Filtering section.
  4. Click the Add Customer Service button.
  5. Type a Service Name that has something to do with Xbox.
  6. Set the Service Type to TCP/UDP.
  7. Set both the Starting Port and Ending Port to 3074.
  8. Enter the IP Address of your Xbox console in the Server IP Address space.
  9. Click the Apply button at the bottom of the page.

 

Disable Port Forwarding

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Port Forwarding/Port Triggering under the Content Filtering section.
  4. Select the rule that refers to either the IP Address of the Xbox Console and/or contains the Port 3074.
  5. Click the Delete Service button at the bottom of the screen.

 

Firewall

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select WAN Setup from under the Advanced section on the left side.
  4. Set NAT Filtering to Open.
  5. Check the Disable SIP ALG box.
  6. Click the Apply button at the bottom of the screen.

 

How to Find Firmware Version

 

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Router Status from under the Maintenance section.
  4. Make a note of the Firmware Version.

 

Firmware Update

 

Auto Update

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select Router Upgrade from under the Maintenance section.
  4. Click the Check button.
  5. If a new version is available a message stating the old and new versions will be displayed.
  6. Click the Yes button to start the upgrade.

 

Manual Update

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Go to http://www.netgear.com/support
  3. Locate the support page for the DGND3700 and download and save the latest version of firmware.
  4. Login to the router by entering 192.168.0.1 in to the address bar.
  5. Enter the username “admin” and the password “password”.
  6. Select Router Upgrade from under the Maintenance section.
  7. Click the Browse button and locate the file that was downloaded from Netgear’s support site.
  8. Once the file is selected click the Upload button.

 

Restore Factory Defaults

 

Through firmware

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Login to the router by entering 192.168.0.1 in to the address bar.
  3. Enter the username “admin” and the password “password”.
  4. Select Backup Settings from under the Maintenance section.
  5. Click the Erase button at the bottom of the screen.
  6. Confirm the action if needed.

 

Through Hardware

 

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Press and hold both the Wireless On/Off and WPS buttons for six seconds.

 

Known Issues

This device has been known to have connectivity issues in Titanfall and Forza.

Known fixes at this time are to update the firmware and if that does not work then to use manual DNS settings in the Xbox.

 

Multiple Console Support

We have no confirmed cases where multiple consoles were able to obtain an Open NAT state while using this device. Please comment on this page if you have been able obtain Open NAT on two or more consoles using this device.
