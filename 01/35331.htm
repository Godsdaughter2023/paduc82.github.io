Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

Run the activation troubleshooter on the new device using the below steps:
Go to Settings>Update and security>Activation and run the troubleshooter. and select I recently changed hardware on this device.

(Please note that Activation troubleshooter requires you to have digital license linked to Microsoft Account.)

a. user say:

I haven't got a clue as to the so called purpose of Libraries as supposed to accomplish, but that is another discussion.

Libraries made their debut in Windows 7. The idea was that instead of only storing your personal data inside the default user folders, they could be stored on other folders on the local computer and even on remote computers (if those folders were indexed on the remote computers or cached locally as offline files) and all those disparate folders could be presented as a single library.

At first Microsoft thought this was a fabulous idea, and so they pushed everyone to store their files this way by making libraries the default unit of storage and featuring them in Windows Explorer (as it was called at the time.) As I recall, you had to edit the registry to avoid using libraries as the unit of storage.

Then came Windows 8 and Microsoft decided that libraries were a terrible idea. Instead we should be storing our files in the cloud. This continued in Windows 10 - where libraries don't even appear in File Explorer by default - and once again Microsoft is pushing everyone to store their files this way by making OneDrive the default unit of storage and featuring it in File Explorer.
