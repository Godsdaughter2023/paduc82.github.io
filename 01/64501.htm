To view or to remove the Read-only or the System attributes of folders in Windows Vista or Windows 7, follow the steps below:

1.     Click Start, click Run, type cmd, and then press ENTER..

2.     To view the syntax for the Attrib command, type attrib /? at the command prompt.

To remove the Read-only attribute and to set the System attribute, use the following command:

attrib -r +s drive:\<path>\<foldername>

For example, to remove the Read-only attribute and to set the System attribute for the C:\Test folder, use the following command:

attrib -r +s c:\test

Be aware that some programs may not operate correctly with folders for which the System attribute and the Read-only attribute are set. Therefore, use the following command to remove these attributes:

attrib -r -s drive:\<path>\<foldername>

For example, to remove both the Read-only and the System attributes from the C:\Test folder, use the following command:

attrib -r -s c:\test

fixer upper say:

Hi Reddogoh,

There are two main options that I can see in connecting your TV to the PC. One is a hardware solution by getting a TV tuner card, the second would be a software solution using your internet connection (this would need to be a high speed/ broad band solution to be effective) to stream TV to your PC. Each of these types has its own unique advantages.

 

Hardware Solutions (TV Tuner):

 

First there is the direct connection method. The main hardware consideration here is the TV tuner card for your PC. An example of a vendor for this would be here:http://www.hauppauge.com/. The consideration here is to make sure that you have the right connection from your TV to your PC. There are several types of connections that could potentially work from your TV to PC: Coax, RCA, S-Video and VGA are some of the more common. One of the more interesting connection types on the site mentioned would be a device that can converts a Coax cable signal from the TV into a USB device that can then be plugged into your computer. One of the advantages of a hardware TV connection is the ability to record live TV, and software for this capability is often included with the TV tuner. Can you use wireless? It may be possible as a type of adapter, but mainly the connections are direct.

 

Software Solutions (Streaming and on demand TV over Broadband):

 

Software solutions can be quite economical as it piggy backs on your existing broadband connection. Another advantage would be that the shows can be streamed on demand.

Prices can be as low as free, an example beingwww.hulu.com where you can stream full episodes of TV shows through a license agreement between the content providers and Hulu.

 

An example of a subscription based model would be  http://www.yourfreetelevision.com/ which is around $30 Monthly.

 

So there are some options and this should definitely point you in the right direction.

These accounts should be left alone as they are part of the operating system structure.

You particularly don't want only one user account with administrative privileges on Windows 7 because the built-in Administrator account (normally only used in emergencies) is disabled by default. If you're running as Administrator for your daily work and that account gets corrupted, things will be Difficult. It isn't impossible to activate the built-in Administrator to rescue things, but it will require third-party tools and working outside the operating system.

The user account that is for your daily work should be a Standard user, with the extra administrative user (call it something like "CompAdmin" or "Tech" or the like) only there for elevation purposes. After you create "CompAdmin", log into it and change your regular user account to Standard. Then log back into your regular account.

If you want to go directly to the Desktop and skip the Welcome Screen with the icons of user accounts, you can do this:

Start Orb>Search box>type: netplwiz [enter]
Click on Continue (or supply an administrator's password) when prompted by UAC

Uncheck the option "Users must enter a user name and password to use this computer". Select a user account to automatically log on by clicking on the desired account to highlight it and then hit OK. Enter the correct password for that user account (if there is one) when prompted. Leave it blank if there is no password (null).
