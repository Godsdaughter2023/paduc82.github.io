samc1 say:

It means [most likely] that the site you're downloading from doesn't output the correct file stream for your computer to save the downloaded file correctly. When files like PDF are transmitted by HTTP server, the file is broken up into "chunks" with the necessary information sent along (there is a well defined protocol for this) so that the client side can reconstitute the chunks into a whole file as it appears at the server. It is usually unlikely for this to go wrong if the job is left to the server to transmit an existing file. If, however, the PDF is generated programmatically on the fly, things can go wrong quite easily (bugs in the script program, say, not sending the necessary file delimiter, etc) and this can confuse the client browser. If you downloaded from another site without any problem, it is unlikely for your browser to be at fault and most likely the website server is causing the problem as highlighted. Hope this helps.
