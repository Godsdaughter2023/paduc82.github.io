lemp say:
The [NONE] that you are referring to means that there are no file extensions associated with things such as Audio CDs, Drives, and DVDs. 

A "file extension" is the 3- (sometimes 4-) character part of a file name that comes after the period.  For example, in "Document.doc" the file extension is doc.

The settings you made were -- as you correctly noted -- are in the Autoplay dialog.  These settings determine what Windows does when a removable storage device is attached.  Depending on the type of multimedia content detected on the device, you can select from a variety of "Autoplay handlers" to deal with that content. 

"Take no action" is not a bad default.

If you want to change your mind, open My Computer and right-click on the icon for a CD/DVD/USB device and select the "Autoplay" tab.  You can select from a variety of "content types" and then pick which application should open.  For more flexibility, get TweakUI --> http://windows.microsoft.com/en-US/windows/downloads/windows-xp and/or see
