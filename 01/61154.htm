    how to create a screen saver - Microsoft Community

    rosereikard say:

    how to create a screen saver

    will windows allow screen savers ?

    davem121 say:
    Hi rosereikard, it depends on your current version of Windows 10

    Open the Settings App
    Go to Personalisation - Lockscreen
    Scroll down to the bottom of that Settings Page
    Do you see 'Screen Saver Settings', if so click that . . .



    If not, then, this is the only way to enable a screensaver:

    Open a Command Prompt as Administrator

    Run this command to list the available screensavers on your system:

    dir c:\windows\system32\*scr /b

    Take note of the Screensaver you want to use


    Run this command to clear screensaver

    reg delete "HKEY_CURRENT_USER\Control Panel\Desktop" /v SCRNSAVE.EXE /f


    Run this command to set your desired screensaver (replace Bubbles with your chosen Screensaver file Name):

    reg add "HKEY_CURRENT_USER\Control Panel\Desktop" /v SCRNSAVE.EXE /t REG_SZ /d C:\Windows\system32\Bubbles.scr /f
