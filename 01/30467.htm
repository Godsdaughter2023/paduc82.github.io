xircal say:

Hi Andrew,

I just checked it myself and see that Microsoft has removed the key in 8.1

No problem though and here's how to create the key yourself.

As a precaution though, create a manual System Restore point. In the event you accidentally delete something, you can revert to that Restore point which will reinstate the original Registry settings.

In case you don't know how to do that, here are instructions.

  1. Right click the Start button and go to System.
  2. Click the link to System Protection on the left near the top and then click the Create button at the bottom of the menu.
  3. Type a name, something you can remember easily i.e. Before editing Registry and then click Create.
  4. Windows will create a Restore point for you after which you can close the menu.
  5. In the event you accidentally delete something in the Registry, go to the same menu and click the System Restore button. You can restore to the state the Registry is in right now from there.

So now we go about creating the key to disable the Charms Bar.

First of all navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\ImmersiveShell\EdgeUI

Next, in the right hand pane, right click and go to New | DWORD (32-bit). Here's a screenshot to show you where you should be (indicated with red arrow at the bottom) and what the menu looks like when you right click in the pane.

Next, type the description for the key as DisableCharmsHint. Here's another screenshot to show that. If you accidentally lose the highlight, right click it and "Rename", or just delete it and create another one.

The next step is the change the value in the Data column shown as (0) preceeded by a number of zeros to (1).  So right click the key you just created and choose Modify as shown in the next screenshot.

In the next menu, change the "0" to a "1" as shown next and then click OK.


You'll see the value change to (1) now as can be seen next.

Now close the Registry editor and reboot. When the machine reboots to the desktop, the Charms Bar won't appear anymore.

If you want the Charms Bar back again, reverse the procedure and change the "1" to a "0" again.

EDIT: After making the change, you can still open the Charms Bar by hitting Windows key + C

E. Create shares as desired. In Windows 7 I usually share out the user's Desktop and the Public directory.

    The new wlan network cards for Windows 10 do not support the hosted network because Windows has provided the mobile hotspot feature in its place. So network adapter manufacturers no longer provide drivers to support the hosted network.
    You can still use the network hosted in Windows 10 on those old network cards where there is the appropriate driver for this feature.Read here: https://www.intel.com/content/www/us/en/support/articles/000023257/network-and-i-o/wireless-networking.html


    Try changing the driver manually:

    1) Go to Device Manager> Network Cards, select the card in use, choose "Update Driver"
      2) Select "Search for driver software on your computer"
      3) Select "Choose from a list of drivers available on your computer"
    4. Two or more drivers will be listed, select a different driver from the one currently used on your computer, then install it.
    If you do not resolve, try to contact the manufacturer of your network card and ask if they can suggest an appropriate driver that supports the hosted network.
    Otherwise you have to opt for the mobile hotspot.
