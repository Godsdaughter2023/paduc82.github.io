junaid_a say:

Hi Aman,

Thank you for writing to Microsoft Community Forums.

Resizing functionality is no longer available with the newer versions of Touch keyboard. However you can restore the previous functionality by doing any of the following:

Method 1

This method is about disabling the new keyboard experience.

Note:

Serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. To backup and restore registry key, refer to this link.

  • Open Notepad and copy paste the following script:

    reg add HKLM\SOFTWARE\Microsoft\TabletTip\1.7 /v DisableNewKeyboardExperience /t REG_Dword /d 1 /f

  • Save the file as ‘.reg’ on your Desktop.

    Note: Do not use the quotation

  • Close Notepad, right click the ‘.reg’ file you just saved and Merge.

  • Restart the machine and see if you have an option to resize.

If Method 1 doesn’t help, try Method 2.

Method 2

Let’s replace the existing Touch keyboard files with the older ones. We need a computer running an earlier build of Windows 10, compared to yours, to extract the files from.

  • On your computer, open File Explorer and navigate to the following location:

    C:\Program Files\Common Files\microsoft shared\ink

  • Backup (copy) TabTip.exe and tipskins.dll and save them in some other location.

Note:

On the files in the folder mentioned above (i.e.,TabTip.exe and tipskns.dll), open Security Permissions and change Owner of files from TrustedInstaller to your local user account). Close and re-open the permissions window and now Change Permission for Administrators Group or add permission for your local account, giving Full permission to your Group or User.

  • Right click the Windows icon on the taskbar, choose Task Manager and end the TabTip.exe process.

  • From a computer that is running an earlier build of Windows 10, copy the TabTip.exe and tipskins.dll files and replace them on your machine in the same location.

  • Run TabTip.exe and see if you can resize.

If the issue persists, I would suggest you to post your query in the IT Pro TechNet Forums, where we have support professionals well equipped with the knowledge on Windows 10 Enterprise edition to help you.

cmdrkeene say:

The Mail app doesn't store any data, ever.  It only connects to the server and shows you what it sees there.  It's like "mail viewer" (like a cell phone email app).  If you deleted it from the server, it would disappear from the Mail app also: you'd be deleting the source of the data.

You have a few options.

  1. You can setup your email to go through the Outlook.com online email client.  Many people think of this like the replacement to hotmail, but it really works like a rich email program (like Outlook) in that it can download mail from other servers (and remove the mail after it does).  Then it can supply that to the Mail app on your computer.  I wrote about this in the context of getting email from POP servers, but it can apply to your situation also.  Check out how in my article here:  http://cmdrkeene.com/use-pop-email-windows-8


  2. You might know that Windows Vista was the last Microsoft OS to come with a desktop email program.  Windows 7 and 8 do not come with any successor to the Windows Vista Mail program.  Instead, you can get it as an optional download from the Windows Essentials page.  Windows 8 does have a bare-bones basic email app that you mentioned, but it's designed for tablets/phones and light use.  If you want something that compares to the Vista mail program (such as pulling emails down and archiving them locally, then deleting them from the server), you're going to want to get the free Essentials download at that link.

My personal recommendation is #1, because it means that your mail is not stored locally. If your computer crashes, gets lost, stolen, wet, or broken, your emails won't be lost.  Option #2 would mean your emails were only on your computer, and subject to loss if something happened to the computer. So while #1 takes a little while to setup, that's what I'd pick.

However if you have a backup strategy (or don't care to have one), and you just want an email program that does what you did before Windows 8, then option #2 is your pick.

cmdrkeene say:

The Mail app is not the "primary store" of data then.  It's not a storage, it's a cache.  I know that sounds like a silly distinction, but it's not.  The data is duplicated for offline viewing, like your Temporary Internet Cache files.  But the Mail app only keep that until the next time it connects to the server and re-synchronizes.

If you delete data from your server, it will delete it from the Mail app as well.  In this way, you can see that the Mail app is not storing data, it's only reading the online source material and caching it temporarily.

If you need a way to actually download, store and archive emails (or a way to delete it from the server without deleting it from the local copy), then you need to use a different program.  Microsoft makes Outlook ($$$) and Windows Essentials (free), both of which can do this for you.

frederik long say:

I wouldn't touch it with a barge pole.

If you like to take risks then you could do this:

  1. Get out your stop watch and measure the time it takes for your machine to boot up.
  2. Create an image of your installation.
  3. Download and install CCleaner.
  4. Repeat the stop watch measurements.

You will most likely find that CCleaner makes a large number of claims about things that "slow down" your PC but that you are unable to actually measure any improvement in speed. At worst it will wreck your installation.

Note also that having a full disk will not slow down a PC, contrary to popular opinion. However, if you have less than 20% free space then you will cause into excessive file fragmentation which will slow things down. Furthermore, Windows will find it difficult to function when the disk is almost full.
