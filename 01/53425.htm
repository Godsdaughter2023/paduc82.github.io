Hi

 

The built-in option to change the Desktop Fonts and Colors was removed in Windows, starting with Windows Vista.

 

The reason is that any font color would become invisible, depending on the Desktop Wallpaper that was being used. An example would be a picture or multi-color wallpaper. If the font is a dark color and the text was positioned over a dark part of the wallpaper.

 

Windows uses a White font with a Drop Shadow effect. This keeps the font visible, regardless of the background color of the wallpaper. In extreme cases where the desktop is a brilliant white, or other very light color, the fonts will turn Black.
