Microsoft Technet: rtheb - posted 2 Sept /09:
SOLVED THE PROBLEM!

The Error was caused by having the "Norton User Account Control tool" in Vista SP2 before the Win 7 upgrade.

Norton UAC needs to be uninstalled from Vista before beginning the upgrade.

If you do the upgrade with the Norton User Account Control tool still installed you'll get the message:

"C:\Windows\system32\*.msc

Windows cannot verify the digital signature for this file.  A recent hardware or software change might have installed a file that is signed incorrectly or damaged, or that might be malicious software from an unknown source."


If you do end up with Norton User Account Control tool still installed in Win 7 then do a Safe Mode Boot, and then go to Control Panel-Programs and Features and uninstall Norton User Account Control tool.

This should be added to the compatibility pre-check for Win 7.

Hi Enon. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

1) There is a new guided walk-through from Microsoft for repairing Windows Update that leads up to resetting components if necessary: https://support.microsoft.com/en-us/help/10164/.... Try running that first.

Open administrative Command Prompt by typing CMD in Start Search, right click result to Run as Administrator, copy and paste or type the following commands one-by-one followed by Enter key.

net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
Ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
Ren C:\Windows\System32\catroot2 Catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver

Close Command Prompt and see if Windows Updates works then

3) You can also install the Updates manually that fail to install which are logged at Settings > Update & Security > Windows Update > Installed Update History, and then search for those to download and install from this Catalog: http://test.catalog.update.microsoft.com/v7/sit...

Is the faulty computer similar to yours?

On a spare working computer perform these steps (skip any if you have already have done them)

1. Visit https://www.microsoft.com/en-us/software-download/windows10
2. Click [Download tool now] & save MediaCreationTool.exe
3. Click File explorer, Downloads, Double-click and run MediaCreationTool.exe
4. Click [Accept], Choose second option (o) Create installation media for another PC, [Next]
5. Untick [/] Use the recommended options for this PC if incorrect
and change Architecture to 64-bit or 32-bit or leave all settings if correct.
6. Click [next] and it will download windows ISO, save in Downloads.
7. Visit rufus https://rufus.akeo.ie/
8. Click Rufus 2.10 portable - https://rufus.akeo.ie/downloads/rufus-2.10p.exe
9. Save rufus-2.10p.exe
10. Click File explorer, Downloads, Double-click and run rufus-2.10p.exe
11. Click [no] to check for updates
12.Insert your USB you want to create a bootable windows 10
13. Click the picture of a CD in Rufus and navigate to downloads
14. Locate windows10.iso file and click open
15. Click [start] to create a windows 10 bootable USB.
16. When complete remove the USB and attach it to the faulty computer
17. Power on the laptop and press F2 to get into the BIOS.
18. Change the boot order to boot from USB, press [F10] to save and boot.
20. On the windows initial install screen, click [next]
21. Click - repair this computer
22. Follow the prompts to try and repair your windows system.
23. Hopefully it repairs windows, remove the USB and reboot.
