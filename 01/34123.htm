[david} say:

Using safe rollback option, tried getting rid of Windows 10. Now stuck in a boot loop. Hp logo appears, windows announces it is trying to recover the previous version and then reboots.

Any ideas? 

Surprisingly, I was able to resolve this issue by accessing the command prompt from the Windows 10 installation USB (select "Command Prompt" in "Advanced options") and entering the following commands (I found this from https://support.microsoft.com/en-us/kb/927392):

bcdedit /export C:\BCD_Backup
c:
cd boot
attrib bcd -s -h -r
ren c:\boot\bcd bcd.old
bootrec /RebuildBcd

After doing this and restarting, Windows 8.1 will load, and at the login screen, you will see "Configuring Windows Update" and "Cleaning up".
