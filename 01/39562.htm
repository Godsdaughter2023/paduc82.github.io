4 (optional): The type of .DMP files located in the Minidump folder are known as Small Memory Dumps. In %systemroot% there will be what is known as a Kernel-Dump (if your system is set to generate). The difference between Small Memory Dumps and Kernel-Dumps in the simplest definition is a Kernel-Dump contains much more information at the time of the crash, therefore allowing further debugging of your issue. If your upload speed permits it, and you aren't going against any strict bandwidth and/or usage caps, etc, the Kernel-Dump is the best choice. sumit (volunteer moderator) say:

Hi,

What you need to do here first of all is to change the way you receive updates. With Automatic Updates enabled, the operating system will reinstall the nagware prompt which is KB3035583 after you uninstall it which will be a step we'll take in a minute.

So proceed as follows:

  1. Right click the Action Center icon on the taskbar and go to "Windows Update".
  2. In the left hand menu, click "Change settings".
  3. Change it from Automatic Updates to "Download updates but let me choose whether to install them".
  4. Click "Save Changes" and then click "Control Panel Home". 
  5. Go to "Programs and Features" and click the link to "View installed updates".
  6. Copy/paste KB3035583 into the Search field top right and when it appears below, right click it and select "Uninstall".This is a large installation so uninstalling it can take several minutes.
  7. Reboot next and then go back to Windows Update again.
  8. Since you now have different WU settings, click the link if there's any which are awaiting your approval which will take you to the menu where you can decide what to install.
  9. If you see KB3035583 in the list, right click it and select "Hide update". The box you normally have to checkmark to select an update will turn grey and when you exit WU, those files will be sent to the repository called "Restore hidden updates".You can restore any updates which have been hidden in there at any time by clicking the link, making your selection and then selecting "Restore".

The next step is to make a registry change which will prevent Windows 10 from downloading and installing.

Hit Windows key + R to launch the Run command, type: regedit and click OK.

Navigate to: HKEY_LOCAL_MACHINE --> SOFTWARE --> Microsoft --> Windows -->   CurrentVersion --> WindowsUpdate --> Auto Update --> OSUpgrade

Click the OSUpgrade key (which looks like a folder) and then in the right hand pane you should be able to see four values called respectively:

AllowOSUpgrade

KickoffDownload

KickoffSource

OSUpgradeInteractive

Some or all of these will have a value of 1. You need to change those to a 0 like you see in the screenshot.

To change a registry value, right click it and go to Modify. In the Value field at the bottom, change the figure 1 into a 0

Click OK and Windows will add the 0x00000000 (0) itself.

If it's already a 0 then you don't have to touch it.

In addition, you need to create a value. To do that, right click a blank part of the right hand pane and choose New --> DWORD with the name DisableOSUpgrade.  Windows will add REG_ to the beginning of DWORD after you create it.

If you inadvertently lose the highlight when you create the new value, right click it and select Rename.

For issue #1 -  How to include captions in Photo Gallery slideshows?

Currently Photo Gallery’s slideshow doesn’t support showing captions or other metadata. The feature you are inquiring about is really a great idea and would be a great feature to have. I will submit this request to our product team. You may also post this as a request in our feedback page at https://feedback.live.com/.

For the mean time you can create a customize slideshow using Windows Live Movie Maker, It will allow you to automatically include the captions presents in the photos.

Here are the steps on how to convert your photos into a movie.

Note: Please make sure that you added the captions you want to add in your photo before you share them to Movie Maker.

  1. Click Start button. In the search box, type Photo Gallery, and then, in the list of results, click Windows Live Photo Gallery.
  2. In Photo Gallery, select the videos and photos that you want to use in your movie. To select multiple photos and videos, select the check box in the upper-left corner for each photo and video you want.
  3. On the Create tab, in the Share group, click Movie.
  4. Movie Maker opens automatically and the selected photos and videos are added to the storyboard.
  5.  In the Add Music dialog box, navigate to the folder that contains the audio files. 
  6. 6. Select the music files you want to add to Windows Live Movie Maker. Click Open. 
  7.  Verify whether the selected file is added to the storyboard pane. Save the project.

You can also add music to your photos. On the Home tab, click Add music, and then select any one of the following options:Click Add music to add the music file to all the media elements added in the storyboard pane. Click Add music at the current point to add the music file from the current point in the storyboard pane.

 

To learn more about Movie Maker please visit http://explore.live.com/windows-live-movie-maker-help-center

 

For issue #2 - How to add text along with the photo when I print or share it?

Printing from Photo Gallery doesn’t support the ability to add caption directly on the photo. If you want the text to be permanently visible along with the photo you can use editing software like Microsoft Paint and once you are done you just need to save the edited photo and open it in Photo Gallery.

Important note: By adding text over the photo you’ll be permanently modifying it so we recommend that you make a copy of the photo first in order to preserve it without the text as removing the text later on may not be possible or may require losing part of the photo by cropping it.

Below are the steps on how to use Microsoft Paint to add text over images

  1. Launch Microsoft Paint. Open Paint by clicking the Start button, click All Programs, click Accessories, and then click Paint.
  2. Click the Paint button, and then click Open.
  3. Find the picture you want to open in Paint, click it, and then click Open.
  4. Use the Text tool to enter text in the picture.
  5. On the Home tab, in the Tools group, click the Text tool.
  6. Drag the pointer in the drawing area where you want to add text.
  7. Under Text Tools, on the Text tab, in the Font group, click the font face, size, and style.
  8. Type the text that you want to add. Click Save to save the photo.
  9. Open the picture in Photo Gallery and the text will now be included as part of your photo and will appear even you share your photo as slideshow or you send it via email, share it in SkyDrive, Facebook and any other sites. It will also appear even when you print your photo.

For further assistance on how to use Microsoft Paint in editing photos, please visit http://windows.microsoft.com/en-US/windows7/Using-Paint
