 Windows 7 or Vista Updates Fail To Install.

 If you find that a particular Windows Update does not install, here are some suggestions:

 First clear up your Temp File/Cache, Reboot and try again. Best and easier to use Ccleaner < Download the basic version via  <http://www.ccleaner.com/download/builds.aspx> The basic version does not contain the disgusted Toolbar>.

Disable your Firewall and try !

In many cases, the update may fail to install in the first instance, but for some inexplicable reason, succeeds in the 2nd or 3rd try. So try a couple of times.

 

It it doesn't help, search on Microsoft, for the patch / update KB number, and download its Stand-alone Installer. Now apply the patch manually. Search for just the number; don't include KB !

 

If it still doesn't help, try this : Rt Click Computer > Manage > Services > Stop the "Cryptographic Service" and rename the "Catroot2" folder situated in C:\Windows\System32\Catroot to something like say Catroot2X. Reboot. Now run the update/hotfix installers again. This action will re-generate a new Catroot2 folder and the updates/hotfixes, should hopefully install OK.

 

If still you have a problem then, go to C:\Windows\WindowsUpdate.log and look for the most recent entry. This will be present towards the end of the log. Any failed updates will have error code/s written next to them. Note them down. If you find just too many entries too confusing delete this WindowsUpdate.log and try installing the updates again. Now open the newly re-created WindowsUpdate log file and have a look at its contents. The warnings will probably appear as -:WARNING: Failed to find updates with error code XXXXXXXX.

Now Rt-click Computer > Manage > Event Viewer > Applications and Service Logs > Microsoft > Windows > WindowsUpdateClient > Operational . Check out for anyCritical Message or WarningFATAL.

considerate_guy say:

As noted in those other threads, Microsoft dropped support for Windows DVD Maker as of Windows 7. It isn't in the Windows Store so there isn't a way to use it on Windows 10.

The latest Movie Maker 2012 saves in .mp4 format by default, but Windows DVD Maker doesn't understand .mp4 format. Instead, you need to change the movie format to Windows Media Video (.wmv). Then you could copy that to a USB drive and use Windows DVD Maker on your Vista machine.

To save in WMV format, when you are choosing the file location and file name of your file, there is also a drop-down in the lower right where you can pick the file format. Change it from H.264 (.mp4) to Windows Media Video (.wmv).

The DVD format is now considered a "legacy" format by Apple and Microsoft. Since it was created in the era of analog TVs, it has a maximum resolution of 480p (NTSC). That means it isn't an ideal format for HD video which will be down-rezzed to the standard-definition format of a video DVD.

If you have a Blu-ray player (or a modern video game console which usually has Blu-ray capability) then you could just put the movie file that Movie Maker creates directly onto an optical disc (even a DVD), but just as a data file. Then that will play in the Blu-ray player in HD quality.

Editing your movie and the authoring of a DVD are treated as separate steps.

You can most definitely still install and use Movie Maker 2012 for the editing step. It will install on versions of Windows 7 and higher (including Windows 10).

As for the DVD authoring step, Microsoft's solution (Windows DVD Maker) is no longer produced. It came with Windows Vista and Windows 7 but isn't available for the later versions.

  • DVD Flick
