Method 1: Run the Windows troubleshooter.

Once you run this tool, you'll be able to see exactly what's causing the malfunction of the Bluetooth device. To run the tool, follow the steps below:

  1. Click the Start button, then select Settings > Update & security > Troubleshoot.
  2. Under Find and fix other problems, select Bluetooth > Run the troubleshooter.
  3. Once the troubleshooter is already running, it will detect and fix the Bluetooth issue automatically. Follow the on-screen instructions and wait for the process of troubleshooting to complete.

Method 2: Enabling the Bluetooth Support Service.

To enable this service, you may need to have the Administrator privilege to proceed using the Bluetooth device bugs troubleshooter. Follow the steps below:

  1. Press the Windows key + R to open the Run box.
  2. Enter services.msc, then press Enter.
  3. Locate the Bluetooth Support service on the Service dialog and right-click it to select Restart.
  4. Double-click the Bluetooth Support service. Change the Startup type to Automatic under the General tab.
  5. Click Apply and OK to save and then exit the service dialog.
