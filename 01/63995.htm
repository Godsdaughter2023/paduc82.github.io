itologist say:
Very simple.

All Windows 10 backup files are stored in a folder called "FileHistory" on a drive that was selected during the backup setup.

There are two ways to delete backup files.

1.  Lower the backup file retention period to whatever it is set. The default is "Forever". You can also change this setting to "Until space is needed".

2. Turn off the Windows backup all together. Go to Setting > Update & Security > Backup

    Turn off the backup.

 Now go to the backup drive and delete the "FileHistory" folder.

 If the folder is locked and cannot be deleted then turn the backup ON again and click on "more options".

 Click on "Stop using drive"

_________________________________________________________________________________________

___________________________________________________________________________________________________

Now try to delete the "FileHistory" folder again.

Windows Key+X (Or right click on Start)

Restart your computer

Go to the website of the audio driver's manufacturer or computer's manufacturer
See if some driver is released for your model and OS

In previous Option, you click Update Driver Software, in place of this, click Uninstall
Install the downloaded driver
Restart your computer

Windows Key+X
Click Device Manager
Expand Sound, video and game controllers
Right click on your audio adpater
Click Properties
Go to Driver tab

Hi fgfleischer,

Network discovery is off by default for Public networks. Changing a network location type to Private is another way to turn on network discovery than turning it on manually.

As a first troubleshooting step, I would suggest you to run the Network Diagnostic Feature in Windows Vista and then try to check if you’re able to turn on network discovery.
Using Network Diagnostics in Windows Vista
http://windows.microsoft.com/en-US/windows-vista/The-doctor-is-in-Using-Network-Diagnostics-in-Windows-Vista

By the way, are you able to ping to the computers on the network without any problem.

I would suggest you to disable your Anti-virus software completely and then try enabling network discovery to check if you’re able to turn it on.
Note: Do not forget to re-enable your security software.

You can also try adding an exception in windows firewall to see if that helps.
1. Click on Start; in the start search type Windows Firewall and hit Enter.
2. Click on Change setttings.
3. Click on exception tab and then click network discovery.

Now try turning ON network discovery to see if that helps.
Post your results back.
