Conexant Audio Message Service is part of the device driver for the Conexant High Definition Audio that comes preinstalled with OEM PCs. It is started as a Windows Service called 'Conexant Audio Message Service' with the name 'CxAudMsg' and described as “Monitors audio device events and forward them to subscribing application. Conexant Audio Message Service runs as a program that can be started by the Service Controller and that obeys the service control protocol. This type of service runs in a process by itself. If this service is stopped, the audio effects will not function properly.

We are looking into this symptom as potentially linked to the latest driver. You may try to update the drivers and check if it helps.
