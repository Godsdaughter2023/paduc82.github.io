Sometimes reseating the video cabling will resolve this problem. If it's a desktop unit, you may also try shutting down and reseating the video card.

Built in Windows critical system services include csrss.exe, wininit.exe, logonui.exe, smss.exe, services.exe, conhost.exe, and winlogon.exe.

This issue occurs when the hibernation file for your virtual machine gets corrupted. For Windows Virtual PC (the version that comes with Windows 7), the default action for close is Hibernate. Hiberfil.sys is a snap-shot of the state of your computer that is taken by windows when it becomes inactive. However, virtual machines don't store their hibernation files in hiberfil.sys like a physical Windows system does. The hibernation file for a virtual machine is referred as the storage of the current state of the virtual machine. So the hibernation file for Windows Virtual Machine is stored in the following location which has a file extension*.vsv

a. Click on Start and click on computer.

b. Go to the following location:

C:\Users\ (Username)\ AppData\Local\Microsoft\Windows Virtual PC\Virtual Machines directory

c. You’ll find a .VSV file in the Virtual Machines Directory.

d. Rename the file. 

 

e. Close all the windows and restart Virtual Machine.
