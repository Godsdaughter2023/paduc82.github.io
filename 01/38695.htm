Hi,

Try running the 8.1 Fixit first of all to reset the Windows Update components: https://support.microsoft.com/en-us/kb/971058

If that doesn't help, it might be due to file corruption. To check, proceed as follows.

  1. Hit Windows key + X and click "Command Prompt (Admin)".
  2. Type (or copy/paste) sfc /scannow and hit Enter.

System File Checker will check Windows system files for corruption and try to fix any found.

On occasions though it can fail in which case you'll see the message: "Windows resource protection found corrupt files but was unable to fix some of them".

If you see that message, copy and then right click and paste the following command into the same command prompt and hit Enter:

DISM.exe/Online /Cleanup-image /Restorehealth

When DISM runs all you'll see most of the time is a plain black screen. But eventually the figure 20% will appear followed by 100% about 15 minutes later. You'll also get a report of what was fixed.

If you see any error messages, make a note of them and post those here please.

ruwim.b say:

Only media files with preview are saved in C:\Users\(username)\AppData\Roaming\Skype\(skype_id)\media_messaging\media_cache_v3/ folder.

These are such files as image files (.jpg, .png), video messages (.mp4), GIF files sent from Skype for Web (.gif) and URL links with preview. The last (URL links) will be shown in the \media_cache folder as the example you have provided in your post (^FEF2B988410E4DD718F310F0C39450B3A5BCB3435463CCFB9F^tfile_urlpv) and will not have any extension. These ...tfile_urlpv files are only used by Skype and cannot be opened by any 3rd party application.

Other media files will have an appropriate extension and you should be able to open them with an application associated with the particular extension.

In the case of image files (.jpg, .png), you can right click on the preview image and select Save as to save the file to a selected folder.

Hi,

I'm a techie who plays console games and PC games, indeed, I built my own gaming PC.  When your talking about textures and pop-in, there are several things in play that can cause this.  

1. Graphics and memory bandwidth.

2. Hard drive read speed.

3. Changes to the graphics engine by the developer to reach a specific performance target.

The Xbox One X maybe 'The worlds most powerful console' but to reach the dizzying heights of full 4K, sacrifices need to be made to reach a designated frame-rate target. Unless you're seeing massive texture and object pop-in, this isn't a fault with your console, it's how the game is meant to perform.
