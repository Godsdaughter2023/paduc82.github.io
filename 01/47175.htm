There are several ways.

2. 3. In Windows Explorer, you can go to C:\Users\<username>\Recent and view the same folder you accessed quicker using method 2 above.

I hope this helps.

Start>Run>msconfig [enter]

john rubdy say:

Hi,

 The cause of this error is that the version of wmp.dll in the registry is out of sync with what is installed. The below steps should resolve the issue you’re encountering:
 

Method 1.
Update the Windows Media Player Version
a. Ensure that Windows Media Player is closed before proceeding.
b. Click Start, click All Programs, and click Accessories.
c. Right-click Command Prompt and select Run as Administrator.
d. Click Continue or provide Administrator credentials if prompted.
e. In the Command Prompt type the following and press Enter:
        UNREGMP2.EXE /UPDATEWMPVERSION
 

Method 2.

If this does not help you can re register the wmp.dll file
a. Log on as an administrator.
b. Click Start, type cmd in the Start Search text box, right-click cmd.exe, click Run as administrator and then click Continue.
c. At the command prompt, type regsvr32 wmp.dll, and then click OK.

 

Method 3.

If the issue still exists disable and re-enable the Windows Media Player.

Here is how:
Step 1.
Disabling the Windows Media Player:
a. Go to Start and in the search type "Turn Windows features On or Off".
b. Click on "Turn Windows features On or Off".
c. Browse to the Media Features and uncheck the mark in front of Windows Media Player.
d. Restart the computer
 
Step 2.
Re-enable the Windows Media Player:
a. Go to Start and in the search type "Turn Windows features On or Off".
b. Click on "Turn Windows features On or Off".
c. Browse to the Media Features and place a check mark in front of Windows Media Player.
d. Restart the Computer.
