lemp say:

By "Windows edition" I meant to ask whether you had Windows 7 Home, Pro, Ultimate, etc.  If you have Windows 7 Pro or better, you can use the Group Policy editor rather than regedit to make the change you're trying to make.

I assume that what you mean by "the changes are made with a standard account that has been given temporary admin rights" is that you right-click on regedit.exe, select "Run as administrator" and respond as needed to the UAC prompt. 

If that's what you're doing, the problem is there is a HKEY_Current_User registry hive for each user, and when you "Run as administrator and then give the credentials of an administrator account, you are indeed "running as" that user.  Thus, you make changes to that user's HKEY_Current_User registry hive. 

See if doing things in the following way works better. 

1.  Log in to an account with administrator permissions; make sure that you are NOT logged in to the user account that you want to change.

2.  Create a manual System Restore point so you can get back to a stable working system if you make a mistake editing the registry.

3.  Open Windows Explorer, click on Tools > Folder Options > View, make the following changes, and click OK

         Select the radio button to "Show hidden files, folders, and drives"

         Clear (uncheck) the box to "Hide protected operating system files (recommended)"

4.  Open regedit.  You'll probably need to respond to a UAC prompt even though you're in an administrator account.

5.  Select (click on to highlight) HKEY_USERS and then go to File > Load Hive

6.  In the "Load Hive" dialog, navigate to C:\Users\<username_of_the_standard_user_account_you_want_to_change> and select NTUSER.DAT then click "Open"

7.  Enter a key name.  You can use anything you want here, for example, the name of the other account

8.  Navigate in this hive (the one you just loaded) to Software\Microsoft\Windows\CurrentVersion\Policies\Explorer and make the required change

9.  Select the hive you had loaded (i.e., click on the name you picked in Step 7) and go to File > Unload Hive (do not forget this step!).

10.  Close the registry editor

11.  Undo the changes you made in Windows Explorer in Step 3.

12.  Log on to the standard account and confirm that the registry edit you made is there.

This Windows 8 Pro CD, one each of 32-bit and 64-bit, are in the original packaging with the product key just as I purchased it.   The Microsoft COA is on the box.  This copy of Windows 8 Pro is not installed on any computer.  I no longer own the laptop I tried it out with, but it was uninstalled from that computer.  Am I ok to sell this item?  Thank you for your help!!


Yes, you can, but make sure you inform the person you are selling it to that it will need to be reactivated by telephone because it was previously in use.

 

1. Press Windows key + X then clickRun, then type: slui.exe 4

2. Next press the 'ENTER' key

3. Select your 'Country' from the list.

4. Choose the 'Phone Activation' option.

5. Stay on the phone (do not select/press any options) and wait for a person to help you with activation.

6. Explain your problem clearly to the support person.
