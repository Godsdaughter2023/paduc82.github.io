bill smithers say:

You can't for many reasons.

A vista Key is not for installing Windows 7.

If you are trying to transfer a Key;

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

aakkam22 say:

Try pressing the Num Lock key to turn it on?

If you want Num Lock to be on during startup, follow the instructions below to edit the Registry:

First, be careful as one small mistake inside the Registry can cause serious system problems.

1. Open the registry (Open Start menu, type regedit and hit Enter)

2. In the left, navigate to and select the following key:

HKEY_CURRENT_USER\Control Panel\Keyboard

3. On the right, right-click and modify the string InitialKeyboardIndicators

4. Clear any existing info, and then type 2 for Value Data

5. Click OK, close the Registry, and reboot.

The Num Lock key should stay on now.

If it doesn't, then you may need to change some settings in the BIOS to allow it to stay on.
