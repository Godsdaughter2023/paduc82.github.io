Now, with all that, will reopening File Explorer show the floppy disc A:\ drive?

NO!"

Link: https://www.bleepingcomputer.com/forums/t/607324/floppy-disk-drive-not-appear-in-file-explorer-when-plugged-in/

Strange behavior by Windows10 I think, but this way is a bit easier than install (or even start-up) a virtual machine with e.g. WinXP to reach files on floppy disks! Thanks "cafejose"!

(4) At a later point I visit the Control Panel Fonts screen and NONE OF THE FONTS ARE VISIBLE THERE! All I can see are 11 of the standard Microsoft fonts (and they all happen to be Raster fonts!)

davidapeterson say: I resolved the problem, here's how.

As Ross Jaburg suggested (thanks!), one must hand delete any font files they no longer want, if those fonts were installed using the shortcut method. The fonts still remain in the registry, however, so I used CCLEANER (a 3rd-party utility) to remove those broken font entries.

Once I did that, I could "magically" see the original fonts that came with Windows 7 in the control panel Font view.

Something is wonky with the font handling of shortcut-installed fonts, so I guess I'd have to recommend not installing them that way.
