a. user say:

Games that date back to Windows 9x are not likely to run - or even install - on Windows 10. Games from the Windows XP era might install or run on Windows 10 - or they might not. It all depends on how they're coded.

It's possible to install old games in what people call 'compatibility mode' and that sometimes works and sometimes not.

But if you really want to experience those games the way they were meant to be experienced, run them inside a virtual machine. A virtual machine is a computer created out of software. Imagine having a Windows 98 computer running on Windows 10 ... it's easy if you know how.

Oracle's VirtualBox and VMWare's Workstation Player both make easy work of creating virtual machines. Once you have your VM setup, install your old game in the VM just like you would in a real computer, because a virtual machine is a real computer.

greg carmack - mvp 2010-2020 say:
Hi Xander. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

That it works in Safe Mode proves the issue is with a driver. That driver is either the Keyboard, Mouse or more likely the USB hub. There is a major problem with Windows Update pushing drivers that are wrong right now, requiring us to roll back or Update the driver.

First try rolling back the Driver in Device Manager reached by right clicking the Start Menu. One by one, expand Keyboard, Mouse and Enhanced and Hub USB controllers, choose Driver tab then choose Roll Back if it's available.

First I would program the Power button to Shut Down the PC so you can restart if you lose mouse or keyboard doing this. Do this at Settings > System > Power & Sleep > Additional Power Settings > Choose what Power buttons do.

If Roll Back is not available, then try Uninstall driver and restart PC to reinstall.

Finally update the drivers for Chipset and USB including USB3 from the PC's Support Downloads web page, for HP using Serial from sticker on side.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
