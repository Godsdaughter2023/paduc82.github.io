Do I need to backup all my files before upgrading to Windows 10? - Microsoft Community

deana j. say:

Do I need to backup all my files before upgrading to Windows 10?

Do I need to backup all the information and files on my computer (pictures, documents, music, etc.) on a separate device before I upgrade to Windows 10? I didn't have to when I upgraded from Win. 8 to 8.1.
tryx3 say:

Deana,

Upgrading an OS is a major task and there are a lot of things that can go wrong.  You would be taking a great risk by not updating your backups before upgrading.

I suggest:

1     Make a [Windows 7] System repair disk or [Windows 8] Recovery drive.  Test it by booting from it so you know it works.

2     Make a system image of the drive that has Windows installed on it.  In Windows 7, this is run from the opening window of Windows backup but I don't know the equivalent location for Windows 8 [Windows Help will tell you].

3     Make a backup of all your own files, all the installation files you've saved for applications you have added and all the product keys you need to activate those applications.

4     [Optional but very useful]  Make screenshots of the settings / options for your applications so you can very quickly get them back the way you like them.  Screenshots are a lot quicker than writing down every individual setting.

[Personally, I am making two backups on separate external drives]

Whatever goes wrong with the upgrade, if you have those tasks done then you will be able to return to the point you started from even if the upgrade fails and its built-in "roll back to previous operating system" capability fails.

Of course, the upgrade might run without any problems whatsoever.  Every one of my installations & updates of the Windows 10 Preview versions has done just that.
