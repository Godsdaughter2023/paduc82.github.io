not able to browse the Internet using any internet browser

 

That symptom is often seen when users update but have a third-party security package installed which sees their job as protecting the user from unknown programs which they consider to be possible "Trojans".  The problem is that the update may have changed existing programs or the programs that they need to use to send requests out so that they too get blocked by such interferers.   So, before reinstalling Windows, try uninstalling all such packages you may have and then see if that changes your symptom.  If they were running during the last cumulative update (aka quality update) you may also need to uninstall and then reinstall that to ensure that the interferers also didn't cause corruption.
