If the PC came with Windows pre-installed by the PC maker then the Product Key will be on the casing or hidden inside the battery compartment [so you don't have problems later on with the label getting worn off].  See the examples at How to Tell – Software  &  How to Tell – Hardware

pcbpcb say:

Here is the solution, for all drives, not just WD.

Bottom line, longer file names are backed up, but are stored in different folders than those with shorter file names. To see this, go to UserName/Desktop-XXXXXX/Data. Then short file names go into /C folder and the hierarchy will look like the files and folders you have on your desktop. Long files names go into $OF folder - but note, the file and folder structure is totally different and you will not be able to find you file easily.

To see all your files in the right folders, search for "Restore Your Files with File History." As you type Restore it will show up some ways down the list. When you click on "Restore Your Files..." it will bring up a window that will have all your backed up files and folders populated with both short and long names. From there, you can browse to whatever file you want and restore it. 

Mithin, please take this info and turn it into an FAQ for users and moderators so that people like me can find the solution quickly. Thank you.

Note also, this is not a problem with WD or any other manufacturer, it is a quirk of Windows/MS.
