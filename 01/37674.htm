Your inbox got corrupted and the messages may have been wiped from it.  Close OE and go to the message store and move the inbox.dbx file to another location.  OE will then create a new one.  To get the messages from the older inbox you can try to extract them (cf. www.oehelp.com/DBXpress/) or you can look in the recycle bin or message store for an inbox.bak file and if it is present, then you can import it back into OE using the method described here:  www.oehelp.com/backup.aspx#imp1  (see that page also for details on the message store location and also see www.oehelp.com/OETips.aspx#2).
