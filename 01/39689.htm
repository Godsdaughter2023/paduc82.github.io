Hello Redoced,

 

.tmp files are created when a program needs to hold temporary information. For instance, when you are on Internet Explorer the program has to store the page information of what you are viewing. The program creates a .tmp files that is usually deleted/cleared out when the program is closed. You can remove .tmp files using the Disk Cleanup tool.

First you will to restart in a clean boot.

·         ·         Click Start

·         ·         Click Run

·         ·         Enter msconfig into the Open: field

·         ·         Click OK

·         ·         Click the Services tab

·         ·         Check the Hide All Microsoft Services box

·         ·         Click Disable All

·         ·         Click OK

·         ·         Click Restart

 

Disk Cleanup:

·         ·         Click Start

·         ·         Click All Programs

·         ·         Click Accessories

·         ·         Click System Tools

·         ·         Click Disk Cleanup

·         ·         Check the boxes of the files you want deleted

·         ·         Click OK

·         ·         Click Yes

 

Replace normal boot settings:

·         ·         Click Start

·         ·         Click Run

·         ·         Enter msconfig into the Open: field

·         ·         Click OK

·         ·         Click Normal Startup

·         ·         Click OK

·         ·         Click Restart (if prompted)

 

You will find that the .tmp files will regenerate, this is because programs use them as part of their functionality. Typically they are cleared out when the program closes, and you may find most of them are 0kb in size.

 

Something to consider is malware can manifest itself within the .tmp files and cause problems within your system. It is recommended that you keep an updated antivirus program running at all times.

 

Hope that helps, look forward to hearing back from you.

Possibly, since you accidentally created new Skype account when the updates initiated, you sign in to it. In line with that, all information like Skype credits, subscription and contacts cannot be found on the said account.

Question: Can I delete the update/new account, thus reverting to my old account?

Answer: Yes, you can delete the new account on your end. Since the migration of Skype accounts to Microsoft account, the account deletion process has been modified wherein users can delete the account on their end. Furthermore, deleting a Skype account will also result in deleting the Microsoft account associated with it.

With regards to the old account, you can still access it using the credential you have used when you initially created the account.

Issue: It had me opening a new account, then wiped out my old account as well as my credit.  Tried all the links recommended ,but to no avail.  I would like my credit re installed, but all attempts fail.

Also, try to sign out to the Skype application on your computer, then sign back in using the same credential when you initially created the account.

Note: If there is an option, "Sign in using different account", on the Skype application, please select that before signing in back.

Refer to the link for additional information. How do I close my Skype account?

Unfortunately the built-in administrator account is not supported by Microsoft WBF. The solution is to create a new account with administrator rights and enroll for this account.

 

Another workaround is to rename the administrator account and import the fingerprints:

1. Use Windows Key +R (Run) and type "control userpasswords2"

2. Create a new user with a password and admin rights, choose a username that you will want to rename the built-in Administrator account to, i.e. SysAdmin, or Admin etc.

6. Once this file is exported, log off the new user, log on to the built-in Administrator, first delete the newly created user (via Run: "control userpasswords2") 7. Next you want to rename your built-in Administrator account in the Advanced tab to the username you just deleted. It must be the same username exactly.

8. Once you have renamed your built-in Administrator account. Log off. Then Log back on.

9. Start Protector Suite and Import the Passport. Once you select your exported file for import, you will generally need to enter the password you created for it when you exported it, but its important to remember that you will only be able to use the fingerprints you enrolled under the other user, you will never be able to add new fingerprints to enroll, or otherwise edit the enrolled fingerprints.
