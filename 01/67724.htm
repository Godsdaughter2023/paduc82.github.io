Windows Media Center - Microsoft Community

travy d say:

Windows Media Center

I have been running Media Center for a 2 years on my computer. THe last month every time that I play music using Media Center my computer will freeze up. Sometimes during the first song and other times after 15 songs.

Thanks,
Travis
imran chand say: Hi travy d,

Thank you for using Microsoft Answers Forum.

This problem occurs if the Windows Media Center update process (Mcupdate.exe) cannot access the Windows Media Center data store. You might experience this problem if the data store is damaged. Try to rebuild the Media Center data store. To do this, follow the steps given below:

1.            Exit any running instance of Windows Media Center.

2.            Delete the mediaCenterDataStore.db file. By default, this file is in the following location:

%SYSTEMDRIVE%\ProgramData\Microsoft\eHome\mediaCenterDataStore.db

To delete the file, follow these steps:

a.            Click Start, and then click Run.

b.            In the Open box, type run, and then click OK

 

If you are prompted for an administrator password or for confirmation, type the password, or click Continue.

c.             In the Open box, type %systemdrive%\programdata\microsoft\ehome , and then click OK.

d.            In the details pane, right-click mediacenterdatastore.db , and then click Delete.

e.            Click Yes to confirm that you want to move the Mediacenterdatastore.db file to the Recycle Bin.

 

3.          Start Windows Media Center.

The Windows Media Center update process (Mcupdate.exe) is a scheduled task that runs two times every day. If this scheduled task cannot access the data store, the Mcupdate.exe process crashes.

You can cancel the Mcupdate.exe scheduled task. In order to do that, follow the steps given below:


1. Click Start. 

2.  In the Start Search box, type task scheduler.
 

3.  In the Programs list, click Task Scheduler.

 

 If you are prompted for an administrator password or for confirmation, type the password, or click Continue. 

4.  In the navigation pane, expand Task Scheduler Library, expand Microsoft, expand Windows, and then click Media Center. 

5.  In the details pane, right-click mcupdate, and then click Disable.

 

To fix this problem automatically, click on the link below:

http://go.microsoft.com/?linkid=9643524

Click Run in the File Download dialog box, and follow the steps in the wizard.

 

This should fix the issue you are experiencing.

I hope this helps you.

Manasa P- Microsoft Support
