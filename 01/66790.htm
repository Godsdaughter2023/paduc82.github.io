erika aro say:

Hi,

Windows.old gives you the chance to go back to the previous version of Windows. If you delete the folder, you won't be able to roll back anymore. You can delete it in the File Explorer by right-clicking the folder and select Delete.

After 30 days from the upgrade, an automatic Disk Cleanup will run and remove Windows.old. If you want to delete it manually via Disk Cleanup, follow the steps below:

  1. Type Disk Cleanup in Cortana search, right-click the icon and select Run as Administrator.

  2. Click Clean up System Files to run, then check the box for Previous versions of Windows.

  3. Click Clean up System Files again.

  4. Check the folder if it's still there in reduced size. It means that further security changes are required to have it completely removed.

    susan_s. say:

    Hello,

    To isolate the issue, please try below steps:

    1. Open the Start menu and click Settings.
    2. Click Apps.
    3. Locate and click Skype from the Applications list.
    4. Click Advanced option.
    5. Select Reset.​​

    You may also try to temporary disable any security program, such as firewall or antivirus on the device, use a different connection, uninstall/reinstall the Skype app and check on the issue again.

    If the issue still persists, try to update audio driver, following below steps:

    • Select the Windows logo/Start button, type Device Manager, and select it from the list of results. 
    • Expand one of the categories to find the name of your device (audio driver), then right-click (or tap and hold) it, and select Update Driver. 
    • Select Search automatically for updated driver software. 
    • If these steps don't work, try reinstalling the driver:  Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall.
