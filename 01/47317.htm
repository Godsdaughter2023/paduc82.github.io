dominicp say:

You need an HDMI Audio Driver to pass sound through your graphics card to the HDMI output. From what you are experiencing, that driver has been removed or damaged.

Uninstalling and reinstalling the sound and/or video drivers should recreate a working HDMI output.

andrewowen88 say:

Cool, thank you for the fast reply. 

How do i install new drivers? i.e. where do i get them from? I can see that there is an option to uninstall drivers but am not sure how to reinstall them. Not sure where the setup.exe file would be located.

dominicp say:

Drivers (and BIOS updates, which might be relevant) are available from your laptop manufacturer's site.

The first step is to uninstall the graphics, sound and HDMI Audio drivers in Device Manager. without removing the drivers themselves, then reboot and let the drivers re-install.

If that does not solve the problem look for updates and solutions on your laptop manufacturer's site and its forums specific to your model.

andrewowen88 say:

Oh ok, that makes sense. 

So if i uninstall the drivers then they WILL automatically reinstall upon a reboot? I'm wary about doing it and then not having them reinstall upon restart.

dominicp say:

As long as you do not remove the drivers themselves - just uninstall the devices and do not check the box offering to remove the drivers if the option is offered - the drivers will still be present and the hardware will be redetected on boot and the drivers installed.

If the driver components are damaged so that they do not work correctly you will need to find replacements from your laptop manufacturer's site. The drivers will be specific to your laptop model.

Your laptop manufacturer may have diagnostic and repair software you could use.

cajuncomputerdude say: Hit the same problem on a couple of new system builds today.  Could not get through the install for Java in Win7, but could for the Win-XP VM on the same machine using the same executable.

My workaround was to run the Java installer in an XP compatibility mode.

Go back to your download directory and right click the install program (e.g. chromeinstall-7u6.exe in my case) and select Properties from the bottom of the menu.

Select the Compatibility tab.

Check the "Run This Program in Compatibility Mode for" check box and select Windows XP (Service Pack 3) from the pull down.

Under Privilege Level, check the "Run This Program as an Administrator" box.  Then hit OK.

Run the installer program now and there should be no hang up.  Was able to run the java checker and website apps we needed them for after this method was applied.

I hope this helps.
