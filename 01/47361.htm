Did you use  Run...'s  Browse... button to see that it's there?  FWIW I didn't even realize that that is where it was.   <eg>

If you are in Administrator Command Prompt that is not the standard directory which it usually starts in.  So that implies that perhaps you do have a functional Path Environmental variable but if you aren't entering the file extension perhaps you don't have a PATHEXT value (in which case I'm not sure why it would decide to show you one).   Which all suggests that either you are using a machine which has been locked down by someone--in which case see that person about whether they want you to try doing these things or that you have malware which has severely taken over your machine--in which case the best thing to do would be to "Nuke and pave".

cd  /d  %windir%

dir/a/b/s  regedit.exe

(I guess I'm assuming that you also have %windir% defined properly for doing that.  It's just a reflexive habit to avoid getting stuck on the wrong volume.  So, if the %SystemDrive% is C:  %windir% (and %SystemRoot% apparently) would usually be  C:\Windows\ )

On your laptop, insert he bootable USB Flash Drive, then just as it boots look for text like, Press F2 to enter Setup (might also be Esc or Del key), press this button immediately!!

 

In BIOS go to the Boot tab and change the boot order so the USB flash drive is top of the list, save changes and restart, the installation should begin. Make sure to check the 'I am reinstalling Windows 10 on this PC' box.

 

NOTE: during the installation your system will reboot, at this point you must go into BIOS again and set your hard drive as top of the boot list and save changes to bios. Otherwise the installation process will start again . . .
