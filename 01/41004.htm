MTU size can be changed, but not with the "ping" command.  The ping command can simply send test packets of various sizes to detect where MTU should be set.  It cannot be changed to be bigger than 1500 but it can be adjusted downward.

Windows XP does not provide an interface for changing MTU..  The easiest way I've found to set the MTU is to use the freeware "DrTCP" program that is also available at the dslreports site.  If you're interested in this, check out their "Tweak Tests" program as well as "DrTCP" -- both available at this page:

bruce hagen ~ mvp say:

What happened to the Run command?

The Run command no longer appears on the Start menu in this version of Windows. The search box that appears on the Start menu provides much of the same functionality as the Run command. However, the Run command is still available if you prefer to use it. You can even add it to the Start menu for easier access.

To add the Run command to the Start menu
Click to open Taskbar and Start Menu Properties.

Click the Start Menu tab, and then click Customize.

In the list of Start menu options, select the Run command check box, and then click OK. The Run command will be displayed on the right side of the Start menu.

Tip:
You can also access the Run command by pressing the Windows logo key + R.

I doubt if updating the driver would help. A newer driver might
improve the feature set of the printer, but an old one wouldn't stop
working. However there is a remote possibility that the older driver
has a conflict with a Windows update that you've installed, and if
that conflict has been fixed in a new driver, updating the driver
might help. I strongly doubt that, but on the other hand, it
wouldn't hurt to try.

And by the way, I wouldn't pay Dell $100 to address the issue. If I
wanted to spend the $100, I'd sooner buy a new printer.
