To manually customize the folder picture that the Thumbnails view uses, put a Graphics Interchange Format (.gif) image file in the folder with a file name of Folder.gif. You can use any graphic editing program to create the .gif image. To use the Microsoft Paint program, follow these steps:
  1. Click Start, point to All Programs (or Programs), point to Accessories, and then click Paint.
  2. To open an existing image, follow these steps:
    1. On the File menu, click Open.

    3. Select the image that you want to use, and then click Open.
  3. Use the tools in Paint to modify the image (if you want to modify it).
  4. Save the image after you modify the image. To do so, on the File menu, click Save as.
  5. In the Save as type box, select GIF (*.GIF).
  6. For the file name, type:
    Folder.gif.
  7. Locate the folder that you want to customize, and then click Save.
When you view the folder in Thumbnails view in Windows Explorer, the folder now has the custom image that you created, unless you use the Customize tab of the folder's Properties dialog box to select a folder picture. Here are some notes that may help resolve problems encountered when installing Windows Updates.

TIP: What On Earth Is Soul Sand Good For? - Microsoft Community

wildattorney say:

TIP: What On Earth Is Soul Sand Good For?

I almost never see a world where someone incorporates soul sand in anything but the occasional dock.  In light of that, I thought it might be helpful to post a little information about the various uses of soul sand in the game.  For starters, the basics:  Soul sand is found in the Nether as a layer of what looks like a darker shade of dirt (with facial images on it) that sits atop Netherrack.  It's primary characteristic is that it slow you down considerably when you walk on it.  This is obviously not a functionality that most people look for in their maps (they'd prefer to go faster, not slower), so it rarely gets used.  There are a number of practical uses for it, however. 

 

First, as I referenced above, it is great for boat docks.  Place the soul sand at water level, so that the top of the soul sand (and only the top of it) is exposed to air (with the rest of block being submerged).  If you then land a boat on it, the boat will stop atop the soul sand and when you leave the boat, the boat will remain in one place.  Without soul sand, you are likely to either crash your boat into pieces or see it drifting off into the ocean. 

 

Second, it's great for one-way doors.  Soul sand is a slightly smaller block that other blocks.  If you make a top block high doorway and place a block of soul sand in the ground in front of the doorway (note, do NOT place it in the floor of the doorway itself, but in the floor in front or in back of the doorway, depending on your perspective), you will not be able to enter the doorway, although someone coming from the other side will be able to enter.  In other words, if you have to step on soul sand to enter the doorway, you will not be able to, but if you step on soul sand in the doorway or as you exit the doorway, you will be able to pass. One caveat to this.  Someone will still be able to get through the doorway if they come at it at an angle.  You can prevent this by making the doorway at least 2 blocks deep. 

 

Third, you can make MOB traps out of it.  Place a piece of soul sand in the ground. Then on the third block up from that soul sand, place four blocks of soul sand in a donut or halo formation, such that the hole is three blocks directly above the soul sand that is in the ground.  If you do this, any MOB that stands 2 blocks high (creeper, zombie, and skeleton) will get stuck once they step on the soul sand and will be unable to move from that point.  You can do the same with MOBs that are one block high (pigs, sheep, etc) by simply making the donut forumation 2 blocks up from the ground (as you see, the key is that the gap between the floor and the donut have to be equal to the size of the MOB you want to trap. 

 

Fourth, while you can use soul sand to make things go slower, you can greatly increase this effect if you put the soul sand on ice.  Now, this has a limited application in this version of the game, since we can't place ice.  So you would have to use pre-existing iced areas of the map. 

 

These are some of the uses of soul sand in the game.  If someone has found other uses, please opine. Of course, I should mention that soul sand can also be used as decoration (including in pixel art if you need something with that exact color).
