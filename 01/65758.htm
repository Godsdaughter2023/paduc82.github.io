I was able to get the NAT to Open on BOTH Xbox Ones in my house, as wall as Open in Black Ops 3 at the same time but I had to get help from tech support. I had to put my Motorola Surfboard sbg6580 modem in to bridge mode, which Arris tech support helped me do over the phone. All of the guides online are wrong. Then the Nighthawk router could not connect to the internet so I called their tech support and they walked me through how to set it up with a bridged modem. This is what Netgear had me do after I got the modem set up properly. This required me to have a desktop to have a wired connection to the router.

1. Reset the router to factory defaults.

2. Reboot the router and open a browser window. It should automatically try and start up the setup genie.

3. When the genie pops up, select No, I would like to set up the router myself. Log in to the router.

4. Select the Internet tab. Scroll down to the options for MAC address. Select GET COMPUTER MAC ADDRESS. Click Apply.

5. Power cycle the modem.

6. Power cycle the router, and the computer plugged in to it.

7. Log in to the router again.

8. To set NAT to OPEN, go to the ADVANCED tab. On the left select the SETUP drop down menu.

9. On the left under that menu go to WAN Setup. Under NAT filtering select OPEN.  Click APPLY.

10. Make sure UPnP is enabled. Go to ADVANCED, then on the left select ADVANCED SETUP.

11. Make sure the box that says Turn UPnP On is checked.

10. Apply the changes, reboot everything again.
