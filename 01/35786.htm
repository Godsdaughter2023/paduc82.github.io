    There's a lot of reasons it could happen, but my guess is a failed shell extension.  She'll Extensions are what add the options to the right-click menus you see when you right-click on various things.  The desktop shell is different from folders, so even if a program (like Google) added an option to the menu in both locations, it's possible for only one to have a problem.  You might be familiar with common shell extensions, like "Scan this file with Norton" on right-click menus and such.

    -Open Control Panel.

    -Type the new name for the account.
    -Click the Change Name button.

    -Open Start.
    -Search for Run, and click the top result.
    -Type netplwiz.
    -Click the OK button.
    -Select the account you want to modify.
    -Click the Properties button.
    -Type a username. (Usually, you want to use a short name that uses one word.)

    -Click the Apply button.
    -Click the OK button.
    -Click the OK button again.
    -After you complete these steps, the sign-in screen will display the name you specified in the Full name field.
