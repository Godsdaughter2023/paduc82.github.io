¡firedog say:

A couple of comments to start with:, as much for the benefit of other users finding this thread in the future as for your own:

 

  • If you ever need help with an error message, it's vital to quote it verbatim. The text you quote isn't a Windows Live Mail error message.
  • The message states clearly that the problem is with data, not with the program itself. Re-installing the program files will have no effect on this fault, as you discovered.
  • This error usually arises because the user has done something that has messed up permissions for access to the store folder, where all the program data are stored. It would have been helpful if you'd mentioned this.

If the normal user account doesn't have permissions to write to the message database index file, you'll see this error. To correct this,

 

  1. Log on to Windows with an administrator account and ensure that no programs are running.
     
  2. Use Explorer to navigate to your store folder, probably at
    C:\Users\{user}\AppData\Local\Microsoft\Windows Live Mail
    where {user} is the Windows user having the problem.
     
  3. Right-click on Properties.
     
  4. On the Security tab, click Advanced, then open the If the Current owner box doesn't contain your normal Windows user account, click Edit then Other users or groups.
    Type the name of your normal Windows user account into the Enter the object name... box and click Check names. The official name of the account should appear in the format {computer name}\{username}.

    Click     OK, select this username in the Change owner to box then OK.
     
  5. Open the Permissions tab and click Change permissions.
     
  6. Click Add and type the name of your normal Windows user account into the Enter the object name... box and click Check names. The official name of the account should appear in the format {computer name}\{username}.

    Click     OK, select this username in the Permissions box and click
    Select the checkbox labelled Full control in the column.
     
  7. Click OK three times to close all dialogue boxes. 
     
  8. Launch Windows Live Mail.
     

If you get stuck anywhere, Cancel your way out and report back.

a. user say:

Hi Jammi,

You can run the compatibility appraiser again by following these steps:

1. Right-click on the start button and select 'Command prompt admin'.

2. Type the following command and press enter

schtasks.exe /Run /TN "\Microsoft\Windows\Application Experience\Microsoft Compatibility Appraiser"

Wait for the test to finish and then check the results.
