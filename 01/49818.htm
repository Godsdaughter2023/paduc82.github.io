Hello, Jaalkaup

Try doing a clean boot of Windows, this will help rule out if another program is interfering with the backup process. To perform a clean boot, do the following:

1.     Log on to the computer by using an account that has administrator rights.

2.     Click Start, type msconfig.exe in the Start Search box, and then press ENTER to start the System Configuration Utility.
If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

3.     On the General tab, click Selective Startup, and then click to clear the Load startup items check box. (The Use Original Boot.ini check box is unavailable.)

5.     Click OK, and then click Restart.

After you perform a clean boot try to create a backup. If you are able to run the backup there is likely another program causing it to fail. You will want to follow the troubleshooting steps at the link below.

Difference between UDF and CDFS

What is the Difference between UDF and CDFS systems? And Whats the best out of them?
shenan stanley say:
What is the Difference between UDF and CDFS systems? And Whats the best out of them?


Use UDF most of the time.

CDFS uses the ATAPI block driver for audio and video navigation and playback. It follows the ISO 9660 specification and supports long file names.

UDF uses the USB block driver or the ATAPI block driver for audio and video navigation and playback. It follows OTSA UDF revisions 1.02 through 2.50. These versions of UDF specifications replace CDFS for optical media file systems that support live file systems in addition to mastered media.

(UDF was made to somewhat replace CDFS... and work with it.)

¡firedog say:

You have now discovered the difference between being signed in and not being signed in in Windows Live Mail.

  • When you're signed in, you see the contacts list belonging to the account you sign in with.
    This list remains synchronized with the one at contacts.live.com, which is also available to you from other devices like your iPad.
  • When you're not signed in, you see the default contacts list.
    This list is not synchronized at all.

The same applies to calendars.


If you want to be able to use the same contacts list in various places, and to keep them safely backed up, then always sign in in Windows Live Mail. Since you constructed your 'proper' list with its categories while you were not signed in, you're going to have to set up the categories again. There is no easy way of transferring contact data from the default set to another set.
