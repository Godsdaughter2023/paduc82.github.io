I doubt you have exactly the same subfolders in both Program Files folders. That's not how it works. a. user say:

Hi Samuel,

Thank you for posting your query in Microsoft Community. Appreciate your interest in using Windows 10. Apologize for the inconvenience caused.

I would like to inform that when an upgrade is done, there will be a folder created in the name “Windows.old” in the drive where the Operating System is installed. It will contain the data, files and program files of the previous Operating System. I suggest you to check if the files are available in this folder.

Also, I would like to inform that Microsoft Works 9 was the final version of Works and the software was discontinued in 2010. However, Windows users can still open WPS files with Microsoft Word or Microsoft Publisher. To open WPS files with either of these programs, you may need to select "Word 2000" or "All File Types" in the Open dialog box. You can also convert Works documents to Word documents using the "Microsoft Works 6–9 File Converter" application.

Hope the information is helpful. Feel free to write to us for any other assistance with Windows, we’ll be glad to assist you.

cbarnhorst say:

If you made the highly advisable system backup before upgrading to Windows 10 you can locate the files in the backup and restore.  You should also be able to locate them in the Windows.old file that Setup made and stored on your computer when you upgraded.  That file will disappear one month after you upgraded.   It is in the root of the system drive, for example:

C:\Windows.old
