or

What is the make and model of your computer.
What type of Internet connection are you on, wireless or wired? If you are on a wireless connection, try switching to a wired connection then try browsing again.

Press Windows key + X
Click Command Prompt (Admin)
At the command prompt, enter the following command, the hit Enter
ipconfig/release
then
ipconfig/renew
then
ipconfig/flushdns
exit the command prompt


If the problem persist, try resetting your router if you are connected one.
Power down the router, disconnect it, wait 10 mins
Reconnect it, power up again
Reconnect the Ethernet cable
Try browsing again

You can also try running the Internet Connections and Network adapters troubleshooters.

Open Start > Settings > Update & security > Troubleshoot
Select Troubleshooter
Click Run the Troubleshooter

When complete, restart the try browsing again.

If that does not work, you likely need to update your computer drivers for the adapter.

Press Windows key + X
Click Device Manager
Expand network adapters
Right click each, then click uninstall
Exit Device Manager then restart

Let it redetect and reinstall the drivers.

Try browsing again.

Learn more about updating drivers:
