¡firedog say:
... account is locked by another session or for maintenance

This is the server's response. Maintenance will happen occasionally, but won't normally last more than an hour or two.

However, there can only be one connection to a POP3 mailbox at once. If you are accessing the same account from a mobile device, and that device is using push mail, then it will monopolize the connection and not allow any other client - e.g. Windows Live Mail - to connect to the mailbox. kevin hau say: Hello Ben,

I hope I can add some clarification to your questions.

When reading about upgrade requirements for Windows 7, we need to remember that many of the requirements are specific to an "upgrade" installation.  This means that you move from an older version of Windows to Windows 7 and all your data is kept on the drive and installed games and programs are still there the way you left them.  The opposite of an "upgrade" is a clean installation, which means that you wipe the hard drive clean and reinstall all your programs and restore your data from your backup (word docs, pictures, etc). 

It's true that you cannot "upgrade" from Vista to Windows 7 if the languages don't match, but you can perform a clean installation type upgrade from Vista (Japanese) to Windows 7 (English).  You will just have to back up your data first and reinstall applications.  Just choose "Custom Installation" from the installation options in the Windows 7 setup.  Here's a link with more details:

http://windows.microsoft.com/en-us/windows7/Installing-and-reinstalling-Windows#section_1

One other important thing to remember is that if you have an upgrade version of Windows 7, you have to have an installation of Windows that is eligible for the upgrade installed in order to perform the upgrade installation.  You can  launch the installation of Windows 7 from within the previous installation of Windows, or you can boot from the DVD and let setup check for the previous installation.  If you are trying to install on an empty formatted drive, you will have to install the previous version of Windows then perform the upgrade to Windows 7.

From a licensing perspective, you are eligible to "upgrade" from any version of Windows since Windows 2k, you will just have certain versions that will require a clean installation, and some that can "upgrade" and keep all your data intact.

Here's a pretty comprehensive document showing what versions of Windows can "upgrade" to Windows 7 and which require a clean installation.

http://www.microsoft.com/downloads/details.aspx?FamilyID=e170eba1-5bab-401f-bbf5-00f0ee7fe0fb&displaylang=en

I hope this helps!
