Problem with Hitman: Blood money - Microsoft Community

xnzlvx say:

Problem with Hitman: Blood money

I recently downloaded Hitman: Blood money and discovered i'm unable to play it. Every time I go to play the game an error shows up on screen saying: This game does not support PAL-50. Please change your display setting to PAL-60. To change your setting, in System, select Console Settings, Display.

This is fine but for some reason when I go to change my settings in the Display, HDTV menu it doesn't show up with "PAL 60".  The settings it does show up with are:

480p, 720p, 1080i, 1080p, 848x480, 1024x768, 1280x768, 1280x1024, 1360x768, 1440x900, 1680x1050

All help is appreciated.

obsessive power say:

Those resolutions show because you're using VGA.  There is an issue with Hitman and VGA with PAL based consoles.  To fix it you need to do the following:

NOTE: YOU ONLY NEED TO DO THIS FIX, ONCE.

With your console off, remove the VGA cable.
Attach either an Xbox 360 Component cable (with the switch set to SD) or Xbox 360 composite cable.  Then connect back up.  If your screen is a computer monitor without Composit or Component inputs, you'll need to connect to Television that has these inputs.

With the SD cable connected, turn on your console, go to System Settings > Display Settings.  From here there should be an option to change the setting to PAL60. 

Turn off your console, re-attach your VGA cable, turn it back on and try your game.  If all goes well, the game should work now.

Please let us know if this has fixed your issue.  Thanks.
