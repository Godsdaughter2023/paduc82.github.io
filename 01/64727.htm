davem121 say:
Hi Steph,

Chromium is a nuisance, and could be classed as malware it did not come with Windows or from Microsoft - it is installed automatically when you install 3rd software and is extremely difficult to remove . . .

The best way to remove it is to manually delete it:

Open Windows file explorer

On the view ribbon, check the box marked 'Hidden Items'

Navigate to (replace username with your login name):

C:\Users\username\AppData\

Open the Local and Roaming folders, and if you see a Chromium folder inside either or both of them, delete it


Press Ctrl + Shift + Esc to open Task Manager
Click on the Startup Tab
See if Chromium is listed there
If it is, select it and click 'Disable'
Close Task Manager

Click your Start Button, type msconfig and hit Enter
When the System Configuration dialog opens, click on the Services Tab
Check the box marked 'Hide Microsoft Services'
In the remaining list see if Chromium is listed there, uncheck any you find
Click Apply and OK

Restart your System and this annoyance will be gone . . .
