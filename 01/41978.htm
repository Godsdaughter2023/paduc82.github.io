Editing process prioities - Microsoft Community

jack raiden. moderator on kdgaming. say:

Editing process prioities

If you edit a process priority in task manager, is that change permanent until you change it back? I don't want the computer to fail and then not be able to change it back.

a. user say:

Hi Jake,

 

I appreciate the efforts which you have put to post the query on this forum.

 

Do you want to change the CPU priority level of processes for Windows 8 store apps?

 

Well, the changes to the priority level for the processes is temporary and not set permanently. Once you close the application or restart the PC, Windows 8 will automatically set the priority back to default for the process or application.

 

Further adding to it, you can follow these steps to change the priority level under Task Manager:

a)      Open the application for which you want to change the processor priority level.

b)      Type “task manager” and click on Task Manager on the Start screen.

c)       Click on More details if the window is not in expanded form.

d)      Navigate to processes tab, right click on the application and click on Go to details.

e)      In the Details tab, right click on the already highlighted name of the application for the process from step 4, click/tap on Set priority, and click/tap on the priority level you want to run this application or process in.

f)       Click/tap on Change Priority to apply.

g)      Repeat steps 4 to 6 above for any other applications or processes you wish to change the priority level of.

h)      Close the Task Manager window.

Note: You will not be able to set the processor priority level of Windows 8 Store apps.

 

Please let us know if you need more assistance on issues with Windows.

Hi Trisha,

Thank you for contacting Microsoft Community. I understand that you have an issue with creating playlist in Windows Media Player. I also see that you issues with searching. I appreciate your efforts to try troubleshooting steps, I will surely help you to fix this issue.

To understand the issue better, let me know what exactly you mean by resetting Windows Media Player to default? Did you try to Turn Windows Features on and off?

This issue could have occurred due to these reasons:
1. Corrupt Windows Media player library database.
2. Corrupt User Profile.
3. Incorrect Search settings.

Try the steps listed here and check if it helps:

Method 1:
I would suggest you to rebuild the Windows Media player library database and check if it helps.

a. Exit Windows Media Player.
b. Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.
c. Select all the files in the folder, and then click Delete on the File menu.
Note: You do not have to delete the folders that are in this folder.
d. Restart Windows Media Player.

Note: Windows Media Player automatically rebuilds the database.

Method 2:
If the issue persists, I would suggest you to check if the issue occurs on a New User Account or not. To create a new User Account, refer to the article and check if it helps.

Create a user account
http://windows.microsoft.com/en-us/windows/create-user-account#create-user-account=windows-8

Method 3:
To resolve the issue with Search, I would suggest you to run “Search and Indexing” troubleshooter which will check if basic search settings are set correctly.

Follow these steps:
a. Press “Windows key + W” on your keyboard.
b. Type “Troubleshooting” without quotes in the search box and hit Enter.
c. Select “View all” from the left panel.
d. Select “Search and Indexing” and follow the on-screen instructions.

Method 4:
If you have not tried to disable and re-enable Windows Media Player from Control Panel, I would suggest you to try the same and check if it helps.

Step 1: Disable Windows Media Player.


a. Press Windows key, type Control Panel, click Programs, and then click Turn Windows features on or off. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
b. Browse to the Media Features and uncheck the mark in front of Windows Media Player.
c. Restart the computer.

Step 2: Re-enable the Windows Media Player.


a. Press Windows key, type Control Panel, click Programs, and then click Turn Windows features on or off. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
b. Browse to the Media Features and place a check mark in front of Windows Media Player.
c. Restart the Computer and check with the issue.

Refer to the article for additional information.

Indexing and Search: Frequently asked questions
http://windows.microsoft.com/en-in/windows-8/search-apps-files-settings
