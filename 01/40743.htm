ronnie vernon say:

Hi

If you just want to change the path that opens when you click the Windows Explorer icon that is pinned to the Taskbar, perform the following.

1. Right click the Windows Explorer icon. This will open the Jump List menu.

2. In the lower section of the Jump List, right click Windows Explorer and select Properties.

3. Click the Shortcut Tab.

4. If you want Computer to open with this icon, enter the following in the Target box and click Apply/OK.

%windir%\explorer.exe e,

(be sure to add the Comma after the e)

To change the shortcut to open a different location, just add a space + the path to that location.

Example: If you want the shortcut to open the C:\Drivers folder, add the following.

%windir%\explorer.exe e, C:\Drivers

To change it to open the root of the system drive:

%windir%\explorer.exe e, C:

Etc.

Let us know if this helps.

Hi Aakash,

Unfortunately, once the Windows 10 Setup closes during the restart, there is no way to resume the download. During the Windows 10 Setup, you will be asked if you want to perform an in-place upgrade or create media for another PC. I suggest that you choose the latter as it might be faster.
