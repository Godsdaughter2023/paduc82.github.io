Hi CJ.75,

Graphics card issue on Win10 is certainly the biggest and most encountered issue so far...

I have had my frustration over it many times. Here's my suggestion to help you either clean install, update driver, or install necessary upgrades -- all smoothly without any hindrance. 

1. If you have an internal iGPU/onboard display, then I suggest you remove the discreet graphics card... and, freshly install your Windows 10. You will be amazed the amount of updates that will be installed; microcodes etc., I have had not noticed such updates when I had only discreet GPU attached. Once you've installed Windows, use only Windows update to let it find the drivers for iGPU/Audio etc.,. You can also right click on the missing device and then click update(to manually update it over the internet). I usually let Windows handle all the necessary drivers that it could find and install without any interference from me.

2. Once that's done. Now, turn of your machine, and install your discreet GPU(nVIDIA or AMD whichever it is). Hopefully, your GPU supports UEFI driver if you've Windows 10 installed with CSM disabled in bios. If you do, then you should see the Windows logon screen. Often people try running nVIDIA/GeForce setup while Microsoft tries to rerun its update for the graphics --- resulting in the device going back n forth between 'Microsoft Basic Display' and 'Video/VGA compatible controller'. nVIDIA setup won't run without seeing 'MS Basic Display'. However, once I enabled both integrated and discreet GPU.... voila! MS started installing updates in a jiff! 

So, whatever CPU processor you got... just make sure it's drivers are updated. Then, add your big card. Also, remove external storage cards, wifi lan card etc., ...they often cause Blue Screen of Death!
