keith a. miller say:

If the file is not in your OneDrive folder or one of its subfolders, it won't be synced.

It sounds like you're placing a copy of an original file saved elsewhere in the file system in your OneDrive folder and expecting those two files to stay in sync --- it doesn't work that way. Only the files in your OneDrive folder are synced --- so you should consider that the primary location for saving/editing/viewing any file you want to be synced.

Keith
