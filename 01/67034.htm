  1. Type Settings in the Cortana search bar and press Enter.
  2. Click Time & language and ensure that the Date and Time are matching your location.
  3. After that, click the Region & language.
  4. Under Languages, click the Add a language and select the language you want to add.
  5. Select the language you want to set as default under the Add a language and select Set as default.

Let us know how did it go or if you need further help.
