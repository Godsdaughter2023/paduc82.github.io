The message that you're getting usually pops-up when Outlook is not finished indexing all the items yet. You can check the indexing status if it's already complete. Just simply go to these options:

Outlook has finished indexing all of your items.
0 items remaining to be indexed.

Mostly keyboard backlight stops working when ATK package is not properly installed or if installed not properly configured on the computer, or if keyboard hotkeys function is turned off from BIOS (Basic Input Output System).

Let’s first check the ATK hotkeys status in programs and features, if the hotkeys does not appear in programs and features try to download and install the ATK hotkeys properly and check after restarting the computer. You may also check uninstalling and reinstalling the ATK hotkeys once again.

  2. Enter control panel in the search box, and then tap or click Control Panel.
  3. Under View by:, select Large Icons, and then tap or click Programs and features.
  4. Tap or click the program, and then tap or click Uninstall.
  5. Follow the instructions on the screen.

After successfully uninstallation of ATK hotkeys, download the latest version of ATK package from ASUS website and install and check.

If the issue persists, I would suggest you to contact ASUS or a local tech shop to check if the feature is turned off from BIOS.

To access the BIOS, you need to either contact ASUS Technical Support or refer to the user manual that came with your computer. Changing any settings in the BIOS is very risky and is always recommended to be done by an expert.

4 year old surface pro 2 - SSD fail ! Need to recover data - Microsoft Community

sneachtapix say:

4 year old surface pro 2 - SSD fail ! Need to recover data

Switched on the pro 2 and it wouldn't boot. Recovery wouldn't boot up either.

Long story short I managed to boot into ubuntu via external usb and run testdisk, Testdisk reported errors on all cylinders and no partitions found.

Is there any hope ???

I'm so disappointed in the SSD failing after only 4 years, worried about my pro 3 now

Thanks for any assistance

james vinyard say:

No!  Your data is going to be gone since the file allocation table is totally corrupted.  Your data will be scattered all over the place and nothing will be recoverable.

However, you may be able to rescue the machine.  If Ubuntu recognized the SSD card, you may be able to save it.  Boot up Ubuntu again and let it load in "live" and when you get to the desktop, run the "Disks" utility.

Within the Disks utility, first, click on the SSD card to highlight it and then click on the 'hamburger' icon on the upper right corner of the utility frame and select the top function"Format".

If all goes well, it will format in the standard universal "Windows" format and the rectangular box should display only a single partition with 100% of the storage displayed.  If this works, the machine is totally wiped out and the SSD card is good.  If it does not, the SSD card did fail and no need to go any further (the machine will be junk).

If it did work, you will need another USB pendrive and with another Windows computer, you will need to download the recovery image for the Surface Pro 2 (https://www.microsoft.com/surface/en-us/support/warranty-service-and-recovery/downloadablerecoveryimage)

Extract the contents of that zip file onto that new USB pendrive and set it aside.  You can now shut off the Pro 2 and remove the Ubuntu USB pendrive and insert the recovery USB pendrive and hold down the Volume Down and the power buttons at the same time like you did before and select recovery from an external drive.

Your data will be totally gone but you will at least be able to save the machine.  If the SSD card had totally failed and did not display in the "TestDisk" or "Disks" utilities of Ubuntu, you could have just chucked the machine in the trash at that time.  You cannot work with a dead SSD drive.
