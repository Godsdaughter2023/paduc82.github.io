You have a printer which uses software built with .NET 2.0. If you uninstall .NET 2.0 the printer software will no longer work.

So yes, just leave them alone.

The "Tuner not found" problem can occur when Windows Media Center canot start the Windows Media Center Receiver Service for any reason. This could be a lack of permissions, a faulty service registration, etc.

Try running Windows Media Center with elevated privileges. Shut it down and then click Start (or tap the Windows key) and type media but do not hit <enter>. Locate Windows Media Center from the list above and right click it and select Run as Administrator from the menu.

With Media Center running, click Start (or tap the Windows key) and type services and hit <enter>. Scroll down the list in the right pane until you reach the three Windows Media Center services.


Figure 1.0 - Windows Media Center Services

All three services (Receiver, Scheduler and Network Sharing) are configured to start Manually. This allow Media Center to start and stop them as needed.

If Media Center starts the Receiver Service and recogizes your tuner now (with elevated priviledges), then the whole problem would point to permissions. You might be able to modify the program menu item Properties to permanently Run as Administrator but further investigation as to what actual permissions are lacking may be warranted.

Click Start > Computer > see which is the problem Drive with low disk space.

 

If it is the Recovery Partition:

Your Recovery Partition was imaged on your Hard Drive during the Manufacture of your Computer by the Manufacturer for these reasons.

 

1.To reinstall Vista from in case of a System crash, by pressing particular Key or Keys at Boot.

F10 or F11 or Alt + F10 are a couple of sequences different Manufacturers use.

You will have to ask your manufacturer for the correct sequence.

 

2. To make Recovery Disks from in case your Hard Drive fails, so that you can then reinstall the Operating System on to a new Hard Drive.

Also ask them how to make the Disks.

 

Your D: or E: Recovery Drive is not to be used for backups, defragged, etc.

Backups should be kept on an External Hard Drive.

 

To fix any problems that have occurred with the D: Recovery Partition, you will have to contact your Computer Manufacturer to remove whatever has been added to it.
