Once you're in the Command Prompt already, type Bootrec / fixmbr and press Enter. Type Bootrec / fixboot and press Enter. Type Bootrec /rebuildbcd and press Enter. Close the Command Prompt window, then restart your device.

Inicialização limpa.

2. Na janela Configuração do Sistema clique na aba serviços;
