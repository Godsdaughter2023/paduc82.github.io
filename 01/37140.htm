Hi

 

You must be very careful when purchasing a version of Windows.

 

If someone is selling Windows, the purchase must include an official COA (Certificate of Authenticity) that contains the Product Key, and Installation Media (DVD).

 

If someone is only selling a Product Key, without the COA or Installation Media, then the chances are very good that the product key is NOT valid.

 

The only thing that you are buying is the License to use Windows which is represented by the Product Key. The physical DVD simply provides a way to install the operating system.

 

How to Tell:
