If you want the convenience of seamlessly switching between Windows and Mac OS X without rebooting everytime, then I strongly recommend you work with VMWare Fusion or Parallels.

If you want the native speed of running Windows 7 on the hardware, then I suggest you use the free built in Boot Camp that comes with Mac OS X and setup a partition to run Windows 7.
