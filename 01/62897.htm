For Upgrade when you first upgraded and it activated it assigned a permanent Digital License to the PC in Microsoft Activation Servers so you can reinstall any time and in any way you want as long as you reinstall the licensed version. Skip the key during install and it should reactivate soon after going online.

If you change hardware follow the steps here: https://support.microsoft.com/en-us/help/20530/.... As long as the original license that was upgraded was not factory or builder's OEM it should allow it to move. If you sign in with the MS Account that is linked to the Digital License and run the Troubleshooter it should automate this, and even if OEM may make an exception if all else is correct.

Factory installs have the Product Key embedded in BIOS or UEFI firmware and should activate themselves, but you can check for the key by installing ProducKey and look for OEM-BIOS key: http://www.nirsoft.net/utils/produkey-x64.zip

Retail keys must be reinserted for reinstalls during install or in Settings.

All methods of activation are converting to Digital Licenses to make it easier to change hardware, however Factory and Retail Product Keys will still reactivate Windows if the DL doesn't.

4. If Required enable USB Boot

6. Save BIOS changes, restart your System and it should boot from the Installation Media

To add a Bluetooth enabled device

Once you've verified that a Bluetooth adapter is installed on your computer, and the device you want to connect with is discoverable, you can add most Bluetooth enabled devices to your computer. Here's how:

  1. Open Devices and Printers by clicking the Start button, and then, on the Start menu, clicking Devices and Printers.

  2. Click Add a device, and then follow the instructions.

  3. Click the Bluetooth enabled device you want to add to your computer, and then click Next. If you don't see the device you want to add, make sure the device is turned on and discoverable. If you just turned on the device, it may take Windows several seconds to detect it.

Prior to the migration of Skype and Microsoft last October 2016, we had three options to create a Skype account, via Skype name, Microsoft and mobile number. Accounts that were created under Skype name, in which you can customize Skype name, the way you want to be searched from the Skype directory, Microsoft account are accounts created under email address which we called "Live ID" and accounts created under mobile number.
