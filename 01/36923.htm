The vsv file should not exist if the Virtual PC has shut down correctly.

Note that because AppData is a junction and not a folder if you try to access normally you will get access denied.

To access open windows explorer click the folder icon in the address bar then type c:\User\[user name]\AppData\Local\Microsoft\Windows virtual PC\Virtual Machines\Windows XP Mode.vsv in the address
