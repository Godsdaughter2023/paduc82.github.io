greg carmack - mvp 2010-2020 say:
I'm Greg, a volunteer installation specialist and 8 year Windows MVP, here to help you.

I understand you were able to get Quickbooks 2007 to run on your previous Windows 10 PC but now it is erroring out on Net Framework 1.1 on a new PC.

Since you were apparently able to install NetFx 1.1 previously, one reason it might not install now is the factory bloatware interfering - another reason most enthusiasts won't even run a Factory Install. I am able to install it just now on my Clean Install.

Try the steps given here: https://support.microsoft.com/en-us/help/922377...

You can enable the necessary NetFx 2.0 which is installed in Windows 10 but hidden by typing Control Panel in Start Search, open Control Panel, choose icon view, then Programs and Features, then Turn Features On, select Net Frameworks, then OK, restart PC.

Then if Quickbooks fails to install, try installing it in Compatibility Mode which tricks the installer into thinking its an older version of Windows: https://www.groovypost.com/howto/run-old-softwa...

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.

andre da costa say:
Which version of Windows 10 is currently installed? See how to verify:
https://answers.microsoft.com/en-us/windows/wik...

If QuickBooks is important to you, you probably should rollback to your previous version of Windows 10 and use that release.

My recommendation to users going forward is not to pounce on every new Windows 10 Feature Update that is released. If your system has been working just fine with the current version, wait it out a while or skip it all together. If there is some specific feature in the update you want, then thats an exception, but these feature updates are becoming more difficult to install on older devices; and it sometimes makes you wonder if the inconvenience is worth the hassle just to keep doing the same thing. 

If you look at whats considered the flagship features of Windows 10 1709, they are aimed at newer generation devices like the new Surface Book 2. Otherwise welcome features like Story Remix, which you have to upgrade to 1709 to get, could be supplanted with third party alternatives.
