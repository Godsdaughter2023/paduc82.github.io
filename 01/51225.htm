Your camera can still be functional in some areas and not others and still have issues.  Problems can be due to the drivers, your OS, privacy settings, or 3rd party software blocking access to your camera.  All of these issues would not be due to Skype but would simply disallow the camera from being functional in Skype.

Please, run the DirectX diagnostics tool. Run dxdiag.exe (Start - Run or Win key +R) then press enter or click "OK". Once the diagnostics completes, "Save All information..." to a text file. 

Since there is no place to upload in this Community, upload this text file to a file sharing/storage site like Onedrive.
