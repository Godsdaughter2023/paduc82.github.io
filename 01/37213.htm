You posted in the forum for the mail client program Windows Live Mail, so that's why I replied as I did.

There are no folders involved, neither in Windows Live Mail nor in People. Windows Live Mail has categories and People has groups. These are two names for the same phenomenon, which you can liken to labels. A particular contact can belong to many categories or groups.

In People, import from your CSV file. When the import is complete, type the group identifier (e.g. 1934)into the search box. This should bring up a list of those who have 1934 in the Notes field (it might also bring up others, e.g. if any of your contacts has a phone number starting with 1934, so be careful). Select the contacts in the group; if there are a lot of them, select the top one, then hold Shift and press the down arrow on your keyboard to select the others. Then click Save as a group on the command bar and name the group by its identifier, e.g. 1934.

frederik long say: I have seen a number of posts in which people insisted that they were unable to boot from the CD/DVD drive. Here are the most common reasons for such failures:
- The poster overlooked the prompt to press a key in order to boot from the CD/DVD drive.
- The boot CD/DVD was not a boot disk or was damaged. Using it on another machine confirmed this.
- Some BIOS setting was incorrect. Booting with some other boot CD/DVD (e.g. from here: http://pogostick.net/~pnh/ntpasswd/) confirmed this.
- The CD/DVD drive was damaged or unable to boot from a boot CD or DVD.
- The machine had a hardware problem (e.g. a motherboard problem) that prevented it from booting in any mode.

When you physically disconnect the hard disk then the machine should still boot from the CD. If it does not then you have yet another confirmation that the machine is always booting from the hard disk.
