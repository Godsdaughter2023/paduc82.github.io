  1. Press the Win + x key on the keyboard and select Command Prompt(Admin)
  2. Type sfc /scannow and hit enter.

Method 2.

  1. Press the Win + x key on the keyboard and select Command Prompt(Admin)
  2. Type the following command: regsvr32 /u wdsutil.dll.
  3. Hit ENTER. This will UN-REGISTER your file.
  4. Type the following command: regsvr32 /i wdsutil.dll.
  5. Hit ENTER. This will RE-REGISTER your file.
  6. Close the command prompt window.
  7. Try to upgrade to Windows 10 and check.

Hope this information helps.
