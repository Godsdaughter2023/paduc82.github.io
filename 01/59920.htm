Correct typing errors automatically - Microsoft Community

¡firedog say:

Correct typing errors automatically

Technical Level : Basic

Summary

The Windows Live Mail function Automatically correct common capitalization and spelling mistakes is intended to correct typing errors. However, with English as the input language, it only works for English (United States). If you prefer Australian, British or Canadian spelling, auto-correction doesn’t work. This article provides a workaround for this shortcoming.

Keywords: auto-correct, autocorrect, autotext, typo, typos.

Details

Windows Live Mail includes a function called Automatically correct common capitalization and spelling mistakes when you’re composing a mail message. It is enabled by ticking the box on the Spelling tab of the program options dialogue. This function will automatically correct common mistakes as you type, so that teh becomes the, abotu becomes about and so on. It will also convert chicago to Chicago. Its basis is the old MS Word Auto-Correct List (ACL) that lists the most common typing errors with their correct forms.

Sadly, the function is buggy: with English selected as the input language, it only works for English (United States). If you prefer Australian, British or Canadian spelling, auto-correction doesn’t work. You have the choice, then, of selecting US English as the language for checking spelling (so that colour will be flagged as wrong) and getting auto-correct as a bonus, or dispensing with auto-correct and having your spelling checked in your preferred flavour of English.

However, there is another undocumented feature in Windows Live Mail 2011/2012 that has not been implemented: Autoreplace. This is analogous to MS Word’s Autotext, which replaces a typed string with another pre-determined one. For example, if I type wlm into a mail message, the program automatically substitutes Windows Live Mail for me. The only way of enabling this feature is by inserting appropriate entries into the registry. By combining these two features, then, we can simulate auto-correction in the non-US flavours of English.

Click here

and select Download to get a file that will insert these registry entries for you. Once downloaded, double-click on the file to merge the entries into the registry. As always when editing the registry, set a system restore point first to reverse your actions if something goes wrong. Once the entries have been merged, reboot the computer, then launch WLMail and see whether auto-correction is working.

  Grateful thanks to Peter.R who first spotted the Autoreplace feature in October 2010 and wrote about it in the newsgroup.

Most of the Foot pedal are Plug and play with a USB and should be detected when you connect them to the computer. There are three controls which are usually setup for rewind (right), play/pause (center) and fast-forward (left) but you can configure them.

a.     If you have a CD that you received with the Foot pedal follow the wizard and install it.
