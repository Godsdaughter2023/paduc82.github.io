considerate_guy say:

Here's how I would go about breaking your project into smaller subprojects.

Let's say you want to split it into 4 subprojects. First make 4 copies of your original (.wlmp) project file. Be sure to keep the original as a backup.

Open up the first copy. Keep roughly the first 1/4 of the project and delete the rest. Double-check that you can both save this subproject and the corresponding movie for that part.

Assuming that all saves as expected, open up the second project copy and delete all but the second 1/4 of the project. Save this project and its movie.

Repeat for parts 3 and parts 4.

Assuming all the resulting .mp4 files are playing as expected, start up Movie Maker with a new project and add the 4 separate movies for parts 1 to 4. Finally save that as your full movie.

• If that didn't work, try this:
○ Uninstall your audio driver: open Device Manager, right-click (or press and hold) the audio driver, and select Uninstall. Restart your device and Windows will attempt to reinstall the driver.
