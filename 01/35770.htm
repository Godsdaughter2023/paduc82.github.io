Hi Tariq

 

The Pashto language is not available as a full language pack in Windows.

 

You can add the Pashto Language as a Keyboard Layout which will allow you to type documents in that language.

 

1. Press the Windows Logo Key + X to display the system menu and click Control Panel.

 

2. In Control Panel, select Language.

 

3. In the next window, click the Add A Language button. In the list of languages, scroll down to the 'P' section and select Pashto. Click the Add button.

 

4. On the right side of the Pashto language, click Options. In the next window, click the Preview link to see an image of the Keyboard Layout.

 

5. Close the Control Panel and go to the Desktop. On the right side of the Taskbar you will see the Language Bar. Click this bar and you can select the Pashto option for the keyboard.

 

You can then type your documents in the Pashto language. To switch back to the default keyboard layout, click the Language Bar again and select the Default Language.

 

Let us know if you have more questions.
