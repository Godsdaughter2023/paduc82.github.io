tartar_sauce say:

You will get the error when there's something wrong with the content of your machine.config file.

 

There are two ways to go about fixing this.

 

First take a backup of the machine.config file.

Simple duplicate the file and name it machine.config_bkup or something like that.

 

1. The Correct Solution

Open the machine.config file with Notepad or any other text editor of your choice.

Carefully look thru the files for any odd characters.

It may be as easy as some garbage characters at the beginning of the file.

Make corrections or deletions as necessary.

Save and close the file.

 

If you can't get it to work, try the following.

 

2. The Workaround Solution

Delete the existing machine.config file.

Make a copy of the machine.config.default file in the same folder,

and rename this to machine.config.

This will restore the default configuration.

While this workaround will resolve your installation issue in most cases,

you may also end up losing some configuration details.

(Not that it really matters, since your file already can't be read!)
