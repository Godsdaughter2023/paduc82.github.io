cliffgu_msft5984 say:

hi seyhun,

if you installed a click-to-run version of office 2013 on your computer, you can get a standalone skype for business 2015 via office 2013 deployment tool.

here are the steps.
1. download office 2013 deployment tool via https://www.microsoft.com/en-us/download/details.aspx?id=36778.
2. edit the configuration.xml file via configuration xml editor.
3. define a path to download the skype for business 2015 installation package. in my environment, i define as c:\cliff.
4. the configuration.xml file looks like below.

<configuration>

    <add officeclientedition="32">

        <product id="o365proplusretail">

            <language id="en-us" />

            <excludeapp id="excel" />

            <excludeapp id="groove" />

            <excludeapp id="infopath" />

            <excludeapp id="onenote" />

            <excludeapp id="outlook" />

            <excludeapp id="powerpoint" />

            <excludeapp id="project" />

            <excludeapp id="publisher" />

            <excludeapp id="visio" />

            <excludeapp id="word" />

            <excludeapp id="access" />

        </product>

    </add>

    <updates enabled="true" autoupgrade="false" />

    <display level="full" accepteula="true" />

    <logging level="off" />

</configuration>
5. open command prompt with the administrator role.
6. change the root to cd c:\cliff.
7. run setup.exe /download configuration.xml to download the skype for business 2015 installation package.
8. run setup.exe /configure configuration.xml to install skype for business 2015.
9. after installing, you need to activate skype for business 2015 for your using.

 
for more detailed information about deploying office 2013 via office 2013 deployment tool, please refer to https://blogs.technet.microsoft.com/odsupport/2014/07/11/using-the-office-deployment-tool-for-click-to-run/.

thanks,
cliff

wilfredo_soto say:
Hello Hashbrownpickles

Solution 1 - Rollback the controller

A possible problem with the HDMI video output is that the drivers can get to problems. The most recommended option in this case is to uninstall the driver and restart the computer, Windows 10 will be responsible for reinstalling the driver, sometimes earlier or the default to which it was installed.
1- Open the Device Manager within System in the Control Panel
2- Right click on the video controller and select Uninstall.
3- It is also important to uninstall the driver software, which can be AMD Crimson or Nvidia GeForce.
4- Then restart the computer.

Solution 2 - Update the driver

Sometimes the problem can be solved by updating the graphics card driver to the latest version, instead of reinstalling the driver or returning to an older driver.
In this case we must go to the corresponding pages of AMD or Nvidia (depending on the model of your graphic card) and check which are the most recent stable drivers that are available.

Solution 3 - Installing the drivers in compatibility mode

If we still do not solve the problem, we can use the 'compatibility' function that is present in Windows 10 to install the driver.
1- We will right click on the executable file of the installer and enter Properties
2- Go to the Compatibility tab
3- Check the Run this program in compatibility mode box and select Windows 7 from the list
4- We accept the changes and execute the driver installation

I hope this can help you
