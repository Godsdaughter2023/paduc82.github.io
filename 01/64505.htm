considerate_guy say:

To make a DVD with multiple scenes, do the following:

  1. Create a project for each scene.  In your case you would have 2 projects.  In each project, put one scene, make all your edits, save the project, and then save the movie to your computer.  Repeat for each of the other scenes.
  2. Start up Windows DVD Maker.  Give it all the movies that you created in step 1.  Under the "Options" link in the lower right, make sure that you have the menu set to play first.  Type a type for your DVD instead of the date.
  3. Choose your menu style and then preview the results.  To see your two scenes, click the 'Scenes' button on the menu.  Then pick which scene you want to see playing.  You should be able to pick the second scene.
  4. If that all previews correctly, burn it to DVD.  When playing it in your DVD player, use the arrow and enter buttons to pick the 'Scene' menu and then the desired scene to play.

which one is stronger ? Please make them in order :

1- WEP 64 bits

2- WEP 128 bits

4- WPA2-PSK

5- WPA-PSK/ WPA2-PSK

 

And which one I should use :

1- AES

2- TKIP

 

patrick barker say: Hi,

Which security / authentication method you choose solely depends on whether or not the network and the devices attempting to connect to said network support the method in which you want. For example, not all devices support connection and authentication to WPA2-PSK so you'll have to use WPA-PSK, etc.

WEP is by far the weakest, and in today's security standards I would never recommend using it. Use WPA2-PSK with AES if your network and devices support it. If not, fall back to TKIP. If WPA2-PSK is out of the question entirely due to device and or network restrictions, use WPA-PSK with AES / TKIP.

AES offers stronger encryption however not all devices support it. TKIP is a little less stronger in terms of encryption but is widely supported by many devices on the market.

If you are using Windows Media Player 12, it monitors your Music library so that you don't have to manually add music for Windows Media Player to play it.

To add a folder to the Music library, simply right click the folder you want to add, select Include in library, and then select the library you want to add the folder to (in this case Music).

Now, when you open Windows Media Player, it will scan that folder for Music and add it to the Media Library.

 

 

Follow these steps:

Step 1: Transfer music to Windows Media Player manually:
a. Open Windows Media Player.
b. Go to "File" drop-down menu and select "Open."
c. Select the music you've downloaded from Frostwire.

Step 2: Configure Frostwire to transfer music automatically
a. Open the Frostwire application.
b. Click the "Frostwire" drop-down menu.
c. Select "Preferences" from the menu. Another window will open, allowing you to browse Frostwire's different preference options.
d. Click "Saving" to show the various options for saving files once they've been downloaded from Frostwire.
e. Click the "Browse" button and search your computer for the music folder used to store songs for your media player.
f. Highlight the folder you wish to save your Frostwire downloads to and click the "Open" button. The window will close once your selection has been made.
g. Click "Apply" in the "Preferences" menu. This will store all the changes you've made.
