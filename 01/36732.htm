steve winograd say: I'm sorry, but Windows 7 Home Premium can't accept Remote Desktop connections from other computers.  That requires the Professional or Ultimate version.

All versions of Windows 7 can connect to another computer using Remote Desktop Connection.  That's what the Help file indicates.

These programs let Windows 7 Home Premium accept remote connections from other computers: TightVNC , TeamViewer , GoToMyPC .
