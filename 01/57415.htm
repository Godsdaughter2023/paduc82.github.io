What game ? Which model Havit controller ?
Most modern games use XBOX (XInput) controller by default, but old games use Dinput, and were too early to have

an Xinput option, so modern XBOX type controllers only work with the old games if they have a switch for both Xinput and Dinput.

available in the game. This software is provided by the mfg of the controller.
