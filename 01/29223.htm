All the different movie maker versions and video formats
and codecs can be quite confusing.

First...When you burned the DVD in Vista...did you use
Windows DVD Maker to create a Video DVD or did you
simply burn a Data DVD? The way to tell the difference
is to look at the directory of the DVD...if it has a VIDEO_TS
folder that contains VOB files...it should play on a free
standing DVD Player. (However...some DVD Players will
not play a home made DVD).

See the following article.

Windows Vista - Burn A DVD Video Disk
http://windows.microsoft.com/en-US/windows-vista/Burn-a-DVD-Video-disc

As for Windows 8.1 Movie Maker 2012 ...if you save the
project in the .WMV  format it should play in Windows
Media Player.

How to save as movie in Movie Maker 2011, 2012
Previously Posted by  Leonard_Su

1. Open Movie Maker and open the project file you have (.wlmp)

2. In the program window, press these keys on your keyboard ALT + F + select
Save Movie and choose Recommended for this project.

3. After that choose your preferred destination where you want to save the
movie.

4. Edit the movie file name and the file format of the movie. (You can
choose .WMV or MPEG-4 if you are using the latest version of Movie Maker
2012. If not you can only choose .WMV)

5. Click Save button.
===
