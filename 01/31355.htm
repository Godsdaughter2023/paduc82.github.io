Method 1: Make changes in Network sharing settings and drivers.

Step 1: Turn off sharing.

  1. Go to Control panel > Click on Network and Internet > Click on Network and Sharing.

  3. Turn everything to OFF on all profiles and options.

  4. Save the changes and then close the applet.

Step 2: Uninstall and reinstall the drivers.

  1. Press Windows andR key together on your Keyboard.
  2. Type devmgmt.msc and hit enter.
  3. Click on View and select Show hidden devices.
  4. Right click on Network adapters.
  5. Uninstall Ethernet and Wireless adapters.
  6. Then click on Action and then select Scan for Hardware Changes.
  7. Allow reinstallation of the devices and then close Device Manger.

Step 3: Turn On sharing.

Follow the step 1 and Turn everything to ON for all profiles and options. Click save changes and close the applet.

Method 2: Modify Local security policy.

  1. Press Windows and R key together on your Keyboard.
  2. Type Secpol.msc and hit enter.
  3. Navigate to: Local policies -> Security options -> In the right pane, search for Network security: Lan Manager…
  4. Double click on Network security: Lan Manager… and select Send LM & NTLM- use NTLMv2 session security if negotiated.

obsessive power say:

Hi,

If you're looking to record and have a high FPS while doing so on your system, then I'm afraid you're going to be disappointed. 

The problem with screen recorders like the ones mentioned is they do all of their video encoding in software, which means a huge chunk of your laptop's CPU resources would have to be dedicated to this process. Hence the cut in performance during game play.

To allow hardware based gameplay video recording, the GPU needs its own built in H.264 encoder - unfortunately, I don't think the 150MX has this. 

You could try nvidia's 'ShadowPlay' which is built into their Geforce Experience software. If shadow play is locked out for you, you could try this guide that might get it to work on your GPU.

If that doesn't work, or doesn't provide the correct performance, I'm afraid nothing can be done, at least not on one computer.  If you had two PC's, you could stream the game to the second PC and use its hardware to encode it.  That's the only way you could do this with your set-up without sacrificing performance.  Or buy a more powerful laptop or PC.

Hope that helps.

the larch say:

Weird.  If you have really exhausted all the repair-type options (other people can reply what they are), then you can consider turning off the indexing process.  Search still works except only Control Panel items and programs will show up in the Start menu search results, so you'd have to search from a window such as for your hard drive (Start, Computer, etc.).

 

From my notes:

How to turn off Windows 7 Search indexing:

Windows 7 Search still works without indexing (except in Libraries windows).  To turn indexing off, click the Start button, Control Panel, Administrative tools, Services, scroll down to find Windows Search in the window that appears and doubleclick it, and click the Stop button.  Indexing is made inactive for all user accounts.  To stop indexing from starting again, click the down-arrow button by “Startup type:”, click Disabled, and click the OK button.  (To turn indexing back on, change “Startup type:” to Automatic, then click the Apply, Start, and OK buttons.)  The Start menu Search box will then give results limited to programs and Control Panel items (any new programs installed will be searchable regardless of the indexing being off); no document files of any kind show up.  Searching in windows’ Search boxes behaves as before, but any changes that you might have done in Indexing Options will no longer be effective.  Also, the “Arrange by” feature in the Libraries windows will no longer work.
