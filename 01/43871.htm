tryx3 say:

1  To open Windows explorer on the folder you want, get to the desired folder then right-click, select Create a shortcut.  Then drag the shortcut to the Desktop & rename if you want.  You can leave it on the Desktop or add it to the Start menu, All programs folders wherever you want [hover over the entry All programs then right-click, Open to see the folders then drill down through to whichever folder you want to add it to].

2  You cannot Pin this shortcut to the Start menu or Task bar directly.  To do this is a bit more involved.  Go to the Start menu, All programs, Accessories & right-click the link to Windows explorer, then select Copy.  Paste this link on the Desktop then right-click & select Properties.  In the Target field add a space to the end of the existing entry then put in the path to your user folder & put in quotation marks if there are any spaces in the path.  You'll end up with this in the Target field

%windir%\explorer.exe "C:\Users\YourUserName\My Documents"

or

%windir%\explorer.exe "C:\Users\YourUserName\Documents"

[both of these do exactly the same thing]

Then you rename the shortcut to something specific so you don't get it mixed up with the normal Windows explorer shortcut then right-click on the shortcut & select Pin to Start menu or Pin to Task bar.  You can then delete the shortcut [or store it away somewhere] as the Pin operation will have created a copy of it in the appropriate settings.

Firmware Update

Login Information

  • Gateway: 192.168.0.1
  • Username: admin
  • Password: (located on the bottom of the device)

How to Find Firmware Version

  1. Open a web browser on a computer.
  2. Type the gateway address: 192.168.0.1 into the web address bar, select apply.
  3. This will take you to a login page, type admin for the username and password (located on the bottom of the device).
  4. Select Modem Status on the main page.
  5. On the next page the current firmware will be located next to Firmware Version under “Modem Status”.

 

Firmware Update

Note: Firmware will be provided to you when you check for firmware update.

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the gateway address: 192.168.0.1 into the web address bar, select apply.
  4. This will take you to a login page, type admin for the username and password (located on the bottom of the device).
  5. Select Utilities on the main page.
  6. On the left hand side select Upgrade Firmware.
  7. Select the Download button to begin the firmware download to your computer.
  8. Once the download has completed select the Choose File or Browse option.
  9. Once you have selected the firmware downloaded to your computer. Select the Upgrade Firmware button to begin the firmware update.
