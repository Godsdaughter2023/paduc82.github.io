dena b say:

Hi coolpolitealex,

Windows Media Player is part of Windows. You cannot completely uninstall Media Player. You can 'turn off' Windows Media Player by following the steps below:

  1. Click on Start
  2. Select Control Panel
  3. Select Programs and Features
  4. Select Turn Windows features on or off
  5. Expand Media Features
  6. Uncheck Windows Media Player
  7. Select Yes
  8. Select OK
