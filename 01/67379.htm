Hello Ron,

If you're able to access the internet through WiFi but are unable to connect via Ethernet cable, the problem most probably is not caused by network issue. It can be caused by hardware issue, driver issue or even your system's settings. You can perform the following methods to address the issue:

Method 1: Use a different Ethernet cable on different ports on the PC and router

Since you've already checked if the Ethernet cable you were using before was not defective, you can further isolate the issue by using a different (working) Ethernet cable on different ports on your computer and router. This is to check if the issue is caused by faulty ports. If you've verified that the issue is not with the ports, proceed to Method 2.

Method 2: Temporarily turn off any Antivirus or Firewall

Some Antivirus programs or Firewall can prevent your computer from connecting to the local network. You can temporarily turn them off to see if this resolves the problem. If not, proceed to Method 3.

Method 3: Make sure the Ethernet is enabled

You can follow these steps to check if the Ethernet is enabled:

  1. Press Windows key + R to open Run dialog box.
  2. Type devmgmt.msc then hit Enter.
  3. In the Device Manager, expand Network adapters category.
  4. Next to the Ethernet card device name, check if you see a computer icon with an arrow on it. If you do, the Ethernet has been disabled.
  5. If it's not disabled, proceed to Method 4. If it is disabled right-click on the device name then click Enable.

Method 4: Make sure the network card driver is functioning correctly

In the Device Manager, check if there's a yellow exclamation mark or question mark next to the network device name. If there is, you need to update the driver.

If you need further assistance, let us know.

The Bluetooth option requires additional hardware and cannot just be enabled, it is difficult to see how a computer can be "messed up" by installing an adapter.
