What are these "registration codes"? 
They are commonly used for software you purchase through the Internet which is restricted in some way until the code is entered. For example, a virus checker may only work for a month then is disabled without the code. It is a way of trying the software out freely to see if it meets your requirements. Every software vendor I know (including Microsoft) uses this system (also known as an Activation code).
