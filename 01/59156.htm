waseemulla shariff say:
Hi Harland,
 
As you've scheduled the scan at 5:00 AM every day, the system has to be turned ON at the scheduled time and allow the scan to complete. If the system is turned OFF or if the system is shutdown during the scan, we come across this issue. However there is a setting that can be applied for the scan to start automatically when the system starts.
 
To enable this setting, try these steps.
1. Click Start, type taskschd.msc and press enter.
2. Click on Continue in the UAC prompt.
3. In the right pane, click on View and click on Show Hidden Tasks to check it. (See screenshot below step 7)
4. In the left pane, click on the arrow on the Task Scheduler Library folder and Microsoft folder to expand them.
5. Click on the Windows Defender folder.
6. In the middle pane, right click on the listed MP Scheduled Scan and click on Properties.
7. Click on the Settings tab.
8. Check the "Run task as soon as possible after a scheduled start is missed box."
NOTE: This will allow Windows Defender to run the scheduled scan the next time you have the computer running if the scan was missed due to it not being able to run at its scheduled time.
9. Click OK button.
10. Close task scheduler window.

For more information on "Using Windows Defender" check this windows help link: http://windows.microsoft.com/en-US/windows-vista/Using-Windows-Defender
 
Hope this helps.
