techfreak_ say:
Unfortunately, you cannot. The video feed is mirrored locally and has been hard-coded in that fashion for the majority of Skype's existence. The only difference in the latest versions is that the local preview window in settings is also mirrored (it was not in prior versions). So in Skype 8.x/14.x there is no non-mirrored local output. If you attempt to mirror the mirrored output it will only display the feed normally for you and mirrored on the other end, which will probably not suffice. The only way to have a non-mirrored local output is to use other software to display the feed non-mirrored locally in that other program and still allow sharing the video feed with Skype .
