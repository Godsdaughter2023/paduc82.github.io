a. user say:

Is your task bar taking up 1/3 of the screen and no matter where you click or drag it won't resize?

Ok here's the solution...

There is a bug with windows when it resizes for different applications here's why. Say the user opens a program that render in a lower screen resolution then windows is currently set to.  Windows auto corrects and displays the largest resolution based upon the program's capabilities.  Once the program is closed windows does not resize correctly leaving you with this really big task bar issue.  TO FIX - First right click on task bar and ensure "lock the task bar" is NOT checked.  Right click the task bar again and choose "Taskbar Settings" then make sure "Automatically Hide the Task Bar in Desktop mode" and "Automatically Hide the Task Bar in Tablet mode" is OFF.  This will now turn your mouse into a double arrow cursor when you hover over the top of the task bar.  Click and drag to resize.
