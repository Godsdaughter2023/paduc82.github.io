Method 2: I would also suggest you to stop the print spooler and delete all spooled files. By doing this, you should be able to print.

a) Click on Windows Vista/Windows 7 Start button and type Command in start menu search field.

d) Then type “del %systemroot%\System32\spool\printers\* /Q” without the quotes then press Enter.

e) Then type “net start spooler” without the quotes and then press Enter.

f) Type exit and press Enter to exit the elevated command window.

msb0742 say:

I found a fix. This has to do with the fact that at some point you altered your icons using a program like icon packager and the upgrade got confused about it. 

Please access your System Registry (WinKey +R) and in the run box type regedit. Then go to the following Key;

HKEY_CLASSES_ROOT\exefile\shellex\IconHandler

On the right side of the screen, if it has a value of {611AD258-4138-4348-A534-9856FA6BA398} you need to rename that registry key from 'IconHandler' to something else, IconHandlerX works.
Then reboot your PC and the Icons should come back.
