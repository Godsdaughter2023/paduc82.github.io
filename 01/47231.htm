phaedra1507 say:

Compacting deleted email messages in Windows Live Mail

I have been told that compacting my deleted email messages in Windows Live Mail completely removes them from the computer.  Is this correct?  And, if so, why is the word "compacting" used when it doesn't mean removal .... only squshing them. 
¡firedog say:

The setting on the Maintenance page (Options > Advanced) reads Compact the database on shutdown ... When this 'compaction' is underway, the progress window reads Recovering unused disk space. There is no mention of Compacting deleted email messages.

All messages are stored as individual files. The database is an index containing the headers of each message along with information about the message's location and status. Occasionally, that information gets out of date in respect of some messages. The historical reasons - as 'compaction'.

If during this housekeeping process, the database discovers that a message has been deleted, all references to it are removed. This is the 'unused disk space' that's recovered, at the rate of 2-3KB per message deleted. There is no need to keep the headers of a deleted message in the database.

A deleted file is marked as such in the Windows file system, meaning that the disk space it occupies is available for other purposes. Whether or when that space gets used so that the deleted file is overwritten depends on many factors, so it may be some time before the file data are gone completely. This explains how undelete utilities can sometimes recover deleted files.

Here's how you stop web results from showing up on every search you do on Windows 10 while improving local results for files, apps, and settings:

  2. Click the Notebook icon.
  3. Click Settings from the list of options.

Note You can say “Xbox, go to settings” from any screen, but you’ll need a controller once you’re on the Settings screen.

5.Select Turn Xbox Off.

6.Select Yes. The console powers down.

7.Turn your console back on by pressing the Xbox button on the console or the Xbox button on your controller.

Note If you don't see the green boot-up animation when the console restarts, repeat these steps. Make sure that you hold the power button until the console completely shuts down.

Physically power cycle the console

1.Turn off your console by holding the Xbox button on the front of the console for approximately 10 seconds. The console will power down.

2.Turn your console back on by pressing the Xbox button on the console or the Xbox button on your controller.

Note If you don't see the green boot-up animation when the console restarts, repeat these steps. Make sure that you hold the power button until the console completely shuts down.

Important If your console is in Instant-On power mode, the above steps will shut the console completely off. Instant-On mode, or the ability to turn on your console by saying "Xbox on", won't be enabled until you restart the console.
