Hello,

This error message usually appears when you are connecting to the Internet using a dial-up modem. Also,
this error may occur if the communication (COM) port used by the modem has been reserved by another software program, or if multiple Internet connections have been set up. As an initial troubleshooting step, you can uninstall and reinstall the modem driver. Check out the steps below:

  1. Press Windows key + R on your keyboard.
  2. On the run command box, type devmgmt.msc, and then hit Enter.
  3. Expand Modems driver. (If Modems is not available from the list, check the Modem driver on Network Adapter driver.)
  4. Right-click on the modem name and then click Uninstall.
  5. Click OK.
  6. Restart your computer. The Modem driver will automatically reinstall after the restart.

Should you need further assistance, feel free to get back to us.

The operating system is irrelevant. If the PCs are networked then you can copy your data across via the network. If they are not networked then you can use a USB flash disk ("memory stick") as a transfer medium.

There are several approaches to doing this. Which is best for you mostly depends on how much data you have to transfer. Here are some choices:

1. Transfer them on CDRs or DVDs.
2. E-mail them from the old machine and read the E-mail on the new one.
3. Network the two computers, and just copy the files over the network.

5. Use a USB thumb drive.

6. Use an External hard drive.

Safety is not an issue with any of these ways.
