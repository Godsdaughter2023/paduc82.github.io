With Windows 8, Microsoft has changed from stickers that have the product key that the user has to type in when installing the operating system to new BIOS embedded product keys. The idea is that by eliminating the sticker, you eliminate one of the easier ways for nefarious users to get a legitimate product key. Make sure the hard drive is set as a slave drive if there are any settings on the drive itself.  Once installed boot windows and go to Start / Control Panel / Administrative Tools / Computer Management / Disk Management and see if you can see the new drive.  It may be there but not named or given a drive letter.  Right click on that drive and make it Active and assign it a drive letter.  Then hover your mouse over the drive an a text box will appear with some information.  Please post that information EXACTLY as it appears in the same order it is presentsd.

If the drive does not show up in Disk Management, go to the drive manufacturer's web site and download the drivers, firmware and/or software for your particular drive and your version of Vista and then install these updates (even if they seem to be the same ones installed).  Reboot and go to Start / Computer and see if the new drive appears.  If not, try the above procedure again to see if it appears in Disk Management and can be configured.

Go to Start / Control Panel / Device Manager and see if the new hard drive is listed and if there are any error symbols next to it.  If it's there double-click on it to see more details.  Confirm that it says the device is working properly.  If the device is not listed then even the device drivers are not installed or recognized let alone the device itself.  That suggests a compatibility issue even if you updated the drivers (they did not install properly for some reason).
