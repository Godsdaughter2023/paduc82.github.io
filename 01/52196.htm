Login Information

 

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

Change Wireless Channel

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Advanced" from the sub menu.
  6. Select the desired wireless channel from the "Channel" drop down menu. (To allow the device to select the channel automatically, select "Auto".)
  7. Select Save/Apply.
