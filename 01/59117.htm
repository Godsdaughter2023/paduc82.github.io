Looks like Documents and Pictures are hidden by setting the "ThisPCPolicy" value to Hide. (Ref: Ronnie's link above)

Copy the following lines and paste into Notepad:

'-------------------------


[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{0ddd015d-b06c-45d5-8c4c-f59713854639}\PropertyBag]
"ThisPCPolicy"="Show"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{f42ee2d3-909f-4907-8871-4c22fc0bf756}\PropertyBag]
"ThisPCPolicy"="Show"

'-------------------------

Save the file with .reg extension ("somefix.reg").

Double-click the file to run it.

Close File Explorer and re-open it.

You can then right-click Documents and click Properties, Location tab. Click Restore Default.
