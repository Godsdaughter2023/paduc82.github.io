Reset "ghost" or nonpresent COM ports with devcon.exe__ - Microsoft Community

jdr2010 say:

Reset "ghost" or nonpresent COM ports with devcon.exe__

Hello,

I would like to "reset" ghost or non-present COM ports using devcon.exe in a batch file.   

The issue is that a device that is attached to USB port and mapped to COM port seems to go "bad" periodically.  I am trying to remove the ghost port then rescan for hardware changes and hopefully be able to pick up the device again in the hope that the device will continue functioning properly.

This device is being used at a hospital facility and it will continue to work when the issue occurs if it is manually unplugged and plugged into another USB port.  However the facility would like to eliminate the need for this manual intervention.

I put together the batch file below.  The devcon. exe command picks up and displays the ghost ports, but it will not remove them.  For the line that has the devcon remove %%a\%%b I get "No devices removed".

Any help, hints, or confirmation that this cannot be accomplished with devcon.exe is much appreciated.

Thanks in advance - jd

@echo off
::COM port info goes to cr.dat file
::XP SP3
::devcon.exe needed, download from
::http://support.microsoft.com/kb/311272 if needed

set DEVMGR_SHOW_NONPRESENT_DEVICES=1
ping 127.0.0.1 -n 5 >nul

devcon findall =ports | find "ECP Printer Port" > cr.dat
::look for card reader port and send to a temp file
::devcon findall =ports | find "USB Serial Port" > cr.dat

::if no COM ports found for card reader exit program
if %errorlevel% NEQ 0 (
  echo %errorlevel%:  ***NO PORTS FOUND TO RESET!***
  goto :EXIT
)

set PORTRESCAN="N"
::parse COM port data in cr.dat and reset COM port
for /F "tokens=1*" %%i in (cr.dat) do (
    echo.
    echo *** *** ***
 for /F "tokens=1,2 delims=\" %%a in ("%%i") do (
    echo Resetting %%a\%%b %%j
    devcon remove %%a\%%b
    IF %errorlevel% == 0  (
      echo DONE!
      set PORTRESCAN="Y"
    ) ELSE (
      echo ***%errorlevel%: ERROR RESETTING PORT!***
    )
   ping 127.0.0.1 -n 3 >nul
 )
)

:: rescan ports
if %PORTRESCAN% == "Y" (
  echo Rescanning COM ports ...
  devcon rescan
  ping 127.0.0.1 -n 3 >nul
)


:EXIT
:: remove cr.dat
del cr.dat
echo *** END OF PROGRAM ***

    If your trying to turn off the automatic renewal on a free trial membership, you can do so on xbox.com by signing in with your Microsoft account and password. Then selecting my account option in the top right after being signed in. Once your account management section loads up, there will be an option in the middle of the page to turn off your auto renewal so it doesn't renew and continuously bill you. If your trying to get rid of the gamertag because it was an uneeded one and was accidently created, you will have to close out the Microsoft account. This is done by going to account.live.com and signing in with the uneeded gamertag. Once your on account summary, all the way at the bottom it gives you the option to close the Microsoft account, Click on that button and that is going to be the only way to delete the gamertag and it free up the email address you were using if you were trying to use the email on a different account. Sometimes that option doesn't work and it will say "you will have to gid rid of any memberships and spend all your Microsoft points before you can close your account" and if this happens when your trying to do it, then your only option is to call support and ask that they do an escalation to get your gamertag removed.
