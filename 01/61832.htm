  2. Turn off Wifi

  5. Go Back to Network & Internet and Turn Wifi back on
  6. Select the Network which has repopulated
  7. Enter the network key (which I now have the option of doing)

tryx3 say:

You can check compatibility at http://www.microsoft.com/windows/compatibility/windows-7/en-us/default.aspx and there's a downloadable system checker there that you can run on each of your old systems.  There's some more info at Will it work on my PC? as well.  You should also consider, at this stage, which version of Windows 7 you would need - see Compare Windows vs. Windows

I'd also recommend checking on the PC maker's support site for Windows 7 upgrade advice. 

As for your existing problems, if XP is at SP3 and Office 2007 is at SP2 then I'm surprised you are having any issues. 

The question of which PC to buy depends on what it is that you want to do with it. 

e.g. 1 If you are sure that you will not need to be mobile over the next five years then you can stick with a Desktop. 

e.g. 2 If you want a large screen you can stick with a desktop or get a laptop with an external monitor for it. 

e.g. 3 If you need lots of RAM for graphics applciations [see 32- vs. 64-bit Windows ] then that could also guide your choice. 

If everything you want to do can be done with your current machine [when it's working properly] then stick with what you've got and solve its problems instead.
