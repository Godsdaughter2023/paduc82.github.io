Hey there!


Xbox 360 consoles themselves were not region locked however depending on where you purchased it, there may be a difference in NTSC vs PAL. If you acquired your console in China, I believe it will be a PAL console which will not be compatible with American games as we use the NTSC format.

steve cochran say:
Well, a number of things may have happened.  As Linney indicates, antivirus software could have interfered and wiped out the messages.  Alternatively the database got corrupted and the messages are still there, but the ones that are zero byte were the ones that showed as blank and there are others that are not zero byte.
 
I would check the original message store directories where the messages should actually be stored (seewww.oehelp.com/backup.aspx#wm).  So under the Local Folders, the directories that correspond to the subfolders should have eml files that are non-zero bytes.  Anything that is zerobyte is blank.  Anything otherwise is likely a real message. 
 
But if as you say, they are all zero bytes in terms of what is left, then something did wipe them and I don't think there is much you can do, if there is no backup.  You can try searching for *.eml and include hidden files and folders in the search and see if they turn up in another location.  If that doesn't work, as a last resort you can try a system restore if you have a point you can restore to where you had those files.  But otherwise, I don't see much you can do in terms of recovery.
 
You can try my WMUtil program to compact and repair the database, but that won't recover the messages. (www.oehelp.com/WMUTil/)
 
Finally, at this point, given you are likely to have to combine all the accounts again, I would check what antivirus you have and make sure you have something that is non-invasive such aswww.microsoft.com/security_essentials/ and then set up a clean message store (just rename the old Windows Mail directory and then WinMail when it starts will create an empty one.  Then you can import again what you need, and backup frequently.  
 
Just as a plug, my OEX program might help you import more easily as it can import across a network of PCs (seewww.oehelp.com/OEX/)
