Que faire si j'ai oublié le code d'accès que j'ai défini pour les contrôles parentaux ?

Si vous avez oublié votre code d'accès, vous pouvez le réinitialiser en répondant à la question de code d'accès définie lors du paramétrage des contrôles parentaux. Procédez comme suit :

  1. Sur la console, accédez à Paramètres, puis sélectionnez Famille.
  2. Sélectionnez un paramètre de contrôle de contenu à modifier.
  3. Lorsque vous y êtes invité, saisissez un code d'accès quelconque (même incorrect).
  4. Lorsqu'apparaît la notification signalant que ce code d'accès est incorrect, sélectionnez Réinitialiser le code d’accès.
  5. Lorsque votre question de code d'accès s'affiche, utilisez votre manette ou votre clavier Messenger pour saisir la réponse, puis sélectionnez Terminé.

Si vous avez oublié la réponse à votre question de code d'accès, vous devrez contacter le support technique Xbox pour obtenir de l'aide.
