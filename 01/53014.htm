Copy below > Windows Key+R and paste into Run window and Enter

Rename it to something else say Microsoft.MicrosoftEdge_8wekyb3d8bbwe_Old
Now, Edge will no longer work.
In future if you need to enable it again, just restore the name of the folder back. So, I will remove _old from the folder name above.

Note - A feature update may bring back Edge. In that case, you will need to do the above steps again.

2. Unlike majority of the default Windows apps, Cortana can not be disabled through Powershell. Microsoft wants people to use Cortana.

If you want to completely turn off Cortana, then you will need to resort to a registry fix (Note - It is recommended that you create a backup of your registry before modifying so that if you commit some mistake, you will be able to restore - https://support.microsoft.com/en-us/help/322756... )

Windows Key+R > Type regedit and Enter
Go to the key - HKEY_LOCAL_MACHINE > SOFTWARE > Policies > Microsoft > Windows
See if you have a directory named - Windows Search
If not, then right click on Windows folder > New Key > Give the name Windows Search
Click Windows Search > In the right-hand side pane, right-click in the empty area and choose New > DWORD (32-bit) Value. Type AllowCortana > Enter
Double-click AllowCortana, and type in 0 under Value Data.
Reboot your computer

baltyofgod say:

Per utilizzare l'aggiornamento di sistema offline, è necessario quanto indicato di seguito.

  • Un PC basato su Windows con connessione Internet e una porta USB.
  • Un'unità flash USB con almeno 4 gigabyte (GB) di spazio disponibile formattata come NTFS.

La maggior parte delle unità flash USB è formattata come FAT32, pertanto sarà necessario riformattarla come NTFS. La formattazione di un'unità flash USB per questa procedura comporta la cancellazione di tutti i file nell'unità. Prima di iniziare questa procedura, trasferire i file dell'unità flash USB o eseguirne il backup. Per informazioni su come formattare un'unità flash USB come NTFS utilizzando un PC, vedere Come formattare un'unità flash come NTFS in Windows.

  1. Collegare l'unità flash USB a una porta USB del computer.
  2. Aprire il file di aggiornamento di sistema offline OSU1.
  3. Cliccare su Salva per salvare nel computer il file con estensione zip contenente l'aggiornamento della console.
  4. Decomprimere il file cliccando con il pulsante destro del mouse sul file e selezionando Estrai tutto dal menu pop-up.
  5. Copiare il file $SystemUpdate dal file .zip sull'unità flash. 
    Nota: i file devono essere copiati nella directory radice e non devono essere presenti altri file nell'unità flash.
  6. Scollegare l'unità flash USB dal computer.
