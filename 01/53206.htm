chet814 say:
... with an Intel Pentium B940 dual core processor, does NOT support "hardware-assisted virtualization".  Will this be a problem... 


No, I do not think this will be a problem.

 

When Windows 7 was first released Microsoft advertised that XP Mode was a major feature for those with older programs; a major selling point for the new operating system. At this time it did require hardware virtualization, however, this requirement wasn't fully explained as much as the feature was advertised. As a result many users sadly discovered that their systems (CPUs) could not run this feature since their CPUs did not support hardware virtualization. It was not a good situation. But (IMO) Microsoft did the right thing and made changes to the XP Mode so that it could be run on systems that did not support virtualization.

 

Windows Virtual PC
Support: FAQ
 http://www.microsoft.com/windows/virtual-pc/support/faq.aspx

 

The following are excerpts for the page above:

 

Is hardware virtualization a must-have to run Windows XP Mode?
No. Windows XP Mode can be run on a CPU without hardware virtualization.

 

Is hardware virtualization better?
For Windows XP Mode the performance will be more than acceptable with or without hardware virtualization. For developers using Virtual PC with Windows Vista or Windows 7 virtual machines we recommend HAV as the non-HAV is only tuned for XP Mode.

 

So again, since your CPU does not support hardware virtualization I doubt if this will be a problem.
