How to install and activate Windows 10 using your Windows 7 or Windows 8 product key

When you upgraded from a previous version of Windows, what happened is the hardware (your PC) will got a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate.

So, there no need to know or get a product key, if you need to reinstall Windows 10, you can use your Windows 7 or Windows 8 product key or use the reset function in Windows 10.
