  1. Go to C:\Program Files (x86)\Skype\Browser
  2. right click on SkypeBrowserHost.exe -> Properties -> Security -> Advanced.
  3. Change permissions
  4. Select disable inheritance, choose explicit...
  5. Remove permissions for ALL APPLICATION PACKAGES, RESTRICTED APPLICATION PACKAGES, All Users to that process.
  6. Apply.
  7. Ok
  8. Quit Skype.
  9. Start Skype
  10. And you done.
