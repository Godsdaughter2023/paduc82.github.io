Upgrading from Windows XP?

Please read these instructions carefully and completely before you begin installing Windows 7: There are important steps that you need to follow to upgrade from your PC from Windows XP to Windows 7 in order to preserve your files and settings.

Before you begin, you'll need to back up all your data to an external storage device, and you'll need all of the installation discs for programs you want to keep. Please make sure you have an external hard drive (see details below).

More detailed instructions are available at windows.microsoft.com/upgrade-windows-xp. Here are the basic steps.

1. Run Windows Upgrade Advisor (windows.microsoft.com/upgradeadvisor) to see if there are any known issues that might affect the installation and whether you should install the 32 or 64 bit version of Windows 7.

2. Save your files and settings on an external hard drive using Windows Easy Transfer (windows.microsoft.com/windows-easy-transfer). If you don't have an external hard drive you won't be able to use Windows Easy Transfer. Alternatively, you can copy files you want to keep on a USB flash drive, CDs, or DVDs.

3. You'll need to manually reinstall your programs, such as Microsoft Office, after you install Windows 7. Gather the installation discs for your programs and any associated product/license keys. You might be able to download some programs from the Internet. For example, Windows Live Messenger can be installed from download.live.com/messenger.

4. Inset the Windows 7 DVD into your PC. When asked "Which type of installation do you want? click "Custom (advanced)."

5. After Windows 7 is installed, use Windows Easy Transfer to restore your files and settings.

If you're not comfortable installing Windows 7 yourself, check with a local retail store or PC services company to see if they offer upgrade services.

From what you describe, You currently have one user account on your system. I'm guessing you signed in with a Microsoft account associated with an email address beginning with 'rarol', hence the automatic folder naming.

'RichardA' is a folder you created and not associated with any account.

The path of least resistance is to accept the automatically created name and simply copy your folders from RichardA to the corresponding folders under rarol.

But if you want a more recognizable and pleasant folder name, this is the time to do it, before you've put much effort into customizing your profile.

To create a user folder name of your choosing, you have to:

  1. Create it as a local account, rather than one linked to an email address. The name you give when creating the account will be the new folder name. (Steps 2 - 6)
  2. Settings => Accounts => Family & other people. Under 'Other People', click 'Add an account to this PC'.
  3. In the Microsoft Account dialog, 'How will this user sign in?", click the link: "I don't have this person's sign-in information"
  4. In  the "Let's create your account" dialog, select the link: "Add a user without a Microsoft account"
  5. Follow the prompts to create the account.
  6. Change the account type from Standard User to Administrator.
  7. Log out of your current account and into the new account. This triggers the actual creation of your new user folder.
  8. Log out of the new account and back into your old account. Go to: Settings => Accounts and under 'Your info',select the link: 'Sign in with a local account instead'
  9. Follow the prompts to convert the old account to a local account.
  10. Log out of that account and back into the new account. Go to Settings => Accounts and select the link: 'Sign in with a Microsoft account instead'. Follow the prompts to link to your Microsoft account.


Now you can copy your files & folders from RichardA to your new account. 
If you have permission issues moving your folders, you may have to take ownership.
