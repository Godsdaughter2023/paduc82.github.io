a. user say:

Hello,

You may not use a Standard account to change the account to Administrator. You may only use an existing Administrator account to do this. Here's how:

  1. Open Control Panel.
  2. In Control Panel, select User Accounts.
  3. Select Manage another account.

This will show all of the user accounts on the computer. If there is an Administrator account it will show that designation.

Follow the steps below to change an account to Administrator type:

  1. Sign out, and log on with any Administrator account.
  2. Open Control Panel, click on User Accounts.
  3. Select Manage another account, and then select the Standard account you wish to change.
  4. Select the Change Account type option.
  5. Select the Administrator option to change the account and click the button.

We look forward to an update from you.

Regards.
