Hi Robert,


The "This device cannot start. (Code 10) A device which does not exist was specified" error message occurs if the hardware key does not contain a FailReasonString value. Usually, the device hardware key contains a "FailReasonString" value that only the hardware manufacturer can define.

Since performing a driver update did not resolve the issue, then you can consider installing the latest driver manually from the device manufacturer's website or ask assistance from their support team in installing the right driver for your device.
