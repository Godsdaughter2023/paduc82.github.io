Method 2: Registry fix.

 

To resolve this problem in Windows , add a EnHcSoftRetry registry entry with a DWORD value of 1 under the following registry subkey:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\PCI\<hw id>\<instance id>\Device Parameters

Note To obtain <hw id> and <instance id>, open the Device Manager property page for each EHCI USB host controller, click the Details tab, and then select Device Instance Path in the list. Most systems have only one or two EHCI host controllers, so this should not be too difficult.

mikey w say:

FIXED!!

After spending a couple hours on the phone with a very helpful Netflix tech it turns out that MS pushed an updated HEVC Video Extension file, which has broken 4K playback through Edge.

To fix go to your applications. Click start, settings, apps and features, type HEVC in the search box and uninstall the HEVC Video Extension app.

Go to this link: https://www.microsoft.com/en- and install the old version back.
