patienceiam say:

I have found a workaround. In case anyone else encounters this problem, I'll describe it here.

Set up the hotspot once. 

In order that it reconnects when the computer wakes, follow these instructions:

1) Set up and save a batch file with one line:

   netsh wlan start hostednetwork

2) In Task scheduler:

    Create Basic Task giving this whatever name you like

    Choose as the action Run Program and link to the batch file you saved.

Now go to the Properties of this task.  On the following tabs:

  General: choose Run whether user is logged on or not

  Trigger:

      Schedule the trigger to be On-Event-log

      Choose the System log. 

      From the Source drop-down list choose the Power-Troubleshooter

      As the Event ID enter the number 1       (one)

All done!

After I did this, on wake-up the hotspot is reconnecting within a few seconds.
