Running System Restores, Refreshes, and Resets - Microsoft Community

hosch250 say:

Running System Restores, Refreshes, and Resets

Technical Level : Intermediate

Summary
Explains the implications of running these three options of system recovery and how to create custom refresh and restore points. Describes how to run them. Basic to intermediate skills required.
Details

Many people run the reset feature when their computer is having problems and are surprised and annoyed when their programs and files are deleted.  They had not read the instructions thoroughly, so they performed the wrong recovery method for their needs.  This article will explain how to access and run the three main methods of restoring your system and what each accomplishes on your computer so you do not get into the same difficulties.  Some of these options will save your files, but it is always recommended to have an up-to-date backup of your files before you run any of these options.

To access any of these options to run them on your computer, boot into the advanced startup mode.  This can be done by holding Shift while you click the "Restart" button, which is accessible in the Settings Charm under the Power heading.  Other ways to boot into this mode involve going to the General tab of the Change PC Settings tool, also available in the Settings Charm, and clicking the "Restart Now" button under the Advanced Startup heading, or to hold the advanced startup key while you boot your computer, which varies for each manufacturer.  Dell's advanced startup key is f8, and HP's is f11; other manufacturer's key combinations can be found on the internet.  Once you have booted into the advanced startup window, follow the prompts about recovering your system until you find the option you want to use; you cannot use your computer while it is in the process of recovering Windows.

System Recovery:

The system recovery option has the fewest implications, so I will cover it first.  This feature restores your computer to the state it was in when the latest restore point was created.  The system recovery files contain shadow copies of all the system files, and will restore only those files that have been modified.  Any desktop programs that you installed after the date of the latest restore point will be removed, but the others will be saved, although they may be corrupted and need to be repaired or re-installed.  I do not know precisely how the Windows Store apps are handled, but I think they follow the same rules as the desktop apps.  All of the files within the C:\Users\UserName folder will be kept.  Before running this feature, you may want to know that it could take up to an hour to complete.

You can create a new restore point of your computer at its current state by following this procedure:  search RECOVERY in the Control Panel and open the "Create a Restore Point" link under the System heading.  Click the "Create..." button, and type the name you want it to have, maybe something like "Just Installed Office."  Click "Create" again, and wait for it to finish creating your restore point.

System Refresh:

The system refresh is slightly less complicated than the system restore, although it has more implications.  It will reload Windows from the currently active Refresh point and remove everything on your computer except for your personal files - those stored in the same place mentioned above.  You can also create a custom refresh image, which will contain all of the programs installed at the time the image was created; when you run this image, the restored system will contain those programs.  For example, if you create a custom image with Office installed, when you use that image, Office will be restored with Windows.  This feature can take a few hours to run.

If you do not create a custom refresh image and your computer's manufacturer did not provide one with the computer (Windows 8 pre-installs only), then you will not be able to use this feature.

You can create a new refresh point using the "recimg" command in the Command Prompt, which needs to be run as an administrator.  There are five available commands in the recimg category:

  • recimg /createimage <directory> 
  • recimg /setcurrent <directory>
  • recimg /deregister
  • recimg /showcurrent
  • recimg /help (also recimg /?)

Terminology: <directory> means to enter the drive location, such as C:\CustomFolder.

  • The first command – recimg /createimage <directory> – creates a new refresh point and sets it as the default.
  • The second – recimg /setcurrent <directory> – sets the custom refresh point in the specified location as the default and deregisters the current refresh point.
  • The third – recimg /deregister – deregisters the current default refresh point and sets the default back to the refresh point that was provided by the computer’s manufacturer, if one was provided.
  • The fourth – recimg /showcurrent – shows the file location of the current default refresh point.
  • The last – recimg /help, or recimg /? – displays the following help prompt:

System Reset:

The system reset is the most basic option of all, but it has the most implications.  This feature will completely wipe your drive and re-install Windows; no programs or files are saved, so you should have a backup of all important files on your computer.  This feature can take anywhere from three to over forty-eight hours to run.  The average time it takes to run on my computer is four to six hours.  You cannot create a new reset point.

Some manufacturers provide a reset drive in the computer (Windows 8 pre-installs only), but otherwise you will need to use recovery media to run a reset.  Personally created recovery media often has problems, so you should use the recovery media available from the manufacturer if possible.

Now you should be able to safely run any of these options without accidentally removing something that you did not mean to remove.  You should also be able to create custom restore and refresh points.  If you have any questions or problems, feel free to post a comment on this article or to use the "Need help?  Ask a Question" link at the bottom of this article.

Maybe I should have specified in the article that the reset feature is only available on computers with recovery media or with a built-in recovery drive, usually called WINRE (Windows Recovery) or something similar, or if you have recovery media available.  The built-in drive is only available on computers that came with Windows 8, and not all, but most, manufacturers included it.

 

You stated that you have recovery media.  I have a story about recovery media that should make you glad that your computer refused it at the beginning instead of following this route:  I had installed the Windows 8.1 Preview, and my refresh drive (not the reset drive) provided by the manufacturer was corrupted.  This was before I realized that it was only the refresh drive, and I had never recovered using media and rather wanted the experience, so I tried to use my recovery media to re-install Windows 8 for college.  My computer accepted the recovery media, locked me out for about six hours while reloading it, rebooted, and told me that the recovery failed.  I contacted HP because I couldn't get MS to give me the Windows 8 ISO download to try re-installing that way.  They sent me recovery media that worked, but my computer was useless for over 24 hours.

 

When you created your recovery media, you may have wiped the built-in recovery drive, if one was provided.  When you create recovery media through the Windows recovery media creation feature, there is an option to remove the built-in recovery drive.  I do not know how to check if you did this, but you can open the Disk Defragmenter tool to determine if you have a built-in drive.  Refer to this article to see how to do that: Using the Disk Defragmenter.  If there is a WINRE (or similar) drive, your computer probably had a built-in recovery drive.

 

My guess is that your problem is that you do not have any built-in recovery media, and that your backup is corrupted.  Your best bet is to contact your manufacturer and tell them about your problem.  They may have some tips specific to your brand and model, or you may need to get recovery media from them.  If you have any more questions, feel free to ask.
