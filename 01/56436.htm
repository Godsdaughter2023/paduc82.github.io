Hi

First, run the Disk Cleanup utility.

1. Go to Start / All Programs / Accessories / System Tools / Disk Cleanup.

2. In the drop down menu, select the drive you want to clean up and click OK. The drive will be scanned.

3. In the next window, select the Clean up system files button.

4. Select the drive again and click OK.

5. In the next window, select the More Options Tab. In the System Restore and Shadow Copies section, click the Clean up button and click OK.

 

6. In the next window, select the Delete Button.

This will remove all of the temporary files and all but the latest System Restore Points.

If you performed a clean install of Windows 7, you will have a folder called Windows.old on the system drive. This folder contains copies of the user files in the previous version of Windows that you upgraded from. If you no longer need these files, you can select the option to remove the 'Previous windows installation' in the Disk Cleanup wizard.

Let us know if this helps.

a. user say:

Hi Ravi TejaNalabothula,

Welcome to the Skype Community Forum.

If the person you're calling is not initiating screen sharing, but your screen is still being shared, I strongly recommend you to completely uninstall and reinstall Skype to reset its configuration settings. 

To completely uninstall, and then reinstall Skype for Mac:

  1. Quit Skype by selecting Skype > Quit Skype.
  2. Open your Applications folder and drag your copy of Skype to the trash.
  3. Then open ~/Library/Application Support * and drag the Skype folder to the trash. The ~ sign represents your home folder. To find your home folder, open Finder and, from the menu bar, select Go > Home.

Important: If you’d like to keep your chat and call history, skip step #3.

  1. Open ~/Library/Preferences and drag com.skype.skype.plist to the trash.
  2. Open Finder and search for “Skype” using the search feature in Mac OS X. Move all of the results returned to the trash.
  3. Click Ctrl and the trash icon, and select Empty Trash.
  4. Restart your computer.
  5. Download and install the latest version of Skype.

*By default, the Library folder is hidden in Mac OS X v10.7 “Lion” and higher. To find your Library folder, open Finder and, from the menu bar, select Go, press and hold the option key, and then select Library. And by the way, make sure that video call button   is selected and not the screen share button.

Please do not hesitate to reply to this thread if you have any additional questions or concerns.

Regards,
