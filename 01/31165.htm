Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

(Not applicable to you)

Select I recently changed hardware on this device.

tryx3 say:

Try these steps

1  Check that WMP is looking in the right places still - in WMP, click on Organise [in the upper left of the window] then select Manage libraries & look at the Music-Viedo-etc entries to check that the folders there are still the ones you need it to look at for your music & playlists.

2 Check that these library locations are also shown correctly in Windows explorer, Navigation pane [i.e. the left-hand pane], Libraries entry - inside this entry you should see the same folders as in step 1.

3  In WMP, press alt then t to get the Tools menu selected then select Advanced & then click on Restore media library.

4 If that doesn't fix things I ought to suggest using Control panel, Troubleshooters, WMP Library & WMP Settings – onboard versions [if you are connected to the internet then Windows should update these to be the same as the online versions when you try to run them]

4.1  Media player library troubleshooter – online version

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Library-troubleshooter

4.2  Media player settings troubleshooter – online version

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter

5  But personally, I think you might as well miss out the troubleshooters & go straight to these steps [they are the closest thing possible to uninstalling then reinstalling WMP & will effectively provide a new installation of WMP]

5.1  Unload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, clear Windows Media Player checkbox, Yes, OK, Restart the PC.

5.2  Reload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, set the Windows Media Player checkbox, Yes, OK, Restart the PC.

tryx3 say:

6 I ignored the bit about Playlists.  After the rest of the WMP library is fixed, check if the playlists have also reappeared. 

If not, check the Playlists folder - this is, by default, the same as the rip folder so in WMP click on Organise, Options..., Rip Music tab & look at the folder shown at Rip music to this location & set it to whatever you had before [you can also check the actual playlist files in Windows explorer as this folder will have a subfolder called Playlists and your playlists should be in that folder as .wpl files].
