So you download the software, and it fails to install since it is corrupted.  The local shop plugs the computer into their network, downloads the same file, and the program installs fine?

Sounds like some corruption is taking place on its way from the remote server to your system.  Perhaps, start by looking and the network hardware you're using outside of your computer - routers and modems.  See if you can swap one out and if it makes a difference.  Do you have another computer you can plug into your network to see if it experiences the same problem?

On later versions, the settings for font styles for reading and for composing mail are on the corresponding tabs in the program options dialogue.

Most emails these days are composed in HTML, and the sender usually specifies which font styles to use, so you have no control over them.
