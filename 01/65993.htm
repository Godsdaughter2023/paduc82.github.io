atom241 say:


1. Right-click on the Taskbar and choose properties

2. Make sure that the "Auto-Hide the Taskbar" box is unchecked and hit apply

3. Right-click on your MAIN displays Taskbar and scroll over the Cortana option

4. Select which ever option is not currently checked

5. Go back into the Taskbar properties and check the "Auto-Hide the Taskbar" box

6. Your Taskbars should now auto-hide

¡firedog say:

Thank you. That's a lot better.

The host 'inbound.mail.att.net' could not be found.

Protocol: POP3

Port: 993

Two things wrong here:

  1. There is no mail server called inbound.mail.att.net . There is one called inbound.att.net, though.
  2. Port 993 isn't used for POP3. The port you want is 995.

Unable to send or receive messages for the Bellsouth (myother) account. Your server has unexpectedly terminated the connection.

This is often a sign that there is a problem with the mailbox at the server. A bit of housekeeping may help:

 

  1. Log in to the account in webmail and start by emptying the folders for spam and deleted items.
     
  2. Create a new folder or folders and move all of the messages out of the inbox and into the new folder.
     
  3. When the inbox is empty, select the account from the dropdown list on the bottom half of the send/receive button in Windows Live Mail. No new email messages should eventually appear in the status bar.
     
  4. Back in webmail, move half of the messages that haven't yet been downloaded by Windows Live Mail back into the inbox and again try send/receive for this account in Windows Live Mail.
     
    If the fault recurs, we now know which half of the messages the bad one is in. Repeat the process, taking half of the messages each time until the dodgy message can be identified and dealt with in webmail.
     

It's not a good idea to use the webmail inbox as a storage folder. Move any messages for keeping on the server into archive folders, leaving only a few days' worth of messages in the inbox at any one time. Empty the spam and deleted folders regularly.
