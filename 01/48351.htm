Actually I saw This Feature On My friend's Laptop And I Really Like It

I Also Want to Use It Although I Don't Have A Microphone :( It Sounds Funny But Yes I Would Even Like It When I Have To Type My Commands On Keyboard


The microphone built into a notebook computer will work fine for an application like Skype. However, speech recogntion requires the purest audio input for accuracy. The input must not only be clean of electronic noise from within the computer but also reject background noise. This is why a good noise canceling USB microphone designed for Speech Recognition will work best.

However, if you are just issuing commands like Show Numbers, Open Microsoft Word etc, the built in microphone will work OK in a quiet environment.

shinmila h say:

Hey Windows Little Helper,

Welcome to Microsoft Answers.

How are you trying to format the partition? Is it from the explorer or disk management or by inserting the Windows installation disc?

It could be possible that the concerned partition is marked as Active.

To set your Vista partition as active, follow either of these methods.

Method 1:

1. Boot to Vista.
2. Click the Start menu and enter “diskmgmt.msc” into the search bar.
3. A window titled “Disk Management” should open displaying basic information about the drives attached to your PC.
4. Right click the Vista partition and select “Mark this partition as active”.

Refer:
Mark a partition as active
http://windows.microsoft.com/en-US/windows-vista/Mark-a-partition-as-active

Alternatively, you can do this from the command prompt.

Method 2:

1. Click on start and type cmd.
2. Right-click on cmd and select “Run as Administrator”.
3. Type DISKPART in the command prompt.
4. Type LIST DISK
5. Type SELECT DISK n (where n is the number of the old Win98 drive)
6. Type LIST PARTITION
7. Type SELECT PARTITION n (where n is the number of the active partition you wish to make inactive).
8. Type INACTIVE.
9. Type EXIT to exit DISKPART
10. Type EXIT again to exit the command prompt
11. Restart your computer.

Once the previous Windows 7 partition is marked as inactive, you can extend your vista partition from the Disk Management console.

1. Open disk management by following the first 2 steps from Method 1.
2. Right-click on vista partition and select “Extend Volume”.
3. Enter the full capacity of the unallocated space.
4. See if it works.

Additional information:

Uninstall Windows 7 on a multiboot system
http://windows.microsoft.com/en-US/windows7/Uninstall-Windows-7-on-a-multiboot-system

Regards,

Shinmila H - Microsoft Support
