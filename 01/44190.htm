It would be helpful to post what brand and model graphics processor your machine uses, but it looks like

You cannot force a graphics processor to use a DX version it was not built to use. This is a hardware limitation.

If it's a laptop you are completely SOL . You cannot upgrade laptop graphics.

DX 11 was introduce to Nvidia with the 400 series - released mid 2010 ( GTX460 being the minimum requirement

for this game).

AMD introduced DX11 a little bit earlier with their HD 6000 series ( HD6850 minimum ).

ddi ?  I assume you are getting the numbers from dxdiag > Display tab.

I'm not using Win7 atm, and don't recall if the terminology was different on Win7,  but on Win10 the Display tab

has an item called Feature Levels. This shows all of the DX versions your graphics processor is capable of using.

If 11_0 or 11_1 is not shown here your graphics processor is not capable or running DX11 effects.

If you had The Elder Scrolls Online, not the Tamriel Unlimited edition you would be able to run the

game, as this earlier version uses DX9.

The earlier version also gets by with a small 512GB VRAM vs 1GB VRAM minimum for the Tamriel Unlimited version.

burrwalnut say:

Here is the full procedure:

Right-click the folder, select Properties > Security Tab > Advanced at the bottom > Owner Tab > Edit > Highlight your username and put a tick in ‘Replace owner on subcontainers…’ and Apply > OK. You will only have the ‘Replace owner on subcontainers…’ box for folders not files. Click Yes when you receive the Security pop-up window then click OK to the Close/Reopen message.

Now go back to the Properties > Security Tab. Click Edit > Add. Type in your User Account Name in the box under Enter the objects name to select. Click on Check Names > OK. Click on your User Account Name to highlight it. Check the box for Full Control > Apply and OK etc.
