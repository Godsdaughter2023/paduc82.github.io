ATT/WESTELL Firmware (Blue/White)

 

Wireless Password

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.254
  4. Select Wireless from the menu on the left.
  5. When prompted, enter the Device Access Code printed on the device.
  6. The wireless passphrase will be listed as "Network Key".

 

Changing Wireless Security  Type/Disabling

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.254
  4. Select Wireless from the menu on the left.
  5. When prompted, enter the Device Access Code printed on the device.
  6. The wireless security type can be selected from the Security drop down. If no wireless security is desired select Disabled. For WPA select WPA-PSK, WPA2-PSK or Mixed WPA/WPA2.
  7. Enter the new wireless passphrase.
  8. Click the Save button.

 

Verizon Firmware (Red/Black)

 

Wireless Password

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. From the menu on the left choose Wireless Status.
  7. The wireless passphrase will be listed as "WEP 64-Bit" or "WPA". 

 

Changing Wireless Security  Type/Disabling

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway into the address bar of the internet browser. Example 192.168.1.1
  4. The router will then prompt you to enter the username and password.the default user name and password are: "admin" and "password"/"admin"/"password1".
  5. Select Wireless Settings from the tabs at the top.
  6. Select Basis Security Settings
  7. In the Turn WEP On section select off.
  8. Click Apply. (If no wireless security is desired stop here)
  9. Select Advanced Security from the menu on the left.
  10. In the Level 1 section select WPA.
  11. On the following screen set the Authentication method to Personal(Pre-Shared Key)
  12. Set the WPA Type to WPA Any.
  13. Enter the desired wireless passphrase into the WPA Shared Key space.
  14. Click the Apply button.
