cmdrkeene say:
Most people leave the option set to "Automatically install updates at 3:00 AM".  You can type "windows update" into your Start menu to configure the automatic updating.  If an update requires a reboot, the system will ask if it's okay to reboot.  If you ignore the warning for an hour, it will reboot anyway.  Because an update is worthless until you reboot, it is critical that it be done immediately.  Otherwise, your system is just as insecure as it would be with no update at all -- and when updates come out, hackers immediately begin searching for unpatched people to prey upon.

If the file that you were working on was open in a Microsoft Office program, such as Word, PowerPoint, or Excel, it would have been saved on your behalf.  Notepad is not sophisticated enough to handle this, however (Sticky Notes are, though).

Unfortunately, there is no way to recover contents of Notepad once the Notepad program is closed.
