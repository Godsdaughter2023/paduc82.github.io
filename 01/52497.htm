pestell159 say:

Before trying to delete an account with Admin rights, you need to ensure that you have another account that has Admin rights as well, otherwise you could back yourself in to a corner that may be difficult to rectify.

There are several ways to see which accounts are part of the Administrators group, and this can vary depending on your operating system.

One way is using an an elevated command prompt.

In Windows 10 click on the Start Button and start typing CMD, when CMD is listed, right click on it and select run as administrator. If you are able to access the command prompt as an Admin without having to enter a username and password then that account is part of the Administrators group.

In CMD window type

net localgroup administrators and press enter.

something like below will be displayed:
C:\Users\Computer>net localgroup administrators
Alias name     administrators
Comment        Administrators have complete and unrestricted access to the computer/domain

Members

-------------------------------------------------------------------------------
Administrator
AdminUser1

AdminUser2


The command completed successfully.

If your account is displayed, and the account that you want to remove is displayed then you should be in luck. Now run the following command:

net user <username> /Delete

i.e. net user AdminUser1 /delete and press enter.

The account will then be deleted from the PC, but their data will still be intact.
