Changing Display Resolution when Docked - Microsoft Community

aj-nyc say:

Changing Display Resolution when Docked

Is there a way to automatically switch resolution levels based on Surface Pro 3 docking status?

I want to use a lower resolution / larger icons when not docked and automatically change to the higher resolution/smaller icons when docking. Only way I found is to manually change things upon docking/undocking, which is a pain if you have to do that a few times a day...

silversee say:

With one caveat, this will happen automatically if you have left the desktop scaling settings set to their default values.  Windows 8.1 will detect the display resolution and panel size and choose the appropriate scaling level automatically for each attached display. 

The caveat is that you may need to log off and back on after docking for the scaling change to become fully active.  This is necessary if you have set up your system to use only the external display when docked.

To check your current desktop scaling settings, right click anywhere on the Desktop, choose Display Resolution, then click on 'Make text and other items larger or smaller'.  To get the automatic behavior, make sure that 'Let me choose one scaling level for all my displays' is NOT selected.

This of course works assuming you are happy with the automatic behavior. The built-in panel scaling factor is normally set to 150%, while most external desktop displays will be at 100%. This will give text and icon sizes that are similar in most cases, though the built-in display will obviously render text more crisply. If for some reason you wanted bigger text and icons on the native display, but smaller text and icons on the external display, then you would have to change this each time I'm afraid.

alden rey say:

Hi David,

Setting up a multi-display environment on Windows 10 is simple. When you connect a second display to your PC, Windows automatically detects the display and displays the desktop UI.

Customizing displays and how they are managed by Windows is easy. But in order to do it, you need to gain access to the display section of the control panel.

  1. Right-click anywhere on the desktop.
  2. Click Display settings.
  3. Click Identify to show numbers on corresponding displays. (1 is the primary.)
  4. Interchange displays by clicking and dragging the numbered boxes.
  5. Click apply.
