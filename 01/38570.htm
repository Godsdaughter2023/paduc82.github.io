a. user say:

Found an answer - if anyone knows of any others please post!


  1.  On the ext HDD, rename FileHistory to FileHistory.old
  2. Start File History such that it creates a new FileHistory folder.
  3. Close File History.
  4. Copy both folders from FileHistory.old to the new FileHistory folder. (If you have a lot of backup data then this may take several hours)
  5. Re-start File History and you should now be able to restore your data using your previous backups.

MS - This is NOT INTUITIVE - please modify File History so that it sees previous backups after a system is re-imaged.
