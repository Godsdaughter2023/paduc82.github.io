Hi mfodor3000,

 

o    What is the location of the pictures?

o    Which browser do you use to surf internet?

 

It could be the pictures from Temporary Internet Files location.

 

Temporary Internet Files folder is used by Internet Explorer to store Web pages, images, audio and video files, and other content from the Web sites that you are visiting. 

 

Follow the step below to verify if the pictures are located in that folder.

 

1.     Open Windows Media Player.

2.     Click on Pictures.

3.     Right-click on the picture and select Open File Location.

4.     If the pictures are located in Temporary Internet Files folder, it indicates that the folder is monitored by Windows media player (WMP).

 

To prevent this from happening in future, configure WMP to stop monitoring the folder.

 

Here's how:

 

1.     Open Windows Media Player 12.

2.     Click the Organize button and choose Manage Libraries from the drop-down menu.

3.     Click on Pictures.

4.     Check the Library locations that are listed.

5.     Remove Temporary Internet Files folder if listed.

6.     If it’s not listed, remove all the folders except My pictures which is the default location.

 

That should be it.
