a. user say:

A reasonable strategy to be reasonably safe without much overhead or system resources is to use MSE and supplement that with the free versions on Malwarebytes and SUPERAntiSpyware.   Maybe run a quick scan with those two one a week or so or whenever you feel that something weird is going on.

If you use the free versions of MBAM and SAS and when installing be sure to choose NOT to let them install any start up items (or disable them later) there should be nothing running from either two to cause a problem.  SAS will usually just find a bunch of tracking cookies but might pick up something MBAM missed.

If you installed the free MBAM with no startup items (check msconfig and uncheck any start up items) I don't see how it could be causing crashes since there should be nothing running.

The free versions are very good and if a system is acting really goofy I might just start with those two to be reasonably sure there is no malware going on using the "if you can't figure out what it is, figure out what it isn't" strategy.

If your system freezes you may have to take a different approach to figure out that.

If your system freezes up quickly and repeatedly without too much effort you might boot the system in Safe Mode with Networking and see if it runs better under that condition.

Safe Mode only loads basic things and while your system should be functional the desktop and display will look odd (probably much larger) since the generic video driver will be used but you just want to run that way long enough to determine if the freezing problem disappears when in Safe Mode.

If the system does not freeze in Safe Mode then we can work on figuring out what is being loaded on a normal boot that might be causing the freezes.  You should see if you can come to the conclusion that:

Yes - No matter what I seem to do the system does not freeze in Safe Mode.

Yes - the system always freeze in Normal Mode.

If that is true we can start to eliminate things from your normal boot until you can isolate the culprit.

To get into Safe Mode restart your system and start tapping the F8 key until you see that Advanced Boot Options menu and choose Safe Mode with Networking:
