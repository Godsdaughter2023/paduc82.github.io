BCCode: D1     0x000000D1

Cause

A driver tried to access an address that is pageable (or that is completely invalid) while the IRQL was too high.

This bug check is usually caused by drivers that have used improper addresses.

If the first parameter has the same value as the fourth parameter, and the third parameter indicates an execute operation, this bug check was likely caused by a driver that was trying to execute code when the code itself was paged out. Possible causes for the page fault include the following:

  • The function was marked as pageable and was running at an elevated IRQL (which includes obtaining a lock).
  • The function call was made to a function in another driver, and that driver was unloaded.
  • The function was called by using a function pointer that was an invalid pointer.

Download the free version of Malwarebytes and run a full system scan first of all deleting anything it finds. https://www.malwarebytes.org/mwb-download/

  1. Next, hit Windows key + E to open File Explorer and click "View" above the ribbon interface. If you can't see the ribbon interface, hit CTRL+F1 (you can toggle it on and off using the same command).

  4. Remove the checkmark from "Hide file extensions for known file types".

Attrib -H -S -R -A /D /S C:\Users\Username\Pictures

Substitute Username with whatever name you log into the computer with.

xircal say:

No. Let's assume that you login to Windows with your first name. In that particular case, the command would look like this:

attrib -r -s -h C:\Users\Jeffrey\Pictures\*.*

So if you don't login with the name "Jeffrey", substitute that with whatever name you do use.

It's better to copy the command after amending it with your login name and then go to the command prompt, right click and select Paste from the menu. Doing it that way means you won't make any mistakes.

*.* on the end of the command which I don't see in your screenshot means apply to every file regardless of name or extension.

If know that the pictures all have a .jpg extension, then you can amend it by substituting the last * (the character above the figure 8 key) and type jpg. In that particular case it would look like this:

attrib -r -s -h C:\Users\Jeffrey\Pictures\*.jpg

So what you're telling the computer to do is to unhide every hidden picture with a .jpg extension in the directory C:\Users\Jeffrey\Pictures (assuming once again that you login with the name Jeffrey).

To get the correct path, you can also do the following.

  1. Hit Windows key + E to open File Explorer.
  2. Double click C:\
  3. Double click Users
  4. Double click Username = the name you're logged in with.
  5. Double click Pictures
  6. In the line where you see This PC | C: | Users | Username | Pictures right click Pictures and select "Copy address as text".

Open Notepad, right click and select Paste. This will give you the same path as C:\Users\Jeffrey\Pictures with the exception that the Username will be whatever name you logged in with.

Here's the command again but with exaggerated spaces so you know where to add one:

attrib     -r     -s     -h     C:\Users\Jeffrey\Pictures\*.jpg
