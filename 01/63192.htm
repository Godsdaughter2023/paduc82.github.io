FYI, before I came across this remedy also tried removing the app and it's associated data using Powershell -  but that didn't solve the problem. Just in case it does work for someone else, here are the steps (on Windows 10):

Completely Remove a Windows Store App:

  1. Type "Powershell" (no quotes) into Cortana.
  2. Right-click the "Windows Powershell" icon at the top of Cortana's results and choose "Run as Administrator". Click "Yes" if you get a security prompt about allowing this program.
  3. Type "Get-AppXPackage" (no quotes) and hit enter.
  4. Find the program you want to clean up in the list of programs.
  5. Copy the exact and complete value listed under Package Full Name to the clipbaord (instructions on how to do this follow these instructions).
  6. Type "Remove-AppXPackage [App Package Full Name]" (where [App Package Full Name] represents the text you copied in step 5 above) and hit Enter.

Copy from Powershell:

  1. Click the command icon in the upper-left corner of the Window (to the left of "Administrator: Windows Power Shell".
  2. Choose Edit -> Mark from the menu.
  3. Highlight the text you want to copy in the window.
  4. Click the command icon in the upper-left corner of the Window (to the left of "Administrator: Windows Power Shell".
  5. Click Edit -> Copy and the selected text will be placed on the clipboard.

In Performance Options look for the "Show thumbnails instead of icons" option about halfway down the list and be sure it is checked, click OK twice, close the System window and see if that fixes things.

Either route should produce the same result.
