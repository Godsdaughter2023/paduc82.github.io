    1. How to Locate the Product Key
    Right Click on Start > Select PowerShell (Admin)
    Copy/paste the following command into the window then press Enter.

    wmic path softwarelicensingservice get OA3xOriginalProductKey

    The Window will reveal the product key for the unit. Write it down and keep it in a secure place.

    2. To Locate the Serial Number of the unit,
    if a Desktop PC, look on the side of the unit.
    If it is a Laptop PC look on the bottom of the unit.

    3. To Locate Serial Number using PowerShell (Admin)
    Right Click on Start > Select PowerShell (Admin)
    Copy/paste the following into the window then press Enter.

    wmic bios get serialnumber

    The window will reveal the serial number.

    4. This command will show you the Manufacturer Product Version, Serial Number and Motherboard Version.

    wmic baseboard get product,Manufacturer,version,serialnumber

    brent-- say: Hi wendymn,

    Let's try clean booting the system to remove 3rd party drivers and see if you still get the error.

    Follow the steps below to place your computer in “clean boot” 

    • Click the Start orb on your Desktop
    • Type msconfig in the Start Search box, and then press ENTER.



    If you are prompted for an administrator password or for a confirmation, type the password, or click Continue. 

    • On the General tab, click Selective Startup.
    • Under Selective Startup, click to clear the Load Startup Items check box. 
    • Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All. 
    • Click OK. 


    When you are prompted, click Restart.


    When the desktop loads, run a few programs, browse the internet for a while, then try shutting down your computer.

    If this does resolve the issue, then see the following Microsoft Article on “Clean boot” and the section “How to determine what is causing the problem” to determine which program or service maybe causing the issue.

    http://support.microsoft.com/kb/331796

    How to set your computer back, to boot normal:

    • Click the Start orb on your Desktop
    • In the Start Search box. Type msconfig, and then press ENTER.


    If you are prompted for an administrator password or for a confirmation, type the password, or click Continue. 

    Click the General tab. 

    • Click Normal Startup - load all device drivers and services, and then click OK. 
    • When you are prompted, click Restart to restart the computer.


    Hope this will fix it for you.
