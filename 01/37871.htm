Thank you for updating us. We've already tried multiple troubleshooting steps and the same issue still persists. In this case, let's try to troubleshoot the issue by creating other user profile and check if it works.

On Windows 10 Home and Windows 10 Professional editions:

  1. First, click Start, then select Settings.
  2. Select Accounts, then select Family & other people.
  3. Then, Add someone else to this PC.
  4. Lastly, enter a user name, password, password hint, and then select Next.

On Windows 10 Enterprise edition:

  1. First, click Start, then select Settings.
  2. Select Accounts, then click Other people.
  3. Then, Add someone else to this PC.
  4. At the bottom of the page, select I don’t have this person’s sign-in information, and at the bottom of the next page, select Add a user without a Microsoft account.
  5. Enter a user name, password, password hint, and then select Next.

Disclaimer: Once this troubleshooting steps, fix the issue, it means that the previous user account is corrupted.

To make the new account an administrator account:

  1. Select Start, then click Settings.
  2. Click Account, then select Family & other people (or Other people, if you're using Windows 10 Enterprise), and select Change account type.
  3. Under Account type, select Administrator, then click OK.

Restart your machine and sign in with the new administrator account.

If the issue still persists, we suggest to perform a system restore. This option takes your device back to an earlier point in time. Restoring won’t affect your personal files, but it will remove apps, drivers, and updates installed after the restore point was made.

To do this:

  1. First, select Start, type control panel and then choose it from the list of results.
  2. Search Control Panel for Recovery.
  3. Then, select Recovery, then select Open System Restore.
  4. Click Next.
  5. Choose the restore point related to the problematic app, driver, or update, and then select Next.
  6. Lastly, click Finish.

Note: If you’re not seeing any restore points, it might be because system protection isn’t turned on.

To check:

  1. Go to the Control Panel, search for Recovery.
  2. Then, select Recovery, the click Configure System Restore.
  3. Select Configure and make sure Turn on system protection is selected.

Update us on the result.
