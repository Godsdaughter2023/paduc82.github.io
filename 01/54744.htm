Internet Explorer is deeply integrated into Windows.  Prior to Windows 7, it was not possible to uninstall IE (not counting customized Windows installations using, e.g., nLite).  Even in Windows 7, however, only the executable part of IE can be uninstalled ... you can't get rid of the HTML rendering engine because that is used by other parts of Windows.

From the issue description, I understand that you are facing issues with connecting to internet.

 

1. How do you connect to internet? Is it a wired or wireless connection?
2. Do you get any error message?
3. Have you made any recent software or hardware changes to the computer?

 

I suggest you to refer to the following Microsoft article and check if it helps.

 

Method 1:

Uninstall the network drivers and install the latest driver from the manufacturer’s website.

Step 1:

Follow these steps to uninstall the network drivers.
a. Open Device Manager by clicking the Start button , clicking Control Panel, clicking System and Maintenance, and then clicking Device Manager.‌.
b. Search for Network Adapters and expand it.
c. Right-click on the network adapter and click on Uninstall.

Step 2:



Go to manufacturer’s website and download the drivers.

 

Method 2:

 

I would suggest you to follow these steps and check if it works.

 

a.     Click Start, click Run, type cmd, and then press ENTER.

b.     At the command prompt, type netsh int ip reset resetlog.txt, and then press ENTER to reset the TCP/IP network protocol.

c.     At the command prompt, type ipconfig /renew, and then press ENTER and check whether you see the error.

 

Reference:

 

How do I fix network adapter problems?


http://windows.microsoft.com/en-US/windows7/How-do-I-fix-network-adapter-problems
