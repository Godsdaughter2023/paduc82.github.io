slance310 say:

Hi,

What you are describing is "Normal"... Even though it seems strange...

The file size, contents, and etc. of the image file are protected, and as a result display no info... Why it is so strict I don't know, but it is normal...

XP is apparently ignoring this protection, or doesn't recognize it..

The reason your image does not appear on your Cavio is because the image is of the OS, and it can not be reloaded\restored while windows is running...

I had to do a quick test to verify this before I posted, and mine is the same...

If you were to boot the machine from a System Repair Disk that is offered when you create the image, your Cavio and the image will appear...

You can also boot the machine with Win 10 installation media, select REPAIR on the lower left of the INSTALL scree, then Troubleshoot... Advanced... you will see an option to Recover an image... among a number of other options...

Your Cavio will appear there as well, along with the image...

Since having a USB drive attached when booting from installation media may cause an issue, it would be best to not hook up the USB drive until you click on Troubleshoot...

Also... the recovering the image process is a bit tricky in some ways...

If you make an image of Win 8.1 while running Win 8.1, the image will not appear if you use Win 10 media to try and recover it...

This may also happen if the version of Win 10 you use to create the media is of a significantly different build of Win 10 that you try to do the recovery with...

If the image file is renamed, it will not appear or be usable...

What is difference between pausing a printer and using it offline, - Microsoft Community

seth d say:

What is difference between pausing a printer and using it offline, especially in regards to network printing?

My printer is a HP 2605dn laser printer, which is on my home network (using IPP to the printer's IP address on port 9100). When I am away from home, I like to send print jobs to the offline printer for actual printing when I return. To do this, I pause the printer when I'm away, and I resume it when I am back at home. However, the printer is automatically "offline" when I am away. What are the differences between a printer that is "paused" and one that is "offline"? If I print to an offline printer, will the jobs automatically start once the printer becomes online (when I reconnect to my home network)?
amrita m say:

Hi Irv,

 

Welcome to Microsoft Answers!

 

In a ‘Paused’ printer the spooler is still working so it will accept the printing jobs and store it in its memory until the printing job is resumed. But if the printer is ‘Offline’ the spooler will have no jobs assigned to it, so there will be no printing jobs in the spooler memory when you put it online.

This means that if you assign any printing job to the printer when it is offline, it will not start the job automatically once you reconnect to the home network.

 

Hope this information is helpful.
