a. user say:

Hi Seth,

Thank you for posting your query in Microsoft Community.

a) What is the exact error message you get?

b) When do you get the error message?

I can imagine the inconvenience you are experiencing and I really appreciate all your efforts taken to resolve the issue.

Ideally, TrustedInstaller is used by a service called “Windows Module Installer” which is used to install Windows 7 system modules. The system modules include Windows Update and Automatic Updates which automatically scans your system to check for any new updates and hotfixes available. So, when scheduled Windows Update is performing its update scan or check on your system, the TrustedInstaller.exe process increases the CPU usage. Hence, it is advisable to wait for a while for the TrustedInstaller.exe to complete its job. Once it finishes running the tasks, the CPU usage rate will return to normal.

Hope this information is helpful. Let us know the asked information, we'll be glad to assist you further.

phillipcaprio say:
Ok I understand, follow these steps first and then try again:

Open Start, type: CMD
Right click CMD
Click Run as administrator

Type in at the prompt OR Copy and Paste these one at a time : (Hit enter after each)

Dism /Online /Cleanup-Image /CheckHealth

Dism /Online /Cleanup-Image /ScanHealth

Dism /Online /Cleanup-Image /RestoreHealth

Also run the System File Checker utility:
Open Start, type: CMD
Right click CMD
Click Run as administrator

At the Command Prompt, type: sfc /SCANNOW
This will check for any integrity violations

Restart your system
