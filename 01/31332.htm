balaji kundalam say:
Hi
Thanks for contacting Microsoft Community.

For installing two Windows OS (called Dual Boot), you need to first decide what exactly the OS are. Based on that, you can proceed with the installation of older OS first, then installing the latest OS. Here, in your case, Windows XP first and then Windows 7.

If you change the order, the latest version of Windows will not work properly.

In addition to that, you can use Virtual PC or VMware to use Windows XP in Windows 7. Windows XP Mode is another Microsoft release that helps you use Windows XP in Windows 7 through Virtual PC. You can download Windows XP Mode from here:
http://www.microsoft.com/windows/virtual-pc/download.aspx

To install Windows XP Mode, you need to have Windows 7 Professional or higher, with hardware virtualisation enabled in your PC (you need to check in your PC BIOS). If you prefer Windows XP Mode, you don't have to install Windows XP the way mentioned above. It will be installed just like any other program in Windows 7.

Note that, if you have Windows 7 Home Premium or lower edition, you can either try Windows XP through Virtual PC/VMware or proceed dual boot.
