Thank you for your reply.

  1. Which updates were installed? What are the KB number of the updates that were installed just before this issue appeared?
  2. Have you checked the status of the sound card in Device Manager? Is there an yellow exclamation mark?

To check the KB article of installed updates:

 

Click Start, click All Programs, and then click Windows Update.

 

On the Windows Update Web page, click View update history. A window opens that shows all the

updates that have been installed or that have failed to install on the computer.

 

To check the status of sound card in Device Manager.

 

1. Click Start

2. Type Device Manager in the Start Search box, and then press ENTER.

3. Click Continue. The Device Manager Dialog box is displayed.

If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

In the list of device types, click the kind of device, and then locate the sound card.

Check if there is yellow exclamation mark on sound card.

 

If no exclamation mark appears next to the name of the device and the device does not function correctly, manually uninstall and reinstall the device drivers. To do this, follow these steps:


1. Right-click the device and then click Properties.

2. Click the Driver tab.

3. Click Uninstall.

4. Click OK.

 

In the Device Manager Dialog box, click the Action menu, and then click Scan for hardware changes.

You may receive a message that states that Windows has to install driver software for your unknown device. If you receive this message, click Locate and install driver software (recommended).

 

If there is yellow exclamation mark please follow the steps in the link below to update the driver:
