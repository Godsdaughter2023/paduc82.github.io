7  Just for completeness, I should point out that user profile errors causing The User Profile Service failed the sign-in for user accounts that have previously worked correctly are a different issue. 

  • If the error only happens once then it is normally assumed to have been caused by a simultaneous attempt by Windows to access the user profile whilst Windows defender was separately trying to scan it.
  • If it happens every time then such user profile corruption is generally not resolvable & the afflicted user account must be replaced. 
  • If the error message is accompanied by being allowed in on a temporary user profile then you can try Fix You've been signed in with a temporary profile - TenForums
  • The best defence against user profile corruption of all kinds is to have a spare [additional, not routinely used] password-protected local Admin account set up on the computer. 
    • This might sit there for years before being used but, if it is ever needed, it will turn a disaster into an annoyance.
    • The password will need to be written down somewhere that is protected from unapproved users & secure from loss.
