Hi Charya,

Thank you for posting in Microsoft Community Forum.

Please be assured that we would do our best to assist you.

This could be caused due to the active partition C: drive was resized.

I suggest you to follow the below methods and check if issue persists:

Method 1

Run CheckDisk

Refer the steps below to run CheckDisk:

  1. Click Start, type “cmd”.
  2. Right- click on cmd and click on run as administrator.
  3. Type “chkdsk /f c:” and press Enter.

Note: Check disk would take more time than expected depending on the size.

Important: Data available on bad sectors might be lost when chkdsk tries to repair your hard disc.

Method 2

Run extend filesystem command from diskpart which extends the file system. Refer the steps below:

  1. Click Start, type “cmd”.
  2. Right- click on cmd and click on run as administrator.
  3. Type “diskpart”
  4. Type List Volume Select volume # (this is the number of the volume listed by the above ‘List Volume’ command)
  5. Type “extend filesystem”.
  6. Type “Exit”
  7. Restart the computer.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.
