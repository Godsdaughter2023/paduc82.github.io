tricky300 say:

It sounds like you have three network connection icons...

1. The local area (wired) connection icon - two screens - probably with a red X

2. The Windows wireless connection icon - one blue screen with 'waves'

3. The Netgear wireless connection icon - you describe it as "green computer screen with waves"

 

It also sounds like maybe you are thinking that 2 and 3 are discrete connections. This will not be the case, they will just be separate indicators of the same wireless connection.

 

Reading between the lines a bit, my guess is that you used to have only 1 and 2 but now you have the 3rd one too. If I am right, probably all that has happened is that you started the Netgear wireless configuration application, which experience tells me has a habit of 'taking over' from Windows wireless configuration without even asking if you want this to happen. All you need to do to get rid of this green screen icon is open the Netgear configuration (via its system tray icon) and look for a tick box to say 'let Windows manage my wireless network connection' (or words to that effect, also, you might see the word 'zero' in the text). If you have no network connection after doing this, then you were always using Netgear's configuration and you will have two choices...

  1. Start the Netgear application form the start menu and reverse the change you just made
  2. Configure Windows to control the wireless connection

If you want to go on using Netgear configuration instead of Windows, I believe that you will have to put up with both icons.
