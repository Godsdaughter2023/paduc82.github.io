Dirty connectors may cause your keyboard to stop working after waking from sleep.

  2. Press and hold the volume-up button and the power button at the same time for at least 15 seconds and then release both.

  3. After you release the buttons, wait 10 seconds.
  4. Press and release the power button to turn your Surface back on. You should see the Surface logo.

trekdozer say: 32 bit Windows does not support booting from a 3TB drive. You need 64 bit.

Windows XP has a limit of 2TB per physical disk, including all partitions.

Windows XP x64 Edition can use GPT disks for data only.

The 32-bit version of Windows XP can not read, write, or boot from GPT disks.



It’s possible to boot from a 3TB+ hard drive with a ﻿GUID Partition Table (GPT) if you have 64 bit Windows 7 and a computer that supports UEFI.
