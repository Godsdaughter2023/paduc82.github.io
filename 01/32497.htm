ben_g0 say: Windows 8 also performs quite bad on my machine with compareable specs, but I found a way to make it slightely better:

Windows 8 comes heavily fragmented, which makes the hard drive respond slowly, especially if you have a rotary hard drive instead of an SSD (SSD's only come standard in quite expensive machines).
Combined with that, windows 8 uses the hard drive A LOT for no obvious reason. Because of this, the hard drive usage will ba almost always close to 100%, which slows down other programs.

You can't fix the high hard drive usage, but you can optimize (defragment) the hard drive to make it more responsive, which results in a slight performance gain. To do this, open the windows explorer (windows button+E), right click the hard drive and select properties. Then, in the 'extra' tab, selecct 'optimize'. It can take several hours and slows down your computer even more while it's running, so do this when you have a lot of free time.

If it still doesn't run at an acceptable speed, you can try a downgrade, but unfortunately, microsoft has done a lot of effort to block other operating systems on windows 8 computers (unless you have one running windows 8 pro), so it's really hard to do.

EDIT: An other thing you can try when defragmenting doesn't work well is the following:
 - open a command prompt (or an explorer window if you can't use the command prompt well)
 - open task manager (ctrl+alt+del > task manager)
 - under 'windows processes', find the explorer process
 - end the task
This gets rid of the metro start screen, the task bar, the desktop, etc. This does improve the performance and stability, but it also stops some of the most used windows features (temporarly - they'll return after a restart). You can still use an explorer window as a substitute for the desktop though.
If you want everything to return, re-open the task manager (or alt-tab to it if it was already open), go to file > new task, and type in 'explorer' (witouth the quotes) and you'll see all windows features coming back, together with a performance drop.

ben_g0 say: Hi,

I've found out of some other tips that may cause your computer to work better:

 - I've installed Start Menu 8 (not Start8, it's basically a free alternative for it), and not only does it bring back the start menu, it also allows you to disable the metro interface (it hides the 'charms bar' on the right, it disables the things that happen when moving the mouse to the corners, etc. ).
It makes windows 8 a lot easyer to use, and with most of the metro interface gone, it seems to be a bit faster and more stable, even though that might be because of some other settings.
 - I've also installed Send Desktop, which makes your in computer immediadely switch to the desktop mode when it starts up. (This does look a bit glitchy as the start screen seems to quickly appear then dissapear, but it works great otherwise).

That were the programs I installed. If they didn't improve the performance and/or stability for you, then these settings may help:

 - Disable fast startup. If you've disabled it, your computer will start up slower, but it will always restart completely. This has the advantage that if something went wrong, causing your system to become less stable, it should be fully reset and fixed when you restart your computer, instaed of the problem being saved so it still buggs you after a restart, where it could get even worse.
If you don't like the slower startup speed, you can always just close your laptop instead of shutting it down completely, and only reboot when you're having performance/stability problems.

 - Some devices attached to your computer have the "allow windows to turn off this device to save power" option (in the properties window when you right-click them in the device manager). It's best to turn this option off, because otherwise, driver glitches can cause them to turn off in the middle of operation. 

 - Finally, find desktop alternatives for the metro apps that you use/need. The metro apps are extremely slow to start up, they are very limited in features, and the controls are very awkward if you don't have a touch screen. Set the default image viewer to "Windows photo viewer" or an alternative instaed of the pictures app. Install adobe pdf viewer as your default pdf viewer instead of that crappy e-reader app, use the website of outlook or MS office outlook instead of the mail app, etc.
When you're done replacing all the metro apps, terminate their background processes or uninstall them completely. Usually, the apps that came preinstalled on windows 8 machines can either be downloaded for free or are just demos, so don't worry about accidentally deleting something you might need in the future.

I hope that this was helpfull for you.

A Code 10 error is generated when Device Manager can't start the  hardware device, a situation usually caused by outdated or corrupted drivers. A device may also receive a Code 10 error if a driver generates an error that Device Manager doesn't understand.
