Safe mode is a troubleshooting option for Windows that starts your computer in a limited state. Only the basic files and drivers necessary to run Windows are started. The word Safe Mode appears in the corners of your monitor to identify which monitors mode you are using.

5. If you see safe mode box has a check mark in it, uncheck the safe mode box and hit apply.
6. You will then see a restart button. Click it to restart your computer.
7. When your computer comes back up, sign in and you will now be in "normal mode".

frederik long say:

Here are a couple of reasons why this might happen:

  • You used the numeric keypad to enter the numbers but the numeric keypad was not active at logon time.
  • You have several keyboard maps installed. The map that is active at logon time is not the same map as then one you have later on.

I also wonder just what exactly you mean with "resorted to blanking the password using some tool". Some password crackers are known to cause permanent damage to the security database. It would have been safer to boot the machine with your System Repair CD, then use System Restore to set Windows back to a point before you changed the password.

To eliminate any doubt about the password you enter, use this method to set it:

  1. Click the Start globe.
  2. Type  cmd.exe  in the Search box.
  3. Press Ctrl+Shift+Enter
  4. Click "Run as Administrator".
  5. Type the following command and press enter:
    net user  "%UserName%"  123456

This command would set your password to 123456. Since you can see it on the screen you know exactly what you typed.
