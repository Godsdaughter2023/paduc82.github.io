Error from the log: 0x80070641 means ERROR_INSTALL_SERVICE_FAILURE - The Windows Installer Service could not be accessed. This can occur if the Windows Installer is not correctly installed

You may try the follow steps to check if these methods fixes the issue:

Method 1: Check Windows Installer service.

  1. Go to Start > Run > type in services.msc > OK.

  2. The Services window will open, scroll down to Windows Installer and double click on it.

  3. The Properties window will open, on the drop down menu (Start up type) select 'automatic' > OK and close windows.

  4. Reboot PC and hopefully it will work for you.

If issue persist, proceed to next method.

Method 2: Unregister and re-register the Windows Installer

  1. Click Start, click Run, type MSIEXEC /UNREGISTER, and then click OK. Even if you do this correctly, it may look like nothing occurs.
  2. Click Start, click Run, type MSIEXEC /REGSERVER, and then click OK. Even if you do this correctly, it may look like nothing occurs, or you may briefly see an hourglass. After you run this command, the operation is complete.

Try your Windows Installer-based application again.

If issue persist, proceed to next method.

How to troubleshoot problems that may occur when you install, uninstall, or upgrade a program on a Windows-based computer

paul_9000 say:

File names of pictures used in Live Movie maker

I'm creating a movie using about 1000 pictures and I was looking for a way to display/download/print a list of the file names of the pictures I've already used in the movie?

Thanks Paul

considerate_guy say:

There isn't a way to retrieve that using Movie Maker, at least not all at once.  You can hover your mouse over a picture and you'll see the file name displayed.  But then you'd have to write that down and continue to the next, and the next, repeating the process.


If you are comfortable with Windows, you can open of the internals of your project file and see that information there.  You would use Windows Explorer, find the project file (i.e. .wlmp file).  Right-click on it and choose Open with.  Then choose the Notepad application.


If you did it correctly, you'll see a block listing your Media Items at the top of the file:


<?xml version="1.0" encoding="utf-8"?>
<Project name="My Movie" themeId="0" version="65540" ...
  <MediaItems>
    <MediaItem id="1" filePath="C:\Backgrounds\blue.jpg" ...
    <MediaItem id="2" filePath="C:\Backgrounds\gold.jpg" ...
    <MediaItem id="3" filePath="C:\Backgrounds\orange.jpg" ...
    <MediaItem id="4" filePath="C:\Backgrounds\pink.jpg" ...
    <MediaItem id="5" filePath="C:\Backgrounds\purple.jpg" ...
    <MediaItem id="6" filePath="C:\Backgrounds\red.jpg" ...
  </MediaItems>

...

paul_9000 say: Thank you, that worked.  With some creative search/replace, creating tables and sorting, I was able to pull out the list that I needed.

If anyone else needs to do this, I would strip out any music or theme, they just add extra lines and text to the files.

Paul
