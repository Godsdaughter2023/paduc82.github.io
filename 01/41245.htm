Abonnement par CB

Il n’est plus possible de désactiver le renouvellement automatique en ligne depuis Février 2010, un appel au Support est obligatoire.

Souscription

Lors de l'acquisition d'un abonnement Gold via le site Xbox.com, il arrive fréquemment que l'utilisateur se connecte au Xbox Live et constate que son abonnement est toujours Silver.
site Xbox.com, il arrive fréquemment que l'utilisateur se connecte au Xbox Live et constate que son abonnement est toujours Silver.

Ce dysfonctionnement est dû à des perturbations du processus de synchronisation des serveurs, pour obtenir l'abonnement Gold sur son compte il faut :

  • Soit patienter... et au bout d'un certain laps de temps, l'abonnement Gold est validé sur le compte.
  • Soit supprimer le profil et utiliser Récupérer le Gamertag.

Comment vérifier la date de fin de son abonnement Gold:

  • Se connecter à l'adresse suivante : Microsoft billing 
  • Depuis la page Aperçu du compte de facturation, dans le panneau Vos services sous Nom du service cliquer sur Abonnement Xbox Live Gold x mois.
    • Sur la page Abonnement Xbox Live se trouve la date d'achat ainsi que la date de fin ou de renouvellement automatique planifié.

Mise à jour des informations concernant le compte:

  • La saisie d'un code d'abonnement prépayé suspend temporairement le renouvellement automatique qui est réactivé lorsque l'abonnement par code se termine.
  • La suppression d'un moyen de paiement (CB) est impossible tant qu'un abonnement Gold acheté avec ce moyen de paiement est actif.
  • Le compte est suspendu et le message demandant de mettre à jour son moyen de paiement est affiché:
    • Lorsqu’une CB est expiré ou non créditée le système Live ne peut débiter les fonds, il peut en résulter une suspension du compte jusqu'à ce qu'une nouvelle CB valide soit ajoutée ou le compte crédité.

Code 80190826

Renouvellement d’abonnement bloqué, il faut vérifier les moyens de paiement via Microsoft billing.

Contrôle parental

Ce dernier est appliqué au compte jusqu'à la majorité du joueur et n'est pas supprimable.
Lorsque l'âge requis sera atteint, il sera possible de promouvoir le compte enfant en compte adulte:
Comment promouvoir le compte enfant

Changement du Windows Live ID.

Lors du changement du Windows Live ID associé au Gamertag :

  • Les MS points sont automatiquement transférés vers le nouvel ID.
  • Un seul changement de WLID par période de 30 jours est possible.

Pour changer le WLID d'un compte enfant, le WLID du compte adulte qui gère le contrôle parental sera demandé.

You may also be able to temporarily resolve the issue by booting to Safe Mode, and then immediately booting back into normal mode.  This workaround may resolve your problem for a while, however the error may return later.

To boot to Safe Mode: 

  1. Hold the Shift key down while you press the Power icon and select Restart.

  2. Once you are within the Windows Recovery Environment, select Troubleshoot, then Advanced options, then Startup Settings, and Restart.

  3. When it restarts, you should see a number of options.  Press 5 or F5 for Safe Mode with networking.

  4. Once you sign into your account in Safe Mode, you’re done.  Just restart your PC to return to a normal boot.

If you are running a 3rd-party antivirus software, we recommend uninstalling and then reinstalling the antivirus software, as this may also provide a workaround for this problem.  The easiest way to uninstall an app is to find it in the Start menu’s app list, right-click the app, and select Uninstall. 

Some customers have reported that adding a new local administrator account has resolved their Critical Error. If that doesn’t work, try removing the original administrator account now that you have a new one created. Details on creating the new local administrator account can be found here along with some additional content on these errors.

gerry c j cornell say:

A digital licence (the Windows 10 equivalent of a product key) is stored in the BIOS. When Windows 10 is reinstalled the licence is checked to see that the existing licence matches the Windows 10 edition being installed and windows is automatically activated in the absence of any discrepancy.

 

As long as Windows 10 has been previously installed and activated any subsequent installation matching the earlier installation should automatically activate.

 

If you have changed the motherboard and the original installation was an OEM copy of Windows you will need to buy a new copy of Windows. If you have changed one or more other hardware devices the new installation may not automatically activate. In this situation you should telephone Support for assistance using the number on the Activation screen.
