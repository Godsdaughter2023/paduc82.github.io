Hi,

Thanks for the update on the issue.

Windows uses the index to perform very fast searches of the most common files on your computer. When we search any file/folder the indexed location saves these files as the history.

You may try to remove the index location items, follow the steps :

  1. Press Windows + X key on the keyboard.
  2. Type Indexing option.
  3. Click Modify.
  4. To add or remove a location, select or clear its check box in the Change selected locations list, and then click OK
  5. If you don't see all locations on your computer in the list, click Show all locations. (If all locations are listed, Show all locations won't be available.)  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  6. If you want to include a folder but not all of its subfolders, click the folder, and then clear the check box next to any subfolder that you don't want to index. These folders will appear in the Exclude column of the Summary of selected locations list.
