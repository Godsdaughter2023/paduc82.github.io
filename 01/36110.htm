loraine mon say:

Hi,



You will have a moderate NAT on your Xbox app or Xbox console if the perimeter network (DMZ) or the network ports are not set up correctly. Enabling these settings will remove the restrictions to the internet. To enable the perimeter network on your router, you can click this link and find the brand of your router. Then, follow the steps to enable the perimeter network (DMZ). You have to use a static IP address on the computer when you set up a perimeter network.


It is also recommended that you open network ports using port forwarding. To open ports, follow the steps below:


1. Open a web browser.

2. In the address bar, type the default IP Address for your router, and then press Enter. Here are the default IP addresses for a few router and gateway manufacturers:


Belkin: 192.168.2.1
Linksys by Cisco: 192.168.1.1
D-Link: 192.168.0.1
NETGEAR: 192.168.0.1. or 192.168.1.1


3. Sign in to the router page.

4. After you're logged in, you should see your router or gateway's configuration page.

5. Open the following ports:

  • Port 88 (UDP)
  • Port 3074 (UDP and TCP)
  • Port 53 (UDP and TCP)
  • Port 80 (TCP)
  • Port 500 (UDP)
  • Port 3544 (UDP)
  • Port UDP 4500 (UDP)
  • Pick a port number between 49152 and 65535. If you don't want to pick one, just use 60209. (Document the port number you choose to be used in part 3.)


If you need further assistance on how to properly configure these settings on your router, we strongly suggest that you contact your Internet Service Provider for assistance.

davem121 say:
Hi Ray, Homegroup was removed form Windows 10 in version 1803, if you are running 1803, you will not be able to set up a Homegroup . . .


To be able to share files between PC's on your Network:

On each PC, create a new folder. right click that folder and choose properties

Click on the Sharing Tab of the resulting dialog and click 'Share'

Then give that folder a Unique Name on your Network



If you have a problem seeing each PC in the Network Section in File Explorer:


Click your Start Button, type Services and hit Enter

When the Services app opens, scroll down to find the following services, then double click each one in turn, set their startup type to Automatic and start the service if necessary

1. Function Discovery Provider Host

2. Function Discovery Resource Publication

3. SSDP Discovery

4. UPnP Device Host

Save that and close the Services app

That's It!



If you get a Network Credentials popup, you should be able to turn off that requirement . . .

Open Control Panel

Go to Network and Sharing Center

Select Change advanced sharing settings in the left pane

Expand the network you want to disable password protected sharing on by clicking the down arrow on the right of the profile

Select 'Turn off password protected sharing' and then click Save changes.

brittanyjames say:

Is there a way I can sync my Facebook events to the Windows 8 calendar app on my computer (no birthdays, just events)?

I just got a new computer with the Windows 8 operating system on it.  I am wanting to sync my Facebook events with the calendar app. Can this be done and if so can someone walk me through how to do this?
blairgranville say: Hi Brittany,

I eventually found a round about way to do this.

It seems from the Calendar settings screen you can only add gmail, hotmail/live, and outlook type accounts. As you have Windows 8 machine, you will definitely have a hotmail/live account, which I think should use the same password you use to log into your computer, and your Windows Live Calendar should already be displaying in your Calendar on Windows 8. So all you need to do is import the facebook calendar into Windows Live Calendar.

First you need to copy the url for your facebook events (or birthday) calendar from https://www.facebook.com/events/calendar.

Click on the little settings wheel next to "Today" in the far right corner, then click export, then copy the urls of either events or birthdays calendar (whichever you want).

Then you need to do is go to your hotmail/windows live calendar (Windows Live Calendar). Click "subscribe" up the top, and subscribe to a public calendar by  adding the url you have copied from facebook calendar.

This means you windows live calendar now has your facebook events, and next time you start the calendar in Windows 8, they should show up.

NOTE: I have also synced my facebook calendar with my gmail calendar, and added my gmail account to the windows 8 calendar, but for some reason this then did not show up. I have to use the Windows Live Calendar to get it to work.
