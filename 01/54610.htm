radney dri say:

Hi,

Deleting or Blocking available networks can only be done using Command Prompt. Follow the steps below:

  1. Open Command Prompt.
  2. Type netsh wlan add filter permission=block ssid="WLAN name" networtype=infrastructure (Replace WLAN name with "hidden network" or the name of the wireless network you want to block) and press Enter.
  3. Close Command Prompt.

To remove/delete the network from the block list. Replace "add" with "delete" on the command in step two.
