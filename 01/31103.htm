lemp say:

The "file association" for exe files has become damaged.

Go to this page:  https://www.sevenforums.com/tutorials/19449-default-file-type-associations-restore.html

Read the directions, all the way down until you see the big blue heading "Default Windows 7 File Extension Types"

Then scroll down the page to "exe" and click on that.  This will download a file named "Default_EXE.reg"  Save this file someplace where you can find it again.  I suggest saving it to your Desktop.

Follow the directions you read at the top of the page.

It's quite possible that other file associations have become damaged as well.  If you receive the same sort of message when trying to open other file types, just come back to the page, download the file for whatever file type you're having problems with, and again follow the directions.
