On Sun, 1 Aug 2010 02:03:00 +0000, oogieb wrote:

Can I scan a document into word and then edit it or change it.

Scanning a document (or anything else) produces a picture of that
document, not the text that's within it.

If you have OCR (Optical Character Recognition) software installed,
you can run that scanned image through it, and turn the picture into
text. The accuracy of that conversion may not be 100%, but if you use
good OCR software and you scanned a clean document, it should be close
to 100%. Whether it's 100% or not, you can then open that OCRed image
in Word (or any other word processing program) and edit it.

jetta48 say:

Hi


Which/what files do you mean?


If you mean personal files such as music, documents, videos, photos, etc. then these are not 32 or 64-Bit specific. Hence you can move them and use them on your new computer. You do still have to have a program that will open them installed  this is nothing to do with 32/64-Bit though.


If you mean programs then 32-Bit programs will usually work on 64-Bit Windows 8. There are a few exceptions so check the program publisher's website for compatibility information. There is also a possibility that some programs that worked on Windows 7 will not work on Windows 8 for reasons unrelated to 32/64-Bit, so again check the program publisher's website.


If you mean hardware drivers then these are 32 or 64-Bit specific they are not interchangeable between 32 and 64-Bit Windows. You must get the correct "bitness" of driver.
