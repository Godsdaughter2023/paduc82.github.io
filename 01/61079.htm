There's more than one way to do a clean install, so you'll need to decide on your own strategy. However, in order to succeed, a clean install must be performed on ...

The reward for all your effort is a computer that simply works without drama and without frustration.

If you want to go to Windows 7, buy a retail copy of it and follow the steps that it gives you.

But note that Windows 7 is very old, and most copies you might see for sale are pirated. So be very careful who you buy it from; Amazon.com is probably the safest source.

8 is also no longer available. You would have to buy a copy of 8.1. But note that you can *not* upgrade from 7 to 8.1. You would have to do a clean installation.

Having said the above, let me tell me tell you that I recommend against going to 7 or 8.1. You have an old computer. It may not be compatible with either 7 or 8.1, and even if it is you might need drivers that are not available for your old hardware.

Two other points:

1. Such old hardware is likely to be way underpowered to run 7 or 8.1 with acceptable performance.

2. The hardware in such an old computer is not likely to last much longer, and as far as I'm concerned, it's not worth putting money into it for hardware upgrades and a new version of Windows. You are almost certainly better off buying a new computer with Windows 10.

Fear not, in the worst case scenario, the data will be accessible by placing the disk into another PC. admittedly, if you cannot, or do not feel confident enough to do this for yourself, you will need to take it to a facility that can, ideally one that you can get a recommendation for.

 

In the meantime, the amount of advice that this forum can give is limited by our code of conduct...

 

If you are using XP home edition, the in-built administrator account probably does not have a password set. You can boot to safe mode where this account will be selectable at logon. Use it to logon and change the password of your normal logon.

To boot in safe mode, keep hitting F8 twice a second after turning on the PC until you get the boot menu where safe mode is selectable by the up/down arrow keys, then hit enter.

 

If you are using XP professional, try logging on using the built-in administrator account. You do not need to do this in safe mode (though you could if you want). If your normal logon screen is a user icon display hit ctrl-alt-del twice to get to a text input logon. Enter Administrator as the user name and try a blank password.

See also http://support.microsoft.com/kb/321305
