can't type lower case letters for changing password, only capital - Microsoft Community

claudia asauer say:

can't type lower case letters for changing password, only capital letters come up

When I go to change a password and you have to copy the letters and numbers, I can only type Capital letters, why don't the lower case letters show up?
kosh vorlon say:

Hi Claudia,

 

I seriously doubt the problem has anything to do with your keyboard itself (unless all cap problems occur in other places as well and on other sites and in non-internet programs like Notepad or Word or whatever) but is a matter of how those things work and are designed by the web site designers on the sites you are using.  What you are dealing with is called a Captcha http://en.wikipedia.org/wiki/CAPTCHA.  While this "caps only" may occur on that particular page or site, I sincerely doubt it occurs on every site that uses such a feature - you may not be using or ever seen a site where it works differently.  Here are a couple of things which may help explain why you are experiencing what is occurring:

 

1.  The process is based on inputting the hard to read characters into a text box to verify that a live person is there and improve security from bots.  Different programs use different features and code things differently.

 

2.  Some Captchas are case-sensitive and others are not (it may depend on the product purchased or how it is programmed on the web site or both).  Generally if it is case-sensitive I've seen it say that's the case in small print beneath the image.

 

3.  When programming it, the designers of the web page can typically decide whether or not to make entry case-sensitive (or just the view of it appear in all caps even if properly entered in both upper and lower case) but often can't control whether or not the presented images show case-sensitive images.  If they want it case-sensitive, then the text box must accept case-sensitive letters (even if they don't show up that way) or the entry will be rejected.

 

4.  If it is case sensitive but the images are not, then one way around that problem from a design perspective is to make the box only accept letters in a certain case and set case sensitivity to only permit entry of that case.  To make it easier on the user and prevent calls that it isn't working, they can control the input box so it only allows upper case letters and set case sensitivity to require all upper case letters.  Thus even though you may see the image in various cases, the box will only allow upper case letters to be entered which will allow the program to accept the entry.

 

5.  If it is not case sensitive, then they may still program the box to show only upper case letters (for whatever reason)which will work fine since it doesn't matter what case is entered in that situation.

 

6.  If you're having trouble with entry and suspect accepance of only upper case letters is the cause, then try several more images before hitting the panic button.  I sometimes find I need to try 4 or 5 (they often have a switch button to choose an image that you think you can read properly when the original or other choices are more difficult - I have trouble sometimes in any case even with my reading glasses and still need to try several times before it works) - so don't immediately assume it's a design flaw rather than simple problems being able to read the wretched things.  If certain there's a problem, try contacting the web site's technical support team for assistance.  They may either be able to get you past the recaptcha image or reset your password (though you probably then won't be able to choose).

 

7.  This can sometimes be a function of the browser being used.  Before assuming it is the keyboard (which is highly unlikely if it only occurs in this one text box), try using a different browser as some pages work differently with different browsers.  If you generally use IE, then download Firefox, Chrome, or Opera and see if the same thing occurs.  If so, it probably isn't the browser. 

 

8.  At that point, if you cannot get access due to this (forget the "problem" if it allows you to enter and get past it despite showing all caps - it isn't you and isn't a problem) and the web site cannot help, try using a different keyboard.  If it's a laptop, try a USB keyboard and see if that resolves the problem or access the onscreen keyboard from Ease of Access in the Control Panel and see if that works.  If so, then you may have a keyboard problem.  First try updating the keyboard driver, firmware, and/or sofware from the computer manufacturer or keyboard manufacturer being certain to use the downloads applicable to your machine and your version of Windows.  If that doesn't work, you may need to replace the keyboard even if it otherwise works fine - but I most sincerely doubt that will happen or prove to be the case unless you have more keyboard-related problems than just that.

 

Somewhere along the way, you will discover it isn't a problem or find out the cause of the problem (there really aren't that many options beyond the above that can cause this type of thing and not be found using the above ideas, tests, or procedures).  I'm almost willing to virtually guarantee that you don't really have a problem or one of these will identity it and depending on what it is, give you an idea where to go next in terms of resolving it.

 

I hope this helps.  Again, I believe you're most likely seeing how the website operates and not a real problem unless you cannot gain access after at least a half dozen tries (and perhaps getting someone else who has better eyesight to try as well) and the site's technical support people confirm that what's happening shouldn't be happening.

 

Good luck and best wishes.

Hi Concerned127,

To resolve this issue, you will need a Windows 7 DVD since going through the installed recovery options have failed. Once you receive a DVD, follow the steps to use the dism tool. this option allows you to delete pending updates.

***Win RE***
Boot from the Windows 7 DVD, select your keyboard and language options, select Repair your computer, then select the Command Prompt.  From the Command Prompt run the following command:
Dism /image:c:\ /get-packages

This will produce a list of installed updates and packages.  They will look like this:

Package Identity : Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0
State : Installed
Release Type : Security Update
Install Time : 2/11/2010 1:01 PM

Find the most recent one and then make a note of the package identity.

 

Then run:

dism /image:c:\ /remove-package /PackageName:Package_for_KB978262~31bf3856ad364e35~x86~~6.1.1.0 (name of update from previous step)

 

You will get a status of the removal until it is complete.  Then reboot your PC.

You will want to start with the most recent and remove them one at a time to find the problem update.

Hi Lisa,

If your Wireless Bluetooth Keyboard suddenly stops working, a force shut down may help resolve the issue. To do this, follow these steps:

  1. Press and hold the power button on your Surface for 30 seconds and then release it.

  2. Press and hold the volume-up button and the power button at the same time for at least 15 seconds and then release both.
    The screen may flash the Surface logo, but continue holding the buttons down for at least 15 seconds.

  3. After you release the buttons, wait 10 seconds.
  4. Press and release the power button to turn your Surface back on. You should see the Surface logo.

After the force shutdown, try to re-pair the Wireless Bluetooth Keyboard.

You may also try to run the Hardware Troubleshooter. Go to Control Panel > Hardware and Sound > Configure a device.

If running the hardware troubleshooter did not work, uninstall and reinstall the Wireless Bluetooth Keyboard driver.

  1. Go to Device Manager.
  2. Expand the Bluetooth.
  3. Right click the Wireless Bluetooth Keyboard driver and choose Uninstall.
  4. Restart your Surface and let Windows reinstall the driver.
