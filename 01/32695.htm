Hi,

Welcome to Microsoft Community. I appreciate your effort in resolving this issue.

When you install software that uses the UIRibbon.dll dependency, the software should automatically register the file for you. In some cases your DLL file may not register properly, and as a result, will provide a "UIRibbon.dll not registered" error. Fortunately, you can use a built-in utility called "Microsoft Register Server" (regsvr32.exe) to re-register your UIRibbon.dll file.

Follow the below steps.

How to re-register UIRibbon.dll from an elevated command prompt.

  1. Press Windows key + X.
  2. Select/click on Command Prompt(Admin).
  3. You will be prompted with a permission dialog box.
  4. Click Yes.
  5. Type the following command: regsvr32 /u UIRibbon.dll.
  6. Hit ENTER. This will UN-REGISTER your file.
  7. Type the following command: regsvr32 /i UIRibbon.dll.
  8. Hit ENTER. This will RE-REGISTER your file.
  9. Close the command prompt window.
  10. Re-start the program associated with UIRibbon.dll error.

Let us know the status.
