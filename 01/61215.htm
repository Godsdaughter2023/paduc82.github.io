Hi Al Ahner,


In fact, my usual approach is:
For example, I wanna change drive E: to drive K:,
in Registry Editor, navigate to HKEY_LOCAL_MACHINE\SYSTEM\MountedDevices, export this hive key to a .reg file, this is very necessary;
in its right pane, first rename \DosDevices\K: to \DosDevices\Z:, because \DosDevices\Z: usually is not be used, this step is very necessary;
and then directly rename \DosDevices\E: to \DosDevices\K:, and then reboot.
So, I don't need to use again the disk management tool.


My experience is:
Either using Disk Management GUI tool, or using mountvol command, or using diskpart command then using its "assign letter=<Drive letter>", and so on, all of these lists are incomplete, only in this registry hive key location, its mounted disks settings' list is complete, and these volume mount points' drive letters also can be renamed or modified very well in here.
