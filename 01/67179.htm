This issue may occur if Windows Backup skips some files while Windows Backup can't find them. When you perform backup it backs up the Documents, Pictures, Music and Videos through user libraries. So, if Windows finds any change in the folder path then the libraries won't recognize them unless you navigate to the corresponding library through Windows Explorer.

As a initial step, I would suggest you to check if Windows Backup service and Volume Shadow Service is enabled.

1. Press Windows key + R top open Run command.

2. Type services.msc and hit Enter.

3. Search for Windows Backup service. Check if it shows running next to it as the status.

4. If yes, right click on the service and click on stop.

5. Now, again right click on it and click on start.

6. Set the startup type to Automatic, if it is set to Manual startup.

7. Search Volume Shadow Service and repeat the steps.

If the issue still persist, try the following steps and check if it helps.

1. Click on "Change backup settings" from troubleshoot dialog
2. Expand the user node whose libraries are getting reported as skipped.
3. Deselect the library that is causing the issue.
4. Expand the "Computer" node in the tree view and select the redirected location of the folders (if any).
5. Save settings.

first this is a common scam.
Likely they told you the OEM SLP key which is common knowledge. Your windows is likely preinstalled so would have the OEM SLP KEY. An OEM SLP key is common to all models of computers of a manufacturer and varies only by the edition of windows.

Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

 

A)   OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the special instructions found only on that Manufacturer's computer hardware. So when Windows was installed using the OEM SLP key (at the factory) Windows 7 looks at the motherboard and sees the special instructions and Self-Activates. (that's why you did not need to Activate your computer after you brought it home)

 

B)    COA SLP: This is the Product key that you see on the sticker on the side (or bottom,[ or inside the battery compartment]) of your computer. It is a valid product key, but should only be used in limited situations (such as if the OEM SLP key stops self-activating for whatever reason). The key must be activated by Phone.
