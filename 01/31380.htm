Hi,
 
1. Were you able to save files earlier without this issue?
2. Do you remember making any changes recently?
 
Follow the steps mentioned below and check if they help resolve your issue.
 
Step 1: Check Event Viewer to get detailed information about the problem. Refer the following links.
http://windows.microsoft.com/en-US/windows-vista/Open-Event-Viewer

http://windows.microsoft.com/en-US/windows-vista/What-information-appears-in-event-logs-Event-Viewer
 
Step 2: Try saving a file on the desktop and check if the computer freezes. Also open task manager and try ending the SearchIndexer.exe process. Do this as mentioned below.
 
a. Right click on the Task bar and select Task Manager.
b. Click Processes tab and search for SearchIndexer.exe.
c. Select it and click End Process.
 
Step 3: Temporarily try to disable the computer`s anti-virus program and check if the issue persists. If the issue does not occur, check if there are any updates available for the security program
 
http://windows.microsoft.com/en-US/windows-vista/Disable-antivirus-software  
 
Note: Ensure to enable your anti-virus back on the computer after you finish these steps. It is not recommended to disable these settings on the computer. This is just to troubleshoot the issue.
 
Step 4: If this does not help, place the computer in clean boot to check if the computer hangs while saving a file. If the issue does not occur in the clean boot state, then you may try the steps from the following Knowledge Base Article and see which application or program is the cause.
 
http://support.microsoft.com/kb/929135
 
Note: Make sure you put the computer back to Normal Startup once troubleshooting is completed.
