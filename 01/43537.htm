Hi JohnParkUZ:

If you were fully patched as of 11-Apr-2017 when extended support for Vista SP2 and IE9 ended (and that may not be the case if you've been manually installing all your Vista SP2 patches for over a year) it's likely that automatic Windows Updates continue to get stuck on "Checking for updates..." if you have MS Office 2007, MS Office 2010 or the MS Office Compatibility Pack SP3 installed on your system.  See my post # 404 in the bleepingcomputer thread Windows Vista Update Hangs at "Checking For Updates" for further information.

If you'd like to check your Vista SP2 machine to see if you're missing any important security updates, see my post # 419 in that same thread.  There are two options presented in that post - either manually pre-install a small number of speed up patches to speed up Windows Update (you only need to apply the speed up patches that are currently missing from your system) or use Torsten Wittrock's WSUS Offline Update tool to install your missing security updates.

If you have MS Office or other Microsoft products that will continue receive security updates after 11-Apr-2017 then learning how to use the WSUS Offline Update tool is the better option.  First-time users sometimes find this tool a bit confusing because it's a two-step process - first run UpdateGenerator.exe and select what updates you'd like to install from the interface (options for Vista SP2 are located on the Legacy Products tab) to download missing updates, then run UpdateInstaller.exe (located in the Client folder) to finish the installation.
