lemp say:
Error message an installation is running. How do I stop it when I don't know what it is. Can't install or uninstall any programs. Been going on for months.

The Microsoft Windows Installer is msiexec.exe.  If you open Task Manager, you should see it there (on the Processes tab) and be able to stop it.

Now see if you can install the application that you want to install.

The potential problem is, however, that even if you successfully stop the installer and successfully install the application that you want to install, that the installer will start up again when you reboot (I assume that you've rebooted the machine at least once if this has "been going on for months").  If this happens, you'll have to investigate the startup entries in your registry. AutoRuns by Sysinternals is a good tool for this.  After it stops scanning, click the "Logon" tab and look at each item to see if it ought to run.
