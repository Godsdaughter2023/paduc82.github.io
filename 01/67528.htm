Tracy -

So is the problem that you want Windows Spotlight images but that they are not changing?

At Settings > Personalization > Lock Screen what is chosen in the drop Down menu under Background? Try Windows Spotlight. Restart PC to see if the pictures change after a few days. Sometimes they hang around awhile.

But be aware that for Lock Screen, Background (Windows Desktop picture in Windows) or ScreenSaver (if you use that) that those pictures in the Spotlight and various Background Themes are much higher resolution and very expensive pictures compared to using ordinary pictures from our Pictures folder.

If you want the Lock Screen Spotlight pictures and they won't change then Try this:

1) Create a System Restore Point: http://home.bt.com/tech-gadgets/computing/how-t...

2) open Settings > Personalization > Lock Screen. Under the Background option, choose Picture and set a picture as default lock screen background.

3) Unhide all hidden folders in File Explorer > View > Change File and Folder Options > View > check box for Show Hidden Folders, Apply , Save.

4) navigate to the following folder,

C:\Users\<username>\AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_<characters>\Settings

Delete roaming.lock and settings.dat files

5) Go back to Lock Screen settings and change dropdown menu to Windows Spotlight.

6) If this does not solve it, right click Start button to open Administrative Powershell.
Copy and paste the following into the box, then press the Enter key:

Get-AppxPackage Microsoft.Windows.ContentDeliveryManager -allusers | foreach {Add-AppxPackage -register "$($_.InstallLocation)\appxmanifest.xml" -DisableDevelopmentMode}

Now check if Lock Screen wallpaper has changed.

Report back results of all steps.
