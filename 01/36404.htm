Data Execution Prevention (DEP) is a set of hardware and software technologies that perform additional checks on memory to help prevent malicious code from running on a system. I would suggest you to turn on DEP on your computer.
 
How to turn on/off DEP in Windows XP.
a. Log in as administrator
b. Click the Start menu, right-click on My Computer and Choose “Properties” from the context menu. On the “System Properties” window, click the “Advanced” tab .
c. Click settings button under Performance
d. Click the “Data Execution Prevention” tab
e. Choose either “Turn on DEP for all programs and services except those I select:” or “Turn on DEP for essential Windows programs and services only to select the Option policy”
f. If you choose “Turn on DEP for all programs and services except those I select:“, click Add to add the programs that you do not want to use the DEP feature.
g. Click OK twice
h. Restart the computer

greg carmack - mvp 2010-2020 say:

 I'd use an imaging solution to transfer the image you have sized down to a smaller size than the SSD.  The best imaging solution I've found is Macrium Reflect is a Free and Easy To Use Backup Utility .

Install Macrium to save externally the image of the System partitions it says are needed.  Create the Rescue Disk.  Swap in the SSD, boot the disk, apply the image to the SSD after clearing the drive if necessary.  All settings are automated, you only need to drag the needed partitions to the SSD, click the link it creates below C to size it precisely to fill out the SSD.  After removing the disk Reboot the PC into Windows 10.   Sometimes it's necessary to run a Startup Repair to get it going. 

Keep in mind that at it's best the Upgrade is an inferior install so I'd also consider doing the vastly superior Clean Install Windows 10 Upgrade. You can buy an inexpensive enclosure for your hard drive to use as an External to import your files.
