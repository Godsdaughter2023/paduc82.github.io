It won't affect an existing program but if you create a System
Restore point and install a new program or update. When you
revert to that system restore point the program or update is
gone.

This is why it is important to create a System Restore point
before installing software or updates...if the new software
causes issues...reverting to the the system restore point will
clear it up.

Hi ronrod60,

 

Welcome to Microsoft Answers community.

 

Was it working fine as you expected earlier?

Which version of Internet Explorer you are using?

 

Try resetting Internet Explorer to default settings.

You can do it by visiting the below link and click on Fix it button/follow the troubleshooting steps provided in it.

http://support.microsoft.com/kb/923737

 

 

The below steps may help you resolve your issue.

1. Right click on Internet Explorer(IE) shortcut icon and select properties.

2. Now select the Maximized option for the Run box. 

3. Now click on OK button.

 

Thereafter, any time you click the IE icon, IE will open in full screen mode.

 

Programs like Internet Explorer will open to the same size and placement as they were when they were closed.

 

If you want Internet Explorer to open full-size when you start it up, you must first start IE, and then maximize the window. Now click File, and then Close. Now restart IE, and it should start up in a full-sized window. If half of the window is off the screen when you start a program, bring the window to the location you want and then click File and then Close and it will re-open in this position the next time.

 

Changing the size of windows that appear when you click on an Internet link takes a few more steps.

 

Open Internet Explorer. Now open another instance of Internet Explorer so that you have two running at the same time. In the second one, right-click a link and select “Open In New Window”. If the new window does not start in full screen, simply resize the window by dragging the edges out to the sides of the screen. Do not use the max button.
