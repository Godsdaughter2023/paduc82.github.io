manuelcapiro say:
Hello UNHAPPY_USER_920,

Windows 10 has a built-in speaker test that allows you to test your 5.1 surround sound. The software is not always reliable and can provide you with false positives.

In order to output 5.1 audio from your PC, the sound card must have 5.1 support. As of now, not all onboard soundcards are configured to output 5.1 surround signal. Desktops with newer motherboards usually have it even on low-end.

Make sure your system is actually able to output 5.1. Furthermore, check if all the cords and wires are properly attached and update the driver for your sound card to the latest version.

Press Windows key + R to open a Run window and type “mmsys.cpl“. Hit Enter to open the Sound Properties.
Go to the Playback and select your playback device that is capable of outputting 5.1 sound.
The name might change depending to your soundcard. With the speakers selected, click on Set Default and then hit the Configure button.
In the Speaker Setup window, select 5.1 Surround and click Next.
If you see multiple 5.1 Surround entries, select the one that resembles more closely how have placed your speakers.
Then, make sure all the boxes under Optional speakers are set and click Next.

Doing this will redirect the audio meant for the missing channel to an active one. This way, you will ensure that you’re not missing any important audio.
This next dialog box is about selecting the full-range speakers. Most 5.1 systems dont have full-range speakers since the audio task is split across multiple channels, then test it

If you have 5.1 speakers but you are unable to select 5.1 Surround after clicking the Configure button (option greyed out or not available), the issue unfurtunatelly is part bug.

Sorry for that

Hope it helps, have a good day

Usually 5.1 and 7.1 Speaker systems have a processor board built into the Sub woofer. You usually only plug in from PC sound card, 3 jacks (L, R) (RL RR) & (C SUB) which go into the sub woofer and then from the sub woofer, the 5 speakers output is from there.

So usually any sound card with these 3 jacks will work. I have VIA HD Audio onboard sound on the PC I built almost 10 years ago. And Logitech 5.1 surround with a manual analog master volume, Bass and Matrix control that sits on my desk.

If the Matrix button is ON, and you run the test for 5.1, which plays a sound for each speaker and shows the graphics of it, then it will only play L & R speakers in the test. But Turning OFF the Matrix, all speakers play a sound in the test.

But with Matrix ON, playing audio, you get the 5.1 sound.
