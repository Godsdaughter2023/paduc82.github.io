ozziedave52 say:

Success!  After some stuffing around, here's how I managed to import my Internet Explorer favourites from my old computer into Edge on my new Windows 10 PC:

  1 - On the old computer, export the favourites to a file:  it will be called bookmark.htm by default. 

  2 - Copy this file onto the new computer in the Windows C-Users-<your.name>-Favorites folder.

  3 - This is important - rename the file to bookmark.HTML (ignore the warning about making the file unusable).

  4 - Open the CHROME browser, then import the favourites from this file.  (If you didn't rename it, Chrome won't even find/display it.  Stupid Chrome).

  5 - Chrome should now show all your imported favourites!

  6 - Now open EDGE and import them easily from Chrome.
