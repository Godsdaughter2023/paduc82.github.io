@Olivier325,

 

It sounds like you are previewing your video inside of Movie Maker and that is where you are seeing the looping.  Yes, the preview inside of Movie Maker loops by design.  What I was talking about earlier was when you have saved your project as a completed movie file (File > Save movie).  The resulting movie should not loop then.  You should only consider Movie Maker as an editor, not for the final presentation; that's what the Save movie option is for.

 

As for your issue with two added audio clips, there are settings on the Project tab to determine the audio balance.  It sounds like you might have it set to emphasize one audio track over another.  Movie Maker will "duck" the audio of the non-emphasized track to make it quieter.  If you don't need this behavior, be sure to choose 'No emphasis'.  Also, double-check your settings on the Audio Mix to ensure that it is right in the middle.

 

Alternatively, if you are adding the second audio after the first, it could just be the format of the second file. Are there any differences in the file types (e.g. .wma vs .mp3)?  Do you have the same problem if you just add the second song to a new project?

 

If it looks like the problem is the second audio file itself, try converting it to .wma format instead.  See if that works differently.
