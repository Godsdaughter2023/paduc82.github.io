nithyananda j say:

Hi Mr Cheese,

A 64-bit version of Windows OS supports the backward compatibility for x86 (32-bit) CPUs. In a 64-bit Windows
you find two folders (as you posted):

C:\Program Files\
C:\Program Files (x86)\

%% These actually represent the current path, they are part of predefined path environment variables. Then of course you have user variables and default system variables. 

Once an application is under the C:\Program Files (x86) this is the classified as a 32-bit application and will use the 32-bit registry entries which are supported.

SysWOW64 Is a known Windows registry entry which hold 32-bit registry data.

WOW64 means Windows 32-bit On Windows 64-bit.

Note: If you move the application installed in "C:\Program Files (x86)" to"C:\Program Files" the applications might not work, as the registry value changes.

I hope you find this information useful.

If you're referring to the icon that shows on the taskbar for a running program (Explorer, for this case), it cannot be done.  Just like if you had a shortcut to Notepad and changed it's icon to be a gold star, the Notepad program icon would still be the blue notebook.

There is no way to change the icon of an EXE application without re-building the application.
