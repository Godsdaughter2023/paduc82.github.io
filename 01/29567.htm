I'll try to outline the steps, but I am thinking some of these are coincidental, but none would hurt. (Hint: use a USB keyboard and mouse to help make these adjustments.) Be sure you are not docked or have any external monitors connected.

  1. shut down
  2. disconnect the type cover
  3. boot up
  4. before logging in, go into the accessibility icon on the login screen. turn all 3 on. use touch for this.
  5. log in
  6. go into settings | system | tablet mode
  7. switch the "make windows more touch-friendly when using your device as a tablet" to a different setting than it has. save the settings
  8. snap connect your type cover now
  9. shut down
  10. boot up
  11. go into the accessibility icon on the login screen. turn all 3 off. use touch for this.
  12. log in
  13. see if type keyboard is still not responding go to step 7.
  14. hopefully you are enjoying your now-working type cover. I think this the issue is caused by something in the new windows 10 tablet mode detection.
