The error "We couldn’t update system reserved partition" might occur if the System Reserved Partition (SRP) is full. The System Reserve Partition (SRP) is a small partition on your hard drive that stores boot information for Windows. Some third-party anti-virus and security apps write to the SRP, and can fill it up.

Try the troubleshooting procedures from below article and check if it resolves the issue.
