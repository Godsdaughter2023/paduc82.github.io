How to resolve problems installing the Windows 10 April 2018 Update. - Microsoft Community

ariel acu say:

How to resolve problems installing the Windows 10 April 2018 Update.

What should I do if I’m having trouble installing the Windows 10 April 2018 Update?  For example, what steps should I try if I’m having any of the following problems:

  • While updating to the latest version, Windows restarts and goes back to the old version.  
  • The update to the latest version of Windows gets stuck for hours, so I had to restart and go back to the old version.
  • I thought I installed the latest version of Windows, but now Windows Update says I need to install it again.
  • I tried to update to the latest version, but now I can’t sign into Windows or reach the desktop.
tiero chi say:
There are many reasons your device may not be able to update to the latest version of Windows. 

If you’re having any problems installing the latest version of Windows 10, first try the steps listed at https://support.microsoft.com/help/4089834/windows-10-troubleshoot-problems-updating.

If you’ve tried installing the latest version of Windows, and now you see a black screen or can’t otherwise reach the desktop, there may be a problem with a driver.  You may need to obtain a video driver from your PC’s manufacturer, and a compatible driver may not yet be available.  To restore your system, try these steps:

1. Try a Windows key sequence to wake the screen.  If you have a keyboard connected, simultaneously press the Windows logo key + Ctrl + Shift + B. If you’re on a tablet, simultaneously press both the volume-up and volume-down buttons, three times within 2 seconds.  If Windows is responsive, a short beep will sound and the screen will blink or dim while Windows attempts to refresh the screen.

2. If you’re unable to wake the screen, use the power button to restart your computer three times. This should start the Windows Recovery Environment.  

3. In the Windows Recovery Environment, at the Choose an option screen, select Troubleshoot > Advanced options > Startup Settings > Restart. 

4. After your PC restarts, you'll see a list of options. Select 4 or F4 to start your PC in Safe Mode. Or if you'll need to use the Internet, select 5 or F5 for Safe Mode with Networking. 

5.      Once you start your PC in Safe Mode, select the Start button, then select Settings > Update & security > Recovery > Advanced Startup > Troubleshoot > Advanced Options > Go back to the previous version.

If you have tried the steps and are still experiencing an issue upgrading to Windows 10 April 2018 Update, please create a new thread with details about the error you receive.
