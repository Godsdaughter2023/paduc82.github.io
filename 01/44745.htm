tzon say:

Try also running them as an administrator (by right clicking on them).

 

If that doesn't work, the only solution I can think is running them in a virtual environment. You can download and install Oracle's Virtual Box from here https://www.virtualbox.org/wiki/Downloads (for Windows host). Then you can use this program to install as a virtual machine a previous version of Windows where the games used to work. You must have the installation media of the old operating system (eg. Windows 95 or XP). You may as well find it and download it from the internet in a .iso format. After successful installation, the older operating system will be running in a window within Windows 8. There you will be able to play your games. For further instructions read the manual here https://www.virtualbox.org/wiki/Documentation.

 

Good luck
