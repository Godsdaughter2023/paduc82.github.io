Windows Live Mail has dozens of different ways of saying can't connect to server, each one corresponding to a different error. If you post the entire error message, we might be able to see what's wrong.

Apart from that, the default DeltaSync connection to Hotmail servers is quite demanding of connection quality - much more so than connecting a browser to a web server, say. So even though the quantity may be good, if the quality's not up to it, you'll have difficulty connecting.

I would set up the account again in Windows Live Mail.

  1. Press Ctrl-Shift-T to start the Add your email accounts wizard.
  2. Enter the Hotmail email address, password and display name (for outgoing messages), then select Manually configure server settings and click Next.
  3. In the Server type box, select POP3, then enter these server addresses:
    Incoming mail: pop3.live.com
    Outgoing mail: smtp.live.com 
  4. Select all three check boxes and ensure that the incoming port is 995 and the outgoing is 587.
    The Log-on username is the full email address. Don't change the Clear text authentication setting.

Now you have two instances of the same account. For the new one, right-click on the account name in the folder pane and select Properties. On the General tab, deselect Include this account when receiving mail or synchronizing, and change the account name (the top box) to something sensible like Hotmail (POP3)

When you're out and about and you find that you can't connect with your old Hotmail account, select the new account from the dropdown beside the Send/Receive button. You should receive new messages in the inbox, and you should be able to send as well. Sent messages should eventually (i.e. when you're back home and connecting properly again) turn up in the Sent items folder under the old account as well.
