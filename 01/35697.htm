You can run the Printer Troubleshooter at Settings > Update & Security > Troubleshoot.

Rather than spending too much more time fiddling I'd simply Clean Install the printer which always works, lasts longer and actually saves time:

1) Power up the printer, if wired unplug it's USB cable from PC.

6) Extract Printer software, right click Setup file to Run as Administrator. Follow the setup screens closely as you install the software. For USB it should prompt when to plug in the printer.

Repair System Restore - The command-line parameter is case sensitive:

1. Insert the Windows XP CD
2. Click [Start] [Run]
3. Type  the following command and press Enter "rundll32.exe advpack.dll,LaunchINFSection C:\Windows\Inf\sr.inf" [Case Sensitive]

System Restore core files should be reinstalled. Alternately, you may also open Windows Explorer, navigate to "C:\Windows\Inf" folder and right-click sr.inf (choose Install) (sr.inf shows up as sr in windows explorer).

System Restore disabled, probably by malware:

HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\Windows NT\SystemRestore
In the right-pane, delete the value "DisableConfig".

The System Restore Utility May Be Suspended on a System Drive Even Though There Is Enough Disk Space -http://support.microsoft.com/default.aspx?kbid=299904

millerjane say:

Green Screen Video

Our students have a green screen and would like to edit their videos in Windows Live Movie Maker 2012 essentials package.  Currently, we do not see this capability. Please advise.

considerate_guy say:

Green screen capability really requires at least two video tracks (one for the green screen and one for the replacement background). It also requires a chroma-key effect that can turn the green areas of the video transparent so the replacement background appears.

Movie Maker has only ever allowed for a single video track and hence has no green screen capability despite years of requests for more than one track (to allow picture-in-picture, picture and video overlays, compositing, green screen, etc.)

Some people have been able to "trick" the older version of Windows Movie Maker 6.0 into doing green screen as a custom transition but that's pretty tricky to accomplish and requires some clean-up afterward.

If you have access to iMovie on a Mac, that has green screen capabilities. Another relatively inexpensive option is Nero Video 2016 which has multiple video tracks and can do green screening.

A free option is to use DaVinci Resolve Lite (free) which has this capability. Another one I can think of is Hit Film Express 3.0 (free).
