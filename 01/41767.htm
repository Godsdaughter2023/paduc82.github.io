    -Function Discovery Resource Publication

    -Peer Name Resolution Protocol

    -Peer Networking Grouping

    What you report suggests that defrag.exe is available but that something got changed in the registry that prevents the program from running normally. To find the cause is very difficult if not impossible. On the other hand the benefits from defragging a disk are now quite small because disk caching is so advanced that it mostly compensates for file fragmentation.     • Use a third-party defragger. This is a risky move.
