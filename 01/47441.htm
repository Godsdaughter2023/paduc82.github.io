Removing malware can be tricky business, though is best done when first noticed and before it has the opportunity to create worse problems. The best way to stop malware, though, is to install and regularly run a trusted antivirus/antispyware program.
 
Multitasking: Many computer users multitask without even realizing it. It's simple common sense: The more programs a computer uses, the higher its CPU usage will go. If you're the kind of user who must be multitasking, tweak your system configuration (msconfig.exe) to halt any unnecessary applications that run at startup.
 
Malfunctioning or Incompatible Software: Some programs have certain hardware requirements (graphics card, RAM, processor speed). Occasionally, a program can simply become corrupt. In some cases, these programs will not open or run. In other cases, these programs will utilize any remaining CPU resources in an effort to function. The solution is to make sure the software is compatible with the “system's operating system and hardware specs” and to remove the software if it is not compatible.
