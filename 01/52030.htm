In that case, you can disable the Windows Automatic Maintenance and re-enable it at the time of your convenience.

To disable Windows Automatic Maintenance, follow the steps below:

  1. Log on to your Windows 8.1 device using your administrator account.
  2. Press Windows + W keys and type Task Schedule.
  3. Open Task Scheduler.
  4. Locate and select Task Scheduler Library.
  5. Under Task Scheduler Library, select Microsoft, Windows, and then Task Scheduler.
  6.  Right click Idle Maintenance and select Disable.
  7. Right click Regular Maintenance and select Disable.
  8. Close the Task Scheduler windows, and restart the device to allow changes to take effect.
