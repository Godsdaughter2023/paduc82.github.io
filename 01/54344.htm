           1: left click on search bar

           2: Type in "regedit" then hit enter

           3: When registry editor opens click on the view tab

           4: Open find

           5: Type in the open dialog box "initialkeyboardsettings" and press the find button

           6: When string appears double click on the initialkeyboardsettings string and change the string value to 2

           7: Next click on view tab then click on find next.

           9: After entire registry has been searched a dialog box will appear stating "find has finishing searching registry editor"  

           10: Shut down registry editor

           11: Shut down all files or programs open

           12: Go to power button and shut down computer

Reinstall Windows Installer Service

Have you recently installed any applications or drivers that might have interfared with the service? You could try removing them temporarily to see if it corrects the problem. Have you also tried doing a System Restore to a point in time before this problem started happening?

To reinstall the Windows Installer, rename the damaged Windows Installer files, and then reinstall the Windows Installer. To do this, follow these steps:
