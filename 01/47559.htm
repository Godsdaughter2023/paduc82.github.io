As Volunteer J noted, this indicates a problem in your project at around the 65% mark (about 2/3 of the way into your project).


It may be caused by an incompatible file type.  If you have a different video clip type or a new audio file added here, it's probably the cause.  Convert problems video files to WMV format.  Problem audio can be converted to WMA or WAV format.  For conversions, Freemake.com has a couple converters for video and audio.  If you do install freeware such as Freemake, be sure to choose the Custom Install option so you can uncheck any add-ons that you don't want.


The other possibility is that the project is a little too complex for Movie Maker to render all at once.  To test this theory, make two more copies of your project file (use File > Save project as).  Working in one of the copies, try deleting everything in the 2nd half.  Now see if it will save as a movie (it should).  If that works, go to the other copy.  Delete everything that is already in the first half.  See if the 2nd half will now save as a movie.


If you can save the movie in parts using the steps above, create one additional project, add the two video (movie) files created from the two halves and then save that as the final joined movie.
