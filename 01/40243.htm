In trying to solve another problem related to the Sound driver on my SP2, I discovered something that might in fact be related to your low volume problem.

There are two sound drivers available on the SP2 (see screenshot):

1) Realtek High Definition Audio Driver (should be the default driver with louder volume)

2) High Definition Audio Device Driver (a generic driver for HD audio from Microsoft without volume boost)

By default, the Realtek one should be what your SP2 is using. However, when you said some settings in the control panel are greyed out, it suggests you might actually be using the generic HD audio driver. I have tested the generic HD audio driver on my SP2, and the volume does indeed become much quieter!

To change the audio driver:

  1. Open Device Manager (You can press the Windows Key + X on your keyboard, or search for this by using the Start Screen) 
  2. Select Sound, video and game controllers.
  3. Check whether High Definition Audio Device is listed there.
  4. If yes, right-click it, and then select Update Driver Software…
  5. Select Browse my computer for driver software, then Let me pick from a list…
  6. You should be able to see a list containing the two drivers as shown in the screenshot.
  7. Proceed with installing the Realtek High Definition Audio driver, and reboot your SP2 when prompted.

Hope this solves your problem.

DriverView - Free - utility displays the list of all device drivers currently loaded on your system.

For Drivers check System Maker as fallbacks and Device Maker's which are the most current.
Control Panel - Device Manager - Display Adapter - write down the make and complete model

UPdate Driver (this may not do anything as MS is far behind certifying drivers) - then Right
Click - Uninstall - REBOOT this will refresh the driver stack.

Repeat that for Network - Network Card (NIC), Wifi, Sound, Mouse and Keyboard if 3rd party
with their own software and drivers and any other major device drivers you have.

Now go to System Maker's site (Dell, HP, Toshiba as examples) (as rollback) and then Device
Maker's site (Realtek, Intel, Nvidia, ATI as examples) and get their latest versions. (Look for
BIOS, Chipset and software updates at System Maker's site while there.)

Download - SAVE - go to where you put them - Right Click - RUN AD ADMIN - REBOOT after
each installation.

Always check in Device Manager - Drivers tab to be sure the version you are installing actually
shows up. This is because some drivers rollback before the latest is installed (sound drivers
particularly do this) so install a driver - reboot - check to be sure it is installed and repeat as
needed.

Repeat at Device Makers - BTW at Device Makers DO NOT RUN THEIR SCANNER -
check manually by model.

Manually look at manufacturer's sites for drivers - and Device Maker's sites.
