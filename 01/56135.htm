wunders say:

Windows XP is not very good at keeping the Root Certificates (those by which all other certificates are verified)  updated.  You should perform a manual update of your Root Certificates.  To do this, go to the following site:

 

"Windows Root Certifacate Program members"

  < http://support.microsoft.com/kb/931125/en-us >

 

then skip down to the paragraph entitled "Root Update Package (Intended for Windows XP only)".  Beneath that is an "Update Root Certificate" link.  Follow that link and eventually you will download a Root Certificate Update package.  After downloading the package, double-click on it to install it.  That should take care of your problem.

Network connection is mostly driven by the network configuration or it could be and issue with the network drivers. Here are some troubleshooting steps that you can take a look into:

Method 1: Turn on network discovery

  1. Open Settings, and go to Network & Internet > Ethernet > Change advanced sharing options.
  2. Tick Turn on network discovery.

Method 2: Find devices and content

  1. Open Settings, and go to Network & Internet > Ethernet.
  2. Click the Ethernet Connected icon and turn on the Find devices and content.

Method 3: Enable the Server service.

  1. Open Services via Start menu search.
  2. Navigate to Server service, and double-click it.
  3. Change the Startup type to Automatic.
  4. Click Start under Service status.
  5. Hit Apply, then OK.

Method 4: Update or reinstall your LAN/network driver
