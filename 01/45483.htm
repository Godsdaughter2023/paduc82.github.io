====================== Keyless Install ================================

How to perform a 'Keyless Install' of Windows:


Boot from the Windows 7 DVD disc you created and start the installation process.


After you enter user name and password you will be asked to: 'Type your Windows product key', do not enter a key.
Also located just below the box where you would enter the key is a box with a check mark to 'Automatically activate Windows when I am online', remove the check mark and click 'Next'.


You will now have a 30 day grace period in which to activate Windows.


During this 30 day grace period you can try resolving your problems. 
After you are satisfied all problems have been resovled then activate Windows.

tryx3 say:

As I also have a Dell Inspiron 1545, I can confirm that Dell will have put the CoA label inside the battery compartment [you have to remove the battery to see it as you have just done].

 

The disk itself is not critical as you can make one of your own [using the link I gave you in my first post] - the critical issue is the loss of the Product Key.

 

If you bought the PC directly from Dell with Windows 7 preinstalled then somebody has removed the label and therefore robbed you.  If you upgraded to Windows 7 later on by buying a retail disk then the disk casing or a card inside it will have the Windows 7 Product Key [or possibly the key is in an email if you bought it electronically].

 

Does your system report that it is activated still?  Look in Control panel, System - if it is activated then it will say so near the bottom of that window.  It might do so as Dell saved a special self-activating Product Key in the motherboard's firmware - this key should survive any reinstallation but it is possible for it to fail and that's when the Product Key on the CoA label comes into play.

 

 What did the "repair" consist of? 

 

If the "repair" did not destroy it then you should still have the Dell Factory Image Restore partition on your hard drive and you should be able to use this if you need to [the procedure is in your PC manual].  If you still have the application Dell DataSafe Backup Local installed then you can make a bootable USB [or DVD-R] of this DFIR partition [this is a valuable protection measure].  Making this USB also confirms that the DFIR partition is still sound [it can't install itself onto the USB if it does not exist & the USB itself is fairly easily testable].
