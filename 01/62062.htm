Hi,

For possible resolution regarding the error "'secure boot violation", see the steps below:

  1. Start the computer/PC, and enter the BIOS by pressing F2 or DEL key.
  2. Go to Advanced Mode by pressing F7. Find the BOOT tab at the top.
  3. Scroll down until you see the Secure Boot.
  4. Enter the setting and Key Management, and select Clear Secure Boot keys.
  5. Confirm that you will delete the keys. Escape out of the main window, and go to the EXIT tab.
  6. Look for the option to Save Configuration and Reset.
