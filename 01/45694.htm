What is a "roaming" Xbox 360 console?

The term “roaming” refers to using content that you purchased on one Xbox 360 console (say, console A) on another Xbox 360 console (say, console B). For example, you might download a game from the Xbox LIVE Games Store to console A, and then attach the hard drive that contains that game to console B. How to find SSID (Network Name) - Microsoft Community

xb1r hydros say:

How to find SSID (Network Name)

Login Information

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

How to Find SSID (Network Name)

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the gateway address “192.168.1.1” into the web address bar.
  4. When prompted for a username and password type in “admin” for both the username and the password.
  5. When you get logged in, on the main page you will find the “SSID”, this is your wireless network name.

noel d paton say:

First things first.....

1) If the Keys in your office are OEM Keys, then they cannot be used on anything other than the original PC with which they came (if they're still the sealed packages, then you should also have the disk!)

2) If the Keys are Retail, then you should have the disk that came with them

3) If they are Volume License Keys, you *may* be able to use them - if the license agreement is still in force.

 

Having said that....

a) You cannot use a Retail XP disk with an OEM Key - it requires a retail Key

b) If your disk has SP3 integrated, you can install using the default Key, and then use the Product Key Update tool to change teh Key to the one on your PC - assuming that the COA license is for Pro.

c) If the licenses in your office are for Retail Pro, simply use your disk and one of the licenses.

d) If the licenses in your office are valid Volume licenses, they still require a valid installation of XP Pro to upgrade
