I've solved the problem.

If anyone else has this issue then this is what I did:

1) Download the .Net 4 Standalone installer from here http://www.microsoft.com/download/en/details.aspx?id=17718

2) Double-click the downloaded file to start the installation process

3) Continue with the installation as far as the .Net splash screen where you are asked to tick the box to accept the T&Cs, but don't click OK.

4) Open Windows Explorer and navigate to your C drive.

5) Locate a new directory that the .Net installer created and extracted the install files to. It'll be named something like 780d92faf7f5c32970.

6) open the directory and locate the setup.exe file. Right-click it and select properties.

7) Click the Digital Signatures tab and click the Microsoft signature then click Details.

8) Click View Certificate, then click Install Certificate.

9) After a few seconds you'll get a "Certificate Successfully Installed" message.

10) Continue the installation from the splash screen and all should proceed OK.

Each brand and model of display monitors are completely different from each other. Manufacturers do not create a hardware or device that is exactly identical to their competitor brands.

If you connect multiple display monitors with different brand and model, your computer will have a conflict with its drivers and will generate communication problem within these monitors.
