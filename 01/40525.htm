cmdrkeene say:

A system image backup won't let you see or browse the backed up files, and can only be restored all at once (you can never use a system image to only restore "that one file I accidentally deleted").  You can restore the entire image, or nothing.

For this reason, I make use of both Windows 10's File History backup, as well as regular backup with system image.

But you should still see the backup file (inside a folder).  If not, I'd run it again.

a. user say:

Hello,

Thank you for posting your query on Microsoft Community Forums.

I would appreciate, if you can provide us the following information to help us understand the issue better?

  • What is the make and model of your system?

I would suggest you to uninstall and reinstall Graphic card drivers and check.

Uninstall and re-install the Graphic card device driver

a) Press “Windows Logo” + “X” keys on the keyboard.

b) Click on “Device Manager” from that list.

c) Go to Display adaptor and expand it  and you would see the Graphics Card device, right click          on it and then select “Uninstall”.

d) On the un-installation window, if you have an option: “Delete the driver software for this device” you may select that and then remove the corrupted drivers from the computer.

e) Follow the on-screen instructions to complete it and the restart the computer if prompted.

After the restart, go to the Device Manager again by following the steps mentioned above and then click on the “Scan for hardware changes” button. Check if Windows  detects the Graphics Card device and installs an appropriate driver automatically.

 If it doesn’t install the driver, I would suggest you to go to the manufacture website and install the latest graphic card drivers.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.
