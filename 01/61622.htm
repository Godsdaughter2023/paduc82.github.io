In most cases, you can safely ignore this error. It is triggered when the License Activation process tries to activate a Microsoft product, but the network is not ready yet.

Right Click on Start > Select PowerShell (Admin)
Copy/paste the following into the window then Press Enter.

slmgr /ato

You should then see the message "Acquisition of End User License was successful"


Event ID: 1000 Faulting App - laclient.exe
Navigate to C:\Users\User_Name\AppData\Roaming\LogitechGaming\laclient
Note: change User Name to your actual User Name in the above.
If there are files or folders inside, move them to another location and created a backup of those files.
Delete these files
In other words leave the folder empty.

Next, reset Windows Store Cache
Start > Type Run > Press Ok
Type WSreset.exe > Press Ok
Retry launching the store

Error 0x80070103
 
Cause
This issue occurs if the following scenarios are true:
  • Windows Update or Microsoft Update tries to install a driver a second time for an additional piece of identical hardware such as a graphics adapter.
  • Windows Update or Microsoft Update tries to update a currently installed driver to an unsuitable version of that driver.
 
Resolution
For the most recent available driver, contact the device manufacturer. This issue can be safely ignored if this applies to a driver. However, the driver may be offered again by Windows Update in the future. To hide the update so that it is not offered again, follow these steps, depending on your operating system:

 

• I.- Visit the Microsoft Update Web site:

<<http://update.microsoft.com>>

• II.- After the scan for updates is complete, click Custom on the Welcome page.

 

• III.- In the navigation pane, click Hardware, Optional. Then, the Update client displays the available updates on the Select Optional Hardware Updates page.

 

• IV.- .Expand the second update for the graphics card, and then click to select the Don't show this update again check box.

 

• V.- Click Review and install updates.

 

• VI.- Verify that the second update for the graphics card is not present, and then click Install Updates.
