When you ask Movie Maker to directly upload to YouTube, it does this internally in two steps.

  2. It logs on to YouTube and uploads the file.

In order to figure the cause of the problem, try these steps manually.

You can pipe the output to a .txt file to see the program that was run.

Example: To have the output of ipconfig /all and pipe it to a text file.
Open a command prompt with an elevated privilege.
Right click on Command prompt. Then select "Run as Administrator."
The command would be: ipconfig /all >output ipconfig.txt

Output ipconfig.txt is the name of the file you saved it in.
