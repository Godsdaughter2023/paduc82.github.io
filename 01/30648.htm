What happens when you try to open an attachment is a Windows function, not Windows Live Mail. Double-clicking on an attachment in Windows Live Mail will always invoke the Windows Shell > Open > Command recorded in the registry for the file type concerned. If the Open command is missing, probably nothing will happen.

If something happens to remove the Open command, Windows won't know what to do when you double-click in Windows Live Mail. How to deal with this depends on the file type concerned.
