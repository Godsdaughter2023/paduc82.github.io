ken blake say:
There are several choices. Here are a few ways:

1. E-mail the data from the old machine and read it on the new one.

2. Network the two machines and transfer the data over the network.

3. Remove the drive from the old computer and install it temporarily (or even permanently) in the new computer or in an external USB case.

4. Use floppies and span the larger files over several diskettes using a file splitting program (or any standard zip utility, which can also do this).

5. Copy the files to CDs or DVDs

6. Copy the files to a USB thumb drive.

Which of these is best for you depends on several things: how much data you have to transfer, whether you plan to keep the old computer, whether you are willing to spend a few dollars for a thumb drive if you don't have one, etc.

Hi

It should be fairly easy to do this.

The only thing you need to be careful about is to make sure that you do not move any of the XP System folders or User folders.

I would recommend that you only move the files, and leave the old XP folders. Once you are done, and you have all of the files you want to save, you can then simply delete that Old Data folder.

For instance, you can move the pictures to the C:\Users\your name\My Pictures folder (where  'your name' is the name of the account you use to log onto the new computer).

You will find My Documents, My Videos, My Music, etc, in the same location.
