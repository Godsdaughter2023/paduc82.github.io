thomas ferguson say:

The thumbs.db file is a thumbnail cache.  It is a file used to store thumbnail images for Windows Explorer‘s thumbnail view. This speeds the display of images because the smaller images do not need to be generated when the user views the folder.

It is possible to turn off the generation of these files; however, the trade-off is some slight decrease in system responsive as cost and vely little increase in available disk space as benefit.

Windows 7:

  1. Click the Start button
  2. Click on Control Panel
  3. Click on Folder Options
  4. Click on the View tab
  5. Under the Files and Folders section check the button next to "Always show icons, never thumbnails"
  6. Click on Apply
  7. Click on OK

andre da costa say:

You are going to need to purchase a new license.

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

As mentioned, if its a system builder license, you can try your luck and use chat support. Some persons have had luck doing so:

Activate Your Windows 10 License via Microsoft Chat Support
