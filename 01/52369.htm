saltgrass say:

The Windows.old folder is an archive created during a large update/upgrade and contains your old files and is used in case a rollback is required.

You can try removing it by using Settings - System - Storage and clicking on the drive at the top of the page.

Then go down to Temporary files and select.  If a Windows.old folder is on your system it should be listed.  Check the box and select Remove files.

There have been some files which were not removable in the normal manner and needed the user to change permissions in order to delete.
