job sam varkey say:

Hi MitchWza,

 

What happens when you try to disable APG/Accelerated Textures?

Why do you need APG/Accelerated Textures disabled?

 

Having the AGP Texture Acceleration setting as "Not Available" is a normal and expected behavior for the Nvidia graphics drivers (production versions 14.0 or greater). This setting does not need to be made available, enabled, or activated, and does not affect the performance of games or 3D applications.

When a game or application is checking for AGP or AGP Texture Acceleration, it is looking for non-local memory. Generally, local video memory is usually faster than non-local. It is recommended to contact the game or application vendor to get a setting or update that allows the game or application to check for local video memory first.

After setting the display language, if you find that the welcome screen or new user accounts is still using the old language setting, you can do like below

  1. Type "Control Panel" on "Search Bar," click on "Control Panel", click on "Clock, Language, and Region", choose and click on "Region".

  3. Then copy your current setting to both of welcome screen and new user accounts, restart it.

At least in my case, above steps, the problem has been solved.

    Welcome to Microsoft Answers community.

    I would suggest you to convert your hard disk from FAT to NTFS and check whether it helps.

    To convert a hard disk or partition to NTFS format

    1.    Close any open programs running on the partition or logical drive to be converted.
    2.    Click the Start button, click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
    3.    At the command prompt, type convert drive_letter: /fs:ntfs, where drive_letter is the letter of the drive you want to convert, and then press Enter. For example, convert E: /fs:ntfs would convert drive E to the NTFS format.
    4.    Type the name of the volume you want to convert, and then press Enter. You must use the existing name of the volume or the conversion will be canceled. You can view all available volumes in the Computer folder under Hard Disk Drives.
    Open Computer by clicking the Start button, and then clicking Computer.

    http://windows.microsoft.com/en-US/windows7/Convert-a-hard-disk-or-partition-to-NTFS-format

    To change the designation of a new drive to be a slave drive, follow these steps:

    1. Check the hard drive documentation or the label on the drive itself for master/slave jumper settings.
    2. Find the jumpers on the circuit board of the drive. Jumpers are plastic plugs with metal sleeves that form a circuit between a pair of pins. Jumper locations are labeled on the board with a J followed by a number (such as J20).
    3. Use tweezers or very small pliers to remove or reposition the jumpers to the configuration shown in the documentation or on the disk label.
    4. Install (or reinstall) the hard drive in the computer
