Reinstall Windows 10 on SSD

I recently finished a new build and inadvertently installed Windows 10 to the HDD drive and not the SSD.  What's the simplest way to reformat the HDD and reinstall on the SSD?

greg carmack - mvp 2010-2020 say:

Unplug the hard drive, boot the installation media to Clean Install Windows 10   to the SSD.

After install you can plug back in the hard drive making sure the SSD remains set first to boot in BIOS setup, wipe HD with Diskpart Clean Command then repartition it in Disk Mgmt for storage:  How to create a simple partition in Windows 7 (Quick Tip)
