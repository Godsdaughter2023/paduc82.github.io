lemp say:

In general, a "Recovery" partition is used to restore a computer to its "fresh from the factory" state.  The recovery process is initiated by some key press just after power on.  See, e.g., http://www.pcadvisor.co.uk/how-to/laptop/how-factory-reset-windows-pc-laptop-or-tablet-3462995/

On the other hand, making a backup image of the working part of your hard drive (e.g., the C:\ partition) would be a good idea.  You can either use the Backup system built into Windows 7 or one of the various third-party backup/imaging tools.

Those tools can also make an image of your D:\partition if you like.

I'm not going to list them all.  You can Google it: https://www.google.com/search?q=backup+software+windows+7&oq=backup+utilities

I happen to use and like Acronis True Image, but it's not free.

You really ought to use a hard drive in an external enclosure rather than a DVD (or a set of DVDs) for system images.

Finally, you should make a Windows 7 System Repair disc.  This isn't the same as a backup, but it can be very useful.  Search for "System Repair" in Windows 7 Help & Support.

What is media streaming and how do I use it? - Microsoft Community

aashish g say:

What is media streaming and how do I use it?

What is media streaming and how do I use it? What is media streaming and how do I use it?
anannya podder say:

Media streaming in Windows 7 lets you access the media on your home PCs while you’re away from home. To use it, you need to set up Remote Media Streaming on both your home computer and the computer you’ll use to play your home media—but after that one-time setup, all you’ll need to do is connect to the Internet, open the Player, and enjoy your media. The basic steps are:

 

1.       Open Windows Media Player and click the Stream menu in the upper-left corner of Player.

2.       Click Allow Internet access to home media.

3.       Click Link an online ID and follow the instructions.

4.       After you link your online ID, in Window Media Player, click the Stream Menu, and the click Allow Internet access to home media.

 

For more details, see Leave the house but not your media: Watch and listen on the road using remote media streaming. (http://windows.microsoft.com/en-US/Windows7/leave-the-house-but-not-your-media-watch-and-listen-on-the-road-using-remote-media-streaming)
