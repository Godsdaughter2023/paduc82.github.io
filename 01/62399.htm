lee drake say:

You will have to swap out graphics cards for another card to get into the machine initially. I'm sorry that's the only solution I could find. Just go buy a cheapo card with the proper interface for your monitor (it can even be VGA), preferably NOT an NVIDIA card. This will let you boot to the desktop so you can uninstall the offending NVIDIA software. Do so, and reboot again, download the latest k4000 drivers from nvidia.  Then tell it to restart in "low rez" mode, which loads basic video drivers.  Shut the machine down when it goes to reboot (don't let it start - turn it off when you get to the bios screen), then swap the video card for the 4000 again.  When it restarts in Low Rez mode install the latest k4000 drivers from nvidia, and shut down again.  You should then get a normal boot.  The issue is that since the nvidia drivers are causing the blackout issue you cannot see the screens (even though the system is booting normally in the background).  Putting the other video card in forces it to ignore the k4000 drivers and use different drivers which work.

Sorry I don't have a better option, but this is how I got mine working - your mileage may vary.  Also thanks Microsoft and Nvidia for not testing something as basic as this.

If you CAN roll back, do so, and then go get the latest nvidia drivers and install them BEFORE doing the anniversary update. It MAY work.

vijay a. verma say:
1. Windows Key+I > Privacy > Camera > Make sure that Let apps use my camera hardware is ON
Also down below Choose apps that can use your camera is ON for Skype

2. Use the Hardware & Devices Troubleshooter
Type Troubleshoot in Windows Start Search box > Click Troubleshoot > Scroll down & Click Hardware & Devices > Click Run the Troubleshooter & follow on-screen directions > Restart the computer after troubleshooting is completed

3. Reset the Skype
Windows Key+I > Apps > Locate Skype > Advanced Options > Reset
Go to Microsoft Store > Click 3 horizontal dots on top right > Downloads and Updates > Check if there is any update for this app > If yes, update it.

4. Uninstall and reinstall Skype app (Note - You will not lose any Skype data because of this)
Uninstall
Windows Key+X > Windows Powershell (Admin) > Copy the below, paste into Powershell > Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Remove-AppxPackage

Reinstall
Copy and paste below and Enter

Get-AppxPackage -allusers Microsoft.SkypeApp | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

5. To isolate the issue, can you make call using either Web Skype or Classic Skype (Download)

Web Skype - https://web.skype.com/

Download and install Classic Skype - https://go.skype.com/classic.skype
