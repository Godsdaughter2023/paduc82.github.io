"Navigation to the webpage has been cancelled" - Microsoft Community

dghrdy say:

"Navigation to the webpage has been cancelled"

Why does this message come up intermittently?

a. user say:

You may have this as a scheduled task in Task Manager.  Go to Start / Control Panel / Administrative Tools / Task Manager and check for an items called InternetServiceOffers and Registration.  If you find them, delete them (or at least disable them if your uncomfortable deleting them) and that may stop the messages from appearing (or at least reduce their frequency).  If you don't have those entries (I don't) then that isn't the problem.

Here's a Microsoft article on this problem with some proposed solutions: http://support.microsoft.com/default.aspx/kb/967941?p=1.  I think this might be you best bet at a solution.

Does this problem occur when you try to go to a website yourself or does the message just magically appear periodically?  What browser are you using and what version?

Here's an article on how to adjust IE settings (if that's the browser you're using) - perhaps some change in the settings will prevent this problem (or you can always restore defaults and see if that helps - though you will then have to reset any changes you made that you want to keep).

This message can also occur if you temporarily loose connection between your wireless device or router and the modem (or perhaps with the internet itself - though that would probably be an ISP issue).  If it continues to occur, you should check these devices for problems (you can for example plug the modem directly into a pC, bypassing the router, and see if the problem persists on that PC).

I hope this helps.  Post back and let us know in any case.

Good luck!
