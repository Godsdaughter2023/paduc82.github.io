Assuming that there is a wired connection to the LAN on the deck in question, what you really need is a "wireless access point" rather than a wireless router.  As it turns out, a wireless access point usually costs more than a wireless router -- but the latter
The following assumes that you DO have access to the wired LAN.  You will connect the E1000 to the LAN -- not "to a computer" that is connected to the LAN.  If there is already a computer using the only available LAN connection (and that computer doesn't have wifi capability), just connect that computer to one of the available LAN ports on the E1000 when you are done following the directions below.  Do
Directions for doing the above are here --> http://ezlan.net/router_AP.html 

The User Manual for the E1000 should be on the CD, but if not, go here --> http://homesupport.cisco.com/en-us/support/routers/E1000


To find the IP address of what the directions call the "main router," go to any Windows computer now connected to the ship's LAN, open a Command Prompt window (Start > Run > cmd > OK), type "ipconfig" (without quotes), and Press Enter.  You will see something like this:

Ethernet adapter Local Area Connection:

        Connection-specific DNS Suffix  . :
        IP Address. . . . . . . . . . . . : 192.168.1.100
        Subnet Mask . . . . . . . . . . . : 255.255.255.0
        Default Gateway . . . . . . . . . : 192.168.1.1

The IP address for the "Default Gateway" is the IP address of the main router.


The default IP address of the E1000 is 192.168.1.1.  This needs to be changed to an IP address that is compatible with the IP addresses assigned by the main router but not the same as any other IP address on the LAN.  This is what the directions mean by "give it an IP that doesn’t conflict with the main router's base IP address, or the DHCP server range." 

Assuming that the "subnet mask" that was shown when you did the above procedure was 255.255.255.0, the first 3 segments of the IP address must be the same.  For example, if the Default Gateway is shown as 192.168.0.1, you need to change the E1000's IP address to 192.168.0.x, where x is between 2 and 254 but not within the range of addresses used by the main router's DHCP server. 

To find the "range of addresses used by the main router's DHCP server," you'll have to access that router's configuration utility.  The corresponding information for the E1000 is shown on page 16 of the User Manual.  The same page also has directions for disabling the DHCP router of the E1000 and setting its IP address.
