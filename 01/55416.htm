We suggest that you enable the hidden administrator in Windows Recovery Environment (WinRe). Kindly download the ISO file provided on this link and create a bootable media.

Once you're done creating an installation media, follow the steps below to access WinRe.

  1. Insert the installation media.
  2. Reboot your computer.
  3. Press any key to proceed when prompted.
  4. On the Install Now window, click Repair your computer.
  5. Choose Troubleshoot.
  6. Select Advanced Options.
  7. Click Command Prompt.
  8. On the Command Prompt, type this command:
    Net user administrator /active:yes
  9. Close the Command Prompt.

Once you're done with the steps, reboot your computer and log in using the hidden administrator that you have enabled. Create a new user account or set your main account as Administrator, then delete the hidden administrator.
