New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

This is how the recovery partition is accessed for the most popular makes..

For Dell, press F8 on the keyboard until the Advanced Boot Options menu appears on the screen.

For HP, press F11 directly after powering up the unit

For LG, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Samsung, press F4 at power up..

For Fujitsu, press the F8 key repeatedly directly after powering up

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

For Sony VAIO, restart and press "F8" or "F10" repeatedly until the "Advanced Boot Options" screen appears.

The Error “Failure configuring Windows updates Reverting Changes” normally occurs when the installation of the Windows Updates fails and thus, while restarting the computer, Windows 8 is not able configure the failed updates and tries to revert back the changes that were made on the computer by these updates. Normally, this situation comes up when you have a large amount of updates to install on the computer. This will also help you to make a track of the failed updates, so that you can come back and try installing the failed ones again.

 

There are many reasons why this issue occurs on the computer. The Windows updates normally fails when there are some issues with the Windows Update Components, corrupted system files, pre-requisites updates not installed, conflicts from the third party software on the computer etc.

 

We have also come across certain instances where the removable media connected to the computer has caused this issue while restarting after installing the updates. Thus, if you have any removable media connected, then remove or unplug them from the computer while restarting the computer after installing the Windows Updates.

 

This issue also happens when there are conflicts from any third party software on the computer. So, while installing the Windows Updates, make sure that the update process is not blocked or conflicted by any process or application on the computer. You may uninstall the unnecessary software on the computer which are no longer in use. Also, always keep your computer up to date.

 

Hope this information is helpful.
