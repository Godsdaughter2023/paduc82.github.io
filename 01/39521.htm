lemp say:
You forgot to identify the wireless adapter.

When you installed the adapter, you installed a configuration utility provided by the manufacturer of that adapter.  This is quite common.  The issue is that a wireless adapter may be controlled (managed) by either Windows or the manufacturer's utility -- not both at the same time.

There likely are 2 different icons in the notification area of the new PC (at the end of the Task Bar next to the time display) that relate to the wireless adapter.  You've been clicking on the icon that goes to the built-in Windows utility (or getting to the Windows utility another way).  You need to get to the other utility -- either to turn it off so that you can use Windows to manage the adapter or to configure the adapter to connect to your router.  To help you do that, I need to know the make and model of the new adapter.

Your Huawei Echolife HG521 router is capable of setting up a wifi network in several modes and with different types of encryption. 

Are you currently using the wifi capability of the router or is your other computer connected to the router using an Ethernet cable? 

There should be a "WLAN" light on the router.  If that light is on or blinking, the wireless mode of the router has been enabled. 

If you haven't already configured the wifi settings on the router, you'll have to do that as part of setting up the wifi connection for the new computer.  In particular, you'll need to know (or create) the wifi encryption password.  This is NOT the password that your service provider gave you to log on to the Internet.

Here's a video that explains how to configure the HG521 (again, this is your router, not the adapter in the new computer) --> http://www.youtube.com/watch?v=XCq0TzDX1Vw

Two comments on that video: 
The default username and password for logging in to the configuration utility of the HG521 are both admin.
When configuring a wifi network do NOT enable "hiding" the SSID. broadcast.   If that box has been checked (as shown in the video) it should be UNchecked.
