Hi,

Thank you for posting in Microsoft Community.

If you do a custom install over Windows 7 or later, the Rollback option is not available. This only works if you had done an in place upgrade from Windows 7 or later. If you had set a password for previous operating system and don't have or cannot remember its not possible to login to the previous OS.

Follow these steps : How to rollback to your previous version of Windows from Windows 10.

1.    Start by opening Settings – Click Start > Settings or type Settings in the search bar and click it or press Windows key + I.

2.   Click Update and Security

3.   Click Recovery then click Get started under Go back to Windows 7  (depending on the previous version you running)

7.   The process can take a while depending on the amount data, applications and settings. Click Next

8.   Click Go Back to begin the process.

9.   Windows 10 will Restart

This will be your screen throughout the recovery process.

When the process is complete, you will be booted into your previous version of Windows environment.

Assuming you can do that, there are several ways to accomplish what you need to do.

In the "Open with" window there are 3 possibilities:

  1. If Word appears in the "Recommended Programs" section, select it (click once on it)
  2. If Word doesn't appear in the "Recommended Programs" section, double-click "Other Programs" to expand that section and if Word appears in the "Other Programs" section, select it (click once on it)

    • In the Office11 folder, find WINWORD.EXE (if you haven't configured things to see file extensions, you'll only see WINWORD).  This file should have an icon like your usual shortcut to Word.  Select this file (click once on it) and click "Open"

Back at the "Open with" window, after you have selected Word using possibility 1, 2, or 3, check the box at the lower left to "Always use the selected program to open this kind of file."  Click OK
