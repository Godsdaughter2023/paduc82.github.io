This is a new Windows 8 setting on top of the normal settings in Power settings.

 

Be aware that the setting of how long time must pass before prompting for a password is required is a per-user setting, so if you need to change it from the default of 15 minutes, it must be for each user created on the system.

 

For me, I've set it to 1 minute rather than always for all users, as it's nice if after an accidential sleep a user can still resume immediately. But anything above that I have a password enforced.

You don't say what Windows you have now.  If you have the Get Windows 10 app on your taskbar and have used it to reserve Windows 10 then you are in a queue for upgrading.  Doing so before you get the email telling you to go ahead is a little risky because you may need a new driver for one of your devices and Microsoft is waiting for the developer to finish it, or something like that.  You can "jump the line" if you know what you are doing by going to
