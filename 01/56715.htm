Hi

To wake up from sleep when need to print through the Network the computer have to be able to Wake on LAN traffic.
This function needs capable hardware and software configuration.

Some computers' hardware cannot Wake on LAN no matter what you do.

Most  computers that are connected to a Network via Wireless can not Wake On LAN.

Boot the computer into the motherboard's BIOS and see if there an entry concerning Wake on LAN. If there is one, Enable it.

Software wise.

Check the Computer's support site and make sure that you are using the latest Drivers*** for the Network cards.

In the Network connector look into the Network card configuration and set it according to your needs.

Example, http://www.ezlan.net/Win7/power_save_win7.jpg

Hi Scand Warrior, there are 2 ways to stop Windows booting into Safe Mode:

1.
Boot into Safe Mode
Click your Start Button, type msconfig and hit enter
On the Boot tab uncheck boot into safe mode
Restart your PC

2.
Boot into Safe Mode
Click your Start Button, type cmd and hit enter
Copy this command into the command Window and hit enter:
bcdedit /deletevalue {current} safeboot
Restart your PC

scottzygadlo say:

I purchased a new Lenovo ideapad 330 in Jan 2019. I had problems with the "gestures" function on the touch pad and wanted to turn the functionality off so the touch pad worked like a regular mouse without the "gestures" features (I don't use a separate mouse, just the touchpad). When I went to touchpad settings, there was no way to turn off the individual "gestures settings" (ie. pinch & zoom, etc.). It was impossible to turn it off my manipulating the mouse settings either. I read in an internet forum that the ELAN HID Touchpad driver had not been updated to allow users the ability to turn off individual "gesture" functions. Sure enough, I searched for an updated manufacturer driver and none existed.

My solution:

- Uninstall the HID Device. In my case, my Lenovo had an ELAN HID device.

Steps:

Option 1: Disable the HID Device

  1. Go to device manager
  2. Expand "Mice and other pointing devices"
  3. Click on ELAN HID Device (Do not click on "HID-Compliant Mouse"). You may have a different manufacturer than ELAN)
  4. Click on the Driver tab
  5. Click "Disable Device"
  6. Click "OK"
  7. Restart the PC

Option 2: Uninstall the HID Device

  1. Go to device manager
  2. Expand "Mice and other pointing devices"
  3. Click on ELAN HID Device (Do not click on "HID-Compliant Mouse"
  4. Click on the Driver tab- Click on "Uninstall Device"
  5. Click on "Uninstall Driver"
  6. Click OK. The device SW and driver will be uninstalled
  7. Restart the PC

The gestures functions will be gone and the touchpad will work like a normal device without the annoying gestures. I did not try to disable the device, but that may work as well and is worth trying first. If that doesn't work, uninstall the device. I never plan to use "gestures, so I chose to uninstall it so it didn't return.
