Microsoft Windows [Version 6.1.7600]

Try setting the program as the default

How to change the default PDF file reader in Windows 10
Method 1. Use the file browser
Step 1. Right click on your PDF file

Right click on a PDF file. Click on "Open with"> "Select another application".

Step 2. Change the default reader

A pop-up window will appear so you can choose your favorite software. Check the "Always use this application" box and click "OK".

If your preferred software does not appear in the pop-up window, click "More applications" to expand the list. Scroll down until you can see the "Find another application on this PC" link and click on that option. Find and select your preferred software to set it as the default PDF reader in Windows 10.

Method 2. Use the control panel
Step 1. Open the control panel

Open the control panel of your computer and click on "Default Programs". Proceed by accessing the option "Associate a file type or protocol with a program". It will take some time for your computer to load all of its file formats.
Step 2. Change the default program

Enter the extension .pdf. Highlight your line and click on the "Change program ..." button. A pop-up window will appear allowing you to choose your preferred default PDF reader.

Step 3. Confirm your settings

If you can not see your favorite PDF viewer, you can find it by selecting the "More applications" link. If you still can not find it, scroll down and click on the "Find another application on this PC" link. Once you have located the software you want, click "Open" to set it as the default PDF file viewer on your PC.
