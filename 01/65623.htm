Hi Steve,

This is intended behavior. As some background, your internal HDDs and some external HDDs have their own $Recycle.bin folder. This is a hidden folder located on the drive. When you delete a file, it is sent to the recycle bin, not permanently deleted. You can disable the prompt for these.

Memory sticks usually do not have their own $Recycle.bin folder because the hardware does not support it. Thus, when you ask to delete a file there, it is permanently deleted. And because you are undertaking this operation that you cannot reverse, Windows must ask you to confirm.

As far as I know, there is not a way to disable the prompts for permanent deletion, only recycling.
