¡firedog say:

I realize that this might come a bit late for you, but I haven't noticed the thread before.

The settings you need to apply are stored variously in the registry and in the store folder, which is by default in the user profile. You may be able to write a script to apply the settings you want, but it will be fraught with difficulty.

Account settings are stored in an XML file in an account folder in the store folder. Both the name of this account folder and the location of the store folder are user-configurable unless you restrict permissions on the keys concerned, so you'd have to get there first.

If account passwords are set to be remembered, they are stored in the XML file encrypted with a key based on the user profile (and possibly hardware). So users would have to apply the password. The rest of the config file could be pre-determined by you. Here's a typical one, called account{A009368C-AFD0-495B-B48D-D22E23C43825}.oeaccount and residing in a folder called Canada Live (the same as the Account_Name value):

<?xml version="1.0" encoding="utf-16" ?>
<MessageAccount>
    <Account_Name type="SZ">Canada Live</Account_Name>
    <Connection_Type type="DWORD">00000003</Connection_Type>
    <Server_Sort_Order type="DWORD">0000000f</Server_Sort_Order>
    <POP3_Server type="SZ">pop.a.com</POP3_Server>
    <POP3_User_Name type="SZ">jnllbⓐlive.ca</POP3_User_Name>
    <POP3_Use_Sicily type="DWORD">00000000</POP3_Use_Sicily>
    <POP3_Port type="DWORD">0000006e</POP3_Port>
    <POP3_Secure_Connection type="DWORD">00000000</POP3_Secure_Connection>
    <Leave_Mail_On_Server type="DWORD">00000001</Leave_Mail_On_Server>
    <Remove_When_Deleted type="DWORD">00000001</Remove_When_Deleted>
    <POP3_Skip_Account type="DWORD">00000001</POP3_Skip_Account>
    <POP3_Prompt_for_Password type="DWORD">00000001</POP3_Prompt_for_Password>
    <POP3_Use_APOP type="DWORD">00000000</POP3_Use_APOP>
    <SMTP_Server type="SZ">smtp.live.com</SMTP_Server>
    <SMTP_Use_Sicily type="DWORD">00000000</SMTP_Use_Sicily>
    <SMTP_Port type="DWORD">00000019</SMTP_Port>
    <SMTP_Secure_Connection type="DWORD">00000000</SMTP_Secure_Connection>
    <SMTP_Display_Name type="SZ">NB alias</SMTP_Display_Name>
    <SMTP_Email_Address type="SZ">jnllbⓐlive.ca</SMTP_Email_Address>
</MessageAccount>
 

The default account is recorded at
     HKCU\Software\Microsoft\Windows Live Mail\Default Mail Account
with the XML filename as data.

The store folder location is specified at 
     HKEY_CURRENT_USER\Software\Microsoft\Windows Live Mail\Store Root
with the full path as data. The default is
     %localappdata%\Microsoft\Windows Live Mail

So if you could restrict permissions on those sub-keys to sysadmins, you might get away with it. You can't restrict the whole program key, because all sorts of program settings are stored there on the fly.

Good luck!

Noel

Since you are using Movie Maker 2012, you can use Add music and it will float locations.  But if you use the Add sound option (under the Record narration button drop-down), then the audio is linked to the item that it is under.  So the feature is there somewhat already.

mre_mann say: Problems solved!

1. Once I completed my project (and made adequate backup!) I made notes of the exact times where specific audio tracks needed to start or coincide with specific video images.
2. I saved the project at an HD quality mp4, which resulted in the various audio tracks changing lengths.
3. Using a free clip converter program, I converted the movie into a .wma audio track.
4. I went back into a copy of the project, removed the audio tracks and replaced it with the single .wma audio track, setting the audio mix to 100% sound.
5. Then, I went through the project one more time, adjusting the lengths of various transitions and stills to compensate for the expansion / contraction of the audio tracks, and (after, again making backups), saved the movie, again to a HD mp4. 

The outcome of this second rendering resulted in no appreciable loss in sound quality, and all the audio track remained in complete sync with the video, as designed.

To add a phone number as a contact on your Skype, please refer to the screenshots below:

For Windows 10:

Open your dial pad.
