Method 1:

Check the status of the Windows Installer service.

* Click Start, type Services.msc into the search space and select the same from the list.
* Find the Windows Installer. Double-click “Windows Installer” in the Services list.
* Check whether the service is running or not. If the service is not already running, click Start, and then click OK. If it is running already try to stop the service and Start it again.

Method 2:

Try to Unregister and re-register the Windows Installer

* Click Start again, type MSIEXEC /REGSERVER into the search space and hit enter.

* Now, try your installation again and see if it allow to do.
