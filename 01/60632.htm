The option for "allow remote connections to this computer" only shows up if you're using Windows professional edition (not home edition).  But, that only applies to the actual Microsoft Remote Desktop system, so it doesn't impact your ability to use VNC.  As long as the "VNC Server" is running down by your system clock, you should be able to connect.

If you click the Edit button in the Photos app, then click "Basic fixes" you'll see the Rotate button.  Click it and it will rotate the image. 

This looks to be caused by the file itself being set to 'read only'.  That is, it is set so you can't make changes to it, thus the rotate button not working or not being available.

You can either use the Photos app Edit mode to rotate and save as a new copy, or you can remove the read-only attribute from the file.
