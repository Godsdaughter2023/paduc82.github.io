We have been seeing black screen issues of several types after upgrading to Windows 10. The main causes observed so far are driver issues, output to unexpected port, and possibly also issues with the Lock Screen app. Please use these steps:

If you see the black screen only after sign-in and you can see a mouse cursor:

  1. While at the black screen, try using CTRL+SHIFT+ESC to start Task Manager.
  2. If it becomes visible, try finding LockAppHost.exe on the Details tab and ending the task.
    1. If this resolves the black screen, use corruption resolution steps (dism, then sfc), and run the all apps PowerShell command in 3080495.

  1. Dism steps

    Run DISM tool.
    Please follow these steps:
    a. Press Windows + X keys and click command prompt (admin).
    b. Type the following command in command prompt and press Enter.
    DISM.exe /Online /Cleanup-image /Scanhealth.
    DISM.exe /Online /Cleanup-image /Restorehealth.

b.Use the System File Checker tool to repair missing or corrupted system files (sfc)
https://support.microsoft.com/en-us/kb/929833
note- it applies to Win 10 as well.

  1. If this does not work, use File, Run new task in Task Manager to start an Administrator CMD window. Use this to create a new administrator user with the following commands (changing “<password>” to something of the customer’s choosing):
    net user test <password> /add
    net localgroup Administrators test /add
  2. This will give you a test user to try, which will confirm if the issue is user or system-specific.

If you see the black screen or have a flickering screen before sign-in:

  1. Try connecting to different video outputs on the PC. Sometimes a change in driver can cause the video to be sent:
    • To a different Adapter: from integrated graphics to a discrete card or vice versa.
    • To a different output: from HDMI to DVI, DisplayPort to VGA, or any other combination.
  2. Try to boot into Safe Mode with Networking or Enable low-resolution video.
    • Advanced boot options, Safe Mode with Networking, or Low Resolution Video
    • Check Device Manager and look for Display Adapters
      • If you see 2 items under Display Adapters, disable to integrated video card (e.g. Intel HD 4000 or AMD Radeon HD 4200) and restart. (try vice versa if it does not work)
      • If you see 1 item under Display Adapters, rollback the driver or uninstall the display adapter in Device Manager, and enable the option to delete driver software.

Check the PC or display adapter manufacturer’s site for an updated driver.

Hope this helps. Let us know if you have any other issues with Windows in future.

To recover from demoting your own account you need to go through a rather complex process. Let's take it in steps to see if you can follow.

Step 1: Go to the Control Panel / Backup and Resture and create (=burn) a System Repair Disk (=CD). If successful, goto Step 2.

Step 2: Change the boot order of your BIOS so that the machine uses the CD drive as its primary boot device. Now boot the machine with your Repair Disk. Go to Step 3 if you get to the Repair Menu without getting challenged for a password.

Step 3: Select System Restore, then restore the machine to a point before you demoted your one and only admin account.

Step 4: Boot up normally, then create, test and document a second admin account in order to be protected against future mishaps.

If you intend to install this on a "home-built" PC, you will need to use the key that you unfortunately lost.

Hi Efrain,

The error code that you’ve encountered on your computer may be caused by some configuration or files mismatched. To sort out your concern, we suggest that you perform the rebuild option on your Windows computer from the Windows Recovery environment using the Command Prompt. This process will repair Windows boot configuration in order to run the Operating System properly. This option will not remove the files on your PC.

To do this, you may follow the steps below:

  1. Restart your computer.
  2. Press the Shift key on your keyboard while pressing F8 quickly and continuously.
  3. On the Choose an option window, select Troubleshoot > Command Prompt.
  4. On the Command Prompt window, press Enter after typing each of the following commands:
    • bootrec /FixMbr
    • bootrec /FixBoot
    • bootrec /RebuildBcd
  5. Once you’re done, you may restart your computer, and then check if the issue persists.
