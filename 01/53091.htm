1.Go to run.(Press Windows key+R)

2.Type regedit.

3.Now in registry go to HKEY_CLASSES_ROOT \ Directory \ Background  \ shellex \ ContextMenuHandlers.

4.Now export each one of the folder to documents(Right click on folder->select export->enter the same name corresponding to that folder and click save).

5.Now delete one and see is problem is solved, if not then replace back that by going to documents and double click on the exported .reg file of deleted entry.Now delete another and see;  if not solved import back , else Done!

6.For example i see 4-5 folders in ContextMenuHandlers. Now i choose a folder named igfxcui and export that folder, by the method in step 4. Now i deleted that folder. Now i go to desktop and do right click to see if it is gone or not. if not import back that folder by going to documents->double click on igfxcui.reg file.

7.Note: This problem is caused by non-windows handler, for me it is caused by folder named igfxcui in registry, so i deleted that one and problem is gone now.

Wow, thats cool, but does that mean, I will have to pay for Windows 10 after the first year?

No. Its a promotion to get as many users to upgrade their existing Windows 7 PCs and Windows 8.1 devices to Windows 10. Once you upgrade within the first year for free, its yours forever at no additional cost. As Microsoft Program Manager for Windows Terry Myerson described it:

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device –at no additional charge.

No, Windows 10 remains a perpetual license, which means, you can upgrade to Windows 10 and use it forever without it expiring or going into any reduced functional mode.

Are you saying there will be no more versions of Windows after Windows 10?

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device – at no additional charge. Windows 10 is free so app developers can standardize on one platform and reduce the fragmented market of different versions and editions of Windows that have been an issue in the past. When everybody is on the same platform, it means better quality apps, a healthier ecosystem and more advances in hardware and software innovation.
