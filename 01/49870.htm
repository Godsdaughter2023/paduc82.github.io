Windows defender scan may freeze due to the bad sectors on the hard drive.

 

Method 1:

Register the Inseng.dll File

a)      Click Start, and then click Run.

c)      When you receive the message that states that DllUnregisterServer in Inseng.dll succeeded, click OK.

e)      In the Open box, type regsvr32 inseng.dll, and then click OK.

f)       When you receive the message that states that DllRegisterServer in Inseng.dll succeeded, click OK.

g)      Restart the computer.

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7

lead3 say:

381b4222-f694-41f0-9685-ff5bb260df2e is the GUID that instructs Powercfg to make a modification to the Balanced power scheme (plan).

powercfg -setactive 381b4222-f694-41f0-9685-ff5bb260df2e makes the Balanced Power Scheme active.

It seems that something made the Balanced Power scheme become inactive (this is usually the default plan), possible something with Winpatrol.
