cameronclements say:

For anyone that's dealing with the same frustration, I have found this works to prevent bloatware apps from reinstalling when adding a new profile.

  1. Preform a clean install of Win 10 (I used a VM)
  2. Install updates or use a recent build like 1703 at the time of this posting
  3. Powershell: Get-AppxProvisionedPackage -online | Remove-AppxProvisionedPackage -online
  4. Powershell: Get-AppxPackage -AllUsers | Remove-AppxPackage 
  5. Unpin all tiles you don't want from the stat menu. You  need to have at lest one tile or the next step will fail.
  6. Powershell: Export-StartLayout.xml <path><filename.xml>
  7. Open Group Policy editor: User Configuration>Administrative Templates>Start Menu and Taskbar>Start Layout
  8. Under options: Start Layout File enter the path to your xml file created above.
  9. Configure anything else you want then sysprep and image the drive.

If you are working in a domain you can do this with a global GPO. This was for a small workgroup with out a DC.

wilfred mac say:

Hi David,

Game Mode is a new feature in the Windows 10 Creators Update that focuses system resources on games when enabled. To help you disable Game Mode, we recommend that you follow the steps below.

Disable Game Mode:

  1. Right-click the Start button.
  2. Click Settings.
  3. Click Game Mode.
  4. Toggle User Game Mode to turn it off.

You can also disable Game bar and Xbox Monitoring. To proceed, you may follow the steps below.

How to disable Game bar:

  1. Right-click the Start button.
  2. Click Settings.
  3. Click Gaming.
  4. Click Game bar.
  5. Click the switch below Record game clips. Screenshots, and broadcast using Game bar so that it turns Off.

How to disable Xbox Monitoring.:

  1. Right-click the Start button.
  2. Click the Search button.
  3. Type regedit and press Enter on your keyboard.
  4. Double-click HKEY_LOCAL_MACHINE.
  5. Double-click System.
  6. Double-click CurrentControlSet.
  7. Double-click Services.
  8. Click xbgm. You might need to scroll down to find it.
  9. Right-click Start REG_DWORD on the right-side pane of the Window.
  10. Click Modify.
  11. Type 4. (The defaults is 3).
  12. Click OK.
  13. The Xbox Game Monitoring is not disabled.

Should you need further assistance, feel free to post anytime.
