I FOUND THIS POSTED BY SOME GENIUS CALLED DARKRAJIN AND IT WORKS ON GENERALS AND GENERALS ZERO HOUR

 

This I found on another page and thought I would share it here as it worked for me.

 

I have a core i5 laptop running windows 7 home premium 64-bit with an ATI graphics card and I had exactly the same problems getting C&C Generals and Zero Hour to work.

Good News - Have just  successfully managed to get it working, and it IS Windows 7 that causes the problem, specifically it's to do with access permissions to folders - Windows 7 blocks access to folders that Generals uses to run.

The way I fixed it was:

In 'My Documents' (usually at C:\users\YOURNAME\documents) there should be a folder named 'Command and Conquer Generals Data' or 'Command and Conquer Generals Zero Hour Data', which was put there when you installed the game.  You need to create a text file using notepad and save it into this folder.  However, you must click 'save as' and change the type drop-down box  from text file (.txt) to 'all files'.  Then save the file as 'options.ini'

Before you save, paste this text into the file to get you started:

AntiAliasing = 1
BuildingOcclusion = yes
DrawScrollAnchor =
DynamicLOD = no
ExtraAnimations = yes
GameSpyIPAddress = 0.0.0.0
Gamma = 50
IPAddress = 0.0.0.0
IdealStaticGameLOD = High
LanguageFilter = true
MaxParticleCount = 5000
MoveScrollAnchor =
MusicVolume = 55
Resolution = 1024 768
Retaliation = yes
SFX3DVolume = 79
SFXVolume = 71
ScrollFactor = 50
SendDelay = no
ShowSoftWaterEdge = yes
ShowTrees = yes
StaticGameLOD = Custom
TextureReduction = 0
UseAlternateMouse = no
UseCloudMap = yes
UseDoubleClickAttackMove = no
UseLightMap = yes
UseShadowDecals = yes
UseShadowVolumes = yes
VoiceVolume = 100

This is simply how the game remembers what options you picked (music volume, brightness and the like) and  you can change them once the game is running using the options menu.

Now the key bits - firstly, make sure that you have full read/write permissions for the Command and Conquer Generals Data folder - do this by right clicking on the folder, then  properties then the security tab and change the permissions for all users.  It might also be worth making sure no files are "Read-Only" in that folder.

Then change the game's compatibility mode by right clicking on the game in the start menu, clicking on properties and then the Compatibility tab.  I used the "Change settings for all users" button at the bottom and changed to Windows XP (SP3), as well as checking the "Run as administrator box".

Hey presto! It worked, debunking the myth that C&C Generals will not run on 64 bit systems (as some people have been saying).  In fairness, I haven't tried online multiplayer yet, just a few skirmishes.

Good Luck!

… So, I Created a New Account in Windows10 – Hey Presto, no more problems…

To be able to work across user accounts you must have Administrator privileges.

Creating a New User Account inherits a bit of work as all the personal stuff must be transferred to the New User Account (Documents, Pictures, Music, Templates, email (pst), etc…); but, you get a ‘Clean’ account to start off with.

DO NOT delete the Old User Account just yet - You will need the Old User Account for a while as the Old Account has all your personal stuff.

As a test before you go through all the work of moving your personal stuff over to the New User Account, boot-up the New User Account and try the Apps…

You will not be able to create a New User Account using the same username as the Old Account; so, to save the Old User Account Rename it:

1 – Right-Click Start; Click Computer Management;

2 – Expand Local Users and Groups;

3 – Expand Users;

4 – Right Click the user account; Click Rename; Key-in a new name (any name will do);

To create a New User Account:

5 – Follow steps 1-2 inclusive above;

6 - Right Click the Users folder; Click New User;

7 – Complete the User Name and Password fields as a minimum; using the check boxes decide if the User must change password at next logon, User cannot change password or Password never expires; Click Create; close the New User box.

8 – Double click the New User Account; select the Member Of tab; to change the default User profile you can Add and Remove user profiles; there are a number of built-in profiles each with their own set of permissions (I prefer NOT to use the computer via the Administrator profile; rather, I use the User profile.  I use the Administrator profile ONLY for making changes/repairs).

9 – To Add a built-in profile to the user account click Add then key-in the user profile type; e.g. Administrators or User then click Check Names, then OK; If you see a Name Not Found box click Cancel and check the user profile you have keyed-in;

10 – To Remove a built-in profile from the User Account, highlight the unwanted profile then click Remove;

11 - Click OK to close the User Account Properties box then close the Computer Management App.

12 – When you are happy that the New User Account is working OK you can remove the Old User Account follow Ramesh.Kumars reply in the article
