When you first power up the system nothing you see being displayed on the monitors has anything to do with Windows, it is all direct from the hardware/firmware in the system. What you're seeing is that the VGA output on your video card is actually the primary output of the card and the HDMI (or DVI) is secondary. After Windows starts running and the video drivers are loaded then Windows can switch to using the monitor you selected to be your primary display.

We have a terminology problem.


The router is not an adapter.  It is a router.  It also is a wireless access point.  And it also is an Ethernet switch (the 4 "local area network (LAN)" ports on the back of the router).


Every computer that is to connect to a network must have a network adapter.  The adapter can either be an Ethernet adapter (typically shown in Windows as a "Local Area Connection" or a wireless network adapter.  Today, almost all laptops have both:  that is, an Ethernet adapter and a wireless network adapter.  Tablets, e-readers, and smart phones usually have only a wireless network adapter.  And desktop/towers usually have only an Ethernet adapter, although it's easy to install a wireless network adapter in a desktop/tower either as an internal card or as an external USB-connected device.


If your desktop computer is connected by a wire (i.e., an Ethernet cable) to your router, the end of the wire connected to your computer is plugged in to your computer's Ethernet adapter.  If you don't have any antennas coming out of the back of your computer or you don't have a USB-connected device that says that it is a wireless network adapter, then you don't have a wireless network adapter.


If your desktop computer has only a Ethernet (wired) network adapter, the network icon will look like the icon on the left; if you have a wireless network adapter, you should see the icon on the right:


If your desktop computer does not have a wireless network adapter, the "issue" is that it can't tell you with an icon whether or not your wireless router (i.e., your wifi network) is working.  If you can access the Internet on the desktop, all that tells you is that the rest of your router (the router part and the network switch part) is working properly.


On the other hand, because your desktop is connected to your router, you can log into the router's configuration utility which will, among many other things, let you know if the wireless access point portion of the router is turned on (you know it is, however, because the Nook can connect to it).  You also can determine whether the router's wifi system is working by looking at the "wireless" LED on the front of the router; it should be solid blue.  Note that there is a button on the back of your router that turns the wifi on and off.

From any computer connected to the router, you can open a browser and go to http://www.routerlogin.net or http://www.routerlogin.com (or simply go to the router's local IP address, which by default is 192.168.1.1 or you can find it as the "Default Gateway" if you do the ipconfig command that I described earlier).  At that point you have to log into the router's configuration utility.  For your router, the User name is "user" (without quotes) and the password is "password" (without quotes) unless you have changed these values.  This information should conveniently be printed on the back of your router, just above the USB port (which is only to be used to connect an external storage device; do not connect computers, USB modems, CD drives, or DVD drives to the router USB port).

If you are not already familiar with your router's configuration utility, the User's Guide should be available on the CD that came with the router or you can download it here --> http://www.downloads.netgear.com/files/GDC/WNR3500LV2/WNR3500Lv2_UM_8Sep11.pdf  I expect that Google support will tell you to do things that involve using your router's configuration utility.
