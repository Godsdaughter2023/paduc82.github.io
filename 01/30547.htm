steve cochran say:
It may be the server that is blocking and not the mail program.  A way around this is to give the attached file a second file extension and then it will get past the screening and then the recipient can rename it back to its original file extension by removing the extra one.
 
First make sure you have "hide extensions of know file types" unchecked under Control Panel | Folder Options | View.  The recipient will need this option unchecked also so they can rename the attachment.
 
For example, if you want to send a file such as myprogram.zip and either WLM or the server blocks that attachment as a "prohibited" then rename the file to myprogram.zip.txt and then send it and most servers are too dumb to realize it and let it through.  Then the recipient can save the attachment and rename back to myprogram.zip and then extract the contents (for a zip file).

PCI Serial Port Disappeared - Microsoft Community

donolive say:

PCI Serial Port Disappeared

Split from this thread.

my pci serial port for xp has disappeared. How do I replace it?

lemp say:

Don,

Not only have you replied to a thread from 6 years ago, but your question is not related to the topic of the thread.  Moreover, you should not be using a Windows XP system connected to the Internet.

That said, I'm not sure what you mean by a "pci serial port."  If you have an add-on PCI card that includes a serial port and that's what's no longer working, it may be that the card has failed or has become dislodged from its socket. 

What happened to your computer shortly before the problem with your PCI serial port started?

With the power off and taking proper static electricity precautions, carefully remove and reseat the PCI card.  The close up the computer and power it on.

If reseating the card doesn't help, take a look in Device Manager (click Start, click Run, type devmgmt.msc in the Open box, and click OK or press Enter).  If there are any items with warning icons, double click on them and see what the "Device Status" is.  If the message indicates a missing driver, you'll have to find the appropriate driver for that device and install it.
