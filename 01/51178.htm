Sometimes your computers firmware, which deals with things like power management, which is related to sleep needs to be updated. Remember, the issue you are experiencing here is outside of Windows, where its not resuming from sleep. Windows is in a saved state on the hard disk and can't manage the hardware resources, thats left up to Firmware (your BIOS) to handle. So, its obvious this more a hardware issue than software, but updated driver software can fix it.

It could suggest you need to update your chipset drivers and BIOS.
I recommend you update the most critical ones: BIOS, Chipset, Display Driver, Network adapters.

If that does not work, I recommend you try two things
First: Boot to the Windows 10 desktop
Press Windows key + X
Click Device Manager
Expand Batteries
Right-click Microsoft ACPI-Compliant System.
Choose Update Driver Software.
Make sure you are connected to the Internet, the check for updates.

When complete, restart then check if you can sleep and wake.

Second option
uninstall your display driver then restart
Boot to the Windows 10 desktop

Press Windows key + X
Click Device Manager
Expand Display adapters
Right click the current display adapter
Click Uninstall
Exit Device Manager
Restart

If that does not work, well, we need to try something else. Try resetting the BIOS:
https://www.groovypost.com/howto/reset-pc-bios-...

Windows 8.1 charms bar search using wildcards or partial matches not - Microsoft Community

-zoltan- say:

Windows 8.1 charms bar search using wildcards or partial matches not working in apps and setting search

When you search for an app from Windows 8.1 charms bar, wildcards are not interpreted in the search string.
Therefore an app or setting cannot be find unless you start typing the characters beginning its name or a phrase in its name.

Example:

hairyfool say:

The key you have recorded is probably the Product ID not the activation key and cannot be used to install Windows.

If you can get in via safe mode you may be able to get the original Windows 7 key using  keyfinder utilities, Showkey plus will find the key for the original license. However given that it is an ex works computer this may not help if they reloaded the computer.

Also Windows 7 OEM SLP keys cannot be used to install Windows 7 from disk. If the key  ends in -6P6GT (Pro) or -3Q6C9 (Home Prem) then these are the HP keys

Also if you can get into Safe mode you might just still be able to create the recovery disks. Unfortunately the Windows 10/8 install tends to break the F11 option but read the following guide from HP

http://support.hp.com/gb-en/document/bph07143/

For Operating Systems Product Keys, Product Key Finder Programs are only useful for finding useable Product Keys for Retail and OEM System Builder's Product Keys for Operating Systems that you install yourself.
