keith a. miller say:

There is no option in Explorer to sort folders with files for any property in a regular Explorer view. So your options are:

  • A third-party file manager that has this option.
  • PowerShell: using a command like:
    gci | sort LastWriteTime
  • Using Explorer's Search bar

If you type an asterisk (*) in the Search bar of an Explorer window, the results will be all files and folders in the folder as well as all subfolders. Xbox live uses content delivery networks to distribute downloads and updates.

What country are you in?

Your DNS settings determine what CDN server you connect to.

If I use my ISPs DNS it connects me to a faulty server which means extremely slow downloads. A couple of weeks ago it took nearly 3 hours to download a system update.
