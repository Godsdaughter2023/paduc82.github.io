The temp folder in the Windows directory is intended to be this way.  Temp files related to programs being run by a user are in your %appdata% directory under your user file.  This is by design to virtualize and isolate processes per user and prevent writing to the system directories.

ravinath p say: Hi Gary M.Ball,

Unfortunately you cannot restore the system restore points, if a restore point is not present or deleted. Restore points are saved until the hard disk space System Restore reserves is filled up. As new restore points are created, old ones are deleted. If you turn off System Protection (the feature that creates restore points) on a disk, all restore points are deleted from that disk. When you turn System Protection back on, new restore points are created.
For more information refer the following link.
Windows XP System Restore
http://technet.microsoft.com/en-us/library/bb490854.aspx
 
Hope this information helps.
