I did have version 3.5 of Software Certification Toolkit installed but it has been removed 


It appears uninstalling Software Certification Toolkit may not uninstall the Systrace.sys driver according to here.

I would suggest to stop the Systrace.sys driver from loading by doing the following:

Open regedit and navigate to the following registry key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\SysTrace

At that key in the right window double-click Start and change the Value data to 4 which should disable Systrace.

Create a System Restore Point beforehand and restart the computer afterward.

Hopefully that should prevent any further BSODs.
