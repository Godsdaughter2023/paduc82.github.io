It seems to me that the root of this issue is actually located in the Chromium source code, (which Google publishes as Google Chrome). It seems to me that this bug originated there, then got scooped up into the Electron framework, which then became the framework on which Skype For Linux is built. I have also noticed this same bug in a wide variety of other Skype-like screen sharing services that use Google Chrome or the Electron framework. Some of these services include:

1. Google Hangouts (When used in Chrome or Chromium)

2. The Discord Desktop app for Linux (built on Electron which uses Chromium source code)

3. appear.in (When used in Chrome or Chromium)

4. talky.io (When used in Chrome or Chromium)

5. Many other screen-sharing services that use Chrome or Chromium.

Thus, it does not surprise me that this bug is present here as well.
