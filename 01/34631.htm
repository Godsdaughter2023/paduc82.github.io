Do you mean icons in the notification area (next to the clock) or buttons in the Task Bar that indicate an active task?  In either case, the icons indicate applications that are starting when Windows starts.  You will have to identify the applications you don't want and prevent them from starting.

If possible, you should prevent an application from starting by using the application's own "preferences" or "options" menu.  For icons that appear in the notification area, these menus often are accessible with a right-click.

Command Prompt:

  1. Click Start button and type (cmd) and press (enter) on your keyboard

  2. Right after C:\Users\John> (or whoever the user is)

  3. Type this command exactly: attrib -s -h -r H:/*.* /s /d and press (enter) on your keyboard and wait.

  4. The (H:) represents the drive letter; you want to fix or unhide the files from. Yours may be different

  5. After the command has done its job, all files will have the Hid_(real file name) example: Hid_Deng, Hid_John etc.

  6. Delete all the corrupt files with 1kb size

  7. You can remove the Hid_ before any file if you want just for having your files back to normal such as = Deng instead of Hid_Deng etc

Note: even if you did not removed the Hid_ before the real file name, the files can still open as normal.
