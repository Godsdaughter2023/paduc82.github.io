You may be having troubles with your mouse device drivers, firmware, and/or software.  This may have been caused by a Microsoft Update or any number of things (and could have happened to both machines if you did the same things in both cases).  Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (and you should get them for your mouse at the very least and I would consider the keyboard as well and make certain they are compatible with both your device as well as your version of Vista) and any you have time to do - they may help in ways beyond just resolving this problem), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows:http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows:http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html.

Also, your mouse may have the functionaality to change font sizes and jump from one end of the screen to the other by doing certain things.  Read the user manual and/or search through Start / Control Panel / Mouse to see if that is the case and if so it may be able to be disabled.  The problem with some of these "features' is that if you don't know about them they can cause difficulties when they activate for seemingly no cause when you actually did do what caused the feature to be activated. It took me a while to realize CTRL-0 would change my IE screen size to 100% and I first thought there was something wrong until I discovered the problem quite by accident.  I say that because the chances of two computers having the exact same flaw are astronomical - it sounds more like a problem as described above or some type of feature that you're not aware of (perhaps in combination with some keyboard entry which is why I suggested you update those as well).

I hope this helps.  If not, I'm sorry to say that your best option would be to return it again to see if they can replicate the problem (you use the computer your way in case it's something you're doing) and either fix it or replace it again (or change it to a different computer entirely).

Good luck!

bossdweeb say:

You don't  defrag SSD's. Defraging an SSD can cause premature wear do to doing too many writes.

SSD's use a feature called TRIM (AKA - Garbage Collection ) which is done automatically while idle.

Running the Optimize option after installing Windows would be a good idea.

I don't don't use automated optimization myself. Regular optimization is not really necessary for most people,

but if your system does not have much idle time it may be worth optimizing occasionally.

- Up to you really. It partly depends on how many files or software you add/install or delete/uninstall.

A bit of experimentation will determine if you need regular optimization or not.

I don't really know if doing small tasks such as these forums is deemed as idle time. The drive certainly

does not get much work, but I only optimize my SSD's manually every six months or so.
Even then I don't notice any perceptible change in performance. TRIM seems to do the job well enough.
