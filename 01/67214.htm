As mentioned in my previous post, Windows 8.1 Modern User Interface apps cannot be downloaded from any third party websites or CD/DVD. You can download them from the Windows Store only. If you want to install a Windows 8.1 Modern User Interface app then, you need to use a Microsoft Account to login to the Windows Store and then you should be able to download and install the app on your Surface Pro 2. However, to do this, you need not convert your local user account to the Microsoft Account on the computer.

 

Windows 8.1 provides an option to login to each apps separately using your Microsoft Account. Thus, this will let you to use your Modern User Interface apps which requires you to sign into them using a Microsoft Account (like Windows Store, Mail and Calendar etc.) from your Local User Account on the computer.

 

To login to the Windows Store App separately from the Local User Account on the computer, you may please follow these steps:

 

a) Open Windows Store by tapping or clicking on the “Store” tile from the Start Screen.

b) Swipe in from the right edge of the screen, tap on “Settings” and then tap on “Your account”.

c) If you are logged into a Local User Account on your device, then you should be able to see a “Sign-in” button. Please click on it.

d) This will take you to a screen with the title “Switch to a Microsoft Account on this PC”.

e) Click on the link “Sign into each app separately instead (not recommended)” at the bottom of the screen.

f) This will take you to the “Add your Microsoft account” screen, where you need to enter your Microsoft Account email address and the password.

g) Enter the email address and the password and click on the “Save” button.

h) Now, you are logged into the Windows Store using your Microsoft Account without switching the user account on your computer and you should be able to download and install your favourite apps on your device from the Windows Store app.

Hi

The issue of whether or not to disable the Virtual Memory in windows has always been contentious.

There are thousands of website articles that say disable it if you have a large amount of RAM and just as many websites say never disable it, regardless of the amount of installed RAM.

 

The fact is that some programs are developed to offload pages to Virtual Memory when they are minimized or not being used. If the virtual memory is disabled, you will see the Low System memory error.

 

The bottom line, after all the arguments are read, is that if you are seeing this error on a system where the virtual memory is disabled, then you need to enable it.

andre da costa say:

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already ownto reactivate if necessary. or use the reset function in Windows 10.

 

Right click the MediaCreationTool.exe file then click Run as administrator.

Note: How to verify if you downloaded or installed the Windows 10 Anniversary Update

Select the option Upgrade this PC now if you want to upgrade right away, of if you want to download a ISO image or create a bootable copy on a USB thumb drive, choose Create installation media for another PC then click Next

A new option has been added to the setup screen for users reinstalling Windows 10 on a previously installed and activated system. If your system has a Digital License attached to it, Windows 10 will automatically upgraded:
