jon.jbm say:

So uh, I just fixed it.

I found this installation log file:

Microsoft .NET Framework 4 Setup_20150422_104550414-MSI_netfx_Core_x64.msi.txt

Found in:

C:\Users\[MYPCNAME]\AppData\Local\Temp

I opened it. I hit CTRL+F and searched for "error". I found the following line: "Product: Microsoft .NET Framework 4 Client Profile -- Error 25003. Error occurred while initializing fusion."

I Googled "Error occurred while initializing fusion."

I found the following post:

http://blogs.msdn.com/b/vsnetsetup/archive/2013/09/30/error-25003-error-occurred-while-initializing-fusion.aspx

I renamed the two files listed in the post. I tried the standalone .NET Framework 4 installer right after and it worked perfectly. I now have .NET Framework 4 installed and working.
