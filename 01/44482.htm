considerate_guy say:

A project file (.wlmp) is not a movie. It's more akin to a recipe for making your movie. The project file contains the *names* of the media files (photos, videos and songs), but it doesn't store a redundant copy of the media. So it is just referring to the files that exist on your computer.

If you move the project file somewhere else, or if you delete, move or rename the original source files, Movie Maker can't load those media elements anymore. Instead it will show you yellow triangles with exclamation points. If you know where those files are, but they are just in a different folder, you can try to fix the problem by double-clicking on a missing clip and then navigating to the location on the disc.

If you don't have the original source files, there's nothing that can be done. It's like you are trying to follow a recipe for baking a cake, but someone took away all your ingredients. You need both your ingredients *and* the recipe; the recipe alone won't do.

So in your specific case, you can't just resurrect things on a different computer using the project file. Go to the original computer where the .wlmp file was originally made and also get the source files used.
