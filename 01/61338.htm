chkdsk /f /r X:( drive letter)

Based on the issue description, I understand that you have added misspelled words to the dictionary instead of your fictional name and now you want to know any method to remove those misspelled words from the from the dictionary.

To understand this issue better and help you to resolve this issue, I would like to know the following information:

1. Which application are you using for fiction writing?

3. When you say " Is there any way I can reach the dictionary to remove this word without resetting the entire list" do you mean the systems default dictionary?

All the words that have added to the system dictionary are saved in the default.dic file. It is located in the following location.

"C:\Users\<user name>\AppData\Roaming\Microsoft\Spelling\<language>\default.dic"

Here user name is your current login name, example: John

  2. Double click Local Disk (C:).
  3. Double click User name
  4. Browse through the following location "\AppData\Roaming\Microsoft\Spelling".

  6. Open the file "default.dic". It should open in Notepad, if it gives you the option how do you want to open this file type (.Dic)?.
  7. Select More options under that select Notepad.
  8. Delete the misspelled words that have added, leaving no blank, and save the file.
