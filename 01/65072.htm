Hi RawrCarrotCake,

 

 

It seems the libraries are causing the issue. You may have to remove the files from the libraries completely and then try transferring the data to the external hard drive.

To remove folders from the Library

  1. In the taskbar, click the Windows Explorer button.
  2. In the navigation pane (the left pane), click the library that you want to remove folders from.
  3. In the library pane (above the file list), next to Includes, click Locations.
  4. In the dialog box that appears, click the folder you want to remove, click Remove, and then click OK.

Also follow the link given below for more information

Libraries: frequently asked questions

http://windows.microsoft.com/en-us/windows7/Libraries-frequently-asked-questions
