Hello Tim,

We'll need to modify the registry of your Windows 10 system in order to set the Windows Media Player (64-bit) as the default program.

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs.   1. Click the Start button and type Command Prompt.

    • unregmp2 /SwapTo:64

  6. Navigate to the path below.
    •  HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\App Paths\wmplayer.exe

  9. Click OK once done and close the Registry Editor.
  10. Restart your PC.

It depends on how Windows 10 was originally configured on your computer.

    If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

    If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

    If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.
