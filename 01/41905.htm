Disconnect that 4TB HDD before you install Windows 10, there is a known bug in the Windows 10 installation routine that will cause the boot sector to be written to the HDD instead of the SSD

diskpart

Run this command and hit Enter:

list disk

Run this command and hit Enter:

select disk disknumber

Run this command and hit Enter:

clean

Run this command and hit Enter:

convert gpt

Then close Command Prompt

[quote user="CaRvInG iZ BaCK"]when I turn my xbox 360 on it goes to where the circle keeps goin in circles like its loading but it never loads and when I unplug the Ethernet cord it loads fine[/quote]

Can you try the following:
Unplug the network cable, and turn on the console (hopefully loading completely now).
Sign out of any gamertag by pressing "Guide" (the silver button in the middle) and then pressing X.
Now plug in the network cable.
Go to the settings hub on the far right.
Go to the System tile in the upper left corner.
Go to the storage options, and highlight any drive. Press the Y button.
In the upcoming menu, choose to clear the system cache.

Now go back to the network settings. And do a connection test to Xbox Live.
Does that test work fine now?

If it does, try to sign in with your Gamertag (press Guide again, then X and select your Gamertag).
See how long it takes to sign in.

If the signing in takes too long, sign out again, try to move your Gamertag to an USB storage device or delete it from the console (when deleting, the console will ask if it should delete associated content with it as well. DO NOT let the console delete the associated content! Let the console delete the profile ONLY!).

Now use the "Download Profile" option in the Guide menu to download a copy of your Gamertag profile again.

See if that now signs in quickly again.

Hi Beverly,

Let’s run through a list of clean up actions we take, and see if we have better luck.

Right click on the taskbar, and open Task Manager - click more details, then click the startup tab. Disable all nonessential apps (Ex. Skype, Office, Adobe Updater, etc) running these things at startup will cause your machine to load windows slower.

Have you deleted items from both your Windows Temp Directory and your App Data directory?

C:\Windows\Temp - ctrl + a and delete the contents.

C:\Users\YOURACCOUNT\App Data\Local\Temp (If AppData doesn’t display, go into control panel, file explorer options, and make sure show hidden files / folders is checked off”

After this, Download and install Ccleaner - https://www.ccleaner.com/ccleaner/download

Note: This is a non-Microsoft website. The page appears to be providing accurate, safe information. Watch out for ads on the site that may advertise products frequently classified as a PUP (Potentially Unwanted Products). Thoroughly research any product advertised on the site before you decide to download and install it.

After installing, run the registry tool. Run the registry tool until there are no more fixes to be made. Once done, and no more fixes are there, reboot.

Click start, type “cmd” - right click and run as admin, then type:

DISM.exe /Online /Cleanup-image /Restorehealth (note the space before each "/") - and hit enter

Wait for that to finish, and move onto the next command. Type:

sfc /scannow (note the space between "sfc" and "/") - and hit enter

Finally,

Chkdsk /F /R

When all these things are done, give your PC a reboot, and see if your OS is responding better.

Step 2. Improve High Startups Impact process

1. Press Ctrl + Shift + Del keys > Select Task Manager;
