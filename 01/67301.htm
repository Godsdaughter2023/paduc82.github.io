mark st... say:
Hello maxwills

When you have Windows Automatic Repair loop issue please follow these recommendations:

1) Refresh PC

Press-and-hold Power button 3 times to forcefully shut down the machine. The system will enter boot repair page after 2-3 times of reboot, choose Troubleshoot, then go on to Refresh PC.

2) Disable early launch anti-malware protection

You can do it using Command Prompt in Windows Boot Options menu which will appears after your computer restarts several times. Choose Troubleshoot > Advanced Options > Startup Settings. Your computer should now restart and give you a list of options, here choose "Disable early launch anti-malware protection". Restart your PC and check if the issue has been resolved.

3) Delete the problematic file

- In Windows Boot Options menu, choose "Troubleshoot > Advanced Options > Command Prompt".
- In the command line, type: C: cd Windows\System32\LogFiles\Srt. SrtTrail.txt and press Enter button.

Here, if you see something like this: Boot critical file c:\windows\system32\drivers\vsock.sys is corrupt, navigate to the specified location using the Command Prompt and enter Del command to delete the problematic file.

4) Manually repair MBR and rebuild the BCD

In the Windows Boot Options menu, choose Troubleshoot > Advanced Options > Command Prompt. When Command Prompt appears, enter the following lines and press Enter after each line in order to run it:

bootrec.exe /rebuildbcd
bootrec.exe /fixmbr
bootrec.exe /fixboot

5) Perform a system restore or reset

in Windows Boot Options menu, choose Troubleshoot > Advanced Options > System Restore. Follow the Wizard instruction and choose the appropriate restore point. Finally, click Finish to restore the system.

a. user say:

Hi John,

 

The above issue might have occurred due to pending printing queue documents given by the printer on the computer. Empty/delete the printer queue documents by following the steps below and check if the issue is resolved.

 

  1. Press Windows logo + R keys on the keyboard, type services.msc and press Enter.
  2. Locate Print Spooler service, right click and select Stop to stop the service if it is running.
  3. Check whether the Startup type is set to Automatic. If not, then set the service to Automatic.
  4. Go to the following location and delete all the pending documents from the Printer. C:\WINDOWS\system32\spool\PRINTERS
  5. Delete all the queued print jobs (select and delete all the files) in the folder so the folder is devoid of pending print jobs. The folder should be empty when you are done.
  6. Restart your system and check the Print Spooler Service again to be sure the service is restarted automatically (since the Startup type is set to Automatic) and the Status column for the Print Spooler should read Started.

 

If the issue persists, download and install the latest and most compatible printer drivers from Epson printer driver website and check if the issue is resolved.

 

Note: Select the appropriate Windows operating system from the device website to download the printer driver on the computer.

 

Write to us with the status of printer not working on the computer for further assistance. Your reply is most important for us to ensure we assist you accordingly.

 

Thank you.
