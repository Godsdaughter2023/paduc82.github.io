petercon [msft] say:

Hi, SimonJBWood

The symptoms look similar to what can happen when processing of font hints goes wrong. The hinting instructions in a font can move around the Bezier control points in a glyph outline, and if the instructions are misinterpreted, the glyph shape can go a bit wacky. When this happens, there may be issues in the font's hinting data, or in the rasterizer, or both.

Since it's only happening when printing to that HP printer, that suggests it's a problem with the driver. (Printer drivers typically have their own glyph rasterizer.) You mentioned that you've tried re-installing the driver, but I wonder if it's possible that you didn't get the latest version. HP released a new version of the driver just a few weeks ago, available here:

http://support.hp.com/us-en/drivers/selfservice/HP-Officejet-Pro-8610/8620/8630-e-All-in-One-Printer-series/5367603/model/5385104

Or you can use this tool from HP, which will retrieve the correct driver for Windows 10:

http://ftp.hp.com/pub/printers/hppiw/hppiw.exe

Let us know if this helps.
