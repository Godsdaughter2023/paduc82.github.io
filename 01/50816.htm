I have now got my Samsung U28D590 working at 4K (3840 x 2160) @ 50Hz with my i7 SP3 using the Intel 10.18.14.4080 driver.

Originally I found I could only get 4K at 30Hz via HDMI (This is a limitation of HDMI standard) but not all cables will work for this, I ended up using the original Samsung HDMI cable with a 'Belikin' brand Mini DP to 4K HDMI adapter - I did try other cables and adapters but had problems with all of them.

The first Mini DP to DisplayPort cable I bought (a generic one) also did not work and my SP3 would not connect to the monitor at all.  I have now got it working using a "Lindy" brand DisplayPort to Mini DP cable, (part # 41551)

The other steps I took to get it working at full resolution were:

  • Install the latest Intel graphics driver - you need to do this as follows because install is blocked by default on the SP3 if you try using the .exe version.
  1. Download the .zip version of the Intel driver from:  https://downloadcenter.intel.com/SearchResult.aspx?lang=eng&ProdId=3721
  2. Unpack the .zip file and save it
  3. Open Device Manager, find the Intel display adapter, right click and choose 'Update Driver'
  4. Select 'Browse my computer for driver software.'
  5. Select 'Let me pick from a list of device drivers on my computer' (note: if you 'Browse' from here it will not work, you need to follow the next two steps)
  6. Click the 'Have Disk...' button
  7. Then browse to find the Intel driver you unpacked in step 2 (for me this was igdlh64.inf, look in the 'Graphics' folder of the unpacked files)
  • Restart the system
  • Connect the monitor with a good quality Mini DP to DisplayPort cable
  • Right click on the desktop and then follow (the new) 'Graphics properties' menu option => click 'Display' in the new window.
  • In the 'Display' window, click on the word 'Display' (top left) and scroll down to 'Custom Resolutions', click 'Yes' at the dialogue box asking if you want to continue.
  • Create a new, custom resolution (I used 3840 x 2160 @ 50Hz with timing mode set to CVT-RB for my U28D590) and save it. (Note: I tried 60 Hz but got a message that it 'exceeds the maximum bandwidth', 50Hz was the highest rate I could get to work).
  • Restart the system
  • Right click the desktop, go to screen resolution => select the new resolution (mine showed as native).  If you still don't see the new resolution, click on 'Advanced', click on the 'Monitor' tab and un-check the 'Hide modes this monitor cannot display' option.
  • You should now be up and running - good luck!
