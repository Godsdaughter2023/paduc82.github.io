How do I disable the lock screen?

How do I disable the lock screen?

wilfredo_soto say:
Hi Robert
I am Wilfredo an Independent advisor


Open the Windows Registry editor by typing the regedit command in the search engine. Press the Enter key to execute it.

Navigate to the HKEY_LOCAL_MACHINE \ SOFTWARE \ Policies \ Microsoft \ Windows path

Create a DWORD entry with the name NoLockScreen and assign a value of 1.

Save the changes and restart the system. You will not see the lock screen again, although the login window will appear if you have it configured.

You can always return to the default configuration by assigning the new entry 0 or deleting it from the system.

a. user say:

You must be logged on as an administrator to perform these steps.


If you receive Windows Update error 80246008 while downloading updates, you might need to change the Background Intelligent Transfer Service (BITS) or Windows Event Log service settings, and then restart each service.

 

To change or restart the Background Intelligent Transfer Service (BITS)

1. Open Administrative Tools by clicking the Start button , clicking Control Panel, clicking System and Maintenance, and then clicking Administrative Tools.

2. Double-click Services. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

3. Right-click the Background Intelligent Transfer Service (BITS) service, and then click Properties.

4. On the General tab, next to Startup type, make sure that Automatic (Delayed Start) is selected.

5. Next to Service status, check to see if the service is started. If it is not, click Start.

 

To change or restart the Windows Event Log service

1. Open Administrative Tools by clicking the Start button , clicking Control Panel, clicking System and Maintenance, and then clicking Administrative Tools.

2. Double-click Services. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

3. Right-click the Windows Event Log service, and then click Properties.

4. On the General tab, next to Startup type, make sure that Automatic is selected.

5. Next to Service status, check to see if the service is started. If it is not, click Start.

 

Repeat here:

What service Background Intelligent Transfer Service needs to function properly:

COM+ Event System

Remote Procedure Call (RPC)

DCOM Server Process Launcher

RPC Endpoint Mapper

 

If the problem persists after you have followed the steps above, see this Knowledge Base article on the Microsoft Help and Support website <http://support.microsoft.com/?kbid=910337> or go to the Windows Update Support Center website to request support by e mail. <https://support.microsoft.com/oas/default.aspx?gprid=6527&st=1&wfxredirect=1&sd=gn>

a. user say:

Hi chjm,

In addition to the information skier-hughes provided. It is also possible that this issue is caused by a large movie project, missing or faulty codecs and outdated display driver, please check the following:

1. If you have a large movie project, splitting the video into smaller projects may help. Open the whole project, split them to multiple parts. Save them as a movie individually. After saving each part as a movie, import and then combine the saved movies in Windows Live Movie Maker.

2. Audio and Video require codecs to play the file and it’s important to have the right codecs installed or updated in your PC. 

• Try running the Microsoft FixIt tool for codecs: http://support.microsoft.com/mats/video_freezes_or_crashes/

• Media info will be able to identify any issues with missing or faulty codecs and whether it’s already installed in your computer or not. You can get it at  http://mediainfo.sourceforge.net/en

3. Movie Maker makes extensive use of the video card GPU (graphics processing unit). The display driver affects the reliability, performance, and compatibility of the video card with various software.

To determine the version and video card driver date, run the DirectX Diagnostic Tool (dxdiag):

DirectX Diagnostic Tool
• Click on Start button and type “dxdiag” in the Search programs and files box and press Enter key on the keyboard.
• Click on the Display tab and you will see the driver details including the driver version.
• If your video card driver seems outdated please go to the video card driver manufacturer and see if a newer version is available.
