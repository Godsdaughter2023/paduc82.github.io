j w stuart say:

The Windows.old folder has two purposes:
1) Can be used to un-install Windows 7 and return your PC to the previous version of Windows.
2) Contains all your personal data files, documents, pictures. Etc.

If you do not need either of these two options the Windows.old folder can be deleted.

Note: Change Windows Explorer option to see the Windows.old folder:
Click Start and in the 'Search' box enter:  "hidden"  (without the quotes)
Click the "Show hidden files and folders" option at the top of the list.
In the 'Folder Options' Window:
Click of the 'View' if it is not already selected.
In the 'Advanced settings'  list locate 'Hidden files and folders'.
Now click on 'Show hidden files and folders'
Next uncheck the following: 
'Hide empty drives in the Computer folder'
'Hide extentions for know file types'
'Hide protected operating system file'

Click the 'Apply' button and the 'OK' button.

Hello, HEIDISCOTT1

A few clarifications here ...

  • When you write that you "just paid for a UK number to call your family," this sounds like you purchased a Skype Number subscription. A Skype Number serves as a land line number people can use to contact you, and you answer those incoming calls on Skype. There is no credit or subscription included with a Skype Number subscription.
  • Unless you set up Skype Caller ID, the Skype Number or your mobile number will not appear, and all calls placed from Skype to land lines and mobile numbers will show as originating from an unknown number.
  • The calls you place to U.K. landlines as well as mobile numbers should connect without any problems if you set up Skype Caller ID.
  • To cover the costs of calls to landline and mobile numbers, you will need to add Skype Credit or a Skype Subscription to your account.

To contact Skype Customer Service, remember these tips:

  • Temporarily enable browser pop-ups as you will be requested to approve an instant message chat dialogue box when the chat application launches as a separate browser pop-up window. 
  • Also, at the Support Request page, select the Payments & Orders option and then Question About a Charge or Purchase. Click Next. Text Chat Support should appear as a contact method.
  • If you do not succeed in reaching an agent, try again using a different web browser, in-private browsing mode, as well as clearing the browser's cache and cookies.

You can clone the installation to the SSD, provided the SSD is large enough to hold the installation to be cloned.
