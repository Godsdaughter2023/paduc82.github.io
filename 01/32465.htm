dariuscollins say:
  • Open "regedit" as adminstrator.
  • Navigate to: HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced]
  • Find "ShowInfoTip", right-click and select "Modify" and change the value to "1".
  • Open Task Manager and select the Processes Tab.
  • Select explorer.exe and hit delete key, then select the End process button in the Windows Task Manager dialog that pops up.  
  • Select the Applications tab and click the New Task... button in the bottom right corner.  
  • Enter "explorer" into the combobox and hit OK.  
  • When the task bar reappears, all task bar tooltips, including jump list tooltips will now work.
