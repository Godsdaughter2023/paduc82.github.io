A sequential power cycle is the first step in clearing connection problems (since you have an 8007* error), and the order and timing of the steps is important to success:

1) turn off the Xbox and unplug the power supply cord from the back of the Xbox then
2) unplug power from the router (Some folks have a combination modem-router AKA 'gateway' or only use a modem with no router. If either, skip steps 3 & 6).
3) unplug power from the modem.
4) Wait at least 5 minutes then replug the modem/gateway power.
5) Wait 'til the light on the front of the modem/gateway which says BROADBAND, ONLINE, READY, CABLE or DSL is on solid (not flashing), then plug in the router. If you have a gateway instead of a modem and router or a modem with no router, then when that light says it's ready, skip to step 7.
6) Wait until the WiFi light on the router is on; you may not use WiFi, but it's still a good indicator your router is finished booting up.
7) Plug in and turn on the Xbox.  For more background on Ports, see http://whatismyipaddress.com/port

If the problem persists, please send us the rest of the info we need from http://forums.xbox.com/xbox_forums/xbox_support/f/9/t/155724.aspx so we may work on this further.

This helps centre old low res game that use a 4:3 aspect ration on the screen. Note that Windows will often force this

setting on games that don't need it, but fails to do so on games that do need it.

- I've also found a couple of games that do need this setting which will fail to launch if it's not set.

- Next, if you are using disc based versions of old games you will find many will be blocked from loading due to a 'security feature'

MS build into Win10 - DRM Blocking. I have details on this scourge in my post at   ---this thread ---

"I know this new Directx 12 version has problems with older games because of its API feature. "

You have been misinformed. Older games do not have any issues with DX12 because they don't use DX12, and the 'API' does

not come into it.

Win10 comes installed with all DX9,10,11 and 12 files. There are numerous sub-versions of DX9, but fewer sub-versions of DX10 & 11.

The game determines which DX files it requires, not Windows, so all DX files need to be installed.

You cannot uninstall (properly) any version of DX,  and you do not install just one version of DX.

There are no off line installers for only DX 10, 11 or 12. In fact there are no off line installers that include DX12 at all. Not yet anyway.

- The site linked to above has been promoted by several people on these forums, but I doubt any of them have actually

downloaded the so called different installers or they would have found that none of them are only for  DX 10 or 11.

The so called DX12 installer is actually a link to the DXSDK 2010, which is a developer tool, and

Use the DirectX Web Installer to verify if all DX files are installed in your Windows.

(including all DX9 files), or show a message to the effect that your DX is up to date.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

- Note that people will often still recommend installing DX9.0c for old games. This installer is redundant. Should there be a need to

reinstall DirectX due to corruption of an unspecified DX file or files use the DirectX Redist (June 2010) installer package.
