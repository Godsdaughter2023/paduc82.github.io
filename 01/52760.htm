sumit (volunteer moderator) say:

Two different cases:

  • Windows Vista and Windows XP: They have been released a separate patch for the latest attack. No prerequisites.
  • Windows 7, 8.1, 10: The updates of these OSes are cumulative. The newer patch contains all the previous patches as well. Earlier 7, 8.1 did not follow this, but now they are. For example Build 14393.953 includes patching for MS01070(or whatever) and the later builds like 14393.1028 also contains the previous fixes.
