If you really have an "80211ab" (i.e., 802.11a and 802.11b) wireless adapter, it's not surprising that you can't connect.  The coffee shop probably configured its router to use 802.11g only.  Although 802.11b is compatible with 802.11g, when both 802.11b and 802.11g clients are connected to an 802.11g router, the performance of the 802.11g clients can suffer. In the worst case, all 802.11g clients will slow down to have the same network speed as the 802.11b clients.  If people were complaining about wifi performance, whoever dealt with the issue may have decided that because 802.11b is so old that hardly anyone uses it anymore it wouldn't matter if the router was set to wireless-G only.

What version of Windows do you have, including service pack?

Assuming that you have some variety of Windows XP ...

A.  If you are using Windows to configure your wireless network adapter

  1. Open Network Connections (Start > Run > ncpa.cpl > OK)
  2. Right click on the icon for your wireless network adapter and select Properties
  3. Click the "Wireless networks" tab
  4. Select each network in the list of "Preferred networks" that relates to these coffee shops and for each one, click the "Remove" button
  5. Click OK
  6. If you are at one of the coffee shops, click "View available wireless networks," select the appropriate network, and click "Connect"
  7. If you are not at one of the coffee shops, click "Close" to exit from the Wireless Network Properties dialog and then connect to the coffee shop network as usual when you next visit

B.  If you are using a third-party utility to configure your wireless network adapter

  1. Delete all "wireless profiles" relating to the coffee shops
  2. Reconnect to the coffee shop network
