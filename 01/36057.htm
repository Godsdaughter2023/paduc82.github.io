¡firedog say:

This is a well-known oddity in the import function that occurs in certain circumstances. To correct it,

  1. Create a new folder for the sent items.
  2. Create a new folder on the desktop.
  3. Select the messages in the imported Sent items folder, drag them out of WLMail and drop them on to the new folder.
  4. Open the new folder and check (e.g. using Details view and sorting by Type ) that all the messages have the extension .eml . Any message whose subject ends with a period ('.') will probably have the extension ._eml . This has to be corrected*.
  5. Select all the messages and drag them back into the folder you created at (1).
  6. If all is well, delete the original imported Sent items folder and the temporary folder on your desktop.

* To deal with them all in one go,

  1. Shift-Right-click on the name of the new folder in Explorer's navigation pane and select Open command window here.
  2. Click inside the window, type


    and press Enter.
  3. Close the command window.

Good luck!
