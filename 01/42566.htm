Windows keeps moving my Word shortcuts. How do I make it stop?

I have several Microsoft Word documents I access on a regular basis, and keep shortcuts for them on my desktop so I can easily access them.  Every time I open one of the documents, however, Windows moves the shortcut icon across the desktop and puts it somewhere else.  The only solution that has been suggested to me so far (often rather insultingly) is to turn off the auto arrange and align to grid options.  That was the first thing I tried, though, and it had no effect (much like trying to set "don't add space between paragraphs of the same style" as the default has no effect).

 

Is there some other way to prevent Windows 7 from moving the icons every time I open a document?

cmdrkeene say:
Windows 7 isn't moving it -- Word is.  When you open a Word file, it creates a temporary copy of the file to hold your edits while you make them.  When you press save, the original file is destroyed and the temporary copy saved with the same name.  Because the file is literally being removed and a new one added, the icon disappears and the new one appears at the top-left of the screen (or first open icon location).

Windows 7 itself can't prevent this, because it can't track the position of a file icon that, in actuality, didn't exist a few seconds ago.

You can store the Word Documents in your Documents folder (or somewhere else) and use only .lnk shortcuts on the desktop.  This should prevent the issue because the temporary file is being created in the folder.
