obsessive power say:

Hard drive usage for Xbox One games, Xbox 360 games or games on any system can not be determined by the size of the game - as in how much you can do or even the size of the game world.  The amount of storage a game uses is mostly down to it's graphics textures.  

Battlefield 4 for example has very varied environments.   This game goes from urban combat, to grassy plains to deserts and even island battles. The textures for these environments are all different and are of much higher resolution than the equivalent texture graphics on the Xbox 360 version, hence the larger data footprint.

As for 'Disc run games', such a concept does not exist on the Xbox One.  All Xbox One games are installed fully from the game disc.  Once you install the game from disc, the console only reads the disc to confirm ownership, the game itself runs entirely from the hard drive.  Xbox One games purchased on disc use exactly the same amount of hard drive space the digital versions.

Xbox One reserves part of the hard drive storage for system software, native apps, system updates and other system related needs.  The user portion of the hard drive is actually 365GB.  This may have been an issue a few months ago, but the most recent updates now allow the use of USB 3.0 external hard drives.  The minimum is 256GB, there is no maximum upper limit.
