How are you accessing Command prompt?

I suggest you to enable the built in admin account from the Windows Recovery Environment, login to that account and then check if you are able to access settings, control panel and start button. I suggest you to go to Windows Recovery Environment and use the Command Prompt there, which is elevated.

Follow the below steps:-

  1. Press Shift and Restart from the power icon (together)

  2. Select Troubleshoot.

  3. Go to Advanced Options.

  4. Select Command Prompt.

  5. Type "net user Administrator /active:yes"

  6. Hit Enter.

A user account with the name Admin will be created. Log in with that account and you will be able to access Admin privileges. You can also create more Administrator accounts with that account. If you are able to access start menu setting etc. I suggest you to create new user account.

Hi carolcoleuz               

Skype Translator is available for Skype for Windows desktop, Skype for Windows 10 and Skype for Web. 

If you are using Skype for Windows desktop, the translator icon is located on the upper right side corner of you and your contact's chat window screen.

To enable Skype Translator: 

  • Select your contact and click the Translator icon.
  • Turn Translator On and then select the language for translation while in a call or instant messaging. 

Please note that Skype Translator only supports the following languages for voice/video calls. 

  • Arabic
  • Chinese (Mandarin)
  • English
  • French
  • German
  • Italian
  • Japanese
  • Portuguese Brazilian
  • Russian
  • Spanish

And over 50 languages for instant messaging. 

Please make sure you are using the latest version of Skype available for your device. If the issue persists, you can proceed with completely uninstalling and re-installing the Skype application. You can use this support link for the step-by-step guide.
