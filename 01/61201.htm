Regarding the issue related red x on the files I suggest you to rebuild the icon cache.

  1. Click on the search bar on the taskbar and type CMD.

  2. Now right click on it and select Run as administrator.

  3. Now in the command prompt window type in the following commands:

ie4uinit.exe –show

taskkill /IM explorer.exe /F

DEL /A /Q "%localappdata%\IconCache.db"

DEL /A /F /Q "%localappdata%\Microsoft\Windows\Explorer\iconcache*"

shutdown /r /f /t 00

Note: After running the last command the PC will restart automatically. Please make sure you save all the files before running the command.
