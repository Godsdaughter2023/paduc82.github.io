    I simply navigated to my NAS drive in file explorer by typing in the name of the drive into the navigation bar like this  \\NASNAMEGOESHERE.  IN my case \\NAS02.  This is the name you gave the specific NAS device when configuring it (or how it came pre-configured).  If you don't remember this you can use a browser to manage your NAS to find out.  Like this:

    Once you have the name type it into File explorer as in the example below:

    Right click on this and select 'Pin to Quick Access'
