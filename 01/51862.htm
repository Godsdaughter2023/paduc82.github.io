The update failed with this status code: 4476 0000 3080 0505 8007 0570

Please help.

According to the first number, your console is already on the latest version, and the second number (all zeroes) indicates that the console is looking for an update that doesn't exist. 

That usually happens, when the storage drive, that the console uses to store parts of the system software became corrupted. 

If you have a 4GB model with built in Flash memory, you're out of luck and have to send in the console for repairs.

If you have a hard drive in the console, you might try to remove the hard drive, and attach an USB drive to the console, and format that as a storage device in the storage settings. 
However, unless you stored your savegames in the cloud storage, your game progress is gone, and you can only recover your Gamertag account by downloading it on a new drive.

Hi

What you are seeing is called a TDR process (Timeout Detection and Recovery)

The system detected that a driver was taking too long to respond to a request. When this happens, windows notifies the system that the component will be stopped and restarted to recover.

In previous versions of Windows, this situation would cause a hard lockup which would require restarting the computer to recover.

The most common cause for this error is usually the graphics device driver, but it can also be caused by other hardware components such as, memory modules, insufficient cooling, or insufficient system power.

 

Besides making sure that all of the hardware drivers are kept up to date, you may need to adjust which programs and the number of programs that are active on the computer at any one time.

 

You might also ask Acer support about any known issues on that system.
