This issue could be caused due to some incorrect settings or due to a misconfigured or corrupt driver. I would suggest you to try these steps and check:

 

First, I would suggest you to check the device status of the Audio adapter in the device manager. Refer to these steps:

 

a)      Press Windows key + X, select Device manager.

b)      Expand the “Audio input and output ” section to locate the Audio.

c)       Check if there is a yellow exclamation mark or a red cross mark over it.

d)      Right click on device and select properties.

e)      In the General tab, check the device status.

 

 

Method 1: Playing Audio

 

This built-in troubleshooter can automatically detect and fix most of the incorrect settings and errors that could cause issues with playing audio. Refer to these steps to run this troubleshooter:

 

a)      Press Windows key +F to bring up the search Charms bar.

b)      Select settings in the drop down menu below search and type Troubleshooting.

c)       Click on Troubleshooting and select the view all option on the left pane.

d)      Click on Playing Audio troubleshooter and follow the prompts.

 

Method 2:

 

If the issue still persists, then I would suggest you to reinstall the driver and check. Refer to these steps:

 

a)      Press Windows key + X, select device manager.

b)      Locate the Audio adapter, right click and select properties.

c)       Click on the driver tab and click uninstall to uninstall the current driver.

d)      Now, restart the computer and Windows will load the generic audio driver for the computer.

e)      Check if you face the same issue with the Windows generic drivers.

f)       If yes, download and install the latest audio driver.

g)      Now, right click on the audio adapter and select “Scan for hardware changes”.

 

Now, check if the issue persists.
