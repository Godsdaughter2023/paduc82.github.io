-  Sony VAIO VPCF126FM with a NVIDIA GeForce GT 330M Video Processor

-  HP ZBook 17 G3 Mobile Workstation with a NVIDIA Quadro M3000M with 4 GB GDDR5 VRAM Video Processor

I use a X-Rite Color Munki to calibrate my Monitors to ensure accurate color rendition for my photography business.  It is absolutely critical that my screen colors match those of the image for accurate web display and printing.

Prior to installing the Feature Update to Windows 10, Version 1903 on June 6, 2019, both laptops would load the most recent color profile that I created on each laptop using the Color Munki and set as the default color profile.  In each case, the computer would load the proper profile during the boot-up process.

Since installing the Feature Update to Windows 10, Version 1903 on June 6, 2019, neither laptop will load any color profiles during or after boot-up from a shutdown state.  Using the Color Management Application has no effect and will not allow me to change the color profile from the drop down list.

If I RESTART the laptops, the computers will load the default color profile that I created with the Color Munki, just as it did prior to installing the Feature Update to Windows 10, Version 1903 on June 6, 2019.  I am also able to select different Color Profiles using the Color Management Application after a RESTART.

If I SHUTDOWN the laptops, neither laptop will load any color profiles during or after boot-up from a shutdown state.

It appears that some changes were made in the Boot-up steps that are bypassing Color Management data, specifically the Color Profile, and setting the display to a default color scheme that is not calibrated for color accuracy.

I have contacted other Windows 10 Users who also calibrate their monitors with a Color Munki or other calibration tools and create computer-specific, display-specific color profiles to ensure color accuracy.  They are experiencing exactly the same issue as I am.

This problem is creating a significant work-around for those of us who depend on custom color management profiles for color accuracy.
