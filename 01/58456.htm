Windows does NOT auto-sign in a user by default. It must be so configured by a user. This command lets you control it:

control.exe userpasswords2

Ticking the box "User must enter a password" will disable auto-sign in.

You DO have full control over your machine if you have the know-how. Here is the administrator's way to delete any account:

  1. Log on under an admin account.
  2. Open an elevated Command Prompt (cmd.exe).
  3. Type these commands and press Enter after each:
    cd  \Users
    dir
    (to see the names of the user profile folders)
    rd  /s  /q  "John Doe"
    (to remove John's user profile folder)
    net  users
    (to see all account names)
    net  user  "John Doe"  /del
    (To delete John's account)

Warning: both actions are irreversible.
