emma_h_15 say:

How to repair registry

Trying to run another program failed because it says flash.ocx is not included in my version of Windows. I believe this is a registry file. Is there any way to replace it, and to find out if other files like this are missing/corrupted?

a. user say:

Stay far away from registry cleaners, registry optimizers, and registry repair tools.  Most do not do any good and many can cause a great deal of harm (to the point where you need to do a clean re-installation - to say nothing of the fact that many of these programs are actually malware in disguise).

That said, there is no real way to "repair" the registry (though there are literally hundreds of products that claim to be able to do so).  You can use System Restore to go back to an earlier point in time when the registry was working fine (it restores the registry as it was at that point in time).  If you want to try this, try a System Restore to a point in time BEFORE the problem began.  Here's the procedure: http://www.howtogeek.com/howto/windows-vista/using-windows-vista-system-restore/.  Be sure to check the box to show more than 5 days of restore points.  If the first attempt fails, then try an earlier point or two.  NOTE: You will have to re-install any software and updates you installed between now and the restore point, but you can use Windows Update for the updates.

What program (name and version and download web site) are you trying to install that won't work because flash.ocx is not included in your version of Windows?  What version of Vista are you using - you provided part of the answer (for example, 32-bit Vista Business SP2)?  Do you have Adobe Flash Player installed on your system and is it the most recent version available and compatible with your OS? 

--------------------------------------------------------------------------------------------------------------------------------------------------------------------

There are methods of detecting and repairing corrupted hard drives and some system files (though I believe your problem is only with Adobe Flash Player and if you fix that, you will fix the problem {I don't think there's anything wrong with Vista itself} - the procedure to fix your particular problem is listed below).  Here are the procedures:

Go to Start / All Programs / Accessories / Command prompt and right click on command prompt and click run as Administrator.

Type sfc /scannow and enter and let it run.  It will scan and try to fix some of your system files.  Hopefully it will complete with no corruption it could not repair (if there is such corruption post back here or try to analyze it to find the problem file(s) using http://support.microsoft.com/kb/928228.  Try to post any corrupted files here so we can see if they can be repaired with good copies from the installation disk (unless there are too many).

While in Command Prompt, type chkdsk /f /r and enter and let it run.  It will scan and try to fix any corruption or bad sectors on your hard drive and mostly remove that as a potential cause.  You may need to reboot your machine because it may schedule itself to only start at the next startup.

-------------------------------------------------------------------------------------------------------------------------------------------------------------------------

Here is an article on how to fix flash.ocx (which is part of Adobe Flash Player (there are no files per se inside the registry)).  The best way to "replace" it is to install the most recent version of the Adobe Flash Player - but I include this article in case it is already installed and it does address that problem directly. http://askville.amazon.com/fix-flash-ocx/AnswerViewer.do?requestId=1395408.  Another option you might try is to uninstall Adobe Flash Player, download the newest version from the Adobe site, and re-install the program.  That might repair the problem without needing to go through the procedures of the above fix.  The chocie is yours.

I hope this helps.

Good luck!
