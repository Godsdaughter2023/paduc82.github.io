ludwig j say:

There are two easy ways to share photos with another user of your computer.

1) Go to the desktop. Open File Explorer (new name for what used to be Windows Explorer). Find the folder you wish to share. Right-click the folder. In the drop-down menu find and click "Share with...". Click on the other user's name.  This is pretty much the same way that you used to share in XP.

2) While in Photo Gallery expand Pictures in the nav pane at the left. Find the "Public Pictures" folder. Every user of the computer has already access to that folder. You can create a new folder inside "Public Pictures". Copy the pictures you wish to share to that folder. The easiest way is to select them, hold down the "Ctrl" key (to "Copy") and drag them to the folder in Public Pictures. The other user can go to that folder and move them (or copy them) to his/her folder of choice.

If your router has a good, working UPnP implementation (many consumer-level routers do not), you should not even have to forward ports, just enable UPnP AND delete all Xbox-related port forwards, disable DMZ, and reboot the Xbox (hold power for 10 seconds).

If using UPnP doesn't get you open NAT, you will need to manually forward ports.  Despite what many here recommend under the mistaken belief it is needed (largely due to misleading information on the MS support site), these are the only ports you need to forward:

88 UDP

3074 TCP and UDP

and if you have COD Ghosts, also:

3075 TCP and UDP

3076 TCP and UDP

Please delete all other Xbox-related port forwards, disable DMZ, forward ONLY the above ports to the Xbox, and reboot the Xbox (hold power for 10 seconds).  Test again and, if the results screen does not show open NAT, hold LT+RT+LB+RB for additional information. Report back with that information if you still have problems.
