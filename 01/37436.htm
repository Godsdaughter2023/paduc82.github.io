is still quite acceptable. Blur when panning may be more pronounced with lower frame rates, but this should

not be an issue with a driving game.

Some games are cpu intensive (particularly poorly optimised ones ). Most are gpu reliant, but monitor cpu

use as well. If the game is also pushing the cpu over 90% you may need to dial back some

settings. AA is always a good start.

- PhysX has been known to cause issues on a number of games.  eg-  Bats-AC is prone to crashing when
