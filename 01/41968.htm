Hello Chugsum

The MS-DOS (MicroSoft Disk Operating System) is an operating system. Its objective is to facilitate communication between the user and the computer, and efficiently use the available resources. The programs that are part of MS-DOS are the tools that the computer will use to perform administrative tasks such as viewing the contents of a disk, creating files, etc.

MS-DOS starts when the computer is booted. The computer checks the memory and looks for the files CONFIG.SYS, AUTOEXEC.BAT and COMMAND.COM on the system disk. The CONFIG.SYS file is a text file that contains special commands. These commands configure the hardware components of the PC so that MS-DOS and applications can use them. When MS-DOS is started, it executes the commands in the CONFIG.SYS file. The CONFIG.SYS file is usually located in the root directory of drive C. The AUTOEXEC.BAT file is executed whenever the system is initialized or reinitialized. The orders that usually contain are usually updating the date and time, file search paths, prompt configuration, keyboard type, etc. It always has to be in the root directory. The COMMAND.COM program processes a series of internal DOS commands and programs. It is an order processor.
