You’ve done almost every possible step to get the mouse to work. Since the behavior is the same for all the three Operating Systems, the issue is more likely related to the mouse itself.

Just to cover all the bases, check in safe mode. See how it behaves. If there’s any improvement, then we can move to the next step which is clean boot. Safe mode starts Windows with a limited set of files and drivers. Startup programs don't run in safe mode, and only the basic drivers needed to start Windows are installed.
