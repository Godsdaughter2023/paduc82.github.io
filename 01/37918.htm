Run these networking commands in a command prompt window since running the Network Troubleshooter didn't work:

srinivas kr say:

Hi,

 

Please follow the methods listed below.

 

Method 1:

Check for the dependencies of Print Spooler service and verify if they are started.

 

a)      Click on Start, in the start search type services.msc and press enter.

b)      Locate Print Spooler.

c)      Right-click on Print Spooler and select Properties.

d)      Click the Dependencies tab.

 

Look for the “Dependencies” under “Print Spooler” and make sure all the dependency services are running and set to “Automatic”.

Print spooler is software that temporarily stores print jobs on the computer hard disk or in memory until the printer is ready to print them.

 

If you've been waiting long time for a document to print, or if you see an error message about the print spooler, you might need to restart the print spooler on your computer.

 

Method 2:

Follow the steps mentioned below

 

a.      Click on Start button, type cmd and press enter.

b.      At the C command type: net spooler stop and press enter. You will get a message when the spooler has stopped. Now you need to clean out the queue.

c.       Click on start button, type: %SYSTEMROOT%\system32\spool\PRINTERS and press enter.

d.      This will bring up the folder. Delete every file in there. (This will not harm anything.) Now you need to start the spooler again.

e.      Go back to the C prompt screen and type: net start spooler, press enter

 

Check your printer queue. Sometimes you may need press f5 to refresh the screen.
