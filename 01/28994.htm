NTFS_FILE_SYSTEM (24)

This indicates a problem occurred in ntfs.sys, the driver file that allows the system to read and write to NTFS drives.

One possible cause of this bug check is disk corruption. Corruption in the NTFS file system or bad blocks (sectors) on the hard disk can induce this error. Corrupted SCSI and IDE drivers can also adversely affect the system's ability to read and write to disk, thus causing the error.

Another possible cause is depletion of nonpaged pool memory. If the nonpaged pool memory is completely depleted, this error can stop the system. However, during the indexing process, if the amount of available nonpaged pool memory is very low, another kernel-mode driver requiring nonpaged pool memory can also trigger this error.

How do I install previous versions of my graphics driver?

So in my ongoing attempts to thwart my sudden issues with OpenGL-based games/programs, I'm attempting to backdate my graphics drivers (I have a laptop with an AMD Radeon HD 6490M card). AMD themselves kindly supply previous versions of the graphics drivers bundled with the "required" Catalyst Control Center. I downloaded one dated late 2013 and ran the .exe file, and all that happened was CCC ran as normal and told me everything was up to date. I'm not convinced this replaced my drivers, so:

What do I need to do, step by step, to backdate my graphics drivers?

cmdrkeene say:

Delete the current set before you run the one you downloaded.  Easy way to do that is to open Device Manager (search by name in the Start menu's search box), find the graphics card driver, go to properties and uninstall.  After it's uninstalled, run the one you downloaded to load the new ('old') driver.

jetta48 say:

Hi

The pre-installed games like Solitaire that you may have had in earlier Windows versions are not available in Windows 8/8.1. Instead there are free Microsoft games available to download install via the Windows Store (the Store tile on your Start screen). The available games from Microsoft include:

  • Microsoft Solitaire Collection
  • Microsoft Mahjong
  • Microsoft Minesweeper
  • Microsoft Taptiles
  • Wordament.

There are 3rd party methods that claim to install the Windows 7 style games on Windows 8/8.1, but I haven't tested them so I will not recommend them.

a. user say:

Hi Kelvin,

 

Thanks for posting in Microsoft Community

 

As I understand you are facing issues with the Display Color Problem. I would recommend you to follow the following steps in order to resolve the issue.

 

A) Press Windows + R on the desktop to bring up the Run Window.

 

B) Open Control panel by typing “Control” in the Run Box.

 

C) Click System and Maintenance, click System, and then click Device Manager.

 

 

Note: If Control Panel is in Classic View, double-click System, and then click Device Manager.

 

 

A) If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

 

B) In Device Manager, double click to expand "Display Adaptor", right click on the Monitor, click "Uninstall" and click "OK".

 

C) Go to the manufacturer’s website and the download the latest set of drivers from the manufacturer’s website.
