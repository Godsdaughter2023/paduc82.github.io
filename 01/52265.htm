statesmanlike m. quadruped say:

It's possible to send Windows 10 into Safe Mode without being logged in first.

Shut down the computer, then do a clean boot. When the Windows 10 splash screen appears, wait for the spinning dots to show up, then press and hold the Power button to force the computer to shut down.

Boot the computer again, and when the spinning dots show up again, do the same force-shutdown thing again.

On the third boot attempt, Windows will prepare "Automatic Repair", and should give you a UI. If you dig into that interface, one of the options it gives you is "Startup Settings". Select that, and Windows 10 will reboot into an interface which gives you basically the same options you used to get from the press-and-hold-F8-when-booting menu - but in a "press this key to choose that result" style, rather than a press-up-and-down-to-change-the-selection menu.

To fix the wrong-resolution problem, I found that the three options labeled as "Safe Mode" didn't help, because they all came up with the same too-high resolution. Instead, I had to choose the option labeled as "low-resolution video mode", by pressing either 3 or F3 at the Startup Settings menu.

Once booted into Windows with that low-resolution video mode, I changed the display resolution to something other than what it was already set to, and rebooted into normal Windows 10. It came back up with the resolution I had specified; from there, I was able to adjust normally.

My experience on this is with Windows 10 version 1709, but in theory most if not all of it should apply to other versions as well.

Can I add my novice advice?

You are changing a lot of hardware all at once, so take baby steps:

-if your old MoBo is working fine, change it last...

-upgrade you video card in the old MoBo, get it's drivers correct and let Win 10 activate with the new hardware (Win10 won't object with a new video card, it's the most common upgrade)...

Get a good image of your system at this stage...

If possible:

-install your new M.2 in the old MoBo and apply the image you just made to it...

-get everything working correct and activated (I don't think Win10 will object to the new HDD/SSD, another common upgrade)...

Get another good image of your system (overwrite the 1st image, as long as everything is working great)...

That leaves your new MoBo swap, Win10 WILL object to this hardware change, there are other threads you should research on these forums on "How to activate Windows10" after a MoBo upgrade... You might get lucky and it will just activate...

IMO, use your old M.2 in the 2nd slot if possible, check your MoBo spec's...

Questions:

-are you using your old CPU in new MoBo? Why? You should be using the best CPU for your system...

-are you using your old Memory sticks? Why?

The MoBo, CPU and memory is to be completely compatible for highest performance of your new system, otherwise "Why" change them?

Without knowing the program, it's impossible to tell you how to do a page break (they're all different).  Neither Notepad nor Wordpad support the pagebreak function (CTRL-Enter as you say for Microsot Word {today's version as well} and some others though I was not aware that this worked in Windows98-supplied "word processing" programs any more than it does today).  Perhaps you were using a different program then (like Word or WordPerfect or Works or something like that).

If you do not have Microsoft Office (hence don't have Microsoft Word), then I would strongly suggest you download the totally free OpenOffice at www.openoffice.org which mirrors much of Office's primary functionality (except for Outlook) and has a program called Writer which is an excellent word processing program - better by far than anything that comes with Vista.  It can even open and work with Microsoft Office documents (even those from 2007 and I believe the beta from 2010).  It will certainly have a page break feature (as well as MANY other features that only come with the expensive office suites (but again, it is free and despite that it is excellent).  If you don't have Microsoft Office or another purchased offfice suite, then I STRONGLY recommend you download and use this set of programs instead of what comes with Vista (though there are times when all you really need is a cheap option like Notepad - and Notepad is also good for making batch script files if you're into that kind of thing).  But if you're doing actual Word Processing, you should definitely switch.  If you switch, try CTRL-Enter for a page break and if that doesn't work then check page break in help to get the proper keyboard command - but I'll bet it's the same.

Don't be too upset that this means you need a 3rd party program to "do something with Vista."  OpenOffice is so much better that it isn't even worth a comparison.  The truth is that the products that come with Vista are just not as good as some of those available on the open market (either for a price or sometimes for free).  I happen to like Vista's Backup and Restore Center (but it's really only good in the more expensive versions of Vista (Business and above) - and yet there's a better 3rd party product out there for this as well (Acronis: http://www.acronis.com/homecomputing/products/trueimage/ but it is not free).

There are some Vista programs that can be valuable.  Some people use Microsoft Mail and Microsoft Contacts as their email client (and contact client) - though many who start this way switch to Microsoft Live Mail (it's a little better than Microsoft Mail) and of course, if you have Microsoft Office, you would almost certainly switch to Outlook (which is MUCH better but it costs money as Office is not free).  As a free alternative, many use Mozilla Thunderbird http://www.mozillamessaging.com/en-US/thunderbird/ as, again, it is a better product with more features and options (and it's free unlike Outlook).  The Microsoft Search program in Vista is very good if you take the time to learn how to use it properly and needs no replacement.

Remember that none of these other programs would function without Windows Vista (or some other OS) behind it - so it does serve an important purpose.  You wouldn't be able to post to this website without Vista and a browser.  If you're using Internet Explorer it's sort of like using Vista because version 7 comes bundled with Vista and cannot be added or removed (it's part of the basic OS) - though you're probably using IE8 which was the upgrade (and worth upgrading).

I hope this helps.

Good luck!
