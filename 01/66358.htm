soundersfan13 say:

VX-1000, VX-3000, and VX-6000 cameras use a custom video streaming driver.  You can see this in the device details from Device Manager.  Each of these device has a .sys file indicating that they have a custom kernel driver. In Windows 8, Store apps are not permitted to use webcams with custom kernel drivers.  For this reason these cameras can only be supported in desktop apps on Windows 8.  Go to the following link and search for LifeCam or your particular model to find which models are Windows 8 compatible.

 

http://www.microsoft.com/en-us/windows/compatibility/win8/CompatCenter/Home
