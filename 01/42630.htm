If you’re having trouble getting connected to a specific website or server, we suggest that you run our Network Troubleshooter. The troubleshooter can diagnose and fix common connection problems. To run the Network Troubleshooter:

  • Select the Start  button.
  • Then type Network problems in the search box.
  • Select Identify and repair network problems in the list.

If the issue still occurs after running the Network troubleshooter, run these networking commands in command prompt window:

  1. Select the Start  button, start typing cmd, right-click Command Prompt in the list, select Run as Administrator, and then select Yes.
  2. At the command prompt, run the following commands in the listed order and then check to see if that fixes your connection problem
  • Type netsh winsock reset and press Enter.
  • Type netsh int ip reset and press Enter.

  • Type ipconfig /flushdns and press Enter.
