Options for freeing up space:
Windows over time will accumulate data outside of the normal types of data we store. This can include things like copies of Windows Updates, memory dump files used to diagnose and troubleshoot error messages, temporary internet files for web pages we often visit and Windows Upgrade files, commonly used to assist with installing updates for Windows. Sometimes we don't need this type of information and removing it can free up precious disk space inaddition to improving overall system performace.

azeez n say: Hi,

A number of "well-known" security identifiers are defined by the operating system so as to ensure that specific system accounts can always be found. Microsoft maintains a complete list of these identifiers in a knowledge base article. This is absolutely normal and the numbers that you see (S-1-5-20) are known as Security Identifier(SID).

Windows grants or denies access and privileges to resources based on access control lists (ACLs), which use SIDs to uniquely identify users and their group memberships. When a user logs into a computer, an access token  is generated that contains user and group SIDs and user privilege level. When a user requests access to a resource, the access token is checked against the ACL to permit or deny particular action on a particular object.

SIDs are useful for troubleshooting issues with security audits, Windows server and domain migrations.

earobinsondo say:

My Acer Aspire M had the same problem.  I tried all the software fixes and nothing worked.  Finally, I found that it was a hardware problem.  The wire connecting the touch sensor to the screen was disconnected.  After days of troubleshooting it was fixed in under 2 minutes.  Turns out the little piece of tape designed to help secure the ribbon strip to the sensor wasn't taped down so it worked itself loose.  As a matter of fact the backing to the tape had never been removed.  I pushed the ribbon strip into the circuit board, removed the tape backing, taped in down to the chip as it was designed and presto, screen is working!

Use your fingernails and a credit card to pop the back off the LED screen off.  In the center of the bottom edge on the back of the screen there is a tiny wire ribbon strip that plugs into a tiny circuit board.  Push it in well.  Chances are the tape backing is still on the securing tape. Remove it and stick it down.  Restart the computer carefully with the back off to see if the screen is now working.  If not working try again.  I can email you pics if needed. Here is a video I made showing how to fix it.  youtu.be/u3DxD23u1c4
