I suggest downloading Ver 1703 directly rather than through Windows update and updating to it using a procedure called a Repair install [also known as an In-Place upgrade].  Its name is a bit misleading, it is not an installation as such [your applications, user accounts & your own files are not overwritten].

2  Boot your computer normally [not from the installation disk]. 

3  Backup your own files and, if you use them, create a new system image.  I have never had a Repair install go wrong on me but I still always take this precaution.

4  Disconnect any unnecessary hardware such as external drives just to keep things simple.

5  Run though the Repair install procedure - Repair Install Windows 10 with an In-place Upgrade - TenForums

6  Uninstall the Media creation tool that you installed in step 1.  It is of no further use to you.

a. user say:

Knowing the specific drivers is only half the tale.  Why are you looking for these drivers?  Are you having problems with your parallel port and/or USB ports?  This could be caused by driver problems or it could be a hardware issue (hard to diagnose a hardware issue without good parts to swap out and test).  If you need to update the drivers, you need to use Device Manager (not msinfo32) - or perhaps clicking on an executable file which loads the drivers automatically (it depends on how it is sent via download).

Go to Device Manager by going to start / search box and type device manager and enter and then double-click on the program icon that appears.  Check each device for a red x, yellow ! or white ?.  These identify devices with problems (probably drivers, but also conflicts or something else).  Click on each for further details and troubleshooting tips.  If you need to get drivers (I assume you will for the parallel port and USB ports), do so from the computer vendor or device manufacturer (NOT from Microsoft Updates).  In fact, you should turn off automatic driver updates from Windows Update as follows: http://www.addictivetips.com/windows-tips/how-to-disable-automatic-driver-installation-in-windows-vista/. Proceed as follows to get the drivers: http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm.  Once you have the drivers, you can install them through Device Manager as follows: http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html.  Or execute the driver installation program provided by the computer vendor or device manufacturer.  Be VERY careful to get the correct Vista-compatible drivers as these devices are hard to identify (they usually don't come with model numbers or anything like that though they may have serial numbers) and using the wrong drivers could make the problem (whatever it is) worse.

If you update the drivers succerssfully and still experience the problem, then it is probably a hardware issue.  In that case either contact your computer manufacturer for technical support (hopeully it's still under warranty) or bring it to a professional computer repair technician (NOT Geek Squad or any of those big store centers as they don't really know what they're doing and charge a lot for that level of expertise) to have the components tested and possibly replaced.

I hlpe this helps.
