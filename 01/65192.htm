lemp say:
System Restore saves as many restore points as it has room for.  When the allocated space fills up, older restore points are deleted to make room for new ones.

Although SR in Windows 7 is far better than it was in XP, I would still hesitate about going back much more than a week.  Windows 7 does warn you: you'll notice on the first System Restore screen a link that says "Scan for affected programs."  Click this to get an idea of what problems might arise if you go ahead with the restore.

Also, the list of restore points that shows initially may not be complete.  Check the box to "Show more restore points" to see the rest of them.

Here's how to adjust the space allocated for System Restore (all versions of Win 7): http://windows.microsoft.com/en-us/windows7/how-much-disk-space-does-system-restore-require

Here are answers to some common questions about System Restore.

Remember, System Restore saves its restore points on the same disk that it is "protecting." If that disk fails, both your system AND your backup are gone.  Best practice would be to save backups on external media. 

If you want to stay with the built-in tools, see, for example:
