praseetha k say:

Hi gregglee,

Thank you for choosing Windows 7 Answers Forums.

First of all try to uninstall and reinstall the Network adapter drivers.

Uninstall the drivers from the Device manager

1. Click start

2. Type devmgmt.msc in Start Search

3. In Network Adapter you will find the device, right click on it and select the option uninstall.

4. Restart the computer and check.

Uninstall the drivers from programs and features

1. Click start

2. Type appwiz.cpl in Start Search

3. Search for the Network driver package and uninstall.

4. Restart the computer and check.

You may also try to install the latest version of the drivers for Network Adapter from the manufacturer’s website.

For more information, you may refer the following link for troubleshooting Network Adapter issues.

http://windows.microsoft.com/en-US/windows7/How-do-I-fix-network-adapter-problems

You may also try the steps provided to perform Power Cycle.

1) Turn off the power to the modem (disconnect from power source). If there is Router turn it off (disconnect from power source).

2) Disconnect the Ethernet Cable that runs from the modem to the back of the computer or the Router to the back of the computer. Disconnect at the computer's end. If there is a router follow the same steps, disconnect at the computer's end.

3) Shut the computer down (Do not restart).

4) Wait for at least 4 minutes.

5) Turn the power on to the modem. If there is a router turn the router on as well. Wait for the lights on the modem\router to stabilize.

6) Connect the Ethernet Cable back to the computer.

7) Turn the computer on. Now try to connect to the internet.

Hope this information is helpful.

Microsoft Answers Support Engineer

                           net stop cryptSvc

                           net stop bits

                           net stop msiserver

3. Now rename the SoftwareDistribution and Catroot2 folder. You can do this by typing the following commands in the Command Prompt. Press the “ENTER” key after you type each command.

                          ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

                          ren C:\Windows\System32\catroot2 Catroot2.old

4. Now, let’s restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt for this. Press the ENTER key after you type each command.

                        net start wuauserv

                        net start cryptSvc

                        net start bits

                        net start msiserver

5. Type Exit in the Command Prompt to close it and then restart the computer.

6.After restarting the computer, now retry to install Windows Updates.
