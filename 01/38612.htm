Windows Photo Gallery has a proprietary data base file that
can become corrupted and may need to be renamed. This
will force the gallery to generate a new database file on the
next launch.

Depending on your version of Live Photo Gallery...search for
Pictures.pd4  Pictures.pd5  or  Pictures.pd6

Rename  Pictures.pd4, Pictures.pd5 or Pictures.pd6 
to Pictures OLD

The next time the gallery is launched it will create
a new database file and the gallery will appear as
it did the first time you ever opened it.

The gallery will need to re-populate and some tags
and metadata may be lost.

If you are unhappy with the results...you can delete
the new database file. If you rename the original back
to Pictures.pd4, pd5, or pd6 your gallery should be as
it was before the change.

Pictures.pd4, Pictures.pd5 or Pictures.pd6 database location:

C:\Documents and Settings\>user name<\Local Settings\
Application Data\Microsoft\Windows Live Photo Gallery\
Pictures.pd4, Pictures.pd5 or Pictures.pd6
