Hi

The "Always show icons, never thumbnails" option is a global setting. If this is selected, it will affect all folders, including the Libraries.

The Optimize option can define the view that you initially see in a folder, such as the General Items, Documents, etc.

 

  • You can disable the Always show icons, never thumbnails option.
  • Set one of the folders to optimized for Documents or General Items.
  • Set the View Menu to any of the non-icon options (ie: List, Details, Tiles, Content).
  • Click Organize/Folder and Search Options/View Tab.
  • Click the Apply to folders Button.
  • This will set all folders that are not optimized for Pictures to the selected view.

 

Only the folders where you specifically select Optimize for Pictures and one of the Icon Views will show thumbnails.
