Hi Everyone!

So i've been having this issue with multiple applications: Atom, Slack, and Skype

It had been getting really annoying to have these pop us happen multiple times in a single session. So, this is what worked for me and hopefully works for everyone else having this issue. 

If you are familiar with terminal this will be a piece of cake, first you want to navigate to your applications directory using the following command on terminal:

cd /Applications/

next you want to list your applications along with the permissions and other information using this:

ls -lah

Now find your application, for instance look for Skype.app.

drwxr-xr-x   3 root   wheel   102B Feb  2 21:40 Skype.app

now I ran the following command:

sudo chown -R $USER:wheel /Applications/Skype.app

After this I stopped receiving the annoying pop-ups
