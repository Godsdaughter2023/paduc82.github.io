shishira d say:

Hi Steve27597,

Welcome to Microsoft Answers Forums.

Please follow the troubleshooting steps below which would help resolve your issue.

Method 1: -

As a first step perform clean boot in safe mode with networking where you are presently working on and restart your system to normal and check if you are able to logon to windows normally.

Clean boot is software configuration utility tool, which starts your computer by disabling startup software programs and some services to determine if they are causing the issue.

 

Please follow the steps below to perform clean boot.

1.     Click Start, type msconfig in the Start Search box, and then press ENTER.

2.     If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

3.     On the General tab, click Selective Startup.

4.     Under Selective Startup, click to clear the Load Startup Items check box.

5.     Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

6.     Click OK.

7.     When you are prompted, click Restart.

8.     After the computer starts, check if the issue is resolved.

9.     If the issue is resolved, you can undo changes done using clean boot.

 

To undo the changes and start your computer normally, please see the steps described below,

1.     Click Start, type msconfig.exe in the Start Search box, and then press ENTER.

2.     On the General tab, click the Normal Startup option, and then click OK.

3.     When you are prompted to restart the computer, click Restart.

For troubleshooting what caused this issue to occur, please follow the troubleshooting steps using clean boot described in the link below,

http://support.microsoft.com/default.aspx/kb/929135

Method 2: -

If the above method fails to resolve your issue I would suggest you to start your computer in normal mode and open outlook in “Outlook Safe mode” and see if you are still experiencing the same issue.

Please follow the steps below to open Outlook in Safe Mode: -

1.     Click on Start Orbit and type Run.

2.     In “Run” window, type “Outlook /safe” without quotes and click on “OK”.

3.     Now, your Outlook 2003 will open in Safe Mode.

4.     Check if now you are able to work with your Outlook client normally.

If yes, probably an add-in installed in Outlook is causing this issue. Please see the link below for instructions to “Add or remove an add-in” in Outlook 2003.

·          Add or remove an add-in,

http://office.microsoft.com/en-us/outlook/HP052427821033.aspx

Note: - The link demonstrates how to remove a COM add-in, you can also try to disable other add-ins and check if issue is still persisting.

Alternatively, you can also visit Office Discussion Groups for more assistance from the link below,

·          http://www.microsoft.com/office/community/en-us/FlyoutOverview.mspx

 

Hope this information helped you.
