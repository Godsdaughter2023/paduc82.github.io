darrell,

I  guess you must be referring to Windows Search which runs as a service. You can't delete it, but you can disable it by doing the following.

  1. Right click the Start button and go to Run.
  2. Type: services.msc and click OK.
  3. Click anywhere and then hit the letter "W" to jump to the list beginning with that letter (saves all that scrolling) and then double click Windows Search.
  4. Change the Startup Type to "Disabled".
  5. Apply/OK.

Windows will no longer respond to search requests afterwards.
