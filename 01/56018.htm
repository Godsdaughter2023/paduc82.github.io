Hi Victor,

Thank you for writing to Microsoft Community Forums.

Gaming laptops are made for intense performance when compared to normal laptops. They have specially designed cooling system and high speed fans for better performance. Gaming laptops have high end dedicated graphics card but normal laptops have integrated graphics. Few laptops come with two graphics card to enhance the gaming experience.

Gamers need bigger screen size and high-quality resolution in their laptops and for a normal laptop user, the balance between size, weight, and resolution is necessary. Gaming laptops have high power consuming processors hence they have less battery life.

If you want to use laptop for work purposes like accessing internet, MS Office, and photo editing, you can buy a normal notebook which has battery life. If you are into gaming, you can go for a gaming laptop considering that they are heavy and have less battery life.

hairyfool say:

It depends on what backup utility you are using because Windows 10 does not in principle have a "backup" utility. Microsoft have retrograde backup per se and switched to the use of the Microsoft Account along with File History.

File History is a continuous timed backup process archiving file changes at intervals you can define. It allows point in time restores of your files. With a Microsoft Account individual settings are stored in your OneDrive profile. After a clean install of Windows 10 logging back in with the same MA credentials restores those settings.

There is a System Image utility primarily to bring backward compatibility with Windows 7 but that puts the entire file set into a compressed archive. This precludes any simple method of adding new files to the archive.

You say once a week 104 disks per year which would indicate 2 disks per backup, therefore one disk must be full anyway, are you expecting to erase disks?

In reality due to the large data sets DVD rarely represent a viable backup solution other that to archive specific file sets. If looking to make regular system backups then you should be looking at external USB3 HDD drives, if you require off site storage then 2 or more rotated. If small enough then consider on line storage as then you don't have to worry about backup.

The unfortunate fact is that domestic backup solutions have not really kept pace with the large drive sizes, a major reason why Microsoft have moved away from drive level backups. The 1st thing is to review what you want from backup as a whole.

hairyfool say:

A System Image is a complete backup of all files on the selected drive so unless there is an issue between backups the second then makes the first redundant. So yes you could erase the older backup

The principle of the system image is that should the drive fail all you need is a new disk and you can restore your system as it was, sometimes described as a "bare metal" recovery.

There are two caveats. 1st is that the System Image file is not bootable, you must create what is called a Recovery disk. This contains the bootable image needed for repairs or system recovery. It can then find the system image file to restore from.

2nd is backup is always a 2 stage process. Lots of people that do backups never test to see if the backup works, waiting to find out of your process is correct until you actually need the backup exposes you to a risk.

I have been doing this job for 30 years and even I do not do "backups" as a routine. I have a Microsoft Account so have OneDrive which also holds copies of a few "life critical" files, I have File History configured to write out to a second hard disk (not a second partition) and sporadically (when I remember) copy my Libraries to an external drive. Since the advent of crypto viruses this disk does not remain connected to my system.

You can click the links to see the difference. 

Clean Install Windows 10 is the gold standard install that wipes the drive and reinstalls Windows 10 in such a way that it remains a great install for as long as you stick with the steps, tools and methods given.  There are zero complaints. 

The Windows 10 Refresh provides an automated alternative that also reinstalls Windows and will also remove and not replace the corrupting factory bloatware (which some Resets include).  The difference is that it doesn't clear the hard drive first to get it cleanest, and it doesn't apply the optimizations that make the Clean Install the best by far.  It is however now built into Windows Defender settings so can be launched from there in addition to the download in the link. 

We had been promised an automated way to shed the factory bloatware in Windows 10, but since many manufacturers customized the Reset Your Windows 10 PC to reinstall the factory lard, this was instead automated in the Refresh and now in Windows Defender's tools as well.  Besides this the difference between the Refresh and Reset is that Refresh is run from a download to Windows or Defender settings, while the Reset can be run from both Settings or from booted media in case Windows won't start.
