Hello Christi,

 

This error can happen if the Component Based Servicing (CBS) manifest has any corruption, (CBS is the application used to install some updates). You can fix this by running the System Update Readiness Tool (Checksur), To install Checksur you can launch Windows Update, just do this:

 

1.       Click Start, click Control Panel, and then click Security.

2.       Under Windows Update, click Check for updates.

 

The update is offered on Windows Update to computers that have a condition that the tool can resolve. If you don’t see the update offered you can click the links in the following list to download the tool from the Microsoft Download Center, just click the link for your version of Vista.

 

Note: The download works best if you download the file to the desktop or other folder on your computer and then double-click the file to run it.


Windows Server 2008 and Windows Vista

Download the System Update Readiness Tool (KB947821) [September 2009] package now.


Itanium-based versions of Windows Server 2008 and Windows Vista

Download the System Update Readiness Tool for Itanium-based Systems (KB947821) [September 2009] package now.

64-bit versions of Windows Server 2008 and Windows Vista

Download the System Update Readiness Tool for x64-based Systems (KB947821)[September 2009] package now.

 

Once you install the update, reboot the computer and then go right to the Windows Vista SP2 update and install it. (Be sure to close any programs that start up with the computer).
