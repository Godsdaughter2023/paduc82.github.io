aileen alf say:

Hi Richard,

Poor Internet connection is one of the reasons why you're experiencing the issue with Windows Spotlight. If in spite of having a strong Internet connection, you continue to face the problem, you may want to reset Windows Spotlight preferences and settings, and see if that helps. We recommend creating a system restore point first before performing the following steps:

  1. Delete the Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy folder.
  2. Open the Start menu, and search for PowerShell.
  3. Copy and paste the following command into the PowerShell prompt, then press Enter:

Get-AppXPackage -AllUsers -Name Microsoft.Windows.ContentDeliveryManager | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml" -Verbose}

Let us know how it turns out.
