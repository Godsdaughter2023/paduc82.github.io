Is that Taskbar set to Auto-Hide, but not doing so sometimes? There is a bug that I have not completely figured out, but some programs make the Taskbar "stick". This is a temporary workaround. Keep notes on what programs seem to cause this action.

 

1. Wait for any copying, deleting, renaming etcetera of files, to finish.

3. Choose <Start Task Manager>.
4. Select the 'Processes' tab.
5. Find 'explorer' and end that process. The Taskbar disappears. (This will also close any Windows Explorer windows, some error messages and the copying, deleting, renaming etcetera of files that may be running.)
6. Select the 'Applications' tab.
7. Click <New task...> 
8. Type [explorer] without the brackets.
9. Click <OK>

frederik long say:

Assuming that you store your data (pictures, music etc) on a partition other than the System partition, 100 GBytes is adequate. I've been doing this for many years and my Windows 10 System partition now uses 50 GBytes. 20 GBytes is nowwhere enough and 200 GBytes is far too much.

There are at least two methods to get your files back in case Windows gets corrupted:

  • From your external backup medium. Remember - important files must at all times be stored on two independent media. The data partition on your internal disk does not qualify as an independent medium.
  • By booting the machine with a bootable Windows Repair flash disk and using console commands to copy the data to an external disk.

You may also check if network discovery and file and printer sharing are enabled on the computers that you are trying to access.

To enable network discovery:

1. Press 'Windows key + I' together to open Settings.

2. Click on 'Network & Internet'.

3. In the panel on the left, click either Wi-Fi(if you are connected to a wireless network) or Ethernet(if you are connected to network using a network cable).

4. Click 'Change Advanced Sharing Settings'.

5. Expand the network profile.

6. Select 'Turn on network discovery'. Also, check the box that says 'Turn on automatic setup of network connected devices'.

7. Click on 'Save changes'.

To enable file and printer sharing:

1. Press 'Windows key + I' together to open Settings.

2. Click on 'Network & Internet'.

3. In the panel on the left, click either Wi-Fi(if you are connected to a wireless network) or Ethernet(if you are connected to network using a network cable).

4. Click 'Change Advanced Sharing Settings'.

5. Expand the network profile.

6. In the file and printer sharing section, select 'Turn on file and printer sharing'.

7. Click on 'Save changes'.
