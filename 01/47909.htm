Why not right-click the job itself in the Task Scheduler, then stop it?

There are several methods to stop the Task Scheduler service this. Here is one of them:
  1. Create a scheduled task called "StopScheduler" as follows:
    - Trigger: At boot time
    - Command: sc.exe
    - Parameters: stop schedule
    - User: System
    - Run with the highest privileges.
  2. Right-click the StopScheduler task, then left-click to run it.
  3. Fix your problem task.
  4. Restart the Task Scheduler.
  5. Disable or delete the StopScheduler task.

cmdrkeene say:

tyhudson say: The "do you like what you see" option does not display in upper right hand corner of page.  Have  the Settings and personalization  options set.  Chose Windows Spotlight under the Lock Screen menu.  The feature does not work.

tyhudson say: Windows spotlight only works after I log in and then hit windows key and L to lock screen,  After logging off and computer restarts it displays old picture and spotlight not on.

cmdrkeene say:

That's how it is supposed to work.  If you haven't actually logged in yet, then no user profile is loaded into memory so it hasn't actually recalled which spotlight/pictures to show yet.  Only after it pulls your profile from the registry into active memory can it actually use that information.  Until you've logged in, your data is locked away and it can only show a basic picture.

Most people don't do full shutdown/restarts, they just use sleep mode (like a phone or ipad... no need to truly shutdown these days), which if you did that, you'd hardly notice.  But if you do decide to do a full shutdown/restart/log out, then yes: it will do exactly as you showed, because your profile and settings are no longer loaded.
