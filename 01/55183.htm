        The file shredders (Eraser, CCleaner, etc.) are only good if you know everything you want erased off the machine.  In the case where you are getting rid of the machine and want to be sure there is no trace of you left on it - you need to use something like DBAN.

        So, quick tutorial.

        First - stop overthinking.  You started mentioning DOS and such when DBAN is self-contained, all in one, everything you need in one package if you get the ISO file, burn that CD image to a CD/DVD, boot with it and go through the dead-simple menus.  Then you know everything is gone.

        DBAN essentially writes zeros to every part of your drive multiple times.  It makes sure *nothing* you had on there exists anymore (Windows and all applications included.)

        It comes in an ISO format, or an image of a CD that - in this case - needs to be changed from that image to an actual CD again.

        DBAN's own FAQ page tells you how to create a true bootable CD using some free software and their download:
        http://www.dban.org/faq/burning

        In general, you get some utility that can burn ISO images onto CDs (they recommend CDBurnerXP, I like ISORecorder) and you download the ISO file from DBAN: https://sourceforge.net/projects/dban/files/dban/dban-2.2.6/dban-2.2.6_i586.iso/download and save it and then use a blank CD-R, your CD/DVD writer and the program you chose to change that ISO image into a real CD per the instruction for that program.  Usually nothing more than a click or two or even a right-click menu choice on the ISO file.

        Once you have the CD created, you boot with it.  In order to do this with your Dell, it is easy.  Put the CD in the drive and close it.  Turn OFF the computer.  Turn on the computer and immediately start tapping the "F12" key once per second - nice and steady and slowly - until a menu comes up asking you what you want to boot from.  Choose your CD/DVD drive and watch it boot.  Choose the proper choices from the menus.  Either of the "DoD" methods should be more than sufficient and choose whatever drives it lists to make sure it gets them all.  Then away you go and you wait.  When it is done, you aren't likely getting anything off that drive.
