bossdweeb say:

Yes, do install 64 bit Windows. This will not affect the vast majority of 32bit programs or games.

As time goes by a lot of programs are getting 64bit versions. In that instance installing the 64bit version when running

64bit Windows is advisable, and in a few instances required by Win10 ( mostly drivers, security software and I believe browsers).

The main exception to 32bit software compatibility is with older software that uses a 16bit installer. 16bit software does not run

on 64bit Windows.

- If you have a lot of software to reinstall it can take a long time getting a clean install of Windows setup, but there is no option

if you want to use the RX480. You cannot simply upgrade from 32 to 64bit Windows, unfortunately.

- If your machine has 4GB or more RAM you need 64bit Windows. 32 bit Windows can only access around 2.75GB RAM.


"I would need to install eg advanced system care..."

No you won't. Because these optimising programs do nothing you can't do yourself in Windows Settings, and a bit of manual

house cleaning, and for the most part Windows does what needs doing without any help.

Running Windows Disk Cleanup occasionally can be useful, but even that can generally be left to cleaning up after you have

done a Win10 version update if you need to free up some drive space. I usually run Disk Cleanup after I have verified the version update

has not borked something, but it's not really necessary. I'm just a bit OCD. ;)

The version updates create backups that can be quite large, but Windows deletes most of these files after a period of time.

I believe it's ten days currently.

I tend to run a disk defrag on my HDD after installing something big like a game as well.


Using any Registry 'cleaner' that is packaged with system 'optimising' software is not advisable. While many people do get away

with running registry cleaners without creating problems (they surface here a couple of times a year), they can bork your Windows install,

and for nothing.  'Cleaning' the Registry does not improve Windows performance.
