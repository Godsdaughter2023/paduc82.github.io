  1. Are you able to access your computer completely?

  2.  You are the administrator of the computer/ the local user?

  3. Is that trying to access the file and folders through the Microsoft account?

Method1:  If you concerned about file permissions?

  • Right click on the folder which you are not able to access and select the Properties of the main user profile folder where the files are located (such as Documents, Pictures, etc.).

  • Go to the Security tab and click Advanced.

  • Check the box at the bottom of this window.

  • It is labeled "Replace all child object permissions with inheritable permissions from this object" and then click Ok.

Method2:  I suggest to follow the below mentioned steps and check with the login user account/ administrator account.

  • Click on Add button, create a new local user account and follow the on-screen instruction to create a new local user account.

  • Click on new user account, Select Properties, and then the Group Membership tab.

  • Select the Administrator tab, then click OK twice.
