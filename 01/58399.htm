Did some testing

Restore windows folder at log on is one thing and separate from what I am listing below. That will only restore folders to auto open.

Ok Windows 10 Fast Start, what it does it preserves all open apps, docs and browsers on a restart.

when you restart PC, and log back on it actually already has all your start up programs, that usually run in taskbar system tray loaded when you log back on. Also browsers will reopen and Docs and most programs will auto reopen.

So here is the catch and fix for it.

When you log off and log back on, all those startup programs have to load one by one such as in taskbar system tray.

Also, any browser, programs and doc will now not auto load on log on.

So instead of selecting restart from desktop, log off and then select restart from the lock/sign in screen.

Then if you restart or just log back in, all those programs, browsers and docs will not auto load.
