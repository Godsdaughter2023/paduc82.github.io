      The only safe way to rid yourself of an MBR infection is by rewriting the MBR prior to installation. Some installations will do this and some won't. Apologies but I do not have a definitive list of either.

       

      Something like DBAN or Killdisk will do this. But here is a big warning...

       

      Some PCs come with an MBR that is non-standard. This is so that you can pre-empt the Windows boot process and divert to (for example) Windows recovery or PC diagnostics. DBAN and Killdisk will wipe not only the custom MBR but also all partitions on the hard drive including (but not limited to)...

      • The main bootable operating system
      • Any diagnostics partition
      • Any operating system recovery partition

      Therefore, when using something like DBAN or Killdisk, you must have installation media available to restore the operating system. These recovery media may or may not restore the deleted 'special' partitions and any associated custom MBR.

       

      A genuine Microsoft XP installation CD will write a standard MBR and re-install the operating system only. Other media may or may not do more than this.
