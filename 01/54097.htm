cmdrkeene say:
I don't know of a way to hide it, but I do have at least a somewhat helpful offer.

  • Option 1:  Group by Offline Availability Status
    Right-click a blank spot in the folder, choose Group By and then More...  You can choose Offline Status as the grouping option.  When you click OK, files will be in a couple groups depending on their availability, which you can collapse with ▼ and ▲ buttons to hide the ones you don't care to see at the time.

    or (probably better)...

  • Option 2:  Sort/filter by Offline Availability Status
    Go to Details column view (where you can see the details/type/size of every file).  Right-click one of the headings and choose "Choose Columns..." to add the Offline Status as a details column.  You can then click the ▼ arrow next to that heading and uncheck the "Not Available" to hide any files that aren't available.

    Super Bonus:  When you filter your view like this, you'll notice that you're no longer "in" your folder: you're in a special kind of live view that is basically showing you real-time search results.   So up in your address bar, you can drag the folder icon down into your favorites or out to your desktop (or to your taskbar) for a quick shortcut that always opens this folder and then only shows files that are available.  I use this to create virtual folders all the time.  There's a big network folder dump on our network that everyone uses, and I have a filter view like this setup to only show files of a certain type that I care about.

How to Find SSID (Network Name) - Microsoft Community

xb1r apollo say:

How to Find SSID (Network Name)

Login Information

 

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 

 

How to Find SSID (Network Name)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter in "root" for the user name and "12345" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Basic" from the sub menu.
  6. The wireless network name will be labeled as "SSID".
