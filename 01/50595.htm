¡firedog say:

Server Error: 0x800CCC()

If you copied and pasted the error message, something very strange is happening, because that isn't a Windows Live Mail error code ...

Server Response: -ERR [SYS/TEMP] problem retrieving message

Windows Live Mail Error ID: 0x800420CD

These two, though, mean essentially the same thing. Because there is a message that Windows Live Mail can't or won't download for some reason, the POP3 session doesn't complete and the delete command isn't implemented to be on the safe side.

Yahoo has a unique way of handling junk mail; it's all kept in the inbox, but marked such that it's not downloaded by POP3 unless the mailbox is configured otherwise. To clear this fault, then, sign into the account in webmail and empty the folders for deleted items and junk mail, then try send/receive again in Windows Live Mail.
