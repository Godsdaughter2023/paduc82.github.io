¡firedog say:

@John: Those instructions will set up a duplicate account, which can cause problems. There will probably be conflicts when the two accounts poll for new messages, and the entire contents of the inbox will be downloaded again to the new account. This could be a large amount of data transferred for no reason. The user might be tempted to remove the old account without realizing that all its folders and their contents would be permanently deleted.

Comcast used to have a page explaining how to edit account details in Windows Live Mail, but they've since taken it down. This is why I gave instructions for amending the existing account to correct the fault that the OP reported. Either this will work, or if it doesn't, we'll see a new error message indicating a different fault to be corrected. Adding the account again is really the last resort.
