arcti say: I have an Asus A55A Laptop w/o built-in Bluetooth running Win 8.1. I'm able to get pictures from my LG840g to the laptop  by doing the following:

1. Pair the devices. When pairing on my computer,  they show they are connected briefly, then they will disconnect.

2. On the task bar, right mouse over the Bluetooth icon and select "Receive a File". You should get a window that says " Bluetooth file Transfer" "Waiting for a connection"

3. On the LG840g, open the picture file you want to send, then select "Send via" then "Bluetooth", then select your computer. The file transfer should begin.

This works in a similar way with v.cards.

If you want to send a file to the LG840g from the computer, right mouse over the file you want to send, select "Send to"> "Bluetooth Device" then select the device.
