mel77084 say:

If we understand correctly, you are saying you installed Windows on the new drive in another computer and then put the drive in place of the old drive in your computer, the toughest problem is the difference in drivers required by the hardware. As Lawrence mentioned, that kind of switch can work between two computers of the same make and model, and in some cases, given enough time, Windows Update can work in the background to find and install the correct drivers needed, so after a while, it can finish booting.

That's one of the main uses for a disk image backup of the operating system partition. To go that route, you need to return the drive that you felt needed to be replaced by a larger one to the computer, run the disk cleaner provided in Windows, and if you have any significant amount of data, such as pictures, videos, music, etc., move that out of the operating system partition into a data partition, which you might or might not have, and therefore, might need to make for yourself.

(Good computer and data management requires that data not be kept in the operating system partition for several reasons.)

Then, download and install one of the free-for -personal-use disk imaging programs. One of them that does the job nicely is Macrium Reflect. After installing it, use it to make a bootable rescue medium, and then make the backup image following the program's directions, and have the program store it on different drive (external or internal) that the computer will be able to see (access).

Now, you remove the old drive and replace it with the new one, and boot to the bootable media that you made with the imaging program. (You might need to change a bios setting to allow booting to the CD or the flash drive.) It will require a minute or so to load the rescue program, and for most of that time, the screen will show nothing. Once it is loaded, you will see the image that you made, and you select the drive onto which you want the image restored. It will warn that whatever might have been on the drive will be destroyed, and you click ok.

In 10 or so minutes, it restore the operating system partition that it imaged from the old drive, the exact same size, containing your operating system and all the installed programs and drivers that were on your old drive.

Now, be sure to have removed the bootable rescue media, reboot the computer. It will boot looking exactly as it did with old drive.

Depending on how many games, etc, are installed, an operating partition won't need to be larger than 80 to 120 GB for proper operation with Windows 10.

Some users say, "I've got a xx TB drive, so I don't need any partitions. I just throw it all in there together." Then, oftentimes, they post that they've lost all their data.

A note on Macrium Reflect: Sometimes new users of it find it to be a little overwhelming, but it's well worth the learning curve to be able to recover in a few minutes from a hard drive failure.

Another good one that seems easier for some new users is AOMEI Technologies Backupper.

Hi eo1966,

 

Problems that cause the 0×80070052 error include:

 

  • The flash media has no sufficient space for your files
  • The flash media has too many folders already on it
  • There is a registry error causing the problem
  • You don’t have the latest updates for Vista

 

Step 1 – Update Vista

 

Microsoft has an update that allows you to quickly work with the various SD cards and USB drives. In case you’ve not applied the update, get it done.

 

Here’s the update:Availability of the Windows Vista Secure Digital (SD) rollup package

Step 2 – Consider Trying To Use A Different SD Card / USB Stick

 

Unfortunately, several SD cards or USB sticks do not support a lot of folders on them. This means that if you try and copy too many directories to the device, it will just show the 0×80070052 error and refuse to copy the files. You should either think about removing some of the folders from your flash media device or using a different one.

 

Step 3 – Clean Out Viruses On Your Computer

 

Using an antivirus program is a great way to fix common errors, such as the 0×80070052 error, because it quickly removes all the infections that your computer has inside.

 

If you have antivirus installed on your computer, update the virus definition of it and scan the computer. If you don’t have one, you may download Microsoft Security Essentials on your computer and run it.
