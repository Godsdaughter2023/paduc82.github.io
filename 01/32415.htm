Hi Alice

I wasn't able to find any information for that HC44 model on the Windows 7 Compatibility Center or the Sony support websites?

When you see the This application has requested the Runtime to terminate in an unusual way error message, this can happen for a couple of reasons. When an application is developed, the program that the developers use will have a particular version of the Runtime files associated with it.

The associated Runtime files are what allow the program to start and run (Runtime) when it is used.

If an older program is being used, the correct runtime library may not be available, for many different reasons. It could also be that the program is not operating with a high enough privilege level.

Here are some things you can do to fix the problem.

1. When you start the Sony utility program for the camcorder, right click the shortcut that you normally use and select the Run as Administrator option.

2. Uninstall the program and then Re-install the program in Compatibility Mode.

Right click the installation file and select Properties / Compatibility Tab.

Place a check mark in the Run this program in compatibility mode for: option.

In the drop down menu, select the Windows XP (Service Pack 3) option.

Click Apply / OK.

Exit the Properties dialog.

Right click the installation file and select the Run as Administrator option, to install.

If neither of these options resolve the problem, I would recommend contacting Sony support for help.

Check the Sony support forums first. There may be some users there who may have experienced and solved this same problem.

Step 1:

c.    Right Click on the Webcam.

e.   Click on Driver tab.

f.     click Update Drivers and lick OK.

g.   Now restart the computer and check.
