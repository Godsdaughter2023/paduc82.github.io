Aha! The cure is somewhat tedious:

  1. Ask a friend to burn a Windows Repair CD on her Windows 7 PC. It's done via the Control Panel / Backup and Restore.
  2. Use this CD to boot your PC into Windows Repair Mode. You must change the BIOS boot priority in order to make it happen.
  3. Use System Restore to set Windows back to a point before your daughter changed the password.
  4. Plan ahead and treat Windows like your house. You have at least two house keys so that you are not locked out in case you mislay one. By the same token, you must have a spare admin account (fully tried and documented) so that accidents like the current one are a minor nuisance instead of a major problem.
