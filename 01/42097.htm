Although I have the Import export etc., I did not have the customize tab.

I checked it on three systems.

By adjusting the registry as shown below, you can enable it.

This is the key for the Contacts folder.

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions\{56784854-C6CB-462B-8169-88E350ACB882}\PropertyBag

In that key you will see a NoCustomize Dword with a value of 1, change to a 0 and you will see the customize tab.

As always changing the registry, if done wrong, can make your system unusable.

Backup the key before changing and set a restore point

Components that need to be uninstalled prior to updating to Windows 8.1
The following components if installed on your PC must be uninstalled prior to updating to Windows 8.1.

 

Uninstall Component List
  1. AMD Display Driver (ONLY required on Model Part Numbers beginning with PSD24U, PSD24M , PSD24P)
  2. Intel Proset
  3. Intel Wireless Display
  4. Nvidia GE Force GTxxxx (includes all drivers that begin with Nvidia GE Force GT)
  5. Realtek Audio driver
  6. Realtek Bluetooth Filter Driver Package
  7. Realtek WLAN Driver
  8. TOSHIBA PC Health Monitor

 

To determine if any of the components are installed on your PC follow the instructions below and remove the listed components, if any, prior to the Windows 8.1 update
  1. Press the Windows logo key + X
  2. Select Programs and Features
  3. Look through the list and cross check it against the Uninstalled Components List above to see if any components are required to be removed from your PC
  4. To remove any Uninstalled Components:
    1. Select the component then click Uninstall
    2. Follow the prompts to uninstall the component
    3. Repeat these steps until all Uninstalled Components listed have been uninstalled from your PC
