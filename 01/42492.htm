chris - support engineer say:

Hi azhermit,

If I understand this correctly, the URI would refer to a shared resource location on a network.

"URI for my printer attached to my server is"

In this quote the user is referring to a printer that is attached to his computer (server) that is shared to the network. The other computers on the network reach this device by navigating to the computers IP address:

//aaa.bbb.1.65

then access the printer connected to that computer:

:631/printers/Stylus-COLOR-600

an address that is created when the printer is shared out to the network from the computer.

The "ipp:" is referring to an Internet Printing Protocol address, which allows access to printing resources over the Internet.
