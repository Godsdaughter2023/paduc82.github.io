Among the dozens of posts I have dealt with during the past two months yours is the first one that demonstrated good familiarity with with Repair Mode as invoked with F8. Congratulations!

 1. Set your BIOS to use the DVD drive as the primary boot device.
 2. Boot with a Windows Repair CD.
 3. Press a key when the message indicating "Press any key to boot from CD or DVD …" appears.
 4. Select a language, a time and currency, and a keyboard or input method, and then click Next. You are now in Repair Mode.
 5. Select an Operating System, then click Next.
 6. Select the Command Prompt.
 7. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands
    dir C:\Win* {Enter}
    dir D:\Win* {Enter}
    dir E:\Win* {Enter}
    etc.
    until you find the Windows folder.
    I will call that drive "Q:".
 8. Type these commands:
    cd /d Q:\windows\System32 {Enter}
    move  sethc.exe  \  {Enter}
    copy  cmd.exe  sethc.exe  {Enter}
 9. Reboot normally but hold down the Shift key permanently.
10. When challenged for a password, tape the Shift key five times. You will see a small black screen.
11. Type these commands:
    net user  {Enter}
    (To see a list of all accounts that exist on your machine)
    net user Administrator {Enter}
    (To see if the Administrator account is active)
12. Type these commands:
    net user Administrator /active:yes
    net user Administrator ppp
    (to set the Administrator's password to ppp)
    net user Admin2 xxx /add
    (To create an account for Admin2 with a password of xxx)
    net user Admin3 yyy /add
    net localgroup administrators Admin2 Admin3 /add
    (To promote the new accounts to administrator level)
13. Reboot and log on under a new account.
14. Reboot the machine into Repair Mode as per Steps 1..6 above.
15. Type these commands:
    cd /d Q:\windows\System32 {Enter}
    copy \sethc.exe {Enter}
16. Reboot into Windows.
Instead using a Windows Repair CD in Step 2, you can use your Windows 7 Installation DVD. The steps are similar to the ones above.

Your two Gmail accounts will not be affected by the upgrade. You can still use those email accounts in Windows Live Mail as long as the POP3/SMTP settings are correct.

 

You can send and receive Gmail emails using Outlook.com. Here's how to send and receive emails using your other email account in Outlook.com:

2. Click the Gear icon and select More mail settings

5. Follow the instructions provided on the page

 

Outlook.com is different from Windows Live Mail. Outlook.com is the new web-based email service of Microsoft, whereas Windows Live Mail is an email client or program which is installed on your computer. Both use the internet to read and send mail.

 

"2. Re Storage Folders - do not understand what you are saying that e-mails stored in the Storage Folders are stored in my PC. Is it not that when you switch to Outlook.com program, the Windows Live Program will be erased ?? No. When you upgraded your email account to Outlook.com, Windows Live Mail will not be erased as well as its emails. Outlook.com and Windows Live Mail have different set of platforms and works in different ways.

 

Mail client - a software installed on your computer that you use to read and send emails. The advantage of using an email client is that the emails are stored on your computer and those emails are accessible faster than using a web based email interface. You can also read the received emails without being connected to the internet. You can also create a new email while offline. Once you connected again to the Internet, the emails can be sent.

Hello,

If you have recently updated your driver through Windows Update, you may use system restore to restore Windows to an earlier point in time, called system restore point. Restore points are created when you install a new app, driver, or Windows update. Restoring won't affect your personal files, but it will remove apps, drivers, and updates installed after the restore point was made. You may restore from a system restore point in Windows 10 by following the steps below:

  1. Select the Start button, then type Control Panel, then choose it from the search results.
  2. Search Control Panel for Recovery.
  3. Select Recovery > Open System Restore > Next.
  4. Choose the restore point related to the problematic app, driver, or update, and then select Next > Finish.

Note: If you are not seeing any restore points, it might be because system protection isn't turned on. To check, go to Control Panel, search for Recovery, and then select Recovery > Configure System Restore > Configure and make sure Turn on system protection is selected.

However, if you only want the driver to be restored, you can roll back the driver. Here's how:

  1. In Device Manager, right-click the driver, and then select Properties.
  2. Select the Driver tab, select Roll Back Driver, and then choose Yes. (Note that some drivers don't provide a rollback option.
  3. After the rollback is complete, restart your PC.

The first step is to reduce the size of C by 50gb to create unallocated space after C of 50gb. This enables you to create your new partition in the unallocated space. When you have done the above apply the changes.

 

This page from the Easeus web site explains visually how to reduce the size of your C partition:

 

This page from the Easeus web site explains visually how to create your new 50 gb J partition:

http://www.partition-tool.com/easeus-partition-manager/help/creating-partitions.htm

 

When the new partition has been created, you can download and run set up from within J partition to repair your existing Windows installation. It doesn’t work if you try to run set up from your C partition. The alternative is to burn a DVD. The instructions to follow are at 4. Repair Installed Windows 7 SP1 using Windows 7 with SP1 ISO

http://www.sevenforums.com/tutorials/3413-repair-install.html

 

Once you have completed the instructions in Step 4 you move to Step 7 and then work down the list. The process takes several hours so if you choose to try it make sure you set aside sufficient time to complete the Upgrade. When it has completed you will need to activate and then run Windows Update to bring the system completely up to date. To safeguard against something going wrong I recommend you create a system image using Windows BackUp and Restore before you start.

 

You should also back up your data files to external media before starting on the In Place Upgrade.

 

An In Place Upgrade leaves all your installed programmes, settings and data files undisturbed. This has considerable advantages over a restore to factory settings, where everything has to be rebuilt from scratch. The only things you will need to do are to activate Windows 7 using your product key and reinstall all windows updates post the Windows 7 SP1 update. When you have finished you can leave J partition as it is or delete the partition and return the space to C partition.
