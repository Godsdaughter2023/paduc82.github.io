a. user say:

Sounds like electromagnetic interference (EMI) which is also called radio frequency interference (RFI.) Any wire can pick it up from nearby electromagnetic sources like overhead power wires or radio station transmitters. Sometimes re-positioning the wires makes a difference, other times not. You may need to buy ferrite beads or chokes from an electronics store.

Another possibility is that you've got a broken wire, which is not uncommon for headphones that get moved around a lot.

frederik long say: Here you go:
1. Keep tapping F8 during the early boot phase.
2. Select Repair from the menu.
3. Enter the Administrator's password when prompted. It is often blank.
4. Select System Restore when prompted.
5. Set Windows back to a point before this problem occurred.

If you cannot log on under Step 3, use this procedure:
1. Set your BIOS so that it uses the DVD drive as its primary boot device.
2. Boot the machine with your Windows 7 Repair CD.
3. Press a key if prompted to boot from the DVD.
4. Select "Repair PC" from the menu.
5. Select Repair when prompted.
6. Select System Restore when prompted.
7. Set Windows back to a point before this problem occurred.

When finished, consider if you intend to continue with just one admin account. Very risky! You should have two fully tested and documented admin accounts, same as you have two keys (or more) for your car.
