kevin d souza say:

Hi Richard,

 

When you backup files from the computer to an external hard drive, it will keep the initial copy of the file. If you make any modifications to the file and then try to backup the same file, then the file in the backup gets updated without actually making two copies of the files. This will help with the disk space management in the hard drive. Instead of saving 2 copies of the same file, it updates the original one.

 

Regarding the number of backup files are being saved on the drive, it depends on how many backup files do you want to save. If you do not need the older backup files, you can go ahead and delete them and keep the most recent ones.

 

When you set up Windows Backup, you can let Windows choose what to back up, which will include a system image, or you can select the items that you want to back up and whether you want to include a system image.
