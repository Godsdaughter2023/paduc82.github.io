mginbend say:

OK, After more research on the internet I figured out how to do this so I'll provide the solution in case someone else has this problem...

Right click on the desktop and click on New > Shortcut

Paste this code in the "Type the location of the item" box:

%windir%\explorer.exe shell:::{3080F90D-D7AD-11D9-BD98-0000947B0257}

Name the shortcut Desktop

Right click on the shortcut, select Properties, Shortcut tab and Change Icon button

Look for Icons in this file: %SystemRoot%\System32\imageres.dll

Select an appropriate icon and click OK.

Move the new shortcut to this folder in your user account folder:

AppData > Roaming > Microsoft > Start Menu > Programs

This will cause the shortcut to show up in the All Apps Start Menu

Open All Apps, right click on Desktop, pin to Start

I believe the problem is that the shortcut was not pointing to %windir%\explorer.exe shell:::{3080F90D-D7AD-11D9-BD98-0000947B0257} since the shortcut was in the right folder but it wasn't doing anything.
