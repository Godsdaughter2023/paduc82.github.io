In windows File Explorer, All you need to do is browse to the folder you wish to remove files from and sort by Clicking on 'Type'.  From there it is easy to select just the .doc files.

DEL and ERASE can not be disabled perse, as what many people are unaware of is that then you say in File Explorer click a set of files to delete, in the background unbeknownst to the average user, under the hood so to speak Windows is actually using the DEL or Erase command to do these processes.

Windows is gearing towards removing the command prompt style itself, and going strictly to Powershell, which is a more powerful than the command prompt.

Please see article listed at: http://www.howtogeek.com/163127/how-powershell-differs-from-the-windows-command-prompt/

As I stated earlier, the DEL and Erase commands in powershell do work like you were expecting it to work in the Command Prompt.  In Windows 8.1 you can even replace the Command Prompt open from Win + X (Right click on Start) options as well.

To do this:

  • Right Click on blank area of Taskbar, and click on Properties
  • Click 'Navigation' Tab
  • Click 'Replace Command Prompt with Windows PowerShell in the menu when I right-click the lower left corner or press Windows key +X' to put a check mark in the box.
  • Click OK

Now whenever you use Win+X, or right click lower left corner, Windows PowerShell, and Windows PowerShell (Admin) replace the Command Prompt and Command Prompt (Admin) options that were there.

Reinstalling Windows 7 - Microsoft Community

f23948 say:

Reinstalling Windows 7

I can format and reinstall Windows 7 all I want just in case if something happens? so i can activate Windows 7 by phone without using the internet?
andre da costa say:

Yes, you can.

If Windows 7 came preinstalled on your computer, you might not need to do Phone or Internet Activation.

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

This is how the recovery partition is accessed for the most popular makes..

For Dell, press F8 on the keyboard until the Advanced Boot Options menu appears on the screen.

For HP, press F11 directly after powering up the unit

For LG, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Samsung, press F4 at power up..

For Fujitsu, press the F8 key repeatedly directly after powering up

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

For Sony VAIO, restart and press "F8" or "F10" repeatedly until the "Advanced Boot Options" screen appears.

If your Windows 7 license is a retail copy and you have the reinstall disc, you can use the following instructions to reinstall Windows 7:

http://windows.microsoft.com/en-US/windows7/Installing-and-reinstalling-Windows-7

If Phone activation is required, see the following:

 How to activate Windows 7 manually (activate by phone)
 
1) Click Start, and in the Search box type:   slui.exe  4
 
2) Next press the 'ENTER' key
 
3) Select your 'Country' from the list.
 
4) Choose the 'Phone Activation' option.
 
5) Stay on the phone (do not select/press any options) and wait for a person to help you with activation.
 
6) Explain your problem clearly to the support person.
 
http://support.microsoft.com/kb/950929/en-us

Also see:

http://techingiteasy.wordpress.com/2012/04/13/how-to-activate-windows-7-oem-license-using-a-retail-disc/
