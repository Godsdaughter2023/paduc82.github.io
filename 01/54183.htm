lou johansen say:

I cannot delete messages from the "Sent" File

 I cannot delete messages from the "Sent" File.  The message, "An Unknown Error Has Occurred", is presented when I try to delete one or all of the messages.
¡firedog say:

[Please don't start multiple threads about the same problem. It only causes confusion for all concerned. I agree that it's difficult to find the right forum for Windows Live Mail, but the moderators will usually be able to move your question to the right place.]

This type of error arises because the message database has got confused. The message list is retrieved from the database's records of which message is where. Every transaction in WLMail (e.g. a message arriving, being read, replied to, moved, copied or deleted) is recorded in a log file. These log files are then replayed in quiet moments and written to the database. If there are still log files to replay when the program is closed down, they are replayed then. If this process is interrupted, e.g. by closing down Windows, the database will get out of touch with reality. No amount of uninstalling and reinstalling the program files will correct faults like this; they exist in the program's data files which remain untouched by the uninstall/reinstall process.

Errors of this nature are usually corrected by closing the program down and waiting a few minutes before relaunching. It's worth watching Task Manager to see when wlmail.exe disappears from the Processes tab before acting.

A re-indexation of the message database may correct the fault. To do this,

  1. On the Home tab, click Work offline.
  2. On the program options sheet (press Ctrl-Shift-O for Options), select the Advanced tab and click Maintenance.
  3. Set Compact the database on shutdown every... to 1 and make sure the check box is ticked.
  4. Click Close, Apply and OK.
  5. Disconnect from the Internet and close down all running programs including your security software.
  6. Close WLMail. A progress window headed Recovering unused disk space should appear.
  7. Wait a few minutes, then reboot the computer and check that your security software is running again.
  8. Re-connect to the Internet, launch WLMail and test.

If this is successful, you can reset the compact counter.

Hi Sandy,

If you want to run Windows 8 on your PC, here's what it takes:

  • Processor: 1 gigahertz (GHz) or faster with support for PAE, NX, and SSE2 (more info)

  • RAM: 1 gigabyte (GB) (32-bit) or 2 GB (64-bit)

  • Hard disk space: 16 GB (32-bit) or 20 GB (64-bit)

  • Graphics card: Microsoft DirectX 9 graphics device with WDDM driver

  • Processor: 1 gigahertz (GHz) or faster with support for PAE, NX, and SSE2 (more info)

  • RAM: 1 gigabyte (GB) (32-bit) or 2 GB (64-bit)

  • Hard disk space: 16 GB (32-bit) or 20 GB (64-bit)
