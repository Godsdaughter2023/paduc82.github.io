arjay dom say:

Thank you for the information.

Try resetting Windows Store apps by following the steps below:

1. Press Windows key + Q.

2. Type powershell and right-click on Windows PowerShell.

3. Select Run as administrator.

4. In the administrative Windows PowerShell window, type the following command correctly and hit Enter:

Get-AppXPackage | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

5. Once the command is successfully executed, close Windows PowerShell and reboot your computer.

Note: We recommend creating a System Restore Point first before resetting Windows Store apps. This will hep you restore your computer system files to an earlier point in time in case the app was reset incorrectly.

Let us know if you need further assistance.
