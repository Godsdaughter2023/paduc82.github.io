hairyfool say:

To switch Windows 8 from one Microsoft account to another and retain all the user settings:

 

Go to PC settings -> Users and change the account from a Microsoft Account to a local account. This sets up the same actual user but with a local username and password.

 

Log back in as that local user and using the same option change that local account to a Microsoft Account. This then asks you the credentials of the Microsoft Account you want to sign in with which can be different to the original one or you can set up an entirely new one.

 

All through this process you are still using the same security ID that the computer knows the user as so settings should be retained.

You may try connecting the device to a different USB port and check if the issue persists.

Also connect the device to another computer and check if the issue persists so we can figure out if the device has gone bad or the USB port.

If the issue still persists, would recommend you to check for the issue in ‘Safe Mode'.

Step1:
Below are the steps which talks about how to get into ‘Safe Mode’:
1.     Restart your computer if it is powered on.
2.     Tap the F8 key after your computer initially powers on.
3.     Once you see the Advanced Boot Options menu you can stop tapping.
4.     Use the up/down arrow keys to highlight your selection.
5.     Select Safe Mode and press Enter.
6.     You should see drivers loading, and then please wait.
7.     You should then be at the Welcome Screen.
8.     Logon to your computer using an account with Administrator privileges.

Step2:
If the issue does not persist in Safe mode, you may put the computer in a Clean Boot state to check which third party service is causing the issue.

Below are the steps to put the computer in a Clean Boot State:

1. Click Start, type msconfig in the Start Search box, and then press ENTER.

 If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.
2. On the General tab, click Selective Startup.
3. Under Selective Startup, click to clear the Load Startup Items check box.
4. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

5. Click OK.
6. When you are prompted, click Restart.
7. After the computer starts, check if the issue is fixed.

If your issue is resolved after performing Clean Boot, then follow the steps mentioned in the following KB article to narrow down the exact source:
