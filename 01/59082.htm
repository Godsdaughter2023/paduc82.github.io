considerate_guy say:

As you have noted, the .mswmm is not a movie.  It is the project file from Windows Movie Maker 2.6.  To create an actual movie you need to use File > Publish Movie.  That will create a .wmv video file.  That also explains why a video converter like Freemake Video Converter can't do anything with that file.

Most often when you are having this problem it is related to two things:

  1. The source files are incompatible with Movie Maker.  If possible, convert all source video files to .wmv format and all source audio files to .wma or .wav format.  Replace the files and then see if you can save your project as a movie.
  2. The project is too complex.  Try making a few copies of your project (.mswmm) file and then remove from the end until the project will save.  If that works, then use the next copy and pare it down to the rest of the movie.  At the end you can join them together in a new Movie Maker project.

WMM 2.6 is designed for those on Windows Vista with insufficient hardware to run the full Windows Movie Maker 6.0.  And both of these were created before the latest Movie Maker in Windows Essentials.  Ideally, as noted by Glenda, you would switch you using the latest version of Movie Maker.  However, if you still need the "classic" version of Movie Maker (with a traditional timeline and custom transitions, titles and effects), you may want to look into loading Windows Movie Maker 6.0 instead of WMM 2.6.

A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you install a program or an update or when you run a program in Windows. You may also troubleshoot or determine what conflict is causing the problem by performing a clean boot. You must log on to the computer as an administrator to be able to perform a clean boot. Your computer may temporarily lose some functionality when you perform a clean boot. When you start the computer normally, the functionality returns. However, you may receive the original error message, or experience the original behavior if the problem still exists. If the computer is connected to a network, network policy settings may prevent you from following these steps.

a. Hover the mouse to the right hand corner of the screen to get the charms bar.

Note: While performing SFC Scan I would recommend you to make sure that there are no applications or programs running on the back ground.

Safe mode starts Windows with a minimal set of drivers and services. If a problem doesn't reappear when you start your PC in safe mode, you can eliminate the default settings and basic device drivers and services as possible causes.
