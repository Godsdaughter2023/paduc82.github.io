spiritx say:
We are volunteers willing to help you try to resolve this issue.

Check these images which show all sorts of SDHC card readers :

https://www.bing.com/images/search?q=computer+s...

Does your computer have an SD card reader slot? Some that do will need an adapter to put the SDHC card into. And some older computers will not read the newer SDHC format.

Your System Maker's Support, documentation, and forums will be a good source of information if required.

If you have a card reader that will accept and read the SDHC card all you have to do is put it in the slot. Then Windows will treat it as another drive.

If the computer does not have a card reader or will not read an SDHC card then you can add a USB Card Reader with the desired abilities and those are not expensive. Computer Shops, most Office Supply Shops, and many on-line retailers sell SDHC USB card readers.

---------------

The other part of the issue is to get the info off of the card onto the computer, Many cameras come with software to help so you just need to install that software. Refer to the camera/software maker for help. Others will use the Windows apps or require File Explorer.

For File Explorer :

Use File Explorer (icon on the Taskbar of the Desktop or Use Cortana and type in File Explorer) then navigate to the old folder and COPY (hold shift to highlight many files or CTRL to copy individual files in a group) then go to where you want them on the C: or whatever drive you will be using on the computer and PASTE. Afterwards you can Delete the original copies in the previous fold if the new copies are OK.

Or you can do the same except use CUT instead of Copy (Cut removes the files when you do the paste).

Please let us know the results and if you need further assistance.
