    keith a. miller say:

    Arbitrary ordering of files & folders is now only possible on the Desktop. It was possible in XP, and possibly Vista, but the code that made it possible and the associated registry values no longer exist.

    The order of items in a folder must be based on one of the properties available as a column in Details view (default is Name). So as Tamara suggested, a numerical prefix to your folder names is your best option.
