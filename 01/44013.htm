auggy say:

To overcome any "read" problem try the following to copy the minidump files to a folder on the Desktop:

First, can you create a folder on the Desktop and call it Minidumps

Then open the C:\Windows\Minidump folder and do the following:

Organize > Select All > Organize > Copy

Then open the Minidumps folder on the Desktop do the following:

Organize > Paste

Then zip up the Minidumps folder on the Desktop.
