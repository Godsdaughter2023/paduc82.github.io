a. user say:

Hi Kian,

Thank you for posting your query in Microsoft Community.

DPI refers to a mouse’s hardware capabilities, while sensitivity is just a software setting. In Windows, the mouse settings can only help you modify the pointer speed and precision in general. You only get the setting to enhance pointer precision but by design it does not provide any option to define or scroll any bar of polling rate.
