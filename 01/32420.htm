elohmn say:

Okay a little confusing but here's how it works. When any External Device be it a HDD or Flash Drive is used as a Storage Device (Memory for the Xbox 360), the Xbox 360 will only recognize it as such (a Storage Device) via the USB. In short, a really huge External HDD becomes a 16Gb (Max) Storage Device from USB.

Some Games and some Features of Xbox LIVE require an Official MS Xbox 360 HDD that is for that particular Xbox 360 Console (ie Slim with Slim HDD). Simply using an External HDD will not work for these cases. Any External Storage Device is meant to supplement, not replace an Official HDD or the performance of one that some Games and Features require.

Users who log on as a guest have a very limited set of permissions. For example, they cannot access other users’ files or perform system wide tasks such as installing software or hardware.
