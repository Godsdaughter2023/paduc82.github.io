In WMP, please go to Tools - Options - Security and click Zone Settings (press Ctrl+M if you don't see the Tools menu). That script error could occur because your Internet zone is set to a custom level and Run ActiveX controls and plug-ins is set to Disable. Hi Missy




InstantGo is installed at the factory by the computer manufacturer. It replaces the traditional Sleep functions in Windows 8x. It is only supported on systems that have the compatible hardware.




You can check your computer to see if it has InstantGo (Also called
Connected Standby).




Go to the Desktop.




Right Click the Windows Logo at the 
bottom/left corner of the screen. In the popup menu, click
Command Prompt (Admin).




In the Command Prompt window type the following command.




powercfg /a




Press Enter.




In the results if you see Standby <Connected> then your system has the InstantGo component installed.




Here is a screenshot of what it looks like.








See the following links for more information.




InstantGo: a better way to sleep

http://blogs.windows.com/bloggingwindows/2014/06/19/instantgo-a-better-way-to-sleep/


 


InstantGo: How your device gets a better snooze than in sleep or standby mode

[quote user="stylerXrussak"]Man hat von 320GB aber nur 292GB frei[/quote]


Bevor einer glaubt hier wird der Kunde betrogen: zum einen steht auf der Packung, dass bei den 320 GB der Packung ein GB 1.000.000.000 bytes (Base 10) entspricht. Aber in der Informatik gibt es diese "runden" Zahlen bei Giga-/Mega- und Kilobyte nicht. Ein
 Kilobyte sind in wirklichkeit 1024byte. Ein Megabyte sind 1024 kilobyte. 



Leider drucken Marketing-Leute immer nur die "runden" Laufwerksgrößen, da dadurch das Laufwerk "größer" erscheint.



Wenn Du mit diesen "base 2" zahlen rechnest, kommst Du auf ungefähr 298GB. Von diesen 298GB zwackt sich die Xbox Systemsoftware für Cache, Title-Update-Speicher und Xbox Emulator nochmal etwa knapp 7GB ab. Und damit kommst Du dann auf die freien 292 GB, die
 korrekt über Base-2 berechnet wurden inclusive abgezogenem Systembereich. 



Als Faustregel kannst Du von dem angegebenen Platz auf der Packung immer nur 7% weniger Platz wirklich nutzen.



Oder um einen Anschaulichen Vergleich zu machen: 



Marketing Leute verkaufen Dir eine Garnrolle, und sagen Du kannst darauf 320 Seilstücke aufrollen (und meinen Damit Seilstücke von je 1000 cm Länge, sagen das aber nur im Kleingedruckten).



Allerdings kommen die Seilstücke vom Rechner bzw. der Xbox nur in einer Länge von 1024cm. Also passen "nur" 298 Seiltücke auf die Spule, weil die Seile nun mal eben 24 cm länger sind, als die Stücke, mit denen die Marketingleute rechnen.
