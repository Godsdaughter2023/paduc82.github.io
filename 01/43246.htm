If the music files transferred from the XP computer are protected by digital rights management (DRM), licenses for these types of files are not moved to the Windows 7 computer.

 

To re-obtain rights to the files, you'll need to contact the provider of the files. Some online stores offer this and might refer to it in different ways, such as computer activation, computer authorization, library restoration, or license synchronization.

 

In the meantime, follow these steps to fix the issue:

 

Step 1: Reset the DRM licenses

 

Important: The ResetDRM tool deletes all existing Windows Media DRM licenses and Zune DRM licenses on your current installation of Windows 7. The Reset DRM tool should only be used if all Windows Media DRM and Zune DRM content playback is nonfunctional. After the following process is complete, you have to reacquire DRM licenses for your Windows 7-based computer.

 

a.    Download the ResetDRM tool and save it to your desktop:

 

http://go.microsoft.com/fwlink?LinkId=105000

 

b.    Right-click ResetDRM.exe, and then click “Run as administrator” to reset the Windows Media DRM.

 

Step 2:

 

Upgrade the DRM by running Internet Explorer as administrator.

 

Follow these steps:

 

a.    Right-click “Internet Explorer” and click “Run as administrator”.

b.    Visit the following website:

 

http://drmlicense.one.microsoft.com/Indivsite/en/indivit.asp

 

c.    Click “Upgrade”.

 

For more information on plug-ins, see Windows Media.
