a. user say:

Hi Laurie Millikin,

 

Since you’re unable to send messages plus you’re getting this error 0x800CCC0D,please be informed of the following possible causes of your issue:

 

This can be broken down to 2 causes:

1. Incorrect POP settings (usually the SSL option for SMTP) – check with provider if it is required or not

2. Firewall blocking ports (usually port 25) – could be 3rd party since the Windows Firewall generally allows this port through, unless specifically told not to do so.

 

We advised you to reconfigure your account by referring to the article below:

How do I add an account to Windows Live Mail?  

 

For firewall issue, you can try to disable the security software/firewall to isolate the issue:

How can I resolve machine configuration issues in Windows Live Mail?

 

Hope this helps.

 

Thanks.
