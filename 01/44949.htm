You don't.

That is not what that registry key is for and you can't abuse it to accomplish your goal.

That registry key tells Windows (the autochk program) to check the dirty bit on every volume and if a dirty volume is found Windows will automatically attempt to fix the volume by running a chkdsk with error correction (chkdsk  /r).

If you want to run a program every time Windows starts the easiest thing to do it make a shortcut to the program and put it in your Windows Startup folder - that's what the folder is for.

Just open your Startup folder, right click, Create new shortcut, enter the command you wish to run and save it.

Cause
