steve merik say:
Hello Christopher, hope this message finds you well.

External hard drive requires the Bitlocker key if it is encrypted with a Bitlocker. If there is an encryption, you will not be able to get access to a different system without the key.
However if you haven't set the BitLocker key, the Secure Boot settings might be switched to Disabled, causing the issue. Please follow the steps to check it.
- Press the power button on your device. After shutting down, press it again to restart the system.
- Find the message on the boot screen and follow the instructions (press the ‘F1’, ‘F2’, or ‘Delete’ button). This message will appear before the manufacturer logo.
- Enter the BIOS settings, find a tab called BOOT and click on that page. You will see BOOT, ADVANCED, STARTUP, etc, it depends on the BIOS manufacturer.
- Search for the settings for the UEFI Secure Boot. Each BIOS could have a different name for this. Try the keywords: UEFI, Secure Boot, Legacy Boot
- Switch the Secure Boot settings to Enabled and try to boot the machine.
In case the password for the BitLocker is set and you don’t remember it, refer to the following article https://support.microsoft.com/en-us/help/402618...
The last option to choose is formatting the hard drive to access it. However if Windows was installed on this the same drive, you will have to reinstall Windows in the end. But hard drive formatting will delete all content.

chaos_edit say:

Try this:

Start>Control Panel>Device Manager>CD/DVD>double click the device>Goto Driver's Tab>click
Update Drivers (this will likely do nothing)>then RIGHT CLICK>UNINSTALL>Restart
this will rebuild the default driver stack.

Then go through these:

Your CD drive or DVD drive is missing or is not recognized by Windows or other programs
http://support.microsoft.com/kb/314060 - a Mr Fixit

Your CD or DVD drive cannot read or write media - A Mr Fixit
http://support.microsoft.com/gp/cd_dvd_drive_problems

CD-R drive or CD-RW drive is not recognized as a recordable device
http://support.microsoft.com/kb/316529/

Hardware devices not detected or not working - A Mr Fixit
http://support.microsoft.com/gp/hardware_device_problems

Hope this helps.
