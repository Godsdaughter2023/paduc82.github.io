alden rey say:

Hi,

The concern you have encountered may be due to a corrupted file or a misconfigured setting.

Many audio problems on a computer can be caused by DPC latency.  DPC stands for Deferred Procedure Call.  In its simplest form, it is the part of your Windows system that handles driver efficiency.  If there is a driver that is taking longer than normal to process, it may prevent other drivers from being processed in time.  The worst case is that it can cause your audio interface driver from responding in time and can cause clicks, pops, distortion and dropouts.

We suggest to update the driver of your audio card. Generally speaking, updated drivers deliver better functionality and improved performance. For the most part, you certainly want to upgrade your drivers unless you have a very specific reason for not wanting to do so. argel obm say:

Hi,

When you access Cortana via "Hey Cortana" voice command, a small window will pop-up which you can use to search something by typing in the widow or by using another voice command. It is the design of Cortana in Windows 10.

Just to clarify, are you playing games which were downloaded from the Windows Store, or are you playing games that you've installed using an installation disc or games that you've downloaded on an online store? Also, what do you mean by "it will open the search tab for cortana and tab me out of the game"? Does the game in full screen mode minimizes when you access Cortana using a voice command?

If you're playing Windows Store games, they will not be tabbed out when you use the "Hey Cortana" voice command. They will continue to run without tabbing out because these kind of applications are designed to work seamlessly with the other features of Windows 10.

If you're playing games that you've installed using an installation disc or games the were purchased from an online store, these application will be tabbed out when you access Cortana using voice command. They will be tabbed out because you're instructing the computer to run a new application which is Cortana. When running these kind of games, Windows 10 is treating them as a whole process. Windows 10 will tab out the game in full screen mode to give way to the new process which is the Cortana search function.
