 Windows 7 Home Premium runs very well in 3 GB as long as you don't start more than 8 to 10 things at once.  One thing that can really affect performance is things that are started automatically when you boot up.  To remedy this, you should click Start and type MSCONFIG in the search box.  Then click on MSCONFIG.EXE.  Click the Startup tab and uncheck everything you see there except your antivirus program and then reboot.  That can make a significant difference in performance.

 If you have the 64 bit version of Windows 7, you could increase the amount of RAM you have.  My Toshiba Satellite L755 can hold up to 8 GB.  The 32 bit version can only address 4 GB, so you wouldn't see much improvement with an extra 1 GB if that's what you have.

If English is not present click on Add language and select it (United States or UK) to install it.
