josh-h say:

Hi,

You should not have anything to worry about with this message, and it is certianly not a virus. Basically what this means is that a file contained within that drive was in use by a program when you formatted the drive. So, in order to continue with the format, you had to "dismount" from that program, as well as any others, so that all the files in that drive were not in use. So for example, if you had a Word document open at the same time you tried to format the drive, this message would appear and you would not be able to continue untill you "dismounted" from Word, meaning that Word would no longer have access to that file untill the formatting was complete.

So unless you are seeing any other issues with performance since formatting, I'd say you have nothing to be worried about.

Why does Windows 7 delete Shortcut icons

I have Windows 7 Enterprise loaded on my laptop. My favorite games are Freecell, Spider Solitaire, Solitaire, Minesweeper, Mahjong Titans, and Chess Titans. As I play these games often I made shortcuts on my desktop. When the screensaver comes on I have it set to 3d text with the time displayed and on resume, display logon screen. I have it set to go to screensaver after 10 minutes then under my custom plan settings the display goes off after 20 minutes. When I return after the display has turned off I type in my password and when the display comes back on the shortcuts for these games have been removed. I have 6 other desktop icons and these are still there only the game icons get removed. Is there a reason this happens and is there a way to keep this from happening?

I applied the answer from dang60 to my computer.

To prevent the task from running you need to disable the task in the task schedular: start-programs-accessories-system tools-task scheduler
Expand the tree in the left hand pane until you get to Task Scheduler Library-Microsoft-Windows-Diagnosis.
In the right hand pane, right click the task named Scheduled and click Disable.

Not everything was under the names he provided but I found and disabled the task named Scheduled.
So far this has worked for me.
