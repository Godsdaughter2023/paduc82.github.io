Maybe a question better asked on the ASUS forums, but I'd guess (only having ever used two

[based on the bandwidth availability.] - should apply to the amount of RAM available to graphics.

With integrated graphics a certain amount of system RAM is allocated to graphics.

- There should be a setting in the BIOS that allows either Auto, or designated amounts up to

1GB (maybe more on the latest motherboards). If the BIOS is set to 1GB it may be the three

monitors are requiring more than 1GB RAM to display.

- If set to Auto performance could be hobbled when running several RAM intensive programs

if the system has only 4GB RAM, as some of the RAM that would be needed for graphics may be shunted to the software.

- Of course if you are running graphics software such as Photoshop  this will have the potential

of using what's left of the available graphics RAM allocation when displaying high res photos,

especially if you have more than one window open.

Long story short - When using three monitors (particularly high resolution - 1920X1080 or higher) you are better off with a dedicated graphics card with 2GB (or more if the 27" a 2560X1440 model ) of dedicated VRAM.
