cbarnhorst say:

Whenever you clean install the installation drive will become C: and other drives will be lettered without regard to any other system on a different partition.  

So don't rely on drive letters.

Name all your partitions by what is on them or how you use them.  Then you don't care what letters get assigned in a particular OS versus how they appear in another OS.  The drive names will always be the same no matter what OS you happen to be running.  C:\Windows 7 System retains the same name no matter the drive letter that happens to get assigned even if you are in Windows 10 where it may appear as F:\Windows 7 System.  The Windows logo always appears above the current system drive to help you as well.

Name a drive by right clicking on it in File Explorer and choose rename.
