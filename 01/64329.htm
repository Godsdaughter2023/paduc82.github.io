Lost previous restore points and update history - Microsoft Community

george wasz say:

Lost previous restore points and update history

Following the Microsoft Anniversary Update (Version 1607) today, I noticed that all my previous restore point and update history is gone?  Does anyone know why or how to retrieve that information?

a. user say:

This is by design, and it will happen again every time a major update to Windows is released. You would not want to restore a restore point from before the update. Were it possible to do so, your copy of Windows would be a mishmash of old and new system files that could cause Windows to become unstable and even to crash.

This brings out one of the weaknesses of System Restore and illustrates why Microsoft is moving away from that technology. A restore point is not a backup. Rather, it's a collection of key system files and drivers needed to restore the Windows user interface, so that people can access Windows' own tools to (hopefully) fix a problem their computer is facing.

Windows now includes its own backup utilities, so that users can make true backups of Windows and their own files and folders. When you restore a backup of Windows, all of Windows is restored to the way it was when the backup was made. You could, for example, restore a backup of Windows made before the anniversary update was installed, and your entire computer would go back to the way it was before the update was installed. This is far superior to the piece-meal approach offered by System Restore.

And if you want to go even further you can learn to use third-party disc imaging tools to achieve features and convenience even greater than what Windows offers on its own.

At Microsoft, we want to help parents create a healthy computing environment for their kids.

 

The only way to work with the Windows Family Safety is to log in with the Microsoft Account and Password which is registered as the Parent account as Windows Family Safety can be accessed globally if you use the Microsoft Account and password to monitor your Kids activity. You will also receive Emails of the Activity Report on the Microsoft Account.

 

If you use Finger print reader or any other bio metric sign in method to log in to the Windows 8.1 computer, it will not affect the Windows Family Safety login as you will still require the Microsoft Account and password to log in to Windows Family Safety.

 

There are many fingerprint readers which are compatible with Windows 8 and Windows 8.1 operating system. Most of the fingerprint readers come built in with the laptop you purchase thus the computer manufacturer's are responsible for providing the drivers/utilities which will make the fingerprint reader and other such hardware work with the version of Windows you have installed on the computer.

The Windows Biometrics Framework (WBF) Win32 client API, driver specification and adapter specification have not changed for Windows 8.1.
