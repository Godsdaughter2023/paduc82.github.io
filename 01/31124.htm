How to unlink Windows 10 lock screen from my Microsoft account - Microsoft Community

hughhasan say:

How to unlink Windows 10 lock screen from my Microsoft account

Hi,

I just upgraded to Windows 10 and in the process connected/linked my Microsoft account to the computer lock screen (unintentionally).  So now, each time anyone in the family needs to unlock the screen they require my Microsoft account password (which naturally, I don't want to share).  I would like to break this connection and have a completely separate password, just for logging into the computer.  How do I do this?  Thanks in advance.

Hugh

a_user_here say:

You have not said how you shared your machine in the past.  As does each person have their own account or are you all sharing the same account?

If each person has their own account (this is the best way to share the computer between people), then those accounts should still exist after the upgrade.  All people should have to do is click on the lock screen and it show where it asks for the password, but at the bottom right there should be a list of user names.  If a person clicks on their user name they can log in with their password.  Note this doesn't log your account out, it allows both people to be logged in at the same time.

If you didn't use separate accounts in the past and want to keep it that way there are a few options.

1) Make it so there is no password to use the machine.  If that is what you want I will let someone else answer that, or you can search for it on here I know it has been asked and answered before.

2) There is more ways to log in other than your password if you set them up.  If you go to Start -> Settings -> Accounts -> Sign-in options, you can create a Picture Password or PIN or both.  Once added any of these methods can be used to log into your account, and you only need to show them what gestures to use for the picture password or given them the PIN.

But the better way is if each person has their own account so that it keeps people out of the other people's documents and such.  And it also allows for the parent to put restrictions on children.

Create accounts for them in: Start -> Settings -> Accounts -> Family & other users

Also if you came from Windows 7 you might be use to using a "local account", as in one not connected to a Microsoft account (but there would still be a password on the lock screen).  If you like you can break the connection with the Microsoft account by selecting:

Start -> Settings -> Accounts -> Your account -> Sign in with a local account instead.

The advantages of using a Microsoft account is that you can log into the store easier, and you can sync your settings to other machines and use OneDrive.
