slance310 say:

Hi,


This is what I had to do for a X5340 in Win 8.1... otherwise reinstalling it made no difference...

Note that this is a USB printer...


One thing you might check first though is to make sure the Print Spooler in "SERVICES" is started, and Bidirectional Support is Enabled...  If this doesn't help, you will probably need to uninstall\reinstall.


To find Bidirectional Support, right click on the printer in Devices and Printers, select Printer Properties, PORTS tab, and it will be near the bottom...


I'm not sure about the USB 3.0 issue... if you haven't changed the plug on the computer, something might be getting messed up and the printer needs reinstalled.


1) Printer must be On for it to show up in Devices and Printers after the Upgrade - Just this one  time.  Otherwise, only the Lexmark Fax shows up and the Printer can't be REMOVED.


2) UNHOOK the Printer from the computer.


3) REMOVE Lexmark Printer and Lexmark Fax (if you have one) in Devices and Printers.


4) STOP the PRINT SPOOLER.


5) Uninstall software - TWO Firewall Security Messages Pop Up about "ALLOWING' Printer Communication..


Select "ALLOW".... if you Allow both Shared and Private on both windows, you end up with two printers in Printer Preferences, Properties, etc. I unchecked a "Shared" on the first window.


6) After REBOOT, STOP Print Spooler AGAIN.


7) Reinstall Software\Drivers ... Hook Printer back up when Prompted.  I used my original Lexmark    installation CD and set the Compatibility Mode to XP SP3 or Vista... both worked...


8) Unhook Printer again and REBOOT ... This step isn't necessary if Bidirectional Support is already ENABLED.


Skip to step 10 if Enabled.


9) Enable Bidirectional Support and Reboot


10) Set up Printer Preferences if desired, and it's ready to go.

I can't change my Gamertag (GT).

Most of the time if you are seeing a message saying the GT you want is unavailable, it’s true. One thing to remember is that spaces and capitalization don’t make your GT unique. For example, “Major Nelson”, “MajorNelson”, and “MaJoRnElSoN” are all the same GT.

GTs are also tied to services like Zune or Games for Windows LIVE. If a GT doesn't exist on Xbox, it might be in use on one of those services. If you try sending the GT you want a friend request and it says it’s not a valid GT that just means it isn’t used on Xbox LIVE, but it could be used for Zune or GFWL.

I heard old GTs are being re-added. When will "This GT" be available?

There is no schedule or estimated dates for when certain GTs will become available again. GTs are not an on-demand service. They will be available when they are available.
