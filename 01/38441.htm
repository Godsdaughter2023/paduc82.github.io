Pleased to know that the issue is fixed. Ideally, you get the exclamation mark on Shut down button when the Windows updates are to be configured or haven't finished getting installed successfully. To avoid this situation, you need to make sure that your system is updated with the latest Windows updates installed. Apart from that, to remove the corrupt entries of pending updates, you may also rename a file called pending.xml located in the winsxs folder as detailed below.

  1. Open an administrative Command Prompt window.
  2. At the command prompt, type the following command:
    takeown /f C:\Windows\winsxs\pending.xml
  3. Rename the c:\windows\winsxs\pending.xml path by using the following command:
    Ren c:\windows\winsxs\pending.xml pending.old
  4. Close the command prompt.
  5. Check for Windows updates and install the remaining updates (if any).
