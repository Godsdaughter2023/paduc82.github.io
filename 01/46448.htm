The disk controller is the circuit which enables the CPU to communicate with a hard disk, floppy disk or other kind of disk drive. Modern disk controllers are integrated into the disk drive. For example, disks called "SCSI disks" have built-in SCSI controllers. In the past, before most SCSI controller functionality was implemented in a single chip, separate SCSI controllers interfaced disks to the SCSI bus.

Where an error occurs, this can result in file access problems, disk writing problems or loss of data. Fixing this means reinstalling the disk controller driver that came with your computer.

You may also try to update the driver from the device manager.

Steps:

a. From desktop screen, press Windows key + X

b. Select device manager.

c. Double click on ‘Storage controllers’.

d. Right click on the device listed.

e. Select Update Driver Software.

f. Select search automatically for updated driver software.

g. Restart the computer.
