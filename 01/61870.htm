kiran jayakumar say:

Hi Mauselet,

 

‘Refresh PC’ will resolve the issue but you can keep this step as a last resort, since performing Refresh will remove all the apps you have installed. Try the following troubleshooting methods if you haven’t already tried them.

 

Check for update to see how many updates are listed, download one of the updates manually from Microsoft Download Center and try to install. To do so, make a note of the KB number when you check for updates and search it in www.download.microsoft.com

 

Rename the Software Distribution folder on your computer. To do so,

a)      Open the 'Command prompt' in elevated mode. To do that; in search, type 'cmd' (without quotes) and right-click on it to select 'Run as Administrator'.

b)      If you are using a local user account, it will prompt you to enter the password of an administrator user account to proceed.

c)      In the command prompt type the following command to stop Windows Update service: net stop wuauserv and hit enter on the keyboard.

d)     Then type in cd %windir% and hit enter on the keyboard again.

e)      Type ren SoftwareDistribution SDTemp, and then press Enter.

f)       After renaming the Software Distribution folder, to start the Windows Update service again; run the following command: net start wuauserv and press enter.

g)      Now try to install the updates and check the status.

 

If the issue persists then run System File Checker scan. This scan will check for any issues with system file and will try to fix them.

Access the command prompt in elevated mode once again and type in the following command: sfc /scannow and press enter. Note that the scan take a while to complete.
