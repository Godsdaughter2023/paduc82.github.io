irfan h say:

Hello Pinak.Tristar,

·         What is the Operating System installed on both your computer?

·         Are you logged in as an Administrator on both the computers?

Go to the computer that is sharing folder you want and do the below steps:
1. Right-click on the folder you want to share.
2. Click on "Properties" (bottom)
3. Click on "Security" tab (third from left at the top)
4. Click on "Add".
5. Click on "Advanced".
6. Click on "Find Now" (mid right side).
7. Click on the user you want to share with and then click "OK" button and then "OK" again.
8. Click "Apply" and then "OK".

If this does not work, try to ping the other computer. Turn off any firewalls for testing temporarily. If the firewall is the problem, you'll have to configure it to allow access the network.

Note: Please make sure that you enable the firewalls after performing the test to keep your computer protected.

Click on start and type cmd in the search field. Right click on cmd and ‘Run as Administrator’:

1.       PING each remote computer by IP address, and if successful, PING by name. Open a command prompt as described above and type.
PING <ip address>
or
PING <computer name>

 

A failure to PING is almost always a firewall configuration issue.

Note: You can obtain the IP address and computer name of a computer by opening a command prompt (DOS window) and typing IPCONFIG /ALL.

Turn off password protected sharing. To turn off password protected sharing, follow these steps:

1. Open Network and Sharing Center by clicking the Start button , clicking Control Panel, clicking Network and Internet, and then clicking Network and Sharing Center.

2. Under Sharing and Discovery, click the arrow button next to Password protected sharing.

3. Click Turn off password protected sharing, and then click Apply.  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

You can also refer the below articles that will give you information on ‘How to troubleshoot File sharing:

http://windows.microsoft.com/en-us/windows-vista/Troubleshoot-file-and-printer-sharing

http://windows.microsoft.com/en-us/windows-vista/Troubleshoot-network-connection-problems
