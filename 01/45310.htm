Hi NOLA. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

There is not 30 day trial for Windows. When you buy a PC with WIndows, the seller must provide a way for it to be activated, or reactivated whenever and however you want to install it. If a shop sold it to you without the proper license with this ability, then it is a theft and you should take it back. You can also call Dell as they include a one year warranty during which everything must be made right.

More likely that's a notice about a trial version of Office which many manufacturers include and normally has a 30-90 day trial during which you can create documents and then feel the need to buy it because you learn it's a trial. But you do not need to do this. You can uninstall the trial version in Settings > Apps & Features and install and use any other Office product you own in it's place. Or you can use one of the popular free Office suites that are fully compatible like Open Office or Libre Office.

johnp lee say:

How can I keep the unwanted keyboard layout from reappearing?

I’ve removed a keyboard layout from my keyboard settings, but if I later try to switch keyboard layouts using the hotkey (Windows Key + Spacebar) or by clicking the keyboard layout on the taskbar, the unwanted keyboard layout sometimes randomly reappears as one of my options, forcing me to go back into keyboard settings to remove it again.

tiero chi say:

If a keyboard layout that you previously removed automatically returns on its own, it's possible that the setting is roaming back to your device from the settings of another of your Windows 10 devices. 

To help prevent this problem from occurring:

  1. Open the Settings app then select Accounts > Sync your settings.
  2. If the Language preferences sync setting is set to On, toggle the setting to Off.  Now you should be able to remove the language or keyboard from the device without it being automatically being re-added.  Note: If you are using an East Asian language IME, be aware that this will also disable the IME’s personal dictionary syncing. In this case, you may want to turn this setting to Off across all devices, remove any unwanted languages/keyboards, wait for 48 hours, then turn the setting back to On.

To remove a language, go back to Settings, and then to Time & Language > Region & language.  Select the language you want to remove and select Remove.  Note that the top language in the list cannot be removed without first moving it further down the list. If you wish to keep the language, but remove one its multiple keyboards, select the language from the list, select Options, select the keyboard, then select Remove. Note that each language must have at least one keyboard, so the Remove button will be disabled for the final keyboard in the list.
