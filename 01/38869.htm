fix typo - Microsoft Community

susannapetrovich say:

fix typo

When I want to switch users on my home computer, one of the names is spelled wrong for the kids.  How do I fix this?

lemp say:

You can do this in (at least) two ways.  Both require that you be logged in as a user with "administrator" privileges.

1.  Click Start, type User Accounts in the Search box, and press Enter

2.  Click "Manage another account"

3.  Click the account you want to edit

4.  Click "Change the account name"

5.  Type in the new name and click the "Change name button"

The above method will change the name you see on the "Welcome" screen as well as on the Start Menu when that account is logged in.  It will NOT, however, change the name of the user directory (folder).  That is, if you use Windows Explorer (the file manager) and navigate to C:\Users, you will still see the old name listed as one of the user folders.

For example, if you had created a user account named Jhon, the file structure where that user's files are saved is:

C:\User\Jhon

    C:\User\Jhon\My Documents

    C:\User\Jhon\My Photos

    etc.

If you fix the user name to John, the file structure remains the same as before:

C:\User\Jhon

    C:\User\Jhon\My Documents

    C:\User\Jhon\My Photos

    etc.

For this reason, especially if there are not many files in the "bad" user account, you may want to create a new account with the correct name.  You do this at the same screen where you selected the account to change by clicking "Create a new account."  If you choose to do this, log into the newly-created account once so that Windows will create all of the default folders.  Then log out of that account, log into your administrator account and copy all of the files and folders from the old Documents, Pictures, Videos, etc., folders to the corresponding new ones.

The default folders in a new user account are as shown below.  AppData is hidden; in order to see it (so you can copy the old one), in Windows Explorer go to Tools > Folder Options, select the View tab, select the radio button to "Show hidden files, folders, and drives" and click OK.

Only after you have logged into the new account and are satisfied that everything got moved properly should you go back to "User Accounts" and delete the old account and its data folders.
