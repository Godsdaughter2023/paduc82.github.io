Unfortunately recovering files doesn't always work. The program will have retrieved any files it finds but that doesn't mean they are complete. They may well be files fragments. As files are written to a disk some deleted files will be overwritten and that is what you might be trying to view.

Look at the file size of the files you have retrieved. You would normally be looking for something around 2-4MB. Retrieved files that are smaller than that are either thumbnail files or just fragments. If you have the retrieved jpg files in a folder then list the files by file size and look at the large files and see if you can view those.

I assume you recovered the files to another drive (and not the one that was formatted).
