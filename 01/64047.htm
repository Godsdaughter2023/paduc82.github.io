Hi Rahul,

This issue might occur due to fast startup settings enabled on the computer. It could also occur if the Remote Procedure Call (RPC) or its dependency services are not running on the computer.

First of all, let’s make sure that the RPC and its dependent services are running in Automatic mode.

Follow the steps:

  • Press Windows+R keys and type “services.msc”. Hit Enter. This will open Services window.
  • Locate Remote Procedure Call service. Double Click on it and go to Dependency tab and check for the dependent services. You need to expand Remote Procedure call service. You would find couple of services: DCOM Server Process Launcher and RPC Endpoint Manager.
  • Close the window and look for these services. Right click on DCOM Server Process Launcher and click on Properties. Set the service to Automatic and start the service. Similarly, start RPC Endpoint Manager Service in Automatic mode. Close the services window. Restart the computer.

Check if you still receive the error message. If yes, proceed further with the troubleshooting methods provided below:

Method 1:

Let’s disable Fast startup and check if it shows all the power options under Control Panel.

Follow the steps:

  • Press Windows+X keys and select Power options.
  • Click on Change what power button does.
  • Click on Change settings that are currently unavailable.
  • Uncheck “Turn on fast startup”.
  • Click on “Save changes”.
  • Note: Make sure that you re-enable the setting after troubleshooting the issue, if you are willing to keep to fast startup option enabled.

If the issue still persists, check with the next method.

Method 2:

Let’s run SFC scan and check if it scans and fixes corrupt system files. 

Refer to the link to run SFC scan:

Follow the steps from section "Windows 8.1 or Windows 8":

http://support.microsoft.com/kb/929833/en-us 

If you still don’t find the power options, move ahead with the next method.

Method 3:

Let’s reset and restore the power plans to default settings. Follow the steps:

  • Press Windows key + X keys. Choose command prompt (admin) from the context menu.
  • In the command prompt, type powercfg –restoredefaultschemes and hit Enter.
  • Exit command prompt.

Restart the computer and check if it fixes the issue.
