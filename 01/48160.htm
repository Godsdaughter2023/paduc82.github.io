Go through your TV display menu and make sure that "Overscan" is turned off - you may have to go into advanced and/or pro settings to find this. I noticed that my Xbox One was doing the same and changing Overscan to off fixed it.

If this setting is turned off and you're still having the issue, or if you cannot find a setting for it, run the Xbox One TV calibration program in settings. You don't have to run it all the way through the colour and brightness settings etc. if you don't want to, but the first step involves formatting the display output to your screen and you can adjust the edges and margins (or the "safe zone" so to speak, similar to how most Call of Duty games ask you to set before playing).
