fergus martin say:

Hello Kyuunex,

The free windows 10 upgrade comes from the update KB3035583, which is installed through the windows updates. To get your windows 10 key, and to upgrade for free, you need to update your PC to the point where you have the GWX applet icon on your taskbar, and to reserve a copy of windows 10. From here, you will be placed into a queue to download the OS. 

It is important to upgrade this way as it will generate a unique windows 10 license key, different to the current key of the machine. Thats why your windows 8.1 key did not work. 

To get the free upgrade, please ensure that all updates have been installed (it may say that all are already installed, but unless you have the icon for Windows 10, then they have not all been installed). Once all the updates are complete, reserve a copy and select upgrade when offered. 

Another check you can to is to run these in command prompt:

dism /online /get-packages | findstr 3035583
dism /online /get-packages | findstr 2952664

These will search for the required updates, and inform you whether they are installed or not. If they are, then it is a matter of patience until the icon appears.

The creation media is used for creating installing the OS manually or creating bootable media, but it will NOT generate the license key needed for an official version of windows 10. 

I hope you find this helpful,

jeremy harding say: I think I was having the same problem as you. Error 1330 on vc_red.cab indicating the certificate was invalid. Upon further investigation, I determined that Windows just could not verify the certificate against CRLs (closed network).

I was able to work around it using the setreg utility that comes as part of the .NET Framework SDK v1.0 and v1.1

Ran the following commands
setreg 2 false
setreg 3 false
setreg 9 false

An expaination of these:

2 — Use expiration date on certificates.

3 — Check the revocation list.

9 — Check the revocation list on the Time Stamp Signer.


After the changes, Visual Studio 2010 installed perfectly.
