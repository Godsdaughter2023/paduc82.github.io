Hi Epsilon,

What the blue tooth creates is a network very similar to wifi, as I'm sure you know.

From that perspective, you might try disabling your firewall software to see if it may be preventing the profile from appearing in the Bluetooth devices, but allow operation.

Older networks used to have similar issues with computers not showing up in the network explorer.

The computers could communicate but the network profile for that computer wouldn't show for several minutes or longer.

The older operating systems would poll the network occasionally to identify the devices on the network, and often were not visible in the network view until this polling was complete.

You might try pairing the devices to see if they show up eventually, maybe an hour or so.

That could shed some light on the issue.

Also, you might checking the installed Windows Updates for the Bluetooth update and uninstall it.

You can also try booting into Safe Mode to see if the profiles are visible. It may not work, but you may be able to find them there and remove them.

If you have Kaspersky installed and can not get Windows 8 Tiles (Metro UI) to work, here is the answer...

I, like many people, have been struggling to get the new Windows 8 Metro UI (Tiles/Apps) to work. Most people who have had this problem have tracked it down to a handful of problems. I would like to share with you how I fixed the problem I was having. 

THE PROBLEM: 

Whenever I tried to open any Apps from the Windows 8 Metro UI Display, the app would open for a short time, show the logo of the App I was opening and then go back to the main screen. 

THE ISSUE:

After a lot of research and a lot of reading, I found out that the problem was with my Anti-Virus program. I was running Kaspersky Pure 2.0. The firewall that comes built in with Kaspersky was blocking the Apps from launching. 

THE SOLUTION:

Step 1) Go to the following website:   http://www.kaspersky.com/pure-latest-versions. Once on this website, download the latest version of the product. 

Step 2) Once the download has finished, make sure that you disconnect from the internet. 

Step 3) Completely uninstall the current version of Kaspersky Pure that you have installed on your computer, via the Programs and Features option in the control panel. making sure to delete all activation information stored on your PC. 

Step 4) Restart your PC. 

Step 5) Install the new version of Kaspersky that you just downloaded. 

Step 6) Restart PC.

Step 7) Once your PC has restarted, open Kaspersky. Go to Settings>Protection and make sure that on the list of features on the left hand side that Kaspersky Firewall is NOT there. If it is not there, then the Windows 8 built in Firewall has been activated and is now the current Firewall running. The Windows Firewall WILL allow the Windows 8 Metro UI App to run and launch correctly. Kaspersky will still handle most of the PC's protection, and your PC will be perfectly safe.

DMZ

  8. Select Save once you have finished to save changes.
