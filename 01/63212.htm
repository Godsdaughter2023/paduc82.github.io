Getting Windows 7 up to date. 

Since the summer of 2016, there have been issues with Windows Update for Windows 7.

And here is a slightly-modified version of Canadian Tech's advice for how to proceed with a new installation of Windows 7 SP1.  This assumes that you're using an installation DVD for Windows 7 with SP 1.  If you have a pre-installed system with SP1, or you install SP1 manually, start with step 6.

  1. Place the Windows 7 install disk in the disk reader and start your computer.
  2. Once the install process is started, choose CUSTOM.  Ignore the check box about drivers, unless you can not proceed further.
  3. When you get to the place where it asks you where to put the installation, click Advanced and delete all partitions.
  4. When the installer asks about Windows update, choose Ask me later
  5. Once Windows 7 SP1 is installed, install the following:
  6. KB3020369
        Win-7 32 bit     https://download.microsoft.com/download/C/0/8/C0823F43-BFE9-4147-9B0A-35769CBBE6B0/Windows6.1-KB3020369-x86.msu
        Win-7 64 bit https://download.microsoft.com/download/5/D/0/5D0821EB-A92D-4CA2-9020-EC41D56B074F/Windows6.1-KB3020369-x64.msu
  7. KB3172605
        Win-7 32 bit,     https://download.microsoft.com/download/C/D/5/CD5DE7B2-E857-4BD4-AA9C-6B30C3E1735A/Windows6.1-KB3172605-x86.msu
        Win-7 64 bit,     https://download.microsoft.com/download/5/6/0/560504D4-F91A-4DEB-867F-C713F7821374/Windows6.1-KB3172605-x64.msu
  8. Open Windows Update, change Windows Update setting to Never check for updates
  9. Do not install anything else at this point – NOTHING
  10. Start the update process.  It is likely to include about 200. It will take some time.
  11. Once you have a list of updates, you need to prevent certain specific updates from being installed to prevent Windows 10 from taking over.
  12. Click once on each Update that is NOT labeled SECURITY and check the date of issue on the right.  If that date is after January 1, 2015, Right-click on it and hide.
  13. Keep running Windows Update again and again till it offers no new ones.
  14. Start Internet Explorer, click the gear (upper right) in IE11 and select Compatibility settings and enter Microsoft.com in the list
  15. Start Windows Update and click the link to include updates for other Microsoft software.
  16. Run Windows Update again and again until you are satisfied you have all the updates you want.
  17. After the install is complete, check Device Manager. Type device in the text box above the start globe when you click it and choose Device manager from the list to find out if Win7 was able to supply the drivers you need.  You should get drivers only from the maker of your computer or Intel.  Its best to get all the drivers at this point.  Win7 install may have installed all of them or nearly. Do NOT use any of those driver download sites.  They are all bogus, have bad drivers, and install malware.
  18. Install your Microsoft Office software and then run Windows Update again and again till no more are proposed.
  19. Defragment your drive.  Type defragment in the text box above the start globe when you click it once.  Choose the Defragmentation link.  Wait till it completes all passes..
  20. Next, you will need to take steps to prevent Windows 10 from taking over.  First step is to change the Windows Update setting to NEVER. Second step is to download and install GWX Control Panel.  http://ultimateoutsider.com/downloads/
  21. New install tip:

After Windows 7, system drivers and all updates are installed and any stable applications like Microsoft Office are installed and updated, and before any data or dynamic applications are installed such as antivirus software, create a system image.  It will take 3 or 6 DVD +Rs (not -Rs) and about an hour.  When you are done you will have a very nice bit of insurance.  Should you ever again need to re-build a corrupted system or replace a hard drive, you will have a precise duplicate of your system as it was at this point.  You can restore that image to a hard drive in about 20 minutes.  Creation of System Image is found in your menu under Maintenance, Backup and Restore.

Another great feature about using the image is that you do not need an install disk or a product key to do the re-install the next time, and you will have saved yourself all the time you put in this time.

I emphasize the need for PLUS R DVD blanks.  Do not use the more common MINUS R DVD blanks.

[If you have an external hard drive, it will be faster to create the system image there.  Of course, an external hard drive is somewhat more prone to an unexpected failure than DVDs that you burn yourself.]
