Change Sticky Notes Font
Windows uses the Segoe Print font. Changing the default is not supported in Sticky Notes. What you can do is type your text in the desired font in a text editor like Word, and copy-paste it here. The Sticky Note should display the text in that font. Let us know if this works for you.
Try:
1. Open a Word Document.
2. Type letters with the font style you want to use in Sticky Notes.
3. Copy the words you typed then paste it to Sticky Notes. The following characters you'll be typing next will now be the same with the font style you used to copy from the Word document.

To reset your Surface from a USB recovery drive:
Step 1:    Make sure your Surface is turned off and plugged in (hold down the power
button for 30 seconds).
