tigerhawkvok say:

I did the nuclear option -- I blew up the index and rebuilt it from scratch, using good registry keys from another computer.

First, go to 

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Search

Right-click the key, then go to "Permissions". From there, click "Advanced". 

Then, go to the Owner (at the top) and change the owner to yourself. Check all boxes about replacing inheritiable permissions/owners. Hit OK.

At this point, it'll seem nothing has happened. Delete the whole key.

Now, go download this registry file:

https://drive.google.com/file/d/0B6LJvOj7KC-cZi1MbVRvMUZwNlU/view?usp=sharing

MD5 hash: 6D8911AC08105AA8532F3E02BB97B16E

This is a registry key for system files. Do not use it if the MD5 does not match.

I pulled it from a fresh Windows 10 upgrade on a Surface Pro 3, and it seems to work fine on my custom desktop.

As soon as I did that, I was able to do Flag+R, type "Control Panel", search "Index", and from there, go to "Advanced" and have all my options again. I rebuilt my index to be safe, and ran

sfc /scannow

to ensure system consistency.
