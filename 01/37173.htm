1. Move the contents of the c:\users folder to another location 2. mount a second hard disk to the c:\users folder 3. move the former contents back to the c:\users location so that it is now on the second drive but still original location as far as the operating system is concerned?
tryx3 say:

The facility for doing what you want is built in already but the procedure you describe might have unintended consequences as there are a lot of hidden folders within Users that the system needs.  I think you'd corrupt your user profile doing it your way.

Instead, after you have created suitably named users' folders on your additional hard drive,

Right-click on each of the visible folders within each C:\Users\EachUserName then select the location tab & tell it where to put the folder e.g. M:\Users\Chris\Documents  M:\Users\Chris\Desktop

As this facility is built-in to the system it is less risky than other ways fo doing the job.
