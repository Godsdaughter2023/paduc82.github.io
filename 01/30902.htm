Hello Sweetb132,

 

To share folders with other users you will need to edit the Permissions settings of those folders. Follow these steps to give other users access to a users documents. (for this example I’m going to share my Documents folder with another user account on this machine)

·         Log into an Administrator account

·         Click the Start Orb

·         Click Computer

·         Open C: (or the partition you want to share)

·         Open Users

·         Open your profile folder

·         Right click Documents

·         Click Properties

·         Click the Security tab

·         Click Advanced

·         A new window will open with a new Permissions tab, click Change Permissions…

·         Click the user you want to add permissions for

·         Click Edit…

·         Here you can set what exactly this user will be able to do with this folder

·         Once you have picked your options click OK

·         Check the Include inheritable permissions from this object’s parent box (if not already checked)

·         Check the Replace all child object permissions with inheritable permissions from this object box (if not already checked)

·         Click OK

·         Click OK

·         Click OK

 

For the user you added permissions for you will need to either add that folder to one of their libraries, or create a shortcut on their desktop to that folder. (You’ll have to navigate through Windows Explorer aka what you were in when you clicked Computer earlier)

 

That list looks long and complicated, but once you’ve done it a few times it becomes much quicker.

 

Hope that helps, look forward to hearing back from you.
