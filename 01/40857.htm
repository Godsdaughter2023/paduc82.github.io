sirbu say: Thanks but problem solved.  The "open" command for .mp4 files in the Registry needed to be modified from "C:\Program Files\Windows Media Player\wmplayer.exe" "%1" to "C:\Program Files\Windows Media Player\wmplayer.exe" /Open "%L".  Simple really!  Then .mp4 files plays ok with installed "ffdshow" codecs.
