Hey Linus -- I think I'm understanding better now.  The setup you see is "correct" in that what you see is the default.  But you can tweak much of it.

Before Windows 10, there was only one taskbar and it could only show on one screen (it didn't necessarily have to be the main/primary monitor, since you can unlock and drag the taskbar to another screen).  

But in Windows 10, there's two taskbars (it doesn't really span multiple displays, but it tries to look similar to that).

The main taskbar is the only one that can ever show the full-size search box, "notification" tray icons, pinned people, the touch keyboard button, or the action center button.  You can see in my example that my left monitor is where I have my primary taskbar, so all those features show up there.   This is also which screen will see the Start menu pop open if I ever tap Ctrl-Escape or the WinLogo on the keyboard.

Under personalize you can decide how it shows the running program icons (which taskbar, or both).  And if you want the main taskbar to be on another screen, you can unlock it and then drag it there.
