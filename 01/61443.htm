hairyfool say:

Unfortunately the only real way to do this is via registry editor.

If you look at the line for the entry you want to remove it will list the registry path which is holding the instruction. You would need to navigate to that location and delete the entry.

As always warned editing the registry should be done with caution and a backup taken 1st. If you are not sure then don't do it.

There is one caveat. If you have removed the line (unchecked it)  then it will not be there. In order to stop it loading MsConfig moves the entry to an inactive backup key. In order to find it at the location you would have to make the line active.

Because it has been moved out it is not processed during startup so other than trying to keep it tidy there is no specific need to delete the line. In fact if the line points to a non existent file or path then it has no effect even if left. Registry processing si much more optimised in Windows 7 and orphaned entries have no measurable effect on boot times. The timeout delays that used to happen have been resolved.

Wireless Security

Login Information

  • Gateway - 192.168.0.1
  • Username - cusadmin/chtruser/chtrtech
  • Password - password/charter

 

Wireless Passphrase

  1. Open a web browser.
  2. In the address bar at the top type in 192.168.0.1
  3. If asked for a user name and password use (cusadmin/password). If your ISP is Charter you can use chtruser/charter or chtrtech/charter.
  4. In the menu at the top, select Setup.
  5. Under the Wireless section on the left, select Security.
  6. Look to see what type of passphrase is enabled from the four drop-down boxes at the top.
  7. For a WPA key, the passphrase will be listed in the box that says WPA Pre-Shared Key. Verify which key the gateway is using by looking at the drop-down box that says current network key. For a WEP key, look in the box that says Network Key 1.

How to Change Wireless Security

  1. Open a web browser.
  2. In the address bar at the top type in 192.168.0.1
  3. If asked for a user name and password use (cusadmin/password). If your ISP is Charter you can use chtruser/charter or chtrtech/charter.
  4. In the menu at the top, select Setup.
  5. Under the Wireless section on the left, select Security.
  6. Enable desired security type. (WPA-PSK, WPA2-PSK)
  7. To turn off wireless security, select Disabled for all security types.
  8. Select Apply.

    andre da costa say:

    Full version (Retail):

    - Includes transfer rights to another computer.

    - Can change the motherboard

    - 90 days of technical support

    - Doesn't require a previous qualifying version of Windows.

    - Expensive

    OEM System Builder version:

    OEM versions of Windows 10 are identical to Full License Retail versions except for the following:

    - OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

    - OEM licenses are tied to the very first computer you install and activate it on

    - OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

    - OEM versions cannot be used to directly upgrade from an older Windows operating system

    You can reinstall and reactivate Windows 10 either full retail or OEM System Builder as many times as you like on the same computer. 

    When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

    Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate.

    There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already ownto reactivate if necessary. or use the reset function in Windows 10.

    If you are prompted to enter a product key, click 'I don't have a key' and 'Do this later'
