zilo say:

Check to see if you have update: KB971029 and read the article below.


In Windows XP, Windows Vista, and Windows Server 2003, AutoRun entries were populated for all devices that had mass storage and had a validly formatted AutoRun.inf file in the root directory. This included CDs, DVDs, USB thumb drives, external hard disks, and any volume that exposed itself as mass storage. This update disables AutoRun entries in AutoPlay, and displays only entries that are populated from CD and DVD drives. Effectively, this prevents AutoPlay from working with USB media.

Many existing devices in market, and many upcoming devices, use the AutoRun feature with the AutoPlay dialog box to present and install software when DVDs, CDs, and USB flash drives are inserted.
Users who install this update will no longer receive a setup message that prompts them to install programs that are delivered by USB flash drives. Users will have to manually install the software. To do this, users click Open folder to view the files, browse to the software's Setup program, and then double-click the Setup program to run the program manually.
Some USB flash drives have firmware that present these USB flash drives as CD drives when you insert them into computers. These USB flash drives are not affected by this update.

The main reason for Update KB971029 was security.

Certain malware, including the Conficker worm, have started making use of the capabilities of AutoRun to provide a seemingly benign task to people – which masquerades as a Trojan Horse to get malware onto the computer. The malware then infects future devices plugged into that computer with the same Trojan Horse.

While presenting an AutoRun task in AutoPlay has been available since Windows XP, we have seen a marked increase in the amount of malware that is using AutoRun as a potential method of propagation.

 

You can remove the update at your own risk!

If you do not have this update check here: Control Panel, AutoPlay. Under Devices, your camera should be listed. Choose your option from the drop down menu.

If you have this update do this: Right click Computer on the start menu and click Open. Right click your camera and choose Import pictures.

The taskbar has a "visible" property that any program can request.  If a notification tray icon adds itself near the clock, for one example, the taskbar will stay visible until you have clicked on that icon.  Same if any program has requested itself as "needy", but that one is more obviously because it flashes the icon (usually orange but that depends on your theme).

Next time it occurs, go to the tray icons near the clock and click each icon, then click the desktop (or anything away from the taskbar) and see if it hides.
