Thank you for posting your query in Microsoft Community Forum!

 

Based on the issue description, it appears that you are getting an error “0x800423F3” while trying to perform backup.

 

I will certainly help you out with this issue.

 

This error occurs due to improper registration of a WMI dll (wmiutils.dll). This dll might have been unregistered manually or by a third party program such as a registry cleaner tool.

 

Let’s try these methods and update us with the results.

 

Method 1:

 

I would first suggest you to run the following commands and check if it helps.

 

a. Click the Start button.

 

b. From the Start Menu, type Run in start search and press enter.

 

c. Type: regsvr32 wmiutils.dll (You should get a prompt that the file was registered successfully)

 

d. Type: net stop winmgmt (press y to stop the service)


e. Type: net start winmgmt (automatically starts the service)

 

 

Method 2: Check the Volume Shadow Copy service.

 

a. Click Start, type services.msc and hit enter.


b. From the list of services, double click on Windows Backup and Volume Shadow Copy services.


c. In the Properties window, select Automatic as the startup type and click Start.


d. Apply the changes and check with the issue.
