Follow these methods.

Method 1: Try to run the Network Diagnostics tool. Follow these steps:

a)    Open Control Panel and click on Network and Sharing Center.

b)    Click on troubleshoot problem.

Method 2: Try to release and renew the IP address on both the computer which has same IP address. Follow these steps:

a)    Click on Start, type cmd in the search box. When cmd.exe appears in the list above the search box, right-click on cmd.exe and select ‘Run as administrator’.

b)    In the C:\ prompt, type ipconfig and press Enter.

c)    Type ipconfig /release and press Enter.

d)    Type ipconfig /renew and press Enter.

Your computer will attempt to obtain a new IP address from the network. This may take a few minutes.
