This issue can occur because other security application is used or the Windows Defender service has stopped or Windows Defender components are corrupted.

 

Do you install any new program recently? Windows Defender is not designed to protect against the full breadth of malicious software. If programs like Microsoft Security Essentials or AVG 9 and Windows Defender are both running, your system may experience performance degradation and other problems caused by the conflict of two services providing real time protection simultaneously. I recommend keeping Defender disabled if another full protection security application is running. These security software's stops windows defender from functioning properly inorder to prevent conflicts.

 

If no such security application is installed, I suggest we perform the following steps to resolve the issue:

 

Step 1: Verify the Windows Defender service
=================================
1. Click Start, in the Start Search box, type: "services.msc" (without quotes) and press Enter. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
2. Double click the service "Windows Defender".
3. Click on the "General" tab; make sure the "Startup Type" is "Automatic". Click on apply. Then please click "Start" under "Service Status" to start the service.


If the issue is not resolved, please refer to Step 2:

 

Step 2: Register related DLL files
========================
1. Click Start, type "cmd" (without quotation marks) in the Start Search box. Right-click CMD listed above and click "Run as administrator".

 

2. In Command Prompt, type in the following and press ENTER at each command:

 

regsvr32  atl.dll

regsvr32  wuapi.dll
regsvr32  softpub.dll
regsvr32  mssip32.dll

3. Close Command Prompt and run Windows Defender again.

edwardhagemeier say:

Pretty old post but in case someone looks for a fix. It still happens to me now and then that some links refuse to delete/remove from Quick Access... I think it's because the target files move/delete/change. I have had one of these in the past and when I recreated the target file, the shortcut removed with the right click / remove. I use the following to force kill all of the pinned folders and files... it can be refined to only delete one or the other by only deleting certain cache files in the folders and not all (*), but I just kill them all and start from scratch. It does also remove the recent files under pinned apps like Excel etc... so the disclaimer is it will start you from a clean slate...

Run the following commands in an admin cmd prompt or save in a batch file and run;

del /F /Q %APPDATA%\Microsoft\Windows\Recent\*
del /F /Q %APPDATA%\Microsoft\Windows\Recent\AutomaticDestinations\*
rem del /F /Q %APPDATA%\Microsoft\Windows\Recent\CustomDestinations\*
taskkill /f /im explorer.exe
start explorer.exe
