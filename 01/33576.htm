This error message can be due to any number of reasons, including:

The following procedure is for accessing and running “Startup Repair Tool” from the Windows 7 Installation DVD:

Warning:
This procedure will not resolve the problem if it is due to a hardware failure.

Notes - Startup Repair Tool:
i) You must first remove any Flash Drives or other USB Storage devices, like External Drives, from your computer before running “Startup Repair Tool”.
ii) If “Startup Repair” finds problems, it will attempt to repair them and your computer may restart several times during this Repair process.
Do not Boot from the Installation DVD [or “System Repair Disc” CD if you you used this] on any Restart. If you do, you will need to restart immediately so the Startup Repair can continue normally.

1) Upon turning on machine, keep pressing the “F2” function key (or follow screen instructions) to open the “BIOS”.
Make the DVD drive the First Boot Device, and exit saving changes.

2) Insert Windows Installation DVD.
The DVD has to be spinning and recognised for the system to “stop at it”, so if the system does not see it first time, leave the DVD in drive, and restart the machine!

3) Following displays:
‘Press any key to boot from CD or DVD…_’.
Press any key to proceed.

4) ‘Install Windows – Enter your language and other preferences’ Screen:
Choose:
Language/
Time and Currency format/
Keyboard or Input method/
Click “Next”.

5) ‘Install Windows – Install now’ Screen:
LOOK AT THE BOTTOM-LEFT OF THIS SCREEN!
Click: “Repair your computer” (bottom-left).

6) Wait for System Recovery Options to locate Windows 7 on your computer.

7) ‘System Recovery Options’ Wizard – Use Recovery Tools or Restore your computer, Screen:

Warning:
If you do not see your OS listed, you might need to load your Hard Disk Drivers, by
Clicking: “Load Drivers” button.

Leave the default Bolted as: “Use recovery tools …”

Note:
Do not worry if the Drive Letter in ‘Location’ column does not match the Drive Letter on which you know Win 7 is installed. Drive Letters are somewhat dynamic, especially when using diagnostic tools like “System Recovery Options”.

When ready, Click: “Next”.

Choose "Startup Repair".

9) ‘Startup Repair – Startup Repair is checking your system for problems…’ Screen:
Wait while “Startup Repair” checks your computer for problems and attempts to Repair them.

If “Startup Repair” finds a problem with an important OS file, the Tool may suggest a solution that you may have to confirm, or may solve the problem automatically.
Whatever happens, follow the prompts as necessary and accept any suggested changes.

10) ‘Startup Repair – Your computer was unable to start’ Screen:

If “Startup Repair” did not find any problems, you will not see this Screen.

If “Startup Repair” found problems, it attempts to repair them.

Click "Finish".

Shut down the machine fully. Startup the machine and as it does so, keep pressing the "F8" function key until the 'Advanced Boot Options' screen appears.
