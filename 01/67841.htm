By what method is this ?
The best way to install drivers successfully is to download the installer from the
mfg's web site (link above), and after uninstalling the current drivers via
Programs and Features , re-boot the machine and install the new downloaded
drivers.

Using various automatic driver install methods (Windows Update, etc.) is not recommended.
The exception to this is when Windows installs generic Windows drivers after you
uninstall the drivers specific to your device (sound card, graphics card, input devices, etc. ).
These Windows drivers are only basic drivers, and drivers from the mfg of the device still need to be installed for full functionality of that device.

Uninstalling drivers via Device Manager is good for Windows generic drivers that
may be corrupt ( ie- USB. HDD's , etc.)  as Windows will re-install them after re-booting, but this is not a good way to install most hardware specific drivers.
