Hi,

Did you run a chkdsk /r to repair problems? Or just the standard chkdsk to detect problems? The former has to be initiated from an elevated command prompt and, as it will be unable to lock the disk, have to be scheduled to run on a system restart.

For an elevated prompt, click start and type CMD. Right click the menu entry that appears and choose 'run as administrator'. Click continue at the UAC prompt. From the command line, run:

chkdsk c: /r
