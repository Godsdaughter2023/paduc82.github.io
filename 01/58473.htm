    Where Can I Find The Times When I Turned My PC On And Off? - Microsoft Community

    cpmaster say:

    Where Can I Find The Times When I Turned My PC On And Off?

    I run Windows7. Where can I find the times when I turned my pc off and on?

    ronnie vernon say:

    Hi

    You can see the information for each time you Logged On and Logged Off on the computer.

     

    Open Control Panel / Administrative Tools and click the Event Viewer.

     

    In the Event Viewer, navigate to the following location.

     

    Applications and Service Logs / Microsoft / Windows / User Profile Service / Operational.

     

    Look for the <date and time> Source: User Profile Service > Event ID 2.

    Event ID 2 records the date and time each time you log on to the computer.

    Finished processing user logon notification on session 1.

     

    The Event ID 4 records each time you log off of the computer.

    Finished processing user logoff notification on session 1.
