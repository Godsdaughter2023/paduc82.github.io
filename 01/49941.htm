Device Manager displays the following message in the Device Properties dialog box, on the General tab, in the Device Status text box: This device is not working properly because Windows cannot load the drivers required for this device. (Code 31)

The issue can be due to the following reason:

Windows cannot load the driver. If the device is a Microsoft 6to4 adapter, the installed network adapter driver may not support the Microsoft 6to4 adapter.

To work on the issue, refer these methods.

Method 1:

If you see the error code associated with a Microsoft 6to4 Adapter on a computer running Windows 7 in a home environment, you may be able to uninstall the 6to4 adapter to resolve the problem.

To uninstall the 6to4 adapter refer these steps.

a) Click Start
b) Type ‘Devmgmt.msc” without quotes and hit Enter.
c) The Device Manager will display all installed and “found” hardware on the computer.
d) Find the Network Adapters Category and click the "+" to the left.
e) The Network Adapters should be listed if installed. Right click the adapter used, select Properties
f) Navigate to the Driver tab in the Network Adapter properties. (6to4 adapter)
g) Selecting the Uninstall button will remove the existing driver from the device manager.
h) Confirm the Uninstall.
i) Close device manager.

After you perform method 1, run an automated troubleshooting service from Method 2.

Method 2:

Run the fixit from the following link.

Hardware devices are not working or are not detected in Windows
http://support.microsoft.com/mats/hardware_device_problems/en-us

Check if it helps.
