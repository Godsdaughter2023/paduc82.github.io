  1. Open the Metro screen (Modern UI screen) of the Windows 8 application and click on search.

  3. Click on create basic task from the list of actions which will be shown on the right side of the popup window.
  4. A new window will open of create basic tasks wizard; give the name of the action you want to perform and click on the Next button.

  7. Select the radio button start a program and click the Next button.
  8. Just put shutdown in the script box, the -s t 55, which stands for shutdown timer 55 seconds in Add arguments and click the Next button.
  9. A popup window will open; click the yes button.
  10. Now the scheduling of the process is completed; click on the Finish button.
