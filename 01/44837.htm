¡firedog say:

Folders don't normally disappear, but they do sometimes get lost. When viewing your message list in Windows Live Mail, press Ctrl-Y to show the entire folder tree. Look in each folder for the missing ones; they may be buried several layers deep, e.g. in a folder called Recovered items. If this folder exists, it won't be empty unless you emptied it:

For your store folder to have Previous versions available, you should include it in File History. 

You can't recover missing folders in the file system (e.g. by messing around in the store folder C:\users\username\AppData\Local\Microsoft\Windows Live Mail, which is hidden by default for a reason); the same messages are present there as you can see win Windows Live Mail, but the folders you see in the program don't necessarily appear as they do in Explorer. This is because the ones you see in Windows Live Mail are virtual folders showing what you want to see; the message database index keeps track of which messages and folders are where, so there will often only be a superficial resemblance between what you see in the program and what you see in Explorer.
