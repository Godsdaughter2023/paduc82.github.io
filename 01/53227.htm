Hi Daniyal. II'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

To show libraries in File Explorer, select the View tab, and then select Navigation pane > Show libraries.

This is easiest handled by funneling all of your Pictures into their User folder at C:\Users\Your User Name\Pictures so they're not spread around in various folders which even if added to a Pictures library still are separate.
