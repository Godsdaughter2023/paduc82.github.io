I would like to inform you that as per the Windows Compatibility Center and documentation for Microsoft LifeCam VX-1000, there are no drivers for Windows 8. This device is not compatible with Windows 8.
However, you can download and install the drivers in the previous Operating System's compatibility mode. Here is the link from where you can download the drivers for the previous Operating System.

http://www.microsoft.com/hardware/en-in/d/lifec...



Follow the steps to install in Compatibility mode:

a. Right click the driver installation file and select Properties then tap or click the Compatibility tab.

b. Place a check in the Run this program in compatibility mode box.

c. Tap or click the drop down box and select a previous Operating System, then tap or click OK.

If that does not work, download the following driver:

https://www.monwindows.com/forum/download/file....

Extract it...

Press Windows key + X
Click Device Manager

Expand Camera or Imaging Device
Click Update driver
Browse to the folder containing the extracted driver file then see if it works.
