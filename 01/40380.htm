j w stuart say:

Thats a good question and according to the Microsoft documentation it seems to imply that your destination drive must be somewhere between 30 and 50% larger than the Windows partition. However I'm in the process of finishing up my next Windows 7 Korner article for pagestart.com and I have been able to backup a 100GB Windows 7 partition (of which only about 25GB is used) to a 32GB partition. This implies the actual partition size no direct bearing on the size of the USB drive.

There are some variables that come into play, one of which is how much of the data on the Windows partition consist of your 'Documents', 'Photos', Etc. The more personal data you have the quicker you are going to file up that external USB drive partition.

So if your 170GB Windows partition has 130GB of used space (mostly .png or .jpg photos for example) then you can expect the amount of space required to perform your first backup will consume about250GB of space.

The only time Windows Backup will backup the entire contents of the 750GB drive is if you place a check mark to include each of the other partitions on the drive. Even then if the remaining partition(s) only includes a small amount of data (say 100GB) then you can expect the total size of the backup to increase by an additional 100GB. So the250GB value I mentioned will now be about 350GB.

Error code 31indicates that this device is not working properly because Windows cannot load the drivers required for this device.

Recommended resolution

Update the driver

You may be prompted to provide the path of the driver.

 Windows may have the driver built-in, or may still have the driver files installed from the last time that you set up the device. However, sometimes, it will open the New Hardware Wizard which may ask for the driver. If you are asked for the driver and you do not have it, you can try to download the latest driver from the hardware vendor’s Web site.

In the device Properties dialog box, click the Driver tab, and then click Update Driver to start the Hardware Update Wizard. Follow the instructions to update the driver.
