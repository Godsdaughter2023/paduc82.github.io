Hi cheselz,

Placing two or more audio files in the same position is not available in Windows Live Movie Maker. As a work around you can import the video/photos then add Music 1 and then save it as a Movie (wmv) to create video with embedded audio. Create another project using this movie, then add Music 2. However, this will limit the adjustments you can do on the project since you'll use a finished movie. Still, I'll be providing the steps for this work around, in case you consider to perform it:

CREATING VIDEO w/ MUSIC 1

1. Import video, stills, photos in Windows Live Movie Maker.
2. On the Home tab of your Windows Live Movie Maker, click Add music, select (Music 1).
3. Modify project according to your preference. Once finished press Alt + F then select Save Movie. This action will turn your project (wlmp) to movie (wmv).

ADDING MUSIC 2

1. Import the created movie which has Music 1.
2. Add Music 2 in location you wish to add it.

Hi,

 

This issue happens if VBScript (Visual Basic Script) is not properly configured on your computer or if the Windows registry contains an incorrect entry for VBScript.

 

I would suggest you to try the steps mentioned below and get back to us with the results.

 

Method 1: To fix error 2738

 

Before you begin in fixing the error, check whether you’ve Windows 7 32 bit or 64 bit installed on your computer.

 

http://windows.microsoft.com/en-us/windows7/32-bit-and-64-bit-Windows-frequently-asked-questions

 

 

Registry disclaimer

To do so: Important this section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:

322756 (http://support.microsoft.com/kb/322756/ )

How to back up and restore the registry in Windows.

 

 

Windows 7 x86 (32-bit) Operating system

 

a. Open a command line window as Administrator, by clicking Start, typing cmd in the search box, then right-clicking on the cmd.exe program and choosing Run as Administrator.

b. Next paste the following in the command line and hit enter.

reg delete "HKCU\SOFTWARE\Classes\CLSID\{B54F3741-5B07-11CF-A4B0-00AA004A55E8}" /f

The above command removes the incorrect entry for VBScript support on 32-bit Windows 7.

Do not close the command prompt and follow next step.

c. Type the following command and press enter to register Visual Script Dll (Dynamic Link Library)

regsvr32 vbscript.dll

 

Now try to install skype and check if it helps.

Now try to install Skype software.
