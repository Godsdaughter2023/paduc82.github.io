techfreak_ say:

VoIP phones are generally incompatible with the standard version of Skype. The implementation Skype uses is proprietary and would require current hardware that has built-in support or a standard OS. Since most if not all older devices with such support are no longer compatible or supported, it would be best to locate a device that supports the latest Windows, iOS, or Anroid OS. This way there is a higher chance a standard Skype client can be installed on it.

Skype Connect and Skype for Business are two other services you can use with VoIP/PBX hardware that have their own configurations/networks.
