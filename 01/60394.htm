xircal say:

Hi,

If it's only happened just recently, you could try running System Restore to wind back the clock to the time prior to when the problem started.

To do that, hit Windows key + X and click System.

In the menu which opens, click the System Restore button and then click Next. Select a Restore point from a time when you know sound was working properly and then click Next and follow the prompts.

If you can't see enough Restore points, checkmark the box to show more.

When System Restore runs, the machine will reboot. At some particular point you'll be looking at a totally black screen where nothing appears to be happening for what seems like quite a long time. But eventually, the usual lock screen will appear and you can login again.

When the machine boots to the desktop, you'll see a message as to whether the restoration was successful or not.

Be aware though that any apps or other software you installed after the Restore point you select to restore to will have to be reinstalled.

Also, see this link from a forum where the user had a similar problem with AOL mail notification in case it's relevant: http://www.eightforums.com/sound-audio/40094-aol-mail-notification-sound-suddenly-gone.html
