a. user say:

Hi Deon,

As per the information related to the events and the error message is concerned, it could occur due to the display driver or, to be precise, NVIDIA driver installed on the computer. Here, "nvinitx.dll" is a

dynamic linking library which is a part of the NVIDIA graphics card drivers.


What is the Make and Model of the computer or the display card driver?

If NVIDIA driver is installed on the computer, I would suggest you to check if the same issue occurs in safe mode with networking. Follow the steps:

Step 1:
Refer to the link to boot the computer in safe mode with networking:

http://windows.microsoft.com/en-in/windows-8/windows-startup-settings-including-safe-mode


If the issue doesn't occur in safe mode with networking, I would suggest you to refer to the computer's or display card manufacturer's website to download and install the latest display driver. But, before installing the latest driver, uninstall the current version of display card driver.
Check with the step 2.

Step 2:
Follow the steps to uninstall the current version of display card driver and install the latest ones:
a) Type Device manager on the Start screen and click on Device Manager.
b) Locate the display adapter and expand the driver icon. Right click on the driver and select Uninstall.
c) Follow the onscreen instructions to uninstall the driver.
d) Locate the folder that has the setup file for the latest driver that you have downloaded. Double Click on the setup file and follow the onscreen instructions to install the driver.


For reference, you can refer to the link:

http://windows.microsoft.com/en-in/windows-8/find-install-display-drivers


Please let us know about the status of this issue.
