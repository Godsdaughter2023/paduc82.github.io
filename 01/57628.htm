Okay after the update this morning I was dead set on setting up DLNA to watch my MKV files so I went ahead and deployed across as many platforms as I could including Windows Phone 8 and Windows Server 2008 r2 and 2012 windows 8 windows 8.1, and one Android Tablet.  All you need to deploy is the Media Player App from the XBOX ONE App Store, and a LAN connected to your XBOX ONE and your DLNA device of choice.


DLNA Win 8.1 AND  Windows Server 2012 Enable
 
 1) Open Charms > Search > "Stream"
 2) Select Media Steaming Options
 3) Click Turn on Media Streaming
 4) Choose Devices to Allow or Block

To Add or Remove folders

 1) Open File Explorer
 2) Expand Libraries (right click left pane to select or deselect libraries from view)
 3) Any system folder located within these directories will be shared
             to ADD :
                 1) Select Folder from folder tree or explorer
                 2) Right Click Select Include In Library > Select Library
            to REMOVE:
                 1) Select folder from LIBRARIES tree
                 2) Right Click Select Remove location from Library    

DLNA Win 8.1 Disable

1) Go to Network and Sharing Center (right clicking your Ethernet/Wifi icon in your taskbar)
2) Click HomeGroup (lower left corner)
3) Click Change advanced sharing settings
4) Click on the arrow next to All networks profile
5) Click Choose Media Streaming Options
6) Click Block All
7) Click OK

DLNA Server 2012 Disable

1) Go to Network and Sharing Center (right clicking your Ethernet/Wifi icon in your taskbar)
2) Click Change advanced sharing settings
3) Click on the arrow next to All networks profile
4) Click Choose Media Streaming Options
5) Click Block All
6) Click OK


DLNA Win Server 2008 R2

3) Add Feature "Desktop Experience"
5) Restart
