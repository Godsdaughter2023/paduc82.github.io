ken blake say:

Windows previous versions isn't the issue. They're not there.

There are two potential issues:

1. The recycle bin. Normally deleted files go to the recycle bin, from where they can be recovered. But if you delete with shift-delete, they don't.

2.  "Deleting" a file doesn't actually delete it; it just marks the space as available to be used. There are third-party programs that can sometimes recover deleted files. If that available space hasn't been used yet, they can be recovered using one of hose programs.

However for someone to use such a file recovery program to look for your sensitive programs, he would have to want to find them, know they are there, and know how to use such a program. If you're concerned about a thief, that would be highly unlikely.

So I wouldn't worry about this the way you apparently do. But if you want to be very sure deleted files are unrecoverable, you should use a third-party program that actually deletes deleted files by overwriting their space. Dban is one such program, but a search can find others. However these are mostly used on entire drives rather than individual files. Which, if any, can be used on an individual file, I'm not sure.
