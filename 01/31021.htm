Hi,

1.       Which operating system is installed on the system?

2.       Have you made any changes to the system prior to the issue?

 

In case you are using Windows 7 then you may try the troubleshooting steps provided below.

You cannot start over with new library in Windows media player but you may clear the Windows Media Player database. To do this, follow these steps:

 

a)   Exit Windows Media Player.

b)   Click Start, click Run, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.

c)    Select all the files in the folder, and then click Delete on the File menu.

Note: You do not have to delete the folders that are in this folder.

d)   Restart Windows Media Player.

 

 Note Windows Media Player automatically rebuilds the database.

 

If the above does not fix your issue then clear the Windows Media Player database cache files. To do this, follow these steps:

a)   Exit Windows Media Player.

b)   Click Start, click Run, type %LOCALAPPDATA%\Microsoft, and then click OK.

c)    Select the Media Player folder, and then click Delete on the File menu.

d)   Restart Windows Media Player.

cbgrant [msft] say:

The quickest way to uninstall apps in Windows 10 is to find the app you want to delete in the start menu app list and then to right click. A pop-up menu should appear with a couple of options. One of the options should be "Uninstall" (Not all apps have the option to uninstall due to the fact that they are necessary for windows to run smoothly, but most are uninstall-able). Once "Uninstall" is clicked the app should be uninstalled automatically with no further involvement needed.

When older apps (non universal or metro apps) are uninstalled however a Control Panel window will appear with a list of all installed (non-universal and metro apps) apps and you will need to find the app in the list that you wish to uninstall, select the app, right click and select "Uninstall".
