    a. user say:

    I don't have fancy titles to impress you, but I can tell you that the cooling fans are monitored by the BIOS in older computers and the UEF in newer computers. If the BIOS or UEF detects that the cooling fans haven't started, you'll see a message to that effect from the BIOS or EFI, and you'll see it before Windows starts. Windows is not in control of the case fans.

    I haven't had this happen to me personally, so I don't know whether a computer will pause when this happens or continue on. I'm going to guess that the computer will pause and make you press some function key if you want to continue.

    The BIOS can't start a fan that hasn't started on its own, so there's really nothing that you can do except to replace the fan in question.

    Some motherboards can detect the heat inside the case and vary the cooling fans' speed accordingly.  (Without this control the fans always run at top speed.) A computer like that has cooling fans with an extra lead that controls the fans' speed. If you replace that fan with a different fan that doesn't have the extra lead, the BIOS will think that the fans haven't started and present the message. This was common in older Dell PCs.

    To summarize:

    1- If you see the message, and your computer doesn't have a motherboard that monitors the heat inside the case, replace the fan in question ASAP. It's a simple repair. (Proof: I did it.)

    2- If you see the message, and your computer does have a motherboard that monitors the heat inside the case, replace the fan in question ASAP.

    3-  If you see the message, and your computer has a motherboard that monitors the heat inside the case, and you installed the wrong type of fan, then live with the message or install the right type of fan.
