Checking Windows 10 license key validity. - Microsoft Community

yoshuahutagalung say:

Checking Windows 10 license key validity.

Hi,
I just bought Windows 10 license key. How do I know that the license is valid without using it yet?

kapil arya [directly] say: Hello YoshuaHutagalung,

I'm Independent Advisor and Microsoft MVP, here to help you with your question.

IMO, there is no way to identify this.

You need to insert product key and check the type of license installed on your system.

1. Open administrative Command Prompt.

2. Type slmgr /dlv and press Enter key.

3. Note the Product Key Channel section Windows Script Host box:

If it mentions OEM <method>, you’ve OEM license installed.

If it says Retail, you’ve retail license installed.

If it mentions GVLK, VOLUME_<method> then you’ve volume license installed. Note that GVLK stands for Generic Volume License Key.

Retail and OEM license are usually supposed to use by end users. If you've Volume license installed on end user machine, the license may not be genuine. Let me know, if that is the case.

Regards.

Yes, in my case, the problem was one or more bad or corrupt messages in the user's Inbox (but not the Conversation History).  Specifically, it was in their Sent Items.  I ended up exporting out each category one at a time (Inbox/sub folders, Sent Items, Deleted Items, etc.).  Once I exported out Sent Items and subsequently permanently deleted them, the problem went away.  This user had roughly 200k of messages (~45 GB), which is why I did one category at a time.

"How to Refresh Windows 8 and Windows 8.1"


  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"
