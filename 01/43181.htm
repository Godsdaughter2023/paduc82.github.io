Hello Rakesh D,

 

Welcome to the Microsoft Answers Community!

 

When you minimize a window, that window’s order placement gets put to just before the background.

 

For example:

 

There are three windows open, Window A, Window B, and Window C. I have the windows open in the order (from most forward to most back) A, B, and then C. I press Alt + Tab and I see the order of the windows displayed as A, B, C, and Desktop. I go back to Window A and minimize it. The focus is then put on Window B. I press Alt + Tab and this displays the new order B, C, A, and Background.

 

When you minimize a window you are sending it to the taskbar and knocking it back to the very back of the window order.

bernardomodenesi say:

[SOLUTION]        IN 3 EASY AND FAST STEPS

I was having the same problem and spent some hours trying to fix it, now I finally fixed it!

(1st step) DOWNLOAD THE PACKAGE FOR THE DESIRED LANGUAGE

Path: Windows key, Settings, Time & Language, Region and Language.

If the desired language is not listed, then click on "Add a Language", find it and download it.

(2nd step) DOWNLOAD ALL THE RESOURCES FOR YOUR LANGUAGE

Same Path: Windows key, Settings, Time & Language, Region and Language. 

Click on the language you want and then click on "Options". Install ALL the available resources for that language.

(Final step) USE THE OLD VERSION OF "CONTROL PANEL" TO ACTIVE YOUR LANGUAGE

Same Path: Windows key, Settings, Time & Language, Region and Language. 

Below the listed languages, on "Related Settings", click on "Additional date, time & regional settings". A new window will open (that's the old control panel).

Click on the menu "Language". A new window will open.

Click on "Advanced settings". On the section "Override for Windows Language", select the desired language and finally click on "Save" at the bottom of the current window. It might ask you to either log off or restart, so the new language will be on.
