Method 1-

Refer the article-

Open the Power troubleshooter

http://windows.microsoft.com/en-US/windows7/Open-the-Power-troubleshooter

Method 2-

You need to try changing the sleep settings for the computer.

Refer the steps-

a)   Click on “Start” and select “Control Panel”.

b)   Now, on the “search bar” type “Power options” (without the quotes).

c)   Open Power Options and select the appropriate power plan.

d)   Now, select “Change plan settings”.

e)   Now, set the options next to “Turn off the display” and “Put the computer to sleep” to “Never”.

f)    Click on “Save Changes” and then restart the computer and check.

Method 3-

If the issue persists, then I would have you restore the power plans to default.

Refer the steps-

a) Click on “Start” and type “cmd” (without the quotations).

b) Once you see the “cmd” icon, right click on it and select “Run as administrator”.

c) Copy and paste the command in the Command Prompt window and hit "Enter".

 

powercfg –restoredefaultschemes

 

d) Wait for process to complete.

e) Close the Command prompt window.

 

Note: Restoring the power plans to default will delete all the customized power plans.

Method 4-

If nothing works, then I would have you install the latest chipset drivers for the computer from the manufacturer’s website. You can ask the manufacturer for assistance as well.

You can also update the drivers for the graphic card as well after installing the chipset drivers.

Note: Installing the drivers for the chipset is risky and needs to be done with caution. Proceed at your own risk.
