tryx3 say:

To create a task for running elevated or to edit any elevated task, you need to open Task scheduler as Admin [i.e. right-click on the Task scheduler shortcut and select Run as Admin]. 

You also need to enter the Admin account name in the use the following user account entry on the General tab.  You will get challenged for the Admin account's password.

Unless you are always logged in on an Admin account, you need to set the checkbox for Run whether user is logged on or not.  Please note that setting this item is not appropriate for any task that has a user interface since a user that is not logged on has no user interface in existence at that moment.
