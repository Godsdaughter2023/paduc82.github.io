Hi RussAdams,


This feature is by design in Windows.

In first scenario, it will start searching for the file with the initial letters of the file rather than the letters present in the middle of the file name.

First...you said you saved the project in Movie Maker
on the Dell and the saved file would not work on the HP?
I'm wondering if you were trying to open the .mswmm
project file...and no...it won't work on a different computer
because it does not include the source files. 

If you Publish the file in the .wmv format...you will have
better luck importing it into the HP Movie Maker.

I have no idea what Microsoft recommends but the
following Microsoft program can convert the files.

Windows Media Encoder
http://www.microsoft.com/downloads/details.aspx?FamilyID=5691ba02-e496-465a-bba9-b2f1182cdf24&displaylang=en

How do I convert other video file formats to WMV?
http://www.microsoft.com/windows/windowsmedia/knowledgecenter/mediaadvice/0065.mspx
(Tip: After clicking the 'Convert a File' button...
when you click the 'Browse' button to select
your .mp4 file...you will not see .mp4 as a choice.

If you open the 'Files of Type' drop window
and choose 'All Files' you should be able select
your .mp4 file.

Under 'Content Distribution'... 'File Archive'
might be a good choice.

Good luck...

jcole66 say:

USB Bluetooth dongle support on Xbox One S?

Does the Xbox One S support a USB BT dongle?  I'd like to use a BT keyboard on my XB1s but realize there is no built in bluetooth support. 

Anyone know for sure if this can work?

internet expert say:

The Xbox One and Xbox One S do not support Bluetooth in any way, and it does not contain any built-in drivers to interface with a Bluetooth dongle.


If you want to use an external keyboard, you can use:

- The keyboard chat-accessory that attaches to the controller.

- Your smartphone with the Xbox app (keyboard pops up when you are ready to enter text on Xbox).

- Standard USB keyboard.

- Wireless keyboard with its own included USB dongle (non-BT)

taidara say:

There's in fact a Bluetooth 4.0 module built-in, which by the Xbox One/Ss design is in a disable state. So, as the previous sayer said; there's currently no support for Bluetooth.
