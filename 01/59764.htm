How to turn off Windows 7 Search indexing:

Windows 7 Search still works without indexing (except in Libraries windows).  To turn indexing off, click the Start button, Control Panel, Administrative tools, Services, scroll down to find Windows Search in the window that appears and doubleclick it, and click the Stop button.  Indexing is made inactive for all user accounts.  To stop indexing from starting again, click the down-arrow button by “Startup type:”, click Disabled, and click the OK button.  (To turn indexing back on, change “Startup type:” to Automatic, then click the Apply, Start, and OK buttons.)  The Start menu Search box will then give results limited to programs and Control Panel items (any new programs installed will be searchable regardless of the indexing being off); no document files of any kind show up.  Searching in windows’ Search boxes behaves as before, but any changes that you might have done in Indexing Options will no longer be effective.  Also, the “Arrange by” feature in the Libraries windows will no longer work.
