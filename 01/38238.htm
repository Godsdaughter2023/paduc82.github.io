danida_u say:

Hi

The statuses available for Skype for Windows 10 are:

  • Online - Set by default when you log in to Skype.
  • Do Not Disturb - You're online but don't want to be disturbed. Other users can still send you messages and call you, but you won't be alerted with a sound.
  • Do not show my presence status - Same as Invisible.  Appear offline to your contacts but still use Skype as normal.
  • Offline - Appears to your contacts when you're not signed in to Skype.

Follow these steps below on how to change your presence status on Skype for Windows 10.

  • Open Skype.
  • Click your profile picture.
  • Below your profile picture, you'll see two available status, Do Not Disturb and Do not show my presence status.
  • Toggle any of the two to change your status.

Hope this helps! Let us know if you need additional information.

Resetting and re-configuring the TCP/IP will help you’re your network adapter to make changes depending on the recommended settings needed. To Reset and Reconfigure your TCP/IP, follow these steps: 

Method 1: Reset TCP/IP  

To do this, follow these steps:  

  1. Run Command Prompt as Administrator.  

  2. Run this command:  

    • netsh int ip reset and hit Enter.  

Exit Command Prompt and monitor your Network Connection.  

Method 2: Re-configure TCP/IP 

To do this, follow these steps:  

  1. Run Command Prompt as Administrator.  

  2. Run these following commands:  

    • netsh int tcp set heuristics disabled  

    • netsh int tcp set global autotuninglevel=disabled  

    • netsh int tcp set global rss=enabled  

Update us on how everything goes.
