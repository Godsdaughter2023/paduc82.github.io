a. user say:

It sounds like your system is Hibernating...

When your system hibernates, either because you do it manually or it is set up to do it automatically, XP writes out the contents of your RAM to a file C:\hiberfil.sys then powers the system off.  The amount of data and the size of the hiberfil.sys file gets bigger the more RAM you have installed.  If you have any work in progress that has not been written to the disk, that gets saved too.

When you power up again, XP issues a message 'Resuming Windows...' and then the c:\hiberfil.sys file begins to load and you see the progress bar along with the message to press F8.

To Hibernate, from the Turn Off Computer menu, press the Shift key and the Stand By button should turn into a Hibernate button you can choose.  Give it a shot so you can see what it looks like.  You can also add a fouth Hibernate button to your Turn Off Computer menu if you are lazy like me (saves a keystroke).  Then power up to restart your system.

If you press F8, you will see a screen like this:

System restart has been paused:
  Continue with system restart
  Delete restoration data and proceed to system boot menu

(Did your message really offer to shutdown and restart?)

This is not the same behavior if you choose to Stand By...

'Continue with restart' resumes reading the hiberfil.sys file, 'Delete restoration data' deletes the hiberfil.sys file and goes to the normal boot process.  The latter could be dangerous if you had a work in progress that was in RAM at the time of hibernation, but may be good if your hiberfil.sys file somehow got corrupted and your system will not resume.

Depending on the size of the hiberfil.sys file (it is based on how much RAM you have), you may find that it takes longer to come out of hibernation than it does to cold boot your system.  For this system with 4MB RAM, it takes (guessing) about 30 seconds to resume from hibernation and that is an eternity of waiting for me.  Maybe it is not 30 seconds, but whatever it is, it is too long to wait.

Since I know it only takes my system 14.82 seconds to cold boot, I don't choose to hibernate anymore.  I just shut down.

If you are not hibernating manually, go to Control Panel Power Options and check to be sure the system is not set to hibernate after a period of inactivity (set things to Never), or you can just disable Hibernation on the Hibernate tab, save any adjustments and reboot to be sure the changes "stick".
