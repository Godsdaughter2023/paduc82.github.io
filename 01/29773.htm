erdeslawe say:

Then is there any way to use windows explorer to see everything that is on the actual desktop?

That's an interesting question, which I suspect you are asking because Windows File Explorer doesn't display all of the visible shortcuts on your Desktop.

The reason File Explorer doesn't display all Desktop content is that not all of the icons you can see are truly on your User Desktop! The Desktop view is actually a merged view of 3 different sources.  Try this:

1. Right-Click on one of the DeskTop Icons that does appear in File Manager > Select "Properties" > Select "Details"  You will see that the Folder Path is C:\Users\UserName\Desktop

2. Right-Click on one of the DeskTop Shortcuts that does NOT appear in File Manager > Select "Properties" > Select "Details"  You will see that the Folder Path is C:\Users\Public\Desktop

3. The System Icons such as Recycle Bin are a 'special case', because you choose whether or not they Show on the Desktop through a Control Panel setting.  They aren't shortcuts and actually truly on the Desktop at all; they are just showing through from their true location. e.g. Check the properties of the Recycle Bin - No mention of the Desktop at all, just a location of C:

So, what you see in Windows File Explorer is your true User Desktop.  If you look at the Public Desktop you will see any Shortcuts Missing from the User Desktop.  You will never see the 'System Icons' because they aren't there, just 'showing' there under User Instructions.

Quite complex, but hope it's clear.

bossdweeb say:

- Laptop or Desktop machine ? Is 2GB for minimum specs or recommended ?

Most desktop machines (and possibly some laptops with a dedicated gpu as well as integrated) have an option in BIOS to select

the amount of system RAM that is dedicated to the integrated graphics.

The maximum varies, and your BIOS may not have an option to set the Video RAM, but the maximum can be as little as 512MB or up to 2GB,

depending on the motherboard or laptop brand and model.

- What game are you wanting to run ? Some games require dedicated VRAM which is found only on dedicated graphics cards

(and is much faster than system RAM). Those games will often use the term VRAM in the system requirements, but not always.

Determining if a game will run well on integrated graphics at all can be tricky as they don't often indicate if a game will run on

integrated graphics in the system requirements.

It's often necessary to trawl though a forum specific to that game to find if people are successful using integrated graphics with that game.

Similarly, if your machine has only 1GB of system RAM dedicated to the integrated graphics, the game may

not see that as adequate even though there is more 'shared' RAM available.

Even though your machine could open up more than the 7.9GB to a game if it were required (not going to happen with a game that

requires 2GB for video), the game may not be coded to look for available RAM other than the limit your machine dedicates to graphics.

cmdrkeene say:

On the login page, click "Sign in options", then click the number pad icon to select the PIN option:




If that's no longer available and you do not know the password, you will need to use another administrator-level account to change your password.


If there are no Administrator accounts on the system, you can boot to Safe Mode to access the Built-in Administrator account. From this fail-safe account, you can force-reset your account's password.

  1. Press +I to display the Settings charm menu.
  2. Click the Power Options icon, then hold down the Shift Key while you click the Restart option. Keep holding shift until the screen says it is restarting.
  3. On the first screen that appears, select Troubleshoot, then select Advanced Options. Finally pick Startup Settings and click the Restart Button.
  4. On the next screen, press the Number 4 Key to choose Safe Mode.

Now that you are in safe mode, you can use the built-in backup Administrator account to perform you necessary recovery. The built in Administrator account should not be used for any other purpose, or it won't be available in situations like this when you need it.

  1. At the login screen, click the Arrow Icon and select the Administrator account.
  2. After Safe Mode starts, press +W and open Control Panel / User Accounts.
  3. Select Manage another account and select the account you wish to change to an Administrator account. You can also use the Change Password option here to reset the other account's password.

When you're done, restart the computer normally.
