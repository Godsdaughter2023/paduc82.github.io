    I have a workaround that's slightly less annoying than the 'restart PC' fix: If you get the error where MYOB freezes up when you try to send an email, open task manager (ctrl-alt-delete or right click on start bar), then go to the 'Processes' tab and close any instances of wlmail.exe using the 'End Process' button.
