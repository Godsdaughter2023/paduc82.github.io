a. user say:

Hi Mia,

Welcome to Microsoft Community.

Did you make any recent changes on the computer prior to this issue?

This issue could have occurred due to damaged Keyboard driver. Let’s check if the issue occurs while using On-Screen Keyboard.

Use the On-Screen Keyboard (OSK) to type.

http://windows.microsoft.com/en-US/windows-8/type-with-the-on-screen-keyboard

I would suggest you to try the below methods and check if that helps:

Method 1: Run the hardware troubleshooter and check if it helps.

  1. Press Windows key + C on your keyboard to show the Charms bar.
  2. Type Troubleshooting and click Troubleshooting under Settings.
  3. Now type Hardware Troubleshooter in the search option.
  4. Click on Hardware troubleshooter to run the troubleshooter.

Method 2:

If the issue persists, I would suggest you to download and install latest Keyboard driver\software from the manufacturer website in compatibility mode. Follow the below steps:

  1. Download the latest Keyboard driver\software for Windows 8 from the manufacturer’s website and save it on your local disk.
  2. Right-click on the setup file of the driver and select “Properties”.
  3. Select “Compatibility” Tab.
  4. Place a check mark next to “Run this program in Compatibility mode” and select Windows 7 from the drop down list.
  5. Let the driver\software install and then check the functionality.

Also you may update the drivers from the Device Manager. Follow the below steps:

1. Press Windows + R, type in Devmgmt.msc and hit Enter.

2. Select Keyboards.

3. Right click on the listed driver and click on Update.

 

Method 3:

Verify the following:

  1. Confirm your keyboard is set to the English language.
  2. Go to Control panel. Click on the "Region and Language" icon. If the setting appears to be in a foreign language, change it to English/U.S.
  3. Click on Keyboards and Language, verify the keyboard language is set to English.
  4. Click on the Change Keyboard and on the General tab, verify you have the correct language.

Also it may be the hardware issue so I would suggest you to contact the Manufacturer.

Keep us posted if you face any issues on windows in future. We will be glad to help you.
