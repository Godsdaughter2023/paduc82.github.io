A .wlmp is not a video file. It's the project file that goes with Windows Movie Maker. The project is more like the save state of your editing session and only contains the *names* of the clips you have used, not actual photos, videos or music.

To turn a project into a movie, you open the project in Movie Maker, then you choose File > Save movie. The result should then be a .mp4 (or .wmv) video file.

considerate_guy say:

Let me repeat -- a .wlmp is NOT a movie file so there is no converter that will make it into anything useful. The only thing you can do is open it in Movie Maker on the PC where it was originally created.

If you move a .wlmp (project) file to another PC, you may be able to open it on another PC if it has the same version of Movie Maker, but it won't be usable. That's because the project just contains the *names* of the media items used. Since you don't have a copy of those on your PC, the project file is useless.

It would kind of be like someone sending you a list of ingredients and saying it was a cake. You'd need both the ingredients and a kitchen to bake that cake before you could enjoy it. The media files (which are still on their PC) would be the ingredients and Movie Maker would be like the kitchen.

Using a different converter to try and do something with this would be like trying to bake a cake in your garage using an electric saw and screwdriver. It's not going to work. It has to be the kitchen with the right tools. Likewise, you need the exact media files on your PC. Without them it would be like trying to use pickles, ketchup and oatmeal in place of butter, eggs and flour.

Tell your friend who gave you the project file (.wlmp) that you instead need an actual movie in order to be able to view it. Have your friend open it in his copy of Movie Maker and choose File > Save movie. The result will be an actual video/audio file that can be viewed on most any PC without needing a converter. Obviously the movie file will be much bigger than the small project file since it contains actual media, not just the names of files.

Thank you for being part of Microsoft Community.

I would like to inform that Windows 10 consumes more power as there are more services running In background. I would suggest you to create a power plan according to your requirements in order to consume less power and increase the backup time for battery.

Also, there is an option Battery saver in Power and sleep setting which will have a control over the background services when enabled and consume less power. But when enabled it will stop the Windows Update from downloading updates and should be disabled for Windows Update to download and install the updates.

You can run the Power troubleshooter to adjust your computer's power settings. The Power troubleshooter checks things like your computer's timeout settings, which determine how long the computer waits before turning off the monitor display or entering sleep mode. Adjusting these settings can help you conserve power and extend your computer's battery life.

Run the Power troubleshooter and see if it helps.

a: Press Windows logo to open the search box.
b: Type troubleshooting, hit enter.

c: Now click System and Security.

d: Click Power.

Do get back to us for any issues pertaining in the future. We will be happy to help.

Try these first to see if they work:
