lemp say:

Unfortunately, when you buy a computer with Windows (any version) pre-installed on the computer by the computer's manufacturer, the license you get is non-transferable.  In other words, you can not legally move that copy of Windows to another computer.

Any new laptop you purchase today is almost certain to come with Windows 10 pre-installed and it is likely that the hardware in the new computer will not support Windows 7 even if you had a legal copy of Windows 7 to install on it.

Any software you installed on your VAIO after you purchased it should be transferable to a new computer.  Unless that software was already old (e.g., software designed for Windows XP or earlier versions of Windows), there is little reason to think that it would not work on Windows 10.  You could, of course, check at the website(s) for any specific programs to determine if the version you have would work with Windows 10 and if not, whether an update/upgrade was available.
