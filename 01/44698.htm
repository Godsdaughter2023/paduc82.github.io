There may be a program running in the Notification Area near the clock on lower right that controls the
device. Some Toshiba models have Card Keys at the top of the screen (move mouse all the way to
the top to see if your model has this feature) which allows control on certain devices. Some models
have Hot Keys (usually indicated on the Card Keys) which control input devices.

If nothing else works try Control Panel - Device Manager - Mice and other Pointing devices - double
click on the device - Driver tab - Disable. I would use this as last resort as the other possible options
allow you to enable and disable the device a lot easier.

Hope this helps.
