volunteer j say:
Two possible methods...
 
1) I'm guessing this is not a not a commercial DVD
with copy protection? If_yes_read on.....

**Be aware that breaking the copy protection
on commercial DVDs is illegal**

Insert the DVD in your DVD Drive...if AutoPlay or
Windows Media Player launch...just close them.

Now...go to...Start / Computer...right click the
icon for the drive the DVD is in and choose...
Open or Explore...this will display the directory
of the DVD.

Find the VIDEO_TS folder and open it...the video
files you are looking for have a .VOB extension.
Copy them to a folder on your hard drive...there
may be several VOB files and the ones with the
largest file sizes are probably the ones you need.

If you rename the .VOB files to .AVI or .MPG you
should be able to play them in Windows Media
Player.

Or you could copy the complete VIDEO_TS folder
to your hard drive. Just create a new folder with a
descriptive name to save it in.

FWIW...the freeware VLC Media Player can play
the files directly from the VIDEO_TS folder. Just
Right Click the VIDEO_TS folder and choose...
Play With VLC...

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

VLC Media Player
http://www.filehippo.com/download_vlc/
(runs on XP/Vista/7)
 
===================================
 
2) Using the freeware Format Factory...you
can download and convert the DVD video to
a format like WMV...then you can play it with
most any media player....and edit it in Movie
Maker.

**Be aware that breaking the copy protection
on commercial DVDs is illegal**

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Format Factory (freeware)
http://www.videohelp.com/tools/Format_Factory
(the 'direct link' is fastest download)
(the file you want to download is : >FFSetup290.zip<)
(FWIW...when installing...you may want to uncheck
'all' the boxes on the final screens)
(Windows XP / Vista / 7)

First you will need to unzip the file or just open the
folder and drag >FFSetup290.exe< out of the folder
and drop it on your desktop. Left click it to install.

Next...after downloading and installing Format
Factory...you may want to open the program and
left click the 'Option' button on the toolbar and
select an 'Output Folder' / Apply / OK.
(this is where you will find your files after they
are converted)

Insert your DVD in your DVD drive.

In the left column of Format Factory...go to...
'ROM Device\DVD\CD\ISO' / DVD to Video File...

Open the DVD drop window and select the drive
the DVD is in. The files on the DVD should appear
in the Output Setting window. Select (check) the
main feature for conversion (the one with the longest
duration will be the actual movie)

Choose WMV as the format and left click the
Convert button. Back on the Main screen...left click
the Start button.

Now, all you have to do is wait. It's best if you do not
multi-task while video is being rendered.

That's about it...Format Factory has many settings
you can adjust but this should be enough to get you
started.

Good luck...

What is Driver Verifier?

Driver Verifier monitors Windows kernel-mode drivers, graphics drivers, and even 3rd party drivers to detect illegal function calls or actions that might corrupt the system. Driver Verifier can subject the Windows drivers to a variety of stresses and tests to find improper behavior.

Essentially, if there's a 3rd party driver believed to be causing the issues at hand, enabling Driver Verifier will help us see which specific driver is causing the problem.

Before enabling Driver Verifier, it is recommended to create a System Restore Point:

Vista - START | type rstrui - create a restore point
Windows 7 - START | type create | select "Create a Restore Point"
Windows 8/8.1 - http://www.eightforums.com/tutorials/4690-restore-point-create-windows-8-a.html

How to enable Driver Verifier:

Start > type "verifier" without the quotes > Select the following options -

1. Select - "Create custom settings (for code developers)"
2. Select - "Select individual settings from a full list"
3. Check the following boxes -
- Special Pool
- Pool Tracking
- Force IRQL Checking
- Deadlock Detection
- Security Checks (Windows 7 & 8/8.1)
- DDI compliance checking (Windows 8/8.1)
- Miscellaneous Checks
4. Select  - "Select driver names from a list"
5. Click on the "Provider" tab. This will sort all of the drivers by the provider.
6. Check EVERY box that is NOT provided by Microsoft / Microsoft Corporation.
7. Click on Finish.
8. Restart.


- If Driver Verifier finds a violation, the system will BSOD. To expand on this a bit more for the interested, specifically what Driver Verifier actually does is it looks for any driver making illegal function calls, causing memory leaks, etc. When and/if this happens, system corruption occurs if allowed to continue. When Driver Verifier is enabled per my instructions above, it is monitoring all 3rd party drivers (as we have it set that way) and when it catches a driver attempting to do this, it will quickly flag that driver as being a troublemaker, and bring down the system safely before any corruption can occur.

- After enabling Driver Verifier and restarting the system, depending on the culprit, if for example the driver is on start-up, you may not be able to get back into normal Windows because Driver Verifier will detect it in violation almost straight away, and as stated above, that will cause / force a BSOD.

- Once in Safe Mode - Start > Search > type "cmd" without the quotes.
