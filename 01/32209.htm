naman r say:

Hi Sunsvalley,

Welcome to Microsoft answers.

·         Are you using any non-Microsoft applications to change the font?

You may try the below steps to change the default font size of the Operating system.

1. Right click on a empty space on the desktop and click on Personalize.

2. Click on the Window Color link at the bottom of the window.

3. Click on the Advanced appearance settings link.

4. Select a Item that you want to change the font for.

NOTE: Not all of these items will have a font for them.

5. To Change the Item Font.

A) Click on the Font drop down menu and select a font.

6. To Change the Item Font Size.

A) Click on the Size drop down menu and select a number for the size. The higher the number, the larger the size.

7. To Change the Item Font Color.

A) Click on the Color drop down menu and select a color.

8. To Have the Item Font in Bold.

A) Click the B button to highlight it. If it is not highlighted, then the font will not be in bold.

9. To Have the Item Font in Italic.

A) Click on the button to highlight it. If it is not highlighted, then the font will not be in italic.

10. Repeat steps 4 to 9 until you have changed all of the fonts for the select items that you wanted.

11. When done, click on OK.

12. Close the Window Color and Appearance window.
