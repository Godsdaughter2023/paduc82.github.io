Unlike iMovie, which has chromakeying built-in, there is no such feature in WLMM.  Windows Live Movie Maker is intended to be a quick tool for pulling together clips and pictures from your digital camera and creating a short movie that you can share with others.  Its goal is speed and ease of use so it doesn't have more advanced features like green screening.

If you want advanced features like chroma-keying you have a few options:
1) Submit it as a feature suggestion for WLMM.  Given the design goals for simplicity and ease of use, it probably isn't at the top of any feature list, but you can try. :)  The link can be found in the Blue|File menu (Alt+F) then About Movie Maker...
2) Install the prior version of Movie Maker (WMM 6.0) and an add-on called ShaderTFX which includes a set of chroma-key transitions.  For more details on this, check the following:
http://windowsmoviemakers.net/Forums/ShowPost.aspx?PostID=126998
3) Purchase other higher-end software that includes full support for green screen footage.

Hi Martin,

With the current design of Windows Live Movie Maker, the option we only have for editing the background of the movie is changing its background color.

How to change the background color:

1. Launch Windows Live Movie Maker
2. Add videos and photos
3. In the Home tab, add your preferred AutoMovie themes
4. Under Video tools, click Edit
5. Click Background color and select your preferred color (different shades of green in More color option)
