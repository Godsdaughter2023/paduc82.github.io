Only Windows 7 Enterprise provide the rights to install 4 additional instances in virtual machine on the Host OS. Volume license customers with Software Assurance you can run up to 4 virtual copies of the OS on the licensed machine. 

 

This does not apply to Windows 7 Ultimate because it is a retail license, you need to purchase the additional licenses.

If you acquire “Software Assurance”, you have the right to use Windows 7 Enterprise Edition on the device instead of Windows 7 Professional. This also permits you to run up to four additional copies or instances on the device.


3) I now have two licence keys - my original Win7 Pro and my Win7 Ultimate upgrade key. Can I re-use either of these keys to activate my bootable VHD or do I need yet another licence key?

 

In regards to VHD Boot, only Windows 7 Enterprise or Ultimate supports it. So if you have Ultimate available as a VHD and the product key is not in use else where, you can use it.
