    Hi Lisa

    Do you mean edit the text in the document?

    When you scan a document it produces a static image (picture) of the document.

    To be able to change the text in a scanned document requires special programs called Object Character Recognition (OCR) software.

    You can use your favorite search engine to find these programs.

    ken blake say:

    IS THERE A WAY TO EDIT A DOCUMENT/IMAGE AFTER IT HAS BEEN SCANNED INTO WNDOWS FAX & SCAN PROGRAM?

    Ouch! Please don't yell at us. We can hear you if you type normally, in mixed case.

     

    You say "document/image," but which do you mean? If you want to treat it as an image. you can edit it in any graphics editing program, such as Adobe Photoshop.

     

    But if you want to treat it as a document, and edit the text that's there. You can *not* do this directly. When you scan a text document, you don't get text; what you get is just a *picture* of that text.

     

    However you can change that picture of text to actual editable text if you use a special kind of program called OCR (Optical Character Recognition). Many scanners come with "Lite" versions of such software, but if you don't have one, you need to get one. The two best such products are Abbeyy Fine Reader and Omnipage, but note that they are not cheap.
