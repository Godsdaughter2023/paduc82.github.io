ron75234 say:

[SOLVED] sort of...

For some unknown reason, I got the blue screen.  KB3084070 had failed to install.  Things seemed thoroughly hosed.  The solution was to to wipe the system disk and start from a clean install.  After 2 days, I have recovered about 80% of my apps; it's been a long time since I did this and Windows 10 is subtly different.  All WUSA updates install incl KB3081455 and KB3084070, although not always on the first try.  A serendipitous benefit is that the clean install works much better than the upgrade, esp. the desktop.

My sad conclusion is that the Windows 10 upgrade has issues.  DISM and SFC don't detect/repair all inconsistencies.  Microsoft is silent on the subject.  Therefore, my course of action was:

  1. Backup everything
  2. Use the Media  Creation Tool to make a Windows 10 install disk from the ISO file
  3. Save the upgrade Windows 10 product key (I used Speccy)
  4. Start from the beginning
