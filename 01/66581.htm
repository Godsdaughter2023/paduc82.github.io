To resolve the issue with automatic rip, here are some general steps to try.

 

Method 1:

 

Verify your options in AutoPlay.

 

a. Click Start button, then Click on Control Panel,

b. Click on AutoPlay.

 

Note: AutoPlay allows you to choose what happens when you insert different types of media or devicein your computer.

If you are inserting an Audio CD, you can choose from the following options:

  • Play audio CD using Windows Media Player
  • Rip music CD using Windows Media Center
  • Play audio CD using Windows Media Center
  • Open folder to view files using windows Explorer
  • Take no action
  • Ask me every time

Please change the option to what you want from the list. Close Auto Play and open Window Media Player and see if it is now choosing the correct option.

If method 1 fails, try method 2 and check.

 

Method 2:

 

To turn off automatic ripping for the CD drive

 

a. Click Start, and then click on Computer.

b. Right-click the CD drive that you use, and then click Properties.

c. In the Properties dialog box for the CD drive, click the AutoPlay tab.

d. In the list of content types at the top of the dialog box, click Music CD.

e. In the Actions area, do one of the following:

 

• Click the Select an action to perform option, and then specify the action you want Windows to perform when it detects a music CD in the CD drive by clicking that action in the list.

 

-or-

 

• Click the Prompt me each time to choose an action option. Windows will then ask you what you want to do each time it detects a music CD in the CD drive.

processing MSWMM files to burn onto DVD - Microsoft Community

malcolmhitt say:

processing MSWMM files to burn onto DVD

Having produced a video in moviemaker it has saved as an MSWMM file, I can't find a means of burning onto a DVD, can anyone help
considerate_guy say:

The .mswmm file is a project file from one of the classic versions of Movie Maker.  It is like the recipe for building your movie, but contains no actual video, photos or music.  It is just a file that shows how your movie is laid out.

 

In order to turn that into something that can be viewed and shared, you must save it as a movie.  The exact instructions depend on the version you are using.  If you are on Windows XP (WMM 2.x), the option is under the File menu and is Save Movie...  If you are on Windows Vista or Windows 7, you might be running Windows Movie Maker 6.0.  Here the option is File > Publish Movie...

 

In either case, once you have save the movie to your PC, you then need software to burn a video DVD.  You can't just burn this file onto a DVD disc and expect it to play in a standard DVD player.  That would be a data DVD.  If your PC doesn't have such software, look for 3rd party programs like DVD Styler or DVD Flick.  If you are on Windows Vista or Windows 7, you can use Windows DVD Maker which should already be installed.
