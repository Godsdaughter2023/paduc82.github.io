Olá NhyuiTop,

Nat duplo geralmente acontece quando você tem dois dispositivos de rede seguidos. Por exemplo, um modem adsl seguido de um roteador. Isso acontece porque geralmente o modem é também configurado como um roteador via cabo, o que faz com que haja dois NATs na rede. No caso do modem adsl (não sei se é o seu caso) corrige-se isso colocando-o no modo Bridge para fazer uma ponte direta para o roteador. 

Para NAT estrito, resolve-se configurando as portas de rede utilizadas pela xbox live no roteador. As portas utilizadas pelo serviço são:

  • Porta 88 (UDP)
  • Porta 3074 (UDP e TCP)
  • Porta 53 (UDP e TCP)
  • Porta 80 (TCP)
  • Porta 500 (UDP)
  • Porta 3544 (UDP)
  • Porta 4500 (UDP)

Espero que isso lhe ajude em seu problema.

[Translation-Hello Nhyuitop,

Double Nat usually happens when you have two network devices in a row. For example, an ADSL modem followed by a router. This happens because usually the modem is also configured as a router via cable, which causes there to be two Natsna network. In the case of the ADSL modem (do not know if it is your case) corrects it by putting it in bridge mode to make a direct bridge to the router.


For strict NAT, it resolves by configuring the network ports used by Xbox Live on the router. The ports used by the service are:

• 88 port (UDP)
• Port 3074 (UDP and TCP)
• Port 53 (UDP and TCP)
• 80 port (TCP)
• 500 port (UDP)
• 3544 Port (UDP)

I would perform a direct clean install...

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.

Go to a working computer, download, create a bootable copy, then perform a clean install.
