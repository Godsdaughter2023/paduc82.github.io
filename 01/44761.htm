Hi,

 

I see that this is your first post in this forum, welcome to the Microsoft Community.

 

This issue could be caused due to some incorrect settings or due to a corrupt or incompatible touch driver. I would suggest you to try these steps and check:

 

First, I would suggest you to check the device status of the touch screen for any errors. The device manager will show any driver or hardware related errors for all devices. Refer to these steps to check the device status of the touch screen:

 

a)      Press Windows key + X, select Device manager.

b)      Locate the Touch screen.

c)       Check if there is a yellow exclamation mark or a red cross mark over it.

d)      Right click on device and select properties.

e)      In the General tab, check the device status.

 

Method 1:

 

I would also suggest you to run the built-in hardware and devices troubleshooter which can automatically detect and fix most of the incorrect settings that could cause issues with hardware devices. Refer to these steps:

 

a)      Press Windows key +F to bring up the search Charms bar.

b)      Select settings in the drop down menu below search and type Troubleshooting.

c)       Click on Troubleshooting and select the view all option on the left pane.

d)      Click on Hardware and devices troubleshooter and follow the prompts.

 

Method 2:

 

If the issue still persists, then it could be caused due to a corrupt touch driver. In this case, I would suggest you to install the latest touch driver from the manufacturer’s website and check. Here is a link to download the drivers for this model:

 

http://www.asus.com/AllinOne_PCs/ET2300IUTI/#support

Since, the latest available driver is for Windows 8, you can download and install these drivers.

 

Try these steps and let us know the results. We’d be happy to help if you need any further assistance.
