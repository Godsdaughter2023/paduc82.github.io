Free Partition Manager - AOMEI Partition Assistant Standard

eg-  fixing any important issues that show in Event Viewer - Windows Logs > Applications and System.

Uninstalling unused software, running Windows Disk Cleanup tool and defraging the HDD.

By the time you've got Win7 back in fighting shape, you may feel less inclined to be seduced by the

'free' Win10 upgrade that you possibly don't need.
