How to insert a photo over video frames but leaving the audio - Microsoft Community

wildcats 007 say:

How to insert a photo over video frames but leaving the audio

Original title: Replacing video with still pics....

 

Is it possible to insert a still pic over a couple of video frames, and leaving the audio?

 

wildcats 007 say:

....with "Windows Live Movie Maker" ....

 

considerate_guy say:

There isn't a direct feature to do this, but it is possible if you separate the audio onto its own track.  The reason it must be separate is that Movie Maker considers a video and its associated audio to be one element.  If you were to split it to insert a photo, you would also split the audio.  So you'll see the video (with its audio), then a break, then it will start again (with its audio) exactly at the split point.


In order to separate the audio, you will need Movie Maker 2012.  Prior versions will only let you add audio files as the "music" track, but now you can pick videos files and MM will use the associated audio.


Normally, when you click 'Add music' you'll get a dialog to pick a music file.  Initially it will show you audio-only files.  But you can change the file type from "Audio and Music" to "Videos" in the lower right.  When you do this, you can then select a video file.  Pick the same video file that is already added to your storyboard.

 

If you did this right, you should hear sound from the original video and the newly added sound track.  So then you want to go back and mute original video.  You can do this globally using the Project tab and Audio Mix.  Set the balance fully toward the added music, rather than the original video.  Now you should hear one copy of the audio.  Alternatively, select the video clip and set its Video Volume to zero on the Video Tools | Edit tab.

 

The tricky thing now is to keep the video and separate audio in sync.  You have to be mindful of the duration of added photos.  So let's say you add a photo of 2 seconds (no transitions).  You then need to split out exactly 2 seconds of video (M = split, before and after), right-click and remove the clip of video.  Now insert the replacement photo.  Doing it this way, when the photo finishes displaying the video will be at the same point it would have been before; and the audio will be still in sync.

 

P.S.  Another option, is to add the audio as sound (rather than "music").  This might be necessary if you also want to add music.  To do this, click the down-arrow next to the 'Record narration' button.  You should see a choice to 'Add sound'.  It behaves similar to added music.

andre da costa say:

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already ownto reactivate if necessary. or use the reset function in Windows 10.

Right click the MediaCreationTool.exe file then click Run as administrator.

Note: How to verify if you downloaded or installed the Windows 10 Anniversary Update

Select the option Upgrade this PC now if you want to upgrade right away, of if you want to download a ISO image or create a bootable copy on a USB thumb drive, choose Create installation media for another PC then click Next
