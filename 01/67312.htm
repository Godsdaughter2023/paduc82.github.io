Creating DVD's with Windows 10 - Microsoft Community

carelann say:

Creating DVD's with Windows 10

Can Movie Maker videos be saved to a DVD? Is there a program, compatible with WIN 10, that would allow me to create DVD's in NTSC and PAL formats?
considerate_guy say:

The ability to burn to the legacy video DVD format was only available on Windows Vista and Windows 7. In Windows 8, 8.1 and 10, the option is no longer there.

The video DVD format requires a specific format with video encoded in MPEG-2. But that requires a licensing fee be paid. Additionally the DVD format only handles up to 480p (NTSC) / 540p (PAL) vertical resolution.

If you still absolutely must output in this older format, you can find software that will do it (say DVD Flick or DVD Styler). But you might want to explore more modern options such as just playing the video (.mp4) from a USB stick to a Smart TV. Or streaming over a network or via something like Chromecast.

Another option, if you have a Blu-ray player (also included in gaming consoles like XBox 360, PS3, PS4, etc.) is to burn the .mp4 file that Movie Maker creates onto a DVD (manually as a data DVD). That is in H.264 format which is understood by Blu-ray players. It won't have any fancy menus, and you are limited to the length of the movie (maybe 30-40 minutes), but it would be in HD quality instead of the older SD format of a DVD.
