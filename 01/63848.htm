a. user say:

The Fax Service depends on the Print Spooler Service.

The Print Spooler Service depends on the Remote Procedure Call (RPC) Service.

The RPC Service doesn't depend on anything, but a lot of things depend on it, so it is usually running all the time.

"Depend" means, it won't run or start without the other one(s) already running first.

Your fax Service is probably set to start Automatically and can't start because the things it depends on are not started either.  If you don't disable it, it will keep trying to start because it thinks it is supposed to be running.

So, check if your RPC service is running, if not start it.  Then your Print Spooler Service should start, then your Fax Service should start.

The only one of those three that you usually must have running is RPC since so many things depend on it.  If it is not running or won't start, you may have some other issue.

If you never use fax, disable the service and it won't try to start (that is how mine is).  If you never print, disable the Spooler Service (that is how mine is).  I don't like the extra junk running or even starting when I am not going to use it.  I would rather my CPU do things I want it to do - and faxing is nothing I do and I print rarely.  If you don't want to Disable the service completely, set it to Manual and it will start if it needs to start.

You can look at your Services by clicking Start, Settings, Control Panel, Administrative Tools, Services.  Manipulate the Services by right clicking them and look at their Properties and the Stop/Start buttons on the screen.  You can see who depends on what there too.

A shortcut to the Services applet is to click Start, Run and in the box enter:

services.msc

Click OK.

How to Connect a Laptop to a TV Using a VGA Cable

 

Step 1 Plug one end of the VGA cable into the laptop's VGA port.

 

Step 2 Connect the other end of the VGA cable to the TV's VGA port. If the TV does not have a VGA port, plug the other end of the VGA cable into a composite adapter and then plug the composite cables into the TV. Hello Daver59,

If Windows remembers the credentials for your wireless networks it means the network profile has been stored by the WLAN auto configuration service. You can delete the stored profile if you wish by following the next steps:

Start a Command Prompt as an Administrator and execute the following commands

netsh wlan show profile

netsh wlan delete profile <Profile Name of the network you wish to delete as shown by the previous command>

Alternatively you can delete all profiles by executing:

netsh wlan delete profile *

Once you have done this, if you wish for windows to ask for the password everytime on your device then uncheck the "Connect Automatically" option when connecting through the network through the graphical user interface.
