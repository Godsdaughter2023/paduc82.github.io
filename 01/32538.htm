then repeat for each drive changing the drive letter, eg

Windows 10: How to add a secondary keyboard layout besides English - Microsoft Community

ahangama say:

Windows 10: How to add a secondary keyboard layout besides English (US) like US-International

Since 2006, a sizable community has been using a keyboard layout somewhat like Icelandic to type Romanized Singhala. It is used in Linux and Macintosh as well.

When Windows 8.1 was updated to Windows 10, we had the OS assuming we are using Unicode Sinhala. We think that it guessed it from the
LOCALENAME    "si-LK"

line on the keyboard definition. So, it added Complex Scripts options to the Font dialog. Then it placed the Unicode font Iskoola Pota as the reining font on the Complex Script side undermining our font. It does this by forcing numerals like a year such as 2006 to be from Iskoola Pota displayed in its size as in the Font dialog.

The result is that we are not allowed to select the font even for characters that are in the SBCS. This is clearly a bug in the OS.

We tried to remove our keyboard, but the OS did not list the keyboard in the installed programs list! We deleted it manually from the sysm32 and sysWoW64 folders and removed an item from the registry too. Incidentally, for the first time, we saw the regedit.exe stop responding toward the end of the database searching for a string. This behavior was repeated on two such searches.

Having removed the keyboard, we changed the problematic line to the following:
LOCALENAME    "en-LK"

It isn't quite clear what you are trying to do. You can't use your SP4 as a wired second display by connecting to its mini display port from another PC.

You CAN use the SP4 mini display port to MDP, HDMI or DVI to connect to the HDMi or DVI *IN* port of another monitor/TV (HDMI is better) and some PC's DO support video in.

If you draw ON your Surface and are connected to another monitor, you would select "duplicate" mode and whatever you are drawing/inking on your Surface would display on the SP4 and the 2nd monitor.
