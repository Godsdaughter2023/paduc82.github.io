Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

arsonlavek say:

how to mix music using windows movie maker 2011

Can someone plz explain how to mix music, not videos, using Windows Movie Maker 2011. For example if i wanted to add two songs on to a picture video, i want one song to start and the 2nd to start before the 1st ends. One song start at 0:00 - 1:30, 2nd song starts at 1:20 - 2:00. I have done this with early versions but i cannot for my life figure it out with the 2011 edition! By the way i have clicked on "Options" and set the timings for both songs but yet didn't work.
Thank you
considerate_guy say:

You can do this in the classic versions (like Windows Movie Maker 6.0) but in Windows Live Movie Maker 2011 and Movie Maker 2012 you are not allowed to overlap music on a track.  The best you can do is fade one out, then fade the next in, but they don't overlap or "mix".  If you would like this feature, you can go to feedback.live.com and suggest it for future versions of Movie Maker.

 

One workaround, if you are using Movie Maker 2012, is to use the two audio tracks it provides.  It has one track for music and another for "narration".  But you could load music files there instead.  To do this you would load song one as "music" and the second song as "narration".  Position the second song at 1:20 and have it fade in.  Set a fade out on the first song.

 

As I noted, you can use the classic version of Movie Maker 6.0 on Windows 7 (or Windows 8).  Or you might just want to look into a separate audio editing program like Audacity.  Do your audio mixing there, then add your video elements inside Movie Maker.
