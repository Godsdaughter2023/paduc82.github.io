lemp say:

You will not be "forced" to move to Windows 10.  Rather, once Microsoft stops providing security patches for Windows 7 in January 2020, Windows 7 will become more vulnerable than it currently is to malware attacks.

When moving to a new computer, normally you have to install software from original installation media using any license key that may be required by the software's publisher.

When Microsoft first introduced Windows 10, it provided a discount for Laplink's PC Mover Express program (https://www.microsoft.com/en-us/windows/transfer-your-data).  Like similar programs, PC Mover Express only transferred user data, not software. 

However, Laplink's "professional" version of PC Mover  claims to be able to transfer at least some software.  Since the original introduction of Windows 10, I believe that other software publishers have also released transfer programs that purport to transfer software, but I don't recall which ones.

PC Mover Professional is not free and I have no personal experience with it.  Still, if you have software that you want to move when you make the jump to Windows 10, it's probably worth investigating.  See https://web.laplink.com/product/pcmover-professional/#tabs2:

PCmover Professional is the ONLY software that automatically moves your applications, files, settings, and even user profiles from an old PC to a new one, or old operating system to a new one. And because most transferred applications are installed ready-to-use on the new PC, there’s usually no need to find old CDs, previously downloaded programs, serial numbers or license codes.*

PCmover Professional is the easiest way to restore or move to a new PC without leaving anything behind, even when there are different versions of Windows on the old and new PC. Nothing is changed on the old PC and nothing is overwritten on the new PC.

*Applications may require re-entry of a license code or re-activating on the new PC. Certain system settings and applications may not be transferred due to their incompatibility with the operating system on the new PC.

For more information, please read the User Guide!
