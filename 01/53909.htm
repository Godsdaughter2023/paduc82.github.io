andre for directly say:
Hi lucdedpet

My name is Andre Da Costa; an Independent Consultant, Windows Insider MVP and Windows & Devices for IT MVP. I'm here to help you with your problem.

When you setup a dual boot configuration on separate drives or partitions, Windows 10 will store the boot information for the second or third installation on the boot drive. This is how dual boot configurations work.

Even if you were to connect one SSD at a time, install Windows 10, remove it then install Windows 10 again on the second SSD, you would need to configure one to be the primary boot drive.

davem121 say:
Hi lucdedpet

If you do not set up a dual boot system, how do you expect to be able to boot to the second version of Windows - the only option then would be to go into BIOS and change the boot order every time, which would be totally inefficient

If you are now presented with a blue boot screen with a choice of OS to boot into, then you system is set up correctly and both operating systems are mutually exclusive . . .
