To set up a Bluetooth enabled device, you need to do three things:

Add a Bluetooth adapter to your computer.

This is usually done by plugging the adapter into a USB port. (If a Bluetooth adapter is already built in or attached to your computer, ignore this step.)

Turn the device on.

To install a Bluetooth enabled device

sandeep ghatuary say:

Hi Gord1956,

·         Was the computer on a domain network?

·         What is the exact issue that you are facing?

Follow these steps:

a)    Make sure you have administrative rights to the computer. If you do not have admin rights you will not be able to proceed. Check with the network administrator to make sure you have the ability to remove the PC from the domain.

b)    Right-click on "My Computer" and choose "Properties" from the menu. Click on the "Computer Name" tab and click the "Change" button.

c)    Click the "Workgroup" button and type the name of the workgroup you wish to join. This can be any name, but you will need to supply a name.

d)    Click "OK" to proceed. You will see a pop-up box asking you for the user name and password--enter the network credentials into those boxes and click "OK." After a few minutes, you will receive a message saying that the change has been completed.

e)    Click "Yes" when asked to restart your computer. When the computer reboots, it will no longer be a member of the domain.

For reference:

================================================
The files on the DVD needed for editing are the VOB files...you will find them
in the VIDEO_TS Folder. You may be able to navigate to the VOB files from
Windows Live Movie Maker and import them but if not...you could try copying
the files to a folder on your hard drive and rename them from .VOB to .MPG.

If you still cannot import them into WLMM you will have to convert the files
to the .WMV format.

There are many programs that can do conversions...
The following freeware is one example...:

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Format Factory
http://www.videohelp.com/tools/Format_Factory
(the 'direct link' is fastest)
(the file you want to download is : >FFSetup260.zip<)
(FWIW...when installing...you may want to uncheck
'all' the boxes on the final screen)

First you will need to unzip the file or just open the
folder and drag FFSetup260.exe out of the folder
and drop it on your desktop. Left click it to install.

Next...after downloading and installing Format
Factory...you may want to open the program and
left click the 'Option' button on the toolbar and
select an 'Output Folder' / Apply / OK.
(this is where you will find your files after they
are converted)

Drag/drop your video clips onto the main screen...

Select 'All to WMV' / OK...

Click...Start...on the toolbar...

That should do it...

Good luck...

Once the files are imported and edited you will have to use Windows DVD
Maker to create a New DVD.
