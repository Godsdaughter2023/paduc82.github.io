Method 2:

Volume Shadow Copy is used by System Restore for the Previous Versions of Files feature in Windows 8. For purposes of testing, you could try and disable the Volume Shadow Copy service.

a. Press Windows key and R key together, type services.msc and press enter.

b. From the Services dialog box, navigate to Volume Shadow Copy and double-click it.

c. From the Volume Shadow Copy Properties dialog box, change the Startup Type to Disabled.

d. Restart your computer and try creating a restore point again.

If it succeeds, you won't have any Previous Versions of Files or the ability to create a backup, but at least you will know if VSS is causing the issue.

 

Method 3:

a) Press Windows key and X key together.

b) Click on command prompt (Admin)
