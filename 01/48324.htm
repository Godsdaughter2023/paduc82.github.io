Bridging Guide

Login Info

 

  1. Gateway Address: "192.168.2.1" or try "172.16.0.1" or try "http://gateway.2wire.net/management"***
  2. Username: Try "admin" otherwise there is no default username.
  3. Password: Try "admin" or try “2wire” otherwise there is no default password.

 

***Note: The Management and Diagnostic Console (MDC) settings are dependent on the 2Wire gateway software release. The MDC are for 2Wire gateways running software release 4.21.x If your gateway is running a software release earlier than 4.21.x, some of these settings may not be available.

 

Bridging Guide

 

  1. Open a web browser.
  2. Type the gateway address: "http://gateway.2wire.net/management"*** into the web address.
  3. When prompted for a password, type in "admin" for both the username and password. If not, check the label on the gateway or ISP documentation for a username and password if you are prompted for a login. 
  4. In the panel on the left hand side under the “Advanced” category click on “Configure Services”.
  5. On the main page uncheck the “Enable Routing” box.
  6. Click on "Submit".

 

***Note: The Management and Diagnostic Console (MDC) settings are dependent on the 2Wire gateway software release. The MDC are for 2Wire gateways running software release 4.21.x If your gateway is running a software release earlier than 4.21.x, some of these settings may not be available.

How to use Malwarebytes after it is installed and Updated:

1. Open Malwarebytes > Click on the Update Tab  across the top> get the latest updates.

2. On the Scanner tab, make sure the the Perform quick scan option is selected and then click on the Scan button to start scanning your computer

3. MBAM will now start scanning your computer for malware. This process can take quite a while.

4. When the scan is finished a message box will appear

5. You should click on the OK button to close the message box and continue with the Malware removal process.

6. You will now be back at the main Scanner screen. At this point you should click on the Show Results button.

7. A screen displaying all the malware that the program found will be shown

Once your computer has rebooted, and you are logged in, please continue with the rest of the steps.

9. When MBAM has finished removing the malware, it will open the scan log and display it in Notepad. Review the log as desired, and then close the Notepad window.

10. You can now exit the MBAM program.

raymond ave say:

Hi Kat,

It's possible that the Ethernet is currently disabled. Kindly follow these steps to help you resolve your concern:

Method 1: Restart your modem and try a different port on the router. If the port being used is broken or doesn’t work well, you cannot be connected to the router. Unplug the cable from the port and plug it into another port to see whether the problem resolves.

Method 2: Check the cable. Make sure the cable is not broken. Swap a cable to do a test. If the problem is caused by the cable, it will work after swapping the cable.

Method 3: Enable Ethernet. 

1. Press Win+R (Windows key and R key) at the same time.  A Run dialog will appear.

2. Type devmgmt.msc in the run box and click on OK button. The Device Manager window will pop up.

 


3. In Device Manager, expand Network adapters category.

4. Next to the Ethernet card device name, if you see a computer icon with an arrow on it, the Ethernet has been disabled.

Following screenshot is for your reference.


5. Then right-click on the device name. Click Enable from the pop up menu.

After doing the said steps, kindly restart your computer.

If you still encounter any issues, you may follow the series of troubleshooting steps on the Fix network connection issues in Windows article.

Should you need further assistance, please let us know.
