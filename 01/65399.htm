j. brooke say:

This problem has disappeared for me.  I think I solved it with a settings change on my router, but since it was always an intermittent problem, it's impossible to be sure.

The culprit, I believe, was a service on the Buffalo routers (and other DD-WRT routers) called "DNSMasq".  DNSMasq, from what I've read, is both a DHCP server and a DNS cache.  I don't know why it would be a problem, but I ran across a few posts with familiar-sounding symptoms interacting with Xbox Live.  Since I turned it off, it has been smooth sailing for me.

On my router, it's necessary to turn it off in two places.

1. Setup -> Basic Setup -> Network Address Server Settings (DHCP).  Uncheck "Use DNSMasq for DHCP" and uncheck "Use DNSMasq for DNS".

2. Services -> Services -> DNSMasq.  Select the "disable" radio button.

Then save, apply, reboot, the usual.

Hopefully this will save some of the rest of you some pain.  Happy zombie hunting, or whatever y'all are into!
