If you don't want your son to create or elevate accounts, don't let him have an Administrator account.

Accounts with Administrator permissions are only intended for tasks that require elevated permissions. A computer only needs one account with Administrator permissions. All other accounts should be Standard accounts. A Standard account is the one to use for day-to-day computing.

Since Administrator accounts have the ability to create other user accounts, they can also create other Administrator accounts. But that's not and never was the way an operating system was intended to be used.
