Jeff,

Before you follow Frederik's sensible advice to search using Google, please try a different image restoration method. 

Restore the image by booting up outside of Windows rather than by using Windows inbuilt method.  This alternative method will be independent of any problems that might exist on your system such as that VSCS issue mentioned above. 

I assume that your image is in its correct folder on your external HDD, WindowsImageBackup 

You will need either a Windows 7 installation disk or a System repair disk.  If you don't have either then try to borrow one temporarily - it must be 32 or 64 bit to match your system but does not need to be the same Windows 7 Edition {Home premium etc} .

You can make a System repair disk - see the opening dialog of Windows backup and look on the left-hand side for the link Create a system repair disk.  If that does not work then see SRD making & testing and try downloading an SRD image to transfer to USB if those images are still available.  Note that SRD testing is essential as you want to be sure that any subsequent problems are down to the image and not the tool you are using to restore that image. 

Image restoration steps

1  Disconnect all external devices including the xHDD on which you have the system image.  You can leave a mouse connected.

2  Connect the installation disk / System repair disk. 

3  Reboot.  When the manufacturers logo appears, start pressing the key* identified onscreen at that point [or in your PC manual] to get into boot options, then select USB / DVD booting as appropriate for the media you are booting from.  {* On Dells you use the F12 key but it varies between PC makers}

4.1  When the installation disk has completed booting, choose your language, time format, keyboard layout then select Repair your computer.  [Fully illustrated procedure - Recovery method 12 - Windows Backup System Image restoration using Windows 7 Installation Disk]

OR

4.2  When the System repair disk has completed booting, choose your language & keyboard layout then click on Next.  [Fully illustrated procedure - Recovery method 11 - Windows Backup System Image restoration using System repair disk]

5  [as also stated in the illustrated procedures] At the System Recovery Options dialog, set the radio button for Restore your computer using a system image that you created earlier, connect the xHDD containing your system image then click Next.

6  Follow the dialogs that appear [and are all covered in the illustrated procedures]

Despite the error message you had earlier, I think this method will succeed.  It is the same method you would use to restore an image onto a blank hard drive and any corruption in your existing Windows installation is bypassed [and overwritten].

Important Info If You Have Received a STOP Code

If you received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss. A hardware device, its driver, or related software might have caused this error. If your copy of Windows came with your computer, call the manufacturer of your computer. If you purchased Windows separately from your computer, Microsoft provides support. To find contact info for Microsoft or your computer manufacturer, Contact Support.

If you have experience with computers and want to try to recover from this error, follow the steps provided in the Microsoft article Resolving STOP (Blue Screen) Errors in Windows.

  1. Download and install updates and device drivers for your computer from Windows Update.
