The BIOS is basically the builtin operating system for the actual motherboard to function. You would normally hit something like F12, ESC, Del at power on or restart to access it... set the default boot device for instance.

EFI\UEFI is a newer type of this, and functions a bit differently. For one thing, it can dramatically reduce the boot time...

The way things are set up is that to use Win 8\8.1 with this type of BIOS, and to get all of it's benefits, it is usually necessary to install Win 8\8.1\10TP from a flash drive...

In order to do this the HDD or SSD must be formatted as a GPT type disk...

Until recently all HDDs were for the most part MBR.. which is how the partitioning type info is stored...

About the only time you will see this is the first time the drive is Initialized, or the partitions are deleted.

You could either do this before you start by using a bootable 3rd party tool, or another machine...

If you were to do a Custom Install, you may be able to do it there as well...

BTW... that product key tool is about useless. It may recover your license key OK, but that is of no benefit...

About any time you attempt to use it you will get some kind of error about a invalid key...

It is an OEM preinstalled volume license key which is not accepted to even download a Win 8 ISO....

Basically this type of license is supported by the computer manufacturer, not Microsoft, so the key is about useless, except when read from the BIOS...

Windows needs to read the key from the BIOS... There may be ways to bypass this, but since you are basically starting with a fresh machine it might be best not to try something like that for the long term...

I'm thinking that Windows won't read the key from the BIOS during the install unless the UEFI and flash drive are used.

5 Home Editions:

  • Windows 8.1=Windows 8 Service Pack 1
  • Windows 8.1 Single Language=Windows 8 Single Language Service Pack 1
  • Windows 8.1N=Windows 8N Service Pack 1
  • Windows 8.1 with Bing
  • Windows 8.1 with Bing Single Language

2 Professional Editions:

  • Windows 8.1 Professional=Windows 8 Professional Service Pack 1
  • Windows 8.1 ProfessionalN=Windows 8 ProfessionalN Service Pack 1

3/5 Home Editions:

  • Windows 8.1 ProfessionalN
