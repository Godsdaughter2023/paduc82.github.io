Thank you for writing to Microsoft Community Forums.

The 1607 error may be caused by windows system file damage. An incomplete installation, or an incomplete uninstallation, or improper deletion of application or hardware can be the reason.

I'd suggest you to do a system restore and check if it helps. System restore points will be created automatically when you install an app, or a driver, or an update. To restore Windows:

1. Press 'Windows key + R' to open 'Run command'.

2. Type 'rstrui.exe' without quotes, and press enter.

3. Select the restore point which was created prior to this issue.

4. Click on 'Next'.

5. Click on 'Finish'.

Same issue here! The difference is that I've searched almost every forum and still no viable answer to this issue. I'm well disappointed about the Standard W 8.0 or W 8.1 features overall.

But, as the search got forward I've manage to find this fix. (http://www.howtogeek.com/forum/topic/autoplay-not-working/page/2)

1. Select "START" and "RUN" then type in REGEDIT.

Then,

Go to the registry

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\Current
Version\Policies\Explorer

The issue is actually the presence of the
NoDriveTypeAutoRun here, not the value.

To fix, rename the key to something else (like
XNoDriveTypeAutoRun) and reboot.

That's it!
