Thanks for posting your query to Microsoft forum. I will certainly help you with this.

Two primary causes of these errors are video drivers or BIOS drivers might be corrupt, there can be a corruption in the Windows drive. Let’s try to resolve it by the performing the steps below.

Method 1: Start your computer in safe mode with command prompt and then perform the following steps:

Perform a check disk operation:

a.   In the command prompt, change the location to C: and then type chkdsk /r and press Enter.

Note 1: Answer Y when chkdsk asks you if you want to check the drive next time Windows boots (restarts).

Note 2: /r = The /r switch tells chkdsk to attempt to recover any bad sectors of the hard drive if any are found. A bad sector is a spot on the hard drive that cannot hold saved data anymore. If you’ve any data on that sector, you’ll lose that.

 

Method 2: Rebuild BCD using the Bootrec.exe tool.

To run the Bootrec.exe tool, you must start Windows RE. To do this, follow these steps:

a. Put the Windows 7 installation disc in the disc drive, and then start the computer.

b. Press a key when you are prompted. Select a language, a time, a currency, a keyboard or an input method, and then click Next.

c. Click Repair your computer.

d. Click the operating system that you want to repair, and then click Next.

e. In the System Recovery Options dialog box, click Command Prompt.

f. Type Bootrec.exe, and then press ENTER.

Refer the following link for more information:

Use Bootrec.exe in the Windows RE to troubleshoot startup issues

http://support.microsoft.com/kb/927392

You may also follow the steps in the given Microsoft article below:

Resolving stop (blue screen) errors in Windows 7

System restore: When you use System Restore to restore the computer to a previous state, programs and updates that you installed are removed.

Check disk: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.

DVD Ripping software can be found on the net
but be aware that breaking the copy protection

For copying personal non-commercial DVD files to your
hard drive.....try the following steps:

Insert the DVD in your DVD drive...close any windows
that may launch.

Now...go to...Start / Computer.

Right click the drive letter for the DVD drive and from
the menu...choose...Explore.

Copy the VIDEO_TS folder from the DVD to a folder on
your hard drive. Creating a new folder in your 'Videos' folder
with the same name as the DVD would be a good idea.

Once you have the VIDEO_TS folder saved on your hard
drive...you can play it with the free VLC Media Player.

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

VLC Media Player
http://www.videolan.org/

Once you have VLC Media Player installed...open it and
go to...Media / Advanced Open File / Disc tab / tick DVD /
Browse to the VIDEO_TS folder / OK / Play.

Or...right click the VIDEO_TS folder...from the menu choose...
'Play with VLC media player'
