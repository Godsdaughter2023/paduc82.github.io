DNS server can&#39;t resolve Xbox LIVE server names. - Microsoft Community

a. user say:

DNS server can&#39;t resolve Xbox LIVE server names.

Before getting started with these steps, you will want to make sure that you have your wireless password if you are on a wireless connection. These steps will clear out any manual settings you have, which could be the cause of any DNS errors. We suggest not setting any network settings to manual to ensure there are no issues with them.

Please follow these steps in this order. The simple things are generally the ones that will resolve your issue! :)

Reset network settings to default

Turn your Xbox 360 on. From your Xbox 360 dashboard (the console’s main menu), navigate to the ‘Settings’ hub, and select ‘System'. From there, you can select ‘Network Settings', and the network you’re on (wired or wireless).

Once you’ve chosen the appropriate network, select ‘Configure Network'. Navigate to the ‘Additional Settings’ tab, and select ‘Restore to Factory Defaults'.

If you have a wireless adapter attached to your console, turn off your Xbox 360 console, and unplug your wireless adapter from its USB port.

Unplug your router (and wireless extenders if you have them) and modem, and leave everything unplugged for 5 minutes. Plug the modem back in; be sure to give it at least a full minute to come back online. When it’s up to speed, do the same with your router, and repeat this step for any additional piece of network hardware you’re using. If you have the wireless adapter, plug it back in to your console before continuing to the next step. When all of your network hardware is running, turn on your console. 

Test your Live connection
From the dashboard, navigate to the ‘Settings’ hub, select ‘System’, and then ‘Network Settings’. Select the appropriate network (wired or wireless), and then select ‘Test Xbox Live Connection'. If you are using a wireless connection, you will need to set up your network again with the wireless password.

If you’re still receiving a DNS error, continue to the next step.

Use manual DNS settings

Obtain public DNS server addresses
The automatic settings should be using the DNS servers that your PC is getting. For this reason, we suggest researching public DNS servers that you can use in place of the ones your internet provider supplies.

Enter the public DNS server addresses on your console
In the dashboard menu of your Xbox 360, go to the ‘Settings’ hub, and select the ‘System’ tile. Then, select ‘Network Settings'. Select your network connection (wired or wireless), then select ‘Configure Network'.

On the ‘Basic Settings’ tab, select ‘DNS Settings', and then select ‘Manual'. Here you will be able to select ‘Primary DNS Server’ and ‘Secondary DNS Server’, into which you will enter the first and second addresses you recorded in the last step, respectively. When you have entered them and selected ‘Done’ for each one, press the (B) button and select ‘Test Xbox Live Connection'.

If you’re still encountering a DNS error, move on to the next step.

Connect directly to the modem

Use an ethernet cable to connect your Xbox 360 console to your modem, bypassing the router entirely. Before running a LIVE connection test, do the full network power cycle that we have suggested above, including the test connection step.

I’m still getting a DNS error. What do I do now?

You may want to give your Internet Service provider a call, to see if they have a resolution for the error on their end. Also, feel free to make a post in the forums, though please be sure to check the questions at the bottom of this post, to better help us help you!

Windows 7 Starter, Home Basic, Home Premium, Windows 8.0 Core, Windows 8.1 Core should install Windows 10 Home

Windows 7 Professional, Windows 7 Ultimate, Windows 8.0 Pro, Windows 8.1 Pro should install Windows 10 Pro
