hal hostetler - mvp-outlook say:
Are you sure they've disappeared? When Archiving is enabled, Outlook will create a new Personal Folders file, "Archive.pst", to hold the archived data.  This file is usually located in the same folder as your main .pst file, the normal path is:

C:\Users\(username)\AppData\Local\Microsoft\Outlook\

You should be able to open Archive.pst from within Outlook using 'File|Open|Outlook Data File'.  Outlook wouldn't normally store anything outside of your comnputer, unless it is used in an Exchange Server (business) environment.  Your .PST files, like all data files, should have been backed up with a disk backup application, if that's what you used.

vijay b say: Hi,

Thank you for being a part of Windows 10.

You will not be able to upgrade from Windows 10 32-Bit to Windows 10 64-Bit. It have to be a clean install. If you are having Windows 8.1 or Windows 7 SP1 32-Bit product key, then you have to uninstall the previous operating system and perform clean install of qualifying operating system in 64-Bit and then you can upgrade to Windows 10 64-Bit provided your computer support 64-Bit Architecture.

Hope this helps. Let us know if you have any other issues with Windows in future.
