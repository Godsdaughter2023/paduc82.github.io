d r h say:

Do not touch the System reserved partition (BTW it should not have a drive letter)

Go into the BIOS and check the boot order. First should be the DVD drive and second the drive which has the system reserved partition. On booting, a list of the OS's on the computer should be viewable and from which the required OS can be selected. If this is not the case then the boot manager(s) need repairing.

For W10, go to Update and Security/ Backup and select the link 'Go to Backup and restore (Windows 7)'. On the left side of that page is 'Create a system repair disc'.  Create the repair disc on a DVD disc.

With the disc in the DVD drive, reboot into the repair disc and you should be able to repair the boot managers for the other drives (exact details are a distant memory).

To remove the drive letter from the system reserved partition, open disc management and right click on the System reserved partition. Select 'change drive letter and paths' and click on 'Remove'.
