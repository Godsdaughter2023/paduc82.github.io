Hi Rosa,

 

Find a paperclip and open it up so you have an inch or two of just the rod.  Somewhere on the CD Drive you will find a very, very tiny hole (about the size of that paperclip tip).  Insert the paperclip tip into the hole and push firmly.  This should open the drive "manually".  Then you can remove the CD.

 

Try another you know works and see if the problem persists.  If so, you may have a defective drive or need to install new device drivers for it from the computer manufacturer's web site (or the CD drive manufacturer's web site).

 

If that still doesn't work and you have a working drive that fits on another computer, swap them and see if the problem persists (or moves).  This will help tell if it's the computer or the drive.

 

After that you should post back to the hardware forum for further instructions, replace the drive if appropriate, or have the computer serviced.  I'd start by getting help from the Windows-Hardware forum here.

 

I hope this helps and the the first step is the only one you really need here.
