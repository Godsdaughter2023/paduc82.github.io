jimnc9 say:

I cleaned the registry as described above, restarted, but the incons came back!!!


Then I found this. It works...


1. Open the Registry Editor
2. If you use Windows 7 browse to:
    HKEY_CURRENT_USER\Software\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\TrayNotify
3. Delete the data of the following keys in this folder:
    IconStreams
    LastAdvertisement
    PastIconsStream
    PromotedIconCache
    UserStartTime
4. Open the Task Manager.
   Switch to the process tab.
   Kill every explorer.exe process
   (this is necessary because the explorer.exe has a backup copy of this keys in the RAM and writes it back to the registry on shutdown)
   Switch to the application tab of the Task Manager.
   Press "New Task".
   Type "explorer.exe" and press [Enter].

Credit to: HeikoM

If the background image your working with is the same size as the
screen resolution, changing those settings won't display any
different. Example, the image is 1920x1080 pixels and your
monitor's resolution is also 1920x1080, it will display identical
at all of the display settings.
 
Find out what the size of the image is vs. your screen
resolution.
 
To actually changed the size of the image you'll need to do that
in an image editor, i.e., such as Paint (which is probably
already installed on your machine).
