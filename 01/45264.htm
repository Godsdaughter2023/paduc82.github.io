delta__vee say:
Keyboard layouts in Windows Vista / Windows 7

What to watch out for:
  • When Windows is installed, for some regions it sets up multiple keyboard layouts for a single language by default
  • Each open application has its own current keyboard layout 
  • If you have multiple layouts set up you can see what's selected only if you click on the keyboard icon on the taskbar
  • Pressing CTRL-SHIFT will change between the layouts without any visible indication
  • Keyboard layouts are configured under the Region and Language control panel, not the Keyboard control panel
  • Removing a keyboard layout in the control panel will not affect any open applications that already have that layout selected
  • Removing all the keyboard layouts but one in the control panel will make it impossible to change keyboard layouts even though some open applications may have other layouts selected
To go to a single keyboard layout in Windows 7:
  1. Save your work and close all applications, because the gotchas above make it very confusing if you don't restart immediately after changing the settings.
  2. Go to the Region and Language control panel (you can find it easily by searching in the box on the start menu)
  3. Choose the Keyboards and Languages tab
  4. Click on the Change Keyboards button. This will bring up the "Text Services and Input Languages" dialog
  5. Go through each of the extra layouts you don't want in the "Installed Services" box, selecting them and clicking on the Remove button, until you're left with just the layout you want. 
  6. Click OK on the Text Services and Input Languages dialog
  7. Click OK on the Region and Language control panel.
  8. If you want to avoid problems with open applications stuck in some other keyboard layout, restart Windows.
Also, from my experience, when you choose English (Canada) during a fresh Windows 7 install, it sets up one language (English) with multiple keyboard layouts (US English, French Canadian, and Canadian Multilingual Standard), and sets the default keyboard layout to US English.
