    1. Which application are you using to perform the print?

    2. Did you try pressing Ctrl+P while using the application to see if the Print Wizard opens?

    Each desktop app determines how you print from that app, so the way to print will vary from app to app. Here are some of the most common ways to print from desktop apps:


    1. Look in the File menu. It's very common for a Print option to be located in the app's File menu.

    2. Press Ctrl+P.  This might bring up the app's Print dialog box.

    3. Look for a print icon or button. App manufacturers commonly put a print icon or button somewhere on the screen. These are commonly located on a toolbar positioned across the top or bottom of the app.
     
    If you are trying to Print a web page then follow the steps below.
     
    1. Click the menu button and then click on Print.
    2. In the standard Print window that opens, adjust settings for what you're about to print, if necessary. See Print window settings below for information on what settings can be changed.
    3. Click on Print.

    Assuming you are reading your email from Windows 10's built-in Mail app.
    Click at the mail you want to print > top right, click the 3-dots icon > click Print at the drop down box.
