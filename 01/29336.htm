Thank you for posting your query in Microsoft Community.

Apologize for the difficulties that you're experiencing in Windows 10.

The kernel power event ID 41 error occurs when the computer is shut down, or it restarts unexpectedly. When a computer that is running Windows starts, a check is performed to determine whether the computer was shut down cleanly. If the computer was not shut down cleanly, a Kernel Power Event 41 message is generated.

An event 41 is used to report that something unexpected happened that prevented Windows from shutting down correctly. There may be insufficient information to explicitly define what happened. To determine what may have happened and to identify a potential resolution, it is important to know what the computer was doing at the time just before the event occurred.

If event 41 is logged because power to the computer was interrupted, consider obtaining an uninterruptible power supply (UPS) such as a battery backup power supply. An underpowered or failing power supply may cause this behavior. For example, if you added RAM or additional devices or hard disks when this problem began, the power supply may cause the problem.

Xbox refusing to rip CD tracks

son ala say:

ok...a few things.

You can store a few MP3's (some formats may not work) onto a usb and play them that way. I did this for a while when I got my new hard drive since it had no music stored on it. A few things to note with this is the folder limitations. Dont just put 1000 songs on a drive and leave it at that, try to make a bunch of folders and sub folders.
When reading MP3 tracks from any device, the xbox will only read the first 50 files. So if you have an artist with 5 albums, dont just have a folder that says the artist's name and throw all of their track into it. Have the main folder and inside that a sub folder for every album..this way every single track can be read by your console.

That happened to me on my new HDD. I have a USB with a live album that consisted of a 4 night event with every song in order, back to back but that 1 folder has over 100 tracks so I needed to break them down into subfolders Night 1-4.

When it comes to ripping a CD, it has to be an audio disc...not a data disc. If you have a CD-RW with 40 tracks stored as data, the 360 will play these while on the dashboard but it wont rip them.

When burning a disc, select audio format and while you're filling the disc with content, you should see a bar fill up that says 20 minutes out of 80 minutes used
instead of 100mb out of 700mb used.

When it comes to retail disc, the only time you wont be able to rip is if the CD itself has the anti piracy lock on it...and that would normally prevent you from ripping on anything.
