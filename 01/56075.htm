DMZ

Login Information

  • Username - admin
  • Password - MI424-WR *Rev A - Rev D* create a password/password1/Serial number on back of device (While entering in password it will pop up with three *** for every letter you type in)

DMZ

  1. Open a web browser.
  2. Enter 192.168.1.1 in the address bar.
  3. This device has several sets of login information:
    (admin/password, admin/password1, admin/password on sticker, admin/serial number beginning with CSJ).

  5. Select Yes when prompted to proceed.
  6. From the menu on the left select DMZ Host.
  7. Check the DMZ Host IP Address box.
  8. Enter the IP Address of your Xbox Console in the boxes to the right.
  9. Click Apply.

Disable DMZ

  1. Open a web browser.

canadian tech say:

So many people seem to be enamored of updating drivers.  Bad mistake!  Drivers should not be updated unless you are solving a specific problem and then only the specific driver that is causing the problem.  The only source for drivers should the OEM support site for your computer.

A tip:

Restore points

The default restore point space allotment in Windows 7 is 3% of drive capacity.  In almost all cases, the computer is using but a very small proportion of the space available.  That makes it very practical to raise that 3% to say 8 to 10%.  This strategy will yield far more restore points that will give you much more flexibility in recovering a bad situation.

To change the allotment, type create in the text box above the start globe.  Choose Create a restore point.  Click configure.  Move the slide to the right as much as you see fit.  Setting it at 50 Gigabytes will have no affect if you have a 500 Gigabyte drive and are only using about 100 or so, which is highly typical.
