A %Temp% folder is the folder which contains all files and folders which are longer being used by Windows XP and can safely be deleted.

All of the folders and files you see in this Temp folder are no longer being used by Windows XP and can safely be deleted.
c. To remove individual folders or files, hold down the Ctrl key while left-clicking on each item you want delete. Release the Ctrl key when you're finished.

g. Confirm that you want to delete the files by clicking Yes on the Confirm Multiple File Delete window that opens.

    The major anniversary update finishes a lot of work after it's done installing, and that work can take several days to finish.  Or even longer if the computer doesn't get much "idle time" to finish. So if you just put it to sleep or turn it off when you're done, it could help it finish by simply leaving it on and idle for several hours or overnight so it can finish the work.

    But that's just one possibility, so let's try to find out what's causing the sluggishness.  To do this, just press Ctrl-Shift-Escape.  Task Manager will pop open, and you can click the CPU column to sort it from most-to-least.  Then just look and let me know what is using the most of your CPU.  If the CPU is consistently 10% used or higher, your computer will feel very slow and sluggish.  Even if you have dozens or hundreds of programs running, the CPU should not be consistently held over 10%.  For example, I have more than 100 programs running right now, including dozens of websites, and as I type this my CPU is still only 3% used (example below).  So if we can find out what is hogging all your computer's CPU, we can determine how to fix it (either by letting it finish working, or by killing the culprit).  So let us know and we'll proceed.

    andre da costa say:
    Open Start, type: CMD
    Right click CMD
    Click Run as administrator

    At the command prompt, type:

    netsh int ip reset

    Hit Enter
    Exit the prompt then restart

    Open Start > Settings > Update & security > Troubleshoot
    Scroll down
    Click Network adapters
    Click Run the Troubleshooter

    When complete, restart to see if the problem is resolved.

    If not, press Windows key + X
    Click Device Manager
    Expand Network adapters

    Right click your adapter
    Click Uninstall

    Exit Device Manager, restart. Windows 10 should detect the network adapter then reinstall it. Check if you can connect and browse.

    also try:

    Open Start > Settings > Network & Internet > Status

    Scroll to the bottom then click Network reset.

    or

    Open Start then type: regedit
    Right click Regedit then click Run as administrator
    Navigate to the following registry key:

    HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\WlanSvc\Parameters\HostedNetworkSettings\

    Delete the following key:
    HostedNetworkSettings

    Restart then try again.

    Run the Network adapter troubleshooter

    Open Start > Settings > Update & security > Troubleshoot
    Scroll down, select Network adapters
    Click Run the troubleshooter

    When complete, restart to see if it works.

    If not, you can try updating both your Network adapter drivers or uninstall them.

    Press Windows key + X
    Click Device Manager
    Expand network adapters
    Right click your adapter, click uninstall

    Exit Device Manager then restart

    Windows 10 will re-detect them and reinstall them.

    Change Wireless Channel - Microsoft Community

    xb1r athena say:

    Change Wireless Channel

    Login Information

    • Gateway - 192.168.2.1
    • Username - N/A (None)
    • Password - N/A (None)

     Changing the Wireless Channel

    1. Plug the computer into a LAN port via an ethernet cable.
    2. Open a web browser.
    3. In the address bar at the top, type in 192.168.2.1
    4. Click Login in the upper right.
    5. Input the password and click Login (by default, the password should be left blank).
    6. Select Channel and SSID from the menu on the left under the Wireless section.
    7. From the dropdown menu labeled Wireless Channel select the desired channel. (Channels 1,3,9,11)
    8. Select Apply Changes at the bottom of the page.
