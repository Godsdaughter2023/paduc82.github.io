With a desktop you need to remove the left hand side panel and remove any dust bunnies clogging fans. Also check that all fans are working. Next switch the computer off again and allow some time for it to cool down. It is best to use a wrist band, which you clip on the computer case to earth you before touching anything inside the casing. You can buy one from a computer accessory shop. Original subject: activation

neilpzz say: An OEM licence is locked to the first hardware it is installed on, so no it cannot be transferred to a new device. Definition of device is the motherboard.

jimwynne say: In addition to neilpzz's reply, the only exception is in the case of a direct replacement motherboard from the computer manufacturer.

lyrrad ✨ say: It is not possible to use a OEM license on a different computer. as the manufacturers license is only intended to use on the same computer where it was Pre-installed.

andre da costa say:

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system 

So, you would need to purchase a new license.
