Downloads + Product keys

You can download your software as soon as your purchase is complete, and you can also download it later from the Downloads + Product Keys section of your Account. Did the PC come with Windows 8 preinstalled at factory, yes or no? 

Either way follow the steps in Clean Install Windows 8.1 paying special attention to Step 4 for how to install 8.1 with a WIndows 8 Product Key which should be embedded in motherboard chip if 8 was factory installed. 

You can read the embedded Windows 8 Product Key using ProduKey which will report it only as an OEM BIOS key. 

After installing with the Generic Key given for your version, uninstall the key from desktop using Command Prompt  running slmgr -upk, then type SLUI 3 to open activation box to insert the embedded Product Key which should activate.

If you have UEFI then you'll also need to boot the media as a UEFI device only.
