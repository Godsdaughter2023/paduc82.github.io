    Security for sign-in passwords is a lot better than it used to be back in the days of Windows 95/98. The user password has been assigned by the previous owner and if you don't know the password then you can't sign-in to that account. Really the shop should not sell the device like this.

    This is not a fault it is designed that way. MS will not be "fixing" anything.

    You could try to enable the built in Administrator account and see if that will allow you to change the user password for "owner".

    At the sign-in screen select Power (in the bottom right hand corner) and hold down shift and then select Restart. Once restarted select Troubleshoot and the Advanced options and then Command prompt. Type in net user administrator /active:yes and press return. Restart and if it has enabled the account it will be listed in the left hand of the sign-in screen. Or you could create a new one but make sure it's set as administrator. Either way then sign-in to the account and disable the built in Administrator account using net user administrator /active:no. You need to do this using a Command prompt (Admin)

    Never use the built in Administrator account for everyday use.
