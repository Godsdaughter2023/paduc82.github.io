lemp say:

Although using Start > Default Programs > Associate a file type is good advice in general, it's not so helpful for zip files.

For one thing, if a user hasn't installed another archiving utility and configured that utility to open zip files, the .zip file extension won't appear in the list under "Associate a file type" (and it might not appear even then).

For another thing, the default handler for compressed folders (zip files) in Windows 7 isn't an exe program but a dll file.

To correct the association for *.zip files in Windows 7, go here:  http://www.sevenforums.com/tutorials/19449-default-file-type-associations-restore.html

Scroll to the bottom of the list and click on the blue "ZIP" to download the file named Default_ZIP.reg

Being sure that you are logged in as a user with "administrator" privileges, find the file that you downloaded and either double-click it or right-click and select "Merge."  Answer Yes if prompted.
