Please answer the following questions:
Is the version of Windows 8 activated on the computer?

If the version of Windows 8 is not activated, you may not be able to change wallpapers on the desktop screen.

You can follow the steps to check if Windows is activated:
a) Type Computer on the Start screen.
b) Right Click on Computer.
c) Select Properties from the options on the bottom of the Window.
d) Check whether Windows is Activated or not.

If it is not activated, you can follow the methods to activate Windows 8 and then try settings up wallpapers:

Method 1:
You can follow the steps to activate Windows 8:
a) Press Windows key + R together on the keyboard to open Run box.
b) Type “slui 3” without quotes and press Enter.
c) Type the product key and click Activate.
 
Method 2:
You can also follow the steps to activate Windows using phone activation:
a) Press Windows key + X and click Run.
b) Type “ slui.exe 4” without and press the 'Enter' key.
c) Select your 'Country' from the list.
d) Choose the 'Phone Activation' option.
e) Stay on the phone (do not select/press any options) and wait for a person to help you with activation.
f) Make sure that you elaborate the issue properly to the support person.

Try to set wallpapers once Windows is activated.
