a. user say:

Hi Asif,

Thank you posting your query on Microsoft Community.

This issue might have caused due to hardware issues or change in USB settings.

To assist you better, I would require the following information:

1. Are you able to use any other external USB devices on that USB port?

2. Did you made any changes to the computer prior to the issue?

Please try to insert the Kingston USB device on a different port and check.

Also try to insert the Kingston USB on a different computer and check the behavior.

To resolve the issue, refer to the below methods.

Method 1

I would suggest you to run the hardware troubleshooter to resolve the issue and check if it helps.

It will automatically diagnose and fix problems with hardware. (USB) Mice and keyboards are not detected by the device manager or when they are not working.

https://support.microsoft.com/en-us/mats/hardware_device_problems/en-us?wa=wsignin1.0

Method 2

I would suggest you to check the status of Kingston USB in Device manager and check if you receives any error on exclamation mark when you insert the Kingston USB.

1. Press Windows Key + R Key.

2. Type “devmgmt.msc” without the quotes and Hit on Enter Key.

3. Search for the Kingston Device and check whether any error on exclamation mark is present on the device.

4. Right click on the Kingston device in the list.

5. Click on Properties and check whether the device is working properly or not.

If the issue still persist, refer to below method.

 

Method 3

If you are able to use other USB device on the same USB port. Then, I suggest you to format Kingston USB device and check if it helps.

1. Insert Kingston USB device into the computer.

2. In the file explorer, right click on the Kingston USB device.

3. Click on Format.

4. Click on Start to format the device.
