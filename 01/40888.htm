a. user say:

Backup and Restore (Windows 7), despite its name, is not a backup program. It's for restoring backups made on Windows 7 to Windows 10.

Backing up with Windows 10's own backup utilities is less than satisfying. The file backup program (File History) doesn't make backups of all your personal data at once; it only backs up individual files. And the imaging program (System Image) lacks many features common to third-party programs and requires Windows to be running in order to restore an image.

There are piles of excellent backup programs available, and you should take advantage of them. Your big problem will be deciding among the many good choices. For file and folder backups have a look at Backup4All, if only to see just what great backup software is capable of. For making disk images look at Macrium Reflect which will give you a good idea of what disk imaging can do. These are only two examples of backup software, and there are lots of other terrific applications.

Once you get into the habit of backing up, you'll wonder how you ever used a computer without it.

Close Device Manager and restart your system
