How do I set up a new administrator account - Microsoft Community

judiththorpe say:

How do I set up a new administrator account

I have installed Windows 8 but typed in my email address incorrectly and this is the administrator account. I would like to set up a new administrator and then delete the incorrect one.
andre da costa say:

a.      Press "Windows + R" to get the Run Window

b.      In the search field type Control Panel

c.       Type User account at the right corner search bar

d.      Select create an account and then select other user in pc settings

e.      Under other users select Add user type in the email address and then click on Next

f.       Select the radio button titled other, and from the drop-down menu, choose Administrators.

g.      Click Finish, which will take you back to the User Accounts dialog box. Click OK.
