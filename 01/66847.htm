How to disable Hotmail email sync'ing? - Microsoft Community

davidaldrich say:

How to disable Hotmail email sync'ing?

My wife and I share a Hotmail account.  We each have a laptop running Windows Live Mail.  We each want to file or delete email without affecting the other person.  However, this doesn't happen by default.  Presently, if I delete an email in Live Mail, that email will shortly disappear from my wife's Inbox.  Is there anyway that I can disable that sync'ing?

 

To make things more complicated, we would like to retain sync'ing of calendars and contacts.  And, wife uses Windows Phone 7.5 as well as Live mail.

¡firedog say:

Synchronization between Windows Live Mail and Hotmail means that, as far as possible, folders and their contents are identical on both the client and the server. It follows that deleting a message in a client will cause it to be deleted from the same folder on the server. It will thus be deleted from the same folder on any other client that subsequently synchronizes with the same account on the server. You can thus only achieve (1) by using POP3, which only downloads new messages from the account inbox. There is no synchronization, so sent messages will only be stored on the computer they're sent from.

The contacts and calendar you see in Windows Live Mail are determined solely by the account you use to sign in with. This is quite separate from the messaging functions, so if you both sign in with the same account, your contacts and calendar will remain in sync regardless of how you arrange your mail accounts.

cmdrkeene say:

The easiest way to really fix this once and for all is to create a new admin-level user account on the computer. Don't worry, this is a non-destructive process (nothing will be harmed by trying it).

Because profiles are separated from each other and the system, creating a new profile on your computer can help troubleshoot and resolve issues. Many times an issue in a profile can be resolved by creating a new one. It takes a little time to set up your preferences and drag your files to the new profile, but it can save many hours of headache. As with the System Restore, this is a “safe option”, because no files will be lost if you try this.

  1. Open the Control Panel or Settings app, go to Accounts and under Other Accounts you can add an account.
    • choose “this person doesn’t have an email” and then “Sign in without a Microsoft Account”.
  2. Login to the new account and see if the issue is fixed. If so, you can open the C:\Users folder and drag your files into the new profile. Although there’s no rush to do it, later you can delete the broken account.

To restore this computer, Windows needs to format the drive that the - Microsoft Community

william838 say:

To restore this computer, Windows needs to format the drive that the Windows Recovery Environment is currently running on.

I had a system with two USB drives plugged in. There was a restore disk on one USB drive and a recovery image on another USB drive.

I had already restored my notebook- but not from a system image.

I was trying to overwrite the notebook restore with a system image restore and got the title message. I Googled and found advice saying that Windows backup and restore is buggy- don't use it. Not exactly helpful, which is why I am writing this.

I decided to make sure that I was running the USB system and discovered that the message was telling me (correctly) that though I had a bootable USB plugged in the system was not running from it.

It had booted using the C drive and the C drive would get zapped by the restore. Restore would fail if it was allowed to run, so it was not allowed to run.

There is on-line help regarding how to select the boot device. I cannot remember how I found it but I remember that there was a blue page listing available boot options (my USB was plugged in and findable as one option) and I chose the Boot device I wanted. It was this screen that told me that the system had booted from the C drive.

When I booted from the USB restore disk the subject message disappeared from System Image Recovery. One note though- be sure that when you get the option of removing disks from the formatting stage you do remove your boot disk. My boot USB was ticked for formatting till I removed it.
