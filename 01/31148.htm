j w stuart say:

If you want a fresh install of Windows 7 on the SSD (does not keep your data files and applications) then you can skip restoring the Image Backup to the SSD. However if you want to save the setting of the current installation of Windows on the hard drive then do the following.

1) Create an Image Backup of the hard disk drive's 100MB 'System' partition and the C: Windows partition to a folder (Give it a meaningful name Example: HDD Windows Backup) on external USB drive. This is called a "System" backup as it will not include any other partitions (such as the D: and E: partitions). Note: In the Acronis 'Option' be sure you change the type of backup from 'Incremental' to 'Custom / Full' and Save the settings. Also make sure that 'Scheduled Backup is disabled.

2) Next create an Image backup of just the Games and Other partition to a different folder on the external USD drive (folder name example: Games and Other). Verify that you are creating a "Full" and not an "Incremental" backup. This is called a Drive and Partition backup (you should remove the check marks from the System and Windows partition as you have already created a backup of these two partitions in step #1.)

3) Create the bootable Acronis 'Recovery' media DVD. Test the media to see if you can boot from the CD/DVD and locate the Image backup on your USB drive.

4) Remove the data cable from the Hard Disk Drive and connect it and a power cable to the SSD.

5) Restore the Windows Image Backup to the SSD using the bootable recovery media you created.

6) Verify you can boot from the SSD.

7) With the computer turned off, connect the hard drive to another SATA and power cable.

8) When you power on the computer enter the BIOS and change the Boot order

From: 1st boot device = DVD drive, 2nd boot device = HDD

To: 1st boot device = DVD drive, 2nd boot device = Your SSD.

Exit and save the changes.

9) Boot to the SSD.

10) Now delete the 100MB 'System' partition and the Windows partition on the Hard Disk drive.

11) Verify Windows still loads (From the SSD) by restarting the computer.

12) Open Disk Management and if necessary change the drive letters of the Games and Other partition back to D: and E:

13) You will now have some "Unallocated" space on the HDD. You choice as what you want to use it for.
