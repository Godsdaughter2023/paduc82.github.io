Press Windows key + X

Click Command Prompt (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

To fix the problem “update 3 of 3 0% complete and then restarts computer to same place and cycles.” With computer that is using Windows Vista and you do not have the Windows Vista disc.

1)Restart/Power On computer and before it starts begin taping the F11 key.

2)When in the Computer Recovery mode, go to Advanced Options.

3)Select Computer Checkup

4)Select the Command Prompt and enter

5)At Command Prompt screen type regedit and then Enter

6)In regedit, select HKEY_LOCAL_MACHINE

7)On the File menu, click Load Hive

8) After Load Hive executes go up to top in Look In to find C: drive.

9)Find C:\windows\winsxs folder and find pending.xml at the bottom of the C:\windows\winsxs folder.

10)Rename pending.xml to pending.old

11)Locate the folder: C:\Windows\system32\config\components and click file Components

12)At the prompt for a name, type Offline_Components

13)In regedit find HKEY_LOCAL_MACHINE and open Components folder

14)Delete both files:

a)Offline Components\AdvancedInstallersNeedResovling

b)Offline Components\PendingXmlIdentifier

15)Exit/close registry editor

16) At the command prompt screen, type Exit to exit command screen.

17)Close Recovery manager by clicking cancel and computer should restart without a problem.

This should take you to the normal startup and get you back into Windows.
