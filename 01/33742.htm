Google Apps Sync and Outlook 2013 - a workaround hack/fix


30 MAY13


This is a bit more technical than our normal blogs, but we're posting it online for reference in order to help others. We have been using and providing Google Apps as an email service to our clients for a number of years now, and have been very happy with the functionality it provides. A key feature is the ability for our clients to continue to use Outlook to collect their email, but have the benefits a cloud email service brings.


 


Unfortunately, Microsoft has changed the way Outlook 2013 works - this has the knock-on effect that Google Apps Sync no longer works with it "out of the box". The team at PCM have been scouring online forums and carrying out research to find a workaround. Our thanks go to members of the Google Apps and Gmail forums (especially Ericbe) for the information provided there. We've gathered all the information together, tested it, and have then drawn up the following steps for users to follow.


 


Warning blurb:


This is not an official supported method. It is not recommended by either Microsoft or Google. It should work but there is no guarantee it will. We are not responsible for any damage done to your computer, registry, Google Profile or any other damage that may appear. It has been tested at PCM and appears to work just fine. Please remember this workaround is provided as is.


 


Requirements (OS, Architecture, Office Versions, etc)


OS: Windows 7, Windows 8 (32bit or 64bit)


Microsoft Office 2010: We used a 32bit trial version


Microsoft Office 2013: - 32bit Only


Google Apps Outlook Sync: latest version


Uninstall ALL existing Office/Outlook software and Google Apps Outlook Sync and Migration Tools from computer


Restart the computer


Install Microsoft Office 2010


Install Google Apps Outlook Sync


Setup new Google Apps Sync user


Open Outlook just to see if opens fine (no need to sync all profile) then close Outlook down


Install Office 2013 - activate product


Go to Control Panel -> Mail -> Make sure it’s selected to use ‘Google Apps” profile and not ‘Outlook’ by default


Do not open Outlook!


Open Registry Editor: regedit


Make sure you first perform full backup of your registry before making any changes!


Navigate to following key:


HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows Messaging Subsystem\Profiles


Right-click on “Profiles” folder and export this key into a new file on desktop.


Edit exported file from your desktop and perform Find and Replace


Find: Windows NT\CurrentVersion\Windows Messaging Subsystem


Replace with: Office\15.0\Outlook


 


Save this file (or Ctrl+S) and re-import this entry into your Registry Editor


Close down Registry Editor


Open Outlook and enjoy your full syncing Google Apps profile with Microsoft Office 2013


Un-install Office 2010 from Control panel > Program and Features and restart.


After all that, your Outlook 2013 should now be syncing with Google Apps once again!

Hi Jaime,

Thank you for posting the query on Microsoft Community.

I am sorry to know that you are unable to get delete confirmation window to delete files or not in File Explorer. Do not worry, will assist you with the issue.

Does it happen on all the programs or is it only on the File Explorer? 

Log into Windows using administrator account and then try the steps below. 

  1. Right-click on Recycle Bin icon on Desktop and select Properties.
  2. Select Local drive(C). Now enable "Display delete confirmation dialog" option and apply the changes.
