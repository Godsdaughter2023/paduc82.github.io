j w stuart say:

In the Troubleshoot / Advanced options choose the Command Prompt option

At the command prompt type the following four commands:

bootrec.exe  /fixmbr    (Press the ‘Enter’ key to execute the command)

bootrec.exe  /fixboot    (may return an 'Element not found' message)

bootrec.exe  /ScanOs  (Optional)

Note: The /ScanOs option scans all disks for installations that are compatible with Windows Vista, Windows 7 and Windows 10. Additionaly, this option displays the entries that are currently not in the BCD store.

Use this option when there are Windows Vista, Windows 7 or Windows 10 installations that the Boot Manager menu does not list.

Finally enter the following command.

bootrec.exe  /RebuildBcd

When prompted to "Add installation to boot list?" press the letter y (for yes) and tap the enter key.

Now remove the media (DVD or USB) and reboot the computer.
