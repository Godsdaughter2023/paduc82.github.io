john rubdy say:

Hi Mena,

Error 1722 is an InstallShield error code. It indicates that the installation process has failed. Please follow the steps below for resolving this issue.

 

 

Method 1: Make sure that the Windows Installer service is not set to Disabled

  1. Press Windows key + R, type services.msc, and then press Enter to open Services.
  2. Right-click the Windows Installer service, and then click Properties.
  3. If the Startup type box is set to Disabled, change it to Manual.
  4. Click OK to close the Properties window.
  5. Right-click the Windows Installer service, and then click Start. The service should start without errors.
  6. Try to install or to uninstall again.

 

 

Method 2: Unregister and re-register the Windows Installer:

  1. Click Start, click Run, type MSIEXEC /UNREGISTER, and then click OK. Even if you do this correctly, it may look like nothing occurs.
  2. Click Start, click Run, type MSIEXEC /REGSERVER, and then click OK. Even if you do this correctly, it may look like nothing occurs, or you may briefly see an hourglass. After you run this command, the operation is complete.

Try uninstalling the application again.

 

 

Method 3: Run this Microsoft fixit tool and check if it helps:

Fix problems that programs cannot be installed or uninstalled

http://support.microsoft.com/mats/program_install_and_uninstall/

Volg de onderstaande stappen als er een rood lampje brandt op de sensor en u een van de volgende foutberichten ziet op het scherm:

Kinect-sensor kan niet bewegen (C0051201)
De Kinect-sensor kan onvoldoende bewegen (C005120A)

Zo kunt u dit probleem verhelpen:

Als het Kinect-sensorlampje nog steeds rood is, controleert u of de consolesoftware actueel is. Dat doet u als volgt:

  1. Zet de Xbox 360-console uit.
  2. Verwijder indien nodig het verpakkingsschuim tussen de voet van de sensor en de sensorkop.
    Opmerking Draai niet zelf aan de sensor.
  3. Zorg dat de sensor zonder belemmering omhoog of omlaag kan kantelen.
  4. Zorg dat de sensor op een vlak en stabiel oppervlak staat. Leer meer over plaatsing van de sensor.
  5. Zet de console aan om te zien of het probleem is verholpen.

Als het probleem met deze oplossingen niet is verholpen, neemt u contact op met Xbox Support.
