    Here you go. Some careful typing is required!

    1. While in the WRE, find the current drive letter where Windows resides. It can be C:, D:, E:, F: but it cannot be X:. I shall assume it is F:
    2. Change to the folder where the registry resides: 
      cd  /d  F:\Windows\System32\config
    3. Create a backup copy of the System hive:
      xcopy   System  \RegistryBackup\
    4. Open the registry editor with regedit.exe.
    5. Navigate to HKLM.
    6. Click File / Load Hive, then specify
      F:\Windows\System32\config\System
      and press Enter.
    7. Give the baby a name, e.g. JohnSystem and press Enter.
    8. Navigate to HKLM\JohnSystem\MountedDevices
    9. Export the key for safety's sake.
    10. Modify the data on the left as you see fit. Changing
      \Dos\Devices\E: to \Dos\Devices\C: will change E: to C:.
      Make sure you have no duplicates.
    11. Navigate to HKLM\JohnSystem.
    12. Click File, then Unload Hive.
    13. Close the registry editor.
