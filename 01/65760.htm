Regarding your OneNote concern, we suggest you to perform the steps listed below:

1. Press Windows logo key + Q.
2. Type Credential Manager.
3. Click Windows Credentials.
4. Under Generic Credentials heading, check for find some credentials with MicrosoftOffice15_Data:live:cid=<16 digit key>. Remove these credentials one by one.
5. Click Yes to the prompt that asks if you if you want to permanently delete the generic credentials.
6. Close the Credentials Manager, then open OneNote to see if the error is still there.

Let us know how it goes.
