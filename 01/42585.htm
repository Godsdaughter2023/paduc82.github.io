Problems caused by an interrupted Windows Update are probably better asked in the Windows Update forum, but I can suggest a few things based on the information you've provided.

Are any error messages received? If so, provide them in full, without paraphrasing.
Check the Event Viewer (Start > Run > eventvwr > OK) for recent error messages (no need to go back very far in time).  Double-click any line in Event Viewer that shows and error to open its "Properties" window for more detail

Event Viewer messages can be copied to the clipboard (and then pasted into an email or forum response) by clicking on the copy icon (below up/down arrows) on the Properties page.

What version of Windows (right-click My Computer > Properties)?

According to the reported data, the computer appears to have a successful wireless connection but the absence of a Default Gateway is suspicious, as is the "No" next to "DHCP Enabled".  I suspect, as Saurabh Joshi apparently also did, that the wireless adapter has been configured manually (i.e., a static IP address).  In any event, you didn't even mention the existence of a wireless network.

The information for the Ethernet (wired) connection definitely shows a failure to obtain an IP address from the router.

Attempting to deal with network issues 3rd hand is really shooting in the dark, but while you are waiting for a response from the Windows Update forum, the following steps shouldn't hurt and might actually fix things.
