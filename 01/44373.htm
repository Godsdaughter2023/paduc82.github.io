After rebooting and before your system starts to load XP, start tapping the F8 key until you get to the Advanced Boot Options menu and from there, chose Last Known Good Configuration.

If you miss the F8 window of opportunity, restart the system and try again.

When your system boots successfully and you shutdown (successfully) you will see that XP says it is "Saving your settings..." and one of the things it does is create what should be a "good" copy of your current registry setup in a separate place in the registry.

When you add some new software or update something that might include a driver, and then your system does not boot (like vidstub.sys), LKGC is the option to choose. 

When you choose LKGC, XP will not try to load the "normal" potentially afflicted registry that points to the troublesome drivers, but will instead use the registry that was saved the last time XP booted successfully.  If the system the boots, you can uninstall or rollback whatever it was that brought on the issue in the first place.

LKGC will not help with many other kinds of no boot scenarios (Sorry, Microsoft Engaged Support Engineer "experts") but is the right choice when you suspect an incompatible driver is the problem.

mikemichaels say:

why does my computer keep waking itself up after i have put it to sleep?

When I put my computer into sleep mode i keep coming back to find it's awake again. This has only begun to happen recently. I am clear of viruses as far as i know. The only other new thing I've done recently is install a new ( canon ) printer connected with a usb. It is switched off though. Any ideas anyone?

 

vishwanath v say:

Hi,

 

· Which operating system are you using (Like XP, Vista or Windows 7)?

 

Method 1: Check which device caused the computer to wake unexpectedly.

 

Open elevated command prompt by following the steps mentioned below:

a)      Click on Start, select All Programs, Followed by Accessories,

b)      Right-click on Command Prompt, and then click Run as administrator.

c)      In the command prompt type in the command powercfg -lastwake without quotes.

This will tell you what woke the computer up.

 

Method 2: Open device manager and uncheck “allow this device to wake the computer” option from network adapter’s properties.

a)      Open Device Manager.

b)      Right-click on the Network adapters present in the device manager.

c)      Select properties, click on power management tab.

d)      Uncheck allow this device to wake the computer

 

Method 3: Try to restore default power plan settings

a)    Open the Control Panel from the Start menu.

b)    Open Power Options in the Control Panel.

c)    Select ‘Change plan settings’ and click on ‘Restore default settings for this plan’

 

Also refer,

http://support.microsoft.com/kb/972664

Note: The steps in the above mentioned article also applies to Windows vista.
