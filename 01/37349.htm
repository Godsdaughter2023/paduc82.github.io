kiran jayakumar say:

Hi,

‘PendingFileRenameOperations’ stores the names of files to be renamed when the system restarts.

 

This entry consists of pairs of file names. The file specified in the first item of the pair is renamed to match the second item of the pair. The system adds this entry to the registry when a user or program tries to rename a file that is in use. The file names are stored in the value of this entry until the system is restarted and they are renamed.

http://technet.microsoft.com/en-us/library/cc960241.aspx

 

If an installer creates an entry under ‘PendingFileRenameOperations’ but does not prompt for a system reboot; you don’t have to restart the computer instead the changes will take effect the next time when the computer reboots.

Hi HUYNHNGHIEM ,

Surface Go power supply unit has 15 volts * 1.6 amps = 24 watts

An AuKey power bank has 5 volts * 10000mAh = 18 watts

From that specification posted above, the power bank can supply a power for the Surface Go since the device needs at least 15 watts of power for it to recharge.
