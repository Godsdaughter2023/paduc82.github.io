bill smithers say:

Hi,

As you are using Vista 32bit, 4GBs of RAM is the maximum amouny of RAM that you can use.

And this explains why there will be a shortfall in the amount of RAM that shows up in the System :

"The system memory that is reported in the System Information dialog box in Windows Vista is less than you expect if 4 GB of RAM is installed"

http://support.microsoft.com/kb/929605/en-us

_________________________________________

To use more RAM, you would need a 64bit Operating System, if your Motherboard was capable of using more RAM.

There is no need to Upgrade the Operating System, unless you wish to.
