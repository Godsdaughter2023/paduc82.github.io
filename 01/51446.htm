pa bear - ms mvp say:

If Windows LIve Essentials (installer) was preinstalled on your computer when you bought it and you just enabled (registered; installed) Windows Live Mail, two things would have happened:

1. Your default update source would have been "flipped" from Windows Update (offers updates for Windows only) to Microsoft Update (updates for Windows Live Essentials, MS Office, Silverlight [1], Microsoft Security Essentials as well as Windows).

2. Automatic Updates would have been changed to the Automatic configuration.

#2 above explains what happened to you.

Note: Both of the above changes are documented deep within the Windows Live Essentials EULA - but very few people bother to read it.

===============================================================
[1] IIRC Silverlight is installed by default when you enable/register/install Windows Live Mail.
