harrisonpickering say:

Hey, I know how annoying and frustrating this can be as I dealt with this problem on my sister's HP Pavilion G6 laptop just a matter of days ago.

I couldn't solve the problem, but I have two solutions to restore your system back to the previous operating system you were running.

The first  and second solutions are cost free, if they work for you.

The last will cost you if you don't have a spare OS lying around on a disc or USB with a product key for it.

1. Safe mode

Now I know that the screen flickering prevents almost everything, but you can open task manager from pressing

On the menustrip at the very top of the task manager, you should see the "File" menuitem. Click it and select the sub-menuitem that says "Run new task" or something along those lines.

In that box, you are going to have to try as best you can to type "msconfig" and click enter. I found it a huge pain and struggle to do this, but I eventually got it. You have to wait until the screen comes back, and then press RIGHT-ARROW and then press the next contributing key to form the word "msconfig". 

Once it has opened, select the "Boot" tab and select "Safe mode" located at the bottom left corner of the tab page.

Shut down the system and boot it back up, even if this means force shutdown via power button on the system itself.

You now want to open the start menu and select "settings", then select the "Update and Security" icon/tab.

Click get started, and you should be able to restore your previous operating system.

If that didn't work for you, then try this second solution, which requires a system restore point.

2. System restore

You need to find some kind of way to access the system recovery

On HP, repeatedly press ESC while the system is booting up, and after until you reach a DOS screen with options.

Then press F11 to get to the system recovery

  1. Select Troubleshoot
  2. Select Advanced Options
  3. Select System Restore
  4. Select the last system restore point you made, or Windows made.

If that didn't work then try the third solution, which requires a copy of an OS with a product key.

3. Clean install

Shut down your system, even if this means force shut down via the power button on the system itself.

If you have a copy of an operating system on a disc, then insert the disc into the system.

Before you reboot your system:

  1. Try find what your system's BIOS key is to access the BIOS ( For HP, typically ESC, then you get some options )
  2. Repeatedly press that BIOS key when you turn the system on, until the BIOS come up

When the BIOS screen appears:

  1. Navigate through the tabs in the BIOS until you find something along the lines of "Boot sequence" or "Boot order".
  2. Navigate down to the list using the UP and DOWN arrow keys.
  3. Use F6 and F5 to change the order of the list.
  4. You want to make sure any form CD-ROM or USB-Flash Drive is above the Boot manager and Hard drive in boot priority.
  5. Now navigate to the exit tab, and click exit and save changes.

After exiting BIOS:

The installer should appear, and I will leave some instructions for a Windows installation here

  1. Select your language ( Select US, even if in UK as UK isn't there, but you can change it later. )
  2. Select Install now.
  3. Enter the product key for the operating system
  4. Accept the license agreement
  5. Select the partition that has a label of Primary on one of the fields of the partition table.
  6. Ignore the warnings it gives you and press okay to continue.

The operating system should now begin to install, and should work fine from there. 

NOTE: Typically, all your old files will be backed up in the Drive letter such as C:\ in a folder called Windows.old

If the installation failed and said the hard drive was locked:

You need to find some kind of way to access the system recovery

On HP, repeatedly press ESC while the system is booting up, and after until you reach a DOS screen with options.

Then press F11 to get to the system recovery

  1. Select Troubleshoot
  2. Select Advanced Options
  3. Select Command Prompt
  4. Run Bootrec /fixMBR
  5. Run Bootrec /fixBoot
  6. Run Bootrec /rebuildBCD
  7. Run exit
  8. Select Troubleshoot ( if not redirected to Troubleshoot > Advanced Options )
  9. Select Advanced Options ( if not redirected to Troubleshoot > Advanced Options )
  10. Select Auto repair or something along those lines

The hard drive should now be unlocked, and you can try the installation once again.

I really hope this helps some of you out there with this problem, as it drove me crazy! It took me a full day to recover the laptop from it's state and back to the previous OS.

  1. Use the File, Run option in Task Manager to open additional tools as-needed. Be sure to check the box to run tools as Administrator. These could include:
    appwiz.cpl:  Opens Programs and Features
    compmgmt.msc: Opens Computer Management
    cmd.exe
  2. You can search for crash events in the Event Logs and find if any third party DLLs are the faulting module in the Explorer,.exe crashes Or uninstall software that is likely to include shell extensions.
  3. If removing software does not resolve the issue, use these commands to repair Windows:
    dism /online /cleanup-image /restorehealth
    sfc /scannow

Removing these apps help resolve the issue:

  • Norton Antivirus
  • Other AV solutions
  • iCloud
  • IDT Audio (faulting module IDTNC64.cpl)

If Task Manager is flickering, or it didn’t open, proceed with display driver troubleshooting:
The likely cause is a display driver or display hardware issue.

  1. Boot to Safe Mode with Networking.
  2. Use the same steps here as for Black Screen issues – if dual chipset, disable built-in graphics, if still flickers, try with only the other chipset enabled.
  3. If this does not work, or there is only a single chipset, uninstall the device and remove the driver software.
  4. Then boot into Windows and update the drivers to the latest from the manufacturer.
