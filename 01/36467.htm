elrumbo say: William,

When rebuilding the index did you try also adding the users folder to the list of included locations? Verify that:

  1. Click start.
  2. Type index.
  3. From the list, click Indexing options.
  4. Verify that the users folder is on the list and that the exclude section has AppData (depending on the number of profiles, you'll see more than two.)
  5. If you don't see the users folder, follow this steps (start at to add a new location to the index...).
By default Win7 doesn't index the contents of C:\Users\username\AppData for all user profiles on your computer. You will have to do that manually. Also, by default, the AppData folder is hidden. To make it visible and exclude them from indexing:
  1. Open Windows Explorer (computer).
  2. Click Organize.
  3. Click Folder and search options.
  4. Click the View tab.
  5. Make sure that the radio button of Show hidden files, folders and drives is selected.
  6. Follow all the steps that I described in the first section of this posting. If you're adding the users folder, make sure that you expand local disk -> users -> username and delete the check mark by the AppData folder.
  7. While in the folder options window, click the Search tab and click the Restore defaults button.
When you finish, I strongly recommend to follow 1-5 just above and to set Don't Show hidden files, folders or drives.
