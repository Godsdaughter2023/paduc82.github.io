waynerupp say:

Raynaldo,

When your computer device is stuck in a black screen or a infinite loop and  unable to access the desktop screen, the following 2-3 hard shut downs and reboots will trigger the Advanced Startup Options menu (ASO).

Hard shut down your device, then perform a reboot, waiting for Manufacturer's logo to appear, immediately another hard shut down.  Repeat this on/off process  until the ASO menu is triggered usually 2-3 times.

Once within the ASO you have many options to choose from.

1.  Continue exit to Windows 10

2.  Troubleshoot

3.  Advanced Options

4.  Startup Repair

5.  System Restore

6.  Safe Mode w/networking in Startup Settings (F4, F5)  then ESC to return to Windows Desktop.

Try many of these one at a time and keep notes of what you have performed to stay organized.

If at any time you are returned to the  stuck black screen, simply repeat the emboldened instructions above, to re-enter the ASO and pick up where you left off.

Sync iPod 4th generation in Windows 10: a How-to - Microsoft Community

ttrout say:

Sync iPod 4th generation in Windows 10: a How-to

It took two iPods and more time than I expected to sync a used iPod 4th generation in Windows 10.  The following may help others who would like to do the same. The version of Windows 10 used was 1511 (build 10586.494). iTunes was 12.4.3.1.

Requirements:

  • Windows 10 32-bit. [With 64-bit, sync would disconnect after 5 +/- songs, reconnect and start sync again.  [There's a lot of overhead to the process; I guessed it would sync about 200 songs an hour.]
  • USB2 port. Connecting the iPod to a USB3 port on my 64-bit system caused the iPod to persistently beep & not connect.
  • Proper driver installed.  If iTunes does not find the iPod, see, for example, this Apple support item.

If you want to try syncing with 64-bit Windows, note that the "official" download may fail with the following error message:

HRESULT: 0x80073715 error when you install an application by using an .msi file if the file is merged with merge modules for the Visual C++ 2005 runtime library

The fix for this to instead install Apple's alternative 64-bit version available here.

Assuming the iPod is now connected to iTunes, the setting that eluded me for some time related to syncing music.  To find this setting, click on the device icon that appears in the top left corner, to the right of the Library dropdown.  In the navigation pane on the left, under Settings, click on Music.  The box for "Sync Music" must be checked. Then click on "Sync" at the lower left corner.

The above is my understanding of the minimum requirements for syncing an iPod 4th generation in Windows 10.  As always, your mileage may vary.  Good luck.

robert j.berg say:

Cumulative Update for Windows 10 for x64-based Systems (KB3081424) - Error 0x80004005

Cumulative Update for Windows 10 for x64-based Systems (KB3081424) - Error 0x80004005
seanob say: THIS SOLVED IT FOR ME 

Reset Windows Update components and install the update.

I suggest you the following steps:

  1. Press Start on the keyboard, type cmd in the search bar and then select “Command Prompt (Admin)” from the menu.

  2. Stop the BITS, Cryptographic, MSI Installer and the Windows Update Services. To do this, type the following commands at a command prompt. Press the “ENTER” key after you type each command.

    • net stop wuauserv

    • net stop cryptSvc

    • net stop bits

    • net stop msiserver

  3. Now rename the SoftwareDistribution and Catroot2 folder. You can do this by typing the following commands in the Command Prompt. Press the “ENTER” key after you type each command.

    • ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

    • ren C:\Windows\System32\catroot2 Catroot2.old

  4. Now, let’s restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt for this. Press the ENTER key after you type each command.

    • net start wuauserv

    • net start cryptSvc

    • net start bits

    • net start msiserver

  5. Type Exit in the Command Prompt to close it.
