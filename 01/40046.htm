samuel christison say:

The "Raptor_v2.3.5_USB3.0_Main_board_VOR2" controller that Sea-gate Externals use before 2014 don't age very well and stop being able to sustain high data rates and long transfers, This leads to an overload of some description and the controller restarts and stops responding to calls from Seagate Manager indefinitely after that point.

If your external isn't showing up or stopped showing up in Seagate Manager but you can still access it and use the files and it seems to be powering down when its under load then the controller is very likely going to die completely soon.

This does not mean you will lose your files, When the controller dies the drive inside the external can be removed and placed into a Desktop PC and at that point it will work perfectly, I just salvaged 6 x 1.5TB Seagate FreePlay 2.5" from faulty externals and they're a great addition to my PC.

TL;DR: The controller in the external is dead/dying and you should pull it apart and put the HDD in your tower.

richardeiler say:

Hi Misla,

By turning it on or off in Setting/Network & Internet/Mobile Hot Spot you are effectively either enabling or disabling the feature on your device. When turned off (disabled) the box in the bottom section of your Internet Access icon on the Task Bar for the Mobile Hot spot should either no longer be seen there or be rendered inactive.

If you have no desire to use it or feel safer having it turned off when not using it turn it off in Settings..., you can always turn it back on if the need or want arises.

1) The password is generated randomly when the service is turned on or enabled, it is displayed to you because it will be needed by the device that will be connecting to the hotspot, just like any protected network. You would be the one to either supply the password to the device user for them to be able to logon or not. The name is the name, basically, that has been given to the host device (your PC). One or both can be changed by you at anytime by clicking on the Edit button in settings.

2) Just turn it off in Settings, if your account (Administrator I assume) is password protected no one else can access (change) the setting unless there is more than one active administrator account on the device or you give your account password out to someone else.

In this way the feature could still be easily used if the need or want ever does present itself.

I hope this helps and that the explanation(s) that I have given have clarified things sufficiently enough to put you at ease.

How to obtain and install Windows 7 SP2 - Microsoft Community

aakkam22 say:

How to obtain and install Windows 7 SP2

Since the release of Windows 7 back in 2009, hundreds of updates have been issued for the aging OS, and anyone who has done a clean install of Windows 7 knows how painful the updating process can be.

Officially, Microsoft only released a single service pack for Windows 7 - Service Pack 1 was released to the public on February 22, 2011. However, despite promising that Windows 7 would only have one service pack, Microsoft decided to release a "convenience rollup" for Windows 7 in May 2016. This single package combines all previously released updates from the release of SP1 to April 2016. So even though Microsoft isn't calling this a service pack, for all intents and purposes, it serves as one.

This article will show you how to install the convenience update (SP2 for Windows 7). 

Note: System Administrators can use DISM to integrate this package into their installation media to simplify future Windows 7 installations.

1. Prerequistes

Before installing the convenience update, make sure:

  • You are running Windows 7 SP1
  • You have at least 5 GB of free space available on your system disk
  • You have installed KB3020369 (download links below)

KB3020369 for x64 systems

KB3020369 for x32 systems

2. Download

Once the prerequisites are met, you can download the convenience update from the links below.

Update for x64 systems (477 MB)

Update for x32 systems (316 MB)

3. Install

When the download has finished, double-click the file, and follow the on-screen instructions. This is a large update, so the searching and installation process may take a while. Please be patient.

Note: Remember, since Microsoft isn't calling this update a service pack, the version number listed in System Properties will remain Windows 7 SP1.

4. Install other Windows Updates

The convenience rollup only contains the updates released after Service Pack 1 and before April 2016. Any updates for Windows 7 that Microsoft has released since then will not be installed, so you should run Windows Update to install any other available updates.

Also, the convenience update does not include Internet Explorer 11 or the latest .NET Framework Updates. You should install those packages manually or from Windows Update if you need them. 

Happy Patching!
