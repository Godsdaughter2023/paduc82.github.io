cari [former ms-mvp] say:

You need to go into the BIOS and stop the PC from looking for a USB device to boot from.  Since you leave your USB drive in situ when booting (starting) the PC, it is looking for the USB device at startup and successfully finding one, but you have no boot files on the drive, so it stalls.

Change the boot order to either ignore the USB device altogether or 'move' the entry to below the Optical drive and Hard drive(s).

Or just remove it before you shut down your PC, and insert it after the PC has started up.

The first means a com object isn't recognised. Something needs to be reinstalled.

 

// MessageId: REGDB_E_CLASSNOTREG
//
// MessageText:
//
// Class not registered
//
#define REGDB_E_CLASSNOTREG              _HRESULT_TYPEDEF_(0x80040154L)

 

The second means a program has run out of memory. This is not related to the amount of memory you have. Adding more won't fix it.

 

Programs store local variables, function parameters, and the address of where to return control to on the stack. A stack must be contiguous. A stack will grow as necessary, it starts at 1 MB (not very much). Your stack has tried to grow and can't. Possibly the memory it wants to grow into is being used.

spiritx say:

Hi,

If it has a WebCam is should also have a mic. The problem might be that the default or older
window drivers are installed which is very typical.

--------------------------------------------------------

How to Enable Hidden Audio Devices in Vista and Windows 7
http://www.vistax64.com/tutorials/143447-audio-device-enable-hidden-devices.html

-----------------------------------------------------------------------------------------------------------------

The drivers might also need to be updated :

Control Panel - Device Manager - SOUNDS - look for HD Audio - Mine says RealTek High Definition
Audio (that is an example and yours may have an entirely different name/maker).

Write down description make and model - double click on it - drivers tab - write down version. Now
Click UPdate Drivers which may not do anything as MS is far behind certifying drivers. Then RIGHT
CLICK on it and UNINSTALL - REBOOT - this will rebuild the driver stack.

If your driver shows 06 you have the windows default and that is part of the issue.

Now go to System Maker's site and get latest Sound/Audio drivers - Download - SAVE - go to where you
put them - Right Click on - RUN AS ADMIN - Reboot

Check the Version on Driver's tab in Device Manager as often sound drivers rollback
and then finally install the current version. Reboot and check after each try, not
unusual to have to run the driver 1-4 times or so. Just make sure the version you
installed shows up in the driver's tab.

Then go to Device Maker's site and look for even newer drivers and repeat install procedure.

Manually look at manufacturer's sites for drivers - and Device Maker's sites.
http://pcsupport.about.com/od/driverssupport/ht/driverdlmfgr.htm

How to Install a Device Driver in Vista Device Manager
http://www.vistax64.com/tutorials/193584-device-manager-install-driver.html

---------------------------------------------------------

After updating driver.

Right click Speaker near Clock - Recording Devices - right click in box - check Show 
Disconnected Devices and Show Disabled Devices - check your mic - highlight them
and then on lower right properties. Make sure the proper one is enabled and check
other settings - on Levels tab make sure MUTE is unchecked and APPLY / OK changes.

Also in Control Panel look for some Audio Manager (maybe named after sound driver maker). Mine
is RealTek Audio Manager, yours can be entirely different name if you have one.

How to Enable Hidden Audio Devices in Vista
http://www.vistax64.com/tutorials/143447-audio-device-enable-hidden-devices.html

Connect a microphone, music player, or other audio device to your computer
http://windowshelp.microsoft.com/Windows/en-US/Help/b0e3d7fb-1b4a-4926-8254-65da289209261033.mspx

Problems with sound and audio or no sound - a Mr Fixit
http://support.microsoft.com/gp/no_sound

Hope this helps.
