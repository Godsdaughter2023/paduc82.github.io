(Google chrome's built-in "snap" feature can use the inside of the monitors). Thanks for your help!

ronnie vernon say:

Hi

The action that snaps a window to the edge of a monitor, when dragging a window, is when the cursor contacts the last row of pixels at the edge of the monitor. Since there is no physical 'edge' where the monitors meet on a dual monitor configuration dragging a window to that location does not work.

You can still snap a window to the inside edge on a dual monitor setup by using the keyboard.

Windows Logo Key+Left arrow key = Snap to left
Windows Logo Key+Right arrow key = Snap to right

Hope this helps.

Thank You for using Windows 7
