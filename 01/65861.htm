To Tricky --

I guess you both have Universal Plug and Play enabled and whatever the discovery protocol is named (I don't recall, and I have both turned off).  That's what puts the "Gateway Device" item in Network Connections.  The idea is to be able to configure the router, for example, without having to access its web configuration utility.

What I assumed -- probably incorrectly -- was that the DSL modem was set to "bridge mode" and that the router was doing the PPPoE login.  AFAIK, that's the more common setup.  Some DSL modems have what amounts to a built-in router (although often there's only one LAN jack).  In this arrangement, the modem is configured with the ISP logon information rather than the router.

The problem with doing things that way is that there are two NAT steps, one in the modem and the other in the router.  Although double NAT is not a good thing, in many situations it will still work, with only a small reduction in performance. I don't think that's the OP's problem.

My next suggestion to the OP (are you reading Roger?) is to reset the TCP/IP stack and reset Winsock. The following instructions assume that you have at least service pack 2 for Windows XP installed.  If you don't, see the second KB article below to reset Winsock using regedit.

  • Log on as a "computer administrator"
  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Type the following command and press Enter
                      netsh int ip reset c:\resetlog.txt

  • Reboot the computer
  • Log on as a "computer administrator"
  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Type the following command and press Enter
                     netsh reset winsock

http://support.microsoft.com/kb/299357

It usually doesn't do any good to manually enter the IP address ... if the adapter is set to get its IP address automatically (the default) and it isn't getting it, something is wrong and manually setting the address won't fix it.

Don't worry about the FireWire adapter.  Having it there isn't bothering anything.

Did you install drivers for the other devices?  I can't think of anything else off the top of my head that would be necessary for the Ethernet adapter to work, but this is Windows, after all.

Now that you have installed the driver for the ethernet adapter, take a look at its properties. You should see this --> http://screenshots.modemhelp.net/screenshots/Windows_XP/Network_Connections/Local_Area_Connection/Properties/General/Index.jpg with at least "Client for Microsoft Networks" and "Internet Protocol (TCP/IP)" checked. 

With TCP/IP selected (as in the image above), click the Properties button and you should see this --> http://screenshots.modemhelp.net/screenshots/Windows_XP/Network_Connections/Local_Area_Connection/Properties/General/TCPIP_Properties/General_(Auto_DNS).jpg

Now do an ipconfig /all command from a Command Prompt window.  The IP address for the "Local Area Connection" should be 192.168.1.xxx and the "Default Gateway" should be 192.168.1.1

As a reminder ...

           ipconfig /all > %userprofile%\Desktop\ipinfo.txt
           notepad %userprofile%\Desktop\ipinfo.txt

It's not clear if you are dual booting Linux with XP on one computer or if your Linux is set up on a different machine.  If Linux is NOT on the same machine as XP, it wouldn't hurt to take a look at the Ethernet jack on the computer and confirm that the "activity" LEDs are lighting up.  (Bad cables/bad hardware are rarer than misconfigured software, but they do happen)

Hi Randy

When all else fails, the manual approach is your best option to transfer the files, the built in Photos App can be a bit unreliable, and you will have more control over the files are transferred . . .

Connect your Phone to your PC, on the phone, swipe down from the top and set USB connection to MTP

Open Windows Fie Explorer (folder icon on your Taskbar)

On the left click 'This PC'

On the right double click your phone or the DCIM folder (whichever shows up)

Then simple select the files you want to transfer and drag them into the Pictures folder in the left pane of File Explorer

That's it!
