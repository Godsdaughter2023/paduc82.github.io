How do I completely reformat the hard drive back to 500Gb?

steve winograd say:

Nothing is missing. You're seeing two different ways of defining a "gigabyte".  Disk manufacturers commonly use the one that gives the bigger number.

From a technical point of view, a gigabyte is 1,073,741,824 (1024x1024x1024) bytes.  But disk manufacturers say that a gigabyte is 1,000,000,000 bytes.  A disk that's advertised as "500 GB" has 50,000,000,000 bytes, which is technically 465.66 GB.

gourav jain say: I guess there are some cached copies of the images you have deleted. 

Do the following:

  • Hover your cursor over the right corner, click Search, type RUN and press Enter.
In the RUN box type: %USERPROFILE%\AppData\Local\Packages\
  • Open the folder with name starting with Microsoft.windowsphoto. Then open LocalState folder.
  • The folder contains cached copies of images which are showing up on Live Tiles. Delete the images you don't like to be shown.
  • After doing it, go to the start screen, turn off the live tile then turn it on again.
  • Done :)

As a piece of advise, use some tools like Tune Up Utilities, CCleaner etc. They can be used to clean all the cache and temp folders with a click and thus reducing a lot of manual work.
