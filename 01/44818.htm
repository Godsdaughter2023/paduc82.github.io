CD /d %userprofile%\AppData\Local
DEL IconCache.db /a
EXIT

TL;DR version:

Windows 10 updates, including Device Manager automatic driver updates, can be disabled with this Group Policy:

Full post:

Since Microsoft was SO helpful here [/sarcasm] I decided to take a closer look at the available Group Policy options. (Start -> Run -> gpedit.msc). First I tried these settings:

Administrative Templates -> System -> Device Installation -> "Specify search order for device driver source locations" enabled, set to "Do not search Windows Update"

Administrative Templates -> System -> Driver Installation -> "Turn off Windows Update device driver search prompt" enabled

Administrative Templates -> System -> Internet Communication Mangement -> Internet Communication settings -> "Turn off Windows Update device driver searching" enabled

Administrative Templates -> System -> Windows Components -> Windows Update -> "Configure Automatic Updates" enabled, set to "2 - Notify for download and notify for install"

Administrative Templates -> System -> Windows Components -> Windows Update -> "Allow Automatic Updates immediate installation" disabled

I then ran my audio driver setup file, but after a restart, Windows immediately installed a new driver. So that didn't work.

I finally resorted to this option:

Administrative Templates -> System -> Internet Communication Mangement -> Internet Communication settings -> "Turn off access to all Windows Update features" enabled
