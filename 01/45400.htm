canadian tech say:

Shawn is absolutely correct.  Do NOT use backup software.  The process is actually quite simple.  

  1. Connect your backup drive
  2. Create a folder named myname&today'sdate
  3. Open a Window with that folder
  4. Open another window with the User folder on your hard drive
  5. Drag and drop (with the right button) folders to the backup folder
  6. When you drop the folder, select Copy from the menu
  7. Do this for Documents, Pictures, Videos, and Music folders.  You will seldom find anything of yours in other folders.

When doing a rebuild:

  1. Type Windows Easy Transfer into the text box that pops up when you click the start button (bottom left) -- this is on the computer BEFORE the re-build.
  2. Connect a backup drive
  3. Create a folder and name it Easy transfer&today'sdate
  4. Follow the prompts. This is the Old computer
  5. This produces a very large single file that can be used for nothing else, but 99% of the time works great.
  6. When you have your newly re-built system in place, after Windows is completely installed and updated, and applications are installed, repeat the same process for Windows Easy transfer choosing "this is the new computer."

Hi,

 

It depends on what type of License you have.

 

If the machine came with the Windows Operating System preinstalled (Dell, HP, Acer, etc), that Product Key that came with the Laptop is for the preinstalled OEM License and is non-transferable.

 

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

 

Only Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. Microsoft licenses it in good faith that customers won't do so, but I guess human nature wins out.
