Search Command Prompt in Start menu and click-right it. (START--Programs--Accesories--cmd)

On the menu that appears, chose Run as Administrator.

On the black windows that appear, write :   slmgr -rearm   and push Enter.

Wait some moments and you`ll see the black screen dissapear and appear a box message that anounce you that your windows is activating for 30 days.

You`ll have to repeat this steps only in the final day of those 30, but before the hour that you make the first activation ! (you must be stil in the activated period to run again that commands).

In this way you can have your windows full option, for 4 month = 120days (first month - evaluating period + other 3 activations !).

After this period, you must change something in registry (to change a value from 0 to 1) and then you`ll have an other 8 month acivation, with the same method.

You can see the days remaining to the activation when you right-click on Computer, from the desktop, choose Properties and on the down page, you`ll see how many days you`ll stil have to the next opperation !

Now, to continue using your Windows (7 or 8 or 8.1) to the 360days, do this:

a) in the search box tape ,,regedit", then go to the key:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\SoftwareProtectionPlatform

b) in the right panel change ,,SkipRearm" value from 0 to 1

c) now you`ll be able to use the command : ,,slmgr -rearm" for an other 8 times.

With this method, you may use your Windows for one year, without buying a licence !!!

So, you`ll have: - 120 days before editing registry ;

                         - 240 days after editing registry

No old copy of Windows XP that could be installed as the host system? If you don't have one, it should be available at a modest price from some online seller. Possibly (I'm not too sure), even an old Windows 98 SE would work. The game (using a 3D engine) won't work under pure DOS, so you need a Windows environment to get it to run. Just FYI: If you get this old classic to run, new issues may arise. Known ones are the "mss32.dll" runtime error (you may have to replace this file with the newer version from SW:KOTOR 1) and the video cutscenes issue (the games uses an outdated video codec, Smacker - either you find an old *.DLL for this or you have to convert the movies to a modern file format).

In conclusion: Getting old games to run on modern software/hardware (it's not only an OS problem) is one of the most fascinating and rewarding challenges for a real fan. But don't expect it will be easy "out of the box" (or simply inserting a CD).
