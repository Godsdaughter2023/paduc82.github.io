You could create a scheduled task using this trigger. It will fire whenever you activate or deactivate power:

Make it invoke this script each time it runs:

'--------------------------------------------
'Generate a pop-up showing the battery status
'19.11.2017 FNL
'--------------------------------------------
Set oWMIService = GetObject("winmgmts:\\.\root\CIMV2") 
Set cItems = oWMIService.ExecQuery("SELECT * FROM Win32_Battery",,48) 
For Each oItem in cItems
    if oItem.BatteryStatus = 1 _
    then MsgBox "Mains power is off", 64, "Battery Monitor" _
    else MsgBox "Mains power is on", 64, "Battery Monitor"
Next

Your task would now consist of finding the switches for powercfg.exe that are appropriate for the two conditions. You can then replace the MsgBox function with a shell call to powercfg.exe.
