The function of a DNS server is to translate human-friendly web addresses such as google.com into IP addresses that can be used by the routers on the Internet to get you to where you want to go.  You don't need the DNS server if you ask for the site by using its IP address. 

Try this:

  1. The next time you get the "DNS server is not responding" message, type the following into your browser's address window and press Enter:
         134.170.188.221
  2. What happens?  You should go to Microsoft's home page.

Here's something else to try:

  1. At a time when everything is working properly, open a Command Prompt window (Click Start, type cmd.exe into the Search box, and press Enter). Then type
         ipconfig
    into the black Command Prompt window and press Enter.  Make a note of the IP address for your "Default Gateway."
  2. The next time you lose connectivity, type the IP address of the Default Gateway into the address box of your browser and press Enter.
  3. What happens?  You should get a log on screen for your router's web configuration utility.   You can find details in your router's User Guide or look here --> http://www.routerpasswords.com/
  4. If you can log in to the router's configuration utility, go to the "Status" page and see if the router is connected or not connected to the Internet.  For example, this is from an old Linksys WRT54G wireless router connected to a DSL modem using the PPPoE protocol:

  • If you don't get to the log on screen for your router, then the problem is with your wi-fi connection. 
  • If you do connect to the router but the router isn't connected to the Internet, then the problem is with your router or your ISP (which includes the cable/telephone line).
  • If you connect to the router and the router is connected to the Internet but you still get the "DNS server is not responding" error message, then the first thing to do would be to configure a different DNS server.

Method 3

Press Windows key and X at the same time.

  1. Choose Command Prompt (Admin) from the context menu.
  2. In the command prompt, type powercfg –restoredefaultschemes and hit Enter.
  3. Exit command prompt.

Note: This would reset the power plan settings to default. Any customized power plans would be removed.
