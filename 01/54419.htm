hairyfool say:

In your second paragraph you state that you hold F8 to select which disk to use. At this stage it is not Windows you are instructing but the BIOS, Windows has not loaded at that point. F8 no longer works for Windows 10 anyway but on some motherboards F8 invokes the one time boot menu where you can override the default boot drive and select an alternative, your description reads as that is what you are doing. This selection is then held until the next cold start. If you tend to always go into standby then this would only occur when Windows calls for a cold start such as when an update needs a reboot to be applied. If you are changing from the default to get the SSD to boot then it cannot be the default.

Make sure you fully shut down then start up but press the Del key to get into the BIOS settings. The next stage can be a bit varied according to the actual BIOS version but you need to make sure the SSD is set as the default boot device. The boot sequence may list all the drives and if so you can position the SSD 1st. Alternatively if it just says "hard disk" then another menu will list the drives and there you set the priority.

Windows or a Windows update cannot ever change the boot order for a couple of technical reasons. The only other possibility is your CMOS battery is getting flat and some of the settings are lost following a full shutdown.
