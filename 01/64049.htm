james vinyard say:

Best way to do that is using OneDrive.  On the Mac, open the App Store and search for "OneDrive" and install the app.  Log in to it with your Microsoft Account and then, on the Surface, copy all of your data to the OneDrive folder and wait for it to 'sync'.

Wait until all the files are synced and then on the Mac, copy the files from the OneDrive folder onto the Mac itself and then empty the OneDrive folder.

You cannot use a hard disk in a normal fashion but the Mac will "read" from a PC hard disk which may be good enough.  In that case, you connect an external hard disk to the Surface (make sure it has external power since the USB port is under powered) and then go to "My PC" from the File Manager and go to the "users" folder and there, go to your account folder and copy all folders from there onto that hard disk.

[Solved:temporary solution]Huawei e8372 Wingle not working on Windows 10

I have just installed Windows 10 on my system(full clean install, not upgrade), previous version was Windows 7, and now my Huawei e8372 Wingle won't work. It shows up as a greyed-out USB. Does anyone know how to fix this.
fingers7606 say:

Finally I've managed to solve my problem(temporary solution). This is what I did:

1. Connect device to your system(PC/Laptop) and install required drivers and/or software. 

2. If the network icon, on the task bar, has a yellow triangle then it mean windows can see your device, but failing to obtain the IP address of your device. (If the network icon has a red cross then Windows cannot see your device, probably due to drivers.)  

3. Right click on the network icon on the task bar(which should have a yellow triangle)  and select "Open Network & Internet Settings"

4. In next window (Network & Internet) select "Change Adapter Options" 

5. In next window(Network Connections) right-click the connection that is assigned to your device(in my case it was "Local Area Connection 2") and select "Properties". 

6. In Properties windows double-click "Internet Protocol Version 4(TCP/IPv4)" (or you can highlight it and click "Properties"). 

7. Now select "Use the following IP address" and insert the following:

*IP address: 192.168.8.100

*Subnet mask: 255.255.255.0

*Default gateway: 192.168.8.1

*Prefered DSN server: 192.168.8.1

8. Click OK, click CLOSE. 

9. Now your Network Status should say that you're Connected. 

10. If your web management tool doesn't open automatically you can open your Internet Browser(IE, Chrome, Microsoft Edge, etc) and enter the following in the URL space: 192.168.8.1 this will run the web management tool. 

Note that when you disconnect your device/modem from your system, the next time you connect the device you'll have to enter those IP Address details again by following the steps above.

Also note that the solution above has been tested only on the Huawei e8372h-927 LTE/4G Wingle and might not work on other modems, Mifi's, Wingles, Dongles, but should generally work for Huawei Wingles, Mifi's and donlges.

Hope this will help others experiencing the same problem.
