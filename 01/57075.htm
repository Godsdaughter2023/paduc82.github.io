Limited User can't manage desktop shortcut - Microsoft Community

coldinertia say:

Limited User can't manage desktop shortcut

Windows 7 64 bit Home Premium

Why do limited user accounts need admin priviledges to delete their own desktop shortcuts? How can I enable them to do so?

ronnie vernon say:

Windows 7 64 bit Home Premium

Why do limited user accounts need admin priviledges to delete their own desktop shortcuts? How can I enable them to do so?

Hi coldinertia

The items that appear on the Desktop can be stored in  2 separate locations.

1. The items in the  C:\Users\your name\Desktop  folder only appear on that users Desktop.

2. The items in the  C:\Users\Public\Desktop  folder will appear on all users Desktop.

A standard user can remove a shortcut from their Desktop if it is stored in their C:\Users\your name\Desktop folder.

However, if the shortcut is stored in the C:\Users\Public\Desktop folder, a standard user cannot delete the shortcut because that action affects all users.

Regards,

Ronnie Vernon MVP – Windows Desktop Experience
