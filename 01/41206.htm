error code 257

I am getting this error code when trying to install Windows XP-Pro. System boots to the cd, let me delete the partitions and recreate them and then this code comes up and will not let me install. Any help would be appreciated!

malkeleah say: Failure to install an operating system is usually caused by (in order of probability):

1. Installation media that is the wrong disc for your computer. What version of Windows is the computer licensed for and what type of installation CD/DVD are you using?

2. You are attempting to install with other devices connected to the computer. Disconnect them before trying to do an installation.

3. Faulty installation media. Check the installation CD/DVD for scratches, smudges. Test in another machine.

4. Faulty optical drive. Test with another bootable CD/DVD or swap it out for a known-working one.

5. Faulty RAM. Test with Memtest86+ from www.memtest.org . You will download the .iso to make a bootable cd. Burn the file with third-party burning software* - as an image, not as data. Then boot with the media you made (you may need to change the boot priority in your BIOS so that the optical drive is the first boot device). The test will run immediately. Let the test run for an hour or two - unless errors are seen immediately. If you get any errors, replace the RAM.

*It is not necessary to have third-party burning software in Windows 7 since Win7 can natively burn an .iso. Other operating systems require it. A good free choice is ImgBurn from www.imgburn.com .

6. Faulty hard drive. Test with a diagnostic utility downloaded from the drive mftr.'s website. You will create a bootable CD with the file you download. Burn the file with third-party burning software (unless you have Windows 7, which can burn .isos natively) - as an image, not as data. Boot with the CD you made and do a thorough test of the drive. If it fails any physical tests, replace it.

7. Other hardware components are faulty.

    donna men say:

    There are 2 methods you can perform to make local drive your default location. For using Group Policy, simply follow these steps:

    1. Simultaneously click Windows + R.
    2. Type "gpedit.msc" in Run dialog box, then hit Enter or click OK.
    3. On the left pane of the Local Group Policy Editor window, navigate to Computer Configuration > Administrative Templates > Windows Component > OneDrive.
    4. On the right pane of the same windows, double-click on Save documents and pictures to the local PC by default under Settings.
    5. New window will pop up, select Enabled, then click Apply followed by OK.
    6. Restart PC to apply changes effectively.

    If your PC doesn't have the Group Policy Editor, you may follow these steps for using Registry to change default file location:

    1. Simultaneously click Windows + R.
    2. Type "regedit" in Run dialog box, then hit Enter or click OK.
    3. On the left pane of the Registry Editor window, navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Onedrive.
    4. On the right pane of the same windows, right-click on black space to create New DWORD value.
    5. Name new DWORD as DisableLibrariesDefaultToSkyDrive.
    6. Double-click on the new value (DisableLibrariesDefaultToSkyDrive), then put "1" as Value data.
    7. Click OK, then restart PC to apply changes.

    If there's anything else, feel free to reach out to us.
