xl3ackfire say:

Solution has been found! If you look back to my original post, you'll see that one of the possible causes was the following:

"I went to Run --> service.msc --> Windows Audio --> Properties --> Log On --> checked Local System Account"

After I selected this option, I could not re-select the original option, which is "This account" (followed by an account name, password, and password confirmation). When I attempted to re-select it by entering the original account name, "LOCAL SERVICE", I kept getting a "Please enter a valid password" error message--despite the fact that the password field was never reset throughout this whole process (same account name, same password--why wouldn't it work, right?)

Turns out that the original account name, "LOCAL SERVICE" is not what needs to be re-entered. The real account name plus the entire process to get it in place is given below:

  1. In your Windows search bar, type Run
  2. In the Run program, type services.msc
  3. You'll see a long list of services. Go all the way down to Windows Audio
  4. Right-click Windows Audio and select Properties
  5. Go to the Log On tab
  6. Select the This account option
  7. In the FIRST field, type NT AUTHORITY\LocalService
  8. EMPTY the Password and Confirm password fields
  9. Click "OK"
  10. There will be some notification windows. Click "OK" or "Yes" to all of them
  11. You should now be back to the Services menu (see step 3 abo

    (Note that steps 12, 13, and 14 may be shortened to "This time, select Restart". However, I did not do this route as the instructions that I was following told me to Stop and Start it as I described.)

    This is what I did and it allowed my sound to work properly again. I really do hope that others can learn from my mistake: DO NOT CHANGE THE "Log On" OPTION IN THE "Windows Audio" SERVICE UNLESS YOU KNOW WHAT YOU'RE DOING (which does not include me!)

    Windows 10 calculator - Microsoft Community

    davidskinnerpe say:

    Windows 10 calculator

    How to use the deg and dms buttons on the Windows 10 calculator.

    To display the deg and dms buttons, put the calculator into Scientific mode, and click the up arrow (↑) button.

    The format for entering and displaying coordinates using degrees, minutes and seconds is DD.MMSSssss where:

    DD = number of degrees        MM = number of minutes     SSsss = number of seconds (optional, may be omitted)
    If the number of minutes is a single digit, the single digit must be preceded by a zero.
    SS is the integer portion of the number of seconds. If the integer portion is a single digit, it must be preceded by a zero.

    ssss is the decimal portion of the number of seconds, if any.

    Once you have entered the number of degrees, minutes and seconds using this format; you can press the deg button to convert the number to decimal degrees.

    If you want to add or subtract a degrees, minutes and seconds number to or from another degrees, minutes and seconds number; enter the first number, press the deg button to convert it to decimal degrees, press the + or – button, enter another degrees, minutes and seconds number, press the deg button to convert the second number to decimal degrees, press the = button, and then press the dms button to convert the answer back to a number that is displayed in the degrees, minutes and seconds format.        (The keyboard shortcut for the dms button is M.)
