Fortunately these errors you are receiving are related to a known bug in Windows 10.

The defaultuser0 is in fact a ghost user account that gets created on some computers running Windows 10, and the messages you are getting makes sense as it is indicating the system cannot find the account.

If you remove this ghost user, I'd suggest doing the backup again afterwards to make certain you have a working backup.
