canadian tech say: Windows Update not working

This is a very common problem lately.  The most important tool you need right now is a lot of patience.  Windows Update resources at Microsoft are prioritizing Windows 7 updates at a very low service level and have been since early Summer 2015.  It is not uncommon for Windows Update to take as long as an over-night.

In many cases, the following rather simple and quick process works.

http://answers.microsoft.com/en-us/windows/forum/windows8_1-update/windows-update-problem-error-code-80080008/de8114b5-1487-4b31-9b28-f11974718df1?auth=1  This thread essentially lists a series of commands that I have found usually fixes WU.

Start, All Programs, Accessories, Right-click on Command prompt, Choose Run as administrator, OK.  Type the following in the black box:

  1. net stop wuauserv
  2. net stop cryptSvc
  3. net stop bits
  4. net stop msiserver
  5. ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
  6. ren C:\Windows\System32\catroot2 catroot2.old
  7. net start wuauserv
  8. net start cryptSvc
  9. net start bits
  10. net start msiserver
  11. pause
  12. Exit

Type services in the text box above the start globe.  Find each of the following and ensure they are set to automatic: Background Intelligent Transfer services, Cryptographic, Windows Installer, Windows Update.  Double click on each item and choose automatic.

If you still have a problem, try this.  It will take a long time, but in almost all cases gets Windows Update back on track.

  1. Download and run the correct version for your system of the System Update Readiness tool KB947821.  a net book last night.
  2. Download KB3102810 directly from the Microsoft web site and install it.  https://support.microsoft.com/en-us/kb/3102810. This took 4 hours on a net book last night.

Now re-start Windows Update.  Do NOT restart Windows.   Be very, very patient, it will likely take a long time.

4 Open the Print Server properties, Remove the printer’s drivers for this printer and for any other printers you don't use presently.

5) If you get a message about access denied or the printer being in use and you cannot find the driver or any installed program that would account for this, you may restart the computer in a diagnostic startup and then use the printui command to remove the driver package(s): https://www.isunshare.com/windows-10/enable-dia...

6) Now type Control Panel in Start Search, open CP, choose Icon View. In Control Panel open Devices and Printers, right click the printer to delete the Printer and any scanner or fax of the same name. Restart PC.

What is your default printer? To check, go to

Settings > Devices > Printers & scanners

and in the 'Printers & scanners' section you can see which printer is the default.

If that isn't the printer you want to use, click once on the printer you do want to use, then click on the gray box that says Manage. That will open a new page where you can select that printer to be your default printer.

Explanation: When a physical printer hasn't been chosen, Windows will default to its built-in fax driver, or a PDF driver, or 'Send to OneNote.' Since neither of these are physical printers, you are asked if you want to "save printer output as."
