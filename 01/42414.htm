roohi_taj say:

Hi FedTKY,

Thank you for writing to Microsoft Community Forums.

During the installation of Windows, you will be provided with an option to select the system language and keyboard layout. Once the installation is complete, this will be your default system language and keyboard layout. We cannot completely remove the keyboard layout from the system, but we have few steps that will address this concern.

I would suggest you to follow these steps:

  1. Click on the Settings icon from the start menu.

  2. Click on Time and language > Select Language from the left pane.

  3. Move English to top, to make it as your Default language.

  4. Now, click on English (United States) language > Click on Options.

  5. Select Add a Keyboard. Select Korean/Japanese as you require. In the same page, click on English keyboard layout and click on Remove. This will remove the English typing language but display language will still be in English.

  6. Now, restart the computer to reflect the changes.

Note: If you have issues with Sign in after these changes, I would suggest you to change the keyboard layout in the bottom right corner of your screen as English and then sign in. As it is difficult to change the layout each time you sign in, I would suggest you to keep a PIN sign in.

Advanced Language settings has been removed in the 1803 build from the Control panel. However, all these options are already added under language option in Settings page.

Hope it helps.
