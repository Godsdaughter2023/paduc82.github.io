Hello Richard,

 

Thank you for contacting Microsoft Community.

 

I understand your requirement and appreciate your effort in fixing the issue.

 

Are you on Microsoft account or Local account?

 

Generally if you see a User Account Control prompt while installing or accessing an application, then it indicates that the User Profile doesn't have administrator-level permissions. In simple the User Profile will be a standard account and require administrator-level permissions while making changes on your PC.

 

You can fix the issue by migrating the Standard User Profile into to the Administrator profile. Suggest you to follow the below mentioned steps and check if they help:

 

* Press Windows Key + R, type netplwiz and hit Enter key.

* If the User Account Control dialog box appears, confirm that the action it displays is what you want, and then click Continue.

* Click and highlight the User profile, which you want to make administrator

* Click on Properties, then select the Group Membership tab

* Select the Administrator, Click Apply/OK

 

This action will convert the User Profile to Administrator.

 

Keep us posted if you require further assistance.

    • The PC restarts into the newly installed copy of Windows.

    If you are looking at the Properties of the Appdata folder, this does not give you an accurate determination of the read only attribute for a folder.

    Unlike the Read-only attribute for a file, the Read-only attribute for a folder is typically ignored by Windows, Windows components and accessories, and other programs. For example, you can delete, rename, and change a folder with the Read-only attribute by using Windows Explorer.

    To determine the Attributes on that folder, perform the following.

    Right Click the Windows Button on the bottom/left corner of the screen to display the System Menu.

    Click the Command Prompt (Admin).

    At C:\Windows\System32>  type  cd\  and press Enter.

    At C:\> type cd Users\yourname  and press Enter.

    (Where 'yourname' is the name of your user account)

    At C:\Users\your name> type  attrib appdata  and press Enter.
