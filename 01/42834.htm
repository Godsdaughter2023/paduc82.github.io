des p d say:

Hello NandanRaj,

The question was asked a while ago, but I thought I would post the answer for others who have the same issue, the app in question Microsoft Paint 3D a Microsoft Sore App.

It seems that it looks for updates constantly, and, since it uses the GPU, it show's up there. The only solution other than uninstalling, which I don't recommend, as it will reinstall on next upgrade, the only solution is to disable automatic updates in the Microsoft App Store.

(This step may not be necessary if you are just rescheduling the updates, go to solution 2.)

Solution 1:

1. Open Microsoft Store App.

2. Right-click the ... beside your signed in logo, top right.

3. Click settings and toggle to off the Automatic updates.

I know, I thought the same thing, disabling automatic updates is a security risk. The solution is to set automatic updates on a schedule, through Computer Management Task Scheduler, Windows Update.

Solution 2:

1. Start Menu ->Windows Administrative Tools.

2. Right click Computer Management, More -> run as Administrator (Type Password!)

3. In Computer Management, Open Menu Task Scheduler -> Task Scheduler Library -> Microsoft -> Windows.

4. Click Windows Update, in the open panel on the double-click Automatic App Update.

5. A separate window opens, click the second Tab, Triggers.

6. Double click one time.

7. Another window opens, from here you can re-schedule any updates to any time that suits your systems usage. (Here I would advise to update as often as possible, this makes your system more secure).

This is what is causing the issue, the updates trigger every 4 hours on my system. I set it to update every week, on a suitable day. That fixes any issues with Paint 3D, and Windows App Store consuming system resources when not needed.

OK, so both of these solution fix the issue, I would advise anyone reading this, not to use both solutions, as you will both switch off updates and reschedule them, I used solution 2.

I hope this helps.

Mark as solution, if it fixes your issue.

Have a nice day!
