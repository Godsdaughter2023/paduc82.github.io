System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

Follow the below steps:

  1. Right click on the Start button, click on Command Prompt (Admin).
  2. In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

    • DISM.exe /Online /Cleanup-image /Restorehealth

Sometimes Microsoft gives you Updates for 3rd Party Hardware: Graphics, Network Adaptors, etc.

You should only obtain the latest Drivers for them from the actual Hardware Manufacturer's website, as Microsoft normally does nor have those latest Drivers available as Microsoft relies of the 3rd Party Hardware Manufactuer to supply them.

But unfortunately, the only way to find out which is the problem update is to install them one at a time, until you find the "bad" one for your setup.

You can do a System Restore to go back to before it was installed (which is NOT the best way to remove an Update) or follow this method:

"Remove an Update - Windows 7"

http://windows.microsoft.com/en-us/windows/remove-update#1TC=windows-7

So that this does not happen again:

When you have found the problem Update from the list of updates that is presented > you then right click on the problem update > select the Hide Update Option > and it will not be offered to you again

"How to Hide or Restore Hidden Windows Updates in Windows 7 and Windows 8"

Download updates but let me choose whether to install them - If you select this option, Windows will download updates to your computer, but not install them automatically. Check for updates but let me choose whether to download and install them - If you select this option, you will be alerted when there are new updates available to download and install. Since we have limited access here at the Skype Community forum to investigate further, and you have done the necessary troubleshooting steps, I strongly recommend you to contact our chat support team in order to make an investigation and collate necessary information from you. 

 

To get in touch with our live chat support, please follow the instructions below:

 

  1. Go to this link: https://support.skype.com/support_selection

  2. Sign in to your account.

  3. Choose Payments & Orders > Help to cancel or get a refund

  4. Click Next.

  5. Click Text chat support

Please make sure your browser allows pop-ups, and you have no plug-ins enabled that could prevent the Live Chat window from opening. If you get a warning about a separate browser pop-up window, you will need to allow it; that's the live chat window.

 

If you're having issue reaching chat support, please check below tips:

 

  • Try again after clearing the browser’s cache and cookies.

  • Try using in-private browsing mode.

  • Try using a different web browser.

  • Try using a different device.

  • Try using a different Internet connection.
