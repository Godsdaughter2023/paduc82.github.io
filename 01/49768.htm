¡firedog say:

A saved Word/Excel document sometimes has a blank box alongside it instead of one showing Word or Excel  ... What am I doing wrong.

Probably nothing at all. The 'blank box' indicates, I think, that Windows doesn't recognize the file type. This can happen if you save a document with the filename that the program suggests. This will be the very first bit of text in the document; if this is a document title or heading that ends with a dot (e.g. Annual report 2015. ), then you'll end up with a filename with two consecutive dots in it, like  Annual report 2015..docx.

Because this can cause problems in certain situations, programs like Windows Live Mail will 'correct' the fault by changing one of the dots to something else, like an underline character. Unfortunately, Windows Live Mail picks the wrong one to change, so instead of transmitting a file with filename Annual report 2015_.docx, it transmits one with filename Annual report 2015._docx. Of course, the extension ._docx isn't recognized, so Windows shows it as an 'unknown filetype' as shown in the picture.

The top one has an unknown filetype, while the bottom one has the extension .xlsx indicating that it's an Excel file.

You can avoid this by always checking the filename to ensure that it doesn't have any extra dots in it.

It's a good idea, too, to make sure that you can always see file extensions. To do this,

  1. Press Windows key  + R, type control folders and press Enter.
    This should open the Folder Options dialogue in Control Panel.
  2. On the View tab, deselect Hide extensions for known file types.
  3. Close the dialogue.
