The download speed testing feature connects to a server and it may not be showing your full download speed because of the bandwidth of the server and how much it can upload to you to receive. There's a lot of factors that can impact your download speed on your console from wet wires on the power line poles to not having your wireless network secure and having someone downloading or streaming things that slow you down.

frederik long say:

You need to use an imaging tool such as AOMEI (free) or Acronis True Image:

  1. Use it to create an image of your system partition
  2. Store the image on an external disk.
  3. Create a bootable rescue flash disk.
  4. Replace the disk.
  5. Boot the machine with the flash disk.
  6. Restore the image.

Please remember to mark this reply as "Answer" to your question.

patrick barker say: Hi,

All of the attached DMP files are of the REGISTRY_ERROR (51) bug check.

This indicates that a severe registry error has occurred.

This error may indicate that the registry encountered an I/O error while trying to read one of its files. This can be caused by hardware problems or file system corruption.

Usually when this bug check occurs, it's due to a large amount of registry and/or file system corruption and the best course of action is a repair or clean install of Windows as the registry is too far corrupted to be repaired, etc.

There is one thing we can try before that though, so let's go through that:

Remove and replace avast! with Microsoft Security Essentials for temporary troubleshooting purposes:

avast! removal tool - http://www.avast.com/uninstall-utility

MSE -  http://windows.microsoft.com/en-us/windows/security-essentials-download

The reason I recommend and mention avast! removal is the only other time a *51 bug check occurs is during a refresh operation failure, which is only used by the security system. Possibly there could be conflicts with avast!. If after removal however you still crash, your best course of action as I said earlier to save all headaches, etc, is repair or clean install.

--------------------------------------------------------------------------------------

Repair install - http://www.sevenforums.com/tutorials/3413-repair-install.html

Clean install - http://www.sevenforums.com/tutorials/1649-clean-install-windows-7-a.html

Repair if you'd like to keep all of your files, clean if you at this time don't care and just want a brand new, fresh, working Windows installation that is GUARANTEED to work.
