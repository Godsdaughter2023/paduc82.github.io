ben herila say: Look in Windows Update and write down the KB numbers of all the updates you need to install. Then go to the Microsoft Update Catalog (http://catalog.update.microsoft.com) and add all those updates to your "cart." When you have added all the necessary updates to your cart, you can download them to a folder on your PC. After the updates are downloaded, double click on each MSU file to install the update. 

For your convenience, you can also open a PowerShell window (note: be sure to run as Administrator) and run the following command once you are in the folder that contains the update: 

ls *.msu -recurse | %{start -wait $_ -argumentlist '/quiet /norestart'}
