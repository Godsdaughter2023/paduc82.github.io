- There is hope. 16bit works on 32bit Windows. If you have access to a machine running 32bit Windows

'98 and older starts getting a bit more hairy.  Since using Win10 I have had at least two games from '02 and earlier,

that ran on 64bit Win7 without using any compatibility setting that now require me to use Win 98 Compatibility.

The reason is these games were developed for single core cpu's , and Win 95/98 options force the game

to run on a single cpu core.

I assume there is something in the way Win10 handles cpu use (probably trying to optimise everything for

quad core ) that it has introduced yet another problem  for gamers ( One more negative vote for  MS's  claim

that Win10 is the 'best gaming Windows ever') .

The usual (but not only) reason that the print spooler stops is that there is either a corrupt print job stuck in the print queue or a corrupted printer driver.

If after completing step A the print spooler still does not stay running after you have issued the "net start spooler" command, you will not be able to follow all of the steps in "Article One."  Instead, go to the other link and download and use "cleanspl" as described there.

A.  Clean out the pending print jobs

  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Type the following in the black Command Prompt window and press Enter after each line (don't forget the double quote marks)

                            net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

B.  Clean out the old printer drivers and install the latest drivers using the directions in Article One.  If you have (or had) a Lexmark printer, follow the directions at this site before installing the new drivers as explained in Article One:  http://members.shaw.ca/bsanders/CleanPrinterDrivers.htm
