If you are not running the 64-bit version of Windows, the extra RAM
will not be used. In fact, all of the 4GB you can use and are using is
something around 3.1GB.

Would more RAM help you in graphics applications? Possibly, but it's
hard to be sure.

Will performance improve with 64-bit Windows? Two points:

1. It will let you use more RAM, and if your applications can use more
RAM, then it will help.

2. Otherwise it will make little difference unless you run 64-bit
applications. There are very few 64-bit applications now, but there
will be more and more in the future.

Hi,

It could be that the Deployment Types for the application set to required. If that is the case, uninstall option will not be available as it essentially violates the rule of being required. The best way to uninstall the program is using Registry. Kindly follow the steps below:

  1. Click the Windows key + R.
  2. Once the Run box open, start by typing regedit and press Enter.
  3. Locate HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\currentversion\Uninstall.
  4. Select the program which you want to Uninstall in the left pane.
  5. In the right pane, select that program’s uninstallstring and double click it.
  6. Copy the string in the value data, paste it in the Run command and press Enter to initialize uninstall of the program.

A lot of different PC's will have different services running in the background, so there is no one script that will work for all PC's

Turn off all unnecessary startup applications and processes:

Open the Settings App
Go to Privacy - Background Apps
Turn off any App you do not need running in the background
