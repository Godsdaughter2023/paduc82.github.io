There is no product key to get.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

wmic path SoftwareLicensingService get OA3xOriginalProductKey

Hit Enter

Edge doesn't store its favorites as files like Internet Explorer does.  Instead, they are stored in a database.  So, that means that you really can't use the File Explorer anymore to view, print, or copy your favorites.

But, there still is a way to perform a backup...  There are two approaches... a simple one that only works if your using the Edge "Sync your content" feature, and a more turn-key version where you install a 3rd-party application

1) If you are using Sync your content, you can copy the contents of the %USERPROFILE%\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\RoamingState for safe keeping.  And then just copy it back after a crash/reset

2) You can use the free 3rd-party EdgeManage application to perform a backup/restore (plus many more features).  This technique works without regard to the "Sync your content" settings.

However, there currently isn't a good way to print favorites...   Yes, it can be done... but it is quite complicated.  It would require EdgeManage, switching views, copy-n-pasting into an Excel spreadsheet and the removing a lot of useless info, and then printing.
