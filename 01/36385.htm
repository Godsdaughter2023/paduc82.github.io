Forgive me, this process might be complex. You will have to create a new registry entry and then merge into the registry.

Note 1: You will have to have "Display file extensions to known file types" set to enabled in "Folder Options" found in "Control Panels" for this to work.

Note 3: If the Windows Key used to activate your installation was issued from an organization's Key service this might explain why your modifications are rejected. However, based on what you stated I will assume that is not the case.

Disable Windows Defender via registry addition
Steps:
1. Open the "Start Menu"
2. Scroll to "Windows Accessories"
3. Run "Notepad" (you should now be presented with a new blank note).
4. Copy and paste the code below, all lines of it, into the brand new blank note, please don't change spacing or characters and paste as is. 
[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender]
"DisableAntiSpyware"=dword:00000001

5. Save the note with the name "Windows_Defender_Off"
6. Be sure to save it as a text file (*.txt) in a place on your computer where you can find it.
7. Now please navigate to the file you just created and change its file type from text to a registry entry by changing the three letters after the period from .txt to .reg
8. Accept the file change warning if one pops up.
9. If the file change was successful you will notice the icon actually changed.
10. Now Right-click this newly modified file and select "Merge" , and answer yes/accept any warnings that might pop up, then restart your computer and check to see if the changes you desired have taken place.

If this solution does not work please let me know.
A consequence of this solution is the registry will have to be manually edited to enable the service.

taramcmullin say: The following fix worked:

Reboot the PC

In Services, find the Windows Media Player Network Sharing Service

Go to Start
Type %LOCALAPPDATA%/Microsoft and press Enter
This folder will open up in Windows Explorer
In this folder, rename the MEDIA PLAYER folder to MEDIA PLAYER OLD
Start WMP, your library will have been reset
