  • Try using a different USB port on your PC.
  • If you use a USB hub between the device and the PC, make sure the hub has power. If it’s still not working, try removing the USB hub and plugging the device directly into a USB port on the PC.
  • Make sure that the cables on your device aren't damaged in any way.
  • Check any battery-powered devices to see if they are fully charged or have fresh batteries.
  • Make sure there’s no dirt blocking keys or sensors.

Hello Rapture00,

The error code you're getting is an indicator when the system is trying to update the driver through Windows Update and is trying to install a driver that is already installed on your computer; which is normally a driver that has a lower compatibility rating than one you already have installed.

To get rid of the notification from Windows Update, you may manually update the driver from the manufacturer's site.

Note: Make sure you're logged on as an administrator to perform these steps.

1. Open Device Manager by clicking the Start button >  Control Panel > System and Security and then, under System, click on Device Manager. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

2. In the list of hardware categories, find the device that you want to update, and then double-click the device name.

3. Click the Driver tab, click Update Driver, and then follow the instructions. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

Now, if you don't want to perform the manual process of updating the driver, you may just configure and hide the notification through the Control Panel:

Turn Off Security Messages and Other System Notifications in Windows 7

Note: Confirm if this is the same process you've completed to hide the notification.
