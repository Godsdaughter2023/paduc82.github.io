jennifer bri say:

Hi Alec,

A recovery drive can help you troubleshoot and resolve issues with your PC, even if it won’t start. To create one, all you need is a USB drive.

  1. From the taskbar, search for Create a recovery drive and then select it. You might be asked to enter an admin password or confirm your choice.
  2. When the tool opens, make sure Back up system files to the recovery drive is selected and then select Next.
  3. Connect a USB drive to your PC, select it, and then select Next > Create. A lot of files need to be copied to the recovery drive, so this might take a while.
  4. When it’s done, you might see a Delete the recovery partition from your PC link on the final screen. If you want to free up drive space on your PC, select the link and then select Delete. If not, select Finish.

Let us know if the steps above worked for you.

locutus deborg say:

"shrink" does not create new partitions
it creates unallocated space to the right of the partition that was shrunk

in an Administrator : Command Prompt

chkdsk /f /v C:

it will prompt to schedule the chkdsk at next boot
type Y

exit the command prompt
then reboot

after chkdsk is completed and you login,
go to disk management and look at it again
if there is no "black" unallocated space, then the previous shrink didn't actually happen
- attempt to shrink the partition again

if there is a "black" unallocated space
- right click it and create a new partition
