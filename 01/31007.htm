Don't let several programs start up with Windows if you don't need them right away.

Many programs add a key in the registry automatically during install to let it start up with Windows.

So once Windows has loaded, these programs stay running in the background without you really needing them.

You can always start these programs manually via Start > All Programs when you need them. However, it is still important that your Antivirus and Firewall stays enabled and starts up with Windows.

You can always enable them afterwards again by selecting them.

"Intel (R) USB 3.0 eXtensible Host Controller -0100 (Microsoft)"

 

The Microsoft Driver is Dysfunctional and there are no other Drivers available!
