OK, you have now established that you wish to search the folder "E:\2016 System". To get these file names into an Excel list you can do this:

Installation

  1. Mark this line with your mouse, then press Ctrl+C to copy it:
    cmd.exe  /c  cd  2>"%UserProfile%\Desktop\FileList.bat"
  2. Press the Windows key + R.
  3. Press Ctrl+V to paste the code line, then press Enter.
  4. Mark this line with your mouse, then press Ctrl+C to copy it:
    @dir /b /s /a-d %1 > "%UserProfile%\Desktop\Files.csv"
  5. Right-click the new desktop file FileList.bat, then left-click "Edit".
  6. Press Ctrl+V to paste the code line, then save and close the file.

Operation

  1. Open File Explorer.
  2. Grab the folder E:\2016 with your mouse, then drop it on the desktop file FileList.bat.
  3. Open the desktop file Files.csv. It will contain a complete list of files found in the folder E:\2016.

Post again if you wish to refine your search, e.g. by looking for certain keywords in the file names.

Hi Robert,

If your old work account is not accessible anymore, you can have it removed on your PC and the meetings associated with it will be removed as well. You can follow the steps below to remove an account on your Windows device:

  1. Click the Search bar.
  2. Click on Cortana’s notebook.
  3. Select About Me.
  4. Highlight the email address you no longer use.
  5. Click the email again, select Sign out.
