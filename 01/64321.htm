marks1964 say:

Hi Garltok,

A back of your computer to an external hard drive is always a good idea, better 2 copies of everything than none at all.

The upgrade simply packs the OLD Windows (7 or 8.1) into a folder called Windows.old and installs the new Windows 10. You are not able to use the old version of Windows.

All of your settings; files; folders and compatible programs will be transferred to Windows 10.

You can use the rollback option feature built into Windows 10 to go back to your OLD Windows providing you use it within the first 30 days, otherwise you are locked into Windows 10 as the Windows.old folder is deleted automatically.

Hope this helps.

Mark

xander1971 say: Is there a way to reformat & start fresh instead of an upgrade?

a_user_here say:

You need to do the upgrade first to establish that the device is qualified to receive the free upgrade.

After that you can do a clean install if you like since with Windows 10 it will connect your license to your device/Microsoft online account (If you log into your Microsoft online account you can see the device registered under Devices).

To do the "clean install", you could just use the "Reset PC" option (Start -> Settings -> Update & Security -> Recovery) to clean out the programs, but leave data.

Or you can use this tool to make an bootable USB Flash or DVD to do a true clean install (not to mention having it for use later)

http://www.microsoft.com/en-us/software-download/windows10

While running the install it will ask for the product key.  You can skip it each time it asks until you get to the point where it asks for your Microsoft online account information.  Once it has logged into that it will automatically activate your copy of windows/license.

By saving the product information in this manner during the upgrade, Windows 10 doesn't require reinstalling the old OS to do the install again like you had to do in the past upgrades of Windows (provides that the device has not changed too much, if that was the case you would have to contact Microsoft support to help you).
