Start your System, then just after the Manufacturers logo disappears and Windows 10 tries to boot, press and hold the power button down for 5 - 10 seconds to perform a hard shut down

xbr dionysus say:

Bridging Guide

Login Information

  • Gateway - 192.168.1.1 or 192.168.0.1
  • Username - admin
  • Password - password

Bridging Guide

*Bridge mode should only be used if you have a router to plug into the device.*

*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.0.1 into your address bar. (Or click either link)
  3. The Gateway may ask for a password. If it does use the following. (admin/password)
  4. Select My Network from the menu at the top.
  5. Select Network Connections from the menu on the left.
  6. Click on Broadband Connection DSL.
  7. If the connection is using PPPoE, proceed to the next step.  If the connection is using DHCP, you will need to click on "Release" and then proceed to the next step.
  8. Locate the "VCs" section, find the line that is enabled and click on the edit button on the right hand side (looks like a little notepad)
  9. Change the protocol from PPPoE to Bridge.  If the connection is using DHCP, it should already be set to Bridge
  10. Once the protocol has been changed to Bridge, make sure that the "Bridge Mode" option is set to just “Bridge”
  11. Click Apply.
  12. Select My Network from the menu at the top.
  13. Select Network Connections from the menu on the left.
  14. Select LAN.
  15. Uncheck "Private LAN DHCP Server Enable".
  16. Click Apply.
  17. If necessary enter the PPPoE settings into the router.

thef055 say:

cloud storage space

How big is the cloud storage space for my Xbox Live Gold account on Xbox One?

cheez-21bae529-f0fc-40ce-a38a-c9532f5c32f5 say:

You get unlimited cloud storage for game saves, game clips, your profile, ect. Please note your OneDrive is not the same cloud storage that your Xbox One uses for game content and your profile. You OneDrive is used to save clips you've edited in Upload Studio or storing any files you wish. You can check how much storage is left in your OneDrive by going to http://onedrive.live.com

To sum it up OneDrive is used to upload game clips and store/share files and you have a limited amount of space (Unless you purchase more), and the separate Xbox One cloud storage saves your profile, game saves, ect and it has an unlimited amount of space.
