USB 3.0 Hub Operating at USB 2.0 Speeds - Microsoft Community

keith- say:

USB 3.0 Hub Operating at USB 2.0 Speeds

My USB 3.0 devices are running at USB 2.0 speeds.  When I look in Device Manager, it lists the USB hub as “SuperSpeed USB Hub (Non Functional)”. My USB 3.0 devices are running at USB 2.0 speeds.  When I look in Device Manager, it lists the USB hub as “SuperSpeed USB Hub (Non Functional)”.
michael. o say:

Windows may treat some early USB 3.0 hubs as USB 2.0 hubs.

 

If this happens, update the firmware on the hub.  You may see a notification in the Action Center, pointing you to a firmware download location.  If not, contact the manufacturer of your USB hub for an update.

 

Most likely, you will notice this when a USB 3.0 device operates at a much slower speed when connected via a hub instead of directly to the USB 3.0 port.  In Device Manager, a USB 3.0 hub will appear as two hubs, a "Generic USB Hub" and a "SuperSpeed USB Hub."  (The manufacturer may supply a new name, but these are the default names).  If the SuperSpeed hub says "(Non Functional)", you are probably hitting this issue.

 

Not all hubs are external devices – some are integrated into systems.  If this is the case, you may not realize that you are using a hub.  To tell, open Device Manager and check for a hub as described above.
