    Hi 2Stuffed,

     

    You can find the format of the DVD movie file by following the steps mentioned below.

     

    a.    Insert the disc in the DVD drive.

    b.    Once the disc gets detected, right click on your DVD Drive and click on explore.

    c.    You’ll have folders and files.

    d.    Go into the folders and check the properties of the file by right clicking on the file and clicking on properties.

    e.    Under the General tab, check the type of file which gives you the file format.

     

    If the file format matches with the list provided under “File format supported by Windows media center”, then you can follow the steps under “Add media files to Windows media center” to import the movies.
