Although the initial version of zip file specification only allowed up to 4 GB (the implementation in XP apparently was limited to 2 GB - LINK), you have a version of the algorithm that (probably*) permits up to about 16 billion gigabytes.

However, the problem you have is that video files generally are already compressed, and attempting to use zip or similar lossless compression algorithms to compress them will often result in a larger file than the original.

You can compress some video files by converting to a different format, but this generally will result in the loss of some detail.  See, e.g., http://www.wikihow.com/Compress-Video
