    how to automatic shutdown? - Microsoft Community

    osmr say:

    how to automatic shutdown?

    I'm trying to set the computer to automatically shutdown by scheduling  shutdown.exe to run daily but that's not working

    please help me and many thanks in advance

    lemp say:
    I don't really want to shutdown my system to check, but I believe that the Run box in Scheduled Tasks should show the program to be run in double quotes followed by any parameters for that program.

    For example, this is the Run box for  a scheduled task automatically generated by Google Updater:

    "C:\Program Files\Google\Update\GoogleUpdate.exe" /ua /installsource scheduler

    The problem in your case is complicated by the fact that one of the parameters you have selected for the shutdown command is the -c parameter, which itself is supposed to use double quotes to delimit the comment.  I don't know if the Scheduled Tasks command line parser will get confused by this.

    I suggest that you first try the following in the Run box:

    "C:\WINDOWS\system32\shutdown.exe" -s -t 120 -f

    If that works, then add the -c parameter as follows:

    "C:\WINDOWS\system32\shutdown.exe" -s -t 120 -f -c “This computer will shut down in 2 minutes, please save all work”

    This should work, but if not, I suggest that you make a simple *.bat file that includes the shutdown command without surrounding double quotes except for the comment:

    C:\WINDOWS\system32\shutdown.exe -s -t 120 -c “This computer will shut down in 2 minutes, please save all work” -f

    The Run box in Scheduled Tasks would then be something like

    "C:\Documents and Settings\cest moi\My Documents\MyShut.bat"
