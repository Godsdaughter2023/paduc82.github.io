As per the issue description you are unable to perform system restore on your computer and getting error code 0X8007005.


This problem occurs because certain files are not replaced correctly if a file conflict occurs during the restore process. This also occurs if there are Windows registry values that are blocking the update from being downloaded and installed or was not properly installed.

  • Are you able to boot the computer to the desktop?

If you are able to boot your computer to the desktop follow method 1.

Method 1: Check the Volume Shadow Copy service.

Check if the Volume Shadow Copy service is running on your computer or not. To check for the same follow the steps below:

a. Click ‘Start’, type “services.msc” without quotes and hit Enter.

b. Locate “Volume Shadow Copy” and check if the service is started.

c. If not ‘Start’ the service.

If the service was already started, I would suggest you to stop the service and restart the same.


Method 2

If issue persists, I would suggest you to perform System Restore from safe mode.


Refer the below link for more details on starting your computer in safe mode:

Start your computer in safe mode
http://windows.microsoft.com/en-US/windows7/Start-your-computer-in-safe-mode

To perform a system restore.

To do that follow the instructions below:

1) Click Start, type system restore in the Start Search box, and then click System Restore in the Programs list.If you are prompted for an administrator password or confirmation, type your password or click Continue.

2) In the System Restore dialog box, click Choose a different restore point, and then click Next.

3) In the list of restore points, click a restore point that was created before you began to experience the issue, and then click Next.

4) Click Finish.

For more information refer this below link.

System Restore: frequently asked questions

http://windows.microsoft.com/en-US/windows7/System-Restore-frequently-asked-questions

Important notes:

1. The system restore done from safe mode cannot be reverted back.

2. When you use System Restore to restore the computer to a previous state, programs and updates that you installed are removed.
