Hello Bryce,

Thank you for posting your query on Microsoft Community.

As per description, you are getting Error code: WerFault.exe - Application error.

The werfault.exe is used for  Windows Error Reporting. It is a feature that allows Microsoft to track and address errors relating to the operating system, Windows features, and applications. It gives you the option to send data about errors to Microsoft and to receive information about solutions.

I would suggest you to follow the methods given below and check if that helps.

Method1

I would suggest that you stop and restart the Windows error reporting service on the computer and later check if the issue persists.

a)   Press Windows Key + X and select Run and type services.msc in the Run box and press Enter.

b)   In the service list navigate to the Windows error reporting service.

c)    Right click and click on properties now stop the service and restart it.

d)   Click Apply and OK

If issue didn't get resolve try method 2,

Method 2: I would suggest that you perform clean boot procedure to check if any third party program installed on the computer is causing the issue.

  1. PressWindows key + X key. And selectRun.
  2. Type msconfig in the Run box and hit Enter.
  3. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then tap or click Disable all.
  4. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.
  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.
  6. Close Task Manager.
  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

Refer: How to perform a clean boot in Windows

http://support.microsoft.com/kb/929135

(Holds good for windows 10)
