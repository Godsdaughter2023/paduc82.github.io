Hi,

What we are going to do is give permission to share file or folder from Windows 10 on your network.

  1. Open File Explorer then right-click the folder you want to share and select Properties.
  2. Click on the Sharing tab.
  3. Click Advanced Sharing.
  4. At the top, tick the box to Share this folder then OK.

Note: By default, when sharing file or folder Windows 10 allows read-only access, which means they can only view it and cannot make any changes. If you want to allow others to make some changes click Permissions and follow the instructions below.

To give permission, right-click on the folder, then click Properties > Sharing > Advanced Sharing > Permissions.

  1. Under the Group or user names box, click Add....
  2. Enter the account name you want to give permission, then click Check Names.
  3. If the user is detected, you should see a path, then click OK to add the user's permission.
  4. Put check on the box under Allow to give the permission they need. Click OK to exit.

Let us know if you need further help.

Hi Maria,

Windows normally offers a Save as option every time you download or transfer a file from Internet or via Bluetooth.

If you did not see the Save as prompt when the transfer got completed, those files will usually reside in a temporary folder by default. Navigate to C:\Users\<profile>\AppData\Local\Temp\ and try searching for the file by sorting out the date and see if you'll be able to find them.

If you can still remember the name of those photos or files, you can utilize Windows Search by pressing Windows key + S and typing the file names. You can also search the files by using File Explorer.
