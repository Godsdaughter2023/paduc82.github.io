How do I change the labels on the Windows boot menu? - Microsoft Community

be creative - be it say:

How do I change the labels on the Windows boot menu?

I have a dual-boot Windows 7 system. One OS is English while the other is Japanese. Both are 32-bit OSes.

 

When I boot up my PC, there is a Windows boot menu (white text against a black background).

 

The menu reads something like the following:

 

Windows 7

Windows 7

 

How do I edit the registry so that I can distinguish between the two OSes, something like:

 

Windows 7 [English]

Windows 7 [Japanese]

 

I appreciate your help with this.

 

 

sathish s v say:

Hi,

 

You can Change the description of the OS on the boot loader through BCDEdit command lines.

Follow these steps:

 

a.       Boot into default Operating System.

b.      Click “Start” button then select “Accessories”.

c.       Right click on “Command Prompt” then select “Run as Administrator” option.

d.      Type “bcdedit” without quotes in the Command Prompt and hit Enter.

It will display the “Windows Boot Manager” and the “Windows Boot Loader”.

Windows Boot Loaders are the Operating System Installed on the computer.

It will be having the “Identifiers” for each Operating System.

 

Current - This identifier indicates the current Operating System in which you are working.

Default – This identifier indicates the default Operating System set on the computer to boot.

If you see alpha numeric numbers like “77da19a1-2ae7-11de-a79b-8ffdb0066c6f” then this identifier indicates that this Operating System is not set as Default and you are not working on this Operating System.

e.      Use this command bcdedit /set {current} description "Windows 7 English”.

Here {current} is the identifier.

If you have set Windows 7 English Operating System as default and you are working on it, then the identifier for this Operating System will be {current}.

 

f.        To change the description for other Operating System use this command.

bcdedit /set {77da19a1-2ae7-11de-a79b-8ffdb0066c6f} description "Windows 7 Japanese”.

Here the alpha numeric code is the identifier displayed for the Operating System on your computer.
