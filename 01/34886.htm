saltgrass say: I was never able to get a Windows 7 Repair CD to boot into the UEFI mode.  Microsoft was made aware of this situation 4+ years ago, and to the best of my knowledge has never done anything about it.

If you want bootable media, create a bootable flash drive and then do the extra step of creating an additional folder in the efi folder named Boot with a file called bootx64.efi in it.  Bootx64.efi is a renamed version of Bootmgfw.efi which is located in your Windows\Boot\EFI folder on your current Windows 7 install.  That is the only way a Windows 7 flash drive will boot to UEFI.

If you see instructions telling you to copy the contents of another boot folder, that is not necessary, just create the Boot folder in the efi folder and then copy the renamed file into it.
