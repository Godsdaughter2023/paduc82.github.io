Have you tried disabling the write-behind caching on the drive?  This way there will never be any unflushed information waiting to be written to the drive.  This write-behind is automatically disabled for all USB external drives, flash drives, and any SSD drives.  But since this is an internal connection, it probably has this caching turned on expecting it to be a permanent hard drive.
