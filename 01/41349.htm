You  may refer to the below suggestions and check.

a.       If you have the Windows 8 ISO file you can create a bootable disk or bootable flash drive.

b.      Click to open the Windows 7 USB/DVD Download tool installation file.

c.       When you are prompted to either save the file to disk or run it, choose Run.

d.      Follow the steps in the setup dialogs. You'll have the option to specify where to install the

Windows 7 USB/DVD Download Tool.


Refer to the following suggestions to use the Windows 7 USB/DVD download tool.


a.       Double click, open the Windows 7 USB/DVD download tool shortcut created on the desktop.

b.      In the SOURCE FILE box, type the name and path of your Windows 8 ISO file, or click BROWSE and select the file from the OPEN dialog box. Click NEXT.

c.       Select USB DEVICE to create a copy on a USB flash drive or select DVD disk to create a copy on a DVD disk.

d.      If you are copying the file to a USB flash drive, select your USB device in the drop-down list and click BEGIN COPYING. If you are copying the file up to a DVD, click BEGIN BURNING.

e.      When your Windows 8 ISO file is copied onto your chosen media, try to boot from it and check.


You may refer to the below steps to boot your computer from bootable device(DVD/Flash drive).

a. Procedure to enter BIOS settings  vary depending on the manufacturer. Usually, you must press a key (such as F2, F12, DEL, ESC) or a key combination immediately after you turn on your computer but before Windows starts.

b. Navigate to the Boot device priority, select CD/DVD drive/ Flash drive as the first boot device (use the arrow keys to change the device priority), press F10, select Yes to save the settings.

c. Now your computer restarts, check if you can boot from bootable device.

For more information, check the information that came with your computer or go to the computer manufacturer's website.


Hope this helps. If you have any further queries, or if this issue persists, reply here and we will be glad to help you.

  7. Click Apply and OK.
