lemp say:

When I use Ccleaner to clean my registry

Old joke:  The patient says, "Doctor, it hurts when I do this." The doctor says, "Then don't do that!"

New version:  Computer user says:  "When I use Ccleaner to clean my registry I have problems."  Tech support says, "Then don't do that!"

Seriously, you should not use registry cleaners or other so-called "optimizers" or "tune-up" programs.  The benefit from using a registry cleaner is speculative at best.  The chance for serious system damage is not negligible.

  https://support.microsoft.com/en-us/help/2563254/microsoft-support-policy-for-the-use-of-registry-cleaning-utilities

Registry entries relating to unknown dll files were probably inserted during visits to dubious websites and/or from downloading software that had bundled applications in the installer.  If the files are missing, it's likely that your antivirus/antimalware software detected and removed the files before they could do any damage.  Such software very often fails to remove registry entries, which is why one of the most common tech support issues is, "When Windows starts, I get an error message that says file xxxx.dll is missing."
