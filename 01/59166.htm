a. user say:

Each time that a printer is re-installed, it's considered a new printer. Even if it's the same printer. Sounds crazy? Not really: Windows can't tell whether you are re-installing the same printer or installing another printer that's exactly like the one you already have. So each one is treated as a separate printer.

Moreover, Windows has to give each printer a unique name in order to be able to distinguish it from other printers. In your case, Windows came up with three unique names for the same printer. In my case, Windows adds 'Copy 2' and 'Copy 3.' Perhaps it has a different naming convention for networked printers.

All you have to do is go into the Devices and Printers control panel and delete the names that are copies of the same printer. (Right-click on the icon, Delete.) Make sure that you are deleting the copies and not the printer itself.
