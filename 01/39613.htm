vijay b say:

Hi Amaan,

If you want to disable systray tool tip, I would suggest you to try the following steps.

a. Press Windows key + R,
b. In the run prompt type regedit.
c. Click on OK
d. Go to the following location.
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced

e. Right click on ShowInfoTip and select modify
f. Change the value to 0
g. Close Registry
h. Reboot computer

Note: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly.
