frederik long say: You use the same method as for all other versions of Windows:
1. Click Start, then type the three letters cmd into the Search box and press Enter.
2. Type this command and press Enter:

    ipconfig /all | more

then look for the word "physical"

Here is an alternative:
1. Click Start, then type the three letters cmd into the Search box and press Enter.
2. Type this command and press Enter:

    ipconfig /all > "%UserProfile%\Desktop\IP.txt"

You can now double-click the file IP.txt on your desktop and examine it at leisure.
