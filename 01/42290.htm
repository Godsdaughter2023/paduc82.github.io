Restore Factory Defaults - Microsoft Community

xbr dionysus say:

Restore Factory Defaults

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin or a custom password was created during setup.

Restore Factory Defaults

 

From Hardware:

  1. Locate the back of the device, where the Ethernet ports are located.
  2. Hold the reset button on the back for 30 seconds. You will need a paperclip, toothpick or pen to hold down the reset button.
  3. Let go of the reset button and wait about 1 min for the device to reboot.

 

From Firmware:

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted for the username type in admin, use the password created during the setup process. If you did not setup one then the use the default password admin.
  5. Once logged in click on the Administration tab on the top of the page.
  6. Click on the Factory Defaults in the subcategory.
  7. Click on the Restore Factory Defaults button
  8. Wait about 1 min for the gateway to reboot.

    The problem is that some codes are just numbers, but Speech Recognition insists on adding hyphens to them.

    As an example, one code is   1101039.

    If I say "remove hyphen from last word" it will result in 110 1039, which is wrong because it has a space.

    I can ask for a correction, and then select the option that doesn't have a hyphen, but this slows things down.

    mrperson(2) say:

    Yeah right, thanks Microsoft. 

    ---------------------------------------------------------------------------------------------------------------------

    Anyhow:  I've found a workaround.

    1.  Say the number.  The computer will get the number right, but add hyphens.

    2.  Say "Remove hyphens from last two words" gets rid of the hyphens.

    3.  Say "Remove spaces" to remove the gap left by the hyphen.

    4.  Profit!  

    A few extra steps, and a little annoying, but works ok in practice.
