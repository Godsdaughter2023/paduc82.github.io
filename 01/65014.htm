Downloading the ISO file

An ISO file is a digital or virtual replica of a physical disc. In order to use an ISO you must burn it to a optical disc. In the case of Windows 10, a blank Dual Layer DVD or an empty USB thumb drive. If you are using Windows 7, you can create the .ISO then burn it using the built in Disc Image utility. If you are running Windows XP or Windows Vista, you can using a third party burning tool such as ImgBurn or Roxio/Nero.

the dod say:

Change the bar orientation on my desktop

After an automatic update the bar that used to be horizontal at the bottom of my screen on my desktop is now vertical on the right side of the screen. How can I get it back the way it was? I tried what was suggested by somene else (ctrl+alt+up key....etc.) but it didn't work. Can any one help? tx.
don patterson say:
  1. Click a blank portion of the taskbar and verify that Lock the taskbar is not checked.
  2. Hold down the left mouse button, and then drag the mouse pointer to the place on the screen where you want the taskbar. In your case, drag it to the bottom of the screen.
  3. After you move the mouse pointer to the position on your screen where you want the taskbar, release the mouse button. Right click on it and select Lock the taskbar.
