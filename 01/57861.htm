System Tray

cmdrkeene say:

It can vary depending on which icon you want to hide.  For many things, it's the same situation as Windows 7 or XP (that is, uninstall or disable whatever program is placing the icon there).

For system icons like power/wifi and such, you can turn it off like you would have in previous versions, from the Settings app (moved from Control Panel but functions the same).

There is two exceptions: the touch keyboard can be turned off by right-clicking, and the Action Center item which can't be outright removed.

rileyhutcheson say: I downloaded Avira anti-virus. An icon was placed on the screen and in the system tray. By removing the program ( app ) the icon was removed from the screen but not from the system tray. I want to remove ( NOT HIDE ) the icon from the system tray.

cmdrkeene say:

The program icon showing in the tray means the program is running (actively processing).  You may be able to double-click it to open the full antivirus program into full view.

To finish removing it, I would either double-click it and then exit, or right-click and choose the exit option (if there is one).

Then, go to your Control Panel, under Programs, and run the uninstaller utility by selecting the Avira program in the list and clicking Remove.
