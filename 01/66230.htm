samuel8843 say:

Thank you Barb. For others using a digital-to-analog cable convertor with an internal analog PC TV tuner, the problem occurs when you lose the TV setup info and need to go through the TV setup process again. Media Center detects the converter as a set top cable box with a signal. If you check other tuner info, you find that it detects the internal analog tuner without a TV signal. If you select manual setup and follow the few questions, you setup only the analog tuner which now receives the converted signal without causing an "All TV Tuners are currently in use" error.
