When you hibernate your computer, Windows saves the contents of the system's memory to the hiberfil.sys file. winkey (that one between ctrl and alt)+r
put gpedit.msc and hit enter
go to user configuration,
then administrative templates,
then windows components,
then click on windows explorer (not the arrow)
then double click remove cd burning features
then click enabled, click OK and close it
that's it no more windows integrated burning discs
