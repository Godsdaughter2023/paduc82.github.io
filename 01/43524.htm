Hi Dizuki,

Thank you for writing to Microsoft Community Forums.

In order to get clarity on this issue and assist you with appropriate troubleshooting steps, please reply with the answers to the questions below.

  1. What is the make and model number of your computer?
  2. Were you facing the same issue when you had Windows 7?
  3. Are you facing this issue only with headphones or with the speakers as well?

Before trying the below methods, I suggest you to play online audio and check if you are facing the same issue.

Method 1: Audio Troubleshooter.

First, I suggest you to run the troubleshooter by clicking on the Start button > Type audio troubleshooter in the search box > Select Find and fix audio playback problems from the list of results > Then, select Next.

Restart the computer once running the Audio Troubleshooter and check for the issue.

Method 2: Installing The Generic Windows Audio Driver.

Open Device Manager > Right-click your audio driver > Update driver > Browse my computer for driver software > Let me pick from a list of device drivers on my computer > Select High Definition Audio Device > Select Next and follow the instructions to install it.

Restart the computer after installing the Generic Windows Audio Driver and check if the issue still persists.

Method 3: Try Different Audio Formats.

  1. Click on the Start button > Type Sound > Select Sound Control Panel from the list of results.
  2. On the Playback tab, right-click the Default Device and then select Properties.
  3. On the Advanced tab, under Default Format, change the setting, and then retest your audio device. If that doesn't work, try changing the setting again.

Method 4: Performing A Clean Boot.

If the above methods don't work, then I suggest you put your computer in a Clean Boot State and check if you are still facing the same audio issue while the computer is in Clean Boot.

Note: A Clean Boot starts Windows with a minimal set of drivers and startup programs, so that you can determine whether a background program is interfering with your game or program. Your computer may temporarily lose some functionality while in a clean boot environment, however, that functionality will return when you reset the computer to start normally after you have finished your troubleshooting.

If the Clean Boot works, then don't forget to place the computer to normal boot by going to Reset the computer to start normally after clean boot troubleshooting section in the Clean Boot link.

To determine what is causing the problem after you do a clean boot, please refer to the section How to determine what is causing the problem after you do a clean boot from the above Clean Boot article.
