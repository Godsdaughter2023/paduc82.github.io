We cannot get sound in safe mode. Safe mode is a diagnostic mode that loads only the drivers and services really necessary. When you boot the computer in safe mode most of the services would be disabled automatically. By default audio service will be disabled in safe mode and this is by design. There is no way you can enable the audio service or hear sound when you boot the computer in safe mode.

a. Hold windows key + R and type "msconfig".

b. Go to Boot tab and uncheck safe mode.

c. Click on Apply and Ok and restart the computer check the issue and let us know.

Hi Tara,

 

Yes, you can skip the option to set up a Microsoft user account when setting up the computer for first time. When you are prompted to connect your PC to a Microsoft account, click on “I’m not this user” or “I’m not username”. This will take you to the local user account page.

If you decide to change your local account into a Microsoft account later, you can refer to these steps:

 

a)      Press Windows key + C, select ‘Settings’ and then ‘Change PC settings’.

b)       Now, click on ‘Accounts’ in the left pane.

c)       Click on ‘Your account’ and then click on ‘Connect to a Microsoft account’ and enter your credentials.

Hi R762mm,

No, there are no other alternatives for Windows Easy Transfer in windows.

Windows easy transfer helps you move your many files, folders, and program settings from one computer to another—all in one process. However, you can use backup and restore to backup files only.

Computer not responding issue may occur, if the hard disk has developed bad sectors. To resolve this issue check for disk errors, run the Checkdsk.exe tool on the drives that have to be backed up

      1.    Click Start, type cmd in the Start Search box, right-click cmd.exe in the Programs list, and then click Run as administrator.

If you are prompted for an administrator password or for confirmation, type your password, or click Continue.

      2.    At a command prompt, type the following command, and then press ENTER:
chkdsk c: /f
chkdsk c: /r
      3.    Press Y when you are prompted to check the disk the next time that the system restarts.

      4.    Close all applications, and then restart the computer.

      5.    After the computer restarts, repeat steps 1 through 4 for the other volume, and then rerun the backup operation.

Now, try to run windows easy transfer again.
