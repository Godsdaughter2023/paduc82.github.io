I had the same issue I thought...then I realized that it was everything I was trying to open. (I open mostly photos so it seemed like it was a photos only problem.) For the sake of anyone else having my same issue I'll share here.)

If you can't open anything by double-clicking (or it seems like most of the time it doesn't work)

What I found was that I must have adjusted my double-click speed on my mouse to 'fastest' and I wasn't ever clicking it fast enough to make it work. 

To check/fix:

-In the Search box type "mouse options" then hit enter, on the top right click "additional mouse settings" 

-Click on the "Buttons" tab

-In the middle area named "Double-Click Speed" if it's set to the fastest option, try it lower (and hit apply) until things start opening for you on a double click. This might not have been your issue if it was already set to the middle or slow-ish.

Ta-da :D
