a. user say:

Here's a lesson I learned by doing things wrong the first time. I learned a lot by this method, unfortunately.

Quick review: The DNS service tells your computer how to find the web page you want to see. Your computer tells the DNS server "I want to see the home page of the Financial Times website" and the DNS server replies with the numeric IP address for that page - like a street address. Most people use the DNS service provided by their Internet Service Provider, e.g., Comcast, Verizon FIOS, etc. You can, if you wish, choose a different DNS service. But you have to make this choice manually and consciously.

One thing hackers like to do after they gain control of your computer is to change the DNS servers being used, usually to a server that they control. Now when you ask your DNS service for the Financial Times website you may get the numeric address of a hacked web page, and things rapidly go downhill from there.

There are two places to choose and change the DNS servers your computer uses. One way is in your router. This is a cinch to do for a hacker if you have not given your router's administrative console a strong password. A hacker can easily get in and make the change. Each admin console works a little different, but if you look in the LAN section you'll see how to do it. Change your DNS server back to your ISP's default.

Another way is via Control Panel > Network and Sharing > Change adapter settings > right-click on your network adapter > Properties> Internet Protocol Version 4 (it's unlikely you are using IPv6) > Properties. You should "Obtain an IP address automatically" and "Obtain DNS server automatically."

To stop this from happening again:

1- remove all malware from your computer

2- give your router's administration console a strong password

3- give your user account a strong password

4- always have your BS detectors on 'high' when browsing the web.

RAM dedicated to hardware - Microsoft Community

suehooker say:

RAM dedicated to hardware

Upgraded RAM form 3GNB to 4GB in Window 7 64-bit OS. My BIOS, my dxdiag etc. identify that I have 4GB installed, but under control panel > system it shows 4GB installed (only 3GB usable); I would like to get that other 1GB usable. Can anyone help? Thx Sue

lemp say:

Even if a hardware device has its own built-in memory (e.g., a video card), Windows has to reserve an "address space" equal to the amount of device memory in order to let the system access and interact with that memory.  That amount is subtracted from the available system memory and is said to be "hardware reserved" and unavailable even though the physical memory is there.

Devices have to map their memory below 4 GB for compatibility with non-PAE-aware

Windows releases. Therefore, if the system has 4GB of RAM, some of it is either disabled

or is remapped above 4GB by the BIOS. If the memory is remapped, X64 Windows can use

this memory. X86 client versions of Windows don’t support physical memory above the 4GB

mark, so they can’t access these remapped regions. Any X64 Windows or X86 Server release can.

X86 client versions with PAE enabled do have a usable 37-bit (128 GB) physical address space.

The limit that these versions impose is the highest permitted physical RAM address, not the size

of the IO space. That means PAE-aware drivers can actually use physical space above 4 GB if

they want. For example, drivers could map the "lost" memory regions located above 4 GB and

expose this memory as a RAM disk.

--  https://msdn.microsoft.com/en-us/library/windows/desktop/aa366778%28v=vs.85%29.aspx#how_graphics_cards_and_other_devices_affect_memory_limits

There's probably not a lot you can do short of disabling some hardware device, but you can see what's using the memory this way:

Open Device Manager (click Start Orb, type devmgmt.msc and press Enter)

Click View and select "Resources by connection"

Expand the Memory node

For more detailed discussion of this see the following articles:

  • Pushing the Limits of Windows: Physical Memory by Mark Russinovich
  • The 3GB-not-4GB RAM problem (MSDN Archive)
  • The usable memory may be less than the installed memory on Windows 7-based computers

In some cases, usually when considerably more than 1 GB shows as "hardware reserved," there may be something amiss.  Read this thread carefully, especially posts #6, 3, and 5. 

http://www.sevenforums.com/performance-maintenance/196224-4-gb-physical-memory-but-1-74-gb-usable.html#post1649304
