    How to disable File Caching (changing the options for the hard - Microsoft Community

    tobi#1 say:

    How to disable File Caching (changing the options for the hard drive does not work)

    Dear all,

    hopefully, someone can help me with the following problem:

    - I need to open a file of 16 GB from within a third-party sofware (Stata) which reads it completely into memory.

    - Because of Windows' file caching, the memory needed for that procedure is actually around 32 GB as both file cache and Stata access the physical memory.

    - As I have only 24GB of physical RAM the file is eventually partly read into the virtual memory which leads to serious performance problems.

    I unchecked the box "Enable write caching on the device" for my hard disk. That did not help. Is there a way to disable the Cache when READING files?

    Thank you very much in advance for any help!

    Tobi

    balaji kundalam say:

    Write caching and virtual memory are different. Virtual memory is an extension of physical memory residing in the HDD where Windows is installed. Normally, virtual memory is about twice the size of the physical memory present. Write caching is a setting exclusive for external HDDs only and doesn't have effect in internal HDDs.

    In your case, kindly note that full 24 GB of RAM will not be allotted to the program. At least 1 GB will be reserved for Windows, so rest of the programs are moved to virtual memory, meaning the HDD. It's not recommended to disable virtual memory as it may cause stability issues. Additionally, files are not copied directly to memory, even if they are large. Instead, parts of file are accessed and that area, including the incremental and decremental values, are stored in memory. So if you suddenly navigate to an area of file which is very far from the previous location, it takes time as the new area as well as the incremental and decremental value of few bytes is also copied to memory. The previously viewed file area is cleared from memory.

    For performance improvements, you can go for SSD, RAM with higher clock speed (must be compatible with the motherboard) as well as upgrading processor. Still, kindly note that the process involved in accessing a file will be same as mentioned earlier, but the speed will increase due to higher bandwidth and clock frequency.
