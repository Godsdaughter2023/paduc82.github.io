a. user say:

Hi Alex,

Thank you for posting your query in Microsoft Community.

I understand you want to set the COM port latency to 1 in the Device Manager. I would suggest you to refer to the following steps and check if the issue persists.

a. Press Windows key + X, click on Device Manager.

b. Navigate to: Ports (COM & LPT) > {Select the device driver}

c. Right click and select Properties.

d. Select the Port Settings tab.

e. Click the Advanced button. 

f. Change the Latency Timer (mSec) to 1 or the lowest possible setting.

g. Click OK on the tabs and exit out the Device Manager.
