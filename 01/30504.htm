john john __ say: The gateway is set at the adapter and you cannot selectively assign different adapter properties for different users, all the users use the same adapter.

With Internet Explorer (doesn't work with other browsers) you can set a proxy server with a fake address, you can do this via Group Policies or you can do it for manually for each individual user via the Internet Options settings.  You can set the proxy server to the loopback adapter address (127.0.0.1) or you can set it to a phony address in your private address range (usually the 192.168.xxx.xxx range)

If you use other browsers you can use NTFS permissions to deny access to the browser or the program's directory or you can use software restriction policies to deny access to specific programs.

You could also use a logon script to enable the Network Connection service when you log on and another one to disable it when you log off, but if you do this you will disable the entire network for the limited users so it may not be a suitable solution for them.  The script can be a simple two line script:

Logon Script:

sc config Netman start= demand
net start netman

Logoff Script:

net stop netman
sc config Netman start= disabled
