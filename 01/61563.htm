How to convert WLMP without movie maker / file will not open - Microsoft Community

mikaelavanity say:

How to convert WLMP without movie maker / file will not open

 In the summer of 2017 I made a video through movie maker that's very precious to me and saved the file to a USB stick. Since then, my computer broke and now my current laptop cannot open the file. I've thought about uploading it to Youtube since my computer couldn't play the video but it needs to be converted from WLMP into another format. This video is so important to me a losing it will be devastating. 

considerate_guy say:

The .wlmp file is the project file. The project file is actually just a relatively small text file that contains the following:

  • A list of all your source media files (file path and name)
  • Data relating to the order of each clip, where you trimmed the clip to start and stop, what effects you applied, etc.
  • Data relating to your overall project such as the aspect ratio and audio balance settings.

The project is essentially useless on its own. It contains no actual media since it is just referencing their location on your original computer that broke. It also can't really be used in any other program except Movie Maker.

Before your computer broke, did you have a backup of the source files (photos, video, audio)? If you did, then you might have a chance if you put those in the same location and also reloaded the same version of Movie Maker on your laptop.

But if you don't have the original media files, there is no way that any program can magically resurrect them from nothing but the file name.

In the future, if you are working on a project, be sure to always finish by rendering your work to a movie file. The movie file can be saved and used on another PC. The project file isn't as transportable, for all the reasons I stated above.

davem121 say:

I have finally narrowed it down and found the culprit for this crackling popping sound - it is the IDE ATA/ATAPI Controller driver. 

 

Windows by default loads a driver called Standard SATA AHCI Controller

 

If you have this device driver loaded, then that is the problem:

  1. Open Control Panel – Device Manager
  2. Expand the IDE ATA/ATAPI controllers section
  3. Right click the Standard SATA AHCI Controller and choose properties
  4. Then click the Details Tab
  5. Change the Property drop down to Hardware Id's
  6. Right click the top item in the list and select copy
  7. Open Google and paste the device id
  8. This will tell you the exact make and model of your chipset
  9. Download and install the correct SATA/AHCI chipset controller
  10. Reboot your system
