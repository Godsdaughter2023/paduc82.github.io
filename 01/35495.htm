Even if you can re-enable the Windows Firewall, you shouldn't run two firewalls at the same time. 

Step 1:

I suggest you to uninstall any third party firewall installed on your computer and check if the issue gets fixed.

Uninstall or change a program in windows.
http://windows.microsoft.com/en-US/windows7/Uninstall-or-change-a-program

a. Right-click Command Prompt (in Start Menu\Programs\Accessories) and choose Run as Administrator. Type (or Copy and Paste) the following command:

winmgmt /verifyrepository

b. If the system returns "WMI repository it is not consistent", type this command:

winmgmt /salvagerepository

The first time you run this command it will fail. (It will issue stop commands to the services causing it to fail, saying, "A stop control has been sent to a service that other running services are dependent on"). It might take a couple of minutes for the services to shut down. Run the last command again. You actually may have to run it 3 times before it finally runs and completes on its own.

c. After running the command you may check if the issue gets fixed.

Step 3:

You can also use the Event Viewer to look for information about the issue.

Refer:

How to view error messages in Event Viewer
http://support.microsoft.com/kb/947321

You can check for information about the issue by the time it occurred, and post that information here to help us narrow down where to focus.

Also refer:

Turn Windows Firewall on or off
http://windows.microsoft.com/en-US/windows7/Turn-Windows-Firewall-on-or-off

If that is all she did, you have not been hacked.  This is a common scam... typically going something like this:
  • someone calls and says they are with Microsoft, or working with them, and they've detected malware on your PC
  • frequently they get you to open the windows event viewer, and point out all the alarms and events as 'proof' of problems.  But... the event viewer is always full of this stuff, and 99.999% of it looks bad but is benign.  It's a great tool for scammers to fool the innocent.
  • The next thing they would have done would be to ask for remote access to the PC, to install their own malware, and then start asking for credit card numbers to charge for cleaning it up, and so on...
You will find this interesting:  Fraudulent Emails and Credit Card Scam Tips | Microsoft Security

And finally, some advice-
1) if event viewer is all she did, you do not need to go to any computer techs for clean up.  You can get a ton of good free advice in this forum, including step by step help if you ever need it.
2) make sure your mom's PC has it's windows firewall (or a 3rd party firewall) turned on.
3) make sure it has active antivirus running.  See above for the MS Security Essentials forum, which is free and highly effective.  You can get it here:  Microsoft Malware Protection Center
4) If you already have another antivirus installed, then make sure it is up to date and run a scan.  Never install more than one real time antivirus at a time, as this causes potential performance issues and impairs your protection.  There is plenty of good advice on that last link I posted as well.

Good luck, and sleep easy... you should be OK.
