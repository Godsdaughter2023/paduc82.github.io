lemp say:
lost my character map, not in system 32 folder, how do I get it back?

That's a pretty good trick, considering that charmap.exe is one of the files that Windows File Protection is supposed to protect.

Do a full-disk search for charmap (no extension), making sure to look in Hidden and System folders.

If charmap.exe is not in C:\Windows\system32 (where it belongs), it probably also isn't in C:\Windows\system32\dllcache (where it also belongs).

If you have a C:\i386 directory (which is common for systems with pre-installed Windows), you should find charmap.ex_ there, which is a compressed version of the file.  You should use the EXPAND command from a Command Prompt window (Start > Run > cmd > OK)  to uncompress it:

    expand c:\i386\charmap.ex_ c:\windows\system32\charmap.exe
    copy c:\windows\system32\charmap.exe c:\windows\system32\dllcache

Don't forget the space before the c:\ in two places in both lines

You'll also find the charmap help files in C:\i386 and can expand those if needed as well.  The expanded files should reside in C:\windows\help

This indicates that a driver has improperly used IoBuildPartialMdl.
