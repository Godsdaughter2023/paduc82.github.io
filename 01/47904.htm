rajesh govind say:

Hi Jared,

 

There are many reasons why the Windows Operating System notifies the issue of low memory on the computer. Some of them are:

 

  • All the installed memory is not completely available for use.
  • A program doesn't free up memory that it no longer needs. This problem is called memory overuse or a memory leak.
  • Low Virtual Memory
  • Unnecessary processes running at the background using a large amount of memory.

 

First of all, I would suggest you to check if the complete 6GB RAM is available for use. This can be check from the “System Properties”. To open the “System Properties”, you may use its keyboard shortcut. Press “Windows Logo” + “Pause/Break” keys on the keyboard and that should open the “System Properties” on your screen.

 

On the “System Properties” window, you may check the installed RAM and the usable RAM (within the braces). Please let’s know the amount of RAM that is available for usage.

 

If this issue is due to the over usage or memory leak by certain applications, then this could a problem with the application. Check if any updates are available from the application developer’s website and install them on the computer and check if that resolves the issue.

 

You may also try increasing the Virtual Memory and check if that resolves the issue.

 

Try Increasing the Virtual Memory on the Computer

 

Follow these steps to increase the Virtual Memory on the computer:

 

a) Press “Windows Logo” + “R” keys on the keyboard, type “sysdm.cpl” in the “Run” command window and then press “Enter”.

b) Click on the “Advanced” tab.

c) On the “Advanced” tab, under “Performance”, click on “Settings”.

d) On the “Performance Options”, click on the “Advanced” tab, and then, under “Virtual memory”, click on the “Change” button.

e) Uncheck the option “Automatically manage paging file size for all drives”.

f) Under “Drive [Volume Label]”, click the drive that contains the paging file you want to change.

g) Select the option “Custom size”

i) Set “Initial size” equivalent to your RAM size in MB.

j) Set “Maximum size” to twice of your RAM size in MB.

k) Click on “Set”, and then click on “OK” thrice.

l) Restart the computer.

 

If the issue still persists, then check if there are any unwanted applications running in the background of the computer. To check this, let us perform a clean boot and check the status.

 

Perform a Clean Boot and find out which application/service is causing this issue.

 

A clean boot helps to eliminate software conflicts and to determine what is causing the problem. To perform a clean boot, please refer to the section “How to perform a clean boot” and follow the instructions for “Windows 8.1 and Windows 8” from this link:

 

http://support.microsoft.com/kb/929135

 

You may please note that performing a clean boot does not fix the actual issue. It just helps us in identifying the root cause of the issue.

 

If the issue does not occur in clean boot, then you may determine which application/service on the computer is causing this issue. This can be done by following the instructions for “Windows 8.1 and Windows 8” given in the section “How to determine what is causing the problem by performing a clean boot” from the same given link.

 

Important: After you have finished troubleshooting, refer to the section “How to reset the computer to start as usual after clean boot troubleshooting” for “Windows 8 and Windows 8.1”, to reset the computer to start as usual from the above link.

 

Hope this resolves the issue. Please get back to us with the troubleshooting results and any additional information regarding this issue.
