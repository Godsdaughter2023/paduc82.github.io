2.   Remove battery and re-connect the AC adapter.

3.   Power on the laptop using the AC adapter.

4.   Open the device manager.  Select 'Batteries'.

5.   Right-click on 'Microsoft AC adapter', then select 'Uninstall'.

6.   Right-click on 'Microsoft ACPI-Compliant Method Battery', then select 'Uninstall'.

7.   Close the 'Device Manager' and power off the laptop.

8.   Re-install the battery and plug in the AC adapter.

9.   Power on the laptop.  The AC adapters will re-install themselves on start up.

10. The laptop battery will then start charging.

Regretfully, if there is a power outage or you unplug the AC adapter; you will have to repeat the process.

WP3 is a Photo Story project file that is only compatible
with Photo Story and is for the purpose of re-editing. This file
is created when you click any of the several...Save Project...
buttons in the program.

To render the WP3 project file into a WMV movie file...all you
need to do is re-open the WP3 file in Photo Story and navigate
through all the screens...

On the last screen that says: "Save Your Story" just click "Next"
one last time and the WMV video file will be created.

Also...on the "Save Your Story" screen you may want to 'Browse'
for a save location and enter a name for your video in the "Specify
the location and file name of your story" section. 'Desktop' would
be a good choice...to make it easier to find the completed file.
