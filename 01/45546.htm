Your Recovery Partition was imaged on your Hard Drive during the Manufacture of your Computer by the Manufacturer for these reasons.

 

1.To reinstall Windows from in case of a System crash, by pressing particular Key or Keys at Boot.

F10 or F11 or Alt + F10 are a couple of sequences different Manufacturers use.

Backups should be kept on an External Hard Drive.

usasma say:
Most black screen errors are related to the video card:
- make sure it's getting cooling (that the fan works and it's not filled with dust/hair/etc).
- make sure that the power plug is firmly seated (and that the card is firmly seated in it's slot).
- make sure that you have the latest drivers from the video chipset manufacturer's website ( http://www.nvidia.com )

Second step - update all other drivers on the system, then get all available Windows Updates.

Additional step - if you get it working, immediately set a System Restore point.  This doesn't always work, but is one of the few semi-reliable fixes for KSoD (Black Screen of Death) issues.

Make sure that you have all updates for the game.

Finally, reinstall Windows if all else fails.  Get all available Windows updates - then make sure to update drivers from the device manufacturer's website (not from the PC manufacturer's website, not from Windows Update, and not from the Update driver function in Device Manager). Then visit Windows Update again and again - until no more updates are available.

Try another video card if the problems continue.
