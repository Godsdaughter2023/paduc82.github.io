Honestly you are best to either restore to factory settings or to perform a clean installation. In my experience repair installs do not fix the instabilities caused by a bloated registry. i.e. are just a waste of time.

This is why Microsoft introduced the Refresh (and Reset) for Windows 8 onwards. The Refresh allows reinstallation of Windows without loss of user data. A Refresh  from Windows 8.1 installation media installs a fresh set of Windows installation files from the Windows 8.1 installation and removes all software and drivers installed. A refresh from a Refresh Image on the other hand reinstalls all the files including drivers and third party programs included in the time the Refresh Image was made again not affecting users files.

The Refresh is not a feature available for Windows 7 installation and in order to get a fresh set of installation files you really need to reinstall everything. Note if you use or have previously used the Dell Windows 7 Reinstallation DVD or Generic Microsoft Windows Installation DVD you will lose/already lost the ability to revert to the Dell Factory settings and the recovery partition is either deleted or useless in such cases.

If you have on the other hand used Windows Anytime Upgrade to go from Windows 7 Home Premium to Windows 7 Professional the factory image and recovery partition will be intact. You can revert to the factory image and then use Windows Anytime Upgrade again to get to Windows 7 Professional.

In order to best help you, let us know if you used the DVD to previously upgrade to Windows 7 Professional or if you used Windows Anytime Upgrade as this will dictate whether the factory image is available or not.

jack mvp say:

Hi

Since you are on Wireless connection it could be that the problem is No the Interent but the Wireless connetion between the Computer and the wireless Router.

-----------

A computer with Wireless connection has to maintain solid connection with the Wireless Router.

If the connection is not stable and often drops, then you can Not use the Internet or the Network even if the other computers are still Ok.

I.e., it is as if someone cut the wire that connects a regular computer to the Router.

Some Wireless Routers behave this way because they are "Quirky" to begin with. Check if the Vendor's support has a new firmware. Download and install new firmware.

Some Wireless cards might need the latest OEM Drivers to be stable. Download and update the Drivers.

Otherwise, try to change the channel that is used by Router for the Wireless transmission. Try all the channels and find which one yields the best results.

Try to take the Wireless card from the computer’s Power Saving loop. I.e., the Wireless card should work at full power and will Not be allowed to Save Power.

If your computer is too far from the Wireless Router, and the environment is electronically noisy it could be that you need to re-work the topology of the Wireless.

It is impossible to give instructions that are more precise since each device has different configuration menus. The extended manual of the devices usually explains how to apply the changes that were mentioned above.

chris cupler say:

A failing router may create internet connection issues; however, it will not cause a PC to hang, crash applications, or throw a blue screen (and auto-reboot.)

As to why the new PC has come down with the same symptoms as the old PC (random hangs, app crashes, blue screen / reboot), well......  frankly, that's rather likely your behavior (download habits, program installs - things like that), or a piece of hardware (and, with it, buggy drivers) that you had with the old PC.

Two things I'd like you to do for the BSoD (and, the other issues) that you're having:
- disable automatic restart.  Refer to this page for how to do it.  Next time the system bluescreens, write down all of the information that's on screen (or, better, take a picture.)  Post results.
Also helpful:  download and install WhoCrashed.  It'll examine the system's crash-dump files, and (usually) can tell you exactly (in fairly plain English) what caused the failure.
- Once point #1 is handled, start Windows in a Clean Boot configuration.  Slowdowns/hangs still happen yes/no?

Regards,

This is more than a one-time upgrade: once a Windows device is upgraded to Windows 10, we will continue to keep it current for the supported lifetime of the device – at no additional charge. Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

However, this is not same for retail licenses i.e. you purchased the license through Microsoft store. That does carry transfer rights. If that is the case, please see the below steps:
(Not applicable to you)
What you need to do is to grab the key using any Software like Showkey Plus, uninstall that key from older PC and use it on another computer.

Download Showkey Plus:
www.thewindowsclub.com/showkeyplus-product-key-finder

Uninstall the key
https://winaero.com/blog/how-to-deactivate-wind...

Use the key by Showkey Plus on the newer computer.

Alternatively, there are two more options available other than this:
(Not applicable to you ends)
Contact Microsoft at https://partner.support.services.microsoft.com/... through chat support. They would take control of your PC and activate(might need a proof of purchase)
Use the Activation troubleshooter on the new PC signed in with same MS account.
Go to Settings>Update and security>Activation and run the troubleshooter.
Select I recently changed hardware on this device.
Please let us know if you need any assistance.
