suzanne s. barnhill say:

If what you have is a single paragraph, then there's no way to make one sentence closer to another, except that, if they are separated by two spaces, pressing Delete will remove one, leaving a single space between sentences. If the paragraph is justified, that space may be larger than desired; you can reduce extra space in the line by judiciously inserting optional hyphens (Ctrl+Hyphen).

 

You can change the line spacing to bring the lines closer together. Or, if each sentence is actually a separate paragraph, then pressing Delete at the end of one sentence will delete a paragraph break, but if the writer pressed Enter twice between sentences, then there is also an empty paragraph to be deleted, so you'd have to press Delete twice.

 

Your best bet for dealing with any of these situations is to display nonprinting characters (formatting marks) by pressing Ctrl+* or clicking the Show/Hide ¶ button in the Paragraph group on the Home tab. This will allow you to see a graphic representation of spaces, paragraph breaks, and other items that affect the formatting of the document (see http://word.mvps.org/FAQs/Formatting/NonPrintChars.htm).

suzanne s. barnhill say:

It appears to me that each sentence is a paragraph. The wide spacing could be caused by any of (at least) four factors:

  1. The paragraphs could have some Spacing Before/After (see the Paragraph dialog). But if you press Delete at the end of one paragraph, it should join with the next paragraph.
  2. The vertical alignment (Layout tab of the Page Setup dialog) could be set to "Justified" rather than "Top". That would spread out the paragraphs, but they should still join if you press Delete at the end of one.
  3. There could be one or more empty paragraphs between the paragraphs you see. Displaying nonprinting characters (Ctrl+*) would help you troubleshoot this. But again, pressing Delete repeatedly should get rid of these.
  4. If you pasted this from the Web, there's a good chance the text is in a table, each sentence in a separate row, with exact row height. If that's the case, pressing Delete would have no effect. When you place the insertion point in one of the sentences, does Word display the contextual Table Tools tab? If so, that's the issue. If you don't have table gridlines displayed (which I assume you don't if this is the problem), then you can click the button for View Gridlines on the Table Tools | Layout tab. But if you want to combine the paragraphs, you'll need to convert the table to text. The Convert to Text button is at the right end of the Table Tools | Layout tab of the Ribbon.
