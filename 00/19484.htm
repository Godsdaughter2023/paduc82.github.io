msoffficeuser say:

Chapter number incorrect in Figure and Table captions for new Style added manually

Dear Friends

I am using Microsoft 2003 Word for creating my project document.

I am using "Heading 1" Style and Formatting with Outline Numbered Bullets and Numbering for main chapters. E.g.
1. Introduction
2. Data analysis

10. Conclusion

But, for Appendix, References etc sections I do not want to use this numbering. So I created a new Style "FreeFloatHeader" which has almost the same formatting as that of "Heading 1" style but without numbering.

Also, I want to use these chapter numbers in the captions of the figures and tables that I am adding in each chapter as well as in Appendix.
So, ideally, if Figure & Table captions would be "1-1", "5-1" etc. For, this we select "Include Chapter Number" option in the "Caption Numbering" box from "Numbering" in "Insert->Reference->Caption" dialog box.

But, now I am finding that, when I add the captions for the Figures and Tables in the Appendices (which are after conclusion, the 10th chapter), then instead of showing the caption as 11-1, 11-2, I am getting the chapter numbers as 10 in the captions.

How can I get the the correct chapter number for the captions for tables and figures in the chapters which does not have Heading without numbering?

Can some one please help me?

Thanks and Regards,
- ganesh

suzanne s. barnhill say:

Is there some reason you are not following the easier path of just using otherwise unused built-in heading styles for your appendix headings. This allows you to use the "Add chapter number" feature directly, without a lot of fiddling with the SEQ fields.

As for the error, is there actually text in the style Heading 1 Appendix, and are both Heading 1 and Heading 1 Appendix autonumbered? FWIW, I would have thought that a style based on Heading 1 would continue its numbering sequence, but I have no experience with this.

suzanne s. barnhill say:

You can create styles that are visually identical to heading styles (but are not based on them) and map them to the same TOC level. I usually create a Front Matter Head style for this purpose (with the same font and paragraph formatting as Heading 1). If you give it an outline level of 1, it will appear in the TOC using the TOC 1 style.

The reason you're getting 10 for the "chapter number" in your appendixes is that the last numbered heading is number 10. You can't have a "chapter number" based on an unnumbered heading (which I gather your appendix headings are), nor can you change the designation of the heading used for the chapter number in any given caption. That is, if you say that the Table caption uses Heading 1 to supply the chapter number, then it must do so for the entire document. If you want to use numbering from a different heading, you must create a different caption label.

ossiemac say:

Try the following code example. The Function is a UDF to find the last used row or column on a worksheet or range.

Note the comments in the code, particularly where you can specify the worksheet name rather than the ActiveSheet.

Run the Sub ApplyFormulaToRange

Sub ApplyFormulaToRange()
    Dim ws As Worksheet
    Dim lngLastRow As Long
   
    Set ws = ActiveSheet
   
    'Alternative code to previous line to specify sheet by name rather than the ActiveSheet
    'Set ws = Worksheets("Sheet1")
   
    lngLastRow = LastRowOrCol(True, ws.Cells)   'Find last used row on worksheet
   
    With ws
        Application.CutCopyMode = False
        .Cells(3, "Y") = "=A3&F3"       'Enter formula in cell Y3
        'Copy cell Y3 and paste to cells down to last used row
        .Cells(3, "Y").Copy Destination:=.Range(.Cells(3, "Y"), .Cells(lngLastRow, "Y"))
        Application.CutCopyMode = False
    End With
   
End Sub

Function LastRowOrCol(bolRowOrCol As Boolean, Optional rng As Range) As Long
    'Finds the last used row or column in a worksheet
    'First parameter is True for Last Row or False for last Column
    'Third parameter is optional
        'Must be specified if not ActiveSheet
   
    Dim lngRowCol As Long
    Dim rngToFind As Range
   
    If rng Is Nothing Then
        Set rng = ActiveSheet.Cells
    End If
   
    If bolRowOrCol Then
        lngRowCol = xlByRows
    Else
        lngRowCol = xlByColumns
    End If
   
    With rng
        Set rngToFind = rng.Find(What:="*", _
                LookIn:=xlFormulas, _
                LookAt:=xlPart, _
                SearchOrder:=lngRowCol, _
                SearchDirection:=xlPrevious, _
                MatchCase:=False)
    End With
   
    If Not rngToFind Is Nothing Then
        If bolRowOrCol Then
            LastRowOrCol = rngToFind.Row
        Else
            LastRowOrCol = rngToFind.Column
        End If
    End If
   
End Function
