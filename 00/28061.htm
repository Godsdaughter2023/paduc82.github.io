Creating multiple/duplicate guitar chord templates - Microsoft Community

bltzhd61 say:

Creating multiple/duplicate guitar chord templates

Hello,

Here's my question/ issue. I want to create my own blank guitar chord template pages. In Word I managed to create one lone diagram. I simply want to copy and paste it across the page giving me six across and six down.  I figured out how to do everything else. I simply don't why I can't copy and paste the chart. Thanks.

a. user say:

I hope Yoga Yao's post has helped you to make progress.

If you were happy with the layout I originally provided, it is possible that what you were seeing was a single grid and a load of things that look like { REF o }. In that case, you should be able to click alt-F9 (or fn-alt-F9/option-F9/fn-option-F9) to show a complete page of chord grids, which you should be able to print or whatever. 

If you want to use the grid you originally created with, the question is whether the problem you were having was making even *one* copy of that grid, or whether the problem was with laying them out in the 6x6 pattern you wanted.

For the 6x6 grid you could start with the layout I provided, but click on each chord grid in turn and delete it (and/or select each { REF o } and delete that. Or you could make a completely new document with your own 6x6 table and use the Table Properties to make the rows and columns the size that you need and make the table lines invisible. Then you would need to copy your own grid and paste it into each of the 36 cells.

If you inserted your original grid by inserting a table, you should be able to mouse over the table until you see the little 2x2 selection box appear at the top left, and click that. Then you should be able to click command-C to make a copy of the table. Then click in cell one of your 6x6 table and click command-V to insert it. Then repeat for each of the other 35 cells.

If you can't select the table by clicking on it, it's possible that you created it in the header (or footer). In that case you would need to use View->Header and Footer first, click in the header, then try clicking the table. (And/or click in the footer, then try selecting the table). Or it may be simpler to create a completely new diagram.

If you created your original diagram some other way (e.g. by using Word Shapes), you may need

 a. to turn it into a single shape before copying it and

 b. to make it an inline shape before copying it into the 6x6 table

If you need to do (a), click on one part of the diagram to select it, then hold down the Command key while clicking on each other part in turn. Then right-click and choose Grouping->Group.

To do (b), right-click on the shape (or the grouped shape), click Wrap Text, then click "In Line with Text" if it is not already selected.

Then select the shape and use command-C, command-V to copy it into each cell of the 6x6 table.

I think the 6x6 table is the easiest way to create and adjust the layout, but there are other ways you could go (e.g. use Format->Columns to set up the document so that it has 6 columns, and take it from there).
