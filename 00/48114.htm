lemp say:
It depends on how "secure" you want your folders to be and from whom you want to protect them.

Encrypting data, including folders and their contents, is a rather extreme method.  Windows XP Pro has a feature called the "Encrypting File System," but this is so powerful and so dangerous that you should be glad that it is not available in XP Home.  Many people have irrevocably lost their important documents and pictures by careless use of the Windows Encrypting File System.

Windows XP implements file protection through a system of permissions.  However, most of the permission-based security available in Windows XP, whether Home or Pro, depends on the files being stored on a disk drive that has been formatted with the NTFS file system.  If your hard drive has been formatted using the older FAT32 file system, these protections are not available to you.

To determine the file system in use, open My Computer, right-click on the drive where the file is located, and select Properties.  The file system in use is shown on the General tab.

Further, Windows XP Home only permits you to "Make this folder private" if the folder is located in your profile.  That is, the beginning of the folder's location must be C:\Documents and Settings\username where "username" is your profile name.  See http://support.microsoft.com/kb/q307286/

If your disk is formatted using FAT32 or you are using XP Home and the file you want to protect is not in your profile, you can ZIP it by right-clicking on the file and selecting "Send to ... Compressed (Zipped) folder."  After the zipped folder has been created, double click on it and then select File > Add a Password.

Your best other alternative is a third-party encryption program such as Truecrypt that Wunders suggested.  Although you are asked for a donation at the Truecrypt site, that is voluntary.  I don't know why you think that "they they want to sell me the software."

  9. Select Reset this PC/Recover from a Drive. Choose Remove Everything and Fully Clean the Drive.
