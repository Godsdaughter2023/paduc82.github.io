Using a list populated from a table as criteria in a Query - Microsoft Community

donald r. cox say:

Using a list populated from a table as criteria in a Query

I don't know if this can be done. I have a table of School Semeters. I have a query that uses the semester field as criteria.  Rather than having a criteria dialog box appear to prompt the user enter a semester, I'd like to have a list of the semesters displayed for the user to select from.

hansv mvp say:

Create a form, name it frmParameters.

Place a list box on the form, name it lbxSemesters.

Set the Row Source of the list box to the name of the semesters table, or to a query based on this table.

If the semester name is the primary key and also the first field of the table, you can use the list box as is.

If the semester name is the second field, with (for example) an AutoNumber field as primary key and first field, set the Column Count property of the list box to 2 and the Column Widths property to 0";1".

Change the parameter in the query to

[Forms]![frmParameters]![lbxSemesters]

jack cook - aka help_line say:

This is the only thing I can offer:

Reset an unresponsive phone

  1. Press and hold the Volume down and Power buttons at the same time until you feel a vibration (about 10 to 15 seconds).
  2. When you feel the vibration, release the buttons, and then immediately press and hold the Volume down button until you see a large exclamation mark.
  3. Once the exclamation mark appears, press the following four buttons in this order: Volume up, Volume down, Power, Volume down. Your phone should now reset and restart itself. (It might take a while for the reset to finish.)

How to get a shortcut for Bing? - Microsoft Community

mammyzx say:

How to get a shortcut for Bing?

Original title: bing needed

i cant get a shortcut to bing as my search engine?

a. user say:

You won't be able to create a shortcut to Bing.  It is a web search engine.

Press Winkey + R to open Run Box

Type www.bing.com > Press Ok to launch an Internet Explorer which opens to Bing

Set Bing as your Default search engine in any browser you may use.  This works for Windows 7 and below.  When you launch the browser and perform a search, Bing will be the search engine. 

To set Bing as your Default Search Engine in Internet Explorer 9, do the following:

 

1.       Launch Internet Explorer and press F4

2.       In the drop down menu, click on Add in the lower right side of the menu which will take you to

          the Internet Explorer Gallery

3.       Select Bing from the list of Search Providers

4.       In the confirmation box that appears, select the ‘Make this as my Default Search Provider’

          check box

5.       Click Upgrade

 

To set Bing as your Default Search Engine in Internet Explorer 10, do the following:

 

1.       Launch Internet Explorer 10

2.       Click the Tools Gear and select Manage add-ons

3.       Select Search Providers in the Add-on Types column to view the current search engine selection

4.       If you don’t see Bing listed, click on the ‘Find more search providers’ link  on the bottom left

          portion of the window which will take you to the Internet Explorer Gallery

 

Set Bing.com for your Home Page and once you launch the browser, it will open to Bing.

http://www.bing.com/set/homepage

Change your Home Page for Internet Explorer

http://windows.microsoft.com/en-US/internet-explorer/change-home-page#ie=ie-11
