Extract unique values only from column A comparing it with another - Microsoft Community

pedrobernardo184 say:

Extract unique values only from column A comparing it with another column

Hi i need to compare two columns from diferente sheets and extract to a new sheet the unique values of only one of the columns for example i have values in column A from sheet 1 and i want to compare it with values that exist in column A from sheet 2 and extract the unique values (values that appear only on column A from sheet 1) to another column in sheet 3

Can someone help please

vijay a. verma say:

Though a single formula can be made but that will be complex. It is better to use a helper column.

1. Let say that column I is a helper column in Sheet 1. Put following formula in I2 of sheet1 and drag down.

=IFERROR(IF(OR($A2="",ISNUMBER(MATCH($A2,Sheet2!$A$2:$A$1000,0))),"",$A2),"")

It will generate all values of column A in Sheet1 which are not in Sheet2.

2. Now, let's assume you want to generate unique list in column C of Sheet3. Put following Array formula in C2 of Sheet3 and drag down -

=IFERROR(INDEX(Sheet1!$I$2:$I$1000, MATCH(0, IF(Sheet1!$I$2:$I$1000<>"",COUNTIF(C1:$C$1, Sheet1!$I$2:$I$1000)), 0)),"")

Note - Array Formula is not entered by pressing ENTER after entering your formula but by pressing CTRL+SHIFT+ENTER. If you are copying and pasting this formula, take F2 after pasting and CTRL+SHIFT+ENTER. This will put { } brackets around the formula which you can see in Formula Bar. If you edit again, you will have to do CTRL+SHIFT+ENTER again. Don't put { } manually.

tom ogilvy say:

Copy the column where you want to find the unique values to the third sheet.

Assume it is column 1 of sheet1 copied to column 1 of sheet3

Assume you have a header in row 1 and values start in row 2

In Sheet3, B2 put in the formula

=Countif(Sheet2!A:A,A2)

then drag fill down the column

Apply a filter to column B and put in a criteria of >0

Select all the rows that are visible and delete them

Remove the filter and you have your list.  (You can clear column B).
