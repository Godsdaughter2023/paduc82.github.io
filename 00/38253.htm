Info for WIndows XP - Windows Picture and Fax Viewer
------------------------------------------------------------------------
OK...but beware...you will lose photo thumbnails in Windows Explorer...
your 'My Pictures' folder...for example...will not display thumbnails.

And...FWIW...WP&F Viewer is not a program...it is just a .dll file. Disabling
the .dll file is basically the same as uninstalling it. (that way...if you do not
like the result...you could re-enable it).  

But if 'gone forever' is what you prefer...just delete the  shimgvw.dll file.

One last warning...your thumbnails in Explorer will be gone.

How to...Disable WP&F Viewer

Go to Start / Run and enter the following command:

Type or copy/paste...        
         regsvr32 /u shimgvw.dll
(yes, the space after the 2 and u is required)

Press...Enter...

(You should see a dialog box that says:
DllUnregisterServer in shimgvw.dll succeeded)

How to...Re-Enable WP&F Viewer

Go to Start / Run and enter the following command:

Type or copy/paste...  
         regsvr32 /i shimgvw.dll
(yes, the space after the 2 and the space after the i is required)

Press...Enter...

You should see a dialog box that says:
DllRegisterServer and DllInstall in shimgvw.dll succeeded

To delete  shimgvw.dll  go to the following folder and right click delete  shimgvw.dll.

       C:\WINDOWS\system32\shimgvw.dll

If you cannot find the path you need to go to...
Start button / Control Panel / Folder Options /
View tab / tick...'Show Hidden Files and Folders'...

The secondary hard drive bay was still empty, so I put my original (traditional spinning-disk) hard drive back in that bay. It was recognized immediately by both my BIOS and Windows so I cut and pasted all valuable data onto my SSD (There wasn't much).  Then I deleted the primary partition, with the intention of freeing up the entire disk for stuff like music.

Much to my dismay, the Disk Management snap-in within the Computer Management utility included in Windows Vista won't let me delete the EISA partition on my old hard drive; when I right-click on the visual representation of the partition, the only option I get is "Help," not "Delete" or other useful functions (which did appear for the main partition, and I was able to delete it).

I don't need the EISA partition because A) it's not even on my boot disk and B) I have all my original OEM recovery discs, and they are in excellent condition.

If someone could tell me how I could delete the EISA partition on a formerly-primary-but-now-secondary 111GB laptop hard drive to reclaim the last 9GB for use by the illustrious Mozart and Tchaikovsky, I would be much appreciative.

I would expect there to be some way to do this in Windows that I just don't know about, which is why I'm posting here, but if not, I am prepared to find and download an ISO of and burn a Knoppix disc to boot into Linux temporarily just to delete that EISA partition (though I'd really rather not do that, for various reasons).

gamer6789 say:

I ended up downloading a free hard drive partitioning tool from softpedia.com and it was able to delete the EISA partition without issue.  I then uninstalled the free tool I had just downloaded (Honestly, I don't even remember the name of the program, but it wasn't hard to find looking through Softpedia's list of hard drive utilities), and then used the Disk Management snap-in within the Computer Management utility to format the disk, which now has my music on it.

I started this thread when I thought I had run out of options, only to promptly realize that some kind of solution could be obtained by downloading something from Softpedia, which was true.
