The Witcher 3 is not a DX12 game, but that's OK because Win10 has DX9,10,11 & 12 and very few games use DX12, yet.

Windows does not determine what version, or which DirectX file gets used, the game does.

- Old games are limited to using DX9 (even older games use only DX7 or DX8).

- More modern games (since Vista & Win7)  use both DX9 and 10 or 11. This allows the games to run on lower spec,

and older systems that don't have the power or capability (the graphics processor has to be capable of using

the version of DX in question) to use DX10/11.

- A number of recent games are restricted to DX10 or 11 and have no option for DX9. I'm not certain if any of the

few DX12 titles have a DX9 option.

- Some games have a DX10/11 option in the games setup, but others use the DX10/11 files automatically when the user

selects Very High or Ultra settings. If the graphics processor is not capable of using a particular version of DX

the relevant graphics settings is normally greyed out in the game graphics setup.

- If your machines graphics processors is up to spec run the DirectX Web Installer to verify if all DirectX files

are installed on your Win10

a. user say:

I'll use Windows Media Player as an example. Other music players work similarly.

  • Open Windows Media Player.
  • Go to Organize > Options > Library
  • Put check marks in 'Display media information from the Internet' and other options that you like.
  • Go to Organize > Options > Library.
  • Put a check mark in 'Retrieve additional information from the Internet.'
  • Go to Organize > Apply media information changes

It will take a while to download all the information from the internet. My library isn't as big as yours - I've got about 3,000 files - and it took about 10 minutes to get it done.

Then you'll want to go through the folders and make sure everything is to your liking. You will almost certainly want to make changes, because music libraries are very personal.

Other popular music players with library functions include MediaMonkey, Music Bee and WinAmp, among many others. And there's always iTunes.

halima s say:

Hi kylejwx,

The error message mostly occur if you do not have sufficient permissions set to access the network drive or you have enabled “password protected” sharing.

First make sure the “password protected” sharing is turned off on the computer.

1. Open Network and Sharing Center by clicking the Start button, clicking Control Panel, clicking Network and Internet, and then clicking Network and Sharing Center.
2. Under Sharing and Discovery, click the arrow button next to Password protected sharing.
3. Click Turn off password protected sharing, and then click Apply. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

If the above step fails then you will need to provide full control permission to the files you want to share.

1. Locate the folder with the files you want to share.
2. Click one or more files or folders that you want to share, and then, on the toolbar, click Share.
3. In the File Sharing dialog box, do one of the following:
• Click the arrow to the right of the text box, click Everyone to share the files with everyone on your network, and then click Add.
• Make sure you provide full control to “Everyone”

Check if it works.

If the above step fails then I would suggest that you refer to the below Microsoft KB article and try the manual steps mentioned.
Error message when you try to access a mapped drive that is mapped to a Windows XP-based computer from a Windows Vista-based computer: "System error 5 has occurred"

Method 1:  Depending on whether you have installation media (such as a DVD or USB flash drive):
