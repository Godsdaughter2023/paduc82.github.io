jameel_ say:

This is a bit old, but i figured I'd paste a solution to this problem as we just encountered it when mmigrating to Win7 and Office 2010.

 

First, this only worked because we were missing both the XPS printer driver and the OneNote printer driver. It wasn't until we fixed the XPS driver that the OneNote driver would reinstall.

 

There are two parts to this fix:

1) Repair XPS services among windows features

2) Repair the Office 2010 installation

 

 In more detail:

 

1) Turn off/on XPS services

* type 'feature' into the start menu search bar and select 'Turn windows features on or off'

* At the bottom of the list is 'XPS services' and "XPS viewer'.  Uncheck both of them and click OK

* The window will close automatically when complete

* Return to Windows Features and re-check XPS Services and XPS Viewer

 

This will install the XPS printer driver to the list of printers

 

2) Repair Office 2010

* Close all Microsoft office 2010 programs (word, excel, onenote, outlook, powerpoint)

* Goto uninstall a program from control panel

* Select Microsoft Office Professional Plus 2010 and click 'Change' at the top

* When the office installation window loads, select "repair' and click next

* Wait 20-25 min for the repair

* Restart the computer (probably unnecessary...)

 

A new printer should show up, "Print to OneNote 2010"

Hi Jim,

Thank you for posting to the forum regarding getting contacts from your Hotmail account to a new iPhone. I will provide a way you can get your contacts exported from your Hotmail account to your PC.  Once this is done you should have multiple options to get the file on your PC to the new phone. 

The steps to export the contacts in a CSV file are as follows:

  1. Log on to your Windows Live Hotmail account.
  2. In the left pane, click Contacts.
  3. On the Windows Live Hotmail toolbar, click Options, then click Export Contacts.
  4. Under Export contacts, click Export contacts.
  5. When prompted to open or save the file, click Save.
  6. Select a location to save the file, and then click Save.

Note The default file name is WLMContacts.csv.
