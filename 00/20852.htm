ken sheridan say: It goes against all my better instincts, but if you are sure this is what you want to do.

The tblMasterPermitsDatabase table should include the following columns: Agency Code, Agency and Agency Type.  the first is the foreign key column referencing the primary key of tblAgency.

The agency combo box in the form for inserting rows into the tblMasterPermitsDatabase should be set up as follows:

Name:    cboAgency

ControlSource:    [Agency Code]

RowSource:     SELECT [Agency Code], [Agency Type], [Agency Name]  FROM [tblAgency] ORDER BY LastName, FirstName;

BoundColumn:   1
ColumnCount:    3
ColumnWidths:  0cm;0cm;8cm

If your units of measurement are imperial rather than metric Access will automatically convert the last one to inches.  The important thing is that the dimensions of the first two columns are zero to hide them.

In the combo box's AfterUpdate event procedure put the following code:

Me.[Agency Type] = Me.cboAgency.Column(1)
Me.[Agency] = Me.cboAgency.Column(2)

The Column property is zero-based, so Column(1) is the second column, Agency Type, and so on.

sven johannsen say:

Two options.  Go to the store, tap the hamburger, and select, my library. This will list everything you ever bought, downloaded, thought about (not really, but it seems like it). Just tap the download symbol at the right. Don't tap the icon as that will open the store to that app, and you will have to get back in and scroll back to where you were.  This might be the better way as you can decide what you do and don't actually want.

The other way involves a reset of the phone. First make sure you have done a backup of the 1020 to the cloud. Then go to settings in the 950XL, about, at the bottom, reset this device. It will spin gears and bring you back to out of the box new. This time when it asks you to sign in with a MS account, do so with the same one you used on the 1020 and backed up. It will shortly offer to restore from a backup and list the ones it knows about. At the bottom of the list is 'set up as new device' BTW. Your latest backup should be right at the top, and if you named your device(the 1020), it will show that name. To the best of its ability it will restore from that backup, background picture, start layout, and the fact you have certain apps. It will need to go download those apps, but will do that by itself. At the beginning, some on the start menu and program list may be grayed out until it re-downloads. You can see progress in the store under downloads and updates. NOTE: it will rename your 950XL the same as your 1020...it is a restore, so go change the name after it is done, so you can tell the backups apart more easily.

Both methods take some time to get all the apps downloaded.  If some apps show errors, just wait till the others are all complete and try them again one at a time. If you are installing all apps to the SD card, you may see an error that a particular app needs to be in main memory, and it will offer to override for just that app. Typically that works. There are of course some apps that don't actually work on 10 that did on 8, so they won't install using the second method, and won't show up in the list on the first method. At the bottom of the My Library list, after the apps that do work in 'this device' is apps you have that won't work on 'this device'.
