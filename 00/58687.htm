Hi Andrew,

Windows Spotlight captures images from the Bing search engine. If you are offline the pictures may not change.

If you are online and the pictures do not change, we suggest that you restart Spotlight and check if it resolves the issue:

  1. Right-click on the desktop and select Personalize.
  2. Press the Lock Screen tab
  3. Under Background, switch from Windows Spotlight to either Picture or Slideshow.
  4. Log off and Log in.
  5. Go back to Lock Screen and select Windows Spotlight again.

If the issue persists, reset Windows Spotlight:

  1. Right-click on the desktop and select Personalize.
  2. Press the Lock Screen tab
  3. Under Background, select Windows Spotlight.
  4. Right-click on the Start menu, select Windows PowerShell (Admin).
  5. Enter the command below and press Enter to reset Windows Spotlight:

Get-AppxPackage -allusers *ContentDeliveryManager* | foreach {Add-AppxPackage “$($_.InstallLocation)\appxmanifest.xml” -DisableDevelopmentMode -register }

 6. Restart your PC.

Let us know if you need further assistance.

By default, the Photos app selects photos to appear on the live tile for you, using an 'interestingness' score that takes into account factors like how recently a photo was taken.  If you do decide to select a photo manually to appear on the live tile, you can always return to the default behavior by bringing out the Settings charm, going into Photos "Settings", and switching the "Shuffle photos on the app tile" toggle to "On".
