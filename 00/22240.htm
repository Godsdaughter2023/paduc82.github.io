Sub Macro9()

ossiemac say:

Hello Christine,

I have been watching this thread with interest because I made a start on a solution and then Bernard had already replied and I waited on your reaction to his reply. However, you might find the following formulas helpful. They do not take into account the actual start times of the workday and then move to the next day if after the finish times of the day and neither do they move the time to the start time of the day if earlier than the required start time. However, it does show how to use the Workday function and add the times to it because workday drops the time off the result.

Finding the next workday using the workday function requires the subtraction of one day and then use workday function to return the next workday (ie. one day for the number of days ahead). Make a list of the public holidays so that you can also use the holidays parameter. In my example I only inserted 2 days for holidays but you can expand that but ensure that you include the full list in the holidays parameter in the formula and the list must be absolute (ie. with the $ signs).

Formula for cell E2:    =WORKDAY($J$3-1,1,$H$2:$H$3)

Formula for cell E3:    =WORKDAY(INT( E2 + D2-1),1,$H$2:$H$3)+MOD( E2 + D2,1)

Formula in E3 gets copied down.

If you would like more assistance to expand the formula so that the times returned in column E will be between start and finish times then let me know the start and finish times and I will attempt to assist further.

Additional info after initial post: If you want the results to be between a start and finish time then J3 should include both date and start time and modify formula in E2 to:   =WORKDAY($J$3-1,1,$H$2:$H$3)+MOD(J3,1)
