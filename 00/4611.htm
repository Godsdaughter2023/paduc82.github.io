Enter below formula as an array formula (Ctrl+Shift+Enter) in cell G2 & copy down:

=IFERROR(INDEX($C$2:$C$100,SMALL(IF(($A$2:$A$100=$E$2)*($B$2:$B$100=$F$2),ROW($A$2:$A$100)-ROW($A$2)+1),ROW(1:1))),"")

This will return multiple matches from column C where E2 = column A and F2 = column B.

The formula
D18: =WORKDAY(C18,F18)
works as expected, the next workday after Tuesday is Wednesday, that is obvious correct.

D18: =WORKDAY(C18,F18-1)
and hide column D.

After that the purple squares shows the same amount as specified in column F.

Uninstalling Microsoft Security Essentials

Technical Level : Intermediate

Summary
This article collects a number of solutions for uninstalling Microsoft Security Essentials.

NOTE: If you are attempting to remove MSE after encountering 0x80073b01 and attempts to remove MSE are met with an error regarding permissions and contacting your administrator, your PC is infected and it has modified the file and folder structure preventing proper functioning or removal of Microsoft Security Essentials in order to reinstall it.

You should download Hitman Pro Kickstart on an uninfected computer, copy it to a USB flash drive and use it to repair MSE and remove the root kit virus, ZeroAccess/Sirefef. on an uninfected computer, copy it to a USB flash drive and use it to repair MSE and remove the root kit virus, ZeroAccess/Sirefef. Instructions and download here: http://www.surfright.nl/en/kickstart

Hi,

Let’s open Excel in application Safe Mode and check how it works.

In the Run open box type in ‘excel /safe’ and hit ok.
