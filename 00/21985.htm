smo# say:

I have struggled with this for so long and finally dug through the internet long enough until I came up with a solution.  This has worked for me, but no guarantees it will work for you.  I have Outlook 2013, so these steps may differ for other versions.

  1. Open a new email message
  2. Click Format Text in the ribbon
  3. Click Change Styles and choose Fonts --> Customize Fonts
  4. Set the font for heading and body to be the font used in your signature
  5. Enter a name and press Save
  6. Click the button to expand the Styles section, and then click the Manage Styles button 
  7. For Normal, press Modify next to the preview.
  8. Set the font size to be the same as used in your signature, and press OK.

jay freedman say:

The Track Changes Options dialog has a separate section for moves (which includes cut followed by paste) with its own set of color options. The behavior you described suggests that the Track Moves box is checked but that the Moved From and Moved To boxes are set to "(none)".

If that's the case, set the Moved From box to "Hidden", set the Moved To box to "Color Only", and choose a color for the Moved To (or select "By Author"). Then your moves will be indicated in the same way as ordinary insertions and deletions. Alternatively, you may want to choose underlining or strikethrough, and keep the Moved From and Moved To colors set to something different from the insertion and deletion colors, so they can be identified as moves.
