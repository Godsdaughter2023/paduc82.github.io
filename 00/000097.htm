julie sheets say:

Hi JP,

Nice to see you here :-)

Project Professional 2003 to Project Professional 2007 (outside server environment) had very few "big" changes.  Project 2007 has multiple level undo (yea!), the ability to create repeating calendar exceptions (for example, set every January 1 for the next 10 years as non-working), and visual reports -- creating pivot tables in Excel for much better time-phased reporting.  However, in my opinion, no real huge changes.

Project 2010 is a big change from earlier releases of Project.  Project 2010 offers a manual scheduling mode, where you can over-ride the scheduling engine and "nail" a task to a specific date -- disregarding links.  I think the manual scheduling mode lowers the entry point for users of project -- they can more easily grow into the tool as their project management processes improve.
