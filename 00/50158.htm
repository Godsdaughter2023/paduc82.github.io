In Windows 10:

Press Windows key + X

Click Windows PowerShell (admin)

Enter the following command:

wmic path SoftwareLicensingService get OA3xOriginalProductKey

system restore and system re-image - Microsoft Community

cw00 say:

system restore and system re-image

What are the difference between system restore and system re-image? Am I correct to say, the re-image gets the whole image of your Windows drive and copies to the Windows drive. The reason I ask is, system restore did not work for me with many tries; I kept on getting error messages, then system re-image as an 'advanced' recovery option worked. 

a. user say:

That's an excellent question. If only more people would ask.

System restore is a Windows terminology. Despite what it says, it doesn't actually restore your system. It restores something Windows calls a 'restore point.'

A restore point is a collection of key system files, drivers and registry keys as they existed on the date of the restore point. The original idea of system restore  - it's been around since Windows 7 - was to restore the Windows interface to a workable state so you can use Windows' own diagnostic and repair tools to fix whatever problem is affecting your computer. Hopefully.

System restore has very limited usefulness and is widely misunderstood. It won't necessarily restore a non-functioning computer to a functional state. It won't restore missing user files. It won't fix a computer that won't boot. Because restore points include default video drivers, restoring a restore point can make it seem like your computer has been brought back to life. But that's just a happy coincidence.

Most important, restore points go out of date quickly. If you try to restore a restore point from, say, 30 days ago, you risk destabilizing your computer with a potentially toxic mix of old and new system files. That's why all restore points are automatically erased when Windows gets a major update.

For these reasons, System Restore is on the way out. You might have noticed that it's not enabled by default in Windows 10. Nowadays we have much better tools to rescue computers in distress. One of the best of these tools is a system image.

Image and re-image are not Windows terms; they just mean special things in the context of computing. An 'image' is an exact copy of a hard drive. That's not 100% technically correct, but I'll come back to that soon.

Remember that computers only work with 0's and 1's. Everything on your computer - your documents, system configuration settings, internet access, everything - is just one huge collection of 0's and 1's. An image is a copy of all those 0's and 1's which is how it can be an exact copy of everything on your computer.

Here are the asterisks: 1) an image doesn't include the unused areas of your hard drive; and 2) an image is actually an exact copy of a partition. Since most folks have only one usable partition on their hard drive, we like to say that an image is a copy of your hard drive. But you can image individual partitions.

Finally, we come to a 're-image.' To re-image means to restore an image. When you restore an image, the hard drive (partition) looks exactly as it did at the time the image was made. It's the only form of time travel that really works.

So if your computer gets messed up - I mean really messed up, the way I do it - you can restore a known good image and in a few minutes, ta da, you're back where you were before like nothing happened. If you think this is a fabulous way to backup, you're right. Why bother with a restore point that may or may not fix your computer or may make it even worse when it's so easy to go back to yesterday, when love was such an easy game to play?

You have choices when you re-image a computer. You can restore an image that you yourself created. Or you can restore an image of a clean install of Windows 10 - what Microsoft calls a 'reset.' Or you can restore an image of your hard drive as it was on the day you first brought the computer home. That one is called a 'factory restore.'

a. user say:

Full disclosure: I do not use Windows' own system image utility. In my experience, third-party image software is more convenient and has more features, including the ability to easily schedule an image backup (which I do.) To schedule an image with Windows' own utility you need to create a command line and then setup the task manually with Windows' Scheduled Tasks utility.

If you want to go that route there's plenty of great image backup software, some of which is free. All the paid products have free trials. I'm a huge believer in backing up (as you might have guessed) so I don't mind paying for image software.

I currently use Macrium Reflect, which also has a free version. It has a great knowledgebase and a helpful user forum. You'll need both because imaging software is hard to understand until you get the hang of it. A web search will provide you with other great imaging programs.

May I suggest that once a week is not sufficient for backing up. I would recommend backing up daily, at least, and why not since once they're scheduled your backup jobs run automatically in the background, and you can continue to use your computer. Mine run overnight while I sleep.
