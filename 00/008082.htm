  • The best way I know is to modify the Table Style being used to include the Keep with Next Paragraph formatting attribute [applied to Whole Table]. NOTE: This can become problematic if the table grows to a size that will not fit on a single page -- You'd be asking Word to do something that cannot be done. If that gets to be the case you'd need to make further adjustments. It can also give unexpected results if Text Wrapping is applied to the table, so keep the default setting of None rather than Around (don't drag the table).
  • Another option is to apply Page Break Before Paragraph formatting to the first cell of the table. The main concern with that is that the table will always begin atop a new page even if there is room for it on the preceding page. Also, it makes it impossible to use the Caption> Above selected item feature.
