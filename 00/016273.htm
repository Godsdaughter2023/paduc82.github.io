a. user say:

I'm not completely sure what you are trying to do here, so I'm going to spell out what I think you're trying to do, and you can correct me.

Let's say you have base doc B (let's say it is called c:\mydocs\mydoc.docx) with a form, and document D. You have a field in B called myfield. What you want is that when myfield is empty (i.e. has the 5 standard spaces), you get [1]. When myfield has content, you get that content.

If you want the content in document B, you need something like

{ IF "{ REF myfield }" = "     " "[1]" "{ REF myfield }" }

If you need the content in Document D, at first sight it looks as if you need this:

{ IF "{ INCLUDETEXT "c:\\mydocs\\mydoc.docx" myfield }" = "     " "[1]" "{ INCLUDETEXT "c:\\mydocs\\mydoc.docx" myfield }" }

where "    " contains five spaces.

However, that doesn't quite work because what Word receives is actually a copy of the entire form field, and that doesn't compare well with "     ". But I think the workaround for that is to do the following:

In document B:
 - make sure myfield is set to "calculate on exit"
 - insert the following SET field somewhere
{ SET myfieldX { REF myfield } }

In document D:
 use
{ IF "{ INCLUDETEXT "c:\\mydocs\\mydoc.docx" myfieldX }" = "     " "[1]" "{ INCLUDETEXT "c:\\mydocs\\mydoc.docx" myfieldX }" }

Calculated field won't calculate - Microsoft Community

sandykayz say:

Calculated field won't calculate

I created a calculated column in Access 2010. All I want to do is add some currency columns together. For the expression, I typed in [Jan13] + [Feb13] + [Mar13] ... and so on. Theoretically it should calculate the amounts in the column right? When I multiply the columns, it's fine. So how do you do a sum? What am I doing wrong? Thank you.

hansv mvp say:

I'd calculate the sum in a query, not in a calculated field.

You can then use

 

Total: Nz([Jan13],0)+Nz([Feb13],0)+Nz([Mar13],0)+...

 

The Nz function ensures that the sum is calculated even if some of the contributing fields are blank. This function can be used in queries, but not in calculated fields.

sandykayz say: Thank you. I just had to use an & instead of a plus sign.

hansv mvp say:
& doesn't add numbers but concatenate strings...

qa guy commelec say:
I just had to use an & instead of a plus sign.

That will not result in the mathematical sum but give you concatenation of the fields.

sandykayz say: I see. How do I insert your Nz string into a query? I know how to add a Total line at the bottom of the query but I want to add columns from left to right. Thank you.

hansv mvp say:

Open the query in design view.

Click in the Field: row in the first empty column.

Enter the expression

 

Total: Nz([Jan13],0)+Nz([Feb13],0)+Nz([Mar13],0)+...

 

You have to write it out fully, of course - don't keep the ...

Make sure that you use the correct field names.

Press Enter when the expression is complete.

How to file actionable feedback to the Windows Camera Team - Microsoft Community

tom a - windows virtual desktop team [msft] say:

How to file actionable feedback to the Windows Camera Team

Hello Insiders,

Tom from the Windows Camera Team here. I just wanted to send out a message to the Insider community to give some instructions on how you can file actionable feedback such that the Windows Camera team can investigate problems you are seeing with your camera on latest Insider builds of Windows. We process feedback on a daily basis and one of the biggest problems we face is feedback isn't actionable. Often cases, feedback is vague ("camera doesn't work") and Insiders are using devices we don't have access too, which makes investigation even more difficult. The best way for you to get your issue investigated is to include three key ingredients: any error message you are facing, screenshot(s), and please use the "monitoring" option so we can get logs from your device.

How to give actionable camera feedback:

The camera team wants to be able to take action on your feedback, in order to do so we need you to provide us with the following additional information.

1. Launch the Feedback Hub, type your issue in search box, search, if you found similar problem upvote and add to the existing issue, or file a new issue if a matching issue isn't found.

2. Mark the appropriate feedback type at the top with suggestion or problem radio buttons
3. Select the appropriate category
-      For camera device feedback [For example: Issues like camera doesn't start] select "Hardware, Devices, and Drivers" category and "Device Camera or Webcams" subcategory
-      For camera app issues [For example: Living images doesn't work] select "Apps and Games" category and "Camera" subcategory
-      If you are unsure, select "Hardware, Devices, and Drivers" category and "Device Camera or Webcams" subcategory
4. If issue is a problem (For example you get error message starting camera) provide us with the following additional information:
-      If there's any error, include error message in the title and text of your feedback
-      Include a screenshot of the problem, especially if there's an error
-      Use the "Begin monitoring" option to reproduce the problem so we have logs to investigate your issue
-      Once you click the "Begin monitoring option", be sure the "Device Camera or Webcams (Default)" option is selected from the drop down.

-      Now, reproduce the problem, after return to the Feedback Hub and click "Stop monitoring"

-      Other information that is really helpful is when the problem started occurring, if you just updated to build X, and your camera was working previously, let us know. Also include what app(s) you are experiencing problems with.

One last thing, if you search and find an issue that looks to match yours, you can add to this feedback by clicking on it, and selecting "Add feedback details." From there you can complete the steps mentioned above. Also, if you find existing feedback, be sure to read the Microsoft Engineer response if one's available, and add your own comments. 

Thanks!

Tom

I'm certain Kent_C's problem is that his boss has used Smart Art.  We use both 2003 and 2007 at my office and have the same problem all the time.  If you're using 2003 with the compatibility pack you can view PPTX files but you still can't edit Smart Art.  If you're using 2007 you can still edit Smart Art in a PPT file.  So the bottom line is that you can only edit Smart Art in versions 2007 & 2010.
