dave oddy say:

My laptop won't connect to my wireless network because it says there are 2 devices with the same IP address, How do I fix this?

I have a PC and 2 laptops, a smartphone along with a wireless bridge and WiFi blue ray player.  My wife's laptop often says it can't connect due to two devices with the same IP address.  I have a Motorola Surfboard modem and a linksys G wireless router.  Is there something wrong with my router?  I also have to reset my modem and router several times a day to connect to the internet on anything.
foxfifth say: It is possible that the ip address conflict is a result of your resetting the router.  If a computer is connected to the router and then goes to sleep or hibernate mode and then the router is reset, when the router restarts it has no way of knowing that the sleeping computer is there and that it already has an ip address that was assigned before the router restart.  When the computer returns to normal mode from sleep mode it may then find that the router has assigned that ip address to another device and there is then a conflict.  That is a normal problem and can be avoided by shutting everything down or waking everything before the router reset.  It isn't normal however to need to reset your modem and router several times a time; I have a Motorola modem and Linksys router and normally don't reset either more than every 2 or 3 months and usually don't need to do it even then.  Unfortunately I can't help on the question of why you need to reset yours so frequently.

m peter ward say:

outlook express updates

Like many others I have lost my old version of Outlook Express with my address book and folders including the emails. It loos like an automatic update has done this but I have tried system restore with no change. I am on Windows XP Home edition. Please help
steve cochran say:

There have been no updates to OE since 2006, so an update did not affect the message store.  Usually antivirus helps wipe out OE's messages, but it can happen spontaneously. 

 

Sometimes OE just resets the location of the message store, so perform an Advanced search for *.dbx and in the Advanced Search options include hidden files and folders.  Make sure first you go to Control Panel | Folder Options | View and uncheck the option to hide extensions of known file types and enable showing hidden files and folders.  See also points 2 and 4 here:  www.oehelp.com/OETips.aspx#4

 

For the address book, search for *.wab and if you don't find that, search for *~* and the address book backup has a ~ in the filename.

 

See if that helps.
