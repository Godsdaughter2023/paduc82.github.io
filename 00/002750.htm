1. To make sure your Office 365 plan has RBAC customization, see Exchange Online Service Description:

https://technet.microsoft.com/en-us/library/exchange-online-service-description.aspx?tduid=(b99b3a06f26a067d425fe307b9c01d8e)(256380)(2459594)(TnL5HPStwNw-9a6hl63sPX8497BYygBo6w)()

3. Create a Management Scope:

a. Connect to Exchange Online using remote PowerShell:

https://technet.microsoft.com/en-us/library/jj984289(v=exchg.160).aspx

b. Run the cmdlets below one by one:

Get-RoleGroup

Get-ManagementScope

Enable-OrganizationCustomization

New-ManagementScope “HelpDesk for New York Users” -RecipientRestrictionFilter {Department -eq “New York”}

4. Create a Role Group.

a. Login to EAC (Exchange admin center): https://outlook.office365.com/ecp/

b. Click permissions->admin roles.

c. Click Add (+), type a name.

d. Select “Helpdesk for New York Users” under Write scope section, add the roles you need under Roles box and then add the admin to the Members box.

e. Click Save.

If this doesn’t work for you, there is no other Microsoft method to meet your requirement.

If you add up the numbers in the parentheses in this cell you would get .5.  I would like to be able to automatically add the numbers despite the text between and have the total appear in a separate cell (or at the end of the text in the cell)  Any help would be much appreciated.

jeeped say:

With an unknown number of bracketed numbers to total, a User Defined Function that could use a VBA loop would seem to be the best solution.

Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module (e.g. Alt+I, M). Paste the following into the new pane titles something like Book1 - Module1 (Code).

Public Function udf_Sum_from_Text(c As Range) As Double
    Dim p As Long, ttl As Double
    p = InStr(1, c.Value, Chr(40))
    Do While CBool(p)
        udf_Sum_from_Text = _
          udf_Sum_from_Text + CDbl(Mid(c.Value, p + 1, InStr(p, c.Value, Chr(41)) - (p + 1)))
        p = InStr(p + 1, c.Value, Chr(40))
    Loop
End Function

Tap Alt+Q to return to your worksheet. Use like any other formula.

=udf_Sum_from_Text(<cell with text>)

vansharma say:

Hi,

 

I don't think it can be service provider related, your wireless carrier has nothing to do with your Wi-Fi settings.

 

Go to Menu > Connectivity > WLAN Wizard when at work. The phone will check for networks. Select your office network and click on Options > Details. Under Security make sure it says 'Open'. if it doesn't, it means there's a pre-shared key or other key required. Go back and then click on 'Edit Access Point'. Make sure that Network Status shows 'Public' and WLAN Network Mode is 'Infrastructure'. Then set WLAN Security Mode to 'Open'. Exit everything and now try. Probably (I'm guessing here) the default security mode is set to WPA/WEP and so the phone is expecting a key.

 

Let me know if this works for you. If it doesn't, check the above settings on another Nokia phone that IS able to connect to your office network and surf the web, and see if you can replicate those settings on your phone.

 

Good luck!

jay freedman say:

Sub x()
    Dim source As Document, target As Document
    Dim i As Long
    Set source = ActiveDocument
    Set target = Documents.Add
    With source
        For i = 1 To .Comments.Count
            target.Range.InsertAfter _
                "[" & .Comments(i).Scope.Text & "]  [" & _
                .Comments(i).Initial & "][" & _
                .Comments(i).Range.Text & "]" & vbCr
        Next i
    End With
End Sub
