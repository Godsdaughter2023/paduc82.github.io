doug robbins - mvp office apps & services (word) say:

When you open a document, a reference is created to the template from which that document was created.  If a password is required to open the template, the creation of the reference to the template will cause a request for that password to appear.  If the wrong password is given or you cancel the request, the reference to the template will not be created and instead, a reference to the Normal.dotm template will be created.

To encrypt the documents created from the template, it is necessary to do that when the document is saved.  You could use some code in the template to do that.  Probably having it in an AutoNew macro would be the best.  Have the code ask for the filename to be used for saving the document and then save it with that name, applying the desired password as part of the save process.

kergga say:

I would start with

http://www.cpearson.com/excel/stringformulas.aspx

Position Of First Non-Digit In A String

This formula will return the position of the first non-numeric character in the string in cell A1.

=IF(LEN(A1)=0,0,MIN(IF(1*ISNUMBER(1*MID(A1,ROW(INDIRECT("A1:A"&LEN(A1))),1))=0,ROW(INDIRECT("A1:A"&LEN(A1))),LEN(A1)+1)))*(ISNUMBER(A1)=FALSE)

This is an array formula so you must enter it with CTRL SHIFT ENTER rather than just ENTER.

then incorporate that in something like (aircode)

=left(C1,IF(LEN(C1)=0,0,MIN(IF(1*ISNUMBER(1*MID(C1,ROW(INDIRECT("C1:C"&LEN(C1))),1))=0,ROW(INDIRECT("C1:C"&LEN(C1))),LEN(C1)+1)))*(ISNUMBER(C1)=FALSE)-1)

so pull everything off the left side minus one char so you don't include the non-numeric, then wrap that in case the whole string is numeric:

iferror([formula above],C1)

or in full,

=iferror(left(C1,IF(LEN(C1)=0,0,MIN(IF(1*ISNUMBER(1*MID(C1,ROW(INDIRECT("C1:C"&LEN(C1))),1))=0,ROW(INDIRECT("C1:C"&LEN(C1))),LEN(C1)+1)))*(ISNUMBER(C1)=FALSE)-1),C1)

again, still an array formula so Ctrl-shift-enter
