But now files that I have created using the 2016 version are changing the text after I re-open them (after restarting the computer).

Most of the time the text is increasing in size, which puts everything out of proportion.

Any thoughts? advice? 

john korchok say:

PowerPoint placeholders are set to automatically adjust text size by default. To prevent this:

  1. With your presentation open in PowerPoint, choose View>Slide Master.
  2. In the left-hand list of slide layouts, scroll to the top and select the larger slide, the Slide Master.
  3. Right-click on the main text placeholder and choose Format Shape... The Format Shape pane opens at the side of your window.
  4. Select Shape Options and click on the square icon with arrows inside it.
  5. In the Text Box section, select the Do Not Autofit choice.
  6. Close Slide Master view and test whether text still changes size. In a normal presentation, setting the slide master placeholder propagates the setting to the placeholders on the slide layouts. If the layouts do not automatically acquire the same setting, you may have to apply the same Do not Autofit settings to the placeholders in those layouts as well.

robert aldwinckle on forums say:

wondering why they wont shutdown the processes completely when I close tabs. 


Because there is something interfering, perhaps an add-on or other started program.   

Try opening an IE window this way.   Run...  (e.g. press Win-R and enter:)   

iexplore.exe  -nohome  -extoff

 

Use Task Manager's Applications tab to find that instance and right-click Go to Process.   If you like, you can confirm that it is the one you started using the Command line field.   Then close the window.   The highlighted instance should go away.

 

If that is not enough to help identify a cause try doing the same test in a safe mode boot.

 

 

HTH

 
Robert Aldwinckle
