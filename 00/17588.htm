VBA in Access Help with changing the value of Checkbox in a field

Ok so Ive got 2 tables
1 - Member
2 - MembershipPurchases

Member contains a checkbox whether they are a Member or not, column is also called "Member"

MembershipPurchases Contains
The Username of the Member - "Username"
The date the membership was purchased - "Purchase Date"
And the date the membership is valid until - "ValidUntil"

I want to be able to change the checkbox of the field Member to False/0 when the field ValidUntil has been reached
I cant seem to get it to work at all using the basic SQL knowledge I have, any help guys?
ken sheridan say: Hans raises an important point.  If MembershipPurchases includes multiple membership renewals, then you'd need to execute the following:

UPDATE Member
SET Member = FALSE
WHERE NOT EXISTS
    (SELECT *
     FROM MembershipPurchases
     WHERE MembershipPurchases.UserName = Member.UserName
     AND ValidUntil > = DATE());
