I want to make this laptop dual boot with Ubuntu OS and Microsoft Windows 10 Home. Although I know how to make a system dual boot with an already installed Windows OS not with preinstalled Ubuntu OS.

Kindly help.


andre da costa say:

You will need to download GParted, create a bootable copy, boot from it then use it to partition the disk and format the partition as NTFS.

GParted -- Live CD/USB/PXE/HD

You might need a utility such as Unetbootin to make it bootable in Ubuntu:

UNetbootin - Linux Downloads

See instructions how to create a partition using Gparted:

Partition Your Hard Drive The Easy Way With GParted - Lifewire

After you setup the partition in Ubuntu, do the following:

Purchase a license for Windows 10:

Buy Windows 10 Home/Pro

Next, you will need to download the Windows 10 ISO for the edition you bought:

https://www.microsoft.com/en-us/software-download/windows10ISO 

You will need to create a bootable copy of Windows 10 using UNetbootin:

Make A Bootable Windows 10 USB Install Stick On Linux With ...

Next, you need to download and copy the EasyBCD executable to the USB thumbdrive, since this will be important for creating a boot menu entry for Ubuntu Linux after installing Windows 10.

EasyBCD - NeoSmart Technologies

Configure the BIOS to boot from USB thumb drive you created:

BIOS/UEFI Setup Guide: Boot from a CD, DVD, USB Drive or SD Card

Choose the partition you had setup earlier using Gparted.

Format it as NTFS by clicking the Format button

Setup might create additional partitions.

Make sure you choose the Primary partition, then click Next to continue with the installation.

After the installation is complete and you arrive at the Windows 10 desktop, proceed to install EasyBCD from the USB thumb drive, then add a boot entry for Ubuntu Linux.

EasyBCD will automatically detect your installed Linux distribution do the rest.
