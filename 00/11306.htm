nuculerman say:

Right click on the worksheet in question and choose "view code."  Copy and paste the following.

Private Sub Worksheet_Change(ByVal Target As Range)

If Target.Address = "$C$2" Then

Rows(7).Hidden = True

If Target.Value <> "DEF" Then Rows(7).Hidden = False

End If

End Sub

ron coderre say: VBA is Visual Basic for Applications. It is a programming language for Microsoft Office.
The real point I want to make is this:
A cell can contain a formula or a value...but not both. At least, not both the way you want it.

You would not be able to enter a formula...like:
A1: =Proper(A1)
Then be able to type text into the cell, replacing the formula, and still have the formula.

Consequently, if you wanted to:
...Type text into a cell
...Press ENTER
...Then have the cell contents become propre case formatted
You'd need to write a small program, triggered by the Worksheet_Change event
that would reformat the text that was just entered.

There is no other way to do what you described you want.

The closest thing would be to enter your text INTO the formula:
A1: =PROPER("my name")
which would display as "My Name"....without the quotes, of course.
