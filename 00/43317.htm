DXDiage.exe and WEI reports in an incorrect DirectX version when - Microsoft Community

bobbymi say:

DXDiage.exe and WEI reports in an incorrect DirectX version when testing DirectX 11 hardware on Windows 7-based computers.

Hello,

 

When I use the DXDiag.exe to check the version DirectX version of my hardware, the version reported is not correct. I then tried Windows Experience Index and it also reports the incorrect version.  The manufacturer of the hardware says that the hardware is DirectX 11 but DXDiag.exe and WEI report DirectX 10.1. Is this a know issue? How do I fix this?

 

Thanks

 

Christi

 

 

bobbymi say:

Found the answer!

 

When you run the DirectX Diagnostics Tool (Dxdiag.exe) on a computer that is running Windows 7 or Windows Server 2008 R2, the diagnostic tool incorrectly reports a DirectX 11 device as a DirectX 10.1 device. This incorrect version information is shown in the DDI field on the Display tab. This problem is only a reporting error and this does not affect the DirectX 11 hardware or software functionality of the graphics card.

 

There is a fix for this issue, for more information on the fix see the following article on the Microsoft web site:

 975243 The DirectX Diagnostics Tool incorrectly reports DirectX 11 devices as DirectX 10.1 devices on a computer that is running Windows 7 or Windows Server 2008 R2

 

Windows Experience Index also has the same issue and Microsoft is aware of this.
