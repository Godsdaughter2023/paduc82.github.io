After 3 days I found a solution, I share it for people having the same trouble.
I think the problem it was the plug and play, so the device was not recognized as a Windows Phone.

So:
1. Install Zune
2. Connect your Windows Phone
3. Zune don't recognize yhe device
4. Go to Device Manger (Control Panel->Device manager)
5. Probably you will see your device on top under "Other device" with a yellow warning
6. Right Click on device and click Update Driver
7. Select manual installation (the 2nd option) and find drver from a list
8. Chose "Portable Devices"
9. Microsoft as manufacturer and Windows Phone 7 driver on the right box
10. Install this driver and the phone will be recognized
11. The device should now be visible under Portable Devices and Zune recognize it, if not try to disconnect and riconnect (may be I rebooted Windows but I don't remember :))

Install a dummy Windows Phone 7 device (right click on device manager and install Legacy hardware)
Then upgrade the drive of this dummy device, and manullay update the driver of your phone to last version.

bob jones aka: cybertaz say:

Additional to John's suggestion, usually the Shift+F5 (which may need to include the fn key if not using a full size keyboard) or Command+Option+Z keyboard shortcut will return you to the location of the last edit, even when you first reopen a document. In fact, either of those keystroke should toggle through the 5 most recent spots where editing has been done.

If you simply need to go to the very end of the existing content, use the Command+End keyboard shortcut.

Other features such as the Sidebar, Find & GoTo may come in handy as well, and for locations to which you may need to return frequently you can set Bookmarks.
