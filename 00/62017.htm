josh83879 say:

I ran into a similar issue today.  My laptop needed a fan replaced, and was shipped off for warranty repairs.  I made sure I had a current backup using File History, and then cleaned my sensitive data off the machine.  I received it back today, and set about to restore my data.  The same thing happened to me, as to trebuin: Windows had no File History at all.

I did some quick research and found that the computer had updated while it was in for repairs.  The file history log must have been replaced or lost in the process.  I tried to roll-back the update, that failed.  Fortunately, the file history log that I needed (pre-update) was stored in Windows.old.

Here is a quick rundown for anyone that finds themselves in this situation again, with Windows 10:

1.  Go to C:\Windows.old\WINDOWS\System32\winevt\Logs

2.  Find the file: Microsoft-Windows-FileHistory-Engine%4BackupLog

3.  Confirm that the time and date stamp matches your most recent backup

4.  Copy this file to the desktop

5.  Ensure you have your microsoft account password handy/memorized.  This is not your pin code; it is the password you use to login to your microsoft account online.  WIthout it, you will be stuck following the next step.  Safe Mode only allows you to  use this password.

6.  Reboot into Safe Mode (CTRL-R and type msconfig into window; select run; navigate to "boot" tab and select "Safe Mode")

7.  Once in Safe Mode, navigate to C:\Windows\System32\winevt\Logs

8.  Copy your logfile from the desktop into C:\Windows\System32\winevt\Logs.  It will ask you if you want to replace the file...yes, you do.

9.  Disable Safe Mode Boot (CTRL-R and type msconfig into window; select run; navigate to "boot" tab and de-select "Safe Mode")

10.  Windows Backup should now recognize your lost backup files.  At least, this is what I did and it worked for me.

I know this post is old but I could not find a comprehensive solution to this anywhere on the web.  I'm sure that others will run into it at some point; hopefully this helps someone.
