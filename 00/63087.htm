Clean up printers

NOTE:  If after completing step A the print spooler still does not stay running after you have issued the "net start spooler" command, you will not be able to follow all of the steps in "Article One."  Use the Special Procedure for Lexmark printers described below and then follow Article One.  If still no joy, use the alternative method linked below.


A.  Clean out the pending print jobs

  • Open a Command Prompt window (Start > Run > cmd > OK)
  • Type the following in the black Command Prompt window and press Enter after each line

                           net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

Open a Command Prompt window (Start > Run > cmd > OK)

      sc config spooler depend= RPCSS
      exit

note that there is no space before the =

bill smithers say:

That depends if you have a system restore point older than 60 days.

Read this information about how it works and what you lose which is bolded:

It all depends on the amount of space allocated for System Restore.

New restore points are being created automatically/regularly and as that happens older ones are deleted to fit that space allocation.

If you find that you need earlier restore points as they are no longer there you will have to alter that space allocation to allow that in the future.

You can't alter the restore points that are available presently. What's there is there, what's gone are gone.

But remember that you use too much hard drive space if you wish to keep restore points back to the year "dot" for your machine and which you would have to keep increasing to accommodate them and it is not practical to do that as you would lose installed programs/app, Windows updates, computer settings, etc. when you do a SR onwards from that earlier restore point.
