el_loco---01 say: Hi Liam, while it is true that formating the HDD with Windows is different than the format in the device that doesn't mean it will break anything. With TWO big exceptions:

1. VERY IMPORTANT. The N91 DOES NOT support NTFS file format. This is the normal file format for Windows XP. IT supports FAT32 so you have to formal in FAT32 and not NTFS. IF you format in NTFS the HDD will cease to work PERIOD! So be sure to format in FAT32.

2. Also I found out is that when you format using the N91, the HDD gets folders created, but if you use Windows no folders are created.

Finally, it should be noted that HDD get fragmented over time and can get some errors so using "Error checking" and "Defragmentation" from Windows (selecting properties then tools tab). So these are useful tools to improve the performance of the HDD.
