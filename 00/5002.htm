Extracting The Time Only from a DateTime Field in Access - Microsoft Community

la lawyer say:

Extracting The Time Only from a DateTime Field in Access

I need to determine whether a DateTime variable has a time (hour and minute) and, if so, convert that portion of the variable to text.  

jeeped say:
In Access (and most any Office application) the Time portion of a DateTime is the decimal portion of a mixed number with the days being 1 for every day past Dec 31, 1899. With that in mind, you can test to see if the DateTime equals the Date, and if not, return the time portion. something like,

SELECT iif(int(tm)=tm,"",format((tm -int(tm)),"hh:mm"))
FROM Table1;

Extracting a Word page - Microsoft Community

billgreer_968 say:

Extracting a Word page

How on earth do I extract one page from a 50 page Word doc and then open it in a completely new file? Alternatively, how do I delete the pages around the page I want to work on? Any advice appreciated. Thanks in advance
bob jones aka: cybertaz say:
Objects are attached to the text in one of 2 ways:

  • In Line with Text, in which case the object is copied/cut along with any other content in the same paragraph as long as it is included in the selection, or
  • Floating -- Having some form of Text Wrap applied but still anchored to the 1st line of a paragraph. A floating object can be deceptive & may shift unpredictably when copied/cut & pasted. It's also entirely possible that the object not be positioned 'in' the paragraph to which it is anchored. There are a number of formatting variables involved, but if the paragraph to which the object is anchored is not included in the copy/cut, the object will not be included, either. The anchor may need to be repositioned.

Perhaps I'm wrong, but I get the impression that you may be intending to take content out, then revise it & put it back in... If I'm guessing correctly I'd seriously advise that you not do so. Edit the content in place within the document.
