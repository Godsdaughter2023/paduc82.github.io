I am not aware of any formula to reliably test if the cell contains just any date. Even my testing with a VBA solution and the VBA Function IsDate is not reliable because it can see a number as a date.

ossiemac say:

I'll give you a lesson in entering Conditional formats.

When you enter a formula for Conditional format, the formula is entered so it references the first cell of the range to have the format. The remainder of the range will format on a relative basis.

Just so that you do not get into bother with multiple conditional formats, firstly clear any Conditional format from the range first as follows.

  • Select the range where the conditional format is to be applied.
  • Select Conditional formatting -> Clear rules -> Clear rues from selected cells.

Now we can start from scratch to apply the Conditional format

  • Select the range where the conditional format is to be applied.
  • Select Conditional formatting
  • Select New rule
  • Select "Use a formula to determine which cells to format"
  • Enter the following formula in the field under where it says "Format values where this formula is true"

          =AND(E2>=DATE(2017,1,1),E2<=DATE(2017,12,31))

  • While the cursor is in the Formula Field, Press F2 to enter Edit mode (otherwise if you use the arrow keys to move through the formula, Excel thinks you are moving on the worksheet and selecting cells to insert in the formula).
  • Edit E2 in the formula (Two instances) to the first cell of the selected range.
  • Click on Format button and set the required format. (Depending on how you want to format, you might need to select the required tab name at the top of the dialog box.)
  • Set the required format.
  • Click OK and OK again to exit the Conditional formatting dialog.

karl timmermans [outlook mvp 2012-2018] say:
What you are describing is the protocol (POP3, Imap, Outlook.com etc) used to configure each email account which has nothing to do with the version of Outlook being used.

In the '2003 scenario, you would had the two email accounts configured as POP3 accounts both of which would have pointed to the same <Inbox> for mail delivery. That would result in two email accounts being configured but a single set of folders appearing in the Outlook Navigation Pane. The ability to have multiple email accounts is only available if the accounts are configured using POP3.

To accomplish the same thing in OL;2013, you need to "manually" configure each account instead of using auto-configure the account by entering the email address/password for the account. For the Shaw account, auto-configure would create the account using IMAP by default and the Hotmail (now Outlook.com) would be configured using the Exchange protocol.

Background info on email account protocols, in case it's of any interest, can be found in the following article

Understanding Email Protocols (POP, Imap, MAPI, EAS, Exchange)
https://www.contactgenie.info/understanding-ema...

To check the protocols used in OL'2013 - from OL menu --> File --> Account Settings --> Email tab

Now to address your question as to how to combine both accounts into a single folder set, you'll need to do the following

#1 - With Outlook closed, create a new Outlook profile by using the Control Panel <Mail> app (don't delete the original profile at this point in case you need to get back to it)

How to create profile and set up an e-mail account in Outlook
http://support.microsoft.com/kb/829918

#2 - You will need to get the POP3 settings for the Shaw account

Setup Email in Windows 10 Mail
https://community.shaw.ca/docs/DOC-5375

*** the above article title references Win 10 Mail but the settings are same for Outlook (any version

#3 - You will need to "manually" configure your Hotmail (Outlook.com) account using POP3

A Guide to Access Outlook.com via POP in An Email Program
https://www.lifewire.com/outlook-pop-access-117...

When you're done configuring the email accounts, be sure to set the new Outlook profile that you've just created as the default for the next time Outlook is opened otherwise nothing will change.
