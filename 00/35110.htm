lz. say:

Dorith,

 

Regarding the system health check I suggested earlier:

  • [Win] + X / Command Prompt (Admin)
  • At message "Do you want the following program to make changes to your computer" click OK
  • Type CHKDSK /F /R and press [Enter]
    a) PC reboot will be required). This will check the integrity of your system partition and repair - where possible - the errors. The /R switch will also locates bad sectors if any and recovers readable information.
    b) For a first check of your system you might want to ignore the /R switch that requires more time for the command to execute. Launching CHKDSK /F /R is typically something you do when you have no other choice (system constantly reporting disk errors) or when you want to make a serious check/repair of your hard drive because you suspect there's something going wrong
  • Type SFC /ScanNow and press [Enter]
    a) No PC reboot required. This will check the integrity of the system and try to repair issues if any. b) When it has finished 3 cases are possible: No issue :-); Some issues but all repaired :-); Some issues but not all repaired :-(. If you experience case #3 no panic. Sometimes you will need to run SFC /ScanNow 2 to 3 times before it can fix everything. If after 3 times there are still unrepaired issues... :-((( Let's be optimistic.....

Hope this helps

Cheers
