Of course the problem is not your lack of understanding of average.  Since in your original example, Excel is displaying 4 for the average of 0, 7 and 0, we know the problem is something else since 2 1/3 would never appear to be 4.  Let's slow down and pay attention to details.

First, if you see 4 for the average of 0, 7 and 0, part of the problem might be the cell format.  Be mindful of that when you look at the result.  You might want to change the cell format to General or to Number with some larger number of decimal places.

Second, if you see 4 for the average of 0, 7 and 0, one of the zeros might be text "0", not numeric 0, especially if you are using a formula like =AVERAGE(A1:A3).  Note that the average of 7 and 0 (just 2 numbers) is 3.5, which can indeed appear to be 4 with the "wrong" format.

Similarly, the average of 0, 0 and 1 might appear to be 0 either because of the cell format (the actual average is 1/3, which can round to zero) or because the one is text "1" instead of numeric 1.

Am using an iPad Pro and the way to get to the Outlook settings options to turn off the <Focused Inbox> is to

#1 - Touch the account avatar on the top left beside <Inbox>
#2 - That brings up the folder list
#3 - At the bottom of the folder list is the <Gear> icon for Settings - select that
#4 - The <Focused> option can be set on or off - if you don't see it immediately, scroll thru the Settings until you find it

*** doing the same thing on an iPad Mini - click on the 3 lines besides <Inbox> to get to the folder list and Settings icon as described above as opposed to looking for an avatar
