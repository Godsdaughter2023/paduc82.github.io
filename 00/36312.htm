To remove the license and the key all together type:

slmgr.vbs -upk(Product Key)

To reinstall the license and the key all together type:

slmgr.vbs -ipk(Product Key)

d. In the Administrator: Command Prompt window, type the following commands. Press the

e. Enter key after each command:

DISM.exe /Online /Cleanup-image /Scanhealth

DISM.exe /Online /Cleanup-image /Restorehealth

f. To close the Administrator: Command prompt window, type Exit, and then press Enter.
