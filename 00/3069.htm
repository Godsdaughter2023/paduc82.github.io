    If cleanup is recommended in the result, namely there prompts: "Component Store Cleanup Recommended : Yes", then could run the following command to clean up the WinSxS folder:

    For getting the same results, also could execute the following command in the command line:
    schtasks /Run /TN "\Microsoft\Windows\Servicing\StartComponentCleanup"
