Saving is not your problem.  "Opening" is your problem.  If you can get them open, you can save them, no problem.

First: Ensure that each file ends with the appropriate file extension.  The file extension is a label that is critical to tell Mac OS X what kind of code a file contains, and thus which application to send it to.  Chances are your file names should be changed to end in ".doc". 
  1. Start Word
  2. On the File menu choose Open
  3. Set Enable to "All Files"
  4. Navigate to the document.

c)  Find someone with a REALLY old PC.  PC Word still had the converters for those old file types until around Word 2000...

6.  On the same Data Files tab click the new PST data file and click Set as Default

7.  Restart Outlook
