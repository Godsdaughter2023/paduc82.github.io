It depends, if you have a full retail copy Windows 10 license, yes, you can transfer it to a new motherboard.

If you upgraded from Windows 7 or 8 to the free upgrade, yes, you can.

If Windows 10 or Windows 7 or 8 came preinstalled with the previous motherboard, then the answer is no, you will need to purchase a new license.

When I upgrade a preinstalled (OEM) or retail version of Windows 7 or Windows 8/8.1 license to Windows 10, does that license remain OEM or become a retail license?

If you upgrade from a OEM or retail version of Windows 7 or Windows 8/8.1 to the free Windows 10 upgrade this summer, the license is consumed into it. Because the free upgrade is derived from the base qualifying license, Windows 10 will carry that licensing too.

If you upgrade from a retail version, it carries the rights of a retail version.

If you upgrade from a OEM version, it carries the rights of a OEM version.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM :

OEM versions of Windows are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

What happens if I change my motherboard?

As it pertains to the OEM license this will invalidate the Windows 10 upgrade license because it will no longer have a previous base qualifying license which is required for the free upgrade. You will then have to purchase a full retail Windows 10 license. If the base qualifying license (Windows 7 or Windows 8.1) was a full retail version, then yes, you can transfer it.

melodie carlisle say:

my computer keeps running out of virtual memory

rohn007 say:
Lets make sure we are talking about the same thing.

Exactly how do you know you are running out of "Virtual Memory" specifically?

Virtual Memory is actually a "Swap file" that is specially allocated space on the HD. Most installations have it set to automatically expand and shrink as needed. So freeing up space on your HD and defragging is reasonable.

How much used and unused space do you have on the HD?

You should also check how much virtual memory allocated. 

  1. Start the "System" dialog using the <WIN><Print Screen> shortcut
  2. Make note of how much "Installed Memory (RAM):" you have.

  3. Tap or click Advanced system settings.  You might be asked for an admin password or to confirm your choice.

  4. On the Advanced tab, under Performance, tap or click Settings.

  5. Tap or click the Advanced tab, and then, under Virtual memory, tap or click Change.

  6. Clear the Automatically manage paging file size for all drives check box.

  7. Under Drive [Volume Label], tap or click the drive that contains the paging file you want to change.

  8. Tap or click Custom size, enter a new size in megabytes in the Initial size (MB) or Maximum size (MB) box, tap or click Set, and then tap or click OK.


In general, recommendations are that you should have 1-3 times your RAM allocated to virtual memory.   Personally, if I have enough free space on the HD, I prefer to set the swap file max and min size the same. 

Originally, back in the 80's and early 90's HD space was at a premium and so was RAM (I paid $5000 for my first computer with 2MB of RAM and 80MB of HD space!). It made sense to let the system allocate more swap file space as needed, then free it up. But, that approach also has the effect of fragmenting the HD.  Now, computers ship with RAM that is literally orders of magnitude larger than back then, and so are HD's.  Actually, if you have a lot of RAM, say more than 4GB for a home user, I have seen a few recommendations to minimize the swap file size. These days even budget computers ship  with more than enough HD space that permanently allocating enough virtual memory is not a hardship. 

If you do set a constant swap file size, it is probably a good idea to defragment the swap file too. That requires special defragging tools.

steve diesel say: Hi,

This problem is due to one or more profiles under HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList with missing ProfileImagePath.

To check whether you have missing profiles:
1.       Open regedit, navigate to the above registry key. (HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\ProfileList). Expand the list

2.       Click on each of the profiles listed. The first 3 profiles should have ProfileImagePath value of
%SystemRoot%\System32\Config\SystemProfile,
%SystemRoot%\ServiceProfiles\LocalService, and %SystemRoot%\ServiceProfiles\NetworkService respectively.

3.       Starting from the 4th profile, the ProfileImagePath should contain path to the user profiles on your machine, such as C:\users\Christine

4.       If one or more of the profile has no profile image, then you have missing profiles.

To work around this, delete the profile in question
