Windows Media Player-shuffle mode - Microsoft Community

johnny nukes say:

Windows Media Player-shuffle mode

Windows media player

i like to listen to my media player in shuffle mode, but it plays a lot of the same songs frequently. many songs have not even been picked yet. is there a setting so it picks songs completely randomly?

tim de baets say:

As far as I know, shuffle mode *is* completely random, but if you don't have many songs yet, it will indeed quickly pick the same songs.

If you want to exclude the songs that have been recently played, then you need to create an auto playlist, with a condition like Date Last Played is before 30 days , and shuffle this auto playlist instead.

By default, this playlist won't include songs that haven't been played yet. For that, you need to OR the first condition with Play count: Total overall is 0 , just like in the example playlist at http://thewelltemperedcomputer.com/SW/WMP/screenshot/Playlist1.jpg .

For more info about auto playlists, refer to WMP Help.
