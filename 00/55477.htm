How can I restore my POP3 Mail Folders which were deleted in Windows - Microsoft Community

bighead one say:

How can I restore my POP3 Mail Folders which were deleted in Windows Live Mail 2011?


How can I restore my POP3 Mail Folders which were deleted in Windows Live Mail 2011?



I'm using Windows Live Mail 2011 on my laptop. I lost my POP3 Mail Folders, upon changing my Mail account to an IMAP account.

a. user say:

Hello,

 

I'd be pleased to assist you with your concern on recovering your deleted mail folders in Windows Live Mail.

 

You may be able to selectively recover the deleted folders using the Previous Versions component. However, you will need to know the name of the folders that were deleted to recover them. Please refer to the steps below:

 

1. You will first need to select the Show hidden files option.

    a. Open Windows Explorer and click Organize / Folder and search options. Click the View Tab.

    b. Place a check mark in the Show Hidden files, folders and drives option. Click Apply/OK.

    c. Close and restart Windows Explorer.

 

2. Open the WLM Store folder.

    a. Navigate to the following folder: C:\Users\your name\AppData\Local\Microsoft\Windows Live Mail (Where 'your name' is the name of the account that you use to logon to Windows 7)

    b. In this folder you will see all of the WLM storage folders. Each email account will have its own group of folders.

    c. Any general storage folders that you created will under the main Storage Folder item.

 

3. Make a copy of the Windows Live Mail folders previous version.

    a. Right click the Windows Live Mail folder and select Properties / Previous Versions Tab.

    b. After the list populates, select a version of the folder with a date and time prior to when the folders were deleted.

    c. Click the Copy Button. In the Copy dialog box, select the Desktop as the location to place the copied folder.

 

4. Move the previous versions of the deleted folders to the WLM store folder.

    a. Go to Desktop and open the copied WLM folder. Locate the folders that were deleted.

    b. Right click the folder and select Copy. Go to the current WLM storage folders and paste the folder into the same location where it was deleted from.

 

After you have retrieved all of the deleted folders, you can delete the copied WLM previous version folder from the Desktop.

 

If you are using POP3 and leaving a copy of the email on the server is not set to delete email from the server, then there should still be a copy of the email available on the server.

 

If the account is a Hotmail account, the email may be recovered on your Deleted folder by referring to this link.

 

Note: Only recently deleted emails can be retrieved by the system. If you were unable to find the emails you were looking for after using the auto restore option, it is possible that it is permanently lost.

 

You can also request the sender to re-send the emails if possible if the steps above were unable to resolve your issue.

 

Let us know if you have any questions.

 

Thank you!
