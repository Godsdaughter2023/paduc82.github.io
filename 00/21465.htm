  3. Don't have multiple worksheets within a workbook.  Put everything on 1 worksheet.

None of the points above are new but they may help.  I rarely have to deal with a corrupt file issue these days...which is probably due to the fact I keep an eagle eye on the file sizes of essential workbooks.  If the file size approaches 30MB, I proactively see what I can do to reduce the contents.

print multiple worksheets based on cell condition - Microsoft Community

johnf2017 say:

print multiple worksheets based on cell condition

I have a workbook with 8 worksheets. I want to print the worksheets based on a condition on the first sheet. 

I have 7 cells on the first worksheet that correspond to values on each of the 7 other worksheets.

I want the first (cover) worksheet and each worksheet that it's corresponding cell is greater then 0 to print.

For instance, If cells 2, 4, and 7 on the cover worksheet each have a value greater then 0, I want worksheets 1 (cover worksheet), 2, 4, and 7 to print.

I want to set this up as a print button on the cover worksheet.

jasles say:

Assuming your cover sheet is called "cover" and it has the names of the sheets in column A starting in row 2 and the number to check is in column B, use this code:

Sub PrintSheets()
Dim x As Integer

ActiveWorkbook.Sheets("cover").PrintOut
For x = 2 To 8
    If ActiveWorkbook.Sheets("cover").Cells(x, 2) > 0 Then
        ActiveWorkbook.Sheets("" & ActiveWorkbook.Sheets("cover").Cells(x, 1) & "").PrintOut
    End If
Next

End Sub

You'll just need to insert a shape onto your worksheet and assign the above macro to it.
