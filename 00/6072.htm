    Hi Erik,

     

    Thanks for your feedback and sorry for this late reply.

     

    As you described, you have encountered the similar situation about the PDF outputs being either too large or too small (too low quality) when adding same pictures for multiple times in one presentation. In this case, PowerPoint may only count the size for pictures displaying in the presentation for one time, no matter how many times they are inserted. However, when you click Print.../Save as PDF, each picture inserted in the PDF file will be counted, including those inserted for multiple times. So the PDF file size is larger than the presentation file size after you saving it as PDF in PowerPoint 2016 for Mac. We apologize for the inconvenience caused by that issue.

     

    At the same time, we would like to suggest you select the PDF quality option in latest version of PowerPoint 2016 for Mac: Preferences > General > Print Quality (Paper/PDF). You can choose between High, Medium, or Low image quality. For printing, we recommend using high quality; for exporting to PDF, we recommend using low or medium quality (depending on your optimal file size).

    jeprox---01 say:

    Nokia charger AC-3 and AC-3U.

    Is there a difference between AC-3 and AC-3U? I bought a nokia phone and it is stated in the manual that my phone is equipped and should be used with an AC-3 charger. But what I can see on my charger's label is an AC-3U. Or are they the same?
    psychomania---01 say: It's the same thing. AC-3 is the type of charger, the "U" is the country code which decides which plug is on it.

    For example:

    AC-3U is for the US and other countries that use the same plug/voltage.

    AC-3X is for the UK with the three pin plug.

    AC-3E is for Europe with the two pin plug.
