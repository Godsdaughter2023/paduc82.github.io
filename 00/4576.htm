heyitstris say:

I think what you're looking for is this:

  1. Open the registry editor regedit.exe.
  2. Locate and then select the following registry entry:
    HKEY_CURRENT_USER\Software\Microsoft\Office\[Version number]\Outlook\Options
  3. Create a DWORD Value with the name DisableEmailAddressesInReplyHeaders.
  4. Change the Value

Have a look at https://support.microsoft.com/en-us/kb/3054903.
