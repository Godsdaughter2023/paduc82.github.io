daniel g. grau say:
Phillip, all fonts have those characters, both in Mac and Windows. Check out characters 210-213 in this table.

---------------------------------------------------------

Bridget, if you need to type them one by one, these are the key combinations (zoom in your browser to see the character details):

  • “ is Option [
  • ” is Option Shift [
  • ‘ is Option ]
  • ’ is Option Shift ]

But there's an easier way, because you can enable automatic replacement without having to type those key combinations:

  • In the main menu bar, select Word > Preferences > AutoCorrect
  • Click on the third tab, AutoFormat as You Type
  • Enable "Straight quotation marks..."
  • At the top of the dialog box, enable Automatically correct...

Those steps will cause all your straight quotes and apostrophes to become curly as you type. It's only in very few instances that Word will err (for instance, if you need to type WellCare/‘Ohana you will get WellCare/’Ohana instead).

Now for the replacement. After enabling "Straight quotation marks..." in AutoCorrect, as outlined above:

  • Just do a global replacement of " to " (quote to quote) and ' to ' (apostrophe to apostrophe)
  • Don't worry about typing them straight or curly, Word will do the right replacement into curly in all instances

And if you ever need straight quotes, you can:

  • Disable the AutoCorrect option, or
  • Paste the character from another place, or
  • Simply Undo with Command Z right after typing the quotation mark

From an earlier post of mine:

This registry edit has fixed this problem for many 64-bit IE10/IE11 users.

You can do this registry edit. Work slowly and carefully and create the registry backup that I mention.

Go to Start> Run, type in regedit and hit Enter. 

In the left pane of the regedit window, select Computer, right-click it and choose Export. Save that REG file to the desktop. That's your backup. By double-clicking or right-clicking> Merging that REG file you can restore the registry and undo changes, if necessary.

You navigate the registry just like you would drill down through a folder structure to get to a specific file (key.) Just click the + to open the next level.

Start> Run> type in REGEDIT and hit Enter.

By clicking in the [+} drill down to

HKEY_CURRENT_USER

  Software

     Microsoft

       Internet Explorer

         Main

With Main selected in the left pane, right-click in the right pane and choose New> DWORD (32-bit)

Name the new DWORD Value TabProcGrowth then right-click it and set the value to 1.

If TabProcGrowth already exists (you didn't have to create it) just change the value from 0 to 1.

- - - - -

This shouldn't cause any problems at all, even if it doesn't help with the PDF problem.
