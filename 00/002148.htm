bill manville say:

The following macro (suitably modified) will do it:

Sub AddHyperlinks()
' add a hyperlink to A1 on sheet Index to B2 on  each worksheet
  Dim WS As Worksheet
  Const HyperAdd = "B2" ' the cell in which you want the hyperlink
  For Each WS In ActiveWorkbook.Worksheets
    If WS.Name <> "Index" Then
      With WS.Range(HyperAdd)
        If .Hyperlinks.Count = 1 Then .Hyperlinks(1).Delete
        WS.Hyperlinks.Add WS.Range(HyperAdd), "", "Index!A1", , "Index"
      End With
    End If
  Next
End Sub

a. user say:

If you use port 25 for your SMTP server, most ISPs block that port if it's being accessed from something other than their email.

Try this: change your SMTP server port to 465 and check the box "use SS"

You might also have to set your SMTP server to authenticate.

Doing both of those should work - no need to change your ISP provider when away as is suggested in the other post.
