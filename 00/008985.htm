If you have configured Hotmail and Roadrunner account as POP account in Windows Live Mail then and folders that you have created in Windows Live Mail will not get synchronized in Outlook 2013 when you configure the email account, you may have to import the emails from Windows Live Mail to Outlook 2013.

 

When you configure Hotmail account in Outlook 2013 all the folders present on web-mail will get synchronized with Outlook.

If you configure Roadrunner account as POP account then only Inbox from web-mail gets synchronized with web-mail.

 

If you want all the folders that you had created in web-mail to present in Outlook 2013 then you can import the data from Windows Live mail to Outlook 2013. Refer the link mentioned below which explains how to import the data:

http://support.microsoft.com/kb/980534

 

You may refer the link mentioned below to configure Hotmail account in Outlook 2013:

http://answers.microsoft.com/en-us/office/forum/office_home-outlook/set-up-hotmail-with-outlook-2013-preview/aad124ad-a167-4122-b812-8a7243090da4

Note: Configure Hotmail account as Exchange Active Sync as explained in the above link.
