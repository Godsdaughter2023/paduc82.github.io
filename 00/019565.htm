hansv mvp say:

The function works correctly when you call it from VBA, but when you use it as a worksheet function, FindNext immediately returns Nothing although there are further matches.

The following version works in a worksheet formula, but you have to specify the range:

Public Function calcGrade(rng As Range) As Double
    Dim graded As New Collection
    Dim credits As Integer
    Dim grade As Double
    Dim cumCredits As Integer
    Dim cumGrade As Double
    Dim finalGrade As Double
    Dim Match As Range

        If Match.Value = True Then
            credits = Match.Offset(1, 0).Value
            grade = Match.Offset(2, 0).Value
            If grade <> 0 Then
                graded.Add (credits * grade)
                cumCredits = cumCredits + credits
            End If
        End If
    Next Match
    Dim val As Variant
    For Each val In graded
        cumGrade = cumGrade + val
    Next val
    finalGrade = cumGrade / cumCredits
    'Debug.Print finalGrade
    calcGrade = finalGrade
End Function

Example of use:

=calcGrade(B2:K10)

    davidf2 say:

    You did not mention what version the older Publisher is, but in general there are some default settings and features that are not the same across all versions of Publisher.

    I would suggest that you compress all pictures in your original file on your home computer to minimize the size of the Pub file. Try creating your 200 message file again. If no joy, then try a workaround by reducing your PDF file to smaller batches. Try 100...try 50...find the number that will work.

    Publisher has always been very printer driver picky. I sometimes find happiness if I install the office printer driver on my home computer and set it as my default for the Pub file while I am designing the publication. You don't need the actual printer to install the driver.

    If none of that works then you can reformat the file on the office computer and the older version of Publisher. You could also try producing one of your newsletters on the older version of Publisher and use it as a template on your home computer using the default settings established by the older version of Publisher.
