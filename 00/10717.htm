Powerpoint file size -- picture types - Microsoft Community

ken hunold say:

Powerpoint file size -- picture types

In trying to reduce the size of a PowerPoint presentation, I tried replacing .bmp pictures with .gif versions.  When I was done, the presentation was larger still.  Does the "replace picture" function really remove the picture in the presentation, or does it keep the picture in the presentation and only change it in the slide?  

Thanks,

Ken

john korchok say:

If the replaced photo or graphic is only used on one slide, then the embedded file is deleted when it's replaced by the new graphic.

GIF is limited to 256 colors, so it's not the best choice for photos. BMPs are a bulky format, I never use them.

The best format for photos is JPEG. A medium quality JPEG is often indistinguishable from a high-quality version in PowerPoint, because PowerPoint automatically blurs placed photos. Medium quality has a much smaller file size than high quality.

Line art, like logos, is best in EMF vector format, as that can't get downsampled by picture compression. Second-best are indexed-color PNGs for logos. While PNGs can have a smaller file size than complex EMFs, the picture compression issue over-rides that concern, at least for me.
