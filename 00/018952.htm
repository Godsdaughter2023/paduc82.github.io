In PowerPoint, the slide size and the paper size the user chooses to print to are two distinct settings.

Set the slide size to 11x17 then in the print dialog box, click the Printer Properties and set the printer to 11x17 also.

Then click the triangle next to "Full Page Slides" to reveal the oh-so-hidden "Scale to Fit Paper" option.  (Yes.  We know.  The people who designed the Print dialog box in 2010 are idiots.  Not our fault.  We're volunteers here.)

If the driver and PPT are cooperating, that should scale the slide contents up to the printable area of the 11x17 sheet.  However, few printers can print clear to the edges of the paper ... that's a physical limitation of the printer, not something that PowerPoint has a hand in or control over.

One other thing: if you've chosen a Ledger Paper (11x17) slide size in PowerPoint's slide setup dialog box, note that the actual slide size is not in the same proportions.  If you truly want to fill an 11x17 sheet, type the actual dimensions into the Width and Height boxes to get a Custom size.
