Financial week From To

per jessen say:

For this special situation, you will need to perform a lookup in a table.

 

With a table like the one you have outlined placed in Sheet2, you can use this formula where you have the actual date in A2:

 

=SUMPRODUCT(--(A2>=Sheet2!B2:B5),--(A2<=Sheet2!C2:C5),Sheet2!A2:A5)

Windows 10 HDMI Audio Solved Today By New Intel Graphics Driver - Microsoft Community

terry walsh say:

Windows 10 HDMI Audio Solved Today By New Intel Graphics Driver

Having built a new PC for Windows 10 with an ASUS Z97I-PLUS, I've been frustrated for the last few months with a lack of audio output from the HDMI port. ASUS ships the motherboard with Realtek drivers and the updated drivers that are being pulled down from WU simply don't work. 

However, I noticed today on the ASUS website that they'd posted a new WHQL Intel Graphics Driver (V10.18.15.4248) - I installed these manually and was surprised to see that my audio issues we fixed! Rather than use the Realtek drivers (which are still installed), my PC seems to have switched to using an Intel Audio HDMI device for output, which has popped up for the first time. 

Who'd have thought an audio issue would be fixed by a graphics driver? :-)

Just passing on in case anyone else is hitting the same issue.
