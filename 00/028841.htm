graham mayor say:
The simplest solution would be to add a new worksheet containing the filtered data and use that as the data source for the merge e.g. The following macro should create a new worksheet based on your filtered data named 'Label Merge Data' Change the bold text as appropriate

Dim xlBook As Workbook
Dim xlSource As Worksheet
Dim xlTarget As Worksheet
Const strSourceName as String = "Original Worksheet Name"
Const strTarget as String = "Label Merge Data"
    Set xlBook = ActiveWorkbook
    Set xlSource = xlBook.Sheets(strSourceName)
    For Each xlTarget In xlBook.Worksheets
        If xlTarget.Name = strTarget Then
            Application.DisplayAlerts = False
            xlTarget.Delete
            Application.DisplayAlerts = True
            Exit For
        End If
    Next xlTarget
    xlSource.AutoFilter.Range.Copy
    xlBook.Worksheets.Add.Paste
    Set xlTarget = xlBook.Sheets(1)
    xlTarget.Range("A1").ClearOutline
    xlTarget.Name = strTarget
    xlBook.Close SaveChanges:=True
    Set xlBook = Nothing
    Set xlSource = Nothing
    Set xlTarget = Nothing

daniel g. grau say:

As long as the captions/titles of figures and tables have the same paragraph style applied (in general this would be "Caption"), all you need to do is this:

  • Insert > Index and Tables
  • In the dialog box, select the "Table of Figures" tab at the top
  • It does not matter what you select under "Caption label"
  • Click on the "Options" button to open the "Table of Figure Options" dialog
  • Place a checkmark next to "Style" and select "Caption" (or whatever common style has been applied to both captions/titles of tables and figures)

Finally, add a "List of Figures and Tables" title above the new table of figures and apply to it whatever style you've already used for title "Abstract."
