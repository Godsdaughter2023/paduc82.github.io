Columns B, C and D have the code values of the source characters and then column E has the source characters sorted.

Refer to the screen shot below.

Cell B2 formula :         =CODE(MID($A2,1,1))

Cell C2 formula:          =CODE(MID($A2,2,1))

Cell D2 formula:          =CODE(MID($A2,3,1))

Cell E2 formula:           =CHAR(SMALL($B2:$D2,1))&CHAR(SMALL($B2:$D2,2))&CHAR(SMALL($B2:$D2,3))

There is no need for Array Formula.
