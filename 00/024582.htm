graham mayor say:
It seems that you have your normal template in the root of the C: drive which is a protected area with Windows 7. You need to ensure that the User Templates folder (File > Options > Advanced > File Locations) location is set to a location that you have normal read/write access to - maybe a sub folder of your My Documents folder, or the original default location
Then with Word closed, move the normal template from the root of the C drive to the chosen location (or rename the normal template and let Word create a new one in the chosen location).

[sxd] say:

The only solution I was able to come up with was as follows:

  1. Go to any OneNote SECTION and select "Insert Page Templates" from the ribbon.
  2. A Templates window should appear on the right side fo your OneNote window.
  3. For me, if I select Default - Blank, it brings up a new blank window with the Calibri Font Size 17 in the Page Name box. 
  4. I may have created that Default- Blank template myself (don't recall).  If so, all you have to do is open a new page, change the font in the Page Name box to Calibri 17.  The hit SAVE CURRENT PAGE AS TEMPLATE.  You can also check the "Set as default template for all new pages in this section" checkbox when you save it.  
  5. All new pages in this SECTION will be as you like - Calibri sized 17.

The only knock is that you have to do this once across all sections.  But once you do, you get the formatting you want on all new pages.
