arminatorx say:

Humming and thumping sounds that happen only on one disc, usually are a sign of a warped or uneven disc.

Either the DVD was poorly manufactured from the factory to be uneven from the beginning, or because of wrong handling (e.g. pulling the disc off the holder in the cover instead of pushing the holder down and letting the disc pop out by itself) the disc is slightly warped.

When you put an uneven or warped disc in the drive, the disc will start to vibrate causing the humming or thumping sound when spun at higher speeds during playback. 

There is nothing wrong with your console, if only that disc is causing the noise.

a. user say:

Hi

Hopefully this is a simple settings issue. Open control panel and select Ease of Access Centre. From here click - Make the keyboard easier to use, then click- Set up filter keys - have a look at the options on this page and also click on Set up repeat keys and slow keys and look at the options here, you have to do a bit of experimenting to get the desired effect should you decide to. Also from control panel click on - Keyboard - and here are some repeat rate settings, there is a test box to test the settings selected.

davem121 say:
Tayy while I hate to advise this, your only option now is probably a clean install of Windows 10

Click this link:
https://www.microsoft.com/en-gb/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB)


Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS)

If you have problems booting from a boot disc, you may have UEFI BIOS, go into your BIOS and make the following changes:
1. Insert the Bootable Media
2. Disable Secure Boot
3. Enable Legacy Boot
4. If Available enable CSM
5. If Required enable USB Boot
6. Move the device with the bootable disc to the top of the boot order
7. Save BIOS changes and restart your System

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers
