Going on the Microsoft information, legally the person who pulled the hard drives out can not install them elsewhere if they are OEM licensed.

Something I put together and something I saved from Microsoft explains that:

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc. machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

This was the original link. Microsoft is altering/deleting links, but the information that was originally there remains the same (I copied it before the link disappeared).

http://www.microsoft.com/oem/en-gb/licensing/sblicensing/pages/licensing_faq.aspx#fbid=iePP5a9JaWJ

Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. If the motherboard is upgraded or replaced for reasons other than a defect, then a new computer has been created. Microsoft OEM operating system software cannot be transferred to the new computer, and the licence of new operating system software is required.

If the motherboard is replaced because it is defective, you do not need to acquire a new operating system license for the PC as long as the replacement motherboard is the same make/model or the same manufacturer's replacement/equivalent, as defined by the manufacturer's warranty.

ruukas say:

Dear All,

Its an old topic, but I would like to share success experience by finding solution for E-studio 161 to work with 64bit systems.

Maybe will help somebody, still.

Toshiba has ended e-studio 161 support at Vista 32bit drivers, which can also be used for Windows 7 32bit.

But it does not work for Windows 64bit systems.

I was desperate to find solution, as all our office PCs moved to 64bit systems - so printer appear

to be not usable due to missing driver, only possible use for copying.

And - one day digging deeper -I found out that Toshiba e-studio 161 is one to one as Sharp AR-M160 (for both all hardware made by Sharp, they are compatible, except some labels and text on body)  ;)

So - there is a solution, bit tricky, but it works, perfectly, when properly done, even via network, through print server.

(note also - for print server Dlink DPR-1061 Toshiba's usb connection did not function, so we had to use parallel port LPT)

1. Find and download Sharp AR-M160 64bit driver for Windows7.

2. Upack it - WARNING - do not install!, only extract to folder.

3. go to folder where driver extracted and go up to  \Printer\EnglishA\GDI\XPVista64  , find there file   sf1cmins.inf

4. open it in notepad

5. find at [Manufacturer]  line  %oem%=SHARP,NTamd64   and change it  into %oem%=TOSHIBA,NTamd64   , save file.

6. now install driver - run  setup.exe  from folder where driver files were extracted. Follow the instructions. Choose AR-M160  model when asked.

Important - first install driver for printer connected to computer! Only then it will possible to use driver

also for network connected printer.

P.S. We tried, but this Windows7 64bit driver does not install for Windows 8, even using compatibility mode.

tiffany mcleod say:

Use Vista Shadow Copy to Restore Files

 

If you are using Vista Business or Vista Ultimate, you can try the "Restore Previous Versions" Shadow Copy feature:

 

Open Windows Explorer and navigate to the file in question.

 

If you want to keep the new file as well as restore the old version, copy the file to another location before proceeding further.

 

Place your mouse over the file and Right-click.

 

From the menu, choose "Restore Previous Versions".

 

Wait for the computer to access the previous versions. This could take a few moments.

 

Select the desired previous version and click the "Restore" button. (Or the Open or Copy buttons if you want to do that instead.) Click "Ok" as prompted.

 

The file will now be overwritten with the older version. You will lose all changes made since the older version you selected above.
