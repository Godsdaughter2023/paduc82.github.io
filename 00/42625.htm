a. user say:

I do not believe these are real system files or even program files (though they may have been so at one time).  They are probably either corruption or malware (probably not even functioning malware but malware leftovers from however your current anti-malware program deleted them in the first place) or leftovers from uninstalled programs that didn't properly uninstall.  IMHO, it is safe to delete these files (but only these files - not the "others like that" unless they are almost exactly like that (you don't want to delete an important file in your zeal to get rid of these seemingly useless ones).  It's better to err on the side of safety.  If I were you, I'd create a system restore point before deleting the files just in case.

If Disk Cleanup does not delete these files (and it's possible it won't - they aren't commonly-named files) then you can try deleting them in the conventional way.  If that doesn't work either, try using one of the following free products to delete the file.  File Assassin http://www.malwarebytes.org/fileassassin.php (with or without Force Delete: http://www.mydigitallife.info/2008/12/27/force-delete-cannot-delete-locked-or-in-used-files-or-folders-with-fileassassin/) or Unlocker at: http://www.softpedia.com/get/System/System-Miscellaneous/Unlocker.shtml.  These programs often work when normal Vista delete functions fail to operate properly.

Try to run anti-malware programs (in safe mode with networking if http://www.malwarebytes.org/mbam.php and http://www.safer-networking.org/en/index.html.  Do these two first because they are the most likely to find and repair any malware on your system (even though I don't recommend Malwarebytes as your day-to-day anti-malware program).  You may also want to try a free trial of Pandasoft (http://www.pandasecurity.com/usa/homeusers/solutions/global-protection/).  Before downloading pandasoft, uninstall whatever anti-virus program you are currently using (because installing and running two at the same time can cause conflicts, freezes, and all sorts of problems).  Once you've uninstalled your current AV program, download, install, update, and scan using Pandasoft.  I think you might be amazed at how much it finds that the other software missed.  Once complete, uninstall pandasoft and re-install your current AV program (unless you decide to switch which I did when I first tried it about 4 years ago).  You may also want to try the new, free Microsoft Security Essentials http://www.softpedia.com/progDownload/Microsoft-Security-Essentials-Download-131683.html (with the same caveat that only ONE AV program can be installed and running on your system at any one time).  You may also want to try the free Avira at: http://www.free-av.com/ and Avast at: http://www.avast.com/index.  Reboot after completing all the scans.  You may also want to try the free OneCare at http://onecare.live.com/site/en-us/default.htm and let it run all the options (not just the anti-virus program but NOT the registry cleaner) because that’s good maintenance (it will take some time to complete but can be done in the background).  I never recommend only one product when there is a real possibility of malware (unless it's a targeted program directed against a specific form of malware and actually not even then) because no one program catches everything and it's important to have a clean machine.

I hope this helps.

Good luck!
