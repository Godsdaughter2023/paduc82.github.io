t. valko say:

 

The 2 named ranges Category & Net must be the same size.


Here's a little known quirk about range size and SUMIF.

A1:A10 = X

B1:B19 = 10

Enter this formula somewhere and copy down 10 rows.

=SUMIF(A$1:A$10,"X",B1)

The referenced range argument is 10 rows so the function will automatically evaluate the sum_range argument to be the same size and starting from the first cell used as the sum_range argument even if the sum_range argument isn't explicitly defined to be the same size.

As you copy down the formula you essentially get:

=SUMIF(A$1:A$10,"X",B1:B10)
=SUMIF(A$1:A$10,"X",B2:B11)
=SUMIF(A$1:A$10,"X",B3:B12)
=SUMIF(A$1:A$10,"X",B4:B13)
=SUMIF(A$1:A$10,"X",B5:B14)
=SUMIF(A$1:A$10,"X",B6:B15)
=SUMIF(A$1:A$10,"X",B7:B16)
=SUMIF(A$1:A$10,"X",B8:B17)
=SUMIF(A$1:A$10,"X",B9:B18)
=SUMIF(A$1:A$10,"X",B10:B19)

The same would apply when using named ranges that are different sizes.

rich michaels - mvp (greatcirclelearning.com) say:

Spelling and Grammar are part of the Word's Language settings and assuming you are writing your paper in one language do the following.

Press Command + A to select the entire document

From the Tools menu choose Language

From the Language dialog select the language you need. Make sure the box for "Do not check..." is not marked. Press the Default button to make this language your chosen default setting and then press OK.

Open Word > Preferences > Spelling and Grammar and in the Grammar section you'll see a Recheck Document button. Click it and close out.

Hopefully all is once again right with the world. :-)

bob jones aka: cybertaz say:

Technically, it sounds like the Spelling & Grammar checking is working as it should, but it's the language attribute of the text that's at fault. For the immediate fix, try this;

  1. Command+A to Select All
  2. Go to Tools> Language
  3. Select your preferred Language from the list
  4. Make sure the Do not check... checkbox is clear
  5. Click OK

Unlikely, but it's possible that you may also have to go to Word> Preferences> Spelling & Grammar to confirm that the boxes are properly checked, then click the Check Document button, then confirm Yes.
