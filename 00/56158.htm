Hi,

To get at your router's admin pages...

  • Click start, type cmd and hit enter.
  • In the resultant black window type ipconfig and hit enter.
  • The router's admin pages can be accessed by the ip address that is defined as default gateway for the network device connected to your router.
  • Open a browser and enter http://<gateway ip address> (gateway ip address is in the format x.x.x.x where x is a number, probably 192.168.x.x).
  • This should open a login page for your router. A typical user name is admin. Typical passwords are password, admin and blank (no password).
  • If the username and passwords you try don't work consult the documentation or go to the manufacturer's website help pages for your router.
  • Once logged in look for wireless settings/encryption/security links.
