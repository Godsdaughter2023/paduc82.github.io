greg carmack - windows mvp 2010-2020 say:
Hi Gorakh. I'm Greg, an installation specialist and 10 year Windows MVP here to help you.

The Product Key on preinstalled WIndows is embedded in the UEFI Firmware chip on the motherboard. You can read it if you want to be sure by installing ProducKey: http://www.nirsoft.net/utils/produkey-x64.zip and looking for the OEM-BIOS key (only).

So it doesn't matter what User accounts are on it for activation purposes.

If you want to change your Local Account to a MS account then go into Settings > Accounts > Your INfo and click the link to Sign in with a MS Account.

If that's not present then go into Settings > Accounts > Family & Other People and create a new MS account under "Other" not family. Sign into it, move your files over, test all programs run correctly, then when ready delete the old account.

You can also delete any other accounts in the same place.

I hope this helps. Feel free to ask back any questions and keep me posted. I will keep working with you until it's resolved.

Hi Gorakh

My name is Andre Da Costa; an Independent Consultant, Windows Insider MVP and Windows & Devices for IT MVP. I'm here to help you with your problem.

If you factory reset it, its no longer in her name if you set it up with your Microsoft Account.

Since Windows Vista, the owner information on the winver dialog has become kind of irrelevant. You can change it if you want using the following steps:

Click Start, type: regedit
Right click regedit
Click Run as administrator


In the left pane, expand the tree view by double-clicking each of the following Registry keys:

HKEY_LOCAL_MACHINE
SOFTWARE
Microsoft
Windows NT

Click CurrentVersion.

If you want to change the owner name, double-click RegisteredOwner. Type a new owner name, and then click OK.

If you want to change the organization name, double-click RegisteredOrganization. Type a new organization name, and then click OK.

Close the Registry Editor.
The owner and organization names are changed.
