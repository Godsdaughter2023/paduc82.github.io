joeu2004 say: Low wrote:
> is a formula to be input at cell B16 that is able to capture the sum of the
> numbers of any group of number with difference lenth in term of number
> of rows ? My idea is to copy that formula and paste it to the bottom of the
> group of the number to get the instant answer.

Normally, I would ensure that there is a blank cell above and below each group of numbers, followed by a formula of the form in B17:  =SUM(B7:B16).  That formula will automatically self-adjust when you insert and delete rows of numbers between the blank cells in B7 and  B16.

But that does not give you the flexibility you are looking for, namely the ability to copy the formula to the end of each group of numbers.

For that, would you be amenable to a VBA function?  See the function below.

Ostensibly, for your purposes, you could put simply =sumUp() into B16, B23, B38 and B47.

More generally, you could use =sumUp(-1), where -1 (default) is the offset to the last of the group of numbers above.  Thus for example, with =sumUp(-2), you could put a blank cell between the group of numbers and their total.

But note that it would be prudent to round whatever formula you use.  For example, =ROUND(sumUp(),2).  This avoids arithmetic anomalies that are common because Excel cannot represent most non-integers exactly.

-----

If you unfamiliar with VBA, do the following steps to add the VBA function below.

  • In Excel, press alt+F11 to open the VBA window.
  • In the VBA window, click Insert > Module to open the VBA editor pane.
  • Copy the text of the function below and paste into the VBA editor pane.
  • You can now close the VBA window.

Unfortunately, the VBA function is volatile.  That means that any formula that uses it is recalculated each time you edit any cell in the workbook.  But I suspect that will be true for any formula that has the flexibility that you ask for.

The VBA function....

Function sumUp(Optional d As Long = -1) As Double
      Dim r As Range, x As Variant
      Application.Volatile
      Set r = Application.Caller
      If d >= 0 Or r.Row + d <= 0 Then Exit Function
      Set r = r.offset(d)
      Do
            x = r
            If IsEmpty(x) Or Not IsNumeric(x) Then Exit Function
            sumUp = sumUp + x
            If r.Row = 1 Then Exit Function
            Set r = r.offset(-1)
      Loop
End Function

Perhaps you should lay out your brochure as a two-column table, putting the pictures in one cell in each row and the accompanying text in the adjacent cell. If you specify a row height or column width (or both), pictures you insert will be constrained to the height/width of the cell.
