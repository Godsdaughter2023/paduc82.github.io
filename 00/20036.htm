' Scan for Word 2016 - 64 Bit
' Author: Günter Born www.borncity.de blog.borncity.com
' Implements a Scan function in Word 2016

Private Declare PtrSafe Function GetTempPath Lib "kernel32" Alias "GetTempPathA" (ByVal nBufferLength As Long, ByVal lpBuffer As String) As Long

Private Function TempPath() As String
  Const MaxPathLen = 256 ' Max length of the path, just as big as possible
 
  Dim FolderName As String ' Name of the folder
  Dim ReturnVar As Long ' Return Value
 
  FolderName = String(MaxPathLen, 0)
  ReturnVar = GetTempPath(MaxPathLen, FolderName)
 
  If ReturnVar <> 0 Then
    TempPath = Left(FolderName, InStr(FolderName, Chr(0)) - 1)
  Else
    TempPath = vbNullString
  End If
End Function

Sub Scan()
'
' Scan Macro, to be invoked in Word 2016
'
  On Error Resume Next
     Dim objCommonDialog As WIA.CommonDialog
     Dim objImage As WIA.ImageFile
     Dim strDateiname
     ' instantiate Scan WIA objects
     Set objCommonDialog = New WIA.CommonDialog
     Set objImage = objCommonDialog.ShowAcquireImage
     
    strDateiname = TempPath & "Scan.jpg" ' set temporary file
'     strDateiname = "C:\Users\Public\Pictures\" & "Scan.jpg"
    
     If Not objImage Is Nothing Then
       Kill strDateiname
       objImage.SaveFile strDateiname ' save into temp file
       Selection.InlineShapes.AddPicture strDateiname ' insert into document
       Set objImage = Nothing
     End If
     Set objCommonDialog = Nothing

End Sub

Copy new sheet with today date when excel open - Microsoft Community

black panther1788 say:

Copy new sheet with today date when excel open

Hi

If I open workbook then it will copy a worksheet with today date and check if current date worksheet exist then it wont create any duplicate.

Kindly help me.

Let me explain you briefly:

I am having one sheet named "Tracker", everyday I used to copy this sheet and rename to today's date.

But how about if it does automatically then it will be great.

So when I open workbook then automatically it will copy "Tracker" with todays date.

myall_blues say: You need to use a macro (VBA code) to do this, Sadly I am not familiar with working with VBA on a mac, but the following site shows how to access the VBA editor.

https://www.techonthenet.com/excel/macros/visua...

From there, click on 'ThisWorkbook' in the resultant window and paste the following code. 
Private Sub Workbook_Open()
    Dim TrackerExists As Boolean
    On Error Resume Next
        TrackerExists = (Sheets("Tracker").Name <> "")
    On Error GoTo 0
    If TrackerExists Then
        On Error Resume Next
            Application.DisplayAlerts = False
            Sheets(Format(Date, "ddmmyyyy")).Delete
            Application.DisplayAlerts = False
        On Error GoTo 0
        Sheets("Tracker").Copy After:=Sheets(ThisWorkbook.Sheets.Count)
        ActiveSheet.Name = Format(Date, "ddmmyyyy")
    End If
End Sub


I have assumed that if you open the workbook more than once in a day you just want to delete the sheet named with today's date. Also assumed you want the date as ddmmyyyy - change the two format statements as appropriate if you want something different.

Note: The linked site is a non-Microsoft website. The page appears to be providing accurate, safe information. Watch out for ads on the site that may advertise products frequently classified as a PUP (Potentially Unwanted Products). Thoroughly research any product advertised on the site before you decide to download and install it.
