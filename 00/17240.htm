Excel Tables are given default names (Table1, Table2, etc.) when they are created. Table names are similar to defined names, which makes them easy use in formulas or to select data. To assign a shortcut, use the Customize Keyboard dialog box (click Tools | Customize, and then click the Keyboard button).
