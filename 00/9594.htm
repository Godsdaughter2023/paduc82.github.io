Left click to insert the cursor at the end of the text you do not want and use your backspace key to delete all the text you do not want.

Hi max2tucker,

Internet Explorer (IE) 8 is saying "Certificate error: navigation blocked". This means that, although the secure server certificate is encrypting data correctly, IE8 cannot verify the domain name. That is because there is not a SSL certificate for your particular domain name.

This does not indicate a problem with the website or control panel. This could be more accurately described as a fault with Internet Explorer being a little overly paranoid on your behalf.

Check to make sure you time and date settings are correct on your computer. These errors/problems occur when your system is set to a date is set incorrectly. Website certificates (even Windows Update uses website certificates) are marked as being good between this date and this date. If your computer is set outside of those dates the certificate won’t be used by the computer and the website/service will not work correctly. You can check your Date and Time settings by;

·         Click Start

·         Click Control Panel

·         Open Date and Time (you may need to change your view to Icons instead of Categories)

·         Change the date and time to their correct settings

·         Click OK

 

Once that is corrected you will need to use Windows Update to install the latest root certificates so that you can use Windows Update and Internet Explorer without encountering certificate errors.
