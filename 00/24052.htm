Sub InsertPicturesMac()

    Dim MyPath As String, MyScript As String, MyFiles As String

    Dim MySplit As Variant

    Dim N As Long

    Dim Fname As String

    Dim OneFile As Boolean

    Dim FileFormat As String

    FileFormat = "{""bmp"",""emf"",""emz"",""eps"",""fpix"",""fpx"",""gif"",""jpeg"",""jfif""," & _

                    """jpeg-2000"",""jpg"",""pdf"",""pict"",""pct"",""png"",""pntg"",""psd""," & _

                    """qtif"",""sgi"",""tga"",""tpic"",""tiff"",""tif"",""wmf"",""wmz""}"

    OneFile = False

    On Error Resume Next

    MyPath = MacScript("return (path to pictures folder) as String")

    If Val(Application.Version) < 15 Then

        'Word 2011

        If OneFile = True Then

            MyScript = _

                "set theFile to (choose file of type" & _

                " " & FileFormat & " " & _

                "with prompt ""Please select a file"" default location alias """ & _

                MyPath & """ without multiple selections allowed) as string" & vbNewLine & _

                "return theFile"

        Else

            MyScript = _

                "set applescript's text item delimiters to {ASCII character 10} " & vbNewLine & _

                "set theFiles to (choose file of type" & _

                " " & FileFormat & " " & _

                "with prompt ""Please select a file or files"" default location alias """ & _

                MyPath & """ with multiple selections allowed) as string" & vbNewLine & _

                "set applescript's text item delimiters to """" " & vbNewLine & _

                "return theFiles"

        End If

    Else

        'Word 2016 (Mac)

        If OneFile = True Then

            MyScript = _

                "set theFile to (choose file of type" & _

                " " & FileFormat & " " & _

                "with prompt ""Please select a file"" default location alias """ & _

                MyPath & """ without multiple selections allowed) as string" & vbNewLine & _

                "return posix path of theFile"

        Else

            MyScript = _

                "set theFiles to (choose file of type" & _

                " " & FileFormat & " " & _

                "with prompt ""Please select a file or files"" default location alias """ & _

                MyPath & """ with multiple selections allowed)" & vbNewLine & _

                "set thePOSIXFiles to {}" & vbNewLine & _

                "repeat with aFile in theFiles" & vbNewLine & _

                "set end of thePOSIXFiles to POSIX path of aFile" & vbNewLine & _

                "end repeat" & vbNewLine & _

                "set {TID, text item delimiters} to {text item delimiters, ASCII character 10}" & vbNewLine & _

                "set thePOSIXFiles to thePOSIXFiles as text" & vbNewLine & _

                "set text item delimiters to TID" & vbNewLine & _

                "return thePOSIXFiles"

        End If

    End If

    MyFiles = MacScript(MyScript)

    On Error GoTo 0

    If MyFiles <> "" Then

        With Application

            .ScreenUpdating =

        End With

        MySplit = Split(MyFiles, Chr(10))

        For N = LBound(MySplit)

            Fname = Right(MySplit(N), Len(MySplit(N)) - InStrRev(MySplit(N), Application.PathSeparator, , 1))

            Selection.InlineShapes.AddPicture MySplit(N)

            Selection.Collapse Word.WdCollapseDirection.wdCollapseEnd

            Selection.TypeParagraph

        Next N

        With Application

            .ScreenUpdating =

        End With

    End If

End Sub

On the subject of another possible approach.

In a .doc, you can insert your pictures using an INCLUDEPICTURE field. To insert such a field, you can use
 a. cmd-F9 (or fn-cmd-F9, depending on your keyboard setup) to insert a pair of the special field code braces {  }
 b. between the {  }, type "INCLUDEPICTURE followed by the double-quoted Posix pathname of the image file, e.g. followed by a \d, so you see, e.g.

{ INCLUDEPICTURE "/Volumes/expa02/scans/slides/1989 trip/box 1/01 canberra cloudscape.jpg" \d }

Select that and press F9 (or fn-F9) to update the field. Mac may present its permission dialog. You can copy/paste a working INCLUDEPICTURE field many times, modify the pathnames for each different picture, then select all the fields and use F9/fn-F9 to update them. The Mac permissions dialog will then come up for each "new" picture.

If you save *that* file as Web Page, Filtered, what you get is a small .html file whose <img> elements point directly to the included pictures. In other words, there is no supporting folder of transformed images. But whether that can get you where you need with conversion to Kindle format, I cannot tell.

_elaine_ say:

Hi, BSC Jim Lawrence

The following may sound a bit complicated: no, the initial Skype Number subscription term cannot be changed.

However, when the current three-month subscription ends, you can then renew the Skype Number subscription for a longer term (one year).

Hint: Cancel the current Skype Number subscription right away - the cancellation only stops the current subscription from renewing for another three-month term, and you still have full use of the Skype Number.

Hint: The day after the current three-month Skype Number subscription term ends, sign on to your account dashboard (portal) at skype.com to renew the Skype Number subscription for the longer term.  The Skype Number itself will be held on your account for 90 days following the initial three-month subscription's expiration date, so you have time to sign on to your account dashboard to reinstate the Skype Number using a new subscription.
