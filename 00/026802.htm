hansv mvp say:

Select G1:L32.

I'll assume that G1 is the active cell within the selection.

Apply green as fill color - this will be the default.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula    =ISODD(A1)

Click Format...

Activate the Fill tab.

Select red, then click OK.

Hi Richard,

 

Thanks for your information.

 

Generally, based on my experience, many admins will use the same UPN in AD and Office 365. It is an easy method to manage users in the organization. If this is your final purpose, you need to add the same SMTP addresses and change the UPN. You don’t need to delete the accounts. UPN is not the first rule to match users. The SMTP address is the first one if you use soft match. Therefore, you need to make sure that the private SMTP address are the same one if you want to do it. Meantime, all cloud attributes will be overwritten with the on-premise attributes.

 

To avoid any potential issues, we recommend you do it at weekend. It will not affect users. I assume your situations may be more complicated with other permissions, you can create a test account to try it first before doing anything. Please let me know if you have any questions when doing this. We can work it together.

 

Soft matching and Hard matching are the only two methods to connect AD accounts and cloud accounts. Otherwise, you may need to create accounts in AD and sync them again.
