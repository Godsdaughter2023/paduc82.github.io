talking moose say:
Outlook for Mac uses the Exchange Web Services protocol (EWS), which is an alternative to the MAPI protocol that Outlook for Windows uses. EWS allows you to sync mail, calendars, contacts, tasks and notes just like MAPI. ActiveSync is for mobile devices only such as iPhones, Blackberries and Androids.

When you set up your account under Outlook menu -->  Preferences... --> Accounts  you want to create a new Exchange account. This will use the EWS protocol. Typically, if you're connecting from outside your company network then you'll use the same server address that you'd use for mobile devices or for accessing your mail via Outlook Web Access (OWA) in a web browser.

Due to limitations with EWS, Outlook for Mac will check for new mail every minute. EWS and Outlook for Mac currently don't work together using push technology.

General Form:
