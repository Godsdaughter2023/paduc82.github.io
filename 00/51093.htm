b) Type “”services.msc” without quotes and hit Enter.
c) Locate Windows Time service and double tap on the service.
d) Tap on Stop to stop the service.
e) Restart the computer.
f) Navigate to the same location to start the service.


Step 2: Sync the time zone again on the computer again:

a) Tap on the time icon at the bottom left corner on the desktop screen.
b) Tap on Change date and time settings.
c) Tap on Internet Time tab and Tap on Change Settings.
d) Make sure Synchronize with an Internet time server is checked and check in OK.
e) Restart the computer and check if you find the correct time on the computer.

 

If it is changing when you reboot, then either the BIOS time is incorrect or the CMOS battery is dead. You can access the BIOS when the system is first turned on.
