macropod say:

I created a table, added text, and set wrap-text to "square," then inserted a picture. Nothing happened as far as text wrapping. I then set the text-wrap for the picture to "square" and similarly nothing happened.

All I can say in reply is that either:
• you haven't done it correctly, with the picture anchored in the cell concerned; or

• the picture is too wide to allow the text to wrap around it.

As far as setting "wrap-text to "square"" is concerned, the only option a table gives is 'around' or 'none' and these settings determine only whether text outside the table will wrap around it. These settings have no effect on what goes on within the table and, even where text outside the table is concerned, text wrapping around the table will only occur if there's space to do so (i.e. the table width is significantly less than the available text width).

As for pictures within a cell, inserting a picture inline within a table cell doesn't result in text wrapping - that only comes into effect with 'square' or 'tight', for example. Even then, text wrapping will only occur if there's space to do so (i.e. the picture width is significantly less than the cell width).

ossiemac say:

Hello Nikki,

 

If I am understanding correctly then something like the following example where you get the input in the main sub and then pass the values to the subs being called. Note that it is rarely necessary to select the worksheets or ranges; just address them by worksheet name and ranges. AutoFilter.Range is a dynamic value for whatever the AutoFiltered range is.

 

Note: A space and underscore at the end of a line is a line break in an otherwise single line of code.

 

Sub CallingMacro()
    Dim FirstDate As Date
    Dim LastDate As Date
   
    FirstDate = Application.InputBox _
            ("Please Enter the First Date in DD/MM/YYYY format: ", _
            "Enter First Date")
    LastDate = Application.InputBox _
            ("Please Enter the Last Date in DD/MM/YYYY format: ", _
            "Enter Last Date")
    
    Call Macro1(FirstDate, LastDate)
    Call Macro2(FirstDate, LastDate)
    Call Macro3(FirstDate, LastDate)
End Sub

 

Sub Macro1(dteFirst As Date, dteLast As Date)
   
    With Sheets("Sheet2")
        .AutoFilterMode = False     'Removes any current filtering
        .UsedRange.AutoFilter       'Turn on AutoFilter for the used range
        .AutoFilter.Range.AutoFilter Field:=18, Criteria1:=">=" _
            & CLng(dteFirst), Operator:=xlAnd, Criteria2:="<=" & CLng(dteLast)
    End With
End Sub


Sub Macro2(dteFirst As Date, dteLast As Date)
   
    With Sheets("Sheet3")
        .AutoFilterMode = False
        .UsedRange.AutoFilter
        .AutoFilter.Range.AutoFilter Field:=18, Criteria1:=">=" & _
            CLng(dteFirst), Operator:=xlAnd, Criteria2:="<=" & CLng(dteLast)
    End With
End Sub

Sub Macro3(dteFirst As Date, dteLast As Date)
   
    With Sheets("Sheet4")
        .AutoFilterMode = False
        .UsedRange.AutoFilter
        .AutoFilter.Range.AutoFilter Field:=18, Criteria1:=">=" _
            & CLng(dteFirst), Operator:=xlAnd, Criteria2:="<=" & CLng(dteLast)
    End With
End Sub
