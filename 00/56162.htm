slance310 say:

Hi,

Normally to do a boot repair you need some type of installation media, or access to the Repair feature in Windows Recovery... You can sometimes get to this through one of the Boot Menus at Power ON...

Something like ESC, DEL, F11, F12... it varies by manufacturer.

When you look at your boot options do you see anything like Recovery, or UEFI Boot ?

One thing you might try first is a different Cloning program, depending on which one you use...

Did this come with the SSD, or other ?

I will include the steps for a boot repair just in case you have a method to get to the Command Prompt when you Power On your machine... If you don't have the necessary media or other access to the CMD Prompt... I will see if I can find another option...

You can skip to Step #6 if you can get to a CMD Prompt without a disk...

1. Put the Windows installation disc in the disc drive, and then start the computer.

2. Press a key when the message indicating "Press any key to boot from CD or DVD …". appears.

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

4. Click Repair your computer.

5. Click the operating system that you want to repair, and then click Next.

6. In the System Recovery Options dialog box, click Command Prompt.

7. Type BOOTREC /FIXMBR, and then press ENTER.

8. Type BOOTREC /FIXBOOT, and then press ENTER.

9. Type BOOTREC /REBUILDBCD, and then press ENTER.

10. Type EXIT, and then press ENTER.
