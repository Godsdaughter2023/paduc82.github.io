Microsoft Word 2010 creating multiple pages of labels - Microsoft Community

karenpace say:

Microsoft Word 2010 creating multiple pages of labels

I am trying to create labels with multiple pages without processing as a mail merge in Word 2010. Please advise how I can type up multiple pages within the same document of labels.

thanks

doug robbins - mvp office apps & services (word) say:

If you want the same text on all of the labels, in the Labels dialog, select the Full page of the same label button and then click on New Document, then print as many copies as you want.

If the data for each label is different, select the Single label button and then click on New Document and in that document, insert the text for each label and then print as many copies as you want.

If you want multiple pages with different text on all of the labels, then it would really be much better to create a data source in an Excel workbook and then use mail merge.
