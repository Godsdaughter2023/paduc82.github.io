¡firedog say:

Have tried Essentials Repair and have disabled email scanning in anti-virus software - makes no difference.

No, the repair wouldn't. You are better off, though, without anti-virus scanning of email.

Windows Live Mail Error ID: 0x800420CD

This error indicates that there's a message on the server that Windows Live Mail can't or won't down­load. Log in to the account in webmail and start by emptying the spam and deleted folders. Then move all of the messages in the inbox to a user-created temporary folder, leaving the inbox empty. Now do a send/receive in Windows Live Mail; you should see a No new email messages status mes­sage. Now move half of the messages from the temporary folder back into the inbox, and again send/receive in Windows Live Mail. Repeat, halving the number of messages moved each time until the error recurs; you should soon be able to identify the message that Windows Live Mail is choking on and deal with it.

You are confusing RAM with Hard Drive/Solid State Drive storage.

RAM is working memory and you will only have Windows 10 running not Windows 8.1 and 10 running simultaneously.

mohammad _ imran say:

Hi Shane,

To share the libraries with the other Homegroup follow the steps provided below.

To share other libraries that you've created, follow these steps:

1.      Click the Start button, and then click your user name.

2.      Select the library you want to share, and then, in the toolbar, click Share with.

3.      Select the people you want to share with.

For more information visit the article below:

Share libraries with your Homegroup

http://windows.microsoft.com/en-us/windows7/Share-libraries-with-your-homegroup

If you want to include a shared folder in a library.

1.      On the other Homegroup member's computer, right-click the folder, click Share with, and then select either Homegroup (Read) or Homegroup (Read/Write).

2.      On your own computer, click the Start button, and then click your user name.

3.      In the navigation pane (left pane), under Homegroup, click the computer you want to access.

4.      In the right pane, right-click and drag the folder you want to your library.

5.      When you release the mouse button, a menu appears. Click Include in library.

For more information on how to Access files and printers on other Homegroup computers visit the article below:

Access files and printers on other Homegroup computers

http://windows.microsoft.com/en-us/windows7/Access-files-and-printers-on-other-homegroup-computers
