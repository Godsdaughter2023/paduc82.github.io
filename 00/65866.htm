sandeep g r say: Hi,

Thank you for choosing Windows Vista Forums.

The method to change the Default Username is same in all the Windows based Computers. You can perform the same steps as in XP.

1. Click the Windows "Start”. Type regedit and press the "Enter" key.

2. Navigate to the following path in the registry directory tree:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon.


3. Right-click the "DefaultUserName" value in the directory details window and select "Edit." If this value does not exist, right-click the "Winlogon" directory and choose "New." Select "Key" from this menu. This opens a window dialog for a new key entry.


4. Enter "DefaultUserName" as the key name. This is already set if you are editing the value.


5. Select "REG_SZ" in the section labeled "Value Type."

6. Enter the default username value you wish to prompt in the login window during boot up. Click the "Ok" button. The value is saved, but changes do not take effect until you reboot the computer.
