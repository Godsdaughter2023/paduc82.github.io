Try code like the following. It will return the caption of the currently active window.

 

Public Declare Function GetForegroundWindow Lib "user32" _
    Alias "GetForegroundWindow" () As Long
Public Declare Function GetWindowText Lib "user32" _
    Alias "GetWindowTextA" (ByVal hwnd As Long, _
    ByVal lpString As String, ByVal cch As Long) As Long

 

Sub AAA()
    Dim WinText As String
    Dim HWnd As Long
    Dim L As Long
    HWnd = GetForegroundWindow()
    WinText = String(255, vbNullChar)
    L = GetWindowText(HWnd, WinText, 255)
    WinText = Left(WinText, InStr(1, WinText, vbNullChar) - 1)
    Debug.Print L, WinText
End Sub

Hi,

 

Try this

 

1. Assume data is in range D6:E16.  Column D has names and E has numbers

2. headings are in D5:E5, say Name and Amount

3. In cell F5 enter Dummy

3. In cell F6, enter =SUM($E$6:$E$16)/COUNTIF($D$6:$D$16,D6) and copy till cell F16

4. Create your pivot table

5. In the pivot table, insert a calculated field with the following formula =Amount/Dummy

 

This will ensure that the denominator will always be picked up from the Dummy column of your source data and therefore applying a filter in your pivot table will not impact the Amount/Dummy calculation.

Thank you for the response.

To launch in-private mode browser, follow the steps below:

Google Chrome

  • Click Settings. From the drop-down menu, click New Incognito Window.

Firefox

  • Click Settings. From the drop-down menu, select New Private Window.

Internet Explorer

  • Click Settings. From the drop-down menu, select Safety and then InPrivate Browsing.

And as for signing in issue, browsers nowadays have the option to verify if you're accessing the same account and will give you an option to remember your credential to avoid logging in manually all the time.

In this case, the best way to avoid this from happening is to install a different browser and from there access either the Skype or Outlook account then on the other browser, access the other account. However, if you're unable to do so, using in-private mode will avoid this conflict.
