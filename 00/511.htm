yangzheng_msft6300 say:

Hi Tommoseley,

From your description, my understanding is that you want the accounts in the 2 domains to only see the accounts of their own domains in the address lists. If so, you can create a separated address list for the accounts in the 2 domains so that they can only see the accounts in their own domains. Here are the detailed steps:

Note: We need to use the mailbox attribute instead of email address suffix as the filter options. To change the attribute, sign in to EAC > Recipients > Mailbox > Edit > General > More Option

1. Make sure that you admin account has the Address List role permission. You can check this in Office 365 EAC (Exchange Admin Center)->permissions->admin role, and add the Address List role to the admin role group where your account is a member.

2. Connect to Exchange Online with PowerShell.

3. Turn on address book policy routing.

4. Run the command below to create a new room list.

New-AddressList -name "Roomlist1" -RecipientFilter {(CustomAttribute1 -eq "domain1") or (CustomAttribute1 –eq “domain2”)}

5. Run the command below to create a new Address List which will contain the accounts in domain1 and domain2.

New-AddressList -Name Addresslist1 -RecipientFilter{(CustomAttribute1 –eq “*** Email address is removed for privacy ***”) or (CustomerAttribute1 –eq “*** Email address is removed for privacy ***”)}

6. Run the command below to create a new Global Address List which will contain the accounts in domain1 and domain2.

New-GlobalAddressList -name "GAL1" -RecipientFilter {(CustomAttribute1 -eq "*** Email address is removed for privacy ***") or ( *** Email address is removed for privacy ***)}

7. Run the command below to create a new Offline Address Book.

New-OfflineAddressBook -Name "OAB1" -AddressLists "GAL1"

9 Run the command below to create a new Address Book Policy.

New-AddressBookPolicy -name "ABP1" -AddressLists "Addresslist1" -OfflineAddressBook "\OAB1" -GlobalAddressList "\GAL1" -RoomList "\Roomlist1"

10. Then run the command below to assign the new policy to the accounts in domain1 and domain2.

Get-Mailbox -Filter {(CustomAttribute1 -like "*** Email address is removed for privacy ***") or (CustomAttribute1 –like “*** Email address is removed for privacy ***)} | Set-AddressBookPolicy ABP1

Regards,
Yang
