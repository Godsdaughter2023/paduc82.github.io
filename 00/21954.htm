what are these two formulas written in words - Microsoft Community

gjc100 say:

what are these two formulas written in words

mike h.. say:

Hi,

The formula

SumIf(B3:M3,"<1e100")

Is summing B3 to M3 and ignoring errors in the range.  If you simply use =SUM(B3:M3) and there were errors in the range then the SUM formula wouldn't work. The  "<1e100" bit ignores the errors because any error value will always be greater then 1E100 which is simply a very large number in scientific notation.

This formula

=IFERROR((AB3*$AH$3)+AB3,"")

is multiplying AB3*AH3 and then adding the value in AB3

The formula is wrapped in an IFERROR functions and if the result of the math is an error then that function evaluates as TRUE and returns a null string ""
