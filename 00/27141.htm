(Please turn off your caps lock key. On the Internet, all caps is considered like shouting.)

In the AutoCorrect Options dialog (reached by the sequence of steps you listed), the replacement of three periods by an ellipsis character is in the Math AutoCorrect tab of the dialog. It's possible that something got messed up there. You can try removing and recreating that entry, with these steps:

  • In the Math AutoCorrect dialog, make sure the "Replace text as you type" box is checked. If it isn't checked, then check it and close the dialog, and test whether the ellipsis is now being replaced correctly.
  • If the box was already checked, click on the ... entry and click the Delete button, then close the dialog and continue with the other steps.
  • In any document, press the Ctrl, Alt, and period keys together to insert an ellipsis character.
  • Select the ellipsis character (being careful not to select any other characters or paragraph mark on either side of it).
  • Open the Math AutoCorrect dialog again. The ellipsis will appear in the With box.
  • Make sure the "Replace text as you type" box is checked.
  • Type three periods in the Replace box, and click the Add button. Then close the dialog.

Don't give up just yet.

To do a mail merge without using VBA code, you need to use Microsoft Word.  The list of e-mail addresses can be in Excel though.

What I gave you in my last post was two pieces of VBA code that allowed you to:

1) Automate the task of extracting ALL the names and e-mail addresses from your Outlook addressbook, and into Excel.  You've already done that manually, so no need to worry about that for now, but if in the future you need to do this again with a large list of other names, you might think about trying the code.

2) Automate the task of doing a mailmerge in Excel using VBA.  If you are not comfortable trying to use VBA code, you'll have to learn how to do it with Word.  This video will help:

http://www.youtube.com/watch?v=_9ExKwXQ3KQ

 (along with the link posted by RonBarker).

Unfortunately, it doesn't look like you can add attachments to e-mails using mailmerge in Word, but you can embed a file in Word by going to the Insert tab > Text group > Object > Create from file tab > and then browse to the file you want to attach, check the Display as Icon option (ALSO TRY WITHOUT CHECKING THAT OPTION - that might do what you need).

So, I would try going to Word, go to the Mailings tab, click Start Mail Merge > Letters, then Insert tab > Text group > Object > Create from file tab > and then browse to the file you want to attach.  Write a message to go in the message, use the Greeting Line button in the Write & Insert Fields group to add a greeting line.

Go to the Finish group > Finish & Merge > Send e-mail Messages.

I could only get it to work (without turning the attached/embedded file into a picture of an icon!) by using the "Mail Format: Attachment" option.  This sends the Word document as an attachment to an e-mail, and IN that attachment is embedded your other file.  Make sense?

If you want to make this "cleaner" and send a plain text message WITH a proper attachment, then I think getting to grips with the VBA is your only option.

This link (in addition to my previous post) may help with that: http://word.mvps.org/FAQs/MailMerge/MergeWithAttachments.htm

Post back if you have any further questions.
