You can control the appearance of the field results by adding a \* charformat switch and applying the required format the first letter of the field.

charles kenyon say:

Please review the responses you have already received. They answer your question. Summarizing:

1. They come out as bold because they were formatted as bold in the original text.

2. Instead of inserting { bookmarkname } to replicate your data, use { REF bookmarkname \* charformat }. This is a formal cross-reference field. In many circumstances, it will work the same as simply naming the bookmark; in some circumstances it will give a better and more predictable result.

You will find the cross-reference command on the references tab. You will still need to edit the created field to add the "\* charformat" switch.
