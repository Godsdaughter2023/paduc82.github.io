john korchok say:

You can't change the default paste in PowerPoint for Mac. You can try trashing the preference files to see if that will reset the default paste:

  1. Close all Office programs.
  2. Hold down the Option/Alt key while clicking on the Go menu and choosing Library. Your user Library folder opens.
  3. Open ~/Library/Preferences and drag com.microsoft.PowerPoint.plist and com.microsoft.Office.plist to the desktop. Office will rebuild them automatically.
  4. Restart PowerPoint and see if your default paste is any better. If it is, you can trash the files you moved.

If that doesn't restore it, you can work around it by adding a normal Paste button to the Quick Access Toolbar:

  1. In PowerPoint, choose click on the down-pointing arrowhead at the right end of the Quick Access Toolbar.
  2. Click on More Commands...
  3. Change the Choose commands from: dropdown to All Commands.
  4. Choose from 13 different Paste buttons. For your purpose, the most likely candidate is Paste. Select it and click on the right-pointing arrowhead button between the 2 lists. The command is added to the right-hand list.
  5. Save out. The new icon is added to the QAT.

Without seeing your source and destination documents, I can't tell why text is bolded or has double carriage returns.

Publisher 2013 Master pages - Microsoft Community

trinitycentre say:

Publisher 2013 Master pages

I use Publisher 2013 to publish a magazine.  The pages have two basic formats that I use: one with  text box for the page heading and a text box with two columns for the body of text.  The other page contains text box only with columns.  Every time I compile the magazine I have to format each page s required.  I thought that I would create master pages with blank text boxes (with heading and without heading) to use as required.  Fine so far, but I cannot seem to be able to apply those pages when I want them.  All tutorials are great on creating Master Pages but not so great on how to apply them.  Any help would be greatly appreciated.
davidf2 say:

A Master Page is used to display page objects in the background on all pages or at least multiple pages. If you insert text boxes on the Master Pages then you have to go to the Master pages to edit them. I don't think that is your best approach.

I would suggest instead that you create Building Blocks of your formatted text boxes and then you can insert them as Page Parts.

Open one of your magazine publications. Go to a page with your page heading text box and two column body text box where you have it formatted as you like. Select all three and Group them by Home tab > Arrange group > Group. Now you can right click the text box groups > Save as a Building Block > give it an appropriate Title, description and for now just save it in the General category. Do the same thing for your two column text box group.

Now you create a new publication or insert a blank page and then Insert > Building Blocks group > Page Parts and click the drop down arrow and under Recently Used or General you should see the two building blocks you created. You can also click the More Page Parts to see more detail. Once you insert your grouped text boxes/page part you can drag them into place. When you are ready to edit your text just click in the text box > Ctrl+A > start typing.

Reference: Use building blocks to quickly create a publication in Publisher: https://support.office.com/en-US/article/Use-building-blocks-to-quickly-create-a-publication-in-Publisher-a3fa9062-0378-426d-a956-3ef7b1fba0b4

Alternatively you could go old school like I would and just pull up an old magazine Publisher file, do a Save As a new file name and start editing that magazine 'template'.
