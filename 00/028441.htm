I'd be glad to assist you with your issue on delayed email delivery.

 

Mail delivery delays are usually caused by one of the following issues:

 

  • The mail servers may be undergoing maintenance, or there may be a service outage.
  • The message was large, or it contained one or more large attachments.
  • Messages that are sent from Windows Live Hotmail, MSN, or Hotmail accounts are subject to the same spam filtering as incoming mail from third-party email services.

These causes can create delays of up to several minutes.

 

Please consider the following:

 

  • Because large attachments or multiple attachments to messages may significantly delay delivery time, use compression software to compress large or multiple attachments. In addition, send fewer attachments in each message.
  • Send a test message to yourself. This method is a good way to ensure that you can send messages because a message that you send must travel to a main server, be sorted, and then return to your inbox.
  • If you send a message that is delayed, check the recipient's email address. If you have a message from that recipient, reply to that message to make sure that you have the correct email address for the recipient.
  • If you are the intended recipient of an email message that is delayed, send a message to the sender and ask the sender to reply to your message to make sure that the email address is correct. Tell the sender to add your email address to the sender's contacts list.

brian tillman [outlook mvp 2007-2019] say:

Enable viewing hidden files and folders.  Then search your old hard drive for the PST file.  The old location default would have been x:\Documents and Settings\username\Local Settings\Application Data\Microsoft\Outlook for Xp or x:\Users\username\AppData\Local\Microsoft\Outlook.  Copy it to the new C drive to, say, Documents\Outlook Files.  Then use the Mail applet in Control Panel to add it to yor mail profile and set it as the default data file.  Define your account(s) if not already done.  Open Outlook and it will contain the exact same data it did on the old PC.
