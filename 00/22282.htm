  1. Open Excel to a blank worksheet and copy your list of names to column A.
  2. Place the following formula in B1 and copy or fill down as necessary:
         =TRIM(RIGHT(SUBSTITUTE(A1," ",REPT(" ",99)),99))
    This should result in something like the following,
                         
  3. Highlight the entire range of data (e.g. A1:B4 in this example) and Select Data then Sort from the Sort & Filter group.
  4. Change the Column, Sort by to Column B and click OK in the lower right.
             
    The result should look like this,
                         
  5. Copy column A back to your Word list/table.
If you have more columns of information than just the names, bring those along with the names to Excel and adjust the column references in the example as necessary.

link an entire excel spreadsheet from one workbook to another in the - Microsoft Community

jhelpa say:

link an entire excel spreadsheet from one workbook to another in the same folder.

I would like to automatically link an entire spreadsheet from one workbook to another. Another person creates a weekly excel spreadsheet containing raw data. I would like to link  to this raw data and do my own pivot tables and formulas. I would like to have the data refreshed automatically when I open my  workbook containing the pivot tables and formulas. How do I link an entire shreadsheet from one workbook to another? I am using excel 2007

Any help would be much appreciated. Thanks.

bernie deitrick, excel mvp 2000-2010 say:

Open both workbooks, and in cell A1 of a blank sheet in your workbook, type an equals sign, and then navigate to the other workbook/worksheet with the data and select cell A1. Press F4 until the $ signs go away, and then press enter. Then copy that cell down and across for as many cells as you need to capture all the values that contain raw data. Then close the data workbook, and your linking formulas should change to include the filepath and filename.

If there are blank cells, you will get zero values - if that is bad for you, change the formula to

=IF(linkformula  = "","",linkformula)

where linkformula is the formula created in the steps above, minus the equals sign.

You could also use a macro to import the data, but that is not automatic unless you set up your workbook to re-rum the macro automatically when opened.....
