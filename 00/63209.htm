basara549 say:

If you actually have a stub of a System partition, that's just too small (mine was 33 MB coming from Win7, others with 8.1 have reported 100 MB), the upgrade won't go through.

It seems that for some, it can expand the partition (possibly even grabbing the space from another partition on the drive, or unallocated space), but for others will not (like me)

If you go into "Manage" for your drives (found by right-clicking the Computer icon on your desktop, that you normally left-click to look at drive contents), You will find an option for "Disk Management"

Going there, you can see the drive partitions. If you can get the System Partition to take the "expand partition" command by clicking on it and choosing from the menu, expanding it to 350 MB should allow the Upgrade to run.

If you can't get it to let you expand the partition, even if you shrink the other partitions to make room, then you'll need to resort to 3rd party software to expand the partition. This can be something from the drive manufacturer, or downloaded from cnet. Shrink the non-system partition (which you should be able to do with "manage") by 350 or so MB, and use the other software to expand the system partition. Most of these partition software fixes will then reboot, run some software outside windows to execute the fix, then put you back into Windows after a couple more reboots. The upgrade should then be able to run.

This is a lot safer than doing a bunch of command prompt commands, and allows you to do the hot upgrade, as opposed to a clean install or having to fully reformat drives.

If there's no System partition whatsoever, I'm not sure how much of this will be applicable.

Hi,

Some Registry keys for local or group policy may have been changed due to some circumstances, causing greyed out LAN Settings on the Internet Options. To troubleshoot this issue, follow these:

If it occurs on every user on your PC:

  1. Press Windows + R to open Run.
  2. Type regedit and press Enter to open the Registry Editor.
  3. Go to HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Internet Explorer\Control Panel
    DWORD: Proxy (Change from 1 --> 0)
    DWORD: Connection Settings (Change from 1 --> 0)

If it occurs on a specific user:

  1. Press Windows + R to open Run.
  2. Type regedit and press Enter to open the Registry Editor.
  3. Go to HKEY_CURRENT_USER\SOFTWARE\Policies\Microsoft\Internet Explorer\Restrictions
    registry keys.
    DWORD: Proxy (Change from 1 --> 0)
    DWORD: Connection Settings (Change from 1 --> 0)

Let us know how things go.
