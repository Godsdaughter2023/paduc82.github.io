Hi,

 

With the search word/phrase in C1 try this which is case sensitive

 

=SUMPRODUCT((LEN(A1:A20)-LEN(SUBSTITUTE(A1:A20,C1,"")))/LEN(C1))

 

or for a non case sensitive count.

 

=SUMPRODUCT((LEN(A1:A20)-LEN(SUBSTITUTE(UPPER(A1:A20),UPPER(C1),"")))/LEN(C1))

 

EDIT...Note that this is not bulletptoof and could get tripped up by the search string appearing within another search string. For example search for 'hop' and it would be found within 'Shop'

 

I'm not too sure that there is a robust formula solution, there would be using VB code.

mike h.. say:

Here's a VB solution that seems pretty robust to me.

 

ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in. Close VB editor.

 

Back on the worksheet call with this

 

=countstrings(A1:A11,C1,1)

 

where

 

A1:A11 is the range to search

C1 contains the search srring

the last number 1 makes the search case sensitive, change this to zero ( 0 ) for a non case sensitive search.

 

 

 

 

Function countstrings(rng As Range, str As String, cse As Boolean) As Long
Dim c As Range, r as string,x as long
r = " " & str & " "
For Each c In rng
If cse = FALSE Then
    For x = 1 To Len(" " & c.Value & " ")
        If Mid(" " & UCase(c.Value) & " ", x, Len(r)) = UCase(r) Then
        countstrings = countstrings + 1
        End If
    Next
    Else
    For x = 1 To Len(" " & c.Value & " ")
        If Mid(" " & c.Value & " ", x, Len(r)) = r Then
        countstrings = countstrings + 1
        End If
    Next
    End If
Next
End Function

I would suggest VBA code using Regular Expressions.

This code assumes that your phrase is a sequence of words, that does not start or end with a character that is not in the set of letters, digits, underscore.

 

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like 

 

=CountPhrase(Phrase, Range to Search, [Case Sens])

 

The optional CaseSensitive argument defaults to False

 

in some cell.

 

=========================================
Option Explicit
Function CountPhrase(Phrase As String, Rng As Range, Optional CaseSens As Boolean = False) As Long
    Dim re As Object, mc As Object
    Dim v As Variant
    Dim i As Long, j As Long
    Dim lCnt As Long
Set re = CreateObject("vbscript.regexp")
With re
    .Pattern = "\b" & Phrase & "\b"
    .Global = True
    .ignorecase = Not CaseSens
End With

v = Rng
For i = 1 To UBound(v, 1)
    For j = 1 To UBound(v, 2)
        Set mc = re.Execute(v(i, j))
        lCnt = lCnt + mc.Count
    Next j
Next i

CountPhrase = lCnt

   
End Function
