luettjelage say:

Hi,

Sorry for my late reply (I saw someone had the same problem in April). After opening an incident ticket with the Microsoft Office365 Support we finally fixed the problem.

The following steps helped me. But be aware: After completing those steps your Outlook is completely fresh (= all mailboxes/ calendars, etc. get lost during this process)! In case you (like me) have everything stored online (in some sort of exchange environment) this should not be a problem as Outlook syncs everything back after you finally reconfigured your accounts.

What we did:

1. Go to Macintosh HD > Users > [User Name] > Library > Group-Containers.

You reach the library on mac by opening finder, holding alt and using the "GoTo" function from the menu bar. By pressing the alt key, the link to "Library" will appear.

2. Delete the three folders starting with "UBF...". Those folders contain all Outlook settings. This is why Outlook comes back completely fresh/ blank (see my warnings above!)

3. Rebuild the main identity using the Microsoft database utility

4. Startup Outlook (should now work).

Many thanks for that solution to the Microsoft technician who helped me. He did a great job!

Good luck to you.
