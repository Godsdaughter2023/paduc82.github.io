rjbayha say:

BackColor setting for UserForm, Frame, and MultiPage Pages for Mac Excel 2011

I have an iMac with Mac Excel 2011. I have developed programs that work with Windows Excel 2000 thru 2010, and Mac Excel 2004. I am in the process of updating the programs to also work with Mac Excel 2011.

 

The annoying issue with 2011 is the BackColor parameter for UserForm and Frame ignores the light grey color (&H000000F&) used as the default color for all previous Excel versions: instead, Excel 2011 uses dark grey. MultiPage Pages uses the same dark grey color, and there is no BackColor setting for Pages in all versions of Excel. All labels and other items retain the light grey setting.

 

1. For Userform and Frame, I set the BackColor parameter in the UserForm_Initialize procedure using the BackColor setting of label.

 

Private Sub UserForm_Initialize()

    UserForm.BackColor = Label4.BackColor

    Frame1.BackColor = Label4.BackColor

    

    ... other code as desired ...

 

End Sub

 

2. For MultiPage Page, I add a label to each Page having a size larger than the page size, and set the Left and Top settings to -1.

 

   a. For a new-blank Page with the label background, simply start adding the items to the page.

 

   b. For a page with items already present, send the label backwards using the VBA Editor Format | Order | Send to Back or Send Backward until all items are visible.
