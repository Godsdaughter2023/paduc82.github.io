Why do envelopes print in red ink in MS Word 2007? - Microsoft Community

leslieharrington say:

Why do envelopes print in red ink in MS Word 2007?

Envelopes are printing in red, even if I change the color in Options.  In the Mailing tab In MS Word 2007 or as a letter document with an envelope, the envelope prints in red.  What is the problem.  I recently upgraded my system but reinstalled Windows 7 and Office 2007.

jay freedman say:
I'm not sure what Options you're referring to. The symptom indicates that you have an incorrect definition for the Envelope Address style. To confirm this (or not):
  • Go to Mailings > Envelope, enter anything in the address box, and click the Add to Document button.
  • In the document, put the cursor in the text of the address.
  • Press Ctrl+Alt+Shift+S to open the Styles pane. The Envelope Address style should be shown with a blue box around it.
  • Hover the mouse pointer over the Envelope Address name in the Styles pane. The popup shows the definition. If the Font section of the definition includes "Font color: Red" (or any other color), that's the problem.
If there is a color assignment in the definition, right-click the Envelope Address style name and choose Modify. Open the color dropdown under Formatting and choose Automatic. Click the option button near the bottom for "New documents based on this template" and then click OK.

If there is no color assignment in the style definition, try modifying the style by changing the color from Automatic to black (these are not quite the same thing). We've heard of cases, possibly depending on the specific printer driver, in which "automatic" black on envelopes is printed by overlapping dots of all three colors, and one of the ink cartridges has run out or is clogged.

bill manville say:

The first thing to try if Excel crashes is to see if the same happens in Safe Mode.

Hold down Ctrl while launching Excel.

If the same does not happen in Safe Mode then the problem is caused by some add-in or program loaded at startup.  Disable all add-ins (File > Options > Add-ins) and then try again running excel normally. As long as the problem does not recur then you can re-enable add-ins you need for your work.

But if the problem still occurs in Safe Mode, there are a number of steps you can try:

- Control Panel > Programs & Features > Microsoft Office 2013 > Change > Repair
- Change your default printer to something like the Microsoft Office Document Writer (in case the problem is caused by the printer driver)
- Tools > Options > Advanced > Display: Disable hardware graphics acceleration (which has been known to cure some issues if there is a problem with the screen driver).

If none of the above solve the problem, please come back.
