wunders say:

>> My issue is as follows: when I run an ipconfig /all from either of the desktops,
>> my Node Type = Unknown; my DNS Suffix Search List is blank

This does not affect the problem you're having.  "Node Type" comes into play with Microsoft File and Printer sharing.  If you don't have a problem sharing files or printing to a printer connected to another machine, then I would not worry about this.

>> my "Connection- Specific DNS Suffix" is blank as well.

It is probably normal for this field to be blank.  If your ISP registers your connection in DNS or you need this to connect to some ISP-supplied resources then it may have something in it.  It does not affect your problem.

>>DNS Suffix Search List is blank

This is normal too.  This is only used to resolve non-fully qualified domain name lookups.  It provides suffixes to append to an IP name with no '.'s in it.

>> my router config page is set, under "Basic Setup," as Automatic-DHCP.
>> About once a day my Internet Connectivity will lapse- I navigate to my Linksys->Status page and my DHCP fields are 0.0.0.0.
>> I have to power cycle the Comcast modem and that usually populates the fields with WAN IP addresses,
>> as it should, and I am back with internet capabilities

The problem you are having is between your Router and your Modem.  It has nothing to do with your Windows machine.  When your Router requests a DHCP address from the modem for its WAN connection, the Modem provides that address and a 'lease' time that the address is valid for.  Apparently in your case, that lease period is one day.  On some routers, one of the status pages will display either the time left in the lease or the lease expiration date/time.  When that lease is approx 1/2 over, the Router should begin to request a renewal of that IP address from the Modem.  This is usually a quick transaction that will extend the period of the IP lease for another lease period (one day in your case).  For some reason, this does not appear to be happening for you and your IP lease ends up expiring on the Router's WAN port and goes to 0.0.0.0.  When you reset your Modem and Router, the Router successfully negotiates another lease from the Modem and you're good to go until that lease expires.  When you first reset things, note the Lease period. When that period is 1/2 over or more, status the router again to see if the lease has been extended.  If it hasn't and runs out, you will be disconnected from the Internet.  This problem has to be worked out between your router manufacturer and your ISP.  You may have a defective modem.  Make sure your router is not set to block ports 67 or 68 (needed to renew IP leases).

Hope this helps,
   JW

vijay a. verma say:
1803 is the latest version not 1709. You should upgrade to 1803 manually by following below steps.

1. Right click on Windows Start > Windows Powershell (Admin)
Type following (or copy and paste following) one by one and Enter

net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
Ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
Ren C:\Windows\System32\catroot2 Catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver

2. Type following in Windows Start Search box and hit Enter %systemroot%\Logs\CBS > Rename CBS.Log to something else.
If it doesn't allow you to rename - Type services in Windows Start Search box > Click Services > Scroll down to find the Windows Modules Installer service and double-click on it > Set its startup type to Manual > Reboot your computer
Go to C:\Windows\Logs\CBS > Rename CBS.Log > Reboot your computer > Then go back into the Services App and set Windows Modules Installer Service back to Automatic Startup (Note CBS log files contain information about windows update, SFC etc)

(Source - https://partnersupport.microsoft.com/en-us/par_...)

3. You can go to https://www.microsoft.com/en-us/software-downlo...

3.1 Click Update now to update to 1803 through Upgrade Assistant.
OR
3.2 You can download the latest Windows 10 package from this link (media creation tool) and follow the instructions there to upgrade. Use the very simple instructions given under Using this tool to upgrade this PC to Windows 10
