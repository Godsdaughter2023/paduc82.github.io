bill manville say:

The criteria blocks for DMAX etc. are inflexible.

I would use an array formula instead.
Assuming you want the maximum value from C2:C2000 that corresponds to a value of A2 in B2:B2000:
=MAX(IF($B$2:$B$2000=$A2,$C$2:$C$2000,""))
entered as an array formula using Ctrl+Shift+Enter


Copy down the remaining cells.

 

Alternatively you could create a pivot table from B1:C2000 making the column B field a row field and the column C field a data field with the operation being MAX.
