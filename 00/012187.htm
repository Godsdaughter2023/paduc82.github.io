How to maximize the ribbon in access every time a report opens - Microsoft Community

alfredo salvadora say:

How to maximize the ribbon in access every time a report opens

 

Problem

I have a report with customized ribbons.  There are times that when I open the report, the ribbons are minimized. 

 

Required Solution

I need the ribbons always maximized every time I open the report.

 

 

Thanks in advance for the help.

 

Alfredo

 

albert d. kallal say:

You can in code the ribbon with this:

To hide the ribbon, use this:

 

Public Function HideReportRibbon()
   
   Dim intRibbonState      As Variant
   
   intRibbonState = CommandBars("ribbon").Height
  
   If intRibbonState = 145 Then
      CommandBars.ExecuteMso "MinimizeRibbon"
      DoEvents
   End If

End Function

 

And to show (MAXIMIZE) the ribbon, use this:

 

Public Function ShowReportRibbon()

   Dim intRibbonState      As Variant
   
   intRibbonState = CommandBars("ribbon").Height
   
   If intRibbonState = 57 Then
      ' ribbon in auto hide state, show it...
      SendKeys "^{F1}", False
      DoEvents
   End If

End Function


So to launch a report and show the ribbon, we go:

 

    DoCmd.OpenReport stDocName, acPreview
    DoEvents
    ShowReportRibbon

 

You might want to "grab" or save the current ribbon setting and thus perhaps  minimize it when you close the report, so in the close event you could go:

 

HideReportRibbon

 

So on report exit re-hiding (minimizing) the ribbon is optional, but even better would be to grab the ribbon state before you launch and leave the ribbon maximized if the user already has ribbon maximized.
