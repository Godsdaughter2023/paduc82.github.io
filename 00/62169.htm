jeeped say:
The Windows 7 Compatibility Center reports full compatibility with the Canon S400 Color BubbleJet printer in both 32-bit and 64-bit Windows 7 (full report here) with no driver download necessary. Since you are having trouble, locating a 64-bit Windows 7 driver would seem to be necessary.

                                                    

The current drivers and support software from the Canon Support site on the S400 page here do not cover Windows 7 64-bit (or Vista 64-bit for that matter), but I was able to locate an appropriate driver from Windows Update. Follow these steps for installation:

  1. With the printer hooked up and turned on, open Devices and Printers from the right side of the Start menu or the Control Panel.
  2. Click Add a Printer from the commands across the top of the Devices and Printers window.
  3. Click Add a local printer from the first page of the Add Printer window. Click Next.
  4. Select an appropriate port. If this is confusing or your USB does not appear listed, select Use an existing Port, LPT1. Click Next.
  5. Click Windows Update and wait several minutes to the lists to refresh and repopulate. Note: it will not appear to be doing anything but it is. Be patient.
  6. When the lists have refreshed select Canon from the left side and Canon Bubble-Jet S400 from the right. Click Next.
  7. Agree to the proposed name or type a new one. Click Next.
  8. The printer software/driver will install. When it is completed you'll have an option to print a test page.

If that doesn't get your printer operational, it is likely due to the wrong port selection. The good news is that the appropriate drivers are on the system now. Turn off your printer and delete the entry from Devices and Printers. Restart your computer then turn the printer back on. A USB connection will locate the appropriate drivers now residing on your system and install them to the connected port.
