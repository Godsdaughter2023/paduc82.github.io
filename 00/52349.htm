    christianrc say:

    My Ethernet connecting/disconnecting all the time drove me insane. I use windows 10/home.

    This is how I eventually solved it:

    • Right click the Start icon.
    • Select Device Manager.
    • Select Network adapters.
    • Right click on  your Ethernet controller (Mine is Realtek PCle FE Family controller)
    • Right click on Properties
    • Click on Advanced
    • Under Property click on Speed & Duplex
    • Under Value you will probably find Auto Negotiation, replace it with 100 Mbps Half Duplex
    • Click OK
    • Close Device Manager

    That did it for me.

    7. In Property-List Type Inventory section you can find the program path ("lower case long path" XML-attribute).

    9. Repeat these steps, for all founded strings with BlockingApplication=True value.

    shenan stanley say:

    Unfortunately - most uninstallers are very untidy and leave behind bits and pieces - probably more than entries in the registry.

    You can uninstall Windows Movie Maker if you like.  After all - it was installed with Windows Live Essentials or similar.

      

    As for Java - you or something you installed likely installed it - if you mean Java Runtime Environment (JRE), the web plugin.  You can uninstall it too - see what breaks and install the latest version if you want.

      

    As for the "Visual Studios" <-- no idea what you mean there.  You may have a Microsoft Visual C++ Redistributable installed that got there with some other installation and was kept updated after that by Microsoft Updates...?

     

    As for the sidebars and gadgets - as long as you got them turned off, I wouldn't do much more with them.  Although - you can uninstall them too, at least their base.

    Go through Control Panel --> All Control Panel Items --> Programs and Features to uninstall things first - that you don't think you need and/or don't want.  I would be careful here - many things get installed because your video card device driver and software need them or some other component, application, drive, etc. that was installed required them to function and there is no NOTE of that nor is there much of any warning that removing it will break that.

      

    You can choose "Turn Windows features on or off" from the left side to perhaps remove some more of the fodder that is installed you do not want...  Individually remove the default Games for example, expand the Media Features section and remove Windows DVD Maker if you don't use/want it, expand Print and Document Services and remove Windows Fax and Scan if you have no scanner/fax system, remove the Windows Gadget Platform and maybe the Tablet PC Components if you don't use the Journal, Math Input panel or handwriting...

      

    At least then you are removing things safely.

      

    Honestly - you stating, "I want my registry free of unnecessary files..." gives me pause - since there are no files 'in' the registry.  ;-)  (Old joke...  *GRIN*)

      

    If you (after doing the above) want to find some orphaned registry entries and want to be very careful about which ones you decide to remove - although I don't recommend registry cleaners - you could use CCleaner (the FREE version) to find the orphaned entries, see a list of them, select what you want to remove and export the pre-cleanup state (although I recommend a system image backup before you start) to do so...

      

    Note that the cleanup of the registry will likely do very little for space, nothing for performance of the system and is a risk.  It is merely for your own edification that you would be doing this, the benefits being almost strictly mental.

      

    For the most part - it's best for most people that they DON'T mess with what is installed and DON'T worry over the space on their drive unless it is starting to get very low (under 15% for small drives of 80GB to 160GB or below and less than 20-40GB for any drives over that size, IMHO.)  A little cleanup in the wrong place can have dire consequences and any application or individual that tells you they can walk you 100% safely through such a cleanup without seeing it themselves - is probably playing on your need to clean up and hoping for a good outcome *at best*.
