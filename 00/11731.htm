ariel acu say:
Hi,

When using your computer and working on something like editing a Word file, you are using the RAM at the same time. RAM is a temporary storage used by your computer processing, managing and using apps. Regarding your concern, we cannot keep a Word file within the RAM because once you click on Save in a Word document, for example, the operating system takes the information that is in RAM and writes it to a file on the hard drive by default. If you're still working on the document file, it also means that you're working with RAM. If you didn't save changes made in the file, RAM will only read the file and show the last saved changes made on it. I hope this information gives you an idea about how RAM works on your computer and why it can't save Word files in RAM.

Let us know if you need further assistance.

charles kenyon say:

Once you have Word open, you can close the current document without exiting Word. (The default, using the close X, is to close and exit if only one document is open.)

Ctrl+W closes the current document without exiting Word.

Alt+F, followed by B closes the current document without exiting Word.

File > Close closes the current document without exiting Word.

This is not a complete answer, but does keep Word loaded as a program for when you next open a document. You need the Word Window to have the focus for Ctrl+N to create a new blank Word document.
