lemp say:

If you have a computer that came with Windows pre-installed by the manufacturer, that copy of Windows has an "OEM license."  For Windows 7, the evidence of your license is a COA sticker fastened to the computer.  The 25-character Product Key shown on that COA is used to "activate" Windows.

Microsoft sold OEM-licensed copies of Windows to computer manufacturers at a reduced cost (to keep the final cost of the computer lower).  In return, the computer manufacturer must provide support (rather than Microsoft) and -- most importantly -- the license is valid ONLY for the computer on which it was first installed.  That means when the computer dies, the license dies with it.  An OEM license is NOT transferable, regardless of the fate of the original computer.

If you purchased a copy of Windows 7 from a retail vendor (i.e., what you called a "distribution copy"), the Product Key was on the DVD's packaging.  The main advantage of a retail-licensed copy of Windows 7 is that it can be used on any computer that meets the system requirements, as long as it is only used on one computer at a time.

Because the "machine that you use most came with Windows 7 installed," it has an OEM license and you can not use the Product Key on the COA affixed to that computer to install Windows 7 on another computer.

If you bought what you're calling a "distribution copy" of Windows 7 and installed it on your old decommissioned desktop computer, then you CAN use the Product Key from that copy to install Windows 7 on the new laptop that came with Windows 8.1 -- if Toshiba has Windows 7 drivers for that laptop, which it may not. 

Do not use the "recovery disk" from that old computer.  Rather, download an ISO file and create either a bootable USB flash drive or a bootable DVD.  You can use the retail Product Key to download the ISO file here:  https://www.microsoft.com/en-us/software-download/windows7

If you want to create a bootable USB flash drive, you may find it easier to use (free) Rufus with the ISO file:  https://rufus.ie/en_IE.html

If you want to create a bootable DVD from the ISO file, I suggest (free) ImgBurn:  http://www.imgburn.com/

These are the basic rules in my words:

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.
