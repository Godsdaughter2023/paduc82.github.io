hansv mvp say:

You could enter the formula you want to use for data validation (including the UDF) in another cell.

Then refer to that cell in the Formula box in Data Validation.

For example: you want to apply data validation to cell B2.

In another cell, say Z2, enter a formula using your UDF that returns TRUE if the value of B2 is OK, and FALSE if the value of B2 does not meet your criteria.

Select B2, and activate the Data Validation dialog.

Select Custom, then enter the formula =Z2 in the Formula box.

You can hide the column with the auxiliary formula if desired.

That is not caused by the UDF itself, but by the fact that we have to rely on the value of another cell (E2).

If you enter an incorrect value in D2, then immediately click in another cell, the value of E2 hasn't been updated yet, so Data Validation doesn't notice that the value is incorrect.
