I don't suppose you remember the files you deleted or the "malware software" you used?

I would start by uninstalling and reinstalling your wireless adapter.

If you have an external wifi adapter, go to the manufacturer's website and download the latest driver for it.

If you have an internal wifi adapter (most likely in laptops), go to the computer manufacturer's website and download the latest driver for your wifi adapter.  Most computer manufacturers will provide a link to the correct driver after you identify the specific model of your computer (in Dell's case, use the "Service Tag").

Save the driver someplace that you'll be able to find it (on a USB flash drive, if you used a different computer)

Open Device Manager (Start > Run > devmgmt.msc > OK)
Expand the "Network adapters" category by clicking the +
Right-click on the entry for your wireless adapter and select Uninstall.
OK your way out and reboot.

If the Found New Hardware Wizard does not open when Windows reboots and discovers the device, go back to Device Manager, right-click on the entry for the wifi adapter and select Update Driver
On the Update Hardware Wizard screen, do not let it connect to Windows Update, but select "Install from a list or specific location"
Click "Search for the best driver" and also "Include this location."  Click the "Browse" button and browse to the folder where you saved the downloaded driver (unzipped already, if you downloaded it in zipped form).  Make sure to select the WinXP subfolder (or similar name).
Click Next and the adapter should install.

Assuming that the above results in an icon for your wireless adapter appearing in Network Connections, there's still a bit more to do.

Right-click on the icon in Network Connections and select Properties.

Make sure that the "General" tab is showing.
In the box under "This connection uses the following items," you should have 4 entries:
Client for Microsoft Networks      {necessary}
Internet Protocol (TCP/IP)            {necessary}
File and Printer Sharing for Microsoft Networks      {necessary if you want to be able to share files and printers}
QoS Packet Scheduler                {optional, but usually present}

If those items are not listed, make it so, using the Install button and subsequent dialogs.

Still in the wireless network properties dialogs, click the "Wireless Networks" tab.
Ensure that the box to "Use Windows to configure my wireless settings" is checked UNLESS you have a manufacturer's utility doing this job.
OK your way out
