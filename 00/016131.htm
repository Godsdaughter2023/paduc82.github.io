Nic --

 

The fact that your formatting changes do not hold when toggling the Slack option on and off tells you that Microsoft has pre-defined styles that you cannot change permanently.  The way around this is to make a copy of the view you want to reformat, select the checkboxes you want on the Format ribbon, and then format the bar styles the way you want for the checked items.  From there, NEVER uncheck the checkbox options for Critical Tasks, Slack, and Late Tasks.

quietman7 say:
MRU-Blaster is an older program created in 2002 by Javacool (now called Brightfort), the creators of SpywareBlaster. As PrashantKumar96 indicates, the program is for detecting and cleaning the MRU lists which contains information about the last files you accessed. When released it was primarily intended for Windows XP and older operating systems but worked on Vista if Run as Administrator. The system requirements page was updated in 2011 to reflect Windows 7 compatibility and should run on both 32-bit and 64-bit versions.

If your concerned about MRU lists, you can use a program like CCleaner which cleans not only Windows MRU keys but also many third party applications.

W32.Blaster.Worm (Win32/Msblast) is something different altogether...it is malware and it is an older infection that targeted a security issue related to the Remote Procedure Call (RPC) function. Symptoms of infection included the computer restarting every few minutes without user input or receiving a System Shutdown dialog box with the message: "This system is shutting down. Please save all work...This shutdown was initiated by NT Authority\System...Windows must now restart because because the Remote Procedure Call [RPC] service terminated unexpectedly." Microsoft addressed this vulnerability with a security update several years ago.

Show or hide file name extensions

A file name extension is a set of characters added to the end of a file name that determine which program should open it. For more information, see File names and file name extensions: frequently asked questions.

Windows hides file name extensions, but you can choose to make them visible.

To show or hide file name extensions

  1. Click to open Folder Options.

  2. Click the View tab, and then, under Advanced settings, do one of the following:

    • To show file name extensions, clear the Hide extensions for known file types check box, and then click OK.

    • To hide file name extensions, select the Hide extensions for known file types check box, and then click OK

PowerPoint merging music and speech simultaneously. - Microsoft Community

ardiaperry say:

PowerPoint merging music and speech simultaneously.

I am presenting a Black History PowerPoint. I am trying to play the I Have A Dream speech by Dr. King along with Lift Every Voice instrumental.

Any suggestions or is this possible?

john korchok say:

You can try it, the effect can be unpredictable. Sometimes just one file will play, sometimes the second will kick in late.

  1. In PowerPoint, click on the Home tab
  2. In the Insert group, click on Media, then on Audio Browser or Audio from File. Audio Browser is better if the tracks are already in iTunes, Audio from file is better if you just have a sound file.
  3. In Audio Browser, drag the tracks onto the slide.
  4. Set the options on the Format Audio tab. If the tracks are to continue over several slides, set Start: to Play Across Slides.
  5. Once you have both tracks on the first slide, test the presentation in Slide Show mode to see if they play together.

This is easier to do if you mix the two files together before placing them in PowerPoint. Then they will always stay in sync. You can use a free program like Audacity to mix the two soundtracks:

http://audacity.sourceforge.net/download/mac

Mixing music with speech can make the voice harder to understand. I would suggest that the music be quieter than the speech.
