Sub changecompanycode()

Dim LR As Long, r As Long, i As Long
    
    With Worksheets("Sheet1")
        LR = .Range("A" & .Rows.Count).End(xlUp).Row
        For r = 2 To LR

                .Cells(r, 2).Value = UserForm2.cmbboxState.Value Then 
                  .Cells(r, 1).Value = UserForm2.txtboxChangeCompCd
                i = i + 1
            End If
        Next r
    End With

john mcghie say:

Hi Jennyfer:

Word 2016 does some very strange things with files, and Office 365 is worse.

First you need to figure out where the file SHOULD be. Then you go look there: and if it is not there, you begin to wonder where it might be.

If the internet was up at the time, AND you saved this to your OneDrive folder, it SHOULD have replicated to the cloud and there should be a copy there.

If you had "Always create backup" turned on in Preferences, there should be a subfolder of the same name as the document in your folder, and the backup copy should be in there.

If you had a Time Machine drive connected at the time, there should be a backup copy in there.

However, if you did not have any of these backup mechanisms in use, then it's entirely possible that the file has gone forever.

You could try a search in Finder.  Pick a word that is likely to occur in the document in question (and unlikely to occur in other documents!). Using Advanced search in Finder is unnecessarily confusing in El Capitan: try this https://support.apple.com/kb/PH21933?locale=en_US&viewlocale=en_US

And you need to have a serious think about your backup strategies :-)  Yes, computers are becoming more reliable, but the value of the information they contain is also increasing rapidly.  Like death and taxes, computers WILL fail at some time, and when they fail, you either have a backup or you have a problem.

Hope this helps
