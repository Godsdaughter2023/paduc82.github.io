tucker [msft] say:

FefaTM,

We've been testing and examining your screenshots and I'm afraid I don't have any good news for you, yet.

While your screenshots show that you are seeing much more jagginess than we see in most inking, it doesn't look like it's a completely new issue. 

We do have a known problem that ink smoothing is not working really effectively in PPT editing.  The problem results from DirectInk actually being more precise and recording a large number of ink points as you draw.  When you draw quickly, the result is that the points are spaced more widely, which gives a smoother looking line, while when you draw slower and more precisely the points are much closer together, which accentuates the jagged appearance.  The result is that the harder you try to be precise, the more likely you are to see the resulting jagginess.

Slide Show doesn't yet use DirectInk, so the ink is actually lower fidelity and looks smoother.  There are a lot of technical improvements around performance and capabilities that makes DirectInk a big improvement, especially in editing mode, which is much more complex than Slide Show and higher performance demands.

As you've noted, OneNote has implemented smoothing that greatly decreases the jagginess.  The core Office apps use a different inking infrastructure than OneNote, so we have to do some deeper changes before we can implement the smoothing, but those fixes are coming.  The work is pretty complex, so it will likely be a couple months.

I'm sorry that I don't have a good, easy workaround for you in the meantime.  You could do any inking in slideshow mode or you could ink in OneNote and copy/paste the ink to PPT.  Neither is an ideal solution, but perhaps that can help until we release an update with improved ink smoothing.

For now the best answer I can give you is that we are aware of the issue and actively working on it. 

Thank you for your comments and for helping us get more details,

Tucker

The builtin Hour() function will return an integer 0-23 for the hour of the day:

SELECT Count(OrderID), OrderDate, Hour([OrderTime])
FROM yourtable
GROUP BY OrderDate, Hour([OrderTime]);

This will group midnight through 12:59:59am under 0, 1:00 thfough 1:59:59 under 1, and so on.

Method 1:

Release and renew IP address for your computer:

  1. Press Windows key + X, Select Command Prompt (Admin) and then press ENTER.
  2. At the command prompt, type the following command and then press ENTER:
    ipconfig /release
    ipconfig /renew
  3. Close the command prompt.
