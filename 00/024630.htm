Here are some troubleshooting suggestions ......

1. Run a full virus scan with the security program you have installed.

2. If the machine is clean, reset Internet Explorer settings.

Start > search box, type internet options > Enter > Advanced tab > at Reset Internet Explorer settings haeding, click Reset
