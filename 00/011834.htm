henry a jordan say:

On Click take unbound List Box Value and GoToRecord on visible bound Form

  

I have a Form which is tied to a Db Table. The table has a field labeled SORT. I have a Query which has data presented in different relationships controlled by a separate selection process. The Query is presented on the Form in a unbound List Box. How can a get the  Query List Box selected Value (value in Field "SORT LIST") to go to the  record on the Form which contains a like value in the table data field "SORT"?  

With a bound ListBox to the Form /Table the code works but with a unbound List Box the built in features with the wizard does not function.

How can I do this with a macro or VB (Use of a separate Command Button is OK)

scott mcdaniel say:

So you're selecting a value in a Listbox, and you want your form to go to that Record?

You can use the Bookmark method:

Dim rst AS DAO.Recordset

Set rst = Me.RecordsetClone

rst.FindFirst "Sort='" & me.YourListbox.Column(0) & "'"

If Not rst.NoMatch Then

  Me.Bookmark = rst.Bookmark

End If

Of course you'd have to change the Field and Control names to match those in your project. I assume that the first column in your listbox contains the value you want to search for. Also, I assume that "Sort" is a Text column, so I've enclosed it in single quotes. If it's numeric, the FindFirst line would look like this:

rst.FindFirst "Sort=" & me.YourListbox.Column(0)
