Hi Elizabeth,

We could first discuss with SharePoint Online permission and then check how to achieve your requirement.

In SharePoint Online, we have site collection, sites in this site collection, libraries or other apps in a site. Normally, library will inherit the permission from the site. It means if user A has edit permission for the root site of a site collection, he may also have edit permission to a library in this root site.

You mentioned that you have the library set up to Read Only, could you capture some screenshots about your set up steps? It may influence the sharing results. Normally, we could click the setting gear in the right top of library>Library Settings>Permissions for this document library>click Stop Inheriting Permissions>select users or groups in the permission list>click Edit User Permissions>change the permission level to Read. After these operations, click Check Permissions button>enter one user account email address>see if the account has permission higher than Read.
