hilary stoupa say:

One thing you could try (save a copy of your template first!) is to save as source files and modify the manifest. Rename the template from .xsn to .cab. Then open the .cab file, and copy all the files out into a new folder.

Open manifest.xsf up with a text editor. About the 10th line, you'll see the document class node: xsf:xDocumentClass

There are two attributes you need to change. From solutionFormatVersion="3.0.0.0" to solutionFormatVersion="2.0.0.0" and from productVersion="14.0.0" to productVersion="12.0.0". Save and close.

Right click the manifest.xsf file and select Design. Your form should now open in IP 2007. You can use File / Save As to pack the files back into a template.

unfukkinreal say: 1) download a working form from the forms library you are working in and look at the version by opening it as a text copy/or through SP Designer (e.g. - 1.0.0.56)
