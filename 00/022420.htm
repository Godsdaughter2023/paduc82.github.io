suzanne s. barnhill say:

My first guess, from your subject line alone, was that you might have inherited a document that used a template, once common for legal pleadings, that had fixed line numbers in a text box anchored to the header (and therefore accessible only when the header is open for editing). But the fact that you can delete the numbers individually suggests otherwise.

A screen shot would be helpful, but the main thing to establish is whether these are actually line numbers or paragraph numbers (if every line ends with a paragraph break, they could well look the same).

If they are paragraph numbers, applied by the Numbering feature, then they could be removed the same way. If they are actually typed numbers, you may still be able to remove them all at once. If they are in a single column to the left of all text, you can use column select to select them and then press Delete. To select a column of text, press the Alt key and the left mouse button at the same time (it may take some trial and error to get the timing right so as not to open the Research pane; just be sure you continue to press the mouse button so that a click is not registered), then drag the mouse to select the text you want to remove. If the column is at least two letters wide, this is fairly easy; it is very difficult to select a column only one letter wide but may be easier if you increase the Zoom.

jay freedman say:

One method is to use the Replace dialog. In the Find What box, type the code ^p^# and in the Replace With box type the code ^p. Click the Replace All button. Change the Find What box to ^p^#^# and repeat the Replace All. That will leave just the first line number to be deleted manually.

If that doesn't work for you, here's another method:

If the line numbers are all in a single column, with all the regular text to the right of them, you may be able to select that column (at least, as much as you can see in one screenful).

Press and hold the Alt key, then use the mouse left button to drag a gray select highlight around just the numbers.

This can be difficult; the highlight may disappear when you release the mouse button. Keep trying. When the numbers are highlighted, press the Delete key to remove them. Repeat as needed.

charles kenyon say:
Truly, I think you've gotten the only answer you will receive.
If this is a truly huge document and you have the original someone may be able to figure out something, but it will not be simple or straightforward.

Documents converted from pdf (or really any other format) to Word can be tough to edit because the conversion process never has a one-to-one matching of how formatting is done. This means that a converted document will seldom be formatted in Word in a way that makes sense. An example is multiple section breaks to change margins, where in Word you would simply change the paragraph indent.

If the conversion is smart enough to figure out these are line numbers it probably can be done. It sounds like your conversion just made the numbers part of the text. There are different ways to convert to Word and they can give different results.

don cuthbert say:

You can Stop Insider Builds at any time through Windows Update - Advance options.

The announcement for Build 10525 says this:

"Windows 10 Insider Preview is intended to be installed on genuine Windows 10 devices – before upgrading to Windows 10 Insider Preview Builds or reinstalling Windows 10 please ensure your device is activated. To confirm go to Settings > Update & security > Activation. With the launch of Windows 10 we introduced device based activation. If you’ve previously activated a genuine copy of Windows 10 on your device, you’ll be able to activate Windows 10 Insider Preview builds of the same edition without the need to enter a product key. This will also enable you to re-install the latest public release of Windows 10 without entering a product key if you no longer want Windows Insider Preview on your device."

Really the Windows Insider Program was never intended to allow users without a Windows 7 or 8 license for their device to install and test Windows 10 Insider Preview.  However, many users did exactly that, as they had old computers that they wanted to use as Windows Insider test devices that had no Windows 7 or 8 license.

Unless there is a very recent change, it is still possible to activate a new device in the Insider Program, by installing a Windows 10 Pro Preview build between 9926 and 10130 and activating it with the 10074 product key.  The device can then be upgraded to activated 10240 using a Windows 10 Pro ISO.  From there it can upgrade to an activated 10525.  We have been told that any Windows 10 installs without an original Windows 7 or 8 license will eventually expire and that this is not a path to obtain a license for Windows 10 on a device with no Windows 7 or 8 license.  However this does allow someone to activate their device as a Windows Insider device, subject to the license terms for the Insider Previews.
