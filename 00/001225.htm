hansv mvp say:

Let's say you want to refer to cell A1 on Sheet1:

=Sheet1!A1

If the sheet name contains spaces or punctuation, enclose its name in single straight quotes (apostrophes). For example, to refer to cell D25 on a sheet named My Sheet:

='My Sheet'!D25

Enter this in a cell and drag down for 3 rows to get the 3 smallest revenues.

=INDEX($A$1:$A$20,MATCH(SMALL($B$1:$B$20,ROW(A1)),$B$1:$B$20,0))

=INDEX($A$1:$A$20,MATCH(LARGE($B$1:$B$20,ROW(A1)),$B$1:$B$20,0))

Lily,

You have to set your default templates location in Microsoft Word. File - Options - Advanced - scroll all the way down to the bottom, there should be a button to set your file locations. Generally the location is set to:

When you save a template PPT should take you to that location automatically. To use your template go to File - New - My templates - your template will be listed there.
