Method 1 - Compatibility mode / Program Comptability Wizard

To try to get the Program Comptibility Wizard to fix the problem for you (automatically), search for "Program Compatbility Wizard" in Help and Support - and find the link that opens a page with a link to open the Program Compatibilty Wizard. Method 2 - DirectX Diagnostic Tool

It might be useful to open the DirectX Diagnostic Tool (to do this, type "dxdiag" in Open in the Run dialog box, and then click OK). When the DirectX Diagnostic Tool window opens, it should have lots of information about the DirectX and related software / hardware on your computer.

If there is / are any problem / problems, then have a look on the More Help tab, and see if there is anything you can do to fix the problem / problems.

 

Method 3 - Running program as administrator

Right-click the program icon (it might be on your desktop*), and then click on Run as... / Run as administrator (or something that says similar (I'm not certain of the exact phrase, but the aim of this method is to run the program through an administrator account)).

If a Run As dialog box appears, select an account that is an administrator (if you don't have one, you can set up a new one), input the rest of the information (e.g. password), and then click OK.

 

Method 4 - Readme file (Troubleshooting / FAQ)

I'm not sure whether you've tried so already (so I'm sorry if this a method you have already tried yourself!), but if there is a Readme file with Troubleshooting and / or FAQ (Frequently Asked Questions) section / sections, see if you can find any useful information and / or method / methods to try here.

 

Method 5 - Uninstalling / installing

Though this can be an inconvenient method, it might be the one that works. It might be a case of uninstalling the game, and then installing it again for the game to work.

 

Method 6 - Safe mode boot-up

Though this method is more of a 'last resort', you could try booting your computer in safe mode; to do this press F8 repeatedly at boot-up.

*(If the program icon isn't on your desktop, the program file (e.g. an .exe file) should be located in an appropriate place in the Program Files folder (which is in your Local Disk folder).)

I hope this helps, and good luck.
