chrismbaylis say: With a little digging around, I discovered this function that (very sensibly) does what I should have been able to do:

Function chkPivotValue(pvtName As Object, pvtField As String, strCheckVal As String) As Boolean
Dim i As Integer

With pvtName
    For i = 1 To .PivotFields(pvtField).PivotItems.Count
        If strCheckVal = .PivotFields(pvtField).PivotItems(i).Name Then
            'value found
            chkPivotValue = True
            Exit Function
        End If
    Next i
End With
'value not found
chkPivotValue = False
End Function

This checks any string passed to it to see if it exists in the listed options ni the pivotitems.

I can therefore amend the final part of my code to work as follows:
