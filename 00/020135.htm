Instead of decreasing the resolution of the monitor, increase the DPI settings instead. This will keep the text sharp while the font increases.

  1. Right click on an empty spot on your Desktop and choose Properties

  3. In the new dialog that popped up you can directly choose "Larger scale (120 DPI)…" in the dropdown list to make the fonts bigger. If you prefer a different font size you can select "Custom DPI…” and type any percentage you like.

If you archive an email for the first time, you will be asked where you want to save the message. It can be in a new folder, an existing personal folder or creating a folder named Archive.

  • Archived emails are counted in mailbox storage
  • Archived folder expires
  • Archived emails are viewable
  • We can delete emails directly from the Archive folder
  • Archived emails can also be seen if account is configured to an email client using a protocol that supports folders
  • Archived emails will also be transferred if account is renamed
  • You can set up a scheduled sweep when archiving emails
  • Emails on any folder can be archived
  • Messages are filtered as they come into the mailbox. So emails are already filtered before you archive your emails

thomas (tom) wood say:

How do you determine FormatConditions.Type?

I've tried everything.  MyCell has three conditions.  The best I can come up with is MyCell .FormatConditions(1).Type.  I'm looking for an VBA expression that is a XlFormatConditionType.  (e.g. values like xlExpression, xlCellValue and such.)

I'm using Visual Basic 2007. 

bernie deitrick say:

You can roll your own UDF from the list of types from MSDN, like so:

Sub CFTestMacro()
    Dim c As Range
    Dim i As Integer
    Dim strM As String

    For Each c In Selection
        strM = ""
        If c.FormatConditions.Count >= 1 Then
            For i = 1 To c.FormatConditions.Count
                strM = strM & IIf(strM <> "", Chr(10), "") & "Cell " & c.Address(0, 0) & " CF#" & i & ": " & CFType(c, i)
            Next i
        Else
            strM = "No CFs on Cell " & c.Address(0, 0)
        End If
        MsgBox strM
    Next c
End Sub

Function CFType(r As Range, i As Integer) As String
    On Error GoTo NoGood:
    Select Case r.FormatConditions(i).Type
        Case 1
        CFType = "xlCellValue"
        Case 2
        CFType = "xlExpression"
        Case 3
        CFType = "xlColorScale"
        Case 4
        CFType = "xlDatabar"
        Case 5
        CFType = "xlTop10"
        Case 6
        CFType = "XlIconSet"
        Case 8
        CFType = "xlUniqueValues"
        Case 9
        CFType = "xlTextString"
        Case 10
        CFType = "xlBlanksCondition"
        Case 11
        CFType = "xlTimePeriod"
        Case 12
        CFType = "xlAboveAverageCondition"
        Case 13
        CFType = "xlNoBlanksCondition"
        Case 16
        CFType = "xlErrorsCondition"
        Case 17
        CFType = "xlNoErrorsCondition"
    End Select
    
    Exit Function
NoGood:
    
    CFType = "Bad combination"
    
End Function
