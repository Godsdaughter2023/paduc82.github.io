amit tandon say:

Refer below image:

Column A has a list of names with no repetitions.

Column B has a list of store numbers with multiple repetitions.

In column E we return the store names (from column A) corresponding to the store number in cell D2. Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell E2 & copy down:

=IFERROR(INDEX($A$2:$A$15,SMALL(IF($B$2:$B$15=D$2,ROW($B$2:$B$15)-ROW($B$2)+1),ROWS($B$2:$B2))),"")

You may change store number in cell D2, and adjust range in the formula per your requirement.

john korchok say:

As Bob said, knowing the Office version you are using would help a lot.

Text placeholders can contain several levels of text, and each level can have different formatting. But the second line can't automatically follow a different style. If you want the second line to follow the second level of formatting, the user must select that text and click on the Home>Indent More button (assuming they are using a Mac. In Windows, this command is called Home>Increase List Level). Now the second line is formatted to match the second level of the slide layout for that type of slide.

The text levels are normally formatted on the Slide Master, where you can create up to 9 levels or different styles. If a particular Slide Layout has a different set of levels/styles, you can also program this on the relevant layout placeholder.

It's routine for me to send a tutorial with the template that mentions this way of switching text levels to access different styles, since it is not intuitive.
