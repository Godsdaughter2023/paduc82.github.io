Based on the sample in your file, try this:

Sub RearrangeData()
   Dim varData          As Variant
   Dim intStartColumn   As Integer
   Dim varLastColumn    As Variant
   
   varLastColumn = Split(Columns(Application.Columns.Count).Address(0, 0), ":")
   
   varData = Selection
   
   ''' get the starting column
   intStartColumn = ActiveSheet.Evaluate("CountA(B2:" & varLastColumn(0) & "2)") + 2
   
   If Selection.Cells.CountLarge > 1 Then
      Cells(2, intStartColumn).Resize(1, UBound(varData, 1)) = Application.Transpose(varData)
   Else
      Cells(2, intStartColumn) = varData
   End If
   
   Selection.ClearContents
End Sub

It will move the range A4:A12 to B2:J2

A14 to K2

A16:A22 to L2:R2

Note there is no need to specify a cell as A2:B2= row A2 column B2. B2 is just enough: column B, row 2.

rohn007 say:

It is a common licensing issue in 2016/365. Many people have had similar questions. They have a valid license that has worked for a while, but all of a sudden one or more applications report they are not licensed.

Basically, MS assumes that everyone who runs an Office application is a thief until they prove otherwise. Unfortunately, their license accounting system is defective, and keeps "forgetting" peoples licenses.

There are 2 possible fixes.

First, try starting the application using Run As Administrator.  Sometimes that is enough to clear up the problem.

If that does not work, then do an "Office Repair". This checks all of the application program files and registry entries, as well as the licensing.  You go into Control Panel > Programs and Features. Look through the list of installed applications for Office. Right click on it and pick "Repair" if it is displayed or "Change" and then Repair.

Note: lately some people have reported that the 365 repair has actually been doing a full uninstall and re-install and unfortunately the re-install is taking an unreasonably long time to happen, like HOURS.  So it is probably a good idea to do this right before you plan on not using Office for a while, like at night or during the day when you leave your computer at home.
