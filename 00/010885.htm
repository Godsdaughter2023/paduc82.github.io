hansv mvp say:

On the Insert tab of the ribbon, click Header > Edit Header.

This will activate the page header and activate the Design tab of the ribbon.

If the "Different First Page" check box in the Options group of the ribbon is ticked, clear it.

If the "Different Odd & Even Pages" check box in the Options group of the ribbon is ticked, clear it.

Type the title that you want, then press the Tab key.

Depending on the length of the title, the insertion point will be in the center, or at the right margin.

If it is in the center, press the Tab key again; you should now be at the right margin.

Press Ctrl+F9 to insert field brackets { } (do *not* type { } yourself).

Type PAGE between the brackets; you should now see { PAGE }

Press F9 to hide the field code.

You should now have a header with the title on the left hand side and the page number on the right hand side.

Click Close Header & Footer in the Close group of the ribbon.

tom ogilvy say:

=countif(B1,"*"&A1&"*")>0

 

will return true if the string in A1 is found in B1.  Now you have to consider what your restrictions are. 

 

A1:  an

B1: antiestablishment

 

this formula would return true.

So to be restrictive, you would want to see that the word in cell A1 is isolated by spaces in B1.   This would do that without altering the values in either A1 or B1

 

=ISNUMBER(SEARCH(" "&A2&" "," "&TRIM(B2)&" "))

 

this would test if   " antiestablishment " contained the substring " an "

and it would fail. 

but

A1:  an

B1:  An owner is needed.

 

would pass.

Do you talk about your Nokia E72 and its USB mode ‘connect PC to web’, in newer phones called ‘use phone as modem’?

In that USB mode, an (outdated) Nokia Connectivity Cable Driver is installed as well. Furthermore, a stripped-down (outdated) variant of the Suite is installed; just the dial-up component. joanne li msft say:

Hi Mike,

 

I'd like to explain that this is happening because changes are made to an Excel workbook when you recover a backup of the file's latest version after Excel crashes. The auto-recovery feature stores backup files in the AppData\roaming\Microsoft\Excel folder. Because the link path to your local drive is relative, it then assumes your linked workbooks are in that same folder, and sets the relative path to the folder.

 

Currently we can't change the behavior and there isn't an out of the box feature in Excel to change multiple link paths at once. How can you scan something into a Word document with Office for - Microsoft Community

teeforetwo say:

How can you scan something into a Word document with Office for Mac?

I am new to Mac and purchased Office for Mac.  I am trying to find out how you can scan something into a Word document.  HELP
john mcghie say:
You use the software that came with your scanner to scan the stuff in as a picture, then you insert that as a picture using Insert>Picture>From File.

Set the scanner to give you TIFF or GIF if you are scanning "text", JPG is too fuzzy.

The TWAIN drivers you are used to in Windows do not exist for Mac, so you cannot scan "into" Word on a Mac.

Hope this helps

bob jones aka: cybertaz say:
As John indicated, what he suggests will provide a picture of the scanned document, not editable text. If you intend to have text you need to use Optical Character Recognition [OCR] software which may have come with your scanner.

If not, you may be able to scan into Apple's Preview app, save as a PDF, then open the PDF using Adobe Reader. Reader will allow you to save in a text format which can then be opened for editing in Word.

Regards,
Bob J.
