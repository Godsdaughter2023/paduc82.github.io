excel formula for auto date footnote - Microsoft Community

dc405 say:

excel formula for auto date footnote

I would like to insert a formula for the DATE SAVED, and the document FILE NAME to update in my Excel document footnote automatically each time I revise and SAVE (not to change when I just open or print, only when I do new SAVE)

Does someone have this formula to share?

gord dibben say:

To insert items in a footer you will need VBA.

 

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
Dim wkSht As Worksheet
    For Each wkSht In ThisWorkbook.Worksheets
        wkSht.PageSetup.RightFooter = ActiveWorkbook.Name & " Last Saved : " & _
            Format(Now, "yyyy-mmm-dd hh:mm:ss")
    Next wkSht
End Sub

 

With your workbook open hit Alt + F11 to bring up the Visual Basic Editor.

CTRL + r to open the Project Explorer.

Select your workbook/project and expand it(+ sign).

Double-click on Thisworkbook Module to open.

Copy/paste the above save event code into that module.

 

Alt + q to return to Excel.  When you save the workbook all worksheets will have a footer with workbook name and save date/time.

#If Mac then

#EndIf

The difference is that using the compiler constants skips the code compiling so you will not get compile time or run time errors, whereas Jim's method requires that the code you are branching around will clean compile in both platforms.

Hi

In an environment where you have many user accounts, you can use an alternate Login Screen.

Press the WinKey+R to display the Run dialog and type Regedit.exe and click OK to open the registry editor.

Navigate to the following key. HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies\System

In the right side of the window, right click the  dontdisplaylastusername  value and select Modify.

In the Value Data box, replace the zero with the number  1.

Press the F5 Key to refresh the window and exit the registry editor.

The next time a user logs on they will see the following screen where they can type their user name and password.
