If you all you want to do is get rid of the superscript attribute from existing and future footnotes, clear the superscript from the Footnote Reference style as Suzanne and Doug have already suggested. To do that in code, you need the following (but, again, using a macro isn't necessary):

Sub ClearSuperScriptFormattingFromNotes()
ActiveDocument.Styles(wdStyleFootnoteReference) _
.Font.Superscript = False
End Sub

Note that the macro clears the superscript in the footnote area and in the footnote reference in the text (because the Footnote Reference style is used in both places).

You will need to use the .Copy method of the Worksheets object to create new workbooks containing the Worksheet applicable to each recipient and then use the .SendMail method of the Workbook object to send each workbook to its intended recipient.
