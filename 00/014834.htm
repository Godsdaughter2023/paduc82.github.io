Hi EAS,

There are some things that causes website to redirect it can be the addon that you accidentally installed after you download something on your computer, it can be due to untrusted site that you browse or default homepage is ruined by malicious sites.

To fix that issue I would suggest to do this things

Method 1: do reset your internet settings

This option is very useful if you want to set your internet settings to default because when you reset your IE it will set all settings to its original content

Open internet option from control panel then go to advanced tab click  restore advanced settings first then hit reset internet settings

Method 2: Change your default homepage.

You can use this option to set the default homepage of your internet explorer if its ruined by a malicious website but this will not work if the malicious application is still present on your computer thats why its better to remove unwanted apps in add/remove programs before doing this method.

Open internet properties from control panel then go to general tab. type your preferred homepage click apply then ok when you are done

Method Three

Check the extended link of your IE browser in the target location of your IE under shortcut tab

You can use this method if you already do all those steps that I mentioned above because some browser hijackers do create an extended link in target location so that the user coudlnt remove it even the user already reset IE settings

right click on your IE icon then go to properties. In properties go to shortcut tab and look for target if there is other link that exist beyond this "C:\Program Files\Internet Explorer\iexplore.exe" better remove it because this might be the reason why your browser is redirecting to other site

Method four

Browser redirects because of browser hijacker contents are left on the registry.

If you did the 3 options that I mentioned above and the same thing occurs definitely browser hijacker left its content on registry thats why its better to check the registry if there are some files that is left.

Open regedit then in registry editor press ctrl + f to find if the browser components is still on the computer. Click next until you find out that there is no other components left on your computer
