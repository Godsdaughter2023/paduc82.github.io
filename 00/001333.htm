Hi. Unfortunately, database tables do not really store records in any order, so determining the 10 "previous" records is a bit uncertain. thedbguy say:

Wait, not you're talking subforms. Is the subform linked to the main form? If so, it will only be able to display records that match the link. If you don't link it, then you can display any records you want. However, without the link, new records won't automatically inherit the PK from the parent as FK on the subform.

To see the previous 10 records based on the ID value, you can try something like the following:
