jeeped say:

In A1, I want the cell to allow only the dates specified in the following criteria:

  • the last day of any month for any year, including the Leap Days (i.e., January 31, 1999, June 30, 2003, February 29, 2012)

Data Validation can contain a custom formula that appears to be a circular reference but isn't due to the external handling of the data validation. Use Data Validation, Custom with a formula like,
=AND(A1<>"",A1=EOMONTH(A1,0))       to force an entry, e.g. no blanks 
... or,
=A1=EOMONTH(A1,0)      to allow blanks 
These custom formulas follow similar rules to Conditional formatting.

In B1, I want the cell to allow only the dates specified in the following criteria:

  • EOMONTH(A1,"X")+1 where "X" is any negative multiple of 12 (i.e., if A1 contains March 31, 2005, B1 will only allow April 1, 2004, April 1, 2003, etc., if A1 contains December 31, 2016, B1 will only allow January 1, 2016, January 1, 2015, etc.))

For B1, you can use another custom formula like,
=AND(DATE(YEAR(B1-1),MONTH(A1),DAY(B1-1))=B1-1,B1<A1,DAY(B1)=1)

This might be better handled by a VBA routine that takes any date input into A1 and converts it to the end of the month. The same VBA could blank a dependent entry (e.g. B1) if A1 is changed and check for your criteria rules.

No, in Word those are two completely separate features.

If you use the Citation method and then a bibliography, you can then select the entry in the bibliography, copy it, and paste it into a footnote; finally, delete the bibliography and the citation, leaving only the footnote. A: No. If you disable the user's archive, you'll have 30 days to enable it again and retain the existing content. After 30 days, all information in the archive will be permanently deleted. If you enable the archive again after this time, a new archive mailbox is created.
