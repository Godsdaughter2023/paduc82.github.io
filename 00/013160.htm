ronrosenfeld say: You could write a User Defined Function in VBA, which would be simple to maintain, or you could use a rather complex function, which I will provide but will leave it to you to make alterations.

The function looks only at A1:A4, and only for a value of 100, and only for strings with three commas (as you show in your example).  You will have to manually change it for other situations:

=SUMPRODUCT((LEFT(A1:A4,FIND(",",A1:A4)-1)="100")*TRIM(
RIGHT(SUBSTITUTE(LEFT(A1:A4,FIND(CHAR(1),SUBSTITUTE(
A1:A4,",",CHAR(1),3))-1),",",REPT(" ",99)),99)))

My error.  I thought you knew more about UDF's.
Also, it now seems that you want to be able to vary which comma group to sum, so you can enter that as a variable in the function.

For example, to sum those values with a Key of "103" that are in the group after the first comma:

=SumCommaGroup("103",A1:A10,1)

If you want to sum a different comma group, merely change the CommaGroup variable in the function when you write it into the worksheet cell.

==================================
Option Explicit
Function SumCommaGroup(Key As Variant, rg As Range, CommaGroup As Long) As Double
    Dim vRg As Variant
    Dim vTemp As Variant
    Dim i As Long
vRg = rg  'using this method since you wrote you have very large ranges
For i = LBound(vRg) To UBound(vRg)
    vTemp = Split(vRg(i, 1), ",")
    If UBound(vTemp) >= CommaGroup Then
        If vTemp(0) = Key Then SumCommaGroup = SumCommaGroup + Val(vTemp(CommaGroup))
    End If
Next i
End Function
=============================

Reopen a notebook to sync OneNote

If the notebook that won’t sync is stored on OneDrive or on newer versions of SharePoint, try opening the notebook in OneNote Online. This can help determine if the problem is caused by the server or by OneNote.

  1. In OneNote, click File > Info.

  2. Right-click the URL shown below the name of the affected notebook, and click Copy.

  3. In your web browser, click inside the address box, and press Ctrl+V to paste the copied URL.

  4. Press Enter to load the notebook in OneNote Online.

If you're able to successfully open the notebook in your browser, the problem may be caused by the desktop version of OneNote. Closing and re-opening the notebook usually clears and successfully resets the connection.

  1. In OneNote, click File > Info.

  2. Next to the name of the affected notebook, click Settings, and then click Close.

  3. Switch back to the notebook in OneNote Online in your browser, and then click Open in OneNote on the ribbon.

If you were unable to open and view the notebook in OneNote Online, the source of the sync problem is likely with your server. If your notebook is stored on OneDrive, check the Microsoft Services status page for any known interruptions or outages. If your notebook is stored on SharePoint or Office 365, you’ll need to contact your administrator to report and troubleshoot the problem.

ahmed_emara say:

Hi Ginger,

Sorry to hear you’re experiencing this. as you are saying that after splitting the notebook into 3 notebooks, 2 of them have sync issues, this strongly suggest you have some faulty pages/sections.
so my suggestion is to do the following for each notebook of the 2 with sync issues:

  • Create new notebook start copying each section one at a time.
  • make sure the new notebook sync fine after copying every section.
  • if the sync issue start to appear then the last section is the one causing problems, so add new section and start copying the content of the faulty section into it.
    (don't copy the pages because this might copy the issue again into the new section just copy content of every page one at a time and make sure sync is fine after each page)
  • delete the faulty section after copying your content to the new section.
  • Repeat these steps for both notebooks.

Note: In general adding big files (pdf, images, ...) into Notebooks make sync slower, so it's highly recommended to split large notebooks into small ones and avoid adding large files, instead you can add them to OneDrive and only keep links to location in OneDrive in the notebooks.

you can also follow the troubleshooting steps provided in the link below to reduce the sync time
