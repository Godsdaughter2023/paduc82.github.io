a. user say:

Breaking it down:

USB means we're dealing with an embedded USB device
VID means Vendor ID
04F2 is the vendor ID for Chicony Electronics
PID means Part ID
B071 is the part ID for the CNF7129 webcam made by Chicony for Asus

MI means Management Interface

00 is the serial management interface, which makes sense as we're dealing with a Universal Serial Bus device.

So we're talking about the Asus CNF7129 webcam in your computer, which was made for Asus by Chicony Electronics. Chicony is a huge Taiwan-based manufacturer of electrical components.

Most of the time this message is actually normal and has no real significance. If you see an entry that says "Device installed" later on in the list of Events, that means the driver for the webcam is properly installed.

However, it's not enough to merely have the latest driver correctly installed.  If Asus or Chicony doesn't have a Windows 10 driver for your webcam, or an earlier Windows 7 or Windows 8 driver that is compatible with Windows 10, the webcam isn't going to work properly in Windows 10.

In fact, a little checking on the Asus website shows that the last driver for that camera was published in 2011. Since your webcam has stopped working, and no more recent driver is available, that's a pretty good sign that your webcam is not compatible with Windows 10, or is no longer compatible with Windows 10, since the OS is being constantly updated.

The solution? Disable the built-in webcam in the BIOS and buy a new webcam. A new webcam is real cheap and it will run circles around the ancient Chicony.
