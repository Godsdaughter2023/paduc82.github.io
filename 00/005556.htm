Private Sub Document_New()
Dim oVars As Word.Variables
Set oVars = ActiveDocument.Variables
oVars("varDate") = Format(Date, "mmddyy")
oVars("varTextDate") = Format(Date, "mmmm d, yyyy")
myUpdateFields

Sub myUpdateFields()
Dim oStyRng As Word.Range
Dim iLink As Long
  iLink = ActiveDocument.Sections(1).Headers(1).Range.StoryType
  For Each oStyRng In ActiveDocument.StoryRanges
    Do
      oStyRng.Fields.Update
      Set oStyRng = oStyRng.NextStoryRange
    Loop Until oStyRng Is Nothing
  Next
End Sub

You could add a new macro to the template

 

Sub UpdateVariables()

Dim oVars As Word.Variables
Set oVars = ActiveDocument.Variables
oVars("varDate") = InputBox("Insert the new value for varDate", "Update varDate", Format(Date, "mmddyy"))
oVars("varTextDate") = InputBox("Insert the new value for varTextDate", "Update varTextDate", Format(Date, "mmddyy"))
myUpdateFields
End Sub

lacrumb say:

To be fully activated you had to upgrade to the RTM even though you were at the same place with the previews.  After upgrading then you could do a clean install.  This is on a qualifying computer.

A VM does not qualify since it is portable.  You have to activate the computer.

New releases are to those that are activated.

If you still have Win 7 SP1 or Win 8.1 you can upgrade to RTM using this;

Go to www.microsoft.com/en-us/software-download/windows10, Do as an upgrade.

If it asks for a key just go to the bottom left and click on Do this later

Then check if your computer is activated;

To check if you computer is activated;
Right click Start, click Command Prompt(Admin)
enter slmgr /xpr  Then hit the Enter key
It will tell you when it expires.  Mine tells me that my computer is permanently activated.

If you make a change to the motherboard then you loose this activation.
