You can use the Split feature on the Video Tools | Edit tab as many times as needed.  The shortcut key is M on the keyboard.

 

However, as in all versions of Movie Maker, when it comes to making this into several separate movies, you have to save one part at a time.  One way to do this in the latest version is to save your project, with all the splits.  Now open it up and delete all but the first clip, save as a new movie.  Now use the Undo feature, or reload the project.  Now delete all but the second clip, and repeat.

 

In the older "classic" versions of Movie Maker, you could do this splitting in a separate "Collection" area.  Then you could add each section to the timeline, publish as a movie, then repeat for the next clip.  If this workflow is more to your liking, you could load the classic Windows Movie Maker 6.0 on Windows 7.  It can coexist with the newer Movie Maker 2012, so you can use whichever one meets your needs for a specific project.

Each of the files will be saved with the new name, and a different sequential number will be appended automatically to the end of each file name.
