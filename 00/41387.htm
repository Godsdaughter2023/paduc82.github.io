    One way to check this is to right click on the Start globe and click on Properties, then click on the Notification Area tab.  Is the box to "hide inactive icons" checked?  If not, please check it.  Next, click on the customize button and check the current items.  Do all of them say "show" and/or are all of them actually showing in the notification area?  If so, there'll be no arrow because there's nothing hidden and thus no need for the arrow to get to non-existing hidden items.

     

    If those missing are any of the four system icons shown on the notification tab's main page, then the problem may be the need to check the boxes to show those system icons (they show if checked and never get hidden, so if missing then checking the boxes will make them appear unless the option is greyed out - in which case rebooting often is enough to fix that or see http://www.vistax64.com/tutorials/106787-notification-area-system-icons.html).   This sometimes happened to me and the most common reason was when I had too much in startup - but cutting back typically resolved that.  Some also say that waiting a few moments to login can help and while it hasn't really seemed to help me much, too many people have suggested it to dismiss the possibility.

     

    If those missing are anything else, then please be very explicit about what you thing should be there (showing or hidden) that is not there (the program behind the icon - name and version), verify that settings in the past item section of the notification are icons do not include the "missing" icon because it is set to "hide" - if so change it "hide when inactive" or "show", make sure the program itself is set to show the icon (some have settings which can enable or disable the notification area icon).

     

    If you're sure there are hidden icons but the arrow is missing, then next I would try making sure that the taskbar is not locked (right click in a free area of the taskbar and uncheck lock the taskbar if it is checked) and then expand the notification area section to be sure that it's not simply due to lack of sufficient space allocation to it when the taskbar is locked and it can't adapt to needed expansions to handle more notification icons than can otherwise fit.  If that fixes the problem, either leave it unlocked or expand the notification area to handle all the icons you expect to ever show - including the hidden ones - and then relock the taskbar with the notification area section properly expanded.

     

    Hopefully, one or more of these will explain or help resolve the problem.  If not, post back with further information about what's missing so we can try to figure out what might be the problem.

     

    Good luck!
