Windows 10 Advanced Sound Settings in Themes - Microsoft Community

texjpc say:

Windows 10 Advanced Sound Settings in Themes

I want to change the New Mail Notification sound from the default to another sound.  After I make the change the new sound works for a day or two but then resets back to the default sound.  How do I make the change permanent?

Thanks

Jerry

ramesh srinivasan say:

See if turning off Theme syncing helps. Open Settings (WinKey + i), click Accounts and click Sync your settings. Turn off "Themes" radio button.

If that doesn't help, or doesn't apply in your case (no multiple Windows 10 devices used), then some other program may be changing the sound setting. You can create a REG file using the following contents and run it from your Startup folder, so that the setting is applied at every login.

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\AppEvents\Schemes\Apps\.Default\Notification.Mail\.Current]
@="C:\\Users\\TESTUSER\\Music\\Email.wav"

[HKEY_CURRENT_USER\AppEvents\Schemes\Apps\.Default\MailBeep\.Current]
@="C:\\Users\\TESTUSER\\Music\\Email Beep.wav"

[-HKEY_CURRENT_USER\AppEvents\Schemes\Apps\.Default\MailBeep\.Modified]

[-HKEY_CURRENT_USER\AppEvents\Schemes\Apps\.Default\Notification.Mail\.Modified]

Change the .wav file paths in the REG file accordingly, maintaining those double-slashes in the file path. Save the REG file in your documents folder, as sounds.reg. Then open your Startup folder (Press WinKey + R, type shell:startup and press ENTER)

In the Startup folder, create a shortcut with the following target:

reg.exe import %userprofile%\documents\sounds.reg

That's it.
