A 0x000000ED UNMOUNTABLE_BOOT_VOLUME is usually caused by a corrupt file system, damaged hard drive or incorrect boot parameters.

There are 2 ways to run chkdsk:

1. Here is a more descriptive document on how to boot into recovery console and run check disk.

http://support.microsoft.com/kb/297185

Following the above article should fix the corrupt file system.

2. You could also re-attach the hard drive to another PC as a2nd hard drive. Once the PC has booted you can go to

Start -> Run -> type chkdsk X: /r -> enter. (WhereX is the drive letter given to the damaged 2nd drive). If it asks you to force a dismount say No. If it asks to schedule a scan at the next startup say Yes. Then restart the PC and let chkdsk run.

 

As Davee said you can also download the hard drive diagnostics program from the hard drive manufacturer's website.

You can find a list of manufacturers and the links to their diagnostics utilities at this website: http://www.tacktech.com/display.cfm?ttid=287

Usually you just download the file, unzip it and then write the ISO image to a CD. You would then boot the PC with the CD and run the long or advanced test.

At the startup menu instead of choosing Normal or Safe Mode try "Disable Automatic Restart on System Failure". This option will then continue with the Windows boot and stop when the BSOD is thrown.

Screen resolution

Color depth

Color space

    andre da costa say:
    Will I be able to play DVDs after upgrading?

    If you already have third-party DVD playback software, Upgrade Assistant helps determine if this software is compatible with Windows 8. If your PC is running Windows 7, you'll have the option to keep this software during the upgrade to Windows 8. If your PC is running Windows XP or Windows Vista, you'll need to reinstall it.

    If you don't have a third-party app that plays DVDs, or if you currently rely on Windows Media Player for DVD playback, you'll need to download and install a third-party app or install Windows Media Center to play DVDs. If you use Windows Media Center on Windows 7 and are upgrading to Windows 8, for a limited time you can get the Windows 8 Media Center Pack for free. For more info, see the Add features webpage.
