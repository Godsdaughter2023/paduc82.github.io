Unlike in 2007, the Print Area can be adjusted in both Normal as well as Page Layout View. In order to do so, though, the Page Breaks must be visible. You can activate them either by:

A different approach:

Sub Comunicate()

    Dim TargetWorksheet As Worksheet
    Dim Cell As Range
    Dim TargetRange As Range
 
    Set TargetWorksheet = ThisWorkbook.Sheets("Index")
 
    For Each Cell In TargetWorksheet.Range("Communications")
        If IsEmpty(Cell) Then

Windows Phone 7;  Hotmail Outlook Connector; VCF; Outlook contacts

If you have trouble adding certain contacts to the Hotmail Outlook Connector that were created while you were a member of a domain, the problem is the email address.  The solution is to delete the email address.  This is a very manual process, find the best way with cutting and pasting to save the actual SMTP*** Email address is removed for privacy *** because what you do not see is that the visible field in Contacts is linked in the background (contact metadata) to the X500 address as it would be found in the Active Directory GAL (Global Address List.)  The X500 address is only valid if the Outlook client is sending while connected to the Exchange server.
