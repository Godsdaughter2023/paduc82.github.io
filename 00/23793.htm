albert d. kallal say:

As a general rule you not see Access as a application "icon" on the web side.

You have to create and START from the Access desktop for the Access client.

So to create a web site with Access you MUST FIRST use the Access client program. When you create a web site, then Access will create a URL for that application. Access is different then rest of office and it not a simple document. (the web site will hold the forms etc.).

So AFTER you do the above, then the resulting application can be used in any web browser and the user does not require Access.

Quite sure the student edition does include the Access web publishing feature. You can try it out by launching Access on your desktop and creating a web application.

So for Access applications, there no "saving" the database "file" on the web. You use the Access client to create + design web forms directly from your desktop (no design occurs on the web side - only can "use" the resulting application).

So you create the web application using Access and then have to use a URL to launch the web site (Access will create this web address for you).

A more "correct" way to think of the Access setup is you are creating web forms in the could but you are using the Access desktop version to create these parts.  Access is not a "document" like Word or Excel, so it works differently.

So each "web" part is created as you work and saved in the cloud as you work with Access. As a result, you don't have to publish or save or copy anything to the web during this development processeb site (Access will create this  since each bit and little part  you create is saved up to the cloud as you work and create your application.

The above works the same for Home edition, small business edition, and quite sure the student edition also includes the Access web publishing feature.

To reset an unresponsive phone

If your phone is unresponsive and you can't reach Settings , you may be able to reset your phone using the hardware buttons. On Nokia Lumia phones (and some others), do the following to reset your phone:

  1. Press and hold the Volume down and Powerbuttons at the same time until you feel a vibration (about 10-15 seconds).
