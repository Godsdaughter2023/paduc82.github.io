robertjwin say:

The root of any drive is the top-most level, i.e. you copy the file to the drive but not into any folder on that drive.

BTW, if you're trying to perform a firmware update of your set-top box then a common mistake people make is to copy the ZIP they downloaded instead of extracting and copying the actual file from inside the archive. Unless the instructions specifically state that you need to copy the archive itself, if you got the files compressed then be sure to extract them first.
