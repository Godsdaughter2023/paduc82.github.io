dax arrowayii say:

Copy Macro w/Add Initials and Date

Here's an interesting one.  I have a simple macro in a Word doc:

 

Sub TextCopier()
    Selection.MoveRight Unit:=wdCell
    Selection.Copy
End Sub

 

To which I'd like to add at the end, the currently logged on user's initials, in Caps, and today's date formatted 00-00-00.  Is there a way to do that? 

 

~Dax

dax arrowayii say:

Actually we figured it out.  It's:

 

Selection.MoveRight Unit:=wdCell
Selection.InsertAfter " " & UCase(Application.UserInitials) & _
" " & Format(Date, "mm/dd/yy")
Selection.Cells(1).Range.Copy
ActiveDocument.Undo 2

Sub FiddleTheChartData()

Dim oCht As Chart
Dim oChtData As ChartData
Dim x As Long

Set oSh = ActiveWindow.Selection.ShapeRange(1)

If Not oSh.HasChart Then
    Exit Sub
Else

    Set oChtData = oCht.ChartData
End If

' Activate the chart and modify some data

With oChtData
    .Activate
    .Workbook.Worksheets("Sheet1").Cells(3, 3) = 42
End With

' This leaves the workbook open in Excel, so:
oChtData.Workbook.Close
