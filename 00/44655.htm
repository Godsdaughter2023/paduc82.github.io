Are you using MM 2012?

You can start a project add your audio, one track and save the movie file.

considerate_guy say:

Assuming you are using Movie Maker 2012, you have two additional audio tracks (beyond the audio that comes with the video.)

For the first track, go to the Home tab and click on Add music. While it is denoted the "music" track, you can add any audio you like (music, sound effects, etc.)

For the second track, go to the Home tab, click the down arrow next to Record narration and choose Add sound. Again, this track can be used for whatever audio you want (music, sound, narration, etc.)

In that way you can have two different sounds playing at once.

If you are on an earlier version of Movie Maker, or if you need more layers of audio, then you can use the layering method noted by skier-hughes.
