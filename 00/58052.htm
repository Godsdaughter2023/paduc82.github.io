a. user say:

If Mixed Reality Portal did not launch automatically when you connected your headset, you probably have an issue with the installed driver. The most likely issue is that your driver needs to be installed. You may follow the steps below on how to install the driver for Mixed Reality:

  1. Open up Device Manager, look under Other devices for a HoloLens Sensors device with an exclamation point.
  2. Right-click the device, and select Properties. If the device's properties read The drivers for this device are not installed (Code 28), exit the window.
  3. Right-click on the device again and select Update Drivers > Search automatically for updated driver software.
  4. After the device updates, you should see a Mixed Reality Device appear in Device Manager.

If manually installing the driver didn't work, or that you do not find it under Other Devices, then you probably need to uninstall the existing driver and reinstall it. Here's how to uninstall and reinstall the driver:

  1. Open Device Manager, look under Mixed Reality devices for your headset.
  2. Right-click the device, and select Uninstall Device.
  3. In the pop-up that appears, select the check box Delete the driver software for this device and click Uninstall.
  4. When that finishes, unplug the headset from your PC, and plug it again. Windows Update will now download and install a new driver.

If the issue persists, feel free to post back to us.
