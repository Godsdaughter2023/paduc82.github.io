shane groff say:

Hi Jason,

Build 16.0.6001.1073 is the most recent build for the deferred channel.  You can see the build numbers for different channels here:

https://technet.microsoft.com/en-us/library/mt592918.aspx?f=255&MSPPError=-2147217396

However, that build does not have the fix for the image issues.  The deferred channel builds are older, and intentionally take a much smaller set of changes in order to keep the build as stable as possible for those organizations that value stability first.   For organizations that want fixes most quickly, there is the current channel, and the first release for deferred channel (which is in between the other two).  Both the current channel and the first release for deferred channel builds have the fix, but the deferred channel build does not.

The good news is that the deferred channel will move forward to a build that has the fix, but I don't know exactly when this will happen.

Shane
