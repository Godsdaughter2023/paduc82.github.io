jay freedman say:

You can do this, but not by using the Quick Parts > Document Property command on the ribbon.

When you use the Quick Parts > Document Property command, it inserts the result within a Plain Text content control. That's why you can't apply formatting to only part of it. There's no way to convince the program to use a Rich Text content control instead.

However, you can insert a DocProperty field without a content control by pressing Ctrl+F9 and inserting the desired field code, so that it looks like this for example:

{ DocProperty Author }

Then press F9 to update it. You can then select any part of the result and format it.

If you specifically want the property to appear in a Rich Text content control (for example, if you're creating a form with the Group command so that only content controls are editable), you can insert the DocProperty field into a Rich Text control; it will continue to behave the same.

If you don't care for the keyboard shortcuts, you can do the same by using the Quick Parts > Field command and choosing the DocProperty field name from the list, then the particular property you want:

Hello Marjorie,

 

Thanks for you the post. As the search feature is related to the Spotlight on your Mac, I suggest you follow the steps below to rebuild the Spotlight index:

1 . Go to Apple menu->System Preference, click Spotlight. Click the Privacy tab.

2 . Drag the folder or disk that you want to index again to the list of locations that Spotlight is prevented from searching. Or click the Add(+) button to select.

3 . From the same list of locations, select the folder or disk that you just added. Then click the Remove(-) button to remove it from the list.

4 . Quit System Preferences. Spotlight will re-index the contents.

Spell check unavailable in protected worksheet? - Microsoft Community

needto know 4 say:

Spell check unavailable in protected worksheet?

When protecting a worksheet, the spell check option is no longer available to use, even in an unlocked cell. How do I change this so spell check is available?
hansv mvp say:

You could create a macro

 

Sub SpellCheckProtected()
    With ActiveSheet
        .Unprotect "password"
        .CheckSpelling
        .Protect "password"
    End With
End Sub

 

where password is the password used to protect the sheet; if you don't use a password, you can omit "password".

When you want to spell check, run this macro. (This of course requires that macros are enabled; the easiest way to ensure this is to place the workbook in a Trusted Location)
