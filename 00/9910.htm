We need to provide emails for someone who requested them as either ASCII Fixed length files or ASCII delimited format.  We are going to send them a PST of all the emails but I’m worried they might want them as just a text file instead of a PST. So the question is are PST's considered ASCII format?

If PST’s aren’t either ASCII fixed length or ASCII delimited format how would I convert them?


Thanks

RG

brian tillman [outlook mvp 2007-2019] say:

ASCII stands for "American Standard Code for Information Interchange".  Typically, when someone requests an ASCII file, they mean they want it to be human-readable; composed only of letters, digits, and punctuation.  PSTs do not fit that.  They contain non-readable (i.e., binary) values as well as readable values.  Moreover, they are essentially indexed files; something a database program would handle.

 

There are tools available that integrate with Office Outlook aand can dump the contents of PSTs into ASCII text.  See http://www.slipstick.com/outlook/email/how-to-save-email-in-windows-file-system/.
