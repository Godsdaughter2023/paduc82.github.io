mac_freak say: Hello.

The memory problem you are experiencing is not related whatsoever with the flash memory you are storing the files into (whether it is the phone or memory card). The memory error stated here is a error related to the lack of RAM (random access memory) in which applications used by the operating system are temporarily dumped in and the software threads used to run different processes in various softwares are run here. If many processes are already consuming a large part of your RAM, opening another application will only be problematic since it cannot be run anymore due to lack of space to run it. If that is the case that it seems pretty impossible to use your phone at all, a hardware problem related to RAM maybe the cause or only a memory buffer overflow bug posed by a faulty coding of the firmware. Better send it out to Nokia support to make sure.

Cheers.
