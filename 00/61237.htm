loboo say:

Reserva espacio para actualizaciones.

De todas las maneras para calcular la capacidad del disco duro se utiliza esta formula:

N*10^y / 2^x

N = número que da el fabricante(500).

y =  exponente del prefijo en el Sistema Internacional (giga = 9).

x = exponente del prefijo en el sistema binario(Gibi (Gi)30).

N*10^9 / 2^30 = 465 aproximadamente.

500 GB = 465 GiB.
