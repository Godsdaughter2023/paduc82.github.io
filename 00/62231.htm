lemp say:

Removing old no longer used printer drivers will not speed up your printing.  In addition, if you have print jobs stuck in the print queue, nothing will print.

Nevertheless, for reference, here's what you can do.

Clean out stuck print jobs

  • Open an Elevated Command Prompt (search for "cmd.exe"; right-click and select "Run as administrator")
  • Type the following in the black Command Prompt window and press Enter after each line

                           net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

Note that if you have installed a fax printer, the above needs to be slightly modified.  In this case, after you type the first command, you'll get a message like this:

C:\Windows\system32>net stop spooler
The following services are dependent on the Print Spooler service.
Stopping the Print Spooler service will also stop these services.
   Fax
Do you want to continue this operation? (Y/N) [N]:

You should respond by pressing the Y key and press Enter.  Then, after you type the net start spooler command, you should also type net start fax and press Enter.
Clean out corrupt or no longer used printer driver

  • Uninstall printer-related software (for the printer you no longer want) from Control Panel > Programs and Features
  • Go to Devices and Printers, right-click the printer that you no longer want, and select "Remove device" 
  • Open an Elevated Command Prompt (search for "cmd.exe"; right-click and select "Run as administrator")
  • Type the following in the black Command Prompt window and press Enter    
            printui /s /t2
            exit
    If you have no other printers from the same manufacturer, remove both the driver and the driver package.  If you have a printer from the same manufacturer that you are going to keep, only remove the driver that you are no longer using.
