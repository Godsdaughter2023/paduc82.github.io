Hi,

Built-in Apps on Windows 10 can be removed by uninstalling them using the Windows PowerShell. To uninstall the Xbox App, you may follow the steps below:

  1. Click Start, and then type Windows PowerShell in the search box.       
  2. Right-click, and then select Run as administrator.       
  3. Type the following command to uninstall Xbox app: Get-AppxPackage *xboxapp* | Remove-AppxPackage       
  4. Hit Enter.       

If you just want to disable the Game DVR, you may perform the following steps on your computer:

  1. Open Registry Editor (Run > regedit)      
  2. Navigate to HKEY_CURRENT_USER\System\GameConfigStore     
  3. Set the value of DWORD "GameDVR_Enabled" to 0     
  4. Go to HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\     
  5. Create key "GameDVR".     
  6. Create DWORD 32bit called "AllowGameDVR" and set to 0     
  7. Restart your computer.

Please do note that these are sensitive files on your computer to make sure that you follow the steps thoroughly.     

Should you need further assistance, you can reply to this post anytime.
