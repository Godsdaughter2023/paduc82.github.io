I'm not sure what's causing your problem (Stefan's long shot might be a hit, though), but instead of using the Tab key to create a first-line indent, it is easier to use Word's built-in first-line indent property. You can apply this by dragging the First Line Indent marker on the horizontal ruler (once you've done this for a single paragraph, it will apply to all subsequent paragraphs until you change it), but a better way is to apply a style that has this indent. Vic.-

I think you have run into a problem with the nature of the English language.

The replace you describe would not need wild cards. You would replace ".[space]" with ".[line break]. The special formatting character for a line break is ^l.

For problems you know about, you could run a fixit replace to replace, i.e. "Dr.[line break]" with "Dr.[space]. This would be run after you run your general replace. Please note that "Dr." is an abbreviation for not only "Doctor" but also "Drive;" similarly, "St." is an abbreviation for "Saint" but also "Street." In one case you may want your line break, and not in the other.

I know of no good comprehensive web tutorial on vba (Word macros). I'm sure that there are some. This is a programming language that is specialized for use with applications. It requires not only programming skill but also a very good knowledge of the application. Otherwise, you'll end up trying to program functions that already exist.

The Word MVP site has a number of good pages that could get you started, though. Macros and VBA (Visual Basic for Applications)

VBA (Visual Basic for Applications) (macros)

Linking and unlinking contacts

You can link contacts to each other to indicate that they are somehow related. In addition, Outlook.com automatically detects contacts that have the same or very similar names and links them so that they appear as a single contact.

Link contacts

  • Select the check box next to each contact in the middle pane, and then on the toolbar, select Link.

Unlink contacts

  1. In the middle pane, select the check box next to a linked contact.

  2. On the toolbar, select the Links drop-down menu.

  3. On the drop-down menu, select the next to the contact that you want to unlink from the others.

Understanding automatic linking

To see which contacts Outlook.com has automatically linked for a particular contact and to see suggestions for other, similar contacts to link that contact to, follow these steps:

  1. Select the check box to the left of the contact name. If Outlook.com has automatically linked any contacts that have the same or similar names, a Links drop-down menu will appear on the right side of the toolbar.

  2. On the toolbar, select the Links drop-down menu.
