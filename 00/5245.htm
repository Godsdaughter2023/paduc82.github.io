Hi,

 

VLOOKUP can't do that but INDEX can.

 

 

Put this ARRAY formula in a cell, ARRAY enter it; see below for how, and drag down as far as you want. Now enter the clothing item you're looking for in D1

 

=IFERROR(INDEX($C$2:$C$20,SMALL(IF($A$2:$A$200=$D$1,ROW($A$2:$A$200)-ROW($A$1)),ROWS(A$1:A1))),"")

 

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

macropod say:

Minor changes in layout are normal for Word documents, as Word optimises the page layout according to the capabilities of the active printer. This can result is differences in line-wrapping and even content shifting from one page to another.

However, your description seems to go beyond that. Changes in colour, etc. are most likely due to the document having the 'automatically update document Styles' option checked (see under Developer>Document Template). Unchecking that option may resolve most of the more significant formatting-change issues. The apparent deletion of spaces in a Word 2010 document is a known issue for Word 2007 installations that haven't been updated in a long time - it was fixed years ago (see http://support.microsoft.com/kb/2633585). So, if someone is seeing this behaviour on a Word 2007 installation, it's because it's long overdue for an update to SP3.

suzanne s. barnhill say:

Seeing the TOC in blue without page numbers suggests that the user may be viewing it in Web Layout view. There is a workaround for this: delete the \h switch from the TOC field. This corresponds to the check box for "Use hyperlinks instead of page numbers" in the TOC dialog (this option applies only to Web Layout and some reading views). Even if the entries themselves are not hyperlinked, the page numbers still will be.

You could also tell recipients not to open the document in Full Screen Reading view (Reading Layout view in Word 2003, Read Mode in Word 2013). Unless they have disabled the option, the default view for a document opened from an email attachment is Full Screen Reading, which is designed for ease of reading and does not represent what will print.

The display of TOCs in reading view varies with the version: Word 2003 shows only the entries (no page numbers), but they are not formatted as hyperlinks (blue and/or underlined); Word 2007 and 2010 show the entries and page numbers, with period leaders, just as in Print Layout; Word 2013 shows only the entries (no page numbers), and they are blue and underlined. All versions show the entries blue and underlined (without page numbers) in Web Layout view.
