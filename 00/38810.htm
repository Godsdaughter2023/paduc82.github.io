  1. Create a Windows Repair CD via the Control Panel / Backup and Restore. Alternatively, copy a Windows 7 installation DVD to your flash disk.
  2. Run this command from a Command Prompt:
    D:\Boot\BOOTSECT.EXE /NT60 G:
    (assuming that Drive D: contains your Windows 7 installation DVD and that G: is your flash disk).
