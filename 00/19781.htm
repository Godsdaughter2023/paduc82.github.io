jim gordon mvp say:
My guess is you are saving in old PowerPoint 97 through 2004 format. Don't. Instead have your Windows users update their PowerPoint 2003 to service pack 3 and install the free compatibility pack. Details on how to do this and links to the compatibility pack are in this article:
http://support.microsoft.com/kb/924074

The default behavior for PowerPoint 2010 for Windows and PowerPoint 2011 for Mac is to always embed videos, not link them. Unless you force a link when you insert a video (or audio or picture for that matter), using standard PowerPoint file format (.pptx) instead of (.ppt) will always internally create a relative link, never a full pathway. This solves the problem of moving a presentation from one computer to another and losing the video or audio content. This is analogous to the Save as PowerPoint Package feature of old versions of PowerPoint. The Save As  Package is no longer needed, as the default (.pptx) format essentially is the same concept by default.

By saving in the old (.ppt) format, you force PowerPoint to make an absolute path link to an external file, which is exactly what you don't want to happen.

On your Mac, use the Compatibility Report to check your presentations for issues with 2003 for Windows before you save, but choose PowerPoint format (.pptx), not the old format.

If you insist, you can create a folder, save the presentation into the folder, save the media files into the same folder in .ppt format (not using any subfolders) and then insert with links to the media files, and then save in (.ppt) format again to create a presentation with relative links.
