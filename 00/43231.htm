ralf_g say:

Just as an FYI, accessing a computer's BIOS is done via hardware (firmware), not software.  BIOS settings are accessed before the computer boots any OS so no operating system is required or involved in the process. If the manufacturer's prescribed keystrokes to access the BIOS don't work then there is potentially a hardware issue. Dual booting could be another potential cause for the original problem, not that you've mentioned ever doing that.

WRT file encryption, I've installed Windows 10 numerous times and done all of the updates but Windows has never spontaneously encrypted any user files. There are viruses which do this however... a possibility to consider.

Another possibility for not being able to access your files is that because you've changed the OS you are no longer registered with the same User ID that created the files you are trying to access, so ordinary Windows security is blocking your current user ID from accessing them. The usual way around that is to have your current user ID take ownership of the folders/files in question.

See this link for instructions on taking ownership:

https://technet.microsoft.com/en-us/library/cc753659.aspx
