a. user say:

Hello,

Thank you for posting your query in Microsoft Office community.

I understand your concern and will help you with the issue.

Follow the steps below to create an envelope and add logo to return address on envelope.

Create the Envelope

  1. Open Word > Open blank document.

  2. Click Mailings tab > select Envelopes located on the left side of the ribbon.

  3. Type the delivery address in the text field at the top of the Envelopes and Labels dialog box and type your return address in the text field below it.

  4. Click "Add to Document" button to insert the envelope into the open Word document. It will appear as a separate section at the top of the document window.

                                              

Add logo to return address on envelope

  1. Click to the left of the envelope's return address to place your cursor there.

  2. Go to Insert tab > select the Picture and insert desired logo.

  3. Double-click on logo to open the Format tab on the ribbon > Click the Position to access a drop-down menu of different picture positions and text wrapping options.

  4. Select the "Top left with square text wrapping" position to line the logo up precisely with the return address in the upper left corner of the envelope.

  5. Select "More Options" from the Position menu and "In Front of Text" if you want to position the logo somewhere else on the envelope that is not relative to the return address text.

  6. Use the "Save As" command under the File tab to save the document that contains the envelope.

john mcghie say:
It is a recognised issue with an explanation, but I would not consider the explanation "simple"  :-)

There are two ways to apply numbering: with a style, or directly.  If you apply numbering directly, Word cannot keep track of which list is which, so you get this problem.

Define your numbering format into your style, and use the style (for example, the style "List Number", which is already built-in to every document) to apply it.

That's the cure to this problem.
