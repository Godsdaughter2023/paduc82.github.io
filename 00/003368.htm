Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell K2 & copy down:

=IFERROR(INDEX($B$1:$J$1,,SMALL(IF(COUNTIF($J2:J2,$B$1:$J$1)=0,IF($B2:$J2>0,COLUMN($B$1:$J$1)-COLUMN($B$1)+1)),1)),"")

robert__p say:

Hi Deneen,

Do you mean to change the shared mailbox folder name in Outlook Web App(https://outlook.office365.com/owa)?

If yes, generally it needs some time to sync the updates. Please right click the shared mailbox folder to remove it. Then right click your name in the left panel and select Add shared folder… to add the shared mailbox to check if the name has changed.

If the name still the same, please follow the steps below to change the display name immediately:

1. Sign in to Portal(https://portal.office.com) with your admin account.

2. Navigate to Admin—SHARED MAILBOXES, click the shared mailbox to change the name first, click save and then change the address of the shared mailbox, click save at last.

3. In Outlook Web App, right click the shared mailbox to remove it, right click your name to open the shared mailbox with the new email address and new display name.

4. Sign in to Admin Portal to change the email address to the original one. Then the display name should be the one you want to change to in Outlook Web App.

Please note that the emails in the shared mailbox will not be affected during the above procedure.

Please let me know if anything is misunderstood.

sheen valentine ali msft say:

Hi Martine,

We'd like to clarify some things before moving forward.

1. OneNote for Mac syncs your notebook to your Microsoft account or your Work/school account, not to your computer account. So switching computer accounts will not affect anything.

2, If you want to sync the same notebook on different computers or computer accounts, you should sign-in to the same Microsoft or Work/school account and use the exact same notebook.

To do this, you can sign-in to OneNote Online, open the notebook, then click File > Edit in Microsoft OneNote.
