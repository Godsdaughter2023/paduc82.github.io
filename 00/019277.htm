Hi onad,

Firstly, in the UI, it is an expected behavior that it will only let you add one user at a time.

However, you can use PowerShell to assign permission to multiple users at one time.

In Office 365 Online, the Default permission has been given. So you need to remove it before you run the Add-PublicFolderClientPermission cmdlet. After you connect your PowerShell to Exchange Online, you can run the following cmdlets:

1.Remove-PublicFolderClientPermission -Identity "\PF" -User Default

2.Add-PublicFolderClientPermission "\PF" -User Default -AccessRights Editor (Editor permission is enough for the users to read and write)

To confirm it, run 3. Get-PublicFolderClientPermission -Identity "\PF"

Subfolder exclusion - Microsoft Community

johnathanstein say:

Subfolder exclusion

If one folder is added to "Excluded files and locations", are all its subfolders also excluded?
kosh vorlon say:

Hi Johnathan,

 

Yes.  Not only the sub-folders but all the files in the primary folder as well as the files in the sub-folders (and no way to avoid this if that's where this question was heading).

 

This should be done ONLY to avoid false-positive scan results or if MSE is blocking the underlying function or program from working.  It should not be used lightly and I would expect it to be needed exceeding rarely.  I have no exclusions and never have.  Remember, every exclusion represents a hole in your security. 

 

There are often other ways around problems where the initial reaction is to exclude the file and/or folder and those should be done before resorting to this.  If doing this because scans are freezing on a certain folder, then while this is an OK temporary workaround (though it involves risk if the reason for the freeze is actually malware), the proper solution is to find the underlying reason for the problem and fix that and keep the exclusions in place for as short a time as possible.  If it occurs only during scans, you may want to consider going to manual scans and excluding them only for the scan and then removing the exclusion.  If it occurs during real-time protection, then leaving them in place until the problem is resolved is really the only choice (unless it involes a rarely used feature, and then the add/remove procedure may be best here as well).

 

I hope this helps.

 

Good luck!

Product Key Cards PKC / OEM / MLK(2007):

·For use on new, preloaded PCs only

Microsoft® Office 2010 Product Key Cards include a 25-character Product Key to activate only one PC preloaded with Office 2010 software. No disc needed. Backup copy available for purchase. The PKC license is not transferable to another PC.

OEM PreInstallation Kit - OPK:

Traditional Discs FPP:

·Home and Student: 1 license, up to 3 devices within same household*

·Home and Business and Professional: 1 license, up to 2 devices for same user

·License is transferable to another PC

·Includes media for use on uncovered PCs (PCs that are not preloaded with Office) and for backup

This license type, often referred to as Full Packaged Product (FPP) in the past, includes a DVD, a 25-character Product Key, and a Quick Start Guide. For example, Office Home and Student 2010 can be used on up to three PCs in a single household, but may not be used for any commercial, nonprofit, or revenue-generating activities. A Traditional Disc product license can be transferred to a different PC if the user upgrades to a new PC.

Copy contacts or pictures from your old deviceWant to copy important information from your previous compatible Nokia device and start using your new device quickly? Use the Phone switch application to copy, for example, contacts, calendar entries, and pictures to your new device, free of charge.

Select Menu > Settings > Sync & backup.

Both devices must have Bluetooth activated.

Select Phone switch > Copy to this.

Select the content to copy and Done.

Select your previous device from the list. If the device is not listed, make sure Bluetooth is activated in the device.

If the other device requires a passcode, enter the passcode. The passcode, which you can define yourself, must be entered in both devices. The passcode in some devices is fixed. For details, see the user guide of the device.

The passcode is valid only for the current connection.

If prompted, allow connection and copying
