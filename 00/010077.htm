First thing to do is go into Insert tab / Text group / Quick Parts drop down / Building Blocks Organaizer command to Display the Building Blocks Organizer dialog. With the latest build, Microsoft has added a new history menu in Microsoft Edge — you can now right-click on the back/forward buttons in Edge to quickly access the web sites you visited recently. Apart from this, there are not any other new features.

Of course, there are some known issues and bugs in the latest build, which are listed below:

Some PC games will crash switching from windowed mode to full screen, upon game resolution change, or upon launch due to a bug in Windows graphics stack. We have observed this with The Witcher 3, Fallout 4, Tomb Raider, Assassin’s Creed, and Metal Gear Solid V but it may occur with other titles as well.

With this build (and with the last build), applications such as Narrator, Magnifier, and third-party assistive technologies may experience intermittent issues or crashes. If you rely on screen readers or other software, you should not use this build. This issue will be fixed with the next build.

You might see a WSClient.dll error dialog after logging in. We’re working on a fix for this but as a workaround, you can run the following in Command Prompt with administrative rights: schtasks /delete /TN “\Microsoft\Windows\WS\WSRefreshBannedAppsListTask” /F

While attempting to update to this build, your PC may show a message that your wireless card is not compatible with Windows 10. The workaround is to visit the support page for your PC or wireless card and install the newest driver that is available.

The Connect button does not show up in Action Center.

The latest build is now available to Windows Insiders via Windows Update. To get the latest build, open up Settings, head over to Updates & Security and Check for updates to download the latest build.

Happy downloading, everyone!

You can create a Registry entry and deploy it by group policy.

Outlook 2016

Thing #2: Isn't there "academic" software distributed directly within an institution's Volume License for a nominal (~$20) fee per copy?

@BagaMcguirk: The license does not actually expire. According to the EULA for Academic Software:
"10. ACADEMIC SOFTWARE. You must be a “Qualified Educational User” to use software marked as “Academic”. If you do not know whether you are a Qualified Educational User, visit www.microsoft.com/education or contact the Microsoft affiliate serving your country."

I do not encourage anyone to violate legal agreements, but from a realistic perspective, however:

IMHO, It's highly doubtful that MS will track you down 10 years from now to see if you're still in school :-) Besides, any given release is only supported & updated for a limited time. You'll eventually upgrade to a newer release at which time you'll have to determine what you qualify for.

I'm writing a book just for giggles, but I want to be able to print back to back and I can't find where to go to change my printing or my preview to back-to back.... HELP!
suzanne s. barnhill say:
If your printer is capable of duplexing (printing double-sided), there will be a setting for this in the printer Properties. If not, you'll need to use Word's manual duplexing feature (in the Print dialog). Alternatively, you can manually print all the odd pages, turn the paper over, and then print the even pages after enabling "Reverse print order." If the document contains an odd number of pages, remove the last page before printing the second side.
