One program that can show formatting marks is Office Outlook.

Turn formatting marks on or off

When composing an email message, you might see some symbols within your text. These are actually formatting marks, such as dots (for spaces) or arrows (for tab characters) in Outlook. Formatting marks assist with text layout. They do not appear on a printed message.

If you have tried but been unsuccessful using the button to toggle formatting marks, you need to use the Display Editor Options dialog box described below to reset your display preferences.

When you turn formatting marks on or off, your choice remains in effect for all subsequent messages you compose, reply to, or forward.

Quickly toggle formatting marks

To turn formatting marks on or off, do the following:

  • In the message window, on the Format Text tab, in the Paragraph group, click the button that looks like a paragraph mark. (When you point your mouse at the button, the tooltip says Show/Hide ¶).

    Keyboard shortcut     CTRL+SHIFT+*. (You must use the asterisk on the 8 key.)

Two controls affect the visibility of formatting marks

Two different controls in Outlook 2016, 2013, and 2010 affect which formatting marks are visible when you compose an email message:

Editor Display options

These settings are always in effect. The only way to change them is in the dialog box.

The Show/Hide ¶ command on the ribbon

This quick command toggles the display of any formatting marks that aren't turned on in the Editor Display Options described above.

Toggle selected formatting marks

(Outlook 2016, 2013, and 2010) If you want certain formatting marks to always be visible, use Display Editor Options. The marks you turn on are not affected by the Show/Hide ¶ command on the ribbon.

  1. In Outlook, choose File > Options > Mail > Editor Options > Advanced >Display.

  2. Use the check boxes to specify the formatting marks that you want to be visible. Or use Show all formatting marks to make all marks visible all the time.

Editing a message with or without formatting marks is a personal preference. You might find them more useful when you are designing content with a specific look — or layout — in mind. Formatting marks are turned off by default in Microsoft Outlook.

rob koch say:

If those of you having these problems with Windows Defender and Windows 10 itself upgraded an older system from an earlier version of Windows, this is likely where your problems began.  Windows upgrades have always been notorious for carrying along any issues that existed in the previous operating system installation, regardless of whether they were apparent to the person using the PC.

Anyone skilled in the use and operation of Windows based computer systems, will tell you that the only method they use when upgrading is a clean install, wiping the hard drive completely and re-installing all applications.

It would be really nice if the Windows upgrade process always resulted in a perfectly stable installation, but in my experience this has almost never been true.  The most effective way out of these issues is typically to collect all of the data backups and other information necessary such as the Windows 10 and other installed software license numbers before performing a complete Windows 10 reinstallation.

If any of you are having these issues with a new PC that was purchased with Windows 10 already installed, you'd be best off contacting whomever the PC was purchased from for support.  If this isn't available, then performing a reinstallation via either recovery or other disks/drives that you created or received with the new PC may be your best method as well.

Attempting to fix problems with Windows Defender or Windows itself in place might work, but if the problems are widespread this may not work.  I've personally found that the time spent performing all of the possible steps required to try and fix a damaged operating system installation is better spent preparing for and performing a clean install.  This is because it not only forces me to create the appropriate data backups and license records for the PC, but also provides the best assurance of a successful resolution to the existing problems.

This is obviously only my personal opinion and some others may offer many different options to attempt to fix the problems individually.  It's fine to try these if you wish, but be aware that unless the problems are simple it's common to spend far more time attempting to fix them individually than the backups and reinstallation of Windows and a handful of truly important applications.  Only large quantities of personal data and/or important applications typically make this process longer.

Rob

rob koch say:

Adware:JS/Pornpop.A is not malware, it is Adware as the threat name indicates and not truly something you need to worry about.  All it does is attempt to display a pop-under advertisement, which really isn't a big deal by itself.

Because you apparently weren't sitting at the keyboard when this occured to see and indicate to the MSE alert screen that you didn't wish this to be displayed, MSe performed the default for this moderate level item and allowed it.  Since there's really no damage done, nor any leftover remnants after this other than maybe a pop-under browser window that needed to be closed, there's really nothing you need to be concerned about.

The only question here is how did this occur when you were away, i.e. who was using the PC or did some other web page you had open generate the pop-under request since this isn't always the result of a pornographic site, it's just been given that name since that's where they most commonly occur.

 

Here's the description of this detection from the MPCC Threat Encyclopedia:

http://www.microsoft.com/security/portal/Threat/Encyclopedia/Entry.aspx?Name=Adware%3aJS%2fPornpop.A

Technical Information (Analysis)

Adware:JS/Pornpop.A is a detection for specially-crafted JavaScript-enabled objects that attempt to display pop-under advertisements, usually with adult content.

It is common for this detection to trigger in a user's Internet cache. This may indicate that the user has recently visited a website that attempted to display pop-under advertisements.

As a JavaScript-enabled object, Adware:JS/Pornpop can run in any JavaScript-enabled browser.
