stefan blom say:
To create a SEQ field, you can do the following: Press Ctrl+F9; Word adds field delimimiters, { }. (Observe that you cannot create the field delimiters by typing them from the keyboard or pasting them in from this message.) Within the delimiters, type SEQ test. In other words, you now have { SEQ test }. Press F9 to update.
 
To create additional fields, you can copy and paste the one you already have created. You'll notice that the fields won't update automatically, but you can force an update by switching to Print Preview and then back to your favorite view (first press Ctrl+F2 and then press ESC).
 
The field construct that you need in this case is: { STYLEREF 1 \n }.{ SEQ test \s 1 } which will produce the current Heading 1 number followed by a period followed by the sequence number (the \s switch tells Word to restart numbering whenever it encounters a Heading 1 paragraph). You can save the whole construct as an AutoText entry and insert it as needed.

suzanne s. barnhill say:
And the advantage of saving it as an AutoText entry is that when you insert the SEQ field it will have the correct value (won't need to be updated), though it will not update automatically if you later add or remove other captions above it.
