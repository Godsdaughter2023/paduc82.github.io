suzanne s. barnhill say:

You probably won't be writing a thesis again soon, but for any similar long document, be aware that Word can handle documents of many hundreds of pages with ease, and it is much simpler to create a long document as a single file from the outset than to try to combine separate files at the end (especially under a deadline, when it is guaranteed that Murphy's Law will operate to the fullest extent possible).

If you want your figure numbers to be continuous throughout the document, then you probably need do nothing further, but if you want them to restart numbering by chapter, then you probably want to use the "Include chapter number" feature, which relies on your having the chapters numbered with an autonumbered paragraph in one of Word's built-in heading styles. You'll want to update your autonumbering when you combine files, anyway; for more on this, see http://www.shaunakelly.com/word/numbering/numbering20072010.html.

There's more on the StyleRef field at http://WordFAQs.mvps.org/StyleRef.htm. Although the StyleRef field is helpful to obviate the necessity to unlink section headers and type the content for each one separately, you'll probably still want to have section breaks between chapters so that you can use "Different first page" to avoid having a header on the first page of a chapter (by leaving the First Page Header empty).

2. Assuming that the caption labels are identical, you can update the caption numbering in the combined document, for example by switching to the Backstage view and then back to the previous view (use Ctrl+F2 followed by ESC).

No need to delete and re-add. Simplest/cleanest approach would be to create a new Outlook profile.

- With Outlook closed, start the Control Panel <Mail> app
- Add a new Outlook profile (no need to delete the old one)
- Configure your email accounts
- Make sure you set the newly created profile as the default for the next time Outlook is opened

All done
