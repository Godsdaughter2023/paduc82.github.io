Why is there no support to watch dvd movies in media player?

Hi

This capability was removed in Windows 8x. It has to do with licensing issues for some of the proprietary DVD Video Codecs.

You can view the following article written by the Windows 8 developers about why this was done.

FAQ - DVD playback and Windows Media Center in Windows 8:

http://blogs.msdn.com/b/b8/archive/2012/05/04/q-amp-a-dvd-playback-and-windows-media-center-in-windows-8.aspx

You can download the free VLC Media Player. It supports all types of DVD Video playback.

VideoLAN - Official page for free VLC media player:
http://www.videolan.org/vlc/

I got Wifi working finally for Opera Mini on my Nokia C3-01 phone. It was working fine for browser and mail application though.

 

I did the following.

1) In Settings >> Configuration >> Def. config. sett.

Delete all entries.

2) Create a access point manually using Settings >> Configuration>>Personal Settings.

It is necessary to create one. It will still work even though it is not pointing to real access point. Just point it to dummy Pack. data acc. pt. access point say "foo".

3) Set the access point created in step 2 as Preferred acc. pt using Settings >> Configuration>>Preferred acc. pt

4) Make sure you had set WLAN >> Internet Conn. to Ask first.

 

Now apps like Opera Mini starts Asking for Access point whenever connecting to internet.

It shows mutiple access point in the list. Usually the first one is your wifi access point and second one is the dummy access point you had as preferred access point.

 

Using the above steps i am able to point wifi access point to access internet in all apps available in my phone. If you want to still access your gprs access point too then point the preferred access point created in step 2 to real gprs access point instead of "foo". Contact your operator to get configuration details for real gprs access point or check out how it is configured in your friends phone using the same operator as yours. Just make sure you are doing it manually and are not using the configuration profile sent by operator direcly to your phone. It blocks wifi access point access on your phone.

    Start the pairing from the PC in devices and printers, add a device. If you have previously paired, unpair and start over.

     

    Once paired with a laptop of tablet that is all that is needed. The connection does not have to be maintained so the phone disconnects. They will be paired, they will not show connected.
