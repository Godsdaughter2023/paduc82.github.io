I think what you are looking for is how to turn on the fast boot in Windows 10.

1) Go to settings (gear icon) from Windows (start) menu

2) Search Power Options

3) Select "Choose what the power buttons do"

4) You should see the options for fast startup

You can update the settings with an admin user account by clicking "Change settings...".

greg carmack - windows mvp 2010-2020 say:
Hi Al.

Go to Settings > System > Power & Sleep > Additional Power Settings > Choose what Power buttons do > Change settings that are unavailable > check Fast Startup.

This is a hybrid Hibernate that replaces ShutDown with Hibernate to have a faster Startup. It does not however have the normal Hibernate function or saving anything you leave out to a Hibernate file before shutting down and then replacing it on the desktop when powering back up.

Sometimes problems can be solved by disabling it, or at least I'd test that to see.

Win7 had a 'Backup now' reminder - has Win10? - Microsoft Community

taffy087 say:

Win7 had a 'Backup now' reminder - has Win10?

On the first day of each month, Windows Action Centre send me a reminder to back up my PC.

I didn't have one this morning on my Win10. Is that procedure now defunct?

taffy087 say:

No reply from the Support Engineer for 12 days!

I've 'accidentally' got the answer, from SomeHope_1 in another thread. I've posted it here in the hope it will help others with this query on such an important topic. Thank you SomeHope_1.

"No it is not defuncted.

If you have a desktop computer, and two hard drives installed in it or more as I have (3 Hard Drives), you have two choices.

1) you can use the file history as you mentioned, to do a system image backup (that include every thing on your primary HD).

2) you can go to the Control  Panel and click on the "Backup and Restore" icon, and also do a System Image backup or use the "Backup or restore your files" option.

Do not forget to create a system repair disc.

If you only have 1 hard drive installed (Primary HD). You will have to create a System restore point. Usually Windows does that on it own, but with all the issues Win.10 has, who knows, maybe Windows forgets to do it.

1) Right click the Start menu icon in the far left of the taskbar.

2) Click on System.

3) Click on System protection.

4) Make sure the Local Disc (C:) (System)is highlighted in blue, if not click on it once,

    also that it is turned on, if not click on Configure and turn it on and click on OK.

5) Now click on Create.
