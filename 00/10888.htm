One global procedure that may help is Font Weeding. Follow that procedure to clear up font conflicts. It wouldn't be a bad idea to run Disk Utility to repair disk permissions afterward. Then do a full Shut Down for a minute or two rather than a Restart.

  6. Then save the new document.

will return any given percentile of a recordset, but will be very slow with a large recordset because of the nested domain function calls.  Translating the logic into a query might give a little improvement, e.g. to return the 75th percentile of a calendar table from 2000-01-01 to 2020-12-31 (7671 rows)

SELECT MIN(calDate) AS Percentile75
FROM Calendar AS T1
WHERE (SELECT COUNT (*)
               FROM Calendar As T2
               WHERE T2.caldate <= T1.calDate)
          >= ((SELECT COUNT(*)
                   FROM Calendar) * 0.75);

Add a background style to your presentation
  1. Click the slide or slides that you want to add a background style to.

    To select multiple slides, click the first slide, and then press and hold CTRL while you click the other slides.

    • To apply the background style to the selected slides, click Apply to Selected Slides.
    • To apply the background style to all of the slides in your presentation, click Apply to All Slides.

This option is available only when your presentation contains multiple slide masters.

      Notes

    • To add the background styles gallery to the Quick Access Toolbar, right-click a thumbnail, and then click Add Gallery to Quick Access Toolbar.
