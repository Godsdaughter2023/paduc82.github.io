Hi,

Thank you for posting in Microsoft Community. I understand how frustrating it is when things aren't the way it is suppose to be. Sorry for the inconvenience caused.

As per description, I understand that you are unable to use apps as you get the error message "This  app has been blocked by your administrator" . I'll do my best to ensure a quick resolution.

Please let us know if your computer is connected to a domain. If so, please contact your Administrator.

Otherwise you may try this:

1: Press Windows +X and select Command Prompt Admin

2: Type the following command without quotes at the Command Prompt and then hit Enter:

net user administrator /active:yes

You should see the message saying "command completed successfully".

 3: Launch Start Menu  and right-click on user-account title at the top-left of the screen. Then click Sign-out.

4: Log into the Administrator account and you will return to the Desktop of that user profile.

5: Locate and run the .exe file for the program which you want to run.

6: As you have already logged into the Administrator account, the Smartscreen protection will be disabled and the installation should complete successfully.

7: Log back into your regular account after logging out of the Admin account.

 8: Disable the Administrator account that was used earlier to prevent malicious access to your system files.

Type the following command from Command Prompt: net user administrator /active:no

Hope this post helps. Get back to us for further queries. We are happy to help!
