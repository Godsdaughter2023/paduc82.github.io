How do I bold all instances of a string using Word VBA - Microsoft Community

la lawyer say:

How do I bold all instances of a string using Word VBA

I want bold all instances of a string of characters (usually a word) in Word 2010 VBA?

I presume that I would use the Range object but I don't know how to use that to bold all relevant instances of a string.

TIA
hansv mvp say:

Try a macro like this:

 

Sub BoldEmAll()
    With ActiveDocument.Content.Find
        .ClearFormatting
        ' Substitute the text you want to make bold
        .Text = "mystring"
        .Replacement.ClearFormatting
        .Replacement.Font.Bold = True
        .Replacement.Text = "^&"
        .Format = True
        .Forward = True
        .Wrap = wdFindStop
        .MatchCase = False
        .MatchWholeWord = False
        .MatchWildcards = False
        .MatchSoundsLike = False
        .MatchAllWordForms = False
        .Execute Replace:=wdReplaceAll
    End With
End Sub

rolando vil say:

Hi David,

Color and Appearance still exists in Windows 10 but it is not easy to access. As you know, the link to Color and Appearance has been removed from the Control Panel. Thus, we suggest that you follow the steps below to access Color and Appearance:

Method 1: Open Color and Appearance via Run command.

  • Press Win + R to open the Run command box.
  • In the command box, type Control Color, and press Enter to open the Color and Appearance window.

Method 2: Create a shortcut of Color and Appearance.

  • Right-click on the desktop, press New, and select Shortcut.
  • Type rundll32.exe shell32.dll,Control_RunDLL desk.cpl,Advanced,@Advanced in the Type the location of the item: box

  • Press the Next button.
  • Type Color and Appearance under Type a name for this shortcut, tap the Finish button to create the shortcut of Color and Appearance on your desktop.

You may leave the shortcut on your desktop or pin it to the taskbar to make it easily accessible.

Let us know if you need further assistance.

I had a bright idea for a temporary fix for Word.  I don't like the idea of actually changing the background colour because it means it will be coloured for other people, and coloured when you try to print.  I thought of temporarily changing the colour of the background, but it's quite a number of clicks to change the colour each time.... So I added macros (and custom buttons in the ribbon) to add and remove the background colour quickly.  Not ideal but better than nothing.

So, other people who are experiencing the same problem with Windows 8 and 10 might want to try the following macros:

Sub PageBlue()
'
' PageBlue Macro
'
'
    ActiveDocument.ActiveWindow.View.DisplayBackgrounds = True
    ActiveDocument.Background.Fill.Visible = True
    ActiveDocument.Background.Fill.ForeColor.RGB = RGB(217, 226, 243)
    ActiveDocument.Background.Fill.Solid
End Sub
Sub BlueOff()
'
' BlueOff Macro
'
'
    ActiveDocument.Background.Fill.Visible = False
End Sub

If you want a different colour that blue, choose the colour that suits you, note down the RGB settings from the colour palette, then edit the bit that says "RGB(217, 226, 243)" to be the colour that you prefer.
