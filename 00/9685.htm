john w. vinson say:

The "Data Entry" property is confusing, or downright misleading. It means that you can ONLY use the form for data entry - adding new records. If it is TRUE then you will not be able to see existing records. A default form (Allow Additions = Yes, Allow Edits = Yes) will let you both add new records and edit old ones.

Setting Allow Edits to No means that you can't change anything in any record, new or old. It's useful if you just want the user to view records but not do anything to them.

The combination of Data Entry = Yes and Allow Edits = No will indeed cause the form to vanish. The former means that the form won't show any existing records; the latter means that it won't show the blank "new record" either. Thus it just comes up blank. How to get Excel to automatically highlight rows depending on dropdown - Microsoft Community

vj ravi say:

How to get Excel to automatically highlight rows depending on dropdown list selection in that row.

For instance. If I have a dropdown menu in a cell that lists priority (High, Medium, Low), and High is selected, is there a way to get Excel to automatically highlight that entire row in a specific color (or better, specific cells in that row)? I'd like the program to highlight in different colors depending on whether High, Medium, or Low is selected.

Thanks,

Vijay

bernie deitrick say:

Use Conditional Formatting, with the Formula option, selecting all the cells in the row to be highlighted, using formulas like

=$F2="High"       

=$F2="Medium"

=$F2="Low"

where F has the dropdown list.  The formula will update as it is copied to other rows, but will always refer to column F.

mike h.. say:

Hi,

Yes you can do that with conditional formatting. See the workbook I made to demonstrate this on my OneDrive called HighLightRows

=$A7="High"
