rohn007 say:
Probably, it depends specifically how you bought Office.

If you bought it factory installed and activated, an OEM license, or you bought a Medialess Kit, MKL license, then no you cannot transfer it. Those licenses are only good on the first computer they are installed on. They only way you can give your friend Office, is to give them the whole computer.

    Third-party font activation in Word 2011 - Microsoft Community

    fontgeek say:

    Third-party font activation in Word 2011

    It appears that fonts activated after Word 2011 do not appear available in the font list until Word 2011 is closed and relaunched? Earlier versions of word appeared to allow activation of fonts through a third party font manager such as FontAgent Pro. It appear that fonts that are added to a system font folder after Word is launched will not be added to the list until Word is closed and reopened. It appears that Word 2011 does not dynamically update its font list. Is this correct?

    john mcghie say:
    Good Question.

    I did not think that Word ever "dynamically" updated the font list.  Word expects all of the fonts to be enabled all of the time.  If they are, Word starts and runs faster.

    Word relies on the System to tell it when the font list has changed, and when it gets that notification, it updates its font cache.

    If you can, change your working method so you never use the Font Menu.  Word is much easier to use if you store all of your formatting, including fonts, in a selection of styles.  Then you can simply apply the required style and know that not just the font but all of the formatting is exactly correct and consistent throughout the document.  You never have to check: if the correct style is applied, the formatting is correct, end of...

    I don't think there has been a change in this regard: not since Word 2008, in any case.  I know they did some work to improve performance in Word, and that may mean it takes longer to "notice" that the font list has updated.

    My suggestion is to use styles exclusively: the font in a style will be applied correctly whether the font currently appears in the font list or not.  If the font is not currently on the system, Word will "fake" the missing font by adjusting the image of either Calibri or Cambria.  When the correct font becomes available, it will be silently re-instated.

    Hope this helps

    stefan blom say:
    Are you saying that you are using the master document feature? In that case, you really shouldn't, because it isn't stable. See http://word.mvps.org/FAQs/general/whymasterdocscorrupt.htm.

    Keeping everything in one file is usually the best solution. You can then use the instructions at http://shaunakelly.com/word/numbering/numberingappendixes.html to create appendix numbering.

    If you really need separate files, look into the INCLUDETEXT field. You can use the Insert File dialog box to create the "master" document. See http://word.mvps.org/faqs/tblsfldsfms/includetextfields.htm. Note that the article hasn't been updated for the most recent versions of Word. In ribbon versions, click the Insert tab, click the arrow on the Object button, and then click Text from File. (The command "Text from File" is misleading, because all the content in the inserted file will be included.)
