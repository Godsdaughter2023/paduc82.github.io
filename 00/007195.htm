ken sheridan say: I can't think of any reason, at least none that are legal, honest and decent, why you'd want to create more than one PDF file of the same invoice, but it's easy enough to do.  With my demo file the button's code would be amended as italicized below:

Private Sub cmdPDF_Click()

  On Error GoTo Err_Handler
    
    Const MESSAGE_TEXT1 = "No current invoice."
    Const MESSAGE_TEXT2 = "No folder set for storing PDF files."
    Dim strFullPath As String
    Dim varFolder As Variant
    Dim n As Integer
    
    If Not IsNull(Me.InvoiceNumber) Then
        ' build path to save PDF file
        varFolder = DLookup("Folderpath", "pdfFolder")
        If IsNull(varFolder) Then
            MsgBox MESSAGE_TEXT2, vbExclamation, "Invalid Operation"
        Else
            n = 1
            strFullPath = varFolder & "\" & Me.Customer.Column(1) & _
                Me.InvoiceNumber & "_" & Format(n, "00") & ".pdf"
            ' get next serial number for PDF file
            Do While Dir(strFullPath) <> ""
                n = n + 1
                strFullPath = varFolder & "\" & Me.Customer.Column(1) & _
                    Me.InvoiceNumber & "_" & Format(n, "00") & ".pdf"
            Loop
            ' ensure current record is saved before creating PDF file
            Me.Dirty = False
            DoCmd.OutputTo acOutputReport, "rptInvoice", acFormatPDF, strFullPath, True
        End If
    Else
        MsgBox MESSAGE_TEXT1, vbExclamation, "Invalid Operation"
    End If

Exit_Here:
    Exit Sub
    
Err_Handler:
    MsgBox Err.Description
    Resume Exit_Here

End Sub

For Windows Phone 8.1, Cortana is available in the U.S., China, and the UK as a “beta,” and in Canada, India, Australia, France, Germany, Italy, and Spain as an “alpha”.

Settings to enable Cortana with Windows Phone 8.1:

8.10.14219.341 Italia italiano (Italia) Italiano (Italia)

When properly enabled, the settings should appear like this example for India:

Region should already be set based on where you live.

If you have more than one language, tap and hold the language appropriate for your region and move it to the top.

Note: If your phone was manufactured for another region, it may not be possible to enable the necessary phone language.

Set Speech language to the choice appropriate for your region.

Note: If you see the error "I'm sorry, but I can't speak your language yet" when starting Cortana, ensure that the speech setting matches the configuration above for your region.
