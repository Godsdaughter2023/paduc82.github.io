Downloading MS Safety Scanner to a flash drive.. Please help! - Microsoft Community

another virus? - really? say:

Downloading MS Safety Scanner to a flash drive.. Please help!

Could someone please explain how to download and run the new MS Safety Scanner software from a flash drive.. Thnx
kosh vorlon say:

Hi Is ths fun - really,

 

It can be downloaded to a flash drive like any other download by saving as and selecting the flash drive.  However, to install and run it, it must be installed onto the system drive of the computer you intend to scan.  You can install it from the flash drive as you would install any other download on a flash drive (or copy the download to the computer and install it from there if you prefer).  But it isn't entirely an online thing where there's the download and then only a matter of running it - it requires installation.

 

Here's info concerning the system requirements for Safety Scanner: http://www.microsoft.com/security/scanner/en-us/SysReq.aspx.  As you see, it needs an operating system.  It will install things in a special program folder, other things in files throughout the Windows operating system files, and make changes to the registry (as with just about any program).  Otherwise it simply can't work.  The flash drive is just a storage device - it does not have an operating system installed on it, so the Safety Scanner cannot be installed on it.

 

Some programs can be installed and run from flash drives that have U3 capabilities (see: http://www.ehow.com/how_5234669_make-usb-drive-smart-drive.html), but only U3-capable software can be installed and run from these drives and there isn't a huge listing of available options.  Microsoft Safety Scanner is not available in a U3 format.

 

Note that the download and the installation is good for only 2 weeks - then it needs to be re-downloaded and installed if you want to scan with it again.

 

So the short answer is that it can't be done and while you can download to the flash drive if you wish, you must install and run it on the computer itself.

 

I hope this helps.

 

Good luck!

 

P.S.  I contacted Microsoft PCSAFETY and verified this with a technician over the phone just to be certain.

marcsroberts say:

Right, so there is a way to trick the Lumia Software Recovery Tool to flash the 10.02.11 059W256 firmware to the phone. This'll give you English language and English T9 Dictionary.

It's reasonably fiddly though and could very easily break the phone if you get it wrong, so really probably best not tried, but if you don't mind risking it:

You'll need a windows tool like hashtab that will show you the crc hash of files.

Load up the Recovery Tool, hook up your phone and click to install the 10.02.11 software it prompts you to, this will make it download the latest firmware for your variant.

After it is downloaded hit cancel at the next prompt (you don't want to actually install it).

Go to the settings>Network tab and enable offline mode.

Go into the downloads tab and click browse and it'll take you to the RM-1035 folder where there's a bunch of files.

Make a backup of this entire folder (you will need *exactly as it is* it if it goes badly wrong).

Open the RM1035_059WXXX_10.02.11_XXX.vpl file in a text editor (the XXX bits will be different for you).

Go here http://www.mobifiles.net/2014/12/nokia-130-dual-sim-rm-1035-v100211_39.html and download all the files apart from the VPL file to the same folder you're in above, and overwrite any that exist. (These files all have some prefix on the file names that you need to remove so they match names listed on the site).

Now, you want to go through the VPL file (it's an XML file) you opened in the text editor and for each <file> block (but not the signature.bin block at the end, just leave that alone) in the <FileList> block, replace the file names with the new ones you downloaded (some are the same but the crcs are different), set <signed> to false and put the crc for your new file in the <crc> block. Save the file.

Once you've done all these you should be able to go back to the first screen of the recovery software and click install. If all goes well (it did for me) your phone should be English now. I did have a few failed attempts in getting this right, if you get told to take recovery steps, something went wrong, put back the backup you made of the folder and then follow the prompts and you *should* get your phone back on your normal firmware. Like I said, this might all just break your phone so you've been warned.
