Fit table to page - Microsoft Word - Microsoft Community

catherinelambertpt say:

Fit table to page - Microsoft Word

Dear Sir or Madam,

I have been trying to make a weekly schedule using the Microsoft Word programme in the Office Suite. I know it is possible to automatically fit a table to contents in the layout tab. However, I would like to know if I can resize the table to fit the page and therefore have larger cells to write in, when I print it? 

Is there a better programme for this sort of schedule plan? I think Excell has cells, but not the design tools I need. 

Many thanks in advance of your response.

Kind Regards,

Catherine

jay freedman say:

It is possible, but it may take a little trial-and-error.

Start by inserting the table with the number of rows you want. The default width of the table is the distance between the left and right margins, so what remains is to make the height of the rows large enough to fill (but not overfill) the page.

First drag the bottom border of the last row to the bottom margin. This is easiest to do if you set the zoom in Word to 50% or whatever percentage lets you see the entire page at once. If the last row jumps to a second page, move its bottom border up a little until it jumps back.

If you turn on nonprinting characters by clicking the ¶ button on the Home ribbon, you'll see that there's an empty paragraph mark below the table; that paragraph can't be deleted. If you need to squeeze the last bit of space into the table, select that paragraph mark and change its font size to 1 point, then drag the bottom border of the last row down a little more.

Finally, click the Table Tools | Layout tab of the ribbon and click the Distribute Rows button in the Cell Size group.
