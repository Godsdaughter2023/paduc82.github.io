Login Information

  • Username: admin
  • Password: admin

 

 

DHCP Reservation

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "LAN" from the menu at the top.
  5. Select "DHCP Settings" from the sub menu.
  6. Select the "MAC-Base Assignment" button.
  7. Enter the MAC address of the Xbox in the "Host MAC Address (xx-xx-xx-xx-xx-xx)" box.
  8. Enter the desired IP address for the Xbox in the "Assigned IP Address (xxx.xxx.xxx.xxx)" box.
  9. Select Assign IP.

Hi,

Thank you for posting the query on Microsoft Community.

Try to run Hardware Troubleshooter. It checks for common issues and makes sure that any new device or hardware attached to your computer was installed correctly.

a. Press Windows Logo Key+X then click on Search screen.

b. Type “Troubleshooting” without quotes and click on Troubleshooting.

c. In the new window, click View all on the left pane.

d. In the list of options, click on Hardware and devices.

e. Click Next and follow the on-screen instructions.

If the issue persists, then try to uninstall and re-install sound/audio drivers in compatibility mode.

  1. Press Windows Logo Key+X. Click On Device Manager.
  2. Then find the driver that you are trying to uninstall and right-click on it to Uninstall.
  3. Follow onscreen instructions to uninstall it.

Once the driver gets uninstall try to install the sound driver in compatibility mode.

Compatibility mode runs the program using settings from a previous version of Windows.

Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 7 operating system from the drop down menu and proceed with the installation. Once this is done, restart the computer and check if the issue persists.

Run Windows Update and download the latest driver:

How to: Install and Update drivers in Windows 10

yosmerythomas severino say:
Ok so there are two methods to working your printer wirelessly one is wifi direct and the other is via your router. Wifi direct once enabled allows you to print with a router and is a direct wireless connect from your wifi enabled device to your printer. The link below shows you how to do enable and use that method

https://epson.com/faq/SPT_C11CF76201~faq-295053

The other method is a bit more work, firstly you must ensure router is ok with wireless printing and you are using a 2.4ghz connection per the printer manual, also you must enter the appropriate information to allow your printer access to your network via the wifi settings. After all that's checked and double checked you need to ensure the software on the computer is up to date and acknowledges you have a wireless printer on your network.

Hi Lazaro. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

There is only one proper way to set up a printer correctly so that it has all of it's needed functions and remains that way. This is NEVER to use the basic driver WIndows may install as it is never fully featured and rarely lasts.

Go to the Printer maker's Support Downloads web page for your exact printer model to download and install the printer driver. While there check for newer firmware to install first if available.

Power up printer and follow the prompts as the software installs.

If this fails or you ever have problems then clean out all printer drivers first that may conflict following this tutorial: https://www.tenforums.com/tutorials/101274-unin...
