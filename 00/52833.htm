Hi ianpark01,

 

This issue occurs when Disk Cleanup come across a corrupt file that can cause it to stop working. This usually happens when you have installed and removed programs/updates from your computer. The most common files that cause this problem are the Windows temporary files and the compressed old files. Therefore to make disk cleanup work properly we need to clean out these files manually.

 

Steps on how to remove a corrupt temporary file so disk cleanup will work again:

a.    Close any programs that are running.

b.    Click on Start; in the start search type “%temp%”(without quotes) which will open your Temp folder.

c.    Select all files and delete them. This is what would have happened if disc cleanup had worked…They are temporary files and not needed to run the computer.

d.    Close this window.

e.    Now open Internet Explorer.

f.     Click on Tools and then on Internet Options.

g.    On the general tab under browsing history, click delete.

h.    Then press ‘delete all’.

This will delete all the browsing history, cookies and temporary internet files.

 

Now you can try running disk cleanup. In most cases it will work now.

 

Hope this information is helpful and let me know if you need any further assistance.

deadmanwalking86 say:
  1. Right-click the desktop. Choose "Personalize."
  2. Click Window Color and Appearance.
  3. Click Advanced Appearance Settings
  4. Go through each item and reset fonts (where appropriate) to Segoe UI 9pt, not bold, not italic. (All the settings in a default Win7 or Vista machine will be Segoe UI 9pt.)

If this doesn't work -- or I should say, if all your settings here are already Segoe UI 9pt -- then you have probably inadvertently changed your ClearType settings. ClearType is how Windows draws fonts on-screen, and depending on the screen, different settings will be needed for optimal font display.

To fix this in Win7:

  1. Right-click the desktop. Choose "Personalize."
  2. Click "Display"
  3. Click "Adjust ClearType Text"
  4. A window will pop up. Make sure its checkbox ("Turn on ClearType") is checked. Click "Next."
  5. Follow the prompts through the wizard, adjusting until the fonts look "right" again.
