maggiemoline say:

HOw do I add a new additional email address to my microsoft outlook account ?

brian tillman [outlook mvp 2007-2019] say:

Do you want this address to have a completely separate mailbox so that messages sent to it are not seen by the first address, or do you want messages sent to either address to appear in the same mailbox?

If the latter, log into http://mail.live.com/, log in with the first address, clock the Options gear, then Options, then "Aliases: manage or choose a primary", then "Add an alias" (down the page a bit).  This will allow you to create a second Outlook.com address which will work exactly like the original address, accepting mail into the same mailbox.  You can use a rule to move those messages to a separate folder, if you'd like, or allow them to deliver to the Inbox.

If the former, open an InPrivate/Incognito browser window, visit http://mail.live.com/ and follow the "Sign up now" link to create a new account with its own mailbox.
