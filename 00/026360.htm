How to modify table of figures - Microsoft Community

angeloganaden say:

How to modify table of figures

I want to create a table of figures where the "Figure" word and Figure

number will be the only one who will appear. Like this

jay freedman say:

This is probably more complex than it should be, but here's the only "automated" solution:

  • Anywhere in a document, create the following nested field (you must create the paired curly brackets by pressing Ctrl+F9, not by typing them on the keyboard):

{ TC \f F "Figure { SEQ \c Figure \* ARABIC }" }

  • Select that whole field, press Alt+F3 to open the Create New Building Block dialog box, and enter a name for the AutoText entry. Leave the rest of the items in the dialog unchanged and click OK.
  • Each time you insert a figure caption, insert that AutoText entry in the caption. It can be placed anywhere within the caption.
  • Click at the beginning of the Table of Figures and press Shift+F9 to display its field code. Alter the code to this, and then update the field:

{ TOC \f F \h \z  }

That will cause the TOC field to use the TC fields instead of the entire Caption style paragraphs.

For more information about the syntax and switches of these fields, see Field codes - TOC and Field codes - TC.
