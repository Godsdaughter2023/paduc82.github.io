Cross reference to a page range (such as see page 3-5) - Microsoft Community

a. user say:

Cross reference to a page range (such as see page 3-5)

I need to create cross references to page ranges in a fairly large (200+ page) document.   The reference needs to saysee pages 3-5 for example.  I have created bookmarks across the page ranges and tried cross referencing to the bookmark but am only getting the first page and am now stuck.  I did think maybe I have to set a start bookmark (on page 3) and and end bookmark (on page 5)and insert a cross reference to each with a hyphen in between, but is there a way to just format the pagefef field to show the range of pages for the bookmark?

Thanks.

doug robbins - mvp office apps & services (word) say:

If you do not need the cross references to act as hyperlinks, if you just insert a single cross reference to the Page number of the bookmark that includes the text to which you want to refer, and then you run a macro containing the following code:

Dim bm As Bookmark
Dim i As Long, j As Long
Dim bmrange As Range, fcode As Range
With ActiveDocument
    .PrintPreview
    .ClosePrintPreview
    For Each bm In .Bookmarks
        Set bmrange = bm.Range
        i = bmrange.Information(wdActiveEndPageNumber)
        bmrange.Collapse wdCollapseStart
        j = bmrange.Information(wdActiveEndPageNumber)
        If i > j Then
            .Variables(bm.Name).Value = "see pages " & j & "-" & i
        Else
            .Variables(bm.Name).Value = "see page " & j
        End If
    Next bm
    For i = 1 To .Fields.Count
        If .Fields(i).Type = wdFieldPageRef Then
            Set fcode = .Fields(i).Code
            fcode = Replace(fcode, "PAGEREF", "DOCVARIABLE")
            fcode = Replace(fcode, "\h", "")
        End If
    Next i
    .Range.Fields.Update
End With

It will replace the PAGEREF field with a DOCVARIABLE field that will display "see page m-n" if the range of the bookmark spans more than one page or just "see page m" if it refers to text that is all on a single page.

If the document is edited after the macro is first run, it will need to be re-run to update the page numbers in the DOCVARIABLE field.  You should also run it again if you change the printer as that may also result in the document being repaginated.

-- Hope this helps.

ossiemac say:

If you have used links in Data Validation then the Data validation needs to be deleted first before the links can be broken.

To locate your data validation cells and remove the links:

  • On Home ribbon select "Find & Select"
  • Select Goto Special -> Validation and OK.
  • All Data validation cells should be selected.
  • Check the validation and use Clear all to delete the validation.
  • Break the links
  • Need to Save the workbook and close and re-open before the links disappear.
