Actually, the method that Shauna Kelly describes in her article makes chapter and appendix numbers part of the same multilevel list. You just modify the lower heading levels so that they look the way you want them to in the appendix. If you are saying that you need all nine levels for ordinary headings, then you can't use Shauna's method.

Method 1: To rename Index.dat file

a.    Open Internet Explorer.

c.    On the General tab under Temporary Internet Files, click on Delete Files

d.    Click Ok when it asks if you want to delete the files

e.    Under the History section on the General tab, click Clear History, then click Ok

f.     Close Internet Explorer

g.    Log off the current user and Log onto another user account such as Administrator

h.    Click on Start, Run

i.     Type CMD and Press Enter to open a command prompt

j.     Change directories to the Temporary Internet Files directory by typing the following command, substituting the correct drive letter and the word username with the correct user in Windows XP.

 

cd drive:\Documents and Settings\username\Local Settings\Temporary Internet Files\Content.IE5

 

Example: cd c:\Documents and Settings\Mark\Local Settings\Temporary Internet Files\Content.IE5

 

k.    type del index.dat and press Enter

l.     Type Exit and press Enter to close the command prompt

m.   Restart your computer

 

Once the computer has been restarted, open a web page with a link to a download. Click on the link and try to download the file. Check if it works now.
