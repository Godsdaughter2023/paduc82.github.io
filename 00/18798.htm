violetchen_msft say:

hi ssjrayden,

here are the answers to your questions:

1. it’s not feasible to make the shared mailbox address as the default from address. it needs to be manually selected.

i understand your concern. therefore, i suggest you submit feedback about the feature via http://office365.uservoice.com/forums/273494-outlook-and-calendar many features have been designed or upgraded based on customers’ comments.

2. to make the outgoing (sent mails) from the shared mailbox addresses appear in the sent items folder of the shared mailbox, you can refer to the steps below:

note: since the parameter can only be run by the admin, if you’re not an admin, let the admin run it. if someone who isn’t an admin runs the parameter, he/she will get error messages.

1) connect to exchange online using remote powershell
2) run the command: set-mailbox <mailbox name> -messagecopyforsentasenabled $true
3) run the command: set-mailbox <mailbox name> -messagecopyforsendonbehalfenabled $true
 

after running the commands, the emails sent from shared mailbox addresses will appear in the sent items folder of the shared mailbox. also you will see these sent emails in the sent items of the main office 365 licensed account.

please let me know if anything is unclear.

regards,
violet

How do I create models represented by concentric circles containing - Microsoft Community

researchnovice say:

How do I create models represented by concentric circles containing text?

Hi,

I want to create something similar to the image below in Microsoft Word.  I need to include concentric circles with the text positioned in the same way as it is below.  Can anybody offer any help?

jay freedman say:

This is about as well as I could manage in Word 2010:

Note that this was done in a regular .docx file that is NOT in Compatibility Mode. If you start with a .doc file that is in Compatibility Mode, WordArt behaves differently so the instructions below don't apply (although you can still do all the needed steps).

Each ring of labels is a WordArt object. The WordArt style you start with doesn't matter. I did the outer ring first, but the order doesn't really matter either.

After typing the text -- with between 8 and 12 space characters between segments -- I pressed Ctrl+Shift+F to open the Font dialog, and set the font to black 24 pt Times New Roman with "Small Caps" (all of the letters were typed in lower case so they'll come out the same size).

On the Drawing Tools > Format tab, set the Text Outline to "No Outline", and set the Shape Fill to "No Fill". Then click the Text Effects button, click Transform, and choose the circle format:

At the right end of the Drawing Tools tab, set the horizontal and vertical sizes to the same number.

If necessary, grab the green-dot "handle" and rotate the WordArt to make the spaces between the labels appear in the correct directions. You may also have to edit the text to add or remove some of the spaces to make everything fit properly.

The second ring is done the same way, but set the horizontal and vertical sizes smaller for the middle ring than for the outer ring. The innermost label is left with no transform, but press the Enter key between the words to put them on separate lines.

Drag the three WordArt objects together so they all center at the same place.

Now you can use Insert > Shape tools to draw the circles and the quadrant lines, and to add the arrows. Word won't give you any help in getting the diagonal arrows to line up, so drag them to where they look as close to correct as you can make them. (Hint: With all three kinds of Shape tools, hold the Shift key while drawing or editing them to make the circles be circular instead of oval, and to make the lines stay exactly straight.)

The result will probably look better if you use a real illustration program instead of Word.
