stefan blom say:
Only the built-in headings can be used with captions, but of course you don't have to use Heading 1. Set up numbering for your heading styles and then select the appropriate heading in the Caption Numbering dialog box.

pamcaswell say: Set up your list in the normal way .  By the looks of it, you can apply the built-in scheme (1 Heading 1,  1.1 Heading 2, etc.) with the cursor in your first Heading 2.  It should then be 1.1. Right click the number and left click Set numbering value > Continue from previous > Advance value.  Set the values to 4 and 21.   Word will add, as hidden text, all the numbers needed to adjust the numbering as you specify.  To help stop others from deleting these hidden numbers, I put a note in the first "heading " such as Do not delete until joined with main list. 

Your chapter-numbered caption should now be correctly numbered.

I think that Word, maybe in earlier versions, applied the level 1 (& so Heading 1) to the paragraph where the cursor is.  If that happens, apply the Heading 2 style to that paragraph.   

Also, depending on the what has happened to the numbering in the document (for example, if the numbering has been removed by clicking None in the List Library), not all headings in the document will update to take on the numbering.  Select all instances of each  heading style and reapply it.
