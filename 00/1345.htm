doug robbins - mvp office apps & services (word) say:

The "Different First Page" setting only applies to the header and\or footer in the Section.

While it can be used to modify (increase) the top and bottom margins, simply by means of increasing the space after\before of the paragraphs in the First Page Header\Footer, if you want to increase the left or right margins, that has to be done by inserting a suitably sized text box (or text boxes) into the header that extend down the page and have the text wrapping attribute set so that the text wraps around them.

If you want to repeatedly create documents with this feature, the thing to do is to set it up in a Template and use the template as the basis of the documents that you want to create.

Hi Anne,

 

We'd like to explain that the members of dynamic distribution group are calculated each time a message is sent to the group, based on the filters and conditions that you define. When an email message is sent to a dynamic distribution group, it's delivered to all recipients in the organization that match the criteria defined for that group.

When you create the dynamic distribution group, you are able to create the rules of the membership list, while there is no SharePoint related rules within them. So it is not feasible to create a dynamic distribution group that has all users in SharePoint.

In fact, are there really any personal files stored there at all, such as pictures or financial records that don't exist somewhere else?  If you don't know, it's likely there aren't any, so what is there to stress about?

If you want to insure that no one can get pictures of you in the future and you rarely if ever actually use your laptop camera for anything like online meetings with Skype or some other similar program, then just place a piece of masking or other non-transparent tape over the camera lens (usually at the top middle of the screen lid above the display).

If you ever need the camera in the future, you can just remove the masking tape.  Don't use something like duct tape or other very sticky tape, since it may be difficult to remove and leave a gummy residue on the lens that's hard to clean off.

No matter what, it's not likely you'd ever get any of these items "back" even if someone had taken them, so its never useful to pay a ransom for their return.  In fact, paying them just tells them you are a fool and they'll continue to contact you, trying to extort even more money from an easy mark.

In other words, just ignore them and delete the email.  And stop stressing, there's nothing you can do about the past even if this were true, so keep backups of important files off the PC and cover the camera lens and next time you'll know this doesn't really matter.

In that case I'd use the AfterUpdate event of the combo box to set the subform's Filter property. Base the subform on a query showing all categories; in the combo box's AfterUpdate event put code like

Private Sub cboCategory_AfterUpdate()
On Error GoTo Proc_Error
If IsNull(Me.cboCategory) Then
   Me.subformname.Form.Filter = ""
   Me.subformname.Form.FilterOn = False

Proc_Exit:
