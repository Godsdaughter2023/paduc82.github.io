ken sheridan say: Having multiple Boolean (Yes/No) columns is what's known as 'encoding data as column headings'.  A fundamental principle of the database relational model is the Information Principle (Codd's Rule #1).  This requires that all data be stored as values at column positions in rows in tables, and in no other way.  Colloquially, the encoding of data as column headings is sometimes known as 'committing spreadsheet', hence Duane's comment.  Having separate tables per classroom is similarly an encoding of data other than as values in tables, in this case as table names.

Taking the situations you mention as an example, the attendance of students at classes is a many-to-many relationship type between student registrations for classes and class sessions, so should be modelled by a table which resolves this into two one-to-many relationship types.  The image below is the model for my StudentLog demo file, in which the CourseAttendances table models such a relationship type:



Students' attendances at meetings would be similarly modelled by a table, and to return the sum of each student's attendances at both classes and meetings would be a matter of aggregating each in a query, and adding the two sums per student to give the total attendances.  This can easily be done by joining the two queries on StudentID in a further query.

The reason why there is a red line thing appeared on the left side part of your document is because you have enabled Track Changes option on Office Word. You may also have accidentally pressed Command+Shift+E which starts Track Changes.

In order to turn off Track Changes, please follow the steps below:

1. Click Review > Track Changes.

2. Permanently remove any markup in your document. The only way to do this is to accept or reject changes and delete comments.

3. Click Review > Next > Accept or Reject.

Word keeps or removes the change and then goes to the next one. To accept all the changes at the same time, click the arrow below Accept and click Accept All Changes.

To have a clear view of the given steps, please check this link.

Let us know if this help resolve your issue.
