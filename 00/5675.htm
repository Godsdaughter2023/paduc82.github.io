suzanne s. barnhill say:

Saving a document as the Normal template is always a very bad idea. See "Styles and AutoText are not copied to a new template based on the Normal.dot template in Word 2002." This is an old article, but it is still valid for current versions. See also http://word.mvps.org/FAQs/Customization/CustomizeNormalTemplate.htm.

 

Many default settings in Word can be changed without opening the Normal template, by clicking the Default... button in, for example, the Font and Page Setup dialogs. To make changes that require actually editing the template (to remove extraneous text, for example), or when changes to the template are not being saved because of the interference of a poorly written add-in, it is necessary to open Normal.dot or Normal.dotm for editing. If you know how to find the Normal template to open it, then this is fine, but if you have difficulty locating it, then this method is very reliable:

 

In Word, press Alt+F11; the Visual Basic Editor opens. Press Ctrl+G to activate the Immediate window. In that window, type

     NormalTemplate.OpenAsDocument

and press Enter. Normal opens, but note that it may not be on top of all windows. You can close the Visual Basic Editor. Make the desired changes to the Normal file, and then save and close it. If the changes you have made have not been recognized as changes (the Undo button is not activated), press Spacebar, then Backspace to "dirty" the file, thus ensuring that the template will actually be saved.

Dim strID As String

strID = Me.Acivecontrol.Form.ID

With Me.RecordsetClone

Dim lngID As Long

david marcovitz say: I think I need a little more detail. If you have:

1.
  a.
  b.
  c.
2.
  a.
  b.

and you want 1 abc to appear and then 2 ab to appear as 1 abc goes away, that is easy. Create an entrance effect on the shape with the whole bulleted list. Go tot he Animations tab on the Ribbon and click Animation Pane. Click the drop-down menu next to the animation in the animation pane and choose Effect Options. Under After Animation, choose Hide on next mouse click.

If you want something more complicated, such as each sub-item to appear one at a time and then disappear all at once, I think you will have to set up separate entrance and exit animations for each sub-item is just a matter of getting them all in the right order and setting the right things to With Previous. For example, you can have an entrance effect for 1a and one for 1b and one for 1c. Then, you can have an entrance effect for 2a along with exit effects for 1a, 1b, and 1c that are all set to With Previous. Remember that each bullet point can have its own entrance and exit effects as well as multiple entrance and exit effects for each bullet point.
