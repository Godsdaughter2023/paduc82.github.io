Hi Django,

Windows Update is preconfigured to initiate the maintenance task and wake event to ensure that it is run. You can disable it by following the steps below:

  1. Click Start Menu and type gpedit.msc on the search box and press ENTER.
  2. Go to Computer Configuration > Administrative Templates > Windows Components > Windows Updates and change Enable Windows Update Power Management to automatically wake up the system to install updates to disabled.

Windows 8 wakes up even if "Allow wake timers" is disabled for the current power source
