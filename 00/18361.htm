But how do I apply this now to the remaining tables? I definitely don't want to have to massage each table individually!

Is there a possibility to add another column and re-adjusts all column width in one go?

Since that's probably asking too much: can I at least carry over the column widths from the first table to all the others?

greg maxey say:

You should be able to do it with VBA.

 

Provided the first table in the document is the table you added and resized columns the following should work:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oTbl1 As Word.Table
Dim oTbl As Word.Table
Dim lngIndex As Long, lngCols As Long
Set oTbl1 = ActiveDocument.Tables(1)
For lngIndex = 2 To ActiveDocument.Tables.Count
  Set oTbl = ActiveDocument.Tables(lngIndex)
  oTbl.Columns.Add
  For lngCols = 1 To oTbl1.Columns.Count
    oTbl.Columns(lngCols).Width = oTbl1.Columns(lngCols).Width
  Next lngCols
Next
End Sub

pa bear - ms mvp say:

I don't have Automatic [Updates] for Win8 turned on.  I like to give patches to windows a few days before installing.

If you want Defender to auto-update itself, Automatic Updates (AU) must be enabled (turned on) and set to the default Automatic configuration.

If you want to "give patches...a few days before installing," you're better off having AU enabled and set to the Check for updates but let me choose... option - but you'd still need to update Defender manually.

This computer came with Windows 7.  I ran Verizon Internet Security Suite and their package included McAfee.

Answer-by-number:

1. How long had the Verizon/McAfee suite been installed?

2. When (approx. date) did you upgrade the Win7 computer to Win8?

3. Has Win8.1 ever been offered to the computer?

4. What Definitions versions are currently installed according to Defender's Update tab?

5. Did you (a) uninstall the Verizon/McAfee application AND THEN (b) download/run the McAfee Consumer Products Removal Tool & reboot (c) BEFORE you upgraded to Win8?

  • Fact: McAfee (and Norton) applications are notorious for not uninstalling or upgrading cleanly. The "leftovers" may be your troublemaker here.
