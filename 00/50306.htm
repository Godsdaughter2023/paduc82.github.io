tryx3 say:

1  This sounds like it was hibernating not sleeping judging by your description.

2  I disagree that it can be anything to do with the BIOS.  Hibernation is controlled by Windows.  Did these "techs" say anything more definite?

3  The first step should be to reinstall the display driver and there is no need to wait until you have studied all the other possible causes as display driver problems account for most sleep & hibernation problems.  I have added instructions for getting the display driver from Dell in case you have never done this before.

3.01  Check the purchase documentation or the little white sticker on your computer to find your Dell Service Tag.  This is nothing to do with any warranty or lack of one and using it will cost you nothing - it is a unique identifier of your system so Dell's

3.02  Go to the Dell website, select the support section, put in your Dell Service Tag, it will offer several sections relating to your computer model including Drivers & downloads - choose that.

3.03  The list of Drivers & downloads that you now see is specific to your computer on the day it was shipped out by Dell [not just your computer model, your computer].

3.04  Look through the available drivers to find the display driver.

3.05  If there is any confusion, look in your computer - Device manager, Display will identify your type of display card and that will match the display driver you see listed.  Actually, if you made a mistake & picked the wrong driver your computer would pick it up and tell you there is no hardware to match the driver you are trying to install.

3.06  Pick the latest available driver - there's nothing magical about later ones but you might as well get the latest as that will have had more bugs removed than any other version.

3.07  Select that driver's title - the titles are always a link to an information page on the driver concerned.

3.08  Print or save the information page so you can refer to its version number later on and also so that you can check its installation instructions.

3.09  Download the driver.

3.10  Install the driver.  Check the installation instructions first but the procedure is almost always to right-click on the driver file and select Run as Admin [because the driver will be an exe file - if it is a zip file then right click on it and extract its contents, which will be an exe file]

3.11  Reboot afterwards [it might tell you to do so but it is handy anyway so you know that you are starting from a clean state].

3.12  Play around with some applications and check that it all seems to be working.

3.13  Look in Control panel, [View by Small icons, if necessary], Display to see if it has automatically set itself to the display resolution you wanted [it normally does unless you have been running it in some deliberately non-standard way]

3.14  Try sleeping to check if the problem has been fixed.

3.15  Try hibernating to check if the problem has been fixed.

4  If the problem remains then look up sleep problems and hibernation problems in Windows help to see the other things you can usefully check. 

All the best,

Denis
