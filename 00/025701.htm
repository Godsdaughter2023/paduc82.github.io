joel ivory johnson say:

It is possible to change the low memory threshold on a Windows Mobile device through registry settings. But this is generally discourage since it can lead to undesirable and unpredictable behaviour from the device.

Also it can't be predicted how much memory a program will need to run. For example, let's say I have a small image editing program and the system sees there is enough memory to load the program into memory. So it does so and the program starts running. Let's also say this program automatically opens the most recently edited picture. That is going to require memory allocation in addition to what was used to just load the program and trigger low memory actions (which may include closing other background programs).
