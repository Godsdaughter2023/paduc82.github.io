lemp say:

When Windows is installed, it configures itself to run with the hardware of the computer on which it is installed.  Unless you used an identical motherboard/CPU, Windows will be confused.  It's possible, but not certain, that you can do a repair install that should keep the programs and data on the drive intact (but you should backup the data first, just in case).  See https://www.sevenforums.com/tutorials/3413-repair-install.html

You will need Windows 7 installation media (see the directions).  If you have a retail product key, you can download an ISO file here:  https://www.microsoft.com/en-us/software-download/windows7

If you have any other sort of Windows 7 product key and need to download an ISO file in order to create Windows 7 installation media, AFAIK the only site that is currently working for Windows 7 is http://windowsiso.net/  You will need a torrent client to download the ISO file from this site.

Note, however, that if you have an OEM product key -- i.e., your installation of Windows 7 was pre-installed on the computer by the computer manufacturer -- activation of your Windows installation will likely fail. 

Microsoft considers that the installation a new motherboard -- unless it is a duplicate of the original or identified by the manufacturer as the replacement for the original -- creates a new computer.  OEM Windows licenses are valid only on the computer on which they were first installed. 

If you attempt activation by phone and speak to a human (don't respond to the automated voice prompts), you may be able persuade the person to authorize activation of your system if you can convince them that you're repairing the existing computer with as close to the original motherboard/CPU as possible.
