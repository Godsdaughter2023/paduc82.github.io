Is it an OEM version of Windows? If it is, and this is not a warranty, exact MB replacement, I believe you'll have to get a new key.

If it's a retail version (full install or upgrade) of Windows, I believe you can install a new MB and hardware without purchasing a new license, but you'll more than likely have to re-activate it because of hardware changes.

Before going through this upgrade (albeit forced due to failure), make an image of your current disk just to be safe. That way if things go horribly wrong, you'll have a safety net to fail back on to. Also, make sure you have all the latest MB drivers beforehand; you'll certainly need them on the first boot.

If the new MB has the same chipset and CPU family/manufacturer as the old one, you shouldn't have too many problems. It's when you change from, say, an AMD CPU to an Intel, or completely different chipsets, that the real fun begins ;-)
