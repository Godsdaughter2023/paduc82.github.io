bossdweeb say:

Hi Ken, Any joystick, controller, or wheel that uses DInput and has a profiling software that is compatible with Win10 can in theory

be used with the game.

The profiling software is just another keyboard emulator (not unlike the 3rd party options, but specific to the brand you are using)

that allows you to assign the keystrokes  used by the game to  whichever button or switch you like on your controller.

- If the buttons on the joystick are registering the desired commands and the stick is not functioning try mapping the

joystick axis to keyboard commands. CFS2 uses number pad keys - 8 for pitch up and 2 for down (or vice-versa). Probably 4&6 for roll .

- I have thought of having a go using my Logitech G27 wheel with flight sims, but never got around to it because I don't run the

flight sims much and my game controllers sticks do the job well enough for casual sessions.

The main issue with the wheel is you cannot push the wheel forward\back for pitch.

My wheel has two 'paddles' on the column (alternatives for gear shift, but can be programmed to whichever function the user wants).

For a flight sim I would probably try to use these for pitch, and the wheel for roll, but while the wheel is analogue the paddles are not,

so they may not be real good for pitch.

The accelerator and break foot controls are analogue and could be used for rudder.

This wheel also has six buttons in the centre, and more buttons on the gear shift console for other functions.
