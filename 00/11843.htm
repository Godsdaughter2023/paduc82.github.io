You don't need to create a group to send everyone a message.

You can click the To button and select all (select first, hold shift and scroll down, select the last entry) - click the BCC button to add all to the BCC field.

You can go into contacts, select all then click the Email button - this will add all to the To field.

One way to send to all of your contacts is to open the Contacts folder, select all the entries using Ctrl+a, then drag the selection to the Inbox. This will open a new message window with all of your contacts in the To field. If you wish to hide the addresses from the recipients, select all of the To field contents (again using Ctrl+a) and drag the selection to the Bcc field.
