SD card installation - Fixes - Microsoft Community

jay_answers say:

SD card installation - Fixes

Technical Level : Basic

Summary

Those who are not on high speed Wi-Fi would prefer to install apps through SD card. Most of the time, this happens as planned. Sometimes it doesn't. Some common fixes are listed.

Note : This article is in addition to information that is available at http://www.windowsphone.com/en-US/how-to/wp8/apps/how-do-i-install-apps-from-an-sd-card . Check that first, then come here if you are still facing issues.


Details

SD card installation issues can come down to following problem points.

  • General stuff
  • Free space
  • Data connectivity
  • Version of XAP file
  • Phone reset

General stuff - app being detected on the SD card, on the phone

  • Ensure that the XAP file is placed in the ROOT directory of the SD card. The root directory is the folder that you see, the moment you open the SD card (on the phone) in your PC.
  • After the XAP file is in the SD card, do a phone restart. This 'forces' the phone to check the SD card for apps.
  • Also, there are occasions when the app won't be detected for a while. Like ten minutes to thirty minutes.
  • Free space

    The way installation works on windowsphone is a little different than other phones. Usually you need to have four times the app size to do installation. That means, if you have an app that is 100 MB, you need to have 400 MB free space on the phone. If you have an app that is 1 GB, for an error free installation, you will need to have 4 GB. Of course, this is not always the case. Most of the time, I am able to install apps with only twice the amount of free space. Still, 4 times the app size is preferred way of installation.

    Data connectivity

    Even if the app is being installed through SD card, the phone still needs to verify the installation. This means, you cannot install apps if your phone is offline. Ensure that the phone is connected to internet  when you try to install apps from SD card.

  • Version of XAP file

    The frequency of updates that a developer pushes to his app can be high (like multiple versions are pushed to the store within a day) to low (no updates even after months). Let's say you downloaded the XAP file two days ago and decide to install it today. Suppose a new version was pushed to the store by the app developer today morning. If so, your XAP is now 'outdated' and will NOT install. You will be required to re-download the latest version of the XAP file and re-try the installation.

  • Phone Reset

  • If all else fails, then you are left with just one option. Something about the OS is preventing you from installing the app. Reset the phone by going to SETTINGS > ABOUT > RESET. Please ensure that you have backed up all you need before you do this.

    talking moose say:
    This is the behavior of Outlook. When you flag a message you're not duplicating it. It's simply appearing in your Tasks list too.

    Flagged messages are flagged because you want to do something with it later. Maybe you want to review it. This becomes a task to complete.

    You can click the flag icon in a message to mark it as complete. The red flag will change to a green checkmark. You can also right-click or Control-click the flag to clear it completely. This will also remove it from your Tasks list.
