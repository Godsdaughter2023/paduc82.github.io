danm---01 say:

The press and hold move was a key in finally figuring this out.  That instruction, by the way, is not listed in the owner guide.  Even then, you have to follow a very long and specific sequence, while ignoring common logic and programming assumptions, to get it to work.  Here's what worked:

 

  • Open player.
  • In top-of-screen menu select one of the choices - All Songs, All Albums, Artist, Genre, or Playlist.
  • You now have to start playing a song to get to the Now Playing screen - only then can you save or create a playlist.
  • In top-of-screen menu select "Save Playlist".  That will save or create that one song in a playlist.
  • To add more songs go back two screens to Songs, Albums, etc.
  • Press and hold a song to add it to the current playlist.  Repeat for more songs.
  • Go back to the Now Playing screen again, and do the "save playlist" process there again.
  • It will ask you to enter a playlist name.  It will not bring up the lists you've already created, so it looks like you're creating a new one - you're not.  Dumb...
  • Enter the exact spelling of the current playlist and save.  It will ask if you want to replace current playlist with this one.  Again dumb.  It's the same playlist, just being updated.  Ignore logic and say yes.
  • Now you have created or updated a playlist.

 

So not only are the guide instructions confusing (and lacking), but so is the actual process.  I fully expected startup issues with this new device, and I'm looking forward to updates for this program, and to new player programs they're currently working on.  In the meantime, I still love the N900!

danm---01 say:

For 5arms:

 

Renaming playlist - the only way I can find is to first load a current playlist in the player, then use "save playlist" in the Now Playing screen's top menu, and put in a different name.  Now you'll have that same playlist under both the original name and the new one.  Delete the old/original by pressing and holding it's title back at the Playlists screen.

 

Deleting tracks - can't find a way to do individual track deletions.  

 

Re-order tracks - no way to do in the Media Player that I can find.  There is a limited "sort" function in File Manager.  However, any sub-folders found or created within the main "Audio Clips" folder in this File Manager do not appear within the Media Player itself (so not sure what purpose they serve, actually).   Maybe sorting an album's track in your main computer before transferring them to the N900 would work.

 

I did find a quicker way to create a playlist.  Open Media Player, click on the little "black bars" icon on the lower right to get to the Now Playing screen and do "clear current playlist" in the top menu.  Now go back and open "all songs", press and hold a track and select "add to playlist", then repeat this on the other songs you want.  Next, go to the Now Playing screen (via that black bars icon on lower right) and do "save playlist" in the top menu.  Done.  This is much quicker, but the two keys to success are, there can't be anything loaded in the Now Playing screen when you start, and if you accidentally click open a song while trying to press and hold it, you'll have to start over, as that mistake will load ALL songs into the Now Playing screen. 

 

Over at Maemo.org there have been comments about some better, more complete, media players being under development/testing there.  I don't remember their working names.  I'll just patiently wait til they're done and hope they solve and un-complicate the current player's issues and operation.
