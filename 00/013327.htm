Keyboard short cuts office 2013 - Microsoft Community

robertensell say:

Keyboard short cuts office 2013

I am working with WORD 2013 marking university scripts. I find the keyboard shortcut for INSERT COMMENT  Ctrl+Alt+M very cumbersome. Can I change it to one that is more suitable for me?
jay freedman say:

Certainly -- you can change it, or you can leave Ctrl+Alt+M assigned to it and add another shortcut to the same command.

The "official" way to assign shortcuts is to go to File > Options > Customize Ribbon and click the Keyboard Shortcuts Customize button in the bottom left corner. In the Customize Keyboard dialog, select the category (or All Commands) that contains the desired command, and then select the command name in the right-hand list. Click in the "Press new shortcut key" box, press the desired key combination, and click the Assign button; then Close.

The name of the command you want is InsertAnnotation, and you'll find it in the All Commands category (for some reason it doesn't appear in the Insert category).

Because the command already has an assigned shortcut, you can get to the right place in the dialog much more quickly: Starting in any document, press Ctrl+Alt+number pad Plus. The cursor changes to a cloverleaf design, indicating that you're in a special mode where the next keypress or ribbon button click will open the Customize Keyboard dialog to the corresponding command (if any). Then press the Ctrl+Alt+M shortcut. Then continue with the "Press new shortcut key" box.

If you want to get rid of the Ctrl+Alt+M shortcut, select it in the Current Keys box of the dialog and click the Remove button.
