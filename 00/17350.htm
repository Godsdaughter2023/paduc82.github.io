OK... we're going to need a bit more information here, so we can help you out.  What exactly are you trying to download? From what sites?

Just some background on Windows Phone:

a) apps -- these must be downloaded/purchased from the store.  There are ways to install from an SD card, but you need specific .xap files (which are usually acquired from the Store).

b) songs, pictures, etc. -- these should also be acquired from a recognized music service/app, rather than just picked up from a download library.  HOWEVER, if you download to your PC first, then you can use the windowsphone app (from Microsoft) to transfer them to the phone.

The main reason you can't directly download and access from the internet is that, for security reasons, applications are 'sandboxed' into their own working environments, and denied access to system functions and other apps working areas.  They can only store information in their own data area -- they can't see or touch others.  There are some specific exceptions (Microsoft office for example) which MS has allowed to see specific things on the phone, but these also are not available to 3rd party developers.

Try this to reapply autofilters across all sheets:

Sub ReapplyFilters()
  Dim wsh As Worksheet
  Dim flt As AutoFilter
  For Each wsh In Worksheets
    Set flt = wsh.AutoFilter
    If Not flt Is Nothing Then
      flt.ApplyFilter
    End If
  Next wsh
End Sub

stefan blom say:

As an alternative, use this macro to remove the shading from all tables in the active document:

   

Sub RemShadingFromTables()

Dim t As Table

For Each t In ActiveDocument.Tables
With t.Shading

 .Texture = wdTextureNone
 .ForegroundPatternColor = wdColorAutomatic
 .BackgroundPatternColor = wdColorAutomatic

End With

Next t

End Sub
