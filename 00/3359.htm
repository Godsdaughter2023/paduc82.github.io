Build Graph from Every Other Column - Microsoft Community

david r. krumholz say:

Build Graph from Every Other Column

I want to graph a single value across time. My dates are in row 1 but they are in every other column so I have dates in E1, G1, I1, K1, etc. My values are lower down in the same columns so they are in E5, G5, I5, K5, etc. What is the best way to graph those values?

barb reinhardt say:

If I were you, I'd create a worksheet level named range ChtLabel with the following definition (adjust to suit)

 

=Sheet1!$E$1,Sheet1!$G$1,Sheet1!$I$1,Sheet1!$K$1,Sheet1!$M$1,Sheet1!$O$1

 

Then create another named range (let's call it YValues) with a similar definition for the row 5 stuff.

 

I've used dynamic named ranges in charts a lot.   While these aren't dynamic, the attached file will give you an idea of how to use them .  If you need some code to create the named ranges, it's pretty easy to do, so let me know how many columns you want to use and I can include it. 

 

http://office.microsoft.com/en-us/excel-help/using-named-ranges-to-create-dynamic-charts-in-excel-HA001109801.aspx

barb reinhardt say:

Here is some code to create the named ranges (to column 25)

 

Option Explicit

Sub Test()
Dim myRange As Excel.Range
Dim aWS As Excel.Worksheet
Dim myCell As Excel.Range
Dim i As Long

Set aWS = ActiveSheet
Set myRange = Nothing

For i = 5 To 25 Step 2  'Change to suit
    Set myCell = aWS.Cells(1, i)
   
    If myRange Is Nothing Then
        Set myRange = myCell
    Else
        Set myRange = Union(myRange, myCell)
    End If
       
Next i

aWS.Names.Add Name:="ChtLabel", RefersToR1C1:="=" & myRange.AddressLocal(ReferenceStyle:=xlR1C1)

'Set value for YRange
Set myRange = myRange.Offset(4, 0)
Debug.Print myRange.Address

aWS.Names.Add Name:="YValues", RefersToR1C1:="=" & myRange.AddressLocal(ReferenceStyle:=xlR1C1)

 

End Sub

a. user say:

I suspect you found the answer to this, but just in case you are still trying, I suspect the problem is that you are trying to use Word's Edit->Paste function to paste the link into the "Link to:" box in the Insert Hyperlink dialog box.

If so,
 a. that won't work, because the Edit->Paste option is there to paste things into the Word document itself. It's not there to paste things into dialog boxes (although I can see that perhaps it should let you do that too!)
 b. You can use cmd-V to paste the Hyperlink that you copied into the dialog box.
 c. When you have done that, the "OK" button should appear in the Dialog Box. Click that, and the link should be pasted into the Document at the place you have selected.
