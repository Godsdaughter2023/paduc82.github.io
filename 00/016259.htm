If the data type of the value being returned is numeric than you can use a conditional sum formula if the combination of the 2 lookup values is unique.

Here's a method that works on any data type.

Let's assume this is your data in the range A2:C10

A X result1
B Y result2
C Z result3
A A result4
B V result5
C B result6
A N result7
B D result8
C F result9

We want to lookup "A" in column A and "A" in column B and return the corresponding result from column C.

Use cells to hold the lookup criteria:

  • E2 = A
  • F2 = A

Enter this array formua** in G2:

=INDEX(C2:C10,MATCH(1,IF(A2:A10=E2,IF(B2:B10=F2,1)),0))

Returns: result4

** array formulas need to be entered using the key combination of CTRL,SHIFT,ENTER (not just ENTER). Hold down both the CTRL key and the SHIFT key then hit ENTER.

To add VBA code to the project you need to do 2 things first.

In the VBA editor Select menu item Tools  and then scroll down to Microsoft Visual Basic for Applications Extensibility 5.3 and check the box. (Ensure you check the box; not just select and highlight the line)

Next in the Excel worksheet window, go to Excel Options -> Trust center -> Trust center settings button -> Macro settings and then check the box against Trust access to the VBA project object model.

You then need to learn the required coding and syntax for creating the code.

However, you compromise the security of your computer when you do the above so I suggest the following method.

Create a template blank worksheet with the required VBA code.

Hide the worksheet.

Then use the following code to copy the worksheet and the VBA code module gets copied with the worksheet. Remember that Me is the generic object name within the worksheet code module for the worksheet to which the code belongs so you can reference the new worksheet.

The code below for creating the new worksheet from the template.

Sub Macro1()
    Dim wsAdded As Worksheet
    Dim lngShtsCount As Long
    lngShtsCount = Worksheets.Count
   
    lngShtsCount = Sheets.Count
   
    Sheets("Template").Copy After:=Sheets(lngShtsCount)
   
    Set wsAdded = Worksheets(lngShtsCount + 1)
   
    wsAdded.Visible = xlSheetVisible
End Sub

daniel g. grau say:
It will not be easy. You can move the objects (not the connectors) in a limited way by selecting the shape and pressing the arrow keys. Pressing Control + Arrow will give you a finer position adjustment, particularly if you zoom in.

After moving the shapes, you will likely find that connectors don't align properly, but they cannot be moved. You'll have to actually hide the misplaced connectors and add new ones. To hide them, do Control-click (or Right-click) on the connector, choose Format Shape > Line and change the line color to "No fill."

When adding new connectors, it will help if you set the Ribbon's Format > Grid align option to "To other objects."

Make sure any new objects you place over the chart allow overlapping. For this, Control-click (or Right-click) on the new object, choose Format object > Layout > In front of text, then click on the Advanced button and place a checkmark on Allow overlap.
