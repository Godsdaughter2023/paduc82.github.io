taurarian say:

Create a shortcut on the desktop to the Excel program you're using and then add the file you wish to open in the target line.

To make it easier, drag and create a shortcut on your desktop by locating the exel.exe file on your system.

Target "C:\Program Files\Microsoft Office\Office14\EXCEL.EXE" "D:\test.xlsx"

Start in: "C:\Program Files\Microsoft Office\Office14"
