keratl say:

You can grab code relate to grouping rows from other past questions; here is a code snippet I had used (in my case, I needed to hide rows that were indented or blank). This shows you how to use the indentlevel property in Excel VBA.

Sub Hide_Indented_Rows()

Dim sh As Worksheet
Set sh = ActiveSheet

For i = 5 To 900 
    Application.StatusBar = "Reviewing row " & i & " of 900"  
    If sh.Range("A" & i).IndentLevel >= 3 Or sh.Range("A" & i).Value = "" Then
    sh.Rows(i).Hidden = True
    End If
    DoEvents
Next i

Application.StatusBar = False
End Sub

dr. strangelove say:

You can have only one realtime antimalware product installed.  Having more than one will cause performance problems and system instability.

This means that you have to uninstall/remove your existing product(s) (ie norton, mcafee, tm, etc) before installing MSE.  It also means you cannot use any realtime features of products (like tea timer from spybot).
