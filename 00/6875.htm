After confirmed with the related resources. Due to the difference in the display resolution and the graphic drives, it's an expected behavior that the column width will change when you open the workbook on another desktop.

 

As a workaround, you could use the VBA code to resize the column A's width while opening the workbook.

--------------------------------------------------------------

Private Sub Workbook_Open()

    Dim wSheet As Worksheet

 

For Each wSheet In Worksheets

        ' Change to the columns you need

        Columns("A:A").Select

        ' Change to the width you need

        Selection.ColumnWidth = 20

 

    Next wSheet

End Sub
