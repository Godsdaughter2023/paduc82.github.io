This is because the "dates" are in fact text values. To convert to dates:

  1. Select the dates
  2. Find, replace, find what "*, " (do not use the quotes, just to show the space. Replace by nothing
  3. Format the range as ddd, dd/MM/YYYY
  4. You will now be able to sort.

"Must be enabled" means "don't disable."  It does not mean "must be running at all times."  The service is started (event-triggered) and stopped by the software downloaded to your device when you join the device to the WIP.  The service would not be contributing to the download issues because it would already be running.  Windows Update would not be offering a build in the first place if it were off.  

Having said that, it is a worthwhile question to explore.

graham mayor say:
The ExcludeUS dictionary needs to be ExcludeUK. You are working in UK English not US English. It means exclude words from the UK dictionary, not exclude American words.

As for the language - the default language is dependent on the Windows regional input language settings in Windows. However every character in a document may have its own language parameter, so just changing the default will not affect the language of imported text with a language parameter already applied. It will keep that language unless you apply a different language.

New documents formatted with styles that have the UK language parameter will be formatted in UK English.

Custom dictionaries may be language dependent or apply to all languages. You can only save words in the default custom dictionary.
