Christi

bobby mikkelson say:

Hello Christi,

 

When changing printer ports, if the printer is in the state "Use Printer Offline", attempting to change the port of the device will successfully change the port but the printer will continue to be in the offline state.  This may be the case when attempting to switch the port from the USB port directly to the printer's TCP/IP address.

 

You can work around this by creating a new print queue for that printer using the desired port, and the new queue will function as expected, then just remove the stuck queue.
