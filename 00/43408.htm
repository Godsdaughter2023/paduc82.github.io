greg carmack - mvp 2010-2020 say:

You can Clean Install Windows 7 while wiping all partitions down to Unallocated Space to get it cleanest.  But you'll need a Windows 7 license to insert Product Key after install once you test it works fine.  I'd also save a Windows 10 Image Backup  in case you decide to go back at any time in the future, or WIndows 7 doesn't work like you want.

But there's no need to uninstall Windows 10 as you can simply shrink its C partition in Disk Management, then boot the Windows 7 installation media to  Clean Install Windows 7 to the space you made.  It will configure a Dual Boot during install, of if not install EasyBCD (click Download - no Name or Email required)  to add the other OS on the Add OS Entry tab. 

This way you can share files between OS's in the File Explorer, go back and forth to optmiize your Windows 10 to greatly improve the inferior factory install following the Windows 10 Install Integrity Checklist.

You will need a license for the Windows 7 partition, and cannot move the old Windows 7 if it came installed on it's old PC.

ellerem say:

Hello,

You are doing the right steps to investigate this issue.  Changing & checking cables and trying a different TV are the lead fixers.

Next try resetting your console display settings, follow these steps:

1. Make sure that there’s no disc in the disc tray.

2. Turn off the console.

3. Turn on the console.

NOTE  If you’re using an Xbox 360 Wireless Controller, make sure only one wireless controller is connected to the console and then turn off and restart the console by using the Guide button on that controller.

4. As the console starts, press and hold Y while pulling the right trigger on the controller. The display settings are reset to the default settings, and then the console automatically restarts. If you only have a wireless controller that is not working - try plugging it in with the recharge cable to make it a temporary wired controller.

NOTE for Xbox 360 E or Xbox 360 S console:

To reset display setting on these console models, you can also just turn off the console and then hold the power button on the front of the console down for about 10 seconds. (The lights will flash several times.)

Good luck & happy gaming

Removing the hard drive is a good step to see if it works without it.  Some pics are here : support.xbox.com/.../add-remove-hard-drive

Alternatively I'd like to ask if you have anything plugged into the USB drives on the front or back of the console?  Items such as USB sticks and external hard drives.

3. Turn console on and see if it is working without freezing.

4. You are welcome to attach each one again and test so as to isolate which attachment is the problem.

USB items can prevent the screen showing anything or freeze the console especially if they are starting to fail or have been corrupted in some way.

Hello,

Previously, you will not be able to download anything from the Microsoft Store without using a Microsoft account. This has changed since the Anniversary Update. You can now use a local account and download any free apps in the Microsoft Store. However, there are still some apps that require a Microsoft account before you can purchase or download them. There are also restrictions when you're not using a Microsoft account such as losing out a number of features to maintain a list of previously owned apps across devices.

You can create a local user account or you can use your existing account then access Windows Store. If you're getting any issues, make sure you have the latest Windows update. To check for updates, go to this link.
