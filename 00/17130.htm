patc---01 say:

Missing Caller ID on Calls/SMS

Technical Level : Intermediate

Summary

Details
Sometimes, caller ID (the name of the caller) does not appear on a call and is replaced with the phone number.

The main reasons for caller ID not working are:

1. Duplicate entries in contacts - by FAR the most common cause and unfortunately, the most awkward one to fix.

If the phone finds the same number against more than one name, it doesn't know which to use so, perfectly sensibly, it defaults to showing the number.

The only way to fix this is to remove duplicates from your contacts. Best way to do this would probably be in Outlook on the PC, but assuming you can't do that, then you have to find the numbers that aren't working a use the LATEST version of the Nokia Search tool to find the duplicate entries:

NOTE: when you search, use PART of the number to avoid problems with formatting: searching for +441234567890 will NOT find +44 123 456 7890 whereas searching for 789 would.

2. Using your international dial code instead of '+'

Not sure if this applies to all phones, but if you store your number using your county's international dial code (eg: 004412345678990 - WRONG) instead of the '+' sign (+4412345678990 - RIGHT), it can cause problems.

3. "(0)" area code prefixes

Only a problem if you synch with MS Outlook. When you specify a country in MS Outlook, it will insert "(0)" into your phone number after the country code - while somewhere in Redmond, Bill Gates cackles - so +4412345678990 becomes +44 (0) 12345678990.

You need to remove the (0).

4. 9 and 10 digit numbers

There has been one report of problems with 9 and 10 digit numbers where one size doesn't work with SMS and the other with calls. Not sure that anyone else has ever come across this. Comment 15 people found this helpful

hansv mvp say:

Here is a version that loops through all worksheets of the active workbook:

Sub ApplesAndBananas()
    Const strMergeCol = "C" ' column with merged cells
    Const lngFirstRow = 2 ' first data row
    Dim lngLastRow As Long
    Dim lngStep As Long
    Dim lngRow As Long
    Dim strValue As String
    Dim shp As Shape
    Dim wsh As Worksheet
    Application.ScreenUpdating = False
    For Each wsh In Worksheets
        For Each shp In wsh.Shapes
            shp.LockAspectRatio = True
            shp.ScaleHeight Factor:=0.5, RelativeToOriginalSize:=False
            shp.Placement = xlMove
        Next shp
        lngLastRow = wsh.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
        lngRow = lngFirstRow
        Do
            lngStep = wsh.Range(strMergeCol & lngRow).MergeArea.Rows.Count
            strValue = LCase(wsh.Range(strMergeCol & lngRow).Value)
            wsh.Range(strMergeCol & lngRow).MergeArea.EntireRow.Hidden = _
                Not strValue Like "*apples*" And Not strValue Like "*bananas*"
            lngRow = lngRow + lngStep
        Loop Until lngRow > lngLastRow
        wsh.Cells.EntireColumn.Hidden = True
        wsh.Range("C1:D1").EntireColumn.Hidden = False
        Application.Goto wsh.Range("C1"), True
        ActiveWindow.Zoom = 100
    Next wsh
    Application.ScreenUpdating = True
End Sub

Replace all special characters with space - Microsoft Community

jw aldridge say:

Replace all special characters with space

I have a string of data... A lot of the data is alphanumeric and many have special characters (!@#$%^&*()_+=-{}[]:;"'<>?,./~` ) 

Example: 6-ct of Apple-skins (#green)

I need to replace all special characters with spaces. 

6 ct of Apple skins  green

Prefer vba (16k+ rows of data) but formula would work as well. 

hansv mvp say:

You can run the following macro (select the range first):

Sub ReplaceSpecial()
    Dim cel As Range
    Dim strVal As String
    Dim i As Long
    Application.ScreenUpdating = False
    For Each cel In Selection
        strVal = cel.Value
        For i = 1 To Len(strVal)
            Select Case Asc(Mid(strVal, i, 1))
                Case 32, 48 To 57, 65 To 90, 97 To 122
                    ' Leave ordinary characters alone
                Case Else
                    Mid(strVal, i, 1) = " "
            End Select
        Next i
        cel.Value = strVal
    Next cel
    Application.ScreenUpdating = True
End Sub
