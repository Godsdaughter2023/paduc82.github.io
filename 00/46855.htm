yolvi ell say:

Hi Josh,

The reason why there are two Recycle Bin icons in the Desktop Icon Settings is for users to have a preview of what the Recycle Bin icon will look like if there are items inside the Recycle Bin.

Taking this into account, it means that there's no other Recycle Bin in a different or hidden file location on the computer. There's only one Recycle Bin, and it's the one that's being shown on the desktop. If the Recycle Bin has files in it, that's the only time that it'll use the Recycle Bin (full) icon.
