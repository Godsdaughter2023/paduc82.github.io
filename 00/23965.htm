How to create macro for custom error bars in office 2010 - Microsoft Community

drcole say:

How to create macro for custom error bars in office 2010

I need to create a macro that adds custom error bars to columns. I can't seem to find any reliable models on the web. For all other tasks I can record a macro and they run but the custom errors bars do not run.

hansv mvp say:

You can use this as starting point:

 

Sub CustomErrorBars()
    Dim cht As Chart
    Dim ser As Series
    Set cht = ActiveChart
    Set ser = cht.SeriesCollection(1)
    ser.ErrorBar _
        Direction:=xlY, _
        Include:=xlErrorBarIncludeBoth, _
        Type:=xlErrorBarTypeCustom, _
        Amount:=Range("B2:B11"), _
        MinusValues:=Range("C2:C11")
End Sub

 

This macro assumes that the chart has already been selected and that you want to add error bars to the first chart series.

It adds both positive and negative error bars; the values for the first are taken from B2:B11 on the active sheet, and those for the latter from C2:C11.

Modify as needed.

lidiaal say:

How to use this code with relative referencies?

Is it possible to modify the macro in a way that it takes the range for the error bar where we point it?

hansv mvp say:

Relative references?

 

You can let the user select the ranges like this:

 

Sub CustomErrorBars()
    Dim cht As Chart
    Dim ser As Series
    Dim rngPos As Range
    Dim rngNeg As Range
    On Error GoTo ErrHandler
    Set rngPos = Application.InputBox(Prompt:="Select range for positive error bars", _
        Type:=8)
    Set rngNeg = Application.InputBox(Prompt:="Select range for negative error bars", _
        Type:=8)
    Set cht = ActiveChart
    Set ser = cht.SeriesCollection(1)
    ser.ErrorBar _
        Direction:=xlY, _
        Include:=xlErrorBarIncludeBoth, _
        Type:=xlErrorBarTypeCustom, _
        Amount:=rngPos, _
        MinusValues:=rngNeg
    Exit Sub
ErrHandler:
    MsgBox "An error occurred!", vbExclamation
End Sub
