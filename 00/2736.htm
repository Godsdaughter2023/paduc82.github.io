    In fact one of the most useful applications of the Solver is to solve for values of 1.0e-4 or 1.0e-5.  Please consider this:

    For a large number of equations where

        y(i) = f(x(j))

    and

        y(i) represents the results of i equations

        x(j) represents j variables

    Generally, j>>i so if we set almost any set of j-i variables as knowns, the system of equations will solve for the remaining set of unknown x(i) variables.

    It is a trivial matter in a spreadsheet to set up an array or list of equations for unknowns, with a set of knowns (or targets) so that all the differences (d(i)) are calculated for the i equations as

       d(i) = f(x(i)) - y(i)

    So the SQRT(SUMSQ(d(i)), becomes the Objective Cell.  Now the Solver can solve to a tolerance (Value of) say 1.0e-4.  Once achieved, the values of all the previously unknown x(i) variables are the solution to the problem.  This value is never zero initially.

    Note that a tolerance here is a better device than just "min" or trying to achieve zero for the Objective Cell as it dramatically reduces solution time trying to achieve levels of significant figures that are very largely unnecessary or that may result in running into stability issues at the limits of Excel accuracy.
