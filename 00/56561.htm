jeeped say:
Many ISPs (and unwitting users) will load a browser with all sorts of useless (to you) add-ons, toolbars and other assorted crapola that only slows down your online browsing experience. Determining proxy status can also stall things for a few seconds.

First, open IE and tap the Alt key to get access to Tools, Internet Options. When the Internet Options dialog opens, go to the Connections tab and click LAN Settings in the lower right. Turn  off everything in this Local Area Network (LAN) Settings dialog. Your North American cable connection doesn't need a proxy server and shouldn't even look for one. Click OK all the way out and close the browser. Start a new browsing session and see if there is an improvement.

Next, use the Start menu to open a copy of Internet Explorer with all add-ons disabled. This is found in All Programs, Accessories, System Tools, Internet Explorer (no add-ons). and can also be opened directly from Start by typing iexplore.exe -extoff and hitting Enter. With all browser add-ons turned off, you should be able to start browsing almost immediately. If this resolved your problem to your satisfaction, use Tools, Internet Options, Programs, Manage Add-ons to disable some or all add-ons. Perhaps even create a menu item or shortcut to IE using the -extoff parameter so you start each browsing session with no add-ons.
