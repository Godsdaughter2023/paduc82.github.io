There is no separate Windows 10 license for Insider builds. Open OneDrive from the Office 365 portal (yes, it is a .sharepoint.com domain)

Click the gear icon and then click "OneDrive settings"

Click "More Settings"

Click "Regional settings"

If you receive the error “Sorry, OneDrive can't add your folder right now when you try to sync your OneDrive folder, this error can occur if following causes are true:

  1. “Require Check Out” was enabled on the library.
  2. There are Required columns
  3. The 'Draft Item Security' is set to 'Only users who can edit items
  4. Syncing using an external or guest account for OneDrive

 

For the resolution, please check the following article:

“Sorry, OneDrive can’t add your folder right now.”

If the issue continues, try syncing OneDrive for business library on a different location on your mac, check if this resolves he issue.

If the issue persists, please provide the following information:

  1. Does your root OneDrive folder name contain space?
  2. Please check if the folder name contains any unsupported character such as <, >, :, ", |, ?, *, /, \
