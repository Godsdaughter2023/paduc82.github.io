Port Forwarding - Microsoft Community

xb1r apollo say:

Port Forwarding

Login Information

 

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 

  

Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select Add.
  7. Select the radio button for "Custom Server" and enter Xbox.
  8. Enter the IP address of the Xbox into the "Server IP Address" box.
  9. Enter 3074 into the "External Port Start", "External Port End", "Internal Port Start", and "Internal Port End" in row 1.
  10. Select "TCP/UDP" from the "Protocol" drop down in row 1.
  11. Select Save/Apply.

 

Disable Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select the radio or check box next to the rule you would like to remove.
  7. Select Remove.

ken blake - microsoft mvp say:

On Sat, 1 Jun 2013 20:03:20 +0000, Sea Hand wrote:

Can anybody advise how to restore my Windows XP bkf files from an external hard drive to my new computer with Windows 8? My old computer crashed but I had backed up my documents and settings to an external hard drive, I can not afford to lose these but Windows 8 does not seem to recognise these bkf files in its restore commands - new to Windows 8 and don't know what I can do.

Bkf files are specific to Windows XP's backup program and can't be
used in Windows 8.

Your only recourse is to find an XP system and restore the bkf files
there. Then use a third-party program to back them up, so they can be
restored using that program on Windows 8. Or instead of a third-party
backup program, if you don't have too much data, you may simply be
able to copy them to a CD, DVD, or thumb drive so they can be copied
to the Windows 8 machine.
