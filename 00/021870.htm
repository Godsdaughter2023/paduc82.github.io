hansv mvp say:

Activate the Replace dialog (Ctrl+H).

Click More >>.

Make sure that Use wildcards is NOT ticked.

Leave the Find what box blank.

With the insertion point in the Find what box, click Format > Font...

Select Bold, then click OK.

In the Replace with box, enter ^p^&^p

Click Replace All.

Is it by design ? If by design then what could be the reason for it.

james yan msft say: Hi Vivjain,

Yes, we cannot delete just the current version of a file without permanently deleting the file and all versions of it. If we make a mistake in the latest version and we want to delete it, we must first restore a previous version and then delete the version that contains the mistake.
In a word, there must be a current version as long as the document exists.
See Restore or delete a previous version of an item or file.
