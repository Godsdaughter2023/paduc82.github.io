·         Click Start

·         Type msconfig into the search field

·         Click Selective startup

·         Uncheck Load startup items

·         Click the Services tab

·         Click Start

·         Type msconfig into the search field

·         Click the Services tab

·         Click OK

·         Click Restart Now

Go to the copied tracks and highlight all of them - right click and select 'Properties' - then the 'Details' tab... example as shown here -

Some of the 'media' fields will show 'multiple values' but you can add 'The Spinners' in both Contributing and Album Artist fields - (note that a semi-colon will appear after the name in 'Contributing Artist' - make sure it's be amended for each track as necessary and note there should be no number before the title in the details tab (it should show in the general tab though - if required). Click Apply and OK after each change...

It's worth going through them all to ensure they are correct before copying and pasting them back to the desired location in the music folder - if you are asked if you wish to replace a track that exists already - agree... any tracks that aren't replaced and still show in WMP with incorrect attributes can safely be deleted from WMP (highlight right click and choose 'Delete... from Library and Computer).

raveydavey say:

I had this problem too and it was really bugging me. Any document I opened had a document map and it would default to thumbnails.

I eventually solved it thus (in Word 2007):

  1. open a single document (default thumbnail view is shown).
  2. close the thumbnail view window.
  3. close Word.
  4. open the document again (no thumbnail should be shown)
  5. Now choose View -> Document Map. (the document map is shown).
  6. Close Word.
  7. Open the document again (the right document map should now be shown)

So it seems Word remember if you had the document / thumbnail window open, but not when its changed.
