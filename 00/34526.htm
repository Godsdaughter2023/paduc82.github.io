skisby2 say:

In my case, setting the registry HKLM\SYSTEM\CurrentControlSet\Services\i8042prt Start value to 1 as suggested in the tomshardware solution did not solve the problem.

I successfully resolved this problem as follows:

1) Powered the computer off, and connected the mouse to the PS/2 port.
2) With no other mouse connected and only a PS/2 keyboard connected, powered on the computer.
3) Using the tab key on the keyboard, navigated to an account that had administrator privilages and signed on.
4) Ran the Windows 8.1 built in "Hardware and Device Troubleshooter" as follows:
- Most systems will boot directly to the Start Screen.
- If you're not at the Start Screen and Classic Shell installed and running: Press Ctrl + Esc to bring up the Start Menu > Down arrow to select the Start Screen > Enter to enter the Start Screen.
- If you're not at the Start Screen and Classic Shell is not installed and running: Press Ctrl + Esc to switch to the Start Screen.
- Once at the Start Screen, press the tab key until the search icon is highlighted near the upper right hand corner of the screen.
- Press the Enter key to enter the "Everything" Search.
- Start typing "Troubleshooting".
- When "Troubleshooting" shows as a search result, press the Down arrow key to select that and press Enter.
- In "Troubleshoot computer problems", press the tab key until "Hardware and Sound" is selected and press Enter.
- Press the Down arrow key until "Hardware and Devices" is selected and press Enter.
- Press the tab key until "Next" is selected and press Enter.
5) The Troubleshooter will then run and report that a repair was made.
6) Using the keyboard computer's power switch, shut down and restarted the computer.
7) The mouse connected to the PS/2 port should now work and Device Manager should now show the mouse driver as installed without error.

Note: If the problem is both the PS/2 keyboard and mouse are not working, but a USB keyboard or mouse work when connected, see http://superuser.com/questions/751799/reinstalling-windows-8-1-ps-2-keyboard-and-mouse-drivers/.
