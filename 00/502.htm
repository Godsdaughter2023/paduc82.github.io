Modify sheet naming macro to delete special characters and constrain - Microsoft Community

a. user say:

Modify sheet naming macro to delete special characters and constrain to 31 characters.

I use a simple macro below to copy the contents of a sheet cell to the tab name of the sheet. This works fine, but to avoid naming errors, I would like to be sure that the tab name is reduced to 31 characters and that it doesn't use any special characters.

Sub RenameTabs()
' Renames all worksheet tabs with each worksheet's cell A1 contents.
'If cell A1 has no content, then that tab is not renamed.
   
  For I = 1 To Sheets.Count
    If Worksheets(I).Range("C2").Value <> "" Then
        Sheets(I).Name = Worksheets(I).Range("C2").Value
    End If
  Next
   
End Function

a. user say:

Most common tips & tricks to try in no particular order:

  • Run Windows Update troubleshooter or wu10.diagcab
  • Run Microsoft's Software Repair Tool
  • Use PowerShell (Admin) to run sfc /scannow and DISM /Online /Cleanup-Image /RestoreHealth
  • Uninstall any 3rd party antivirus program
  • Disconnect from the internet once the download is complete
  • Set Feedback & Diagnostics to FULL
  • Verify MS account
  • Unplug any USB devices
  • Switch Insider Ring to FAST
  • Reset Windows Update (use the auto-script)
  • Delete the "Download" folder
  • Rename the "SoftwareDistribution" folder by running RenameWU.cmd with admin rights

If all failed, re-installing on top of itself or a clean install should be considered (ISO image required).

Read this thread for instructions on how to manually download any Windows ISO image.
