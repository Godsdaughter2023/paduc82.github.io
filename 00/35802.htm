tryx3 say:

I used to have problems with updating media properties.  I now avoid them by editing properties in Windows explorer whilst WMP is closed.

In Windows explorer, right-click on the selected files, select properties, select the Details tab.  Sett he values in the fields in that tab, OK.  These changes should now be visible to WMP as well and should be retained without any problems.

Does this procedure not work for you?  If not, when does it go wrong?  When WMP has been started?

It sounds to me that you might have a Library problem in WMP.  I suggest checking the whole set of WMP troubleshooters.

1 Control panel, Troubleshooting, Hardware & sound -

then the online versions [this might be a duplication of effort but I've never been able to confirm that]

2 Media player library troubleshooter

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Library-troubleshooter

3 Media player settings troubleshooter

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter

I am suggesting 3 in case it is a configuration problem that is causing the library to go wrong.

If that has not resolved things then

4 Delete the WMP library database to force a complete rebuild - Close WMP, in the Start menu search box type 

%LOCALAPPDATA%\Microsoft\Media Player

In the Windows explorer window that appears delete all the files [but not sub-folders]. See if that has fixed the problem.

5 Delete the WMP database cache - Close WMP, in the Start menu search box type 

%LOCALAPPDATA%\Microsoft\Media Player

In the Windows explorer window that appears, select the Media Player folder in the left-hand pane & delete it together with its sub-folders.

6 If that hasn't fixed it then do both these steps [they are the closest thing possible to uninstalling then reinstalling WMP]

6.1 Unload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, clear Windows Media Player checkbox, Yes, OK, Restart the PC.

6.2 Reload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, set the Windows Media Player checkbox, Yes, OK, Restart the PC.

At the end of 6.2 you will effectively have a new installation of WMP. It will build a new library free of whatever was causing the previous problem unless there is a general Windows 7 error that is showing its symptoms just in WMP - I cannot imagine what sort of error could be doing that.
