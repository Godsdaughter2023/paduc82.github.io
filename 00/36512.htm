Reconnecting to network drives on boot up or restart causing Red - Microsoft Community

dubbin say:

Reconnecting to network drives on boot up or restart causing Red Crosses on drives and pop up warning - an answer for some.

For a long time now I've been trying to find a solution to the annoying problem of Windows not reconnecting to all networked drives on booting up or restarting my desktop. I've read many postings and discussions and not yet found one that works for me. I have a wired and WiFi home network with a NAS serving three computers and its only the wired desktop experiencing the "Could not reconnect all network drives" pop up with the red crossed NAS drive in File Explorer.

My answer came when I downloaded Windows 10 TP to a dual boot on my Win 8.1 desktop. I noticed that Win 10 had not mapped my NAS drive instead it had gaven it a Network Location. The NAS showed up in File Explorer as a networked file folder and acted just like a mapped drive but without a drive letter.

When I booted into Win 10TP or restarted I didn't get the annoying pop up or any red crosses in File Explorer.

I tried the same thing on my Win 8.1 installation. I disconnected the NAS by right clicking its icon in File Explorer and selecting disconnect from the menu. I then rebooted, entered File Explorer, selected "This PC" and clicked on the "Add a network Location" item in the Ribbon Menu then followed the "Add a Network Location Wizard" that popped up. I typed in my chosen network location, which in my case was \\192.168.1.69\Public , this being the address of my NAS drive's Raid 0 folder. Once finished, I had easy quick access to my NAS files, no red crosses, no annoying pop up warning and apart from no Drive Letter everything worked fine.

I realise this will not work for a lot of folk who may depend on having a mapped drive letter, but I'm sure it will be a great solution for lots with NAS drives on home networks. At least until Microsoft sort out the boot timing issues that caused the red crosses.

I know there are a lot of people looking for a solution and have been for ages, perhaps a more elegant explanation of this process by someone out there may help.
