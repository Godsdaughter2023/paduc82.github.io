Excel 2016 and 2013: Format Object: Move and Size with Cells does not work and Object moves freely on sheet

Excel 2016 and 2013: Format Object: Move and Size with Cells does not work and Object moves freely on sheet. In older copies of MS Office Excel when you posted a picture or object to a cell and right click/format Object/ Properties lists 3 options for Object Positioning:

1. Move and size with cells. In older versions a picture or the object would fill the cell and if it was made smaller or larger the object remained in the cell. 

2. Move but don't size with cells. This appears to be the only working option available. The object icon can still be moved out of the cell and misplaced on the sheet.

3. Don't move or size with cells. In my opinion this is the only working option, but this selection really means the object is not assigned to any cell and if the cell is made smaller the icon is free floating. 

This option does not work in Excel 2013 or 2016.

The option used to make the icon fill the cell and if the cell was made smaller the icon got smaller or larger if the cell size was increased. The reason this option is handy and needed is because it was the only way to post your file icons as objects and have them remain in a specific cell for reference. Currently, if you use an icon and the cell is resized then you will end up with all your icons outside the cell and not know which icon belonged to each reference. Without being able to contain the icon to one cell no matter how the cell is sized makes it useless to use icons to keep track of files. Your only safe option is to use a link which stays within the cell it is assigned to. Please fix this problem.
