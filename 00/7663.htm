aelearllo say:

I use Word.doc  2007 in makings lists, etc. within a folder.  Is there  a way you can select just the title of  each file without it going beyond the title into it? I would like just the file titles for references, etc. when working on a given project.

 

Dr. Alton Loveless

doug robbins - mvp office apps & services (word) say:

If you run a macro containing the following code, it will open a dialog in which you can select the folder containing the files and then it will insert the path and filename of each into the active document:

 

Dim fname As String

Dim PathToUse As String

Dim fd As FileDialog

Set fd = Application.FileDialog(msoFileDialogFolderPicker)

With fd

    .Title = "Select the folder containing the files."

     If .Show = -1 Then

         PathToUse = .SelectedItems(1) & "\"

     Else

         MsgBox “You did not select a folder.”

         Exit Sub

     End If

End With

Set fd = Nothing

fname = Dir$(PathToUse & "*.*")

While fname <> ""

    ActiveDocument.Range.InsertAfter PathToUse & fname

    fname = Dir$()

Wend

If you do not want the path, replace

 

    ActiveDocument.Range.InsertAfter PathToUse & fname

 

with

 

    ActiveDocument.Range.InsertAfter fname

swischner say:

This does not explain why your original .one files on your laptop seem to be corrupted but there is a good chance you do have another copy.

OneNote 2010/2016/2016 has an autobackup mechanism. the default setting just creates a copy of all (open!) notebooks every 14 days but maybe that'll help.

You find the location of the backup files in OneNote under Options -- Save (or the like; sorry I just have a german OneNote at hands). In the upper right there are three default paths shown. One of them is the location of the backup files. 

Important: Those backup files are in .ONE format. So there is a file for every section in the notebook (represented by the folder the ONE files are in), but the backup mechanism does not create a .ONETOC2 file. That one is a kind of index file containing a list of all sections and their correct order. So when restoring the backup file by file (you can do that from OneNote as well. File menu, top right corner, Restore Backups or the like) you will have to recreate the order of your notebook.

Hint: To transfer notebooks from one (office!) OneNote to another next time export them in ONEPKG format from within OneNote. On the target pc you just have to open that ONEPKG using the Explorer. OneNtoe starts and guides you through the steps of creating a new notebook and importing all content from the ONEPKG.

hansv mvp say:

The check boxes are contained in a text box. This text box has a transparent fill but a solid white line around it.

Right-click the white line and select Format Shape... from the context menu.

Click Line Color.

Select No Line.

Click OK.

 

The gray box itself is also a text box. The Line Color for this box has been set to black.

You can't specify individual borders for a text box - it's all around or nothing. You'd have more control over the borders if you used a table.
