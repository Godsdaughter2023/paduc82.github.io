jay freedman say:
The logic here gets a bit twisty, but stick with it. I think you can make it work.

To answer 'Can I programmatically set the "Default personal templates location,"' yes you can, by writing to the registry. First, find out whether you need to write it and, if so, what to write.
  • Open the key HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Word\Options. Try to read the value PersonalTemplates in that key. If it exists, the user has entered a "Default personal templates location". You should put your templates in the folder named in that value, and not write to the registry.
  • If the PersonalTemplates value doesn't exist yet, look for the folder %userprofile%\Documents\Custom Office Templates. If it exists, write to the registry and specify that location as the PersonalTemplates value.
  • If the PersonalTemplates value doesn't exist yet and there is no Custom Office Templates folder, I'll suggest creating the folder %userprofile%\Documents\Custom Office Templates, storing your templates there, and specifying that location as the PersonalTemplates value. (You can create and specify another folder, but then Word won't use the Custom Office Templates folder, and that might cause support problems later.)
The PersonalTemplates value has to be a REG_EXPAND_SZ value. If your installation tool supports writing a REG_EXPAND_SZ value by specifying a string, you're done. If you have to use a .REG file, that requires converting each character of the string to hex and specifying the value's data type as hex(2), for example:

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Word\Options]
"PersonalTemplates"=hex(2):25,00,61,00,70,00,70,00,64,00,61,00,74,00,61,00,25,\
  00,5c,00,4d,00,69,00,63,00,72,00,6f,00,73,00,6f,00,66,00,74,00,5c,00,54,00,\
  65,00,6d,00,70,00,6c,00,61,00,74,00,65,00,73,00,5c,00,50,00,65,00,72,00,73,\
  00,6f,00,6e,00,61,00,6c,00,00,00


  • After the Personal Templates value is specified in the registry, and Word is restarted, the path will appear in the Options dialog's Save page. In the New tab of the Backstage view, there will be links labeled FEATURED and PERSONAL just below the search box (but if the Workgroup Templates location has also been specified, the labels will be FEATURED and CUSTOM -- a complication that I think was a big mistake by Microsoft). Your templates and any that the user has saved will be accessible by clicking the PERSONAL (or CUSTOM) link. This doesn't require any individual setup, but will require a little documentation so users can find your templates.
