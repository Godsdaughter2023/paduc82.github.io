how to format the underlines in the foreign language written text ?

I have wrote in a different language , world underlines the content .

daniel g. grau say:
Do they look like this?


If so, go to Word > Preferences > Spelling and Grammar. Under Spelling, disable Check spelling as you type (red underlines). Under Grammar, disable Check grammar as you type (green underlines).

You should apply the proper language to whatever text you are typing. To apply the proper language, select the text, then choose Tools > Language and apply the proper language. (Note that language is not a Word setting, but a text attribute, like bold or font color.)
