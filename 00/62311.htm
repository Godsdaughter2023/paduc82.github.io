Welcome to Microsoft Answers.

A) Unregister Windows Installer, and then reregister Windows Installer. To do this, follow these steps:

1. On the "Start" menu, Type Run in the search bar and hit enter.
2. In the "Open" box, type "msiexec /unreg", and then press ENTER.
3. On the "Start" menu, Type” Run” in the search bar and hit enter.
4. In the "Open" box, type "msiexec /regserver", and then press ENTER

B) The Windows Installer service may be disabled on the machine.

1. Go to "Start" -> "Run", type "services.msc" and press on "Enter" button.
2. Double-click on the service named "Windows Installer".
3. Check the value in the "Startup type:" field. If it's currently set to "Disabled", then this is the problem. Change it by selecting "Manual" from the drop-down box and “Start” the service.

lemp say:

That sounds like a reasonable analysis.

I can give you a place to start looking, but I can't give you step by step instructions on the modifications you'll need to make to the Registry.  Be sure to backup the Registry before you start experimenting.

The lists of drivers and services loaded in Safe Mode and Safe Mode with Networking are located at these two Registry keys:

     HKLM\SYSTEM\CurrentControlSet\Control\SafeBoot\Minimal

     HKLM\SYSTEM\CurrentControlSet\Control\SafeBoot\Network

On my Win 7 Pro system, the third subkey (which starts with 4D36E967) is the ClassGUID associated with disks and other storage devices, including SSD, spinners, USB flash drives, and other solid state storage devices.   You can see that string, for example in the data for HKLM\CurrentControlSet\Enum\IDE\<name of disk drive>. 

Although each different device shows a different numeral at the end of the string for the "Driver" value (e.g., the "0001" in the entry for an SSD shown below), all of the entries under HKLM\SYSTEM\CurrentControlSet\Control\Class\{4D36E967-E325-11CE-BFC1-08002BE10318}\00nn appear to be the same ... but perhaps that's because all of the storage devices on this system use the same generic Microsoft drivers.

I'm sorry, but I don't know where you go from there or what class you need to add under HKLM\...\SafeBoot\Minimal
