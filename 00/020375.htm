Highlight entire row based on one cell containing a "last month" date

The charts will vary in size so I need it to select the entire sheet and highlight rows in which column B contains a date from last month.

 

I don't want to have to update the formula every month. Is there a way to do this?

Thanks!

 

jwess say:

By following ideas from this thread, I was able to come up and use this with VBA:

 

Sub highlight row()
'highlight entire row in grey if date in B column is from last month
 
    Range("$A$2:$M$1000").Select
    Selection.FormatConditions.Add Type:=xlExpression, Formula1:= _
        "=MONTH($B2)=MONTH(TODAY())-1"
   
    Selection.FormatConditions(Selection.FormatConditions.Count).SetFirstPriority
    With Selection.FormatConditions(1).Interior
        .Pattern = xlNone
        .Color = 12632256
        .TintAndShade = 0
    End With
    Selection.FormatConditions(1).StopIfTrue = False


End Sub
