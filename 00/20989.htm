jay freedman say:

Even when the "Automatically use suggestions from the spelling checker" is checked, there's an additional criterion that isn't mentioned: the automatic replacement will occur only if the spelling checker has exactly one suggestion. 

For example, type the phrase

don't stap believig

You'll see that 'believig' is automatically corrected to 'believing' because that's the only suggestion from the spelling checker. However, 'stap' remains underlined and uncorrected because the spelling checker returns five suggestions for it.
