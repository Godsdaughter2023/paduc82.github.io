Something like the following.

    Dim wsActive As Worksheet
    Set wsActive = ActiveSheet
    Worksheets("Sheet3").Activate
    wsActive.Activate

Alternatively, hide the worksheet and then unhide it again. (Needs to be reactivated after being unhidden because another worksheet becomes the active sheet when a worksheet is hidden.)

    Dim wsActive As Worksheet
    Set wsActive = ActiveSheet
    wsActive.Visible = xlSheetHidden
    wsActive.Visible = xlSheetVisible
    wsActive.Activate
