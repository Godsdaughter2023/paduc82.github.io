ken sheridan say: A report would be the best way to do this, but it's not a trivial task.  It involves writing code in the report's detail section's Format event procedure which steps through the controls from left to right and, for each control which is Null, firstly sets the Visible property of the control and its label to False, and secondly adjusts the Left property each label and bound control to the right of the hidden control by subtracting a constant value (in twips) from the default Left property of the label and bound control.

If more than one control has been hidden, then the value by which the Left property of the controls to its right must be reduced is a multiple of the constant value of course.  The default Width properties of each control must be equal if the Left properties are to be adjusted on the basis of a single constant value.  Otherwise it would be necessary to have a different constant value relating to each control, and rather than reducing the Left property of a control to its right by a multiple of a single constant value, it would be reduced by the sum of the constant values relating to those controls which have been hidden.
