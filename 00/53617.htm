1. What is the make and model of the computer?

2. Did you make any recent changes on the computer prior to this issue?

This issue could have occurred due to these reasons:

1. Conflict between third party applications.

2. Corrupt audio driver.

Try the steps listed here and check if it helps:

Method 1:

Follow these steps:

a. Close all back applications, especially, Sound Recorder or other multimedia applications.

b. At Windows Start Tile screen, move the cursor or touch lower right screen & bring out the "Search" function.

c. At the "Search" panel, look for "Control Panel" and then launch it.

d. At the "Control Panel", look for "Sound" and then launch it.

e. Click on the Recording tab.

f. Select the recording source, for example, Line In 2/Mic 2, and then click Set Default. Recording device will be marked with a green tick mark.

g. Adjust the input level if you have difficulty recording. Click Properties, and then click on Levels.

Method 2: Update the Sound Card Drivers.

Step 1: Check the Windows updates for any latest driver updates.

a) Press “Windows Logo” + “W” keys from the keyboard.

b) Type “Updates” in the search bar and click on “Install optional updates” from the left pane.

c)  From the “Windows Update” window, click on “Check for Updates” on the top left corner to receive a fresh list of updates and install all the updates listed there.

Step 2: Try updating the drivers from the Device Manager.

a) Press “Windows Logo” + “X” keys on the keyboard.

b)  Click on “Device Manager”.

c)  Search for the Sound Card from the device list and right click on it and select “Update Driver Software”.

d)  Follow the on-screen instructions and complete the installation of the latest driver.

e) If no driver updates are available, you may also search for the updated sound card drivers in the “Drivers” and “DriverStore” folders in the location “C:/Windows/System32” by selecting the option “Browse my computer for Driver Software”.

Step 3: Update Driver Software from the Manufacturer’s website:

Go to the computer or sound card manufacturer’s website and search for the latest Windows 10 audio drivers available for the sound card model and then install it by following the instructions given in the website.

    Canon MP730 x64 Drivers - Microsoft Community

    shaananc say:

    Canon MP730 x64 Drivers

    Hi,
    I'm trying to connect to my Canon MP730 Multifunction printer via the network as I did under XP and Vista and there are 64bit drivers avaliable for the printer connected via USB however I can't get them to work over the network,
    Thanks.
    Shaananc

    takeo yoshida say:

    Hi Shaananc,

    This solution enable to print any networked printer (served with any windows OS)

    Try following steps,

    ===================================
    Printer Server side Vista(64bit) (or any windows OS)
    ===================================
    0)Must be login with administrator authority.
    1)Check/Set computername "vista64" (any name).
    2)Check/Set workgroup name "myhome" (any name).
    3)Share your vista64's printer name "mp730" (any name).
    4)Get your vista64's IP address.(run "cmd", and type "ipconfig" in the cmd window)
      ex)192.168.1.12

    5)install printer driver.
      If your printer is Canon, so thier latest driver can be download from your country Canon web site.
      for just in case, here is Japanese Canon's original download site.
      http://cweb.canon.jp/drv-upd/multipass/mp-win.html

     ex) Vista64bit and MP730
     This printer can be used vista's standard printer driver, you may upgrade it with..
      http://cweb.canon.jp/drv-upd/bj/aomvstea23.html


    ===================================
    Client side  WindosXP(32bit) (or any windows OS)
    ===================================
    0)Must be login with administrator authority.
    1)Check/Set workgroup name "myhome" (must be same as Server side).
    2)Install local printer.
    3)Use LPT1(printer port)
    4)Select your Maker/Printer Name, and install driver.

     ex) If client is WindowsXP(32bit), then you should download from following URL.
     http://cweb.canon.jp/drv-upd/multipass/pixusmp700w2k.html

    After installed MP730 printer(as local printer)

    5)Open this Printer's property screen.
    6)Select "Port" tab
    7)Button down "Add Port"
    8)Select "Local Port"
    9)Button down "New Port"
    10)Enter port name
     \\vista64\mp730  (\\vista64's machine name \ shared printer name)
     or
     \\192.168.1.12\mp730 (\\Vista64's ip address \ shared printer name)

    11)Button down "OK"


    Now you can print via network to your VISTA64's MP730.
    I hope this help you.

    Takeo Yoshida
