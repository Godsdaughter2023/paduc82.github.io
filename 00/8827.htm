hansv mvp say:

In a standard module, created by selecting Insert > Module in the Visual Basic Editor:

Public dtmTime As Date
 
Sub StartMail()
    Call SendMail ' this is the macro that sends out emails
    dtmTime = Now + TimeSerial(0, 30, 0)
    Application.OnTime EarliestTime:=dtmTime, Procedure:="StartMail"
End Sub
 
Sub StopMail()
    Application.OnTime EarliestTime:=dtmTime, Procedure:="StartMail", Schedule:=False
End Sub

And in the ThisWorkbook module:

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Call StopMail
End Sub
 
Private Sub Workbook_Open()
    Call StartMail
End Sub
