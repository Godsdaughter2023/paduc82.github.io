eddieb- say: Hi,
A couple of different things here:
1. when you "rip" a cd you are actually taking the musicfrom the cd and reproducing it somewhere on your hard drive or media card. (where ever your settings are set to place the music)
2. Whichever program is set as the "default" media player will have access to those files (if they are all stored in the same place)
Example, I too use Win Amp as my primary music player, but Windows Media Player is still set as my default. So If I try to rip a cd, WMP comes up. If I simply double click on a random music or video file, WMP opens the file and that is no matter where the file is located on my hard drive or a media card I insert. Now I can change that by making Win Amp my default media player, if I wanted to. Neither of those programs will "move" files around. If you want to change the location where files are stored after being ripped or downloaded, you can do that in the program's settings.
Hope this helps,
Eddie B

The desktop and the "don't-call-it-metro" modern environments can actually have two totally different DPI pixel density (size) settings.  It sounds like you're already done with the one for the desktop environment.
