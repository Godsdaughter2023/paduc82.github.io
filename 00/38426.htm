Hi,

You do not say what type of License it is/was on the old machine.

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) or is an OEM System Builders License, that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) and OEM System Builders License are tied to the first machine it is installed on and dies when that machine dies.

Only Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

And Retail Licenses can also only be installed on one machine at any particular time.

davem121 say:

If you have a lot of files in a folder, the best way is to run a script in Command Prompt to create a listing of the file names and then paste that list into Excel:

Go into the folder in Windows File Explorer

Press Shift whilst right clicking an empty area in that folder

Select 'Open Command Prompt Window here' 

To get a list of all files in that folder, run this command:

dir /b /a-d *.* >%userprofile%\Desktop\list.txt

To get a list of just jpg files in that folder, run this command:

dir /b /a-d *.jpg >%userprofile%\Desktop\list.txt

Then you can open the text file, which you will find on your Desktop, copy all and paste into an Excel document. The file names will automatically be placed into separate cells in Excel


==================================================================================

Please Note: if you are running the Creators Build of Windows 10, Command Prompt is not accessible by Shift + Right Click in Windows File Explorer, only Powershell is and this command does not work in Powershell . . .
