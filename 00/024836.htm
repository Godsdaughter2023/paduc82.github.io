tim liu msft say:

Hi SunChatterjee,

As you mentioned, Distribution list cannot be used in SharePoint Online. We need to use Office 365 group, Security group or Mail enabled security groups in SharePoint Online.

We could access Office 365 admin center>Groups>Add a group>select Security groups (or other two types of groups)>add owner and users to that group. And then in SharePoint Online, you may share items (click Share>enter the group name) or add permissions with these groups (click setting gear>Site Settings>Site Permissions>Grant permissions>enter the group name). 

So, if in steps mentioned above, you add the Distribution list to the security group, you could change to add individual users to the group. And then check the outcome.
