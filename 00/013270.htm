Hello Kciwart,

 

 

You may try the following:

  1. In Outlook 2007, click on the Tools menu, and then click Account Settings.
  2. Click the E-mail tab, and then click New.
  3. Click to select Microsoft Exchange, POP3, IMAP, or HTTP, and then click Next.
  4. Click to select the Manually configure server settings or additional server types check box, and then click Next.
  5. Click to select Internet E-mail, and then click Next.
  6. On the Internet E-mail Settings screen, type the following setting information:
  7. In the Your Name box, type your name as you want it to be displayed to recipients.
  8. In the E-mail Address box, type your email address. For example, type *** Email address is removed for privacy ***.
  9. In Account Type, select POP3.
  10. In the Incoming mail server box, type pop3.live.com.
  11. In the Outgoing mail server (SMTP) box, type smtp.live.com.
  12. In the User Name box, type your MSN account name.
  13. In the Password box, type your MSN password.
  14. Click to select the Remember Password check box, if you want to save it.
  15. Leave the Require logon using Secure Password Authentication (SPA) check box unchecked.
  16. Click More Settings.
  17. Click the Outgoing Server tab, and then click to select the My outgoing server (SMTP) requires authentication check box.
  18. Click the Advanced tab, and then click to select the This server requires an encrypted connection (SSL) check box under Incoming Server (POP3).
    Note The Incoming Server (POP3) box should be set to port 995.

  19. Select TLS for Use the following type of encrypted connection under Outgoing server (SMTP), and then click OK.
    Note The Outgoing Server (SMTP) box should be set to port 25. If port 25 is blocked in the network or by your ISP, you can set SMTP port to 587.

  20. On the right side of the Internet E-mail Settings scree, click Test Account Settings.

    If you do not receive errors, your email account is set up correctly. Go to step 8.

    If you receive an error, make sure that you have entered all the information correctly, and then click Test Account Settings again.
    If all the information is correct and you continue to receive errors, go to the "Next Steps" section for more help.

  21. Click Next, and then click Finish.

 

PS: Read more at http://support.microsoft.com/kb/287604/en-us

 

 

Cheers,
