chip pearson say:

Try code like the following. Change the marked with <<< to the appropriate range. When the procedure ends, LastCell will be a Range object referencing the cell containing the closest date to the current date.

 

Sub AAA()
    Dim R As Range
    Dim LastDiff As Date
    Dim LastCell As Range
    LastDiff = DateSerial(9999, 1, 1)
    For Each R In Range("A1:A10") '<<< CHANGE
        If Abs(R.Value - Now) < LastDiff Then
            Set LastCell = R
            LastDiff = Abs(R.Value - Now)
        End If
    Next R
    Debug.Print LastCell.Address, Format(LastDiff, "#,##0")
End Sub

talking moose say:
Outlook and mobile devices such as iPhones and iPads use two different
protocols to connect. What works for one doesn't mean the other will
work. However, most administrators make the settings match to simplify
things.
 
You'll need to contact your organization's Exchange administrator or
Help Desk and ask if they allow external connectivity via the Exchange
Web Services (EWS) protocol. That's what Outlook for Mac needs. It's not
enabled by default. This should be the same address you would use to
access Outlook Web Access (OWA) via a web browser.
 
Hope this helps!
