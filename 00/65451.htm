Hi Daniel

Windows 10 comes in a few different Editions - Home - Home Single Language - Pro . . .etc.

On the Media Creation tool, you may be offered just Windows 10, if that is the case, then on the next Window you should be offered to install either the Pro or Home Edition

If you want to install a specific edition of WIndows and you are not offered that option, you can force the Installation media to give you that option

After you create the installation media on a USB flash Drive

Right click your Desktop, choose New - Text Document

Paste the text below into that document

[EditionID]
Core
[Channel]
Retail

Then choose File - Save Aa, at the bottom of the Save As dialog, change file type to 'All Files' then save that file as ei.cfg (note you must manually add the cfg file extension)

Then copy that ei.cfg file into the Sources folder in your USB Installation Media

Then when you begin the Installation, you should be asked which Edition you want to install . . .

 Open Services

Open File Explorer

Create a new Folder called Backup

Move the following key which starts with f686aace6942fb7f7ceb231212eef4a4_ to the Backup folder

Go back on level to Go to C:\ProgramData\Microsoft\Crypto\RSA\

Click Apply

------------------------------------------------------

Hi, these extra user profiles are created during a Windows 10 update.

When Windows 10 performs a major update, your user profile is copied to a temporary profile and once the update completes, the profile is copied back.

What happens sometimes is that the process of copying the profile back does not complete and you end up with extra profile(s) or you might even end up signed into a temporary profile and not be able to find your files . . .
