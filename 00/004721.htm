vijay a. verma say:

I have created a mock-up for you which can be downloaded from following location 

https://1drv.ms/u/s!Akd5y6ruJhvhiSESMfmmLNbicc82

1. Put all your images in a tab. I have named this tab as Images. Measure the width of your columns and rows of your merged cell in first sheet and create the cells in Images tab in column B for this size. (This is required for aesthetics purpose, you can do manually also). Insert all your images in different cells of column B.

2. Align your pictures appropriately in these cells.

3. In the View tab, uncheck the gridlines so that gridlines don't appear in Images tab (otherwise grids around images will also appear in your first sheet)

4. In column A of Images tab, put the names which you have in dropdown list against those images (better, you create your dropdown from this column)

5. Formulas tab > Name Manager > Create > Name it ShowPic (or any other name) and put following formula in Refers To:

=IF(Sheet1!$B$2="","",INDEX(Images!$B:$B,MATCH(Sheet1!$B$2,Images!$A:$A,0)))

You will have to change references appropriately like B2 in your case is F11.

6. Insert a picture (it doesn't matter which picture from your list)

7. Select the Picture and put =ShowPic in formulabar.

8. You are good to go.

bernie deitrick say:

This should do it:

Private Sub Worksheet_Change(ByVal Target As Range)

    If Intersect(Target, Range("F:F")) Is Nothing Then Exit Sub
    ActiveSheet.Unprotect ("password")
    Cells(Target.Row, "H").Locked = (Not (UCase(Target.Value) = "Y"))
    ActiveSheet.Protect ("password")
End Sub

But if you use this to protect the sheet, then VBA can make changes without the protection toggle

Sub ProtectSheet()

    ActiveSheet.Protect "password", UserInterFaceOnly:=True

End Sub

And your code could be

Private Sub Worksheet_Change(ByVal Target As Range)

    If Intersect(Target, Range("F:F")) Is Nothing Then Exit Sub
    Cells(Target.Row, "H").Locked = (Not (UCase(Target.Value) = "Y"))
End Sub

If you need to apply the locking without making changes to F again (since you already have it done) then you could use a regular macro:

Sub UnlockH()

    Dim R As Range
    ActiveSheet.Unprotect ("password")
    For Each R In Intersect(ActiveSheet.UsedRange,Range("F:F"))

          Cells(R.Row, "H").Locked = (Not (UCase(R.Value) = "Y"))

    Next R

    ActiveSheet.Protect ("password")

End Sub
