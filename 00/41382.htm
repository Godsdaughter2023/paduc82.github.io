dominicp say: Any 32-bit Windows can address a maximum of 4GB, but hardware addresses take part of the address space so you will never be able to use more than about 3.5GB of RAM.

If you had a 1GB graphics card fitted you might only be able to use around 2.7GB of RAM.

64-bit Windows with your i7 would be able to use all 4GB of RAM because the hardware addresses with 64-bit would be mapped above 4GB.

Hi Ben

Rest Assured, it is safe to delete all files suggested by Disk Cleanup, that utility will never suggest anything that is not safe to delete, removing any of the items suggested will never have an adverse affect on your PC

Disk Defragmenting:

When files are stored on your hard drive, the hard drive tries to store the file in one location, this is not always possible and sometimes bits of the files are stored in different empty parts of the hard drive

Defragmenting a hard drive will put move the data on your hard drive around so all those bits are put together and will make it faster to open those files, because the heads on the hard drive will not need to look all over your drive for those bits

If you have an SSD Drive rather than a standard hard drive, then you never need to defragment your hard drive

Defragmenting can be set to run on a schedule and automatically keep your hard drive in good order

Hello Ianculescu,

Microsoft uses Windows Error Reporting (WER) to transmit troubleshooting information and updates for specific problems in Windows.

WER is a multi-stage communication protocol. The first stage of WER that includes the basic report parameters is not encrypted. Any detailed information that is used to help determine the cause of a problem is sent in the second stage of reporting. This second stage is encrypted over a Secure Sockets Layer (SSL) connection.

This means that it is a part of the Windows error reporting system that collects and send error logs from your system, to Microsoft.

Microsoft Windows Error Reporting Service helps Microsoft and Microsoft partners diagnose problems in the software you use and provide solutions. This does not entirely mean that wermgr.exe is the issue, it means that something triggers WER causing the pile-up of notification in your system.

Disabling Error Reporting is not really a resolution, it will only hide the notifications your system is trying to alert you about.

The issue you're facing is normally resolved by a patch or update released by Microsoft once we collate the data your system sends.
