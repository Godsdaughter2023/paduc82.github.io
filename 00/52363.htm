This is is caused by having too large an allocation unit size for the number of files on the drive.  You will need to reformat the drive to a much smaller allocation unit size to accommodate the more than 500,000 files you are storing.  The easier way to determine what your current allocation unit size is, is to run chkdsk on the drive. At the bottom of the chkdsk report will be listed the number of bytes in each allocation unit.  I recommend you use 4096 bytes in each allocation unit.

To learn more about this google "calculate allocation unit size"
