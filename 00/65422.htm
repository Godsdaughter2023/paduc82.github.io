IE can create two different types of Internet Shortcuts --- .url (Internet Shortcut) & .website (Pinned Site Shortcut)

.url .website
created via:

File -> Send -> Shortcut to Desktop

      --- or ---

Right-click in page background ->Create Shortcut

 Tools -> Add Site to Apps
Created in: Desktop

All Apps

(shell:start menu)
Opens with: Default Browser Internet Explorer
(but can be modified)

So if you want shortcuts that can be pinned to Start, they need to appear in All Apps, .website files appear there when cfeated, but you need to change the association for them if you want them to open in Edge.  I describe that here:

http://answers.microsoft.com/en-us/windows/forum/windows_10-start/renaming-tiles-in-windows-10/2ed46ae5-d2c5-42af-bc39-255a84975397?page=5#LastReply

To get the correct size of the desktop background picture on the screen of the computer, you may select the option to fit the picture with the computer screen. Please follow these steps:

d) Now, on the left-bottom corner of the “Desktop Background” window, select the option “Fit” from the “Picture Position” menu.

e) Click on “Save Changes” button and then close the window.

f) Now, come to the Desktop Interface and check if the desktop background picture fits with the computer screen.
