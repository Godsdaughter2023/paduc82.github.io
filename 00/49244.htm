If I set the power setting to never shut off the computer everything runs as scheduled. What's the problem/solution?

shinmila h say:

Hey gapoore,

 

Perform the following steps to fix the wake up issue of external drive.

 

Method 1: Set USB selective suspend to Disabled.

 

a.    Go to Start, type  Power Options in the start field and press Enter.

b.    Select Change plan settings next to the power plan you want to edit.

c.    Select Change advanced power settings.

d.    Select Change settings that are currently unavailable.

e.    Set USB selective suspend setting to Disabled under USB settings and then click OK to save the changes.

 

Method 2: Update the Seagate drivers to the latest.

 

Check the support section of Seagate website for latest updates.

Here’s the link: http://www.seagate.com/www/en-us/support/downloads/

 

Method 3: Reinstall USB controller to see if it resolves the issue.

 

To do so, follow these steps:

 

a.    Type Device Manager in the start orb search field.

b.    Click on  Device Manager.

c.    Expand Universal Serial Bus controllers.

d.    Right-click every device under the  Universal Serial Bus controllers node, and then click Uninstall to remove them one at a time.

e.    Restart the computer.

 

Windows will automatically install drivers for the controllers.

 

Test and validate if this changes anything.

 

Method 4: Perform Windows Updates

 

Update your computer to make sure that you have the latest updates for improved security and reliability.

 

Refer:

 

Install Windows  updates

 

That should be it.

Regards,

Shinmila H - Microsoft Support

 How to activate Windows 7 manually (activate by phone)

1) Click Start, and in the Search box type:   slui.exe  4

5) Stay on the phone (do not select/press any options) and wait for a person to help you with activation.

 

Additional information:


Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

 

A)   OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the special instructions found only on that Manufacturer's computer hardware. So when Windows was installed using the OEM SLP key (at the factory) Windows 7 looks at the motherboard and sees the special instructions and Self-Activates. (that's why you did not need to Activate your computer after you brought it home)

 

B)    COA SLP: This is the Product key that you see on the sticker on the side (or bottom,[ or inside the battery compartment]) of your computer. It is a valid product key, but should only be used in limited situations (such as if the OEM SLP key stops self-activating for whatever reason). The key must be activated by Phone. (Note: All manufacturers that use the OEM SLP system are required by contract to include a Certificate of Authenticity (COA) sticker, that has a COA SLP key, on the computer)
