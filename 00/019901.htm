    How to automate conversion of 1.x-5.x versions of Word files? - Microsoft Community

    davidcb2 say:

    How to automate conversion of 1.x-5.x versions of Word files?

    I have a large number of 1.x-5.x versions of Word files that I would like to open and save as .doc or .docx in 2008 (OS X 10.6.8) so they are compatible with 2011. (If it would be easier, I could reinstall 2004 and open and save them from that.)

    What is the best way of automating this?

    Thank you,

    David
    daniel g. grau say:
    You probably won't be able to open them automatically (as in selecting several files in the Finder and opening them) because Word 2011 has a safety trigger, as those files are considered "insecure." You can still open them from the File > Open menu, but this would take a long time.

    Your easiest method would be going back to Word 2004 (I believe it does not have the "security" feature, but I cannot test because I don't have any old files available) and run the following five-line macro:

    Sub SaveAllAsDoc()
    For Each aDocument In Application.Documents
        aDocument.SaveAs Filename:=aDocument.FullName & ".doc", FileFormat:=wdFormatDocument
    Next aDocument
    End Sub

    Open several documents at once and run the macro to SaveAs in the same folder (the macro will add the extension .doc to all of them, so it won't overwrite them, but you may end up with documents named .doc.doc).

    If you want them to close automatically, use this six-line version instead:

    Sub SaveAllAsDoc()
    For Each aDocument In Application.Documents
        aDocument.SaveAs Filename:=aDocument.FullName & ".doc", FileFormat:=wdFormatDocument
        aDocument.Close
    Next aDocument
    End Sub

    If you don't know how to install a macro, see Install a Macro in Word or Macros in Mac Word.

    daniel g. grau say:
    A macro could be devised to make Word go through your whole disk, but it's too much trouble. What you can do is use the Finder's Find to locate the files by date, for instance, or better yet…

    FileBuddy
    Drop one of the old files on FileBuddy to obtain it's File Type and Creator (both are 4-letter codes). Then do a Find operation to look for those files. Once you have them in the results window, just select them all and open them in Word 2004.

    Function ThirtyDaysAfter(dt As Date, vHolidays As Variant) As Date
    Dim vwh(1 To 7, 1 To 2) As Variant
    vwh(1, 1) = 0: vwh(1, 2) = 1
    vwh(2, 1) = 0: vwh(2, 2) = 1
    vwh(3, 1) = 0: vwh(3, 2) = 1
    vwh(4, 1) = 0: vwh(4, 2) = 1

    vwh(6, 1) = 0: vwh(6, 2) = 1
    vwh(7, 1) = 0: vwh(7, 2) = 1
    ThirtyDaysAfter = Int(add_hours(dt, 30.0001, vwh, vHolidays))
