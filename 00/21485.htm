You need to enter this formula in N2

=IF(SUMPRODUCT(--NOT(ISERR(SEARCH($Q$1:$R$1,$C2,1))))=1,D2,"Not Found")

 

The SUMPRODUCT formula returns 1 if cell C2 contains Coles or Safeway, you need to wrap it in an IF statement to return other values.

brian tillman [outlook mvp 2007-2019] say:
This happens to most people from time to time. When a spammer sends his or her spew, the sending address is often set to some innocent person's address. This is called address spoofing. Now, a spammer's message is sent to lots and lots of addresses, many of which are non-existent. When the receiving domain gets the message for delivery, that mail service sees that the address doesn't exist and sends a non-delivery report back to the sending address, which in this case is yours because that's what the spammer inserted in the message. Hence, you receive NDRs from people you never sent to and whom you don't know. This is often called backscatter and is very difficult to eliminate. Usually you can create a rule that handles it.

Can't provide a definitive answer without knowing what type (POP3, Imap, Outlook.com/Exchange) accounts are involved.

A generic overview would be

1) For POP3 - copy the PST file associated with the POP3 account and make sure that you use it when configuring the same account on the new machine with '2019
On the system with OL '2013, go to Outlook File --> Account Settings --> Email tab --> highlight the POP3 account --> go down to the <Change Folder> button and take note of the folder path/file name of the PST file where emails are being delivered.

Then switch to the Data Files tab in Account Settings and take note of all other PST files configured in the profile. All those need to be copied to the new machine and added to '2019

2) For IMAP - if there are contact/calendar folders in the IMAP folder tree, those folders should be exported to a PST file and copied to the new machine otherwise that data will be lost

3) For Exchange - nothing to copy, everything is sync'd with the server.

As a side note, you should also take note which data file in the profile is set as the "default data file" which is where the default contact/calendar folders are stored. In OL'2013 - go to Account Settings --> Data Files tab

For the most part, there should be no reason to "import" anything but can provided more specific info once the above info about the accounts is provided.

Word 2007 number format that includes spacing after 3pt - Microsoft Community

don899 say:

Word 2007 number format that includes spacing after 3pt

In Word 2007, I would like to have a number format that already includes the parameters: Left indent 0", text indent .2", spacing after 3pt, line spacing single. How do I create a style with these parameters that I can select for each document?

jay freedman say:
Your request doesn't quite make sense -- the parameters you mentioned all belong to paragraph styles, and have nothing to do with a "number format". In other words, such a style can be applied to any text, not just numbers. Also, Word doesn't have any concept of "number format" the way Excel does, except when you're formatting the result of a field that gives a numeric result.

To create a paragraph style with the parameters you described:
  • In any paragraph in any document based on the Normal.dotm template, right-click and choose Paragraph.
  • In the Paragraph dialog, set the Left indent to 0". Under Special, choose "First line" (I think this is what you meant by "text indent") and set the box to its right to 0.2". Set the Spacing After to 3 pt and set the Line Spacing to Single. Click OK.
  • Click the dialog launcher (the little arrow in the bottom right corner) of the Styles group on the Home tab (keyboard shortcut Ctrl+Alt+Shift+S) to open the Styles pane.
  • At the bottom of the Styles pane, click the first of the three buttons to open the Create New Style dialog.
  • In the dialog, assign a name to the style. Click the option button at the bottom for "New documents based on this template", which will add the style to the Normal.dotm template. Click OK.

The style will be available in any new documents you create based on Normal.dotm. To make it available in existing documents, you'll have to use the Organizer to copy the style from Normal.dotm to the document. (Click Office button > Word Options > Popular and check the box for "Show Developer tab in the Ribbon" and click OK. On the Developer tab, click Document Template; in the dialog that appears, click Organizer at the bottom.)
