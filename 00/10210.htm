The problem is that PowerPoint (all versions) does not measure in cm or even inches. It uses POINTS under the hood. One point=0.013827 inches! You will see often 72 points= 1 inch but even that isn't exactly correct.

So the figure you see in the boxes is a conversion from points and hardly ever exactly accurate. Because 2013 rounds to 3 places and 2010 to 2 places of decimals you see the difference.

compile error in hidden module1 - Microsoft Community

stefan ht say:

compile error in hidden module1

When I´m opening Word program I always get an error, Compile error in hidden module: Module1 can anybody guide me thru this. it is frustrating 
jay freedman say:
It means you have an add-in (usually a template stored in the folder %appdata%\Microsoft\Word\Startup) that contains one or more macros, and at least one of those macros has an error that prevents it from loading. It may be that the macro was written for an earlier version of Word and isn't compatible with your current version, or it may have always had wrong syntax.

The steps to fix the problem are:

  1. Shut down Word.
  2. Go to the folder %appdata%\Microsoft\Word\Startup and move all the templates (*.dot and *.dotm) from that folder to some other folder such as C:\temp.
  3. Restart Word to check that the message doesn't appear. Shut down Word again.
  4. If the Startup folder had more than one template in it, move one of them back into the Startup folder and then restart Word. If the message doesn't appear, repeat this step with another template.
  5. When you move one template and the message appears again, you've found the one containing the error. If it's something you don't need, just delete it. If it's something important, find out where it came from and see if there's a later version available.
