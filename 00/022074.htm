The links are to an outlook item in the user's Outlook for Windows data file. The link doesn't work on the Mac and there is no way to work around it. Sorry.

onenote:outlook?folder=Calendar&entryid=[big code] where the big code is the id of the item in the mailbox. Mac doesn't use the onenote: protocol.
