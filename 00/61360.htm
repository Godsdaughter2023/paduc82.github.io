stuart harrison - mvp say:

Hi

I assume you are using the native Movie Library in Windows 7 Media Center and DVD disc rips.

The native movie library should see the DVD movies that you have ripped to your hard drive after you have specified a watch folder.

It should download covers / metadata  for those movies via an online service called AMG. 

The folder location for these cached DVD covers is:

C:\Users\Username \AppData\Roaming\Microsoft\eHome\DvdCoverCache

There is also another folder where XML files are stored for each movie that contain the metadata information

C:\Users\Username \AppData\Roaming\Microsoft\eHome\DvdInfoCache

Take a look in these folders to see whats in there? You might have to un-check the option "Hide protected operating system files" in the Folder Options.

Movies should have their own folder in your file structure. For example:

D:\Movies\nameofmovie1\VIDEO_TS

D:\Movies\nameofmovie2\VIDEO_TS

D:\Movies\nameofmovie3\VIDEO_TS

In each movies folder "nameofmovie" there should be a cover image JPG file called folder.jpg

 

You might have better luck using a 3rd party tool, for obtaining meta data for your movie collection.

I personally use My Movies (A replacement movie library for media center) and their Collection Management program.

However if you want to stick with the Microsoft native movie library in Media Center you could use something likeYamm or Media Center Master for collecting correct metadata, which will make the movies appear correctly in the movie library. Neither of which I have used personally, but I hear good things about them both.

Good luck.
