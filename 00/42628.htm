5. Type following commands one-by-one and press Enter key after each:

BOOTREC /SCANOS

BOOTREC /FIXMBR

BOOTREC /FIXBOOT

BOOTREC /REBUILDBCD

6. Close Command Prompt and see if you can boot now.
