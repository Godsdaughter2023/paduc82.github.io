keith- say:

I see no option to keep apps, files, or settings when upgrading from Windows 7 to Windows 8

I’m trying to upgrade my Windows 7 PC to Windows 8.  I’m trying to upgrade my Windows 7 PC to Windows 8.  According to the “Upgrade to Windows 8” support website, I should be able to keep my apps, Windows settings, and personal files when I upgrade from Windows 7 using the Upgrade Assistant.  However, when I’m prompted to “Choose what to keep”, the only option I’m given is “Nothing”.
joe-c. say:

This problem will occur if you have moved the Users directory or the ProgramData directory to any drive other than the %systemdrive% in Windows.  The %systemdrive% (typically the C: drive) is defined as the disk drive that contains the Windows directory. 

 

To keep your personal files, you must back up your files manually and restore them after installing Windows 8.

 

You can find more information about relocation of the Users directory and the ProgramData directory at http://support.microsoft.com/kb/949977.

quynhdao say: Hi,

I have resolved the problem as follows :
1 - Correct the misspell of "IRPStackSize" : I wrote IRPStarckSize instead!! in Registry
2 - Create / modify the value of "IRPStackSize" to "25" in both XP and Win7 Starter Registry
4 - Reboot both computers.
5 - Add printer in Win7 starter , using "Add a local printer menu"
6 - Select "Create a new port". Choose "Local Port" as the type of the port.
7 - In the box "Enter a port name", type the address in the following format:
     \\my XP Pro SP3 name\my shared  HP C5280 printer name 
8 - Click Next
9 - Choose the correct driver in the driver list. 
       If no available driver, click the button Windows Update, wait for the process finishes and then look for the
       driver again.
10 - Finish the installation  by printing a test page.
