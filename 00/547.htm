Common Processes found in Task Manager:

System is a process in system. Every process started by Windows, except for the System process, runs in user mode which is limited in terms of what system resources they have access to. The User mode processes must request use of the kernel by means of a system call in order to perform privileged operations on their behalf. Kernel mode has unrestricted access to system resources and controls scheduling, thread prioritization, interrupt handlers, memory management and the interaction with hardware. The system process cannot be terminated.

System Idle process is used for measuring how much 'idle' time the CPU is having at any particular time (100% minus the sum of all tasks CPU usage). It accounts for processor time when the system is not processing other threads and will display how much CPU resources, as a percentage are 'idle' and available for use. One instance of this process operates per CPU, and runs to occupy the processor when other threads are not running. System Idle process also issues HLT commands which put unused parts of the CPU into a suspend mode, thereby cooling the processor. Normally this process should take up at least 90%+ of processor time on average (this is the value in the CPU column). In non-technical terms, this figure represents how much CPU time has not been requested by anything else on your system.

suzanne s. barnhill say:

My guess is that these are non-breaking hyphens (Ctrl+Shift+Hyphen). When nonprinting characters are displayed (see http://word.mvps.org/FAQs/Formatting/NonPrintChars.htm), the non-breaking hyphen is displayed much like an en dash. It does not respond to Alt+X, and on my system, if I select it and go to the Symbol dialog the 7/8 character is selected.

This is not too surprising: these nonprinting characters are just artifacts, and in this case the character that is being represented is not (apparently) a Unicode character but just a function in Word. You can, however, search for these characters using Find. The code for a non-breaking hyphen is ^~ (you can insert that using the Special list in the Find dialog).
