andre for directly say:
Check for an outdated webcam driver

Select the Start button, type Device Manager, and then select Device Manager from the search results.
Find your webcam under Imaging devices or Sound, video and game controllers.
Press and hold (or right-click) the name of your webcam, and then select Properties.
Select the Driver tab, select the Driver Details button, and look for a file name that includes stream.sys. If it’s there, your webcam was designed before Windows 7 and you'll have to replace it with a newer webcam.
If you don't find a file name that includes stream.sys, try rolling back your webcam driver.

Roll back your webcam driver

In Device Manager, press and hold (or right-click) your webcam, and then select Properties.
Select the Driver tab, select Roll Back Driver, and then choose Yes. (Note that some drivers don’t provide a rollback option.)
After the rollback is complete, restart your PC, and then try opening the Camera app again.
If rolling back didn’t work or isn’t available, uninstall your webcam driver and scan for hardware changes.

Uninstall your webcam driver and scan for hardware changes

In Device Manager, press and hold (or right-click) your webcam, and then select Properties.
Select the Driver tab, select Uninstall > Delete the driver software for this device, and then select OK.
In Device Manager, on the Action menu, select Scan for hardware changes. Wait for it to scan and reinstall updated drivers, restart your PC, and then try opening the Camera app again.
If your webcam still doesn’t work, try looking for help on the webcam manufacturer’s website.

You do understand that "upgrade" is just a marketing term, right? If you let yourself be lead around by companies' marketing departments you're in for lots of frustration.

I don't know whether, in fact, your processor is not supported on Windows 10, but I can tell you what 'not supported' means:

Each time an operating system gets updated - any operating system - hardware manufacturers need to test their products to make sure that they still work with the operating system. If the hardware no longer works with the updated version of the OS, hardware manufacturers have to decide whether they will modify their products so that they'll continue to work with the new OS.

That doesn't mean that hardware manufacturers take out old parts and put in new parts - it means that they have to update their drivers. A driver is the software that a hardware manufacturer installs in the operating system, so that the hardware device and the operating system will work together.

The initial driver for a hardware device was written to work with the operating system at a point in time. If the operating system subsequently changes ('gets updated') the driver may have to change too, so it will still work properly with the updated operating system.

If a hardware manufacturer says they will support Windows 10, that means every time Windows 10 changes, the manufacturer will test their driver on the new version of Windows 10 and if need be they will update their driver, re-test, update again if needed, re-test, then document the changes and update their support staff.

All that costs money, which the manufacturer is willing to spend for as long as the hardware keeps bringing in the money. Once the cashflow dries up, the manufacturer will likely stop support.

When a hardware device loses support that doesn't mean it will stop working. It means that the manufacturer doesn't know if it will work with subsequent updates to the operating system because they aren't testing it. For the same reason, the manufacturer won't help you if you run into trouble - they don't know what to tell you.

For its part, the hardware device may keep on working perfectly well for months or years, regardless of how much the operating system has changed over time. Then, one day, an update to the operating system comes along that the hardware driver can't work with, and that's where the trouble starts.

Maybe the driver only needs a small adjustment, or maybe it needs a total re-write. You'll never know, because the manufacturer has chosen to not support that device any longer.
