View, add or delete background tasks on System Tray on task bar - Microsoft Community

carlstaley say:

View, add or delete background tasks on System Tray on task bar

When I install some applications they but background tasks on the System Tray on the task bar that initiate on Startup.  How do I view, add or delete them?
lemp say:
Generally, the best way to manage applications that begin when Windows starts is to use each such application's own "options" or "preferences" menu rather than directly modifying Windows registry entries.

Also, keep in mind that not every task that starts up when Windows starts is actually using much in the way of your computer's resources.  Many tasks have a burst of activity right at start-up and are then paged out to the paging file and just sit there waiting not using any CPU cycles or RAM.

Notwithstanding the above, there are sometimes when you need to modify a registry entry for an application that starts with Windows.  The best tool for this is Autoruns.  Because it shows (and lets you change) start-up information for far more tasks that start when Windows starts than tools like the one you mentioned, Autoruns can be both intimidating and dangerous.  If you stick to making changes in entries shown in the "Logon" tab, you'll be OK.  Don't make changes elsewhere unless you're sure of what you're doing.

By the way, Spybot Search & Destroy used to be a recommended anti-malware tool.  These days, better choices are Malwarebytes Anti-Malware and SUPERAntiSpyware.

ren C:\Windows\System32\catroot2 Catroot2.old

net start cryptSvc

net start bits

net start msiserver

e) Type Exit in the Command Prompt to close it.
