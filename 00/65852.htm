Method 1:  System File checker (SFC Scan).

Follow the steps to run the System File Checker

  1. On the Search bar type “cmd” and click ctrl + shift + Enter to open the Elevated Command Prompt. (If      prompted for a password, type the password and click Allow)
  2. Type “sfc /scannow” and hit     Enter.
  3. Exit Command Prompt.

For more information about SFC Scan click on the link below: https://support.microsoft.com/en-us/kb/929833

Method 2: DISM TOOL

  1. On the Search bar type “cmd” and click ctrl + shift + Enter to open the Elevated Command Prompt. (If      prompted for a password, type the password and click Allow)
  2. Type Dism /Online /Cleanup-Image /ScanHealth and click Enter.
  3. Type Dism /Online /Cleanup-Image /CheckHealth and click Enter. 
  4. Type Dism /Online /Cleanup-Image /RestoreHealth and click Enter.

For more information about DISM click on the link below: https://support.microsoft.com/en-us/kb/947821

Hope this helps. Please reply if you have further related queries.
