Hello Scooterfuller,

Thank you for posting.  I have a very straight forward answer for you.  all you have to do is create the folder that you want everything from the usb drives to go into, then copy by drag and drop method.  I've included the steps below on how to do this.

Moving and copying files by using the drag-and-drop method

 

The most common way to copy and move files and folders is by using the drag-and-drop method, which means that you select one or more files and drag them to another location. For example, you can drag a file to the Recycle Bin to delete the file, or drag a file to a folder to copy or move it to that location.

How to drag files

 

When you drag a file or folder, exactly what happens depends upon what you drag and where you drag it. Pop-up windows containing information appear when you drag a file over a destination—pay attention to these to find out what will happen when you release the mouse button. You can also refer to the following table to learn more about what happens when you drag files and folders.

Tip

  • If you press and hold the right mouse button while dragging, a menu appears that lets you choose a specific action, such as copying or moving.

Dragging files

Into a folder on same hard disk

The file is moved to the destination folder.

Into a folder on a different hard disk

The file is copied to the folder on the destination disk.

Into the Navigation pane (left pane) of a folder

Nothing. You can't add files to the Navigation pane.

Dragging folders

Into a folder on same hard disk

The folder is moved to the destination folder.

Into a folder on a different hard disk

The folder is copied to the folder on the destination disk.

Into the Navigation pane of a folder

A link to the folder is added to the Navigation pane, and the link is then available from the Navigation pane of every folder

aileen alf say:

Hi,

The Windows Store Applications in Windows 10 are installed in the WindowsApps folder located in the C:\Program Files folder. By default, almost all system files, system folders and even registry keys in Windows 10 are owned by a special built-in user account called TrustedInstaller. You might need to take ownership of the files and folders in order to access these files and make modifications.

Here's how:

  1. Go to C:\Program files using the File Explorer.
  2. Change the File Explorer settings temporarily to show hidden files and folders.
  3. Select the WindowsApps directory. Right-click on it and select Properties.
  4. Go to the Security tab and click on Advanced.
  5. The window Advanced Security Settings for WindowsApps is now visible.
  6. Click on the Change link next to the Owner label.
  7. The window Select user or group is now available.
  8. Type your user account in the area Enter the object name to select and  click OK.
  9. Set full access to the app folder for your account. Right-click on the WindowsApps folder > Properties > Security > Add.
  10. The window Permission entry for data is now visible.
  11. Click on Select a principal, then add your account.
  12. Set permissions to Full control.
  13. Run PowerShell as an admin and re-register all of the packages by typing the command: Get-AppXPackage | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

Let us know how it turns out.

If you think this was useful, you may "mark it as an answer" to help those who are facing the same problem.
