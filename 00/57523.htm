Original title: Trying to run backup.

afzal taher say:

Hi,

This problem occurs because the backup process detects that system drive is dirty.
Windows Vista Backup always includes the system drive as part of backup. If the system drive is dirty, the backup process skips the system drive,
and no snapshot of the system drive is created. Therefore, the backup cannot locate a snapshot of the system drive later, and the error occurs.

Refer this KB article and follow the steps to resolve the issue:

Error message when you try to perform a system backup operation on a Windows Vista-based computer: "Catastrophic failure (0x8000FFFF)":
http://support.microsoft.com/kb/973558
