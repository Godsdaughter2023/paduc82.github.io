amit tandon say:

A solution using Excel formulas:

If you wish to determine the 100th non-blank value within a column (say column B), then you may use either of below formulas - enter as array formula (CTRL+SHIFT+ENTER):

=INDEX($A$8:$A$1002,SMALL(IF(1-($B$8:$B$1002=""),ROW($B$8:$B$1002)-ROW($B$8)+1),100))

or

=IFERROR(INDEX($A$8:$A$1002,SMALL(IF(1-($B$8:$B$1002=""),ROW($B$8:$B$1002)-ROW($B$8)+1),100)),"")

If you wish to determine the 100th non-blank value within the range B8:E1002 wherein non-blank cells will be counted across each row (in all 4 columns simultaneously), then you may use either of below formulas - enter as array formula (CTRL+SHIFT+ENTER):

=INDEX($A$8:$A$1002,SMALL(IF(1-($B$8:$E$1002=""),ROW($B$8:$E$1002)-ROW($B$8)+1),100))

or

=IFERROR(INDEX($A$8:$A$1002,SMALL(IF(1-($B$8:$E$1002=""),ROW($B$8:$E$1002)-ROW($B$8)+1),100)),"")

Pls note that the above formulas will not determine the 100th non-blank value in case you want to count column wise - first non-blank values in column B, then non-blank values in column C, and so on.
