Hello,

If you removed all apps and lost the Windows Store app as well, Powershell has the ability to retrieve it. For us to assist you with this concern, kindly follow the steps below:

  1. Right click the Windows icon and run PowerShell as Administrator.
  2. Type the following command on the Powershell and press Enter:
    • Get-Appxpackage -Allusers
  3. On the list, locate the Microsoft.WindowsStore entry and look for the text PackageFileName.
  4. Run this PowerShell command and replace the ****** with the PackageFileName from the list:
    • Add-AppxPackage -register "C:\Program Files\WindowsApps\******\AppxManifest.xml" -DisableDevelopmentMode
  5. After pressing Enter, this will reinstall Windows Store. Close your Powershell and restart your computer to check.

¡firedog say:

You can't rename (or move or remove) the system folders that are automatically created under a mail account and under Storage folders. These folders (Inbox, Drafts, Sent items, Junk email and Deleted items) are required by the program for proper functionality.

You can give user-created folders any name you like, but when the equivalent folder is created in the file system, its name will be truncated to 15 characters. This can cause problems with some backup software, so it's best to restrict folder names to eleven characters or fewer.

User-created folders will be arranged alphabetically below the system folders after a program re-start. If you want them in a particular order, start their names with, say, (1), (2) and so on. You can nest folders several levels deep, so it's easy to avoid having to use long folder names by creating sub-folders instead.
