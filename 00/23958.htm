AutoText entries are listed in the dialog with AutoText in the Gallery column.

To set the GAL as the default Address Book: From the main page of Outlook, (...we have it from Office 2013), click on Address Book, then the Tools menu -> Options. Click on the Custom radiobutton. Now click on the Add button, choose the Global Address List, click Add. Move it to the to top of the list. Voila! You're now using the GAL as the first data set in order.

gilguzman7180 say:

Think of the Global Address List as your company's main phone book.  It gets all the updates immediately.  The Offline Address List gets a copy (sync) from the GAL after a set period in time, usually 24-48 hours.

Any changes you make get updated in the GAL.  Give time and it'll show up in the OAL or you can force a sync, but it's best to just allow the GAL to sync on it's own.
