An actual "clean install", where you begin by deleting all disk partitions until there is only unallocated disk space, is a good idea from time to time.  The next public release will be version 1803, which stands for March 2018, so it will be available this month or early April.  I would recommend that you consider doing a clean install of that version, if you can wait a few weeks as you are.  If you have room to relocate your existing data to another partition on the hard disk, you could do a clean install to the first part of the disk and then copy your data back in.

If your computer on the Activation page shows that it is activated with a digital license, you do not need any Product Key to do a clean install.  The hardware itself has been registered as licensed and will be recognized and Windows Activated once online.  In fact, during a clean install, it specifically says not to enter any Product Key if you are reinstalling Windows 10.  Most upgraded Windows 10 have the same generic Product Key per Edition (Home has one, Pro has one, etc.) but you don't need to enter it in during a clean install.  Also, if a PC shipped with Windows 8, it should have the Windows 8 Product Key embedded in firmware and it can be looked up (but is rarely needed for anything other than reinstalling Windows 8).

A second install of Windows does not take long compared to an upgrade, so if you want to prove to yourself that it will auto-activate, you only need a minimum of 20GB disk space for Windows 10 64-bit.  You can shrink C: by that much and install Windows 10 there just to prove that it activates without a product key.

There is also a way to save the current activation status to a small XML file, by running gatherosstate.exe from the install media.  But that is usually not necessary.

4.      Try installing Office again.

5.      Once Office is installed on your PC, go back to the list of services and open Print Spooler.

6.      Tap or click Start, and then tap or click OK.

http://support.microsoft.com/kb/929135

Note: Ensure to restart the computer in normal mode once troubleshooting is completed.

 

Clean boot is performed to disable all startup programs and 3rd party services on the machine and check if they are conflicting with the Office program. If so, you may disable the unwanted software those conflict with the suite or get them updated.
