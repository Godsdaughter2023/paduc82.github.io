karl timmermans [outlook mvp 2012-2018] say:

Just to clarify Roady's response - there is no option to "selectively" export contacts but there would be no need using the suggestion of category assignment and printing of the resulting filtered view.

The only way to create an external file of "selected contacts only" would be to create and then "copy 'n paste" the selected items from the view into an Excel worksheet. Also, note that the Street Address is always contained in a single field and multi-line street addresses would "appear" truncated unless the cells were expanded to show the entire value of any given field.

Multi-line street addresses will appear as a single line in any <table> view and would need to be sized accordingly prior to printing to encompasses the largest street address to ensure that no information is truncated when printing the filtered view. Would also suggest printing in "landscape mode" since it is unlikely that all selected fields would appear on a single line/page if expanded.

stefan blom say:
I believe you are referring to the "Set up your Heading paragraph styles" section in the article. It refers to the "Style based on" setting in the Modify Style dialog box and it is different from the links between style names and numbering levels in the Multilevel List dialog box.

Also, in the Modify Style dialog box, make sure that "Style for following paragraph" is set to "Heading 1" for the Heading 2, that it is set to "Heading 2" for Heading 2 paragraphs, etc. This controls which style is applied to the new paragraph inserted as you press Enter (at the end of the paragraph).

As far as style sets are concerned, I would avoid them in conjunction with numbered paragraph styles. Just apply the styles directly.

Once you have the numbering set up in a document, click File | Save As and change the file format to template. Later, when you want to create a document using that particular numbering, create a new document based on the template.

dstone001 say:

Thanks. you.  Can you explain what "Style Set" is?  I realized that I don't know.  

And is there any way I can change the what styles appear on the Home Tab when I open a new document?  

stefan blom say:

The current style set is just the collection of styles you can see in the Styles group on the Home tab. Applying a different style set (via the Design tab) replaces the styles in the Styles group and reformats text using those styles in the document.

You can add styles to the current style set by selecting the "Add to Styles gallery" option in the Modify Style dialog box.

However, as I wrote, style sets are not very useful with numbered paragraphs. I would even call them buggy, as numbering may be unpredictable when using them.
