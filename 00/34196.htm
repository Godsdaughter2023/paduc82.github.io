Hi jderpiz,

Movie Maker usually compresses video to fit on DVDs, but depending on the size of the video, it may be too large to compress to a DVD.

Can you please specify what DVD are you using? A standard DVD's size is 4.7GB. You can try to save the movie first to your computer, and burn them later to a DVD. Please follow these steps:

1. After editing the video on Movie Maker, click File, then select Save movie.
2. Scroll down and select Custom setting.
3. Select For computer, then change the Bit rate to a lower one (you can try 2000 kbps).
4. Change the name to Custom, then click Save, then save the setting to your computer.
5. Go again to File > Save movie, and then select Custom.
6. Save it to your computer.
7. After saving, click Open folder. Right-click the file, then select Properties.
8. If the size is less than 4.7 GB, proceed with the next step. If it is still big, create a custom setting again, and try lowering the Bit rate and Audio settings.
9. Open Windows DVD Maker, and add the video. Follow the on-screen instructions to burn it.

Please take note that using a lower bit rate will reduce the quality of the video.
