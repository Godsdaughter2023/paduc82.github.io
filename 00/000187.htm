a. user say:

If it's a one-off, or the structure of your Excel workbook is going to change over time, it's probably simplest to copy the data into a single sheet and use that as the data source.

Another way, for a small, fixed number of sheets with the same column count and column names, might be to use Word VBA to specify a UNION query, e.g.

Sub MultiSheetConnect()

    ActiveDocument.MailMerge.MainDocumentType = wdNotAMergeDocument

    ActiveDocument.MailMerge.MainDocumentType = wdCatalog

    ActiveDocument.MailMerge.OpenDataSource Name:= _

        "C:\mywb\union.xlsx", _

        SQLStatement:="SELECT * FROM [Sheet1$] UNION SELECT * FROM [Sheet2$] UNION SELECT * FROM [Sheet3$]"

End Sub


I've never used it except in simple cases so there may well be limitations. It has to be a small number of sheets because the total length of the SQL is at most around 512 characters and possibly no more than 255.

Another way you could achieve much the same thing if you also have Access is to created a linked table for each sheet, create a UNION query (as in the above example) that unions all the tables. However, Word can't see Union queries in Access so you have to wrap that query in a simple query such as
SELECT * FROM myunionquery.

We were looking at the Additional Actions Bitmask for actions that might allow the remediation to complete, presuming that it hasn’t. In addition to the Action Success, I would be looking at the Threat Status ID:

 

ThreatStatusID

Data type: uint8

Access type: Read-only

The Threat Status ID - Enumeration

Unknown (0)

Detected (1)

Cleaned (2)

Quarantined (3)

Removed (4)

Allowed (5)

Blocked (6)

CleanFailed (Blocked)

QuarantineFailed (102)

RemoveFailed (103)

AllowFailed (104)

Abondoned (105)

BlockedFailed (107)

 

But I’d be concentrating on Additional Actions with Defender and “Additional Actions” with third-party malware-removal apps if there’s even a chance that you’re looking at a remediation failure. And of course you should make sure that Cloud-Delivered Protection and Automatic Sample Submission are turned on and connected:
