Equation editor: how to obtain a small fraction by keyboard - Microsoft Community

marten jan de ruiter say:

Equation editor: how to obtain a small fraction by keyboard

A skewed fraction a/b can be entered in Word by pressing <Alt><=> to enter the equation editor mode, and entering "a\sdiv b ", where the spaces after \sdiv and b trigger the equation editor to process the formatting.

Similarly, "a/b " produces a stacked fraction and "a\ldiv b " produces a linear fraction.

Small fractions are available the Equation tools menu. Can these also be entered using a keyword?

Unicode Technical Note 28, "Unicode Nearly Plain-Text Encoding of Mathematics Version 3" by Murray Sargent III suggests the "\ndiv" keyword on page 5, but that does not work in Word 2010.

marten jan de ruiter say: Reply to bump

marsf say:

I found that "\ndiv" sometimes does something and sometimes doesn't. But whenever it did work, it acted the same as standard division. But I found how to type the small fraction a different way. I converted the small fraction added from the Equation menu to linear format and saw it can be entered the following way:

"\box(64&x/y)".

Replacing "x/y" with the actual fraction you want to use. See section 3.7 in "Unicode Nearly Plain-Text Encoding of Mathematics Version 3" (http://www.unicode.org/notes/tn28/UTN28-PlainTextMath-v3.pdf) to understand the \box option more. Basically the "\box" 64 option applies script size text to the expression "x/y".

As you've discovered, changing the sampling rate fixes it, though not the same sampling rate works on every config (Conexant seems to like 192 KHz, for example). Outlook will not open error: Cannot start Microsoft Outlook. - Microsoft Community

diane poremsky mvp (slipstick.com) say:

Outlook will not open error: Cannot start Microsoft Outlook. Cannot open the Outlook window.

You receive this Outlook error message: "Cannot start Microsoft Outlook. Cannot open the Outlook window. The set of folders cannot be opened."

This is typically caused by one of the following issues:

  1. Your data file or profile is corrupt. 
  2. Another application or a COM Addin, such as a sync utility or Skype, is accessing Outlook’s data file.  

  3. The navigation pane is corrupt. 

See I can’t start Microsoft Outlook or receive the error “Cannot start Microsoft Office Outlook. Cannot open the Outlook Window” for the troubleshooting steps and solutions.

Office 365 business subscribers can use the Support and Recovery Assistant to diagnose the problem. 

A less common cause, affecting Office 365 business accounts, is having two (or more) Microsoft Exchange accounts in your profile when one account has delegate or full access permissions on the other account. Remove the delegated account from your profile to solve.

Here is how I fixed it.

1. Control Panel -> Mail -> Email Tab

2. Delete all the Microsoft Exchange / Office 365 / Hotmail / Outlook.com email accounts.

3. Manually go to the location (this can be found in the Data Files Tab) of the OSTs and delete them.

4. Open Outlook (It should open now)

In my case, the issue was due to the OST getting corrupt and Outlook not having the correct privileges to work in them.
