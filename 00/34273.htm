You can use the storage sensor to remove unnecessary or temporary files from the device.

(Can not find the storage sensor on the device with the steps below, in that case, see the section Releasing space with the Disk Cleanup section below).

Select the Start button, and then select Settings> System> Storage.
Under Storage Sensor, select Free space now.
Windows will take a few minutes to determine which files and applications consume the most space on your PC.
Select all the items you want to delete, and then Remove files.
Note

The Recycle Bin and some other configuration options are not selected by default. You may have to scroll down to select these items.

Select the Recycle Bin to free up space occupied by previously deleted files. Windows moves the files that you delete to the Recycle Bin in case you change your mind and need them again in the future. You may be able to free a significant amount of free space by selecting Recycle Bin to remove the files that you deleted before permanently.


If you activate the storage sensor, Windows will automatically free up space by deleting files that you no longer need, such as temporary files and items from the Recycle Bin.

Select the Start button, and then select Settings> System> Storage.
Active Storage sensor.
To choose which files you want the Storage Sensor to remove automatically, select Change the way to automatically free space, and then check or uncheck the options for Temporary files, Locally available cloud content, and Free space now.

If the system does not have a storage sensor, you can use the Disk Cleanup tool to remove temporary files and system files from your device.

Select the Start button, type disk cleanup, and then select Disk Cleanup in the list of results.
Check the box next to the type of files you want to delete. (To see a description of each type of file, select the name). By default, the options for Downloaded Program Files, Temporary Internet Files, and Thumbnails are selected.
To free up even more space, select Clean system files, and then select the type of system files you want to delete.

Note

If less than ten days have passed since you upgraded to Windows 10, the previous version of Windows will be displayed as a system file that you can delete. If you have to free storage space, you can safely delete it, but keep in mind that you are going to delete the Windows.old folder, which contains files that offer you the option of going back to the previous version of Windows. The removal of the previous version of Windows can not be undone.



After selecting the file types you want to delete, select OK, and then select Delete files in the confirmation window to start the cleanup.
