    john korchok say:

    This is a different problem than what you originally posted. 

    When you create a new document, that documents takes its styles from your Normal.dotm template. When you have an older document that has been edited over time, it can have a different set of styles.

    When you apply bullets from the ribbon, that is not part of the style, it is local formatting and can easily be over-ridden by text pasted that has a different style.

    Then when you paste, your document styles over ride the styles in the new document. You can eliminate this problem by creating a template that has the correct styling, like a Heading 2 style with a bullet. You can apply that to existing documents and you can create new documents from it, and all will have identical styling. Then pasting is a non-issue, as none of the formatting changes.

    If you need help implementing a template with styles, please post again and I'll cover the steps.

    1. Go to Search programs and files by clicking Start button.

    2. Type in Services. You can select Computer Services or Services (the one with cogwheels icon) directly.

    3. Locate Windows Search.

    6. Under General tab, you will be able to choose the Startup Type. Make sure that it's set to Automatic.
