Giving oneself control is a two step process. Taking ownership and then giving oneself permissions.

 

Ownership of a file allows the owner to change permissions even if they don't have permission to do so. This is the only thing that ownership does. The person that creates an object owns the object. For admins, objects created are owned by the Administrators' group not the individual.

davidmcandy say:

Autochk is unusual is that it is not a Windows program.

 

There is the Windows NT operating system. It runs other operating systems. Windows 32 and 64 bit, Unix, and in earlier versions OS/2. (MSDos and Windows 16 bit are run in a Windows 32 bit program that emulates a 16 bit environment).

 

Operating systems do two things - control the hardware/provide services (Windows NT) and provide a programming interface (Windows 32/64 bit).

 

Autocheck is one of the rare NT Native Mode programs (because it runs before Windows starts), the other main one is the program that actually provides Windows (CSRSS.exe - Client Server Runtime Sub System).

 

Your AV program may be confused by a non windows program.

- Apparently running the Steam.exe as admin helps some people. Don't know why though.

Running the Steam client as admin can cause other issues.

- What graphics processor does your machine use ?

- If using a laptop with an Nvidia graphics, the game does not support  Nvidia Optimus.
