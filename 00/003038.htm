go to the print contacts button on the right - click on it

select the print button on the bottom

Column A   Column B   column C

Put the Last name only in Column B

desiree yau say:

Hello,

If the sync feature of your device is on, the Outlook app will keep on syncing so long as it's connected to either your WiFi or mobile data which could be the reason why it's using your mobile data allowance. Sometimes, even if you have connected your mobile device to your WiFi if the mobile data is on, it will still use some of your data allowances. That being said, we advise turning off the mobile data when you connect your phone to your WiFi to avoid using your data. In line with this, you can also turn off the sync feature on your phone if you are not using the Outlook app. It will not just save your mobile data but will also save your battery.

In case you have performed the things that we have suggested, we can do some troubleshooting steps to isolate what's causing the issue. However, before we proceed, we'd like to know more about this concern. Kindly answer the following questions:

  • Is this the first time that you encountered this concern?

  • Which server setting are you using? Is it IMAP, SMTP, or POP? To determine which server setting you are using, kindly check this article.

In addition, if there's a feature on your phone where you can set a data limit in each of your apps, please do so. Doing so the app will only use limited data.

Reply to us with the information so that we can assist you better.
