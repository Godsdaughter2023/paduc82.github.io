Format based on cell contents as formula or number.

I would like to format some cells based on whether they contain a formula or a value/number.  Thanks for any advice.

lars-Åke aspelin say:

I would like to format some cells based on whether they contain a formula or a value/number.  Thanks for any advice.

If you enter the following UDF (User Defined Formula) to a module of your workbook it can be used as a conditional formatting formula:

Function hasformula(c As Range) As Boolean
  hasformula = c.hasformula
End Function

Alt + F11, select the Project/Workbook, Insert Module (if you dont already have one) and enter the code above inte the code window.

Test the formula by writing =HASFORMULA(B4) in any cell. If B4 has a formula the result should be TRUE, if not it should be FALSE.

When the HASFORMULA UDF is working you can use it for conditional formatting like this:

Select the cell, e.g. B4, that you want conditionally formatted and click on Conditional Formatting in the Styles section of the Home tab (if it is on the same place in Excel 2010 as in Excel 2007).

Click on New Rule... and then Use a formula to determine which cells to format

Then in the text box Format values where this formula is true, enter the following formula:

=HASFORMULA(B4)

B4 should be the address of the cell that you originally selected.

Click on Format... and choose a format you want to have when the cells contains a formula.

OK yourself out.

Hope this helps / Lars-Åke

Use the following macro to paint the cells in red (existing values) or yellow (existing formulas) :

Sub ValueOrFormula()

Dim c As Range

For Each c In ActiveSheet.Cells.SpecialCells(xlCellTypeFormulas)

    c.Interior.ColorIndex = 3

Next c

For Each c In ActiveSheet.Cells.SpecialCells(xlCellTypeConstants)

    c.Interior.ColorIndex = 6

Next c

End Sub

Paste the following code in the sheet module to color the cells when you enter values or formulas.

Private Sub Worksheet_Change(ByVal Target As Range)

Dim c As Range

For Each c In Target.SpecialCells(xlCellTypeFormulas)

    c.Interior.ColorIndex = 3

Next c

For Each c In Target.SpecialCells(xlCellTypeConstants)

    c.Interior.ColorIndex = 6

Next c

For Each c In Target

    c.Interior.ColorIndex = xlNone

Next c

End Sub

suzanne s. barnhill say:

There are various ways to do this, but probably the examples you saw used MacroButton NoMacro fields, as described at http://word.mvps.org/FAQs/TblsFldsFms/UsingMacroButton.htm or (in more detail) in the second section of http://word.mvps.org/FAQs/Customization/FillinTheBlanks.htm.

While the MacroButton field is intended to run a macro, it has the useful characteristic that it is designed to show a prompt (which can be any kind of text, an underline, or even a picture) and that when you click on it, the entire field is selected, so that anything you type replaces the field.

Note that, unlike a form field, a MacroButton field is not reusable (when you type in it, you replace the field), so you definitely do need to use it in a template, not a document.
