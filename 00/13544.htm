petrib say: Packet data and access points do not have anything to do with Bluetooth, really.

A packet data connection is a data connection over the mobile network (GPRS, EDGE or WCDMA/UMTS/3G data connection).

An access point is the mechanism how the connection "context" is established.

The easiest way to define/set up an access point (or several; usually network operators offer three: one for Internet access, another for WAP and a third for MMS, Multimedia Messaging Service) is to request/order a configuration message with the settings to be sent to our phone.

You can usually do so, by calling your operator's customer service (helpdesk) and ask them to send you the settings, or you can do it from their web site, or Nokia's web site.

See "Get settings" here: http://europe.nokia.com/A4144888

WHY DOES MICROSOFT WORD PERIODICALLY ERASE ALL MY MACROS??

rohn007 say:

By default macros are saved in NORMAL.DOTM unless you explicitly save them in the explicitly formatted DOTM document.

MS issued an Office update that "accidentally" lost macros by renaming the NORMAL.DOTM, forcing it to be recreated.

Since you are having this issue, you may want to get into the habit of backing up your NORMAL.DOTM file.  I used to use a batch file to start Word. It would copy my normal.dotm from a backup location, start word, then after word closed it would copy normal.dotm to the backup location.

Buttons in the QAT are stored separately, NOT in the NORMAL.DOTM. That is why the buttons remain, but the macros are missing.

2015 12 23- Yet more patch troubles for Office users - Macros Lost - Normal.DOTM

http://new.office-watch.com/2015/yet-more-patch-troubles-for-office-users/
