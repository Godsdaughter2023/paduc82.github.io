Saving custom shapes - Microsoft Community

gwnatl say:

Saving custom shapes

When using Word, I have several shapes that I use regularly. I have customized them to my liking. Is there a way to save them to a gallery like you can do in Visio
jay freedman say:

Yes, you can save shapes as Quick Parts (also called Building Blocks).

Click on one of the shapes to select it. Then go to the Insert tab, click the Quick Parts button, and click Save Selection to Quick Part Gallery. This dialog opens:

Enter a name for this shape (spaces and capitalization are allowed here). You can leave the Category box set to General, or you can open the dropdown, click Create New Category, and enter a descriptive category name. Open the Save In dropdown and choose the template where you want this shape to be stored (Building Blocks.dotx, Normal.dotm, or the base template for the current document if it isn't the Normal template). Click OK. Repeat for the other shapes, changing the name but using the same values for all the other settings.

When you want to insert one of these shapes, click the Quick Parts button and select the shape from the gallery.
