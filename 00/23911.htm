Just started using Outlook at work. Have lists of email addresses to add, probably 125 names. Can I put the addresses into Excel import to Outlook without having to create new contacts for each?
talking moose say:
Yes.

I suggest you open Outlook first and enter at least one example contact. Then select File menu --> Export... --> Contacts to a list (tab-delimited text) and save the new file.

Open this file in Excel and you'll see all the potential headers you can use. Even if your existing spreadsheet uses different headers you will still have the option to pair your file's headers with Outlook's headers so that information is imported correctly.

When you're ready to import select File menu --> Import... Contacts or messages in a text file and select your tab-delimited file.

From the issue description I understand that you are unable to access Internet Explorer 11, as it closes automatically.

-Are you referring to the Desktop Internet explorer or the Modern UI Internet explorer or any other web browser?

-Have you made any changes to your computer recently?

This issue may occur due to corrupted Internet Explorer settings or some browser add-ons.

Disable all add-ons and check if internet explorer works

While browser add-ons can enhance your online experience, they can occasionally interfere or conflict with other software on your computer. Try starting Internet Explorer without add-ons to see if the problem goes away.

Manage add-ons in Internet Explorer

http://windows.microsoft.com/en-us/internet-explorer/manage-add-ons#ie=ie-11

If disabling all add-ons solves the problem, you might want to use Manage Add-ons to disable all add-ons and then turn on add-ons only as you need them. This will allow you to figure out which add-on is causing the problem.

If disabling add-ons doesn't solve the problem, try resetting Internet Explorer back to its default settings. This removes all changes that have been made to Internet Explorer since it was installed, but it doesn't delete your favorites or feeds.

Reset Internet Explorer settings

http://windows.microsoft.com/en-us/internet-explorer/reset-ie-settings#ie=ie-11
