You can test the position of each shape on the sheet by testing whrere the top_Left is. If it's in column 1 (Or any other column) you can capture the row number and do thing on that row

 

 

Sub ShapesPostion()
Dim sShape As Shape
Dim ShpRow As Long
Dim ShpCol As Long
For Each sShape In ActiveSheet.Shapes
    ShpRow = sShape.TopLeftCell.Row
    ShpCol = sShape.TopLeftCell.Column
    If ShpCol = 1 Then
   
        'do things on row number ShpRow
      Stop
    End If
  Next
End Sub

How to reference a table cell text entry - Microsoft Community

chelseacat say:

How to reference a table cell text entry

I'm not sure if I've described this correctly, but I'd like to reference a text entry from one table cell in another one.  It's similar to what you'd do in an Excel spreadsheet if you were to type a text entry in one cell and then refer to that cell number to result in the same text value.  Is this possible in Word?  I've tried assigning the original cell entry as a bookmark and then pasting it into the new cell but the result is "0."  Would string functions work?

Thank you in advance for any responses!  I don't know if version number makes a difference but I have all releases of Word going back to 2003.

 

 

jay freedman say:
Table functions in Word are extremely limited compared to Excel. You might do better by embedding a section of an Excel worksheet in your Word document.

If you need to stay with Word alone, the rules are a bit strange:
  • Formula fields (described in http://office.microsoft.com/en-us/word-help/use-a-formula-in-a-word-table-HA102329800.aspx) work exclusively with cells that contain numbers. There are no string functions, unless you want to get involved with macros. If you refer to a cell whose contents can't be interpreted as a number, the value of the reference will be 0.
  • If you need to repeat a non-numeric string from one cell to another, you need to bookmark the first cell and put a REF field (containing the name of the bookmark) in the second field.
  • In both cases, the reference formula or field will have to be updated manually (by selecting the field or the whole table and pressing F9, or by going to Print Preview and back again) -- again, unless you use a macro. You might be able to get automatic updating if you use STYLEREF field (http://sbarnhill.mvps.org/WordFAQs/StyleRef.htm).
There aren't any changes in fields or functions from Word 2003 through Word 2010.
