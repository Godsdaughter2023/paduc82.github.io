If the data base has info in a field called Received_By return that value and use in the mail merge field.

doug robbins - mvp office apps & services (word) say:

Use an If...then...Else field construction

{ IF { MERGEFIELD Received_By } <> "" { MERGEFIELD Received_By } ""}

using Ctrl+F9 to insert each pair of field delimiters.

Similarly for the Prepared_by case.

macropod say:

Make two copies of your «Received_By» field and one of your «Prepared_By» field, thus:

«Received_By» «Prepared_By» «Received_By»

Select all three fields and press Ctrl-F9 to enclose them in a set of field braces, thus:

{ «Received_By» «Prepared_By» «Received_By» }

Finally, fill in/around the fields so you end up with:

{IF«Received_By»= "" «Prepared_By» «Received_By»}

then press F9 to update the field.

Your mailmerge is now ready to run.

Note: the indicated spaces in the final construction are all required.

In your example, it would appear that there are otherwise underlying, invisible, multiple, active processes within workbooks 1 and 3, thus they are included in the 'application window stack' multiple times. You have to Alt+Tab twice before you get away from the Excel application window as the Find dialog makes an additional entry in the enumerated 'stack'.
