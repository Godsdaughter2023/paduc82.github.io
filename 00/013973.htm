cody c. say:

Hey Ronaldo,

 

It's a little hard to tell exactly what issue you're running into. Here are the two most common font related issues:

 

  1. Increasing the font size of the Outlook for Mac user interface

    Aside from decreasing the screen resolution on your Mac's display (Choose the Apple Icon on the Menu bar > System Preferences > Displays >Resolutions), it's not currently possible to increase the font size of the Outlook interface. 


  2. Font size changes when sending from Outlook for Mac to Outlook for Windows

    Outlook for Mac uses Apple's open sourced WebKit based text editor. This editor displays font sizes in pixels. This is unlike Outlook for Windows or other Office applications that use Word as the text editor and displays font sizes in points. 

    While the font sizes differ, the change shouldn't be drastic.     At most the font size should only increase by 4 or 5 points. In most cases, the difference is a single font size. This is expected behavior when emailing between the two applications.
