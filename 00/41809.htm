vitaliyg say:

I might have found a solution... Had this exact issue on several machines trying to set up my children as family. Had a hunch Microsoft might program something like this in -- ghost accounts that get in a way. ;)

The solution I used is this.

First, you need to get to "Family". Log in as an adult set up with Microsoft account already, go to manage your account (I left click start button, right click on my account on upper left corner, and choose "change account settings")

You need to get to "Family and other users" -- and under your family, enable everyone to log in.

Then, go to control panel (right click on start button, choose control panel) and go to "User Accounts" (you might need to show all options on control panel to get there). DELETE all new accounts you just enabled (obviously, leave accounts your kids use already or the ones you converted -- just delete the new ones that popped up when you enabled log in via "Family").

Now you should be able to connect existing accounts with Microsoft accounts.

Worked for me like a charm, hopefully will work for you.

Method 1:

Run the Windows Memory Diagnostics Tool and check if it helps.

  1. Press Windows key + X.
  2. Click Control Panel.
  3. Open Administrative Tools (in "Large icons" view).
  4. Double-click to initiate the Windows Memory Diagnostics.
  5. Click on Restart now and check for problems (recommended).

Method 2: Clean Boot.

A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you install a program or an update or when you run a program in Windows 8.1, Windows 8, Windows 7, or Windows Vista. You may also troubleshoot or determine what conflict is causing the problem by performing a clean boot.

System File Checker (SFC) is a utility in Microsoft Windows that allows users to scan for and restore corruptions in Windows system files. Perform System File Check (SFC), and then check if this fixes the issue.
