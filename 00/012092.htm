Hi Steve

I'm not sure I've absorbed everything, but I can shed some light on what you're seeing...

Doug is correct about the bookmarks to which I was referring. You can see these in Insert/Bookmark if you activate/deactivate/activate the "Hidden bookmarks" checkbox.

Concerning the formatting of the cross references. If you press Alt+F9 you'll see that these are maintained by REF fields. If the field contains the \h switch, that means it's been created to be a hyperlink. Thing is, Word is designed to create print media, so generally, one would expect this text to not display the hyperlink formatting generated by the Hyperlink character style. Any other character style, yes.

Given what I understand, I'd try

1. Create the cross references without the hyperlink. There's a checkbox in the dialog box you can deactivate for new hyperlinks. For exising ones, with the field codes dispalyed, you can do a "Find" on [space]\h[space] and "Replace" with a space. (Just type a space where I have [space], don't type that literally.)

2. Create a style that makes text look like a hyperlink, don't use the built-in Hyperlink style.

3. Create the hyperlink back to a bookmark over the entire text (as you've tried to do). To remove the Hyperlink style, select the text, then press Ctrl+Spacebar. Now apply the new style.

At this point, everything should appear the same in the Hyperlink, the hyperlink should work no matter where the user clicks or hovers.

Does that take care of the first two problems?

For Problems 3 & 4, I recommend you start a new message thread, concentrating on that.
