ocryzen say:

Games on a PC (not all games), create more CPU interupts (CPU Load) & require you to install the games to the HDD, to keep the game running smoothly, by running it off the HDD & lowering the interupts + loading times of the Larger games etc..

But, in the case of Steam Networks, You actually download the games from their server (even though you have the disc), as it is part of their Piracy Prevention & it also puts a unique ID on the Data, so you have to have Steam's Program open to run the game...

Consoles Games & PC games, are programmed differently & PC games can require a Higher Polygon rate, than the consoles & games on a console, are limited to a certain level of performance.. So you can utilise the Optical Drive to run the game, as the consoles game is also optimised for the console to run with less CPU Interupts.. You will always get faster loading times, if you install the game to the HDD though..

icestorm iii say: Games on the 360 are designed to stream in content from the disc. At 12x, the data rate is about 15MB/sec, so it's not terrible as long as there's limited seeking. Careful arrangement of content on the disc mitigates the amount of seeking required.

There are games which can elect to use the HDD as cache space (Halo 3 and Oblivion do this). They typically exhibit longer loading times without a HDD.

a. user say:

The discs do not always run at 12x. Most often they will run at a slower speed, only rising to 12x when necessary to decrease loading time (usually when transitioning to a new area.)
