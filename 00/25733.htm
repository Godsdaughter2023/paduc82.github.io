Hi Dx,

 

You can go to the old admin center to check if the problematic account still exists.

 

If you ensure that the MA account permission is unnecessary in your organization, you can run the cmdlet below to remove the permission:

 

Remove-MailboxPermission -Identity Test1 -User Test2 -AccessRights FullAccess -InheritanceType All

 

This example removes user Test2's full access rights to Test1's mailbox.

 

If you want to delete permission on all mailboxes, you can try using this cmdlet:

Get-mailbox | Remove-MailboxPermission -user identity -AccessRights FullAccess -InheritanceType all

 

Then use Get-MailboxPermission -identity <alias> to check if all the full access permission has been removed.

vijay a. verma say:
1. For online purchases product keys are not provided. To download and install Office 365 or 2016, you may login to the My Account page on Office.com with the Microsoft account you used to register or purchase.

http://office.microsoft.com/myaccount

If you have purchased Office 365 or 2016 from a retail store (Product Key Package), then you may register the purchased product key with the Microsoft Account and then download and install Office. Refer to the following link to sign in or create a Microsoft account to download and install Office 365.

https://setup.office.com/

2. You don't require a product key for MS Office if you have installed and activated it. You require it just once, for the first time installation and activation.

Once you install MS Office 365 / 2016 through Microsoft site and activate it, a digital signature corresponding to your computer gets stored in Microsoft servers.

Now whenever you login to your Microsoft account, it will check whether you are trying to install on the same computer or not. It won't ask for the product key again.

Hence, go ahead and reinstall. Follow the Help below to reinstall -

https://support.office.com/en-us/article/Downlo...

In case, you want to install on a different computer, you will have to first deactivate Office license which was installed on old computer otherwise it will be counted as a license. Hence, if you were allowed installation on one computer, you will not be able to activate on new computer. If your license was for say 5 computer as in case of Office 365 Home, now your validity will be for 4 computers.

Hence, deactivate first and then install and activate.

Deactivate your existing installation - https://support.office.com/en-us/article/Deacti...
