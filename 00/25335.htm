stefan blom say:

In Word, line spacing is measured between text baselines, which means that, for the text to be readable, line spacing can't be so much smaller than the point size. I'm referring here to a line spacing set to "Exactly." If you choose "Multiple," you shouldn't choose a value much smaller than 1 line.

doug robbins - mvp office apps & services (word) say:

Single line spacing is normally provides the optimal line height to accommodate the font that is selected.  While you may be able to reduce it slightly, parts of some of the characters missing is the expected result.  The only option is to use a smaller or narrower font and\or to condense the spacing between the characters via the Format>Font>Special dialog.

If you condense the font spacing, you will have to expect some degradation in the appearance of your document.

how to jump to a specific cell in excel 2003 - Microsoft Community

muhammad faizan attari say:

how to jump to a specific cell in excel 2003

how to jump to a specific cell in excel 2003

I am making my own invoice template for my small business. I need to Jump to a specific cell upon Enter.

bernie deitrick say: You can control which cells are selected after entering by using the change event - let's say that after you enter date in cell B3, you want to jump to F7. Copy this code, right-click the sheet tab, select "View Code" and paste the code into the window that appears.

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Address = "$B$3" Then Range("F7").Select
End Sub

Maybe you want to cycle through several cells. . . . . .

 

 Private Sub Worksheet_Change(ByVal Target As Range)
     Select Case Target.Address
        Case "$C$2"
             Range("C5").Select
        Case "$C$5"
             Range("E2").Select
        Case "$E$2"
             Range("E5").Select
     End Select
 End Sub
