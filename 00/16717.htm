jan karel pieterse say:

Assuming your columns start in K1...

If you use this function you will get the column number (start counting in column K) which has today's month:

=MATCH(TODAY(),$K$1:$AA$1,1)

Since you need the 3 months prior to that, this should be subtracted by 4 to get the starting column. The Offset function then can give the three month's data:

=AVERAGE(OFFSET(K2,0,MATCH(TODAY(),$K$1:$AA$1,1)-4,1,3))

ronrosenfeld say:

Instead of using AVERAGE, just use SUM and divide it by 3.

In other words, replace Jan's solution with SUM instead of AVERAGE, and enter it as an array formula.

eg.

=SUM(OFFSET(Q$1,ROW()-1,0,1,3))/3

The only issue is how to compute the starting column for the range, and that depends on the real contents of the labels row.

If the real contents is a date, formatted to look as you show, then:

=SUM(OFFSET($O$1,ROW()-1,MATCH(TODAY(),$O$1:$AO$1)-4,1,3))/3

could be entered in AP21, and fill down.  The formula is an **array-formula** and must be entered by holding down <ctrl+shift> while you hit <enter>.  If you do this correctly, excel will place braces {...} around the formula.
