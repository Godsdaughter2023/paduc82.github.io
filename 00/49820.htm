Hi

The only way that I'm aware of to remove the Device Manager from the Computer Management console is by using the Group Policy.

Group Policy is only available in Windows 8.1 Pro. If you are using the Pro edition, perform the following.

1. Close Computer Management. Press the WinKey+R to display the Run dialog.

2. Type  gpedit.msc  and click OK. This will open the Group Policy Editor.

3. Navigate to the following location.

 User Configuration\Administrative Templates\Windows Components\Microsoft Management Console\Restricted/Permitted snap-ins\Extension snap-ins

4. In the Extension snap-ins, double click the Device Manager setting to open.

5. If the Disabled option is selected, click the Not Configured  option.

6. Click Apply / OK  and exit the Group Policy Editor.

Open Computer Management and check the results.

Let us know if this helps.
