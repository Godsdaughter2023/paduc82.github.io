Word file over 175MB! - Microsoft Community

neudorfer say:

Word file over 175MB!

I have a word file with 200 pages of text and 100 black & white high quality jpeg photos.  The book has become 195MB in size.  I've tried the reduce file size under the file menu but it's still over 100M.  This is randomly causing an error that says Over 32MB and can't open.   Even if I save the file as a pdf it is still over 100MB in size.   I want to keep the photo quality up but I"d rather not have to re export and  import all of them again in a smaller size.  Any suggestions?


bob jones aka: cybertaz say:
There's no question that the images are responsible for the size of the file, but there isn't much you can do to reduce the size from within Word. I get the impression that you've already used the Reduce File Size feature Daniel suggested. Problem is that JPEG by its very design is already compressed, so it won't have a significant impact. If you have used it I'm surprised the file size dropped as much as it did -- perhaps due to B/W rather than color, but if you haven't used the feature, don't expect more than a 20-25% reduction.

Although he's also right about the 32MB spec, I'm not so sure I agree that the message is "flawed" :-) More likely, IMHO, there is a degree of corruption in the file which has not yet grown to be a roadblock. The images could also contribute to that & it may also be contributing to the size of the file.

Try this;

Use Command+A to Select All
Use Shift+Left Arrow to deselect the last ¶
Use Command+C to Copy
In a new blank document Use Command+V to Paste
Save in Word Document (.docx) format

See if that results in a [somewhat] smaller file.

The only other options I can suggest for reducing the size of the Word file both require re-inseting the images;

Use a decent image editing app to set the required print dimensions & resolution (300 ppi is ample for most non-commercial printed output.), then insert the new versions, or
Replace the embedded images with Linked images [far preferable when using even half as many as what that document contains).

BTW -- Generating a PDF doesn't necessarily result in a file smaller than the original, especially when the original is largely graphic in nature.
