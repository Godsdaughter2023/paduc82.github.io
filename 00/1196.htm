In order for you to setup a GPO with trusted sites in Windows 7 /IE8 you will need setup a windows 7 machine with IE8 and install the RSAT tools.  Then you will need to configure the Trusted sites on this machine how you would like them and use the Group Policy Management tool that is installed with RSAT to import those setting in to your group policy.

 

Kind of a pain, but it’s the best way to deal with it that still allows other uses to add their own trusted sites.
