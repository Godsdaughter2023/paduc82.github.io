Italics command is a toggle. To see this, if you select a word and use Ctrl+I repeatedly, the word will toggle Italics.
 
If at sometime either underlying style was in Italics and Italics had been removed using direct formatting, when you apply a new style, it toggles the style. So what was manually applied non-italics becomes italicised.
 
So as Pam advises, if you select the text and use Ctrl+Spacebar, it will remove the directly applied formatting back to the underlying style.

Your code indicates you are fairly competent with VBA code so you should be able to follow the following example to incorporate into your code.

  • The code copies the worksheet to a new workbook.
  • Then saves the workbook as an xlsx file that automatically removes any VBA code within the worksheet module.
  • Copies the Used Range of all worksheets in the new workbook and then PasteSpecial -> Values that will remove the formulas.

Sub Macro2()
    Dim wbThisWB As Workbook
    Dim wbDestin As Workbook
    Dim ws As Worksheet
   
    Set wbThisWB = ThisWorkbook
   
    wbThisWB.Sheets("Sheet2").Copy
    Set wbDestin = ActiveWorkbook   'The Destination workbook becomes the ActiveWorkbook
   
    'Following code saves as xlsx workbook without any VBA code
    Application.DisplayAlerts = False   'Supress message re saving without VBA code
    wbDestin.SaveAs Filename:="New Book without VBA code.xlsx", _
                    FileFormat:=xlOpenXMLWorkbook, CreateBackup:=False
    Application.DisplayAlerts = True    'Restore Alerts ASAP
                   
    'Following removes all formulas and replaces with values
    'Links should also be removed with this code.
    For Each ws In wbDestin.Worksheets
        ws.UsedRange.Copy
        ws.UsedRange.PasteSpecial xlPasteValues 'Removes all Formulas and replaces with values
    Next ws
   
    Application.CutCopyMode = False

   wbDestin.Close SaveChanges:=True        'Added with Edit after initial posting

End Sub
