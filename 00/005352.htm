Amylove,

I just checked.

This has been the same for several years.

An address added as a connected account will be able to send and receive emails.

An existing address setup as a alias may be able to send emails, but will not be able to receive emails.

ron sommer mvp 2009-2017 say:

Emails go to the server for that account.

An existing address added as an alias may be able to send emails.

It will not receive emails, because Outlook.com does not download emails from the account's server.

When an address is added as a connected account, the server for that account receives the email.  Outlook.com downloads the email from the account's server.
