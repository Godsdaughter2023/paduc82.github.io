space after colon - Microsoft Community

david p. vancouver say:

space after colon

I have set my "spaces after sentence" to 2.  Now how do I set the "spaces after colon or semi-colon"?
jay freedman say:

Word does not have any setting to examine the number of spaces after colons or semicolons, because those characters are not considered to be sentence-ending characters. In fact, the setting in the Grammar Settings dialog does not change the number of spaces between sentences, but only flags instances that don't match the setting -- you still have to correct those instances manually.

To make the changes to two spaces, regardless of whether they're marked with blue wavy underlines, use the Replace dialog.

  • In the dialog, click the More button to expand the dialog box.
  • Check the "Use wildcards" check box.
  • In the Find What box, enter the expression

: ([A-Z])

There is one space between the colon and the opening parenthesis.

  • In the Replace With box, enter the expression

:  \1

There are two spaces between the colon and the backslash.

  • Click the Replace All button.
  • Now repeat these steps with a semicolon as the first character of each expression. You may repeat again using a period (full stop), a question mark, and an exclamation mark as the first character.

To make all of this into one operation that is easily used, record a macro while making the replacements, and assign a keyboard shortcut or a Quick Access Toolbar button to the macro.
