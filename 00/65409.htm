Cannot Remove Read-Only from Folder

I can't remove read-only from a folder. I am the owner of that folder and when I apply the changes it back to read-only.

I am on Windows 10 Build 18362 I am on a laptop from Dell.

keith a. miller say:

Are you having issues with saving, renaming, editing, or deleting files? If you are receiving any error messages, please post them.

If you've just noticed the Read-only checkbox in the folder's Properties dialog, that is not a status indicator!

That checkbox is a 3-state switch. It's default state, the gray square, means that if you click OK or Apply, the read-only attribute for the files within that folder will not be modified from their currnent state. Checking the box will set the read-only attribute on the files within the folder. Clearing the checkbox will clear the read-only attribute on the files within the folder.

Keith
