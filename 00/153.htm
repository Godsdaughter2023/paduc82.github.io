Pasting into visible cells only excel 2016

I saw this code from Bob Umlas.

I am always copying and pasting from the same area, so I dont need the pop up part. How can i edit the code so it always copies from $B$999:$B$1009 and pastes to $A$1:$A$1000

 

Sub Copy_Filtered_Cells()
    Set from = Selection
    Set too = Application.InputBox("Select range to copy selected cells to", Type:=8)
    For Each Cell In from
        Cell.Copy
        For Each thing In too
            If thing.EntireRow.RowHeight > 0 Then
                thing.PasteSpecial
                Set too = thing.Offset(1).Resize(too.Rows.Count)
                Exit For
            End If
        Next
    Next
End Sub

Using audible.com audiobooks sucessfully on a Nokia 5800- guide - Microsoft Community

nickb99---01 say:

Using audible.com audiobooks sucessfully on a Nokia 5800- guide

There are a few threads on this subject but many are outdated/confusing. So pulling them all together with some extra info here.  Got audible.com audiobooks working well on my 5800 xpress music yesterday.  This is a great forum with lots of useful info so hope this gives something back in a small way.

 

Before you start open an audible.com account and get your user name (email address) and password. You will need it for the install to the phone.

 

1. Attach 5800 to yr pc with USB cable, choose 'mass storage' connection

2. Download from audible.com their client. I selected mp3/phones option when asked.

3. During its install select 'Nokia phone' and 5230 as the model.

4. Select the 'phone not connected to PC' option when asked. In the next step it will ask you to use a memory card or similar to get the audible client onto the phone.

5. As you already have the phone connected that is easy. Select the phone memory card (it will be E: or similar) and the audible software will be copied over.

6. Once the PC client is finished installing go into File Manager on your phone and search for the install file. It will be a SIS file in a Audible folder on the memory card. Run it to install the phone client. You will need your audible account name and password at this point. You can delete the SIS file once the install is complete.

7. Don't worry if you get 'connection error' when your phone tries to sync the library over the air.

8. Once the phone software is installed plug it back into the PC, using Mass Storage again.

9. When you run the Audible manager on your PC it should now see the phones memory card as the storage location.

10. Buy some books on audible.com, they will flow down to your audible client on your pc. Then use the audible client easy function to move the books to the phone memory card.

11. Final step - on my phone the audible client could not see the audio book files until I moved them into the E:\Audible folder. The audible PC software had placed them in E:\, the root of the memory card. I will try and correct this setting in the pc client but it may be something 5800 users have to live with.

12. Once that is done select 'local library' on the audible software on the phone and you should be able to see and select your audiobooks to play :smileyhappy:

 

My 5800 still won't sync over the air (connection to Library always fails with Connection error) but everything else seems to work well. Enjoy your audiobooks.

 Price.SiteID, Price.MaterialID, Price.QuantityID, Site.ProducerID
FROM (Material AS mtrl INNER JOIN (Quantity AS qty INNER JOIN (Price INNER JOIN Transactions AS trx ON Price.PriceDevID=trx.PriceID) ON qty.QuantityID=Price.QuantityID) ON mtrl.MaterialID=Price.MaterialID) INNER JOIN Site ON Site.SiteID=Price.SiteID
WHERE qty.QuantityMeasure="Tonnes" And Year(trx.TranDate)+1>=Year(cdate(Forms!frmWebsiteUpload!UploadToDate)) ANDtrx.TranID NOT IN (SELECT TranID FROM qryTrxRecyclingRate)
GROUP BY Site.ProducerID, mtrl.MaterialType, mtrl.MaterialRecyclable, Price.SiteID, Price.MaterialID, Price.QuantityID
