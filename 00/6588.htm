Private Sub Worksheet_Change(ByVal Target As Range)
  Dim R As Range
  Dim j As Long
 
  'Do nothing if more then one cell is changed
  If Target.Count > 1 Then Exit Sub
 
  'Get all contiguous cells around
  Set R = Target.CurrentRegion
  'Exclude the top rows
  If R.Row < 3 Then Set R = R.Offset(3 - R.Row)
  'Reduce to the 4 columns of the region (A-D, E-H, ...)
  j = (Target.Column \ 4) * 4 + 1
  Set R = Intersect(R, Columns(j).Resize(, 4))
  'Sort by top left cell of this region (DATE)
  R.Sort R.Cells(1, 1), Header:=xlYes
End Sub

greginmich say:

DOS/Alureon is the detection for a corrupted MBR (or VBR), which is difficult to repair automatically.

http://www.microsoft.com/security/portal/threat/encyclopedia/Entry.aspx?Name=DOS%2fAlureon

 

The MPC encyclopedia entry for “DOS/Alureon” claims that “Microsoft security software detects and removes this threat.”, and then goes on to describe the manual steps required to repair the damaged MBR. But since “DOS/Alureon” is nothing more than the detection for the damaged MBR, the claim that “Microsoft security software detects and removes this threat.” isn’t quite true. Microsoft AV products can’t repair the damaged MBR.

 

The MPC encyclopedia recommends Windows Defender Offline in the “Additional recovery instructions” for this threat, but we’ve never seen any evidence that WDO is actually capable of completing the removal of DOS/Alureon, while we’ve seen plenty of cases where a PC infected with DOS/Alureon is rendered unbootable when it’s scanned with Windows Defender Offline – so I wouldn’t risk it:

http://www.microsoft.com/security/portal/threat/encyclopedia/entry.aspx?name=Trojan%3aDOS%2fAlureon.J

http://answers.microsoft.com/en-us/protect/forum/protect_other-protect_scanning/windows-defender-offline-removed-alureonj-now/c74319b4-6a5a-437d-b93a-122e70394722

 

Instead, follow the steps in this Malware Removal Guide, which starts with TDSSKiller, a tool that’s generally recognized as the most effective weapon against Alureon bootkits.

http://www.selectrealsecurity.com/malware-removal-guide

No, you should not need to open Adobe Reader to get a pdf to open.  However, the association of pdf files with Adobe Reader may need to be set.

 

Right click on a pdf file in Windows Explorer and select Open with... and in the Open with dialog, select Adobe Reader  and check the box for Always use the selecte program to open this kind of file.
