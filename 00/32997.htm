The current Blu-ray technology makes it unfeasible to stream a game directly off the disc, as the disc cannot be read fast enough - all games must be installed to the hard drive in order to run. Games today are far more graphically complex nowadays, and thus require dozens of gigabytes of storage space (on average).

Technically blu-ray has a faster transfer rate than DVD. It's not the speed that makes blu-ray ill suited to play from but how it is used.

DVD runs at a variable rate. It speeds up and slows down as needed. So the drive can speed up and return to a previous section of the disc to pull off data.

Blu-ray runs at a constant speed. whatever the speed drive is rated at, will be the speed it always runs at. keith- say:

Why does the screen brightness on my tablet keep changing unexpectedly?

Why does the screen brightness on my tablet keep changing unexpectedly? Why does the screen brightness on my tablet keep changing unexpectedly?
michael. o say:

Some tablets have an adaptive brightness sensor that detects the current brightness in the room and adjusts the screen's brightness accordingly.  If the light hitting the sensor is constantly changing, or if you accidentally cover this sensor with your hand or thumb, it can fool the sensor into thinking that you're in a dark room and it will lower the brightness on the screen.

 

In Windows 8, the adaptive brightness setting is enabled by default.  You may be able to prevent unwanted changes in screen brightness by disabling the adaptive brightness setting. 

1.     From the Start Screen, type Control Panel and select the Control Panel.

2.     Set the View by: dropdown to Category

3.     Select Hardware and Sound.

4.     Select Power Options.

5.     Select the Change plan settings link on your current power plan

6.     Select Change advanced power settings

7.     Expand Display

8.     Expand Enable adaptive brightness

9.     Change the setting to Off and select OK.
