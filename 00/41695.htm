Based on my online research, the issue seems to occur on computers that have Windows Firewall disabled. Windows Firewall service has to be running even if you’re using another antivirus cum firewall program on your computer.

 

I’d recommend you to check for the status of the “Windows Firewall” service and set it to “Automatic”.

 

Follow these steps:

 

a.    Click “Start”.

b.    Type “services.msc” in the Start search box and hit “Enter”.

c.    Right-click “Windows Firewall” and click “Properties”.

d.    In the “General” tab, select “Automatic” under the “Startup type”.

e.    Click "Start" to start the service.

f.     Click “Apply” and “OK”.

 

Restart the computer and check if the issue gets fixed.
