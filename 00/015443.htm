It would probably be easier to have all of the data in an Excel spreadsheet with one row of data for each of the 19 documents and set up the Skeleton document as a mail merge main document with mergefields in it for each of the data items.

This will give you a low tech (non VBA) method of populating the document from the data from the desired record in the Excel data source.

I would suggest that you save the skeleton document as a template with the Excel data source attached to it, and then when you want to create one of your document, use File>New and select that template and then use the Mail Merge Preview facility to select the required record from the data source and then use CTRL+a to select the content of the document and then CTRL+SHIFT+F9 to unlink the fields in the document and then save it.  By doing that (creating a new document from the template), the template itself will remain unchanged so that you can repeat the process over and over.

Sub Delete_SubFolder()
Dim obj As Object
Dim ff As Object
Dim myPath
myPath = "C:\Folder1\Folder2\"   '<<< NOTE: search, in folder2 to delete the desired subfolder
Set obj = CreateObject("Scripting.FileSystemObject")
Set ff = obj.GetFolder(myPath)
For Each f In ff.subfolders
On Error Resume Next
If MsgBox("delete folder: " & f.Name & " ?", vbOKCancel) = vbCancel Then GoTo aaa
Kill myPath & f.Name & "\*.*"    ' <<< step1 delete all files in subfolder
RmDir myPath & f.Name   ' <<<< step2 delete empty subfolder
aaa:
Next
End Sub
