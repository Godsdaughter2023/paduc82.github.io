hansv mvp say:

Make sure that all cells are initially unlocked.

Protect the worksheet (with a password, if you like)

Right-click the sheet tab.

Copy the following code into the worksheet module:

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim r As Range
    If Not Intersect(Range("H:H"), Target) Is Nothing Then
        Me.Unprotect Password:="Secret"
        For Each r In Intersect(Range("H:H"), Target)
            r.Offset(ColumnOffset:=-7).Resize(ColumnSize:=7).Locked = (r.Value = True)
        Next r
        Me.Protect Password:="Secret"
    End If
End Sub

Remarks:

Change the password as needed. Use "" if you don't want to use a password.

This will only work if the user sets cells in column H to True, not if column H contains formulas.

Save the workbook as a macro-enabled workbook (.xlsm).

Make sure that you allow macros when you open the workbook.

I suspect that the part your are trying to highlight is not recognized as a "part" by Word. It sounds like you are trying to edit an image (scanned?) or a protected portion of a document.
