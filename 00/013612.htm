mikebmassey say:

Define the width of a column in a powerpoint table

Is there a way, via the Powerpoint or via a macro, where I can define the width of a column in a table?  You can control the size of the table (for example - 5 inches by 4.5 inches), but if I want to specify the width of a column in that table to be 1 inch, the only way I can see to do so is to drag the column smaller or larger, but it may not be exactly an inch.  

Is there a way to do this?  
steve rindsberg say:
Here's how you'd do it with a VBA macro:

Sub SetColWidth()
' This assumes you've selected a table
' It won't work otherwise
'
' It sets the first column width to 144 points (2")
' Edit to whatever value you like.  72 points = 1"
' Values are in points

' Change Columns(1) to whatever column# you'd like to edit

    Dim oSh As Shape
    Dim oTbl As Table
    
    Set oSh = ActiveWindow.Selection.ShapeRange(1)
    Set oTbl = oSh.Table

    With oTbl
        .Columns(1).Width = 144
    End With

End Sub

ActiveDocument.CompatibilityMode is a half baked, poorly documented, Long, not a Boolean.
 
Depending on what condition you want to check for, you could either make explicit checks against the version constants (wdWord2003, etc.), or take a flyer and compare it to Application.Version which appears as though it ought to work into the future (it may not do what you want in future releases, of course!).
