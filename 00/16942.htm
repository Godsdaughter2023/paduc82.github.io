jeff b141 say:

Change Office 365 default Address Book

I have Office 365 Home Premium, and need to change the default address book from Contacts ( This Computer Only ) to Contacts.     The "Contacts (This Computer Only) "  folder is empty.   The straight "Contacts"  file is the one I imported from Office 2007 off of my old machine.   Can this be done ?
diane poremsky mvp (slipstick.com) say:

The default Contacts folder is in the Default data file - if you set the data file Contacts is in, as default in File, Account settings, Data files tab, it will be set as default. If you just want to see that contacts folder first when you click the to button or open the Address book dialog, you can change it. Open the address book (ctrl+shift+b) - then click it's Tools, Options and you can set which contacts you see first  in the last option on the dialog.
