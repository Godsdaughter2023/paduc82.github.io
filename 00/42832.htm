divya r say:

Hi Cosmic108,

 

 

-      Did you turn off the System Restore earlier?

-      Are you referring to Disk usage space by System Restore or is it the processor usage that you are referring to?

 

 

 

System Restore might use between three and five percent of the space on each disk. As the amount of space fills up with restore points, it deletes older restore points to make room for new ones. System Restore doesn't run on disks smaller than 1 gigabyte (GB).

 

Method 1: If you are referring to the disk usage space by System Restore, here’s how you can adjust it:

 

 To adjust disk space usage:

a. Right click on My Computer and select Properties - System Restore tab.

b. Under “Disk Usage Space” move the slider appropriately to utilize the space specified by you for System Restore. Click OK.

 

Note:If you decrease the disk space to be smaller than the space that is currently being used by system protection, older restore points, including previous versions of files, will be deleted.

 

More information is available at:

 

How the System Restore Tool Handles Hard-Disk Space Usage

 

 

Method 2: If you still wish to repair System Restore, disable and enable System Restore function to do the same.

 

Step 1: To disable System Restore

 

 

a. Right click on My Computer and select Properties – Click on System Restore tab.

b. Check the box “Turn off System Restore” and click OK. Restart your computer.

 

Step 2: To enable System Restore

 

a. Right click on My Computer and select Properties – Click on System Restore tab.

b. Uncheck the box “Turn off System Restore” and click OK. Restart your computer.

 

Disabling and re-enabling System Restore should fix the issues related to this function.

Hi cevicho,

Welcome to Microsoft Answers!!!

Follow the steps below to disable the built in Administrator account.

Method 1 : Using Command Prompt

1. Log out from the Administrator Account and log in on another user account.
2. Go to the Start menu and in Accessories list, select Command Prompt
3. When the window appears, enter the command net user administrator /active:no
4. Your log in screen should now display only your common user accounts.

Method 2: Using User Account Control

1. Go to your Start menu and right-click on the My Computer icon.
2. Choosing Manage from the list will prompt a User Account Control (UAC).
3. When you are done, the Computer Management console will appear. Expand the Local Users and Groups option with the use of the arrow next to it and choose Users folder.
4. You can now select the Administrator Account and go to the Actions panel to open More Actions.
5. Choose Properties  to open the Administrator Properties dialog box and check the option labeled Account is Disabled.
6. When done, click OK to confirm changes.

For more information on built in Administrator account access the link below.

http://technet.microsoft.com/en-us/library/cc766343%28WS.10%29.aspx

When you locate the key in Windows Registry the right pane is where you would see Attributes listed.

I could right click in the right pane > Select New DWord to add the word Attributes to the menu

I could then right click on Attributes > Select Modify and give it a value of (usually) 1 or 0 (when working with registry keys).

Before attempting the following, backup Windows Registry.  If you experience issues, you may then restore the Registry to previous state.

How to back up and restore the registry in Windows

https://support.microsoft.com/en-us/kb/322756/

If you're uncomfortable with doing the following, then don't do it.

To add the key to Windows Registry:

If you can locate this much in the Registry "HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\" and what is missing is the string number:

7516b95f-f776-4464-8c53-06167f40cc99\8EC4B3A5-6868-48c2-BE75-4F3044BE88A7

Click on Edit > New > String Value to add the string number

Once you've added the string value

Select it.  Do you see Attributes in the Right Pane?

If no, right click in the right pane > select New > Dword (32-bit)

Type Attributes

Right click on Attributes in the right pane > Select Modify

Change the value to 2

Close Windows Registry > Reboot computer for changes to take effect
