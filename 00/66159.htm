lemp say:

The best and safest way is to uninstall it using Control Panel > Programs and Features and then reinstall it using the game's installation media (CD/DVD or downloaded installation file).  Check with the game manufacturer's website to save and move any saved games, customizations, settings, and the like.  If you need a license key or other authentication, be sure to find this out before uninstalling.

Usually -- but not always -- when you install a program you will have a "custom install" option where you can select the installation location.  Even when this is available, however, there likely will be some data saved on the system drive no matter what you do.

Attempting to "move" an already-installed application is fraught with danger.  Applications typically have pieces in multiple locations and games often are the worst offenders at this.  You may easily miss some component.  Moreover, almost all Windows applications other than the very simplest ones will have multiple entries scattered throughout the Windows Registry.  You would have to find and edit all those entries to point to the correct new location(s).

a. user say:

Don't bother with the embarrassing 'Microsoft Support Engineer.'

There's no such thing as a default user account - that would be a big security risk.

 If you are the only user of your computer, you need to have two accounts:

  • an Administrator account - that's a user account with Administrator permissions - in order to make system-wide changes to the computer. You wouldn't store your personal files in this account. Your Administrator account is only used when you need to make changes that affect the entire computer, like installing software; and
  • a Standard account - that's a user account that does not have Administrator permissions. This is the account that you use for your day-to-day computer use, and it's in this account where you store all your personal files (documents, pictures, bookmarks, etc.)

Your Standard account can be either a local account or a Microsoft account. A local account can only be used on your own computer; a Microsoft account can be used on any computer. Your Administrator account should be a local account.

Combining all your personal files into one place from the four corners of the earth is easy-peasy: Once you've settled on a Standard account, move all your personal files from the other accounts into your Standard account. You'll need to do this from your Administrator account, because you're making a change that affects the whole computer. Every time you move files, you'll be asked if you want to take ownership of the files, and you'll say 'yes' each time. In Windows, each user account is considered a separate person - that's why you have to obtain permission to access other user accounts.

Software should be installed in the same account as Windows. That's usually the C: drive. Since you need to be use an Administrator account to install software, your software programs should already be in your Administrator account, and that's where they'll stay.

Most software installs itself in a way that makes it available to all users. If you installed software for only one user, and now you want to change that user, you'll have to uninstall the software and re-install it.
