I am creating computer documentation with 100's of screenshots that I capture to the clilpboard.   I want to paste into word2007.  To do this, I do "Insert-Shape-New Drawing Canvas".   Unfortunately it does not paste into this area from the clipboard, but rather goes into an adjacent text area.    I can paste the clipboard into some other app like Paint and save as a BMP, which then can be pasted in as a picture in this drawing canvas.   I can then insert overlaying documentation, shapes, arrows, etc. and group then along with the pasted in picture to make a group that is scalable, moves nicely, etc.

How do I paste the clipboard into the canvas without having to go to the 3rd party program and the hassle of all the file names, etc
mikhail_str say:

Follow next steps

  1. Snapshot an object (text/image/etc.) of .pdf document, it is automatically coped in Clipboard of Word.
  2. Paste the object; it appears

    To make that object to be inside of the Canvas (and linked/moved together with it, and other objects in it)

    1. Right-click the object on the page (not in Clipboard!), and Copy it.
    2. Create New Drawing Canvas (or make existing one active)
    3. Right-click in the canvas area, a dropdown menu appears, select Paste; the object appears inside canvas

    I do not know why, but it does work.

    Good luck.
