suzanne s. barnhill say:

You don't need anything after \p. Adding the \p switch causes the StyleRef field to display "above" if the referenced style occurs before the StyleRef field or "below" if it occurs after the field (see http://office.microsoft.com/en-us/word-help/field-codes-styleref-field-HP005186193.aspx). This switch actually makes more sense in the REF field, where those words may take the place of a page number, heading number, etc. If you want a reference such as "See Table 1 above," you'll actually need two StyleRef fields, just as you do when you want to reference both the paragraph number and the paragraph text.

 

As a simple test, insert any text in a new blank document and format it as Heading 1. Press Enter to create a Normal paragraph below it. In that paragraph, insert a StyleRef field with the \p switch; when I insert it with the Insert | Field dialog in Word 2003, the result is { STYLEREF  "Heading 1" \p  \* MERGEFORMAT }, but of course the \* MERGEFORMAT switch is unnecessary, so { STYLEREF  "Heading 1" \p } will suffice. This should display as "above."

If anyone wants a macro to give them running headers then here it is:


Sub runningHeaders()
'
' runningHeaders Macro
' Macro to create running headers - document title (at left) and chapter headings (at right) in header.
'
   
    Selection.Fields.Add Range:=Selection.Range, Type:=wdFieldEmpty, PreserveFormatting:=False
   
    Selection.TypeText Text:="STYLEREF ""Title"""
    Selection.Fields.Update
    Selection.EndKey Unit:=wdLine
   
    Selection.TypeText Text:="" & vbTab & vbTab
   
    Selection.Fields.Add Range:=Selection.Range, Type:=wdFieldEmpty, PreserveFormatting:=False
   
    Selection.TypeText Text:="STYLEREF ""Chapter"""
    Selection.Fields.Update
    ActiveWindow.ActivePane.View.SeekView = wdSeekMainDocument
   
   
End Sub



You can edit the name of the headings to suit your needs. As it stands, the macro picks up the first text on the page formatted in "Title" style and puts it in the header of the document (left hand side of the header). It also picks up the first instance of text on any page which has been formatted in a style called "chapter" and places that in the header (right hand side of the header). Hope this is useful for someone.



The macro has been designed such that you double-click on the Header first to enter Header editing mode and then you run the macro. The macro will set up the header, update the fields and then close the header.

pbezucha say:

The neatest macro I have met, to produce stage plots without distorting original data, comes originally from Lori Miller (4/23/2009), I have only made it rather lubricated. Still, I admire it as a result of almost hermetic science, as you can see on the definition of WB Names in the macro.

All what you need here prepared is to have the user defined chart template, saved under the name “xyStageChart”. Basically, it is a xy chart, where you must define two series. You create it on the base of dummy data – follow the help on the topic. Series(1) should have a format of the final stage line, the format of Series(2) is arbitrary.

For creating an embedded dynamic stage chart in any workbook, keep this workbook with macro open, select the two ranges of x and y data in a worksheet with chart to be inserted,  much like as with common Chart Wizard, and trigger the macro.

The sub is not entirely debugged for wrong selections. Tested only with xl2003.

 

Option Explicit

Sub StageChart()

'PBezucha, 2010 (basic idea Lori Miller (4/23/2008)

Dim Xs As Range, Ys As Range, XAddress As String, _

YAddress As String, ASName As String, AWName As String, _       

ErrMsg As String

On Error GoTo ErrExit

Set Xs = Selection.Areas(1)

Set Ys = Selection.Areas(2)

On Error GoTo 0

ASName = ActiveSheet.Name

AWName = "'" & ActiveWorkbook.Name & "'!"

XAddress = ASName & "!" & Xs.Address

YAddress = ASName & "!" & Ys.Address

With ActiveWorkbook.Names

 .Add Name:="XsN", RefersTo:="=" & XAddress & ""

 .Add Name:="YsN", RefersTo:="=" & YAddress & ""

 .Add Name:="XSerN", RefersTo:="=XsN+{0,0}"

 .Add Name:="YSerN", RefersTo:= _

"=LOOKUP(ROW(YsN)-{1,0},ROW(YsN),YsN)"

End With

Charts.Add

ErrMsg = "UserDefined Chart Not Available"

On Error GoTo ErrExit

ActiveChart.ApplyCustomType ChartType:=xlUserDefined, _

 TypeName:= "xyStageChart"

ActiveChart.Location Where:=xlLocationAsObject, Name:=ASName

ActiveChart.SeriesCollection(1).Formula = _

 "=SERIES(," & AWName & "XSerN," & AWName & "YSerN,1)"

On Error Resume Next

ActiveChart.SeriesCollection(2).Delete

Exit Sub

ErrExit:

If ErrMsg <> "" Then MsgBox ErrMsg

End Sub
