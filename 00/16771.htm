amit tandon say:

Refer below image:

Enter below formula in cell D5 & copy to the right:

=IFERROR(INDEX($F$59:$O$59,MATCH(D$3,INDEX($F$60:$O$60,,),1)),"")

This formula returns the corresponding Phase from row no 59 ($F$59:$O$59), when the value in row 3 (starting from D$3) equals or crosses the value in row 60 ($F$60:$O$60). In this example D3 value of 6 is less than 10 (F60) hence it returns a blank. The E3 value of 12 crosses F60 value of 10 hence the F59 value (Phase 1) is returned in E5. And so on.

It is presumed that values in row 60 ($F$60:$O$60) will be in ascending order.

Not sure if you mean the above. The earlier formula can be reconciled with this objective if required where the increments are same.

Explaining the earlier formula - =OFFSET($J$19,,-ROUNDUP(MATCH($A$3,INDEX(ROW(1:50),,),0)/5,0)+1):

1) ROW(1:50) returns an array of values 1,2,3,4, ... 50;

2) INDEX(ROW(1:50),,) is used to obviate using an array formula - else we will need to enter as Control+Shift+Enter;

3) MATCH($A$3,INDEX(ROW(1:50),,),0) returns the match of A3 value in the array of values 1,2,3,4, ... 50. In the above example 8 is returned by this;

4) ROUNDUP(MATCH($A$3,INDEX(ROW(1:50),,),0)/5,0) returns the batch number where each batch of array 1,2,3,4 ...50 is divided by 5 so that each batch consists of 5 numbers. This example returns a batch number of 2 (ie. 8 falls between 6-10 where first batch is 1-5, second batch is 6-10, ...);

5) -ROUNDUP(MATCH($A$3,INDEX(ROW(1:50),,),0)/5,0)+1 - because we are doing a reverse offset from cell J19, we use a negative sign, and then add 1 for offset;

The Footer is on a Master Page which would show

to create multiple Master Pages. Creating a Master

The following excerpt was copied from the Microsoft
Publisher 2007 Help Files:

===
Microsoft Publisher 2007

Hide the header or footer on any page

You can hide a header or footer on any page by creating a duplicate master
page, deleting the header or footer, and then applying it to the publication
page where you want the header or footer hidden.

1.. Create a duplicate master page that contains the header and footer you
want to hide.

How?

1.. On the View menu, click Master Page.

2.. In the Edit Master Pages task pane, click the arrow next to the
master page you want to duplicate, and then click Duplicate.

3.. In the Duplicate Master Page dialog box, do one or more of the
following:

a.. In the Page ID (1 character) box, type a single-character
identifier for your new master page. This can be any single Unicode
(Unicode: A character encoding standard developed by the Unicode Consortium.
By using more than one byte to represent each character, Unicode enables
almost all of the written languages in the world to be represented by using
a single character set.) character.

b.. In the Description box, type a brief description of your new
master page.

Note    The display of this description is limited to 40 characters.

c.. Click OK.

2.. Delete the contents of the header or footer on the duplicate master
page.

3.. Apply the duplicate master page to the publication page.

How?

1.. In the Edit Master Pages task pane, click View publication pages.
    The task pane will switch to Apply Master Page.

2.. Navigate to the page where you want to apply the master page.

3.. In the Apply Master Page task pane, in the box below the page icon,
select the master page you want to apply.

===
