benwells say:

how can i merge two excel sheets together using a unique ID field?

i have two spreadsheets with a common ID field. I would like to merge these two spreadsheets, updating the values in sheet 1 with values from sheet 2
rich007a say:

Ben,

Your description is a vit vague, but here's my interpretation.

If you have two workbooks (files) and you want the data that resides in one file to appear in the other do this:

Open both files.

Go to the file where you want the data to appear.

Select the cell where you want the data to appear.

Type equals  (=)  then go to your other workbook and click on the cell containing the data.

Hit Enter.

The formula of the cell where you want the data to appear should now say something like:

=[OtherWorkbookFileName.xlsx]Sheet1!A1

 

The other way to do this (especially useful if you want to do this with multiple cells is to go to the file containing the data you want to copy.  Select the cells containing the data and copy (hit Ctrl+V).

Now go to the other file and right-click > Paste Special.  Click the Paste Link button at the bottom right.

 

Was that what you wanted to do?
