swapnil rustagi say:

512 MB fits you fine for Windows XP. OK, the application might run slower than on a modern computer - but the performance is reasonable.

Like me, I have a computer with 1 512 MB memory chip and another 128 MB memory chips - making the total upto nearly 640 MB - but I use Photoshop CS3 daily on it and it works fine for most tasks.

However, if you want to get some more performance improvements from your computer for RAM intensive applications, I would recommend increasing the virtual memory.

Find out more about Virtual memory here-

http://ask-leo.com/what_is_virtual_memory.html

To increase Virtual Memory-

Right-click My Computer > Click Properties > Advanced tab > Settings button under 'Performance' > Advanced tab in Performance Options window > Change button under 'Virtual memory'.

Now select Custom size option.

Below the options, you will see some details like-

Minimum allowed - XX MB

Recommended - XXXX MB

Currently allocated - XXXX MB

Make sure you set the value of Initial size (MB) close to the Recommended space. Set the value of Maximum size to a value quite higher than Currently allocated space. (But still set Maximum size value to less than 2048 - you don't need that much).

That might give some amount of performance increase - but it's not going to give you a magical performance boost.

andre da costa say:

Yes, you can use that ISO copy, but you must ensure it is bootable. Because it is a new computer without a qualifying license, you will need to purchase one.

Prepare Windows 10 ISO file

After you have downloaded the Windows 10 ISO, you need prepare it, this is especially important for Windows 7 users..

After obtaining the .iso file you can use the built in Disc Image burning tool or Microsoft .iso to USB/DVD tool to create a bootable DVD or USB (requires a blank DVD or USB flash stick of at least 4 GB).

After downloading the .ISO file, right click it and click Burn disc image

Insert a blank DVD

Click Burn

Windows 8/8.1 users can mount the .ISO file and start the upgrade automatically. Learn more here

Important:

If you are creating a bootable USB thumb drive, make sure the thumb drive is first formatted as FAT32.

Connect the thumb drive

Open Computer

Right click thumbdrive

Click Format

Make sure FAT32 is selected as the file system.

For UEFI based systems

If your computer is UEFI based, these are normally systems that come pre-loaded with Windows 8 or later, you will need to prepare the ISO file for such a configuration or you will receive an error message during setup. The thumbdrive needs to be formatted as FAT32 and use the GPT partitioning scheme. To do this, you need to use Rufus, a small tool you can download for free.

 

After you have installed Rufus:

  • Launch it
  • Select ISO Image
  • Point to the Windows 10 ISO file
  • Check off Create a bootable disk using
  • Select GPT partitioning for EUFI firmware as the Partition scheme
  • Choose FAT32 NOT NTFS as the File system
  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete

Buy Windows 10 Home/Pro

Enter the product key after setup:

Click Start > Settings > Update & security > Activation > Change the product key then enter the new product key then click Next to Activate
