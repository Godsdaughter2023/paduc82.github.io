Most crypto malware ransomware is typically programmed to automatically remove itself...the malicious files responsible for the infection...after the encrypting is done since they are no longer needed. That explains why many security scanners do not find anything after the fact. The encrypted files do not contain malicious code so they are safe. Unfortunately, most victims do not realize they have been infected until the ransomware displays the ransom note and the files have already been encrypted. In some cases there may be no ransom note and discovery only occurs at a later time when attempting to open an encrypted file. As such, they don't know how long the malware was on the system before being alerted or if other malware was downloaded and installed along with the ransomware. If other malware was involved it could still be present so be sure to perform full scans with your anti-virus.

Important: Keep in mind that when dealing with ransomware it is best to quarantine malicious files rather than delete them until you know what infection you're dealing with. N73 How to Hide Files (Photos/Videos) from Gallery view - Microsoft Community

dan955uk say:

N73 How to Hide Files (Photos/Videos) from Gallery view

To All Nokia N73 Gallery Sufferers!!!

I bought my N73 over the weekend, and the problem of Hiding my files from the Gallery view has plagued me ever since I held this phone in my hand...but I've found a solution at long last which hopefully by sharing it here may help others.

The Gallery is excellent for certain things, and I quite like the slide show feature in it..but its total inability to differentiate between what you need to view and what you don't, is a real pain to put it mildly... whoever designed it should be shot!!!

So here's the solution.

Firstly, you need to get a piece of software, a file explorer called Y-Browser, you can get it here:

http://www.drjukka.com/YBrowser.html

With this, you can change the attributes of your folders and files (pictures, videos etc.,) so you can hide them from Gallery... I've read about a million articles and none of them explain it clearly, so here's how I did it:

1. Download Y-Browser onto PC

2. Click on the SIS file, and it starts the Nokia Installer, accept the install on your phone...Once its loaded, you'll find in your MENU screen right at the bottom an icon for DrJukka.Com select it, then select the Y-Browser. This application allows you to see your phone internal & external memory as a series of drives, and clicking on the drives you will see all the folders underneath. We need Y-Browser to view hidden folders and hide our own folders.

3. In this next step we want to let the phone create a hidden directory called _PAlbTN, which the phone uses to store thumbnails for your gallery view. This is a hidden folder so you won't see it in the File Explorer supplied with the N73, thats why we need Y-Browser.

a) From your PC, go to your phone and in the Phone Memory (or your external memory), under the folder called DATA, create a new folder, lets call it SECRET for arguments sake... then copy a few photos into it...

b) Switch your phone to Gallery view, and you should see the photos come up as the application finds the new images recently added.

c) Now, fire up Y-Browser again, and use it to go to drive, and the DATA folder, where you should see the SECRET subfolder... click on this and you will see your images together with a sub-folder called _PAlbTN, in there you will see 3 other folders for different sized thumbnails.

d) Go back up to the contents of the SECRET folder, highlight the _PAlbTN folder, then click Options, File, Attributes, then change Hidden to 'No' (scroll down & click on it). This unhides the folder so we can see it via the PC.

e) On your phone, switch to gallery view, you should still see the images.

f) On your pc, go back to the DATA Folder, and in the SECRET sub-folder you should see your images and the now un-hidden folder _PAlbTN. Now all we have to do, is on the pc, drag the images into this folder...

g)As you do this, as the files are moved into this _PAlbTN folder, you will see the images magically (!!!) disappear from the Gallery view.

h) Thats it, you have now created a SECRET folder in which you can hide whatever you want from the Gallery view. You can create as many of these folders as you want to hide whatever you need.

4. OK, how do you get them back into Gallery so we can still use this applications strengths...because I still like the slideshow option.. well this is what I do...I use the File Manager supplied with the Nokia N73 to copy the files I want to view back into the normal IMAGES folder in the DATA directory, and delete them once I've finished. I know its a pain, but it works, and I try and keep my Gallery tidied up under various secret folders which it can't see, and only view things as I need them.

I hope that helps everyone who's struggling with this... and if I'd found a post like this, it might have saved me many hours of searching... I know a lot of people have given up and probably deserted the ship... but Nokia did get it wrong with the Gallery application, and I have yet to see them own up to its shortcomings or come up with a fix to this problem.

Enjoy!
