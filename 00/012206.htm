There should be an entry in MSCONFIG for MSE. The icon you use in the Start menu.  Right click on it and click on properties to see the location.  it should show the path to a file called msseces.exe.  Make a note of that path.

jim gordon mvp say:

If you can open the file after manually changing the file extension, then the file was not properly saved or named in the first place.


A file that is saved in older formats would have an extension of .xls.

A file saved in a current open format would have a file extension of .xlsx


You can't change the format by manually changing the file extensions. You MUST use File > Save As to change the file type.


See Choosing Excel File Formats in Office 2011 for Mac

and Office 2011 for Mac: Save Excel Workbooks in Old File Formats
