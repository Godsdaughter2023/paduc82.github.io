a. user say:

Hello MelissaDBaggett:

 

Thank you for posting to the forum regarding your question about importing playlists created in Xbox Music to Windows Media Player.  I am afraid this is something that is not possible.

 

The reason for this is that the Xbox Music application does not create a local playlist file.  All playlists created in the Xbox Music application are created and added to your cloud collection and can then be synced with other devices that support the cloud, Windows phone 8, Xbox consoles, Surface devices and Windows 8 PC's.  As there is no local file on the PC there is nothing to export and Windows Media Player does not support cloud access.

 

With that said you have the option to create playlists via Windows Media player if you do not want to try and sort out the issues you are having with Xbox Music.

 

If you would like to try and get Xbox Music sorted out, we would be happy to help.  I am not sure I fully understand the issue based on the information provided up to now.  If you would provide some greater detail related to the steps you are taking and the results we would love to see if this can be corrected.  If you are able to provide screen shots of what is going on that would also be helpful.

 

The troubleshooting I can offer at this point is to uninstall the Xbox Music app from your computer.  Once this is done clear your PC temp files, do this by pressing the Windows key + R > type %temp% in the search box > press Enter > Highlight all the files and folders > right click them and delete.  If any files/folders refuse to delete, skip them.  Once this is done, restart the computer and try the application again.

 

We look forward to hearing back from you in regards to this issue.

 

Mark W.

gary's student say: Try:

Sub Converter()
Dim rr As Range
Set rr = ActiveSheet.UsedRange.Cells.SpecialCells(xlConstants)
For Each r In rr
If Right(r.Value, 1) = "-" Then
    v = Mid(r.Value, 1, Len(r.Value) - 1)
    r.Clear
    r.Value = -v
End If
Next
End Sub

hansv mvp say:
Do you happen to know where I can get the macro to do this. It seems that my macros got wiped out, I had it at one time.


Here is a variation of a macro originally from Gord Dibben. It operates on the selected range of cells.

 

Sub NegSignLeft()
     ' Gord Dibben
    Dim cell As Range
    On Error GoTo ExitHere
     ' Move minus sign from right to left on entire worksheet
    For Each cell In Selection.SpecialCells(xlCellTypeConstants, xlTextValues)
        If IsNumeric(cell.Value) Then
            cell.Value = CDbl(cell.Value)
        End If
    Next cell
ExitHere:
End Sub
