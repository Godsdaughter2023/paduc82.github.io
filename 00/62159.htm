ken blake say:

OK, I suspected that that was what you meant. Those are not USB *hubs*. They are USB ports.

A USB hub is a small device that plugs into a USB port; it itself has several USB ports in it. It's typically four ports, so effectively a hub changes a single USB port into four USB ports.

If it was a hub, the easiest thing to do would be to throw it away and buy a new one; they are not expensive. But with ports built into the computer, I'm not sure what to do. You could try alcohol, but I don't know how well it would work. If you can't clean them, you might need to take the computer to a service company (avoid the Geek Squad and any other big box store) and see whether they can fix the ports or replace them.

On the other hand, there's little downside to first trying to clean them with alcohol.

how to formate my 32GB sandisk pan drive

earlier i was make my pen drive bootable but now i want to format my pen drive. when i going to format m pen drive there shown "This disk is right protected".
mightymiro_wd say:

Hi there.

Some pen drives have a physical switch that allows you to lock write-protection for it, so if possible take a look if your pen drive has such. In case it has such, then move the switch to unlock the pen and this should disable the write-protection.

If it doesn't have one, then you can try to format it via the CMD. In order to do that you can follow these steps:

  • Go to cmd, type diskpart, press Enter.
  • After that type list disk, press Enter.
  • Type select disk X (where X is the number your drive shows up as), press Enter and type clean, press Enter.
  • After that type create partition primary, press Enter, type format quick fs=ntfs, press Enter, type assign letter=F (or the letter you want to assign to the drive), press Enter and lastly - type exit.

Hope this helps and let me know how it went. :)
