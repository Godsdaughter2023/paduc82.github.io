Resolved the problem. It seems that windows security update KB2918614 is causing issues with the MSI installer.

When I reviewed the installer log for Garmin I found the entry "CryptAcquireContext Could not create the default key container" which I googled and found this:

http://windows-installer-xml-wix-toolset.687559.n2.nabble.com/SECREPAIR-CryptAcquireContext-Could-not-create-the-default-key-container-td7596414.html

Which had narrowed the issue to KB2918614. Once I uninstalled that update and rebooted my computer I was able to install without issues.
