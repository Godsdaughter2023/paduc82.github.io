Function missing(mydate)
   Application.Volatile
   lastrow = Cells(Rows.Count, "A").End(xlUp).Row
   olddata = 0
   mymiss = 0
   For k = 2 To lastrow
     If Cells(k, "A") > mydate Then Exit For
     If Cells(k, "A") = mydate Then
       mycount = mycount + 1
       newdata = Cells(k, "B")
       If olddata = 0 Then
         olddata = newdata
       ElseIf newdata > olddata + 1 Then
         mymiss = mymiss + (newdata - olddata) - 1
       End If
       olddata = newdata
    End If
    Next k
    missing = mymiss 
End Function

The above formula uses INDIRECT Function. Below option can also be used which avoids using INDIRECT:

=SUM(IF(FREQUENCY(IF(A2:A100=E2,B2:B100),IF((ROW(1:5000)>=500)*(ROW(1:5000)<=MAX(IF(A2:A100=E2,B2:B100))),ROW(1:5000)))=0,1))-1

If Outlook stops responding you can close Outlook, start it in safe mode, then close it and open it normally to fix the problem.

  1. Close Outlook.

  2. Launch Outlook in safe mode by choosing one of the following options.

    • In Windows 10, choose Start, type Outlook.exe /safe, and press Enter.

    • In Windows 7, choose Start, and in the Search programs and files box, type Outlook /safe, and then press Enter.

    • In Windows 8, on the Apps menu, choose Run, type Outlook /safe, and then choose OK.

  3. Close Outlook, and then open it normally.

If Outlook isnt' working or this didn't resolve your issue, continue to the methods below.

Install the latest Windows and Office updates

Outlook works best when it's up to date. We recommend setting Windows Update to automatically download and install recommended updates. Bug fixes, new features, and performance enhancements are released regularly.

  • To install the latest Windows and Office updates, see Install Office updates.

If installing the latest updates didn't resolve your issue, move on to the next step.

Repair your Office programs

Repairing your Office programs can resolve issues with Outlook not responding, hanging, or freezing. Errors in Office files are repaired automatically.

  1. Quit any Office programs that are running.

  2. In Control Panel, choose Programs and Features.

  3. In the list of installed programs, right-click Microsoft Office 2016 or Microsoft Office 2013, and then choose Change.

If repairing your Office programs didn't resolve your issue, move on to the next step.

Repair your Outlook data files

When you install Office, an Inbox Repair Tool (SCANSPST.EXE) is also installed on your PC. The Inbox Repair Tool can resolve issues by scanning your Outlook data files and repairing errors.

  • To use the Inbox Repair Tool, exit Outlook, go to Repair Outlook Data Files (.pst and .ost) and follow the relevant steps.

If repairing your Outlook data files didn't resolve your issue, move on to the next step.

Create a new user profile

Your profile stores a variety of settings that control how Outlook looks and feels. You can create a new profile and then add your email accounts to the new profile.

  1. Close Outlook.

  2. Choose one of the following:

    • In Windows 10, choose Start. Type Control Panel and press Enter. Choose User Accounts > Mail > Show Profiles.

    • In Windows 8, open the Apps menu, choose Control Panel > Mail > Show Profiles.

    • In Windows 7, choose Start > Control Panel > Mail > Show Profiles.

  3. Choose Add, and in the Profile Name box, type a profile name, and then choose OK.

  4. Enter an email address and a password for the primary email account you use in Outlook and then walk through the steps to create the profile.

  5. Once the profile is created, on the Show Profiles dialog, choose Prompt for a profile to be used.

  6. Start Outlook and choose your new profile.

If creating a new profile solves your issue, you can return to the Show Profiles dialog and delete your old profile.

I agree with Laura.   We're not hear to be screamed at.   We are mostly all volunteers here, and we'll thank you to kindly respect that fact.  Having said that, I noticed that you mentioned Clean Install of Windows Vista, yet you indicated the installation of Norton 360.  In my mind, that is now the farthest thing from being a "clean" install of Windows Vista. 

 

REMOVE NORTON 360

Accordingly, Norton 360 would be the first thing that I would remove.  Then I would download and run the Norton Removal tool to get rid of *all* signs of anything norton-related (orphaned files and folders, Registry settings, etc) on your system in order to ensure that this product was not contributing to your problems.

 

CLOSER TO BEING PRISTINE
After having performed this step, your computer *should* be closer to being pristine except for the WMDC 6.1.   Because of all of the problems you are having, I would remove that as well, rebooting your computer (if asked to do so as a result of having removed this component).  

 

RUN CHECKDISK STRATEGY
I would then run whatever Check Disk utility is available in Vista.  Unfortunately, I am not familiar with this OS and can't say whether or not this program is still available.   But from XP, you could issue these commands to run it:

 

1.) Click Start > Run > "cmd" (to get the command prompt), then Enter key to get the command prompt box.

2.) Type in "chkdsk" then Enter Key and confirm that it passes the entire test (usually split up into 3 stages).  Don't worry about the "/F" message.  I'm not trying to fix anything here, just confirming no pre-existing problems.

3.) After completion and pressuming success, type "exit" then Enter Key and the window will close.

 

If you got any errors, then you have problems with your Hard Drive, or the files on the Hard Drive, which could be contributing to the situation.  Your choices at this point are to research how to fix this kind of problem yourself, or you can take your computer to a computer repair shop.  If your computer is under warranty, you can obviously return the computer to the vendor for Warrranty repair.

 

DOWNLOAD A FRESH COPY OF WMDC
I would now re-download the WMDC program from here (just to eliminate the possibility that your existing WMDC installer is corrupt).  DO NOT INSTALL THE PROGRAM.  Save it instead to the hard-drive where you can find it later and run it.

 

REBOOT THE COMPUTER
I would now suggest rebooting the computer after having run the Check Disk utility and re-downloaded the current version of WMDC.

 

INSTALL WMDC AGAIN

Now re-install WMDC again, and let us know whether this fixed the problem or not.  If it doesn't, please try to be civil when you respond.  Thank you for your understanding in this matter...
