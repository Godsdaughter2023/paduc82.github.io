ken sheridan say: Do you want to return all rows, but show a value in the computed column if the 'Warning – expires in one month' date has been reached or passed, or do you want to return only those rows where this is the case?

If the former then you can use the IIf function in the expression to return a value of your choice, e.g.

IIf(Date() >= [Warning – expires in one month],"Expired","Unexpired")

This will return 'Expired' if true, 'Unexpired' if false.

If the latter do as Doug says and put True in the 'criteria' row of the computed column in query design view and uncheck the 'show' checkbox.

An do take note of what Doug says about table and column names.  'Warning – expires in one month' is pretty bizarre as a column name, and even more so when it is a column of date/time data type.  Normally something like ExpiryDate, which represents the attribute which the column represents, would be used.

I solved this on my Mac. I figured it was a permissions problem but in El Capitan, it's no longer as easy as launching Disk Utility. All Disk Utility allows you to do is fix disk errors but not permissions. 

So I booted into Safe Mode (restart your Mac and right after you hear the chime, hold down the SHIFT key until the Apple logo appears).

Once you log in, (you will notice that Safe Mode appears in the top right corner in red), go to Applications>Utilities and launch the Terminal app.

When in Terminal, copy and paste the following:

sudo /usr/libexec/repair_packages --repair --standard-pkgs --volume /

You will be promoted for your password but the blinking cursor doesn't move while typing it. Once you hit enter, the fix permissions process will start. Wait until it's finished, open up Mail>Preferences and change the default mail handler to Outlook.

You might be able to solve this without booting into Safe Mode but I did it as a precaution because in normal mode, there could be any number of services running that might affect the permissions process.

This isn't an add-in issue so starting Word in Safe Mode, as advised in this thread, is a waste of your time. Text highlighting doesn't work like the Bold or Italic format does.
