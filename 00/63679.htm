Pokki update

davem121 say:
Hi Susan,

Was your PC originally Windows *, then updated to Windows 10?

Pokki is a utility used to add the missing Start Menu to Windows 8

You can safely uninstall Pokki from Control Panel = Programs and Features

Also check your Startup for that Pokki Updater:

Press Ctrl + Shift + Esc to open Task Manager
Click on the Startup Tab
See if Pokki updater is listed there
If it is, select it and click 'Disable'
Close Task Manager

Click your Start Button, type msconfig and hit Enter
When the System Configuration dialog opens, click on the Services Tab
Check the box marked 'Hide Microsoft Services'
In the remaining list, see if Pokki updater is listed there, uncheck any you find
Click Apply and OK

Restart your System and this annoyance will be gone . . .

How you activate Windows 10 after reinstalling it depends on how Windows 10 was originally installed on your device.

stevenparks58 say:

After upgrading from Windows7 to Windows10, network interface is not functional

This morning I tried doing a Windows 7 to Windows 10 upgrade.

Everything was fine before the upgrade, entire network was rock solid.

After the upgrade, the main network interface (Intel 1217-V) comes up with "limited access", which in this case means "no access".

I tried connecting up a USB Ethernet Adapter and that works fine, so I had a 100Mbit connection at least.

I tried updating the driver using the automatic search utility, it said I had the latest driver.

I tried installing and uninstalling the driver

I tried ipconfig/release and ipconfig/renew.

I tried setting a fix IPv4 address.

I even tried to let Windows diagnose the problem but that didn't work (not that I expected it to).

I finally found a help page entitled "Windows 10 Limited network over Ethernet "Unidentified network""

and a second one entitled "Intel I217-V network adaptor Windows 10 SOLVED"

These articles suggested going straight to the Intel driver download page at

https://downloadcenter.intel.com/download/25016/Network-Adapter-Driver-for-Windows-10

I downloaded the driver for Windows 10, 64 bit and installed it.

And it worked!

So now I have my GigE network connection working again.

The lesson learned here is don't trust Windows when it says you have the latest driver.
