The solution is to rename the existing event routines with the same names, e.g.:

Private Sub Worksheet_BeforeDoubleClick_A(ByVal Target As Range, Cancel As Boolean)
  If Target.Column >= 4 Then
   Cells(Target.Row, 11) = Target.Value
  End If
End Sub

Private Sub Worksheet_BeforeDoubleClick_B(ByVal Target As Range, Cancel As Boolean)
    Cancel = True
    Target.Interior.Color = vbGreen

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)

jllatham say:

[Edit - see my next post for an easier way]

I would do this in 2 stages.

First I would take the formulas in rows 1 through 30 and make the cell addresses absolute.  This will keep those references from changing when you do the copy.  As an example, 'C01'!C1 would become 'C01'!$C$1  and 'C01'!X6 would become 'C01'!$X$6.

It will take several steps to do this.  Select all of the cells with formulas in them for ='C01'!  and use Edit Replace to work through the different column references for columns C, J, Q, X, etc.  Use a form for the Edit to Find the column letter along with the ! symbol, as  Find:   !X   and then replace with  !$X$

Once you've got those changed, you can then select all 30 rows of formulas and copy them to the next area on your worksheet.  Choose all of the copied cells and use Edit - Replace to then find C01! and change it to C02! within that area.

Repeat for all 10 sheets.  Make a copy of your workbook to practice this with in case you make an error in any of the Edit - Replace operations.

Wish there was an easier way, but sometimes there just isn't.  I could think of a couple of other ways to do this, but each would take just as large an effort to do.
