hansv mvp say:

If you want the value to begin with B, you should not have an asterisk * before the B:

 

WHERE text_new Like "B[0-9]*"

 

This will return all values that begin with B followed by a digit, and possibly by other text.

ADDED ======
You have 'AND' in the WHERE statement so it would have to be like this --
GS5xxxByyy   OR ByyyGS5xxx,   having both 'GS5' AND 'B' in the same record.

marshall barton say:

I don't follow your explanation along with that query and the G5 thing, but maybe you're asking for something more like:

SELECT main_id

FROM local_main

WHERE (text_new Like '*B#*' OR text_new Like '*B##*' OR text_new Like '*B###*' OR text_new Like '*B-#*' OR text_new Like '*B-##*' OR text_new Like '*B-###*') AND text_new Like '*G5[5-9]*'

 

Note that # is a wildcard that does the same as [0-9]
