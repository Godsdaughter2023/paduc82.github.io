daniel g. grau say:
Word is telling you that because the selected text already has that style applied, but also has additional modifications relative to the way the style is currently defined in the document.

To stop that message, apply the style before entering any formatting changes, or don't have any text selected when you apply the style (place the cursor between characters).

Think along the lines of Word when encountering that situation:
  • The user is telling me to apply this style to the selection, but I have already applied it.
  • In addition, the user has changed the looks of the selected text.
  • Is the user: (1) unhappy with the way the associated  style looks? (2) unhappy with the way the selected text looks?
  • What should I do? What should I do?

This results in Word asking you whether you repent of your selected format changes or repent of your current style definition.

If you ask me, Word should stick to following orders, rather that attempt to be obtrusively helpful.
