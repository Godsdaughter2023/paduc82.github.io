SUMIFS across multiple pages

Hello -

I am trying to sum a particular cell across multiple sheets, but only if a certain cell on that sheet meets a certain criteria. tom ogilvy say:

What you show is a 3D Range.   You can only use a 3D range with a relatively small subset of functions.  The xxxIF functions such as SUMIF do not support 3D ranges. 

Sumif can actually work across multiple sheets.   If I have sheets named Sheet1, Sheet2 and Sheet3 then

=SUM(SUMIF(INDIRECT("Sheet"&{1,2,3}&"!A1:A10"),"High",INDIRECT("Sheet"&{1,2,3}&"!B1:B10")))

if I have sheets named AAA, BBB and CCC then

=SUM(SUMIF(INDIRECT({"AAA","BBB","CCC"}&"!A1:A10"),"High",INDIRECT({"AAA","BBB","CCC"}&"!B1:B10")))

These are not array formulas.  They can be entered normally.   If you add and remove sheets, it won't adjust like the 3D formula reference.

--

Regards,

Tom Ogilvy

Prevent losing email in Outlook.com when you've set up POP3

Outlook.com has a feature that prevents losing email messages if you accidentally set up POP3 to delete them after reading them on the server. Here’s how it works:

    The first time that Outlook.com receives a POP3 instruction to delete email messages after reading from an email app, Outlook.com won't delete the messages.     Outlook.com keeps the email messages in the POP folder until you sign in to your Outlook.com account and verify whether Outlook.com should respect or ignore the POP3 delete command.

To verify your POP3 settings:

Don’t let another program delete messages from Outlook.com. (If your other program is set to delete messages from the server, we’ll simply move them to a special POP folder. They won’t be deleted.)

Nokia Belle FP2 -Pros and bugs. - Microsoft Community

srikanthsd say:

Nokia Belle FP2 -Pros and bugs.

Nokia Belle FP2(Sw.Version 113.010.1506) seems to be another cool release by Nokia with many bug fixes and lot of improvement in ease of usability.

 

Here are some of the things I would like to share:-

1.New music player with new appearance and correct album list.

2.Search shortcut on the homescreen.

3.Music player pause and play in the dropdown status bar.

4.Improved transition effects.

5.Very good in place pop up options for fast selection of items.

6.Improved video player aspect ratio switching with double tap.

7.Improved speed in file manager.

8.Cool new extra widgets.

9.Nice looking security lock keypad.

10.Nokia 808 style camera interface is a highlight in FP2 with lot of speed improvement in capturing an image!

11.Very good new lock screen with fast access to alerts.

12.Camera can be opened while keypad is locked!

13.New keyboard with native language.

14.Improved volume level design.

15. Improved selection of text for copying and pasting.

16. Apps open much faster especially like the Dictionary app.

 

 

Bugs/Drawbacks:-

 

1.The worst keyboard ever.

 

A. Lots of eye movement, have see both body of typing and suggestion. No predictions is used, though prediction is present, it is not used for helping the user to write the message fast but to struggle in selecting appropriate words.

First type regardless of tiny mispellings and check whether the word is in the suggestions or not and select appropriate word, instead of the phone selecting appropriate word itself and helping the user to select the word by itself.

user has to see the suggested words and select the word instead of looking at the message body.

please don't make our phones unusable.

 

B. No right and left arrow keys to move between tiny letters. 

These keys appear only when you tap on the body of the message in messaging only, not in an app like opera mini.

Try and try again until you get the cursor beside correct letter!

 

C. The suggetions on top of the keypad takes extra speace and blocks the typing area apps like opera mini. See attachment.

 

 

2. Unusable gallery app.

 

A. Now there are only two options in gallery. Add tag and delete with mark option. No mark option to select multiple images and send them via mail or bluetooth. You can send one by one by opening an image and then by using options-send.

 

B. There is no add to favourites folder or any other custom folder to move your important photos or images of your important documents etc.... you need to scroll to the correct image and open it. Now the more images you have more  scrolling and time it takes for you to see your image.

 

C. The new gallery app cannot zoom in high resolution photos or the photos from 808 pureview.

It shows blank black screen when you zoom.

 

D. Image editor cannot zoom the image like as with old editor. Very limited zooming. like you zoom in gallery.

 

 

3. Battery performance seem to be improved now, but not like initial Belle.

 

 

4. Camera:

Option to select front camera has been removed. Very bad.

 

 

5.Mass storage mode.

Now with FP2, Connecting in Mass storage mode hangs your phone completely, only way is to remove the battery.

 

 

 

Thanks.

In love with Nokia.

We have run multiple tests with an IMAP server known to support IMAP IDLE and any mail sent to an account on this server arrives within a minute from the moment it is sent when the mailbox is setup as an IMAP mailbox. When set up as a POP3 mailbox the mail arrive at the intervals set in the mailbox sync settings.

 

We created a Microsoft account (*** Email address is removed for privacy ***), Google account (*** Email address is removed for privacy ***) and 'regular' POP3 and IMAP account and all accounts receive new mail at either set interval (15mins/POP) or within a minute(IMAP4) from sending a mail to the account.

 

This test was done using a Nokia 808 running Nokia Belle FP1. We will repeat this test when FP2 becomes available again, but from what we have been able to verify either polling at the set interval or through the IMAP IDLE function works for us.

 

Kosh
