robinblackdiamond say: This sounds like their may be whats called an "open handle" for the files you're trying to delete/move.

What this means is that another process on your system has them open for some sort of file operation (such as read/append/write/etc).

This could be a Windows system process, an application, or even anti-virus.  Some times anti-virus applications latch on to some files and fail to release their handle preventing other applications/processes from gaining access.  The typical response is "access denied" or a system error message #5.  In this case it isn't because you don't have the required permissions to access/delete/move the file, it's simply because something else has it first and isn't letting go of it.

The easiest way to resolve this is to close any open applications that may be using these files and trying again.  If that fails, try a restart.  If that fails, try temporarily disabling your anti-virus software and trying again.

If all of that fails, you can dig a little deaper.  If all of the above are failing its most likely something caused by the system or a process that runs when you start Windows.  Starting in Safe Mode can prevent some things from starting up and allow you to delete the files possibly.  To start in safe mode, you'll need to reboot and press F8 before you see the Windows splash screen and select start in safemode option.

There are utilities such as the Windows Process Explorer from System Internals that you can download from the following Microsoft site:
http://technet.microsoft.com/en-us/sysinternals/bb896653.aspx

This tool will allow you to search for your files and identify the processes that have open handles to them and forcefully remove the handles through Process Explorer.

If you've made it this far and you're still having problems then please post back and provide some additional details specifically about the path and files you're trying to delete.

Sometimes a corrupted print job will get stuck in the print queue and cause subsequent print jobs to stack up waiting for the first one to be cleared.

Click the Start Orb, click Devices and Printers, right-click the icon for your HP DJ 2540 and then click "See what's printing."

  • Open an Elevated Command Prompt (search for "cmd.exe"; right-click and select "Run as administrator")
  • Type the following in the black Command Prompt window and press Enter after each line

                           net stop spooler
                           del /Q "%windir%\system32\spool\PRINTERS\*.*"
                           net start spooler
                           exit

  • Uninstall printer-related software from Control Panel > Programs and Features
  • Open an Elevated Command Prompt (search for "cmd.exe"; right-click and select "Run as administrator")

            exit
