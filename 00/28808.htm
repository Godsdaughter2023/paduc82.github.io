In order to achieve the task function to be repeated you need to change a task to a recurring task in Outlook 2013.

Follow the steps given below in order to change the task into recurring task.

1. Open the task you want to change to a recurring task.

2. In the ‘Task’ tab > click on ‘Recurrence’ group.

3. In the ‘Task Recurrence’ window > check the option ‘Regenerate new task [ ] week(s) after each task is complete. Also chose the appropriate option in that window as per your requirement.

Note: Recurring tasks appear one at a time in the task list. When you mark one occurrence of the task complete, the next occurrence appears in the list.
