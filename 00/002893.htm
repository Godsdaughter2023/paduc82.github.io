kevinwhitect say: The "fix" is quite simple, and I found it searching through other threads.

Do a Windows Update.

If needed, download Classic Shell, set it to the Windows 7 Start Menu, and search the programs and files for Windows Update.

Then, do a Windows Update.

Cortana will work immediately, but the start menu command in Classic Shell will NOT pop the Windows 10 Start Menu window.

Removing Classic Shell returns the Windows 10 Start Menu to full functionality.
