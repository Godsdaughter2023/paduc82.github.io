Hi,

Take a look at this page: http://www.slipstick.com/outlook/rules/outlooks-rules-and-alerts-run-a-script/

You could use VBA code like this:

 

Private Declare Function ShellExecute _
    Lib "shell32.dll" Alias "ShellExecuteA" ( _
    ByVal hWnd As Long, _
    ByVal Operation As String, _
    ByVal Filename As String, _
    Optional ByVal Parameters As String, _
    Optional ByVal Directory As String, _
    Optional ByVal WindowStyle As Long = vbMinimizedFocus _
    ) As Long

Sub EmailProcessor(Item As Outlook.MailItem)
    Dim strSub As String, strOrderNum As String, strURL As String
    Dim lSuccess As Long
   
    strSub = Item.Subject
    strOrderNum = Replace(strSub, "Order ", "")
    strOrderNum = Replace(strSub, " has been delivered", "")
    strURL = "www.mysite.com/EmailProcessor.aspx?oid=" & strOrderNum & "&status=delivered"
    lSuccess = ShellExecute(0, "Open", strURL)
End Sub

Then in Outlook, do Tools > Rules and Alerts > New Rule
Start from a blank message > Check messages when the arrive > Next
from people or distruibution list (and add the sender's e-mail address)
with specific words in the subject (and add " has been delivered") > Next
run a script (and select the EmailProcessor script from the list) > OK > Next
Next
Specify a name for the rule
Finish.
