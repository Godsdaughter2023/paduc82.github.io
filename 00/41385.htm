a. user say:

Why did you have to install a new hard drive?

If chkdsk runs automatically on a reboot there is a reason - either Windows has detected the volume is "dirty" or a chkdsk has been scheduled to run by you on the next reboot.

It could take the chkdsk a very long time to run (several to many hours) depending on the size of the volume being checked, the amount of data on the volume, the speed of your system and what the chkdsk finds to do.

Interrupting the process will not keep it from running again the next time you restart and may make things worse so it is best to wait it out and let it complete naturally.

When Windows starts it checks the "dirty bit" on all volumes to see if a chkdsk with error correction needs to run and if a volume is dirty Windows will automatically run the chkdsk with error correction.  The only way to clear the dirty bit on a volume is to let a chkdsk with error correction do it for you by running to completion.

The dirty bit can be set on a volume from an aborted restart, ungraceful shutdown, power failure, power interruption, using the reset or power button, battery removal, etc.  Have any of those events occurred?

You can query the dirty bit on a volume from the command prompt window with Administrator privileges.

Click the Start orb and in the box enter the following:

cmd

Right click cmd.exe and choose to Run as Administrator:


To query the dirty bit on drive C, in the Command Prompt window enter the following command:

fsutil  dirty  query  C:

Sample output should be one or the other of:

Volume C: is dirty
Volume C is NOT dirty

You should check each of your volumes or each of your hard drives (C, D, E, etc.).

Type 'exit' to close the command prompt window and report your results.

Is the volume dirty or is it not?

Windows might also run a chkdsk on reboot if you scheduled one to run so if that is the case you can remove that by entering the following command from your Command Prompt window which will remove any scheduled chkdsk on all voumes:

chkntfs   /d

a. user say:

Hello,

 

Welcome to Microsoft Community. I will certainly help you with the issue that you are facing.

There is no tool or setting in Windows where you can set the priority, if you are using multiple audio devices. However, as a work around, you may try to set the TV as a default playback device. So when TV is connected, it will be the default device and when you disconnect it Windows will automatically choose Headphones if they are plugged in the headphone jack and not the USB jack.

To set TV as a default device.
  a. Right-Click on Volume icon in the Task bar notification area and select ‘Playback devices’.
 b. See if the headphone is detected when plugged in.
  c. If it’s not detected, right-click on the blank area and select “Show Disabled Devices”.
 d. If it gets detected, select TV and select “Set Default”. Check if it helps.

Hope the information helps. Let us know if you need further assistance with Windows related issues, we’ll be glad to assist you.
