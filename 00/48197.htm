anannya podder say:

Hi Morne,

 

 

Temporary files are files that are stored and used in the system and/or user TEMP and TMP folders temporarily while a task is being performed.

Well if you want to delete the temp files then try these steps:

 

a. Press Windows key + R on the keyboard. Type %temp% in the run box and hit Enter.

b. Press Ctrl+ A to select all the temp files and press Shift+Delete to delete the temp files.

 

Once you are done deleting the temp files you can initiate the download and verify the issue status.
