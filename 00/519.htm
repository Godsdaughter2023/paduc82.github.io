Excel macros: Stripping Domain names from a list of users within a - Microsoft Community

steven andrews say:

Excel macros: Stripping Domain names from a list of users within a column

Hi all,

 

Is it possible to write a macro for an Excel spreadsheet that will strip out a domain name and the backslash from a list of users?  The format is domainname\username.  I'd like to be left with just the username.  Making these changes manually is taking a very long time and there must be a way to ensure that it doesn't have to be done manually.

Any pointers appreciated

rick rothstein, mvp - excel say:

Assuming you have selected the cells that you want to process, this simple macro will do what you want...

Sub LeaveOnlyUsername()
    Selection.Replace "*\", "", xlPart
End Sub

You can also to what you want without using a macro. Select the cells you want to process, press CTRL+H to bring up the Replace dialog box, type *\ (that is just the two characters asterisk backslash) into the "Find what" field, leave the "Replace with" field empty, then press the "Options>>" button to reveal all the options if they are not already displayed and make sure the "Match entire cell contents" checkbox is not checked, then click the "Replace All" button.

For example:

Option 1: Expensive Item - higher charges - more insurance - more expensive shipping

 

Option 2: Cheap Item - lower charges - less insurance - cheaper shipping

I have two spreadsheets at the moment, but was wondering whether there is a way we can combine them into one, but have certain cells (e.g the additional charges) change depending on the drop down selection.

Sorry if this does not quite make sense.

hansv mvp say:

One tab = one worksheet; there is no way to have two worksheets in one sheet tab.

But you can create formulas whose return value depends on the selection in a dropdown list.

For example, let's say that the Expensive item/Cheap item dropdown is in cell B1.

The cell for shipping could contain a formula such as

=IF(B1="","",IF(B1="Expensive item", 50, 25))

Similar for the rest. So you'd need only one worksheet/tab.

andreas killer say:
At first a short note:
There are 2 kind of controls in Office: "Form Controls" and "ActiveX Controls".
Form Controls are very easy, you can link them to cells and they are ready to use.
ActiveX Controls are more powerful, but you have to write VBA code to make them work (in many cases).

Back to your question:

Right click the button and choose "Format Control..."
In the dialog in tab Control click inside the "Cell link" field
Select cell A1 and click Ok

That's it. For other controls read also this article:
https://support.microsoft.com/en-us/help/291073...

Andreas.
