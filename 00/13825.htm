justin cavida msft say:

Hi MK,

When you upload a folder via portal or copying to your OneDrive folder and sync, it won't actually move the folder, it will just copy your folder to your OneDrive and treat it as new. As a result, the date created and date modified will change to when you upload/copied the folder.

In order to retain the original dates, you must move the folder by dragging it to your OneDrive folder. This will retain the original date created and date modified of your folder.

Furthermore, you can vote for this entry in OneDrive UserVoice for the related team to know your demands. It is the best way to improve the products.

jim gordon mvp say:

A dictionary is a pretty simple thing. It's just a text file that has one word on each line (designated by a paragraph mark), each word representing the correct spelling of a word. These text files have a .dic file extension and need to be saved with particular encoding. Any list of words can be saved as a dictionary file.


There are loads of dictionaries out there. Some are for languages. Others for special fields (medical, legal, scientific). Most are free, but some you have to pay for. The big headache is trying to find one that already exists. Search engines like Google, Bing, and Yahoo have had their effectiveness neutralized by advertising and scam sites that flood the search results with useless garbage. You literally have to spend hours wading through the search results to find the one or two useful results.


I gave it a shot, and within about 1/2 hour found this particularly useful site. It offers many language dictionaries, including Afrikaans:

http://www.winedt.org/Dict/


Go to the site, and click the UNICODE download for Afrikaans. You'll get a zip file. Double-click the zip file in Finder to expose the unzipped Za folder. Inside that folder is a file called za.dic, which is the dictionary file we're after. Move Za.dic file to ~Users:YourUserName:Library:Preferences:Microsoft:Office 2011:    and be aware that if you are using Lion that the Library folder is hidden. see http://notebooks.com/2011/07/30/how-to-unhide-the-library-folder-in-lions-home-folder/


In Word, go to Word Preferences and choose the Spelling and Grammar preferences. Where it says Custom dictionary, click the Dictionaries button. In the resulting dialog, click the Add button and choose Za.dic. Click OK to close the dialog.


That should do the trick. Search the Answers Word forum for the word dictionary and you'll find some threads that go into great detail about working with custom dictionaries, so I won't rehash all that here.
