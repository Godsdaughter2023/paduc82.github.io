cmdrkeene say:

Since you disabled that entry in Task Manager, it will never actually run (or even try to run).  Although even if you re-enabled it, it wouldn't run since the actual file itself got deleted.  So even though it won't affect system performance or anything, I can still appreciate your desire to clean up this useless entry from the list.  And that is, indeed, typically handled in the registry. Or it may be in the Startup folder.  Task manager lets you know where to look:

Step 1: Figure out where the startup item is coming from

If you right-click a column header in Task Manager on that Startup tab, one of the options you can turn on is "startup type" and it will tell you where the the startup item comes from.

If the startup item type is "registry", use these steps to remove it:

  1. Use the Run box to run regedit, and then navigate to this location (if you have the latest updates to Windows 10, you can copy/paste this into the Registry Editor "address bar" and hit enter to go there quickly).

    Computer\HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

  2. In the list of keys, find the one related to iSkysoft and delete it.

  3. If the startup item was configured to run for any user (not just when you login), you may need to go to this section of the registry to remove it from there (only difference is "local_machine" instead of "current_user").

    Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Run

If the startup item type is "folder", use these steps to remove it:

  1. Use the run box to run shell:programs\startup and delete the icon
  2. If the startup item was configured to run for any user (not just when you login), you may need to run this instead:  %programdata%\microsoft\windows\start menu\programs\startup
