hansv mvp say:

You were almost there - you should use FIND(".pdf", E124)+3 instead of FIND(".pdf", E124)+4. With +4 you include the space after the file name, and this messes up the rest.

Here is a single formula to return the file name, assuming that it doesn't contain spaces:

=SUBSTITUTE(RIGHT(SUBSTITUTE(LEFT(E124,SEARCH(".pdf",E124)+3)," ",REPT("@",255)),255),"@","")

Hi Anne,

There are two ways to save a document in Microsoft Word. First is Save, this procedure will save the file in a default location, usually the Documents folder. Second option is Save As, this will ask you to name the file before you save it. This will also save the file in a default location.

To look for the saved file, type Word on Cortana's search field on the lower left part of the screen, next to the Windows button. Then click Word 2013. Once Word 2013 appears you will see the lists of saved files or documents on the left part. After that choose the document you wish to open.
