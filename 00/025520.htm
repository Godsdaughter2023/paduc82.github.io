doug robbins - mvp office apps & services (word) say:

I cannot find any way in the document.xml to control the size so I think that only thing that you can do is use some vba code to set the size of the Content Control.  If you use the tag "Picture" for each of the controls that you want to be sized and the use the following code in the ThisDocument object, each of the controls so tagged will be set the dimensions that are inserted in parentheses after the InchesToPoints

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)

Dim acontrol As ContentControl

For Each acontrol In ActiveDocument.ContentControls

    If acontrol.Tag = "Picture" Then

        With acontrol.Range.InlineShapes(1)

            .Height = InchesToPoints(3)

            .Width = InchesToPoints(3)

        End With

    End If

Next

End Sub
