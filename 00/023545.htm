You could force the task pane to close when you open that workbook using the Workbook_Open() event handler.  Here's the code:

Private Sub Workbook_Open()
    Application.CommandBars("Task Pane").Visible = False
End Sub

With Excel 2003, it's easy to put the code into the proper place in the workbook:  Open the workbook, then right click on the small Excel icon just to the left of the File option in the menu toolbar.  Choose [View] code from the list that will appear.  Copy the code above and paste it into the code module that was presented to you.  Save the workbook.

The task pane should be forced "closed" when that workbook is opened in the future.

sharonmary say:

Hi JC555,

You cannot delete the final paragraph marker after a table. If it just so happens that the paragraph marker slips on to the next page, then there is a little trick that I learned eons ago.

You'll need the Show/Hide characters switched on to do this:

If you don't have them switched on already, click the Show/Hide button in the toolbar on the Home tab (it's the one with the ¶ symbol in it) to display your document formatting, then:.

1. Select the final paragraph marker

2. Right click on it and choose Paragraph... from the pop-up menu

3. Under the Spacing category, make sure you have 0 in Before: and After:

4. Click OK if you made any changes to the Spacing

5. With the paragraph marker still selected: click on the little arrow in the Font Size button in the Font category and type 1 (you won't find it to select in the list)

    >>> This will ensure that the font is the smallest size possible. With a bit of luck it will make the rogue paragraph marker jump on to the preceding page.

 

>>>

 

All the best,

Sharon

 

Sharon Roffey
Queensland, Australia
