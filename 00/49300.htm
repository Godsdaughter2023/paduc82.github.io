greg carmack - mvp 2010-2020 say:

The error was reminding you that an Upgrade must be run from the desktop and not from booted media.  If this is reliable Windows 10 media created from Windows 10 Download site to Create Media then open the stick in Explorer, right click Setup, Run as Administrator and you're on your way.   If booted it would Clean install which will not Activate until you first do the in-place Upgrade and confirm it's Activated. That means you have a license permanently tied to that hardware which will reactivate on re-installs.  You can even go back to Windows 8.1 which is not affected by Windows 10, as long as you delete the Windows 10 partition first.  Then you can go back to WIndows 10 at any time too. 

It was good you had the Windows 8.1 backup image.  I would also have your files backed up separately as a hard drive can die at any time.  If that is an image of preinstalled Windows 8.1, it likely still has all the bloatware which throttles performance and can affect the Upgrade's chances. 

To improve the Upgrade's chances turn off everything at Startup:

In Windows 8.1, right click Task Bar to open Task Manager, uncheck everything in Startup Tab, Apply, Reboot.

Uninstall any anti-virus except Defender.  Bloated AV's like Norton, MacAfee, AVG and Avast need to have special tools run to remove them from Uninstallers (removal tools) for common antivirus software - ESET Knowledgebase.

Stick with built-in Defender in Windows 10 for best performance and adequate protection.  If you get infected add the inexpensive Real Time protection from the best-in-class free malware scanner Malwarebytes .  I'd also periodically scan with SUPERAntiSpyware or the more powerful AdwCleaner, also best-in-class and free.

van_m say:

Hi IvaniniBarduchi,

 

You can try this command on the classic version of Skype.

 

  1. Sign in to the Skype application.

  2. Go to one of your Skype contact’s messaging field.

  3. Type in the command “/showplaces”and send it as a message. This will show if you are logged in to other devices / endpoint

  4. Sign out of the Skype application and then sign back in.

  5. Go again to the messaging field of any of your Skype contacts and Type in the command “/remotelogout” and send it as a message. This will log you out on other devices.

 

Note: The command wouldn’t appear as a message on the other end user as it is a special command.
