Sub FindXinColA()
    Dim rngFound As Range
   
    Set rngFound = Columns("A:A").Find( _

                        After:=Range("A" & Rows.Count), _
                        LookIn:=xlValues, _
                        LookAt:=xlPart, _
                        SearchOrder:=xlByRows, _
                        SearchDirection:=xlNext, _
                        MatchCase:=False, _
                        SearchFormat:=False)
   
    If Not rngFound Is Nothing Then 'the search term was found

    End If
End Sub

Function FindFirstRowInCol(vCol As Variant) As Long
' Function returns the row number of the first populated row
' in the parsed column vCol.
' If the whole column is empty the function returns -1.
'
    Dim rng As Range
   
    Set rng = Columns(vCol)
    If rng.Cells(1).Value <> "" Then
        FindFirstRowInCol = 1
    Else
        FindFirstRowInCol = rng(1).Cells.End(xlDown).Row
        If FindFirstRowInCol = Rows.Count Then
            If rng.Cells(Rows.Count).Value = "" Then
                FindFirstRowInCol = -1
            End If
        End If
    End If
End Function

I think your question is (or was) how to find the first occurance of "x" after the first cell of a range?  If I remember your code from your previous posts, you use P4 as a string containing the value of the cell P4 (despite my recomendations not to do this!), and P4 contains a reference to some cells.  e.g. P4 = "B2:B10".  So:

will indead return the first cell of whatever range is returned by Range(P4).  But note that there's no need to select that cell.  That will just slow your code down.

 
Now, here's a question for you.  Do you:

a) want to find the next occurance of "x" AFTER the first cell in your range anywhere on your worksheet?,

or

b) want to find the first occurance of "x" WITHIN your range (which could include the first cell in that range)?

 

The VBA code: 

    Set rngFound = Range("B2:B10").Find( _
                        What:="x", _
                        After:=Range("B10"), _
                        LookIn:=xlValues, _
                        LookAt:=xlPart, _
                        SearchOrder:=xlByRows, _
                        SearchDirection:=xlNext, _
                        MatchCase:=False, _
                        SearchFormat:=False)
 

will find the first occurance of "x" WITHIN the range "B2:B10".  Note how by using the LAST cell in that range (B10) for the After arguement, means the Find straight away loops to the first cell in the range (B2) and looks there first.  If B2 contained "x", the above would find it before any other "x" in the range.  But if we set After:=Range("B2") then it would check all the other cells in the range (i.e. B3:B10) first, before looking in B2.  Make sense?

 

You used Cells.Find, which will give the result described in (a) above: the next occurance of "x" AFTER the ActiveCell, anywhere on the sheet (which could end up being BEFORE the ActiveCell, depending on what is on your sheet!).

 

Also, you need to protect for the event that Find doesn't find what you were looking for.  That is why you should assign the output of Range.Find to a Range object:  Set rngFound = Range("B2:B10").Find(...  and then you can test to see if rngFound is nothing:

    If Not rngFound Is Nothing Then

        ....do your code...

 

Make sense?

 

So I think you need:

either a)

    Dim rngFound As Range
    Set rngFound = Range(P4).Find(what:="x", After:=Range(P4).Cells(1, 1), _

        LookIn:=xlFormulas, LookAt:=xlWhole, _
        SearchOrder:=xlByColumns, SearchDirection:=xlNext, _

        MatchCase:=True, SearchFormat:=False)

 

which will find the first occurance of "x" WITHIN Range(P4), AFTER the first cell in Range(P4),

 

or b)

    Dim rngFound As Range
    Set rngFound = Range(P4).Find(what:="x", _

        After:=Range(P4).Cells(Range(P4).Rows.Count, Range(P4).Columns.Count), _

        LookIn:=xlFormulas, LookAt:=xlWhole, _
        SearchOrder:=xlByColumns, SearchDirection:=xlNext, _

        MatchCase:=True, SearchFormat:=False)

 

which will find the first occurance of "x" WITHIN Range(P4), since, Range(P4).Cells(Range(P4).Rows.Count, Range(P4).Columns.Count) gives the LAST cell in Range(P4).
