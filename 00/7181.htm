grschinon say: This is a classic bug.

If there are too many audio files in memory that can potentially be used as ringtones, the phone is unable to populate the list with them normally.

If the file you want to use as a ringtone is on your memory card then copy or move it to the phone's memory first.

Now eject the memory card and you'll be able to change the ringtone because the list will be populated normally. Then re-insert the memory card.

You cannot literally "upgrade" a system from 32 bit to 64 bit ... it must done as clean install ... and yes, since you will need to purchase a license in order to activate your new computer, I certainly would install 10 directly, rather than installing an older OS and then upgrading to 10.

The activation doesn't distinguish between 32 bit and 64 bit, the difference is between home and pro ... a machine is digitally activated running either home or pro ... and if it has been activated to run pro, you can downgrade to home, but not visa versa.

If a motherboard is replaced with a like one because of failure of the old one, Microsoft will allow the existing activation to continue, but if it's deemed to be an upgrade replacement of the motherboard, then, it's a new computer, requiring a new activation.

steve rindsberg say:
I've tested several programs that claim to be able to convert PDF to PowerPoint.  Some do a better job than others, but none that I'm aware of will convert the PDF, even one created in PPT originally, back to anything like a properly constructed PPT file.  

You'll get a page of graphics (some raster, some vector depending on the source and the program used for the conversion) that LOOK like the PDF, but bulleted text, for example, won't necessarily turn into a normal PPT body text block.

Getting images to PPT or even entire PDF pages *as* images into PPT is quite simple and very effective, so long as the images in the PDF itself are of reasonable resolution.

The trick is to zoom in on the PDF in Reader before using the graphics snapshot tool to choose the area to be copied.

The snipping tool only allows you to capture what's visible on the screen.

If you use the snapshot tool in Reader, you can zoom way in first; the snapshot tool will scroll the zoomed document so you can capture a huge chunk of a "virtual" screen.
