here's the solution you need:

1. go to Device Manager

2. find DISPLAY ADAPTERS -> click on Intel HD 615 

3. click Driver-> update driver-> Browse my computer for driver software-> Let me pick from a list of available drivers  from my computer

4. select & install Microsoft Basic Display Adapter

now finally you can install intel's latest driver 

Go to intel's website, download & install the driver for HD615. 

Good luck!
