Hi Michael,

Thank you for writing us.

I understand that that you are trying to share files by creating DLNA server on his Windows media player and you are unable to see the files on other computer though it gets connected to the server without any problem.

The issue could be caused if Internet Protocol security (IPsec) is enabled on the computer. We won't be able to use the media sharing feature if Internet Protocol security (IPsec) is enabled on the computer.

Note: Windows Media Player has an option under ‚ÄúStream‚Äù to ‚ÄúAutomatically allow devices to play my media‚Äù which has the same effect. You have to be careful if you decide to use this option on a laptop computer that you use outside the home.

Important: To Set Up the DLNA Media Client in Windows Media Player.

You need to make sure that the player PC is on the same network as the media server. Now if you go to another Windows PC and call up Windows Media Player, you should be able to see, in the left column, under ‚ÄúOther Libraries‚Äù, the shared Library that you just set up. You may have to right click ‚ÄúOther Libraries‚Äù and select ‚ÄúRefresh‚Äù if you don‚Äôt see the content right away.

Windows 8 DLNA Media Server does not have any access control. It has no logins and no permissions.

So whatever you make available through your DLNA server is visible to everyone. It is best to use the Media Streaming Options screen to control who you want to share with if you plan on using that computer outside the home.

Please check the Internet Protocol security settings, as it must be set to Automatic so change the same manually.

To change the IPS settings, follow the steps below:

  1. Press Windows Key + R key type control and click Ok.
  2. Click on Network and sharing center.
  3. Click on the internet access type then go to properties.
  4. Select Internet protocol version 4 then click on properties.
  5. Change the IPS settings manually and save it.
  6. Restart the computer and test the issue.
