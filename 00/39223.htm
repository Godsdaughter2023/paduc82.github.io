arminatorx say:

You don't "need" a Freesync monitor. The Xbox One X will run perfectly fine on any "normal" monitor. However, it won't use the increased refresh rate. 

Freesync only eliminates an effect called "tearing", which happens when the console draws a newly rendered image before the TV has finished drawing the previous image on the screen. 

If the image is scrolling horizontally, it will appear as if someone cut the image horizontally, and shifted the other part a bit sideways. So for example a tower, chimney or lamp post or any vertical edge will have a cut in it and be shifted. 

A monitor/TV that works with Xbox One X's Freesync would be the Samsung Q9FN.
