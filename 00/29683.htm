Here are explicit details for the creation of a simple specimen signature, based on my own, in HTML. It shows two tables, 14% of the width on the left with an image (can be jpg, gif, etc.), which could be a logo, picture, avatar, etc. and has text in the rightmost 86%. Open Notepad and copy all the text into it. Now edit it to suit your personal details, i.e. change the table sizes (width=) image (<img src….) and email link (<a href….) lines. Also change other attributes (color, text size, font) or remove lines. Save the file with a meaningful name, e.g. signature01.html, preferably in its own folder, e.g. c:\signatures. Also, any pictures you intend to use must be saved in this folder.

 

Now open the Windows Live Mail program, hold down both Ctrl and Shift and press O for Options then the Signatures Tab > Put ticks in ‘Add signatures to….’ and optionally ‘Don’t add….’ then click New > File (at the bottom) and Browse to the name of your file (make sure the drop-down menu points to HTML files) then Apply/OK your way out. The HTML code can be edited by right-clicking the filename and choosing Notepad and double-clicking the filename will display your signature in a browser window. Finally, send an email to yourself to make doubly sure it displays exactly how you want it to, e.g. the size of the image, color, wording, etc. Here is the specimen code, which is about 1KB in size:

<table width="100" border=0>

<TR><TD></table>

<td valign=top width="86%">

<B><font face="times new roman, verdana, arial" color=blue size=5>

John William Smith</B><BR>

<font color=black size=3>

Hi Priesteye,
 
When you create a shortcut to a shared folder or computer on a network (also called mapping a network drive), you can get to it from Computer or Windows Explorer without having to look for it or type its network address each time.

1.  Open Computer by clicking the Start button, and then clicking Computer.
2.  Click Map network drive.
3.  In the Drive list, click a drive letter.
You can choose any available letter.
4.  In the Folder box, type the path of the folder or computer, or click Browse to find the folder or computer.
To connect every time you log on to your computer, select the Reconnect at logon check box.
5.  Click Finish.

Your computer is now connected, or mapped, to the network drive.

Note: If you can't connect to a network drive or folder, the computer you're trying to connect to might be turned off or you might not have the correct permissions. If you can't connect, contact your network administrator.

Spraying compressed air into an enclosed space like a drive is a bad idea. It can create clumps which can get stuck on mechanical parts (either the laser track or the disc eject mechanism) and cause problems. If there's any problems reading the disc the game will halt and the console will tell you to clean the disc.
