Macro to select and reformat beginning of line with a colon in it - Microsoft Community

gabor5454 say:

Macro to select and reformat beginning of line with a colon in it

I would like to use a macro (or VBA script) to select from the beginning of a line to colon- to change the first part of the line to bold, but leave the text after the colon unbolded 

i.e. change the body parts but not "normal" in the following text

Neck: Normal
Chest: normal
Abdomen: normal.

(if I use "*:"  in find and replace then the "normal" in the previous line is also changed)

thanks in advance

RG
jay freedman say:
In the Replace dialog, first click the More button and check the box for "Use wildcards".

Then put the following expression in the Find What box:

([!^13]@:)

Put this in the Replace With box and then press Ctrl+B:

\1

Click the Replace All button.

The Find What expression has the following meaning: The code ^13 represents a paragraph mark. The exclamation before it means "NOT", and the [ ]@ means "any repetition of characters that match the expression in the brackets". So the part of the expression  [!^13]@ means "any sequence of characters that are not paragraph marks". The colon just represents a colon following the sequence. Finally, the parentheses around the whole thing let you use the \1 in the Replace With box to represent "the part of the document that matches the expression".

There's lots more about wildcard searches at http://www.gmayor.com/replace_using_wildcards.htm.

If your sub-folders were created <prior to> when you backed up your PST file, there would be absolutely no way those would be missing if the correct backup file was used.
