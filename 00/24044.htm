a. user say:

Hi Dominic,

Thank you for posting in Microsoft community.

I suggest you to follow the methods below.

Method 1: I suggest you to run Printer troubleshooter and check if that helps.
Fix printer problems.
http://windows.microsoft.com/en-us/windows-10/fix-printer-problems

Method 2: Perform Clean boot

Place your computer in clean boot state and try installation printer driver in compatibility mode.

How to troubleshoot problem by performing clean boot in Windows: https://support.microsoft.com/en-us/kb/929135

Note: Once you are done with the troubleshooting, perform the steps in “How to reset the computer to start as usual after troubleshooting with clean boot “to reset computer as usual.

Method 2: Compatibility mode

I would suggest you to uninstall and install the latest available drivers on manufacturer's website in compatibility mode.

Follow the steps below on how to install driver in compatibility mode:
1. Download the driver from the manufacturer’s website and save it on your local disk.
2. Right click on the setup file of the driver and select “Properties”.
3. Select “Compatibility” Tab.
4. Place a check mark next to “Run this program in Compatibility mode” and select operating system from the drop down list.
5. Let the driver install and then check the functionality.
