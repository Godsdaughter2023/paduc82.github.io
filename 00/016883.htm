Hi,

use the following event macro:

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
Dim rngRange As Range
  Cancel = True
  Set rngRange = Me.Range(Target.Address & ":" & Me.Cells(Rows.Count, Target.Column).End(xlUp).Address)
  rngRange.Find(What:=Target.Value, LookIn:=xlValues, LookAt:=xlWhole).Select
End Sub

 

Right click on sheet tab > choose View code > copy the above code in the codemodul

a)   To do this, click  Start, click Programs.

b)   Point to Accessories, point to  System Tools.

c)    Click Internet Explorer (No Add-ons).

 

If you start Internet Explorer with add-ons disabled, you may find that this resolves the issue. Then, you can locate and disable the add-on that conflicts with Internet Explorer. Please refer this article for help.

Internet Explorer add-ons: frequently asked questions

 Method 2:

Register the following files:

 

¯ regsvr32 ole32.dll

¯ regsvr32 oleaut32.dll

¯ regsvr32 actxprxy.dll

 

Here’s how:

 

a.    Go to Start and type cmd.

b.    Right-Click on cmd and select “Run as administrator”.

c.    Type “regsvr32 ole32.dll” without the quotes in the command prompt and press Enter.

d.    Repeat the same for the remaining two files.

e.    Close command prompt.
