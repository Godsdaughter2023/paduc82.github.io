Need help writing a COUNTIF() formula for visible cells only

 I am trying to figure out if it is possible to write, any formula, that can perform a special kind of COUNTIF().

For example, I have data in Range(F3:AP3), F3, H3, J3, L3 etc., can have PLAN or ACTUAL G3, I3, K3, M3 etc. have dates.

At any point, some of these columns will be hidden and that depends on user selections somewhere else in the worksheet.

I need to write a COUNTIF(), or any other formula, that would count the number of VISIBLE cells between F3:AP3 that contains the words ACTUAL.

 

Is this possible?

 

Thanks.

jeeped say:
Dealing with hidden rows is generally handled with SUBTOTAL() but SUBTOTAL() does not like dealing with hidden columns and it does not accept a range modified with the TRANSPOSE() function.

The best solution may be a User Defined Function (i.e. UDF). Here is a quick one,

Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert,. Module. Paste the following into the new pane titled something like Book1 - Module1 (Code), 

Public Function udfCountIf_VisibleOnly(vr As Range, sc As String, _
    Optional cs As Boolean = False) As Long
    Dim i As Long, r As Range
    For Each r In vr
        i = i + (Abs(Not r.EntireColumn.Hidden) * _
                 Abs(LCase(r) = LCase(sc)) * _
                 Abs(Not cs Or r = sc))
    Next r
    udfCountIf_VisibleOnly = i
End Function

Tap Alt+Q to return to your worksheet.

Usage:

=udfCountIf_VisibleOnly(<range of cells>, <criteria>, <optional case sensitivity>)

For your purposes try something like,

=udfCountIf_VisibleOnly($F$3:$AP$3,"ACTUAL", TRUE)

or,

=udfCountIf_VisibleOnly($F$3:$AP$3,"actual")

Edit: edited out Debug.Print.
