a. user say:

Hi Bojan,

 

Thank you for your response with the current update on the issue.

 

Before performing the steps below, you can create a System Restore point and then perform the troubleshooting steps. So that you can restore back to best known configuration on Windows 10.

Create a system restore point:

http://windows.microsoft.com/en-us/windows-10/create-a-system-restore-point

I recommend you to disable the User Account Control notification from the registry editor. 

1. Press Windows Key + R Key.

2. Then type "regedit" in the Run box and hit on Enter Key.

3. Follow the path below: 

HKEY_LOCAL_MACHINE > SOFTWARE > Microsoft >Windows > CurrentVersion > Policies >System

4. In the right side pane, search for EnableLUA and double click on it.

5. Then modify its value from 1 to 0.

6. Close the registry editor. 

7. Restart your computer.

Registry disclaimer:
