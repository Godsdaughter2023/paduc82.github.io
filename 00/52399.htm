considerate_guy say:

I agree - the video creation tools in the Photos App (aka the "Story Remix" feature) basically let you pull together some photos and video, add a single song soundtrack and create a video that you can quickly share. But it's not going to be a full-fledged video editing suite. It's got even fewer features compared to Movie Maker, like you said.

I would not consider it a replacement for Movie Maker. It's more like a reimagining of the minimum, easy-to-use set of features that will let a user create a "slideshow" type video. It's more akin to what you can do on your phone using Google Photos where you select some clips, trim them, add a song, add a title and then share your result. It made for a cool demo when "Story Remix" was first showcased with dinosaurs and fireball soccer shots, but in practice it just doesn't give that level of fun or amazement. It certainly isn't trying to be the "new and improved Movie Maker with all the old features and a few more". It's a different animal with a subset of the features that Movie Maker had to make it even simpler for a novice to create and share a video. If that's what you want, that's what you got. If you want something akin to the older versions of Movie Maker, either keep using them or look for 3rd-party options like Pinnacle, Cyberlink, etc.

Hi Billy,

The simple way to disable sign-in so that you can just dive right in and start using your PC. Here’s how you do it.

  1. Right-click the Start button and select the Run option. Type “netplwiz” and hit enter.
  2. This will open the User Accounts window. Choose the one that you would like to skip the whole login and password option for. Uncheck the ‘Users must enter a user name and password to use this computer’ and click Apply at the bottom.
  3. Restart your system and you will see you no longer have to enter a password to start using it.

New operation system question, I am going to install a new operating system(aka windows 7 professional) alongside my original (aka windows vista), I have a dead monitor and use the FN keys a lot, will windows 7 recronize the FN keys? If the answer is NO how do I get it to recronize the keys? 
PS: I am getting Windows 7 Disk from Best Buy and it is just windows 7
cmdrkeene say:
Windows itself doesn't "see" the FN keys.  That's up to to a small processor built in to the keyboard itself.  For example, my laptop has a special FN key combination that adjusts the brightness of the screen, and another that turns the wifi on and off.  Although both of those affect Windows -- Windows itself doesn't 'know' it's happening.  Pressing those keys won't stop the screensaver for example.

Short story: if your computer has those keys, they'll work regardless of what operating system you have installed, because they operate outside of the OS's control.
