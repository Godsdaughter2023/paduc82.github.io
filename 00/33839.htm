Installing or downloading the same file on multiple users. - Microsoft Community

stardust747 say:

Installing or downloading the same file on multiple users.

If you install a game or download a file on multiple users, does it just share one file between the users or does it make duplicates for the different users and take up memory?
robertjwin say:

By "take up memory" I assume you mean "take up disk space". Anyway, the answer is that it depends on where the program's installed or the downloaded file is saved.

If every user installs the same program under their own user profile in %AppData% (C:\Users\username\AppData\Roaming) for instance, obviously there will be multiple copies. However if they all install to the same location in Program Files or another drive or wherever, overwriting the previously installed copy each time, of course there will be no extra space used. Admin users can also choose to install programs for all users simultaneously.

Downloads work the same way. If all users saves the same file to their own Downloads folders (C:\Users\username\Downloads), naturally there will be multiple copies. But if every user saves to D:\Downloads with the same file name... Well, I think you can figure it out by now. :)

An error 1075 means that the Dependencies for the Service may be afflicted.
