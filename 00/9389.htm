Excel for Mac 2016--need fast way to insert entire rows - Microsoft Community

margotzook say:

Excel for Mac 2016--need fast way to insert entire rows

I need a way to quickly insert rows. The toolbar function is ridiculously cumbersome (and located inconveniently with no way to customize it) and I found a list of keyboard shortcuts but this does not seem to have any shortcut.

bob jones aka: cybertaz say:

As alternatives to the keyboard shortcuts;

To insert a single row:

  1. right-click the Row Number below where you want the new row to be inserted & select Insert from the contextual menu, or
  2. click the number of the row below where you want the new row, then right-click anywhere in the selected row & select the Insert command, or
  3. just right-click any cell in the row where you want the new row inserted. That calls the Insert dialog where you can select the Entire Row option, then click OK or press return
To insert more than one row:
  1. select that number of Row Numbers, then use the #2 method, or
  2. select a range of cells in a column, right-click the range & do as described in #3
