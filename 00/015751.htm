Use a query to create list of dates and corresponding numbers.

SELECT DeliveryDates.F_Date_Required, Sum(IIf([XX].F_Date_Required<DeliveryDates.F_Date_Required,1,0))+1 AS DateNUM

FROM DeliveryDates, DeliveryDates AS XX
GROUP BY DeliveryDates.F_Date_Required;

Use a union query to join date with data.

SELECT DeliveryOrder.F_Date_Required, DeliveryOrder.F_Type, DeliveryOrder.[F_Work_Order_#], DeliveryOrder.Row, DeliveryOrder.DateNUM
FROM DeliveryOrder
UNION ALL SELECT NULL, NULL, DeliveryOrder.F_Date_Required AS [F_Work_Order_#], 0 AS Row, DeliveryOrder.DateNUM
