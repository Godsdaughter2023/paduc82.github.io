tina zhou msft say:

Hi MJbarentine,

The workaround for this issue is to type "+n " (n is the number of pages) after the &[Page] or &[Pages] code in the header or footer.  Excel will add this amount to the page number.  For example, if you want the first page to start at page 4, the header/footer code would be "Page &[Page]+4 of &[Pages]+4 ". It's important that the number of pages to add must not be the last character in the string that makes up the header/footer.  Even if you just include a space at the end, it will work. If you don't include the space, it would print something like "Page 5 of 54" (if there are 5 pages), appending the "4" rather than adding 4 to the number of pages. If you put a space at the end, it would print "Page 5 of 9".

Regards,
Tina
