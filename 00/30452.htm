Should be ok. Just make sure it has the same configuration (i.e. number of pins, size/design). This is very important for laptops.

bill.p. say: Hi there, thanks for your question.

If you have an OEM license for Windows 10 Pro, you can only use that on the PC it was preinstalled on. You won't be able to use it on the NUC as the license is tied to the motherboard, which is why the NUC keeps going back to Home edition.

If you'd like to upgrade the NUC to Windows 10 Pro, you'll need to purchase a retail license which can be installed on any PC:

https://www.microsoft.com/en-us/p/windows-10-pr...
