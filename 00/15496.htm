Assuming A2 and B2 contains real times, then this in say C2 should work fine:
=IF(COUNT(A2:B2)<2,"",IF(OR(A2>--"8:15 AM",B2<--"4:00 PM"),"Not Completed Time","Completed Time"))

If "numbers" align left in cells they are seen as  text by Excel and sort as text.

bernie deitrick say:

'For example, I want to copy the values  of  6th column (F) of sheets 1-20, in Workbook 1
'to  the corresponding 6th column ("F") in sheet 1-20 in Workbook 9.\

Sub TestMacro()
    Dim wkbk1 As Workbook
    Dim wkbk2 As Workbook
    Dim wsht1 As Worksheet
    Set wkbk1 = Workbooks.Open("Workbook 1.xls")
    Set wkbk2 = Workbooks.Open("Workbook 9.xls")

    For Each wsht1 In wkbk1.Worksheets
        With wsht1
            .Range(.Cells(2, 6), .Cells(Rows.Count, 6).End(xlUp)).Copy _
                    wkbk2.Worksheets(.Name).Cells(Rows.Count, 6).End(xlUp)(2)
        End With
    Next wsht1

End Sub
