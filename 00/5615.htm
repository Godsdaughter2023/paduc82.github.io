Private Sub Worksheet_Change(ByVal Target As Range)

    Case Is = "$B$4"

Get message "install OCR from CD" when attempting to scan. - Microsoft Community

judiromine say:

Get message "install OCR from CD" when attempting to scan.

I have a Lexmark all-in-one.  When I try to scan I get the message "install OCR from CD."  This in reference to the drivers CD.  I believe the MSE firewall is the problem.  How do I set up my scanner to pass through the firewall? 
kosh vorlon say:

Hi Judi,

 

MSE does not include firewalls or any type of firewall protection - so that cannot be the issue with the Lexmark.   MSE depends on Windows Firewall to provide that type of protection and would not have altered the Windows Firewall settings when it was installed.  I do not believe that Windows Firewall would allow all other functions on the Lexmark to work except scanning if the Lexmark were being blocked somehow (and it doesn't really block attached devices from functioning - especially not only some features of those devices).  MSE doesn't do the things you suggest may be causing this problem.  This is being caused by something else.

 

I believe OCR is software on the CD that provides the ability to do Optical Character Recognition if you have your Lexmark configured to convert the document to text during the scan.  Perhaps you have it set to use OCR and thus the message is being generated if you didn't install the OCR software, or the software isn't working and needs to be re-installed, or the printer has that capability but didn't come with OCR software which is needed to allow that feature to operate.  Normal scanning does not require OCR unless you are trying to scan into a program (for example, you have a document you want to scan into Microsoft Word and then perhaps edit or work with as a .doc or .docx file and so some type of OCR software is needed to convert what is on the document to text that Word can then work with).  Lexmark may or may not have come with such software so while it can naturally scan, conversion by OCR needs additional software.

 

For assistance, I recommend you contact Lexmark technical support starting here: http://support.lexmark.com/index?page=localeSelect&jumppage=home.

 

I hope this helps.

 

Good luck!
