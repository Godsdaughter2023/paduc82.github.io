Of course, this assumes that only single characters will exist between parentheses and that they will be the digits you wish to remove.

If there might be more than one character, you can use (*).

If you only want to remove digits, and there might be non-digits between the parentheseses, you will need to either use a VBA macro (to remove the characters in place), or use a formula in an adjacent column.

First be sure you are clear as to what you require.  In particular, consider the difference between deleting cells and clearing them.  And also that all you want to do is replace one particular string with another.


In its simplest form, a macro to do exactly what you request (and no more) might look like


==========================
Option Explicit
Sub ReplaceAndDelete()
With ActiveSheet
    .Rows(1).Delete
    .Range("D:J").Delete
    .Cells.Replace what:="16:00(5)", replacement:="16:00", lookat:=xlWhole
End With
End Sub

================================

palcouk say:

My understanding of MailStore, is that the archives can be opened in a number of email clients, and if neccessary replied to via that client using their Servers.

However your current IMAP provider is, may have a maximum size for the data store.

To move any Exchange data you would first need to create the matching folders in the target folder set, then copy the contents of a folder to the target folder.

You would need to wait for it to sync, before moveing on to the next folder, to repeat.

Allthough since MailStore can Archive immedietly 'on the fly' you could simply archive the contents, and if nessessary open the Archive to check.

Most data stores provided free by an ISP have no guarentee of uptime or content retention. There have been reported cases of major free mail providers losing all content, or the provider terminating the service.

If the IMAP account is terminated or reset all current content on the local device is lost, unlike a pop type.

I have in the past recovered data from an *.ost data file on the local device, when the provider terminated the service, but it required specialist software and was a tedious process.

One possible workaround:

Switch to Notes Page master and rearrange the notes page layout so that you have a slide image on the left and your notes on the right (you'll likely want to switch the notes page layout to landscape first in the Slide Setup dialog for the presentation).
