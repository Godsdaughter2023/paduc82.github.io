How do I change existing custom number formats using vba? - Microsoft Community

jbttx say:

How do I change existing custom number formats using vba?

I have several workbooks with custom number formatting that is applied in various places in several sheets in each.  The custom number format has text added to it like: "" 0.0% " of XYZ Cost & Contingency".  I need to loop through all the custom number formats, look for "XYZ" and replace it with "ABC", so the new format will be "" 0.0% " of ABC Cost & Contingency".

 

I have tried everything I can think of, even changing the xlsm file to a "zip" file and searching through the xml. No luck.  Nor can I find any vba examples that deals with modifying existing custom formats.  Any help or suggestions would be great!

 

Thanks!

jeeped say:

... it adds a new custom number format so now all the old ones are still there.  Any way to just modify them without adding a new one?  Or to delete the old ones?


The trick here is to store the old custom number formats while changing them, then delete them after they have been changed. If you delete them on-the-fly as you are changing them, a cnf in another cell may be removed before you have adjusted it.

Sub mcrReplaceCNFs()
    Dim stmp As String, vtmp As Variant, i As Long, rng As Range
    stmp = vbNullString
    For Each rng In ActiveSheet.UsedRange
        If InStr(1, UCase(rng.NumberFormat), "XYZ") > 0 Then
            stmp = stmp & rng.NumberFormat & "¶"
            rng.NumberFormat = Replace(rng.NumberFormat, "XYZ", "ABC")
        End If
    Next rng
    vtmp = Split(Left(stmp, Len(stmp) - 1), "¶")
    For i = LBound(vtmp) To UBound(vtmp)
        ActiveWorkbook.DeleteNumberFormat vtmp(i)
    Next i
End Sub
