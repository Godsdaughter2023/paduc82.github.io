Draw line macro - Microsoft Community

cousin excel say:

Draw line macro

Hi,
Can I have a macro drawing a horizontal line starting from the position of the cursor and to the right hand side with a length of some centimeters.
Thanks and regards,
Cousin Excel
chip pearson say:

Try the following code.

 

Sub AAA()
    Dim LL As Shape
    Dim WS As Worksheet
    Dim BeginX As Double
    Dim EndX As Double
    Dim BeginY As Double
    Dim EndY As Double
    Dim Centimeters As Double
    Dim Length As Double
   
    Set WS = ActiveCell.Worksheet
    Centimeters = 5 '<<< CHANGE 
    Length = Application.CentimetersToPoints(Centimeters)
    BeginX = ActiveCell(1, 1).Left
    BeginY = ActiveCell(1, 1).Top
    EndX = BeginX + Length
    EndY = BeginY
   
    Set LL = WS.Shapes.AddLine(BeginX, BeginY, EndX, EndY)
    With LL.Line
        .DashStyle = msoLineSolid
        .ForeColor.RGB = RGB(255, 0, 0) ' red
        .Weight = xlThick
    End With
End Sub

 

Change the line marked with <<< to the length of the line in centimeters. Change the LL.Line properties as desired.  This will draw a level, thick, red line from the top left of the active cell to a point 5 centimters to the right. You can manipulate the Begin and End variables to position the beginning of the line at different points of the active cell (e..g., the left, center, bottom, etc).

anna bon say:

Hi Leticia,

Encrypting an email message in Outlook means it's converted from readable plain text into scrambled cipher text. Only the recipient who has the private key that matches the public key used to encrypt the message can decipher the message for reading. Encryption applies to all messages sent out using Office Outlook. Let us try the steps below:

  1. In message that you are composing, click File > Properties.

  2. Click Security Settings, and then select the Encrypt message contents and attachments check box.

  3. Compose your message, and then click Send.

Let us know if this will help send out encrypted emails to your distribution list.
