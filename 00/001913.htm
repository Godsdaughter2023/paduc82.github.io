macropod say:

The behaviour of newspaper-style columns depends on how you've formatted the Section breaks that follow them, combined with the layout options for the document.

If you use a 'next page' Section break, the columns will occupy the full page but won't be balanced.

If you use a 'continuous' Section break, you can balance the columns by unchecking the 'Don't balance columns at the start of continuous Sections' option, under File|Options|Advanced>Layout Options, but then the columns will occupy only as much of the page as they require. Checking that option results in each column filling the page height before starting the next column - the same as they do if you use a 'next page' Section break.
