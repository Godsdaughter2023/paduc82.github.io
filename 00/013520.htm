davem121 say:
Hi Techit

The following code can be used to run an executable or batch file (you need to change the batch file name and the location of that file)

Private Sub Command1_Click()
Dim myPath As String
myPath = "C:\Program Files\mytool\MyFile.bat"
Call Shell(myPath , 1)
End Sub
