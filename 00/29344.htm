If you are seeing message like this:

No Disk, Exception Processing Message c0000013 Parameters 75b6bf7c 475b6bf7c 75b6bf7c

Until you can determine what the "real" problem is with your hardware, you can disable the pop up message by making a registry adjustment.

Before making registry changes, backup your registry with this popular free and easy to use tool:

http://www.snapfiles.com/get/erunt.html

Click Start, Run and in the box enter:

regedit

Click OK to open the registry editor and navigate to the following key:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Windows

On the right hand pane, check the value of the ErrorMode parameter.  

If the ErrorMode parameter is set to 0, double click ErrorMode and change the value to 2 (2 = disabled).

Click OK to save the changes, then click File, Exit to close the registry editor.

Reboot your system and check things now.

Setting the ErrorMode parameter to 2 stops the pop up message but will still log any hard errors to the Event Viewer log.

Read more about it here:

http://support.microsoft.com/kb/124873

In order to make a video, you need:

  1. Source files (videos, photos and music)
  2. A video editor
  3. A location to upload your video for sharing.

The exact answers depend on what it is you want to do exactly.  If you are using your own camcorder, camera or webcam, then you can use something like Movie Maker 2012 which is part of the free Windows Essentials download.  If you go to the Movie Maker page, there are a few mini-tutorials on how to use the program.


Using Movie Maker you should be able to edit your video and then upload it to sharing sites such as YouTube, Vimeo or OneDrive (formerly SkyDrive).


If instead you want to record from your webcam and use Skype, that is also possible. A final option would be to use a Windows App called Movie Moments, assuming you just want to share a short (1 minute) video.
