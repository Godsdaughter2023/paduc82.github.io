a. user say:

Three possibilities (there are very likely to be more!):
 a. Your Normal document (which Word uses as the starting point for new documents) is damaged. See e.g. http://word.mvps.org/Mac/MacWordNormal.html 
- it doesn't cover Word 2011 but may be helpful.
 a. In Format->Document->Layout, ensure "Vertical Alignment" is set to "Top" (actually, I do not think this would cause exactly the behaviour you describe, but it might)
 c. ensure that the format of the paragraph (or the paragraph style with which it is associated, or the following paragraph defined in the style) does not have spacing characteristics that are causing the problem

For (c), before pressing the return key, 
1. click Format->Style, and note the name of the style that is highlighted (I would guess it is Normal).
2. Click Modify, and look through the various settings for the paragraph layout for example:
    - if "Style for following paragraph" is different from the name of the style you are currently looking at, you may need to look at the settings for that style as well
    - Click on the Format dropdown at the bottom left and select Paragraph. Then look through the options there and see if any of the spacing is too large, modify that option and see if that fixes the problem. You may then need to try to ensure that the style modification is written back to the template - I think "Add to template" will do that, but if not I expect someone will jump in here and tell us.
 3. if nothing looks wrong in the style for the current paragraph, or the options for the style for the following paragraph, try looking at Format->Paragraph and looking for a spacing problem there.

kristine arp say:

Hello,

 

 

If you are trying to sort meeting invites, updates, and responses, there are other options to do it aside from filtering using flag status. Here are the three methods that you can use to filter meeting invites, updates, and responses.

 

Method 1: Using the Filter Feature.

 

Instead of the flag status, you can create a filter for the duration. This way you can filter all emails with meeting invitations, meeting responses, and meeting updates. Follow the steps below:

 

  1. Go to the View tab.
  2. Click View Settings.
  3. Hit the Filter button.
  4. Click Field.
  5. Select All appointments fields.
  6. Select Duration.
  7. In the Condition text box, select exists from the drop-down list.
  8. Click Add to List.
  9. Hit OK.
  10. Click OK to close the Advanced View Settings dialog box.

 

Method 2: Using Custom Search Folder.

 

It enables you to automatically filter all meeting invitations, updates, and responses from a specified mail folder easily. Here are the steps:

 

  1. Go to the Folder tab.
  2. Click New Search Folder.
  3. Scroll down and select Create a custom Search Folder.
  4. Click Choose.
  5. Enter a name for the search folder.
  6. Click Criteria.
  7. Go to the Advanced tab.
  8. Click Field.
  9. Select All appointments fields.
  10. Select Duration.
  11. In the Condition text box, select exists from the drop-down list.
  12. Click Add to List.
  13. Hit OK.

 

Method 3: Using Rules.

 

This will allow you to automatically filter all meeting invitations, updates, and responses from a specified folder to the other folder. Follow these steps:

 

  1. Click Rules from the Home tab.
  2. Choose Manage Rules & Alerts from the drop-down.
  3. Click the New Rule button.
  4. Select Apply rule on messages I receive.
  5. Click Next.
  6. Check the box beside Which is a meeting invitation or update.
  7. Click Next.
  8. Check the box beside move it to the specified folder.
  9. Click the word specified.
  10. Highlight a mail folder where you will move your meeting invitations, updates, and responses.
  11. Click OK.
  12. Click Next until you reach the last Rule Wizard dialog box.
  13. Check the option of Run this rule now on messages already in “your folder name”.
  14. Click Finish.
  15. Hit OK.

 

Let us know if you need additional assistance.
