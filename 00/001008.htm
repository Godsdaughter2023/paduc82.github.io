callkathy say:

There really isn't a way to do exactly what you want to do. However, I believe you could get to the same destination by using tagging.

Tag your list items with one of the to do flags. When you update the list, generate a summary page containing only the uncompleted items flagged with that flag.

To generate the summary page:

  • Click Home--> Find tags
  • Ensure the tags are grouped by Tag Name
  • Ensure that "Show only unchecked items" is selected
  • Ensure that the search scope (at the bottom of the task pane) is set to "All Notebooks"
  • Click "Create Summary Page" (if you want a printable list)

The items in the search results list (the tag summary) will all be clickable. Click any piece of text and it will take you directly to the original note.
