Hi Erin,

Regarding ‘’Information Rights Management" (IRM) in your post, it means that the emails you received is using restricted permission with IRM.

For the first time to open such type of file, you need to connect to a licensing server and if the lincesing server verified your credentials, it will issue a use license for you. After that, you can view this file. 

In addition, as mentioned in your post, this message can only be opened using Outlook. If you have installed Outlook client in your PC, when you try to open this message in Outlook client, you will receive an pop-up window and you just need to click on the Request Permission button to ask for license to open this message.

Moreover, if you don't have Outlook client installed on your PC and you need to read this email message, if you can sign in Outlook on the web or Outlook Web App on a web broswer, then you can view restricted messages here.  

If you don't have a Microsoft account email address, you can quick get a free Microsoft account in this link: https://signup.live.com/?lic=1. For your reference: Introduction to IRM for email messages.

Try this:

 

Sub Run_Presentation()
Dim prApp As Object
Dim prPres As Object
Dim prSlide As Object
Dim prName As String
Dim noSlide As Integer
prName = "presentation100.pptm"
noSlide = 1
On Error Resume Next
Set prApp = GetObject(, "PowerPoint.Application")
If Err.Number <> 0 Then
Set prApp = CreateObject("PowerPoint.Application")
prApp.Visible = True
End If
On Error GoTo 0
On Error Resume Next
Set prPres = prApp.Presentations(prName)
If Err.Number <> 0 Then
Set prPres = prApp.Presentations.Open(Filename:=prName)
prPres.Open
End If
On Error GoTo 0
With prPres.SlideShowSettings
.ShowType = ppShowTypeSpeaker

     .StartingSlide = noSlide
     .ShowWithAnimation = msoCTrue
     .AdvanceMode = ppSlideShowUseSlideTimings
     .Run
End With
End Sub

Updating Formatting of Data in Mail Merge - Microsoft Community

ccapano say:

Updating Formatting of Data in Mail Merge

How do I update formatting of mail merged data from Excel into Word (i.e. date, % and dollar amounts)?

a. user say:

Not sure what you mean by "update" here, but to apply formatting to merge fields in your document,
 a. use option-F9 or fn-option-F9 (depending on your keyboard setup) to toggle field code display on - in this view, field codes look like this:

{ MERGEFIELD myfield }

(In some versions of Mac OSX, the F9 key has been commandeered by Exposé - see e.g. http://word.mvps.org/mac/fieldsinmacword.html for details)

 b. You can also toggle individual field codes using shift-F9/ fn-shift-F9.

 c. Then click after the "myfield" and insert the formatting switch that you need. e.g., for US Dollar amounts, a commonly used format is "$,0.00", which you can apply by modifying the field to look like this:

{ MERGEFIELD myfield \#$,0.00 }

Graham Mayor has a page that lists many of the commonly used formatting options for "Western" formats (there are quite a lot of other switches e.g. for other calendars, numbering systems and so on).

http://www.gmayor.com/formatting_word_fields.htm
