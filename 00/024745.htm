john mcghie say:
Hi Richard:

This is a big miss in the Word Help.  You need to understand "wrapping" and "layering", and they left that out of the help.  Most of the "help" is about SmartArt, which is useless: for almost anything.

To get the Help improved, use Help>Send Feedback to document your issue with Microsoft.  They left it out of the help because they think nobody is using it.  Nobody is using it because they can't find it: it's not in the help!  :-)

You need to set your picture to be "Inline with text".  That causes Word to sit the picture in a paragraph in the text layer and treat it like a very large character.

You will then find that the arrow will happily slide over the top of the picture.

To format the picture, right-click it and choose Format Picture>Layout>Wrapping style "Inline with text".

A Word document is like a sandwich: the text layer is the "filling".  There are 128 graphics 'layers' in front of the text', and 128 layers 'behind the text'.  You can order objects backwards or forwards in the stack like a real Dagwood sandwich to produce the effect you require.

Right-click the arrow and use the commands in the "Arrange" menu to shuffle the object forward or backward in the layers.

If you are going to do a lot of this, do it in PowerPoint.  The same graphics engine is common to all of Office, but the controls are much better presented in PowerPoint.

Hope this helps

suzanne s. barnhill say:

Unfortunately, as you have discovered, text boundaries in Word 2013 are useless. This is due to the new layout engine used in Word 2013. It turns out that Word's developers had no idea that so many users depended on text boundaries, which we have been told were kind of a mistake to begin with, "a feature that was never supposed to be a feature," "a window into the internals of word" that should probably never have been exposed to users.

 

The feature "just draws a border around [Word's] internal drawing rectangles," and those have now changed with the switch to a hardware-accelerated display. So Word "is doing precisely what it has always done (drawing borders around internal data structures). Unfortunately, this means that scenarios that used to work accidentally now don’t work the same way. There’s really no way to bring back the old functionality – it truly doesn’t exist."
