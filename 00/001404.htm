docking onenote rearranges icons, and undocking doesn't put them - Microsoft Community

j.r. bob dobbs say:

docking onenote rearranges icons, and undocking doesn't put them back

The title says it all.  Docking Onenote 2010 in order to use the Linked Notes functionality messes up the desktop icon arrangement, and undocking doesn't correctly put them back where they started.

Has anyone found a solution to this that doesn't involve 1) manually using an icon position restoring utility, 2) changing my workflow by installing an icon dock or "fences" utility, 3) keeping all my icons on one side of the screen all the time?
omer atay say:

This is unfortunately due to the way the Windows desktop reacts to screen size changes. Docking OneNote changes the size of your desktop, similar to how it would be if you went to the screen resolution options and changed it to a smaller resolution (say from 1920x1080 to 1280x800). When the screen gets smaller, Windows will move the icons that no longer fit the screen so that they are still visible. When the screen gets larger there aren't changes to the position of the icons as they still fit the screen. Therefore when you dock OneNote the icons that were in the space that OneNote now occupies will be moved out but they won't move back when you undock.

 

Note that even if OneNote wanted to provide some way of fixing this, it couldn't, as Windows does not support programmatic access to the position of icons on the desktop. So technically no application can change the position of your icons on desktop. The third party tools that seem to do this are using undocumented/unsupported techniques which may stop working next time you get an update from Windows Update. So I wouldn't expect OneNote to have such option in the future and I would recommend not putting icons to the side where you dock OneNote if their position is important.

         - To continue, you may need to reboot the phone by holding down the Power + Volume Down keys until the device vibrates.
