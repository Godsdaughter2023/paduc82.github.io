It seems to happily stream as long as the developer tools stay open.  You can drag them down out of the way but don’t close them.


This adjustment in Document mode worked at other camera sites not using AMC as well.

stevelindsey- [o365] say:

when adding shared mailboxes to mobile devices, you need to add them as an IMAP connection.  

When adding this account, you will select IMAP & Not Exchange as the type of mailbox you are adding

Enter the Shared mailbox display name, shared email address and YOUR Domain password on the "New Account" screen and click Next.

Enter the incoming and outgoing server names (provided below in table), the username is going to be: *** Email address is removed for privacy ***\sharedmailboxAlias format and then YOUR password then click Next. The mailbox is now ready to use.

*Note*

           Depending on which Apple iOS you are running, you may need to also enter the user name & password like you did on the incoming server section. For the version we tested on we did not have to.

Server & Port Settings

The IMAP and SMTP server names for Office 365 accounts are in OWA Options. Apple devices should get the correct Port and Encryption settings from autodiscover records.

Server name: outlook.office365.com

Port: 993

Encryption method: SSL

Server name: smtp.office365.com

Port: 587

Encryption method: TLS
