The macro recorder is useless for this - it does not record any of the operations related to pictures.

Starting with the code below, you can experiment in the Visual Basic Editor to find out how to set many of the picture options.  To play with it, select a picture in your document and run the code below.  Set a breakpoint after "Set myShape = ".  In the Immediate window, type (without quotes) "?myShape.".  As soon as you type the ".", you will see many of the options available to set the format of a picture.

Sub SetPictureFormat

    Dim myShape as InlineShape

'

    Set myShape = Selection.InlineShapes(1) ' Set a breakpoint here

'

    myShape.Reset ' Set the picture back to its original state

' Add other commands here, for example:

    myShape.Fill.ForeColor.RGB = RGB(120, 255, 240)

'

End Sub

For example, you will things such as:

myShape.Fill - you can set myShape.Fill.ForeColor.RGB = RGB(120, 255, 240), etc.

myShape.Line - you can set myShape.Line.Weight = 2.5, etc.

myShape.Shadow - you can set myShape.Shadow.Style, myShape.Shadow.Blur, etc.

myShape.Glow

myShape.SoftEdge

myShape.PictureFormat

Do a little exploring and you will find the things you are looking for.  Then add them to the sub above to see what they do!

Use File --> Export to store a whole notebook in one .onepkg file you can distribute like any other file.
