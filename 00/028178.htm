    OK, Exec Bad Access is a software error (the software is trying to reach some section of the software that is not there or corrupted).

    Since you have been able to re-install, you will be able to do that again, but NOT from Time Machine.  Time Machine can bring software modules back with permissions that don't relate to the current hardware, and that may be what has happened.

    Reinstalling does nothing at all to files that already exist, so to make this work, you will need to remove Office entirely, then re-install, then re-update.  Re-update before you try to use any of the applications.

    To remove, follow this procedure exactly:

    http://support.microsoft.com/kb/2398768

    It's important to carry out the entire procedure, exactly as written.  Don't assume that because you have already done some of the steps that you can leave them out: you have to do all of them, each time.

    Then re-install.

    Then re-start the computer.

    Then re-update (and re-start again).
