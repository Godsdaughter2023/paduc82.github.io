How to get a CMD window on a folder now that it's been removed - Microsoft Community

win10nuc2 say:

How to get a CMD window on a folder now that it's been removed from Explorer's menu

Microsoft have deemed that using the simple, straightforward CMD shell is to be outlawed, and have removed it from Explorer forcing us to use Powershell. This is enormously complex and unwieldy, complete overkill when I just want to run a "dir /a" command as admin. Fortunately it is easy to reinstate the traditional right-click on the folder for DosHere, and to get it to run as admin. Just add this to the registry:

[HKEY_CLASSES_ROOT\Directory\shell\runas]
@="DosHere"
[HKEY_CLASSES_ROOT\Directory\shell\runas\command]
@="c:\windows\system32\cmd.exe /k cd /d \"%1\""

It took me some time (too much time) to discover that simply calling the key "runas" will cause the command to be run as admin. Otherwise it is the same as the good old Win95/NT powertoy - that only runs as user.

There are two things that commonly cause the memory card to be 'invisible' to the phone which would make it revert to the default Nokia tune:

 

(1) When the phone is connected to a PC by mass storage mode, the memory card appears on the PC as an external drive and is not accessible by the phone, so if somebody calls the file is 'missing' so the phone defaults to standard tune.

 ... surely there is an easier way?

You could add this VBA to your PERSONAL.XLSB and a couple of buttons to your QAT (one for each sort direction):

====== BEGIN CODE ======

Option Explicit
'
Sub SortColumnAscending()
    Call SortThisColumnOnly(xlAscending)
End Sub
'
Sub SortColumnDescending()
    Call SortThisColumnOnly(xlDescending)
End Sub
'
' SortThisColumnOnly Macro
' Macro to sort only the selected column and not surrounding columns.
'
Private Sub SortThisColumnOnly(theOrder As XlSortOrder)
    Dim theColumn As Range
'
    Set theColumn = ActiveSheet.Columns(Selection.Column)
'
    ActiveSheet.Sort.SortFields.Clear
    ActiveSheet.Sort.SortFields.Add _
            Key:=Range(theColumn.Cells(2, 1), theColumn.Cells.End(xlDown)), _
            SortOn:=xlSortOnValues, _
            Order:=theOrder, _
            DataOption:=xlSortNormal
    With ActiveSheet.Sort
        .SetRange theColumn
        .Header = xlYes
        .MatchCase = False
        .Orientation = xlTopToBottom
        .SortMethod = xlPinYin
        .Apply
    End With
End Sub

====== END CODE ======
