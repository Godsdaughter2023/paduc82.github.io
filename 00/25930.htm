Where is the inverted or turned italic iota? - Microsoft Community

galahadbr say:

Where is the inverted or turned italic iota?

Cambria Math has an inverted, or turned, Greek iota ( ι ) at U-2129. But it's not italic, in the math container it can't be made italic, and yet logicians prefer the inverted iota to be italicized. Its absence is unexpected, since the equation editor supplies a full set of italicized Greek letters.

 

Is there a workaround, besides splitting an equation into two and inserting a non-containered inverted iota between the two halves? 

 

galahad72 say:

I found the solution. You can italicize math characters that are defaulted to roman in the equation zone. The turned small Greek iota is U+2129. It means "the" in symbolizing a definite description. Is it an operator? Word doesn't include it in the operators but in the Letter-like Symbols group, where some are italic and some aren't. Unicode does not specify it as italic. Logicians generally prefer to see it as italic, starting with Principia Mathematica 1: 30, and perhaps even with Peano (who did his own typesetting).

 

To italicize a roman character in the equation zone or container, while staying within Cambria Math, select the character and go to Equation Tools | Design | Normal Text and press Control-i. There is no need to change the typeface (and to do so isn't straightforward).

PowerPoint Keywords as Filters? - Microsoft Community

sarahratekin say:

PowerPoint Keywords as Filters?

My company has a "master slide deck" that is about 100 slides.

We'd like to "tag" each slide somehow so we can sort/filter/create targeted presentations without having to manually hide the extraneous slides for each application.

We use PPT 2016.  Is this an option?  

Moved from:  Office / PowerPoint / Windows other / Office 2016

steve rindsberg say:

It's not possible right out of the box, but it can be done with VBA or some other programming language that can talk to PowerPoint.

In PPT, presentations, slides and even shapes on slides can have multiple tags (called that, in fact).

You could do something like this in VBA (warning: improvisational code, ripped straight out of the air between my ears and dropped on the screen). It's a starting point only.

This'll let you have a Dog show, a Pony show or a Dog and Pony show.  Select the dog slides, run TagAsDogs.  Select the pony slides, run TagAsPony.  After that, you can run DogShow, PonyShow or DogAndPonyShow to hide/unhide the appropriate slides.

Option Explicit

Sub TagAsDogs()
    Dim oSl As Slide
    For Each oSl In ActiveWindow.Selection.SlideRange
        oSl.Tags.Add "DOG", "Y"
    Next
End Sub

Sub TagAsPonies()
    Dim oSl As Slide
    For Each oSl In ActiveWindow.Selection.SlideRange
        oSl.Tags.Add "PONY", "Y"
    Next
End Sub

Sub DogShow()
' Hide any slide w/o a DOG tag
    Dim oSl As Slide
    For Each oSl In ActivePresentation.Slides
        If oSl.Tags("DOG") <> "Y" Then
            oSl.SlideShowTransition.Hidden = True
        End If
    Next
End Sub

Sub PonyShow()
' Hide any slide w/o a PONY tag
    Dim oSl As Slide
    For Each oSl In ActivePresentation.Slides
        If oSl.Tags("PONY") <> "Y" Then
            oSl.SlideShowTransition.Hidden = True
        End If
    Next
End Sub

Sub DogAndPonyShow()
' Unhide all of the slides
    Dim oSl As Slide
    For Each oSl In ActivePresentation.Slides
        oSl.SlideShowTransition.Hidden = False
    Next
End Sub
