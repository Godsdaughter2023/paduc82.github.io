marthaevansxr say:

how to get Windows 10 Home from Windows Pro

My computer crashed recently and a "Friend " fixed it by putting Windows 10 Pro on it in which I do not have the product key. how can i put Windows 10 home back on my computer so it will be activated,
andre da costa say:

When you boot from the install media, it should prompt you to select Windows 10 Home during setup. Did you get to select that option?

If not, you can try the following option:

  1. Open Registry Editor(WIN + R, type regedit, hit Enter)

  2. Browse to key HKEY_Local Machine > Software > Microsoft > Windows NT > CurrentVersion

  3. Change EditionID to Home (double click EditionID, change value, click OK). In your case it should at the moment show Pro

  4. Change ProductName to Windows 10 Home. In your case it should at the moment show Windows 10 Professional

  5. Browse to key HKEY_Local Machine > Software > Wow6432Node > Microsoft > Windows NT > CurrentVersion, change the same two values as in steps 3 and 4 above

  6. Close the Registry Editor

  7. Insert the Windows 10 Homeinstallation media, do an in-place upgrade by launching the setup from desktop (not booting with Windows 10 Home install media)

  8. When asked, enter your validWindows 10 Home product key and select what to keep(settings, personal files and apps, only personal files or nothing)

  9. Windows 10 Home in-place upgrade will be done now.

Tip: after downloading the ISO, you can double click the ISO file to mount it then start setup

Overview

Login Information

 

  • Gateway: 192.168.0.1
  • Username: admin
  • Password: admin

 

How to Find SSID (Network Name)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Basic from the menu at the top, if it is not already selected.
  6. Check to see which SSID is enabled next to the option "Enabled SSID#:" (By default, #1 will be selected.)
  7. The network name will be listed in the field below as SSID.

 

How to Find Wireless Passphrase (WPA/WEP)

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the menu at the top.
  6. If the "Encryption" is showing WEP, the passphrase will be based on which key is selected next to "Default Key," and will be labeled as "Encryption Key".
  7. If the "Encryption" is showing WPA, the passphrase will be labeled as "Pre-shared key".

 

Changing Wireless Security Type/ Disabling Wireless Security

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Security" from the menu at the top.
  6. Select the desired security under "Encryption". (To turn off security, select "Disabled")
  7. Select Apply.

 

MAC Filtering

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Firewall" from the menu to the left.
  5. Select "MAC Filter" from the menu at the top.
  6. Uncheck the "Enable MAC filtering" box.
  7. Select Apply.

 

Change Wireless Channel

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Basic" from the menu at the top, if it is not already selected.
  6. For "Auto Channel" select "Disable" (to let the channel auto select leave "Auto Channel" as "Enable").
  7. Select the desired channel from the "Channel" drop down box.
  8. Select Apply.

 

Change Wireless Mode

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Basic" from the menu at the top, if it is not already selected.
  6. Select the desired wireless mode from the "Band" drop down box.
  7. Select Apply.

Change Wireless Band (Width)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter "admin" for the user name and "admin" for the password.
  4. Select "Wireless" from the menu to the left.
  5. Select "Advanced" from the menu at the top.
  6. Select the "20 MHz" radio button for the "Channel Bandwidth".
  7. Select Apply.



 

DHCP Reservation

 

  1. Open a web browser.
  2. Log in to the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "System" from the menu to the left.
  5. Select "DHCP" from the menu at the top.
  6. Check the "Enable Static DHCP IP" box.
  7. Enter the IP address and MAC address of the Xbox into the labeled boxes.
  8. Select Add.
  9. Select Apply.

UPnP

  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Advanced Settings" from the menu to the left.
  5. Select "UPnP" from the menu at the top.
  6. Select either "Enable" or "Disable" based on current configuration.
  7. Select Apply.

 

DMZ

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Firewall" from the menu to the left.
  5. Select "DMZ" from the menu at the top.
  6. Check the "Enable DMZ" box.
  7. Enter the Xbox IP address into the "Local IP Address" box or select the Xbox from the drop down box.
  8. Select Apply.

 

Disable DMZ

Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Advanced Settings" from the menu to the left.
  5. Select "Port fw." from the menu at the top.
  6. Check the "Enable Port Forwarding" box.
  7. Enter Xbox into the "Description" box.
  8. Enter the IP address of the Xbox into the "Local IP" box.
  9. Select "Both" from the "Protocol" drop down.
  10. In both the "Local Port" and the "Public Port," enter 3074.
  11. Select Add.

  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.

  4. Select "Advanced Settings" from the menu to the left.
  5. Select "Port fw." from the menu at the top.
  6. Select the port forwarding rule that you want to remove.

  3. When prompted, enter in "admin" for the user name and "admin" for the password.

How To Find Firmware Version

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "System" from the menu to the left.
  5. Select "Status" from the menu at the top, if it is not already selected.
  6. The firmware version will be listed as "Kernel version".

 

Firmware Update

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. When prompted, enter in "admin" for the user name and "admin" for the password.
  4. Select "Tools" from the menu to the left.
  5. Select "Firmware" from the menu at the top.
  6. To download the firmware navigate here.
  7. Select the "Downloads" tab.
  8. Select and download the latest firmware.
  9. Save the firmware somewhere easy to find, such as the desktop.
  10. Navigate back to the "Firmware" window for your router.
  11. Select "Browse" next to the text box.
  12. Locate and find the firmware that was downloaded.
  13. Select Apply and let the installation complete without interruption.

 

Restore Factory Defaults

 

Through Firmware

  1. Plug the computer into one of the LAN ports on the router via an Ethernet cable.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  4. When prompted, enter in "admin" for the user name and "admin" for the password.
  5. Select "Tools" from the menu to the left.
  6. Select "Back-up" from the menu at the top.
  7. Select the "Reset" button next to "Restore to factory default:"

 

Through Hardware

  1. Plug the computer into one of the LAN ports on the router via an Ethernet cable.
  2. Locate the reset button on the device.
  3. Using a paperclip/pen/toothpick, hold the reset button down for 15 seconds.

  6. Enter the PPPoE login and password.
  7. Select Apply.
