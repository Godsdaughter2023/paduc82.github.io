ossiemac say:

 THEN would like it to automatically calculate, save & close workbook..  or closeall.

I interpreted the above as you have 2 buttons; one to close ThisWorkbook and another to close all open workbooks.

Try the code below and test if it does what you want. The code firstly identifies if any workbooks have changes since last save and if so, calculates. Then based on the button used (one or all workbooks)  it processes the Close.

Note that if closing all then ThisWorkbook (with the code) must be last to close otherwise the code is lost.

Feel free to get back to me if you have any questions regarding it. I'll leave you to incorporate your clipboard clearing code.

Sub ClickCloseThisWB()
Call altClose(False)    'False only close this workbook

End Sub


Sub ClickCloseAllWBs()
Call altClose(True)     'True close all open workbooks

End Sub

Sub altClose(bolAllWBs As Boolean)
   
    Dim bolSaved As Boolean
    Dim response As Variant
    Dim wb As Workbook
   
    For Each wb In Workbooks
        If wb.Saved = True Then
            bolSaved = True
        Else
            bolSaved = False
            Exit For    'If at least one workbook not saved then process remainder
        End If
    Next wb
           
   
    If Not bolSaved Then
        response = MsgBox(Prompt:="Workbook not saved." & vbCrLf _
                & "Do you want to save before closing?", Buttons:=vbYesNoCancel)
       
        Select Case response
            Case vbYes
                Calculate   'Calculate all open workbooks
                If bolAllWBs Then
                    For Each wb In Workbooks
                        If wb.Name <> ThisWorkbook.Name Then
                            wb.Close SaveChanges:=True
                        End If
                    Next wb
                End If
                ThisWorkbook.Close SaveChanges:=True
               
            Case vbNo
                If bolAllWBs Then
                    For Each wb In Workbooks
                        If wb.Name <> ThisWorkbook.Name Then
                            wb.Close SaveChanges:=False
                        End If
                    Next wb
                End If
                ThisWorkbook.Close SaveChanges:=False
               
            Case vbCancel
                'Following is optional message to user.
                'Comment out message if not required
                'MsgBox "User cancelled close and save." & vbCrLf & _
                        "Workbook not Calculated, Saved or Closed."
        End Select
    Else
        If bolAllWBs Then
            For Each wb In Workbooks
                If wb.Name <> ThisWorkbook.Name Then
                    wb.Close SaveChanges:=False
                End If
            Next wb
        End If
        ThisWorkbook.Close SaveChanges:=False
    End If

End Sub

abigail faye m. say: Hi everybody,

 

I know how frustrating it is to be unable to receive important emails. We have to consider several causes why you were not able to receive emails such as:

 

A. Filter Settings of the account

 

To ensure that your account filter settings are not causing incoming email not to be delivered to your account, I suggest that you visit this link:

 

I am not receiving Email sent by friends/family

 

B. Inbox is full.

 

There are some users not receiving emails due to there is not enough space from the Inbox.


Free account initial storage limit is 5GB and 10GB for premium account. If you've exceeded the initial storage limit, please try to get rid of unnecessary email to free up some space from your account.

 

C. Mail Client Settings

 

You might not also send or receive emails due to incorrect configuration when accessing account using a Mail Client. For this reason, please reconfigure your account by following the settings of the link below:

 

Send and receive Hotmail from an email app

 

D. 3rd Party Email account

 

If you are using University email account such as (.edu) or other 3rd party account, it is possible that there is an issue with your network/server. You might need to contact your domain administrator for further assistance. They may have changed some settings in your domain's system that's why you are having problems receiving emails.

 

If the above steps provided did not resolve your issue, I would like to ask additional details regarding your concern:

 

> Are you trying to access your account using a web browser or a Mail Client such as Windows Live Mail?

> Have you tried to send a test mail to your own email address? did you receive the test email?

 

Thank you!
