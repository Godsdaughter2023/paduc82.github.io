Try this macro:

Sub change_formulas()
   Dim ws As Worksheet
   Dim r As Integer
   Dim my_formula As String
   Dim my_new_formula As String
   Dim par1 as String
   For Each ws In ActiveWindow.SelectedSheets
     ws.Unprotect("rudolph")
     For r = 25 To 29
       my_formula = ws.Cells(r, "B").Formula
       par1 = Mid(my_formula, InStr(my_formula, "(") + 1, InStr(my_formula, "-") - InStr(my_formula, "("))
       my_new_formula = "=COUNTIFS(" & par1 & "8," & par1 & "2)"
       ws.Cells(r, "B").Formula = Replace(my_new_formula, ">=""&TODAY()-2", "<""&TODAY()-2")
     Next r
     ws.Protect Password:="rudolph"
   Next ws
End Sub

Before you run the macro, you should select all sheets where you want the formulas to be replaced.

You should not change the aspect ratio of a picture, which is what you would be doing if you changed the width without changing the height (or vice versa).  Instead, you will need to crop or resize the picture in photo editing software (even Microsoft Office Picture Manager, if you have it, can do this) to get the desired aspect ratio before inserting the picture.
