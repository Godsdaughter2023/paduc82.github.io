Fix Excel file name Start with ~$ sign - Microsoft Community

22836 say:

Fix Excel file name Start with ~$ sign

Need your help to get rid of the ~$ signs in the excel file. i am using ms office 2007. i googled it but didn't get any permanent solution. why these signs appear in the start of Excel file?. 

Thanks 

hansv mvp say:

A file whose name starts with ~$ is almost certainly a temporary file created by Excel when you open a workbook.

For example, when you open a workbook named Products.xlsx, Excel will create a temporary file ~$Products.xlsx.

You should only see such files if you clear the check box "Hide protected operating system files (recommended)" in the View tab of the Folder Options dialog.

Under normal circumstances, Excel will remove the temporary file when you close the workbook. So there is no need to worry about them. Only if Excel crashes, the temporary file will be left behind.
