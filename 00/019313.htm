Private Sub CommandButton4_Click()

    Dim i As Long
    Dim r As Long
    Dim c As Long
   
    Sheet14.Activate
   
    'Clear any existing data
    Range("A1").CurrentRegion.ClearContents
   
    'List the column headers in the first row, starting at A1
    With Me.ListView2
        For i = 1 To .ColumnHeaders.Count
            Cells(1, i).Value = .ColumnHeaders(i)
        Next i
    End With
   
   
    'List the selected items and subitems starting at Row 2
    r = 2
    With Me.ListView2
        For i = 1 To .ListItems.Count
        If .ListItems(i).Checked Then           

                 Cells(r, "A").Value = .ListItems(i).Text
                For c = 1 To .ListItems(i).ListSubItems.Count
                    Cells(r, c + 1).Value = .ListItems(i).ListSubItems(c).Text
                Next c
                r = r + 1

        Next i

3.  Click on GO options beside Manage, uncheck all the add-ins and click Ok.
4.  Try to open Outlook and check if it works fine.

hpholm say:

Reconfigure the phone to use another Microsoft account is only possible by performing a factory 'hard reset' of the phone from Settings > About > Reset... Then during the first start-up select the new account.

Please notice this will return the phone to its out-of-the-box state and erase any personal content on the phone, so first copy everything you want to keep to a PC. Contacts will also have to be restored from elsewhere, and you need to manually make a list of favourite apps to be installed again. Any purchases in the Store are also lost.
