In order to assign a keyboard shortcut to a macro, you have to use the Customize Keyboard dialog.

  1. Right-click on the Ribbon or QAT and choose Customize the Ribbon.

What is WBEMTEST.exe: Original use was a Microsoft support team tool

Steps to clear AntiVirus registry values from Microsoft Security Center:

1)Click Start- Search

2)  Search for  WBEMTEST.exe   

3)  right click WBEMTest.exe  

a. right click WBEMTest.exe in the results list and select Run as Administrator <- edit noticed by Stephen Boots

b.  runas /user:computername\Administrator WBEMTEST.exe <- this will allow WBEMTEST.exe to be run in administraor mode

4)  Click Connect 4)  Click Connect

5) In the top field enter root\securitycenter

6) Click Connect

7) Click Query
8) Type the following Query:

 

SELECT * from Antivirusproduct

 


9) Select the instance returned - If any

10) Click DELETE
11) Close the query window

12) Click Query again
13) Run the Query:

SELECT * from Firewallproduct

14) Select the instance returned
15) Click DELETE

 

Microsoft security center is now clean of any AntiVirus reg...values left behind after the product was uninstalled!

sven johannsen say:

OK,

In the message app if you tap on the phone number at the top of the thread screen it will open up an UnKnown Contact screen which will let you phone, text, OR SAVE that number (disc icon at bottom). If you tap the disk (SAVE) it will take you to contacts, and give you the opportunity to select a contact to assign it to, or ADD a new contact (+ sign at bottom). If you choose ADD it presents a new contact screen with that number in it.

So for someone you get a text from you don't have in contacts, Tap the phone number, hit Save (disc icon), tap Add (+ sign) and fill it out.

As for saving text content...not the greatest process but can be done. Tap and hold on a text bubble. Notice an option is copy. That copies what is in that bubble. You can then paste it somewhere. I like to use OneNote. Easy to create a new note and paste that in, and if you have it set up, OneNote syncs to Skydrive and is then available wherever. Could also paste to an e-mail or word document, and of course send the latter to SkyDrive.

For multiple posts, just get into the message thread, copy first, get to OneNote and paste. Then hold down back arrow to quickly switch back to messages, tap-hold copy....hold back arrow to go back to the oneNote Note, paste, lather rinse repeat. You can name the Note appropriately at the top.

 

The bad news is the copy-paste thing doesn't retain who the messages are from, or any time stamps, so if you are looking for a 'legal' record as some have, this isn't going to do it.

Re 1:

If the document originates in your company, it should use the letterhead template, be that a modified form of the Normal template or some other. karl timmermans (outlook mvp 2012-2018) say:
>>
Now I'm getting error while setting data file of Imap acc to default.
<<

#1 - What error exactly are you encountering?
#2 - Would it be correct to assume that you have another account coonfigured using Exchange?

If the answer to #2 is yes, then unfortunately you are not able to set the IMAP OST file as the default data file. However, that is actually a good thing since you really do not want to maintain your default contact/calendar data in an IMAP data file where the folders will have the added description of "This Computer Only".

If the IMAP account is removed from the Outlook profile, the associated data file will also be deleted and all data in "This Computer Only" folders will be lost. Similarly, if Outlook can't find the OST file for any reason, a new one will be created, the email folders will be sync'd with the server but the contact/calendar folders will be empty.

You should also note that an OST file can only be opened/read by Outlook using the original Outlook profile/email account that created it. If the OST file is backed up separately, you will need to use a 3rd party OST2PST conversion tool.

The recommended alternative to using an IMAP ost file as the default data file is <either> to maintain the default contact/calendar items in an Exchange based account OR to add a new PST file to the profile and set it as the default data file.
To add a new PST file, go to
Home Ribbon --> new items --> more items --> Outlook Data File
when done, set the PST file as the default data file.
