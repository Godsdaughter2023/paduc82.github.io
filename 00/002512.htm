Hi Rick,

I’d like to answer your questions one by one.

1. I'm assuming that needs to be manually moved from the current hosts to Office 365 host servers, right?
The answer is right.

2. Am I right in thinking the exchange data is in an ost file instead?
The answer is that both the “pst” file and “ost” file are the two types of Outlook data files. The “ost” file is kept in the Exchange server and the “pst” file is kept in your PC (local). The answer is that you can backup the previous data into a “pst” file and import it into the Outlook client. More information about Exporting from the Outlook client and Importing from the Outlook client are for your reference.
