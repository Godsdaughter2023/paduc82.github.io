graham mayor say:
With the cursor in the table - Layout > View Gridlines.

As for the labels not aligning correctly:

1. You have added a header/footer to the normal template (rename the normal template.)
2. The page size set in the printer properties doesn't match the page size of the label sheet.
3. You have a scaling option set in the zoom section of the print dialog
4. Your printer has poor paper handling. Some printers will slip when faced with label stock and thus are incapable of feeding the label correctly. Cleaning the rollers in the print path may help.

word document Bullets - Microsoft Community

philkinneyoi say:

word document Bullets

I can't remove bullets from my document

john korchok say:

Try:

  • Clicking to the left of the bullet and pressing the forward delete key. If you can delete them this way, they aren't a real bullet list and can be manually deleted.
  • Selecting the bulleted text, then using Format>Bullets and Numbering>Bulleted>None command.
  • Selecting the text, then applying a non-bulleted style to it from the Quick Style Gallery. Normal style is usually good.
