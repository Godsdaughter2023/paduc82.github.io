ruz san say:

Hi JW,

If you are unable to keep the preferred input language on your computers, you can manually change it through the Registry Editor. This will change what the PC recognizes as the default settings when using your keyboard. Please see this page first for information about editing the registry.

To proceed with changing the settings, refer to these steps:

  1. Right-click on the Windows icon on your home screen.
  2. Choose Run.
  3. Type regedit then press Enter.
  4. Navigate to HKEY_USERS\.DEFAULT\Keyboard Layout\Preload.
  5. Look for the string value named 1.
  6. Right-click on the value then choose Modify....
  7. Take note of the current Value data.
  8. Replace it with 00000809. This is the code for UK input language.
  9. Click OK.
  10. Restart your computer and check if UK has been set as the default input language.

Do the same steps for the other computers.

Let us know if you need more help.
