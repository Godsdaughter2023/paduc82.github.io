ken sheridan say: Forget about the InParam function; that's for use only where the parameter is a comma separated list of multiple values, i.e. for when a query is required to return multiple rows where the value in a column is any one of the values in the value list.  You simply want a single value as the parameter because, for each PDF file you create, the query will be returning only the row or rows which match that value, i.e. the basis for your 'group'.

The reason the InParam function works in my demo but not your file is because it is included in a module in the database.  Although published by Microsoft, the function is not built into Access. It has to be added to the database in question.

So in query design view, where the query is a 'totals' query you'd add the column on which you want to restrict the query, and hence the report which uses it as its RecordSource, to the query design grid a second time, and in the 'total' row select 'where'.  In the 'criteria' row of the same column enter the reference to the text box control on your form as the parameter in the format:

[Forms]![YourFormName]![YourControlName]

You don't need a list box or any query to populate it.  Your code will establish the recordset in the way I described and loop through it assigning the Header column's value to the text box at each iteration of the loop, and calling the OutputTo method to create the PDF file.  This will use the current value of Header, which is now in the text box, as the parameter via the report's query, so should create a PDF file for the data relevant to the current value of Header.

Unlike the very "old" DOS days when there was a signature file on the floppy, there is nothing unique on the CD to identify it. The uniqueness is provided via the Product key you use to activate it. So, you can borrow a CD without threatening their installation/activation.
  2. Ask at work if they have CD you can borrow
  3. Try to find a local "PC Computer Club". There is almost certainly a geek or three with disks you could borrow.
  4. Find a small local computer store or repair business. They often have old versions of software because they are still supporting customers using them.
  5. You could buy it online. Be very careful who you give your money to
  6. Try websites like Kijiji or Craig’s List to see if you can borrow or buy it
  7. Try getting one from MS (if it works, please post back here, there are many interested people who would like to know that MS is still providing that kind of support) http://support.microsoft.com/kb/326246/ 

Once you get the installation file, make sure you burn a copy for yourself before returning the disk if it is borrowed.

I strongly suspect it will work like a normal install. If you chose the default installation it will automatically, with NO WARNING!, uninstall your old office. Very rude!. You should do a "Custom Install", and explicitly select the option to Not uninstall old applications.

Different back grounds configurable per screen. Either via colour or desktop image.

Taskbar clue as what screen one is on. This must take up a small area only.

  1. EASY to use keyboard shortcuts – configurable (default Alt 1,2,3,4 or Shift/Ctrl  1, 2, 3, 4) These keys either  to override desktop app shortcuts or not.

     1, 2, 3, 4) These keys either  to override desktop app shortcuts or not.

  4. Optionally, each screen to be able to have its own desktop back ground

  5. Optionally leave one monitor, in a multi monitor setup as a fixed desktop, main or secondary to be the switchable desktop.

  6. Optionally each screen to be able to have its own theme

  7. Optionally, each screen to play a different sound when entered
