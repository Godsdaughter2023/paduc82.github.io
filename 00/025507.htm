graham mayor say:
There is a minor typo in Doug's code ('-' instead of '=') which will cause it to fail, however you may find it easier to work in inches than points, by using the InchesToPoints conversion thus

With ActiveDocument.PageSetup
    If .Orientation = wdOrientPortrait Then
        .TopMargin = InchesToPoints(1)
        .LeftMargin = InchesToPoints(1.5)
        .RightMargin = InchesToPoints(1)
        .BottomMargin = InchesToPoints(1)
    Else
        .TopMargin = InchesToPoints(1.5)
        .LeftMargin = InchesToPoints(1)
        .RightMargin = InchesToPoints(1)
        .BottomMargin = InchesToPoints(1)
    End If
End With

Hi Constance,

Based on the error message and my research, the issue should be caused by too many connections to the mailbox. To resolved this, you can increase the limits for that mailbox.

  1. Create a new throtting policy:
    New-ThrottlingPolicy -Name HighUsage -OwaMaxConcurrency 100 -RcaMaxConcurrency 100
  2. Apply the throttling policy to the problem mailbox:
    Set-Mailbox -Identity tonysmith -ThrottlingPolicy HighUsage

For more information, please refer to the two articles below:

Change user throttling settings for specific users

Set-ThrottlingPolicy
