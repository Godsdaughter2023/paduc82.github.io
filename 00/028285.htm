doug robbins - mvp office apps & services (word) say:

Bibliography sources you create are all listed in the following file: %APPDATA%\Microsoft\Bibliography\Sources.xml

Or, to put that in a bit more layman type language

c:\users\[username]\AppData\Roaming\Microsoft\Bibliography\Sources.xml

Now, if you copy one of these files from one computer and copy it into another folder and change its name to say Sources1.xml and then copy it into the Bibliography folder on the other computer, then by clicking on the Manage Sources button in the Citations & Bibliography section of the Ribbon, you can then browse to that file via the Browse button in the Source Manager dialog and then copy the citations from one list to the other.

erich h. collarcollar say:

macropod say:

1. Quit Word & Excel.
2. Click Start, click Run, type regedit, and then click OK. This opens the Registry Editor.
3. The Registry Editor has a structure reminiscent of a Windows folder structure. Navigate to:

HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Excel\Options

4. With 'Options' selected, right-click and choose New>DWORD value.

5. Type QFE_Boston, then press <Enter>.
6. Right-click QFE_Boston, then click 'Modify'.
7. In the 'Value' data box, type 1, then click OK.
8. Quit Registry Editor.
