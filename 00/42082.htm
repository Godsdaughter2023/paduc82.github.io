jimwynne say:

AOL compresses graphics files (even ones like jpg that are already compressed) in order to facilitate page loading.  It gives them a .art extension.  Somewhere in the AOL browser options you will find something that allows you to turn that compression off and thus be able to download photos/images in their native formats. 

ETA:  Note that when you find that option and fix it, you should clear the browser cache before testing it because you might reload images from the cache that are still in the .art format.

The sad fact is that removing the glare is not possible because there is
no image detail "under" the glare...it's just obliterated.

So...what to do?

In extreme cases I have copied undamaged eyes from other photos of
the same subject and pasted them into the damaged photo. With some
blending...you would never know the difference. This requires editing
software that can create cutouts like PhotoShop. Personally, I use
MS Digital Image Suite but it has been discontinued.

The following article suggests replacing the damaged sections of the
image with undamaged sections.

Have a look:

Correcting eyeglass glare
