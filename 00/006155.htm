Following code in ThisWorkbook module.

Private Sub Workbook_Open()
   'The "Exit Sub" line used during testing and editing VBA.
    'To suppress invoking timeout, uncomment "Exit Sub"
    'This sub can be run from VBA editor by commenting out "Exit Sub" _
     and placing cursor within this sub and pressing F5
   
    'Exit Sub    'Uncomment this line to suppress invoking timeout when opening workbook
    
   'Save user Status Bar default setting
    blnStatus = Application.DisplayStatusBar
   
   'Turn on status bar display
    Application.DisplayStatusBar = True
   
    Call StartTimeOutClock

End Sub

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Call StopClocks

End Sub


Private Sub Workbook_SheetCalculate(ByVal Sh As Object)
    Call StopClocks
    Call StartTimeOutClock
   
End Sub


Private Sub Workbook_SheetSelectionChange(ByVal Sh As Object, ByVal Target As Range)
    Call StopClocks
    Call StartTimeOutClock

End Sub

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    Call StopClocks
    Call StartTimeOutClock

End Sub

Following code in Standard Module.

'Public variables and Constants to be at top of standard module before any subs.
Public dteNoActivity As Date
Public dteStatusDisplay As Date
Public blnSuppressStartClocks As Boolean
Public blnStatus As Boolean
Const strTimeOut As String = "00:00:30"        'Edit strTimeOut interval to desired value
Const strStatusInterval As String = "00:00:02" 'Edit Status update interval to desired value


Public Sub ShutDown()
    On Error Resume Next
    Application.DisplayAlerts = False
   
    blnSuppressStartClocks = True
   
    Application.Wait (Now + TimeValue(strStatusInterval))   'Optional force wait
   
    Call StopClocks
   
    With ThisWorkbook
         Application.DisplayStatusBar = blnStatus  'Reset to User's default
        .Save
        .Close
    End With
  
End Sub


Public Sub StartTimeOutClock()
    If blnSuppressStartClocks Then Exit Sub
    On Error Resume Next
    dteNoActivity = Now + TimeValue(strTimeOut)
    Application.OnTime dteNoActivity, "ShutDown"

    Call StartStatusClock

End Sub


Public Sub StartStatusClock()
    If blnSuppressStartClocks Then Exit Sub
    dteStatusDisplay = Now + TimeValue(strStatusInterval)
    Application.StatusBar = "Time left before shutdown " & Format((dteNoActivity - dteStatusDisplay), "hh.mm.ss")
    Application.OnTime dteStatusDisplay, "StartStatusClock"
End Sub


Public Sub StopClocks()
    On Error Resume Next
    Application.OnTime dteStatusDisplay, "StartStatusClock", , False
    Application.OnTime dteNoActivity, "ShutDown", , False
End Sub

john mcghie say:

If you have an "English" version of Word 2011, the English US proofing tools are installed if the application is installed.

To see a complete list of the proofing languages available, read the Word help topic "Proofing tools that are available for each language".

So the only way you could see that error message is if either your Office installation is one that does not contain any form of English (in which case, the user interface would not be in English either) or something has removed one of the languages from the application bundle.

There are Rubbishware applications out there that promise to "reduce the size of the office installation" by "removing unnecessary files".  Monolingual is one such.

If you run those, they can indeed delete one or more proofing languages.

If you look in /Applications/Microsoft Office 2011/Office/Shared Applications/Proofing Tools, you should see five files with "English" as the start of their name. These are the source files for the Hyphenator, Speller, Thesaurus, Grammar and Word Forms.

If they are not there, copy them back from your Installation disk.  If you do not have an installation disk, you will need to restore them from your backup. If you have Office 365, a remove and replace will put them back for you.

Hope this helps
