considerate_guy say:

This is a difficult question to answer since there are many variables.  It really depends on what you are trying to accomplish and how you expect your recipient to view your movie.

 

If they have a Windows PC that is capable of viewing the movie formats you are creating (e.g. .wmv or .mp4 files) then you can copy them on to either disc and they can then play them on their computer.

 

However, if you are trying to make something they can put in their DVD player and show on a TV, you can't just copy the file onto a disc.  DVD players expect video to follow the DVD standard with menus, .vob files encoded in MPEG2 video format, etc.

 

This requires a specific program to create the DVD in this standard format.  Windows 8 doesn't come with any such program so you'll have to look for other options.  A few freeware choices are Freemake Video Converter, DVD Styler and DVD Flick.  You'll have to search the web for tutorials on how to use them, but basically you want to:

  1. Save your final movie from Movie Maker onto your PC using File > Save movie
  2. Verify this movie plays as expect on your computer using Windows Media Player.
  3. Open the desired video DVD program and give it this file as the source file.
  4. Walk through the options for creating a DVD.
  5. Insert your blank DVD disc and choose the option to burn a movie.
  6. Verify this disc works as expected in your own DVD player.

Other things to consider:

  • The DVD standard is an SD (standard definition) format with a maximum resolution of 720 x 480 (NTSC). Expect high resolution photos and video to look less clear on a standard DVD.
  • A DVD disc can hold about 2 hours of video.  If you plan to add that much video, consider doing it as several projects from Movie Maker (say a separate movie for each "chapter") then add them all to the final DVD.  This is a better idea than trying to edit a full 2 hours of video in a single Movie Maker project.
  • You might want to consider other options for sharing the video such as uploading it to a sharing site like YouTube or Vimeo and just emailing a link rather than the actual video.
  • CDs are generally used for music and DVDs are generally used for video with the DVD holding more information.  If you plan to make more movies on DVDs, you should stick with DVD disc.  Also, the RW versions (read/write) are more expensive (since they allow you to rewrite the contents) and wouldn't be required unless you plan to reuse the discs. However they can be helpful when starting out because if you make a mistake you can erase and start again on the same disc without wasting a disc.  Once you have the process down, you can stick with DVD-R discs instead of the RW variety.

considerate_guy say:

The CD and DVD can both be used for storing video disc that plays in a standard DVD player, you'll need to make sure that you are storing those files on a DVD and in the appropriate format for a video DVD; you can't use a CD in a DVD player unless that model can handle data files on a CD disc.  And that same disc may not be playable in other DVD players that only understand the video DVD standard.

Hi DebDoc,

Login to your Husband’s account and then try the below steps:

1. Check if you the credentials have been saved in the Credentials Manager.

Click on Start, click on Control Panel, All Control Panel Items>Credentials Manager.

2. Check if ‘Connect Automatically’ option is checked in the Network and Sharing Center properties.

a. On the right side of the taskbar, you will see a wireless network icon like the one below. Click on it.

b. A window with available network connections will open. As you can see from the screenshot below, the list is split by the type of available network connections. At the top you will have dial-up and virtual private network (VPN) connections, while at the bottom you will have a list with all the wireless network which Windows 7 has detected.

c. Once you decided on which network to connect to, click on it. If you plan to use that network in the future, make sure you check the box that says 'Connect automatically'. This way, when you start your laptop next time, in the same area, it will automatically connect to this wireless network without requesting any manual intervention. Next, click on the Connect button.

NOTE: be cautious with wireless networks which have no security enabled. They can be used to steal personal data. If you connect to such networks make sure your security solutions are turned on.

d. After a few seconds, you will be asked to enter the security key. Ask the administrator of the network for the wireless security key or, if you are in your own home network, take it from the control panel of your router. If you are in a public place, it is best to check the 'Hide characters' box so that other people don't see what you are typing. Then type the security key and click on OK.

e. If you typed an incorrect password, Windows 7 will request you to type it again and again until it matches the password of the network you are connecting to. If everything is OK, Windows 7 will connect to the network you selected using the given security key. When the connection is successful, the wireless icon from your taskbar changes as shown below.

NOTE: This procedure works only if your wireless adapter is enabled. If it is not enabled, you won't see any wireless networks being available. The procedure of enabling the wireless adapter is different for each model of laptop.

Restoring Windows Live Mail from a disk - Microsoft Community

laurjoost say:

Restoring Windows Live Mail from a disk

Hi all!

Background:

New computer with new OS (Win 7 Pro 64 bit).

Old computer was W8 32 bit, the motherboard fried. HDD is available and working.

Using Windows Live Mail 2012 on both

Goal:

Mails as they were on the old computer, running on new.

What I've tried:

  1. Using the old computer's hard drive (actually an image thereof) I've tried to copy the whole WLM storage folder on the old computer, replacing the same on the new one. (As also seen here: http://answers.microsoft.com/en-us/windowslive/forum/livemail-files/need-to-recover-windows-live-mail-messages-from/9f0b4562-3e4a-419a-af79-ed115085e705)
    • Problem: After opening WLM, the folder tree and accounts were as expected, but mails could not be accessed (while showing correct read/unread status <- SOMEWHAT IMPORTANT). Additionally, All the mails now had duplicate, unread, variants that opened.
    • Further: After closing WLM, the old folder tree along with the unaccessible mails disappeared, and all mail was now under Recovered items.
    • Further: Every time I reopen WLM, the recovery runs again, NESTING the bloody Recovered items folders (sigh!)
  2. Using Import Mail from the old storage location
    • Problem: Will not maintain read/unread information
    • Further: Lots of manual one-by-one folder moving in a bad example of a tree structure.

So:

How can I make WLM recognise the preexisting storage folder?

EDIT:

Nevermind. Got it done manually.

What I did:

In a clean installation I:


  1. Imported using File -> Import Messages -> Windows Live Mail
  2. In the old storage folder I selected all folders EXCEPT "Calendars", "Compact", "Proof", "Sentinel", "WLMDSS.tmp", "Your Feeds" and their subfolders.
    • Note that subfolders of the actual mail folders like "Drafts" need also be selected
  3. Manually moved the folders and mails from "Storage Folders -> Imported items" to their proper locations.
  4. At some point, WLM will ask you the account passwords of the different mail accounts, once for each account.

That caused me to lose the read-unread status of messages (They're all marked "read". Since this is not my computer, I at this point decided that I don't care.

However, it is potentially possible to determine the unread status from the first recovery option (see "What I've tried" above), write down the list of unread messages and then manually mark the unreads among the imported messages. If the amount is large, you'd need some way to automate it, at which point you either pay someone to do it for you, or dive in and spend a lot of time coding, with the hope that if this ever comes up again you can use your previous work (you can't, cause by that time there's a new mail client and you'll be solving the problem of importing your old stuff into it.)

Good luck.

Something like this:

Has the system ever been upgraded to Windows 10 and then reverted back to Windows 7?

Do you now or have you ever had a third party defragmentation application installed?

Describe your antivirus/antimalware environment.

Windows 7 comes with a Scheduled Task to defragment all volumes every Wednesday at 1:00AM.

If you have ever upgraded the system to Windows 10 and then reverted back to Windows 7 sometimes that will corrupt every Scheduled Task on your system (sometimes fixable).

If you have installed some third party application that includes it's own defrag or other kind of maintenance operations it may have disabled the built in Windows 7 task. 

If you have disabled the built in Windows 7 task yourself you will see that message when you attempt to open defrag (non believers try it).

If the task has been disabled and you click the Remove settings button that should enable the task again and (after a moment) you should see:

Something you have installed could prevent or override that setting though.

Click the Start button/orb and in the box enter:

Task Scheduler

Choose to Run as administrator:

If Task Scheduler opens with lots of complaining about corrupted tasks that is a symptom of a botched Windows 7 to Windows 10 back to Windows 7 reversion and is a different problem (sometimes fixable).

If Task Scheduler opens fine check the Defrag task to see if it has been Disabled.

When you look in Task Scheduler you should see the task and be sure the Status column says Ready and not Disabled:

If the task is Disabled, right click and Enable it:

See how things look now.
