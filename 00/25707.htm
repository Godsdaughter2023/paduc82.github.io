bernie deitrick say:

You can do it by not merging the cells, and by entering Trip into all six cells. Or you can use VBA:

Function TripCount(r As Range, s As String) As Integer
   TripCount = 0
   If Not r.Find(s) Is Nothing Then TripCount = r.Find(s).MergeArea.Cells.Count
End Function

Used like

=TripCount(A1:AE1,"Trip")

bernie deitrick say:

Functions and formulas ignore merged cells, with one exception that would require your users to perform a complicated procedure.  Usually, they just reference the upper left cell of the merged range. So you have to use VBA to keep it simple for your users.

".... doesn't fill the requirement of NOT using VBA."

I would like to eat all day without getting fat, but that ain't gonna happen either  ;-)  Unless I'm training to run an ultra- marathon. So here's the procedure that allows functions to count merged cells, just so you can decide. 

Have "Trip" entered into a single, non-merged cell.

Merge the cells of interest.

Right-Click the merged cells, choose Paste Special, then choose Formula  and click OK.

Then Countif(cells,"Trip") will count those merged cells.

  • Emails rejected due to policy reasons
  • Temporary errors (No adequate servers, Connection timed out, Resources temporarily unavailable, Out of memory)
  • Incorrect or misspelled recipient's address
  • Wrong outgoing mail server set on your email program or mobile device.
  • Mailbox full or Quote Exceeded
  • Mailbox unavailable
  • Browser issue

  • Email address in question.
  • Are you using an email client like Windows Live Mail, mobile device or Outlook.com (web) to send emails?
  • Were you able to successfully send an email individually?
  • Full message header of the bounce message. These are the details that we need.  

- Final Recipient

- Action

- Status

- Diagnostic Code

 

To get the bounce message, refer to these steps.

 

1. Go to Inbox.

2. Right-click the bounce message.

3. Choose View message source, a new tab will appear.

4. Copy and paste all the contents and paste it in the private message area.
