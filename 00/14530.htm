graham mayor say:
The following Outlook macro will resave all your contacts to save you re-editing them all manually. The macro can be modified to add fx to fax numbers that don't have it already.

The fx prefix appears to work, though I found it necessary to restart Outlook before the change was reflected in the address book. Another issue is that if you have software that uses the addressbook for preparing faxes, then the cobtacts with faxes may not appear. certainly I have written procedures in the past that take advantage of the fact that the addressbook lists fax numbers.
If you click first character of the e-mail address in the To dialog, you will get a list of the addresses you have used previously which may be a better approach?

Sub ResaveContacts()
Dim olNS As Outlook.NameSpace
Dim olContactFolder As Outlook.MAPIFolder
Dim olContact As Outlook.ContactItem
Dim olItems As Outlook.Items
On Error Resume Next
Set olNS = GetNamespace("MAPI")
Set olContactFolder = olNS.GetDefaultFolder(olFolderContacts)
Set olItems = olContactFolder.Items
For Each olContact In olItems
    With olContact
        .Display
        .Close (olSave)
    End With
    Err.Clear
Next
Set olItems = Nothing
Set olNS = Nothing
Set olContactFolder = Nothing
Set olContact = Nothing
End Sub

A couple of other approaches that would speed up the process (assuming that removing/re-adding works as described) would be:

#1 - Manual approach without code

Customine the Phone View to add the Fax numbers involved in the view. Just need to click the field name in the view to sort it accordingly at which point can just go down the list making in place table edits

For Each olContact In olItems

        .Display
        .Close (olSave)
    End With

Replace with the following:

------------------------------
On Error Goto Skip_Item

For Each olContact In olItems
     With olContact
           If Left(.BusinessFaxNumber, 2) = "fx" then
                .BusinessFaxNumber = mid(.BusinessFaxNumber, 3)
                .Save
                .BusinessFaxNumber = "fx" & .BusinessFaxNumber
                .Save 

          EndIf
    End With

Skip_Item:
Next
-------------------------------


*** above assumes the prefix (fx) is always going to be the first 2 characters of the fax number - otherwise adjust the value of the "prefix" accordingly. Other two fax number field names are ".HomeFaxNumber" and ".OtherFaxNumber"

There is presumably at least one cell in the range that you are summing that contains a #N/A value.  If you want to ignore such values change
=SUM(range)
to
=SUM(IF(ISNA(range),0,range))

and enter this formula as an array formula using Ctrl+Shift+Enter.

Alternatively change the cells that contain #N/A values so that they don't.
