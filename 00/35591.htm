    Why do you create all these partitions on the first install... only one is necessary, so if your Windows installation cannot find the O/S, it might be installing not on the first partition.  Try again, deleting the volumes, and creating a completely empty, unformatted drive, then just create one single partition and tell Windows to use that.  Do not create further partitions or do any more formatting until Windows is running.  It can all be done from Disk Management within Windows.

    Another good idea is to install Windows 7 WITH ONLY ONE SINGLE HARD DRIVE INSTALLED IN THE PC.  Poor Windows does tend to get very confused if you try to install and you've got 6 other drives connected.  You don't have to remove them from the case, just disconnect either the power or the data cables from the rear of the drive(s).

    BTW, a hard drive isn't 64 bit or 32 bit, it's just a drive!
