PARAMETERS [Forms]![Stock]![Picked] TEXT(255);

You can either open the query directly as a datasheet, or open a form or report with the query as its RecordSource.

User32.dll is a very important file for Windows, and there are some viruses that try to infect it. In some cases, a damaged Windows registry may result in a damaged User32.dll file.
