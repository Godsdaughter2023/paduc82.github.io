macropod say:
The problem appears to be that you have mixed data types in column B - especially if all of rows 1-8 are empty or have something other than a very long string. When Word does a mailmerge, the OLE DB server it uses queries the data source for information about the data. The data types are determined according to whatever's in the first 8 rows. The DDE server doesn't do this and simply treats everything as text.
