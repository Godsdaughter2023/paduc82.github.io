I have found the problem for my specific case. In the registry HKEY_CLASSES_ROOT > Applications > ruby.exe > shell > open > command, the Default value string value refers to an out-of-date location for the ruby.exe. HI Elsa,

USB devices such as a flash drive or an external hard drive won’t get detected when they are not getting installed on the computer. Although the device status externally shows that the device is connected (the device light is glowing), it’s important to find out whether the device is showing up Device Manager. If it does, then do we have an error code or message accompanied with the device status for the drive.

  1. What is the Make and Model of the computer? (To find out if the chipset driver installed on the computer is the latest one as compared to the version available at manufacturer’s website of the computer.)
  2. Is the issue specific to this external hard drive? Are other external devices showing up when connected to the computer?

Try connecting the drive to a different port and check if it getting detected (rare and front ports). If it’s not getting connected, proceed further. Also, check if the drive is working when connected to a different computer.

To check the device status, follow the steps:

  • Press Windows+X keys and select Device Manager.
  • Verify if the drive is showing up under Universal Bus Controllers icon. If yes, check if the there is an exclamation or a question mark on the device icon. If yes, right click on the icon and select Properties. Make a note of the error code/message that you receive under Device status.

For further troubleshooting, you can run the hardware and device troubleshooter and check if it helps. Follow the methods listed below:

Method 1:

The hardware and device troubleshooter checks for common problems and makes sure that any new device or hardware that is connected to your PC is installed correctly.

Connect the drive and then refer to the link to run the troubleshooter:

Refer to the section “Run the Hardware and Devices troubleshooter”:

http://windows.microsoft.com/en-US/windows-8/what-device-isnt-installed-properly
