jeeped say:

As mentioned, Excel considers each day as one (1). A week is simply 7 days or (7*1).

D2 should be =D1+(C2*7)

D3 should be =D2+(C3*7)

Format the date as you wish. Don't be surprized if you originally get a number like 40394. It is simply the number of days since 01-Jan-1900. Use the Home tab's Number section to select a date format you are comfortable with.
