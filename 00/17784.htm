Sounds like you are doing it right. Make sure to turn off the "Link to Previous" feature in 4 place, both left and right headers and footers.

Word treats "pages" as a logical concept only. So when you are using automation like "odd page" Word keeps track of it, but does not actually show a blank page in any of the preview modes.

The best way you can tell that the odd even page thing really is working is to use the gutter feature. When you are printing a book to be bound, real typesetters add an extra "gutter" margin on the side that is being bound so that you don't lose any of the text in the binding.

After you have the everything setup:
sections
unlinked headers and
different odd/even pages
page numbers

(hint I added text that explicitly said Section one Odd, (etc) so that I could assess if
I agreed that what Word was doing matched what I expected it to do.

Then you can go to Page Layout tab > Page Setup group > Margins drop down > Custom Margins command > to display the Page Setup dialog with Margins tab showing. In the Margins section set a value for "Gutter" (I used 2 inches to make it obvious, it will be removed or reduced after you are convinced) and gutter position to "Left".

Now if you look at the pages in "Print Layout" view you will see that things are working the way you want.
