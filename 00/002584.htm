Going back to the original post, there is a way to put a floating graphic in a specific spot on a page and keep it there regardless of the editing of the text.

  • Select the graphic.
  • On the Picture Tools | Format tab of the ribbon, click the Position button and click More Layout Options at the bottom of the gallery.
  • In the Vertical position section of the Picture Position tab of the dialog, change the right-hand dropdown from Paragraph (the default setting) to Margin or Page. The measurement in the left-hand dropdown will be automatically adjusted to the graphic's current position with respect to your choice.
  • Click OK.

There is one caveat: If editing of the text before the graphic's anchor paragraph causes the anchor to move to a different page, the graphic will also move to that page -- the graphic can never be on a different page than its anchor.

Remove Odd page break - Microsoft Community

cindyhutchings say:

Remove Odd page break

I have a large document.  Pages 1-15 are good, but page 16 keeps appearing after 15 which I want blank and want to start page numbering at 16 after blank page but stubborn section break (odd page break) won't delete.

suzanne s. barnhill say:

There are two ways to accomplish what you want:

  1. At the end of page 14, insert a page break. On the new page 15, insert another page break. This will leave page 15 blank (except for any header/footer), and the following page will be page 16.
  2. If you want page 15 to be entirely blank (no header or footer), use an Even Page section break after page 14 instead of a page break. An entirely blank page 15 will be inserted (though you won't see it unless you view facing pages in Print Preview), and the next page will be page 16.

Since you already have an Odd Page section break, you can change the section start type to Even on the Layout tab of Page Setup. For more information on "blank" pages and related issues, see http://WordFAQs.mvps.org/BlankPage.htm.

cindyhutchings say: I probalby was not clear enough.  My document is numbered great except after page 15, I want page 16 but the stuborn page 17 won't go away.

suzanne s. barnhill say:

An Odd Page break will force the page to have an odd page number and will insert a blank even page to accomplish this. You will not see the blank page (which is page 16) until you print unless you enable either "Mirror margins" or "Different odd and even" and view two pages in Print Preview (which will then show facing pages).

 

If you don't want an Odd Page break, then change the section start type to "New page" or "Even page" on the Layout tab of the Page Setup dialog. The easiest way to open the Page Setup dialog is to double-click at the top of the horizontal ruler, but you can also use the dialog launcher (small arrow in the bottom right corner) in the Page Setup group on the Page Layout tab.

ron sommer mvp 2009-2017 say:

Emails sent using the Samsung or Sony mail app will duplicate in the Sent folder.

The emails duplicate every time the Samsung or Sony mail app syncs.

The Sent folder can grow so much that the account may be blocked.

You will have to fix the duplicating problem.

 

There is an update to the Samsung mail app.  It fixes the Sent folder duplicating messages. 

 

Or remove the account from the phone app.

Download and install the outlook.com app.

Add the account.

 

Or you can setup the account to use 'Microsoft Exchange ActiveSync'.
AndyWickens replied on January 1, 2014

 
FIXED ON SAMSUNG GALAXY S2?

 

I had the same problem with hundreds of duplicated emails in my outlook sent mail folder. I have a Samsung Galaxy S2 and I use the native email app (which I prefer after trying many alternatives). I cleared the problem using the native email app on my phone.  I deleted the outlook account [open app; settings; bin in top right corner; delete] then added it back selecting 'Manual Setup'.  When the app then asks 'What type of account?' I selected 'Microsoft Exchange ActiveSync'.


The settings needed are as follows...

 

Domain\user name: \[Your outlook email address]

Password: [Your outlook password]

Exchange Server: s.outlook.com

Use Secure connection (SSL) [Tick yes]

Use client certificate: [Leave blank]


All your emails will be back in place on the phone within a few minutes once you have worked through a few more settings which are all ok at their default values.


I can now send emails from my Samsung with only one instance appearing in my outlook sent folder :)


If I get any further problems I will report back.


I got the Exchange ActiveSync settings from the following

http://windows.microsoft.com/en-GB/windows/outlook/send-receive-from-app
