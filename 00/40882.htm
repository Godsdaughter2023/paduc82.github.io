Login Information

  • Gateway - 192.168.0.1
  • Username - admin
  • Password - password

 MAC Filtering

  1. Open web browser.
  2. Login to router by entering http://192.168.0.1 into the address bar
  3. The router will then prompt you to enter the user name and password. The default user name and password are: "admin", "password".

  5. Select the Setup Access Control List button.
  6. Uncheck the Turn Access Control On box.
  7. Select Apply.
