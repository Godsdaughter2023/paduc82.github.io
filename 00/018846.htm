You didn't mention your version of OS X, but my guess is that you may be using one of the newer ones which has left Carbon-based software pretty much behind. My understanding is that the behavior you'd prefer to see [persistent folder location] is related to the OS now being completely Cocoa-based. Office 2008 is obsolete, deprecated software for which all support has been withdrawn. There will not be [and probably couldn't be] any steps taken to change that behavior.

In order to expedite accessing a certain folder I'd suggest that you either;

  • move your pictures into the Pictures folder which should be directly available from the Sidebar, or
  • temporarily add the folder containing your pictures to the Finder window Sidebar, or
  • select your folder from the dropdown list of Recent Places in the Choose a Picture dialog, or
  • Import your images into the Office 2008 Clip Art Gallery - which isn't strictly for 'clip art'.

Other file management options may be available as well depending on OS X version.

This issue might occur if some of the system files got corrupted.

I suggest you to try the below methods and check if it helps.

Method 1: Perform System File Checker (sfc) scan.

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

To perform SFC Scan, refer to the below link:

Deployment Image Servicing and Management (DISM) is a new command line tool that you can use to service a Windows image or prepare a Windows PE image. The DISM tool replaces the Package Manager (pkgmgr.exe), PEimg, and Intlcfg tools. DISM consolidates the functionality found in those three tools, as well as delivers new functionality to improve the experience of offline servicing.

Follow the steps to Run DISM Commands:

1) Press Windows + x on your keyboard.

2) Select Command prompt (Admin) to open Administrator Command Prompt .

3) In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

dism.exe /online /cleanup-image /scanhealth

dism.exe /online /cleanup-image /restorehealth
