Hi John,

Changing the clock format from 12-hour to 24-hour in Windows 10 is very simple. To do that, follow these steps:

1. Right click the Start button, then go to Control Panel.

2. Click Clock, Language, and Region, then click Region

3. Click the drop-down arrow next to Short time and select HH:mm

4. Then, click the drop-down arrow next to Long time and select HH:mm:ss

5. Click Apply, then OK.

Usually, when an audio device is connected to a PC, the default audio settings must be changed to allow Windows to use the connected device for audio. If that settings remains unchanged, even if the audio device is connected, the device will not work as desired.

Right click on the speaker/volume control:

Click on Playback devices. You can see a dialog box with different audio devices, like in the image:

Click on Bluetooth Headset (with mic) or Bluetooth Headphones (only audio output), the click on Set Default (greyed in the image, but the button works if the device is connected). Then click on Apply and then Ok.

If you can't see Bluetooth Headset/Bluetooth Headphones option, right click on the empty space inside the dialog box, click on Show Disabled Devices and Show Disconnected Devices options (make sure a tick is present). Then, you'll be able to see all the audio devices installed in the PC.

Still, if you're unable to see the Bluetooth devices option, you need to install additional audio drivers. Depending on the audio device present, you may need to install audio drivers from some other manufacturers (like Dell or HP) to use the Bluetooth functionality.
