Is it possible to get Solver run using my range of y values as objective values for each iteration? Alternatively, if there isn't a way to set the objective value to the value in a cell, is there another way to use a macro to run Solver through iterations with different objective values? Thanks.

bernie deitrick say: Of course, though you left out a lot of information. For example, with a formula that references D1 and D2 in D3, with E7:K7 containing the target values, and D8:D17 containing your other variable....

Put this formula in D3

=D2*D1

and fill E7:K7 and D8:D17 with integers, and run this - which requires a reference to SOLVER in your VBA project:

Sub SolverMacro()
    Dim X As Range
    Dim Y As Range

    For Each X In Range("D8:D17")
        For Each Y In Range("E7:K7")
            Range("D1").Value = X.Value
            SolverOk SetCell:="$D$3", MaxMinVal:=3, ValueOf:=Y.Value, ByChange:="$D$2"
            SolverSolve True
            Cells(X.Row, Y.Column).Value = Range("D2").Value
        Next Y
    Next X
End Sub

Web layout is more freeform, it is not limited by the printed page margins. But other than that it maintains most formatting

Outline view focuses on the heading. Page formatting is out of the window, most text formatting is still maintained.

Draft view strips out all formatting, just shows the text. It is intended to make editing as fast as possible.
