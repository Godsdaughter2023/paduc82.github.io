jack cook - aka help_line say:

So at first glance Andrew, it seems to me that probably the best and quickest way to get your space back (and more importantly to get your upgrades) is to strongly consider performing a hard reset. It will not take that long and the advantage is that you get a fresh start. If it were me, I would NOT restore from a backup but rather start fresh.

There are a couple of reasons for doing that. All too often we have (overtime) installed a lot of programs that we never use or rarely use so they just take up space. Most likely, if you did not reinstall them, you would not even miss them. If you restore from a backup, it will set all those programs as pending and it always seems to take forever to download.

I find that not restoring from a backup, my device gets setup very quickly and I am up and running in a short period of time.

Use your same Live ID so all your contacts and calendar return and as long as you have enabled auto backup to OneDrive when near a Wi-Fi, all your photos have been stored in OneDrive. You might want to check that your photos are there before you hard reset if you choose to go this route. 

Once it is setup and BEFORE you add back any must-have apps, check for updates. It is a perfect time to install them since there is all sorts of room on a freshly setup device. Once completed you can then add back the programs you need and slowly add back others that are not as important or rarely used.

That is my advice, I have been doing it this way for a long time and it has always worked well for me. In fact, I just hard reset my 950XL two weeks ago. Over time I had accumulated too much and it is nice to start fresh ... it works well!

Just as an aside, if you have deleted and there is still not enough room, something that was installed did not delete the installation files as it should and there are a lot of temporary files stored for some reason. I truly believe that a fresh start is the way to go. In the final analysis, it is a faster and more efficient way to get your device back the way you want it. Just my thoughts....

Good luck with your decision!!

larryyu_msft3110 say:

Hi OliverMunday,

This may because the shared boxes’ size is too large. By default, the shared box items are downloaded into your local cache OST file if your Office 365 mailbox is configured in Exchange cached mode. A too big OST often leads to bad Outlook performance.

To determine this, please try to export the shared items to a PST file as backup and then delete some old items to reduce the size. After that, you can also let the users view shared items via opening the PST file in the Outlook client. Feel free to let us know if you encounter any problem during the progress.

If the issue persists after that, please let us know and we will further investigate it.

Larry

The article applies to Outlook 2010 and Exchange Server 2010.
