ken sheridan say: Could you inadvertently be printing the form?  This could happen if you've moved focus back to the form while the report is still open in print preview.  When adding such a button to a form I would normally either add a second button to print the current record, changing the View argument to acViewNormal, or include a 'Send to printer' check box, with a default value of False, in the form and then change the code to:

    Dim strWhere As String
    Dim intView As Integer

    If Me.chkSendToPrinter Then
        intView = acViewNormal
    Else
        intView = acViewPreview
    End If

    If Me.Dirty Then    'Save any edits.
        Me.Dirty = False
    End If

    If IsNull(Me.Inv) Then     'Check there is a record to print
        MsgBox "Select a record to print", vbInformation, "Invalid Operation"
    Else
        strWhere = "[Inv] = " & Me.[Inv]
        DoCmd.OpenReport "Preprinted", View:=intView, WhereCondition:=strWhere
    End If
 
Note the amendment to test for a current record by checking to see if the primary key, which I assume to be Inv, is not Null.

BTW I'd recommend you add error handling to this procedure in case an attempt is made to save the current record while any required field is Null.

Can a Desktop PC works like a Laptop in PowerPoint Presenter View? - Microsoft Community

matthewwong say:

Can a Desktop PC works like a Laptop in PowerPoint Presenter View?

Consider this:

You use a Laptop and go for a presentation in a Lecture Theater.  You connect it to the projector there.  The Laptop is set to "Duplicate" so that you can show some web pages or the whole Desktop to audiences. 

Your PowerPoint has the option "Advanced"; "Display" - "Automatically extend display when presenting on a laptop or tablet" select.  So when you show a PowerPoint Presentation, it will automatically change the display to "Extend" and use "Presenter View".  After showing the PowerPoint, it will automatically change back to "Duplicate" and you can continue to show your Desktop.

We want to install a Desktop PC in Lecture Theater to do the same thing.  The PC supports 2 monitors.  The problem is that it does not work like a Laptop.  The option "Automatically extend display when presenting on a laptop or tablet", as it said, does not work for Desktop PC.

Is there a way to enable it on Desktop PC?

This apply to Windows 7, 8.1, 10 and Office 2010, 2013.

Thanks.

a. user say:

Hello Matthew,

Welcome to Microsoft Community and thank you for posting your query.

Please provide more information before we proceed.

Did you set monitor under Slide Show as Automatic?

I suggest you to change the Monitor setting to Automatic under slideshow tab and check if it works. To do so, you may open PowerPoint, connect secondary display and click on Slide Show tab> Monitors> select Automatic from the dropdown. You may also check if Presenter view check box is checked under Slide Show tab> Monitors> Use Presenter view.

Try running slideshow now and check if it works as expected.

Hope this information helps. Please post back with the updated issue details for further assistance.

Thank you.

matthewwong say:

Thanks for your help.

We know that in order to use Presenter View, the Windows Monitor setting (Display under Control Panel) must set to Extend.

Yes. When the Desktop PC is set to Extend, Presenter View works perfectly. 

In Laptop PC, PowerPoint can automatically change the Monitor (the setting under Display of Control Panel, not Slide Show of PowerPoint) from Duplicate to Extend if the PowerPoint option "Automatically extend display when presenting on a laptop or tablet" is set.

This option does not work if it is a Desktop PC.

The "Slide Show tab> Monitors> select Automatic (or selected monitor)" and "Slide Show tab> Monitors> Use Presenter view" is a must on both Laptop and Desktop for Presenter View to work.

It is just that for a Desktop installed in Lecture Theater, it would be better for it to have Duplicate mode (for showing Windows Desktop) automatically change to Extend (for PowerPoint Presenter View) like what can be done on Laptop.

I cannot find a way for Desktop to do so.  And it is strange why Microsoft make the option "Automatically extend display when presenting on a laptop or tablet" not work for Desktop PC.

Regards.
