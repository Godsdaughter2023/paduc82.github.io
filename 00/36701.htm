lemp say:

When you use msconfig to change how the system starts, those settings will remain until you run msconfig again and select "Normal startup."

Start your computer.

Does it actually say "Safe Mode" on the screen when the computer starts?  If so, you need to make a change on the "Boot" tab of msconfig as explained below.

Click Start, type msconfig.exe in the Search screen, and press Enter.

Select the "Boot" tab.  UNcheck the box next to "Safe boot"

Now select the "Services" tab.  Check the box at the bottom left to "Hide all Microsoft services."  Click the "Disable all" button.

Now select the "General" tab.  You should see the following, with the radio button next to "Selective startup" selected and the box next to "Load system services" unchecked.  The other boxes under "Selective startup" will be checked.

At this point, you can either click OK and restart or uncheck the box next to "Load startup items" and then click OK.  If you uncheck "Load startup items," you'll see that the radio button next to "Diagnostic startup" gets selected instead of "Selective startup."  This is the "clean boot" configuration. 

The procedure described in that other thread left the startup items (programs) enabled and only disabled all non-Microsoft services (and then re-enabled 5 of those services at a time).  You might as well try this.  If you don't get the error message when non-Microsoft services are disabled but normal startup items are allowed to start, you won't have to go through the tedium of selectively re-enabling the startup items to see if it's one of them that's causing the problem.

When you're done -- or anytime you want to get back to a "normal" startup (that is, not starting in Safe Mode), run msconfig and select "Normal startup" on the General tab and then click OK.  That should re-set everything for the next restart.

Hi Aditya,

 

Hybrid sleep is designed primarily for desktop computers. Hybrid sleep is a combination of sleep and hibernate. Hybrid sleep saved any open documents and programs in memory and on the hard disk, and puts the computer into a low-power state so that you can quickly resume your work. 1. Stop the Windows Update service (make sure you are logged on as an Administrator)

Double-click Services

Start button || Control Panel ||System and Maintenance ||Administrative Tools
