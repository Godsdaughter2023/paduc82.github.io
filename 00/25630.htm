tasos_k say: Hi,
try this code...

Sub PrintPreviewEachRow()
Dim ws As Worksheet
Dim myRow As Range
Dim i, c As Long
Set ws = ActiveSheet
Application.DisplayAlerts = False
For i = 1 To 5 ' PRINT PREVIEW from Row1 up to Row5
c = ws.Cells(i, Columns.Count).End(xlToLeft).Column
If Cells(i, 1) = "" And c = 1 Then GoTo abc
Set myRow = ws.Range(ws.Cells(i, 1), ws.Cells(i, c))
With ActiveSheet
With .PageSetup
.PrintArea = myRow.Address
.Orientation = xlLandscape
.CenterHorizontally = True
.Zoom = False
.FitToPagesWide = 1
.FitToPagesTall = 1
End With
End With
ActiveWindow.SelectedSheets.PrintPreview
abc:
Next i
ActiveSheet.PageSetup.PrintArea = ""
Application.DisplayAlerts = True
End Sub

jay freedman say:
It sounds like your table has been split into separate page-sized tables. You should be able to rejoin them by deleting whatever is separating them.

First, turn on nonprinting characters by clicking the ¶ button on the Home tab. At the bottom of each page or at the top of the next page, look for a paragraph mark (the ¶ symbol) or a dashed line that contains the words "Page Break" or "Section Break (Next Page)" (or maybe some other kind of Section Break).

Each time you find one of those line separating two sections of table, place the cursor in the left margin next to that line and click to select the paragraph, and the press the Delete key. When you finish, you should be able to select and sort the whole table as one piece.
