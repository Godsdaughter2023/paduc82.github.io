Why are there different language settings in the same PowerPoint - Microsoft Community

a. user say:

Why are there different language settings in the same PowerPoint document

In PowerPoint, why are the language settings separate for the context boxes in slides and the speaker notes, even after changing default language settings? What are the reasons for this?

bob jones aka: cybertaz say:

The Default Language affects nothing other than what is used when creating a new presentation file or adding a text container of some sort to an existing presentation file. It does not change anything about the ones in an existing file that have been formatted to use a Language which differs from the Default.

You can select multiple text containers on a single slide to change their Language formatting, but there is no provision in the program to universally change the Language of all containers in the file at once except for text which is in standard Placeholders.

For content in Placeholders you can use Outline View to select all text displayed there, then go to Tools> Language to select the one you want to apply & click OK [not Default]. However, any content in Text Boxes or other drawn Shapes will not be affected.

nayana s m say:
Follow the steps below to enlarge a worksheet:
1.       Select the range of cell that you want to print.
2.       Click File > Print > Page Setup
3.       On the Page tab, under scaling, Click Adjust the percentage.
Follow the steps below to fit the worksheet to the paper width of printed pages:
1.       Select the range of cell that you want to print.
2.       Click File > Print > Page Setup
3.       On the Page tab, under scaling, Click fit to
4.       In the second fit to box enter 1 to fit the worksheet to the width of one page and so on.
