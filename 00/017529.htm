rajesh c. say:

YES - very much.

You can achieve the functionality for looking Left and not Right.

But this cannot be achieved by using VLOOKUP.

You will have to use a combination of INDEX and MATCH functions together.

The INDEX function is used as follows:

=INDEX(A1:D20,12,3)

In the range A1:D20 (see the 1st parameter in the above formula), the index function will

go to the 12th row (see the 2nd parameter in the above formula)

and the 3rd column (see the 3rd parameter in the above formula)

and fetch the value from that position and get it for you.

Now, the MATCH function is used as follows:

=MATCH(A1:A20, 12, 0)

In the range A1:A20 (see the 1st parameter in the above formula)

the MATCH function will go to the 12th position (see the 2nd parameter in the above formula)

and get you the value from that position.  Usually the 3rd parameter is always 0 (or FALSE).

Now, in MATCH function above we have taken the range as a column and hence we are able to fetch

a row value from that column.

One can similarly fetch a column-value from a row using the following MATCH function:

=MATCH(A1:D1,3,0)

In this case the match function will fetch the value from the 3rd position (cell) in the range A1:D1.

If we combine the INDEX and MATCH function as follows then you can easily fetch a value from the

left side of your key column (a functionality to VLOOKUP on the Left side instead of Right).

=INDEX(A1:D20,Match(RowSearchKey,A1:A20,0),MATCH(ColumnSearchKey,A1:D1,0))

In this case, the first match function will fetch you the row position (12 in our above example) and the

second match will fetch you the column position (3 in our above example) thereby

bringing the Index function to look like =INDEX(A1:D20,12,3).

How to Delete Line Following a Table - Microsoft Community

curious query say:

How to Delete Line Following a Table

Can anyone tell me if it is possible to delete the line that is inserted after a table?

herb tyson say:

If you mean the paragraph mark, no, it cannot be deleted. All Word documents contain at least one paragraph mark. So, if your document contains only a table, you will always have a paragraph mark following that table. If your document has a table as the last content, it will always be followed by a paragraph mark.

Why do you want to delete it? Is it causing a problem?

Or... if you mean an actual line, then you'll need to explain what you're seeing, since inserting a table does not ordinarily produce a line after it (just table borders, but presumably that's not what you mean, since they're part of the table, and do not come after it).
