Hi mate.

I've experienced the same issue with FSX numerous times, even though it's supposed to create the  desktop  'shortcut'.    There may be other ways of resolving this issue, but I only know of one, which is as follows :

1.  On your pc,  open 'My Computer',  or use  'File Explorer'  and  find /  go to the main internal hard drive on your system.  It might appear as  OS (C:)  or just  C:

2.  Once there  'double click'  on the  C,   and it  should  produce  all of the programs and files within it.

In that list  there  'should'  be  a file / folder  called  'Program Files'.  Double click / open  'Program Files'  folder.

3.  Once  you're in the  'Program Files'  folder,  look for  'Microsoft Flight Simulator X'.

Note  -  if it is  'not'  there,  find a folder called  'Microsoft Games'.  FSX  'should'  be in there if not as described above.

4.  Once you've found the folder,  double click  again to open it.

5.  When you're in the folder  (i.e.  Microsoft Flight Simulator X),  there are many other sub-folders, programs and files that make up the  FSX software/program.   Scroll down and search for the  FSX icon   (blue square with a white aeroplane).  You will probably find  several results, but to my understanding,  the relevant one simply reads as  ''fsx''  in lower case,  and is called an 'application'  in the 'Type'  column  if/when viewing the folder as a 'List' of items as opposed to other ways the folders contents can be viewed.

6.  Using your mouse pointer,  put it over the icon  and then  use / press your   'RIGHT Click button'.   This should open up a  small  window with  a list of  options of things you can choose to do,  (from  'Open',  or  'Run as Administrator',   to  'Properties'  and more).

What you are looking for,   (as per the way I've resolved the problem)   on  this pop-up list/window  is  .  .  'Send to'    with  a pointer  like this    >   

7.    If you move your mouse pointer over   'Send to',   which  opens a further list.  From the options in this list, ther  'should'  be  one  that  reads  .  .  Desktop (create shortcut).

Now check your desktop, and the icon you just clicked should be there for you to use to start / open  fsx.

Summary

a.  open  'My Computer'  -  find the main internal hard drive  (e.g.  C:)  and open it.

b.  Find  'Program Files',  and  open that folder.

c.  If  FSX  is not listed within 'Program Files',  check  'Microsoft Games'  folder.

d.  Open the FSX folder,  find the  fsx  icon,  'RIGHT Click',    select  'Send to'  . . . . .

e.  From  'Send to'    select  'Desktop (create shortcut)'.

f.  Check the icon/shortcut is there and see if it open's  FSX.

If you haven't been able to find fsx where it's been described above,   maybe use 'File Search'  in File Explorer in case it's been saved elsewhere.

One last note  -  in the end description where you get the window open from 'Right Click',  there is an option in that list  labeled  as  'Create shortcut',  without even going to the  'Send to'.    I recall I tried that option recently for something else and it failed to  produce a shortcut,  or if it did, it's hidden it from being easily found.  Just mentioning it so that you don't end  up  with a negative result.

I hope this helps, and good luck.
