ossiemac say:

I realize that this post is quite old and wonder if you ever found the correct answer or that you are still a member of the forum. The only way to group Forms Control Option Buttons on a worksheet is to use the Group Box around them.

Selecting the required option buttons and then right click and selecting Group does NOT work as far as separating the option buttons into separate groups.

Create the Option Buttons and then select the Group Box (Form control) and size the Group Box to entirely contain the option buttons for the group. The Option Buttons must be entirely contained in the Group Box and not hanging outside at any point.

If you then set a Linked cell for one of the Option Buttons then that same linked cell automatically applies to all of the Option Buttons in the Group. However, instead of the cell displaying True or False, it displays a number equivalent to the number of the option button in the group. The numbers are automatically applied and Excel reads the Option Buttons across and down within the group. You can get the True/False of each clicked option button but referencing the number in the linked cell is and easy method to determine which option button is selected.

Tip: For the linked cell you can hide the value by setting the Font color to same as Fill color. I do this and use a cell that is under either the first or last option button of the group and the number is not visible.

steven. s say:

Hey Grandpa,

 

The "Safe Search" referenced in those steps above is a general term for the filter that is enabled by default in most search engines to filter out mature content. To complete those steps you will need to visit the website of the search engine you want to unblock and use their preferences to choose that option. Usually the search engine will save this information with a cookie and you will be able to search going forward without the filter enabled.

 

In short, to accomplish this you will need to visit the actual website of the search engine you want to use (instead of using the search bar at the top right corner of Internet Explorer). Then find that search engine's preferences area and disable the content filter.

 

Hope that helps to clarify why you were getting different results from the steps above.
