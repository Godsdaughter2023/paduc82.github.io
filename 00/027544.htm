karl timmermans [outlook mvp 2012-2018] say:
>>I'm thinking to delete the email account which created the ost file (which can't be imported in future?)

Just to get a couple of things out of the way for the sake of accuracy (which appear to be missing from the original description

#1 - There are on;y 2 types of email accounts that would result in an OST file being created - (1) Outlook.com/Exchange & (2) IMAP
#2 - Only an IMAP account would have any folders created with the added description "This Computer Only" (as of Outlook '2013/'2016)
#3 - IMAP accounts do not sync contact/calendar data so if the OST file is lost - that data will also be completely lost unless backed up (exported) to a PST file
#4 - Generally there is no need to import a PST file, just needs to be opened directly

Primary question(s) that come to mind are "why do your PST files have so many "This Computer Folders" unless imported from different accounts?
