rich007a say:

Hi.

The max of a data set can be returned using, for example:

=MAX(C1:C30)

If tonnage is in C1:C30, and the crew is listed in B1:B30, the crew on the day the max tonnage was hauled is given by:

=INDEX(B1:B30,MATCH(MAX(C1:C30),C1:C30,0))

 

and likewise if the date is in cells A1:A30, the date on which the max tonnage was hauled is given by:

=INDEX(A1:A30,MATCH(MAX(C1:C30),C1:C30,0))

 

Hope that helps.

Cheers
Rich

PS. Note that if the same maximum tonnage was hauled on more than one day, the above formulae will return only the first day/occurance/crew/etc. that that maximum tonnage was hauled.  For this reason, it might be better (depending on your needs) to sort the data so you can see what really happended in a given month.  Use Sort on the Data tab for this.

how to restore backup of my Lumia 520

jack cook - aka help_line say:
Restore a backup to my phone

If you lose your phone or decide to upgrade it, you can restore your apps, text messages, accounts, and many settings from your old phone when you set up a new phone. All you need is a Microsoft account and a backup from your old phone.

To learn how to create a backup, see Back up my stuff.

To restore a backup

  1. When you first set up your phone, sign in using the same Microsoft account you used when you created your backup.

  2. When prompted to restore your phone, tap the backup you want to use, tap Next, and then tap Next again.

  3. Wait for the backup to download, and then tap Next.

  4. Do one of the following:

    • Enter passwords for your restored accounts, and then tap Next.
    • Tap Skip if you want to enter your passwords later.

  5. Tap Done.

Notes
  • Previously saved app data won't be restored.
  • Start will return to its factory default layout when you restore a backup.
  • You'll need to set up Wi-Fi connections again after you restore.

jay freedman say:
Probably the simplest way is to create a document that does contain the 'No spellcheck' style, apply the style to any bit of text -- such as a paragraph mark -- and copy that to the clipboard; then open an existing document that doesn't contain the style and paste into it. By looking at the Styles pane (Ctrl+Alt+Shift+S) you can confirm that the style is now present in the document, and you can save the document.

A somewhat more complicated but efficient method is to use the Organizer. With the Styles pane open, click the Manage Styles button (the third button from the left at the bottom of the pane). In the Manage Styles dialog, click the Import/Export button, which opens the Organizer dialog.

The Normal.dotm template's styles should be shown on one side (usually the right). On the other side, click the Close File button and, when it changes to an Open File button, click it again. Choose any document that you want to add the 'No spellcheck' style to. Click the 'No spellcheck' style in the Normal.dotm list and click the Copy button. Click the Close File button and agree to save changes. Now you can click the Open File button and repeat the procedure for another existing document.

If you want to add the style to all existing documents, you need a macro. I don't have one handy for this particular purpose, but the one in http://www.word.mvps.org/FAQs/MacrosVBA/BatchFR.htm could be adapted to do it.

charles kenyon say:
Word is a word processor, not a programming language, unlike FORTRAN, which does not do much as a word processor.

Documents are intended to be something that are printed. Templates are intended to be used to create new documents. Adding a style to templates allows all new documents based on the templates to use that style.

You could easily create a global that contains your style. A macro in it could be set to fire on the Document_Open event and transfer the style to the opened document. After a period of time all of your most-used documents will have the style. You could then modify the global template so that it would fire the macro upon use of a keyboard shortcut instead.

You could also create a macro that adds the style to all documents and templates in a given folder.

jay freedman say:
OK, this code should go into the ThisDocument module of your Normal.dotm template (not a global add-in in the Startup folder):

Private Sub Document_Open()
    Dim newStyle As Style
    Dim styleToAdd As String
    styleToAdd = "No Spellcheck"
    If Not StyleExists(styleToAdd) Then
        MsgBox "adding " & styleToAdd & " to " & ActiveDocument.FullName
        Set newStyle = ActiveDocument.Styles.Add(Name:=styleToAdd, Type:=wdStyleTypeCharacter)
        newStyle.BaseStyle = ActiveDocument.Styles(wdStyleDefaultParagraphFont)
        newStyle.NoProofing = True
    End If
End Sub

Function StyleExists(styleName As String) As Boolean
    Dim tryStyle As Style
    On Error Resume Next
    Set tryStyle = ActiveDocument.Styles(styleName)
    If tryStyle Is Nothing Then
        StyleExists = False
    Else
        StyleExists = True
    End If
    Err.Clear
End Function

It will add the No Spellcheck style to each existing document you open that doesn't already contain that style. You can delete the MsgBox line if you don't want to see a popup when the style is being added.

Note to Charles: AutoXYZ and Document_XYZ macros don't run if they're stored in add-in templates in the Startup folder. Refer to http://word.mvps.org/FAQs/MacrosVBA/DocumentEvents.htm.
