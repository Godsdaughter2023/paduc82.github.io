On some occasions the CHKDSK would end quickly without any message(s), sometimes it would fail with 'An unspecified error occurred (696e647863686b2e dee).'

Either way, the write-protect problem usually went away, but it also came back after the next boot, not necessarily on the same drive(s).

The drives that were randomly going into write-protect were all connected via a GBB36X ( rebadged JMB36X) controller chip on my GA-P35C-DS3R m/b, with chip driver version 1.17.42.8 (Win 7 couldn't find a later driver).

brian tillman [outlook mvp 2007-2019] say:
First question: Is there a difference between Windows Mail and Windows Live Mail?

Second question: Is one of the two already in Windows 7?

Third question: I was going to do a Windows Easy Transfer from my computer running XP with Outlook Express to my new computer with Windows 7. If Windows 7 already has Windows Mail or Windows Live Mail will all my messages etc. in Outlook be transferred into the new mail program?

Fourth question: If Windows 7 does not have one of the mail programs mentioned should I download one of them into the old computer and move my messages etc. from Outlook into the new program and THEN do an Easy Transfer into the new computer (after I also download the same mail program into that computer if it doesn't already have it)?

If this has already been addressed in another post please direct me there.  If I should be asking this question elsewhere please tell me where.

1) Windows Mail is the Vista equivalent of Outlook Express.  Windows Live Mail is very similar to both of those.  It is a POP/IMAP/NNTP client just like the OE and WM, but it also implements the DeltaSync protocol that lets it access a Hotmail/Live account via MAPI, giving it access to the contacts and calendars available to a Hotmail/Live mailbox.

2) Neither are built into WIndows 7.  Windows 7 contains no built-in email client at all.  (In reality, the Windows Mail executable is on Windows 7 but it doesn't work.  Some people have gotten it to work, but I wouldn't want to rely on it.

3) Outlook Expresss and Outlook are not the same thing.  Outlook, a part of Office, is a PIM (Personal Information Manager), which is more than just a mail client.  Using Windows Easy Transfer (or any other simple copy procedure) can transfer an OE database to WIndows 7 and WIndows Live Mail can import it.  I personally would not use WET.  Transferring Outlook data between PCs using WET is problematic and WET usually damages an Outlook mail profile.  There are better ways to transfer Outlook data between PCs.

4) If you purchase a license for Outlook and install it on the old PC, you can transfer your Outlook Express contacts and email to Outlook on that PC and then install Outlook on the new PC and transfer the Outlook data.  You can also install Windows Live Mail on the old PC, transfer the data from Outlook Express to WLM, then transfer the WLM database to the new PC and import it into the second installation.
