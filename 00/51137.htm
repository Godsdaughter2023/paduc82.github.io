cmdrkeene say:
It is likely that the program is changing the registry directly (which is not the correct method a program should use, but then again some programmers are lazy and don't follow the rules).  When the registry is modified directly, the checkbox in the options gets out of sync (which is why you have to uncheck it and re-check again).

You might explore the programs options to see if there is a setting there to stop it from making changes.  This would be the best solution, of course, because it solves the issue at the source.  If not, I'd suggest contacting the company that makes that program.



Having said that, you might also be able to simply take away the program's ability to change that registry option.  Doing this is not recommended or supported.  If the program tries to change the registry option and gets blocked, the program might crash or behave unexpectedly.  Also, making changes to the registry can result in problems -- including problems that might render your computer unusable.  If you decide to make registry changes, you should make a System Restore checkpoint first so that you can revert back if problems develop. It's also always a good idea to have backups of your system and/or files, just in case.

With that caution in mind, here's what I'd do if it was my computer and all other solutions have failed.  As long as you follow these instructions, there should be no danger involved.
  1. First, make sure that Window dragging is working as you want.


  2. Open the Registry editor (regedit) and return to the same location as before:
    HKEY_CURRENT_USER\Control Panel\Desktop


  3. Right-click the Desktop key (folder) and choose Permissions...


  4. Select your username in the list, then un-check the Allow Full Control box.
    Caution: Do not check the deny box!  Doing this will cause you to be unable to change this setting in the future, meaning you'll never (ever!) be able to change your desktop settings again.


  5. Do the same thing for the Administrators:  Select the Administrators item in the list, then un-check the Allow Full Control box.  Again, never choose deny.


  6. Click OK to save these Permission settings, and close the registry editor.

After you do this, you are taking away the ability to modify the desktop settings.  Note that you'll not be able to change your wallpaper, colors, or other options either (unless you undo the above changes to restore your permissions).

Again:  this is not a recommended or supported solution, but it's something I would do (because I'm a nerd and get a sick satisfaction out of tweaking my computer in improper ways).
