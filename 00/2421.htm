Why do my files end with.docx, not .doc? - Microsoft Community

jetpack_b say:

Why do my files end with.docx, not .doc?

Why has the file extension for Microsoft Office( in this case, MS Word). changed to end with x on the end of the file extension? Can I still open files from old MS Office, like .doc files?
jetpack_b say: As of MS Office 2007+, your files will now be saved with an 'x' on the end of the file. There is no need for concern. You can still open .doc files with MS Office Word, but you will be prompted to convert them to .docx constantly, although this is not required to read the file.

The x on the end of the file extension is no cause for worry. This does not indicate a virus or problem with your software, but simply that the file was created with MS Office 2007 or later.
This format is called OOXML.

More information on OOXML can be found here: http://searchwinit.techtarget.com/definition/OOXML

Try:

Alt-F11 to open the VBA editor

Alt-IM to insert a new code module

In the new module, paste the following code

'==========>>

Option Explicit

'---------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet, destSH As Worksheet
    Dim srcRng As Range, destRng As Range
    Dim ArrIn As Variant, arrOut() As Variant
    Dim i As Long, j As Long
    Dim CalcMode As Long
    Const sReportSheet As String = "Report"  '\\ Name of Report sheet  '<<====  Change
    Const iColumns As Long = 20  '\\ Number of report columns            '<<====  Change

    Set WB = ActiveWorkbook
    On Error GoTo XIT
    With Application
        CalcMode = .Calculation
        .Calculation = xlCalculationManual
        .ScreenUpdating = False
    End With

    With WB
        Set destSH = .Sheets(sReportSheet)
        ReDim arrOut(1 To .Worksheets.Count - 1, 1 To iColumns)
    End With

    For Each SH In WB.Worksheets
        With SH
            If Not .Name = sReportSheet Then
                i = i + 1
                Set srcRng = .Range("B1").Resize(iColumns)
                ArrIn = srcRng.Value
                For j = 1 To iColumns
                    arrOut(i, j) = ArrIn(j, 1)
                Next j
            End If
        End With
    Next SH

    Set destRng = destSH.Range("A2").Resize(i, iColumns)
    destRng.Value = arrOut

XIT:
    With Application
        .Calculation = CalcMode
        .ScreenUpdating = True
    End With
End Sub
'<<==========


Alt-Q to close the VBA editor

Alt-F8 to open the macro window

Select Tester | Run

To populate a report sheet which contains only the column headers or to add only new sheet data to a populated report sheet, replace the previous code with the following version:

Alt-F11 to open the VBA editor

Alt-IM to insert a new code module

In the new module, paste the following code:

'==========>>

Option Explicit

'---------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet, destSH As Worksheet
    Dim srcRng As Range, destRng As Range, findRng As Range
    Dim ArrIn As Variant, arrOut() As Variant
    Dim i As Long, j As Long, LRow As Long
    Dim CalcMode As Long
20  '\\ Number of report columns            '<<====  Change

    Set WB = ActiveWorkbook
    On Error GoTo XIT
    With Application
        CalcMode = .Calculation
        .Calculation = xlCalculationManual
        .ScreenUpdating = False
    End With

    With WB
        Set destSH = .Sheets(sReportSheet)
        ReDim arrOut(1 To .Worksheets.Count - 1, 1 To iColumns)
    End With

    For Each SH In WB.Worksheets
        With SH
            If Not .Name = sReportSheet Then
                With destSH.Columns("B:B")
                    Set findRng = .Find(What:=SH.Range("B2").Value, _
                                        After:=.Cells(1), _
                                        Lookat:=xlWhole, _
                                        LookIn:=xlValues, _
                                        SearchOrder:=xlByRows, _
                                        SearchDirection:=xlNext, _
                                        MatchCase:=False)
                End With
                If findRng Is Nothing Then
                    i = i + 1
                    Set srcRng = .Range("B1").Resize(iColumns)
                    ArrIn = srcRng.Value
                    For j = 1 To iColumns
                        arrOut(i, j) = ArrIn(j, 1)
                    Next j
                End If
            End If
        End With
    Next SH

    With destSH
        LRow = .Cells(Rows.Count, "A").End(xlUp).Row
        Set destRng = .Range("A" & LRow + 1).Resize(i, iColumns)
    End With
    destRng.Value = arrOut

XIT:
    With Application
        .Calculation = CalcMode
        .ScreenUpdating = True
    End With
End Sub
'<<==========

Alt-Q to close the VBA editor

Alt-F8 to open the macro window

Select Tester | Run

If the report sheet has only headers, this version will add data for all sheets;  if the report sheet is already populated with sheet data, this code will append data only for new/missing sheets.
