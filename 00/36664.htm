considerate_guy say:

In the 2.x versions of Movie Maker, your photos get stretched to match the aspect ratio of your project. 1024 x 768 is a 4:3 aspect ratio, so if you want to avoid stretching, set your project to the same aspect ratio (Tools > Options > Aspect Ratio).

The other option, if you want a 16:9 project is to crop the photos to that ratio. Paint Shop Pro can be configured to crop to a 16:9 aspect ratio, if that's what you want.

Another choice is to use WMM 6.0 which won't alter the aspect ratio of the photo but will instead center it on a black background.

Install Windows Movie Maker 6.0

Version 6.0 takes advantage of the GPU to render effects and transitions and has a 64-bit version rather than just 32-bit, but has some minor differences particularly for the scrolling text which isn't as steep as in 2.x. I wouldn't upgrade however, if you have a lot of custom transitions and effects unless you want to take advantage of the new capabilities that the GPU rendering adds.

Custom Transitions/Effects/Titles for WMM 6.0

If you do want to try WMM 6.0, I would back up your collection files and your project files. Realize that once you open a project file from 2.x in WMM 6.0, it gets converted to the new format and wouldn't be readable going back. If you save with a different name or have a backup, then you could revert to WMM 2.6 if you wanted to.

IMPORTANT
If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media
