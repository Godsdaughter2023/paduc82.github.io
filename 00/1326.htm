How to insert a date range in criteria of funtion SUMIFS in excel - Microsoft Community

amarjeetmehto say:

How to insert a date range in criteria of funtion SUMIFS in excel

I wanted to insert a date range in the criteria of funtion SUMIFS, how to do it ??

=SUMIFS(Sheet1!J:J,Sheet1!D:D,Sheet2!A5,Sheet1!G:G,{date range})

Column G of Sheet1 is a date column.

Date range : "01-05-2016 to 10-05-2016"

jacques raubenheimer say:

The trick to this is realising that SUMIFS allows you to specify the same criteria range multiple times, each time with a different criterion.

=SUMIFS(Sheet1!J:J,Sheet1!D:D,Sheet2!A5,Sheet1!G:G,"<"&DATEVALUE("2016/04/01"),Sheet1!G:G,"<"&DATEVALUE("2016/02/01"))

You could also use Date(Y,M,D) instead of DATEVALUE, as you prefer.

Try below formula:

=IF(D7=D13,3,IF(OR(D8=D13,D9=D13),2,IF(AND(D10=0,D10<>""),1,"")))

If D7 = D13, formula returns 3;

If D8 = D13 OR D9 = D13 (and D7 <> D13), formula returns 2;

If D10 = 0 (and D7 / D8 / D9 <> D13), formula returns 1;

If D10 <> 0 (and D7 / D8 / D9 <> D13), formula returns BLANK - the formula returns BLANK even if D10 is blank in this case;
