karl timmermans (outlook mvp 2012-2018) say:
>>
Are there any suggestions on whether I might still recover my notes and how to re-stabilize Office without losing anything else?
<<

From the description, what you have is an orphaned OST file which may have the data file you're looking for but unfortunately the only way to determine that is to use an OST2PST conversion program. An OST file can only be opened by the original Outlook profile/email account that created it. It seems that either the account was reconfigured or Outlook created a new OST file for the account due to some kind of problem which would have resulted in the data in folders with the added words "This Computer Only" in the folder description being lost.

There are no shortage of OST2PST conversion programs when doing a search on "OST2PST". Download and test any program prior to making any purchase.

In order to avoid this type of issue recurring in the future, would advise doing the following

#1 - Create a new PST file in the profile
Go to Home ribbon --> <New Items> --> <More Items> --> Outlook Data file

#2 - When #1 done, make this PST file the default data file for the profile
File --> Account Settings --> Data Files tab --> select the PST file --> set it as the default

#3 - Copy the items from within the "This Computer Only" folders to the corresponding folders in the PST folder tree
- Select the desired source folder
- press CTRL+A to select all items
- right-click on selected items --> Move --> Copy to Folder --> select destination folder
* am suggesting <Copy to Folder> instead of <Other Folder> purely as a precaution in case something goes wrong during the copy phase. The <Copy> option will not delete anything from the source folder whereas the <Move> option does
** for Calendar and Notes folder, you will need to change the <View> to a "List" in order to complete the above steps
