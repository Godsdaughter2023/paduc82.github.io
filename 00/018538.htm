How to set password protection using Mac OS - Microsoft Community

rich3500 say:

How to set password protection using Mac OS

I would like to set password protection for a document created on a Mac with OS 10.13.6 (High Sierra) and using MS Word for Mac v15.22.1.  The web advises me to set this under "Word", "Preferences", "Security", then type in the proposed password.  Problem is the version of MS Word noted above does not include that option under that pathway.  I also tried as part of the "Save As" process, which works on a PC but I do not get any security options when using the Mac.  Can someone point me in the right direction to password protect a document on a Mac?

bob jones aka: cybertaz say:

First, do as Eric Lu suggested - Update Microsoft AutoUpdate [MAU] then update your Mac Office software. The build you're currently running is obsolete by nearly 3 years.

Once you've updated the software: What you've read on the internet is incorrect. Password protection of a document in Mac Word 2016 can be applied by going into the Password Protect dialog in either of 2 ways:

  • Tools> Protect Document... in the main menu, or
  • By clicking the Protect Document tool on the Review tab of the Ribbon.

There are 2 types of protection but you don't specify which you want to use:

  1. Security - or File Level protection - which prevents anyone from opening the file without supplying a password. It also includes the option to allow modification (make the document open as Read Only) which allows anyone to open but not make any changes unless they have the correct password. Both can be used at the same time.
  2. Protection which does not prevent anyone from opening the file but does prevent the type of changes listed in the Password Protect dialog depending on the selected option (most frequently used for Forms). Only one option can be selected at a time & use of a password is optional.
