a. user say:

Hi Fredrik. From your description it sounds like you are looking to put all data, including shared project files, in to a single user's OneDrive for Business? If so, whilst this can be done it is not usually a recommended approach.

The ideal approach is that each user has their own OneDrive for Business to store their working project type files they are not ready to release to their colleagues yet. The same kinds of files that you would find in the user's My Documents or home drive.

Users can share out documents from their own OneDrive for Business if they need to.

For team or project type documents, the ideal location is team sites. This makes it easier to manage and maintain when you need multiple team members to collaborate on the files. Or, use the new Office 365 group files feature - this is a quick and easy way to set up document repositories for teams to share files.

The problem with all files in a single OneDrive for Business is if that user leaves the organisation then there is a lot of data at risk tied to that user account.
