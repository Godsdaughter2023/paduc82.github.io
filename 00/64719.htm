exit

Then reboot.

donlekei say: If you have EVER signed in to a windows live account, after the update you will have this problem.

The problem is that the new version of Windows Live Mail tries to use your contacts in your windows live account EVEN if you are not signed in, or if your windows live account is not tied to a hotmail account (eg. if it is tied to a gmail or ISP mail account.

You need to manually go in and disable windows live account "functionality".

2) Select Options, then Mail.

3) Select the Connections tab.

4) Click on "Stop signing in".


Restart Windows Live Mail. Your contacts should be working now. If you messed it up in your attempts to solve this problem you may need to import your contacts from your last CSV backup.

xircal say:

Nicholas,

Please do the following.

  1. Right click the Start button and go to "Command Prompt (Admin)".
  2. Type (or copy/paste) sfc /scannow and hit Enter.

System File Checker will check Windows system files for corruption and attempt to repair any found.

If it can't, you'll see a message to that effect. In that event, use the same command prompt to run the following command which I suggest you copy/paste:

DISM.exe/Online /Cleanup-image /Restorehealth

If neither of these fixes the problem, open another Command Prompt (Admin) and run the following command to check your hard drive:

CHKDSK /f /r

You'll get a message that the tool cannot be run because the disk can't be locked and whether you want to schedule a diags check on the next reboot. Click "Yes" and then either reboot immediately, or finish what you're doing and then restart.

If CHKDSK finds bad sectors, it will mark them as such so that they can't be used. But if the tool appears to hang for more than a couple of hours at a certain percentage, then that would indicate that your hard drive is defective and will need to be replaced.
