teylyn say:
Hello,

a cell can hold either a formula or a value. You can help the user select a valid value with data validation, like a drop-down list. But you can not have a formula in the same cell that then performs a lookup with the selected value and returns the result in the same cell. It's easy to do that in a different cell, though.

You can use VBA to manipulate the cell after a value has been chosen. The VBA routine can perform the lookup and write the looked up value into the cell.

So, either re-model your sheet or engage VBA.

ofer gal say:

10.0.10240 as ISO

Anyone knows where I can download 10.0.10240 as ISO? 

The latest I have is 10.0.10074 and it does not upgrade anymore. 

a. user say:

ISO for 10240 is not available (yet or maybe ever)

Yes, 10074 is upgradable to 10240 if 10074 is still activated.

New clean installs will have an issue activating.

1. Login with your Microsoft System Account (same account you registered with for the Insider program)

2. Change the registry key using windows + R keys (run) and type regedit

Change the entries from fbl_Impressive to TH1

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsSelfHost\Applicability

Ring = WIF

RingBackup = WIS or Enabled (if you have this key)

EnablePreviewBuilds = 2

BranchName = TH1
UserPreferredBranchName = TH1

3. Reboot your pc

4. Login and run windows update and it should download and install TH1 Professional Windows build 10240.

If you changed the Ring key from WIS to WIF - you might need to wait one day before the update works.
