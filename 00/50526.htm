MAD Mouse - Microsoft Community

capn larry say:

MAD Mouse

On connecting a Garmin GPS 76 to a USB port on my computer using a Garmin USB to RS-232 Converter cable, my mouse cursor goes mad - enlarging or reducing programs already in use, opening apps, etc, etc.  If I turn the GPS off, the problem stops - but on turning it back on the problem continues. I am using Microsoft Windows Vista with a Gateway computer.  I have attempted downloading an updated driver to no avail.  I have tried turning on the computer first, or second.  It looks to me as if the computer is sensing the GPS as mouse input.  This did not happen with my previous computer which used Windows XP.  Occasionally I have had it running, but haven't been able to do so with any consistency.

Thanks in advance for any help, suggestions or insight.

a. user say:

Capn Larry,
Thank you for visiting the Microsoft Answers forum.

The problem is caused by Windows seeing the serial information coming through the USB port as coming from a Microsoft serial mouse. To solve the problem:
With the GPS receiver plugged in, navigate to Start>Control Panel>System
When the System Properties window opens click on the Hardware tab and then click on Device Manager. Expand the Mice and other pointing devices heading by clicking on the plus sign.
In addition to your mouse or touchpad, you will also see listed a Microsoft serial mouse or Microsoft serial pointing device.
Right click this entry and then click on the Disable button.
Your cursor should immediately settle down.
Now close all open windows and do a restart with the USB GPS receiver still plugged in.
It should show up on reboot and there should be no more crazy cursor.
Disabling the Microsoft serial mouse device will be permanent until you go back through the above path to re-enable it if you should ever need to plug in a Microsoft serial mouse.

Let us know if this information was helpful or if you still need assistance.
