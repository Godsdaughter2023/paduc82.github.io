robertjwin say:

Drag & drop works differently if you do so between different partitions or the same partition. If you're unsure, always use the right mouse button to drag & drop instead of the left one. With the RMB you'll get a handy menu when you release from which you can select whether you want to copy or move.

However I suggest you forget drag & drop entirely for the moment. Just do what people have always done going back to much, much before XP. Select your files/folders, right-click and cut or copy. Then go to the destination, right-click and paste. Or use the Cut/Copy/Paste/CopyTo/MoveTo buttons in the new ribbon you'll see at the top of File Explorer (what used to be called Windows Explorer in XP).

Seriously, it's simple and none of these basic cut/copy/paste file management techniques have changed much (if at all) in decades.

How to delete all the restore points of Windows 10 at the same time.

If what you want is to delete all the restore points (backup) created in the Windows operating system of your computer, the first thing we will have to do is write the term: Disk Cleanup in the Windows search field. Your laptop probably has a recovery mechanism accessible by F-key or modified F-key (e.g. by Ctrl) depression at boot time. To advise further on this we need details of your laptop (brand and model). You may find this documented from somewhere on the Start menu or in a booklet that came with the laptop.

 

In addition there is probably a facility to create your own recovery CDs/DVDs. Again, look on the Start menu and in any paper documentation. If this facility exists, you should do it asap because this will be your only recovery method in the event of hard disk failure.

 

Failing the inbuilt recovery and/or creation of recovery media, you will have to contact the manufacturer for recovery media.

 

You can download a standalone SP3 installer from...

http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=24

...this is only a service pack installer, not a full XP OS installer. You can ignore the text that says that it is for network installations and also ignore the text advising Windows/Microsoft Update if only updating one PC. You will not be asked for a product key.

 

In fact you will need this SP3 installer to update the OS before Windows Update will be operational. Install it before even attempting Windows Update.

 

A manufacturer's recovery would not normally require entry of the product key from your sticker. But if it does, the key on the sticker is good for the recovery and any updates including SP3.

 

After recovery, you should uninstall whatever AV solution came pre-installed, do the SP3 upgrade without being connected to the internet and then install, for example Microsoft Security Essentials for which a standalone installer can be downloaded here...

http://www.microsoft.com/en-us/download/details.aspx?id=5201

...or here...

http://windows.microsoft.com/en-US/windows/security-essentials-download

 

Then you can connect to the internet and download MSE (or other AV solution) updates and Windows Updates.

 

A standalone IE8 installer can be downloaded here...

http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=43
