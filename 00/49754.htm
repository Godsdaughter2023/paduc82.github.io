xircal say:

It's advisable to change the way you receive updates to prevent the same thing happening again. Here's what to do.

  1. Right click the Action Center icon (looks like a flag) on the taskbar and click "Open Windows Update".
  2. In the menu on the left, click "Change settings".
  3. Change Automatic Updates to "Download updates but let me choose whether to install them"
  4. Click OK to exit.

In future when updates become available, you'll see a message on your login screen. Microsoft releases Windows Updates in the second Tuesday of every month usually although there are times when the odd update will appear out of band.

But don't install KB3035583 because this is the Windows 10 upgrade prompt. Instead, right click it and click "Hide update". This will turn the box you have to checkmark grey and it be moved to the "Restore hidden updates" repository when you exit. It'll remain there just in case you decide at a later date to make the switch anyway.

Having said that though, Microsoft seems to be re-releasing it just lately and it might well appear again in Windows Update in which case just hide it again.

There's also a small free tool to prevent the upgrade which you can download called GWX Control Panel: http://blog.ultimateoutsider.com/2015/08/using-gwx-stopper-to-permanently-remove.html

It's a standalone application which means you don't have to install it and it will run from wherever you choose to keep it.
