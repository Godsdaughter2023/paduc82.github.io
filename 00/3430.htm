stefan blom say:
If you are editing in those two places, you can use Shift+F5 which moves between the most recent edit points.

pamcaswell say: You can split the screen and show both places in the document at once. Click View tab > Window group > Split icon.  Or press Alt+Ctrl+s.  Drag the split bar to where you want it, then click the bar.

ruwim.b say:

The answer you have received from Microsoft Agent JosephV_A has very little to no relevance for your original question.

 

Skype for Web application is silently automatically updated without any interaction from the user. There is no message or prompt provided to the users when this application is updated. The currently latest version is 1.124.27.

 

I'm not sure what you mean by "Microsoft account Skype", but if by this you mean Microsoft Skype for Windows 10 (the 12.X or 14.X version), then this application is also silently updated from Windows Store.

 

With the new Microsoft Skype for Desktop (the 8.X version) you may see a notification to update in the Notifications tab, but it has been now long time ago that the update notification has been shown. In the case of Windows users the Check for Update option may still show that they are on the latest 8.30.0.50 version, while the actually latest version is 8.32.0.53.

 

If you are on a Windows computer, then the latest 8.32.0.53 version can be installed using this link:

 

https://endpoint920510.azureedge.net/s4l/s4l/download/win/Skype-8.32.0.53.exe

The only Skype application that in the past had the option to switch automatic updates on/off was the older Skype 7.X version. In the latest 7.41.0.101 version, this option is now disabled too.
