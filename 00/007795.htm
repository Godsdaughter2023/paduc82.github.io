aw1991 say:

Excel- move data based on conditional formatting

I have a workbook with multiple sheets in it (about 7). One is the Mastersheet where I will be putting in the data (and changing it often). e.j. gun say:

Here is an example of how you can do this.  You will need to duplicate the rules in your conditional formatting in the equations below.  In my example, I used your example criteria above.

First, start with your database.  I made one up:

Criteria Value 1 Value 2 Value 3 Value 4
aaab -81 955 224930 589
aacb 69 725 238716 585
aabb -70 953 104213 519
aaab 54 893 131026 601

aacb -37 418 124214 500
aabb -12 650 193245 556
aaab 8 710 148116 664
aaab -18 120 212953 640
aaab -96 386 161630 734

Now, for each criteria you want to extract, create a new worksheet.  Name the worksheet for that criteria:

Now go to tab "aaab", and enter two formulas as shown below:

In cell A2: 

=MID(CELL("filename",$A$1),FIND("]",CELL("filename",$A$1))+1,256)

In Cell B2:

=IFERROR(INDEX(Database!B$2:B$37,SMALL(IF(Database!$A$2:$A$37=$A$2,ROW(Database!A$2:A$37)-ROW(Database!A$2)+1),ROWS(Database!A$2:Database!A2))),"")

Make sure the formula in B2 is entered using SHIFT-CTRL-ENTER to make it an ARRAY FORMULA.

Drag the formula in B2 to the right all the way to E2.  Then Drag B2:E2 down enough rows to account for the number of entries in your original database.

You should now have an extracted list of occurrences of "aaab".  Repeat for other desired criteria.

Hope this helps,

Eric

a. user say:

Hello Mikaelhagerman:

 

Thank you for posting to the forum and providing the requested details.  I have looked into this error a bit and found that it is most often related to the PC's ability to properly connect with Windows update.  Please take a look at this article, http://windows.microsoft.com/en-us/windows7/Windows-Update-error-80070424.  Please try all the following steps that are outlined in the article:

 

Run a Windows Update troubleshooter

  1. Click this button:

    Fix this problem

  2. In the File Download dialog box, click Run, and then follow the steps in the wizard.

  3. Open Windows Update, and try to install the update again.

If that didn't work, try the following steps.

 

Start required Windows Update services

  1. Open Administrative Tools by clicking the Start button , and then clicking Control Panel. In the search box, type administrative tools, and then click Administrative Tools.

  2. Double-click Services. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

  3. In the Services dialog box, click Background Intelligent Transfer Service (you might need to scroll to find it; the entries are in alphabetical order).

  4. Note the status of the selection (the status appears in the third column). If it shows Stopped, right-click Background Intelligent Transfer Service, and then click Start.

  5. In the Services dialog box, click Windows Update (you might need to scroll to find it; the entries are in alphabetical order).

  6. Note the status of the selection (the status appears in the third column). If it shows Stopped, right-click Windows Update, and then click Start.

  7. In the Services dialog box, click Workstation (you might need to scroll to find it; the entries are in alphabetical order).

  8. Note the status of the selection (the status appears in the third column). If it shows Stopped, right-click Workstation, and then click Start.

  9. Close the Services dialog box, and then run Windows updates again.

     

If the Background Intelligent Transfer Service doesn't appear in the list of Services, see the resolution section of this article about problems with the Background Intelligent Transfer Service on the Microsoft website.

 

If the Windows Update Service doesn't appear in the list of Services, see this article about installing the Windows Update agent on the Microsoft website.

 

After you have done all of the above, you should check your PC for available updates, http://update.microsoft.com/microsoftupdate.  If you are informed that the PC is up to date, be sure to click the link that reads "check online for updates from Microsoft update".  If any updates are found, run and install them.  After installation completes, restart your computer and try to update the phone again.

 

The final thing to do if you continue to get this error is to try the update from another computer.  We suggest you try another computer on a different network.  If progress is made on the other computer, there is likely something on the PC that is preventing the update from installing to your phone.  If you receive the same error then we may be looking at an issue with the phone itself.

 

Please let us know if you have any other questions or concerns.

Try something like this. . . .

Assuming you have unlocked certain cells then protected
the sheet and allowed unlocked cells to be selected

Private Sub CommandButton1_Click()
    Dim ts As Date
    If Selection.Count > 1 Then GoTo stopper
    ActiveSheet.Unprotect
    With Selection
        .Columns.AutoFit
        .Value = Now
        .NumberFormat = "h:mm AM/PM"
    End With
    ActiveSheet.Protect
    Exit Sub
stopper:
    MsgBox "more than one cell selected"
End Sub
