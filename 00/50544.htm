For information only ...

Usually, the number on the holographic sticker on the bottom of a laptop is the 25-character Windows license key (xxxxx-xxxxx-xxxxx-xxxxx).  Do NOT post this type of key in this forum or anywhere else online.

Sometimes, you'll see a reference to a "product ID" or product code.  This usually is a 20-character number with this format: xxxxx-yyy-zzzzzzz-zzzzz.  This identifies the version of Windows.  On a "brand name" computer, the second group of characters is OEM, as in 55274-OEM-zzzzzzz-zzzzzzz.  In this example, the 55274 means a generic OEM version of XP Pro.

Microsoft Office programs use a 32-character product code or Globally Unique ID (GUID) such as

hairyfool say:

Just to clarify. It is not possible to upgrade from 32 bit to 64 bit as far as the accepted meaning of the term is concerned. It requires a clean install of the new operating system removing the existing, there is no option to retain programs and settings.

The only reason other versions can do it for "free" is because the license key is valid for both architectures. Moreover those users that upgraded to Windows 10 had the upgrade based on the qualifying O/S architecture, if they had Windows 7 32 bit they got Windows 10 32 bit and the Digital license issued is specific as to 32/64bit. OEM licenses were technically only valid for the original architecture and were sold specifically as 32 bit OEM packs and 64 bit OEM packs, only retail versions had both media either of which could be used (but only one).

However your particular problem arises over the Enterprise License. These licenses are only issued as multiuser licenses intended for the corporate market and are not to be resold individually. Enterprise licenses also did not qualify for the free Windows 10 upgrade program.

The principle behind this is that in a corporate environment individual users would not upgrade their computer, it would be managed through the organisation that owns the license, most likely a dedicated IT department planned rollout. Similarly a switch from 32 to 64 bit would be centrally managed as they own the license and merely requires them ordering 64 bit media.

So the reason why you do not find Enterprise media being electronically distributed by the same channels is whoever actually owns the Enterprise license has to request it from the service they bought their license from in the 1st place. Obviously the question then arises as to why you have an Enterprise installation and who you need to speak to to get your system upgraded.

andre da costa say:

If you got your Windows 10 Enterprise license through the organization you work for, you will need to contact your system administrator for assistance with obtaining the 64 bit version.

Windows 10 Enterprise is what you call a volume license client, which is normally available through the Volume License Service Center.

If you are able obtain the install media, you can switch from 32 to 64 bit without it affecting your activation status.

With the release of Windows 10 November Update (1511), and later versions such as the Windows 10 Anniversary Update (1607), you can switch directly to a 64 bit version of Windows 10 with or without a product key.

  • Switching to Windows 10 64 bit is free, you don't have to pay for it. As long as you have a qualifying Windows 7 or 8 or Windows 10 license, you are entitled to use it.
  • Switching requires that you perform a new installation, that means files, applications and settings will be deleted.
  • You will have to reinstall applications and native 64 bit drivers.
  • Backup before migrating: How to: Create a system image in Windows 10

Step 1: Download Windows 10 64 bit then create a bootable copy:

How to download official Windows 10 ISO files

  • How to: Perform a custom installation of Windows - Choose this option if you want to recover files from Windows.old
  • How to: Perform a clean install of Windows 10 - Choose this option if you want to erase the drive, files and applications will be deleted.
