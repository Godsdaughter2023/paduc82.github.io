leighf. say:

Perfect, thank you! This seems to work. Using a function to set SearchTermFound wasn't working quite right so at this point it's just this repeated for all 10 search terms but I can clean it up later if it's too slow. I ran it with a practice powerpoint and it worked perfectly.

        Dim sSearchTerm As String
        Dim oSl As Slide
        Dim oSh As Shape
        Dim SearchTermFound As Boolean

        For Each oSl In ActivePresentation.Slides
            oSl.SlideShowTransition.Hidden = False
        Next

        sSearchTerm = "two"
            For Each oSl In ActivePresentation.Slides 
                For Each oSh In oSl.Shapes 
                    If oSh.HasTextFrame Then 
                        If UCase(oSh.TextFrame.TextRange.Text) = UCase(sSearchTerm) Then 
                            SearchTermFound = True
                            Exit For 
                        Else
                            SearchTermFound = False
                        End If
                    End If
                Next 
                            If SearchTermFound = True Then 
                                oSl.SlideShowTransition.Hidden = True 
                            End If
            Next
