9:30:00 AM to 9:30:?

hansv mvp say:

Right-click the merge field and select 'Toggle Field Codes' from the context menu.

You'll see something like

{ MERGEFIELD MyField }

or

{ MERGEFIELD MyField \* MERGEFORMAT }

Make it look like this:

{ MERGEFIELD MyField \@ "h:mm AM/PM" }

Press F9 to hide the field code and to update the result.

Two places to look for a fix..........

 

1)  When you upgraded to Win 8.1/IE11, in the setup there was a Skydrive window that, if you clicked "Next" would automatically backup all of your data to cloud storage.  When that happens, it leaves overlays on the Desktop internet icons telling you that the file is syncd with Skydrive backup storage.  You can remove this by doing a simple registry edit..

 That prevents Windows from finding the Skydrive checkmark overlay files. (you can do the same with whatever file you find there, eg Dropbox, etc.)
