Stolen Laptop, Stolen Software - Why Not Shut It Down? - Microsoft Community

mdmesser say:

Stolen Laptop, Stolen Software - Why Not Shut It Down?

My laptop was stolen from my vehicle, a police report has been filed.  The deductible on my car insurance is $500 and the laptop was $320.  I realize I won't get it back, but why can't Microsoft disable the Windows system, as long as I have a valid police report?  The laptop will eventually log into Windows Update and then it should be rendered useless.  Doesn't this make sense?
shenan stanley say:
My laptop was stolen from my vehicle, a police report has been filed.  The deductible on my car insurance is $500 and the laptop was $320.  I realize I won't get it back, but why can't Microsoft disable the Windows system, as long as I have a valid police report?  The laptop will eventually log into Windows Update and then it should be rendered useless.  Doesn't this make sense?

First - it is unlikely that would affect anyone (thief or innocent victim after the theft) in any manner.

Secondly - there is no way of connecting that installation of Windows to you.  Did the laptop come with the edition/version of Windows it has on it pre-installed?  If so, it is unlikely you were ever using the product key you *may* have recorded off the sticker on the laptop in any case - so reporting the product key to have it disabled would be pretty much worthless (it'll probably never be used in reality.)

In what other way did you think the 'right to use' license you had for Windows was tied to you?  If it came with the computer, it was tied to the computer.  If you have the serial number of the computer and a receipt tying that serial number to you - perhaps you can prove that system and that right to use Windows is yours - but if the Windows came pre-installed, your right-to-use that particular Windows license was tied to that specific computer (OEM licensing, etc.)  You cannot take that right to use that license anywhere else, even if it was stolen.

As already pointed out, the product key you might have (unlikely) recorded off the sticker on the computer was probably not even in use (thanks again to the OEM) - so there is nothing unique identifying that SPECIFIC system to "Windows Update", where you think some sort of identification could happen and disable said system.  If you used some product key finder and recorded the key it found, the one you got is likely the OEM SLP and they are not going to disable thousands of legitmately owned machines for your (really?  a $320 laptop?!  Sounds more like a netbook...) one stolen one.

So - questions for you..

  • Do you have the Windows Product key off the sticker on the machine?
  • The Windows Product key that was in use on the machine?
  • The serial number of the machine?
