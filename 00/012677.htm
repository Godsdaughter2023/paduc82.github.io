Your  " cell j19, in sheet2, in sheet1," is a bit confusing.

If you meant that J19 is on Sheet1, use this:

Sub DeleteColumnWithValue1()
    On Error Resume Next
    Worksheets("Sheet2").Cells.Find(Worksheets("Sheet1").Range("J19").Value).EntireColumn.Delete
    On Error GoTo 0
End Sub

If you meant that J19 is on Sheet2, use this:

Sub DeleteColumnWithValue2()
    On Error Resume Next
    Worksheets("Sheet1").Cells.Find(Worksheets("Sheet2").Range("J19").Value).EntireColumn.Delete
    On Error GoTo 0
End Sub
