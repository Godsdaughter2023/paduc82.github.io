Re-running Hybrid Configuration Wizard - Microsoft Community

a. user say:

Re-running Hybrid Configuration Wizard

I recently gracefully uninstalled Exchange 2013 from a Windows 2012 server that we'd run the Hybrid Configuration Wizard on (the new HCW, very slick) in order to blow away and rebuild the server with 2012 R2.  No major issues involved, but I did have a question:

When re-running the HCW, there is a step where the wizard prompts for which servers you'd like to include in the connector configuration.  Is it necessary to select all the servers in your environment currently serving in a hybrid capacity (we have four), or can you simply select one server and run the wizard?  For example, let's say we decided for whatever reason to add an additional hybrid server to the environment - can we simply just check the option for that one server or should we select all the hybrid servers and allow the HCW to (re)configure all of them?

Appreciate any feedback, thanks!

a. user say:

Sorry, folks, for some reason I wasn't getting the email notifications of updates to the forum.  I ended up checking with Microsoft directly and it turns out you don't need to run the HCW when adding a new server.  You just have to make sure of the following:

1) The new server has the proper certificate imported into it's store

2) The server is added to any outbound connectors to O365

3) For inbound mail flow, make sure you update any hardware load balancers to include the new server name (or if exposing the servers directly, add the server in lieu of a smart host)

4) Make sure the MRS Proxy endpoint is enabled and again, that the server is properly configured in any load balancers (or otherwise has a migration endpoint)

So in short, no need to rerun the HCW; just build the server, and make sure the above steps are completed properly.

This was very specific to my scenario, so apologies if it was a little vague.  Hope this helps anyone else in a similar position.

Thanks, all!

  7. Click "OK" to close the "Internet Options" box

joel in vic say:

I had exactly the same issue on a 920.

 

When I had it search for networks it could "see" them, but it just wouldn't register with them.

 

I finally found some information that hinted to the SIM card reader on the 920, (shared with the N9 and L800) was known for having problems.

 

I bought a replacement part on line, and took it into a cell phone repair shop to have them use their hot air soldering iron to replace the SIM card reader.  It worked!

 

What I did learn is that actually replacing the part might have been more work than was actually necessary.

 

There are 12 small solder connections between the SIM card reader and the mother board, one fore and aft of each "prong" connection.  I suspect all I really needed to have done was to use a fine tip on a good old fashioned soldering iron and I could have re-heated each of the connections in turn.  The attached pic shows the back side of the SIM card reader so you can understand where the connections are.
