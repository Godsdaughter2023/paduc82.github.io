jan karel pieterse say:
Hi Elton,
 
Simple:
 
Sub CloseAllButMe(bSaveChanges As Boolean)
    Dim lCt As Long
    For lCt = Workbooks.Count To 1 Step -1
        If Workbooks(lCt).Name <> ThisWorkbook.Name Then
            Workbooks(lCt).Close bSaveChanges
        End If
    Next
End Sub
 
To close all without save, call this like so:
 
CloseAllButMe False
 
To close and save:
 
CloseAllButMe True

It’s the new Windows Update Agent. You can no longer use the old (wuauclt.exe /detectnow) commands to schedule a Windows Update scan via the command prompt. Windows Update settings has been moved away from the legacy Windows Control Panel to the new modern System Settings app in Windows 10. If you are trying to manage Windows Updates on a Windows 10 or Server 2016 / 2019 computer you have no doubt already figured out that WUAUCLT no longer functions. 

Windows Update uses usoclient.exe, usocoreworker.exe, usoapi.dll, usocoreps.dll, & usosvc.dll to either scan for updates, install or resume updates.  Windows will also call this service when adding new features to Windows.



You won’t find any reference to usoclient.exe or usocoreworker.exe in pretty much any MS documentation.  Microsoft has made no information available on this app and has gone so far as to say:

"This command isn’t meant to be called outside of the internal OS. Nobody outside the OS should be trying to run the usoclient directly"
