aileronroll say:

Hi there! The error code C000-0185 refers to the update failing to load on your XBOX. Here's a couple steps to help fix the problem:

1. Download the update again. Press the guide button and select Settings, then System Settings. Go to Network Settings, Wired or Wireless Network (depending on which you are using), Test XBOX Live Connection, then Yes. A pop-up window will appear that will prompt you to download the update again.

2. If that doesn't help, check your XBOX Live connection. Just follow those steps above to do that.

3. If all is well in your connection, make sure you have enough hard drive space. You need at least 190MB of hard drive space in order for your download to begin. To do this, just click on the link as follows to learn how to move, copy and delete your files: support.xbox.com/.../manage-games-data

4. If you already have enough storage for your XBOX, try to clear the system cache.

5. If you cleared the cache but the error code still comes up, you can try to update it via the CD/DVD drive or a flash drive. Click on the link to learn how to do this: www.xbox.com/system-update-usb

6. If you have already done that, you can try to remove the hard drive from the console. Click on this link to see how to do that: support.xbox.com/.../add-remove-hard-drive

7. If all else fails, do what the Hitchhiker's Guide To The Galaxy suggests in big bold letters: don't panic. You can still contact the fine folks over at XBOX Support by clicking on the link as follows: support.xbox.com/contact-us

Hope this helps!

Startup Repair is a Windows recovery tool that can fix certain system problems that might prevent Windows from starting. Startup Repair scans your computer for the problem and then tries to fix it so your computer can start correctly.

I suggest you to download the Windows 10 ISO and create a Media Creation tool and then perform Automatic

Repair.

Refer to the following link:
http://www.microsoft.com/en-us/software-download/windows10

Once you have the installation media, follow the steps: 

1. Start your computer from the installation media. Insert the disk or USB stick and restart your computer. If you see a message asking you to “Press any key to boot from DVD,” do so. 2. In the Windows Recovery Environment, on the Choose an option screen, click 'Troubleshoot'.
3. Click, Advanced Options and then click on 'System restore' .
4. After this complete the repair and see if you are able to boot in to your computer.

If the System Restore fails follow the same procedure and on the Step 3 choose Automatic repair. 

Note: There will be no data loss by performing Startup repair. It will format c: driver and reinstall Windows 10.

Hope this helps in resolving the issue. If the issue persists, do get back to us. We will be happy to assist you.
