Hi,

Every email you receive in Outlook 2016 has a unique thread ID. When someone replied to the original message, the "reply email" will have the same thread ID as the original email with additional characters at the end. For example, the thread ID of the original message is abcd1236=. When someone replied to the original email, the "reply email" thread ID will be abcd1236=+7856de. In this case, the two emails (the original email and the "reply email") will become a thread. If the thread ID of the "reply email" will become abcd1326=, it will become a separate email and will not form a thread. We need to find out if the original email and the reply email have the same thread ID. To do this, follow the steps below:

  1. Double-click the original email.
  2. A window will pop up. In the window, click File.
  3. Select Properties.
  4. In the Internet headers box, take note of the the thread ID of the email.
  5. Close the window.
  6. Double-click the reply email of the original email.
  7. Do steps 2-6.
  8. Check if the thread ID of the two emails are the same. Please be reminded that the "reply email" will have some additional character at the end but the first set of it's thread ID will be the same as the original email.

Please inform us if the thread ID of the two emails are the same or not.

sdp_2009 say:

The following approach works for my N97 Mini. 

 

If you want to disable your internet connection (to save your battery) WITHOUT having to disable or uninstall each web-driven widget on your home screen (what a pain), do this: On your home screen tap on Options, then select Content to offline mode. You will eventually see the Facebook widget display a cross to declare a loss of connection. The same for any other internet-based widgets you might have set up. The phone's internet connection can be re-enabled the same way. Note: the only app/widget that remains connected to the internet even when you take the abovementioned action is Nokia's own email messaging widget. If you want to disable that too, you'll need to go into its own Settings sub-menu. I've not noticed any battery problem by leaving the email messaging widget on all day.
