pamdemonium---01 say: If your phone asks for a restriction code, it is NOT a phone problem.

Network operators sometimes offer phones free or at a discount if you sign a contract with them. The phone they give you is tied to them so that you cannot use other sim cards with the phone.

The SIM card they give with the phone when the phone was first bought is the ONLY SIM card that can be used with the phone until the network (not Nokia) gives you the code to unlock it.

Did you buy the phone second-hand or used from an unauthorised seller? If so, they sold you a locked phone that someone else bought on a contract. You need to go back to the seller and ask for a refund or get them to fix it for you.

If you bought the phone brand new, then check the box for a SIM card that it might have come with. If not, go back to your seller.

There is no general code for restriction codes. The only people who can give you a restriction code, like grschinon and petrib said, is the network that the phone is ORIGINALLY tied to.

I hope this explains the situation for you.

    Trendline for only part of graph data range - Microsoft Community

    chrtod say:

    Trendline for only part of graph data range

    Windows 7 home premium and Excel 2010 prof. I have a graph showing daily stock prices for several years. I know how to insert a trendline covering all the data, and extrapolating. But is it possible to insert a trendline covering only a specified period (e.g. the last 3 months). I hoped maybe such an option would be available in excel 2010, but if it is I can't find it. Of course I know this could be done by adjusting the data range of the graph so that it only covers the dates I am interested in.

     

    bernie deitrick, excel mvp 2000-2010 say:

    You could:

    1) insert a new series, and calculate only the trend values for the X-values (time period) of interest - if, for example you wanted the best fit straight line, you could use an array formula like this, entered using Ctrl-Shift-Enter

     

    =SLOPE(IF($A$3:$A$20>DATE(2012,1,10),$B$3:$B$20),IF($A$3:$A$20>DATE(2012,1,10),$A$3:$A$20))*A3+INTERCEPT(IF($A$3:$A$20>DATE(2012,1,10),$B$3:$B$20),IF($A$3:$A$20>DATE(2012,1,10),$A$3:$A$20))

     

    2) Insert a new series, and only populate it with the points of interest, and base the trend-line on the new series - you could also then just hide the new series so it doesn't clutter things up.
