Original Title: "Book formatting issues"

suzanne s. barnhill say:

We'll need more detail about what sorts of changes you're making and what results you're experiencing, but a few general comments:

1. If you're applying formatting to a single paragraph that is propagated to other paragraphs, see http://word.mvps.org/FAQs/Formatting/WholeDocumentReformatted.htm.

2. If you are making changes to document/section properties (such as page size, margins, number of columns, header/footer), then those changes affect the entire document or section.

3. If you make changes in a header or footer that is linked to the following section(s), the changes will affect all the headers and footers.

4. Some page layout settings made in a single section affect all of them, even if you select "This section" in the Page Setup dialog. Notably, the "Different Odd and Even" setting for headers and footers applies to the entire document; "Different First Page" can be applied to selected sections.

5. Word is not page layout software; it is word processing software. Text reflows from page to page, so if you add or remove text on one page, this will affect the pagination of text on following pages.

Date-Time Difference - Microsoft Community

pauldnb say:

Date-Time Difference

I am trying to calculate the difference in hh:mm:ss between 2 dates/times.

I followed the example in Excel Help...using the following formula = ((End_Date+End_Time)-(Start_Date+Start_Time))*24

But I get #NAME? as a result.

I set the date format to 9/25 for the date fields and the time format to 37:30:55 for the time fields since the time difference could be in excess of 24 hours.

I'd appreciate some guidance on why the formula is not returning a value.

Thanks.

hansv mvp say:

If you want the difference in hh:mm:ss, do not multiply with 24 - you only need to do that if you want to calculate the difference as decimal hours, e.g. 8.3 hours.

For End_Date, substitute the cell that contains the end date, etc.

So if the start date is in A2, the start time in B2, the end date in C2 and the end time in D2, the formula would be

=(C2+D2)-(A2+B2)

Format the cell with this formula as 37:30:55 (or the custom number format [h]:mm:ss)
