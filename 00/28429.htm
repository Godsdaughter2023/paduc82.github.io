Hi

I have two Access tables (A & B) with 6  fields as listed below:

Company Name      Line Number     Column Number    Description    Amount      Year

I would like to append table A to table B and at the same time, eliminate records where data in the 6 fields are identical.

Thank you for taking the time to read and answer this question.

Al

ken sheridan say: You can avoid inserting rows which already exist in B by extending the SQL statement as below:

INSERT INTO B([Company Name],[Line Number],[Column Number],Description,Amount,Year)
SELECT DISTINCT [Company Name],[Line Number],[Column Number],Description,Amount,Year
FROM A
WHERE NOT EXISTS
     (SELECT *
      FROM B
      WHERE B.[Company Name] = A.[Company Name]
      AND B.[Line Number] = A.[Line Number]
      AND B.[Column Number] = A.[Column Number]
      AND B.Description = A.Description
      AND B.Amount = A.Amount
      AND B.Year  = A.Year);
