You cannot grab text from a document without opening it in order to read it.

If you want to import a range of five paragraphs after a particular bookmark ("myBookmark") in a document ("C:\Path\SourceDocName.docx") into the document that you are working on you would need something like


Dim oSource As Document

graham mayor say:
You cannot obtain a selection of text from a document with vba without opening the document.

Insert text essentially opens the document also.

As I said earlier, you can still use an includetext field in your target document to insert a bookmarked region of a document, but it still opens and closes the document transparently whenever that field is updated. The first example in my last post shows how to set the range to the bookmark and the five following paragraphs

'Set the range to the scope of the bookmark
Set orng = oSource.Bookmarks("myBookmark").Range
'Then move the end of the range five paragraphs
orng.MoveEnd wdParagraph, 5
'orng contains the required paragraphs

If you want to select from the start of one bookmark to the start of another then that takes us back to my original suggestion - in this case

'Set the range to the scope of the first bookmark
Set orng = oSource.Bookmarks("StartBM").Range
'Set the end of the range to the start of the second bookmark.
orng.End = oSource.Bookmarks("EndBM").Range.Start
'orng contains the content from the start of the first bookmark to the start of the second

You can now insert the range into the target document as shown in the previous post.

Change the bookmark names to the names in your source document.
