Hi,

Windows 10 introduced Tablet Mode, which is designed to make the desktop easier and more intuitive for touch devices.  This mode may hide some desktop icons.   If you are unexpectedly missing some of your icons, you may be in Tablet Mode.

You can turn off Tablet Mode through the Action Center, or through Settings.

To turn off Tablet Mode using the Action Center:

  1. Tap or click the Action Center icon in the notification area.

  2. Tap or click the Tablet mode tile to toggle it on or off.  (Tablet mode is on when the tile is blue, and off when the tile is grey.)

To turn off Table Mode within Settings:

  1. Open Settings, and select System.

  2. Select Tablet mode.  Toggle the slider off under Make Windows more touch-friendly when using your device as a tablet.

If the missing icons are not related to Tablet Mode, you can pin missing program shortcuts directly to the desktop by performing the following steps:

  1. Search for the application you want to add to the desktop.

  2. Right-click the application and select Open file location.

From the file location that opens within File Explorer, right-click the selected application’s shortcut file and select Send To and then Desktop.

a. user say:

Hi Onyancha,

 

Refer to the link and follow the troubleshooting methods and check,

 

Method 1: Make sure that the date and time are correct on the computer.

 

a)      Check the setting for Date/Time by choosing the Desktop tile and then left click on the date/time in the corner.

b)      Click on change date and time settings. And then verify the time zone is correct as well as the internet time tab has the option to Synchronize with Internet time server.

 

Method 2:

 

a)      Press Windows Key + Q on the keyboard.

b)      Type Command Prompt.

c)       Right click on the Command Prompt from the left window and click on Run as Administrator.

d)      At the command prompt, copy and paste (or type) the following command and then press ENTER:

DISM /Online /Cleanup-Image /RestoreHealth

e)      Reboot the computer.

 

Method 3: Reset Windows Store cache.

 

a)      Press the ‘Windows + R’ key on the keyboard.

b)      Type "Wsreset.exe" and press Enter.

Note: When you reset Windows store, all the apps in the windows store will get refreshed.

 

Method 4: Rename the Windows Update components folder.

 

a)      Press "Windows key + X" and click "Command Prompt (Admin)".

b)      Type "net stop wuauserv" and press Enter.

c)       Type “rename c:\windows\SoftwareDistribution softwaredistribution.old" and press Enter.

d)      Type "net start wuauserv"and press Enter.

 

There is no need to install the .NET Framework 3.0 on Windows 8 Operating System. The .NET Framework 4.5 is included with Windows 8, offers a full installation package which includes the full runtime installation packages for .NET Framework 4.0 as well as .NET Framework 3.5, 3.0 and 2.0.

 

Method 5: Disable third-party Security Software,

 

a)      On your Computer screen Task Bar next to System Clock.

b)      You will see the third-party Security Software Icon.

c)       Click on the third-party Security Software Icon, you will see a pop up.

d)      In that pop and select the option to disable, click on it.

 

Now the third-party Security Software will be disabled.

 

Important Note: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you have to temporarily disable it to install other software, you should re-enable it as soon as you're done. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

I have found a way to solve the error problem being cited by Windows Store whenever one tries to download and install apps. Guys online cited one to download a registry repair program. I have to caution anyone that the program you opt to download may have some implications with the default programs in Windows 8.
