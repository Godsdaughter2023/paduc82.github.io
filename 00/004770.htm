daniel g. grau say:
By static text I meant unchangeable in content, i.e., whatever appears in the document that the user filling-in the information cannot directly alter. In my screenshots, it would be the bold titles and the red-framed text (there are also two framed text fields).

As seen above, if a text field has several lines, it could push down whatever follows it, which is where the "maximum length" specification comes handy (test by typing all lower-case Ms). For formatting purposes, try to avoid multiple-line fields. Otherwise, just leaving ample space at the end of a page  should suffice (you should add Page Breaks to fix the start of the next page).

Alternatively, instructions after a multi-line text field can be placed within a floating text box placed relative to the page (or margins or column), so such instructions won't move down if the paragraph it's attached to does move down on the page.

Your document would look like this:
  • instructions
  • text field(s)
  • instructions
  • text field(s)
  • … etc.
  • white space
  • Page Break
  • instructions
  • text field(s)
  • … etc.
It's recommended to enable nonprinting shades to visually mark your fields: Word > Preferences > View > Field Shading > Always. This is a local preference, so it does not necessarily appear on the final user's screen.

To transfer the Dictionaries

1. The first three steps apply to the computer you are moving from—the first computer. In the Windows Explorer window, go to the following folder: [%userprofile%\Application Data\Microsoft],and open the Proof folder.

2. Copy your custom dictionaries to your transfer location. If you've saved other custom dictionaries in different folders, copy them too. If you're not sure where they are, open Word, choose Tools --> Options 3

3. To find the custom dictionaries, open Word, choose Tools --> Options, click the Spelling & Grammar tab.

4. Click the Custom Dictionaries button. Select each dictionary file in the Custom Dictionaries dialog box, and read the "Full path".

5.Go to the second computer, the computer you are moving dictionary files to. To add your custom dictionary, first move the custom dictionary files to the folder you copied it from (i.e., somewhere inside the Proof folder). Then, on that same computer, choose Tools --> Options, and click the Spelling & Grammar tab. Click the Custom Dictionaries button, and then click the Add button. edit

Tips: The [%user profile%\Application Data\Microsoft] directory is where other Office-related settings can be found.

Specify the names of the customization or settings if you are searching for any others.
