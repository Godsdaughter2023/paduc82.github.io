blair armstrong say:

How to extract a unique distinct list from multiple worksheets

Hi,

I am fairly good at Excel but I seem to have come across a problem that I cannot solve.  I have a workbook with 5 identical worksheets (i.e. Formatting, formulas, etc. are all the same).

In column D of each of the 5 sheets the worker using the document will enter a client's identification number.  This number may or may not be repeated on any of the other sheets.

I was able to extract a unique distinct list from the column of the sheet using the following formula:

{=IF(ISNA(INDEX(D3:D101,MATCH(0,COUNTIF($A$2:A5,D3:D101),0))),"",INDEX(D3:D101,MATCH(0,COUNTIF($A$2:A5,D3:D101),0)))}

The above formula is working fine for each of the 5 worksheets, but I need to develope a unique distinct list from each of the 5 worksheets onto a sixth worksheet (essentially I need a tally sheet of each ID number from the 5 sheets).

I am hoping someone can help me with this, I have tried several different variations of this formula with no luck.

Thanks in advance,
Blair
jllatham say:

Here is a macro that should do the job for you.  You will need to make sure that some of the "pointers" to column IDs and starting row numbers are correct [I think they are] and you will definitely have to change the names of the worksheets involved.  I've made those lines of code bold to point them out to you.

 

Here is a web page that shows how to add code to a regular code module:

http://www.contextures.com/xlvba01.html#Regular

 

Here is the actual code.  After you run it, you can always manually sort the list to get the entries in order which would also offer a quick visual check to make sure they aren't in the list more than once.

 

Sub DevelopUniqueList()
  'change these Const values to match data sheet setups
  Const srcIDCol = "D" ' column on 5 sheets with data we need
  Const srcFirstRow = 2 ' first row with number to deal with
  'this should point to column to put list into on 6th sheet
  Const destListCol = "A"
 
  Dim destWS As Worksheet
  Dim dataSheets(1 To 5) As String
 
  '********************************************************
  'enter the names of the 5 sheets to gather data from here
  dataSheets(1) = "Sheet1"
  dataSheets(2) = "DataSheet2"
  dataSheets(3) = "ThirdDataSheet"
  dataSheets(4) = "MoBettaDataHere"
  dataSheets(5) = "Last Data Sheet"
 
  'point this to the sheet to put the unique list onto
  Set destWS = ThisWorkbook.Worksheets("Sheet6")
  '********************************************************
  'end of user changable definitions
  Dim searchRange As Range
  Set searchRange = destWS.Range(destListCol & ":" & destListCol)
  Dim srcWS As Worksheet
  Dim srcList As Range
  Dim anySrcCell As Range
  Dim SLC As Integer ' sheet loop counter
  Dim lastRow As Long
  Dim findWhat As String
  Dim foundMatchCell As Range
 
  For SLC = LBound(dataSheets) To UBound(dataSheets)
    Set srcWS = ThisWorkbook.Worksheets(dataSheets(SLC))
    'get last used row in the column on current sheet
    On Error Resume Next
    lastRow = srcWS.Columns(srcIDCol & ":" & srcIDCol).Find(what:="*", _
     After:=srcWS.Range(srcIDCol & 1), _
     SearchOrder:=xlByRows, _
     SearchDirection:=xlPrevious).Row
    If Err <> 0 Then
      lastRow = srcFirstRow ' an unexpected error indication
      Err.Clear
    End If
    If lastRow < srcFirstRow Then
      lastRow = srcFirstRow
    End If
    On Error GoTo 0 ' reset error trapping
    'set reference to entries
    Set srcList = srcWS.Range(srcIDCol & srcFirstRow & ":" _
     & srcIDCol & lastRow)
    For Each anySrcCell In srcList
      findWhat = CStr(anySrcCell)
      Set foundMatchCell = searchRange.Find(what:=findWhat, _
       LookIn:=xlValues, lookat:=xlWhole, MatchCase:=True)
      If foundMatchCell Is Nothing Then
        'a new entry, add it to the list on the 6th sheet
        On Error Resume Next
        lastRow = searchRange.Find(what:="*", _
         After:=destWS.Range(destListCol & 1), _
         SearchOrder:=xlByRows, _
         SearchDirection:=xlPrevious).Row + 1 ' point to empty cell
        If Err <> 0 Then
          lastRow = 2
          Err.Clear
        End If
        On Error GoTo 0
        destWS.Range(destListCol & lastRow) = anySrcCell
      End If
    Next ' anySrcCell loop
  Next ' end loop through 5 worksheets
  'do some housekeeping cleanup
  Set srcList = Nothing
  Set srcWS = Nothing
  Set searchRange = Nothing
  Set destWS = Nothing
  MsgBox "Unique List Created"
End Sub
