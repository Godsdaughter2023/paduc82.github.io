  5. Start File Explorer, navigate to C:\Windows and scroll down to the Temp folder.
  6. Right-click the Temp folder and click Properties.
  7. Go to the Security tab and click [Advanced].
  8. At the top, you should see an owner listing, along with a Change link next to it.  Click Change.
  9. Type in "Administrators" and click [OK].
  10. Back in the Advanced Security Settings dialog, you should now see an option right under the ownership information named Replace owner on subcontainers and objects.  Tick this option on.

  12. In the dialog that opens, click Select a principal.
  13. Type your local login name (or your Microsoft login email address) and click [OK].
  14. Tick Full Control on and click [OK] again.
  15. Back in the Advanced Security Settings dialog, tick Replace all child object permission entries with inheritable permission entries from this object on, and click [OK].

  17. If asked, confirm that you wish to change the permissions on a system folder (click [Yes] again).

1703 has made it possible to force full screen in all games, should you wish, via your graphics processors
