What about Licensing?

You do not tell us what type of License you have.

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.

This was the original link. Microsoft is altering/deleting links, but the information that was originally there remains the same (I copied it before the link disappeared)

Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

A. Generally, an end user can upgrade or replace all of the hardware components on a computer‚Äîexcept the motherboard‚Äîand still retain the licence for the original Microsoft OEM operating system software. If the motherboard is upgraded or replaced for reasons other than a defect, then a new computer has been created. If the motherboard is replaced because it is defective, you do not need to acquire a new operating system license for the PC as long as the replacement motherboard is the same make/model or the same manufacturer's replacement/equivalent, as defined by the manufacturer's warranty.

lemp say:

I gave you two methods to do the migration to new hardware.  One free (SevenForums method) and one for which you would likely have to buy software (Acronis).  Each of these methods will let you move your current installation to new hardware with the OS (Windows) at its current update state (i.e., with all updates and service releases intact) and with all of your programs installed and configured as you currently have them.

In addition to those methods, here's a third one:  as part of its effort to persuade people to move to Windows 10, Microsoft partnered with Laplink to offer -- as a replacement for Windows Easy Transfer -- Laplink's PC Mover Express.  https://www.microsoft.com/en-us/windows/transfer-your-data

Like Windows Easy Transfer, PC Mover Express only moves data files and some settings; it doesn't move installed programs.  However, Laplink also has a commercial program, PC Mover Professional, that does move installed programs (or at least it claims to do so -- I don't have personal experience). 

Microsoft appears to have arranged discounts for both PC Mover Express and PC Mover Professional:  http://pcmover-10.laplink.com/

A clean install is likely to give you a smoother, faster, running system when you're done, but the trade-off is the need to update Windows and re-install all of your application programs.

Personally, I'd go with Acronis.  After you do the migration, you can continue to use Acronis to do regular backups.
