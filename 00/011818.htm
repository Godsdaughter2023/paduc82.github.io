iliru say:

Hi,

The DATEDIF function does not directly calculate the number of weeks that are included between two dates. The syntax of this function is this:

DATEDIF(start_date,end_date,unit)

(for more information, visit this Web site: https://support.office.com/en-us/article/DATEDIF-function-25dba1a4-2812-480b-84dd-8b32a451b35c)

If you want to calculate the number of weeks between two dates, you can use this formula:

=INT(DATEDIF(start_date,end_date,"d")/7)

for example:

=INT(DATEDIF(A1,B1,"d")/7)

where in cell A1 is given the start date and in cell B1 the end date. Both of these cells will need to be formatted as the date, while the cell where the result will be obtained will need to be formatted as General.
