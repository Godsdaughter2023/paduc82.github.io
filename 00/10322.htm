1. Move the scrap file to your computer's desktop, if it is not already stored there.

What is the boot order?
When the computer starts, the system BIOS begins to check a pre-determined list of drives and devices that may contain an operating system such as Windows. The computer can boot an operating system from a hard drive, floppy drive, CD or DVD optical drive, USB storage device, or a network. The order in which the devices are checked is configurable through the BIOS setup menu.
What is the default boot order?
The default boot order settings for the computer are configured in the factory. The default boot order for HP notebook PCs is listed below.
  1. Floppy disk drive
  2. Optical drive (DVD, CD-ROM)
  3. Hard drive
  4. USB device
  5. Network adapter
An operating system not found error message may appear if a floppy disk or an optical disk is inserted into a computer that does not contain an operating system. If this error occurs, remove the disk and restart the computer.
If you have installed a second hard drive and the computer does not boot, please select the hard drive which contains the operating system as first boot device in the hard drive group.
Configuring the boot order
The boot order can be configured on the Advanced tab in the BIOS settings menu. The steps for modifying the boot order may vary depending on the model of the computer.
BIOS configurations vary depending on the computer. For further information on a specific computer, refer to the documentation provided with the computer or go to the HP Drivers and Downloads page. The BIOS settings are generally listed in the Maintenance and Service Guide.
Follow the steps below to configure the boot order on most computers.
  1. Turn on or restart the computer.
  2. While the display is blank, press the f10 key to enter the BIOS settings menu.

  5. Follow the on-screen instructions to change the boot order.

To reset the boot order to the default settings, follow the steps below.

  2. While the display is blank, press the f10 key to enter the BIOS settings menu.

doug robbins - mvp office apps & services (word) say:

In the mail merge main document, use an If...then...Else field with symbols for the checked and unchecked boxes that you insert from the Insert Symbol dialog.

To do that, insert the mergefield into the document, and then press Alt+F9 to toggle on the display of field codes and you will see the

«mergefieldname»

replaced by

{ MERGEFIELD "mergefieldname" }

Select that and then press Ctrl+F9 so that you have

{ { MERGEFIELD "mergefieldname" } }

and then modify it to

{ IF { MERGEFIELD "mergefieldname" } = "True"  "symbol for checked box" "symbol for unchecked box" }

Use Alt+F9 again to toggle off the display of the field codes.  Note, you may or may not need to the quotes around the True.  Try it both ways.  You might also try using -1 in place of it.

Also a better way might be to do the data manipulation in an Access query.
