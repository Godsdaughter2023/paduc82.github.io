Set oRangeA1 = Nothing

strRangeA1 = "$A$2:$Z$43"

john hon say:

How to make _italics_ work with copy and paste

Bascially, _italics_ works when i type it in normally, however when i copy and paste the same thing from another application, word doesnt recognise it. I need word to recognise _italics_ when i paste it and turn it into italics however i dont know how?
suzanne s. barnhill say:

When you paste, AutoFormat As You Type (which is what is converting _italics_ to italics) does not act. But you can accomplish the same thing with AutoFormat, though it requires some initial setup:

  1. At File | Options | Proofing | AutoCorrect Options select the AutoFormat tab (not the AutoFormat As You Type tab) and make sure that "*Bold* and _italic_ with real formatting" is checked. Clear the check boxes for any other formatting you don't want.
  2. Click OK to close the AutoCorrect dialog and then select Quick Access Toolbar in the Word Options dialog.
  3. Set Choose commands from: to Commands Not in the Ribbon.
  4. Scroll down to AutoFormat Now, select it, and click Add>>. Click OK to close the dialog.

Now, when you paste in text that has the _italic_ formatting, you can select it and click the AutoFormat Now button to effect the formatting. You can also use the command on an entire document provided you have disabled the features you don't want. I find that if I disable the entire top section (Apply) and leave most or all of the Replace features enabled, this is a very helpful command.

One really useful thing it will do is remove empty paragraphs from text pasted from the Web (or elsewhere). This is a huge timesaver for me because once a week I paste a list of names, each in a separate paragraph, with empty paragraphs between, and AutoFormat cleans up the list with one click.
