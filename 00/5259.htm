suzanne s. barnhill say:

If you use one of the "Automatic Table of Contents" selections in the TOC gallery, I suspect you're pretty much stuck with what you get. I find these very tricky to edit. You'll probably find it more satisfactory to "roll your own" by using the Insert Table of Contents... command on the Insert | Table of Contents menu. This gives you the classic TOC dialog where you can make your own choices, including setting options for what styles/levels to include and modifying the formatting of the TOC styles.

In the TOC itself, you can change the formatting (paragraph, tabs, font) of a single TOC entry at a given level, and the change will be propagated to the rest of the entries at that level because TOC styles (by default) are set to update automatically.

For help with aligning page numbers and setting tabs/indents, see http://sbarnhill.mvps.org/WordFAQs/NumberAlignment2007.htm. For help on omitting page numbers and tips on formatting TOCs, see http://sbarnhill.mvps.org/WordFAQs/TOCTips.htm.

In some cases, what you want to accomplish with a TOC can't be done through the TOC dialog or the TOC itself and requires actually editing the TOC field. There is more about this in http://word.mvps.org/FAQs/Formatting/TOCSwitches.htm and in the aforementioned http://sbarnhill.mvps.org/WordFAQs/TOCTips.htm.

daniel g. grau say:
There is a "Computer Scripts" folder here:

  • Macintosh HD/Library/Scripts

Each folder in there corresponds to a category in the "Scroll" icon (next to the clock) that should be showing in your menu bar. To access that folder, you may need to use the Finder's Go > Go to folder menu item and paste the path /Library/Scripts.

There is also a "User Scripts" folder:

  • /Users/Your_Name/Library/Scripts

The above contains an /Applications/Word folder that used to be available in Word's "Scroll" menu (additional to the other "scroll"), but Word now looks for scripts in this folder:
  • /Users/Your_Name/Library/Application Support/Microsoft/Office/Word Script Menu Items

AppleScripts in the above folder can be "loose" or classified in folders (these appear as categories).

There is this other location, which is probably from an older version of Word (can't recall), but you may find some scripts there:

  • /Users/Your_Name/Documents/Microsoft User Data/Word Script Menu Items
----------------------------------------------------

You can write a one-statement Workflow with Automator, using the "Rename Finder Items" command.
