    hangc say:

    Auto populate text from one slide to another

    We have a template that is being used by people in the office but one of the main complaint was having to type the same information in on different slides. Is there a way on the master slide to create a link between two slides to have the information carried over?

    john korchok say:

    The only objects that act even close to that are the footer and date fields (Insert>Header& Footer). To use them, you would place the footer or date on the slide layout in the position where you need the information to appear. Then in the Insert>Header & Footer dialog, you would edit the Date and/or Footer to have the information you want to appear, then click on Apply to All.

    To make the Date field display information other than the date, choose the Fixed option in Header & Footer, then type in whatever text you need instead of a date.

    That's as close as you can get. But I imagine you're looking for something like Word's Content Controls, which are not found in PowerPoint. Others have had the same idea, here is a suggestion to that effect in the PowerPoint User Voice forum. You can add your vote. Microsoft uses these votes to prioritize new programming: References to Fields/Values (like Word Cross References)

    steve rindsberg say:

    John's summed up the main answers, but a couple other sorta "outlier" ideas that might help in some cases:

    1) Type the text into a cell in a SAVED Excel worksheet, format it to taste, then copy it and Paste Special/Link it into PowerPoint on any slides where you want the text to appear.  When someone updates the text in Excel, PPT will update it on your slides then next time the presentation's opened.

    2) It might suffice to create a new layout with the repeated text on it.  Any slides that you apply the layout to will inherit the text.  Update it in the layout and it'll update on any slides based on the layout.

    If you need to have lots of different bits of text updated this way, let us know. There may be other options.

    santanu suna nayak say:

    Open the Start Menu, click on Control Panel and then open the System and Maintenance section.
    Double-click on "System" and then click "Remote Settings" to open the Remote Desktop Settings window.

    Click "Don't allow connections to this computer" to prevent anyone from accessing your computer.
