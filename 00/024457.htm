po2 smith say:

How do you do an IF formula with more than 1 criteria

How do you do a formula such as =IF(ISERROR(SEARCH("one criteria",$C2)),"No","Yes") with more than one search criteria, such as looking for dogs, cats, mice.  I am searching 8500 lines of data and want to do something to making my search combinations easier.
t. valko say:
How do you do a formula such as =IF(ISERROR(SEARCH("one criteria",$C2)),"No","Yes") with more than one search criteria, such as looking for dogs, cats, mice.  I am searching 8500 lines of data and want to do something to making my search combinations easier.


To answer your specific question, try it like this:

=IF(OR(ISNUMBER(SEARCH({"dogs","cats","mice"},$C2))),"Yes","No")

However, you might like this version better:

=IF(SUM(COUNTIF(C2,{"dogs","cats","mice"})),"Yes","No")

toddallcock say:

The reason Jack is asking you if this is the primary account (I suspect) is because the best troubleshooting option now is to simply delete the account from the phone, then recreate it. This will force the phone to sync with the Outlook account's current state because that will be the only source of data available.

 

Of course, if this the phone's primary account, you can't delete it without a hard reset (complete wipe of all phone data) which is probably a little drastic in this instance. If this is the primary Microsoft account, I'd probably try to force a clean sync by going into Settings > email & accounts, select that account, uncheck "email" under "items to sync", save that (which should force a sync and delete the emails from the phone, but not the server), then after it finishes syncing, reselect "email", save it, forcing a resync, and see if only the correct email returns.

 

If that doesn't work, try creating a temporary folder on the web to move the (web) inbox contents into, (so now the web inbox is empty), sync the phone (which should remove the matching emails from the phone's inbox), then manually delete everything else from the (phone) inbox that's still there, sync both while empty (hopefully "resetting" the sync between them), then move the emails back from the temp folder to the web inbox, sync the phone again and see if the emails return to the phone.
