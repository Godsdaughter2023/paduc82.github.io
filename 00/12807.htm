Hi,

Welcome to Microsoft forums.

I understand that you are facing issues with pairing the Bluetooth device with the computer. We will help you to resolve the issue.

1. Do you get any error message when you are using Bluetooth?

2. What is the make and model of the computer?

3. Have you made any recent hardware or software changes to the computer prior to the issue?

First of all, I suggest you to start the Bluetooth service and set it to Automatic:

a. Go to the Start screen.

b. Type “Services.msc” without quotes and press Enter.

c. Right click on Bluetooth support service and click on Properties.

d. Set the Startup type as Automatic and then click Start under Service status.

e. Click on OK.

If the issue persists, I suggest you to try the following steps and check if it helps.

Method 1: Run the Hardware and Devices troubleshooter

 

The Hardware Troubleshooter is an automated tool which checks the hardware on the computer for any known issues and provides the details on how to fix them. Follow these steps to run the troubleshooter.

a) Press “Windows Logo” + “W” keys from the keyboard.

b) Type “Troubleshooting” in the search bar and press “Enter”.

c) In the “Troubleshooting” window, click on “View All” on the left pane.

d) Click on “Hardware and Devices”.

e) Click on “Advanced” and then click on “Run as Administrator”.

f) Click “Next” and follow the on-screen instructions to complete the troubleshooting process.

Method 2:

If the issue still persists, I suggest you to uninstall the Bluetooth driver and install the latest driver on the computer.

 

Step 1:

Uninstall the current version of the Bluetooth driver on the laptop:

 

a) Press Windows and x together on the desktop screen.
b) Select Device Manager.
c) Locate Bluetooth devices and expand the driver icon.
d) Right Click on the driver icon and click on Properties.
e) Click on Uninstall and follow the onscreen instructions to uninstall the driver.

 

Step 2:

 

Logon to the computer manufacturer website and install the latest driver for the device.

Hope this helps.

Please post back if the issue persists and we will be glad to assist you further.
