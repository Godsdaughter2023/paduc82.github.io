Change Print Settings to Entire Workbook - Microsoft Community

jane_hancock say:

Change Print Settings to Entire Workbook

I need an event code that changes the Print Settings option to "Print Entire Workbook" when one particular file is opened and then reset it to my default of Print Active Sheet when the workbook closes.

 

I don't want the workbook to go directly to the printer ... I want my user to be able to review the doc view Print Preview before advancing the print.

 

Any ideas?

andreas killer say:
No problem, we can select all visible sheets first.

Andreas.

Option Explicit

Dim WithEvents App As Application

Private Sub Workbook_Open()
  Set App = Application
End Sub

Private Sub App_WorkbookBeforePrint(ByVal Wb As Workbook, Cancel As Boolean)
  Dim S As Object, SaveSelectedSheets As Object
  Dim SheetNames() As Variant
  Dim Count As Integer
  'If not our workbook...
  If Wb.Name <> ThisWorkbook.Name Then
    'Cancel the user print
    Cancel = True
    Application.EnableEvents = False
    'Count all visible sheets
    For Each S In Sheets
      If S.Visible Then Count = Count + 1
    Next
    'Retrieve all names from the visible sheets
    ReDim SheetNames(1 To Count)
    Count = 0
    For Each S In Sheets
      If S.Visible Then
        Count = Count + 1
        SheetNames(Count) = S.Name
      End If
    Next
    'Save current selected sheets
    Set SaveSelectedSheets = ActiveWindow.SelectedSheets
    'Select all visible sheets
    Sheets(SheetNames).Select
    'Print all visible sheets
    ActiveWindow.SelectedSheets.PrintPreview
    'Restore selected sheets
    SaveSelectedSheets.Select
    Application.EnableEvents = True
  End If
End Sub

julie fka profjulie say:

I think the problem you describe is more to do with using a wifi connection exclusively on a phone for push email than it is for how outlook.com pushes email to the phone. 

The nature of a wifi connection as it relates to a windows phone (or probably any other connected device) is that as long as the device is alive (not in sleep mode) the wifi connection remains active, but when the device goes to sleep or standby mode the wifi connection becomes inactive (or goes to sleep) and so any notifications that might be pushed to a device would be delayed as long as the wifi connection is inactive.  Since an active wifi connection can be a power drain on a mobile device, most mobile devices suspend the wifi connection when the device goes into standby or sleep mode. A data connection is continuously connected to a phone (or at least as long as the phone is "in network range" of the cellular system's network), so the phone doesn't experience the same kind of notification delay it might encounter while only connected via wifi connection.

Windows Phone 8 has an option that you can set that will keep the wifi on even after the screen on the device goes dark.  This setting is turned off (by default) since turning it on can potentially drain the phone's battery.  You can change the option on your phone by going to Settings > wi-fi > tap the "advanced" button > place a checkmark in the box next to "Keep Wi-Fi on when the screen times out."
