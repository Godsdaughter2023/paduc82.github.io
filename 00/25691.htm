    Developer Preview Program FAQ - Get Windows Phone 8.1 Update 1 today - Microsoft Community

    7thsense say:

    Developer Preview Program FAQ - Get Windows Phone 8.1 Update 1 today

    Technical Level : Intermediate

    Summary
    This wiki article answers the most frequent questions concerning the Windows Phone Developer Preview program and getting early access to Windows Phone 8.1 Update 1.
    Details

    What is the Developer Preview Program?

    The Developer Preview program gives the developer community early access to new Windows Phone updates. This will give you, as a developer, time to test your apps with new OS versions ahead of the official roll-out. This is only Windows Phone OS updates and does not include manufacturer firmware updates. The update is labeled Preview but is extensively tested and is at least a RTM candidate. This is not a beta program.

    How can I join the Developer Preview program and update to Windows Phone 8.1 Update 1 today?

    1. In order to take part in the Developer Preview program you need to be a registered developer. Either register as a Windows Phone developer or as an App Studio developer. If you choose to register as a Windows Phone developer, you need to download the developer tools and developer unlock your phone. 
    2. To setup your phone to search for previews, install the Preview for Developers app on your phone and accept the terms and conditions in the app. After enabling the preview through the app, you may uninstall the app from your device.
    3. Finally check for updates on your phone: Settings > phone update > check for updates (after the install finishes, make sure you check again since there may be more than one update pending.)

    Is the developer preview version the same as the one in the official RTM roll-out?

    If any critical issues are found in the preview they will be fixed before RTM.

    Will I still get the official update when it rolls out?

    Yes, if the official RTM is a new build, you will receive it. You will also get any manufacturer firmware updates separately.

    Can I roll-back my phone to the latest official version?

    Only Nokia phones can be rolled back by flashing the phone with the latest official release. This is done using Nokia Software Recovery Tool. No other manufacturers are supplying such tool (yet).

    Where can I turn to if I run into problems with the Developer Preview?

    Visit the official Developer Preview support forums.

    Will the Developer Preview void my warranty?

    You may void any warranty of your mobile operator and OEM.

    Will the installation erase my phone?

    No. The update will only update the OS. All apps, games, images, settings, etc., will remain the same.

    What are the pre-requisites of the Developer Preview?

    Apart from being a registered developer, your phone has to be running Windows Phone OS 8.0.10322.71 or newer (check your OS version under Settings > about > more info). If you have an older version of the OS on your phone, check Settings > phone update to see if a more recent update is available. If not, contact your carrier (or OEM if your phone is unbranded) to find out when they plan to roll-out a newer version.

    Where can I find more information on the Developer Preview program?

    Head over to the official Developer Preview site.

    Is there any other way I can get Windows Phone 8.1 Update 1 early?

    No. You will have to wait for the official roll-out for your carrier (or OEM if you phone is unbranded) in the your region.

    I'm 100% sure that the device is Developer Preview enabled but I'm still not getting the update?

    A small number of devices need a firmware update from the OEM/carrier before they can receive the update.

    Printing labels in Office 2013 - Microsoft Community

    bobh21 say:

    Printing labels in Office 2013

    I use mail merge to print a short list of labels from Excel. Is there a way to start printing the next list on the same sheet so as not to waste the unused labels? Thanks.
    doug robbins - mvp office apps & services (word) say:

    One way to do it is to insert the requisite number of rows before the first row of data in the data source.

    If the data source is a Word table, the following macro will do that

    Macro to set the first label on a part sheet of labels for a label type mailmerge.

    Dim MMMDoc As Document

    Dim dsource As Document

    Dim dtable As Table

    Dim i As Long, j As Long

    Set MMMDoc = ActiveDocument

    With MMMDoc.MailMerge

        If .MainDocumentType = wdMailingLabels Then

            If .State = wdMainAndDataSource Then

                Set dsource = Documents.Open(.DataSource.Name)

                Set dtable = dsource.Tables(1)

                i = InputBox("Enter the number of labels that have already been used on the sheet.", "Set Starting Label")

                If IsNumeric(i) Then

                    With dtable

                        For j = 1 To i

                            .Rows.Add BeforeRow:=.Rows(2)

                        Next j

                    End With

                End If

                .Destination = wdSendToNewDocument

                .Execute

            End If

        End If

    End With

    dsource.Close wdDoNotSaveChanges

    If the data source is in Excel, you can copy and paste it into a Word document so that you can use that macro.

    Filter top Percent in MS Excel - Microsoft Community

    nasirbashir11 say:

    Filter top Percent in MS Excel

    I want to extract data that constitute top 25% of the total of cloumn. but when I apply top percent filter, it gives wrong data.. Plz F1.

    bill manville say:

    top 25% gives you the largest items in the column numbering 25% of the total number of items - so if the column conains the numbers 1 to 12, top25% will give you 10, 11, and 12.

     

    To get what you want will require a rather complex formula to work out the minimum value (N) such that the top N items provide at least 25% of the total.

     

    I would use a macro to compute N as it is easier than trying to write a worksheet function to do it and you will need to use a macro to apply the filter anyway.  In my example I had a column of numbers in A2:A13 with a heading in A1.

     

    Sub Test()
      FilterToTopPercent 25, Range("A1:A13")
    End Sub

     

    Sub FilterToTopPercent(Percentage As Integer, R As Range)
      Dim dTotal As Double
      Dim dSum As Double
      Dim iCount As Integer
      dTotal = WorksheetFunction.Sum(R)
      For iCount = 1 To R.Count - 1
        dSum = dSum + WorksheetFunction.Large(R, iCount)  'add the nTh largest in the range
        If dSum >= dTotal * Percentage / 100 Then Exit For    'reached or exceeded the target %
      Next
      R.AutoFilter
      R.AutoFilter 1, iCount, xlTop10Items
    End Sub
