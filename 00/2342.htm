doug robbins - mvp office apps & services (word) say:

If you insert the filename \p field into a table cell in the footer, the path\filename will wrap within that cell so that it does not encroach upon the page number.

Alternatively, if you put a { DOCVARIABLE varPath } field in the footer, you could use a macro containing the following code to set the value of the varPath variable to the path of the document

Dim i As Long, j As Long
With ActiveDocument
    .Variables("varPath").Value = .Path
    For i = 1 To .Sections.count
        With .Sections(i)
            For j = 1 To .Footers.count
                .Footers(j).Range.Fields.Update
            Next j
        End With
    Next i
End With

jim gordon mvp say:
To summarize (please correct me if I am wrong about any of this):

In PowerPoint 2010 and PowerPoint 2011, the default behavior is the same. All movies and audio files are embedded by default. When saved in standard format, the audio and video content is within the file and PowerPoint will play the content.

PowerPoint 2010 for Windows (with free Apple QuickTime also installed) and PowerPoint 2011 for Mac (with free Flip4Mac QuickTime codecs installed) will play all the same audio and video content. This is a major step forward, as linking is no longer a worry. PowerPoint 2010 for Windows finally enables the Windows version of PowerPoint to play the same audio and video files that have been supported on PowerPoint for Mac for decades. Likewise, the free Flip4Mac codecs bring wmv to the Mac.

As we go back in time to older versions, more considerations come into play.

* Text, animated text over video, and overlapping video is supported only in PPT 2010 and PPT 2011

* The only version of PowerPoint for Windows that supports QuickTime formats is 2010. Older versions of PowerPoint for Windows do not support QuickTime formats. Use wmv for cross-platform movie support (requires Flip4Mac on the Mac to play them)

* PowerPoint 2007, 2008, 2010, and 2011 can play presentations that have embedded video that was saved in standard file formats

* I don't have PowerPoint 2004 or 2007 to test whether or not these versions can play embedded video via the compatibility pack. From what I see here in this thread, I would not expect them to (although they might).

* Only PowerPoint 2010 for Windows and 2011 for Mac can embed video. Although 2007 and 2008 can play embedded video, they can not make files with embedded video.

* By default all older versions of PowerPoint for both Mac and Windows older than 2010 (ie. 2008, 2007 and so on all the way back) require that movies be linked when added to a presentation. For older versions of PowerPoint, for more see information in these links:
 this FAQ
and
 http://office.microsoft.com/en-us/powerpoint-help/my-movie-doesn-t-play-HA010077716.aspx

To make a version of your presentation for an older version of PowerPoint: first, put all the media files into a folder. Then use File > Save As and choose 97 through 2008 (.ppt) file format and save a copy of your presentation into the same folder. Then make "relative links" to the media files as described here: http://www.pptfaq.com/FAQ00155.htm
Be sure to choose to Link rather than embed in the Insert > Movie > From File dialog.
