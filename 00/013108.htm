klehlke say:

Formula - count if criteria is met in two different columns

I have a large spreadsheet with many columns....

I have column A that has either the numbers 1, 2 3 or 4 or blank.  I also have another column (column Q that will contain a date once something has been accomplished.

I am trying to write a formula that will count how many that have a 1 listed in column A and a date in column Q over multiple lines and then copy it and do the same for those that have a 2 in column A and a date in column Q etc.

Any help is appreciated.

bernard liengme say:

Let us assume the data ranges over rows 1 to 12

In A16:A19 type 1, 2, 3, 4

In B16 enter =COUNTIFS($A$1:$A$12,A16,$Q$1:$Q$12,">0")

Copy this down to row 19

This will not work for a blank, so in B20 use =COUNTIFS($A$1:$A$12,"",$Q$1:$Q$12,">0")

best wishes

amit tandon say:

Refer below image:

Enter below formula in cell B17 & copy down:

=SUMPRODUCT(($A$1:$A$14=$A17)*($Q$1:$Q$14>0))

This will work for blanks also ex. cell A20 is blank so cell B20 returns the blanks in the range $A$1:$A$14 which have a date in the range $Q$1:$Q$14.

Regards,

Amit Tandon

amit tandon say:

To update, enter below formula in cell B17 & copy down:

=SUMPRODUCT(($A$1:$A$14=$A17)*($Q$1:$Q$14>0)*(ISNUMBER($Q$1:$Q$14)))

This will also validate a number in the range $Q$1:$Q$14 (ISNUMBER($Q$1:$Q$14)) besides > 0 ($Q$1:$Q$14>0), for date purpose.
