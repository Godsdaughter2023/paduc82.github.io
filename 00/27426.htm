Public Function GetAlphaPosition(varIn As Variant)
    Const cAlpha As String = "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
    Dim x As Integer
    
    For x = 1 To Len(varIn)

            Exit For

gary's student say:

Install the following User Defined Function:

 

Function SavedDate() As Date
Application.Volatile
SavedDate = ActiveWorkbook.BuiltinDocumentProperties.Item(12)
End Function

 


User Defined Functions (UDFs) are very easy to install and use:

1. ALT-F11  brings up the VBE window
2. ALT-I
    ALT-M opens a fresh module
3. paste the stuff in and close the VBE window

If you save the workbook, the UDF will be saved with it.

To remove the UDF:

   1. bring up the VBE window as above
   2. clear the code out
   3. close the VBE window

To use the UDF from Excel:

=SavedDate()

To learn more about macros in general, see:

http://www.mvps.org/dmcritchie/excel/getstarted.htm

or

http://www.cpearson.com/excel/WritingFunctionsInVBA.aspx
for specifics on UDFs

 

 

chip pearson say:

You can use the following VBA function to get the last save time of the file.

 

Function LastSaveTime()
    Application.Volatile
    LastSaveTime = FileDateTime(ThisWorkbook.FullName)
End Function

You can call this from a worksheet cell with

 

=LastSaveTime()

That's never been a problem for me, but I've read of other users who have lost data when they deleted a partnership.

So to safeguard against data loss, backup your PIM data first either by:

  • Copying the PIM.vol file to your storage card: Open up File Explorer on the windows phone and navigate to the root folder.  You'll find a PIM file there.  Copy that file to your storage card. Then go back to the root folder and rename the PIM file to something else (or delete it); OR
  • Downloading and using the free application called PIMBackup
