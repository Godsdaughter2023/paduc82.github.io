Hi,

try this vba code..

download a picture from web (url address)

target cell is A1 in activesheet.

(change cell's dimensions as needed)

Sub Add_Picture_In_Cell()

'Apr 08, 2016

Dim rng As Range
Set rng = ActiveSheet.Range("A1")
Dim v As Variant
Dim p As Picture
Application.ScreenUpdating = False
For Each p In ActiveSheet.Pictures
If Not Intersect(p.TopLeftCell, rng) Is Nothing Then p.Delete
Next
v = "http://www.desicomments.com/wallpapers/force_of_nature/force_of_nature_33.jpg" ' << url address

'v ="D:\Folder A\Folder Pictures\picture1.jpg"  '<< from folder

Set p = ActiveSheet.Pictures.Insert(v)
With p
.ShapeRange.LockAspectRatio = False
.Height = rng.Height
.Width = rng.Width
.Top = rng.Top
.Left = rng.Left
.Placement = xlMoveAndSize
End With
Application.ScreenUpdating = True
End Sub

gregak say:

Hi Grandy,

OneNote page templates are stored locally and do not sync as part of the notebook. Your templates are saved to C:\Users\<username>\AppData\Roaming\Microsoft\Templates\My Templates.one, and you can copy this file file between computers by placing it in the same location.

Note that the contents of the "Roaming" folder in AppData does replicate to your other computers if Windows is domain joined to a managed network (such as a work network).  If this is a personal workstation, you will have to copy the files manually.

If you'd like to see OneNote sync templates like it does notebook content, please vote for it at https://onenote.uservoice.com/forums/256328-retired-onenote-users?query=sync%20templates.
