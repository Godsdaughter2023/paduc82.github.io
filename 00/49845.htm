Hi, has anyone seen this issue? Steps to reproduce issue:

Start Windows Explorer
Local C: Drive
Right Click on C: Drive
Click Properties

The Properties dialog box will default to the Sharing tab instead of the General tab. The order of the tabs from left to right is Sharing > Security . Previous Versions > General. The row above it is Tools > Hardware > Quota.

It would seem somehow the tabs got reversed.

This only happens when viewing the properties of the disk drive itself. (Note, can be a hard drive, CD Drive, Removable Drive, etc.) Does not happen on folders/files.

Please assist in figuring this out. Thanks.
kevin789 say: I have found the solution to my own problem.

In the registry I was missing the following value:

[HKEY_CLASSES_ROOT\Drive\shellex\PropertySheetHandlers]
@="{5F5295E0-429F-1069-A2E2-08002B30309D}"

I had no value set in there which caused the properties to always default to the sharing tab instead of the general tab. After entering the value {5F5295E0-429F-1069-A2E2-08002B30309D} all was resolved.

I hope this helps anyone else that comes across this issue.

mouneshwar r say:

Hi Cam65,

 

What exactly you mean by “computer all of a sudden goes into safe mode”? Do you mean to say that you get an option to boot into safe mode or the computer boots into Safe mode without your intervention?

 

Step 1: Uncheck “Safe boot” option under Microsoft System Configuration Utility

Windows 7 will not boot into Safe mode until and unless it is forced to boot into Safe mode. There’s an option in the well-known System Configuration utility that will let you force Windows to always boot into Safe mode until you turn the option off.

a.    Open up the start menu search or run box, and type in msconfig and hit the enter key.

b.    Select the Boot tab, and then check the box for “Safe boot” and make sure the radio button below is set to “Minimal”.

c.    Once you hit the OK button you’ll be asked if you want to Restart now or wait until later. Either way, the next boot will take you into safe mode. Uncheck this option if it is checked.

 

Else otherwise, once you restart, you’ll realize that you are in Safe Mode because it not only places that text in the four corners of the screen, but it even opens up the safe mode page from the help file in Windows 7 or Vista.

 

Step 2: Perform a startup repair

If Safe boot option is not checked in System Configuration utility and windows still wants to boot into Safe mode, then I would suggest you to perform a startup repair.

 

Startup Repair: frequently asked questions

http://windows.microsoft.com/en-US/windows7/Startup-Repair-frequently-asked-questions

 

Hope this information is helpful and let me know if you need any further assistance.
