jeeped say:
This could be accomplished with a simple array formula using VLOOKUP() and IF().

           

The formulas in F2:F6 are,

=VLOOKUP("A",IF($B$2:$B$10>10,$A$2:$C$10),3,FALSE)
=VLOOKUP("A",IF($B$2:$B$10<10,$A$2:$C$10),3,FALSE)
=VLOOKUP("A",IF($B$2:$B$10>=10,$A$2:$C$10),3,FALSE)
=VLOOKUP("A",IF($B$2:$B$10<=10,$A$2:$C$10),3,FALSE)
=VLOOKUP("A",IF($B$2:$B$10<>10,$A$2:$C$10),3,FALSE)

Each of those requires Ctrl+Shift+Enerr rather than simply Enter. But as you specifically requested a User Defined Function or UDF, paste this into a standard module,

Public Function SPECIALLOOKUP(sLU As String, rR1 As Range, sCRT As String, rR2 As Range, _
  rRTRN As Range, Optional iINST As Long = 1) As Variant
    Dim r As Long, i As Long
    If rR1.Rows.Count <> rR2.Rows.Count Or rR2.Rows.Count <> rRTRN.Rows.Count Or _
      rR1.Columns.Count <> rR2.Columns.Count Or rR2.Columns.Count <> rRTRN.Columns.Count Or _
      (rR1.Rows.Count > 1 And rR1.Columns.Count > 1) Then
        SPECIALLOOKUP = CVErr(xlErrValue)
        Exit Function
    Else
        SPECIALLOOKUP = CVErr(xlErrNA)
    End If
    If rR1.Rows.Count > rR1.Columns.Count Then
        For r = 1 To rR1.Cells.Count
            If rR1.Cells(r, 1) = sLU Then
                i = i + Abs(Application.Evaluate(rR2.Cells(r, 1).Address & sCRT))
                If i = iINST Then
                    SPECIALLOOKUP = rRTRN.Cells(r, 1).Value
                    Exit For
                End If
            End If
        Next r
    Else
        For r = 1 To rR1.Cells.Count
            If rR1.Cells(1, r) = sLU Then
                i = i + Abs(Application.Evaluate(rR2.Cells(1, r).Address & sCRT))
                If i = iINST Then
                    SPECIALLOOKUP = rRTRN.Cells(1, r).Value
                    Exit For
                End If
            End If
        Next r
    End If
End Function

Tap Alt+Q once you have that pasted to return to your worksheet. The syntax is,

=SPECIALLOOKUP(<first lookup>, <first range>, <logical criteria>, second range>, <return range>, [optional] <instance to return>)

From the image above G2 would be, 

=SPECIALLOOKUP("A",$A$2:$A$10,">10",$B$2:$B$10,$C$2:$C$10,1)
... or,
=SPECIALLOOKUP("A", $A$2:$A$10,E2,$B$2:$B$10,$C$2:$C$10,1)

You may note that I've added a parameter in addition to the ones in your description. If you wish to return the second, third or any subsequent match rather than just the first, you can increase the integer given in the last optional parameter. If this is omitted, it defaults to 1 (the first). I put in a check to make sure that the rows and columns of all three ranges match.  I also added the functionality for an HLOOKUP() in additional to a VLOOKUP() where you can look across a row rather than down a column.

Edit¹: added a check to make sure that either the rows or columns were 1 wide.
Edit²: added #N/A error returned on no match.
