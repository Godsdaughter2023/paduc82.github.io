ken sheridan say: You could try using the following to get the phone number:

Public Function GetPhone(varPhone As Variant)

    Dim strPhone As String
    Dim strChr As String
    Dim n As Integer

    If Not IsNull(varPhone) Then
        For n = 1 To Len(varPhone)
             strChr = Mid(varPhone, n, 1)
            If strChr = "E" Then
                Exit For
            Else
                If IsNumeric(strChr) Then
                    strPhone = strPhone & strChr
                End If
            End If
        Next n
        
        GetPhone = Format(Trim(strPhone), "(###) ###-####")

    End If

End Function

and the following to get the extension number, if any:

Public Function GetExt(varPhone As Variant)

    Dim intExtPos As Integer

    If Not IsNull(varPhone) Then
        intExtPos = InStr(varPhone, "EXT")
        
        If intExtPos > 0 Then
            GetExt = Trim(Mid(varPhone, intExtPos + 3))
        End If
    End If
    
End Function

    Thanks. The problem is caused by the left and right page margins - they have been set to 0 for the EAE sheet; the new sheet uses the default margins of about 0.7".

    In addition to setting the print area, you also have to set the margins, and perhaps the paper size:

     

            With wbPrint.Worksheets(1).PageSetup
                .PrintArea = wbThis.Worksheets("EAE").PageSetup.PrintArea
                .LeftMargin = wbThis.Worksheets("EAE").PageSetup.LeftMargin
                .RightMargin = wbThis.Worksheets("EAE").PageSetup.RightMargin
                .TopMargin = wbThis.Worksheets("EAE").PageSetup.TopMargin
                .BottomMargin = wbThis.Worksheets("EAE").PageSetup.BottomMargin
                .PaperSize = wbThis.Worksheets("EAE").PageSetup.PaperSize
            End With

    Starting point when Word does something unexpected

You may even find some settings that you want to try that have nothing to do with the current problem.

    1.       Always check Tools Options
