Can't update from Windows 7 to Windows 10

don cuthbert say:

You might be short of space, or possibly there are some damaged files, either in Windows 7 or in the Windows 10 upgrade files.

To check your Windows 7 files, open an Administrator Command Prompt and run sfc /scannow. It will take quite a while to run, but it may find and fix some errors.

Try creating Windows 10 install media on an 8GB or larger USB stick, by running the Media Creation Tool from the link below:

https://www.microsoft.com/en-us/software-download/windows10

Rub setup.exe from the USB stick to upgrade.

axk say: Can I just delete stuff on computer to get more space or do I need to use a USB stick?

don cuthbert say:

I don't know if space is the issue. But since your error said during Migrate Data it made me wonder.

If space is a problem, you can just delete large files, or move them elsewhere. Video and photo files are usually the largest. Documents are tiny in comparison.

Even having an NTFS blank, formatted USB stick inserted during an upgrade can make an upgrade that is short on space succeed. Windows will use the space on the USB for temp files.

The error may be about corrupted files, so running sfc /scannow is a good idea.

You don't have to use a USB stick, but Windows 7 cannot mount an ISO file, which is one way to download Windows 10. A program like 7-zip can extract the files to a folder if you have room for 4GB of files. But that would be on top of the 3-4GB ISO file that you downloaded. So putting the install files on a USB stick might be simplest. A small USB stick is very inexpensive, easy to find at many stores and can be reused.
