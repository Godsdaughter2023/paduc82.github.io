Hi,

Thank you for posting in Microsoft Community.

Disk Defragmenter is a tool that rearranges the data on your volume and reunites fragmented data so your computer can run more efficiently.

Fragmentation happens to a volume over time as you save, change, or delete files. The changes that you save to a file are often stored in a different place on the volume than the original file. This doesn't change where the file appears in Windows—only where the bits of information that make up the file are stored on the actual volume. Over time, both the file and the volume itself become fragmented, and your computer slows down as it has to look in different places to open a single file.

Note: If the disk is already in exclusive use by another program or is formatted using a file system other than NTFS file system, FAT, or FAT32, it can't be defragmented. Network locations can't be defragmented.

    This is an interesting article about file deletion and recovery of same:

    Why Deleted Files Can Be Recovered, and How You Can Prevent It

    When you delete a file, Windows removes the pointer and marks the sectors containing the file’s data as available. From the file system’s point of view, the file is no longer present on your hard drive and the sectors containing its data are considered free space.

    However, until Windows actually writes new data over the sectors containing the contents of the file, the file is still recoverable. A file recovery program can scan a hard drive for these deleted files and restore them. If the file has been partially overwritten, the file recovery program can only recover part of the data.

    Will Disk Optimization on SSD? - Microsoft Community

    srosea say:

    Will Disk Optimization on SSD?

    Hi,

    Cam I run Windows Disk Optimization on my Samsung 840 Pro SSD in Windows 8.1 Pro RTM?

    Saroseus

    a. user say:

    Hi Saroseus,

    No, it is not recommended to run Disk Optimization on Solid State Drives. Defragmenting SSDs isn't particularly helpful as it can potentially shorten the life of the drive, not to mention reduce performance.

    In Windows 8/Windows 8.1, the "Optimize Drives" feature will automatically defragment HDDs and TRIM SSDs as needed on a schedule, or when you manually run it. It sends a complete set of trim hints for the entire volume again - this is done at idle time and helps to allow for SSDs that were unable to cleanup earlier. We do not do a traditional defrag (moving files to optimizer there location for space and performance) on SSDs.

    Refer to the link to get more information on this feature:

    http://windows.microsoft.com/en-CA/windows-8/improve-performance-optimizing-hard-drive

    Refer to the suggestions from Ronnie Vernon on July 16th, 2012:

    http://answers.microsoft.com/en-us/windows/forum/performance/optimize-and-defragment-drive-on-windows-8/8e87d228-24e9-4ce0-9dd3-4388bd6ee835
