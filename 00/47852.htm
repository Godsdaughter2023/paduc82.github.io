aakkam22 say:

If I do that, will MS come back and do it again later anyway? And is there anything in it that is important that I should have (like security or something)?

Yes and no.

If you're running Windows 10 Pro, you can defer the upgrade for at least 8 weeks. Go to the Windows Update page in the Settings app, click "Advanced Options" and check the box to "Defer Upgrades".

If you're running Windows 10 Home, you won't have the option to defer upgrades, so you will be forced to download it again.

Not Recommended: Disable Windows Update altogether

If you are running Home and really don't want the AU, you can choose to nuke Windows Update altogether. Not only will this stop the AU from installing, it will disable updating altogether. You won't get security or feature updates anymore. You decide if you want the tradeoff.

1. Open the start menu, type services.msc and hit Enter.

2. Find the Windows Update service from the list of services.

3. Double-click it and set the startup type to Disabled. Restart your computer.
