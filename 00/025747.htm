How can be the default value of spinbutton set as the value of - Microsoft Community

kkleung say:

How can be the default value of spinbutton set as the value of activecell

How can be the default value of spinbutton set as the value of activecell? The setting of value in property only accept numbers rather activecell or name.

hansv mvp say:

In the code behind a userform, Me refers to the userform itself. You are not required to use it, but when you type Me and then a point (period, dot), like so:

Me.

you will get a popup list of available controls, properties and methods of the userform. When you then type the first letters of the name of the spin button, the name will be selected in the list and you only have to press the Tab key to enter the complete name into the code module.

The same holds once you have Me.SpinButton1 and type a point again:

Me.SpinButton1.

You then get a list of the properties and methods of the spin button. In this example, we want to set the Value of the spin button.

hansv mvp say:

You can use the UserForm_Initialize event procedure of the userform for this.

In the Visual Basic Editor, double-click an empty part of the userform.

Copy the following code into the code module that appears:

 

Private Sub UserForm_Initialize()
    On Error Resume Next
    Me.SpinButton1.Value = ActiveCell.Value
End Sub

 

where SpinButton1 is the name of your spin button. Each time the userform is loaded, the spin button will get the value from the active cell. The line On Error Resume Next suppresses the error message that you'd get if the active cell contains an invalid value.
