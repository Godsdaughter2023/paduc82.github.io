If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

Refer the article ‘Which version of Windows operating system am I running’ to provide the details.

 

However, you may try the below suggested methods to remove Run as administrator from the context menu.

 

Method 1

Note: Please ensure you’ve logged in as an administrator before performing the below steps.

  1. Right click on the program’s shortcut or exe file and choose Properties.

  2. Switch to compatibility tab and Uncheck the box which is next to “Run this program as an administrator”

  3. Click on Apply and OK.

  4. Repeat the same steps for the other application for which you wanted to remove Run as administrator from the context menu.

If you do not see any changes after trying Method 1, you may try Method 2.

 

Method 2

  1. Click on Start, type in Control Panel.

  2. Go to System and security group of settings > Click on Security & Maintenance and expand the options under Security.

  3. Scroll down and see the Windows SmartScreen section.

  4. Click on Change settings(You’’ need admin rights to make these changes)

  5. Toggle the options to Warn before running an unrecognized app, but don’t require administrator approval.

  6. Click on OK and check the status of the issue.

 

If the issue still persists, please reply to us with the details mentioned above and we will assist you further.
