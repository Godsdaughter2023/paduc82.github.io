You can use any of the events linked to the worksheet - the Sheet Activate event would probably be the best - to change the size of the buttons to match their parent cell, like so:

1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
6) Make changes as needed - one loop for each button

Private Sub Worksheet_Activate()
    With Me.Buttons("Button 1")
        .Left = .TopLeftCell.Left
        .Top = .TopLeftCell.Top
        .Width = .TopLeftCell.Width
        .Height = .TopLeftCell.Height
    End With
End Sub

    DoCmd.OpenForm "frmYourFormName", DataMode:=acFormAdd

Using Word as an Alternative to Scrivener - Microsoft Community

raindog308 say:

Using Word as an Alternative to Scrivener

One of the best software products for writing fiction is Scrivener, which I own.  However, I also have a Surface tablet and Scrivener has not been ported to it (nor will it likely ever be - they haven't even done an iOS version).  This means lugging a laptop if I want to write and I was hoping to make travel tablet-only.

My use of Scrivener is really light and I'm wondering if there is a way to replicate the core functionality in Word.

What Scrivener does is:

  • On the left plane there is a list of texts - think of them as passages.
  • On the right is the editor window, which is just a text editor.
  • Select which you want on the left and edit on the right.

The key is that you can move the passages around in the left pane which is massively useful - and you don't have to do a copy/paste as you would in Word if you wanted to move a scene from Chapter 4 into Chapter 2.  It also allows you to see all your scenes in one glance instead of scrolling through 100 pages.

Another key feature is the ability to say "now assemble all those scenes that I've ordered the way I want into an RTF".

I don't mind getting down and dirty into macros if I need to - just wondering if there is this word of "outline overview" view or if it can be simulated/recreated in Word somehow.

jay freedman say:

In Word 2013 there are at least three ways to accomplish the rearrangement. All of them depend on each of the "passages" starting with a paragraph with a Heading style. These can be the built-in Heading 1, Heading 2, etc. or custom styles you define with appropriate outline levels.

Method 1: On the View tab, click Outline. In the Outline view, you can select the depth of outline levels you want to display; all the subordinate levels will be collapsed. You can then drag-and-drop the displayed paragraphs to a different order, and their subordinate paragraphs will be moved with them. You can double-click the plus-in-circle icon next to any heading to expand its subordinates for editing. The article at http://www.word.mvps.org/FAQs/Formatting/UsingOLView.htm was written for Word 2003, but most of the description there is still valid for Word 2013.

Method 2: In the Print Layout view, when you hover over any Heading paragraph, a small triangle appears in the left margin. Clicking that triangle toggles the collapse and expansion of all the text subordinate to that heading. Right-click any Heading paragraph and click the Expand/Collapse item, and then you can choose to collapse or expand all headings. When they're collapsed, you can drag-and-drop in the same way as in the Outline view. You can expand any single heading to see its subordinates.

Method 3: When the Navigation pane is displayed (on the View tab, check the Navigation Pane box; it also appears automatically when you press Ctrl+F to start a Find), you can drag-and-drop the headings within the list, and the subordinates will go with them.

Once the document is in the desired order, use the File > Save As command (shortcut F12) and set the "Save as type" dropdown to Rich Text Format. If you prefer, you can go to File > Options > Save and set the "Save files in this format" dropdown to Rich Text Format to make that the default format (but you can always change the format setting in the Save As dialog to something else for one save).
