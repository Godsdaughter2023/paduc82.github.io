bob jones aka: cybertaz say:
Tracked Changes are never removed from a document until you:
  1. Turn Track Changes Off
  2. Accept/Reject all changes
  3. Save the file [or Save As to create a copy not containing the changes.

For more discussion if you need it, use the term track changes in the Search field in the orange banner atop the ANSWERS web page.

BTW: You actually aren't using "2008 in 2004 compatibility mode" :-) You're using Word 2008 but you're working with a document which is saved in the deprecated .doc file format. Especially if you intend to use Track Changes I'd suggest you save your work in the .doxc file format introduced roughly 5 years ago. Among other advantages it's considerably more stable.

teylyn say:

Hello,

you need to use an XY scatter chart, since line charts don't show units smaller than a day on the x axis.

Format the X axis and set the major unit to 0.010417, which is the numerical equivalent of 15 minutes. Note that with 15 minute increments, you need to make the chart about three screens wide, so all data labels show. You also may want to turn the axis labels 90 degrees.

You can set the number format of the axis to "Linked to source" so it will show the same format as your source data.

You need 2 things to re-install Office, 2013 or 2010: your Unique 25 character Product Key and the associated generic installation file / disk.

The key point is you have to find the 2013 25 character Product Key you used.  After you have that it is easy to get the installation file.
