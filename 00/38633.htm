Alternative to upgrade Windows 10 and also solve Error - Microsoft Community

gungurru say:

Alternative to upgrade Windows 10 and also solve Error code:-0xc0000017. “There isn’t enough memory available to create ramdisk device.”

I tried different methods such as Windows Update Downloading, changing system language and region,  boot from USB with Windows 10 media file (downloaded from Microsoft website) & etc. to upgrade Windows 10 Pro from Windows 8.1 Pro.  All failed several times with different error codes.

 

Run setup.exe from the USB with Windows 10 media file in Windows 8.1 Pro operating system. Still failed from 1st reboot with Error code:-0xc0000017.

 

“There isn’t enough memory available to create ramdisk device.” 

 

Spending several days research and using the below method.  It worked perfectly and upgrade to Windows 10 from Windows 8.1 Pro and Windows 7 Home Premium on my two computers without any issue.

 

  1. Run Windows 7 or 8.1 operating system as normal;
  2. Insert the DVD or USB Windows 10 with correct system and edition created  through Microsoft Media Creation Tool;
  3. Run the following command in an administrative command line window clearing the bad memory:

            “bcdedit /deletevalue {badmemory} badmemorylist”  (No quotes)

  1. Execute Windows 10 setup.exe file from DVD or USB to upgrade Windows 10.

 

This is the only solution working perfectly for me.  Hopefully the aforesaid information can help others.
