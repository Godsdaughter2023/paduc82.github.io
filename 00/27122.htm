O365 Business offers MS-Office Business equivalent to MS-Office Standard

O365 Pro Plus offers MS-Office Pro Plus which is a higher feature version of MS Office.

The restriction for 300 in the O365 Business family is that you can only subscribe maximum of 300 licenses within a single Office365 account (we refer to this as Office365 Tenant).

O365 is priced by per user per month. So USD8.25 is for one user. If you need 300 users is USD 8.25 x 300 per

month.

Also, the 5 installs refer to 5 devices belonging to the SAME user (e.g. PC, Notebook, MacBook, Home PC..etc)

So it is NOT 60 licenses and install 5 devices so become 300 machines installed.
