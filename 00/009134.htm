That error almost never has anything to do with the amount of memory available.

What is happening is that Word is failing to write the file to disk.  The only indication it gets from the OS is "Write Failed", and the only reason Word can think of is that you may be out of memory.

In fact, it's likely to be a corrupt document.  If Word can't correctly ready the code in the document, it is unable to save it.

The Maggie:

3. Copy it.

5. Save under a new file name and close all, then re-open.
