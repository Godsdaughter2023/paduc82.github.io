The answer given by the MSFT folks above is (unfortunately) woefully incomplete.  This change will cause new installations to be redirected -- this is true.  But any references to
 %ProgramFilesDir% elsewhere that expect to find where programs were already installed will now be broken.  If you change this registry entry, you are unfortunately taking it upon yourself to find all the myriads of variations of specifying this folder in the registry  ("C:\Progra~1", C:|Program Files, "%SystemDrive%\Program Files", etc., etc.).  And you must also move all the existing files to the new location (sometimes requiring tedious ownership and permission changes and conflicts with files in use, etc.)  If you don't do all this, you will have two (or more) locations known at one time or another as the ProgramFilesDir, and things will be broken for some programs.

erdeslawe say:

When I go to save a document windows 10 keeps telling me I need the administrators permission.  why is this happening and how do get Windows 10 to stop telling me this so I can just save my documents in my documents file and not have to save it elsewhere?

My guess would be that you either did not disable/uninstall your Anti-Virus/Security Software as advised by Microsoft, or a Cloud Syncing Application (such as the Desktop version of OneDrive) when you performed the Windows 10 upgrade; and, as a result the system's protection has kicked in and changed all of your user Folders to 'Read Only' access.  

You need to take ownership of those Folders and restore full user permissions again, using the Admin Account, e.g. My Documents, Music, Pictures etc.
