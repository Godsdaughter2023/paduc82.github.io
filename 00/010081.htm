Might be you would rather use code to motorize the process?

 

Sub Fill_Blanks()
'by Dave Peterson  2004-01-06
'fill blank cells in column with value above
Dim wks As Worksheet
Dim rng As Range
Dim lastrow As Long
Dim Col As Long

    Set wks = ActiveSheet
    With wks
        Col = ActiveCell.Column
    'or
    'col = .range("b1").column

        Set rng = .UsedRange        'try to reset the lastcell
        lastrow = .Cells.SpecialCells(xlCellTypeLastCell).Row
        Set rng = Nothing
        On Error Resume Next
        Set rng = .Range(.Cells(2, Col), .Cells(lastrow, Col)) _
                .Cells.SpecialCells(xlCellTypeBlanks)
        On Error GoTo 0

        If rng Is Nothing Then
            MsgBox "No blanks found"
            Exit Sub
        Else
             rng.NumberFormat = "General"
            rng.FormulaR1C1 = "=R[-1]C"
        End If

    'replace formulas with values
        With .Cells(1, Col).EntireColumn
            .Value = .Value
        End With

    End With

End Sub

suzanne s. barnhill say:
FWIW, the font in use appears to be Helvetica Neue, which is not the same as Helvetica, so if you do install a new font, make sure you get the right one.

john korchok say:

As Suzanne has pointed out, you don't have Helvetica Neue installed on your computer, so Word is substituting a different font. You'll only get Arial automatically if the original font is plain Helvetica. But you can fix this. With your demo document open in Word:

  1. Choose File>Options>Advanced.
  2. Scroll down to the Show document content section and click on the Font Substitution button.
  3. The left-hand column includes Helvetica Neue. Select it.
  4. Change the Substituted font dropdown on the right to Arial. OK out.

Word will remember your selection and display Arial when the Helvetica Neue font is used in a document.
