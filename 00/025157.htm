Each time there is a software update, the Outlook sounds revert back to the current Microsoft set.  I agree that it is difficult to hear new mail when out the room with the "tink" sound.  I like a much older set better.  I saved a message several years ago from someone who had the answer. I didn't save his name so can't give credit but here is how to change one or more of the individual sounds. You can change just one sound or all 4. This is for Outlook for Mac 2019 but also worked for the 2016 version.

Make sure you have closed Outlook.

Navigate to the folder where Outlook is installed. Typically, it’s in the Applications folder.

Right-Click on Microsoft Outlook app. and click Show Package Contents

Navigate to Contents -> Frameworks

Find the OutlookCore.framework file and expand the tree.

Open the Resources folder.

Backup your newmail.wav file into a local path just in case something goes wrong

Replace the newmail.wav file with any wav file of your choice

Note: You will likely need to authenticate by entering your administrator password to replace the file.

Start Outlook and try it out.

This has always worked for me. I change mailerror.wav, mailsent.wav, newmail.wav and nomail.wav back to my old reliable sounds with every MS update.   The easiest way is to keep a folder with these 4 sounds (or any you choose to change) on an external HD.  Then when you copy them into the outlook folder on your main machine or laptop, the originals remain in the folder on your ext HD.
