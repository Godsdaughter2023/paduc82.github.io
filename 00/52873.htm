If you don’t get the option to upgrade it doesn’t necessarily mean that your system can’t run Windows 10. Download the MEDIA CREATION TOOL and upgrade using that. This will activate your upgrade and a unique activation key based on your hardware configuration (hardware hash) will have been generated during the upgrade process and will be deposited on a Microsoft server.

You can use the same MEDIA CREATION TOOL to create a win 10 bootable usb disk or dvd to do a clean install of Windows 10. If you haven’t purchased a key then you MUST upgrade first to obtain that hardware hash key. During a clean install skip the 2 requests for an activation key (unless you purchased one). You’ll find that Windows 10 will activate after connection to the internet because the key which was generated during the previous upgrade process will be retrieved from the Microsoft server. Before you do a clean install, after the upgrade, this is a good time to put in a new hard drive or SSD as this won't affect the hardware hash.
