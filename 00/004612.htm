Some users might be experiencing a glitch with starting the Threat Service (Windows Defender Antimalware Service) that was delivered on Patch Tuesday. So the first thing to try would be manually restarting the PC several times, with a few-minute break between the restarts (Start button > Power > Restart), as suggested by Le Boule in this thread:

 

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning-windows_10/windows-defender-virus-threat-protection-restart/9215a1e1-d386-477b-b22f-d608365b255a

 

If that doesn’t work, then check the Security Center for the status of Windows Defender, and for any messages pertaining thereto (Control Panel > Security and Maintenance).

 

Refer to this thread for further troubleshooting steps:

 

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning-windows_10/windows-defender-the-threat-service-has-stopped/ff4395cf-d93c-4e1c-a6fc-d14cdcaf8bc7

 

Additionally, you can query the registry to make sure that Defender is properly enabled in its own registry folder.

 

Right-click on the Start button and launch Windows PowerShell; and then copy, paste, and enter these commands

 

REG QUERY "HKLM\SOFTWARE\Microsoft\Windows Defender" /v DisableAntiSpyware

 

REG QUERY "HKLM\SOFTWARE\Microsoft\Windows Defender" /v PassiveMode

 

Both parameters should be set to the value of 0.

 

Also, if you’ve already run the cleanup tools for any preinstalled or previously installed AV apps, and also confirmed the integrity of the system with both the System File Checker and DISM, then you can try rebuilding the WMI Repository to clear any possible corruption in the Security Center:

 

https://support.avast.com/en-us/article/Troubleshoot-unknown-Antivirus

 

If you can’t get Windows Defender up and running, then you’ll either have to resort to the recovery options,

 

https://support.microsoft.com/en-us/help/12415/windows-10-recovery-options

 

Or else install a free third-party AV app for real-time protection so that you’ll be able to connect to the internet safely:

 

https://www.pcmag.com/article2/0,2817,2388652,00.asp

 

 

GreginMich

You may turn off automatic app updates and install it manually when you are not using Internet for other purposes. Check if it helps.

  1. Open Windows Store.
  2. Click on round icon near to search box in the top right corner.
  3. Click on Settings.
  4. Move the slider and turn off automatic app updates.

To check the app updates manually, follow these steps:

  1. Open Windows Store.
  2. Click on round icon near to search box in the top right corner.
  3. Click on Downloads and updates.
  4. Click Check for updates.

Hope this information helps. Reply to the post with an updated status of the issue so that we can assist you further.
