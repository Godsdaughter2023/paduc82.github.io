OK,

Multiple steps here so please be patience with me... 

Preparation:  We should assume the AV you have running failed to do the job and is likely compromised.  If you paid for that software, figure out how to preserve the license for reinstall later.  Once you have done that, uninstall it (for now) so it doesn't prevent the following steps from working as intended.

Cleaning Up:

1.  Check for hidden Hooks:  I prefer an off-line scanner when removing malware because sometimes the malware hooks the operating system.  When it hooks the OS, it can hide part of itself from AV and re-infect the system after you clean it off (sounds like what you have happening currently).  I'd try Windows Defender Off-line.  From a clean system download the appropriate version, double click on the file and it will step you through building a DVD which you will use to boot from on the suspect system.  The file you need for this step is found at:  http://windows.microsoft.com/en-US/windows/what-is-windows-defender-offline

  Note:  Know how to force your PC to boot from DVD before you attempt this (see mfr user guide ).   Typically its a function key (e.g. F10) that you hit when the system initially starts. 

2.  Hedge your bets:  Use a 2nd Anti-malware scanner such as McAfee Stinger (free) after the system boots up normally.  You can download stinger from:   http://www.mcafee.com/us/downloads/free-tools/stinger.aspx

By using two diverse AV engines, your likelihood of detecting and removing the malware is better.

3.  If your Files are Encrypted:  According to research, depending on the version you have, this malware could leverage Crypto-ransomware capabilities.  These types of malware encrypt files and offer to sell you the decryption key via bitcoin or Greendot card.  Depending on the malware, its possible (not guaranteed and totally at your own risk) that the files could be decrypted using a utility offered on Kapersky's site:  http://support.kaspersky.com/viruses/disinfection/2911#block2

4.  Perform Spring Cleaning:  Clear your browser's Cache, the likely location of any additional malware lurking that could re-infect you.  To clear you cache using the appropriate link from below:

Windows Internet Explorer:  http://windows.microsoft.com/en-us/windows-vista/delete-temporary-internet-files

Google Chrome:  https://support.google.com/chrome/answer/95582?hl=en

5.  Remote Stinger and reboot.  This prepares your system for installation of a more permanent AV solution.

6.  Locate and install a full package Anti-virus application; ensure you choose to allow any webpage safety review services (may have prevented this in first place).  Selecting AV products is like buying a new car; its not a one size fits all answer.  I tend to recommend staying with industry leaders (e.g. Mcafee, Symantec, Kaperski, ESET, etc..).

I hope this works for you.  Good luck

e - All error messages

”*” - Wildcard, log all information except for the v and the x option. To include the v and the x option, specify “/l*vx”.

Hi,

 

You are losing the structure of the ElseIf statement on this line

 

ElseIF Cell.Value = Cell.Value

 

The structure is

 

Elseif - condition - then - result

 

as you can see you have no THEN in that line but in any case the line would do nothing because I think you're trying to say if none of the conditions are met then do nothing so corrected the code becomes like below. Note to try and make it clearer i've included an ELSE statement which isn't actually required but as I said I hope it makes things clearer

 

Sub Change_Cell_Value()
Dim Cell As Range
For Each Cell In ActiveSheet.Range("A2:A1000").Cells
    If Cell.Value = "NON-CRP" Then
        Cell.Value = "YES IT WORKS"
    ElseIf Cell.Value = "C03" Then
        Cell.Value = "CMC3"
    ElseIf Cell.Value = "Host K27" Then
        Cell.Value = "HK27"
    Else
        'Do nothing
    End If
                
Next
 End Sub
