You need a couple of macros.  Right click the sheet Tab and choose View Code.  Copy this code into the sheet code module.

Private Sub Worksheet_Change(ByVal Target As Range)
Set Target = Range("B6")
HideRows
End Sub

This just calls the HideRows Macro that must be placed in a standard module.

Press ALT + F11, Insert Module and copy the following into the module

Sub HideRows()
Dim cell As Range
Set cell = Sheets("Sheet2").Range("B6") 'Change the sheet name to suit
If cell = "Yes" Then
    Rows("53:64").Select
    Selection.EntireRow.Hidden = True
Else
    Rows("52:65").Select
    Selection.EntireRow.Hidden = False
    Rows("56:65").Select
    Selection.EntireRow.Hidden = True
End If
End Sub

Press ALT + Q to return to the sheet

You will have to save the workbook as a macro enabled workbook
