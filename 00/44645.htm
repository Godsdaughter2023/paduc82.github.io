Ah, you bring up a great point.  So, beginning with Windows NT 4.0 (shortly before Windows 95), Microsoft started this "My Documents" craze.  I say craze because soonafter, everything had to have the word  "My" in it:  My Documents, My Computer, My Pictures, My Videos, My Music, My Network Places, My My My My.  Ugh.  I digress...

Well, some relief came. As you've seen, the folders may say "My Documents" as their name, but their truth path is just "Documents".  So how's that "My" getting in there anyway?  It's from a hidden file within the folder called desktop.ini.  Almost every folder has this hidden file.  This hidden file includes the display name the folder should call itself.  Your Documents, Videos, Pictures, and Music folders have special version of this desktop.ini file that uses a variable.

So if you go go your computer from a network drive, for example, the folder would appear to be called "Shawn's Documents".  The path is still \\computer\users\shawn\documents but the display name would include my name.

So that's the rational behind that weird naming convention.  There is good news though.  From the command prompt you can use variables like %userprofile%\documents to access the current user's documents folder, regardless of the path.
