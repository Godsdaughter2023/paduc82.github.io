Microsoft word custom tables - Microsoft Community

stephensmith4 say:

Microsoft word custom tables

When I create a table and do things like change the size and add colours to the boxes etc is it possible to save that table to a button and add it to one of the bars?

So I can reuse it without having to do all the formatting again?

charles kenyon say:

You have two possibilities:

  • Add to the Tables Building Blocks (and/or)
  • Add to the Table Styles

Both require that you have the table selected.

If you want to reproduce the entire table, you can put it in the Insert Table gallery by adding it to building blocks. If you want to be able to share this you will want to save it somewhere other than in the Building Blocks.dotx file. It will appear under Quick Tables.

  1. With the table selected, press Alt+F3. This opens a dialog box.
  2. Make sure the Gallery is Tables or Custom Tables.
  3. Give your entry a name. The one shown starts with an underscore so that it will show up at the top of the gallery (under the Category).
  4. Again, if you want to be able to share this, you want to save it in a different template than your normal.dotm template or the building blocks.dotx template. Where can Building Blocks be stored?
  5. Galleries are displayed in alphabetical order. Even if you store your Table in a different template, you can use the Built-In category. If you create your own category you will want it to appear before "Built-In." The easy way to do that is to begin the name with an underscore.

If you want to save it as a Table Style, that can be done as well. I don't use Table Styles and so do not know as much about that. I do know that the Style can be saved to a different template, but it is more difficult to make that easily sharable for all documents. A Global StyleSheet in Microsoft Word?

  1. With your table selected, go to the drop-down for Table Styles. This is under the Table Tools > Design Tab.
  2. This gives you the full Table Styles gallery with the command to create a new Style.
  3. You will have the opportunity to name your Style and pick a place to save it. Choices for saving are limited to the open document or its template.

Hope this helps.
