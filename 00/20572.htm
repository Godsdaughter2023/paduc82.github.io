My document contains many occurences of an text string (letters) ending in various numbers; I need to find each occurence and enclose in square brackets

e.g. have xx-x-1 xx-x-10 xx-x-100 and need [xx-x-1] [xx-x-10] [xx-x-100]

Is there a method where I can do this?

deanh say:

My document contains many occurences of an text string (letters) ending in various numbers; I need to find each occurence and enclose in square brackets

e.g. have xx-x-1 xx-x-10 xx-x-100 and need [xx-x-1] [xx-x-10] [xx-x-100]

Is there a method where I can do this?


With your example here you can use the following in the Find What box:

[A-z]{2}-[A-z]{1}-[0-9]{1,3}

Specifically 2 letters dash one letter dash upto three numbers

Use the following in the Replace With box

[^&]

Where the ^& means what you found (do not change) and place square brackets at either end.

You must have Use Wildcards checked in the More options.

It works on a little example I ran here.

If you have already started adding the square brackets, you can Next these occurances if you do the Find one by one, or you can alway Find Replace [[ and ]] with [ ] later (without Wildcards checked).  :-)

Hope this helps
