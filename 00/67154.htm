a. user say:

Windows logs you into a temporary profile when your own user account has become corrupted to the point where it is no longer usable.

The purpose of the temporary profile is to give you the opportunity to create a new user account for yourself. You can't use a temporary profile as a user account. Well, you can, actually, but being temporary, none of the changes you make from a temporary file will be kept after a reboot.

Use the temporary profile to switch to a user account with Administrator permissions. In the Administrator account, create a new Standard User account and - this is critical - move all your personal files from your old, corrupted user account (documents, pictures, videos, etc.) to the new user account.

Once you are sure that there's nothing left in your old user account that's of any value to you, you can delete the old user account. and continue on with your new user account.

les_p say:

After  spending hours and trying everything I could think of, I solved the problems. I believe that it should help in any OS.

I have W7pro, IE9 and WLM 2011.

The error message was: "an incorrect password was entered" with server error: 0x800CCC90 and WLM error:0x800CCC92. It wasn't true. There are several forums with the same problem and the same error messages. 

I changed passwords and could log in from "Webmail" with no problem. I had no problem using WLM with other email accounts (Shaw). 

SOLUTION

When opening a new email account in WLM, you have to fill in information in shown windows. The first one is "email address"  (e.g.: *****doe.com).  When you come to the next page, there is "log on user name" window. The program (WLM) already filled it in from the entered email ("john" taken from the email address). You have to change it to fill in full email address (******doe.com) as the "log on user name". Then it works with no error messages.

WLM error  "an incorrect password was entered" is missleading. It's not an error with a password.It's the error in WLM program, taking only the front part of the email address as a default for the user name.

viennafiji say:

There is 5 possible reasons for the blue screen,

1) Component on the laptop is on its way out or the link between the screen and the laptop is squashing the cable some how.

2) Ram or Harddrive installed is not seated correctly or cable to harddrive is not firmly fitted right.

3). Power issue with fan or unit is cause interruptions.

4) Bad software file associated to the graphic driver such as a bad update or wrong device driver or software for your laptop.

5) Third party anti-virus software or Windows defender is blocking the loading of the file associated to sleep and waking up, and or bad indexing build has occurred.

The last two 4 and 5 can be corrected through system restore point and or via good backup if you have one.

After wise Turn off any tamper protection, random folder protection feature of your anti-virus program or Windows defender then proceed to uninstall software in the order you installed them until you locate the issue then try reinstalling them one by one creating system restore point after each successful install.

if none of that work uninstall your graphic device drivers via the device manager restart your computer and then wait 15 minutes for Windows to install its basic drivers before installing the newer up to date graphic driver for your model of laptop and see if that solves the issue before turning on any tamper protection or random folder protection software.
