This section does not mean, however, that you have the right to install the software, or use the primary function of the software (other than the features listed in this section), on any of these other devices.

Unfortunately since these are not associated file types (like DOC/DOCX for Microsoft Word, or PDF for Adobe Reader) it's difficult to impossible to un-do the associations you have made for them using other programs such as Notepad.  Note that simply putting them back to their original association (or deleting the association you made) doesn't prevent someone else from coming along and doing the same thing you did, opening them with another program and changing their association to the new program.  If you don't have a System Restore point that goes back far enough then you may have to live with them as they are.

rgharper911 say:

Well yes, that would work, but it would be an extreme action to take to fix a problem that is really not a problem.  It doesn't matter whether the file shows a Notepad icon, or some other icon, or even just a generic (non-associated) icon.  The file will still work as it was intended to even if it has an icon different than the one you expect.  It's the fact that you associated the file with Notepad, etc. that caused the icon to change - nothing else has changed other than the icon.
