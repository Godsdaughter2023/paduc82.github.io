The twice-yearly "full build updates" reset settings more globally then a typical update. 

Although they don't get new brand names like they used to, these updates (like 1709 and 1803) are actually "entirely new versions of Windows" and they install in much the same way as if you upgraded from Windows Vista to Windows 7.  In those days if you did a full system OS replacement like that and some options got reset, it was usually seen as "expected", because a pro user installing an entirely new OS wouldn't typically think twice about a setting like that getting reverted.

But now when these types of full OS replacements happen in a more transparent manner, these settings getting reverted are perceived as more intrusive.  The underlying cause is the same, but the way the updates are handled where they try to give the illusion of "just an update" rather than a full OS upgrade, it is frustrating like you said.
