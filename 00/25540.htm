jay freedman say:

If you want to use a .docm instead of a .dotm template, add this code to the end of the code in the other thread:

Private Sub wdApp_DocumentBeforeSave(ByVal Doc As Document, _
                SaveAsUI As Boolean, Cancel As Boolean)
    Dim prompt As String
    Dim newName As String
    Dim newType As WdSaveFormat
    newName = Doc.Name
    While UCase(newName) = "LETTERHEAD.DOCM"
        prompt = "To prevent overwriting, you must supply a " & _
            "different file name for this document."
        newName = InputBox(prompt, "Letterhead")
        If Trim(newName) = "" Then
            Cancel = True
            MsgBox "The file has not been saved! Try again."
            Exit Sub  ' stop, don't save at all
        End If
    Wend
    
    newType = WdSaveFormat.wdFormatDocument97
    If UCase(Right(newName, 5)) = ".DOCX" Then
        newType = WdSaveFormat.wdFormatXMLDocument
    ElseIf UCase(Right(newName, 5)) = ".DOCM" Then
        newType = WdSaveFormat.wdFormatXMLDocumentMacroEnabled
    End If
    
    Cancel = True ' prevent saving with old name
    Doc.SaveAs FileName:=newName, FileFormat:=newType
End Sub

This will automatically be called whenever the document is about to be saved, but before the save happens. (This includes pressing Ctrl+S, clicking Office button > Save or Save As, pressing F12, or closing the document or Word while there are unsaved changes.)

rick rothstein, mvp - excel say:

First thing I would do is Conditionally Format columns E and F to give your boss a warning when he accidentally puts two numbers on any one row in those columns. (**See Edit Note Below**) To do this, select Columns E and F, then click Format/Conditional Formatting from Excel's menu bar, then select "Formula Is" from the drop-down box labeled "Condition 1" and Copy/Paste this formula into the adjacent field...

=COUNT($E1:$F1)>1

Next click the Format button and select the Patterns tab on the dialog box that appears, then click on one of the color squares and OK your way back to the worksheet. Now, if your boss enters two numbers in Columns E and F on the same row, those cells will highlight in the color you selected. (** Continue from Edit Note here**) Okay, now back to your question - put this formula in G7 and it can be copied up/down as needed...

=E7&F7

Note: This formula will show both values in E7 and F7 if your boss puts numbers in both cells which is why the Conditional Formatting is an important step to implement.

Click the Conditional Formatting button on the Home tab in the Styles palette, then click "New Formatting Rule" in the popup menu that appears and select "Use a formula to determine which cell to format"... enter the first formula I gave you above into the indicated field and then press the Format button, select the Fill tab and pick the color you want, then OK your way back to the worksheet. On Error GoTo StopMacro

StopMacro:

If Err.Number <> 0 Then
