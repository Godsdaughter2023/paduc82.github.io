amit tandon say:

Enter below formula in cell B2 & copy down:

=IF(A2<>"",IF(COUNT(A$1:A1),A2-LOOKUP(2,1/(A$1:A1<>""),A$1:A1),""),"")

You may use / copy this to any range / column.

In case this is your requirement. If you want to return the SUM of each of these deductions in a single cell (ie. without calculating each value in column B but calculate directly from column A), a formula can be given though it might be somewhat complex.
