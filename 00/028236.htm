ossiemac say:

Why not simply use the checkbox changes. However, it depends on whether you are using ActiveX CheckBoxes orForms CheckBoxes.

 

The first solution is for ActiveX CheckBoxes. Right Click the worksheet tab that has the CheckBox and Select View Code. Copy the code below into the Wotksheets Module. There is no need to use Select for the Worksheets and Ranges. Simply address the worksheets and ranges as I have done.

 

If your ActiveX CheckBox name is not CheckBox1 then you will need to alter the sub name to reflect the name of the Checkbox. Note that the name of the sub is critical and must be in the following format with only the CheckBox1 changed if necessary to match your CheckBox name.

 

Private Sub CheckBox1_Click()
    If ActiveSheet.CheckBox1 = True Then
        Sheets("modules").Range("B10").Copy _
                Destination:=Sheets("BallParkFigure").Range("G6")
    Else
        Sheets("BallParkFigure").Range("G6").Clear
    End If
End Sub

 

The second solution is for Forms CheckBoxes. Open the VBA editor and the code goes into a standard module like Module1 or Module2. Note after copying the code into the module, you need to return to the worksheet and Right Click the Forms CheckBox and select Assign Macro and then select the Macro name below to be run when the control is clicked.

 

The name of the sub for Forms Controls is not critical so long as when you assign the macro as described above, you select the correct sub to run.

 

Sub CheckBox2_Click()
    If ActiveSheet.CheckBoxes("Check Box 2").Value = 1 Then     '1 = Checked
        Sheets("modules").Range("B10").Copy _
                Destination:=Sheets("BallParkFigure").Range("G6")
    Else
        'Note: If not True then following If /End If statment not really required _
                   because must be False if not True.
        If ActiveSheet.CheckBoxes("Check Box 2").Value = -4146 Then   '-4146 = Unchecked
            Sheets("BallParkFigure").Range("G6").Clear
        End If
    End If
End Sub
