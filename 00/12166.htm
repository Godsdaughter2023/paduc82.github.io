don cuthbert say:

Yes, the standard Windows GPT partitioning has the Recovery partition of 500MB, 100MB System/EFI, 16MB MSR, followed by the rest of the disk as Boot/Windows (drive C:).  So an upgrade adds the new Recovery partition at the end of C: and makes that the current Recovery partition, leaving the original Recovery partition intact, but no longer in-use.  If a rollback is done, it can just switch back to the original Recovery partition at the beginning of the disk and it works as before the upgrade.

In previous Insider build cycles, I remember seeing multiple Recovery partitions.  I had at least 3 at a time after multiple upgrades from build to build.

Working with Commandbars in VBA is well documented at MSDN and at hundreds of sites on the internet.

To make a commandbar programmatically you use DIM to make it. You can also work with commandbar buttons and pictures, both in Word's interface and via VBA. I'm not going to write a full tutorial here, as they already exist in many places on line. To get started with these items

    Dim MyBar As CommandBar
    Dim MyBarButton As CommandBarButton
    Dim MyBarPic
    Dim oInsertPictureCommandBar As CommandBar

Then you make the commandbar

    Set oInsertPictureCommandBar = CommandBars.Add(Name:="InsPic5")
    oInsertPictureCommandBar.Visible = True

Then you can add a button this way

Set oInsertPictureControl = CommandBars("InsPic5").Controls _
        .Add(Type:=msoControlButton)
    With oInsertPictureControl
        .FaceId = 1106
        .OnAction = "ShowPicturePlacer"
        .TooltipText = "Insert and scale one picture"
        .Caption = "Picture from File or Network"
        .DescriptionText = "Inserts a picture and displays " & _
                           "slider tool"
        .Visible = True
    End With

or this way

    Application.CommandBars("InsPic5").Controls.Add Type:=msoControlButton, Id:= _
        111, Before:=14

You can dock a toolbar

MyBar.Position = msoBarTop

You can also put commands onto menus and contextual menus programmatically

You can put a picture onto a toolbar button by choosing any of the hundreds of built-in button faces using faceid (see example above). There are many free faceid add-ins on the internet. Some will work in Office 2011.

Or, your can paste a picture onto a button by right-clicking on a button and choosing Properties. Click the little triangle next to the existing button face and you can paste any picture (preferably one sized right for a button) from the clipboard.
