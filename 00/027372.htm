bill_in_az say:

macro to define a named range from dynamic data

I know that I can define a named range with vba, but how would I go about defining that named range if the range changes from time to time?

 

So I know this...

jeeped say:
That code does not actually define a named range. If you are looking to set Rng1 as a Range type variable then this,

Sub SetRange()
    Dim rws As Long, Rng1 As Range
    With Sheets("Sheet1")
        rws = Application.Max(.Cells(Rows.Count, "A").End(xlUp).Row, _
                              .Cells(Rows.Count, "B").End(xlUp).Row)
        Set Rng1 = .Range("A1:B" & rws)
        Debug.Print Rng1.Address(0, 0)
    End With
End Sub

If you actually want a named range, you might be better using Formulas ► Name Manager and creating the named range using this as the Refers to:,

=OFFSET(Sheet1!$A$1,0,0,MAX(IFERROR(MATCH(1E+99,Sheet1!$A:$A),0),IFERROR(MATCH("zzz",Sheet1!$A:$A),0),IFERROR(MATCH(1E+99,Sheet1!$B:$B),0),IFERROR(MATCH("zzz",Sheet1!$B:$B),0)),2)

That formula can be shortened significantly if the nature of the values is known, whether there are always values in A and B and if the ranges can be shortened from full column references. In VBA you would create it once and then use code to modify the .RefersTo parameter of the ThisWorkbook.Names object.

Edit: the OFFSET() formula translated itself. I've corrected it.

Sub macro_01()

Dim myWB As Workbook, WB As Workbook

Dim ws As Worksheet
Dim L As Long, r As Long

Dim myPath

myPath = "c:\FolderA\myfolder\"

Dim myFile
myFile = Dir(myPath & "*.xls*")
Application.ScreenUpdating = False
Set ws = Sheets.Add(before:=Sheets(1))
L = 1
Do While myFile <> ""
Set WB = Workbooks.Open(myPath & myFile)
r = WB.Sheets(shName).UsedRange.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
WB.Sheets(shName).Rows("1:" & r).Copy
myWB.Sheets(1).Cells(L, 1).PasteSpecial xlPasteValues

Application.CutCopyMode = False

L = myWB.Sheets(1).UsedRange.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row + 1
WB.Close False
myFile = Dir()
Loop
Sheets(1).UsedRange.EntireColumn.AutoFit
myWB.Save
Range("A1").Select
myWB.Save
Application.ScreenUpdating = True
On Error GoTo 0
End Sub
