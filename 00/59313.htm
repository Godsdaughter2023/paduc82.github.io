File History replicates the folder hierarchy of files which you choose to backup, at target location. This helps you quickly find files from backup locations by only identifying their location in source path.

I would like to inform you that the Drive letter will not be displayed during the process of restoring files. File history will display only the contents of the drive. This is the reason you are not able to view the drive letter.
