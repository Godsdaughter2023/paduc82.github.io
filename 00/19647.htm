How do I re-indent an entire MS Word document? - Microsoft Community

michael hemp say:

How do I re-indent an entire MS Word document?

I have been given a manuscirpt in MSWord Mac 2004 for a Kindle book. I want to change all the indents in the entire document to upload to Amazon-Kindle. Please indicate the steps on a completed doucment, not just a paragraph.

Thanks,

Michaenl Hemp
The History Company
john mcghie say:
As an alternative to what Adam offered;

1)  Find out what the problem actually is.  It may be the indents, it's more likely to be the "margins".

So first go to Format>Document and look on the Margins tab.  If the margins are not set correctly, fix them.

If the margins are set correctly, investigate the definition of the Style applied to the text.  Start with the Normal style, because every other style probably inherits its indents from there.

Use Format>Style>Normal>Modify>Paragraph and check the indents specified for Left and Right.

If you fix the Normal style, chances are the entire document will come right.  If it doesn't, you need to find which styles are applied to the text and fix those.

This is a more elegant solution than simply slapping a hard format indent over the top: it will enable you to make further editing changes throughout the document without disconcerting results...

Hope this helps
