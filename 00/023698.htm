Hi Brett,

To let ATP protect the shared mailbox, we can apply the safe attachments/links policy to the shared mailbox by following the steps below:

2. Select advanced threats-> safe attachments-> click “+” to create a new safe attachments policy.

4. Go to Applied to, and then choose The recipient is-> select the shared mailbox.
