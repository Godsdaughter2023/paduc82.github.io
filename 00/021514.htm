jeeped say:

If you can add a helper column that will fill in the blanks (which can optionally be hidden after it is created), a simply pseudo-MINIF() function can be created with MIN() and INDEX().

The formula in the helper column's F2 is,

=VLOOKUP(1E+99,$A$2:$A2,1)

Fill down as necessary. The CF rule that produced the orange highlights was created after selecting B2:D11 (with B2 as the active cell) and using the following formula,

=$D2=MIN(INDEX($D$2:$D$9999+($F$2:$F$9999<>$F2)*1E+99,,))
