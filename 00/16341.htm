rob koch say:

What you don't understand Sam is that regardless of whether any security program still works with Windows XP or not, the system can never be truly secure.

That's due to the fact that most of the critical security updates delivered to all of the other Windows operating system versions since the time WinXP ended support haven't been made available for installation on WinXP.

What this means is that no matter what the AV products which still work with WinXP attempt to do, they can't fully protect it from most current malware.  They may try to claim that they can, but in reality we've always known that the best protection comes from the operating system itself, while the security products are nothing more than a last resort when all built-in protections in the operating system have already failed.

Since "new" malware today comes at a much faster pace than in the past, it's common for security programs to miss them, which with a newer operating system might be ok if the built-in protections it contains block it anyway.  However, with Windows XP this simply won't happen, so if the security program fails the system will be infected, guaranteed.

So the cost of continuing to use Windows XP is that there's truly no security anymore and so you can't trust it for any personally critical use.  That's the simple truth and no amount of wishing, wanting or believing can change this.

Bottom line:  Windows XP is guaranteed to be insecure.  Get rid of it.

Rob
