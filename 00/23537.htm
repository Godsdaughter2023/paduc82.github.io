rena yu msft say:

Hi SamKam,

 

Refer to Self-service password reset in Azure AD deep dive, there are only ‘Notify users on password resets” and “Notify all admins when other admins reset their passwords” notifications available. The notification you require is not available. However, the users can only reset their password if they have data present in the authentication methods that the administrator has enabled and requires. The authentication methods can be:

  • Email
  • Mobile phone
  • Office phone
  • Security questions

You can set “Methods available to users” and “Number of questions required to reset” to control the SSPR. In this way, any users who doesn’t meet the requirement will not be able to reset password.
