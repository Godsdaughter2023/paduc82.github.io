How to make onscreen keyboard display and stay on top - Microsoft Community

barbaraewell say:

How to make onscreen keyboard display and stay on top

I have clicked Onscreen Keyboard in Microsoft Ease of Access.  It appears briefly on WELCOME screen, but then disappears when PC goes to main screen.  I cannot find it. 

a. user say:

Hi Barbara,

 

This could be an issue with the settings for On-Screen keyboard which you have setup on the computer. It might also be an issue with the system files.

 

Did you make any changes to the services or the settings related to the On-Screen keyboard?

 

First of all, let’s check if the “Touch keyboard and Handwriting panel service” is Started and running in Automatic mode. Follow the methods:

Method 1:

Follow the steps:

a)      Press Windows and R key and type “service.msc” without quotes.

b)      Hit Enter. Locate the “Touch keyboard and Handwriting panel service” and right click on it. Click on Properties.

c)       Check if the service is started and the startup type is Automatic. If not, make the required changes and then click on Apply. Click on OK.

d)      Restart the computer and then check if it fixes the issue.

Note: If they are already enabled, stop and restart the service. Restart the computer and check for the issue. If this does not fix the issue, follow the next method.

 

Method 2:

Let’s disable and then re-enable On-Screen keyboard. Follow the steps:

a)      Press Windows and X key together and click on Control Panel.

b)      Click on Ease of Access.

c)       Click "Use the computer without a mouse or keyboard" and uncheck the box which says "Use On-Screen keyboard" .

d)       Restart the computer and then navigate to same location to check "Use On-Screen keyboard".

e)  Restart the computer and check if it fixes the issue.

 

If the issue still persists, proceed to the next method.

 

Method 3:

Let’s run SFC scan and check if there are issues with the system files.

 

SFC Scan:

System File Checker is a utility in Microsoft Windows that allows users to scan for and restore corruptions in Windows system files.

 

Follow the steps:

a.      Type “cmd” without quotes on the Start screen.

b.       On the left pane, right click on the “Command Prompt” and select “Run as Administrator”.

c.      Type ‘sfc /scannow’ without quotes and hit Enter.

 

Please let us know about the results.
