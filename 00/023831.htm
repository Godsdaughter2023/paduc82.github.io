pablopoo---01 say:

Possible solution for &quot;Connection failed. Access denied&quot; message after Belle update

Hi,

 

If you get a message "Connection failed. Access denied" and can´t connect to any access point (wireless - mobile) the reason could be that you installed an application that use a sniffer plugin to monitor your data usage, like SPB Wireless Monitor. 

 

If you had installed SPB Wireless Monitor before belle upgrade and after the upgrade you lost the network connectivity, try this:

 

  1. close Nokia Suite (file->exit)
  2. Connect your phone to computer with USB cable in mass storage mode (Notification bar->USB->mass storage)
  3. In your computer, go to (phone drive letter):\Private and rename or delete these folders:
    1. 20042D1D
    2. 101f7989
  4. Close explorer, eject the drive and remove the USB cable.
  5. Reboot the phone.
  6. Configure your wireless network or try to connect using your mobile data plan. Should work now.
  7. Now you can open Nokia store and reinstall SPB Wireless Monitor

If you have Symbian Anna and want to update to belle, uninstall any data monitor like SPB Wireless Monitor before updating.

 

--

Pablo
