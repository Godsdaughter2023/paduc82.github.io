Importing 2007 Outlook.pst file into Outlook 2013 - Microsoft Community

m. holleran say:

Importing 2007 Outlook.pst file into Outlook 2013

I attempted to import my contacts from an Outlook.pst data file saved on a flash drive into 2013 Outlook on new computer.  Instead of importing the data file it set it up the data in another contact folder under "My Contacts."  The program will not import them directly.  When I try to import the contacts using the wizard I get a message that "this Outlook data file (.pst) is already in use in the current profile."  I can only access the contacts if I leave the USB flash drive plugged into the computer. 
karl timmermans [outlook mvp 2012-2018] say:
The program will not import them directly.  When I try to import the contacts using the wizard I get a message that "this Outlook data file (.pst) is already in use in the current profile."  I can only access the contacts if I leave the USB flash drive plugged into the computer. 


There is a difference between importing and opening a PST file. The message you've encountered means that you already have the PST file open and as a result have the contacts showing in another contact folder as well as "I can only access the contacts if I leave the USB flash drive plugged into the computer".


In short,


Simply copy the contacts from the contact folder located in the PST file residing on your USB drive and then close the USB based PST file.


There are different ways to go about this - the following is just one approach


- select(highlight) the contact folder in the USB based PST file

- use Ctrl-A to highlight all the contact items in the folder

- right click and select either (move/copy to other folder) - suggest you use "copy"

- when complete - click on the root of the USB PST file in the navigation pane --> right click --> close
