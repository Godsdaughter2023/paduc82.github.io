How do you scan downloaded files for viruses and what not. - Microsoft Community

dalilamahammer say:

How do you scan downloaded files for viruses and what not.

How do I scan a file that I have downloaded or an e-mail for viruses?

a. user say:

Most AV/AM programs have the ability to scan individual files.  Right click on the file you want to scan and see if you have options listed like "Scan with Microsoft Security Essentials" or "Scan with Malwarebytes" or whatever.  If not, the program itself (open it) probably has a custom scan option where you can select what you want to scan so just use whatever process it provides to select the file you want to scan and do so.  If that doesn't work, contact technical support for the security program you are using and ask them how to do it as each one is different (and it may or may not be possible - but chances are good it is possible for files).

Not all AV/AM programs are able to scan the e-mails themselves (individually or collectively).  E-mails are rarely a problem - it's the attachments to them or the links in them where the problem lies.  If it has it, it will be in the setup section and it may include the ability to scan an individual e-mail or it may only do all of them - perhaps as a special scan or perhaps only in real time.  It really depends on the program.  What AV/AM program are you using (name and version)?

I hope this helps.

Good luck!
