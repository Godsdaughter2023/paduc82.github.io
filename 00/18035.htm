jay freedman say:
Excellent. The error message is telling us that the ActiveDocument.FormFields collection does not contain a form field addressed by ActiveDocument.FormFields(myformfield).

This might seem puzzling, since the immediately preceding statement (before the comment) should set the variable myformfield to the bookmark name of the form field that contains the cursor, Selection.Bookmarks(1).Name. But there is one situation that can cause this: the bookmark name of that form field is an empty string.

The usual reason that a form field has an empty bookmark name is that, when setting up a form, you copy one form field and then paste copies of it elsewhere in the document. Because there can never be more than one bookmark in the same document with the same name, each form field you paste causes the bookmark name of the original or the most recent copy of it to be wiped out.

You can manually work your way through all the form fields in the document, opening the Properties dialog (by double-clicking the field) and typing in a name if the Bookmark box is empty. If there are a lot of blank-named form fields, you might want to write a macro to assign the names, in which case http://www.word.mvps.org/FAQs/MacrosVBA/AssignNameToFmFld.htm will he helpful.

joeu2004 say:

Louis wrote:

I can't seem to find the formula for the average annual appreciation for a condo purchased in 2004.               

What information are you given?

Typically, the purchase price (e.g. $100,000), an annual rate of inflation or appreciation (e.g. 2.5%), and the number of years (in this case, 2018 - 2004 = 14).  The current price might be:

=FV(2.5%, 14, 0, -100000)

Note the use of "signed cash flows" -- in this case, negative purchase price so that the current price would be positive.

The average annual appreciation is (FV(2.5%, 14, 0, -100000) - 100000) / 14.
