The formula should be

diana d say:

Hi,

 

Remove and then re-install the Active X control.

 

You can only delete ActiveX controls that you have downloaded and installed. You cannot delete ActiveX controls that were preinstalled or add-ons of any kind, but you can disable them. To delete an ActiveX control that you have installed, use the Add-on Manager. If the add-on cannot be removed in Add-on Manager, you might be able to uninstall it through Control Panel.

 

To delete ActiveX controls you have installed in Internet Explorer 8

a.    Open Internet Explorer by clicking the Start button , and then clicking Internet Explorer.

b.    Click the Tools button, and then click Manage Add-ons.

c.    Under Show, click Downloaded controls to display all ActiveX controls.

d.    Click the ActiveX control you want to delete, and then click More information.

e.    In the More Information dialog box, click Remove. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

f.     Repeat steps d and f for every control you want to delete. When you are finished, click Close.

 

 

Also refer to “How to troubleshoot script errors in Internet Explorer on Windows-based computers”: http://support.microsoft.com/kb/308260

Solved: Groove music can't find my music though file location was given before

Hey there,

I was facing problem on Groove music. I added my music file to the searching preference long ago. But recently, I noticed, whatever the new songs I was adding to the file were not actually getting added to to the library. I searched for the error and even went through the questions. But none of them worked. Then what I simply did was to remove the location and add it back again and it got fixed.

Steps:

1. Go to "Settings ", marked on the left corner below the app.

2. Click on "Choose where we look for music" under "Music on this PC" title.

3. Click on the Box titled with the file name, that contains the music to be added. This removes the location.

4. A box will pop up to confirm the removal. Confirm it by clicking on "Remove Folder".

5. Then click on the box labeled "+".

6. A window will pop up which allows to add the file that should be added in the search directory of Groove Music. Simply add the file that was recently removed.

Done!

I hope this will add the songs those were not being added. But the problem is, this problem may occur again. Just follow the simple steps above each time. Looking forward to see Microsoft solving this bug.

For more solution, go to this link: Q: The Groove Music app can't find music files.
