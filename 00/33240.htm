    mouneshwar r say:

    Hi Brooklyn,

    As per my understanding, you want to export selected contacts in a contact group. Is that right?

    This is not possible through Windows Contacts. But this is possible in Windows Mail. What you need to do is to export the contacts to a folder and then import it to Windows Mail and then through Windows Mail you can choose selected contacts to export to the desired location.

    Import, export, or change the format for contacts
    http://windows.microsoft.com/en-us/windows-vista/Import-export-or-change-the-format-for-contacts
     
    1. Open Windows Mail.
    2. In the File menu, select Export and in the submenu, select Windows Contacts…
    3. In the Export Windows Contacts window, select the vCards format and click the Export button.
    4. Next, select a folder for the exported contacts and click OK.
    5. Click OK to confirm the message informing you of the successful export, and then click Close.
    6. Now open Microsoft Outlook.
    7. In the File menu, select Import and Export…
    8. In the Import and Export Wizard, select the vCard format for import and click Next.
    9. Next, select the folder for your exported .vcf files and select the contact to import.
    10. Click Open to import the contact.

    If you want to export and import all contacts in one go, you need to use the CSV format:
    1. Open Windows Mail and in the File menu, select Export, and then Windows Contacts…
    2. Now select the CSV format as the export format, and click the Export button.
    3. Again, select the location for the export file and click Next.
    4. The next step is to select which fields you want to export.
    5. Once you have selected the fields you want to export, click the Finish button.
    6. Click OK and then click Close.
    7. Start Outlook, and select your Contacts folder.
    8. In the File menu select Import and Export…
    9. Select Import from another program or file, and click Next.
    10. Select Comma Separated Values (Windows) and click Next.
    11. Next, select the file to import the contacts from and specify how to deal with duplicate contacts.
    12. Click Next.
    13. Select the destination folder for you import, which by default will be the Contacts that you already selected, and click Next.
    14. Click Finish.
