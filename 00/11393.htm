mike h.. say:

Hi,

I don't know your date format so in this I used mine of DD/MM/YYYY if yours is different then change to MM/DD/YYYY I have included the military time format.

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
Dim ws As Worksheet
Set ws = ActiveSheet
With ws.Range("A1")
.Value = Now
.NumberFormat = "dd/mm/yyyy hh:mm"
End With
ws.Range("A2") = Environ("Username")
End Sub
