john mcghie say:

Hi Tanja:

OK, you have opened a very large can of worms here :-)

The bottom line is: "The obsolete .doc format cannot store all of the things that modern versions of Word can create; and is not as robust."

Because .doc cannot store all of the artefacts, the file is down-converted to remove the information .doc can't store whenever you save in .doc format.  You will notice this loss of data particularly in graphics.

In version 2011, Microsoft removed the software to edit vector graphics, which is possibly why your editing is not working.  When you save to .doc, the graphic is downgraded to a bitmap, and you can crop those.  My strong suggestion is to do your graphics editing in a graphics program: Word's graphics ability is not really ready for prime time yet.

Performance in .docx is not where they want it to be yet: I expect large improvements in that in the next versions.  However, one cause of "document slowness" is Tracked Changes.  Users sometimes "hide" tracked changes when they intended to remove them.  The changes continue to be tracked, the inside of the document becomes extremely complex, Word slows down, and eventually cannot open the document at all.

Go to the Review tab and "Accept all changes in document" to remove any changes that may exist.  Then save and close the document (the removal of the tracks from the file does not take place until the document is closed).

You will also find that a document containing lots of "floating" graphics will become very slow, because the code becomes very complex.  There's not a lot you can do about that, other than try to cut your graphics to size and shape before you insert them in the document.

For example: if I place a full-resolution .jpg in a Word document and size it to 10 cm x 8 cm (quarter page) Word is still storing and processing a 24 MB inclusion.  If you re-sample it to 300 dpi (the most the majority of colour printers can print) you reduce the stored image to 84 kb a reduction of 285 times. If you do this to all of your images, the document will speed up dramatically.  Keeping the total file size of the .docx below 2 MB will pay big dividends.

Of course, you can allow the .docx to grow to several gigabytes.  But you will need a MacPro to achieve acceptable handling speeds. If you resample your images down to the maximum you can actually USE, your life will improve a lot.  A typical home or office 1,200 dpi colour printer cannot achieve a colour resolution of greater than 300 dpi (max resolution divided by the four ink colours).

Hope this helps

        Else
            bool3 = Evaluate(rngR3.Cells(cc).Value & strC3)
        End If
        If rngR4.Cells(cc).Value = "" Then
            bool4 = True
        Else
            bool4 = Evaluate(rngR4.Cells(cc).Value & strC4)
        End If
        If rngR5.Cells(cc).Value = "" Then

marshall barton say:

Are we talking about the DLookup being in a text box control on a form?  Not sure what else it could be and make sense ...

If it is a form, then Refresh the form only updates the displayed records with any changes made by other users.  This is an unusual thing to do that is rarely useful.

If you want the text box to redo its expression, then you need a line of code in an event in the form:

   Me.[the text box name].Requery

Don't forget that if the reason you need to redo the DLookup is because you changed a value in a field in the form's record source table/query.  Then the changed record needs to be saved before DLookup can see it.  More details about what you are doing when you want to redo the DLookup might allow me to be more specific about exactly what is needed and where it can be done.

 

Important note:  You should NEVER use SendKeys to get Access to do something.  SendKeys is so fraught with problems that it was removed in A2010.

dirk goldgar say:

Anyhow, even though I got this to work, I would like to know why it works in some forms and it didn't work in this form that I have?  Also, what did you mean by "text boxes do not have the Recalc method"?


"Refresh" means, "save the data in the form's current record if it's dirty, then go back to the database and get the latest values for all the records displayed by the form."  Your form is unbound, so it has no records to refresh.

The form has a Recalc method, which tells it to recalculate the values of all the calculated controls on the form.  You could call this method from code behind the form by saying "Me.Recalc".  That could be used to force your DLookup() text box to recalculate, but it's overkill.  There's no such method for the text box alone, though.  In this case, telling the text box to Requery itself accomplished what you want in the most efficient way.

LAPTOP AS A ROUTER - Microsoft Community

zenog---01 say:

LAPTOP AS A ROUTER

CAN I CONNECT MY MOBILE N95 TO INTERNET WLAN USING MY LAPTOP AS A ROUTER?????

mccbleue say: No, the equipment in the phone and laptop is the same thing, both need to connect to a router, neither can function as one.

grschinon say: Actually, this isn't entirely true.

The phone certainly can function as a router for the laptop by using 3rd party software such as joikuspot, and the reverse is possible as long as you know exactly what you're doing in your network settings both on the computer and on the phone:

1) Find out the IP addresses of your ISP's nameservers. You'll need them in the next step.

2) Assign IP addresses outside your router's DHCP range (but obviously elsewhere still within the subnet) manually to the phone and to the laptop. Enter the nameserver addresses found previously in the access point settings both on the computer and on the phone. They need to be specified explicitely because you're no longer using the router's DHCP functions having assigned IP addresses manually.

3) Set the computer's default gateway address to the IP address of the router.

4) Set the phone's default gateway address to the IP address of the computer.

5) Activate IP forwarding on the computer. Microsoft used to call this "Internet Connection Sharing" back in the days of Windows 98. It might be called something different, but it all boils down to IP masquerading at the end of the day.

6) Enjoy the Internet on your phone, connecting through the laptop over the WLAN.
