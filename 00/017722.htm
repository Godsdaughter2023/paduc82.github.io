[Edit..]

Sub FolderPicker_ExportData()

'Nov 19, 2014

Dim wb1 As Workbook, wb2 As Workbook
Set wb1 = ThisWorkbook
Dim sPath As String: Dim sFile As String
Dim L As Long
With Application.FileDialog(msoFileDialogFolderPicker)
.Title = "Please select one folder"
.AllowMultiSelect = False
If .Show = True Then
sPath = .SelectedItems(1) & "\"

sFile = Dir(sPath & "*.xls*")
If sFile <> "" Then

Application.ScreenUpdating = False
L = 1
Set ws = wb1.Sheets.Add(before:=wb1.Sheets(1))
Do Until sFile = ""
Set wb2 = Workbooks.Open(sPath & sFile)
For x = 1 To wb2.Sheets.Count
ws.Cells(L, "A").Value = wb2.Sheets(x).Range("I3").Value
ws.Cells(L, "B").Value = wb2.Sheets(x).Range("J3").Value
ws.Cells(L, "C").Value = wb2.Sheets(x).Range("L3").Value
L = L + 1
Next
wb2.Close False
sFile = Dir()
Loop

ws.Range("E1") = sPath   '<< folder's path in cell E1

Application.ScreenUpdating = True

Else
MsgBox "no files found"
End If
Else
MsgBox "Cancel"
End If
End With
ActiveWorkbook.Save
End Sub

scott .. b say:

Depending on the version of Windows 8 Pro you have, e.g. Desktop, Tablet, etc.  you may be able to restore your computer to its original factory default settings by doing a system REFRESH.

 

Internet Explorer 10 is part of the Windows 8 operating system.  The only way you can re-install it is by reloading Windows 8.  This is possible, through something called REFRESH, which is a WIN 8 option that saves your personal files (documents, pictures, videos, Windows Store apps) and uninstalls everything else. Then it reinstalls Windows 8, updates your system, and reinstalls your personal files.  It will also reinstall a clean copy of Internet Explorer 10.  Essentially, REFRESH puts your computer back the way it was when you bought it.

 

I don't know how many programs you may have bought, downloaded, or whatever, but if you do a REFRESH it will uninstall all of them and you will have to reinstall them one at a time.  It will leave a file on your Desktop telling you which programs it uninstalled, so you can reinstall them.  NOTE:  You may need the original discs or product code to reinstall some applications, so before doing a REFRESH, be sure you can do that.  Also , if you bought your own antivirus software (other than that which came with your computer) you will need to contact your Antivirus software provider and request a temporary deactivation of the software, so that when you go to reinstall it, you won't have to buy another license.  REFRESH will not remove any pre-loaded programs that came with your computer (MS Office etc,);  only those that you have added since buying your computer.  When you re-install your programs, it is a good idea to install only one and then check your computer's operation to see if your original problem reappears.  Then move to the next one, etc, then to peripherals.  You never really know where the culprit is but, as long as you're starting over, it is a good time to eliminate potential problem causers, one by one.

 

You should write down the website address of all sites you have Pinned to Start, because they will be gone and you will have to go back to the site and pin them again.

 

When REFRESH is finished, it will automatically start applying all of the Windows updates that have accumulated over the past few months.  This will take awhile, but what you have pre-installed on your hard drive needs to be updated.  While REFRESH is running, DO NO TURN OFF YOUR COMPUTER!  It may seem like nothing is happening at times.....just wait.  You don't have to do ANYTHING!.  When REFRESH has finished it will return you to the Start screen.

 

 

To make it happen..................

At the Start screen type REFRESH

Select "Settings"

Select "Refresh your PC"
