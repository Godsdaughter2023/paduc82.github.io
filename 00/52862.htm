Casey, please ignore the bad information supplied by the previous poster.

It depends on how Windows 10 was originally configured on your computer.

If its a full retail store bought license online or offline, it is transferable to a new computer or motherboard.

If its the free upgrade from a retail store bought Windows 7 or Windows 8 license, it is transferable to a new computer or motherboard.

If its the free upgrade from a OEM preinstalled Windows 7 or Windows 8 license, it is NOT transferable to a new computer or motherboard.

http://www.groovypost.com/howto/transfer-window...

http://www.groovypost.com/howto/activate-window...

Microsoft's view is, as long as you are not abusing the license, you should get a pass.

In your case in particular, you have a digital license which should allow you to reactivate on the new machine since it originated from a retail license.

This will reduce the amount of memory you have for taking new pictures and, eventually, you will run out of memory and you will need to copy the pictures to another storage device.

 

If you are just using the computers memory then the camera should have a USB lead which allows you to fasten one end to the camera and the other end to your PC's USB port to allow you to transfer data between the two.

 

I am, however, assuming that your PC has an SD Memory card which stores the photographs.
