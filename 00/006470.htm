Hi,

 

This snippet will find the last used row and which column it's in

 

Dim LastRow As Long, WhichCol As Long
Dim Rng As Range
Set Rng = Columns("A:B")

LastRow = Rng.Cells.Find(What:="*", After:=[A1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row


WhichCol = Rng.Cells.Find(What:="*", After:=[A1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Column
