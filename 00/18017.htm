chrismccready say:

Automatically calculate the sum of data separated by spaces in a single column with the sums in the blank spaces?

Is there a way to calculate the sum of a set of data separated by spaces in a very long list of data in a single column?

I have a large data set in a single column, but need totals in the areas where the data ends (the blank space/cell). Is there a way to have excel calculate the long columns and output the totals with out having to go to each end of the data and autosum?

bernard liengme say:

Here is a VBA solution

best wishes

Sub tryme()
   lastRow = Cells(Rows.Count, "A").End(xlUp).Row
   For k = 1 To lastRow
    If Cells(k, "A") <> "" Then
      Subtotal = Subtotal + Cells(k, "B")
    Else
      Cells(k, "B") = Subtotal
      Subtotal = 0
    End If
   Next k
   Cells(lastRow + 1, "B") = Subtotal
End Sub
