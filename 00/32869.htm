The Group Policy Client service failed the logon, Access denied - Microsoft Community

marcolr say:

The Group Policy Client service failed the logon, Access denied

Hello,

I just had this issue, as soon as I switched on my Win 8.1 PC this morning. It's always up-to-date, including all Windows Updates. Let me say that I haven't installed any new software/hardware or anything like that.

First of all I'd like to know what might have caused this issue, please.

Also, I found a solution without needing to reinstall, which in my case was not viable, in fact I would have had to spend long hours doing so.

The solution I found on the Web site of an important Italian PC magazine implies editing the registry, so of course I am not suggesting you to do that unless you know what you're doing, so do it at your own risk ;-)

This is the solution (it is in Italian, sorry...):

http://www.pcprofessionale.it/2014/12/09/come-risolvere-il-problema-del-client-di-criteri-di-gruppo/

I am providing a rough translation, I don't have the exact Windows terminoliogy in English since my Windows is in Italian:

1) Launch Regedit and go to HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services, check if you have the Gpsvc key. If it is there you can proceed. If not it means you have to reinstall/repair Windows.

Thanks goodness it was there for me :-)

2) FInd HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\SVCHOST. You should have a key here called GPSvcGroup, if it is not there you can create a new one and assign to it the value GPSvc.

3) Create two dword values in Gpsvc and name them AuthenticationCapabilities and CoInitializeSecurityParam. Then assign the following value to AuthenticationCapabilities: 0×00003020

and the following value to CoInitializeSecurityParam: 0×00000001

Close everything and reboot. Once rebooted, first of all check if the gpsvc Service is running. If it is, then you should be ready to go.


In my case, this solved my issues of a basically unusable PC, with core programs such as the whole Office 365 not working at all.

I hope this can be useful to someone else.

Once again, it's always very scary that Windows does such things without any reason... It's not the first time that this happens, you don't do anything wrong and then maybe to some Windows Update or similar your PC stops working.

This is very bad.

Regards,

Marco

eryksun say:
Is there a way to make the mouse slower during the logon screen in Windows 7, because now it's difficult to choose an account.

I've changed the mouse settings in all accounts, but the mouse is still very fast before choosing an account.

Thanks.
I had to figure this out on Vista because the pointer stick of my cheap wireless keyboard is painfully slow at 1920x1080 screen resolution using the default settings. Hopefully this will work on Windows 7.

First run regedit and verify that you have the following two registry keys:
  • HKEY_CURRENT_USER\Control Panel\Mouse
  • HKEY_USERS\.DEFAULT\Control Panel\Mouse
They should have entries such as MouseSpeed, MouseSensitivity, etc. If these are present, then Windows 7 probably hasn't changed this configuration.

The logon settings are in the key for the .DEFAULT account. You can directly edit them, but I think it's simpler to configure the mouse using the control panel interface in your current account and then apply the settings to the default account all at once (as opposed to figuring out the range of sane values for directly editing the registry keys).
  1. Once you have the settings the way you want them for your CURRENT_USER, go back to regedit and right-click the "Mouse" key for the CURRENT_USER in the left-hand pane. Choose export. Save it as mouse.reg someplace convenient like the desktop.
  2. [Optional but Recommended ]  Also export the "Mouse" key for the .DEFAULT account in case you need to restore your old settings.
  3. Right-click mouse.reg and choose "edit". Replace [HKEY_CURRENT_USER \Control Panel\Mouse] with [HKEY_USERS\.DEFAULT \Control Panel\Mouse].  Save the file.
  4. Double-click mouse.reg to import the new settings into the registry.
Hopefully this will either work fine or someone can hack it for Windows 7.
