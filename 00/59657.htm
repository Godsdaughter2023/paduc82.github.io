john john __ say: You are trying to install on a SATA drive or your drives are on an add-in controller card and the Windows XP x64 setup program doesn't have the required driver for the controller so it cannot detect the drive.

You have three options:

1-  Use the F6 method to supply the mass storage drivers to the Windows XP x64 setup program.  The drivers the setup program will not accept the drivers from any other media source.

2-  Slipstream the drivers into your Windows XP x64 CD.  Note that you will find reference to slipstreaming SP3 to your Windows CD which
3-  (For on-board SATA controllers) Go in the BIOS and put the controller in IDE mode, most motherboard with onboard SATA controllers have an option to do this.  When in IDE mode the Windows XP setup program will see the drive and you will be able to install Windows.  After Windows is installed you will be able to install the SATA drivers from within the up and running Windows XP installation.  You need to install the SATA drivers before you change the controller back to SATA mode in the BIOS or else when booting Windows will throw up a BSOD with an 0x7b error.

Windows Vista is Homogeneous Multi-adapter system. The term "homogeneous multi-adapter" is used to refer to cases when more than one graphics adapter is in use but all adapters use the same graphics driver. As in your case two identical cards from the same graphics hardware vendor. So if your driver is working fine on one, its should work fine on another.

 

Windows Vista does not support "heterogeneous multi-adapter" multi-monitor implementations. The term "heterogeneous multi-adapter" is used to refer to multiple graphics adapters using multiple graphics drivers in a single PC. A common example is the use of graphics adapters from two different manufacturers, each of which requires a different graphics driver from the respective manufacturer.

In a system configured with heterogeneous graphics adapters (that is, adapters that use different drivers), Windows Vista will disable one of the two adapters and present an error message:


Incompatible display adapter has been disabled.
At least one display adapter on the system has been disabled becuase its driver is not compatile with the driver for the VGA adapter.

 

Device Manager will show the disabled device with an Error Code 43. Event Log Viewer will also have a message logged that contains the same text as in the error message displayed.

As if your other card is being seen not identical with your other card.
