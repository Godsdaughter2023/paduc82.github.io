That is only possible using a macro, copy the code below into a regular module.

Select the data to copy and execute the macro.

Note:

To get all data from the current region select only one cell inside the data. The macro enlarges the selection automatically.

It is not possible to get the cell formats, we can copy the data only.

Andreas.

Sub MySpecialCopy()
  Dim Data
  Dim R As Range
  Set R = Selection
  If R.Areas.Count = 1 And R.Rows.Count = 1 And R.Columns.Count = 1 Then
    Set R = R.CurrentRegion
    R.Select
  End If
  Data = R.Value
  If Not IsArray(Data) Then
    ReDim Data(1 To 1, 1 To 1)
    Data(1, 1) = R.Value
  End If
  Set R = Nothing
  On Error Resume Next
  Set R = Application.InputBox("Select the cell to paste", Type:=8)
  If R Is Nothing Then Exit Sub
  With R.Cells(1, 1).Resize(UBound(Data), UBound(Data, 2))
    .Value = Data
    .Parent.Select
    .Select
  End With
End Sub
