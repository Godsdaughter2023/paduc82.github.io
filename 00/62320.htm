canadian tech say:

If you have system file corruption, you are likely at the end of the rope- so to speak.  Try running SFC again.  Sometimes a 2nd or even 3rd run corrects the corruption.

Try System Restore.  It may fix your problem if it has not been there too long.  Usually it is best to do the restore starting from Safe Mode (F8 at startup)

Much more likely you will need to re-build.  Choices: 

1. restore from factory partition

2. full re-install from a Win7 disk

3. In place non-destructive install

If I were in your shoes, I would re-install Windows 7.  It is a time-consuming process -- likely 6 to12 hours elapsed time depending on the speed of your Internet connection and your computer and how much memory it has.  This kind of a process will lead you to a purely clean and absolutely virgin system that will have dramatically improved performance.

First backup your data, but do one more thing.  Use Windows Easy Transfer to create a file for later transfer. Use a large external hard drive for this.

Now find the drive tester software from the manufacturer of your hard drive.  Run all three tests to ensure you have a reliable hard drive before you begin this long process.  This may take a couple of hours, but I would not do this until this is done and you are confident.  If necessary replace the hard drive -- not a difficult or particularly expensive process.

Choice 1: If your computer came with a factory restore partition, restore that.  Find out how to access the factory restore partition.  It is likely to be found by using F8 during startup of the existing system. 

Depending on what you bought, you will then have to a lot of un-installing of un-desirable programs.  Do not let an antivirus or internet security system install until you are at the end of this process - the very last thing you do.  Also do not put a password on your computer until you are absolutely done.  If antivirus or Internet security is installed in this choice 1, uninstall it.  Now startup Windows Update, but first change it so that it will give you all Microsoft updates, not just Windows ones. There will likely be hundreds of them and it will take hours, but you do not have to watch it.  There will be multiple rounds of updates.  Keep Checking for new updates until there are none. 

Now install your applications, but only the stable unchanging ones, e.g. Microsoft Office.  You may have a couple more rounds of updates after this.  Now do a defragmentation.  No data yet. 

Next take a step almost no one does but should you ever be back in a similar situation again, you will thank yourself for  having done it.  Create a system image using Maintenance, Back up Restore.  Use DVD's -- from 2 to 4.  It will take close to an hour, but when you are done, you will have a set of DVD's that can be used to restore the system to the same or a new drive that is precisely what you have so far.  While you are at it spend 5 more minutes and create a Windows7 System Repair CD.

Of course after this, you can add your volatile applications like Flash Player and the like.  Then run Windows Easy Transfer again to get all your data and settings back.  It is positively amazing how well this works, you will find you will not see a difference from before except that it will be stunningly faster.

Choice 2: I actually prefer this option. Use an original Microsoft Windows 7 DVD, preferably Service Pack 1. It can be any Windows 7 DVD of the same product variety -- e.g. Home Edition and either 64 or 32 bit depending on your system.   Make certain you have the Microsoft Windows Activation key written down.   The only downside to this option is that it is possible but not likely that you may have to find a driver.  This does not happen often because even the drivers that the Windows 7 install process itself does not provide, will most likely be provided in your first Windows Update pass. Once you get Windows install completed, everything else above will be the same.

Choice 3:  Follow this link for a good description of how to do this

http://windowssecrets.com/top-story/win7s-no-reformat-nondestructive-reinstall/   This choice will require extensive Windows Updates time as well.

Notes:

I will not work on a computer with ANY Internet security product on it.  All you want or need is an antivirus product.  The other stuff that is part of their packages will only cause you problems. 

Intel has a driver update utility (assuming yours is Intel based) that is absolutely excellent.

You do not need a disk burner program.  Windows has it built in.

After you are done create a restore point.

I am a long-term systems professional who looks after 160 client "consumer" type computers.

"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it.
