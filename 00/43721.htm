davem121 say:
OK, this is a common enough problem . . .

On your D Drive, create a new folder, name it Desktop
IMPORTANT - Move all the files and folders that should be on your Desktop into that new folder
Click on Quick Access in the left pane of Windows File Explorer
Then click on your Desktop in Quick Access

IMPORTANT - when asked if you want to merge files and folders - CHOOSE NO !

Now that folder (D:\Desktop) will be your Desktop Folder

After a restart you should be able to move your Desktop folder to its default back on your C drive or leave it in the new folder . . .

marilyn o say:

If you have a System Restore point to before the issue was occurring, then do a System Restore.

To choose a System Restore point,

·         From the Desktop click the Pearl (start) button
·         In the Search Field type “System Restore” and hit enter
·         Once System Restore is open, select the option “choose a different restore point” and hit next

Once on the next screen, pick a restore point to go back to when it last worked for you. Once you select the restore point, click next, then select finish, this will start the restore process.

a. user say:

Hi Dorothy,

Startup sound can be the same with Windows logon sound, since the setting for both are on the same place. However, Windows logon sound is hidden as default, and needs to be reveal for it to work. That said, modifying the Windows Registry is part of the steps. Here's what you need to do in the Windows Registry:

  1. Press the Windows key + R on your keyboard, type in "regedit", then hit the Enter key.
  2. Navigate to this folder "HKEY_CURRENT_USER\AppEvents\EventLabels\WindowsLogon"
  3. Double-click the ExcludeFromCPL, then change it from (1) to (0).
  4. Close the Windows Registry, then open up the Windows Sounds Dialogue.
  5. At the Sound tab, then look for Windows Logon under Program Events.
  6. Click on Browse, then choose your preferred sound.
  7. Make sure to put a check on "Play Windows Startup sound".
  8. Hit Apply, then OK.

Close everything on your computer, then restart it. Once done, check if the issue persists.

Disclaimer: Serious problems might occur if you modify the Windows Registry incorrectly. Make sure to create a backup of the Windows Registry before modifying it. Check this article to learn how to create a backup of the Windows Registry.
