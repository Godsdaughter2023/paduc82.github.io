Hi Bob, if you want to remove that screen altogether and just go straight to the Login Screen:

Click your Start Button, type regedit and hit Enter to open the Registry Editor

In the left pane, navigate to:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

Right click the Windows Key (folder) and choose New - Key

Name the key Personalization

Click the new Personalization key

In the right hand pane, right click and choose New - DWORD 32bit

Name it NoLockScreen

Double click NoLockScreen and set its value to 1

Close the Registry Editor and Restart Windows for the change to take effect

That's It!

Dean

hairyfool say:

It says exactly what it means. Windows Setup will not run if it detects that the laptop is running on the battery and the AC is not plugged in.

If you are actually trying to run setup with AC on then ACPI power management is not indicating this to Windows Setup properly.

Check that the BIOS is correctly configured, that you have the latest BIOS and if necessary reset the BIOS back to default settings. It may however be indicating a problem with the laptop.

You could try running setup with no battery fitted.

hairyfool say:

Most laptops have two separate power circuits when on AC, one to charge the battery and one to run the system board. It rather looks as if that second circuit has failed, if the laptop power pin is multi pole then it could be the power supply but this is rare.

Check the manufacturers label of the PSU, if it indicates two voltages or two current ratings it might just be worth trying another power supply.
