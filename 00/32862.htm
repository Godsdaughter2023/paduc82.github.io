Slow restart windows 10 solved - Microsoft Community

servanschreiber say:

Slow restart windows 10 solved

Hi Guys,

After a long search on te web I finally found a sollution to my problem with the restart issue.

If shutdown works fine but the restart takes like minutes or hours or forever the following steps can work:

1. start regedit in administrator modus.

2. find registry key : Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control

3. set value WaitToKillServiceTimeout to 1000

4. find registry key : Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management

5. set value ClearPageFileAtShutdown to 0

6. close regedit and restart

What was forever became immediatly

Hope it works for you

There is no Active partition on a UEFI install which uses an EFI System partition to boot rather than the Legacy MBR System Reserved.

how can i backup my drives incase anything goes wrong? - Microsoft Community

mohitsaxena say:

how can i backup my drives incase anything goes wrong?

my laptop came with 1 TB hard disk which was all in C drive but i needed one more so i partitioned it and made another one of 500 GB storage and named it F. I store all my stuff in it like videos, important softwares, memorable pics, movies, etc. so is there any way i could back all this up internally in this computer so that if anything ever goes wrong and i had to reinstall windows so that i could get all that stuff back as it is right now?

a. user say:

There two ways to backup: You can backup just files and folders and you can backup an entire partition.

Backing up files and folders is as easy as copying from one place and pasting in another place. With backup software this can be almost effortless, and you can schedule automatic backups to run daily.

To copy an entire partition you need disk imaging software. It takes a bit of work to learn how to do this, but your efforts will be amply rewarded. With a disk image backup you will never need to reinstall Windows or your software, just restore your image.

Two suggestions:

With all deference to Microsoft, you will find backing up and imaging much more convenient and easy if you use third-party programs. Windows' in-built backup capability is limited and awkward.

You are best advised to store your backups outside your computer. An external USB drive or 'the cloud' make good destinations. That way, even if your computer is lost or physically damaged, you haven't lost anything.
