jay freedman say:

The "No Spacing" style (with a space in its name) is a built-in style, not a custom style that you created.

Word won't allow any built-in styles to be deleted. If the built-in style has been modified, the attempt to delete it will return it to its default definition.

If you have two custom styles, with Style 2 based on Style 1, you can delete Style 1 from the document. Then Style 2 will automatically change to being based on the Normal style, with the addition of any formatting that used to be part of Style 1. In other words, having another style based on Style 1 does not prevent the deletion of Style 1, as long as it isn't a built-in style.

When you want to find out whether a style is built-in or custom, click the Manage Styles button at the bottom of the Styles pane; click the dialog's Recommend tab; and click the Select Built-In button. If the style you're looking for is in the selected group, it's built-in; otherwise it's custom.

graham mayor say:

Enhanced Metafile is essentially a graphical representation of the table, and what you see is what it does.  If copying and pasting tables between two Word documents then why not simply copy (Ctrl+C) and paste (Ctrl+V) the table (as a table)? Any differences in presentation will then be solely due to the margins in the two documents and the styles used to format them and the table remains a table.

If the new document revolves around the table, then create a new document using the original document with the table as a template. The new document will then be identical to the original, complete with styles and margins and you can replace the stuff that is not the table with the new associated text.

I am not good at VBA, so I do not want to create a UserForm or a Module. Is there a way to do this from the "Record a Macro" tab?
lz. say:

Hi

Assuming your table is named Table1 and is in Sheet1 (replace below where appropriate) use the following code for your New Record button;

Sub AddRowToEndOfTableAndMoveThere()

    Dim mySheet As Worksheet
    Dim myTable As ListObject
   
    Set mySheet = ThisWorkbook.Worksheets("Sheet1")
    Set myTable = mySheet.ListObjects("Table1")
   
    With myTable
        .ListRows.Add
        Application.Goto Cells(.ListRows.Count + .HeaderRowRange.Row, .DataBodyRange.Column)
    End With
   
End Sub
