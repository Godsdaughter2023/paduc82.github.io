ksmith2---01 say: Hello,

I also just purchased the Mybat USB cable on ebay, model is NK-6030. It works perfectly with the Nokia 6061 phone. Note that the only software that supports 6061 is Oxygen Mobile Phone Manager, which has trial version for download at www.oxygensoftware.com I tried MobiMB and Nokia PC Suite, neither of them detected the 6061

The trial version of Oxygen does NOT let you write to the phone, but it did find the phone as RH-74 (nokia's internal model number) and downloaded a backup of all the data. You can buy a "lite" license for $20 Euro.

When installing the Oxygen software, it does not auto-detect the cable, but after the manager is installed, you can manually add a new connection type, then select the Prolific PL-2303 virtual COM port, and it will label it as cable type DAU-9p. The default and recommended baud rate will be 115200 N-8-1. On my system it appeared as COM3.

Note that when connecting the MyBat cable to the bottom of the 6061 or 6030 nokia phone, do not insert it all the way. Only insert the cable about halfway into the phone. That will make better contact with the leads on the phone.

Note that when you install the USB cable, MyBat NK-6030, you must FIRST install the driver from the CD, THEN install the hardware. If you have done the reverse, you must delete the driver from the Device manager, under Control Panel / Hardware Tab / Device Manager button... Then go to Ports (COM & LPT). You will ONLY see the entry when the device is installed. Make sure the USB cable is installed, then delete the bad driver. Then remove the cable, install the real driver from the CD, then install the cable and it should detect and install the correct driver for you automatically.

If you are having trouble finding drivers for the cable (PL-2303), try the Prolific web site at www.prolific.com.tw/eng/Products.asp?ID=59

Hope this helps, and spread the word. The MyBat + Oxygen + 6061 is a great combination... no need to re-enter 200 names manually in the phone book!

Keith.
