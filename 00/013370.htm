rik_uk say:

Hi Ted

This is a UDF written to return the RGB of a given cell, so place it in a standard module of the wb and it can be used as a formula

Function getRGB(RefCell)
Dim mystr As String
Application.Volatile
    mystr = Right("000000" & Hex(RefCell.Interior.Color), 6)
    getRGB = Application.Hex2Dec(Right(mystr, 2)) & ", " & _
             Application.Hex2Dec(Mid(mystr, 3, 2)) & ", " & _
             Application.Hex2Dec(Left(mystr, 2))
End Function

So if cell A1 has a fill of red and in B2 you type  =getRGB(A1), then the result is  255, 0, 0
