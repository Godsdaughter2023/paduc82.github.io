a. user say:

Hi Naoki,

You might be experiencing this issue because of a configuration error or the current network driver is not playing well with your current Windows build. If you were connected before and recently installed a new network adapter driver, rolling back your driver to a previous version might help. Here's how:

  1. In the search box on the taskbar, type Device Manager, and then select Device Manager from the list of results.
  2. In Device Manager, select Network adapters > the network adapter name.
  3. Press and hold (or right-click) the network adapter, and then select Properties.
  4. In Properties, select the Driver tab, select Roll Back Driver, then follow the steps. If the button is unavailable, that means there's no driver to roll back to.
  5. After rolling back to the previous version of the driver, select the Start button > Power > Restart if you're asked to restart, and see if that fixes the connection issue.

It is possible that you are using an incorrect network adapter driver. If you still have the CD or DVD that came with your computer, we suggest that you install the driver from there. You can also try looking for one on the device manufacturer's website and follow their instructions.

We also recommend that you download and install any pending updates that need to be installed on the computer through Windows Update and check if it helps resolve the issue. If you are currently using Windows 10 version 1709, the latest Windows update is KB4089848 (OS Build 16299.334).

For additional troubleshooting steps that you can try, you may refer to the following links:
