lemp say:
In general, wireless keyboards and wireless mice are uniquely paired with the wireless transceiver that comes with them. Otherwise, people using the same model wireless keyboards within say 20 to 30 feet of each other would interfere with each other.

Some older versions of these wireless devices had "connect" buttons (on both the transceiver and the keyboard/mouse) that you had to press to sync with the transceiver, but most do not.

Unfortunately, for wireless devices that don't have "connect" buttons, you need the transceiver that was packaged with the keyboard or mouse. If the manufacturer won't supply you with a replacement given the part's serial number, there's no practical way for you to get one.

I hope you didn't pay too much for the keyboard.

vijay a. verma say:
Try following and check after each

1. I am assuming that in File Explorer > View > Extra large icons. large icons or medium icons - One of them is checked, As these are only views which would show a thumbnail of picture.

2. Open File Explorer > View tab > Options > View tab > Uncheck Always show icons, never thumbnails (If this is already unchecked, check it and OK..Then go back again and uncheck it and OK

3. Type control in Windows Start Search box > Click Control Panel > Type system in CP search box > Click View advanced system settings > Click Settings under Performance > Check Show thumbnails instead of icons

Source - https://answers.microsoft.com/thread/f0c429b7-d...
