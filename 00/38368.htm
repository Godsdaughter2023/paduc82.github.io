mouneshwar r say:

Hi Lori,

 

Step 1: Check if you can delete the fonts from safe mode

a.    Use H8 while restarting the computer.

b.    Under Windows Advanced Boot Options menu, select Safe mode and press Enter.

c.    Once you login to Safe mode, try deleting the fonts to see if you can delete them successfully.

 

Step 2: Turn off User Account Control

You can try turning off User account control in normal windows mode and then check if you’re able to delete the fonts successfully. Visit the link below to know how to turn user account control on or off.

http://windows.microsoft.com/en-US/windows-vista/Turn-User-Account-Control-on-or-off

Note: Microsoft wouldn’t recommend turning off UAC. Please do turn on once you’re done with troubleshooting.

 

Step 3: To take the ownership of C: drive

a.    Logon to your computer as administrator.

b.    Click on Start and then on Computer.

c.    Right click on C: drive and choose properties.

d.    Click on the Security tab.

e.    Click on the Advanced button and click on the Owner tab.

f.     Change the current owner to 'Administrators' (not 'Administrator' - not the 's') you will be changing the owner to the 'Administrators' group (not the user 'Administrator').

g.    Click on Apply and Ok until you exit out of all the windows.

 

Step 4: To take ownership of Windows Font folder

a.       Right-click on the Windows Font folder (Location - C:\Windows\Fonts), and then clickProperties.

b.       Click theSecurity tab, click Advanced, and then click theOwner tab.

c.       ClickEdit. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

d.       Click the name of the person you want to give ownership to.

e.       If you want that person to be the owner of files and subfolders in this folder, select theReplace owner on subcontainers and objects check box.

f.        ClickOK.

 

Now try deleting the fonts and check if you’re able to delete.

 

Hope this information is helpful.

How to uninstall Windows PowerShell

To uninstall Windows PowerShell 1.0 in Windows Vista, follow these steps:

  1. Click Start, in the Start Search box, type appwiz.cpl, and then press ENTER.
  2. In the list of tasks, click View Installed Updates.
  3. In the Uninstall an update list, right-click Windows PowerShell(TM) 1.0 (KB928439), click Uninstall, and then follow the instructions to remove Windows PowerShell 1.0.

To disable Windows PowerShell 1.0, follow these steps:

  1. Click Start, in the Start Search box, type appwiz.cpl, and then press ENTER.
  2. In the list of tasks, click Turn Windows features on or off.
  3. Click to clear the check box for Windows PowerShell.
