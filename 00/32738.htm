Ignore the squares,  remove them if you like or change them to things that you would find useful.   The only thing that is really different is that when Windows first starts up, in W7 terms it is as if the Start Menu was opened automatically and spread all over the Desktop.   To dismiss that you would press Win- or Ctrl-Esc, or just ignore it and open the Run... dialog (e.g. press Win-R).   But in W7 you would also probably often not see what you wanted to use immediately and then have to open the Start Menu and start typing.   So, in W8 with the Start Screen already opened you can start typing.   Just imagine that you have an invisible input box to type your first letter in--which is true--as soon as you start typing you will see its Search bar.   Another difference would be that in W7 you would be able to see your Taskbar.    That shows up as soon as you dismiss the Start Screen.   In fact, as a reminder you could leave the Desktop tile in position one and then "launch" the Taskbar that way simply by pressing Enter (since the first tile on the Start Screen has keyboard focus when Windows starts up--but, again, you may find that you have a more useful default than that for that feature.   ;  )

 

Everything else, e.g. on the Taskbar is essentially the same, so I don't understand your comment about minimizing programs?   "How to open new apps"  may need some customizing of the Start Menu or the Search, depending on how the "new apps" were installed.
