Here's how to drastically cut upgrade time with no need for an SSD - Microsoft Community

hula_baloo say:

Here's how to drastically cut upgrade time with no need for an SSD

I've found the answer!

 The first time I upgraded from build 10041 to 10049 it took me a LLLLOOOONNNNGGGG 10 hours.  Since then I've read a number of posts suggesting that if you disconnect or otherwise disable your Internet connection, you can significantly reduce this time.  So yesterday I deleted my Windows 10 build 10049, installed 10041 from a DVD I made from the ISO, and tried it.  It didn't make any difference.  So today I decided to try it again, but this time I disconnected the network cable after the download was complete but BEFORE it rebooted.  VOILA!!  From the "Preparing to Install" message until I was up and running with 10049 took all of 62 minutes.  That's a 10:1 performance improvement just by disconnecting the network cable at the right time!  And I was doing all this on an ordinary several years old 320 GB SATA hard drive.  I have 6 GB RAM and an Intel E7500 2.93 GHz dual core CPU.
`

 So let me just sum it up.
`

1) Start the upgrade via the usual Settings (fast track) update.
`

2) Wait for the download to get to 100% and give the message "Preparing to install"
`

3) At this point you'll have roughly 10 minutes before the computer will reboot.  This is the time to unplug the network cable (or disable your WiFi connection).

greg wilson 2 say:

Nimalan:

The problem is the LockAspectRatio property. By default it is set to True. You therefore cannot change the picture's proportions. If you attempt to do so then the opposite dimension automatically adjusts to maintain the original proportionality (aspect ratio).  If you set this property to False and change the proportions you distort the image. 

I suggest that, immediately after inserting the picture, you test for the picture's aspect ratio first (it will differ depending on the camera) by dividing the height by the width. Then adjust the proportionality of the cell range instead so that it equates with that of the picture. Then size the picture and insert it into the cell range.

Shown below is code that disabled the LockAspectRatio property and adjusted the picture to exactly fit the cell range. As mentioned, it will distort the picture unless the cell range proportionality matches the picture's aspect ratio.

Sub x()
Dim pth As String, fnm As String
Dim apicture As Excel.Picture
Dim mycell As Excel.Range

pth = "C:\Users\Greg Wilson\Desktop\xlPhotoMap\xlPhotoMap Photos"
fnm = "Capture_00001.JPG"

With ActiveSheet
    Set mycell = .Range("J5:S26")
    Set apicture = .Pictures.Insert(pth & "\" & fnm)
End With

With mycell
    apicture.ShapeRange.LockAspectRatio = False
    apicture.Top = .Top
    apicture.Left = .Left
    apicture.Width = .Width
    apicture.Height = .Height
    apicture.Placement = xlMoveAndSize
End With

End Sub
