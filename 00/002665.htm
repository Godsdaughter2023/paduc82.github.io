danja9191 say:

OK, figured it out. Somehow it eluded me during my first google search. Since someone already had a "Me Too" to this question, here's how you get text to align with the rule lines.


1. Close OneNote


2. Open the registry editor (type regedit into the Windows 8 start menu)


3. Navigate to: HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\OneNote\Options\Editing. Double click on "DefaultFontSize" and set the value to whatever you want. For Verdana font it's 19.5 to align with the rules.


4. Start OneNote


You'll notice at the very beginning the font is set to something else because the selected field is for the section header, which apparently uses a different default value. As soon as you actually click on the page and start typing, it will change to the value you set in the registry.


You may notice that your text is not directly on the lines. You can fix this by grabbing the top of the text box while holding Alt (to disable snapping to the grid) and dragging it such that the text you typed in the first line is correctly positioned.

I think I've figured out what the issue was.

We all probably know that Microsoft took off its links to build 10240. Apparantly, if you don't have the option to download from other PCs on the internet and the local network enabled under "Choose how you download updates" under "Advanced Options" the updates app won't see any 10240 update.

Basically, just turning back on the downloading from other computers on the internet will likely fix your problem as well.
