How do I export Office365 distribution groups and import it to another office365 Tenant

How do I export Office365 distribution groups and import it to another office365

Get-Group | Export-Csv C:\Group.csv ??

anna ma msft say:

Hi Michael Amaro,

May I know are you trying to export all distribution groups and their attributes, or export the specific distribution group members?

Generally, to export Office 365 distribution groups, you may try the following steps:

  1. Connect to Exchange Online PowerShell.
  2. Run the PowerShell command to export all distribution groups and their attributes : Get-DistributionGroup -RecipientTypeDetails MailUniversalDistributionGroup | Export-Csv -Path “C:\Users\Administrator\Desktop\Example01.csv"  For your reference: https://docs.microsoft.com/en-us/powershell/module/exchange/users-and-groups/get-distributiongroup?view=exchange-ps .

To import csv to create the new distribution groups:

  1. Connected to Exchange online PowerShell.
  2. Run the command: Import-Csv -Path "C:\Users\Administrator\Desktop\Example01.csv" | ForEach-Object {New-DistributionGroup -Name $_.Name  -DisplayName $_.DisplayName -Alias $_.Alias}
  3. For your reference: https://docs.microsoft.com/en-us/powershell/module/exchange/users-and-groups/new-distributiongroup?view=exchange-ps .
  4. Please note that if you want set Primary smtp address directly in PowerShell, make sure that the custom domain of each groups in exported csv file are the same in new Office 365 tenant.

How to paste a number into Excel as text? - Microsoft Community

m22d say:

How to paste a number into Excel as text?

I commonly paste lengthy numbers from other programs into Excel. Since they represent a part or sku ID, and the info is alphanumeric by nature, I need to find a way to consistently paste all such numbers so that Excel immediately recognizes and formats them as TEXT.

In particular, the problem is that some such "numbers" are IDs that have leading zeros (eg 00123456). When pasted as a number, the leading zeros disappear, and the ID is no longer valid.

What's the secret I'm missing? And is there a way to do such pasting without going thru a right-click menu?
jllatham say:

You could preformat the columns/cells that you will be pasting into as Text and see if that doesn't work for you.

Another option would be to copy the ID number and then in the cell where you want it pasted, first type in a single quote mark (apostrophe) and then paste the number behind that apostrophe.  The apostrophe tells Excel to treat whatever follows as text even if it appears to be a number.  Actually you may have to type the apostrophe first then do the copy and paste.  You could "pre-load" all of the cells you plan on pasting into with the single quote mark to facilitate things.

But Excel should recognize mixed alpha-numeric entries as text immediately.
