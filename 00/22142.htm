Using the Shift Key in PowerPoint

Technical Level : Basic

Summary
Learn how you can improve your workflows by holding down the Shift key while performing everyday PowerPoint tasks.
Details
PowerPoint almost always allow you to perform the same task using multiple, differing steps. For example, there are many ways to arrange, position, or resize your slide objects such as text, pictures, tables, etc. Using some of the easier options can save you hours of work. Clip art that you insert from online sources also get resized in the arrow direction.


Video clips, Smartart graphics, and Shapes also get resized the same way.

It's not a good idea to lock a user in a loop so this gives the option to try again for a correct sheet name or cancel out.

Sub copy_data()
Dim response As String, ws As Worksheet
Dim sht As String
100:
response = Application.InputBox("Select a cell in the worksheet to copy from", "Select", Type:=2)
On Error Resume Next
If response = "False" Then Exit Sub
Set ws = Sheets(response)
On Error GoTo 0
If ws Is Nothing Then
    MsgBox "Invalid sheet name, enter again or Cancel to exit."
    response = ""
    GoTo 100
End If
ws.Range("A1:X76").Copy Sheets("Copy Data").Range("A1")
End Sub

You have what's known as a "branded" phone, which means that the mobile network operator has installed their custom version of the firmware in the phone.
