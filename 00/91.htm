bob jones aka: cybertaz say:

You cannot create your own version of the Normal.dotm template simply by saving a file using that name. If that's what you're trying to do it will fail every time.

Only Word can generate a valid Normal.dotm file which the user then can modify. In this case you don't need a new template, though. Do exactly as described in the instructions you cited... Which document you do it in is irrelevant, nor does the document, itself, need to be saved. Just confirm Yes when prompted after clicking the Default button.

What may be giving you the impression that It doesn't "take" is the fact that Word 2016 does something which it did not do in prior versions - It takes its Language cue from the Keyboard Input Source specified in System Preferences. IOW, if Word detects that your using a Spanish keyboard Input Source it will automatically set the Language to Spanish, both in new documents as well as the current location in existing documents regardless of what your Default language may be set to in Word. Apparently your current Input Source is English. You'll need to either;

  • Set the keyboard Input Source to match the Language you want to use, or
  • Manually select the preferred Language each time you create a new document.

Unfortunately, there is no preference or option in the program for changing this behavior. Please use the ☺︎ at the right end of the Ribbon in Word to send a message to Microsoft expressing your opinion on it.

Nokia Xpress 5800 - Camera Not Working - please help!

I have a Nokia Xpress Music 5800 and am having poblems with the camera.  It is showing on the applications menu that it is still open (with the green circle next to it) but when you press the camera icon or the camera buttons on the phone, it wont load.

 

I have tried taking the battery off and then switching it back on which seems to 'close' the application, I can then load the camera, take one picture and then it says it is processing the image and freezes. I then have to turn the phone off. 

 

Strangely though it seems to be ok in video mode once I've restarted it.

fender77---01 say: Hi,

First I would pull out the memory card when the phone is turned off, and check how the camera behaves when it has to store the pictures to C memory... Does everything works now?

If it does, I would try to back up what needed from the memorycard and then format it.

If it still misbehaves, do the soft reset which will wipe your phone memory and verry often it clear out any glitches. ( You diall *#7370# with sec code 12345 by default) - Because it's gonna wipe your phonememory you should back up contacts, messages, calendar notes etc - so you can restore them afterwards.

While youre at it, you should check if there are any updates available for your phone. Check here: http://europe.nokia.com/support/download-software/device-software-update/can-i-update

If it is: Follow the instructions and update your phone - Then you can see if it works before concidering doing a soft reset. (Note that I would recommend you to do it after a fw update, to start out fresh).
