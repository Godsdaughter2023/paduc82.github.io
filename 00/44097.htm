Step 1: Follow the steps to uninstall a program/game:

 

a. Press Windows key + X and clickon Programs and Features.

b. Select the program/game and click Uninstall.

c. Restart the computer and then try step 2.

 

Step 2: Refer to the Microsoft Help article to install the game in compatibility mode.

 

Make older programs compatible with this version of Windows:

http://windows.microsoft.com/en-us/windows-8/older-programs-compatible-version-windows

 

Method 2:

 

If the issue persists, I suggest you to update the graphics card driver and check if it helps.

 

Follow the steps to update the graphics card driver:

 

a.             Press Windows key + X and click on Device Manager.

b.             Search for Display Adapters.

c.              Right-click on the Graphics Card and click on Update Driver Software.

Hope this helps.
