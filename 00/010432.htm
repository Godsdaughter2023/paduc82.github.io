Do you have a Section Break between Page 5 and the start of the Exhibit A?   If so, if it is set to be an Odd Page Section Break, a blank page will be inserted when you print the document so that the Exhibit starts on an odd page.

You will not, however, be able to change the section break type if numbering is restarted in the exhibits section (so that the first exhibit is on page 1, which Word knows is an odd page). Backup, restore, & clean of Microsoft Edge's Favorites by - Microsoft Community

trybird say:

Backup, restore, & clean of Microsoft Edge's Favorites by batch files

Here's my way on backup, restore, & clean of Microsoft Edge's Favorites by batch files.


You might use notepad to make three batch files, and individually name them as EdgeFavoritesBackup.bat, EdgeFavoritesRestore.bat, EdgeFavoritesClean.bat, then save them into your own custom backup folder, then in File Explorer navigate to the folder which the batch files saved in, then right-click on the batch file you wanna run, then a shortcut menu will pop up, then please click on "Run as administrator" menu item on the shortcut menu, then User Account Control dialog box will pop up, then please click on "Yes" button, then it will be running quickly within an opened "Command Prompt (Admin)" window, then its window will close automatically, means that the execution or performance is complete, - the result you wanna has been reached.


Note:
My Microsoft Edge's Favorites backup folder is located in E:\Software\Settings\Edge, so, please replace the E:\Software\Settings\Edge placeholder with the location of your customized settings' backup folder.


Below are my three batch files for backup, restore, & clean of Edge's Favorites.


If you wanna to back up your Microsoft Edge's Favorites, might run this batch file:
EdgeFavoritesBackup.bat (note: this line is just the file name, not file contents, yet the below lines are file contents)
taskkill /f /im MicrosoftEdge.exe /t
for /f "delims=" %%d in ('dir /ad /s /b %LocalAppData%\Packages\DBStore ^| findstr /v "LocalState"') do robocopy "%%d" "E:\Software\Settings\Edge" spartan.edb /b /copyall

If you wanna to restore your Microsoft Edge's Favorites from the spartan.edb file you previously already have backed up, might run this batch file:
EdgeFavoritesRestore.bat
taskkill /f /im MicrosoftEdge.exe /t
for /f "delims=" %%d in ('dir /ad /s /b %LocalAppData%\Packages\DBStore ^| findstr /v "LocalState"') do robocopy "E:\Software\Settings\Edge" "%%d" spartan.edb /b /copyall

If you wanna to reset and completely delete (remove) all of your Favorites data in Microsoft Edge, might run this batch file:
EdgeFavoritesClean.bat

taskkill /f /im MicrosoftEdge.exe /t

for /f "delims=" %%f in ('dir /ad /b %LocalAppData%\Packages\Microsoft.MicrosoftEdge_*') do (
del /f /s /q /a "%LocalAppData%\Packages\%%f\AC\MicrosoftEdge\User\Default\DataStore"
rd /s /q "%LocalAppData%\Packages\%%f\AC\MicrosoftEdge\User\Default\Favorites"
)


Good luck.

gary's student say:

Enter the following event macro in the worksheet code area:

 

Private Sub Worksheet_Calculate()
If Range("F34").Value < 0.99 Then
    MsgBox "F34 below limit"
End If
End Sub


Because it is worksheet code, it is very easy to install and automatic to use:

   1. right-click the tab name near the bottom of the Excel window
   2. select View Code - this brings up a VBE window
   3. paste the stuff in and close the VBE window

If you have any concerns, first try it on a trial worksheet.

If you save the workbook, the macro will be saved with it.


To remove the macro:

   1. bring up the VBE windows as above
   2. clear the code out
   3. close the VBE window

To learn more about macros in general, see:

http://www.mvps.org/dmcritchie/excel/getstarted.htm

To learn more about Event Macros (worksheet code), see:

http://www.mvps.org/dmcritchie/excel/event.htm
