Event Viewer doesn’t really sense anything at all – it just logs the reports that it receives from an app or a service, in this case the Microsoft Antimalware service. In order to reconstruct the history (detection, action, and errors) for any particular threat, you need to filter for all of the relevant event IDs:

MSE (Windows 7)

1.) Open Event Viewer by typing “eventvwr” in the search box and hitting the Enter key.

2.) Expand Windows Logs and click on the System folder.  

3.) Click on the Filter Current Log option in the Actions pane or Action menu.

4.) Select Microsoft Antimalware as the Event source and enter “1116, 1117, 1119” for the Event IDs.

5.) Click on Save Filter to Custom View and name the Custom View “MSE Detections”.

The next time you open Event Viewer, simply expand Custom Views and click on “MSE Detections”.

Windows Defender (Windows 8/8.1/10)

1.) Press the Windows key + R to open the Run window.

2.) Open Event Viewer by typing “eventvwr” in the Run window and hitting the Enter key.

3.) Navigate to Applications and Services Logs/Microsoft/Windows/Windows Defender/Operational 

4.) Click on the Filter Current Log option in the Actions pane or Action menu.

5.) Enter “1116, 1117, 1119” for the Event IDs; and click OK.

1116 – Malware detection

1117 – Malware remediation

1119 – Remediation error

I am fairly sure that I now understand the problem.

 

When you set the DTPicker Format property to 2-dtpTime, even though it is not displayed, there is actually a date included in the time value and that defaults to today's date. Therefore it is necessary to subtract the date from the value to just leave the time. Times are a fraction of a day and the days are whole numbers so to leave just the time we subtract the integer of the value from the total value.

 

When combining the date and time in a single cell it is best to sum the value of the date picker with the time from the time picker. Because it is a real date and time in the cell we can use Number format to display that date and time in a format of choice.

dd MMM yyyy hh:mm t    24 Mar 2013 10:12 P
