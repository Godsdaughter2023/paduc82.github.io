To reset your Xbox 360 console, follow these steps:
 

  1. Press the Guide button, and then select Turn Off Console.
  2. Turn on your Xbox 360 console.


Note If you are using an Xbox 360 Wireless Controller, you can turn on the console by using the Guide button on the controller

    2. Select Download Profile.

      Note If your account is an Xbox Live child account, the parent’s Microsoft account information is required to download the profile.

sheryl85_ say:

How to disable the lock screen from sleeping or hibernate mode

I've disable the lock screen when the computer is turned on, but can't seem to disable it when it goes into sleep mode, can someone help.  I've read the other posts and I've done them, but it still doesn't work.  Thanks.

a. user say:

Hello Sheryl,

Thank you for posting your question on Microsoft community.

I appreciate you for providing details about the issue.

I also appreciate this fact that you want to disable lockscreen when Windows goes to sleep.

I would suggest you to follow these steps and check.

a. Press Windows + R keys, type regedit in run window and press Enter.

b. Navigate to the following registry key location:

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Personalization

c. If you do not see "Personalization" key, right-click "Windows" key above it, point to "New", and create a key named "Personalization".

d. Right-click in the right pane and create a new "DWORD" value named "NoLockScreen".

e. Double-click the "NoLockScreen" value and enter a value of "1".

f. Close the registry editor and restart your computer.

Note: Serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:
https://support.microsoft.com/en-us/kb/322756

I hope this information helps.

Please do let us know if you need any further assistance.
