Hello Kalee,

Outlook for Mac doesn't allow scripts to send email automatically. 

I suggest you create a template and send the emails via different template every time.

Steps:

1. First, update to the latest version of Outlook for Mac client:

Click Help > Check for updates > tick manually check > check for updates. 15.36.170721

2. Save an email message as a template

Select the email message you want to use as a template.

Provide a name for the template, such as "Monthly Status".

Select the location where you want the template to be saved.

Select Save.

3. Create a new email message from a template

Select the template location and name. The name will end with .emltpl.

Select Open.

Add your email message content and recipients, and click Send.
