Hello Michael,

Thank you for posting your query in Microsoft Community.

We're sorry to hear the inconvenience you have experienced. Let us know few things to assist you in this regard.

What happens when you tried to access Start menu or Microsoft Edge?

Does they throw any error message on to the screen?

Which Insider Build are you using?

If Start Menu stops responding on your PC, then it might be due to one of the below reason.

* System might be loading still. (You confirm this by waiting on desktop screen without triggering an action for 2-4 Minutes)

* Some drivers or applications creating conflicts.

* User Profile got corrupted or some system files might be missing or infected.

We have observed that the issue has been caused due to some incompatible drivers or some application which are creating conflicts. But it’s not always obvious what may have caused the issue. Let us confirm this issue by following simple troubleshooting steps.

  1. Restart your PC. When you reach the sign-in screen, hold the Shift key and select the Power button, and then select Restart.

  2. After your PC restarts, select Troubleshoot > Advanced options >Startup settings > Restart. After your PC restarts, you should see a number of options. Press 5 or F5 for Safe Mode with networking.

  3. When you're on to the Safe Mode, check whether basic features of Windows work fine there. Like Start Menu, Notification bar(I.e. in your system tray) and some other things. However Microsoft Edge won't work properly in Safe Mode. Safe Mode, check whether basic features of Windows work fine there. Like Start Menu, Notification bar(I.e. in your system tray) and some other things. However Microsoft Edge won't work properly in Safe Mode.

Note: Safe mode starts Windows with a limited set of files and drivers. Startup programs don't run in safe mode, and only the basic drivers needed to start Windows are installed. Only the basic files and drivers necessary to run Windows are started. The words Safe Mode appear in the corners of your monitor to identify which Windows mode you're using.

If everything works fine there, then the possible cause of the issue is due to some of incompatible drivers or applications. Try to uninstall any application or driver you have installed recently and check if it fix the issue. Check and install all compatible drivers from Manufacturer website.

Also ensure that you're system is Up to date with Windows Update. There are lot of updates rolled out to fix the most common issues that are faced with compatibility on Windows 10. Go to Start Button/>Settings/>Update & Security/> Windows Update and check for any available updates.

If you think the problem is due to corrupted User Profile then you try below mentioned troubleshooting steps.

Use DISM tool and system file checker to repair or replace corrupted system files.

  1. Press CTRL+Shift+ESC to open Task Manager.

  2. Click File > Run New Task

  3. Make sure you have a check mark beside “Create this task with administrative privileges”

  4. Type CMD and hit Enter key

  5. Type or copy/paste the following commands at the CMD prompt and hit enter:

DISM.exe /Online /Cleanup-image /Scanhealth

DISM.exe /Online /Cleanup-image /Restorehealth

sfc /scannow

To close the Administrator: Command prompt window, type Exit, and then press Enter. Reboot your system to see if problem still occurs.

If problem still persist, then I suggest you to add another Microsoft account by following the steps mentioned below and see if the problem persists:

* Press CTRL+Shift+ESC to open Task Manager.

* Click File > Run New Task

* Make sure you have a check mark beside “Create this task with administrative privileges” . Type ms-settings: and hit enter.

* Navigate to Settings/>Accounts/>Your Account/> Family & other users, on the right pane click Settings/>Accounts/>Your Account/> Family & other users, on the right pane click Add someone else to this PC under Other users.

* If you have another Microsoft account try to use that or else follow the steps to sign-up for new and switch with new Microsoft account.

* Sign-out from the old and Login with new Microsoft account.

Once you're sign-in with new Microsoft account, then try work around to see if problem still persists.

Keep us posted if you require further assistance.
