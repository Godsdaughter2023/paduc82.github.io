bob jones aka: cybertaz say:

In word processing programs this is virtually impossible unless it's an intricately planned objective at the very inception of the project... Even then it can be a challenge. After-the-fact it could take a good deal of time & work [along with a possibly good amount of compromise] because this is a meticulous aspect of Typesetting rather than word processing. There simply are too many variables involved which impact vertical spacing.

For example, you have top alignment set but among other considerations have you:

  • used the same font & font size exclusively?
  • set the identically same Line Spacing throughout?
  • calculated the imposition of Space Before/After Paragraph Spacing based on the number of paragraphs & Headings in the document & where those Headings fall?
  • used Keep with Next or Keep Lines Together paragraph formatting?
  • included footnotes at all?
  • inserted any graphic objects?
  • included any Tables?
  • used any Bulleted/Numbered Lists?

It's less challenging if working with a plain text manuscript which hasn't been formatted to deviate from the defaults. Otherwise, the needed changes most likely will have an unwanted influence on text flow which also will have to be corrected.

For a more thorough explanation & recommended approach please see this tutorial by MVP Suzanne Barnhill. It's PC oriented but the concepts, etc. are identical... only the UI varies:

“Improving the bottom line”—tips for achieving more uniform page length

hansv mvp say:

To create a hortcut to the database:

  • Right-click the database in Windows Explorer.
  • Select Send To > Desktop (as shortcut) from the context menu, This creates a shortcut.
  • Rename the shortcut.
  • If the database contains an AutoExec macro, it will be run when you open the shortcut.
  • If the database has a startup form, that form will be displayed when you open the shortcut.

To create a shortcut to any database object:

  • Open the database.
  • Make sure that it is not maximized and that you can see part of the Windows desktop.
  • Drag the database object (form, report, ...) from the navigation pane to the desktop and drop it there. This creates a shortcut.
  • Rename the shortcut
