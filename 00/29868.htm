Figured it out! Type in 'cmd' in search box. You'll see Command Prompt listed. Right-click and select Run as Administrator.

Type 'tzutil /l' to see a list of all time zones. Once you find your time zone, copy the description that is on the second line.

Example: 

(UTC-06:00) Central Time (US & Canada)
Central Standard Time   <---- copy

Afterwards, type 'tzutil /s "(paste the time zone here)" '

Like this:

tzutil /s "Central Standard Time"     (make sure to include the quotations!) then ENTER.

Exit the Command Prompt, go to your time settings, and it should be correct!

Hope this helps!!!

By doing so, it will then allow you to run an ‘anti‘ program to remove any infection(s). There are 4 versions of Rkill; exe, com, scr and pif. Some malware may recognise the program and stop the exe version from running, if so, try one of the others. Rkill doesn’t delete any programs, it just stops them from running, removes rogue entries from the registry and restarts Windows Explorer. It will take less than a minute to run and produces a log file showing which processes have been stopped. Don’t restart the computer until the infection has been removed, as any processes that Rkill stops may be restarted when you restart the computer, including, of course, any nasties that haven‘t been removed from startup.

Start a new career and when the character creation starts there should be an option called Select Existing ID. If this exists, you will be able to start a new game with a character you had previously finished the game with, not as good as having all your saves back but you will at least keep your level and equipment.

 

If even that option is not there, then I'm afraid everything has been lost.
