cmdrkeene say:

You are absolutely right: registry cleaners do far more harm than good.  The registry is just a simple database of reference. For example, in your registry you'll find something like this example:

WallpaperPicture = "C:\Users\Shawn\Pictures\Motorcycle.jpg"

As you see, it's just a reference that specifies which picture file I want to use as my wallpaper.  There is no "cleaning" that needs to be done to the registry.  Even if there were extra entries, it doesn't matter unless some other program looks up and asks for that entry.  It's much like the reference desk at the library.  Just because they have extra copies of the encyclopedia set, doesn't make them any slower.

There's no blanket "registry repair" that ever needs to happen.
