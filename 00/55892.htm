"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"

tryx3 say:

Ryan is correct.

The message you get has nothing to do with pairing. 

The message you get merely means that your metered setting prevented Windows going online to check whether or not it ought to download updated device drivers.  It is not claiming that anything is needed, merely that it does not know either way.

As I also use a metered setting for my networks, I get this for every device in Settings, Devices, Bluetooth & other devices and in Settings, Devices, Printers & scanners. 

If I ever suspect that a device is not performing correctly then I can either enable the Download over metered connections for a while or get the latest driver from the device maker.  The second option is always the better choice because not all OEM device driver updates make it to Windows update & because not all Windows updates device driver updates are the right ones.
