doug robbins - mvp office apps & services (word) say:

I have many macros in my Normal.dotm template, but for the more complex ones, I have them stored in a template that I save into the Word Startup folder.

It is convenient to use the Normal.dotm template for storing macros, because it is always possible to get at the code in them, whereas, if they are stored in a template that is in the Word Startup folder, it is necessary to use File>Open and open that template to be able to access the code if you want to modify it.

If you need a macro to set a document variable to the path of the document, the code to use would be

Dim i as long, j as long

With ActiveDocument
    .Variables("varPath").Value = .Path

    For i= 1 to .Sections.Count

        With .Sections(I)

            For j = 1 to .Footers.Count

                .Footers(j).Range.Fields.Update   

            Next j

         End With

    Next i

End With

tim liu msft say:

Hi Customer,

I'd like to answer your question as below:

Question 1.

If in question 1 you mean you upload the folder MyStuff in browser, it means you upload this folder to cloud. It doesn't set up the sync between the local MyStuff folder and the cloud one. We can say those contents uploaded to the cloud are copies of those local files. As there is no sync process between the local MyStuff folder to OneDrive cloud, when you create new folders in the local folder, they won't be synced to cloud.

Question 2.

Actually, when you click Sync button in the cloud, it sync all those contents in your cloud OneDrive to a local synced folder. We can say this is the operation to set up your OneDrive sync client on your computer. However, in this step, when you select the folder C:\MyStuff, OneDrive will create a folder OneDrive-company name under that folder. I'd like to explain that when we select a local folder, OneDrive will create it's own OneDrive-company name folder in it. All items created in this synced folder will be uploaded to cloud automatically. All items added from OneDrive cloud also will be synced to this synced folder automatically.

Note: We cannot set the synced path like C:\MyStuff, OneDrive will create the folder OneDrive-Company name as synced folder.

So, in this case, C:\MyStuff\OneDrive-company name is the synced folder. All things in it will be synced automatically. For other contents out side of OneDrive-company name folder in MyStuff folder (I mean those duplicates files), you may keep them as back up or directly delete them if you don't need them any more.

A proper process could be firstly click Sync button to set up the synced folder on you computer. And then move all things in MyStuff folder to that synced folder. Things will be synced back to cloud.

    2)  At the finish of work each day, log out of your Mac.  This cleans up the memory and enabels certain house-keeping tasks.

    3)  At the end of each week, shut the Mac down.  When you restart on Monday or whenever, OS X will completely clean p the memory and temporary files for you.
