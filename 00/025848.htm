norman david jones say:

Hi Wyatt-2016, 

Thank you for your help! Last question, how do I have the macro place the sum in a cell of my choice? For example A500? Thank you. 

Try:

'=========>>
Option Explicit

'--------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim aComment As Comment
    Dim dSum As Double
    Const sStr As String = "visa"
    Const myCell As String = "A500"               '<<=== Change to suit

    Set WB = ThisWorkbook
    Set SH = WB.Sheets("Repairs")

    For Each aComment In SH.Comments
        With aComment
            If InStr(1, .Text, sStr, vbTextCompare) Then
                dSum = dSum + .Parent.Value
            End If
        End With
    Next aComment

    'MsgBox Format(dSum, "$#,##0.00")

    With SH.Range(myCell)
        .Value = dSum
        .NumberFormat = "$#,##0.00"
    End With

End Sub
'<<=========

Replace A500 with the address of the required recipient cell.

If you also want to retain the message, delete apostrophe which now precedes 

the instruction:

      MsgBox Format(dSum, "$#,##0.00")
