    jay freedman say:

    Obiwan (can I call you "Ben"?),

    Save your form as a macro-enabled template (*.dotm file) in your Templates folder. Press Alt+F11 to open the macro editor. In the Project Explorer pane in the upper left, expand the icon of the template and double-click the ThisDocument icon inside the Microsoft Word Objects folder. Then copy/paste the code below into the big empty area on the right.

    Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)
        Dim testCC As ContentControl
        Dim iReviewed As Integer
        Dim iTotalScore As Integer
        Dim iSectionScore As Integer
        Dim sngPct As Single
        
        ' if user erased the list value, set it back to N/A
        If CC.Type = wdContentControlDropdownList And CC.ShowingPlaceholderText Then
            CC.DropdownListEntries(CC.DropdownListEntries.Count).Select
        End If
        
        For Each testCC In ActiveDocument.ContentControls
            If Not InStr(testCC.Range.Text, "N/A") > 0 Then
                iReviewed = iReviewed + 1
                Select Case Left(testCC.Range.Text, 1)
                    Case "A": iSectionScore = 10
                    Case "B": iSectionScore = 7
                    Case "C": iSectionScore = 3
                    Case "D": iSectionScore = 0
                    Case Else: iSectionScore = 0
                End Select
                iTotalScore = iTotalScore + iSectionScore
            End If
        Next testCC
        
        If iTotalScore = 0 Then
            MsgBox "No sections reviewed", vbCritical
            Exit Sub
        End If
        
        sngPct = Int(CSng(iTotalScore) * 10# / CSng(iReviewed))
        
        ActiveDocument.Tables(1).Rows.Last.Cells(1).Range.Text = "Score = " & CStr(sngPct) & "%"
    End Sub

    Private Sub Document_New()
        ' initialize all dropdowns to N/A (last entry in list)
        Dim CC As ContentControl
        For Each CC In ActiveDocument.ContentControls
            If CC.Type = wdContentControlDropdownList Then
                CC.DropdownListEntries(CC.DropdownListEntries.Count).Select
            End If
        Next CC
    End Sub

    Here's an explanation of what it does:

    • When you use the File > New or double-click the template's file in File Explorer, Word creates a new document based on the template. The macros in the template are available to run in the document. The Document_New macro runs automatically then, and it sets all the dropdowns to show the "N/A: 0" entry instead of the "Choose an item" placeholder text that all dropdowns display by default. This assumes that you've made N/A the last entry in the list, like the sample in your first post.
    • Each time the cursor exits from a dropdown, the Document_ContentControlOnExit macro runs.
      • The code loops through all the dropdowns in the document. This does not depend on the number of dropdowns, so you can add more of them to the template without changing the code.
      • If any dropdown is still showing N/A, that loop does nothing. Otherwise, the variable iReviewed is incremented by 1 and the rest of the loop continues.
      • The code looks at the first character of the text in the dropdown box. If it's an A, B, C, or D then the corresponding number is added to the total score. (Note that this does instead a message is displayed and the macro stops.
      • If iReviewed is greater than 0, the percentage is calculated. (I simplified the equation a little.)
      • Finally, the percentage is written in the first cell of the last row of the table (I assumed there is only one table in the document, and that you want the answer in the first cell of the last row).
    • As you score more sections, the percentage is recalculated and the table is updated every time you leave that dropdown.
    • If you save, close, and reopen the document, you can continue scoring more sections. (The setting of the defaults to N/A happens only when a new document is created.)

    Access 2010 Date/Time Stamp for Record Edit - Microsoft Community

    jim9988 say:

    Access 2010 Date/Time Stamp for Record Edit

    How do I accomplish this?
    graham r seach say:
    Hi Jim,
     
    The easiest way is to create a table trigger.
     
    1. Open the table in design view.
    2. Add a new datetime field.
    3. From the Design group, select Create Data Macros | Before Change.
    4. Display the Action Catalog.
    5. From the Action Catalog, double-click SetField.
    6. In the [Name] textbox, enter the name of the date field you want to change.
    7. In the [Value] textbox, enter Now().
    8. Click Save, then Close.
    . Save the table design, and close it.
     
    Now, every time you create or edit a record, the date will be recorded in the changed row.
