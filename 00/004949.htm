Thank you for sharing that to me, IndrajitDutta1.

You can manually register the Ac32BitAppServer.dll using the Microsoft Register Server from an elevated command prompt. Here's how:

  1. Click the Start button, then type "command" in the search box.
  2. While holding Ctrl+Shift keys on your keyboard, hit the Enter key.
  3. Click Yes on the permission dialog box that will pop up.
  4. Type: regsvr32 /u Ac32BitAppServer.dll, then press the Enter key from the keyboard to un-register the file.
  5. Type: regsvr32 /i Ac32BitAppServer.dll, then press the Enter key again to register the file.
  6. Close the command prompt window.
  7. Reboot PC, then try to install the program you're trying to install.
