    jetta48 say:

    Hi 

    WLMP files are project files (that contain the video editing info.) in Movie Maker rather than a more universal video file that other apps such a Windows Media Player . You need to open the files in Movie Maker then save them as WMV video files. The WMV files should then be playable in Windows Media Player.

    Assuming you have Movie Maker 2012 already installed:

    1. Make a copy of the WLMP file and work from the copy just in case.

    2. Open Movie Maker 2012.

    3. Select File then Open project.

    4. Browse to the copy of the WLMP file then select it and select Open. Hopefully the project will appear without any yellow triangles with ! in them.

    5. Select File then Save movie then the option that suits your needs (such as For computer).

    6. Browse to your preferred save location, give the file a name and select WMV  from the Save as type menu at the bottom then select Save. The saving process may take some time depending on factors such as the length of the video, its' resolution and the specifications of your computer.

    7.  Open the saved WMV using Windows Media Player to check it worked.

    NOTE: Movie Maker is no longer supported by Microsoft - please see:
