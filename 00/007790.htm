suzanne s. barnhill say:

AutoText entries are stored in templates. By default, they are stored in the Normal template, Normal.dotm. Only AutoText entries stored in Normal.dotm will generate AutoComplete tips, but they can be stored in any template.

You can create a special template that users can load as an add-in: if the template is stored in Word's Startup folder, it will be loaded at startup, and AutoText entries in the template will be available to all documents.

If users want to be able to take advantage of AutoComplete, they can move the AutoText entries from the add-in to Normal.dotm using the Building Blocks Organizer. Select any entry, click on Edit Properties, and, in the Save in dropdown, change the location from the add-in to Normal.dotm.
