Hi Chris,

Re:  Memory Issues.   The HIGHMEMFIX=1 entry essentially deals with allocation of memory when FSX is under load when using textures.   Typically, users often find that parts of their aircraft are either missing or 'dismembered' or scenery around them doesn't look right/disjointed.   This entry usually fixes these anomalies.

To do this, go to your fsx.CFG file (found here:  Your drive letter:\Users\Your Name\AppData\Roaming\Microsoft\FSX).  Then scroll down for fsx.CFG (ensure that you Folder Options in your Search box, then go to the View tab, and locate:  Hide Extensions for known file types.   This entry should be unticked.  You can also go through other entries there and ensure that they reflect what you require in the other areas of your computer.   Then click  Ok.

Once the file (fsx.CFG) is opened (in Wordpad), go down your list to the Heading, [GRAPHICS].   Go to the end of that list under the heading and simply add this line:   HIGHMEMFIX=1   Save the file and close everything.   FSX should now be equipped to handle this type of memory mismanagement.   There is no Registry change here.

I've attached this link for your extra reference:    https://flyawaysimulation.com/news/4737/ 

Installation of Service Pack 1 is straightforward (I assume that you have already downloaded one or both the packs).   I then keep these downloaded files stored elsewhere (not in FSX directory) in the event I need to use them again if having to reinstall FSX.

Hope it all goes well for you.

Chris
