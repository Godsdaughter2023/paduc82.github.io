If you install any version of MS Outlook and add a mail account it will sync with the server (the send/receive process)

A pop mail account downloads, by default all mail from the server, unless you set the option to keep a copy on the server.

A pop mail account will not be in sync on any other device.

An IMAP account is a copy of what is on the server and mail will be in sync (the same) on any other device

An Outlook.com or other Exchange type mail account will be fully in sync, mail/calendar/people with all other devices

MS Outlook by default checks for mail when its started and every 30 minutes (Polling interval)

If your MS Outlook is syncing/using bandwidth it is receiving data, if you set Outlook Offline it will not sync

Well, you've been misinformed :-) You can't assign a keyboard shortcut to a specific word or phrase, but even if you could it probably wouldn't be the best approach. You could crate a macro that inserts the word & assign a kybd shortcut to it or you could create an AutoText item & assign one to that, but both are overkill for what you're describing.
