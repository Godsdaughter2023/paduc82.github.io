Help with how to calculate a date in Access - Microsoft Community

strobn say:

Help with how to calculate a date in Access

I am working on creating a vaccination database.  I am needing to have it calculate when the ext vaccination is due.  Example:  I need to know 30 / 90 / 150 days out from a date.  What would be the best way to have this calculated, as well as displayed.  Should I place this information into a form, or place it into the table?  Side note, I have about 300 records already that have the date entered.  
john w. vinson say:

The DateAdd() builtin function can do this very easily. It can be used on a Form or in a Query; but derived values like this should not generally be stored in a table. The only reason one would do so is if you want the calculated field to be editable - e.g. the calculation says March 31 but you want to override that and specify March 30.

In a Form or Report you can set the Control Source property of a textbox to

=DateAdd("d", 30, [VaccinationDate])

to display the ext date. Instead of a literal 30 you can use a field or some other expression, for instance looking up the number of days using the DLookUp function.

hansv mvp say:

You can calculate the dates in a query based on the table, and use this query as record source for forms and reports.

Let's say you have a date/time field Date0.

Create calculated columns in the query:

 

Date30: [Date0]+30

Date90: [Date0]+90

Date150: [Date0]+150

Creating a Custom Word Quick Part - Microsoft Community

ernestoglenn-preston say:

Creating a Custom Word Quick Part

how can I create a custom Microsoft Word quick part (what's the syntax) so that there is a field at the end of each paragraph I type that gives the word count ?
rich michaels - mvp (greatcirclelearning.com) say:
To make a Quick Part you create the text and other items you want, select them, and then go to your Ribbon > Insert > Quick Parts > Building Blocks Organizer and hopefully it is self explanatory from there on.

However, why do you want to create a Field that tells you how many words are in the paragraph you just typed? You can get that information very easily. Do the following...
  1. Right click on the status bar at the bottom of your Word window and make sure the Word Count checkbox is marked. (You only need to do this once and it will stay set.)
  2. Type a paragraph of text.
  3. Triple click in the paragraph you just typed, which will select the entire paragraph.
  4. Look at the Word count in the Status bar and it will show an x/y count of words. X = number of words selected and Y = total number of words in the document.
Hope this helps
