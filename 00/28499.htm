jim_282 say:

Windows 10 Start Menu - Why Not Customizable?

Why not allow the user to fully customize their Start Menu? The Start Menu is one of the more important parts of the user interface, and one that many people disagree about. If you let each person design their own, you will end this battle, and it will be a big win for Microsoft.

1. The Start Menu modifying app could have some standard looks, like Windows 8, 7, and XP look (we like the XP/98 look here, even though it is simpler, because it is much easier to read and faster to use - the transparent look is definitely disliked here). Then the standard look could be further customized by the user.

2. Allow the user to change the colors, transparency, etc. - everything. It's a computer - give us the power to make it look the way each of us wants it too, not one standard look, that some like and some don't. We don't all live in houses that are the exact same color and layout. Instead of Microsoft trying to figure out the one look that everyone will like, which is impossible, let each of us make our Start Menu into what we want, and what works best for our unique situations.

3. Allow the user to add or remove jump lists. The jump lists off the start menu items are extremely useful if you use a lot of different files, like we do here. This is my number one favorite feature of Windows 7, because it saves me so much time.

4. If this Start Menu modifying application gets too large, and you want to keep the code base smaller, make this a downloadable app that only those who want it can download it.

5. Allow the user to save their menu customization (this file could then be shared on a Microsoft website for others to use (and this would give you great free publicity), or given to friends, or it could be used to standardize the Start Menus of different computers across an organization, for what works for that organization).

In other words, let each of us build the Start Menu we want. Those who like the Windows 8 look can have it. Those who like the Windows 7 look can have it. Those who like the Windows 10 look can have it. And best of all, those who don't like any of these, can design what they want/need for their use. This would allow Microsoft to meet the Start Menu needs and wants of every customer. It would be a huge win for Microsoft!

It is possible to disable NCSI by a registry setting if you don’t want Microsoft to be able to check your internet connection.  Internet connection issues causes Activation to fail.
   
1. As Administrator, run Regedit
2. Go to HKEY_LOCAL_MACHINESYSTEM > CurrentControlSet > Services > NlaSvc > Parameters > Internet
3. Under the Internet key, double-click EnableActiveProbing, and then in Value data, type: 0
     The default for this value is 1. Setting the value to 0 prevents NCSI from connecting to a site on the Internet during checks for connectivity

4. Reboot then it will work

suzanne s. barnhill say:

There are two ways to do this. They could be described as "the right way" and "the wrong way," but in fact there are numerous other "wrong" ways (such as trying to use newspaper-style columns or a two-column table), and of these two, there are arguments for both. Here's how to do either:

  1. Go to the Page Setup dialog. If you have the horizontal ruler displayed, the easiest way to access the dialog is to double-click on the top of the ruler. Otherwise, click the dialog launcher (absurdly tiny arrow in the bottom right corner) in the Page Setup group on the Page Layout tab.
  2. On the Margins tab of the Page Setup dialog, under Pages, there is a dropdown labeled "Multiple pages." Click the arrow beside "Normal," and you will see three other choices: "Mirror margins," "2 pages per sheet," and "Book fold."
  3. The latter two are the ones you're interested in.

You can get two logical pages on a physical sheet by using "2 pages per sheet," and there are plenty of good applications for this setting. If you just want to print two A5 fliers on the same A4 sheet, for example, this works well. I especially like it for printing two 3" x 4" name badge inserts on a 4" x 6" index card. But it is not what you want for a booklet unless you want to have to specify the page printing order. For example, if you have an eight-page booklet, you would have to tell Word to print pages 8,1,2,7,6,3,4,5.

The other setting, "Book fold," is the one you want. Onscreen, it looks exactly the same as "2 pages per sheet." Unless you display two pages, you'll see only one page at a time (and you won't see facing pages even so), and you treat these "pages" the same as you would a full-sized page. You can use page borders, add page numbers, and do anything else you would do with a full A4 page. You create the pages in numerical order, just as you would with a full-size document. "Mirror margins" is enabled automatically when you choose "Book fold," so you can set a larger Inside margin if desired. The process is entirely transparent to the user. But when you print the document, Word will order the pages such that you can fold them into a booklet. If your printer duplexes automatically, you'll end up with finished pages; otherwise, you'll have to take your single-sided pages and have them copied or printed double-sided.

There is more on this subject at http://wordfaqs.mvps.org/2PagesPerSheet.htm.
