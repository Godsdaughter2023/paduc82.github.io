a. user say:

Hello Pian,

Thank you for posting your question on Microsoft community.

I appreciate you for providing details about your query.

If you choose to run the file (for some file types your choice will be "open" instead of run), the file is saved to a temporary folder on your computer and run or opened automatically.

If you choose to save the file, you will have the option of deciding where the file is saved on your computer and exactly when you want to run or open it.

You have more control over the file by choosing save option.

When you select run, it begins a two-step process:

• The file you have requested is downloaded to your machine and placed in a temporary location.
• The file then runs exactly as if you used Windows explorer to navigate to the temporary location and double-clicked the file.

The temporary location is either your browser’s cache or temporary folder, or one of your Windows temporary

folders.

In fact, it really only needs to be preserved long enough for "Run" that happens next, and could be deleted at any time thereafter.

When you select save (sometimes only offered as Save As…) in response to a download, the browser simply downloads the file.

The file is placed into a folder that you specify or know how to access, and will not be automatically deleted.

Also see:
Save a file
http://windows.microsoft.com/en-us/windows/save-file#1TC=windows-8

You may move files from temporary folder to any other file location.
