john_barnett say:

Hello Ja047son0165,


The problem with formatting is it over-writes everything with zero's and one's. Once you have done that twice you have completely obliterated what files were there in the first place. Even if you could 'recover' files the chances of them being completely intact is virtually none existent.


While there are file recovery applications available, if parts of a file are not recoverable then the whole file will not be readable. You could try the freeware application Recuva http://www.piriform.com/recuva but I doubt it will be much help.


There are, of course, companies that will attempt to recover lost files for you using forensic applications; however, this is a last resort option and very, very expensive.
