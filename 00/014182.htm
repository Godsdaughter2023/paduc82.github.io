jay freedman say:

According to https://support.microsoft.com/en-us/help/211489/operating-parameter-limitations-and-specifications-in-word,

The maximum file size is limited to 32 MB for the total document text only and does not include graphics, regardless of how the graphics image is inserted (Link to file, Save with document, or Wrapping style) into the document. Therefore, if the file contains graphics, the maximum file size can be larger than 32 MB.

and the maximum size of a file that Word can open is 512 MB. Together, that means you may be able to write your document as a single file. Long before that, though, Word will probably slow to the point that the document is difficult to work with.

I agree with Charles that a master document and subdocuments is not an advisable solution. However, there is an alternative. Write the series of smaller documents, carefully basing them all on the same template and using only paragraph styles and character styles for formatting. To collect them, make one more document that contains a series of INCLUDETEXT fields, one for each "subdocument". That document can also hold a table of contents and an index for the entire collection.

jim gordon mvp say:

Hi Herman,

I don't think PowerPoint has the ability to do what you are asking for.

If it were me, I would insert the video, then copy the video on your slide then paste it back into the slide for as many breaks that you need. Doing that does not actually copy the video multiple times. PowerPoint is memory efficient and will only use the original file for the pasted copies.
