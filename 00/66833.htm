peter.r. say:

Hi GKon,

 

The Manual for the Canon CanoScan LiDE 700F scanner reveals that Windows Live Mail is not in the list of supported e-mail programs. That is why it is not listed by default in MP Navigator EX's list of Mail Programs. Nevertheless, Windows Live Mail can work correctly with Canon's scanning software MP Navigator EX. Indeed, the versions of MP Navigator EX (version 4.1 and higher) provided with the latest model Canon scanners now provide soft-coded support (via an .ini file) for Windows Live Mail.

 

But adding Windows Live Mail to MP Navigator EX's list of Mail Programs via  One-click [button]->Attach to E-mail [button]->Attachment Settings [section]->Mail Program:->[Drop-down arrow]->Add...->Add to List [button]  and navigating to wlmail.exe will not get Windows Live Mail to work properly with MP Navigator EX.

 

However, we can tweak an .ini file to add Windows Live Mail to the list of compatible Mail Programs that work with MP Navigator EX. This enables the Canon scanner's E-MAIL button (if it has one), and MP Navigator EX's E-mail functions to work correctly with Windows Live Mail (a Windows Live Mail compose window will open with the resultant file attached - although MAPI 'attached' image files are treated differently in WLMail 2011 & 2012 (they are not actually 'attached').

 

Below I will outline in detail the method for adding Windows Live Mail to the compatible Mail Programs list in Canon's scanning program MP Navigator EX. For the benefit of others, I will generalise the method somewhat, as it can be applied to any Canon scanner using the MP Navigator EX scanning program.

 

Basically, with a CanoScan LiDE 700F scanner, it involves adding the property Mailer3=Windows Live Mail to the [Application] section of csLiDE700F.ini, which is found within the Device sub-folder of MP Navigator EX's Installation Folder.

 

1. Firstly, if you have already added Windows Live Mail to the list of Mail Programs in MP Navigator EX via the Add...->Add to List option in the Mail dialog box and navigating to wlmail.exe within the Select Mail Program dialog box (One-click [button]->Attach to E-mail [button]->Attachment Settings [section]->Mail Program:->[Drop-down arrow]->Add...->Add to List [button]->wlmail.exe), please remove it (or them if done multiple times) by right-clicking on the Windows Live Mail entry (or entries) that you added, and click on the 'Delete' option that pops up.

 

(See Screenshot 1)

 

You will not see a 'Delete' option appear for the Windows Live Mail entry that will appear there after you add it to the [Application] section of the relevant .ini file.

 

2. MP Navigator EX has one or more .ini files named according to its compatible scanner models, at the following location:

64-bit systems:

C:\Program Files (x86)\Canon\MP Navigator EX x.x\Device

32-bit systems:

C:\Program Files\Canon\MP Navigator EX x.x\Device

(Where x.x is the version number of MP Navigator EX - in this example it will be MP Navigator EX 2.0)

 

So navigate to that location, then copy and paste to your Desktop (or some other convenient location), the .ini file that is named according to your model of scanner - in this example it will be csLiDE700F.ini

 

Note that by default in Windows 7, the option to show file extensions is disabled in Windows Explorer, so if you cannot see the .ini file extension, please open Windows Explorer (i.e. open any folder) and enable viewing of file extensions via: Tools->Folder options...->View [tab]->Advanced settings: [section]->[ ] Hide extensions for known file types [uncheck]

 

The Menu bar in Windows Explorer may also be hidden, so if you cannot find the 'Tools' menu, please show the Menu bar via: Organize->Layout->Menu bar

 

3. Open (double-click) the .ini file that you placed on your Desktop (it should open in Notepad), and look for the [Application] section. It will look something like this:

 

[Application]
Mailer0=Outlook Express
Mailer1=Microsoft Outlook
Mailer2=Windows Mail

 

4. Below "Mailer2=Windows Mail", add a new line (Enter key) and type (or paste) the following:

 

Mailer3=Windows Live Mail

 

Note: use the correct consecutive number if there are more, or less e-mail programs under [Application].

 

It should then look something like this:

 

[Application]
Mailer0=Outlook Express
Mailer1=Microsoft Outlook
Mailer2=Windows Mail
Mailer3=Windows Live Mail

 

(See Screenshot 2)

 

Now save this edited .ini file...

 

5. Back at the location C:\Program Files (86)\Canon\MP Navigator EX x.x\Device (where x.x is the version number of MP Navigator EX - in this example it will be MP Navigator EX 2.1) - rename the old .ini file by adding .bak to the file extension - in this example, csLiDE700F.ini is renamed to csLiDE700F.ini.bak

 

6. Now Copy and paste the .ini file that you edited, back into its original location at C:\Program Files (86)\Canon\MP Navigator EX x.x\Device (where x.x is the version number of MP Navigator EX - in this example it will be MP Navigator EX 2.1).

 

(See Screenshot 3)

 

If you ever upgrade MP Navigator EX, you will need to repeat this process, because the upgrade will overwrite your edited .ini file.

 

7. Now select the Windows Live Mail entry from the list of Mail Programs within MP Navigator EX:

One-click [button]->Attach to E-mail [button]->Attachment Settings [section]->Mail Program:->[Drop-down arrow]->Windows Live Mail [select]->Apply [button]

 

(See Screenshot 4)

 

8. You will also want to select the Windows Live Mail entry within the Preferences dialog box of MP Navigator EX: Preferences [button]->General [tab]->Application to run [section]->Attach to E-mail: [drop down arrow]->Windows Live Mail [select]->OK [button]

 

(See Screenshot 5)

 

9. To otherwise test if Windows Live Mail is the default e-mail program for MAPI:
Right-click a .jpg or .pdf file and select Send to->Mail recipient

 

A Windows Live Mail compose window should open with the file attached - although MAPI 'attached' image files are treated differently in WLMail 2011 & 2012 (they are not actually 'attached').

¡firedog say:

OK, thanks. I hope that warning didn't indicate that you'd tried to run wlmail.exe, did it? The program will have to be installed first.

Did you normally sign in in Windows Live Mail with a Microsoft Account? If so, we probably won't have to worry about contacts.

There's a possibility that you can get Windows Live Mail up and running with your old data on W8, but it's a bit tedious.

First, locate the store folder on the old drive. You didn't say which OS it used; on XP, the folder to find is

{Z:}\Documents and Settings\{olduser}\Local Settings\Application Data\Microsoft\Windows Live Mail

and on Vista/W7

{Z:}\Users\{olduser}\AppData\Local\Microsoft\Windows Live Mail

where {Z:} is what used to be C: and {olduser} is the Windows user account that was using the program.

Copy this entire folder to

C:\Users\{newuser}\AppData\Local\Microsoft\Windows Live Mail

where {newuser} is your current Windows user account.

Now if the old OS was XP, do the same with the folder

{Z:}\Documents and Settings\{olduser}\Local Settings\Application Data\Microsoft\Windows Live Contacts

If it wasn't, there may be a problem.

When those two folders are in place, visit the WE offline installer page and download the installer for Windows XP. Don't be tempted - this will not work if you select a later version.

Then run wlsetup-all.exe as Administrator. With any luck, the installer will both work and find the store folder and take it into use. I have grave doubts about the contacts, though.
