How to create a hyperlink in word calling an executable with arguments - Microsoft Community

vidyamurugesan say:

How to create a hyperlink in word calling an executable with arguments

How to create a hyperlink in word calling an executable with arguments. eg: C:\a.exe -help

where a.exe is an executable in c drive and it takes an argument -help

jay freedman say:
Ah, thanks Peter, I misconstrued "calling".

The way to call an executable is to use the Shell function in a macro (see http://www.gmayor.com/installing_macro.htm). For the given example, the macro would be

Sub LaunchProgramA()
   Shell "C:\a.exe -help", vbNormalFocus
End Sub

To run this macro from the body of a document, use a MacroButton field like

  { MacroButton LaunchProgramA Click here to run a.exe -help }

You must press Ctrl+F9 to insert the field markers; do not type braces. Then press F9 to update the field and collapse the field code to just the "Click here..." part. You can format the field to look like a hyperlink by applying the Hyperlink style to it.

A MacroButton field normally needs a double-click to activate it. To make it run the macro with a single click, put this macro into the Normal template:

Sub AutoExec()
    Options.ButtonFieldClicks = 1
End Sub

abi99 (2) say:

What are the differences between AC60X & AC60?

There is none. That ‘X’ means, a UK plug is attached to that charger. ‘Nokia AC-60’ is just the generic term to identify all local variants (like US, EU, UK; source). For example, I have a Nokia AC-60E because I have the EU-plug variant. Your issue sounds like a faulty Nokia DT-601 – perhaps it draws more current than it should. Therefore, did you consider to replace the Nokia DT-601? Alternatively, did you consider to go for a USB charger which gives 2000 mAh?

kolort, Microsoft does neither leverage Microsoft Answers as feedback channel nor for bug-reporting. Why ever. Microsoft wants you to use the app Windows Feedback which is included since Windows 10 Mobile. Sounds like an app went wild and bogs the processor. The processor then draws so much current, the charger is not able to supply. Although we have the app Field Medic, I have no idea how to read its output. Therefore, did you consider to go for the Recovery Tool via a Windows computer? Then, do not re-apply the backup of your settings but install only that what is really necessary for your daily life. Perhaps you are able to rule out the culprit app or setting that way.
