Macro to search and copy rows to different sheets based on cell value - Microsoft Community

t_rousey say:

Macro to search and copy rows to different sheets based on cell value

I have a workbook that I would like to create a macro to search and copy rows to different sheets.

Here is how I have the workbook set up:

I have two "Input" sheets where I am pasting in reports that are generated by a different program that I am using.  I need to copy rows from these input sheets to other sheets in the workbook based on the value of Column C.  The possible values for column C are as follows; DC, DJ, DW, GT, JF, KY, MC, MW, SB, TW.  These rows need to be copied from the input sheet to its corresponding sheet based on the value of column C (the other sheets are labeled the same as the possible values for column C). 

I think that I will need to write two macros, one for "Grade 4 Input" sheet and one for "Grade 5 Input" sheet.

Any help would be much appreciated.
bill manville say:
Sub CopyData()
  CopySheetData "Grade 4 Input", "A3"
  CopySheetData "Grade 5 Input", "H3"
End Sub

Sub CopySheetData(stSheet As String, stAddr As String)
  Dim WS As Worksheet
  With Sheets(stSheet).Range("A1").CurrentRegion
    .AutoFilter
    For Each WS In Worksheets
      If Len(WS.Name)=2 Then  ' e,g, "DC"
        ' clear previous data
        WS.Range(stAddr).Resize(WS.Range(stAddr).CurrentRegion.Rows.Count,6).ClearContents

        .AutoFilter 3, WS.Name  ' filter column C for the name of the sheet
        If .Columns(1).SpecialCells(xlVisible).Count>1 Then
          ' there is some data to copy; leave the headings behind
          .Offset(1).Resize(.Rows.Count-1).SpecialCells(xlVisible).Copy
          ' paste it to the address passed in
          WS.Paste WS.Range(stAddr)
        End If
      End If
    Next
    .AutoFilter
  End With

End Sub

contacts subfolders missing when connecting old .pst file from Outlook 2007

Hello I just set up a new computer that came with outlook 2010 and used an existing .pst file.  Everything worked fine except there were no subfolders in my contacts.  If I looked at the folder size it displayed all the subfolders in the window along with corresponding size.

I searched long and hard to no avail for a solution.

In the end it was simple and I repurposed a fix from MSOutlook.info on importing contacts that worked a charm.

Start the Import and Export Wizard;

  1. Outlook 2010
    File-> Open-> Import
  2. In the Import and Export Wizard dialog choose;
    Import Internet Mail and Addresses
  3. Click Next.
  4. Set your selection to; Outlook 4.x, 5.x, 6.x or Windows Mail
  5. At the bottom set your selection to “Import Address book” only.
  6. Click Next.
  7. Select how you want the import wizard to treat duplicate items.
  8. Click Finish

All subfolders loaded up and I am a happy camper.

 

This is not really a question but maybe I can save someont else some greif if you post this solution.
