Conditional Formatting a row based on value of one cell in that row

I want to conditionally format rows according to the value in the last column, i.e. if last value is greater than 2 shade the whole row yellow, If the last value is greater than 4 shade the row amber etc.
andreas killer say:
If you want to auto detect the last value in each row do as follows:

Select cell A1
Open the name manager
Create a new name "LastValue" and apply this formula:
=INDEX($A1:$F1,LOOKUP(2,1/($A1:$F2),COLUMN($A1:$F1)))

(Note: This formula works for columns A:F, you have to customize the last column if you have more data columns)

Ok
Close
Select the rows to color
Open the Conditional Formatting
Create a new rule
Use this formula
=LastValue>4
and add the amber fill
Create a new rule
Use this formula
=LastValue>4
and add the yellow fill

That's it.

Andreas.
