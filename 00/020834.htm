To turn off Parallels' feature to allow you to open files in Windows applications:
  1. Open Parallels and select Window menu --> Virtual Machines List.
  2. Select (don't start) the first machine in the list. This will open the VM's window.
  3. Select Virtual Machine menu --> Configure...
  4. Select the Options tab.
  5. Select Applications on the left.
  6. UNcheck the option Share Windows applications with the Mac on the right.
  7. Close the window to save your settings and close the VM's window.
  8. Repeat these steps for each Windows VM you have.
Hope this helps!

jean-yves tfelt say:

Hi,

 

The first SaveAs creates a copy, the second Saveas will try to overwrite the first copy.

Using On error Resume next will jump over the execution.

This amemnded code will work if you want to allow overwritting the copy.

 

Private Sub SaveButton_Click()

'Last updated Jan 24/13 by Amanda Meyer
'This code saves the ticket

Dim ThisFile As String

    'Saves the document with the ticket # as the name
    ThisFile = Range("B1").Value
    'On Error Resume Next
    Application.DisplayAlerts = False
    ActiveWorkbook.SaveAs Filename:="C:\Temp\" & ThisFile
    Application.DisplayAlerts = True
End Sub
