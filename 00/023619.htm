brian tillman [outlook mvp 2007-2019] say:

You also could have searched for "mail" using the Search box at the upper right of the Control Panel window.

When you create a new profile, the autocomplete cache starts empty and must be built up over time. The goal is to see if addresses added to it stick.

If you have a previous cache that you know contains addresses, you may be able to add it to the new profile by copying it from the old profile.
