Your Office 2011 AutoRecovery folder is not in the Applications folder. The path is:

~UserName:Library:Application Support:Microsoft:Office

Depending on your version of OS X [10.7 or later] to access the Library folder for your User Account, you may need to hold the Option key while you open the Go menu in Finder.

Unless you've modified your Spotlight Preferences, however, that location isn't included when doing a Spotlight Search. If you did, in fact, manually save a file to that location, though, it still should be there. Only those temporary files created by the Office programs are deleted automatically from that folder.

rob koch say:

Though such messages can be fake, they're most often real notifications from Microsoft, but it's easy to confirm.  Few of the messages sent from Microsoft display the green shield icon anymore, since most email programs won't display it anyway, so I've personally stopped expecting to see that.

Just go to the following page (easier from your PC) and once logged in, click the "Security" link either along the top menu bar or in the second row of selections below.  On that Security basics page, select Sign-in activity and you'll find your most recent login information.

https://account.microsoft.com/

If the email message you received was fake, it usually wouldn't matter if you opened it from a phone, since most malicious actors wouldn't think to include anything that affected a phone as they're expecting you to be using a PC.  Since there's little true malware that affects phones anyway, it's unlikely that any problem would result.

a. user say:

Currently, there are two "Outlooks":

Outlook client (Outlook 2010, 2013, 2016) - This is an email program, by itself it doesn't do much. To check your emails using this "client" you need to add your email accounts. Once you add your account, you can use this program to send and receive emails. You can also do other things like Contact and Calendar. You can also add multiple accounts.

It is possible to also have multiple Outlook clients on a computer like if you have an old version and a new version (Outlook 2010 & Outlook 2016). In this case, you may uninstall the old version if you are not using it, make sure to transfer any data left

Outlook.com - Web based email server. You use a web browser to access your emails and you can send and receive emails from here. You also use your Microsoft account, but you can also add other accounts.

** I'm also not overly familiar with the other one that you have mentioned (the one with the upgrade to a premier). It might be Windows Live Mail from Windows 7 or an old Outlook program that was carried over from Windows 7.

For what to use, each has pros and cons. Outlook client is a powerful tool and lets you do a lot of things with email. Outlook.com is basic but it has the advantage of being able to access your email everywhere (most devices have a web browser).

Either way, you can use both at the same time, you can connect your Outlook.com account to Outlook 2016. Hope this helps! I'm sure other members can explain it better.
