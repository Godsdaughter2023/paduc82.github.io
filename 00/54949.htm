Hi,

Windows SmartScreen helps keep your PC safe by checking downloaded files and web content within apps to help protect you from malicious software and potentially unsafe web content. It is your discretion if you want to run an app or not.

  • Close all web browser windows. Press the Windows key + R.
  • A run dialog box will pop-up, then type inetcpl.cpl
  • Select the Advanced tab, and then click Reset
  • In Reset Internet Explorer Settings dialog box, select Reset
  • When it's finished applying default settings, select Close, and then select OK. Restart your PC to apply changes.
