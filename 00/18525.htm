I've had this issue too. Until yesterday (June 1 2016) ...I realize this is an old thread, but it was the first in the search list, so I'm spreading the wealth here lol.

I then exited and restarted Outlook.

Then I added each Exchange (and hotmail.com/live.com/outlook.com) account Inbox back to Favorites.

(I exited and restarted Outlook 2016 each time I added an Inbox back to Favorites.)

After each restart, the sort order was exactly the sequence I'd added the Inboxes back to Favorites in.

I've probably restarted Outlook 2016 several dozen times, just to check if the Favorites sort order has changed.

Nope: the sequence has been consistent ...which hasn't been the case for almost two years.

So I'm pretty happy about this.

That's the good.

The bad is that I'm afraid if I change the sequence (and I'd really like to have the Gmail Inbox the 2nd in sequence in the sort order ...but "oh well"), I'm going to end up with a hot mess again.

My "theory" is that the Favorites list somehow "checks" connectivity of available accounts in the Favorites list, and for whatever reason, if you put a MAPI account Inbox (or any folder, probably) "higher" in the Favorites list (sandwiched between Exchange type accounts), the sort sequence just ...crumbles.

...I'd add that I've tried about every "fix" to the problem I've come across during the past two years (the 2nd best was the "right click to open the main Inbox in Favorites in a new Window" trick), and nothing - absolutely nothing - has kept the sort order sequence as consistently as this one.

So.

Apparently there's some issue with mixing MAPI and Exchange sequences when folders are added to the Favorites list that screws things up.

I dunno what it is.

But at least now, you can work-around the problem by putting Exchange accounts first (adding them in the sequence you want: it doesn't seem to matter if that's alphabetical or some other sequence priority ...whatever floats your boat, just put Exchange accounts first) and then adding the MAPI accounts (probably POP too: I don't prefer to use POP anymore so I can't say for sure).

And then leaving the sort order alone!

(I suspect not changing the sort order is essential, but I haven't test it yet.)

But - at least in my case - it's more important that the sort order is consistent than that my 2nd most used account (Gmail) is 2nd in sequence.

I'm just happy all the Inboxes stay in the same place in the list - now lol - when I exit Outlook and restart (and yes, even after a system reboot).

We appreciate your response. If you are trying to export all emails from a mail folder in Outlook to Excel with Copy feature, you may follow the steps below:

  2. Turn off the Reading Pane by clicking View > Reading Pane > Off.
