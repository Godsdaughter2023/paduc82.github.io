Assuming all the first names are one word names...

Use Text to columns.

First, make sure the column to the immediate right is empty so that it can accept the parsed data.

Select the range of names

Goto the menu Data>Text to Columns

Select Delimited>Next

Select Space>Finish

Just thought of something...

That may not work the way you want if there are last names that are more than one word names.

If you want a formula, and, assuming the first names are only one word names...

Enter this formula in B1 to extract the first name:

=LEFT(A1,FIND(" ",A1)-1)

Enter this formula in C1 to extract the last name (or whatever follows the first name):

=MID(A1,FIND(" ",A1)+1,50)

The only thing worse than parsing names is parsing addresses!
