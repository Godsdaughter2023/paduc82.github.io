There's no doubt more complex ways, but at a very basic level you could change it to this:

Private Sub CheckBox1_Click()
ActiveSheet.Unprotect ("password")
If CheckBox1.Value = True Then Range("AA10").Value = 0.15
If CheckBox1.Value = False Then Range("AA10").Value = 0
ActiveSheet.Protect ("password")
End Sub

Omit the "password" if you're sheet is just protected without a password.
