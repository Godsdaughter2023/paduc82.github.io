aits say:

Your comment about .csv not retaining text formatting sounds correct. 

Regarding, contacts modified dates not being preserved, you need to compare apples with apples and not apples with oranges. When you upgraded from Outlook 2000 to Outlook 2007 you were using the same database structure (.pst) and same file system (NTFS). Moving to Outlook 2011, you have moved to a totally different machine that uses a different database structure (separate files rather than a consolidated .pst) and a different file system (HFS+). 

Copying and synchronising are two different things. Copying creates a replica of a file (offline version) which usually preserves the date modified information but may not if it is being moved to a completely different file system. The key point to understand with a copy is that you end up with 2 different files. With synchronisation, it is just one file but it can be accessed and edited from different locations and devices. If you change the file on one device, you will notice the change instantly when viewed from another device. Synchronising requires server-side features so no, it won't work with your external drive. Well known synchronisation options are Apple iCloud, Microsoft One Drive (formally Sky Drive), Google Drive and Dropbox etc. The reason I suggested iCloud is because the Windows version of Outlook allows you to sync your iCloud contacts (i.e. those on your iPhone) by installing the iCloud Control Panel (for Windows). Your contacts are stored in the Cloud and updated in the Cloud so the date modified would always look correct no matter what device, operating system etc that was being used. Like I said but, it looks like Outlook 2011 doesn't support iCloud contacts and calender sync which really does show that Microsoft is not too fussed about making it work as they want you to keep using Windows.

Hi TLadyDay,

We can easily check the weather for a specific day via the Calendar tab in Outlook.com. I’ve checked on my end and the default location stated there is in Redmond, WA. However, you can add another location and select it at the same time. Upon doing so, the set location will already appear and the weather for that place will also be available. Here are the steps that you can follow to perform this action:

  1. Go to Outlook.com and sign in to your account.
  2. Click on the Calendar tab.
  3. Click the cloud icon on any date, then select Edit locations.
  4. Click on Add another location and search for the place you wanted to add.
  5. Click on the location that you’ve searched and make sure that it is highlighted in blue.
  6. Click OK.
  7. Check the calendar and see if the changes already applied.
