You can have two different headers/footers within a section if you enable "Different first page" (on the Header & Footer Tools Design tab).

Since your document has more than one section, if you want to keep header/footer content separate, you will also have to turn off "Link to Previous" for both the first page and the main page header (and similarly for footers). You may be asked to restart your device."

 

Sounds like you possibly have another security program installed or there may be remnants of such a program you previously used that are still on the computer and are interfering with Windows Defender.

 

On Windows 8.1/10 when a third party antimalware program (such as AVG, Avast, McAfee, Norton, etc) is installed the Windows Defender (WD) program is disabled by design to prevent potential software conflicts and performance issues.

 

If you wish to use WD on Win 8.1/10 you should ensure that any third party antimalware program that may be on the computer (or was previously on the computer – possibly before it was upgraded to Win 10) has been uninstalled and that no remnants remain.

 

Note – most new computers included a trial version of various antivirus programs (McAfee, Norton, etc) depending on what the vendor/manufacturer wanted to be included on the computer.
