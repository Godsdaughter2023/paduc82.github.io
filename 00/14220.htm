    VB code to Update multiple table of contents on save - Microsoft Community

    juliaclaire say:

    VB code to Update multiple table of contents on save

    I am working on a document that includes multiple Table of contents, Table of figures and form fields.  Is there any VB code that will on saving update all the Table of Contents, Table of Figures and Field Codes?
    graham mayor say:
    I would be inclined not to run the macro when the document is saved, however the following should update everything you need.


    Sub UpdateStuff()
    Dim oStory As Range
    Dim oTOC As TableOfContents
    Dim oTOF As TableOfFigures
    For Each oStory In ActiveDocument.StoryRanges
        oStory.Fields.Update
        If oStory.StoryType <> wdMainTextStory Then
            While Not (oStory.NextStoryRange Is Nothing)
                Set oStory = oStory.NextStoryRange
                oStory.Fields.Update
            Wend
        End If
    Next oStory
    Set oStory = Nothing
    For Each oTOC In ActiveDocument.TablesOfContents
        oTOC.UpdatePageNumbers 'oTOC.Update
    Next oTOC
    For Each oTOF In ActiveDocument.TablesOfFigures
        oTOF.UpdatePageNumbers 'oTOF.Update
    Next oTOF
    End Sub


    Note that if your TOC is in a section of its own with different header footers from the rest of the document, then updating the TOF field as opposed to updating just the TOC page numbers (alternative code commented out) will screw up the header footers. This appears to be a bug that has been around for many releases of Word and occurs whether you update manually or with vba. I haven't tested it but I suspect it does the same with TOF fields also.
