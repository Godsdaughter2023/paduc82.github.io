cbarnhorst say:

That is not a Windows function.  That is a function of the hardware as detected by the BIOS/UEFI before Windows loads.  It is only changeable at computer startup and before Windows loads.  The order is initially set by the computer or motherboard's manufacturer.  

Your present boot order is a good one because it allows you to boot from dvd or usb if a bootable dvd or usb is inserted and the computer restarted.  If the harddrive was higher in the order than the dvd or usb you could not do that unless no OS was detected on the HDD..

If you insert a bootable dvd or usb you will see a message asking you if you want to boot from it and a five second delay to allow you to confirm by hitting any key.  If you don't confirm, the hardware will continue to the next bootable device (presumably the HDD).

kimi gao msft say:

Hi Ramy,

 

It’s not feasible to directly create a drop-down list of hyperlinks, but there is a workaround to make a drop list for hyperlinks.

Follow the steps below to achieve it:

1.Create a new work book and save it with a name (I have created a workbook named droplist). Create several worksheets. (I will create a drop list in sheet 4).

 

2. Choose a column, click Data>Data validation>choose List in Allow>select the range>OK to create a drop-list.

3. In cell B2, enter the formula as below:

=IF(B1="Sheet 1",HYPERLINK("[droplist.xlsx]Sheet1!A1","linktosheet1"),IF(B1="Sheet 2",HYPERLINK("[droplist.xlsx]Sheet2!A1","linktosheet2"),IF(B1="Sheet 3",HYPERLINK("[droplist.xlsx]Sheet3!A1","linktosheet3"))))

4. We can click on the hyperlink on B2 when we change the drop-list. For example, if we choose Sheet 1 from the drop-list, the B2 cell will show the hyperlink of Sheet 1.
