john w. vinson say:

Will there ever over the course of the use of this application be a SiteC?

Are you absolutely certain there won't?

If there will, a much better - and easier to use! - table structure would be:

Sites
   SiteID <primary key>
   Description

Locations
  LocationID <primary key>
  Description

Distances
  DistanceID <autonumber primary key>
  SiteID
  LocationID
  Distance

Or you could do away with the Sites table altogether, and just make each Site a "privileged" Location; this would allow you to store the distance from any location to any other location.

Given this structure, you wouldn't need any IIF or DLookup at all - just a query with criteria for the site and the location.

In any case your Trips table should not contain a Distance field at all - it's redundant, since it already exists and can be readily looked up at any time in the Distances table. With your current structure it's doable but difficult - your choice of two mutually exclusive yes/no checkboxes on a form makes it hard to construct a simple and efficient DLookUp.

techfreak_ say:

Echo123 is considered a person, not a bot in the Skype directory. If it is added to a group conversation, then it will exist in the conversation until it is removed. Its existence in a group does not affect the normal use of the test account.

In regards to the image you supplied, you need to type in echo123 exactly with no spaces. Inputting in a variation like a space, will bring up any global contact or local group chats/conversations with echo and 123 in their display names.


If other contacts in the Skype directory called themselves echo123 in their display name, those will still show up under more people. That is how the Skype directory works.  If you added contacts to echo123, it was not actually added to the echo123 account. A new group chat/conversation was created with those that were added. If that bothers you, you can go to the group chat and remove everyone except yourself from that group (view profile). If no group chats/conversations currently exist, then you have nothing to change.

After removing everyone from the group except yourself:

Notice that the real account will have the Skype logo next to it with the exact name echo123, as seen below and above. Any other variation is not the Echo / Sound Test Service account.
