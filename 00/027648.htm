Thank you for contacting Microsoft Community. As per the issue description you want both the table and the chart to be present in word so that when you enter the data in the table it will be updated automatically in the chart.

 

It is not possible to automatically update the chart when the table is updated when you have inserted both table and the chart in Word document.

 

You may follow the steps mentioned so that both the table and chart will be displayed in Word and you don’t have to switch between applications:

·         Open Word.

·         Go to Insert tab, under Text group click on Object.

·         Under Create New select Microsoft Excel Binary Worksheet and click Ok.

·         In the Excel sheet inserted in Word enter the table values and go to Insert tab, select appropriate chart under Charts group to insert the chart.

Note: When you edit the table in the Excel sheet inserted in Word it will not automatically update the chart. You have to highlight the chart and extend the chart table selection for the table entry.

 

I hope the above suggestion helps. If you need further assistance on any Office related issues let us know and we will be glad to help.
