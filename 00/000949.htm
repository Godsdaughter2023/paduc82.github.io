Microsoft removed this saying that the transparent visual blur effects were bad for power life (batteries would run out while trying to compute all the correct blurry glass effects).  The actual technology is still there: the Aero underlying system is called DWM (Desktop Window Manager) and it's still there, it just doesn't do the transparent class blur anymore.

If you want the glass blur and don't care about battery life, you can download a program like Window Blinds to "skin" your operating system.  Window Blinds is made by a Microsoft partner company StarDock Inc, which makes many tools for Windows.

I can't speak for them, but historically Microsoft is a platforms company.  There's so many capabilities in the OS but they don't "light it up".  For example the entire theme system from Windows XP upward was actually a joint development and they licensed the rights to include the themes from the company that build some of the technology. That's why Windows XP only had a few stock themes and you couldn't make your own unless you used the underlying API's to develop your own customization program (companies like StarDock took this to their advantage and instead of just making their own skin, they made an entire "style your own PC" software called WindowBlinds that uses the capabilities Microsoft built.

Basically Microsoft doesn't build every possible function, they just make sure the architecture is there so others can build upon it.  Microsoft doesn't want to support and patch code for extras, they're basically as minimal as possible (which is a stark contrast to the days of Vista's "Ultimate Extras" and "we can code everything!" attitude -- but then, we all saw how successful Vista was lol).
