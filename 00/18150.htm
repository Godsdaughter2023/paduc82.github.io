Does Word have something like Master Slide in PowerPoint? - Microsoft Community

a. user say:

Does Word have something like Master Slide in PowerPoint?

You know how in PowerPoint you can create a master slide – background graphics, font style and size – and then none of that can be edited from the normal slide view?

Is there something like that for Word?


The only thing that is close is headers and footers, however I need a graphic to spread across the entire page.

So I want my client to be able to open up a Word file and they get this cool graphic in the background, but there are two text fields that are live that they can type in. The font style and size is already dictated in the document.

Thanks!

bob jones aka: cybertaz say:

What you are aiming for falls into the category of graphic design. Word for mac really is not equipped to easily handle this type of project - simple as it may sound :-)

In a standard Word document or template:

To have the image display on-screen at full intensity it must be placed in the text layer of the document. That leaves it open to be altered by any user who opens the document.

To deter access to the image it must be in the Header/Footer or inserted as a Watermark, but it will not display with full intensity although printing would not be affected.

One way to avoid that is to design the document as a Form. The general process is as follows seasoned to your own taste:

  • The document first needs to be set up with 0 margins for all margin values including Header & Footer margins. Also, the Paper Size needs to be specified as US Letter Borderless.
  • For the text you'll need to position either;
  • A 2-cell Table, or
  • 2 separate Table Cells, or
  • 2 Frames (FWIW, my preference)

Note that drawn Text Boxes cannot be used because Text Box Form Fields cannot be inserted into them.

  • Insert a Text Box Form Field into each cell or Frame
  • When you insert the image, set it's Text Wrap to Behind Text. Specify the image dimensions as necessary. Use the Align tools set to Align To Page then use Vertical> Align Top & Horizontal> Align Center
  • To apply Protection, go to Tools> Protect Document & check the Protect document for box, then choose the Forms option. In that way you can include a password. If you just use the Protect Form button on the Developer tab there is no opportunity to create a password, so any user can simply turn the Protection off.
  • Save the document using the Word Template (.dotx) file type if the user will need to repeatedly issue documents using the same template.

How do i move a font from Font Book to Word 2011? - Microsoft Community

sarah..bee say:

How do i move a font from Font Book to Word 2011?

I've downloaded a font from another website but can't figure out how to move it from font book to word. Please Help!
bob jones aka: cybertaz say:
Fonts do not get 'moved' to applications. Properly installed, valid fonts are available to all programs having access to the Fonts folder in which the font is installed. However, not all fonts are usable by all programs -- and Word is very finicky. It's possible that the font is not designed in a manner that makes it usable by Word.

If the font is available in other program's it apparently has been installed properly. The next steps:

  1. Quit all programs
  2. Launch Font Book to run its Resolve Duplicates & Validate Fonts routines as described here:  Font Weeding.
  3. Run Disk Utility to repair disk permissions on your HD
  4. Restart your Mac after a brief Shut Down

If the font still does not appear in Word's Font list it may simply not be a usable font. Not knowing any details about it makes it impossible to be sure why, but you may want to contact the source for more insight.

Forgive me if this is info you're already familiar with, but even if the font can be used it may not work out the way you expect :-) If a document is displayed on a different computer where the font used is not available, the document may very well not be viewed or printed as intended.

Regards,
Bob J.

Finally, after testing different solutions that I found in Google during the last hours, the only one that worked in my case was next one:

1) right click in the problematic inbox folder inside Outlook

2) go to IMAP folders

3) click the Subscribed tab and then Query

4) a list of folders should appear now so you should have the problematic inbox in this list

5) click the inbox and then select Unsubscribe

6) click OK

7) go to Send/Receive menu and click the Work Offline option

8) close the Outlook application (previous command is needed in order to avoid Outlook crashing due to the infinite synchronizing status with the problematic inbox)

9) relaunch the Outlook application (if it crashed in previous step, maybe you need to restart the PC before relaunching Outlook)

10) go to Send/Receive menu and unclick the Work Offline option to enable connection to server again (the inbox should still not synchronize until next steps)

11) right click again the problematic inbox that should not be in synchronizing mode anymore and select again the IMAP folders

12) in the All tab, click the Query and you should see the inbox in some unselected mode

13) select the inbox and click Subscribe

14) click OK and click F9 to resynchronize all your folders

15) you should see now the inbox working normally again

Hopefully this should help you too.

Good luck and let us know if it worked.

This has happened to me twice.  First time I had Microsoft assistance and we created a new profile but that was a hassle since I had to then re-do all my settings, export and import my calendar, and I lost all my auto-typed email addresses from anyone I sent a message to.  This happened again and so I did more research and tried more options.  One had you uncheck your Inbox for synchronizing and then setting outlook to be offline and then re-checking the inbox for synch, didn't work.  But this below worked.

  1. Go to to Send/Receive, then Send/receive groups, and Define Send/Receive Groups.
  2. On the right side of the send/receive groups window click on the Edit button.
  3. Uncheck the "Get folder unread count for subscribed folders".
  4. Under Received mail items check the option "Use the custom behavior defined below".
  5. In the next window below select INBOX and leave the other folders unchecked.
  6. Click OK and then exit out of your options.

Below are the command to get the members of all the Guest Users in an Office 365 tenant by means of Azure AD PowerShell cmdlets. The Command pulls the output for first 1000 users. Could you please help us to pull the command for all the Guest Users in an Office 365 tenant.

############################################################################################################################################
# Script that allows to get all the members of all the Guest Users in an Office 365 tenant by means of Azure AD PowerShell cmdlets
# Required Parameters:
#  ->sCSVFilenName: Name of the file to be generated
############################################################################################################################################

$host.Runspace.ThreadOptions = "ReuseThread"

#Definition of the function that get all the Guest Users in an Office 365 tenant through Azure AD PowerShell cmdlets and export the results to a CSV file
function Get-O365GuestsUsersThroughAzureAD
{   
    param($sCSVFileName) 
    Try
    {   
        [array]$O365GuestsUsers = $null
        $O365TenantGuestsUsers=Get-AzureADUser -Filter "Usertype eq 'Guest'”
        foreach ($O365GuestUser in $O365TenantGuestsUsers) 
        { 
            $O365GuestsUsers=New-Object PSObject
            $O365GuestsUsers | Add-Member NoteProperty -Name "Guest User DisplayName" -Value $O365GuestUser.DisplayName
            $O365GuestsUsers | Add-Member NoteProperty -Name "User Principal Name" -Value $O365GuestUser.UserPrincipalName
            $O365GuestsUsers | Add-Member NoteProperty -Name "Mail Address" -Value $O365GuestUser.Mail
            $O365AllGuestsUsers+=$O365GuestsUsers  
        } 
        $O365AllGuestsUsers | Export-Csv $sCSVFileName
    }
    catch [System.Exception]
    {
        Write-Host -ForegroundColor Red $_.Exception.ToString()   
    } 
}

$sMessage="Introduce your Office 365 Credentials"
