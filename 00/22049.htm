Local notebooks in Onenote are nothing else than folders in the Windows file system.

There are two possible scenarios:

Using RecordsetClone & Bookmark in a Separate Form to Refer to Another Form

Hello.

 

I currently have a form (frmALPHA) with a textfield and a button. When I type the ID (EventNum) into the text field and press the button, I am taken to the exact point on my form where my EventNum = the textfield. I use the code below and it works correctly.

 

Option Compare Database

Option Explicit

 

Public Sub GoToEventNum(EventNum As Long)

With Me.RecordsetClone

.FindFirst "[EventNum] = '" & EventNum & "'"

If Not .NoMatch Then Me.Bookmark = .Bookmark

End With

End Sub

 

Private Sub btnSearchEventNum_Click()

Me.GoToEventNum Me.txtEventNum

End Sub

You can use the same method, but you have to get the RecordsetClone from that form:

Public Sub GoToEventNum(EventNum As String)

Dim rst As DAO.Recordset

Set rst = Forms!frmALPHA.RecordsetClone

rst.FindFirst "[EventNum] = '" & EventNum & "'"

If Not rst.NoMatch Then

  Forms!frmALPHA.Bookmark = rst.Bookmark

End If

End Sub
