  • All of the file servers are XP
  • All of the other computers are XP except my Vista, Win7, and Mac machies.
  • Vista and Mac can see the file server (and allow me to sign in with user/password)
  • Vista can see all the XP computers, and allows me to remote connect to all of them.
  • Win7 can not see any of the computers.
  • Win7 can map network drive to XP file servers by server name
  • Win7 cannot see those servers from network view
  • Win7 can remote desktop by ip address (but that is not good as with DNS the ip addresses change regularly)

sirrond say:

Problem SOLVED!!

In my case it turned out that the two machines Vista and Win7 ended up connected to two different wireless devices.

Device 1 (with vista) was Bridged, and allowed the network to assign the IP and all network connectivity.

Device 2 (with Win7) was Nat, and assigned the IP bypassing network assignment. So of course, the Win7 machine could not see any of the hard-wired devices or any device on the internal "bridged" wireless device.

Changing the Win7 machine to the correct access point (bridged in my case) resolved the issue.  I can now see all of the other computers in this network.

Hi Siua,

 

From the issue description, it seems that you are trying to reinstall Windows 7 on your computer which currently has Linux Mint 15 installed on it.

 

This can be done only by performing a clean install of Windows 7. It means to remove an existing operating system and replace it with a fresh installation of Windows 7.

 

The most important thing to realize before performing a clean install of Windows 7 is that all the information on the drive where the current operating system is installed on, (probably the C: drive) will be destroyed during this process. So, you should back up all your important data before starting with the clean installation process.

 

I would like to know – Did you change the boot device order from the BIOS after inserting the Windows 7 disk in the DVD drive?

 

To perform a clean installation, you need to boot the computer from the Windows 7 installation DVD. To do this, you need to change the boot device order from the BIOS of the computer. Normally, you can access the BIOS by pressing the “Del” or “F2” button on the keyboard as soon as the computer starts with the “POST” process. (A message to press either of these keys (to access the BIOS) is normally displayed on the screen as soon as the computer is powered on.)

 

For more information on how to access the BIOS, you may contact the computer manufacturer or refer to the user manual that came with your computer.

 

Once you are able to access the BIOS, you need to change the boot device order to CD/DVD and then restart the computer after inserting the Windows 7 installation disk in the DVD drive.

 

Please refer to this article for more information: Start your computer from a Windows 7 installation disc or USB flash drive

 

Important: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Anleitung: Xbox 360 – Profilschutz (Roaming) - Microsoft Community

cool ferret say:

Anleitung: Xbox 360 – Profilschutz (Roaming)

Vorwort

Hallo an alle Zocker,

wie ihr vielleicht alle wisst gibt es das klassische „Gamertag retten“ seid dem neuen Update nicht mehr.

Es nennt sich ab jetzt nur noch „Gamertag herunterladen“ und befindet sich in der Profilauswahl eurer Xbox 360 Steuerung.

Das hat folgenden Hintergrund...

Es ist ja nun möglich sich an jeder Konsole mit Internetverbindung egal wo dies auch ist z.B. Freund, Freundin etc. mit seinem Profil anzumelden und dieses in vollem Umfang nutzen (auch die Spielstände siehe Cloud – Speicher).

Es geht nun darum, dass man wenn man sich an einer anderen Konsole sein Profil heruntergeladen hat, gefragt wird ob sich die Konsole das Passwort merken soll oder nicht, damit man nicht jedes mal beim Anmelden nach dem Passwort gefragt wird.

Ist man wieder zu Hause kann man seine eigene Xbox anmachen und das Profil auch weiterhin 1:1 nutzen.

Doch halt... die Xbox auf der man sein Profil heruntergeladen hat um es bei Freund, Freundin etc. zu nutzen hat ja auch noch vollen Zugriff drauf... hm was kann man da machen?

Das Zauberwort nennt sich Xbox 360 Profilschutz und versteckt sich in eurem Konto auf der Xbox – Homepage.

 

Anleitung:

Loggt euch auf der Xbox HP mit eurer Live-ID und Passwort ein und wechselt zu „Mein Konto“ was oben rechts auf der Seite steht.

Es folgt eine Übersicht in welcher ihr unten - rechts einen Punkt sehr welcher sich Xbox 360 Profilschutz nennt.

Klickt auf diesen und es folgt eine weitere Übersichtsseite.

Ihr seht dort die Punkte „Konsole welche das Passwort benötigen“ und „ Konsolen welche das Passwort NICHT benötigen“.

Unter den jeweiligen Punkten stehen jeweils ggf. Neue Konsole oder Alte Konsole und ein Datum dahinter.

Angenommen ihr habt eine Xbox zu Hause und eine bei nem’ Kumpel stehen auf welchen sich euer Profil befindet und möchtet nun das die Xbox vom Kumpel nichts mehr mit eurem Profil anfangen kann, so müsst ihr die Box unter dem Punkt „Konsole welche das Passwort benötigen“ mit dem neusten Datum suchen und rechts den Button „Profil-Download erforderlich“ klicken.

 Wenn der Kumpel nun versucht sich auf das Profil einzuloggen bekommt er die nette Mitteilung, dass dieses Profil auf dieser Konsole ungültig sei und er es erneut herunterladen müsste wofür er ja Live-ID und Passwort braucht was er ja hoffentlich^^ nicht hat.

 

Anmerkung zum Profil herunterladen:

Wenn ihr das Profil erneut heruntergeladen habt (davon ausgegangen es befinden sich alle Daten eures Avatars auf der Festplatte) müsst ihr nichts mehr einstellen und alles ist wie vorher.

Schaut höchstens mal in die Datenschutzeinstellungen da gibt’s ein paar neue Punkte.

 

Fazit:

Wenn man das System mal verstanden hat ist es echt ne’ feine Sache wobei ich mich Anfangs nicht getraut habe es auszuprobieren.

Zur Not ist der Support ja auch noch da.

Sollte ich hier etwas falsches verzapft oder gar vergessen haben bitte ich die Moderatoren dies zu ergänzen weil mit editieren ist es ja dann zu spät.

 

Liebe Grüße und frohes Zocken.

Dann auch noch eine Anmerkung zur Profilschutz Seite:

Da hat Microsoft wohl wieder den Praktikanten an die Deutsche Übersetzung gesetzt. Die musst man also nicht selber löschen.
