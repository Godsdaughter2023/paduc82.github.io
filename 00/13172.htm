Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
  Dim dataWS As Worksheet

  Set dataWS = ThisWorkbook.Worksheets("YourSheet Name")

When i print it says spooling - printing

when i try printing with microsoft word, it says spooling - printing and doesnt print, i can print with any other program but only this one spooles, help me!!!
somasundaram pandiaraj say:

Hi,

     You could try the following steps to overcome the spooling message while printing.

1)      Print the document without being spooled. To do that,

a)     Click “Start à Control panel à Devices and printers”

b)     Right click on your default printer and click Printer Properties.

c)     Click “Advanced” on top and then select the option “Print directly to the printer”.

2)      Try turning the spooling service off.

a)      Click “Start à All programs à Accessories à Run”.

b)      Type in CMD and click OK.

c)      Type in net stop spooler and press Enter on the keyboard. Now check if you are able to print a document.

(Note: To turn the print spooler service type in net start spooler in the same command prompt window)
