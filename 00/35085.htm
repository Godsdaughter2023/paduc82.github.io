Hi,

WMP uses no longer directshow decoders so it is not so easy as before.

But there is a way, I have done it.

WMP uses media foundation decoders and all you have to do is:

1.- Install an MP4 splitter like Haali (http://haali.su/mkv/). The splitter splits the mp4 file into its components (video, audio, etc.)

2.- Install a video/audio decoder like ffdshow (http://www.xvidvideo.ru/ffdshow-tryouts-project-x86-x64/)

Up to this stage nothing has changed for WMP. It still uses media foundation.

3.-Install Preferred Direct Show filter tweaker (http://www.codecguide.com/windows7_preferred_filter_tweaker.htm)

One of the added benefits of this filter tweakers is that you can enable hardware acceleration for mp4 and mkv's so you can put to work your gpu instead of the cpu.
