Hi


Let me inform you that even though it is not recommended and the latest build will install automatically in the future, we can use the go back feature to roll back to the previous version if it has not been more that 10 days of installing the anniversary update.

 

  • Navigate to Settings > Update & security > Recovery.

  • Under Go back to an earlier build section, click the Get started button.

  • If you’re asked, provide feedback why you’re uninstalling Anniversary Update and going back to the previous build of Windows 10.

  • If you see the Check for updates? Screen, click No, thanks button to proceed further.

  • Next, you will see Don’t get locked out screen with “If you used a password to sign in to your previous build, make sure you know it” message. Put simply, if you used a different password for your account before installing Anniversary Update, make sure you remember it. The new password (if set after installing Anniversary Update) can’t be used to sign in to your previous build.

  • Finally, click Go back to earlier build button to restart your PC and begin restoring your previous build of Windows 10.

Hope this information is helpful.

Thank you for bringing this up to Microsoft Community. There are many possible contributors to this kind of issue. Software problems as well as hardware degradation can be a factor. Have check if there any apps running in background as well? Unnecessarily running programs can cause slow system behavior and affect overall stability of the Device.


I suggest that you run the Surface Diagnostic Toolkit to check for possible software errors or you may download drivers and firmware to get the latest firmware updates Apart from that, please run the Power Troubleshooter to see if the current power plan is not effective for your Surface and follow the other steps as well. Here's how:

Step 1: Run Power troubleshooter.

  • Open Settings.
  • Go to Update & Security.
  • Choose Troubleshooter.
  • Select Power.
  • Click on Run the troubleshooter.

Step 2 Check Battery Saver Settings.

  • Open Settings > System > Battery

  • Check that Turn battery saver on automatically is enabled and that it is set to at least 20%.

  • Check that Lower screen brightness while in battery saver is also enabled.

Step 3 Check app usage.

  • Open Settings > System > Battery and then select See which apps are affecting your battery life.

  • From Time period, select 6 hours, 24 hours, or 1 week depending on which period is closest to the your reported problem.

  • From Show, select Apps with usage.

Look for apps that have high battery consumption. If there are apps with high usage that the customer is not using, you should close them.

If the apps start automatically, open Task Manager and then select the Startup tab. The you can disable any apps that they don’t need to start automatically.

Lastly: Reset/Fresh Start

  • Fresh Start is the least intrusive form of reset. You will not lose any files or personal information, you will only need to reinstall desktop programs.
