  • Select ISO Image
  • Point to the Windows 10 ISO file
  • Check off Create a bootable disk using
  • Select GPT partitioning for EUFI firmware as the Partition scheme
  • Choose FAT32 NOT NTFS as the File system
  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete

sergent---01 say:

Hi shay0689,

 

Welcome to Nokia discussions board!

If the timestamp displayed for incoming messages is incorrect, the time or time zone configured on your phone is most likely incorrect. 

To fix this, you can set your date and time to "Automatic" or "Network Provided."

Go to Menu >  Settings >  Date and Time > Auto update of time > On. 

With this setting enabled, your mobile provider will set your phone to the correct time. If the incorrect time is displayed even with this setting enabled, this is an issue with your network. Please contact your mobile provider to address the issue or the Whatsapp team.
