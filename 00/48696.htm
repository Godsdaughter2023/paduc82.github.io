Set up your printer with Google Cloud Print

"i didnt open anything"

- If not, you have Steam setup to launch when you boot into Windows. This can be disabled either in Steam Settings

or in Task Manager > Startup tab.

It looks like you've ran two searches which Windows is holding in RAM as well. Not a problem.


Win 10 does it's best to prevent your RAM just sitting there looking pretty, so it loads a fair bit of stuff, even at idle.

If the remaining RAM looks to come up short when you start running something that needs a lot of RAM, such as a game,

Windows will free up more RAM for the game or other foreground program.

Note that if you run a game or other high RAM use program Win10 will often hold some of that data in RAM after you close

the game or program as well. For how long I haven't worried about checking.
