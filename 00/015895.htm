Please help, I am using Microsoft reader app to open my pdfs.

jay freedman say:

Use Word 2013 instead of the reader to open the PDF file (using the File > Open command, or Ctrl+O).

If the PDF was originally created from editable text and graphics, rather than from a scan, it will open in Word as an editable Word document. If it was a scan -- meaning that the only things the PDF contains are pictures of the pages -- then the Word document will contain pictures of the pages. In either case, you can save from Word as a Word document (*.docx file). In turn, that can be inserted into another Word document, either by copy/paste or by using the Insert > Object > Text From File command.

If the contents of the PDF are just pictures of pages (which I suspect is the case if the reader doesn't offer a copy command), and you need to edit the text, then you'll need to use optical character recognition (OCR) to convert it to editable text. You can try using OneNote to do that, as explained in this article, or you can look for other OCR software.

Unless you have a printer that supports full-width bleeds, such as a photo printer, it doesn't matter what kind of file you try to print or what application you use, you won't be able to avoid the white portion - which represents the unprintable area reserved by the printer's manufacturer for paper handling - appearing on the printed page. ron de bruin say:

Both workbooks use a different date system

See the text from the article
http://msdn.microsoft.com/en-us/library/bb277364(office.12).aspx
 By default, Excel in Microsoft Windows uses the 1900 date system. However, many Apple Macintosh users use the 1904 date system. Dates in Excel are actually numbers. That is, if you enter 1 in an empty cell and format the cell as date, 1900-01-01 is displayed in the 1900 date system, and 1904-01-02 is displayed in the 1904 date system.

To change this setting in Excel 97-Excel 2003, on the Tools menu, click Options, and then click the Calculation tab. To change the setting in Excel 2007, click the Microsoft Office Button, click the Excel Options dialog box, and then click the Advanced tab. If you copy dates between workbooks, and both workbooks use a different date system, there is a difference of four years (1462 days) between the dates.

The following steps illustrate one way to compensate for this difference:
 1.Type 1462 into any empty cell in the workbook.
 2.Copy the contents by selecting the cell and pressing CTRL+C.
 3.Select the cell that contains the date you want to update.

Excel 2007 only:

On the Home tab, in the Clipboard group, click Paste, click Paste Special, select Add (or Subtract depending on which workbook you are pasting to), and then click OK.

-or-

Excel 97, Excel 2000, Excel 2002, and Excel 2003 only:

On the Edit menu, click Paste Special, click Add (or Subtract depending on which workbook you are pasting to), and then click OK.
