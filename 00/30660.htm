xircal say:

You can try by all means, but I'd be surprised if it's present.

What you can do instead is to create a System Image. That has the advantage that it not only contains the operating system, but everything else which is on the machine such as all your personal data, all the drivers and all the programs you have installed.

I create one of these myself every month after each Patch Tuesday. I've only ever had to resort to using it once, but it worked flawlessly.

To create one, you'll need an external hard drive with at least 50GB free space available. You can use the rest of the storage space for whatever you wish.

To create a System Image, proceed as follows.

  1. Attach your external hard drive and then right click the Start button and go to Control Panel. If you have the Category view enabled, change it to "Small icons" via the link top right.
  2. Click the link to File History next.
  3. In the menu which opens, click the link in the bottom left hand corner to "Create System Image". I've attached a screenshot of that.
  4. In the next menu select your hard drive which the tool identifies and then click Next.
  5. In the following menu checkmark the EFI partition and the C: drive and then click Next.

Once the process to create a system image starts it usually takes about 20 mins to complete. The folder it creates is called WindowsImageBackup.

If you decide at a later date to create another one, but don't rename the first one, the tool will overwrite all the files. That takes about 45 mins. Personally, I find it easier simply to rename the first image to XWindowsImageBackup and then let the tool create a new one.

I created an illustrated guide on how to restore to a System Image which you'll find here marked as Answer:

The icon for the webpage used for favorites/bookmarks ("favicon") is stored in the temporary internet files cache. If it gets cleaned (using Disk Cleanup, CCleaner, or any other cleanup utility), or if the user clears their internet cache files, the icons are gone and will show as the default icon for .url file types (if Chrome was your default browser, they would all show as the chrome icon instead of the globe for example).
