No, that is just to try and track down the cause! What it tells us is that one of your startup files is causing the problem.

So, restart Excel normally, and check if you have any add-ins loaded. If so, disable them all and see if the problem goes away. If it does, re-enable one at a time and see if the problem recurs.

If you don't have any add-ins, or if disabling them all doesn't help, the most likely issue is a corrupt .xlb file. On Win7 you'd find this at C:\users\your user name\appdata\roaming\microsoft\excel and it will be named based on your Office version (eg Excel14.xlb for Office 2010, Excel15.xlb for 2013). You need to have hidden folders visible to locate it. Once you find it, close Excel, and either rename the .xlb file (eg Excel14.xlb.old) or move it somewhere else temporarily, and then restart Excel and see if the problem has gone.
