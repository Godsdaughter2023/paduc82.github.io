julie fka profjulie say: Check ActiveSync's settings:

On your desktop computer, open up ActiveSync, select File > Connection Settings
About 1/2 way down you'll see "This computer is connected to:" with a box underneath that.  By default that is set to "Automatic", but you can select the down arrow and try the other two options there (Work Network & the Internet).  See if one of those options works for you.

Also, the computer's firewall could be blocking the connection (it does on my computer), so I disable the firewall when I want to use the passthrough connection.
