If you put a { DOCVARIABLE FileName } field where you want the filename to appear and then run the following macro, the field will display the filename without the extension

With ActiveDocument

    .ClosePrintPreview
End With

=Environ("username")

IF Me.NewRecord Then

   Me![Created By] = Environ("username")
