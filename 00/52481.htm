a. user say:

The built-in Administrator is not a user account, so it doesn't work like a user account. The built-in Administrator is a system account that is your lifeline when you can't sign in to any of your user accounts.

If your computer does not already have a user account with Administrator permissions, use the built-in Administrator to create one. Use the (new) Administrator account to create a 'standard' user account - that's what you should be using for day-to-day computing.

Once this is done, move any files you have stored in the built-in Administrator to your standard user account, then restore the built-in Asministrator to the way it was when you first started to use it. Then hide the built-in Administrator.
