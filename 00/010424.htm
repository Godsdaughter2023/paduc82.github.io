stephen boots say:

These are not "errors" as such, but normal return codes, despite the developer's use of the uppercase ERROR in each line. :-)

0x00000005 is an access violation, meaning that the scanner could not attach to the running process.

0x00000020 means "The process cannot access the file because it is being used by another process.

0x0000012B means STATUS_FILE_LOCKED_WITH_WRITERS -- yet a variation of a process not being available for scanning

0x0000054F means either error copying file or internal error, which isn't helpful since it appears in a line item for a service.

In any event, why are you scanning with the Safety Scanner? What is your installed antivirus? If Defender is active and running, the Safety Scanner is redundant as it uses the same detections and engine. If you are using a 3rd party antivirus program, then scanning with the Safety Scanner for a 2nd opinion is valid, but it will often return errors such as those you posted due to conflict with the resident antivirus.

-steve

I'm afraid that you can't avoid Bing when inserting pictures in Word.

However, if you want to prevent OneDrive from being the default in Office, do the following: In Word, click File | Options | Save. Click the option "Save to Computer by default." Clear the option "Show additional places for saving, even if sign-in may be required."

You may also want to select "Don't show the Backstage when opening or saving files"; after that, you can access the classic Open and Save As dialog boxes by using keyboard shortcuts: Ctrl+O for Open and F12 for Save As.
