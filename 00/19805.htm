john korchok say:

Many image editors can create transparent colors or transparency layers in a PNG file. Then import it to PowerPoint.

Within PowerPoint, you can select and image, then choose Picture Tools Format>Color>Set Transparent Color to set a single transparent color. This works best with line art or graphics with areas of flat color, like logos.

For photos, select the photo, then choose Picture Tools Format>Remove Background. Make any corrections to the Background Removal by drawing with the Keep and Remove tools, then click on Keep Changes when you're done.
