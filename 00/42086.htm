Hi thengtientek,

Go to Start then search "Your account picture and profile settings" without quotes.

There should be a new item named just that, click on it.

This is only if you want to change the picture.

If you want to remove the account picture, right click the Start button, then click Run, then enter the following: %APPDATA%\Microsoft\Windows\AccountPictures

Delete every picture in there, and reboot. Hope that helps.
