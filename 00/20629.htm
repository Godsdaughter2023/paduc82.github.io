hansv mvp say:

Here is a macro you could run:

Sub SelectIt()
    Dim cel1 As Range
    Dim cel2 As Range
    Set cel1 = Range("A:A").Find(What:="(Question-", LookAt:=xlPart)
    Set cel2 = Range("A:A").Find(What:="Solution:)", After:=cel1, LookAt:=xlPart)
    Range(cel1, cel2).Select
End Sub

brent eubanks say:

This was a terrible design decision on Microsoft's part.  I use a trackpad and I curse Excel at least a half dozen times a day because this unwanted feature intrudes into my workflow and suddenly I am looking at 10000% or 5% zoom.

Poor show, Microsoft.  Very poor show.

That said, I think I have found a workaround.

First, you ENABLE the "zoom in intellimouse scroll" option discussed above.

Then you install and configure a X Mouse Button Control (free, here: http://www.highrez.co.uk/downloads/XMouseButtonControl.htm)

Define a profile for Excel, and tell it to replace WheelUp with CTRL+WheelUp.  Ditto for WheelDown.

Should look like this when you are done (the middle button doesn't have to be disabled; that's up to you).

This causes X Mouse to intercept wheel up/down and replace it with CTRL+Wheel Up/Down.  Excel then interprets this as normal scroll, because you enabled the "zoom on scroll" feature, so it treats CTRL+Scroll as just scroll.  

Convoluted and stupid (thanks again Microsoft!) but it works.  Unfortunately, it does not work on Word because (as far as I can tell) the "zoom on scroll" is not a selectable option on Word.
