2. Highlight the screen you want to change (if you have multiple monitors) and change Orientation between Landscape and Portrait.

How to rotate a computer screen
On Windows PC (Windows 10)
Press "Windows", type "Display Settings" and hit "Enter".
(If "Display Settings" is not displayed, click on "..." icon located at the top right corner to expand the list and select "Settings" to filter search results)

I have more than one computer, can I install the same software on any of them?

You may install the software on any computer you own.

Note: Software that requires a product key for installation can only be installed on a single computer. The Microsoft Imagine license does allow for this type of software to be installed on up to two computers, but a second product key may be needed. In order to install software on a second computer, you must request another product key through the Support section of the Microsoft Imagine website. This system was designed in order to guard against abuse of the license.
