If the problem involves the use of a Desktop PC note that many PC's (not all) still have power at the USB ports when Shutdown and the power supply is not switched off or unplugged.

What your doing here is removing the hardware from the PC and then booting up the system with it removed and then waiting until the system is completely and fully up and running then essentially plugging in new hardware.

If that does not work the first time repeat the procedure doing a restart (not a shutdown and start) after booting up the first time, sometimes that is necessary for some reason. I've used this procedure a number of times with success.

If the drive is good and the data still intact it should work. Check in disc management if the drive still shows up after you boot up the PC before you plug the drive back in, if it is still shown, remove it and then do a restart and recheck.

The anniversary update was actually a full blown new OS install procedure, the new OS must recognize and install the removable hardware, this applies to USB sticks (flash drives), external disc players, etc., as well.

If you have access to another PC you can plug-in the portable external drive to it to verify that it is good and did not get corrupted while being plugged-in during the update.
