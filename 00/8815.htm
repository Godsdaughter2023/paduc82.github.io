julie fka profjulie say:

 

The way ActiveSync works is if data is on the mobile device but not on the computer, the data on the mobile device should sync to the computer.  The only time the sync rules apply is if the same data exists on both the mobile device and the computer and that data or some portion of that data is changed on both devices.  When syncing that data, ActiveSync would recognize that a conflict exists and apply the sync rules.

 

If you are syncing to a new computer and setting up a new ActiveSync partnership or if you delete your existing ActiveSync partnership and create a new one, Activesync should recognize on the first sync that data exists on the Mobile device and ask you how you wish to proceed with 3 choices:  1) Combine the data on both devices; 2) Replace the data on the mobile device with data from the computer; 3) Do nothing.

 

If you choose the 1st option (Combine), you may risk duplicating data on both the computer and the mobile device; if you choose the 2nd option (Replace), you will most certainly lose any data on the mobile device; and if you choose the 3rd option....well, I'd never choose that option because I want to sync the data on the two devices.

 

If you are in a situation where you want to copy the data from the mobile device to the computer, you should always start out with a new syncing partnership and you should always choose the Combine option.  When I have done this in the past, I have spent time cleaning up the data on my computer to minimize the amount of duplicates that might get created....I have even gone so far as to removing all the data on the computer.

 

Of course, if the data is only on the mobile device, you should always always always backup the mobile device before syncing to the computer just in case things do not go as planned.  Then if the data on the mobile device gets wiped, you can always restore your data from the backup file.

michaelryba say:

How do I subtract one date / time from another and get the answer in minutes?

I need to know how to calculate the difference between two date/time cells with the resulting answer in minutes. As an example, following are my two cell values:

2015/03/03 18:02:50

minus

2015/02/24 15:06:30

How do I accomplish this?

Thanks.

 

I have changed the title from:-

 

Date / Time Calculation in EXCEL

 

 - to:-

 

How do I subtract one date / time from another and get the answer in minutes?

 

jp ronse say:

Hi,

Say A1: 2015/03/03 18:02:50

B1: 2015/02/24 15:06:30

Enter in C1: =(A1-B1)*1440

EDIT after reading Bernard's reply.

You must be aware that multiplying the value by 1440 "overwrites" the original interpretation that Excel can give to a value.

=A1-B1 gives 7.122... expressed as days and parts of it.

If you need to keep that value for further calculations, you can also apply the format [m] to the cell, it will also display 10256 but the Excel time value is still in the cell for further calculations.

vijay a. verma say:

An alternative way -

=--TEXT(B1-A1,"[mm]")

If you don't need to worry about whether B1 is bigger than A1 or vice versa, use following which will always give +ve result -

=--TEXT(ABS(A1-B1),"[mm]")
