nevermind, figured it out.

Just in case someone wants to do something similar, here is a possible solution

Column A has the addresses

in Column B I ran this formula =FIND("@",A2) - This gives the number of characters before the @ sign. Could not find a function that gave me the number of characters after the @ sign

In Column C I used this =LEN(A2) to find the total length of the text

ossiemac say:

Hello Spike,

See if the  code example below will do what you want. I have assumed that you want the user's windows login name but I have included an alternative line of code where you can enter any string you wish for the default value for the InputBox.

Because you have mentioned the "Selection Change event ", I assume that you know how to install the event code. Please get back to me if you need further assistance with this. Also feel free to get back to me if not doing all that you want.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
   
    Dim strPrompt As String
    Dim strDefault As String
    Dim strTitle As String
    Dim varUser As Variant
    
    If Intersect(Target, Range("A:A")) Is Nothing Then     'Edit "A:A"  to column required
        Exit Sub        'Not required column
    End If
   
    strPrompt = "Press Enter to accept default user or enter another name." _
                    & vbCrLf & "Cancel to exit."
    strTitle = "Get User Name"
   
    strDefault = Environ("Username")    'Users windows login name
   
    'Alternative to previous line that gets the users windows login name
    'strDefault = "Users name"       'Can have a name other than the login Username as default
   
    Do
   
        varUser = Application.InputBox(Prompt:=strPrompt, Title:=strTitle, Default:=strDefault, Type:=2)
                                   
        If varUser = False Then
            MsgBox "User cancelled. Processing terminated."
            Exit Sub
        Else
            If varUser <> "" Then
                Exit Do
            End If
        End If
    Loop
   
    'Following line for testing only. Replace with code required after user enters the name
    MsgBox "User name entered is " & varUser
       
End Sub
