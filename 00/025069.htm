you can add a column with a formula, starting in A2 and copied down

Not quite. Icon sets are cannot be configured to compare the previous cell with > and = and <

The only comparison operators are >= and >. Also, it is not possible to reference the previous cell in icon sets. By default, Icon sets will highlight in which bandwidth cells fall. This is quite apparent in the dialog

hansv mvp say:

Apparently, the contents of the locked cell have a different meaning  between locations. Let's take euros against US dollars:

1 EUR ~ 1.39 USD and 1 USD ~ 0.72 EUR

If the locked cell contains 1.39 (the value of 1 euro in dollars), an amount in euros is converted to dollars by multiplying by the value of the cell. E.g. 100 euros = 100*1.39 dollars.

If the locked cell contains 0.72 (the value of 1 dollar in euros), an amount in euros is converted to dollars by dividing by the value of the cell. E.g. 100 euros = 100/0.72 dollars.

If you want the formulas to work the same way for every currency, you have to ensure that the locked cell always contains the same type of factor, for example *always* the value of 1 local unit in USD, or *always* the value of 1 USD in local currency. Not one type for some currencies, and the other type for other currencies.
