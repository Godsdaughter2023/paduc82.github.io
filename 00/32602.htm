breckenridge st say:

what is the best network (home, public or office) for a computer based at home so others cannot connect or see private information and how do I secure it

al jarvi say:
what is the best network (home, public or office) for a computer based at home. how do I secure my computer from others outside my home from gaining my information


Is this a standalone computer connected directly to the public internet through your modem or do you have a wired/wireless router between your computer and the public internet?

If its the former choose Public as you network location type. That disables network discovery, file sharing, etc.

***************************

http://windows.microsoft.com/en-US/windows7/Choosing-a-network-location

http://technet.microsoft.com/en-us/library/bb878110.aspx

For the Public location type, the assumption is that the computer is directly connected to the Internet and therefore exposed to incoming malicious traffic from the Internet. Because of a possible hostile networking environment, the following default settings are automatically configured for networks with the Public location type:

• Windows Firewall is turned on

• Network discovery is turned off

• All forms of file and printer sharing are turned off, including file sharing, printer sharing, public folder sharing, and media sharing

If you change these default settings, they will be applied to every network with the Public location type.

****************************

If your behind a wired router and have more than one computer but don't want any type of file sharing access on that particular PC use the public network location. Additionally don't setup a HomeGroup, which is not available in a public network type anyway AFAIK.

If your behind a wireless router you can either disable the wireless radio on the router, note this is not available on all routers so check the users manual for help, or a better solution is to use strong encryption like WPA2 or at a minimum WPA with a long random ASCII key to keep unwanted computers from connecting to your network. Personally I use WPA2-Personal, aka WPA2-PSK [AES], and 63-character random ASCII keys to protect my two home wireless networks.

http://theillustratednetwork.mvps.org/LAN/SoHoWirelessSecurity.html

As far as sharing files or not on your home network see these links for help. In general I would select the Home network location type if your behind a router (wired or wireless) and want to selectively share files/folders and/or printers. If you absolutely don't want to share anything select public.

http://windows.microsoft.com/en-US/windows7/File-sharing-essentials

http://windows.microsoft.com/en-US/windows7/Share-files-with-someone

"Right side"?  That's where the Charms bar is.   There is a Start charm in the Charms bar but to actually use it would mean that you were both inadvertently "swiping" from the right as a Touch user (e.g. moving to the left while starting beyond the right edge of the screen) AND simultaneously clicking the Start charm.   The Start charm does have default keyboard focus so it is possible that instead you would be somehow sending an Enter once the Charms bar was rendered and given keyboard focus.

 

All this would suggest that you have a problem with your Touchpad.   Then a solution would be disabling your Touchpad (however your manufacturer provides for that use case or at least disabling its Touch gestures) and using a more conventional mouse (e.g. a wireless one) separately.

 

BTW I don't normally use my Touchpad since my screen has Touch support but while researching a reply to a similar question recently I had another look at its Touch gestures.   I think it would take about a week long course to learn the functionality and master it.   Just reading the documentation left me believing it would be quite likely that I could cause myself more confusion than convenience by trying to enable all its features.   I feel the same way about my Tablet's Pen gestures but have left them enabled because so far having them enabled has not caused me any grief.

Yes, dual boot of Ubuntu and Windows is very sensitive in order of things.

Windows OS should be installed first, because its boot loader is very particular and the installer tends to overwrite the entire hard drive.

GRUB is the boot manager installed in Ubuntu by default. GRUB is an open source boot manager that install the main parts of the boot loaders inside Ubuntu. This means Ubuntu is independent and avoids any need for writing to other operating systems
