Hi

You can download a Windows XP recovery console CD image (ISO) from here http://www.mediafire.com/?ueyyzfymmig

Once you have downloaded the file burn the image to CD. You can do this with a program like Imgburn. Imgburn can be downloaded from http://www.imgburn.com/index.php?act=download.

Once you have written the CD you must restart the PC and boot the PC with the CD. To do this you might have to change the boot order in the BIOS to boot with the CD Drive first. To enter the BIOS you usually press F2, F10 or DEL. (When you switch the PC on it will say press XX key to enter Setup/BIOS). Once in the BIOS search through the menu for something along the lines of "Boot Priority" or "First Boot Device" or "Boot Order". Once you have found the setting change it so that the CD Drive is the first Boot Device. Then Save the settings and Exit the BIOS.

Now you can boot the PC with the CD. If you get the message saying press any key to boot from CD - press a key.

After the CD has booted you will get a menu on screen. Press "r" to enter the Recovery Console.

Follow the instructions on screen. You will be given a list of Windows Installations. Press "1" to log onto the primary Windows Installation.

You will be asked to enter the Administrator password. Enter it or leave it blank if there is no password and press enter.

You should now be at the Command Prompt C:\Windows>

Now type in

chkdsk c:/r

press enter. Allow chkdsk to repair any disk errors it may find.

Once done type in

cd system32
