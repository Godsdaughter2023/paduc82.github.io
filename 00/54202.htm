System Restore points are either complete or not available. You never get "half" a restore point, e.g. due to lack of space. When insufficient space is available then old restore points are deleted to make room for a new point.

Port Forwarding - Microsoft Community

xb1r apollo say:

Port Forwarding

Login Information

  • Gateway: 192.168.1.1
  • Username: root
  • Password: 12345

 

 

Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select Add.
  7. Select the radio button for "Custom Server" and enter Xbox.
  8. Enter the IP address of the Xbox into the "Server IP Address" box.
  9. Enter 3074 into the "External Port Start", "External Port End", "Internal Port Start", and "Internal Port End" in rows 1 and 2.
  10. Select "TCP" from the "Protocol" drop down in row 1, and select "UDP" from the "Protocol" drop down in row 2.
  11. Select Save/Apply.

 

Disable Port Forwarding

 

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. When prompted, enter "root" for the username and "12345" for the password.
  4. Select "Advanced Setup" from the menu on the left.
  5. Select "NAT" from the sub menu, then "Virtual Servers" below that.
  6. Select the radio or check box next to the rule you would like to remove.
  7. Select Remove.

gerry c j cornell say:

File names in blue normally signify that the file has been compressed to save disk space. To decompress right click on the file in Windows Explorer, select Properties general tab, Advanced, uncheck the box before Compress contents to save disk space, click OK and Apply.

If all files in a folder and sub-folders are to be decompressed (or compressed) follow the same procedure on the top folder.
