If Windows is installed on an SSD drive, ReadyBoost can't be used because an SSD is already fast. So ReadyBoost cannot improve its performance.
However I can recommend you a few steps to fix the issue.
Method A:
Try to enable Superfetch/Sys Main service from services.
Note: For Windows 10 1803 version - the service will be Superfetch; for Windows 10 1809 - the service will be Sys Main
- Press Windows + R to open Run and type in services.msc. Press enter
- Search for the service Superfetch (Sys Main)
- Right click on the service and tap on Properties
- Type (select) Automatic under Startup
- Tap on Start button to start the service
- Click on Apply and press ok
Method B:
Format the memory card. Format the memory card to use it for ready boost. Follow the steps, please.
- Press Windows + E
- On the left pane, tap on This PC
- Choose the appropriate drive for the Memory card
- Right click on it and choose Format
- Select the check box Quick Format under format window.
- Tap on Start
