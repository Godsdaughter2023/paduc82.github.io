You can't turn AutoSave off permanently. However, you can turn it off on an individual file and Excel will remember to keep it off for that file each time you open it.

The ""Save autorecover information every ? minutes" option under File --> Save" is different from AutoSave. You turn off/on AutoSave for a file by clicking the button on top left of Excel.

Also, AutoSave only works for the new file formats, and if they're saved to SharePoint or OneDrive. So if you save elsewhere or in the old format, AutoSave will be off for those files as well.

yes I think we are saying the same thing.

As I understand it you have separate affiliation tables. Each table handling a different type of affiliation because different data is needed, correct? Each affiliation table will have a foreign key to link back to the primary key in the main record. Then each subform will be linked to each affiliation table PK to FK. This will allow you to show as many affiliations as the main record has. Each affiliation will be shown in the subform on each tab. So if a main record is a donor and a volunteer, then the donor subform will show the data for that person and the volunteer subform will show the data only for that person. Other subforms will be blank until you start entering data into them. When you start entering data, the FK will automatically be populated.

To say it another way, a subform works by displaying data related to the current record displayed in the main form. This is accomplished by linking the subform to the main form on a key field. You can have multiple subforms, each with the same linking fields because the subforms are bound to a separate table (or query). You have to tell Access to link based on the key field. But if there is no record in an affiliation table for the current main record, that that subform will be blank.

Microsoft Word booklet printing - Microsoft Community

gveneven say:

Microsoft Word booklet printing

How to continue printing booklet? I mean if I want to continue from 30th page, where can I change this page number?

suzanne s. barnhill say:

Printing specific pages of booklets has always been problematic. In theory, if you specify a page to print, Word will print that page and the facing page, plus (if you're duplexing) the pages that would be on the reverse side. So, for example, in an eight-page booklet, if you wanted to print page 5, you would get pages 4 and 5 back-to-back with pages 3 and 6.

In my experience, this almost never happens. Your best bet is to change the "Multiple pages" setting from "Book fold" to "2 pages per sheet." This will make absolutely no difference in the way the book looks to you in Word, but you can then specify the pages to be printed (in the example, you'd print 4,5,6,3).
