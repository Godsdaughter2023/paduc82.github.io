You can try the following:

Reset Windows Update:

1. Open notepad.

2. Paste the following code:

 net stop wuauserv
 cd %systemroot%\SoftwareDistribution
 ren Download Download.old
 net start wuauserv
 net stop bits
 net start bits
 net stop cryptsvc
 cd %systemroot%\system32
 ren catroot2 catroot2old
 net start cryptsvcf
 wsreset

3. Choose the "Save as" option; under the "Save as type" option choose "All files"and name it UpdateRepair.bat

4. Run as Administrator.
