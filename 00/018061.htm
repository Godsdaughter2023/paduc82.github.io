dirk goldgar say: If your form is bound -- that is, it has a recordsource that binds it to a table or a query -- then most of the text boxes on it are probably bound to fields in that table or query, in which case their behavior is governed by the properties of that field.  So then the data type of the field determines whether it can hold decimal places or not.  If the field is one of the whole-number types -- Type = Number, Field Size = Long Integer, Integer, or Byte -- then any decimal places you enter into it will be rounded off, because the field can't hold them.
