It's very unusual to have this error code being encountered when using Photo Gallery and at the same time getting some issues with Movie Maker. This can possibly happen if you are using an outdated version or some files under the Windows Live folder are corrupted.

To resolve this, we suggest you reinstall Windows Live using the latest version. Please follow these steps:

 

1. Restart your computer.

2. Optimize your Internet Explorer browser. We need to reset the settings to make sure the browser is working correctly. Follow this link:

How to optimize Internet Explorer browser

3. Disable your third-party security software. You may restore the service after updating Windows Live. Please also make sure Windows Firewall is enabled. It is necessary to have this up to avoid any errors.

4. Uninstall Windows Live. How to uninstall Windows Live Essentials. In case Windows Live Essential is not found on the Add or Remove Programs, proceed to the next step.

5. Delete the old Windows Live folder.
   5.1 Go to C:\Program Files (in case you are using a 64-bit Windows OS, go to (C:\Program Files(x86) instead) and delete "Windows Live" folder.
   5.2 Go to C:\Program Files\Common Files (C:\Program Files(x86)\Common Files for 64-bit Windows OS) and delete "Windows Live" folder.

6. Install Windows Live by downloading the full installer first. Follow this link:

http://windows.microsoft.com/en-US/windows-live/essentials-install-offline-faq
