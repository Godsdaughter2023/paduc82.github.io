macropod say:

Try:

Sub UdpateMMDateFields()
Application.ScreenUpdating = False
Dim Fld As Field
Const SearchDateChar As String = "_DT"
Const DateEscapeChar As String = "\@"
Const DateFmtDB2 As String = """\@ YYYY-MM-DD"""
With ActiveDocument
  For Each Fld In .Fields
    With Fld
      If .Type = wdFieldMergeField Then
        With .Code
          If InStr(.Text, SearchDateChar) > 0 Then
            If InStr(.Text, DateEscapeChar) = 0 Then
              .Text = .Text & DateFmtDB2
            End If
          End If
        End With
      End If
    End With
  Next
End With
Application.ScreenUpdating = True
End Sub

A B C D E F G H I J
  10 15 5 10
I want the formula to return 3 indicating that the 3rd column is the first one to contain a value.  I Also want a column to indicate that column 6 is the last column to contain a value.

Any help would be appreciated.
jeeped say:
A  B  C  D  E  F  G  H  I  J
        10 15 5 10

The following suggested formulas assume that your numbers are within the first row (e.g. 1:1).

For the first column that has a value>0 use an array formula like,
=MATCH(TRUE,1:1>0,0)
... which is an array formula and needs to be entered with Ctrl+Shift+Enter rather simply Enter.

For the last value in the same row try this non-array formula,
=MATCH(1E+300,1:1,TRUE)
The 1E+300 is just an incredibly large number.

If you want the cell address wrap the formulas in ADDRESS() like this,
=ADDRESS(1,MATCH(TRUE,1:1>0,0))     e.g. $C$1 
... and,
=ADDRESS(1,MATCH(1E+300,1:1,TRUE))     e.g. $F$1
