teylyn say:
Hello,

if you use shared workbooks, please be aware that they are notoriously unreliable, prone to corruption and data loss. Excel experts all over the world agree that using shared workbooks is a very bad idea. Excel was not designed for simultaneous multi-user access and handles this very badly.

In your scenario, since every person is just using one sheet, you could create ten different workbooks instead. Protect each workbook with a password to modify. There are different security levels for read only and for editing. 

If you need to consolidate data from the ten workbooks, that can be done with a reporting workbook that looks up values from all ten files. 

Depending on your data structure, you may also want to consider a database solution like Access or SQL. Both data entry and reporting can be built in Excel, but the storage and permissions will be handled with Access.

cheers, teylyn
