cmdrkeene say:

You can customize it, but the default settings if enabled are to create Restore points every week, and just before significant system events, such as the installation of a program or device driver. However the program or driver being installed needs to be "system restore aware" and know to ask for a restore point to be created.  And just remember that System Restore itself may be disabled entirely.

Restore Points created use the Task Scheduler engine to launch the creation process once a week.
