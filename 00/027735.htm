I will appreciate any advice as to how to phrase my query

bill mosca say:

First, your logic is wrong. You don't want records with Dogs AND you don't want records with Cats. If you use OR you are saying "give me records that don't have dogs" That would be ones with anything but dogs which would be cats. Then you want records that don't have cats. That would include records with dogs. See why you are getting both?

<>"Dogs" AND <>"Cats"

Put Is Null on the second line of the criteria or switch to SQL view and write it this way:

WHERE (Animals <>"Dogs" AND Animals <>"Cats") OR Animals Is Null

You can also write it this way:

WHERE (Animals NOT IN("Dogs", "Cats")) OR Animals Is Null

lukangel say:

Hello Scott,

We did a little more digging into this.  Unfortunately, the various technologies you have are not exactly compatible with Miracast.

In order to use Miracast with the projection menu in Windows 10, both the WiFi driver and Graphics driver must support the functionality.  The graphics device you have should be able to support Miracast (although it looks like the current driver on the system is out of date, a newer one may be available directly from AMD).  The WiFi device, however, was built before the WiFi direct was supported and does not appear to be capable of the technology.  Because of this, the PC is unable to perform Miracast.  Note that even if it were, interoperability with the RockBird sink via Miracast still may not work properly as it is not Miracast certified.

EZCast is a proprietary technology that uses a different protocol from Miracast.  The technology appears to leverage infrastructure WiFi rather than WiFi direct.  The RockBird device also seems to natively support the EZCast protocol in addition to Miracast.  As such, you are able still able to connect to the sink via the EZCast application using the EZCast protocol.  I'm glad to see that this functionality still works with Windows 10, however, since it is not supported natively, it will not integrate with the projection menu.

I hope this clarifies things for you.

Thanks,

Luke

Select C4:C28 on Sheet1.

Enter the formula

Click Format...

Activate the Fill tab.

Select a suitable fill colour.

Click OK, then OK again/
