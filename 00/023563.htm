jay freedman say:
The display in your first picture cannot be changed in any way. The Microsoft designers apparently thought that everyone enjoys looking at huge, space-wasting pictures with no usable information attached to them. :-(

Instead, customize the Quick Access Toolbar to add the "New Document or Template" button (from the All Commands setting of the Customize dialog). That button opens the New Document dialog box from Word 2003, which can be set to the Details view by clicking one of the three icons just above the Preview pane.

The slight drawback of this dialog is that it displays only templates that are stored in the C:\Users\<your name>\AppData\Roaming\Microsoft\Templates folder and its subfolders. By default, Word 2013 is programmed to save new templates that you create in the My Documents\Custom Office Templates folder, but the New Document dialog can't see that location. The solution is to go to File > Options > Save and enter the C:\Users\<your name>\AppData\Roaming\Microsoft\Templates folder in the "Default personal templates location" box. If there are already some templates in the Custom Office Templates location, move them to the Templates folder.

jay freedman say:

There is no default assignment of a shortcut key to the new-document dialog, but you can assign one yourself. If you want to use the shortcut all the time, you don't need the button on the Quick Access Toolbar.

  • Click File > Options > Customize Ribbon and click the Customize button in the bottom left of the dialog, which opens the Customize Keyboard dialog.
  • Scroll the Categories list to near the bottom and click All Commands.
  • Scroll to and click the FileNewDialog command. (Hint: click anywhere in the list and press the F key to get close, then scroll down a bit.)
  • Verify that the "Save changes in" box is set to Normal.dotm.
  • Click in the "Press new shortcut key" box, and press the keys for the shortcut you want. You could reassign Ctrl+N, which by default creates a new blank document based on the Normal template, or you could choose another combination.
  • Click the Assign button, and then the Close button. (Careful: if you click Close first, the shortcut won't be assigned.)
  • You should be prompted when you close Word to save the Normal template; click Yes.
