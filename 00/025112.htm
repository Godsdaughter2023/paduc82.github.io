Unfortunately MS does not make the sort of information you are looking for terribly easy to find. You can get bits and pieces.

For Office, the first place to look is File menu > Help command > On the far right side of the "backstage pane" (that is the official name of the whole screen wasting "dialog box" that automatically "drops down" from the menu) you will see the Office logo. Under that will be:

  1. a line for the activation status, ie "Product Activated", "Trial Product"
  2. a description of the bundle you have, ie "Microsoft Office Home and Student 2010" or "Microsoft Office Professional 2010"
  3. and the applications included in it.
  4. version number
  5. a hyperlink to "Additional Version and Copyright information" (the "old" About dialog)
  6. a hyperlink to MS Support
  7. the "Product ID"  (different from the Product Key
  8. a hyperlink to the Office License terms document

I don't have access to an Academic license, but I suspect that would be included in the item 2 description.

"Commercial", what is that?

OEM is the only license type that is explicitly displayed.  If you look in item 7, Product ID, Office OEM licenses will have the letters "OEM" in the 3 digits after the first hyphen.

jp ronse say: Hi All,

As Belgian user, Windows 7, Excel 2010, so same conditions as Benmie, I've tried all suggestions but it looks as we cannot force Excel to use a comma a listseparator. Changing the regional settings does not change Application.International(xlListSeparator). It always remains a semi-colon, even after restarting Excel.

Reversing the thousand and decimal separator is IMO not a good idea.

Hereafter already a basic sub to write CSV files.

HTH

Wkr,

JP Ronse

Sub WriteCSVFile()   ''' should ask for separator & textqualifier
   Dim lngFirstRow      As Long, strFirstColumn As String
   Dim lngLastRow       As Long, strLastColumn As String
   Dim strEvaluate      As String
   Dim strSeparator     As String
   Dim lngCount         As Long
   Dim fs               As New Scripting.FileSystemObject
   Dim f                As Scripting.TextStream
   Dim varFile          As Variant
   Dim varYesNo         As Variant
   Dim strTextQualifier As String
  
   strSeparator = Chr(34) & "," & Chr(34)
   strTextQualifier = Chr(34) & Chr(34) & Chr(34) & Chr(34)
  
   ''' get a filename to save
   varFile = Application.GetSaveAsFilename(, "CSV Files (*.csv),*.csv", , JPE_APP_TITLE)
  
   If varFile = False Then Exit Sub
  
   If fs.FileExists(CStr(varFile)) Then
      varYesNo = MsgBox(Prompt:="File already exists, do you want to overwrite?", Buttons:=vbYesNo, Title:=JPE_APP_TITLE)
      If varYesNo = vbNo Then Exit Sub
   End If
  
   Set f = fs.CreateTextFile(CStr(varFile), True)
  
   With ActiveSheet.UsedRange
      lngFirstRow = Replace(Split(ActiveSheet.UsedRange.Address, "$")(2), ":", "")
      lngLastRow = Split(ActiveSheet.UsedRange.Address, "$")(4)
      strFirstColumn = Split(ActiveSheet.UsedRange.Address, "$")(1)
      strLastColumn = Split(ActiveSheet.UsedRange.Address, "$")(3)
   End With
  
   For lngCount = lngFirstRow To lngLastRow
      strEvaluate = "RangeConcat(" & strFirstColumn & lngCount & ":" & strLastColumn & lngCount & "," & _
            strSeparator & "," & strTextQualifier & ")"
      strEvaluate = Evaluate(strEvaluate)
      f.WriteLine (strEvaluate)
   Next lngCount
  
   f.Close
   Set f = Nothing
   Set fs = Nothing
End Sub
