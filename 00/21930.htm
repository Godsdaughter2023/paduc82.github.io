Count Unique Values - Microsoft Community

otto moehrbach say:

Count Unique Values

I have a linear range (one column, consecutive cells) set to the variable TheRng.  I wish to count the number of unique values in that range using VBA.  All the values are strings.  How would I do this?  Thanks for your time.  Otto

hansv mvp say:

You could use a formula: let's say the data are in E1:E100. The following array formula, confirmed with Ctrl+Shift+Enter will return the number of unique entries:

=SUM(IF(FREQUENCY(MATCH(E1:E100,E1:E100,0),MATCH(E1:E100,E1:E100,0))>0,1))

Using VBA you could use the following as starting point:

Function CountUnique(rng As Range) As Long
    Dim col As New Collection
    Dim cel As Range
    On Error Resume Next
    For Each cel In rng
        col.Add Item:=CStr(cel.Value), Key:=CStr(cel.Value)
    Next cel
    CountUnique = col.Count
End Function

Use like this:

MsgBox CountUnique(Worksheets("Sheet1").Range("E1:E100"))

jllatham say:

Are there any empty cells within TheRng ??  As Chip noted, if there are empty cells it will not work properly.  It returns a #VALUE! error in such a case, and that could explain the error you got.

I did some testing of all options offered so far and I found the following:

FIRST: using a list of 1,440 entries with 9 unique entries and no empty cells in it, everything works as advertised and return the proper results.

But there are two that will fail if there are any empty cells in TheRng:

HansV's worksheet array formula will return #N/A, and as cautioned by Chip, his Evaluate() returns the #VALUE! error.

That kind of leaves us with my code and  HansV's.  Both work fine with all 1,440 entries filled and  Hans V's code runs in about 2/3 the time that mine does (i.e., about 30% faster).  BUT when we empty out some of the cells, it counts an empty cell as one additional unique value.

My vote would go to Hans V's code with a minor change so that it only returns the number of non-empty unique 'values' in the list:

Function CountUnique(rng As Range) As Long
     Dim col As New Collection
     Dim cel As Range
     On Error Resume Next
     For Each cel In rng
      If Not IsEmpty(cel) Then
         col.Add Item:=CStr(cel.Value), Key:=CStr(cel.Value)
      End If
     Next cel
     CountUnique = col.Count
 End Function

It's fast, and no errors creep in due to empty cells.  I'd say that HansV MVP gets the 'Answer' for this one, certainly got my Helpful vote although I like his other formula and Chip's a lot as long as you keep their limit in mind.
