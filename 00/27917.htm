If you want to, you can include some other heading level than Heading 1 with your captions. Your choice applies to all captions using that caption label.

stefan blom say:

Yes, for a given caption, Word picks up the numbering of a specific built-in heading style, only. You can't get around this without manual intervention, and that would be a nightmare to maintain, obviously.

pamcaswell say: I found this answer from HansV some time ago in this or another Word forum:

If you click in the chapter/section reference of the caption and press Shift+F9, you will see a field code like { STYLEREF 2 \s }


The number 2 is the number of levels to include. You can change this, for example so that it reads

{ STYLEREF 4 \s }, and your caption number would be 1.1.1.1-1.


Press F9 to hide the field code and update the value.


Note that once you change the switch, you cannot use the caption dialog to generate new captions because Word will change all captions of a type (say, figure) to match whatever level is specified in the dialog at that time.  (This is normally a good thing.)  Instead, you can keep all the levels in your clipboard and paste when needed or for longer projects, save each caption level as autotext.

I have used this a few times. Because my client insisted subchapter numbering, the alternative without modifying the fields was manual numbering. There were 20 and more figures in some subchapters, so I felt that using modified fields was the better choice.  It worked and my documents were stable. But at that point in the documents' production, I was the only person making changes to it.  We were delivering hard copies and pdfs, so there was also no danger of the client messing it up.

Indeed, that's "manual intervention." :-)  As you wrote, as long as there is only one editor of the document, it may work fine.
