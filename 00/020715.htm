doug robbins - mvp office apps & services (word) say:

For each new user of Word, a C:\Users\[UserName]\AppData\Roaming\Microsoft\Templates folder is created and populated with a Normal.dotm template that contains the Word default attributes.

For what you want to do, It would really be best to use a template other than the Normal.dotm and have your users use that template whenever you want a document that contains the styles that you want to be used.  Such a template could be saved in a Workgroup Templates folder, but better option is to use the distribution method detailed in the article "Distributing macros to other users” at:

http://www.word.mvps.org/FAQs/MacrosVBA/DistributeMacros.htm

suzanne s. barnhill say:
To add to what Doug has said, you should not be trying to create a Normal.dotm; only Word can do that correctly. Instead, you should modify and save an existing Normal.dotm. Once you've created one, you could distribute that to other users, but I would still recommend, as Doug does, that you instead create a specific document template to be used by your users, leaving their copies of Normal.dotm alone, as the Normal template is intended to be the location of individual personalizations and customization, and users won't thank you for wiping out any personal settings they may have made.

volunteer j say:
The "make transparent" tool on the picture toolbar within Publisher doesn't seem to work completely - it only makes part of the selected background color transparent and leaves a big smudge of the original background color in the picture - Can I get rid of all the background color?

===========================================
The smudge you describe occurs when the area you select
to be transparent has a color variation. The pixels that are
exactly the same color as the pixel you clicked will become
transparent and pixels of a different hue will not.

Best bet for a perfect shape with a transparent background
would be would to create a cutout. (think paperdoll)

The following article describes a way to create a cutout. It
can be a tedious process but should produce the result you
are seeking.

Microsoft Publisher 2003 / 2007
Fine-tune how text wraps around a picture

http://office.microsoft.com/en-us/publisher-help/fine-tune-how-text-wraps-around-a-picture-HP005115008.aspx
(scroll down to : Change the shape of the picture's outline)

When your cutout is satisfactory you can right click it and
'Save As Picture' to save it separately for future use. Saving
as .png or .gif will preserve the transparency.

A few alternative apps that can create cutouts are:

Windows Paint, IrfanView, Paint.NET.
