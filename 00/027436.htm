    Although the name is the same WordArt is a feature which has been totally redesigned from the ground up. Most of the instructions for the version in Office 2008 & prior will not apply.

    In order to scale text when resizing the shape you first must Transform the WordArt object... Select it, then click the Text Effects button [in the WordArt Styles group on the Shape Format contextual tab of the Ribbon]. Point to Transform & select any of the Warp styles.

    For more on working with the current WordArt features please see Help> Word Help, then search using the term wordart.
