Hi div-munroe,

There are several causes of Code 39 errors:

  • A required device driver is missing.
  • A required binary file is corrupted.
  • There is a problem with the file I/O process.
  • A driver that is referencing an entry point in another binary file is unable to load.

Uninstall and re-install the driver for the device to fix this issue.

Step 1: Uninstall the CD/DVD driver. Follow this step to do the same:

a.    Click the Start button, click Control Panel, click System and Maintenance, and then click Device Manager. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

b.    Expand DVD/CD Rom Drives-> Right click on the Tsstcorp CDDVDW TS-L633m ATA Devicedriver and click Properties.

c.    On the device Properties dialog box, click the Driver tab, and then click Uninstall.

d.    Restart your computer.

Step 2: Re-install the CD/DVD driver to the latest by visiting the manufacturer's website.
