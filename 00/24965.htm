Is there anyway to change the DPI Value of Individual Slide Objects (not the whole slide) that are saved as pngs.ie when you right click any object and select "save as picture..." Can this be changed in REG EDIT like the DPI value of whole slides that are saved as .pngs.

steve rindsberg say:

DPI isn't a measurement of image size.  It's a ratio between the number of pixels in the image and some arbitrary number of inches.

If you know the number of inches, then a 150 dpi image would be Inches x 150 pixels

To get the number of PIXELS you want in your images, you can use VBA:

Sub TestExport()
    ExportTheShape ActiveWindow.Selection.ShapeRange(1), 2500
End Sub

Sub ExportTheShape(oSh As Shape, lDesiredImageWidth As Long)

    Dim lDesiredImageHeight
    Dim lSlideHeight As Long
    Dim lSlideWidth As Long
   
    ' we work this out:
    lSlideHeight = ActivePresentation.PageSetup.SlideHeight
    lSlideWidth = ActivePresentation.PageSetup.SlideWidth
    lDesiredImageHeight = (lSlideHeight * lDesiredImageWidth) / lSlideWidth
   
    With oSh
   
        .Export "c:\temp\blah.jpg", ppShapeFormatJPG, _
            (lSlideWidth * lDesiredImageWidth) / .Width, _
            (lSlideHeight * lDesiredImageHeight) / .Height
   
    End With
   
End Sub
