SUMIF division - Microsoft Community

bellottir say:

SUMIF division

From Sheet 2, I'd like a formula to calculate the following using an array: If Sheet1 Column A = Sheet2 Column A AND If Sheet1 Column K > 0 THEN Divide Sheet1 Column K by Sheet1 Column L Can anyone help? Thanks. Bob
jim thomlinson say:

You can try this...

=IF(SUMIF(Sheet2!$A$2:$A$10, Sheet1!A2, Sheet2!$L$2:$L$10) = 0, "No Value", SUMIF(Sheet2!$A$2:$A$10, Sheet1!A2, Sheet2!$K$2:$K$10)/SUMIF(Sheet2!$A$2:$A$10, Sheet1!A2, Sheet2!$L$2:$L$10))

With AFA in Cell A2 on Sheet1 and your data in A2:L10 on sheet 2 this seems to return the number you were looking for. It also handles Divide by zero...

Hi.. I found a workaround to the problem:- I created a following macro, and assigned it Ctrl+Alt+V keys. No I just copy an image from the IE, Firefox, etc. And then I move to the Word document where I want to post the image. Then I simply press the Ctrl+Alt+V keys.. and Hey! The VBA macro-code is sampled below:

Sub Paste_Pic_Device_Independent()
'
' Paste_Pic_Device_Independent Macro
' Keyboard Shortcut: Ctrl+Alt+V
'
    Selection.PasteSpecial Link:=False, DataType:= _
        wdPasteDeviceIndependentBitmap, Placement:=wdFloatOverText, DisplayAsIcon:=False
    Selection.TypeParagraph

If you face problems in placement, you can replace the Placement:=wdFloatOverText with the following code:

Placement:=wdInLine
