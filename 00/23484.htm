a. user say:

Re:  identify conditional formatted colors

There is no direct way to determine a CF color for a cell.
The CF color is actually laid over any non-CF color in the cell.  The cell contains both colors.

Code must parse the CF formula that is applied to the cell and then determine if the CF condition has been met then determine what color is specified in the formula.  It is not an easy task to code and subject to problems.
Also, since there are thousands of colors available, colors that appear to be identical may not be.

       


Another approach is to your problem is to count those cells whose values meet one or more criteria.
The CountIf or CountIfs functions might work for you.

For Excel 2003 and earlier versions I have a free add-in (Determine Colors) that displays cell & border color names on the right-click pop-up menu.  It does both standard & CF colors.


[EDITED on July 29, 2015 -jbc]
To clarify: the DisplayFormat object (introduced in xl2010) will return the color number of a conditionally formatted (or normally colored) cell...  
'---
Sub CountCcolorR1(range_data As Variant, C_criteria As Variant, lngTotal As Long)
'Jim Cone - July 2015 - Portland, Oregon USA
'Counts the quantity of cells in the specified range that are the same color as the selected cell.
'Only valid on xl2010 or later excel versions.
'Standard and conditional formatted colors are counted.
'Merged cells are considered one cell.

 On Error GoTo Black
 Dim datax As Range
 Dim xcolor As Long
 Dim ycolor As Long
 Dim arrMerged() As Variant
 Dim N As Long
 Dim Ndx As Long
 Dim blnFlag As Boolean

 If TypeName(C_criteria) = "Range" And TypeName(range_data) = "Range" Then
   ReDim arrMerged(1 To range_data.Cells.Count, 1 To 1)
   Set arrMerged(1, 1) = range_data(range_data.Cells.Count).Offset(1, 1)

   xcolor = C_criteria.DisplayFormat.Interior.Color
   For Each datax In range_data.Cells
      ycolor = datax.DisplayFormat.Interior.Color
      If datax.MergeArea.Cells.Count > 1 Then
        For Ndx = 1 To range_data.Cells.Count
          If Not IsEmpty(arrMerged(Ndx, 1)) Then
            If Not Application.Intersect(datax, arrMerged(Ndx, 1)) Is Nothing Then
              blnFlag = True
              Exit For
            End If
          Else
            Exit For
          End If
        Next 'Ndx
        If Not blnFlag Then
          N = N + 1
          Set arrMerged(N, 1) = datax.MergeArea
          If ycolor = xcolor Then lngTotal = lngTotal + 1
        Else
          blnFlag = False
        End If
      Else
        If ycolor = xcolor Then lngTotal = lngTotal + 1
      End If
   Next 'datax
 End If
 Exit Sub

Black:
 MsgBox Err.Number & " " & Err.Description
End Sub

But the Run Macro action setting could call a macro in the current presentation and that macro could be something like this:

Sub CallMeFromRunMacroSetting()
  Application.Run ("ADDIN_NAME!Subroutine_Name")
End Sub

Substitute the name of your add-in for ADDIN_NAME
Substitute the name of the Subroutine in the add-in for Subroutine_Name
Be sure that the subroutine within the add-in isn't declared as Private
And the add-in must be loaded for this to work.

cgyq say:

Hover the cursor over Windows Media Player's Album column.  The word or phrase will turn blue and underline.

(Sometimes it won't work until you hover over another category, then you can hover over the desired category.)

Click on the bottom right corner of your touchpad.  If you have a touchbar,

click on the right corner.  I'm guessing this a right-click on a mouse.

A box will come up with only one option, "Edit" Cursor down and click normally.

The category may then be edited. 

This works for Album, Artist, Genre and date, which are all under the (left side) Album column in each album's page.

After I discovered this on my own, I was also able to search for album info, again by bottom-right clicking on the square 'album' icon just further to the left of the categories, and still inside the Album column. Searching for Album info helped me verify dates and reinstate most album covers, which is a helpful visual when searching for music.
