Ok, This is how I would do it.

To make this easier. Add another table:

tblSort

SortID (PK Number)

Add 12 records to this table numbered 1-12

Now create this query:

INSERT INTO tblBooking ( LoadID, BookingDate )
SELECT tblLoad.LoadID, [Enter Date] AS BookDate
FROM tblLoad, tblSlot;

To create this query open Query Design mode and switch to SQL View, then copy and paste the above SQL Statement into the query.

SELECT tblLoad.LoadID, Forms!formname!controlname AS BookDate.
