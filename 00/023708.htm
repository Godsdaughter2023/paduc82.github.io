Sub ExportPDF()
    Dim myFile As String, myPath As String
    Dim fldr As FileDialog

    
    myFile = "thisisatest" & ActiveSheet.Cells(1, 1).Text & ".pdf"
   
    Set fldr = Application.FileDialog(msoFileDialogFolderPicker)
   
    With fldr
        .Title = "Select a Folder"

Without the use of InitialFileName then default folder is the current folder (or last folder used for save in the current session). However, if InitialFileName has been previously used then the dialog box retains the last use of InitialFileName in the current session. Therefore, instead of removing  the command and allowing it to default to last time used, set it to a zero length string like the following so that it does not retain the previously used value.


    With fldr
        .Title = "Select a Folder for Saving File '" & myFile & "'"
        .AllowMultiSelect = False
        .InitialFileName = ""
        If .Show <> -1 Then GoTo NoPath
            myPath = .SelectedItems(1)
     End With


Other options:


Set the value to the path of ThisWorkbook like the following.


.InitialFileName = ThisWorkbook.Path


Set the value to the current directory. (Last folder to which a file has been saved or opened; Not necessarily the folder for ThisWorkbook.)


.InitialFileName = CurDir
