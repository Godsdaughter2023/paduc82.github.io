Excel makes formating the list easy but creating pages is difficult.  Word makes the columns easy to create but getting the format correct can be a pain.  I know nothing about how to write a VBA script to do any of this for me.  I don't mind the final list being in Excel or Word.
tasos_k say: Hi Bryan,
you can do that you want using a code..
let's say that data are on sheet1 (in  row1 are headings)
Export data in a new sheet in columns A,B,C,D.

1) press ALT+F11 to open VB editor.

2) Insert > module and paste the code below on the right

Sub abc()
Dim ws1 As Worksheet, ws2 As Worksheet
Dim r As Long, N As Long
Set ws1 = Sheets("Sheet1") 'change sheet1 name as yours
Set ws2 = Sheets.Add
r = ws1.Cells(Rows.Count, 1).End(xlUp).Row
ws1.Range("A1:B1").Copy Destination:=ws2.Range("A1:D1")
ws1.Range("A2:B" & N + 1).Copy
Cells(2, 1).PasteSpecial xlValues
ws1.Range("A" & N + 2 & ":B" & r).Copy
Cells(2, 3).PasteSpecial xlValues
Application.CutCopyMode = False
Columns("A:D").AutoFit


3) Close Visual Basic
