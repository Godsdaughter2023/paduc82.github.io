- Use the 'Backspace' key to zoom out

I J K L

# - Zoom Out
8 - Page Overview
9 - Go To

jacobsladder---01 say:

A-GPS works like this. Phone gets cell tower information and date and time and passes that to A-GPS server in the network using GPRS. The server then returns the data telling the phone which GPS satelites are visible and so phone GPS chip doesn't have to scan all codes and can get the first fix in 10 seconds or so. It is then the internal GPS that provides all the location data for maps and so the data transfer has only been for those first few seconds.

 

If you switch off A-GPS then the GPS chip has to scan for every code until it gets a fix, that can take anything upto 5 minutes.

 

In theory the phone should work with the A-GPS disabled, but it will take a long time to get the fix.
