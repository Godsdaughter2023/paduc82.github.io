rob koch say:

The answer to this depends upon the specific details of the writing/reading hardware and the particular type of CD or DVD involved.

Commercially produced Music CD or Video DVD formats are relatively limited formats when run on a dedicated player device, since these usually treat the data only as that and not a applications that can be executed.  However, if these same disks were placed in a PC drive instead, the enhanced capabilities of that device might be used to execute an application embedded on the removable disk, so an application produced and included on the disk by the vendor could be run by the PC if the code is both appropriate for that operating system and hardware (e.g. Intel x86).

However, since a commercial CD or DVD is typically created by pressing rather than being written with a writable drive on a PC, these disks can't have such programming added later as with a CD-R[/W] or other writable disk format.  So no, in your example a typical commercial CD/DVD is read-only, so it can't become infected with malware after the fact even when inserted into a CD/DVD ROM writeable type drive on your PC.

Any writeable format disk placed in a PC drive though could have anything written to it including malware up until the "finalization" process is performed on that particular disk and thus the directory format is written and locked as a result.  Though in theory it might be possible to override or ignore the finalization process and write additional data to the disk anyway, the standardized formats used in the finalization process would inherently become corrupted as a result, so this would typically generate disk read errors when inserted and read later in another PC.

Though the exact results are thus somewhat unpredictable, this wouldn't be an easy task for malware to perform, since it would not only need to write to the disk, but also modify the normal operation of the PC DVD device driver software as well in order to bypass the rules built into that software which protect a finalized disk from such an overwrite.

And even if malware intended for a Windows PC were written to a Music or Video CD/DVD format, it would have no purpose or functionality on a non-Windows PC device such as a commercial DVD player or even another computer operating system like an Apple or Google device.  That's because these use different code for their processors and calls within their operating systems, so the Intel x86 code targeted at a Windows operating system would be gobbledygook to these other devices.

In short, I would have no concern about any Music or Video only CD or DVD disks used in an infected PC as long as they were created and finalized at an earlier time on a known safe PC or by a commercial organization.  Only data disks created on the PC in question that hadn't been previously finalized or those specifically set up with a writeable format which allows adding data later would typically be at risk.  However, for safety's sake you might want to manually scan any writeable type PC data disks inserted in an infected PC on the chance that these could have had data added that included malware.
