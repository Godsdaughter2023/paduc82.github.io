gary zhu msft support say:

Hi Sounder,

 

The error indicates that you need to use Exchange Management Shell (EMS) from our on premises Exchange Hybrid server to enable Office 365 mailboxes for archiving. To enable all Office 365 mailboxes for archiving at once, please run the following cmdlet in EMS.

Get-RemoteMailbox -ResultSize Unlimited -Filter {(RecipientTypeDetails -eq 'RemoteUserMailbox')} | Enable-RemoteMailbox –Archive

 

After that, please force a directory synchronization via the following steps:

  1. Login to AD Connect Server;
  2. Open PowerShell;
  3. Type Start-ADSyncSyncCycle -PolicyType Delta, and then press ENTER.

 

Thanks,

Gary
