hansv mvp say:

Let's say the sheet with the data is named DataSheet, with headings in row 1, countries in column A (starting at A2) and data in columns B:E.

Select the list of countries A2:A..., then click in the name box on the left hand side of the formula bar, type Countries and press Enter. The list has now been named, and you can refer to Countries elsewhere in the workbook.

Select the entire data range from A2 to E... and name it DataTable using the same method.

 

On your form sheet, click in the cell where you want the dropdown. Let's say it is A2.

Activate the Data tab of the ribbon and click on Data Validation.

Select List from the Allow dropdown, and enter =Countries in the Source box.

If you wish, you can enter an instruction for the users in the Input Message tab, and a message to be displayed if the user enters a nonexistent country in the Error Alert tab.

Finally click OK. A dropdown arrow will appear in the cell and you should be able to select a country.

 

Next, click in the cell where you want to display the cards/landline rate.

Enter the formula

=VLOOKUP(A2,DataTable,2,FALSE)

Explanation: the formula tells Excel to look up the value of cell A2 (the selected country) in the first column of DataTable, and to return the corresponding value from the 2nd column. The last argument FALSE tells Excel that you don't want an approximate match but an exact one.

In the next cell, enter

=VLOOKUP(A2,DataTable,3,FALSE)

etc.

ron coderre say:

The standard approach to sum the same cells in contiguous sheets is:

• Insert a blank sheet BEFORE the sheets to be summed

• Name that sheet: First

• Insert a blank sheet AFTER the sheets to be summed

• Name that sheet: Last

• Put the Summary sheet anywhere before First or after Last, but not in between

 

Now, to sum cell C12 on the other sheets, use this formula:

=SUM(First:Last!C12)

 

Any sheets you put between First and Last will be included in the sum.

What is Power Pivot and Why You Should Care
https://exceleratorbi.com.au/what-is-power-pivot/
The answer to the question “What is Power Pivot?“.

Power Pivot is a tool that allows you to prepare your data in a way that Pivot Tables can use.  Power Pivot is actually
.  1.  Take data from one or more sources and combine it together into a single data source (this is called the data model).
.  2.  Enhance the underlying data so that you can create new derived concepts out of the data.  So if you have Sales $ and Cost $ in your source data, you can create Margin $ and Margin % and make these new concepts available to a Pivot Table.
.  3.  The new concepts you build in Power Pivot are infinitely reusable in your Pivot Tables.  In traditional Excel you need to create formulae in individual cells by pointing to other source cells.  If you want to reuse a formula you have created on another Sheet that is not 100% identical to the first sheet, then you effectively have to rewrite the formula.  This is not the case with Power Pivot.  You write the formula once in the data model and then it can be reused over and over in as many Pivot Tables as you like.
.  4.  Power Pivot compresses the source data and stores it as a database within the Excel workbook.  This  means that you can store 10s (even 100s) of millions of rows of data directly in an Excel workbook in a highly compressed and performant way, and distribute it to other people without them needed to have access to the source database.  Excel can “natively” only store 1 million rows of data. Excel it is not optimised for large database crunching work, Well before you hit the 1 million row limit, Excel will start performing very poorly, particularly if you have lots of VLOOKUPs etc. 

So Power Pivot is not “Pivot Tables on Steroids” because Power Pivot does not change the functionality of Pivot Tables.  What Power Pivot does is change the source data so that you can use data that could not previously be used, and you can get more out of data that you previously used inside of a Pivot Table.  So Power Pivot is actually your source data on steroids.
