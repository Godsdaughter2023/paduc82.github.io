Hi,

This is incorrect, if SFC is saying there is a Hash Mismatch it means that the file is corrupt, it doesn't mean that it has been removed. SFC will check the associated manifest file for that the SHA256 hash is supposed to be of the file in question, if the hashes dont match then this error will occur. If the file had been removed from the system then the CBS\CSI would of removed all of the traces of this update (along with all of the components) from the machine.

This file is the one of I have seen most corrupt on many windows 8 machines, it is rather easy to repair.

Download and Run SFCFix

Please download and run SFCFix from > <Removed><

It will take about 15 minutes to process. Once the scan has completed a notepad file will launch with the results. Please copy and paste the entire contents of the results and post them back into this thread.

SFC /SCANNOW

  1. Right click on the Start button
  2. Click on Command prompt (Admin) => Press Yes on the prompt
  3. Inside the Command Prompt windows copy and paste the following command
    SFC /SCANNOW
  4. Please wait for this to Finish before continuing with rest of the steps.

Convert CBS.log to CBS.txt

  1. Right click on the Start button
  2. Click on Run => Inside the run box copy and paste the following command:
    cmd /c copy %windir%\logs\cbs\cbs.log %userprofile%\Desktop\cbs.txt
  3. Press Enter
  4. Once this has completed please go to your Desktop and you will find Dropbox or One Drive
