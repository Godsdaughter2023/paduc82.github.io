a. user say:

Hi MaryMC19,

Windows Live Mail 2011 converts quotes and apostrophes to a strange character due to some settings involved using Unicode characters. The message will be transmitted using Unicode (UTF-8) encoding regardless of the setting at Options > Send > International settings. If the recipient tries to view that message using another encoding – like Western European – he will not see the characters you intended him to.

Windows Live Mail will use the encoding specified at Options > Read > International settings if Use default encoding for all incoming messages is selected. Otherwise, it will use the encoding specified in the message headers, or if none is given, its best guess at an encoding based on the content of the message.

If you see weird characters in a received message, click the Encoding button on the ribbon and select a different one, like Unicode (UTF-8). If the wrongly-displayed text doesn’t change, it may have been hard-coded into the message by the sender, for example by forwarding or replying to a message wrongly displayed at his end.

I find it best to leave Use default encoding for all incoming messages not selected.

You have two options on how to correct this sending behavior:

A. Press Backspace or Ctrl-Z or click Undo after entering a quotation mark or an apostrophe, or

B. to stop the behavior altogether. To do this, please follow the steps provided below:
  1. With Windows Live Mail closed, type  regedit  into the start search box and press Enter. Give permission where necessary.
  2. In the left hand pane, navigate to
    HKEY_CURRENT_USER\ Software\ Microsoft\ Windows Live Mail\ Compose\ Preferences\ PostEditor\ Autoreplace
    and click on Autoreplace.
  3. In the right-hand pane, double-click on each of the four Names where the Type column reads REG_DWORD. In each case, type 0 into the Value data box and click OK.
  4. Press F5 and then close the window.
  5. Launch Windows Live Mail and test.

 Click Start -> Computer and position it to a folder you'd like to specify columns for.  Click the "More options" button (3rd from the far right of the menu bar) and click Details view.  Then right-click the column header near the top of the right pane, and you'll get a list of fields you want to have shown.  Put checks by the ones you want.  This sets up your prototype for all folders you'll set up next.
