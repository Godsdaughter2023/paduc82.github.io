Enter below formula in cell B2 and copy down - for the number preceding the 1st '':

=TRIM(RIGHT(SUBSTITUTE(LEFT(A2,FIND(CHAR(1),SUBSTITUTE(A2,"''",CHAR(1),1))-1)," ",REPT(" ",255)),255))

=TRIM(RIGHT(SUBSTITUTE(LEFT(A2,FIND(CHAR(1),SUBSTITUTE(A2,"''",CHAR(1),2))-1)," ",REPT(" ",255)),255))

convert data from active sheet to csv format (semicolon)

(results on desktop/ name yyyymmdd_hhmmss)

Sub ActiveSheet_to_csv_utf8()
'Jun. 03, 2018
 Const sDelim As String = ";"   '<< select comma or semicolon
 Dim ws As Worksheet
 Set ws = ActiveSheet
 Dim sFile As String
 sFile = Environ("USERPROFILE") & "\desktop\" & Format(Now, "yyyymmdd_hhmmss") & ".csv"
 Dim r As Long, c As Long, i As Long, j As Long
 r = ws.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
c = ws.Cells.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column
Dim v() As Variant
ReDim v(1 To c)
 Dim obj As Object
 Set obj = CreateObject("ADODB.Stream")
 obj.Type = 2
 obj.Charset = "utf-8"
 obj.Open
For i = 1 To r
 For j = 1 To c
 v(j) = ws.Cells(i, j).Value
Next
 obj.WriteText Join(v, sDelim), 1
 Next
 obj.SaveToFile sFile, 2
 Dim Npad As Variant 'open csv via notepad
 Npad = Shell("C:\WINDOWS\notepad.exe " & sFile, 1)
 End Sub

stefan blom say:

The STYLEREF field would only display an error if there is no paragraph at all using the referenced style, so the conditional field isn't going to work.

What you can do is create a character style; in sections where Heading 4 exists, apply the character style to the Heading 4 paragraph. In each section where there is no Heading 4, apply the character style to the appropriate Heading 2 paragraph instead. This would be a "semi-automatic" solution, of course, and in a long, complex document it may be difficult to maintain.

Office 365 Groups Files & Folders Protection - Microsoft Community

ooimeng say:

Office 365 Groups Files & Folders Protection

How to prevent Office 365 Groups member from deleting files/folders created by the group owner?
cliff gu msft say:

Hi OoiMeng,

Generally, if the files are created by you(the group owner) and you only don’t want users to download files in the Office 365 group, you can customize users’ permissions via the following steps.

1. Navigate to the group.
2. Click More>Site>Documents.


3. Click Settings>Library settings>Permissions for the document library under Permissions and Management.
4. Click on the members group and delete all users in it.
5. Click Manage Parent. By default, the folders’ and files’ permissions inherit from the group site.
6. Click Permission Levels>select Edit>Copy Permission Level>name the customized permission level as customize>Uncheck Delete Items and Delete Versions.

7. Return to the permission page>Grant Permissions>Invite people>SHOW OPTIONS>customize.

If you have other requirements, you may try the similar steps above to see the outcome.

Thanks,

Cliff
