john mcghie say:

It's a badly-worded dialog.  What it wants you to do is:

  1. SELECT the item it shows you in the dialog,
  2. THEN click "Select".

That will grant access to the file in question.  This is an artefact of Apple's sandboxing requirement.  A sandboxed application has access to only a narrow range of locations.  If you want to use files in other places, you have to grant access to each location.

Once you have granted access, Word will store the location in your preferences, and will not ask again for that location.  Irritating, but that's the way Apple made it.

I would run a mile from the Reset Password utility.  The person who suggested that should be taken out and shot: that has the potential to lock you out of the entire content of your hard disk forever.  You will note that the advice was first suggested by someone struggling with running two Font Managers concurrently.  I assume you're not doing that!

However: The fact that the password utility finds "no available users" is deeply disturbing.  You ARE logging on to your Mac, right?  With a username and password?  In which case, I think I would make sure your backup is complete, then take that machine to your nearest Apple Shop.  The recovery from there has the potential to wipe the disk, so make sure you have a working backup.

You are correct in your suspicion that if the software is not activated, nothing is going to work.

There is a backdoor way to get into the Activation dialog.  If you have not tried the suggestions here: https://support.office.com/en-us/article/What-to-try-if-you-can-t-install-or-activate-Office-2016-for-Mac-using-Office-365-Home-Personal-or-University-5efba2b4-b1e6-4e5f-bf3c-6ab945d03dea

You could work through those, carefully and in sequence.  Ignore the one about re-installing, you've already done that.  I would start with Verifying your Office 365 Account.

Hope this helps

stevenrivera281 say:         

Fix for the 80240020 error when trying to install Windows 10

What this error means is, your Windows 10 installation folder is unfinished and/or corrupted. Now, how do you fix that?
1. Go into "C:\Windows\SoftwareDistribution\Download" and delete everything in that folder. (If you're unable to complete this step, look at "edit 5" down below)
2. Now, run the command prompt as an administrator. Type in "wuauclt.exe /updatenow".
3. Go to your Control Panel > Windows Update and your Windows 10 should start re-downloading from scratch, hopefully without flaws this time.

EDIT: For those who still have issues, several users reported that setting their time forward (to the First of August for example) and then doing the above steps again helped them resolve the issue.

EDIT 2: If none of the above works, you can try to download Windows 10 from Microsoft's website: http://www.microsoft.com/en-us/software-download/windows10 . Just be cautious that you want to do an upgrade, not a clean install! If you do a clean install, Windows 10 will not activate. You would need a new product key for that. But if you do an upgrade, your product key from Win 7/8/8.1 will carry over to your Win 10. Once you've completed the upgrade and checked that Windows is activated, you can do a clean reinstall if necessary.

EDIT 3: Another possible fix from Microsoft themselves but requires you to tamper with registry keys: /u/ritoman)

EDIT 4: Microsoft support is endorsing this as an official fix! Now, ain't that an accomplishment? https://twitter.com/WindowsSupport/status/626461440568872960 (thanks /u/RxBrad) Thanks for all the gold and kind words, guys!

EDIT 5: If you're unable to delete the contents of the folder due to them being in use, try the following:
1. Open the folder "C:\Windows\SoftwareDistribution\Download"
2. Run the command prompt as an administrator. Type in "net stop wuauserv". You should see a message indicating that the Windows Update service has been stopped.
3. Now, go back to the folder and now you should be able to delete all its contents.
4. Go back to the command prompt and this time, type in "net start wuauserv". This time, the message will tell you the Windows Update service has been started.
5. Go back to step 2 in the original post.
