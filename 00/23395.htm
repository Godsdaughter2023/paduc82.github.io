Excel 2007 VBA abort button - Microsoft Community

fred conklin say:

Excel 2007 VBA abort button

Need to build an abort button that will when activated let me cleanly shut down the VBA code.  If needed it can just stop the code and leave - but it would be nice to clean things up a bit

bernie deitrick, excel mvp 2000-2010 say:

You can use the DoEvents command to allow Excel events to run, then use them to find out if your want to stop the main code. For example, paste this into a standard codemodule

Option Explicit
Public Continue As Boolean

Sub TestMacro()
    Continue = True
    Dim r As Long
    Dim s As Long
    For r = 1 To 1000
        For s = 1 To 1000
            DoEvents
            If Not Continue Then
                MsgBox "I was stopped at " & r & " " & s
                Exit Sub
            End If

        Next s
    Next r

    MsgBox "I was not stopped."

End Sub

Then right click the sheet tab, select "View Code" and paste this code into the window that appears:

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
If Target.Address = "$A$1" Then Continue = False
End Sub

Run the macro TestMacro, and don't do anything, and the "I was not stopped." message should appear. Run the macro again, and select cell A1 (or another cell then cell A1 if A1 is the activecell when you run the macro). You should get a different message, like "I was stopped at 308 456" actual values being dependent on your timing.
