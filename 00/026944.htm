Windows Device Recovery Tool "Proxy settings error"-FIXED and - Microsoft Community

stranger_hanyo say:

Windows Device Recovery Tool "Proxy settings error"-FIXED and solutions for other problems

Hello,

I have been fidgeting with Windows Device Recovery Tool (WDRT) since the day I started installing Insider builds on my phone. It's a very useful software and saved a lot of phones but unfortunately for the last few days it was giving me trouble. Every time I tried using it, the package download would stop midway and throw an error "Please check if proxy settings are correct and click 'Try again'." This made me search the forums, try all given solutions and whatnot but nothing helped. Only today did I solve it and now it is all fine. While searching through the forums, I found that none of the solutions helped anybody and so I am going to post the solution in case anyone faces the same problem and gets stuck.

1. Open Control Panel.

2. Click on Windows Firewall option( if you use another firewall, you'll find these settings in its UI)

3. Click on "Allow an app or feature through Windows Firewall"

4. Click on "Allow another app"

5. Browse on your PC for the WDRT file and add it

6. Try downloading the files for your phone again, it should work now. 

Update 1.

There seems to be some issues after the recent upgrade to the software. Just re-install it and it should start working again.

Update 2.

If the above solution still doesn't work and WDRT fails to flash your phone, try the steps given below. WARNING!: This may void the warranty of your phone and if you want to stay on the safe side, visit your nearest service centre or try WDRT on another PC.

1. Install "Windows Device Recovery Tool" 
2. Download the firmware(only the ffu file) from this site http://www.lumiafirmware.com/  (open your phone cover and you'll get your phone model and code. This will be written somewhere on a sticker. Make sure you downlaod the right file)
3. And now open CMD as administrator
4. Typ "cd C:\Program Files (x86)\Microsoft Care Suite\Windows Device Recovery Tool" for 64bit PC.
   "cd C:\Program Files\Microsoft Care Suite\Windows Device Recovery Tool" for 32bit PC.
   Remove the quotes and post the rest of the command line as it is.
5. Connect your phone to your PC.
6. Typ in cmd "thor2 -mode uefiflash -ffufile "C:\(the location of the ffu\file name.ffu)" -do_full_nvi_update -do_factory_reset". Replace "(the location of the ffu\file name.ffu)" with the proper name of your downaloded file and location).
7. Wait now until your phone gives a green screen
8. Typ now in cmd "thor2 -mode rnd -bootnormalmode"
9. Wait until your phone starts up normally again.

As a reminder, always download all available WDRT updates. I'll be adding more solutions for problems to WDRT and if you face some issues using WDRT, drop a reply here.

Thanks.
