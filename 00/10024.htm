john mcghie say:

There is a setting, but it is not "hidden".

On the File menu there's an option "Reduce File Size".  That enables you to choose between 96, 150, and 220 pixels per inch.

Use 96 for email, 150 for inkjet printers and 220 for colour laser printers.

"Keep current resolution" keeps the image at whatever resolution you inserted it at.

A couple of things to note:

a)  Once you have applied this command, it's permanent.  You can't go backwards.

b)  There is no point in sending a device more pixels than it can output: it will simply cost you storage space, and throw the pixels away when it gets them.

c)  A normal colour printer can print a maximum of max-resolution divided by 4 in colour.  So a 1200 dpi colour printer can output about 300 dpi in colour, a 300 dpi printer usually has only 3 colours, so 100 dpi.  Of course, a commercial printer can do much better than this.  But in that case, the printing company will usually ask for the originals of the photos, so they can process them accordingly.

Once you start "dragging" in the document, either to position the picture or to resize it, all bets are off.  Word will reprocess the picture "to make it fit", and you have no real control over what's going to happen.

You should use a graphics editor (iPhoto will do well for simple stuff, or GIMP if you want to get really serious) to produce the photo at the size and resolution that you intend to output it.  Then use Insert>Picture>Picture From File to insert the picture where you want it, at the size you want it.

Word will then print what you gave it with no changes.

Provided that you do not use "Reduce File Size" :-)

Hope this helps
