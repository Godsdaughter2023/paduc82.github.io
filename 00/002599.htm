rohn007 say:
"Compatibility Mode" simply means that the file is in an older format, DOC rather than DOCX. In Compatibilty Mode, Word disables features that are not supported by the old file format.  The most obvious example is in the Home tab / Font group, the "blue A" Text Effects button is greyed out.

Rather than doing a SAVE AS, use the convert function. In the File tab / Info command, there is a big "Convert" Button displayed next to the "Compatibility Mode" description. Click on the button and follow the prompts.  Note, you may want to do this on a copy of the file as it changes the file structure and extension of this document rather than making a copy of it.

Describe exactly what sort of formatting differences your are seeing, I don't have a telescope and can't see that far ... <grin>

The changes in formatting are probably due to differences in the underlying formatting assumptions. If you go into File tab / Options command / Advanced option / Layout options, and scroll through the list you'll see a long list of formatting assumptions and rules that have changed over the years.

Take a closer look at the formatting applied to the the "good" and "bad" paragraphs. Here is some help on how to do that:

Reveal Codes in Word - Is there life after “Reveal Codes”? - http://sbarnhill.mvps.org/WordFAQs/RevealCodes.htm .  This is an excellent article describing all of the native Word tools for discovering formatting.


Formatting Problems? Install CrossEyes - http://www.levitjames.com/Products/CrossEyes.aspx

This tool adds a pane that converts Word formatting into HTML/WordPefect like “codes”, displaying a WordPerfect like “Reveal Codes” pane. It is a great complement to MS’s inadequate attempt in the <SHF><F1> “Reveal Formatting” pane.
