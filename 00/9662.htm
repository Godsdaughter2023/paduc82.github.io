johnlv say:
Sorry, I had just answered another question about 2010.  That code would work in 2007 or 2010, but not 2003.

Why are you trying to avoid VBA in 2003?  The macro facility in that version is very limited.  To do it with macros in 2003, you would need to have a hidden control on your form where you could stuff the search value, then do the Find after clearing the edit.  The sequence would look something like:

Condition: If Not IsNull(DLookup("[Family TItle]", "Family", "[Family Title] = '" & [Family Title] & "'"))
Action:  CancelEvent

Conditon: ...
Action:  MsgBox 
Message: That Family Title already exists in the database."

Condition: ...
Action: SetValue
Item:  [strHidden]
Expression:  [Family Title]

Condition: ...
Action:   RunCommand
Command:  Undo

Condition: ...
Action:  FindRecord
FindWhat: =[strHidden]

You would need a text box on your form called strHidden that has its Visible property set to No.

Note that using three dots in Condition on each line is imperative so that the additional statements get executed if the first Condition is True.

In VBA:

Private Sub Family_Title_BeforeUpdate(Cancel As Integer)
Dim strFamily As String
    If Not IsNull(DLookup("[Family TItle]", "Family", "[Family Title] = '" & Me.[Family Title] & "'")) Then
        Cancel = True
        MsgBox "That Family Title already exists in the database."
        ' Save the value the user typed
        strFamily = Me.[Family Title]
        ' Cancel the update
        Me.Undo
        ' Find the original record
        Me.Recordset.FindFirst "[Family Title] = '" & strFamily & "'"
    End If
End Sub
