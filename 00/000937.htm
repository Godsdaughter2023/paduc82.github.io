Function CountColour(rng As Range, clr As Range)
Application.Volatile
Dim c As Range
For Each c In rng
    If c.Interior.ColorIndex = clr.Interior.ColorIndex Then
        CountColour = CountColour + 1
    End If
Next
End Function

a. user say:

The correct way to disable Cortana in windows 10

This is the correct way to disable Cortana in windows 10.

1. Cortana is integrated into the Windows operating system and cannot be uninstalled.

(Yes, if you are foolish enough to try to rename or delete Cortana file or registry entries then search will not work)

(Also after each major windows update Cortana will be reinstalled again)

2. You can disable Cortana via Group Policies.

3. Cortana will still be seen in Task Manager but that is only it's local search functionality.

You need to have Windows 10 professional or Enterprise to turn off Cortana via Group Policies.

Press Windows key + R  (run) or right-click start, run
Type gpedit.msc (click ok or press enter)
Navigate to Cortana Group Policy objects under & click and reconfigure each option below

Computer Configuration
> Administrative Templates
> Windows Components
> Search

1. Allow Cortana
(Choose whether to let Cortana install and run on the device).

2. Allow search and Cortana to use location
(Choose whether Cortana and Search can provide location-aware search results).

3. Do not allow web search
(Choose whether to search the web from Windows Desktop Search).

4. Don't search the web or display web results in Search
(Choose whether to search the web from Cortana).

5. Set what information is shared in Search
(Control what information is shared with Bing in Search).
