a. user say:

I solved this one.

 

My ASUS K50ij Windows 7 began to take around 6 minutes to start up and I found the \Device\Harddisk0\DR0 error hundreds of times in the Event Viewer.

 

First I tried Skandisk, Malwarebites, Commercial AVG, but they showed no errors. I checked as well that I had the most recent BIOS from ASUS. Additionally, an update with DriverReviver provided me with some updated drivers. Nothing from the above solved the issue but I ecourage you to do it all.

 

Then I used the AIDA64 to find out the vendor of my hard drive (apologies, maybe there is an easier way to do this), which happened to be Seagate.

 

In the Seagate site I downloaded a tool to check and repair their hard disks, called SeaTools, DOS version. You have to save the mirror image of the downloaded .iso file it into a CD: I used the ISO Recorder freeware software for this.

 

Obviously, I had to change the BIOS so that start is done first from the CD.

 

Bounced the laptop and the SeaTools was launched from the CD. In the SeaTools menu, I selected Basic tests ->  Long Test (please be careful with Advance tests option, as it can erase your hard drive data)

 

After 1.5 hours the analysis finished and the tool found an error, but it was able to repair it.

 

Closed, changed back the BIOS start option, and bounced.

 

Voila, 1.5 minutes to start up, not a single error in the Event Viewer.

 

Probably your hard drive vendor has similar analysis and repair tools.

 

Hope it helps. Sorry for my english and for my poor explanations, I'm not an IT guy.

afzal taher say:

Hi Hugh Montgomery,

You can restore files using the Restore Files wizard. You can restore individual files, multiple files, or all files in a backup.

a. Open Backup and Restore by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking Backup and Restore.
b. Do one of the following:
•To view your files, click Restore my files.
•To view all users' files, click Restore all users' files.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
c. Do one of the following:
•To browse the contents of the backup, click Browse for files or Browse for folders.
When you are browsing for folders, you won't be able to see the individual files in a folder. To view individual files, use the Browse for files option.
•To search the contents of the backup, click Search, type all or part of a filename, and then click Search.

One of the most common problems when using a network firewall is that it sometimes blocks network traffic that you want to allow.

Please follow the below step to troubleshoot with the issue:

  1. Open the “Control Panel“.
  2. Select “System and Security.
  3. In the Windows Firewall section, select “Allow a program through Windows Firewall“.
  4. Unchecking the box to the left of the application name disallows it from accessing network resources, while checking it allows access.
  5. If the program you wish to block or unblock is not listed, you can click the “Allow another program…” button to add it. Choose the application in the list and select “Add“. If the program is not in this list, use the “Browse…” button to select the program file manually.
