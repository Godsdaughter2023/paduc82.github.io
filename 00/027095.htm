myall_blues say:

There might be a way to do it with conditional formatting but that is beyond me.

You could do it with a helper column and a Worksheet_change event.

For the helper column (say column AO), enter this array formula (enter using CTRL+SHIFT+ENTER) at the first row of your data and then copy down. Change '2' to the relevant row

=MAX(COLUMN(AF:AM)*(AF2:AM2<>""))

This gives the column number of the rightmost value in AF:AM

Code for worksheet change event

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim iColor As Variant
    iColor = 39 ' Set preferred colour
    '
    ' If the changed cell is not in range, exit macro
    '
    If Intersect(Target, [AF:AM]) Is Nothing Then Exit Sub
    '
    ' Reset colour over range.
    '
    Range(Cells(Target.Row, "AF"), Cells(Target.Row, "AM")).Interior.ColorIndex = xlNone ' or change to your yellow colour
    '
    ' Apply the preferred colour over range. Change 'AO' to helper column address
    '
    Range(Cells(Target.Row, "AF"), Cells(Target.Row, Cells(Target.Row,"AO").Value)).Interior.ColorIndex = iColor
End Sub

wyman w say:

Below is the sample code and sample file. I trust that you will be able to modify the code on your own after reading my example.

http://wymanwmwong.synology.me/example/transfer.xlsm

Public Sub transfer()

    dtmDate = Range("dtmDate").Value
    txtVendID = Range("txtVendID").Value
    paymentID = Range("paymentID").Value
       
    dataEntryFirstRow = 6
    dataEntryLastRow = dataEntryFirstRow + Application.WorksheetFunction.CountA(Sheets("Data Entry Form").Range("A6:A18")) - 1
    summaryFirstRow = Sheets("Total Summary").Cells(Rows.Count, "A").End(xlUp).Row
    
    counter = 1
    For r = dataEntryFirstRow To dataEntryLastRow
        With Sheets("Total Summary")
            .Range("A" & summaryFirstRow + counter).Value = dtmDate
            .Range("B" & summaryFirstRow + counter).Value = paymentID
            .Range("C" & summaryFirstRow + counter).Value = txtVendID
            .Range("D" & summaryFirstRow + counter).Value = Sheets("Data Entry Form").Range("A" & r).Value
            .Range("E" & summaryFirstRow + counter).Value = Sheets("Data Entry Form").Range("C" & r).Value
            .Range("F" & summaryFirstRow + counter).Value = Sheets("Data Entry Form").Range("D" & r).Value
            .Range("G" & summaryFirstRow + counter).Value = Sheets("Data Entry Form").Range("E" & r).Value
            .Range("H" & summaryFirstRow + counter).Value = Sheets("Data Entry Form").Range("H" & r).Value
        End With
        counter = counter + 1
    Next r
End Sub
