You'll have one problem with this: the zero preceding the last digit is static text, not part of the numbering scheme. Now I have a situation where I wish to Sort my WS's ... but then later Return them to their original position.  I am not intimate with VBA code (I record Macros only) so I have no clue if this can be done (with or without code)??? ... Can this be done? ... If so, how to do it would be awesome.

 

Thanks ... Kha

chip pearson say:

First, you need to store the original order, and then restore to that order. The first procedure below saves the worksheet names in order as a defined name with the sheet names separated by a ":" character.

 

Sub SaveSheetOrder()
    Dim S As String
    Dim WS As Worksheet
    For Each WS In ThisWorkbook.Worksheets
        S = S & WS.Name & ":"
    Next WS
    S = Left(S, Len(S) - 1)
    ThisWorkbook.Names.Add "SheetOrder", S, False
End Sub

Then, your code does its thing. Finally, you need to restore the worksheet order to that saved in the defined name:

 

Sub RestoreSheetOrder()
    Dim N As Long
    Dim S As String
    Dim SS() As String
    Dim WS As Worksheet
    S = ThisWorkbook.Names("SheetOrder").RefersTo
    S = Replace(S, "=", vbNullString)
    S = Replace(S, Chr(34), vbNullString)
    SS = Split(S, ":")
    For N = 1 To UBound(SS)
        Set WS = Worksheets(SS(N))
        WS.Move after:=Worksheets(SS(N - 1))
    Next N
End Sub
