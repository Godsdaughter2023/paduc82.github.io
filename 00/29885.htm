waseemulla shariff say:

Hi gregdes,

In addition to Andre's suggestions, Windows 7 has options to install the legacy device drivers as well as the devices. To install the legacy devices, try the following steps.

Step 1: Install legacy device
a. Click Start, type devmgmt.msc and hit enter.
b. Click Computer, click Action menu, and click "Add Legacy Hardware"
c. Follow the instructions on the wizard to install the device successfully by locating the device driver file.
d. Check if it works fine on Windows 7.

Step 2: Run the device driver in Windows XP compatibility mode manually
a. Right click on the device driver setup file and click Properties.
b. Click Compatibility tab.
c. Under compatibility mode section, check mark "Run this program in compatibility mode for" and select "Windows XP Service Pack 2" from the dropdown list.
d. Under "Privilege Level" section, check mark "Run this program as an administrator"
e. Click ok to apply the changes.
f. Connect the device and open the driver file and install it.

See Make older programs run in this version of Windows:
http://windows.microsoft.com/en-us/windows7/Make-older-programs-run-in-this-version-of-Windows

Important: Windows XP Mode can only be used on computers running Windows 7 Professional, Enterprise, or Ultimate editions.

Note: Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.

Follow these steps and check if it fixes the issue.

  1. Press Windows key + R.
  2. Type services.msc and hit enter.

  5. Go to C:\windows\system32\wbem\repository.
  6. Delete the repository folder ONLY.
