lemp say:

Sometimes, programs fail to release a file when done using it.  If there is no obvious program that might be using the file in question (e.g., if it's a *.doc file, close Word), there are utilities that will force a file to be released.  One of the better known ones is Unlocker:  https://www.majorgeeks.com/files/details/unlocker.html

Other than using something like Unlocker, there are a few things you can try.  In order of increasing disruption to your workflow:

1.  Open Task Manager, select "explorer.exe" in the list of processes (NOT IExplore.exe) and click "End Process."  Then, still in Task Manager, click "File > New Task (Run...)" type explorer.exe in the "Open" box and click OK.

2.  Log off (click Start, click the arrow to the right of the "Shut down" button, click "Log off").  Then log back on to your account.

3.  Shut down the computer.  Remove the flash drive (or other device).  Turn the computer back on.

xircal say:

Hi,

Check first of all to see if the file is located in the Recycle Bin. There's a shortcut to the Recycle Bin on your desktop in the top left hand corner of the screen.

If the file is in there, right click it and choose "Restore" from the menu. Windows will replace it in the same location it was deleted from. You don't need to do anything else afterwards.

If the file isn't in the Recycle Bin, do the following.

  1. Right click the Start button and go to "Command Prompt (Admin)".
  2. Next, copy/paste the following command and hit Enter.

DISM.exe/Online /Cleanup-image /Restorehealth

DISM will download original system file(s) from Microsoft servers and replace those which are missing.

After it finishes, using the same command prompt to run the following command: sfc /scannow
