If my value is more than column 1 then red 

if my value is less than  column 1 then green

For eg.

scottang say:

As icon sets cannot have relative references in the criteria, therefore create one rule for each row of data using vba.

Assuming data is in column A and B, with row 1 as header, and no blank rows in between:

Sub condFormat()

    For iRow = 2 To Range("A" & Rows.Count).End(xlUp).Row

        With Range("B" & iRow).FormatConditions
            .Delete
            .AddIconSetCondition
        End With
        
        With Range("B" & iRow).FormatConditions(1)
            .ReverseOrder = True
            .IconSet = ActiveWorkbook.IconSets(xl3Arrows)
            .IconCriteria(2).Type = xlConditionValueNumber
            .IconCriteria(2).Value = "=$A$" & iRow
            .IconCriteria(3).Type = xlConditionValueNumber
            .IconCriteria(3).Value = "=$A$" & iRow
            .IconCriteria(3).Operator = 5
        End With
        
    Next iRow
    
End Sub

jack cook - aka help_line say:

Because I truly want to maximize my battery, I always send receive manually ... personal preference. 

Since you have different accounts setup and the send/receive is different on each account it will "coordinate" the intervals with the other account even though they are set for different intervals.  When they both have to sync at the same time, I would expect that it will do one, complete the process then do the other.

I might suggest that you test it out using a 5 min and 10 minute cycle.  Setup up the schedule then sync emails.
