You can help solve some computer problems and improve the performance of your computer by making sure that your hard disk has no errors. If system restore doesn’t help, then use the command prompt tool from the Windows Recovery options and then perform check disk.

greg carmack - windows mvp 2010-2020 say:
Hi Peter. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.


You can run the Printer Troubleshooter at Settings > Update & Security > Troubleshoot.

But rather than spending too much more time fiddling I'd simply Clean Reinstall the printer by cleaning out the old drivers which always works, lasts longer and actually saves time:

1) Power up the printer, if wired unplug it's USB cable from PC.

2) Go to Settings>Apps>Apps and Features to find and Uninstall the printer software.

3) Next type CMD in Start Search, right click Command Prompt result to Run as Admin, copy and paste in the following command and press Enter: printui /s /t2 (Note that there is one space before each forward slash).

4 Open the Print Server properties, Remove the printer’s drivers for this printer and for any other printers you don't use presently.

5) If you get a message about access denied or the printer being in use and you cannot find the driver or any installed program that would account for this, you may restart the computer in a diagnostic startup and then use the printui command to remove the driver package(s): https://www.isunshare.com/windows-10/enable-dia...

d0nnyr say:

Ok, So I've figured the solution to this problem, and I'm going to post it here at the top so that it will be easily seen by others, and hopefully I'll explain this in a way that other users with this issue will understand the solution.

 

So in the AMD VISION Engine Control Center you have the options to create presets. The presets apply to whatever page/subpage/action you have completed/opened/visible at the time.

 

Since I'm going to assume that most people with this problem or at this point will have already created an Eyefinity Display Group in the AMD VISION Engine, I'm going to write this with that assumption in mind.

 

1.) Disable Eyefinity Display.  If you notice your displays are duplicates of each other, so then we..

 

2.) Go to Desktop Management -> Creating and Arranging Displays in the AMD VISION ECC. Where it allows you to select a display, click the drop down arrow and select "Extend". Do this for each display you have.

 

3.) Drag and rearrange your displays.  This may not be necessary for everyone, but if you try to scroll from one display to another and you find that it's in the wrong spot, this is how you will resolve this.

 

4.) Presets-> Add Preset.... A dialogue window will pop and here you will enter in a Name (I chose "extended desktop"), a tool tip which a more thorough description. Leave the location settings by default. These allow you to access this preset by having it pinned to the Start Menu, available in the system tray, etc. I assigned a hot key using the modifier Ctrl+alt+shift and key M because this is probably not used by windows by default.

 

At this point you have now created an Start Menu icon/hotkey shortcut/system tray link to a proper 3 monitor, Windows 8 desktop with full Windows 8 Metro Support.

 

5.) Now, you recreate your Eyefinity Display Group. Since anyone reading this will have already made one, I'm not going to outline those instructions.

 

6.) Properly set up, arrange, create bezel compensation, ETC for your Eyefinity Display Group.

Once you've done this...

 

7.) Presets-> Add Preset.... Name this preset something like "Eyefinity Desktop". Follow the same steps before in regards to adding a preset.

 

8.) Test functionality.   I did this by switching back to the first preset I made which was the extended desktop; If you created your presets properly, your monitors will likely flash on/off as the changes take place.
