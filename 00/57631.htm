lemp say:

Yes.

And unlike many Windows error messages, this one is actually pretty clear.  It means that if another program -- likely the program that came with the laptop or wireless adapter -- is running and managing the wireless adapter, then Windows can't manage the adapter. 

Typically, there is an option somewhere in the other program to turn it off and allow Windows to manage the adapter.  Otherwise, you have to uninstall the other program.  In addition to the MS KB article, seehttp://www.ezlan.net/wzc.html

If you provide the make and model of your wireless adapter (and laptop, if you're using a laptop), then perhaps someone can direct you to the instructions for your particular program.

Some like to have the manufacturer's utility manage the wireless adapter instead of Windows Wireless Zero Configuration, because the manufacturer's utility often has features that the Windows service lacks.
