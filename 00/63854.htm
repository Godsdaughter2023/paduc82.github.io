    Let's get your File History addressed by following the workaround indicated below:

    1. Open File History. Click Select Drive. Then, choose a different drive that is showing in File History.
    2. Set up File History on the drive that you chose. Afterwards, turn off the File History in that drive for the files not to be transferred.
    3. Go to the external USB drive that is not showing. Find the File History folder and rename it to "File History.old" or delete it.
    4. Go to your C:\Users\your user folder\AppData\Local\Microsoft\Windows. Find the File History folder and rename it "File History.old" or delete it. 
    5. Re-open File History. Click Select Drive. Choose the original external USB drive that is not showing. The external drive should be working fine now with File History.
    6. Lastly, delete the File History that was created in procedure 1-5.

    Note: The above process is a must when File History configuration files were not done right due to altered drive letters when adding, removing drives, or moving the system between drives.
