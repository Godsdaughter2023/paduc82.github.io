A Windows license has never been for the life of the User, only for the life of the machine it is activated on.

'Upgrading' the mobo and cpu is effectively  building a new machine, and the mobo has always been the bit of hardware

Win10 licensing is no different. OEM = one machine only. Retail = several machines, but it's still not endless. I haven't verified

the limits for Retail Win10, but prior to Win10 it's been for 5 different machines. Successive or concurrent. Not both.

- Exception is generally made if your mobo dies and your replace it with the same series mobo using the original, or same model cpu.

You may still be able to sweet talk the supervisor, if you use phone activation rather than automatic activation, and get the new hardware

activated ... ( 'My motherboard went up in smoke and I can't replace it with the same model. Take pity kind sir\madam' ).
