mitch h. say:

Hi,


The error message, "Windows Update Standalone Installer. The update is not applicable to your computer.", that you're receiving when trying to install KB4022716 update might be due to these common reasons:

  • Update is superseded
    If the update that you're trying to install already has a newer version of the payload on your system.
  • Update is already installed
    If the update that you're trying to install was previously installed, for example, by another update that carried the same payload.
  • Wrong update for architecture
    If the update doesn't match the architecture for your CPU.
  • Missing prerequisite update
    If you are missing a prerequisite update.

Having said that, have you checked if you have previous updates that failed to be installed?

In the meantime, you can download the Windows Update Troubleshooter from this link. Select the sub-link entitled, "Download troubleshooter for Windows 10" to download and run the troubleshooter.

If the issue persists, let's reset Windows Update components for further troubleshooting.

You can also consider manually downloading the Windows update you're trying to install from the Microsoft Update Catalog if the issue occurs.
