Alias is just another name for the same thing. The White House is an alias for 1600 Pennsylvania Avenue. 


...  it asks me if I want to create a new account or add an existing account


The choice is Add an existing email address as a Microsoft account alias. An address that is a Microsoft Account in its own right can't be an alias of a different Microsoft Account. You can use an email address that isn't registered as a Microsoft Account as an alias of a Microsoft Account.

All Microsoft-provided public email addresses - e.g. @outlook, @live, @hotmail, @msn, @windowslive - are automatically registered as Microsoft Accounts, so for an existing address to be usable as an alias, it must be a non-Microsoft address - e.g. @gmail, @yahoo, @aol, @comcast or whatever.
