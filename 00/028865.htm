Print 11B envelopes in word - Microsoft Community

narelletaylor say:

Print 11B envelopes in word

Hi,

I need to print 11B size envelopes in word but when I try to change the custom settings to 14.5cm by 9cm, it says the settings are too small and the minimum dimensions are 16.25cm by 5.4cm.

How do I get around this?

Is there another program I can use?

Thanks

doug robbins - mvp office apps & services (word) say:

That minimum size is set by the size of the Frame that is included in the Envelope Address Style, but it can be overcome as follows:

  1. Accept the 16.25cm, then add the envelope to a document
  2. Click on the Show Hide (¶) button so that you can see the paragraph marks ¶ in the document.
  3. Click on the ¶ in the middle of the envelope where the address would go and you will see the dotted outline of the frame



  4. Move the mouse cursor to the edge of the frame until it changes to four opposed arrows and then click to select the frame - The border will changed from dots to a hatched style.
  5. Right click and then click on Format Frame and in the Frame dialog, reduce the Width of the frame to (say) 11 cm.
  6. Click out side of the Fram and then click on the dialog launcher in the Page Setup section of the Page Layout tab of the ribbon and go to the Paper size tab and change the Width to 14.5 cm
  7. You may want to click in front of the ¶ where the address will go and reduce the Left Indent of the paragraph to provide more space for the address.
  8. Scroll down into the document below the envelope and then access the Page Setup dialog again and change the top and bottom margins to 1 cm and go to the Layout tab and change the Section Start to Continuous.  The second page will then disappear.
  9. Save the document (Envelope) as a template with the name of Envelope 11B.

Then whenever you want to use one of these envelopes, use File>New and select your Envelope 11B template.

Hi Tim,

To use message encryption in Office 365, we need to activate the Azure Rights Management first. Please note that not all Office 365 plans have Azure Rights Management, plans have this are for example E3 or E5 plan. So I would like to confirm what Office 365 plan you are using at the moment.

I would like to confirm whether you have activated the Rights Management. If not, follow the steps to do it:

1. Sign into Portal (https://portal.office.com) with your Office 365 admin account, Click Go to the old admin center, navigate to Admin – SERVICE SETTINGS – Rights Managements and activate it there.

2. Connect to Exchange Online Using Remote PowerShell.

3. Run the following command. Just select one based on your region:

For North America:

Set-IRMConfiguration -RMSOnlineKeySharingLocation “https://sp-rms.na.aadrm.com/TenantManagement/ServicePartner.svc”

For European Union:

Set-IRMConfiguration –RMSOnlineKeySharingLocation “https://sp-rms.eu.aadrm.com/TenantManagement/ServicePartner.svc”

For Asia:

Set-IRMConfiguration –RMSOnlineKeySharingLocation “https://sp-rms.ap.aadrm.com/TenantManagement/ServicePartner.svc”

For South America:

Set-IRMConfiguration –RMSOnlineKeySharingLocation “https://sp-rms.sa.aadrm.com/TenantManagement/ServicePartner.svc”

4. Run the command:

Import-RMSTrustedPublishingDomain -RMSOnline -name "RMS Online"

 

5. Run the command and make sure the result is PASS:

Test-IRMConfiguration -RMSOnline

 

6. Run the command:

Set-IRMConfiguration -InternalLicensingEnabled $true

 

Please note: this action will need hours or a day to take effect. For your reference:

https://technet.microsoft.com/en-us/library/dn569291.aspx
