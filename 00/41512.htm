¡firedog say:

If there are folders or messages that you can see in the store folder in the file system but not in the Windows Live Mail interface, then it's the database index that has become confused or corrupted. Open your store folder, then:
 

  1. Rename the file Mail.MSMessageStore to Mail.old
  2. Delete the folder called Close Explorer, then type wlarp into the start search box and press Enter. In the window that opens, select Repair ...
    (I don't know if this necessary or helpful, but it might be and it won't hurt).
  3. Reboot the computer, then disconnect from the Internet (e.g. by switching off Wi-Fi or unplugging the Ethernet cable). This will prevent the program from trying to sign in and call home on launch.
  4. Start Windows Live Mail.
    If the program starts, it will need some time to create the new database index. Stay disconnected and leave the program alone until it starts responding.
