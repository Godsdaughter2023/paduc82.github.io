A formula such as

 

=COUNTIFS(A1:A20000, "xyz", B1:B20000,"I'm ringin to speak to a manager to complain")

 

should work regardless of the ' in the text.

 

If you still want to replace, you could use something like

 

Sub ReplaceApostrophes()
    With Range("B1:B20000")
        .Replace "I'm", "I am", xlPart, , True
        .Replace "You're", "You are", xlPart, , True
        .Replace "He's", "He is", xlPart, , True
        .Replace "She's", "She is", xlPart, , True
        .Replace "We're", "We are", xlPart, , True
        .Replace "They're", "They are", xlPart, , True
    End With
End Sub

Headers/Footers by default are properties of the Document... IOW, a continuous run from start to end. If you want to isolate a Header or Footer of any given Section in order to make it different from the others what you have read is correct -- You need to Unlink the Section's Header and/or Footer from that of the previous Section.
