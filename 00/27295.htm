How can you have formatting (such as text color) defined for a - Microsoft Community

dianea521 say:

How can you have formatting (such as text color) defined for a drop down list display when a list item is selected.

For instance, I have a list that contains 3 values, Pass in green, Fail in red and NA in blue. But it doesn't display that way in the cell with the drop down when you select from the list.
jllatham say:

I don't know if your Pass/Fail/NA entries have green/red/blue cell fills or font colors, but that doesn't really matter.

I presume that the drop down is set up with Data Validation (DV) from your color coded list and that the entries in the drop down list are black (no way to change that), and when you choose one of them in the DV cell it is still black.  This we can change by using Conditional Formatting on the DV cell.

Select the cell with the DV and use Conditional Formatting and set up 3 rules, 1 for each of the words.

Start with "New Rule"

Choose the 2nd entry in the list: "Format only cells that contain"

In the "Edit the Rule Description" where it starts out with [Cell Value] use the drop down to select [Specific Text]

Enter the word (Pass or Fail or NA) that you're creating the rule for and then click the {Format} button and choose the fill or font format (or both) that you want that cell to take on when the word is chosen from the list.

Repeat for all 3 possible choices.
