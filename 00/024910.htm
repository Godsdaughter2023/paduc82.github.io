dgsarnow say:

Hyperlink based on the text value of a cell

I have a workbook with two sheets. One has a large amount of data in a table, call this SHEET2. This holds my accounts. This table can be sorted and filtered by users. My other sheet holds cells with just the name of my accounts and I would like to create hyperlinks for these so it acts like a table of contents allowing people to jump to an account wherever it is in the table, Call this SHEET1.

I am trying to create a Hyperlink function that would look at the name of an account from SHEET1 and use vlookup or match or some function to populate the address of the cell the hyperlink points to.

I know the formula is wrong, but it gets at what I am trying to do.

EXAMPLE:

Cell A1 on SHEET1 contains the text "ACME Construction"

In cell A2 on SHEET1, I want to have a hyperlink something like this
=HYPERLINK([Book1.xlsx]'SHEET2'!(vlookup(A1,SHEET2!B:B,1,FALSE))

I cannot link this manually because I have too many accounts. HELP!
gary's student say:

Try this:

 

=HYPERLINK("#"&"SHEET2!" & ADDRESS(MATCH(A1,SHEET2!B1:B100),2), A1 )

 

If there is is cell in SHEET2 column B that matches the contents of A1, we will jump to it.
