Cannot connect to Internet on only one of 3 PCs - Microsoft Community

sandyyn say:

Cannot connect to Internet on only one of 3 PCs

About a week ago, I unexpectedly cannot connect to the Internet on one of my PC's, though my other 2 still connect fine.  I've tried to repair my LAN in Network Connections, with the result that my IP cannot be renewed.  Also tried an ipconfig /renew, which gives the result of "unable to contact your DHCP server".  Ipconfig shows my IP as 169.254.90.197, same for my default gateway.  Tried turning off and resetting my router.   My Network card shows working.  Any ideas? 
lemp say:
 Ipconfig shows my IP as 169.254.90.197, same for my default gateway.

This means that your NIC is set to "Obtain an IP address automatically" (which is the default and which you confirmed is, in fact, your setting) and that the computer can't connect to a working DHCP server.

In virtually all home networks, the DHCP server is part of the router.  So the message means (a) computer can't connect to the router, (b) the router's DHCP server is turned off, or (c) the router's DHCP is broken.

Because your other computers work, options (b) and (c) are ruled out.  That means that the problem is (a): the computer can't connect to the router.

The most common cause for a failure to connect to the router is that there is a misconfiguration of wireless security, although you didn't mention whether the problem computer is connecting via a wireless NIC or a wired NIC.

As a first step, access your router's configuration utility (from one of the computers that can connect to the Internet through the router).  Confirm that
  • the DHCP server is working (of course it is)
  • the DHCP server is configured to give out a sufficient number of IP addresses (this probably is true, unless someone has been messing with your router's configuration behind your back)
  • there is no MAC filter or other filter set up in your router that is excluding your "problem" computer (like the previous point, this probably also is true for the same reason)

If you are using a wired NIC, the problem likely is hardware:  either the cable is bad, the port in the router is bad, or your network card is bad.  The easiest way to test for these conditions is to swap known working parts.


If the connection is wireless, go to the wireless section of your router's configuration utility.  First, make sure that your router is configured to use the proper wifi band that is compatible with all of your wireless devices (A, B, G, N).  Although it is unlikely that this setting will have changed since a week ago, it's worth confirming (the usual problem is with an older wireless adapter, say wireless-B, trying to connect to a router set to "G-only" rather than "B & G"; the same for wireless-N).


At this point, you have two options:

Option 1:  write down all of the pertinent wireless configuration settings and then go back to the problem computer and try to connect using the correct information.

Option 2:  temporarily disable all wireless security on the router and see if you can connect.


Whichever option you take, you'll want to remove the wireless "profile" from your computer before you try to connect.  If you're using Windows WZC, right-click on the icon for the wireless adapter in Network Connections, choose Properties, then the "Wireless Networks" tab.  Find your network (SSID) in the list of Preferred Networks and remove it.

ken blake say:

*No* version of Windows has ever come with a word processor. If an older computer of yours came with a word processor (Word, Microsoft Office, or Works), it was because its manufacturer bundled it with what he sold you, not because Windows came with it.

A minor exception to the above is WordPad, which *does* come with Windows. However, as far as I'm concerned, it's just a glorified text editor, not a real word processor. However

What kind of documents are "all the documents [you] have saved"? Are they .doc or .docx files, created by a prior version of Word? Or if they are something else, please tell us what they are.

If they are .doc or .docx files, you have several choices:

1. Buy Word 2016.

2. Buy Microsoft Office 2016, which includes Word 2016.

3. Find and buy an older version of Word of Microsoft Office.

4. Buy WordPerfect (my personal favorite word processor), which is largely compatible with Word .doc and .docx documents.

5. Try WordPad, which is also largely compatible with .doc and .docs files, on your documents, and see how compatible it is with your documents and how you like it.

6. Download and install the free OpenOffice (http://www.openoffice.org/download/) or LibreOffice (http://www.libreoffice.org/), both of which are largely compatible with Word.

I recommend Trying WordPad first, then giving either OpenOffice, LibreOffice, or even both a try. You will get to see how compatible they are with your files, and also see how you like them. If you don't like them, you can easily uninstall them (OpenOffice and LibreOffice, not WordPad), and buy Word, or Microsoft Office, if that's what you want.

  2. Click System and Security and click Power to run troubleshooter.

  2. Click Change plan settings on the power plan (ex: High performance) that you want to reset its settings to default.  

  3. Under Edit Plan Settings, click Change advance power settings.  

  4. Under Power Options on Advanced settings tab, select the power plan (ex: High performance) you want to reset its settings to default by clicking the drop down arrow.  

  5. Click the Restore plan defaults button.  

  6. Click OK to confirm and check if the issue persists.
