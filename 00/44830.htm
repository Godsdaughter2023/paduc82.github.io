turn off the lock screen

How do I turn off the LOCK SCREEN using Windows 8

ronnie vernon say:

How do I turn off the LOCK SCREEN using Windows 8


Hi

 

How you do this depends on which version of Windows 8 you have installed.

 

If you Have Windows 8 Pro:

 

1. Press the WinKey+R to display the Run dialog. Type  gpedit.msc  and press Enter. This will open the Local Group Policy Editor.

2. Navigate to Computer Configuration / Administrative Templates / Control Panel / Personalization.

3. Double click “Do not display the lock screen” and select Enabled from the dialog that pops up. Click OK.

 


 

If you have the Windows 8 core version, you can disable the Lock Screen using the Registry.

 

1. Press the WinKey+R to display the Run dialog, type  regedit.exe  and click OK to open the registry editor.

 


2. Navigate to the following registry key.

 

 

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

 

3. Right click the Windows key and select New/Key. Rename the new key  Personalization.

 

4. Right Click the Personalization key and select New/DWORD (32-bit) Value.

 

5. Name the new value NoLockScreen  (Case sensitive). Right click the NoLockScreen value and select Modify.

 

6. In the Value data box, type the number 1  and click OK.


7. Exit the registry editor and restart the computer to see the change.
