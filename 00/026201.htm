First sign that this is a phishing scam is that it's a message coming "from you" "to you"

rohn007 say:

I haven't personally run into this problem (yet, and I hope to never have the "pleasure").

Sounds like your customized file is either being deleted or overwritten.

I can think of 2 "old school" approaches to the problem.

First.

  • Make all of your customization changes.
  • Navigate to your normal.dotm file in file explorer
  • (with word still open)
  • Select, Copy, paste the normal.dotm to create a copy of the file

(You can do this, even with the file open by Word. It may not have all of the changes because some are them are not saved until Word closes.)

  • Then close Word. 
  • Assuming the normal.dotm file has not been deleted, make another copy (So you now have 2 backup copies)
  • right click on normal.dotm
  • select "Properties" from the context menu
  • turn on the "read only" file attribute.

Now nothing should be able to delete (or change) the file without asking your permission.

The other approach is a little more complex. You create a batch file. In it you copy a backup copy of normal.dotm to become your new normal.dotm, then start the Word program. After word closes, you copy your current normal.dotm to the backup copy.

This second approach is handy when you are on a corporate network that is rude enough to download a corporate normal.dotm, overwriting your customizations.
