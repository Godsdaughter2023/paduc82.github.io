vanessa yar say:

Hi fuadhüseynli,

Two of the possible reasons why you're not getting any sound from your device speaker are: (1) The device speaker may not be configured yet, or (2) The speaker was not set as the default playback device. To ensure that your speaker is properly configured and is set as the default playback device, please follow these steps:

Set up speaker on your device.

  1. Open Control Panel and select Sound.
  2. Select the appropriate speaker and click Configure at the bottom left.
  3. Once done, select again the speaker and click Properties.
  4. Verify all settings are correct. If possible, change the speaker icon.
  5. Click Apply, then OK.

Set speaker as the default playback device.

  1. Open Control Panel and select Sound.
  2. On the Playback tab, select the appropriate speaker.
  3. Click the drop-down arrow in Set Default option, then select Set as Default Device.
  4. Click OK.

Set an output device.

  1. Navigate to Settings > System > Sound.
  2. Under Choose your output device, select the appropriate speaker.
  3. Set the volume to 100.

Once done, please test if you're now able to hear sound from the device speaker. If not, restart your Dell Venue 8 Pro, and try to play some music.

If you continue to have this issue even after performing the above steps, please reply to this thread. This way, I'll be notified and will respond back to you as soon as I can.

You will need to close Sticky Notes before attempting to copy and overwrite the existing plum.sqlite inside the LocalState folder.
