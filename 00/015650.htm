daniel g. grau say:
The sanctioned formats are very strict. If you want to depart from the standards, the easiest thing to do is to wait until your document is final. Then:

  1. Save a copy using Save As (to preserve the original document) or make copy in the Finder.
  2. You'll have to manually go through each citation, click on it and then click on the blue triangle to select "Convert citation to static text.")  
  3. Select each citation (now converted to editable text) and modify as needed. If it appears several times in your text, copy it and paste into Edit > Find > Advanced Find and Replace to do a global replacement.

File History is one of the feature available on Windows 8 operating system and above. It saves copies of your files and has an ability to get them back if they’re lost or damaged. This will be applied if this feature is enabled on your computer. To verify if this feature is enabled on your computer by going to the Control Panel > System and Security > File History. Option 2: OneDrive

 

OneDrive is an online storage where you can save your files and access it in other devices once it’s associated with your Microsoft account. This will be useful if you use this to save your worksheets. Go to this link and click Sign in at the top right-hand corner and then log in to your Microsoft account. Hi,

 

So you want to drag the formula to the right and copy a column from another sheet, try this.

 

=INDIRECT("'Sheet4'!"&ADDRESS(COLUMN(A1 ),1))

 

 

First change the sheet name to the correct one and be careful not to delete the ' marks either side of the sheet name. You then need to get the correct start cell. The formula is currently set to copy column A starting from A1 and as you drag right the next cell is A2.

 

The column is the last number, 1 is Col A and a 2 would be Col B etc.

 

The start row is this bit COLUMN(A1 ) and all you change is the A of the A1 bit. A1 is row 1 so B1 would be row 2

Hi,

 

OFFSET is also a volatile function as is INDIRECT

 

An excellent resource on the use or avoidance of volatile functions can be found at the link below.

 

Received wisdom is that we should avoid them and while I don't really disagree with avoiding volatile formula I think some perspective is required.

 

I copied your formula into range A1:AOU1 which is 1000 cells and did the same with my formula by copying into b2:AOU2 and ran some timings.

 

Time to calculate the 1000 cells of each formula was

 

OFFSET = 0.00495 seconds

INDIRECT = 0.00685 seconds

 

So OFFSET is faster then INDIRECT by 19/1000th of a second when calculating 1000 cells but I doubt that the difference is going to be noticeable accept in very complex workbooks.

 

FWIW. I would have posted (and used) your formula had I thought of it<g>
