Windows 10 is not designed to be run using the Built-In Administrator account.  Unlike Windows 7, the Built-In Administrator account in Windows 10 has several undocumented limitations.

In Windows 10, the only practicable use for the Built-In Administrator account is to rescue you from a condition of having no functioning Admin-level accounts of your own on the computer. 

I suspect that you will not be able to solve your current problems without using the Built-In Administrator account to create an Admin account of your own and then using that one instead. 

You will need to use the Command prompt to create the new user account as the Built-In Administrator account cannot create user accounts through the UI - the commands you need are explained in paras 2.2-2.5 of Use the Built-In Admin account in Safe mode - Wiki.  You should also look through the tidying up suggestions given in section 3.

john mclellan 246 say:

Headphones issue - Very Simple Solution - Unplug them on Shutdown, Replug only when Win10 fully restarted

Hi,

Finally I have a Very Simple Solution to my Headphones not working....Unplug Headphones before shutting down computer. Re plug only when Windows 10 has finished starting up.

In Detail......

Today my headphones stopped working after I put my Laptop into sleep mode with the headphones still plugged in. When I restarted my laptop with the headphones still plugged in they no longer worked.

I checked all the obvious setting and nothing sound related had changed in Device Manager. I then did the following steps:-

1     Unplug the headphone jack plug with the computer still running.

2     Shut the computer down (total shutdown).

3     Restart the computer

4     Wait for Windows to finish starting up

5     Plug in the headphones.

The headphones then worked correctly.

I have repeated this to make sure the fault happens if the headphones are left plugged in and the computer restarted, and that the fault goes away after removing them, shutting down (total shutdown) and then restarting, only plugging them in once Windows 10 has finished starting up.  The process is repeatable on my laptop.

It is quite possible that not all of these precise steps are needed to get headphones to work, but this can be explored in due course.

I have not seen any Microsoft Updates take place since I discovered this cure / workaround. It may be necessary to disconnect the headphones, restart and then reconnect them following an update, again time will tell.

I hope this Very Simple Solution works for others. Please leave a reply if it works for you.

John
