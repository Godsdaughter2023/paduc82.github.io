        Will adding more RAM erase my existing system files? - Microsoft Community

        solutions ten say:

        Will adding more RAM erase my existing system files?

        Original Title add memory

        I just bought a new HP computer with window 7, however, I want to add more memory to its maxi capacity.  I do undwerstand the new pc has some nre files loaded into, if I take the original 4 GB memory chip out and insert two new 4 GB memory into the slots, will I going to loss the original loaded files ?? Just want to make sure before do it. Thanks. 
        ken blake say:
        I just bought a new HP computer with window 7, however, I want to add more memory to its maxi capacity.  I do undwerstand the new pc has some nre files loaded into, if I take the original 4 GB memory chip out and insert two new 4 GB memory into the slots, will I going to loss the original loaded files ?? Just want to make sure before do it. Thanks. 



        Several points:

        1. Changing the amount of installed memory has no effect on what is stored on your disk drive. So no, you will not lose anything if you do this.

        2. 32-bit Windows 7 can only use about 3.1GB of memory. If you are running 32-bit Windows 7, adding more than you have will do nothing for you. Only 64-bit Windows can make any use of the addition.

        3. Your motherboard may or may not be able to take additional memory. You should find out whether it can before you make any changes.

        4. Assuming that you have 64-bit Windows and that your motherboard can take more memory, whether adding more memory will improve your performance depends on what applications you run. For most people going from 4GB to 8GB will provide little or no improvement and shouldn't be done.
