For a list within a document -- with each item in a separate paragraph or in a separate row of a table -- use the Sort command

If you have a "ribbon" version of Word (Word 2007 or later), select plain text and click the Sort button  in the Paragraph group on the Home ribbon.

Or select a table and click the Sort button in the Data group on the Table Tools > Layout ribbon (this ribbon is visible only if the selection is in a table):

If you have a menu version (Word 2003 or earlier), the Sort command is on the Table menu. It will display either the Sort Text or the Sort Table dialog, depending on what is selected in the document.

leahtheintrovert say:

(ANSWER) "The file .docx cannot be opened because there are problems with the contents"

I had the problem where i was compiling scanned documents into Microsoft word. Any time I tried I saved as I was going along, sometimes it was fine and sometimes I got this message "There is insufficient memory or disk space. Word cannot display the requested font". After a while, when I tried to save it crashed by document and would not open it. Instead it gave me this message "The file "abcd".docx cannot be opened because there are problems with the contents". When I clicked on details the message I kept getting was "The name in the end tag of the element must match the element type in the start tag".

I tried for hours attempting the various recommendations of Microsoft easy fix, downloading the service pack for office, even trying to fix the xml document myself but nothing worked. THEN I did the following:-

  1. Click start menu
  2. Go to computer
  3. Click "Organize"
  4. Go to "Folder and Search Options"
  5. Select the "View" tab
  6. Scroll down to "Hidden files and folders" and select "Show hidden, files, folders and drives"
  7. Scroll down further and check to see if "Hide extensions for known file types" is currently selected. If so, click to uncheck it. If it is not currently selected, leave it alone.
  8. Just below that, uncheck "Hide protected operating system files (Recommended)". When you do, a warning will pop up saying "You have chosen to display protected operating system files (files labed System and Hidden) in Windows Explorer. These files are required to start and run Windows. Deleting or editing them can make your computer inoperable. Are you sure you want to display these files?" Select "Yes".
  9. Select "Apply" then "Ok".
  10. Go to the folder where you last saved the document. You should now see some additional files that look transparent. Look for one that ends in .tmp. The name of the file might look something like this ~WRL5401.tmp. Right click on it and Open with "Microsoft Word". You should be able to get the document open that way.
  11. As soon as I got my document opened this way, I DID NOT CLICK ON SAVE. Instead I adjusted some formatting issues of the document then went into the File tab and selected Save As. I put the name of the document BUT I CHANGED THE FILE ~WRL5401.tmp. Right click on it and Open with "Microsoft Word". You should be able to get the document open that way.
  12. As soon as I got my document opened this way, I DID NOT CLICK ON SAVE. Instead I adjusted some formatting issues of the document then went into the File tab and selected Save As. I put the name of the document BUT I CHANGED THE FILE TYPE TO PDF instead of Word Document or any such format. Later I used an online software and changed back my pdf to word to edit it.
  13. NB It is important to go back and HIDE PROTECTED OPERATING SYSTEM FILES and HIDDEN FILES AND FOLDERS when you are finished!

It would be a lot simpler to use ONE Update query to update all the records at one time.
