I believe the source of the problem to be a permissions error on Windows server.

When opening a file from the file server, MS Office for Mac creates temporary files in a ‘.temporaryitems’ folder. If the folder does not exist, it attempts to create the ‘.temporaryitems’ folder.

Sub AutoNew()
Dim oBm As Bookmarks

    Set oBm = ActiveDocument.Bookmarks

Function UpdateBM(strBookmark As String, strText As String)
Dim oRng As Range
    If oDoc.Bookmarks.Exists(strBookmark) Then
        Set oBookmark = ActiveDocument.Bookmarks(strBookmark)
        Set oRng = oBookmark.Range
        If Len(strText) > 0 Then
            oRng.Text = strText
            oDoc.Bookmarks.Add strBookmark, oRng
        End If
    End If
End Function

daniel g. grau say:
You can turn off spell-checking as you type, but that's not what you need.

The language setting works like a text attribute. A given portion of text can be defined as Bold, as 10 Points, as Red, and as English, German or whatever as well. Just like there will always be color to the text, there will always be a language to it.

What you need to do is tell Word not to spell check a portion of text. To do that:
  • Select the text
  • Go to Tools > Language
  • Choose a language
  • Place a checkmark on Do not check spelling or grammar

If you click on the Default button you would be adding that setting to the document's original template, so don't use it unless you want that. Just click OK.

 Another way to do it is to redefine the Normal style in your document:
  • Go to Format > Style
  • Select Normal in the list to the left
  • Click on Modify
  • Choose Language in the bottom dropdown menu and continue as before

jllatham say:

As written it would change the name on the tab when the user entered a name into A2 -- it would not change the name if the content of A2 is a formula that changes based on other cell contents. 

That could still be done automatically, but I'd have to see the formula(s) in A2 of the various sheets: we would have to go back to some source cell somewhere that the user does make entries into to get the name to change to do it automatically.

Here's a bare minimum skeleton for code that would work when a button was clicked on a sheet, this would go into a regular code module:

Sub ChangeSheetName()
  Dim testWS As Worksheet
  If IsEmpty(ActiveSheet.Range("A2")) Then
    ActiveSheet.Name = "Default " & ActiveSheet.Index
  Else
    'test if name used on another sheet
    On Error Resume Next
    Set testWS = Worksheets(ActiveSheet.Range("A2").Text)
    If Err = 0 Then
      'name already in use!
      ActiveSheet.Name = ActiveSheet.Range("A2") & " (" _
       & ActiveSheet.Index & ")"
    Else
      'name not used as sheet name yet
      ActiveSheet.Name = ActiveSheet.Range("A2")
    End If
  End If
  Set testWS = Nothing
End Sub
