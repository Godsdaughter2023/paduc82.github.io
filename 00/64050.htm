tryx3 say:

You can use Windows own tools for this [see Windows Help] or you can use Acronis TI [WD Edition].  I have referred to Acronis methods below because that is what you asked about and there are distinct advantages to using it.

[But please note that this user-to-user mutual support forum is actually for Microsoft products not Acronis ones]

The latest Acronis TI WD Edition is a limited version of Acronis TI 2015 and your user interface marks the limited items that you do not have.  You have the ability to make & restore images but do not have all the auxiliary tools such as scheduling & encryption.  You will probably not feel any real disadvantage from not having these extra tools.

WD provide a user manual for the application.  According to the

You can also use the Acronis website itself - they have many helpful articles in their Knowledge base.  You can also get a copy of the user manual there [chose Acronis TI 2015] but it does not identify the limitations of your WD Edition.  Acronis also have

In addition to this the application has a Help system.  Look at its left hand side and click on the icon that looks like a book.  Then click on Open Help.  Look through the introductory section but then study the article at Backing up data, Backing up partitions and disks.

I cannot guide you through the procedure for making a system image in any detail because the Acronis TI user interface is too difficult to explain without working through compiling an illustrated guide.  So, very briefly indeed,

  1. Open Acronis TI [WD Edition]
  2. Click on Operation,
  3. Click on Edit settings,
  4. Click on the big icon for Disks & partitions so you can select what you want to make an image of,
  5. Click on the big icon for Destination [or Custom destination] so you can tell it where to save the image [in a folder on your external hard drive],
  6. Click on Save in the bottom right corner,
  7. Click on Backup now in the same place.

That's it.  The image will be created.

When it has finished, you should also make a rescue disk. 

  • If your Windows 10 installation was ever so corrupted that it would not boot up for you to be able to use the application itself then you would use the rescue disk to boot up from because it is able to control restoring a system image that you had previously made. 
  • To make the rescue disk, look at the icons on the left-hand side of the application's user interface and click on the Tools icon then select Rescue media builder.  That will guide you through making a rescue disk that can boot your computer and restore a system image. 
  • You only need to do this once [i.e. not after every image]. 
  • Make sure you study the Help entry about using the rescue disk because you may not be able to get into your system to read it if there is an emergency need to use this procedure.

5. One special thing about being an admin is you can give permissions to yourself. It's just an extra step to do anything. All system files allow only the Windows Modules Installer (trustedinstaller.exe) full access. To decode 0x80070002. The 0x means it's a hexadecimal number, the 8 means error, the first 7 means it a windows error, and the rest of the number, 2, is the actual Windows error.
 
To look up the error we need it in decimal format. Start Calculator (Start - All Programs - Accessories - Calculator) and choose View menu - Scientific, then View menu - Hex. Enter 2. Then View menu -  Decimal. It will say 2. (I add this as a general how to for numbers over 9)
 
Start a Command Prompt (Start - All Programs - Accessories - Command Prompt) and type
 
net helpmsg 2
 
and it will say
 
The system cannot find the file specified.
 
Use Notepad to search for FATAL and 80070002 in C:\Windows\Windowsupdate.log.

a. user say:

How can I prevent certain programs from starting when Windows starts?

Go through the menus of those programs until you find the setting that lets you choose whether or not the program should start with Windows. Turn that setting off.

Most programs will allow you to make this choice, but some will not. For those that don't, ask the program developers' technical support if they have a way to prevent the program from starting with Windows.

The reason for asking the developers' technical support, rather than taking action on your own, is because you always want to operate all your software in a way that the developers support, if at all possible.

Will startup programs slow my computer down?

Yes, if they continue to run in the background and use a lot of your computer's resources (CPU, RAM.)

And is there a way to modify the program so it will not start up when Windows starts up during the installation process?

Some programs will ask you during the installation process if you want the program to start with Windows. Most do not.

Don't stop a program from starting with Windows merely because you feel it makes your computer slower. The program may be doing a task that is vital to your computer.

Restart your PC
