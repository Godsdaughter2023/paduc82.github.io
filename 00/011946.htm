rob koch say:

One glance at that page immediately tells anyone with any knowledge of Microsoft that it's a fake.

The Microsoft Logo is always accompanied by the logotype, which means that the Microsoft name is displayed to the right of the logo symbol as covered in the "What artwork should be used?" section of the following document.

Microsoft Corporate Logo Guidelines

The page tab displayed in my Internet Explorer browser is an apparently default "Home Page" with an image of pages, while a Microsoft page always contains their logo symbol and often the logotype as well.

There are other ways to determine who owns a website such as using the Whois utility at any major registrar of Internet domain names, but you'd need to learn how to read these.  In most cases, there are so many clues on the fake web pages used for phishing and similar attacks that it's a wonder anyone believes them.

For example, this fake page contains very little text and extremely simple formatting.  This avoids the more common issues like misspellings or bad grammar, but it looks nothing like any real Microsoft page I've ever seen.  For example, just compare it to either the http://Microsoft.com or any other primary Microsoft websites linked from that main page such as Support or the Microsoft Store.

In other words, just think before you click and use any of the official websites as a simple way to verify that the page looks anything like them.

Rob

< EDIT > Note that there are many other things wrong with that web page such as the certificate being registered to the domain name itself rather than to a company, let alone Microsoft.  Any Microsoft website is typically named something>.Microsoft.com so not seeing that is an immediate red flag.

This could go on for hours, so the point is to look for anything missing that would normally be present on an official Microsoft web page, which is why I suggested looking at the main home page, since that's a simple example to compare against.
