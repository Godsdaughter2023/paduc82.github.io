jay freedman say:

If you want to paste Text Only by default all the time, in all documents -- and use the Paste Options button to change that choice when necessary -- you can set that default in the Word Options dialog, on the Advanced tab; in the "Cut, copy and paste" section, set some or all of the four options to "Keep Text Only".

If you need to keep different settings for most documents and "Keep Text Only" just for this one template, you need a macro. Before showing that, I want to emphasize that you shouldnot paste text into a template, but into a document based on the template. Templates are meant to be relatively unchanging, supplying only styles and macros and such.

Store the following macro in the template:

Sub EditPaste()
    On Error Resume Next
    If ActiveDocument.AttachedTemplate = ThisDocument Then
        Selection.PasteSpecial DataType:=wdPasteText
    Else
        Selection.Paste
    End If
End Sub

When you paste, the macro will paste plain text if the document into which you're pasting was based on the template containing the macro, even if the options are set some other way. If the document receiving the paste is based on any other template (such as Normal.dotm), then the formatting will be determined by the option settings.

Also note that if the clipboard contains a picture in addition to text, the picture won't be pasted; and if the clipboard contains a picture or other object but no text, then it will appear to do nothing at all.
