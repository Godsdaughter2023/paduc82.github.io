graham mayor say:
You cannot do this with fields in Word. There is unfortunately no Left (or Mid or Right) function associated with fields. You can however use a macro run on exit from the dropdown field  to populate another field, when you tab out of it e.g.

With ActiveDocument
    .FormFields("Text1").Result = _
    Left(.FormFields("Dropdown1").Result, 5)
End With

will populate text formfield Text1 with the first five characters of the value selected in dropdown field Dropdown1

how do I repeat footnotes on every page in document - Microsoft Community

steph_901 say:

how do I repeat footnotes on every page in document

Table headers are repeated on each pabe of my doc but the footnotes that I have for each on show up on the first page.
daniel g. grau say:
To do what you want you'll have to resort to page footers, because footnote texts can only appear once per document:

  • Insert a section break before the table (and another after, if your document continues)
  • Make sure the footers are unlinked from the previous footers

Then you can do proceed in two ways:

Use simulated footnotes
  • Add a superscript number in the table's repeating heading (instead of the current footnote reference).
  • Go to the footer and add the same superscript number, plus the "footnote" text (plus an optional border above). This will cause it to appear at the bottom of every page in the section containing the table. 

Use a real footnote and reference it
  • Add the footnote reference and text like you did.
  • Select the footnote text (you won't be able to select the separator above it).
  • Do Insert > Bookmark  and give the bookmark a name (say "FFF").
  • Go to the footer, select Format > Document and enable "First page special" (this will give you a different first page for the section containing the table).
  • In the general footer (i.e., not the first page of the section) insert a REF field through Insert > Field and link it to the "FFF" bookmark. The field should look like this: { REF FFF }. Add an optional border above it.
