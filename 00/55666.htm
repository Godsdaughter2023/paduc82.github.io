You're asking the wrong question. The question you should be asking is "How can I avoid destroying my computer?" I had that same question. And then I found the answer - which, as I later discovered, was no secret. Here it is:

1- Don't install an operating system on hardware that is not compatible with the operating system.

3- Only install software programs that are supported on the operating system.

4- Don't change your configuration settings until you've done the research and are sure that you're doing the right thing in the right way. (That was the second-best advice I ever got.)

5- Backup every day. (That was the best advice.)
