1) Go offline by opening Action center at the right end of Task Bar, then clicking on Airplane mode to turn off internet.

felixwong_ say:

I had the same issue with my Microsoft Wired Keyboard 200 for 5 months before figuring out the solution.  Apparently, many other people have had this problem too judging by the Amazon reviews for the keyboard.

The root cause is the computer shuts off the keyboard in order to save power. It does this because of a default setting.

To fix, go to Control Panel → Hardware and Sound → Devices and Printers → USB Keyboard/Properties → USB Input Device/Properties → Change Settings → Power Management tab and uncheck "Allow the computer to turn off this device to save power."

The above might be a little confusing so I posted step-by-step instructions with screenshots here.
