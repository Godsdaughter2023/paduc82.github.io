* * Note that Excel reverses the arguments to ATAN2 – see notes below

Since atan2 returns values in the range -π ... +π (that is, -180° ... +180°), to normalise the result to a compass bearing (in the range 0° ... 360°, with -ve values transformed into the range 180° ... 360°), convert to degrees and then use (θ+360) % 360, where % is modulo.

There is a function that can do exactly this:

IMARGUMENT(complex(North1-North2;East1-East2))*180/PI()+180 will give you the angle in degrees.

Edit: I made a correction to move from -180->180 to 0->360...

North1,East1 are your reference coordinates, North2, east2 the target.

andrea jones - all about resources say:

The Modulo function gives the remainder after diving the value by 360 so to get the answer for (θ+360) % 360 in Excel you would use

=MOD((θ+360),360)

(you would, of course, replace θ with the reference of the cell containing your angle or substitute your trigonometry calculation).
