    yaqub k say:

    Hi Noddy,

    Does your computer freezes only when the Automatic Maintenance takes place on the computer?

    Windows 8.1 automatically performs a number of maintenance tasks for you, including smart system diagnostics, checking to see if any updates are available for your software and performing security scans. Automatic maintenance does too many functions, so turning it totally off is not advisable.

    The Automatic Maintenance normally runs at a time when the computer is idle. When it finds that the computer is in use at a time scheduled for the automatic maintenance, then it does not start automatically and instead it waits for the next time when the computer is idle.

    It is apparently not possible to disable it completely. We do not recommend you to disable Automatic Maintenance. If you wish to disable Automatic Maintenance you may disable it, but only at your own risk. Follow the steps to disable Automatic Maintenance:
    a. Press Windows + W keys and type Task Schedule.
    b. Open Task Scheduler.
    c. Browse to Task Scheduler, Microsoft, Windows, Task Scheduler.
    d. Right click Idle Maintenance & Regular Maintenance and select Disable and exit the task scheduler window.

    Note: Regular Maintenance automatically re-enables itself after system restart.

    You could at least disable it from waking your computer.

    a. Click on the Action center flag on the right hand bottom corner to enter Action center.

    b. Expand Maintenance and click on Change maintenance settings

    c. Uncheck the option which says allow computer to wake up my computer at the scheduled time

    d. Set the Automatic maintenance Schedule time to a time when the computer is powered off / not in use

    Note: The task will remain enabled, but with triggers disabled, it won't actually start.

    As you have mentioned that your computer freezes and you are not able to do anything on your computer, I would suggest you to follow the methods mentioned below.

    Method 1:

    If you are using any third party security program, then try temporarily disabling the security program and check if that helps.

    Step 1: Temporarily disable the third party security program. (If you have installed any)

    If you have installed a third party security program, then I request you to contact its technical support to temporarily disable the program and then check the issue.

    Note: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you have to temporarily disable it to install other software, you should re-enable it as soon as you're done. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

    Step 2: Let’s place the computer in clean boot and then check if the computer freezes. Clean boot-   It’s a process wherein you can start Windows 8 by using a minimal set of drivers and startup programs.  A clean boot helps eliminate software conflicts.

    Follow the link below to perform clean boot:

    How to perform a clean boot in Windows

    http://support.microsoft.com/kb/929135

    Note: Please go through the section: How to reset the computer to start as usual after troubleshooting with clean boot of the Kb article to boot the computer in normal startup after fixing the issue.

    Step 3: Run the System Maintenancetroubleshooter.

    a. Press Windows key + W and type Troubleshooting.

    b. Select Troubleshooting.

    c. Select view all on the top left corner.

    d. Click System Maintenance, follow onscreen instructions.

    If the issue still persists, check if the computer is infected by any malicious program. For this, we can run Microsoft Safety Scanner and check.

    Method 2: Run Microsoft Safety Scanner and check if there are any infections in the computer.

    I would also suggest you to check if the computer is infected by any malicious software. For this let’s try running the Microsoft Safety Scanner and then check the issue. You can download and run the Safety Scanner from this link:

    http://www.microsoft.com/security/scanner/en-us/default.aspx

    Note: When running a scan using an Anti-Virus program, any data files that are infected may be cleaned by deleting the file entirely, which means there is a potential for data loss.

    Open Event Viewer and check for error logs. Post the logs here so that we can find out what exactly is causing the problem.

    Press Windows key+ F, type “event viewer” in the search, select “view event logs”

    jesse rosenbaum say:

    Hey Adrian,

    I was just experiencing the same issue and someone recommended booting into safe mode then uninstalling my display adapters in Device Manager, reboot then install the latest Nvidia drivers.  This didn't work for me, but here is what did.

    Keep in mind, my laptop is probably not the same as yours, but it's a Lenovo W540 with an Nvidia Quadro K2100M along with an Intel HD Graphics 4600 chip.  What worked for me was to boot into safe mode (using msconfig.exe) then disabling the Intel display adapter.  Now the PC works great.  No issues with Windows (file) Explorer.

    Just in case you need steps on how to boot into Safe Mode

    1. Press Windows Key + r to get the run prompt

    2. Type msconfig and click OK (press enter)

    3. In the new window click on the Boot tab

    4. Check the Box for Safe Boot

    5. Click the check box for Base Video

    6. Click OK

    Then when your system reboots just go into Device Manager and disable the Intel video card, assuming you have an add-on card like I do with the Nvidia.  Reboot and see if Windows Explorer works now.  Hope this helps.
