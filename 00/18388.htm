The best practice is to save the covers in the Word template's AutoText (Insert menu>AutoText). Then a user can insert whichever cover they prefer. One additional hint. You asked about a document and then talk about templates. There is a big difference. You can save AutoText (and building blocks) in Templates, not in documents.

This is true in both the Mac and Windows versions, AFAIK.

Templates in Microsoft Word

Where can Building Blocks be stored?
