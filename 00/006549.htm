justin cavida msft say:

Hi Jeanette,

To insert an Equation box as Rohn mentioned, press Alt and "=" simultaneously. This will automatically add an Equation box to your document. You can then simply insert a fraction by using "/" symbol (e.g. 1/2) and press Space or Enter.

There is no default shortcut for inserting fractions using the Ctrl key as of now. However, you can use the Alt accelerator sequence to navigate the Ribbon. This will let you choose a fraction template from the fraction gallery. You can follow the steps below. You can also try Rohn's suggestion about using LaTex.
1. While the cursor is in the Equation box, press and release the Alt key, which puts up the tooltips that show the accelerator keys.
2. Press J and then E to activate the Equation Tools tab.
3. Press F to activate the Fraction gallery.
4. Use the arrow keys to select the fraction template you want.
5. Press Enter to use the selected template.

Creating a macro that will hide rows on specific multiple worksheets - Microsoft Community

bkwsnx say:

Creating a macro that will hide rows on specific multiple worksheets based on specific criteria

Hello,

There are specific worksheets in a spreadsheet (I'll call them Worksheet_1, Worksheet_2, Worksheet_3, etc.) that I want to use to write some reports. There are ranges of certain rows in these worksheets (i.e. row ranges 6:12, 45:90 in Worksheet_1, row ranges 13:93, 100:110 in Worksheet_2, row ranges 7:35, 36:66, 67:100 in Worksheet_3, etc.) that will generate the text "N/A" or "" (blank) near the end of the range.


What I would like to do is to create a macro that will check numerous row ranges in multiple specific worksheets all specified in the macro, check each row range for the first row that contains anything except "N/A", "" or completely empty rows, find the last row in this range that contains anything except "N/A", "" or completely empty rows, then hide the rows outside that row range.

Here are some specific examples on how I'd like to deal with specific named ranges in just one of multiple worksheets: In rows 6-12 in Worksheet_1, there would be text in row 6, blank space in row 7, numbers in row 8 and 9, while rows 10-12 each have multiple cells with the formula result of "N/A". In 45:90 in Worksheet_1, there would be blank space in rows 45-48, rows 49-75 would contain numbers and rows 76-90 would contain the formula result of "".

My desired macro result would keep rows 6-9 as is but hide 7-12 in row range 6:12 & leave rows 49-75 alone while hiding rows 45-48 & 76-90 in row range 45:90.

Any help on this would be greatly appreciated.

bkwsnx say:

Figured it out myself. I realized that if I made the spaces between the ranges I wanted to keep intact as an IF statement based on other criteria where the output was " " if I wanted to keep it or "" if it needed to be hidden, then it would be considered in the following formula properly. There was also at least one column in each of these worksheets that had the data required, "N/A" or "", namely the A column. 

I wrote this to get the rows I wanted hidden:

 

Sub HideRowsInDifferentWorksheets()

Sheets("Worksheet_1").Select
For Each cell In Range("A6:A12,45:90")
    If (cell = "N/A" Or cell = "") Then cell.EntireRow.Hidden = True
Next cell

Sheets("Worksheet_2").Select
For Each cell In Range("A13:A93,100:110")
    If (cell = "N/A" Or cell = "") Then cell.EntireRow.Hidden = True
Next cell

Sheets("Worksheet_3").Select
For Each cell In Range("A7:A100")
    If (cell = "N/A" Or cell = "") Then cell.EntireRow.Hidden = True
Next cell

End Sub

I really hope this can help somebody else in the future.

vasil l. michev say:

No data will be removed when you lift the hold, unless you have manually specified some retention policy with the Delete action. Or the user has already deleted the messages (i.e. messages in the Recoverable Items subtree might get deleted after you remove the hold). His 'regular' messages in Inbox and other folders will remain.

brookzhang_msft8047 say:

Hi GarethEvans6847,

When in-place hole is enabled for a mailbox and related items are deleted from the Deleted Items folder, all items are moved to Recoverable Items folder, which is invisible. After a mailbox user is removed from In-Place Hold or Litigation Hold, items in the DiscoveryHolds, Purges, and Versions folders are purged by the Managed Folder Assistant.

In your scenario, if these email between January and May are deleted from the Deleted Items folder (invisible for you), it’s not feasible to recover them. 

If they are visible for you (in Inbox, Deleted Items folder etc.), apparently you can retain them.
