slance310 say:

Hi,

Do you want to install Win 10 Pro to Legacy Bios or UEFI Bios settings...

That is where the "GPT" comes in... It is required for UEFI Bios, and "MBR" is required for Legacy Bios...

How the Flash Drive is created can also determine how the installation tries to install... If it is created with a NTFS file system it will most likely try to install for Legacy Bios and look for a MBR Drive...

It is created with  a FAT32 file system it will most likely try to install for UEFI Bios and look for a "GPT" drive...

You can not simply have a "GPT" or "MBR" partition... The entire drive must be GPT or MBR created when the drive is first "Initialized" after having every partition on it deleted...

If you can determine if the machine is set for UEFI or Legacy it would be easier to help... One thing to check for... If you have a small FAT32 "EFI" partition, it is UEFI Bios... If you do not, it is Legacy...
