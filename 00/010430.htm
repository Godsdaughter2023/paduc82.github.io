    The question now is i dont seems to able to convert the tabulated format into the original text file format.  can some help on this?  I used the below instruction to convert from Text to EXcel. 

      • In Excel 2010:
        1. Select the File tab.
        2. In the left column, select Open.
        3. To the right of the File name: text box, from the drop-down list, select Text Files.
    2. Select the document you want to convert.
    3. Select the Delimited radio button and click Next.
    4. Under Delimiters, clear all the check boxes.
    5. Under Delimiters, place a check in the check box(es) corresponding to the separator(s) your data contains.
    6. Click Next.
    7. Click Finish.

    Assume that your original data is 12*34, you import the data as 12 in A1, 34 in B1 (if thats what you mean tabulated format)
