ongcaps say:

Hide rows based on a numeric value in a cell using VBA

Dear Community,

I am trying to create Worksheet_Change() event macro to unhide rows based on the numeric value of one cell.

I have a table B13:B513. I want to unhide a number of rows equal to the value in C7. For example, if C7=10, then B13:B22 will unhide and the rest will still be hidden.

I have seen several ways of doing something similar but using "case" for each option. In my case I have 500 options. I am sure there is a more efficient way of doing this.

I am using Excel 2010.

Thanks in advance!

Daniel

bernard liengme say:

This worked for me

best wishes

Private Sub Worksheet_Change(ByVal Target As Range)
    If Intersect(Target, Range("C7")) Is Nothing Or Target.Cells.Count > 1 Then Exit Sub

    If Target.Value > 500 Then Exit Sub

    Application.EnableEvents = False
    LastVis = Target.Value + 12
    ActiveSheet.Rows("13:" & LastVis).Hidden = False
    LastVis = LastVis + 1
    ActiveSheet.Rows(LastVis & ":513").Hidden = True
    Application.EnableEvents = True
End Sub

williamquaintance say:

Replace All in Microsoft Excel

Dear Microsoft,

 

After much effort with MS Excel 2010, I finally figured out the relationship between the Replace All button, the Within Sheet/Workbook option, and whether or not I have multiple cells selected when I clicked on Replace All.  You should add more description or otherwise make it clear how this works.  The Replace box does not give this information.  Here's how it works:

 

First, select multiple cells in the visible worksheet.  Then open the Replace window (ctrl-H).  If the Within option is set to Sheet, then clicking the Replace All button will replace all within the selected cells only, not the whole sheet.  If the Within option is set to Workbook, the Replace All button will replace all in the entire workbook, regardless of your selected cell range.  If you have only one cell selected and the Within Sheet option, then it will Replace All in the whole sheet you are working on.

 

I had a hard time before I figured all this out.  I had previously used the Within Workbook option for its intended purpose.  Later, I selected some cells, clicked Replace All, and could not understand why things outside my selection were changing.  Then I found someone on the internet say that my selected range of cells has no effect if Within Workbook is chosen, but does have effect if Within Sheet is chosen.

 

I do like having all these options, but for God's sake please make it understandable and obvious.
