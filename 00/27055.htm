jeeped say:

In a 2010 workbook, .AutoFilter might be a better choice.

Sub mcr_Remove_This_but_not_That()
    Dim c As Long, ws As Worksheet
    Dim sContainsThis As String, sButNotThis As String
    Set ws = ActiveSheet
    sContainsThis = "XYZ"
    sButNotThis = "XYZ ABC"
    With ws.Cells(1, 1).CurrentRegion
        c = .Columns("A").Column     'filter on column A
        .AutoFilter
        .AutoFilter Field:=c, Criteria1:="=*" & sContainsThis & "*", _
            Operator:=xlAnd, Criteria2:="<>*" & sButNotThis & "*"
        If Application.Subtotal(103, .Columns(c)) > 1 Then _
            .Offset(1, 0).EntireRow.Delete
        .AutoFilter
    End With
    Set ws = Nothing
End Sub

Edit: I made a modification to the way that the code determines whether there are any rows to delete. I had forgotten that you would likely have a columns heading.

ken sheridan say:

If I understand you correctly you want to open the appropriate form for whatever the event type is for the current date.  Right?  If so you need to look up the value returned by the query.  Yo cannot simply reference the query in the way you are attempting.  To look up a value from a table or query the DLookup function can be used.  This will return the value from the first row returned unless you include a criterion, but as far as I can see your query only returns one row.  So the amended code would be:

Private Sub Command43_Click()

On Error GoTo Err_Command43_Click

 

   Dim varEvent as Variant

    Dim stDocName As String

 

  varEvent= DLookup("[Event]","[Event ListRRR Query]")

 

   If Not IsNull(varEvent) Then

      stDocName = "Ast" & varEvent

     DoCmd.OpenForm stDocName

    Else

      MsgBox "No event today",vbInformation, "Warning"

    End If

   

Exit_Command43_Click:

    Exit Sub

 

Err_Command43_Click:

    MsgBox Err.Description

    Resume Exit_Command43_Click

   

End Sub

 

Because your form names are using a naming convention of 'Ast' followed by the event  value you can assign a value to the stDocName variable in a single line by concatenating 'Ast' with the event name.
