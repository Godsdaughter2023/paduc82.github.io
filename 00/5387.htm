john mcghie say:
No, you can't turn that off in Word for Mac.  The command to change it was left out of Mac Word.

Nor can you lock the styles in the document so they cannot change them.  Again, that is a PC Word function that was left out in Word for Mac.

The only thing you can do is preemptively rest the styles to those in the template when you get the file back.  To do that, use Tools>Templates and Add-ins, attach a template, and check the "Automatically update styles" box.

Uncheck the box immediately, it works instantly, and you don't want to leave it set.  Don't use it on a document that contains numbering: it will reset all the numbering.

Phone set to English-US and Greek keyboards

Phone set to Greek UI.

Fallback to 70-characters-per-SMS is triggered by either one of the following conditions:

1) Inputting even one lowercase Greek character.

2) Inputting even one uppercase Greek character which has a equivalent of a latin character.

For example, inputting "μ" or "Ρ".

Only way to keep using Greek 160 characters-per-SMS is the following:

Use uppercase Greek characters and when an equivalent latin exists, switch to latin keyboard, input that character, then switch to Greek keyboard for the non-latin character.

Example: NAMAΣTE.

In the above example, only "Σ" must be inputted in Greek alphabet. The rest of the letters MUST be inputted using the English-US keyboard. If they are inputted using the Greek keyboard, the 70-character limit kicks in. Otherwise, you get the usual 160-character limit.

Before some update, you could write in lowercase Greek, using punctiation just like "Νάμαστε", which, when sent to another phone via SMS, it was automatically converted to "ΝΑΜΑΣΤΕ" so that it wouldn't break the 160-character-per-SMS utility. But this is no longer the case.

I think Nokias have a setting in SMS+ that you can choose which behaviour you want (automatically converting Greek characters to uppercase and keeping 160-characters-per-SMS limit, or using full uppercase/lowercase notation and going to 70-characters-per-SMS).
