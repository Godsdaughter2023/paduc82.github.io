- Uninstall KB3035583 and KB2976978 updates

Windows Registry Editor Version 5.00

Hi Kristi,

Xbox App is one of the built-in applications on Windows 10. To uninstall it from your computer, you have to perform the following steps using the Windows PowerShell:

  1. Click Start, and then type Windows PowerShell on the search box.      
  2. Right-click, and then select Run as administrator.      
  3.  Type the following command to uninstall Xbox app: Get-AppxPackage *xboxapp* | Remove-AppxPackage
  4. Hit Enter.  

If you are unable to uninstall the app, reset Windows Store to clear the store cache and try to uninstall the Xbox app again.

Here's how to reset Windows Store:

  1. Press Windows key + X on your keyboard.       
  2. Click on Command prompt (Admin).       
  3. Type wsreset.exe, and then hit Enter.
