jeeped say:

Text values that masquerade as numbers are sorted by Excel as if the digits in the pseudo-numbers were characters in a word so what you have is going to sort as R250, R50, R7 etc.

Typically, a helper column would be used to split off the numeric portion and convert this to a true number that reacts as one would expect numbers to sort in an ascending or descending manner. In an unused column to the right, use this formula,

=VALUE(MID(A2, 2, 99))

  ... or,

=--MID(A2, 2, 99)

Fill down as necessary. You can now sort your block of data using this column as the primary sort order to achieve the results you are expecting.

If there are more variations on the alphanumeric part numbers (e.g. with more than a single character prefixing the number), post back with examples and additional help could be provided.
