Stefan, unfortunately the cited article is for Microsoft Search Server, available from SharePoint, but it is not part of Office. The search function built into Word does not support Boolean search operators.

Tom, if you can write and use Word macros written in Visual Basic for Applications, you can incorporate RegEx ("Regular Expressions") in a macro to perform these searches. Introductions to RegEx syntax and capabilities include

https://www.regular-expressions.info/vbscript.html

https://blog.udemy.com/vba-regex/

https://www.ocpsoft.org/tutorials/regular-expressions/or-in-regex/

https://www.ocpsoft.org/tutorials/regular-expressions/and-in-regex/

There are many articles on the internet about using RegEx in Excel macros, but anything that works there will also work in Word.

To use RegEx in a Word macro, create or open a module in the macro editor, and then click Tools > References and put a check mark next to "Microsoft VBScript Regular Expressions 5.5" in the list. That attaches the code from vbscript.dll (a standard Windows system file) to your module, and you can then call any of the RegEx functions.

The SUMIF function has been in Excel for a long time. I am assuming you mean SUMIFS, which was introduced in Excel 2007. You can  use SUMPRODUCT instead of SUMIFS. For example,

 

=SUMPRODUCT(--(A1:A6="a"),--(B1:B6=2),C1:C6)

 

This will sum the values in C1:C6 when the corresponding value in column A is equal to "a" and the value in column B is 2.
