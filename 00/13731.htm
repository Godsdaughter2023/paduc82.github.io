Create two calculated columns in the query:

TheYear: Year([DateField])

and

TheMonth: Month([DateField])

AveargeIf doesn't act on the average - it averages cells that meets the if part. Do your cells have 0 values or are they blank? Do you want to only average non-zero values or do you want the average to not be zero? Examples of what you might have and what you want from each of the examples would be helpful.

=IFERROR(1/(1/AVERAGE(S26:U26)),IFERROR(1/(1/AVERAGE(P26:R26)),"Missing"))

=IFERROR(AVERAGE(S26:U26),IFERROR(AVERAGE(P26:R26),"Missing"))

violetchen_msft say:

hi philip,

to make the outgoing (sent mails) from the shared mailbox addresses appear in the sent items folder of the shared mailbox, you can refer to the steps below:

note: since the parameter can be only run by the admin, if you’re not an admin, you need to let the admin to run it. if someone who isn’t an admin runs the parameter, he/she will get error messages.

1) connect to exchange online using remote powershell
2) run the command: set-mailbox <mailbox name> -messagecopyforsentasenabled $true
3) run the command: set-mailbox <mailbox name> -messagecopyforsendonbehalfenabled $true
 

after running the command, the emails sent from shared mailbox addresses will appear in the sent items folder of the shared mailbox. also you will see these sent emails in the sent items of the main office 365 licensed account.

regards,
violet
