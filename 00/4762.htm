Let's assume cell A1 holds the true Excel date 05/01/2011.

=YEAR(A1)&TEXT(A1,"mm")

That will return the result as a TEXT string. If you want the result as a numeric number then:

=--(YEAR(A1)&TEXT(A1,"mm"))
