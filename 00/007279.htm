ashish mathur say:

HI,

 

The double das symbol is used for converting TRUE to 1 and FALSE to 0.  So while this portion of the formula will return a virtual array of TRUE's and FALSE's

 

ROW(OFFSET($A$1,,,ROWS(Source!$C$4:$C$425)))>=COLUMN(OFFSET($A$1,,,,ROWS(Source!$C$4:$C$425)))

 

the one below will convert the TRUE's to 1's and FALSE's to 0's

 

--(ROW(OFFSET($A$1,,,ROWS(Source!$C$4:$C$425)))>=COLUMN(OFFSET($A$1,,,,ROWS(Source!$C$4:$C$425))))

 

Hope this helps.

You sync media (photos, music, podcasts) between a computer and windows phone via the Zune desktop software.
