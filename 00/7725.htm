Outlook Online (OWA) / Retention Policies available - Microsoft Community

charlesmip say:

Outlook Online (OWA) / Retention Policies available

Hello,

We have users using Outlook Online which "by default" have in their "Retention Policies" all the policies we created.

Some others just have an empty list and have to use the "+" button to add them one by one.

We have try to set again the "Default MRM Policy" on the latter, but result is the same.

One user with all policies (that's why we'd like to have for all our users):

Another user with no policy:

Any idea why some have all the policies added by default and others don't?

Thanks!

charlesmip say:

We found the explanation to this one.

If you apply to the user a retention policy that includes a default retention tag configured with a:

• defined retention period, then the user will get by default all the retention tags we made available in the policy (case "user with all policies" in the question). He will get the “Assign policy” contextual menu with all the retention tags we’ve made available.

• unlimited retention period (hold indefinitely), then the user will have no retention tags in his retention policy (case "user with no policy" in the question). Consequently he will not even get the “Assign policy” contextual menu unless he adds retention tags manually using mail settings > retention polices > add (+).

When we started to enable Online Archive, we got feedback from user complaining their emails were archived without them knowing why. This was because when the Online Archive is enabled, the default retention policy is applied to the user mailbox. In this default retention policy, we had a default retention tag (the one that applies to the whole mailbox) defined to move items to archive after a 2 year retention period.

We decided to edit the default retention tag to set the retention period to "hold indefinitely", which basically tells "never archive" automatically. Doing so, we were able to enable the Online Archive without having emails being archived without user knowing why.

But, this is when we started to have two groups of users: the ones for who the Online Archive was enabled before the change, and the ones for who it was enabled after. This explained why we had users with "all policies" and others with none by default.

How to record FM radio - Microsoft Community

michaels---01 say:

How to record FM radio

Some of our members have been asking about FM radio recording, a feature that is available in a small number of Nokia phones, so here is a quick guide:

 

Which Nokia Phones support FM recording?

  • Nokia C1-02
  • Nokia X2-02
  • Nokia X2-05
  • Nokia 110
  • Nokia 112
  • Nokia 113
  • Nokia 114
  • Nokia Asha 200
  • Nokia Asha 201

 

Is FM recording available everywhere?

Due to legal restrictions in some countries, FM recording is not available on all variants of the phone models mentioned above. The feature is available only in phones sold in the following countries:

 

Bahrain, Bangladesh, Brazil, Brunei, Cambodia, Chile, China (Mainland), Colombia, Dominican Republic, Ecuador, Egypt, Guatemala, Honduras, India, Indonesia, Iran, Iraq, Jamaica, Kuwait, Laos, Lebanon, Malaysia, Mexico, Morocco, Nepal, Pakistan, Palestine, Panama, Paraguay, Philippines, Singapore, Syria, Tanzania, Thailand, Tunisia, Turkey, Uganda, Uruguay, Venezuela, Vietnam and Yemen.

 

How do I use FM recording?

 

1. Ensure that you have the wired headset connected to the phone

2. Select Menu > Music > Radio to open the FM Radio.

3. Whilst the radio is playing, select Record to begin recording.

4. To end recording, use the same button to select Stop.

5. After recording, you will be prompted to enter a name for the file. You can type the filename of your choice or just keep the suggested name. When you are ready to save, select OK.

6. To play back recordings, use the left-side button to select Options...

7. ...then scroll down and select Recordings.

8. Choose the saved recording that you wish to play then select Open.

9. During playback of your recordings, you can pause and resume the playback by pressing the navigation key.

  

 

Why does my Nokia C1-02, X2-02, X2-05, 110, 112, 113, 114, Asha 200 or Asha 201 not have the Record option?

If the Record option is not displayed when you are listening to the radio, this means that your phone was manufactured for one of those countries where the FM Recording feature is not available due to legal reasons.
