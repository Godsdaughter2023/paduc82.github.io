jeeped say:
You can actually accomplish this with a standard formula using INDEX() in its array form.

     

The formula in G3 is,

=IFERROR(INDEX($C$2:$C$999,SMALL(INDEX(ROW($1:$998)+((($A$2:$A$999<>$E$2)+($B$2:$B$999<>G$2))*1E+99),,),ROW(1:1))),"")

This does not require Ctrl+Shift+Enter. Copy or fill right then down. I've included both on the same worksheet. Post back if you have trouble transcribing this for your own purposes with the results on a second worksheet.

camcafe say: Ashish,

Your example is only based on one criteria. I need it based on two criteria.

jeeped say:
FALSE is the equivalent of 0 (aka zero). Since you are looking for the SMALL() (or MIN()) ROW(), the first match would be any FALSE since rows start at 1 and you are returning a 0. Returning 1E+99 when FALSE will never match a MIN() or SMALL() when looking for a ROW() as it is larger than the largest ROW() number on any worksheet.

Put another way, if you run the array formula through the Formula tab's Evaluate Formula, you will see the array of TRUE and FALSE turn to row numbers and 1E+99's. Remove the 1E+99's and you get row numbers and zeroes. A MIN() or SMALL() will always return a zero before a valid row number. The <> is used because you are trying to exclude non-matches. By strict definition, if you exclude all non-matches, then you are left with matches.

  1. Enable the display of nonprinting characters through the ¶ button in the Standard toolbar.
  2. Create a new document.
  3. Go to Page Setup > Manage Custom (third dropdown) sizes, click on the+ sign and create a US Letter Quarter page (4.25 x 5.5) with zero margins all around.
  4. Go to your source document, press End (to go to the end of the document) and press Return. This will create a blank paragraph containing a lot of information (like page size) that you don't want to copy.
  5. Do Edit > Select All (or Command A), then press Shift and hit the left arrow once. This will deselect that last unwanted paragraph.
  6. Do Copy and then Paste into the blank document. You should end up with a blank paragraph at the end.
  7. In the new document, go to Format > Document  and redefine the margins so your coupon will look OK and/or centered. Fine-tune the paragraph margins. You could set Format > Document and Copy to the clipboard.
  8. Again, go to the end. Press Shift Enter (or Insert > Page break) 
  9. Repeat so you have four identical pages. If you want to delete the last ¶ mark, use a Forward Delete.
  10. Do Print and in the third dropdown menu, select Layout.
  11. Choose Pages per sheet = 4 and perhaps add a framing Border.
  12. Click on Preview to check.

If your model takes about a whole US-Letter page
  1. Do Steps 4 & 5 and Copy. Do Steps 9 & 10. This will give you 4 pages.
  2. Do Steps 11 & 12.
  3. In Paper handling, select "Scale to fit paper."
  4. Do Step 13.

or:

{IF«State»= "local" {IF«brfcat»= "a" "Text1"}}
{IF«State»= "local" {IF«brfcat»= "c" "Text4"}}
{IF«State»= "local" {IF«brfcat»= "e" "Text3"}}
{IF«State»= "foreign" {IF«brfcat»= "d" "Text1"}}

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. Likewise, you can't type or copy & paste the chevrons (i.e. '« »') - they're part of the actual mergefields, which you can insert from the 'Insert Merge Field' dropdown. The spaces represented in the field constructions are all required. Although the fields are shown laid out on separate lines, that's just for legibility; you'd probably want them all on the same line.

The question-square represents certain special characters.  In your case, they are probably linefeed characters (CHAR(10)).

Carriage return (CHAR(13)) does not appear as question-square.  It is not displayed, although it does count as a character in the text.

So the actual characters around the question-square might be just linefeed (CHAR(10)), carriage-return linefeed (CHAR(13) CHAR(10)) or linefeed carriage-return (CHAR(10) CHAR(13)).

If you simply do not want to see the question-square, you can enable wrap-text.  Select the cell(s), then press the wrap-text button on the Home tab, indicated below.

However, if you want to remove them altogether, the following should do the trick:

=SUBSTITUTE(SUBSTITUTE(A1,CHAR(10),""),CHAR(13),"")

If that formula is in B1, you could then copy B1 and paste-special-value into A1.  Then you can delete B1.
