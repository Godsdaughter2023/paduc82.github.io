Hello Tha Sandman101,

     Thank you for patiently waiting for our official response. We were able to determine the reason that content gets removed from the Microsoft Store. The main reason that content gets removed is because the developer chooses to stop providing support for games or providing content for those games. The developer has an agreement with Microsoft to maintain the game, but once the developer chooses to stop supporting the game it may get removed. This does not occur with backwards compatible games as Fable's developers are no longer in business, but the game is still available on the Microsoft Store.

     The type of games that most commonly experience removal from the Microsoft store are franchise games. These types of games are NBA 2k, WWE 2K, Madden and more. The developer will come out with a new version of the same game almost every year and continue to provide support for some of the previous versions of the game, but will stop supporting much older versions of the game. The reason that Microsoft will remove a game from the Microsoft Store can also be attributed to copyright or licensing issues. This situation does not occur very often, but it is possible.
