suzanne s. barnhill say:
  1. Press Ctrl+H to open the Replace dialog, then select the Find tab.
  2. Click More>> to expand the dialog.
  3. Click Format, then Font, and choose 16 as the size (don't change any of the other settings).
  4. Click OK. You'll see "Format: Font: 16 pt" beneath the "Find what" box.
  5. Click Find In and select "Main Document." All the 16-point strings will be selected.
  6. Click the title bar of the window to transfer focus back to the document without deselecting the selected text.
  7. Press Ctrl+C to copy.
  8. Press Ctrl+V to paste the text into a new document. Each string will be pasted as a separate paragraph.

macropod say:

You can do this with Find/Replace.

First, mark all text as hidden. If everything disappears, click on the ¶ symbol on the Ribbon's Home tab to make it visible again.

Then use Find/Replace to find all 16pt text and change it to unhidden.

Another Find/Replace for hidden text will delete anything not in 16pt.

What you'll be left with is a document containing just the 16pt text. Save that as a new document.

Notifications in Outlook for Mac 2011 - Microsoft Community

jc_26 say:

Notifications in Outlook for Mac 2011

For some reason Outlook for Mac 2011 only provides sound notifications for the main Inbox Folder. I setup folders and rules to get emails on specific folders, but there is no sound or notification when emails arrive to the selected folders, which is not ideal. There is only sound (and sometimes notification) for emails arriving to the main Inbox...Any idea how to correct this issue? Thanks.
jc_26 say:

Hello,

I finally found the solution.

The problem is that Rules are setup on the Exchange Server and they apply without the need to have Outlook open; one restriction of these Exchange Server Rules is that they cannot display notifications or play sounds.

To play notifications, sound and some other options for emails arriving to a folder following an Exchange Server rule:

  • Under Outlook Preferences select "Sync Services" and "On My Computer". You need to select a folder on your HD to add new items to.
  • Under "Rules" select "Edit Rules". You will find two categories: "EXCHANGE SERVER" and "ON MY COMPUTER"
  • Select "Exchange" under ON MY COMPUTER
  • Click the "+" at the bottom to create a new rule.
  • Give the rule a name.
  • In the From - Contains add the email of the recipient that the rule will apply to (you can avoid this step if you select an email from the recipient before starting the process as Outlooks adds it automatically here)
  • Under "Do the Following:
    • Select  "Change Status" and change it to "Display Notification" - you can select Show Subject Only or Show Subject and Preview
    • Select "Set Category" and change it to "Play Sound" - the New Mail Sound is selected automatically
  • The new emails on your folders will now have Notifications and Sounds :-)

Note: Before I went to the process above I decided to remove the selection on the "Do not apply other rules to messages that meet the same conditions" in the EXCHANGE SERVER rules (you edit them using a double click). I tested the above process sending an email to myself and it works (best to give things as they are when it works, right?).

Hope the above is useful.
