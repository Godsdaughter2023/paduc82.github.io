Some of the entries would pertain to the built-in Windows Virtual CD-ROM that appears when mounting an ISO image.

In any event the Code 39 error may be from the Afc value.

Afc may indicate the afc.sys was installed on the system.

This afc.sys driver may have been installed with a third party software associated with the drive perhaps an Arcsoft program.

This program may have been uninstalled but the reference to the afc.sys remained in the registry.

The afc.sys is trying to be loaded via the registry but as the afc.sys may no longer exist on the system the Code 39 error occurs.

To remedy the issue do the following:

 

First create a System Restore Point.

Restart the computer.

Upgrade to Windows 7 from Vista - Microsoft Community

armyfiremn say:

Upgrade to Windows 7 from Vista

If i upgrade to Windows 7 from Vista will i lose my Adobe and Microsoft Office Programs???
andre da costa say:

If you select the option to upgrade during installation, your personal files, settings and installed programs will be preserved, so yes, Adobe and Microsoft Office would not have to be reinstalled.

Upgrade Paths:

When making significant changes to your computer such as an upgrade, it is always recommended you backup your system prior to installing a new version of Windows. In your case, you can upgrade to Windows 7 depending on edition and architecture of Windows Vista you have installed. If you are running a 32 bit version of Vista and want to upgrade to a 64 bit version (edition of Windows 7), you will have to a do clean install, this applies vice-versa.

If you are running consumer editions of Windows Vista such as Home Basic or Premium, you can only upgrade to either Windows 7 Home Premium or Ultimate. Also, you cannot do downgrades, which means, if you are running Windows Vista Ultimate now, you cannot downgrade to Windows 7 Home Premium. You must do logical upgrades:

Windows Vista Home Basic > Windows 7 Home Basic, Premium or Ultimate
Windows Vista Home Premium > Windows 7 Home Premium or Ultimate
Windows Vista Business > Windows 7 Professional, Enterprise or Ultimate
Windows Vista Enterprise > Windows 7 Enterprise
Windows Vista Ultimate > Windows 7 Ultimate.
