darian tab say:

Hi,

You can manually permit a program to access the internet by opening a firewall port. You will need to know what port it uses and the protocol to make this work. Let's follow these steps to open ports in Windows 10 firewall:

1. Navigate to Control Panel, System and Security and Windows Firewall.
2. Select Advanced settings and highlight Inbound Rules in the left pane.
3. Right click Inbound Rules and select New Rule.
4. Add the port you need to open and click Next.
5. Add the protocol (TCP or UDP) and the port number into the next window and click Next.
6. Select Allow the connection in the next window and hit Next.
7. Select the network type as you see fit and click Next.
8. Name the rule something meaningful and click Finish.

Let us know if you need any other assistance.

Regards.

a. user say:

Hi,

Thank you for posting your query in Microsoft Community.

Sorry for the inconvenience caused.

Your default keyboard layout or input method is the one that’s automatically used with the language you see Windows in (for example, the QWERTY keyboard for American English). If you want to keep your Windows language the same but use a different keyboard automatically, follow these steps:

  1. Open Control Panel by right clicking on the Start Button.
  2. Enter Add a language in the search box, and then tap or click Add a language.
  3. Tap or click Advanced settings.
  4. Under Override for default input method, choose the keyboard layout or input method you'd like to be your new default, and then tap or click Save.

Note: If you don't see the keyboard layout or input method you want, then you haven't added it to your input method list. To Add a keyboard layout or input method for a language see the steps below:

  1. Type Region And Language in the Windows search Bar and select  Region and language.
  2. Tap or click the language you want to add a keyboard to, and then tap or click Options.
  3. Tap or click Add a keyboard, browse the input method list for the one you want to use, and then tap or click it.

Hope this helps.
