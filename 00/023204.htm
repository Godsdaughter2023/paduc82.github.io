rangao_msft4227 say:

Hi Ashok,

There are no settings on the Office 365 portal to disable the notification.

For individual users, you can change the following value in registry key to disable the notification.

  1. Open Registry Editor and go to:
    HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\15.0\ClickToRun\Updates
  2. Change the value for “UpgradeAdvertised” to 0.

Note: Always back up your registry before making any changes.

In addition, you can disable the download of Office 2016 in Office 365 admin center. Go to Office 365 admin center -> Service Settings -> User Software, then uncheck the Office 2016 version.

    gord dibben say:

    This sample event code will do that without hitting a button.

    Copy/paste the code into Worksheet module of Sheet2

    Private Sub Worksheet_Change(ByVal Target As Range)
    'assumes dv list in Sheet2 B3
    Dim sht1 As Worksheet
    Dim destrng As Range
    Set sht1 = Sheets("Sheet1")
    Set destrng = Sheets("Sheet2").Range("G3")
        If Target.Address = "$B$3" And IsNumeric(Target.Value) Then
            Select Case Target.Value
                Case 1: sht1.Range("A3:E7").Copy _
                                Destination:=destrng
                Case 2: sht1.Range("A8:E12").Copy _
                                Destination:=destrng
            End Select
        End If
    End Sub
