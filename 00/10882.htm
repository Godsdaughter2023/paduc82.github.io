Fixed range of 11 cells would be something like

Range("A1:K1")

The next empty cell for, say, column K is:

Cells(Rows.Count,"K").End(xlUp)(2)

The row number of the last filled cell in, say, column A is:

Cells(Rows.Count,"A").End(xlUp).Row

To fill K through U to match column A, would be this

    Dim lngR As Long
    Dim rngC1 As Range
    Dim rngC2 As Range
    
    Set rngC1 = Cells(Rows.Count, "K").End(xlUp)(2)
    lngR = Cells(Rows.Count, "A").End(xlUp).Row
    Set rngC2 = Cells(lngR, "K")
    Range("A1:K1").Copy Range(rngC1, rngC2).Resize(, 11)
