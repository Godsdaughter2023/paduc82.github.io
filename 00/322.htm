How to display students with highest score in each subject - Microsoft Community

paulanim say:

How to display students with highest score in each subject (section).

Hi Support,

I have two tables namely; Score and Students tables. The fields of the tables are shown below.

I want to write a query to display students with the highest average mark in each section. I want the query to display the student name and **** from the student table and the score (average field value) .

Thank you

ken sheridan say: The most efficient method is normally by means of a JOIN to a subquery which returns the MAX average per section:

SELECT Students.*, S1.Section, MaxAverage
FROM (Students INNER JOIN Score AS S1
ON Students.APID = S1.APID)
INNER JOIN
    (SELECT Section, MAX(Average) AS MaxAverage
     FROM Score
     GROUP BY Section) AS S2
ON S1.Section = S2.Section AND S1.Average = S2.MaxAverage;

Alternatively, but usually less efficient, is to use a correlated subquery to restrict the outer query:

SELECT Students.*, Section, Average AS MaxAverage
FROM Students INNER JOIN Score AS SC1
ON Students.APID = SC1.APID
WHERE Average =
     (SELECT MAX(Average)
      FROM Score AS SC2
      WHERE SC2.Section = SC1.Section);

The latter, unlike the former, will return an updatable result table, though it's hard to envisage a legitimate situation in which you would want to update any of the values returned.

One thing to which I would draw your attention, however, is that Name is a bad name for a column.  As the name of a built in property, as a 'reserved word' it should not be used for object names.  Name the column StudentName or similar, or even better split the name into FirstName and LastName columns for example.  You can easily return the full name by concatenating the values of the two columns, whereas parsing a single value in to its constituent elements is a little trickier.
