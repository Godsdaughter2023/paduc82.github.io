kimi gao msft say:

 Hi Elena,

 

An Office account and a domain name is one-to-one relationship which means a domain name can only be used by one Office 365 account.

The users using the seconddomain.com need be changed the domain name or removed if you have applied for a new Office 365 account with seconddomain.com. If you just change the domain name of the users in the original account, the mailboxes information will be kept.

Meanwhile, you need to manually add the new users with the seconddomain.com in your new Office 365 account and new users will automatically generate mailboxes. The procedure of the migration should be complicated, but this is the only method meets your requirement. We suggest you take a full consideration to carry it out.

Regards,

Kimi

kimi gao msft say:

Hi Elena,

Your understanding about the migration of SharePoint team site domain is definitely correct. The SharePoint online site domain is related with the Office 365 account name. If our account’s domain is exampledomain.com, then the SharePoint team site’s domain name is exampledomain.sharepoint.com and it can’t be changed. We need to apply for a new Office 365 account if we want to show the SharePoint team site’s URL with another domain name as you said.

After you have registered for a new Office 365 account using seconddomain.sharepoint.com domain name, to attach the seconddomain.sharepoint.com in your new account, you need to remove all the things related with seconddomain.sharepoint.com in your original account so that the new SharePoint team site will work with seconddomain.sharepoint.com.

Hello Island_Girl_FL

Based on your description, you are asking about an Outlook.com account (the forum in which you posted) and using the Outlook desktop program, part of the Office 365 suite. On that basis

#1 - Your Outlook.com account (outlook.live.com) is an <email account>
#2 - The Outlook desktop program is an <Email client> which connects to <email accounts>

*** just in case it's of interest, I have an article on the difference between the two at

Understanding Email Accounts and Email Clients
https://www.contactgenie.info/understanding-ema...

#3 - Suspect that you have not configured (added) an (your) Outlook.com account to the Outlook desktop program. In order to sync contact/calendar data between Outlook desktop and an <email account> requires that the email account is configured as an Exchange account (the default for Outlook.com).

#4 - Contact/calendar data that is contained in folders for any account configured as either a POP3 or IMAP account does not get sync'd with the server. If you're not sure how your accounts have been configured, go to Outlook desktop menu --> File --> Account Settings --> Email tabs (type of account is in the right hand column)

#5 - At this point, suspect that you only have one contact and calendar folder configured in Outlook desktop. In case you have multiple email accounts configured, you can see which folder tree the folders belong to by clicking CTRL+6 in the Outlook Nav Pane to see the full folder list. As a FYI, the Outlook Nav Pane lists all the data files configured in Outlook desktop - it is not a list of <email accounts>.

Summary answer to your question

a) add your Outlook.com account to the Outlook desktop program

Add an email account to Outlook
https://support.office.com/en-us/article/add-an...

b) within Outlook desktop, go to FIle --> Account Settings --> Data FIles tab --> select the Outlook.com .ost data file and set it as the default for the profile (the default data file is where the default contact/calendar folders are stored)

b) once the account is added, move any contact/calendar data you want to share between devices to the respective contact/calendar folders in the Outlook.com folder tree.

If anything is not clear - just ask.

bernie deitrick, excel mvp 2000-2010 say:

% required is simply  TotalBills/TotalIncome  

The specific formula will depend on your cells, but something like:

=H8/H9

where H8 is the pulled bills total, and H9 is the pulled income total, formatted for percentage.

Amount required from earner 1 is simply % times Monthly income, like

=Sheet1!B2*$H$10

Note the aboslute reference to the cell with the percentage (H10) which will stay constant when the formula is copied down, to reference the other incomes.
