malcolm makin say:

How To Activate Animation on Slide Show Window using VBA

I'm trying to use VBA to trigger an animation for shape on SlideShowWindow.  The best way I could think to accomplish this would be to set animation's trigger to a click event on another shape, and then use the Windows API to simulate a click on the triggering shape.  I actually have gotten far in figuring this out, but where I am currently stuck is getting the coordinates of the shape as it appears on the SlideShowWindow.  Shape.Top and Shape.Left give values in points relative to the slide, and if I try converting to pixels they don't seem to line up to where they should be (comparing to coordinates using a screen capture).  So, I guess my question could be answered in one of two ways: 1. How to trigger an animation on a SlideShowWindow in VBA.  or 2. How to retrieve the coordinates of a shape on the SlideShowWindow, and convert into pixels that match the screen dimensions.
malcolm makin say: Hi Cristin,

Thanks for your reply.  I meant to post back here earlier but have been a bit busy lately.  After much searching, I have come up with something that does what I want.  It took a lot of investigating to determine what was happening in PowerPoint, and how to convert points to pixels, determine screen ratio, etc.

My solution, broken down, is this:
  1. Set each animation to be triggered by a click action on another shape (since this is a gameshow presentation, there are corresponding shapes on the slide already, so I didn't have to add arbitrary shapes).
  2. Use the Windows API to determine the slideshowwindow handle
    a) This was achieved by using FindWindowEx, searching for the text "PowerPoint Slide Show - [Presentation Name]". 
    b) However, that handle is not the correct window to send messages to (oddly enough), it is a child window.  So, I looped through all child window handles, returned a windows RECT using GetWindowRect, and examined the dimensions of each child window.  Only one of them actually has width & height, so that is the one to use.
  3. Determine the zoom ratio using SlideShowWindow.View.Zoom / 100
  4. Get the screen coordinates of the shape to be clicked on (to animate the desired shape)
    a) Taking into account that the active slide does not fill the entire screen when presented (if using a widescreen display), Shape.Left is the X coordinate, but needs to have the "offset" added to it: (SlideShowWindow.Width - (Presentation.PageSetup.SlideWidth * ScreenRatio)) / 2
    b) Using the calculated offset, the shape's screen coordinates are (in points, not pixels) determined as: Shape.Left * Ratio + Offset;  Shape.Top * Ratio
    c) Now convert the coordinates into pixels.  There are plenty of examples of how to do this, so I won't list it here.
  5. Now that the coordinates have been determined, use PostMessage to send a mousedown and mouseup messages to the SlideShowWindow at the specified location
You are indeed correct, it did take a bit of code to figure out, but I am happy with the result.  I am just surprised in all my searching that someone hadn't posted an answer such as this yet (even though this question has apparently been asked for years now but numerous people).

hansv mvp say:

Here is a macro you can use. Modify the constants at the beginning to match your situation. For example, if the data begin in row 2 instead of row 1, change

    Const FirstRow = 1

to

    Const FirstRow = 2

To be on the safe side, make a copy of the worksheet or workbook before running the macro.

Sub Zipper()
    Const Col1 = 1 ' column A
    Const Col2 = 2 ' column B
    Const FirstRow = 1
    Dim CurRow As Long
    Application.ScreenUpdating = False
    CurRow = FirstRow
    Do
        If Cells(CurRow, Col1).Value <> "" And Cells(CurRow, _
                Col1).Value < Cells(CurRow, Col2).Value Then
            Cells(CurRow, Col2).Insert Shift:=xlShiftDown
        ElseIf Cells(CurRow, Col2).Value <> "" And Cells(CurRow, _
                Col2).Value < Cells(CurRow, Col1).Value Then
            Cells(CurRow, Col1).Insert Shift:=xlShiftDown
        End If
        CurRow = CurRow + 1
    Loop Until Cells(CurRow, Col1).Value = "" And _
        Cells(CurRow, Col2).Value = ""
    Application.ScreenUpdating = True
End Sub

Sub DeleteTrailingComma()
    Dim cel As Range

If the values in your columns are numbers:

Sub Zipper()
    Const FirstCol = 1 ' column A
    Const LastCol = 5 ' column E
    Const FirstRow = 1
    Dim CurRow As Long
    Dim CurCol As Long
    Dim MinCol As Long
    Dim MinVal As Long
    Application.ScreenUpdating = False
    CurRow = FirstRow
    Do
        MinVal = 999999999
        For CurCol = FirstCol To LastCol
            If Cells(CurRow, CurCol).Value <> "" And Cells(CurRow, CurCol).Value < MinVal Then
                MinCol = CurCol
                MinVal = Cells(CurRow, CurCol).Value
            End If
        Next CurCol
        For CurCol = FirstCol To LastCol
            If Cells(CurRow, CurCol).Value <> "" And Cells(CurRow, CurCol).Value > MinVal Then
                Cells(CurRow, CurCol).Insert Shift:=xlShiftDown
            End If
        Next CurCol
        CurRow = CurRow + 1
    Loop Until Application.CountA(Range(Cells(CurRow, FirstCol), Cells(CurRow, LastCol))) = 0
    Application.ScreenUpdating = True
End Sub
