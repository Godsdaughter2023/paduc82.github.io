Because the default entry for Excel is time - hours and minutes.  That is just the way it is.  If you want, you can use event code like the snippet below, to change hours and minutes to minutes and seconds: This works on entries in column D.

Copy this code, right-click the sheet tab of the sheet of interest, select "View Code" and paste the code into the window that appears. Save the workbook as a macro-enabled .xlsm file.

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Column <> 4 Then Exit Sub
If Target.Cells.Count > 1 Then Exit Sub
If Target.Value = "" Then Exit Sub
If Target.Value < 1/24 Then Exit Sub
On Error Resume Next
Target.Value = Target.Value / 60
End Sub

La semaine dernière notre équipe chargée des mises à jour logicielles a commencé le déploiement de la dernière mise à jour pour le Nokia N9 pour un certains nombres de Pays.  Aujourd'hui, la mise à jour logicielle est disponible pour plus de 70 pour cent des terminaux, et ce chiffre continue d'augmenter. Si vous n'avez pas encore reçu une notification de mise à jour, merci de patientez encore un peu ou contactez votre opérateur local pour savoir si il a autorisé la mise à jour.

 

La version logicielle 30.2012.07 (PR1.2) apporte plein  de nouvelles fonctionnalités tel que:

- la compatibilité avec l'appel vidéo Gtalk (si vous ne l'avez pas encore, téléchargez le sur l'app store)

- la compatibilité avec les dossiers dans la visualisation des Applications

- une amélioration des fonctionnalités de l'Appareil Photo, y compris la prise de clichés en rafale, et la reconnaissance faciale pour les photos, pour que vous puissiez tagger facilement les visages avec des noms

- des nouvelles fonctionnalités pour Nokia Drive, tel que les options d'itinéraire, les favoris, les alertes en cas de dépassement des vitesses autorisées et la compatibilité avec la boussole

- une nouvelle visualisation des emails et la possibilité d'accéder au répertoire de la société dans Mail for Exchange (MfE)

- la possibilité de création des listes de lecture et la possibilité de supprimer les morceaux via le lecteur de musique

- des améliorations dans la visualisation de l'historique de navigation web

- la possibilité de copier-coller lors de la navigation internet et dans les documents

- la compatibilité avec les langues Thaïlandais, Hébreu, Perse, Vietnamien et Kazakh

 

Donc, allumez votre Nokia N9, activez le WiFi et soyez prêt à télécharger la dernière version logicielle en cliquant soit sur la notification de mise à jour logicielle ou en effectuant une vérification manuelle via Paramètres >  Applications > Gérer les Applications puis en cliquant sur l'onglet Mise à jour logicielle. Si vous n'avez pas de point d'accès WiFi proche de vous, vous pouvez aussi obtenir la mise à jour logicielle via le logiciel Nokia Software Updater (pour le télécharger, vérifiez les pages de support du Nokia N9 via www.nokia.com/support ou rendez vous sur swipe.nokia.com).  Si vous souhaitez faire une sauvegarde de vos données, vous pouvez utiliser Nokia Link.

Setting up iCloud with Alias Email in Outlook Mac - Microsoft Community

jdlandifer say:

Setting up iCloud with Alias Email in Outlook Mac

There is another, closed thread on this, but I wanted to post clearer instructions.  Here is the previous thread that, at least to me, is pretty ambiguous

Here's what I did to set up Outlook for Mac client app (Version 16.16) to send and receive from my iCloud alias email:

  1. Set up an app-specific password in iCloud.  This Apple Support thread gives detailed instructions. The step is necessary if two-step authentication is enabled (which it should be!) on your iCloud account.  Apple gives you an app-specific password that you can copy/paste into Outlook account setup
  2. Add the iCloud account using your default (non-alias) email or login ID.  Go to Settings/Accounts in Outlook Preferences menu and add a new account.  Enter your iCloud account login and paste in the app-specificpassword.  Outlook takes care of the rest of the configuration
  3. In the Outlook account's preferences, change the E-mail address field to your iCloud alias address but leave the User name field as your default iCloud login name

This results in emails sent from the account as the alias address, and the account will receive emails addressed to either the alias or default icloud address.  In the Outlook email pane, the Inbox etc. will be listed under the default icloud address.
