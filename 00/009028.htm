I don't believe that syntax is available in Access queries. You may need to use a subquery.

For numbering the orders over customers in the Northwind Orders table:

 

SELECT Orders.CustomerID, Orders.OrderID, Orders.OrderDate,

(SELECT Count(*)

FROM ORDERS O

WHERE O.CustomerID = Orders.CustomerID and O.OrderDate <=Orders.OrderDate) AS Row_Num
FROM Orders
ORDER BY Orders.CustomerID, Orders.OrderDate;

gary's student say:

Say we want to make cells A1 and B2 mandatory on Sheet1.  Enter the following event macro in the workbook code area:

 

 

Private Sub Workbook_BeforeClose(Cancel As Boolean)
Sheets("Sheet1").Activate
Set r1 = Range("A1")
Set r2 = Range("B2")
If IsEmpty(r1) Or IsEmpty(r2) Then
    MsgBox "please fill the required cells"
    Cancel = True
End If
End Sub

john korchok say:

Different versions of Word come with different defaults for the Normal.dotm template. This template creates your document formatting when you create a new blank document.

You can set a new linespacing that will be recorded in Normal.dotm, so that all future documents will use your revised spacing:

  1. Select text a paragraph of text that is to be the basic text style. This is most often in the default Normal style.
  2. Choose Format>Style from the OS X menu at the top of your screen.
  3. Click on the Modify button.
  4. Check the Add to template option. This is the option that adds the style change to Normal.dotm, so all future documents will use this styling.
  5. Click on the Format dropdown button and choose Paragraph.
  6. Set your preferred before and after spacing. OK out.
  7. Create a new blank document and check that the paragraph spacing looks the way you want it.

brian tillman [outlook mvp 2007-2019] say:
When you have Outlook configured with only POP accounts, the default account will be used when you create a new message.  No Outlook prior to 2010 had more than one delivery location, technically (although IMAP accounts always had their own folder sets), so the default account was always used for a new message.  Now that Outlook supports multiple delivery locations, my guess is that Microsoft thought it made more sense to use the account associated with the delivery location.

graham mayor say:
There is no autosave in any version of Word. There is only the saving of autorecovery information in the event of a crash, which may then aid the recovery of the document next time Word is started. This information is not stored in the default save path.

If you want autosave it can be achieved with an add-in - see http://www.gmayor.com/automatically_backup.htm
