suzanne s. barnhill say:

Okay, I think your Word version was reported incorrectly. If you have the Developer tab, then you definitely don't have Word Starter. I'm away from the computer where I created this, but I'll try to remember the steps I used:

  1. I inserted a picture, sized it to "bullet size," and set the text wrapping to In Line With Text.
  2. With the picture selected, I inserted a frame. I have an Insert Frame button on the QAT, but you can use the button in the Legacy Forms palette of the Legacy Tools in the Controls group on the Developer tab.
  3. You'll need to remove the border from the frame, so select the frame (not the graphic in it) by clicking on the hashed border. You'll then be able to use the No Border button on the Borders dropdown in the Paragraph group on the Home tab.
  4. I found that the graphic in the frame (which is now in its own paragraph) had the same formatting as the paragraph in which I'd inserted it. That happened to be Normal, so it had 1.15 line spacing and 10 pts Spacing After. I had to set it to Single with no spacing to make the text wrap correctly around the frame. You can also adjust the "Distance from Text" in the Format Frame dialog (double-click on the frame border to access that).
  5. Once you've got the framed graphic sized and placed to your satisfaction, select it and press Alt+F3. This will open the Create New Building Block dialog with AutoText selected as the default. Type a name for the entry (you can also change the gallery if desired) and click OK.
  6. You should be able to insert your bullet by typing the first four letters of the entry name and pressing Enter when you get the AutoComplete suggestion, but if that doesn't work, you can type a few letters and press F3 or use Insert | Text | Quick Parts | AutoText. The second bullet in my screen shot was inserted this way.

My example was just a rough-and-ready trial, but you should be able to duplicate it using your graphic. It's nowhere near as convenient as a true bulleted style, but at least it's a workaround.

With IMAP, you can only sync email and you can't sync anything with POP3. To sync content from these account types with an iPhone, you'll need to use iCloud or iTunes from Apple.

The best solution would be to sync directly with the mail server as you'd always be up-to-date as long as you have an Internet connection. EAS stands for Exchange ActiveSync which is a synching protocol targeted towards mobile devices. You don't necessarily need a corporate Exchange server account for this; Outlook.com also offers support for this protocol.

Outlook.com allows you to configure it with your current POP3 or IMAP address so you can use it without needing to change your address.

Sub Copy2Final()
'
' Moves Data from Main Sheet to Final Destination
'
Dim TerminationDate(1000) As String
Dim EmployeeId(1000) As String
Dim EmployeeName(1000) As String
Dim JobTitle(1000) As String
Dim RecordNumber(1000) As String
Dim wkMainSheet As Worksheet: Set wkMainSheet = Sheets("Main Sheet")
Dim wkFinalDestination As Worksheet: Set wkFinalDestination = Sheets("2017") ' Change the year value to match new sheet
Dim termDateCellColumn As Integer: termDateCellColumn = 1
Dim employeeIdCellColumn As Integer: employeeIdCellColumn = 2
Dim employeeNameCellColumn As Integer: employeeNameCellColumn = 3
Dim jobTitleCellColumn As Integer: jobTitleCellColumn = 4
Dim recordNumCellColumn As Integer: recordNumCellColumn = 5

Dim targetCell As Integer

'Iterate through source sheet and grab data
For x = 2 To 1000
    If wkMainSheet.Cells(2, 2) <> "" Then
        TerminationDate(x - 2) = wkMainSheet.Cells(x, 1) 'termDateCellColumn)
        EmployeeId(x - 2) = wkMainSheet.Cells(x, 2) 'employeeIdCellColumn)
        EmployeeName(x - 2) = wkMainSheet.Cells(x, 3) 'employeeNameCellColumn)
        JobTitle(x - 2) = wkMainSheet.Cells(x, 4) 'jobTitleCellColumn)
        RecordNumber(x - 2) = wkMainSheet.Cells(x, 5) 'recordNumCellColumn)

    Else
        'if the cell is blank then exit the loop
        Exit For
    End If
Next

'Iterate through each array and merge them into the main sheet
For x = 2 To 100000
    If wkFinalDestination.Cells(x, termDateCellColumn) = "" Then
        targetCell = x
        Exit For
    End If
Next

For x = targetCell To (targetCell + 1000)
    wkFinalDestination.Cells(x, termDateCellColumn) = TerminationDate(x - targetCell)
    wkFinalDestination.Cells(x, employeeIdCellColumn) = EmployeeId(x - targetCell)
    wkFinalDestination.Cells(x, employeeNameCellColumn) = EmployeeName(x - targetCell)
    wkFinalDestination.Cells(x, jobTitleCellColumn) = JobTitle(x - targetCell)
    wkFinalDestination.Cells(x, recordNumCellColumn) = RecordNumber(x - targetCell)

Next

' Clears old info out of Main Sheet
    Sheets("Main Sheet").Select
    Rows("2:70000").Select
    Selection.Delete Shift:=xlUp
    Range("A2").Select

MsgBox ("Operation Complete")
