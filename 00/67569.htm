UEFI Secure Boot and Ubuntu Live CD - Microsoft Community

nharp0386 say:

UEFI Secure Boot and Ubuntu Live CD

Since the UEFI Secure Boot feature that's included with Windows 8 PCs is designed to prevent the user from running operating systems other than Windows 8, is it legal to run an Ubuntu 12.10 Live CD on one of those machines and use the "Try Ubuntu" option?

Thanks!

spiritx say:
Hi,

UEFI does not prevent using other operating systems. UEFI itself is a lightweight
operating system with many capabilities. It is true that a OS can be written so
that UEFI can be used to prevent other OS's from being used. However that
ability could be used by Windows, Apple/Mac, Linux, or others. The main function
of the UEFI and Secure Boot is to prevent malware from loading at boot.

UEFI
http://en.wikipedia.org/wiki/Unified_Extensible_Firmware_Interface

Check with Ubuntu Support and Forums for methods to use Linux with UEFI
Also check with the System Maker (Motherboard Maker for custom systems)
for methods to use alternative OS with or without enabling UEFI if possible.

The problem is the OEM password used when UEFI is enabled is controlled
by the individual hardware manufacturer not Microsoft. Making the password
public would totally negate the ability of UEFI to protect the system since the
malware authors could also use it.

Linux and Windows 8 UEFI secure boot issues
http://www.itproportal.com/2012/12/31/linux-and-windows-8-uefi-secure-boot-issues/

UEFI - check the UEFI Learning Center
http://www.uefi.org/home/

Hope this helps.
