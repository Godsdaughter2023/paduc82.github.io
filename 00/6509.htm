To disable recent used list in Attachment file drop down, you will need to create the Reg_DWORD Key under following registry location.

Hi Shaimaa,

 

Here are the answers.

 

A:

(1) Regarding Exchange in premium which Exchange plan 1 with no In-Place Hold, is the archiving here will keep even the deleted emails.

  The answer is no. 

(2) Can the user call an email from the archiving folders then delete it.

Yes, as the archive mailbox also has Delete Items folder and Recoverable Items folder. It is the same as our primary mailbox.

(3). Is there is a way that the admin can prevent losing the deleted email in Ex Plan1.

No, we suggest you additionally purchase a standalone Exchange Online Archiving plan. So that you can use the In-Place Hold feature.

 

B:

I would like to confirm if this is your situation, your Office 365 tenant has added both Office 365 Business Premium and Office 365 Enterprise licenses, now some users are assigned the SharePoint Online (Plan 1) license from Office 365 Business Premium, and some users are assigned the SharePoint Online (Plan2) license from Enterprise E3.

That means, the users who has been assigned SharePoint Online (Plan1) license only can use the search features that are provided by Plan 1. For the services compared between SharePoint Online Plan1 and Plan 2, see this article: https://technet.microsoft.com/en-us/library/sharepoint-online-service-description.aspx.

 

C:

The in-place hold(Preservation hold library) is available in Plan 2 but is not available in Plan 1. In this situation, if a user is assigned SharePoint Online (Plan 1) license, then he cannot benefit this feature. About how In-place holds works, see: Overview of eDiscovery and in-place holds in SharePoint Server 2013
