Set the Controlsource of your text box on the report to: =[Enter an as of date]

That should do it.

Personally, I wouldn't do it that way. I would use a form to enter the criteria. Create a blank form and put an unbound textbox named txtAsOfDate. Then add a button on the form to run the report. Set the criteria in your query to:

=Forms!formname!txtAsOfDate

Then use the same expression in the ControlSource of the textbox on the report.

Using a form to enter criteria instead of a parameter prompt has many advantages. For example, you can make sure the user enters a valid date before you run the report.

hansv mvp say:

The MultiSelect option is only intended to work within a single folder, just like you can select files in Windows Explorer. When the user selects a different folder, the file selection is reset.

If you want the user to be able to select files from different folders, you'll have to display GetOpenFilename more than once, and keep track of the selected files in an array.

SUM a Row Based on Conditional Number of columns - Microsoft Community

howardl00 say:

SUM a Row Based on Conditional Number of columns

I need to sum a number of cells on a Row always starting at the same column and going forward X number of columns where X can vary and is contained in a specified cell. So for example, if X = 10 and my first cell to sum is E5, then the SUM should deliver E5:E14. If X changes to 8, then the SUM should deliver E5:E12. Any assistance would be appreciated.
ron coderre say: EDITED TO CORRECT MY EXAMPLE FROM ROW TO COLUMN DATA:
With
E5:E16 containing up to 12 values

and
A1: the number of cells to sum....eg 10

This formula sums the number of cells in E5:E16 that are indicated in A1.
=SUM(E5:INDEX(E5:E16,A1))

In the above example, the formula sums E5:E14...(10 cells)

Note: I used 12 contiguous cells because that construct is usually applied to budgets, actuals, forecasts, etc.

If you want more flexibility, this formuls will sum any number of cells, beginning with E5
=SUM(E5:INDEX(E:E,A1+ROW(E5)-1))

Does that help?
