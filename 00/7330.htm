swischner say:
Sounds strange. The only reason I can think of, is that you have set a limited page size. By default OneNote pages are of unlimited size in both dimensions. But with OneNote 2016 you can optional limit the page to a specific size like Letter or A4. If you do that, you are still allowed to write "out of bounds"; the limit is more of an optical reminder. Those out of-bounds-areas are displayed in a rather dark grey and it's almost impossible to see the cursor on them.
To "unlimit" the page size, go to "View" menu, click on "Paper Size" and then to the right set "Size" to "Auto".
If this does the trick, it may still be, that every new page created in that section (only) will show the same size limit. In that case a page template for this section has been set. To remove it, do the following:
Go to "Insert" menu, click on "Page templates" to unfold the menu from where you choose the last entry ("Page templates..."). In the following dialog ther is an area in the lower area titled "Always use a specific template". Set that to "No Default template".
