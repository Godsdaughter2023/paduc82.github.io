Hi geordie,

 

For the average user, it is the only primary real-time anti-malware, anti-virus, and anti-spyware program you'll probably require (though you also need to use Windows Firewall which uses a replacement in that Norton suite).  It's what I use, but it's not all I use.  And a lot of protection is what you do as the user much more than what program you use.

 

For what it's worth (and all of these are totally free), here's what I do (and it isn't 100% but nothing is, and in some cases only catches things after they've already gotten into the system, but it's a good process while remaining reasonable and they can all work together - as proven by the fact that I use them).  Use what you want and ignore what you don't want and do them at a frequency you feel comfortable doing, but regardless of those decisions, do pay attention to the bold section because THAT you should do no matter what else you decide:

 

I use MSE for real-time protection and run scheduled daily quick scans with a scheduled bi-weekly full scan.  I've been using it since the day after it was released as a beta and I'm totally and completely satisfied.  In my opinion it's the best option out there.  In addition, given your post, I'll add that I would not let Norton come within 100 miles of any of my computers or those of any of my clients - but again, that's a personal opinion.

 

I update and periodically (every couple of weeks or so) run full scans with Malwarebytes: http://www.malwarebytes.org/ and SuperAntiSpyware: http://superantispyware.com/ and ESET online scanner: http://www.eset.com/us/online-scanner Get only the free versions. Any that also have paid versions include features like real-time protection that cannot be used along with MSE anyway (in these cases), so there's no point in spending the money anyway.

 

I use Secunia PSI http://secunia.com/vulnerability_scanning/personal/ to make sure all my programs are updated and current (since updates often plug security holes in the software and thus can be important as many people never check and/or update their software unless prompted - and this way I don't need to check everything since notifications and sometimes updates are automatic).

 

I always follow the following advice from Stephen Boots - MVP:

 

-Make sure that the Windows Firewall is enabled. No other firewall truly offers better protection or is needed. This one is perfectly fine.

 

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

 

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

 

-In Internet Explorer 8 or 9, use the SmartScreen Filter.

 

-Make sure that IE Internet Security settings are at least set to medium-high (default).

 

-Enable the pop-up blocker in IE.

 

-On Vista and Windows 7 make sure that User Account Control (UAC) ON and not running with elevated privileges.

 

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

 

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

 

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

 

-Never open unsolicited email attachments.

 

-Practice safe web browsing.

 

I hope this helps.  If you do decide to change to MSE and would like some very detailed instructions on how to do it right and completely the first time (including troubleshooting tips if you run into trouble and configuration recommendations and more), then just ask and I'll post them as soon as I see the request.

Ahhh - that explains the problem.  No need to send any files.

Visio 2010 introduced new Callout shapes and capabilities, which are not compatible with the older Visio versions.  If they are edited in the older version, they will not maintain connectivity.

To maintain expected behavior, you can continue to use the older Callout shapes by opening the older Callouts stencil manually:

1. In the Shapes window select More Shapes

2. From the flyout select the Visio Extras folder

3. Select the Callouts stencil

These are the same legacy Callout shapes that ship with Visio 2007, and should round trip from 2007 to 2010 and back.  These older shapes had their own issues, which is why we created the newer mechanism and shapes for 2010, but the 2007 version doesn't have the ability to use the newer ones.
