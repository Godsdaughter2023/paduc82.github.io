Windows 10 Review: OneDrive, Help Services, Miscellaneous - Microsoft Community

andre da costa say:

Windows 10 Review: OneDrive, Help Services, Miscellaneous

Technical Level : Basic

Summary
OneDrive would probably be described as one of the most important parts of Windows 10, it defines what the platform means to be connected to the cloud, and it is in fact the key to using Windows 10s new features. With Windows 10, Microsoft is deprecating the dedicated modern app and focusing exclusively on the desktop client instead. Another major effort is the unification of the sync engine, instead of separate engines for its consumer and business versions, OneDrive will have just one unified engine for both markets.
Details

Previously:

Windows 10 Review: Introduction, Specifications and Setup

Windows 10 Review: Daily Usage

Windows 10 Review: Microsoft Edge web browser

Windows 10 Review: Multimedia and Productivity

Windows 10 Review: Advanced features, Security and Connectivity

Windows 10 Review: Windows Update, Support, Recovery, Easy Upgrade

OneDrive takes a new approach to syncing files called selective sync

The placeholder feature introduced in Windows 8.1 is being replaced in Windows 10 because it was kind of confusing by providing shortcuts to files that were not available offline, this was of course great for devices with limited storage. The Windows 10 version of OneDrive replaces this with selective sync; with this option you can choose to sync just the folders you often access. I find this option way better since I won’t have to be forced into syncing everything. I mostly need just my Office files, which I have in a special documents folder on OneDrive, so I will just have that synced and leave everything else on OneDrive. Another feature OneDrive offers is remote access; you can enable this through a settings option to access your files remotely from another computer as long as the computer is available. AutoPlay now supports saving Pictures and Videos directly to OneDrive in addition to that, Microsoft has also added music and playlist integration in OneDrive. OneDrive also adds performance improvements for syncing large amounts of files that demand a lot of bandwidth.

Windows 10 includes a new web based help system to help users familiarize themsevles with the new OS.

GETTING STARTED – An improved help system is included in Windows 10, also a modern app. I believe this was a miss in Windows 8, Microsoft expected users to just learn the OS without having a clue about its features or how to navigate it effectively. The Getting Started app which provides mostly web based content is an end to end collection of tutorials for learning about Windows 10 and all its features, what’s different, improved and familiar. It is also updated with the latest information so the content stays current. I notice some help links will require that you have Internet access. For instance, when clicked the link to learn more about Deferred Updates in Windows 10, it loaded Bing with a Definition of the function.

Many of the help links in Windows 10 will require Internet access to view them. The above utilizes Bing for definition.

Miscellaneous:

There are a host of improvements and changes throughout Windows 10 that are quite small, but deserve a mention none the less:

  • Print to PDF is now built in.

  • Built in Windows Phone Companion app with support for iOS, Android and Windows Phone devices.

  • Command Prompt includes a wide selection of new keyboard commands

  • Backup and Restore options first introduced in Windows 7 have been restored to their original locations – System Image.

  • New Date and Time Calendar in the Notification Area.

  • Action Center has now been renamed Security and Maintenance

  • Refreshed icon design set – users should note, the icon set in Windows 8.1 is more than a decade old so it was definitely in need of an update. The Recycle Bin users will notice shares a resemblance to the old style Windows 95 recycle bin.

  • Modern Skype has now been replaced by a Desktop client.

  • Battery Saver setting for limiting background activity when on battery power.

  • XBOX app features a built in screen recording functionality you can use in apps.

  • Improved Pen support in Remote Desktop when used with the upcoming Windows Server 2016 release.

  • App and Start layout syncing have been deprecated

  • Device app install limits have been reduced from 81 to 10.

 

Windows 10 includes a built in Windows Phone Companion app that works with iOS and Android devices too. The app can be used to easily transfer files to and from your phone.

Is Windows 10 faster?

I wouldn't describe it as any faster than Windows 7 or Windows 8.1 and I had both on this same laptop. Claiming faster these days doesn't really seem necessary since hardware seems to have caught up nicely over the past 5 years. What I will say is there are more improvements that make the interface more productive and feel faster at accomplishing task:

- The return of the Start menu, quicker access to familiar functions and launching apps

- Snap assist for managing on screen windows

- Task View for managing multiple desktops

- Quick Access for frequently accessed folders

- Cortana for finding information quickly

- Network Flyout for connecting to wireless networks faster

- Ability to pin frequently accessed System Settings to Start

- Live Tiles for viewing information at a glance without launching an app

- Action Center for viewing app and system notifications

All of these things make working in the Windows interface faster and efficient. Combined with the familiar Windows 7 user experience, many users coming from the six year old OS should hit the ground running with minimal learning curve.
