a. user say:

If you format only the C partition, no other partition is affected. If you format the entire hard drive then every partition on the drive is affected.

There's really no need to format your system partition as a separate step if you want to go back to Windows 8. If you're doing a clean install or a factory restore, the system partition is formatted at the beginning of the process.

I'd like to suggest you consider keeping your data files in a separate partition, like the one you setup. There's a great advantage to doing this, as you have found: If you need to restore your software, your data isn't touched.

All that said, backing up is always a good idea. You should be doing it every day.
