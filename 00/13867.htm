drtimr say:

Moving Time Machine backup of msoffice 2011 to clean install of new os?

How do I move Time Machine backup of legitimate copy of msoffice 2011 to clean install of new os?

I struggled with this as many others have from various forums and found it very frustration however I now have the solution.

1) After cleaning up your Mac and reformatting it. Install OS you want. I installed the new "EL Capitain".

2) Restore your Time Machine Backup but exclude the Applications including MS Office.

3) Go into a Time Machine backup that occurred BEFORE you cleaned up your Mac and go into Applications. Drag the MSOffice folder onto your Mac desktop.

4) Drop this folder onto your new OS Applications folder.

5) Open up a MS Office Document and the relevant MS Office Program reloads. (If you try and open MSOffice App directly it will tell you it is a TimeMachine backup and won't open. You need to open a document.)

(The reason is that the files with your microsoft licence information were restored from Time Machine in your Licence and other folders folder.)

NB: To keep legitimate make sure you only reinstall this copy onto the same computer you upgraded.I have no idea if this works on a new computer but that was not whatI was interested in. I am a strong believer in using legitimate purchased software so the developers can keep up their good work.

If you are trying to extract all characters to the right of "-" then try below formula:

If you want to extract all characters to the left of "-" then try below:
