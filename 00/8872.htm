    Formula to find intersection based on row and column value - Microsoft Community

    todtown say:

    Formula to find intersection based on row and column value

    I used to have a great formula for this but....I USED to have a lot of things. :0)

     

    Here's the scenario. Column A contains a list of unique IDs. Row 1 contains a list of unique headings. I want a formula that will return the cell where the row of a given ID intersects the column of a given heading.

     

    [_]  Floor   Wall    Door 

    123

    456                       

    789

     

    If the ID is 456 and the heading is Door, I want the formula to find Cell C2.

     

    tod

    hansv mvp say:

    Since 456 is in A3 and Door is in D1, I would have expected that you wanted the value of cell D3.

     

    Let's say you enter the ID (e.g. 456) in cell K1 and the heading (e.g. Door) in K2.

     

    =INDEX($B$2:D$4, MATCH(K2, $A$2:$A$4, 0), MATCH(K2, $B$1:$D$1, 0))

     

    You can expand the ranges of course. If you want to suppress the #N/A error value that would be returned if K1 or K2 is empty or contains an invalid entry:

     

    =IFERROR(INDEX($B$2:D$4, MATCH(K2, $A$2:$A$4, 0), MATCH(K2, $B$1:$D$1, 0)),"")
