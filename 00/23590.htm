vornel say:

For Exchange i had this problem and a few others - this is how I fixed it:

Locate your account setup file here: ~/Library/Group Containers/xxxx.Office/Outlook/Outlook 15 Profiles/Main Profile/Data/Exchange Accounts/xx/

(the file will not have any extension)

I am an experienced windows user that has switched to macs. All 3 of my macs have this problem, but i noticed that if i change the server, it works really well until auto discover changes the server reference - even though I am sometimes prompted to accept the change I decline, but the server is still modified.

My question was how do I locate my account set up file? For ex-windows users, who may not be familiar with macs, this is what you do:

  1. Open outlook
  2. Change the server
  3. Quit outlook (as the auto discover can work at any time and you may lock the file in the wrong state)
  4. Click on Spotlight Search the hour glass in the top right hand corner
  5. Type in ~/Library/Group Containers/
  6. Go through the directories until you come to the last folder indicated above.
  7. The file will not have an extension but will end in "ExchangeAccount"
  8. Right click on the file 
  9. Select "get Info"
  10. check the "Locked" box
  11. Open outlook

If you are prompted for a password, even though you know it is there and correct, maybe insert the server without the "https://" proceeding it, then repeat the steps above.

I did look at the file (copied it to my desktop and changed its extension to "txt" and notices that it has a call to auto discover - i suppose that this "fix" does not allow it to happen.


Actually seems to be working and as changing the server always worked (but was tedious) i have high hopes for this fix. This sometimes used to happen with my iPhone and iPad when auto discover would change the server name. How to back up and restore the registry in Windows

b.       2.Locate and then click the following registry subkey: HKEY_CURRENT_USERS\SOFTWARE\Microsoft\Internet Explorer\Zoom

c.       3.In the details pane, double-click ZoomFactor.

d.      Click Decimal.
