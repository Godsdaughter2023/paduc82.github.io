To set the location as private -

Firstly, you need to set UAC to its default level in Control panel, User accounts, Manage User account control settings.

You can then change it to a private network by setting it to allow the computer to be discoverable in Settings, Network, Status, Change connection properties, Make this PC discoverable.

You can then put UAC back to its highest level if you wish.  [and the setting you just changed will no longer be visible]
