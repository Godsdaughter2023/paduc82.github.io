Hi Naseefo,

Which operating systems was installed on the computer first?

 

Every operating system has an EFI file to boot in EFI mode. If you do not see the list of installed operating systems while booting the system, the reason for this issue might be that the Ubuntu UEFI EFI file is behind Windows in the priority list. Let’s see if we can invoke the Ubuntu Grub, which would allow us to make the operating system selection on boot up.

 

  • Boot into Windows 10

  • Open the Command Prompt (in Administrator Mode).

  • Go to Start -> Command Prompt -> Right Click and "Run as Administrator".

  • Type in the following command and press enter:

  • bcdedit /set {bootmgr} path \EFI\ubuntu\grubx64.efi

  • Now Restart your system, and the GRUB menu should be loaded. Select the OS you want to boot into.

     

    Hope it helps.
