Support for content controls is very limited in Word 2011. Checkbox content controls created in (say) Windows versions of Word are displayed with their current value but cannot be checked/unchecked. The simplest way to implement something similar is to use a "legacy" checkbox control.

Are custom dictionaries specific to a template? - Microsoft Community

inagalaxyfarfarawry say:

Are custom dictionaries specific to a template?

In Word 2016, is it possible to add a custom dictionary to a specific template? What I mean is I would like this custom dictionary to be accessed by documents using a specific template, not the Normal.dotx template. Or are custom dictionaries added and removed only to the whole application?
graham mayor say:

The short answer is no, but you might be able to achieve your aims with a bit of lateral thinking. For example let's assume that you want to have the customdictionary - "C:\Path\Template.dic" as the default dictionary for a particular template. Save the following macros in the template in question then use an autonew macro to call SetDictionary and an autoclose macro to call ResetDictionary as below.

I have left in message boxes to show that the default dictionary is being changed.

Option Explicit

Const strDic As String = "C:\Path\Template.dic" 'The name and path of the dictionary

Sub AutoNew()
    SetDictionary
lbl_Exit:
    Exit Sub
End Sub

Sub AutoClose()
    If Not ActiveDocument = ThisDocument Then
        ResetDictionary
        UpdateTemplate
    End If
lbl_Exit:
    Exit Sub
End Sub

Sub SetDictionary()
'Graham Mayor - http://www.gmayor.com - 23/09/2016
Dim oOriginalDict As Dictionary
Dim oCustDict As Dictionary
    Set oOriginalDict = Application.CustomDictionaries.ActiveCustomDictionary
    ThisDocument.Variables("varDict").Value = oOriginalDict
    Set oCustDict = Application.CustomDictionaries.Add(strDic)
    Application.CustomDictionaries.ActiveCustomDictionary = oCustDict
    MsgBox Application.CustomDictionaries.ActiveCustomDictionary
lbl_Exit:
    Exit Sub
End Sub

Sub ResetDictionary()
'Graham Mayor - http://www.gmayor.com - 23/09/2016
Dim oCustDict As Dictionary
Dim oOriginalDict As Dictionary
    Set oCustDict = Application.CustomDictionaries.Add(strDic)
    Set oOriginalDict = Application.CustomDictionaries.Add(ThisDocument.Variables("varDict").Value)
    Application.CustomDictionaries.ActiveCustomDictionary = oOriginalDict
    oCustDict.Delete
    MsgBox Application.CustomDictionaries.ActiveCustomDictionary
lbl_Exit:
    Exit Sub
End Sub

Sub UpdateTemplate()
'Graham Mayor - http://www.gmayor.com - 23/09/2016
Dim bBackup As Boolean
    bBackup = Options.CreateBackup
    Options.CreateBackup = False
    ThisDocument.Save
    Options.CreateBackup = bBackup
lbl_Exit:
    Exit Sub
End Sub
