Every page has the same header. This has been set by repeating one or more rows of a table on ever printed page. In Excel go Page Setup > Print Titles > Rows to repeat at top. Then save as PDF.

It just appears that the content only is scrolling. If you view the pages side by side or in scrolling mode you can see that the title is on every page.

The file has been set up to suppress scrolling and instead show the same viewport when paging through the document. 

In Acrobat Reader, this setting can be found in View > Page Display > Single Page View

Change the view back to scrolling and that illusion disappears.

It is not an Excel setting, but an Adobe Reader setting. I don't know if this setting can be manipulated from Excel, but I doubt it.

Okay, I'm pretty sure the issue here is the limitation with Exchange and Exchange ActiveSync. They do not make it easy to move calendar events between accounts, possibly to avoid accidental data leaking as Exchange is primarily used in corporate settings. The only sure way to get this event into your calendar is to export it as an ICS file, then import that ICS to your Outlook.com calendar using the web interface.
