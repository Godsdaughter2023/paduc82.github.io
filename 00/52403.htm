Why does File Explorer always open on start up from sleep mode?

Generally, when I expect to be away from my PC for a period of time I place it in "sleep mode". I find this is great in that you don't need to wait for the startup process to go through its steps and you can quickly get back to work.

I have noticed though that when I "wake up my PC from sleep that File Explorer always opens immediately. Has anyone else had this experience, if so did you find a way of stopping file explorer from opening?

Update: Problem solved by myself: I was able to solve this problem by doing a sequence of things.

1. Open All settings in the Action Center.

2. Click on Personalization.

3. Click on Start (Near bottom of start is choose which folders appear on start (in blue text color); Click on the blue text).

Anew page with title CHOOSE WHICH FOLDERS APPEAR ON START then appears on the screen.

In my case the top option of File Explorer was set to the ON position. As soon as I sawthat the switch was ON I felt I knew that the setting was the problem solver. I immediately changed the setting to the OFF position. I then closed the page.

Next I tested to see if this solved the problem:

-a. I put the PC to sleep and waited a couple of minutes.

b. I then woke up the PC and SURPRISE  the desk top display appeared which is what I wanted. Obviously, the test confirmed the setting had been the problem. I now have things the way I want them. I still can not remember ever being on that page before since I upgraded to Windows 10 - now doesn't matter.
