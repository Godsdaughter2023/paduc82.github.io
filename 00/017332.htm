You can optionally specify a different delimiter like this

=ConCat(A1:A6,",")

Function ConCat(rng As Range, Optional delimiter As String) As String
Dim c As Range
If delimiter = "" Then delimiter = "_"
For Each c In rng
If c.Value <> "" Then
ConCat = ConCat & c.Value & delimiter
End If
Next
ConCat = Left(ConCat, Len(ConCat) - 1)
End Function

Change the extension of the file to .txt.

When you open the .txt file in Excel, the Import Text Wizard will start.

Specify Delimited, then click Next.

Specify Comma as delimiter, then click Next.

Click on the date column, and specify MDY as format.

Click Finish.
