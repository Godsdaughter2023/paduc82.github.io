kari p say:

Formula for auditing tool

Hello,

I am creating an auditing tool with 5 categories to be scored. I am trying to create a formula that will let me add the sum for the row of the audit, however divide by the "possible score" and multiply by 10. I want to add a condition that if 1 cell is left blank that it will not be included in the sum so the score will not be lowered.

Is this possible to do? I tried a SUMIF =(SUMIF(B5:F5,"<>"&"")/SUM($B$3:$F$3))*10 formula but it was still dividing like there was an option to score in the category.

Example: You can score a 10 all together, however if 1 category was not audited, Cell E3(or the one that was not used) will be blank, therefore the possible score will not be picked up in our range.

Thanks!

bernie deitrick say:

I think this is what you actually want:

=SUM(B5:F5)/SUMIF(B5:F5,"<>"&"",$B$3:$F$3)*10

For scoring purposes, it is handy to use a weighting function:

=SUMPRODUCT(B5:F5,$B$3:$F$3)/SUMIF(B5:F5,"<>"&"",$B$3:$F$3)*10

Where B3:F3 has relative weights for the factors (a weight of 2 is twice the weighting as a 1, but half that of a 4, for example), and B5:F5 have scores that are, say, 1 to 10 or blank.

importing data from multiple workbooks - Microsoft Community

seviercountychoppers say:

importing data from multiple workbooks

I want to create a workbook with a list of all excel files in a folder and data from a specific cell (E7 - Balance Due) in each of those files.  I've managed to get the list of files using a query but can't figure out how to get the data from cell E7.  Any help is appreciated.
bernie deitrick say:

Put this into a blank workbook, and create a button that runs the macro. Save the workbook as a macro-enabled .xlsm file, then copy it into any folder of files. When run, it will create a link to cell E7 of every sheet named "Statement" - change that to the actual sheet name that you have in those files - for all the workbooks in that current folder.

Sub GetDataFromAllFilesInFolder()
    Dim strFName As String
    Dim strPath As String
    Dim strWFile As String
    Dim wkbkWF As Workbook
    Dim strFormula As String
    Dim strSheetName As String
    Dim i As Integer

    Application.DisplayAlerts = False

    strFName = ThisWorkbook.Name
    strSheetName = "Statement"  'This sheet needs to exist in all files
    
    With ThisWorkbook.Worksheets(1)
       .Range("A1").Value = "File name"
       .Range("B1").Value = "Balance due"
    End With

    i = 2
    
    Application.EnableEvents = False
    Application.Calculation = xlCalculationManual
    
    strPath = ThisWorkbook.Path & "\"
    strWFile = Dir(strPath & "*.xls*")
    
    Do While strWFile <> ""
        If strWFile <> strFName Then
            'Generate strFormula through string manipulation
            strFormula = "='" & strPath & "[" & _
            strWFile _
            & "]" & strSheetName & " '!"
            'Create the list
            With ThisWorkbook.Worksheets(1)
                .Range("A" & i).Value = strWFile  'Put the file name into a list
                .Range("B" & i).Formula = strFormula & "E7"  'Get the balance due
             End With
            i = i + 1
        End If
        strWFile = Dir()
    Loop

    Application.EnableEvents = True
    Application.Calculation = xlCalculationAutomatic

End Sub
