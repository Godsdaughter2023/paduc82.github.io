stefan blom say:

The Paragraph dialog box displays the so-called Outline Level for the current paragraph:

Outline Level in that dialog box is the same as the TOC level. For example, if the Outline Level is "Level 1" in the dialog box, Word will include the paragraph in a table of contents and format the corresponding entry with the TOC 1 style. (If the Outline Level is "Body Text," Word won't add the paragraph to the table of contents at all.)

I'm wondering if you are thinking of the Table of Contents dialog box (References | Table of Contents | Custom Table of Contents) which can be used to customize the appearance of the TOC. By default, this dialog box includes Heading 1, Heading 2 and Heading 3 which means that text formatted in these styles will be added to the table of contents. The heading styles are added by default (and their Outline Levels are hardcoded so that you can't modify them). In the dialog box, you can specify that any paragraph style should be added to the TOC, even if the Outline Level for that style is set to "Body Text."

It is not the TOC entries that show the outline level but the headings themselves. A Heading 1 paragraph should show Level 1 as the outline level, and so on. mike h.. say:

Hi,

A very similar question arose a few days ago and I proposed the solution below, What it does is when a worksheet is de-activated an exact copy is made to another worksheet. In your case this would require the code in 2 sheets because you want the changes to reflect both ways.

In the code below the user also wanted formatting to be copied across, if you don't want formatting it can be done in real time using the change event as opposed to when the sheet is deactivated

code in sheet 1

Private Sub Worksheet_Deactivate()
Dim c As Range
Application.ScreenUpdating = False
    For Each c In Sheets("Sheet1").UsedRange
    c.Copy
    Sheets("Sheet2").Range(c.Address).PasteSpecial , Paste:=xlPasteAll
    Next
Application.ScreenUpdating = True
End Sub

code in sheet 2

Private Sub Worksheet_Deactivate()
Dim c As Range
Application.ScreenUpdating = False
    For Each c In Sheets("Sheet2").UsedRange
    c.Copy
    Sheets("Sheet1").Range(c.Address).PasteSpecial , Paste:=xlPasteAll
    Next
Application.ScreenUpdating = True
End Sub
