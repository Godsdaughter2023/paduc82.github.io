How can I limit rows and scroll faster in Worksheet - Microsoft Community

bwduncan say:

How can I limit rows and scroll faster in Worksheet

Original Title - sheet too long

I have a worksheet that I took out of a database my information only goes to 5000 rows but if I scroll it goes all the way to over 1,000,000 rows.  Is there a way to limit how many rows there is so that I can scroll through the 5000 rows faster?

jeeped say:

It would appear you've run into the infamous LastCell property of Excel. To verify the cell that Excel believes is the last relevant cell, you can use the GoTo function. Press F5, then ALT+S, ALT+S <enter>. This will take you to the cell that Excel believes to the the outer extent of information. The cell may not contain any information but the row and/or column it is on either has information or had information at some point.

To reset the LastCell to your current range of information, first you need to ensure that no formulas or spaces populate any cells beyond the range you wish to work with.

  1. Go to column A in the first blank row below the last row you wish to keep.
  2. Press CTRL+SHIFT+<down arrow>. You should be at the absolute bottom of the worksheet (row 1048576). If not, keep the CTRL+SHIFT keys down and press the down arrow until you get there.
  3. Press ALT+<spacebar>. This selects the entire rows. Right click the highlighted rows and select Delete.
  4. Go to row 1 in the first blank column to the right of the last column you wish to keep.
  5. Press CTRL+SHIFT+<right arrow>. You should be at the absolute right of the worksheet (column XFD). If not, keep the CTRL+SHIFT keys down and press the right arrow until you get there.
  6. Press CTRL+<spacebar>. This selects the entire columns. Right click the highlighted columns and select Delete.Now you can create and run a snippet of macro code to reset that pesky LastCell property.

    1. On the Developer tab of the ribbon, click Macros and type in Reset_LastCell as the macro name. Click Create button.
    2. In the macro editor, edit the code to read:
        Sub Reset_LastCell()
            ActiveSheet.UsedRange
        End Sub
    3. Press ALT+Q to exit the macro editor and return to your worksheet.
    4. Click Macros again from the Developer tab. This time highlight Reset_LastCell and select the Run button.

    You're done. Verify the reset of LastCell to a new location by F5, then ALT+S, ALT+S <enter>.

    For the record, this can also be done by going to your own desired LastCell, then CTRL+SHIFT+HOME, CTRL+C, <go to a new worksheet's A1>, CTRL+V, then deleting the original worksheet and renaming the new sheet if necessary.

    Due diligence: There is the potential here to delete a lot of possibly relevant data and/or formulas. Make sure you have a backup copy!

    Select Done and refresh the browser.

    Mozilla Firefox

    If you’re using Windows, in the Firefox window, select Open menu Firefox browser options menu > Options.

    Tip: If you’re using a Mac, go to Firefox > Preferences.

    Select the Privacy tab.

    In the History section under Firefox will, select Use custom settings for history.

    Make sure Accept cookies from sites is checked and Accept third party cookies is set to Always, and then select OK.

    Safari

    On your Mac, go to Safari > Preferences > Privacy.

    Under Cookies and website data, select Always allow.

    Select Close and refresh the browser.
