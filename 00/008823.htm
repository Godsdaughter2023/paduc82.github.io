Question : How can I paste data from a column to a filtered column with no code and no pasting on non selected lines ?

tasos_k say:

Hi,

try this vba macro

(make a copy before)

Sub PasteInVisibleCells()
'sheet1 not filtered/sheet2 filtered
Const colSh1 = "A" '<<< data on sheet1 in col A, change
Const colSh2 = "A" '<<< data on sheet2 in col A, change
Dim ws1 As Worksheet, ws2 As Worksheet
Dim r As Long, t As Long

Set ws1 = Sheets("Sheet1")   '<<< change name
Set ws2 = Sheets("Sheet2")   '<<< change name
r = ws2.Cells(Rows.Count, colSh2).End(xlUp).Row
t = 2
For Each rr In ws2.Range(colSh2 & "2:" & colSh2 & r).SpecialCells(xlCellTypeVisible)
rr.Value = ws1.Cells(t, colSh1)
t = t + 1
Next
End Sub

how??

step2

1) press ALT+F11 to open Visual Basic

2) Insert > module and paste the code above on the right

3) Press ALT+Q to Close Visual Basic

step3

To run the macro, press ALT+F8, 

select <PasteInVisibleCells> from the list and click the run button.

VISUAL BASIC EVENT WHEN EXIT A CELL - Microsoft Community

huggy2 say:

VISUAL BASIC EVENT WHEN EXIT A CELL

I would like to display a message if the user exits a specific cell without entering data. I can use VB to display a message but don't know how to display the message when the user exits the cell without entering any data.

I have the following code, but only want it to run when the user exits the cell.

Sub Initial_CTMS_Date()
    If Cells(10, 3) = "" Then
    MsgBox "Please enter Intial CTMS Date", vbOKOnly
    End If
       
End Sub

I have excel 2007.

per jessen say:

You need an Selection_Change event and a variable to record the previous selected cell.

Insert this in the code sheet for the desired sheet. Notice that the variable declaration has to be at the very top of the module and outside the event code, just as below:

Dim LastCell As Range

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
If Not LastCell Is Nothing Then
    If LastCell.Address = "$J$3" And LastCell.Value = "" Then
         msg=MsgBox ("Please enter Intial CTMS Date", vbOKOnly+vbExclamation, "Missing data")
    End If
End If
Set LastCell = Target
End Sub
