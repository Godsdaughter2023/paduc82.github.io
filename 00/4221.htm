bob jones aka: cybertaz say:

You have at least 2 options:

Option 1 - In Word or comparable use Find & Replace to replace the commas with carriage returns [¶] then save the .txt file. When you import each file the content will come in as rows in a single column rather than a single row spanning multiple columns.

rodney zhang msft say:

Hi Robyn,

In your case, you can refer to the following steps to see if it works:

1.  Right click on any of the ribbon tabs and select Customize Quick Access Toolbar.

2.  In the Excel Options dialogue box, select All Command in the ‘Choose Command from’ drop down.

3.  Scroll down to select Send to Mail Recipient option.

4. Click on the Add button. This will add the Send to Mail Recipient option on the right pane.

5. Click OK.

6. This will add the Send to Mail Recipient icon in the Quick Access toolbar.

7.  Right-click on the sheet that you want to send and select Move or Copy.

8. Select (new book) under the To book drop down. Also check the Create a copy option at the bottom left.

9. Click OK. This will create a new workbook that will have the copied worksheet only.

10.  With the newly created workbook activated, click on Send to Mail Recipient icon from the Quick Access Toolbar.

11.   Select the option “Send the entire workbook as an attachment”.

12.   Click OK. This will open an outbound email with the workbook attached. Since this is the new workbook that we created, and only has one sheet (that we moved), you are sending a single worksheet only.

13.   Input the recipient email address and click on Send.

If the above don’t work, could you share your Excel client version with us? You can find the version number via Excel>File>Account.
