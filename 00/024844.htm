hansv mvp say:

Hold down the Shift key and right-click a .doc file.

Select Open With > Choose Default Program from the context menu.

If Microsoft Word is listed, select it.

Otherwise, click Browse..., navigate to C:\Program Files\Microsoft Office\Office10 and select WINWORD.EXE, then click Open.

Tick the check box "Always use the selected program to open this kind of file", then click OK.

 

Do the same for a .docx and a .docm file.

 

Alternatively, perform a repair installation of Office XP from the Add/Remove Programs control panel. This should restore the file associations for all Office documents.

suzanne s. barnhill say:

The easiest solution is to reregister Word, that is, reestablish the file association between Word and .doc files in the Windows Registry. To do this, press Windows Key + R (or click Start | Run) to open the Run box. Type "winword /r" (without the quotation marks, and note the space before /r) and press Enter. It may not appear that anything is happening, but Word will be reregistered.

 

Alternatively, in Windows Explorer (My Computer) press Shift and right-click any Word document and choose "Open With." Do not select Word even if it is offered, as this does not give you the opportunity to make Word the default. Instead, select "Choose program." Then select WINWORD.EXE from the list and make sure the box for "Always use the selected program to open this kind of file" is checked. For more detailed instructions, see http://support.microsoft.com/kb/307859.
