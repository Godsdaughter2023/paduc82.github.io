jay freedman say:
The code is just

      Selection.Delete

As an aside, your macro doesn't actually have to "select" the bookmarked text in order to delete it. If the name of the bookmark is "MyBookmark", for example, the single statement

    ActiveDocument.Bookmarks("MyBookmark").Range.Delete

will remove the marked text. The advantage is that the cursor on the screen never moves, so the user doesn't lose the previous position. Also, if the bookmarked text was off-screen at the time, selecting it would force the screen to scroll, while just deleting the bookmarked text doesn't scroll.
