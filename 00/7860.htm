stefan blom say:

I'm assuming here that you have set up your heading numbering properly so that each level is assigned to a heading style. See http://www.shaunakelly.com/word/numbering/numbering20072010.html.

Once heading numbering is up and running, you will have to create a "non-heading" style which is based on the corresponding heading. One style is required for each heading level. That is the only way to re-use the number formatting.

For example, you would need a style called (for example) Body Text Level 2 paragraph style that is based on Heading 2, and you would need a Body Text Level 3 paragraph that is based on Heading 3.
