How to calculate same store sales data using Excel - Microsoft Community

randy_barn say:

How to calculate same store sales data using Excel

I am trying to calculate same store sales using Excel.  Year 1 there were 300 stores.  Year 2 there were 350 stores. Some new stores were added and some stores were deleted.  Net increase in stores was 50 stores.  Tyring to determine same store sales between the stores that existed in Year 1 and Year 2.

 

 How can Excel assist me in the calculation of same store sales?

jeeped say:

... Tyring to determine same store sales between the stores that existed in Year 1 and Year 2.


I am envisioning your problem as using the same column (e.g. Sales Date) for both years. I believe you will have to stack a couple of formulas to determine which stores existed in both years, and sum the total sales from that criteria.

          

The formula in K3 is,
=IF(AND(SUMPRODUCT(($A$2:$A$14=$E3)*(YEAR($B$2:$B$14)=F$1)),SUMPRODUCT(($A$2:$A$14=$E3)*(YEAR($B$2:$B$14)=H$1))),SUMPRODUCT(($A$2:$A$14=$E3)*($C$2:$C$14)),0)
... which can be filled down as necessary. The formula in J3 is,
=--AND(SUMPRODUCT(($A$2:$A$14=$E3)*(YEAR($B$2:$B$14)=F$1)),SUMPRODUCT(($A$2:$A$14=$E3)*(YEAR($B$2:$B$14)=H$1)))
... which results in 1's and 0's so the cell uses a custom number format of,
[Blue]"Yes"_);[Blue]"Yes"_);[Red]"No"_);_(@_)
... which can be filled down as necessary.
I applied some conditional formatting to show the stores that were active in both years.

I've uploaded that sample worksheet to my SkyDrive here. If you can use the methods described here for your own application, download a copy to use for example.

tom-s say:

Randy_Barn,

Here's a possible way to go with this but might need some adjustments.

 

Assuming you have column headers in A1:D1 - Y1 Stores, Y1 Sales, Y2 Stores, Y2 Sales

Y1 data (Stores and Sales) is in A2:B301

Y2 data (Stores and Sales) is in C2:D351

 

Assume next that it's possible some of the Y1 stores closed and so don't appear at all in the Y2 data. You report in column E on the stores from the Y1 list, where if the store remained open you just see the store name, but if the store closed there's a brief note. So make E1 header say Y1/2 Stores, and put this formula in E2 and fill down to E301:

=IF(ISNA(MATCH(A2,C$2:C$351,0)),"Y1 store " & A2 & " closed",A2)

 

Assume next that the sales report you want, in column F, is the difference in sales between Y2 and Y1 (for those Y1 stores that didn't close). So make F1 header say Y2-Y1 Sales, and put this formula in F2 and fill down to F301:

=IF(RIGHT(F2,6)="closed",F2,VLOOKUP(F2,C$2:D$351,2,FALSE)-VLOOKUP(F2,A$2:B$301,2,FALSE))

This applies to all the Menus including the Windows Explorer
