select whole rows where a specific word exists in a column - Microsoft Community

keith r13 say:

select whole rows where a specific word exists in a column

I have a 20 column by 200 row spreadsheet and I would like to create something (macro?) that will show whole rows where a specific word exists (amongst many) in any of the cells in column 20; how can I do this?
mike h.. say:

Hi,

 

You can do it with this code. Change the worksheet and the string your looking for to the correct ones.

 

Sub Hide_Rows()
Dim MyRange As Range, ws As Worksheet
Dim CopyRange As Range, Mytext As String
Dim MyString As String
Set ws = Sheets("Sheet1") ' change to suit
MyString = "xxx" 'change to the word you're looking for
Lastrow = ws.Cells(Cells.Rows.Count, "T").End(xlUp).Row
Set MyRange = ws.Range("T1:T" & Lastrow)
For Each c In MyRange
    If Not InStr(c.Value, MyString) > 0 Then
        If CopyRange Is Nothing Then
                Set CopyRange = c.EntireRow
        Else
                Set CopyRange = Union(CopyRange, c.EntireRow)
        End If
    End If
Next
If Not CopyRange Is Nothing Then
CopyRange.EntireRow.Hidden = True
End If
End Sub
