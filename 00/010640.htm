Accoring to the error message "Your mailbox has been temporarily moved to Microsoft Exchange server", I suggest we first try following general steps:

1. To prevent data loss in following steps, please backup all email items, calendar events and contacts in all your accounts to the .pst file. For details, please refer to this article: Export or backup email, contacts, and calendar to an Outlook .pst file.

2. Exit Outlook and open Control Panel, click on Mail > Show Profiles. Note: if you cannot see the Mail icon, change your view to Large icons.

3. Remove all Outlook profiles except the primary profile (the primary one of the two account). If you see a pop-up window show a warning message, please double-confirm to make sure you have backup the datas of this account, then you can click on Yes to continue.

2. When you finished, remove the primary one (the one you left in above step 3).

3. In the Mail dialog box, make sure to set "Always use this profile" to use the new one, then re-start Outlook to see how it will goes.
