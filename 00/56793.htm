In order to save costs, the interior packaging is the "standard" packaging for every console. With special bundles like your FIFA bundle, only a cheap "wrapper" is put around the standard interior box, as it's cheaper to print a bundle specific wrapper, than to print a complete box for each and every bundle. 

If you are sure you can't find the code on any of the included cards (maybe the FIFA code is printed together on a card), it MIGHT be, that a previous customer bought the bundle, redeemed the FIFA code, and then returned the console to Media Markt for whatever reason.

greg carmack - mvp 2010-2020 say:
Hi Frank. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.


Recent Windows Updates are causing device problems in Windows 10, including USB flash and external drives. You should be able to access and back up your files on another PC. Then here are some things to try:

Find the devices in Device Manager reached by right clicking the Start Menu. From Driver tab choose Roll back. If that fails or isn't available choose Uninstall, restart PC to reinstall driver.

If this doesn't help the driver may be the USB controller for that USB port. Try the above for all USB Host and Hub controllers in Device Manager USB category.

If your keyboard and mouse are USB, first program the power button to effect the needed restart after Rolling back or uninstalling the driver. Go to Settings > System > Power & Sleep > Additional Power Settings > Choose what Power buttons do to choose Shutdown.

Run the Hardware Troubleshooter at Settings > Update & Security > Troubleshoot.

Update the USB 3 or Chipset drivers from the PC or Device's Support Downloads web page. While there check for newer BIOS, Display, Sound, Bluetooth, Network and all other drivers to make sure the ones presently installed in Device Manager are latest.

You can check which Updates were installed just before this happened, then uninstall them at Settings > Update & Security > Windows Update > Installed Update History - look first for installed drivers. Then you can hide these Updates using the Hide Updates tool: https://www.howtogeek.com/224471/how-to-prevent...

You can also try to System Restore to a point before this happened, then check for Updates using the Hide Updates Tool to block the Updates offered: http://home.bt.com/tech-gadgets/computing/how-t...

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
