Method 1: Update Network adapters.

3. Double-tap or double-click the device name.

4. Tap or click the “Driver” tab, tap or click “Update Driver”, and then follow the instructions.  You might be asked for an admin password or to confirm your choice.

 

Method 2: Rename Software Distribution and Catroot2 folders.

Step1.

Stop these services:

Press Windows key + R to open the run window.

Type “services.msc” and hit enter to open the services window.

We need to stop these three services.

·         Right click on “BITS” service and select stop.

·         Right click on “Cryptographic” service and select stop.

·         Right click on “Windows Update” service and select stop.

 

Step2.

Once this is done we will rename these two folders.

Press Windows key + E to open Windows Explorer.

Navigate to C/Windows/Software Distribution.

Right click on this folder, select rename to “SD.old”.

Navigate C/Windows/System 32/Catroot2.

Right click on this folder, select rename to “catroot2.old”.

 

Step3.

Press Windows key + R to open the run window.

Type “services.msc” and hit enter to open the services window.

We need to start these three services.

·         Right click on “BITS” service and select start.

·         Right click on “Cryptographic” service and select start.

·         Right click on “Windows Update” service and select start.

jane lla say:

Hello Andres,

The reason why you're getting this error message whenever you try to save said file, is due to English being set as the default language.

If the offending characters are from a language other than English but one Microsoft supports, you can install the relevant MUI (Multilingual User Interface) language pack from Windows Update, or the relevant LIP (Language Interface Pack) from here as a possible fix.

In addition to that, the system locale determines the default character set (letters, symbols, and numbers) and font used to enter and display information in programs that don't use Unicode. This allows non-Unicode programs to run on your computer using the specified language. You might need to change the default system locale when you install additional display languages on your computer. Selecting a different language for the system locale doesn't affect the language in menus and dialog boxes for Windows or other programs that do use Unicode.

Follow these steps to change the system locale:

1. Go to the Start menu
2. Type Control Panel in search bar and click on it
3. Select Region and Language
4. Click Administrative tab
5. Click Change system locale tab (If you're prompted for an administrator password or confirmation, type the password or provide confirmation)
6. From drop down menu, check and make sure the language is selected as you require and click OK.
7. Restart computer and check.

Update us with the outcome.
