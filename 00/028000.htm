Hi,

 

I'm afraid it all gets a bit marginal over whether a robust formula can be developed because:-

 

We now have commas and you seem to want to keep them.

Some last names are 'double-barelled' and other not
Some have 2 forenames and others may not have.

 

Because of this here's 2 formula. The first one works for both your posted examples.

 

=LEFT(A2,FIND(",",A2))&" "&TRIM(MID(SUBSTITUTE(" " & SUBSTITUTE(A2,",","")&REPT(" ",6)," ",REPT(" ",255)),3*255,255))

=TRIM(MID(SUBSTITUTE(" " & A2&REPT(" ",6)," ",REPT(" ",255)),1*255,255))

 

 

Note the 1 towards the end. This makes it pull out the first word, change this to a 2 for the second etc.

pamcaswell say:

Yes, you can.

The first field code of each two-field caption (press  Shift F9 to reveal them) likely reads

{ STYLEREF 3 \s }

The number 3 is the heading level to include. You can change that number to 1 so the code looks like this:
{ STYLEREF 1 \s }

Then press F9 to hide the field code and update the value. 

You can use different codes in different sections of a document.  Just remember that if you again enter the caption dialog to create a figure caption, all captions will be changed to the level last requested. You can, instead, save as AutoText the label, number, and punctuation for each chapter-numbering scheme you want and insert it where needed.

Also remember to note somewhere how you did this so others don't ruin your work or so they will know how to duplicate it.

marites_p say:

Hi

Please be advised that Skype Community forum is an option to post your concerns and questions, or look for the same thread that is similar to your concern which has been resolved or answered.

Going back to your concern, there is no option to recover/retrieve your deleted Skype conversation history.

When you delete a conversation, all the messages in the conversation are deleted, and it's removed from your history.

The conversation will be deleted from Skype on that specific device, but will still be visible on other devices unless you delete it from there too.
