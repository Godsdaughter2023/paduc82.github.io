bernie deitrick say:

This will do what you want, but it may be better to fill in A and B with values.

Sub TestMacro()
    Dim lngR As Long
    Dim v As Variant
    Dim strC As String

    strC = "C"

    For lngR = Cells(Rows.Count, strC).End(xlUp).Row To 2 Step -1
        v = Split(Replace(Cells(lngR, strC).Value, " ", ""), ",")
        If UBound(v) > 0 Then
            Cells(lngR + 1, strC).Resize(UBound(v)).EntireRow.Insert
            Cells(lngR, strC).Resize(UBound(v) + 1, 1).Value = Application.Transpose(v)
        End If
    Next lngR

End Sub

herb tyson say:

I suffer from occasional optic migraines, and on occasion, I find the default background white to be blinding. I've eased the visual shock somewhat by changing Windows' default window background from white to a shade of gray. This solves the problem not just in Word, but in all other applications that use Windows' default settings.

If you're using Windows 7... click Start - Control Panel - type window color in the search box - click Change window colors and metrics. Set Item to Window. Set Color 1 to a shade of gray (you might need to experiment to find something comfortable). Click Apply and see if a Word or other application is now less blinding. Continue to experiment until you find something that's comfortable, and click OK when you're satisfied. This will not affect how documents print... just how everything display on your own computer.
