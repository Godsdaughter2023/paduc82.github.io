How do I keep the home tab on top - Microsoft Community

loriubell say:

How do I keep the home tab on top

I'm not even sure if I have the terminology correct, but in the ribbon, there are tabs marked File, Home, Insert, etc. My documents come up with the File tab. A lot of the stuff I used regularly is in Home. I looked in the Customize Ribbon section, but don't see how to get the Home tab to come up and stay up. I am using Word 2013 on a Windows 7 PC.
jay freedman say:

The File "tab" is not like the others -- it opens what Microsoft calls the "Backstage", with commands Info, New, Open, Save and so forth down the left side. It may appear to be selected because it's blue or black, but if you can see your document at all, File is not selected.

The Home tab is selected when you first open a document. When it's selected, the Clipboard group (Paste, etc.), the Font and Paragraph groups, and the style gallery are visible. The others -- Insert, Design, etc. -- become selected when you click them. Then there's a whole set of "contextual" tabs that appear when you click on something they apply to -- tables, pictures, WordArt, and so on -- and disappear when you click something else.

There is, by design, no way to keep the Home tab permanently selected. If there were some way to do that, you wouldn't be able to get to any of the other commands.

There are two things you can do:

  • One is to add the commands you most commonly use to the Quick Access Toolbar. Right-click any button on any of the ribbons and choose Add to Quick Access Toolbar. Those will always be available; and you can easily right-click a button and choose to remove it when you don't need it any more.
  • The other is to add custom groups to the Home tab in the Customize dialog (click Home in the list of tabs, then click the New Group button), and add your favorite commands to those groups. (You can't change the content of the built-in groups, although you could hide a built-in group and then create a custom group with the same buttons plus additional ones.) That will minimize the number of times you have to switch from Home to another tab and back again, although it won't prevent the contextual tabs from appearing.

The usual shortcut for inserting a page break is Ctrl+Enter. You could use the Customize Keyboard dialog to restore that shortcut (the command you want is InsertPageBreak under All Commands). The only problem with that is that (a) there appears to be no command for the line break, and (b) it is not possible to assign Shift+Enter to a command (at least not through the Customize Keyboard dialog). Some of the complexity involved is suggested by http://en.wikipedia.org/wiki/Newline.

Since I have been unable to assign Shift+Enter through the UI, it must have been programmatically (via VBA) so a VBA (macro) solution may be required. Or it may be that reassigning Ctrl+Enter to InsertPageBreak (and removing the Shift+Enter assignment, if it exists) will free up Shift+Enter to work for line breaks again.
