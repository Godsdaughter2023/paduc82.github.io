    fahimulla k say:

    Hi,

    The items that appear on the Desktop can be stored in two separate locations.

    a. The items in the  C:\Users\your name\Desktop  folder only appear on that users Desktop.

    b. The items in the  C:\Users\Public\Desktop  folder will appear on all users Desktop.

    If you create or change the desktop shortcut C:\Users\your name\Desktop, it will not affect other users on the computer. On the other hand if you’re trying to create or change the desktop shortcut from C:\Users\Public\Desktop it will affect all the users on the computer.
