petrib say: The VPN is a bit problematic, I'm afraid.

For actual VPN tunnels, it will probably work with most any IPSec capable and standards compliant VPN gateway.

What's problematic is that in order to configure the client software on the phone, you must first create a policy file that tells the app where to go and how.

That isn't directly the problem, but what is, is the fact that currently the only way to get a policy file created is to have a special Nokia Security Service Manager server that is used to create the policy (as a directly downloadable policy file, or a "standalone" .sis file to install on the phone separately).

The only way I know how to get a Nokia Security Service Manager, to create the policy file, is by also buying a Nokia VPN gateway (which most companies or universities or other schools probably haven't done and won't be doing, either).
