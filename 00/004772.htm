Hi Creaturebot

 

Welcome to Nokia forum! :Cool:

 

In the majority of these instances, the following procedure will fix the issue:

 

  1. From the main menu (and with the SIM Card still inserted), select Settings.
  2. Select Configuration.
  3. Select Preferred Access Point.
  4. Select a listed access point, usually named after the operator e.g. Internet, SmartInternet, ORANGE.
  5. Go back to the main menu.
  6. Select Messaging.
  7. Open Email and proceed.

If a Communication Error still occurs, please follow these steps:

 

  1. Go to your Network Operator's support page and request the Access Point configuration settings for Internet (GPRS). The Operator might be able to send these settings to your phone via a text message. If the Operator can send via a text message, go to step 7.
  2. From the main menu on your device, select Settings.
  3. Select Configuration.
  4. Select Personal Config Sett. (or Personal settings)
  5. Select My Access Point.
  6. Enter a name for your access point and input the settings obtained from your Network Operator for Internet settings. The settings may already exist. Please ensure the information is correct and create a new access point name if the Operator's most current settings are not in the list of specified access points.  
  7. Select the newly configured access point name.
  8. Try  to sign in with email again.

Check that you can reach Internet by using other applications.

neo wu msft say:

Hi Harm,

 

Thanks for your posting.

 

The sites will be locked and have the home page removed while the process wait to confirm that the Group is cleaned up. The site will typically be removed anywhere from 5 min to a few hours.

If 24 hours have passed then the site will be orphaned. Based on the lock state, a 404 would be expected if the Group was deleted and site locked.

 

That's the reason why deleted site's link still in there.

 

Here is a workaround:

  1. Run Set-SPOSite on your site to set the lockstate to "unlock". ( set-sposite -identity "site URL" -lockstate "unlock")
  2. Run Remove-SPOsite to delete the site.

 

For your reference: Connect to SharePoint Online PowerShell

 

Please have a try and share with me your result.

 

If you still can see your site in SharePoint homepage, please share with me more info as below:

  1. Detailed steps about how you create this site, how you deleted and recreated it.
  2. How long since you deleted this site?

 

Thanks,

Neo
