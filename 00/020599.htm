jim gordon mvp say:
As Phillip and Bob pointed out, Word is unable to embed video. The best you can do is link to video,

Your first challenge is to get the video from YouTube and turn it into a movie file on your Mac.

Then make a new folder. Make sure both the Word document and the movie file are closed. Put the Word document file and the movie file into the same folder. Do not make any subfolders.

In Word, insert a picture that is to be the placeholder for the movie.
Then, right click on the placeholder and choose Hyperlink from the pop-up menu. 
In the Insert Hyperlink diaog, choose the middle Document option and then click the top Select button. Navigate to the movie file and then click the Open button.
Click OK to close the Insert Hyperlink dialog

Now you should be able to click on the picture and the movie will play using the default player for movies on your Mac.

It is important that you now think of the entire folder as a unit. If you don't rename the movie file and you keep the folder intact when moving the document from place to place, the link will not break.

Problem solved?

 

My solution was found to be to simply refresh (reload) the USB driver for c:\program files\common files\apple\mobile device support\drivers.

 

Chris Lim explains the method using Windows 8.1, and it worked exactly the same for me in Windows 10Technical Preview build 10122..   see https://www.youtube.com/watch?v=vkG9NfKR1DA  

 

Procedure

  • Plug in your iPad or iPhone
  • select Device Manager
  • select Portable devices
  • select iPhone or iPad
  • select Update Driver software
  • RIGHT click and select  'Browse computer for driver software'
  • Browse to c:\program files\common files\apple\mobile device support\drivers (assuming 64bit) or c:\program files (x86)common files\apple\mobile device support\drivers for 32bit systems.
  • Click on OK
  • Click on Next
  • You should see a message saying..   'Windows has successfully updated your driver software'   This should now mean that the Apple Mobile Device USB Device has now been renewed/refreshed/updated
  • Click on Close

 

I hope this works for you too!

SELECT Notes.[person id], Notes.[First Name], Notes.[Last Name], Notes.[66 Date/Time], Notes.[67 Entered By], Notes.[68 Note], Notes.Credential, Notes.Type, Notes.[Expiration Date]
FROM Notes;

Some people have one line entry, some people have 2, etc.  The most I have seen is 5.  Each entry per person has a different date.  I am trying to find a way to get the most current note so I don't have to manually go through my 2500+ records.

I have tried several things without much luck.  Data was pasted in a table from Excel.  The 66 Date/Time field is date and time, but I have formatted it to short date.

 

Any help would be appreciated.

john p spencer say:

One method is to use a correlated sub-query to get the latest date (max) for each person id.

SELECT Notes.[person id]
, Notes.[First Name]
, Notes.[Last Name]
, Notes.[66 Date/Time]
, Notes.[67 Entered By]
, Notes.[68 Note]
, Notes.Credential
, Notes.Type
, Notes.[Expiration Date]
FROM Notes
WHERE Notes.[66 Date/Time] =
   (SELECT Max([66 Date/Time])
    FROM Notes as Temp
    WHERE Temp.[Person Id] = Notes.[Person Id])

This is probably the simplest way to get the results you want.  There are faster ways, but since your field names require the use of square brackets, they require two queries.  Hint: Table names and field names should consist of only letters, numbers, and underscore characters (no spaces, dashes, slashes, etc.).
