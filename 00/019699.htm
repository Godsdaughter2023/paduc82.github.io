Eeeewww....  Just a minute there...  Are you actually talking about TC fields, and not TOC fields?

TC fields are very rare.  Bob has given you the correct answer for TOC fields.

If you are using TC fields, the answer is "Yes.  TC fields contain static text that you must manually update."  So:  Do not use TC fields with Autonumered headings.

Instead, use the Heading Styles to apply the autonumbers, and create the TOC using STYLES not "Table entry fields".  A TOC build from styles uses invisible bookmarks to automatically update the TOC entry with the currently generated autonumbering.

In the Word Help, follow the procedure "Create or edit a table of contents" exactly, and you will get an automatically-updating table of contents.

Note: the formatting of the Styles is irrelevant, set the formatting as you wish.  But you must use the built-in heading styles, they contain "secret sauce" that make Tables of Contents and Numbering "just work".
