ossiemac say:

What do you mean by "Compress"? Do you mean to reduce the saved size Byrtes or to reduce the dimentions of the picture to fit in a given area on the worksheet?

The following code is an event code example that inserts a picture at a given location. The required picture is selected in a DropDown on the worksheet. It adjusts the dimensions of the picture to a given range area on the worksheet. If a picture is already present at the llocation then it deletes it first and replaces with the selected picture.

Event code is installed by right clicking on the worksheet tab name and select View code which opens the module attached to the specific worksheet. Copy and paste the code into the VBA editor. Do NOT change the sub name or it will not work as event code.

You will need to edit the code where indicated with the comments.

If you need more help then please feel free to get back to me but I will need more information on exactly what you require.

Private Sub Worksheet_Change(ByVal Target As Range)

    'Following line: Edit "$B$2" to cell address of DropDown
    If Target.Cells.Count = 1 And Target.Address = "$B$2" Then
      
        Dim rngPicArea As Range
        Dim shpPic As Shape
        Dim strPath As String
        Dim strFile As String
        Dim dblLeft As Double
        Dim dblTop As Double
       
        Set rngPicArea = Range("G4:J9")     'Edit to required range for picture
       
        strPath = ThisWorkbook.Path & "\Pictures\" 'Edit to you path to the pictures
       
        'Alternative example to previous line to set path other than ThisWorkbook
        'strpath = "C:\Users\OssieMac\Documents\Excel\Pictures\"
                 
       
        'DropDown options must exactly match picture file names
        strFile = Target.Value & ".jpg"
       
        'Adding 1 to the Left and Top ensures that the picture is within the range
        'and not marginally adjusted to cell outside the range.
        dblLeft = rngPicArea.Left + 1
        dblTop = rngPicArea.Top + 1
       
        Call DeleteExistPicture(rngPicArea) 'Delete existing picture from range (If exists)
       
        'Setting Width and Height to -1 retains picture current dimensions
        'and locks the aspect ratio.
        Set shpPic = ActiveSheet.Shapes.AddPicture(Filename:=strPath & strFile, _
                                    LinkToFile:=msoFalse, _
                                    SaveWithDocument:=msoCTrue, _
                                    Left:=dblLeft, _
                                    Top:=dblTop, _
                                    Width:=-1, _
                                    Height:=-1)
                                   
        'Resize the picture to the width of the required range area
        With shpPic
            '.LockAspectRatio = True 'Can be True or false
           
            'Subtract 2 from width and height to ensure that picture _
            'is entirely contained within the range
            .Width = rngPicArea.Width - 2
            .Height = rngPicArea.Height - 2
        End With
    End If
   
End Sub

Sub DeleteExistPicture(rngPic As Range)
    Dim shpPic As Shape
   
    For Each shpPic In Me.Shapes
        If shpPic.Type = msoPicture Then
            If Not Intersect(rngPic, Range(shpPic.TopLeftCell, shpPic.BottomRightCell)) Is Nothing Then
                shpPic.Delete
            End If
        End If
    Next
   
End Sub
