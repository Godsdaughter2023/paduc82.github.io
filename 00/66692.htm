bobby mikkelson say:

Hello Christi,

 

When using some video cards, if the resolution is not set to a recommended resolution you may see black bands on the sides or the top and bottom. For example: on the NV G100,ATI HD4530 on L194WT LCD with a recommend resolution of 1440x900, the display cannot be full screen on resolutions 800x600，1024x768，1152x864 or 1280x720. These settings will result in cropping (black bands).

This behavior will also affect the game software that resets the resolution to no-recommended settings.

Hi Jennifer,

I just resolved the issue.

Because the concerning application don’t have neither CLID nor SID then I suspected that the culprit could be OLE (Object Linking and Embedding).

I compared HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Ole on the faulty computer with a completely healthy computer with the same configuration. See screenshot:

What I did:

1)      Exported the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Ole from both computer to REG files.

2)      Deleted the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Ole from the faulty computer.

3)      Imported the healthy computer REG file to the faulty computer and rebooted it.

That's it. The error did not show up anymore.

Why the faulty computer has a “Default access Permission” data field is a mystery.

You should ask a Microsoft engineer / programmer.
