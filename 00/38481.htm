neil prague say:

I spent ages experiencing the same problem and trying to get it sorted. This page is a bit out of date but it's one of the first relevant pages listed when you do a search, so here's my pennyworth. If I can spare someone the same hassle I had I'll be happy!

Assuming the printer itself is functional and you've installed the HP Windows 8 drivers successfully, it's pretty likely that it's a USB issue. The older models of this printer have USB 1.1 ports and most new computers come with USB 3.0 ports. If you're lucky your new computer will have one USB 2.0 port - if so, that could well solve your problem. If not, I'm afraid you won't be able to use your printer until this compatibility issue is addressed.

If you get Windows to try to find a solution when the printer is in an error state it will tell you that you need to use a USB 2.0 port. I spoke to HP and they told me the same thing.

There isn't much you can do. Believe me I've searched far and wide and there isn't yet an adapter for turning a USB 3.0 port into a USB 2.0 - or at least I can't find one. Until something like this is available  you probably will have to use another printer.

I know that this is an excellent (and expensive) printer and it's galling that you can't use it. As I see it the advantages of USB 3.0 (whatever they are) are outweighed by the fact that you can't use a printer that should last a lifetime. Whatever happened to "if it ain't broke don't fix it?"

I'm not claiming this is definitely the cause of the problem, but it is very likely. Windows 8 (which seems pretty good, once you've downloaded a program to get the Start button back!) isn't to blame here. HP should support their products and produce something to link the incompatible USB ports together. Maybe they will, but I'm not holding my breath.

I hope this is of some help.

Neil
