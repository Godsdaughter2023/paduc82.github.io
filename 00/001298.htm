graham mayor say:
The filter (as explained in the link http://support.microsoft.com/kb/196506) is intended to insert HPGL format graphics files into Word. It does not convert printer instructions to graphics.

Insert > Picture > select the graphics file.

Word cannot open graphics files directly.

Incidentally if you have a 64 bit version of Windows, the file locations in the link are incorrect. It would then have to go in
C:\Program Files (x86)\Common Files\Microsoft Shared\Grphflt and the registry key that holds the settings would  be
[HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\Microsoft\Shared Tools\Graphics Filters\Import

Just click on or select the various paragraphs and type a paragraph style name in the field of the Format toolbar:

In the screenshot above, I've moved the toolbar down and placed it over the document.
