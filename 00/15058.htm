There are two possible solutions.

  • Move both the Word document and the Excel workbook to a folder that doesn't contain your login name, preferably a new folder that you create with the name of the project. andreas killer say:

Execute the macro below in your file (with Sheet1 activ). Then setup the page layout so that 4 or 5  tickets fits on one page. Print it out and cut it into pieces.

 

Andreas.

 

Sub CreateTickets()
  Dim Count As Integer, Number As Long
  Dim i As Integer
  Dim Source As Range, Dest As Range
  Count = Application.InputBox("How many tickets?", Type:=1)
  If Count <= 0 Then Exit Sub
  Number = Application.InputBox("Starting number?", Type:=1)
 
  Set Source = Range("A1:K11")
  Application.ScreenUpdating = False
  Sheets.Add
  For i = 0 To Count - 1
    Set Dest = Cells(i * 11 + 1, 1)
    Source.Copy Dest
    Dest.Offset(9, 2) = Number
    Dest.Offset(9, 7) = Number
    Number = Number + 1
  Next
  Application.ScreenUpdating = True
End Sub

a. user say:

Hello Cathy:

 

Thank you for posting to the forum.  The error code 80072FA8 is one related to the connection.  I have seen a few things lead to this error.  The most common issue is with the APN settings so we will start there.

  • First, please go to your phone and turn off your cellular data connection.  Tap settings > mobile network/cellular > OFF.
  • With the data connection turned off, go to settings > Wi-Fi > find a local Wi-Fi connection and connect.
  • Once you are connected to the Wi-Fi, go to settings > emails + accounts > tap and hold the email you are trying to use > tap the sync option.  If you see the words "attention required" at any point, let us know the details of this error.
  • Check your inbox for new emails and try to send an email from the current location.

If I am mistaken that you are connecting to Wi-Fi at work, you will want to contact your mobile operator and get the proper or alternative APN settings and add them to the phone.  This information is added from settings > network/cellular.  You will find a add APN button at the bottom of the screen.

 

If you are connecting to a Wi-Fi network at work and the above steps did not help to resolve the issue please try disconnecting from the Wi-Fi network and reconnecting.  There could be an issue with the current Wi-Fi details.

 

Finally, if this is an email that was manually added to the phone, you may want to remove it and readd it to the phone.  The steps to add an email manually can be found on windowsphone.com here, http://www.windowsphone.com/en-US/How-to/wp7/people/set-up-an-email-account.

 

Please let us know if you have any additional questions or concerns.

 

Mark W.
