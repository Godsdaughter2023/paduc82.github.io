VBA experts,

I cobbled together this macro to scale the selected graphic to 130%. I am cutting and pasting a lot of them from Framemaker to Word 2003 and they always insert at a smaller size than the originals. Sub ScaleTo130()
'
' ScaleTo130 Macro
' Macro recorded 1/24/2012
'
Dim response, errormsg As String

On Error GoTo ErrorHandler    ' Enable error-handling code.

With Selection.InlineShapes(1)
    .ScaleHeight = 130
    .ScaleWidth = 130
End With

Selection.MoveRight Unit:=wdCharacter, Count:=1

response = MsgBox("Selected graphic has been scaled to 130%.", vbOKOnly, "Scale Graphic")

ErrorHandler:
If Err.Number = 5941 Then
errormsg = MsgBox("You need to select a graphic first.", vbOKCancel, "Error")
End If

One of the things you'll find as you work with VBA is that what you think might be parallel constructions often aren't. In this case there are several big differences in the way VBA deals with InlineShapes and Shapes.

The first one is that there's no such thing as Selection.Shapes; instead, it's Selection.ShapeRange. (Don't ask why -- that way lies madness.) But then you can use Selection.ShapeRange(1) to get the first shape in the selection, similar to Selection.InlineShapes(1).

The next thing is that the .ScaleHeight and .ScaleWidth for an InlineShape are properties that are simple numbers, but the same members for a Shape are methods with two required parameters (Factor, which corresponds to the simple value, and RelativeToOriginalsize which is a Boolean value) and an optional third one (Scale, which says which part of the picture sits still while the rest is resized). To top it all off, while the scaling numbers for an InlineShape are percentages (130 in your case), the values of Factor for a Shape are multipliers (1.3 in your case). Got headache?

Getting the If...Else structure is a piece of cake after all that...

Sub ScaleTo130()
'
' ScaleTo130 Macro
'
On Error GoTo ErrorHandler    ' Enable error-handling code.

If Selection.Type = wdSelectionInlineShape Then
    With Selection.InlineShapes(1)
        .ScaleHeight = 130
        .ScaleWidth = 130
    End With

    Selection.MoveRight Unit:=wdCharacter, Count:=1
  
    MsgBox "Selected graphic has been scaled to 130%.", vbOKOnly, "Scale Graphic"
ElseIf Selection.Type = wdSelectionShape Then
    With Selection.ShapeRange(1)
        .ScaleHeight Factor:=1.3, RelativeToOriginalsize:=msoTrue
        .ScaleWidth Factor:=1.3, RelativeToOriginalsize:=msoTrue
    End With

    Selection.MoveRight Unit:=wdCharacter, Count:=1
  
    MsgBox "Selected graphic has been scaled to 130%.", vbOKOnly, "Scale Graphic"
Else
    MsgBox "You need to select a graphic first.", vbOKOnly, "Error"
End If

Exit Sub   ' <== need this to avoid running the error handler when there is no error
ErrorHandler:
    MsgBox "Error " & Err.Number & vbCr & Err.Description, vbOKOnly, "Error"
End Sub
