If you already have the outline/subtotals/autofilter applied, you can protect
the worksheet in code (auto_open/workbook_open??).

Option Explicit
Sub auto_open()
    With Worksheets("sheet1")
        .Protect Password:="hi", userinterfaceonly:=True
        .EnableOutlining = True
        '.EnableAutoFilter = True 
        'If .FilterMode Then
        '   .ShowAllData
        'End If
    End With

Short answer: No - not without purchasing a 3rd party OST2PST conversion program which can get expensive.

For future reference - backing up an OST file is of no use. As OST file represents the data contained on the server with the exception of any folder tag as "This Computer Only" in the description which is the case for IMAP accounts in OL'2013/'2016. An ost file can only be opened in Outlook by the original email account/outlook profile that created it. Configuring the same account again results in a new .ost file being created. 

Going forward - suggest you create a new PST file - set it as the "default" for the profile which will result in the default contact/calendar folders located in the PST file which is always accessible as opposed to the .ost file. Meantime, without purchasing a 3rd party tool - your contact/calendar data should be considered lost.
