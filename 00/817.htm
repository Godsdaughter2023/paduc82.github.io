david marcovitz say:

I'm on my iPhone so I can't type full code, but this is easy to do. 

Put two buttons on the first slide. Each will run several lines of code to hide and show slides and then go to the next slide. To hide a slide (code copied from an earlier answer given by John)

ActivePresentation.Slides(x).SlideShowTransition.Hidden = msoTrue

just replace x with a slide number you want to hide. Repeat the line for all the slides you want to hide. Then change True to False and repeat for all the slides you want to show. 

However, you might want to look into custom shows and not bother with VBA. Just set up two custom shows that only contain the slides you want to show. Button A links to one custom show, and button B links to the other. No  VBA and very simple to implement.
