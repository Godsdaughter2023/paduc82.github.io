How to reformat an existing Excel spreadsheet - Microsoft Community

rdp5 say:

How to reformat an existing Excel spreadsheet

I have a spreadsheet containing questions and answers to open ended survey for each attendee of a course.  Right now my columns contain first name, last name, questions, answers.  I would like to have the questions across the top of the spreadsheet with each of the respondents answers in the row.  Meaning each row would display the attendee data as well as their answer for each question.  How do I reformat my existing spreadsheet to be the way I need it?.
jllatham say:

This macro code should do the job for you.  It doesn't copy the "labels/questions" but does move all the data you want from the original sheet to another sheet that you can designate in the code by editing it.  Everything you may need to define is set up as a Const value which you can edit to match your workbook/worksheets setup.

To put the code into your workbook, open it up and press [Alt]+[F11] to open the Visual Basic Editor (VBE).  In the VBE choose  Insert --> Module.  Copy the code below and paste it into the code module presented to you and make any edits to the Const values in the code.  Close the VBE.  Run the macro from the [Developer] or [View] tab.

Sub TransposeMyData()
  'this refers to the source sheet as you have it now
  'change any Const values to match your actual setup
  'especially the NumberOfQuestions value
  Const SourceSheetName = "Sheet1"
  Const FirstRowWithData = 2
  Const LNameCol = "A"
  Const FNameCol = "B"
  Const QColumn = "C" ' column question is in
  Const AColumn = "D" ' column answer is in
  Const NumberOfQuestions = 3
  'just need the name of the second/Destination sheet
  Const DestSheetName = "Sheet2"
 
  'working variables
  Dim sourceWS As Worksheet
  Dim sourceList As Range
  Dim destWS As Worksheet
  Dim sourceLastRow As Long
  Dim nextDestRow As Long
  Dim SourceRow As Long
  Dim LC As Integer

  Set sourceWS = ThisWorkbook.Worksheets(SourceSheetName)
  Set destWS = ThisWorkbook.Worksheets(DestSheetName)
  sourceLastRow = sourceWS.Range(LNameCol & Rows.Count).End(xlUp).Row
  SourceRow = FirstRowWithData
  Do While SourceRow <= sourceLastRow
    nextDestRow = destWS.Range("A" & Rows.Count).End(xlUp).Row + 1
    'copy the names
    destWS.Range("A" & nextDestRow) = sourceWS.Range(LNameCol & SourceRow)
    destWS.Range("B" & nextDestRow) = sourceWS.Range(FNameCol & SourceRow)
    For LC = 1 To NumberOfQuestions
      destWS.Range("A" & nextDestRow).End(xlToRight).Offset(0, 1) = _
       sourceWS.Range(AColumn & SourceRow)
      SourceRow = SourceRow + 1
    Next
  Loop
'a little housekeeping and we're done
  Set sourceWS = Nothing
  Set destWS = Nothing
End Sub

Sorry, I forgot to add the code to unprotect/protect the sheet.

If all sheets in your workbook are protected with the same password, you can use the following:

Sub RefreshPT()
    Dim wsh As Worksheet
    Dim pvt As PivotTable
    For Each wsh In Worksheets
        wsh.Unprotect Password:="secret"
        For Each pvt In wsh.PivotTables
            pvt.PivotCache.Refresh
        Next pvt
        wsh.Protect Password:="secret"
    Next wsh
End Sub

where "secret" is the password.

CTRL+CLICK selects individual messages, SHIFT+CLICK selects everything between two clicks.
