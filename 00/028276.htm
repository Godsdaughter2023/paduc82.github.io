andythebaker say:

Hi keparkin

This worked for me as I had exactly the same issue

Step 1: Go to the Tools menu, and select Internet Options.

Step 2: Once the Internet Options window appears, select the Security tab, and click Custom Level.

Step 3: Scroll down to the ActiveX controls and plug-ins group, and ensure that all ActiveX options are set to either “Enable” or “Prompt”. Since the remote viewing software is published in an unsigned state, please make certain that the options “Download unsigned ActiveX controls” and “Initialize and Script ActiveX controls not marked as safe” are set to “Prompt”. This way, when you visit the demo, you will be asked if you want to install the software.

Step 4: Click “OK” until you return to the browser.
