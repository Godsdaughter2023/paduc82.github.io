Never mind, I will repeat it here:

1)  Click in a paragraph formatted with Normal style.

3)  Adjust the format of the paragraph properties to your tastes.

suat.cini say:

My registry must be mucked up... 14 failed attempts at the CU .73... troubleshoot hasn’t fixed error 0x80070311

Before you start doing anything else, disable or uninstall any third-party antivirus, antimalware products that you may have. If this does not apply to you, then you should continue reading.

There may be another possibility for your failed attempts: Your disk might have some sectoral damage where the downloaded files land and get corrupted. In this case, it is important to rename SoftwareDistribution folder in Windows directory, rather than clean inside or delete it, and start Windows Update again.

Renaming is important. If you delete the folder, then next time you download the same update, the same files will land on the same location and get corrupted.

If you wish to try again and follow my suggestion, then please open a command prompt with admin rights and run the following commands:

net stop wuauserv
net stop cryptsvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

ren C:\Windows\System32\Catroot2 Catroot2.old

Restart your PC and run Windows Update to check for updates. If your Windows drive is not C:, then change it to the correct drive letter.

Hope this helps.
