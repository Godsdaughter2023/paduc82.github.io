Type of email account (POP, Imap)?

To determine the type if you don't know go to

File --> Account Settings --> Email tab shows all the email accounts configured along with the type of account

If it's POP/SMTP - simply remove the email account which does not remove the associated PST file.

If you delete the IMAP account before completing the export, you'll lose all the data.

Outlook is not a web browser.  Because it uses Word to render HTML, you'd get a singularly incomplete experience.
