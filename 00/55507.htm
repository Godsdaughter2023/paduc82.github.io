kahtleentasman85 say:
Interrupt the normal boot process of your Windows 10 PC
If Windows 10 fails to boot normally three times over, the fourth time it enters by default in an Automatic Repair mode. Using this mode, you can boot into Safe Mode. To trigger the Automatic Repair mode, you must interrupt the normal boot process three consecutive times: use the reset or the power button on your Windows 10 PC to stop it during boot, before it finishes loading Windows 10. If you use the power button, you might have to keep it pressed for at least 4 seconds to force the power off
Then you proceed to repair the computer:
With a reset, all of your Installed applications are lost, but you get an option to save your files if you want. If a system image from the OEM exists-called a recovery partition, then it also gives you an option to restore it. It would make your PC load to default factory settings.
Navigate to Troubleshooting section, again.
Select Reset.
If you select keep my files, your files would be kept. You might also get the third option-restore to factory settings-which isn’t applicable on the PC.
After selecting the desired options, you are presented with a confirmation, something like this- This indicates what you have chosen.
Click on Reset and the resetting should begin.

This is a strange problem that affects some versions of both Chrome and Firefox. The file isn't missing; it's just that it's not found when these browsers invoke Windows Live Mail, but no-one knows why, apparently. An easy solution is to copy the file to the Windows Live Mail program directory. To do this,

 

1.   Close down all running programs.

 

2.   Copy the text in the appropriate box.

 

For a 32-bit system: 

copy "C:\Program Files\Windows Live\Shared\wldcore.dll" "C:\Program Files\Windows Live\Mail"

copy "C:\Program Files (x86)\Windows Live\Shared\wldcore.dll" "C:\Program Files (x86)\Windows Live\Mail"

 3.   Press Windows key + R to open a Run dialogue, type cmd and press Ctrl-Shift-Enter to open an administrator command window. Give permission when asked.

 

4.   Right-click inside the window and select Paste, then press

 

5.   Take note of any message, then close the command window.

 

6.   If there was an error message, quote it in your reply.

      If there was no error message, reboot the computer, launch your browser and test whatever you were trying to do that caused the error to be thrown.

a. user say:

Be careful with your simile: Thanks to global warming, icebergs are moving faster than ever.

A computer runs slowly for either (or both) of two reasons. Either it's being asked to do too much work for the resources it has available, or the hardware has been misconfigured by the user.

A computer is working too hard when it's trying to accomplish too many tasks at the same time. If you open Task Manager to the Startup tab you'll see all the other applications that start along with Windows. Some of them may be using huge amounts of RAM and disc space. Ask yourself: Do they really need to start with Windows?

Malware infection can also be a huge drain on system resources. Well constructed malware will try to use as much as it can.

It's less likely that you've misconfigured your hardware. That usually happens when someone follows some of the ridiculous advice found on the internet for 'tweaking' their computer.
