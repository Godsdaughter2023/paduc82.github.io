Hi rezakim,

Error code 0x80004005 will occur if you're trying to access/manage a folder that you don't have access rights permissions to. The user might be accidentally removed from the user names on the folder’s Security tab. That's why the access permission was lost on that specific folder. Let's try to give full access of the folder to the user that is currently logged in. Kindly follow these steps:

  1. Right-click the folder that you're trying to rename.
  2. Select Properties.
  3. Click the Security tab.
  4. Click Edit....
  5. On the new window that will pop up, click Add....
  6. Another window will pop up. Under Enter the object names to select, type Everyone, then click Check Names.

  7. Hit OK.
  8. Going back to the previous window, under Group or user names, highlight Everyone.
  9. Under Permissions for Everyone, check the box to Allow Full control.

  10. Click Apply, then OK.
  11. Close the remaining sheets.

If the issue will persist after performing the steps above, try to create a new user profile, and check if it'll make any difference.

    • I assume, for example, that you are able to print to PDF using the built-in Microsoft print to PDF printer.  Please try this now if you have not used it recently.
    • I suspect that your printer connecting to that store's computer successfully was a red herring.  Unless they ran through the installation procedure then they could only have automatically installed the MS-sourced printer driver and that proves nothing other than that the printer & its USB cable have not blown up or melted.  That MS-sourced driver is probably 10MB but the HP software is round about 100MB because it does a lot more than just install a driver.
    • My printer is an HP DJ 2540 so I imagine that there is hardly any difference in the procedures required.  Unlike you, however, I hardly ever use my printer nowadays.

    1  Go to the Printers & devices dialog, right-click on the printer, check all the various properties entries looking for anything that might be useful diagnostic data.  {Start menu, Windows system, Control panel, Devices & printers [or Hardware & Sound, View devices & printers]}

    2  Go to the Start menu, HP entry.  If an uninstall item is there then use it as it will give you the cleanest possible state from which to start again.

    3  If there was no uninstall item then check your HP user manual for uninstall instructions and use them. 

    4  Check the HP website for uninstall instructions and use them. 

    • Start at https://support.hp.com/gb-en/product/HP-DeskJet-2130-All-in-One-Printer-series/7174550 and see the guidance in all the tabs on the left.
    • Then do a search for uninstall printer / delete printer / remove printer / ...
    • Then go to their user forum - https://h30434.www3.hp.com/t5/forums/searchpage/tab/message?advanced=false&allow_punctuation=false&q=deskjet+2130 and see what guidance exists.
    • Then do a search in the user forum for uninstall printer / delete printer / remove printer / ...

    5  Go to Settings, Devices, Printers & scanners and select any remaining entries that refer to your printer then click on Remove device.  If you are given any options such as Remove all its drivers & settings? then say Yes.

    6  Go to Device manager. Set View, Show hidden devices.  If any entries for this printer still exist then right-click on them and select Uninstall device.    {Start menu, Windows system, Control panel, Device manager [or Hardware & Sound, Device manager]}

    7  Before proceeding with another attempt, stop all Windows hardware updates [this is a broad weapon and it means you will have to check for hardware updates manually every so often] - go to Start menu, Windows system, Control panel, System [or System & security then System], Advanced system settings, Hardware tab, Device installation settings, No, Save changes.  We can also discuss the ramifications of this setting afterwards.

    8  Go to the HP link again and download their latest software for your printer.  Choose the full software download not just the basic driver.  Check that it says it is the Windows 10 version of the software.  While it's downloading, check their installation instructions [they normally tell you at what stage of running their software to connect the printer - they used to insist that you uninstalled the previous version first but recent versions have not done so].

    9  Disconnect from the internet.  This is just to ensure that Windows cannot start doing anything distracting.

    10  Run their installation software.  {When it completes, it will probably ask you to register online but you can skip that or just do it whilst not connected to the internet and it should then confirm that it has finished}

    11  Test the printer.  

    • Use the buttons on the printer to print its test page [to prove the printer itself is okay]
    • Go to your Start menu, HP group and run the software.  It will attempt to communicate with the printer so might give you useful information straightaway.  The opening page of the HP Printer assistant window that opens probably has an entry for Maintain your printer & that has a Print diagnostic test page item.
    • If it all works then try printing from one of your applications as you would normally do.

    12  If the printer still does not work then go to your Start menu, HP group and run the software.  In the HP Printer Assistant window, select Utilities in the toolbar/ribbon, select HP Print & scan doctor [it will download HPP&SD if it needs to so you might need to reconnect to the internet at this stage].  In my case, this tool was wonderful and immediately identified a stupid mistake I had made in setting up my printer.

    13  If this does not work then I suggest running SFC and then trying again.

    14  If this does not work then I suggest running a "Repair install" and then trying again.  I can guide you through that but, in essence, you create a Windows 10 installation disk on USB then run its setup.exe file from within File explorer so it overwrites Windows files [run this way, it does not reinstall as such so your own files & your installed applications are not affected even though it is still worthwhile getting your backups up to date first].
