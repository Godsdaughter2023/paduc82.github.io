aksharpd say:

how much data does internet radio uses?

Considering no Fm option in Lumia 920, If I use internet radio for an average of 2-3 hours per day can any one tell me how much data i may end up using every month ?
crazy kal say:

Depends on the bitrate of the stream. Often you can choose between various bitrates and also between different codec types.
Lowest bitrate I have seen for internet radio is 24kbit/s ,  highest is 320kbit/s.

Now, if I have done my maths correctly, one hour of streaming at:

24kbit/s    - uses approx. 10.5MB data / hour.

32kbit/s    - uses approx. 14MB data / hour.

64kbit/s    - uses approx. 28MB data / hour

128kbit/s  - uses approx.  56MB data / hour

320kbit/s  - uses approx.  140MB data / hour

 

The higher bitrates give a better sound quality, because the compression is less.
 

(For listening to music on a smartphone I personally find 64kbit/s is pretty good. 32kbit/s is easily good enough for speech. With higher quality headphones 128kbit/s might be worthwhile. Best to try.)

Select B:E; I'll assume that B1 is the active cell within the selection.

Save an excel file as both a .xlsm and .xlsx? - Microsoft Community

jane ann w. say:

Save an excel file as both a .xlsm and .xlsx?

I have my file run every morning, and it saves as a .xlsm. How do I write a script that will save it as an .xlsm as well as an .xlsx? There is information copied into the spreadsheet and the code runs when it hits save and will run then save then close.

[Moved from: Office / Excel / Windows 10 / Office 2016]

ossiemac say:

Try the code below.

The code has been tested but ensure you have a backup of your original file in case something goes wrong.

 

The following code goes in the Macro Enabled workbook. It first saves the Macro Enabled workbook and then it saves as the xlsx copy with the same name as the macro enabled workbook but with the xlsx file extension.

Reason for closing the xlsx workbook immediately after saving it is that the code does not actually disappear from Excel until the workbook is closed. The code is NOT removed from the macro enabled work.

Sub SaveXLSXCopy()
   
    Dim strFileName As String
   
    'Assign the Macro enabled workbook path and filename to a string variable
    strFileName = ThisWorkbook.FullName
   
    'Replace the .xlsm with .xlsx
    strFileName = Replace(strFileName, ".xlsm", ".xlsx")
   
    ThisWorkbook.Save       'Save Macro Enabled workbook first
   
    Application.DisplayAlerts = False   'Suppress message re macros will not be saved
    'Save as xlsx workbook
    ThisWorkbook.SaveAs Filename:=strFileName, FileFormat:=xlOpenXMLWorkbook, CreateBackup:=False
    Application.DisplayAlerts = True    'Re-Enable alert messages
   
    'Close the workbook (which is now an xlsx workbook)
    ThisWorkbook.Close

End Sub
