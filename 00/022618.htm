I don't want to stop using all of the AutoCorrect replacements; I just want to delete some of them, and I want those to be deleted for every Word document from now on. Can you help me figure out how to do that?

Thanks!

jay freedman say:

All AutoCorrect entries that use formatted text are stored in your Normal.dotm template. (The plain text entries are stored in .ACL files so they're available in other Office programs; the Normal.dotm template is specific to Word.)

Whenever you make changes to the formatted entries in the AutoCorrect list, Word will (or at least should) update the Normal.dotm template when you shut down Word. So the first thing to try is to delete the items and then close and reopen Word, maybe restarting the computer in between.

If the entries don't permanently go away when you delete them and close Word, then the Normal.dotm template must not be getting updated. That could happen if there's a permissions problem in the Templates folder where the template is stored. It could also happen if you have an add-in that's telling Word not to save the template.
