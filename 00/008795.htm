copy sheet 1 to the end of the workbook

Put the date into b1 of that new worksheet

prabhakaranradhakrishnan say:

Perform Backup & Restore operations in SharePoint Online

Hi,

I would like to know the SharePoint Sites/Site Collections backup and restore process in SharePoint Online. Can you please answer the below questions.

1) Is this possible to restore only sub site in SharePoint Online?

2) If full site collection restore to the same url by overwritten, then we may have data loss for the recent changes?

3) If someone need 2 or 5 years old data to restore, will it possible in SharePoint Online?

Please explain about SharePoint online has set an RPO and RTO in the event of disaster.

sheen valentine ali msft say:

Hi Prabhakaran,

1. If you have deleted only the subsite, it is possible to restore it via the site collection's own recycle bin. If you have deleted the entire site collection, then you would need to restore the site collection to restore the subsite.

2. After deleting a site collection, it will go into the site collection recycle bin where it will reside for 30 days before it is permanently deleted.

During this 30 days period, if you create a site collection with the same URL, SharePoint will give you a warning and asks you to permanently delete the previous site collection before continuing. This provides a safety net to ensure that if you want to restore the previous site collection, there won't be a new site collection with the same URL in it's place.

In case you accidentally deleted the site collection from your recycle bin, you need to open a support ticket to Microsoft immediately as there is a 14-day window restriction to restore the site collection.

In this scenario, different situations have different restore option, and will be treated on a case-by-case basis.

3. No, backups are performed every 12 hours and retained for 14 days.


SharePoint Online has set an RPO and RTO in the event of a disaster:
1 hour RPO: Microsoft protects your SharePoint Online data and has a copy of that data that is equal to or less than 1 hour old.
6 hour RTO: Organizations will be able to resume service within 6 hours after service disruption if a disaster incapacitates a hosting data center.

The information I provided, as well as more information on SharePoint Online's Service Description can be found under the article below:

SharePoint Online Service Description.

Regards,

Sheen
