When you download Win10 Home China through MediaCreationTool 1709, it - Microsoft Community

allinwonder say:

When you download Win10 Home China through MediaCreationTool 1709, it is not actually Win10 Home China

It actually downloads the same ISO as Windows 10 normal chinese language version (same checksum). It was not the case for 1703.

However, due to Microsoft licensing FUBAR, Windows 10 Home China *is not* the same as Windows 10 Home Chinese or Windows 10 Home Chinese single language, and therefore if your computer comes with Windows 10 Home China, it cannot be activated through OEM license if you install using the normal Windows 10 Chinese Language ISO.

It has been posted on the Chinese language community many times and the staff there doesn't even acknowledge this is an issue.

Further, there is no known way to find older version of MediaCreationTool to download the 1703 ISO, so you are stuck with an OS that can never be activated unless you pay for a retail license, or reinstall when the correct MediaCreationTool somehow somewhere gets fixed.

If this file is over say... 1GB deleting the file and rebuilding the index may get rid of the error ?

Windows.ebd file. Either delete the file, or rename it (so you have it for backup if necessary).
- Reboot.

Method 5:

Internet service providers can charge by the amount of data used (the amount of data sent and received by your PC). That's called a metered Internet connection.

 

These plans often have a data limit, and if you exceed the limit you might have to pay extra. In some cases, you aren't charged extra but your connection speed becomes slower until the billing cycle ends.

 

You may have to change Metered network connection to non- metered connection and check if this fixes the issue.

 

Refer to the steps “How do I change my current network connection to metered (or non-metered)?” from the article “Metered Internet connections: Frequently asked questions” to know how to change the Metered network connection to Non-metered connection.

 

Method 6:

 

If you have proxy server configured, then you may have to disable proxy server for the Windows Store to function properly.

 

Follow these steps to disable Proxy Server:

a.       Press “Windows Key +R” from the keyboard.

b.      Type “inetcpl.cpl” without quotes in the Run window and hit Enter key.

c.       Click the “Connections” tab, and then click “LAN settings”.

d.      Remove the check mark next to “Use a proxy server for your LAN”.

e.      Click Ok.

 

Method 7:

 

Clear Windows Store cache and check how it works.

 

Follow these steps:

a.       Press “Windows key + R” from the keyboard.

b.      Type "wsreset.exe" without quotes in Run window and press Enter.

c.       This will clear the Store cache.
