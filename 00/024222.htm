andreas killer say:
If you debug your macro, the VBA application modifies the state of the task / your machine, means the code runs in a different way.

The VBA application hooks into the interrupts of the Excel application, so Excel runs in a different way.

Any code is a sequence of bytes, the CPU gets the first byte, this is the command, recognizes if it has arguments and if so takes the next bytes. Then the process starts over. That means there is nothing that stops the execution.

A debugger set a machine interrupt hook, this interrupt is called if a command is finished inside the CPU. Then he moves the code segment pointer to the next code line to it's own code segment and the CPU executes his code.

Such things are very hard to explain and I did not know the technical details how VBA and Excel collaborates. Therefore I can not give any better explanation.

jack cook - aka help_line say:

However, although the contacts are already on the Microsoft account, there is nothing appearing in the People Hub on the phone. When I go to Settings => Microsoft account, there is just "Email" opt at the "Content to sync", no Contacts, no Calendar or any other.

SO when you set up your phone with a Microsoft ID, it automatically syncs the Contacts and Calendar from that account. There is no option to sync (or not sync) Contacts and Calendar because it is always syncs them.

As you add additional email accounts (like a second Live account or gmail account), you now get those options for the additional accounts if those accounts allow (have the option for) syncing of Contacts and Calendar. That explains why in your case when you added the account as a second account, the option appeared to sync (in that account) although it is not needed since it should be syncing from the initial setup.

If for some reason they were not appearing in your Contacts or calendar then verify that the filter is set correctly to have them appear. Go to People, click the "..." in the bottom right and click on Settings. From there you will see a list from all accounts that you can show Contacts from. As for Calendr, it is very similar. Go to Calendar then press on "..." and Settings. Make sure that specific calendar is turned on.
