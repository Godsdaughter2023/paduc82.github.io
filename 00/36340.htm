I have resolved the issue.

It was basically BIOS setting. I have search enough to understand that every system is with different BIOS interface and settings. Some of them needs upgrade in order to enable bit-execution in BIOS. 

Here is the best procedure for windows 10 upgrade.

1. Download Coreinfo.cmd from the following link:

https://technet.microsoft.com/en-us/sysinternals/cc835722.aspx

Run cmd.exe as Administrator and type coreinfo.cmd along with path where you saved the file.

Check the specs. Your CPU should support NX, PAE and SSE & SSE2.

NOTE: Do not download any other software (CPU-Z, etc.)as they wont tell you the exact system support features.

2. Your system should have all the updates recommended by Microsoft on you current OS (Operating System) i.e. Window 7, 8 and 8.1

3. Make sure your DEP (data execution prevention) should be on.

  open Control panel>System>Advance System Settings> in advanced tab open Performance settings Look for the tab Data Execution Prevention.

NOTE: if it is does not allow you to change the settings in DEP (data execution prevention), follow the link and you will be able to change it:

http://www.thewindowsclub.com/disable-data-execution-prevention

4. Restart your system and open BIOS settings and enable bit-execute in Security or Configuration (look for it in each option in that way you will learn as well)

NOTE: By doing this if you find any problem its better restart your system, open BIOS settings again and restore it as Default setting.

5. If you can`t wait for the schedule upgrade time then download MediaCreationTool.exe (written as Download Tool Now) from the following link and upgrade it:

https://www.microsoft.com/en-gb/software-download/windows10

I hope this is the best procedure.
