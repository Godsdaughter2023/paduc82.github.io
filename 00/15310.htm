mike h.. say:

Hi,

 

Keeping the list of keywords and worksheets within a worksheet makes for easy maintenance meaning you can add/alter delete keywords as you require.

 

Create a worksheet call Keywords and paste the data in your last post into columns A and B

 

With the summary sheet as the active sheet run this code and it will place things on the correct sheet

 

Sub search_Word()
Application.ScreenUpdating = False
Application.Calculation = xlCalculationManual
Dim LastRow As Long, MyRange As Range, CopyRange As Range
Dim c As Range, WordToFind As String, DstSheet As String
Dim Keywords As Range, LastRow1 As Long, TheRow As Long
LastRow1 = Sheets("Keywords").Cells(Cells.Rows.Count, "B").End(xlUp).Row
Set Keywords = Sheets("Keywords").Range("A2:A" & LastRow1)
LastRow = Cells(Cells.Rows.Count, "B").End(xlUp).Row
If LastRow < 22 Then
    MsgBox "No data to search"
    Exit Sub
End If
Set MyRange = Range("B22:B" & LastRow)
For Each c In MyRange
    If Not IsError(Application.Match(c.Value, Keywords, 0)) Then
        TheRow = Application.Match(c.Value, Keywords, 0) + 1
        DstSheet = Sheets("Keywords").Cells(TheRow, 2)
        LastRow = Sheets(DstSheet).Cells(Cells.Rows.Count, "A").End(xlUp).Row + 1
        c.EntireRow.Copy Sheets(DstSheet).Cells(LastRow, 1)
    End If
Next
Application.ScreenUpdating = True
Application.Calculation = xlCalculationAutomatic
End Sub

suzanne s. barnhill say:

Word 2010 introduced a "safety net" for users who (inadvertently or ill-advisedly) close a document without saving. Word saves a copy of such files for a limited period (about 48 hours, I think). To access these files, click File and choose the Recent tab. At the bottom of the screen, click on Recover Unsaved Documents. These are .asd files, the same kind of AutoRecovery backups that you would get if Word hung or the system crashed, but now, instead of deleting them when you close the file or quit Word, Word saves them for a little while longer, "just in case."

Note that this is a user option: at File | Options | Save, there is a check box for "Keep the last autosaved version if I close without saving. What this implies is that the version saved may not be the final version that you would have if you had saved manually, especially if your AutoRecovery interval is long. Even with the default 10-minute interval, the "autosaved version" could be missing up to 10 minutes' worth of work done just before you closed the file.
