hairyfool say:

Wargaming (the developers for WoT) are themselves a Belarus/Cypriot company and as such are likely to use servers based anywhere in Eastern Europe.

On line games will by default try to attach to the nearest server because they will produce the best network performance, in fact geographical distance is meaningless. The game itself will contain little to define where it connects to so there is nothing that Windows Store can vet for in terms of malicious code.

The servers can change for a couple of reasons. Local network performance can change due to problems making a more distant server appear a better connection. Wargaming do not own the servers, instead they lease hosting services and these contracts can change. Finally if the server starts to get congested and overloaded it will have a mechanism to reject new players forcing a reassignment. In some cases games will stick with the last known good connection until they are forced to change as above

The ransomware tools are themselves executable programs and system security should as suggested block anything getting on to the PC. In game traffic would not normally include such files either, the Game engine on your PC would not know what to do with them.
