eli qian msft say:

Hi ANDOAK,

Based on my knowledge and test, I don't get the related settings and shortcuts to move bullets from a cell to another cell directly. I try use a macro to achieve the same effect but bullets cannot copy and paste from a cell to another cell actually. Only the text will be moved. If your table is similar with the following table, we could use the following marco to switch the adjacent cells. (Note: The third column is the extra column to store temporary cut variables.)

For example, you move the first row A woman to the second row by pressing Alt+shift+arrow down. The number 1 and 2 will be switch as well. It's not the result you desired. Run the macro which has been assigned with a new keyboard shortcut, you could switch number 1 and 2 again.

The resut is as below:

Sub MoveCells()
'
' MoveCells Macro
'
    Dim wApp As Word.Application
    Dim wDoc As Word.Document
    Dim wTable As Word.Table
    
      Set wDoc = ActiveDocument
      Set wApp = wDoc.Application
     
      Set wTable = wDoc.Tables(1)   ' you need to do some work here.  Which table?
     
      wTable.Cell(1, 2).Select
     
      wApp.Selection.Cut
     
      wTable.Cell(1, 3).Select
     
      wApp.Selection.PasteAndFormat wdPasteDefault
     
      wTable.Cell(2, 2).Select
     
      wApp.Selection.Cut
     
      wTable.Cell(1, 2).Select
     
      wApp.Selection.PasteAndFormat wdPasteDefault
     
      wTable.Cell(1, 3).Select
     
      wApp.Selection.Cut
     
      wTable.Cell(2, 2).Select
     
      wApp.Selection.PasteAndFormat wdPasteDefault
     

End Sub
