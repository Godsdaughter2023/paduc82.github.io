brumanchu say:

Footer has three brackets }}} I can't delete

Hello,

I have a Word document where i built a standard footer.  However, it has added three brackets }}} to the end of the footer that both display and print.  The problem is, I cannot select them nor delete them.  If I delete the entire text in the footer, they still appear.

 

Any ideas on what these are and how i can get rid of them?

Thanks,

Bruce

jeeped say:
It would seem that those are field code braces. If they are, they should appear bold. If those are field code braces, you cannot delete one brace without deleting the entire field. Try pressing ALT+F9 to toggle from field code view to field results view. This should show you what the field codes are representing and you should be able to delete them once you can see them in their entirety.

Reference style - Microsoft Community

rotorwing say:

Reference style

Are there any other reference style format available?  The Vancouver style is the standard in all medical publications and would be extremely helpfull if it was available.  Any help?
yves dhondt say:

Microsoft has not released any other styles. However, the referencing framework is completely open and everyone who has some xslt knowledge can add their own style.

A basic (!) Vancouver style can be found at http://bibword.codeplex.com/releases/view/19772 . Make sure to read the remarks at the bottom if you want your in-text references to be superscripted.

BibWord: Microsoft Word Citation and Bibliography Styles
