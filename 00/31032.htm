rcwhite say:
Hi, zefie.
 
I don't know how it impacts your problem, but ".." (two periods in
succession) has long been the symbol for "parent of the current directory"
(dating back to MS-DOS days when folders were directories). Of course, a
single period means "the current directory" in that same context, but is
often a valid character in a filename - as well as the separator between the
actual filename and the extension.
 
Can you substitute some other character or combination for the ".."?
 
Or, in a Command Prompt window, you can use the Dir command with the /x
switch to see the 8.3 filename for each file. (As usual in a Command Prompt
windows, type the command followed by /? to see a "mini-Help" file listing
all the switches and parameters available with that command.) Even a
one-character filename can be an LFN (Long File Name) if that one character
is not "legal" in an 8.3 filename (also known as an SFN or Short File Name).
In a Command Prompt window, go to that "Within Temptation" folder and enter:
Dir /x
 
You should see a normal Directory listing of all the files in that folder -
but there will be an extra column before the LFN column, with an SFN for
each LFN that does not also qualify as an SFN. Then use that SFN to delete,
rename or otherwise deal with that file.
 
Another possibility is that it is not the ".." that is causing your problem.
You ARE enclosing the entire path in double-quotes, aren't you? That is
required because of the spaces in "Within Temptation" and in the filenames.

How can I fix or erase the background of the photos in my - Microsoft Community

sunlight78 say:

How can I fix or erase the background of the photos in my project???

I want to remove /change the "black "color  on both sides of the photos in my project...How can I possibly do it??? SOS PLS! Thanks...

considerate_guy say:

Movie Maker starts with a black background and then puts your photos into that rectangle.  If you are working with a widescreen project (16:9 setting on the project tab) then unless your photos are also 16:9, they will show black on the sides.  Similarly, even if your project is a 4:3 project with 4:3 photos, if you have vertical (portrait) photos, they won't fit in the rectangle and you'll see black.

 

There isn't a way to change that background color, nor is there a way (within Movie Maker) to crop those photos to fit completely in the rectangle.

 

There are a few solutions/workarounds:

  1. Crop all your photos to the matching aspect ratio in Photo Gallery (or another application).  For example, if your Movie Maker project is 16:9, crop your photos to the same horizontal 16:9 aspect ratio.
  2. Try using some of the Pan & Zoom animations.  Those will zoom the photos in and make the black background less apparent.
  3. Submit a feedback suggestion suggesting that Microsoft add a crop feature, or an ability to automatically "zoom" your photos to eliminate the black background.  An alternate option would be to add a feature to let you change the background color or make it dynamic based on the dominant color in the photo.
  4. Ignore the issue and just be happy that Movie Maker isn't arbitrarily cutting out parts of your photo or stretching it to a distorted aspect ratio.
