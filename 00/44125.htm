Hi ItsTheHappyPanda,
I am Vijay, an Independent Advisor. I am here to work with you on this problem.

S means Store. Windows 10S will allow you to download apps from Store only.
Windows 10 in S mode is designed for security and performance, exclusively running apps from the Microsoft Store. If you want to install an app that isn't available in the Microsoft Store, you'll need to switch out of S mode. Switching out of S mode is one-way. If you make the switch, you won't be able to go back to Windows 10 in S mode. There's no charge to switch out of S mode.

You should move from S mode to either Home or Pro.

To switch your PC out of S Mode, please follow these steps:

Open the Settings App

Go to Update and Security - Activation

In the Switch to Windows 10 Home or Switch to Windows 10 Pro section, select Go to the Store. (If you also see an "Upgrade your edition of Windows" section, be careful not to click the "Go to the Store" link that appears there.)

On the Switch out of S mode (or similar) page that appears in the Microsoft Store, select the Get button.

After you see a confirmation message on the page, you'll be able to install apps from outside of the Microsoft Store.

We can see above that what's occurring is the thread is waiting for an event object which happens to be a notification event. You should not be using "CD-RW" discs.  You should use CD-R.  The technology used to permit optical discs to be "rewritable" (e.g., CD-RW) leads to a shorter lifetime for the discs compared to "write-once" discs (e.g., CD-R).

More to the point, you should be using DVD+R discs (assuming that the optical drive in your computer can write to DVD+R) because a DVD can hold almost 8 times as much data as a CD (about 4.7 GB compared to the 700 MB capacity of the CDs you bought). 

Whatever optical discs you use, you'll have to save the photos in batches such that each batch is no larger than the capacity of the disc to which you are writing.

You probably are referring to this screen:

That screen refers to the format in which files are saved on the CD. 

The first choice -- "Like a USB flash drive" -- means that you record (burn or write) to the CD in multiple "sessions" until the disk is full.  You can also delete files that you have written to the CD (they are not actually deleted, but Windows will no longer find them when you "look" at the CD).  If you use this method, you will only be able to read the CD with a computer (and perhaps only a Windows computer).  The Windows 7 Help system says that you will be able to "use the disc in a Windows XP or later computer." 

The second choice -- "Mastered" -- means that you will record all of the files in one session and that you will be able to use the disc in a non-computer CD player or a non-Windows computer.

For more a more detailed explanation, click Start, then click Help and Support.  Type << which CD format >> into the search box in the Help window and then select "Which CD or DVD format should I use?"

As long as you're going to be using the same Windows computer, you could use either choice ... but if you are thinking that you would like to save the photos on the CD as an "archive," you're probably better off using the "Mastered" format.


Here's how to get the pictures from the different locations on your computer onto the same CD using the "Mastered" format.

1.  Start by going to the first location, selecting the pictures you want to copy to the CD, and click the "Burn" button in the tool bar.

2.  A Windows Explorer window will open with the title "Files Ready to Be Written to the Disc" and you'll see the pictures you had selected in Step 1.  A pop-up at the bottom of your display will also show up, and you can click on it if the Windows Explorer window didn't open automatically.

On the Taskbar at the bottom of your display, you should see the icon for your open Windows Explorer window, like this:

3.  Right-click on the icon for the window with the waiting files and select "Windows Explorer" to open a second Windows Explorer window (you could also press the "Windows Logo" key together with the E key).  Arrange the two Windows Explorer windows side by side on your Desktop.

4.  In the second Windows Explorer window, navigate to the next location where there are pictures that you want to burn to the CD.  Select the pictures and drag them to the window where the files are waiting to be burned.

5.  Repeat Step 4 as often as needed, navigating to the different locations where your pictures are located. 

6.  When you have copied all of the pictures to the window that has the "Files Ready to Be Written to the Disc," click somewhere in that window and press CTRL+A (or click Edit > Select All in the menu bar).  You can then check in the Status Bar at the bottom of the window to make sure that the total size of all of the files is less than the 600 MB capacity of your CD.  Then click "Burn to disc" in the tool bar at the top.
