The email triggers by system automatically when the connected account is out of reach. Whenever you connect another account to your Microsoft account and you login or your Microsoft Account ties to sync with the connected account and it cannot, the email is triggered.

To run these networking commands in a command prompt window
3.1. In the search box on the taskbar, type Command prompt, press and hold (or right-click) Command prompt, and then select Run as administrator > Yes.

Type netsh winsock reset and press Enter.
Type netsh int ip reset and press Enter.
Type ipconfig /release and press Enter.
Type ipconfig /renew and press Enter.
Type ipconfig /flushdns and press Enter.

a. user say:

Hello,

You create additional folders in the Outlook app on your iPad to help you organize your emails. To do this, simply follow the steps below:

  1. In the navigation pane, click Mail   , Contacts   , Tasks   , or Notes   .

  2. In the navigation pane, select the folder under which the new folder will be saved.

  3. On the Organize tab, click New Folder.

  4. In the navigation pane, type a name for the new folder.

    Notes: 

    • As an alternative to having multiple folders, you can organize your items with categories. Not only are categorized events displayed in different colors in the item list, category filters let you show events, contacts, tasks, and notes only in the categories that you want.

    • The above procedure explains how to create a subfolder of an existing folder. To create a folder on the same level in the hierarchy, select a folder, and then press SHIFT + OPTION +  + N .

    • To delete a folder that you create, click the folder in the navigation pane, and then on the Edit menu, click Delete.

Should you have further questions, please let us know.

Hi Jessicac1.

Thank you for posting on the forum.

I understand that your Windows Phone device is lost or stole and you're trying to locate it using the Find My Phone function.
Signing In to the Devices web site ( https://account.microsoft.com/devices ) does not show your device.

Let me first point out that what LCRT has posted is very much correct.
The ONLY way to Remove the device from the Microsoft Account it was registered on, is by Signing in to the Microsoft Account's Devices Web Page and Remove it manually over there. So whatever is happening to your phone right now, without the correct Email Address and Password to the Microsoft Account that was used on it, there is no possibility that it was removed.

In my experience the most common issue in similar situations is if the user is Signing In to the Microsoft Account web site, using different Microsoft Account than the one that was actually set to be used on the phone.

My suggestion would be to try to Sign In to the Microsoft Account web site with any and all the Other email addresses that you have used in the past.
Chances are you've set your phone to use another Microsoft Account then the one you're checking.

Other than that, I would suggest contacting your Mobile Phone Operator as some of the Mobile Operators provide additional services, useful in such situations.
