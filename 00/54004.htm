I got this solution from Silicondust Tech Services which solved this problem:

When you're connected to the new router, you'll need to go into the View network status
and tasks page in Control Panel and change the network to Private/Home network rather
than the Public it probably defaulted to. Windows will block discovery of network devices
when the network is marked as public, thus preventing WMC from accessing the tuners. The
HDHomeRun software uses a different mechanism to find the tuners, so it could still work
even if WMC wasn't.

Hi,

If you would like to change the default Windows sounds, then follow the below instructions:

1.       Right-click on Desktop and select Personalize.

 

2.       Click on the link named “Sounds” at the bottom of the window.

 

 

3.       Next, select a sound event from the list and then hit Browse button. Note that you need to have your customized new sound file in .wav format.

 

4.       Select your desired .wav sound track and click Open and then click Apply.

 

5.       Once you change the sound, just hit the Play button to test the customized sound.
