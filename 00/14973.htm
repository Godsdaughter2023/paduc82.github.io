There has never been the option to re-use an existing .ost file. .ost files can only be used by the original Outlook profile/email account used to create it and are data stores used by Imap and Exchange/Exchange AcitveSync (Outlook.com) accounts.

You can use pre-existing pst files when configuring POP3 accounts either at the time the email account is configured or afterwards by changing the destination delivery location.

Tab Jumping Too Far Right

I'm working on a Word doc and suddenly, after 50 pages, the tab does not indent as it should (.5") but rumps far to the right of the page. This is so frustrating and I've tried resetting it in "Paragraph" but it doesn't work. I would hugely appreciate any help!

suzanne s. barnhill say:

Tab stops operate in two ways:

  1. If you have not explicitly set any tab stops in a paragraph, then there are built-in tab stops at regular intervals. By default, the interval is 0.5" (you can change this in the Tabs dialog).
  2. When you set a custom tab stop, all the built-in tab stops to the left of that position are deleted, so when you press Tab just once, you go to the tab stop you have set.

See http://wordfaqs.ssbarnhill.com/SettingTabs.htm for more.
