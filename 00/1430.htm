benoit barabe say:

Currently there are only two ways, neither are very intuitive, to move a notebook from Skydrive to a different location.

1) Create a new notebook and move the content
    - Create a new notebook on Skydrive (from OneNote or from windows live then click Open in OneNote)
    - When you have the two notebooks opened in OneNote, move each section from the original notebook to the new one
    - Close the original notebook

2) Move the notebook locally then re-share
    - Right-click the notebook you want to move, choose properties, click "Change Location" and pick a new local folder on your computer
    - On the Share tab, share this notebook

If you have already moved the notebook on Skydrive and all you need is to open in OneNote, go to OneNote Web and click Open in OneNote from the home tab. It should open the notebook in your client OneNote and start to sync.

Hope this help.

Benoit Barabe, OneNote Development Manager.

They are genuine Microsoft product keys for retail licenses, but not for Retail sale. hansv mvp say:

I assume that you meant "separate the info in column E" instead of "separate the info in column D".

The following macro should be run from within Excel, while the sheet with the data is the active worksheet.

Sub SplitRows()
    Dim cel As Range
    Dim arr() As String
    With Range("E:E")
        Set cel = .Find(What:=";", LookAt:=xlPart)
        If Not cel Is Nothing Then
            Application.ScreenUpdating = False
            Do
                arr = Split(cel.Value, ";")
                cel.EntireRow.Copy
                cel.Offset(1, 0).EntireRow.Insert
                cel.Value = arr(0)
                cel.Offset(1, 0).Value = arr(1)
                cel.Offset(1, -4).Value = "A" & cel.Offset(0, -4).Value
                Set cel = .FindNext(After:=cel)
            Loop Until cel Is Nothing
            Application.CutCopyMode = False
            Application.ScreenUpdating = True
        End If
    End With
End Sub
