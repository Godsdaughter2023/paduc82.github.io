Try this:

Place a text box in the Shop Header.

  • Set its Visible property to No.
  • Set its Control Source property to =1
  • Set its Running Sum property to Over Groups.
  • Set its Name to txtCount.

Create an On Format event procedure for the Shop Header section:

Private Sub ShopHeader_Format(Cancel As Integer, FormatCount As Integer)
    Me.Company.Visible = (Me.txtCount = 1)
    ...
End Sub

where ShopHeader is the name of the group header section, and where Company is the name of a control in the Shop Header that was originally part of the Company Header.

You can add lines for other controls as needed.

Can i run 2 OS on one computer? - Microsoft Community

mmudloff say:

Can i run 2 OS on one computer?

I have an 8 core with 2 video 2 hard drives 4 monitors can I run 2 Op sys at once.
cbarnhorst say:

The hardware makes no difference.  The only way to run two or more OSs simultaneously on a computer is to run one of them as the supervisor and the others in a virtualization program such as VMWare Player, VirtualBox, or under Hyper-V (on Windows 8Pro or Windows Pro).  

You cannot run more than one natively because of collisions in operating system calls at the same level (ring).  There is no way to even try it.  One OS must host the other(s).  It doesn't matter how may cpus or how many cores each cpu has.  One OS must manage the system and any others must run in an evironment subservient to the host (in higher rings).
