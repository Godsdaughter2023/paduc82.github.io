I have solved our issue to this. Its not the best, but solved the issue. 

But the following are the things that let up to the resolution:

  • On of my coworkers notice a memory.dmp occurred around the time of the 1803 upgrade. Once I examine with the WinDbg tool, it stated, "Probably caused by: tpm.sys (tpm!WppCleanupKm+63)". Then when I did a '!analyze -v' it stated, "KERNEL_SECURITY_CHECK_FAILURE (139)'.
  • Cleared the TPM on the machine, just to be safe.
  • Tried re-applying the Office product key, and had no joy.
  • I decided to do a rollback to 1709 since it was still available.
    • Rollback successfully. Confirm popup was not occurring and Office was still activated.
  • So, now I decided to upgrade back to 1803 via SCCM, just to see if we can reproduce the issue.
    • 1803 upgrade was successful. Popup about license did not re-appear. Office was still activated and work properly.

So I would assume that there was a hiccup or file corruption that occurred during the first 1803 upgrade. 

So remediation - rollback to 1709, and then re-install 1803 upgrade. Hope this information is helpful for others.

Drew Davis

batch in Digital Image if you go to...
......Touchup / Batch Edit In Mini Lab / Save As.....

If any of these files happen to be layered projects
like Greeting Cards...saving in a standard image
format like .jpg will only save the front page and
future editing of the layers will not be possible.

If you plan to purchase a copy of Digital Image
2006 and re-edit the layered files...be sure to
save them in the PNG Plus format.

The number of files you can convert at one time
will depend on your system resources but I
would say try 50 and see what happens.

Restore point before installing software or updates)
