vijay a. verma say:
Attach this drive in another computer and verify whether the drive is working or not. Also try putting in different USB ports.

Assuming that there is no hardware problem with your drive -

1. Right click on Windows Start > Device Manager > Click the topmost entry i.e. your device > Click Action in Menu > Click Scan for hardware changes
Now you should be able to see your USB drive

2, Right click on Windows Start > Device Manager > Expand Disk Drive > Locate the Drive > Right click on this > Uninstall > Reboot your computer
Now it should be detected

3. Use the Hardware & Devices Troubleshooter
Type Troubleshoot in Windows Start Search box > Click Troubleshoot > Click Hardware & Devices > Run the Troubleshooter

4. Whether the Drive is assigned a Drive Letter or not
Right click on Windows Start > Click Disk Management > If you see your Drive is listed there but there is no Drive Letter > Right Click it > Click Change Drive Letter and Paths > Add > Click Assign the following drive Letter > Click in the Drive Letter List Box > Choose an appropriate Drive Letter > OK > OK

5: Fix USB Root Hub
Windows Key+X > Click Device Manager > Expand Universal Serial Bus controllers > Right-click USB Root Hub option and click Properties > Go to Power Management tab and uncheck the Allow the computer to turn off this device to save power. Click OK. > If you have more than one USB Root Hub, repeat this step for all

6. Update / Reinstall Generic USB Hub
Windows Key+X > Click Device Manager > Expand Universal Serial Bus controllers > Right click on USB Hub > Select Update Driver Software > Browse my computer for driver software > Click on Let me pick from a list of device drivers on my computer > From the list, select Generic USB Hub > Next and follow onscreen instructions > Reboot your Computer

If problem not solved, right-click USB Hub of your USB drive and select Uninstall
Reboot your Computer to install driver automatically

The "mediacenterdatastore.db" is not available in Windows 7, it is in Windows Vista. In Windows 7, he needs to find "mcepg2-0.db". I would suggest you to follow the steps mentioned below:

Method 1:

re-register the Windows Media Center components. Follow these steps and check if the issue persists:

a.       Click the Start, All Programs and Accessories.

b.      Right click Command Prompt and choose Run as administrator.

c.       Type or copy the following commands in this order and press Enter after each line.

regsvr32 jscript.dll

regsvr32 atl.dll

cd C:\WINDOWS\eHome

ehSched /unregServer

ehSched /service

ehRecvr /unregServer

ehRecvr /service

ehRec.exe /unregServer

ehRec.exe /regserver

ehmsas.exe /unregServer

ehmsas.exe /regserver

d.      Exit Command Prompt and check if the issue persists.

Method 2:

Disable and enable Windows Media Center from Windows feature and check if the issue persists, follow the steps below:

a.       Click the Start button, click Control Panel, click Programs, and then click Turn Windows features on or off.  If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

b.      To turn off Windows Media Center, click on Media feature. And uncheck Windows Media Center and click on OK.

c.       Restart the computer to apply the changes.

Note: follow the above steps to turn on or enable Windows Media Player and check if the issue occurs.

For more information, refer to this link Turn Windows features on or off:

http://windows.microsoft.com/en-US/windows7/Turn-Windows-features-on-or-off
