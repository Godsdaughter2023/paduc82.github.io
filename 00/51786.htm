a. user say:

The catalog.wci folder is used by the XP Indexing Service.

If you are running the Indexing Service and you don't know or are not sure why you need it or if you need it, I would Disable it since it only slows your system down.

Once the Indexing Service has been stopped, you can just empty the catalog.wci folder, or maybe the files will then defragment.  I don't know for sure if they will defragment since I will never enable the Indexing Service except to test how much it slows my system down.

Sometimes the Indexing catalog file gets corrupted and the way to fix it (if you are compelled to keep using the Indexing Service) is to stop the Indexing Service, empty the catalog.wci folder and restart the Indexing Service.

If/when you start the Indexing Service again, it will repopulate the folder.

Read more a little about that here:

http://support.microsoft.com/kb/209304

However you should not be able to get into the System Volume Information folder without a little extra effort since it is a hidden, protected, read only system folder, but if you are compelled to delete the files, we can tell you how to do that.

There are other system files (like registry and paging files) that will sometimes not defragment and for that, you may find PageDefrag of some use:

http://technet.microsoft.com/en-us/sysinternals/bb897426.aspx

The XP defragmenter is certainly capable of a result of zero fragmented files but sometimes you have to run it many times and use PageDefrag with it or do some unnatural things to get to that point.

Remember that the XP defragmenter is a slimmed down version of Diskeeper from Condusiv Technologies and while it is certainly possible to have zero fragmented files with the built in XP defragmenter, some people like third party tools better.

Here are some examples (in no particular order) of popular third party tools that you can investigate and you can do some Google searches for others:

Some people like the Piriform suite of products and their Defraggler application which you can get here:

http://www.piriform.com/

Some people like the Auslogics suite of products and their defragmenter application which you can get from here:

http://www.auslogics.com/en/

Some people like the PerfectDisk suite of products and their defragmenter application which you can get from here:

http://www.raxco.com/

Diskeeper (not free anymore)

http://www.condusiv.com/products/diskeeper/
