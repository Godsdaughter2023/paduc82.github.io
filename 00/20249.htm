You have Access 2010--rest assured. By default, both Access 2010 and Access 2007 create databases in the .accdb file format, which is referred to as the Access 2007 file format. The utility of this apparent misdirection comes clear when you open an Access in an earlier format... the latest format for Access is 2007.

In File - Options - General, drop down the Default file format for blank database, and you'll see that the available defaults are Access 2007, Access 2002-2003, and Access 2000. The format for Access 2010 is called "Access 2007" because the format wasn't changed when Access 2010 came out.

I hope this helps ease your mind.
