markinthebox say:

WiFi is turned off when the phone sleeps.  This is by design, in order to preserve battery life.

The exception is when the phone is plugged in to a power source.  In this case, WiFi will remain ON (if it was on when the phone goes into standby).

So if your phone is plugged-in all night, it should still be using your WiFi connection.  This seems to be working correctly for me, though I haven't checked carefully to be absolutely certain my overnight data usage isn't increasing.

ossiemac say:

Try the code below.

Backup your workbook in case the code does not do as you expect.

Note the comments where you may need to edit the code.

Sub Grouping()
    Dim rngSource As Range
    Dim cel As Range
    Dim strCol As String
    Dim strTemp As String
    Dim lngCount As Long
   
    strCol = "A"        'Edit "A" to your column Id with the source data.
   
    With Worksheets("Sheet1")   'Edit "Sheet1" to your worksheet name
       'Following line starts at row 2 (Assumes column headers in row 1)
        Set rngSource = .Range(.Cells(2, strCol), .Cells(.Rows.Count, strCol).End(xlUp))
    End With
   
    lngCount = 0
   
    strTemp = Mid(rngSource.Cells(1, 1).Value, 6, 3)
   
    For Each cel In rngSource
        If Mid(cel.Value, 6, 3) = strTemp Then
            lngCount = lngCount + 1
        Else
            strTemp = Mid(cel.Value, 6, 3)
            lngCount = 1
        End If
        cel.Offset(0, 1) = "h" & Format(lngCount, "000")
    Next cel
       
End Sub

Table cell alignment - Word 2011

  • If I make the cell heights VERY large, i can see the difference between centre, top centre and bottom centre - but the heights I need to make are unacceptable
  • Finally, when I have chosen 'centre' and made my cells very tall, the text is still not perfectly vertically centred, but slightly to the bottom almost like I have a <return> at the top or something (which I have confirmed that I do not)

Thanks in advance guys!
daniel g. grau say:
  • The Cell Alignment option 'centre' looks exactly the same as 'top centre' and 'bottom centre' 

You won't see a difference unless there is, in the same row, a cell taller than the height that your vertically-centered cell would otherwise require.

  • If I make the cell heights VERY large, i can see the difference between centre, top centre and bottom centre - but the heights I need to make are unacceptable

This depends on whether you've set Space Before/Space After in the neighboring paragraphs, or if you've specified a fixed row height. Select/Click on the row, do Table > Table Properties > Row and take the checkmark off Specify height, if one is there.

  • Finally, when I have chosen 'centre' and made my cells very tall, the text is still not perfectly vertically centred, but slightly to the bottom almost like I have a <return> at the top or something (which I have confirmed that I do not)

This can happen if the text has the Space Before attribute. Select your cell, then do Format > Paragraph and set "Space Before" to zero.

Similar effects, albeit in the other direction, can be caused by differing Line Spacings (1.5, Double, Exactly) and Space Afters.

a. user say:

try:

1. Select your formatted character

2. Click the Insert tab, then the Quick Parts button in the text group.

3. Save Selection to Quick Part Gallery, give your selection a memorable name, and select "AutoText" as the Gallery (I think that's optional), and click OK.

4. Click the Word Office Button->Word Options->Customize->Keyboard shortcuts Customize.... In the Categories column, select Autotext (near the bottom of the column). In the AutoText column, locate and select the memorable name you specified in (3), then assign your keystroke and click Assign.
