That's all simply normal operation for the update process for Java when using a Windows 7 PC with UAC (User Access Control).  You are misunderstanding the prompts, since to uninstall Java you must accept that the program make changes to your computer or the uninstaller can't perform the necessary operations to remove itself.  So in effect, you are telling the program to uninstall and then when it tries, telling it "no, you can't do that".
