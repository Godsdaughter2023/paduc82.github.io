Don't bother with partition wizard.

1. Boot to the recovery environment (a system repair disk will do this for you)

2. Press Shift+F10 on your keyboard to bring up a command prompt

3. Type diskpart and hit Enter

4. Type list vol and hit Enter

5. Now you are looking for a small volume (usually 50 MB to 100 MB in size). This volume contains the boot loader and is the one that needs to be set as active. 6. Type active. You should get a success message.

7. Close the command prompt and reboot.
