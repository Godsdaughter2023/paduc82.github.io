rohn007 say:
Back in early versions of Windows, when you logged in as Admin, you had full admin rights all of the time. But that was a security risk. Back a couple of Windows versions ago MS introduced "UAC", User Access Control, a feature that requires you to provide the admin id and password when doing things that require higher access rights. By Windows 10 they extended UAC to include the Admin ID. So even though you are logged in as "admin", you are operating at standard (or maybe slightly higher, I'm not positive) Windows access rights.

When you see this UAC prompt, the proper action is the submit the password on your admin account and press OK.

If it was created under Skype name, log on at account.live.com using Skype name and password, it will prompt you to enter your Microsoft account.
