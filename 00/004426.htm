cbarnhorst say:

Both are in the Fast Ring.  Fast Ring has two channels. 

One is Fast Ring, Active Development of Windows.  Currently, this channel is receiving Redstone 4 builds.  Redstone 4, aka Spring Creators Update, is coming to an end in a few weeks when the SCU is released for use by the public.  This channel is open to all Insiders.

The second channel is Fast Ring, Skip Ahead to the next Windows release, or just "Skip Ahead" or "Skippy."  This channel is receiving Redstone 5 builds.  Redstone 5 (not yet named by Microsoft) is just underway and is due for public release this Fall.  This channel has limited seating and is currently not accepting new Insiders.  When seats open up the team will announce it in the Feedback Hub.

The idea behind the two channels is that some Insiders (a lot of Insiders, actually) don't want to look for nitpicky bugs (which is all that is left to do for the Spring Creators Update) and want to work with much more interesting software.  So MSFT lets a few of them start finding bugs in Redstone 5.  However, MSFT still needs lots of eyes looking at SCU so that they can release it in as good a state as possible so only a few Insiders get to work with Skippy and most are still working with SCU's final spit-and-polish stage.

After Spring Creators Update (yes, a lot of us think it is a dumb name) goes public next month both channels will begin receiving the same Redstone 5 builds.  This will continue until late Summer when Redstone 5 is almost ready and once again those in Skippy will start getting Redstone 6 builds.  There is always some turnover in who is in Skippy, but so far there doesn't seem to be much.

Those are the two faces of Fast Ring.

thetao---01 say:

Is this a personal data file you deleted (e.g. a text file, picture, etc.) or just some random Nokia file?  If it's the latter, do you remember the name or location of the file?  (Nokia's File Manager gives you access to only a small subset of the data on the phone, making it generally very safe.)  If your phone still works normally, I wouldn't worry about the loss.  I have never used an E7, but the phone memory on my E73 contains only a few log files and remnants of system .sis files that my phone will never miss.

 

If you've accidentally deleted a personal file from the phone memory and don't have a backup, then you may be out of luck.  I am not aware of a "file undelete" utility for Nokia phones.

 

Hope that helps!
