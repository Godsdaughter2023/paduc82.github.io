The formulas copy automatically when the column in which you enter data is used in the other formulas.

Try the following experiment on a blank worksheet.

  1. Enter some numbers in cells A1:A5 (Any random numbers but NOT formulas)
  2. In cell B1 enter the formula =A1*A1 (Could use any formula that incorporates A1)
  3. In cell C1 enter the formula = B1*10 (Again can use any formula so long as source goes back to column A)
  4. Copy the formulas in cells B1:C1 down to row 5 (Same last row as column A)
  5. Now enter any number in the next cell in column A (Should be cell A6 for this experiment)

peggyshepard say:

Hi ctdd,

Another attempt to answer your question; these are different formulas than were in my previous post.

Column D - Used from First Earned is in response to your post: "formula that will subtract hours used from the oldest hours earned first"

Row A B C D E F
2 01/01/10 8 1 1 7 0
3 01/02/10 8 0 0 8 0
4 02/01/11 8 0 8 0 0
5 02/15/11 8 12 5 0 3
6 03/01/11 0 1 0 0 3

Column Column Head                      Formula 
A          Date                                  no formula 
B          Earned                               no formula 
C          Used                                  no formula 
D          Used from First Earned        D2 =MIN(B2,SUM(IF((A:A<=A2+365),C:C)))                D3  =MIN(B2,SUM(IF((A:A<=A3+365),C:C))-SUM(D$2:D2))
E          Expired                              E2 =IF(A2<TODAY()-365,B2-D2,0) 
F          Avail to Use                        F2 =SUM(B$2:B2)-SUM(D$2:D2)-SUM(E$2:E2) 

Column D, use the first formula for D2, use the second formula for D3 and the rest of the cells in column D.

     The MIN is to catch Used hours that will exceed the hours earned as of that date

     The Sum(if((A:A<A2+365),C:C)) is an array formula, so the entire formula should be entered with Ctrl+Shift+Enter keys.

          This portion of the formula is summing used hours, used before 365 days after the corresponding line's date.

     The Sum(D$2:D2) is a running total of Column D, up to the line above, which is subtracted from the formula in the previous line.

Column E, checks whether the earned hours have been used within one year from today for the corresponding line; if not then the expired hours for that line are displayed in this column.

Column F, shows how much of the corresponding line's cumulative hours that are still eligible to be used.
