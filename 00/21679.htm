In PowerPoint, Word and Excel there is an Advanced Option: "Do not compress images in file" that needs to checkmarked on each file.

andreas killer say:
We can do this with a registry key.

Open REGEDIT
Go to HKEY_CURRENT_USER\Software\Microsoft\Office\14.0\Excel\Options
Add a new DWORD key and name it AutomaticPictureCompressionDefault
The value of that key should be 0
Open Excel
Go to the option, you'll see a checkmark.

Repeat the same for PowerPoint and Word.

If want to enable the compression, set the value of that key to 1.

Andreas.

suzanne s. barnhill say:
What you're describing is not "Alt Text" but additional text added to the page number. While it is possible to use the "Include chapter number" feature to add a number to a page number (2-1, for example), and that will be picked up by the TOC, the only way to add text in this way is to type it in manually or use a StyleRef field; in neither case will the added text be included in the TOC. Even if you use TC fields, you can prescribe only the entry text in the TOC; you can't influence the format of the page number.
