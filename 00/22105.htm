Make sure that the number format of the cells is NOT set to Text, but to General or Number.

Select an empty cell.

Select the cells with numbers.

Click the lower half of the Paste button on the Home tab of the ribbon.

hi brandon,

you need to modify a user role in the role assignment policy. i suggest you create a new role assignment policy, for test at first referring the following steps.

1. connect windows powershell to exchange online.

new-managementrole -name mybaseoptions-nofwd -parent mybaseoptions

3. set the role entry for the new created role, and remove the forwarding address and forwarding smtp address parameter:
set-managementroleentry "mybaseoptions-nofwd\set-mailbox" -parameters forwardingaddress, forwardingsmtpaddress –removeparameter

4. log into ecp, and create a new role assignment policy. please choose mybaseoptions-nofwd under other roles, as in the following example:
