List of Malware Removal Tools - Microsoft Community

prashantkumar96 say:

List of Malware Removal Tools

Technical Level : Basic

Summary
This wiki aims to list out some reputed on-demand malware scanners.
Details

When a malicious activity is suspected, it is recommended to full-scan your system with installed antivirus. In addition, it is also recommended to scan with an on-demand scanner as a second opinion. An on-demand scanner is a tool that scans the system only when user asks it to. The main difference between an antivirus and an on-demand scanner is that an antivirus provides real time protection, on-demand scanner don’t. Real Time Protection means antivirus will start as soon as system starts and keep monitoring your system while you are using it. Thus, if malware tries to execute, Real Time Protection will instantly stop it. On the other hand, on-demand scanner will not stop any malware in real time, it will catch malwares only when it is manually ran by user.

Important Note: Nearly all tools mentioned here are not Microsoft products, they are third party tools which hold a good reputation in removing malware. Most of the community members are individual volunteers here and these tools are recommended by most of them.

1. Microsoft Safety Scanner

The Microsoft Safety Scanner is a free downloadable security tool that provides on-demand scanning and helps remove viruses, spyware, and other malicious software. It should be noted that if you have Windows Defender as your current antivirus then scanning with this tool will be of no help as both Windows Defender and this tool share same definitions.

Microsoft Safety Scanner will expire after 10 days after being downloaded.

Download Microsoft Safety Scanner: https://www.microsoft.com/security/scanner/

Note: As reported by PaulSey... in this thread, Microsoft Safety Scanner works in Windows 8(.1) and Windows 10 too.

2. Malwarebytes

Malwarebytes offers a 14-day trial of their Premium edition. This Premium edition provides Real Time Protection and can act as a full antivirus suite. While Malwarebytes claim that they built their product to be compatible with other antiviruses including Windows Defender, we have seen some reports of possible conflict.
After expiration of 14-day trial Malwarebytes becomes Free edition and act as on-demand scanner.

Download Malwarebytes: http://malwarebytes.org/premium

3. Emsisoft Emergency Kit

Emsisoft Emergency Kit is another free on-demand scanner which is portable, which means you can carry it in a USB drive and scan a system offline. Here is a nice guide from Emsisoft on how to use this scanner: https://blog.emsisoft.com/en/16796/how-to-find-and-clean-malware-infections-with-emsisoft-emergency-kit-2/

Download Emsisoft Emergency Kit: https://www.emsisoft.com/en/software/eek/

4. HitmanPro

HitmanPro is an on-demand scanner only. However, it requires you to run a one time 30-day trial if you want to remove malwares it found. After this one time license is used, you have to buy this product. Here is its support link: https://www.hitmanpro.com/en-us/support-hmp.aspx

Download HitmanPro: https://www.hitmanpro.com/en-us/hmp.aspx

5. ESET Online Scanner

Use ESET Online Scanner: https://www.eset.com/us/home/online-scanner/

6. Kaspersky Virus Removal Tool

Kaspersky Virus Removal Tool is a free tool for scanning and disinfecting Windows computers.

Download Kaspersky Virus Removal Tool: http://support.kaspersky.com/viruses/kvrt2015

If a rootkit gets detected during the scans, anti-rootkit utility TDSSKiller can be used: https://support.kaspersky.com/viruses/disinfection/5350

7. Avira PC Cleaner

Download Avira PC Cleaner: http://install.avira-update.com/package/pccleanerwebloader/win32/en/avira_pc_cleaner_en.exe

8. Norton Power Eraser

Norton Power Eraser is a free tool that can be downloaded and run to remove threats from your computer.

Download Norton Power Eraser: https://security.symantec.com/nbrt/npe.aspx?&NUCLANG=en-us

rob koch say:

If you've read the threads referenced by Smeed in his recent post above, you'll see that there are many reasons that a scan, especially a full scan of many more files, may vary in length of time required.

However, I don't believe that any of these threads or other posts above mentioned the most common reason these scan times can vary on the same exact PC using the same selections with Windows Defender.

In this case, it's often the changes that occur to Windows Defender's malware detection definitions and associated scan engine code extensions, that evolve over the typical 1 to 2 month lifetime of a set of engines, which causes the greatest variance in scan times.

Though these scan engines are extensions to the core scanning code contained in Defender itself, they are intimately linked to the current set of detection definitions contained in the local database stored on each PC.  This is why these engines and definitions are downloaded and updated together, typically on a monthly schedule that resets just a few days after mid-month, entirely replacing both the engines (code) and definitions (detections database) to reflect the most prevalent malware and families currently known to be infecting Windows systems worldwide.

The reason this monthly definitions update reset can have a significant effect on scan time is that the engine code snippets it includes are re-optimized at this time to perform most effectively with the set of detection definitions they are initially released with.

However, over the month or so period this particular set of definitions and engines exist, the definitions database itself will grow in size, reducing the performance of any scans and so increasing the time required to complete.

As you've already learned from other posts and threads referenced above, due to the real-time monitoring for malware by Defender, many of us rarely perform full scans any longer and I hadn't done one on this Windows 8.1 laptop in possibly a year.  So I just performed one out of curiosity and though its Solid State Disk (SSD) drive and dual-core hyper-threaded processor had typically completed a full scan in from 9 to 12 minutes in the past, the current scan required 37 minutes for approximately 551,000 items.

If I were to try another full scan when the next engines and definitions database, reset update occurs, likely in the next few days, this would likely be reduced significantly.  Though I doubt it would return to the 9 minutes it used to take due to both increases in the number of files the system contains and the overall number of malware likely defined in a more recent definitions database set.
