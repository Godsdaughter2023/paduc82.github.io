Hi, I recommend following steps.

  • One column must contain the date in Microsoft Format (DD.MM.YYYY).
  • An other column "Submit-Month" will convert this date in format "YYYY.MM"
  • Use following function to convert Date into Text String. =STXT(F908;7;4)&"."&STXT(F908;4;2)

Secondary email addresses on a mailbox are receive only - you need to have the address on a contact, DL, or shared mailbox and give the user send as permission on it. Or you can use a 3rd party utility to sent the from field.
see https://www.slipstick.com/exchange/exchange-ser... for more information and links to utilities.
