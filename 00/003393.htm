If you start with a two-column slide layout and you're typing in the first column, you can either click in the second column and continue typing your data OR press Ctrl+Enter to do the same thing.

Hi,

create first to cells with data validation, assume cell D1 for the depts and cell E1 for the functions. Your data is in range A1:C12. In this case use the formula To sumin cell F1 and To count in cell G1:

Excel 2003:

To sum: =SUMPRODUCT(($A$2:$A$12=$D$1)*($B$2:$B$12=$E$1)*$C$2:$C$12)

To count: =SUMPRODUCT(($A$2:$A$12=$D$1)*($B$2:$B$12=$E$1)*($C$2:$C$12<>""))

Examples:

D1: Admin
E1: Working
F1: =SUMPRODUCT(($A$2:$A$12=$D$1)*($B$2:$B$12=$E$1)*$C$2:$C$12)
Result: 10

D1: Admin
E1: Working
G1: =SUMPRODUCT(($A$2:$A$12=$D$1)*($B$2:$B$12=$E$1)*($C$2:$C$12<>""))
Result: 2

 

Excel 2007/2010:

To sum: =SUMIFS($C$2:$C$12,$A$2:$A$12,$D$1,$B$2:$B$12,$E$1)

To count: =COUNTIFS($A$2:$A$12,$D$1,$B$2:$B$12,$E$1)

Examples:

D1: Admin
E1: Working

D1: Admin
