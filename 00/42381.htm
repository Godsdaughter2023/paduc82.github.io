Can i make a bootable Windows 10 DVD that supports UEFI? - Microsoft Community

matteomatteonx say:

Can i make a bootable Windows 10 DVD that supports UEFI?

So, it is simple: can i make a bootable Windows 10 DVD that supports UEFI?

davem121 say:
Hi Matteo, the Windows 10 installation Media created from the Media Creation Tool will install onto either MBR (Legacy BIOS) or GPT (which supports UEFI)


To manually wipe a drive and convert it to GPT:

Turn off the PC, and put in the Windows installation DVD or USB Drive

Boot the PC to the DVD or USB key in UEFI mode

Once the Installation Starts, press Shift + F10 to open Command Prompt:

Run these commands, one at a time and hit Enter after each one:

diskpart


Now make a note of the drive No. to be converted

list disk


Now select the drive (replace DISKNO)

select disk DISKNO

clean

convert gpt

exit

Close the command prompt window

Continue the Windows Setup installation.

When choosing an installation type, select Custom. The drive will appear as a single area of unallocated space.

Select the unallocated space and click Next. Windows begins the installation.

  1. On a Windows 10 device, search for “Feedback Hub” in Cortana search, then launch the app.
  2. Navigate to Feedback in the left menu, then press + Add new feedback.
  3. Select the Problem, and share any details you think are relevant, and choose an appropriate category and subcategory.
  4. Important: If possible, reproduce the problem(s) after clicking Begin monitoring (or Start capture) near the end of the form; Stop monitoring when you’re done.
  5. Once you’ve completed the form and monitoring, click Submit.
  6. Click Continue using Feedback Hub.
  7. Click My feedback at the top, then find the item you have submitted (it may take a while to appear).
  8. Open the item you submitted, then click Share.
  9. Click Copy link, then paste the link here in your response.
