a. user say:

If you can use VBA, the following macro may do it for you, but I have not done much testing, so proceed with caution, e.g.

 a. save and close all your documents

 b. make a copy of your large document

 c. reopen the document and run the macro.

 

Graham Mayor has an article about installing and running macros at http://www.gmayor.com/installing_macro.htm

This code is only intended for the specific scenario you describe, and relies on 

 a. the dates having the same format throughout (nn/nn/nn) 

 b. each date following a full paragraph mark (not just a hard line end, for example. 

Sub reverseparas()
Dim b As Bookmark
Dim al As WdAlertLevel
Dim finished As Boolean
Dim i As Integer
Dim r As Range
Dim source As Range
Dim target As Range
al = Application.DisplayAlerts
Application.DisplayAlerts = wdAlertsNone
On Error GoTo problem
Set r = ActiveDocument.Content
If Right(r.Text, 2) <> Chr(13) & Chr(13) Then
  r.InsertParagraphAfter
End If
Set b = r.Bookmarks.Add("source", ActiveDocument.Range(0, r.End - 1))
Set target = ActiveDocument.Range(r.End - 1, r.End - 1)
finished = False
Do Until finished
  Set r = b.Range
  With r.Find
    .MatchWildcards = False
    .Text = "^p^#^#/^#^#/^#^#"
    If .Execute Then
      Set source = ActiveDocument.Range(0, r.Start + 1)
    Else
      finished = True
    End If
    target.Select
    target.FormattedText = source.FormattedText
    target.Collapse direction:=wdCollapseStart
    source.Text = ""
  End With
Loop
problem:
Application.DisplayAlerts = al
Set r = Nothing
Set source = Nothing
Set target = Nothing
ActiveDocument.Bookmarks("source").Delete
End Sub

jay freedman say:
If the graphs are set to In Line With Text (instead of any of the other text wrapping settings), you can use the Keep With Next paragraph setting to keep each title and its graph together.

The best method is to define a style that you apply to the titles. In the dialog where you define that style, click the Format button and click Paragraph in the menu. In the Paragraph dialog that opens, click the Line & Page Breaks tab, and check the box for "Keep with next". You can also define the font and spacing you want for the style.

When you apply the style to a title, and put the graph in the following paragraph, those two paragraphs will always be on the same page.

If the graphs need to have text wrap around them, this idea won't work. Something that may help instead is to put the title and the graph together into a text box or a frame.
