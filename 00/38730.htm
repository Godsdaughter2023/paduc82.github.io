Is it possible to get back the data from a hard drive formatted - Microsoft Community

deli46000a say:

Is it possible to get back the data from a hard drive formatted several times?

My friend recently asks me to help him restore his data from an external hard drive. But, he says this drive has been re-formatted for several times anyway. Is it still possible to restore the contained files back?

I need you help! I need you help!

omniplex say: Simplified a hard drive can be formatted as "superfloppy" (one volume) or consist of one or more partitions, each partition corresponding to a volume.   Disk management or the command line tool diskpart can create, delete, or modify partitions.  If your friend only rearranged partitions without actually formatting the corresponding volumes the old data on this drive might still exist.

For a given volume (incl. the superfloppy case) "format" means writing some file system structures depending on the file system (NTFS is different from FAT32, etc.) and cleaning the data area for the future files.  For big volumes that takes minutes, and it's not really necessary.  

A "quick format" only creates the file system structures, leaving the data area as it was.  In that case parts of the old files still exist in the preserved data area, but the info how these parts have to be arranged to recreate the old files is lost.  If the old files were fragmented (= stored in multiple parts) the chances to reconstruct anything are slim.  Otherwise (not fragmented) it might be possible to find some old files.

Check out TestDisk to find lost partitions, and PhotoRec to reconstruct old files -- as the name says folks often try to recover photos, and for some types of files, e.g., JPG, their known internal structure can allow to find the next piece (fragment.)                   

delores00s say:

Well! Have you saved anything new on this drive since then?


Nowadays, the formatting process, especially the quick formatting process, often could not erase the stored data completely. It is only marked as “erased” and wait for new data to overwritten itself. Therefore, as long as you’ve not overwritten them, you always have possibilities to get it back.


That’s to say, in this regard, it doesn’t matter with how many times you’ve reformatted. But, for the absolutely safety of inner data, you’d better not to do the reformatting things any longer.
