'Default gateway not available' - Regular connectivity issues.

Since moving into a new shared house I have been experiencing a connectivity issue that seems to also have affected lots of other people. When connected to my wireless router, I regularly see that I have limited connectivity to the network and no access to the internet. The issue is easily solved by disconnecting and reconnecting to the network, or by using the troubleshooter (which advises that that issue was that the default gateway was not available). Though it is easy to solve, it is still incredibly frustrating in that it happens every few minutes at the least.

Like many of the others with the same problem, I am running Windows 7. Specifically, it is Windows 7 Starter, on a Samsung N150 Plus netbook. 

The issue is most frequent when I am streaming or downloading content.

I have read elsewhere that it could be an issue caused by the combination of me running Windows 7 and that I am using an old wireless router (specifically iot is a Thomson TG585 v8). Unfortunately this choice is beyond my control, so I'm going to need to get Windows 7 and the old router talking.

I've tried disabling IPv6 and this didn't solve the issue. I've downloaded and installed the most latest broadcom updates for my netbook from Samsung's website and this also didn't solve the issue.

I attempted to manually set the default gateway though struggled to follow the instructions.

Any help on how I might fix this would be greatly appreciated. And if there is some common, easy fix that I'v somehow missed, please feel free to point me in the direction of it!
