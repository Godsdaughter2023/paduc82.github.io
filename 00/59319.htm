Hi Francis,

 

Have you disabled Prefetch and Superfetch?

This truly is a personal preference question.

SuperFetch and prefetch are storage management technologies in Windows that provide a fast-track access to data on traditional, slower hard drive

Example of a working of prefetch and Superfetch: When you are using the computer, the prefetch files are created for the programs you normally use most often. This allows these programs to load faster. The purpose of SuperFetch is to increase the performance of the PC during regular tasks performed by the user

Also, it is advised not to use third-party PC optmizers. However, if you are still using them, then please be very careful about the files that you remove.
