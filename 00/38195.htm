Welcome to the Microsoft Windows Vista Communities site.

 

 

Windows Update being reoffered…

 

I’ve provided you with some troubleshooting to clear a few registry values that may be causing this behavior. Before running proceeding with troubleshooting I strongly encourage you to create a System Restore point as you will need to modify the registry.

 

Registry Disclaimer: Modifying REGISTRY settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of REGISTRY settings can be solved. Modifications of these settings are at your own risk.

 

Clearing Required Reboot and Pending Rename Operations…

1.     Click Start, click All Programs, click Accessories, right-click Command Prompt, and select Run as Administrator.

2.     Click on Continue or provide Administrator credentials if prompted.

3.     Select all the below commands, right-click, and click Copy.

REG DELETE "HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\Session Manager" /v PendingFileRenameOperations /f

REG DELETE "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\Auto Update\RebootRequired" /f

REG DELETE "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsUpdate\Auto Update" /v RebootRequired /f

REG DELETE "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Updates\UpdateExeVolatile" /f

4.     Now right-click the Command Prompt window and select Paste.

5.     The commands should all complete. You may see one or two errors those are expected – they simply mean the registry value does not exist.

6.     Restart the computer, and see if the issue is resolved.

7.     If the issue persists proceed to the next group of troubleshooting.

 

How to clear the Windows Update cache…

1.     Click Start, click All Programs, click Accessories, right-click Command Prompt, and select Run as Administrator.

2.     Click Continue or provide Administrator credentials as needed.

3.     Select all the below commands, right-click, and click Copy.

net stop wuauserv

ren %windir%\SoftwareDistribution SoftwareDistribution.OLD

bitsadmin /RESET /ALLUSERS

net start wuauserv

exit

4.     Now right-click the Command Prompt window and select Paste.

5.     Now click Start, click All Programs, and then Windows Update.

6.     Then click Check for updates, and see if the issue is resolved.
