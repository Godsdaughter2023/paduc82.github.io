peer_sam say:

Hi Rishi,

 

Thank you for writing to Microsoft Community forums.

 

When you say that booting the computer from USB flash drive does not work, what happens when you try to boot the computer from it?

What is the make and model of the computer?

 

Bit locker can block you from booting the computer from the hard drive it is enabled on, but it won’t block you from booting the computer from an external bootable device like a flash drive.

 

Ensure that you change the boot priority in BIOS settings and set the USB flash drive as the first boot device to be able to boot the computer from it. Please refer to the computer’s manual to know how to access the BIOS.

Note: any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.
