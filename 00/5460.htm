Though you cannot change the on-screen keyboard shortcuts for the Ribbon, you can still create custom keyboard shortcuts for the menu commands themselves:

  1. In Word, click on the File tab.
  2. Click on Options.
  3. Click on the Customize Ribbon tab.
  4. Click on the Customize button beside Keyboard shortcuts:
  5. Check that Save changes in: is set to Normal.dotm.
  6. Find the command for which you want to change the shortcut and select it.
  7. Click on the Press new shortcut key: field
  8. Type the new keyboard shortcut.
  9. Click on the Assign button

Repeat steps 5 through 8 for each additional command. Click Close and OK out when you're done.
