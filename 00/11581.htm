How do I transpose data in a cell separated by a comma, example Last - Microsoft Community

siggiestillman say:

How do I transpose data in a cell separated by a comma, example Last Name, First name to First Name, Last Name?

How do I transpose data in a cell separated by a comma, example Last Name, First name to First Name, Last Name?

It's driving me crazy.

siggie
bobgreenblatt say: Assuming your names begin in A1, enter this formula in a spare cell to the right:
=MID(A1,SEARCH(",",A1)+2,256)&" "&LEFT(A1,SEARCH(",",A1)-1)

Fill this down for as many cells as needed. You can then copy that range and paste special-values to itself to eliminate the formula. And, if necessary then paste that column back into A to replace the original.

Bear in mind that this formula assumes "lastname, firstname" and may not properly handle middle names and/or Jr/Sr etc.
