rich007a say:

Try this:

Sub ToBulletOrNotToBullet()
    Dim para As Paragraph, i As Long
    For Each para In ActiveDocument.Paragraphs
        i = i + 1
        If para.Range.ListFormat.ListType = wdListBullet Then
            MsgBox "Paragraph " & i & " is in a bulleted list and is at Level " _
                & para.Range.ListFormat.ListLevelNumber
        End If
    Next para
End Sub

I think it might do what you need.

Microsoft deliberate discourages deletion of recoverable deleted items - Microsoft Community

a. user say:

Microsoft deliberate discourages deletion of recoverable deleted items in Outlook?

When accessing Outlook e-mail through a browser, there is a ”Recover deleted items” link in the trash which opens a pop-up displaying deleted e-mail, contact and calendar information. Microsoft seems to have deliberately made it difficult to select multiple or all recoverable deleted items and permanently delete (”purge”) them.

When moving the mouse over an item a checkbox appears with delay to the left of the single item. By displaying a single checkbox at a time and with delay, Microsoft makes the manual selection of multiple items slower than for example when selecting e-mails to be deleted in the Outlook inbox.

On the other hand, there is no option available to select all to be able to purge all items faster or a ”purge all” button. In account settings, there is also no choice to disable the ”Recover deleted items” option. The user is also not informed clearly that an item which has been moved to trash and manually deleted from trash has not been deleted ”permanently” in the sense of being unrecoverable to the user.


A select-all button which exists in other parts of Outlook, a purge-all button in the window or a "disable deleted item recovery" option under account settings are examples of ways to allow the Outlook user to "permanently" (unrecoverable to the user) delete e-mail in a way that is as easy as regular-delete. The lack of these features suggests Microsoft wishes these items to remain recoverable to the user. Also, not informing the user about the ability to recover deleted items is an issue for privacy-concious users who see this feature not as a way to undo mistaken deletions but as a way for Microsoft to keep more information about them for longer.
