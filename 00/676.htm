Bought a yearly subscription within the last 30 days, OR.

Bought a monthly subscription, and cancel within 30 days of your last renewal date.

a. user say:

Hi Paul,

Welcome to Microsoft Community.

This issue might be due to corrupted driver.

Try to uninstall the network adapter driver, and then restart your computer and have Windows automatically install the latest driver. Consider this approach if your network connection stopped working properly after a recent update.

Before uninstalling, make sure you have drivers available as a backup. Visit the PC manufacturer’s website and download the latest network adapter driver from there. If your PC can't connect to the Internet, you'll need to download a driver on a different PC and save it to a USB flash drive, so you can install the driver on your PC. You’ll need to know the PC manufacturer and model name or number.

1.       In the search box on the taskbar, type Device Manager, and then select Device Manager from the list of results.

2.       In Device Manager, select Network adapters > the network adapter name.

3.       Right-click (or press and hold) the network adapter, and then select Uninstall > Delete the driver software for this device check box > OK to verify you want to uninstall it.

4.       After uninstalling the driver, select the Start button > Power > Restart.

After your PC restarts, Windows will automatically look for and install the network adapter driver. Check to see if that fixes your connection problem. If Windows doesn't automatically install a driver, try to install the backup driver you saved before uninstalling.

Windows Defender Error Code: 0x80073afc - Microsoft Community

stephen boots say:

Windows Defender Error Code: 0x80073afc

Technical Level : Intermediate

Summary

In recent weeks there have been a high number of reports of encountering error 0x80073afc when trying to run Windows Defender on Windows 8.x.

Some users have reported that a System Restore resolved the problem. Others reported that it did not help.

Some users also reported that a System File Checker scan resolved the problem, while others reported that it also did not help.

This article is a copy of a post from Microsoft Malware Protection Center to this thread on 22 June, 2015.

If you do not wish to troubleshoot the problem, you may wish to consider --

Restore, reset, refresh Windows 8: http://windows.microsoft.com/en-US/windows-8/restore-refresh-reset-pc -- this may actually be the quickest solution if you don't have lots of installed programs to reinstall.


Details

We’re looking into this error code.

To help us address this issue, you can submit your diagnostic logs through our malware submission portal: 

In Windows 7 or Windows Vista, if you are using Microsoft Security Essentials

  1. Open an elevated command prompt:
    a. Click the Start button.
    b. Click on All Programs, then click on Accessories.
    c. Right-click on Command Prompt.
    d. Click Run as administrator.
    e. Click Yes when asked if you want to run this program.
  2. Type cd "\Program Files\Microsoft Security Client" and press Enter.
  3. Type MpCmdRun.exe -GetFiles and press Enter.
  4. Submit the MpSupportFiles.cab file through our malware submission portal:
    a. On the top right of the submission page, click Sign in and sign in with your Microsoft account. This means we can contact you if we have any further questions or information for you.
    b. On the submission page, click Browse… and find the MpSupportFiles.cab file as described at the bottom of the command prompt window. Click OK to attach the file to the submission form.
    c. Enter the following text in the comments box “Diagnostic logs for Error Code 0x80073afc”.
    d. In the Product menu, click Microsoft Security Essentials.
    e. Ensure all other fields are complete and then click Submit sample.

In Windows 8.1, if you are using Windows Defender

  1. Open an elevated command prompt:
    a. From the Start screen type “Command Prompt”.
    b. Right-click on Command Prompt when it displays in the search results.
    c. Click Run as administrator.
    d. Click Yes when asked if you want to run this program.
  2. Type cd “\Program Files\Windows Defender” and press Enter.
  3. Type MpCmdRun.exe -GetFiles and press Enter.
  4. Submit the MpSupportFiles.cab file through our malware submission portal
    a. On the top right of the submission page, click Sign in and sign in with your Microsoft account. This means we can contact you if we have any further questions or information for you.
    b. On the submission page, click Browse… and find the MpSupportFiles.cab file as described at the bottom of the command prompt window. Click OK to attach the file to the submission form.
    c. Enter the following text in the comments box “Diagnostic logs for Error Code 0x80073afc”.
    d. In the Product menu, click Windows Defender.
    e. Ensure all other fields are complete and then click Submit sample.

Thanks, MMPC

-----------------------------------------------------

To locate (Browse for) the MpSupportFiles.cab files requested for submission the paths are as follows:

Windows 7 and Vista: C:\ProgramData\Microsoft\Microsoft Antimalware\Support\MpSupportFiles.cab

 

Windows 8/8.1: C:\ProgramData\Microsoft\Windows Defender\Support\MpSupportFiles.cab

Please note that in order to access these files the folder options/view will need to be changed to 'Show hidden Files" and "Unhide file extensions" before Browsing for the files.

For Windows 7 and Vista users click on Organize in the upper left corner of menu and then select Folder and search options.

Next click on View tab and under Hidden files and folders put a dot in the circle next to Show hidden files, folders and drives. and uncheck the box next to Hide extensions for known file types.

For Windows 8 users click on View in menu and put check-marks in the boxes next to File name extensions and Hidden items.

How to add hyperlinks to drop down list items? - Microsoft Community

bern_dv say:

How to add hyperlinks to drop down list items?

I have a drop down list created using data validation. I want the entries in the list to be hyperlinked. How can I do this?

 

Thanks.

bill manville say:

You can't do it directly.
Assuming that your list is fed from a range of cells, you could make those cells hyperlinked and you could have a Worksheet_Change event procedure for the worksheet which detects that the relevant validated cell has been changed; it would then find the entry in the list and use the FollowHyperlink method.

Here is an example, in which it is assumed the cell is named ValCell and the list is named ValList.
The code goes in the module behind the worksheet on which ValCell is situated:

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim vMatch
  Dim H As Hyperlink
' test to see if the cell named ValCell has been changed
  If Intersect(Target, Me.Range("ValCell")) Is Nothing Then Exit Sub
  If IsEmpty(Me.Range("ValCell")) Then Exit Sub
' find it in the validation list (named ValList)
  vMatch = Application.Match(Me.Range("ValCell"), Me.Range("ValList"), 0)
  If IsError(vMatch) Then Exit Sub  ' not on the list
  Set H = Me.Range("ValList").Cells(vMatch).Hyperlinks(1)
  ThisWorkbook.FollowHyperlink H.Address & IIf(H.SubAddress <> "", "#" & H.SubAddress, "")
End Sub
