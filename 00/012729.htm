Hi Vassilis

Yes this can be fixed with a registry edit as below:

For Office 2016

 

Windows Registry Editor Version 5.00

 

[HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Outlook\Security]

"EnableRoamingFolderHomepages"=dword:00000001

 

[HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Outlook\Security]

"NonDefaultStoreScript"=dword:00000001

 

For Office 2013

 

Windows Registry Editor Version 5.00

 

[HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Outlook\Security]

"EnableRoamingFolderHomepages"=dword:00000001

 

[HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Outlook\Security]

"NonDefaultStoreScript"=dword:00000001

If you want to put them in the policies folder you can using the below:

For Office 2016

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Policies\Microsoft\office\16.0\outlook\Security]
"EnableRoamingFolderHomepages"=dword:00000001
For Office 2013 
Windows Registry Editor Version 5.00
[HKEY_CURRENT_USER\Software\Policies\Microsoft\office\15.0\outlook\Security]

VBA code to export all tables in an Access 2007 db as .csv or .xls - Microsoft Community

larana626 say:

VBA code to export all tables in an Access 2007 db as .csv or .xls

I need vba code to build a module that will export all tables in an Access 2007 db as .csv or .xls; depending on the user's choice.

The code will need to ask the user for a destination folder.

Thank you,

hansv mvp say:

Try this:

Sub ExportExcelCSV()
  Dim strOut As String
  Dim tbl As AccessObject
  Dim f As Boolean

  With Application.FileDialog(4) ' msoFileDialogFolderPicker
    .Title = "Please select the target folder"
    If .Show Then
      strOut = .SelectedItems(1)
      If Not Right(strOut, 1) = "\" Then
        strOut = strOut & "\"
      End If
    Else
      MsgBox "You didn't select a target folder.", vbExclamation
      Exit Sub
    End If
  End With
  f = (MsgBox("Do you want to export all tables to Excel (No = CSV)?", _
    vbQuestion + vbYesNo) = vbYes)

  For Each tbl In CurrentData.AllTables
    If Not tbl.Name Like "MSys*" And Not tbl.Name Like "~" Then
      If f Then
        DoCmd.TransferSpreadsheet acExport, acSpreadsheetTypeExcel12Xml, _
          tbl.Name, strOut & tbl.Name & ".xlsx", True
      Else
        DoCmd.TransferText acExportDelim, , _
          tbl.Name, strOut & tbl.Name & ".csv", True
      End If
    End If
  Next tbl
End Sub
