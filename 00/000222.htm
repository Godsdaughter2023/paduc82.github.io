AFAIK there is no simple way, because Powerpoint opens Excel like this:

Set xlApp = CreateObject("Excel.Application")

And such an instance did not load anything automatically.

But if we use the macro to initiate the process, we can load the AddIns by ourself.

Andreas.

Sub EditChartData()
  'Issue:
  '  My Excel-AddIn won't load when right click on a PowerPoint chart and select "edit data"
  Dim Sh As Shape
  Dim xlApp As Object 'Excel.Application
  Dim xlAdd As Object 'Excel.AddIn
  'Be sure there is a selection
  If ActiveWindow.Selection.Type = ppSelectionNone Then GoTo ErrorOut
  'Find the chart
  For Each Sh In ActiveWindow.Selection.ShapeRange
    If Sh.Type = msoChart Then
      With Sh.Chart.ChartData
        'Open Excel
        .Activate
        'Refer to the application
        Set xlApp = .Workbook.Parent
        'Look for installed addins
        For Each xlAdd In xlApp.AddIns
          If xlAdd.Installed Then
            'Load it
            xlApp.Workbooks.Open xlAdd.FullName
          End If
        Next
        'Done
        Exit Sub
      End With
    End If
  Next
ErrorOut:
  MsgBox "Please select a chart and try again", vbExclamation, "EditChartData"
End Sub

imran chand say:

Hi Lars 45,
To Disable or Turn Off ‘Do You Want To View Only the Webpage Content that Was Delivered Securely Security Warning’, you need enable ‘Display mixed content’ option in Internet Options.

Mixed content situation can happen on HTTPS secure websites such as online stores or financial sites that display images, banners, or scripts that are coming from a server that is not secured. IE has defaulted to prompt user for next course of action when encounter mixed content as displaying mixed content carries risk that non-secure web page or script will be able to access information from the secure content.

Having to click “No” (or “Yes” to block non-secure content) button for every visit to website with mixed content can be troublesome and annoying. Besides, its possible user accidentally clicks “Yes”, which prevents some insecure elements from loading and displaying. For user who confident that he or she does not visit possibly malicious site, it’s possible to instruct Internet Explorer to display mixed content automatically, instead of asking for input. 

Follow the steps below to enable ‘Display mixed content’ option:
a.    Launch Internet Explorer.
b.    Going to Tools->Internet Options->Security.
c.    Select the ‘Security’ tab.
d.    Click the ‘Custom Level’ button.
e.    In the ‘Miscellaneous’ section change “Display mixed content” to Enable

For detailed information, refer to the suggestions provided byvinod Sundarraj on Wednesday, May 20, 2009 2:14 AM in the below link:
