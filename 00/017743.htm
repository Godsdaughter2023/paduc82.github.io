shyam pillai say:

Hi,

Here is the vba code to add a bookmark trigger from my page: http://skp.mvps.org/2010/ppt002.htm

Sub AddBookmarkTrigger()
Dim oShp As Shape         ' Shape with animation that needs to be triggered via bookmark
Dim oShpMovie As Shape    ' Media shape

'Assumes that we have a media shape with 'Bookmark 1' on slide 1 of the presentation
With ActivePresentation.Slides(1)
        Set oShpMovie = .Shapes(1)
        Set oShp = .Shapes(2)
        ' Assign an trigger to shape
        ' if no bookmark name is specified, it will default to 1st. If no bookmark exists then it fails
        Call .TimeLine.InteractiveSequences.Add.AddTriggerEffect(oShp, msoAnimEffectPathCircle, _  
                                msoAnimTriggerOnMediaBookmark, oShpMovie, "Bookmark 1")
End With

End Sub

Decimal Places formatting

-All decimal places should show out to however many places were typed in and nothing more. (0.5314 or 0.11 or 0.123)

-Numbers greater than 1 should only ever show 2 decimal places (1,000.004 displays as 1,000.00)

hansv mvp say:

Set the number format to Number with 2 decimal places using the thousands separator.

Select Conditional Formatting > New Rule...

Select 'Format only cells that contain'.

Leave the first dropdown set to 'Cell Value'.

Select 'less than' from the second dropdown., and enter 1 in the box next to it.

Click Format...

Activate the Number tab.

Select Custom in the list of categories.

Enter 0.############## in the Type box.

Click OK, then OK again.

catherinehardy say:

I see these options in Excel, but I'm trying to format my forms and reports in Access. Any help on how to go about that?

Thanks!

hansv mvp say:

My apologies, I didn't look carefully enough. You cannot set the number format through conditional formatting in Access. You'll need VBA code in a form bound to the table. For a control named Text0:

Private Sub Form_Current()
    FormatControl Me.Text0
End Sub
 
Private Sub Text0_AfterUpdate()
    FormatControl Me.Text0
End Sub
 

And in a standard module:

Sub FormatControl(ctl As Control)
    If ctl = 0 Then
        ctl.Format = "0"
    ElseIf ctl > 0 And ctl < 1 Then
        ctl.Format = "0." & String(Len(CStr(ctl)) - 2, "0")
    ElseIf ctl > -1 And ctl < 0 Then
        ctl.Format = "0." & String(Len(CStr(ctl)) - 3, "0")
    Else
        ctl.Format = "#,##0.00"
    End If
End Sub

This will only work in a single form, not in a continuous form.

For a report, you'd need to use the On Format event of the section containing the control. For example, in the Detail section:

Private Sub Detail_Format(Cancel As Integer, FormatCount As Integer)
    FormatControl Me.Text0
End Sub
