I've seen this issue with Default UAC permissions, so that's not the issue.

The problem is the window is hidden so you can't see it.  To see what is prompting for Admin access, click Ctrl+Alt+Del > Task Manager.  Click Fewer Details (if shown).  Right-click the app (should be the only one if no other user apps are running) and click Properties.  Verify it's a trusted application in a trusted location (e.g. a Microsoft executable in c:\windows\...).  Then right-click the process again and click "Switch To".  This will force the hidden window that is prompting for admin access to be displayed.  If you verified it's a trusted application from a trusted location (above) and the publisher of the app prompting for admin permissions is someone you trust (e.g. Microsoft), then it's probably best to click OK on the elevation of privileges since it's probably a windows update or something making changes to your system.  However, If you don't recognize the app, don't click OK, as it could be malware, etc.  In that case, I would recommend downloading the Microsoft Malicious Software Removal Too to make sure your system doesn't have a virus or malware:
