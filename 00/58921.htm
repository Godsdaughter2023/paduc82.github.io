Hi,



Streaming issues on Windows 10, may occur due to several reasons. If you have a poor network connection or an older wireless network card driver, it can initiate scans of wireless networks in the background. Some, but not all, wireless cards can be asked not to scan as frequently.


For Intel Wi-Fi devices:


  1. Right-click on the Start icon.
  2. Select Device Manager.
  3. Expand Network Adapters.
  4. Right-click on Intel <name of wireless card>.
  5. Select Properties from the menu.
  6. Select the Advanced tab.
  7. Select Roaming Aggressiveness from the Properties list.
  8. Change value to 1. Lowest from the drop-down list.


This will put the Intel driver in the lowest scanning mode.
