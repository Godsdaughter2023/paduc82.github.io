jllatham say:

I'm not at all certain that a reasonable formula can do this - I think you'd need many sub formulas strung end to end for as many possible duplications of a part number as you could ever have.  That means a lot of work done, quite often probably to return nothing.

 

Realizing you are averse to a VBA solution, I've prepared one anyway.  You can elect not to use it if you like and wait until someone wiser than me [Edit: I think his name may very well be Bernie :-)] comes up with a formula to do this.

 

This is a User Defined Function (UDF).  That means you can use it on a worksheet just like a built-in Excel function such as SUM() or AVERAGE() or COUNTIF().  You would use it in this fashion: in cell C2 put this formula:

=JoinNarr(A2)

where A2 is the cell with the part number.  Fill down.  Only the first instance of a part number will then have all the narrative pulled together into the cell.  It could be changed to do it for all of them, but I figured the clutter would be simply awful, with numerous duplicates.  This way you can filter for non "" cells and copy and paste to a new sheet.

 

See this web page for instructions on how to put this code into a REGULAR CODE MODULE
http://www.contextures.com/xlvba01.html#videoreg

After adding the code to the workbook, be sure you save it as a 'Macro Enabled..." file, type .xlsm or .xlsb

The function's code:

 

Function JoinNarr(partNo As Range) As String
'see this web page for instructions on how to put this code
'into a REGULAR CODE MODULE
' http://www.contextures.com/xlvba01.html#videoreg
'
'To use the function on a worksheet, simply type it in
'like a regular formula and reference the cell with the
'part number in it, as:
'   =JoinNarr(A2)
'where A2 has the part number, and column B has text to combine
'
  Dim pnList As Range
  Dim anyPN As Range
  Dim tempText As String
 
  tempText = Left(partNo.Address, InStr(2, partNo.Address, "$") - 1)
  Set pnList = Range(partNo.Address & ":" & _
   Cells(Rows.Count, partNo.Column).End(xlUp).Address)
  'display results for first occurrance of the part number
  If Application.WorksheetFunction.CountIf(Range(tempText & 1 & ":" _
   & tempText & partNo.Row), partNo.Value) = 1 Then
    tempText = ""
    For Each anyPN In pnList
      If anyPN = partNo Then
        tempText = tempText & Trim(anyPN.Offset(0, 1)) & " | "
      End If
    Next
  Else
    JoinNarr = ""
    Exit Function
  End If
  JoinNarr = Left(tempText, Len(tempText) - 3)
End Function
