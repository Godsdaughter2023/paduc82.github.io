don cuthbert say:

The way to resolve this depends on the way your disks are initialized:  GPT or MBR.  With an MBR disk you can remove the Active flag on the old HDD.  With a GPT disk you could try formatting the HDD's System partition (which is a FAT32 partition on GPT disks).  I had trouble sorting this out when I installed an SSD and chose to delete the hard disk's System partition, so that it would not appear as a UEFI boot option.

Launch Disk Management to see what the partition layout of your 2 disks is.  On a UEFI BIOS mode computer, it is best to use GPT disks.  GPT disks have a separate System partition (FAT32 format) so look for that.

If your HDD is MBR, you can just remove the Active designation so that it can't boot from it.

Outlook re-downloading entire email server contents - Microsoft Community

tholtx say:

Outlook re-downloading entire email server contents

I have Outlook 2011 14.0.0, have downloaded current update.  Use a MacBook Pro, OSX 10.7.2 Lion.  Sometimes when I ask Oulook for new email messages it starts downloading the entire contents of the server including scores of messages already downloaded. Entourage 2004 used to do this too. What causes this and can it be fixed?  
dianeoforegon say: Updaters need to be installed in order. You need the following updaters:

14.1.0
14.1.4 (This update includes all the improvements that were released in all previous Office 2011 updates since the release of the Microsoft Office for Mac 2011 14.1 Update)

Follow directions here to update:

How To Install, Reinstall and Update Office 2011

I suspect you have a POP account.


When your mail server receives a new message it gives the message a
unique ID. This is what mail applications use to determine whether or
not to download messages. New IDs get downloaded. Existing IDs don't.

When mail servers are rebooted or reset in some way they often reset the
unique IDs. The next time the mail client connects it sees that it has a
lot of "new" messages and downloads them again.

If you downloaded the messages in Entourage and now want to download into Outlook, Outlook will download everything left on the server. Outlook has no way to know what messages Entourage has downloaded.

You can work through this a couple of different ways:

1. Edit your POP account settings to delete messages that are older than
a certain length of time. I use five days. If the server is reset then I
only receive the last five days worth of messages again.

2. Convert to an IMAP account instead of POP. IMAP is made to
synchronize with your mail server unlike POP, which downloads and deletes.

Trying to use your POP server as an IMAP server by leaving messages on
your server is not a good solution.
