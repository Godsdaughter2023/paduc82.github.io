Hi,

To simply return a Boolean you can do this. TRUE is spelt correctly

Sub spell_me()
Dim r As Boolean
r = Application.CheckSpelling(Word:=ActiveCell.Text)
MsgBox r
End Sub

To colour the cells we can negate the TRUE and FALSE with NOT and do this

Sub spell_me2()
Dim r As Boolean
If Not Application.CheckSpelling(Word:=ActiveCell.Text) Then
ActiveCell.Interior.Color = vbYellow
End If
End Sub

jllatham say:

Another victory for the Good Guys!!  None of that stuff seems to be actually documented in the on-line help, or is very well camouflaged.  But then, 'Help' seems to have become a forgotten word since they put it on-line.  The reference I found said "Applies To" with Excel 2003 only listed.

Thanks, Mike H. for teaching me something new today.

We could even take it a step further by highlighting the individual words in a phrase that are incorrect in a cell like this (so easy when someone has shown you how).  Only problem I see is making sure we maintain original font coloring, which could be dealt with using a little more code.

Sub SpellingHive()
  Dim response As Boolean
  Dim words As Variant
  Dim wordCount As Long
  Dim startAt As Long
 
  words = Split(ActiveCell.Text, " ")
  'set all of the text to automatic color
  ActiveCell.Font.ColorIndex = xlAutomatic
  For wordCount = LBound(words) To UBound(words)
    response = Application.CheckSpelling(word:=words(wordCount))
    If Not response Then
      'find out where it is in the text and color the font red
      startAt = InStr(ActiveCell.Text & " ", words(wordCount) & " ")
      ActiveCell.Characters(Start:=startAt, Length:=Len(words(wordCount))).Font.Color = vbRed
    End If
  Next
End Sub
