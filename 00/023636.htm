EUREKA!  bhringer, I think I've found the answer!  While visiting a web site, "tenforums.com", I ran across a post which suggested a Registry modification.  So, using RegEdit, I visited the portion of the Registry concerned (HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer).  The Key concerned is, (SmartScreenEnabled=Warn).  In that Key, it was in fact set to WARN.

I then changed that Key from WARN to OFF (thinking that at worst, I would only turn SmartScreen OFF).  Sure enough, when I rebooted the computer and checked "Windows Defender Security Center > App & browser control > Check apps and files", it was then set to OFF!

Using RegEdit, I went back to (HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer), changed the (SmartScreenEnabled=Off) Key to WARN, rebooted the computer, and voila!  When I re-opened "Windows Defender Security Center > App & browser control > Check apps and files," it was then set to WARN!

I rebooted the computer again to confirm the setting did not change, and it didn't--problem fixed!

I hope that action didn't affect anything else...  I suppose whatever the issue may have been--IF it concerns UAC--the underlying issue may still exist.  I guess I'll find out eventually (and probably at the worst possible time LOL).  Anyway, I wanted to thank you again though, for your help.  Thanks again!
