e.j. gun say:

The answer you received in your previous post (here:  Previous Post) should work quite well, and contrary to your response, it is not dependent on the version of Excel.

Let's say the cells you want to monitor are A1:A3.  In B1:B3, insert a circle using the Webdings font by entering a lowercase "n" in each cell, and changing the font to Webdings.

Now set up two conditional formatting rules in cells B1:B3:

  • Select B1:B3
  • On the Home tab, select Conditional Formatting, then New Rule.
  • Select "Use a formula to determine which cells to format"
  • In the formula box, enter =(A1<>"")
  • Click Format..., then Font.  Change the font color to Green.
  • Repeat to add another formula, but this time use =(A1="") and set the font color to Red.

Now, when A1:A3 have nothing in them, B1:B3 show a red stoplight.  When you enter anything into one of those cells, the stoplight changes to Green.

What is it about that solution that doesn't work for you?  I did this with Excel 2010.
