You should delete the value named:

{754AC886-DF64-4CBA-86B5-F7FBF4FBCEF5}

That is pointing the Desktop shortcut under This PC to your Downloads folder.

After that, log off & log back on.

If you still see duplicate names & icons, even though the actual folder contents are different, then you need to edit the desktop.ini file:

Open PowerShell by right-clicking on Start & selecting Windows PowerShell

Type the following commands:

cd desktop

The file will open in Notepad. Edit the contents to match the text below. then save and close Notepad.

[.ShellClassInfo]
LocalizedResourceName=@%SystemRoot%\system32\shell32.dll,-21769
IconResource=%SystemRoot%\system32\imageres.dll,-183

Return to PowerShell and type the following:

attrib +h +sdesktop.ini

Log off and log back on. Things should be OK now...

-Make sure that the Windows Firewall is enabled. No other firewall truly offers better protection or is needed. This one is perfectly fine.

 

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

 

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

 

-Never open unsolicited email attachments.

 

-Practice safe web browsing.
