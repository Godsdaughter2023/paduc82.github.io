ken sheridan say:
SELECT T1.Section, T1.Area, T1.Date, T1.HoursBS, NZ(T1.HoursBS -
          (SELECT HoursBS
            FROM YourTable AS T2
            WHERE T2.Area = T1.Area
                 AND T2.Date =
                (SELECT MAX(T3.Date)
                 FROM YourTable AS T3
                 WHERE T3.Area = T1.Area
                 AND T3.Date < T1.Date)),0) AS DELTA
FROM YourTable AS T1
ORDER BY T1.Area, T1.Date;

I would advise against the use of Date as a column name, however.  As the name of a built in function in Access it is a reserved keyword, and should not be used as an object name.

It can also be done with a single subquery using the non-standard TOP predicate if portability of the query is not important:

SELECT T1.Section, T1.Area, T1.Date, T1.HoursBS, NZ(T1.HoursBS -
        (SELECT  TOP 1 HoursBS
          FROM YourTable AS T2
          WHERE T2.Area = T1.Area
          AND T2.Date < T1.Date
          ORDER BY T2.Date DESC),0) AS DELTA
FROM YourTable AS T1
ORDER BY T1.Area, T1.Date;
