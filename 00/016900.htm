Corruption in the History folder is a common cause of this. Clearing the History should fix it.

From Internet Options> General > Browsing History> Delete (next screen) *uncheck Preserve Favorites website data* then check and delete History and Temporary Internet Files.

hansv mvp say:

You could create a macro in your personal macro workbook Personal.xlsb and assign it to a keyboard shortcut, a Quick Access Toolbar button or a custom ribbon button.

The macro could look like this:

 

Sub PasteNoWrap()
    ActiveSheet.Paste
    Selection.WrapText = False
End Sub

 

(preserves formatting), or

 

Sub PasteNoWrap()
    ActiveSheet.PasteSpecial Format:="Unicode Text"
End Sub

 

(doesn't preserve formatting)

    Set OutApp = Nothing

graham mayor say:

As others have indicated, the reason is that you have inserted a date field which shows the system date. What you need to do is change those date fields for createdate fields - ALT+F9 change { DATE } or { TIME \@ "d MMM yyyy" } to { CREATEDATE \@ "d MMM yyyy" } then F9 and ALT+F9 - and change the date in your letterhead template so that future letters based on it show the correct dates. The switch  \@ "d MMM yyyy" may be different at your location but you will have to add a formatting switch if your original field was a DATE rather than a TIME field.
