michael bauer (vboffice.net) say:

As the colors are stored per mailbox you need to set up the same master category list on every computer.

You could do it completely manually. Or deploy the category names via GPC and assign only the colors manually. 

The Category-Manager Addin allows to share the master category list with colors.

michael bauer (vboffice.net) say:

You don't edit colors, but category names. You add names to the master category list (mcl), and you can assign a color to each category name. While you can add as many categories as you want, there's only 25 colors available.

When you assign a category to a calendar item, you write only the category name into the item. If another user looks at the item, Outlook searches the category name in the user's own mcl. If found, it can display the color.

Using PERCENTILE.EXC function with a changing range of cells

If I'm using the PERCENTILE.EXC function in a worksheet and I am routinely changing the range of cells that it is using in the calculation, then how can I use a formula to automatically adjust the range of cells that it is using?
vijay a. verma say:

Apart from INDIRECT, another approach is to use OFFSET.

Say your data is in column C

In this case, your formula becomes where 0.25 can be replaced by your %.

=PERCENTILE.EXC(OFFSET(C1,0,0,COUNTA(C:C),1),0.25)

This is a dynamic formula. Say you have value from C1 to C20. It will automatically become C1:C20. If it is from C1 to C100, it will automatically become C1:C100.

Replace C1 and COUNTA(C:C) appropriately with your start point and column.
