suzanne s. barnhill say:

Word intends "Book fold" documents to be printed in their entirety or not at all. From Word 2002 (the first version that included this option), results of attempting to print (or reprint) specific pages have been unsatisfactory. There are (at least) two ways to get around the problem:

  1. Change the "Multiple pages" setting from "Book fold" to "2 pages per sheet." This will not change the layout of your document at all; it will look just the same on the screen. The only difference is that it would print in page number order instead of the order to create a folded booklet (for more information, see http://wordfaqs.mvps.org/2PagesPerSheet.htm). Then, on the Print tab of the Backstage, under Print All Pages, select Print Custom Range and enter either 1,4 (if that's what you really want) or 4,1 if you want the pages the way they would appear in a booklet.
  2. Save the document as a PDF and print it from Adobe Reader, which permits printing specific sheets (in this case you will get 4 on the left and 1 on the right).

hansv mvp say:

A list box doesn't have AutoComplete. It works as follows instead:

If you type a letter, the first item beginning with that letter is selected.

If you type the same letter again, the next item beginning with that letter is selected, etc.

When the last item beginning with the letter is reached, you'll jump back to the first one.

If you type a different letter, the first item beginning with that letter is selected.
