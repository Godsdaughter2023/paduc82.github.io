This macro will replace both cells that are formatted as italics and individual words within a cell that are formatted as italic. An added side effect is that the italics (and any other formatting) is stripped. This may or may not be desirable.

Tap Alt+F11 and when the VBE opens, immediately use hte pull-down menus to Insert, Module. Paste the following into the new pane titled something like Book1 - Module1 (Code), 

Sub mcrMake_HTML_Italics_Code()
    Dim c As Range, i As Long, str As String, bI As Boolean
    For Each c In ActiveSheet.UsedRange.SpecialCells(xlConstants)
        If c.Font.Italic Then
            c = "<i>" & c.Value & "</i>"
            c.Font.Italic = False
        Else
            bI = False
            str = vbNullString
            For i = 1 To Len(c.Value)
                If c.Characters(Start:=i, Length:=1).Font.FontStyle = "Italic" Then
                    If Not bI Then
                        str = str & "<i>"
                        bI = Not bI
                    End If
                    str = str & Mid(c, i, 1)
                Else
                    If bI Then
                        str = str & "</i>"
                        bI = Not bI
                    End If
                    str = str & Mid(c, i, 1)
                End If
            Next i
            If bI Then str = str & "</i>"
            c = str
        End If
    Next c
End Sub

With that pasted, tap Alt+Q to return to your worksheet. Tap Alt+F8 to open the Macros dialog and Run the macro.

If they're being received in the Junk folder, then the junk filter is working correctly.  Why would it bother you if things are working the way they are designed to work?  The whole purpose of the junk filter is to keep the junk out of your Inbox, not out of all folders in the mailbox.  The whole point of the Junk folder is so the filter has a place to put what it considers spam so it doesn't end up in the Inbox.  You then have the freedom of either just emptying the Junk folder or of reviewing the subject lines of the contents to verify that all the messages are junk before purging them, allowing you to salvage messages that were incorrectly identified.

bob jones aka: cybertaz say:

I'd suggest that you first troubleshoot a bit further by doing each of the following tests;

  1. Hold Shift while you launch Word from the Applications/Microsoft Office 2011 folder or the Dock. See if the problem persists. If it does, the issue is most likely related to Normal.dotm or Word's Preferences file.
  2. Create a new Standard User Account. Log in as that User & see how Word behaves. If the problem doesn't occur the cause is specific to your current User Account but not directly attributable to Word, itself.

Starting with OS X 10.7 Apple decided that the Library folder for User Accounts should be Hidden.

Once you log in as the original User & Quit all Office programs, go to your Desktop, open Finder's Go menu & press the Option key. The menu will expand to display Library directly below Home. From there go into the Preferences folder then delete the file named: com.microsoft.Word.plist
