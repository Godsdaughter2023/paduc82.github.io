john korchok say:

You can create an animation where you stack still frames and then make them appear. I'd recommend using smallish graphics, both in dimensions and file size, so that PowerPoint can manipulate them rapidly enough.

  1. Stack the graphics on a slide with the beginning image at the bottom.
  2. Open Home>Select>Selection Pane... This pane allows you to select graphics even when they're not directly visible.
  3. Start with the bottom most image and give it an Appear animation.
  4. Set the Start to On Click, if you would like to start the effect manually. This will apply only to the first image.
  5. Select the image second from bottom. Give it an Appear animation, with Start set to After Previous.
  6. Repeat these for all the rest of the images.
  7. Preview to see the effect, then adjust the Timing of each animation if necessary, adding a small amount to the delay if the animations run too quickly. 0.03333 seconds is the equivalent of 30 frames per second, 0.06666 is 15 f.p.s.

IMHO, Charles is muddying the waters!

PUT THE CREATEDATE FIELD IN THE HEADER!  Then when you create a document from the template, the selection will be located in the body of the document, where you want to start typing and what you type will have no effect on the CreateDate field!

If when you change the spacing after a paragraph, it affects all paragraphs in the document, see the article "Formatting applied to one paragraph affects the entire document” at:

http://www.word.mvps.org/FAQs/Formatting/WholeDocumentReformatted.htm

Of course, if you wanted to put the CreateDate field in the body of the document and have a bookmark somewhere and use a macro to make the selection go to that place whenever a new document was created, you could save the template as macro enabled template (dotm) and have the macro in an AutoNew macro in that template.

Method: 1

Open task manager and check if multiple instance of Outlook is running at the back ground. If found, end the process by following the steps below and then try to open and close Outlook and check if it closes fine:

· Right click on the taskbar

· Start task manager

· Under Process tab check if you find multiple instance of Outlook.exe

· If found, highlight it right click and end task

 

Method: 2

You may start your Outlook application in safe mode and then try to close the application and check if it take long time to close Outlook.

Click Start, type ’Outlook /safe’ (without quotes) in search programs and files box above Start and press enter.

If it opens and closes the application fine, then it is due to add-ins.

Follow the steps below and disable add-ins in Outlook 2010.

Open Outlook, Click on File, Options, Add-ins, and Click on ‘GO’ beside Manage – COM Add-ins and uncheck the add-ins and click OK

open and close the application and check if it works fine.

 

Method: 3

You may start the computer in clean boot using the article below and try to open and close the Outlook application and check if it works fine:

http://support.microsoft.com/kb/929135

Note: After you have finished troubleshooting in clean boot, ensure to restart the computer in normal mode
