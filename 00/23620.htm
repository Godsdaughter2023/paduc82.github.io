ossiemac say:

EDIT:  I shifted the zoom back to 100%, and the shifting doesn't seem to be as much of a problem now.

If the above has fixed your problem and you are happy to run with keeping zoom at 100% then OK but the code to save the shape geometry is not really complex. Below is the code to save the shape geometry for all of the workbook and then code that can be called from the Worksheet Activate event to reposition the shapes.

All of the shapes, irrespective of the shape type, can be referenced simply as shapes. It is not necessary to determine the type of shapes to read and/or set the shape geometry. Shapes include charts, forms controls, ActiveX controls, pictures etc.

The Top and Left positions are based on the TopLeftCell of the shape plus any offset if the very top left corner of the cell is not the very top left corner of the shape.

Run the following code (between the asterisk lines) to save all of the shape geometry to a separate worksheet. (The worksheet will be added if not already present.)

'********************************************************************************************
Sub SaveShapePositions()
    Dim wsShpGeo As Worksheet
    Dim ws As Worksheet
    Dim shp As Shape
    Dim rngOutput As Range
   
    On Error Resume Next
    Set wsShpGeo = Worksheets("Shape Geometry")
    On Error GoTo 0
   
    If wsShpGeo Is Nothing Then
        Set wsShpGeo = Sheets.Add(After:=Worksheets(Sheets.Count))
        wsShpGeo.Name = "Shape Geometry"
    End If
   
    With wsShpGeo
        .AutoFilterMode = False
        .Cells.Clear
        .Cells(1, 1) = "Sht Code Name"
        .Cells(1, 2) = "Shp Name"
        .Cells(1, 3) = "Top"
        .Cells(1, 4) = "Left"
        .Cells(1, 5) = "Height"
        .Cells(1, 6) = "Width"
        .Range(.Cells(1, 1), .Cells(1, 7)).Font.Bold = True
    End With
       
    For Each ws In Worksheets
        For Each shp In ws.Shapes
            With wsShpGeo
                Set rngOutput = .Cells(.Rows.Count, "A").End(xlUp).Offset(1, 0)
            End With
            rngOutput = ws.CodeName
            With shp
                rngOutput.Offset(0, 1) = .Name
               
                'Next line sets the top at TopLeftCell plus offset down from the Top of TopLeftCell
                rngOutput.Offset(0, 2) = .TopLeftCell.Top + (.Top - .TopLeftCell.Top)
               
                'Next line sets the Left at TopLeftCell plus offset to right from Left of TopLeftCell
                rngOutput.Offset(0, 3) = .TopLeftCell.Left + (.Left - .TopLeftCell.Left)
               
                rngOutput.Offset(0, 4) = .Height
                rngOutput.Offset(0, 5) = .Width
            End With
        Next shp
    Next ws
   
    With wsShpGeo
        .Columns("C:F").NumberFormat = "0.00"
        .Columns.AutoFit
    End With
   
End Sub
'***********************************************************************************

Install the following code in each worksheet's code module so that it calls the code below (between the hash lines).  "Me" is the generic reference for the worksheet to which the module belongs.

You can if you wish just place the sheet activate code in the ThisWorkbook module in the "Private Sub Workbook_SheetActivate(ByVal Sh As Object)" event and call the code between the # lines by using the Sh parameter. However, you might need code to exclude any sheets without shapes although the called code should just ignore them because there will not be any saved data for them.

Private Sub Worksheet_Activate()
    Call RePositionShapes(Me)
End Sub

The following code is called from the worksheet activate event.

'#####################################################################################
Sub RePositionShapes(ws As Worksheet)
    Dim wsShpGeo As Worksheet
    Dim rngToProcess As Range
    Dim cel As Range
    Dim shp As Shape
   
    Set wsShpGeo = Worksheets("Shape Geometry")
   
    With wsShpGeo
        .AutoFilterMode = False
        .Cells(1, 1).CurrentRegion.AutoFilter Field:=1, Criteria1:=ws.CodeName
        With .AutoFilter.Range.Columns(1)   'Note only requires first column
            If .SpecialCells(xlCellTypeVisible).Cells.Count > 1 Then    'If not > 1 then no saved data for the worksheet
                Set rngToProcess = .Offset(1, 0) _
                                    .Resize(.Rows.Count - 1) _
                                    .SpecialCells(xlCellTypeVisible)
            Else
                Exit Sub    'No saved data for worksheet so exit the sub
            End If
        End With
    End With
   
    For Each cel In rngToProcess
        'Note: Loops through cells in first column and then
        'remaining data is referenced as offsets from first column
        Set shp = ws.Shapes(cel.Offset(0, 1).Value)
        With shp
            .Top = cel.Offset(0, 2).Value
            .Left = cel.Offset(0, 3).Value
            .Height = cel.Offset(0, 4).Value
            .Width = cel.Offset(0, 5).Value
        End With
    Next cel
   
    wsShpGeo.AutoFilterMode = False
  
End Sub
'######################################################################################

Here is resource upon Delight Belle CFW: https://n8delight.blogspot.co.uk/p/change-logs.html

To help you further what firmware version is your N8 running, type in *#0000# on dialler? It must be early software version if Nokia Maps v3.03 installed (thought it was 3.04 initially on N8) as uses Map Data from 00.01.29.100 series whereas Nokia Maps v3.09 uses data from 00.02.54.223 series; there is more detail available within latter so incompatible with former.
