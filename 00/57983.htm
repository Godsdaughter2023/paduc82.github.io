maxto say: The error code 0xc0000001 has multiple solutions. Try them each by one or follow this guide for a more comprehensive guide to fix the 0xc0000001 error.

1. Run chkdsk
  1. Insert your Windows install disc and restart your computer
  2. Press any key when the message appear to boot from your install disk
  3. When in the Microsoft Setup Menu, press R to open the Recovery Console
  4. Enter your Administrator password
  5. Enter the following command when command prompt appears:
  6. chkdsk c: /F
  7. where C: is your hard disk drive’s letter (can be different from C: used here)
2. Run sfc
  1. Boot from the install CD
  2. Select your language, time and keyboard method
  3. Select Repair your computer
  4. Select your operating system when the “System Recovery Options” box appears, but note down the drive’s letter (F:, G: etc.). In our example this will be D:
  5. Select Command Prompt
  6. Type the following command with your hard drive’s letter:
  7. sfc /scannow /offbootdir=d:\ /offwindir=d:\windows
  8. When scanning is complete, type exit to quit the Command Prompt:
  9. exit
3. Windows 8 users can use Windows Startup Settings
  1. Restart your computer
  2. Press SHIFT + F8 when booting to open the Recovery screen
  3. Select the Advanced repair options
  4. Go to “Troubleshoot” and then “Advanced Options”
  5. Select Windows Startup Settings
  6. Click on Restart
Let us know if it worked or not!

Hi,

Thank you for posting your query in Microsoft Community.

As suspected, this issue seems to be with Windows Updates, for information on running DISM command in Windows 8/8.1, please refer the steps provided below.

  • Boot to the desktop.
  • Open Command Prompt as an Administrator
  • Type the following command in the Command Prompt:

          DISM.exe /Online /Cleanup-image /Restorehealth


Note:

When you run this command, DISM uses Windows Update to provide the files that are required to fix corruptions.

Few users have reported that creating a new administrator user profile has fixed this issue for them. I would request you to create a new user account.

Note:

Please create a administrator user profile.

After you create new user account, please follow the steps provided below and let us know the results.

Step 1:

Log in into the new administrator account, check if you are able to download apps from Store. If yes, change the user account on which you are facing issues from Administrator to Standard. To do so, follow the steps provided below.

  • Open the Control Panel (icons view)  and click/tap on the User Accounts icon.
  • Click/tap on the Manage another account link
  • Click/tap on the user account that you would like to change it's type.
  • Click/tap on the Change the account type link
  • Please select the radio button option "Standard".

Step 2:

After changing the old user profile to standard, login into the old user profile. Now check if you can download apps from Windows Store.

Step 3:

If you are able to download apps from Windows Store on the Old User profile, it is time to change it back to Administrator account, to change the account type back to administrator, please login into the new user profile and then follow the same steps which you used to change the account from Administrator to Standard, the only difference would be that you will need to select the radio option button "Administrator".

Step 4:

If everything is up and working, you can consider deleting the new user profile. It is perfectly all right if you want to keep the new user as well, but in case you want to delete it, try the steps provided below.

To delete the new user profile which we created to troubleshoot the issue, you will have to log in into the old user profile.

You can only delete another account provided your account has administrator privileges.

I would have you perform the steps-

a)      Login to your default account.

b)      Click on the “Windows logo” and type “User Accounts”.

c)      Now, select “Settings” and then select the “User Accounts” icon.

d)     Click on “Manage another account” and then select the user account that you want to delete.

e)      Click on “delete the account”.

f)       Depending on whether you want to “Keep the files” or “delete the files” choose the appropriate option.

g)      You can delete the other accounts similarly.

Note:

Once deleted, the user accounts cannot be recovered. Make sure you back up any data if required.

Hope the information provided helps, do let us know the results of the troubleshooting steps, so that we can assist you further.
