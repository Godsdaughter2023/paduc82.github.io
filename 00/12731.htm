Hi Elizabeth,

AutoSave is enabled when working on a file that is saved to OneDrive, OneDrive for Business, or SharePoint Online. If the file is saved to another location, then AutoSave is disabled. diane poremsky mvp (slipstick.com) say:

Just because Microsoft is ending "support" doesn't mean you can't use it till it dies - it just means you can't call Microsoft (or HP) for help if you have a problem with it.  Microsoft won't release updates and new software won't work on it, but as long as a antivirus vendors support it, you can keep using it.

 

You mean a document header as in headers and footers? Those are not supported in email. Upgrading will not fix your problem - it's not a problem with the software, it's the medium you are using.

 

Copy the document body to the email - because email is one document, you don't need a header or page #, because there is only 1 long page, but if you do, type it at the top of the page and apply formatting.

 

If you want to use page #, you need to put them in the document body.

 

If the message is not arriving, how large is the email? It's possible its getting "lost" because it's too large. It could also be getting flagged as spam. Will the other party accept a link to skydrive, google drive, drop box etc?
