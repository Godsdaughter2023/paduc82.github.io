erica blair say:

Hi Joflow,

You will need to use the Indirect() function to generate a reference in the way that you're attempting:

instead of doing something like:

=ISERROR(MATCH(Sheet1!A1,array((Sheet1!A2:A(COUNTA(Sheet1!A:A)))),0))

try:

=ISERROR(MATCH(Sheet1!A1,INDIRECT("Sheet1!A2:A"&COUNTA(Sheet1!A:A)),0)

Please let me know if this answers your question and if you need further assistance.
