er. nitin pandey say:

how to use Vlookup in vba?

hi,

         will you please tell me that how to use Vlookup formula in vba. i am using this way but its showing error : "type mismatch"

          ans = Application.VLookup(arg1, myrange, arg3, arg4)

 

         and if i am usin that code then its showing :- "unable to get vlookup property of the worksheet function class"

          ans = Application.WorksheetFunction.VLookup(arg1, myrange, arg3, arg4)        

please help me, any help will be appriciated.

Thanks, Nitin.

dave peterson say:

Make sure you declare ans as a variant.

Dim Ans as variant
dim myVal as string 'long, double, whatever
dim myRng as range

set myrng = worksheets("sheet99").range("a:e")
myVal = "something here"

ans = application.vlookup(myval, myrng, 3, false)

if iserror(ans) then
   ans = "Not found!"
else
   'do nothing
end if

msgbox ans

==========
If there is no match, then Ans needs to be able to hold an error -- something
strings and numbers can't do.

==========

And when/if you use application.worksheetfunction.vlookup(), you'll have to trap
the error.  This syntax doesn't return an error.  It causes a runtime error.

Dim Ans as variant
dim myVal as string 'long, double, whatever
dim myRng as range

set myrng = worksheets("sheet99").range("a:e")
myVal = "something here"

on error resume next
ans = application.worksheetfunction.vlookup(myval, myrng, 3, false)
if err.number <> 0 then
   ans = "Not found!"
else
   'do nothing
end if
on error goto 0

msgbox ans

Hi Bab,

The migration method is so easy and it should be done per user basis.

However, as a Global admin, you are able to migrate files for your end users as long as you have full control permission on user's site. For example, I want to migrate User a2's local files to his OneDrive for Business library. Then I go to Office 365 admin center -> Active user -> tick a2 ->OneDrive settings -> click access files.

After I log into user a2's OneDrive for Business library, i copy his OneDrive for Business site URL, such as https://domain-my.sharepoint.com/personal/a2_domain_onmicrosoft_com -> launch SharePoint Management tool -> type into your credential (Global admin) -> follow the prompt and type user a2's personal site URL above.
