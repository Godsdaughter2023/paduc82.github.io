dononewco say:

Found the answer...

This turns out to be a "Samsung Update Utility" issue. Seems that Samsung doesnt seem to care that much about this issue so in their update they never put a proper driver in the update package. So in order to fix this I did the following.

1) Research the hardware and find out which display setup my tablet was using (for me it was the "Intel HD Graphics 4000" but for others it may be something else).

2) Go directly to the Manufacturers Website of the hardware I need the driver for, so in my case it was the "Intel" web site. THEN DOWNLOAD THE DRIVERS IN ZIP FILE FORMAT and unzip files to your favorite location.

**Zip file format is very important because the exe installer will abort because it is not approved by the tablet manufacturer in my case.

3)Open up device manager and goto display adapter right click and update driver. Select browse to the file option and go to the place where you unzipped the files and select ok. Then finish the install, restart the computer and all should be well.

In my case the generic Samsung approved driver did not lend support to the miracast or widi functions on my tablet.

If you had your Windows 10 license activated previously, you got a digital license. You will never need the product key again once you get digital license. tricky300 say:

You don't need to change cables, disk drives or any other hardware if you change the BIOS setting. In simple terms, changing the setting to IDE runs your current SATA equipment in "IDE mode".

 

Stop 7B (minus leading zeros and all the other hex numbers) is what I predicted and switching to IDE mode should prevent this stop.

 

Vista and later operating systems support AHCI natively (i.e. drivers for AHCI are incorporated). XP does not have these drivers and consequently you have to provide them via the F6 mechanism which only supports floppy disk drives, i.e. supplying the drivers via other devices is not going to work. If you switch to IDE/ATA mode these drivers are not needed and installation should progress.
