How do you change the color of "check box content control" - Microsoft Community

sabortooth say:

How do you change the color of "check box content control" "when selected"

How do you change the color of check box content control when selected? I want the box black, but when it is selected I would like it to turn red?

Thanks, Scot

macropod say:

You could do it with a ContentControlOnEnter macro:

Private Sub Document_ContentControlOnEnter(ByVal CCtrl As ContentControl)
With CCtrl
  If .Type = wdContentControlCheckBox Then .Range.Font.ColorIndex = wdRed
End With
End Sub

Presumably, you'll also want to change it back again when you exit it:

Private Sub Document_ContentControlOnExit(ByVal CCtrl As ContentControl, Cancel As Boolean)
With CCtrl
  If .Type = wdContentControlCheckBox Then .Range.Font.ColorIndex = wdAuto
End With
End Sub

paul van der heu say:

The current Firmware  for the 1020 supports the full highresolution images in either JPG or RAW (.DNG) format. You can not directly extract these from the phone to an iOS device as you would need MTP support which it does not support. So to get the photos off of the phone you would need a supporting device from where you can probably get the images to an iPad. You need some sort of cable to attach remote storage to the iPad to get the files on it or use cloud storage. I guess you can edit JPG on iPad, no so sure about DNG files.

 

Frankly, a Windows 8.1 tablet would be much more efficient and easy to work with here.
