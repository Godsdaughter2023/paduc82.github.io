ipconfig /registerdns

ipconfig /release

ipconfig /renew

nathaniel mon say:

You can try disabling the automatic updates using the Registry. Kindly follow these steps:

  1. Press Windows key + R to open the Run command.
  2. Type regedit, and click on OK to open the Registry.
  3. Browse this path: HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows.
  4. Right-click on the Windows (folder) key, select New, and then click on Key.
  5. Use WindowsUpdate as the name of the new key and press Enter.
  6. Right-click on the newly created key, select new, and click on Key.
  7. Use AU as the name of the new key and press Enter.
  8. Inside the newly created key, right-click on the right side, select New, and click on DWORD (32-bit) Value.
  9. Use AUOptions as the name of the new key and press Enter.
  10. Double-click on the newly created key and change its value using one of the following options:
  • 2 - "Notify for download and notify for install."
  • 3 - "Auto download and notify for install."
  • 4 - "Auto download and schedule the install."
  • 5 - "Allow local admin to choose settings."

11. Click on OK.

12. Close the Registry to complete the task.

Note: Editing the registry is risky, and it can cause irreversible damage to your installation if you don't do it correctly. It's recommended that you make a full backup of your computer before proceeding.
