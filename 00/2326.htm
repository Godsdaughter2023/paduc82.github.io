IF you really want VBA code, then, one way to meet the requirements you request:

=====================================
Option Explicit
Sub SetZeroValueFontWhite()
Dim c As Range
Dim FirstAddress As String
Application.ScreenUpdating = False

With Sheet1.Range("A1:G10")
    .Font.Color = vbBlack 'or whatever color
    Set c = .Find(what:=0, LookIn:=xlValues, lookat:=xlWhole)
    If Not c Is Nothing Then
        FirstAddress = c.Address
        Do
            c.Font.Color = vbWhite
            Set c = .FindNext(c)
        Loop Until c.Address = FirstAddress
    End If
End With

Application.ScreenUpdating = True
End Sub

If you want a little more control over things, you can use a snippet of code to format your code:

        

Sub AddLineNumbers()

    Dim x As Long
    Dim oSh As Shape
    Set oSh = ActiveWindow.Selection.ShapeRange(1)

    With oSh.TextFrame.TextRange
        For x = 1 To .Paragraphs.Count
            .Paragraphs(x).Characters.InsertBefore (CStr(x) & " ")
        Next
    End With

End Sub

If you want to number lines rather than paragraphs, change it to:

For x = 1 to .Lines.Count

Will you really use all 750 in one sitting?  Lots of slides in rapid-fire animation, for example?

If there's no overwhelming reason to put all of them in one presentation rather than breaking them into multiple (possibly linked) presentations, then there's your answer.  Put them in multiple presentations.
