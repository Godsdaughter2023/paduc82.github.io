ewgssi say:

An old adage in database computing is: If there is more than one copy of a data item, then all but the original copy are redundant. In light of this, why do you need to create a second (redundant) copy!

[It is a simple exercise to find the index to the column header text using a MATCH function, then use that index value in an OFFSET function to define the column range; e.g:

   col.Index=MATCH(col.Text,Sheet1!$1:$1,0)

   tab.Table=OFFSET(Sheet1!$A:$A,0,col.Index-1)

   where col.Index is the index  value to the desired column headed by the text value in col.Text.

   Then the definition for the desired table named tab.Table is the column range OFFSET from the first column in the sheet by one less than the index value.]
