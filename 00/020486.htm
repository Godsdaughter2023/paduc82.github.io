hansv mvp say:

Since a default new workbook doesn't have a Visual Basic project until you create a module or some code in a workbook/worksheet module, you'd have to do something like this:

  • Create a new empty workbook.
  • Activate the Visual Basic Editor.
  • Select Insert > Module to create a module. It may remain empty.
  • Select Tools > References and set the references that you want.
  • Switch back to Excel.
  • Save the workbook as a macro-enabled template named Book.xltm (this name is obligatory; however it depends on the language, in French it would be Livre.xltm) in the folder C:\Users\<yourusername>\AppData\Roaming\Microsoft\Excel\XLSTART.
  • Quit and restart Excel.

Side effect: if you save a new workbook as a .xltx workbook, you'll get a warning that the VB Project will be lost if you continue.

Blinking message icon on N96 - Microsoft Community

adamf---01 say:

Blinking message icon on N96

Technical Level : Intermediate

Summary

Details
If there is a blinking message icon on the display, the main reason for this is that SIM card message memory is full.

Class 2 messages (e.g. service messages from operators) are stored in SIM card message memory. Once the SIM card message memory is full, class 2 messages can't be received.

One reason for this might be that the user has used a S40 product before.
In S40 products SIM card message memory is filled before phone message memory.
Once you insert a SIM card with full message memory to the phone, symptom described above will occur.

Solution:

  1. Messaging
  2. Options
  3. SIM messages
  4. Delete messages

This will remove the message icon.
