The reason there's a difference is because the "older" version of Outlook had the same email account configured as a POP3 account with the "Leave messages on server" option <unchecked>.

When the account was configured on the new computer with Outlook '2019, you no doubt added the email account to Outlook just by entering the email address and password which would have resulted in the account being configured as an IMAP account.

The difference between a POP3 and IMAP account is that POP3 only downloads the email items in the <Inbox> whereas with IMAP all email folders are sync'd between server and computer. Nothing gets deleted from either side unless you explicitly delete/move something from one of the sync'd email folders.

To get back to where you were, suggest the following steps

With Outlook closed, create a new Outlook profile by opening the Control Panel <Mail> app --> click on <Show Profiles> --> Select <Add> --> give the profile a name and then add your email account. Check the option for <Manual Setup or additional server types> and configure the account using the settings applicable to the account.

The POP3 account would have had a PST file associated with it which is where the data (email, contacts, calendar etc) would have been stored. If you still have access to that PST file, suggest you copy it to your new computer if you haven't already and make sure to set Outlook to use that PST file instead of creating a new one.

There's no need to delete the original Outlook profile at this point. Just make sure that you set the new Outlook profile as the default for the next time Outlook is opened.

If you have any questions regarding the above, please ask away.
