If you also have a Factory Recovery partition (larger than 10gb) on the drive then it most likely won't run any longer.  You can try generating your Recovery media if it still will, even try triggering Recovery from boot as specified by your specific model, and if it will not run then you can delete it to recover that space.  But if you think you may ever want to restore to factory condition (a much inferior install to the Clean Install Windows 10) you can also save it because there is a method from a working OS to Boot Recovery Partition using EasyBCD .

Also do not confuse Recovery partitions - which in Windows 10 are for the WinRE repair tools which are separated out from C for the first time, or a Factory Recovery partition which is much bigger - with the System Reserved or EFI System partitions which exist to boot Windows.  Only in rare cases are these System partitions merged with WinRE Recovery by manufacturers at the factory, which may explain why a System partition is much bigger.
