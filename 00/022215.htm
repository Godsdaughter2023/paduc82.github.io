ihidalgo say:

I need to count text, minus the Grand Total bar

Hello,

I am trying to count the cells that contain text from A1 to A10 and A11 being the Grand Total. I used an erray formula =Counta(A1:A10). When I filter the cells my counta formula counts the grand total as text as well. Meaning it includes this total cell in the results making it inaccurate. I am trying to just count for example the names of cities, but when I filter if I get less cities, the formula counts Grand Total cell as a city as well. I know why excel is doing it, I just don't know how to fix it.How can I leave this bar out so that I only count the number of cells that contain city even when I filter?

Thank you for your help!

amit tandon say:

=COUNTA(A1:A10) will count all non-blank cells in range A1:A10;

=COUNTA(A1:A11) will count all non-blank cells in range A1:A11 so that "Grand Total" in cell A11 is also counted;

=COUNTIFS(A1:A11,"<>Grand Total",A1:A11,"<>") will count all non-blank cells in range A1:A11 & will exclude cells containing text "Grand Total";

=SUBTOTAL(3,A1:A10) SUBTOTAL always excludes cells that are hidden by filtering with Autofilter (does not exclude cells that are hidden other than autofilter);

=SUBTOTAL(103,A1:A10) SUBTOTAL always excludes cells that are hidden by filtering with Autofilter (using 103 also excludes cells that are hidden other than autofilter);

in SUBTOTAL, using 3 & 103 indicates COUNTA.

ahg1 say:

You could try this cross tab query:

Create a new query using Query design mode. Then switch to SQL view and paste the following query into the SQL window.

Replace "YourTable" with the actual table name:

---------------------------------------------------------------------------------------------------------------------------

TRANSFORM Last(YourTable.[Expired]) AS LastOfExpired
SELECT YourTable.[Company Name], YourTable.[Address], YourTable.[City], YourTable.State, YourTable.Zip
FROM YourTable
GROUP BY YourTable.[Company Name], YourTable.[Address], YourTable.[City], YourTable.State, YourTable.Zip
PIVOT YourTable.[Type];
--------------------------------------------------------------------------------------------------------------------------

This will return the last instance of the expired dates. I f you want the first set change the last function to First. (You can also use Min or Max.
