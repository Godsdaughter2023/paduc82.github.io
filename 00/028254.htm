steven. s say:

Hey JJ’s Mom,

 

You do have the option of enabling Parental Controls on a standard user account. Although you would have to set up a separate account, this will remove the ability to use InPrivate browsing. This will work with any parental control setting, ranging from no restrictions to completely blocking Internet access. For more information on Parental Controls and how to set it up:

Windows 7 – Parental Controls

Windows Vista – Parental Controls

Windows XP – Family Safety (this can also be used in Windows Vista or 7)

 

Alternatively you can disable InPrivate browsing with a group policy, but this may not be an option depending on your version of Windows. You can test to see if this is available by:

·         Press Windows key + R on your keyboard

·         In the Open: filed, type gpedit.msc

·         Click OK

·         Open this path: Computer Configuration > Administrative Templates > Windows Components > Internet Explorer > InPrivate

·         In the right column will be the option to “Turn off InPrivate Browsing”

 

Keep in mind that using group policy to remove InPrivate Browsing can be bypassed if the user is also using an administrator account because they can simply undo/redo the steps you just took.

 

Hope that helps.
