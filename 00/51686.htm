a. user say:

Any program that saves data to your hard drive, can save its data anywhere possible.

Any computer program will (shall) let you choose where to save important or voluminous files.

Computer programs can identify where your Documents folder is, to use it by default.

You must check settings for each program, individually, if you want to make changes.

Every program you install (or not) will have its own location to save files.

From the desktop, you can't set desktop as the default. You need to modify each program's settings.
