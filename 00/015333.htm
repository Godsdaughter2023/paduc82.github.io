rich007a say:

Hi,

Say you have two excel files called Chinese.xlsx and English.xlsx

In Chinese.xlsx on Sheet1 is the Chinese text and

in English.xlsx on Sheet1 is the English text.

In an empty worksheet, in cell A1, enter the following formula:

='[Chinese.xlsx]Sheet1'!A1&CHAR(10)&'[English.xlsx]Sheet1'!A1

Drag that formula down and across.

Make sure you format the cells to Wrap Text (there's a button on the Home tab).

That said, if I were doing this myself, I'd copy the two worksheets into the same Workbook so you have a sheet called "Chinese", a sheet called "English" and one more sheet for your combination.

You would then use the formula  =Chinese!A1&CHAR(10)&English!A1

If you want to delete the original data, select all the cells and copy & paste as values.

Hope that helps.

Cheers
Rich

Wildcard replacement patterns in MS Word - Microsoft Community

the hit man say:

Wildcard replacement patterns in MS Word

Based on the sample format (Opening Bracket Numeric Value Closing Bracket Sample:[169]) How can I use a wildcard to replace multiple occurences with different numeric variables?
macropod say:

A wildcard expression like:
Find = \[[0-9]{1,3}\]
will find numbers of 1-3 digits length within a set of square brackets. Depending on what you want to replace these numbers with, you may need a macro. For example, if you want to increment the numbers, a macro would be required.

graham mayor say:
\[[0-9]{1,}\]

will find a number in square brackets - http://www.gmayor.com/replace_using_wildcards.htm
but you can only set 1 replacement value (unless you propose reordering the numbers found).

To replace a variety numbers with other numbers, you are going to need a macro and a list of the numbers and their replacements e.g. each item in vFindText must have a corresponding item in vReplaceText.

Dim vFindText As Variant
Dim vReplaceText As Variant
Dim oRng As Range
Dim i As Long
vFindText = Array("[169]", "[234]", "[111]", "[732]")
vReplaceText = Array("[321]", "[456]", "[777]", "[921]")
For i = 0 To UBound(vFindText)
    Set oRng = ActiveDocument.Range
    With oRng.Find
        Do While .Execute(FindText:=vFindText(i), _
                          MatchWildcards:=False, _
                          Forward:=True) = True
            oRng.Text = vReplaceText(i)
            oRng.Collapse wdCollapseEnd
        Loop
    End With
Next
