james bernie say:

Possibly what you are looking for requires you to access the registry to do the rename.

HKLM\Software\Microsoft\Windows NT\CurrentVersion\NetworkList\Profiles

Go through the list of profiles to find the one you need and then change the ProfileName string value.

You can also do this when you change wireless adapters and reconnect to the same network and the connection then shows up as "wireless name 2" or "wireless name 3" instead of just "wireless name", you can delete the old profiles of the same name and then rename the current one without the number on the end.

After making the changes just refresh your Network Connections list to see the changes.

As always, making changes in the registry is not advised, do so with caution, take a backup before making changes etc etc.
