teylyn say:

Here is a formula way to create the layout table.

Copy and paste the first column of your original data into a new column, starting in G3. In H1 enter this formula:

=INDEX($A:$A,CEILING(COLUMN(C1)/2,1))

Copy this as far right as you have data. 

In cell H2 put this formula to create the X axis values and copy to the right:

=INDEX($B$2:$E$6,MATCH(H$1,$A$2:$A$6,0),ISODD(COLUMN())+1)

In cell H3 put this formula to create the Y axis values in the correct places. Copy the formula down and across as far as required.

=IF(H$1=$G3,INDEX($B$2:$E$6,MATCH(H$1,$A$2:$A$6,0),ISODD(COLUMN())+3),NA())

Now select the data table from cell G2, down and across. Do not select row 1. Create the XY chart with lines.

This formula works great (very similar to mine). One thing to note is that if there is a space in the name of the worksheet it dies. You may want to amend the formula to include single quotes around the sheet name

 

=INDIRECT("'" & A$1&"'!R"&ROW(A12)&"C"&COLUMN(X12),FALSE)

I am guessing you want this...

=INDIRECT("'" & A$1 & "'!" & ADDRESS(ROW()+10, COLUMN()+23))

=INDIRECT("'" & A$1 & "'!" & ADDRESS(ROW(X12), COLUMN(X12)))
