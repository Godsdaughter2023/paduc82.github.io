- remove the linked folder

or

- update the credentials

In one case when EVERYONE in an organization was affected - it was simply that a share they all used had changed credentials.  For a single user - it was that they had uniquely linked to a folder whose permissions changed.

Mac 2008 Word Formula Syntax??? - Microsoft Community

billmce say:

Mac 2008 Word Formula Syntax???

I have two tables in a Word Mac 2008 document, and the last cell in the lower right hand column of each table has a total amount for that table (calculated by selecting Table>Formula and doing a simple "=SUM(ABOVE)" calculation.  I need to calculate a sum for the two table values.  How to I reference the tables and cell in order to add them together in my document.
bob jones aka: cybertaz say:
Each of the 2 tables needs to be Bookmarked:

  1. Select a table (Table> Select> Table)
  2. Go to Insert> Bookmark
  3. Type a bookmark name [such as TblOne]
  4. Click OK
  5. Repeat for the second table [using a different name, of course :-)]

To total the 2 sums, identify the cells containing the totals [eg., 3rd column, 3rd row would be C3) then in the location where you want to create the grand total:

  1. Go to Table> Formula
  2. Enter the formula:  =SUM(TblOne C3, TblTwo C3)

Regards,
Bob J.

a. user say:

Unless you know the absolute cell address of both totals, the simplest approach is probably as follows:

In the first table, instead of { =sum(above) }

Use

{ quote "{ set total1 { =sum(above) } }{ total1 }" }


In the second table, use the same nested fields, but using "total2" rather than "total1"

To calculate the overall total, use

{ =total1+total2 }

Or you may need

{ ={total1}+{total2} }

All of the {} need to be the special field code braces that you may be able to insert using cmd-F9, OSx permitting
