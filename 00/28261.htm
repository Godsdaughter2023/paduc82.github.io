This code will select the used range of a sheet.

Sub select_used_range()
    Dim LastRow As Long
    Dim LastCol As Long
    Dim myRng  As Range
    With Worksheets(1)
        LastRow = .Cells(.Rows.Count, "A").End(xlUp).Row
        LastCol = .Cells(1, .Columns.Count).End(xlToLeft).Column
        Set myRng = .Range("A1", .Cells(LastRow, LastCol))
        myRng.Select
        MsgBox myRng.Address
    End With
End Sub

    WayBackMachine - Internet Archive - Find missing Web pages - Error 404 - Microsoft Community

    rohn007 say:

    WayBackMachine - Internet Archive - Find missing Web pages - Error 404

    Technical Level : Basic

    Summary

    The internet is like a living organism.  New sites are created every day and existing sites are taken down. Sometimes web sites are reorganized, changing the links.  So one way or another ever internet users hits an "HTML ERROR 404", page not found at some time.

    The "WayBackMachine" on the Internet Archive site is a partial answer to that problem.

    The site uses tools called "crawlers" that scan websites. These crawlers take snapshots of every page they scan and send them back to the archive.  The website has the option of blocking crawlers, so not all pages are archived.

    The name came from a 1960's cartoon about a boy and his talking dog who had a time machine called the "WABAC" machine.

    Bottom line, if you get a web page not found message,

    Google/Bing to the WayBackMachine


    Details

    If you get a HTML Error Code 404 - Page Not Found it would be nice to get the information somewhere else.  That somewhere may be in the Internet Archive "WayBackMachine"

    The site Home page looks like this:

    Paste the link to the missing web page into the input box surrounded by the green highlight box.

    If your page is found in the archive, you will see something like this:

    1. The search input field showing your search value and where you can put new links for searching
    2. A link to the searched page that was found
    3. A graphical presentation of how often the page was captured over time
    4. A calendar view showing dates the page was crawled as blue dots

    Click on one of the blue dots to retrieve the archived page

    When you click on one of the blue dots the retrieved page looks like this:

    1. The internet archive address of the retrieve page
    2. Search input box with your original web address
    3. Abbreviated graphical timeline
    4. Your web page retrieved from the archive.

    NOTE: sometimes even though the page was recorded as crawled it may not be there.  In that case you can click on the arrows to scroll through the history to pick another date to find the page.

    Have fun!
