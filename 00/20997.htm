I suspect that you have a blank cell at the bottom of the list that you have applied to the Data Validation list. If a value in the list matches the current value in the cell then the DropDown defaults to the selection on that value and hence if the current cell is blank and a blank is included at the bottom of the DropDown list then the last value is the default that gets highlighted.

In the first snippet where I have used the values in column A for the DropDown list and included a blank cell (A10) at the bottom in the DropDown list. Then because the cell with the DropDown is blank then the DropDown defaults to the blank value at the bottom of the list.

The yellow fill color of the cell is just to highlight the cell containing the DropDown and is irrelevant to the explanation here.

The second screen snippet does not include a blank cell in the list. ie the list is only A1:A9 and therefore nothing in the list matches the empty cell and the list starts from the top.
