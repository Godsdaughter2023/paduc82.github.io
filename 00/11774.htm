Sub dural()
Dim r As Range

Next r
End Sub

pc-learner say:

1. Shut down the PC.

2. Insert a bootable USB with Windows 10 1803 (or later) on it.

3. Power up the PC and enter the BIOS - either Setup (usually DEL or F2) or Boot (usually F12).

4. Tell the BIOS you want it to boot from the USB, not the hard drive.

5. Turn off Secure Boot and make sure the USB matches the BIOS mode (MBR USB = Legacy BIOS; GPT USB = UEFI BIOS).

6. Boot from the USB.

7. When the Windows screen comes up, do not Install Windows

8. Repair your installation using the Troubleshooting options.

9. If that doesn't work, do a clean install at step 7. If you need this, you may want to get the latest Insider ISO and recreate the USB first.

10. Reinstall your user files, if step 9 was necessary, from your backup.
