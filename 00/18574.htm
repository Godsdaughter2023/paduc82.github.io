Sub CombineWorkbooks()
  Dim Source As Range, Dest As Range
  Dim FName As String
  Dim WB As Workbook
  Dim SaveCalculation As XlCalculation
 
  'Install an error handler
  On Error GoTo ErrorHandler
  Application.EnableCancelKey = xlErrorHandler
  'Switch some things off => runs faster
  Application.ScreenUpdating = False
  Application.EnableEvents = False
  SaveCalculation = Application.Calculation
  Application.Calculation = xlCalculationManual
  'Set the destination for the data
  Set Dest = Range("A" & Rows.Count).End(xlUp).Offset(1)
  'Find the first file in the same directory of this file
  '  Note: This finds *.XLSM and *.XLSX files too!
  FName = Dir(ThisWorkbook.Path & "\*.xls")
  'Found?
  Do While FName <> ""
    'Skip our file
    If FName = ThisWorkbook.Name Then GoTo Skip
    'Open the file
    Set WB = Workbooks.Open(ThisWorkbook.Path & "\" & FName, False, True)
    'Give the user a change to abort
    DoEvents
    'Get the data
    Set Source = WB.ActiveSheet.UsedRange
    'Copy the data to our file
    Source.Copy Dest
    'Give the user a change to abort
    DoEvents
    'Set the destination to the next empty cell
    Set Dest = Dest.Offset(Source.Rows.Count)
    'Close the file
    WB.Close False
    'Give the user a change to abort
    DoEvents
Skip:
    'Find the next file
    FName = Dir
  Loop
  'Done
  Application.ScreenUpdating = True
  Application.EnableEvents = True
  Application.Calculation = SaveCalculation

Excel IF Formula's to create a RAG status column - Microsoft Community

andrewgreen_745 say:

Excel IF Formula's to create a RAG status column

I have a spreadsheet with a status column (column G) and a column with how many days old the record (Column I) is that i have pulled from a data base, i need a column that will tell me if:

Status is AQ , result is RED

Status APV Red and 7 or less days old, result AMBER - 8+ days old, result RED

Status APV and 7 or less days old, result AMBER - 8+ days old, result RED

Status AI Red and 7 or less days old, result Green, 8-14 days old, result AMBER, 15+ days old, result RED

Status AI and 14 days old or less, result GREEN, 15-28 days old, result AMBER, 29+ days old, result RED

Status Free fill and 3 or less days old, result GREEN, 4-7 days, result AMBER, 8+ days old, result RED

Status IA, OT or AD, result GREEN 

either it coming up with the word in column K or changing the colour of column D would be brilliant

jasles say:

You can use conditional formatting to do that. Create three different rules - one for each color. I assumed that the text after each Status in your question above was the text in the cell in column G. You may have to change the actual text in the formulas below. I also assumed that the days old column would contain just a number for number of days.

Assuming your data starts in row 2, click in cell D2, click "Conditional Formatting" under Styles on the Home tab of the ribbon, select "New Rule", select "Use a formula to determine which cells to format". Then paste the first formula I have below - this is for green formatting, and then click the Format button and select the green color fill. Repeat this for amber and again for red.

Once you're done, copy cell D2 down to the bottom of your data and you're good to go.

Green rule formula:

=OR(AND(G2="AI Red",I2<=7),AND(G2="AI",I2<=14),AND(G2="Free fill",I2<=3),G2="IA",G2="OT",G2="AD")

Amber rule formula:

=OR(AND(LEFT(G2,3)="APV",I2<=7),AND(G2="AI Red",AND(I2>=8,I2<=14)),AND(G2="AI",AND(I2>=15,I2<=28)),AND(G2="Free fill",AND(I2>=4,I2<=7)))

Red rule formula:

=OR(G2="AQ",AND(LEFT(G2,3)="APV",I2>=8),AND(G2="AI Red",I2>=15),AND(G2="AI",I2>=29),AND(G2="Free fill",I2>=8))
