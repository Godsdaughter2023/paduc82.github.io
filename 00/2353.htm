julie fka profjulie say: If you want to do much of anything on your phone, you'll need to enter a windows live id and password into it when prompted to do so (or if you don't do it then, you'll have to set up a windows live account later).  Unless you input a windows live id and password into the phone, you will not be able to use the Marketplace, a ZunePass, the xBox features of the phone....The only way to sync calendar and contact information onto the phone is by syncing to a cloud-based service (Google, Windows Live) unless you have access to an Exchange Server (for work, perhaps).  Now, you can directly sync media files (music, photos, video) between your computer and phone via a USB connection, but the only way you can get other files on the phone are either by emailing them to yourself as attachments to an email address you've set up on the phone or by downloading the files, again, from some cloud based service.

As Entegy stated in his reply, syncing and downloading from a cloud-based service is not the same thing as sharing files.  Most all cloud based services provide a way to share files with other people, but you have to enable that option.

If you input a windows live id into the phone, any contacts or calendar items that are associated with that account will automatically sync to your phone.  If you don't want these kinds of things to sync to your phone, then you need to remove them from your windows live account (by going online and deleting them).  You can also set an option within the windows live account that determines how often the phone syncs to your windows live account by going to Settings > email + accounts > select your windows live account > set the "download new content" to "manually" if you don't want the phone to automatically sync to your windows live account....

Not really sure what you mean by "why does your phone relay info to the cloud without your permission."  I've already explained how you can change the sync options on a windows live account, but you can do the same for any other account you've set up on the phone.
