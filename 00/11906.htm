Sorry, there is no way to specify "Merge format but only a few parts of the format". It's all or nothing.

After you paste with Merge Format, select the pasted text and change the font.

You can try applying the Normal style to the pasted text, but the result will probably vary from one instance to another, depending on how much of the text has the bold, italics, and underline formatting -- if it's more than half of the text, Word will make the entire selection have that formatting. It can be very difficult to predict when that threshold has been crossed, although you can press Ctrl+Z (Undo) if the results are wrong.

I prefer dvd. once started do nothing unless it hangs, which has happened to a few. it will take about an hour to install and will restart as needed. it will create a boot menu to allow you to use both versions, win 10 preview and the 8.1. wait about 24 hours to update to build 10166 through windows update.

when doing a dual boot there is no capture of a previous license as there is no previous version on the selected drive or partition you install to. the drive or partition is empty and you'll be doing a clean boot, not an upgrade from a previous version of windows. you will have 2 separate operating systems, the windows previous version on one drive or  partition and win 10 preview on another drive or partition.

if you use the upgrade method from a previous version of windows 7 or 8.1, the license will not allow you to re-use it to do another separate install of the upgraded os unless you roll it back to the original os.

Option Explicit

Sub Frustration()

    Dim sFileDirectory As String, strFileName As String
