Hello,

Welcome to Microsoft Community.

To resolve this issue we need more information. Let us know the following.

What is the type of email account you have setup in Outlook 2010 and Outlook 2016? For example POP3, IMAP, Exchange Active Sync (EAS) or Microsoft Exchange. To check the type of email account configured in Outlook 2010 and Outlook 2016 click on File > Info > Account Settings > Account Settings drop-down. EAS account refers to Hotmail.com, Outlook.com or Live.com email accounts.

In case you have a POP account in Outlook 2010/Outlook 2016 then you need to export the PST file from Outlook 2010 and import the same in Outlook 2016.

You may follow the steps as shown in the following Microsoft Article to export the PST file from Outlook 2010.

https://support.microsoft.com/en-us/kb/287070/

In order to import the PST file in Outlook 2016 follow these steps:

  1. Click on File > Open & Export > Import/Export > Import from another program or file > Outlook Data File (.pst).

  2. Click on Browse and select the PST file and click on Open > select Do not import duplicates.

  3. Make sure Outlook Data File is selected and Include subfolders is checked > under Import items into the same folder in: make sure to select the PST and not any sub-folder

  4. Click on Finish.

If you are using other email accounts types: IMAP, EAS or Microsoft Exchange, then you do not need to export and import you just have to reconfigure your email account/s in Outlook 2016 and synchronize.

Check if the issue resolves following the steps mentioned above. Let us know the result.
