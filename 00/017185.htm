bernard liengme say:

Excel sees your first diagram

1896,2051,2576,2587,2598,2609,2620,2631,2642,2653,2664,2675

as a number. But Excel can handle a max of 15 digits so it rounds this and displays it in scientific notation.

You need to proceed it with a single quote (aka an apostrophe); '1898,2051....

Then Excel will recognize it as text.

You might be able to replace 1896 by 3333 after that

Depending on how you generate this entry, you might be able to format the cell as Text before adding the entry

Krebs is referring to a feature in Windows 10 called Wi-Fi Sense. I do understand that programmers have the ability to test and multiply features a normal user cannot. However, That's not how Wi-Fi Sense works.

Yes, the option to allow sharing is enabled by default.

Wi-Fi Sense is absolutely secure and no one is going to connect to your network without your permission.
