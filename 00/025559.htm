In Access macro commands are called Actions and VBA command are Methods. I gave you the format for such a WHERE clause in my response.

scottgem say:

This is not something you use a Macro for. If you want to filter a report for a specific date or range of dates, you have 2 options. You can either include criteria in the query that is the Recordsource of the report or you can add a WHERE clause to the OpenReport method (or action).  A WHERE clause takes the form of:

fieldname = variable

Where fieldname is the name of the field in the Report's Recordsource and variable is the value you want to compare it to.

Is that what you are looking for?

Then

  1. in sheet OOD select your data list and apply the following conditional formatting formula:
    • =ISNA(MATCH(B2,Watchbill!$B$2:$B$7,0))=FALSE
  2. in sheet Watchbill select your data list and apply the following conditional formatting formula:
    • =ISNA(MATCH(B2,ODD!$B$2:$B$13,0))=FALSE
  3. in  sheet INFO select your data list and apply the following conditional formatting formula:
    • =ISNA(MATCH(B2,Watchbill!$B$2:$B$7,0))=FALSE

Perhaps you should use structured references.

  1. Transform your data sets into tables (Ctrl+T) and give them appropriate names (e.g. tbl_odd, tbl_watchbill, tbl_info);
  2. in sheet Watchbill select your table's column and apply the following conditional formatting formula:

    • =ISNA(MATCH(INDIRECT("tbl_info[@Months]"),INDIRECT("tbl_watchbill[Months]"),0))=FALSE
