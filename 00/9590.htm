Tips for Making USB Sticks & Hard Drives Bootable... - Microsoft Community

slickclick say:

Tips for Making USB Sticks & Hard Drives Bootable...

This install would be rather tricky for the home user -I have WinXP32-; however there is a couple tips to an easy smooth install...

To do a 32-bit install you can have no less than a 4Gb Memory stick...

To do a 64-bit install you will need at least an 8Gb Memory stick...

If you are using a memory stick do not waste your time with any other apps or programs other than "rufus"..!
I tried a few others; but this is the only one that did the job.

Also Microsoft has an ISO installer tool that works as well XP users have 2 additional downloads with that; but my goal was to boot from a USB Hard Drive...

So; if you have a USB Hard Drive that you are trying to toss this onto its a bit more in depth...

USE A BLANK DRIVE!!! I HIGHLY SUGGEST NOT USING A DRIVE WITH DATA ON IT!!!

As you will be making this drive "bootable"...

To do this you will need a "32-bit bootsect.exe" file... -I went here: http://85.25.103.133/Getintopc.com/bootsect.exe -

I use an app called MagicDisc to load ISO files as a Disc Drive; so I would recommend that to use...

Load up the Win10 ISO and copy the "boot" folder to your C: drive -yes just the root folder-

Now open C:\Boot and delete the bootsect.exe from there,

then copy and paste in the downloaded 32-bit one; this is applicable for 64-bit installs from older 32-bit OSes -WinXP-

-NOTE: I didnt actually try doing this with the 32-bit Win10 ISOs-

Plug in USB hard Drive,

WARNING FROM THIS POINT FORWARD BE SURE OF WHICH DRIVE IS YOUR USB DRIVE OR YOU COULD ERASE THE WRONG DRIVE!!!

Open a "COMMAND PROMPT" from the "run" bar on the start menu type "cmd" and hit enter -without quotes-
Now in the Dos Box; type "CD\"

You should now see this:

C:\>

Now type "diskpart" and hit enter; this will begin a dos tool; the next steps you want to do exactly as follows:

  1. DISKPART> list disk                                 /* shows list of active disks */
  2. DISKPART> select disk #                         /* # is the number for your USB flash drive */
  3. DISKPART> clean                                     /* deletes any existing partitions on the USB flash drive */
  4. DISKPART> create partition primary     /* create a primary partition on the USB flash drive */
  5. DISKPART> select partition 1                 /* select the newly created partition */
  6. DISKPART> active                                    /* make the new partition active */
  7. DISKPART> format FS=NTFS                 /* format the USB drive with NTFS file system */ NOTE: If this Command does NOT Work KEEP GOING!!!
  8. DISKPART> assign                                   /* assign a volume and drive letter to the USB drive */
  9. DISKPART> exit                                        /* exit Disk Partition */

-Above list taken from: http://blogs.msdn.com/b/habibh/archive/2011/09/14/how-to-create-a-bootable-usb-flash-drive-to-install-windows-8-developer-preview.aspx -

Leave the DOS Window open; if you got a list during the "format" command simply double click "my computer" and right click on the USB Drive and select "format" when the menu comes up select "NTFS" for the file system, allocation size is default, and check the "quick format" box, once that is done go back to the DOS Window/C: prompt...

now you type in "CD BOOT" and hit enter, you should see this:

C:\boot>

now type in bootsect /nt60 E:                              /* where E: is the drive assigned to the USB

Now you can close the DOS box

Depending on your windows version all you have to do now is extract the ISO file onto the disk drive;
DO NOT FORMAT IT AGAIN OTHERWISE IT WILL NOT BOOT FROM THE HARD DRIVE!!!
I use WinRAR and simply right click on the WIN10ISO File, select "extract..." and select the E: drive -or which ever is your USB Hard Drive-

That should get you going...

If anyone has anything else or perfected methods please feel free to add...

doug robbins - mvp office apps & services (word) say:
I have made a form on Microsoft Word and in some of my text form fields I have put them to a date setting. When I email this form to a client and they come to a date form field I need them to only be able to type in a date that contains day,month and year not a date that only contains month and year or even just year. Is this possible and if so how do I do this. Thank you


Given that you are emailing the form to your clients, it is almost impossible to achieve what you are after for the following reasons:

The Date Picker to which Heather makes reference will only be available if the clients are using Word 2007 or later.

A macro such as the following could be run on exit from the formfield, BUT, you cannot be sure that the client's macro security settings will allow it to be run, particularly as the document is being received by email.

However, if you run the following code on Exit from the TextBox, it will check what has been entered into the TextBox to determine if it is a valid date and if it is, it displays a MsgBox give the date in the format dd mmmm, yyyy.

TheDate = TextBox1.Value

If Len(TheDate) = 10 Then

    If Mid(TheDate, 3, 1) = "/" Or Mid(TheDate, 3, 1) = "-" Then

        If Mid(TheDate, 6, 1) = "/" Or Mid(TheDate, 6, 1) = "-" Then

            Daynum = Val(Left(TheDate, 2))

            MonthNum = Val(Mid(TheDate, 4, 2))

            YearNum = Val(Right(TheDate, 4))

            GoTo Check

        End If

    End If

ElseIf Len(TheDate) <> 8 Then

    MsgBox "You must enter the date as ddmmyyyy."

    GoTo Done

End If

Daynum = Val(Left(TheDate, 2))

MonthNum = Val(Mid(TheDate, 3, 2))

YearNum = Val(Right(TheDate, 4))

GoTo Check

Check:

On Error GoTo Invalid

Serial = DateSerial(YearNum, MonthNum, Daynum)

Dim Mon$(12)

Mon$(1) = " January, ": Mon$(2) = " February, ": Mon$(3) = " March, ": Mon$(4) = " April, ": Mon$(5) = " May, ": Mon$(6) = " June, ": Mon$(7) = " July, ": Mon$(8) = " August, ": Mon$(9) = " September, ": Mon$(10) = " October, ": Mon$(11) = " November, ": Mon$(12) = " December, "

MsgBox Str(Daynum) & " " & Mon(MonthNum) & Str(YearNum)

GoTo Done

Invalid:

MsgBox "That is not a valid date.  Please re-enter it in the form ddmmyyyy"

Done:
