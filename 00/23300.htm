EXCEL 2010 - Book1

hansv mvp say:

If you want to change the blank new workbook that is opened when you start Excel or press Ctrl+N, modify the blank workbook to suit your needs, then save it as a Microsoft Excel template named Book.xltx (this name is obligatory) in your Excel startup folder. By default, that is

C:\Users\<username>\AppData\Roaming\Microsoft\Excel\XLSTART

AppData is a hidden folder, so you must show hidden files, folders and drives in Windows Explorer to see it.

Also create a version of the workbook with just one worksheet, and save it as a template named Sheet.xltx in the same folder.

It will then be used when you insert a new sheet into an existing workbook.

Seamless Animation Loops - Microsoft Community

mestompu say:

Seamless Animation Loops

I am using the swivel animation to continously loop an object in a slide until the slide ends. The problem I'm having is once the object repeats the animation it is not seamless causing a jump in the animation. Thoughts?
john korchok say:

One of PowerPoint's imperfections. Swivel is not symmetrical on a timeline: it starts differently than it ends. This kind of animation is not a good candidate for looping. It starts with no shape and ends with the shape at full width. So repeats have a jarring transition at the end of each loop.

Feel free to report the bug by clicking on the small smiley-face icon in the upper-right corner of the program window. Choose Tell Us What Can Be Better and describe the problem. submitting will send the report to the Mac programming team.

That said, I doubt they'll do anything about this. This is an Entrance animation that is intended to introduce a shape, then leave it on screen. It just doesn't work when repeated, and I don't think that's an accident. It would be nice if they included an extra option to end the loop then the shape disappears rather than when it is at full width. But I think you just need to choose a different animation that works better when looped.
