alanwu_msft say:

hi edward,

we have 3 different types regarding the retention policy tags. one is personal tag, one is the entire mailbox tag, and the other is the default folder tag.

for the personal tag, we can manually assign to a single folder like what you have mentioned for the sub folders under the inbox folder.

for the default folder tag, which is used to automatically applied to the default folder types in the mailbox, like inbox folder, deleted items folder, etc.

for the entire mailbox tag, it’s used for the entire mailbox automatically.

 

back to the case, given the information above, we’re not able to manually assign the retention tags to the inbox folder, since it’s a default folder type. if you want to control the retention settings for the inbox folder, you can go to eac (exchange admin center) of office 365, then create a new retention policy tag which will be applied to the inbox folder, then add the tag into the retention policy and assign it to the user mailbox. please note that the action will need to involve our office 365 admins.

That said, Watermarks are still intended for use in printed documents (and pdf).
