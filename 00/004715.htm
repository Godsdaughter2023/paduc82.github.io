{Sigh}  No Problem!  We'll have to do this the hard way!  Make sure you have your Office CD and Product Key handy, you're going to need them:

1) Quit all Microsoft applications.


2) Track down all instances of pre-2008 Normal template on your computer,

and drag them to your desktop. The file is called simply "Normal" and has

no extension.


3) Find and drag the file Normal.dotm to your desktop. Unless you have

moved it, it should be in

/Users/ ~ /Library/Application Support/Microsoft/Office/User Templates/


4) If the following files exist, Remove or rename them:


~/Library/Preferences/Microsoft/Word Settings (10)


~/Library/Preferences/Microsoft/Word Settings (11)


User/Library/Preferences/com.Microsoft.Word.plist


User/Library/Preferences/Microsoft/Office 2008 (the whole folder!)


5) Then Repair Permissions with Disk Utility.


6) Then re-install, and reapply the 12.1.0 and 12.1.5 updates, in that order.


7)  Repair permissions again, and it should work.


Be thorough with this, if you leave any of the old files behind Word 2008

will find them and won't replace them. Do it right and Word will construct

a new, clean, set of preferences and everything should now work.

steve rindsberg say:

This came up for another user just the other day ... same thing, I think.  Letters piling up atop one another.

Try this:

Open the problem file.

Select the text in question.

On the Home tab, check to see what font PPT says it is.

Close the file, quit and restart PowerPoint, specifically by clicking the Windows Start button and typing POWERPNT ... that will show the PowerPoint desktop app as the best match. Click it to start PowerPoint.

Choose to start a new blank file, type some text into the title placeholder, select the text then on the Home tab | Font group see if you can choose the same font as you're having trouble with in the problem file.

If not, it means that the font isn't installed on the computer, or that it's a font type that PPT doesn't accept. 

In that case, you'll need to use a different font in the "problem" files or find a TrueType or OpenType/Truetype version of the font to install.

jaysnelson say:

I found the issue that was causing it for me. 

There's a feature with OneDrive to allow for better collaboration between people using the same office file. I turned that feature off in the OneDrive settings and I no longer have the issue. 

Go into the settings for OneDrive (right-click the cloud icon on your task bar, choose settings). On the "Office" tab, uncheck "Use Office 2016 to sync Office files that I open".
