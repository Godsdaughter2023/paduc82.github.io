Just to clarify: A Table cannot have "two primary keys" ([Highlander]There can only be one![/Highlander].) It can however have a single primary key which consists of two - or three, or even ten - fields, which jointly must be unique.

How to find the size of each slide in KB

Is it possible to find the size in Bytes or KB of each slide in a presentation using VBA?

 

 

shyam pillai say:

Hi,

There is no built-in way to get this information. You would get an approximate value by exporting individual slides out to disk and then reading the file size on disk

 

Export Slides within a Presentation as files

http://skp.mvps.org/ppt00036.htm#2

 

Each slide is saved to disk using the PublishSlides method in 2007/2010.

'Control individual slides export
Dim oSld As Slide
For Each oSld In ActivePresentation.Slides
     Call oSld.PublishSlides("c:\temp\", True, True)
Next
