Problem with different versions of Word Documents - Microsoft Community

jonathan713tan say:

Problem with different versions of Word Documents

I have a problem with different versions of Word; 2007 and 2010. When a document is saved in Word 2007 and re-open again in Word 2010, the alignment of the wordings and pictures are out. Anyone has a solution for this?
rohn007 say:
In theory that shouldn't be happening.

Assuming you are working on 2 separate machines, are they running the same version of Windows? 

Actually, the important thing to confirm is that both machines are using exactly the same version number for the Printer Driver.  Word takes a lot of subtle, and not so subtle formatting keys from the printer driver.  If they are using different printers, or even different version numbers of the same printer driver, that could cause your problem.

Another thing to confirm is that you are using the same NORMAL.DOTM file. Copy it from one machine to the other (preferably from 2007 to 2010.  Settings in the normal.dotm could affect the layout.
