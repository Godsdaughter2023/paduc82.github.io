Excel 2010 - Create a Calendar in Worksheet - Microsoft Community

jgreg7 say:

Excel 2010 - Create a Calendar in Worksheet

I am trying to create a record keeping form that displays a three month calendar below a set of checkbox criteria.  The form is printed and used to check off activities based upon the checkbox criteria.  The idea is that days could then be simply circled (in pen) to indicate the date the checkbox criteria occurred on.

 

The template would then be set so that a single input "start" date would generate the three-month calendars, and the form could be printed. 

The general idea is to be able to generate the latest form by simply inserting the start date.

 

The format of the calendar is intended to be similar to any normal printed calendar (e.g. weekdays across the top row, with the weeks on rows below, one month per view).  I would use three of these side-by-side for the monitoring period.

 

Is there an easy way to do this?

 

 

bernie deitrick say:

Calendars are easy - the tricky part is putting the first day into the first displayed Sunday.

 

Say you want a start date of 1/1/13. Enter that into cell A1. Then in the first cell of the calendar - the Sunday anchor - let's say that will be cell A3) - use the formula

=A1 -IF(WEEKDAY(A1 + 6)=7,0,WEEKDAY(A1 + 6))

 

In the cell to the right of that (B3), use

=A3+1

and then copy to the right for 7 columns.

In the cell below (A4), use

=A3+7

and copy right and down for as much as you want to show.

 

Then format the cells the way you want for date (usually mmm dd) and base color.  Then apply CF, with the custom formula

=MONTH(A3)/2 = INT(MONTH(A3)/2)

and pick a color - the odd months will retain the base formatting, and the even months will get your new color. (change the A3 to the first of the selected cells when you apply the CF).
