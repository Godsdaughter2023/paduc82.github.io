Hello Scheduler84,

If you created the custom calendar in one file, you can most certainly copy that custom calendar to the 'global' template file.  Once in the global template, it is available in every project file created on your computer from that point forward.  To copy the custom calendar:

  1. Open the project file containing the custom calendar.
  2. Choose Tools > Organizer to open the Organizer dialog.
  3. Click the Calendars tab.
  4. On the right hand side, select your custom calendar in the list.
  5. Click the copy button to copy the custom calendar and add it to the calendars in the Global.mpt file.
  6. Close the Organizer dialog.

scottgem say:

Ok first problem is your WHERE clause. A WHERE clause has to parts, the field being compared and the value being compared to. You only have the value being compared to.

So you should have:

WHERE Tag <> "Snag" AND Plot = Forms!View_Datasheet!Plot

I'm guessing the fieldname you want to filter for is Plot.
