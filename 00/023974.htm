print multiple envelopes in Word (not mail merge) with same - Microsoft Community

7031cc say:

print multiple envelopes in Word (not mail merge) with same delivery address

I want to print multiple envelopes with the same delivery address so it looks neat, and not just a label stuck on. I can't find anyplace that will have that as an option.

I have to type the address each time for each envelope.

jay freedman say:
The quick-and-dirty way is to click the Add to Document button in the Envelope dialog, and then use the regular Print command where you can set the number of copies you want. Since the Add to Document button creates the envelope as a new Section 1 followed by the rest of whatever document was open, you need to enter s1 in the Pages box of the Print dialog to avoid printing the rest of the document.

A better approach is to use Graham Mayor's Envelopes add-in from http://www.gmayor.com/EnvelopesAndLabels.htm.

suzanne s. barnhill say:
In addition to what Jay has said, if you're creating an envelope that you will use repeatedly, you can use Add to Document to create the envelope, then remove the attached document and save the envelope alone for reuse. See http://WordFAQs.mvps.org/DetachEnvelope.htm

wdt_935 say:

Private Group on Yammer Disappeared

I created a private group under my company but the group name has disappeared from the navigation pane.  I first thought I had accidentally deleted it, but when I try to recreate it under the same name, it says it already exists.  What can I do to find it again and restore it to the navigation pane so I can continue to access/administer it?  There is currently only 1 other member of this group.

a. user say:

Hello,

 

Thank you for contacting us.

 

You can find the group is still there on your home network by following below steps:-

  1. Login into yammer network.
  2. Click on three ellipses next to Home, message and notification icon
  3. Select Group> All Groups
  4. That group name under the list.

 

Or you can try below steps to find the group details by following below steps:-

  1. Login into yammer network as a verified admin
  2. Click on Settings> Network Admin> Export data
  3. Click Save.

 

In the export zip folder you will find a file as Groups.csv in that you will find group id, group name, created_by_id, created_at and updated_at.

Unfortunately we cannot restore once its deleted, but you can create a new group with same name.

 

If you are a verified admin of yammer network and the group is showing active in the export group file than you can delete the group by following below steps and can try to recreate the group.

  1. Login into yammer with verified admin rights
  2. Now click on Setting> Network Admin> Content Mode
  3. Select the check box in Private content mode
  4. Than select that group which you want to delete
  5. Select setting top right
  6. Now there is option Delete this group.

 

You can refer below link for reference:-

https://support.office.com/en-us/article/Monitor-private-content-in-Yammer-verified-admins-b5271fe7-62aa-4dc4-a1d7-b2b0bd23edae

 

Kindly let me know if the information provided was helpful.

 

Regards,

Sushil Dhiwa
