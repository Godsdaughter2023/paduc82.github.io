Windows 10 vs. Windows 7; concerns about the direction Microsoft - Microsoft Community

john r moffett say:

Windows 10 vs. Windows 7; concerns about the direction Microsoft is going in

I have concerns about the direction Microsoft is taking the Windows operating system. Windows 8 was not a viable desktop operating system, and the speed at which Microsoft rolled out Windows 10 as a replacement makes it apparent that they got the message from their customers. I would like to start a productive discussion about how Microsoft could make improvements to Windows 10 that would make me and millions of others willing to update to the new version. Hopefully Microsoft will get the message again. Here are some of my main concerns that I know are shared by many other Windows users. 

 

1. Windows 10 is not as attractive looking as Windows 7. It is just too flat and plain looking. The interface alone is enough to make me avoid Windows 10.

2. Users can’t change how it looks into something they might like better due to reduced customization options. If you like a 2D, flat look then you won’t mind, but I much prefer Aero over the current Metro/desktop hybrid.

3. Many users have complained about updates being forced on them, especially because some have actually made computers unusable. We should be allowed to control what we download and install if we want to.

4. The Metro (winrt) paradigm is slow in many areas, such as launching so-called apps, which are distinct from real programs or applications.

5. Speaking of apps, they were originally intended for phones. There is no reason to design a computer operating system around apps. But of course MS is interested in selling apps through the app store built into the start menu, so that is why they have spent a lot of effort on making Windows 10 app-centric. This is a big downside as far as I am concerned. The limited functionality of apps makes them useful for small, limited devices, but it makes no sense on a real desktop computer.

6. Windows 10 has more data collection/transmission (telemetry and tracking) than all other versions put together. I don’t want to use something that’s chewing up my bandwidth for activities that do not benefit me. Microsoft needs to be very upfront about all of the telemetry and tracking features, what they do in detail, and allow users to opt out after reading the detailed descriptions. If it all is truly for fixing problems, great, but that does not seem to be the only reason for the telemetry and tracking. But of course we don’t really know because Microsoft is being very vague on the subject.

7. I’m not allowed to control or even know anything about the vast majority of this collection/transmission process, and this reduces trust.

8. I don’t like being advertised to or given suggestions for things to buy in my operating system. Offer a store and let me choose to use it or not. Any store built into Windows needs to be something I have to open, not something on the start menu and open by default.

9. Many things that I use in Windows 7 have been removed or replaced with less capable versions. Things like WMC and DVD maker.

10. Microsoft’s take it or leave it attitude where they are going in a different direction (mobile computing/app store OS) and there isn’t anything the customers can do to change that, despite the fact that the vast majority of the Microsoft user base use desktop computers or laptops without touchscreens. Windows 10 is still not as good on desktop computers as Windows 7, so unless you have an urge to have a new buggy operating system, there isn’t any compelling reason to switch. However, MS is making the upgrade harder and harder to avoid if you leave automatic updates turned on in Windows 7. 

11. Microsoft's inability to recognize the sheer number of people who want a choice is a problem, for example the Aero interface, a choice that MSFT could easily provide in a very robust way. Why didn’t they improve on Windows 7 by offering both modes and themes? You could have the desktop mode with Aero themes and all the customization options in Windows 7, or the Metro mode for mobile devices, with its own themes. Users should have endless options for tweaking and changing every aspect of how Windows looks. Microsoft should be expanding choices, not limiting them.

12. Upgrades that do not respect my choice of default programs. I set my PC up the way I want it, not the way MSFT dictates. There are more than a dozen locations on the MSDN library where MSFT explicitly warns developers about respecting the User's choice. Take a peek at ShutdownWindowsEx or BroadcastSystemMessage. So why doesn’t MSFT respect my choice of browser?

13. Windows 10 brings me nothing that I can’t already do with Windows 7. I still use Windows 7 Ultimate, and Windows 10 pro seems like a big downgrade for me. I don’t see anything that would make me want Windows 10. At best it does what I already do like open my programs. At worse it looks like it was designed for small mobile devices and focuses too much on apps. There is not a single thing in 10 that will help me be any more productive, allow me more entertainment, or be any more fulfilling on my PC. Just because it’s the latest version does not make it better, just newer (and buggier).

14. Windows 10 has undergone surprisingly little change during its lifecycle, at least as far as most users can tell. Less than most other versions in the same timeframe. Windows 10 today looks, acts, works almost exactly same as it did last January and it’s probably how it will be next January. Maybe it is a bit more stable, but forced updates still cause problems for lots of users by breaking things that were working.

15. Adding a phone based operating system to a desktop one was a poor design decision, and it is not too late to have two independent operating system development paths, one for desktops and one for mobile devices. Whoever thought it was a good idea to mash them together was just plain wrong.

16. It appears that Windows 10 was designed to run on older hardware and low spec systems. That’s great if you are running it on a phone or have an XP laptop from a decade ago, but what’s the motivation now for hardware manufacturers to push out faster products? Very powerful computers are becoming more affordable every year, but now Windows doesn’t need any of that increased horsepower.

17. The default Microsoft online account. I don’t like that at all. I know you can shut it off, but I shouldn’t have to do that.

18. Windows 10, like it’s even more problematic predecessor, Windows 8, is an experiment, and the users are guinea pigs. The forced updates take virtually all control away from users, and so far the security rational for doing this does not seem to be panning out. Many Windows 7 users who don’t want the upgrade are shutting off automatic updates to avoid a forced Windows 10 installation. That makes for a poorer security situation, not an improved one.

19. Very sparse information on what the OS is doing at any given time. Some people don’t care what their OS is doing as long as they can play Angry Birds. Others care a lot because they are technically oriented tinkerers. People who wanted their computer to be like a reliable toaster bought Macs, people who wanted a computer they could endlessly customize and tweak bought Windows computers. Not so much anymore. Microsoft is being very tight lipped about many updates, for example, a number of recent updates for all versions of Windows only say “an update to Windows Update”, or “makes the transition to Windows 10 smoother”. They might as well have said: “it’s supercalifragilisticexpialidocious!” Lots of other aspects of the OS functionality are hidden completely from users. There need to be options for getting more information about what the OS is doing, and why. Many of the concerns over Windows 10 are due to this opaqueness on Microsoft’s part. A much more open and honest explanation of various updates, and functions such as telemetry and tracking, would have gone a long way toward heading off much of the backlash against Windows 10.

20. I don’t use Microsoft browsers. I use Opera, which is based on Chrome. I don’t care about Edge. I also don’t want or need Cortana because I already have Dragon Naturally Speaking Medical version which does what I need. I don’t use apps on my desktop computers, so I don’t need the app page or app store. I realize others might want some of these features, but they are not an improvement over Windows 7 for most desktop users who have different software preferences.

21. Lack of a paid-for Windows 10 Ultimate version (like the Enterprise version) is another glaring issue. Until MS comes out with an ad/app store/Metro/telemetry/tracking/forced updates-free consumer version that I can buy, I will be sticking with Windows 7 Ultimate. Microsoft would solve almost all of their problems by updating Windows 7 Ultimate with new features and improvements, including improved security features, and releasing it for sale at the original price ($299). This would allow people like me to happily pay and upgrade, and it would be another nice revenue stream. As it is now, they are offering me nothing that I would want, and lots of things that I really, really don’t want, in Windows 10 Pro. Maybe if they make a Windows 10 Ultimate version that consumers can buy, I will have another look. But right now, I have made sure all of my computers do not upgrade to Windows 10 automatically.

 

It is not about going forward or backward, it is about giving users more choices, more information, more control, and more respect. That’s it in a nutshell.
