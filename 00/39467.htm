rob koch say:

The WannaCrypt ransomware attack is simply one malware attack leveraging the known Server Message Block 1.0 server (SMBv1) vulnerability that was patched via the MS17-010 Security Update released by Microsoft on March 14th.  Any supported Windows system that's been properly maintained with updates should have installed the appropriate patch moths ago, but here's the official support article providing links to all individual update packages.

MS17-010: Security update for Windows SMB Server: March 14, 2017

The patch is NOT for WannaCrypt itself, since this malware is constantly evolving using other exploits.  So keeping all Windows operating systems and other applications updated using their respective update systems is always important, regardless  of which specific vulnerability or malware might be in the news today.

Since the above patch was only initially released for currently supported operating systems, however, Microsoft chose to provide this for a few older unsupported versions including Windows XP, 8.0 and Server 2003 as well.  The downloads for these additional patches can be found near the bottom of the following article discussing the WannaCrypt attack.

Customer Guidance for WannaCrypt attacks
