teylyn say:

Hello,

you can use an Index/Match combo to do that. 

The first Index/Match looks like this:

=INDEX(B2:D2,MATCH($E$2,$B$1:$D$1,0))

in words: look at B2 to D2 (the Index) and return the cell where B1 to D1 is the same as E2 (the Match)

Do the same for F2 and divide the two. Then copy down.

Note the placement of the $ signs. They are important to lock the Match to look in row 1 even when the formula is copied down and use the values from E2 and F2 for a currency match. The complete formula as in the screenshot is

=INDEX(B2:D2,MATCH($E$2,$B$1:$D$1,0))/INDEX(B2:D2,MATCH($F$2,$B$1:$D$1,0))

mike middleton say:

The XY Scatter chart type requires numerical values for both the X and Y axes.

To have text values for the horizontal X axis, you could use a Line chart type.

Select both adjacent columns of data with text in the left column and numbers in the right column, choose Insert > Chart > Line With Markers.

If you want the text labels of the horizontal axis title at the bottom of the chart, select that axis title, choose Format Axis > Axis Options > Interval Between Labels > Label Position > Low.

If you want markers without lines, click on a data point to select the data series, and choose Format Data Series > Line > No Line.

End Sub


Function dhTrimNull(ByVal strValue As String) As String

On Error GoTo TrimNull_Err

    ' Find the first vbNullChar in a string, and return

    ' everything prior to that character. Extremely

    ' useful when combined with the Windows API function calls.

    ' From "VBA Developer's Handbook"

    ' by Ken Getz and Mike Gilbert

    ' Copyright 1997; Sybex, Inc. All rights reserved.

    ' In:

    '   strValue:

    '       Input text, possibly containing a null character

    '       (chr$(0), or vbNullChar)

    ' Out:

    '   Return Value:

    '       strValue trimmed on the right, at the location

    '       of the null character, if there was one.

    Dim intPos As Integer

    intPos = InStr(strValue, vbNullChar)

    Select Case intPos

        Case 0

            ' Not found at all, so just

            ' return the original value.

            dhTrimNull = strValue

        Case 1

            ' Found at the first position, so return

            ' an empty string.

            dhTrimNull = ""

        Case Is > 1

            ' Found in the string, so return the portion

            ' up to the null character.

            dhTrimNull = Left$(strValue, intPos - 1)

    End Select


TrimNull_Exit:

    Close

    DoCmd.Hourglass False

    DoCmd.SetWarnings True

    Exit Function


TrimNull_Err:

    Select Case Err.Number

        Case Else

            Close

            DoCmd.Hourglass False

            DoCmd.SetWarnings True

            MsgBox "Error #" & Err.Number & "   " & Err.Description

            Resume TrimNull_Exit

    End Select


End Function
