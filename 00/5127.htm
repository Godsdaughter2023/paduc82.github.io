how to number invoices in mail merge - Microsoft Community

marycook_824 say:

how to number invoices in mail merge

We have members that we want to invoice and send renewal notices to. Their names etc are in excel and I am creating a merge file in Word 2010. I would like to have Word assign an invoice number rather than have to type it on each renewal notice.

Page numbering doesn't work, it there another option?

macropod say:

Ideally, your invoice #s would be in the data source, so that: (a) you have a more permanent record of it; and (b) it could be merged from there. Although you could insert a MERGESEQ field or a MERGEREC field into your mailmerge main document and generate sequential numbers that way, each time you do the merge the numbering would re-start at 1. Either that or additional field coding would be required so you could tell it which number to start counting from. The field code for the latter would look something like:

{QUOTE{ASK Offset "What is the starting Invoice #?" \o}{SET Val {MERGEREC}}{=Offset+Val \# 0000}}

The \# 0000 switch controls the number format, inserting up to 3 leading 0s.

Another approach might be to use the member ID (if you have one) plus the year, in a field coded like:

{QUOTE "«Member_ID» - {DATE \@ YYYY}"}

Note: The field brace pairs (i.e. '{ }') for the above examples are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. Likewise, you can't type or copy & paste the chevrons (i.e. '« »') - they're part of the actual mergefields, which you can insert from the 'Insert Merge Field' dropdown. The spaces represented in the field construction are all required.
