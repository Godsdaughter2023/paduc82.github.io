  • The Table tab isn't showing because you have not selected a cell in the table. It appears that Column F is selected which is adjacent to the table but not an integral part of it.
  • However, if the tab still doesn't appear when you select a cell with content in Column A-E it very well could be that it is not a Table, it's simply a range which has been formatted with banded rows to give the appearance of a ledger sheet.
