cindy meister say:

Hi Lark

The TOC functionality in Word relies on bookmarks it creates for the various sections. If a bookmark can no longer be found, you'll get the message you're seeing.

You can try using Ctrl+Shift+F9 to make the TOC fully static text, or Ctrl+F11 to lock the TOC field so it can't update.

But before you do, you should probably compare the TOC with the pages it references to make sure everything is correct. That error you're seeing could indicate some things have changed.

You can also try clicking in the TOC, then pressing F9 to force an update and see if this clears up the problem when converting to PDF.

It's just going to depend a bit on exactly what's wrong...

Cindy Meister, VSTO/Word MVP

short: january build will make authentication attempt if protocl Level is >=smb2.

Workarounds:

1) Change registry in installed january build to HKLM\SYSTEM\CurrentControlSet\Services\LanmanWorkstation\Parameters\AllowInsecureGuestAuth to 1

2) Set protocol Level to smb1 on NAS

3) request authentiction on the NAS
