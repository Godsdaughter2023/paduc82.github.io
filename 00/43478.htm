To block communications from another player on Xbox Live:

  1. From the dashboard, select the Friends tile, or say "Xbox Select Profile".

How can i change the old users name to mine? - Microsoft Community

perezc say:

How can i change the old users name to mine?

I want all the files in my computer to have my name and not the old users account.
diana d say:

Hi Perezc,

• Are you using a new user account now?
• What type of files are you referring to?
• Are you the administrator of the account?

Important: For added protection, you can create a restore point before performing any steps.
1.Open System by clicking the Start button , clicking Control Panel, clicking System and Maintenance, and then clicking System.
2.In the left pane, click System Protection  If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
3.Click the System Protection tab, and then click Create.
4.In the System Protection dialog box, type a description, and then click Create.
“How to create restore point manually” http://windows.microsoft.com/en-us/windows-vista/System-Restore-frequently-asked-questions 

After creating the restore points you may do the following:
Change User account.

Note: You can change the user account on any account on the computer as long as you are logged in as the computer's administrator. If you are not the administrator, you can only change the user account name for the account you are currently logged in as
i.  Click on the "Start" button and then "Control Panel."
ii. While in the "Control Panel," switch to the Classic View by clicking on it in the left pane. Click on "User Accounts.
iii.  Select the account that you want to change. Click on "Change your account name." Click "Continue" when a dialog box appears asking if you are sure. Type in your new user account name and click on "Change Name."
iv.  Notice that the user account now has the new name. After you change your user account name, the name of the user file will be changed also.

“User accounts: frequently asked questions” http://windows.microsoft.com/en-us/windows-vista/User-accounts-frequently-asked-questions 

Change Name of Computer
If you like your user account name and computer name to match, you should change your computer name soon after changing the user account name.

To do this,
i. Click on the "Start" button and then "Control Panel."
ii. Click on "System and Maintenance" and then "System."
iii. In the left pane, click on "Advanced system settings." The "System Properties" dialog box appears.
iv. Click on the "Computer Name" tab. Click on the "Change" button at the bottom of the page next to "To rename this computer."
v. Type in the new computer name in the next dialog box and click "OK."
“Change your computer name” http://windows.microsoft.com/en-US/windows-vista/Change-your-computer-name
