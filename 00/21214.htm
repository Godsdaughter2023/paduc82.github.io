dale howard [mvp] say:

Vagner --

To create and edit enterprise calendars, you MUST USE the Microsoft Project Professional desktop application for this purpose.  You CANNOT do this in PWA.  So, your first step is to install Microsoft Project Professional on your PC. 

Once you have Microsoft Project installed on your PC, you will need to complete the following steps to create the login account:

1.  Launch Microsoft Project.

2.  Click File > Info > Manage Accounts.

3.  In the Project Web App Accounts dialog, click the Add button.

4.  In the Account Properties dialog, enter a friendly name in the Account Name field.

5.  In the Project Server URL field, enter the URL for your PWA (NOTE:  Omit the /default.aspx from the URL).

6.  Leave the Set As Default Account option selected and then click the OK button.

7.  In the lower left corner of the Project Web App Accounts dialog, select the Choose An Account option.

8.  Click the OK button.

9.  Exit Microsoft Project Professional and then re-launch the software.

10.  In the Login dialog, click the OK button.

Now you are ready to create and edit enterprise calendars using the steps you quoted in your previous post.  You start the process of creating and editing enterprise calendars in PWA, but the system hands off the process to Microsoft Project Professional.  Again, I want to stress to you that you CANNOT create and edit enterprise calendars using the PWA user interface alone.  Hope this helps.

john - project say:
Maria,
First of all what version of Project are you using? Is it updated with the latest service pack?

There are various ways to assign multiple resources to a task. In Project 2010 select the task of interest then go to Resource/Assignments group/Assign Resources icon. That will bring up the Assign Resources window. Assign how ever many resources you want for that task. Then select the next task. The Assign Resource window will stay visible until you close it.

If all you want to do is to enter work in days, simply enter that into the Work field (i.e. 2.5d). If you also want it to display as days, go to File/Options/Schedule group and set the "Work is entered in:" to "Days".
