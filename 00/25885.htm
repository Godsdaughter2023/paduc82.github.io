Backing Up OneNote SkyDrive Files - Microsoft Community

roystreet say:

Backing Up OneNote SkyDrive Files

Hello...
   This is simply what I'm trying to do.  I create OneNote files & they are stored on my SkyDrive account so I can also modify them using my mobile device.  I would like to backup the OneNote files on my server as well as on SkyDrive.  I have a planned backup routine, using SyncToy, but I don't know what files to back up & if I needed to recover those files how can I do so?

Thanks!
omer atay say: For most people the fact that the notebook is in SkyDrive and the backups OneNote takes in Backups folder are good enough. If your computer dies, your notebooks are on SkyDrive. If for some reason the notes get corrupted or deleted from a device and is not found in the recycle bin, you have backups. If you still want an external backup, you can backup the Backups folder. Recovering from them are as easy as opening one of them and copying content. Backing up the cache and trying to recover from it is problematic and not recommended.

omer atay say:

Are you trying to backup or are you trying to recover? If you are trying to recover, you can replace the OneNoteOfflineCache.onecache and see what it can recover from it. To do so:

 

  1. Take a copy of your OneNoteOfflineCache.onecache file (in the same place as the backup folder)
  2. Disconnect network so OneNote doesn't try to sync the data in the backed up cache file you will be replacing
  3. Copy the backed up cache file in place of the cache file you have
  4. Boot OneNote
  5. Save any sections you want to recover (File > Save As)
  6. Close OneNote
  7. Switch back to your original onecache file

Then you can open the saved files and copy anything out of them.
