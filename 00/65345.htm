Times to use:

Moving from a 32 bit Windows system to a 64 bit system, or changing languages

Don't want to do an in place upgrade, or you want a fresh/clean install

1. Run Windows Upgrade Advisor - i﻿t scans your PC for potential issues with your hardware, devices, and installed programs, recommends what to do before you upgrade, and whether you can install the 32 or 64 bit version of Windows 7.

jessica l. say:

How to limit standard user account from changing background and screensaver?

I have two accounts set up on the computer - one is an admin account and the other is a standard user account.  Is there any way to limit the standard user account from changing the background and the screensaver?  It is a computer that middle school students use and limiting their access to these features would be very helpful.

Thank you for your help.

david. f say:

Hello, Jessica

Thank you for using the Microsoft Answers Forum!

If your computer is connected to a domain, you should talk to your administrator about setting up a group policy to do this.

If this is a standalone machine, you could change the Local Policy.

This list is a bit long, but it will ensure non-administrators aren’t able to change the desktop or screen saver while still allowing administrators the option.

To change the Local Policy, login to an administrator account and do to following:

·         Go to Start, in the search box type “mmc” without quotes and press enter. You may be prompted to enter an administrator password or to continue.

·         In the Console1 screen, click File and select Add/Remove Snap-In.

·         On the Add or Remove Snap-ins screen in the list on the left select the “Group Policy Object Editor” item and click the Add button.

·         A wizard should come up, click the Browse button, on the screen that comes up click the Users tab and select the Non-Administrators item and press OK, then Finish, and OK again.

·         On the Console1 screen, click the arrow next to an item to expand the item on the tree to the left.

·         Expand Local Computer, User Configuration, Administrative Templates, Control Panel, and select Display.

·         On the list in the middle of the screen, you can select an item to view more information about an items settings and uses.

·         Double click the Prevent changing wallpaper item, and the options screen should come up. Select “Enabled” to disable users from changing the wallpaper.

·         To disallow users from changing the screen saver, you will need to double click the Screen Saver executable name item, set the Enabled option and specify the screen saver executable name. The default Windows Logo screen saver’s file name should be “logon.scr”. You can search your computer for “*.scr” and this should find all the screen savers.

·         You may want to double click the Screen Saver item and set this to Enabled as well.

After that is all done, close out of the Console1 screen, it will ask you if you want to save, you can save this for later reference so you do not have to go through the beginning setup steps again – you will just need to open the file that you save and it will open up so you can just expand the lists and set your options.

You will need to logoff and back on for these settings to take effect.
