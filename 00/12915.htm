amit tandon say:

Formula in cell H2 - Address of G2 Value in Range:

=ADDRESS(SUMPRODUCT((($A$2:$E$10)=G2)*ROW($A$2:$E$10)),SUMPRODUCT((($A$2:$E$10)=G2)*COLUMN($A$2:$E$10)))

Formula in cell H3 - Row no of G2 Value in Range:

=SUMPRODUCT((($A$2:$E$10)=G2)*ROW($A$2:$E$10))

Formula in cell H4 - Column no of G2 Value in Range:

=SUMPRODUCT((($A$2:$E$10)=G2)*COLUMN($A$2:$E$10))

Please Note: The above formulas are valid in case there is only one occurrence of the value (the cell G2 value). In case there are multiple occurrences of the value then the formula can be modified to return the co-ordinates of say the first occurring value.

amit tandon say:

Formula in cell H2 - Row No of Value in Range:

=LOOKUP(2,1/(G2=INDEX($A$2:$E$10,,MAX(INDEX(($A$2:$E$10=G2)*COLUMN($A$2:$E$10),,)))),ROW(INDEX($A$2:$E$10,,MAX(INDEX(($A$2:$E$10=G2)*COLUMN($A$2:$E$10),,)))))

Formula in cell H3 - Column No of Value in Range:

=MAX(INDEX(($A$2:$E$10=G2)*COLUMN($A$2:$E$10),,))

Formula in cell H4 - Address of Value in Range:

=ADDRESS(LOOKUP(2,1/(G2=INDEX($A$2:$E$10,,MAX(INDEX(($A$2:$E$10=G2)*COLUMN($A$2:$E$10),,)))),ROW(INDEX($A$2:$E$10,,MAX(INDEX(($A$2:$E$10=G2)*COLUMN($A$2:$E$10),,))))),MAX(INDEX(($A$2:$E$10=G2)*COLUMN($A$2:$E$10),,)))

Please note that the above formulas return in respect of the last column & last row of the multiple occurrences of the G2 value in the range. There are 4 occurrences of the G2 value (420) and the co-ordinates of the last column / row of its occurrence ie. C5, is returned.

ron sommer mvp 2009-2017 say:

Windows Live Mail can sync the messages back to the server.

I know nothing about Mac Mail, but I imagine that any messages moved back to the server will be downloaded by Mac Mail again.  I would move a few messages to see how you are going to prevent the doubling of messages in Mac Mail.

 

You haven't mentioned how many messages are involved.  Move a few hundred messages at a time.

 

On the old computer, open Internet Explorer, Alt + F, Work offline.

Open Windows Live Mail and do not go online.

Right click on the Hotmail account name in the folder pane, Properties.

Uncheck Include this account when receiving mail or synchronizing.

Now go back to Working online.

Setup the Hotmail account again in WLMail.

After all folders and messages have synced to the new account, copy or move some messages from the old Inbox into the new Inbox.

Do a Send/Receive.

The messages will sync back to the server.

 

After you have synced all messages back to the server, you can remove the old account.

scottgem say:

There are a couple of ways to do this. So I'm not sure what I suggested back then. But the way I would do it now is to set the default values of the two text boxes to:

=DMin("[datefield]","Recordsource")

and 

=DMax ("[datefield]"."Recordsource")

So when the form is opened you have a date range covering the earliest and latest dates. Then all you need is to enter either a start or end date or both.

Then the OK button uses:

DoCmd.OpenReport "reportname",acViewPreview,,"datefield BETWEEN #" & Me.txtStart & "# AND #" & Me.txtEnd & "#"
