rohn007 say:
Since most of us don't work for MS, the Windows Update process is a "black box". But, windows is a big spiderweb. Office applications do share some code modules.

Each update often fixes more than one problem and more than one program.

When you read the summary description, it mentions the main program(s) affected.

As well, if you have installed and uninstalled programs, not all of the support files are always removed.

And, even though the initial Windows update scan indicates you need a specific update, when you do the actual update it sometimes realizes that the update is not really needed, so it is not installed.

The final summary is that:
- updates can patch more than one program, without always detailing all programs being updated
- orphan files from uninstalled files may need to be updated
- not all updates are actually installed.
