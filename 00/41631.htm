Generally, BIOS (basic input/output system) , is a function of the system hardware.  WindowsXP Home Edition doesn't have a BIOS.  Your motherboard does and how it functions is determined by the board manufacturer.  "Flash" utilities, whether run under Windows or as stand-alone programs under DOS, re-write the electronically erasable programmable read only memory (EEPROM) found on your motherboard.  If your board manufacturer, and sometimes the BIOS firmware generator (Award, Phoenix, American Megatrends, etc), has provided a Windows based flash utility, you can use that to change your BIOS.  Limitations to a BIOS utility's functionality are determined by the provider.  You should not update a BIOS casually and unless the newer version has specific new functionality you need.  IF IT AIN'T BROKE, DON'T FIX IT.
