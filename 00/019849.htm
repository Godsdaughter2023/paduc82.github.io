In the end I found a work around.

1. Run Outlook as admin

2. Right click on Inbox and select IMAP folders 

3. Use the Query button then highlight all the folders and subscribe unsubscribe over and over to the folders.

4. Un-tick the checkbox at the bottom of the window, which enables the apply button.

5. Repeat 2-4 times and miraculously Outlook starts working again!
