Hi All,

The issue is with the generic Microsoft wireless adapter driver against the HP Probook 4430s hardware (I presume you all have Probooks with Ralink wifi card, like I have).
The surest way to install any version of 64 Bit Windows 8/8.1 - Pro/Enterprise on ProBook 4430s is to get hold of screw driver and remove Ralink wifi card.
Follow these steps:
1. Unplug charger, power off turn around notebook, the battery release latches towards the battery icons to disengage the battery. (Note, you might want to remove any residue current in your PC by pressing the power button for five-ten seconds with battery removed)


2. Remove transverse P1 screw from battery bay area near the battery connectors. (No it is not same as the M2.5X4. This screw is hidden on the side with a lock symbol near it)

3. Slide the battery release latches towards the battery icons further until the service door disengages from the base enclosure.

4. Pull the wires off the wifi card (Card is located adjacent to HDD and RAM DIMM slots)

5. Remove screws (2 in #) from the card and pull it out. Place it safely.

6. Use insulation tapes to cover free ends of metallic portion of wires you just removed from card to avoid any short circuit.

7. Slide in service door.

8. Put Battery in its place.

9. Clean install 64 Bit Windows 8/8.1 Pro/Enterprise.

10 Pray, if all is well, you will not see BSOD error and complete installation with reboot.

11. Go to device manager and uninstall network adapter drivers (My computer>right click>manage>device manager>network adapters> rightclick on each component and uninstall one by one.

12. Download all HP ProBook 4430s Drivers for your Windows version from HP Website (or atleast Ethernet and Ralink Wifi drivers)- http://h20565.www2.hp.com/portal/site/hpsc/public/psi/swdHome/

13. Install drivers. (if prompted to restart chose restart later.)

Shut down. and repeat steps 1, 2 and 3.

14. Replace your card back into the slot. put screws. Remove insulation tape from wires, & Fix wires on Ralink card. (Wires are numbered 1 and 2 on the plastic tag on them. Match wires with sockets #1 & #2 printed near the sockets on your wifi card)

15 Slide in service door, ensure you also put the service door lock screw. Put battery in place.

Start PC.

Viola !!

you should be up and running with your 64 bit Windows 8.1 pro/enterprise.

Regards,

AJ

Hi AnandJ & AJ,

I was able to bypass (avoid) the physical removal of the card as suggested by following something like this procedure below (very rough from long ago memory):

1) Bios disable WiFi

2) Clean install of windows 8.1 and all updates

3) Download and locally expand latest HP RaLink WiFi driver from website

4) Right mouse click --> install on the inf file. I think this pre-populates the driver database (or just copies the driver definition to c:\windows\inf etc).

5) Bios Enable WiFi

6) boot to Safe mode (or diagnostic mode or other non driver loading mode of windows GUI, what ever it is called now...some key combination before windows loads)

7) From device manager, software disable the WiFi card

8) Restart to normal windows (sadly ironic !)

9) From device manger install the HP driver manually by pointing to the inf file on the local driver expansion.

10) the RaLink device loads and enables and all is good

11) reboot again to check.

From my testing in Dec 2013 there was also a Windows Update for the RaLink card which also broke a good system. Do not do this update if it still exists. Keep the HP driver (RaLink 5.0.37.0).

Good luck,

Andre

No, you cannot use it, because it is an OEM license that is forever attached to the broken laptop.

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

You will need to purchase a full version license.

Because Windows 7 is in limited stock because Microsoft ended sales in October 2013, this has created demand because of its scarcity. This means retail boxed copies demand a premium, while OEM System Builder copies remain affordable.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system
