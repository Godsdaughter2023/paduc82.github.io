First, let me point out that no version of Word (nor any other
significant application software) has ever come with any version of
Windows. If your computer came with Word, it's because the computer's
vendor bundled Word with it, not because it came with Windows. And
these days, whenever a vendor bundles Word with a computer, it's
almost always a trial version of Microsoft Office, and the trial will
shortly expire.

and getting a bit frustrated with it. I am scanning documents on my printer and trying to can them into pdf files so they are easier to send to someone. I downloaded adobe,

Adobe is a company, not a product. Which Adobe product did you
download? Perhaps Adobe Reader? Adobe Reader can read pdf files but
not create them.

but I still can't figure out how to scan the documents as pdf files; it is not an option.

If it's Adobe Reader, that's correct. It can not create pdf files. The
only Adobe product that can create pdf files is Adobe Acrobat. But
Adobe Acrobat is expensive, not free. Moreover it can't create a pdf
file from a picture.

Any help in the matter?

Note the word "picture" in my last sentence above. When you scan a
document, you produce a picture. It's a picture of the text, not the
text itself. Adobe Acrobat (or any other pdf-creating program) needs
text (for example from a word-processing program) to produce a pdf
file. A picture of text won't do.

What you need to have is an OCR (Optical Character Recognition)
program. An OCR program processes the picture, recognizes the
characters (text) within it, and produces a text file from it. Some
scanners come with lite versions of OCR programs, but of you don't
have one, you'll need to buy one. Two of the best ones are OmniPage
and Abbeyy Fine Reader. They are also both expensive.

Ken Blake
