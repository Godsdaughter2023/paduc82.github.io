don cuthbert say:

Maybe you are not on a public build. You have to install build 15063 to have a non-expiring build. You can't just quit when using a pre-release build and expect it not to expire. The Digital License of your Device for Windows 10 is permanent. You just need to install a non-expiring, public build.

You can't "upgrade" to a lower build number if you have a build number higher than 15063. You could possibly stay on the build you have until September or so and then upgrade to the Fall Creators update. Otherwise, you would have to do a backup, then a clean install of build 15063, then restore your data and reinstall Apps.

cbarnhorst say: Expiration of the build has nothing to do with Windows licensing.  Windows licensing is permanent for the device.  It is only the build that is time-bombed.  You can reinstall the Creators Update or you can click the "Stop getting previews" button in Settings/Update and Security/Windows Insider Program and choose a method from the list that appears.  You should not be using a primary or production computer for preview builds so there should not be anything lost reinstalling Windows.  If you did not follow instructions and are using previews on your main computer then you should stay with the program until this cycle completes in the fall and stop getting preview builds once the Fall Creators Update releases to the general public (no definite date yet but probably around October or November).  You will be on the new version of Windows at that point and no additional steps will be needed to leave.

paulewills say:  If you are using a desktop & have a spare HDD, you can always dual boot between pre-release builds & the stable public release build.

cbarnhorst say: Actually, making a small partition on a laptop drive works just as well.  Anyway, the OP wants to leave the program.

paulewills say:  Or stay on the slow ring builds which are more stable.
