jay freedman say:
It isn't a bug from Word's point of view -- you just need to know what's going on under the covers.

The section formatting of each section in a Word document is stored in the section break at the end of that section, not at the beginning of the section. (This seemed logical to someone at MS when Word was first designed, and it's never changed despite the years of confusion that it causes.)

When you insert an Even Page section break, the second section is set to start on an even page -- but that information is stored in the hidden section break that's part of the final paragraph mark at the end of the document, not in the section break you added (which now contains the information for the layout of the first section).

When you then delete the section break between sections 1 and 2, the section info at the end of the document applies to the whole document. It includes the "start on an even page" setting, so you can only set its page number to an even value, including 0 or 2 but not 1.

To change it back to "normal", go to the Page Layout tab and click the dialog launcher in the Page Setup group. In the Page Setup dialog, click the Layout tab. Change the Section Start dropdown from "Even page" to "New page" and click OK.

For more info, see http://www.word.mvps.org/FAQs/Formatting/WorkWithSections.htm.

Modifying Default Audio Playback Settings - Microsoft Community

elearn partners say:

Modifying Default Audio Playback Settings

Is there any way to change the default audio playback settings in PowerPoint (Audio Tools [contextual tab] > Playback) so that "Hide During Show" is checked instead of unchecked? The Audio Tools do not seem to have an "Apply to all" command, making it necessary to manually check "Hide During Show" individually for each slide in a presentation after inserting an audio file.

Moved from: Office / PowerPoint  / Windows 10  / Office 365 Home

john sr wilson say:

No way I know in the GUI but it is possible to write a fairly simple macro to check all slides and set "Hide During Show" in one go.

Sub testin()
Dim osld As Slide
Dim oshp As Shape
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.Type = msoMedia Then
If oshp.MediaType = ppMediaTypeSound Then
oshp.Select
If Not CommandBars.GetPressedMso("MediaClipToolsHideDuringShow") Then
CommandBars.ExecuteMso ("MediaClipToolsHideDuringShow")
DoEvents
End If
End If
End If
Next
Next
End Sub

Technically it would be possible to write an AddIn to set the default but this is a lot more tricky. I have done this for big clients but they had a budget!
