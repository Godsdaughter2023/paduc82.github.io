Ok, let's start at the beginning. Use auto scheduled tasks because manual scheduling is just a waste of time and effort. Estimate a duration for the task. That duration column is not right there on the entry table for no reason. That first task with that resource looks like about 5 weeks or 25 working days. That's your duration. Go to the resource sheet and make the resource, and the other one as well. Set max units to 20% for the first and 50% for the second. Go back to the Gantt chart. Use the Resource tab and Assign Resource dialog to assign both of them, the first at 20% and the other at 50%. Use the task usage view to check the total work for each resource for each day. If I got this wrong, adjust as necessary. Before doing that, go to task information, advanced and set the task type to fixed duration (the default is fixed units with effort driven off).

dale howard [mvp] say:

Michael --

Really good question with a thorough description.  Thanks for that.

Go to the Resource Sheet view of your project.  Enter the name of the first resource (available 10 hours/week) and set their Max. Units value to 25%.  This means they are available 10 hours/week at a maximum.  Enter the name of the second resource and set their Max.
