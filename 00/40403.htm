This error means "DLL initialization failed".

You may perform a clean boot to verify if any third party application services or startup item is causing this issue. To perform a clean boot we will have to run System Configuration Utility.

To perform clean boot, follow the steps:

1. Click Start, type msconfig in the Start Search box, and then press ENTER. If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

2. On the General tab, click Selective Startup.

3. Under Selective Startup, click to clear the Load Startup Items check box.

4. Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

5. Click Apply and OK.

6. When you are prompted, click Restart.

7. After the computer starts verify whether you can download files.

If your issue is resolved after performing Clean Boot, then follow the steps mentioned in the following KB article to narrow down the exact source:

http://support.microsoft.com/kb/929135

Also, see the section on how to return your computer to a Normal startup mode by following the steps under “Reset the computer to start as usual”.

samuraisshadow say:

Hello Foo Box!

I had a console that did this too. Nothing appeared to be causing the problem, but it may have been because it was an older model.

There are a few ways you can make your console less prone to overheating:

1)  Place it vertically. I've found the temperature in the console isn't as high, this way, as when it's horizontal.

2) Install your games. If the disc isn't spinning inside the drive then it stands to reason that it won't produce as much heat. This sounds like it could work because your console works fine when running Netflix (Which doesn't use a disc)

3) Make sure nothing is on the floor, especially if it is carpeted. Place it on a table, this'll make it cooler. Also, make sure the power brick is not on the floor either.

If none of these appear to fix it, you may have a faulty console. That's what happened to me. If that's the case, you should call Xbox Support, or contact them online, and they'll be able to send you a replacement console.

If you have any further questions, don't hesitate to reply.

Best wishes,

SamuraisShadow.
