1. Type "Account Control" in the search bar on desktop and press "Enter".
2. In the "User Account Control Settings" window, move the slider to the default settings and click "OK".

If issue still exists, follow the troubleshooting steps listed below and check if it resolves the issue.

Step 1: Adjust your Date and Time Settings

Having an incorrect date/time settings can cause the Windows Store not to work. To adjust your date/time settings, follow these steps:

1. Type Date and Time in the search bar and select it from the results.

2. Adjust the correct date and time and exact time zone depending on your region.

Step 2: Disable Proxy Connection

Incorrect Internet settings might also block Windows Store from working.

1. Right click on Start and select Control Panel.

2. Select the "Internet Options" which opens up "Internet Properties" window.

3. Under the "Connections" tab click on "LAN Settings".

4. Uncheck "Use Proxy server" option and check "Automatically detect settings" and click on OK and Apply to

    save the changes.

Step 3:

Reset Windows Store Cache

If issue persists, reset Windows Store to clear the Store cache. Here's how:

1. Go to the search screen and type wsreset
2. Right click and select Run as administrator.
