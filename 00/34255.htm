Windows 10 login password

Why is Windows 10 checking my password BEFORE I press ENTER, and tells me password is invalid?

frederik long say:

This is very frustrating. How can this behavior be changed?

This behaviour is self-inflicted. At some stage in the past, someone automated the logon process for you on this PC. In the meantime you must have changed your password but did not update the remembered password.

Press the Windows key + R, then type control  userpasswords2 and press Enter to get to the screen where you can activate or deactivate the automatic logon process.
