    You may need to install using Elevated Permissions.

    Download the program to the desktop or insert your C/DVD. If an installation initiates, cancel it. Locate the software's installation program in Explorer. Right click the shortcut or installation program you are try to use (e.g. Setup.exe...?) and select Run as administrator.
