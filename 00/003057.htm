Dates and Vlookup...

I have a list of dates in column A.  They are the first date of each month and formated to show only the month and year.  Column B has a list of values that correspond to the month in column A.  Given a specific date (day month and year), how do I find the value associated with that date.

 

Example

May-11 1234123
June-11 35463
July-11 52352
August-11 4574562
September-11 235423465

Given the date 06/14/11 returns 35463, and 8/2/11 would return 4574562.

max, singapore say: One generic way is via this index/match with the advantage
that it'll work regardless whether the return col contains text or numbers 
Assume your data as posted is in A2:B6, real dates in A2:A6, data to be returned in B2:B6
Assume your input date is in C2 (this is presumed a real date)
Place this in D2, normal ENTER to confirm will do:
=INDEX(B2:B6,MATCH(TRUE,INDEX(TEXT(C2,"mmmyy")=TEXT(A2:A6,"mmmyy"),),0))
The TEXT function is used to "reduce" the real dates to "mmmyy" text strings to enable matching/comparison

tushar mehta say:


That's an interesting idea of creating a big version of a sparkline.

To automate the process I selected the selection event (rather than double click).  Put the following in the worksheet's code module:

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Target.Cells.Count = 1 Then AutoBigChart Target
    End Sub

Then, put the following in a standard module.  Once the code below is in place you can either just select a cell to get the big version of the sparkline (if any) in that cell.  You can also invoke the code by selecting a cell with a sparkline and running the makeBigChart procedure.

Option Explicit

Const BigSisRatio As Integer = 5
    'big chart is this much larger than cell size
Function getBigChart(aCell As Range) As ChartObject
    On Error Resume Next
    Const BigSisName As String = "SparklineBigSis"
    Dim aChartObj As ChartObject
    Set aChartObj = aCell.Parent.ChartObjects(BigSisName)
    On Error GoTo 0
    If aChartObj Is Nothing Then
        Set aChartObj = aCell.Parent.ChartObjects.Add( _
            aCell.Left + aCell.Width, aCell.Top, _
            aCell.Width * BigSisRatio, aCell.Height * BigSisRatio)
        aChartObj.Name = BigSisName
    Else
        With aChartObj
        .Left = aCell.Left + aCell.Width
        .Top = aCell.Top
        '.Width = aCell.Width * BigSisRatio
        '.Height = aCell.Height * BigSisRatio
            End With
        End If
    Set getBigChart = aChartObj
    End Function
Sub deleteAllSeries(aChart As Chart)
    Do While aChart.SeriesCollection.Count > 0
        aChart.SeriesCollection(1).Delete
        Loop
    End Sub
Sub setChartType(aSparkline As SparklineGroup, aChart As Chart)
    Select Case aSparkline.Type
        Case xlsparkline:
            aChart.ChartType = xlLine
        Case xlSparkColumn:
            aChart.ChartType = xlColumnClustered
        Case xlSparkColumnStacked100:
            aChart.ChartType = xlColumnStacked100
        Case Else:
            MsgBox "Unknown type of sparkline chart (=" _
                & aSparkline.Type
            End Select
    End Sub
Sub AutoBigChart(aCell As Range)
    With aCell
    If .SparklineGroups.Count = 0 Then GoTo XIT
    Dim aChartObj As ChartObject
    Set aChartObj = getBigChart(.Item(1))
    deleteAllSeries aChartObj.Chart
    aChartObj.Chart.SeriesCollection.Add _
        "='" & .Parent.Name & "'!" & .SparklineGroups.Item(1).SourceData
    setChartType .SparklineGroups(1), aChartObj.Chart
        End With
XIT:
    End Sub
Sub makeBigChart()
    AutoBigChart ActiveCell
    End Sub
