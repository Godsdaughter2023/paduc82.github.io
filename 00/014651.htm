bernard liengme say:

If can be a problem to do something based on formatting like colour. But how about having the macro use the same criteria that is used in the conditional formatting rule. Example: I filled a column with numbers in range 1 to 10. Then I used conditional formatting to make cells with values >4 green. My macro (below) checks for such values when deciding which rows to hide. Note: when hiding or deleting rows programmatically it is best to start at the bottom. best wishes

Sub HideRow()
 LastRow = Cells(Rows.Count, "A").End(xlUp).Row
 For j = LastRow To 1 Step -1
  If Cells(j, "A") > 4 Then Rows(j).Hidden = True
 Next j
End Sub

Then try this:

Plug in your phone
