If Disk Check is having to move files off of sectors to restore the file system then it can take a long time like that, depending upon how full the disk is.

Make sure you have the latest BIOS, Chipset, Display (Video), Sound, USB3, Bluetooth, Network and all other drivers, or else download and install the latest now.
