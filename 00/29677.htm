update error 80004005

a. user say:

Hello,

Error 80004005 usually appears when you're trying to update your PC. This can be

Sometimes restarting your device fixes the issue. Once your device has restarted, do the following:

  1. Press the Windows key + X and choose Control Panel.
  2. Select Windows Update.
  3. Choose Change Settings.
  4. Change the settings for updates to Automatic.
  5. Choose OK.
  6. Restart the device.

If that did not change anything, download the Windows Update Troubleshooter and run it. Select Next and follow the steps in the wizard to find and fix problems with Windows Update.

Note: If you select Save, you need to go to the folder where the troubleshooter is downloaded to after the download is complete, and then double-click the latest wu.diagcab file to run the troubleshooter.

Download the most recent Servicing Stack Update (SSU)

  1. From Start, choose Settings.

  2. Select System.

  3. Click About.

  4. Scroll down to System type.

Download the version of the latest SSU (KB40490111) that matches your version of Windows from the Microsoft Update Catalog.

Restart your device and check for updates.

Let us know how things go so we can further assist you.

Best,

Anne
