Thank you for posting your query in Microsoft Community.

BSOD Error DRIVER_OVERRAN_STACK_BUFFER indicates that a driver has overrun a stack-based buffer.

This is the classic "buffer overrun" hacking attack. The system has been brought down to prevent a malicious user from gaining complete control of it.

Method :

You may check the hard disk for errors.


I suggest you to disconnect all the external devices apart from keyboard and mouse if connected to the laptop.

Before Chkdsk,

a. Press “Windows Key + Q” to open Charms Bar.
b. Type “cmd” without quotes in the search box.
c. On the left pane, right click on the “cmd” option and select “Run as Administrator”.
d. At the command prompt, type Chkdsk /f, and then press ENTER.
e. Press Y when you are prompted to check the disk next time that the system restarts.
f. Close all applications, and then restart the computer. Note During the restart process, Windows checks the disk for errors and then starts.
g. After the computer has restarted, try installing again.

Warning: Microsoft does not recommend interrupting the CHKDSK process. Microsoft does not guarantee the integrity of the disk if the CHKDSK program is interrupted. For safety you can back up your data before using the check disk command
