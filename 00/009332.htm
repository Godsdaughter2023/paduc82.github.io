Problem with Linear Trendline - Non-Symmetric Fit to Data - Microsoft Community

markbohon say:

Problem with Linear Trendline - Non-Symmetric Fit to Data

I have found what appears to be an error with the linear trendline function in Excel 2010. If it is not an error, I would like an explanation of the behavior as it is not intuitive.

I have a set of data: (A1, B1), (A2,B2), (A3,B3), etc. The A's are measurements made by Technique A at various locations. The B's are measurements made by Technique B at those same locations. To predict the value of A for a given value of B, I have Excel 2010 fit a linear trendline through the (A,B) data plotted in an XY scatter plot. I then use the trendline to predict the value of A for a given value of B. However, the value of A that is predicted for a given value of B depends on whether the B's are plotted on the X axis or on the Y axis! This is illustrated below. In the graph on the left, the B's have been plotted along the X axis. The linear trendline Excel 2010 fitted to the data predicts an A value of 35% for a B value of 29%. In the graph on the right, the same data is plotted but with the B's plotted along the Y axis. The linear trendline Excel 2010 fitted to the data now predicts an A value of 50% for a B value of 29%. It is the same data. Intuitively, I would think I should get the same value of A for a given value of B, regardless of whether the B's are plotted on the X axis or Y axis. Why does the answer depend on whether I've plotted the B's along the X axis or the Y axis?

mike middleton say:

In general, the variable plotted on the vertical axis is the variable of interest or dependent variable, often some type of performance measure. The variable plotted on the horizontal axis is an explanatory variable, i.e., it hopefully explains the variation in the variable of interest. In some situations we try to detect a cause-and-effect relationship, because we think that the variable on the horizontal axis causes the variation observed in the variable plotted on the vertical axis.

When we look at a scatter plot, there are many possible straight lines that we could use to describe the average relationship between the vertical-axis variable and the horizontal-axis variable. If there is any scatter, i.e., if the points are not exactly in a straight line, then for a specific line there will be a vertical distance between a point and the line. This vertical distance is called a deviation (or error or residual).

Excel's linear trendline feature uses the standard method for determining the line of best fit by minimizing the sum of squared deviations. If you plot A (vertical) vs B (horizontal), the vertical deviations are different than when you plot B vs A, and the best-fit line is different. The only situation where you will see "symmetry" is the situation where there is no scatter.

For more discussion of your question, use Google to search for "regression x on y or y on x" (without the quotes) or a similar phrase.

Assuming you are starting with your 1-100 in A1, in B1 try,

=IF(ISNUMBER(A1),IF(A1<75,1,IF(A1<83,2,IF(A1<90,3,IF(A1<95,4,5)))),"")
