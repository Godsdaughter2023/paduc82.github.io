jay freedman say:
To find characters in Unicode fonts such as Arial Unicode MS and Lucida Sans Unicode, go to http://en.wikipedia.org/wiki/List_of_Unicode_characters and use your browser's Find command to look for what you want. For example, you'll find the schwa at U+018F (capital) and U+0259 (lower case). In Word's Symbols dialog, choose the font and set the "from" box to "Unicode (hex)", then type the number (without the U+) into the character code box.

Not all of the Unicode characters listed on the Wikipedia page exist even in the "large" Unicode fonts, and this does nothing for the symbols in the non-Unicode fonts that Charles mentioned. But it's worth a try when you're searching for something.

Another approach is to print out "cheat sheets" for the symbol-type fonts on your computer. Read http://www.word.mvps.org/FAQs/Formatting/FontSampleGenerator.htm (and http://www.gmayor.com/installing_macro.htm if needed).

Start Internet Explorer without add-ons by right-clicking the IE icon on the desktop. Choose Start without add-ons.
or
from Start> Programs> Accessories> System tools> Internet Explorer (no add-ons)
If the problem goes away, an add-on is causing the problem.
