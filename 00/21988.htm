That is a symptom of doing a search from the Address bar without typing question mark space to avoid any ambiguity about whether you are trying to do a search or looking up an Intranet single word host name without using a protocol prefix while also using the option which says that you want to be able to do that (in Options, Advanced, Browsing section). 

To avoid this symptom:  press Ctrl-e when you want to do a search from the Address bar regardless of whether you are going to be typing one word or two.
