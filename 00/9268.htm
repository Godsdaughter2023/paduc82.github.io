That is correct unless the sheets are adjacent, but you don't need to type the formulas, just create them by pointing and clicking, Excel will take care of the sheet names.  You will need to hold down the Ctrl key to select non-touching cells and you will need to type , (comma) when you want to switch to a different sheet.

=SUM(Sheet2!A3,Sheet2!A9,Sheet2!A14)  yes

=SUM(Sheet2!A2:A6,Sheet3!A2:A6)   yes

=SUM(Sheet2!A3,A6,A9)  no, this will get A3 from sheet2 and A6 and A9 from the current sheet.

=SUM(Sheet2:Sheet3!A6:D10) yes if the sheets 2 and 3 are adjacent.  To make this selection type =Sum(  then move to sheet2 and highlight the range A6:D10, hold down the Shift key and click on Sheet3's tab and press enter.

When you upgrade to Outlook.com from Hotmail, you get the new Outlook.com service, and you keep everything about your account intact:

  • Your email address continues to work - no need to change it
  • You will continue to sign-in with your @hotmail.com using the same password

rich007a say:

To be clear, in your .xls file, you want to insert a hyperlink to an existing .pdf file?

If so, select a cell in Excel, then do Insert tab > Hyperlink > Existing File or Webpage (on the left edge of the Insert Hyperlink dialog) > then browse to the file and select it > OK.

Works for me, whether my Excel file is saved as .xls or .xlsx.

 

But, if you want to have a drop-down list, using data validation, that contains a selection of hyperlinks, you can do the following.

Set up a list of URLs in a range of cells, e.g. B2:B10

Select the cell for the drop-down, let's use D2.  Go to Data tab > Data Validation

Set Allow: to List

Set the Source: to =$B$2:$B$10   (i.e. the cells containing the various URLs).

OK.

In another cell (e.g. E2) use the following formula:

=HYPERLINK(D2,"link")

The user can now select the hyperlink from the dropdown, but he/she would have to click on the next door cell which would contain the word "link" to follow the hyperlink.

Hope that helps.
