4. Ran some 3rd party tools against the SSD drive which indicated that the partition table on the drive didn't have the appropriate endpoint hex value.

5. Ran chkdsk <drive:>/r. Finally some luck. This process ran for several hours and repaired almost 1 million files. The drive became browseable through explorer. The directory structure on the drive was restored along with the files (so far I haven't found any missing files).

halima s say:

Hi lulamka momo,

 

1.    Was the printer working fine under Windows 7 64bit earlier?

 

 

As per the Windows 7 compatibility center there is no information about this printer model there is a possibility that this printer may not be compatible with Windows 7.

 

http://www.microsoft.com/windows/compatibility/windows-7/en-us/Search.aspx?l=en-us&type=Hardware&s=Canon%20pixma%20ip1980

 

Method 1

As a work around you may check if the print spooler service is running on the computer.

A print spooler is service that temporarily stores print jobs on the computer hard disk or in memory until the printer is ready to print them.

You must be logged on as an administrator to perform these steps.

a)   Open Administrative Tools by clicking the Start button, and then clicking  Control Panel. In the search box, type  administrative tools, and then click  Administrative Tools.

b)   Double-click Services. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

c)    Right-click the Print Spooler service, and then click  Properties.

d)   On the  General tab, next to Startup type, make sure that  Automatic is selected.

e)   If the service is not already running, under Service status, click  Start, and then click OK. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

 

Why can't I print?

Method 2

If the above step fails then run the troubleshooter available in the below link.

Printer problems in Windows

http://windows.microsoft.com/en-us/windows/help/printer-problems-in-windows

 

I was unable to find any drivers for the printer in the Canon website as per the link below , you will need to contact Canon for more information about the drivers.

volume control icon on toolbar

My volume control icon on my toolbar disappeared - I right clicked the toolbar near the clock in the right corner and proceeded to access properties - however, when I checked the "Notification Tab" the word Volume was "grayed-out" and wouldn't let me "check" it to restore the volume icon on my toolbar as answers to this question we're posted [see below] - I'm still looking for an answer to get my volume control icon back on my toolbar.

Been there, done that.

Right click in the task bar near the time display and you will get a pop up menu.
Click on properties.
Under system icons to always show, check volume.
Click Apply. 

Also, if you don't want inactive icons to be hidden, make sure that box is not checked.
