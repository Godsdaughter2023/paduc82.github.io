considerate_guy say: The timeline contains a different thumbnail picture for each photo or clip that you add to the project.  It's essentially a frame from the beginning of that clip.  I'm not sure which demo video you are referring to (link would be helpful), but I suspect it had lots of pictures or perhaps lots of individual clips.  If you have one long movie (whether it is 2 minutes or 90 minutes) it will be one single clip.  If you want to move parts of it around you'll need to make splits at various points.  The quickest way to do this is to put the playback marker at the point you'd like to split and press 'M' on the keyboard.  Repeat until you have your long clip split as desired.  Then move things around, apply effects, add transitions between parts, etc.

Another possible problem might be the codec used to encode your .avi file.  Apart from fiddling with codecs until you get one that might allow you to edit DviX/XviD video, you might just want to stick with Movie Maker's preferred .wmv format.  Consider converting your other clips to .wmv format using something like Freemake Video Converter.  That should at least get you something rather than gray or black. But you'll still need to split it if you want individual parts; WLMM doesn't do that for you automatically.

Motherboard failure.

Deduced mother board failure when it failed to boot by checking system components, HD, memory etc in another PC.  If I install another motherboard, can I use the Digital Licence Key again, or Do I need to reinstall all my software again. What is the best way as reinstalling the software will mean reregistering Microsoft suite, Visio, Planner, AutoCAD, etc... will take days or will I be best to go back to Windows & Pro.

Thanks

SteveO

 
hairyfool say:

The Windows 10 upgrade program issues a digital entitlement license for the computer. Changing the motherboard invalidates that license. If the new motherboard is fairly close in specification the original disk may boot and be able to sort out the driver changes. If the hardware is significantly different then it is highly likely that boot would hang anyway.

There is no key as such, activation of the installation is automatic and will fail. If you use a key finder the key it finds is effectively a placeholder that tells the system to verify the digital entitlement license, all computers that used the free upgrade program have the same key.

If you can get it to boot into Windows you could try using the call Microsoft option and speak to an operative. In some circumstances they have the leeway to allow reactivation following hardware failure but this is not always possible.
