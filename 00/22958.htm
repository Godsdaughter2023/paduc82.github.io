If you put all of the files in a folder named Test, you could try running a amcro with the following code:

Dim myFile As String
Dim PathToUse As String
Dim myDoc As Document
Dim strPassword As String
strPassword = InputBox("Enter the password to open the documents", "Password")
PathToUse = "C:\Test\"
MkDir "C:\Test\Processed"
SavePath = "C:\Test\Processed\"
'Close all open documents before beginning
Documents.Close SaveChanges:=wdPromptToSaveChanges
myFile = Dir$(PathToUse & "*.doc")
While myFile <> ""
    'Open document
    Set myDoc = Documents.Open(filename:=PathToUse & myFile, PasswordDocument:=strPassword)
    myDoc.SaveAs SavePath & myFile, Password = ""
    myDoc.Close
    'Next file in folder
    myFile = Dir$()
Wend

If you really are using PowerPoint rather than Word, here's a simple trick:

Save a copy of your presentation as a PowerPoint Picture Presentation.

Open the picture presentation version of your PPTX and note that each slide has been converted to a picture.  

Zoom out to 30%

Drag the picture on the slide to the right until the picture's LEFT edge snaps to the right edge of the slide.

Drag the middle selection handle on the RIGHT edge of the picture to the LEFT until it snaps to the slide's LEFT edge.

Your slide (or at least the picture of it) is now reversed.

Print away.
