It's rather simple if you are booted into Windows. You go into disk management and attach the vhd you want to restore. When I bought Windows 7 it came with a 32 Bit and a 64 Bit disc but only one Product Code. Ie. The License covers you for both 32 bit and 64 bit. Which you install is up to you or what your computer can handle.

At anytime you can change from 32 Bit to 64 Bit but you will have to do a complete reinstall of Windows 7 using the 64 Bit disc, otherwise know as a clean install. This will delete everything on your hard drive, so back up all you data first. If your computer has your OS installation set-up on a partition instead of discs, the partition will not be deleted.

If you have a 32 Bit OS and you "UPGRADE" to windows 10 you will still have a 32 Bit OS. The only way to change to a 64 Bit Windows 10 is to download the Windows 10 64 Bit ISO and do a clean install.

As I said at the beginning, your license covers you for both 32 Bit or 64 Bit, so it should not cost you anything to go from 32 Bit to 64 Bit.

However, if you go from Windows 7 Home 32 bit to Windows 10 Pro 64 bit, there will be a charge. This is because you license only covers you for a Home version. It does not cover you for a Pro version.

Depending on the cost, it might be worth it to get the Windows 10 Pro.
