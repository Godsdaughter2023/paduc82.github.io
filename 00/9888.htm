jay freedman say:

Yes, because a docx file is really a zip file, you can open it up and see what's inside.

If you have a separate zip program such as 7zip, you can use the Open With command on the docx file's right-click menu and select that program. Otherwise, make a copy of the docx file and change its extension to .zip, and then double-click the zip file to open it in File Explorer.

Navigate to the \word\media folder inside the docx\zip file. You'll find a list of all the graphics in the document (although all the names will be "imageXX.yyy") and you can see their file sizes. Extract particularly large ones to disk as separate files and open them to see what they are. Then you should be able to locate them within the document.

        Case "4"
            LastDateofSelectedQuarter = LastDateofQ4

    iCountWeek = 0
    iCountWeekCounter = 0

    Do While iCountWeek <= 91

        RollingWeek = FirstDateofFirstWeekNumber + iCountWeek
        StartDateofWeek = RollingWeek

        iCountWeek = iCountWeek + 7
        iCountWeekCounter = iCountWeek / 7

        FinishDateofWeek = StartDateofWeek + 7

       'Exit Loop if the next month is started
        If FinishDateofWeek > LastDateofSelectedQuarter Then
            Exit Do
        End If

        'Gustav Brock Code

        WeekNumber = ISO_WeekNumber(StartDateofWeek)
        WeekYearNumber = ISO_WeekYearNumber(StartDateofWeek)

        Me.Controls((iCountWeekCounter) & "_Label").Caption = WeekYearNumber
        Me.Controls((iCountWeekCounter) & "_Range").Caption = RollingWeek

    Loop

Done:
    Exit Sub
ErrorHandler:

gustavbrock say:

First a note, as this won't work as intended:

Dim LastDateofQ1, LastDateofQ2, LastDateofQ3, LastDateofQ4 As Date

because only the last variable will be declared as Date; the preceding variables will be Variant. Thus use:

Dim LastDateofQ1 As Date, LastDateofQ2 As Date, LastDateofQ3 As Date, LastDateofQ4 As Date

Then, weeks are normally not related to quarters, so you must decide for a rule:

  • If quarter is, then filter on the quarter of the dates, and then find the week number. This will, in most cases, cause the first and the last week to have data for less than seven days and make them also to appear in the next and the previous quarter respectively.

  • If week is, you must calculate which quarter a week "belongs" to in the same way a for the year; the quarter of the date of Thursday is the quarter of that week.

Then you should be able to modify your report.

Public Function ISO_DateOfWeek( _
  ByVal intYear As Integer, _
  ByVal bytWeek As Byte, _
  Optional ByVal bytWeekday As Byte = vbMonday) _
  As Date

' a year according to ISO 8601:1988 standard.
'
' Notes:  Years less than 100 will be handled as
'         two-digit years of our current year frame.
'         Years less than zero returns a zero date.
'         A weeknumber of zero returns the requested
'         weekday of the week before week 1.
'
' 2000-12-17. Cactus Data ApS, Gustav Brock.

  ' the first week of year intYear.
  Const cbytDayOfFirstWeek  As Byte = 4
  ' Number of days in a week.
  Const cbytDaysOfWeek      As Byte = 7
  ' Month of January.
  Const cbytJanuary         As Byte = 1
 
  Dim datDateOfFirstWeek    As Date
  Dim intISOWeekday         As Integer
  Dim intWeekdayOffset      As Integer
 
  ' No specific error handling.
  On Error Resume Next
   
  If intYear > 0 Then
    ' Date of fourth of January in year intYear.
    datDateOfFirstWeek = DateSerial(intYear, cbytJanuary, cbytDayOfFirstWeek)
   
    ' Weekday of requested weekday.
    intISOWeekday = Weekday(bytWeekday, vbMonday)
    ' Calculate offset from requested weekday in first week of year intYear.
    intWeekdayOffset = intISOWeekday - Weekday(datDateOfFirstWeek, vbMonday)
   
    ' Date of requested weekday in first week of year intYear.
    datDateOfFirstWeek = DateAdd("d", intWeekdayOffset, datDateOfFirstWeek)
    ' Date of requested weekday in requested week of year intYear.
    datDateOfFirstWeek = DateAdd("ww", bytWeek - 1, datDateOfFirstWeek)
  End If
 
  ISO_DateOfWeek = datDateOfFirstWeek
 
End Function
