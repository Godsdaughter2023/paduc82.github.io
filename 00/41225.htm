Safe Mode is a Diagnostic Mode of Microsoft Windows Operating System running with minimal set of services and drivers.

You can configure Windows to not install any device drivers at all - you will be in charge of keeping your device drivers up to date.

Make your system Device Installation Settings look like this:

Here's how:

https://support.microsoft.com/en-us/kb/2500967

If you don't pay attention to the Windows Updates you are offered or just choose to blindly install all of them you could end up with a system that used to work fine and then suddenly it doesn't have sound, video, network or printers that work (for example). 

A common complaint in the community after Patch Tuesday is "Windows Update broke my computer!" and the problem is often that folks just blindly let MS update install everything that comes down the line and that can include the wrong drivers for your devices.

If you see a device driver offered in Windows Update you also can choose to hide it but I would just configure Windows to never install device drivers and then you can be responsible for keeping things up to date by visiting the WWW site of the device manufacturer.

If you find that a Windows Update has afflicted one of your devices you may be able to "roll back" the update from Device Manager which will restore the previous version of the driver and the hide that update so you don't see it again:

There are lots of driver updater programs that will scan your system and tell you what they think you need and sometimes they will also suggest the wrong drivers or want you to pay them to get what they think are the correct drivers - probably not worth it since you can do that yourself.

Some folks think those programs are great and some folks don't like/trust them at all.

You can always run one of them and then check the manufacturers WWW yourself if it says a certain device needs updating.

When you upgraded from a previous version of Windows or receive a new computer preinstalled with Windows 10, what happened is the hardware (your PC) will get a digital entitlement, where a unique signature of the computer will be stored on Microsoft Activation Servers. The Windows 7 or Windows 8 genuine license you were previously running will be exchanged for a diagnostics key.

Anytime you need to reinstall Windows 10 on that machine, just proceed to reinstall Windows 10. It will automatically reactivate. You will prompted to enter a product key a couple times through the installation, click I don't have a key and Do this later.

There is no need to know or get a new key, but if you have issues with Digital Licensing or the Activation Troubleshooter, you can utilize an existing Windows 7 or Windows 8/8.1 product key you already own to reactivate if necessary. or use the reset function in Windows 10.

Go to a working computer, download, create a bootable copy, then perform a clean install.
