doug robbins - mvp office apps & services (word) say:

A few tips before you start.

  1. The first thing that you should do is create new Heading styles for each level of each language, and I would suggest that you call them EHeading 1, EHeading 2, EHeading 3, EHeading 4 for the English headings, SHeading 1, SHeading 2, SHeading 3, SHeading 4 for the Spanish Headings and RHeading 1, RHeading 2, RHeading 3, RHeading 4 for the Russian Headings.

    You need these set up first so that when you define the numbering as in Figure 5 in the Late Shauna Kelly's article, the styles will be available for you to link the numbering to them.
  2. You will need to repeat the steps 1 - 6 of Shauna's article for each of the languages, and I would suggest naming the List Styles and English Headings, Spanish Headings and Russian Headings.
  3. It would probably be best to initially set up the headings for one language and make use of them so that you can make any required adjustments to indents, which MUST be done in the Define New Multilevel List dialog, and when you have them the way that you want, make a note of those settings so that when you are setting up the numbering for the other languages, you can assign those settings at the outset.  Otherwise, you may be forever going back to the Define New Multilevel List dialog to fine tune the indents for each language.
  4. Don't be tempted to base the Heading Styles for one language on the Heading Styles for one of the other languages.  You will not be able to get the numbering that you want that way.

You have to understand how to use it.

You turned your phone off and installed an SD card, new I hope. If previously used in another device you should have formatted it before installing.
