The default save folder for Movie Maker .mswmm
project files on a Windows XP machine is:

C:\Documents and Settings\>user name<\
My Documents\My Videos

If you do not find them in the My Videos folder...
just do a search for .mswmm and you will find them.

All you need to do to back up the .mswmm files is
copy them and paste the copy in a different folder
or save them on external media.

Another thing that is important are your source files...
photos, video clips, music files, etc... be sure that
they remain in their original locations until you are
positive that you're finished with the project. The
.mswmm project file does not include the source files...
it only records the path to the original locations.

Might be a good idea to make copies of the source
files and store them in the folder with the backed up
.mswmm files.

Step 3: Wait for the Assistant to download the Windows 8 Pro installation package and create the bootable media or .iso file. The flash drive must be have 3GB or more of free space available.

Step 4: Burn the .iso file, if that's the path you chose, to a blank DVD.

Step 5: Use that USB drive or DVD to boot the PC.

Step 6: When you get to the screen in Setup that asks, "Which type of installation do you want?' choose "Custom."
