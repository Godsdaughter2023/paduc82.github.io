tryx3 say:

1  You can wake a Windows computer from Sleep using Task scheduler even if it is on battery power.  I use this a lot as Sleep only uses up about 1% of the battery power an hour so an overnight wake-up poses no problems.  You can use Windows Help or this forum to find out how to use Task scheduler.  You would need to set up Task scheduler to run some task or other but that could be as trivial as opening Notepad or running some auxiliary task such as this command to create a file on your desktop

PowerCfg -LastWake >D:\Desktop\LastWake.txt

{You would get info that might help with future troubleshooting in the file created and you would always be able to look at the file's properties to confirm that the wake-up happened when you wanted it to.}

2  Every Windows computer I have had has needed AC power to be connected for Task scheduler to be able to wake it from Hibernate whatever settings had been chosen in Task scheduler.  You would have to try it to get a definite answer for your computer.

Windows update will run after wake-up as long as the internet re-connects automatically [which many Ethernet or WiFi connections will do].  So, unless your UpdateOrchestrator utility is doing something else for you at the same time, I don't think it will be needed for this wake-up then update procedure to work.

tryx3 say:

Yes, after a computer is woken from Sleep it will consume normal levels of power and therefore only last a limited time [for my laptops, this would be about 4 hours for one of them and about 2 hours for the other].

If you need to worry about power consumption because of the particular time that you are waking the computer, when you put it to Sleep switch power plans to one that puts it back to Sleep after, say, two hours idleness or change your normal power plan to this setting so that you never have to bother switching

As you rightly imply, a Sleeping computer should never be left in circumstances where it might overheat unless you are able to monitor it properly & have ensured that there are no impending wake tasks.  For example, I leave the computer in Sleep in my bag if I'm going to the station & intend to work on the train but not if the computer is simply going to be luggage.
