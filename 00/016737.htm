Nokia E51 WLAN Solution - Microsoft Community

clemencemlambo say:

Nokia E51 WLAN Solution

After 5 days of shouting Nokia support people and my service provider with no solution at all- imagine these guys could not give a solution for their own product- (this is sad, I decided to used my basic networking knowledge to solve the WLAN problem, so this is what I did and it worked like a charm

ROUTER SETTINGS

1 Access your router through internet brower by entering 10.0.0.02 in the web address.

2 Click on Interface Setup, and click on WIRELESS and enter following settings:

Access point = Activated
SSID = enter any name you want
Broadcast SSIS = Yes
Channel ID = choose want you want
Authentication type = choose WEP 64 bits (easier to work with)

Then Enter 4 hexadecimal keys with a prefix of 0x (zero x) e.g 0xa9bc1d3ef2

Make sure you use 802.11 b+g option, under advanced settings depending on your router

SEARCHING WLAN CONNECTION ON E51

3 Then USE your NOKIA E51 to scan for WLAN networks. when you find this network - with SSID name you set above, try to browse so that you get an option to save it as a DEFINED ACCESS POINT. save it

GETTING ROUTER IP SETTINGS

4 Now go to DOS command on the computer where the ROUTER is connected, and type ipconfig /all Then a list of your windows IP configuration will come on. Write down the IP Address, Subnet Mask and Default Gateway, you will need these later on

SETTING UP YOUR E51

5 Now go to your E51 and go to MENU-TOOLS-SETTINGS-Connection-Access Points-Choose the WLAN name that you just saved and ENTER- now you are the Access points ( Network SSID name) configuration page. Enter the following settings:

WLAN SETTINGS ON E51

Connection Name = should be the same with your SSID
Data Bearer = Wireless LAN
WLAN Network Name = SSID (as the connection Name above)
Network Status = Public
WLAN Network Mode = Infrastructure
WLAN Security Mode = Same as your authentication mode/type in the ROUTER e.g WEP
WLAN Security Settings = Enter the Keys as they appear in the Router

You are done with this,

IP SETTINGS ON E51

On the same screen Press OPTIONS - Advanced Settings. LEAVE everything else AS IS. Just change the following:
IPv4 Settings (press enter) Get these from the "GETTING ROUTER IP SETTINGS" section above, DOS- ipconfig /all command- remember??

Phone IP Address = the one you wrote down, but with a bit of modification e.g if your wrote down IP is 10.0.0.100, here you will put 10.0.0.110 or so, only change the last 2 digits. Make sure you understand this

Subnet Mask = the one you wrote down

Default gateway = the one you wrote down

GO back to CONNECTION then choose Wireless LAN - Show WLAN Availability (enter your preference - YES or Never)

Now on the same screen Press OPTIONS button on the left side- Choose Advanced Settings and YES to continue

LEAVE everything else AS IS, ONLY CHANGE the following settings

Automatic configuration = Disabled
Long retry limit = Choose higher number eg 20
Short retry limit = Choose higher number eg 60
Power Saving = Disabled

EXIT by pressing BACK button. BROWSE THE INTERNET YOU ARE SET

I hope you will find this helpful. I hate Nokia support service

doug robbins - mvp office apps & services (word) say:

Probably easier to have two separate macros.  One that has all of what you want and then a second one with just the repeating section, which the user would call after positioning the cursor.

 

However also see the article “Graphics on Labels” on fellow MVP Graham Mayor’s website at

http://www.gmayor.com/graphics_on_labels.htm

Or use the following macro in conjunction with Mail Merge to create the labels:

Macro to set the first label on a part sheet of labels for a label type mailmerge.

Dim MMMDoc As Document

Dim dsource As Document

Dim dtable As Table

Dim i As Long, j As Long

Set MMMDoc = ActiveDocument

With MMMDoc.MailMerge

    If .MainDocumentType = wdMailingLabels Then

        If .State = wdMainAndDataSource Then

            Set dsource = Documents.Open(.DataSource.Name)

            Set dtable = dsource.Tables(1)

            i = InputBox("Enter the number of labels that have already been used on the sheet.", "Set Starting Label")

            If IsNumeric(i) Then

                With dtable

                    For j = 1 To i

                        .Rows.Add BeforeRow:=.Rows(2)

                    Next j

                End With

            End If

            .Destination = wdSendToNewDocument

            .Execute

        End If

    End If

End With
