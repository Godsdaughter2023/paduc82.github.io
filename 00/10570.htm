daniel pineault say:

If Stephen Lebans says it is so, then it is so!

But as usual, he did all the work for the rest of us and provided with a solution to the problem.

(1) import the module into your db
(2) in your form's module add in the following

Private Sub VerticallyCenter(ctl As Control)
Dim lngHeight As Long
lngHeight = fTextHeight(ctl)
' Rounding will result in a 1 to 2 pixel margin of error
' of every control before it renders text.
ctl.TopMargin = ((ctl.Height - lngHeight) / 2)

End Sub

(3) in your form's onload event add in (For each control you wish to vertically align):

VerticallyCenter Me.YourLabelControlName
