There may be some VBA method of changing only the border color but that's outside my realm.

rich michaels - mvp (greatcirclelearning.com) say:

Try this:

Sub ChangeBorderColor()
    Dim doc As Word.Document, rng As Word.Range
    Dim tbl As Word.Table
    On Error GoTo errHandler
    Set doc = ActiveDocument
    If Selection.Start = Selection.End Then
        Set rng = doc.Content
    Else
        Set rng = Selection.Range
    End If
    
    If rng.Tables.count = 0 Then
        MsgBox "No tables exist in selected range.", vbExclamation, "Change Border Color"
        Exit Sub
    End If
    
    For Each tbl In rng.Tables
        If tbl.Borders.OutsideColorIndex = wdAuto Or _
            tbl.Borders.OutsideColorIndex = wdBlack Then
            'for grayscale coloring there is only wdGray50 or wdGray25
            tbl.Borders.OutsideColorIndex = wdGray50
        End If
        If tbl.Borders.InsideColorIndex = wdAuto Or _
            tbl.Borders.OutsideColorIndex = wdBlack Then
            tbl.Borders.InsideColorIndex = wdGray50
        End If
    Next
    Exit Sub
    
errHandler:
    MsgBox Err.Description, vbCritical, "Change Border Color"
    Err.Clear
End Sub

Hope it helps

This issue may be caused by multiple product keys installed in office, down below will be attached solutions to uninstall and install a new product key to the office package:

Check and uninstall if needed multiple product keys in Office:


1. Please check if there is any product key installed on the computer.


A. Use the following command in the command prompt with admin:


   for 32 bit office -


cscript "C:\Program Files (x86)\Microsoft Office\Office16\ospp.vbs" /dstatus


for 64 bit office -


cscript "C:\Program Files\Microsoft Office\Office16\ospp.vbs" /dstatus

B. Use the following commands in the command prompt with admin to remove the product keys

cscript "C:\Program Files (x86)\Microsoft Office\Office16\ospp.vbs" /unpkey: (the last 5 digits of the PK that you see in the previous step)


 For 64 bit:


cscript "C:\Program Files\Microsoft Office\Office16\ospp.vbs" /unpkey:(the last 5 digits of the PK that you see in the previous step)

Repeat the same steps until all Product keys are uninstalled.


C. Open an desired office application

D. Continue the activation progress.
