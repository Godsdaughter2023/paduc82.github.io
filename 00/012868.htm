ken sheridan say: Another way is to store the URL in a column in a table as text and set the IsHyperlink property of the control bound to the column to True (Yes).  That way it will look and behave as a hyperlink in the form.  No code is required.

ken sheridan say:

I do not see IsHyperlink in Access 2013.


The IsHyperlink property is a property of a text box control, not a column in a table.  As all user interaction with data should be via forms, never in a table's datasheet, it is in the bound control in a form where the hyperlink is created by setting this property to True (Yes).  the column in the table is a simple Text data type.

However, I was wrong to say no code is needed.  I should have examined my own demo more closely!  The IsHyperlink property really does no more than format the text as a hyperlink.  It is also necessary to call the FollowHyperlink method of the Application object in the bound control's Click event procedure:

Application.FollowHyperlink Me.[NameOfControlGoesHere]
