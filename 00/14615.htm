ossiemac say:

Am I interpretting this correctly in that you want to centre the image in a cell? If so then try the code below.

 

I have used ActiveSheet.Shapes.AddPicture in lieu of ActiveSheet.Pictures.Insert. In xl 2010 AddPicture only stores the link to the file but with Shapes.AddPicture the parameters can be set to save only the link or the actual picture with the excel file. (Shapes.AddPicture works in xl2007 also.) I have set the parameter to save the actual picture with the excel file.

 

You will also see some example code between the asterisk lines that is commented out to demonstrate how to import a picture and retain the Aspect Ratio. When Aspect ratio is retained only the Height or Width needs to be changed. If both are set in the code then the last one set takes precedence and resets the previous to retain Aspect ratio.

 

Sub IMPORTPHOTOS()
    Dim dblLeft As Double
    Dim dblTop As Double
    Dim shpImage As Shape
    Dim Image As String
    Dim X As Integer
    Dim TotalRows
   
    TotalRows = ActiveSheet.UsedRange.Rows.Count
   
    For X = 1 To TotalRows
        'Set the row height and width of the cell to contain image
        ActiveSheet.Range("h2").Offset(X - 1, 0).ColumnWidth = 15
        ActiveSheet.Range("h2").Offset(X - 1, 0).RowHeight = 84
       
        'Assign left and top of cell to variables
        dblLeft = ActiveSheet.Range("h2").Offset(X - 1, 0).Left
        dblTop = ActiveSheet.Range("h2").Offset(X - 1, 0).Top

        Image = "D:\Desktop\horeca\curver small\" & Range("A2").Offset(X - 1, 0) & ".jpg"
       
        'Code to set Width and Height at time of import _
         using Shapes.AddPicture
        Set shpImage = ActiveSheet.Shapes.AddPicture _
                    (Image, False, True, dblLeft, dblTop, 80, 80)

        '*********************************************************************
        'Alternative code to lock aspect ratio during import with _
         parameters -1 for Width and Height lock the aspect ratio.
        'Set shpImage = ActiveSheet.Shapes.AddPicture _
                    (Image, False, True, dblLeft, dblTop, -1, -1)
       
        'Following code only required if dimensions set after import. _
         When Aspect ratio is locked then only need to set either Height or Width _
         and the other is set automatically. If both set then will set both _
         Height and Width keeping Aspect ratio by the latter line of code.
        'With shpImage
        '    .Height = 80
        'End With
        '***********************************************************************
       
        'Center the image in the cell.
        With shpImage
            .Left = .TopLeftCell.Left + (.TopLeftCell.Width - .Width) / 2
            .Top = .TopLeftCell.Top + (.TopLeftCell.Height - .Height) / 2
        End With
   
    Next X
   
    Range("I6").Select
End Sub

Added with Edit:

If the width and height of the cell to contain the picture is smaller than the picture then the picture is still centred in relation to the cell by using the the formula in the code but it will overlap the borders of the cell.
