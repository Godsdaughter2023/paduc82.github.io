joel ivory johnson say:
Just long-press the back-button :)

The applications that display there are in most cases not running. Once an application is no longer in front of the user it get's a few seconds to run (up to 5 seconds) so that it can save progress or do anything else it needs to. After that the application is suspended. If the user doesn't navigate back to it the application is cleared from memory. The sense that the application hasn't been closed is maintained by the phone keeping a screenshot of it's last view and applications usually being able to restore themselves to their previous state. 

Some applications can perform certain task in the background. Applications that can do this are in settings. But even these background task get very little execution time and usually are not running. There is one type of background task that *can* run continuously (Resource Intensive Task) But such a task can only execute under certain conditions including the phone being connected to a power supply and Wi-Fi being turned on.

brian tillman [outlook mvp 2007-2019] say:

A person who is to get one of your Premium address must first have an existing mailbox. You then send that mailbox an invitation to use an address in your custom domain. When that person accepts, any messages addressed to the custom address will forward to that person's normal mailbox. If that person wishes to send mail using the custom address as the sending address, they must configure their mailbox to use the custom address as an alias.

If the person doesn't have a mailbox, create a new ordinary @outlook.com mailbox, then send that mailbox the invitation. The custom address will get added as an alias of the @outlook.com mailbox.
