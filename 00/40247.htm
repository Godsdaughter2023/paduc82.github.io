While certainly good reading, neither of those links seem to tell you how to add the missing Print Spooler Service, and the Microsoft link will certainly not help you add the missing Service and is a waste of time.

If the Print Spooler Service is missing from Services, you must add it (how do you suppose that happened?).

I do not have XP Home, but this method works on XP Pro and I can't imagine why it would be any different.


Here's how to add the Print Spooler Service:

If your system is Windows XP Service Pack 3, you can add the missing Print Spooler Service to your system using a registry import.

Before making any changes to your registry, back up the registry with this popular free tool:

http://www.snapfiles.com/get/erunt.html

Reinstall the missing Print Spooler Service by creating a registry file that you can import into your system (tested on Service Pack 3). 

If you already have a registry entry for the Print Spooler Service, you may want to export it to a file and save it.  The following example is the generic registry code from a working XP Pro Service Pack 3 system.

Open notepad to create a new text file by clicking Start, Run and in the box enter:

notepad

Copy and paste the following lines of bold text into the new notepad file:













Save the new text file with a .reg extension to your desktop or a folder with a name you can remember, something like:

addspooler.reg

After saving the file, close notepad.

Locate the addspooler.reg file on your desktop and double click it.

Alternatively, you can right click the addspooler.reg file, choose Open With... and select the Registry Editor which will then import the file.

Respond in the affirmative to the question... are you sure you want to add the information to the registry?

You should then see a message that the information was successfully entered into the registry.

I would reboot at this point to make sure the updates "stick".

Now you should see the Print Spooler Service and you can right click it, choose Properties and (if it is not already) set the Startup type to Automatic and start the service if it is not already running after the reboot.

Sine the Print Spooler Service should not be missing, I suggest you follow up with this (or do it first):

Download, install, update and do a full scan with these free malware detection programs:

Malwarebytes (MBAM):  http://malwarebytes.org/
SUPERAntiSpyware: (SAS):  http://www.superantispyware.com/

They can be uninstalled later if desired.

For Deleted Items, just right click on it and  Empty Deleted Items Folder.

For Sent Items:

To delete all:  Highlight one and Ctrl + A will highlight them all. - Delete.

To delete a section: Highlight the first one you want to delete. Scroll down to the last one and click on it while holding the Shift key down. - Delete.

To delete multiple messages that are interspersed: Hold down the Ctrl key while you click on each message to highlight. - Delete.

When you're done, make sure to compact your folders as described below.

General precautions for Outlook Express:

Do not archive mail in the Inbox or Sent Items. Create your own user defined folders and move the messages you wish to save to them. Empty Deleted Items folder daily. Although dbx files have a theoretical capacity of 2GB, I recommend about a 300MB max for less chance of corruption.

Information about the maximum file size of the .dbx files that are used by Outlook Express:
