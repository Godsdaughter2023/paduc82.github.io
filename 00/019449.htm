Hi Iwata.ty,

First of all, the settings for "automatically update document styles" means to update the style definitions of a document with the style definitions in its attached template.

This means each time the document is opened, it will check for the style from the original template. This is what is creating the "Contacting..." dialog. If you uncheck that particular setting, it should hopefully resolve the issue. This is the behavior I'm getting on my end.

However, as Charles has explained in the other thread, this should not cause Office to crash. It does not on our end. Nevertheless, you should be able to resolve it once you uncheck the button. 

The default setting should be unchecked for Word to "automatically update document styles". Additionally, this settings affects only  that particular document. We suggest you edit your dotx (template) to uncheck this, so that each time you create a  new docx from the template, the settings should not be applied.

Let me know if this resolves the issue.
