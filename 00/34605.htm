The 1st connector is DVI, the second is VGA or also known as D-Sub. A monitor itself requires no software to operate.


What connectors are you attaching the leads to in the computer? Connectors in the same group as the USB, audio and network sockets are referred to as being on board, connectors in the parallel slots separated from those ports will be a graphics card. In this picture an on board graphics connector would be where the serial port is. 



Not all motherboards will allow you to use both the on board and graphics card ports at the same time. Installing the card would automatically turn off the on board card.


In this example the graphics card has both types of port (Analogue video port = VGA) and in this case both ports can be used.


Some graphics cards only have DVI ports but most will support VGA with the use of a DVI-VGA adapter

guru at guru graphics say:

I found a work around because CorelDraw inheritance the settings from Windows and Windows 10 does not have any where you to change title bar or font colors unless you just use the contrast settings which is "Blinding" to me.

Here Is a work around:

You can run CorelDraw in Win XP SP 3 "Compatibility Mode" and the menu bar shows up just fine. (Why Windows XP?? Because it did and does not use the Windows settings. It has it's own.) However, every time you start CorelDraw in XP Compatibility Mode you have to answer yes to the Authorization Pop-Up..

Right click on your CorelDraw Icon, select Properties, select the "Compatibility Tab" at the top, in the "Compatibility Mode" box click on the box by the words "Run this program in compatibility mode for:" so there is a checkmark in it the box, then select Windows XP (Service Pack 3).

Left click on apply at the bottom then okay.

Then open CorelDraw. Remember, you'll have that popup to answer yes to then go on using Corel..

Hoping Microsoft fixes this.

There are some "Hacks" out there that say they have fixes but the ones I've tried do not fix the Corel problem..
