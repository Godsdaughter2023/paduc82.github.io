To add to paperplane's post...

If your email addresses are in Excel, you can save your Excel file as a *.csv file. You can do this in Excel by opening the spreadsheet, choosing File > Save As > From the Format pull down choose  Comma Separated Values (.csv). 

Once you have your *.csv file, follow the steps below to import the file into Outlook:

  1. On the Outlook for Mac File menu, click Import.

  2. Click Contacts or messages from a text file, and then click the right arrow.

  3. Click Import contacts from a tab- or comma-delimited text file, and then click the right arrow.

  4. Locate the text file that you want to import, and then click Import.

    On the Import Contacts screen, the Mapped fields box shows the fields that Outlook matched. The left column of the Mapped fieldsbox shows the fields that are available in Outlook. The right column of the box shows the fields of the imported text file.

  5. Drag fields from the Unmapped fields box to the right column of the Mapped fields box. For example, drag the "Business Phone" field to the right column of the Mapped fields box next to the "Work Phone 1" field.

    You can also drag fields within the right column of the Mapped fields box.

  6. To view other contact records and verify that the data is mapped correctly, click Next or Prev.

  7. When you have finished mapping fields, click Import.

Hi

Targeting for small items, such as the Notification Icons is important.

According to user studies the developers can determine an optimal distance for the spacing of components. It has to be large enough to work if the user slightly misses the actual icon with the Cursor.

If you hover the Cursor over one of the Notification Icons, you will see an almost transparent Box appear on all sides of the icon. This is the Target area for the icon.

Anything less than that target area would result in some very frustrated users when trying to hit one of those icons.

This targetting is in almost every part of Windows.

Here is a two-step process for superimposing the normal density curve onto a histogram:

(1) First, browse to BetterHistogram.com, go to the Free Download page, and download the Better Histogram ZIP file. Use the add-in that will (a) create the frequency distribution from your raw data and (b) create the better histogram.

(2) Second, browse to MikeMiddleton.com, go to the Excel page, and download the Histogram With Normal Curve workbook, which includes terse instructions for creating the normal density curve.

(The normal density curve shown here has approximately 99.7% the area under the histogram. If the curve is extended several more standard deviations in each direction, the two areas become essentially equal.)
