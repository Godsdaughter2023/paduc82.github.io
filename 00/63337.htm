michaelclark_01 say:

Passwords are kept in a registry file called SAM. You can restore this file manually like so:

1. Go back into Repair Mode as before.
2. Select the Command Prompt.
3. Identify the correct drive letter for your Windows installation. It could be a drive other than C: but it won't be X:. You do it by typing the commands
    dir C:\Win*
    dir D:\Win*
    dir E:\Win*
    until you find the Windows folder.
    I will call that drive "Q:".
4. Type these commands:
    cd /d Q:\windows\System32\config
    xcopy *.* Q:\RegBack\
    cd RegBack
    dir
5. Examine the date of the file called SAM. How old is it? Did you change any passwords since that date?
6. Type this command:
    copy /y Sam ..
    (the two dots are part of the command)
7. Reboot normally. If things do not work out then you can backtrack by copying the original registry files from Q:\Regback to Q:\Windows\System32\config.
