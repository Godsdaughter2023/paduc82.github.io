doug robbins - mvp office apps & services (word) say:

You will need to use some code to do that.

With your document as the active document, press ALT+F11 to open the Visual Basic Editor and then locate your document in the Project pane, and if necessary click on the + beside it and then click on the ThisDocument item in the list.  Then in the code window on the right, paste in the following code:

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)
Dim score As Long, i As Long
score = 0
With Selection.Rows(1)
    For i = 2 To 7
        Select Case .Cells(i).Range.ContentControls(1).Range.Text
            Case "Definite"
                score = score + 2
            Case "Potential"
                score = score + 1
