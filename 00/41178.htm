Method 2:

If files from an indexed folder are not showing up then it might be because of the security settings for that folder. For a folder and its content to be indexed, it must be configured with the  System permission. Most folders already have this, but there are some cases where a folder might be missing the System permission.
To add the System permission to a folder

a. Right-click the folder, click  Properties, and then click the  Security tab.
b. If you don't see  System listed under  Group or user names, click  Edit.
c. Click  Add, type  System in the box that appears, and then click Check Names.
d. Under  matching names, click  System, and then click OK until all of the open dialog boxes you've opened are closed.
You will need to rebuild the index once you've made this change. To rebuild the index you may refer to the below link change advanced indexing options.
