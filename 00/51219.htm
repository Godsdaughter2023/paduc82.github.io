Restore Factory Defaults - Microsoft Community

xb1r athena say:

Restore Factory Defaults

Login Information

  • Gateway – 192.168.0.1
  • Username – admin
  • Password – admin

 

Restore Factory Defaults

Through Hardware

  1. Locate the Reset button.
  2. Using the tip of a ballpoint pen or toothpick press and hold the reset button for 15 seconds.
  3. Release the reset button.
  4. Wait for the device to restart.

Through Firmware

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select System Tools from the menu on the left.
  5. Select Restore to Factory from the submenu on the left.
  6. Click the Restore button.
  7. Select Yes to confirm.
  8. Wait for the router to reboot.
