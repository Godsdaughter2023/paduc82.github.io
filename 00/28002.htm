barb way say:

In Visio 2010, the Swimlane shapes in the Cross-Functional Flowchart template get their fill settings from the Style selection on the Cross-Functional tab.  By default, the selected Style has No Fill settings for the swimlanes. You can select a different Style from the drop-down (look in the Design grouping on the tab), and choose a different option which includes Fill.  Then you can select lanes and apply fill manually, or use the themes on the Design tab, and the swimlanes will be able to fill with color.

suzanne s. barnhill say:

You may see lots of workarounds for various situations involving quotation marks in Word (such as typing a space, then the quote, then deleting the space), but it is much more efficient to learn the simple keyboard shortcuts that you can use to force Word to give you exactly the character you want:

 

Single opening quote: Ctrl+`, ` (that is, press Ctrl together with the grave accent key--on U.S. keyboards, it's at the top left, combined with the tilde--then release and press the accent key again)

 

Single closing quote or apostrophe: Ctrl+', ' (same procedure except using the apostrophe key)

 

Double opening quote: Ctrl+`, " (Ctrl and accent, then shifted apostrophe)

 

Double closing quote: Ctrl+', " (Ctrl and apostrophe, then shifted apostrophe)

 

With practice these become second nature so that you can use them easily; there is no excuse for an opening single quote where an apostrophe is required (as in 'tis or 'n')!

I had same issue. The process below saved me!

The answer is in Time Machine or your backup.

1.     Turn off all internet connections or One note will start syncing before you can complete the recovery

2.     Delete/remove One note from Applications folder.

3.     Find: “com.microsoft.onenote.mac” (see steps a - e below)

a.     Close everything on desktop

b.     Click on Go in the menu bar

c.     Hold down option key, now you should see library click on Library

d.     Open Containers folder

e.     Scroll down to find “com.microsoft.onenote.mac” should be in there (on to step 4)

4.     Open Time Machine while you are at this location in finder. 

5.     Go to date you know files were there. (Once you do this you may have to open containers folder again while in time machine)

6.     Highlight

7.     Choose replace the existing file

8.     Then go to applications folder for the same date in time machine and recover the OneNote app

9.     Open OneNote

10.  Your files will be there but you’re not safe yet!

11.  Make another Notebook call it backup (or whatever you want)

12.  Copy all other note books contents into the new notebook you just created

13.  Turn on your internet connection allowing one note to sync

14.  Now some of the original files may be gone from original locations but if you copied them into the new notebook = “backup” they will all be there and you can put them back to their rightful place.

Fix for how to connect to a Windows Small Business Server 2008 server - Microsoft Community

john ericson say:

Fix for how to connect to a Windows Small Business Server 2008 server with Windows 10 TP

I noticed that the fix for Windows 8 to connect to Windows SBS 2008 no longer works in Windows 10 Technical Preview.

Instead you can VPN/RDP directly to a server without using Internet Explorer and visiting the company website. All you need is the included RDP program that comes with Windows. Do like this:

  1. Win + R and type "mstsc.exe".
  2. In Computer type in short name of server, for example SERVER.
  3. Make sure Username is correct. If server uses another domain, type it in like this: DOMAIN\user.
  4. Show options and go to Advanced and then Settings.
  5. Select "Use these RD Gateway server settings". In Server name type in adress of website, for example: vpn.company.com.
  6. Make sure Log-on-settings is correct.
  7. Press OK and Connect.
Feel free to make a wiki page of this article.

Sometimes when the page is refreshed unexpectedly, we are unable to notify the server that you have left the document.
