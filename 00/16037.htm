How do I get X² or X³ superscripts missing in OneNote office - Microsoft Community

paulcofrancesco say:

How do I get X² or X³ superscripts missing in OneNote office 2010, office 2016 etc.

How do I get X² or X³ in OneNote office 2010, office 2016 etc when they are missing from the equation / formula dialog box?

SOLUTION: just type the unicode 00B2 or 00B3 after that it reverts to something more normal 2074 for X⁴ and 2075 for 5th power etc.

Steps: (office 2016)

Click Insert Tab, Symbol, More symbols, type 00B2 or 00B3 into the character code box.  For some reason in the translation form decimal to hexadecimal before x⁴ which is code 2084 it reverts to 00B#, after x⁴ it is easy x5 would be 2085, x6 is 2086 and so on.  The problem is Microsoft sorts them in a way that ends up separating something that would normally be grouped in the same place.

Shortcut: you can cut and paste right from here to add X² X³ to your document.

Note: this question was posted and answered on the ms forum in 2015 but the thread was closed to new posts and the posters neglected this simple solution of typing the code in the character code box "00B2" (zero zero B 2) not the letter 'O'.

  • PROXYCFG –P 1.1.1.1:8080
  • PROXYCFG –P MYPROXY.NET:8080

tasos_k say:

Hi,

try this code

export data from active sheet to CSV format

file name: ActiveSheet.Name

Sub ADODB_method()

'Apr 15, 2014

Const myDelim As String = ","  '<<< select comma or semicolon
Dim WS As Worksheet
Set WS = ActiveSheet
Dim r As Long, c As Long, i As Long, j As Long
r = WS.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
c = WS.Cells.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious).Column
Dim myPath As String
myPath = ThisWorkbook.Path
Dim myFile As String
myFile = myPath & "\" & ActiveSheet.Name & ".csv"

Dim obj As Object
Set obj = CreateObject("ADODB.Stream")
obj.Type = 2
obj.Charset = "utf-8"
obj.Open
Dim v() As Variant
ReDim v(1 To c)
For i = 1 To r
For j = 1 To c
v(j) = WS.Cells(i, j)
Next
obj.WriteText Join(v, myDelim), 1
Next
obj.SaveToFile myFile, 2
MsgBox "done"
End Sub
