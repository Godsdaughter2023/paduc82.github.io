When creating a MINIFS() or MAXIFS() type of array formula, you cannot use AND() or OR() in the conventional sense as they don't follow the pattern of array computing. You have to nest the IF(IF(IF())) statements. Try,

=MIN(IF(Sheet1!A:A=Sheet2!A4, IF(Sheet1!B:B=Sheet2!B4, IF(Sheet1!C:C=Sheet2!C4, Sheet1!D:D))), 99999)

don cuthbert say:

"Factory Reset" usually refers to restoring an OS image that matches whatever the computer shipped with when it was new. If that is still working, you would revert to that version and Edition of Windows, whatever it was (such as Windows 8.1 Home).

A Disk Image is a backup of your current OS and data. Restoring that gets you back to however things were when the backup was done. For example, if you make a disk image on backup media and then replace your hard disk with a new one, restoring that image will replicate what you had on the new disk. If you have Pro and it is activated, it will remain activated on the new disk.

If you do a Reset from the Settings App it will simply Reset the current install of Windows Pro, close to the state it would be from a "clean install". The user accounts remain as they are and you have the choice to keep data, but most Apps and software will have to be reinstalled.

If your computer came with some version of Windows Home, but now has Windows 10 Pro activated and you do a "clean install" (where you either begin with a new hard drive or wipe all partitions from the existing one) often the embedded "Home" Product Key might be detected during install and Windows 10 Home will be installed. If this happens, you can just use a Pro Product Key to upgrade to Pro or running the Activation Troubleshooter may find your Digital License for Windows 10 Pro for that PC. For the Troubleshooter to work, you would have to login to your currently activated Windows 10 using a Microsoft Account (which will link the License to your Microsoft Account).

If you have any doubt that your activated Windows 10 Pro will activate after a clean install, there is also a way to save the activation status to a small XML file, by running gatherosstate.exe from the Windows 10 install media. Here is an old article describing this process:

http://betanews.com/2015/08/31/how-to-do-a-clean-install-of-windows-10-the-easy-way-no-upgrade-required/
