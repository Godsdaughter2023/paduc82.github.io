hi simon,

if you’re using onedrive for business next generation sync client, as “shared with me” section is just a view that shows the documents or folders other users have shared with you, the shared files or folders are still physically in their own libraries, so it’s not feasible to sync and see the files via onedrive for business next generation sync client when opening the onedrive for business synchronization folder (file explorer).  in this situation, we suggest you vote on uservoice via the link below. many features came directly from your feedback.
onedrive for business - implement shared folder syncing to desktop client

we’re unable to sync the sharepoint team sites libraries to the local via onedrive for business next generation sync client. however, the ability to sync sharepoint document libraries including office 365 groups is in development for onedrive for business next generation sync client, you can install the latest sync client after the feature is rolling out to make sure you get the best service, for the latest updates, you can follow up on office 365 roadmap:
http://fasttrack.microsoft.com/roadmap 
 


if it’s the old onedrive for business sync client, then you are able to sync the onedrive for business libraries and sharepoint team site libraries to the local (file explorer). also for the files and folders in the “shared with me” section, you can consider going to the owners’ onedrive for business libraries directly and syncing them to the local.with this, users will be able to access the files from file explorer (including use of the preview pane)

for the ability to open the files in the shared location via office applications, we can store all the shared files in one team site, then let each user add a place in the office applications to access the library directly like the following screenshot shows:
 


also we can let each user sync the team site library to the local via the old onedrive for business sync client, then open the files via office applications after clicking computer option. if your company are in the same ad domain, you (as the admin) can also choose to sync the team site library to one file server, then share the file server with your users for their access.

thanks for your understanding.
