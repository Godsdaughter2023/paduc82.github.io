Windows XP Fax Services is not a very robust application.  The usual response to questions like yours is, "If the modem isn't on the Windows XP Hardware Compatibility List, then it probably won't work with XP Fax Services."  The problem with such a response is that the XP HCL (or whatever Microsoft changed its name to) is no longer available, and even if it were, you probably couldn't find any of the listed modems for sale anymore.

Your phone numbers are plainly not in the U.S.  In the U.K. perhaps?  If you use Dialing Rules, both the called number and the calling number must be entered in "Canonical Form"http://support.microsoft.com/kb/318575/

I'm pretty sure you need the plus sign, almost sure you need the parentheses, and don't know about spaces and dashes. For this reason, I generally don't use Dialing Rules.

When you leave the box for Dialing Rules UNchecked, you must enter the number EXACTLY as you would if you were dialing a normal phone using the line to which the modem is connected. 

Try sending a fax to your own phone (mobile or not, it doesn't matter).  Does the called phone ring? When you answer it, do you hear fax tones?

lilrayna say:

To elaborate, A US manufactured Xbox console will be using the NTSC region code, while Europe (and other areas) use a PAL region code. If you have bought your console from the US (NTSC) and are trying to run a region locked game that is coded as PAL it will not work (or vice versa). Both the game and the console have to be from the same region to work. There are some instances where some games are region free; meaning they'll work on any console. Normally this is not true for the "big name games" like MW3 for example. I would recommend purchasing only games from the same place you've purchased your console and games that are labeled region free. Usually this information will be present on the back label of the game. I hope this helps.
