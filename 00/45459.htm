The error 0xC0000005 can be caused by an incorrect or corrupted device driver, faulty RAM, software that needs to be updated, or a malware infection.

To resolve your concern, we'll be using the Command Prompt from the built-in Windows Recovery Environment. To access the Command Prompt, follow the steps below:

  1. Completely turn off your computer.
  2. Turn on your computer and keep on tapping the F8 key on your keyboard before the Windows logo appears.
  3. You will be redirected to the Advanced Boot Options for Windows 7.

  6. On the Command Prompt, type the commands below and press Enter each of the following lines:

    • attrib c:\boot\bcd -h -r -s
    • ren c:\boot\bcd bcd.old
    • bootrec /fixmbr
    • bootrec /fixboot
    • bootrec /rebuildbcd - type Yes and press Enter when prompted after running this command.
