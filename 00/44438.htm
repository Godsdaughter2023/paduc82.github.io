brett lyons say:

I've found a workaround.

  1. Got to https://catalog.update.microsoft.com/v7/site/Search.aspx?q=xbox%20one%20controller
    This requires Internet Explorer and will probably ask you to install an add-on.
  2. This should list 4 drivers. The version of the files I had listed was 6.2.11059.0 so you need something like that rather than the new 10.* versions.
  3. Ignore the "DFU" drivers. Click the link for one of the other two and check that it matches your CPU type (AMD64 or X86). If it matches then click "Add to Basket" then close the window. If it doesn't match then close the window and add the other driver to your download basket.
  4. Click "View Basket" then click "Download". Specify the folder to save to and download the drivers.
  5. Go to the folder you specified and there should be a sub-folder called "Microsoft - Other hardware - Microsoft Xbox One Controller" that contains the driver download file which should be xxxxxx.cab.
  6. Extract the contents of the .cab file using 7-Zip (www.7-zip.org).
  7. Find the *.inf file (mine was "xb1usb.inf") and install it by right-clicking and selecting the "Install" menu option. Once you accept the update confirmation prompt it will do this silently so just allow 5 or 10 seconds for it to finish.
  8. Go to Device Manager (right-click Windows icon and select "Device Manager")
  9. At the bottom of the screen expand "Xbox Peripherals". Right click the item for the Xbox controller under that (mine was "Xbox Elite Controller") and select "Update Driver Software".
  10. Choose "Browse my computer for driver software" then "Let me pick from a list of device drivers on my computer"
  11. Choose the driver with the version number you just downloaded and click "Next" to install it. Reboot may be required.

Unfortunately with this driver you can no longer use the Xbox Accessories app to configure the Elite controller. However the configuration is stored in the controller so you can configure it on another machine (Xbox One or PC) and then connect it back onto the one with the workaround driver installed.
