Adding bookmarks to Word 2007 document using VBA - Microsoft Community

cmna say:

Adding bookmarks to Word 2007 document using VBA

Can anyone advise on a bookmark problem?

I have a VBA macro which automatically adds bookmarks, eg

'Insert Bookmark:

Selection.HomeKey Unit:=wdStory
ActiveDocument.Bookmarks.Add Name:="Chapter_1", Range:=Selection

This works fine and places the bookmark at the start of the document.

But, if I then want to add another bookmark in front of the first of these bookmarks, I get problems.

For instance:

Set myRange = ActiveDocument.Range(Start:=0, End:=0)
ActiveDocument.Sections.Add Range:=myRange

ActiveDocument.Bookmarks.Add Name:="Table_of_Contents", Range:=myRange

This adds the bookmark "Table_of_Contents" ok - but it places it after, not before, the bookmark "Chapter_1"

I've tried plenty of variations on this code example, but I'm obviously missing something.

Any suggestions would be appreciated.

jay freedman say:

When the macro finishes, you want the Chapter_1 bookmark to be at the beginning of the newly created second section, not at the beginning of the document. So do it this way:

    Selection.HomeKey unit:=wdStory
    ActiveDocument.Bookmarks.Add Name:="Chapter_1", Range:=Selection
    
    Set myRange = ActiveDocument.Range(Start:=0, End:=0)
    ActiveDocument.Sections.Add Range:=myRange
    ActiveDocument.Bookmarks.Add Name:="Table_of_Contents", Range:=myRange
    
    Set myRange = ActiveDocument.Sections(2).Range
    myRange.Collapse direction:=wdCollapseStart
    ActiveDocument.Bookmarks.Add Name:="Chapter_1", Range:=myRange
