Hi little09red,

 

Here’s a little insight on how Library works. A Library can display files that are stored in several folders at the same time. However the catch here is that Libraries don't actually store those items. They just monitor folders that contain a user’s items and provide a single access point to those contents. The point behind this feature is to provide an easier and faster way to search, view, and access documents, photos, music, videos, and other data.

For pictures you have accidentally deleted, use your favorite search engine and search for recovery software from a trusted publisher. That should help you in restoring the pictures back.
