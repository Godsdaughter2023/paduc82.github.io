A better sequence would be:

  remove all partitions using windows disk manager.

  install windows from DVD to the unallocated space.

- or -

  boot from dvd and do custom install

  use the windows installer to remove ALL partitions

  then install windows to the unallocated space.

Either method will allow you to control the size of the partition or shrink a partition to leave room for Linux after the windows OS is installed.

Note: if you use any third party format or partition software, there is no guarantee that the $Bitmap, $Top, or other hidden system format files will be written correctly. To achieve a true "CLEAN" install, you must start with a disk with NO PARTITIONS and then partition it with the OS you are installing on the PC you will be using the drive in. mike h.. say:

Hi,

You know what the code does so we can skip over that but you asked how.

First the code needs to establish the last used cell in a Column and only the programmer knows why Col H is used when the looping is through column B but column H was chosen and this finds the last used cell. It works by reading up from the bottom of the worksheet until it finds a populated cell and returns the row number.

 LastRow = .Cells(.Rows.Count, H_COLUMN).End(xlUp).Row

The coder does that for efficiency so only rows with data in are processed and not every cell in the column or some other wasteful very large range.

Having established the last row it's then used in this line. Here cell is dimmed as a range object and the line of code says look in EACH range object (the variable cell) in the range B2:Bn where n is the lastrow

 For Each cell In Range("B2:B" & LastRow)

As an observation the code is a bit lacking inconsistency. the coder uses

With Worksheets(sSheetName) but then defines sheetname as the active sheet so the WITH statement isn't necessary as it would work on the active sheet without that. We can see that in action in this code where despite having used the with statement the coder then ignores it on this line:-

For Each cell In Range("B2:B" & LastRow)

For consistency what should have happened is that there should have been a period in front of the range like this

For Each cell In .Range("B2:B" & LastRow)

That period now links the range to sheetname.
