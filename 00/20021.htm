bobgreenblatt say:

TEXT is completely wrong here. the date you are working with is already text, NOT a date. The TEXT function converts a number into text according to the format specified.

If you want that string to become a "real" date, then you need this formula:

=Date(left(a1,4),Mid(a1,5,2),right(a1,2))
