In general, if the only problem is that of extra spaces, merely replace A4 with TRIM(A4) to get rid of them.
 
But my suggestion would be a macro.
 
You indicate you wish the results to be in A1:A3, but do not indicate where the originals are, or where the 2nd set of names should go.
 
So I assumed you would SELECT the originals, and that subsequent sets of names would go in adjacent columns:  e.g. b1:b3, c1:c3, etc.
 
To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.
 
To use this Macro (Sub), first select the range of names to parse. 
Then <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.
 
===========================================
Option Explicit
Sub SplitName()
    Dim rg As Range, c As Range
    Dim rDest As Range 'Destination of results
    Dim re As Object, mc As Object, m As Object
   
    'list of delimiters are inside the brackets [...]
    'note that the "dash" must be first or last
    '\s is a token for a <space>
    Const sPat As String = "[-\s,/]+"
   
    Dim P As Long, I As Long, J As Long
    Dim SplitArray()
 
Set rg = Selection
 
'clear target area
Set rDest = Range("A1")
Range(rDest, rDest(3, rg.Count)).ClearContents
 
Set re = CreateObject("vbscript.regexp")
    re.Global = True
    re.Pattern = sPat
    J = 0
For Each c In rg
    Set mc = re.Execute(c.Text)
    ReDim SplitArray(0 To mc.Count)
    P = 0
    I = 0
    For Each m In mc
        SplitArray(I) = Mid(c.Text, P + 1, m.firstindex - P)
        P = m.firstindex + m.Length
        I = I + 1
    Next m
    SplitArray(I) = Mid(c.Text, P + 1)
        For I = 0 To 2
            rDest.Offset(I, J).Value = SplitArray(I)
        Next I
        J = J + 1
Next c
End Sub

When you put the product key in, MS Office contacted the Microsoft Server, which checked the product key was valid, then returned an Activation Key (different number) to your computer.

You can't reconstruct the product key from the activation key, and you can't activate without it.

Are you sure you did not write the Product Key on the backup disk?  Most folks do... :-)
