jeeped say:
... How do you keep '$0.00' from showing when the value is '0'?

Typically, a hyphen is used to express a zero dollar amount so that the user can see that a formula is in place but the amount has a decreased visual impact compared with other cells that have amounts greater or less than zero. Use an Accounting style number format for this.

Another option would be to use an IF() formula like,
=IF(C4-K4, C4-K4, "")
... since zero equates to a boolean False and anything that isn't zero is True. The above leaves a blank cell (i.e. empty string) if C4-K4 is zero.

john korchok say:

If you create a text box by choosing Insert>Text Box, then single-clicking on the screen, the resulting text box is normally just big enough to fit the insertion point. If, instead, you drag across the screen, the initial text box will be 1 line high and the width of your drag motion. If you double-click on the Insert>Text Box button, a text box 1" x 1" will be created in the center of the slide.

You can change some of these default behaviors by drawing a text box, then right-clicking and choosing Format Shape. In the Format Shape pane, choose Text Options, then click on the Text Box icon. Select Do not Autofit or Shrink Text on Overflow, then set the default text box. Now when you create a text box by dragging, the text box remains the full height and width of your drag distances.

If you set the default text box in a presentation, then make that presentation your default presentation, you'll always have the text box style you prefer. Here's an article on how to do that: Create your own default presentation For Office 2016 and 2019, use the Office 2013 steps.
