To check your PID press the win key and pause at the same time, or open system properties through the control panel, scroll down to activation section and look at the PID It will look something like this 12345-067-1234567-54321, the portion you are interested in is the three digit section, if it matches or is within one or two digits of the examples below you probably have an MSDN key.

065, 066, 067, 068 for Ultimate, 220, 221 area for Pro and the 230, 231, 221, 232 area for premium, there are probably others but they should be close to those posted

About MSDN or TechNet Product Keys

They are genuine Microsoft product keys, they are actually retail licenses, but it is intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. The main purpose is for evaluation purposes. The great thing about them, unlike trial Microsoft software, MSDN or TechNet keys don't expire. Because the agreement under which the subscription is provided is a single license, none of the software should be distributed outside of it. Even though its $50,000 worth of licenses, it is for one person only to use and no one else. Unfortunately, regardless of the licensing terms, persons still abuse the program, either giveaway product keys or resell it on auction sites. Microsoft licenses it in good faith that customers won't do so, but I guess human nature wins out.

In your case, what probably happened is, you bought an MSDN licensed key, which carries up to 10 activations unlike full packaged retail licenses which only carry 1 activation. The person who sold it to you probably sold it to 10 other persons. Somewhere along the way, one of those persons might have installed it on a second system, activated it, because it went past 10 activation threshold, Microsoft detected it that it was being abused and blocked the key from further use.

Finally, A Solution to the Game-Breaking Glitch to Microsoft Treasure - Microsoft Community

achievehunter21 say:

Finally, A Solution to the Game-Breaking Glitch to Microsoft Treasure Hunt?!?

Hi!  I posting this here since the Facebook group doesn't appear to give feedback.  I'm hoping this will reach the right ears within Microsoft.

As you know, there is a game-breaking glitch within the game of Microsoft Treasure Hunt.  Whenever the game is closed voluntary or shutdown involuntary, the player's progress as well the items he/she obtained will be lost for good.  The player is then forced to start all the way back to the very beginning to level 1 to start again.  This is particularly heartbreaking as this game is really fun.

While working with the game, I confirmed reason on why the game often forces a shutdown.  The longer any program remains open on Windows, the taxing it is to keep open, this is why, the more you progress, a noticeable lag starts to affect the gameplay.  The major reason the program starts to become taxing are the videos that are programmed to play whenever players enter the Bonus room.  Videos are very resource hungry and, sooner than later, will force the game to close abruptly.

Well, I found a way in which the game will continue to keep my progress and my items, even if I am forced to close the game when I'm not finished.  The solution: a) keep the game window at least half the size of the monitor one is playing on and b) keep the game minimized in the Windows taskbar when not in play.

Keeping the game window small will help the program not get bogged down on needing so many resources to keep open.  As for minimizing the game, I haven't fully figured out why, but often when I'm forced to reboot my PC for another Windows update or when my screensaver inadvertently forces my game to close, each and every time, when I open the game back up again, my progress is still there, my items is still there and, most importantly, my patience is still there.

It's tough to have wait 3+ years for a solution on a glitch that frustrates so many players.  My thinking is that a more readily accessible save-state would better serve the game and its players.  But, until then, I recommend to try these to tips and see if it works out for you.

*This works for PC users.  For those with tablets, I don't know if the same solution applies.

brandonstead say:

WindowsUpdate_800705B4 error forNVIDIA driver update

I'm trying to complete some optional updates for windows 8.1 and the NVIDIA driver update listed under optional updates downloads and fails to install displaying the following error code: WindowsUpdate_800705B4


I followed the error code info the Microsoft website and ran the Windows Update Troubleshooter program which stated the following:


Problems with recent updates: Not fixed

Check for missing or corrupt files: Detected

Windows update error 0x80070057 (2013-11-28-T-11_06_44P): Fixed


The issue still remains installing the driver. The update is no longer appearing under new windows updates possibly because I have already downloaded it. I'm clueless as what to do now. What should I do?

a. user say:

Hi Brandon,

You are getting this error code because you have already installed the latest driver for NVIDIA on the computer. If you try to install the driver from the Windows update then driver clashes will occur and the graphics card won’t work properly. So I suggest you to hide this update if it offers again.

 

To hide an update

 

a.       Swipe in from the right edge of the screen, and then tap Search.
(If you're using a mouse, point to the upper-right corner of the screen, move the mouse pointer down, and then click Search.)

b.      Enter Hide or unhide updates in the search box, and then tap or click Hide or unhide updates.

c.       Do one of the following:

    If you have any important updates, click the link that tells you important updates are available.

    If you have any optional updates, click the link that tells you optional updates are available.

d.      Select any update that you want to hide, press and hold or right-click it, and then tap or click Hide update.

 

Please post back if you face any issues related to Windows and we will be glad to assist you further.
