How can I delete an 'I-beam' artifact from a Word - Microsoft Community

darrenmcintosh say:

How can I delete an 'I-beam' artifact from a Word template?

This artifact appears on a few pages in our template. I've tried everything I can think of to delete this. Any ideas? 

jay freedman say:

It appears to be a bookmark. That's what a bookmark looks like when it doesn't enclose any text -- the two brackets [ ] are pushed together so they share the vertical edge.

If you're using the bookmark for some purpose, such as the target of a hyperlink, then you can just turn off the display of bookmarks. Go to File > Options > Advanced, scroll down to the "Show document content" section, and uncheck "Show bookmarks".

If you don't need the bookmark at all, go to the Insert ribbon and click the Bookmark button in the Links group. If there are multiple names in the list in the Bookmark dialog box, select one and click the GoTo button; repeat until the cursor lands on the bookmark you want to remove. With that one selected in the list, click the Delete button.

Can a Windows product of Office be used on a Mac?

If you have a program for Office 365 for a windows PC, and you later purchase a Mac, will you be able to use the windows office 365, or do you have to purchase the office produc specifically for Mac?

rohn007 say:

Office 365 is different from "traditional" Office installations before 365.  Before 365 you had to buy the Office specific for PC or MAC. You could not run the version on the "wrong" operating system. You could not use a Windows key on a Mac installation (and visa versa).

365 has changed that somewhat.  You still need to use the installation file specific for the operating system, but your 365 license (product key) allows you to use your installation count on either operating system, as you need.
