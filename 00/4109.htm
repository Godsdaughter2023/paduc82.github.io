How to enable desktop Control Panel on Start menu in Windows 10 - Microsoft Community

andre da costa say:

How to enable desktop Control Panel on Start menu in Windows 10

Technical Level : Basic

Summary

Since the release of Windows 8, Windows now includes two applets for configuring system settings. One in particular is optimized for Touch (PC Settings) while the other is more geared towards desktop users utilizing mouse and keyboard. In this wiki article, we show you how to enable the classic Control Panel for quick access.


Details

UPDATE: The information in this article has been deprecated in Windows 10 builds 9926 and later. The classic Windows Control Panel can only be pinned to Start. It cannot be pinned to the Recent apps list like in previous builds.

To Pin classic Control Panel to your Start, type Control Panel in Search box on the Taskbar. Right click it and click Pin to Start.

If you pin the Classic Control Panel to your Taskbar though, you can still access the Jump List.

- Andre

23/03/2015

If you are familiar with your keyboard commands, the classic Control Panel can be accessed quickly. Just press Windows key + X on your keyboard.

This will launch a contextual menu where you can easily find Control Panel.

If you want to have it pinned on the new Start menu in Windows 10, here is how you do it.

Right click the Taskbar > click Properites

Select the Start menu tab and click Customize

Check off Control Panel under 'Pin to Start list'

Click OK

The Apply and OK

Click Start and you will see the Control Panel pinned to your Start menu.

carey dean say:

The summary task duration unit of time insists on reporting days only.

In Project 2007, I customized a duration field with formula to display totals from two other duration fields. The subtasks are fine, but the summary tasks insists on using the Days unit of time.

I ran the macro to change all durations, and the clicked on the cell, Edit/Clear all but to no avail. I would appreciate any information regarding this issue. Thanks.

john - project say:
Carey,
Pardon my bumping in. Sometimes you just gotta go with the flow. Assuming your two duration fields are Duration1 and Duration2, the following macro will place the values you want into extra field Text1 for both summary lines and performance tasks. You can adjust the time dimension (e.g. hours, days, etc.) with a few tweaks to the code.

Sub RollupDurInHrs()
Dim t As Task, st As Task
Dim TSum As Single, RunSum As Single
For Each t In ActiveProject.Tasks
    If Not t Is Nothing Then
        If t.Summary = True Then
            RunSum = 0
            For Each st In t.OutlineChildren
                TSum = (st.Duration1 + st.Duration2) / 60
                st.Text1 = TSum & " hr"
                RunSum = RunSum + TSum
            Next st
            t.Text1 = RunSum & " hr"
        End If
    End If
Next t

Hope this helps.
