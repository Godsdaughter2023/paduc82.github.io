¡firedog say:

OK. The first rule of troubleshooting is change one thing at a time and test, because otherwise you won't be able to tell whether in fact one of them was the right one. When you conducted this latest test, you were connecting from somewhere else (the IP address belongs to NTL). And it I would guess that you were connecting from a different machine, one with Avast antivirus installed. However, this guess may be wrong and it was in fact the same machine.

If there is no connection to the server ("Cannot connect to SMTP server"), the server cannot return an error ("Server Response: 421"). So the error message is most likely not coming from the server, but from an anti-virus program that has insinuated itself between client and server. If Avast is installed, switch off its Mail Shield. This adds no extra security (especially on outgoing messages) and can cause problems.

Port 587 is the normal one for not-secured, authenticated SMTP. Because SSL is a complex process more prone to failure simply because of that complexity, it's always worthwhile to check whether not using SSL improves performance. NTL clearly doesn't accept this.

Now, having turned off mail scanning in your security software, let's try with SSL (on the same NTL connection as last time). On the Advanced tab, select This server requires a secure connection (SSL) for the outgoing server and change the port to 465. While on that page, increase the server timeout to two minutes. Then remove any unsent messages from the outbox and close down WLMail. Wait a few minutes, then re-launch the program and try composing and sending yourself a new test message with a different subject (so we know which message isn't getting through). Post any new error message here.

NTLWorld accounts are notorious for poor performance, as you can see on the Virgin forums. A common remedy which succeeds for some customers is to use the Virgin SMTP server (smtp.virginmedia.com) instead of the NTLWorld one. If you still can't send, this is the next thing to try. The port settings should remain the same.

Noel

New computers that come preinstalled with Windows 7 often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

 

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee.

------------------------------------

This is how the recovery partition is accessed for the most popular makes

For Dell, press CTRL + F11 directly after powering up the unit

For HP, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

internet expert say:

The article that Miss Bijoux linked suggests several things that could be at fault:

  • A) You aren't signed into the same profile that purchased the digital game.
  • B) The DRM license of the game does not match the console it was originally purchased on.
  • C) There is something wrong with the DVD drive.
  • D) Your game installation is corrupt.

I don't know why the third thing takes you to the disc drive troubleshooter, but it will tell you to remove any flash drives that are plugged into the console, and then it will eventually tell you to take it in for a service repair if it still doesn't work.

The "disc unreadable" error occurs when the Xbox is unable to read files from the source destination, which in OP's case, is the hard drive.

So, the solution here (in order):

  1. Ensure you are signed into the same profile that purchased the game.
  2. If you originally purchased the game on a different Xbox 360 console, perform a license transfer.
  3. Remove any flash drives plugged into the console.
  4. Uninstall, re-download, and reinstall the game.

If your game still does not launch, then you may very well have an issue going on with the hard drive, and the hard drive must ultimately be replaced.

Hope this all helps!
