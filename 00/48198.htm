a_user_here say:

Here is my long winded answer on this.

There is only one reason that going to 64-bit would make your programs run faster, and that is if the programs are starved for memory(Highly unlikely, you would know by how slow they are running).  Video editing "might" push that a bit.  You can look at this in the Task Manager.  (Just search for it). In general you could just look at how much memory each program is using and get an idea each of them is using in different situations.

The max memory 32-bits can use is 4GB, but most likely it is even less that that because some of the address space might be used by hardware like your display.  You can see how much memory your machine actually can use with Win key + X -> System ... (XX GB usable)

Now if you go to 64-bit you will be able to use more of the memory, but at 5 GB it might be pretty much of a wash, because see some addresses will take 64-bits instead of 32-bit.  Not all instructions have addresses and not all of them would have to be the full 64-bits, but on average you can figure 64-bit programs will take between 1.25 and 1.5 more memory for the same program.

You can go to from 32-bit to 64-bit for free, but it will have to be a clean install, not an update.  So you would have to backup your data files, and restore them after the install.

Here are some details on the procedure.

http://answers.microsoft.com/en-us/windows/wiki/windows_10-win_upgrade/how-to-switch-from-32-bit-versions-of-windows-to/40d90c4e-8845-4bcb-bb2f-aeab312caf74
