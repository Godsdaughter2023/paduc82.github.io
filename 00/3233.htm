Private Sub browse_file_Click()

Dim wb1 As Workbook
Dim wb2 As Workbook
Dim Sheet As Worksheet
Dim PasteStart As Range

Set wb1 = ActiveWorkbook
Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = "RawData"
Set PasteSpecialFormatsStart = [RawData!A1]

FileToOpen = Application.GetOpenFilename _
(Title:="Please choose a Report to Parse", _
FileFilter:="Report Files *.xlsx (*.xlsx), *.xlsm (*.xlsm),*.xls (*.xls),")

If FileToOpen = False Then

    For Each Sheet In wb2.Sheets

            Set PasteSpecialFormatsStart = PasteSpecialFormatsStart.Offset(.Rows.Count)

    Next Sheet

petrib say: What does "I cant UPDATE" mean, exactly?

Is the 2630 supported with Nokia Software Updater (NSU) in the first place?

If not, then you have to take the phone to a Nokia service/repair center and ask them to update it for you.

If the phone model is supported for NSU updates, then the question is whether an update is available for your phone model in the first place.

If it is, then the next question is, whether a version is also already available for your country/region.

If it is, then the next question is, if you are using a network operator (carrier) customized/subsidized phone, whether the operator has ordered (paid) for a new customized version to me made, and at what stage in development/availability it is.

And if you have a phone supported by NSU and a new firmware version is available, and it still doesn't work for you, then you need to explain more about how you attempted to use it, and how it failed (such as any and all error messages, etc.).
