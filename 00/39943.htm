Unhandled Exception Error (0x800401A) when starting Windows 7 for the first time.

When I start Windows 7 for the first time I am receiving this error message:

“Unhandled exception has occurred in your application. If you click Continue, the application will ignore this error and attempt to continue. If you click Quit, the application will close immediately.

 The server process could not be started because the configured identity is incorrect. Check the username and password. (Exception from HRESULT: 0x800401A)”.

Why is this happening?

dan v [msft] say:

Hi John,

This message may appear when the following occurs:

- You are starting Windows 7 for the first time
- You have clicked on the Ease of Access button in the lower left corner of the setup window
- You click Hear text on screen read aloud (Narrator) and click Apply
- You click the link labeled Control whether Narrator starts when I log on in the Microsoft Narrator screen

This issue can occur during the Out of Box Experience (OOBE) portion of setup as the system is in a state where user accounts have not been created. The error will only happen during this portion of setup.

Once you have created a user account (at a later point) and log on to this user account the link for Microsoft Narrator will work as expected.

sigmathedj say:

It probably is coil whine, in which case the only real options are to get it replaced or repaired, assuming it's really bothering you. If it's under warranty, contact Xbox Support and they'll be able to help you.


The reason it happens in certain situations and not others is because a certain level of power draw can cause components inside the console to vibrate, which is what generates the noise.
