a. user say:

Hi,

Thank you for your update on the issue.

Yes, you need to reconnect the external drive to backup the image on your portable hard drive.

On the other hand, Event ID: 12348 is generated for the Volume Shadow Copy Service (VSS) if the service does not have access to the volume root directory, the drive partition in which the operating system is installed and so the shadow copy creation of the volume may fail.

The Volume Shadow Copy Service (VSS) provides the ability to create a point in time image (shadow copy) of one or more volumes that can be used to perform backups. The service is also used during restores of applications.

So, for the prevention of this event generation, we need to ensure that the VSS has access to the volume root directory. To do so, you may follow the following steps.

  1. Go to Start, click All Programs and open Accessories.
  2. Right click on Command Prompt in the Program list and then select Run as administrator. If you are prompted for an administrator password or for confirmation, type your password, or click OK.
  3. In the Command Prompt, type the following command, and then press ENTER:
    icacls  C:\  /grant  system:f
    (Note: The above command is applicable if your Windows is installed in the partition C: of the drive. You may it replace it with the appropriate drive letter.)

Hope this information is helpful. Please post back for any further assistance.

arya s asok say:

Hi Western Massachusetts,

 

In Windows you do not have the option to change the color of the icon.

 

Firstly, you need to restore all the folders icons back to default Windows folder icons. Also remove any third party application you have used to change the folder icon and colors to get the default view of the folder icons.

 

To restore the icons to default Windows icon, follow the steps below.

 

1.      Navigate to the location of the folder.

2.    Right click on the folder and select "Properties".

3.    Select "Customize” tab.

4.    Click onthe "Change Icon" button is at the bottom and click on restore default button.

 

This will change the icon to default Windows Icon. Later you will have to search internet using your favorite search engine for some third party application to change the folder icon colors.
