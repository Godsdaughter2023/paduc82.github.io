Remove Bing as the default homepage or search engine - Microsoft Community

debbie_h say:

Remove Bing as the default homepage or search engine

Summary
Technical level: Basic You can change the default homepage of your browser from Bing to something else.Also, you can change your default search engine if you no longer want Bing Search.
Details

How to change the default homepage:

 

1.  Open Internet Explorer

2.  Click on Tools -> Internet options

3.  Under the General tab, remove the URL contained in the Homepage box and replace it with whatever website you want as your new default homepage, example: http://www.cnn.com etc.

4.  Click OK

 

Note:

  • Instructions are the same for Internet Explorer 10. 
  • These settings are not available / do not apply to the modern version of Internet Explorer.

 

How to change the default search engine in Internet Explorer 9 and previous:

 

1.  Open Internet Explorer

2.  Click on Tools -> Internet options

3.  Under the General tab, click Settings on the Search section

4.  On the left pane, choose Search Providers

5.  A list of the available Search Providers will show up, highlight the one you prefer and click on Set as default

6.  Click Close and OK

 

How to change the default search engine in Internet Explorer 10:

 

1.  Open Internet Explorer

2.  Click Tools and select Manage add-ons

3.  Under Add-on Types, select Search Providers to view the current search engine selections

4.  If you don’t see the one you want, click the ‘Find more search providers’ link on the bottom left portion of the window for more options

 

To see a list of Search Providers and other add-ons available for Internet Explorer, you can look at the Internet Explorer Gallery here.

vchirlanza,
There are various ways to do what you want. Perhaps the easiest is to break the work into monthly tasks and then assign the resource accordingly. For example, break the electrical work into a July task and an August task. For July assign 23 electricians and for August assign 11 electricians.
