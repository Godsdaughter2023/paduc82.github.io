Runs when you select a sheet with this code in it.

Private Sub Worksheet_Activate()
    MsgBox "hello"
End Sub

Placed in Thisworkbook module runs when any sheet selected.

Private Sub Workbook_SheetActivate(ByVal Sh As Object)
   MsgBox "hello to all"
End Sub
