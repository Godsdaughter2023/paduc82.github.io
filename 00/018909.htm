Grant File Access - Question - Microsoft Community

lpkw76 say:

Grant File Access - Question

I am working on a document in microsoft word. However, when I go to open my document, there is message with the heading: Grant File Access. Microsoft Word needs access to the following files. I am wondering what this means? Should I grant access? 
john mcghie say:

Microsoft is not putting up that message, Apple is. :-)

Apple requires all applications sold through the Apple Store to be "sandboxed", and Microsoft has to comply along with everyone else.  As Philip mentions, the idea is to make your computer more secure against nasties.

Out of the box, Word installs with access to anything within your user "Documents" folder, and anything within its application folder /Users/<Your_Login_Name>/Library/Group Containers/UBF8T346G9.Office

If you have a document that is attached to a file outside those locations, you will be asked, one-time only, to grant access to the location. If you choose to do so, your decision will be remembered and you will not be asked again.

Files that can cause this are Templates, if they are outside the Documents folder, spelling dictionaries (especially custom dictionaries, if they are in the old location, and perhaps backup files (unlikely)).

In each case, Apple will show you where the wanted file is, and you can make your decision.  If you say "No", whatever was in that file will not be available. So your Custom Dictionary, for example, may not work.

Normally, in the early days of a new installation or a new Mac, you may see this prompt three or four times, and your decision will normally be "Yes". If you have been using Word a month or two and you see the prompt, especially if you did not expect it (have not, for example, just opened a document...) then think very carefully, because some bad actor may be trying something nasty on your computer.

You may also wish to review the locations within which you store things.  Most users would never see this prompt, or see it only once.  If you see it more than that, it could indicate that you are storing things like templates and spelling dictionaries where Apple does not expect them.  If that is the case, should you move them?

Only you can say!  I moved my templates into my Documents folder (where I should have had them in the first place, so they would be correctly backed up...).  I chose not to move my spelling dictionaries, because I want Office 2011 to have access to them as well.

Your mileage may and will vary!

hansv mvp say:

Save the workbook first, specifying the file format, then close it. With a variable wbk of type Excel.Workbook:

wbk.SaveAs Path & "\" & FileName & ".xls", xlExcel8
wbk.Close

If you use late binding (i.e. you have not set a reference to the Microsoft Excel 12.0 Object Library), replace xlExcel8 with its value:

wbk.SaveAs Path & "\" & FileName & ".xls", 56
wbk.Close
