¡firedog say:

You should be able to return the messages in the deleted folder to WLMail by using the Import messages function (Alt-F I). Point the wizard at the backup folder containing the folder containing the messages, not at the folder containing the messages itself. This should populate an Imported folders folder under Storage folders.

You can then re-create the account folder and right-click > Copy to folder to move messages from Storage folders to the re-created folder under the account. It's best to do this while Working offline to avoid interference from any sync process. Only Go online when WLMail has settled down after relocating the messages, to give the database time to catch up. If a lot of messages are involved (hundreds, say), it might be best to let WLMail settle down, then close down the program to allow the log files to be replayed. Re-launch after a few minutes, Go online and test.

If this is successful, you can then delete any supernumerary folders and messages under Storage folders.

This might be due to incompatible\outdated display adapter.

I would also suggest you to check with a different HDMI cable and check if the issue persists.

Let us try the following steps, and check if it helps. 

Step 1: Run the Hardware and Devices troubleshooter.

Hardware and Devices troubleshooter checks for common problems and makes sure that any new device or hardware connected to your PC is installed correctly.

  1. Type troubleshooting in the search bar.
  2. Select Troubleshooting.
  3. Select View all on the top left corner.
  4. Click on Hardware and Devices.
  5. Follow the on-screen instructions to run the troubleshooter.
  6. Check if issue persists.

  1. Press Windows key + X and click on Device Manager.
  2. Expand Display Adaptor.
  3. Right click on the video driver and select Update Driver Software.
  4. Restart the computer.

3. Select “Compatibility” Tab.
4. Place a check mark next to “Run this program in Compatibility mode” and select operating system from the drop down list.
5. Let the driver install and then check the functionality.
