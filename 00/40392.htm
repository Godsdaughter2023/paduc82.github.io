dave n9 say:
The documents are really in the second folder, the Documents and Settings is
a junction point.
 
In Vista and in Windows 7, "Documents and Settings" is not a folder.
Vista/Win7 uses a different file structure than XP does/did. Those familiar
folder names you may be used to, like "My Documents", "Documents &
Settings", etc. are not folders in Vista/Win7. They are junction points,
and are used for legacy programs which were written to utilize the XP file
structure.
They will redirect the programs to use the equivalent Vista/Win7 folders.
If you keep the protected operating system files hidden, you won't see them.
 
In Vista/Win7......
Documents & Settings -> \Users
My Documents -> \Users\youraccount\Documents
My Music -> \Users\youraccount\Music
Application Data -> \Users\youraccount\AppData
etc.

Hi Theo,

Thank you for writing to Microsoft Community Forums.

Follow the below steps in order to create one partitions from two different partitions.

NOTE: Before performing the below steps, backup all the data which is in H and G partitions as you will lose all the data in these partitions, which is not recoverable.

Step 1: Press the Windows key and the R key from your keyboard and type Diskmgmt.msc and hit Enter.
Step 2: Now, find your H and G partitions in Disk Management > Right-click on the H partition > Click on Delete Volume > Yes.
Step 3: Now, right-click on G partition > Delete Volume > Yes.
Step 4: Once you have deleted both the partitions, a new Unallocated partition will be created. Right-click on that unallocated partition > New Simple Volume > Next > Next > Assign any letter for the new partition and click on Next > Again, click on Next for the File System Format and click on Finish.
Step 5: Then, go to File Explorer and Format the disk if it prompts and you will see one partition in File Explorer.

Hope it helps.
