Col A                         Col B
7/26/2012 01:00     1(call)
7/26/2012 03:00     1(call)

So between 01:00 & 03 :00 I have no data & I need to insert these timesteps with a zero as the call data in the next column . Trouble is, the spreadsheet is massive & will take an eternity to do manually. Any quick workaround for this?
Steven
mike h.. say:

Hi,

 

This code will input this missing timestamps. ALT+F11 to open vb editor, right click 'ThisWorkbook' and insert module and paste the code in on the right and run it

 

I have assumed you 1 header row in these columns

 

Sub insert_Times()
Dim LastRow As Long, x As Long
Dim diff As Long, c as Range
LastRow = Range("A:A").Find(What:="*", After:=[A1], _
SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row
For x = LastRow To 3 Step -1
    diff = (DateDiff("n", Cells(x - 1, 1), Cells(x, 1)))
    If diff > 30 Then
        Cells(x, 1).Resize((diff / 30) - 1).EntireRow.Insert
    End If
Next
  LastRow = Cells(Cells.Rows.Count, "A").End(xlUp).Row
  For Each c In Range("A2:A" & LastRow)
    If IsEmpty(c) Then
    c.Value = c.Offset(-1).Value + TimeValue("00:30:00")
    c.Offset(, 1).Value = 0
    End If
  Next
End Sub
