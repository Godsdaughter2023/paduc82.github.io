No you do not need a lot of data to start with. It is actually easier to start with a smaller set of rows while you are setting up the formulas you want to work with. Actually, you only have 5 columns of data, columns A to E, ie Date, Description, Withdrawls, Deposits and Balance.

after you create the formula the first time you can Select it, copy it then paste it into the rows below. Excel will adjust the cell names for you.
