Hi

I see that there is some confusion being created by the use of the term AutoSave. I can easily understand how you might be confused by this term because it sounds very similar to another feature called AutoRecover.

I have little patience for Microsoft agents who are not absolutely clear about these two features. They should know these features cold. They don't.

Please allow me to explain.

There are two very similar sounding features, and they both work to recover files in the case of a crash. In both cases, you must have saved your file at least once before you can recover it:

AutoRECOVER: This feature is for files saved locally on your Mac. You can specify the location for AutoRecover files in Word > Preferences > File locations. Check there first. If you specified a location, go there to find your AutoRecover files. Otherwise, in Finder, click the Go menu and choose Library and navigate to /Containers/com.microsoft.Word/Data/Library/Preferences/AutoRecovery

AutoSAVE: This feature only activates when you have saved your file to OneDrive or SharePoint. If you save your file locally on your Mac, this feature is disabled.

The way Word is designed to work, if there is a Word document in the AutoRECOVER location, Word will attempt to open it when Word opens. If that particular document is corrupt, it can cause Word to crash or have other undesirable behavior. The cure is to remove all Word documents from the AutoRecover location. Word must be completely closed before you remove files from this location. If in doubt, restart your Mac before removing these files before you open Microsoft Word.

gary's student say:

I have had good luck with this approach:

  • put a formula on the worksheet that will calculate if the data in cols A thru C is refreshed
  • use a calculate event macro to timestamp the time of calculation

So:

  1. Make sure D1 is empty
  2. Make sure data refresh is enabled
  3. Pick a cell and enter   =COUNTA(A:C)
  4. finally enter the following macro in the worksheet code area:

Private Sub Worksheet_Calculate()
If IsEmpty(Range("D1")) Then
    Range("D1").Value = Now
End If
End Sub

 


Because it is worksheet code, it is very easy to install and automatic to use:

   1. right-click the tab name near the bottom of the Excel window
   2. select View Code - this brings up a VBE window
   3. paste the stuff in and close the VBE window

If you have any concerns, first try it on a trial worksheet.

If you save the workbook, the macro will be saved with it.


To remove the macro:

   1. bring up the VBE windows as above
   2. clear the code out
   3. close the VBE window

To learn more about macros in general, see:

http://www.mvps.org/dmcritchie/excel/getstarted.htm

To learn more about Event Macros (worksheet code), see:

http://www.mvps.org/dmcritchie/excel/event.htm
