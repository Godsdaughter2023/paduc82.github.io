hansv mvp say:

Try this macro. Please test it on a copy of your document.

Sub ChangeText()
    Application.ScreenUpdating = False
    Selection.HomeKey Unit:=wdStory
    With Selection.Find
        .ClearFormatting
        .Font.Underline = wdUnderlineSingle
        .Font.ColorIndex = wdRed
        .Text = ""
        .Wrap = wdFindStop
        Do While .Execute
            Selection.Font.Underline = wdUnderlineNone
            Selection.Font.ColorIndex = wdAuto
            Selection.Text = String(Len(Selection), "_")
            Selection.Collapse Direction:=wdCollapseEnd
        Loop
    End With
    Application.ScreenUpdating = True
End Sub

christine_ say:

As far as I know you still can't mix landscape and portrait slides in the same presentation.

Best practice is to use just one orientation (landscape filling the available space best), but if it is absolutely imperative to use a portrait slide, you could develop two presentations (one portrait, one landscape) and then use the Action function to link your portrait slide to the first slide in the landscape deck.

You can find Actions under the Insert tab. Choose an object on the slide you want to "jump" from, go toInsert - Action , underMouse Click (I prefer this to Mouse Over because "I" would inadvertently set the action off before I was ready...YMMV) chooseHyperlink to: and scroll down toOther PowerPoint Presentation . At that point you may choose the whichever slide you wish to "jump" to in the second presentation.

When presenting, have both presentations open so when you initiate your action from your first (portrait) slide it should move smoothly onto your second (landscape) slide and continue on with that presentation.

Kind of a work around which I'm sure Steve has some super duper plug in to take care of, but it gets the job done! :-)

How about photo thumbnails for Word document icons? - Microsoft Community

pamelapratt say:

How about photo thumbnails for Word document icons?

How about photo thumbnails for Word document icons?
jay freedman say:

They're available. When you save a new document (or an existing document to a new name or folder) and the Save As dialog appears, check the box at the lower left for "Save Thumbnail".

I wouldn't advise doing this for all Word documents, for two reasons. The main one is that you can't tell a lot about most Word documents by looking at a thumbnail; they all look more or less the same, and it would be better to rely on using descriptive file names. The other reason is that the thumbnail is saved in the document's file, making the file larger and slower to load than necessary.

jay freedman say:

If you want one non-default picture icon applied to all Word documents, you can do that. Either find an icon file (extension .ico) on the Net or use an icon editor such as this one to make an icon file from a picture.

Then look at http://www.online-tech-tips.com/computer-tips/change-file-icon/ or http://www.howtogeek.com/howto/12383/change-a-file-types-icon-in-windows-7/ for instructions on how to assign that icon to the file type.

If you want a different picture for each document, you'll have to do that to shortcuts to the files, not the files themselves. See http://www.7tutorials.com/how-change-icon-shortcut.
