Yes, but it will get EXTREMELY slow if you force it to monitor 5,000 folders.

Go to Outlook>Preferences>Accounts, select your IMAP account, then click Advanced.

Click the Server tab, the enable "Synch all IMAP Folder every nnn minutes"

You must set the number of minutes long enough so it can complete synching before starting again, otherwise it will lock up.  And note that you will be burning through a lot of bandwidth, so don't enable it on a cell phone connection :-)

But why would you NEED to synch all 5,000 folders?  Is there any way emails can get into those folder without you putting them there?  (In my case, I have server side rules running, but they don't shift anything from one folder to another, so this is not a consideration for me.)

I synch only the Inbox, where stuff arrives.  If it subsequently goes to a folder, that's because I put it there, and Outlook will synch that folder when I do.
