andreas killer say:
Obviously I can turn Protection Off & On again, but would love to understand why it works in only one case?
I can not understand why your code works at all. :-)

The first thing I did is to remove the "On Error Resume Next". After that your code did not work properly, when I click the ToggleButton. But when I run the sub manually it works. The question is why.

The reason is simple, you can only change Validation.ShowInput when a cell in the sheet is selected. But when you click the button, the button is selected and no cell! Is it a BUG? Seems so.

The simple workaround it so re-select the current selection. I suggest to rewrite your macros like the sample below.

Andreas.

Private Sub tog_DATA_Valid_Help_Click()
  Dim Cell As Range
  'Handle the errors!
  On Error GoTo ErrorHandler
  'Don't trigger the selection event!
  Application.EnableEvents = False
  'Re-select the current selection in the sheet
  ActiveWindow.Selection.Select
  Application.EnableEvents = True
  'Visit each cell
  For Each Cell In DATA.Range("FullData")
    Cell.Validation.ShowInput = tog_DATA_Valid_Help.Value
NextCell:
  Next
ErrorHandler:
  Select Case Err.Number
    Case 0: 'No error
    Case 1004
      'This happens when the cell is protected, etc.
      'Show the error in the immediate window
      Debug.Print "Error in cell : "; Cell.Address(0, 0)
      Resume NextCell
    Case Else
      If Err.Source = "" Then Err.Source = Application.Name
      Debug.Print "Source     : " & Err.Source
      Debug.Print "Error      : " & Err.Number
      Debug.Print "Description: " & Err.Description
      If MsgBox("Error " & Err.Number & ": " & vbNewLine & vbNewLine & _
          Err.Description & vbNewLine & vbNewLine & _
          "Enter debug mode?", vbOKCancel + vbDefaultButton2, Err.Source) = vbOK Then
        Stop 'Press F8 twice
        Resume
      End If
  End Select
End Sub

ken sheridan say: You are catching it too late in the day with the Load event procedure.  The form's Open event procedure would be a better solution as this has a Cancel argument whose return value can be set to True:

Dim PassWord As String
PassWord = InputBox("Enter Administrator Password")

If PassWord <> "OBFF2013" Then
    ' cancel form
    MsgBox "You're not authorized for Administrator access.", vbExclamation, "Invalid operation"

End If
