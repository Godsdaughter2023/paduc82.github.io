What size do you want it?

To facilitate inserting multiple copies of it at the same size, insert one cell table into a document that has the desired cell dimensions and into that cell, insert the symbol and size it to fit.  Then select the table and go to File>Options>Proofing>AutoCorrect Options and add it as an autocorrect entry.

In that case, you could in another document

  1. Create a two row by two column table of the required dimension
  2. Colour the individual cells


  3. Select the table and use CTRL+c to copy it to the clipboard
  4. Then use Paste Special and paste the table as a Picture (Enhanced Metafile)

When you need a different set of colours and symbol, go back and repeat steps 2 thru 6

    **************************************************************

    1. Disable the Linked mailbox from Exchange (Using Powershell: Disable-Mailbox -Identity "MailboxName")

    3. Select User mailbox when Reconnecting and you should be able to see user mailbox instead Linked
