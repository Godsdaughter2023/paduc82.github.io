Some characters, with accents like: á é í ó ú, or with tildes like: ñ, are not displayed correctly, they are REPLACED by other characters, "

etc

I was about to format my pc when I read this:

Sometimes you install a font  that is incomplete: the creator made the font for english languaje, so he didn't draw the "ñ" or the accents, when you use that font those letters will be empty, the browser will search for that font if the webpage need it, if it didn't find it the browser will change to a "default font" like in windows, but if the browser do find it (like in case of using a mac computer or have installed the font in your pc) the browser will use it, and in all character like ñ or á will change it...  it happens a lot with the font HELVETICA
Try to identify that font and delete it from your PC, or install one with TILDE SUPPORT

There are a few reasons why the update is not appearing yet.

From what I have read they are as follows:

  1. It has been less than 31 days since you upgraded to Windows 10
    • The Threshold 2 update removes your ability to roll back to Windows 8.1
  2. You have defer upgrades checked
    • Check in Settings > Update & Security > Advanced Options
    • Make sure Defer upgrades is not checked.
  3. You previously installed Threshold 2 and then you removed it.

These are the most common reasons it would not appear other than as Bulldog mentioned the update just hasn't gotten to you yet.
