misterfizzle say:

Hi there Preached,

Sorry to hear you're having issues downloading Episodes 2 and 3.

I've done some digging into this and it seems like this is a common issue with those episodes.

Here's one fix I've seen posted that a few people confirmed worked for them:

[quote]

1. Make sure you have uninstalled any episodes causing the problem, either it is Episode 2, 3 or both of them. The uninstall process should proceed automatically after the installation failure, so just make sure the uninstall is finished before you proceed to step 2.

2. Reboot your PC. Don't Shutdown and then Power it on. Do a Reboot.

3. Close all programs

4. Go to Control Panel -> Windows Defender -> Settings and uncheck the "Turn on real-time protection..". Click Save changes. If you have some other Antivirus installed, disable it.

5. Go to Computer Management -> Services and Applications -> Services and locate the "Windows Installer" service. Click on it and go to the menu Action -> Start. Make sure it started.

6. Go and install the episode. If you face problems with both episode 2 and 3, first install episode 2. Don't start both episodes installation. Just one at a time.

7. Stay in the Adera screen and watch the Download and Installation process. Don't minimize or loose focus from the application. I know it takes a long time, but you have to be patient.

8. The installation should complete successfully after a few minutes. If you installed Episode 3, you are good to go. If you installed Episode 2, in order to install Episode 3, follow the steps 2 to 7 again.
 

Hopefully, the above steps, if followed carefully, will let you install the episodes.
 

Please take notice that maybe some of you have already managed to install it following SOME of the above steps. As the source of the problem is not always the same and varies from PC to PC, make sure you follow ALL the steps, even if some may not be needed in your case. Doing so will let you manage to perform the installation with minimum number of retries and lost time.[/quote]


Please let us know how that works for you and we'll go from there.

rjcrane say:

Hi AtomicVariable,

What you seeing is called Image Retention, this is caused by the phosphors that create the image to continue to glow after being in an excited state for a given time. The result leaves a temporary image trace that can linger on screen. The brighter the image on the screen, the more it excited the phosphor causing the trace to possibly last longer on screen.

To protect your plasma TV from image retention caused by the Xbox, try on to stay on the home or menu screens for a long period of time and try not to have your image brightness set too high. Also most modern day plasma TVs have something called Anti Image Retention under the settings that help avoids these screen burns, so check your TV setting to see if you have this and turn it on.

Let's create installation media for Windows 8.1 and perform automatic repair.

You may follow the steps below to re-download Windows 8 and also you can create installation media as well.

  2. When prompted to run or save the file, choose Run
