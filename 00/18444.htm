Hi Corrie,

 

If you want to filter on more than one value, I suggest you use a Query String (URL) Filter Web Part. You can follow the steps below to check the outcome.

1. Create a list containing the Multiple Choice checkboxes column named choice.
2. Create a page and insert the list.
3. Add the Query String (URL) Filter Web Part on the page.
4. Type choice in the Query String Parameter Name box, select Send all values in Advanced Filter Options and then click OK.
5. Select Connection in the list Web Part and then select Get Filter Values From> Query String (URL) Filter>choice in Consumer Field Name to connect the list Web Part to the Query String (URL) Filter Web Part.
6. Click Save to save the page.
7. If you type the page URL?choice D&E, it will return all items containing choice D and E. If you type the page URL?choice B&C&D, it will return all items containing choice B, choice C and choice D.
