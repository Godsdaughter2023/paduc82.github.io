I've never had Excel get confused with function definitions that are exact, especially when stored in the workbook that uses the function.

What may restore your sanity is to change from

=PrevSheet(B4)

to using built-in worksheet functions like

=INDIRECT("'"&$A$1&"'!B4")

For every sheet, you need to enter the previous sheet's name into a cell - whatever you want  in place of $A$1 - you could use a macro to enter the names, like

Sub EnterNames()

    Next i

gord dibben say:

The Prevsheet UDF version you posted makes no allowance for more than one workbook open so fails.

This version is designed to overcome that problem.

But Bernie may have made the use of Prevsheet UDF unnecessary so info only.

Function PrevSheet(Rg As Range)
'accounts for more than one workbook open
'and has hidden sheets
'Bob Phillips  October 4, 2009
Dim N As Variant
    With Application.Caller.Parent
        N = .Index
        Do
            If N = 1 Then
                PrevSheet = CVErr(xlErrRef)
                Exit Do
            ElseIf TypeName(.Parent.Sheets(N - 1)) <> "Chart" And _
                .Parent.Sheets(N - 1).Visible = xlSheetVisible Then
                PrevSheet = .Parent.Sheets(N - 1).Range(Rg.Address).Value
                Exit Do
            End If
            N = N - 1
        Loop
    End With
End Function

The actual issue that I missed is that you don't specify the parent workbook on this line:

Public Function PrevSheet(rCell As Range) As Variant

While you can add and remove elements in VBA, it makes for easier development (and maintenance) to create the whole form in design mode. Private Function GetATPUICultureTag() As String
    Dim shTemp As Worksheet
    Dim sCulture As String
    Dim sSheetName As String
    
    sCulture = Application.International(xlUICultureTag)
    sSheetName = sResourcePrefix + sCulture
    
    On Error Resume Next
    Set shTemp = ThisWorkbook.Worksheets(sSheetName)
    On Error GoTo 0
    If shTemp Is Nothing Then sCulture = GetFallbackTag(sCulture)
    
    GetATPUICultureTag = sCulture
End Function
