Select the range to which you want to apply the formatting. I'll assume that the active cell within the selection is cell A2.

=A2=A1

If you were talking about Windows Mobile technically Microsoft never did any updates for the phones themselves. They only did updates on the kits used to make the OS for the phone and it was up to the OEM to decide if they wanted to use the updated kit to make an updated OS image. 

For Windows Phone there's not a set number of updates or amount of time over which you will receive updates. The statements I've heard from Microsoft representatives is that phones will receive updates for as long as they are capable of running the update. Of course at some point in the future it might not be feasible to deploy certain features to phones that are not capable of meeting the performance or memory demands of the feature. But right now it's too early to start making projects of when that will be.
