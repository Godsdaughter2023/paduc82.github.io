yang zheng msft support say: Hi Okigs,

If you configure an IMAP account into the Outlook client, the calendar of this IMAP account will not sync with the IMAP server. So all the calendar items created in the Outlook client will only exist on this computer. In this case, the ‘This computer Only’ calendar will be automatically created for this IMAP account. If you remove this calendar, the calendar for the IMAP account could be removed. So you can’t remove this ‘This computer Only’ calendar.
