It would appear that you have the upper levels in the wrong order in the "Number format" box for those lower levels. When this happens, you need to either start over with the numbering scheme that shows 1, 1.1, 1.1.1, linking it to the desired styles, or start over in a given level by clearing the box and reinserting each element in order, starting with (for Level 3) the previous level number for Level 1, then Level 2, then the number style for Level 3.

A helpful tip that will make it easier to see when levels go wrong is to temporarily set the starting number for each level to the number of that level. In other words, set Level 2 to start at 2, Level 3 to start at 3, and so on. When you do this, then for Level 4, you should see 1.2.3.4. If you see something different, then you know the levels are in the wrong order. (Don't forget to set them all back to 1 after fixing the numbering and before leaving the dialog.)

My apologies. I've completely misread your requested conditions  by mashing up the first paragraph with your expected results.

=SUMPRODUCT(ISNUMBER(A1:A9)*((B1:B9="New")+(B1:B9="Renew")))

... or possibly,

=COUNTIFS(A1:A9, ">="&DATE(2000,1,1), B1:B9, "*NEW*")

The second one comes with a couple of caveats. First, Excel treats dates as numbers in a worksheet, so we can make an arbitrary filter for actual dates by specifying the lowest date accepted. andreas killer say:
Have a look at the code below how to get and copy data from an active Autofilter in a better way.

Andreas. 

Private Sub Example_GetAutoFilterRange()
  Dim Source As Range, Dest As Range
  
  'Setup the destination cell
  Set Dest = Sheets(2).Range("A2")
  
  'We can get the filtered cells from different objects in different ways:
  
  'From the active sheet, exclude headings
  Set Source = GetAutoFilterRange
  'From a special sheet, include headings
  Set Source = GetAutoFilterRange(Worksheets("Whatever"), False)
  'From a table in a sheet, exclude headings
  Set Source = GetAutoFilterRange(Worksheets("Whatever").ListObjects(1))
  
  'Do we have some?
  If Source Is Nothing Then
    'No, done
    Exit Sub
  End If
  
  'For example we want to have the 1st, 3rd and 4th column:
  Set Source = Intersect(Source, Source.Range("A1,C1:D1").EntireColumn)
  
  'Clear previous result, exclude headings
  Dest.Parent.UsedRange.Offset(1).ClearContents
  'Copy the values
  Source.Copy
  Dest.PasteSpecial xlPasteValues
  Application.CutCopyMode = False
End Sub
Function GetAutoFilterRange(Optional ByVal Parent As Object, _
    Optional WithoutHeader As Boolean = True) As Range
  'Returns the visible range of an Autofilter, Excel 2010 and above
  Dim r As Range
  If Parent Is Nothing Then
    Set Parent = ActiveSheet
    If Parent Is Nothing Then Exit Function
  End If
  'No filter, return nothing
  If TypeOf Parent Is Worksheet Then
    If Not Parent.AutoFilterMode Then Exit Function
  ElseIf TypeOf Parent Is ListObject Then
    If Parent.AutoFilter Is Nothing Then Exit Function
  Else
    Err.Raise 438, "GetAutoFilterRange", "Object " & TypeName(Parent) & " not supported"
  End If
  
  With Parent.AutoFilter
    'Get the whole range
    Set r = .Range
    'Remove headings?
    If WithoutHeader Then
      If r.Rows.Count = 1 Then Exit Function
      Set r = r.Resize(r.Rows.Count - 1).Offset(1)
    End If
    'Filter active?
    If .FilterMode Then
      'Error's off, we get an error if no cells are visible
      On Error GoTo ExitPoint
      Set r = r.SpecialCells(xlCellTypeVisible)
    End If
  End With
  'Return the result
  Set GetAutoFilterRange = r
ExitPoint:
End Function
