Like Steve says, it's hard to tell without seeing it first hand, but it sounds to me as though you saved your template with an existing Title Slide. Try this...

  1. Open the actual template [.potx] file
  2. Make sure you are in Normal View
  3. In the Slides Pane, select & delete all slides
  4. Save & close the .potx

If I'm right, you're currently getting a combination of what was called for by the original Title Slide Layout & your new version of it. Reselecting from the ribbon essentially is re-applying the new Title Slide Layout only. Removing the existing slide from the template should obviate the problem.

steve rindsberg say:

It's hard to tell w/o actually seeing this but it sounds like the kind of thing that happens when you remove the title placeholder from a master that expects to HAVE title/subtitle placeholders.  If you remove them, PPT will add them to new slides, and in a format/location of its own making.

It's better to leave the normal placeholders and format them the way you want OR start with a blank layout and format that.  

If you include a slide that's based on Blank and formatted the way you want it  in the template, it'll automatically become part of any new presentation people start from your template (though if they start from some other template and later APPLY your template, they won't get your slide).

johnb123rd say:

Within a 2011 Powerpoint Table, is there a way to add Tab Leaders when using tabs?

I have created a table in Powerpoint 2011 using the table builder function. But when I add a tab between two words and try to insert a dotted Tab Leaders, I can't find a method for adding the Tab Leaders like I can in Words table builder function. Is there a way to add Tab Leaders using Powerpoint 2011's table builder function?
julia.sommer say:

Since there is no dot leader tab setting in PowerPoint, the work-around I use is to set the tab as

  1. For each line in the 'table', select/highlight the tabbed area
  2. On PPT for Windows, in the Home ribbon Font group, click the corner "launcher" to access the full settings, then change the Font "Underline style" from "(none)" to the dotted line option.  [To access the dotted underline option on a Mac, right-click (or Command+Click) the selected tab to bring up a contextual menu and select Font.]
  3. Repeat formatting for each tabbed over line one at a time (F4 can repeat the last command and save clicks).

The above screenshot is a table of contents which is not an actual table, just text followed by a tab to a  page number.

In an actual table, use Ctrl+Tab to tab over to the end of the cell, select the tab and set the font to a dotted underline as described in step 2 above.

To avoid tabbing every 1/2" (the default), select the column of cells which will have tabs with dots, then set a tab stop (or stops) in the Home ribbon, in the Paragraph group, by clicking the corner settings launcher, then the "Tabs" button in the dialogue box, then entering the desired location for a tab stop.  Although the tab marker to the left of the ruler disappears when selecting more than one cell, the option is still available to set an entire column of selected cells when going through Paragraph > Tabs.

steve rindsberg say:

PowerPoint doesn't have tab leaders, unfortunately.  In regular text, the sometimes-useful workaround is to create the text in Word, then copy/paste into PowerPoint.  Whether this would work within table cells, I don't know.  Give it a try and let us know how it works out.
