tom ogilvy say:

Since you are running the code from a sheet module, object references for ranges assume they are qualified with me such as

cells.Select

are treated as me.cells.select ;  me referring to the Setup worksheet since that is where your code is located. 

You can change the code so those references are more fully qualified:

Sub HideUnHide()
'
' HideUnHide Macro
'
Dim SelectedRow As Integer
Dim WKSNm(1 To 10) As String
Dim sh As Worksheet
'
SelectedRow = Worksheets("Setup").Range("B4").Value
'
If (SelectedRow < 1) Or (SelectedRow > 32) Then
    MsgBox ("Invalid Selected.... Pls give row number in 1 - 32 range")
    GoTo ABORT:
End If
'
For i = 1 To 10
    WKSNm(i) = "R" & i
    Worksheets(WKSNm(i)).Activate
    Set sh = ActiveSheet
    sh.Cells.Select
    Selection.EntireRow.Hidden = False
    sh.Rows("3:34").Select
    Selection.EntireRow.Hidden = True
    sh.Rows("3:" & SelectedRow + 2).Select
    Selection.EntireRow.Hidden = False
    sh.Range("A2").Select
Next i
Worksheets("Setup").Activate
Range("B4").Select

ABORT:

End Sub

What does BLUE CLOUD beside a file in OneDrive mean? - Microsoft Community

carey vandenberg say:

What does BLUE CLOUD beside a file in OneDrive mean?

The STATUS in 1 of my OneDrive folder shows a BLUE CLOUD.  All the rest show GREEN CIRCLES WITH A CHECK MARK IN THE MIDDLE.  What does the BLUE CLOUD mean????
hansv mvp say:

The blue cloud icon means that the folder is only stored in the cloud, on OneDrive, without a local copy on your hard disk. So if you don't have an internet connection, you won't be able to view or edit the files in the folder.

The circle with check mark icon means that a local copy has been stored on your hard disk, so you can view and edit the files in the folder even if you don't have an internet connection.
