toddallcock say:

Sadly, the SD card on WP is for user content only- photos and videos created on the device, and synced music , photos and videos. For security reasons third-party apps have very limited access to the SD card, and store their own data in phone main memory. So, make sure any synced media you store on the phone is being synced to SD card, and not main memory, so more main memory is available for your apps.

ray gao msft say:

Hi Cdtekcfc,

 

PendingActivation and PendingProvisioning are two very similar provision status. PendingActivation means work load is available for user/tenant but need to backfill/provision. PendingProvisioning means we will provision the service when user start using it (just in time provisioning).

 

When users start to use the service which says PendingActivation or PendingProvisioning, the provision status will turn to PendingInput or Success accordingly.
