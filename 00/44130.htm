¡firedog say:

You didn't mention your version of Windows Live Mail, which has probably changed since this thread was started. I assume you're using WLMail 2011.

If you need only a list of names and email addresses for the category, you can get one thus:

  1. Start a new mail message (Ctrl-N) 
  2. In the To... box, type the name of the category
  3. Click the '+' sign to reveal the category members
  4. Press Ctrl-A C to select All and Copy
  5. Paste into your favourite word processor and use Find and Replace to convert the list into a neat table or a list of comma-separated values.

If you need all the contact data for the members of a category, you could use vCards:

  1. Create a new temporary folder on your desktop
  2. In WLMail's Contacts window (Ctrl-3), select the category in the left-hand pane
  3. Press Ctrl-A to select All
  4. Drag the selection and drop it into the temporary folder
  5. Open the folder, select All the .vcf files
  6. Right-click and select Send to > Compressed (zipped) folder .

You can now do what you need with the .zip file. Most mail clients can import .vcf files; in W7, you can import them en bloc into Windows Contacts.

The average Windows 10 machine has around 150 tasks, many more than you think. To see if one of them runs once every minute, you could do this:

  1. Open an elevated Command Prompt.
  2. Type these commands exactly:
    cd  /d  c:\Windows\System32\Tasks
    for  /r   %a  in  (*.*)  do  type  "%a" | find  /i  "/Interval"  >> "%UserProfile\Desktop\Tasks.txt"
  3. Open the desktop file Tasks.txt. It will look like so:
    <Interval>PT1H</Interval>
    <Interval>PT1M</Interval>
    <Interval>PT30M</Interval>
  4. See if you can find a line that contains the letters PT1M.

    mark ole say:

    Hi William,

    This could be a lockout issue due to incorrect PIN entry for BitLocker. You can try to suspend Bitlocker and see if your device will be able to boot Windows. To do that, follow these steps:

    1. Choose the “Skip this drive” link at the bottom of the page where you are asked to enter the recovery key. You should be presented with a menu that will let you get to a command prompt (The sequence is Advanced options -> Troubleshoot -> Advanced options -> Command prompt).
    2. Once you have a command prompt,  use the following command to check the BitLocker status of the C: drive:

      manage-bde -status c:

    3. If the status is returned as locked, you’ll need to use the following command to unlock it using your recovery password:

      manage-bde -unlock c: -rp <your 48-digit recovery password> 
                
    4. Once the drive is unlocked you'll need to use the following command to suspend protection:
       
      manage-bde -protectors -disable c:          

    5. Then exit and reboot.

    Reset your device if your device is still unable to boo in to Windows. If resetting the device did not work, restore you device using the recovery image.

    Feel free to post back if you have any concerns.

    cmdrkeene say:

    It depends if you had encryption turned on or not.  If you had encryption on, then you'll need a copy of your encryption key or the files will be locked forever.

    If you didn't encrypt your files, then you can recover the files simply by putting that hard drive into another computer and browsing it as if it was a USB stick (it's just a drive), you can even take ownership and re-set the permissions on the folder.

    Another option is to boot up using any Windows installer CD or USB stick.  At the install prompt, ignore it and press Shift+F10 to open a command prompt.  Then reach that drive (the letter may not be C: in this mode) and use the copy command to copy the file to another drive or USB stick.
