The issue was affecting all disabled users that has inactive mailboxes and "Hide from address list" check in on-prem Exchange.

We have found a workaround for this users.

Why does the option for Manual/Auto calc change? - Microsoft Community

bill_in_az say:

Why does the option for Manual/Auto calc change?

I work with many files daily and I find that some files seem to change my excel options from Auto calculating formulas to manual. Problem being that this setting stays that way when I open other files that I know are not supposed to be set that way.

 

Why does this happen, and/or how can I prevent it from happening?

joeu2004 say: Bill wrote:

I work with many files daily and I find that some files seem to change my excel options from Auto calculating formulas to manual. Problem being that this setting stays that way when I open other files that I know are not supposed to be set that way.   Why does this happen, and/or how can I prevent it from happening?


The calculation mode when the file is opened depends on the calculation mode already set in the same Excel instance.  Refer to http://support.microsoft.com/kb/214395.  In a nutshell:  "The first document opened uses the calculation mode with which it was last saved. Subsequently opened documents use the same mode".

Unless you need multiple files open in the same Excel instance due to external references, I suggest that you open each file in a separate instance.

That is not always easy to control.  Although you can change the properties of the Excel file, I have had some problems with that.  Moreover, any change you make to the properties will be undone if you update or repair the Office products.

For me, the more reliable method is to start Excel by clicking on the Excel program icon, not an Excel file icon.  Then use File > Open or Recent to open the desired file.

Or you might have an Open event macro in either the Personal workbook or the Excel file that forces Manual calculation mode, perhaps only under some conditions.
