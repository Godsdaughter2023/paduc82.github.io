ranmo_msft say:

Hi SAM,

DDGs in on-premises Exchange server cannot be migrated to Office 365. You need to create in the cloud manually.

You can decommission the existing Exchange server after all mailboxes are migrated to Office 365 and then you can solely use Office 365 as the Email service with DirSync to manage users and groups from AD. But please kindly note that, we strongly recommend our customers keep at least one Exchange server in local AD after migration so that you have a supported method to manage objects Exchange related attributes. Using the tool like ADSI edit to manage the objects Exchange attributes will put you in an unsupported scenario.

If anything is unclear, please feel free to let us know.
