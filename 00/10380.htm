You will not be able to set 7-7.9, 8-8.9 using Group functionality. But you should understand the way Excel groups. 7-8 : This has value for 8 associated with this group. Arial Unicode MS is a "last resort" font that applications such as Word will substitute if you add a character to the document that does not exist in the font you are using.
