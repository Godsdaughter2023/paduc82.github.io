Does anybody know how to remove the automatic default value of - Microsoft Community

elietouloupis say:

Does anybody know how to remove the automatic default value of $0.00 from a currency field in Access 2016?

Does anybody know how to remove the automatic default value of $0.00 from a currency field in Access 2016?
ken sheridan say: While it is very simple to delete the zero from the column's DefaultValue property, with a column of currency data type this is rarely appropriate for a column of Currency data type.  The column will have to allow Nulls, i.e. its Required property must be False (No).  Null is not a value, but the absence of a value, so is semantically ambiguous.  Consider a Null at a CreditRating column position in a row in a Customers table.  Does this mean that no credit is advanced to the customer?  Does it mean that the customer is allowed unlimited credit?  Or does it mean something else?  The answer is that it could mean any or of these; there is no way of knowing.

Null is also problematical because it propagates in arithmetical operations, so any arithmetical expression which includes a Null will evaluate to Null, regardless of other values in the expression.  Any such expressions consequently require the Nz function to be called to return a zero in place of a Null.  Note that the propagation of Null does not apply to aggregation operations such as SUM, which is a set operation rather than an arithmetical operation.

While there might be occasions when a Null is appropriate in a column of Currency data type, I would need to see a cogent reason for this before recommending it.

brian tillman [outlook mvp 2007-2019] say:

What it sounds like you want is a second alias of your mailbox. Your first address is called a primary alias and you can create other addresses that access that same mailbox so that no matter which of the two addresses someone uses when sending mail, you'll receive it in the same Inbox. You can, instead, have a rule that places the messages arriving for the second alias into a separate folder. When you reply to a message addressed to the second address, the sender of the reply will be the second address as well.

See this article: Add or remove an email alias in Outlook.com.

brian tillman [outlook mvp 2007-2019] say:

> I am struggling to understand what is required to set up the inbox rule

You should have been given the option to create the rule when you created the alias.  Anyway, the rule would look like this:

-----

After the message arrives and...

a recipient address contains the words "youralias"

Do the following...

Move the message to folder "youralias"

and stop processing more rules on this alias

-----

where "youralias" is the portion of the alias preceeding the @.

So, in the "Inbox and sweep rules" option, click the plus sign to create a new Inbox rule. Give it a name. In the "When the message arrives" box, choose "it include these words in the recipient's address", put the username portion of the alias in the "Specify words of phrases" dialog, click the plus sign, click OK, then in the "Do all of the following" dropdown, choose Move, copy, or delete>Move the message to folder, select the folder (or create a new one) when the "Select folder" dialog appears, click OK, then click OK at the top of the page.
