Sub ExportSheetsFromDatabaseBasedOnColumnIComissionReports()
 'Based on the value in the the second column
 Sheets("Combined Data").Select
 Dim myCell As Range
 Dim mySht As Worksheet
 Dim myName As String
 Dim myArea As Range
 Set myArea = Range("A1").CurrentRegion.Columns(9).Offset(1, 0).Cells
 Set myArea = myArea.Resize(myArea.Rows.Count - 1, 1).SpecialCells(xlCellTypeVisible)
 For Each myCell In myArea
 On Error GoTo NoSheet
 myName = Worksheets(myCell.Value).Name
 GoTo SheetExists:
NoSheet:
 Set mySht = Worksheets.Add(After:=Worksheets(11))
 mySht.Name = myCell.Value
 With myCell.CurrentRegion
     .AutoFilter Field:=9, Criteria1:=myCell.Value
     .SpecialCells(xlCellTypeVisible).Copy _
         mySht.Range("A1")
     mySht.Cells.EntireColumn.AutoFit
 End With
 Resume
SheetExists:
 Next myCell

End Sub

    Hi DT_IN, 

    Welcome to the Microsoft Community.

    I understand you would like add an existing Hotmail account as an alias for your Outlook.com account.  Let me assist you.

    It is not possible to add an existing Microsoft account as an alias to another Microsoft account.  To add an alias, it should be a non-existing Microsoft account or an existing third-party email account.

    However, there are workarounds.  If there are no important emails or contacts on your Hotmail account, you can just delete or close the account, wait for 30 days then add it as an alias on your Outlook.com account. 

    Another thing you can try is to rename the said account, delete it, wait for thirty days then add it as an alias to the Outlook.com account.  Before you rename the account, we strongly suggest you backup important emails for security.
