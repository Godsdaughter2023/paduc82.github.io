This could be caused by incorrect Power plan settings. You can run the Power troubleshooter to scan and fix issues with the Power settings. To do this follow the steps below:

a. user say:

Hi Kurt,

Safely removing the hardware is important to avoid losing data when copying to and from your USB drive. However, if you do not see the icon from the taskbar, it could be that the icon is hidden. Kindly select Show hidden icons, and see if the icon appears.

If the icon does not appear from the hidden icons, it could be that the option to show the icon is not enabled in Settings. Kindly go to Settings > Personalization > Taskbar, then select Select which icons appear on the task bar under Notification area. From there, scroll down to see the option Windows Explorer: Safely Remove Hardware and Eject Media and turn it on.

If the icon still does not appear, kindly make sure that the device has stopped all activity like copying or syncing files.

If you need further assistance, feel free to ask us.
