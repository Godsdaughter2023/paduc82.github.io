linking cell information to multiple pages of a workbook - Microsoft Community

paul w gilmore say:

linking cell information to multiple pages of a workbook

How do I have information entered into a cell of page one copy to the same cell location of all pages of a workbook?

bill manville say:

If it's static information, select all the sheets before entering the data.  Right-click a sheet tab and Select All Sheets. 
When done, remember to right-click a sheet tab and Ungroup Sheets

If you want to be able to change the information on sheet 1 at any time and have the change reflected on the other sheets then select all sheets except the first (select the second sheet and shift+click the last sheet tab) and, if you want A1 in each sheet to match A1 in the first sheet, enter =Sheet1!A1 in A1 of the active sheet.  You can copy this formula to other cells.
When done, select sheet1 to ungroup the sheets.
