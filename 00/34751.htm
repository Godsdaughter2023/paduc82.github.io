dougeatfresh say:

Pre-loading icons:

>Download nircmd.exe from Nirsoft and use it as follows:

>Create and save a batch file (as "all files" type rather than txt file).

Here's the structure of the batch file (rename the folders, of course):

@echo off
explorer C:\Users\MyUserName\FolderName\SubfolderOneName
nircmd.exe cmdwait 3000 win close class "CabinetWClass"
explorer C:\Users\MyUserName\FolderName\SubfolderTwoName
nircmd.exe cmdwait 2000 win close class "CabinetWClass"
explorer C:\Users\MyUserName\FolderName\SubfolderThreeName
nircmd.exe cmdwait 5000 win close class "CabinetWClass"
exit

>Include every folder you desire in this batch file.

The number in each line specifies milliseconds to wait while the icons load. Experiment with these numbers until delay is only as long as required for each folder: the more icons there are in a folder, the longer it will take to load them.

>Put the batch file in your root folder or in your User folder, create a shortcut to it, & put the shortcut in your Quicklaunch bar or on the desktop, or just run it at startup.

This works! I use it daily.
