maggieheller say:

User Account Control Doesn’t Allow to Enter Password/Click Yes in Windows 8/Windows 8.1

User Account Control always works perfectly well and allows the users to either provide the administrator password (in case of prompt for credentials) or click the Yes button (in case of prompt for consent) before the users can perform any administrative task on the computer. If your copy of Windows 8/Windows 8.1 doesn’t allow you to provide the password or click Yes when the UAC confirmation box is displayed, it means that there is some issue with the currently logged on user.

In this situation, you can create a new user but for that you must enable the built-in Administrator account by going to the external command prompt, and then enabling the account.

You can get the above task done by following the steps given below:

  1. Log on to Windows 8/Windows 8.1 using any account.
  2. From the Start screen, hover mouse to the bottom-right corner of the window.
  3. From the displayed Charms bar, click Search from the top.
  4. In the available field in the Search pane that comes up in the right, type CMD.
  5. From the available options, click Command Prompt.
  6. In the opened command window, type SHUTDOWN /R /O -T 00 and press CMD.
  7. From the available options, click Command Prompt.
  8. In the opened command window, type SHUTDOWN /R /O -T 00 and press Enter.
