stefan blom say:

Select an existing caption (label plus number) and click References | Insert Caption. In the dialog box, click the Numbering button, select "Include chapter number" and choose the "Heading 1" style. Click OK twice. Word reformats all captions of the same type.

Note that after you have replaced one type of caption with another like that, you usually have to update all of your cross-references. To trigger an update, click File | Print so that Word displays the Print Preview and then go back to Print Layout view by pressing ESC.
