Various things have been done to allow build upgrades to proceed:

  • get the current ISO for public Windows 10 
  • ensure that you completely uninstall any anti-virus software except Windows Defender
  • use msconfig.exe to disable all non-Microsoft Services
  • use Task Manager to disable all Startup programs
  • Restart after disabling above items

I would like to inform you that by choosing to view a site in compatibility mode while using Internet Explorer 11, does not make Internet Explorer 11 vulnerable.
