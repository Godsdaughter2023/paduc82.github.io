1. In explorer select Tools, Internet Options, browsing history, delete then check off temporary internet files, history and cookies. (temporary internet files should do it, but I do all three.

2. Start Task Manager by left clicking toolbar or ctrl-alt-del

3. select process "explorer.exe" and click End Process

4. select File, New Task (run...), type cmd.exe  and enter (the command processor window opens)

   then type the following commands:

cd /d %userprofile%\AppData\Local  (The directory will change)

attrib -h IconCache.db        (IconCache will be unhidden so it can be deleted)

del IconCache.db     (This deletes IconCache so that it can be rebuilt)

5. select File, New Task (run...), type explorer.exe and enter                 (this restarts explorer)

6. restart your system
