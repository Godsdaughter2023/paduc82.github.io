Do you intended to print to a file rather than to a printer?

If you want a paper copy, not a file, then click the Printer dropdown and, at the bottom of the list, click the "Print to File" item to turn off its check mark. Then, when you click the Print button, the document will go to the printer instead of a file.

If you really do want to print to a file (a fairly rare requirement, useful mainly when the physical printer isn't attached to your computer, and the file will be moved to a computer that does have that printer), then the Print to File item should be checked BUT you cannot use the name of the document you're printing. When it asks for a file name, you MUST give it a different name than the current document, or else select a different folder than the one that contains the current document.

If you add more than 5k items to a library, most operations won't work rendering the library unusable. This is because Sharepoint is an archaic legacy platform and when you do something in a library like use a filter, every item has to be touched. There are supposedly some workarounds but none that I've seen work well so it's best to stay below the 5k mark.
