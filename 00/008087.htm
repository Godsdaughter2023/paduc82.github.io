If you are trying to boot a Legacy BIOS device with media which has been setup for UEFI, it will not work.  If you take that same ISO file, but use Rufus to create a bootable USB stick for Legacy/MBR then it should work fine.  The resulting Windows partition is the same either way.  Only the BCD and System files used are different.
