How did you reopen the file?  If you opened it in Excel, you need to realize that CSV files do not retain formatting codes; so if your default format does not include seconds, you'll probably need to account for that.

In addition, if your Windows REgional settings default short date format is not also MDY, you may get really messed up.

As I wrote. So your problem is NOT that the seconds are left off.  It is that you are not SEEING them.

ronrosenfeld say:

If you must send the file as CSV,  The receiving party has to adjust the format.  As I wrote before, CSV documents do NOT include the necessary formatting codes.

There are any number of possible solutions, but either you have to send them a file that retains the formatting (eg an Excel workbook); or develop a method for them to add the necessary formatting at their end (education; vba macro, etc).

EDIT: Another possibility that occurs -- can you put the date and the time in separate cells, and save it to CSV that way?  In that case, it is likely that the time portion will be correctly formatted by the recipient  as h:mm:ss
