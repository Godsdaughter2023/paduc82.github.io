lead3 say:

In sleep, if the UPS had the capabiity to wake the PC, the it could run the shutdown software. It is the motherboard and BIOS setting that determine the power to the USB ports.  In hibernate, all power is off on the PC, if your power fails, it would lose nothing. You have already done a controlled shutdown.

You could use hybrid sleep which is a combination of sleep and hibernate. It takes a little longer to enter that mode since it stores the memory data to hard disk just like hybernate. This is a controlled shutdown as well.

If you don't lose power, it will return quickly just like sleep, if you lose power during hybrid sleep it will start back as if it were hibernated. Again no loss of anything.
