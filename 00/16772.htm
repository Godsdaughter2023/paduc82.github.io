john mcghie say:
Hi James:

No, Time Machine backups are fine.  They are not "deltas" as understood by the version control industry, as you say, they are exact duplicates: they are complete files.

Many version control applications either re-name the template to a different file name (ProjectWise, Documentum, I'm looking at YOU...) or they "Diff" the template to produce a "delta" (a partial file containing only the differences) (CVS, RCS etc).

Either completely prevents a Word template from functioning.

Time machine simply takes a checkpoint copy of the template at a specific scheduled time.  The Time Machine copy is a whole file, and can be substituted for the current version, because it is a complete binary structure.

Hope this helps
