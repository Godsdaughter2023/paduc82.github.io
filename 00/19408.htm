Try this ARRAY formula, see below on how to enter it. This does Col A, drag right for other columns

 

=MIN(IF(A1:A12000>PERCENTILE(IF(A1:A21000<>0,A1:A12000),0.9),A1:A12000))

 

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula

jay freedman say:

As the article says, although briefly:

Select some or all of the part of the document that isn't showing spelling errors.

On the Review tab, click the Language button and select Set Proofing Language.

If the check box for "Do not check spelling or grammar" has a check mark, click to turn it off, and click OK. If the box has a solid fill, that means that only part of the selected text has the "Do not check" turned on; if that's the case, click the box twice -- the first click puts in a check mark, and the second click turns it off.

jay freedman say:

Read this article about unchecking the "Do not check spelling and grammar" setting.

That setting is like bold or underline -- once it gets set (and I don't know how it would get set without your knowledge, but Word can do strange things sometimes), it continues to be set for everything you type after that point. Since it's saved as part of the document file, it won't disappear if you take the document to another computer, and uninstalling/reinstalling the program doesn't do anything to it.
