bob jones aka: cybertaz say:

I'm afraid I don't know what "stages" & "boxes" you're referring to, so please excuse me if this is redundant information. There is very little that need be done to set the Default Language in Office 2011;

  1. Launch Word
  2. Go to Tools> Language
  3. Select the preferred language [English (UK)]
  4. Confirm that Do not check spelling or grammar is not checked
  5. Click the Default... button
  6. Confirm Yes to updating your NORMAL template
  7. Click OK or Close to dismiss the Language dialog window

Keep in mind, though, that this establishes the default language only for new documents you create from that point on. It sets the Language specification of your Normal paragraph style as well as all other built-in styles according to what you select.

However, it does absolutely nothing to change the language(s) previously applied in existing documents. Language is an attribute of the text, comparable to Bold, Italic, etc. In any existing documents you must select the text, then apply the appropriate language to it. In general, it's best to include the Language specification in the Styles used in your documents.

For more comprehensive info, please see:  Spelling, Language, Dictionary

If you continue to have a problem it may be that your Normal.dotm template is damaged in some way. Please provide as much additional detail as possible, including exactly what steps you're taking as well as the specific results along the way.

net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

net start cryptSvc
net start bits
