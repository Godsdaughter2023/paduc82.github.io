https://docs.microsoft.com/en-us/skypeforbusiness/sfb-client-updates#lync-2013-client-updates

Then try to clear Skype for Business cache as following steps (I assume you are using Windows OS):

1. Exit Lync 2013 completely.
2. Open Windows (File) Explorer and navigate to the folder: %UserProfile%\AppData\Local\Microsoft\Office\15.0\Lync
3. Delete your sip folder and all the contents of Tracing Folder (Don’t delete the Tracing Folder itself). 
4. Restart the computer and Lync client.

3. If the issue happens to every Skype meetings you recorded?
