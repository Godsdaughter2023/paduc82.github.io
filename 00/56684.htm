    You posted in the forum for the client program Windows Live Mail. If this is what you're using to manage your email, then there is no 'preview' pane - there is a reading pane. Click the Services tab, check the "Hide All Microsoft Services" box and click Disable All (if it is not gray). Click the Startup tab, click Disable All and click OK. Restart the computer and then try the upgrade again.

    Note: Clean Boot is a troubleshooting step. If some programs have been disabled during the troubleshooting, we can re-enable them later. If a System Configuration Utility message appears, check the "Don't show this message" box and then click OK.

    If the error goes away, perform the troubleshooting steps at the following link: How to troubleshoot a problem using a clean boot in Windows Vista.

    To re-enable normal startup, start the System Configuration Utility again, and select the Normal startup option on the General tab.
