I have a nice vbs script which cleans the event-log;

 

(but I am sure this is not slowing down your pc... A fresh XP install is the only solution as I experienced computer speed issue...)

 

run it as "cscript file.vbs"

 

strComputer = "."
Set objWMIService = GetObject("winmgmts:" _
    & "{impersonationLevel=impersonate}!\\" _
    & strComputer & "\root\cimv2")
Set colLogFiles = objWMIService.ExecQuery _
    ("Select * from Win32_NTEventLogFile " _
    & "Where LogFileName='System'")
For Each objLogFile in colLogFiles
    Wscript.Echo objLogFile.NumberOfRecords
Next


strComputer = "."
Set objWMIService = GetObject("winmgmts:" & "{impersonationLevel=impersonate,(Backup)}!\\" & strComputer & "\root\cimv2")  
Set colLogFiles = objWMIService.ExecQuery _
    ("Select * from Win32_NTEventLogFile " _
    & "Where LogFileName='Application'")
For Each objLogfile in colLogFiles
    objLogFile.ClearEventLog()
    WScript.Echo "Cleared application event log file"
Next

Set colLogFiles = objWMIService.ExecQuery _
    ("Select * from Win32_NTEventLogFile " _
        & "Where LogFileName='Security'")
For Each objLogfile in colLogFiles
    objLogFile.ClearEventLog()
    WScript.Echo "Cleared Security event log file"
Next


Set colLogFiles = objWMIService.ExecQuery _
    ("Select * from Win32_NTEventLogFile " _
    & "Where LogFileName='System'")
For Each objLogfile in colLogFiles
    objLogFile.ClearEventLog()
    WScript.Echo "Cleared System event log file"
Next

harry stevens say:

Hi,

If you no longer have any other accounts on the computer, what you can do instead is to enable the in-built Administrator of your computer, then create a new user with admin privileges instead.

To do this:

1. You will need to download a bootable Windows 10 USB or DVD. You can download one here.

2. After downloading Windows 10 and making a bootable USB / DVD, restart the computer. It will start the Windows 10 Installation.

3. Click "Repair your Computer" on the bottom left side of the installation window.

4. You will be taken to the Windows Recovery Environment - select  "Troubleshoot".

5. Then click "Advanced Options".

6. From there, click "Command Prompt".

7. The Command Prompt window will open, type the command:

net user Administrator /active:Yes

8. Press Enter, and you'll be told that the command was successfully initiated. Enter the command "Exit" to restart the computer.

9. Remove the USB/DVD on your system and let it start normally.

If done correctly, you will see a new user "Administrator". This doesn't have any password, so you can easily access it.

You will be able to create a new user from there. 

Hope this helps.
