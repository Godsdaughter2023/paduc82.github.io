john visio mvp say:

Just a few comments on the code

1. Avoid hardcoded values. It is difficult for others to read. Instead of 1021 and 1022 use

If nScopeID = Visio.VisUICmds.visCmdUFEditCopy
If nScopeID = Visio.VisUICmds.visCmdUFEditPaste

2. Visio shapes are defined as Visio.Shape and they have collections of shapes defined as Visio.Shapes. So rather than

Private oShape(1 To 100) As Visio.shape

You should use

Private Oshapes As Visio.Shapes

So you will not be limited to 100 shapes.

3. When deleting, start with the last one first. In an array you will end up deleting only half the shapes. Delete the first shape, the second shape becomes the first shape. You then move to the next shape to shape(Count) which would be Shape(2) which was the old Shape(3).

 

The basic problem you are trying to resolve is that you need to determine what your new page is. How do you want the user to select the new page?

Hello

We understand how important this is for you to be resolved. As a workaround, please follow the troubleshooting steps below:

  1. Sign in to Skype for Windows 10.
  2. Choose any contact, in the chat box type the command /dumpmsnp and click Enter.
  3. Type /msnp24 and hit Enter.
  4. Restart Skype.

If the above information does not work, please do not hesitate to reply to this post.
