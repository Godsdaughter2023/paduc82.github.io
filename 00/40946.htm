rajesh govind say:

 

Hi Jason,

 

In Windows Operating System, the Check Disk utility is initiated at the boot time in 2 scenarios:

 

1)       When you attempt to perform a check disk in the last session and Windows could not perform it as the drive on which you were trying to run the check disk was not unlocked.

2)       If Windows itself suspects some bad sectors on the hard disk, then it initiates the Check Disk utility automatically without any previous attempts by the user. This usually happens when the shutdown process is not properly done.

 

If the Check Disk utility automatically initiated by the operating system while booting due to some bad sectors on the hard disk, then it would not display the name of each files that are checked by the Check Disk utility.

 

But, when you launch the Check Disk within Windows operating system using Command Prompt (Admin) using its appropriate switches, then it should display the details of the files that it checks.

 

To run the Check Disk utility on Windows 8, you may follow these steps:

 

a)       Press “Windows Logo” + “X” keys on the keyboard.

b)       Select “Command Prompt (Admin)” from that list.

c)       On the Command Prompt window, type the Check Disk command using its appropriate switches and then press Enter.

 

The Check Disk Command format is as given below:

 

chkdsk [drive letter]: [switches with space inbetween]

 

[drive letter] – Replace this with the drive letter on which you are trying to run the Check Disk utility.

 

[switches with space inbetween] – replace this with the different switches of the Check Disk utility.

 

 

On FAT or FAT32 volumes, “/V” is used with the Check Disk command to display the name of each file in every directory as the disk is checked. For example: The command chkdsk L: /f /r /v, will run this utility on the L: drive. The “/f” parameter tells CHKDSK to fix any errors it finds; “/r” tells it to locate the bad sectors on the drive and recover readable information; “/v” displays the name of each file in every directory as the disk is checked.

 

If it prompts you to restart the computer to lock the drive before running the check disk, then you may restart the computer and check if it displays the details as per the switch that you have used while running it at the start up.

 

Hope this is helpful.

===================================================
First...the total file size of 40 photos may to be way too much for a single e-mail.

Actually if you just upload the photos to a photo sharing site like Flickr then all
you have to do is send your recipients the link for viewing.

You can multiple select photos in Photo Gallery by holding your Ctrl Key while
clicking your selections. With the group selected...go to...E-mail on the toolbar.

If you prefer to create a New Folder...right click your desktop and from the menu...
choose...New / Folder...enter a name for the folder if you wish...

Now open Photo Gallery and in the file tree on the left of your screen scroll down
until you find the New Folder. Now you can drag/drop or copy/paste files into the
new folder. Then go back to your desktop to e-mail from the New Folder.
