THM movie file extension

Hello, I was editing a short video in my Windows MM program and it automatically saved it with a file extension THM. Now it won't play and the file conversion site I use will not upload it to change the extension. What do I do to 'fix it' so I can play it on my computer? 

considerate_guy say:

That's not a movie.  That's a thumbnail file.

WLMM doesn't "automatically save".  You have to explicitly save your project (File > Save) and also create a final movie (File > Save Movie).  If you do the latter (Save Movie) it will create a .wmv file that can be played on your computer.

greeves55 say: When I saved it I didn't click on "SAVE AS." I just clicked SAVE. So it got saved as a THM and now doesn't exist as a vid file. I don't have the original vid anymore so it looks like it's lost forever, right?

considerate_guy say:

Windows Live Movie Maker doesn't touch your original video files at all.  When WLMM works, it remembers a reference to the file on your computer.  In other words, it remembers the file name and all the entry/exit points, plus any splits, effects, transitions, etc. applied.

The video, photos and music are still on your hard disk as the ingredients of your movie.  But much like a recipe, they are only listed by name, not copied into Movie Maker..  You must keep your ingredients around if you want to make your movie, just like you need ingredients to make a cake.  You can't do much if you just have a project (aka recipe) and no ingredients.

So if you deleted the file, go to your recycle bin and retrieve it.  Hopefully you'll find it there.  Otherwise, you'll need to consider other options like file "undelete" utilities or perhaps recapturing it from the camcorder or digital camera (assuming you haven't cleared it from there).

Unfortunately I don't believe this will ever be possible. Windows RT is a special kind of Windows. Its been built to run on the special kind of processor that is in the Surface and other mobile devices (ARM processors). Windows Pro however, is built to run on standard computer processors (x86 and x64).

No. The Surface RT comes with Windows RT built in. There's no way to change its operating system to Windows 8.

The Surface RT uses a special chip designed for long battery life, and Windows RT takes advantage of that chip to provide give you many hours of computing time.

Windows 8, by contrast, requires a different chip.
