    Windows 7 and memory 32 bit vs 64 bit? - Microsoft Community

    jaiyp54 say:

    Windows 7 and memory 32 bit vs 64 bit?

    I am presently running 32bit Windows 7 with 4gb RAM installed. Using Photoshop I get memory use issues (out of memory...) Is there any advantage to increasing the RAM I have now with the 32 bit version or do I need to dump it and install 64 bit Windows 7 to fully use the higher memory? Thanks.

    ken blake - microsoft mvp say:
    On Sat, 20 Nov 2010 20:15:25 +0000, JaiyP54 wrote:
     
     
    > I am presently running 32bit Windows 7 with 4gb RAM installed. Using Photoshop I get memory use issues (out of memory...) Is there any advantage to increasing the RAM I have now with the 32 bit version
     
     
     
    ---------------------------------------------------------------------------------------------------------
     
     
    No. All 32-bit client versions of Windows (not just XP/Vista/7) have a
    4GB address space (64-bit versions can use much more). That's the
    theoretical upper limit beyond which you can not go.
     
    But you can't use the entire  address space. Even though you have a
    4GB address space, you can only use *around* 3.1GB of RAM. That's
    because some of that space is used by hardware and is not available to
    the operating system and applications. The amount you can
    use varies, depending on what hardware you have installed, but can
    range from as little as 2GB to as much as 3.5GB. It's usually around
    3.1GB.
     
    Note that the hardware is using the address *space*, not the actual
    RAM itself. If you have a greater amount of RAM, the rest of the RAM
    goes unused because there is no address space to map it to.
     
     
     
    ---------------------------------------------------------------------------------------------------------
     
     
    > or do I need to dump it and install 64 bit Windows 7 to fully use the higher memory? Thanks.
     
     
    ---------------------------------------------------------------------------------------------------------
     
     
    Yes. And running Photoshop, you may be one of those people who can
    make use of the additional memory.
     
    Ken Blake (MS MVP)

You can increase the size of Virtual Memory/Paging file in Advanced System settings / Performance. The normal size is between 1.5 to 3 times the amount of installed RAM.     You can open disk management and extend the partition.

    Press Windows Key + R

    Type: diskmgmt.msc

    Right click the System volume and click the ‘Extend Volume’ option on the contextual menu, click ‘Yes’ when the warning appears.

    A wizard will now begin that will guide through the steps to merge back the unallocated space with your system partition.

    The wizard provides a simple procedure to merge back the unallocated space with the system drive. Once you have selected the space, click Next, at the end of the wizard, you will see the amount allocated.
