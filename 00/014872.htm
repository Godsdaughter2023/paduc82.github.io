Need VBA help to open/create a folder based on student ID which is 9 digit text type. For example, when a button is clicked for student with ID 999887777, folder s:\maxwell\doc\999887777 opens. If it does not have that folder, create and open it. The s:\maxwell\doc\ is fixed. Thanks.
hansv mvp say:

The following assumes that you have a form with a control StudentID and a command button cmdOpenFolder:

 

Private Sub cmdOpenFolder_Click()
    Const strParent = "S:\Maxwell\Doc\"
    Dim strStudentID As String
    Dim strFolder As String
    Dim fso As Object
    ' Get student ID from control
    strStudentID = Me.StudentID
    ' Full path
    strFolder = strParent & strStudentID
    ' Create FileSystemObject
    Set fso = CreateObject("Scripting.FileSystemObject")
    ' Check whether folder exists
    If fso.FolderExists(strFolder) = False Then
        ' If not, create it
        fso.CreateFolder strFolder
    End If
    ' Open it
    Shell "explorer.exe " & strFolder, vbNormalFocus
End Sub

Did you check if nvstor.sys is effectively loaded?

You can check this with windbg using a kernel dump as target, or, you can use Process Explorer to watch the drivers loaded in System process.

nvstor.sys shouldn't be loaded, the disk IO driver stack for SATA devices, is completely handled by MSFT drivers, third parties can have drivers in the upper layer (filter drivers), but not in the lower layers. In the lower layer you should find storport.sys and disk.sys, but no nvstor.sys.
