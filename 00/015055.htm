Highlight rows on an Access report based on current month.

marshall barton say:

Use VBA code in the detail section's Format or Print event:
If Me.datestamp < DateSerial(Year(Date), Month(date), 1) Then
   Me.Section(0).BackColor = vbRed
Else
   Me.Section(0).BackColor = vbWhite
End If

To do the same thing using Conditional Formatting, create a text box that covers the entire detail section and set its CF - Expression Is option to:
   [datestamp] < DateSerial(Year(Date), Month(date), 1)
and the Back color property to red.

john p spencer say:

You can filter based on the date fields using

[DateField]< DateSerial(Year(Date()), Month(Date()),1)

You can use conditional formatting on the controls in the report using that expression.  You can apply the expression to every control.

Expression Is:  [NameOfDateField]< DateSerial(Year(Date()), Month(Date()),1)

Obviously if you filter the records to just the overdue records then there is no need to use conditional formatting to "highlight" them, since by definition, they are all overdue.
