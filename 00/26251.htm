gh541 say:

Sorting of numerals and alpha-numeric numbers in excel

Hi everyone,

I have a column in a spreadsheet with clothing size ranges however the sizes are out of order and not from smallest to largest. An example of 4 of the rows is as follows:

18|10|4|6|16|24|26|8|12|14|20|22

102|112|132|87|122|127|142|92|97|107|117|137

M|2XL|3XL|S|5XL|L|XL|XS

5XL|XS|L|XL|2XL|3XL|4XL|M|S

I just wondered if anyone knows if there is some formula that I could use that would sort the numbers or alphanumeric numbers from smallest to largest while still keeping each number separated by the pipe (|) symbol?

Would really appreciate any help.

Thanks in advance.

lars-Åke aspelin say:

You have to tell Excel the sorting order of all possible sizes.

I suggest that you do that in a helper column, column H in the formula propsal below but could be a column on another sheet if you want.

The helper column should contain all sizes that can occur in your spreadsheet, sorted from the smallest to the largest. One size per cell and no pipes. If alphanumeric and numeric sizes can never occur in the same cell, as you examples suggest, you can just put the alphnumeric sizes at the bottom of the list. If they can occur in the same cell, you have to put them in the proper place in the helper column.

Assuming your column with the unsorted, pipe separated, sizes are column A starting on row 1, try this formula in cell B1 (or any other column of your choice).

=MID(CONCAT(IF(ISERROR(FIND("|"&H$1:H$33&"|","|"&A1&"|")),"","|"&H$1:H$33)),2,999)

Note: This is an array formula that has to be confirmed with CTRL+SHIFT+ENTER rather than just ENTER.

H$1:H$33 is the range in the helper column. (I found 33 different sizes in your example, expand the range if you need to handle more sizes)

Hope this helps / Lars-Åke
