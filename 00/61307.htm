Login Information

MAC Filtering

  1. Open a web browser.

Are you sure that figure isn't off by a factor of 1,000?  

When I used the WLM Export feature, it moved everything, leaving nothing on my computer;

Export doesn't move anything. It creates copies of each message and adds a folder-name file to each folder. Something else must have removed messages from your computer. How did you get them back?

Robocopy is a robust file copy application built in to recent versions of Windows. It may serve your purpose.
