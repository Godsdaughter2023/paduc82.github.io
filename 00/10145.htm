Private Sub UserForm_Initialize()
  Dim C As MSForms.Control
  Dim i As Integer
 
  For i = 1 To 10
    Set C = Me.Controls.Add("Forms.Textbox.1")
    C.Left = Rnd * 200
    C.Top = Rnd * 200
    C.Value = C.Name
  Next
 
  Me.ScrollBars = fmScrollBarsBoth
  For Each C In Me.Controls
    If C.Top + C.Height > Me.ScrollHeight Then Me.ScrollHeight = C.Top + C.Height
    If C.Left + C.Width > Me.ScrollWidth Then Me.ScrollWidth = C.Left + C.Width
  Next
  Me.ScrollLeft = 0
  Me.ScrollTop = 0
End Sub

bernard liengme say:

1) There is no "use" for 9e+307; it is a number expressed in scientific notation.

2.5E+02 means the same as 2.5 × 10² or 2.5 × 100 or 250

 

9E+307 is a very large number! 9 with 307 trailing zeros.

The formula =LOOKUP(9E+307,B1:B10) finds the last entry in range B1:B10 because no cell in the range could hold a bigger number.

 

2) CSE or array functions are very common in Excel. If you do an internet search using <Excel CSE functions> you will find lots of material that will explain them to you
