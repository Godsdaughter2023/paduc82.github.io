t. valko say:

By the way, the formula

=CELL("filename")

will return the path+filename+sheetname. To get just the filename:

=MID(CELL("filename"),FIND("[",CELL("filename"))+1,FIND("]",CELL("filename"))-FIND("[",CELL("filename"))-1)


That formula will return the file name of the *last* open file to perform a calculation. So, if you have more than one file open you could get an incorrect result.

Just include a cell reference and it will take care of that:

=MID(CELL("filename",A1),FIND("[",CELL("filename",A1))+1,FIND("]",CELL("filename",A1))-FIND("[",CELL("filename",A1))-5)

Note that the file must already exist. That is, it must have been given a name and saved at least once.
