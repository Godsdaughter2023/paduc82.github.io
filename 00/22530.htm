Try

Dim myrange As Range, bmk As Bookmark

bernd p. say:

Don't rename notebooks with Onenote.

The new name is only a local display name and gets lost after closing that notebook. 

This local renaming feature is a design bug from the start on.

Real renaming (for notebooks on Onedrive) only can be done on Onedrive.

I don't know the actual state of the mobile clients respecting renaming, but normally they lose the connection to the renamed notebook because their path to it is now invalid.

The most reliable way to rename a notebook is

  1. close the notebook on all devices
  2. rename it on Onedrive
  3. re-open it on all devices
