cmdrkeene say:

There's 4 places to check for timeout settings (don't get me started...).  Probably one of these just needs to be edited:

  1. Right-click Desktop > Personalize > Screensaver
    Even if you don't use a screensaver, this timer value is used to lock the screen.  Set this to "none" andmake sure the checkbox is off so it doesn't require a password.


  2. Power Options > Change when to turn off the display

  3. Power Options > Change when the computer sleeps

  4. Settings App > Lock Screen Settings

It's probably number 1, because even if your screensaver is set to "none", the timer on that page still applies. So it will time out after that time and go to the lock screen, which then also goes to sleep.

So again, screensaver is the most likely cause, especially since you already checked the actual power settings.  And again, the screensaver option applies even if you never had or used a screensaver, because the timer value still matters.
