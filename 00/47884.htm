There is a PowerShell command that registers all the modern app packages in your system. But a minority of users reported that running that command broke existing (modern) apps which were previously working well.

If you'd like to give it a try, use these steps:

Get-AppxPackage -allusers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}
