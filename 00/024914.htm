andreas killer say:
Excel has no function for it, but you could do that with a macro.

Select a worksheet that is just behind a worksheet named with a month name and run the macro below.

If your sheet or month names look different, you'll need to adjust the code.

Andreas. 

Sub MonthSequenceInExcelTabs()
  Dim i As Integer
  If ActiveSheet.Index = 1 Then
    'The first sheet is January always
    ActiveSheet.Name = MonthName(1, False)
  Else
    'Check the previous sheet for all other sheets
    For i = 1 To 12
      If Sheets(ActiveSheet.Index - 1).Name = MonthName(i, False) Then
        ActiveSheet.Name = MonthName(i + 1, False)
        Exit For
      End If
    Next
  End If
End Sub

It would seem that the Section Break is too close to the TOC.

Ensure that there is at least one paragraph between the end of the TOC and the section break, so that when the TOC is refreshed the section break is not caught in the update.

If space is an issue then format the paragraph(s) as 1pt.
