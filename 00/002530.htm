Function DaysInWeek(rDates As Range, Index As Long, ParamArray RangesAndCriteria() As Variant) As Long
  'Index:  0 is this week; -1 last week; +1 next week
  Dim V As Variant
  Dim I As Long, J As Long, K As Long
  Dim dtStart As Date, dtEnd As Date
  Dim bRC As Boolean
dtStart = Date + 7 * Index + 1 - Weekday(Date)
dtEnd = Date + 7 * Index + 8 - Weekday(Date)

V = rDates
bRC = True
For I = 1 To UBound(V, 1)
    If V(I, 1) >= dtStart And _
        V(I, 1) < dtEnd Then
            For J = 0 To UBound(RangesAndCriteria) - 1 Step 2
                If RangesAndCriteria(J).Cells(I, 1) <> RangesAndCriteria(J + 1) Then bRC = False
            Next J
        If bRC = True Then K = K + 1
    End If
Next I
  DaysInWeek = K
End Function

They are based on the fact that:   TODAY()+1-Weekday(Today()) will --> Last Sunday, unless TODAY() is Sunday, in which case it will --> Today.

Last Week:  =COUNTIFS(ColD,">="&TODAY()-6-WEEKDAY(TODAY()),ColD,"<"&TODAY()+1-WEEKDAY(TODAY()))

This week:  =COUNTIFS(ColD,">="&TODAY()+1-WEEKDAY(TODAY()),ColD,"<"&TODAY()+8-WEEKDAY(TODAY()))

Next week:  =COUNTIFS(ColD,">="&TODAY()+8-WEEKDAY(TODAY()),ColD,"<"&TODAY()+15-WEEKDAY(TODAY()))

Replace ColD in the above formulas with the reference to your list of dates.

Since you asked for a VBA function, here is a UDF that takes the date range as the first argument, and an Index as the second.  If the Index = 0 , then it will count the dates in this week; -1 would be last week; -2 would be two weeks ago; 1 would be next week; and so forth:

It uses the same algorithm to compute the week interval, and does not require that the dates be in a single column.

===========================================

Function DaysInWeek(rDates As Range, Index As Long) As Long
  'Index:  0 is this week; -1 last week; +1 next week
  Dim V As Variant
  Dim I As Long, J As Long, K As Long
  Dim dtStart As Date, dtEnd As Date
dtStart = Date + 7 * Index + 1 - Weekday(Date)
dtEnd = Date + 7 * Index + 8 - Weekday(Date)

Debug.Print Format(dtStart, "ddd mmm dd, yyyy")
Debug.Print Format(dtEnd, "ddd mmm dd, yyyy")

V = rDates
For I = 1 To UBound(V, 1)
    For J = 1 To UBound(V, 2)
        If V(I, J) >= dtStart And _
            V(I, J) < dtEnd Then
            K = K + 1
        End If
    Next J
Next I
  DaysInWeek = K
End Function

cliffgu_msft5984 say:

Hi Simon,

If your DNS hosting provider is Office 365, you can’t edit the SPF record because Office 365 adds it automatically.

Please check if the domain you said is managed by Office 365. [You can check the domain by selecting the domain > Domain settings > DNS management.]

If you want to edit the SPF record to add additional values, please let your own DNS hosting provider outside Office 365 manage the DNS records. In this case, you need to change the nameservers in the domain registrar and make them point to your own hosting provider. Then, re-add the DNS records Office 365 needs in the DNS hosting provider outside Office 365.

Change how DNS records are managed with Office 365

Create DNS records at any DNS hosting provider for Office 365

How do I convert a microsoft word 2000 document into a microsoft word - Microsoft Community

crystal2you say:

How do I convert a microsoft word 2000 document into a microsoft word 2007 document from my flashdrive?

I have the microsoft word (millenium 2000 software) but I need to convert the word doc.
sharonmary say:

Hi Crystal2you,

You will be able to open Word documents created in earlier versions in Word 2007 but ... rule number 1: never do this directly on a flashdrive. Any time you open from or save to a flash drive you run the risk of corrupting the file. So, do this:

1. Copy the Word document from the flash drive to a folder (My Documents ...) on your computer

2. From Word 2007 open the file

3. Make any necessary changes

4. Click on Save and check the entry in the Save As type: box

    >>> Most likely it will be Word document (.docx), if not you can select that version from the list when you press on the little down-facing arrow.

Remember though!  If you want to put it back on to your flash drive, don't save it directly from Word, save it to your Documents folder and then copy from there to the flash drive.

Hope that helps?

All the best,

Sharon
