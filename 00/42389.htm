Let's analyze and identify the cause of the issue. Perform the steps provided here. This issue normally occurs when there is any issue with the wireless adapter that you are using.

Method 1: Verify the service

First of all, please verify if the WLAN service is started and set to automatic. If it is stopped, then I would suggest you to start the service and set it to Automatic.

  1. Click on Start, type in services.msc in the Start search and hit ENTER.
  2. Select WLAN Autoconfig service from the list, and double-click on it.
  3. Change the startup type to Automatic and click Start to start the service.
  4. Click Apply and then click OK.
  5. Restart the computer and check if the network is detected. If not, proceed to Method 2.

Method 2: Reset DNS and TCP/IP settings

  1. Click the Start button, click All programs, and click Accessories.
  2. Right-click Command Prompt, and click Run as administrator. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.
  3. Please enter the following commands, and press enter after each one.
    NETSH INT IP RESET
    IPCONFIG /FLUSHDNS
  4. Restart the computer to check the issue.

Check if the system is able to detect the network.

Looking around on the web, I have only found this pertinent detail about the Neomail address book: Neomail only has two fields in the addressbook, the nickname and the email address, on this page: http://blog.jaguarpc.com/web-hosting/transferring-address-book-from-neomail-to-horde-and-squirrelmail/

If that is the case, your CSV file should look something like

Nickname,Address

John Doe,john.doe @ domain.com

Jane Doe,jane.doe @ domain.com

...

or possibly

Nickname,Address

"John Doe",john.doe @ domain.com

"Jane Doe",jane.doe @ domain.com

...

 

[spaces before and after the '@' only to fool the forum software]

 

Open your file in Notepad and check that this is what they in fact look like, with a single comma in the middle of each line. If the character between the nickname and the address is a different one - a semi-colon, for example - use Find and replace to change them to commas.

You'll have to do any parsing of the Nickname into First name and Last name manually. It will be simple enough if there are only two names per contact. Just change the first line to
First name,Last name,Address
Remove any quotation marks and then change all spaces to commas.


If some contacts have more than two names, you'll have to add a field to accommodate the extra ones, for example Middle name. "John James Doe" would thus be changed to John,James,Doe and "Jane Doe" to Jane,,Doe
The first line would then read
First name,Middle name,Last name,Address

A CSV file is simply a text file, so if you have your exported list from Neomail in the format I illustrated, all you have to do is save it in Notepad with the extension .csv.
