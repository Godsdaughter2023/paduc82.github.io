what is the input mask to automatically change text to title case in - Microsoft Community

casi hopewell say:

what is the input mask to automatically change text to title case in access 2007?

I am wanting my forms to display the text in certain feilds as title case. I can see how to do lowercase and upper but no title! is it possible?
hansv mvp say:

No, you can't do that with an inout mask. You could use a calculated control with control source

 

=StrConv([FieldName],3)

 

where FieldName is the name of the field that you want to display. Please note that a calculated control cannot be edited, its value is read-only.

 

Or you could convert the text entered by the user to proper case. Let's say you have a text box txtAddress:

 

Private Sub txtAddress_AfterUpdate()
    Me.txtAddress = StrConv(Me.txtAddress, vbProperCase)
End Sub

 

StrConv with the vbProperCase argument is not perfect - it'll change McDonald to Mcdonald and O'Leary to O'leary.

    andreas killer say:
    The reason is that Range("A1"):CurrentRegion returns more cells as you can count with a Long data type. Therefor you must use Variant instead.

    But the RTE is not related to the variable, but is caused by the Count method, which has a Long as return type. Therefor since XL2007 you have to use the CountLarge method.

    The next issue is that CountLarge returns a datatype that you can't use with FOR loops and you get RTE 458... see code below.

    BTW, just to be sure: You're sure you want to sort every cell in a rectangular area? With a bubble sort?

    Andreas. 

    Sub Workhistorydata()
  Dim i As Variant, j As Variant, temp As Variant, rng As Range
  Set rng = Range("A1").CurrentRegion
  i = 1
  Do While i <= rng.CountLarge
    j = i + 1
    Do While j <= rng.CountLarge
      If rng.Cells(j) < rng.Cells(i) Then
        'swap numbers
        temp = rng.Cells(i)
        rng.Cells(i) = rng.Cells(j)
        rng.Cells(j) = temp
      End If
      j = j + 1
    Loop
    i = i + 1
  Loop
End Sub
