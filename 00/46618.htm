don cuthbert say:

The Windows 10 free upgrade does not activate with any Product Key.  It activates only by upgrade from an activated qualifying OS (Windows 7 or 8).  There are no unique product keys for Windows 10 free upgrade, so there is no way to activate it (for free) once it has been already installed.

That leaves the only way to activate Windows 10 for free is to reinstall Windows 7 or 8.  You can then upgrade by using the Windows 10 Media Creation Tool.  Directions are here:

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install#

So if Windows 10 is not activated, you have to install and activate Windows 7 again, using the Windows 7 product key.  But you can use the Media Creation Tool to upgrade right away, instead of doing all of the Windows 7 upgrades and waiting for the Windows 10 to download.

Once you have upgraded and activated Windows 10 you can do a clean install any time in future on that same device, without needing Windows 7 again.

Hi Sai,

We appreciate you for being part of Windows 10.

The error code might show up when there is an issue with the network connection. This might also occur due to issues with the Microsoft account settings apart from Store app settings.

Please assist us with the following information related to the issue:

  1. Have you enabled any proxy setting on the computer?
  2. Are you able to access other apps from the Start screen and connect to internet using the      internet browser apart from Store app?
  3. Is there any third party security software installed on the computer?
  4. Are you able to install the updated offered from Windows Update?

If you aren’t able to access other apps or if you have enabled a proxy setting, this is how you can remove proxy setting and enable automatic settings:

  • Open Internet Explorer on the desktop.
  • Click on Tools icon option on the top right corner.
  • Select Internet Option from the list.
  • Click on Connection tab.
  • Click on LAN Settings.
  • Uncheck the box which says“Use a Proxy Server for your LAN”. Check the box which says “Automatically      detect settings” option.
  • Click on OK.

For further troubleshooting, I would suggest you to check if running the app troubleshooter fixes the issue. Follow the methods:

Method 1:

Run the Windows store apps troubleshooter and check if it helps.

  1. Press Windows + X keys on the keyboard and select Control Panel
  2. Type Troubleshooting in the search bar on the Control Panel.
  3. Click on View all on the left navigation pane.
  4. Click on Windows store apps to run the Windows store apps troubleshooter.
  5. Click on Advanced on the Windows store apps troubleshooter dialog box and click on Run as      administrator (Enter the credentials if required).
  6. Click on Next and follow onscreen instruction complete the Windows store apps troubleshooter.

Check if the issue still persists. If yes, check with the next method.

Method 2:

Let’s reset Store app by removing the temporary files to troubleshoot the issue further. Follow the steps to reset the Store:

  • On the desktop screen, press Windows + R key to open run box.
  • Type WSReset.exe and hit Enter.

I hope the above information helps. Kindly let us know if you need any further assistance with Windows. We are glad to assist you.

Thank you.
