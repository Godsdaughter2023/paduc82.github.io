shenan stanley say:

Update the drivers.  Do not depend on Microsoft for ANY of your non-Microsoft hardware.

[Some further explanation of what I mean.]

I realize before that you stated, "has all the latest drivers and windows 7 professional updates" - but to different people that means different things.  Some people are satisfied that Windows has checked for updates, found some updates including hardware drivers, and they have installed those and it has found no more - done.  Others people are satisfied when they visit the manufacturer's web site for the system (in your case HP) and utilized the support page for the model of their system (you'd need to be more specific than "Pavilion DV7" with the model) to download the hardware device drivers and install them from there.  Other people (category I fall in) want the latest/greatest from the actual hardware manufacturers whenever possible: motherboard chipset, video, network and audio chipset drivers are those that are usually more easily obtained this way - which are the heart and soul of your computer in many ways.  These people seek out the actual manufacturer drivers for each individual component.  This is what I am about to suggest:

Visit ATI (AMD's) web page and download/install this driver & software:
http://support.amd.com/us/gpudownload/windows/Pages/radeonmob_win7-64.aspx
(11.7 released on 7/27/2011.)

Visit the Intel web page and use their driver scanner utility to scan for/locate and download (then you install) the latest drivers for your chipset, built-in video and etc.:
http://www.intel.com/p/en_US/support/detect
(Check your system for the latest updates, install the "Husdawg, LLC" stuff, etc., expand the lists, etc.)

Being a laptop - and specifically one that *will* run hot - make sure you do not set it on pillows/soft tops and keep it on a flat/hard surface during use.  You may even need/want to purchase some sort of laptop cooler base for it to sit on when you want to utilize it in more 'relaxing' spots.

What you are doing is eliminating the software issues and the obvious hardware possibility; if the drivers and the heat dissipation doesn't help - the symptoms point to existing hardware issues, not software ones.  Defective (probably from heat damage) video display device is the most likely suspect.  That will lead you to here: http://www8.hp.com/nz/en/support-drivers.html

Hope that helps, somewhat!

This issue might occur if welcome screen language settings are change.

anannya podder say:
Hi,
Are you able to boot to Windows Recovery Environment?
If yes, perform a (check disk) chkdsk from WINRE (Windows Recovery Environment) and then try to perform a clean install to see if it works:
Reference:
http://windows.microsoft.com/en-US/windows7/What-are-the-system-recovery-options-in-Windows-7
You can follow these steps to run CHKDSK:
a. Use the Windows 7 DVD to start the computer in Windows Recovery (WinRE).
b. In WinRE, open a command prompt. To do this, follow these steps:
c. On the Install Windows screen, select the appropriate Language to install, Time and currency format, and Keyboard or input method options, and then click Next.
d. Click Repair your computer.
e. Click the Windows 7 installation that you want to repair, and then click Next.
f. Click Command Prompt.
At the command prompt, type the following command, and then press ENTER:
chkdsk drive: /p
chkdsk drive: /r
For reference: http://windows.microsoft.com/en-US/windows7/Check-a-drive-for-errors
Chkdsk disclaimer:
Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.
Note:
If the above does not help then probably it’s a corrupt DVD issue, I would suggest you to contact the manufacturer for assistance.
