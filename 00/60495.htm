¡firedog say:

Is there a way to successfully copy these across to the new laptop ... ?

Using the official process: Export and import messages

An alternative, which might be preferable if there are not too many messages to be transferred:

  1. Create a temporary folder on your desktop to contain the messages to be transferred.
  2. In this folder, create folders corresponding to each folder whose contents you want to transfer.
  3. Rearrange the windows on your desktop so you can see both Windows Live Mail and the desktop folder (tip: click in a window and press Windows key  + left arrow key to pin it to the left-hand half of the screen, etc.).
  4. Select messages1 in a Windows Live Mail message list, then drag them across and drop them on the corresponding folder in the desktop folder.
  5. Copy the desktop folder to the new machine's desktop, e.g. over the network or using a USB drive.
  6. Reverse the process to drag messages from a folder in the desktop folder into the message list of the corresponding folder (create new ones if necessary) in Windows Live Mail.

One caveat: dragging out of Windows Live Mail into a file system folder will generate a new file for each message. The file will have the subject of the message as its file name and .eml as its extension. Files with the same subject will have a serial number appended to the file name.

If the subject of a message ends with a period ( . ), the file created will have the extension ._eml to avoid having two periods together. Check carefully before copying the folder to the new machine that there are no messages like this; if there are, right-click on the file name, select Rename and remove the underscore ( _ )character. If you don't do this, drag and drop will fail ...
