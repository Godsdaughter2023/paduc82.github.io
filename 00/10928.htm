doug robbins - mvp office apps & services (word) say:

If there is no reason for the Different First Page setting, you should uncheck the box for Different First Page in Sections two and four and also uncheck the Same as Previous.  Before unchecking the Different First Page box, you may want to copy the content from the First page Header and\or Footer and then Paste it into the Primary Header as the First Page Header and Footer will disappear when you uncheck that box.

If you have a reason for the Different First Page setting, then you need to replicate the page numbering fields in the Primary Header\Footer of the second and fourth sections of the document.  Use the Next and Previous buttons in the Navigation section of the Header & Footer>Tools tab of the ribbon to navigate through the Headers and Footers.

mike h.. say:

Hi,

Because you're using worksheet-calculate it will pop up no matter which cell is change, what you need is worksheet_change and to look at the range being summed by the 2 formula in A10 and E10 like the code below. Now the message will only pop up if a cell in the range A11:A1000 is changed and A10 is >3, like wise for Col E

I also included a marker cell(s) Y10 & Z10 and these cells are change to 'Advised' if the message has been popped up, preventing it popping up a second time. Manually deleting the content of these 2 cells resets things. You can use any 2 cells for this.

I don't understand the £ sign in N13 & N16

Private Sub Worksheet_Change(ByVal Target As Range)
If Not Intersect(Target, Range("A11:A1000")) Is Nothing And _
Range("Y10") <> "Advised" Then
If Range("A10").Value > 3 Then MsgBox "Employee has reached the tardy threshold. Contact HR Coordinator."
If Range("A10").Value > 3 Then Range("Y10") = "Advised"
End If
If Not Intersect(Target, Range("E11:E100")) Is Nothing And _
Range("Z10") <> "Advised" Then
If Range("E10").Value > 2.667 Then MsgBox "Employee has reached the 64-hour threshold for sick leave.Contact HR Coordinator. If employee provides medical verification, enter hours over 64, using calculator on the right, in a new row under the Pre-Approved and Authorized Absences section."
If Range("E10").Value > 2.667 Then Range("Z10") = "Advised"
End If
End Sub

ashish mathur say:

Hi,

Try this

1. Click on cell K16 and go to Home > Conditional formatting > New Rule > Use a formula to determine which cells to format

2. In the Formula bar there, enter this formula

=OR(ISNUMBER(SEARCH("elect",K16),ISNUMBER(SEARCH("util",K16)),ISNUMBER(SEARCH("prop",K16)))

3. Click on Format and choose your desired formatting

4. Click on OK/Apply

You may now copy this formatting to other cells.

Hope this helps.
