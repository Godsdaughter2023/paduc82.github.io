jay freedman say:

The error in the template code is easy to fix (most of the following is just how to find the right place!):

  • Locate the downloaded template file -- it should be in a hidden folder at %appdata%\Microsoft\Templates\Lesson plan calendar.dotm
  • Right-click the template file and choose Open from the menu.
  • Cancel the two dialog boxes that appear when the template starts up.
  • Press Alt+F11 to open the macro editor.
  • In the Project pane in the top left of the editor, expand the TemplateProject )Lesson plan calendar) icon, and under that also expand the Forms icon. Then right-click the frmSelectMonthAndYear icon and choose View Code from the menu.
  • Scroll down a bit to see the line Private Sub cmdOk_Click(). Below that line, there are some lines that start with the word Dim. The Microsoft programmer omitted the one that defines the variable named j. Insert a new line (Enter key) anywhere in that group of lines and type in Dim j As Integer
  • Save and close the template, and close the macro editor.

The template will now work as expected.

Inserting Excel Spreadsheet - Microsoft Community

tl2012 say:

Inserting Excel Spreadsheet


Hi,

I'm using Word for Mac 2011. I want to insert an excel spreadsheet. I've seen references to Insert Excel Spreadsheet on the Toolbar.
This is not a menu option that I have.
bob jones aka: cybertaz say:
There's most likely something missing from the references you've seen or your recollection of them rather than from the program :-)

The actual menu path is: Insert> Object in the menu bar -- From there you can either insert a new Excel Worksheet Object based on several options or create one from an existing Excel file. Although some people may have added a toolbar button there isn't one by default in Word 2011. The closest is a button in the Data group of the Charts tab in the Ribbon, but it's only active & useful when you select an existing Chart for which you want to edit the data it plots.


Personally, I'd avoid using the feature altogether. For one thing, no matter which you use, the objects are limited to 5 Columns by 10 Rows. Instead, I'd suggest that you actually create the sheet in Excel. Copy it & use Edit> Paste Special to paste it into the Word document. Try some of the Paste Special options to wee which suits you best.

Either way, you typically get a more presentable result & aren't restricted to a 5x10 matrix.

If the administrator of office365 education removes my account, can I - Microsoft Community

redapple0204 say:

If the administrator of office365 education removes my account, can I retrieve the onedrive file?

Hello everyone
I applied for a student version of office365's account at my school mailbox. If I graduated, my administrator deleted my account number. How can I retrieve my files on onedrive?
Thank you for your help.
tim liu msft say:

Hi Redapple0204,

If the account is deleted, you will lost the access to the cloud OneDrive for Business library. You can only check if those files are still in the synced folder (the premise is that you use OneDrive for Business sync client to sync them). Copy them to another local folder and store them to your personal cloud storage. If you already cannot find them in your sync folder, files may be removed by your tenant admin or school IT (they can take owner of your OneDrive for Business library) or you already don't have permission to sync them. In this situation, you cannot retrieve those files from your side. You need to contact your IT or admin to check if they can help you find those files.

So, if you have work/school account and use the OneDrive for Business service in it, I will suggest you copy those files in it to your local drive or other cloud storage before leaving school.

Best regards,

Tim
