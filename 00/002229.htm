Unfortunately, the new UUP system for providing the new build is quite slow, and the progress indications are quite misleading, not correctly indicating what's going on ... the Downloading updates 100% is, in most cases, left over from the prior download of Windows Defender definitions update.

What's really going on is the determination of what files exist in your current build, so that they won't need to be downloaded again, which will result in a smaller download. When that finishes, eventually, the 100% indication will change to 0%, the actual build download will finally begin, but on some machines, it doesn't progress to beginning the download until the machine is restarted, so if you haven't yet, it wouldn't hurt to restart, and check for updates again.

macropod say:

There's a few tricks you can employ to speed up processing:

1. disabling screen updating;

2. for long tables, especially, turning off autofit.

Both can be restored after processing; and

3. Giving Word some breathing room for housekeeping every now and then.

The revised code below does all three and gives a progress report on the status bar.

Sub AutoTableShader()
Application.ScreenUpdating = False
Dim i As Long, j As Long, bAutofit As Boolean
With ActiveDocument
  On Error Resume Next
  For i = 1 To .Tables.Count
    With .Tables(i)
      bAutofit = .AllowAutoFit
      .AllowAutoFit = False
      For j = 1 To .Rows.Count
        Application.StatusBar = "Processing Table " & i & ", row " & j
        With .Rows(j).Shading
          If .BackgroundPatternColor = RGB(128, 128, 128) Then
            .BackgroundPatternColor = RGB(211, 211, 211)
          End If
        End With
        If j Mod 100 = 0 Then DoEvents
      Next
      .AllowAutoFit = bAutofit
    End With
    DoEvents
  Next
End With
Application.StatusBar = ""
Application.ScreenUpdating = True
End Sub

Create a new sheet in your file and name it "Show Tabs".

(You can name it as you like, but the word "Show" must be in front)

Write the names of the sheets to be hidden/shown into column A (first cell A1)

Copy the code below into the code module of that sheet.

Andreas.

Option Explicit
Option Compare Text

Private Sub Worksheet_Activate()
  Dim R As Range
  Dim S As String
  S = Me.Name
  On Error Resume Next
  For Each R In Range("A1").CurrentRegion
    Sheets(R.Value).Visible = Left(S, 4) = "Show"
  Next
  Mid(S, 1, 4) = IIf(Left(S, 4) = "Show", "Hide", "Show")
  Me.Name = S
End Sub

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
  Worksheet_Activate
End Sub

Yes, take that exact same disc that you burned the .iso to and use it to do another install of Windows -- when you do that, you can choose to have it reset your computer, refresh your computer, do a clean install, etc.  Is that what you were asking?
