Hi rickellen,

  • Is this the only game which has been disappeared from the games folder?
  • Did you remember making any changes due to which it might have happened?

As you have mentioned that you do not have the Windows XP disc, I will suggest you to follow the steps given below which will not require the disc. However, if the issue is not fixed, you might have to arrange for a Windows XP disc in order to get the game back.

a)      Click Start and then click Run.

b)      Type “Spider” in the run box and hit Enter.

c)       If the game opens, you probably just need to create a new shortcut of the game which was deleted somehow.

d)      Navigate to the location c:\windows\system32\spider.exe.

e)      Copy “spider.exe” from the above location.

f)       Now, navigate to the location c:\Documents and Settings\Allusers\programs\games.

g)      Paste the “spider.exe” file copied earlier.

h)      Rename the file to “Spider Solitaire”.

i)        Close all the windows and check if the game appears in the games list.

If incase the spider.exe file does not exist on the machine, you will have to arrange for the Windows XP disc to get the game back. Once you have the disc, you may follow the given article which will help you restore all the games back the way they were.
