It doesn't matter how you have partitioned your HD on another pc, if the BIOS of your Dell can't handle the 320Gb size of your new HD then it just won't see the HD at all.

How I got the update to install...

To install the update I did what it says below. I just added a '2' character to the name of the "$WINDOWS. ~ BT" folder.  The update creates a new "$WINDOWS. ~ BT" anyway, so you can't go back and change the name of the folder back to how it was after the update.

This is a major update, so don't forget to backup first.  Some programs change too, so look into exactly what the update changes if you are concerned.  E.G. Sticky Notes updated to a worse version - in my opinion.  Good luck.

Credit to MrFraztastic in this post. 

  • Go to your C: drive in the file explorer
  • Click View and check the box that says "Hidden Files"
  • There will be a folder named "$WINDOWS. ~ BT", rename that folder to anything you'd like.
  • Attempt the update again.
