Can send to limited inter-office addressees, only.

marcosaraujo2 say:
Hello TCVee

This error means that Office 365 has added the user to the blocked senders' list because this person's emails have been recognized as spam. This might happen, for example, if the user has sent an email to a large number of recipients, exceeding the limits of the Office 365 plan.

To solve this problem, the admin of the Office 365 tenant needs to:

Make sure that the user account has not been hacked or compromised in any way.

If the tenant admin is certain that the account is safe, they need to follow these steps to remove the account from the block list and resume its mail flow:

Open Exchange admin center.

Select protection from the left menu and go to the action center

Find and select the blocked user from the list and click the Unblock Account link in the pane on the right.

Click Yes to confirm and wait for the changes to propagate. The propagation usually takes around 2 hours, but if the user sends any emails during this time, it might take longer. After the process is finished, the user should be able to send emails.

Office 365 can notify you when your users are blocked. It can also forward all suspicious emails to your mailbox. You can set these notifications on the outbound spam.
