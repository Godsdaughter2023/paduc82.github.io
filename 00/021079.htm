Replacing an embedded Hyperlink in an Excel Spreadsheet - Microsoft Community

john f. say:

Replacing an embedded Hyperlink in an Excel Spreadsheet

I have a Excel spreadsheet of 2415 rows - each row being of a different named sport person.

In each of these rows, I have inserted an hyperlink into one cell (hence, having 2415 hyperlinks), so that clicking on each of these hyperlinks, will take

you to a different set of on-line results, for each of the 2415 players.

However, the on-line Server has now been replaced, so that the results have now been saved to a different Server, which means that I will have to change part of the embedded URL, so that -

"http://www.evc2011.com........." is to be replaced by "http://www.pingpong.cz/6/eurochamp/mev2011.........."

Hence, my question is - Does anybody know of an Excel macro, or a Javascript routine, which I could run, to replace these URLs.

 

 

hansv mvp say:

Here is a macro you can run:

 

Sub ReplaceHyperlinks2()
    Const strOld = "..." ' old server address
    Const strNew = "..." ' new server address
    Dim hyp As Hyperlink
    For Each hyp In ActiveSheet.Hyperlinks
        With hyp
            .Address = Replace(.Address, strOld, strNew)
            .TextToDisplay = Replace(.TextToDisplay, strOld, strNew)
        End With
        hyp.Range.Replace strOld, strNew, xlPart
    Next hyp
End Sub

 

Substitute the correct addresses in the constants at the beginning of the code.
