aradmin say:

I was able to find a work-around in regards to error code 800488EB.

The symptom I observed was in regards to logging into Windows Family Safety client. The issue appears to actually have something to do with the 'Windows Live ID Sign-in Assistant' service.

 

Stopping both the 'Windows Live ID Sign-in Assistant' and 'Windows Live Family Safety Service' in the Services Mangment Console allowed my Windows Family Safety client to finally log-in so I could adust settings again.

 

Here are the steps to stop the two services (in Windows 7):

1. Close any open windows/programs so that only the Windows desktop is displayed.

2. Click the Windows 'START ' button

3. In the 'Search programs and files' text box, type: services.msc

    then press ENTER

4. After the 'Services' Window displays, on the right pane, locate the service named:

    Windows Live ID Sign-in Assistant

5. RIGHT-CLICK on the 'Windows Live ID Sign-in Assistant' service

6. On the popup-menu that displays, click STOP

7. A progress bar will display while the service stops.

8. After the progress bar dissapears, repeat Steps 1-7 for the service named: 'Windows Live Family Safety Service'

9. Close the 'Services' Window.

10. Run the Windows Family Safety program again and attempt to login.

 

The login time on my workstation for Window Family Safety took longer than usual (approximately 1-3 minutes) but it did eventually log-in completely without the 800488EB error code. I was able to adjust user settings again.

 

I suspect that the Windows Family Safety client restarts both stopped services when you attempt to login durring step 11...only this time it does so w/o whatever issue causes error code 800488EB.

Original credit for this work-around goes to: Steph_The _Gamer in the Microsoft Answers thread: http://answers.microsoft.com/en-us/windowslive/forum/messenger-program/windows-live-messenger-error-code-800488eb/e7ac1f72-49a5-414c-9f8b-483b589536e5

 

That particular thread actually provides several possible solutions to this related issue. The steps above worked in my case.
