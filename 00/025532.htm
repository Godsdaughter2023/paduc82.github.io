Question: How can I get my mobile voice only number verified to use as my Skype Caller ID?

Answer: Skype only offers two options to users to set up their caller identification. You can only use a Skype Number or a mobile number.

If you choose to use a Skype Number, the system will not send a verification code as this is a pre-saved number on the system and caller ID will be activated immediately. How do I assign a shortcut key to a phrase that I use often in typing - Microsoft Community

kisa1955 say:

How do I assign a shortcut key to a phrase that I use often in typing documents?

In Microsoft Word Perfect, I was able to assign a shortcut key (Ex. F1) for phrases that I type often in documents.  How can I do this in Microsoft Word?
charles kenyon say:
It depends on the length and type of phrase. It also depends on which functions are available in the Starter edition.

The two methods that come to mind are AutoCorrect and AutoText.
AutoCorrect automatically replaces a code word with the text, for instance, "twimc" with "To whom it may concern:" It does this when you press the space bar or Enter, or punctuation.

AutoText is used primarily for longer items and is stored as a "Building Block" in a template. You can assign a keyboard shortcut to that, but I don't think the ability to do this is enabled in the Starter edition.

See Automated Boilerplate Using Microsoft Word

Macros are not enabled in the Starter edition, but even if they were, you would not want to use them for this purpose. Macros are for manipulating documents, or parts of them, not for typing text in Word.

You would not want to reassign the function keys (or at least I would not want to) because they all have useful assignments in Word already. However Alt- Ctrl- Ctrl-Alt Shift-Alt- and Ctrl-Shift-Alt- combinations with letters are all available. You would probably want to avoid both the simple Alt- and Ctrl- combinations because many of them also have defined uses. In particular, the Alt key triggers a keyboard step-through of the Ribbon and menus.

If AutoCorrect is available it would be available through the Proofing Options under the File menu.

chip pearson say: In Excel, press ALT F11 to open the VBA editor. There, open the Project window with CTRL R if it is not already open (typically on the left side of the screen). In that window, find your workbook, expand the tree nodes, and double-click on the ThisWorkbook code module, In that module, paste the following code:

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
Dim WS As Worksheet
Dim R As Range
Dim S As String
Dim N As Long

If SaveAsUI = False Then
    Exit Sub
End If
Application.EnableEvents = False
On Error GoTo EndSub:
Set WS = ThisWorkbook.Worksheets("Front Sheet")
With WS
    Set R = .Cells(.Rows.Count, "A").End(xlUp)
End With
N = CLng(R.Value) + 1
N = Application.InputBox(prompt:="Enter version number", Default:=CStr(N), Type:=1)
If N <= 0 Then
    Application.EnableEvents = True
    MsgBox "You didn't enter a valid version number."
    Cancel = True
    Exit Sub
End If
S = Application.InputBox(prompt:="Enter a comment", Type:=2)
If S = vbNullString Then
    Application.EnableEvents = True
    Cancel = True
    MsgBox "You didn't enter a comment."
    Exit Sub
End If
Set R = R(2, 1)
R.Value = N
R(1, 2).Value = S

EndSub:
Application.EnableEvents = True
End Sub

Close the VBA editor and save (not save as) the file. Now, when the user tries to do a Save As via the dialog, he will be prompted for the version number and a comment.

chip pearson say: Shah,

In order to use any VBA in a workbook, you must use the XLSM format, the XLSB format, or the old XLS format. The XLSX format is specifically designed to prevent any VBA code in the workbook. 

You also need to enable macros. By default, macros are disabled. To do this, first you must make the Developer tab of the Ribbon visible if it is not already visible. Go to the File menu and choose Options. Then, go to the Customize Ribbon page. There, on the right-hand list, put a check in the box for the Developer tab. Click OK to return to Excel. Then, go to the Developer tab and click the Macro Security button on the Code panel. Under Macro Settings, choose "Enable All Macros".  Click OK to get back to Excel. Save (not Save As) the file. Now, when you attempt to do a Save As from the dialog, the code should run and should prompt for a version number and comment. 

As far as I know, there is no way to enable macros for one workbook while disabling them for other workbooks. Moreover, I don't think you can have Excel prompt the user when the workbook is opened whether to enable macros. Such functionality was removed in 2007 (I think).
