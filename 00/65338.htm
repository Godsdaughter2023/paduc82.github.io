Login Information:

To log into your routers configuration page open up a web browser on your computer and in the address bar type in the gateway address below. You won’t be prompted for a username and password because it is not required.

  • Gateway - 192.168.1.1 or http://vizio.home
  • Username - N/A (None)
  • Password - N/A (None)

Restore Factory Defaults

Via Firmware

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open web browser.
  3. In the address bar at the top, type in 192.168.1.1.
  4. Click on the icon of the router at the top. (The symbol in the middle of the three icons)
  5. Click Advanced next to Firmware.
  6. Click on Restore Defaults.

Via Reset Button

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Locate the reset button on the device.
  3. With paperclip/pen/toothpick hold down the reset button down for 30 seconds.
  4. Wait for lights to turn off then return to normal.

Found this answer on the Focusrite site. 

Why is my audio slowed down in Windows 8 when the computer starts?

We are aware of an issue when using Windows 8 that causes audio to be slowed down when the computer starts up, when using a Scarlett usb interface. This issue will only affect certain systems and there are workarounds to prevent this from happening.

The first workaround is disconnect and then reconnect the usb cable to the Scarlett once the computer has started. The second workaround is to change the sample rate that the interface is running at either in Windows sound preferences or in a piece of dawsoftware. The third workaround, which removes the need to do either of the previous ones, is to disable Windows Fast Boot.

To do disable fast boot please go to Control Panel > Hardware and Sound > Power Options > Click on "Choose what the power button does" and in the next window untick "Turn on fast start-up". The next time the system starts you should find that the issue does not occur. 

Not ideal but faster than rebooting every time.
