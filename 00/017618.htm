Complicating things a bit more is the physical location of the workbook after making the links. If you use relative links, but then move the workbook to a new file directory, you break all the links.

stop processing more rules - Microsoft Community

raja@ranashoponline say:

stop processing more rules

This issue always pops up when I try creating a rule, is there a way to fix this?

The rule has a condition that the server cannot process. The action 'stop processing more rules' will prevent all remaining server rules from being carrier out. Are you sure this is what you wanted to do?

diane poremsky mvp (slipstick.com) say:

That applies just to the message and that rule - if a message matches a rule, outlook stops checking it against additional rules and starts on the next message.  It can be faster when you have a lot of rules because outlook stops checking additional rules when the first match is found. 

Say you have a rule to move all mail from Bob to a folder and one to move all mail with "account 254" in the subject to another folder. If bob sends you a message with account 254 in the subject, stop processing in the bob rule means the account 254 rule won't touch it. If you want it in both folders (duplicate messages), remove stop processing. It will not affect other messages received, they will be tested against all rules until a match is found (if using stop processing). 

http://www.outlook-tips.net/tips/tip-42-stop-processing-more-rules/ has a little more information.

Hi Gerald:

You can do this in Word, depending on how exacting you want to be.

As Bob said, a "shape" (i.e. a vector graphics object) is a mathematical formula that computes displacement from an "origin".

When you select a shape, the various control handles that enable you to adjust it appear.  The light blue square ones enable you to re-size the shape in a single dimension.  The light blue round ones enable you to scale the shape in two dimensions equally.

The green one enables you to rotate it.  The Yellow diamonds enable you to adjust the values in the formula itself to adjust the shape.

The yellow diamond at the base of the arrow head enables you to adjust the included angle of the apex of the triangle that forms the arrow head (the "sharpness" of the arrow.  The one closest to the tip of the arrow enables you to adjust the width of the wings of the arrow.  The one at the other end of the arrow enables you to adjust the thickness of the shaft of the arrow.

To match colour, outline, gradient fill etc, use the Format Painter.  Select a shape and click the Format Painter (Paintbrush on the Formatting toolbar).  Then click the object you want to change.  The colour will be copied from one to the other.  If you double-click the format painter, it will stay on, colouring everything you touch, until you hit Esc to cancel it.

If you wanted to do a lot of this professionally, then you would buy a copy of Adobe Illustrator.  If you did not want to do this all day, or did not want to do it for a living, you would probably be able to find all you need in a cheaper alternative.  Some of the better ones are mentioned here: http://www.onextrapixel.com/2011/03/15/10-best-alternatives-to-adobe-illustrator/

Since you know the "elements" of your stored value are always "dd/mm/yyyy", but you can't be sure of the target machine's regional date value, you can coerce your stored value to behave like the target machine's date methods:

Dim CheckDate As Date

Dim StoredDate As String

StoredDate = <get the date value stored here>

CheckDate = DateSerial(Split(StoredDate, "/")(2), Split(StoredDate, "/")(1), Split(StoredDate, "/")(0))

The input for DateSerial is (Year, Month, Day), so this would create the CheckDate variable using the Year, Month and Day of the value you've stored, but it would do so in the machine's regional settings. You could then compare that to the current date:

If Date > CheckDate Then
