a. user say:

The built-in Administrator was never intended to be a user account. The built-in Administrator is system account provided as a lifeline in the event all user accounts are corrupted. In earlier versions of Windows there was no enforcement of the built-in Administrator. This changed in Windows 10.

You should always operate Windows with a user account, not a system account. The safest way to operate a computer is with a Standard account, although every computer needs to have one user account with Administrator permissions in order to perform tasks that require Administrator permissions.

A Standard account is safer because it has fewer permissions on the computer - there are many administrative tasks it can't do. In the event that your user account gets hacked, the attacker won't be able to make changes that only an Administrator can make.
