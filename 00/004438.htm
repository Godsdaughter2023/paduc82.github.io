Office 2016 does not have toolbars. The QAT is its own thing, not a toolbar. You can copy your macros to a document or your Normal template using Organizer.

    johannswart say:

    Check Box: Content Control control cannot be deleted

    I am constructing a Word table matrix with row titles and column titles. The other cells all contain a check box each which the user either selects or deselects. I want to have all the check boxes with the attribute "Content control cannot be deleted", but this attribute does not seem to copy when I copy the check box. Without having to laboriously set this attribute for each individual check box (a few hundred), is there another option available to assign this attribute to all check boxes?
    charles kenyon say:

    You need to select a "range of text" for the group command to activate. Word is picky about this.

    For purposes of grouping, in a table a cell can be a range of text. One or more contiguous rows can be a range of text. A column cannot be a range of text. Multiple cells (but not a complete row) cannot be a range of text.

    An entire table and more can be a range of text.

    The following macro should set all checkbox content controls in a document's body with the property of "no delete."

    Sub CheckboxNoDelete()
    '   Charles Kenyon
    '   May not work for checkboxes inside headers/footers or textboxes, haven't checked
    '   Sets property for all checkbox content controls in body of active document to "no delete"
    '   Checkbox may still be checked on unchecked
    '   21 June 2017
    '
        Dim CCtrl As ContentControl
        For Each CCtrl In ActiveDocument.ContentControls
            If CCtrl.Type = wdContentControlCheckBox Then CCtrl.LockContentControl = True
        Next CCtrl
    End Sub

    For instructions on using, see Graham Mayor's Installing Macros.

    You might want to set the dteOld using an actual date to count back 14 days if the 14 days is not counted from Date (ie. Date is todays date and calculates on day of running the code). It is possible to use an InputBox for this date if you want.

    Sub DeleteOldData()
        Dim ws As Worksheet
        Dim dteOld As Date
        Dim rngToDelete As Range
       
        Set ws = Worksheets("Sheet1")   'Edit "Sheet1" to your sheet name
        dteOld = Date - 14      'Count back 14 days from today
       
        With ws
            .AutoFilterMode = False  'Turns off AutoFilter (if on) to initialize without filters
            
            'Following sets Autofilter from row 1 (Column headers) to last data before a blank row
            .Range(.Cells(1, "A"), .Cells(1, .Columns.Count).End(xlToLeft)).AutoFilter
           
            'Following line is generic code for Filtered range irrespective of how many rows
            'Note that it is a nested With / End With (Inside of With ws)
            With .AutoFilter.Range
           
                'Set filter on date column to less than 14 days ago
                'Use m/d/yyyy format with Filter dates in VBA code (irrespective of your regional date format)
                .AutoFilter Field:=4, Criteria1:="<" & Format(dteOld, "m/d/yyyy")
                   
                'Next code assigns the visible rows to the variable (Excludes column headers)
                'Offset moves down one row off column header (then includes additional row at bottom)
                'Resize removes additional row at bottom created with the Offset
                Set rngToDelete = .Offset(1, 0) _
                                    .Resize(.Rows.Count - 1, .Columns.Count) _
                                    .SpecialCells(xlCellTypeVisible)
                                   
                rngToDelete.EntireRow.Delete    'Delete the visible rows
                   
               
            End With
           
            .AutoFilterMode = False  'Turn off AutoFilter
           
        End With
       
    End Sub
