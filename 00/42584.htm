[sequence corrected 14 June 2017 - 1 & 2 were the wrong way around so the solution would only work for those who had previously set the folder to be a Music type folder]

  1. In File explorer, right-click on the folder in the Navigation pane, select Properties, Customise, Optimise this folder for - Music.
  2. Set up the folder the way you want it [the type of View, the columns that appear in the Details view, the sort order].
  3. Click on the File tab at the top of File explorer, select Options [in older versions, Change folder & search options], select the View tab.
  4. Click on Apply to folders i.e. apply what you have just done to all folders of the type Music

All existing & future Music folders will now appear the way you have just set up.  You can set up all other folders types the same way [the different types are shown in the dropdown list you saw at Optimise this folder for.
