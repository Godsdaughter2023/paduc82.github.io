Trucs et Astuces utiles pour le Nokia N900 - Microsoft Community

stephabcd say:

Trucs et Astuces utiles pour le Nokia N900

Bonjour, 

 

 

Nous commençons ici un message sur les trucs et astuces pour le N900. Si vous en connaissez d'autres n'héistez pas à compléter. 

 

Bonne Lecture...

Cordialement,

Stéphane 

NokiaCare

 

 

Vous pouvez supprimer la sonnerie lors d'un appel entrant en renversant votre mobile de bas en haut.

Lorsque vous appuyez sur une touche du clavier, les contacts qui commencent par cette lettre s'affichent.

Appuyer deux fois sur le bouton marche/arrêt pour verrouiller l'appareil.

Lorsque vous êtes sur une page Web ou en train de lire les messages, glissez votre doigt sur l'écran de gauche à droite pour faire apparaître un pointeur de souris, en tapant l'écran vous pouvez sélectionner/copier le texte d'une page web ou d'un message. 

Appuyez simultanément sur ctrl + retour arrière pour faire apparaître le gestionnaire de tâche.

Lorsque vous êtes dans le navigateur tapez sur la touche retour arrière pour atteindre la dernière page.

Lorsque vous êtes dans le navigateur, utilisez le bouton volume pour zoomer sur la page.

Appuyez sur Ctrl + Shift + P pour prendre une photo instantanée

Appuyez deux fois sur la flèche bleue pour verrouiller les nombres et les caractères bleus du clavier. De la même façon vous pouvez verrouiller les majuscules en appuyant deux fois sur la flèche vers le haut.

Basculer le téléphone de haut en bas horizontalement éteint l'alarme. 

Depuis la liste des contacts, appuyez sur Ctrl vous permet d'ajouter un nouveau contact. De même en appuyant sur Ctrl lorsque vous êtes sur une page web vous permet d'afficher une nouvelle fenêtre.

Dans le navigateur par défaut, glissez votre doigt sur l'écran dans le sens des aiguilles d'une montre /ou dans le sens inverse pour zoomer sur la page.

Dans le lecteur vidéo/audio/internet/radio vous pouvez effectuer une recherche en saisissant des caractères comme pour les contacts.

Dans l'application Appareil Photo, le bouton volume supérieur permet de zoomer dans les deux sens.

Il est bien connu que dans le navigateur, la barre d'espacement fait défiler la page vers le bas et que la touche retour arrière ramène sur la page précédente. Il est un peu moins connu qu'en appuyant sur shift et les mêmes touches produit l'effet inverse. 

Un  appui long sur le bureau fait apparaitre le Menu du Bureau immédiatement d'appuyer sur le bureau puis sur la touche circulaire.

Si une application que vous utilisez tel que le navigateur se bloque ou ne réponds pas, cliquez sur bouton marche/arrêt et choisissiez Finir la tâche courante comme dans le gestionnaire des tâches de Windows.

Lorsque le lecteur de média est sur “En écoute" appuyez sur l'image de l'album pour basculer entre la chanson actuelle et la liste des chansons de l'album.

Dans le navigateur web vous pouvez utiliser ctrl + flèche droite pour atteindre le champ suivant dans un formulaire (comme lorsque vous appuyez sur un clavier normal). Ctrl + flèche gauche produit la même chose que shift-TAB, soit un retour au champ précédent du formulaire.

Ctrl b ajoute une page aux favoris

Choisissez le mode "Stockage de Masse" lorsque vous connectez le N900 à un PC via l'USB permet un chargement plus rapide.

don cuthbert say:

When you power on your Surface Pro 4, as soon as you see swirling dots, hold in the power button for 5+ seconds until it powers off.  Do this twice.  On the third power-on, it should go to Windows Repair Environment.  From there, you can follow the menus until you find Command Prompt.

At Command Prompt, try the following:

sfc /scannow /OFFBOOTDIR=C:\ /OFFWINDIR=C:\Windows

If sfc finds things that it cannot fix, run DISM:

DISM /Image:C:\ /CleanupImage /RestoreHealth

If you can use another computer, you can create Windows 10 install media on a USB stick (size between 4GB and 32GB) by running the Media Creation Tool found here:

https://www.microsoft.com/en-us/software-download/windows10

It is good to have the install media to boot from in case your Windows Repair Environment does not work sometime.

Another thing you could try from the Repair Menu, is startup options to go into Safe Mode.
