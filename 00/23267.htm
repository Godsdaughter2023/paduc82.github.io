Select B4:D13 and in the Conditional Formula dialog use Manage Rules;  Add New Rule: Use a Formula .... (last item in list)

You cannot count the CF colours directly but you obtain the required number by using the same logic as that used in the CF formula. Therefore, assuming that the first row of interest is row 10 and the last is row 100, try the following formula:

    =SUMPRODUCT(--(T10:T100="Y"),--(R10:R100<=0))

Dynamically linking task between project plans - Microsoft Community

dave shultz say:

Dynamically linking task between project plans

Can someone point me to a good summary of best practices concerning dynamically linking task between project plans? 

john - project say:

Dave,

This certainly isn't a complete list but the three most important best practices are:

1. Do NOT link summary lines to any performance task or other summary lines. Note: this applies whether cross-linking between projects or linking within a single project. Summary line linking in Project is permissible but should be used only by the most experienced users (that excludes about 99% of the Project user community)

2. DO keep all schedule logic in a "forward pass" structure. When linking between multiple projects there is sometimes a tendency to lose track of logic flow and end up with links that loop back around (backward pass). This results in circular relationships.

3. Once all linking is set up, do NOT rename or move files. Doing either will corrupt the link structure over time.

In addition when you say, "dynamic linking" I assume you are talking about predecessors and successors and not about paste links. The latter are ripe fodder for corruption.

As far as implementation, the easiest approach is to create a dynamic master and then set up the links at the master level. It is much easier to isolate desired tasks for linking when all subprojects can be displayed together. After links are set, first save the source file using Save As to its original name and then do likewise with the destination file, for each subproject. Once this is accomplished you can either keep or delete the master.

John

Smart Phones stream audio over a specific Bluetooth protocol called Advanced Audio Distribution Profile (A2DP).  While there isn't a standard streaming Wi-Fi protocol, there are technologies like DLNA.  The Lumia Play To app can stream media to compatible devices like TV's and speakers.  I'm not aware of any programs for the PC that can receive a DLNA stream, it usually goes the other direction.

You may want to use the Windows 8 Community forum to see if anyone has a fix for your Bluetooth problem with Windows 8.
