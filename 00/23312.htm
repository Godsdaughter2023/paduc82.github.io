When you create a document from a template, the macros in the template are available for use in that document, because a "reference" is created to the template.  If the document is however saved (even as a docm) and then sent or moved to another computer on which the template containing the macros is not available, the Normal.dotm template on that machine will be substituted for the original template and as the macros are not in that template, they will not be available for use on that document.

To get multiple toggle buttons to behave as radio buttons you could use something like this:

 

Private Sub ToggleButton1_Click()
    If Me.ToggleButton1.Value = -1 Then
        Me.ToggleButton2.Value = 0
    Else
        Me.ToggleButton2.Value = -1
    End If
End Sub

Private Sub ToggleButton2_Click()
    If Me.ToggleButton2.Value = -1 Then
        Me.ToggleButton1.Value = 0
    Else

marites_p say:

Hi,

To turn off the automatic start of Skype, kindly follow the steps below:

For the classic Skype version (7.xx):

Go to tools > options > general settings > untick Start Skype when I start Windows

For new Skype version (8.xx):

Go to your profile icon > select application settings > under startup and close > toggle off automatically start Skype.
