considerate_guy say:

What you are talking about is the project file (.mswmm) which is where Movie Maker stores the instructions for assembling your movie. A project file is really just a list of media file names and then the parameters for trimming, adding effects, etc.

The .mswmm format, in particular, is from a "classic" version of Movie Maker, either from Windows XP (Windows Movie Maker 2.x) or from Vista (Windows Movie Maker 6.0).

The latest versions of Movie Maker (Movie Maker 2010, 2011 and 2012) can open the .mswmm files from WMM 6.0 directly.  But they can't open up the project files from the older 2.x versions (2.0, 2.1 and 2.6).

What you can do is install Windows Movie Maker 6.0 which can understand .mswmm project files from any of the classic versions (WMM 2.x or 6.0). And it would then be able to save the project in 6.0 format which could later be used in Movie Maker 2012, if you wanted.

There are a few caveats:

  1. There are slight differences in the way effects/transitions/titles render in 6.0 compared to 2.x. The most notable is the Scroll, Perspective title is not as tilted back and "Star Wars-like" as in 2.x.
  2. No project file is sufficient on its own to create a movie. You must have all the media source files in the same locations as before or else you'll see Red Xs for the missing clips. If this happens, double-click on a missing clip, navigate to the new location and Movie Maker will reconnect the links.
  3. It's best, when done with edits to actually save a standalone copy of the movie in .wmv or .mp4 format. In that way you don't even need Movie Maker for future playback. The project on its own isn't enough.
