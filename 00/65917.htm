lemp say:

This should not be the case.  Other than hardware-related software, such as printer drivers, and very old software, any software that worked on a 32-bit Windows computer should work on a 64-bit Windows computer.  Even some printer driver packages will work for both 32- and 64-bit systems.

If you have any specific software that will not work, what is its name and version and what is the error message you get when you attempt to use it or install it?

You didn't answer most of the questions that I asked.  If your "HP 750" printer is, as I guessed, an HP PSC 750, then there is a Windows 7 64-bit driver readily available, as shown in the following screenshot from a Windows 7 64-bit computer:

When you connect the printer to the computer, Windows should recognize it an automatically install the driver.  In the case of old printers like the HP PSC 750, you will likely see a pop-up message asking for permission to search Windows Update for the driver.  You should grant this permission (and, of course, the computer will need to be connected to the Internet).  It may take 5 minutes or more to fully update the list of drivers stored on your computer, but eventually the correct driver will be downloaded and installed.
