If there are merged cells on the row then wrap text will not adjust the row height.

Otherwise it should.

If there are merged cells that don't need text wrapping themselves, unmerge them - you may need to use the horizontal alignment "Center across selection"  to get the effect you want.

If there are merged cells that do need text wrapping the only options are to do the row height adjustment manually or to use a macro to do it for you.

macropod say:

Word's hidden text display always applies to the whole document, not to any particular range. Basically you have two options:

1. Set some custom document property values or document variable values to True/False and embed the conditional content in IF fields in the document that test those property/variable values; or

2. Directly update specified (e.g. bookmarked) ranges according to the checkbox state.

For the first approach, you might create two custom document properties, named Chk1 and Chk2, respectively. In the body of the document, you'd have a pair of IF fields coded as:

{IF{DOCPROPERTY Chk1}= True "Content to display when CheckBox1 is checked"}

{IF{DOCPROPERTY Chk2}= True "Content to display when CheckBox2 is checked"}

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practicable to add them via any of the standard Word dialogues.

Your macros would then be coded along the lines of:

Private Sub CheckBox1_Click()
With ActiveDocument
 .CustomDocumentProperties("Chk1").Value = CheckBox1.Value
 .Fields.Update
End With
End Sub

Private Sub CheckBox2_Click()
With ActiveDocument
 .CustomDocumentProperties("Chk2").Value = CheckBox2.Value
 .Fields.Update
End With
End Sub

For the second approach, you might create two bookmarks, named Chk1 and Chk2, respectively. Your macros would then be coded along the lines of:

Private Sub CheckBox1_Click()
Dim StrTxt As String
With ActiveDocument
  If CheckBox1.Value = True Then
    StrTxt = "Content to display when CheckBox1 is checked"
  Else
    StrTxt = ""
  End If
  Call UpdateBookmark("Chk1", StrTxt)
End With
End Sub

Private Sub CheckBox2_Click()
Dim StrTxt As String
With ActiveDocument
  If CheckBox2.Value = True Then
    StrTxt = "Content to display when CheckBox2 is checked"
  Else
    StrTxt = ""
  End If
  Call UpdateBookmark("Chk2", StrTxt)
End With
End Sub
End Sub

Sub UpdateBookmark(StrBkMk As String, StrTxt As String)
Dim BkMkRng As Range
With ActiveDocument
  If .Bookmarks.Exists(StrBkMk) Then
    Set BkMkRng = .Bookmarks(StrBkMk).Range
    BkMkRng.Text = StrTxt
    .Bookmarks.Add StrBkMk, BkMkRng
  End If
End With
Set BkMkRng = Nothing
End Sub
