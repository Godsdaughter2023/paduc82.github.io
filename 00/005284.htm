pampieroni say:

Copy a doc with Track Changes

I'm trying to to copy a few pages of a document that has track changes with comments showing, but when I copy and paste, the tracked changes and comments will not show up. I want them to be included in the pasted new doc.  How can I do this??
rich michaels - mvp (greatcirclelearning.com) say:
Hi Pam,

There shouldn't be a problem with Comments, they should copy and paste to the new document without problem. Tracked Changes however requires some special handling.

In the source document turn OFF the Track Changes toggle. Then copy the text that you want and paste it into the new document. Markings of the changes should carry over when pasted. It is important that the toggle if Off before you copy select and copy the text.

Make sure that the Track Changes toggle is Off in the receiving document, otherwise everything will be marked as a change. After you have done this you can reset the Track Changes toggle in the source document.

Hope this helps

    shyam pillai say:

    Sub RandomizeTriggerItemsOnShape()
    Dim intSequence As Sequence

    If ActiveWindow.Selection.SlideRange(1).TimeLine.InteractiveSequences.Count = 0 Then
        MsgBox "This slide has no triggers"
        Exit Sub
    End If

    Set intSequence = ActiveWindow.Selection.SlideRange(1).TimeLine.InteractiveSequences(1)

    Dim I As Integer
    Dim J As Integer

    J = intSequence.Count
    For I = intSequence.Count To 1 Step -1
        intSequence(I).MoveTo GetRandomNumber(J)
    Next

    End Sub

    Function GetRandomNumber(x As Integer) As Integer
    Randomize
    GetRandomNumber = Int(x * Rnd) + 1
    End Function
