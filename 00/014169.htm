anula d'souza say:

Hi David,

 

As I understand Outlook hangs while working on it and also you are unable to open multiple emails in Outlook.

 

Follow the steps mentioned and check if it helps:

Step 1: Open Outlook in safe mode:

Open Outlook in safe mode to check if the issue is caused due to add-ins. Follow the steps mentioned to open Outlook in safe mode:

·         Close Outlook.

·         Press the windows key+R on the keyboard to open the Run window.

·         In Run window and type outlook /safe

Note: There is a space between 'outlook' and '/'.

 

If you are able to work with Outlook without any problem then the issue might be due to add-ins. In order to disable the add-ins follow the steps given below:

·         In Outlook, click on File< Options< Add-ins.

·         Under Manage select COM Add-ins.

·         Click on GO options beside Manage, uncheck all the add-ins and click Ok.

·         Try to open Outlook and check if it works fine.

 

Step 2: Repair the Outlook data file:

Scan the Outlook data file to check for errors and if errors are found repair the data file and check if Outlook work fine. Refer the link mentioned to repair Outlook data file:

http://office.microsoft.com/en-us/outlook-help/repair-outlook-data-files-pst-and-ost-HA010354964.aspx?CTT=1

Note: Ensure to run the scan PST till you get the message No errors found.

 

Step 3: Boot the system in clean boot:

Clean boot is performed to disable all startup programs and 3rd party services on the machine and check if they are conflicting with the Office program.

Refer to the link mentioned to boot the system in clean boot:

http://support.microsoft.com/kb/929135

Note: After checking in clean boot ensure to reboot the system in normal mode which is explained in the same article.

 

If Outlook works fine in clean boot then follow the complete article, perform advanced clean boot to check which program is causing the conflict. Once you have found the problem program, contact appropriate technical support team to update the program.

 

Step 4: Create new profile:

If the above steps do not fix the issue then create new profile and check. Refer the link mentioned to create new profile:

http://support.microsoft.com/kb/829918

 

Note: If Outlook works fine in new profile and continue using new profile and set new profile as default which is explained in the above article.

 

I hope the above suggestion helps. Let us know if you need any further assistance.

At the moment, it prints one sided - which means there are blank pages in between each page. I tried going into Printer Settings and changing it to print "double sided" but when I printed the document it did not print in the correct order. My printer is HP CM6040MFP.

Please help :)

john korchok say:

In the Publisher Print panel in the Settings section, ensure that the dropdown below Pages: is set to Booklet, side-fold. Then 2 dropdowns further, set it to Print on Both Sides Flip sheets on short edge.

Then click on Printer Properties below the selected printer and enable 2-sided or Duplex Printing. OK out. 

The Preview in the Publisher Print panel will now display a field with 1 in it. To the right of that field it should say of 2, since that's the total number of sheets you're printing. Then to the right of that is Front and Back. You have the correct setup when:

Sheet 1 Front is numbered 8 and 1

Sheet 1 Back is numbered 2 and 7

Sheet 2 Front is numbered 6 and 3

Sheet 2 Back is numbered 4 and 5

You may still have to play with the settings in Publisher and in your print driver to find a combination that prints the pages in correct order, but the Preview at least saves you some paper.
