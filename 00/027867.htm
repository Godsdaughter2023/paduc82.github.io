3. This you perform as a last resort. Nothing else is working.If the phone is not showing any activity, proceed with hard formatting , turn off your phone, hold the following buttons while pressing the power button. (the default code is 12345)

grschinon say: As for the "how it works" bit, you just dial the number from standby. The phone will intercept the key sequence, ask you to confirm and prompt you for the security code (the default is 12345 as radical24 pointed out).

You'll see all sorts of wacky things for the names of all the codes.

The *#7780# code is just intended to restore factory default setings and doesn't reset the phone at all.

*#7370# is called a "soft reset" because it resets the phone in response to a software condition (the keys *, #, 7, 3, 7, 0 and # being pressed in sequence).

The hard reset is so called because it prompts a reset in response to a hardware condition, namely 3 keys being pressed down at the time the phone switches on.

Don't forget to reformat the memory card as well if you're sending that back with the phone...

"MC Office"?  Is that a typo, do you mean "MS Office"?

Your Office license gives you the right to run either 32 or 64 bit Office.  You have to uninstall 32 bit Office first.

You can use your Product key to download the required installation file from this link:

Manage, download, back up, or restore Office 365 / 2016 / 2013 / 2010 / 2007

https://products.office.com/en-us/download-back-up-restore-microsoft-office-products   365 / 2016 / 2013 / 2010 / 2007 (you may have to scroll down)

 

MS still recommends you install 32 bit rather than 64 bit. Mainly because many software builders (including MS) have not bothered to create 64 bit versions of their addons.

For Excel the main reasons for switching to 64 bit are:

  • file size is approaching 2GB on HD
  • when open the file is using approaching 2GB of RAM (due to many formulas, or formula complexity)

other than that there does not appear to be any benefit to switching to 64 bit.
