gord dibben say:

This UDF can help with that.

Copy/paste this UDF to a general module in your workbook.

Function PrevSheet(rg As Range)
    n = Application.Caller.Parent.Index
    If n = 1 Then
        PrevSheet = CVErr(xlErrRef)
    ElseIf TypeName(Sheets(n - 1)) = "Chart" Then
        PrevSheet = CVErr(xlErrNA)
    Else
        PrevSheet = Sheets(n - 1).Range(rg.Address).Value
    End If
End Function

Example of usage...................

Say you have 31 sheets,  sheet1 through shee31...........sheet names don't
matter.

In sheet1 you have a formula in A10  =SUM(A1:A9)

Select second sheet and SHIFT + Click last sheet

In active sheet A10 enter =SUM(PrevSheet(A10),A1:A9)

Ungroup the sheets.

Each A10 will have the sum of the previous sheet's A10 plus the sum of the
current sheet's A1:A9

steve rindsberg say:
PowerPoint doesn't use inches internally, only points.

72 points to the inch, so it just takes a bit of math.

Or a function:

Function Points(sngInches as Single) as Single
  Points = sngInches * 72
End Function

usage:

Shape.Height = Points(10)  

Now as to maintaining proportions, you can either do the calculations yourself based on the proportions of the thing originally or you can let PPT do the work for you.  If you set .LockAspectRatio to true, PPT will keep the shape proportional as you change either height or width.

Dim lLockAspect As Long
With oSh
    ' save its current Lock Aspect Ratio setting
    lLockAspect = .LockAspectRatio
    ' lock the aspect ratio
    .LockAspectRatio = True
    ' now change either height or width
    ' but not both
    .Height = 144
    ' and reset lock aspect
    .LockAspectRatio = lLockAspect
End With

get emails from microsoft outlook 2010 to android smartphone - Microsoft Community

a. user say:

get emails from microsoft outlook 2010 to android smartphone

How do I get my emails from outlook 2010 to my android smartphone?
brian tillman [outlook mvp 2007-2019] say:

Connect both Outlook 2010 and your Android device to the same mailbox and you should see the same messages arrive in the Inbox. Since you say you use a POP account for Outlook 2010, make sure you configure the account to leave messages on the server so that the Android device can see them. That setting is on the Advanced tab of the account properties for POP accounts.

If your mail provider offers IMAP, changing to IMAP in Outlook 2010 will allow both devices to see the complete folder set and the data in those folders, including sent items and allow changes on one to be reflected on the other.

Note that neither IMAP nor POP can handle anything by mail messages. No contacts or calendar events. If you want hose to sync, you need to use a mail service that allows either Exchange or Exchange ActiveSync.

desiree yau say:

Hello,

The option to transfer the ownership of a notebook is currently not available. However, there's a workaround that we can do to prevent losing the notebook you've shared. Since the notebook has been shared, it's automatically saved to your OneDrive. That being said, we advise not deleting the file from your OneDrive and just close it from your OneNote 2013. From there, you'll no longer have a visibility on the shared notebook when you use your OneNote 2013. Just an added information, if you will delete the said notebook from your OneDrive, that's the time they'll no longer have access to it.

In case the workaround doesn't work for you, we highly suggest that you post this concern to our OneNote UserVoice, and suggest to have a feature that can transfer ownership. Actually, this is a good suggestion since it'll help other users who have the same concern. We appreciate any suggestions or feedback since it will help us improve our products and services. In addition, if some users have already posted the same concern, you can also upvote those posts.

Hope this information helps you. In case you need further assistance, don't hesitate to post back.

bernd p. say:

One possible procedure if the sharer has write access and OneNote 2016:

The sharer:

  1. Export the notebook (open in OneNote 2016) as Package
  2. Close the notebook
  3. Open the Package (double click in explorer)
  4. Is new owner now (of a local notebook)
  5. If wanted: Share the notebook to his own OneDrive

You after the sharer took ownership:

  1. Close the notebook
  2. Delete it on OneDrive

Bernd
