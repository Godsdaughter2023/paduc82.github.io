Hi Jeffrey,

In Windows 10, you're free to sideload an app from outside the Windows Store. However, please note that it's not recommended to sideload apps. If there's a need to sideload apps, you should only install from a trusted source. By default, the operating system  blocks you from sideloading apps. This is to protect you and your computer, as getting certified apps from the Store ensures that you're getting a good app that will not harm your device. To allow Windows 10 to sideload apps, you may follow these steps:

  1. Open Settings.
  2. Click on Update & security.
  3. Click on For developers.

  4. Under Use developer features, select the Sideload apps option.

  5. Click Yes to confirm the risks involved running an app outside the Windows Store.

  6. Restart your computer.

I had the same problem and this is how I fixed it:

I went into C:\Windows and right-clicked on the SysWOW64 folder and selected PROPERTIES.

I clicked on SECURITY and clicked on ADVANCED to give me ownership of this folder. then I went back to SECURITY and gave myself (as Administrator) full control of this file. Then I deleted the dllhost file in the SysWOW64 folder and rebooted. The problem was gone.

You will probably have to open Windows Task Manager, check the "show processes from all users" box and continually right-click END PROCESS to stop the offending dllhost files in order to get this done, but it will work.
