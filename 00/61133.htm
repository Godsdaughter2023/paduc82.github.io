considerate_guy say:

I'm not sure what you mean by "scene name".  Inside of Movie Maker as you add items to the storyboard they appears as rectangles.  Each rectangle represents a photo or video.  If you hover your mouse over the rectangle, you'll see the file name but you can't change the file name from within Movie Maker.

 

Perhaps you are referring to the scenes on a DVD?  If you choose to save your movie and pick the "Burn to DVD" option it launches Windows DVD Maker.  When you do this, Windows DVD Maker will automatically split your movie into several sections and add them to the "Scene" menu of the DVD.  However, you have no control over where these scene sections are created.

 

If you want more control over the process, you can create a separate movie for each scene on the DVD.  You'll want a separate project for each scene.  Save that as a project and then also as a movie on your computer.

 

When you are ready to make your DVD, run Windows DVD Maker directly.  It will then ask for the movies you want.  Give it the several movies you created for each "scene".  When you do it this way, each scene will start at the beginning.  However, back to your original question, I don't believe you have the ability to name each of these scenes.

peteratavalon say: Thanks for the quick reply. What you say makes sense from what I see in the program, but I'm looking at this "tip" site: http://desktopvideo.about.com/od/desktopediting/ss/mmeditvideo_2.htm that shows actually labels under the clips. Am I looking at a different animal?

considerate_guy say:

You are looking at one of the prior versions of Movie Maker, specifically Windows Movie Maker 2.1 from Windows XP.  In the prior versions there was a concept of a "collection" area where you could organize your clips.  It allowed you to essentially add start and end points, make virtual splits, name the parts of a clip with different names, etc.  You would then drag these parts from the collection to the storyboard/timeline area.

 

In the latest versions of Movie Maker, it was decided that this extra work area could be confusing, especially to novice users.  The Movie Maker team opted to instead have you bring everything into the storyboard area to work on it there.  You can make splits and set start/end points, but there wasn't really a need to have you name these clips.

 

Anyway, hopefully that bit of history helps and explains why you don't see the exact same thing as described in that tutorial. 

 

I suggest looking at these video tutorials for the latest version, since they will be more relevant.

Since the issue happened after a recent update, you can perform a System Restore. System Restore takes your PC back to an earlier point in time, called a system restore point.
