Hi chandg999 and welcome to the Skype Community!

Currently, both features cannot be disabled. There is no option in Skype to disable saving your chat history since this is native behavior of the app. The quick sign-in on the new Skype on Mobile is also a newly added feature requested by a number of Skype users so our developers included on the current version releases. 

What I can suggest as a workaround is to delete your chat conversations. To delete a chat on the new Skype: 

  • Tap and hold the conversation you want to delete from the Chats screen. 
  • Click Delete chat from the given menu.
  • Then tap Delete to confirm it.

unexpected changes to desktop icons - Microsoft Community

ganutcracker say:

unexpected changes to desktop icons

When I setup access to an online program with a icon i WIndows 7  it promptly changes to a generic windows icon. How do I maintain the icon i have downloaded?

anannya podder say:

Hi,

 

 

I see that when you setup access to an online program with an icon it promptly changes to a generic Windows icon.

 

 

 

 

Does the issue occur with all online programs?

 

 

You may try these steps:

 

 

Step 1:

If the desktop shortcut icons for programs have changed to Windows default icon then. You may delete them and go to C:\Program Files\ folder and create new shortcut and check if it helps.

a. Right click on the desktop shortcut icon; which has changed to Windows Default icon; and select delete. Click Yes to confirm the deletion.

b. Click Start and click “Computer”.

c. Go to the following location C:\Program Files\ and locate and open the program’s folder whose icon you deleted.

d. Right click on the program’s executable file and click on “Send to” and then click on “Desktop (create shortcut).

e. Check if the right icon appears for the program.

 

 

Step 2:

Rebuild the Icon Cache Database and check if it helps.

a. Close all folder windows that are currently open.

b. Launch Task Manager using the CTRL+SHIFT+ESC key sequence, or right click on taskbar and click on "Start Task manager".

c. In the Process tab, right-click on the Explorer.exe process and select End Process.

d. Click the End process button when asked for confirmation.

e. From the File menu of Task Manager, select New Task (Run).

f. Type "cmd.exe" (without quotes), and click OK.

g. In the command prompt window, type the commands one by one and press ENTER after each command:

CD /d %userprofile%\AppData\Local

DEL IconCache.db /a

EXIT

h. In Task Manager, click File, select New Task (Run).

i. Type "explorer.exe" (without quotes), and click OK.

j. Check if the issue is fixed.

 

Hope it helps!

If you need further assistance on this issue or any Windows issue in future let us know.
