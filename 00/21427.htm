diane dennis say:

Tabbing down in table cells rather than across...

Hi All!

I create forms utilizing tables and fill-in fields.

I need to be able to TAB down in the table cells rather than across.

Thank you!
Diane Dennis
rich michaels - mvp (greatcirclelearning.com) say:

Hi Diane,

If you upgraded to Office 2011 you could use this code. I've posted it in case there's anyone who has Word 2011 and would like the functionality. Insert it into your Normal template and assign a keystroke sequence.

For anyone who's wondering ... to make it behave correctly you have to use a Range command when moving down and a Selection command when moving up.  Why? It's not in my pay grade to know. :-)  At least that occurs on Word 2010 where I wrote it and it tested out fine on 2011.

 

Sub MoveDownTableColumnByCell()
    Dim rng As Word.Range
    Set rng = Word.Selection.Range
    If rng.Information(Word.WdInformation.wdWithInTable) Then
        If rng.Rows(1).IsLast Then
            Beep
        Else
            rng.Cells(1).Select
            rng.Collapse (Word.WdCollapseDirection.wdCollapseEnd)
            Selection.MoveDown unit:=Word.wdLine
        End If
    End If
End Sub


Sub MoveUpTableColumnByCell()
    If Selection.Information(Word.WdInformation.wdWithInTable) Then
        If Selection.Rows(1).IsFirst Then
            Beep
        Else
            Selection.Cells(1).Select
            Selection.Collapse (Word.WdCollapseDirection.wdCollapseStart)
            Selection.MoveUp unit:=Word.wdLine
            Selection.Cells(1).Select
            Selection.Collapse (Word.WdCollapseDirection.wdCollapseStart)
        End If
    End If
End Sub

ex.

A B C D E F G
1)   "               SKU         "               ,               "             Title          "


Now I need to take the data in each one of these cell and put them in one cell

ex.

A
1)    "SKU","Title"

I have over 5000 products so I need to do this in one big step. How do I do this...Please help! :(


gary's student say:

Use concatenation.  Since you already have data in column A, insert a new column to the left of column A

 

Now in the newly blank A1 cell, enter:

=B1&C1&D1&E1  add as many terms as you need.

Then copy A1 and paste it down the column as far as you need.
