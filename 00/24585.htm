rob koch say:

The reason is WOW64 as described in this article at Wikipedia:

http://en.wikipedia.org/wiki/WoW64

 

WOW64 allows 32-bit applications to run on 64-bit versions of Windows and is always included as a subsystem, so without the x86 portions of MSE it couldn't protect either any 32-bit applications or in fact the entire WOW64 subsystem itself from getting malware.  Since this malware would still have complete access to your entire PC via WOW64, this would effectively leave the PC unprotected, regardless of whether you ever use a 32-bit installaed application or not.
