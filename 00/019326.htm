charles kenyon say:

Ignore Charianne's response which is about AutoCorrect, not AutoText. (If you are writing about AutoCorrect, by all means read it.)

It depends on where your AutoText entries are stored in Word 2003. They will be in one or more templates. The default storage place is in the normal.dot template. If that was the case, open the normal.dot template in Word 2016. Save it as a .dotx template in Word 2016 and move it to your Word Startup Folder or in the Building Blocks folder. See Where can Building Blocks be stored?

Saving as a .dotx file will strip out any macros. If it is stored in the startup folder, it could be simply the .dot file which would preserve any macros. If stored in the Building Blocks folder, it must be a .dotx file. The AutoText will be available if it is stored in either location.
