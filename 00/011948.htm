ronrosenfeld say: How did you use the IF...OR?   One of the problems with worksheet functions is the difficulty of searching for "1" and finding it in the number "10".

Here is a User Defined Function  (UDF) that may accomplish this task.  It searches only for "whole words" where a word consists of only letters, digits or the underscore character; and the word is bounded by either a non-word character (e.g. your comma), or the beginning/end of the string.

To enter this User Defined Function (UDF), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this User Defined Function (UDF), enter a formula like

=FindWords(SearchIn, WordsToFind)

WordsToFind can be a cell reference (with a single "phrase" per cell; it can be a single phrase or number, or it can be an array constant.

e.g:  =FindWords(A1,$F$1:$H$1)

====================================
Option Explicit
Option Base 1
Function FindWords(SearchIn As Range, WordsToFind)
    Dim RE As Object
    Dim I As Long, lCnt As Long
Set RE = CreateObject("vbscript.regexp")
With RE
    .ignorecase = True
    Select Case TypeName(WordsToFind)
        Case Is = "Range"
            lCnt = WordsToFind.Count
        Case Is = "Variant()"
            lCnt = UBound(WordsToFind)
        Case Else
            WordsToFind = Array(WordsToFind)
            lCnt = 1
    End Select
   
    For I = 1 To lCnt
        If Len(WordsToFind(I)) > 0 Then
            .Pattern = "\b" & WordsToFind(I) & "\b"
            If .test(SearchIn) = True Then
                FindWords = 1
                Exit Function
            End If
        End If
    Next I
End With
End Function

Streaming Videos - Microsoft Community

mbamg---01 say:

Streaming Videos

Technical Level : Intermediate

Summary

Details
Almost all Nokia phones have a streaming client built into them that is able to stream video's from websites or even network operator sites such as Vodafone Live.

In order to stream correctly you will need the correct streaming settings from your network operator. If they were not supplied with your handset from out of the box it is advisable that you contact your network operator and ask them to send them to you OTA.

General rules for streaming:

1: The access point used for the internet connection and streaming must have the same APN and proxy.
2: Streaming will work best when you have 3G coverage so avoid trying to stream in areas where signal quality is low or you do not have 3G connectivity
3: Streaming will make the phones battery run down a lot quicker and it will also cause the phone to become quite hot
4: Depending on your network operators tariffs streaming can be charged at the standard data rate or at a flat rate (consult them before attempting any lengthy streaming)

You can also obtain settings from this site

http://www.nokia.com/phonesettings
