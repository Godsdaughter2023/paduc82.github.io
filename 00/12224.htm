Deepak,

Your excel file gets a temporary lock file (also called "Owner File" ~$samplefile.xlsx) when it is opened. That's why you get a notification that the file is locked for editing by 'PersonXYZ', open 'Read-Only' or click Notify. 

One another way out is to delete that file. It is a hidden file. 

Follow the below steps :

1. Open Folder Options under Control Panel (from the start menu)

2. Click on the View Tab

3. Click Show hidden files, folders and drivers under Advanced settings

4. Uncheck Hide protected operating systems files (Recommended) and Click OK

You should see that file in the same directory the original file is in. Delete it and open the document again.

Extract all unique text values from four separate columns, across multiple sheets

Hi,

  • I have four columns (J, O, T, and Y) that I want to extract distinct values from, without regard to case. (So, RED would be the same as Red or red, and the distinct value would just be "red")
  • All sheets have the same columns.
  • I would like to extract all the distinct values to a combined new sheet. 

Could someone please lend some assistance? Any help is greatly appreciated!

tasos_k say:

try this code

remove duplicates from all sheets (columns J,O,T,Y) in active workbook

results in a new sheet

Sub UniqueValues_from_FourColumns()
Const sName$ = "Unique_Data" '<< new sheet name/unique values
Const N As Integer = 1 '<< data from row 1/no headers, change as needed
Dim newWS As Worksheet

Dim r As Long, i As Integer, x As Integer, t As Long
Application.ScreenUpdating = False
For Each ws In Sheets
Application.DisplayAlerts = False
If ws.Name = sName Then ws.Delete
Application.DisplayAlerts = True
Next
Dim v As Variant
v = Array("J", "O", "T", "Y")
Set newWS = Sheets.Add(after:=Sheets(Sheets.Count))
newWS.Name = sName
t = 1
For i = 1 To Sheets.Count - 1
r = Sheets(i).UsedRange.Rows.Count
For x = 0 To UBound(v)
Sheets(i).Range(v(x) & N & ":" & v(x) & r).Copy
Cells(t, 1).PasteSpecial xlPasteValues
t = Cells(Rows.Count, 1).End(xlUp).Row + 1
Next
Next
Application.CutCopyMode = False
With Range("A:A")
.RemoveDuplicates Columns:=1, Header:=xlNo
.Sort Key1:=[A1], Order1:=xlAscending, Header:=xlNo
End With
Application.ScreenUpdating = True
End Sub
