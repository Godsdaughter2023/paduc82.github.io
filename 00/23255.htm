jay freedman say:

Simply copying and pasting a plain-text content control from the Developer tab has never provided an automatic update.

There are specific content controls in the Insert > Quick Parts > Document Property dropdown that do automatically update, but that's because they're "mapped" to XML in the document's file structure. When you exit one of these controls, the new value is copied into the XML node, and that updates all the other controls that are mapped to the same node.

You can create mapped content controls outside of the Document Property group. For more information on mapping and an add-in tool for creating mapped content controls, see these pages:

http://gregmaxey.com/word_tip_pages/mapped_content_controls.html

http://gregmaxey.com/word_tip_pages/content_control_tools.html
