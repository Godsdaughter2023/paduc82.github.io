Minimizing Windows live mail in Windows 8 - Microsoft Community

rpvinny say:

Minimizing Windows live mail in Windows 8

How do I minimize windows live mail in Windows 8 and not have it show in the task bar on the left side.
cmdrkeene say:
The program wasn't built to allow that, so we can't offer much help.  There are some tricks you can use to try to fool it into working (like this), but that's not supported and if it causes problems you're on your own.  It might not even work at all.

If your Taskbar is on the default setting ("show just the icon"), the point is fairly moot anyway.  Keeping it in the left side provides some substantial benefits:

  • Consistent access from the same spot at all times
  • If the icon is pinned, it's even there when the program isn't running
  • Notification (smaller icon inside the icon) to show when you have new messages
  • Jump list (right-click the icon for direct links to common tasks)
  • Keyboard shortcut by taskbar position.  Example, if Mail is the 3rd icon over, press Windows+3 to open Mail (or switch to it).
On the other hand, having the icon disappear when minimized causes impacts to workflow:

  • Accessing the program would require going to different places depending if the program was closed, open, or minimized.  Example:  "Start the program from your Desktop.  Unless it's not there, then go to Start / All Programs.  Unless it's already running, then it's on your taskbar.  Unless it's minimized, then it's over by the clock.  Unless it's under the hidden icons".  User feedback shows users don't like playing hide-and-seek with their icons, and such scenarios are no longer supported.
