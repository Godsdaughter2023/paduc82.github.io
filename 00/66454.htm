greg carmack - mvp 2010-2020 say:
Hi Richard. I'm Greg, a volunteer installation specialist and 8 year Windows MVP, here to help you.

The new User name is an abbreviation of your Windows Account email address which is what is assigned when you choose to sign-in with the account during install of Windows. Apparently it cannot connect at that moment to the account to learn the name associated with it so it just uses the first five letters.

This has annoyed many and I reported it to the Windows 10 team for correction repeatedly. Only recently they replied that this was intended and would not likely be changed soon.

Unfortunately that User account name cannot be changed without reinstalling and first setting up a Local Account with the preferred name. If one tries web methods to change the account it can result in lock-out requiring undoing the registry fix from boot media.

Sorry that this is the way it is, but just know that someone tried to get it changed as best I could and it was not deemed important enough to change.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.
