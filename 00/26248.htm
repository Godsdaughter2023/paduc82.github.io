macropod say:

I agree with Charles - up to a point. Instead of:

    With Selection.Font
        .Name = "Times New Roman"
        .Size = 12
        .Bold = False
        .Italic = False
        .Color = -587137025
    End With

you should apply those characteristics to a suitable paragraph Style and apply that Style to the whole document. Even so, that won't cure all 'direct formatting' that might have been done.

The following macro assumes you've applied the desired attributes to the 'Normal' Style and removes any 'direct formatting' that might have been done. As for the colour & bold application, that too could be done via character Styles for each of your Slow, Very Slow, Normal, Strong, & Very Strong texts, but you'd still need a similar macro to apply those Styles. Note the addition of:

    .MatchCase = True
    .MatchWholeWord = True

to overcome the problems you mentioned.

Sub Demo()

Application.ScreenUpdating = False
With ActiveDocument.Range
  .Style = "Normal"
  .Font.Reset
  With .Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Replacement.Font.Bold = True
    .Replacement.Text = "^&"
    .MatchCase = True
    .MatchWholeWord = True
    .Forward = True
    .Wrap = wdFindContinue
    .Format = True
    .Text = "Slow"
    .Replacement.Font.Color = RGB(238, 132, 52)
    .Execute Replace:=wdReplaceAll
    .Text = "Very Slow"
    .Replacement.Font.Color = RGB(214, 40, 40)
    .Execute Replace:=wdReplaceAll
    .Text = "Normal"
    .Replacement.Font.Color = RGB(238, 198, 67)
    .Execute Replace:=wdReplaceAll
    .Text = "Strong"
    .Replacement.Font.Color = RGB(13, 171, 118)
    .Execute Replace:=wdReplaceAll
    .Text = "Very Strong"
    .Replacement.Font.Color = RGB(11, 83, 81)
    .Execute Replace:=wdReplaceAll
  End With
End With
Application.ScreenUpdating = True
End Sub

davem121 say:

Hi Euge

This macro should do that for you . . .

Sub MultiplySelection()

   Const multiplier As Double = 1000000 

   Dim c As Range

   For Each c In Selection
      c.Value = multiplier * c.Value
   Next c
End Sub
