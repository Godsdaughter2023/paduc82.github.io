[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\FVE]
"UseAdvancedStartup"=dword:00000001
"EnableBDEWithNoTPM"=dword:00000001
"UseTPM"=dword:00000002
"UseTPMPIN"=dword:00000002
"UseTPMKey"=dword:00000002
"UseTPMKeyPIN"=dword:00000002

Now have my drive encrypted.. Hope this helps out anyone else who has this issue.

Only systems that come preinstalled with Windows 8 Pro or 8.1 Pro carry downgrade rights to Windows 7 Professional. Retail and OEM System Builder licenses do not carry downgrade rights.

Using OEM System Builder license is not illegal, its just that it has certain restrictions how it is licensed, otherwise, its no different from retail licenses. The only area of concern where OEM System Builder license might be a problem is when transferring it to new hardware such as new computer.

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system 

Even under these restrictions, you can still try phone activation to get it reactivated:

shilohbob say:

http://windows.microsoft.com/en-us/windows/preventing-low-memory-problems#1TC=windows-7

The link above gives a explanation for 'low memory problems' even though it refers to W7 it is still relevant to W8,  I understand that you may have read this before.

Using 'Task Manager' you need to determine which application/process is utilising high memory resources;

I would put my  money on either Chrome or Norton AV or both being the problem.

If you search thru various forums both have causes problems with W8.1

Personally I would uninstall, first Chrome, making sure it is completely removed, restart the laptop, run 'Task Manager', checking if memory usage has reduced.

Secondly uninstall Norton Av, again making sure it is completely removed, using Norton uninstall tool, link below:

https://support.norton.com/sp/en/us/home/current/solutions/kb20080710133834EN_EndUserProfile_en_us

Restart laptop and recheck memory usage.

If after carrying out action above you reduce your memory usage considerable, you have identified the problem, if memory usage is not reduced, further action to identify cause is required, by observing memory usage in 'Task Manager'

You could also increase 'Virtual Memory' or RAM, but that is not actually identifying the cause of the 'low memory' warnings. (Although it may get rid of the warnings.)

Unfortunately tracing 'low memory' warnings can be an arduous tasks.
