If column B in worksheet Main has a value of "HOSP" then return the data in column A (of the same row)

 

To column A of the same row to the Customer Service 1 worksheet.

bernie deitrick say: On Customer Service 1, do you want the values in the same rows as they are on Main (as you show) or grouped together starting in, say, row 2?

shaqueawhite say: Grouped together starting in row 3 would be perfect. I wasn't sure how possible that would be.

bernie deitrick say:

Array-enter (enter using Ctrl-Shift-Enter instead of just Enter) in a cell of row 3:

=IFERROR(INDEX(MAIN!A:A,SMALL(IF(MAIN!$C$1:$C$1000="HOSP",ROW(MAIN!$C$1:$C$1000)),ROW(A1))),"")

Then copy down for as many rows as you expect to need (at least until it returns blanks) and then copy across to pick up column B from MAIN.
