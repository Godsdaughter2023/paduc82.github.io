geethu b say:

Hi Jurgen,

Thank you for the reply.

Teredo is a tunneling protocol designed to grant IPv6 connectivity to nodes that are located behind IPv6-unaware NAT (Network address translation) devices. It defines a way of encapsulating IPv6 packets within IPv4 UDP (User Datagram Protocol) datagrams that can be routed through NAT devices and on the IPv4 internet.

Teredo is installed by default and enabled so that it remains in a standby mode and comes into action only when required. This is typically when you install an application that needs Teredo to function properly. The main problem with Teredo is that it involves a bit of heavy-duty work, and if it is switched on unnecessarily, it might slow down your network connection.

I would suggest you to uninstall and reinstall the Microsoft teredo tunneling adapter and check if it helps.

a.      Click Start, type Device Manager in the Start Search box, and then press ENTER.

b.      Click Continue. The Device Manager Dialog box is displayed.

c.       If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

d.      In Device Manager, expand Network Adapter right click on "Microsoft teredo tunneling adapter ", click "Uninstall" and click "OK".

Restart your computer and Windows 7 will reinstall all USB controllers automatically.
