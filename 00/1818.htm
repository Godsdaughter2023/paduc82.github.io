Revised code for new ranges

 

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Target.Cells.Count > 1 Then Exit Sub
    If Not Intersect(Target, Range("B10:B38,D10:D38")) Is Nothing Then
        Calendar1.Top = Target.Top + Target.Height
        Calendar1.Left = Target.Left + Target.Width / 2 - Calendar1.Width / 2
        Calendar1.Visible = True
        Calendar1.Value = Now
    ElseIf Calendar1.Visible Then Calendar1.Visible = False
    End If
End Sub
Private Sub Calendar1_Click()

End Sub

bilo cape town say:

The auto upgrade makes you do it all again unfortunately.

But the GOOD news is that Microsoft made the Media Creation Tool available so that you can save the installation ISO on your PC - on a USB Flash Drive - or a DVD.

Get it here: https://www.microsoft.com/en-us/software-download/windows10

Skip the top option to 'Upgrade now'. The second option on that page is to 'Save the media for another PC'.

That means ANY PC. Including the one you are using.

The defaults offered will be for the version (Home / Pro / Ultimate W7 / Win10) that you are currently using.

If you need a different version - uncheck the box below the defaults.

The options are then selectable if you need a different version.

Then you get 3 options.

1. Burn a bootable USB Flash Drive

(The BIOS of the PC you want to upgrade must have the option to boot from a USB device.)

2. Burn a bootable DVD

3. Save the ISO on the PC you are using currently.

If you save the default version of the ISO on the PC you are using:

1. To upgrade - just execute the SETUP.EXE file in the ISO file

2. Use this utility to burn an installation DVD or USB Flash Drive.

    Copy and paste in your browser's address bar to download: http://wudt.codeplex.com/#

The DVD or USB Flash drive that you burn can be used to do an upgrade.

After an upgrade has been done on a PC - you can do a CLEAN INSTALL,

to start afresh without any add-ons or 'trial software' that often comes on new PC's.

The clean install will retain the activation after an upgrade is done.

If you do the upgrade and prefer to continue staying on your current Win 7 or Win 8

you can 'Roll Back' to it after doing the upgrade. The registration is preserved.

The Identification of valid systems is stored by Microsoft, so that you can do the upgrade

again any time in the future from the ISO you saved -  even if it is past the 29th July 2016 cutoff date for the FREE upgrade.

Get rid of any 3rd party antivirus and make sure that Windows Defender is active in Windows 10.

It is part of the Windows 10 system and is automatically updated for you.
