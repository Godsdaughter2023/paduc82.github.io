anula d'souza say:

Hi Henry,

Thank you for posting in Microsoft Community. As per the issue description you're unable to open the Office applications.

I understand how hard it is when the program do not work the way it should be. The issue might be due to the .exe files that are running in the background (Clview.exe and OLicenseHeartbeat.exe). Let me help you with the issue.

 

Follow the steps mentioned below and check if you are able to open the Office applications:

Step 1: Boot the system in clean boot mode to check if any non-Microsoft programs are causing the conflict.

Refer the link mentioned below to boot the system in clean boot mode:

http://support.microsoft.com/kb/929135

Note: If Office applications works fine in safe mode then follow the complete article to perform advanced clean boot to check which program is causing the issue. Ensure to boot the system in normal mode which is explained in the same article after checking in clean boot mode.

If you receive the error message in clean boot mode too, then repair Office 365 installation when the system is booted in clean boot mode and check.

 

Step 2: Uninstall Office 365 completely from the system and re-install it:

Remove the Office 365 setup files using the Microsoft Troubleshooter from this KB article and then try installing the Office 365

http://support.microsoft.com/kb/2739501
 

Note: If you have a 64 bit operating system then download the fixit tool save it on the system and then try to run the fixit from the saved location.

Once the uninstallation is complete, reboot the system and then try to install Office 2013 and check.

 

I hope the above suggestion helps to resolve your issue. If you need further assistance on this particular issue or any other Office related issues let us know and we will be happy to help you.

Thank you.
