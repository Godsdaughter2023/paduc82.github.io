  1. You can fix the issue by going to Microsoft  Preferences.
  2. Click Spelling & Grammar
  3. Make sure the item highlighted above in screenshot is either set as shown or click on Don't Check to reveal Choices.
  4. Set either to Don't Check or to one Space.

The two spaces after a sentence is a holdover from typewriter days.  

In the heyday before the IBM Selectric®, and the DaisyWheel Printers all typewriters used a Monospaced font either Elite (a smaller thinner print), or Pica (a Larger, thicker Print).

With the advent of Daisy-wheel printers and with replaceable daisy-wheels type could be variable spaced fonts could be created.

In the days of Monospaced fonts, it was necessary to add an additional space to distinguish between sentences.

Now forty, fifty years later we have Two groups that Hang on to such archaic methods. The APA  and perhaps the Legal Profession and they have one additional requirement Double spacing of line of type. 

Many College and universities base their requirements for writing Papers and dissertations on APA guidelines.

If you are in such institutions either as students or employees then go by the institution's Guidelines. But if you are an average person It is not necessary. So adjust as I have described above.

Find Replace Macro - Microsoft Community

mordecai98 say:

Find Replace Macro

In Word 2010, every day I have a new doc  which I need to find and replace the same things.  Is there a macro or something that can be embedded into Word so I can quickly find and replace instead of doing a whole bunch of them one at a time?

Thanks
Mordecai
pesach shelnitz say:

Hi Mordecai,

 

Multiple replacements can be done using a macro. The following macro is an example for replacing various fractions written in the in-line form, for example, 1/2, by the single-character equivalent, for example, ½. The array members in this macro can be replaced by the texts that you want to find and the replacement texts.

Sub ConvertFractions()
    Dim FindArray As Variant
    Dim ReplaceArray As Variant
    Dim MyRange As Range
    Dim i As Long
    Dim pos As Long
   
    FindArray = Array("1/4", "1/2", "3/4", "1/3", "2/3", _
        "1/8", "3/8", "5/8", "7/8")
    ReplaceArray = Array(ChrW(&HBC), ChrW(&HBD), ChrW(&HBE), _
        ChrW(&H2153), ChrW(&H2154), ChrW(&H215B), _
        ChrW(&H215C), ChrW(&H215D), ChrW(&H215E))
    Set MyRange = ActiveDocument.Range
    pos = MyRange.start
    For i = LBound(FindArray) To UBound(FindArray)
        MyRange.start = pos
        With MyRange.Find
            .Text = FindArray(i)
            .Replacement.Text = ReplaceArray(i)
            .Forward = True
            .Execute Replace:=wdReplaceAll
        End With
    Next
End Sub

This macro would be efficient if you are searching for exactly the same texts and replacing them by the same texts every time. If however, you are replacing the same texts repeatedly by different texts in the same basic document, I suggest that you consider creating a form with a pop-up dialog box for entering the replacement texts.

Write back with more details if you need help in using this macro or getting started with creating a form.

 

Hope this helps,
Pesach

eileen gal say:

Hi Carol, 

The notification that you are getting when opening files is a default message particularly when you are opening or creating files from an external storage. 

As a workaround, you may have this option disabled by modifying the registry key. Please note that the said changes will only apply to the current user's profile. Please follow the steps below: 


  1. Click Start, and then click Run. 
  2. In the Open dialog box, type regedit, and then click OK.

    Notes:
    • You have to modify only one of these registry subkeys. You do not have to modify both.
    • If the Security subkey already exists, go directly to step 6 after you select the Security subkey. 
  3. In Registry Editor, locate one of the following registry subkeys for 2016 Office:
    HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common
    Or, in Registry Editor, locate one of the following registry subkeys for Office 2016:
    HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common 

  4. Click the registry subkey, point to New on the Edit menu, and then click Key. 
  5. Type Security, and then press Enter to name the key. 
  6. On the Edit menu, point to New, and then click DWORD Value. 
  7. Type DisableHyperlinkWarning, and then press Enter to name the entry. 
  8. In the right pane, right-click DisableHyperlinkWarning, and then click Modify. 
  9. In the Edit DWORD Value dialog box, click Decimal, and then type 1 or 0 under Value data.

bob jones aka: cybertaz say:
When you get to step #6 you can select either of the 3 options for completing the merge:
  • Merge to Printer: Use this if immediately printing labels, name badges, form letters, etc.
  • Merge to New Document: use this to create a Word document file containing a copy of the Main Document addressed to each recipient. This would commonly be used if the printing is not to be done immediately, to archive a copy of the merge, or to create a completed document for reprinting on an as needed basis rather than having to repetitively redo the merge.
  • Generate e-mail messages: Use this to send copies of the Main Document to each recipient as an email message

If you're doing a merge to email click the 3rd button.


From you're description it sounds like you're clicking the 2nd button which creates a new Word document containing a copy of your Main Document for each recipient, then trying to use the Mail Merge Manager to send that document as email messages. You can't go on to use that 'end result' to create another 'end result' - IOW, that is a type of merge, not a step in the emailing process. The button is dead because you're no longer in a merge-related document.


If you don't want to create the new document, don't click the 2nd button. OTOH, if you do want the document but still need to do the email merge, go back to the Main Document & click the 3rd button which will be active once you get there.
