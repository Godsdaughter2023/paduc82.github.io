shyam pillai say:

As John says, you need to use the objects in context with the model.

This should work for your needs.

 

 Dim I As Integer
 Dim J As Integer
 Dim K As Integer
 Dim shp As Shape
 Dim sld As Slide
 
'Assuming that the shape selection is a table.
 
Set sld = ActiveWindow.Selection.SlideRange(1)
 Set shp = ActiveWindow.Selection.ShapeRange(1)
 
With shp.Table
     For I = 1 To .Rows.Count
 
        For J = 1 To .Columns.Count
             If .Cell(I, J).Shape.TextFrame2.HasText Then
                 With .Cell(I, J).Shape.TextFrame.TextRange

                     'Use TextFrame and not TextFrame2 if you want the hyperlinks info
                     'Check if the textrange has any hyperlinks on it.

                     If .ActionSettings(ppMouseClick).Action <> ppActionNone Then

                        'A run with a hyperlink will show up as a separate run in 2010

                        For K = 1 To .Runs.Count
                            If .Runs(K).ActionSettings(ppMouseClick).Action <> ppActionNone Then

                                With .Runs(K)
                                    Call sld.Shapes.AddShape(msoShapeRectangle, _
                                                shp.Left + .BoundLeft, _
                                                shp.Top + .BoundTop, _
                                                .BoundWidth, _
                                                .BoundHeight)
                                End With

                            End If
                        Next
                    End If
                 End With
             End If
         Next
 
    Next
 
End With

austinfrank say:

Prevent Formulas from Changing when Inserting New Column

I am relatively new to Excel, but I am wondering how I can set a formula so that it does not change when I insert a new column into the sheet. For example: I want C3: =F3, but when I insert a new column, I dont want C3: =G3. I want it to remain as it was before. I also need to be able to copy and paste the formula if possible because I have about 100 rows where I need this to happen as well. Sorry if this is confusing. If anything needs to be clarified let me know thanks!
jeeped say:
This proposed solution may be an option for you is you only want 100 rows worth of the formula. See below for calculation lag caveat¹.

If you use INDIRECT() to note the cell reference, you will be supplying a text value that will not change when columns and/or rows are inserted or deleted. From your example, in C3 try,

=INDIRECT("F3")

... or to provide the capability to fill the formula down,

=INDIRECT("F"&ROW(3:3))

This latter version will reference F4, F5, etc when filled down.

¹Caveat - INDIRECT() is a volatile function. This means that it will recalculate whenever any value in your workbook changes, not just when a value that it references changes. For only 100 rows, I do not see a problem, but expect calculation lags if you run many times more than that.
