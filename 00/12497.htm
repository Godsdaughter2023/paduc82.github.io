The following link has a document with four similar fields using dropdown forms fields. Add whatever identifying text you want to the document. Select the lot then press ALT+F3 to save as an autotext entry in the normal template.

http://dl.dropbox.com/u/57986755/Doc1.docx

Without turning this into a major programming exercise, you may be able to achieve much of what you want with the aforemetioned building block and two simple macros.

http://www.gmayor.com/installing_macro.htm

The first macro adds the building block at the end of the document and you can then add yoyur scores. The second macro adds those scores to the next line of  an Excel worksheet
"D:\My Documents\Test\Merge\WorkbookName.xlsx" starting at column A. You can change the path and filename in the macro as you wish.


Sub AddScores()
Dim oRng As Range
Dim i As Long
ActiveDocument.Range.InsertParagraphAfter
Set oRng = ActiveDocument.Range
oRng.Start = ActiveDocument.Range.End
oRng.InsertBreak wdSectionBreakContinuous
oRng.Start = ActiveDocument.Range.End
NormalTemplate.BuildingBlockEntries("Scores").Insert _
        Where:=oRng

For i = 1 To ActiveDocument.Sections.Count - 1
    ActiveDocument.Sections(i).ProtectedForForms = False
Next i
ActiveDocument.Sections(ActiveDocument.Sections.Count).ProtectedForForms = True
If ActiveDocument.ProtectionType = wdNoProtection Then
    ActiveDocument.Protect wdAllowOnlyFormFields
End If
End Sub

Sub CopyScoresToExcel()
Dim xlApp As Object
Dim xlBook As Object
Dim xlsheet As Object
Dim LastRow As Long
Dim i As Integer
Dim Score1 As String
Dim Score2 As String
Dim Score3 As String
Dim Score4 As String
Const strWorkBookName As String = "D:\My Documents\Test\Merge\WorkbookName.xlsx"

On Error Resume Next
Set xlApp = GetObject(, "Excel.Application")
If Err Then
    Set xlApp = CreateObject("Excel.Application")
End If
On Error GoTo 0
Set xlBook = xlApp.Workbooks.Open(FileName:=strWorkBookName)
Set xlsheet = xlBook.Sheets(1)
LastRow = xlsheet.Cells(xlsheet.Rows.Count, 1).End(-4162).Row
With ActiveDocument.Sections(ActiveDocument.Sections.Count).Range
    Score1 = .FormFields(1).Result
    Score2 = .FormFields(2).Result
    Score3 = .FormFields(3).Result
    Score4 = .FormFields(4).Result
End With
xlsheet.Cells(LastRow + 1, 1) = Score1
xlsheet.Cells(LastRow + 1, 2) = Score2
xlsheet.Cells(LastRow + 1, 3) = Score3
xlsheet.Cells(LastRow + 1, 4) = Score4
xlBook.Save
xlBook.Close
Set xlApp = Nothing
Set xlBook = Nothing
Set xlsheet = Nothing
End Sub

You can insert a text form field in a table cell. If the height is set to Auto, then text will wrap naturally within the cell, and the cell will expand vertically as required.
