mike h.. say:

Hi,

 

Right click the sheet tab, view code and paste this in on the right.

 

I used the worksheet_activate event so you will see the message box every time you go to the particular sheet.

 

Private Sub Worksheet_Activate()
Dim lastrow As Long, x As Long
Dim Msg As String
Msg = "Expired cells" & Chr(10) & Chr(10)
lastrow = Cells(Rows.Count, "Q").End(xlUp).Row
For x = 2 To lastrow
    If Cells(x, "Q") > 5 Then
    Msg = Msg & Cells(x, "Q").Address & Chr(10)
    End If
Next
MsgBox Msg
End Sub
