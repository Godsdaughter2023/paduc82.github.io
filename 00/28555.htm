Well, perhaps a little macro would help you.  Copy the code below and paste it into a regular code module (it could be in the same workbook you are working with, or in another you'd open just to use the code while working with another workbook).

 

The Process:

Select the cell you want to place the picture into

Insert the picture

The code to copy and paste into a workbook (starts at   Sub...   ends at   End Sub)

 

Sub SizePictureToCell()
  Selection.ShapeRange.LockAspectRatio = msoFalse
  Selection.Left = ActiveCell.Left
  Selection.Top = ActiveCell.Top
  Selection.Height = ActiveCell.Height
  Selection.Width = ActiveCell.Width
End Sub

 

Note that "Selection" is expecting to be the picture itself.

Let me condense the significance of the protocol used from the article link previously provided

POP3 - One way from server to device - <Inbox> folder only. Mutliple devices can download the same <Inbox> contents providing every other device has the option unchecked for "Leave messages on server"

Absolutely nothing else gets downloaded - nothing is "sync'd" which by definition implies bi-directional communication

IMAP - email folders only - bi-directional server<->device sync of email folders

EAS (Exchange ActiveSync) & Full Exchange - full sync of all folders

Have no doubt what you were told but likely the person providing that info had an Outllok.com EAS account in mind (Outlook.com accounts can be configured as POP, IMAP & EAS)
