There are 3 possible XLSTART folders to look in.
They vary by Excel version and Windows version.
They can be located by opening Excel, Alt+F11 to the visual basic editor, Insert > Module, paste in the following code and run the ShowXLSTARTFiles macro:

Sub ShowFiles(Path As String)
  Dim stFile As String
  stFile = Dir(Path & "\*.*")
  Do Until stFile = ""
    Debug.Print Path & "\" & stFile
    stFile = Dir()
  Loop
End Sub

Sub ShowXLSTARTFiles()
  ShowFiles Application.StartupPath
  If Application.AltStartupPath <> "" Then ShowFiles Application.AltStartupPath
  ShowFiles Application.Path & "\XLSTART"
  Debug.Print "Done"
End Sub

In the immediate window (Ctrl+G if it isn't visible) you will see listed any files in any of the 3 folders.

If there are none it will just show "Done"

 

In the absence of any files in XLSTART, the error may be coming from an installed add-in.
Go to Options > Add-ins and remove any active add-ins (Manage: Excel Add-Ins > Go)
Then add them back one at a time until you find the culprit.

people/ contacts - Microsoft Community

dennisfecci say:

people/ contacts

I have 200 contacts on my PC running Windows 10 and I can't figure out how to move them to People. How can I do this?

parmi mehta say: Hello Dennis,

My name is Parmi, I am an Independent Advisor. Thank you for posting your question. I'll be happy to help.

I am assuming you need to export contacts from windows 10 mail app and want to import them to Outlook people ?

Please follow below to export contacts from Windows 10 mail app

Export Contacts and Email Addresses From Windows Mail
To save your Windows Mail 8 and earlier contacts to a CSV file:

Select Tools | Windows Contacts from the menu in Windows Mail.
Click Export in the toolbar.
Make sure CSV (Comma Separated Values) is highlighted.
Click Export.
Select Browse.
Pick a folder to receive the exported contacts or open a new folder.
Type a folder name such as "Windows Mail contacts" under the File name.
Click Save.
Click Next >.
Make sure all address book fields you want to be included are checked. Note that Windows Mail does not export first and last names separately, even though there are First Name and Last Name fields. Choose Name instead.
Click Finish.
Now click OK.
Click Close.

To Import to outlook people, sign-in to this using your Microsoft account you want to import to.

https://outlook.live.com/owa/?path=/people

From there, you go to "Manage" \ "Export Contacts" and you can export them all to a csv file. If you then go back into your Outlook application on your computer, it is a simple matter to import this file.


Please let me know if, you have any other queries.
