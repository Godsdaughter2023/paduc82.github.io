The code is pretty straight forward:

Private Sub CmdButton3_Click()

End Sub

I am assuming that it is something to do with the file extension, but idk.

Bonus Question: Is there a way to use this so that the Active workbook doesn't become that new saved file? (Act more like an export I guess)

hansv mvp say:

GetSaveAsFileName only prompts the user for a filename, but it doesn't actually save anything. You have to do that in the code:

Private Sub CmdButton3_Click()
    Dim FileName As String
    FileName = Application.GetSaveAsFilename(FileFilter:="Excel Files (*.csv), *.csv")
    If FileName = "False" Then
        MsgBox "No filename specified!", vbExclamation
    Else
        ActiveSheet.Copy
        Application.DisplayAlerts = False
        ActiveWorkbook.SaveAs FileName:=FileName, FileFormat:=xlCSV
        ActiveWorkbook.Close SaveChanges:=False
        Application.DisplayAlerts = True
    End If
End Sub
