dhookom say:

If it's not possible to combine the data into a single table, you can combine them using a union query with SQL like:

SELECT #1-Jan-2012# as TheDate, Customer, Days, Product, Amount
FROM Data201201
UNION ALL
SELECT #1-Feb-2012#, Customer, Days, Product, Amount
FROM Data201202
UNION ALL
--- etc ---

 

You can the build a crosstab query based on the union query and the Legend table. Set the criteria under the Days column to Between MinDays and MaxDays. Use the LabelUsed as the Row Heading, format TheDate as the Column Heading, and sum of Amount as the value.

Hi.  Excel, (and Solver) can not work per say at 2 decimal precision.

They are both machine precision numbers.  What you are seeing is just the display.

If interested...

Your model is more complex, but If you have 3 cells   x+y+z = 1

x=.142857...

y=.428571...

z= ?

Excel has to guess for z an infinite number of values to trigger True (x+y+z=1)

Even if it guessed close, and got 0.99987, this is still false at machine precision, and Solver will continue.

It will use a derivative method to get close, but you run the risk of the values not summing to 1.

The "Technique" I would use would be to remove the =1 constraint.

Replace the z cell  with =1-x-y

Changing cells are now just 2 (x & y)

The constraints are:

{x,y,z} >= 0

{x,y,z} <= 1

This will make the problem mush easier, and x+y+z = 1 with much higher precision.

Hi,

Error 0x803D0008 could occur if the user's Kerberos token is too large. This can occur if the user is a member of a large number of groups ("nested" groups can be a major contributor to this problem).

Try removing the user from some groups to see if this resolves the problem.

Resources:

Problems with Kerberos authentication when a user belongs to many groups
https://support.microsoft.com/en-us/kb/327825

There is also this KB that references header size/token size.

Enterprise single sign-on users in Office 365 can't sign in to Skype for Business Online from inside their corporate network
https://support.microsoft.com/en-us/kb/2839539
