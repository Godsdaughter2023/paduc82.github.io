If you are using Exchange Server, it has a feature that lets Exchange 2003 operate without the Message Transfer Agent (MTA). If a message was sent incorrectly by using the MTA route, this delivery status notification is returned to the sender.

Note: Although Exchange 2003 can operate without the MTA, we do not recommend or support this configuration.

To turn on this feature and to prevent messages from queuing to the MTA, follow these steps:

  1. Disable the MTA service.
  2. Set the DWORD value to 0 in the following registry subkeys for every information store database and public folder store:
    HKLM\System\CurrentControlSet\Services\MsExchangeIS\<Server Name>\<PrivMDB_GUID>\Gateway In Threads
    HKLM\System\CurrentControlSet\Services\MsExchangeIS\<Server Name>\<PrivMDB_GUID>\Gateway Out Threads
    Note: When you do this, you make available the store resources that are associated with MTA delivery.
  3. Restart the information store.

To troubleshoot this, check your routing topology. Use the WinRoute tool to make sure that the routes are replicated correctly between servers and routing groups.

2. If you cannot get it on I would suggest to alternate a few times with the powerbutton and power charger - This is just a shot in the dark and will probably not work, but you might be lucky. - It's hard to tell what the actual problems are.

3. Hard reset it. Press and hold these three keys while powering on: Green/call button + * (star key) + 3 (number 3). Hold them keys down til youre past the Nokia handshake logo screen. Note that the hard reset will wipe your phonememory and contacts, messages, calendar notes etc will be deleted.

If you get it on: I will suggest you to reinstall fw. And reset the phone and format the memory card anyway.

If your phone still has problems you will need to take it to a Nokia care point. - You can find such information by clicking the "contact us" link on top of this page.

Remember that a format of the memory card will delete everything that's on it, so back up if possible. It's the same with the phonememory (hard/soft reset) as i mentioned.

If you get the phone to work normally, then you should keep an eye on what you install, and what you do to it. That way you will more likely be able to know what caused the issue. - But it could be a hardware issue with the phone it self. If youre not able to reinstall the firmware - a Nokia care point can help you.
