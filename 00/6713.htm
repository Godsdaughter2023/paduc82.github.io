billgrice say:

Build 10525 ATI graphic drivers.

If you have just updated to build 10525 and have an older ATI graphics card you may experience driver problems but there is a work around. If you go to device manager and click display adapters and then your device and see a dialog saying there are no drivers installed for device this is what you can do to solve the issue.
Before you follow this proceadure make sure automatic driver updates are disabled or windows will update to a non working driver again.

1. Go to your control panel programs and if you have AMD catalyst install manager installed uninstall it and reboot it is important you do this.
2. Go to the AMD website and go to drivers and support and search for your card and select Win 8 as your operating system 64 bit or 32 bit depending on what Windows 10 system you have. Download the installer and you should have the 13- 4- legacy_vista_win7_win8_64_dd_ccc in the case of 32 bit systems it will be _32_dd_ccc
3. Once you have downloaded the driver software copy it to your desktop and rightclick to run the installation wait for it to unpack then close the installation once the catalyst installation screen appears.
4. Go to this pc and select your c drive there should now be a folder AMD open the folder and navigate through the sub folders untill you find the 13-1 legacy folder, open this folder and go to the packages folder and drivers folder then display you should find a folder called W86A_INF you will find 3 INF files right click the first one and open with either notepad or editor and scroll to the botom for the list of supported devices, if your device is not there open the next INF file and repeat the proceadure untill you have found the INF file with your device and make a note of it location for later.
5. Right click this PC open properties, open your device manager and select your device and open the driver menu and update driver use the option browse my computer for driver software. Now open the Let me pick from a list of device drivers  on my computer. in the dialouge select Have Disk, now use browse to navigate to your INF file in your AMD folder. Doublle cick the INF file you need and the driver will install, once it as installed return to your device manager and check your device you should now see This device is working properly.
6. If you want to install the catalyst software re run the installer when asked do you want to install drivers select do not install. once your installation as finished reboot.

I would also sudjest you keep your catalyst installer some were safe in case you need it again in the near future.

ron sommer mvp 2009-2017 say:

Emails are only sent once.

When you send using the email app, the email app is saving a copy of the sent message and the server is saving a copy.

The solution is to not have the app save a copy of sent messages.

The problem is the save a copy setting applies to all accounts setup in the app.

vega hassan say:

check this link: https://support.office.com/en-us/article/Recover-deleted-items-in-Outlook-2010-cd9dfe12-8e8c-4a21-bbbf-4bd103a3f1fe

note: 

If you can't find an item in the Deleted Items folder, the next place to look is the Recoverable Items folder. This is a hidden folder, and it's the place where items are moved when you do one of the following things:

  • Delete an item from the Deleted Items folder.

  • Empty the Deleted Items folder.

  • Permanently delete an item by selecting it and pressing Shift+Delete.
