Try the below formula in cell C1 and copy across as required. Please note that this is an array formula. You create array formulas in the same way that you create other formulas, except you press CTRL+SHIFT+ENTER to enter the formula. If successful in 'Formula Bar' you can notice the curly braces at both ends like "{=<formula>}"

=IF(COUNTIF(Sheet2!$B$1:$B$1000,$B1)<COLUMN(A1),"",INDEX(Sheet2!$A$1:$A$1000,SMALL(IF(Sheet2!$B$1:$B$1000=$B1,ROW($A$1:$A$1000)),COLUMN(A1))))
