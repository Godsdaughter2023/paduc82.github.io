    rick rothstein, mvp - excel say:

    If you wanted a macro to do this with, try this one...

    Sub AddSevenPointFivePercent()
      Dim NumberRange As Range, Cell As Range
      Set NumberRange = Selection.SpecialCells(xlCellTypeConstants, xlNumbers)
      For Each Cell In NumberRange
       Cell.Value = 1.075 * Cell.Value
      Next
    End Sub

    Note: This macro operates on a selection of cells (that selection can contain numbers, text, formulas, etc.) and it will only process the constant numbers within that selection. So, to use the macro for the first time on a worksheet, simply click the Select All button (the blank square where the row number headers and the column letter headers meet) first and then run the macro. I set the macro up this way so you can add new data to the worksheet in the future and then change only those newly added rows of number by selecting only those newly added rows of data. If you will NEVER be adding new data to the worksheet that will need to be processed after you run the macro, you can use this version of the macro instead (that way, you will not have to select anything)...

    Sub AddSevenPointFivePercent()
      Dim NumberRange As Range, Cell As Range
      Set NumberRange = Cells.SpecialCells(xlCellTypeConstants, xlNumbers)
      For Each Cell In NumberRange
       Cell.Value = 1.075 * Cell.Value
      Next
    End Sub

    Oh, I guess I should mention how to install the macro just in case you have not done this before. From any Excel worksheet, press Alt+F11 to go into the VB editor, then click Insert/Module on the VB editior's menu bar and then copy/paste the macro into the code window that appears.
