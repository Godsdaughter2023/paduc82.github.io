The Network Access Protection (NAP) agent service collects and manages health information for client computers on a network. It is mostly for in case the computer needs to access a corporate network domain.

You can follow the steps to enable ‘Network Access Protection Agent’ service.

a. Press Windows key + R, type in services.msc and press Enter.

b. Check for the ‘Network Access Protection Agent’ service name.

c. Right-click and select properties.

d. And in the Start-up type select Automatic.

e. Under service status option click the ‘start’ button and then click on Apply and then OK.
