a. user say:

The manufacturer of your computer has updates for your BIOS on their technical support website, as well as detailed instructions on how to perform the update. Just remember to install the update for your specific model number.

The BIOS and Windows are two separate things. The BIOS comes from the computer manufacturer (actually, the motherboard manufacturer, but that's just a detail) and Windows comes from Microsoft. However, one of the things that the BIOS does is to pass on to Windows a list of the computer's hardware, which Windows uses to install drivers.

BIOS updates sometimes improve compatibility with newer version of Windows 10 and expose your system to newer functionality unique to the version of the operating system. It could be improved security, better audio, graphics etc. 

BIOS updates can also improve stability in order to support new hardware such as add on cards for graphics, Ethernet etc.
