I just recently bought a vista 64 bit machine and moved files from my XP machine.   Lots of the files now show as .db files  or thumbnail.db filesand say I cannot open them.   What are these db files and how can I open them?
anthony - support engineer say:  

Hello gary1346,

 

Thank you for visiting the Microsoft Windows Vista Community site.

 

Thumbs.db is a file existed and used in Windows XP, Windows Server 2003, and Mac OS X to store thumbnails cache of various graphics, pictures, photos or some documents and files such as those with JPEG, BMP, GIF, TIF, PDF and HTM extension when these files are viewed in Windows Explorer‚Äôs Thumbnail or Filmstrip view. Thumbnail preview of video files are also supported in Windows XP Media Center Edition, but the previews are stored in ehThumbs.db file. Thumbs.db database files are stored in each directory where thumbnails are created.

 

In Windows Vista , users will be unable to find Thumbs.db or ehThumbs.db files scatter around and place in many directories or folders again. Instead, Vista system will store all thumbnails and previews generated in a centralized thumbnail database . The centralized thumbnail database is located in the following folder:

 

\Users\[User Account Name]\AppData\Local\Microsoft\Windows\Explorer
