Yes, it's because the focus is in a text box (drawing object). Drawing objects are managed in a different layer of the document (one that supports "Office", rather than "Word") so some things don't work the same, or at all, when you're in a text box.

Test whether this works for you:

1. Right click the edge of the text box (the blue line that you don't see when you're not in the text box).

2. Select "Format text box" from the context menu (at the bottom)

3. In the dialog box, go to the tab "Text box".

4. Click "Convert to Frame"; click OK when you get a warning about losing functionality (after all, this is a test)

Frames are managed in the document layer, not the drawing layer. So you can insert a footnote into a frame. Frames are older than the current version of a "text box", and can't do some of the same things a text box can. But text will flow around a frame, so if that's all you're using a text box for, this should work for you.

The other possibility would be to use a Table with a single cell, rather than a text box.

With $A$2 in Cell B2 you cluld use a formula like this

=indirect(B2)

Note that this formula is volatile and has high calculation overhead. A better solution might be to put the number 2 in Cell C2 and use a formula such as this

=index($A$1:$A$1000, C2)
