elbab say: Hi there!

If you add a phone number to a contact, you can call their mobile or landline or send them text messages (SMS) from Skype. You'll need a little Skype Credit (for pay as you go calls and sending SMS) or a subscription.

To add a phone number to a contact

In your contact list, select the contact you’d like to add a phone number to.
In the chat window, click their profile picture.
In the contact profile, click Add Number.
Select the number type (Mobile, Home, Office, Other), choose a country from the drop-down, enter a valid phone number and then select tick icon to add the number to the contact.
Now you have the option to call the contact on Skype or on their phone.

Footers are in section breaks. The last footer is in the paragraph mark at the end of the document. Before you delete the section mark before page 13, copy your first footer into it if the text (not the page number) is different.
