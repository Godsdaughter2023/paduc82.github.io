Hello James,

Stuttering may cause by a lot of reasons. Anything from drivers, software, or hardware problem. With that, I would like to know more information regarding the issue so that we can isolate it:

  • What is the make and model of your device?
  • When did this issue start?
  • Did you make any changes prior to this issue?
  • Is this also happening even if the computer is in idle?
  • Are you using a third party antivirus?
  • Is it still the same after performing a hard reboot? (Pressing the power button for 10 seconds or until the computer is completely off, then turn it back on after a few seconds)
  • Have you checked the memory usage in Task Manager if it is spiking when the issue is happening?

It's not obvious what exactly the cause of the issue. Our initial troubleshooting step is to perform a Windows update. There are a lot of updates that was released to fix common incompatibility issues with Windows 10. To perform a manual Windows update, go to Start > Settings > Update & Security > Windows Update and click the Check for updates button.

I would also recommend that you download and install all latest drivers for Windows 10 from the manufacturer's website.

If the problem still persists, the other option is to perform a clean boot. A clean boot is to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you install a program or an update or when you run a program in Windows 10:

  1. Press Windows Key, type msconfig in the search box, and hit Enter.
  2. On the Services tab of the System Configuration dialog box, tap or click to select the Hide all Microsoft services check box, and then click Disable all.
  3. On the Startup tab of the System Configuration dialog box, tap or click Open Task Manager.
  4. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.
  5. Close Task Manager.
  6. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

If a service is causing issues, I suggest you turn on 5 services at a time, reboot and then see if the problem reappears, keep repeating this until you find a group of 5 services, which are causing this issue, then check the one by one, till you find the one which is causing your issue. Perform the same divide and apply for Startup's too.

You can also refer to this article on how to perform a clean boot.

Note: Once done with the troubleshooting, refer to How to reset the computer to start normally after clean boot troubleshooting from the same article.

Let us know how it goes.
