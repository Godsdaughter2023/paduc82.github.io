#VALUE! is still returned. I have the most recent update of Office 2011 for Mac installed. Can anyone help with this, or is it just a bug that I should report to Microsoft? It would be helpful to know if it occurs for others as well.
j.e. mcgimpsey say: Are your System Preferences set to m/d/y  or d/m/y? XL takes its default date input format from System Preferences.

DATEVALUE() will return #VALUE! with that string if System Preferences are set to d/m/y, since there's no 22nd month. If that's the problem then it should work fine with =DATEVALUE("22/8/2008").

Note that if you change System Preferences you'll need to close and restart XL, since it only reads the format on startup.

How do I make a template with a different watermark on all 3 pages and - Microsoft Community

nickt303 say:

How do I make a template with a different watermark on all 3 pages and merge fields on all pages.

I need to make a template which has a different watermark on each page along with merge fields on all  3 pages.

jay freedman say:
As long as the document is never more than three pages long, you can do it easily. More than three pages will require a different approach.

The first thing you need to know is that a watermark in Word is really just a picture or a WordArt object whose anchor is in the header of the page, and usually washed out (with low contrast and high brightness) so it doesn't hide the text of the main document. The Watermark command also adds a special hidden bookmark, so it can delete any existing watermark when you add a new one -- which you don't want it to do -- and it adds the same watermark to all the pages of a document, which you also don't want. So avoid the Watermark command and just add the picture or WordArt as you would any other graphic.

First, if the document or template doesn't already have all three pages, insert manual page breaks (Ctrl+Enter) to create the extra pages.

Next, right-click the header area and click Edit Header. On the Header & Footer Tools ribbon, check both the boxes for "Different First Page" and "Different Odd & Even Pages". This creates a separate header on each page, labeled "First Page Header", "Even Page Header", and "Odd Page Header". You can use the Next and Previous buttons on that ribbon to move from one header to another.

With the cursor in the First Page Header, use the Insert > Picture command or the Insert > WordArt command to make the "watermark" for the first page. Set its Text Wrap to Behind Text, and change its size, color, outline, rotation, etc. Drag it into the middle of the page.

Move the cursor to the Even Page Header, and create the "watermark" for the second page in the same way. Then move to the Odd Page Header and create the "watermark" for the third page. Finally, close the header pane and save.

'=========>>
Option Explicit

'--------->>
Public Sub Tester()
    Dim Rng As Range, Rng2 As Range

    With ActiveSheet
        Set Rng = Intersect(.UsedRange, .Rows(2)).Offset(0, 1)
    End With

    On Error Resume Next
    Set Rng2 = Rng.SpecialCells(xlCellTypeBlanks)
    On Error GoTo 0

    If Not Rng2 Is Nothing Then
        With Rng2
            .Value = .Offset(0, -1).Value
        End With
    End If
End Sub
