a. user say:

Hi Simon,

 

I understand that you are facing an issue where the additional clock displays wrong time.

 

Usually if the Daylight Savings time is observed in a particular time zone, the additional clock will automatically show you the correct time and will use the Daylight time instead of showing the standard time.

 

For Example: California observes Pacific Standard Time and when daylight is observed the clock is automatically set to use the daylight standard time. Though the time zone will reflect as GMT -8.

 

If the time is not displaying correctly, I suggest you to ensure that the time zone selected is PST.

 

You may also follow the steps below to synchronize the time with the server to make sure that the time displayed is correct:

 

  1. Press Windows key + R
  2. Type timedate.cpl
  3. Select Internet Time
  4. Click on Change settings
  5. Make sure that the server name is time.windows.com and select Update now
