jennifer bri say:

Hi McPherson,

The error code indicates that the camera app is unable to create the file, this usually indicates that there is a permissions issue.

We can try the methods below to check which one will resolve the issue:

Method 1: Hardware and Devices troubleshooter

  • Type troubleshooting in the search bar then select Troubleshooting from the results.
  • Select View All on the left side.
  • Click on Hardware and Devices.

  • Follow the on-screen instructions to run the troubleshooter.

  • Check if issue persists.

Method 2: Update the camera driver from Device Manager manually

  • Press (Windows key + X) and click on “Device Manager”.

  • Expand “Other devices”.

  • Right click on the “Integrated webcam” drivers and select “Update Driver Software”. Once this is done, restart the computer and check if the issue persists.

Let us know if the methods above worked for you.
