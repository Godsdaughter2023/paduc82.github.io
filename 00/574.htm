Select cell A1 and add your validation list.

Add this code to the worksheet code module:

Private Sub Worksheet_Change(ByVal Target As Range)

    If Not Intersect(Target, [A1]) Is Nothing Then
        Application.EnableEvents = False
        If [A1].Value = "Other..." Then
            Application.Dialogs(xlDialogFunctionWizard).Show
        Else
            [A1].Formula = "=" & [A1].Value & "($B$1:$B$10)"
        End If
        Application.EnableEvents = True
    End If

End Sub

The canned functions will operate on B1:B10.

Add this code to the worksheet code module:

Private Sub Worksheet_Change(ByVal Target As Range)

    Const RangeReference = "$B$1:$B$10"
   
    Dim Value As Variant

    If Not Intersect(Target, [A1]) Is Nothing Then
        Application.EnableEvents = False
        Select Case [A1].Value
            Case "None"
                Value = vbNullString
            Case "Average"
                Value = "=AVERAGE(^RangeReference)"
            Case "Count"
                Value = "=COUNT(^RangeReference)"
            Case "Count Numbers"
                Value = "=SUMPRODUCT(ISNUMBER(^RangeReference)*1)"
            Case "Max"
                Value = "=MAX(^RangeReference)"
            Case "Min"
                Value = "=MIN(^RangeReference)"
            Case "Sum"
                Value = "=AVERAGE(^RangeReference)"
            Case "StdDev"
                Value = "=STDEV.P(^RangeReference)"
            Case "Var"
                Value = "=VAR(^RangeReference)"
            Case "Other Functions...":
                Application.Dialogs(xlDialogFunctionWizard).Show
        End Select
        If VarType(Value) = vbString Then
            [A1].Formula = Replace(Value, "^RangeReference", RangeReference)
        ElseIf Not IsEmpty(Value) Then
            [A1].Value = Value
        End If
        Application.EnableEvents = True
    End If

End Sub

jay freedman say:
Yes, it's quite easy.

In the equation content control, start by typing
   f(x) =

On the Equation Tools ribbon, click the Bracket button and choose the single brace in the bottom left corner of the gallery. That inserts a one-line brace and a placeholder -- but the brace will expand to fit the depth of anything you put into the placeholder.

With the placeholder selected, click the Matrix button on the ribbon and choose the two-cell vertical arrangement. Select the top cell and enter the
   λexp(-λx), x ≥ 0
part of the first equation. Then select the bottom cell and enter the
   0, x < 0
part of the second equation.

By default, the two cells will be aligned on their centers. If you want both to be left-aligned at the brace, right-click them and choose Column Alignment | Left.

Hi. For Linux, you can get decent support and performance on Linux Mint 14.

 

Download it here: http://www.linuxmint.com/

 

Just install it like you normally would for other distributions. Namely:

 

1) Burn the ISO to a DVD, or copy it to a bootable USB drive. If you are on Windows, you can use RUFUS to make a bootable USB drive.

2) Boot using the DVD/USB drive.

3) Once you are in, open the "Install Linux Mint" application and follow instructions.

 

After installation, there are a few more things you need to do in order to make things work:

 

1) Fix suspend/sleep:

Open the terminal and enter this command:
sudo mv /usr/lib/pm-utils/sleep.d/99video /usr/lib/pm-utils/99video

 

2) Fix brightness:

Open the terminal and enter this command:
sudo pluma /etc/default/grub

Once the text editor has come up, look for the line that reads:

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash"
and then add "acpi_backlight=vendor" to it. So it'll read like this:

GRUB_CMDLINE_LINUX_DEFAULT="quiet splash acpi_backlight=vendor"

Save and exit, then go back to the terminal again and enter:

update-grub

And then restart the computer. It should work.

 

3) Optional, but if you would like to apply color correction:

Enter this command into the terminal
xrandr --output LVDS-0 --gamma 1.1:1.12:0.78

Play around with the values (1.0), (1.12), and (0.78) to better fit your display. The values are gamma values for Red, Green, and Blue accordingly.

You can also apply this on boot by adding a new entry into "Startup Applications" in "Control Center"

 

And that's it. Flash doesn't work as well as on Windows (fullscreen lags and stutters), videos in general are dreadful, and there is no hardware acceleration. But it works far better than other Linux distributions, and it requires very minimal tweaks and modifications.
