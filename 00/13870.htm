vinay, india say: A possible approach could be to use data validation.

Assuming your data is in Columns A, B and C and it is without any headers.
On Cell B1, Add a DATA VALIDATION Condition (DATA MENU > DATA VALIDATION).
Select Custom and write the following formula:

=COUNTIFS($A$1:$A$10,$A1,$C$1:$C$10,$C1)>1

Click on the Error Alert and write the message: Duplicate Payment warning.

You may select STOP or WARNING type alert which would decide if the user is allowed or not to make the entry after a message is displayed.

To copy the validation to all rows, copy the cell and do a PASTE SPECIAL > VALIDATION over the remaining cells. You may feel the need to copy this validation to the cells in Column A and C also.

An alternative approach could be to have a helper column like say Column D with the following formula in D1.

=IF(COUNTIFS($A$1:$A$10,$A1,$C$1:$C$10,$C1)>1,"Duplicate Entry detected in Row " & MAX(N($C$1:$C$10=C1)*N($A$1:$A$10=A1)*N(ROW($A$1:$A$10)<>ROW(A1))*N(ROW($A$1:$A$10))),"")

This needs to be entered as an ARRAY FORMULA using CTRL+SHIFT+ENTER instead of normal enter.
