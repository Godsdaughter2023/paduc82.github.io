a. user say:

Hi Pete,

 

Important: It is recommended that you do not configure Windows 8.1 to auto login if you're using a mobile computer like a tablet, laptop, etc. Requiring a password to access Windows 8.1 is a simple and reasonably effective way of keeping your data secure if your computer ends up in someone else's possession.

 

If you're sure you want to do this, just follow the steps below to make Windows 8.1 automatically logon to your account.

 

a. Press Windows key + R and type netplwiz in the run box and hit enter.

 

b. The Advanced User Accounts program will open.

 

c. On the Users tab, which should be where you're at now, uncheck the box next to Users must enter a user name and password to use this computer.

 

d. Tap or click the OK button at the bottom of the window.

 

e. When the Automatically sign in box appears, enter the user name you wish to automatically login to Windows 8.1 with.

 

Important: If you're using a Microsoft account, be sure to enter the entire email address that you use to sign in to Windows 8.1 within the User name field. What defaults there might instead be the name associated with your account, not your actual user name.

 

f.  In the Password and Confirm Password fields, enter the password you sign in to Windows 8.1 with.

 

g. Tap or click the OK button.

 

The Automatically sign in window, as well as the User Accounts window, will now close.

 

h. Restart your computer and make sure that Windows 8.1 automatically logs you in.

 

Also, refer to the link to know more about system restore:

 

How to refresh, reset, or restore your PC

http://windows.microsoft.com/en-US/windows-8/restore-refresh-reset-pc

 

Note: Ignore the Reset and Refresh options from the article.

 

Note: When you perform System Restore to restore the computer to previous state, programs and updates you installed are removed.
