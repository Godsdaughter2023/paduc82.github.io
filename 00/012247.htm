Hi,

 

There is an issue with Adobe Reader on 64-bit IE10. Here are two solutions:

 

1. Install Foxit reader (free) http://www.foxitsoftware.com/pdf/rd_intro.php and make it the defalt program for opening PDFs. You will still be able to right-click a local PDF file and open with Adobe Reader if you wish.

 

2. This registry edit has resolved the issue for many users.

 

Start> Run> type in REGEDIT and hit Enter.

By clicking in the [+} drill down to

HKEY_CURRENT_USER

  Software

     Microsoft

       Internet Explorer

         Main

With Main selected in the left pane, right-click in the right pane and choose New> DWORD (32-bit)

Name the new DWORD Value TabProcGrowth then right-click it and set the value to 1.

If TabProcGrowth already exists (you didn't have to create it) just change the value from 0 to 1.

AFTER THE HARD RESET
