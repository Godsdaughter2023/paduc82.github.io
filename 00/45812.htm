Dear Joe,

Thank you for contacting Seagate support.

Please refer to the DISKPART procedures below to Clean the disk, initialize it as MBR, then create new volume/partition NTFS.

***All data on the hard drive will be erased during this procedure ***
*** DO backup your important data or do self data recovery before proceeding ***

Firstly, please check what number the drive is detected as in your list of disks in your Disk Management utility.

To access Disk Management, right click on My Computer / This PC, select Manage, and then select Disk Management on the left column.

Within Disk Management this drive should show in your list of disks in the lower portion (Disk 1,2 3 etc etc). Make sure you get the correct Disk number, Right click the Disk, select Properties to verify.

Take note of the Disk number for the drive. You will need it shortly.

1. Go to Command Prompt:
-In Windows Vista/7/8/10, click the Windows icon and then type cmd in the search field, Right click the cmd.exe, click Run As Administrator.
-In Windows XP, click the Start menu, select Run, then type cmd and press Enter.

2. In the command line, type:
diskpart
and press Enter.

3. DISKPART command line appear, type:
list disk
and press Enter to show all disks in this machine.

4. Figure out which disk you want to erase - it will be identified with a number.
Then type:
select disk X
where “X” is the number of the disk you want to erase. This will set the focus to the specified drive.
For example:
select disk 2

5. Type:
clean
and press Enter to erase the disk.

6. Go back to Disk Management. You should see that the disk is "Unallocated" / "RAW" with black color.
- If it asks you to Initialize the disk, click Ok.
- If there is no message, Right-click on the "Not initialized" disk, select "Initialize Disk"
Choose MBR
Then the drive should be ready to partition and format.

7. Now right-click on the "Unallocated" area, select Format / New Simple Volume

Agree with everything by default value until you see this:

Default
NTFS
New Volume = (name the drive whatever you like)

how do i get rid of a pop up - Sttray.Exe-entry point not found - Microsoft Community

lewisday say:

how do i get rid of a pop up - Sttray.Exe-entry point not found

when I start my computer I get the following pop up : Sttray.Exe-entry point not found. The procedure entry point localize string could not be located in the dynamic link library stlang.dll. Please how do I get rid of this pop up, and what does it mean ? Thank you. ps my computer is a Dell 9400 Inspiron, and I run Windows XP.
kosh vorlon say:

Hi Lewisday,

 

I do not believe this is a virus or infection problem - it is a startup failure notice and not an infection type of popup.  I believe you're having a problem with loading the Sigmatel Audio Driver.  Go to Device Manager and check to see if it shows any problems with the driver.  Go to Control Panel / Sound and verify you can configure everything as you should be able to do so - if it isn't loading I'd expect you to be having some type of audio problem.

 

Here's info about sttray.exe: http://www.what-is-exe.com/filenames/sttray-exe.html.

 

I also happen to have a Dell 9400/E1705 Laptop and this applies to the Sigmatel Audio Driver.  You should check at Dell by your Service Tag number, but will probably end up here: http://www.dell.com/support/drivers/us/en/19/DriverDetails/DriverFileFormats?DriverId=NN9X1&FileId=2940571307&productCode=inspiron-9400&urlProductCode=False.

 

It runs at Startup for me.  It seems that you're having trouble with the driver.  I recommend you download and update the driver on system and reboot and that may well solve your problem.  You'd do so using Device Manager.

 

Without this, you may have problems with your Sound (maybe not completely - but possibly in terms of getting the best quality).  I've never tried without it and I have mine upgraded for Vista now.

 

If you would like to see what happens if you simply don't run this application at startup (which will remove the popup as well) and since it most likely already isn't running anyway and yet you don't mention any sound problems (maybe you simply don't use anything but the computers basic audio that perhaps doesn't need that - I'm not certain), then you can go to Start / Run and type msconfig.exe and enter.  Go to the Startup tap and uncheck the box next to Sigmatel or sttray.exe, save the change and reboot.  I'll bet the notice is gone - but do you notice any changes to your audio?  If not, then maybe you can live without it if the above didn't resolve things.

 

In any event, it's not a virus thing, removing it from startup will remove the popup, and updating the driver may resolve the problem and allow the functionality to remain (if you notice or care).

 

I hope this helps.

 

Good luck!

 

Kosh
