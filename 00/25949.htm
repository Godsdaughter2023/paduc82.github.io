If the UPN and the Primary SMTP address don't match, it can cause some issues.

bob jones aka: cybertaz say:

Another variation not offered in the conversation Daniel provided the link to because the poster was convinced that modifying the document body content & text flow was a better option than using a simple keystroke :-)

At the spot where you leave off...

  1. Use Shift+Command+F5 to open the Bookmark dialog
  2. Type a simple Bookmark name [e.g., last]
  3. Press return or click the Add button
  4. Save & Close the document

When you reopen the document...

  1. Use the same keystroke to open the Bookmark dialog
  2. Double-click your Bookmark name (among several other ways)
  3. Delete the Bookmark & Close the dialog window
  4. Reset the Bookmark elsewhere when/if necessary

Note that if using a laptop or condensed keyboard you may have to include the fn modifier key or change your OS X Keyboard settings or assign a different keystroke in Word to the EditBookmark command. That command can be added to a Toolbar as a button if you wish. Also, the processes can be automated with macros as Daniel suggested... especially useful if you consistently use the same Bookmark name in all documents.

charles kenyon say:
You are likely switching column settings in your newsletter. This creates a new section.

Sections / Headers and Footers in Microsoft Word 2007-2010

Page numbering is a creature of sections.

The default setting is to continue page numbering from the previous section. However if you set a section to begin at page 1, when a new section break is inserted, it can pick up that setting as well.

You need to find each section (there can be multiple sections on one page) and have them all set to continue page numbering.

If all else fails, with a copy of your document, remove all section breaks and then start reinserting them as needed.
