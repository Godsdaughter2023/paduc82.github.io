lemp say:
The Intel PRO/100 is an Ethernet network adapter.  It is not a WiFi adapter.  You need to buy one.

There is quite a variety of WiFi adapters available. 

The two most common WiFi standards for home use today are "Wireless G" and "Wireless N."  Wireless N is the newer standard and is considerably faster and longer range than Wireless G.  For these reasons, Wireless N adapter are likely to be more expensive.  If you have a 4G LTE hotspot and will be using it in an area where you will get a solid 4G connection, you probably ought to get a Wireless N adapter to make full use of the speed capability of your MiFi device (you better have an unlimited data plan, though).

The other choice you need to make is the packaging of the adapter.  Assuming that you're going to be using the MiFi with a laptop located close to the MiFi device, get an adapter that looks like a USB drive.  If you plan to use a desktop or tower computer, you might want a different type of device so that the antenna isn't buried between the back of the tower and the wall.

This is just an example, not a recommendation (but the reduced price looks like a good deal):  http://www.newegg.com/Product/Product.aspx?Item=N82E16833124428
