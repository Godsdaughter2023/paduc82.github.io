a. user say:

you can get a little more detail on what type of error you have.  by pressing the sync and eject buttons you can retrieve a 4 digit error code which is equivalent to the error code given when the console boots up and displays an error screen rather than not booting at all.  the error screen gives this code in the E## format.

to get your secondary code, hold sync, then press eject.  this will give you the first digit based on the number of flashing lights.  press eject a second time for the second digit, again for the 3rd, and again for the 4th.  pressing it again will return to the original regular red light pattern.  (on the older consoles, a number of light will light up 1-4, 4 is equal to a 0 in the code.  think of the 4 lights making a circle or 0.  newer console do not have the dual green-red leds, so they flash a particular number of times if i am not mistaken, rather than lighting up a certain number of ring sections.)

knowing this code can sometimes help narrow down the issue.  here is a list of all the flashing light error codes, including the E## code they correspond to, and the general issue related to the code:
