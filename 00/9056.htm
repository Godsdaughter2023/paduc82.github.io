How do i get all of my internet favorites transferred to my new - Microsoft Community

danielmalek say:

How do i get all of my internet favorites transferred to my new computer?

I would like to transfer my internet explorer favorites from my old computer with Windows XP to my new computer With windows 8.  How do I do this?

a. user say:

Hi, DanielMalek,

 

If you use Internet Explorer on more than one computer, you can export your favorites from one computer and import them to another.

To export favorites:

  1. Open Internet Explorer.

  2. Click the Favorites button.

  3. Click the arrow next to the Add to favorites button, and then select Import and export.

  4. In the Import/Export Settings dialog box, select Export to a file, and then click Next.

  5. Select Favorites (and any other settings) you'd like to export, and then click Next.

  6. Select the folders of favorites that you want to export, and then click Next.

  7. By default, Internet Explorer creates a file called bookmark.htm in your Documents folder. If you want to store the exported favorites to a place other than Documents (like a disc or a flash drive), click Browse, and then select a different drive. Click Save.

  8. Click Finish.

To import favorites:

  1. Open Internet Explorer.

  2. Click the Favorites button.

  3. Click the arrow next to the Add to favorites button, and then select Import and export.

  4. In the Import/Export Settings dialog box, select Import from a file, and then click Next.

  5. Select Favorites (and any other settings) you'd like to import, and then click Next

  6. By default, Internet Explorer imports from a file called bookmark.htm in your Documents folder. Click Next to import the default file, or tell Internet Explorer to import favorites from a difference place. Click Browse and select a file, or type a location and file name to import. Click Next.

only common being say:

Hello!

I found an anwser, not sure if it's working though, because the program (Trackpad++) that was "unsigned" doesnt work anymore.... but the watermark is gone:

[FIX]

Open elevated command prompt.

Enter this command.

Code:
bcdedit -set TESTSIGNING off
Reboot.


[EXPLANATION]
Windows since Windows Vista 64-bit, does not allow unsigned or test-signed kernel mode code (drivers) to be loaded at all.

In order for developers of drivers to be able to properly develop the driver, they need to rapidly load different versions of it into windows. This would require them to get each version signed by Microsoft. That's not reasonable, so Microsoft added the ability to allow loading of test-signed drivers through the use of "Test Mode".

End users don't have much use for this feature and it is a security risk.

The only use for an end user, is if they need to use a driver for some reason that hasn't been certified by Microsoft. Certification costs money, and some independent or amateur developers opt not to do it.

The text cannot be disabled or removed so long as you are in test mode. Only disabling test mode will remove it.
