don cuthbert say:

Why not try a clean install of Windows 8.1 Update to an empty partition and then upgrade that partition to Windows 10.  There may be some un-resolvable issue in your current install.  You can download the most recent ISO for Windows 8.1 Update for free.

http://windows.microsoft.com/en-US/windows-8/create-reset-refresh-media

To retrieve your original OEM Windows 8 product key, if you do not know it, use this command and record the key for use activating this new clean install of Windows 8.1 Update.

wmic path softwarelicensingservice get OA3xOriginalProductKey

A program called Produkey can also look up your current product key.

Shrink your C: partition by 40GB if possible using Disk Management to make some un-partitioned space for the Windows 8.1 clean install.  Create an ISO for Windows 8.1 on a 4GB or larger USB flash drive.  I just bought some 16GB USB 3.0 flash drives for $9.99 each.  A utility called Rufus can create a bootable FAT32 flash drive to install Windows 8.1.  Boot your computer with the Windows 8.1 ISO and choose Custom to install it to the 40GB un-partitioned space.  Activate Windows 8.1 Update with your OEM Windows 8 product key.

Then you can upgrade to Windows 10 on that partition by running the Windows 10 Media Creation Tool and choosing "Upgrade this PC now".

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

The end result should be an activated Windows 10 install as a dual-boot with your existing Windows 8.1.  If it works as planned, this will register your PC as a licensed Windows 10 device, allowing future clean installs of Windows 10 using no product key.  Meanwhile your existing install has not been removed, so there is little risk involved.

Once you have done this, you can back up all of your data, clean install Windows 10 and restore your data to the clean Windows 10 install.  This should resolve whatever issues you have that are preventing the upgrade now.

Creating a heading sub-shape

The heading of a container is simply the container shape’s text for basic containers.  However, it is possible to achieve more sophisticated visual styles if the heading becomes its own shape.  For this construction the container becomes a group and the heading is added as a sub-shape of the group.  The primary benefit is that Visio can account for the heading sub-shape in sizing and layout operations to avoid putting member shapes on top of the heading.  To designate a sub-shape as a heading for the container, add the cell User.msvStructureType to the sub-shape and give it the value “Heading”.  Also you should properly set theUser.msvSDContainerHeadingEdge cell in the sub-shape.  These cells helps Visio understand how the heading is positioned relative to the interior of the container.  For example, if the heading is near the left edge of the container then the correct value for the cell is “Right” (regardless of the heading’s angle).  The interior of the container is on the right side of the heading.

jeeped say:
From your sample data above try,

=INDEX($A1:$K1,MATCH(MAX($C1,G$1,K1),$A1:$K1,0)-2)

The formula can be filled down to cover subsequent rows. It has the limitation of only finding the first maximum number, e.g. if K1 was 40 as well as G1, only Joe is returned. A Conditional Format could be used to identify rows with more than one matching maximum value.
