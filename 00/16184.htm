bob jones aka: cybertaz say:

Please do not "Uninstall Office" as suggested by another user. Most likely there is nothing wrong with your current installation.

It sounds like what you are looking at is the area of the Spelling & Grammar Preferences dialog window which pertains to Custom Dictionaries. That simply is a listing of any Custom Dictionaries you create in addition to the mandatory Custom Dictionary which is provided by default. Custom Dictionaries are what terms are added to when you choose to add words that are not included in the standard [main] Dictionary.

The Proofing Tools supplied with Office 2016 are not user-accessible files... nor is there any reason to interact with them. They will be used automatically by the program when it encounters text that has been marked as being in an included language. IOW, the user doesn't select which Proofing Tools should be used, the program does - It's the user's responsibility to format the text appropriately so the program knows which Proofing Tools to use. You may find this more comprehensive article of interest:   Spelling, Language, Dictionary

Dutch Proofing Tools most definitely are provided with the standard installation and will be used when text formatted as Dutch is encountered:
