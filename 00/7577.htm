david m [msft] say:

PowerPivot requires an add-in for Excel 2010, and as such, is not native Excel functionality.  This add-in uses the MSOLAP provider, which Excel sees as an external data connection.

PowerPivot is not supported in Windows Live.  To use PivotTables in Windows Live, create a PivotTable off of data that exists in the workbook, without using the PowerPivot add-in, and without adding any external data connections to the workbook.

Add a pop-up picture to a cell in Excel 2019 - Microsoft Community

ryeguy-33d0e284-a0dd-43e2-9a50-bf0d24a469f5 say:

Add a pop-up picture to a cell in Excel 2019

In Excel 2019 the steps are the same as previous versions of Excel, except that you insert a NOTE and not a comment now! Just insert a NOTE where it says Comment and it will work as before. Here are the updated steps changed to Note instead of Comment:

  • Right Click on a cell and choose New Note (Or from the Review Tab: Notes: New note)
  • Right-click the border of the comment (not the interior!).
  • Select Format Comment... from the context menu.
  • Activate the Color and Lines tab.
  • In the Fill section, Click the Color dropdown and select Fill Effects...
  • Activate the Picture tab.
  • Click Select Picture...
  • Browse to the picture you want to display.
  • If you don't want the picture to be distorted, tick the check box 'Lock picture aspect ratio'.
  • Click OK, then OK again.
  •  Click and hold the left mouse button on the bottom right corner of the image and drag to resize if needed
  •  Press Escape on the keyboard and now when you hover over that cell, the picture will appear!

  • To edit the picture, right click in the cell and choose "edit note", press escape to close the empty comment
  • Repeat above steps to insert a new picture or resize
