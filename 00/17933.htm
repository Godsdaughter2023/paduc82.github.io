Continuous Numbering - Multiple documents MS Word 2007 - Microsoft Community

laurenlevia say:

Continuous Numbering - Multiple documents MS Word 2007

OK. so...

I have followed similar (read, identical) instructions from several different sources on how to continue (sequential) numbering across multiple document. 

  1. At the end of the first document insert a PAGE field; select it.
  2. Format the field as "hidden".
  3. Place the field in a bookmark (Word 6/95: Edit/Bookmark; Word97 and later: Insert/Bookmark).
  4. Save this file; open the next file.
  5. In the header or footer, where the page number should appear, insert the following set of fields:
    { = { PAGE} + { INCLUDETEXT "C:\\My Book\\Chap1.doc" Chap1Pages \* CharFormat \! } }
  6. Repeat the steps for each document in sequence.

Now these are a little vague and incomplete. This is what i actually did;

  1. On the second last page of the first document insert a continuous page break
  2. on the last page of the first document open the footer
  3. link from previous
  4. insert PAGE field
  5. Ctrl+Shift+H to hide field
  6. Highlight Field and Ctrl+F5 to open bookmarks (while field is selected)
  7. Add Bookmark
  8. Save and close first document
  9. Open Second Document
  10. Open Footer
  11. Insert > Quick Parts > Field > IncludeText (Making sure Field Codes are on) 
  12. INCLUDETEXT  "Z:\\FG Administration\\EA Admin\\Lauren Docs\\Admin\\Procedures\\3. Jobs\\301 New Job Rev01 LL 16apr14.docx LPN\\* CharFormat\\!" 
  13. Back in footer select Field and press F9  

This is all good and well. I do not get any errors. I get a page number. I GET THE PAGE NUMBER OF THE LAST PAGE IN THE FIRST DOCUMENT!

That is not "continuing" that is copying the last page number of the first document into the first page of the second document. So now. if i were to print both documents i would have two page fours!

How do i create CONTINUOUS SEQUENTIAL page numbering across two or more documents without using macros?

macropod say:

Try it this way:

1. Anywhere in the source document footer, press Ctrl-F9 twice to create a nested pair of fields, thus { { } }.

2. Edit these fields so you get {SET PageCount {NUMPAGES}}, then press F9. The fields will probably disappear, but that's normal.

3. Save the source document.

4. In the destination document footer, insert an INCLUDETEXT field pointing to the PageCount bookmark in the source document.*

5. Select the number inserted by the INCLUDETEXT field in the document and press Ctrl-F9 to enclose it in field braces (e.g { 25 }).

6. After the number inserted by the INCLUDETEXT field  (25 in the example above), press  Ctrl-F9 again to generate another pair of field braces '{ }'. You should now have { 25{ } }

7. Edit the field code so you end up with {=25+{PAGE} \# 0}

8. Press F9 to update the field.

* If you do this via the Insert|Quick Parts|Field>INCLUDETEXT dialogue, you should use:

INCLUDETEXT "Z:\FG Administration\EA Admin\Lauren Docs\Admin\Procedures\3. Jobs\301 New Job Rev01 LL 16apr14.docx" PageCount

and uncheck the 'Preserve formatting during updates' option.
