steve rindsberg say:

If you need an immediate solution that'll do what you want and a good deal more, have a look at my Merge add-in for PowerPoint.  The free demo is completely functional; you can test thoroughly at no cost.

http://www.pptools.com/merge/

But if you want to DIY, this'll get you started; if you need to run it from within Excel, you'll need to modify it a bit, but by using the Replace method as shown, it'll preserve the formatting of any text it replaces.

Sub TextReplace(sSearchFor As String, sReplaceWith As String)

    Dim oSl As Slide
    Dim oSh As Shape
    
    With ActivePresentation
        For Each oSl In .Slides
            For Each oSh In oSl.Shapes
                With oSh
                    If .HasTextFrame Then
                        If .TextFrame.HasText Then
                            .TextFrame.TextRange.Replace sSearchFor, sReplaceWith
                        End If
                    End If
                End With
            Next
        Next
    End With

End Sub

Assign start number for caption - Microsoft Community

crystalj123 say:

Assign start number for caption

I am attempting to create a document to be used in a legal case. It was requested that we caption all tables and figures with Exhibit 100..101..102 etc. I can find how to add the Exhibit caption, and how to include chapter numbers in the caption, but nothing for assigning a start number for you captions. Is there a way to do this? Thank you in advance for your help.

Microsoft Word 2010

Windows 7 Professional

doug robbins - mvp office apps & services (word) say:

If you use Alt+F9 to toggle on the display of the field codes in the document, you will see something like

Figure { SEQ Figure \* ARABIC }

If you want the numbering to start at 100, modify the first such field as follows

Figure { SEQ Figure \r 100 \* ARABIC }

The \r switch sets the starting number to the number that follows it.

You only need to make that modification to the first of each type of caption.

Use Alt+F9 again to toggle off the display of the field codes.
