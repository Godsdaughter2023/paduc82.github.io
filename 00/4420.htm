ossiemac say:

Following Guidelines to insert the code below in case you require them. (I don't know what your expertise is and the guidelines saves holding you up if you don't know how to insert the code).

  1. In the Developer ribbon turn on Design mode. (In Office 2010, "Design mode" is on when the icon has orange background around the icon and off when there is no background around the icon).
  2. Right click the ActiveX Combo Box and select "View code" (Opens VBA editor and auto inserts a Sub and End sub)
  3. Copy the code below and paste into the VBA editor to replace the Sub and End sub that is automatically created when you selected "View code".
  4. Close the VBA editor (Cross top right of VBA editor)
  5. Turn Off Design mode.
  6. Save workbook as Macro Enabled
  7. Ensure you have macros enabled (See Help for how to do this and option to "Disable all with notification should be OK")

The code below sets the linked cell to wrap text and then autofits the row height.

Private Sub ComboBox1_Change()
    Range(ComboBox1.LinkedCell).WrapText = True
    Rows(Range(ComboBox1.LinkedCell).Row).AutoFit
End Sub
