filldown

In VBA for Excel, how do you perform a Select Filldown (copying data down in one column) to the last cell in the column based on the last cell with data in another column?

rick rothstein, mvp - excel say:

Here is a non-looping solution that you may find interesting...

Sub FillDownNoLoop()
  With Range("B2:B" & Cells(Rows.Count, "A").End(xlUp).Row)
    .SpecialCells(xlCellTypeBlanks).FormulaR1C1 = "=R[-1]C"
    .Value = .Value
  End With
End Sub

Here's a different way

 

Sub FillDown()
Dim Area As Range
MyCol = "A"
Lastrow = Cells(Cells.Rows.Count, MyCol).End(xlUp).Row
  For Each Area In Columns("B").Resize(Lastrow). _
                   SpecialCells(xlCellTypeBlanks).Areas

  Next
End Sub

john w. vinson say:

The SEEK method (and therefore setting indexes) applies only to local table-type recordsets, not to linked ones.

As Hans suggests, using an OrderBy in a query and the FindFirst method will work with either local or linked tables. There might be a slight performance advantage to Seek - and, if you really need it, you can create a new Database object on the backend database to use it - but in general DAO recordsets and Find methods are easier to use and preferable.

 

hansv mvp say:

If you want to sort the records in the recordset, you can use an SQL statement:

 

Dim rst As DAO.Recordset
Set rst = Db.OpenRecordset("SELECT * FROM Loans ORDER BY ID", dbOpenDynaset)

 

where ID is the primary key field.

 

If you want to find a record, try using FindFirst instead of Seek:

 

rst.FindFirst "ID=37"

Only way I found to execute that command with the keyboard:

Press the context menu button which is available on most windows keyboards, usually between the right <alt gr> and <crtl keys>. (it should also work with <shift>+F10 but does not in my windows 10 or with my keyboard) That makes the context menu pop up. Now look for the underlined letter in the "copy link to paragraph" menu entry. I am using a German OneNote so I can't tell you what it is in the English version. Pressing the corresponding key does the trick.

So its:

Press and release the context menu button

Press the letter for the menu entry

additional tip:

If <shift>+<F10> is working to popup the context menu you could use an external tool like Autohotkey to define a macro which you assign to a single key press or combination. You don't need an external tool if you happen to use a programable keyboard or programmable mouse keys to assign the two key actions to a single button or click.

Hope this helps

SharePoint Template - How to open a new document - Microsoft Community

deevincent-day say:

SharePoint Template - How to open a new document

In desktop Word, I can go to a folder and right click a template then select new.  This opens a new (Unsaved) document in word and includes the content of the template.  Can anyone tell me If I can do the same thing in SharePoint?  I tried right clicking and it does not offer new as an option.  

The issue is I have a library of templates but the resulting document needs to go in a different location when saved.  

I can get around this by using synched files on my local desktop but I would be happier if I could do it within SharePoint.  

cliff gu msft say:

Dear Dee,

 

Thanks for posting the thread on our forum.

 

About the first question, you can achieve the requirement via adding a content type of the template file in the template library in SharePoint Online. You can follow the steps below to see the result.

1. Create a site library calling Template Library.

2. Go to Settings in the site>Site settings>Site content types under Web Designer Galleries>click on Create to create a new document content type and make settings as follows.

 

3. Go to Advanced settings on the following page>choose Upload a new document template>upload the template file.

 

5. Go to the template library>Library settings>Advanced settings>check Yes in Allow management of content types?.

 

6. Return to the Library settings page>Add from existing site content types to add the template file to the template library.

 

7. When clicking on New>document template 1, a new file with the template file’s data will be created.

 

If you want to change the template file, you can change the template file on the page showed in step 3.

 

Generally, if you click on the newly created file with the template file’s data, you will open it in Word Online if you don’t change some settings. After you edit the file in Word Online directly, the file will be automatically saved in the template library.

 

So about “The issue is I have a library of templates but the resulting document needs to go in a different location when saved. “, could you show me some related screenshots for our clarification?

 

Best Regards,

Cliff
