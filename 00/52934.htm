    This game should run on Windows 7 without any problems (I had it running, at the moment uninstalled due to temporary space problems). Be aware of the standards:

    1) Never install a game into the \Program Files folder of your Windows partition. This folder is protected against accidental changes, and UAC may deny access to some game functions. If the game doesn't know anything about permissions, elevated privileges etc., it will simply cease to react.

    2) Install and run the game in XPSP2 compatibility mode. Run it as administrator. Don't forget to apply the latest patch before (2.28).

    3) This game specifically has a copy protection that will conflict with any virtual discs and disc emulator software installed. Reportedly, it also doesn't work with all antivirus programs, some may block it. If you're unsure, better disable all possible troublemakers while you're playing.

    4) Check if your chipset drivers, DirectX version and graphics drivers are indeed updated. Some people have strange understandings what "up-to-date" is. Utilities like Everest or Sandra can help you to see how old those basic installations are. Concerning the graphics drivers, both AMD/ATI and NVIDIA offer drivers which work better on one Windows version and worse on another. Try until you find the one that suites you.
