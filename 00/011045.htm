scottgem say:

First, What you have is a standard configuration for Multi-User applications. What you refer to as MDB#1 is called the Front End and MDB#2 is called the back end.

The front end will link to the tables in the back end so all the tables are actually in the back end (there may be some temporary tables that are kept in the front end).

If you want to do this on a one time basis, its very easy. Make a copy of both the FE and BE in a separate folder. Open the FE uses the Shift Bypass key so the startup is not run. Then use Linked Table Manager to re-link to the test BE. At the bottom left corner of Linked Table Manager there is a check box to prompt for prompt for each table. If all the linked tables are in the same file then you will be prompted once to select the file you want to link and it will relink all the files.

If, however, you want to automate the process so that when you distribute a new front end, it will automatically relink to the production back end, then you will have to use code to do that.

a. user say:

Hello DeepakkumarGupta:

 

Thank you for posting to the forum regarding your connection issue.  From your description there are three likely issues at play.

  1. The sync cable is faulty.  I personally have had this issue and resolved it by using a friends USB cable.  It does sound like you stated multiple USB cables have been used.  If you have not, please try to get your hands on another sync cable.
  2. This may be an issue on the computers end.  There may be something on the computer that is blocking the drivers from fully installing.  Please try connecting your phone to another computer on another network.  If the phone is recognized, we know the issue is the PC.
  3. Finally, this may be an issue with the phone itself.  The above test will help us determine if this is the case.  In most situations, issues with the phone need to be resolved with a hard reset or by having the phone looked at by the phone manufacturer.  We will leave this as a last resort.

In addition to the above, please provide the following details:

  • Is the phone seen in device manager?  To get to device manager, go to your computer and click start > right click computer > select manage > click device manager.  When the phone is connected to the PC you should see it listed as a portable device.  It may be listed as an unknown or other device and have a yellow ! next to it.  Please detail what you see here.
  • If the phone is seen in device manager but is simply not being seen in Zune, please let us know the exact error you see when you go to the device section of the Zune software.
  • What is the current OS running on your PC, on the phone tap settings > about > more info.?
  • What OS is running on your PC?

Please let us know if you have any other questions or concerns.
