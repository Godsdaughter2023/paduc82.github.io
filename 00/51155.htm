my cursor keeps spinning does anyone know why?? - Microsoft Community

shooksi say:

my cursor keeps spinning does anyone know why??

Even when i'm not doing anything its still spinning!!
gerry c j cornell say:

A spinning cursor is indicative of a user started task that is still being processed i.e. the task has not completed. There are a number of explanations why it occurs for long rather than short periods of time.

1. The computer has insufficent RAM, or processing capacity to efficiently carry out the task.  Adding RAM is a simple solution. Changing a CPU may or may not involve changing the motherboard but regardless the CPU processor is one of the more expensive components in a computer. Check programme system requirements are more than exceeded as system requirements tend to be understated by programme providers.

2. The computer has not been properly maintained. Removal of temporary files and defragmenting will help.

3. Faulty or incompatible hardware needing replacement. Check for problems with drivers in Device Manager.

4. Incorrect system settings. You can troubleshoot using Event Viewer reports. Having programmes loading at startup when they might better be loaded on demand.

5. Software that needs updating. Even brand new computers often  require immediate attention because they have been sold needing an updated BIOS and motherboard drivers.

6. Faulty software containing bugs or which have been poorly written. The solution is to check for updates but better still look for independent advice not PC Magazine Reviews before purchase.

7. Poorly written or older web sites or web sites than cannot adequately handle the number of users trying to access at any one time. Having too many toolbars and search engines.
