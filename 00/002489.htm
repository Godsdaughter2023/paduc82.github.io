bilo cape town say:

 Small headsup on upgrading.

When you have created the ISO from the ESD source,

the BEST option is to make at least one BOOTABLE DVD

or Flash Drive in case you need to do a CLEAN install on any of your PC's.

This can be done with the Media Creation Tool.

The Media Creation Tool gives you TWO initial options.

1. Upgrade this PC now

2. Create media for another PC. (Meaning this one and any other PC that has the same version (Home / Pro) installed.

The version is selected automatically for the PC you are using, but you can select another version / language / 32Bit / 64Bit if you uncheck the checkbox

where it shows the default selection.

Thereafter, another option can be selected to 'Save the ISO on this PC'.

Choose a convenient location.

If the Media Creation Tool is closed without sending it to a DVD or Flash Drive, you can download the Win7 DVD/Flash Drive Utility to do the ISO burning.

Available as a direct download from: http://wudt.codeplex.com/#

The DVD / Flash drive will initiate the upgrade as soon as it is loaded in the Drive / USB Port as long as Autorun is enabled. Otherwise browse to the media in the File Explorer end run the Setup.exe.

However, the ISO can be COPIED to a Flash Drive and copied onto each PC that you plug it into. If you have Network Sharing enabled, copy the ISO across your LAN to any/all your other PC's that need to be upgraded.

To perform the upgrade:

Boot up the PC as usual on the old system and browse in the File Explorer to where the ISO is stored (or even to the Flash Drive).

Be sure you have 'Show hidden files and folders' enabled.

Double click on it if you are on Windows 8 to expand the contents. Then execute the SETUP.EXE  at the bottom of the list of contents.

This holds good as long as you do not have another 3rd party compression utility installed.

In Windows 7

RIGHT CLICK on the ISO file and select 'Open with' Windows Explorer.

Then execute the SETUP.EXE from the contents.

Hello,

the easiest way might be to turn the data into an Excel table. This works best if the first row of data contains column titles. If you upgraded to Windows 10 preview and are activated, you can also upgrade to the RTM and will remain activated and gain the ability to clean install Windows 10 with no product key on that laptop.

To upgrade, create the matching Edition of Windows 10 official ISO and run setup.exe, choose to keep Windows settings, and do not enter any product key when asked for one.
