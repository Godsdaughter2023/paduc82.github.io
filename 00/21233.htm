Okay solution very simple:

FMP17 fires applescript to open word doc and save as pdf in a temp directory.

Permission requested once to grant access to this temp folder and pdf file saved

After a little digging around I've found these variations seem to work to find yesterday's messages via raw query:

kMDItemUsedDates == $time.yesterday
kMDItemUsedDates == $time.today(-1)

For more raw query syntax information read: File Metadata Query Expression Syntax

To find additional query items you can run the "mdls" command in the Terminal on any .olk14MsgSource file found in ~/Documents/Microsoft User Data/Office 2011 Identities/Main Identity/Data Records/Message Sources/xT/xB/xM/xK/.
