aakkam22 say:

You've tried a lot already, that's impressive.

Examine your BIOS. Do the following:

1. Disable Secure Boot

2. Disable UEFI boot and enable support for Legacy boot

3. Set your disk controller to use SATA, Legacy, or IDE mode, but do not use AHCI or RAID configurations.

Next, exit the BIOS, make sure the new HDD is connected and disconnect all other drives, and then boot a live Linux disk (preferably Ubuntu, but other distros will work).

Use the Linux terminal to format the brand new disk. Then, set it to use MBR format instead of GPT. This step is important. Refer to

Afterwards, try booting the Windows 7 Disk (not Windows 10) and see if you can at least reach the "Welcome to Setup" page.

vijay a. verma say:
A. Try to boot into safe mode

Power ON your computer and as Windows / manufacturer's Logo appears, Power OFF - Repeat 3 times
Now your computer will go into Advanced Recovery Environment
Advanced Options > Troubleshoot > Advanced Options > Startup Settings > Restart
Press 4 or F4 to enable Safe Mode (5 or F5 to enable Safe Mode with Networking)
Log in once Windows starts in safe mode

Once you are able to boot into safe mode, check the integrity of the system

B. Another option when you are in Advanced Options in previous step is to choose (Note - You won't lose any files in this step)
1. System Restore - This will restore your system to a point when things were working fine.
OR
2. Startup Repair - Startup Repair upgrade fixes all Windows errors and retains all files, applications and settings.

C. Penultimate option could be Reset this PC.
In Step A > Troubleshoot > Reset this PC - Keep my files and Remove everything. Choose this carefully as most likely you would like to keep your files.
All apps will be removed and windows will be reinstalled.

D. Last option could be to Go back to the previous Build -
In Step A > Troubleshoot > Advanced Options > Go back to the previous Build

E. Otherwise, you can resort to below
To solve this problem, please following the below help guide written by Sumit Dhiman, MVP

What to do if the computer is broken after Windows 10 1803 update - https://www.yourwindowsguide.com/2018/05/what-i...

Do let me know if you have any more question or require further help. I would be happy to work with you to take you though this problem.

quetzalita say: Thank you for your suggestion! You'll see above I had tried that but I didn't have the Roxio Easy CD Creator software in my computer. However, I DID find the solution. My mistake was trying to install Windows XP Professional as an upgrade, which didn't work.

So I finally found a suggestion online that did work.
1)  I restarted and selected "Press any key to boot from the CD" (something like that!) and installed Windows XP Pro as a new program.
2)  I followed directions from the readme file, and when I got to the question of partitions, I deleted the existing two partitions (one was where the old Windows ME operating system was located, and the other was a "system save" program.
3)  After deleting the two partitions, I created a new one, following the written instructions in the installation program, and installed Windows XP with no problem.
4)  Note: when trying to install XP as an upgrade, and getting the above error message about the missing file, and cancelling, the computer still gave me the option to click "Install." I tried that a few times and each time, it told me that my Product Key was invalid. However, when I installed it as a new program, everything installed easily and it accepted my Product Key as valid. So folks, don't believe the "invalid" message immediately, you may indeed also have a valid Product Key.

I'm trying to uninstall it now, and it is taking a very long time. Tomorrow I'll determine exactly what to buy and install.
