  1. Press Windows + R on your keyboard, and then type services.msc. 
  2. Hit Enter or click OK. 
  3. In the list of services, look for Windows Update and make sure that Startup Type is set to Manual or Automatic. 
  4. Set the Service status to Start. 
  5. Click Apply and OK to save the changes. 

You can also re-register Windows Store by following the steps below:

  1. Press the Windows key + X on your keyboard.
  2. Open Command Prompt (Admin).
  3. Paste the following command: PowerShell -ExecutionPolicy Unrestricted -Command "& {$manifest = (Get-AppxPackage Microsoft.WindowsStore).InstallLocation + '\AppxManifest.xml' ; Add-AppxPackage -DisableDevelopmentMode -Register $manifest}"
  4. You should see a progress bar that re-register the Windows Store on your computer
  5. Restart your computer for the changes to take effect.

If the issue persists, manually uninstall and reinstall the Windows Store and the Xbox App from your computer by following the steps below:

  1. Click Start, and then type Windows PowerShell on the search box.       
  2. Right-click, and then select Run as administrator.       
  3. Type the following command to uninstall Windows Store: Get-AppxPackage *windowsstore* | Remove-AppxPackage    
  4. Type the following command to uninstall the Xbox app:Get-AppxPackage *xboxapp* | Remove-AppxPackage       
  5. Hit Enter.       
  6. Restart your computer.    
  7. After restarting your computer, go to the Windows PowerShell as administrator again and then type the following command to install the apps back to your computer: -Register "$($_.InstallLocation)\AppXManifest.xml"}
  8. Hit Enter.
