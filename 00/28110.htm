Thank you for contacting Microsoft Community.

I understand it’s annoying when things do not work as expected.

This issue could occur due to some change in Internet Explorer settings.

Please follow the methods below to resolve the issue.

Method 1

Check to make sure the Delete browsing history on exit box in your Internet Options is not checked.

  1. Open Internet Explorer 11.
  2. Click the Tools button, and then click Internet Options.
  3. Under the General tab, and then under Browsing History, make sure the Delete browsing history on exit box in not checked.  

Check whether the problem is fixed. If not go to the next method.

Method 2

Check to see if there is enough free space on your hard disk.

  1. Open Internet Explorer 11.
  2. Click the Tools button, and then click Internet Options.
  3. Under the General tab, and then under Browsing History, click Settings. Under Disk space to use, make sure the disk space is set to 250MB, and under History make sure the Days to keep pages in history is set to 20.

Note: If you don’t want a webpage history kept, set the number of days to 0.

If the issue still persist please refer method 3.

Method 3- Delete your temporary Internet files

It’s a good idea to delete temporary Internet files periodically to free drive space.

  1. Open Internet Explorer 11.
  2. Click the Tools button, and then click Internet Options.
  3. Under the General tab, and then under Browsing History, click Delete.
  4. In the Delete Browsing History dialog box, select Temporary Internet files, Cookies, and History, click Delete, and then click OK.  

Note: Your changes will take effect the next time you open Internet Explorer.

If the issue still persists please refer method 4.

Method 4- Reset Internet Explorer 11

Please refer the link below to reset the Internet Explorer 11 Settings.

            http://windows.microsoft.com/en-us/internet-explorer/reset-ie-settings#ie=ie-11

By resetting Internet Explorer settings, you return Internet Explorer to the state it was in when it was first installed on your computer. This is useful for troubleshooting problems that might be caused by settings that were changed after installation.

Disclaimer: Resetting Internet Explorer's settings is not reversible. After a reset, all previous settings are lost and cannot be recovered.

If you have gone through the "scorched earth" complete Removal. 

  1. When you open Word (could apply to PowerPoint or Excel, or Outlook) Go to Word Menu
    a) Click  on About Word.
    b) Look and see what Version you have (for example as shown in this screenshot on my install of Word 2011).
  2. If version 14.1.0 is shown then you need to reset the message you're getting by:
    a) Quit Word (or any and all Office apps you have open.
    b) Open Disk Utility, and Repair Permissions. (Run twice to make sure all Are repaired there may be some false positives.

  3. Restart Computer.

  4. Now reopen Word, if the update message does not  return or changes, then continue with update to current version.

  5. If instead you get an exact Repeat of the message, do the following:
    a) Quit any open Office application(s).
    b) Locate the ".plist" file for the Microsoft Auto Update. Use either Spotlight, from Finder go to FileMenu and choose Find File, or download my favorite tool called Easy Find; type in AutoUpdate.plist. Check location to see if associated with Microsoft Office. 
    c) now  remove to Trash.
    d) Restart computer.
    e) Now reopen updater should clear up the issue.

  6. Now download and update the 14.5.1 update (or which ever is in effect at time this is written).

  7. Continue with install.

  8. Before opening run Disk Utility and repair permissions, before opening Word.
