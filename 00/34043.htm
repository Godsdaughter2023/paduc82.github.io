bossdweeb say:

If all of your files/software are on the same partition as Windows they will be transferred to the new HDD when

your clone/image/migrate Win10 to the new HDD.

If you have data or software on a separate partition or HDD to Windows you can  simply copy that data

to the new HDD.

Be careful to use the same drive letter for the partition that the data is stored on now in case there

is something that requires the original file path ( software in particular, but some data may as well ).

There are a number of cloning tools. I use the free version of AOEMI Partition Manager, but HDD mfg's

also have similar software available from mfg's web site.

Free Partition Manager - AOMEI Partition Assistant Standard

- I appreciate that 3TB may be more than enough space for your needs, but you should seriously consider

getting an SSD for Windows.  120GB is adequate, but 240GB is more cost effective per GB.

Of course this is a bit of a bother if you have everything on C:\ , but  Windows loads faster and runs quicker

from and SSD. Once you've used Windows on an SSD you will never want to go back to HDD...

Sooo slow... Software also benefits some, but games are fine on an HDD as they only benefit from

slightly decreased loading times on SSD, and of course, data if fine stored on an HDD.

keith a. miller say:

When you get multiple System Folders pointing to the same folder, you have to edit the Registry to unscramble the mess.  Make sure the folders you want to be your System Folders have been created in the file system:

  1. Open the Registry Editor by typing "regedit" in Search or the Run dialog.
  2. Ctrl+F will open the Regedit search dialog.  Type: UserShellFolders.  Select the checkbox to search Keys; clear the checkboxes for Values & Data.  Click 'Search' and you should find yourself at: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders.
  3. You should see the stored path names in the right-hand pane.
  4. Edit Values in the right-hand pane by double-clicking the Value name.  Type the correct path for each folder.
  5. Repeat the above for values found under: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders.  (these are there for legacy software.  May not be needed by any programs you run, but good form to be consistent).
  6. Close Regedit & reboot.

Good Luck,

Keith

misssvsiebert say:

I can't change the backround.

On my computer, I usually change the backround by going into slide show, and then clicking on a picture and choosing make my backround, but it hasn't been working lately. instead, nothing changes. can anyone help? 

a. user say:

Hi,

 

Are you referring to the Slide Show for the background picture which does not change or is it an isolated desktop background that you are unable to set?

 

Well, if you are referring to the multiple desktop background pictures that you have set as slide show (which would start according to the time that you have set), this could be due to the fact that the slide show might be paused under power options.

 

Check if these steps helps to reset the settings:

a) Press Window+X key together and select Control Panel.

b) Click on Power options.

c) Click on Change plan settings.

d) Click on Change advance power settings. Click on Change settings that are currently unavailable.

e) Under Desktop background settings, select Available for Slide show section.

f) Click on Apply and then on OK.

 

Check if the desktop background works as a part of slide show.

 

If it’s an isolated picture that you are trying to set as background, Corrupted TranscodedWallpaper.jpg could be one of the different reasons. Follow the methods to try and fix the issue:

Method 1:

Follow the steps to rename corrupt TranscodedWallpaper.jpg:

a) Press Windows and R to open Run. Type the following location and hit Enter:

%USERPROFILE%\AppData\Roaming\Microsoft\Windows\Themes\

b) Right-click the TranscodedWallpaper.jpg file, and then click Rename.

c) Change the file name to TranscodedWallpaper.old, and then click Yes when you are prompted.

d) If slideshow.ini is listed, double-click this file. Slideshow.ini should open in Notepad. Select all the text in slideshow.ini if any exists, and then press Delete on the keyboard.

e) On the File menu, click Save.

f) Close Notepad.

g) Close the File Explorer window, and then browse to the picture that will be set as the background image.

h) Try to apply a background picture again.

 

If this does not work, proceed to the next method.

 

Method 2:

Check the Remove background setting under the Ease of Access settings. Follow the steps:

a) Press Windows+X key to select Control panel and then click Ease of Access Center.

b) Under Explore all settings, click Make the computer easier to see.

c) Make sure that the option to Remove background images not selected.

d) Click OK twice, and then close the Ease of Access Center window.

 

Check if the required changes help you to retain the background wallpaper.

 

Please let us know about the status of this issue.
