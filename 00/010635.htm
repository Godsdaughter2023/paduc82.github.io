bernie deitrick say:

It depends - do you have + and - letter grades? Or just whole letters? What is your rule for, say, an average of 2.5?  2.7? 2.9?  Post your VLOOKUP formulas and table that you use to convert to/from letters/numbers.

 

If you use just whole letters to grade the sub-scores and typical rounding, then try this array formula: enter using Ctrl-Shift-Enter:

 

=CHAR(ROUND(AVERAGE(IF(B2:D2<>"",CODE(B2:D2))),0))

 

If you want the average to have + and - letter grades based on full letter grades, use this array function (again, enter using Ctrl-Shift-Enter)

 

=CHAR(ROUND(AVERAGE(IF(B2:D2<>"",CODE(B2:D2))),0)) & CHOOSE(ROUND(MOD(AVERAGE(IF(B2:D2<>"",CODE(B2:D2))),1)*3,0)+1,"","-","+")

 

Note that if you have more than 3 letter grades, you would change the B2:D2 to a bigger range, but leave the *3 alone - that is used to bin the + and -, not get an average of 3 letters....

dhookom say:

This query gets you the current and previous dates:


(SELECT TOP 1 DateCompl FROM tblCamCafe A
WHERE A.Part = tblCamCafe.Part and A.Order = tblCamCafe.Order AND
 A.DateCompl<tblCamCafe.DateCompl ORDER BY A.DateCompl DESC) AS PrevDate
FROM tblCamCafe;

This query gets you a date difference:


WHERE A.Part = tblCamCafe.Part and A.Order = tblCamCafe.Order AND
 A.DateCompl<tblCamCafe.DateCompl ORDER BY A.DateCompl DESC) AS PrevDate, DateDiff("d",(SELECT TOP 1 DateCompl FROM tblCamCafe A
WHERE A.Part = tblCamCafe.Part and A.Order = tblCamCafe.Order AND
 A.DateCompl<tblCamCafe.DateCompl ORDER BY A.DateCompl DESC),[DateCompl]) AS Days
FROM tblCamCafe;

You could change the increment to hour and then divide by 24 to get something more accurate. I don't have the time to figure out how to remove weekends. This would take a user-defined function.

jeeped say:
Your original question dealt with A1 and A2 with A1 taking precedence. Now the cells are A1 and B1 with no mention of precedence. Hopefully, this modification should take care of that as well as the notification.

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("A:B")) Is Nothing Then
        On Error GoTo FallThrough
        Application.EnableEvents = False
        Dim ab As Range
        For Each ab In Intersect(Target, Range("A:B"))
            If Not Application.Sum(ab) > 0 Then _
                ab.ClearContents
            If Application.Count(Cells(ab.Row, 1).Resize(1, 2)) = 2 Then
                'MsgBox "cannot provide data in both " & _
                  Cells(ab.Row, 1).Address(0, 0) & " and " & _
                  Cells(ab.Row, 2).Address(0, 0), vbExclamation
                ab.ClearContents
            End If
        Next ab
    End If
FallThrough:
    Application.EnableEvents = True
End Sub

To enable the warning message, remove the ' from 'MsgBox "[...] leaving MsgBox "[...]. I've left it commented out of processing so you can see how to turn it on and off and you will likely find it annoying during preliminary testing.

strath say: Thank you for the response.  I guess my follow-up question is then, how do I edit notebook names in the OneNote app on iPhone and iPad?  When I click on the edit icon, it is only allowing me to delete the notebook, and seemingly nowhere to edit the name?

bernd p. say:

I knew you would ask this ;-)

The general answer is: rename the notebook master. The original notebook.

This is valid for all possible locations (on your computer, on a network drive, on Sharepoint or Onedrive).

In your special case of a master on Onedrive you must rename it on Onedrive. That means using Onenote Online aka the Onenote Web App in your browser.

But expect, depending on your Onenote version(s), to lose the connection to your notebook, because you changed the path. On any client where you see this effect just close the notebook and re-open it with the new name.

Bernd
