"I disabled it and even it is shown as disabled still Defender detects  false positives and is conflicting a third party antivirus"

Not a bug in Windows or Defender in this case, but with your 3rd party antivirus program perhaps.

Your problem is that Defender should *not* be active *at all* program on Windows 10 (and Windows 8.x). When a 3rd party antivirus registers to Windows at install (as it is supposed to, if it is compatible), Defender is de-activated to allow the 3rd party product to be the registered antivirus/antispyware protection.

If the 3rd party program has not registered itself properly to Windows, Defender will remain active and you have a conflict condition. You should never have more than one security product installed on the PC providing active protection/scanning. This can cause performance issues, system instability, and can hinder the effectiveness of both products at providing protection.

Ignoring the primary issue of the 3rd party antivirus being present, Defender can be disabled from the interface, but it will re-activate automatically because that's what it is designed to do. You have to also stop the service and change it to be manual in Services if you want to shut Defender down.

hansv mvp say:

It depends on the way you create the hyperlinks. Here are a few simple experiments.

Select A1 and insert a hyperlink.

Selection.Hyperlinks.Count returns 1, as expected.

Now fill down from A1 to A4.

Each of the cells in A1:A4 shows a hyperlink, and if you select any of those cells by itself, Selection.Hyperlinks.Count will return 1, still as expected.

But if you select A1:A4, Selection.Hyperlinks.Count returns 2, not 4. What happened?

The original hyperlink in A1 counts as 1, and the copied hyperlink in A2:A4 also counts as 1. Not as 3 separate hyperlinks! If you select A2:A4, you'll find that Selection.Hyperlinks.Count returns 1.

So I suspect that you created individual hyperlinks in A1:A29 (counting as 29), then filled or copied right to column L (creating 1 new hyperlink for each row, so 29 new hyperlinks, for a total of 29+29=58 hyperlinks.
