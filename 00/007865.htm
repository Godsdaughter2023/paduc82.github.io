ben wil say:

I may have found the answer. I've been plagued by this problem for the last 24 hours, and reddit came through for me with my answer after crawling through search results for answers. Here's what worked for me:

  1. Run the Windows 10 Media Creation tool from the website. Be sure to get the version that matches what you currently have installed.
  2. Let it download and verify the download. With me, this is where it failed immediately. It would say "Creating Windows 10 media" and show 100% and say that it stopped working.
  3. Close the tool and open a run box. (Windows Key + R)
  4. Paste this: C:\$Windows.~WS\Sources\Windows\sources\setupprep.exe
  5. Voila! Your installation should begin.

As far as I know, this only works if the download is 100% but the installer stops unexpectedly. Hope this helps!
