Port Forwarding - Microsoft Community

xbr dionysus say:

Port Forwarding

Login Information

  • Default Gateway - gateway.2wire.net / 192.168.1.254
  • Username - N/A
  • Password – “Device Access Code” on the side of the device

 

Port Forwarding

  1. Open Web Browser.
  2. In the address bar of the browser enter in "gateway.2wire.net" (if this does not work, try one of the following IP address: 192.168.1.254).
  3. If asked for a password it ill be located on the side of the device labeled “Device Access Code”
  4. Click on the Settings tab along the top of the page.
  5. Click on the Firewall tab in the submenu.
  6. From the Sub-Sub-Menu select Applications, Pinholes and DMZ.
  7. In section 1, choose the Xbox as the computer that will host applications.
  8. In section 2, select Add a new user-defined application.
  9. Name the rule Xbox.
  10. Select TCP for Protocol.
  11. Type 3074 for both Port boxes.
  12. Keep Application Type to defaults.
  13. Click Add to List.
  14. Repeat steps 7-12 only change the name of the rule to Xbox2, and the Protocol to UDP.
  15. Click Back.
  16. Select Allow individual applications.
  17. From the list of applications select Xbox and Xbox2 and select Add.
  18. Select Save.

 

Disable Port Forwarding

  1. Open Web Browser.
  2. In the address bar of the browser enter in "gateway.2wire.net" (if this does not work, try one of the following IP address: 192.168.1.254).
  3. If asked for a password it ill be located on the side of the device labeled “Device Access Code”
  4. From the menu at the top select Settings.
  5. From the sub-menu select Firewall.
  6. From the Sub-Sub-Menu select Applications, Pinholes and DMZ.
  7. In step 1, select the blank spot.
  8. Click Save.

To remove printer drivers:

a.     Click on Start, type Print Management in search box and click Print Management.

b.    Open Print Management.

c.     In the left pane, click Print Servers, click the applicable print server, and then click Printers.

d.    In the center pane, right-click those printers that use the driver that you want to delete, or change the driver that is used by each printer to another driver.

e.    In the left pane, click Print Servers, click the applicable print server, and then click Drivers.

f.      In the center pane, right-click the driver and do one of the following:

·         To delete only the installed driver files, click Delete.

·         To remove the driver package from the driver store, completely removing the driver from the computer, click Remove Driver Package.
