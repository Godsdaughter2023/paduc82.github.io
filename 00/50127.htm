You would receive this prompt if multiple operating system are installed and required to be chosen when you boot the computer.

 

Well, rather than removing the operating system, you can follow another way to boot the computer to Windows 8 operating system within seconds from the boot screen.

 

Follow the steps:

a)      Boot the computer and choose Windows 8 from the options.

e)      Press Windows and C key to open charms bar and click on power icon.

h)      Click on Change the timer.

i)        Click on 5 seconds.

j)        Click on “Choose a default operating system: Current default Windows 8”.

k)      Click on back button.

l)        Click on Windows 8 to boot the computer.

m)    Restart the computer and check whether the computer chooses Windows automatically 8 after 5 seconds.
