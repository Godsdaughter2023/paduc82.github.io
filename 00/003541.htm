classie say:

bchapman255 say:

Fax Mail Transport can work in a 64-bit operating system under the following conditions:

1) You're not using Vista.

I'm investigating workarounds for this one, as I found this one out the hard way in a production environment and neither I nor my client are happy about it.  Vista spits out an error message "The requested operation failed." if you go to Tools -> E-mail Accounts (Outlook 2003) or Account Settings (Outlook 2007) -> click Next (2003), select "Fax Mail Transport" and click "Change."  When that happens, any outgoing faxes will not appear in the server's Outbox and gets stuck in the wild blue yonder.

2) You have Windows Fax and Scan installed (or the Fax service installed if XP x64 or Server 2003 x64 are what you want to use).

3) You have Windows Fax and Scan configured to connect to a Fax Server or Fax Modem of some kind.  In other words, if the status bar shows "All fax accounts are accessible," then you're in business.

4) You're not using the 64-bit build of Outlook 2010.  It does not have Fax Mail Transport.

Once these conditions are met, Fax Mail Transport should be available as a choice when adding a new account under "Other" or "Other account types."
