deepak b say:

Hi Melissa,

 

Thank you for reaching out to Microsoft Community where you can find all the required information about Windows Operating Systems!

 

You want to know what are desktop.ini and Thumbs.db files and the computer runs slow.

 

desktop.ini is a file used to customize settings for the Windows folders that contain the file.

 

Thumbs.db has information that helps to speed up the showing of thumbnails when you are viewing a folder in Thumbnail view.

 

Deleting these files would not harm the computer in any way.

 

To work with the computer being slow, you could refer to the site:

Slow PC? Optimize your computer for peak performance

http://www.microsoft.com/athome/setup/optimize.aspx#fbid=SiScpw55dsV

 

Note:  While checking a drive for errors, if bad sectors are found then the system tries to repair the sector. Any data available might be lost. Hence take a backup of data.

 

Note: Any data files that are infected may only be cleaned by deleting the file entirely, which means there is a potential for data loss.

 

For any Windows related issue, feel free to contact us.
