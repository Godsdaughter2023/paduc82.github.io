My conclusion is as follows:

Because I'm running a French OS when I repaired the OS the account was first provisioned with the data for French (hence the "fr-FR" folder). When I then reinstalled the US English language pack (obviously after logon) and everything got installed for that user account to use the "display language = US English" setting (that got preserved across the repair), except for the necessary "en-US" folder. Can you please pass the message along within Microsoft?

For the record, the fix for anyone encountering this issue should be the following, starting from a broken account X using display language Y (so, language pack

1. create another account T

2. set that other account's display language to Y

3. make sure settings search work under that T account (might take some time for indexing)

4. copy the folder for indexed settings data for language Y from c:\users\T\AppData\Local\Packages\windows.immersivecontrolpanel_cw5n1h2txyewy\LocalState\Indexed\Settings to c:\users\X\AppData\Local\Packages\windows.immersivecontrolpanel_cw5n1h2txyewy\LocalState\Indexed\Settings

5. Delete the folder for indexed settings data for language Z from c:\users\X\AppData\Local\Packages\windows.immersivecontrolpanel_cw5n1h2txyewy\LocalState\Indexed\Settings

6. Wait for the indexing service to index that data

That should do the trick.

Hi,

Thank you for posting your query in Microsoft Community.

I understand how frustrating it could be when things do not work as expected. Please do not worry I will try my best to resolve it.

Microsoft does not support changing the location of the Program Files folder by modifying the ProgramFilesDir registry value. If you change the location of the Program Files folder, you may experience problems with some Microsoft programs or with some software updates.

To change the location of the Program Files folder back to the default location, follow these steps:

a. Press Windows + R, type regedit , and then click OK.
b.Locate and then click the following registry subkey:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion

c. In the details pane, right-click the ProgramFilesDir registry value, and then click Modify.
d. In the Value data box, type the default location for the Program Files folder, and then click OK.

Note The default location of the Program Files folder is systemdrive\Program Files. For example, if Windows is installed on drive C, type C:\Program Files in the Value data box.
d. Exit Registry Editor.

Note: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. Hi,

Changing the language of your Windows 10 can be done through Settings app. Follow these steps to change the language:

  1. Press Windows key + I to open the Settings app.
  2. Click on Time & language.
  3. Click on Region & language tab.
  4. Under Languages, click on Add a language.
  5. Select the language that you want to add, and then select the specific variation if applicable.
  6. Select the new language, and then click on Options button (download any the language packs as necessary).
  7. On Region & language tab, select the new language and click on Set as default button.
  8. If applicable, under Country or region, select your location to enable the Windows 10 to give you local content.
  9. Click on Start.
  10. Type Control Panel, and then press Enter.
  11. On the top-right corner, set View by to Small icons or Large icons.
  12. Select Language.
  13. On the left pane, click on Advanced settings link.
  14. Click on Apply language settings to the Welcome screen, system accounts, and new user accounts.
  15. On the Administrative tab, under the Welcome screen and new user accounts, click on the Copy settings button.
  16. Check the Welcome screen and system accounts and the. New user accounts options.
  17. Click on OK button.
  18. Click on OK button on Region window.
  19. Reboot your computer and upon restart, you should see the new default language globally set through the operating system.
