How to avoid having to click enable editing. - Microsoft Community

southern minnesota periodontics say:

How to avoid having to click enable editing.

Is there a way to save a document so that when you go to open the document you don't have to click on the "enable editing" or having to click the view tab and then edit document?  

jay freedman say:
  1. Do not ever open a document while it's still attached to an email with the intention of editing it. The article http://www.gmayor.com/outlook_attachments.htm explains why. Always save the document as a separate file on the drive, and open that.
  2. If the document originates from another computer, right-click the file in the folder where you saved it, click Properties, and unblock the file (that's a button or a check box in the dialog, depending on your version of Windows). If there is no Unblock item in the dialog, don't worry about it.
  3. If you save the file in a folder that you've set in Word as a Trusted Location, you won't get a warning banner. (To set a location, go to File > Options > Trust Center > Trust Center Settings > Trusted Locations.) But be careful about what files you put in a trusted location, because such files aren't subjected to any security measures.

It's possible to disable Protected View in the Trust Center, but that's a really poor idea. That stuff really is there for your protection against malicious files.

ken sheridan say: It sounds like the page design width, plus whatever the right and left margins are set at, is wider than the paper size in use.  Open the report in design view and  first make sure that it is set to landscape format in the Page Setup ribbon.  If so select Page Setup in the same ribbon to see what the margin settings are.  Then in the report design window see what the design area extends to on the ruler bar at the top and add the two margin dimensions to that.  If the total is above the width of the paper then you need to either reduce the margins or reduce the width of the design area.

Reducing the margins is done in the Page Setup dialogue which you opened to check the margin settings.

To reduce the page design area you'll first have to resize any controls which extend beyond the limit to which you want to reduce the design area.  To do this click in the top ruler bar beyond the point you want to reduce the area to.  This will select all controls which extend to that point.  The control in the footer which shows the 'Page of Pages' will be one of these but there could be others such as dividing lines which the wizard has inserted.  You can resize them all together by resizing one while all are still selected.  To do this grab the handle on the centre of the right side of any of the control.  The mouse pointer will turn to a horizontal double-headed arrow.  You can then drag the right edge of the controls to the left so they all are within the new area.  Having done this you can then drag the right edge of the design area itself (it's the white gridded part of the window) to the left to its new position.  Unlike a control the design area doesn't have handles; you just position the mouse pointer over the edge of the area until you get the double-headed arrow.

If any for the control's you resized are now too small for their contents you can select them individually by clicking on them and dragging the control's left edge to the left to enlarge it.

Even after many years of creating Access databases I still find the report wizard useful as a starting point when creating a report, but invariably then make changes to its layout manually in design view.  These are usually customizations, however, rather than correcting something like the problem you've encountered, so I'm not sure just how it's arisen in your case.

Ken Sheridan, Stafford, England
