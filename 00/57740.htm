cody c. say:

Thorlain,

My understanding is that Windows Defragmenter and Defraggler determine a fragmented file differently. Defraggler will consider a file fragmented if it has a single fragment where as Windows will allow a handful of fragments before feeling the file needs to be defragmented.

The reason this is important is that if your HDD has large files (2-10 GB in size) and they have a single fragment, this can cause Defraggler to show a very high fragmentation level where Windows would not.

Hope this makes sense! Let me know if you need further help!
