myall_blues say:

For the first part, I think the apparent rounding is probably due to the way you have formatted the labels (and therefore the values). Although it shows as 23, it will still be plotting as 22.5.

For the second part, when you add a second Y axis Excel treats them independently, so you have to scale one or the other so that they line up.

For example, from your data, the value of 300 corresponds to 49.65%, but from the scaling of the secondary Y axis in your image it corresponds to about 51%, which explains why they don't correspond. To fix it you need to make them scale equally, so for example set the primary Y axis to go from zero to 300, and the secondary Y axis to go from zero to 49.65.

myall_blues say:

As noted the scales are independent and there is no option in Excel to force them to match. The only way to do what you want is with VBA.

For example, you could put some formulas in a group of cells that calculates the axis max, min (and interval optionally) for both the primary and secondary axes based on your dynamic data (i.e. the values change as your data updates). The code example below sets the primary and secondary axis scales based on the values in E2:G2 for the primary and E3:G3 for the secondary. Select the chart and then run the code.

If you wanted you could hook up the code to a worksheet event (such as Worksheet_Change) to make it trigger automatically when your data update. Jon Peltier has an example of how to do this on his website.

https://peltiertech.com/link-excel-chart-axis-scale-to-values-in-cells/

Private Sub AlignAxisScales()
    With ActiveChart
        With .Axes(xlValue)
            .MinimumScale = ActiveSheet.Range("E2").Value
            .MaximumScale = ActiveSheet.Range("F2").Value
            .MajorUnit = ActiveSheet.Range("G2").Value
        End With
        With .Axes(xlValue, xlSecondary)
            .MinimumScale = ActiveSheet.Range("E3").Value
            .MaximumScale = ActiveSheet.Range("F3").Value
            .MajorUnit = ActiveSheet.Range("G3").Value
        End With
    End With
End Sub

I want English spell-checking. What can I do?

daniel g. grau say:

Bear in mind that language is a text attribute, not a setting or preference. Therefore, every word you write has an associated language to it, just like it has "underline/no underline" or a specific font. This allows Word to have multiple languages in the same document.

The solution is to select the text in question (probably a Select All with Command A will do fine), then go to Tools > Language and apply the proper language parameters.

If the language error is happening with all your new documents, then also click on the Default button to propagate that change back to the Normal template (the mold from which all your blank documents are cast).
