suzanne s. barnhill say:

As a diagnostic step, try starting in Office Safe Mode (press Ctrl while loading Word and say yes when asked if you want to start in Safe Mode). Set "Files of type" to "All Word Documents" and close it, then reopen. Does it stay the same? Try some other selections and test. It should retain the last selection every time you open it.

If it works as expected in Safe Mode, then you know there is a problem with Normal.dot, the Registry, the printer driver, or an add-in. In most cases the problem is caused by interference from a third-party add-in, but in this case it might also be a Registry entry, though I'm not sure that refreshing the Word Data key (a solution to many problems) would be the solution here; there may be some specific key that pertains to that particular setting.
