    Hello,

    Thank you for writing to Microsoft Community Forums.

    What exactly do you mean by 'Cortana pop-up'?

    Cortana is optimized for specific language and market pairings. It works the best if Languages (this is your device language), Speech language and Country or region are set to the same language. Make sure the country selected corresponds with the Windows Display language set in the Language setting. The following are some things you can try if you're having problems with Cortana and see if it helps.

    Method 1: Make sure Cortana is enabled in Settings and all recently released updates have been installed. For more information, you may refer the article on Fix problems with the Start menu or Cortana.   

    Note: Do not follow the last step (Reset your PC) from the above article.

    Method 2: Country, display and speech language settings.

    Cortana will refuse to work if you have set different speech language and display language. If you're in U.S. region, then you might be using English (United States) as your display language and English (United States) as your speech language. If you set English (United Kingdom) instead of English (United States), it won't work as the languages don't match. To change the speech and display language settings, you may refer the article on Cortana's regions and languages.  

    Method 3: Restart Windows Explorer service.

    It might not work due to the service which is not running. So let's check if the service is enabled.

    • Go to search window and type services.msc and click on Enter.
    • Narrow down to Windows Search service and right click to select Run.
    • Thereafter you need to select the running type to Automatically in service Properties and click on Apply and OK.
    • Right click on the Windows Search service and choose Restart option.

    Method 4: Re-register Cortana.

    I would suggest you to re-register Cortana as its stopped working. To do that, follow the below mentioned steps.

    • Right click on the Start menu and select Windows PowerShell.
    • In the PowerShell window, type the following command to uninstall Cortana -

    Get-AppxPackage Microsoft.Windows.Cortana | Remove-AppxPackage

    • After uninstalling, run the below mentioned command to reinstall Cortana -

    Get-AppXPackage -allusers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

    • Then press the Enter key and Restart your computer.

    Hope it helps.
