tryx3 say:

Brett,

1  The ISO is no longer available for preinstalled Windows 7, as you have already discovered.  There seems little point finding any way around this limitation as any website providing downloads will be doing so without a licence and might have added anything they wanted to the installation files [it's difficult enough getting rid of any normal malware infection but malware that had been written into a bastardised operating system could be well-nigh impossible to remove].

2.1  You can borrow somebody else's installation DVD to do it. 

2.2  You should borrow a Home Premium installation DVD and you should check that it is the DVD for your type of architecture [32 or 64 bit].

2.3  Installing with a borrowed DVD will mean that you will definitely have to enter the Product Key from your Certificate of authenticity [CoA] label.

2.4  It is possible to backup that installation DVD to a USB and your licence allows you to do so.  It's a bit convoluted but I'll explain if you are going to do it [you run some specific DiskPart commands and then just copy the entire DVD across to the USB].

3.1  The potential advantage of using a DVD from the preinstaller [iBuyPower] only exists if your Windows 7 was self-activating when you got your computer i.e. it ran you through the setup process but you did not have to enter the Product Key from your CoA in order to activate Windows 7.  [The Product Key is provided in case this self-activating mechanism should ever fail] 

3.2  The advantage is that using their recovery DVD means that the same will happen again - you will not need to enter the Product Key.  Ask them that direct question to avoid disappointment.  There can be a secondary advantage in that it is likely to include the correct hardware drivers as well - that will save you a bit of messing about getting everything working afterwards.

3.3  Make sure that they mean what you want them to mean.  There is no commonly agreed definition of a "recovery disk" so there is a danger that they might just mean a disk with all the drivers on it rather than a disk from which you can reinstall Windows 7.  Ask them that direct question to avoid disappointment.

3.4  To qualify for the upgrade, you need to complete all windows 7 updates up to and including SP1 so you might also want to check with iBuyPower whether or not their recovery disk is already at SP1 because that would save you a lot of effort.

4  Do note that many people have had problems upgrading to Windows 10.  Some have then had problems running Windows 10 [many have found that OneDrive has messed up access permissions to their own user folders, just for example].  Some have had problems after then reverting back to Windows 7.  So having a reinstallation disk handy is an important tool to hold ready before you attempt the upgrade again [along with a system image of the completed Windows 7 installation since that will enable the quickest possible return to your starting point if it all went horribly wrong for you again, probably only about 20 minutes].  Do also bear in mind that there is no rush, we have until 29th July 2016 to do the upgrade - waiting a few months might mean that many of these glitches have been fixed.  I'm using the Preview version at the moment and am not upgrading my main Windows 7 installation until late November.  iBuyPower's "several weeks" postal delay might actually rather suit you.

5  You did not ask about this but ... .  Do you realise that, once you have an activated Windows 7 SP1 installed, you can run the Windows 10 upgrade as a Repair install [aka InPlace Upgrade].  See my paras 5-9 in this thread  [**] for the procedure.  If it did all go horribly wrong and you had to revert to Windows 7 SP1 then start again, at least you would have a Windows 10 installation disk ready to go rather than have to repeat the whole download phase yet again.  [** If what I wrote in that other thread needs any clarification then please post back here in your own thread not the other one as it could all get pretty confusing otherwise.]

All the best,
