Hi Victoria,

I had the same problem and was able to fix it by the following.

  • Close your Windows Mail app.
  • Click start and type in Windows Powershell
  • Right-click and choose run as administrator
  • type in or copy and paste the following:  
Get-AppxPackage -AllUsers| Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}
  • Press enter and wait for the operation to finish reinstalling all Windows built-in apps.
  • Restart your windows.

Above fixed it for me. Same procedure can be used to fix some other problems with built-in Windows apps.

Good luck!

To know the activation key of Windows without any program you must follow these steps:

Open the start menu and type "regedit", right click on the search result and select the option "run as administrator".

After opening the registry editor, open the path: HKEY_LOCAL_MACHINE> SOFTWARE> Microsoft> Windows NT> CurrentVersion> SoftwareProtectionPlatform

After clicking on SoftwareProtectionPlatform look for the entry BackupProductKeyDefault and the value that appears as a third party in the list is your Windows activation key.

The key has a format of five groups of five characters each.

It looks something like this: TG59Ñ-W3M1G-FGBSX-4YWX6-6F2CY. (This is an example)
