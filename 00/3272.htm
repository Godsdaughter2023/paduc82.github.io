if so,

try this

(expected results in same forlder with xl file, text file named: textANSI.txt)

if you are not familiar with macros...

first,

take a look here... enable macros

second,

step1

Save As, your Workbook with extension .xlsm (macros enabled)

step2

1) press ALT+F11 to open Visual Basic

2) select: Insert > Module and paste the code below on the right

Sub XLS_to_TXT()
'data in active sheet
Dim r As Long, c As Long, i As Long, j As Long
r = Cells(Rows.Count, "A").End(xlUp).Row
c = 3 ' max number of columns
Dim myPath As String
myPath = ThisWorkbook.Path
Dim v() As Variant
ReDim v(1 To c)
Open myPath & "\" & "textANSI.txt" For Output As #1
For i = 1 To 10
For j = 1 To 1
v(j) = Cells(i, j)
Next
Print #1, Join(v, vbTab)
Next
For i = 11 To 40
For j = 1 To 2
v(j) = Cells(i, j)
Next
Print #1, Join(v, vbTab)
Next
For i = 41 To r
For j = 1 To 3
v(j) = Cells(i, j)
Next
Print #1, Join(v, vbTab)
Next
Close 1

MsgBox "done"

End Sub

3) Press ALT+Q to Close Visual Basic

step3

To run the macro, press ALT+F8, 

select:  XLS_to_TXT from the list and click the run button
