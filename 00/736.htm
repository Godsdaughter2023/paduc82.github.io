albert d. kallal say:

I think this is a great question (probably because I have a good and easy answer!). By the way, nice formatting for the macro code. I am going to copy that...

 

The way lookup works is that if the record is found, then code inside of the code block runs (code indented inside of the lookup). If record is not found, then all of that indented (nested) code does not run.

 

So in your code example, you could just remove the if..then and you would be fine.

 

In most cases when looking to do something with the record then it is easy since all the code simply runs inside of that indented code block.

 

I would say there more of an issue for when the record is not found. In that case you would have to test/check and set some type of flag. (access team missed a great idea of adding "else" to ANY code block in these cases). So, in the case for not found, set some value before lookup. somthing like:

Eg:

SetLocalVar  (NoMatch, True)

·        Look Up A Record In tblEmployees

       Where Condition = [tblEmployeesID]=100

             setlocalVar    (found,false)

            Code here if record found runs....

 

If localVars!NoMatch = True then

   code here for no match.

End If

 

> A related question, how do you find out how many records the LookupRecord has returned?

 

No great answer here. While there is a [LastCreateRecordIdentity] to get the last id created, I don't see anything for a record count. So, you have to run a for-each record and increment a counter value. Note again the for-each record works just like the above lookup and the nested code runs for each reocrd.

 

I suppose you could also setup a trigger that maintains the count of all reocrds. However that idea would ONLY work for total table count and not your question about a given condition. So for-each is about the best bet I can see.

Hi,

If you have a list of file to import, then it may be simpler just to avoid the UI altogether.

Sub SampleTest()
Call InsertAudio("G:\Music\track1.mp3", ActivePresentation.Slides(1))
Call InsertAudio("G:\Music\track2.mp3", ActivePresentation.Slides(2))

End Sub

Sub InsertAudio(Track As String, oSlide As Slide)
Dim oShp As Shape
Dim oEffect As Effect

'Add the audio shape
Set oShp = oSlide.Shapes.AddMediaObject2(Track, True, False, 10, 10)

'Set audio to play automatically
Set oEffect = oSlide.TimeLine.MainSequence.AddEffect(oShp, msoAnimEffectMediaPlay, , msoAnimTriggerWithPrevious)
oEffect.MoveTo 1

'Hide during slide show
With oEffect
    .EffectInformation.PlaySettings.HideWhileNotPlaying = True
End With

End Sub
