bill manville say:

Oh dear.  That is messy isn't it?

My approach would be to write a macro to clean up the data, putting the account number and Account balance into a new worksheet from where you could do the VLOOKUPs quite easily.

Let's say the text-to-columns data is in a sheet named "Dirty" and we have a sheet named "Clean" which has A1="Account" and B1="Balance"

Sub CleanUp()
  Dim rClean As Range
  Dim rDirty As Range
  Dim iOffset As Integer
  Dim stBalance As String

  Do Until rDirty.Row > Worksheets("Dirty").UsedRange.Rows.Count
    If rDirty.Value <> "" Then
      ' an account number
      rClean.Value = rDirty.Value
      iOffset = 0
      Do Until rDirty.Offset(iOffset, 1) = "Account Balance"
        iOffset = iOffset + 1
      Loop
      stBalance = rDirty.Offset(iOffset, 2)
      stBalance = Trim(Replace(Replace(stBalance, "$", ""), Chr(160), "")) ' there are non-breaking spaces (character 160)
      rClean.Offset(0, 1) = Val(stBalance) ' convert to number
      Set rDirty = rDirty.Offset(iOffset)
      Set rClean = rClean.Offset(1)  ' ready for the next one
    End If
    Set rDirty = rDirty.Offset(1)
  Loop
End Sub

You have to accept the changes first, then turn track changes off.
