how to Highlight the inserted text along with font colour through - Microsoft Community

i-ingenuous-i say:

how to Highlight the inserted text along with font colour through Track changes insertion option

I want to highlight the text which I type through track changes insertions option. I just found colour,  strike through or its similar option but my intention is to highlight the along with colour. please help me
jay freedman say:

As you found already, Word doesn't offer highlighting as part of the Track Changes markup (except for table cells). Instead, you need a macro to apply highlighting to insertions after they're made. This will do it (and you can run it as often as needed).

Sub HighlightInserts()
   Dim wasTCF As Boolean
   Dim rev As Revision
   
   With ActiveDocument
      wasTCF = .TrackFormatting
      .TrackFormatting = False
      For Each rev In ActiveDocument.Revisions
         If rev.Type = wdRevisionInsert Then
            rev.Range.HighlightColorIndex = wdYellow
         End If
      Next
      .TrackFormatting = wasTCF
   End With
End Sub
