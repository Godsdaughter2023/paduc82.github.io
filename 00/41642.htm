Align Objects in 3D Builder

Hello, 

i am wondering how to align objects in 3d builder exactly , e.g. different circles should have the same center 

Thanks in advance

louis tl hong say:

Hi,

The easies way to align object is to use the "Move" tool (keyboard shortcut R). When you select an object and then click the "Move" tool, the numeric value shown in the tool is the center of the object. For instance, in the screenshot below, the tall cylinder is selected, and its center is at (0,0), so the value 0 is shown in the tool. This is true when you click either the x-axis arrow or the y-axis arrow (the green arrow is the one currently picked). To align the tall cylinder with the short cylinder, you just need to make sure that the numeric values of the two models are the same when you click the x-axis arrow and y-axis arrow.

In addition, 3D Builder does automatically snap objects so that their centers are aligned. Try dragging the x-axis arrow of one of the cylinders slowly, and you should see the cylinder snap to the center as you get close to it.

The snapping option can be changed by going to the top-left corner menu and then go to Settings.

davem121 say:

Start your Laptop, then just after the Manufacturers logo disappears and Windows 10 tries to boot, press and hold the power button down for 5 - 10 seconds to perform a hard shut down

Do this Twice

On the third boot attempt, Windows will boot into the Windows 10 Repair Environment, from there you can access Startup options -  Safe Mode

If your system can boot into Safe Mode, try uninstalling the latest Update

If you cannot boot into Safe Mode, from the Recovery Environment, try to see if you have a restore point in System Restore that pre-dates the latest update
