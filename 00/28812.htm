andreas killer say:

Right-Click the sheet tab and paste in all code below.

Save the file as XLSM

Andreas.

Option Explicit

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim R As Range
  Dim Value
  'Reduce to the used range
  Set Target = Intersect(Target, Me.UsedRange)
  'Exit if outside
  If Target Is Nothing Then Exit Sub
  'Events off, otherweise we call ourself
  Application.EnableEvents = False
  'In each changed row
  For Each R In Target.EntireRow
    'Get the date from B
    Value = Range("B" & R.Row).Value
    'Be sure
    If IsDate(Value) Then
      'Only till that date
      If Date <= Value Then
        'Update A
        Range("A" & R.Row) = Date
      End If
    End If
  Next
  Application.EnableEvents = True
End Sub

ossiemac say:

The letters and numbers are displayed when you press the Alt key. The letters and numbers tell you which key to use in conjunction with the Alt key as a short cut to the function that the icon they are against would normally invoke.

With the other problem it appears similar to what you get if the worksheet is protected. You can determine if the worksheet is protected by selecting Home ribbon -> Format (Located in the Cells block which is the second last block from the right of the ribbon) and in the resulting dropdown if the worksheet is protected then there will be an option to Unprotect. If not protected then the option displayed is Protect.

You will possibly need a password to Unprotect although it is possible to Protect without a password in which case a password is not required to unprotect.
