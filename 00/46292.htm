I found a way to fix DRM playback problems in Windows Media Player. This works on Windows 7 Media Player 12.

If you purchased a video or song you may need to enable DirectX Video Acceleration.
Another thing is that you want to make your paging space at the recommended setting, for both minimum and maximum to be the same value. When it shows you the max, make the minimum match the highest number in maximum. This will insure that your pc is running smoothly and gives the files a place to process from during playback.

To Fix Problems with VLC, streaming or other slow playback issues. First start VLC.
Next do Ctrl-H to show the options at the top of the application, Media tab, File tab, Show More options, Change Caching from XXXms if is lower then 500ms to 500ms.
If having trouble with the network, dic, or capture device do the same. Adding more delay is building a cache to work from.  

To Fix problems with Windows Media Player playback do the following.
Open Windows Media Player.
Right Click the center of the screen in it.
Select More Options.
Go the the performance tab.
Detect Connection Speed - Enabled
Buffer -Enabled and set to10 seconds
Use Video Smoothing - Enabled
Display Full-screen controls - Enabled
Turn on DirectX Video Acceleration for WMV Files - Enabled  =  very,very,very important.
High Def videos will have trouble if this isn't turned on.
