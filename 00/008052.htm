I suspect the problem is in those short, continuous sections you created for the newspaper columns. doug robbins - mvp office apps & services (word) say:

Select the date entered by the date picker and add a bookmark to the document (for my test of this, I just used date as the name of the bookmark.  Then in the footer, insert a cross reference to the text of the bookmark and after toggling on the display of the field codes, add \@ "M/d/yyyy" inside the closing } so that you have

{ REF date \h \@ "M/d/yyyy" }

You can delete the \h if you want to as having the cross reference hyperlinked to its source is of no value when the field is in the footer of the document.

Note that if the date picked changes, the cross references field will need to be updated and that can be done by doing a print previewi of the document.

stefan blom say:

If page numbers are being restarted following a section break, Word will add "filler" pages (if necessary) so that a right-hand page will always be odd-numbered. This occurs even if you have cleared both "Different odd and even" and "Mirror margins" in the Page Setup dialog box (or the corresponding options on the Header & Footer Tools Design tab on the ribbon).

Using the options for Page Setup or Page Number Format, you can't get around this behavior, which is by design.

However, you could see if the article at https://wordmvp.com/FAQs/TblsFldsFms/InsEvnPgEndChap.htm will do what you expect.

suzanne s. barnhill say:

Word adds a blank even page only if the next section starts with an odd page and the previous section ends on an odd page. This can be (as Stefan and Doug have pointed out) either because you have inserted an Odd Page section break or because you have restarted numbering at 1 (or an odd number), in which case Word will treat a Next Page break as Odd Page. It does this because it sees you are planning to print on both sides (because you've enabled Different Odd and Even), and this is what you want it to do. For a book, you should also have Different First Page enabled, and the First Page Header should be blank (you can have a page number in the footer if desired).

But none of this will cause Word to add a blank page unnecessarily; it will happen only to present two odd pages in succession, which would spoil the book.

To remove the current product key, you can follow the steps below:

 

1. Sign out of the Office applications and close all of them.

2. Windows key + R  -> type cmd to open the command prompt window

3. Input the following command:

cd C:\Program Files (x86)\Microsoft Office\Office16. If you can't find the path, type cd C:\Program Files\Microsoft Office\Office16  

cscript ospp.vbs /dstatus

cscript ospp.vbs /unpkey:xxxxx (copy the last five digits of the product key)

4. Restart computer and reactivate Office.
