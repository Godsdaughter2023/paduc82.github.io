If you hyperlinks are to bookmarks, or if you insert a bookmark at the target of the hyperlink, you could accompany the hyperlink with a cross reference to the page of the bookmark, such as

on page { PAGEREF [bookmarkname] \h }

That will display the number of the page on which the bookmark is located and it will update (Ctrl+A the F9) if editing results in the page number of the page on which the bookmark is located changing.

There are other things to which a { PAGEREF } can be applied via the References>Cross-reference facility.
