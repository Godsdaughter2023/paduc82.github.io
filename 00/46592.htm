Microsoft is aware that previously-hidden partitions are improperly being assigned drive letters after installing the April 2018 Update. As a result, they now appear in Disk Management and File Explorer. These partitions are typically recovery partitions used by the Windows Recovery Environment or by the Original Equipment Manufacturer. It is not recommended that you attempt to delete or reformat these partitions as this may have adverse effects on your ability to recover Windows. Microsoft is currently working to address the issue in a future Windows Update, and we’ll update this thread when a solution becomes available.

(Update 5/14/18: The solution that will soon be released via Windows Update will only prevent this issue from occurring for devices that install the April 2018 Update after the solution is released. If you've already installed the April 2018 Update and are experiencing this issue, you'll need to follow the steps below if you want to hide these partitions.)

Although you can safely ignore these partitions, if you prefer do away with the low disk space warnings and hide the partitions so they no longer appear in File Explorer, you can do so by manually removing the partition’s assigned drive letter. To do this:

1. In Cortana, type cmd and press Enter. Then right-click Command Prompt in the search results and select Run as administrator.

2. In the Administrator: Command Prompt window, type diskpart and press Enter.

3. Type list volume and press Enter.

4. Make note of the volume number for the drive letter you want to remove. For example, if the drive letter you want to remove is Drive “G”, note the volume number for Drive G.

5. Type the following command, and press Enter.

select volume <volume number>

When typing this command, in place of <volume number>, substitute the volume number you made note of in Step 4. In the example above, if the volume number for drive G is 5, you would type: select volume 5

6. Type the following command and press Enter.

remove letter=<drive letter>

When typing this command, in place of <drive letter>, substitute the drive letter you want to remove. In this example, you would type: remove letter=G

7. Close the Command Prompt window.

Workarounds and solutions for common Windows 10 issues and complaints - Microsoft Community

deadlysensi say:

Workarounds and solutions for common Windows 10 issues and complaints

The following is various tricks I've learned by trial and error and just sheer persistence. This takes some of the most common issues and gripes people have with Win 10 atm and offers valid solutions that I've used myself. Please feel free to add to this or correct anything posted here.


Issue 1) Windows update runs constantly without my consent or input. I am not comfortable with this. How do I stop windows update and only run it when I want to?

Solution 1. Disable windows update completely)
There is a simple way to do this. It works as follows.
-Search for "Services" or got start menu/all apps/windows administrative tools/services
-Scroll down until you find a service called "Windows Update". It is set at "enabled" by default.
-Change this service setting to "disabled."

That's windows update fully disabled. You can re-enable it at any point you wish and check for updates. You can alternate between modes at any point to suit yourself.

Solution 2. Alter group policy and change the windows update rules)
-Search for and run "Edit group policy"
-In the left navigation bar, follow the path Computer configuration/administrative templates/windows components/windows update
-In the right policy page, select Configure Automatic Updates
-You will have to set to Enable and the old options for windows update become available in the dropdown box. Ie they un grey out.
This allows you to set to "Notify for download and notify for install", "Auto download then notify for install", "Auto download and schedule install" and "Allow local admin to choose setting". You can also set the scheduled update time.


Issue 2) Win 10 by default is mining and passing on far too much personal information about me and my activities. I am not comfortable with it. How do I take control of this?

Solution)
-Go start menu/settings/privacy
-Adjust all the settings in this control panel to suit yourself. Pay particular attention to the general tab, the location tab, the message tab, the contacts tab and don't forget to opt out online for Microsoft's targeted advertising. Also unless you want to participate in feedback or Cortana's important to you, set diagnostics and feedback to basic.

Please note that disabling some of these options will disable certain features such as Cortana. It's really down to yourself what you are comfortable with. If that solution still isn't enough. If you are hyper paranoid and need more then post in the reply's. I'll show you what services to disable and which .exe's to block in your firewall that will ultimately permanently block all data mining.


Issue 3) I am trying to install drivers/software but I am getting error messages saying the publisher is untrusted or untrusted source. I have tried to remove the publisher from the untrusted publisher list in internet explorer but it is empty. How do I install this software?

Solution)
This is a workaround for this issue.
-Copy the installer to the root of your c drive
-Rename it something simple like "installer.exe"
-Open a command promt in admin mode
-Navigate to the root of your c drive and execute the installer by typing it's name. Ie "installer.exe"
That gets around the block and will install the software you want.


Issue 4) I am trying to install a modified/custom/unofficial driver but it fails with errors just as it's about to install the actual driver files.

Solution)
The install is being blocked by driver signature verification. You need to disable it for a single boot to allow the driver to install. Here's how.
-Press and hold in the shift key. At the same time, go start menu/power/reset. Keep the shift key held in untill the Update and Recovery section loads up
-Select Troubleshoot
-Select Advanced Options
-Select Startup Settings
-This will reboot into the old familiar safe mode selection screen you used to get when f8'ing on the way in to older windows. One of the options is Disable Driver Signature Verification. Select it
This will start windows with Driver Signature Verification disabled for one boot only. Your driver will now install and will not be blocked.


Issue 5) Windows 10 is running slower than 7/8 with games that use alot of vram/Windows 10 is using far to much vram for the desktop and there is no "Disable visual themes" or "Disable desktop composition" options in the compatability tab for my games.

Solution)
To get windows to significantly reduce the amount of vram it's using for desktop, you are limited in options. About the only thing you can do is adjust the advanced performance settings.
-Search for "Adjust the apearance and performance of windows"
From there it's down to you. I personaly disable everything bar "Smooth edges of screen fonts" and "Use drop shadows for icon labels on the desktop". This reduces windows vram usage by almost half and frees up well over 100mb of vram for games. Trust me it makes a massive difference :)

What is Driver Verifier?

Driver Verifier is included in Windows 8/8.1, 7, Windows Server 2008 R2, Windows Vista, Windows Server 2008, Windows 2000, Windows XP, and Windows Server 2003 to promote stability and reliability; you can use this tool to troubleshoot driver issues. Windows kernel-mode components can cause system corruption or system failures as a result of an improperly written driver, such as an earlier version of a Windows Driver Model (WDM) driver.

Essentially, if there's a 3rd party driver believed to be at issue, enabling Driver Verifier will help flush out the rogue driver if it detects a violation.
