john p spencer say:

Shot in the dark.

Close the database.

Check the directory (folder) that contains the database and look for a small file with the name of your application and the extension of .ldb.  If the file exists, try to delete it.

The NameOfDatabase.ldb file is the lock file and normally deletes when the database application is closed.  However, the file may not be deleted if the database application is abnormally terminated or if a user working with the data does not have delete privileges on the folder.

If you cannot delete the lock file, then check your privileges on the directory.
