how to convert classic site to modern team site - Microsoft Community

a. user say:

how to convert classic site to modern team site

I need to associate a previous created classic site with the hub site, so want to convert it to modern team site, how to do it? Thanks
cliff gu msft say:

Dear Jack,

 

As the classic team site and the modern team site are different and some customizations on the home page of the site may not be integrated, I suggest you not convert it and you may not need to do that if you just want to associate it with a hub site. The classic site can also be associated with a hub site via going to the modern SharePoint admin center>Active sites>finding the classic team site>clicking on Hub site>selecting Associate with a hub site.

 

As the hub site link can’t appear on the classic team site home page, you may need to click on Documents and find it on the modern library page. However, if you don’t make some customizations on the classic site home page and you are an administrator, you can convert the home page to a modern home page via PnP PowerShell.

The steps to do that are the following.

1. Install SharePoint Online Management Shell.

2. Open SharePoint Online Management Shell with the administrator right.

3. Run Install-Module -Name PowerShellGet –Force to install PowerShellGet.

4. Run Install-Module SharePointPnPPowerShellOnline to install PnP PowerShell.

Connect to SharePoint Online via the following command.

$adminUPN="<the full email address of a SharePoint administrator account, example: jdoe@contosotoycompany.onmicrosoft.com>"

$orgName="<name of your Office 365 organization, example: contosotoycompany>"

$userCredential = Get-Credential -UserName $adminUPN -Message "Type the password."

Connect-SPOService -Url https://$orgName-admin.sharepoint.com -Credential $userCredential

5. Copy and paste the following scripts (change the site URL and the page name accordingly) to Notepad and then save it as a .ps1 file to a folder on your computer. Generally, the Home page is Home.aspx.

# Connect to the classic site with pages to modernize 

Connect-PnPOnline -Url https://contoso.sharepoint.com/sites/teamsite 

 

# Modernize page1.aspx and add the page keep/discard banner on the page 

ConvertTo-PnPClientSidePage -Identity page1.aspx -AddPageAcceptBanner 

6. Type cd <the folder path> in SharePoint Online Management Shell.

7. Type ./<ps file name>.ps1 to run the script.

 

After running it successfully, a new modern Home page such as Migrated_Home.aspx will be created in the page library and you can make it home page to show the hub link in the classic team site.

 

If you make some customizations on the home page, the customizations will not be shown on the converted modern home page. In this situation, I suggest you use the classic home page or degrade it as normal page and create a new modern page directly and make it home page.

 

Best Regards,

Cliff

Using Visual Basic for applications, you can write any code/note you wish.  The challenge becomes defining what problems you are looking for, and then how to notify the user.  I have written an Add-In for MS Project 2007 and later that makes over 30 tests on the schedule and then displays in the flag or text field of your choice. It also will put it into Excel very nicely.  Not freeware, just letting you know what is possible with code ...

What you can do quite quickly and easily is insert the column "Status" or "Status Indicator" and it will let you know if a task is behind schedule, etc.  That comes as out of the box behavior.

Now to get a bit more robust.  Suppose you are looking for detail tasks that have a finish date that has slipped from the baseline.  You can set a formula in any of the 20 custom flag fields iif([Finish Variance]>0, "yes", "no") which will set the Flag to "yes" of there is a positive finish variance.  You can change the 0 to 5 or whatever is a desired threshold.  Then, rename the Flag field to be something like "Late Finishes" and that becomes your note.  Since there are multiple custom text, flag, and number columns, you have a lot of places to store your results.

To create the formula, insert the field of your choice into a view.  Right click on it (say Flag1) and select "Customize Fields"  In the dialog box, you can set a formula.

Finally, after customizing the Flag field, you can use a display indicator.  The ability to display a flag, circle, dot, triangle, square, etc. of various colors is a good way to bring attention to a specific set of tasks (you can even filter to it, ie: Flag1="yes").

To get much more complicated than this will require the use of code.
