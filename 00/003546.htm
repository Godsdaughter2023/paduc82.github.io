daniel g. grau say:
You need a preemptive behavior, because tracking changes can quickly tax Word out of resources. This would consist of saving, closing and reopening your document before the problem occurs (i.e., before your 3 problem-free hours have elapsed). A similar effect can be obtained through Save As (no closing needed), but you'll need to number the versions orderly.

Do enable Word > Preferences > Save > Always create backup copy only or that document*. It's a lot safer than AutoRecover, in the sense that you get a fully-functioning version of the file every time you save (mind you: no save, no backup). The backup copy is one version behind, which ensures that if the current version screws up, the file named "Backup of..." will contain the previous version, minus the steps that led to the screwup.

AutoRecover does not work if your file is closed normally, because Word will eliminate all the recovery files, since they are no longer needed. It only serves its purpose if Word freezes, which you never want, so the best course of action is to prevent this from happening.



* I say "only for that document" because if you leave it on, pretty soon you'll find your disk littered with "Backup of..." files.

jllatham say:

This is due to the speed of today's systems: the userform doesn't get a chance to be fully drawn before you get into your loop.  That is dealt with using the UserForm1.Repaint instruction.

Next issue would be that because the loop is so tight, the system never gives you a chance to click that checkbox.  The DoEvents statement takes care of that (tells VB to do system processes in the background so that things like you clicking the checkbox can take place and be noticed).

Sub Main()
  Dim i As Long
 
  UserForm1.Show ' property set to non-modal
  UserForm1.Repaint ' make sure it is all visible
 
  Do
    'changed to give us time to test click
    For i = 1 To 1000
      DoEvents
      'Cells(i, 1).Value = i
      'changed so we can watch the value easily
      Cells(1, 1) = i
    Next i
  Loop Until UserForm1.CheckBox1.Value = True
End Sub
