No you can't and a question that comes up repeatedly.

The Windows 8 install was licensed by Acer as a major systems manufacturer and is authenticated with the motherboard BIOS, this license is only for Windows 8, not transferable and locked to the system. This is why, Robert, the manufacturer cannot give you an 8.1 key as you would then have 2 licenses and the motherboard would not recognise it.

In order to install Windows 8.1 directly you have to have a Windows 8.1 key, the 8 one is not valid. If you were to Reset the system it has to be returned to 8 and the 8.1 upgrade applied again.

You could "trick" that download page which may still work. Go through the dialogues to download Windows 8 (as you have the key) and then abandon the download. Restart the process again and the system should recognise that you have already authenticated the download but let you select the 8.1 ISO image instead.

you're very close.  Here's the code I use to rename all files (in this case, it removes all - hypen marks out of the file name.

Get-Item .\*.* | ForEach-Object { Rename-Item $_ ($_.Name -replace "-", "") }
