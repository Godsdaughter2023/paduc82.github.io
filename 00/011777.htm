hansv mvp say:

You could create the following custom function in a module in the Visual Basic Editor:

Function Get10Digits(s As String) As String
    Dim re As Object
    Dim m As Object
    Set re = CreateObject("VBScript.RegExp")
    re.Pattern = "\d{10}"
    Set m = re.Execute(s)
    If m.Count > 0 Then
        Get10Digits = m(0)
    End If
End Function

In a cell formula:

=Get10Digits(A1)

where A1 is a cell containing a string as in your example.

In VBA:

    Dim strInput As String
    Dim strOutput As String
    strInput = "xyz 4257 yht 1254879652 45875"
    strOutput = Get10Digits(strInput)
    Debug.Print strOutput
