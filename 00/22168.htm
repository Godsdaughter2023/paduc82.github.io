CD/DVD DISK SPEED FOR STORING WORD/EXCEL FILES - Microsoft Community

ronaldlegner say:

CD/DVD DISK SPEED FOR STORING WORD/EXCEL FILES

What is the best/only CD or DVD type and speed that I should get/have to store the subject files???  Where should I have been looking to find this information???
jllatham say:

OK, the 8x, 16x ... 52x refers to the maximum speed that it can record (be written to) at.  So if the speed is at least what your burner can burn at, then it would be optimum.  You'd have to check your computer's specs to see what that speed is.  But even if you buy 52x and your burner can only burn at a maximum of 16x, that's no problem at all.  Most good disk burning software can even test to see how fast it could burn given the specific combination of burner and disk - and it will vary with almost every burn.  Again, no big deal.

Possibly the most important thing as far as using them to archive files is the quality of the disk.  Stick with good names like Maxell, Ritek, Ridata, Verbatim and others that you might be familiar with or are on the shelves at your local Staples, Office Depot or Office Max.  I try to stay away from "house brands" myself since their source is pretty much unknown.  A good source of these things is through this site:

http://www.supermediastore.com/

Although not always the lowest price, they are a reliable and trustworthy site with good pricing on their entire line of products.  If you need a big bunch of disks, they can sell you bulk packages easily.

Generally these days all burners support -R type disks.  The -R/W are 're-writable' disks that you can save later versions of a file to a limited number of times.  These can be problematic - I'd stay away from them, plus not all CD players can read that format because the "session" is never finalized or "closed".

Remember that a CD can hold about 700 MB of data, a DVD can hold about 4.7GB (about 6 CD's worth) of data.  CDs in bulk run about $0.25 or so each, while DVD's in bulk only run a few cents more.  On a 'per byte of storage' ranking, DVDs are much less expensive than the number of CDs needed to store data that won't fit on a single CD.

Actually, you've probably got much of the info you need already right here?  But you can find some more about both CDs and DVDs from a practical standpoint at that site I mentioned:

http://www.supermediastore.com/category/u/blank-recordable-dvd-r-disc-discs-media

http://www.supermediastore.com/category/u/blank-recordable-cd-r-rw-disc-discs-media

You're possibly going to want to know about the various label-side surfaces:  some disks have the non-data surface (the label side) made so that you can (with the proper software on your computer) 'print' the label in your CD burner: you use that software and put the CD/DVD in it upside down for that operation; these will be marked as LightScribe type disks.  There are also disks that you can print on with either an ink-jet printer or thermal printer - the printer has to be able to deal with them (and thermals can't be printed with an ink jet printer, and those made to work with an ink jet printer won't work with a thermal printer).  Epson makes several inkjet printers that can do this - I've used various models from them over the years with just outstanding looking results.  They come come with a piece of software called PrintCD that allows you to design the layout of the label and print on the disk without spraying ink all over areas that don't exist on the disk.  Then, of course, there are disks without any special surface on the label side - you can print on paper labels with sticky backs and put those on the disks, or simply write on that side of them (never write on the data side) with a felt-tipped permanent marker like a Sharpie.  I don't recommend using paper stick-on labels: they can tear and shred during use and really mess up the insides of your disk drive.

graham mayor say:

Buy the discs that are compatible with your writer. The speed is of secondary importance.

Do not use this type of medium for direct access from Word or Excel. Always copy too and from the hard drive. Use only for backup purposes.

Note that CD/DVD is a write once medium. There are applications that enable such discs to be used as a 'big floppy', but they are notoriously flaky and I wouldn't entrust my files to such a process.

You would be much better using flash media. You can get an enormous amount of data on an SD card and if your PC is not compatible with such media an external card reader will not break the bank.

The 2 features you mention are actually one in the same -- simply different methods of imposing scaling of the content. The Fit To option allows Excel to scale the content as necessary whereas the Scale value allows you to specify the preferred percentage yourself. [Pursuant to that, Fit To adjusts Scale value but entering a Scale value does not activate (i.e., check the box for) Fit To settings.] For example, I have a file which is too wide to fit on a single sheet of paper based on paper size & margin settings. If I use Fit To for 1 page wide x 1 page tall Excel automatically sets the Scale value to 83%. If I clear the Fit To check Excel sets the Scaling value back to 100%. IOW, Fit To is a convenience feature for Scaling the Print Area.

Additionally, Scaling is worksheet specific. It is neither a workbook setting nor a global Excel preference. Each sheet in a book can have a different scaling percentage based on the volume of content in its Print Area & the needs for that particular sheet. Further, the values set are saved with the workbook file, so each time the file is opened each sheet's Layout specifications are as they last were, assuming the workbook was saved after the adjustments were made.

Sub AutoOpen() '   Written by Paul Epstein

    Dim TOC As TableOfContents
    For Each TOC In ActiveDocument.TablesOfContents
      TOC.Update
    Next
End Sub
