mariuspetrescu say: These behavior has the following cause:
- There is a new network adapter in the system created by BlueSoleil, the "Bluetooth PAN Network Adapter".
- The adapter is in an new network, acting also as DHCP server (this is the default behavior)
- That network is not identified and appears as "Unidentified Network" public network in the network and sharing center
- If no other PAN device is connected, the status of that network device is "media disconnected"
- Now since the network icon in the task bar shows a global network status and one of the network adapters has its media disconnected, you will get the red cross over the network status in your taskbar.

So nothing to worry.
Try connecting a PAN device to your bluetooth adapter and the cross will go away.

Anyway, an option in windows not to monitor a network adapter media status would be nice...

We receive this error message when you are trying to update driver from Windows update and installing a driver that is already installed on your computer or a driver that has a lower compatibility rating than one you already have installed.

http://support.microsoft.com/kb/952032

i.                     Click Start, type devmgmt.msc and hit enter.

ii.                    On the “Device Manager” window, expand the item, right click on the sub item and then click “Update Driver”.

iii.                  Restart the computer once the driver is updated.

i.               Click Start, type devmgmt.msc and press enter.

ii.                     Expand the driver and right click on the sub item and click uninstall.

iii.                   Restart the computer.

iv.                   Visit the manufacturer website to download the latest driver.

v.                     Install the driver and restart the computer for the changes to take effect.
