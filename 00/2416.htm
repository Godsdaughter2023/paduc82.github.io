bernie deitrick say:

First off, use the IFERROR function instead of nested  IF and ISERROR

=IFERROR(AVERAGEIFS(Sheet2!J$5:J$1045, Sheet2!$B$5:$B$1045,$A$3, Sheet2!$E$5:$E$1045,$B3)),0)

For the Median IFS, you need to array-enter this formula, using Ctrl-Shift-Enter instead of just enter:

=IFERROR(MEDIAN(IF(Sheet2!$B$5:$B$1045=$A$3,IF(Sheet2!$E$5:$E$1045=$B3,Sheet2!J$5:J$1045))),0)

nathan bloomfield say:

Thanks Bernie!

Should the same function work for "Mode"?  I tried and im getting a 0 result? For the same formula but with MODE instead of MEDIAN.

Thanks again!

bernie deitrick say:

It will work - just make sure that you array enter it.  If that formula returns 0, it means that there is no MODE - which is a repeated value.  MODE works on unrounded numbers, so you may want to round the values to get a mode:

=IFERROR(MODE(IF(Sheet2!$B$5:$B$1045=$A$3,IF(Sheet2!$E$5:$E$1045=$B3,ROUND(Sheet2!J$5:J$1045,0)))),0)

tom ogilvy say:

Mode can only work if there are truly duplicate values that meet the criteria.  (It would return a #N/A in that case and your iferror would return the zero)  

If there are true duplicate values that meet the criteria, then I would expect mode to work the same as median.  (did in some light testing i did)

On 22 June 2011, Nokia made an agreement with Accenture for an outsourcing program. Accenture was supposed  to provide Symbian-based software development and support services to Nokia through 2016; about 2,800 Nokia employees became Accenture employees as of October 2011. The transfer was completed on 30 September 2011. Although support was promised until 2016, there was little development from Accenture, where most Symbian developers had already left by 2012. In January 2014, Nokia stopped accepting new or changed Symbian software from developers.

Nokia terminated its support of software development and maintenance for Symbian with effect from 1 January 2014, thereafter refusing to publish new or changed Symbian applications or content in the Nokia Store and terminating its 'Symbian Signed' program for software certification.

Lumias will be supported as announced, depending on which OS they are running.
