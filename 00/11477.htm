The place to create a custom bullet is in a Style. The bullet styles listed under the Bullets button that you circled are under program control and can't be changed in a reliable way. The styles you create should be saved in a template file (not Normal.dotm) and the template file should be distributed to users. The Custom Office Templates in each user's Documents folder is a good location for a custom template. As an alternative, you can set up a Workgroup Templates folder on your server, so each user can access the template from a central source.

To prevent users from applying the default bullets, you can protect the template to allow Styles (Review>Restrict Editing>Check Limit formatting to a selection of styles>Choose styles to be allowed>Click on Yes, Start Enforcing Protection). Then user can't use the default bullets, only your bullet style.
