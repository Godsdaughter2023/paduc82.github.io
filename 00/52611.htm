Hi,

The Error code 1935 when installing a software on Windows 10 occurs when the Windows file system transaction log is corrupted upon installation. To resolve your concern, you need to clear the log by following the steps below:

  1. Press the Windows key + X on your keyboard.
  2. Select Command Prompt (Admin).
  3. Type the following command: fsutil resource setautoreset true C:\
  4. C:\ is the drive where Windows 10 is installed. If it is installed on a different drive, kindly change it to the appropriate drive letter.
  5. Restart your computer for the changes to take effect.

adrianne buc say:

Hi Ella,

Upgrading from an older version of Windows might have caused some issue with the previously compatible CD/DVD drive drivers. To correct this problem, we suggest following these troubleshooting steps:

Reinstall or update your driver from the Device Manager

  1. Open Device Manger by searching it from the Start menu or Cortana.
  2. Click the drop-down arrow next to DVD/CD-ROM drives to reveal the existing optical drives.
  3. Right-click on your optical drive and select Uninstall
  4. Restart your PC after uninstalling the optical drive. Windows will automatically detect the drive then reinstall it. You can also use the option Update Driver Software from the Device Manager.

Should the issue persists, we suggest removing the virtual drive software if you have any then repeat the steps above.
