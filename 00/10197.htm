dr. strangelove say:

Windows Defender, as you noted, is disabled by MSE on Vista.  This is correct, as there is no need for Defender.

To control and manage startup programs, you can go to START, RUN (or START, SEARCH) and enter MSCONFIG.

To temporarily disable MSE to download programs, which you should NOT need to do, can be accomplished by opening MSE, settings tab, select real-time protection, and UNCHECK "Turn on realtime protection".  Just remember to turn it back on.

Windows update will now provide definition updates for MSE, as well as MSE doing them automatically, or you doing them manually (by opening MSE, update tab, and clicking on UPDATE). 

Verify that you are using the latest build of MSE, which is 1.0.1961.0.

Just move all this code into the form's Timer event. The only thing that the Form_Open event needs to do (for this purpose)  is

Me.TimerInterval = 60000

This will cause the form to call the Timer() function once a minute.

You can simplify the If statement your code to

If Weekday(Date()) = vbMonday And Time() > #12:55:00 PM# Then

Right before the Exit Sub (which is actually not needed, since there's nothing after it) put a line

Me.TimerInterval = 0

 

The code will only fire once a minute; if it hasn't reached 12:55 it won't do anything, but the next time it's called thereafter it will run your report and then shut itself off.
