I'm sorry, I'm having difficulty visualizing the display you're seeing.

However, I'd just say that if you imported a lot of folders and messages, it will take some time for Windows Live Mail to index them all. The message list is simply a set of entries from the message database index, and until the index is complete, it will not necessarily be an accurate representation of the messages actually residing in a particular folder. When you select one of the messages, the program tries to find the message you selected to display its contents. This may produce unpredictable results if the index is incomplete. It's as well to allow the program to get on with its housekeeping undisturbed - e.g. overnight - before using the program.

I would suggest that you go through this procedure, e.g. before bedtime:

  1. In Windows Live Mail, press Ctrl-Shift-O for Options and on the General tab deselect Send and receive messages at start-up and Check for new messages every n minutes. This is to avoid any disturbance to the indexing process caused by the arrival of new messages.
  2. Press Ctrl-Shift-Escape to open Task Manager. You may have to adjust the size of the window to be able to see the lists of processes. Look for Windows Live Mail (32 bit) in the Processes list.
  3. Close down Windows Live Mail and wait until Windows Live Mail (32 bit) has disappeared from the Task Manager Processes list.
  4. Re-launch the program and leave it alone for several hours, e.g. go to bed!

By morning, the program will most probably have finished indexing all those new files. Have a look and then come back here and tell us what you found.

a. user say:

It's hard to make a firm diagnosis without knowing the full text of the message you are seeing.

In general, when a web browser tells you that your connection is not secure, it means to say that the web page you are trying to view cannot establish a secure connection with your web browser. That's because the owner of the web page has not arranged to have a secure connection. There's nothing you can do about that. If the connection between the web page and your browser isn't secure, an attacker can observe everything you are sending to and receiving from the web page and tamper with it.

The real solution to this is for the web page owner to ensure that your connection is secure from the very first second that you connect. Otherwise, there may be methods that setup a secure connection after your web browser has already connected to the web page, and that's what you may be seeing.

a. user say:

Hello,

Thank you for posting your concern here in Microsoft Community and welcome to the Windows 10 family. I would be happy to assist you.

From the description provided, I understand that you want to sync the lost calendar and contact files in Outlook from iCloud.

I would suggest you to follow the below steps and check if it helps to restore the lost contacts.

Export iCloud contacts to vCard

1. Sign in icloud.com.

2. Select Gear icon to bring up settings.

3. Select “Select All” or press and hold “Shift” and select the contacts you want to export

4. Select “Export vCard.”

5. Ensure “Save File” is selected and select “OK” to export.

Your iCloud contacts are saved as a vCard on your computer.

Import the vCard file into Windows Contacts

1. Open Windows File Explorer.

2. Enter c:\users\%username%\Contacts into the address bar to open the contacts folder.

3. Select Import > vCard > Import.

4. Browse to the location of the vCard file that you exported from iCloud.

5. Select “OK” for every contact.

Export contacts to CSV file

1. Select Export > CSV (Comma Separated Values) > Export. 

2. Click “Browse” to select the location for the file to be exported.

3. Enter a name for the file and select “Save” to create the CSV file.

4. Select “Next” to select the fields to export.

5. Select the fields to export.

6. Select the fields that you wish to import and click on “Finish” to export as a CSV file.

7. Select “OK” and “Close” to export your contacts into a CSV file.

Import contacts Microsoft Outlook

1. Open Microsoft Outlook.

2. Select File > Open & Export > Import/Export.

3. Select “Import from another program or file” and “Next.”

4. Leave “Comma Separated Values” as the selection and “Next.”

5. Select “Browse” to locate the file to import.

6. Select the CSV file that you have just exported from Windows contacts and click “OK.”

7. Choose how you want Outlook to handle duplicates, then click “Next.”

8. Select the destination folder and “Next.” (Most likely “Contacts.”)

9. Make sure that the CSV file is selected, and click “Finish.”

Microsoft Outlook imports your contacts. The amount of time it takes depends on how many contacts you import. Once done, all of your exported iCloud contacts appear in Outlook Contacts in the People tab.

To Import or subscribe to a calendar through Outlook.com, refer to the below link.

http://windows.microsoft.com/en-in/windows/outlook/calendar-import-vs-subscribe

Hope the above information was helpful. If you need further assistance, please do let us know and we would be happy to help you further.

Thank You.

A lot of problems arise from setting Windows Update to automatically update. You may prefer to change this setting. Select Start, All Programs, Windows Update, Change Settings. Under Important Updates is a box showing the current setting. Click on the Arrow down to the right and change the selection to “Check for Updates but let me choose whether to download and install them” and click on OK.

 

If you encounter problems updating or you are not using the Automatic Update you should manually update by selecting Start, All Programs, Windows Update, Check for Updates. Check the boxes before the updates you want to install, click on Install and click again on Install in the next window and the process of installing starts. You should let the process proceed without interruption. Multi-tasking at the time updating is in progress is unwise. The progress window will tell you what is going on. You need to be patient.

 

You may have a single update that causes all in the batch to fail to install. If this occurs you should install one at a time so that you can find out which update is causing the problem. Manually installing and restarting your computer also allows you find out more quickly whether the problem is resolved. Automatic updating is a scheduled operation, which may execute up to 24 hours later.
