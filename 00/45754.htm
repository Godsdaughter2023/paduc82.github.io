canadian tech say:

Please click on Computer.  You should then see a window that has your C: drive in it and it should tell you its capacity and how much of that capacity is in use. My reason for wanting to know this is that many people find themselves in trouble when there is not enough space for the operating system to work on Drive C: which should be your main drive.

The most common cause of failure of laptop is the hard drive.  Your hard drive was manufactured by Seagate.  Download and run the Seatools for Windows program and run all the tests, do not attempt any fixes unless you have done a complete backup of your

http://www.seagate.com/ca/en/support/downloads/item/seatools-win-master/

This tests the HARDWARE of the drive.  It could be starting to fail. If all tests are past, at least you know you are working with good hardware.

Have you ever updated your BIOS or any drivers?  Suggest you find the Lenovo support web site and find the latest drivers for you model.  There may be an updated BIOS that should be applied.  If you feel uncomfortable doing this, find someone who is not to do it for you.

Caution:  do NOT use any of the hundreds of driver update web sites you find on the web.  They are usually full of infections and seldom have anything good. Use only Lenovo for this.

In general, you should know that your laptop at 4.5 years is statistically not likely to live a lot longer.  Laptop life used to be 3 years on average and has improved over the last several years to 4 or 5.

melowin mar say:

Hi Gauranga,

Follow the steps below if the Hibernate option was not found in Start Menu.

  1. Select the Start button>All Programs>Accessories and then right-click on Command Prompt and select Run As Administrator. Enter the password and continue. You can also open Command Prompt in Administrator mode by typing cmd in Start Menu Search box and press Ctrl+Shift+Enter.
  2. Provide the following command: powercfg /hibernate on
  3. Then you have to exit command prompt so that Hibernate option will show in Start Menu.

If the above the steps doesn't work for you, make sure the Hybrid sleep option is turned off in Advanced Power Options.

  1. Open Power Options from Control Panel or by using powercfg.cpl command in RUN dialog box.
  2. Now click on Change when the computer sleeps link present in left sidebar.
  3. Now click on Change advanced power settings link and expand Sleep setting and set Allow hybrid sleep option to Off.
  4. Apply changes and then restart your computer.

Let us know how it goes.

Restart the computer a few times and check for Windows Updates.
