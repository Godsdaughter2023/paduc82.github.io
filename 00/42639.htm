lemp say:
I'm surprised that whatever Norton product you have hasn't removed the malware. 

Sometimes, the security software removes the actual infected DLL file but doesn't remove the left-over system configuration that attempts to load the DLL file (using svchost or a similar mechansim) at system startup.  That typically results in error messages such as "Error loading C:\WINDOWS\mopscatp.dll" where the name of the DLL file is usually some random string of letters and numbers.  If this is the case, you have to remove the registry entry that is causing the automatic startup.  The preferred method of doing that is by using Autoruns (a free tool from Microsoft).

That, however, doesn't sound like the situation you have.

In any event, I suggest that you download, install, update, and run a complete scan with each of the following free programs.  Do not get the trial of the "Professional" version of either program.  Be careful during the installation process that you don't inadvertently install any "add on" software such as the Ask Toolbar which the publisher tries to sneak into the installer.

Don't run the scans simultaneously.  Each scan may take quite some time, depending on the size of your hard drive, so start one up and then go do a non-computer chore for a couple of hours.  When the scan completes, allow the program to delete or quarantine anything it finds. 

When you are finished, I suggest uninstalling SUPERAntiSpyware (which I find to be more intrusive than I like) but keeping MalwareBytes AntiMalware so that every week or so, you can update the MBAM database and do a scan.
