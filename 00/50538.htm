Here's how.  This works on all versions of Windows back to Windows '95.

  1. Open the Character Map program. To do this on Windows 8, just open the Start screen and begin typing the name.  By the time you get to "char", it will probably already be found in the search results.




  2. Now that you opened Character Map, click the Advanced View checkbox at the bottom.  You'll see this provides a search box now.

  3. Search for the word "degree" and you'll find your symbol, which you can select and copy.




  4. Notice at the bottom right, it says Keystroke: Alt+0176.  That means you can get the degree symbol in the future by holding Alt while you type the numbers 0176 on your number pad.  No need to even open the Character Map.
