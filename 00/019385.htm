jason (office engineer) say:

The limits are for each individual Sway.  The paragraph limit counts every new line in each Text card, Title card, Heading card, or captions in other cards.  It doesn't count as a new one when the text wraps to a newline, but if you press Enter to manually get a newline then that counts as a new paragraph.  Bullets and numbering also each count as a new paragraph.  Even if your second Sway is longer with number of cards, there may be more paragraphs within the cards of your third Sway.  Yes, you can add links to go from one Sway to another and that won't cause the limit to be applied to the Sways together.

Also, if you go over the limit, even though the content will show in the Storyline, the content over the limit will not show in the Sway.

daniel jackson say: Hello Denise,

Thank you for posting your query in Microsoft Office Community.

When you set the  option 'Recur every 2 weeks', the series starts from the following appointment's  weekday, and the loop continuous with the interval of 2 weeks from the appointment date.

This is not based on the 2nd and 4th week of the month, but on the 2 week interval from the creation date.

As a work around you may create 2 recurring appointment with the following recurrence pattern and see if it helps.

1) New Appointment > Recurrence > Monthly > The second > Wednesday of every 1 month.
2) And another appointment with the recurrence pattern 'Monthly > The fourth > Wednesday of every 1 month.

You may also check if you can use code to create recurring appointment with 2nd and 4th week recurrence pattern in a single appointment. For assistance related to creating code, post your query in following section of Microsoft Office Community:

mauro gamberini say:
I am using Excel 2003 and would like to set the tab key to only work in 3 columns, e.g. I want to use tab to navigate through columns A,B, and C in row 1 and then I want the tab key to move me to cell A2.  Is this possible?


In the code module of ThisWorkbook:

Private Sub Workbook_Open()
    Application.OnKey "{TAB}", "m"
End Sub

In a standard module:

Public Sub m()
    If ActiveCell.Column = 3 Then
        ActiveCell.Offset(1, -2).Select
    Else
        ActiveCell.Offset(0, 1).Select
    End If
End Sub

New Word Styles - using in other documents - Microsoft Community

oldhasbeen say:

New Word Styles - using in other documents

I've created a new Word style - how can I make it usable in all my documents (i.e. get it to appear in the style gallery)

I realise I could create a new template containing the new style,  but this would only be useful for brand new documents.

stefan blom say:

You can copy the style to your template. The easiest way is to select "New documents based on this template" in the Modify Style dialog box; doing that adds the style to the template which is attached to the current document.

If you want to add the style to other documents, you must copy them separately, for example via the Organizer dialog box.

Here's how to display the Organizer dialog box: In the Styles pane (which you can open by pressing Ctrl+Alt+Shift+S), click the Manage Styles icon. In the Manage Styles dialog box, click the Import/Export button.
