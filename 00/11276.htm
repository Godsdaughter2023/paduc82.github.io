jasles say:

What if you added sheets for each week of the year (the naming wouldn't matter) in the ProductionWeekly workbook, and then have each week in the Report workbook, RNTCombined sheet.

Then in column B for each week, use this formula:

=LOOKUP(2,1/(NOT(ISBLANK([ProductionWeekly]08-13-17!$K:$K))),[ProductionWeekly]08-13-17!$K:$K)

You'd have to reference each weekly sheet in each row's formula, but it should work.
