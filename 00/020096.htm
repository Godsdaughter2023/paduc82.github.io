ossiemac say:

Now I understand the problem. The formulas do not return an exact value. This is caused by the binary method used by computers to perform calculations. Unfortunately, many calculations are not 100% correct and need to be handled by the user. This is simply an idiosyncrasy of computers.

As a test, insert the following formula in any blank cell

=10.0385 - 10

Now copy the cell and Paste Special -> Values over it to remove the formula and you will see the following result in the formula bar.

0.0385000000000009

What we have to do is handle the final result with the ROUND Function like the following so that the result is rounded to the required number of decimal places. (The example below is one decimal place)

=ROUND((B4-A4)*1000,1)

Therefore the formulas for the Conditional formatting will be like the following for Sheet1.

For Green:

=AND(ROUND((B1-A1)*1000,1)>=31.5,ROUND((B1-A1)*1000,1)<=38.5)   

For Red:

=OR(ROUND((B1-A1)*1000,1)<31.5,ROUND((B1-A1)*1000,1)>38.5)

Other functions for rounding that you should investigate are as follows (Google the functions for more info on them.)

ROUNDUP

ROUNDDOWN

CEILING

FLOOR
