How to Configure Computer to Enable BitLocker without Compatible TPM:

Administrators must follow the steps below to configure their Windows 10 computers to allow enabling Bit Locker Drive Encryption without compatible TPM:

a. Log on to Windows 10 computer with the account that has administrative privileges.

b. Assuming that the computer has been configured to display classic start menu, click Start and at the bottom of the menu in search box type GPEDIT.MSC command and press enter key.

c. On the opened Local Group Policy Editor snap-in from the left pane expand Computer Configuration > Administrative Templates > Windows Components > Bit Locker Drive Encryption and from the expanded list click to select Operating System Devices.

d. From the right pane double-click “Require additional authentication” at startup.

e. On the opened box click to select Enabled radio button and ensure that under Options section Allow Bit Locker without a compatible TPM checkbox is checked.

f. Once done, click Ok button to allow the changes to take effect and close Local Group Policy Editor snap-in.

Hope this information helps.

    greg carmack - windows mvp 2010-2020 say:
    I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

    The hardware signature changed so will need to be reactivated. See here for how to move Windows 10 to new hardware and reactivate: https://support.microsoft.com/en-us/help/20530/...

    If the original license was using a retail and not OEM license, including the underlying license for an Upgrade, then it can transfer. If it was OEM it cannot - although the troubleshooter may make an exception is everything else is legit. But you must have linked the old Activation Digital License to your Microsoft Account and now be able to sign into that Account to run the Activation Troubleshooter to transfer it.

    If the new PC came with the same Windows 10 license then you can activate with it by running ProducKey: http://www.nirsoft.net/utils/produkey-x64.zip to locate the OEM-BIOS key and copy and paste it into Activate, or if you have a retail license key for that version you can insert it.

    If you need help you can contact Microsoft Support, asking for Activation help, then keep asking for an agent. https://partner.support.services.microsoft.com/...
