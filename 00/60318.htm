azeez n say:
Hi Gary Rich,
 
USB 2.0 VGA Adapter is not a program. It's a hardware device.
 
1. What automated troubleshooters have you tried?
2. Are you referring to secondary monitor by any chance?
3. What Operating system is installed in your computer?
 
Well, since, you have mentioned USB 2.0 VGA Adapter which , I am sure the issue has to be with secondary monitor.
 
You would be interested in following the below given steps in order to fix the issue.
 
Step 1:
 
You may access the link below and follow the steps to setup dual monitor on your computer and check if it helps in detecting the secondary monitor.
 
http://support.microsoft.com/default.aspx/kb/976064
 
http://windows.microsoft.com/en-us/windows7/work-and-play-better-with-multiple-monitors#section_1
 
Refer the links below for more details on troubleshooting multiple monitors:
 
Move windows between multiple monitors
http://windows.microsoft.com/en-US/windows7/Move-windows-between-multiple-monitors
 
Troubleshoot multiple monitor problems
http://windows.microsoft.com/en-US/windows-vista/Troubleshoot-multiple-monitor-problems

Step 2:
 
If the issue is still not resolved, I’d suggest you to uninstall the display drivers and re-install the same and check if that fixes the issue.
 
a. Click Start, type Device Manager in the Start search box and hit Enter.
b. Locate Display adapter and expand the same.
c. Select the display adapter you wish to uninstall.
d. Right-click and select Uninstall.
 
Follow the on-screen instructions. Restart the computer, if prompted.
 
After the computer restarts, visit the manufacturer’s website and download and install the latest chipset and display adapter drivers and check if that fixes the issue.
 
Step 3:
 
If that doesn’t help, you may also try by reducing the video hardware acceleration and check if that fixes the issue.
 
a. Open Display Settings by clicking the Start button, clicking Control Panel, clicking Appearance and Personalization, clicking Personalization, and then clicking Display Settings.
 
b. Click Advanced Settings, click the Troubleshoot tab, and then click Change settings.
 
    Note that some video card drivers do not permit you to change settings.
 
c. Move the Hardware acceleration slider to different levels, apply the changes and then try to play the game.
