For the first part of the question (changing 2 to 3 etc) you could put the numbers 2-? above each respective column and then refer to the cell in the formula rather than the actual value. For example:

Formula in F2: =VLOOKUP($E$2,$A$2:$C$2,F$1,FALSE)

Formula in G2: =VLOOKUP($E$2,$A$2:$C$2,G$1,FALSE)

This will change automatically as you drag it. I'm guessing this is Spanish? so you probably need to change the commas in the formula to semicolons as appropriate.

For the second part, 'lock' the row number with a $ sign e.g. $A$2:C$780

Hi, There is a simple way to change the column number automatically in VLOOKUP() and you do not need to type manually in anywhere in the worksheet and use that cell as reference. You can simply use COLUMN() or COLUMNS() to get it. Like, based on your VLOOKUP() use below formula to get automatically Column number:

=VLOOKUP($A783,$A$2:C$780,COLUMN(B2),FALSE)

In the above formula, A783 is your Lookup_Value where Column has locked but Row is free. $A$2:C$780 is your data table range, where $A$2 cell has locked due to it is the starting point of the range and at the ending point you need to return data from dynamic columns, so that "C" is free but row has locked to $780. In COLUMN() returns the column number. As you are expecting to return data from SECOND Column, so COLUMN(B2) returns 2. And if you drag it to the right, then it will automatically changed to C2, D2 etc. and C2 returns 3, D2 will return 4.

By this why if you drag the formula to the right, then you will get the automatically answer.

Hope this will help you.
