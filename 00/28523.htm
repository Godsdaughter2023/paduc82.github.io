jay freedman say:
Hans's reply in that thread is one method. Another is to create a character style that consists only of the custom color, which you can apply onto any text that needs it regardless of the paragraph style that's already applied (Normal, Body Text, Heading X, etc.).

In a document based on your Letterhead template, use the More Colors dialog to apply the custom color to some selected text, or select some text that already has it. On the Home tab, click the lower down-arrow of the Styles gallery and click Save Selection as a New Quick Style. In the small dialog that appears, enter a name and click the Modify button. (Important: Don't click OK, because then you'll get a paragraph style, which can't be converted into a character style.)

In the next dialog, change the Style Type dropdown to Character. If you want to add this new style to the template on which the current document is based, click the option button at the bottom for "New documents based on this template" -- that will make the style available in all new documents (but not existing documents) that you create from that template.

Don't change anything else, just click OK.

I certainly don’t buy into the philosophy that you can’t take privileges away from malware without also taking those same privileges away from the user; but if taking unnecessary privileges away from the user also takes those privileges away from malware; then I think that’s a step in the right direction.
