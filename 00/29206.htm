With no information about your system, I have noted these observations for that error:

In addition to the good ideas from ZigZag3143, here are some other thoughts...

This is almost always a hardware problem - either faulty RAM, a failing hard drive, low CMOS battery, loose connections, or it could be something like a marginal power supply.

(if your system is a desktop especially) etc. so I would make an effort to rule some of these things out first.

If your system boots, you can look in the Event Log for messages like this.  They are really not worth deciphering until you rule out hardware issues since you will probably find that it is a hardware issue:

Event Type:    Error
Event Source:    System Error
Event Category:    (102)
Event ID:    1003
Description:
Error code 000000f4, parameter1 00000003, parameter2 89fe0da0, parameter3 89fe0f14, parameter4 805d297c.

If the message contains 00000003, that is your clue that it is a hardware issue.

Has the system been relocated, dropped, bumped, kicked, punched or whacked for misbehavior prior to this message?

Have you made any hardware changes lately...  added additional RAM, new video card, or anything like that?

If you have any external USB device installed that are not necessary to boot, you can unplug everything that is not necessary to boot leaving just the keyboard and mouse at least temporarily to eliminate them from the equation.

A low CMOS battery on your motherboard could be a suspect, so replace it to get that out of the troubleshooting equation.

I would run a chkdsk with error correction either from Windows or if your system will not boot, run it from the XP Recovery Console and it never hurts to do a memory test.

You could also run some diagnostics on your system RAM memory and to be thorough you can use the one from Microsoft and the popular memtest86+ and also reseat your RAM (remove one stick of memory at a time and replug it) in case there is a marginal connection.  If you need directions for how to do that, say so and I will post some.

After unplugging your system from the power source (like the wall or power brick) discharge any static electricity in your body by touching a metal part of the system chassis and carefully unplug and replug every connection and cable in your system that comes undone without too much force being sure to only do one thing at a time so you don't get thing mixed up.  Just unplug it, and plug it right back in.  Things that don't come loose easily, just give the a couple wiggles to be sure it is secure.

If you have a discrete separate video card that is not built into the motherboard, remove it and put it right back in to be sure there is not a loose connection there.

Some OEM computer manufactures (Dell, HP, IBM, etc.) and most hard drive manufacturers have non destructive diagnostics you can download from their  WWW support pages to test your HDD.
