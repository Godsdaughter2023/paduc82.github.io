This is a fairly common complaint - one cause is the reading pane setting to mark as read automatically but its rare for all mail to be marked automatically in newer versions of Outlook - only the first message would be marked. Another cause is download mail via POP3 from one system but using a server-based account elsewhere. Addins and utilities that process mail could cause it.

 

Unless you are actually using OWA, the OWA settings won't affect this. If you use OWA to read mail, then yes, it could be one factor, but it goes away when you close OWA.
