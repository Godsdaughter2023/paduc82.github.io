ken sheridan say:
Now, what I need is:  That pressing the CheckBox, ignore both conditions and that the font color is set GRAY

You can do that with conditional formatting of the Field Date control as follows.  First set the default ForeColor of the Field date control to grey.  Then in the conditional formatting dialogue, for the first condition, setting the ForeColor to red:

Expression is        [CheckBox] = False And [Field Date] >= Now()

and for the second condition, setting the ForeColor to green:

Expression is        [CheckBox] = False And [Field Date] < Now()

Note that I've used <Now() for the second condition rather than <=Now() as you can only test for equality in one of the conditions.  Alternatively you could use >Now() for the first condition and <=Now() for the second.  Bear in mind that the Now() function returns the current date and time, so if you want to test the date only you should use the Date() function rather than the Now() function.

mike h.. say:

Hi,

I'm afraid that's default behaviour and there's no setting to change that. What you can do with some VB code is make every comment appear in the centre of the window when you select a cell with a comment.

Code courtesy of contextures. Right click your worksheet tab, view code and paste the code below in. Close VB editor and select a cell with a comment in it.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
 'www.contextures.com/xlcomments03.html
 Dim rng As Range
 Dim cTop As Long
 Dim cWidth As Long
 Dim cmt As Comment
 Dim sh As Shape
 
 Application.DisplayCommentIndicator _
    = xlCommentIndicatorOnly
 
 Set rng = ActiveWindow.VisibleRange
 cTop = rng.Top + rng.Height / 2
 cWidth = rng.Left + rng.Width / 2
If ActiveCell.Comment Is Nothing Then
  'do nothing
Else
  Set cmt = ActiveCell.Comment
  Set sh = cmt.Shape
  sh.Top = cTop - sh.Height / 2
  sh.Left = cWidth - sh.Width / 2
  cmt.Visible = True
End If


End Sub

Edit page numbers in Word 2010 - Microsoft Community

grandpa jim say:

Edit page numbers in Word 2010

My doc was created in Word 2003. I now use Word 2010.

I want to change the font and location of the page numbers. For some reason I'm not understanding the Help file. Is there a simple way to do this?

suzanne s. barnhill say:

Double-click in the header to open it. If the page numbers are in the footer, you will then be able to scroll down and click to position the insertion point there. Wherever the page numbers are, you can select them and change the font. If they were inserted in Word 2003 using Insert | Page Numbers, then they will be in a frame, which you can drag. Otherwise, they will have been positioned using paragraph alignment or tabs. It will be helpful to display nonprinting characters (formatting marks) in order to see what you're dealing with. If there's a chance there are frames involved, you'll also find it helpful to display text boundaries (see http://word.mvps.org/FAQs/Formatting/UnaccountablyIndented.htm).

If you want to start from scratch, select the page number (and any surrounding frame) and delete it, then use Alt+Shift+P to insert a page number anywhere you want it. If you want to change the font for all page numbers, note that Page Number is a character style that can be modified.

jr de pijper say:

Is there a way to change the actual notebook folder name so that the change propagates to all platforms?

 


No, there isn't. OneNote notebooks correspond to actual folders in the file system, that is probably the reason. When you change a notebook's name, you really change its "NickName" property. As you found out, you can also do this in OneNote on the desktop, by changing the "Display Name", but not in the mobile version.
