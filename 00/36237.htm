ben rhouma say:

Ce message apparaît lorsque l’application universelle utilise votre emplacement. De nombreuses applications telles que les cartes ou le courrier et les calendriers peuvent utiliser votre emplacement. Dans ce cas, vous recevrez ce message dans votre barre des tâches. Beaucoup d’utilisateurs ne sont pas à l’aise de partager leur emplacement, et aujourd’hui nous vous montrerons comment le désactiver Votre emplacement est actuellement utilisé message.

La fonction de localisation peut être très utile pour certaines applications, mais si vous n’êtes pas à l’aise de partager votre emplacement avec les applications Windows, vous devriez désactiver cette fonction. Pour ce faire, il vous suffit de désactiver cette fonction dans les options. C’est très simple et vous pouvez le faire en suivant ces étapes :

  1. Appuyez sur la touche Windows + I pour ouvrir l’application Paramètres.
  2. Une fois que l’application Settings s’ouvre, naviguez jusqu’à la section Privacy.
  3. Allez à Location sur la gauche. Dans le volet de droite, cliquez sur le bouton Changer dans la section Emplacement. Définissez l’emplacement de cette unité sur Off. Assurez-vous également de désactiver le service de localisation .

Après cela, le service du site est désactivé et les applications ne peuvent plus utiliser votre site.
