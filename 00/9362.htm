Microsoft Organization Chart Add-In box background colors - Microsoft Community

kmorrell60 say:

Microsoft Organization Chart Add-In box background colors

Is it possible to use box background colors, other than the drop down color choices, in Microsoft Organization Chart Add-In?  The light blue color is printing out very dark, making it impossible to read the text.

jay freedman say:

The colors offered in the dropdown are "theme colors", and the blue background in the org chart boxes is the Accent 1 color in whatever is the current theme. You can choose a different theme that has a better Accent 1 color, or you can keep the same theme but select a different color or shade for its Accent 1.

To change the theme, go to the Page Layout ribbon and click the Themes button. It its dropdown, mouse over the various themes to see the effect this has on the org chart. You may not like any of the choices -- if so, proceed to the Colors button next to the Themes button, and try mousing over the entries in that dropdown.

Most likely, you'll want to choose a similar blue but in a lighter shade. To do that, first choose the theme that you want (or keep the default one). Then click the Create New Theme Colors entry at the bottom of the Colors dropdown. In the dialog, click the Accent 1 button.

In the top row of color blocks, the current Accent 1 color will have a red border. Click one of the lighter shades in the same column, such as Accent 1, 40% Lighter. Enter a name for this modified color set in the Name box and click the Save button.

As soon as the dialog closes, all the boxes that use Accent 1 will display the new lighter shade. Verify the choice by printing a sample chart.

If you want to use this color set elsewhere, you'll find it has been added to the bottom of the list of color sets in the Colors dropdown.

From what I've been able to figure out, this issue is not limited to BetterSnapTool/BetterTouchTool specifically, but any window-management software that allows Windows-style window "snapping", where dragging the window to the left or right side serves as a shortcut for making the window fill that half (and dragging to the top maximizes).
