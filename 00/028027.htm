greginmich say:

Is there any way to stop the  tech support scam popups that lockup the computer until I close my browser?

A few of these Tech Support Scam pages actually are the handiwork of malware (PUPs or malicious browser add-ons); but those would normally be the full screen mock-ups, or pages that pop up and need to be force-closed repeatedly, e.g.:

https://blog.malwarebytes.org/fraud-scam/2015/07/techsupportscams-and-the-blue-screen-of-death/

In most cases, though, there won’t be any malware to hunt down with these Tech Support Scam web pages, because the “malicious code” here is nothing more than a script that’s run by the page itself. The script loops the alert message box, and since that message box is modal, the browser deactivates its controls and waits for the user to close the message box. But the loop prevents that from happening and effectively locks up the browser. So disabling active scripting for the browser should prevent the lockup:

https://blog.malwarebytes.org/fraud-scam/2015/09/avoid-this-bsod-tech-support-scam/

https://support.microsoft.com/en-us/gp/howtoscript?wa=wsignin1.0

More on that here:

http://answers.microsoft.com/en-us/protect/forum/protect_other-protect_scanning/potential-malware-blue-screen-with-dialog-box-on/3569bb1b-c05e-4247-98dc-6bf22da581e0?page=4#LastReply

Now we just need to find a way to disable scripting for Edge, because it’s turned on by default and there’s no way to switch it off in the user interface. A. Sometimes, other add-ins can cause problems. To determine, if add-in is a problem start your application in safe mode and see if problem has gone away.

To start in Safe Mode -

Hold CTRL key and click on application icon and don't release the CTRL key till it asks you for Safe mode confirmation.

If yes, then start application normally and disable add-ins one by one and start application again every time you disable an add-in to determine the culprit add-in.
To disable add-in > File > Options > Add-ins > Down below you have Manage: Com Add-in : Press Go > Untick the Add-ins to disable and tick to enable them

B. There is another possibility of having another version of Office which can create this conflict because of 32 / 64 bit conflict. If there is another version of Office, you will need to uninstall other Office versions and keep only latest. After uninstall, you will have to repair Office 2016.
