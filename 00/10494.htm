ossiemac say:

Try the following code and see if it helps.

It first attempts to set the object variable to Word if Word is already open and if it is not open then it creates an instance of word.

DoEvents forces it to finish what it is doing before commencing the next line of code.

Before setting object variables to nothing, clear the contents of the variables by closing the document and quitting word.

Sub buttontest1()
    Dim objWordApp As Object
    Dim objDoc As Object
    Dim FileName As String
   
    On Error Resume Next
    'If Word is already open then assign to the variable
    Set objWordApp = GetObject(, "Word.Application")
    On Error GoTo 0
   
    'If Word was not previously open then object variable will be nothing
    'Therefore Create the Object
    If objWordApp Is Nothing Then
        Set objWordApp = CreateObject("Word.Application")
    End If
   
    'FileName = "/Volumes/256SSD/word docs/shortcut keys.docx"
   
    With objWordApp
        .Visible = True
        Set objDoc = .documents.Add ' .documents.Open(FileName)
    End With
   
    DoEvents        'Yield to operating system and allow it to complete all external events
   
    objDoc.Close    'Close the document before setting the object to nothing
   
    objWordApp.Quit 'Quit Word before setting the object to nothing
   
    Set objDoc = Nothing
    Set objWordApp = Nothing
 End Sub

shashirekha say:

Leave a copy of messages on the server option is only available if you have a POP3 account configured in outlook. Only POP accounts download emails from the server to your computer.

If you have an IMAP, Exchange or outlook connector account the emails are not downloaded on to the computer but outlook syncs with the server creating a mirror image of the server therefore the option is unavailable in these accounts.

Determine if you have POP3 account configured.

If you have POP3 account configured then click on file->account settings-> Click on the account->Change->More settings->advanced and you will find a check box “Leave a copy of messages on the server”.
