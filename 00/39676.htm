The .wlmp is the project file.  It contains the list of files you are using in your project, but is not a source file for playing video.  You mentioned you are using an .avi file from dxtory.  My guess is that the codec used to encode the video/audio in that file isn't 100% compatible with Movie Maker and that is why it fails to save as a movie (stops at 43%).

 

The quickest fix would be to convert the .avi from dxtory into .wmv format using something like Freemake Video Converter.  If you do install Freemake, be sure to choose the 'Custom' option so you can control what it installs or doesn't install.

 

The resulting .wmv file should edit much better in Movie Maker and not have the saving problem you are having now.

 

Another possible workaround is to split your project into smaller parts.  Make several copies of your .wlmp file.  In the first copy, delete all but the first third.  See if that will save as a movie (you said the full project got to 43%, so 1/3 would be 33% and should save).

 

If that works, then take the second project copy, delete the first and last thirds, leaving the middle.  See if that will save as a movie.  Do a similar thing with the last project copy (keeping the last third).

 

In the end, when you have 3 movies created for each third, create one new project where you add those 3 video files together.  Save one last time as a full movie.

Yes, you can.

Because Windows 7 is in limited stock because Microsoft ended sales in October 2013, this has created demand because of its scarcity. This means retail boxed copies demand a premium, while OEM System Builder copies remain affordable.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

There is nothing wrong with using it too, especially since all future releases of Windows will be full versions anyway.

You can find OEM System Builder software from dozens of online merchants. The current price for OEM Windows 7 Professional at cost only

There are no technical limitations to prevent you from using OEM software on your own PC, although this software will work only for a clean installation and not for an upgrade.
