Where did my Favorites go in Windows 10? - Microsoft Community

tdehan say:

Where did my Favorites go in Windows 10?

e.gray say:

I used to be able to access my FAVORITES folder from the Start menu in Windows 7.  Where did that go in Windows 10?  

This has been a long suffering thread with lots of twists and turns...  in fact, I'm not really sure what the original question is!

So, let me see if I can shed some light on this topic.

  • Internet Explorer has pretty much always used the C:\User\YourNameHere\Favorites to store favorites
  • These favorites were stored as files
  • Edge has never used this location and does not share favorites with Internet Explorer at all

  • The version of Edge in original "RTM" release of Windows 10 kept favorites as files buried 11 folders deep at: C:\Users\YourNameHere\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\AC\MicrosoftEdge\User\Default\Favorites
  • You could use the File Explorer at this location to Edit, Add, Delete favorites, but it took a registry tweak for a lot of the changes to appear in Edge

  • The current version of Edge (from the Windows 10 "November update") now uses an database at: C:\Users\YourNameHere\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\AC\MicrosoftEdge\User\Default\DataStore\Data\nouser1\120712-0049\DBStore\spartan.edb
  • So that means that you can no longer use the File Explorer to manage favorites
  • There may be favorites left over at the "old" location, but they are no longer being used.  (That's why when people see these files, they notice that they are not up to date).
  • The "Microsoft way" is to manage your favorites from inside the Edge application

However, there is a 3rd-party application called EdgeManage that will allow you to manage your Edge favorites.  Take a look if you want:

http://www.emmet-gray.com/Articles/EdgeManage.html
