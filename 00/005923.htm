jasles say:

You only need one data source, regardless of the amount of PivotTables that you have - no need to copy the tabs and then delete data from the source. This could be causing your problems.

You can filter each PivotTable to report only on the data that you want. So you could have one pivot table filtered to show only by Company and Level, and another pivot table of MGMT data, and so on.

The pivot tables all will be using the same raw data on your Labor Estimate Data tab, but let you filter what data is shown on each one.

It seems that in step 3 of your process above you have created partly duplicate data.

You should:

1) Have only one tab of raw data

2) Use that one tab of raw data as the data source for all of your pivot tables

3) Add report filters to each pivot table to filter what data is shown for each

If styles in a Word document are changed if a document is opened on another computer, the first thing I would check is whether "Automatically update document styles" is checked.

Open the document. Click on the Developer tab (how to show developer tab). Click on Document Template. If the Automatically update document styles is checked, uncheck it.

It actually updates all of the styles in the document with the styles in the attached template. If you have created a document based on MyLetter.dotx, and you send the document to me, the document will no longer have access to MyLetter.dotx, since I don't have a template named MyLetter.dotx. Instead the Normal.dotm will be the attached template when I open the document. If "Automatically update......" is checked, all of the styles in the document will be updated based on the styles in my Normal.dotm.

It's actually a pretty good feature, but only when you know exactly how it works. Then it's actually possible to update styles that others have changed in a document to the styles the document was created with (if you do have the right template).
