cmdrkeene say:

Hi all,

This is a bit of a complicated issue, and works differently on different versions of Windows.  As someone else already posted above in this conversation, it works pretty splendidly with Windows 10.

The story with Windows 8 is the worst. Since

However, even in Windows 8.1, you're not going to have the best time quite yet. There's a lot of backstory to this, but the short version: Just like email can sync using POP/IMAP/Exchange and other protocols, Calendars use protocols too. Google changed from using "Exchange Active Sync" to using something else (I think it wasn't the CalDAV standard but actually something Google-specific).  But regardless, it doesn't sync to the Windows 8.1 calendar directly.

Like email, the best way to sync a calendar is to store it somewhere online (like an email server) and let all your phones/tablets/programs just point to that server so everything stays in sync.  But since the Windows 8.1 calendar can't talk to the Google calendar servers, that's an issue.  So you have two options:

  1. Don't use Google Calendar
    Option 1 is to use a calendar server that will work with all your devices/phones/tablets. And at the risk of sounding like a Microsoft fanboy, Outlook.com's calendar can be the central calendar sync point, and it doesn't require changing your email address or anything. In fact, just by posting here on this Answers website, you already have a Microsoft account capable of using that Calendar.  And it supports all the standards, even syncs to iPhone or Android, email programs, etc.


    Or,

  2. Don't use Windows 8.1
    Option 2 is to keep your calendar where it is, but stop using any programs that can't sync to it. So goodbye, Windows 8.1 calendar. Instead, use a full program that can talk to Google calendar server (I don't know if Office Outlook can or not).

    With Windows 10 coming in about 9 days, that basically does option #2 for you and solves this problem. So if you are really happy with Google Calendar but also want to use the Windows app, you have 9 days left to wait :)

It is possible to pull in your Google calendar events into the Outlook.com calendar, so they will show on Windows. That's called an iCal calendar subscription file, and it's read-only. Your appointments would show up, but you couldn't edit/remove/change them from the Windows app, so most users wouldn't like that option either.

So super short version: Windows 8.x calendars can't sync to Google calendar service. Windows 10 fixes it in about a week, or use a more open server.
