Unfortunately, these type of malware attacks are difficult to keep up with because they trick you into letting them install. They usually come from an infected web site, and usually through an advertisement. You get a pop-up from the infection and you click it to close the pop-up - which allows the infection to install.  They can also be delivered in a "drive-by" fashion with no action needed by the user due to the system being unpatched, no matter what security software is running.

 

When you encounter one of these fake virus pop-ups while browsing, immediately do the following:

-Do not touch any browser window to close it or browse further.
-Immediately press Ctrl-Alt-Del and bring up Task Manager and forcibly end all instances of iexplore.exe, if using Internet Explorer, or the executable for your browser for any other web browser.
--or--
-Go to Start/Shut Down and restart the PC without touching any browser windows.
-If you used task manager to close browser instances, reboot the machine.
-Then go to Control Panel/Internet Options and delete all temporary Internet Files and cookies. If you are using an alternate web browser, open the browser settings to do the same - delete the local cached files and cookies.
-Perform a full scan with MSE.

The above steps should prevent the infection from taking hold.

Start here - https://support.microsoftsecurityessentials.com/

and select the link that says - I think my computer is infected. Options will vary by region, but phone support leads you to Microsoft Answer Desk (http://www.answerdesk.com/) in the US at this time. Besides MSE, the following recommendations will assist in protecting the PC from infection:

-Make sure that the Windows Firewall is enabled.

-Make sure that all important/critical updates, including service packs for the operating system and programs are installed from Microsoft Update (Windows Update).

-Make sure Internet Explorer is at version 8 or 9 and updated with all patches.

-In Internet Explorer 8 or 9, use the SmartScreen Filter.

-Make sure that IE Internet Security settings are at least set to medium-high (default).

-Enable the pop-up blocker in IE.

-On Vista and Windows 7 make sure that User Account Control (UAC) ON  and not running with elevated privileges.

-Make sure that Windows Automatic Updates are set to at least notify, but the preferred setting is to download and install automatically. If you update manually, be sure to update as soon as possible after being notified of available updates.

-Make sure that installed applications, especially Adobe Acrobat, Adobe Flash, and Java are at their latest versions. Many vendors are regularly updating and patching for security holes.

-Never click through links from unknown sources and use caution even if they are from a "trusted" source.

-Never open unsolicited email attachments.

-Practice safe web browsing.

rohn007 say:

Word Options: "Fast Save" vs "Background Save" are they the same?

Many years ago, when all parts of computing were relatively much slower than they are today, there were some "Options" to make saving files appear to go faster.

As I understand it:

"Allow Fast Save" turned of the "housekeeping" function that truly deleted "deleted" text, ie undo would not work after a save. Leads to bloated file sizes. It has been completely removed from 2010.

"Allow background saves" turned on multi threading, in theory allowing you to get back to work while the actual save went on in the "background". Still an option in 2010.

hansv mvp say:

They are quite different.

If "Allow Fast Saves" was turned on, saving a document would not rewrite the entire file. Instead, the changes since the previous save would be tacked on to the existing file. This would indeed result in a "fast save" especially for a large document with only a few changes. But it meant that each time you opened the document, it had to be reconstructed from the first saved version plus all successive changes, and this was a surefire way to corrupt the document eventually. So the widespread recommendation was to turn off fast saves first thing after installing Word.

If "Allow Background Saves" is turned on, you can continue editing in Word while a document is being saved. If off, Word locks up temporarily while saving.

"opens a new form in Add mode" do you mean Data Entry is set to Yes or that you've used VBA  of style:


DoCmd.Openform "YourFormName",,,,acFormAdd



In either case, when in Data Entry mode (which acFormAdd is) the form does not display any records and only brings up a blank version of the form for 'data entry purposes'.




An alternate approach would be to open the form normally

DoCmd.Openform "YourFormName"

and then use the form's load event to navigate to a new record.


Private Sub Form_Load()
   DoCmd.GoToRecord , , acNewRec
End Sub

jay freedman say:

If your computer is a laptop, you might have a key labeled Fn or something like that, which serves as a sort of shift key for the function keys. Pressing the two keys together would give the F12 function instead of Airplane mode. You may have to look at the HP user's guide to find out how you're supposed to do this.

You can indeed move the command to a different key or key combination. 

  • Go to File > Options > Customize Ribbon.
  • Click the Customize button in the bottom left corner.
  • In the Categories list, select File Tab.
  • In the Commands list, select FileSaveAs. Note that F12 appears in the Current Keys box.
  • Click in the "Press new shortcut key" box and press the key or keys that you want to use for the Save As command. I suggest F11, which is by default assigned to the rarely used NextField command (and that command has a second combination, Alt+F1, if you do want to use the command).
  • Click the Assign button
  • If you never want to use F12, click on F12 in the Current Keys box and click the Remove button.
  • Click the Close button.
  • When you shut down Word, if you're prompted to save changes in the Normal template, answer Yes.
