You have to run something when the button is clicked.  You can put it all in one procedure or you can call one procedure from another:
 Sub YouHaveToRunSomething() 'Run using your button
If MsgBox("Do you want to run this macro?", vbYesNo, "Decisions, Decisions") = vbYes Then
  MsgBox "Yipee!! I can run."
 'The rest of your macro code here"
End If
End Sub

Sub BeSure() 'Run using your button
If MsgBox("Do you want to run this macro?", vbYesNo, "Decisions, Decisions") = vbYes Then
 ReallyRunMe
