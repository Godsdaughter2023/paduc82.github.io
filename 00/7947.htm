    shortcut key for last used row in a column and last used column in row

    Hi everyone

    Is there a shortcut key for the last used row in a column?

    Also, what about what the last used cell in a row?

    If no shortcut key exists, has anyone seen VBA code to make one?

    Thanks

    Josh
    bill manville say:

    Sub LastCellInRow()
      Dim R As Range
      Set R = Cells(ActiveCell.Row, Columns.Count)
      If IsEmpty(R.Value) Then
        R.End(xlToLeft).Select
      Else
        R.Select
      End If
    End Sub

    Sub LastCellInColumn()
      Dim R As Range
      Set R = Cells(Rows.Count, ActiveCell.Column)
      If IsEmpty(R.Value) Then
        R.End(xlUp).Select
      Else
        R.Select
      End If
    End Sub
