stanysteve say:

If you're using the Printer Properties link in Word's Print page, settings made there are only for the current session. To make it permanent:

In Windows 7, open the Control Panel (Windows key + X, P) and click Devices and Printers. Right-click your printer (which should be set as the default printer) and click Printing Preferences.

Check the printer's Duplex (double-sided) option and click OK. Restart Word.

Now, each time Word starts, the "Print on Both Sides" option should appear on the Print page.

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
If Target.Address <> "$A$1" Then Call myClock

End Sub

step2

In a regular module paste in the following vba macros:

Sub myClock()
Range("A1").Value = Format(Now, "dd/mm/yyyy hh:mm:ss;@")
Call myUpdate
End Sub

Sub myUpdate()
Dim dt As Date
dt = Now + TimeValue("00:00:01")
If Selection.Address = "$A$1" Then Exit Sub
Application.OnTime dt, "myClock"
End Sub

now,

start the clock: double click in a cell (NOT in A1)

stop the clock: select cell A1
