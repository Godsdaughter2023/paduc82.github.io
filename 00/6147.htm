davehessling say:

Creating multiple lines of text in a text box

I've created a text element that I need to enter remarks into. I need to know how to enable multiple entries into that data element. When I hit enter to create a new line of text, Access takes me to the next data element. Any ideas? Thanks.
john w. vinson say:

You'll need to make this a MEMO type field rather than TEXT, since Text fields are limited to 255 characters (not very much for comments!) As Hans says you can change the Enter Key behavior property of the textbox; just so you know, you can also type Ctrl-Enter to enter a newline character.

If you have some table and want to record multiple separate remarks for each record, you may want to consider instead having a one-to-many relationship to a Remarks table. This would let you do things like timestamp each remark, capture the identity of the person making the remark, allow for more flexible searching... but at the cost of another table and subform to maintain.
