Unable to update and launch Store (Beta) - check the setting - Microsoft Community

elise [msft] say:

Unable to update and launch Store (Beta) - check the setting of automatic app update

The Store in Windows Technical Preview (Build 9926) does not support manual updates of apps. While going through setup of a new machine, please do not turn off automatic app updates. If you do so, you'll be unable to update apps manually. If the Store itself requires an update, you will be unable to use the Store. In order to turn auto updates back on, run the following command from an elevated command prompt:

reg add "HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\WindowsStore\WindowsUpdate" /v AutoDownload /t REG_DWORD /d 4 /f

There is another known issues with updating the Store while the Store is open. We suggest you to close the Store and wait. But this only works if auto-updates are enabled, and it may take up to a day to resolve itself.

Hi,

Try this

1. In sheet2, type the selective headings (of the 27 columns) from cell A1 to the right, say till T1.  When typing the headings, please ensure that the spelling of the headings in range A1:T1 (assumption) of sheet2 is the same as the spelling of the heading in range A1:Z1 (assumption) of sheet1

=IFERROR(INDEX(Sheet1!$A$2:$Z$24,SMALL(IF(Sheet1!$C$2:$C$24>0,IF(Sheet1!$E$2:$E$24>0,ROW(Sheet1!$A$2:$A$24)-ROW(Sheet1!$A$1))),ROW(1:1)),MATCH(A$1,$A$1:$Z$1,0)),"")

Change the $A$2:$Z$24 and $A$1:$Z$1 to your actual range.
