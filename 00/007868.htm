_elaine_ say:

Hello, FeebyFeeby

You are not missing anything - it is not the Skype Number that is being forwarded, it is your entire Skype account whose calls are being forwarded.  Skype does not provide a way to forward each Skype Number purchased on one account to a different mobile or landline number. 

The work-around others have used was to create a new Skype account (and now, a new Microsoft account), and start a new Skype Number subscription on that account.  Although this will facilitate call forwarding, answering calls to this second account will prove tricky as the New Skype desktop application does not support more than one account being signed in and used on the same computer.  (The older versions on Windows did this via what is called a secondary switch, and on Mac it was a bit of a challenge that was never officially supported.) The second account would have to log in on a different device.

Anticipating your next question, unfortunately subscriptions and paid-for features cannot be transferred from one account to another. 

You ought to be able to select which of your two Skype Numbers to use for caller ID purposes.  If this is still not showing as an option in your account dashboard, please contact Skype Customer Service directly to sort this.

When you contact Skype Customer Service, remember these tips:

  • Temporarily enable browser pop-ups as you will be requested to approve an instant message chat dialogue box when the chat application launches as a separate browser pop-up window. 
  • When you land on the Support Request page, select the Payments & Orders option and then Help with a Subscription.  Click Next.  A link should appear on the next page to Text Chat Support.
  • If you do not succeed in reaching an agent, try again using a different web browser, in-private browsing mode, as well as clearing the browser's cache and cookies.

Regards,

Elaine

Windows 10 can be installed in multiple partitions (parallel installations) without need of any additional licenses so long as you are installing the same edition that has already been installed on the device at least once.  What you cannot do for free is install Home on a computer that is running Pro and never has been a registered Windows 10 Home device, or vice-versa.
