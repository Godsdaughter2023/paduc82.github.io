    Thanks again for your great response.

    Create a new section using a Next Page section break.

    Make sure that none of the headers are linked to previous. This is a setting for each of the three potential headers in the section.

    Remove the watermark(s) from all headers in that section by deleting them from the header, not by using the Remove Watermark command.

    Insert your text into the new section that does not have the watermark(s).

    Header/Footer Settings Recap

    There is nothing static about Word. It is intended to be used as a word processor with text flowing from page to page. Watermarks are creatures of headers which are creatures of Sections. With different sections you can have different headers and footers, but the default, reasonably, is to continue from previous sections. You need to tell Word what you want.
