suzanne s. barnhill say:

Don't bother looking at the article/video Parmi suggested; it's about writing equations in Word, and that's not what you want when you just want to write a chemical formula. AFAIK, there is no way to make Word automatically correct such formulas (though you might post a suggestion for Microsoft at http://word.uservoice.com/), but perhaps you are unaware that there are keyboard shortcuts for subscript and superscript (and for chemical formulas I believe you need subscript rather than superscript as shown in your question).

Subscript: Ctrl+=

Superscript: Ctrl+Shift+=

These commands are toggles, so you can use the same command to toggle off subscript/superscript, but it is generally easier just to press Ctrl+Spacebar, which removes all direct font formatting.

scottgem say:

This is more an Excel question than an Access question. Tables linked to Access from Excel are one-way. meaning that you cannot make changes in Access to a linked Excel table. However, you can also link an Access table or query FROM Excel. The Excel sheet would then Update the linked data when you open the sheet or Refresh the links. 

However, such a Data Connection is linked to a specific Access file. If the link was originally created to an Access MDB in a specific location and the file name or folder has changed, the Data Connection needs to be updated. To do so, go to the Data ribbon in Excel and select Connections icon. In the list that opens select the Connection you want to modify and open the Properties. In the second tab it will show a Connection string with the folder and file being connected to as well as the table/queryname (or a SQL statement). You can just edit the Connection string to change mdb to accdb or any other changes needed.

Hi Carvalhoj,

On the Home Tab, click 'Find and Select' - Replace

In the 'Find What' box enter: R$ and a space

Leave the 'Replace what' box empty

Click 'Replace All'

Also, sometimes the editor forgets to apply the new color, and the change level is lost.

We have tried track changes - that does not work as the track changes document becomes to unstable after a number of authors work on it.  Also, the changes do not appear as marked revisions after the changes are accepted.

Styles are just as cumbersome since the text must still be highlighted, and the style applied.

Is there a way to change the color of all new text added to the document?

In a word, no. You have already eliminated Track Changes and styles, which are the only mechanisms Word offers.

When a document is issued as a revision controlled document, any revisions have to be identified in the document.  This is normally done with some type of indication in the margin such as a vertical bar, then a revision level notation.

stefan blom say:

You can set options for Tracked Changes so that you get the behavior you want. For example, you can have a Word document (or template) track edits but not formatting changes. The problem is that some of the display options for Track Changes are user- and machine-specific rather than document-specific, which means that they probably wouldn't be useful to you.

If the ability to set up numbered versions would suffice for you, have a look at MVP Graham Mayor's article at http://www.gmayor.com/SaveVersionsAdd-in.htm.
