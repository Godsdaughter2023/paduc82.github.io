original_red say:

I defragged the Sd card in my 5800 a couple of weeks back and it did speed things up a bit, but i know the card's life has been drastically shortened (i'll be fine as long as it lasts another year).

 

The one problem with defragging a SD card or SSD drive is that it kills write cycles - a much safer way is to copy everything from the disc onto your computer, format the SD/SSD and then put it all back on (in one operation - don't start one file operation and then start another at the same time). This has exactly the same (often slightly better) effect as defragging, but uses less write cycles (a lot of SSD drives carry a warning about using defrag, remember they are basically fast, high capacity SD cards with a SATA interface).


As a preventative measure: when copying things from your computer to the phone, do it as ONE operation, not several concurrent operations - which is the major cause of fragmentation.
