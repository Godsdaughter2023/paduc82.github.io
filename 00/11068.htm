Create a Folder shortcut on the Desktop that has links for all of the Programs/Apps that are installed on the computer. These links will open the Program/App when double clicked. You can then Right Click any of the apps in this folder and select the Create Shortcut option to place a shortcut on the Desktop for that app.

2. In the Location box, copy/paste the following line of text.

%windir%\Explorer.exe Shell:::{4234d49b-0245-4df3-b780-3893943456e1}

3. Click Next and name the shortcut All Apps. Click Finish.

Hi Paul,

Thanks for replying to us with the status of your issue.

As the issue still persists, after running Windows app troubleshooter, I suggest you to run Powershell commands to re-register Windows app and check if it helps.

Follow the below steps:
