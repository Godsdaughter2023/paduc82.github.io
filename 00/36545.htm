How did you get WRITE.EXE there?  I promise you it should not be there.  It hasn't existed for over 18 years.

 
You're referencing a LNK link (the Start menu shortcut) instead of the actual wordpad file.  Here's what you need to use in your batch file (or if this is the only command, you can just use this as your scheduled task -- no batch file needed)

Hi Jackos,

Thank you for posting your query on Microsoft Community.

I suggest you to create a system image of Win7/Win10 and apply it to SSD.

To create system image on Windows 10 follow the below steps.

  1. Type File history in the search area on the taskbar and select File history.
  2. Click on System image backup on the left bottom corner of the Window file history Window.
  3. Click on Create System Image on the left panel of the System Image backup windows.
  4. Select the driver, where you want to create system Image.
  5. Follow on screen instruction.

If you want to perform a dual boot you can refer to the below link and check if it helps:

http://answers.microsoft.com/en-us/insider/wiki/insider_wintp-insider_install/how-to-dual-boot-windows-10-with-previous-versions/9695dfc7-1c13-4d8d-b10b-587e78c6ac36
