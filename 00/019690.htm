To Uninstall Bing from Windows 8/8.1 - Microsoft Community

hughgo-a-go-go say:

To Uninstall Bing from Windows 8/8.1

To remove Bing from Windows 8.1

  1. Open the Windows Control Panel.
  2. Click on ‘Network and Internet’.
  3. Click ‘Internet Options’ in the pop-up ‘Network and Internet’ box.
  4. Click on ‘Programs’ tab.
  5. Click on the ‘Manage add-ons’ button which is in the ‘Manage add-ons’ section.
  6. Click on ‘Search Providers’ which is under ‘
    Types’ in the ‘Manage Add-ons’ pop-up box.
  7. Right-click on ‘Bing’ where is listed under the ‘Name:’ column.
  8. Click on ‘Remove’ from the drop-down menu.

Cheers,

Hugh

e.j. gun say:

You should try to reference the specific document in your code, and not the "active document" or "active window".  For example:

 

Sub MySub()
    Dim thisDoc As Document
'
    Documents.Add Template:= _
        "C:\Program Files (x86)\Microsoft Office\Templates\1033\MedianLetter.Dotx" _
        , NewTemplate:=False, DocumentType:=0
    Set thisDoc = Application.ActiveDocument
'
' Hide all open windows
'
    Application.Visible = False
'
' Do work on the new document
'
    thisDoc.InlineShapes.AddHorizontalLineStandard
'
' Show everything when we're done
'
    Application.Visible = True
End Sub

kaykayme say: Thank you.  How can I reference a specific document if it is new and has not been saved?

e.j. gun say:

See the line that says "Set thisDoc = Application.ActiveDocument"?  That's how you reference it.  The line before that created it as a new document from a template.

After that, you refer to "thisDoc" instead of "ActiveDocument" or whatever.

Note the "Dim thisDoc as Document" at the beginning.  That's what tells Word that "thisDoc" is an object that represents a Word document.

HTH,

Eric

If your first installation of Windows 10 was a clean install then you will not be able to activate the current copy without a retail product key.

It gets a little complicated, but essentially if you clean installed a preview back before the official release of Windows 10 then you had to maintain a continous upgrade path through the Insider builds in order to maintain activation.  But once you clean install after the official release that chain is broken.  

To activate now requires that the installation to have been an upgrade from Windows 7sp1 or 8.1 via the Get Windows 10 app or from any version of Windows 7 or 8 via an upgrade with Windows 10 usb you created with the media creation tool. 

In particular, build 9926 will not activate after a new installation because its product key was been blocked by Microsoft in mid-July.  Existing copies already activated continued to function.  I believe 9926 is now expired anyway.

You need to reinstall your old Windows 7 OS and upgrade that to the current Windows 10 in order to get Windows 10 for free.  

An upgrade means that you do NOT start the computer with the usb you made, but start the computer normally and then insert the usb and upgrade via the Windows 7 desktop.

Because your copy of Windows 7 is the Home edition, your Windows 10 will be also.

greg maxey say: Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oRng As Word.Range
Dim oPar As Paragraph
For Each oPar In ActiveDocument.Range.Paragraphs
  Set oRng = oPar.Range
  oRng.Collapse wdCollapseStart
    Do While oRng.Characters.Last Like "[1-9.^t]"
      oRng.MoveEnd wdCharacter, 1
    Loop
    If Len(oRng) > 1 Then oRng.Delete
Next oPar
End Sub
