ashish mathur say:

Hi,

Try this

1. Select A2:B15 and press Ctrl+F3 > New > Name.  In the Name box, type c_group

2. Select C2:D15 and press Ctrl+F3 > New > Name.  In the Name box, type hr_group

3. Select E2:F15 and press Ctrl+F3 > New > Name.  In the Name box, type sa_group

4. Delete + from A15,C15 and E15

5. Enter this formula in cell G19 and copy down

=D19*VLOOKUP(F19,INDIRECT(LEFT(B19,SEARCH(" ",B19)-1)&"_group"),2,1)

=ROUND((E22-C22)/30,0)
