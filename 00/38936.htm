a. user say:

Hello,

The Runtime Broker process was created by Microsoft and is a core process in Windows. If you’ve just logged into Windows 10 and haven’t run any apps yet, you probably won’t see RuntimeBroker.exe running yet. RuntimeBroker.exe is triggered by metro apps, and if the process is ended all currently open Metro apps will immediately fully close. 

The Runtime Broker handles checking if a Metro app is declaring all of its permissions (like accessing your Photos) and informing the user whether or not its being allowed. In particular, it is interesting to see how it functions when paired with access to hardware, such as an app’s ability to take webcam snapshots. Think of it as the middleman between your apps and your privacy/security.

Most of its related registry entries and the process itself can be found in these locations:

  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsRuntime
  • C:\Windows\System32\RuntimeBroker.exe

The error normally means that someone has activated Windows with a Volume License Key meant only for business or organizations that activate a large number of PC's, which need to check in with the organization's server regularly to reactivate.

Sometimes this is done by shops reinstalling Windows, unnecessarily because the PC either came with a Windows Product Key embedded in the BIOS/UEFI firmware, or it was already activated for free Upgrade and would have reactivated after a reinstall had they given it the chance and not interrupted this by installing the illegal key.

If you know Windows was originally activated legitimately by one of these methods then to fix the problem what I would do is go offline to remove the Volume License key and then restore the Generic Product Key that is used to activate with the Digital License issued by Upgrade or Factory install, the factory key embedded in the UEFI/BIOS firmware chip, or the retail key you used. 1) Go offline by opening Action center at the right end of Task Bar, then clicking on Airplane mode to turn off internet.

2) Next type CMD in Start Search, right click to Run as Administrator, then right click to copy and paste this command in and press Enter: slmgr -upk

3) Restart PC and then insert Generic Product Key for WIndows 10 Pro from here: http://winaero.com/blog/generic-key-to-install-...

This sets everything back to default so that if there is a Digital License issued during a proper Upgrade it should reactivate within a few days. If not we will need to try something else:

If that doesn't activate after going online, then read the Product Key embedded in the UEFI firmware chip at the factory by running ProducKey: http://www.nirsoft.net/utils/produkey-x64.zip and looking for the OEM-BIOS key, copy and paste it into Settings> Update & Security > Activation.

hothead75 say:

How do I reset a passkey on xbox one x even though I didn't set it up?

I was setting up my xbox one x, and when I was navigating through some of the screens, some of them required a pass key, though I never had a chance to set one up. How do I reset it?

andre' arnett-braun say:

Hello, hothead75!

I'm sorry that you're experiencing issues with the passkey.

To reset a passkey, you need to enter the wrong passkey in three times (if you have forgotten the passkey), and it will prompt you to enter you Microsoft account password instead. After completing this step, you can follow the steps below to either remove or change your passkey:

Remove Your Passkey:

  1. Press the Xbox button to open the guide.
  2. Select System > Settings > Account.
  3. Select Sign-in, security & passkey.
  4. Select Delete my passkey.

Change Your Passkey:

Once your passkey has been removed, you can create a new one:

      
  1. Press the Xbox button to open the guide.   
  2. Select System>Settings>Account. 
  3. Select Sign-in, security & passkey. 
  4. Select Create my passkey.
  5. Enter a six-digit passkey, and then re-enter it to confirm.

Let us know if this helps!
