swischner say:
First: Although it mostly works (there may be issues with some Linux based servers), Microsoft isn't officially supporting sharing OneNote notebooks on a network share anymore. Their official answer will always be "Use OneDrive, Sharepoint or OneDrive for Business".

About your specific issue: As Ugo Lopez already correctly stated, you apparently copied a ONEPKG file to the network drive. ONEPKG is just an archiving format (basically it's a ZIP file with a different extension) and only used to recreate a new notebook based on it's archived content. A locally stored notebook is made of more than just one file.

You have to either create the notebook in the network share (which ideally is mapped to a drive letter) from within OneNote (File, New, This PC, Create in a different folder) or move an existing locally save notebook by moving it's complete directory structure (not just single .ONE file as those are just sections).

A properly stored local notebook's structure is:

A folder with the notebook name
This folder holds .ONE files for every section in the notebook.
If there are subsections in the notebook they appear as subfolders in the notebook folder with more .ONE files inside.
If there are deleted notes there will also be a recycle bin subfolder
A "open notebook.onetoc2" file in the notebook folder. That's a kind of index file containing a list of the section and subsection structure.

If you copy or move that complete structure to your network drive, everybody who has read/write/create access to that network drive will be able to open the notebook from there. This can be done from within OneNote or by using the explorer and double-clicking the .onetoc2 file or any ONE-file in the same directory.

suzanne s. barnhill say:

If you have the ability to rescan the letterhead, then that's the way to go. Scan it in three separate pieces: the top, the side, and the bottom, and save those pieces as graphics (PNG for preference, or JPEG). If you can't do that, then you have a couple of other ways to go.

  1. Use Snapshot to select and copy the pieces of the letterhead from the PDF and paste them in. They probably won't be very high-quality, though if you increase the zoom on the PDF (in Adobe Reader) as high as you can and still get the piece all on the screen, then they'll be the best you can get.
  2. Insert the entire PDF, anchored to the header (First Page Header) and formatted as Behind Text. You'll need to set the position as Top Relative to Page and Left Relative to Page and make sure the size matches the page size. You'll then need to set your margins to clear the background pieces. If this is to be a multipage letterhead, set the header margin for what you want on the second and subsequent pages, then add space to the First Page Header to increase the effective margin on the first page.

If you can rescan the letterhead and save it in three pieces, then you'll insert the top piece in the First Page Header, the bottom piece in the First Page Footer, and the side piece anchored to the First Page Header. Unless you have cropped the graphics to the fit within the header/footer margins (in which case you can insert them In Line With Text), you'll need to format them all as Behind Text so that you can align them with the edges of the page (the side piece will have to be Behind Text or Square in any case).

Whether you can crop the graphics before you insert them depends on how you are scanning them. I usually scan directly into Corel Photo-Paint, which allows me to edit the scans before saving them.
