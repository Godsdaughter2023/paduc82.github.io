You can still copy the data that is on a hidden worksheet and in a protected workbook. So just copy the data to another workbook and print from there. It is not cracking passwords etc; just a feature of Excel.

If only a worksheet is protected then you can just add a worksheet but as you have said that you have protected Structure and Windows then it needs to be another workbook because cannot add a worksheet.

Sub PrintHiddenProtectedWorksheet()

    Dim wbThis As Workbook
    Dim wbPrint As Workbook
   
    Set wbThis = ThisWorkbook
   
    Set wbPrint = Workbooks.Add     'Becomes the ActiveWorkbook when added
   
    wbThis.Sheets("worksheetB").Cells.Copy Destination:=wbPrint.Sheets(1).Cells(1, 1)
   
    'Alternative Copy and PasteSpecial if problems experienced with Links or formulas
    'wbThis.Sheets("worksheetB").Cells.Copy
    'wbPrint.Sheets(1).Cells(1, 1).Cells(1, 1).Select
    'Selection.PasteSpecial Paste:=xlPasteValues, Operation:=xlNone, SkipBlanks _
        :=False, Transpose:=False
    'Selection.PasteSpecial Paste:=xlPasteFormats, Operation:=xlNone, _
        SkipBlanks:=False, Transpose:=False
   
    'Include code here for page setup if required
   
    ActiveWindow.SelectedSheets.PrintOut Copies:=1
   
    wbThis.Activate
    wbPrint.Close SaveChanges:=False

End Sub
