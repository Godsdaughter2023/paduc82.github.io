rohn007 say:

The install disk should have the 64 bit version in a separate folder. Look carefully at the folder names.

You can also download the installation file from MS. This is probably a better approach since the download should include all/most of the updates that have been issued. That eliminates the need to run Windows update multiple times to get all of the updates applied.

You can go to this webpage and submit your product key to download the installation file:

https://support.office.com/en-us/article/download-and-install-or-reinstall-office-365-or-office-2019-on-a-pc-or-mac-4414eaaf-0478-48be-9c42-23adc4716658?ui=en-US&rs=en-US&ad=US  (including downloads OTP 2010-2016)  Look for prompts to confirm if you are getting 32 or 64 bit. Check the file name also, there should be an indication.

Scotts comment was true until recently.  Just this year MS has finally changed their "official" position. A couple of months ago MS announced that the default Office install will now be 64 bit on 64 bit Windows.   The original decision to go with 32 bit was based on the knowledge that many/most addon makers, including MS!, had not bothered to create 64 bit versions of their addons. If people had installed 64 bit Office, they would have lost the functionality of those addons.  And, (independent) testing showed that there was no perceptible performance benefit to using 64 bit in most cases.  The one exception is in file size. 64 bit can handle significantly larger files. But very few people would encounter that size limit.
