I found the solution to the problem. Here are the instruction:

  1. Create envelope and add to the document with these steps.

1.1.   Click on “Mailings” in the menu bar.

1.2.   Click on “Envelopes” in the Mailings Menu.

1.3.   Enter the delivery address

1.4.   Click the “Add to Document Button”. 

  1. Make edits to the return address in the document to include any desired graphics.
  2. Set the Windows default Paper Source with these steps:

2.1.   Click on “Layout” in the Menu Bar.

2.2.   Click on the little arrow at the bottom right corner of Page Setup tab to expand the menu (see screenshot).

2.3.   Under page source, “First Page”, change the source from CD/DVD Tray to the correct source (e.g. Cassette 2), and then click on “Set as Default” button.

  1. Save the document.
  2. Print the first page using an envelope in the correct position.
