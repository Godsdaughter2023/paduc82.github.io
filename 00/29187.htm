Button mapping - Microsoft Community

sqrlley say:

Button mapping

when making profile 1 and adding to slot 1 it defaults back to default settings in slot 1 how do you change it to use profile one 

xbf diego m say:

Hey sqrlley,

I'm sorry to hear about the issue being experienced. I'd be happy to help.

I'm assuming you are talking about the Xbox Elite controller? As that is the only first party controller with the ability to switch between button mapping slots.

Whenever you try to add a profile to a slot and it reverts back to Default, that means the profile you made had no changes so it stays on Default. The slot name will change from Default to Profile 1 or however you named it when you make a button map change or change any of the other settings the Xbox Elite controller can configure. Once you save the changes, Slot 1 will change to Profile 1 or whatever you named it to.

A power cycle to your console may also be a good idea to start off fresh.

sqrlley say: Thank you for the answer it is not an elite controller so that answered why it defaults back due to it is a plain xbox one controller was just curious on why it kept defaulting back

xbf diego m say:

Hey sqrlley,

You can button map a standard controller as well. It's not unique to the Elite controller, it's only that the Elite controller has the ability to switch between two different profiles while you game by just flipping a switch on the controller from Slot 1 to Slot 2. You can make more than one profile on a standard controller, but you would just need to open up the Xbox Accessories app anytime you want to switch profiles.

When you open the app, it will show you a picture of your controller. Select "Configure" and then select "New Profile" you can name it however you want. Button map using the drop down menu you see or by holding the button you want to change and then you press the corresponding button you want to change it to. Once you've set up your button mapping press B to back out and then you can select the dropdown menu saying "No Slot" for the profile, from there select Slot 1 and it should now say Profile 1 or whatever you named it to.

Follow this guide for further information.

I hope this is helpful!

david. f say: This is by design.

As a work around, do the following:
  1. Right-click the program's file name or shortcut.
  2. Click Advanced, click Run with different credentials, and then click OK.
  3. Start the program.
  4. When you are prompted, type an administrative user ID and password.
http://support.microsoft.com/kb/893677

kg4vbo say: Thank you, that solved the problem. I had to sign into my administrator account, find the shortcut in the "Documents and Settings/<user name>/Desktop" folder, and click it there. I then ran the program once from my account, entering my password, and now I can access it from their limited user account as long as they uncheck "protect my computer and data from unauthorized program activity."
