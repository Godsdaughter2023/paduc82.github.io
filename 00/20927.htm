Excel Group & Sum - Microsoft Community

kirk p. say:

Excel Group & Sum

How can I programatically take this data

and grouping by like Category and GPS Class, provide the sum in each of the columns C through C+119?  Results would look like this:

hansv mvp say:

This version works on GPS Input, whether it is the active sheet or not, and whether it is visible or not:

Sub Summarize()
    Dim lngRow As Long
    Dim lngLastRow As Long
    Dim lngCol As Long
    Dim lngLastCol As Long
    Dim rngToDelete As Range
    Application.ScreenUpdating = False
    With Worksheets("GPS Input")
        ' Optional - make sure that the range is sorted
        .Cells(1, 1).CurrentRegion.Sort _
            Key1:=.Cells(1, 1), Key2:=.Cells(1, 2), Header:=xlYes
        lngLastRow = .Cells(Rows.Count, 1).End(xlUp).Row
        lngLastCol = .Cells(1, Columns.Count).End(xlToLeft).Column
        For lngRow = lngLastRow - 1 To 2 Step -1
            If .Cells(lngRow + 1, 1).Value = .Cells(lngRow, 1).Value And _
                    .Cells(lngRow + 1, 2).Value = .Cells(lngRow, 2).Value Then
                For lngCol = 3 To lngLastCol
                    .Cells(lngRow, lngCol).Value = _
                        .Cells(lngRow, lngCol).Value + .Cells(lngRow + 1, lngCol).Value
                    If rngToDelete Is Nothing Then
                        Set rngToDelete = .Cells(lngRow + 1, 1)
                    Else
                        Set rngToDelete = Union(.Cells(lngRow + 1, 1), rngToDelete)
                    End If
                Next lngCol
            End If
        Next lngRow
        If Not rngToDelete Is Nothing Then
            rngToDelete.EntireRow.Delete
        End If
    End With
    Application.ScreenUpdating = True
End Sub

How to have an image and page number in footer in MS Word? - Microsoft Community

shalinigandham say:

How to have an image and page number in footer in MS Word?

I am trying to make an official document, in which I have a brand element in the footer. Now when I try to insert page number the footer vanishes. 

Any solution around this?

I need the image and page number in the footer.

doug robbins - mvp office apps & services (word) say:

I assume that you are using one of the Built-In footers available from the Footer drop down in the Header & Footer section of the Insert tab of the ribbon.  

Instead of doing that, click on Edit Footer in that drop down which will open a blank Footer pane.

By default the paragraph in the Footer pane is formatted with a center aligned and a right aligned tab.  If it suits, you can therefore use tab spaces to position the elements that you want in the footer and I would recommend that the image be inserted so that it is in-line with the text.  To insert the page number move the selection to where you want the page number to appear and then press ALT+SHIFT+p, or from the Page Number drop down in the Header & Footer section of the Insert tab, click on Current position.

While it may not be necessary, using a borderless table can be a good way to control the layout of images and text in the footer (or for that matter, anywhere in a document).

If you can't get it set up exactly the way that you want, feel free to send me a copy the document\template, referencing this thread and providing details of the requirements in the covering email message, and I will help you.
