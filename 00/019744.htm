palcouk say:

If set up as IMAP on both devices any mail deleted will be deleted on both devices.

You cannot delete on the mobile but still have them available on the desktop.

If both devices were set up for pop, provided the account setup options on both devices are set to leave a copy on the server, you can delete on one device without impacting on the other.

But note any mail sent from one device will only show in that devices folders
