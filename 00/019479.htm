No, Excel does not provide a way to print a custom view directly - you have to show the view first, then print.

If you would like to be able to print a specific view quickly, you could create a macro, for example

 

Sub PrintMyView()
    ActiveWorkbook.CustomViews("MyView").Show
    ActiveWorkbook.PrintOut
End Sub

 

when you run the macro, the custom view is shown and the workbook is printed.

serds_p say:

Hi,

To pin your Skype app to the taskbar, simply launch the app and right click on the Skype icon on the task bar and select pin to task bar.

To create a Skype desktop shortcut, open your file explorer icon from the taskbar. Click this PC, then select your C: drive and open program files then look for the Skype file. Right click on the Skype file and send to desktop (create a shortcut). Take note that this will not work if you are referring to the built-in Skype for Windows 10 version.

If you are using Skype for Windows 10 (built-in Skype application for Windows 10 OS), create a desktop shortcut by dragging the Skype icon from the Apps list to your desktop.
