bill manville say:

For a purely functional solution (with no VBA code) you could do it using a deliberate circular reference.
You would need to set the calculation mode to manual and to enable iterative calculation (Office button > Excel Options > Formulas > Enable iterative calculation (check the box); maximum iterations = 1.
You would have a checkbox (i suggest from the Form controls set on Developer > Controls > Insert) linked to a cell for each die.
If the dice values are in B2:F2 and the check boxes are linked to cells B3:F3 then the formula in B2 would be
=IF(B3,B2,RANDBETWEEN(1,6))
and you can copy the formula to the other dice value cells (C2:F2)

When you press F9 the dice which have not been held by checking the relevant box should change.

However, if you are using any VBA macros in your workbook then it would be simplest to get the VBA to set the dice values.
