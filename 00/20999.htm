tiger glider say:

If you're seeing something that looks like Microsoft Office 15_Data:live:cid=<numbers_and_letters>

then this is a saved password for Office. Specifically, the CID (Customer ID) is an identifier for your Microsoft account.

Office uses your Microsoft account to so several things, but mainly so you can save documents to a private OneDrive online document folder.  Saving documents to your OneDrive folder is very secure.  It not only protects you from losing documents if your hard drive crashes, it is also super nice if you have several computers since you can save your stuff to your OneDrive online folder, then you can access those documents from any of your machines, without having to copy them onto a removable drive, and finding yourself with multiple copies of the same document with different edits. 

Geek Details:

This is an entry in the Credential Manager, which is where Windows safely stores saved passwords.

You can find this by going to Control Panel / User Accounts / Manage your credentials / Windows Credentials.

You can delete this entry safely, but shouldn't need to.  If you do, Office will just prompt you for your password the next time you open any of the Office applications.  Since you don't want to have to enter your password every time you open Word, this is a good feature and nice convenience.  CredMan is secured using DAPI (a crypto API) which is very secure.

Hope that helps,

Scott
