Hi,

take a look the sample below

1) sort by column D (custom list)

v = Array("XA", "PA", "AA-01", "QA")

2) sort by column E (descending order)

pic1 (before)

2) pic2 (after)

vba macro

Sub Sort_by_Columns_D_E()
Dim ws As Worksheet
Set ws = ActiveSheet
v = Array("XA", "PA", "AA-01", "QA")
Dim r As Long, c As Long
r = ws.Range("A1").CurrentRegion.Rows.Count
c = ws.Range("A1").CurrentRegion.Columns.Count
With ws.Sort
.SortFields.Clear
.SortFields.Add Key:=ws.Range("D2:D" & r), SortOn:=xlSortOnValues, CustomOrder:=Join(v, ","), DataOption:=xlSortNormal
.SortFields.Add Key:=ws.Range("E2:E" & r), SortOn:=xlSortOnValues, Order:=xlDescending, DataOption:=xlSortNormal
.SetRange ws.Range("A1").Resize(r, c)
.Header = xlYes
.MatchCase = False
.Orientation = xlTopToBottom
.SortMethod = xlPinYin
.Apply
End With
End Sub

Using VBA to find a range of cells containing a constant - Microsoft Community

no1_uknow say:

Using VBA to find a range of cells containing a constant

Hello,

 

I am trying to determine the code need for VBA to search a row (row 24) that contains a constant (e.g. Jan, Feb, etc) and determine the start and end address of that data so I can name that range.  The data will always be consectutive.

 

For example:

       A          B         C       D       E       F        G       H         I           J          K         L        M         N

24| JAN    JAN    JAN   FEB   FEB   FEB   FEB   MAR   MAR   APR   APR   APR   APR   APR

 

I would want the VBA to find all the JANs and determine the range as A24:C24 so I can name that range JAN, and APR would be J24:N24, etc.

 

Thanks for the help.

ronrosenfeld say:

This should get you started.  The code looks at row 24 and extracts a list of unique values (constants) into a collection.

These values are then used to populate a range array with the ranges corresponding to each unique value.  You could then use these to develop NAME'd ranges.

======================================
Option Explicit
Sub NameRanges()
    Dim r As Range, c As Range
    Const RowNum As Long = 24
    Dim colConstants As Collection
    Dim colItem As Variant
    Dim v() As Range
    Dim i As Long
    Dim FirstAddress As String

Set r = Range(Cells(RowNum, 1), Cells(RowNum, Columns.Count).End(xlToLeft))
Set colConstants = New Collection

'generate list of constants
On Error Resume Next
    For Each c In r
      If Len(c) > 0 Then _
        colConstants.Add Item:=c.Text, Key:=c.Text
    Next c
On Error GoTo 0

'populate range array
ReDim v(1 To colConstants.Count)
For Each colItem In colConstants
    i = i + 1
    With r
        Set v(i) = .Find(what:=colItem, LookIn:=xlValues, lookat:=xlWhole)
        FirstAddress = v(i).Address
        Set c = v(i)
       
        Do
            Set c = .FindNext(after:=c)
            If c.Address <> FirstAddress Then
                Set v(i) = Union(v(i), c)
            End If
        Loop Until c.Address = FirstAddress
    End With
Next colItem

'See the results
For i = LBound(v) To UBound(v)
    Debug.Print v(i)(columnindex:=1).Text, v(i).Address
Next i
   

End Sub
=============================================
