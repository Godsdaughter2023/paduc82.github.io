You cannot create Normal.dotm by saving another file under this name or by renaming a file. You have to start with the Normal.dotm you are given and make changes to it, then save.

If the changes you are making to the defaults are not being saved, then perhaps an add-in is preventing the Normal template from being saved. In such cases, the ultimate solution is to identify the offending add-in and disable it. As a temporary measure, however, you can open the Normal template directly for editing. If you know how to locate and open Normal.dotm, you can do so. If not, here's a shortcut:

In Word, press Alt+F11; the Visual Basic Editor opens. Press Ctrl+G to activate the Immediate window. In that window, type (or paste)

 

NormalTemplate.OpenAsDocument

 

and press Enter. Normal opens, but note that it may not be on top of all windows. You can close the Visual Basic Editor. Make the desired changes to the Normal file, and then save and close it. If the changes you have made have not been recognized as changes (the Undo button is not activated), press Spacebar, then Backspace to “dirty” the file, thus ensuring that the template will actually be saved.

suzanne s. barnhill say:

Alt+Ctrl+Shift+S is just the easiest way to open the Styles pane. You can also open it using the dialog launcher (small arrow) in the bottom right corner of the Styles group on the Home tab.

Normal.dotm is the new template because it is in the new file format used by Word 2007 (just as the files you create are .docx rather than .doc). If you installed Word 2007 over an older version, your template should have been saved as NormalOld.dot, and most of your customizations will be retained in the new version. These include macros, shortcut keys, AutoText, and the like but do not include styles and formatting.

If you are not being prompted to save the Normal template, then go to Office Button | Word Options | Advanced: Save and make sure that "Prompt before saving the Normal template" is checked. If you have already explicitly saved the template when you opened it and made changes, then you will not be prompted to save it again when you quit Word, but if you have made changes in a document and told Word to save them as the default for new documents, you should get the prompt, and, if you don't, this is an indication that an add-in is preventing Normal from being saved.

Many changes that you make in the Normal template (such as changing the view or zoom) don't register as "changes" that can be undone (the Undo arrow is not enabled). If you click Save in such instances, the template will not actually be saved because Word does not regard it as "dirty" (changed). In order to force Word to save it, you have to make a change that dirties the file.
