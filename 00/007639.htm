ken sheridan say: If you first create and fill a table TimeSlots with all of the date/time value against which you wish to check the occurrence of calls, then you should be able to return the count of calls in progress at each time with a query like this:

SELECT TimeSlot, COUNT(*) AS NumberOfCalls
FROM, TimeSlots, CallLog
WHERE TimeSlot >= StartTime
AND TimeSlot <=End Time
GROUP BY TimeSlot;

You can easily fill the TimeSlots table, e.g. with every 15 minute time interval for 2016, 2ith:

     Dim dtmTimeSlot As Date
     Dim strSQL As String
     
     For dtmTimeSlot = #1/1/2016# To #12/31/2016 11:45:00 PM# Step #12:15:00 AM#
        strSQL = "INSERT INTO TimeSlots(TimeSlot) VALUES(#" & _
            Format(dtmTimeSlot, "yyyy-mm-dd hh:nn") & "#)"
        CurrentDb.Execute strSQL, dbFailOnError
    Next dtmTimeSlot
