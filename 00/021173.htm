ERROR MESSAGE:

CANNOT FIND https://hubblecontent.osi.office.net/ContentSVC/Content/Download?provider=MicrosoftIcon&fileName=Document.svg

Make Internet Explorer your default web browser.  Microsoft products depend upon Internet Explorer to access the online library.  It throws an error when Google Chrome is the default browser.  I know... me too.

scott mcdaniel say:

You can create an Alias for your returned query fields. For example:

SELECT (First_Name + ' ' + Last_Name) AS FullName, DateOfBirth AS [Date Of Birth] FROM SomeTable

Note if you're Grouping, you have to use the full "alias" in the Group By section of the statement. For example:

SELECT (First_Name + ' ' + Last_Name) AS FullName, DateOfBirth AS [Date Of Birth] FROM SomeTable Group By First_Name + ' ' + Last_Name

In other words, you don't use the "aliased" name, but rather the expression you used when creating that alias.

To create an alias in the Query Design Grid, use this syntax in the Name row:

FullName: First_Name + ' ' + Last_Name
