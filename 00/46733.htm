Hi Yusuf. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

Open File Explorer, right click C to view Properties. Is there enough space for your files? If not Please post a screenshot of Disk Management, which I read like a doctor reads X-rays. Follow the steps here so I can see everything needed to advise you: http://answers.microsoft.com/en-us/windows/foru...

Some PC's now have a separate Data partition so you can move the User folders there to keep the C image smaller for imaging purposes. Then if C ever becomes irreparable you can reimage C in 20 minutes with a great free imaging app like Macrium and the data will be current and safe in it's own separate partition.
