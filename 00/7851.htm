Workaround to fix “Server Stumbled” issue with Windows 10 - Microsoft Community

venkat iyer say:

Workaround to fix “Server Stumbled” issue with Windows 10 Store on PC (error code 0x80072EFD)

We have seen multiple reports of Windows 10 Store users running into “Server Stumbled” issue with error code 0x80072EFD. This error manifests in Network connectivity issues from the Client to the Store Services. There is a workaround available to get beyond this issue and the steps are listed below: 

1) Using an administrator account on the affected machine, open the registry editor and navigate to the following key “HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\NetworkList\Profiles”

2) Perform the following steps

a) Right click on the Profiles key and go to Permissions.
b) Click Advanced
c) Check the ‘Replace all child object permission entries with inheritable permission entries from this object’
d) Click Apply

3) Then restart the machine.

4) Try connecting the Store and you should not see the 0x80072EFD anymore

- Venkat (on behalf of the Store Quality team)

stefan blom say:

Clearing the "Track formatting" option for the Normal template only affects future documents that you create based on the template.

For an existing document, you have to (a) turn off format tracking and (b) accept any tracked formatting already present in the document.

(a) is discussed in my previous reply in this thread, but if you are using Word 2013, there are additional steps required: Click the dialog launcher button in the Tracking group on the Review tab, click Advanced Options and then clear "Track formatting." 

Here's how to do (b): On the Show Markup drop down, deselect all options besides "Formatting." Then click Review tab | Accept | Accept All Changes Shown.

Once you have performed the steps above, it doesn't matter if "Formatting" is selected on the Show Markup drop down (it usually is selected).

stefan blom say:

How did you save the change to the Normal template? Did you open the template as a document, via File | Open in Word, and then clear "Track formatting" in the Track Changes dialog box (called Advanced Track Changes dialog box if you are using Word 2013)? Doing this prevents Word from tracking format changes in the future (in new, blank documents that is).

The check mark next to "Formatting" in the Show Markup drop down is more difficult to get rid of permanently... 

In fact, Word forces the display of tracked changes in any document that you open. You can control this to a certain extent after you have cleared "Make hidden markup visible when opening or saving" at File tab | Options | Trust Center | Trust Center Settings | Privacy Options, but you can't control it fully. Note that "Make hidden markup visible..." is not document-specific; it only applies to your machine. When someone else opens the document, what they see is determined by settings on their machine.

In other words, the best you can do is turn off format tracking in the Track Changes dialog box for the relevant templates and documents, accept any existing format changes, and ignore the "Formatting" option on the Show Markup drop down.
