Windows 10 upgrade leading to mail stuck in outbox in Outlook 2013 - Microsoft Community

steveb_946 say:

Windows 10 upgrade leading to mail stuck in outbox in Outlook 2013

There are various posts going back some years about mail getting stuck in your Outlook outbox. Here is a situation which I resolved today with the help of Microsoft Support which I hope will help other users in a similar position who are just as non-technical as me.

PROBLEM

I was finding that emails which I thought I had sent were remaining in my Outlook 2013 outbox for no apparent reason.

SOFTWARE CONFIGURATION

I bought a new PC last week which came pre-installed with Windows 8.1. The first thing I did was take the free upgrade to Windows 10, after which I installed a new copy of Office Home & Business 2013.

SOLUTION

Having double-checked that I was not trying to send overly big attachments, that I was online and connected to the Outlook server and that I had not inadvertently clicked the Send/Receive 'Work Offline' button I figured I needed more detailed technical support.

The Microsoft helpdesk person told me that this is a known problem resulting from the Windows 10 upgrade. You should first exit from Microsoft Outlook and then apply a fix as follows:

1. Type CMD into the search bar (bottom left of your screen) but do not press Enter

2. This will reveal 'Command Prompt' desktop app as the best match at the top of your Start menu

3. Right-click on Command Prompt and choose 'Run as administrator'

4. Click 'Yes' to verify that you do want to do this

5. In the Command Prompt window, type sfc /scannow (i.e. type the letters 'sfc' followed by a space, then a forward slash, then 'scannow') and press Enter

6. You have now instigated a system scan which takes a few minutes to complete

7. Once the scan progress reaches 100% you should get a message saying that 'Windows Resource Protection found corrupt files and successfully repaired them.'

8. Restart Outlook 2013 and you should now be able to successfully send email
