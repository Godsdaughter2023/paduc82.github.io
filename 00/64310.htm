This issue might be due to hardware issues or services might have turned off.

Method 1: Run Hardware troubleshooter as it will help in resolving the issues with hardware and resolve any simple issues found.

  1. Type troubleshooting in the search tab on the desktop and press Enter.

  2. Select “Hardware and Sound” from the Troubleshooter Window.

  3. Select “Hardware and Devices”.

  4. Follow the on screen instructions.

Method 2: To remove drivers from Programs and Features:

  1. Right click on Start button, and then select Programs and features.

  2. Under Uninstall or change the program, select the Bluetooth driver to uninstall.

  3. Right click on the driver and select Uninstall.

  4. Once the Bluetooth drivers are uninstalled, restart the computer.

If the issue still persists then follow the below:

Method 3: Check if the Bluetooth Service is started in the computer.

  1. Type Services.msc in the Run prompt. (Press Windows key and the R key to open the Run prompt)

  2. Locate Bluetooth Support Service. Right click on it and click on Properties.

  3. Select the Startup type to Automatic and then Start the Service.

  4. Use the Function key combination to enable the Bluetooth Device.

For more information, go through the below article

http://windows.microsoft.com/en-IN/windows-8/why-isnt-windows-finding-device
