hansv mvp say:

Toggling AllowEdits shouldn't cause the On Current event of the form to fire, but toggling DataEntry will certainly do that.

There is no global EnableEvents in Access.

You could create a variable at the top of the form module:

 

Private blnStopEvents As Boolean

 

In your code:

 

Private Sub cmdOldNewMode_Click()
    blnStopEvents = True
    ...
    ...
    blnStopEvents = False
End Sub

 

and in the On Current event:

 

Private Sub Form_Current()
    If blnStopEvents Then Exit Sub
    ' your code here
    ...
End Sub
