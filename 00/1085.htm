myall_blues say: I am in the habit of shading all cells that contain a formula on my worksheets to a grey colour so they are easy to spot. The code below does that and should give you a start with what you are trying to do.
I chose to do it in two steps - first find all the formulas (Sub FindFormulae) and then do the shading (Sub Colour Formulae) but there is no reason you couldn't combine these into one step. There's also a sub to clear the shading on selected cells.
Obviously you could hook each one up to a custom macro button, or I put a menu front end on it and saved it as an Excel addin that then appears whenever I open Excel. If you would like the more complete code for an addin let me know - mine has other bits and pieces in it. 

Sub FindFormulae()
    '
    ' M. Tyler
    '
    ' Find and select all formulae in a workbook
    '
    Dim strPrompt As String
    On Error Resume Next
    Err.Clear
    Selection.SpecialCells(xlCellTypeFormulas, 23).Select
    If Err.Number = 1004 Then
        strPrompt = "No formulae found in selected area"
    Else
        strPrompt = "An unspecified error has occurred"
    End If
    If Err.Number <> 0 Then
        MsgBox strPrompt, , "Error"
    End If
End Sub
Sub ColourFormulae()
    '
    ' M. Tyler
    '
    ' Shade cells containing formulae to grey
    '
    Dim FormulaCells As Range
    '
    ' Apply borders/fill to selected cells
    '
    For Each FormulaCells In Selection
        If FormulaCells.HasFormula Then
            FormulaCells.Borders.LineStyle = xlContinuous
            FormulaCells.Borders.Weight = xlThin
            FormulaCells.Borders.ColorIndex = 48
            'FormulaCells.Interior.ColorIndex = 22
            FormulaCells.Interior.ColorIndex = 15
            FormulaCells.Interior.Pattern = xlSolid
            FormulaCells.Interior.PatternColorIndex = xlAutomatic
        End If
    Next
End Sub
Sub ClearFormulaeShading()
    '
    ' M. Tyler
    '
    ' Clear shading and borders set on cells containing formulae
    '
    Dim FormulaCells As Range
    For Each FormulaCells In Selection
        FormulaCells.Borders.LineStyle = xlNone
        FormulaCells.Interior.ColorIndex = xlNone
    Next
End Sub

a. user say:

Hi Michael Lofty,

Welcome to Microsoft Community.

I understand how frustrating it could be for you, when the screen color changes every time you insert a USB drive. Don’t worry, we will help you to resolve this issue.

This issue might occur due to the graphic drives not updated.

Method 1: Update drivers.

I suggest you to update graphic drivers and check if it helps. Follow the steps given below.

  1. Press Windows key + X key and click on Device manager.
  2. Search for Display Adapters.
  3. Right-click on the graphics card and click on Update driver software.

You may refer to the following Microsoft article provided below.

http://windows.microsoft.com/en-in/windows-8/all-drivers

If the issue still persist, follow method 2.

 

Method 2:

 

Step 1: Safe Mode with Networking

I suggest you to Start Windows in Safe Mode with Networking and check if.

Safe Mode is a troubleshooting option for Windows that starts your computer in a limited state. Only the basic files and drivers necessary to run Windows are started.

Follow the instruction provided below. 

  1. Press the ‘Windows + C’ key on the keyboard.
  2. Click on Settings, Change PC Settings and then General.
  3. Scroll down to Advanced Startup, click Restart now.
  4. Click on Troubleshoot, Advanced options, Startup Settings.
  5. Click Restart, select Safe Mode with Networking and then press Enter.

 

Note: Restart the computer to start in normal mode.

 

If the issue is doesn’t occur in Safe mode with networking then place the computer in a Clean Boot and check.

 

Step 2: Clean boot

A Clean boot is troubleshooting option for Windows in which all the third party services are disabled and checked, if this services are causing the problem.

Follow the Microsoft help article to perform clean boot.

https://support.microsoft.com/en-us/kb/929135

Note: The steps for Windows 8/8.1 applies to Windows 10 Technical Preview.

Hope it helps. Do let us know if your issue is resolved. For any further query, feel free to contact us.

Thank you.
