cmdrkeene say:

That's the default for Windows 8, so there's no "fix" persay, because user folder names cannot be changed after they are created.


However, you can create a new local user account named Ray (do not create it as a Microsoft Account).  The folder will then be called Ray.  You can drag all your files over to that user folder and then delete the old user account (which will delete the 000 folder also).  Then when you link your local Ray account to your online Microsoft Account, it will not rename the folder (because like before, it cannot be changed after it is created).
