The 360 is intended to be used for media playback, not media serving. Besides the console baring copying of untrusted content to its HDD, it has no facilities to "serve" content to something else.

The 360 does not permit copying of data from external sources to trusted storage devices. It is not possible to copy video, pictures, or audio files to the 360's HDD or USB Memory Unit storage devices.

 

XBOX Music does not allow songs to be downloaded to the 360. They can be streamed. That's it.

 

The 360 is a playback device. It's not a serving device. It not only doesn't have the storage capacity, it doesn't have the software required to serve files.

Screen resolution changes on external monitor when closing lap top lid - Microsoft Community

rkpatel say:

Screen resolution changes on external monitor when closing lap top lid on Dell Studio 1737

With some trouble, I can set up the display resolution on my Dell Studio 1737 17" lap top display (to 1440x900) and 15" external monitor (to 1024x768).  I have then changed my Power Management settings so that when I close my lap top cover/lid, my lap top does not go to sleep and the display switches to my 15" external monitor only.  However on closing the lap top, the external monitor resolution then changes to something like 800x600 making the screen completely unusable as the text is too big.

I have an "Intel Graphics Media Accelerator Driver for Mobile" and "Mobile Intel(R) Series Express Chipset Family".  I have checked and my driver software is up to date.

How can I get my external monitor to keep its resolution setting to 1024x768 when I close the lap top lid?

a. user say:

rkpatel,
Thank you for visiting the Microsoft Answers community forum.

This problem occurs when the monitor driver is unloaded when the lid of the computer closes.  In this case, the Transient Multi-monitor Manager (TMM) reduces the resolution to the default resolution because it cannot verify the monitor.

To work around this problem, stop the TMM (Transient Multi-monitor Manager) service. To do this, use one of the following methods.
Method 1: Use Task Scheduler
1.  Click Start , type Task Scheduler in the Start Search box, and then click Task Scheduler in the Programs list. 
2.  Expand Task Scheduler Library, expand Microsoft, expand Windows, and then click MobilePC. 
3.  Right-click TMM, and then click Disable.

Notes • If the TMM task is not displayed, right-click MobilePC, point to View, and then click Show Hidden Tasks. 
• This fix does not disable any of the docking features on the portable computer. If you find that an external monitor does not work correctly, you can re-enable the TMM task. 

Method 2: Use a script
1.  Click Start , type cmd in the Start Search box, and then click cmd in the Programs list.

 If you are prompted for an administrator password or for a confirmation, type the password, or click Continue. 
2.  At the command prompt, type the following command, and then press ENTER:
Schtasks /End /TN “\Microsoft\Windows\MobilePC\TMM” 

If this does not work you should check Dell's support site.  I saw this issue discussed on HP's web site, the same exact problem, and their fix was to download a new Intel video driver, not sure if Dell recomends the same.

Let us know if this was helpful or if you still need assistance.
