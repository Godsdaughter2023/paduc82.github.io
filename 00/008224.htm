Use timing, animation, or sound in a presentation with hyperlinks - Microsoft Community

ellen finkelstein say:

Use timing, animation, or sound in a presentation with hyperlinks

Technical Level : Intermediate

Summary
This article will teach you how to solve a problem with timing, animation, or sound that don't play in a presentation with hyperlinks. 
Details

The problem: If you create a timed presentation that automatically advanced after a certain number of seconds -- usually done in a kiosk situation where viewers view the presentation on their own -- and include hyperlinks to allow viewers to review the presentation, after they click a hyperlink to go backwards, the presentation stops advancing. Or animation doesn't work/a sound doesn't play  the second time a slide is displayed.

Apparently, PowerPoint was designed so that if you hyperlink backwards to a slide that you've already viewed, automatic timing stops and animation/sounds don't play.

The workaround: Create duplicates of each slide. Then set the timing for the first copy of each slide to 00:00. Because it's an exact duplicate of the following slide, you don't notice the duplication at all. (You might think that you wouldn't see any slide set for 00:00 timing, but you actually see it for a split second, so if the slide is not an exact duplicate of the next slide, you'll notice it.) Then hyperlink to the first slide of each pair of duplicates.

I found that the easiest way to do this is the following:

  1. Complete the presentation, including the timings.
  2. Click each slide  (in the outline pane on the Slides tab or in slide sorter view). Press Ctrl+C to copy it, and immediately press Ctrl+V to paste it. You get a copy right after the original slide.
  3. Select the first copy of each slide, choose Slide Show>Slide Transition, and change the timing to 00:00.
  4. If you have added sound to any slide, remove the sound icon from the first copy of each slide (the 00:00 ones). You wouldn't want the sound to start to play twice.
  5. Create the hyperlinks. You can copy and paste them throughout the presentation. What's important is that the hyperlinks link to the first copy of each slide, the one set for 00:00. Because it has no timing, PowerPoint won't stop the timing when you hyperlink back to it.
  6. If you use Action Buttons, set them for hyperlinks. Instead of setting them to Previous Slide or Next Slide, in the Hyperlink To drop-down list of the Action Settings dialog box, scroll down to Slide. Then choose the first copy of the previous or next slide.

If you later change a slide, be sure to delete its copy, copy and paste again to duplicate it, and change the first one's timing.

If you use the Slide Numbering feature to number the slides, the numbers won't be correct; instead you need to manually add text to indicate the slide numbers.

This original post, Use timing, animation, or sound in a presentation with hyperlinks, is on my website.

liza.z say:

Hi Jason,

 

Thank you for posting in Microsoft Community Forums.  Please be assured that we will do our best to answer the question

 

As per the description, you want to disable the message “Show All Content” in Internet explorer 9.

If it happens every time you open IE, then chances are your home page has this feature. It normally only occurs when it encounters a page where that situation exists. But you probably don't want it happening anywhere whether at startup or during a browsing session.

 

a.      Go to Tools,

b.      Select  Internet Options,

c.       Click on Security and then Internet / Custom Level.

d.      Scroll to one marked "Display mixed content". It is probably set on prompt.

e.      To remove the message, choose either Enable or Disable (you probably want Enable) and click OK.

f.        Close all IE windows and tabs to make it effective and the next time you open IE and go to a page with that feature; it will automatically open both secure and unsecure content without giving you that message. You can of course choose to Disable the option as well so only secure content is displayed - but that's your decision to make.

 

Do let us know if you have any other queries pertaining to Windows in the future. We will be happy to help. We, at Microsoft strive towards excellence and provide our customers with the best support.
