vanderghast say:

Assuming there is no space character in the data:

Replace( LTRIM(Replace( fieldName, "0", " ")), " ", "0")

 

The part in bold replace all zeros by spaces, and then, left-trim that result, ie, remove starting spaces. The external Replace replaces the spaces left in the string to their initial 0 character.

ken sheridan say: Try this:

Public Function LTrimZeros(varNumber)

    Dim strChr As String
    Dim n As Integer
   
    strChr = "0"
    n = 0
   
    If Not IsNull(varNumber) Then
        Do Until strChr <> "0"
            n = n + 1
            LTrimZeros = Mid(varNumber, n)
            strChr = Mid(varNumber, n, 1)
        Loop
    End If
       
End Function

Let's try to apply these steps in case you have not done them before.

1.- Open Microsoft Word to the document containing the footer you want to adjust.
2.- Select the "Insert" tab located at the top-left corner of the program window.
3.- Look to the "Header & Footer" section of the resulting menu bar. Click the "Footer" button and then select the "Edit Footer" option from the sub-menu that follows.
4.- Select the "Design" tab that appears along the top of the program window.
5.- Look to the "Position" section of the resulting menu bar and locate the "Footer from Bottom" option.
6.- Click the upward-facing arrow to increase the height of the document footer, or click the downward-facing arrow to decrease its height.
7.- Click "File" and select "Save" to confirm the change.
