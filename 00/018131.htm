robf-yorkshire say:

As a short-term fix.  If you create a html file (create text file and change the extension to html) and put into it the following:

 

<INPUT TYPE="button" VALUE="Add Search Provider" onClick='window.external.AddSearchProvider("http://www.iegallery.com/DownloadHandler.ashx?ResourceId=13438");'>

 

Save it, and then double-click on it to open, now just click the button to add Google as your provider.
