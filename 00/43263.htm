a. user say:

There are several suggestions so as you work on them just do one thing at a time, reboot and see how things look before moving on to the next.

Don't make a bunch of changes all at once or you won't know which adjustment fixed the problem or how to undo things in case things get worse.

Notice in your Task Manager that there is a svchost Process consuming at that moment 25% CPU - that is not normal.  If you watch it for a while it will probably bounce around constantly.

The svchost Processes are hosting Windows Services.  That is why the description reads "Host Process for Windows Services".  There might be one or more important Windows Services running under each svchost Process you see.  It is normal to see several svchost Processes running at the same time.  It is not normal to see one of them consuming a bunch of CPU when the system is idle.  They should be consuming very little CPU (usually zero) and not be consuming 25% of the CPU.

Hi Tricia

What you are describing is normal and by design.

If you look closely at those folders, you will see a Shortcut Icon on those folders. This shortcut icon identifies those folders as Junction Points. A Junction Point is used to redirect windows components to a different location.

Some folder names that were used in earlier versions of windows have been changed in Windows 8x.

Components such as software installation programs, for older programs, are Hard Coded to look for the older names of these folders. This can cause these installations to fail. The Junction will redirect these installation programs to the new folder name and installation will succeed.

Here is a list of the Junctions that you named and the new location in Windows 8.1.

My Documents = C:\Users\user name\Documents.

Local Settings = C:\Users\user name\AppData\Local.

Nethood = C:\Users\user name\AppData\Roaming\Microsoft\Windows\Network Shortcuts.

Printhood = C:\Users\user name\AppData\Roaming\Microsoft\Windows\Printer Shortcuts.

Send To = C:\Users\user name\AppData\Roaming\Microsoft\Windows\SendTo

Start Menu = C:\Users\user name\AppData\Roaming\Microsoft\Windows\Start Menu

Templates = C:\Users\user name\AppData\Roaming\Microsoft\Windows\Templates

These Junctions cannot be used as a regular folder, since doing so can cause corruption on the system. To prevent this all Junction Points have the Access Denied - Everyone  security setting.

Let us know if you have more questions.
