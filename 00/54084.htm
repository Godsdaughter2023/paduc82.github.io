In order to increase performance, Windows can enable write caching, which simply means that the system writes data to a temporary location in RAM (e.g., when you click "save" on a file) and then goes on to do other things.  In the background, the system transfers the data from RAM to the more permanent storage (e.g., hard drive).  The danger here is that if you abruptly disconnect the hard drive, there may be data left in the cache that the file system "thinks" was saved but that has not yet actually been written to the drive.  "Safely remove hardware" flushes the RAM cache, ensuring that all data that was supposed to be written to the drive is actually written.

This feature is controlled from the hard drive's Properties > Policies tab, accessed from Device Manager.

If you're going to format the drive anyway, it doesn't really matter whether you just disconnect it -- but doing so might cause the system to throw repeated errors until the next restart.

There are a number of possibilities for things that can keep the drive "open."  You can use a tool such as Process Explorer to track down what process is accessing a file or files on the drive and then stop that process, or probably more easily, just re-start the computer in Safe Mode,then click the "Safely Remove Hardware" icon, and the drive should have no problem stopping.
