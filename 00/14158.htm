Something that sometimes works in this situation is as follows:

 h. Re-open, and see if the data source is now re-connected automatically

 i. If you had any sorts/filters, re-create them, then save the document and re-open

steve rindsberg say:

Try viewing the same presentation on one or more other computers.  If the problem only appears on your computer or on your and similar computers, it could be a video driver problem.

To help verify whether this is the case, go to File | Options | Advanced

Scroll down to Display

Put a checkmark next to "Disable Slide Show hardware graphics acceleration" and "Disable hardware graphics acceleration"

Restart PowerPoint and see if that's solved the problem.

If so, try removing each of the above checkmarks in turn then testing again to see if one or the other is the solution to the problem or whether you need both checkmarked.

If that sorts it out, you can either leave things as is and go on with life, or visit the video driver manufacturer's site (or perhaps the computer manufacturer's) to see if there are updated video drivers.  These might solve the problem.  Or not.
