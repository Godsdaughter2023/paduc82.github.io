lance cai msft say:

Hi James,

Creating a preservation policy alone is half the task. To control which content is preserved and for what duration, we need to also create and assign a Preservation rule to the policy, and this is done via the New-RetentionComplicanceRule cmdlet.

Here are the steps:

  1. Connect to the Office 365 Security & Compliance Center using remote PowerShell wih an admin account.
  2. Run the cmdlet below to achieve your requirement:

new-RetentionComplianceRule -Name <the rule name> -Policy "<the policy name you create>" -PreservationDuration Unlimited

At this point, go back and check the Compliance center GUI, we will notice that the error message has now disappeared.

However, the feature is newly published in office 365 environment, I would to provide you with another suggestion about the data retention.

You can take a look on this part - Preservation policy vs. eDiscovery hold in this article: Overview of preservation policies

The e-Discovery hold feature is easier to create and apply to and you can also add any condition you want.

If you are interested in the e-Discovery hold feature in Office 365, here are some articles for your reference:

import a style from another word document WITHOUT opening that - Microsoft Community

bfincher say:

import a style from another word document WITHOUT opening that document

I want to import a style into a Word document without opening the document I am importing the style from.
kristine arp say:

Hi,

 

 

You can directly import the file from the Developer tab in Word. If you don’t see the Developer tab in the Ribbon, you might have to add it to the commands by following these steps:

 

  1. Click File.
  2. Choose Options.
  3. Select Customize Ribbon.
  4. In the Main Tabs section, check the box beside Developer.
  5. Hit OK.

 

Once the Developer tab is already in the Ribbons, follow the steps below to import the template:

 

  1. Click Developer.
  2. On the Templates tab, choose Document Template.
  3. Click Organizer.
  4. Click Close File in the right section of the dialog box.
  5. Click Open File.
  6. Select the file you want to import style from then click Open.
  7. Select the style that you want to import from the right box and click the Copy button.
  8. Hit OK.

 

Let us know if you need additional assistance.
