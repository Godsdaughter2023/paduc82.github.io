zynlex say:

Hello,

The Xbox One Media Remote uses a infrared (IR) transmitter, which is fairly common in television remotes and it's probably the same technology used to control your LED lights which is the problem your running into.

The Xbox One controller users a different technology which is why you don't have that problem with the controller as you do with the media remote.

IR requires a straight line to the console in order to control it, so you essentially need to move the LED lights IR module away from the console so when you attempt to use the media remote, the LED lights don't detect it. I'm not sure if this is a feasible option but that's probably your only option.
