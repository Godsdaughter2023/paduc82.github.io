1) Open the dialog for customizing keyboard shortcuts
- File/Options/Customize Ribbon/Keyboard shortcuts: Customize...

- Move focus to the Press new shortcut key edit field.
- Press Ctrl-F.
- Select All Commands in the Categories listbox.
- Select EditFind in the Commands listbox.
- Click Assign button.

3) Enable the good old F3 as the find next function
(Assuming that you just completed step 2)
- Press F3.
- Select RepeatFind in the Commands listbox.
- Click Assign button.

hansv mvp say:

F3 is the keyboard shortcut for Find Next in Notepad and in the Visual Basic Editor, but not in previous or current versions of Word or Excel.

In Word, F3 is the default keyboard shortcut for inserting AutoText, and in Excel, F3 is the default keyboard shortcut for pasting a defined name into a formula.

The keyboard shortcut for Find Next in Word and Excel is Shift+F4.

How to hyperlink an Excel sheet to a Word document? - Microsoft Community

How to hyperlink an Excel sheet to a Word document?

I need to add a hyperlink in word to go to a specific tab in excel.  How can I do this
jeeped say:

From the Insert tab's Links group, click Hyperlink. Provide a Text to display and(optionally) a Screen Tip. Use the buttons and folder navigation to locate your Excel workbook. Add the pound sign (#) and the worksheet name, then an exclamation point (!) and a cell to start as in,

C:\Users\<username>\Documents\Book1.xlsx#Sheet3!A1

That should place a CTRL+<click> hyperlink into your word document.
