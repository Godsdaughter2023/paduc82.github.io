Try changing how markup is displayed.  Go to the Review tab, and change to All Markup.  Sometimes Word does not know how to manage hidden text, special characters, format codes, etc.  All Markup helps because Word just displays everything without applying all the formats.

rohn007 say:

File format XLSX vs XLSM has no impact on image file formats supported.  I was able to save a TIFF in an XLSX, in PC not Mac (hopefully PC v MAC is not significant in this situation, another Mac user will have to confirm either way).

The x file format was introduced in 2007.  Prior to that all default Office file formats supported macros. As well the file formats were "binary", not readable in any program that had not explicitly coded to handle the file format.  In 2007 Office changed the default file format to the new "X" format files. The difference between "X" and "M" files is macro support, hint, the "M" files support macros. This was because most files do not use macros and there was a problem with malware macros infecting files.  Co-incidentally, the new x-file formats are XML based, meaning they are text based and therefore any program that can handle text formats can read most of the content.

The "hard drive full" message is excessively misleading (yes it is a pet peeve of mine) because it is not talking about the entire HD, which can have lots of free space.  The error is happening because part of the open process creates "temp" files, and sometimes those temp files are located in folders that have defined space limits and that limit is being exceeded. 

jim gordon mvp say:

Methinks there is something about the particular TIFF file you have that is problematic. Perhaps if you open it in a graphic editor such as PhotoShop or GIMP, make an inconsequential change to it, then save it with a new file name the problem with this TIFF file might go away.

Adding to what Rohn007 said about file formats:

XLSX - is the default file format for current versions of Excel. The X of xlsX indicates the file is OOXML file format (Office Open XML, which is plain text in XML). This format never has any macros.

XLSM - same as XLSX except that this format allows for including VBA macros within the file. The M of xlsM means macro,

XLST - same as XLSX except it acts as a template. The T of xlsT stands for Template.

XLSB - Binary format instead of text. This format supports macros as does XLSM. The B stands for binary. These files are smaller. They open and save much faster than the other file formats. Useful for large data files.

XLSA - Excel add-in. The A stands for add-in, but you wouldn't be wrong in saying it also stands for application.

All of the above formats support JavaScript add-ins.
