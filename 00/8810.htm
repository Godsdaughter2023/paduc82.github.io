suzanne s. barnhill say:

When you insert a cover page, Word enables Different First Page and sets the numbering to Start at 0, which is why you still get 1 on the first page after the cover page. If you change the numbering to "Continue from previous section" instead, it will start at 1 for the cover page.

 

If you want the original first page to still be 1 but the number of pages to be correct, there are a couple of workarounds. You can use { NUMPAGES -1 }. Or, if the remainder of the document is a single section, you can insert a section break between the cover page and the rest of the document (disabling Different First Page) and use the SectionPages field instead of NumPages.

 

Note that the \* Arabic switches are not necessary and in fact will be ignored by a TOC if the number format is set to something different. Just setting the number format to 1, 2, 3 is sufficient.

charles kenyon say:
Julie,

This is not going to happen using the Insert Cover Page function. That cover page is always going to be page 0 and not count in your total pages in Word 2010-13. In Word 2007 it is 0 and is counted in the total pages.

You can create a custom (building blocks) gallery of cover pages not in the Cover Pages gallery. They will not have this problem but will probably have other problems for you.

The Cover Pages are very specially formatted.
  • They insert a page.
  • That page is at the beginning of the document.
  • They replace any existing cover page.
  • They don't mess up page numbering.

If you create your own gallery you lose all of these features.


You could create a macro that would let your users change the page numbering if you wanted and store that macro, attached to a keyboard shortcut, in your template.

gina whipp say:

Not that I hve ever seen BUT you could make ythe rows in your grid style report grow evenly...

 

In Design View of the report, for each Field you want to grow evenly, place Border on the Tag line in the Properties window, then in the OnPrint event of the Detail section of your Report place everything between the bold lines...

 

Private Sub Detail_Print(Cancel As Integer, PrintCount As Integer)
'Duane Hookom 3.18.2008
   
    Dim intMaxHeight As Integer
    Dim ctl As Control
    'Find highest control in Detail section that has a tag property of "Border"
        For Each ctl In Me.Section(0).Controls
            If ctl.Tag = "Border" Then
                If ctl.Height > intMaxHeight Then
                    intMaxHeight = ctl.Height
                End If
            End If
        Next
       
        'Draw a box around each control in Detail that has a tag property of "Border"
        For Each ctl In Me.Section(0).Controls
            If ctl.Tag = "Border" Then
                Me.Line (ctl.Left, ctl.Top)-Step(ctl.Width, intMaxHeight), vbBlack, B
                'Me.DrawWidth = 6  'play with this number to make line thicker
        End If
        Next
End Sub

Hi, arbensinanaj. When you say personal videos, are you referring to the video recorded by the phone itself? If yes, have you tried Soft and Hard reset? Soft reset will not delete any personal data on the phone.
