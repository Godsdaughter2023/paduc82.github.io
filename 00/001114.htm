sachin shetty say:

Which operating system do you use?

 

Try to open Excel in safe mode using the following steps and verify the status:

1.     Open Run (press Windows key + R)

2.     Type Excel /safe

Note: add a space between Excel and /

3.     Try to print the Sheet or use Print Preview

 

If the issue still persists set a different printer (XPS document writer) as default and verify the result.
