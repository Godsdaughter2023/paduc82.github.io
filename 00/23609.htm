Your formula, as written, will not return the results you want.

I used Sheet1, assuming you don't actually have a sheet named "sheet".

allan lu msft say:

Hi Denise,

 

Yes, by default, a user with valid Microsoft account or Office 365 account can accept the invitations which are specific to another individual.

 

We can change this to make it only the individual we requested can accept it by the following settings. Go to SharePoint admin center > sharing, under Additional settings, choose “External users must accept sharing invitations using the same account that the invitations were sent to”.
