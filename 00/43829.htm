You will not be able to use the Upgraded copy of Windows 10 on another computer
unless the version of Windows 7 was a Full Retail Version. And even then you can't
use it on both the present and new computer. In fact you can't use the original Windows
7 and the upgraded Windows 10 at the same time on the same or different computers.

the license is only for the original computer's hardware.

Windows 10 from the present machine and install a Clean Windows 10 on the new
computer.
