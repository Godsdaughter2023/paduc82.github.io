  2. Under Filter Keys, make sure the toggle is set to Off.

Method 2: Check your keyboard language

  1. Go to Start, and select Settings > Time and language > Region and language.
  2. Make sure your region is selected in the drop-down box.
  3. See what language is listed under Languages.
  4. If you don’t see the language you want, select Add a language and select the language you want to add.
  5. To delete a language you don’t want, select it and select Remove.

Method 3: Check for special keyboard layouts

  1. Go to Start, and select Settings > Time and language > Region and language.
  2. Select your language and select Options.
  3. To add a layout, select Add an input method and choose the layout from the list.
  4. To remove a layout, select it and select Remove.

Method 4: Try CAPS+Fn (to toggle back and forth from the function keys (F1,F2,F3,...) and the special keys.

Method 5: Restart your Surface. Press and hold the volume-up button and the power button on the your Surface at the same time for 15 seconds and then release both of them. The screen may flash the Surface logo, you just need to continue holding down the buttons for 15 seconds.

Method 6: Run a system restore.
