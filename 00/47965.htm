lemp say:

AFAIK, there is no limit ... but there are certain practicalities that you need to consider.

First of all, you need to distinguish "partitions" or "volumes" from physical devices.  Most people refer to partitions as "drives," but a physical hard drive can have many separate partitions.

In Windows, partitions, are typically identified by letters followed by a colon.  A: and B: are reserved for a floppy drive (which almost no modern computer has anymore).  The remaining letters, C: through Z: may be used for partitions on hard drives (or other devices).  Typically, C: is the system partition, i.e., the place where Windows is located.

An optical drive (CD or DVD) counts as a partition and takes up a drive letter.  So does each slot on a multi-card reader, whether or not there is a card in the slot.

Each USB flash drive counts as a partition -- or more than one, if you have created multiple partitions on the device.  You can "partition" a USB flash drive just the same way you can partition a physical hard drive.  Unlike the card reader, however, the USB flash drives only "count" when they are connected.

So, let's assume that you have a typical system with a single hard drive that has only one partition and that partition of course includes Windows.  You also have one optical drive (e.g., a DVD drive) and a 4-slot card reader.  You don't have any floppy drives.  Here's what happens:

  • Drive letters A: and B: are reserved even though there are no physical devices.  Actually, you can manually use these letters for other partitions, but most people don't,
  • Drive letter C: is assigned to your system partition on your hard drive
  • Drive letter D: is assigned to your DVD drive
  • Drive letters E:, F:, G:, and H: are assigned to your card reader

Because the English alphabet has 26 letters, you then have 26 - 8 or 16 letters left.  These can be assigned multiple physical hard drives connected to the computer through the motherboard; multiple partitions either on the main hard drive you already have or on external drives; or to drives connected via USB.  These latter might be either flash drives or "real" drives in external cases.

For example, this is what "My Computer" looks like on a system that has 3 physical drives installed inside the computer, 1 physical drive connected by USB, 1 USB flash drive; and 4-slot card reader.  One of the internal disks has 2 partitions, as does the external drive, and there are no cards in the reader.  The system happens to be Windows 7, but XP would be the same.

Disk 0 -- Partition C:

Disk 1 -- Partitions D: and E:

Disk 2 -- Partition H:

Disk 3 (a physical disk in an external enclosure connected via USB) -- Partitions F: and G:

Disks 4 - 7 (the card reader) -- Partitions I:, J:, K:, and L:

Disk 8 (the USB flash drive) -- Partition M:

DVD drive -- Partition Y:

You are not, however, limited by the 26 letters.  A partition can be "mounted" and referenced as a directory as long as you are using the NTFS file system.  This essentially gives you the ability to have an unlimited number of partitions (at least I don't know of a limit).  See this article for more information --> http://ask-leo.com/26_drives_is_there_a_way_around_the_26_drive_limit_in_windows.html

just WHAT is Drive Q: ?  I can't access it.

Starting with Office 2010, Microsoft introduced click to run, "a streaming and virtualization technology that is used to install Office products."  If you elect to use click to run rather than the traditional "MSI" installation, you can start using an Office product within about 90 seconds, rather than waiting until the installation is complete.  Microsoft creates and uses the Q: drive to accomplish this virtualization.  In fact, if you had already assigned Q: to a physical partition, you would get an error when you attempted to install Office using click to run technology. 

What is the "Q:" drive?

Office 2010 Click-to-Run suite such as Office Starter 2010, Office Starter To-Go, Office Home and Student 2010, and Office Home and Business 2010 are based on some application virtualization technology. The Q: drive is the virtual file system drive where virtualized applications are located in the file system namespace. This Q: drive is not a typical drive. The Q: drive has no space that the user can access directly and is inaccessible from Windows Explorer or My Computer.

Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

However, this is not same for retail licenses i.e. you purchased the license through Microsoft store. That does carry transfer rights. If that is the case, please see the below steps:

Run the activation troubleshooter on the new device using the below steps:
Go to Settings>Update and security>Activation and run the troubleshooter. and select I recently changed hardware on this device.

(Please note that Activation troubleshooter requires you to have digital license linked to Microsoft Account.)

If the troubleshooter hasn't helped, you can get in touch of an agent using the link below who can help you to get your copy activated:
https://partner.support.services.microsoft.com/...

alan williamson say: When your laptop connects it's using DHCP to automatically get an IP Address from your router. You could try configuring a static IP address on the system to see if that works (so you know it's a DHCP problem assigning you an IP, not a wireless connection issue for example).  

Also you could open an elevate command prompt (right click on cmd, Run As Administrator) and run the following:

  1. ipconfig /release
  2. ipconfig /renew
  3. ipconfig /all
The first 2 would be to fix the lack of dynamic IP address (DHCP), the 3rd is for you to see that you get one (it you get an address starting with 169.254 then you probably didn't get an address).
