gandallas say:

TO LET YOU KNOW. THIS ARE THE CORRECT PROCEDURE. IT WORKS!!!

1. At an elevated command prompt, type the following command, and then press ENTER:

takeown /f c:\windows\system32\rasadhlp.dll

2. Type the following command, and then press ENTER to grant administrators full access to the file:

icacls c:\windows\system32\rasadhlp.dll /grant administrators:F

3. Type the following command to replace the file with a known good copy of the file (from a retail build of windows, download and save it in temp folder):

copy c:\temp\rasadhlp.dll c:\windows\system32\rasadhlp.dll


Type exactly and it will work
