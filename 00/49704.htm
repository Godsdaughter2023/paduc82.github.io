a. user say:

Ah yes ... the (in)famous Logitech Download Assistant. Users of Logitech hardware know it well, including me.

This is one of two totally unnecessary software utilities that Logitech likes to install, the other being their product registration utility, which will continue to start with Windows even if you've already registered your hardware with Logitech.

Logitech's software almost never gets updated, but the download assistant insists on installing itself and fights our attempts to prevent it from starting with Windows.

If you have Logitech's hardware, the only thing you need to install is the hardware driver, if one is needed in the first place. Other Logitech software, like Setpoint or LWS (Logitech Webcam Software), is really unnecessary and will install one or both of the above utilities.

Here's the safe way to get rid of both utilities - permanently:

1- Uninstall all your Logitech software.

2- If your Logitech hardware needs a driver - a webcam needs one - be sure to install only the driver, and nothing else.
