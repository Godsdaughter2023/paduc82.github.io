There're 2 locations where Windows NT 6.x stores Start menu content:
%appdata%\Microsoft\Windows\Start Menu\Programs\

%programdata%\Microsoft\Windows\Start Menu\Programs

The trick is that modern apps aren't shortcuts (links) as we've been used to them so they aren't stored there. However, it looks like you can't rearrange any shortcuts and folders from within the Start menu itself right now. This is one of the quirks of running Technical Preview.
