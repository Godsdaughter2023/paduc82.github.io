Hi

There is controversy whether changing the setting of the "LanmanServer\Parameters\Size" cache works on Win 7.

As far as my experience goes, it seems to help on a Win 7 computer that is used as Hi traffic server.

Try to do this.

Set the Dword value of the following registry key to 1

HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management\LargeSystemCache

Set the Dword value of following registry key to 3

HKLM\SYSTEM\CurrentControlSet\Services\LanmanServer\Parameters\Size

If it does not do anything, you can leave it as is, or reverse the settings.
