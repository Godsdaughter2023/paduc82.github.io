a. user say:

Hi Ankit,

Yes, you can install a printer using TCP/IP port through advance printer setup under Control Panel.

Follow the steps to do so:

  • Press Windows+X keys and select Control Panel.
  • Click on Devices and Printer. Choose Add a printer.
  • Click on Next. Select Add a Printer using a TCP/IP address or hostname. Click on Next.
  • Select Custom and then on Settings.
  • Type the IP address of your Print Server in Hostname or IP address. The print server’s IP should in same subnet with your router.
  • Select the correct driver for your printer. If you can’t find the model of your printer, please connect your printer to PC with USB cable and install the driver first. Click Next.
  • Click Next and then on Finish.

For further reference, you can refer to the link:

http://windows.microsoft.com/en-ca/windows/install-printer#install-printer=windows-8

Please let u know if the issue still persists.

5. Turn on the router, wait for two minutes.

6. Boot the computer.

Get-AppXPackage *WindowsStore* -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

Built-in : Windows Key+X > Click Settings > Click Update & security > Click Troubleshoot > Click Windows Update > Click Run the Troubleshooter
