Here is what you need to do to clean install Windows 10 on a computer system.

9.Reboot the PC.

In order to ensure an uniform distribution in each case, we must apply a procedure that has a slightly different output than your example.

Below is the code (for interested followers), here is my sample file:
https://www.dropbox.com/s/n24pt8gwgu6huxg/b4f0e...

Press ALT-F8 and execute the macro.

Andreas. 

Option Explicit
Sub Test()
  Dim Table As ListObject
  Dim Data, Item
  Dim Dict As Object 'Scripting.Dictionary
  Dim Key As String
  Dim i As Long
  
  'Refer to the right table with the codes to distribute
  Set Table = ActiveSheet.ListObjects("Table2")
  
  'Read in all data
  Data = Table.DataBodyRange.Value
  
  'Create a dictionary for faster access
  Set Dict = CreateObject("Scripting.Dictionary")
  Dict.CompareMode = vbTextCompare
  For i = 1 To UBound(Data)
    'The key is the Branch Code from the 2nd column
    Key = Data(i, 2)
    If Not Dict.Exists(Key) Then
      'The 1st slot is a count, the others are the "EMP No"
      Dict.Add Key, Array(0, Data(i, 1))
    Else
      'Add "EMP No" for existing keys
      Item = Dict.Item(Key)

john korchok say:

I still have Word 2004 installed, so I was able to do some tests.

You don't have to attach the old template to a document. Just use Word's File>Open command to navigate to the old template and open it. The Word screen should read Normal [Compatibility Mode] in the blue bar at the top. If it says Document1, you've created a new document instead.

I was able to copy over AutoText, Styles, Toolbars and Macro Projects with no problem. Toolbars will appear in a new Add-ins tab on the Ribbon.

Keyboard shortcuts can't be transferred in the same way. But to print out a list while you have your old template open:

  1. Choose File>Print.
  2. Change the Copies & Pages dropdown to Microsoft Word.
  3. Change the Print what dropdown to Key assignments, then click on the Print button.

Your Normal.dotm file is the host to all this information (including key assignments). Once you get everything moved, then you can copy the new Normal.dotm file to your laptop and everything will get moved with it. When you get to that stage, here are the steps:

  1. Close Word on both computers.
  2. Hold down the Option/Alt key on your desktop, then click on the Go menu and choose Library. Your user Library folder opens.
  3. Navigate to Library/Group Containers/UBF8346G9.Office/User Content/Templates and copy Normal.dotm.
  4. Move Normal.dotm to your laptop.
  5. Repeat steps 2 and 3, but copy Normal.dotm over the Normal.dotm on your laptop.
  6. Start Word on your laptop and check that all content is there.
