a. user say:

Hello,

 

Thank you for posting your query in Microsoft Community.

 

We do understand your concern and we will be happy to help you regarding this.

 

I would like to inform you that the position in the taskbar is set before you open any new desktop. If you change the position of anything in the taskbar, it will come back to the same place where you initially had the program.

For example, if the Process Explorer tab is on the first position before creating any new desktop. Then it will remain on the first position. If you go to the new desktop and if you come back to the first desktop, Process Explorer tab will stay on first position. And if you drag the Process Explorer tab to the third position after opening a new desktop, the Process Explorer tab will reset back to first position after toggling between the desktops.

 

I hope this helps you. If you have any further queries, feel free to contact us. We will be happy to help you.
