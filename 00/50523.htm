lemp say:

"Broken Registry Items" is terminology invented by companies that want to sell you something.

  • Is it possible for an entry in the registry to become corrupt?  Yes.  Is this common?  No.

  • How does a registry entry become corrupt?
    • When a poorly or maliciously written program makes a change in the registry.
    • When a person with insufficient knowledge makes manual changes to the registry.
    • When the computer is shut down improperly while registry writing is underway.  See https://support.microsoft.com/en-us/help/822705/registry-troubleshooting-steps-for-advanced-users:  A power failure or some other unexpected shutdown event may cause a corrupted registry hive. .... In this case, some process may have been modifying part of the registry hive, and the computer lost power before that change could be completed. This leaves the registry hive in an inconsistent state.
    • When there is a problem with the storage media where the registry is located (e.g., your hard disk is failing)
  • Is it possible for a malformed Windows Update to cause registry corruption?  Yes.  Is this common?  No.
