flag duplicate records - Microsoft Community

javablood say:

flag duplicate records

I have a table with the following fields: ID, Date, Time, Gallons, Comments.  In this table there are 6 separate IDs, e.g., TR-2, TR-4, etc.  Data are automatically collected once a day and then downloaded quarterly for me to update my table by a specific date range.  However, I just noticed that through the years the download has included duplicate dates with different times because evidently the computer program (propreitary and not ours) recorded multiple data on the same dates with no siginificant pattern.  What I need to do is flag the duplicate records (ID & Date) for each ID so that I may quickly evaluate the records that should be deleted.  I searched the answers but did not find something similar enough to my needs to work.  Any help is appreciated.  Thanks.

 

hansv mvp say:

I'd do this in two steps, otherwise it becomes rather complicated.

First, create a query

 

SELECT MyTable.[ID], MyTable.[Sample_Date], MyTable.[Sample_Time], MyTable.[Gallons], MyTable.[Comments]
FROM MyTable
WHERE (((MyTable.[ID]) In (SELECT [ID] FROM [MyTable] As Tmp WHERE Tmp.[Sample_Date] = [MyTable].[Sample_Date] GROUP BY [ID], [Sample_Date] HAVING Count(*)>1)))
ORDER BY MyTable.[ID], MyTable.[Sample_Date];

 

and save it as qryDuplicates.

Next, create a new query based on qryDuplicates:

 

SELECT qryDup.ID, qryDup.Sample_Date, qryDup.Sample_Time, qryDup.Gallons, qryDup.Comments
FROM qryDup
WHERE (((qryDup.Sample_Time)>(SELECT Min([Sample_Time]) FROM qryDup AS T WHERE T.ID=qryDup.ID AND T.Sample_Date=qryDup.[Sample_Date])));

 

This returns all duplicate records except for the first one for an ID and date.

You can even turn this into a Delete query:

 

DELETE qryDup.ID, qryDup.Sample_Date, qryDup.Sample_Time, qryDup.Gallons, qryDup.Comments
FROM qryDup
WHERE (((qryDup.Sample_Time)>(SELECT Min([Sample_Time]) FROM qryDup AS T WHERE T.ID=qryDup.ID AND T.Sample_Date=qryDup.[Sample_Date])));
