The steps below might help you.

Step 2:

 

Delete the “Microsoft Silverlight” directory from under Program Files.

 

ü From an administrator command shell on an x86 you can do this by running:
rmdir /s /q "%ProgramFiles%\Microsoft Silverlight

 

ü If you are on a 64-bit version of Windows, substitute ProgramFiles(x86):
rmdir /s /q "%ProgramFiles(x86)%\Microsoft Silverlight

 

Here’s how you do it.

 

a.    Go to Start and type cmd.

b.    Right-Click on cmd and select “Run as administrator”.

c.    Type the above command whichever is applicable for your version and then press Enter.

 

Step 3:

 

As with many uninstallations, this may not delete Silverlight entirely. In particular, it will probably leave behind some registry entries. You can remove them from the registry by following these methods.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Back up the registry

http://windows.microsoft.com/en-us/windows7/Back-up-the-registry

b.    Click on regedit. This opens registry editor.

c.    Navigate to [HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Silverlight]

d.    Delete Silverlight key.

e.    Close registry editor

 

Step 4:

 

Install Silverlight by downloading from:  http://www.microsoft.com/getsilverlight/Get-Started/Install/Default.aspx

remove letter="Z"
