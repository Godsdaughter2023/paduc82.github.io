palcouk say:

You can try running scanpst.exe, the inbox repair tool against this data file, but generally that error indicates that your method of creating or copying the data file was at fault

eg if you copied an outlook data file, pst, that was connected to OL, whilst outlook was open that would have corrupted the data file.

If you used a cd to copy the data file, in the properties for that data file, ensure any read only flag is off.
