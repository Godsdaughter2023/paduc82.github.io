hairyfool say:

There are limited options with PS3 network management so have to work on the PC


Switch on the PS3 and make sure it is connected to the internet with the laptop turned off.


Turn on the laptop and allow it to try and connect. If it comes up with the same error don't worry.


Open a command prompt window at an elevated level and type in the following commands with enter after each line.


ipconfig /release


ipconfig /renew


Windows stores the last network configuration as its preferred setting and will request that same value the next time it joins the network. If the PS3 fails to acknowledge that it is already using that address the router as the DHCP server will issue it again to the laptop. The release command will make the computer "forget" its preferred setting and because the router has only just issued the address to the PS3 should offer a different one to the laptop on the renew command.


If the problem repeats itself try logging into the router and see if you can assign a specific address to the PS3. All devices initiate a connection using its MAC address in order to request the IP address. With many routers you can tell the system to always give one specific address to the PS3 MAC address. In order to avoid conflict the router then flags that address to never be assigned to anything else even when the PS3 is not connected.



This same trick can be used to "fix" the address of printers and stop connectivity problems.
