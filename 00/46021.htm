How To Permanently Turn off Windows Defender in Windows 10?

How can I permanently turn off windows defender in windows 10. I am running F-Secure and Windows 10 tells me that I have no antivirus protection!

profrowe say:

Got it... 

  1. Open Local Group Policy Editor (start -> Run -> gpedit.msc)
  2. Go to Computer Configuration, Administrative Templates, Windows Components, Windows Defender
  3. Find the setting called "Turn off Windows Defender"
  4. Enable it (double click, Enabled, OK).

Job done, defender is gone.  (Service that Admin didn't have rights to stop in services.msc is now stopped)
