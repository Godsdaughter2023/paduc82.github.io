bossdweeb say:

Your old Sidewinder game pad uses DINPUT, which is not compatible with  XINPUT, so in the absence of

profiling software that is compatible with Win10  it is e-waste.

MS introduced  XINPUT  with the XBOX controller. The introduction of XINPUT coincided with the advent

of "Games for Windows" and MS apparently doing a 'deal' with game developers to make the XBOX controller

the default setup in their games.


"Modern" games mostly default to the XBOX (or other controllers that have  XINPUT compatibility).

In theory this is supposed to make for a 'universal' controller setup, but in practice it causes issues,

particularly with older games which use DINPUT  which is not available on the XBOX controllers.

It's a typical one size fits all, and don't  worry about the collateral damage, or mountains of e-waste

that it contributes to, approach that MS tends to promote.

Logitech, Razor, and a few other mfg's make controllers that have a switch that allows you to change between

DINPUT and XINPUT. These controllers use a profiling software that allows the user to program the buttons

to the keyboard keys that the games uses when playing using a keyboard and mouse, making these controllers

better all-rounders than the XBOX, or XINPUT limited controllers.

The Logitech F310 is one example of a good value controller that has both XINPUT and DINPUT capability.
