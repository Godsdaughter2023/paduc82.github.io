lemp say:
Your symptoms are a bit peculiar.  A paper jam in the printer shouldn't have anything to do with software -- it's just a mechanical problem.

The information you forgot to supply was the actual error message(s) that you received.

Msi (really msiexec.exe) is the Windows Installer.  It uses files with the file extension msi to install applications (e.g., TrayApp.msi, ScannerCopy.msi).

Sometimes, the installer itself becomes corrupted, and that's the situation where you may get "Windows installer is not accessible.  Installation can't continue because the Windows Installer could be accessed.  Error Code 1601".

Sometimes, the installation package is missing or corrupted, and that's when you may get something like "The feature you are trying to use is on a network resource that is unavailable. Click OK to try again or enter an alternate path to a folder containing the installation package 'TrayApp.msi' in the box below."

If the installer is stuck attempting to complete an install where it can't find the necessary msi file, you may not be able to install anything else.  That's what seems to have happened when you tried to install the Adobe Reader update, although it's also possible that the installer itself has become damaged.

If you saw the eHow instructions, you've probably already seen this HP page that explains some of the above --> http://h10025.www1.hp.com/ewfrf/wc/document?cc=us&lc=en&docname=c00811375

The applications that appear to be having a problem getting installed are related to non-printing features of your HP All-in-One.  Are these features available from Start > All Programs (I don't have an HP AIO, so I can't give you the exact names, but there probably is an HP or Hewlett Packard entry in your All Programs list)?  Can you scan or copy?

If all of the HP software is installed and working (I suspect that it is not), you may still have to repair the installer itself.  Use the Command Prompt commands I suggest.  They won't hurt and they may help.

If you use those commands and you still have problems, here are two further steps:

Download and install the latest Windows Installer from here --> http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=8483
Read the Instructions on that page and be sure that you download the correct version for your computer.  That page has versions for a number of different Windows operating systems.

If you continue to have problems related to missing HP files, uninstall all of the HP software using Control Panel > Add or Remove Programs.  Then use either the Microsoft Windows Installer Cleanup Utility mentioned mentioned on the HP page linked above or free Revo Uninstaller to remove any remaining remnants of the HP software installations.

If you follow the link in the HP page, you will find that the Windows Uninstaller Cleanup Utility has been removed from the Microsoft download servers because it apparently sometimes removed more than it was supposed to (I believe this was related to Office uninstalls).  You can still obtain the Windows Installer Cleanup Utility here --> http://majorgeeks.com/Windows_Installer_CleanUp_Utility_d4459.html  

Revo Uninstaller is an alternative that shouldn't have any issues. 

Before you use these cleaners, you should create a System Restore point first (or backup the Registry with ERUNT) just in case something other than the HP software gets removed or some other unforeseen issue arises.  See http://windowsxp.mvps.org/registry.htm  A full disk image would be even safer if you have that capability but is probably overkill.


Once the HP software has been completely uninstalled and you reboot, you should have no further problems installing unrelated software like the Adobe Reader update.  You may have to redownload that update.

Finally, reinstall the HP software.  Either use the CD that came with your AIO or go here --> http://h10025.www1.hp.com/ewfrf/wc/softwareCategory?os=228&lc=en&cc=us&dlc=en&sw_lang=&product=428800#N299 Note that there are several patches and updates available.  Most of these are dated before the "full feature" download.  I expect that you don't need the earlier-dated updates.
