-Scan -ScanType 3 -File

ossiemac say:

You have a syntax error in the line. You have left out .Range in front of ("A6:G6").

Sheets("Sheet1").Range("A6:G6").AutoFilter Field:=1, Criteria1:=ary_1, Operator:=xlFilterValues

However, once the filters have been applied then you can use a generic range reference like the following.

Sheets("Sheet1").AutoFilter.Range.AutoFilter Field:=1, Criteria1:=ary_1, Operator:=xlFilterValues

Also you do not need to test if AutoFilterMode is True before setting it to False.

You can replace the following line

If Worksheets("Sheet1").AutoFilterMode = True Then Worksheets("Sheet1").AutoFilterMode = False

With the following line

Worksheets("Sheet1").AutoFilterMode = False
