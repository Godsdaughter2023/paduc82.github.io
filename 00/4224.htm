stephan prudhomme say:

Google Analytics and Publisher 2013

Someone knows how to add Google Analytics tracking code to Microsoft Publisher 2013?
davidf2 say:

If you are talking about a Publisher 2013 web publication, then you can't...at least not in the same way as in the past. 

 

MS has been deprecating the web functionality of Publisher since Pub 2010, and as of Pub 2013 they stripped the ability to insert an html code fragment into a web publication. You must have this tool in order to insert the Google code snippet. If you have access to an older version of Publisher then use it.

 

You do have the option of editing the source code manually after you produce it. Open the index.htm file and insert the Google code snippet. I can not give you the specifics of how to do this and would suggest you read the Google instructions. Here is a Publisher site where the Google Analytics are used and where you can study the source code to see if that helps: http://www.europe-hotels.gr/cyprus/index_paphos.htm

 

Do you really need the Google Analytics? Most web hosts offer pretty good stats. Look at what your web host provides and you may find it is enough.

 

Perhaps you should also rethink how much effort you are going to put into building your site with Publisher give the hobbled web functionality and the lack of a future in Publisher. Expression Web is now free if you are willing to learn basic html and css coding. Serif WebPlus is a good alternative and offers a free version. Most webhosts offer a web building tool for basic sites. There are lots of alternatives out there, and perhaps you should consider switching before you invest a lot more time in using Publisher.
