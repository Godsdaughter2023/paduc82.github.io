dan at it associates say:

The key difference between the Microsoft Office 2007 OEM license and the MLK license is that the OEM license is an agreement between the manufacturer and the end user, while the MLK license is an agreement between Microsoft and the end user.

 

From the Office 2007 SLTs:

 

 

"2. OEM LICENSE TERMS

These license terms are an agreement between you and the device manufacturer that distributes the software with the device, or the software installer that distributes the software with the device."

 

"3. MEDIA-LESS LICENSE KIT TERMS

These license terms are an agreement between Microsoft Corporation (or based on where you live, one of its affiliates) and you. Please read them. They apply to the software that accompanies these license terms, which includes the media on which you received it, if any."

 

 

 

So your point about "...the original OS never had the pre-installation files for MS Office on it in the first place..." is irrelevant, not only because the Operating System is never pertinent to the discussion, but because the MLK terms are not contingent upon the software being preinstalled on the device (the computer).

BlackJack II Connects as Guest Cannot Establish Partnership -- Found a Fix!

I have a new BlackJack II.  I have ActiveSync 4.5 installed on my home computer and I can sync with that one just fine.  It's Windows XP Home and I have Outlook 2000.  ActiveSync 4.5 on my work computer is installed and runs fine, when I connect the phone it connects as a Guest.  It doesn't give me any other options, and never asks me if I want to establish a partnership.  At work, we have Outlook 2003.  I can see the files on the phone and transfer files on and off of it in both places, but at work I cannot establish the partnership.  Any ideas?

 

Thank you in advance for your help.

 

UPDATE:  I found a fix that worked for me:

http://www.pocketpcfaq.com/faqs/activesync/guest.htm#Guest%20Issues

 

There are 9 things to try, and what worked for me was #8 -- copied here for your convenience:

Check and see of the registration key at HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows CE Services and see whether a key named GuestOnly has a value of 1.  If it does then change the value to 0 to allow you to setup a partnership.  An alternative is to use a free application application called ActiveSync Keys which is available at Undocumented ActiveSync Keys.  NOTE: Microsoft does not support end users editing the registry so there is no support for performing this.

 

I only had to edit the key and it worked, I did not use that free application that was mentioned.

 

Good Luck and I hope that this can help someone someday.
