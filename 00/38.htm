I think what you want would be something like this:

Function myUDF(Optional iCell As Range) As String
  If iCell Is Nothing Then
    myUDF = Application.Caller.Address
    Exit Function
  End If
  myUDF = "all ok"
End Function

You can use either Application.Caller or Application.ThisCell to reference the cell containing the UDF. E.g.,

ossiemac say:

If I understand correctly then the following code returns the required parameters.

    Dim dblLeft As Double
    Dim dblTop As Double
    Dim dblWidth As Double
    Dim dblHeight As Double
   
    dblLeft = Sheets("Sheet1").ChartObjects("Chart 1").Left
    dblTop = Sheets("Sheet1").ChartObjects("Chart 1").Top
    dblWidth = Sheets("Sheet1").ChartObjects("Chart 1").Width
    dblHeight = Sheets("Sheet1").ChartObjects("Chart 1").Height

To get the bottom of the chart just add Height to Top and for Right side of chart add Width to Left.

As a little extra, you can use code like the following to position the top left of an existing chart relative to cell positions.

    Sheets("Sheet1").ChartObjects("Chart 1").Left = Range("F16").Left
    Sheets("Sheet1").ChartObjects("Chart 1").Top = Range("F16").Top

And the following will return the top left cell address of an existing chart.

    Dim strTopLeftCell As String
    strTopLeftCell = Sheets("Sheet1").ChartObjects("Chart 1").TopLeftCell.Address

Plus to finish off the following returns the row and column of the top left cell

    Dim lngRow As Long
    Dim lngCol As Long
    lngRow = Sheets("Sheet1").ChartObjects("Chart 1").TopLeftCell.Row
    lngCol = Sheets("Sheet1").ChartObjects("Chart 1").TopLeftCell.Column

You can also use BottomRightCell in lieu of TopLeftCell to return the position of the right and bottom of the chart.
