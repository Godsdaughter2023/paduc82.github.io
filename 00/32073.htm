Force the LPT Port to LPT1 if not there

Start up

Finished the wizard
You should now be able to see the share name on the network
Right clicked on your computer name in device manager and view the print queue
Delete the printer that is NOT copy 1 or the last one you installed from queue
Reboot
Open device Manager
