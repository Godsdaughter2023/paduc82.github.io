rtfreeman say:
Hello Xaessya -

I don't recommend disabling Windows Updates as it leaves your computer vulnerable.

However, you can disable the Windows Update service which should stop the pop-ups.

To do so:

1. Press "Ctrl + R" to open the Run command.
2. Type in "services.msc"
3. Scroll down until you find "Windows Update"
4. Right click "Windows Update" and select "Properties"
5. Click "Stop" to stop the service from running.
6. Under the Startup Type, select the dropdown box "Disabled" to prevent it from starting again.

When you're ready to run updates, which I encourage, do the same thing but under Startup Type, select "Automatic."

I hope that helps!

yolvi ell say:

Hi,

A computer name can be easily changed through the Settings. Before doing so, keep these things in mind:

  1. The name may contain the numbers 0 to 9, uppercase and lowercase letters from A to Z, and the hyphen (-) character.
  2. The name cannot consist entirely of numbers.
  3. It can't have spaces.
  4. Special characters ( ` ~ @ # $ % ^ & ( ) = + [ ] { } | ; : , ‘ “ . < > / ?) aren't allowed.

Once you've decided on the PC name that you want, make sure you're signed in as an administrator before following the steps below:

  1. Click Start.
  2. Type Settings and press Enter.
  3. Click About.
  4. Under PC name, click Rename PC.
  5. Click Next.
  6. Choose Restart now if you're ready to restart your computer or Restart later.

For the user folder name, a five-character abbreviation is used by default if you use your Microsoft account when you first sign in. If you want a different user folder name, you can override this by creating a new local user account with the user folder name that you want. The steps for creating a new user account can be found here. The steps for changing the account type to an administrator account can be found there as well.

Once you've set up your new local user account, remove the Microsoft account association of your old user account by following these steps:

  1. Click Start.
  2. Type Settings and press Enter.
  3. Click Accounts.
  4. Click Your info on the left side.
  5. Choose Sign in with a local account instead.
  6. Enter your password and click Next.
  7. Type a user name, password, and password hint, then click Next.
  8. Click Sign out and finish.

Lastly, you'll need to sign in to your new user account and associate your Microsoft account with it. Here are the steps to do it:

  1. Follow steps 1 to 4 above.
  2. Choose Sign in with a Microsoft Account instead.
  3. Enter your email address and click Next.
  4. Enter your Microsoft account password and click Sign in.
  5. Enter the password for your local user account and click Next.

After following these steps, check if the user folder name is the name that you want and let us know if you need more assistance.

Windows Installer and Substituted Drives

Users cannot install software packaged in the Windows Installer format to a drive that is substituted for another drive.

  • Try installing the package into another non-substituted drive.
  • Also, try un-substituting the drive being installed to and try again with the original drive letter. NOTE: This may cause some software to stop working.
Windows Installer and the SYSTEM Account

The Windows Installer software uses the SYSTEM account for installation purposes. Because of this, it is necessary for the SYSTEM account to have full control of any folder being installed to. Ensure that the SYSTEM account has the appropriate privileges as follows:

  • Double-click “My Computer.”
  • Right click the drive that the software should be installed to and click “Properties.”
  • Select the “Security” tab.
  • For Windows 2000, verify that the “Name” box has the “SYSTEM” user account in it.
  • For Windows XP, verify that the “Group or User Names” box has the “SYSTEM” user account in it.
    If the “SYSTEM” user account does not appear, add it.
  • Click the “Add” button.
  • When the “Select Users or Group” dialog appears, Click “SYSTEM” in the “Name” field.
  • Click “Add.”
  • Click “OK.”
  • In the “Permissions” section, click the “Full Control” checkbox under “Allow” then click “Advanced.” A dialog box will appear.
  • If using Windows XP, select the “Replace permission entries on all child objects with entries shown here that apply to child objects” checkbox next to the “SYSTEM” user account. Click “OK.”
  • For other Windows systems, select the “Reset permissions on all child objects and enable propagation of inheritable permissions” checkbox next to the “SYSTEM” user account. Click “OK.”
  • If a security dialog box pops up, be sure to click the “Yes” button.
