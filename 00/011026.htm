jvinicius say: Friends,

After several hours looking for a solution for this error, I just found the cause of it and an easy way to solve it.

According to the error message, looks like the system is looking for an application called EXCEL.EXE  and didn't find it or found one that is not according to what it expected (corrupted)

I check the file EXCEL.EXE in (Windows 7) C:\Program Files (x86)\Microsoft Office\Office12 and I found it with the size of 0 kb. Of course it was corrupted. What did I do? I went in my other computer(PC)  and
  • copied the file that was there  (17904 KB) and 
  • sent the file to my own email
  • changed the name of the old file from EXCEL.EXE to Excel-OLD.EXE
  • pasted the file from the PC(received from email)  into the folder  C:\Program Files (x86)\Microsoft Office\Office12 with the original name (EXCEL.EXE).
And it worked. Easy like that. (if you don't have the a second computer to copy from, ask a friend. Pay attention if the version of Windows Offices are the same)
