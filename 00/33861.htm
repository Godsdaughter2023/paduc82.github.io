ronnie vernon say:

Hi

 

This may vary slightly according to the system, but normally the 'memory recognized' amounts to the total available memory for the system. A good way that you can track the usage is to use the Resource Monitor.

 

Go to Start and type  resmon  in the Search bar and press Enter. This will start the Resource Monitor.

Click the Memory Tab.

In the lower section of the window, you will see a graphic display of the total available memory and how the physical memory is being used.

Hover the cursor over any of the values and a pop up will show a definition of that value.

The Hardware Reserved is the value that the BIOS allocates for installed hardware, such as the Display Adapter.

 

One thing to consider is that we are still in the early stages of the transition to a pure 64bit computer. Most of the available software, even the 64bit versions are still unable to use the huge amount of memory that can be installed.
