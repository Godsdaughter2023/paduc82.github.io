In the meantime, I suggest you to follow the below methods and check if they help.

Method 1: Disable narrator from Startup items and registry

I'd suspect that narrator is present at Startup items either in the Start menu's Startup folder, or your registry. 

Step 1:

Disable narrator from Startup.

a) Type MSCONFIG into your Start menu and press enter to start the Microsoft Configuration Utility.

b) On the Startup tab, search for Narrator.exe.  If it exists, uncheck it to disable it from startup.

 If issue persists, refer step 2.

Step 2:

Change the values in registry.

a) Open regedit.exe and navigate to: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options

b) Create a folder (key) called Narrator.exe

c) Inside the key, create a new String value called Debugger, and set its value to %1

d) Close registry editor 

From now on, every time Windows will attempt to execute Narrator.exe, nothing will happen. If you need Narrator.exe back, just remove the Narrator.exe registry key.
