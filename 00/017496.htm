In your case it's even more problematic since you are linking summary lines in a linked structure. A few years back someone sent me their master/subproject structure to troubleshoot. In Project linked structures (e.g. master/subproject, resource pool/sharer) may seem like a nice convenient way to set up their organizational structure but unfortunately it is a very tenuous file structure and subject to corruption. Try this ARRAY formula, see below for how to enter it and change the range to suit.


=TEXTJOIN(",",TRUE,IF(MATCH(A1:A10,A1:A10,0)=ROW(A1:A10)-ROW(A1)+1,A1:A10,""))

This is an array formula which must be entered by pressing CTRL+Shift+Enter

around the formula {}. You can't type these yourself. If you edit the formula

you must enter it again with CTRL+Shift+Enter.

        Also see the IMPORTANT NOTE about not using more than one anti-virus program in this topic:
        Choosing an Anti-Virus Program

        IMO MSE is just as good as any other major anti-virus solution including Norton (Symantec) but leaves a smaller footprint...meaning it uses less system resources which can affect performance.

        No single product is 100% foolproof and can prevent, detect and remove all threats at any given time. The security community is in a constant state of change as new infections appear and it takes time for them to be reported, samples collected, analyzed, and tested by anti-vendors before they can add a new threat to database definitions before they can add a new threat to database definitions. Malware writers have the advantage since no matter how hard security vendors attempt to stay on top of new threats, there is always a short time-frame in which a new malicious file goes undetected and can infect a computer without detection. Just because one anti-virus or anti-malware scanner detected threats that another missed, does not mean its more effective. Every vendor's virus lab and program scanning engine is different. Each has its own strengths and weaknesses and they often use a mix of technologies to detect and remove malware.

        Security vendors use different scanning engines and different detection methods which can account for discrepancies in scanning outcomes. Depending on how often the anti-virus or anti-malware database is updated can also account for differences in threat detections.

        Further, each vendor has its own definition naming standards of what constitutes malware and scanning your computer using different criteria will yield different results. The fact that each program has its own definition files means that some malware may be picked up by one that could be missed by another.
