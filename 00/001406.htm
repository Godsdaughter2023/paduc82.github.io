    How do I undelete my history? - Microsoft Community

    cb_02 say:

    How do I undelete my history?

    I accidentally deleted my history and I need to get that back to resolve an issue.  Any help?
    kosh vorlon say:

    Hi CB,

     

    Mike's right. It cannot be undeleted. But it may still be possible to get to the data (though I don't know if deleting the history also deletes this log and I don't care to test in case it does as I want to keep my log). The log is in C:\ProgramData\Microsoft\Microsoft Antimalware\Support\MPLog-<numbers>.log. Open it with Notepad. Some permission changes may be needed to access these folders and files along the way - so be logged in as an Administrator when doing it.  If it's empty, then it got deleted in the process (or otherwise deleted and I've no idea how to recover it).


    This is NOT an easy log to read and is filled with extra data you'll consider irrelevant. It also goes back some time (mine goes back to the 9/11 upgrade) so the best thing to do is scroll to the earliest date you want to start looking. Just look and you'll notice the dates - they're not conveniently sorted or listed but easy enough to find. Mine show earliest first and most recent last. Since the History file only shows 30 days anyway, if you were hoping to use that, then that's as far back as you need to go. If you assumed it somehow kept more or want more, then go to where you want to start or to wherever the log actually starts (it'll be the date in the first entry at the top of the file).

     

    Obviously, we're looking for threats. There's no easy way to find them. In Notepad, click edit and then click Find. I used the term Threat. Search down if you're starting at the earliest date or go to the very bottom of the file and select search up and go until you get to a date where you are willing to call it quits. There were many false leads (mostly one or two per day - so not too bad and I wanted to be sure to use a term that would get them all), but eventually I found what I was seeking.
