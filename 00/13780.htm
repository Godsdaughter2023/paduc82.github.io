stephanb1987 say:

I have the same problem, though I found a way to get into the Windows Defender interface and perform a scan.

Open your Task Manager, click on "More Details" (if only a summarized/simplified list is displayed) and look under "Background processes" for "Windows Defender notification icon", right-click it and select "Open File location" this should open the Windows Explorer opened at the Windows Defender folder. The file "MSASCuiL.exe" is highlighted, but look at the file above this, which is named "MSASCui.exe" right-click this file and select "Run as administrator" and after waiting the odd 10 seconds the Windows Defender window should open and you should be able to use it as normal.

This is however only a workaround and does NOT solve the actual problem of Windows Defender not opening when clicking the notification icon on the task bar, or the "Open Windows Security Center" option in the Windows 10 Settings interface (under "Update & Security" --> "Windows Defender").
