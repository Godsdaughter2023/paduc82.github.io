Sweeping and blocking the junk emails that you received is not enough to stop them from coming through on your inbox. By setting your junk mail filter to Exclusive, only messages from your contacts and Safe senders lists will be allowed to go to your Inbox.

 Please refer to these steps:

 

1. Sign in to your account and go to your Inbox page.

2. If you're using the old interface, click on Options located at the right upper part of your screen and then select More options. But if you're using Outlook.com interface, click on gear icon next to your username and then select More Mail settings.

3. Under the Preventing junk email, click on Filters and Reporting.

4. Under Choose a junk email filter, make sure that the option Exclusive is selected.

5. Click on Save.

 

As for creating a rule, it depends on how you set it up. You can create a rule that if the subject of the message contains the word "addict" it will automatically delete the email. This rule will filter all incoming emails whose subject contains the word "addict" and will delete it automatically which means you won't be receiving any emails containing that word in the subject line.

To know how to do this, please follow the steps provided below.

1. Sign in to your account.
2. Click Options beside the question mark icon on the right most part of the screen and on the dropdown list, select More options.
3. Under Customizing Hotmail, click on Rules for sorting new messages.
4. Click New button.
5. Under Step 1: Which messages do you want this rule to apply to?, on the first box with dropdown list, select Subject and on the second box
with dropdown list, select contains word.
6. On the third box, type addict.
7. Under Step 2: What action do you want to apply?, select Delete these messages.
8. Click on Save button.

1. Sign in to your account and go to your Inbox page.

If you’ll save a file from Excel Online, a desktop version of it is needed to complete the process. For a reason that the Office Excel app has extensive features that users can utilize. To save an Excel file to a computer, follow these steps:

  1. Click File> Save As> Download a Copy.
  2. Click Save.

Once you’ve downloaded the file, here are the next instructions that you’ll follow to save it as a delimited file:

  1. Go to File > Save As.
  2. The Save As dialog box appears.
  3. In the Save as type box, choose the text file format for the worksheet.
    - Click either Text (Tab delimited) or CSV (Comma delimited).
    - Note: The different formats support different feature sets. For more information about the feature sets that are supported by the different text file formats, see File formats that are supported in Excel.
  4. Browse to the location where you want to save the new text file, and then click Save.
  5. A dialog box appears, reminding you that only the current worksheet will be saved to the new file. If you are certain that the current worksheet is the one that you want to save as a text file, click OK. You can save other worksheets as separate text files by repeating this procedure for each worksheet.
  6. A second dialog box appears, reminding you that your worksheet may contain features that are not supported by text file formats. If you are interested only in saving the worksheet data into the new text file, click Yes. If you are unsure and would like to know more about which Excel features are not supported by text file formats, click Help for more information.
