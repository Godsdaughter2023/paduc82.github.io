hpholm say:

I can't fix this issue for you, only help you to help yourself.
Please complete each step before proceeding.

Safely eject and physically remove the SD memory card from the phone.
Install the Windows Device Recovery Tool on the PC. This should install all the required drivers. Start up the recovery tool and connect the phone via a good quality USB data cable. It must connect without errors, and WDRT must offer to do what it does.
Verify there are no warning triangles in Window's control panel regarding drivers.

Disconnect the phone from the PC.

SD memory cards are often counterfeit. Most issues occur because of fake cards. Get a reputable brand Micro-SD class 10/UHS-I from a reputable dealer. Genuine cards come properly formatted out of the box = It is probably fake if the phone or the PC prompts to format it before use. It is also probably fake if it was cheap, also if it has all the right logos and came in a pretty display box.

SD card manufacturers are fighting an uphill battle with the pirates. There is often a help page how to spot the genuine items on the SD card manufacturer's website, and they may have a forum to discuss interoperability issues, and instructions how to report fakes.

If in doubt replace the card. I spent ~€30 per 36 color pictures back in the 1980's. A less than €100 memory card to hold thousands of pictures and hours of video, and re-use again and again, is a bargain in comparison.
Kingston RAM and SD cards always works for me, and comes with a no-nonsense warranty through the nearby store if I ever need it.

Since the card has been used in a PC, it may have issues, even if it is genuine:
Insert the SD memory card and let the phone format the card.
It must format without errors and it must display the advertised capacity.
From now on, the SD memory card stays in the phone for the rest of its useful life.

Verify that it works correctly both for app installs, still image and video recording, also when it fills up beyond 32 and 64 GB used space. Any media files must be transferred to the phone via USB cable or from the cloud. Do not remove the card from the phone.

There may be BIOS updates and chipset driver updates from the PC or motherboard manufacturer. Windows does not install those automatically.
Intel, Asus, Lenovo, HP etc. all have tools and websites with instructions and user support forums to assist.

This is the best I can do without hands-on the phone and the PC.

Please refer to a qualified technician if the issue is still not resolved.

Cheers,
Hans

Start with Word's Help on the topic: Word keyboard shortcuts

Most of the keyboard shortcuts used in the Mac version are directly comparable to those used in the Windows version. The major differences are often just a matter of differing keyboard layout. On a Mac the Command key are typically the equivalent of Alt & Control, respectively, on a PC.

Depending on which style of Mac keyboard you may also need to use the key labeled Command+End on  a Mac full keyboard, but if you have one of the compact keyboards or are using a Mac laptop there isn't a dedicated key for fn+Command+Left Arrow/Right Arrow, respectively. e.j. gun say:

I had a very similar problem with our HP DesignJet plotter.  If I used the "scale to fit page" option, the print preview (and printout) would show up as actual size (too small for my ANSI C sized sheet).  For me, however, manual scaling (175%) did the trick.  I knew it worked why my Print Preview view changed to show my data filling the page.

 

My settings were:

Print Active Sheets

Collated

Landscape Orientation

ANSI C - 17" x 22"

Last Custom Margins Setting (Left: 0.2", Right: 0.2", Top: 0.5", Bottom: 0.5")

Custom Scaling (Adjust to 175% normal size)

 

Make sure your print area is correct.  Make sure you pick the correct sheet size for the output you want.  Try some other sheet sizes to see if anything changes.
