marshall barton say:

If you are using a standard Access query that uses a local or linked table, the only reason I can think of that would cause that error is if the function is in a form or report module,  It must be in a standard module.

I think the better way to do this without using a function with data values imbedded in it, would be to have a table (named Exceptions) with two Text fields (named ExName and ExValue) one for the computer name and the other for the exception value,  E.g.
   S51515A01   SL03
   S54658A01   SI01
     . . .
Then your query can use an outer join to the Exceptions table to pick up the exception's value. maybe something roughly like:

SELECT sometable.computername, Nz(Exceptions.ExValue, sometable.transit) As MyTransit ...
FROM sometable LEFT JOIN Exceptions
   ON sometable.computername = Exceptions.ExName

or in an Updte query:

UPDATE sometable INNER JOIN Exceptions
   ON sometable.computername = Exceptions.ExName
SET sometable.transit = Exceptions.ExValue

If you are just trying to get a list of each of the drop down values, it's probably easier to use a string array to grab that list, and then iterate through the array, like:

beth melton say:

If you've added the TOC and deleted it several times, it could be the hang is being caused by too many extra bookmarks. Press Alt+F11 to open your VBA Editor. From the Insert menu, click Module, copy/paste this macro into the new module. Place your insertion point in the lines of the macro and then press F5 to run it.

Sub DeleteTOCBookmarks()
Dim oBmk As Bookmark
  With ActiveDocument
    .Bookmarks.ShowHidden = True
    For Each oBmk In .Bookmarks
     If .Bookmarks(oBmk).Empty = True Then oBmk.Delete
     If Left(.Bookmarks(oBmk).Name, 4) = "_Toc" Then oBmk.Delete
    Next
    .Bookmarks.ShowHidden = False
  End With
End Sub

rohn007 say:

"Office (version) 15" is another name for Office 2013.

Do you have an MS Office license?  If not, you can uninstall it to free up some space

On your own, you may be able to free up enough space to get back to work without paying someone else a lot of money.  Just be very cautious about what you delete.  If you are not sure, if you don't know what it is, DON'T delete it!

You can start by buying an external USB based drive. Depending on how much money you have you can buy a relatively small, simple USB "thumb drive" in the 64GB or less range for around US$50 or less.  For US$50 and up you can buy an external USB based hard drive with 500GB or more.  If you can afford it, buying a drive of 1TB or more is a good investment.

Starting by copying your personal document files, an informal "backup", protects them in case you do something to make your computer unusable.

After you have the USB drive, the simplest way free up space is to temporarily MOVE all of the files in your "MyDocument" folder to the USB drive. Once those files have been copied to the new drive and deleted from your C: drive, the system will have some "breathing room" to work with.

Hopefully you will buy a larger USB HD (1TB or more). After you have freed up that space you should do an 'image copy backup' of your system.  That way if you 'oops' and delete something you shouldn't, you will have a known good image copy to work with. This will allow you to restore your computer back to the working condition it was in when you made the backup:

http://www.howtogeek.com/136522/geek-school-learning-windows-7-backup-and-recovery/

After you have image copy backup you can proceed to do more cleanup to free up more space by running the "Disk Cleanup" tool as described below.

How big is your current HD?

  • Look in Windows Explorer,
  • right click on C:\ or "My Computer"
  • select properties
  • On the General tab it will show "used space" and "free space", let us know what those numbers are
  • click on the "disk cleanup" button
  • a job will run in background for a couple of minutes then display the "Disk Cleanup" dialog

The dialog will list categories of files temporary it can delete and show roughly how much space they are taking up.  Clicking on the "View Files" button will open Windows explorer to the currently selected category so you can look at them to decide if you want to keep or delete them.  After you select the categories to cleanup

There is also a "System Files" button.  This enables a few more categories to cleanup.  Stay away from them for now. But you will want to look at them later.

Here are a some links about the disk cleanup tool:

https://davescomputertips.com/free-up-lots-more-disk-space-with-windows-new-disk-cleanup-plug-in/

This link is for Win10, but the principles apply to Win 7, just slightly fewer features:

A Tip to Free Disk Space in Windows 10

https://www.petri.com/tip-free-disk-space-windows-10

Once you use disk cleanup you can copy the 'Mydocument' files you still need to use actively to your computer.

Remember, before doing any cleanup work, start with backups so you don't lose any files you need.
