Hi,

Although a full Windows repair would be ideal, below are other things you can try to improve your PC's performance.

Method 1: Temporarily disable Windows Search. Windows Search is what makes searching for files on your PC faster.

a. Press Win+X, and then select Command Prompt (Admin).

b. Type the following command and then hit Enter:
net.exe stop "Windows search"

Method 2: Disable SuperFetch. SuperFetch is a Windows service that helps speed up your PC's performance over time.

a. Open Run (Win+R), type services.msc, and then click OK.

b. Locate Superfetch.

c. Right-click on Superfetch, and then click on Pause/Stop.

Method 3: Adjust the system page file to improve system performance.

a. In the search bar, type Adjust the appearance and performance of windows, and then press Enter. This will open the Performance Options dialog box.

b. Go to Advanced tab and click the Change... button in the Virtual memory section.

c. Uncheck the check box for Automatically manage paging file size for all drives option.

d. Select drive C: or the drive where Windows 10 is installed.

e. Select Custom size by ticking its radio button.

f. Change the Initial size (MB) and Maximum size (MB) to the recommended value that you will find just below the Total paging file size for all drives section.
