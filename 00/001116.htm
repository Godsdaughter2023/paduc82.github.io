Search words starts with Capital letters - Microsoft Community

void func say:

Search words starts with Capital letters

Hi,

Is it possible to search words that starts with Capital letters?. My aim is to identify words which starts with capital letter but not the first word of a sentence. This is to make sure that, fisrt word in a sentence alone should start with Capital letter

Thanks!

deanh say:

You can use Wildcards to select the captial letter at at start of a word, the code for the Find box is <[A-Z]

To use Wildcards in the Find/Replace dialog, click on the More button and check the appropraite box on the left of the dialog.

This code will find all captials at the start of any word - that includes at the beginning of sentences.

What you could do is several Find/Replaces with the use of Highlight, found under the Fomat button

First - do the find with <[A-Z], and replace with Highlight (ensure you have a colour selected in the Highlight icon.

Then do finds for a captial that is after a paragraph, ^13[A-Z] and replace with Not Highlight (click Highlight twice).

The repeat that for a capital that is after an end-sentence punctuation mark, ([.?]) [A-Z] - there is a space between these two sets. This is looking for only one space because as you know with modern best practice there should only be one space between sentences, so you may need to do a replace of multiple spaces to one space :-)

Anyway, you should now only have captial letters highlighted that are at the start of a word and NOT at the start of a sentence highlighted, now you can do a search for only highlight and deal with each transgressor as required.

Hope this works for you.
