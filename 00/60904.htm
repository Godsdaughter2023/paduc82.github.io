andre furtado [msft] say:

To avoid the Start Menu to store and display recently opened programs, do the following:

  • Right-click your taskbar
  • Click in Properties
  • Go to the Start Menu tab
  • Uncheck the desired checkboxes in the Privacy group

To adjust Windows search settings, try:

  • Open the Control Panel (Start Menu -> Control Panel)
  • Search for "Search settings" in the top-right search textbox
  • Click on Change how Windows searches
  • Click on the Modify button
  • Remove the programs which you don't want Windows search to index

    dominicp say:

    If your system shuts down without warning or event log entry it is your hardware which is shutting the system down.

    If your system has been in use for some time, a common problem is dust build-up on the CPU cooler. There is a heat sensor under the CPU which will shut down the system if the CPU is detected as overheating. Check and clear any dust mat between the CPU fan and heat sink body.

    A failing power supply unit can also cause such problems.

    To avoid loss of work, set up any automatic backup options in your applications or get a third party application which will do continuous backups.
