stefan blom say:

What you describe is a result of running the Document Inspector on the document and choosing to remove personal information. Once you have done that, Word continues to remove personal information each time the document is saved and reopened.

To fix the problem, make sure the affected document is the active document in Word, click File | Options | Trust Center | Trust Center Settings | Privacy Options, clear the "Remove personal information from file properties on save" and click OK twice.

Note that the above steps won't restore lost author/reviewer names, but they will prevent Word from removing names from future edits made to the document.

huni shen msft say:

Hi Felix,

 

If you’ve tried using Support and Recovery Assistant for Office 365 and repaired Outlook Data file and the issue still persists. You can try to start your Outlook in safe mode and disable add-ins and see if it can be helpful:

 

Start Outlook in safe mode

  1. Choose Start > Run.
  2. Type Outlook /safe, and choose OK.

 

Disable add-ins

  1. Open File > Options > Add-ins
  2. Select “COM Add-ins” > Go
  3. Uncheck any add-ins that you’re not using
  4. Click OK

 

Meanwhile, you can try running the /resetnavpanecommand removes all customizations to the navigation pane.

  1. Choose Start > Run.
  2. In the Open box, type Outlook.exe /resetnavpane, and then choose OK.
