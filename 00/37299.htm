a. user say:

That's a pretty glaring error for those folks.  There are other typos there as well.

If there is no boot.ini, it is true that there will be course be no BOOT.INI tab in msconfig.

But if the boot.ini is afflicted and you use bootcfg /rebuild and answer all the queries, it will build on top of the afflicted boot.ini so before doing that, you should really delete or rename the afflicted boot.ini and then use bootcfg /rebuild and it will make a new boot.ini.  This tidbit is missing from the Microsoft instructions so trust me on this one (or try it).  Some people run bootcfg /rebuild repeatedly and that might work eventually but now your boot.ini is very sloppy.

And many people either don't have a genuine bootable XP installation CD or even a CD/DVD drive - then what are you going to do?

In a single partition configuration, XP will boot just fine if there is no boot.ini.  It will complain, but still boot.   Non believers, try it!  So from the Recovery Console, you can rename/delete the afflicted c:\boot.ini (so there isn't one), boot normally, then you can either fix your afflicted boot.ini or just create a new one (if it is missing)  by hand from My Computer, Advanced, Startup and Recovery, Settings, Edit, (copy, paste, save):

[boot loader]
timeout=5
default=multi(0)disk(0)rdisk(0)partition(1)\WINDOWS
[operating systems]
multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /fastdetect

So if XP boots okay, so I would just make a new boot.ini from My Computer - that might be a little easier since some folks have trouble with Recovery Console or they don't have a CD, or they can't get past the password prompt (especially with XP Home) to get in, etc.

I hope it works out.

Hi soniawarner,

Your bluetooth is not showing up in Device Manager, right? In that case, you can check if the Bluetooth Support Service is turned off. If the Bluetooth Support Service is turned off, the bluetooth won't show up in Device Manager.

You can follow the below steps to check for it. If it's turned off, just turn it on.

1) On your keyboard, press the Win+R (Windows key and R key) at the same time to invoke the run box.


2) Type services.msc and press Enter on your keyboard. 

3) Double-click the Bluetooth Support Service.

4) If you see the Service status is Stopped, click the Start button and click Apply. If you see the “Service status” is “Running”, skip this step.

5) On the Startup type list, click Automatic then click Apply.

6) Check to see if the problem is resolved.
