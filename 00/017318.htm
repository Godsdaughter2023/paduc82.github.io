I understand that you want to remove the value selected if it was selected earlier...

Try

Private Sub Worksheet_Change(ByVal Target As Range)

'Code by Sumit Bansal from https://trumpexcel.com

' To allow multiple selections in a Drop Down List in Excel (without repetition)

Dim Oldvalue As String

Dim Newvalue As String

Application.EnableEvents = True

On Error GoTo Exitsub

If Target.Column = 2 Then

  If Target.SpecialCells(xlCellTypeAllValidation) Is Nothing Then

    GoTo Exitsub

  Else: If Target.Value = "" Then GoTo Exitsub Else

    Application.EnableEvents = False

    Newvalue = Target.Value

    Application.Undo

    Oldvalue = Target.Value

      If Oldvalue = "" Then

        Target.Value = Newvalue

      ElseIf InStr(1, Oldvalue, Newvalue) = 0 Then

            Target.Value = Oldvalue & ", " & Newvalue

      Else

        ' Target.Value = Oldvalue

        ' Replace the value if already present

        Target.Value = Replace(Oldvalue, Newvalue, "")

        ' This is to remove the comma and space left behind

        If Left(Target.Value, 2) = ", " Then

            Target.Value = Mid(Target.Value, 3, Len(Target.Value))

        ElseIf Right(Target.Value, 2) = ", " Then

            Target.Value = Mid(Target.Value, 1, Len(Target.Value) - 2)

        Else

            Target.Value = Replace(Target.Value, ",  , ", ", ")

        End If

        End If

      End If

    End If

Application.EnableEvents = True

Exitsub:

Application.EnableEvents = True

End Sub

john korchok say:

The simplest would be to resave as JPG. Here are the steps:

  1. Make a copy of a presentation, then place it in a folder.
  2. Change the file ending to .zip. Confirm this choice when macOS asks if you want to change it.
  3. Right-click and choose Open With>Archive Utility. A new folder named after the original file name is created beside the zip file.
  4. Open the folder, then look in ppt/media for all your picture files. They won't have the original names. Open all the PNG files in Preview, then File>Export as JPEG.
  5. Open the original presentation and replace the graphics.
