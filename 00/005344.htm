jllatham say:

You'll need to do some up-front work to get this enabled, but once that's done it should work quite well for you.

 

First, links cannot be updated without opening the workbooks with links in them.  A physical limitation that there is no way around.  Any changes made in the source are not 'seen' in the linked workbooks until they are opened.

 

This code assumes that the 20 workbooks have links on a sheet with the same name and cell addresses with links on the sheet are all the same.  If that's not the case, we can work around it later.

 

The up-front work:  add a sheet to the master workbook (I called mine LinkSheet] and in cell A2 on that sheet set up a link to one of the 20 workbooks (for future planning, a link to one of the linked cells in that workbook would be a good one to use, otherwise a link to any cell on any sheet in that other workbook).   Repeat, working down column A with 1 link to each of the other 20+ workbooks.  That completes the up-front work.

 

Now anytime you want to verify the data, just run the macro below.  There are some Const values you'll need to change, and I've made them bold to call your attention to them.

 

Sub VerifyLinkedData()
  'quickly opens and verifies data in
  'other workbooks based on a list
  'of cells in this workbook that
  'are linked to the other workbooks
  'Name of sheet with the links to the other
  'workbooks is "LinkSheet" and
  'the links to the other workbooks
  'begin at cell A2 on that sheet
  '
  'The other workbooks are ALL presumed
  'to be closed when you run this code
  '
  'CHANGE THESE AS REQUIRED
  '
  'assumes all other workbooks have
  'sheets of same name and linked cells
  'on them are all the same
  '
  'These are the name of the sheet in the
  'other workbooks that have links to this workbook
  'and the addresses of the cells with links on
  'that sheet to be tested
  Const testSheetName = "Sheet1" ' change!
  Const testCell1 = "Q9" ' change
  Const testCell2 = "X44" ' change
  'these are the name of the sheet in THIS workbook
  'that the other workbooks link to and
  'the addresses of the cells they link to
  Const myDataSheetName = "Sheet2" ' change
  Const checkCell1 = "B5" ' change
  Const checkCell2 = "C5" ' change
 
  Dim otherWB As Workbook
  Dim cellAddress As String
  Dim otherPath As String
  Dim linksList As Range
  Dim anyLink As Range
  Dim thePath As String
  Dim bookName As String
  Dim sheetName As String
  Dim char1Pos As Integer
  Dim char2Pos As Integer
  'change LinkSheet in this next line if needed
  Set linksList = ThisWorkbook.Worksheets("LinkSheet").Range("A2:" _
   & ThisWorkbook.Worksheets("LinkSheet").Range("A" & Rows.Count).End(xlUp).Address)
 
  For Each anyLink In linksList
    otherPath = anyLink.Formula
    otherPath = Replace(otherPath, "=", "")
    otherPath = Replace(otherPath, "'", "")
    'get sheet name needed in other workbook
    char1Pos = InStr(otherPath, "]") + 1
    char2Pos = InStr(char1Pos, otherPath, "!")
    sheetName = Mid(otherPath, char1Pos, char2Pos - char1Pos)
    otherPath = Left(otherPath, char1Pos - 1)
    otherPath = Replace(otherPath, "[", "")
    otherPath = Replace(otherPath, "]", "")
    'other path should now be valid path
    'to the other workbook, open it
    'with "update paths set to True, Read Only set to true
    Application.DisplayAlerts = False
    'if otherWB has macros, don't run them
    Application.EnableEvents = False
    Set otherWB = Workbooks.Open(otherPath, True, True)
    Application.EnableEvents = True
    Application.DisplayAlerts = True
    ThisWorkbook.Activate
    'the testing
    'the 'error' message output could go to another sheet
    'and could contain more info like sheet name(s) and
    'cell address(es).
    If otherWB.Worksheets(testSheetName).Range(testCell1).Value <> _
     ThisWorkbook.Worksheets(myDataSheetName).Range(checkCell1) Then
      MsgBox "Value in " & vbCrLf _
       & "'" & otherWB.FullName & "'" & vbCrLf _
       & "has NOT updated.", vbOKOnly, "Data Mismatch"
    End If
    If otherWB.Worksheets(testSheetName).Range(testCell2).Value <> _
     ThisWorkbook.Worksheets(myDataSheetName).Range(checkCell2) Then
      MsgBox "Value in " & vbCrLf _
       & "'" & otherWB.FullName & "'" & vbCrLf _
       & "has NOT updated.", vbOKOnly, "Data Mismatch"
    End If
    otherWB.Close False
  Next
  Set otherWB = Nothing
  MsgBox "Data Verification Finished", vbOKOnly + vbInformation, "Task Completed"
End Sub
