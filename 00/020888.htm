  • Certain spam-fighting techniques can intentionally introduce delays
  • Various networking issues can cause an email server to appear down, with the same result.

Important Note: Antivirus software can help protect your computer against viruses and other security threats. In most cases, you shouldn't disable your antivirus software. If you're connected to the Internet or a network while your antivirus software is disabled, your computer is vulnerable to attacks.

When an email is delayed for a significant amount of time, the best way to check why an email is delayed is to analyze the email headers to see where the delay is occurring.

  • Login to your Hotmail.com (For webmail)
  • Select the email that has been delivered lately
  • Right Click the email and select View message source, it will open up a new page.
  • There you will find the relevant portions of the email header of message sent from the delayed account domain to your regular email account.
  • Email headers are read from the bottom up.

In that page you will find the IP of the sender. And list of all the mail server it has reached before delivering the email to the right address.

ossiemac say:

Try the following code. Because you want to keep the first and last sheet as is then it reads the intermediate worksheet names into an array and sorts the array. It then moves the sorted worksheets to their respective sorted position and leaves the first and last as is.


Sub Macro1()
    Dim SortOrder As Variant   'Needs to be variant so Cancel will work
    Dim strPrompt As String
    Dim wsArray()
    Dim i As Long
    Dim j As Long
    Dim varTemp
    Dim lngSht As Long
   
    strPrompt = "Enter A for ascending sort or D for decending sort." _
                & vbCrLf & vbCrLf & "Click Cancel to exit process."
    Do
        SortOrder = Application.InputBox(Prompt:=strPrompt, Type:=2)
        If SortOrder = False Then
            MsgBox "User cancelled. Processing terminated." 'MsgBox is optional
            Exit Sub   'User Cancelled
        End If
       
        SortOrder = UCase(SortOrder)
        If SortOrder = "A" Or SortOrder = "D" Then
            Exit Do   'Entry valid
        Else
            strPrompt = "Enter A or D only or Cancel to exit process."
        End If
    Loop
   
    'Dim array size to hold sheet names from second to second last sheet
    ReDim wsArray(1 To Worksheets.Count - 2)
   
    'Assign the worksheet names from the 2nd sheet to 2nd last sheet to the array
    For i = 1 To UBound(wsArray())
        wsArray(i) = Worksheets(i + 1).Name
    Next i

    'Sort the array
    For j = 1 To UBound(wsArray())
        For i = 1 To UBound(wsArray()) - 1
            If SortOrder = "A" Then     'Sort Ascending
                If wsArray(i) > wsArray(i + 1) Then
                    varTemp = wsArray(i)
                    wsArray(i) = wsArray(i + 1)
                    wsArray(i + 1) = varTemp
                End If
            Else        'Sort Decending
                If wsArray(i) < wsArray(i + 1) Then
                    varTemp = wsArray(i)
                    wsArray(i) = wsArray(i + 1)
                    wsArray(i + 1) = varTemp
                End If
            End If
        Next i
      
    Next j
   
    lngSht = 1 'Initialize to first worksheet number
   
    'Move the worksheets to their relative sorted position
    '(First and last sheet should remain as is.)
    For i = 1 To UBound(wsArray())
        Worksheets(wsArray(i)).Move After:=Sheets(lngSht)
        lngSht = lngSht + 1
    Next i
   
End Sub

It appears that you have deleted the post that references your problems using dates as worksheet names but I will answer it anyway.


Some characters are illegal in worksheet names. (In particular the slash (/).) If you copy the following code into a blank workbook and run it then it will list the illegal characters.


Sub SheetIllegalChars()
    Dim r As Long
    Dim varErrNumb As Variant
    With Worksheets(1)
        .Select
        .Columns("A:B").Clear
        .Cells(1, 1) = "ASCII code"
        .Cells(1, 2) = "Character"
        For r = 0 To 255
            varErrNumb = 0
            On Error Resume Next
            ActiveSheet.Name = "Sheet1_" & Chr(r)
            varErrNumb = Err.Number
            On Error GoTo 0
            If varErrNumb > 0 Then
                .Cells(.Rows.Count, 1).End(xlUp).Offset(1, 0) = r
                .Cells(.Rows.Count, 1).End(xlUp).Offset(0, 1) = Chr(r)
            End If
        Next r
    End With
End Sub



If you want to use dates as worksheet names and you want to sort the worksheets then they need to be formatted as Year Month Day in numeric format like the following:-


2013 12 13


The reason for Year Month Day is that the worksheet name is treated as text when sorting. (Using the serial number you mentioned will sort correctly as text.)


Note you cannot sort the worksheets if you use the alpha month like the following.


2013 Dec 13


To reproduce a date in the Year month day format for a worksheet name then use the following code example. Note that hyphen is a legal character in a worksheet name


Sub DateSheetName()
    'Assumes that the date is in cell A2 of Sheet2 and you want
    'to rename the 3rd worksheet in the workbook.

    Worksheets(3).Name = Format(Worksheets("Sheet2").Range("A2"), "yyyy-mm-dd")

End Sub
