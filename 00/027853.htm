david marcovitz say: I can think of two ways to do this. One would be fairly easy with VBA code, but VBA code is not that easy. The alternative is to use a complex set of animations. This would be easiest if you only show the line for the correct answer. If the line will show up for any answer they choose, this would be far goo complex. Create 8 text boxes on the left. Create 8 text boxes on the right. Duplicate the 8 text boxes on the right so they cover the 8 text boxes already created. Draw the 8 lines. You now have 32 objects and a whole bunch of animations. Each of the topmost text boxes on the right will need an appear animation to be triggered by clicking on the choice on the left that matches it. Additionally, each of those text boxes will need a disappear animation triggered by the 7 text boxes on the left that don't match it. So far, this means that a click on a box on the left will make the 7 duplicate wrong answers disappear, and the 1 right answer appear. If you do this right, the change will not be visible on the screen because the original answers on the right will always be there. Next, you need to animate each of the lines to appear (or fly in or whatever) triggered by when the duplicate text box is clicked. 72 animations later, it should all work by clicking on the left and then clicking on the correct choice on the right.
