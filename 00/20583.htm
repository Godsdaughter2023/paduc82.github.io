Tablet/Desktop Mode and the device type distinction are obsolete - Microsoft Community

joscelin.trouwborst say:

Tablet/Desktop Mode and the device type distinction are obsolete legacies, get rid of them ...

I would like to discuss the following.

Tablet Mode, Desktop Mode, I feel these terms and the modal UI/UX switch keep us in a fundamental mind frame of a complete difference between the two, while this is actually (no longer) the case, let's get rid of it.

'Tablet' used to refer to a portable device with a display size from about 7" to about 10" and with touch screen input only.

'Desktop' used to refer to a non-portable device with a display size from about 15" to about 23" and with keyboard and mouse input only.

Also ...

'Smartphone', a pocketable device with a display size up until about 6".

'Notebook', a portable device with a display size from about 13" to about 17".

Meanwhile, the collection of input options has grown from

  • keyboard
  • mouse
  • touch screen

to include

  • pen
  • voice
  • gesture
  • dial.

Input options are no longer limited to specific device types and can all be available at the same time over a display size range from 3.5" to wall mount size!

Hence, the Tablet Mode / Desktop Mode UI/UX distinction has become non-sense.

However, there will always be an individual user preference in UI/UX, reason to continue and extend those options in Windows, but no longer dependent on 'Tablet/Desktop'.

Looking forward to comments improving or countering my reasoning and ideas for UI/UX options.

Note, that the required distinctive qualification for this post to be for 'PC' or 'phone' is obsolete legacy as well.

Sub Compare_two_wbks()
'June 23, 2016
Const v1A As String = "A" ' wb1 column 1st
Const v1B As String = "B" 'wb1 col 2nd
Const v2A As String = "A" 'wb2 col 1st
Const v2B As String = "B" 'wb2 col 2nd
Const sWB1 As String = "WorkbookA.xlsx" 'wb1 name
Const sWB2 As String = "WorkbookB.xlsx" 'wb2 name
Const sName1 As String = "Sheet1-A" 'wb1, data in sheet name
Const sName2 As String = "Sheet1-B" 'wb2, data in sheet name
v = Array("Names", "Hobbies")
Dim wb As Workbook, wb1 As Workbook, wb2 As Workbook
Set wb = ThisWorkbook
Dim rr As Range
Dim sPath
sPath = "E:\folder A\folder B\"  'wbks in this path
Dim r1 As Long, r2 As Long, t As Long, x As Long
Application.ScreenUpdating = False
Set wb1 = Workbooks.Open(sPath & sWB1)
Set wb2 = Workbooks.Open(sPath & sWB2)
r1 = wb1.Sheets(sName1).Cells(Rows.Count, v1A).End(xlUp).Row
r2 = wb2.Sheets(sName2).Cells(Rows.Count, v2A).End(xlUp).Row
wb.Sheets.Add before:=wb.Sheets(1)
wb.Sheets(1).Cells(1, 1).Resize(, 2) = v
t = 2
For x = 2 To r1
Set rr = wb2.Sheets(sName2).Cells(2, v2A).Resize(r2).Find(What:=wb1.Sheets(sName1).Cells(x, v1A), SearchOrder:=xlByRows, SearchDirection:=xlPrevious)
If Not rr Is Nothing Then
If wb1.Sheets(sName1).Cells(x, v1B) = wb2.Sheets(sName2).Cells(rr.Row, v2B) Then
wb.Sheets(1).Cells(t, 1).Value = wb1.Sheets(sName1).Cells(x, v1A)
wb.Sheets(1).Cells(t, 2).Value = wb1.Sheets(sName1).Cells(x, v1B)
t = t + 1
End If
End If
Next
wb1.Close False
wb2.Close False
wb.Save
Application.ScreenUpdating = True
End Sub

Sub Compare_Two_WBooks_2()
'June 24, 2016
Const v1A As String = "M" ' wb1 col 1st
Const v1B As String = "N" 'wb1 col 2nd
Const v2A As String = "D" 'wb2 col 1st
Const v2B As String = "E" 'wb2 col 2nd
Const sWB1 As String = "WorkbookA.xlsx" 'wb1 name
Const sWB2 As String = "WorkbookB.xlsx" 'wb2 name
Const sName1 As String = "Sheet1-A" 'wb1, data in sheet name
Const sName2 As String = "Sheet1-B" 'wb2, data in sheet name
v = Array("GATE-1", "GATE-2/WB1", "GATE-2/WB2")
Dim wb As Workbook, wb1 As Workbook, wb2 As Workbook
Set wb = ThisWorkbook
Dim rr As Range
Dim sPath
sPath = "E:\folder A\folder B\"  'wbks in this path
Dim r1 As Long, r2 As Long, t As Long, x As Long
Application.ScreenUpdating = False
Set wb1 = Workbooks.Open(sPath & sWB1)
Set wb2 = Workbooks.Open(sPath & sWB2)
r1 = wb1.Sheets(sName1).Cells(Rows.Count, v1A).End(xlUp).Row
r2 = wb2.Sheets(sName2).Cells(Rows.Count, v2A).End(xlUp).Row
wb.Sheets.Add before:=wb.Sheets(1)
wb.Sheets(1).Cells(1, 1).Resize(, 3) = v
t = 2
For x = 2 To r1
Set rr = wb2.Sheets(sName2).Cells(2, v2A).Resize(r2).Find(What:=wb1.Sheets(sName1).Cells(x, v1A), LookIn:=xlValues, LookAt:=xlWhole)
If Not rr Is Nothing Then
wb.Sheets(1).Cells(t, 1).Value = wb1.Sheets(sName1).Cells(x, v1A)
wb.Sheets(1).Cells(t, 2).Value = wb1.Sheets(sName1).Cells(x, v1B)
wb.Sheets(1).Cells(t, 3).Value = wb2.Sheets(sName2).Cells(rr.Row, v2B)
t = t + 1
End If
Next
wb1.Close False
wb2.Close False
With ActiveSheet.UsedRange
.HorizontalAlignment = xlCenter
.Columns.AutoFit
End With
wb.Save
Application.ScreenUpdating = True
End Sub
