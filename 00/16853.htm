To find or replace a nonprinting paragraph mark, use the code  ^p . To find this and other codes, click the More button in the Advanced Search/Replace dialog and then click the Special button at the bottom of the dialog. Each item in that list inserts the corresponding code in the search term or the replacement term (wherever the cursor is when you click the button).

If you're doing a wildcard search, use the code  ^13  instead. Also see http://www.gmayor.com/replace_using_wildcards.htm.
