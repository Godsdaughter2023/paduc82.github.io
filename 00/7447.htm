File recovery software is unlikely to help you because you are dealing with different versions of a file rather than deleted files.

You could open Windows Explorer, then right-click the problem file, then click "Restore previous version". It is possible that there is a version there that is more recent than 28th December.

a. user say:

Hello,

Thank you for posting your query in Microsoft Community.

It happens when there is some other program or application is under process of installation. You'll receive to get this error if that existing installation gets corrupted or stuck in queue. Hence system won't allow for another installation, till the time it completes. I suggest you to try the below mentioned troubleshooting steps and check if they helps. 

Method 1:

Check the status of Windows Installer Service.


* Click Start, type Services into the start search box and select the same from the list when it appears
* Find Windows Installer, double-click “Windows Installer” in the Services list.
* Under General tab, make sure the service is started under “Service status”. If the service is not already running, under Service status, click Start, and then click OK. 

Method 2:

Unregister and re-register the Windows Installer

* Click Start, type MSIEXEC /UNREGISTER into the search box and hit Enter

* Click Start again, type MSIEXEC /REGISTER into the search box and hit Enter

* Now, try installation again and check if it allows.

Keep us posted if you require further assistance, we're glad to assist you.
