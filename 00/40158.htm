ian ser say:

Hi Johann,

Error code 80072EE2 is caused by the network adapter TCP autotuning configuration. To resolve this issue, follow these steps:

  1. Press Windows key on your keyboard and then type CMD.
  2. Right-click CMD from the search result and then choose Run as administrator.
  3. Type this command on the window that would appear:

    netsh int tcp set global autotuninglevel=disabled
  4. Restart your device and run a Windows update again.

Update us if there are improvements.
