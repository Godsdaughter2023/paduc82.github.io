considerate_guy say:

If it doesn't save as a movie on the original computer, I have a feeling that moving it to another computer may not help.  I'd solve the issue on the original computer before introducing other variables.

The two main reasons that a movie won't save:

  1. Original source files are not 100% compatible with Movie Maker.  If this is the case, you'll need to convert the problem source videos to a compatible format like WMV.  Replace the clips and see if it will save.
  2. Project file has become too long and/or too complex.  Did the project save when it was smaller?  If so, the best solution is to create several copies of the project file and pare each one down to just part of the longer movie.  See if a smaller portion of the project will save as a movie.

For more help, check out www.papajohn.org > Problem Solving > Can't Save a Movie

P.S.  As PapaJohn noted, a project file (size = 34KB = ~34,000 bytes) is actually pretty small.  That's because a project file is not the container of all the source video, audio and photos.  It just references them off your computer. Video is usually on the order of MBs (million bytes) or even GBs (billion bytes). That's going to be the harder task for moving, but I don't see why an external hard drive wouldn't be able to move everything though.
