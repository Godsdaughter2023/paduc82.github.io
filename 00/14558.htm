Thank you for getting back to us. Just to reiterate the steps provided on this thread, please perform the instructions provided below to isolate the issue:

1. To add or remove note space in OneNote, use the “Insert Space” button. This button appears in the “Insert” button group on the “Insert” tab of the Ribbon. 3. When that icon appears in the correct position in the note, click and drag downward with the mouse to create the extra space. All the content below the line is shifted downward in the page as you drag. After selecting the desired amount of space, release the mouse button to add the selected space.

rick xu msft support say:

Hi Tad,

 

It’s not feasible to control receiving or replying to read receipts on an organization level. As Lance mentioned above, “whether to send a read receipt depends on which client (device) the recipient read the email requesting a read receipt. The action that send a read receipt is processed by the client instead of the mail server.

”

The PowerShell command “Set-MailboxMessageConfiguration *** Email address is removed for privacy *** -ReadReceiptResponse NeverSend” is applied to when you read the message in OWA (Outlook for Web App) and it will not work for a smart phone.

 

Regarding this problem, it’s suggested you submit feedback via this link: https://office365.uservoice.com/forums/273494-outlook-and-calendar . Many features have been designed or upgraded based on our customers’ constructive comments.

 

Thanks for your understanding.
