The session time out error in Excel online appears when the maximum memory or file size limit for Excel Online is exceeded. In the 32-bit version of Office, the maximum files size for a workbook containing a Data Model is 2 GB, and the maximum memory that can be consumed by a workbook is 4 GB. shirley wang msft say:
Hi Andy,

Here’s what happens after Office 365 ProPlus is installed on a computer that has shared computer activation enabled.

1. A user logs on to the computer with her account.
2. The user starts an Office 365 ProPlus program, such as Word.
3. Office 365 ProPlus contacts the Office Licensing Service on the Internet to obtain a licensing token for the user. 
4. If the user is licensed for Office 365 ProPlus, a licensing token is stored on the computer in the user's profile folder, and Office 365 ProPlus is activated. The user can now use Office 365 ProPlus.

The licensing token that is stored on the shared computer is valid only for a few days. As the expiration date for the licensing token nears, Office 365 ProPlus automatically attempts to renew the licensing token when the user is logged on to the computer and using Office 365 ProPlus.

If the user doesn’t log on to the shared computer for several days, the licensing token can expire. The next time that the user tries to use Office 365 ProPlus, Office 365 ProPlus contacts the Office Licensing Service on the Internet to get a new licensing token.Therefore, we need to log into the client with the account every several days. 

Given the situation, please check if activation for Office 365 ProPlus succeeded via going to the following folder on the shared computer%localappdata%\Microsoft\Office\16.0\Licensing. If activation succeeded, there are some text files in the folder. Don’t make any changes to these files.

Shirley
