Yes what you’re not seeing is used by Windows and my guess is that you set-up Windows Backup to backup onto your C drive. What happens is that the backup becomes a system file that isn't read unless you’re restoring from the backup. So when you try to get the size of the backup you will always see 0 when you check properties. To see the backup you will need to take Ownership of the backup and sub folders. Then you need to add permissions for you to take full control of the file and sub folders. After doing this you will find the lost GB you’re looking for.

This is usually heat related, defective hardware, memory or even processor though it is  "possible" that it is driver related (rare).
