Contact data does not sync when using an IMAP <email account>. Has nothing to do with Outlook but with the IMAP protocol which is an industry standard and applicable to any <email client> (of which Outlook is just one of many).

>>
with the only contact option to choose 'contacts (on this computer)' which is empty
<<

"This Computer Only" means just that - data is stored on the computer on which the account has been configured. As a FYI, it's really not a good idea to maintain data in any folders (i.e. contact/calendar) labeled as "This Computer Only" since if the account is ever been removed from the Outlook profile, the underlying .ost file in which these folders exist, will also be automatically deleted resulting in any data in those folders also being lost. Also, if Outlook can't find the .ost file, it will automatically recreate the file and re-sync the email folders with the server. The "This Computer Folders" will be empty

>>
what I would really like to do is use another 'contacts' folder that has my contacts in it.
<<

You can't simply select another contact folder. What is required is to maintain your contacts in an Outlook.com/Exchange account.
