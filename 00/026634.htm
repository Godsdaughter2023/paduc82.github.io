Because column E is string data, you need to separate the data to allow sorting by both house number and street name.

 

For example:

In F1 enter:

=--LEFT(E1,FIND(" ",E1)) and copy down

In G1 enter:

=MID(E1,FIND(" ",E1)+1,256) and copy down

 

So if E1 has 1000 Maple Avenue, then F1 would have the 1000 and G1 would have the Maple Avenue

 

Just sort by column G and then column F.
