saltgrass say: It appears your current install is configured as Legacy or MBR.  UEFI is the newer configuration and will normally be set up for OEM computers that come with Windows 8 pre-installed.

"Windows cannot be installed to this disk. The selected disk has MBR partition table. On EFI systems, Windows can only be installed to GPT disks."

This message would indicate you are booting to the UEFI version of the install media, which will not install on a MBR configured drive.  If you want to do a clean install, delete the current partitions during the install and then let it install as it wants.  If you want the Windows partition to be smaller than ~ 460 GB, create a partition the size you want to use and let the install go in that area.  If it gives you an error when you create a partition during the install, just let it install on the entire drive and shrink the partition later.

Whether you boot to the UEFI or Legacy version of the install media is controlled in the Bios.  It may be set to see UEFI only media, or Legacy only, or both.  If you open the Boot Device Menu, for a DVD you should see two entries with one of those showing a UEFI annotation.
