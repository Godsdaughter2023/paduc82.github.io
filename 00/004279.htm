Organizing favorites in alphabetical order for Internet Explorer in - Microsoft Community

cruzr77 say:

Organizing favorites in alphabetical order for Internet Explorer in Windows 7 & 10

I was trying to answer someone's question on this subject, but somehow wound up here. I hope it's OK, and I hope that it can still help folks.

Although I performed this on Windows 7, I am told it can also be done on Windows 10.

Please forgive the messed up numbering, but something happened when I transferred this from a word doc. 

After researching help pages at Microsoft, it became apparent that there is no one-button solution to this dilemma.  After a couple of minutes I realized that each time a person adds a new folder or file, it is place at the bottom of the FAVORITES list within Internet Explorer. Thus, the folders and files are arranged by the order in which the favorite is saved.  After a couple of minutes I realized that each time a person adds a new folder or file, it is place at the bottom of the FAVORITES list within Internet Explorer. Thus, the folders and files are arranged by the order in which the favorite is saved. Therefore, Microsoft Window’s ascending rules do not apply here. After another couple minutes, I found a work-around to the problem of alphabetically organizing your favorites within Internet Explorer. While it takes around 5 to 10 minutes to complete (or longer if you desire to redo your folders) it is safer and less expensive than using a third party software; and definitely a whole lot faster than trying to alphabetically organize your favorites within Internet Explorer (God, what a nightmare).

While this solution looks lengthy, it really is quite simple. I just tried to explain it so that even a novice could do it. If I had the know-how, I would put it on video and upload it somewhere. Even if someone has already done so, it is still worth reading this as I offer suggestions to structuring your files for quicker and easier access. Hope this all helps. It took me (a two finger typist) a while to put this together.

  1. First open up WIDOWS EXPLORER and create a folder in;

    1. DOCUMENTS

    2. or  DESK TOP

  2. Name the folder something that will keep it at the top of the list.

    For example:        “_ 1a Temp Holding Folder – Favorites”   (without the quotation marks)

    This will make it easier to access

  3. IMPORTANT – For the following 3 folders perform the operations in step 4:  

     

    1. For the folder you just created,

    2. For DOCUMENTS or DESK TOP (where you created the new folder),

    3. For the FAVORITES folder within WINDOWS EXPLORER.

  4. If not already set, please set the 3 folders above, to the following selections:

    1. Select folders to display Details

      1. All the way to the right of the bar above the DOCUMENTS LIBRARY window select the chevron next to the CHANGE YOUR VIEW button.

      2. From the drop-down window select DETAILS.

    2. Select to arrange by folder

      1. At the top right corner of the DOCUMENTS LIBRARY window select ARRANGE BY: FOLDER

    3. Select to have folder and files in ascending order

      1. In the DOCUMENTS LIBRARY window click on “NAME” until folders appear in ascending order.  Of course Microsoft’s ascending rules will apply:

         Of course Microsoft’s ascending rules will apply:

        1. Dash  (_)    comes before;

        2. Space    comes before;

        3. Numbers (starting with “0”)     comes before;

        4. Letters (starting with “A”)

        5. If I missed something please forgive me; I didn’t bother to look it up.

  1. AT THIS POINT – It would be a good time to organize all your favorites into folders in the order you desire them to appear in the FAVORITES DROP-DOWN WINDOW in INTERNET EXPLORER. For myself, instead of having a bunch of sub-folders and files within a file called “Shop”, I prefer to have them easily accessible in the root directory of the FAVORITES WINDOW. Example:

Shop – Automotive - _Temp Holding File

Shop – Automotive – Factory Manuals

Shop – Automotive – OBDII Diagnostic Code – Info

 Shop – Automotive – OBDII Diagnostic Code – Reader Co’s

Shop – Electrical Electronics - _Temp Holding File

Shop - Electrical Electronics - How To

Shop - Electrical Electronics – Products

Shop - Woodworking - _Temp Holding File

Shop - Woodworking – Fasteners

Shop - Woodworking - How To

Shop - Woodworking - Project - Saw Router Stands

Shop - Woodworking - Projects - Plans

  1. Cut and Paste all your favorites from the FAVORITES folder to the folder you created. IMPORTANT – DO NOT COPY (folders and files must be moved away from this folder).

    IMPORTANT – DO NOT transfer the folder called “FAVORITES BAR”.

    1. Open the FAVORITES folder within WINDOWS EXPLORER

    2. Within the Tool Bar select ORGANIZE

    3. From its drop-down window click on, “SELECT ALL”

    4. Scroll down to FAVORITES BAR and press and hold “Ctrl” key while clicking on FAVORITES BAR (this will deselect FAVORITES BAR so that it is not moved).

    5.  Right click on the highlighted area of the favorites selected and select “CUT”.

    6. Go over to the folder you created; right click on folder or right click within the folder; and select “PASTE” from the pop-up window. All your favorites should now be moved to the new folder you created.

    7. IMPORTANT  è    ONE-BY-ONE, and from top to bottom – drag your favorites folders and files from the folder you just created, back to the FAVORITES folder and drop them in. The order, in which you drag each folder and file, will be the order in which they will appear on the FAVORITES BAR within INTERNET EXPLORER. DO NOT DO A GROUP DRAG-AND-DROP or your favorites will be out of order, again.

    8. Now open the FAVORITES BAR in INTERNET EXPLORER and verify that everything is in the order you have created.

    9. If you desire a certain favorites folder or file in a different order you can do this by:

      1. Selecting the “STAR” icon at the top right corner of INTERNET EXPLORER page.

      2. In the drop-down window click on the chevron next to the ADD TO FAVORITES button.

      3. In the drop-down window select “ORGANIZE FAVORITES”.

      4. In the pop-up window place the curser over the item you wish to reorganize; left click and hold; then drag up or down to the location you wish it to appear.    

           

  2. Retain the file you created and the setting you made earlier in this instructions. It will make it that much easier and quicker to periodically complete this task from time to time.

  3. Get in the habit of saving your favorites into folders and keeping them in order. To help in this, create TEMP HOLDING FOLDERS for specific groups as in my example above.

  4. Remember – What’s the point of saving things if you have to hunt around for hours to retrieve it, especially when you will more than likely need it NOW!

  6. * Moved from Insider

jeeped say:

1000 rows of data  want to remove the duplicate rows A, B, C  but sum the numbers that are in Column E

You are looking to change a value based upon a cell's value and another cell's value, retain the value and remove the other entry. This will likely require VBA. Any formula based solution will change when one of the entries is deleted.
Press Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module. Paste the following into the new pane titled something like Book1 - Module1 (Code),

Sub mcrCombineAndScrubDups()
    For Each a In Range("A1", Cells(Rows.Count, "A").End(xlUp))
        For r = 1 To Cells(Rows.Count, "A").End(xlUp).Row - a.Row
            If a = a.Offset(r, 0) And a.Offset(0, 1) = a.Offset(r, 1) And a.Offset(0, 2) = a.Offset(r, 2) Then
                a.Offset(0, 4) = a.Offset(0, 4) + a.Offset(r, 4)
                a.Offset(r, 0).EntireRow.Delete
                r = r - 1
            End If
        Next r
    Next a
End Sub

Press Alt+Q to return to your worksheet. Press Alt+F8 to open the Macros dialog and Run the macro. It may not be the most elegant code, but I have the impression that this is a one-off or rarely needed procedure.

Your workbook will have to be saved as a macro-enabled workbook.

jay freedman say:

The \* charformat switch doesn't change the size of the barcode. What does is to add a \h switch followed by the height in twips (20 twips = 1 point). The aspect ratio of the barcode is fixed, so increasing the height also increases the width and the spacing of the bars.

The default size with no \h switch appears to be about 1700 twips, so to add about 30% to the size you can use the field code

{ displaybarcode "{ MERGEFIELD Barcode }" code39 \h 2200 }
