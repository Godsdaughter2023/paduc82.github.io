This will export the latest date's values from Preview to the workbook "YTD Adherence.xlsx" on the worksheet "Export" starting in cell A1

Sub ExportLatestDateToSeparateSheet()

    'Export is based on the date value in column A

    Dim shtData As Worksheet

    Set shtData = Worksheets("Preview")

    dteFilter = Application.Max(.Range("A:A"))
        With .Range(.Range("A4"), .Cells(.Rows.Count, "C").End(xlUp))
            .AutoFilter Field:=1, Operator:= _
        xlFilterValues, Criteria2:=Array(1, Format(dteFilter, "m/d/yyyy"))
            .SpecialCells(xlCellTypeVisible).Copy _
                    Workbooks("YTD Adherence.xlsx").Worksheets("Export").Range("A1")
            .AutoFilter
        End With
    End With

End Sub

Check after each step.

Internet Explorer 8 starts in "No Add-ons" mode when you use the desktop icon to start Internet Explorer:
http://support.microsoft.com/kb/976915
The Internet Explorer icon is missing from your desktop:
http://support.microsoft.com/kb/945402

4. From Start> Programs> right-click the Internet Explorer icon> Properties>
Shortcut. If the Target is
"C:\Program Files\Internet Explorer\iexplore.exe" -extoff
edit out the  -extoff  (leave the quotes) then OK your way out of
Properties.

There might be more that one IE icon- one at the top of the Start Menu,
another in Programs, etc. If the issue persists, check all IE icons.

Activex Filtering is a feature of IE9 and IE10. You don't have this feature in IE8.

Also, getting IE out of no add-ons mode may resolve this.

Which updates won't install? What's the error message?
