john korchok say:

I've built forms that do this kind of thing, but it gets complicated fast. 

You'll have to use VBA. Unfortunately, Content Controls are not easy to use with VBA. It's simpler to implement a Legacy DropDown Form Field to run a macro that inserts building blocks. Legacy Form Fields require that your document be protected for forms. Depending on how you set up your document, it may be necessary for the macro to unprotect the document, insert the building block, then reprotect for forms.

  1. Exit all Office applications including the OneDrive for Business sync client.
  2. Reset Internet Explorer settings.
  3. Removing all the stored credentials in the credentials manager (Control Panel > User Accounts > Credential Manager > Windows Credentials). (NOTE: This will remove your stored passwords.)
  4. Run regedit as administrator.
  5. From Registry Editor, browse to:
    HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Common\Identity
  6. Delete the Identity folder.
  7. Open an Office application like Word and sign in with your Office 365 account.
  8. Open OneDrive for Business to continue syncing.
