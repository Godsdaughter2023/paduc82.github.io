How can I get the footer from page 3 to show up on the blank page (as page 4)?

suzanne s. barnhill say:
When you insert an Odd Page section break, the even page that is automatically created if the previous section ends on an odd page is totally blank by design; this is in conformity to publishing convention. If you need to print something on the "blank" page (this should be something other than the footer, which should not be printed if the page is otherwise blank), see http://word.mvps.org/FAQs/TblsFldsFms/InsEvnPgEndChap.htm.

How do you edit and add to an existing sway? - Microsoft Community

lindsayalvarez say:

How do you edit and add to an existing sway?

How do you edit and add to an existing sway?

jason (office engineer) say:

Hi Lindsay,

Viewers and authors use the same URL to open a Sway.  Upon opening it, if you are signed in as an author, you will be taken to the edit page; if not, you will be taken to a view-only page.  By default, the person who creates a Sway is the only author on it.  More authors can be added if the owner sends out the special edit link (from the Share dropdown).  When a non-author follows that link, they will be given an option to become an author on that Sway.

Hope this helps!

Jason
