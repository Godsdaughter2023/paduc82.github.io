Unfortunately manually deleting files from the Winsxs Component Store can quickly corrupt the operating system.

tryx3 say:

Just to be cautious for a moment because the term recovery can often be used quite loosely -  

  • I think you are referring to a partition that was several GB big [mine, for example, is 10GB]. 
  • If you referring to something that is anything up to 1GB then that is a completely different thing.

The purpose of the OEM recovery partition is to ease reinstallation of Windows by providing a combined source of

  • Windows, and
  • Hardware drivers for your computer model, and
  • Some [possibly all] of the computer maker's utilities that were pre-installed on your computer when you got it.

You can get these from elsewhere so you have not really lost anything -

  • Windows - you can always make an installation disk [USB/DVD] of the latest Windows version using http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install
  • Hardware drivers - the latest versions of all your hardware drivers will be available somewhere in your computer maker's support website
  • Pre-installed computer maker's utilities - the latest versions of most, if not all, of these utilities will be available somewhere in your computer maker's support website

Notes -

1  Even if you had not deleted that partition, its contents would have been frozen at the date it was written so if you ever used it you would then have to mess about getting the latest Windows, the latest drivers & the latest utilities anyway.

2  Some computer makers allow you to download a copy [an "image"] of that partition so you can use it to create a USB that has all its contents.  Dell provides that but I have not taken advantage of this for my computers because I do not think it would be useful to do so [see Note 1]

3  Having a Windows installation disk [USB/DVD] of your own can be very useful in its own right.  It can be used as a Repair tool as well as an installation tool.  You can, just for example, conduct a Repair install using it.

Additionally, if your current Windows is "activated with a digital licence" [see Settings, Update, Activation] then you do not need a product key.  If you ever need to reinstall Windows,

  • you skip the dialog asking for a product key,
  • you complete reinstallation then go online,
  • Windows will then automatically contact the Microsoft activation servers, find its record of your current activation & reactivate automatically {you do not need to enter any product key at any stage, you do not need to do anything during reactivation}.

So, if I have understood you correctly, you do not have a problem.
