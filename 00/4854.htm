Ah, this is the "other Compare"! Compare and Merge Documents (which you stumbled upon) is a markup operation. What you want is Compare Side by Side, which is now called View Side by Side. Open the two documents, select the View menu, and in the Window group, click View Side by Side.

If you have more than two documents open when you click View Side by Side, Word opens a dialog called Compare Side by Side (oops!) and gives you a list of the documents other than the one that currently has the focus to "Compare Side by Side with." You choose the second document to compare.

If you have two versions of the same document open for comparison, you'll probably want to use the default Synchronous Scrolling (a button in the same group), but if you have two copies of the same document open (using New Window) so that you can view different portions of the document, you'll want to turn that off.

OK since you have reformatted already you should be good for a completely clean install.

One thing that works well for me when I have had issues on some builds is to disable all network adapters (Ethernet and Wireless) in the laptop BIOS before installing Windows 10.  You can re-enable them in BIOS after you have Windows 10 installed.

Boot with a Windows 10 install USB or DVD and choose Repair your computer.  Find the Command prompt.  At Command prompt launch diskpart.

Assuming that your "old laptop" is BIOS-based and not UEFI, at diskpart prompt, type "select disk 0", then "clean" and then exit.  Type exit to return to the various Repair options.  If you are using a Windows 10 install USB or DVD, choose Exit and continue to Windows 10.

Boot to the Windows 10 Setup, Choose Custom install, and install to the unallocated space of your disk drive.

The above will ensure that the Windows 10 install creates all of the needed partitions properly.  Pre-formatting the drive might not set up the desired partitions right.
