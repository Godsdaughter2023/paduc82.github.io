Breaking down the INDEX() function to its syntax reveals your formula in INDEX()'s array form.

  1. Log on as a user other than the user whose profile you are copying files to or      from.

  3. In      Windows Explorer, click Tools, click Folder Options, click the View tab, click Show hidden files and folders, click to clear the Hide protected operating system files check box, and then      click OK.
  4. Locate the C:\Documents and Settings\Old_Username folder, where C is the drive on which Windows XP is installed, and Old_Username is the name of the profile you want to copy user data from.
  5. Press and hold down the CTRL key while you click each file and subfolder in this      folder, except the following files:
    • Ntuser.dat
    • Ntuser.dat.log
    • Ntuser.ini
  6. On the Edit menu, click Copy.
  7. Locate the C:\Documents and Settings\New_Username folder, where C is the drive on which Windows 10 is installed, and New_Username is the name of the user profile that you created in the "Create a New User Profile" section.
  8. On the Edit menu, click Paste.
  9. Log off the computer, and then log on as the new user.

Disclaimer: You must import your e-mail messages and addresses to the new user profile before you delete the old profile.

  1. Sign into the new user profile.
  2. Press Windows key + letter ‘R.’

  4. Select the old profile name and select Remove.
  5. Sign out and Sign back into the new profile.

  1. Go to Start Menu.
  2. Choose Settings.

Hi,

Try this ARRAY formula, see below for how to enter it

=PRODUCT(--MID(SUBSTITUTE(UPPER(A1),"X",""),ROW($A$1:INDEX($A:$A,LEN(SUBSTITUTE(UPPER(A1),"X","")),1)),1))

This is an array formula which must be entered by pressing CTRL+Shift+Enter

around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.
