"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.

  • The PC restarts into the newly installed copy of Windows.

• If you go in Task Manager and look at RAM usage it will show active memory. If you close apps using lots of RAM it will decrease the RAM usage. If you go into system resource monitor then it will break down for you what type of RAM and how much is in use. This is mostly about active memory and cache but there's also hardware reserved RAM.


• If you are seeing large amounts of active memory then this using more RAM for performance isn't the reason. If you leave apps open but computer is left RAM will remain high. If you close the apps then you're RAM will go down. Even if you aren't doing anything and it's idea if an application is open its in RAM.


• If you RAM usage is much higher when no apps are open then you likely have a lot of background processes running. Look in task manager and sort the processes by RAM usage to determine what's going on. Also go to the startup tab and look there. Disable all start up apps that you don't need.
