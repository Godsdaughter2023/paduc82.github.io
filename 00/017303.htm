Microsoft is aware of the ongoing issue with the merged table cells not working correctly in Word2016. As of September 11, a workaround was released and a resolution is tentatively scheduled for October 3. As a workaround, you can uninstall both the latest update for Word 2016: KB3213656and KB4011039. Your merged table cells should return to normal after removing both KBs. For more information about this known issue, you can refer to this article. Follow the steps below on how to uninstall an update on Windows 10:

  1. Close Word 2016.
  2. Select the Start button.
  3. Go to Settings > Update & security > Update history > Uninstall updates.
  4. Look for KB3213656 and KB4011039.
  5. Select the updates and then click Uninstall.
  6. Click Yes or enter the administrator password to proceed with the removal process.
  7. Wait for the process to complete.
