mohammad _ imran say:

Hi RWAP,

 

Follow the methods listed below:

 

Method 1: Use Device Manager to scan for hardware changes

Use Device Manager to scan for hardware changes. After your computer scans for hardware changes, it might recognize the USB device that is connected to the USB port so that you can use the device.

To scan for hardware changes, follow these steps:

1.       Click Start, type device manager in the Start search box and hit Enter.

In Device Manager, click your computer so that it is highlighted.

2.       Click Action, and then click Scan for hardware changes.

3.       Check the USB device to see whether it is working.

4.       Restart the computer.

If scanning for hardware changes fixed the problem then issue will be fixed. If this method did not fix the problem, go to Method 2.

 

Method 2: Disable and re-enable the USB controller

Use Device Manager to disable and re-enable all the USB controllers. This lets the controllers recover the USB port from its unresponsive condition. The USB controllers represent the USB ports in Device Manager.

To disable and re-enable the USB controllers, follow these steps:

 

1.       Click Start, type device management in the Start search box and hit Enter.

2.       Expand Universal Serial Bus controllers.

Note: You might have to scroll down the list to find this item.

3.       Right-click the first USB controller under Universal Serial Bus controllers, and then click Uninstall to remove it.

4.       Repeat step 4 for each USB controller that is listed under Universal Serial Bus controllers.

5.       Restart the computer. After the computer starts, Windows will automatically scan for hardware changes and reinstall all the USB controllers that you uninstalled.

6.       Check the USB device to see whether it is working.

 

Method 3: Delete the USB Upper Filter & Lower Filter entry.

Before we go any further, please first backup the Registry.

 

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, click the following article number to view the article in the Microsoft Knowledge Base:

Back up the registry

http://windows.microsoft.com/en-us/windows7/Back-up-the-registry

 

1.       Click Start, type regedit in the Start search bar and press Enter to open Registry Editor.

2.       Locate the UpperFilters value under the following key in the registry:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{36FC9E60-C465-11CF-8056-444553540000}

3.       On the Edit menu, click Delete, and then click OK.

4.       Locate the LowerFilters value under the same key in the registry:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{36FC9E60-C465-11CF-8056-444553540000}

5.       On the Edit menu, click Delete, and then click OK.

6.       Quit Registry Editor and restart the computer.

 

For more information on updating drivers, see Update drivers: recommended links

 

Hope this helps.

Hi,

The issue usually occurs when your system display card settings are too high for your monitor. Incompatible or corrupted drivers can cause this issue. I understand how difficult it is, when the device doesn't work in the way it should. I appreciate the troubleshooting steps that you've tried so far. We can try a few steps and check if we can fix this.

Method 1: Change the Screen refresh rate and check:

1. Boot into Safe mode.

2. Press Windows key + I together to open Settings.

3. Click on System > Display > Advanced display settings.

4. Click on Display adapter properties.

5. Go to Monitor tab, and select 60 Hertz as Screen refresh rate. 

6 Click on Apply and OK.

7. Restart the system and check.

Method 2: Change the display resolution and check.

1. Boot into Safe mode.

2. Press Windows key + I together to open Settings.

3. Click on System > Display.

4. Under Resolution, decrease the resolution using the drop down.

5. Restart the system and check.

Method 3: Uninstall the devices showing under Display adapters and Monitors and check.

1. Boot into Safe mode.

2. Press Windows key + R together to open Run.

3. Type devmgmt.msc and press Enter.

4. Expand Display adapters and Monitors sections.

5. Delete each of the devices in these two sections by selecting a device and by pressing Delete key on your keyboard.

6. Restart the system and check.

You can check this article on Start your PC in safe mode in Windows 10. 

Let us know how it goes.
