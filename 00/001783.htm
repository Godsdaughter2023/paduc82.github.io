stephen boots say:

When you download Windows Defender Offline, you select the architecture for the PC you are going to scan. So, on a machine running 32 bit Windows, if you select the 64 bit version of Windows Defender Offline, the Wizard will create the bootable media for your target architecture -- x64.

On an XP machine, which would be a x86 or 32 bit operating system, you can create WDO media for a Windows 7 machine running 32 or 64 bit -- you need to know which version of Windows 7 is on that target machine.

 

This is contrary to what downloading 32 bit vs. 64 bit means. The WDO download that runs to create the boot media can run on either architecture. The critical part is that the architecture of the bootable media created matches the OS of the PC to be scanned.

 

What infection are your trying to resolve? What exactly is the "blue screen" text or error message? I suspect that you may actually have a hardware problem with the Windows 7 PC.
