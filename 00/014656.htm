Alarm (Reminder) in excel - Microsoft Community

m.y.d say:

Alarm (Reminder) in excel

Dear all,

Is there a way to add a reminder/alarm in Microsoft Excel for Windows Desktop?

for ex: setting a specific date and time as an alarm for specific selected cells in a sheet 

Thanks.

wyman w say:

Here is a VBA solution.   Alt+F11 > insert Module > Paste the following code. Press F5 to start the code, Ctrl+Break to stop.

Public Sub RunTimer()
    runtime = Now() + TimeValue("00:00:01")
    Application.OnTime runtime, procedure:="refreshTbl"
End Sub

Sub refreshTbl()
    If Now() = Range("A1").Value Then
        MsgBox ("reminder")    'reminder action
        Exit Sub
    End If
    RunTimer
End Sub

In the above code, I refer the reminding time to be value in A1, you can change to whatever cell you want by modifying the code

I have also set a reminder action to be a message box, you can replace the code  MsgBox ("reminder")

The value in A1 should be in format of something like "24/1/2015  16:52:00"
