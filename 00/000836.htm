Go to References, Insert Caption.

Create a New Label (such as "doc") and check the box next to "Exclude label from caption."

In your table, put a tab space between the number and the doc title (use one table cell) to keep them together.

As you add doc titles to your table, just copy and paste the seq number (field) plus the tab space, and then update (F9).

Will there be only one matching copy, or might there be multiple matches with different names ?

Are all the files in the same folder, or could they be in subfolders ?

If only one match and all at the same level:

Sub tester()
    Dim c As Range, tmp, fldr
   
    fldr = ActiveSheet.Range("A1").Value
    fldr = fldr & IIf(fldr Like "*\", "", "\") 'ensure trailing slash
    If Len(Dir(fldr, vbDirectory)) = 0 Then
        MsgBox "Network folder not found!"
        Exit Sub
    End If
   
    For Each c In ActiveSheet.Range("A3:A100").Cells
        tmp = Trim(c.Value)
        If Len(tmp) > 0 Then
            With c.Offset(0, 4)
                .Value = Dir(fldr & tmp & "*.zip")
                If .Value = "" Then .Value = "N"
            End If
        End If
    Next c

End Sub
