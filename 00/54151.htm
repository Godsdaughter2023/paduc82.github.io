  1. Locate the printer entry (printer XYZ on http://yourcupserver in "printers and faxes" )
  2.    - right click and select printer properties
  3.      - select the ports tab
  4.        - the port being used should already be checked. Click on "configure port"
  5.         - Select "use the specified user account", fill in the details and click OK.

You're done!

stevenjayrocks say:

I have this problem as well. The only way to use the "Mail to" feature via File Explorer is to install a different email program outside of the Windows 8 Mail App. So, in this instance, you would have to install Windows Essentials which includes Windows Live Mail 2012 -- OR -- Microsoft Office Outlook.

 

If you wish to use the Windows 8 Mail App to send photos, then use must browse your photos using the "Photos App" that comes with Windows 8.

 

1. Browse to the folder that contains your photos that you wish to share using the Photos App.

2. Zoom out to see all the photos by either holding down the Ctrl button and scrolling the mouse wheel down, clicking on the "-" sign in the lower right corner of your screen, or pinching the screen when viewing the photos in the folder you're browsing.

3. Hold down Ctrl, while holding down Ctrl, click each photo you wish to share using the mouse. Not sure how this would work with touch only.

4. When happy with your selection, swipe from the right or mouse over to the lower/upper right corner of your screen and click/tap share.

5. Here you will be able to choose Mail to send your photos.

 

Looks more complicated than it's worth, but in order to use the Mail App, this is how you do it. Otherwise, the other method is it for now.
