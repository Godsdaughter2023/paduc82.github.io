ossiemac say:

You could use either of the following 2 examples. The second is just an alternative method of writing a range with the square brackets and no double quotes.

Note that the second cell of the range is always B1 for the examples.

Sub ClearCell()
     If Range("B3") = "False" Then
        Range(Cells(3, "B").Offset(-1, 0), Cells(1, "B")).ClearContents
     End If
End Sub


Sub ClearCell_2()
     If [B3] = "False" Then
        Range([B3].Offset(-1, 0), [B1]).ClearContents
     End If
End Sub
