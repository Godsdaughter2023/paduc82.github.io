marilyn-- say:  Hello GP1111,

Thank you for posting to the Microsoft Answers Community Forum.

How to create a sync partnership:
Open Windows Explorer,
right-clicking on the drive letter assigned to the USB drive
Click on Properties from the drop down menu,
Select Customize
Set to Documents or All Items whichever  you prefer.

If your device does not appear in the list of available sync partnerships and you have already tried connecting it to your computer again and turning it off and on, Windows might not detect the device or it might not work with Sync Center.

Before you ask the manufacturer if the device supports this version of Windows, try installing any software that came with your device. This often includes any driver that allows Windows to recognize your device. This software might also include sync capabilities. You might have to sync your device using this program instead of Sync Center. Try opening this program and checking if it includes sync capabilities.

If no software came with the device, you might be able to download a compatible program from the manufacturer's website. Not all sync software works with Sync Center. If it doesn't, you will have to sync your device using software from the manufacturer, rather than Sync Center.

Note
If you are unable to get a device to work with Sync Center, it may help to check if your device is listed in Device Manager. Even if Windows recognizes your device, however, it might not be compatible with Sync Center.
To check this, see Device Manager. 
You must be logged on as an administrator to perform these steps.
If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

Another option:
You could also sync to a USB Flash Drive by installing SyncToy from Microsoft.
