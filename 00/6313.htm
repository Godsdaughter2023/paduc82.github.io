dirk goldgar say:

I have a main form with one subform.  The subform has multiple records, within those records are multiple calculated fields (premiums, rates etc).  The fields are calculated based on values placed in both the sub & main forms.  If a value changes in the main form, how do I get all the affected fields & records updated in the subform?

 


If the values are calculated by reference to controls on the form and subform, and not values in the underlying tables, then you can probably just Recalc the subform:

    Me.sfYourSubform.Form.Recalc

However, in the more likely case that the subform calculations involve data in the the tables, and you've changed a value that ought to be stored in a table, then you have to do two things: (1) force the main form's record to be saved, and (2) force the subform to refresh (or maybe requery).

    Me.Dirty = False

    Me.sfYourSubform.Form.Refresh
'       or    Me.sfYourSubform.Form.Requery
