suzanne s. barnhill say:

The keyboard shortcuts Ctrl+Home and Ctrl+End are assigned to StartOfDocument and EndOfDocument, and this is what they do. If your document contains no text at all, then the start and end will be at the same place. But these shortcuts do not ever take you to the top and bottom of a page unless the document is exactly a page long.

There are other commands that will do other things. PageUp and PageDown actually move the document by one screen, but Ctrl+PageUp (BrowsePrev) and Ctrl+PageDown (BrowseNext) will navigate backward and forward by one page when the browse object is set to Browse by Page.

Check the HOSTS File

The HOSTS file is a text file that you can edit with any text editor (such as Notepad). If your network uses HOSTS files for host name resolution and you cannot connect to the other computer using its host name, there may be an invalid entry in your HOSTS file. Search your LMHOSTS file for the NetBIOS name of the other computer, verify that there is only one entry per NetBIOS name, and then verify that the entry for the NetBIOS name of the other computer is correct.
