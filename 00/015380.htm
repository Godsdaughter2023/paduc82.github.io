    I have a page which has three columns. However, in addition, I would like to add a header (or even just a title) that is only one column across, above the 3 columns below. Is this possible on the same page? I also only want the header to appear on that page. Any help appreciated.
    pamcaswell say: I'm guessing that you want a heading over the three columns below it.  You can do this with section breaks.

     **Type the heading and its return in a one-column section of the document.  Click  Page Layout tab > Page Setup group > Breaks > Continuous. With the cursor below the break, setup up the columns as you want them.  When you want a new heading that must span the columns, insert another section break (use a Next Page section break if you want the heading  to start a new page),  set the number of columns to 1 and  repeat from **.
