a. user say:

Some extra details ...

Windows 10 (Home or Pro) is available in 32-bit (x86) and 64-bit (x64) versions. If you have 32-bit hardware you must install the 32-bit version. If you have 64-bit hardware you can install the 32-bit or the 64-bit version of Windows, but it makes sense to install the 64-bit version.

The 64-bit version of Windows 10 is backwards compatible with 32-bit software, however 64-bit hardware is not backwards compatible with all 32-bit software.

Program Files (x86) is where Windows installs your 32-bit software. Program Files is where Windows installs your 64-bit software. You should not change this.

The essential difference between 32-bit and 64-bit software is the amount of memory (RAM) it can use. 32-bit software can use up to 4 GB of RAM, which is 2 raised to the 32nd power. 64-bit software can use a ridiculous amount of RAM (2 raised to the 64th power.)

To maintain backwards compatibility there are few other differences between 32-bit and 64-bit Windows, although what differences there are can be significant.
