gord dibben say:

Whichever column holds the active cell.

 

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Application.Intersect(Target, _
            Columns(ActiveCell.Column)) Is Nothing Then
        On Error GoTo Endit
        Application.EnableEvents = False
        With Target
            If .Value <> "" Then
                .Offset(0, 1).Value = Now
            End If
        End With
    End If
Endit:
    Application.EnableEvents = True
End Sub

rich007a say:

Hi.

Try this.  Right-click on the tab of the Worksheet and View Code.

Paste this code:

Private Sub Worksheet_Change(ByVal Target As Range)
    If Not Intersect(Target, Range("A:A")) Is Nothing Then
        If Target.Value <> "" Then Target.Offset(0, 1).Value = Now
    End If
End Sub

 

This will put the current date and time in column B whenever any cell in column A is changed (although I'm not sure if the event will fire when cells are changed by your barcode scanner....).

The A:A in the above code states that only when any of all of column A is changed then the code runs.

The Offset(0,1) means that the cell 1 column to the right of the changed cell will have the date stamp entered in it.

 

Does that work for you?
