username#1 say:

I fixed "Automatic repair loop" windows 10 upgrade.

I must mention the need to recover your files from your loop drive before you try doing anything. What I did was I removed the drive and plugged it into a working PC; copy any filed to either the PC's hard drive or USB stick/external any files like the desktop, downloads, pictures and documents, etc. those files are probably found under windows/users or windows.old/users folders, look around until you find your files that you want to save...you will lose apps or programs.

My system had originally win7, toshiba satellite, i5, 4gb, and I did a clean install of Win 10 (I though it was a "clean" install, hmmm), then a couple days ago I was notified that there was another big win 10 upgrade and the computer needed to restart...long story short the "automatic repair" loop started. I know a little bit about computers so I tried every little trick that I could think of, reading all the complaints and suggestions online, etc... Nothing helped. I mean nothing (bios, cmd, reinstall, chkdisk., etc..nothing helped). The loop always came back, even after reinstalling win7 there was still a part of win 10 hiding in the background, then after formatting the drive to reinstall win7 (which worked, the loop was gone) but then upgrading to win 10 again brought me back to the loop, aghhh!!!

Sorry if this is taking a while to get to the point but I was frustrated with this issue and I wasted two days on trying to figure this out.

While I was trying to figure out what was happening, everything pointed to the new system drive X:. Why did the win10 installation use this X: drive? Why not use the system drive that was created when I installed win7 instead? Nothing made sense.

So my focus shifted to getting rid of this infamous X: drive which win 10 used during installation and/or as system drive.  

Let's just say I tried to reinstall win 10 multiple times with win7, win8 and without(can't do it without, need key)...unless you have a product Key for win 10 you are stuck having to install win 7 or 8 first. So you are stuck with this system drive created by win 7 or 8 which the big win 10 upgrade for some reason can't use so it has to create this mysterious X drive...hmmm.... what to to...???

Here is what you need:

NEED NOW: your old win 7 or 8 installation disk, or if you have a manufacturer recovery disk that would work too, borrow a friends, I don't know, do something.

My point here is this...you are going to run into this problem of this hyper loop... if you are upgrading or updating.... unless you have a supper clean install of win 10 on a brand new hard drive.... sooner or later.... this is how you prevent this from happening.... i hope.

0. WARNING. There is no going back after this point you will lose all your data, everything, unless you have backed it up just FYI.

1 Insert DVD or USB containing windows 7 or 8 OS.

1.2 It should boot now.

2. Choose advance/custom install once you have the option, don't select which drive to install just yet.

3. Pay attention here, Delete your system drives at this point, they are very small probably around 100MB or so, one of them is the win 10 X: drive, good riddance.

5.1 Select the unallocated space, you will see an option for "new" click new and it will ask how much of this you want to allocate, default being everything so that's what we will do but if you want to make a partition you can allocate as much as you want as long as its large enough to receive the operating system and apps... for a 1TB (1000GB) hard drive I usually split it in half. but that is not important here because as you will see we are going to have to do this process one more time so you can do it now or later if you want or not, whatever.

5.2 So once you select apply you should get a message pop up that the operating system will create a system partition bla bla bla, hit yes... you will see two new drives being created one is really small(couple hundred MB) and one will be your drive C: basically, this is where we will install windows, 7 or 8.

5.2.1 Format  this larger drive that will receive windows just in case any windows 10 parts are still hiding ha-ha, it only takes a few seconds.

3. If it asks for Product ID click I don't have one (this option is only available because it already recognized your other operating system and its free to upgrade anyways).

4. You will have to go through the steps (see instructions on how to install win 7/8 above) on how to delete the system and allocated drive space and then allocating a new one form the allocated space again.
