Problem in Outlook 2016 mail single-line layout and conditional - Microsoft Community

pda2010 say:

Problem in Outlook 2016 mail single-line layout and conditional formatting

If you change the font size for all rows in Outlook 2016 (Mail folder), the "normal" font size setting in conditional formatting does not reflect the font size you've set but always uses 8pt.

This bug is very easy to replicate:

1) When viewing your mail in Outlook 2016, ensure that you're viewing in "single-line layout" (so that each email takes one line).  You can do that by widening the width of the mail list or setting "Always use single-line layout" in "Advanced View Settings".

2)  In "Advanced View Settings", change the font size of the "Row Font" to something other than 8pt.  Click "OK"

All read mail in the mailbox will reflect the new font size.  However, the font size of all unread messages will not change.  This is because unread messages are made bold as a result of a conditional formatting rule, which uses a font size of "Normal".  "Normal" should use the font size for the row set by the user but, because of this bug, always uses 8pt.  Note that in "Compact Layout" (i.e. each email takes multiple lines), it works correctly.

Two questions:

1) Is Microsoft aware of this bug in Outlook 2016?  If you have any difficulty replicating it, I can provide further instructions.

2) Will Microsoft issue a patch to fix this bug?

Thanks!
