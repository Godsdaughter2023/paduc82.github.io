VBA code to highlight an entire row based on multiple cell values - Microsoft Community

mooshak say:

VBA code to highlight an entire row based on multiple cell values

I'm looking for a macro formula that would allow me to automatically highlight entire rows based on the values in multiple cells.

Such as if column D = Library AND Column L is >= 28, then highlight the entire row.

gord dibben say:

Sub ColorRows_2Params()
Dim iLastRow As Long
Dim i As Long
    iLastRow = Cells(Rows.Count, "D").End(xlUp).Row
    For i = iLastRow To 1 Step -1
        If Cells(i, "D").Value = "Library" And _
            Cells(i, "L").Value >= 28 Then
            Rows(i).Interior.ColorIndex = 4
        End If
    Next i
End Sub

lordalukard say:
Dear Yom

In this case you will need to set up rules https://support.office.com/en-us/article/use-in... and disable Stop processing new rules.

So let's say you have person A and B. You will need to create a rule that when A replies email should be moved to inbox and should go to A folder. The same for B person. If you need only their replies separately on different folders, you do not need other rules, so you may select stop processing more rules.

All messages that are being received are checked against all the rules that you have configured. This checking begins with the first rule in the list and ends with the last rule. If a rule applies to a message and this rule has the "stop processing more rules" action enabled, then that message will not be checked against any other rules and Outlook will skip to the next received message that will be checked against all rules again starting with the first rule.

I hope this make sense, but if you will still need explanation, please let me know.
