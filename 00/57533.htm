My internet comes from a dial up and am look for a way to share this internet with a computer running xp. that is hooked up through the Ethernet chord.

steve winograd say: Enable Internet Connection Sharing on the dial-up connection:

1. Press WindowsKey+R , type ncpa.cpl in the box, and click OK .
2. Right-click the dial-up connection and click Properties > Sharing .
3. Put a check mark in the Internet Connection Sharing box and click OK .

msq2 say: I have tried this method and the xp computer has not been able to connect to the internet thru this method.

steve winograd say: Make sure that you're enabling ICS on the dial-up connection.  If it asks you to select a private network connection, select the Local Area Network Connection, not the Wireless Network Connection.

Check the TCP/IP properties of the Local Area Network connection on both computers. 

Windows 7 should show:

   IPv4 Address: 192.168.137.1
   IPv4 Subnet Mask: 255.255.255.0
   IPv4 Default Gateway: blank
   IPv4 DNS Server: blank

Windows XP should show:

   IP Address: 192.168.137.x,  1 < x < 255
   Subnet Mask: 255.255.255.0
   Default Gateway: 192.168.137.1
   DHCP Server: 192.168.137.1
   DNS Server: 192.168.137.1

If those values are right, open a command prompt window on XP and type these commands.  Each one should get four replies:

   ping 192.168.137.1
   ping 8.8.8.8
   ping yahoo.com

Please report anything that doesn't work as expected.
