5.  If you want to see your SkyDrive account, you'll need to go to the "Open" folder and add your SkyDrive account with the "Add" button.  You just do this one time.

Sub PrintHex()
  Dim s As String

What type of controls are you using in the document?  Legacy Formfields or Content Controls?

With Legacy FormFields, you could use code such as

With ActiveDocument
    .Unprotect
    If .FormFields("check1").CheckBox.Value = False Then
        .Bookmarks("check1text").Range.Font.Hidden = True

    Else

        .Bookmarks("check1text").Range.Font.Hidden = False

    End If
    .Protect wdAllowOnlyFormFields, noreset
End With

in the validation button, where the text associated with the checkbox (and probably the checkbox itself, would be enclosed in a bookmark with the suggest name of check1text.

For it to function correctly, the display (and more particularly, printing) of Hidden text would need to the turned off.  Having the display of hidden text turned on, does however allow the text to be reinstated by checking the box, if that is necessary.

If you want to insert a general background picture that is rectangle, then draw a 'donut' on the blank template that has the circular guidlines.
