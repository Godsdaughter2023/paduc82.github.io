tom_bpc say:

Toni,

This is fairly straightforward for linked subprojects as (I think) you describe.  You have to insert the task in the appropriate "source project" that the "subproject" (which is more or less a shell) links to.  Below  is a quick bit of vba code to get you started.  It inserts a single task at the end of each of the connected projects, named according to your initial post.  You can add more parameters to the "Add" methods below - like exactly where to insert the new task, and you can add more tasks as needed.  You'll also have to add more code if you need to include durations, logic, custom fields, etc.  No doubt you'll need to do some debugging.

I'm curious if your company has used linked master/sub-project structures for very long.  There seems to be a general consensus among the experts in this community against them - it seems extremely difficult (or impossible) to prevent eventual corruption of the links, so the whole exercise is more trouble than its worth.


Sub AddSubPTasks()

    Dim SubProj1 As Subproject
    Dim SrcProj1 As Project
    Dim casenumber As Long
    
    'Use input boxes to collect user data
    casenumber = InputBox("Enter Case Number:")
    
    'Open each subproject and connect to the source project
    For Each SubProj1 In ActiveProject.Subprojects
            Set SrcProj1 = SubProj1.SourceProject
            'Add specific tasks to the source project
            Select Case SrcProj1.Name
                Case "Sales.mpp"
                    SrcProj1.Tasks.Add (casenumber & " Sales")
                    'Add other tasks as needed
                Case "Architectural.mpp"
                    SrcProj1.Tasks.Add (casenumber & " A-Drawing")
                Case "Manufacturing.mpp"
                    SrcProj1.Tasks.Add (casenumber & " Manufacturing")
                Case "Foundation.mpp"
                    SrcProj1.Tasks.Add (casenumber & " Foundation")
                Case "Construction.mpp"
                    SrcProj1.Tasks.Add (casenumber & " Construction")
            End Select
    Next SubProj1
End Sub
