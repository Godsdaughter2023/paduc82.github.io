cristina a. say: Hi Cathy,

I am an independent advisor, I would like to help you with my experience in Microsoft products.

The workaround is to export the secondary calendar and import it into your main calendar. This process works well for calendars you create, but you should verify none of the calendars are internet calendars or published calendars. Removing shared calendars will impact updates.

You can verify if you have any of these calendar types by going to File | Data Management. There are tabs for each of these items. Even though my travel calendar is an internet calendar, I’m comfortable removing it after the merge because I don’t anticipate any changes. In my case, I could always resubscribe to it.

To Export an Outlook Calendar

From the File menu, select Import and Export…
In the Import and Export Wizard, select Export to a file.
Click Next
In the Export to a File panel, select Comma Separated Values (DOS).
Click Next.
In the Select folder to export from section, select your secondary calendar file. It may show in a different area like Internet Calendars.
Click Next.
Note the exported file name and path.
Click Next.
Because we’ll be importing this file back into Outlook, you can skip Map Custom fields….
Click Finish.
Provide a data range for the exported calendar.
Click OK.

The above process exported your secondary calendar to a CSV file. The remaining part is to import this file into your main calendar. As you might expect the process is similar.

To Import a file into your main Outlook Calendar,

From the File menu, select Import and Export…
In the Import and Export Wizard, select Import from another program or file.
Click Next.
In the Select file type to import from panel, select Comma Separated Values (DOS).
Click Next.
In the File to import: section, make sure the CSV file you just exported shows. If not, browse to the file.
In the Options: section, select the radio button that suits you needs.
Click Next.
Select the Destination for this file. (In my case, I’m importing this file into the Calendar that is in my Personal Folders.)
Click Next.
Confirm the listed action and click Finish.
Review your main calendar to make sure it shows entries from both.

The last step is to delete your secondary calendar. If you’re not comfortable with deleting it, uncheck it so it doesn’t display in Microsoft Outlook.

I hope this works for you, if you need more help, just let me know.
