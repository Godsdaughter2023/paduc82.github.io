cmdrkeene say:

Your OneDrive folder is already local if you want it to be.  That is, OneDrive already keeps a real solid copy on your drive, and also copies them up to the OneDrive server.  So you already have both an offline and an online copy, without any work required.

But if you really want two local copies, one in the OneDrive folder and another somewhere else, you could accomplish this a few ways.

1) A little utility like Microsoft SyncToy can be set to do exactly what you want

2) You could do it without software by writting a batch file (which in this case would be a single command) to tell it to copy files to another location, skipping files that are already copied.

Both of these can be scheduled to occur with Windows Task Scheduler, and you can make it do it every night at midnight, or every 10 minutes, whatever you prefer.

a. user say:

DaveM121 says an unexpected shut down won’t damage the hard drive.

On the contrary, an unexpected shut down can absolutely damage your hard drive. That's why people (and businesses) buy a UPS (uninterruptible power supply.)

When your computer suddenly loses power, all work in process stops. Unsaved data is lost and your computer is left in an unstable state.

Running the CHKDSK command after an unexpected power cut - if your computer is still able to boot - will identify and try to repair damage to the file system. This may or may not succeed. Even if it does succeed, any unsaved data is permanently lost.

Windows has changed enormously since Vista. It's a mistake to think, "This is the way it was in Vista so it must also be true for Windows 10."
