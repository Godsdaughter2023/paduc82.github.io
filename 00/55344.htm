Winload.exe is the operating system boot loader, that is to say, the files needed to start Windows. If those are missing or corrupt, the computer won't boot.

How to Remove Blue Double Arrows and Stop Automatic Compression of - Microsoft Community

doctorcoconut say:

How to Remove Blue Double Arrows and Stop Automatic Compression of Files and Folders.

I'm posting this as an answer in the questions section because I believe that more people will see it there than in the discussions section.

Recently I noticed that my icons on my desktop had a pair of blue arrows in the top right corner. This was also true for my documents folder. Any new files that I created had the blue arrows.

I found out that it was due to my files being compressed after looking through other forum posts of people with similar issues. The answers provided didn't actually fix the problem, so I figured it out myself and brought it to attention while I was chatting with Microsoft support.

To resolve this:

>Open file explorer,

>Right-click on the affected directory (desktop/documents in my case),

>Go to "Properties,"

>"Advanced,"

>Uncheck "Compress contents to save disk space,"

>"Ok,"

>"Apply."

The files will begin uncompressing, and this will prevent new files from becoming compressed automatically. 

This may be caused by an update, as this exact issue was present across three different devices of mine.

Hi Phil 1953:

Are you still getting an error 80248015 today when you run Windows Update?

There was a known issue where many Windows 7 users who ran a Windows Update on 04-Dec-2017 saw an error 80248015 with the message "Windows Update cannot currently check for updates, because the service is not running. You may need to restart your computer".  I don't know if Vista SP2 was also affected but Microsoft has now fixed the problem - at least for Win 7 users.  See Martin Brinkmann's article Getting Windows 7 Update Error 80248015? You Are Not Alone on ghack.net as well as Gunter Born's recently updated blog entry Windows 7: Error 0x80248015 in Update Search (Dec. 4, 2017) for more information.

If you see another error 80248015 there is one possible workaround I know of that works for Windows 7 but it requires users to temporarily disable the setting for Microsoft Update (Windows Update | Change settings | Microsoft Update | Give me updates for Microsoft products and check for new optional Microsoft software when I update Windows), and once that setting is disabled the option to receive Microsoft Updates is completely removed and can be very difficult to get back again.
