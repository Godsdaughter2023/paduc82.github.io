barb bowman say:

Got an official explanation for you:

"

Actually what customers are seeing is a new windows features called adaptive hibernation.  It’s been turned on for older chipsets for a while (SP3, S3), but only recently got turned on for Skylake chipsets (SP4, Book).

 

https://msdn.microsoft.com/en-us/library/windows/hardware/mt732711(v=vs.85).aspx

 

What will happen is that the system will learn times when you’re likely to have stopped using your device (for example, if you generally stop using the device around 5 pm, and then it almost always ends up going into hibernation afterwards).  During those times, Windows will try to be more aggressive about hibernating to help conserve battery.

 

If someone doesn’t want this behavior, turning it off should be as easy as going into power options and resetting the hibernate time to a different value.
