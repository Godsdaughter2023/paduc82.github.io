Powerpoint - using search and replace to change font/colour - Microsoft Community

apple_peartree say:

Powerpoint - using search and replace to change font/colour

Is there a way in powerpoint that  I can search all the texts on different slides and replace a particular word in all of them with a different colour? For example if I've already written a presenation on cats, I now want to go back and every time I wrote the word cats I want it to appear in bright red colours.

Any ideas would be great.

john sr wilson say:

You cannot use the slide master to change the colour of a specific word as the MSFT person suggests.

You can use code to do this:

If you do not know what to do with the code try googling for USE VBA code in POWERPOINT.

Sub toRed()
    Dim osld As Slide
    Dim oshp As Shape
    Dim W As Long
    Const strWORD As String = "cats"    'change the word to suit
    For Each osld In ActivePresentation.Slides
        For Each oshp In osld.Shapes
            If oshp.HasTextFrame Then
                For W = 1 To oshp.TextFrame.TextRange.Words.Count
                    If UCase(Trim(oshp.TextFrame.TextRange.Words(W))) = UCase(strWORD) Then _
                       oshp.TextFrame.TextRange.Words(W).Font.Color.RGB = vbRed
                Next
            End If
        Next
    Next
End Sub

john sr wilson say:

Glad it sparked an interest!

I kept the code intentionally easy(ish) to follow.

Try this for a slightly more complex answer.

Sub To_Red2()
'NOTE Does not look in Tables, Smart art etc
Dim oPres As Presentation
Dim oSld As Slide
Dim oShp As Shape
Dim Word_to_Change As String
Dim b_Whole As Boolean
Dim oTR As TextRange
Dim oTR_TEMP As TextRange
Word_to_Change = InputBox("Enter word to search for")
If MsgBox("Look for whole words only?", vbYesNo) = vbYes Then b_Whole = True
For Each oPres In Application.Presentations
     For Each oSld In oPres.Slides
        For Each oShp In oSld.Shapes
            If oShp.HasTextFrame Then
        If oShp.TextFrame.HasText Then
            Set oTR = oShp.TextFrame.TextRange
            Set oTR_TEMP = oTR.Find(FindWhat:=Word_to_Change, MatchCase:=False, WholeWords:=b_Whole)
            If Not oTR_TEMP Is Nothing Then oTR_TEMP.Font.Color.RGB = vbRed
           Do While Not oTR_TEMP Is Nothing
           Set oTR_TEMP = oTR.Find(FindWhat:=Word_to_Change, _
           After:=oTR_TEMP.Start + oTR_TEMP.Length, MatchCase:=False, WholeWords:=b_Whole)
           If Not oTR_TEMP Is Nothing Then oTR_TEMP.Font.Color.RGB = vbRed
           Loop
       End If
    End If
        Next oShp
    Next oSld
Next oPres
End Sub
