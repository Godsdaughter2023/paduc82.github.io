walsterq say:

How do I select multiple objects in Word 2010?

I was having a hard time selecting multiple objects of various types in Word 2010. I believe the reason is that this is simply not possible when the file is in the native XML format (ie, .docx).

I really do not like using the drawing canvas, so after reading various posts, I found the solution:

First, save the document as Word 97-2003 (*.doc).

Quit word and open the .doc document again.

Make sure the Layout for all the object you want to select is the same and not inline (eg, infront of text, not "in table cell", allow overlap, etc). You can do this more quickly by changing the Layout properties of one object, then clicking on other the objects and pressing F4. You can do this step when the document is .docx or .doc.

Now you can draw the selection box around all the objects you want! (Home, Select, Selec Ojects).

Some features not existing in standard release can be used in the targeted release. These features are not only including Office client features, but also including some Office 365 features.
