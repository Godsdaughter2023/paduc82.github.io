jllatham say:

See this page, pay particular note to the comments after the code.

http://www.cpearson.com/excel/BlinkingText.aspx

See this page for instructions on placing code into a Worksheet module, into the ThisWorkbook object, and into normal code modules since you'll have to do all 3:

http://www.contextures.com/xlvba01.html

You will also want some code associated with the sheet that the A10 cell is on to start and stop the blinking.

This code goes into the code module for that sheet - right-click on the sheet's name tab and choose [View Code] and copy this code and paste it into the module:

Private Sub Worksheet_Change(ByVal Target As Range)
  If ActiveSheet.Range("A10") = 1 Then
    StartBlink
  Else
    StopBlink
  End If
End Sub


 

You will need the Workbook_BeforeClose() event handler from the Pearson web site also.

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    StopBlink
End Sub

 

Finally, you need this modified code from Chip Pearson's site modified to work on your single sheet and put it into a standard code module.

Public RunWhen As Double

Sub StartBlink()
    With ActiveSheet.Range("A10").Font
        If .ColorIndex = 3 Then ' Red Text
            .ColorIndex = 2 ' White Text
        Else
            .ColorIndex = 3 ' Red Text
        End If
    End With
    RunWhen = Now + TimeSerial(0, 0, 1)
    Application.OnTime RunWhen, "'" & ThisWorkbook.Name & "'!StartBlink", , True
End Sub

Sub StopBlink()
    ActiveSheet.Range("A10").Font.ColorIndex = _
        xlColorIndexAutomatic

    On Error Resume Next
    Application.OnTime RunWhen, "'" & ThisWorkbook.Name & "'!StartBlink", , False
    If Err <> 0 then
       Err.Clear
    End If
    On Error Goto 0
End Sub

Run-time error '1004':

Insert method of Range class failed

 

The macro is:

Sub a_row_insert()
'
'
' a_row_insert Macro
'

'
    Selection.EntireRow.Insert
End Sub

 

How can I fix this? I need to make sure that only certain cells can be edited in the worksheet so that users don't inadvertantly delete or change anthing.

jllatham say:

Write your code like this:

 

ActiveSheet.Unprotect Password:="yourActualPassword"

Selection.EntireRow.Insert

ActiveSheet.Protect Password:="yourActualPassword"

 

Naturally, replace "yourActualPassword" with your actual password.  If you didn't actually assign a password, then use:

ActiveSheet.Unprotect

Selection.EntireRow.Insert

ActiveSheet.Protect
