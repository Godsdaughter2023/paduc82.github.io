Assuming that this is about a PST file associated with a POP3 account and that the "Personal Folders" data file is not associated with any other email account then without knowing anything else

#1 - Reality is that Outlook doesn't care what you use - it will utilize the data file that has been associated with the email account

#2 - Outlook wouldn't have created a PST file with the name <Personal Folders – Duplicate> - that's something that would have been done manually

#3 - If the data file named <Personal Folders> is no longer being used by anything - just delete it from the profile as long as you're sure there is nothing left in there that you want. Even then, deleting the PST file from the profile does not delete the file from your system should you ever decide that you need to go back to it for some reason

#4 - Just rename the <Personal Folders - Duplicate> to <Personal Folders> (or any other name - makes no difference)
