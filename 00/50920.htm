Fact is,  almost every game released after 2003 runs as well on Win10 as they do on earlier versions of Windows.

Games older than this very often also run well on Win10, but will often have issues that require special attention.

However, there are two stumbling blocks on Win10, and neither have to do with compatiblity.

The first one (Direct Play - below) is easy to fix from within Windows.

The second problem only affects some (GTA-SA not included) disk based games. DRM Blocking.

This issue has nothing to do with compatibility. This is a problem created by MS.  Fortunately it is also easy to fix (for most games),

a. user say:

Hi Spencer,

As per the query description, if you have already signed in to the computer using the Hotmail account, you’re automatically signed in to Mail, and will see your email from the Microsoft account. (Hotmail account in this case)

Are you using a different password to sign in to the computer using that Hotmail account? (As compared to the password that you use to sign-in to the account on the internet browser)

A Microsoft account is an email address and password that you use to sign in to Windows. It's free and easy to set up, and you can do so with any email address you choose, or get a new email address. (For example, you can use an Outlook.com, Gmail, or Yahoo! address for your Microsoft account.). If you have changed the password for the Hotmail account (that can only be done online using outlook mail on the web browser), then, the password will automatically sync with the Hotmail account used on Mail app.

Refer to the link to get the details on Microsoft account:

http://windows.microsoft.com/en-ca/windows-8/microsoft-account-tutorial

If you have just upgraded to Windows 8.1 and then reinstalled Mail app from Store, it could have happened that the Mail app is not installed properly. I would suggest you to uninstall Mail app, reinstall it from Store and then add the Hotmail (Outlook) account on it.

Follow the steps mentioned below:

Step 1:

Refer to the link to uninstall and reinstall the app from “Your apps” section in Store. Right Click on the Mail app on the Start screen and select Uninstall;

http://windows.microsoft.com/en-us/windows-8/what-troubleshoot-problems-app

Note: When you uninstall Mail app, People and Calendar apps will also get uninstalled. You will have to reinstall them.

Proceed to the next step.

Step 2:

After reinstalling the app, you will find Hotmail account already synced on Mail app as you are using the same account to login to Windows. If not, refer to the link to add/setup account on Mail app:

http://windows.microsoft.com/en-ca/windows-8/add-accounts

If the mails don’t show up for the Hotmail account on Mail app, follow the steps change the settings:

  • Type “mail” and Click on Mail to get Mail app.
  • Right Click on Mail app window and click on “Sync” option from the bottom.
  • Press Windows and C keys together and select Settings.
  • Click on the account which you have synced with Mail app.
  • Check if “Download new email” are marked as “As items arrived”.
  • Check if Download email from is selected as “Anytime”.
  • Check the box besides “Email” under “Content to sync”.
  • Check if it’s turned on for “Show email notification”.

You can also find out how to sync mails by navigating to Settings menu on the same website that is provided above.

Please get back to us with the status of this issue.
