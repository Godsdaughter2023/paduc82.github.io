    It probably doesn't have a drive letter assigned, but is it viewable there?

    It may be that doing so will get it functioning properly.
     
    If it is not viewable in disk management, you might try an "In-Place Upgrade".    You will need to get the data you want on the system backed up to an external device and have all the programs and media keys for the programs installed on your system before doing so.  You will also want to have a good restore point in place incase you need to restore your system.
     
    Immediately following the In-Place Upgrade and before making any modifications or adjustments, connect your external USB drive and see if it will recognize the drive then.  If it still does not, I suggest a Clean Installation of Windows 7, or if you have 16gb of unused space on your system hard drive, you may make a new partition and do a clean install of Windows 7 to test if the USB drive is found in a clean installation of Windows 7.
