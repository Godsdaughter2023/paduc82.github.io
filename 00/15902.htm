jay freedman say:
You can install the printer driver for a PostScript printer in Windows -- you don't have to have the actual hardware attached to your computer. For example, you may find that Windows includes a driver for an HP Laser Jet 2300L PS printer -- the PS indicates PostScript.

To print a document to a PostScript file from Word 2003:

  • Click File > Print.
  • Select the PostScript printer in the Printer Name box.
  • Check the "Print to file" box".
  • Click OK. The Print To File dialog (similar to the Save As dialog) appears.
  • Choose the folder to save in, and the name for the file. By default, Word will add a .prn extension, but you can force it to name the file with a .ps extension by typing the name and extension in double quotes, for example  "MyDoc.ps" .
