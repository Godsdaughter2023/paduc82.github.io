Keyboard- When I press + I type 0? - Microsoft Community

irvingletto say:

Keyboard- When I press + I type 0?

When I press + I type 0?

* Moved from Virus & Malware

* Original title: Keyboard

rob koch say:

Typically that sort of problem is due to a short n the keyboard circuitry, though how it happened varies depending on whether the PC is a laptop or desktop model.

Desktops often have space between the keys where paperclips or other small items can get between them causing a short.

Laptops are typically better sealed, but they can still be affected by liquid spills, which when something other than pure water is involved may still cause a short.

These issues can also be due to software failures including drivers or even malware, but these are far more rare and typically affect more than a single pair of keys.

So initially I'd just try turning over the keyboard or laptop (turned off first) and give it a couple good taps (including the keys themselves) while upside down to see if anything becomes dislodged.  Also pay attention to whether either key seems "sticky" or even stuck in the pressed position.

If the device is a laptop it might require removing the keyboard and either cleaning it with pure water or even replacing it physically.

However, the first thing to try before something that drastic is to swap out the keyboard with another on a desktop, or add a second external one temporarily to a laptop to see if the problem exists there as well.  In most cases, the problem won't occur for the second keyboard, indicating that what ever the precise issue, it's related to the keyboard hardware rather than software.

Hello! I am an Independent Advisor, I would love to help you out!

I have found some information about registry changes that may be able to remove the "fun facts".

Those are:
HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\ContentDeliveryManager

Set the DWORD value ContentDeliveryAllowed to 1.
Set the DWORD value RotatingLockScreenEnabled to 1.
Set the DWORD value RotatingLockScreenOverlayEnabled to 0.
Set the DWORD value SubscribedContent-338387Enabled to 0.

Why this works:

ContentDeliveryAllowed must be enabled for any of the dynamic content to work and should be enabled by default unless you've turned it off with policy. RotatingLockScreenEnabled enables the dynamic background picture instead of a static one. RotatingLockScreenOverlayEnabled is the "Get fun facts" option in the Settings app and setting it to 0 disables it.

Each of the SubscribedContent values appears to control a different part of the Windows UI, such as the start menu, taskbar, notifications area, etc., and 338387 seems to be the one for showing tips on the lock screen.

Bluetooth error code 43 for Acer Iconia W4 solved - Microsoft Community

special character say:

Bluetooth error code 43 for Acer Iconia W4 solved

Today when turning on my tablet, both my Bluetooth mouse and keyboward were not working. I saw that under device manager the Bluetooth presented an error code 43.

I tried the solution from in the microsoft site as well as all solutions and recommendations posted in this forum. None of them worked, it was very frustrating.

I was about to either upgrade to the preview version of windows 10 or reinstall windows 8. Both ways mean wasting a lot of hours fixing something that actually I didn’t break, plus I had already lost several hours trying all possible fixes.

I found a solution. It makes no sense however.  Before explaining I must mention that all my drivers were updated and all updates from Microsoft were installed (even before the problem started). Also I used intel’s tool for checking for any necessary updates and all was updated, the same with acer’s update tool.

The solution: I went to the Acer site, looked for drivers related to the acer w4 and downloaded the Master Installer version 1.16 (it is a zip or rar file). When launching the setup file I got an error message with something like “plug in your adapter” and it didn’t run. So I checked the folders contained in the zip/rar file and in the MI folder I found a folder called “Drivers” along with a setup file. I launched it, it warned me that the drivers that I want to install are not newer than the ones already installed, I clicked continue and all drivers were installed again. After that I restarted the computer and the Bluetooth was working again. By the way, the installer is from Intel, it is called “Intel Atom Processor Z3700 Series Driver”.

If you don’t have this tablet I would recommend updating your drivers. If they are updated then install them again. Get them from the manufacturer site.

Good luck.
