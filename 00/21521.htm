hansv mvp say:

A tabstrip merely provides a strip with tabs. It is entirely up to you as designer/programmer to make something happen when the user clicks on one of the tabs, and you have to do it all in code. So if you want a text box to be visible when the user clicks the first or fourth tab but not when the user clicks the second or third tab, you must write code that sets the Visible property of the text box to True or False depending on the value of the tabstrip (the value of the tab strip is 0 when the user clicks the first tab, 1 for the second tab, etc.)

This means that the tabstrip is very flexible but a lot of work for the designer/programmer.

A multipage control has tab pages. Each tab page contains its own controls (text boxes, labels, etc.). When the user clicks on a tab, only the controls on the corresponding tab page are displayed, the controls on all other tab pages of that tab control will automatically be hidden. No code is needed for that.

So for most purposes, the multipage control is much easier to work with - just place controls on each tab page, and the showing/hiding is done for you.

  • Open a new blank document.
  • In the old document "Select All", using <CTL><A>
  • in the new blank document, use Paste > Special > Text only
  • apply the required new styles

Note: rather than recreating the new styles, you can use the "Organizer" tool to copy just the styles you need from an existing document (or template) into this new document. If you don't know how, I can tell you.

First is use Find and Replace:

  • in the document, open the Replace dialog
  • click on more button (only have to do this once)
  • remove any text in the Find input box
  • click on Format button (at bottom of Replace dialog) > Style command
  • in the displayed "Style" dialog select the old style to find > click OK
  • back in the Replace dialog, move to the Replace input field, remove any text
  • repeat selecting the new style to apply
  • back in replace dialog, click on "Replace All" > wait ...

Repeat for each old style.

The other way is use the Style Gallery or Style pane:

  • In the style gallery, or style pane if you display it, right click on the old style name
  • select the "Select all" option. This selects all paragraphs with the style. It may take some time
  • when selection is done, click on the new style you wish to apply.

After you apply the new styles you should "clean up" the document by removing the no longer required styles. They are just "clutter" that increase the size and complexity of the document for no purpose.

hansv mvp say:

Application.WorksheetFunction.Sum expects a Range as argument, not a String.

Here is a working version using Evaluate:

Function DynamicTotal(RowNumber)
    Dim LastDataSheetSuffix As Integer
    Dim DynamicSumF As String
   
    LastDataSheetSuffix = Application.Sheets.Count - 2
    DynamicSumF = "DATA:DATA" & LastDataSheetSuffix & "!B" & RowNumber
    DynamicTotal = Evaluate("=SUM(" & DynamicSumF & ")")
End Function
