Print position adjustments for documents and labels - Microsoft Community

honest abe say:

Print position adjustments for documents and labels

Is there a way to make small adjustments in print alignment, for different printers, without affecting the formatting of a document?

THE DILEMMA - The paper registration for many printers can be slightly off and there is often no way to mechanically correct this problem.  Furthermore, many printer configuration utilities do not provide options for adjusting the print alignment to compensate for registration problems.  These small registration differences can be extremely noticeable, especially when working with small tolerances related to labels or other publications that require accurate layout.  While there are a myriad reasons for a paper registration problem, even small differences in registration for plain paper, labels, photo paper, cards, brochures, and other stock can create problems on a single printer.

WORD DOES NOT PROVIDE A CLEAN SOLUTION - Adjusting the document settings in Word, in order to print properly on a specific printer, often means the document will now print wrong on other printers.  This is especially a problem when the document is widely distributed.  Moreover, it is extremely unintuitive, not to mention confusing, for users when required to set margins to 1 1/8 inches, for example, just to get a standard 1 inch margin on a specific printer.

BEST SOLUTION - Many accounting programs provide a feature that allows the user to make fine adjustments to the print position of the final printout.  This is especially useful in printing checks properly on a preprinted form without having to change the check design format.  Some graphics and publishing programs also provide such a feature for properly registering output on different printers.  In most cases, the feature is in the form of a small dialog that allows the user to specify small TOP and LEFT adjustments for changing the default starting print position.  These settings can be changed at any time for printing on different printers.  If not present already, we hope Microsoft will consider such a feature for all OFFICE applications.

vittles123 say:

Thanks Jay!!  That put me on the right path.

I had to change the optional execute find/replace lines to a single one and then I added coding that changes the list paragraph style to actual bullets with a .5 indent.

Other modifications include removing the *s and changing the document font to our standard and paragraph spacing to single.  Not sure why Word defaults to the 1.08 spacing...but that is a wish list item for a future build.  :)

See below for the revised coding - may be helpful for someone else too.  :)

Sub eRSScTESSFormat()
    'This macro turns * to bulleted paragraphs
   
     With ActiveDocument.Styles("List Paragraph").ParagraphFormat
        .LeftIndent = InchesToPoints(0.5)
        .RightIndent = InchesToPoints(0)
        .SpaceBefore = 0
        .SpaceBeforeAuto = False
        .SpaceAfter = 8
        .SpaceAfterAuto = False
        .LineSpacingRule = wdLineSpaceMultiple
        .LineSpacing = LinesToPoints(1.08)
        .Alignment = wdAlignParagraphLeft
        .WidowControl = True
        .KeepWithNext = False
        .KeepTogether = False
        .PageBreakBefore = False
        .NoLineNumber = False
        .Hyphenation = True
        .FirstLineIndent = InchesToPoints(-0.25)
        .OutlineLevel = wdOutlineLevelBodyText
        .CharacterUnitLeftIndent = 0
        .CharacterUnitRightIndent = 0
        .CharacterUnitFirstLineIndent = 0
        .LineUnitBefore = 0
        .LineUnitAfter = 0
        .MirrorIndents = False
        .TextboxTightWrap = wdTightNone
        .CollapsedByDefault = False
    End With
    ActiveDocument.Styles("List Paragraph"). _
        NoSpaceBetweenParagraphsOfSameStyle = True
    With ListGalleries(wdBulletGallery).ListTemplates(1).ListLevels(1)
        .NumberFormat = ChrW(61623)
        .TrailingCharacter = wdTrailingTab
        .NumberStyle = wdListNumberStyleBullet
        .NumberPosition = InchesToPoints(0.25)
        .Alignment = wdListLevelAlignLeft
        .TextPosition = InchesToPoints(0.5)
        .TabPosition = wdUndefined
        .ResetOnHigher = 0
        .StartAt = 1
        With .Font
            .Bold = wdUndefined
            .Italic = wdUndefined
            .StrikeThrough = wdUndefined
            .Subscript = wdUndefined
            .Superscript = wdUndefined
            .Shadow = wdUndefined
            .Outline = wdUndefined
            .Emboss = wdUndefined
            .Engrave = wdUndefined
            .AllCaps = wdUndefined
            .Hidden = wdUndefined
            .Underline = wdUndefined
            .Color = wdUndefined
            .Size = wdUndefined
            .Animation = wdUndefined
            .DoubleStrikeThrough = wdUndefined
            .Name = "Symbol"
        End With
        .LinkedStyle = "List Paragraph"
    End With
    ActiveDocument.Styles("List Paragraph").LinkToListTemplate ListTemplate:= _
        ListGalleries(wdBulletGallery).ListTemplates(1), ListLevelNumber:=1
    With ActiveDocument.Styles("List Paragraph")
        .AutomaticallyUpdate = False
        .BaseStyle = "Normal"
        .NextParagraphStyle = "List Paragraph"
    End With
   
   Dim rg As Range
   
 
   ActiveDocument.Range.Find.Execute FindText:="*", Replacewith:="^p*", Replace:=wdReplaceAll
  
   ' turn off all AutoFormat options except bullets
   With Options
     .AutoFormatApplyBulletedLists = True
     .AutoFormatApplyFirstIndents = True
     .AutoFormatApplyHeadings = False
     .AutoFormatApplyLists = False
     .AutoFormatApplyOtherParas = False
   End With
  
   Set rg = ActiveDocument.Range
   With rg.Find
     .Text = "*"
     While .Execute
       If rg.Start = rg.Paragraphs(1).Range.Start Then
         ' star is first character in current paragraph
         rg.Style = ActiveDocument.Styles("List Paragraph")
         rg.Paragraphs(1).Range.AutoFormat
       End If
      
       ' prepare to search for next star
       rg.Collapse wdCollapseEnd
     Wend
   End With
  
   ' restore default settings
   With Options
     .AutoFormatApplyFirstIndents = True
     .AutoFormatApplyHeadings = True
     .AutoFormatApplyLists = True
     .AutoFormatApplyOtherParas = True
   End With
  
   Selection.WholeStory
    Selection.Font.Name = "Arial"
    Selection.Font.Size = 11
    With Selection.ParagraphFormat
        .SpaceBefore = 0
        .SpaceBeforeAuto = False
        .SpaceAfter = 0
        .SpaceAfterAuto = False
        .LineSpacingRule = wdLineSpaceSingle
        .LineUnitBefore = 0
        .LineUnitAfter = 0
    End With
   
     ActiveDocument.Range.Find.Execute FindText:="*", Replacewith:="", Replace:=wdReplaceAll


 End Sub

Try this

 

Sub Summarise_Data()
 Dim IDNo As String
 Dim LastRow As Long, OutRow As Long
 Dim c As Range, Project As String
 LastRow = Cells(Cells.Rows.count, "A").End(xlUp).Row
 Set MyRange = Range("A2:A" & LastRow)
 OutRow = 2
 For Each c In MyRange
     If c.Offset(1) = c.Value Then
         Project = c.Value
         IDNo = IDNo & ", " & c.Offset(, 1).Value
     Else
         IDNo = IDNo & ", " & c.Offset(, 1).Value
         Cells(OutRow, 4) = c.Value
         Cells(OutRow, 5) = Mid(IDNo, 2)
         Project = ""
         IDNo = ""
         OutRow = OutRow + 1
     End If
 Next
 Columns(5).AutoFit
 End Sub
