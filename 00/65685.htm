marvs barce say:

Thank you for getting back to us. Since the newly created user is working, there's a possibility that your old user account is corrupted. We suggest that you migrate all the files from the old user into a new user account. Follow these steps:

Method 1: Make sure that the account you created is running as an administrator:
Control Panel > User accounts > Change your account type > You will be prompted for the administrator password then type the password. > Set the account type to Administrator.

Method 2: Transfer the files from the old user account to the new one.
Go to File Explorer.

  1. Under This PC, select Local (C:) drive.
      
  2. Select Users.
      
  3. Look for the username folder of the old account (ex: admin).
      
  4. Copy all the folders from the old user account or you can choose which to copy.
      
  5. Look for the new user account folder and paste the one you copied. Highlight all the folders and click paste or you can press Ctrl + A and Ctrl + V to copy and paste the items. This will copy all the files and folders from the old user account to the new user account.

In addition to that, you can check this article on how to sign in your Microsoft account to your PC. Also, to delete a user account, you can follow the suggestion of Deepika Gowda on this thread for the steps.

Looking forward for the result.
