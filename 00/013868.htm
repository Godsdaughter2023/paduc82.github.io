rich007a say:

Hi,

Try this:

 

Sub ColourTableCells()
    Dim tbl As Table
    Dim cl As Cell
    Dim r As Long, c As Long
   
    For Each tbl In Selection.Tables
        For r = 1 To tbl.Rows.Count
            For c = 1 To tbl.Columns.Count
                Set cl = tbl.Cell(r, c)
                Select Case Replace(Replace(UCase(Trim(cl.Range.Text)), Chr(7), ""), Chr(13), "")
                    Case "R": cl.Shading.BackgroundPatternColor = RGB(255, 0, 0)
                    Case "Y": cl.Shading.BackgroundPatternColor = RGB(255, 255, 0)
                    Case "G": cl.Shading.BackgroundPatternColor = RGB(146, 208, 80)
                End Select
            Next c
        Next r
    Next tbl
End Sub

bernd p. say:

#1a

Onenote always has a local cache file. This cache contains a copy of all open notebooks in Onenote.

For performance and for working offline. Offline = no access to the location of the original of the notebook.

This can be the case if your original notebook is not on your computer, e.g. on a server in your LAN or on Onedrive.

#1b

YES

#2

Yes, a shared notebook gets moved to Onedrive. The original notebook is now on Onedrive.

But you have a locally cached copy as long as the notebook is open.

#2a

See my answer to #1a

#2b

Working offline means you are changing the locally cached copy on your computer.

So if you want to work offline you have to open the notebook before you kill your Internet connection.

BTW: you should never close a notebook you're working on.

Naturally you cannot open a notebook on Onedrive without an Internet connection.

After regaining your Internet connection your offline changes are synched to the original notebook on Onedrive.

Bernd
