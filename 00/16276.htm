My argument comes down to this: there is a memory leak problem in the kernel, a problem caused by the slide of use as wallpaper, and the only remedy is not to use slide as wallpaper, and happens mostly in users of video cards AMD.

ossiemac say:

In addition to Dave's suggestion, you might like to add the lines of code as follows to save the zoom setting and then reset it; particularly if other people use the worksheet.

Dim dblZoom As Double

dblZoom = ActiveWindow.Zoom 'Save Zoom setting

ActiveWindow.Zoom = 100 'Set Zoom to 100%

Dim c As Range

  For Each c In Range("A18", "A116").Cells
 
    If c(1, 2).Value <> "" Then
   
      ActiveSheet.CheckBoxes.Add(c.Left, c.Top, c.Width, c.Height).Select
     
      With Selection
     
        .LinkedCell = c(1, 49).Address
        .Display3DShading = True
        .PrintObject = False
        .Characters.Text = ""
         
      End With
       
    End If
         
  Next

ActiveWindow.Zoom = dblZoom 'Reset Zoom to previous setting

 

Regards, OssieMac
