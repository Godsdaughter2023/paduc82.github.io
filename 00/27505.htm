greginmich say:

 

 

This error message is only an issue for people who are still using the old Windows Defender interface (MSASCui.exe), which has now been replaced. You can create a taskbar shortcut for the new Windows Defender interface by clicking on the Start button and then right-clicking on the entry for Windows Defender Security Center > More > Pin to taskbar. Or you can unhide the icon by dragging it out of the hidden icons box in the notification area (the up arrow) and then dropping it into the notification area proper. Even if you search for "Windows Defender" you'll see that the search result is now limited to the Windows Defender Security Center app.

 

https://docs.microsoft.com/en-us/windows/threat-protection/windows-defender-antivirus/windows-defender-security-center-antivirus

 

I've submitted a bug report for this issue at the Feedback Hub, and you can upvote that report if you’d still like to have the option for using the old Defender interface. But there really isn’t much point in fretting over problems with an obsolete interface when its designated replacement is already up and running. To find the bug report, just open the Feedback Hub from the Start menu and search for "0x80029c4a". The status of this bug report was recently updated from "Looking into it" to "Working on it", so I'm hopeful that there's a fix on the way.

 

A fix for this issue was posted by Community Moderator

 

Right-click on the desktop and select New > Shortcut.

 

Then copy and paste this line for the “location”:

 

"C:\Program Files\Windows Defender\MSASCui.exe"

 

Now right-click on the shortcut and choose “Run as administrator”.

 

You can also start a fully functional instance of the classic Windows Defender interface from the search box – just type “MSASCui.exe”, right-click on the entry for the app, and “Run as administrator”.

 

 

GreginMich
