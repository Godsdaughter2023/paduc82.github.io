Thank you in advance!!!!

PS: There are 81 pages of landscape tables, so if there's any way to do the sideways page numbers automatically . . . . well, I'd be really appreciative and happy!

john mcghie say:

The complete answer to your question is "Insert the page number from outside the header/footer area, so Word creates it in a floating text box that you can then drag to anywhere you like on the page!"  However, you asked for a simple explanation, which necessarily involves a lot of words, so here they are:

Page numbers are a component of the running Header/Footer.  That means they are either in the header or the footer of the page. They are stored in the section break.

However, they can either be part of the text of the header or footer, or in a floating text box.  The second is what you need.

To enable you to drag the page number, it must be inserted in a floating text box.  We normally recommend against this method because it means you need to manually control the position of the page number.  In your case, it's the only way to achieve your aim

  1. Before you can do any of this, you need to enclose the sideways page(s) in section breaks.  Otherwise, any changes you make will affect the entire document including the non-sideways pages.  So click before each sideways page, and from the Insert menu choose Insert>Break>Section Break (Continuous). Any kind of section break will do, but the Continuous kind will avoid creating a new page.
  2. After the end of each set of sideways page(s) (i.e. at the end of each wide table) insert another Continuous section break.  This "fences off" the area between the second breaks so we can have different page numbers.
  3. Now:  click in a wide table that you just fenced off.
  4. Go to View menu and choose "Header and Footer".  On the Ribbon, click the "Header and Footer" tab (which should now be purple indicating that you are in that view).  The Header and Footer tab won't appear unless you are in the header or footer.
  5. Look for the "Link to Previous" checkbox and uncheck it if it is checked.  This is a critical step: if you forget it, any change you make to the header or footer (the page number) will affect the entire document.
  6. Now double-click in the text of the document, in Page Layout view.
  7. Choose Insert>

    Once you have done this a few times, it will become quite simple.  The essential "secret" is to NOT be in the header or the footer view when you insert the page number.  If you are, Word inserts the page number into the header or footer as part of the text of the header or footer, without the text box, and you can't drag it anywhere.

    Hope this helps

    original title: Number format in mail merge
    macropod say:

    Hi Nancy,

    To control number & currency formatting in Word, add a numeric picture switch to the mergefield. To do this:
    • select the field;
    • press Shift-F9 to reveal the field coding. It should look something like {MERGEFIELD MyData};
    • edit the field so that you get {MERGEFIELD MyData \# $,0.00} (or whatever other numeric format you prefer - see below);
    • position the cursor anywhere in this field and press F9 to update it;
    • run your mailmerge.

    Note1: The '\# $,0.00' in the field is referred to as a numeric picture switch. Other possibilities include:
    . \# 0 for rounded whole numbers
    . \# ,0 for rounded whole numbers with a thousands separator
    . \# ,0.00 for numbers accurate to two decimal places, with a thousands separator
    . \# $,0 for rounded whole dollars with a thousands separator
    . \# "$,0.00;($,0.00);'-'" for currency, with brackets around negative numbers and a hyphen for 0 values
    The precision of the displayed value is controlled by the '0.00'. You can use anything from '0' to '0.000000000000000'.
    If you use a final ';' in the formatting switch with nothing following, (eg \# "$,0.00;($,0.00);") zero values will be suppressed. Note that this suppresses 0s resulting from empty fields and from fields containing 0s.

    Note2: If you use a decimal tab or right-aligned tab to align the values, wrap the switch in quotes (i.e. \# "$,0.00") and insert a tab into the field code after the $ sign, you can have the values output like:
    $ 999,999.99
    $     9,999.99

    graham mayor say:

    Are you pasting from the keyboard? i.e Edit Paste (CTRL+V)
    In that case the following macro should do the trick

    Sub EditPaste()
    Dim oRng As Range
    Selection.Paste
    If Selection.Information(wdWithInTable) = True Then
        Set oRng = Selection.Cells(1).Range
        oRng.End = oRng.End - 1
        Select Case Trim(oRng.Text)
            Case "1": Selection.Cells(1).Shading.BackgroundPatternColor = wdColorRed
            Case "2": Selection.Cells(1).Shading.BackgroundPatternColor = wdColorOrange
            Case "3": Selection.Cells(1).Shading.BackgroundPatternColor = wdColorGreen
            Case "4": Selection.Cells(1).Shading.BackgroundPatternColor = wdColorBlue
            Case "#ERROR": oRng.Text = ""
            Case Else
        End Select
    End If
    End Sub
