    don varnau say:
    Been there and done that---tried it both ways and I do NOT want to use metro---

    Then you should try this workaround to open those Internet Shortcuts in a new window which can be set to open maximized.

    Create a LNK file instead of a URL file and you will have the Properties> Shortcut> Run option.

    Find "IExplore.exe" (probably in Program Files\Internet Explorer)
    Right-click and drag IExplore.exe to the desktop and choose "Create
    shortcut." Right-click that shortcut> Properties> Target, modify the Target
    to include the URL.
    Example:
    "C:\Program Files\Internet Explorer\IEXPLORE.EXE"
    becomes
    "C:\Program Files\Internet Explorer\IEXPLORE.EXE" www.yahoo.com
    Note a space between   EXE" and www
    Rename the shortcut if desired.

    That should always open a new instance of IE for that shortcut and you can specify a window size of Normal, Maximized, etc.

    Don
