To the extent that "fix" works, all it does is workaround some problem/bug the One has with correctly detecting then reporting your NAT status.  It does not and cannot magically open your NAT if your router is not actually properly configured to be open.  The only way to get open NAT is to use a router with good, working UPnP--in which case the One will automatically open the ports it needs--or manually forward the necessary ports via port forwards, port triggers, or DMZ (don't use DMZ!).  From OP's post, it doesn't sound like OP has configured his/her router yet.

a. user say:

Hi Justin,

Thanks for the update. Are the SD cards recognized when you connect it through the other card reader?

If yes, then probably the issue is with the SD card reader and not with the SD cards. In this case, updating the USB controller drivers can help. Did you visit the computer manufacturer’s website to check for the latest USB controller drivers?

You can also check for updates through the device manager. Refer to these steps:

  1. Press Windows key + X, select ‘Device manager’.
  2. Expand the ‘Universal Serial Bus controllers’ and locate the USB controllers.
  3. Right click on the device and select ‘Properties’.
  4. In the driver tab, click on ‘Update driver’ and check.

Try these steps and let us know the results for further assistance.
