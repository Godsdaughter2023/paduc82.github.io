bob jones aka: cybertaz say:

Rows appended to the top or bottom of the original range do not modify the  linked Excel Object in the document. If the new rows are inserted within the original range the linked Excel Object will expand accordingly. BTW: The same is not true of columns. The number of columns doesn't change in the document regardless of how/where they're added in the sheet.

Often, the simplest solution is to replace the original linked object with a new one.
