a. user say:

From your description I would suggest that you may be creating a cross-reference (a) to the Caption and (b) using the "only label and number" option. If you use "Entire Caption" you may see the whole thing. 

The thing to bear in mind is that when you create a cross-reference, Word actually creates a hidden bookmark that "covers" the text (including the SEQ field) that it is going to use, then inserts a REF field that uses that bookmark. If you alter the text in/around the SEQ field, the position of the bookmark can change in unhelpful ways and updating the REF field can reflect that.

I think the idea "Mr. Andrew" had in the other conversation was to create your own bookmark names, in which case you need to make the Cross-reference to the Bookmark you created, not the Caption.

You can avoid the problem with Word using special formatting for captions by inserting the SEQ field "by hand", e.g. 

press ctrl-F9 to insert a pair of the special field code braces { }
type "SEQ Exhibit \*Arabic" (without the quotes) between the braces, so you see
{ SEQ Exhibit \*Arabic }

Use Alt-F9 to show/hide the code, and select & press F9 to update the result. 

You can save the code as an Autotext for easier insertion.

Something I wasn't quite sure about in this case was whether the Exhibit numbers can be "fixed" before you complete your document, i.e. so that you can no longer rely on SEQ to provide the correct numbers.

graham mayor say:
The main thing to remember with sequence fields is that they don't update automatically if you make changes to the order of the document. A macro will help with that - see the example macro at http://www.gmayor.com/installing_macro.htm  

I would be inclined to make the construction into a formatted autocorrect entry e.g. change #ex to (Exhibit { SEQ Exhibit }) where  { SEQ Exhibit } is the SEQ field. Then use the macro to update the numbers (and any REF fields).

talking moose say:

Two questions:

  1. What type of account (POP, IMAP or Exchange)?
  2. How many messages are in your Inbox?

I’m going to assume you’re using either IMAP or Exchange and that most messages are in your Inbox.

The Inbox is synchronized more frequently than any other folder in your account. Outlook must not only routinely check for new mail but also keep its index of messages updated. When Outlook doesn’t have time to complete its update to the index before it checks for more messages, you’ll see a performance problem. Messages won’t download in a timely manner or maybe not at all.

If you can access your account via web browser and move most messages to another mail folder outside the Inbox then performance should improve after syncing with Outlook. This will probably take quite a while.

Hope this helps!

If you're using POP then only the Inbox will download into Outlook. You should be able to access your Yahoo mail via browser, create a mail folder there and move all messages to prevent them from downloading to Outlook. You can leave only those messages you want to sync to Outlook in the Inbox.

keerthi hiremath say:

HTML signatures do work, but you have to paste a chunk of HTML-formatted signature text. For anyone looking how to get an HTML signature on Outlook app on iOS, here's how.  I tried it on iOS 10.3.2 and Outlook 2.25.

The old "shake-to-undo" trick that works for the native Mail app in IOS does not work.  I almost fractured my wrist trying that with Outlook.

1. Paste formatted signature in the left box of http://wordtohtml.net.  You can compose your signature in Word.

2. Select any  tags you want removed from the checkboxes below the left box and click on the “Cleanup my HTML” button.

3. Copy the html generated in the right window.

4. Send an email to yourself with the only the html signature in the body of your email.  Remove any signatures that are already there.

5. Open the email in Outlook app on your iOS device.  Copy the HTML signature.

6. Go to Settings (gear icon) in Outlook app and then to the signatures section.

7. Paste the HTML signature there.

8. Send test email.

You can use any HTML editor to format your signature.  My steps above are for those who do not have such tools.
