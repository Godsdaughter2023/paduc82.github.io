john korchok say:

There might be a way to link a spreadsheet to PowerPoint using OneDrive, but I haven't figured it out yet. The closest I can get is to embed it, but that doesn't let you update the link.

However, you can create a reliable link in 2 other ways:

  1. If the other users are on the same local network, you can create a shared folder on the network, place the Excel file in it and create a link from there.
  2. If users are in different locations, you can put the Excel file at the root of your C: drive or in a folder like C:/ExcelLink. Create the link to PowerPoint, then distribute a copy of the Excel file to each user to be placed in the same spot on their machine. Of course, this will mean that when you update the Excel file, you'll have to distribute copies to users again.

When you Paste Special an Excel file, you can choose to embed it as a Microsoft Excel Worksheet Object, which places an editable copy of the spreadsheet in your presentation. If you don't need to update the Excel data later, this is the simplest solution.
