a. user say:

Hi Kirk,

Thank you for posting in Microsoft Community Forum.

Please be assured that we would do our best to assist you.

Everyone has their own way of organizing what's important to them, and you can use Windows 8 to be just as particular about all of your important files and folders.

The four basic ways to organize how your files and folders appear in Windows 8 include arranging, sorting, grouping, and changing your view.

Even though you set the sorting order in Windows Explorer, it won’t be showing in the Open file dialog box. You need to sort it when you open the dialog box.

Group by (Create visual breaks for easy organization)

You can group files by property. Like sorting, grouping won't significantly change the way files are displayed. It just breaks them into different sections, depending on which property you choose. So, if you have a folder full of text documents, you might want to group them by type to see which program was used to create them or you can group them by date to separate them by the date they were taken.

Sort by (Sequence your files to suit you)

Sorting options are similar to arrangements, except they don't change the way your files appear they just reorder them. Generally, files are listed in alphabetical order by name. But this convention doesn't always make the most sense.

To group your files by property in open dialog box, follow the below steps: (Follow the same steps in all the programs which you need)

  1. Open the dialog box, change the view to details under more options.
  2. Point to Date modified, and then click a property.

Note: If you don't see the Arrange by, Sort by, or Group by menus, it means that these options aren't available for the particular browsing state that you're in. For example, arrangements are only available in libraries, so you won't see this option when you're browsing outside a library.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.

sujatamoktan say:

Hi,

I suggest you to uninstall Windows Live Mail, please back up your messages and contacts by following the steps provided on this link: How do I import and export email, contacts, and calendars with Windows Live Mail?

After you back-up your emails, contacts, and calendar, please visit: How do I uninstall one or more Essentials programs to remove components such as Windows Live Mail or the whole program of Windows Live Essentials.

We can now proceed to the clean removal of registry keys associated with the previous Live Mail installation. It is highly recommended that you know how to restore the registry just in case a problem occurs. For information about how to back up, restore, and modify the registry, please read this article.

Then, please follow the steps below on how to do the clean removal process:

1. Open Registry Editor (Click the windows icon also known as the Start button, on the Start Search box, type regedit, and then press ENTER to open a Registry Editor window)
2. Click HKEY_CURRENT_USER, click SOFTWARE, and then click MICROSOFT. Right click Windows Live Mail and then press Delete to remove the registry key.
3. Click HKEY_LOCAL_MACHINE, click SOFTWARE, and click MICROSOFT. Right click Windows Live Mail and then press Delete to delete the registry key.
4. Exit Registry Editor and then restart the computer.

Once you're done with the clean removal Windows Live Mail, you can now download and install the latest Windows Live Mail version through this http://windows.microsoft.com/en-us/windows-live/essentials

 and follow the instructions on how to add your account.

Do you only experience this issue when pairing this specific mouse? Let's try using the Add to Personal Area Network feature to pair this particular mouse. Here's how:

  2. On the top-left corner of the screen, click Add a new Bluetooth device.

  5. Restart your computer, and then try pairing the mouse again.
