Using the SUBTOTAL function to SUM will not include other SUBTOTALS in the SUM (so that using SUBTOTAL will sum everything in the range except values in SUBTOTAL formulas).

nathan o. - microsoft support say:

Gina is correct.  You should be able to run Access 2003 on a Windows 7 machine (32 bit or 64 bit).  With that being said, could you run into issues, possibly.  As mentioned by Gina, if you are going to run multiple versions of Access on the same machine, they should then be installed in the order of the oldest version to the newest version.  In addition to this, if you have multiple versions of Access on the same machine a few things to keep in mind are:

 

·         You should never have multiple versions of Access open at the same time.

·         When you flip between versions, Access will need to reconfigure in order to load all the proper components for that specific version of Access.  When this occurs, you will need to be sure you have proper permissions on your machine for each version to reconfigure properly.  If you don’t, you could end up having references in your database that point to the wrong object library files.  If you run into an issue like this, you could try changing the setting for msaccess.exe to always run as administrator.  To do this, you can navigate to each version of msaccess.exe on your machine, right click on it, and choose properties.  In the properties dialog, click on the compatibility tab and choose “Run this program as administrator”. 

·         If you continue to have issues after doing this, then there may be a problem specific to your database and / or the install on the specific machine you are running on. If running a repair of that version doesn’t help then it may be a good idea to open a ticket directly with Microsoft to look into the issue specific to the machine you are running on.  But as mentioned by Gina and others, you should be able to run Access 2003 on a Windows 7 32 bit or 64 bit machine.

Why doesn't LEN work in SUMIF formula? - Microsoft Community

kate von say:

Why doesn't LEN work in SUMIF formula?

Column A is a text column with values like 1507, 1507-1, 1507adj, 1508, 1508-1, etc.

Column B has numbers

I want to add all the numbers in B, but only if the length of the text field in A is 4 (1507 and 1508 meet the criteria)

Why does this formula return an answer of 0?

SUMIF(A1:A5,LEN=4,B1:B5)

hansv mvp say:

LEN by itself is meaningless, and you cannot use it this way in SUMIF. You could use the following instead:

=SUMPRODUCT((LEN(A1:A5)=4)*B1:B5)

or, as an array formula confirmed with Ctrl+Shift+Enter:

=SUM(IF(LEN(A1:A5)=4,B1:B5))

jay freedman say:
The default printing options are set in Windows, not in Word.

In Windows 8.1, press the Windows key and the X key together to open a menu, and press P to open the Control Panel. Click or tap the Devices and Printers item. In that screen, right-click or tap/hold the icon of your printer and choose Printing Preferences.

That dialog will probably look like the one you get from clicking Printer Properties in the Print dialog of Word, but this one controls the printer's defaults. Set one-sided printing (or turn off duplex printing) and click OK.

Conditional Formatting dates in adjacent cells - Microsoft Community

lilyhopkins say:

Conditional Formatting dates in adjacent cells

I have an excel worksheet that logs all of the equipment our department holds, I want to change the cell (E5) to red when something hasn't been returned by the next day.  Once the item is returned and a date has been entered into cell (E7) I want the cell to change back to its original formatting of no colour background and automatic text colour, so that when somebody checks the spreadsheet it is easy to see what equipment is still on loan.  I would be grateful for any help anyone could give me.  Thank You.
bernie deitrick say: Select E5 and to the right (assumes that your equipment names are entered in row 5 starting in column E), click Conditional Formatting on the Home tab, select "New Rule"  then under rule type select "Use a formula to determine...." and use the formula

=AND(E5<>"",E7="")

and choose red fill as the format.

Then, any cell in row 5 that is filled without a date (or any entry, actually) in row 7 will be colored red.
