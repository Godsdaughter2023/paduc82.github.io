doug robbins - mvp office apps & services (word) say:

In the footer of the document, insert

 

{ IF { PAGE } < { NUMPAGES } "Contd. P / { = { PAGE } + 1 }" "" }

 

For the text at the top of the subsequent pages, insert the following field construction into the header of the document.

 

{ IF { PAGE } > 1 "Page - { PAGE }" "" }

 

You must use Ctrl+F9 to insert each pair of field delimiters { } and you use Alt+F9 to toggle off their display.

 

You will probably need to do a print preview to get the field constructions to update to see the results.
