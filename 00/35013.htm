helpster say:

Hello,

I'll throw a couple persoanl observations out there.

1) TouchPads are EVIL. A slight brush of your thumb when typing fast and you are opening tem things! If you are using a laptop with a touchpad, diable it for a bit and see if that stops the odd behavior. If it does, get a nice USB mouse (wired or wireless)

2) Speaking of 'Wireless'. Wireless keyboards can have a lag. This can cause odd issues. I just had a change mine out because of similar issue.

3) The system is running slow...just not keeping up. Make sure that you don't have many thing open at one time. Though Windows can multitask well, your hardware might not be able to keep up the pace! Check the Notification Bar in the lower right corner of Taskbar. There should only be a few things running. If you have more than 6 itens, I would see which ones you can eliminate.

Here are the basics: Network, Speakers, Anti Virus, Safely Remove Hardware, Battery(laptop), Outlook(if that is what you use).

4) Make sure you have the latest drivers for your Computer. Mouse and Keyboard to start with, but everything else as well. Go to your computer manufacturers site to make sure you have the latest versions.

5) Check System Performance by opening Task Manager. Right-click the Taskbar and select it. Look at the Performance tab. If it is using a LOT of system memory and/or CPU, you should look at Processes to see what is using it all.

6) Windows 7 should run good with 1GB RAM, but if you have a lot of programs running, you will really be taxing the system.

I hope this helps
