graham mayor say:
Create a new document in Word 2007.

Add a module to the VBA project of that new document - see http://www.gmayor.com/installing_macro.htm

Copy the macro code from the message or move it from your normal template to that new module.

Save the document as a macro enabled template DOTM in the Word Startup Folder. That folder is (by default) located by typing - %appdata%\Microsoft\Word\Startup in the address bar of Windows Explorer (or the file save as dialog).

Add a button to the QAT or (if you are feeling adventurous) to the ribbon of that template to call the macro then save it again. - http://gregmaxey.mvps.org/word_tip_pages/customize_ribbon_main.html

Close and re-open Word. The template will load with Word.

Copy to the users' Word startup folders. When Word is restarted the macro should be available.

As for the wildcard, you cannot use the replacement wildcard in taht particular string, so you would need a different technique e.g.

Sub ReplaceList()
Dim vFindText As Variant
Dim vReplaceText As Variant
Dim strReplace As String
Dim orng As Range
Dim i As Long
Dim sAsk As String

vFindText = Array("prior to", "Acknowledgements", "towards", "about ([0-9]{1,})")
vReplaceText = Array("before", "Acknowledgments", "toward", "")

For i = 0 To UBound(vFindText)
    Set orng = ActiveDocument.Range
    With orng.Find
        .ClearFormatting
        .Replacement.ClearFormatting
        Do While .Execute(FindText:=vFindText(i), _
                          MatchWildcards:=True, _
                          Forward:=True, _
                          Wrap:=wdFindStop) = True
            orng.Select
            If i < 3 Then
                sAsk = MsgBox("Replace - " & vbCr & orng.Text & vbCr + vbCr & _
                              "with - " & vbCr & vReplaceText(i), vbYesNo, _
                              "Replace from Table")

                If sAsk = vbYes Then
                    orng = vReplaceText(i)
                End If
            Else 'Items after the third item in the ar ay.
                strReplace = Chr(126) & Mid(orng.Text, 7, Len(orng.Text) - 5)
                sAsk = MsgBox("Replace - " & vbCr & orng.Text & vbCr + vbCr & _
                              "with - " & vbCr & strReplace, vbYesNo, _
                              "Replace from Table")
                If sAsk = vbYes Then
                    orng = strReplace
                End If
            End If
            orng.Collapse wdCollapseEnd
        Loop
    End With
Next
End Sub

Note you don't need an array unless you are processing a list of terms.

You don't print forms, you print reports. Forms use mainform and subforms to group records. Reports don't need to. 

Your report should have a recordsource of a query that returns the data you want to see in the report. You use Grouping to eliminate duplication of printing. 

If you want to print a report filtered for the current record on a form, then you use:

DoCmd.OpenReport "reportname",,,"[PrimaryKey] = " & Me.primarykeycontrol

where primarykeycontrol is the name of the control bound to your PK field.
