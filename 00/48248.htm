Please follow the steps for "Horizontal" Spacing and "Vertical" Spacing

1. Press the Win+R keys to open the Run dialog, type regedit, and click/tap on OK to open Registry Editor.
2. In Registry Editor, navigate to the location below.
HKEY_CURRENT_USER\Control Panel\Desktop\WindowMetrics

3. To Change the Desktop Icon "Horizontal" Spacing

A) In the right pane of WindowMetrics, double click/tap on IconSpacing to modify it.
B) Type in a value between -480 to -2730, and click/tap on OK.
Note

lucille15 say:

How to get rid of spooled file sitting in print queue.

I cancelled spooled file, but it won't go away. It is showing 'deleting', but it is misleading. The file is still there, blocking everything else ready for printing. How can I delete this cancelled file to open my printer to next jobs?
babu v say: Hi Lucille,


Method 1: Turn the printer off and on and check if it helps.

Next, try turning the printer off and then on. With certain printer models, this cancels the print job.

If this method worked for you, you are finished with this article. However, you may want to read the "Prevention tips" section to avoid this problem in the future.

 

Method 2: Let’s stop the Print Spooler service and then delete the files in Printer folder.
a.    Press “Windows Key+ R”, type “services.msc”, hit Enter.

b.    In the services window, use the scroll bar to locate " Print Spooler service”, double click on it, click Stop

c.       Navigate to the folder below, and open the PRINTERS.

C:\Windows\system32\spool\PRINTERS

 

d.      Click/tap on Continue and Yes (if prompted by UAC) to approve opening the PRINTERS folder.

e.      Delete all files in the PRINTERS folder until it is empty, then close Windows Explorer.

f.        Note: DO NOT DELETE the PRINTERS folder itself. Only delete its contents.

g.       Open Services, and start the Print Spooler service.

umesh p say:

Hi,

You may disable those entries in msconfig and check if there is any undesirable behavior. If there is no any undesirable behavior then you can enable them back in msconfig and then delete them in the registry if you want.

Steps to disable entries in msconfig:

1. Click on start > Run
2. Type msconfig and press enter
3. Click on startup tab
4. Clear the check mark for those entries and click on OK button.
5. Restart the computer.

cba10075 say:
Hello Umesh, thanks.  Unchecking the referenced entries in caused msconfig to go haywire, sort of.  It turned a bit messy with a frozen system and a forced 'selective starup' .. which I, at first, could not reverse.  Moreover, two new 'strange character' entries suddenly showed up under 'Startup' in msconfig.

Long story short: a few reboots later (plus a 'last good' re-start), the initial two 'funny' entires were gone, however, instead I had two new ones I couldn't get rid of.  The solution was to use regedit to delete this entry found under MSConfig\startupreg:
 
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\MSConfig\startupreg]

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\MSConfig\startupreg\Load]
"key"="SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\Windows"
"item"="蜈 "
"hkey"="HKCU"
"command"="蜈 "
"inimapping"="1"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\MSConfig\startupreg\Run]
"key"="SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\Windows"
"item"="蜈 "
"hkey"="HKCU"
"command"="蜈 "
"inimapping"="1"

Once deleted, all funny entries were gone from Startup and msconfig went back to a normal boot sequence.
