danno mck. say:

If you have an HDTV, you don't want to use the non-HD composite cable (yellow for video, red & white for audio) that comes with the 360S; it will only support up to 480P. You'd need to be more specific about which cable you used with your older console, as those came with a component cable sometimes (at least, my Elite did, I don't know about Arcade or Pro), but then they changed to including only the composite cable (since many people would use HDMI, anyway).

If you do have a component cable from your old console, you could continue using that with the new console. Many people don't see a lot of difference between component 1080P and HDMI, though the latter is inherently more stable (but it can vary with the screen) and carries audio in the one connection. Look around for an under-$10 (US) HDMI cable and I think you'll appreciate the upgrade.

I use HDMI cables, because it frees up a component slot on my TV for a different device.  Sure, it can be hard to tell the difference between the HDMI video output and the component cables, but it's easier and far more streamlined (one cable > five cables).
