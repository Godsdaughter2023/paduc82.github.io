Hi

 

Using IE10 Desktop:

1. Open IE10 desktop.

2. Go to the webpage you want to pin.

3. Click on the Tools "cog" (top-right) then select Add site to Start Screen.

4. Click Add in the box that then appears.

Using IE10 Modern (aka Metro):

1. Open IE10 Modern.

2. Go to the webpage you want to pin.

3. Right-click on blank area of the screen.

5. Select Pin to Start  then Pin to Start again in the box that appears.

(you may need to scroll to the far right of your Start screen to then find it)
