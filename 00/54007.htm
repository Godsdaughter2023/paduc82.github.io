  • Look on the bottom of the wireless device for a Reset button. Resetting the device will disconnect and then reconnect the wireless connection. There might also be a Reset button on the USB receiver for your device.

  3. Under Hardware and Sound, select Configure a device.

  1. Click the Start button, enter "Windows Update" in the search box, and then, in the list of results, select Windows Update.
  2. In the left pane, select Check for updates.
  3. On the Select the updates you want to install page, look for updates on your device, select the check box for the driver that you want to install, and then click OK. There might not be any available driver updates.

Download and install a driver yourself

  1. Click the Start button, type "device manager" in the search box, and then select Device Manager.
  2. In the list of hardware categories, find the device that you want to update, and then double-click the device name.
  3. Click the Driver tab, click Update Driver, and then follow the instruction.
