If we install any major Windows updates, we will possibly encounter a driver issue. To resolve this one, we suggest reinstalling the audio device driver. To do so, please follow the steps below:

  1. Press Windows key + X, then select Device Manager.
  2. Under Device Manager, navigate Sound, video and game controllers.
  3. Double-click Sound, video and game controllers to expand it.
  4. Right-click on the audio device installed (e.g Realtek High Definition Audio), then click Properties.
  5. Under audio Properties, select Driver.
  6. Under Driver tab, click Uninstall.
  7. Click OK.
  8. Once your done, click Scan for hardware changes to install the most updated driver for the device.

These problems may occur if you start Microsoft Windows Explorer before a logon script has completed.

In this case, we need to turn off Fast Logon Optimization via GPEDIT .
Follow these steps:

1. Run gpedit.msc via Start menu.

1. Run gpedit.msc via Start menu.
2. Under User Configuration, stretch the Administrative Templates.
3. Stretch System.
4. Click Scripts.
5. Look for  Run logon scripts synchronously, double click it and then select Enabled radio button above left.
6. Click Apply, then OK.

Restart computer afterwards and check if issue persists.
