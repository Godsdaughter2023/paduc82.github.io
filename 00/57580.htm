Thank you for posting your query on Microsoft Community.

As per the issue description you are facing performance issue with your computer and could not change the resolution.

I would suggest you to perform System Maintenance troubleshooter on the computer and check if it fixes the issue.

Follow the methods below and check if it works.

Method 1: Perform System maintenance check to improve performance of the computer:

a. Press Windows and x key together and select Control Panel.
b. Click on Troubleshooting.
c. Click on View all on the left side panel.
d. Click on System Maintenance and follow the onscreen instructions.

Method 2: Refer to the link to improve PC performance:

Ways to improve your PC's performance

http://windows.microsoft.com/en-IN/windows-8/improve-pc-performance

You can also refer to the link to improve the computer’s performance by optimizing the hard drive.

Improve performance by optimizing your hard drive

http://windows.microsoft.com/en-in/windows-8/improve-performance-optimizing-hard-drive

Important: Data available on bad sectors might be lost when chkdsk tries to repair your hard disc.

Method 3: Boot the computer into Safe Mode and check if you experience slow performance.

Step 1: Refer to the link to boot the computer into Safe Mode:

Windows Startup Settings (including safe mode)

http://windows.microsoft.com/en-us/windows-8/windows-startup-settings-safe-mode

Step 2: If the issue does not occur in safe mode, then you may boot the computer into Clean Boot state and check if any program or application might be the reason behind this issue.

How to perform a clean boot in Windows

http://support.microsoft.com/kb/929135
