swischner say:
You are right. Apart from the backup option in Windows Deesktop OneNote (eg. OneNote 2016) there is no easy way to duplicate notebooks.
Here is what the OneNoite developers themselves suggest:
Create a new, empty notebook (perferrably with another name) and copy (not move!) section by section from the original notebook to the new one.

Your attempt to copy the link (placeholder) file from within OneDrive using a browser should also work. At least I successfully copied several notebooks that way and indeed created different versions. I just copied the notebook placeholder to another folder.

There is a third, also inofficial option: If your notebooks are placed in folders instead of the "main directory" of OneDrive, you are able to download that complete folder to your computer using the web interface. This is not possible with single notebooks (no Download option there), just with complete folders. That will create and download a ZIP file that actually holds all the notebook data in the "classic" OneNote compatible format (eg. .ONE files for sections). So this is a real local backup.
But: To restore a notebook from these files you absolutely NEED OneNote 2013/2016 for Windows. They cannot be opened with another OneNote version.
