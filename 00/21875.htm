jay freedman say:

In order to repeat on every page, the text box must be part of either the header or the footer -- although it doesn't have to be in the header or footer area, it does have to be anchored there. This is covered in detail at http://wordfaqs.mvps.org/AnchorToHeader.htm.

The key idea is that the cursor must be in the header or in the footer at the time you click the Insert > Text Box > Draw Text Box command. Then you can draw it wherever you want it.

Because you want it in the lower part of the page, it seems more logical to anchor it to the footer, but it really doesn't matter whether the anchor is the footer or the header.

steve rindsberg say:

Given a shape containing text and some text you want to apply a hyperlink to, here's a start:

                        

Function ReturnRange(oSh As Shape, sText As String) As TextRange

    Dim lStartChar As Long

    With oSh.TextFrame.TextRange
        lStartChar = InStr(.Text, sText)
        If lStartChar > 0 Then
            ' the text we want is there
            Set ReturnRange = .Characters(lStartChar, Len(sText))
            Exit Function
        End If
    End With

End Function

Sub TestMe()
    Dim oSh As Shape
    Dim oTxtRange As TextRange

    Set oSh = ActiveWindow.Selection.ShapeRange(1)
    Set oTxtRange = ReturnRange(oSh, "Click Here")

    If Not oTxtRange Is Nothing Then
        With oTxtRange
            MsgBox .Text
            .ActionSettings(ppMouseClick).Hyperlink.Address = "http://www.pptfaq.com"
        End With
    End If

End Sub
