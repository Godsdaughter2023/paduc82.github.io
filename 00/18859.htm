sanjaymehta---01 say:

The certificate needs to be in .der format. You probably have it in .cer (PEM) format right now.

 

You can convert it using openssl. Change the filenames appropriately:

 

openssl x509 -outform der -in MYCERT.pem -out MYCERT.der 

 

Then send the .der file to the phone and open it. The phone should offer to install it as a certificate.

Using your data in my test, if you use code such as the following to load the combobox from an array, with MatchEntry set to 0 - fmMatchEntryFirstLetter when you type the first number of the data in the first column, that row in the combobox is selected

myarray = xlsheet.Range("A1").CurrentRegion.Value
ComboBox1.ColumnCount = xlsheet.Range("A1").CurrentRegion.Columns.count
With ComboBox1
    For i = LBound(myarray, 1) To UBound(myarray, 1)

        For j = 2 To UBound(myarray, 2)

    Next i
End With
