ozhunter say:

use named range as control source for user form combo box

Hello

 

Is it possible to use a named range as the control source for a combo box on a userform.

 

If so, how do you reference the named range? I have tried

 

Me.cboDate.ControlSource = NamedRange

 

and

 

 Me.cboDate.ControlSource = "NamedRange"

 

and

 

Me.cboDate.ControlSource = 'NamedRange'

 

all with no success.

 

Any advice gratefully accepted at this point.

 

 

hansv mvp say:

Me.cboDate.ControlSource = "NamedRange"

 

is the correct syntax, but keep in mind that the ControlSource is the single cell that will display the selected item in the combo box. Perhaps you intended to set the range that populates the list of the combo box? If so, you need to set the RowSource property:

 

Me.cboDate.RowSource = "NamedRange"

 

Also, keep in mind that the named range must either have the workbook as scope, or it must be local to the active sheet.

If you define a named range that is local to Sheet1, and you run the userform while Sheet2 is active, you can't use that named range.
