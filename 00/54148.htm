net stop dnscache

Hi,

The errors you are referring appears in a computer, when a program installed in the computer is not communicating with the volume shadow copy service properly.

 

Method 1:

 

Check the status of the Volume shadow copy in services.

 

a.    Click on Start button.

b.    In search box type services.msc, hit enter.

c.    Check for the Volume shadow copy services.

d.    If it is disabled, Enable it by right click on the Volume shadow copy service and click on Enable.

e.    Once again start the Volume Shadow copy services.

 

 

Method 2:

 

Perform Clean boot and check:

 

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7

http://support.microsoft.com/kb/929135


Please note:  After the troubleshooting, make sure to boot your computer to normal mode by following the step 7.

 

 

Method 3:

Perform a system file checker scan on your computer and check for the issue.

How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7

http://support.microsoft.com/kb/929833

For more information please refer the below link:

http://social.technet.microsoft.com/Forums/zh/w7itproperf/thread/01436ec7-113f-49b1-9bed-29478fea77f6
