Dynamic Host Configuration Protocol (DHCP) is a client/server protocol that automatically provides an Internet Protocol (IP) host with its IP address and other related configuration information such as the subnet mask and default gateway. RFCs 2131 and 2132 define DHCP as an Internet Engineering Task Force (IETF) standard based on Bootstrap Protocol (BOOTP), a protocol with which DHCP shares many implementation details. Outpost Firewall Pro isn't just a firewall, it also includes always on spyware and program monitoring, which are real-time functions similar to those found in MSE.  This means that it's highly likely these two programs are interacting and though they may or may not directly conflict, are at least causing the excess cpu overhead by monitoring each others activity.

This is why Microsoft recommends using the included Windows firewall, which simply operates as a firewall and not an active protection component, since MSE performs these network monitoring functions itself.  However, if leak-tests are your only measure of security, you may be better off with Outpost and another simple dumb antivirus, since that's how it is designed to operate.  I personally prefer an integrated security application that performs all the anti-malware functions in one application, since it doesn't have these questions of application overlap and/or 'holes' between programs designed with completely different criteria.
