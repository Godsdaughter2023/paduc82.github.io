greg wilson 2 say:

See help for the Characters property. The syntax is Characters(start, length), where start is the start character number and length is the number of characters.

Example reading and writing individual characters:

Sub Test()
Dim i As Integer
With Sheets("Sheet1").Cells(1, 1)
    Sheets("Sheet1").Cells(10, 1).Value = .Value
    For i = 1 To Len(.Value)
        If .Characters(i, 1).Font.Underline <> xlNone Then
            'Note that the Underline property is not a simple True or False but a numeric value corresponding
            'to the particular underline style of the individual character referenced (single, double, etc.).
            'So I equate the destination cell individual character's underline state with that of the corresponding
            'character in the source cell.
            Sheets("Sheet1").Cells(10, 1).Characters(i, 1).Font.Underline = .Characters(i, 1).Font.Underline
        End If
    Next
End With
End Sub

Unfortunately, it does not appear that userform controls support modifying the underline state of individual characters within a text box or label. So I can't help you there.
