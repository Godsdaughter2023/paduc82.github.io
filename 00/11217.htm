ossiemac say:

Following solution involves a UDF (User Defined Function) with VBA code as below

To install the code

  • Alt and F11 to open the VBA editor
  • Select Menu item Insert -> Module
  • Copy the code below and paste into the VBA editor.
  • Close the VBA editor (Cross very top right of window)
  • Use "Save As" to Save the workbook as Macro Enabled
  • Ensure that macros are enabled. See Help for how to do this. (The option to "Disable all with notification" should be OK)
  • In a cell enter the following formula where A1 contains the string to search for the 7 digit number. The number of digits for the number is represented by 7 parameter so the UDF is more generic and can be used for any specified number of digits.

=FindNumeric(A1,7)

The above formula returns the 7 digit number as text. If you want to return it as a number that you can use in maths functions then expand the formula as follows to convert to a number value.

=VALUE(FindNumeric(A1,7))

Both of the above formulas will return an error if a 7 digit number is not found. Expand the formula as per the following if you would like the cell to appear blank if a 7 digit number is not found.

=IFERROR(VALUE(FindNumeric(A1,7)),"")

Copy all of the code below into the VBA editor.

Function FindNumeric(strToSearch As String, n As Long) As String
    Dim objExec As Object
    Dim objRegex

    Set objRegex = CreateObject("VBScript.RegExp")
    With objRegex
        .Global = True
        .Pattern = "\d{" & n & "}"
        Set objExec = .Execute(strToSearch)
    End With
    FindNumeric = objExec(0)
End Function
