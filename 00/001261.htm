Just for the sake of accuracy - email accounts are removed from W10 (or any OS for that matter) - the account(s) are removed from the <email clients> in which it's configured such as Outlook desktop, W10 Mail etc.

In terms of Outlook - to remove the account - two ways to go about it

#1 - From within Outlook 

File --> Account Settings --> email tab --> remove the GMail account. If the account was configured as a POP account, go to the Data Files tab and remove the associated PST file taking note of where on your computer it is located. The PST file will not be physically deleted from your computer so will need to do that as well if you want everything removed.

#2 - with Outlook closed - using Control Panel <Mail> app --> same instructions as item #1

greginmich say:

[Original Title: Windows defender]

We already have the option for turning off Windows Defender’s Automatic Remediation:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/disable-automatic-quarantine-in-w10/8183c443-6ad0-4e28-a114-085a3db32bbf

Or you can run (copy, paste, and enter) these commands at the Administrator Command Prompt to toggle the Automatic Remediation setting:

Turn off Automatic Remediation:

REG ADD "hklm\software\policies\microsoft\windows defender" /v DisableRoutinelyTakingAction /t REG_DWORD /d 1 /f

Turn on Automatic Remediation:

REG DELETE "hklm\software\policies\microsoft\windows defender" /v DisableRoutinelyTakingAction /f

I don’t recall ever having seen a single complaint about having to choose between the different remediation options - just complaints about not having those options available for real-time detections (like the one here). Remember that once Automatic Remediation is turned off, all you have to do to initiate it is just click on the big red Clean PC button in the "red screen". But if you want to make changing the default action possible, then you have to click on the Show details link. And then if you actually want to take any action other than the default, you have to click on the Recommended action down-arrow and change the default selection. So you’re certainly never put in a position where you’re forced to make a choice:

This is one of the things that I always change on all my Windows 10 PCs. The other setting that always gets changed is the one that’s described in this thread:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/windows-defender/235ac1ff-b772-4568-975f-0d656d9f80a5

Who wants to click on that stupid View details button over and over again just to see the different components in the History tab?
