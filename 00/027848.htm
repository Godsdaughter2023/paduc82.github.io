jllatham say:

See if this helps any.  It depends on one, and only 1 semicolon being in cells where there is text and a number.  If no word(s) then no semicolon, just the number:

For an entry in cell A1:

=IF(ISERR(FIND(";",A1,1)),A1,VALUE(RIGHT(A1,LEN(A1)-FIND(";",A1,1)))+1)

It simplifies to English kind of like this:

IF(ISERR(FIND(";",A1,1)), A1

If there is no ; in the cell, then just return the cell's contents (number value).

The lack of a semi-colon will generate an error, that we have trapped/tested for.

Ok, there was no error, means there is a semi-colon and we assume a word or phrase to the left of it that counts as 1, and all to the right of the semi-colon makes up a number:

,VALUE(RIGHT(A1,LEN(A1)-FIND(";",A1,1)))+1)

The VALUE() turns text that looks like a number into a number, and we add 1 to that number to get the total.

Here's test stuff I used showing all the formulas for each row, data in A, formula in B.  Should work for any number from 1 to whatever Excel can handle, just don't use commas or dollar symbols.  Periods (a decimal point) should be ok.
