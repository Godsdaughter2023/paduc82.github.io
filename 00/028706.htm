As Doug wrote, using the Multilevel List dialog box is crucial when modifying indents for a numbered paragraph. In this case, click in the first Heading 1 paragraph of your document. Extracting Part of a Field

Hi,

Can anyone tell me what function I should use so that it only shows anything before the third "/"

 

Eg.

AJ

marshall barton say:

You can use the InStrRev function to find the last / in the string and Left to get what you want:

   Left(thefield, InStrRev(thefield, "/") -1)

But, that will only wotk if the are always exactly 3 /s in the field.  If that's not your situation, please explain, beyond your simple examples, what variations are possible so we don't waste time guessing what other kinds of values might be in the field.

ken sheridan say: You need to use the Like operator for a pattern match, not the equality operator, and the arguments should be separated by commas, not semi colons:

Expr1: IIf([Order Number] Like "*/*",Left([Order Number],InStrRev([Order Number],"/")-1),[Order Number])

You can if you wish alias a computed column with the name of the column itself provided that you qualify the references to the column in the expression with the table name:

Order Number: IIf([Orders].[Order Number] Like "*/*",Left([Orders].[Order Number],InStrRev([Orders].[Order Number],"/")-1),[Orders].[Order Number])

BTW while the expression will work in a query, it would fail in VBA as in code the expressions for both possible return values are evaluated, so there an error would be raised by the second argument of the IIf function where the value in the column did not contain a slash character.  In code you'd need to use an If....End If construct.

How do I import favorites into Edge from another browser?

You can also try the following:

  1. Open your old browser’s main bookmarks folder. You can also choose to import only a subset of your bookmarks. In either case, make sure you select all the desired bookmarks.
  2. Drag and drop the bookmarks on to your Windows 10 desktop, or any distinct (and separate) folder of your choice.
  3. Open the ‘C’ drive on your computer. If your OS is installed on any other drive, open that one.
  4. Under the ‘Users’ section, locate the name of your system’s main admin account and open its folder.
  5. Locate the folder named ‘favorites’ and open it. In short, the path for this folder becomes as follows (assuming the system drive to be ‘C’ and your account name to be ‘XXXXX’).
    C:\users\XXXXX\favorites\
  6. Copy all the bookmarks you saved in step 2 to the location accessed in the previous step.
  7. Close this folder after making sure that all the bookmarks have been copied successfully.
