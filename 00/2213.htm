Excel not adding correctly - Microsoft Community

a. user say:

Excel not adding correctly

according to excel 8.48+.04=8.519999 instead of 8.52.

to reproduce add .04 to itself and then autofill until the total is 8.48 

mike h.. say:

Hi,

This is a known issue and is caused by the way Excel does floating point arithmetic. Excel works in binary and often there is no precise binary equivalent of a floating point number and this precision error occurs. It is fully documented here.

https://support.microsoft.com/en-ca/help/78113/floating-point-arithmetic-may-give-inaccurate-results-in-excel

In your specific case you're working to 2 decimal places so you can eliminate the error using ROUND like this

=ROUND(S1+0.04,2)

EDIT Please note that reducing the number of decimal places may make the number appear correct but the underlying value will still contain the precision error unless you use ROUND

how do i stop longing running scripts?? - Microsoft Community

junker69 say:

how do i stop longing running scripts??

I want to stop these long running scripts that keep messing up my computer NOW!!!

don varnau say:

I want to stop these long running scripts that keep messing up my computer                

I'll assume that you're running IE11 on Windows 10, so...

If this is happening only on one (or a few) website(s) there really is (by Internet Explorer standards) a long running script on the page and IE is doing you a favor by providing the "Stop script" option. Click on the Stop script button.

- -

Again, if only a site or two are affected...

It might break a desirable feature on the site, but placing a site in Restricted Sites will disable JavaScript and the script messages will stop.

Internet Options> Security> Restricted Sites...

Internet Options> Advanced, unchecking Use smooth scrolling might help.

The script may be coming from an ad. Running an ad blocker like Adblock Plus may help on some sites.

https://adblockplus.org/

- -

If you're seeing the message on all or nearly all sites, a browser add-on might be causing it.

Note: The Google Toolbar has caused script problems for a few users. See if disabling that toolbar stops the messages.

Start Internet Explorer without add-ons from Start> Programs> Accessories> System tools> Internet Explorer (no add-ons)
or

from Start> Run (or Windows logo key + r) type in iexplore.exe -extoff

(iexplore.exe<space>-extoff) then hit Enter.

If the problem goes away, an add-on is causing it.

To determine which add-on is causing the problem see this old (but still relevant) article:

IEBlog  Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

- -

Memory  usage: Internet Explorer will allow memory usage to increase to unreasonable levels Closing all iexplore processes from Task Manager and starting with a new process may be inconvenient but will get memory usage back to a reasonable level. This might help with script problems.

- -

* * * Another option: Other browsers like Edge, Chrome, Opera and Firefox are more forgiving of these script problems. Consider using another browser on affected sites.
