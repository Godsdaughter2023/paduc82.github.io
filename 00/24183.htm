c. Wait for your phone to reset, during the reset you will see gears rotating on the screen for up to 5 minutes, after which the screen goes blank for ~30 seconds and then your phone will restart.

Hi,

 

Here are some possible causes why you encountered this error based on the error code (5.7.1) you provided on your post:

  • General access denied, sender access denied; the sender of the message does not have the credentials required to complete delivery.
  • You are trying to relay your mail through another SMTP server, and that server does not allow you to relay.
  • The recipient might have mailbox delivery restrictions enabled. For example, a recipient's mailbox delivery restriction was set to receive from a Distribution List only, and the email message of a non-member is rejected and generates this error.
  • For Exchange Server 2003, a distribution list can be configured to restrict mail delivery from unauthenticated users. Mail that is sent by using an unauthenticated SMTP session is rejected.

As troubleshooting step Iyou can check system permissions and attributes for the contact, and then try to send the message again. Also, make sure that you are running Exchange 2000 Service Pack 1 or a later version to account for other known problems.

john sr wilson say:
You cannot reset a single placeholder but you can set all date placeholders to right aligned.

Sub Fix_Date()
Dim osld As Slide
Dim oshp As Shape
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.Type = msoPlaceholder Then
If oshp.PlaceholderFormat.Type = ppPlaceholderDate Then
oshp.TextFrame.TextRange.ParagraphFormat.Alignment = ppAlignRight
End If
End If
Next oshp
Next osld
End Sub

ALTERNATIVELY if you want the alignment font etc to match the slide master you could try

Sub Fix_Date()
Dim osld As Slide
Dim oshp As Shape
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If oshp.Type = msoPlaceholder Then
If Not getDatePH(osld) Is Nothing Then
getDatePH(osld.CustomLayout).PickUp
getDatePH(osld).Apply
Exit For
End If
End If
Next oshp
Next osld
End Sub

Function getDatePH(osld As Object) As Shape
Dim oplc As Shape
For Each oplc In osld.Shapes
If oplc.Type = msoPlaceholder Then
If oplc.PlaceholderFormat.Type = ppPlaceholderDate Then
Set getDatePH = oplc
Exit Function
End If
End If
Next oplc
End Function

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
On Error Resume Next
If Not Intersect(Target, Range("A1")) Is Nothing Then
Set r = Target
If r.Value = "" Then
With r
.Formula = "=CHAR(252)"
.HorizontalAlignment = xlCenter
With .Characters(Start:=1, Length:=1).Font
.Name = "Wingdings"
.Size = 14
.Color = vbGreen
End With
.Offset(, 1).Select
End With

Rows("5:15").Hidden = True

Else
r.Value = ""
r.Offset(, 1).Select
Rows("5:15").Hidden = False
End If
End If
End Sub

Hi,

add a check box (active X) in a new cell
select a cell or a group of cells and run this code.
now check box is 'anchored' in cell.

Sub Add_CheckBox_ActiveX()
'Oct 29, 2014
Selection.ClearContents
For Each r In Selection
Set sh = ActiveSheet.OLEObjects.Add(ClassType:="Forms.CheckBox.1", Link:=False, _
DisplayAsIcon:=False, Left:=r.Left, Top:=r.Top, Width:=r.Width, Height:=r.Height)
With sh
.Placement = xlMoveAndSize
'.LinkedCell = r.Offset(,1).Address
.Object.Value = False
.Object.Caption = ""
.Object.BackColor = RGB(153, 204, 255)
End With
Next
End Sub

NOTE

under one condition:
if you hide the specific row, where check box is.
and then, save/close and reopen the file,
now check box has been moved.
