If the option is set to Yes, the default view of a list or library displays results the following way.

If the list or library does not exceed the List View Threshold, libraries display folders first, then the files and lists display items from newest to oldest by the ID column.

If the list or library does exceed the List View Threshold, libraries display folders and files sorted together in alphabetical order by the name column and lists display items from oldest to newest by the ID column.

michelb62 say:

Create custom signature in Outlook 2016

I seem not to be able to (re)create a signature for my mail.

I used to use a table of 2 columns, 1 row. In the left cell I have a picture. In the right cell I have my contact information. I have a Word document in which I create the signature and I copy it in Outlook to the signature editor. But now in 2016 I have problems getting it right.

Today I learned You can change your signature which is stored on your harddisk:

C:\Users\%username%\AppData\Roaming\Microsoft\Signatures

Imagine I have named my signature 'MySignature'. It should be stored as 'MySignature.rtf'.

But when I copy my signature from my document to the rtf file and look what I got in Outlook, it is an empty signature.

What can I do?

michelb62 say:

And finally I solved it myself.

So:

  1. Create the signature you want in Word. Copy it.
  2. Open the signature editor in Outlook (Outlook: File --> Options --> Mail --> Signature).
  3. Paste the signature to the editor.
  4. Test it in a new email
  5. When you are not quite happy with the result, edit your new signature in the signature rtf file, located in:

C:\Users\%username%\AppData\Roaming\Microsoft\Signatures
