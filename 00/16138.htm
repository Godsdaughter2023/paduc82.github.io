=DATEVALUE("4/1/"&TheYear)-WEEKDAY(DATEVALUE("4/1/"&TheYear))+1

A robust formula for AprSun1  should be

=DATEVALUE("1-Apr-"&TheYear)-WEEKDAY(DATEVALUE("1-Apr-"&TheYear))+1

Another way around the problem is to update only fields that aren't TOCs:

    Dim fld As Field
    For Each fld In ActiveDocument.Fields
        If fld.Type <> wdFieldTOC Then fld.Update
    Next fld

To answer your original question, this code will prevent the document from blinking, although the ribbon area will still blink:

    Application.ScreenUpdating = False
    With ActiveDocument

        .ClosePrintPreview
    End With
    Application.ScreenUpdating = True

        objCC.Title = "Select"
        If objCC.ShowingPlaceholderText Then _

        For j = LBound(arrEntries) To UBound(arrEntries)

        Next j
