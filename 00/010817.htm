marksingletary say:

I had the same issue and finally came up with a fix.

XSN files are just cab files.

  1. Create a copy of the2010 xsn file.
  2. Change the extension of the copy to .cab
  3. Extract the whole cab file to a folder..
  4. Use a text editor to edit the manifest.xsf file
  5. Locate the <xsf:solutionProperties element and set the allowCustomization attribute to ‚Äúyes‚Äù
application="InfoPath Design Mode">
    <xsf:solutionProperties
lastOpenView="view1.xsl"
lastVersionNeedingTransform="1.3.0.0" fullyEditableNamespace="http://schemas.microsoft.com/office/infopath/2003/myXSD/2009-09-25T15:31:22"
automaticallyCreateNodes="no"

allowCustomization="yes"

scriptLanguage="managedCode">
</xsf:solutionProperties>
                </xsf:applicationParameters>

  1. Save the file back into the extracted folder.
  2. Use the IEXPRESS wizard to build a new CAB
    1. Open a CMD window and type IEXPRESS
    2. Choose Create New Self Extraction Directive file. Click Next
    3. Choose Create Compressed File Only. Click Next
    4. Click Add and select all the files from the extracted folder. Click Next
    5. Check the STORE FILES USING LONG FILENAME box.  Click browse and navigate to where you want the new cab to be built and give the cab a name.  Click next.
    6. Click next. Click Next. Click Finish
    7. Go to where you stored the new cab.  Rename the cab file with an xsn extension.
    8. Right Click the xsn file and select design to open the InfoPath designer.
      ( Notice there is no protection warning.)
    9. If the original template was digitally signed, you will need to sign it again.
    10. Save the file.

You now have an Unprotected Template.

mel sampat [msft] say: Hi CoyoteBob
You can't just move an entire folder to the storage card, because the system will think it's still there and apps will continue looking for it. However, what you can do is move as much of the contents of the folder to your storage card as you like. If you have any documents, photos or music in the My Documents folder, simply move them into a new folder on your storage card, but leave the original folder as is. Same for apps and games - you cannot simply move them after they have already been installed. However, you can uninstall them first (from Start > System > Remove Programs), and then reinstall them to the storage card.

If you want to go further, there are various 3rd party utilities you can try that will clean up your file system automatically. One that comes to mind is MemMaid, although I'm sure there are numerous others (both paid and free).
