Interface Continuity - Microsoft Community

jonathaningham say:

Interface Continuity

I've been using Windows 10 for a few days now and think Microsoft is off to a great start (no pun intended). While I've only been running it in Virtual Box, the performance has been excellent. I have always been a supporter of new, big and bold ideas when it comes to software. I even admired the boldness of Windows 8's new look. The problem I have always had with both Windows 8 and now Windows 10 is the lack of interface continuity. While the OS is much more familiar than Windows 8, there are still a few design elements that I find detract from the overall user experience. For instance, the control panel and PC Settings - I believe these should be rolled into a new, complimentary element. As it stands, it feels like the new modern UI and classic UI are battling it out rather than melding together as the OS evolves. I realize it is important to make transitions gradually so everyday users are not completely lost. That being said, I don't see why control panel even needs to exist as a separate entity from PC settings. Lastly - and this is just a personal quirk - I would like to see the classic icons (documents, pictures, etc.) within the file explorer revamped to match the modern UI. There is a delicate balance to moving users to a new OS without jarring them and I personally would argue that interface continuity is key for Windows 10's success.  I am excited to see where Microsoft goes with this new OS. Without question I believe they're on the right track to bridging the bright new UI of Windows 8 with the familiarity of previous versions.
