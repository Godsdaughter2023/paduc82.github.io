Hi Surveyor,

 

There are 2 locations for the desktop in Windows 7.

C:\Users\Public\Desktop is the location for All Users. Any shortcut icon in this folder will appear on every user desktop.

C:\Users\user name\Desktop is the location for each individual user. Any shortcut icon in this folder will only appear on that individual user desktop.

To get the result you want, if a program installation places an icon in the “All users” desktop folder, you can simply move that icon from that folder to your individual desktop folder. i.e You can remove a shortcut from an individual user's desktop on either OS by moving the icon from the All Users (on Windows 7, Public) Desktop directory into the Desktop directory(s) for the user(s) you want.
