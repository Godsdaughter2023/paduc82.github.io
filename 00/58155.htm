canadian tech say:

"Backing up" the OS is useless.  You can not use it.  If you need to restore your system from scratch - meaning re-install Windows, you have to do that from a Win7 install disk or by Factory restore.

The same applies to ALL application programs.  You can not restore them from backup.   They must be installed from their original source along with their activation codes.

The only way to "backup" the OS is to do a System Image.  That kind of a backup can then be used to restore the entire system with data, programs and every single bit to a state that is identical to its state when the Image was created.  The System Image is a gigantic file and you can not restore any part of it.  Only the entire image. Restoring an image erases everything on the disk and replaces it in entirety with the image.

I do recommend System Image creation, but that should be done immediately following OS load, all updates and drivers, activation, etc.  BEFORE any data or applications like Antivirus are added.  That kind of a System Image is typically done on 4 or so DVD +R's.  That image is an insurance policy that enables you to re-image a corrupted hard drive or even a brand new drive.  Its beauty (as an insurance policy) is that you START with a Complete windows installation with every thing done but updates since its creation.  Then you install your programs and copy your backed-up data.  One of the great pluses is that you do not have to have a Win7 install disk or an activation code, because it is already done.

The likely reason you run into the message that is stopping you is that the operating system on your system that is performing the backup has the file in use and it can not copy it.

Bottom line:  Backup your DATA and possibly your settings.

Backup software makes what should be a simple task way too complicated and locks you into dependencies you just do not need.

considerate_guy say:

Photoshop Elements stores the order of the photos separately in an album description. Any other program in Windows will have no understanding of this order that you have chosen. The only exception is if Photoshop Elements gives you a way to rename the files, say by pre-pending the file name with a sort number.

Movie Maker can accept a batch of images. Click on the 'Add videos and photos' button and then you can select them by clicking on the first photo, holding down Shift and then selecting the last photo. By default, this will be in file name order. You can try other sorting options by using a detail view and then sorting the photos. For example, you might try date taken instead.

However, if the order doesn't match with what you would like, you can always rearrange them in the storyboard/project area as needed.
