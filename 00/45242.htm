¡firedog say:

Do you mean that by merely signing into my MS Account[1] that Windows Live Mail contacts are automatically synchronized with the MS cloud?

You have at least two different contacts lists in Windows Live Mail: 
 

  • When you're Home tab of the ribbon reads Sign in and the contacts list you see is the default list.
  • When you are signed in with a Microsoft Account, the right-most button on the Home tab of ribbon reads something other than Sign in and the contacts list you see is the one that belongs to the account you're signed in with. The list synchronizes with the one at contacts.live.com. Any change you make to this list in Windows Live Mail will propagate to the online list and vice versa.
  • Use the button to change your signed-in state and hence the contacts list you see. If you sign in with a different Microsoft Account, you'll see a different contacts list.
  • There is no connection between the different contacts lists you see in Windows Live Mail.
  • All of your contacts lists are stored locally on your computer.

I assume this automatically occupies some space on my allocation on OneDrive or is it somewhere else? 

No. There is a storage allocation for contacts separate from OneDrive included with the Microsoft Account.

... what else might be stored on OneDrive and resynchronized when I sign into my MS Account.

None of this is stored on OneDrive. The calendar will also synchronize with the one at calendar.live.com when you sign in.

Does this also apply if I don't use my MS Account but a Local Windows Account to log into my computer?

Can I get the same effect if I sign into e.g. http://answers.microsoft.com from my browser? 

This has nothing to do with your Windows user account or any other use of your Microsoft Account. We're only talking about signing in in Windows Live Mail.

I took the precaution of exporting my contacts via .csv

Here's a suggestion:

  1. Open the People page by visiting contacts.live.com using your browser (you may find that Internet Explorer works best for this sort of thing) and sign in with your Microsoft Account. 
  2. Use the Import contacts function to import your CSV file.
  3. Use the Clean up contacts function under Manage to merge any duplicates.
  4. In Windows Live Mail, switch to the contacts window (ctrl-3), then use the right-most button on the Home tab of the ribbon to Sign out of Windows Live Mail if you're already signed in.
  5. When you see the default contacts list, use the right-most button on the Home tab of the ribbon to Sign in. This should load the cleaned-up contacts list you created at (3).

I don't see a good way to deal with this other than manually looking through them to find omissions or additions and creating one master in Windows Live Mail contacts.

Sounds like a plan.

Depends what you want to know. Asking questions at this site will usually produce an answer if there is one.
