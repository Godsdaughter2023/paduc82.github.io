windows 10 says i have 6.4 gb of ram comitted when i have 5.75 gb - Microsoft Community

shoopnoop say:

windows 10 says i have 6.4 gb of ram comitted when i have 5.75 gb installed

the task manager says i have more ram than installed

it also seems to use a good amount of ram can anyone help

i have 6gb ram installed and 5.75 useable 

a. user say:

That's correct. Committed RAM is not the same thing as physical RAM.

When your applications need more RAM than your computer has, Windows takes some of what's currently loaded in RAM and moves it to a temporary storage area on your hard drive. This is called swapping or paging, depending on the way it's done. When physical RAM needs the data that was moved out, it gets moved back in. That's how Windows can appear to be using more RAM than you actually have.

Committed RAM is the total amount of data committed to RAM whether it's located in physical RAM or on vacation on your hard drive.

Q. If my customer asks me to upgrade a PC with new hardware components, when is a new operating system needed? When would the PC be considered "new"?

A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. If the motherboard is upgraded or replaced for reasons other than a defect, then a new computer has been created. If the motherboard is replaced because it is defective, you do not need to acquire a new operating system license for the PC as long as the replacement motherboard is the same make/model or the same manufacturer's replacement/equivalent, as defined by the manufacturer's warranty.
