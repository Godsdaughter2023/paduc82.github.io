At the end of each Section of the document, insert a Bookmark Section# where # is the number of the Section.

Then in the Headers, use the following field construction for the Page X of XX

Page { = { PAGE } - { PAGEREF Section# } } of { SECTIONPAGES }

You must use CTRL+F9 to insert each pair of field delimiters and use ALT+F9 to toggle off their display.

You will need to unlink the Header in each Section from that in the previous Section.

Use the normal page numbering { PAGE } in the Footers

the BEST way to determine if an application has ended and returned its result is like this:

  while(1)

  {

      DWORD dwExitCode;

      if(WaitForSingleObject(hProcess, 5) != WAIT_TIMEOUT)
      {
        if(!GetExitCodeProcess(hProcess, &dwExitCode) ||

        {

        }
        else // if(dwExitCode == STILL_ACTIVE)
        {
          Sleep(1); // lose time slice, continue loop
        }
      }

virfranz man say:

Hello,

You can hide the comments on a Word file when saving as a PDF file by using the No Markup option. Kindly follow the steps below on how to hide comments:

  1. Open Word file with comments.
  2. Go to Review tab.
  3. Under Tracking pane, click the All Markup drop-down option.
  4. Select No Markup.

You can also hide the comments on the said file.

  1. Follow the steps 1 and 2 above.
  2. Under Tracking pane, click the Show Markup drop-down.
  3. Uncheck Comments.

stefan blom say:

It isn't exactly what you are looking for, but you could run a macro that hides comments and inserts comment text directly in the document, and then the macro can print without any highlights visible.

The below macro is a very basic approach but it works. You'll have to close the file without saving to prevent the "in line" comments from being saved in the document.

Sub PrintCommentsInlineWithText()
'Basic macro which inserts comment text "in line with text"
'and hides comment balloons before printing the document
'Stefan Blom, MVP, Januari 2018
Dim i As Comment
For Each i In ActiveDocument.Comments
i.Reference.InsertAfter " [" & i.Range.Text & "]"
Next i
ActiveWindow.View.MarkupMode = wdInLineRevisions
ActiveDocument.PrintOut
End Sub
