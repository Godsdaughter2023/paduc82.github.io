Hotmail has a feature that prevents losing email messages if you accidentally set up POP3 to delete them after reading them on the server. Here’s how it works:

tom van stiphout say:

As you found out, the runtime has a few limitations. This is one of them. However you can get the menu back by creating your own. In my startup code I write:

If SysCmd(acSysCmdRuntime) = True Then Application.ShortcutMenuBar = "macCreateRightClickMenu"

Then I create that macro with one line, that uses the AddMenu action. It refers to a second macro which implements the options, such as Cut/Copy/Paste, Filter, etc. Those all use a RunCommand action.
