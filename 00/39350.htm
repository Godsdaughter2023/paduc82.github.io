I've checked and find no instance in that directory on my computer. However I find 15 instances in C:\Windows\winsxs\amd64_microsoft-windows-consolehost_

Does Win7 Ultimate not use or need conhost?


Hi

 

Yes, the Conhost.exe file is needed in Windows 7.

 

It was included in Windows 7 to solve a couple of problems that appeared in Windows XP and Windows Vista.

 

You can see a good explanation of these issues and the reason the conhost.exe is needed in the following article.

 

What is conhost.exe and Why Is It Running?:

http://www.howtogeek.com/howto/4996/what-is-conhost.exe-and-why-is-it-running/

 

 

If you are sure the Conhost.exe is not appearing in the System32 folder, perform the following test.

 

1. Click Start and type  cmd.exe  in the Search box.

 

2. In the Results, right click the Cmd.exe file and select the Run as Administrator option. This will open a Command Prompt window.

 

3. Leave the Command Prompt window open. Right click the Taskbar and select Task Manager.

 

4. In Task Manager, select the Show Processes from all users button at the bottom of the window.

 

5. Make sure the Processes Tab is selected and check to see if you can find an instance of  the Conhost.exe file in the Image Name column.

 


Let us know the results.
