jeeped say:
I have a predicament of how to run a kind of nested IF statement based on four conditions. And the bad part is that I will have probably eight metrics to do this on so once I get one, I will need to do it 7 more times for different results ...

I don't know about the patterns for the other seven formulas, but this seems to cover the conditions you've set here.

=LOOKUP(D2*100-IFERROR(MATCH(C2,{"Month Three","Month Two","Month One"},FALSE)*IF(D2*100<6,0,2),0),{-1E+99,0,84,86,88,90},{0,1,2,3,4,5})

... Therefore I need something even my amateur brain can understand and replicate ...

I ran into some floating point errors when dealing with the percentages so I multiplied them up to integers. The,
MATCH(C2,{"Month Three","Month Two","Month One"},FALSE)
,,, returns 1, 2 or 3 depending on the value in C2. Normal is caught as 0 (default) by the wrapping IFERROR(). Your pattern broke at 0 since there wasn't a 0, 2, 4, 6 stagger at zero so I had to add the IF() statement to cover that anomaly. The combined IFERROR(MATCH()*IF()) functions produce a modified D2 value which is compared to a base set of values with LOOKUP() to return a 0, 1, 2, 3, 4 or 5.

Broken down into component sections, it isn't too bad. Post back with any questions.

kincavel say:

How to create new data file in Outlook - Office 365

Hi, I'm running Office 365 with Outlook on Windows 10, Surface Pro 3 and no longer seem to be able to create new data files without them being attached to an email account. I was always able to do this previously with Windows XP so that I can save emails from various email accounts into one folder.

Does anyone know if this is still possible?

Many thanks

a. user say:

Hello,

As per your query, you can create a PST file without creating a new account.

Please follow the steps and check if you are able to resolve your query. Open Outlook > click on File > Account Settings > Account Settings > under Data files click on Add. This will add a data file without linking it to any email account.

Please post to us if you need any further assistance with Outlook and we will assist you further.

Thank you.
