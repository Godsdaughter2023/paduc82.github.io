john mcghie say:
Sadly, that's right.  The Table of Figures will indeed not include anything that is in a TextBox.

It is a bug in Word.  It's a design bug: TextBoxes are not part of the Main Text Story, so the TOC generator (which does the Table of Figures) cannot see them.

Word 2010 is one work-around.  The other is to convert the text boxes to "Frames".  Which you cannot do in Word 2011 unless you downgrade the document to .doc format (and don't risk that).

Another workaround is to copy the caption and paste it into the text right next to the picture, then format it as "hidden" text.

Not very clever, is it?  Best not to use floating pictures, so you don't get the problem.

Or:  You could ask yourself why you NEED a List of Figures and List of Tables!  I have removed them from all the documents I produce.  There has never been any usability study evidence of "readers" actually using them!  Not ever!

Back in the ancient days of moveable type when book pages were made up with a ruler and Stanley knife, the LOF and LOT were used to help the compositors get the tables and figures on the correct pages.  I dimly remember those days, because I am so old I actually worked with that briefly.  In modern times when documents are entirely produced electronically (and often, consumed that way...) there is absolutely no need for them: they just waste time and space.

Hope this helps

Method to total up section page numbers in Page X of XX format in - Microsoft Community

wordweary say:

Method to total up section page numbers in Page X of XX format in multisection document?

I have a multi-sectioned report document with TOC, switching page orientation (forcing sections), and appendices.

 

I desperately need a method to correctly show the totalled page numbers for only certain sections of a document.  Is there a way to formulate either {NUMPAGES} or  {SECTIONPAGE} to do this?

 

ie. Sections 1, 2, 3 totalled showing as Page X of XX (total of Sections 1,2,3)

Sections 4 showing as Page 1 of XX (Section 4 only total)

Section 5 showing as Page 1 of XX (Section 5 only total), etc

 

I am using Word 2010 on Windows 7 Enterprise, and don't have much experience with field codes. I have odd/even pages and different first page in the doc.  MUCH APPRECIATED!!

jay freedman say:
If the arrangement you listed is an accurate representation of the document (continuous page numbers through the first three sections, then page numbering restarts at 1 for the next two sections), then the scheme is fairly simple.

Place a bookmark on the last page of Section 3, naming it (for example) Sec3LastPg. In the headers or footers of Sections 1, 2, and 3, insert the expression

   Page { PAGE } of { PAGEREF Sec3LastPg }

The field markers { } must be created by pressing Ctrl+F9; if you try just typing them, they won't make working fields. Select the whole line and press F9 to update the fields, collapsing the codes and showing the numbers.

For the headers/footers of Sections 4 and 5, you can use the usual expression

   Page { PAGE } of { SECTIONPAGES }

because those sections don't have page numbers that continue into following sections.

Where you have other consecutively numbered multiple sections, use the same idea as in Sections 1 to 3, just using a different name for the bookmark and using that name in the PAGEREF field.

forcing capital letter on first cell text entry - Microsoft Community

brilliantb say:

forcing capital letter on first cell text entry

I need to make the first letter of the first word in a cell always a capital - can that be done?
gary's student say:

Say we are entering text in column A and we want to autoconvert the first letter of the first word to upper case.

 

Put this event macro in the worksheet code area:

 

Private Sub Worksheet_Change(ByVal Target As Range)
Dim t As Range, A As Range
Dim s As String
Set t = Target
Set A = Range("A:A")
If Intersect(t, A) Is Nothing Then Exit Sub
s = t.Value
s1 = UCase(Left(s, 1))
Application.EnableEvents = False
t.Value = s1 & Mid(s, 2, Len(s) - 1)
Application.EnableEvents = True
End Sub

 

  Because it is worksheet code, it is very easy to install and automatic to use:   1. right-click the tab name near the bottom of the Excel window 2. select View Code - this brings up a VBE window 3. paste the stuff in and close the VBE window   If you have any concerns, first try it on a trial worksheet.   If you save the workbook, the macro will be saved with it.     To remove the macro:   1. bring up the VBE windows as above 2. clear the code out 3. close the VBE window   To learn more about macros in general, see:   http://www.mvps.org/dmcritchie/excel/getstarted.htm   To learn more about Event Macros (worksheet code), see:   http://www.mvps.org/dmcritchie/excel/event.htm
