Taking a look at  Mike H..'s excellent formula, replacing the 2 with a progressive COUNTIF() would adjust for an unknown number of identical entries.

               

The standard formula in E2 is,
=IFERROR(LARGE($A$2:$A$10,ROWS($1:1)),"")
... which only needs Enter↵ and the array formula in F2 is,
=INDEX($B$2:$B$999,SMALL(IF($A$2:$A$999=E2,ROW($A$2:$A$999)-ROW(E$2)+1),COUNTIF(E$2:E2,E2)))

Hi,

 

So you want to look up 100 in Col A and return the second match and then return Col B. Try this ARRAY formula, the very last 2 is what makes it return the second match, you can change this for other matches

 

Lookup value (100) goes in e1

 

=INDEX($B$1:$B$20,SMALL(IF($A$1:$A$20=$E$1,ROW($A$1:$A$20)-ROW($E$1)+1),2))
