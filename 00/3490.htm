ruwim.b say:

The new Microsoft Skype for Windows 10 (Skype UWP) application is using local storage for chat messages in a way identical to how this was the case with Skype for Windows Desktop. In both cases, chat messages are saved in the main.db file on your computer.

 

When you first time start the new Skype UWP application, only the about latest 30 days messages are transplanted from Cloud servers to your local main.db file. The consequent messages will also be saved in the main.db file and kept there forever.

 

If you want to transfer the old messages from the desktop application to Skype UWP, then the only what you need is to transfer the main.db file from desktop Skype to the location where this database file is located in the UWP application.
