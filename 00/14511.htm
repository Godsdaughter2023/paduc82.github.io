andreas killer say:

I'll try to answer your second question first:

Correct, one is an Active-X object and the other one is a DropDown object. The DropDown is a relict from ancient times and supported for backward compatibly only. But many users use such old controls, because they are easier to use if you have no VBA knowledge.

On the other hand, the new Active-X objects (introduced with Excel 2000) are much more comfortable in there settings, supports events, ..., but also requires (often) code.

Now your first question:

Both controls are objects and if we want to access them, we must get a pointer to it first (using the SET statement).

Active-X objects are public in a sheet, that's why you can say i.E. "Combobox1.ListRows = 60" in the code module of sheet.

Below is an example how to get a pointer to the objects from anywhere.

Make a new file, add a DropDown and a ComboBox to a sheet, run the macro below and if the code stops, follow the instructions in the comment.

So you get an impression what objects are and how to find properties that you did not noticed before.

Andreas.

 

Sub Test()
  Dim MyDropDown As DropDown
  Dim MyComboBox As MSForms.ComboBox
 
  Dim S As Shape
  Dim i As Integer
 
  'Search all objects in the sheet
  For Each S In ActiveSheet.Shapes
    'Mark the variable S above, press Shift-F9, click Add and explore the properties by click the +
    'Then continue the macro step by step with F8 and compare the code with the object properties
    Stop
    'What type is it?
    Select Case S.Type
      'Form control
      Case msoFormControl
        'Is it a dropdown?
        If S.FormControlType = xlDropDown Then
          'Yes, get the object
          Set MyDropDown = S.OLEFormat.Object
          'Remove all items
          MyDropDown.RemoveAllItems
          'Add some items
          For i = 1 To 1000
            MyDropDown.AddItem i
          Next
          'Increase the dropdown
          MyDropDown.DropDownLines = 60
        End If
       
      'Active-X control
      Case msoOLEControlObject
        'Is it a combobox?
        If InStr(1, S.OLEFormat.ProgId, ".ComboBox.", vbTextCompare) > 0 Then
          'Yes, get the object
          Set MyComboBox = S.OLEFormat.Object.Object
          'Remove all items
          Do While MyComboBox.ListCount > 0
            MyComboBox.RemoveItem 0
          Loop
          'Add some items
          For i = 1 To 1000
            MyComboBox.AddItem i
          Next
          'Increase the dropdown
          MyComboBox.ListRows = 60
        End If
    End Select
  Next
End Sub
