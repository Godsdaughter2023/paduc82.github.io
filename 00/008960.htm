In the Customize Keyboard dialog, it is possible to remove or reassign shortcuts for specific documents. This may have happened -- although I'm at a loss to explain how -- in some of your documents.

To check out this theory, open one of the documents that don't save when you press Ctrl+S. Go to File > Options > Customize Ribbon and click the Customize button at the bottom left of the dialog.

In the Customize Keyboard dialog, first set the "Save changes in" dropdown to the name of the document --

In the Categories list, select File. In the Commands list, select FileSave. If Ctrl+S doesn't appear in the Current Keys box (along with Shift+F12 and Alt+Shift+F2, which are other default assignments for the same command), then click in the "Press new shortcut key" box, press Ctrl+S, and click the Assign button. OK both dialogs and save the document.

If that works, you need to repeat it in each document that has the problem.
