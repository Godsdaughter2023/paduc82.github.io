There are 2 locations for the desktop in Windows 8.

C:\Users\Public\Desktop is the location for All Users. Any shortcut icon in this folder will appear on every user desktop.

C:\Users\user name\Desktop is the location for each individual user. Any shortcut icon in this folder will only appear on that individual user desktop.

If a program installation places an icon in the “All users” desktop folder, you can simply move that icon from that folder to your individual desktop folder. i.e. You can remove a shortcut from an individual user's desktop on either OS by moving the icon from the All Users (on Windows 8, Public) Desktop directory into the Desktop directory(s) for the user(s) you want.

Note: You may get an administrator prompt when you add or remove icons from the all users’ location.
