roady [mvp] say:

You'll need to create an Advanced Filter based on the Message Class.

The following filter criteria will result in only Meeting related messages to be shown:

Field: Message Class

Condition: Contains

Value: IPM.Schedule.Meeting

Note that deleting invites will not remove the tentatively accepted meeting from the Calendar, you'll need to decline it instead.
