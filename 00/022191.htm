billphillipsnc say:

Hello Levtov, (read with caution, more than you ever wanted to know, I'm sure :)

I must explain that I am not a computer tech guy, just a regular guy trying to understand this as you seem to be. So, consider my answers as a good effort, but not entirely reliable. All these comments are based on using MS Outlook for email, contacts, and calendar. And, you asked, "how do I know if I'm using POP or IMAP?" In Outlook, you should find it under Account Settings and the Email tab. One must be fearless to make changes in these settings. The results can be unexpected.

POP, IMAP, and Exchange Active Sync (EAS), are email protocols. (The word "protocol" meant a set of "procedural rules" for social organizations before the tech community adopted it for email.)  As I understand it, the different protocols are different sets of rules by which email is transferred between your home computer and the server in some remote location. Say, for instance, the protocols are the US Post Office, UPS, and Fed Ex.  You gotta pick ONE to get your mail, or send something.

Each does it in a different way with different results. Exchange Active Sync is not a protocol you buy, it's just Microsoft's latest server set up for individuals. It runs your email, contacts, calendar, etc through Microsoft's Outlook.com.  The advantage, I think, is that EAS will synchronize your email, contacts, and calendar between a home computer and a laptop, and hopefully a 3rd device (a mobile phone.)  POP and IMAP are older protocols that either do not sync between devices, or do not do it very well.

The presumed advantage of using EAS is that any entry you make on any of your three devices automatically is entered on the other two. This of course would only happen when there is an internet connection for all three devices. If there is no connection, the entry on one device "waits" until it recognizes a connection, then it happens. andreas killer say:

Instead of Sheet4 use an Autofilter in Sheet3.

Insert (at min.)  one row at the top of the sheet and apply the Autofilter to your data.

Copy the code below into the code module of the sheet.

Write e.g. "a*" (without the "") in the 1st row above the name column to show all names that begins with an "a".

Andreas.

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim R As Range
  'Only when an Autofilter is present
  If Not Me.AutoFilterMode Then Exit Sub
  'Only when a cell in row 1 is changed
  Set Target = Intersect(Target, Rows(1))
  If Target Is Nothing Then Exit Sub
  'Only when a cell within the data range is changed
  Set Target = Intersect(Target, Me.AutoFilter.Range.EntireColumn)
  If Target Is Nothing Then Exit Sub
 
  With Me.AutoFilter.Range
    'In each changed cell
    For Each R In Target
      If IsEmpty(R) Then
        'Remove the filter
        .AutoFilter R.Column
      Else
        'Set the filter in this column
        .AutoFilter R.Column - .Column + 1, R.Value
      End If
    Next
  End With
End Sub
