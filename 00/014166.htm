Hello,

I want to create a document with Word 2016 that has two different sections with different custom margins. However, it seems that Word only allows one set of custom margins in a single document, which means that when I change the custom margins for one section, it also changes them for the other section. Is there any way to get around this? Not sure if I'm able to set up a document template where I'm able to incorporate two different sets of custom margins that I've been able to specify myself?

Any help would be greatly appreciated!

doug robbins - mvp office apps & services (word) say:

You should be able to achieve what you want by having Different Odd and Even Headers\Footers and in the headers of the second section of the document (which should have equal margins of 2 cm in your case), insert textboxes into those headers that have their layout set for the Text to wrap around them, and position and size them to cause the text begin\end at what would have been the mirrored margins if you could have had them in just that section of the document.

The headers in the second section will need to be unlinked from those in the first section before the textboxes are inserted.

In the following screen shot, the text boxes are shown with an outline so that you can see them, but which you would want to turn off in your case

DKU-2 and CA-53 are cables for "real" USB devices (the phone is a USB device from Windows point of view)

DKU-5 and CA-42 cables are for phones like 3220 that connect to Windows as serial port (virtual COM is assigned) devices, the cables have imbedded emulation firmware to allow the cable to be connected to physical USB port in PC.
