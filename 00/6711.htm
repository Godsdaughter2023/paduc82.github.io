Unsatisfactory Solutions:

     Shrinking the pictures to fit.

     Increasing the size of the table cells to fit.

     Cropping the pictures before importing them.

Is there something I've missed, or is this just arbitrarily not allowed in Word? I feel like it should be an easy thing to do.

jasonszulc say:

Excellent, Thanks Doug :)

For anyone else that wants to do this:

Create a rectangle and set the size in Drawing Tools -> Format.

Right Click the shape and select Format Shape.

On the panel on the right select Shape Options -> Fill -> Picture or Texture Fill.

Insert the picture (e.g. from file) - It will be shrunk to fit.

Check Tile Picture As Texture - It will be full size and fitted to the top left corner.

Select Alignment = Centre - It should now be full sized, centred and cropped to fit.

Change the shape's border to the right look (e.g. black line, 1pt thick)

Change the Shape's Wrap Text to In Line and then copy and paste the shape to give a table structure, it will automatically go in to the right place. Trying to put them in a table is likely to cause headaches.

Select each shape and use Shape Options -> Fill -> Picture or Texture Fill to change the picture file.

Check Tile Picture As Texture - This is automatically unselected, but Alignment =Centre is kept.

It's disappointing that the table cells aren't set up to do this as well, from an underlying coding point of view table cells are just shapes. Oh well, job done.

For anybody who needs to do this but cannot access the PDF file for some reason, here are the basics:

Disk Preparation using Linux

 

This is a ‘quick, down, and dirty’ guide on how to use Linux as a utility to wipe out Drive C: on a Surface Pro series tablet.  The particular Linux distribution used has Surface basic drivers for which should support the keyboard/touchpad cover and has been tested on a Surface Pro 3.

Materials needed:

  • 2 USB pendrives – one 8GB or higher and the second 16GB or higher and both formatted in FAT32 format.
  • Recovery image for your particular Surface Pro model (available here: https://www.microsoft.com/surface/en-us/support/warranty-service-and-recovery/downloadablerecoveryimage)
  • ISO image of Ubuntu Desktop Linux.  Available here: https://www.ubuntu.com/desktop
  • Rufus Tool to burn ISO image onto 8GB pendrive.  Available here: https://rufus.akeo.ie/

Preparation of media:

  • Insert 8GB pendrive in USB port and launch the Rufus Tool located in your Downloads folder.  Change the “FreeDOS” setting to “ISO” and click on the folder icon to its right and navigate to  the Ubuntu ISO image and burn it.  When finished, remove the pendrive and exit the Rufus tool.
  • Insert 16GB pendrive and open the File Manager and navigate to the Downloads folder and double click on the “ZIP” file which is the Windows 10 recovery image for your machine.  Within  the unzip folder, go CTRL + A (highlight all}, CTRL + C (Copy), navigate to the  pendrive folder and go CTRL + V (Paste).  When finished, remove the pendrive.

Booting Linux on Surface Pro tablet:

  • Insert the 8GB pendrive and press and hold down the “Volume Down” button at the same time, hold down the “Power” button for about 15 seconds and a blue screen with white text will appear and choose the second option to recover from external drive.  Ubuntu should come up with a black screen with white text and select “Try Ubuntu without installing”.  Wait until the Unity desktop appears.  Click on the spiral icon on the upper left corner to get rid of the panel in the center of the screen.  Press that icon again and at the top of the screen panel is a search box and type in “disks” (without quotes) and you should see the “Disks” utility appear just below it.  Click on it.
  • The Disk Manager will appear and on the left side column, it should display the SSD card in highlight.  You can tell by the storage capacity.  Now, click on that hamburger icon on the upper right corner of the program frame and click on “Format Disk”.  When finished, shut down the tablet by clicking  on the upper right corner of the taskbar (top) and hit the power icon.  Wait until the machine  is shut down completely.

Restoring Windows:

Not much is needed to be done here except to remove that 8GB pendrive and replace it with the 16GB pendrive and use the volume down, power button sequence again to boot up to the recovery screen and choose the second option again.  This time, it will start the Windows 10 installation.

Run all updates when complete and then add your data and additional programs to the system.
