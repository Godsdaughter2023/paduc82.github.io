sam wang msft say: Hi DamianJHT,

Firstly, please be noted that users can’t log into a shared mailbox directly using Outlook as it doesn’t have its own user name or password. You must first assign permissions (the Full Access permission) to the user to access the shared mailbox, and then they can access it using Outlook. And it’s the same case when multi-factor authentication is enabled on the user’s account. For more details, please refer to this article:
https://support.office.com/en-us/article/Create-shared-mailboxes-in-Office-365-871a246d-3acd-4bba-948e-5de8be0544c9

Moreover, here are two recommended methods to access a shared mailbox in the Outlook client:
1. Grant the full access permission of the shared mailboxes to you and let them appear automatically via automapping.
2. Or grant the full access permission of the shared mailboxes to you and remove automapping for the shared mailbox, then add them manually as additional mailboxes (In FILE> Account settings> double-click your account> More Settings> Advanced> Add) in the Outlook client.

You can refer to either one and let us know how it works. Just let me know if anything is unclear.
