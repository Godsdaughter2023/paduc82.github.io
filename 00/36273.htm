I have finally managed to upgrade to Fall Creators Update build 16299.

Here is the process which eventually worked.  It involved finding the "incompatible" file which was preventing the upgrade from proceeding.

Open Event Viewer (Find by typing "Event Viewer" in bottom left search box)
Click Windows Logs (left hand side)
Click Application (left hand side)
Go to Action menu
Find "winsetupdiag02"
Click "Event Properties" in right window.
Scroll down in box and double click on SetupAct.log
This should give clues (lots of them!!) about the problem.
In my case I found "Manual uninstall required for vsdatant.sys".
I opened a File Explorer window, click on "This PC" typed the filename in the search box and found the file that was causing the problem.
Deleted it.
Tried upgrade again.
