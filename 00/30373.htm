Plenty. I know, because I've done it all. Every mistake you've made and plenty more that you would never make because, seriously, who could be that dumb? Here are some things I've learned along the way.

Updates to Windows presume that they are being installed on a fully supported and properly configured computer. If either of those are not present, the update can fail spectacularly.

If your computer has hardware that isn't supported on Windows 10, there's no way to know what will happen when an upgrade gets installed, because the hardware manufacturer hasn't tested the hardware on Windows 10 and hasn't made any attempt to keep their driver up to date. Sometimes unsupported hardware can work just fine for years, like my ancient graphics card (shhh!) but then along comes an update that the hardware can no longer deal with and whamo.

The same is true of unsupported software applications, although Windows has the ability to run unsupported software and it may work fine or it may not or it make cause unpredictable problems in the future. As in the case of unsupported hardware, there's very little that Microsoft can do about this because they didn't make the product in the first place.

Then there is the issue of all the crazy things people do to their computers. We've seen it all here. An update can't possibly anticipate all the messed up configurations it has to contend with.

This is what I learned from the 'smart' computer users; the experts, the professionals who don't seem have serious trouble running their computers: A clean install on supported hardware, configured as Microsoft supports, running with supported software, always works.

That said, nothing made by human beings will ever be perfect so all software has bugs, including piles of bugs that won't be discovered for a long time. But these bugs don't prevent people from doing the simplest things on their computers. That comes from lack of knowledge and experience.

When I installed Windows for the first time, I didn't upgrade what I had before - I did a clean install. But before my clean install I checked to be sure my hardware would run properly on Windows 10. After my clean I made sure to only install supported versions of software. After that, I did not make any changes to my configuration unless I thoroughly checked it out first to make I was doing the right thing the right way.

You can call me the Pollyanna of the PC, but I don't have any trouble with my computer. And daily backups are my insurance policy.

cody gatewood say:

Changing tabs using the Track pad

On my Chromebook I can switch between tabs by using three fingers to swipe left or right. I want to be able to do the same on my Dell Windows 10 but cannot figure out how. Anyone know if this is possible and if so how to do it?

bravoman21 say:
There is a way to make this work using just the features provided by Windows, but you will need to have a precision trackpad in order for this to work (most Dell computers do).

To determine if you have a Precision Trackpad: open your start menu and type "Trackpad settings" and hit enter. The Trackpad settings window should appear. If you have a precision trackpad you should see the phrase "Your PC has a precision touchpad." at the top of the Trackpad settings window.

If you do have a precision trackpad, look over towards the far right of that Trackpad settings window, select the "Advanced gesture configuration" setting.

From there, navigate down to the Left and Right options listed under "Configure your three finger gestures", click on the Left dropdown and select "Custom shortcut" and you should receive some additional settings to record the key combination for this gesture.

For Google Chrome, to switch to the previous tab, the keyboard shortcut for this is "Ctrl + Shift + Tab", hit record and press those three keys together, then save the Custom shortcut.

Do the same for the Right option, but the keyboard shortcut to switch to the next tab is "Ctrl + Tab".

Let us know if this worked for you or if you need any further assistance.
