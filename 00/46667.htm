Hi painterofbridges,

 

·        Do you want to customize your screensaver and desktop icons?

The source of the download should also contain the instructions on how to install them on the computer. However, you can check if the following helps.

 

If you have downloaded new screensavers and icons and want to activate them then, you need to save them to your computer first. (On a desired location on the hard drive)

 

To install the screensaver:

 

a.       Click the Start button and choose Control Panel.

b.      Select “Appearance and Personalization" in Control Panel. If you have previously set Vista to display the Control Panel in the Classic view, the title of the icon is "Personalization."

c.       Choose "Change Screen Saver" in the Personalization section.

d.      Install a customized photo screen saver for Vista by selecting "Windows Photo Gallery" from the Screen Saver Settings drop down box.

e.       Use the "Settings" button to select options for this screen saver. In the Windows Photo Gallery Settings, select the first radio button to let Vista display all photos on your hard drive or to include or exclude them, using tags.

f.        Pick the second radio button to let Vista display all the photos in the hard drive's Photos folder. Use the "Browse" button and select the correct folder where you have downloaded your screensavers, if yours are not in Photos.

To install the icons:

 

a.       Click the Start button and choose Control Panel.

b.      Scroll to the "Personalization" icon and double-click. Select the "Change Desktop Icons" link on the left side of the screen.

c.       Choose the icon you want to change in the box. Click the "Change Icon" button. Select the "Browse" link. Navigate through your hard drive to the downloaded folder. Select the icon you want. When you finish, select "Apply" to save changes. Then, "OK" to leave the icon settings.
