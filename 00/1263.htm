david a matson say: Greetings RJZ2!  Welcome to the Windows Phone forums!

Sorry for the lateness of the response.   Did you know about the duplicate finder tool?

1.) Login to the Windows MyPhone website.
2.) Up towards the top is the option to check for Duplicates. 
3.) Click on this option, then select "all" duplicates option just beneath that.
4.) Delete the duplicates by item category (contacts, calendar, etc).
5.) When you next connect your phone, only your duplicates will disappear (the newly created items will remain).

I hope this helps with your main problem.

InStr() which will return the starting position of one string within another (hint: find a space). 
Left() which returns a number of characters on the left end of a string.
