saltgrass say:

Why should I use RAID for an M.2 SATA, and why should I use AHCI for an M.2 NVMe?


Because the SATA controller setting in the Bios determines the functionality of the M.2 slot.  If set to AHCI, you can run an NVMe drive.  If set to RAID, you can run a SATA version or an NVMe drive configured to run as SATA, which is how Dell ships their systems.

6. Click OK

5. Click "Restart the computer"
