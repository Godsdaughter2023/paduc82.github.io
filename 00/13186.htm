paulcordts say: The issue is using the Indirect(A1) as a validation criteria.

What I have done for this issue is to use a sheet change macro to reset the validation criteria.

This way when you change A1, the macro changes the validation setting

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.row=1 and Target.column=1 then 'This restricts the action to only changes for cell A1
  With Range("B1").validation
     .Delete 
     .Add Type:=xlValidateList, AlertStyle:=xlValidAlertStop, Operator:=  xlBetween, Formula1:="=" & Range("A1") .IgnoreBlank = True
     .InCellDropdown = True 
     .InputTitle = "Your Title Here" 
     .ErrorTitle = "" 
     .InputMessage = "Your message here" 
     .ErrorMessage = "" 
     .ShowInput = True 
     .ShowError = True 
   End With
End if
End Sub

I would recommend you use the macro recorder and change the validation manually once to get the conditions from .IgnoreBlank through .Show Error and replace those above with your recorded setting.

If you have many rows for which this is repeated change the beginning to this

If  Target.column=1 then 'This restricts the action to only changes for column A
With Target.offset*0,1).validation
   .Delete 
   .Add Type:=xlValidateList, AlertStyle:=xlValidAlertStop, Operator:=  xlBetween, Formula1:="=" & Target.Value
