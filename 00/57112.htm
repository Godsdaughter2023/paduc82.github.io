How to repair and reboot to Windows 7:

a) Insert the Windows 7 System Repair DVD or Installation DVD and reboot.

b) Select the correct keyboard input method and click ‘Next’.

Note: If you are using the Windows7 Install DVD then at the 'Install Now' prompt (Do not pick the Install Now option) instead choose the 'Repair your computer' option located in the lower left of the display.

c) The System Repair tool will search for a valid operating system and will report that “Windows found problems with your computer’s startup options. Do you want to apply repairs and restart your computer?”

d) Click ‘Repair and restart’ and reboot again to the Windows 7 System Repair environment.

e) Click ‘Next’ at the keyboard selection prompt and when the ‘System Recovery Options’ screen appears verify that the “Use recovery tools that can help fix problems starting Windows” is selected.

Note: On a Dual Boot system there may be only one operating system listed "if you have removed one of the two versions of Windows 7". Click the ‘Next’ button.

f) Now from the menu list select the ‘Command Prompt’ option.

g) At the DOS prompt type the following three commands:

bootrec.exe  /fixmbr

bootrec.exe  /fixboot (may return an 'Eliment not found' message)

bootrec.exe  /RebuildBcd

h) Close the command prompt window and click the ‘Startup Repair’ option.

i) The repair process may take some time, so wait until the process completes at which point you will see two messages “Windows cannot repair this computer automatically” and “Startup Repair cannot repair this computer automatically”.

Note: Please perform step j as detailed, startup repair must be run multiple times.

J) Close this message window by clicking on the X in the upper right corner of the Send/Don’t send window. Next click on the ‘View advanced options for system recovery and support’ option and from the main menu once again click ‘Startup Repair’. This time the repair process may only that a few seconds and when prompted “Startup Repair could not detect a problem”, click ‘Finish’  and run the ‘Startup Repair’ option one more time, click ‘Finish’, and then remove the System Repair DVD and click ‘Restart’.  Reboot the computer.

cmdrkeene say:

It's not always significant trade-offs. It's just worth considering. For example, if you use the device offline or have to pay by usage on your internet connection, you may prefer to lose some hard drive space but have a large cache of web content, or allow more system restore points to pile up in case you need them.  In general if you have enough space for Windows and the apps you use to function, you don't need to manage these too much.  But if space is at a premium, you can turn off system restore, prevent caching, turn off some detailed logging/error reporting collection, and clear out a number of temp files.

Most (but certainly not all) of the locations you can clean out are in the cleanmgr.exe program, Disk Cleanup. It comes from Windows 95 Plus, but it's been updated over the years and even plays key rolls in Windows install. A few days after installing a windows upgrade, cleanmgr comes back and clears out the windows.old folder (so be sure to get what you need from it, if anything).

When you run it, be sure to click "Clean Up System Files".  This button just re-launches the process but running as administrator, allowing it to clean a much wider scope of locations that are otherwise outside the user sandbox.

 

You can schedule this program to run automatically in the background in task scheduler.  In fact, you can choose up to 65,500 different configurations (not that you'd need that many).  For example, you could have this too empty out things you don't care about every day at 2:00 AM, but another scheduled version of it could clean out other areas on a longer interval, say once a month.  Just remember the trade off: you might be losing valuable backup information, files from the recycle bin you might have needed, or just costing more data download over the network. Maybe you don't care about those things.

Windows 10

I think my recommendation is to clear out all you can ahead of the upgrade to Windows 10 so that there is plenty of space to install.  Afterwards, don't clean it out immediately (I don't want you to lose your ability to roll back to your previous install).  But once you're comfortable and are okay losing that backup, I can tell you that you'll be pleasantly surprised at how much free space there is on Windows 10 versus Windows 8.1. I know I was on my Dell Veneu 8.
