Insert borders to a single column - Microsoft Community

jamesavery 804 say:

Insert borders to a single column

I'm working a two column page in office 2013. How do I insert a border to single column not the whole page?  HELP!
jay freedman say:

There are several ways to border only one column. The question that makes the most difference is "Does the text from the bordered column have to be continuous into the next unbordered column?"

If the answer is no -- if the two columns are unrelated -- then you can use a text box or a table to contain the bordered text.

If the answer is yes, you can use text borders around the paragraphs in one column; you might need a column break at the beginning of the paragraph after that, to keep it in a separate column. Or you could use two text boxes, one with a border and the other without a border, and link them together so that any text that overflows the first box will go into the second one.

Finally, you could just type the text in the two-column layout, and then insert a rectangle shape (set its text wrapping to In Front of Text) with a border but no fill to cover one column. This is best done after all editing of the document is completed, to avoid the possibility of the shape being pushed out of place or needing to be enlarged or reduced to fit other edits.

Listing all Combinations of a Table - Microsoft Community

chris molland say:

Listing all Combinations of a Table

Hello, and thanks for taking a look at this question.

I thought this would be simple, until I tried it!

I'll describe a simple example to help clarify my question. Let's say I have a table which holds survey information in a school class which shows how many Boys/Girls like different fruit. The table has the following Rows: Apples, Oranges, Bananas and Grapes. The table has the following columns: Boys and Girls.

Each column contains the number of Boys/Girls which like the different fruit.

What I want is a list which shows all the combinations like this:

Apples  Boys  2

Oranges  Boys  5

Bananas  Boys  8

Grapes  Boys  3

Apples  Girls  5

Oranges  Girls  6

Bananas  Girls  2

Grapes  Girls  7

I used to be able to do this in Excel 2004 by creating a Pivot Table and Pivot Chart Report. In the dialog box I would select 'Multiple Consolidation Ranges' and 'I will create the Page Fields'. I then selected to place the Pivot Table on a new worksheet. I then clicked on the 'Layout' button, dragged off the Row and Column headings and clicked 'Finish'.

In the resulting Pivot Table I would double-click the Grand Total and another worksheet is created with the results I wanted.

I just can't seem to re-create this in Excel 2007 (PC) or Excel 2011 (Mac), its too different.

I would be very grateful for some advice here.

Regards

Chris
chris molland say:

OK, I've got it.

You can open the old PivotTable Wizard, it's just that you need to know the secret keyboard shortcut. On my Mac this is Cmd + Alt + P.

Once you're into the Wizard you simply follow these steps:

  1. Select 'Multiple Consolidation Ranges'.
  2. Click Next.
  3. Select 'I will create the page fields'.
  4. Click Next.
  5. Select all of the Cells including the Row and Column headers for the Range. Click 'Add'.
  6. Click Next.
  7. Choose to place the Pivot Table on a new sheet or existing sheet.
  8. Click Next.
  9. Open the Pivot Table Builder, if it's not already open. In the 'Field name' box un-check Row and Column leaving only Value checked.
  10. Drag Value down to Row Label.
  11. In the Values box have 'Count of Values'.
  12. The Pivot Table now should have all of the individual values in the left column, these would be the number's of Children who like each fruit. In the right column are the count of each value.
  13. Double-click on the Grand Total Value (the number not the words Grand Total).
  14. A new worksheet is created with 3 columns (Row, Column and Value) with each combination and the value.
  15. Brilliant!
I hope this helps.

Chris
