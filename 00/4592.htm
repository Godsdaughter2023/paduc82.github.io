rohn007 say:
short answer: No you can't get office to show more than one version of Office.

Long Answer: There are 2 things going on here at the same time.

First is that windows is really simple. It can only assign one program to be default at a time.

The other is that Office ASSUMES that the most recently run application sets the default for all Office applications by running the "Configuring ..." dialog. You have to do a registry hack to block that "configuring ... " process.

After you have done the registry hack, then you can use the Control Panel > Default programs > Associate a file type with a program > select a file type > click on Change program button > More apps > Look for more applications on this computer > then you can specify the exact path to the specific version application you want to make default for that specific file type.
