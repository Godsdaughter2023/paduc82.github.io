xbr dionysus say:

Overview

Engenius ESR-1221

Login Information

  • Gateway – 192.168.1.1
  • Username – admin
  • Password – admin

 How to Find SSID (Network Name)

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Management from the menu on the left.
  5. Select Status from the sub menu.
  6. The SSID is located on this page labeled SSID.

 

How To Find Wireless Passphrase (WPA/WEP)

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Wireless from the menu on the left.
  5. Select Security from the sub menu.
  6. The passphrase is on this page labeled PRE-Shared Key.

 

Changing Wireless Security Type/ Disabling Wireless Security

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Wireless from the menu on the left.
  5. Select Security from the sub menu.
  6. Select desired security type from the Encryption drop down menu. (To turn off the security select None)
  7. Select Apply Changes.

 

MAC Filtering

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select MAC Filtering from the sub menu.
  6. Uncheck the Enable MAC Filtering box.
  7. Select Apply Changes.

 

Change Wireless Channel

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Wireless from the menu on the left.
  5. Select Basic Settings from the sub menu.
  6. Select desired channel from the Channel Number drop down menu. (1,3,9,11)
  7. Select Apply Changes.

 

Change Wireless Mode

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Wireless from the menu on the left.
  5. Select Basic Settings from the sub menu.
  6. Select 802.11g only from the Band drop down menu.
  7. Select Apply Changes.

 

Change Wireless Band (Width)



This router only supports the 20MHz band.



DHCP Reservation

*This feature is not available on this device*

 

UPnP

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select TCP/IP Settings from the menu on the left.
  5. Select WAN Interface from the sub menu.
  6. Check or uncheck the box next to Enable uPNP.
  7. Select Apply Changes.

 

DMZ

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select DMZ from the sub menu.
  6. Check the Enable DMZ box.
  7. Enter the Xbox’s IP Address into the DMZ Host IP Address box.
  8. Select Apply Changes.

 

Disable DMZ

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select DMZ from the sub menu.
  6. Uncheck the Enable DMZ box.
  7. Select Apply Changes.

 

Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select Port Forwarding from the sub menu.
  6. Check the Enable Port Forwarding box.
  7. Enter the IP Address of the Xbox into the IP Address box.
  8. For Protocol select Both.
  9. In both Port Range boxes type 3074.
  10. In the Comments box type Xbox.
  11. Select Apply Changes.  

 

Disable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select Port Forwarding from the sub menu.
  6. Click the Select radio button next to the port you want to remove.
  7. Click the Delete Selected button.
  8. Click the Apply Changes button.

 

Firewall

*There are no firewall settings that need to be configured.*

 

How to Find Firmware Version

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Management from the menu on the left.
  5. Select Status from the sub menu.
  6. The firmware is located on this page labeled Firmware Version.

 

Firmware Update

  1. Plug a computer into the LAN port on the back of the device.
  2. Open a web browser.
  3. Go to the Engenius support page and find the newest version of firmware for your router.
  4. Download it and save it to the desktop.
  5. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  6. If prompted to login enter “admin” as the username and “admin” for password.
  7. Select Management from the menu on the left.
  8. Select Upgrade Firmware from the sub menu.
  9. Select the Browse button.
  10. Find the file you just downloaded and click Open
  11. Select the Upload button.
  12. If asked to confirm click Yes.
  13. Wait for the router to finish the update and reboot.

 

Restore Factory Defaults

Through Hardware

  1. Locate the Reset button.
  2. Using the tip of a ballpoint pen or toothpick press and hold the reset button for 15 seconds.
  3. Release the reset button.
  4. Wait for the device to restart.

Through Firmware

  1. Plug a computer into the LAN port on the back of the device.
  2. Open a web browser.
  3. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  4. If prompted to login enter “admin” as the username and “admin” for password.
  5. Select Management from the menu on the left.
  6. Select Save/Reload Settings from the sub menu.
  7. Select the Reset button.
  8. If asked to confirm click Yes.
  9. Wait for the router to finish the update and reboot.

 

 Entering PPPoE

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select TCP/IP Settings from the menu on the left.
  5. Select WAN Interface from the sub menu.
  6. In the WAN Access Type drop down menu select PPPoE.
  7. Enter the User Name and Password.
  8. Select Apply Changes.

 

Known Issues

If you have Surewest for your ISP you may not be able to access the router settings. Surewest requires all changes to be done remotely from the ISP.

 

If there is no modem and just an ethernet jack into the wall, you may have irresolvable NAT issues.

 

Multiple Console Support

We have confirmed that this network device is unable to support more than one console with an Open NAT.

When you add a child's Microsoft account to your family, you'll get regular activity report emails summarizing how much time they spent on the PC, the websites they visited, the games and apps they used, and the terms they've looked up in search engines like Bing, Google etc.

The email summary gives you a lot of info about your child's PC activities at a glance. But you can always delve into more details—and change permissions and other settings based on the activity info—by selecting Recent activity at account.microsoft.com/family. Here's how:

1. Go to account.microsoft.com/family and sign in.

If you don't want to receive weekly activity reports in your email, you can turn that feature off and just check your kid's activity reports pages manually.

mark ole say:

Hi Richard,

It is possible that the noise is coming from the fan of your Surface device. It is normal to hear the fan on Surface devices and in some cases the sound of the fan will change slightly. If the fan is not working correctly, you will see the “Thermometer Gauge” warning indicating the internal temperature is above what the Surface CPU can operate at.

We recommend that you install the latest updates to keep your Surface performing its best. You can get updates now by checking Settings> Update & Security> Windows Update.

You can try to force shutdown and restart your device and see if it helps resolve the issue. You can do this by holding the power button for 20 seconds until you see the Windows logo.

If you tried to force shutdown your device and it did not make any difference, bring your device to a Microsoft Store near you and have it assessed.

Let us know if you need further assistance.
