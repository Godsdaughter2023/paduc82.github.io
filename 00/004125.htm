jay freedman say:

The Word developers tried to make the reviewer identity of existing revisions and comments as permanent as possible. There's no way within the user interface to change a reviewer's name, and the VBA macro language considers it to be read-only (https://docs.microsoft.com/en-us/office/vba/api/word.revision.author). You might think that the only way to change the name is to visit each revision and comment, one at a time; copy the replacement text or comment; reject the revision or delete the comment; and then recreate the revision or comment so it will use the current user name. That process is too painful to consider...

There is another way. It's a large number of easy steps:

  • In Windows Explorer, copy/paste the document to a new file.
  • Change the extension of the new file from .docx to .zip. (This works because a .docx file really is just a zip file with a different extension.) When asked whether you're sure you want to change it, click Yes.
  • Double-click the zip file to open it. If you have a dedicated zip program, the file will open in that program; otherwise it will appear in Windows Explorer as a folder. In either case, there will be lots of subfolders within the zip.
  • Open the subfolder named 'word'.
  • Select and copy the two files named comments.xml and document.xml.
  • Open or create some other folder outside the zip and paste the two files into it.
  • Start up the Notepad.exe program and open comments.xml in that program. In the Open dialog, you'll have to change the file type from Text (*.txt) to All Files (*.*) to be able to see xml files.
  • Open the Replace dialog (Ctrl+H).
    • In the Find What box, enter author="your email address" (of course, type your actual email address in the quotes). Note that there are no spaces around the equal sign.
    • In the Replace With box, enter author="your user name".
    • Click the Find Next button to make sure it properly matches what's now in the file. If not, fix any typos. When you can successfully match the email entry, click the Replace All button.
    • Close the dialog.
  • Save the file and close Notepad.
  • Copy/paste the comments.xml file back into the zip folder, replacing the one that's there.
  • Repeat the use of Notepad to open the document.xml file, replace the email address with the user name (this will change the reviewer in the tracked changes), save the result, and copy the file back into the zip folder.
  • Go up to the folder that contains the zip, and rename the zip file to the .docx extension, answering Yes when prompted.
  • Open the modified document in Word and verify that the name has been changed properly in the comments and revisions. If you're happy with the result, delete the original document and rename the modified copy to the original name.

    There is a macro in the post by user scenography in the thread at https://stackoverflow.com/questions/38225918/vba-to-change-name-of-author-on-tracked-changes-in-word that automates a process similar to this. I haven't tried it.
