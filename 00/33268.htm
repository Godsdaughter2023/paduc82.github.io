james.orr say:

That's getting way too complicated for what you're trying to do. I haven't got how to keep the command prompt open for putting in other commands, but you can just use a batch file for the icon that pings.

Make a new text document, name it ping or whatever you want, and in the document, type in "ping <ip>" so for example ping 192.168.100.100, then save it. With just that, once it's done pinging the cmd window will auto close, so if you want it to stay open to see the results, add pause to the line under the ping line.

Then rename ping.txt to ping.bat to make it a batch file, and now if you double click it, it'll open command prompt, do the ping command, then exit or if you added pause it'll pause and say press any key to continue.

You can edit what is being pinged by right clicking the batch file and selecting edit, it'll open like a text document and you can edit it from there to do whatever you want.
