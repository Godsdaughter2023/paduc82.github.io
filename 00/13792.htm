it will simply open MyLinkedTable. It doesn't matter whether the link to MyLinkedTable was created last year or thirty seconds ago; it just looks for that name.

Of course when you relink, you must be sure that the names, datatypes and meanings of the fields are still correct. If you're linking to the same file (of a file with the same structure) you're good; if the file has new fields or changed fields you will need to tweak the query to match.
