themanindbox say:

How to fix Windows 10 Updates that are not properly applying

So you bought windows 10, and you are having issues with updates, well you aren't alone.. 

There is a problem in the way windows 10 applies updates, which as a result causes things to get stuck, or fail completely to install. 

You may also have issue where you will get to 5% or so complete on download before the updates freeze or fail to work.. 

Don't worry, 

this is a simple fix.. 

On your machine, 

Log in.. 

On your keyboard you will have a key on the left that looks like a windows ICON.. 

Click that button and hold it, and while holding it, press the R key on the keyboard. 

This will open RUN. 

In RUN type in Services.msc and press the OK button. 

You will get a window that will show all the windows services running. 

In this window, locate; Windows Update and Right Click it. 

Click on Properties

In the properties of Windows Update, you will want to click the drop-down where it says startup type and select Disabled

Next you will want to click Apply. 

Next you will want to click on the stop button. 

Lastly click Apply, and OK. 

After stopping the Windows update service; 

Use the Run command again (see above) and paste in;

 %windir%\SoftwareDistribution\Datastore

Once in that folder, highlight everything and Press Shift and Delete at the same time. 

This will delete all the folders. 

Reboot.. (Save this page or text) 

Once your machine is back up.. 

Use the Run command again (See above for how to get to run) 

Type in Services.msc

Locate Windows update, 

Right click

Choose Properties

Click the drop down next to Startup type and choose automatic

Click Apply

Click Start

Click apply 

Click OK. 

Reboot (Once you have rebooted, click on the Update option and update the Windows 10 machine. 

This will resolve nearly 90% of windows update issues on windows 10.
