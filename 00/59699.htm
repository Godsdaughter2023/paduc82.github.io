monica cam say:
Hi Rupert_W,

Windows Update won't fully install without rebooting the computer, it is working as designed. When your computer restarts all the programs and processes end leaving only the power on. A restart is necessary as it is the last stage to apply the changes made to the device.

As per your query about re-opening your previous browser session, you can follow the steps below if you are using Microsoft Edge:

  1. Open the Microsoft Edge browser.
  2. On the left side select the three-dotted menu , select Settings.
  3. Under Open Microsoft Edge with, select Previous pages from the drop-down list.

To avoid Windows Update from interfering from what you are working on to your device, I suggest that you set Windows Update to install on the time you least use your device. Set "active hours" to the time that you are using your computer so Windows will not restart during those hours. To do this, follow the steps provided in this article.

Should you have other questions, let us know.
