Merge multiple worksheets into one - Microsoft Community

trinakropf say:

Merge multiple worksheets into one

I have a spreadsheet with multiple identical tabs.  I would like to compile all the data from the tabs into one spreadsheet, but only need data from 2 of the columns.  Is there VBA codes to do this?
per jessen say:

Sub CompileData()
Dim TargetSh As Worksheet
Dim DestCell As Range

Application.ScreenUpdating = False
Application.DisplayAlerts = False
If Sheets(2).Name = "Compiled Data" Then Sheets(2).Delete
Application.DisplayAlerts = True

Set TargetSh = Worksheets.Add(after:=Sheets(1))
Set DestCell = TargetSh.Range("A1")
TargetSh.Name = "Compiled Data"

For sh = 3 To Sheets.Count
    Sheets(sh).Range("B3:B5").Copy DestCell
    Sheets(sh).Range("IK5:IK77").Copy
    DestCell.Offset(0, 1).PasteSpecial xlPasteValuesAndNumberFormats
    Set DestCell = DestCell.Offset(73, 0)
Next
Application.CutCopyMode = False

End Sub

Did you initialize the new hard drive? That must be done. Most hard drive manufacturers provide info concerning that on their support site.

Once that is done the Windows 7 installation disc will see it if your DVD drive isn't so old that it needs a driver installed for it before it can run the DVD.
