Cause

A driver tried to access an address that is pageable (or that is completely invalid) while the IRQL was too high.

This bug check is usually caused by drivers that have used improper addresses.

If the first parameter has the same value as the fourth parameter, and the third parameter indicates an execute operation, this bug check was likely caused by a driver that was trying to execute code when the code itself was paged out. Possible causes for the page fault include the following:

  • The function was marked as pageable and was running at an elevated IRQL (which includes obtaining a lock).
  • The function call was made to a function in another driver, and that driver was unloaded.
  • The function was called by using a function pointer that was an invalid pointer.

In general such information is always useless, since there's absolutely nothing that you or anyone else for that matter can do about such port scanning.  In fact there's nothing you need to do if your PC firewall is doing it's job, since there's no reason to let anything into or receive a response from your system unless your actions made the initial request.

These types of notifications are simply a way of helping sell security products to consumers by convincing them that the product is actually doing something, when in truth it's simply adding to the FUD (Fear, Uncertainty and Doubt), that all consumers have about their security.

In this latter case, the use of something like Malwarebytes Anti-Malware (MBAM) may be advisable, since it's the Potentially Unwanted Programs (PUPs) like adware that often get installed unnoticed by less careful users that MBAM deals with quite well.

https://downloadcenter.intel.com/Detail_Desc.aspx?agr=Y&DwnldID=22375&lang=eng


I tried the newer release (3/21/13) and it would not install on my Surface Pro.  Go figure.

 

Then get the 4/3/13 release of the DisplayLink core software from the DisplayLink website and install it.


http://www.displaylink.com/support/downloads.php

 

Your Netflix and Hulu apps should now both work.  Also, you should notice that your metro apps don't have a 10 second lag upon opening anymore.   You'll see what I mean when you open the IE metro app.  Your DisplayLink adapter should also be working as well.

arya s asok say:

Hi Kyo-San,

 

Since the troubleshooter was not able to see the CD DVD drive, you may try to navigate to the following location in registry editor and delete the upper filter and lower filter keys and check if it helps you to get the CD DVD drive working.

 

1.     Click Start, and then click All Programs.

2.     In the start search box type regedit, and press enter. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

3.     In the navigation pane, locate and then click the following registry subkey: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Class\{4D36E965-E325-11CE-BFC1-08002BE10318}

4.     In the right pane, Check for UpperFilters and LowerFilter keys and select it.
Note You may also see an UpperFilters.bak registry entry. You do not have to remove that entry. Click UpperFilters only. If you do not see the UpperFilters registry entry, you still might have to remove the LowerFilters registry entry.

5.     On the Edit menu, click Delete.

6.     When you are prompted to confirm the deletion, click Yes.

7.     Exit Registry Editor.

8.     Restart the computer.

 

Note: Sometimes this problem is caused by two Windows registry entries that have become corrupted. To fix the problem, you have to use Registry Editor to delete the corrupted Registry entries.

 

However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs.

 

For more information about how to back up and restore the registry, follow the steps as in the KB Article: http://support.microsoft.com/kb/322756/

 

Access the link below to know more about the CD drive or the DVD drive that does not work as expected on a computer.

http://support.microsoft.com/kb/929461

 

Note: the above article is for Windows Vista and the same holds good for Windows 7.
