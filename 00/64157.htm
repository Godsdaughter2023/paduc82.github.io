deleted material - Microsoft Community

end the junk say:

deleted material

When is deleted material really deleted?
brian tillman [outlook mvp 2007-2019] say:

When you pulverize the disk it's on.

Under ordinary conditions, files you delete are never actually deleted.  First, they move to the Recycle Bin, but you can still recover them there easily.  When you empty the Recycle Bin, the file itself isn't deleted, only the pointers that allow the file system to locate it.  The data remains on the disk.  There are many tools that can read the space between files and locate that data again.  Since the file system doesn't track it any more, you may overwrite the data the next time the file system needs space for another file, which would overwrite the data at that point, but there are forensic tools that can recover even that in many cases.

Doing a full format of a hard drive generally is sufficient to ensure the data can't be recovered, but even then there are tools that can at least partially recover it.
