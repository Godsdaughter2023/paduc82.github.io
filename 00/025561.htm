If you don't want the comments to appear, delete them.  Running a macro containing the following code will do that:

Dim acomment As Comment
With ActiveDocument
    For Each acomment In .Comments
        acomment.Delete
    Next acomment
End With

After running that, save the file with a different name if there is a likelihood of your later needing to access the comments.

Virus and Malware Transmission By Opening Email? - Microsoft Community

lwapner say:

Virus and Malware Transmission By Opening Email?

I'm running Windows 8.1 and use Windows Defender.  Can my computer become infected (virus or malware) by merely opening an email without downloading attachments or clicking on links?

I've received mixed replies to my question posted elsewhere.

Thanks!

rob koch say:

This depends on the email client you use and how susceptible it may be to being invoked by those items.  Since most modern versions of email applications like the full Outlook have been designed not to display these items in a preview mode, they typically can only invoke them if you purposefully click on them.

If you instead use a browser based client such as Outlook.com, it's not typically possible for these to be invoked or viewed without clicking.  However, since some graphics that contain "web bugs" can be displayed and so invoked (e.g. touched) simply by opening the message, this may allow the sender to confirm that your address is active.  This is why messages detected with these sometimes don't display images and/or they may be placed in a Spam or Junk folder which also inhibits both these images as well as possibly other links in order to protect you.

The specifics of this depend upon the email system and viewer you use, so that's why you'll receive such a confusing array of answers, since few other than those who've managed email systems like myself truly understand all of the potential subtleties.  And only by testing or reading about these specifics in your own email client's documentation will you be able to confirm exactly how yours may work.

Rob
