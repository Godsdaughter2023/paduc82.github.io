The WdSaveFormat Enumeration is for the FileFormat argument. We're not talking about the FileFormat argument here, but about the CompatibilityMode argument. That uses the WdCompatibilityMode Enumeration, with values

11 = wdWord2003

12 = wdWord2007

hansv mvp say:

You declare a variable strDoc but use strDocName. The following works OK for me:

 

Sub Save_Docx()
    Dim strDoc As String
    Dim intPos As Integer
    strDoc = ActiveDocument.FullName
    intPos = InStrRev(strDoc, ".")
    strDoc = Left(strDoc, intPos - 1) & ".docx"
    ActiveDocument.SaveAs2 FileName:=strDoc, FileFormat:=wdFormatXMLDocument, _
        CompatibilityMode:=14
    ActiveDocument.Close
End Sub
