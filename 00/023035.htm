s0nspark say:

The synchronizing you are seeing is normal behavior for IMAP and cannot be turned off.

 

With POP email accounts, mail is downloaded to your machine during a send and receive. With IMAP accounts, however, mail is stored on the server and your view is simply updated (i.e. synchronized) during send and receive... What you are seeing is Outlook scanning through the folders on the server and making your view of the folders match what is there.

 

If this concerns for some reason you then you may be able to set up the account as POP instead of IMAP if the provider supports access in that manner...

I had the same problem, after removing the touchpad driver, the mouse started working again.

My laptop has also touchscreen, so I could still get around in the settings. You might need an external mouse, I did not test if that worked.

Step by step instructions:

1. Connect a mouse to your Laptop and check that it works, if it does not work, you will have to use the keyboard or get help

2. Right click on the Windows logo a.k.a. Start Menu

3. A menu will pop-up, choose "Control Panel"

4. If your control panel is listed by Category, choose "Uninstall a program", otherwise choose "Programs and features"

5. From the list search for the touchpad driver, in my case it was ELAN touchpad driver

6. Right-click and select Uninstall

7. After this reboot your Laptop and then run windows update, this should install the correct driver for your touchpad
