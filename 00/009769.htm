    mDate = Format(myDate, "mmmm")
   
    Select Case Right(dDate, 1)
        Case 1: dText = "st"
        Case 2: dText = "nd"
        Case 3: dText = "rd"
        Case Else: dText = "th"
    End Select
    OrdinalDate = dDate & dText & " " & mDate & " " & mYear
OrdinalDate.Characters(3, 2).Font.Superscript = True                  (problem is here)
End Function

If, as seems likely, you are using your function as a UDF (User Defined Function) on a worksheet, you cannot add Excel formatting instructions. More generally,  with very few highly esoteric and particular exceptions, a worksheet function can only return a value to the calling cell; it cannot change or modify its environment.

If you wish to convert a cell value to its corresponding ordinal equivalent and apply specific formatting, you will need to use a VBA procedure rather than a UDF and this cannot be called from a worksheet formula. 

Note also that formatting is applied to a range object. As written, OrdinalDate is not a range object and will have no Characters property. Therefore,  your problematic instruction would need to be revised to apply to a specific cell or range.
