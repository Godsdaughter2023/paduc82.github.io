john korchok say:

Color sets are file-wide, not application-wide. You can set colors for a template, and as long as you open that template, you'll have the same set of colors.

With Excel, PowerPoint and Word, you can set default templates. Then when you open any of these, you'll see your chosen color set, until you open a file using a different template.

In addition to the 12 basic colors in a color theme, in Word and PowerPoint you can also set up to 50 custom colors in a template. Here is my article on the subject: XML Hacking: Custom Colors

Please note, custom colors do not travel with a theme, only with a template. This means that to get the same set of colors in Word and PowerPoint, you'll have to hack both a Word and a PowerPoint template with your custom colors. Excel ignores custom colors.
