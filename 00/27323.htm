  1. Close Outlook if it is running
  2. From control panels - select the User Accounts and Family Safety item, then the Mail (Microsoft Outlook 2013) item and then the Email Accounts option.

(Note domain and username are separated by a backslash (\) character This changes the domain. Enter your password as usual.
  7. I was prompted to do this twice - same each time. Enter domain\username in the username field and usual password
  8. Account is then created.
  9. Open Outlook to complete the process and synchronise data with Exchange.

You need to add a Windows Media Player control object thingo:
