Trying to create a pop-up to send reminder using the VBA Code. - Microsoft Community

zahiral say:

Trying to create a pop-up to send reminder using the VBA Code.

Hi 

I need a little help - i am trying to create a pop-up message from my worksheet for when a cell in Column G turns to 'SendReminder' after using this code:

=IF(H4<TODAY()+30,"Send Reminder") in the worksheet. 

The VBA code which I am trying to use is as follows but however it does not work:

Private Sub Worksheet_Change(ByVal Target As Excel.Range)
If Not Intersect(Target, Range("G1:G1000")) Is Nothing Then
If UCase(Target.Value) = "SendReminder" Then
    MsgBox ("Email to be sent")
End If
End If
End Sub

Could someone please help?

jllatham say:

The _Change event does not fire when a formula revises its result (calculates).  The event you need to work with is the Worksheet_Calculate() event.

The most basic routine to achieve this would be something like this:

Private Sub Worksheet_Calculate()
  Const rPhrase = "Send Reminder"
  Dim searchRange As Range
  Dim foundEntry As Range
 
  Set searchRange = Range("G1:G1000")
  Set foundEntry = searchRange.Find(what:=rPhrase, LookIn:=xlValues, lookat:=xlWhole, MatchCase:=False)
  If Not foundEntry Is Nothing Then
    MsgBox "send email reminder"
  End If
  Set searchRange = Nothing
End Sub

Now for all the problem associated with this:

The main problem is that this process is going to run every time any formula on the worksheet recalculates - so unless you get rid of the "Send Reminder" message that the formula put up, you're going to continue to get the pop-up message over and over and over.

Possibly the easiest way to deal with this would be to put some indication on the worksheet itself that the reminder for a row has been displayed and work that into the routine.  You'd need to use an available, currently unused column for this.  For this code, I'm assuming that column X is available.

Here is a routine that I believe would deal with all of this well for you, just change the "G" and "X" that are set up in it now to the actual columns involved; "G" should be correct based on your original post, but I don't know what column you need to use as the "marker" column.

Private Sub Worksheet_Calculate()
  Const rPhrase = "Send Reminder"
  Const fColumn = "G" ' formula column
  Const mColumn = "X" ' marker column
  Dim searchRange As Range
  Dim anyCell As Range
  Dim offsetToMarker As Long
 
  offsetToMarker = _
   Range(mColumn & 1).Column - Range(fColumn & 1).Column
 
  Set searchRange = Range(fColumn & "1:" & fColumn & "1000")
  'run through all of the cells in G and test them
  For Each anyCell In searchRange
    'we test for "Send Reminder" in G and
    'for an empty cell in X
    If UCase(Trim(anyCell)) = UCase(Trim(rPhrase)) Then
      'test to see if anything is in column x
      If IsEmpty(anyCell.Offset(0, offsetToMarker)) Then
        'has Send Reminder in G, but nothing in X
        MsgBox "Send email Reminder"
        'and flag as having had this done!
        anyCell.Offset(0, offsetToMarker) = "reminded"
      End If
    Else
      'does not have "Send Reminder" in G,
      'make sure nothing in X
      anyCell.Offset(0, offsetToMarker).ClearContents
    End If
  Next ' end anyCell loop
  Set searchRange = Nothing
End Sub
