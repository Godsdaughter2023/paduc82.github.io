Print Selection means "Print the slides I've selected" ... it doesn't print the selected shapes.

So if you haven't selected at least one or more slides, the option will be grayed out.  

And if no shapes are selected, then from PPT's point of view, the current selection is the current slide ...and with a slide selected, you DO get the Print Selected option.

rich michaels - mvp (greatcirclelearning.com) say:

To make your UserForm appear when the end user either clicks directly or "tabs" to your document based form control you need to set the Run macro on Entry parameter on the forms Option control with the name of your Macro.

The VBA coding for your UserForm needs to have private routines that look something like this:

Private Sub CloseButton_Click()
    Unload Me
End Sub

Private Sub SubmitButton_Click()
    Dim i As Integer
    For i = 0 To Me.ListBox1.ListCount - 1
        If Me.ListBox1.Selected(i) = True Then
            If i < Me.ListBox1.ListCount - 1 Then
                ActiveDocument.FormFields("Text1").Result = _
                    ActiveDocument.FormFields("Text1").Result & Me.ListBox1.List(i) & vbCr
            Else
                ActiveDocument.FormFields("Text1").Result = _
                    ActiveDocument.FormFields("Text1").Result & Me.ListBox1.List(i)
            End If
        End If
    Next
    Unload Me
End Sub

Private Sub UserForm_Initialize()
    With Me.ListBox1
        .AddItem "Red"
        .AddItem "White"
        .AddItem "Blue"
    End With
End Sub

The UserForm_Initialize routine loads the list items and the SubmitButton routine takes what the end user has selected and puts it into the document form as if the end user had typed it. This is just one crude example of how it would work and hopefully you will be able to customize the SubmitButton routine to meet your specific needs.

Finally you need a public subroutine in a VBA module that displays the UserForm. This is the subroutine that you list in the Entry Marco setting for the document form.

Option Explicit

Sub RunUserForm()
    Dim frmUserForm As New UserForm1
    frmUserForm.Show
End Sub

Hopefully this gets you on your way to completing your project.
