According to your description, we would like to know whether you checked deleted notebook in Second-stage recycle bin. If no, please check whether deleted notebook is present in Second-stage recycle bin, if yes, restore it.

 

Open the SharePoint site> Settings> Site settings (Site information> view all Site settings)> Recycle bin under Site Collection Administration> Click Second-stage recycle bin under Recycle Bin Page.

    Technical Level : Basic

    When browsing the catalog on Groove, or any other site, there are sometimes when you run into content that should be there; but isn't. When you have found something that is missing, sometimes finding how to report it to the correct team is quite a puzzle.

    You can't report missing content through the Groove App, as of yet. So to report missing content, you will need to visit the website, which is: https://music.microsoft.com

    Once you are at the website, simply sign into your Microsoft Account, using the sign in option at the top right hand side corner of the site. 

    Once signed in, you will need to click onto the gear icon, that you can see in the image above in the red box, for a drop down menu to appear.

    Now you will need to press the down arrow in the red box, to bring up a list of categories. From that list, you will need to select on Request Missing Content. 

    Once the pop up window changes to the window like the image above, please make sure you provide as much detail as you can, and fill out the form. This will allow the team to review what is missing, and to see if they can add the missing content. 

    Once you have completed the form, please click onto submit for the information to be sent off.

    Private Sub Worksheet_SelectionChange(ByVal Target As Range)

    If Not Intersect(Target, Cells) Is Nothing Then

    t = 0

    s = 0

    For i = 1 To 10

    If Cells(i, "A") > 0 Then

    t = t + 1

    s = s + Cells(i, "A")

    End If 

    Next i

    If t = 0 Then 

    Range("A11") = 0

    MsgBox "There isn't positive number"

    Exit Sub

    End If 

    End If 

    Range("A11") = Round(s / t, 2)

    End Sub


    (you may change: the numbers 1, 10, column A  and cell A11 as needed)
