A spreadsheet would be easier to update than calculations by hand. However, there is no format that seems to work, other than Number and showing everything in seconds. ronrosenfeld say:

You can enter in one of these formats:

h:mm:ss.fff  (fff = fractions of a second)

mm:ss.fff

If you use any other format, Excel will not interpret as you prefer.

Then format the cell in the manner you wish to have the data displayed.

Ignore what you see in the formula bar -- you will not be able to change that by formatting or data entry methods, so long as your cell is formatted to show time.

As an example, if you want to enter 97.22 seconds, you would need to enter it as

0:97.22

If the cell format is, for example [h:mm:ss.000] the cell will show:  0:01:37.220 (and the formula bar will show 12:01:37 AM)
