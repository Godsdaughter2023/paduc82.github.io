"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.

Fastest and easiest way into Safe Mode is to RIGHT click the Start Button select "Run".  Box opens type in MSCONFIG click OK.  Select Boot tab at top.  Check  the Safe Boot box,  click Apply, click OK.

Box opens that restart is necessary.

To return to normal boot you will have to reverse everything done above by unchecking the Safe Boot box, Apply, OK.

The longer way to enter Safe Mode is to Hold down the SHIFT key while clicking Restart from the shutdown icon below the Settings icon.  This will activate the Advanced Startup Options menu.  Choose Troubleshoot >

Advanced Options >Startup Settings > Select F4 or F5 for Safe Mode.  This method takes a little longer to do but you don't have to reverse anything to get back to normal boot.

slance310 say:

Hi,

From the command prompt in the Advanced section of the Installation Media...

WARNING... Be cautious with these commands if you need to login with a Microsoft Account...  Safe Mode does not always provide access to Login, even when choosing "With Networking"

Use the correct bcdedit command as shown below based on which

Safe Mode option you'd like to start:

Safe Mode:

bcdedit /set {default} safeboot minimal

Safe Mode with Networking:

bcdedit /set {default} safeboot network

Safe Mode with Command Prompt:

bcdedit /set {default} safeboot minimal

bcdedit /set {default} safebootalternateshell yes

To turn off Safe Boot use... bcdedit /deletevalue {default} safeboot

If that gives an error, use... bcdedit /deletevalue {current} safeboot

To try a manual boot repair...

UEFI Boot Repair...

For Legacy select the drive without the UEFI Prefix...

Be advised that booting with a Flash Drive requires that the drive is specifically configured for Legacy or UEFI Bios... They are not interchangeable...

Legacy requires a MBR type drive formatted NTFS...

UEFI requires a GPT type drive formatted FAT32...

Not all machines will boot into UEFI from a DVD...

1. Put the Windows installation disc or Flash Drive in machine, and then start the computer.

2. Press F9, F11, F12, ESC, or whatever your machine uses to access Boot Options at the Manufcturer's Logo Screen..

Select the CD\DVD Drive or Flash Drive that starts with "UEFI"... Press any key when

"Press any key to boot from CD or DVD …". appears. You may see this for a Flash Drive as well...

3. Select a language, a time, a currency, and a keyboard or another input method, and then click Next.

4. Click "Repair your computer" on the lower left of the "Install" screen....

5. Click the "Troubleshoot" or "Advanced" depending upon which appears...

6. Select "Command Prompt". (Capital letters not necessary)

7. Type BOOTREC /FIXMBR, and then press ENTER.

8. Type BOOTREC /FIXBOOT, and then press ENTER.

9. Type BOOTREC /REBUILDBCD, and then press ENTER.

NOTE... If you have multiple HDDs\SSDs, it would be best to unhook all except fro the OS drive..
