SOLUTION
========

DO NOT size your table by dragging the left edge of the table.  When the requirement is to have the left of the table align with the document left margin, change your settings as follows:

Table Properties, Table tab, Options button.  Set the left and right cell margins to 0.

If you are printing borders on the table and require the text to be indented from the left and right:
Paragraph dialog, set a left and right indent (0.2cms)

How do I get a letter in one cell to equal a number range in - Microsoft Community

stephanieeaves say:

How do I get a letter in one cell to equal a number range in another?

I would like to use a formula to provide a letter for a number score earned through the use of a grading scale. For example: 90-100 = A, 80 -89=B, 70-79 = C etc.

myall_blues say: There's a few ways of doing this - the best depends a bit on whether you have a lot of grade scales or not.

For just a few, like in your example, you could use a simple IF statement. So assuming the number score is in A1, then you could bet the letter in B1 by inserting the formula:
=IF(A1>=90,"A",IF(A1>=80,"B",IF(A1>=70,"C","Ungraded")))

If there's a lot more options then nested if statements can become a bit cumbersome, and you might try to use VLOOKUP. So somewhere on your sheet (for example E1:F5) create a table with scores in column E and grades in column F. Then in E1:E5 put 50, 60, 70, 80, 90 - note these must be in ascending order. In F1:F5 put E,D,C,B,A.

Then your formula in B1 can become:
=IFERROR(VLOOKUP(A1,E1:F5,2,TRUE),"Ungraded")
