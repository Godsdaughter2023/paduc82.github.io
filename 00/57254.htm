cmdrkeene say:

If you accidentally set two special system folders equal to each other, you'll have to use the registry to fix it.  The registry is advanced, but this particular task is not too difficult. Here's how to fix it:

  1. Press WindowsLogo+R to open the Run box.  Type regedit and click OK to open the registry editor.  You may be prompted for administrator access.

  2. In the navigation pane on the left, find this folder (called a key in the registry).

    HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders


  3. Find the Music entry on the right side, and double-click it to edit it.  Change it to be the right folder for your music files (probably %USERPROFILE%\Music), then click OK and exit the registry editor.


  4. Reboot your computer.


  5. Open your folders again and see if the music and downloads are now separate.  If it's good, you're done.


  6. If it still shows two Music folders, you may need to go into folder options, turn on "show hidden files and folders" and then delete the desktop.ini file inside the folders (if there is one) and it will restore the proper name of the folder (and icon).
