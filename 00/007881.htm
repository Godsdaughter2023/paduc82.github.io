charles kenyon say:
Attaching the Normal template can be done with a macro but that is probably overkill. On the merge I do every day, there is a finalization macro that does attach to Normal, but that is to get rid of unneeded toolbars.

Again, have you tried with the template open (not a doc based on template) Mailings > Start Merge > Normal Word Document. Then save the template.

When you reopen the template, or create a new document based on the template, it should not display the message. Instead, it should give you a normal Word document to which you have to attach a merge data source to have it run as a merge.

In my daily merge, I use a macro that
1. creates a new document based on the template (which is not attached to a data file)
2. attaches my datafile
3. starts the merge to a new document for editing

I have a separate macro that then attaches a separate utility template that has toolbars to use in editing the final merge document. On one of those toolbars there is the command to finalize the merge and clear markup formatting. That macro also attaches the Normal template to the document.

The macro to attach normal is below. Again, I think it may be overkill.

Sub AttachNormal()
'
' AttachNormal Macro
' Macro written 8/17/2007 by Charles Kenyon
'
    If ActiveDocument.Name = "MyMerge.dot" Then
        MsgBox "Sorry. You need to merge to a new document first. (Alt + Shift + N)"
        Exit Sub
        ' This checks to make sure it is being run in the merge result document rather than the template.
    End If
   ' CleanUpText 'See procedure below
    'BoldBack ' See procedure below - undoes part of CleanUpText
    With ActiveDocument
        .UpdateStylesOnOpen = False
        .AttachedTemplate = _
            Application.Options.DefaultFilePath(wdUserTemplatesPath) & "\Normal.dotm"
    End With
End Sub

entegy say:
The lock screen has five slots. The first two are taken for missed calls/voicemail and notifications from the Messages app. This cannot be changed.

The next three slots are taken by emails. The third slot usually ends up being your Windows Live Hotmail account. Turning that off frees up the slot.

The order in which you add accounts to the phone does matter. So does linked inboxes. The first three accounts you add get a slot each. After that, no slot on the lock screen. However, a linked inbox can be used to get the fourth-or-beyond account on the lock screen. Open up one of the inboxes that have a lock screen slot and link one of the other inboxes. Linking two inboxes that each have a slot will free up one slot.

Here's how this works on my phone. I have four email accounts on the phone: My personal, my professional, my public and my work email. I added them in that order, which means personal, professional and public got slots 3, 4 and 5. Work will not show up on the lock screen. I linked personal and public. This freed slot 5, which work moved into. Here's where it gets weird. I haven't tested it, but I don't think unlinking personal and public will give public's slot back and boot work off the lock screen again.

This is a long way of saying that order matters to get a slot on the lock screen and to use linked inboxes to "cheat" the slots.
