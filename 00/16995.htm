Macro to highlight texts based on list - Microsoft Community

retdlj say:

Macro to highlight texts based on list

Hi -- I'm hoping to develop a macro that can basically find words or strings of words and highlight that text in a Word doc. The words/word strings would be listed in another, separate Word or (better yet) Excel doc.

Any help would be much appreciated, thanks!

john korchok say:
                                    

Sub HighlightFromTableList()
  'Set the variables
  Dim oHighlights As Document, oDoc As Document
  Dim oTable As Table
  Dim oRng As Range
  Dim rFindText As Range
  Dim i As Long
  Dim sFname As String

  'Change the path in the line below to reflect the name and path of the source words
  sFname = "Mac HD:Users:YourUserName:Documents:WordHighlights.docx"

  'The document to be highlighted
  Set oDoc = ActiveDocument

  'Open the source word doc
  Set oHighlights = Documents.Open(FileName:=sFname)

  'Find the first table
  Set oTable = oHighlights.Tables(1)

  'For each row in that table
  For i = 1 To oTable.Rows.Count
    Set oRng = oDoc.Range

    'Get the text
    Set rFindText = oTable.Cell(i, 1).Range

    'Remove the cell ending characters
    rFindText.End = rFindText.End - 1
    With oRng.Find
      .ClearFormatting
      .Replacement.ClearFormatting

      'Find all instances of the word
      Do While .Execute(findText:=rFindText, MatchWholeWord:=True, MatchWildcards:=False, Forward:=True) = True

        'Highlight them in yellow
        oRng.HighlightColorIndex = wdYellow
      Loop
    End With

  'Get the next word to be found
  Next i

  'Close the source word doc
  oHighlights.Close wdDoNotSaveChanges
End Sub

Disabling Proofing for All Unrecognized Words in a Document

When I go to “Review” > “Spelling & Grammar” and click on “Ignore All,” the program marks as ignored only the same words as the word currently selected, but not all the other words in the document. How to make the program mark as ignored all words which it does not recognize?

charleskenyon (ia) say:
1. If you click ignore on a word and then copy that into another document, it will be checked in that document.
2. There is no automatic way to do the ignore for everything that has popped up.

A character style with the do-not-check proofing language characteristic is the closest you will come.

If you select all text in your document and apply that proofing language characteristic text copied from there into another Word document will not be checked.

Note that in some ways this can be like a virus so that if you have one of these words and continue typing, the do-not-check characteristic will be carried forward. This is like if you have a word in Bold and continue typing the continued text will also be bold. The difference is that with the Bold formatting you notice it. With the proofing language it is invisible.
