¡firedog say:

OK. I've just re-read your original post. You say both running Windows 8.1 and Windows Live Mail with the same email address. It sounds as if this means that you are signed in in Windows Live Mail with the same Microsoft Account on both machines. If this is the case, then the contacts list from that account will be available in the contacts window. There is no need to import anything - it happens automatically. If you do import a contacts list, then all of the imported contacts will appear in addition to the ones that were there to start with. If it's the same list that you import, then of course there will be duplicates ...

Many users don't grasp the significance of signing in in Windows Live Mail. When you do, the right-most button on the Home tab of the ribbon will show your Microsoft Account display name and avatar, if there is one. When you are signed in, you see the contacts list from people.live.com and the calendar from calendar.live.com. If you don't sign in, or sign out, then you see the default contacts list and calendar that only exist on the computer you're using at the time.
