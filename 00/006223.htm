Cannot update. Database or object is read-only.

Microsoft Access can't find the wizard. This wizard has not been installed, or there is an incorrect setting in the Windows Registry, or this wizard has been disabled.

mukherjee ashutosh say:  

The page setup in publisher needs to match the paper size used for printing

Following steps will guide you on how to check the page size in publisher:

1. Open the publication already created.

2. Click on 'Change page size' on the 'Task Pane' on the left side of the document.

3. Choose the exact paper size that you will use to print the document.

4. Click on File > Print > paper size> choose the same paper size used for printing.

 

For more information on page size and orientation in Publisher  you can check the following link: http://office.microsoft.com/en-us/publisher/HP051148411033.aspx

Hi Chan,

Thank you for posting your query on Microsoft Community.

I appreciate you for providing details about the issue and we are happy to help you.

You can change your primary alias up to twice a week. If your primary alias ends *** Email address is removed for privacy ***, @msn.com or @live.com, and you change it, you won't be able to make it primary again.

You can create up to ten new aliases per year, for an overall maximum of ten. Deleting an alias removes it from the overall count, but not the yearly limit. Existing @hotmail.com, @live.com, Outlook.com and @msn.com addresses can't be added as aliases.

You can create an alias using an existing email address from other providers such as Gmail, Yahoo! Plus, or AIM Mail, as long as you haven't used the address to create a Microsoft account.
