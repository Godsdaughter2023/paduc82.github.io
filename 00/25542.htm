diane poremsky mvp (slipstick.com) say:

A pst file cannot be opened by outlook.com on the web (or by smartphones) but if you move or copy them into an outlook.com mailbox, it will be just like it was with the pst, but everything will be stored in an Exchange mailbox online, available anywhere you open the mailbox.  Instead of having the pst open in Outlook desktop, you'll open the mailbox. 

You will need to get the items into the Outlook.com folders - you can use cut or copy & paste or using the Move to/Copy to folder command, or Import. Once you do this, you will have them everywhere you view the outlook.com mailbox - using a web browser, outlook on your desktop, or using a smartphone. 

If you want to keep the pst as a backup, use Import rather than Move -

1. Add the Outlook.com account to your Outlook desktop profile, if it's not already there. 

2. Close the pst if its open in outlook

3. Then go to File, Open & Export, choose Import & Export and follow the steps to import, choosing the Outlook.com mail box. You can import the entire pst file or just the Tasks folder. 

Not to make it more confusing (because its really a simple process, easier to do than it sounds)... but if your microsoft account address is not an Outlook.com address, you'll need an outlook.com alias added to the microsoft account to outlook desktop.
