Based on my knowledge, the membership list for dynamic distribution groups is calculated each time a message is sent to the group, based on the filters and conditions that you define. Not on Windows XP! The default location for User Templates is %appdata%\Microsoft\Templates which on Windows XP equates to

C:\Documents and Settings\<user name>\Application Data\Microsoft\Templates.

It is also possible to set the User Template location to a preferred different location, though that does not appear to have been the case here.

However as others have indicated, the Word 2010 File Open/Save dialogs have the correct location pre-programmed in the Places Bar to the left of the dialogs as 'Trusted Templates'.
