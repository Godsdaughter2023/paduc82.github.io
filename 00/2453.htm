ken sheridan say: I think you meant 'long text'.

You could open the zoom box as Daniel suggests with the following code in the DblClick event procedure:

RunCommand acCmdZoomBox

Alternatively you could open your other longer form, or an abbreviated version of it which includes only a text box control bound to the long text field, in dialogue mode, filtered to the current row.  The code would be along these lines:

Const FORM_NAME = "NameOfFormGoesHere"
Dim strCriteria As String

' ensure current record is first saved
Me.Dirty = False

strCriteria = "MyID = " & Me.MyID

' open form filtered to current MyID value
DoCmd.OpenForm FORM_NAME, _
    WhereCondition:=strCriteria, _
    WindowMode:=acDialog

where MyID is the primary key of the subform's table, and is a numeric data type.  If the key happens to be of text data type then the code would be amended as follows:

strCriteria = "MyID = """ & Me.MyID & """"

Step 1 - Fix network issues

It is noteworthy that this solution is at the top because the network issue is primarily responsible for any Windows update. If you are suffering from any kind of upgrade problem then, first of all, check out the Internet / Network issues.
Please apply the following steps accurately:

a) Go to the search box and type "Identity and Repair Network Problems"
b) Select and execute "Identifying and repairing network problems".
c) Automatically scan your PC for any type of network or internet problem.
d) If any problem is detected, it will display it and ask you to fix it.
 
Step 2 - Enable and disable Windows Firewall

a) Look for the "Firewall" and click the Windows Firewall (Control Panel).
b) In the left pane, click Enable or Disable Windows Firewalls.
c) Then click the Enable Windows Firewall button and click OK.
 Structure of the action of this step (enable - disable - and activate ok)

Step 3 - Try uninstalling the Microsoft .NET Framework 4 Client Profile:

a) Go to Start> Control Panel> Programs and Features
b) Find the Microsoft .NET Framework 4 Client Profile
c) You now need to uninstall the Microsoft .NET Framework 4 Client Profile
d) Right-click and select Uninstall / Change
e) The uninstallation process may take a few minutes.
f) Then restart your computer
