ossiemac say:

Example workbook at the following link. The code is in Module1. I have deleted my email addresses that I used for testing and left it up to you to insert whatever addresses you wish for testing.

Example Workbook to Send Emails

The following is the code example.

Sub SendEmails()
    Dim objOutlook As Object
    Dim objMailItem  As Object
    Dim wsSource As Worksheet
    Dim wsUniqueList As Worksheet
    Dim rngEmailAddr As Range
    Dim rngUnique As Range
    Dim rngCel As Range
    Dim rngToFind As Range
    Dim strTo As String
    Dim strCc As String
    Dim strSubject As String
    Dim strBody As String
    Dim c As Long
   
    On Error Resume Next
    Set objOutlook = GetObject(, "Outlook.Application")
    On Error GoTo 0
   
    If objOutlook Is Nothing Then
        Set objOutlook = CreateObject("Outlook.Application")
    End If
   
    'Assign the worksheet with the email addresses to a worksheet variable.
    Set wsSource = Worksheets("Open Issues")
   
    'Add a temporary worksheet to create a unique list of Email addresses
    Set wsUniqueList = Sheets.Add(After:=Sheets(Sheets.Count))
   
    With wsSource
        .Select
        'Turn off any set AutoFilter filters
        If .AutoFilterMode Then
            If .FilterMode Then
                .ShowAllData
            End If
        End If
       
        'Assign column Q email addresses (including column header) to a range variable
        Set rngEmailAddr = .Range(.Cells(3, "Q"), .Cells(.Rows.Count, "Q").End(xlUp))
       
        'Copy the email addresses to the temporary worksheet
        rngEmailAddr.Copy Destination:=wsUniqueList.Cells(1, "A")
    End With
   
    With wsUniqueList
        'Note: Next line INCLUDES column headers. List not unique (contains multiple listings of email addresses)
        Set rngUnique = .Range(.Cells(1, "A"), .Cells(.Rows.Count, "A").End(xlUp))
       
        'Create a unique list (no duplicates) of the email addresses
        rngUnique.RemoveDuplicates Columns:=1, Header:=xlYes
       
        'Note: Next line EXCLUDES column headers. (This is the unique list of Email addresses)
        Set rngUnique = .Range(.Cells(2, "A"), .Cells(.Rows.Count, "A").End(xlUp))
    End With
   
    For Each rngCel In rngUnique    'Loop through unique list of email addresses
        If rngCel <> "" Then    'If any blank email addresses then exclude from processing.
            With rngEmailAddr
                Set rngToFind = .Find(What:=rngCel.Value, _
                                LookIn:=xlFormulas, _
                                LookAt:=xlWhole, _
                                SearchOrder:=xlByRows, _
                                SearchDirection:=xlNext, _
                                MatchCase:=False)
               
                'Next line probably superflous because all email addresses should be in the unique list.
                'but left in place in case other changes made that could result in NOT found.
                If Not rngToFind Is Nothing Then
                    strTo = rngToFind.Value 'Assign found cell value to the To variable
                   
                    'Create a string with the cc email addresses (if any)
                    strCc = ""  'Remove any addresses left from previous loops
                    For c = 1 To 3
                        If rngToFind.Offset(0, c) <> "" Then
                            strCc = strCc & rngToFind.Offset(0, c).Value & ";"
                        End If
                    Next c
                   
                    'Assign Subject column value to Subject variable
                    strSubject = rngToFind.Offset(0, c).Value
                   
                    'Create a message for the body of the email
                    strBody = "This is the body of the mesage."  'Edit as required
                   
                    Set objMailItem = objOutlook.CreateItem(0)
                   
                    With objMailItem
                        .To = strTo
                        .CC = strCc
                        .BCC = ""
                        .Subject = strSubject
                        .Body = strBody
                        .Display    'Can be commented out after uncommenting Send
                        '.Send  'Leave commented out until finished testing to suppress sending the emails
                    End With
                End If
            End With
        End If
    Next rngCel
   
    Application.DisplayAlerts = False
    'Delete the temporary worksheet with the unique list of email addresses
    wsUniqueList.Delete
    Application.DisplayAlerts = True
   
    'Clean up object variables
    Set rngToFind = Nothing
    Set rngCel = Nothing
    Set rngUnique = Nothing
    Set rngEmailAddr = Nothing
    Set wsUniqueList = Nothing
    Set wsSource = Nothing
    Set objMailItem = Nothing
    Set objOutlook = Nothing
   
End Sub

List styles, as defined in Word, only provide the numbering. For stability in numbering, you'll have to combine them with paragraph styles (one paragraph style per numbering level).

For heading and appendix numbering, the best solution is to make use of the built-in heading styles. See https://shaunakelly.com/word/numbering/numbering20072010.html.

suzanne s. barnhill say:

As Doug and Stefan have said, for situations such as appendices, using unused heading levels is the recommended solution. Word's built-in heading styles have a number of special properties not found in any other style (see https://shaunakelly.com/word/numbering/usebuiltinheadingstyles.html), including being able to use the style's paragraph number as a "chapter number" in page numbers and captions.

That said, however, if you just want a style to be displayed in the Navigation pane and included in the TOC, all you have to do is give it an outline level other than Body Text (this setting is in the Paragraph dialog). If you set it to Level 1, for example, it will be displayed at the same level as Heading 1.

The "default" aspect applies to the currently opened document. Once you make a shape the default, all newly-created shapes

Therefore, to make a shape parameter become the default in a blank document, you have to implement the change

We have to open if from within Word, as any other method will only open a copy of the template (i.e., a blank document). Use File > Open and navigate to Your_User_Name > Library > Application Support > Microsoft > Office > User Templates to open Normal.dotm. (If you can't see the Library folder, go to the Finder, open your home folder, then select View > Show View Options and place a checkmark next to "Show Library Folder.")

Once you are editing Normal.dotm, draw a shape (or paste it from somewhere else), set it as default, then erase it (otherwise it will appear at that position in every new document), save Normal.dotm and close it.
