You can define up to 9 different indent levels for each paragraph  Ruler object is tied to each level and not the entire shape

The following code first defines 3 indenet levels for each paragraph and then for each level defines a different indent level

With ActiveWindow.Selection.ShapeRange.TextFrame
   .TextRange.Paragraphs(1).IndentLevel = 1
   .TextRange.Paragraphs(2).IndentLevel = 2
   .TextRange.Paragraphs(3).IndentLevel = 3
   .Ruler.Levels(1).FirstMargin = 20
   .Ruler.Levels(2).FirstMargin = 40
   .Ruler.Levels(3).FirstMargin = 60
End With

 

Best regards, Harvey
