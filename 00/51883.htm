considerate_guy say:

Let's first get one thing straight. When you are working on a project within Movie Maker, what you are saving are the instructions for how to build your movie. Each step that you do (adding media files, making trims and splits, applying transitions and effects, adding titles, etc.) is recorded into a project file.

The project file is nothing more than a text file listing all the file names, the split points, etc. As such, the project file is NOT a movie. Nor does the project contain any actual video or audio. It's more like a save point for your editing session.

The next time you reopen a project, Movie Maker will read the list of files and then go search for them where they were before (by drive letter, folder name and then file name). So if these files are on a removable drive (say on a memory card instead of on the hard drive) or if the folder was renamed or the file names were moved, renamed or deleted, then you get a yellow exclamation point instead of a thumbnail for that clip.

That tells you that Movie Maker can't find the clip where it was before. So if you are using removable media (USB drive, memory card, external drive) and you think it won't be there the next time you open the project, you should instead copy all those files to the hard drive before you start working in Movie Maker. And don't touch those copies until you have actually finished all your edits and have made your movie.

And that's the key -- when you want to finalize all your work, you need to save it as a movie. At that point Movie Maker will follow all your instructions, load each of the source files, make all the actual splits, add effects, etc. and will make a new video/audio file that you can play back in your favorite media player, or you can share and/or upload to video sites.

So, don't go looking for a corruption in Movie Maker; this is all expected behavior. Instead, its just that you aren't keeping the source files in the same spot each time you restart the project. To avoid this, be sure you are only using files that are on your hard disk and in a place that won't get changed.

If for some reason you do need to move your source files after you have started your project, you can double-click on one of the broken yellow exclamation points and you'll be prompted to navigate to the new location. After you do that, you'll get the thumbnail back and the exclamation will go away. Any other videos that were in that same folder will also get fixed in this step.
