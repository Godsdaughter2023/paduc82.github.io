considerate_guy say:

To help explain things, I'm going to use the analogy of constructing a house.

You have all your construction materials (2x4s, plywood, drywall, windows, etc.) -- these are like your source files (video, photos and music)
Then you have your blueprint which tells how all those pieces go together, how things are cut/assembled/painted -- this is like your project.
Finally you have a completed house-- this is like your movie.

What you did was send the equivalent of just the blueprint without any of the construction materials.  The project just has a list of file names, where to split those files, how long they should be, what titles/captions to add, what animations/transitions to apply, etc.  But if you don't also send all the source video, photos and music, their project can't be used to make a movie.  Instead they'll see yellow triangles with exclamation points for all the missing files.

So in order for them to edit the blueprints, they would need both the project and the source files.  Additionally, they would need them to be in the same location as on the first computer.  So if you had your videos in C:\Users\NBB\Videos\MyFolder but they put theirs in C:\Users\HenryJones\Videos\DifferentFolder, the references in the project won't match and they will still see yellow triangles with exclamation points.  At least at that point, they can click on one of the triangles and choose 'Fix'.  They would then navigate to the new location and that would fix the references.

Getting back to the question of sending the movie, that would be like sending an actual copy of the final house.  Yes, they can view it (say in Windows Media Player), but if they want to make further edits, it's going to be a little harder.  That's because the video file has everything incorporated and you can't really pull it apart.  Similarly with a house, you might be able to paint it a different color, but it takes a lot more effort if you wanted to move a wall around.  But if you send them the blueprint and the materials, they could change the plan as much as they like and build their own version of the house.
