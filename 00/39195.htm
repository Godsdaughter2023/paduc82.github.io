graye say:

Microsoft Edge Favorites are located here:

C:\Users\userprofile\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\AC\MicrosoftEdge\User\Default\Favorites

Edge no longer uses that location (since November 2015).  Instead, Edge now stores its favorites in a database (and doesn't store them as files like you're used to with Internet Explorer).   So, those files (if there are any) are from last November, and won't be the current versions that you had before the crash.

Yeah, this problem is well documented on this forum... favorites just tend to magically disappear (typically associated with a windows update).  For some, it happens more than once... and for countless thousands of others, it never happens at all.

For the vast majority of folks, there is no recovery of your favorites. They're gone...

We haven't heard from Microsoft what the root cause is... or how to prevent it from happening again.  Instead, they merely suggest that you ignore the problem and reload your favorites from Internet Explorer.

However, "Reload your favorites, and hope it doesn't happen again"... isn't very comforting.

So, about the only thing that we can offer you is a way to perform a backup and restore of your favorites for "the next time" this happens. (Yeah, I know...)

There are two approaches... a simple one that only works if you're using the Edge "Sync your content" feature, and a more turn-key version where you install a 3rd-party application

1) If you are using Sync your content, you can copy the contents of the %USERPROFILE%\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\RoamingState for safe keeping.  And then just copy it back after a crash/reset

2) You can use the free 3rd-party EdgeManage application to perform a backup/restore (plus many more features).  This technique works without regard to the "Sync your content" settings.

. My recommendation would be to install the updates in batches, such 5 at a time, exclude hardware/driver updates. Avoid installing a combination of Microsoft Windows and Office updates at the same time. Also, try installing the small updates first. Restart each time you install the updates (its a chore, but it guarantees a more reliable experience). Try starting out with Microsoft Office Updates for programs such as (Word, Excel, Outlook, PowerPoint).

How do I reset Windows Update components

How to troubleshoot a problem by performing a clean boot in Windows Vista/Windows 7
