robert aldwinckle on forums say:

I found them in the HP_Administrator drive.


On a different drive?   Not just in a different directory?  What changed?

If it is only in a different directory usually all you need to do is switch the shell:Favorites setting.

First check to see where the OS thinks your Favorites are supposed to be.   E.g.  open a Windows Explorer window (Press Win-E and Alt-d etc.)  and enter:   shell:Favorites  in the Address bar.  

Unfortunately XP does not provide a nice UI solution to moving the location.  You need to use a registry hack to change it.   In fact though, in XP there is a slightly scarey quirky alternative:  you can drag-move the existing Favorites folder somewhere else and then the Shell Folders registry location for Favorites will be updated accordingly.   So I suppose if you didn't have a Favorites folder where you wanted it and you drag-moved the existing one there you would then have your Favorites where you wanted them.   However, I'm pretty sure that this would not work across drives.  E.g. all drags across drives are copies not moves.   Also, even within drives it is sometimes difficult to make XP know that you want to do a move, not a copy.   The registry entry will only be updated if you do a drag-move, not a drag-copy.   Right-click drag might be the safest way of trying to ensure that that happened.    Note to Vista or W7 lurkers:  this drag-move feature does not work in your OS.   W7 users have a new Properties Location tab dialog for much better usability of this case.   I don't know what Vista users see.

 

FYI
