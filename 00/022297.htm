rick rothstein, mvp - excel say:

Give the following a try. Put the indicated module code in a Module (Insert/Module from the VB editor's menu bar) and put the indicated sheet code in the code window for the worksheet you want to have this functionality (double click the worksheet item in the Project Window to bring up the worksheet's code window). Assign the ToggleColorSwapModeOn macro to one of your buttons and the ToggleColorSwapModeOff macro to the other. Then, click the "on" button and select cells inside and outside (or a combination of both) of your matrix area and watch the cell colors toggle back and forth depending on their current colors. Click the "off" button to turn this functionality off.

'*************** START MODULE CODE ***************
Public ColorSwapModeOn As Boolean

Sub ToggleColorSwapModeOn()
  ColorSwapModeOn = True
End Sub

Sub ToggleColorSwapModeOff()
  ColorSwapModeOn = False
End Sub
'*************** END MODULE CODE ***************

'*************** SHEET CODE ***************
Private Sub Worksheet_SelectionChange(ByVal Target As Range)
  Dim Cell As Range, IntersectMatrix As Range
  Const CellIndexColor As Long = 6
  If ColorSwapModeOn Then
    For Each Cell In Target
      If Not Intersect(Cell, Range("A1:C10")) Is Nothing Then
        If Cell.Interior.ColorIndex = 3 Or Cell.Interior.ColorIndex = xlColorIndexNone Then
          Cell.Interior.ColorIndex = xlColorIndexNone + 3 - Cell.Interior.ColorIndex
        End If
      End If
    Next
  End If
End Sub
'*************** END SHEET CODE ***************

Hi,

Are you transferring offline contacts to Windows Live Hotmail? If so, Windows Live Hotmail contacts can be imported through a (.CSV) file. You've mentioned that your tech guy already created a back up copy of the contacts. The next to do is to export the contacts to your computer to save it on a folder provided that the output file is (.CSV) format. For better assistance please refer to your email provider to walk you through the steps.

After you have exported the contacts, you may refer to the following steps on how to import it to Hotmail.

1. Sign in to https://contacts.live.com
2. Select Import from file. Microsoft Outlook (using CSV) should be selected.
4. Click Browse and locate file for the contacts you've exported.
5. Select the file then click Open.
6. Click Import contacts.

In addition, Windows 7 only supports Windows Live Mail as its mail client. The imported contacts through web mail will automatically reflect on Windows Live Mail client when you sign in.

If you're having an error woth your Windows Live Mail, please provide us the exact error for investigation.
