Tip #5 in this link may give you what you want in a slightly different way
5 Tips to make you a Rockstar

http://chandoo.org/wp/2008/03/13/want-to-be-an-excel-conditional-formatting-rock-star-read-this/


I have not been able to find an example where you can use icons to indicate a cell comparison directly. BUT, you can work around that limit using a helper column.  Setup a true/false indicator column that  you use to key the icons.

I defined the following formula to set the conditions =IF(A1>B1, 100, 0)  So if a is > b, set 100, else true.

Copy the formula to the rest of the helper column (C in this case)

Select an Icon set:

  1. Home tab
  2. Styles group
  3. Conditional Formatting drop down
  4. Select Icon Sets
  5. Select the 3 icon set: Green Up / Yellow Horizontal / Red down

This gives you the arrows you want, but still show the helper values, so with the column selected go back and edit the rule :

  1. Home tab
  2. Styles group
  3. Conditional Formatting drop down
  4. Manage Rules command
  5. make sure the right column is indicated in the "Applies to" column
  6. Edit Rule button
  7. click on the "Show Icon Only" check box
  8. OK button to close edit rule
  9. Apply button to see the new rule (hide numbers) applied, you can see it happen behind the dialog
  10. Ok button to close the dialog

Now when you change the values in the watched columns the arrows will switch to match.

OK. Assuming that the source string is in cell A2 then the following will extract the string separated by the first and second space.

 

=MID(A2,FIND(" ",A2,1)+1,FIND(" ",A2,FIND(" ",A2,1)+1)-1-FIND(" ",A2,1))

 

However, it gets more complex if you have a string with more than 2 spaces and want the string say after the 3rd space.

 

 

 

 

ossiemac say:

However, it gets more complex if you have a string with more than 2 spaces and want the string say after the 3rd space.

 

 


Further to my previous post, if it is always spaces that are the separators then the following will return from the 2nd space to the next space.

 

=TRIM(MID(A1,FIND("#",SUBSTITUTE(A1," ","#",2)),FIND("#",SUBSTITUTE(A1," ","#",2+1))-FIND("#",SUBSTITUTE(A1," ","#",2))))

 

Reason I said it works with spaces is because the Trim function takes care of leading and trailing spaces. If it is another character then need to use some maths within the formula to adjust start position and number of characters to return.

 

The # character can be be any character that will not be found in the source string.

 

Where you see 2 in the formula indicates to start from the 2nd space and finish at next space. Change that to a different number to return a different section of the source string.

 

If you want to return from the last space to the end then the easy way is to concatenate a space with A1 so that there is another space at the end of the string. Note that this only occurs in the formula; it does not affect the source string actually in cell A1.

 

=TRIM(MID(A1&" ",FIND("#",SUBSTITUTE(A1&" "," ","#",4)),FIND("#",SUBSTITUTE(A1&" "," ","#",4+1))-FIND("#",SUBSTITUTE(A1&" "," ","#",4))))
