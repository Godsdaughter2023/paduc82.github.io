Hi guys, this issue was frustrating a lot of people already with Windows 8 and Quickbooks 2012 (that is already unsupported so won't be updated). Luckily the solution to this is fairly easy!

  1. Download and extract the Windows 7 XPS document writer printer drivers from here Microsoft XPS Document Writer
  2. Open up your Control Panel. (If you open the Windows Explorer, Click on computer, and then Computer at the top, Control Panel is under there)
  3. Open Devices and Printers.
  4. Right click the Microsoft XPS Document Writer, and press Remove Device.
  5. Click Add a Printer.
  6. Choose Add a Local Printer.
  7. Select Create a New Port.
  8. Select Local Port. then click Next.
  9. In the Enter a Port Name field, type in XPS Port, then click OK.
  10. In the Manufacturer list, choose Microsoft.
  11. Select “Have disk”, and locate the inf file in the folder that you extracted earlier.
  12. Finish the Add new printer wizard.

Then open Quickbooks and enjoy PDFing again...

Hope this helps.

This device is an Intel Centrino wi-fi/Bluetooth adapter.  It is used in at least Dell and Toshiba computers, and probably other brands as well.  If you're having problems, then "Go to the support web site for this device and download and then install its latest driver."

You did not provide the LMP version as requested.  According to Intel's spec sheet, as released this device was compatible only with "Bluetooth 2.1, 2.1+EDR, 3.0, 3.0+HS".  I do not believe that there is a driver available that will add support for more recent versions of Bluetooth.  If the Bluetooth device that you're having difficulty with uses Bluetooth 4.x or later (as many newer Bluetooth devices do), your current adapter won't work.

You probably should buy a USB-connected Bluetooth adapter that is capable of Bluetooth 4.x or later.  If you do get one of these, you may want to disable the built-in Bluetooth adapter.  I don't know if it will be possible to disable Bluetooth separately from wi-fi (the built-in adapter is a combination wi-fi and Bluetooth device).
