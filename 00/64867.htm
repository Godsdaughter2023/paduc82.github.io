praseetha k say:

Hi grnangela,

• Since when are you facing this issue?

• Did you make any recent changes to the computer?

First check whether the SSDP Discovery and TCP/IP NetBIOS Helper are started. If not started, then try to start these services first and try to start Windows Media Player Network Sharing Service.

1. Log on to your computer as an administrator.

2. Click Start, and then click Run.

3. In the Run box, type services.msc.

4. In Services, find " SSDP Discovery and TCP/IP NetBIOS Helper services ", and then verify the status.

5. If the services are stopped, double click on the services.

6. In that window, ensure that the startup type is "Automatic."

7. Click on the Start button in that window to start up the services.

8. Close the services windows and check whether you can turn on Network Discovery.

You may try to temporarily disable the third party Security Software and firewalls and check what happens.

If the issue occurs due to third party Security Software, then try to install the new version.

Note: Enable the security software back after resolving the issue.

Hi Allison,

Which is the base language in the computer?

 

The base language is the language which was installed when you purchased the computer. If the base language was Danish, then you would have installed English language pack. If you are seeing the start screen and the start menu in Danish even after installing the English language pack, it is possible that the Windows Display language is still set to Danish. To change this setting, try these steps:

 

a)      Press Windows key +F, select the “Settings” tab below the search bar.

b)      Type “Language” and click on the Language tile.

c)       In the list of languages installed on the computer, click on the options link against English.

d)      Check the status under “Windows display language”.

e)      If the language pack is not installed, it will show if that language pack is available to be installed.

f)       Download and install the language pack.

g)      Save changes and reboot to check.

 

Try these steps and let us know the results. We’d be happy to help if you need any further assistance.
