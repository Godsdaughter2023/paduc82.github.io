Just to be a little pedantic, memory and disk space are not the same thing.  You did not run out of memory, you ran out of disk space.  You should consider getting a larger disk, if you can add one to your computer.

How did you configure the accounts in Outlook?  The Exchange account should not be a problem because it would have all the data on the Exchange server.  If you had an IMAP account configured for gmail, that's the same - all the data is on the server and will come back when you re-add the account to Outlook.  For Hotmail, you have a choice of access methods: POP, IMAP, or Outlook Connector.  Which were you using?  Only the POP account type will have issues because it uses a PST and if that PST can't be seen as an Outlook Data File by Outlook, then it's pretty well toast and you'll need to recover your data from your backup.  You do make frequent backups, correct?
