jllatham say:

We haven't directly addressed the issue of resizing the height to fit the text, so...

You'll notice on the contextures page that there is a section on resizing comments.  To resize for the height, they use a factor of either 1.1:1 1.2:1  but those are 'best guesses on average".  The only way to be certain you come close is by first doing some trial and measurements, because what you need to know is how many characters of a given font style and size will fit on 1 line of a comment box of the width you want.  Then you'd have to examine the text in the comment box, determine it's length - divide that by the average # of characters per line and round up.  Then you need to set the height of the comment box to that calculated value * the height needed to show 1 line of text.
