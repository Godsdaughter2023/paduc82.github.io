per jessen say:

Maybe you can use a listbox, and set the ColumnCount property to the required number of columns. The ListBox can be populated using the RowSource property.

Private Sub UserForm_Initialize()
Dim MyRng As Range
Set MyRng = Worksheets("Sheet1").Range("A1:C10")
Me.ListBox1.ColumnCount = MyRng.Columns.Count
Me.ListBox1.RowSource = MyRng.Parent.Name & "!" & MyRng.Address
'RowSource = Sheet1!A1:C10

End Sub

per jessen say:

The ColumnCount property is not the issue here.

Using a ListBox on a worksheet, we don't have a RowSource property, but a ListFillRange property!

Sub FillListBox()
Dim MyRng As Range
Set MyRng = Worksheets("Sheet1").Range("A1:C10")
With Worksheets("Sheet1")
       .ListBox1.ColumnCount = MyRng.Columns.Count
       .ListBox1.ListFillRange = "'" & MyRng.Parent.Name & "'!" & MyRng.Address
End With
End Sub
