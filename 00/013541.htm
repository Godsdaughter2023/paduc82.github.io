If you've successfully pasted the stock data into your worksheet and cannot use it in formulas, then the data as likely carried along some leading or trailing spaces or non-breaking spaces. This is common on data that has been copied from an external source, particularly that which has been copied from a web site.

Select all of the numbers and tap Ctrl+H (aka Find & Replace). Put a space in the Find what: and leave Replace with: blank. Click Replace All. Repeat with a non-breaking space. A non-breaking space can be either gently copied from one of the values holding it in the first or last place or you can hold down one of the Alt keys while typing 0·1·6·0 on the numpad (with Num Lock on) and then releasing the Alt key.

If this doesn't provide a solution then consider posting a readacted sample of the data to a public access file area like OneDrive, DropBox or Google Drive and post a link to the publicly shared file back here. A firsthand look at the actual data would help immensely.
