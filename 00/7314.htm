jay freedman say:
There's a very complex procedure in http://support.microsoft.com/kb/912679 (which is for Word 2003 but could be adapted to Word 2010), but its result is a hyperlink with fixed display text such as CLICK HERE. You  would have to run a macro on the merge output to change the display to the hyperlink's address.

If you're going to have to do something to the output anyway, there's a much easier solution. First, do this one-time setup:
  • Go to the Options dialog, click Proofing, click the AutoCorrect Options button, and go to the AutoFormat tab of the dialog (note: not the AutoFormat As You Type tab).
  • Clear all the checkboxes except the one for "Internet and network paths with hyperlinks".
  • Click OK.
  • In the Options dialog, click Quick Access Toolbar.
  • Set the category to Commands Not in the Ribbon.
  • In the list of commands, click AutoFormat Now.
  • Click the Add button in the middle of the dialog, and then click OK.

Run the merge, using the Edit Individual Documents command on the Finish button. The addresses are all plain text. Click the AutoFormat Now button on the Quick Access Toolbar, and the addresses will all be converted to hyperlinks.

suzanne s. barnhill say:

Since you're merging to a new document, which allows for post-processing, here's a technique you can try.

  1. First go to File | Options | Proofing: AutoCorrect Options.
  2. On the AutoFormat tab (not AutoFormat As You Type), clear all the check boxes except "Internet and network paths with hyperlinks."
  3. OK back out to the Options dialog and select Quick Access Toolbar.
  4. Under "Commands Not in the Ribbon," select the AutoFormat Now command and add it to the QAT.
  5. When you click that button on the QAT, your text email addresses will be converted to hyperlinks.

jay freedman say:
Oops, sorry about that. The VBA documentation is incorrect -- the AutoFormat method is documented as belonging to the Document object (and apparently the VBA compiler accepts it as such), but it actually works only when it's called as a member of a Range object. (That probably sounds like gobbledygook to you, but Doug will understand it.)

That means you should change the inserted line to

    bDoc.Range.AutoFormat

Because the document's content is a specific type of Range object, you could use

    bDoc.Content.AutoFormat

to do the same thing.

how to select and open all excel files in folder at once

chip pearson say:

Use code like the following:

 

 

Sub OpenAll()
    Dim WB As Workbook
    Dim FName As String
    Dim FolderName As String
   
    FolderName = "G:\A" '<<< CHANGE FOLDER NAME
    ChDrive FolderName
    ChDir FolderName
    FName = Dir(FolderName & "\*.xls*")
    Do Until FName = vbNullString
        Set WB = Workbooks.Open(Filename:=FName)
        ' Do something with WB if desired
        FName = Dir() ' Dir returns empty string if no more files
    Loop
End Sub

 

Change the line marked with <<< to the appropriate folder name.
