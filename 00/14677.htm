jeffreycoley say:

Excel can't add a trend line to certain types of graphs - mine for example is a stacked bar, and Excel grays out the "Trend line" option.

Here's my workaround: I added a third data series for the "total" (sum of my stacked data series); this gives me a stacked bar twice as high as I want. Then I right-click on my new "total" series and change the chart type to "line", giving me my original stacked bar plus a line connecting the tops of all the bars.

Right-click on the "Total" line, and "Add a trendline" will be active. Click on it and Excel will create the trend line for the "total" series.

Lastly, get rid of that "Total" line cluttering up your chart. Right-click on the "Total" line and select "Format Data Series". Change formatting to make the markers and fill "No Color" and line style "No Line". Hey presto! Now your stacked bar chart has a trend line, based on the now invisible "Total" line.

Hope this helps!

gavinbollard say:

You can't actually uninstall OneDrive. It's built-in to Windows 10. 

I'm surprised that Colin told you to unlink the OneDrive though. 

There are two main ways to work with OneDrive / SharePoint files. 

1. In the Cloud using the OneDrive WEB interface

2. In file explorer

Personally I'd recommend the cloud but it takes a lot of getting used to. Most of our people still want to use the familiar windows file explorer.

If you want to use File Explorer, you have to keep "synching" 

however... 

You can control what is actually synched. 

If you right click on the blue cloud in the system tray (near the clock) you can bring up settings. That will tell you what areas are synching.  If they're not synching, then they're not available via file explorer. 

If you've got all the current windows updates, you can go into your file explorer window and look at the icons next to the folders.

  • The Solid green ticks mean that the folder is permanently synched.
  • The Green ticks on a white background mean that the folder is synched now because you've recently used the file (it will synch all files that are recent).
  • The cloud icons mean that the file is not synched (only the filename/placemarker). It's actually cloud-only.
  • There's a couple more icons (blue arrows for "synching now" and red stop sign for "issues").

To make changes, right click on various folders and choose 

  • Always Keep on this Device (for stuff you want locally)
  • Free up Space (for stuff that you don't want synched)

So.... the two recommendations are;

1. If you don't need to use file explorer, don't synch anything,

2. If you do need to use file explorer, synch the folders but then choose to "free up space" so that nothing is used on your hard drive. 

Good luck.
