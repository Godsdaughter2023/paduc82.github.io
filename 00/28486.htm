How can this be done. 

Thanks

Tim

tconway say:

Found the answer:

1. Open Excel for IPad, then go to the Open Excel files section where you can add a Place, Open Files and your previous files are all listed. 

2. In the Left vertical Taskbar where you'll also see New and Recent, select the main account Icon which should be the IPad User's Name or similar (the top icon on the task bar) 

3. In the list of Places (where you will see "OneDrive", "IPad", "+ Add a Place" and if you've added a new one, that one as well) at the very top you will see the word Account and to the right, Edit in green.  Press Edit

4.  Now the newly added "Place" will have a red dot with horizontal white line, press it and Remove Place button will appear. 

5.  Press Remove Place

Tim

Close Word and locate the Word startup folder - If you have not changed the preferred startup folder it can be located (in English language versions of Windows) by typing or copying

This is an add-in that frequently causes problems and the first named file is its lock file orphaned following a Word. crash.
