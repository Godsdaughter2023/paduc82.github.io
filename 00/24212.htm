rick song msft say:

Hello Stefantel,

 

The ost file is a copy of the items that are saved on a mail server. When you work offline in Outlook client, you make the changes to the ost file. And when you work online again in Outlook client, the data will sync to the mail server.

 

However, when you sign in Outlook Web App(OWA), it may need some time to sync the data from mail server to the OWA. So it is a normal behavior that you need to open the folders to let them sync to the OWA, which means, the data needs to sync to the server, then sync to the OWA when you sign in OWA.
