ken sheridan say: Just noticed a missing dot operator, it should have been:

SELECT OrderDetails.ProductID, UnitPrice,
Quantity, Quantity*UnitPrice AS GrossPrice
FROM OrderDetails INNER JOIN ProductPrices
ON OrderDetails.ProductID = ProductPrices.ProductID
AND (OrderDetails.Quantity BETWEEN ProductPrices.LowerVol AND ProductPrices.UpperVol);

The above is the SQL statement for a query.  If you've set up the tables as I described, then open the query designer in the usual way, switch to SQL view and paste the above into the window in place of what's there already.

While this could be used as the RecordSource for a report or a non-updatable form, the query is not updatable, so would not be suitable as the RecordSource of a form in which order details are entered.  For that you'd need to call the VBA DLookup function in the query rather than using a JOIN.  It would then be updatable:

SELECT OrderID, ProductID, Quantity,
DLookup("UnitPrice", "ProductPrices",
    "ProductID = " & [ProductID] &
     " And " & [Quantity] & " Between LowerVol And UpperVol")
AS [Unit Price], Quantity*[Unit Price] AS [Gross Price]
FROM OrderDetails;

Normally order details would be entered via a subform in an Orders form (see Northwind for an example), so the above could be used as such a subform's RecordSource.  One thing to note is thet the Quantity column in OrderDetails must be given a DefaultValue property of zero as the DLookup function would return an error otherwise until the quantity is entered in the form.

Invoices and orders are of course essentially modelled in the same way as orders, so the above would work with an invoicing system.  If you look at Northwind you'll see that in it an invoice is just an expression of the order data in a different way.

This is required as per the Audit by the Project

Can the below Macro be fine tuned for the same. The Below Macro list  all the fonts other than arial & give me the page number where the font is located in the 300 page document. I need to modify the macro to give me all the fonts & its size in the new macro

 

Sub FindAllFonts()
    Dim lWhichFont As Long
    Dim sTempName As String
    Dim sBuffer As String
    Dim newDoc As Document Dim p As Long
    Application.ScreenUpdating = False
    For lWhichFont = 1 To FontNames.Count
        sTempName = FontNames(lWhichFont)
        If sTempName <> "Arial" Then
            p = FindThisFont(sTempName)
            If p > 0 Then sBuffer = sBuffer & sTempName & " on page " & p & vbCrLf
            End If
        End If
    Next lWhichFont
    Set newDoc = Documents.Add
    Selection.TypeText Text:=sBuffer
    Application.ScreenUpdating = True
End Sub

Function FindThisFont(sName As String) As Long
    Selection.HomeKey Unit:=wdStory
    With Selection.Find
        .Text = ""
        .ClearFormatting
        .Font.Name = sName
        .Forward = True
        .Format = True
        If .Execute Then FindThisFont = Selection.Information(wdActiveEndPageNumber)
        End If
    End With
End Function
