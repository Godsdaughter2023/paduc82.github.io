"I keep getting the error message that it can't load the 3D....whatever."

- Unfortunately 'whatever' doesn't quite fill in the error message blanks. There are a number of potential 3D whatevers.

- What is the name of the game you are having problems with ?

If the game runs when you reboot or shut down and restart, there is no problem with DirectX. If there were, the game would never run.

Can you also be more specific about when the game crashes ? Does it crash when trying to play after resuming from Sleep or Hibernation ?

- It's not possible to 'downgrade' DirectX, and most likely, no need to 'update' or reinstall DirectX.

Windows 10 comes with DirectX 9,10,11 & 12, and while there may be older installers, the most recent installer (June 2010)

has the same files as the older installers (9.0c), plus newer DX Files that are not in the older installers.

This is the best first option if a corrupt DX file is suspected.

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

- There is also a -  DirectX Web Installer - This is an on line tool for automatically 'updating' or verifying all  DirectX files are installed.

This it the best first option if a DX files is 'missing'.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center


- Windows does not select which DX version or file the game is going to run on. The game decides which DX file it requires and

selects that file from Windows.  DX10 & 11 files do not come into play unless you select graphics settings in the game that

use DX10\11 files. Game that use only DX9 files are not bothered by DX10\11 issues.

- Nvidia do have a Win10 driver for your GT230M, but Nvidia also have a disclaimer regarding their generic driver being fully

suitable for laptops. This is because the mfg of the laptops often configure their hardware in such a way that you need to use

their driver for the best performance.

https://www.nvidia.com/Download/Find.aspx?lang=en-us
