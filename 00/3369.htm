When Word opens a document, it checks each word in the text to see which language it is marked with.

Word then starts the proofing tools (spelling and grammar) and calls the needed languages.  For this purpose, English US and English UK are two different languages.  There are 13 versions of "English".

So have a look in one of those documents and see what language(s) it is in.  Click in the text and go to Tools>Language... and see which language is highlighted.  Tell us which language it needs and we can tell you how to proceed.

Of course, if you installed Office without installing the proofing tools, or if you have run one of those applications such as "Monolingual" that falsely claim to "speed up" Office by deleting "unnecessary" files, then this problem is the result.  You will need to put the spelling checker files back.

[Edit] a little research and I found some comments that Charles Williams made regarding 2007 that shed light on it all, and it's actually simple to turn multi-threading off/on.  When you turn it off, it should go to single processor mode.  When you turn it back on, it goes back to the way the settings were previously.

 

Sub TurnOffMultiThreading()
  Application.MultiThreadedCalculation.Enabled = False
End Sub

 

Sub TurnONMultiThreading()
  Application.MultiThreadedCalculation.Enabled = True
End Sub

You can also set the .ThreadMode to either automatic or manual, and in manual, you can specify the number of CPUs to use with these internal values:

Application.MultiThreadedCalculation.ThreadMode can be xlThreadModeAutomatic
or xlThreadModeManual.
if set to Manual then Application.MultiThreadedCalculation.ThreadCount can
be set to the number of threads to use.

a. user say:

oh yeah, they will come back with a vengeance, this is

  1. run windows update
  2. on the left side click Change settings
  3. under the Important Updates section select the drop down box and change it from Install Updates Automatically (recommended) to download updates but let me choose weather to install them
  4. click Ok
  5. on the left bottom click Installed Updates
  6. uninstall KB2952664
  7. uninstall KB3022345
  8. uninstall KB3035583
  9. go back to the windows update main screen, click Check for updates
  10. the above three updates will appear, right click each one and select Hide Update
  11. at this point you can either leave updates set to download updates but let me choose weather to install or you can change it back to Install Updates Automatically (recommended)
  12. go to the windows update main page
  13. on the left side click Change settings
  14. under the Important Updates section select the drop down box and change it from download updates but let me choose weather to install them back to Install Updates Automatically (recommended)
  15. click Ok

The three hidden updates should now NOT re install

for more information on what the three updates do, see these threads or search the above KB numbers on this forum, the official info from MS is sneakily vague to say the least.
