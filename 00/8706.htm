The firmware 510.21.009 introduces severe stability problems with the USB connection to the PC. Problems start immediately during software update. Part of the software is a reset and installation of some support programs on the phone. That's the first that fails.

    How to verify data in excel - Microsoft Community

    mitsostsampallas say:

    How to verify data in excel

    Hello there, 

    I am working on a worksheet with 50 different names. On a daily basis, these names change cells. For example, 30 get the day off and the other 20 do other things around the company. These 30 are put together in a separate column. 

    My question is, how can I verify that the names put in this separate column can not be found anywhere else in the worksheet? And after verifying this, how can I check that the total number of  names is 50?

    Thank you for your time in advance. 

    andreas killer say:

    "anywhere in the worksheet" is too inaccurate, I will simplify the task to show you a possibility.

    Let us assume we want to check if column A contains only one "A", one "B" and one "C".

    C1:  A
    C2:  B
    C3:  C
    D1:  =COUNTIF(A:A,C1)
    D2:  =COUNTIF(A:A,C2)
    D3:  =COUNTIF(A:A,C3)
    D4:  =SUM(D1:D3)

    That's it, means make a list with all name, use COUNTIF and search for the names and if the sum is equal to the number of names your check is done.

    Andreas.

    mitsostsampallas say:

    Thanks for the answer. 

    Let me give a more specific example. 

    We have these 50 names. 30 of them are in cell ranges A10:A25, B1, C1:C5, D3, D5, D7, E25:E29. The other 20 are in column B40:B59.

    The numbers are random and do not correspond to any real data. 

    My question is, how can I verify that none of the first 30 names are found in range B40:B59? And, after verifying that, how can I check the total sum to be 50?

    Thanks again for your answers in advance. 

    andreas killer say:

    To create a sample file with formulas follow these steps:

    Make a new file

    Right-click on the sheet tab
    Choose "View Code"
    Within the menu click Insert \ Module
    Paste the following code
    Close the VBA editor

    Press Alt-F8 and run Test

    Columns G:H shows the check for B40:B59

    Columns I:J shows the check for all names

    If the sum in H21 is 0, the names in B40:B59 are unique.
    If the sum in K51 is 60, all names are unique.

    Otherwise a name appears more than once.

    If you want to find a duplicate name, select column A:E, press CTRL-F, enter the name and click Find All.

    Andreas.

    Option Explicit

    Sub Test()
      Dim MyFormula As String
      Dim Arr, Item
      Dim i As Long
      Dim AreaA As Range, AreaB As Range, A As Range, B As Range
      Dim Dest As Range
      Dim All As New Collection
      Set AreaA = Range("A10:A25, B1, C1:C5, D3, D5, D7, E25:E29")
      Set AreaB = Range("B40:B59")
     
      'Fill some random names in all cells
      For Each A In Union(AreaA, AreaB)
        Item = Chr(WorksheetFunction.RandBetween(65, 90)) & Chr(WorksheetFunction.RandBetween(65, 90))
        A = Item
        All.Add Item
      Next
     
      'Create formulas to count the names in AreaB in column G:H
      Set Dest = Range("G1")
      AreaB.Copy Dest
      Set Dest = Dest.Resize(AreaB.Rows.Count)
      ReDim Arr(1 To AreaA.Areas.Count)
      For Each A In Dest
        i = 0
        For Each B In AreaA.Areas
          MyFormula = "COUNTIF(Where,Item)"
          MyFormula = Replace(MyFormula, "Item", A.Address(0, 0))
          MyFormula = Replace(MyFormula, "Where", B.Address)
          i = i + 1
          Arr(i) = MyFormula
        Next
        A.Offset(, 1).Formula = "=" & Join(Arr, "+")
      Next
      Set A = Dest.Cells(Dest.Cells.Count + 1)
      A = "SUM:"
      A.Offset(, 1).Formula = "=SUM(" & Dest.Offset(, 1).Address(0, 0) & ")"
     
      'Create formulas to count all names in column I:J
      Set Dest = Range("J1")
      i = 0
      For Each Item In All
        Dest.Offset(i) = Item
        i = i + 1
      Next
      Set Dest = Dest.Resize(i)
      ReDim Arr(1 To Union(AreaA, AreaB).Areas.Count)
      For Each A In Dest
        i = 0
        For Each B In Union(AreaA, AreaB).Areas
          MyFormula = "COUNTIF(Where,Item)"
          MyFormula = Replace(MyFormula, "Item", A.Address(0, 0))
          MyFormula = Replace(MyFormula, "Where", B.Address)
          i = i + 1
          Arr(i) = MyFormula
        Next
        A.Offset(, 1).Formula = "=" & Join(Arr, "+")
      Next
      Set A = Dest.Cells(Dest.Cells.Count + 1)
      A = "SUM:"
      A.Offset(, 1).Formula = "=SUM(" & Dest.Offset(, 1).Address(0, 0) & ")"
    End Sub
