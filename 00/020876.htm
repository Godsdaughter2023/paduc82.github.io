how to auto fill in sequence numbering on a filtered data excel - Microsoft Community

makhosonke mncube say:

how to auto fill in sequence numbering on a filtered data excel spreadsheet

how to auto fill in sequence numbering on a filtered data excel spreadsheet?
teylyn say:
Hello,

You can do this if you have a range that is NOT an Excel table. If you are using a table, you need to convert it to a range first.

Filter the range
Select the cells where you want the auto-numbering to go, for example the visible cells in column B, starting in the first visible row below the header 
hit F5 > Special > Visible cells only > OK
If the first visible row of data is in row 43, enter this formula into B43 -- but don't hit Enter yet!

=MAX($B$1:B42)+1

The formula refers to all rows above. In this example, row 43 is the first visible data row in the filtered range, so the formula refers to rows 1 to 42.

Hold down the Ctrl key and hit Enter. The formula will now be applied to all the visible cells.

Now you can remove the filter to see that the consecutive numbering has only been applied to the rows that you had previously showing.

Does that help?

sebastianj---01 say:

Indeed it does. I don't consider that granting application permissions is the best way to configure a firewall (personally I prefer to fix apps to specific ports and then open those ports in the FW).

 

I prefer it that way mainly because depending on the application, it could change ports on its own (for example, my BT client has a list of possible ports, and if one is closed, it goes through the ports range seeking for an open one). Hence, port mapping could get overly complicated and may be different at different moments. 

 

However, if its correctly set up, granting application permissions can be a good choice too.

 

TCPView is an utility from MS (Sysinternals). You can download it from here.
