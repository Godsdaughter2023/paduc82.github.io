Instead of using Top and Bottom wrapping, can you just insert the images In Line With Text? When an image is inline, an inserted caption will also be inline. You can format the image as "Keep with next" to keep it with the caption.

When an image has any text wrapping style other than In Line With Text, the caption will be inserted in a text box, which will also be wrapped. toddalbertson say:

This is probably not the answer you're looking for, but I've been formatting large documents for over a decade and I've found that building tables of contents or figures are much easier when done manually, and you don't have problems like the ones you describe.  The way captions and tables of figures are done manually is with bookmarks and fields.  If you're interested, it's done like this:

1.  Insert your illustration.

2.  Extend your canvas (or the space after the graph or table) so there's some whitespace below it.  This whitespace will contain your caption.

3.  Enter your caption as a textbox in the whitespace.

4.  Within the textbox, insert a bookmark in the figure's title.  This is found under Insert/Links/Bookmark.  When you name the bookmark, use underscores instead of spaces.  The bookmark will look like this when you're done.

5.  Now, where you want the table of figures, set up a blank page and create your styles for that page, as appropriate.

6.  For a dissertation, you'll probably want tabs with leaders.  Right-click your style (here, it's the one called "Figures Row") and select Modify.  The Modify Style dialogue box opens.  Select Format.

7.  Select Tabs.

8.  The Tabs dialogue box opens.  Insert a right tab where you want the page numbers to appear.  Ensure you select the appropriate leader.  Click Set to create the tab.

9.  Click OK in the Tabs dialogue box.  Click OK in the Modify Style dialogue box.  Now, enter your figures with a tab at the end of each line.

10.  Place the cursor at the end of the line that refers to the figure.  Go to Insert / Text / Quick Parts.  Select Field.

11.  The Field dialogue box opens.  Scroll down to PageRef.  Under Bookmark Name, select the bookmark you created earlier.

12.  Select OK and the field dialogue box closes.  The page number will now appear in your table.

You'll want to enter bookmarks for each illustration and PageRef fields for each line in your table.  The benefit of this method is having captions that always "stick" to your figures, illustrations, charts, and graphs.  The drawback is a little more time inserting bookmarks and making tables, but that time is normally saved not having to deal with the headaches you describe.

Whether you do this or find another solution, good luck!

ossiemac say:

COUNTIF function will count where there is one condition to be met.

COUNTIFS function where there are multiple conditions to be met.

However, attempting to compare  "" for a match is a problem and needs a workaround. The following combined use of both functions counts the number of Males disregarding column L and then subtracts the number that matches with "Male" in column F and then match with "x" in column L which leaves the number of males without "x" in column L.

=COUNTIF(F11:F32,"Male") - COUNTIFS(F11:F32,"Male",L11:L32,"x")

You could make it easier and just use COUNTIFS function on its own if you replace the "" with another alpha character.

bernie deitrick say:

What's wrong with it is that your code is based on a poorly written example in the help file - "Type" in the example is just a place- holder, and not the name of the property being set, which is xlChartType.

And parens are only used if an object variable is being set to the return from the addchart method.

expression.AddChart(Type, Left, Top, Width, Height)

So this would work:

    ActiveSheet.Shapes.AddChart xlColumnClustered, 100, 375, 75, 225

If you ever want to know what the property name actually needs to be to use the := form, use the autocomplete popup to help, as below, which shows "xlChartType" and not Type

ActiveSheet.Shapes.AddChart XlChartType:=xlColumnClustered, 100, 375, 75, 225
