OK, now I figured it out. Here are my instructions on how I did this. Hopefully it will help you out too!

File

Page Set Up

Now Click the Size Tab and change the 0 to 90 where it says Rotation and Click OK and you‘ll see that it rotated it.

Now Highlight all the letters inside the box by putting the cursor at the end of the last letter of the last word and hold down the Shift Button and press the Up Arrow twice.

Now Right Click and Click Copy.

After the last letter of the last word, hit the Enter Button twice and then Right Click and Click Paste etc for each.

Before you move on any further, you will need to Stretch the Text Box all the way over to the right and left side of the page as described above already.

Automatic capital letters at the beginning of lines - Microsoft Community

susanalexander2 say:

Automatic capital letters at the beginning of lines

I am trying to type poetry but each time I press return to start a new line the line automatically corrects to start with a capital letter. This is infuriating! How can I stop this happening?
charles kenyon say:

The reason for this is that you are pressing the Return key.

That tells Word you are starting a new paragraph, which you are not intending to do.

A new paragraph always starts with a new sentence and the AutoCorrect option capitalizes that new sentence.

Rather than turn off this handy feature, consider not starting a new Word paragraph. Instead, use Shift+Enter and start a new line within the existing paragraph (and sentence).

You won't trigger the AutoCorrect.

One other thing with annoying AutoCorrects is you can always use the Undo key Ctrl+Z.
