What changes when you want to record a program longer than 2 hours is that the DVD recorder has to compress the video at a higher ratio in order to fit more time on the 2-hour disc.

In other words, by compressing the video into smaller file sizes, you can fit more recording time (4, 6, or 8 hours) on the same, 4.7 GB 2-hour-sized disc.

However, keep in mind that recordings made at a longer mode length that 2 hours will be lower in quality due to the increased compression, it not only affects the video quality, can also affect playback on some DVD players. nareshtongbram say:

Can I delete Downloads folder & WinSxS folder

Original Title: C:Drive size went up like a Tsunami

Is this normal? Or do i need to delete it? PLiz Help me 

a. user say:

Hi Naresh,

Thank you for posting your query in Microsoft Community Forums.

Please be assured that we would do our best to assist you.

All the Windows updates get downloaded to the Downloads folder as a temporary location, so you can go ahead and delete downloads folder frequently for more disk space.

All of the components in the operating system are found in the WinSxS folder – in fact we call this location the component store.  Each component has a unique name that includes the version, language, and processor architecture that it was built for.  The WinSxS folder is the only location that the component is found on the system, all other instances of the files that you see on the system are “projected” by hard linking from the component store.  Let me repeat that last point – there is only one instance (or full data copy) of each version of each file in the OS, and that instance is located in the WinSxS folder.

In addition, to increase hard drive space, you may follow the below methods:

Disk Clean Up

  1. Click the Start button, in the search box, type Disk Cleanup, then click Disk Cleanup.

  2. In the Drives list, click the hard disk drive that you want to clean up, and then click OK.

  3. On the Disk Cleanup tab, select the file types you want to delete.

Note: To Remove all System Restore Points except the Most Recent - Click "Clean up System Files" and you will get another tab called More Options. Now click More Options, then click "Clean up" under "System Restore and Shadow Copies" then confirm the deletion.

   4. Click OK on the bottom of the Disk Cleanup window and in the message that appears, click Delete files.  

Uninstall Programs You Don’t Need

  1. Press Windows key+R, then type Control Panel.

  2. Click Add/Remove Programs and uninstall programs you don’t want.

Hope it helps. For any further Windows related assistance, feel free to contact us and we will be happy to help.

igor leyko say:
Hi Summer203,
task manager counts (on first tab) only a memory used by processes themselves. It does not count memory used by drivers, memory used by memory mapped file (some programs require such usage) and memory used by system.
But on performance tab all sorts of usage are counted.
Detailed analysis of memory usage one can get with RamMap tool.

a. user say:

There is no necessary correlation between the amount of RAM in use and the number of running processes in Task Manager. Moreover, Task Manager is a high-level overview; it's not a professional tool. The same is true of many utilities built in to Windows.

If you see high CPU utilization accompanied by hard-working ventilation fans, then you would be suspicious of having been attacked.
