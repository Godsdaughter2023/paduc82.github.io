These are EFI system and/or recovery partitions.  Microsoft is aware that these drives are appearing in Optimize Drives.  There is no need to be worried about them – they do not indicate a problem, and they do not need to be optimized. 

Microsoft is working to make changes in a future release to hide these unactionable partitions.
