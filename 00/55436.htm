lemp say:
  • Open Device Manager (click Start Orb, type Device Manager in the Search box, press Enter)
  • Expand the "Network adapters" section by clicking the arrow to its left
  • Right-click your wireless adapter and select "Uninstall"
  • Confirm that you want to uninstall, but do NOT delete the driver software

  • Check Network and Sharing Center to see if the "Manage Wireless Networks"link is now present.

If that doesn't work, the next step would be to completely uninstall the wireless adapter and its driver and then reinstall as if it were new.  To do this, you should download the current driver from the web site of the adapter's manufacturer.  What is the make and model of the wireless adapter?

What the information in the Network and Sharing Center shows you is that "ROMA-PC" is connected to a network by an Ethernet cable (that's what "Local Area Connection" means). 

If you were connected by wireless, you would see something like this:


If the router to which ROMA-PC is connect happens to be a wireless router, then ROMA-PC and all of your other computers that connect to that same router via wifi (whether the router's "network name" is "default" or something else) will all be connected to the same Local Area Network.

What was the problem that made you want to "manage wireless networks" on ROMA-PC in the first place? 

If there are several wireless networks near you that you want to be able to connect to using ROMA-PC, then you'll need to buy a wireless network adapter.  USB wifi adapters are the simplest to install, but keep in mind that the "ultra-mini" adapters do not have very sensitive antennas, so if the adapter is going to be placed between the back of your desktop and a nearby wall, you may not get very good connections.
