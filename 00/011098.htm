Using templates with ComboBox and AddItem - Microsoft Community

edentistry say:

Using templates with ComboBox and AddItem

I have created a Word template, with several ActiveX ComboBoxes. The ComboBoxes have items within them, when are added using AddItem command using the Document_Open event.

They work when I open the actual template file to edit it, but when I try and create a new document from the template, the ComboBoxes don't get the items added to them.

Why is that? I've tried using the Document_new event instead, but that doesn't work either.

Here is an example of the code I've used at the moment:

Private Sub Document_Open()
    With ThisDocument.ComboBox1
        .AddItem "1"
        .AddItem "2"
        .AddItem "3"

    End With

End Sub

Can anyone help me with this? Is there a better way to get the AddItems to occur to the ComboBoxes? I've tried the ComboBox1_Click event to use the AddItem, but this doesn't work either!

jay freedman say:

The root of the problem is that ThisDocument refers to the template (technically, to whatever file contains the executing code). Instead, refer to ActiveDocument, which is the document that has the focus at the time the macro executes -- either the new document based on the template, or an existing document that's being reopened.

For maintainability, such as adding or deleting choices in the combobox, make the initialization into a separate subroutine that you call from both Document_New and Document_Open:

Private Sub Document_New()
    LoadCombo
End Sub

Private Sub Document_Open()
    LoadCombo
End Sub

Private Sub LoadCombo()
    With ActiveDocument.ComboBox1
        .AddItem "1"
        .AddItem "2"
        .AddItem "3"
    End With
End Sub

Sub ColorFontAllRed()
    'Charles Kenyon
    ' 9 Feb 2019
    ' Changes all font colors other than black or Automatic to Red
    Dim i As Integer
    Dim j As Integer
    Dim rngWord As range
    Dim rngText As range
    '
    For i = 1 To ActiveDocument.StoryRanges.Count
        Set rngText = ActiveDocument.StoryRanges(1)
        For j = 1 To rngText.Words.Count
            Set rngWord = rngText.Words(j)
            If rngWord.Font.ColorIndex < 2 Then
                rngWord.Font.ColorIndex = wdClassicRed
            End If
        Next j
    Next i
    Set rngWord = Nothing
    Set rngText = Nothing
End Sub
