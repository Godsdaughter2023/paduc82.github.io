ross jaburg say:
I have an hp dv9500 and want to know if I should install 32bit or 64 bit Windows 7? Also do my nvidia drivers need updating before or after I install

Bear in mind you cannot do an in-place upgrade from 32-bit to 64-bit. It will have to be a fresh install.

If you have a 64-bit CPU, definitely go with the 64-bit version of OS.
You are not required to have more than 4GB of RAM, but if you want it at least it will be supported when you are ready.

32-bit OSes do not support more than about 3GB (varies by machine.)

Yes, you will need to update to 64-bit drivers for video and other devices.
Also, 64-bit OS requires all drivers to be digitally signed so some devices may not have updated drivers yet ... or ever (depends on mfr) but you may still be able to use the 32-bit ones that are available.

compatibility between a 32 -bit and 64-bit office documents

I am investigating the likely issues I may have in sharing office documents and other common file types between a Win10 Home 32-bit PC and a Win10 Pro 64-bit PC.

My understanding is that provided software uses the same address space (32-bit / 64-bit) on networked computers then files can be transferred between them, and accessed on any of them, whatever the CPU address space.

Therefore, if there is a 32-bit PC in the network, then the software on all the platforms has to be 32-bit for seamless file sharing and file access.
