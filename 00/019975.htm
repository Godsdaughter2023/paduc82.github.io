anoop k m say:

No updates were released last week that can corrupt the rules in Outlook.

 

What type of account is configured in Outlook? (POP/IMAP/Exchange/MAPI)

 

Create a new PST file in Outlook, then set that PST file as the delivery location and check if the rule works fine.

http://office.microsoft.com/en-us/outlook-help/create-and-use-pst-data-files-in-different-versions-of-outlook-HP001231987.aspx

 

If it works fine, you may run a scanpst on the original PST file using the Inbox Repair Tool and see if it helps.

http://office.microsoft.com/en-us/outlook-help/repair-outlook-data-files-pst-and-ost-HA010075831.aspx?CTT=1

Note: Close Outlook and take a backup copy of the PST file before you run the scanpst tool.

 

If this fails, create a new profile in Outlook, then configure the account and check if the rule works fine in the new profile.

http://support.microsoft.com/kb/829918
