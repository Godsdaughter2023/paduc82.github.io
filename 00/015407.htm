stefan blom say:
"List Paragraph" is a paragraph style that Word uses by default to format any paragraph (spacing before/after, line spacing, etc.) to which you apply a numbered or bulleted list. It has nothing to do with the list format that you select.

Note that you get more control if you actually make use of a bulleted or numbered style. For example, try the built-in List Bullet or List Number styles.

Recent versions of Word should be able to open PDFs, but you may not be able to edit the files, depending on how they were first created. Typically, PDFs consisting of text are editable in Word, but not PDFs created by a scanner which must first be converted into an editable format via OCR (optical character recognition) software.

andreas killer say:

Try the code below.

BTW, there is not really a need to make the sheet visible, you can also copy the values from a hidden sheet.

Andreas.

Option Explicit

Sub SecReports()
  Dim path As String, ppe As String, filename As String
  Dim Wb As Workbook
  Dim Dest As Range
 
  Application.ScreenUpdating = False
  Application.DisplayAlerts = False
  path = "C:\Priya\"
  ppe = "2017-0702"
 
  'ppe is going to be a user defined input, but I kept it as a variable to make testing easier

  filename = Dir(path & ppe & "\" & "*.xl??")
  Do While filename <> ""
    'Find empty cell in column A in our file
    Set Dest = ThisWorkbook.Sheets("Sheet1").Range("A" & Rows.Count).End(xlUp).Offset(1)
    'Open the file
    Set Wb = Workbooks.Open(path & ppe & "\" & filename)
    'Run the macro from the regular module
    Application.Run filename & "!ComputeSecReport"
    'Copy all
    Wb.Sheets("SecReport").UsedRange.EntireRow.Copy
    'Paste into our sheet
    Dest.PasteSpecial xlPasteValues
    'Done
    Wb.Close False
    'Find next file
    filename = Dir()
  Loop
End Sub
