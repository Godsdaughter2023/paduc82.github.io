how do I add page numbers 1a, 1b, 1c, etc between pages 1 and 2?

jay freedman say:
  • Insert a Next Page break (Page Layout > Breaks > (Section Breaks) Next Page) between the original page 1 and the new material. Insert another Next Page break between the new material and the original page 2.
  • Open the header pane of the original page 2 (double-click in the header area). [Note: if your page numbers are in the footer, then open the footer pane instead.] Click the Link to Previous button in the Navigation section of the Header & Footer Tools ribbon to turn it off. Click the Page Number button on the same ribbon, choose Format Page Numbers, type the number 2 in the Start At box, and click OK.
  • Click the Previous button on the Header & Footer ribbon (above the Link to Previous and Next buttons) to go to the header (footer) of the section of the new material. Click the Link to Previous button to turn it off. Right-click the page number and choose Edit Field; in the Field Properties part of the dialog, choose the "a,b,c..." format, and click OK. In the header pane, type the number 1 (just as plain text) to the left of the page number so it says 1a etc.
  • Close the header pane. Your numbering should now work as you want.

gord dibben say:

You can use this UDF to keep track of summing previous sheets.

See this site for explanation of usage.

 

https://groups.google.com/forum/#!topic/microsoft.public.excel.newusers/WeFlVKJK3FY

 

Function PrevSheet(rg As Range)
'accounts for more than one workbook open
'and has hidden sheets
'Bob Phillips  October 4, 2009
Dim N As Variant
    With Application.Caller.Parent
        N = .Index
        Do
            If N = 1 Then
                PrevSheet = CVErr(xlErrRef)
                Exit Do
            ElseIf TypeName(.Parent.Sheets(N - 1)) <> "Chart" And _
                .Parent.Sheets(N - 1).Visible = xlSheetVisible Then
                PrevSheet = .Parent.Sheets(N - 1).Range(rg.Address).Value
                Exit Do
            End If
            N = N - 1
        Loop
    End With
End Function

Word Shows me the following Pop-Up every time I open:

john korchok say:

Assuming you are creating a new blank document, this message is telling you there is a problem with your Normal.dotm template. If you are trying to create a document from a particular template, it means that template has an error. You should replace the template file with a working copy.

The following assumes the issue is with every new blank file you create. To replace the Normal.dotm template, please follow these steps:

  1. Close Word.
  2. While holding down the Alt key, click on the Go menu and choose Library. Your hidden Library folder opens.
  3. Navigate to ~/Library/Group Containers/UBF8T346G9.Office/User Content/Templates.
  4. Drag Normal.dotm to your desktop.
  5. Open Word. It should open without the error message.
  6. Close Word. It creates a new Normal.dotm template with the default settings.

If you had kept custom styles, autotext, toolbars or macros in your Normal.dotm template, you can use Tools>Templates and Add-ins...>Organizer... to copy them from your old Normal.dotm to the new one. Otherwise, just put the old file in the Trash.
