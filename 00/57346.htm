Hello,

We're here to help.

Here are the steps to block software from accessing the internet:

  1. Return to the main Windows 8 interface (the one w/ all the ‘Tiles’ and is often referred to as “Metro”). Simply start to type out the words Control Panel. This will immediately bring up the “App Search” window, and from here, click on the Control Panel entry from the left column.
  2. From the Control Panel, click the System and Security link.
  3. Click Windows Firewall.
  4. Click the Advanced settings link in the left column.
  5. From here, you can monitor all of your Windows Firewall activity, and of course, create new rules for the firewall.
  6. Click the Outbound Rules link from the column of the left side of the window. Now, click the New Rule button from the column on the right side of the window.
  7. Select Program, and then click the Next button.
  8. Select This program path: and then click the Browse button.
  9. Navigate to the program you wish to ensure has no access to the Internet, select it, and click the Open button. If you have any difficulty finding the program you want to block, check the sub-folders in C:\Program Files (x86)\ and C:\Program Files\ – those two folders store most of the programs on your PC.
  10. Double-check to ensure you have the correct program selected in the This program path: box, and then click the Next button.
  11. Make sure the Block the connection entry is selected, and then click the Next button.
  12. Make sure there are ‘check-marks’ in all of the entries listed in the When does this rule apply? section, and then click Next.
  13. Give your new rule a name (descriptive helps) and then optionally, give it a more in depth description. Click the Finish button.
  14. In the Name column, your newly blocked program will be listed. In the Action column, you’ll see that your newly created rule is set.

Let us know if it helps.

michaelbaffoni say:

I ran into this problem as well, and found some other posts on this subject that led me down the path.

First I had to delete the "$WINDOWS.~BT" folder.  This was tough, even booting into safe mode and running these commands didn't fully delete everything in the folder (some files would say open by system), make sure to run from cmd "run as administrator":

takeown /F C:\$Windows.~BT\* /R /A
icacls C:\$Windows.~BT\*.* /T /grant administrators:F
rmdir /S /Q C:\$Windows.~BT\

Then I renamed "$WINDOWS.~BT" to "$WINDOWS.~BT.old" and rebooted.  I still wasn't able to delete all of the files/folders under that folder (something really weird there), but I was able to mount the .iso and run install properly (at least I didn't get the something is wrong reboot message).

Hope that helps!
