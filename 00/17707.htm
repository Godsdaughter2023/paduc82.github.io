n7i2s5 say:

If the Outlook icon is in the Notification Area (by the clock), then Windows will automatically minimize the window. To prevent this open Outlook from the Start Menu or pin it to taskbar.

If this problem occurs when Outlook is pinned to the taskbar, find the Outlook icon in the Microsoft Office folder in the Start Menu, Right-click on it, and click 'Properties'. In the 'Shortcut' tab, go to 'Run' and select 'Normal window' or 'Maximized'. Then try to open Outlook again. If it still opens minimized, right-click on the Outlook button on the taskbar and click 'Unpin this program from taskbar'. Then go back to the shortcut in the Microsoft Office folder in the Start Menu, right-click and select 'Pin to Taskbar'.

If you cannot tell whether the Outlook icon is pinned to the taskbar or is in the notification area, click on the Outlook button, if it is in the notification area, a seperate taskbar button will appear. If it is pinned to the taskbar, the Outlook icon will turn into a button, which may or may not have a text label.

Hopefully this is somewhat helpful.
