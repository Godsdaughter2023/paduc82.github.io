Location of files if I 'Reset this PC' with 'Keep my - Microsoft Community

david_designer say:

Location of files if I 'Reset this PC' with 'Keep my files' option

If I "Reset this PC" with the "Keep my files" option, do those files have to be in a specific location, such as a user Documents folder, or can they be anywhere on drive C?

My thanks, in advance, for clarification of this point.

a. user say:

That's an excellent question. Your files need to be in the default locations where Windows expects to find them. That means the Documents, Music, Pictures and Videos folders; and that is true for each user account on the computer.

But - there's always a 'but' - there are some tricky details.

The Public account and the Guest account aren't user accounts - they're system accounts. We're not supposed to be storing our data there, so any data stored in those places won't get backed up.

The contents of each user's desktop won't get backed up. That's because the desktop is also not a place to store data, even though everybody does it. The desktop is for shortcuts to data, not data itself. (There's a trick to keeping data on the desktop - but it's easier to simply copy everything on the desktop to a folder in Documents before you reset.)

Some user data is not stored in the 4 folders described above, including two things you'll want to keep:

* your email, calendar and contacts are stored in different places, depending on which services or programs you use for email. If all your email comes from online services - e.g., Outlook.com, Gmail, Yahoo! mail - there's nothing to backup since everything is stored online. If some of your email comes via an application - e.g., Office Outlook or Thunderbird - you need to get specific directions for backing up those applications.

* your internet bookmarks may or may not need to be backed up. If you sync your bookmarks to the cloud, e.g., with a Microsoft Account or Google Chrome or Mozilla Firefox, there's nothing to backup since everything is in the cloud. Otherwise you have to get directions on exporting bookmarks for your particular web browser.

Keep in mind that a reset is basically a clean install. Your hard drive will be erased and only your user accounts, data and Windows Store apps will be restored. Any software you purchased outside Windows Store will be gone, so make sure you have license keys for paid software.

Finally: It's always a good idea to backup your data before doing a reset. Because life holds no guarantees.

shinmila h say: Hey Captain Bigs,

You can use the Xcopy utility to determine whether the disc or the disc drive is bad. To run the Xcopy command, follow these steps:

1.    Insert the game discs in the CD drive. If the game setup starts, exit it.
2.    Click Start, point to Run, and then type the cmd in the start search field.
3.    Right-click on cmd and select “Run as administrator”.
4.    At the command prompt, Type the following command, and then press ENTER to access the root of the hard disk, which is typically drive C:
cd\
5.    At the command prompt, type the following command, and then press ENTER:
Xcopy CD drive letter:\*.* /s c:\text.txt
You receive the following message:
"Does C:\text.txt specify a file name or directory name on the target? (F=file, D=Directory)."
Press F for file. Click A to select All. Depending on the speed of the computer’s CD or DVD drive, the steps to run the Xcopy command may take several minutes to finish. If there are any files that cannot be read from the CD, the CD may be dirty or damaged, or the game CD may be corrupted.

In that case, it is advisable to get a new CD.

If that doesn’t help, try re-registering Windows Installer and install the game one more time.

To unregister and reregister Windows Installer, follow these steps:
1.    On the "Start" menu, click "Run".
2.    Type msiexec /unreg, and then press ENTER. This unregisters Windows Installer.
3.     On the "Start" menu, click "Run".
4.     In the "Open" box, type msiexec /regserver, and then press ENTER. This registers Windows installer.
Once you’re through with the registration, run the installation and see how it goes. You may opt to follow the above copy method to do the same.
