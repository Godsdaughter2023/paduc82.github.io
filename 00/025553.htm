stefan blom say:

If others should use the macro, it must be installed on their machines (because you can never force a person to run a macro). But note that once headers and footers are properly linked, the macro is no longer needed: By default, when a new section is created in Word, its headers and footers will be linked to their "neighbors" in the preceding section.

To delete a section in Word, you can delete the relevant section breaks in the document. Beware that you may lose formatting, as Word inserts section breaks if you format (say) part of a page in two columns. The rules that apply are far from obvious, and they are discussed in the article at http://word.mvps.org/faqs/formatting/WorkWithSections.htm.

To see section breaks, show nonprinting marks, for example by pressing the ¶ button on the Home tab.

stefan blom say:

Note that a section can have up to three different headers (and footers), which is the case if you have enabled "Different odd and even" and "Different first page" in Page Setup. Each header (and footer) must be separately linked to its neighbor in the preceding section. You can use a macro to accomplish this quickly:

 
Sub LinkAllHeadersAndFooters()
Dim s As Section
For Each s In ActiveDocument.Sections
s.Headers(wdHeaderFooterEvenPages).LinkToPrevious = True
s.Headers(wdHeaderFooterFirstPage).LinkToPrevious = True
s.Headers(wdHeaderFooterPrimary).LinkToPrevious = True
s.Footers(wdHeaderFooterEvenPages).LinkToPrevious = True
s.Footers(wdHeaderFooterFirstPage).LinkToPrevious = True
s.Footers(wdHeaderFooterPrimary).LinkToPrevious = True
Next s
End Sub
 
For assistance with the macro, see http://www.gmayor.com/installing_macro.htm.

I've just had exactly the same issue. I tried to solve it hardly, but appeared that only a few simple steps can resolve the situation.

Hi DeviousDevil,

 

I would like to know what view settings is applied on the both the Mail and Calendar navigation tabs.

By default Calendar is in Calendar view.

 

And on Outlook client normally the settings are:

Folder Pane – Normal

Reading Pane – Right

 

AS you are using Outlook 2016, try disabling the Compact Navigation (View > Layout > Folder Pane > Options… > Navigation Options).



 

This way setting the Folder Pane to “minimized” still shows the navigation options at the bottom.
