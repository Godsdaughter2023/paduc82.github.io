alik2010---01 say:

OK, I found it myself...

 

Open Email application (In Applications folder), open Settings, select your mailbox (e.g GMail), open Mailbox settings,  go to Advanced mailb. settings (the last item in the list). There you can see two options:

  • Incoming email settings
  • Outgoing email settings

 

Open Incoming email settings and change Access point in use  from Always ask to User defined. A new box pops-up and gives you a list of available access points.

Do the same thing for Outgoing email settings.

 

This will make e-mail app to use the defined access point for sending and receiving e-mails in that mailbox. If you have multiple mailboxes, then you have to repeat this procedure for each of them.
