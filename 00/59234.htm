Player.

The question is simple, how can I make windows to include subfolders in desktop wallpaper path in windows 7?


Hi

The default path to the Wallpapers is the C:\Windows\Web\Wallpaper folder. You can create new folders and sub-folders in there and they will all be visible, in separate sections, when you click the Desktop Backgrounds button, in the Personalize options.

If you use the Browse button and select pictures from any other location, any subfolders will not be shown and must be selected individually.
