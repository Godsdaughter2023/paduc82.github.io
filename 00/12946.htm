ken sheridan say:

I want that when I select a Test (by marking it True in Tests Table), and refresh Datasheet view of Results Table, the output records should be limited to this/these test(s) only.

Why on earth would you want to do that?  A table's datasheet should be no more than a basic representation of the actual values in all columns in all rows of the table.  You have already compromised that by using a combo box as the DisplayControl property of the Students_FK column.  The place to use such a combo box to return a value from a referenced table is in a bound form, not in the table definition.  A user should never interface with the data in a table's datasheet, but always in forms.

Moreover, you already have a qryResults query which returns the data as you want, though this is also poorly designed because it uses the combo box DisplayControl inherited from the table definition.  The query should return the values from the referenced tables by including those tables in the query.

One other point on which I would express caution is in your use of a student's name and date of birth as a distinct identifier.  I was once present at a clinic when two patients arrived within a few minutes of each other, both female, both with exactly the same names, and both with the same date of birth.  In most situations like this I would use address data as the distinguishing factor, though in a student environment, where many students might be housed together in a college or hall of residence, this would not be a reliable tie-breaker.

i assume that:
1) in row 1 are headings (Sheet1, Sheet2)
change sheet name as yours
2) export data in a new Sheet
in columns A:F
try this code..

Sub Compare2Shts()
Dim ws1 As Worksheet, ws2 As Worksheet, ws As Worksheet
Dim r1 As Long, r2 As Long, r As Long, X As Long, i As Integer
Set ws1 = Sheets("Sheet1") '<< change
Set ws2 = Sheets("Sheet2") '<< change
r1 = ws1.Cells(Rows.Count, 1).End(xlUp).Row
r2 = ws2.Cells(Rows.Count, 1).End(xlUp).Row
Application.ScreenUpdating = False
Application.DisplayAlerts = False
For Each ws In Worksheets
If ws.Name = "CompareSheet" Then ws.Delete
Next
Application.DisplayAlerts = True
Set ws = Sheets.Add(before:=Sheets(1))
ws.Name = "CompareSheet"
ws1.Range("A1:C1").Copy Destination:=ws.Range("A1:F1")
ws1.Range("A2:A" & r1).Copy
ws.Range("A2").PasteSpecial xlValues
r = ws.Cells(Rows.Count, 1).End(xlUp).Row + 1
ws2.Range("A2:A" & r2).Copy
ws.Range("A" & r).PasteSpecial xlValues
r = ws.Cells(Rows.Count, 1).End(xlUp).Row
ws.Range("A1:A" & r).Sort key1:=ws.Range("A2"), header:=xlYes
For i = r To 2 Step -1
If ws.Cells(i, 1) = ws.Cells(i - 1, 1) Then Rows(i - 1).Delete
Next
r = ws.Cells(Rows.Count, 1).End(xlUp).Row
For i = 2 To r1
X = Application.Match(ws1.Cells(i, 1), ws.Range("A1:A" & r), 0)
ws1.Range(ws1.Cells(i, 2), ws1.Cells(i, 3)).Copy
ws.Cells(X, 2).PasteSpecial xlValues
Application.CutCopyMode = False
Next i
For i = 2 To r2
X = Application.Match(ws2.Cells(i, 1), ws.Range("A1:A" & r), 0)
ws2.Range(ws2.Cells(i, 1), ws2.Cells(i, 3)).Copy
ws.Cells(X, 4).PasteSpecial xlValues
Application.CutCopyMode = False
Next i
For i = 2 To r
If ws.Cells(i, 2) = "" Then ws.Cells(i, 1) = ""
Next
ws.Columns("A:F").AutoFit
Application.ScreenUpdating = False
End Sub
