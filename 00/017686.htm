graham mayor say:

Not strictly what you had in mind, but if you create a set of rules to shuffle your inbox messages to their respective folders then uncheck them all in the Rules Editor list they will not run automatically. You could then run an Outlook macro from time to time to run some or all of the rules on the content of the inbox. The macro takes the form of a Userform with a list box and a command button combined with the following code (you can download the userform fromhttp://www.gmayor.com/Extras/RulesForm.zip, Unzip and import it into Outlook vba if you prefer.)

NOTE:- The macro will run on ALL the messages in the inbox, so read them before running the macro.

Option Explicit
Private st As Outlook.Store
Private myRules As Outlook.Rules
Private rl As Outlook.Rule
Private count As Integer
Private ruleList As String
Private i As Long

Private Sub UserForm_Initialize()
Set st = Session.DefaultStore
Set myRules = st.GetRules
Me.BackColor = &HC0FFFF
Me.Caption = "Select the Rules to Run on the Inbox"
Me.CommandButton1.Caption = "Run the Selected Rules"
With Me.ListBox1
    .Clear
    For Each rl In myRules
        .AddItem rl.Name
    Next rl
    .MultiSelect = fmMultiSelectExtended
End With
End Sub

Private Sub CommandButton1_Click()
Me.Hide
Set st = Session.DefaultStore
Set myRules = st.GetRules
With Me.ListBox1
    For i = 0 To .ListCount - 1
        If .Selected(i) Then
            Set rl = myRules.Item(.List(i))
            If rl.RuleType = olRuleReceive Then
                ' if so, run it
                rl.Execute ShowProgress:=True
                count = count + 1
                ruleList = ruleList & vbCrLf & rl.Name
            End If
        End If
    Next i
End With
ruleList = "These rules were executed against the Inbox: " & vbCrLf & ruleList
MsgBox ruleList, vbInformation, "Outlook Rules"

Set rl = Nothing
Set st = Nothing
Set myRules = Nothing
Unload Me
End Sub

The userform is then called with the macro

Sub RunMyRules()
frmRunRules.Show
End Sub

Follow the instructions on the userform

FPP was originally "Full Product Package", referring to the boxed CD/DVD packaging.  It has been extended to include online downloads that are explicitly identified as a FPP license which cost somewhat more than the PKC equivalent. The most obvious license term that confirms FPP is identifying it as a 2-computer or 3-computer installation.

Unfortunately the marketing of University is extremely vague on the point of license type. I have seen hints of both types of license, but nothing definitive.

PKC or Product Key Card license is a variation of OEM and MLK (MediaLess Kit") licenses, with the same terms. Install on only one computer, not transfer rights.

The bundle you described is somewhat a mongrel, combining identifying features of both current types of license (installation disk and key card).  Since your package came with a PIN, I am going to GUESS that you have a PKC license. For you the key identifying feature will be whether the license or packaging says anywhere how many devices you may install it on.
