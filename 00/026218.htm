maria fab say:

Hi Vital,

The error 0x800CCC90 occurs if your Internet Service Provider (ISP) limits the number of POP3 connections that you can make at the same time. For possible resolution, let's separate your POP3 accounts into smaller Send/Receive Groups, and then connect with each Send/Receive group separately. Here's how:

  1. In your Outlook Mail app, go to File > Options > Advanced > Send and receive.
  2. Click on the Send/Receive tab. In the Send/Receive Groups dialog box, click New.
  3. Specify the name for this send/receive group, and then click OK. The Send/Receive Settings dialog box appears for the New Send/Receive Group.
  4. All available accounts are listed under Accounts. Select the account, and then put a check mark in the Include the selected account in this group check box. After you select the check box, the options in the dialog box are available.
  5. Click Download complete item including attachments, and then click OK.
  6. Repeat steps 2 through 5 for each account that you want to include in this Send/Receive Groups.
  7. To create another Send/Receive Group and add an account to that group, repeat steps 2 through 6.
  8. When you have created all the Send/Receive Groups that you want, click Close.
  9. To send and receive e-mail for each group that you have created, go to the Send/Receive > Send/Receive Groups, click the drop-down arrow, and then click the name of the Send/Receive Group that you want.

Let us know how it goes.

tom ogilvy say:

cwong668,

 

put this formula in F2 and drag fill down until you start to see #N/A errors - then your list is complete.  You can now use the same countif formula to get your counts.  This formula just builds the list of unique values.

 

=INDEX($A$1:$C$3, MIN(IF(COUNTIF($F$1:$F1, $A$1:$C$3)=0, ROW($A$1:$C$3)-MIN(ROW($A$1:$C$3))+1)), MATCH(0, COUNTIF($F$1:$F1, INDEX($A$1:$C$3, MIN(IF(COUNTIF($F$1:$F1, $A$1:$C$3)=0, ROW($A$1:$C$3)-MIN(ROW($A$1:$C$3))+1)), , 1)), 0), 1)

 

It must be array entered using Ctrl+Shift+Enter rather than just enter

 (Cmd-Return for Mac)

 

I modified the formula at this site

the formula is explained there, so you might want to take a look.

It also works for numbers or a mixture with some cells having numbers and some cells having text.

Hi,

 

I an't do that with a formula. But here'ssome code that will do it. ALT+F11 to open VB editor, right click 'ThisWorkbook' and insert module and paste the code below in and run it.

 

You will be prompted to select the range of input data and the first cell for the output

 

 

 

Sub Count_Mixed()
Dim SrcRange As Range, c As Range
Dim DstRange As Range, r As Long
Set SrcRange = Application.InputBox("Select data range", Type:=8)
Set DstRange = Application.InputBox("Select start cell for output", Type:=8)
DstRange = "Unique list"
DstRange.Offset(, 1) = "Appears"
For Each c In SrcRange
    If Application.CountIf(DstRange.EntireColumn, c) = 0 Then
        Cells(Cells.Rows.Count, DstRange.Column).End(xlUp).Offset(1, 0) = c
       r = WorksheetFunction.CountIf(SrcRange, c.Value)
       Cells(Cells.Rows.Count, DstRange.Column).End(xlUp).Offset(0, 1) = r
    End If
Next c
End Sub
