hansv mvp say:

It's in the very first line:

 

Private Sub PageFooterSection_Format(Cancel As Integer, FormatCount As Integer)

 

If the footer happens to have another name in your report, e.g. just PageFooter, you should use that:

 

Private Sub PageFooter_Format(Cancel As Integer, FormatCount As Integer)

 

and in the unlikely case it's named JustinBieber, you'd use

 

Private Sub JustinBieber_Format(Cancel As Integer, FormatCount As Integer)

 

One other thing: your report should contain a text box whose control source refers to the total number of pages in the report, e.g. =[Pages]. This text box can be hidden, but it's essential that it exists.
