cmdrkeene say:

In Outlook you could program a VBA macro module to export the email contents into a spreadsheet or text file (perhaps comma separated), but that is very advanced programming.

If you have Microsoft Office 2013, you could use Microsoft Access to "pull" the data out of Outlook.  Here's what you'd want to do:

  1. Make sure all the emails you want to extract are in the same email folder (you might setup a rule for this to organize incoming messages).
  2. Open Access, then choose More > From Outlook Folder in the External Data tab
  3. It will pull in the contents of all the emails in that folder.

What's great about this is that you can link it to the Outlook folder, so that next time you need not do any of these steps.  You simply open Access again and choose "Refresh".

Once the data is in Access, you can export it to a text file for use in the other software program.
