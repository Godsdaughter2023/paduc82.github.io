It would seem that the default file association for the .PDF filetype has been changed from Adobe or removed altogether. This is fairly easy to reinstate.

  1. Open Explorer and locate a .PDF file. Right-click it and select Open with >, then Choose default program from the flyout.


    Figure 1.0 - Open With dialog

  2. Make sure that Always use the selected program to open this kind of file is checked in the lower left.
  3. If your Adobe Reader is in the Recommended Programs list, double-click it. The file will open in Adobe Reader and you are fnished.
  4. If your Adobe Reader is not in the Recommended Programs list, click Browse and locate your Adobe Reader application. The default location is something like:

    ... or more commonly:
    C:\Program Files\Adobe\Reader 10.0\Reader\AcroRd32.exe
  5. Once that is selected you will be returned to the Open With dialog with Adobe Reader selected within Other Programs. Click OK to accept it. The file will open in Adobe Reader.

Your .PDF file association should be back to normal. Repeat for DesignCAD 3D Max and a .DCD file.
