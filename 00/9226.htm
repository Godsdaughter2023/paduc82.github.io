aaususer say:

Alright, going to answer my own question.

The solution I found to get subtitling to even appear as an option on playback was to mux my MP4 and the SRT captions file. (I used an open source PC utility called My MP4Box GUI - but there's MP4Box for the command line warriors.)

Bit of a roundabout trip and would be lovely if Microsoft prioritised support for hearing impaired users a little better but there you go.
