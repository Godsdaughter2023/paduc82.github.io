adam vero say:

Essentially you cheat.

Create "helper" columns so that you end up with two series, one with the values you want in one colour, one that only has the other values for the second colour. The "gaps" must have a "#N/A" in them so that the chart simply omits these points. You can use a formula for this using the NA() function such as:

=IF(C2="Y",B2,NA())

then plot both helper series, and colour accordingly.

As a variation, you might simply plot the whole of the original series with lines but no points, then the second series only as points but no lines. Any line with points and no lines will always be drawn in front of another series with lines visible (if they are on the same axis), so the layering order is one less thing to worry about.

There are loads of articles about this sort of "conditional formatting" in charts at Jon Peltier's site:www.peltiertech.com
