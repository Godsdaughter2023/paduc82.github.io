ruwim.b say:

The main.db file from the \skylib\<Skype Name> folder is a standard SQLite database with a structure practically identical to the one used by the classic Skype (the 7.X version). However, you will not find any chat messages in this database file.

 

This main.db file can be opened and viewed by any SQLite browser. The one I prefer is Db Browser for SQLite.

 

http://sqlitebrowser.org/

 

If you open the main.db file in this browser, you will see that the Messages table, where chat data were saved by the classic Skype, is empty.

 

 

 

 

Actually, most of the Tables previously used by the classic Skype, are now empty. The only useful information I could find in this file is in the CallMembers table, where info on the call partners Skype version is saved in the endpoint_details column.
