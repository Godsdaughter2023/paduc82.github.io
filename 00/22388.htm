vincentchoy say:

AD-FS vs DirSync are just different forms of authentication.

AADSync (DirSync is name of older version) copies AD credentials to O365, O365 does the authentication.

ADFS - O365 will pass the authentication back to your AD. O365 does not do authentication. This is more complicated and costly as it requires 2 server roles running in HA mode to connect O365 to your AD (if O365 cannot connect with your AD it will not allow anyone to login).

In both cases, this has nothing to do with  user self-service password reset. In both cases the passwords are stored in your corporate AD and your user has to go to your AD to reset the password. Password change via ADFS is possible however.

Azure AD Pro provides enhanced features that allow user self service reset but that comes with additional cost.

Hello, Agha Jamil MirzaBaig,


Are you referring to Bing Toolbar or Bing Search?


In Internet Explorer you can manage, remove or disable addons, toolbars, extensions, etc via Manage Addons


To remove Bing Search, using Internet Explorer

Go to Tools > Internet Options > Programs Tab > Manage Add-ons

Select Search Provider in left index

Select the Search Provider you prefer from the list on the right

Click Set As Default

Click Close


If you do not have a list of search providers, click the dropdown arrow in address bar

Select Add to add different search providers from the Gallery


To remove toolbar

Open Control Panel > Uninstall Programs

Look for Bing Toolbar

Select it and click uninstall

Reboot computer when finished
