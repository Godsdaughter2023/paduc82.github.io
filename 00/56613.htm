Hello,

had similar problem with USB not working on an HP ProBook 450 G3, even downloading and installing chipset and USB3.0 drivers from HP website didn't solve the problem. 

I was still having a not recognized device with hardware ID: IUSB3\ROOT_HUB30&VID_8086&PID_9D2F (and two more simular ID entries).

But drivers from HP get unzipped all inside the "C:\swsetup" windows folder. So in order to successfully update the driver for the undetected device I did the following:

  • right clicked on the device into the device manager
  • selected to update the driver
  • in the next window i chose to manually detect the driver on the computer
  • So I selected the path "C:\swsetup" including subfolder

Windows 7 after searching for a while finally managed to find the required driver and as a result a Composite USB devices and an Intel 3.0 Root Hub have been added under the USB Controller branch of the device manager tree.

USB then started working properly.
