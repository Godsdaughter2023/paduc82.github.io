1. Page Layout tab - Breaks - Insert either a page break or a next page section break.
2. Select the text in question - Home Tab - Click the arrow bottom right of the Font group and select the Outline option.

rohn007 say:
#1) alternate method, use <CTL><Enter> shortcut to insert page break

#1 Bonus points) use <CTL><SHF><Enter> to insert column break. In a one column section (default) it will act as a page break

#2) if you are talking about the font effect "outline", then Graham has described it
      if you are talking about a story outline, you mark the outline points using heading styles, then  click on the "Outline" view button on the bottom right corner of the application window OR View tab > Document Views group > Outline button.

#3 select the table by clicking on the little "+" icon that appears when you hover the mouse near the upper left corner of the table.
