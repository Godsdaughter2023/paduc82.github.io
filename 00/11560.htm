nathan li msft say:

Hi kfafNQ,

 

In OneDrive for Business, the Edit permission include the permission to delete items. So you may not be able to do this in OneDrive for Business. Although OneDrive provides different ways of retrieving your files after being deleted:

Restore deleted files or folders in OneDrive

Restore your OneDrive

 

However, you may create such a permission level in a SharePoint site. My suggestion is that you may store your files in a SharePoint site, then create a unique permission which include permission of editing but not deleting of items by the following process:

  1. On the SharePoint site, click the gear icon > Site Permissions > Advanced permission settings;
  2. In the PERMISSIONS ribbon, click Permission Levels > Add a Permission Level;
  3. Add all other permissions contained in Edit, but leave Delete Items and Delete Versions unchecked.

You may also refer to the following article for the detailed steps:

How to create and edit Permission Levels

 

After that, you may grant the users this unique permission by the steps in this article:

Customize permissions for a SharePoint list or library

 

Please let us know if you have any further concerns.

nayana s m say:

Hello Josh,

Thank you for choosing Microsoft Community.

Unfortunately, there is no feature within Outlook to set the folders as expanded in navigation pane.

That said, Outlook will remember which folders were expanded when closing. To have all the folders as expanded when opening, you need to expand all the folders/sub folders before exiting Outlook.

If that is not working in Outlook, try to reset the views in Outlook to the default ones. To do that, try the following step:

Open Outlook > View tab > Click on ‘Reset View’ under current view.

Note: Resetting the views would change the views to factory settings.
