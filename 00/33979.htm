Hi

You can activate an OEM installation an unlimited number of times, as long as it's installed on the same computer where it was first installed by the computer vendor. Since the installation was already activated once, you may need to use the telephone option and call the activation center to get it activated.


1. Press the Windows Logo Key+R. This will bring up the Run dialog box.

2. In the Open: box, type  slui 4  and press OK. This will start the manual activation wizard.

3. Select your Country in the drop down menu and click Next.

4. The wizard will display a Toll Free phone number. Leave the wizard open on the computer and call that phone number.

You will be given some recorded options. Select the option to talk to a live representative.

The representative will manually activate the system for you.

One other important question would be, how did you reinstall Windows 7? If you used a Recovery Disk that came with the computer, you may find that the same 'rental agent' component might be reinstalled from that recovery disk.

If this happens and you own the computer, I would call the rental store where the purchase was made and demand that they remove that component.
