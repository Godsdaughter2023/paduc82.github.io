Gaming, multiple monitors and power users? Feature request. - Microsoft Community

mariogoebbels say:

Gaming, multiple monitors and power users? Feature request.

Power users these days are likely to have two or even more displays. When gaming, it may happen more often than not that the game's not spanning all of them, intentionally or not.

There is an issue with that. The secondary display(s) will be powered up, but functionally useless, if the game started on the primary display and captured the mouse cursor. Yet, Windows doesn't seem to have any trouble to keep rendering active content on the secondary screen (animated GIFs for instance keep playing in a browser instance left open). So:

Let us make use of the secondary displays, while leaving the game oblivious about it. This would entail two things, for one, prevent the games from capturing the mouse cursor exclusively, and second, leave the game up and running instead of sending it to the background whenever another window gets focus. While this is partially possible today, it requires the game to support running in a window without frame, instead of fullscreen mode. Not much of them do that, and it'll be performance impaired by the DWM, because the rendering has to be funneled through it.

A feature like this would be handy for any sort of games where you have to frequently consult the internet for help. Be it MMOs, complex games like upcoming Elite, or whatever else. One display runs the game in true fullscreen mode, the other would run the part of the desktop that would have been visible anyway, and be usable without the constant mode switching (which might upset some game engines, too).

Thanks for considering.
