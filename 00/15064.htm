Refer below image:

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell E2 & copy down:

=IFERROR(INDEX(A$2:A$100,SMALL(IF($C$2:$C$100=0,ROW(A$2:A$100)-ROW(A$2)+1),ROWS(A$2:A2)))&"","")

Enter below formula in cell F2 & copy down:

=IF($E2="","",VLOOKUP($E2,$A$2:$B$100,2,FALSE))

Presuming Number in column B is distinct for each name. 

Please Note: If number in column B is NOT distinct for each name then enter below formula as an array (CTRL+SHIFT+ENTER) formula in cell F2 & copy down:

=IFERROR(VALUE(INDEX(B$2:B$100&"",SMALL(IF($C$2:$C$100=0,ROW(B$2:B$100)-ROW(B$2)+1),ROWS(B$2:B2)))),"")
