I have been building PCs for over 12 years. Not a long time but a lot of PCs.

 I don't remember any windows OS that on the first boot to system that I didn't get a window asking if I want to check windows update. At that point I always select 'Turn Automatic Updates' "OFF".

 In fact no matter what program it is I DO NOT allow automatic updates.

 It takes a little more effort on your part but will keep you from having unexpected problems that seem to come out of the blue.

 Through 'Windows Update' I run a check. Then when it has the list ready I select "Important Updates"

 Uncheck all then read through the list and select critical and security updates only. Let them download + restart. Run update check again and staying in "Important updates" select malware tool and defender updates. At the next run of update you will/should only have updates that state "For a complete listing of the issues....." I never install these without checking "More Information" first to see if the system and how it would be used would make it needed to download. Other then that they stay on the list.

 I very rarely install "Optional" updates.

  And NEVER install any drivers offered by win-upds. I don't know how many times in the far past that windows update installed new drivers for something and the hardware quit working immediately after the install reboot. Need new drivers go to manufacturers website for them.

 Auto updates in other programs? I have an Antivirus program that within it's updater has an automatic system updater. So it, Virus program,  installed all the windows updates on it's own bypassing my selection for manual windows updates resulting in crashing the system. 

 So if you are installing any program that has an auto-update feature? Be sure you have the option to turn it off. If not then get rid of it.

 It is not hard to run a system this way. Usually when I first log onto the network I run an antivirus definition update. Windows only gets checked once or twice a month.

 Also if your computer is always on then you need to make sure you can disable/turn off the internet connection at your will.

 The decision to turn on auto updates would depend on who is going to use the machine.

There are tree main ways to activate Windows.
