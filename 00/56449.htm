chriso747 say:

This is a fairly common error in FSX (there are many others that come up of course!).   The UIAutomationCore.dll that FSX tries to use when needed is found in System 32 and SysWOW64 in your Windows directory.  However, FSX is not able to use that newer version of the .dll and so it crashes.   Usually this error comes up when the FSX Menu bar items are used often. 

To get around this error, one needs to download an earlier version of the .dll.   There are two versions of the .dll that can be used:  one is 6.0.5840.16386 and the other, 6.0.6001.18000

You can go to your preferred site to download this, or else, you can try here:

For the 16386:   http://www.search-dll.com/dll-files/download/uiautomationcore.dll.html

For the 18000:   http://dllworld.org/uiautomationcore-dll/

Since FSX is a 32 bit application, select the .dll for that one.   After you download it, copy/paste it in the FSX Directory. FSX will search its own backyard for the .dll before it searches elsewhere in Windows.   

Hope this fix will help you.
