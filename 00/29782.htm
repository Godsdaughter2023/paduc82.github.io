is an audio "new e-mail" signal available for desk tops ? - Microsoft Community

fmwoodman say:

is an audio "new e-mail" signal available for desk tops ?

for Windows 7

lemp say:
Yes.

Start > Control Panel > Sounds > Sounds tab

If you have "Category" view of Control Panel, then Hardware and Sound > Sound > Change System Sounds.

Scroll down in "Program Events" to "New Mail Notification."

The default is Windows Notify.wav ... but there are different files with that same name depending on which "Sound Scheme" you have selected (at the upper part of the same screen).

You can listen to the sound by clicking the "Play" button or click "Browse" to look for a different *.wav file.

The built-in Windows 7 system sounds are located at C:\Windows\Media\<soundschemename>

where "soundschemename" is one of the following:
Afternoon
Calligraphy
Characters
Cityscape
Delta
Festival
Garden
Heritage
Landscape
Quirky
Raga
Savanna
Sonata

You also can use pretty much any *.wav file you like.  There are many web sites that offer such files for free.  For example, http://www.thesoundarchive.com/email.asp

Of course, whether your particular (unidentified) email program will USE that sound event is a different story.

Hotmail is a web-based email service.  You won't get notified about email until you go to the Hotmail web site (which is now Outlook.com or mail.live.com) and sign in.

However, if you use a locally-installed email client (e.g., Mozilla Thunderbird or one of THESE) to access your Hotmail account, that client will usually provide a new mail notification.
