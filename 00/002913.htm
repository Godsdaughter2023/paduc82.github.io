petrib say: From your wireless (Wi-Fi/WLAN) router you need to get information regarding what kind of encryption and access keys, etc., it requires. Usually you do it by logging on to the router using a browser (a web user interface).

Once you have that, then on the phone go to: menu > Tools > Settings > Connection > Access points

There hit Options, and choose New access point > Use default settings

Give it a name (Connection name). Any name you wish to identify it with.

For Data bearer choose "Wireless LAN".

For WLAN netw. name enter whatever is the "SSID" value you got from your wireless router.

Same for Network status: "Public" or "Hidden" depending...

WLAN netw. mode must be "Infrastructure"

WLAN security mode to corresponding router setting, again. As it is not open, it is probably WEP or WPA/WPA2 for your router.

Then choose WLAN security sett.

From here things are different from whether your router is using WEP or WPA security. In any case you get the correct values from your router.

Once the access point is defined, then you can start using it in apps like the browser ("Web"), Messaging (for email), etc.

Sub GetAllPageNumbers()
  Dim objWordApplication As Word.Application
  Dim nPageNumber As Long
  Dim nWordNumber As Long
  Dim objFile As Variant
  Dim objFileSystem As Object
  Dim objFolders As Object

  nPageNumber = 0
  Set objWordApplication = New Word.Application
  Set objFileSystem = CreateObject("scripting.filesystemobject")
  Set objFolders = objFileSystem.GetFolder("C:\Users\11467\Desktop\New folder")

  For Each objFile In objFolders.Files
    '  Only process non-hidden files
    If (objFile.Attributes And 2) = 0 Then
      With objWordApplication
        .Documents.Open (objFile)
        nPageNumber = nPageNumber + .ActiveDocument.ComputeStatistics(wdStatisticPages)

        .ActiveDocument.Close (False)
      End With
    End If
  Next objFile

  MsgBox ("pages:" & nPageNumber & " " & "words:" & nWordNumber)

  Set objFile = Nothing
  Set objFileSystem = Nothing
  Set objFolders = Nothing
  Set objWordApplication = Nothing
