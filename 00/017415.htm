hansv mvp say:

Access has a feature called Name AutoCorrect. If this is turned on (File > Options > Current Database > 'Track name AutoCorrect info' and 'Perform name AutoCorrect' both ticked), changing the name of a database item will automatically update the references to this item too. If Name AutoCorrect is turned off, you're responsible for updating those references yourself.

Unfortunately, Name AutoCorrect is slightly wonky - it has been known to cause problems - so it's best to keep it turned off, and perhaps only turn it on temporarily when you want to change the name of an item and would like Access to update references to it.

You can round corners using the Line dropdown from the Shape Styles. Choose Line Options. Set the rounding Size. This will set the shapesheet's Rounding cell in the Line Format section. The changes will not be reflected in the Geometry section.

jay freedman say:

When you insert a TOC from the References ribbon, each TOC style is given a tab stop (left aligned, no leader) where the text after the numbering should begin.

The Reveal Formatting pane (Shift+F1) should display this setting.

If that tab stop has been removed, the result will be what you showed -- the tab character between the number and the first word will push the text to the right-aligned, dot-leader tab stop at the right margin.

You should be able to fix this by right-clicking anywhere inside the TOC and clicking Update Field in the menu, then choosing "Update entire table" in the prompt.
