For reference, this is how I have the code structured right now:

Public Function TestApplicationVLOOKUP(ByVal Name As String) As Integer

bernie deitrick say:

Dimension your return as a Variant so any error gets passed correctly, and fully qualify your range's parent sheet:

Public Function TALOOKUP(ByVal Name As String) As Variant
    TALOOKUP = Application.VLookup(Name, Worksheets("Sheet1").Range("A:E"), 2, False)
End Function

If you want 'other stuff' to happen in your same function, then you could use

Public Function TALOOKUP(ByVal Name As String) As Variant
    Dim vRet As Variant
    
    vRet = Application.VLookup(Name, Worksheets("Sheet1").Range("A:E"), 2, False)

    If IsError(vRet) Then
        'Code for other stuff
    Else
        TALOOKUP = vRet
    End If
    
End Function
