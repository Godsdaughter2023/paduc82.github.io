Hello SabineSchoppe

First and foremost, it is highly unlikely that you would need to export anything to configure the same account on the 2nd computer. What would be required depends almost solely on the type (POP3 or IMAP) of email account configured on
Computer 1.

<IF> the email account is configured as a POP3 account, all you need to do is copy the PST file associated with the account (with Outlook closed) from Computer1 to Computer2. On C#2 - make sure that you use the PST file just copied for the account instead of letting Outlook create a new one

<IF> configured as an IMAP account on C#1 there is nothing to copy in terms of email folders. Just configure the account as an IMAP account on C#2 and Outlook will sync the email folders with the server.

The caveat for an IMAP account is that if you have any folders with the words "This Computer Only" in the folder description (i.e. Contacts/Calendar) that contain data you want to keep, be sure to export those folders to a PST file, copy that PST file to C#2, add it to your Outlook profile and finally set the PST file as the "default data file" which is where the default contact/calendar folders will be stored. If this scenario applies and you require more info, please just ask and will respond accordingly with more detals.

Regardless of whether POP3 or IMAP - export/import is not a required (nor recommended) to accomplish what you need to get done

cindy meister say:

Are you making sure you're in the Header/Footer area for a section before inserting the water mark?

If you insert the water mark in the main body of the document (instead of the header footer) then what you describe could happen if the paragraph to which the graphic is anchored is deleted, or moves to a different page. In the first case, deleting the paragraph will delete the anchor, which will delete the watermark.

In the second case, you'd probably find two (or more?) watermarks are superimposed. I suspect this is what is happening in your case.

Cindy Meister, VSTO/Word MVP

graham mayor say:

For the watermark tool to work, you need to ensure that your section breaks are in place and that the second section header is NOT linked to the first section header (header tools) and that the appendix header is NOT linked to the body header, and that all the headers in the contiguous sections in which you want the headers ARE linked. Then put the cursor in the body where you want the watermark and run the Watermark tool.

If you have watermarked sections that are not contiguous or want some other arrangement, insert the watermark. Open the header, select and copy the watermark to the clipboard. Remove the watermark using the Remove watermark tool. Then open and paste the clipboard content into the header of each unlinked section where you want the watermark to appear.

Essentially a watermark is merely a graphic set behind the text in the header view of the particular section.
