¡firedog say:

Why does Live Mail add an underscore to the extensions on email attachments?

It does it to avoid having two dots ahead of the extension, which was widely not allowed when this program was under construction. It's common for a Word document, say, whose title ends with a full stop to be saved with the title as the filename, so that there are two dots before the extension. Trouble is, Windows Live Mail converts the second rather than the first.

You can remove the extraneous character when you Save as. You could also set up a file association for the new extension, so *._pdf opens with Adobe Reader, for example.

This should be fixed and distributed during Windows Updates ... shouldn't it?

Fixes to Windows Live Mail have never been distributed via Windows Update. There haven't been any fixes at all since 2012, as far as I can remember. I'll be pleasantly surprised if we see any in the future.
