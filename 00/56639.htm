a. user say:

I have user accts admin., to-pc and tj. How can I block access for tj to view certain files and or folders?

Here is another way to restrict an user's access to certain files and folders :
Each user account has its own folders in which the user keeps his/her personal files. These files cannot be accessed by another user, because each account is protected by login password.
The only files/folders that can be accessed by all users are those in thePublic Folders.

Based on that principle, if there are files or folders you do not want tj to access, move those files away from the Public folders to you user account.

How to MOVE folders :
Open the Public Folder > right click on the items inside the Public folder > click Move To Folder > on the next window, use the scroll bar, navigate to the personal folder you want, click it > then click MOVE
If there are a few files/folders to move, just repeat the steps.
