I want to pull the numbers out of the string into seperate columns.  So for example cell A1 would have IU-96.750A-48.00-0.162-0.938Q-L2-X.  I want to have cell B1 show the value 96.75, C1 show 48.00, D1 show .162, and E1 show .938.  The number of characters between the dashes can change but the pattern of dashes doesn't.  By the way the last number will always have a letter behind the number that needs to be dropped.  These part numbers are listed out in a column and I was hoping to just add 4 more columns and pull the numbers out into them.

jeeped say:
You might be able to do this with a convoluted combination of nested functions but a User Defined Function (aka UDF) would serve you better.

     

Tap Alt+F11 and when the VBE opens, immediately use the pu;ll-down menus to Insert, Module. Paste the following into the new pane titled something like Book1 - Module1 (Code),

Public Function udf_Part_Number_Pieces(rPN As Range, p As Long, d As String) As Variant
    Dim vPCS As Variant
    vPCS = Split(rPN, d)
    If IsNumeric(Left(vPCS(p - 1), 1)) Then
        Do While Not IsNumeric(vPCS(p - 1))
            vPCS(p - 1) = Left(vPCS(p - 1), Len(vPCS(p - 1)) - 1)
        Loop
        udf_Part_Number_Pieces = CDbl(vPCS(p - 1))
    Else
        udf_Part_Number_Pieces = vPCS(p - 1)
    End If
End Function

Tap Alt+Q to return to your worksheet. Use just as you would any native Excel function.

Usage:
=udf_Part_Number_Pieces(<address of cell with PN>, <position to retrieve>, <character to split on>)

In the case of your sample data in the image above this would translate to a formula in B1 as,

=udf_Part_Number_Pieces($A1,2,"-")
... or to make it easier to copy right,
=udf_Part_Number_Pieces($A1,COLUMN(B:B),"-")

I've assumed a hyphen separates the pieces of a part number. COLUMN(B:B) simply translates to 2 but will increase as it is copied to the right so that the 3rd, 4th and 5th piece gets returned. A1 is column-anchored as $A1 but will become $A2, $A3, etc as it is copied down.

Function Fx(Rng As Range)
    Dim v, j
    v = Rng.Value
    v = Split(v, "-")
    For j = 1 To UBound(v) - 1
        v(j) = Val(v(j))
    Next j
    Fx = Array(--v(1), --v(2), --v(3), --v(4))
End Function

Probably not needed in this particular case, but if we want to avoid the limitation to four numbers, the following modified function may be of help:

Function Fx(Rng As Range)
    Dim v, j
    v = Split(Rng.Value, "-")
    For j = 1 To UBound(v)
      v(j - 1) = Val(v(j))
    Next j
    Fx = Array(v)
End Function

The 'call' of this function needs to be changed to =--Fx(A1) (still entered as an array formula) if you want the result to be numbers rather than text.

There is still an assumption of the first "number" being after the first '-'.

3. Import the data to the new Office 365 tenant. The admin can use the network upload to import the PST file to Office 365 if the data size isn't large. The end user also can import their pst file to new Outlook client by themselves. 

4.Make sure that everything works in that new Office 365 tenant, and then cancel the original Office 365 Subscription.
