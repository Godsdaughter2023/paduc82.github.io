It uses a more robust connector, longer shielded cables, and stricter (but backward-compatible) electrical standards. eSATA speeds are up to six times faster than other external hard drive cable types.

ESATA, external Serial ATA, is an external device connectivity specification for hard drives and optical drives. The eSATA specification is a direct extension to the SATA specification used for internal hard drives and optical drives. ESATA provides for transfers speeds of up to 6.0 Gbit/s and does not suffer from performance loss due to converting from a hard drive interface to a USB or FireWire interface. Using eSATA to connect a drive to your computer is a simple task that any computer user can accomplish.

If the Spell Check is checking a portion of the sheet but not checking other portions, the most likely reason is that the ignored portion is formatted as a Language for which there are no Proofing Tools. That can easily happen by accident or when content is copied from another source, such as the web. See if it helps to:

  1. Select the entire sheet [Command+A]
  2. Go to Tools> Language
  3. Select the correct Language
  4. Run the Spell Check again after to OK out of the Language dialog.

stefan blom say:

First of all, have you actually installed all the languages that you use? Just having Word apply language formatting definitely won't work.

As you have noticed, "Detect language automatically" isn't very useful. If you are creating different parts of the document in different languages, apply different paragraph styles (with different language settings) to each part. On the other hand, if one and the same paragraph may contain text in different languages, make use of character styles instead.

stefan blom say:

I've only seen face recognition being used in movies, so I can't comment on that one. :-)

Anyway, automatic translation is far from perfect. Generally, anything to do with languages usually requires human "interference."

Using styles is essential if you want to be efficient when working in Word, so adding language formatting to those styles won't require much extra work.
