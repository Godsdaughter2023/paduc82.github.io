rich007a say:

The FlexFind addin can do this: http://www.jkp-ads.com/OfficeMarketPlaceFF-EN.asp

Otherwise you're looking at a VBA solution... try this:

 

Sub ChangeAllCharts()
    Dim sht As Worksheet
    Dim cht As ChartObject
    Dim ser As Series
   
    For Each sht In ActiveWorkbook.Worksheets
        For Each cht In sht.ChartObjects
            For Each ser In cht.Chart.SeriesCollection
                ser.Formula = Replace(ser.Formula, "$D$5:$D$26", "$D$5:$D$27")
            Next ser
        Next cht
    Next sht
End Sub

 

To run the macro, copy the above code from here, then in excel press Alt+F11, Insert > Module, then paste the code.

In Excel do View tab > Macros, select the ChangeAllCharts macro and click Run.

Sub ReplaceInSeriesFormula()
    Dim ws As Worksheet
    Dim cht As Chart            'Chartsheets in the workbook
    Dim chtObj As ChartObject   'Charts on worksheets
    Dim ser As Series
    Dim strOld As String, strNew As String
    Dim strEndSerForm As String
   
     '-- UPDATE AS REQUIRED --
    strOld = "$27,"      '<-- YOU ONLY NEED TO CHANGE HERE
    strNew = "$23,"      '<-- YOU ONLY NEED TO CHANGE HERE
     '------------------------
   
     'Replace in all charts in chartsheets:
    For Each cht In ActiveWorkbook.Charts
        Debug.Print "Chart name: " & chtObj.Name
        For Each ser In cht.SeriesCollection
            Debug.Print "Series name: " & ser.Name
            Debug.Print "Start series formula: " & ser.Formula
            strEndSerForm = Replace(ser.Formula, strOld, strNew)
            Debug.Print "End series formula: " & strEndSerForm
            ser.Formula = strEndSerForm
            Debug.Print "Final series formula: " & ser.Formula & " DID IT CHANGE OK?"
        Next ser
    Next cht
   
     'Replace in all charts in worksheets:
    For Each ws In ActiveWorkbook.Worksheets
        For Each chtObj In ws.ChartObjects
            Debug.Print "Chart name: '" & chtObj.Name & "' in sheet: '" & ws.Name & "'"
            For Each ser In chtObj.Chart.SeriesCollection
                Debug.Print "Series name: " & ser.Name
                Debug.Print "Start series formula: " & ser.Formula
                strEndSerForm = Replace(ser.Formula, strOld, strNew)
                Debug.Print "End series formula:   " & strEndSerForm
                ser.Formula = strEndSerForm
                Debug.Print "Final series formula: " & ser.Formula & " DID IT CHANGE OK?" & vbCr
            Next ser
        Next chtObj
    Next ws
End Sub

tony chen chn say:

Hi,

Following these steps you can save a Custom Quick Style
1. Change the styles in a Quick Style set according to your requirements.
2. On the Format Text tab, in the Styles group, click Change Styles.
3. From the Change Styles drop-down list, select Style Set, and then select Save as Quick Style Set.
4. In the Save Quick Style Set dialog box, enter a file name for the new Quick Style set, and then click Save.
You can reuse the newly created Quick Style set by selecting it from the Style Set gallery.

You can also set the current style set and theme as default.
On the Format Text tab, in the Styles group, click Change Styles, then click Set as Default.
