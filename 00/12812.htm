In PowerPoint 2013, going to File > Info > Check for Issues > Inspect Document, can tell me that Embedded Documents *exist*, but it does not tell me how many or what they are.  Is there a way to get a list of all the embedded documents in the deck? 
steve rindsberg say:

Other than VBA, I don't know of any way to do this.  If you want to have a play with some VBA, try this.  You'll want to press Ctrl+G in the VBA editor to make sure that the Immediate Window is displayed ... that's where you'll see the results:

Sub ListEmbeds()

    Dim oSl As Slide
    Dim oSh As Shape
    
    For Each oSl In ActivePresentation.Slides
        For Each oSh In oSl.Shapes
            If oSh.Type = msoEmbeddedOLEObject Then
                Debug.Print "Slide:" & vbTab & CStr(oSl.SlideIndex) & vbTab & oSh.Name
            End If
            If oSh.Type = msoPlaceholder Then
                If oSh.PlaceholderFormat.ContainedType = msoEmbeddedOLEObject Then
                    Debug.Print "Slide:" & vbTab & CStr(oSl.SlideIndex) & vbTab & oSh.Name
                End If
            End If
        Next
    Next    ' Slide

End Sub
