jay freedman say:
The simplest way to make sure they're all the same size is to draw the first one, copy it to the clipboard, and make all the others by pasting from the clipboard. Then you can rotate or otherwise format each one individually, but be careful not to drag their bounding boxes.

The other way, if you must deal with shapes that have already been drawn separately, is to select one and note the measurements in the Size group of the Drawing Tools tab of the ribbon. Then select another one and set the Size measurements to the same values; and repeat for each of the others.
