You may use the following personal function (from www.excelabo.net). Note that a change of color does not provoke a recalculation of the function :

 

Function CountColor(c As Range, Colour As Integer)

 Application.Volatile

 For Each cell In c

    If cell.Interior.ColorIndex = Colour Then

       CountColor = CountColor + 1

    End If

 Next

 End Function

The arguments are 1. the range and 2. the colorindex of the background color, eg. =CountColor(F1:F9,3)

If I understand correctly, I expect you would need a unique field in the first table (tblLinda). I added an autonumber field [ID].

Then create a cartesian/union query (qcarLinda) like:

FROM tblLinda, TableX
WHERE [PeriodNum]=1

SELECT qcarLinda.ID

Rounding using formatting

" If you're an accomplished Excel user, you might be tempted to skip this topic. After all, Excel rounds digits for you when you specify the number of decimals to be displayed in a cell. And you are right, it does--but the results may not be what you expect!

Take the example below. A simple equation calculates Gallons * Markup and displays the results in the Total column. A grand total of 10.206 appears at the bottom of the Total column.

Gallons    Markup    Total

1.701      3         5.103

1.701      3         5.103

                    10.206

If you want to display the results in the Total column showing only two decimals, highlight the cells and click the Decrease Decimal button on the Formatting toolbar.  Using the exact same formula, the example below shows the difference between the unformatted Total and the formatted Total.

Gallons    Markup    Total    Formatted

1.701      3         5.103     5.10

1.701      3         5.103     5.10

                    10.206    10.21

Formatting the cells to display only two decimals hides the digits you don't want displayed. But the complete numbers are still stored in the cells and are reflected in the grand total (10.21). At first glance (especially to someone who is not the originator of the worksheet) it appears as though the numbers don't add up. The calculations in the Formatted column all end in zero, yet the grand total ends in a one. Someone reading this worksheet would think there is an error, when in fact Excel is merely rounding the numbers for display purposes.

Some people accept this discrepancy, while others (especially Accountants) want the displayed numbers to add up. You can resolve this discrepancy by using Excel's rounding functions.


FIXED

An alternate to the "permanent rounding" is the "fixed" function in Excel.

 Stephen can create a new column next to the column that has all  his numbers (let's say it's column D), create a formula in D1:

     =FIXED(C1,2)

and copy/paste or fill down. This way, the data in D will be  dynamic, and he doesn't have to repeat the process when his data  changes.

INT

Most people think the INT function gives you the integer portion of a number. Well, yes and no. With positive numbers the INT function gives you the integer portion of a number. So that:

=INT(5.16) results in 5

=INT(5.66) results in 5

However, with negative numbers, the INT function does not give you the integer portion of the number:

=INT(-5.16) results in -6

=INT(-5.66) results in -6

According to Excel Help, "INT rounds down to nearest integer". The problem here is with the word "down".  For negative numbers, the INT function actually gives you a larger negative number.

TRUNC

If you really only want the integer part of a number, use the TRUNC function. Regardless of the number TRUNC simply strips the fractional part of the number. So that:

=TRUNC(5.16)  results in  5

=TRUNC(-5.16) results in -5

=TRUNC(5.66)  results in  5

=TRUNC(-5.66) results in -5

The TRUNC function has a optional argument, num_digits, where you can specify the decimal places you want displayed when the number is truncated. If omitted, TRUNC displays only the digits to the left of the decimal. Unlike the Round functions, you cannot specify a negative num_digits argument for TRUNC.

These are 7 things you should already know about a hacked account.

1-Recover your account (if you can)
Important: If you cannot recover access to your account, then it is now someone else's account.
It is now the hacker's account.
Unless you've backed up, everything in it is gone forever and you can skip the next two items.

One you regain access to your account, or if you never lost it, you should immediately change your password.
