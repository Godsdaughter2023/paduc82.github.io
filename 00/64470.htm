great white north say:

Hi JoannaFreedendal:

Unlike Microsoft Outlook and many other e-mail programs that store messages, contacts, attachments, etc. into a single file (e.g., a .PST Personal Information Store file), Vista's built-in Windows Mail program stores each message as a separate .EML file in the hidden system folder C:\Users\<yourusername>\AppData\Local\Microsoft\Windows Mail\Local Folders. See the BleepingComputer tutorial How to See Hidden Files in Windows Vista for instructions on how to temporarily show hidden files and folders on your system if you'd like to view the contents of this AppData folder. I've never used Recoverit for recovering deleted e-mails but I assume you would be looking for files with a .EML extension that were deleted from your C: drive, or perhaps an entire folder that was located in C:\Users\<yourusername>\AppData\Local\Microsoft\Windows Mail\Local Folders.

Before trying to recover your e-mails with a data recovery tool like Recoverit, are you absolutely certain the messages were "permanently" deleted after you emptied the Deleted Items folder in Windows Mail?  It's very easy to accidentally drag an entire e-mail folder and drop it inside another folder, so be sure to expanded all your e-mail folders in Windows Mail in case your missing folder is now hiding as a subfolder in the wrong location.

Also note that some online webmail applications allow you to recover deleted e-mails from their backend mail server.  For example, I have a free Windows Live (Outlook.com) e-mail account (<myusername>@live.ca) and if I log in to the Outlook.com mail application at https://outlook.live.com/owa/ I have up to 30 days to recover any e-mails deleted from my Windows Live account - see the support article Restore Deleted Email Messages in Outlook.com.  If you have an e-mail account with another e-mail service (e.g., Google Mail, your Internet service provider, etc.) you should check first to see if they offer similar recovery of deleted e-mails.



Just an FYI that there is a free version of Wondershare's Recoverit at https://recoverit.wondershare.com/data-recovery-free.html that allows users to recover up to 100 MB of lost data at no charge. Wondershare has a 7-day money back guarantee for their recovery software (see their return policy <here>) that you might qualify for.  In any case, you can contact their  Customer Support at http://support.wondershare.com/ to open a support ticket if you need help recovering your e-mails with Recoverit.

andre for directly say:
You can create a stop and start script for Windows Update.

Create a Notepad file with each of the following and save them with the .bat extension.

stop-updates.bat:

sc config wuauserv start= disabled
net stop wuauserv

sc config bits start= disabled
net stop bits

sc config dosvc start= disabled
net stop dosvc

pause

start-updates.bat:

sc config wuauserv start= auto
net start wuauserv

sc config bits start= auto
net start bits

sc config dosvc start= auto
net start dosvc

pause

Now you can simply right click on stop-updates.bat and select “Run as Administrator”
