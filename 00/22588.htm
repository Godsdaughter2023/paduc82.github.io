haytham amairah say:

No matter if paragraphs being entered into single cells.

The Check Spelling takes into account this case and looks into each word in a cell.

This is your code but with major adjustment:

Sub SpellCheck()
    Application.ScreenUpdating = False
    On Error Resume Next
    
    Dim password As Variant
    password = Application.InputBox("Enter the password: ")
    
    ActiveSheet.Unprotect password
        
    Cells.CheckSpelling SpellLang:=1033
        
    ActiveSheet.Protect password
    
    password = Null
    Application.ScreenUpdating = True
    On Error GoTo 0
End Sub

tisky zheng msft say:

Hi Aaron,

 

Please try the following steps to create a Drop-Down Form Field.

 

1. Click the Developer tab, in the Controls group, click the Legacy Tools and choose Drop-Down Form Field.

2. Repeat step 1 to add a second dropdown field.

3. Right click the Region dropdown field and choose Properties.

4. In the Drop-Down Form Field Option. Add North, South, East and West in my example.

5. In the Bookmark control, enter ddRegion.

6. Right click the state dropdown field and name it ddState.

7. Launch the Visual Basic for Applications by pressing Alt+F11.

8. from the Insert menu, choose Module.

Copy the following code as an example:

 

Sub PopulateddState()

 

 Select Case ActiveDocument.FormFields("ddRegion").Result

 

 Case "North"

 

 With ActiveDocument.FormFields("ddState").DropDown.ListEntries

 

 .Clear

 

 .Add "Michigan"

 

 .Add "Ohio"

 

 End With

 

 Case "South"

 

 With ActiveDocument.FormFields("ddState").DropDown.ListEntries

 

 .Clear

 

 .Add "Georgia"

 

 .Add "Texas"

 

 End With

 

 Case "East"

 

 With ActiveDocument.FormFields("ddState").DropDown.ListEntries

 

 .Clear

 

 .Add "New York"

 

 .Add "Maine"

 

 End With

 

 Case "West"

 

 With ActiveDocument.FormFields("ddState").DropDown.ListEntries

 

 .Clear

 

 .Add "California"

 

 .Add "Oregon"

 

 End With

 

 End Select

 

 End Sub

 

Click Save then click Yes.

 

9. Right click ddRegion and choose Properties.

10. From the Exit dropdown, choose PopulateddState.

11. Click OK.

12. Click the Developer tab and click Restrict Editing in the Protect Group.

13. In the resulting task pane, click Editing Restrictions and choose Filling in forms.

14. Click Yes, Start Enforcing Protection.

15. Enter the password then click OK.

Sub CopyPaste_ListObj_ValuesOnly()
