Issues I found on Windows 10 Technical Previews - Microsoft Community

gee law say:

Issues I found on Windows 10 Technical Previews

Let's keep a comprehensive and growing, and hopefully shrinking in later versions, list of found issues in Windows 10 Technical Previews!

Welcome to reply an issue.

Warning: This thread is provided as-is and without any warranty.

------------------ The following issues are found on Windows 10 JTP ------------------

    1. Sometimes one cannot remove a group in Start. That is, if a group is named, the group might not disappear when no tile belongs to it.
    2. Search charm legacy problems:
      1. Search charm can be started from any Modern app that has a Charm button on the title bar.
      2. Search charm might start Search Hero app when searching Everywhere, which is now replaced by Cortana and will hang on the splash screen with magnifier logo.
      3. Search charm now shows in-app search options for any searchable app on that desktop.
    3. People tile does not show contacts' profile photos. Plus, there are lots of other Live Tile issues. I hope this is just because it's a preview.
    4. OneDrive synching goes back to Windows 7 style. Bad thing, really bad. It weakens OneDrive. Please do take back Windows 8.1 smart files!
    5. Secondary Tile legacy problems:
      1. When upgraded from Windows 8.1 with Update, previously pinned websites shows up as blank shortcuts and have to be unpinned manually.
      2. Secondary Tiles no longer offer "small" size.
      3. Tapping a Secondary Tile for a contact does not show the contact. I hope this is just because it's preview.
      4. Pinning a Secondary Tile no longer needs user confirmation, which makes it impossible to choose the initial size of the tile.
    6. When using "show search icon" for Cortana on the taskbar, the search icon turns into search box when start menu is activated. The box overlaps the icons next to it.
    7. Multiple selection of tiles on Start is not supported.
    8. Bad behaviour of Snipping Tool under multiple desktops:
      When using multiple desktops on a single screen,       Snipping Tool detects and shows windows that are not on the current desktop when user creates a new Window Snip. When user chooses to snip the misdisplayed area, the resulting picture is the screenshot of the area on the current desktop, instead of the window it represents.
      Should owe this to the "preview". It takes time before Microsoft itself gets to know this compatibility issue.
    9. Calculator and modern Calculator are combined but Sound Recorder and modern Sound Recorder aren't. Strange. Should own this to the "preview". It just takes time.
    10. Item Picker contract legacy problems:
      1. Item Picker contract is not supported in Windows 10 JTP but was supported in last preview build.
      2. Windows Store apps that call into Item Picker to pick a file get a Common Dialog.
      3. Conclusion
        1. Cannot pick multiple files on different locations.
        2. Cannot pick files from an app directly.
        3. Strangely, you still cannot pick two same files in 微博 one by one. Like you can't insert a picture twice in a single Weibo from 微博 (Windows Store app, Universal Windows app). When you pick a second time, the file disappears from the Common Dialog.
    11. After a PIN is added, user cannot sign in Windows with that PIN. I see the role of PIN has changed. This might be by design.
    12. Loading screens (splash + progress ring) of some apps have wrong logo position and wrong scaling level. This can be reproduced with Surface Pro 3 running Weather app. Should be a "preview" problem.
    13. Settings charm legacy problems:
      1. Settings flyout of a Modern app comes out from the rightmost of the screen instead of the app window boundary.
      2. Windows + I is no longer Settings charm. It's now Settings app. Settings charm key (on Type Cover 3) also goes to Settings app.
    14. Sometimes, for example, tapping the toast notification of a new mail or tapping Mail from Start menu, focusing a Modern app does not bring it to front.
    15. Toast notifications doesn't shows app defined colour. This seems to be by design.
    16. Aero Peek for Music doesn't support thumbnail buttons. This should be a "preview" problem.
    17. Aero Peek Inconsistent behaviour:
      1. Aero Peeking a Modern app that is minimized with minimized button on the title bar doesn't really peek it.
      2. Aero Peeking a Modern app that is minimized by clicking its icon on the taskbar does peek it.
      3. Aero Peeking a minimized desktop app does.
      4. When Aero Peek does not really peek the app, the wallpaper is not rendered.
      5. When Aero Peeking Modern apps on another desktop, the wallpaper is not rendered.
      6. Conclusion: the two way of minimizing does not do the same thing. Perhaps Windows have a virtual extra desktop to deposit those Modern apps minimized with minimize button.
      7. Another knowledge: Modern apps are rendered by ApplicationFrameHost.
    18. Snapping supports at most 2 apps in Tablet Mode. This should be by design?
    19. When explorer.exe is restarted from crash, it may be unable to show start menu, start Cortana, launch Task View, show taskbar icons or some notification area icons.
    20. Pen caps for Surface Pro 3:
      1. Clicking the pen cap means start associated OneNote program. No OneNote on the lock screen.
      2. Double clicking the pen cap means Windows + PrnScr, or Windows + VolDown.
    21. Tablet Mode: User cannot show Application Bars of an app by swiping up/down from the edge. User must swipe down from the top to see the title bar, tap the Charms (3 horizontal lines) button and tap App Commands to see commands, which might be covered by the title bar.
    22. Windows + P inconsistent behaviour:
      On Start Screen, Windows + P shows Windows 8.1 style projection option page.
      On Desktop, Windows + P shows Connect flyout.
    23. "Wifi" can be a misspell.

------------------ The issues above are found on Windows 10 JTP ------------------

doug robbins - mvp office apps & services (word) say:
If you execute the merge to a new document (Edit Individual letters) and then run a macro containing the following code when that is the active document, it will send each newsletter to the printer as a separate print task so that the folding and stapling should work.
 
Dim i As Long
With ActiveDocument
    For i = 1 To .Sections.Count
        ..PrintOut Range:=wdPrintFromTo, From:="s" & i, To:="s" & i
    Next i
End With
