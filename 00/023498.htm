Hi Bill,

When printing a web page, browsers usually don't include the href attribute in the anchor tags. However, if there are Cascading Style Sheets (CSS) linked to the web page, it may contain a rule that tells the browser to include the href attribute when printing. If possible, kindly check the style sheets linked to the page if they have this section of code:

@media print {
      a[href]:after {
          content: " (" attr(href) ")";
       }
}

If it has this section of code, you may remove this part of the code, or override it with your own print style sheet. If you are printing this page from the Internet, you can try to save the web page first to your computer, and then try to edit the style sheets. If the web page prints differently when saved offline, you may need to contact the website owner again to adjust their codes.

If you need further assistance, please do not hesitate to send a reply.

rajesh sinha. say:

Jeanette W:

If fact your formula needs little corrections only, and it should written like show below.

=IF(B1=40,0.01923,IF(B1=80,0.03846,IF(B1=120,0.05769,IF(B1=160,0.07692))))


Your original Formula have has few bugs:

=IF(B1=40,0.01923),IF(B1=80,0.03846),IF(B1=120,0.05769),IF(B1,160,0.07692)

  • Check the last IF is written,,   IF(B1,160,0.07692)

Note, after B1 instead of ,(Comma) you must write = sign.

  • In your Formula =IF(B1=40,0.01923) has 0.01923 is TRUE part    but Formula is not considering the next IF as FALSE part, since the TRUE part 0.01923 is finished with  ).

As an alternative I would like to suggest this one, will makes the Formula more competent since it check,, 

  1. Whether the Cell B1 has value or is Blank.
  2. And returns the expected results, when B1 has either of the values, 40, 80, 120, 160.

=IF(OR(B1=40,B1=80,B1=120,B1=160),IF(ISBLANK(B1),"",IF(B1=40,0.01923,IF(B1=80,0.03846,IF(B1=120,0.05769,IF(B1=160,0.07692))))),"")
