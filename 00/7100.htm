a. user say:

> ..  the formula returns 0.3447

Hi.  Once you get this number, you could use LEFT(0.3447,5) to return the string  "0.344"

However, I believe you want to round the value to 0.345.

One of the issues you have is that both numbers  0.1, and 9.9 both consider the leading digit as significant.

Therefore, the easier Log10 functions will give different values.

One additional idea might be to round these values up to the nearest multiple of 10.

Therefore, 0.1 and 9.9 are rounded to 10, and we can now work from here.

We can take advantage that our values fall within the range 0-700.

Here, I used Switch.  You could use the function REPT if you wish.

=TEXT(A1,SWITCH(CEILING(LOG10(CEILING.MATH(A1,10)),1), 1,"0.000",  2,"00.00",  3,"000.0"))

0.345
