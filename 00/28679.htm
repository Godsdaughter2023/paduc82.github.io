You could create a custom layout called LISTS that has the correct formatting for lists. Your users would select that layout when they click the New Slide button to create a slide with the pre-formatted and positioned placeholders.

Ha, so some more poking around and I found a way to fix my problem.  Aliases, as well as the traditional auto sorting, in the new Outlook.com are all now just 'rules'.  So:

  1. rename the old folder (just add a 1 or start to the front of it)
  2. create a new folder with the name you want
  3. move any subfolders to the new folder,
  4. go to 'Settings (cog wheel) -> View all settings -> Mail -> Rules'
  5. Scroll down until you find the rule that moves all that mail to your 'old' folder
  6. Click the pencil to edit the rule
  7. Fix the rule so the new stuff is delivered to the right folder
  8. Test it out from somewhere
  9. Delete the old root folder
