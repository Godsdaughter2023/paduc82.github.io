Sorting data with spaces in between - Microsoft Community

benc44 say:

Sorting data with spaces in between

Hi,

 

I have a spreadsheet that looks something like below - using .... to seperate columns to show what it looks like.  I need to insert more companies below the last one as and when needed.

 

Client number.........Client name.........................Billings

1234..........................Joe Bloggs Ltd....................2010

...................................................................................2009

...................................................................................2008

8888..........................Test Co 1..............................2010

...................................................................................2009

...................................................................................2008

 

To the right of the years are 15 columns with various numbers denoting what services they had bought.

 

Is it possible to sort the spreadsheet by the client name  while keeping all the relevant data associated with that client together?

rich007a say:

I have understood your question to mean you have a text file that uses spaces to "arrange" data into columns, but you want to work with this in Excel, this being the case then...

You should convert the text to columns.  On the menu bar, click Data > Text to Columns...

Looking at your sample, you will probably have to use the Fixed Width option.

 

Next you need to make sure the “empty” cells in column C are really blank (and do not contain spaces).  To do this, select column C and press Ctrl+H.

In the “Find What:” field, enter two spaces.  Leave the “Replace with:”  field blank.  Click Replace All.

That will have cleared most of the spaces, but there could be single spaces left. 

Now, in the “Find What:” field, enter a single space (make sure there is only one), click the Options button and check “Match entire cell contents”. Click Replace All.

 

All cells in column C should now contain client names or will be completely blank.

To be able to sort, you need to fill in the empty cells in the client name column.  Quickest way to do this is to insert a new column to the right of the Client Name column.  So assuming the Client Name in in cell B1, you'd right click on the top of column D and Insert Column.  Then in cell D2 (to the right of "Joe Bloggs Ltd"), enter this formula:

    =IF(ISBLANK(C2),D1,C2)

and then drag down as far as your data to fill column D.

Check column D looks sensible, then copy column D and paste special as values.  Now you can sort your data by client using Data > Sort...

Hope that helps.

Cheers
Rich

Outlook 2016 - Microsoft Community

debsmccarthy say:

Outlook 2016

I have Outlook 2016 and when I do a search it doesn't find all the emails relating to that search.  How do I fix this?

anna bon say:

Hi DebsMcCarthy,

If you search for items in Outlook and don't receive the results that you expect, the indexing of your Microsoft Windows and Microsoft Office system data files (the search catalog) may not be complete. You can rebuild the search catalog, which restarts the indexing of your data files. The search catalog is a file where all of your Outlook and Microsoft Windows items (data files) are indexed. Please follow the steps below:

1. On the File tab, choose Options > Search.

2. Under Sources, click Indexing Options. In the Indexing Options dialog box, under Index these locations, make sure Microsoft Outlook is an included location. You can click Modify if necessary to change selected locations.

Check the Outlook search scope. Click the Current Mailbox list to check the Scope group:

All Mailboxes
Current Mailbox
Current Folder
All Subfolders
All Outlook Items

If it doesn't work, let us try the steps below:

1. Close Outlook and open Control Panel. Depending on the version of Office you're using, you might need to click the Start button, and on the Start menu, right-click Control Panel.

2. Do one of the following:

        a. Choose Indexing Options.
        b. In the Search box, type Indexing, and then choose Indexing Options.


3. In the Indexed Locations dialog box, under Change selected locations, select the check box for the location you want to modify, and then click OK.

4. In the Advanced Options dialog box, on the Index Settings tab, under Troubleshooting, click Rebuild. A Rebuild Index dialog box opens warning that the "index might take a long time to complete." Choose OK to start rebuilding the index.

Once done, please check if you will encounter the same issue.

Let us know the result of the suggested steps.

Thank you.
