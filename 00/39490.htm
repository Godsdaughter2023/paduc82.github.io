a. user say:

Hi,

 

This issue could be caused if the sleep settings in the current power plan are configured incorrectly. By default, the sleep settings are configured in such a way that it enters hibernation if kept on sleep mode for a long time, to save the battery consumption.

 

Refer to these steps to change the power settings:

 

a)      Press Windows key +F to bring the search charms bar.

b)      Type Power and press enter; click on the Settings tab below the search box.

c)      Click on the Edit power plan tile and click on Change advanced power settings.

d)      Expand the Sleep option; change the Hibernate after option to Never and the Allow wake timers to Disable.

e)      Click Ok to save the changes and check if the issue persists.

 

Try these steps and get back to us with the results. We’d be happy to help if you need any further assistance.
