Hi Ercan,

In the OneDrive for Business Next Generation Sync Client, the 20,000 items limitation has been removed.

You can refer to this article to setting it up:

Get started with the OneDrive for Business Next Generation Sync Client in Windows

Get started with the OneDrive for Business Next Generation Sync Client on Mac OS X.

If you are using OneDrive for Business 2013 or 2016, the limitation is still exists. In this situation, the best practice for now is to split folders\files into separate document libraries based on your business. For example, store the files in different document libraries by different contractors or projects. In this way, we can identify the files quickly in different libraries and the file\folder name can be shorten. OneDrive for Business can also be used to sync multiple SharePoint libraries using the steps below:

1. In the task bar, right click on the OD4B icon (little blue clouds) and select Sync a new library.

2. In the window that pops up click Sync a different library instead. Enter the URL to the site containing the document libraries you want to sync and click Sync Now.

3. Add as many libraries as you like (remember they are synced to your local machine so be aware of space requirements).
