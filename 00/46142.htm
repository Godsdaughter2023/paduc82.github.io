From your description and from judging on the appearance of your screenshot, it seems to me that the app you're using is not "high-dpi aware".  It doesn't know how to scale up text/assets to view on your high-resolution monitor.

You can read about this here: http://cmdrkeene.com/dpi-scaling-with-multi-monitor

It's for a related topic, but it has the background about what causes the error.

The DPI settings, if you want to experiment and find what works best for you, is here:

Right-Click Desktop > Resolution > Make Text and Other Items Larger or Smaller (link at bottom)

It is clear to me from your screenshot that the program's interface is enlarged (tabs/menus) so it looks like your current DPI scaling is set to probably 125% or 150%.  But the program itself is not scaling up the internal text (the actual code).  That means the program doesn't understand how to use high-definition screens, so you can either bump up the font size the program is using, or lower your DPI so that your fonts are at least consistent (use 100%).

Hi

 

Running 2 anti-virus programs on the same computer can cause major conflicts.

 

Windows Defnder is designed to disable itself if it detects another product is installed.

 

To enable Defender, you will first need to uninstall the Norton product.

 

Norton removal requires a special tool. Go to the following website to download the tool.

 

Download and run the Norton Removal Tool to uninstall your Norton product:
https://support.norton.com/sp/en/us/home/current/solutions/kb20080710133834EN_EndUserProfile_en_us;jsessionid=B906151014A898270B141F50D9E83257.4?product=home&product=home&pvid=f-home&version=1&version=1

After Norton is removed, reboot the computer and enable Defender.
