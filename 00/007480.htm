  4. Locate and delete the config or config.xml file.

  6. Restart your Windows computer.

yehia zakaria say:

history of outlook

i create a new profile in outlook and all mail history not found

can i recover it

i havenot add any one to outlook contact


pls help
brian tillman [outlook mvp 2007-2019] say:
What there a specific reason you created a new mail profile?

What type of mail account do you use?  If an Exchange account, all your past messages shoould still be in your Exchange mailbox.  Just recreae the Exchange account.  If you use an IMAP account, the answer is the same: all your messages should still be on the server and creating the account will restore them.  Likewise, if you use a POP account and you left copies of the messages on the server, they'll all download again and you'll have them.

If use POP and don't leave messages on the server, then it's likely thay are still on your PC.  The first thing to do is to open the Mail applet in Control Panel (Start>Control Panel>Users Accounts category>Mail for the Category view or Start>Control Panel>Mail for the Classic view) and click Show Profiles.  DO you see more than one mail profile listed?  If so, you can select the one not currently selected and tell the applet to make Outlook use that profile when starting or you can allow Outlook to ask which profile you want.  Then start Outlook.  Do you see your data now?

If you have only one mail profile, then click Data Files and then Add.  You should get a browser widget that shows you your existing data file (PST).  It may also show you another.  Select that one and click OK.  Close the Mail applet and start Outlook.  You should see two sets of folders now.  Does the second set contain your data?  If so, click Tools>Account Settings>Data Files, select that data file, and click Set As Default.  Now stop and restart Outlook.  It should look as it did before.  Now click Tools>Account Settings>Data Files and remove the extra data file created when you created the new profile.
