hansv mvp say:

Here is a custom function:

Function ConcatResults(Headers As Range, Values As Range) As String
    Dim s As String
    Dim i As Long
    For i = 1 To Headers.Count
        If Values(i) <> 0 Then
            s = s & ", " & Headers(i) & "(" & Values(i) & ")"
        End If
    Next i
    If s <> "" Then
        s = Mid(s, 3)
    End If
    ConcatResults = s
End Function

Use like this for the example that you provided:

=ConcatResults($A$1:$D$1,A2:D2)

For more columns, simply expand the ranges. Since the 40th column is column AN:

=ConcatResults($A$1:$AN$1,A2:AN2)

  • The new blank workbook should be created only if you directly launch the program from the Applications folder or from the Dock/Mission Control/etc. If you double-click an Excel file the program should launch without creating the blank workbook, and
  • Even if the blank workbook is created by launching as described above, it should evaporate when an existing file is opened provided no changes have been made in the blank file.

If you're experiencing something other than what is described above, please provide a complete & accurate step-by-step explanation of what method you're using to open files along with the exact results.
