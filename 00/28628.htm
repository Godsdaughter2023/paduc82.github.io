Mr Faisal,

Thanks for your feedback and I'm glad the VBA is doing what you wanted!

While I was looking through the code I realised that the next to last activation of "Book3" is unnecessary and may slow down the overall execution of the routine, so I would suggest the following instead:


Sub BranchBooks()
'
Dim i As Integer
Dim j As Integer
Dim br_code As Variant
'
Workbooks("Book3.xlsm").Activate
'
Worksheets(1).Range(Cells(2, 1), Cells(2, 7).End(xlDown)).Sort _
    key1:=Worksheets(1).Cells(2, 1), Header:=xlNo
'
i = 2
j = 1
br_code = Worksheets(1).Cells(i, 1)
'
Do Until br_code = ""
'
    Workbooks.Add
    '
    ActiveWorkbook.SaveAs Filename:=ThisWorkbook.Path & "\Branch Code " & br_code, _
        FileFormat:=xlWorkbookDefault, ReadOnlyRecommended:=False, CreateBackup:=False
    '
    Workbooks("Book3.xlsm").Activate
    Worksheets(1).Range("A1:G1").Select
    Selection.Copy
    Workbooks("Branch Code " & br_code & ".xlsx").Activate
    Worksheets(1).Range("A1:G1").Select
    ActiveSheet.Paste
    '
    Workbooks("Book3.xlsm").Activate
    '
        Do Until Worksheets(1).Cells(i + j, 1) <> br_code
            '
            j = j + 1
            '
        Loop
    '
    Worksheets(1).Range(Cells(i, 1), Cells(i + j - 1, 7)).Select
    Selection.Copy
