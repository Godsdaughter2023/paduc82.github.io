suzanne s. barnhill say:
You can use a SEQ field. It will not update automatically but will update
when you update fields. To insert the field:
 
1. Press Ctrl+F9 to insert field braces.
 
2. Type SEQ flag between them, and F9 to update. This will give you the
number alone.
 
3. Type SIDE FLAG and a space before the number field.
 
4. Select the entire phrase and save it as an AutoText entry.
 
When you want a new side flag number, insert this AutoText entry (the number
will be correct upon initial insertion but, as noted, will not update
thereafter unless you explicitly update fields by printing, switching to
Print Preview, or pressing Ctrl+A, F9).
