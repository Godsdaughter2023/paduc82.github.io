This is the issue:

Resource and Performance Monitor does NOT find the antivirus product, since
it is searching in the Windows Management Instrumentation (WMI).

Since, the Resource and Performance Monitor is searching using old query,
with old path.

Old Query: root\SecurityCenter:SELECT * FROM AntiVirusProduct
New Query: root\SecurityCenter2:SELECT * FROM AntiVirusProduct

Now, since the case is about the older query, we end-up getting a message about that

drkrieger say:

Just a heads up if anyone else comes across this issue, check to see if your system partition is full. The system partition on a non-UEFI install of Windows 10 is typically 100-260MB. If it's UEFI, it's typically 500MB.

If you do not have enough space in this partiton DO NOT delete any files within it. Simply use a utility like Mini-Tool Partition Wizard to shrink/move your main OS partition by ~100MB, then extend your system partition to give it more space.
