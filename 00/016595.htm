batch convert xls to xlsx

Hello,

 

Is there a tool that would allow me to batch convert 30 xls files within a single folder into xlsx files?

Mark

hansv mvp say:

Here is a macro you could run:

 

Sub ConvertToXlsx()
    Dim strPath As String
    Dim strFile As String
    Dim wbk As Workbook
    ' Path must end in trailing backslash
    strPath = "C:\Excel\"
    strFile = Dir(strPath & "*.xls")
    Do While strFile <> ""
        If Right(strFile, 3) = "xls" Then
            Set wbk = Workbooks.Open(Filename:=strPath & strFile)
            wbk.SaveAs Filename:=strPath & strFile & "x", _
                FileFormat:=xlOpenXMLWorkbook
            wbk.Close SaveChanges:=False
        End If
        strFile = Dir
    Loop
End Sub

 

Warning: .xlsx workbooks don't store macros, so any macros from the .xls workbooks will be lost.

daniel g. grau say:
I don't know why you are not getting any error messages. I certainly am with the parameters you indicated.

First of all, set your units in Word > Preferences > General > Measurement units. (You should have done this already, since you are using centimeters.)

If you take a look at your values, the vertical length required is:

  • Top margin + (Vertical pitch x Number of labels down) = 1.5 + (4.2 x 7) = 30.9 cm

That won't fit in 29.7 cm down. You can only fit six labels.

Now, using your horizontal values, the required width is:

Again, that won't fit in 21 cm.

Error or no error, make sure your values result in measurements that can fit on the specified page, error or not.

ruchi bisht say:

Ensure multiple versions of Microsoft Office do not exist on the computer.

 

Try the step suggested by Stefan; alternatively you may also try to register the DWORD value using the following steps which doesn’t involve opening the Registry Editor.

 

Note: read the disclaimer given in the link below before modifying the registry in Windows.

 

Press Windows key + R, in the Run Window, type the following commands one by one and then press Enter:

 

1.     reg add HKCU\Software\Microsoft\Office\12.0\Word\Options /v NoReReg /t REG_DWORD /d 1         

2.     reg add HKCU\Software\Microsoft\Office\12.0\Excel\Options /v NoReReg /t REG_DWORD /d 1         

3.     reg add HKCU\Software\Microsoft\Office\12.0\Outlook\Options /v NoReReg /t REG_DWORD /d 1      

4.     reg add HKCU\Software\Microsoft\Office\12.0\PowerPoint\Options /v NoReReg /t REG_DWORD /d 1 

5.     reg add HKCU\Software\Microsoft\Office\12.0\A

 

If the issue still persists then uninstall Office 2007 using the steps givenhere and then reinstall it. Keep the disc and the product key handy for Office 2007 reinstallation. For Windows 7, follow the steps given for Windows Vista.
