myronh say:

There are two ways this may be fixed. The easier way requires a registry change though so you might not want to go that route.

1.       Click Start Menu, and then type regedit in the Start Search and press ENTER. Right click on regedit from the search results and run as administrator.

2.       Navigate to the following registry branch:

HKEY_CURRENT_USER\Software\Microsoft\Windows\Curre ntVersion\Explorer\FileExts\[ext]

3.       Replace the [ext] with the actual file extension that you want to restore its file type association to revert back to original Windows 7 default (probably .lnk). If you unsure, simply browse through all the sub-key under FileExts.

4.       Delete the sub-key named UserChoice.

Exit from Registry Editor.

The second method is quite a bit more work, may not fix it, but doesn't involve the registry.

Since the default action is caused by a user choice (inadvertant as it was), you may be able to restore simply by creating a new user account. See if the icons return to their natural state and now work. If so, just move all of your files and folders over and delete the old account. If you use Outlook you may also have to reset where it looks for the Outlook files, but it might be automatic (don't try starting Outlook until everything is copied over though). If you use this method, don't forget to copy the hidden appdata folder. Just unhide everything before you start the copy.
