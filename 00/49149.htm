patrick barker say: Hi,

Hardware acceleration in computing terms is when the OS uses the hardware to perform tasks so that it gets done faster, etc. For example, normally when rendering a video it's done by the CPU. However, if the editing program, Flash player, etc you have supports rendering by Hardware acceleration, you can then render with the GPU (video card) and have it go by much faster.
