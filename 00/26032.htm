ron coderre say:

There are 2 templates you'll want to create: One for new workbooks and one for new worksheets.

 

1) Create a new workbook and customize it with any formatting and structure preferences you want (fonts, headings, formulas, number of sheets, etc)

2) File.Save

...Save_as_type: Excel Template (*.xltx)

...File name: BOOK.XLTX .....Not BOOK1

....Browse to your XLSTART location (Mine is: C:\Program Files\Microsoft Office\Office14\XLSTART)

...Save the file there.

 

3) Create a new workbook with only ONE worksheet.

...you might want to just use the BOOK.XLTX workbook and remove all but one sheet

4) File.Save

...Save_as_type: Excel Template (*.xltx)

...File name: SHEET.XLTX

...Save the file in the same XLSTART location

 

5) Close and re-open Excel

Now, when you create a new workbook, Excel will use the BOOK template.

When you insert an new worksheet, Excel will use the SHEET template.

(If you don't create a SHEET.XLTX template...new worksheets will have the default Microsoft formatting)

 

Does that help?

doug robbins - mvp office apps & services (word) say:

At the point in your Template where you want the information to appear (probably in place of the Rich Text Box), press Ctrl+F9 to insert a pair of field delimiters { } and inside them type

 

DOCVARIABLE varA2

 

Press Alt+F9 to toggle off the display of the field codes.

 

Then replace the following section of your code

 

' enter text to Template
With wrdDoc
ActiveSheet.Cells(R2, C1).Select
Selection.Copy
wrdDoc.Selection.Paste
End With

 

with

 

' enter text to Template
With wrdDoc
    .Variables("varA2").Value = Range("A2").Value
    .Range.Fields.Update
End With
