    techfreak_ say:

    Since applications like Xsplit will only natively function with desktop applications (not Apps), you will either need to use a desktop version of Skype or a version (desktop/App) that has NDI support. 

    Skype for Windows 10 App (12.x/14.x) compatibility:

    1. Xsplit is incompatible with Windows 10 Apps. Xsplit, not Skype, would need to create a version that is natively compatible for the virtual devices to be detected and available to any Windows 10 App like the Windows 10 Skype and Camera Apps.

    2. Xsplit should function over NDI. If future NDI support is added, it will more than likely exist first in the Skype 14.x insider release. If you are part of the Windows Insider program, you can get these releases with your Window 10 Insider builds.

    Skype for Windows Desktop (7.x 8.x) compatibility:

    1. Xsplit is compatible with desktop applications. Due to this, Xsplit virtual devices should be available in the audio/video settings of any Skype desktop application.

    2. Xsplit should function over NDI. NDI support exists in the current insider release of Skype 8.x, It can be downloaded from the Skype Insider page. If you do not get a Skype for Windows drop-down (when signed in), try loading the page in another browser like FireFox.

    I suggest if you require compatibility now that you resort to using either the release or Insider pre-release of Skype 8.x. These releases are currently compatible with Windows 10.
