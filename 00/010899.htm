The account under which the messages are to be sent must be set as the default account under Outlook.

In Outlook 2010, that is done by clicking on File>Account Settings>Account Settings, which will display the Account Settings dialog and then on the E-mail tab of that dialog, select the account from which the messages are to be sent and then click on the "Set as Default" item in the toolbar below the tabs in the dialog.
