In Excel 2007, one rather simple way is to format the range as a table.  When you click the Format as Table button in the HOME tab, you can check the box for My table has headers.  Doing this will also automatically engage Filter and you can then click the filter dropdowns on any column and the first two options are sorting options.

=MODE(IF(SUBTOTAL(3,OFFSET(Table1['# hours],ROW(Table1['# hours])-MIN(ROW(Table1['# hours])),0,1)),Table1['# hours]*{1,1}))

Copy a worksheet contents to another workbook's worksheet - Microsoft Community

rt_56 say:

Copy a worksheet contents to another workbook's worksheet

Is there a way (using VBA) to copy the contents of a open worksheet, then open an existing workbook and place the copied results to a worksheet based on which worksheet the user selects in the existing workbook?

jllatham say:

There were a few things not mentioned, like if the user would be opening the other workbook or if they need to pick that also.

To give them a list of worksheets to choose from, it's easiest (in my mind) to set up a UserForm with a ListBox to put the list of sheet names into and let them pick from that and then do the copy.

To that end, I put together the workbook below.  If you don't understand userforms and such, just say so, but I kept the default names for the form and the controls on it, and I will post all of the code for both the code module in the workbook and the userform below.

Basically you start with the Sub CopyToOtherWB() which does the job of letting the user pick the other workbook to be used, opening it and setting a reference to it that the userform can use, then it opens the userform.  The userform does the work of copying and once done, it closes the other workbook, saving the changes and removes itself after announcing that the job was done.  It will copy your selection to the next available row on the destination sheet beginning in column A.

Link to the 'master file':

http://www.mediafire.com/download/kicwia26catieif/For_RT_56_MasterBook.xlsm

Code for the code module in the workbook, note that otherWB is a Public variable declared before the Sub starts so that it can be used by the userform:

'this needs to be Public so UserForm1 can 'see' it
Public otherWB As Workbook

Sub CopyToOtherWB()
  Dim fileToOpen As String
  'get user to identify the other workbook
  'you could 'hard code' this if it is always
  'one particular workbook
  fileToOpen = Application.GetOpenFilename(filefilter:="Excel Files (*.xls*),*.xls*", _
   Title:="Select Destination File", MultiSelect:=False)
  If fileToOpen = "False" Then
    Exit Sub ' user cancelled file select
  End If
  Application.ScreenUpdating = False
  'open other workbook: Update Links = False, Read Only = False
  Set otherWB = Workbooks.Open(fileToOpen, False, False)
  ThisWorkbook.Activate
  UserForm1.Show  ' where the actual copying is done.
End Sub

'********

and now the code for the UserForm

Private Sub UserForm_Initialize()
  Dim anyWS As Worksheet
  ListBox1.Clear ' remove any old entries
  'otherWB must have been opened before
  'this userform is brought into play.
  For Each anyWS In otherWB.Worksheets
    ListBox1.AddItem anyWS.Name
  Next
End Sub

Private Sub CommandButton1_Click()
'this is the Copy... button
  Dim selectionToCopy As Range
  Dim lastRow As Long
  Dim otherWS As Worksheet
 
  If ListBox1.ListIndex < 0 Then
    MsgBox "Please Select from the list on the left.", _
     vbOKOnly + vbExclamation, "Try Again"
    Exit Sub
  End If
  Set otherWS = otherWB.Worksheets(ListBox1.List(ListBox1.ListIndex))
  'find last used row on the other worksheet
  On Error Resume Next
  lastRow = otherWS.Cells.Find(What:="*", _
   After:=otherWS.Cells(1), _
   Lookat:=xlPart, _
   LookIn:=xlFormulas, _
   SearchOrder:=xlByRows, _
   SearchDirection:=xlPrevious, _
   MatchCase:=False).Row
  On Error GoTo 0
  'put selected cells onto the clipboard for copy
  Selection.Copy
  otherWS.Range("A" & lastRow + 1).PasteSpecial xlPasteValues
  Application.CutCopyMode = False
  'we are done, clean up and exit
  Set otherWS = Nothing
  'notify user that it actually worked
  MsgBox "Copy accomplished, " & otherWB.Name & " will now be closed.", _
   vbOKOnly + vbInformation, "Task Completed"
  'save changes to the other workbook and close it
  otherWB.Close True
  'release resource back to the system
  Set otherWB = Nothing
  'remove this form from screen/memory
  CommandButton2_Click
End Sub

Private Sub CommandButton2_Click()
'this is the Cancel/Exit button
  'if otherWB is not already closed, then
  'close the other workbook, do not save any changes made
  If Not otherWB Is Nothing Then
    otherWB.Close False
    Set otherWB = Nothing
  End If
  Unload Me
End Sub
