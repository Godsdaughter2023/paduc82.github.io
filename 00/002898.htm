How can I search for italics along with particular unitalicized - Microsoft Community

jselhorst say:

How can I search for italics along with particular unitalicized characters following the italics?

I'm working in Word 2010, and I need to find all instances of italics in a document and make sure that any commas and periods that directly follow the italics are also italicized (the style my client requires). Most of them are not already italicized, and it is sort of time-consuming just to search for any italic format and then leave the search box to individually italicize each piece of punctuation that needs changing. Is there a way to search for italic format followed by a nonitalicized period or comma, and then replace that with italic format that includes the period or comma?
jay freedman say:
The expression ActiveDocument.Range in the previous macro refers only to the main body of the document. This slightly modified version will handle the endnotes, as well as footnotes, text boxes, and a number of other parts of the document that are ignored by the previous one.

Sub ItalicPunctuation()
    Dim punct As String
    Dim rg As Range, rgTest As Range, rgStory As Range
    Dim i As Long
  
    ' Add any other desired characters between the quotes
    punct = ",."
    For i = 1 To Len(punct)
        For Each rgStory In ActiveDocument.StoryRanges
            Do
                Set rg = rgStory
                With rg.Find
                    .Text = Mid(punct, i, 1)
                    .Wrap = wdFindStop
                    While .Execute
                        Set rgTest = rg.Duplicate
                        rgTest.Start = rg.Start - 1
                        rgTest.End = rg.Start
                        rgTest.Select 'debug
                        If rgTest.Italic Then
                            rg.Italic = True
                        End If
                        rg.Collapse wdCollapseEnd
                    Wend
                End With
                Set rgStory = rgStory.NextStoryRange
            Loop Until rgStory Is Nothing
        Next rgStory
    Next i
End Sub

Even this version isn't comprehensive and general-purpose, for reasons explained at http://www.word.mvps.org/FAQs/Customization/ReplaceAnywhere.htm. I didn't take the time to apply the full suite of techniques explained there to handle gaps in headers and footers, or story ranges that are nested inside other story ranges, on the assumption that your documents aren't that complicated.

jay freedman say:
It isn't possible to find a string with mixed italic and non-italic formatting (or any similar character formatting, such as bold or underlined) by using the Find or Replace dialog. However, it is possible to solve your problem with a macro.

The logic of the following macro is to find each punctuation mark -- using the list of marks in the string named punct -- and examine the character that comes before it. If that character is italic, then the punctuation is made italic. (The macro doesn't check whether the punctuation is already italic; making an italic character have italic formatting is the same as leaving it as is.)

Sub ItalicPunctuation()
    Dim punct As String
    Dim rg As Range, rgTest As Range
    Dim i As Long
   
    ' Add any other desired characters between the quotes
    punct = ",."
   
    For i = 1 To Len(punct)
        Set rg = ActiveDocument.Range
        With rg.Find
            .Text = Mid(punct, i, 1)
            .Wrap = wdFindStop
            While .Execute
                Set rgTest = rg.Duplicate
                rgTest.Start = rg.Start - 1
                rgTest.End = rg.Start
                If rgTest.Italic Then
                    rg.Italic = True
                End If
                rg.Collapse wdCollapseEnd
            Wend
        End With
    Next i
End Sub

See http://www.gmayor.com/installing_macro.htm if you need instructions.
