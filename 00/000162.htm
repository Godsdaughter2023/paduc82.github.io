amit tandon say:

To List all values once (Unique List) which are common in both columns A & B, enter below formula in cell D2 & copy down:

=IFERROR(INDEX($A$2:$A$100,MATCH(0,INDEX(COUNTIF(D$1:D1,$A$2:$A$100)+(COUNTIF($B$2:$B$100,$A$2:$A$100)<1),,), 0)),"")

Note: If cell D2 formula is in a different column, replace "D$1:D1" in the formula with that column (ex. if formula is in column F then update to "F$1:F1")

To List all Values Once (Unique List) from columns A & B, enter below formula in cell F2 & copy down:

=IFERROR(IFERROR(INDEX($A$2:$A$100, MATCH(0,INDEX(COUNTIF(F$1:F1,$A$2:$A$100)+($A$2:$A$100=""),,), 0)),INDEX($B$2:$B$100, MATCH(0,INDEX(COUNTIF(F$1:F1,$B$2:$B$100)+($B$2:$B$100=""),,), 0))), "")

Note: If cell F2 formula is in a different column, replace "F$1:F1" in the formula with that column (ex. if formula is in column G then update to "G$1:G1")

jeeped say:
In addition to Héctor Miguel's suggestion, a UDF could be based upon the .Text property of the cell. The .Text is the displayed text (complete with AUD, GBP, etc) rather than the underlying numerical value.

Public Function udf_Sum_Currency(rng As Range, curr As String)
    Dim r As Range, ttl As Double
    For Each r In rng
        If Right(r.Text, Len(curr)) = curr Then _
            ttl = ttl + CDbl(r.Value)
    Next r
    udf_Sum_Currency = ttl
End Function
