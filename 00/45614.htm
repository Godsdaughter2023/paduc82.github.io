hairyfool say:

Both of the Output interfaces are just that, an Output channel. You still only have one audio device converting the digital data stream into an audio waveform. Because Bluetooth is itself a digital device it then has to convert it into A2DP to transmit it.

In order to do what you want you would need to feed each audio file/source directly to the Bluetooth but that is in a format it cannot handle. To do what you want each speaker has to be LAN based so it can upload its data file individually and then convert it into the audio waveform and not process it on the computer. As a server Windows can easily feed two files at a time because it doesn't process them in any way.
