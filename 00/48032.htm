how do i open the file explorer on windows 8

I have a document in file that I created yesterday and today I cant find the tile that opens the screen on which the document is listed.  I had it yesterday but today I don't see it.
jetta48 say:

To find File Explorer and Pin it to the taskbar or Start screen:

1. Move mouse pointer to bottom right or top right of screen to bring up the Charms bar then select Search

2. Make sure Apps is highlighted beneath the Search box then type File Explorer into the Search box.

3. Now right-click on the File Explorer icon that should have appeared on the left and select Pin to Start or Pin to taskbar depending on what you want.

cubman1941 say:

All of a sudden I find I can receive but not send emails, why?

I can receive email but can not send any.  I get Failed to send (now 7 emails)
¡firedog say:

If you have one message that failed to send in your outbox, it may be preventing subsequent messages from being sent. Click Work offline, then remove all the messages from the outbox (below your account folders in the folder pane). Now send a new test message.

The error message you see will explain why send is failing. Click on the error message to select it, then use right-click to copy it and Ctrl-V to paste it into your reply here. There can be no troubleshooting without seeing the entire error message.
