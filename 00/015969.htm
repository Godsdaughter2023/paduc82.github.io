A combination of the Replace() and Int() functions should work:

Int(Replace(<TheField>, "-", ""))

hansv mvp say:

Create a query based on the table.

Let's say the field is named MyField.

Create a calculated column

JustDigits: Replace([MyField],"-","")

and another one

RealNumber: Val(Replace([MyField],"-",""))

milbourne say:

turn off autonumbering in word tables

Am curious: When I insert a new row into a table within Word 2013, it automatically updates the numbering so that it

* continues from the previous row

* but resets the font and indentation

My question is how can I continue the same font and indentation from the previous row BUT more importantly how do I turn off the autonumbering?

Regards

doug robbins - mvp office apps & services (word) say:

You can turn off the autonumbering by clicking on the Numbering button in the Paragraph section of the Home tab of the ribbon.

That will however remove the indentation and maybe the font formatting if the numbered paragraph in the preceding row has been formatted with a font that is different from that of the Normal style.  You will therefore need to modify the formatting of the paragraph in the new row so that it has the required font and indent.
