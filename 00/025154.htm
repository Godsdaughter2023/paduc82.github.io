This formula returns 50% i.e. 50% of Ford cars that have a colour are green

joeu2004 say: PAL17 wrote:
Let's forget "yes" for now.  Lets say that column D are autos, column F is colors.  A4 is Ford.

I am looking in column D for all the Fords and then I would like the percentage of autos that are green.  If there are 5 cars and only 4 have colors, give me the percentage of cars that are green from the 4.  If there are no colors for all the fords, then return a blank.

Let's try this:  the percentage of all the cars=A4 that are Green; if there are no cars=A4, then return blank.

=IFERROR(COUNTIFS(D1:D100,A4,F1:F100,"green")/COUNTIF(D1:D100,A4),"")

Note that I use "" (null string) instead of " " (one space).  The latter will be harder to deal with later.

If you truly want to ignore cars=A4 that have no color stated, then:

=IFERROR(COUNTIFS(D1:D100,A4,F1:F100,"green")
/COUNTIFS(D1:D100,A4,F1:F100,"<>"),"")

If F1:F100 might contain cells that  appear  blank but actually contain null strings that would like to ignore (like formulas above), then:

=IFERROR(COUNTIFS(D1:D100,A4,F1:F100,"green")
/COUNTIFS(D1:D100,A4,F1:F100,"?*"),"")

That will not count cells in F1:F100 that contain numbers.  Is that a potential issue?  If so, try the following as a last resort:

=IFERROR(COUNTIFS(D1:D100,A4,F1:F100,"green")
/SUMPRODUCT((D1:D100=A4)*(F1:F100<>"")),"")
