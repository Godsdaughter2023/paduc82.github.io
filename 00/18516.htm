Save custom watermark in MWord 2016 - Microsoft Community

robertchmielinski say:

Save custom watermark in MWord 2016

I have a constant need for unique watermarks for my documents in MWord 2016.  I can write them under the "Design" tab, but I can't figure out how to save them so that I don't have to re-write them each time.  Does anyone have a procedure for this?
jay freedman say:

After you create and insert a custom watermark that you will reuse:

  • Open the header pane (double-click in the header, or right-click there and click Edit Header).
  • Click on the watermark to select it. You should see the selection handles at the corners and sides of the watermark.
  • Press Alt+F3 to open the Create New Building Block dialog.
  • Enter a name for the watermark.
  • In the Gallery dropdown, click Watermark.
  • In the Category dropdown, the first time you do this, click Create New Category and enter a name for a group of watermarks. If you start the name with an underscore, the group will appear at top of the Watermark gallery. For later watermarks, you can choose the same group or create another category.
  • Enter a description (this step is optional).
  • In the Save In dropdown, choose which template will store the building block. If the current document is based on a specific template, that template will be in the list; otherwise you'll be shown only Normal.dotm and Building Blocks.dotx (which are specific to your computer).
  • Click OK.
  • On exiting Word, if you're prompted to save the template you chose above, say Yes.

The building block you created will now appear in the gallery when you click Design > Watermark.

Select text and Convert to Table (2 columns, separate at paragraph marks)

Select column 1 and apply a style

Select column 2 and apply a different style

Now convert the table back to text (choosing paragraph as the separator again)

Macro method (untested concept)

Sub ZebraPara()

Dim i as integer

For i = 1 to ActiveDocument.Paragraphs.Count step 2

  ActiveDocument.Paragraphs(i).Style = "Normal Indent"

Next
