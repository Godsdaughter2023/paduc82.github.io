Open Outlook.

Whenever you add a hotmail account in outlook it automatically creates an ost file. This ost is needed to link outlook and the webserver.

You cannot transfer the ost file to a different computer and sync it with the hotmail account created. The ost willl work only with the account with which it was created.

If you need to backup the ost file, you can export the file as a pst and move it to a different machine.

Merging emails in outlook 2010 - Microsoft Community

tammy brandon, ms say:

Merging emails in outlook 2010

How do I merge all of my 4 emails into one outlook inbox, sent box, etc.   This is the way it was set up in 2003 version and it works the best for me.
brian tillman [outlook mvp 2007-2019] say:
Simply tell Outlook to use an existing PST instead of creating a new one when you add an account.  For existing accounts, use File>Account Settings>Account Settings.  On the E-mail tab, use the Change FOlder button to tell your accounts where to deliver the mail.  Specify the Inbox of the PST you want to use.

Adding to Brian’s suggestion, when you add the accounts Outlook 2010 defaults to giving each account their own folder. 

Go to the File tab > Account Settings button > Select the account then click the change folder and select the folder you want to use.

Note: this applies if you are using a POP account.
