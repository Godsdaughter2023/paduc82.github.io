However you want to move all fonts so all fonts are usable with all users.
  1. System Fonts can not and should not be moved unless you want your computer to Crash
    Plus the fact system fonts are accessible to all users anyway.

  2. With all applications shut down and just showing finder:
    1. Go to the Go Menu
    2. Hold Option (⌥) key down
    3. Go to Library when shows up in the menu
    4. Go to Library
    5. Go to Fonts
    6. Move all Contents to Desktop
    7. Close Fonts > Then Library then User

  3. While still in Finder Double-Click the Hard Drive to open.
    Open Library
    1. Open Fonts
    2. Drag fonts on desktop to this Fonts Folder

  4. Now open FontBook
    1. Check for Duplicate Fonts and for defective fonts, if none fond your ready to go.

To check fonts what's recommended in either of these two Links:

http://www.officeformachelp.com/office/font-management/


And Be sure while doing either of two you look at this:

http://www.phillipmjones.net/Font_Book.html


From now on when you add a Font with Font Book specify which Library to place the font. Read Help in Fontbook to find out what library to place Fonts.
