1972vet say: The appropriate command would be:
chkdsk /f

You can copy and paste that or type it into your command prompt to run the disk check, or you can use Windows GUI by doing the following:
Right-Click on the c:\ drive and select "Properties-->Tools. At the top, under the "Error-checking" heading, click the "Check Now..." button. Check the box in both boxes, then click "Start". You'll need to click to apply and ok that, then close down and reboot the computer. The chkdsk utility will scan automatically and when it completes, windows will automatically reboot on it's own. Note--you should do nothing else with the computer while this is underway. You can view the results in the event log:
  • Click start-->run...type eventvwr.msc then click "OK".
  • In Event Viewer, click the "Application" log.
  • In the right pane, look down the Source column for "Winlogon" and double-click on that entry to view the results.

    • At this point, you should be able to run your defrag using the same instructions from the above, except instead of clicking the "Error-checking" button, this time click the "Defragment Now..." button.

    a. user say:

    Do you mean the Windows Experience Index (WEI)?

    Why do you think turning on or off DEP will help your WEI score - and turning off DEP is never a good idea unless you know of a poorly coded third party program that you must have/run that is so sloppy that it causes DEP exceptions.   DEP helps protect your system from programs that might do it harm so it is best to leave it alone.

    The WEI base score is a combination of several factors and what you see as the Base score is the lowest subscore of those components.

    Many experienced community contributors suggest the WEI scoring is irrelevant overall but to some people it seems to be important.

    Click the link next to the rating to see how the rating was calculated and find out which component has the lowest score. 

    In this example, the Graphics (video) component has the lowest score of 4.2:

    The lowest subscore is 4.2 so that will be the Base score rating you see on the top page.

    In that example the video subsystem might be considered a "weakness" of the system so if you wanted to improve the rating, get a better video card but then the lowest score will never be higher than 5.5 (the RAM) so if the system can handle more RAM you could add more RAM.

    If that rating number is important to you, find out what your lowest score is and consider how you might improve it with some Google searches like this and you can get almost 3 million ideas:
