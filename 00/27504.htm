NDR Certification validation failure - SubjectMismatch

3/20/2016 6:43:48 AM - Server at TY1PR06MB0926.apcprd06.prod.outlook.com returned '550 5.4.300 Message expired -> 451 4.4.0 Primary target IP address responded with: "454 4.7.5 Certificate validation failure, Reason:SubjectMismatch." Attempted failover to alternate host, but that did not succeed. Either there are no alternate hosts, or delivery failed to all alternate hosts. The last endpoint attempted was 131.242.135.171:1701'
3/20/2016 6:33:46 AM - Server at jpn01-internal.map.protection.outlook.com (25.152.184.27) returned '451 4.4.0 Primary target IP address responded with: "454 4.7.5 Certificate validation failure, Reason:SubjectMismatch." Attempted failover to alternate host, but that did not succeed. Either there are no alternate hosts, or delivery failed to all alternate hosts. The last endpoint attempted was 131.242.135.171:1701'

The endpoint mentioned is an Exchange 2010 server (and the only one responsible for sending emails). The only other consistent part is the jpn-01-internal address mentioned. I understand that has two IPs - the only failures we see is when 25.152.184.27 is used.

Anyone have any suggestions?

owenbarnes5169 say:

I ended up fixing this. The issue was our 'Inbound from Office 365' connector had out of date IPs specified on the Network tab.

When Exchange Online would attempt to send email to our server from newer IPs, it would use the default connector (rather than our o365 one) - resulting in the NDR with the certificate failure.

I used the following script to update the IPs on the connector:

$FormatEnumerationLimit =-1

$ip = Get-HybridMailflowDatacenterIPs

Get-ReceiveConnector "Inbound From Office 365" | Set-ReceiveConnector -RemoteIPRanges $ip.DatacenterIPs

Once updated, Exchange Online was able to contact and use the correct connector and has been working normally since.

Is the document being created by mail merge?

If not, try

Dim i As Long, j As Long

Dim DocSource As Document

Dim Doctarget As Document

Dim prange As Range, trange As Range

j = 1

With DocSource

    While .Range.Start <> .Range.End

        Set Doctarget = Documents.Add

        For i = 1 To 6

            Selection.HomeKey wdStory

            Set prange = .Bookmarks("\page").Range

            prange.Cut

            Set trange = Doctarget.Range

            trange.Collapse wdCollapseEnd

            trange.Paste

        Next i

        Doctarget.SaveAs2 "Document" & j & ".docx"

        j = j + 1

    Wend

End With
