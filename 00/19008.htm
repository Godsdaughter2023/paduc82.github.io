rich007a say:

I have been able to get it to calculate an equivalent amount in Euros based on a date range through manual entry via IF formulas e.g. IF(A1>"1/1/2014,A1<"31/1/2014", TRUE, FALSE)

To use an if statement to return TRUE or FALSE is unnecessary.

=1=1  in a cell will return TRUE

=1=2  in a cell will return FALSE

=A1=DATEVALUE("1/1/2014")      would return TRUE if A1 contains the date 1/1/2014

Also, if you want to do two tests within an IF function, you have to use the AND function:

=IF(AND(  A1>DATEVALUE("1/1/2014"),  A1<DATEVALUE("31/1/2014")  ),value_if_true, value_if_false)

But note that that will miss out both 1/1/2014 and 31/1/2014.  Always make sure you cover all the possibilities in your range by using a <= or a >= somewhere.  e.g.

=IF(AND(  A1>=DATEVALUE("1/1/2014"),  A1<=DATEVALUE("31/1/2014")  ),value_if_true, value_if_false)
or:
=IF(AND(  A1>=DATEVALUE("1/1/2014"),  A1<DATEVALUE("1/2/2014")  ),value_if_true, value_if_false)

will reture the value_if_true result for all dates in Jan 2014

(that last one is greater than or equal to 1-Jan-2014 AND less than 1-Feb-2014 in dd/mm/yyyy date format)

Cheers
Rich

  2. Make sure you're on the "Home" tab

  8. If you'd like to keep this as your default view when you open this inbox, return to "Arrange By" drop down one last time, and select "Make This the Default View"
  9. Otherwise, click on the Filters button again and select "Clear All Filters" from the dropdown
