andreas killer say:

how to position a chart on a chart sheet using vba

Make a new file and debug this macro, watch into Excel after each line.

Andreas.

Sub Test()
  Dim Sh1 As Shape, Sh2 As Shape
 
  'Create 2 charts
  Range("A1:A4") = WorksheetFunction.Transpose(Array("Nr", 1, 2, 3))
  Set Sh1 = ActiveSheet.Shapes.AddChart(xlColumnClustered)
  Set Sh2 = ActiveSheet.Shapes.AddChart(xlBarClustered)
 
  'One as chart sheet
  Sh1.Chart.Location xlLocationAsNewSheet
  'Note: The active sheet is now the chart sheet
 
  'Move the 2nd also into this place
  Sh2.Chart.Location xlLocationAsObject, ActiveSheet.Name
  'Grab the shape again  (Note: Sh2 becomes invalid!)
  Set Sh2 = ActiveSheet.Shapes(1)
  'Move it to the top left
  Sh2.Top = 0
  Sh2.Left = 0
End Sub

Any ideas on how to fix this?

suzanne s. barnhill say:

Possibly the page number in the Even Page Header is not a PAGE field. Select the existing page number and delete it; then insert a PAGE field using Alt+Shift+P.

Note that "Link to Previous" does not link Odd and Even headers/footers; each header or footer (Odd Page or primary, Even Page, or First Page) is linked to the corresponding header or footer in the previous section. Each header and footer is independent of all the others, so you can have, for example, all the footers linked, but only some of the headers (First Page Headers, for example) linked.
