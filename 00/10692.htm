Hi,

 

Data Execution Prevention (DEP) is a security feature that helps prevent damage from viruses and other security threats. They monitor your programs to make sure they use system memory safely. Data Execution Prevention, or DEP raises an exception if it detects code loading from the default heap or stack. Since this behavior is indicative of malicious code (legitimate code does not generally load in this manner), DEP protects the browser against attacks rendered. For example: Via buffer overflow and similar type vulnerabilities. Internet Explorer Enable DEP is a program developed by Microsoft.

 

You may go ahead and un-install Internet Explorer Enable DEP from your computer. Follow the steps.

a.  Press Windows key + X.

b.  Click on Program and Features.

c.   Under Un-install or change program, select Internet Explorer Enable DEP from the list and click on un-install.

d.  Follow the prompts. A progress bar will show you how long it will take to remove Internet Explorer Enable DEP.

djsteele, mvp say:

Try

Private Sub AppendBtn_Click()
On Error GoTo Err_AppendtoCWIPMasterTblbtn_Click
Dim qdf As DAO.QueryDef
Dim prm As DAO.Parameter
Dim sSQL As String
Dim curdate As Date

  curdate = Now()
  MsgBox "At This Time: " & curdate & " - You are about to Append " & _
    [VAC_SELECT_1] & "-CTRL#: " & [CWIP_POS_CTRL_NUM] & Chr(13) & _
    "to the CWIP Master Faces Table." & Chr(13) & _
    "Org Title Must be Filled In! and CTRL Number can NOT be a Duplicate, i.e., already in CWIP." & Chr(13) & _
    "Query Appends Names, numbers, titles, emp type, status, CIV data, Comments, etc." & Chr(13) & _
    "Please Remember to fill in blank data when possible, Location and POC sections can be updated by Data Stewards."

  sSQL = "qry_Append_Vac-to-tblMasterTEST"
  Set qdf = CurrentDb.QueryDefs(sSQL)
  For Each prm In qdf.Parameters
    prm.Value = Eval(prm.Name)
  Next prm
  qdf.Execute dbFailOnError
  If qdf.RecordsAffected > 0 Then
    MsgBox "Append was successful"
    Me.VAC_STATUSCombo = "ARCHIVED"
  Else
    MsgBox "Something went wrong Records Affected NOT greater than 0"
  End If

Exit_AppendtoCWIPMasterTblbtn_Click:
  Set prm = Nothing
  Set qdf = Nothing   
  Exit Sub

Err_AppendtoCWIPMasterTblbtn_Click:
  MsgBox Err.Description
  Resume Exit_AppendtoCWIPMasterTblbtn_Click

End Sub
