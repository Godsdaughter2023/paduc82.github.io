jllatham say:

The problem is that values in TextBoxes (and combo boxes for that matter) is TEXT not numbers even though they may look like numbers or times.  Convert the value in both to real numbers before doing the math:

 

'declare some variables to work with

Dim myTime As Date
'then do the conversion

myTime = Me.txtTime ' Excel coerces the input into real time!

'now your problem line of code becomes

.Offset(cboDay,9).Value = Val(Me.TxtMiles.Value) / (myTime * 24)
