For A, Parts 1 and 2, I would stick with the three column table use {SEQ } fields for the numbering
 
{ SEQ  English }, { SEQ French } and { SEQ Khmer }
 
With that method, if you add or delete a paragraph, the numbering will update to accommodate the change.
 
 For A, Part 3 Do you want to combine three separate documents into one document set up as a three column table.  If that is the case, I would probably create a macro to do the combining and renumbering.
 
Similarly for the footnote issue, I would use a macro that converted footnotes into "text notes"
 
Here is the code of a macro that does that, but it would need to be modified for your 3 language situation
 

' Macro created 29/09/99 by Doug Robbins to replace endnotes with textnotes at end of document<o:p></o:p>

' to replace the endnote reference in the body of the document with a superscript number.<o:p></o:p>

'<o:p></o:p>

    Dim aendnote As Endnote<o:p></o:p>

    For Each aendnote In ActiveDocument.Endnotes<o:p></o:p>

        ActiveDocument.Range.InsertAfter vbCr & aendnote.Index & vbTab & aendnote.Range<o:p></o:p>

        aendnote.Reference.InsertBefore "a" & aendnote.Index & "a"<o:p></o:p>

    Next aendnote<o:p></o:p>

    For Each aendnote In ActiveDocument.Endnotes<o:p></o:p>

        aendnote.Reference.Delete<o:p></o:p>

    Next aendnote<o:p></o:p>

    Selection.Find.ClearFormatting<o:p></o:p>

    Selection.Find.Replacement.ClearFormatting<o:p></o:p>

    With Selection.Find.Replacement.Font<o:p></o:p>

        .Superscript = True<o:p></o:p>

    End With<o:p></o:p>

    With Selection.Find<o:p></o:p>

        .Text = "(a)([0-9]{1,})(a)"<o:p></o:p>

        .Replacement.Text = "\2"<o:p></o:p>

        .Forward = True<o:p></o:p>

        .Wrap = wdFindContinue<o:p></o:p>

        .Format = True<o:p></o:p>

        .MatchWildcards = True<o:p></o:p>

    End With<o:p></o:p>

    Selection.Find.Execute Replace:=wdReplaceAll

--
Hope this helps.
