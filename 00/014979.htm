to change the weather location for all users, use the cmdlet below:
get-mailbox | set-mailboxcalendarconfiguration -weatherlocations 'locationid:<locationid>;name:<name>;latitude:<latitude>;longitude:<longitude>'

for example, locationid:105808079;name:redmond, wa;latitude:47.679;longitude:-122.132

dilyrosario say:
Hello KrystalParr,

Thank you for posting in the Microsoft Community.


Select the white or blue OneDrive cloud icon in the Windows taskbar notification area.
In the activity center, select More > Settings.

Select the Account tab, and select Choose folders.

Note: You might need to select the Show hidden icons arrow The Show hidden icons button. next to the notification area to see the OneDrive icon.

If you uncheck a folder you are syncing to your computer, the folder will be removed from your computer. The folder and its contents will still be available online.
