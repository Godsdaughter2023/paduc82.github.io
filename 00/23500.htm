For example, if I want to type "empires" (i.e. the word EMPIRES within quotation marks) the keyboard will instead place a diaeresis (i.e. two small dots) over the first letter, for example: ëmpires

The only way to overcome this is to press the " key (which does not type the first quotation mark), then press the space bar (which causes the first quotation mark to then appear), then type the word, then press the " key, then press the space bar (to make the second quotation mark appear), then press the space bar again to type the next word.
