How can I change my footer page numbers??? - Microsoft Community

ericfadely say:

How can I change my footer page numbers???

 It currently shows 1 of 2 on the first page and 1 of 2 on the 2nd page.  When I go to the second page to change it to 2 of 2 it changes the first page to 2 of 2...
pesach shelnitz say:

Hi Eric,

In order to have the page numbering that you want in the center of your footers, you must have the following throughout your document:

1) A section break that starts a new page or a new odd page at the end of each section except the last one. To verify that you have these section breaks, press Shift+Ctrl+8, navigate to the end of each section, and verify that something like one of the following is there:

     ===Section Break (Next Page)===

     ===Section Break(Odd Page)===

If you don't see such a section break, on the Page Layout tab, in the Page Setup group, click Breaks, and then click Next Page or Odd Page.

Also make sure that there are no section breaks in places where you don't want a new section to begin. You can find them by searching for ^b .

When you finish checking this, press Shift+Ctrl+8 again to hide the non-printing characters.

2) The following field codes at the center of your footer in each section.

Page { PAGE } of { SECTIONPAGES }

To check that they are present, double-click within a footer, press Alt+F9, and then on the Design tab under Header & Footer Tools, in the Navigation group, use the Next Section and Previous Section buttons to navigate between sections. If these field codes are missing in any of your sections, you can create a PAGE field by pressing Alt+Shift+P, and you can create a SECTIONPAGES by typing SECTIONPGAES, selecting it, and pressing Ctrl+F9. After you create any field, press F9 while your cursor is still in it before moving on.

When you finish this phase, press Alt+F9 again to hide the field codes and display the numbers.

3) PAGE fields that are configured to start at 1 in each section.

As you navigate through your sections, on the Design tab under Header & Footer Tools, in the Header & Footer group, click Page Number, and then click Format Page Numbers. Verify that Start at is selected and that the number 1 appears In the Start at box.

If you want two numbering schemes working simultaneously in your document, get this to work properly first, and then write back with a more detailed description of what you want.

Hope this helps,
Pesach

The page header is formatted before the page footer, so you must use the PageHeader_Format event procedure for this:

 

Private Sub PageHeader_Format(Cancel As Integer, FormatCount As Integer)
    If Me.Pages > 0 Then
        Me.PageHeader.Visible = (GrpArrayPage(Me.Page) > 1)
    End If
End Sub

 

The code for PageFooter_Format can remain as it was originally.

bernard liengme say:

You asked:

Is it possible for a cell to have a formula and allow for manual input?

The answer is: no

But you could use =IF(OR(A1="i",A3="ii",A3="iii"),B1,H1)

And enter the 'manual' data in H1

Is it possible that you meant to type =IF(OR(A1="i",A1="ii",A1="iii"),B1,H1)

If so, then you can make a neater formula with =IF(OR(A1={"i","ii","iii"}),B1,H1)
