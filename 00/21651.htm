suzanne s. barnhill say:

The latest version I have is Word 2013, but the gallery there is exactly as shown in the article you cite. In order to see the gallery, you must first choose Top of Page or Bottom of Page. Then you must scroll down past the Simple and Page X selections to the Page X of Y selections. They all have bold numbers, but you can unbold them.

If you want to insert the numbers into an existing header or footer, you can select Current Position instead of Top of Page of Bottom of Page (the catch here is that the Top/Bottom of Page selections are actually full headers and will replace whatever header is already in place). You must then place the insertion point where you want the number and choose the desired format (there are fewer selections because they're not in triplicate--left, center, right).

In neither of the above cases do you have to actually enter the required fields, but if you do want to do it manually, the combination you want is Page { PAGE } of { NUMPAGES }. You can insert the PAGE field by pressing Alt+Shift+P; there's no convenient shortcut for the NUMPAGES field (though I suppose you could assign one), so you must press Ctrl+F9 to insert the field braces, then type NUMPAGES between them. When you then press F9, you'll see "Page 1 of 1" (or whatever).

jay freedman say:

Charles is correct that Word can handle quite a large document as a single file. The number of pages isn't the controlling factor, but rather the number of characters and the complexity of the layout. The documented maximum is 32 MB of text, plus an unspecified quantity of graphics. However, documents with lots of graphics, tables, and various other complications do tend to be slow to respond to editing.

There is an alternative to the buggy Master Document feature, so you can maintain each chapter as a separate file if you prefer. (It's best if all the files are based on the same template, to avoid conflict among styles with the same name but different definitions.)

When it's time to bring all the files together into one piece, for creating a table of contents and index and for printing, do this...

  • Create another blank document based on the same template, or use the document containing the frontmatter.
  • Press Enter once or twice to create space for the TOC. Insert a section break (probably a Next Page break or an Odd Page break) at the end.
  • Place the cursor after the section break. On the Insert tab, click the down arrow on the right side of the Object button, and choose "Text from File" in the menu.
  • In the Insert File dialog (which looks much like the Open dialog), select the first chapter's file. Click the down arrow on the right side of the Insert button, and choose "Insert as Link" from the menu. The content of the file will appear at the cursor in the document.
  • Save.
  • Go to the end and insert another section break, insert another chapter, repeat as needed.

Because of the "Insert as Link" command, what is really inserted in the document is an INCLUDETEXT field (you can see this by right-clicking in the inserted text and choosing Toggle Field Code). It contains the path to the included file, such as

{ INCLUDETEXT "D:\\Documents\\Chap1.docx" }

If you edit text in the chapter document, then you'll have to update fields in the "master" document that contains these fields.

You'll also have to set up the headers in the "master" document, because the INCLUDETEXT field doesn't bring headers and footers with it.

Transferring headers and formats from one worksheet to another. - Microsoft Community

nancy kop say:

Transferring headers and formats from one worksheet to another.

Is there a quick way to transfer a custome header/ footer from one worksheet to another?  I have a detailed set-up that I would like to copy to a new worksheet without having to re-type the whole thing each time.
ed ferrero say:

Right-click on the sheet tab, Select Move or Copy, make sure that 'Create a copy' is checked.

That copies the whole sheet including header/footer.

Alternatively, select all the sheets by holding the Shift key down and clicking the first and last sheet tab. Then you can edit the header for all sheets at once.

Third method, does not work in XL 2003, but does in 2010 (and maybe in 2007). Hold the Ctrl key down and select (by clicking on the sheet tab) the sheet with your custom headers and the new sheet. Then edit the header footer (it will show your custom header) and click Ok. The custom header is now copied to the new sheet.
