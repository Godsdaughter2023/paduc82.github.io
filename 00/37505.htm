As we know Windows 10 Auto Updates installs Synaptics driver Version 19.0.12.95, that keeps resetting/defaulting some settings.

Download and install the Synaptics driver "Version: 19.0.12.95 Rev: A Pass: 14" released by HP below, it works...

19.0.12.95 Rev: A Pass: 14    Last Modified  (07/08/2015)
ftp://ftp.hp.com/pub/softpaq/sp72001-72500/sp72120.exe

After installing the driver (no need to uninstall the previous drivers) Reboot, set your settings, reboot and your settings wont reset anymore, but make sure to not let Windows update it again, it will reinstall its own which keeps loosing settings. You could do that manually or through configuring policies as below:

Run "gpedit.msc" --> Computer Configuration --> Administrative Templates --> System --> Device Installation --> Device Installation Restrictions --> (Double Click) Prevent installation of devices that match any of these device IDs or Prevent installation of devices not described by other policy settings --> Select "Enabled" and perform the necessary changes.
Detailed procedure:

https://technet.microsoft.com/en-us/library/cc732727%28v=ws.10%29.aspx?f=255&MSPPError=-2147217396

https://msdn.microsoft.com/en-us/library/bb530324.aspx

p.s. To allow Windows to install/update drivers for this device again, perform the above and set it back to "Not Configured".
