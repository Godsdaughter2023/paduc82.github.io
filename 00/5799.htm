michael bauer (vboffice.net) say:

The master category lists are stored in mailboxes, public folders don't have that. If it's a public folder, Outlook looks for the color in the user's mailbox; if it's a shared mailbox, then Outlook looks in that mailbox for the color.

 

So if you're using a public folder, you must ensure that all users have the same master category list. You can synchronize the master category lists with Category Manager.

karl timmermans (outlook mvp 2012-2018) say:
>>
Recently lost my outlook contacts and calendar. Would I be able to recover them from the carbonite back up and what kind of file would i be looking for.
<<

From the description, sounds like the account was configured as an IMAP account. If that's the case, the contact/calendar data would be stored in an .ost file where the contact/calendar folders would have the added words "This Computer Only" in the folder description.

Am not familiar with Carbonnite in terms of whether it actually backs up .ost files (some backup programs do not).

An .ost file can only be opened/read by the original outlook profiel/email account that created the file. If the .ost file is orphaned, you would need to use a 3rd party OST2PST conversion app to retrieve any data within the ost file.

The following doesn't resolve your current problem but 2 approaches to mitigate this problem from occurring again which are as follows

Options #1

1) Add a new PST file to your Outlook profile (from Home ribbon --> New Items --> More Items --> Outlook Data file)
2) Go to File --> Account Settings --> Data Files tab --> select the newly added PST file --> set it as the default for the profile. This results in the default contact/calendar folders now being stored in the PST file

Option #2

If you want to share your contact/calendar data with other devices

3) Create a new Outlook.com email account (if you don't already have one)
4) Add the Outlook.com account to Outlook
5) Perform Step #2 above selecting the ost file for the Outlook.com account

Whichever approach you select, your default contact/calendar data will be stored in the applicable folders in either the PST file or the OST file for the Outlook.com account. Outlook.com is an Exchange account and all info is sync'd to the server which includes contact/calendar data unlike an IMAP account which only sync's email data

Recode numbers - Excel - Microsoft Community

sriebl say:

Recode numbers - Excel

Hi all,

Does anyone know if it is possible to recode numbers in Excel?

For instance, I would like to recode numbers 1-6 into various numbers like 0.73, 0.98, and 1, and 2 (these last two would be from like 5 recoded to 1 and 6 recoded to 2).

Thank you for any recommendations or suggestions. If I am not clear enough I apologize and I will attempt to explain what I am trying to do better.

Thanks again!
mike middleton say: One way is to use the CHOOSE worksheet function.

For example, if the integer 1 to 6 is in cell A1, enter in B1 =CHOOSE(A1,0.73, 0.73,0.98,0.98,1,2).

A more complete answer to your question depends on how your data are organized and where you want the results.

-  Mike Middleton, www.MikeMiddleton.com

j.e. mcgimpsey say: One way:

In an out-of-the-way location, put your equivalents, e.g.:

       J                   K
1   Original   Recode
2       1            0.73
3       2            0.98
...
7       6              2


Next, assume the number you want to recode is in A1. In B1 enter

   =VLOOKUP(A1, $J$2:$K$7, 2, TRUE)
