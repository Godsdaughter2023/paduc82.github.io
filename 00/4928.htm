It's also a good idea to put Option explicit at the top of the VBA code.  It forces you to declare your variables and so both makes you more disciplined & yields more informative error reports if things do start to go wrong.


So, the start of this would be


Option Explicit

    Sub Jack()
Dim OriginalFile, UsersFile As String


Adding As String after the variables is also useful for the same reasons [it also helps to reduce memory usage but that is not a concern here].  Without declaring what type of variable something is, you leave VBA to make its own decision based on how you first use it.  If you then try to use it as something else, the error report is more likely to guide you to correcting things.


You don't need to repeat Option explicit at the start of every Sub, just put it at the top of each VBA module & that's enough.
