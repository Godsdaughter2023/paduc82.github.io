-> You could create a symbolic link to keep this program happy. Here is how you do it:

  1. Create a desktop shortcut for cmd.exe.
  2. Right-click this shortcut, then left-click "Run as administrator".
  3. Accept the challenge. You will get a black frame.
  4. Type these commands EXACTLY and press Enter after each:
    mklink  /d  "c:\Users\Graham"  "C:\Users\waveb"
    exit
Whenever your program looks at Graham it will be automatically redirected to waveb.

davem121 say:
Hi GTAComputertech

The problem is that Disk is set as MBR which has a Maximum partition size of 2TB, that drive needs to be converted to GPT

Since you do not have any data stored on that drive, this will be easy to do:

Close Disk Management

Click your Start Button, then just type cmd

Right click Command Prompt and choose 'Run as Administrator'

Run this command and hit Enter:

diskpart

Run this command and hit Enter:

list disk
(Make note of the disk number you want to convert and enter it into the next command - replace disknumber)

Run this command and hit Enter:

select disk disknumber

Run this command and hit Enter:

clean

Run this command and hit Enter:

convert gpt

Then close Command Prompt


Open Disk Management, that drive should now be all unallocated space, right click that and choose New - Simple Volume, accept the default size, make sure the file system is NTFS and accept the default Drive letter

That's It . . .
