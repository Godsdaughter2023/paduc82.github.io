c. In the right panel, double click on ‘Code Signing for Device Drivers’.

e. Click Apply, OK. Restart your computer to install unsigned drivers.

bastian weber say:

This time the in-place installation using the media creation tool worked (at least that's what it seems like now). Everything is basically as it was before, and the start menu works again. I guess I was lucky this time.

For all of you out there who are wondering what I did:

  1. Download the media creation tool from here
  2. Grab a USB-Stick that is empty or does only contain data that you need no longer
  3. Run the media creation tool and select that USB stick to make it a bootable windows installation medium
  4. When that is completed, open the USB Stick in Windows Explorer and run the setup.exe
  5. Select that you want to upgrade your system, that updates shall be installed, and that you want to keep all your apps and settings

The rest of the process will be fully automatic. Apparently the critical components of the system will be reinstalled without altering your applications and settings.
