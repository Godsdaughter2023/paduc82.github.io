I am guessing I should use the /XN switch, but the help description isn't obvious, and some of what I've read on the internet seems to imply this would have some other effect.

eryops say:

Okay, I found the answer myself.  And it turns out I misunderstood how RoboCopy worked.

The answer, if anyone else is after the same answer:

By default Robocopy will overwrite any file in the destination folder than differs from the source version (both older and newer).  So using the default settings does what I want.

Using /XN will prevent newer source files overwiting older destination files, which is irrelivent when restoring from a backup where everything should be the same or older than the destination.

arminatorx say:

As for the "best settings", there is no real recommendation, as the picture settings are also dependent on where you set up the TV. If the TV is in a room with windows and a lot of natural light, you need different settings than using the TV in a dark room with closed shades, where you use artificial light, and again different settings if you turn the light completely off just like in an actual cinema. 

This support page from LG gives you some pointers what the picture experts from LG found for different lighting environments to get the best out of your TV.

Additionally, the TV features a Game setting. I highly recommend that setting, as it turns off any further image enhancing features, that introduce "lag" to the action on-screen. 

If you're watching non-interactive content like broadcast TV or watching a movie, the TV can "improve" the image by "making up" in-between images to make movements appear smoother, or suppress compression artefacts that happen when the video is compressed to "fit" a TV-station's broadcast bandwith. To do that, the TV collects some video frames, does some calculations on those images to determine what it "thinks" it could improve, and after a while it will then show these images on-screen. In broadcast TV or movie watching it doesn't really matter if you see a sports goal a few milliseconds later.

However, when you have interactive content like games, especially rhythm games or fast action games, you will always see an image "from the past" if you have those image enhancements on. 

For example, by the time you see the muzzle flash from the gun of an opponent on your screen, for the console, the shot already hit you.

Or if you press a button in a rhythm game when you see the "note" hit the mark, you will always miss, as the console is "further ahead" from what you are seeing on the TV. If you hit the note "perfectly" on screen, it actually is already well past the perfect spot for the console. 

And there are no specific game settings for each individual game. So you shouldn't go "Oh, I'm playing RDR2 now, so I have to set brightness to 65, contrast to 22, and... let's see, color balance to 8015k. Oh, the game needs an update. So, I'll go to Assassins creed, and have to set brightness to 60, contrast 50..."

The Xbox has a built in tool to help you calibrate your TV for the optimum display settings, and you should then only change these settings, when the lighting of your room changes significantly (e.g. you decide to convert your living room with bright windows to an actual home cinema with no natural light from windows).

To access the calibration tool, go to the console settings, and under Display and Sound go to Video output. 

There is a button to calibrate your HDTV. 

Follow the instructions on the screen to set up the TV for the most optimal viewing experience.
