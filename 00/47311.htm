Get rid of the new Quick Access/Home view in File Explorer

Follow these simple steps to bring back the old This PC view in Windows 10 File Explorer:

  • Show frequently used folders in Quick Access

shakthi weerawansa say:

Hello Jamie! Have you checked whether all the speech, region, language, time settings are all set to one of the supported countries? Make sure that all of them are set to the same country/region. Follow these steps:

  • Check these settings in 'Settings' and 'Control Panel'(these two are entirely different)

First change in 'Settings':

  1. Open 'Settings'
  2. Go to 'Time and Language' section
  3. Then open the 'Region and Language' tab
  4. Set the country to your region and set the keyboard to the same region (by clicking add a language)
  5. Go over to the 'Speech' tab and make sure that the speech is set to the same region as the previous setting. If that speech setting is not there, go back to 'Region and Language' tab and click on the language, click 'Options' and download 'Speech'.

Then change in 'Control Panel':

  1. Open 'Control Panel'
  2. Type 'Region' in the search box, open it
  3. Make sure the 'Format' is set to "Match Windows Display Language"
  4. Click the 'Location' tab and make sure the location is set to the same region as you put in 'Settings'
  5. Now click OK
  6. Minimise Control Panel for a bit and change to the appropriate date and time by changing the time settings on the taskbar(right click the clock and click 'Adjust Date and Time')
  7. Save changes(if you made any) and go back to Control Panel
  8. Type 'Language' in the search box and open it
  9. Find the language that is the same as the last settings. If it is not there, click 'Add a language'
  10. Now click on 'Options' for that language
  11. Under 'Windows Display Language' click 'Make this the primary language'
  12. Windows will say 'You will need to log off for changes to take effect
  13. Log off and log on

Now see if Cortana can be turned on

Hope this works for you.

Shakthi
