jay freedman say:
Shortcut keys are not stored as part of the styles, so copying styles from one document or template to another is never going to bring the shortcuts with them.

Similarly, saving a template from the document that does contain the shortcuts, and putting it in the same folder as Normal.dotm, doesn't make the shortcuts available in other documents. That template is just sitting there, doing nothing. If you used the New command to make a new document based on that template, then you would get the shortcuts because the template is associated with the new document.

There are two ways to get the shortcuts on the second computer:
  • Move the template containing the shortcuts to the folder %appdata%\Microsoft\Word\Startup. Any template stored in that folder is automatically loaded when Word starts. The shortcuts (and macros, if there are any) in that template will be available in all documents. Of course, the styles applied by the shortcuts must also exist in the current document, either because they're defined there or because the INCLUDETEXT fields bring them from another document.

OR

  • Re-create the shortcuts in the Normal.dotm template of the second computer. You might find this easier than messing with templates and unfamiliar folders.

mike h.. say:

Hi,

Pasted data from external sources often contains characters you can't see so functions like VLOOKUP don't work reliably or at all. There are workarounds but I find the best way is to clean the data and to do that I use this code. Put the code in a general module and then select the data to clean (column 1) and run the code and then try your VLOOKUP again.

Sub CleanMe()
Dim c As Range
    For Each c In Selection
    If Not c.HasFormula Then
    With c
        .Value = WorksheetFunction.Trim(.Value)
        .Value = Replace(.Value, Chr(160), "")
    End With
    End If
    Next
End Sub

       Group1       Group 2

    A     B     C    D   E     F

pamcaswell say:

AS far as I know, you cannot set up multiple header rows in the table style so that, when applied, the multiple rows are in place.  You can set certain items like inside borders (for when the row header had dark shading and  you want the inside borders to be grey or white, for example).   

To apply the  header row formatting to multiple rows in the table: after the table style is applied,  select the rows you want included in the "header row" and on the Table tools > Layout tab, click Repeat Header Rows.  If header row is ticked in Design tab > Table style options, you will see your  header row formatting.

Pam
