frank blt say: We figured out why the problem occured with our images not being accessible by the Proxy.

We have a cluster of servers running Apache and a single NGINX server running on top of them to distribute the load (and do caching on certain file types). What happened is both our Apache servers and the NGINX servers were serving images with GZIP compression. This means NGINX would receive a GZIP compressed image and run a second GZIP compression on it before serving it. This double GZIP compression apparently created a situation where the Live Proxy couldn't properly read the images.

So, we made sure only the NGINX server did GZIP compression on the images. The images started to load in Outlook as soon as this was done.

Hopefully this might help someone who uses a similar setup.

gd1977 say:

return string to left of character

All-

I am hoping within VBA that I can take a string and return all of the characters to the left of a character. For example, I have data as 12243.334_abcd

I would like to search for the period "." and only return 12243

An issue I am having is the value I want does vary and will not always be a set number of characters

Any help is greatly appreciated!

per jessen say:

No problem:

 

Dim MyString
MyString = "12243.334_abcd"
MyString = Left(MyString, WorksheetFunction.Find(".", MyString) - 1)
MsgBox (MyString)

rick rothstein, mvp - excel say:

An alternative to calling out to the worksheet function FIND is to use VB's built-in InStr function...

Dim MyString
MyString = "1224.3334_abcd"
MyString = Left(MyString, InStr(MyString, ".") - 1)
MsgBox MyString

I would note that, as written, both Per's and the above code will error out if MyString does not contain a dot to find. The simplest fix for this is to concatenate a dot onto the end of MyString inside the InStr (and FIND) function...

Dim MyString
MyString = "1224.3334_abcd"
MyString = Left(MyString, InStr(MyString & ".", ".") - 1)
MsgBox MyString
