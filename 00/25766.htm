john - project say:
ShaheerAziz,
There are various ways to do this. If you simply want a text based checklist, you can use the Notes field to set up your checklist and annotate it as each item is received. However, if you want the checklist items to physically affect the schedule (i.e. must have them in order to proceed), you can set them up as individual milestones and link them as predecessors for the schedule task of interest. So if you have for example 5 items that are needed before the task can start, set up 5 milestones with the expected date of completion. As each document/deliverable is completed, set that milestone to 100%. Tie them all with Finish-to-start links to the task. When all the milestones are complete the task can start.
