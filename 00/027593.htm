roady [mvp] say:

No, there is nothing you can do about this the way you are working with it now. As soon as you connect to a pst-file with Outlook, a file lock is written and several other things as well. While you might not be changing any content, meta data of the pst-file is changed and thus also the modified date. Setting the pst-file's properties to Read Only won't work either since Outlook requires Read/Write access to the pst-file. This has nothing to do with the indexing process either.

As you are working from an archive location (meaning that this data shouldn't ever change), you really shouldn't be opening files directly from that location but work with copies instead. You can change the NTFS permissions of that location to prevent deletions and modification but allow copying new content to it. This way, you are automatically forced to create a copy before you can open it.

To disable on Windows client machines

daniel g. grau say:
You don't need a macro for that. Although it could be done that way, it's a lot easier to use an AutoCorrect sequence:
  • Insert the heart in the middle of some Auto-colored text. It's available through Insert > Symbol > Advanced Symbol > Zapf Dingbats (if you have this font) or through Apple's Character Viewer (just type "heart" in the search field).
  • Select it and set it to red
  • While it's still selected, go to Word > Preferences > AutoCorrect
  • Place a checkmark at the top, next to "Automatically correct..."
  • Select the AutoCorrect tab at the top
  • Enable "Replace text as you type"
  • Click on the "Formatted text" radio button so the red color, font and size will be kept
  • In the "Replace" field, type a unique sequence. For instance, "h\" or "/h/" or "\h\" (if you want to be able to insert it with no spaces around it, like in the middle of a word, use the latter sequence, as the backslashes terminate words).
  • Click on Add and test by typing the sequence. The text should switch back to Auto after the red heart is inserted.
If you still want a macro, post back.
