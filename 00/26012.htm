What should I know about the 30-day waiting period?

Even when you choose to delete your out-of-date security info, MS don't fully remove it for 30 days. This waiting period helps to deter hackers from updating your security info and gaining access to your account without your knowledge.

After 30 days, the user of the account will be able to access the security settings and update his/her preferred information.

But the waiting period doesn't mean you're locked out of your Microsoft account.

Check to see if you can still access your emails via https://mail.live.com.

Dim olExp As Outlook.Explorer
Dim olSel As Outlook.Selection
Set olExp = Application.ActiveExplorer
Set olSel = olExp.Selection
Set olItem = olSel(1)
strForename = ""
'Split the body of the selected message by paragraph
vBody = Split(olItem.Body, vbCr)
'Check each paragraph
For i = 0 To UBound(vBody)
    'If the string "Name:" is at position 2
    If InStr(1, vBody(i), "Name:") = 2 Then
        'split the paragraph by spaces
        vName = Split(vBody(i), " ")
        'assign the first word of that paragraph to a string
        strForename = vName(1)
        'then stop looking
        Exit For
    End If
Next i
'if the string is not found warn the user and quit
If strForename = "" Then
    MsgBox "Message does not fit criteria", vbCritical
    GoTo Cleanup:
End If
'the string was found so repeat the search for the e-mail address
(though if the address if the same as the sender's address you could use that instead.)
For i = 0 To UBound(vBody)
    If InStr(1, vBody(i), "Email:") = 2 Then
        vName = Split(vBody(i), " ")
        strEmail = vName(1)
        Exit For
    End If
Next i
Create a new message using the information obtained
Set olNewItem = Application.CreateItem(0)
With olNewItem
    .BodyFormat = olFormatHTML
    .To = strEmail
    .Subject = "RE: " & olItem.Subject
    .Body = "Dear " & strForename & vbCr & _
    "Thank you for submitting:" & vbCr & vbCr & _
    olItem.Body
    .Display
End With
Cleanup:
Set olItem = Nothing
Set olNewItem = Nothing
Set olExp = Nothing
Set olSel = Nothing
End Sub
