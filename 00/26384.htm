How do I apply stationery to a word document - Microsoft Community

susantrusina say:

How do I apply stationery to a word document

How do I apply "My Stationery" to a word document
jay freedman say:

According to http://www.sevenforums.com/general-discussion/102578-my-stationary-folder.html, that folder is created by Windows Live Mail. It doesn't have anything to do with Word.

Depending on what kind of files are in that folder, you may be able to use them as page backgrounds in Word. Try this to see whether it works:

  • Click the Page Layout tab on the ribbon.
  • In the middle of that tab, click the Page Color button. In the dropdown menu, click Fill Effects.
  • In the Fill Effects dialog box that opens, click the Picture tab.
  • Click the Select Picture button.
  • In the Select Picture dialog that opens, navigate to the My Stationery folder. The dropdown to the right of the File Name box should say "All Pictures" followed by a list of the file extensions that Word recognizes. If any files are listed in the dialog, try selecting one and clicking the Insert button.
  • You're returned to the Fill Effects dialog box. Click the OK button.

If you were able to select a stationery file and complete the procedure, the image will appear as the background of the page. Anything you type will appear in front of it. Depending on the type, size, and aspect ratio of the original picture, the image may be "tiled" (more copies of it added to fill out the page).

There are other adjustments you may have to make: Changing the text margins to avoid part of the background such as a left-side dark image; changing the text font color so it doesn't disappear into the background; etc. If you want to print the document on paper or as a PDF file, you need to go to the Options dialog's Display page and check the box for "Print background colors and images".

If the background is tiled and that's unacceptable, an alternative is to open the header, insert a rectangle shape there, drag its edges to cover the whole page, and then select the stationery as the rectangle's fill. Again, this will depend on whether the stationery file is in a format that Word can handle.
