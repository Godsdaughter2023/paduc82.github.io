hounddog999 say:

 I resolved this problem.

In Windows' Device Manager (devmgmt.msc), the SXUPTP driver was showing as disabled. Windows gave a warning that the device was known to have stability issues. I found out that SXUPTP is a driver from a company called Silex. Silex makes network USB hubs, so I assume that the Belkin hub is a rebranded Silex hub.

In order to update the Silex driver follow these steps:
1) Close the Belkin network hub's tray application.
2) Download the latest version of SX Virtual Link (sx-virtual_link_win_3120.zip).
3) Extract all of the files.
4) In Windows Explorer, navigate to sx-virtual_link_win_3120\s-33a3120\Files\Bin\Sxcsapi\WinVistaX64 if you are running a 64 bit OS, or sx-virtual_link_win_3120\s-33a3120\Files\Bin\Sxcsapi\WinVista if you are running a 32 bit OS.
5) Right-click on "Sxuptp.inf" and select "Install" from the menu.

The driver will be installed and the warning icon in the device manager should disappear. You may need to reboot for the changes to take effect.

Hope this helps.
