a. user say:

Hi,

 

1. Have you checked the free space on your computer and on the destination media?

2. Do you have any third party antivirus software installed on the system?

 

This error occurs if there is no enough disk space in the storage location where you are trying to create the system image backup or you aretrying to backup to a FAT32 drive. FAT32 drives have a file size limit of 4GB, and your backup is trying to exceed that limit.

Let’s try the following steps and check if it helps.

Method 1:

Check the disk usage in disk management

According to the error message, it indicated that there is not enough disk space to do a system image backup on the hard drives. So, I would first suggest you to check the disk usage in disk management carefully and make sure all the partitions have enough space to do system backup image.

To open the Disk Management, you can follow these steps:

a. Go to the desktop Interface and right click on “Computer” and select “Manage”.

b. In the “Computer Management” window, expand “Storage” and click on “Disk Management”.

Here, you can see the usage of the whole hard drive. Please check it carefully and make sure all partitions have enough space.

Notice: If you have some hidden partition on the computer, please also make sure the hidden partition has the enough rest space to do the system image on the computer.

 

Method 2:

If there is enough space on the hard disk, then you may try the next method to check if there are any errors with the disk (both on C: drive and the external drive you are using).

Method 2: Check the disk where you are trying to backup or create the system image and make sure that the disk is error free.

To check the disk for any errors, follow the steps provided “To Repair a Drive” from the article:

Improve performance by optimizing your hard drive

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost.

For further information, refer to the article:

How to use File History

 

Method 3:

NTFS is the preferred file system for this version ofWindows. Make sure the hard drives are in NTFS format.

If you need to format it to NTFS format, then try to format it to NTFS format and check if it helps.

Note: Make sure you have backed up the data in the drive before formatting to NTFS.

a. Press Windows Key + C to open Windows Charms Bar. Then Go toSearch and type Control Panel in the Search Bar and press Enter. The Control Panel opens.

b. Click on System and Security.

c. In the System and Security tab click on Administrative Tools.

d. Double click on Computer Management.

e. Click on Disk Management and then select the partition you want toformat. Right click, select Format and select NTFS.

f. Click “OK” to proceed formatting.

 

Hope the information helps. Let us know if you need further assistance with Windows related issues. We will be happy to help.
