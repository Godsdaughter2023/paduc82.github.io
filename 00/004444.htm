    If you use the first option, you can use Power Query (Data ribbon > Get and Transform group > New Query > From File > From Folder) in Excel to list all the files in a folder, extract the version number and the person from the file name (if it follows a regular pattern) and then save that data in a table in the workbook.     jasles say:

    On the Separate Tab worksheet, use this formula:

    =SUMIF('Current Tab'!A:A,A2,'Current Tab'!B:B)

    Change the sheet names and cells in the formula above to match your spreadsheet. The A2 above is looking in cell A2 for what to seach for. You could also type the name "Adam", but A2 allows more flexibility.
