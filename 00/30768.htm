If you have many files the easiest way to do this is to use a duplicate file/folder finder utility to find and delete the duplicates, There are only a handful of good utilities that will find duplicate folders and none are free, there are a bit more duplicate file finders available, some of them freeware.

http://www.freeware-guide.com/dir/util/cleanup/DuplicateFileFinder.html

Duplicate Cleaner Pro by Digital Volcano Software - it works well and has an easy to use modern interface.  Note that only the Pro version of the software will find duplicate folders.

Directory Report by AMB Software - Don't let the old ancient looking interface fool you, this is one powerful completely up to date enterprise class utility!  It's a bit more difficult to use and get the hang of than my other recommendation but it is faster and can work on hundreds of thousands of files across networks, I settled on Directory Report rather than Duplicate Cleaner Pro but by all means Duplicate Cleaner Pro is also nice.
