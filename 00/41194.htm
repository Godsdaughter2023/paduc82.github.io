lemp say:

I assume that you were attempting to download from this Microsoft site: https://www.microsoft.com/en-us/software-download/windows7

It appears that the DVD you purchased was an "OEM" product, most likely a "System Builder" version. 

The Microsoft download site does not support product keys for OEM versions.  In other words, it's not that your key is "invalid," it's that it's "for a product not supported by this site."

It seems to me that this used to be explicitly explained on the website.  These days, that fact is buried in the FAQ:  https://www.microsoft.com/en-us/software-download/faq

My Windows 7 product key won't verify. What's the problem?

The most common issue is the use of a product key for a product not currently supported by the site such as an Upgrade key, an MSDN key, product keys for pre-installed media or an Enterprise edition key. For access to MSDN products or Enterprise edition visit the MSDN Portal or the Volume Licensing Service Center. Upgrades and pre-installed media are not currently supported by the tools on the site.

I understand that your situation isn't "pre-installed media," but in Microsoft's view, all OEM products are "pre-installed."  That is, at least for Windows 7 (Microsoft repeatedly has changed its policy on this topic), Microsoft's stated policy is that OEM System Builder versions of Windows 7 are not to be used by end users for their own computers:

Use of this OEM System Builder Channel software is subject to the terms of the Microsoft OEM System Builder License.This software is intended for pre-installation on a new personal computer for resale.This OEM System Builder Channel software requires the assembler to provide end user support for the Windows software and cannot be transferred to another computer once it is installed. To acquire Windows software with support provided by Microsoft please see our full package "Retail" product offerings.

See, e.g., http://winsupersite.com/windows-7/no-oem-you-windows-7-oem-packaging-not-individuals
