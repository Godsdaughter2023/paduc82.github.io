Power on your computer and power it off as soon as you see the Windows logo. Do it three times.

mike h.. say:

Hi,

 

Try this code, change the sheet names to suit

 

Sub FindComments()
Dim c As Comment, x As Long
Dim src As Worksheet
Dim dst As Worksheet
Set src = Sheets("Sheet1") 'Change to suit
Set dst = Sheets("Sheet2") ' change to suit
x = 1
If src.Comments.Count > 0 Then
    For Each c In src.Comments
        dst.Cells(x, 1) = c.Parent.Value
        dst.Cells(x, 2) = c.Text
     x = x + 1
    Next
End If
End Sub
