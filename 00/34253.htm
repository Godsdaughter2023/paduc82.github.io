tryx3 say:

Vee,

The Password reset disk only has any meaning for local user accounts i.e. user accounts that are not linked to online accounts or, in other words, user accounts that only exist on the computer [just like user accounts used to be in Windows XP or, for the most part, in Windows 7].

  • You are using an MSAccount-linked user account.  That allows you to use various online facilities directly such as email.  I imagine that it allows you to come straight into this forum as well.  It also allows you to use OneDrive for online documents & to set up syncing between Windows 10 devices.
  • I am using a local user account so to use online MS facilities I have to go to, for example, the live.com webpage, this forum or OneDrive then log in by entering my email address & my password.  I cannot use the syncing capabilities of Windows 10 at all.

Windows was not designed to allow MSAccount-linked user accounts to make a Password reset disk because they are not relevant but the way it has been implemented does allow you to find the tool & to try making one, albeit unsuccessfully - the fact that you have managed to find the tool is a bug in Windows 10's design.

So, just to make this completely clear - it is not that there is "no need of password reset disk for it, IMO" but that a Password reset disk cannot be made for, and will not achieve anything for, an MSAccount-linked user account. 

  • For MSAccount-linked user accounts you go to When you can't sign in to your Microsoft account and use its link to Recover your account
  • I can make a Password reset disk if I want to but there is little point as it achieves no more than writing my password down achieves.
