  2. In the Manage features section, click Caller ID.
  3. Select your country from the drop-down list.
  4. Enter your mobile number in the text box and click Verify number.
  5. You will receive an SMS text message on your phone containing a confirmation code. Enter this code in the field displayed.
  6.  Click Verify.
  7. If you entered the code correctly, you will see a confirmation message, and your Caller ID will be activated in 24 hours.

Create a "spot" color in Word - Microsoft Community

al the printer say:

Create a "spot" color in Word

How do I create a "spot" color in Word so that a print provider can separate the colors for his printing plates?
bob jones aka: cybertaz say:
I'm afraid I don't understand your question :-} AFAIK, Spot Colors are premixed inks used in order to (a) insure consistency of the color throughout the job and/or (b) avoid the higher cost of CMYK process printing (4 color) if fewer than 4 colors are needed in the job.

However, Word is not equipped for color separation to begin with, so there is no way to distinguish between 'spot' & 'process' color. This is capability you'd find in desktop publishing & graphic design software. Although it's possible to determine the RGB, HSB or CMYK values of a color used in the document the color is being displayed as RGB only. Also, color values of a color to be used in the document can be specified in the Color picker, but I can't attest to the accuracy it provides. What prints may be anywhere from slightly to extremely different.

Without knowing any more about the extent of color use in the document I can't be sure, but if it's just a matter of a single color it shouldn't matter what actual color is applied in the document. The print color choice can be made based on swatches from the provider as to what color ink should be used anywhere the color in the document is other than black.

Have you consulted with the print provider on what is actually required?

Regards,
Bob J.
