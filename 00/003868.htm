rich007a say:

You can do this using a simple script file.

If you right-click on a file in Windows Explorer, you should see things like Open, Edit, Preview, Print, etc.  This script will automate that for you on all files in a folder.  Try the following:

1) Open Notepad.

2) Paste in the following:

TargetFolder = "C:\temp"
Set objShell = CreateObject("Shell.Application")
Set objFolder = objShell.Namespace(TargetFolder)
Set colItems = objFolder.Items
For Each objItem in colItems
    objItem.InvokeVerbEx("Print")
Next

 

3) Change the target folder path in the first line to a folder containing the files you want to print.  Start with a folder containing just a few files and work up from there.

4) Save the notepad file with a .vbs extension, e.g. PrintMyFiles.vbs. 

5) Close notepad

5) Double-click on your .vbs file.  The script should run and providing there are no strange file types in the folder you specified, and you have a printer connected (as the default printer in windows), all the files in the specified folder should print.

Here's where I found the script: http://blogs.technet.com/b/heyscriptingguy/archive/2004/10/04/how-can-i-print-text-files-using-a-script.aspx

Hope that helps.
