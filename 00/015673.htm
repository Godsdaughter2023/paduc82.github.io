john w. vinson say:

The Format is irrelevant: it just controls how the Double Float number in the Date/Time field is displayed. To get the elapsed time in hours, you can create a Query and in a vacant Field cell type:

TimeSinceReferral: DateDiff("h", [Referral], [Appointment])

If your fields are actually named [Referral Date/Time] etc., I'd suggest changing the fieldnames: non-alphanumeric characters such as / can be misinterpreted (as a division operation), and even blanks can cause problems.

Note that DateDiff returns whole numbers of hours, counting the number of hour boundaries. For example, DateDiff("h", #10:00 am#, #11:59:59 am#) and DateDiff("h",#10:59:59 am#, #11:00:00 am#) both return 1 hour, although the actual duration is almost two hours for the first and only one second in the other. If you want fractional hours, use "n" (for miNutes, "m" is Months) and divide by 60.
