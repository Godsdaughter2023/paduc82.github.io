Getting error message: A Newer Version of OneDrive is Installed

Applies to: OneDrive personal on Windows 10

Technical Level: Basic

Introduction

Windows 10 has its own version of OneDrive which is built into Windows 10. This version can not be uninstalled and is updated via Windows Update. However, some people get the error message "A Newer version of OneDrive is installed" when they are using another OneDrive App on their computer. So this Article will talk about how to resolve that error.

Details

If you are getting this error, then the first thing you will need to do is uninstall the OneDrive App you have downloaded. You can do this through the Control Panel program that is built into Windows 10. To do this, please:

  1. Click into the search bar on your Taskbar,
  2. Search "Control Panel",
  3. Select the Control panel program from the list of results,
  4. Select Programs,
  5. Select uninstall programs,
  6. Select OneDrive,
  7. and uninstall.

You will need to wait for OneDrive to uninstall from the computer. Then you can search OneDrive in Cortana/search bar and click onto the OneDrive App. This will then load up the OneDrive setup which you will need to sign into your Microsoft Account and choose what files you want to sync. 

After you have done this, you will notice the OneDrive Icon on your Taskbar and your files will be starting to sync to the cloud. 

For more information on how to get started with OneDrive on Windows 10, and how to use it with Microsoft Office, please follow the link underneath:

  • What is OneDrive?
