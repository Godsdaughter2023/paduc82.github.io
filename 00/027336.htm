Refer below image:

Enter below formula in cell H2 & copy to the right till column M and down - this returns a value for each procedure & if procedure not listed or blank then returns "NA":

=IFERROR(CHOOSE(MATCH(B2,{"Intensive Care Medicine","Geriatric Medicine","Acute Internal Medicine","Emergency Medicine","Gastroenterology","General Psychiatry"},0),1,4,1,1,4,3),"NA")

Formula in cell N2, to be copied down - this sums all values:

=SUM($H2:$M2)

In case you only want the SUM of values assigned to all procedures WITHOUT adding each specific value as above, enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell O2 & copy down:

=SUM(IFERROR(CHOOSE(MATCH($B2:$G2,{"Intensive Care Medicine","Geriatric Medicine","Acute Internal Medicine","Emergency Medicine","Gastroenterology","General Psychiatry"},0),1,4,1,1,4,3),0))

This formula is independent of any values calculated in columns H to M.

You can't do that. When you use "Include chapter number" in the Caption Numbering dialog, you have to select one specific heading style for all items using that caption (Figure, Table, etc.).
