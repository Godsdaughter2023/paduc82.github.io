jay freedman say:
In Word 2013, create a template (or multiple templates for different kinds of documents) and save it with the .dotx extension. On the Review tab of the ribbon, click the bottom half of the Track Changes button and click Lock Tracking. (This command doesn't exist in earlier versions.) Enter and re-enter the password, and save the template.

Users should then use the template(s) to create new documents. Those documents will automatically have Track Changes turned on, and it can't be turned off except by clicking Lock Tracking again and entering the password. No macros are needed.

You must use Word 2013 to turn on the Track Changes and lock it on, and the Admin must use Word 2013 to unlock it, but users with Word 2007 or 2010 will be able to use the template and documents based on it.

Hi Dawn,

In VBA, you can set a comment's .Done property to True. You can also add a reply in VBA with the comment's .Replies.Add method. For example, assuming there is already at least one comment in the document, this adds a reply to the first one and then marks the thread as done:

Sub x()
    With ActiveDocument.Comments(1)
        .Replies.Add Range:=.Range, Text:="Second comment"
        .Done = True
    End With
End Sub

julie isaac say:

POP3 account in Outlook 2016 has no contacts or notes

I have a POP3 account in Outlook 2016 in which I can only see email.  If I click on Contacts, Notes, Tasks the only items are from another account.  There is no toggle on the left side to switch between accounts.  

To troubleshoot I added a connection to an old pst file too see if it had something to do with POP3 (versus exchange).  My other POP3 is listed when I click into Contacts, Notes, and Tasks.  

How do I get these features enabled for that particular account?  I want it to have separate Contacts, Notes, and Tasks from my other accounts.  THANKS for any answers!!

karl timmermans [outlook mvp 2012-2018] say:

How do I get these features enabled for that particular account?  I want it to have separate Contacts, Notes, and Tasks from my other accounts.  THANKS for any answers!!

Based on the description - going to guess you're referring to the following behaviour

#1 - You configured your first POP account which had the contact/calendar folders. That would also have set the PST file for that account to be the "default data file" for the profile which contains the default contact/calendar folders. (There can only be one "default" contact & calendar folder".

#2 - You configured subsequent POP3 accounts where the contact/calendar folders didn't appear in the folder tree - normal behaviour.

If you want contact/calendar folders in each of your POP3 PST files go to 

File --> Account Settings -->Data Files tab

set each PST file as the default once - exit and return to repeat the process. The contact/calendar folders will now appear in each PST folder tree. Just remember which one of those is designated as the "default" for the profile for reason mentioned above.
