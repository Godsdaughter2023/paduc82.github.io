cookymonzta say:

UPDATE:  SOLVED!  I just found out why my floppy disappears whenever I reboot with a USB drive plugged in:

 

In my UEFI, there is a category called BOOT OPTION PRIORITIES, and on that screen it gives you the option of setting the boot order, or the order in which the machine searches for the OS or any other boot program.

 

For the better part of the year, I've had my UEFI boot order set this way:

 

1.  Primary Hard Drive (where Win7 is stored)

2.  Blu-ray Burner

3.  Floppy

 

The reason my floppy started to disappear whenever I rebooted with a USB device plugged in was because I changed the boot order and had no idea what it would do or had done to my machine...

 

...In other words:  THE FLOPPY MUST ALWAYS BE NUMBER ONE IN PRIORITY!

 

1.  Floppy

2.  Primary Hard Drive

3.  Blu-ray Burner

 

I tried this boot order and rebooted with a powered USB device and a USB flash drive plugged in.  The floppy responds and Win7 recognizes it.

 

If the floppy is first in the boot option, Win7 will activate it.  If not, and a USB drive is plugged in, the USB drive will supersede the floppy every time and the floppy will be kicked out of Win7.  Unlike WinXP, Win7 always expects the boot devices to be in alphabetical order, and if a USB device is plugged in, and device A is anywhere except first in the boot order, it will bypass A and read devices C, D and so on.
