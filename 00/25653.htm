grschinon say: It's the "made in Finland" phones that you want to be wary of in the Middle-East.

There is absolutely no difference whatsoever between a genuine Nokia handset made in Finland and one made in China. However, the Middle-East is notorious for some unscrupulous retailers ordering normal stock from Nokia that happens to be made in China, printing out fake "made in Finland" labels, sometimes with the product code and IMEI number of a totally unrelated model, and selling these "improved" phones at a premium price, thus perpetuating the myth that phones made in Finland are better than those made in China.

Just buy the phone from regular channels. Even if it's made in China you have nothing to worry about. You should worry about a retailer trying to sell you a "made in Finland" phone at a premium price because the chances are that the product identification label in its battery compartment is fake and that you therefore have no warranty on the phone.

faysal farooqui say:

Assume that, A1:A3 is the data range where A1=142.5, B1 = 99.8 and C1 = 171.0. Now use the below formula in A3 and copy right to the end of C3.

=IF(ROUND(A1,-1)>A1,ROUND(A1,-1),ROUND(A1,-1)+2.5)

hansv mvp say:

With a number such as 105 in A1, you can use the formula

=MROUND(95%*A1,2.5)

This formula will round to the nearest multiple of 2.5. It can be filled down. If you prefer to round down, use

=FLOOR(95%*A1,2.5)

and if you prefer to round up:

=CEILING(95%*A1,2.5)

amit tandon say:

MROUND Function Returns a number rounded to the desired multiple.

Alternates:

You may also use below - CEILING Function Returns number rounded up, away from zero, to the nearest multiple of significance:

=CEILING(A2*0.95,2.5)

or below - FLOOR Function Rounds number down, toward zero, to the nearest multiple of significance:

=FLOOR(A2*0.95,2.5)

don cuthbert say:

Well the simple part is to use Disk Management to shrink drive C: by as much as you wish to dedicate to the second OS. Then simply create bootable Windows install media for it, boot the PC and choose Custom install during the Windows setup process. There you can choose the unpartitioned space (that you created by shrinking drive C:) to install Windows to for the dual-boot setup. Windows setup will proceed with as many restarts as is necessary until Windows has been installed again. This second install will become the default OS. The default can be easily changed under the Advanced tab of System Properties by choosing the  Startup and Recovery Settings button.

Use a Command Prompt (Admin) to rename the current OS as you wish, so that you can tell the installs apart:

bcdedit /set {current} description "Insider"

Note: When a new Insider build is released, ensure that the Insider OS is set as default during the upgrade. This will ensure that the mid-upgrade restarts launch the correct OS during the upgrade.

don cuthbert say:

It is possible to install Windows to run right off of a USB stick (sometimes performance is not adequate) but what I am talking about is to install the second build of Windows 10 to a separate hard disk partition (or a separate internal hard disk).  Because you cannot "upgrade" to a lower build number, you would need to clean install the public build.
