amit tandon say:

Rajesh is correct. You may use SUMIFS likewise:

=SUM(SUMIFS(D2:D18,A2:A18,DATE(2017,1,1),B2:B18,{"A","G"},C2:C18,{"Example 1";"Example 2"}))

Explanation:

1) One of the array constants should be a single-column array, while the other should be a single-row array. 

2) Ensure to have comma in one of the array constants while semi-colon in the other array constants. This has the effect to coerce Excel into a "two-dimensional" set of returns, so as to count all possible combinations for these two sets of criteria.
