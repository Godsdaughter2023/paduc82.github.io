karl donaubauer say:

J-Lew wrote:

Microsoft 2007

One of my Access fields is called picture. The picture field is a
hyperlink to an image. I would like to have the picture display AS A
PICTURE in an Access 2007 report. No success :(

In the table you should replace your hyperlink field by a text field
with the path (incl. file name) of the picture. Then you can use
a picture control in the report and set its property "Control source"
to the name of the text field.

If you can't replace the hyperlink field you could try to extract the path
and file name in a query by using a calculated field:

HyperlinkPart([YourHyperlinkField], 5)

If this shows the correct path then use the query as record source
of the report and bind the picture control to the calculated query
field as described above.
