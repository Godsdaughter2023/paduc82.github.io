earond say:

Windows Defender runs Quick Scan every 30 minutes

Build 10074. Windows Defender runs a quick scan every 25 to 30 minutes. The system is noticeably slowed down. Is there any way to change this schedule?

a. user say:

 

Hi,

 

Welcome to Microsoft Community. We appreciate your interest in Windows 10 Technical Preview.

 

I can understand how frustrating it is when things do not work in the usual manner. I believe that the scan schedule in not set up correctly.

 

The Windows Defender scheduled scan will run in the background of Windows 8 at the time you set it to run. When it is running, you will notice the Action Center icon on the taskbar notification area will display the black maintenance icon under it like below. You can click/tap on the Action Center icon, and click/tap on Scheduled Scan link to open Windows Defender to see what it's doing if you like.

 

Let us schedule the scanner as per our needs.

 

1. Press the Windows + R keys to open the Run dialog, type taskschd.msc, and press Enter.

2. If prompted by UAC, then click/tap on Yes.

3. In the left pane of Task Scheduler, double click/tap on Task Scheduler Library, Microsoft, and Windows folders to expand them, then open the Windows Defender folder.

 

4. In the middle pane of Windows Defender, double click/tap on the Windows Defender Scheduled Scan task.

 

5. Click/tap on the Triggers tab, and click/tap on the New button.

 

6. While using On a schedule, select and set when (ex: One Time, Daily, Weekly, or Monthly) and how often you would like the scheduled task to run. Make sure Enabled is checked, and click/tap on OK.

 

7. Click/tap on the Actions tab, and click/tap on the Edit button.

 

8. Under Program/script, make sure that the default path is wrapped in quotes like below. It's not by default.

 

9. To the left of Add arguments (optional), copy and paste the argument in the table below to have Windows Defender run a quick or full scan when scheduled, then click/tap on OK.

 

10. Click/tap on the Conditions tab, and set the Power options for if you want this task to be able to run on AC power, batter power, and able to wake the computer to run, then click/tap on OK.

 

11. After a brief pause, you will now notice the Triggers and Next Run Time you set at step 5 and 6 appear for the Windows Defender Scheduled Scan task.

 

12. You can now close Task Scheduler if you like. Windows Defender will now scan on a schedule you set.

 

Hope this information is helpful. Let us know the results.

How can I create a template for labels that are pre-printed with our - Microsoft Community

pattyf73 say:

How can I create a template for labels that are pre-printed with our logo at the top?

Our logo and address, etc. are printed at the top of our shipping labels (Avery 5164).  How can I set it up so that when I type the name and address on it, the text appears in the middle of the label. When I change the pitch of course the whole label moves down, and when I changed the margins the same thing happened, if I remember correctly.

I used the ruler to move the text over, but can't use it to move it down (it moves the whole label) - and I'd have to do it for every label I do.

Advice is appreciated.  Thanks.
suzanne s. barnhill say:

To add to what Charles has said, the easiest way to do this is to change the top margin of the cells. Don't try to use the built-in label feature except the first time, to create your template.

  1. In the Labels dialog, click New Document. This will create a sheet of the labels.
  2. Make sure you have table gridlines displayed so you can see what you're doing. When the insertion point is in the table, you should see the contextual Table Tools tab. On the Layout tab of Table Tools, click the button for View Gridlines.
  3. Right-click anywhere in the table and choose Table Properties from the menu (or click the Properties button next to View Gridlines).
  4. On the Table tab of the Table Properties dialog, click Options...
  5. In the Table Options dialog, change the top margin to accommodate your preprinted return address. This may require some trial and error; you can type in some trial text, print the sheet on plain paper, and compare it to your labels. Repeat as needed.
  6. When you have the top margin set as required, click on the table handle to select the table and press Delete (not Backspace, which will delete the table) to clear all the text you've typed in the table.
  7. Save the table as a template. In the Save As dialog, choose "Word Template (*.dotx)" as the file type. If you navigate to the folder where your other templates (such as Normal.dotm) are stored, then your template will appear on the Available Templates tab when you use File | New. When you select it, you will be creating a new document based on the template (a new sheet of labels), and the template itself will remain unaffected.

lars-Åke aspelin say:

If by "total repeatability between test-1 and test-2" you mean the "test-restest repeatabilty" as defined in this paper

http://www.eri.harvard.edu/faculty/peli/lab/woods/publications/VSIA95b.pdf

i.e. 1.96 times the standard deviation of the difference in scores at test and retest, and your test data are in cells A2:A12 and B2:B12 respectively, try this formula:

=1.96*STDEV(A2:A12-B2:B12)

Note: This is an array formula that has to be confirmed with CTRL+SHIFT+ENTER rather than just ENTER.

Hope this helps / Lars-Åke
