a. user say:

Hi Pavan,

If you want to install Windows 7 on a preinstalled Windows 8.1 computer, you can create a new partition drive specifically for Windows 7 and then install it on that drive as a dual boot.

Considerations before you start:

The changes that are required to be done before going for the installation involves changes in BIOS and drive partitions. You can contact HP at this point of time to confirm about the successful dual boot setup.  

Disclaimer: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

a) Create a USB recovery drive or collect a recovery disk from HP (Recovery drive might already there as it Windows 8.1 preinstalled):

http://windows.microsoft.com/en-CA/windows-8/create-usb-recovery-drive

b) Turn off Secure boot temporarily:

To know if secure boot is set or enabled, follow the steps:

  • Type Powershell on the start screen and right click on Windows PowerShell. Select Run as administrator.
  • Run the following command:

Confirm-SecureBootUEFI

Refer to the link to get the details on Confirm-SecureBootUEFi:

http://technet.microsoft.com/en-us/library/jj603041.aspx

c) Refer to the HP website and make sure that there are drivers for Windows 7 for the model that are using at the moment.

d) Disable Fast startup:

How to disable ‘Fast Boot’:

Make the following changes to the Windows 8 Power Options.

  • Control Panel --> View by: Small Icons -->Power Options
  • Select "Choose what the power button does" on left hand side.
  • Click on "Change settings that are currently unavailable".
  • Uncheck "Turn on fast startup" under shutdown settings.
  • Click on the ‘Save changes’ button.

Now, to start installing Windows 7 in a different partition, create a drive partition under Disk Management and then name it as Windows 7. Then, we will use Windows 7 installation disk to install the operation system. Follow the steps:

  • Press Windows+X keys and select Disk Management.
  • Change the ‘Volume name’ to “Windows 8.1 Pro” (Without the quotes): Right click on the Windows 8.1 partition drive and select ‘Properties’. In the ‘General’ tab change the existing name to: Windows 8.1 Pro and click the ‘Apply’ button.
  • Shrink the Windows 8 partition to create unallocated space for use when installing Windows 7: Right click on the Windows 8 partition and select the ‘Shrink Volume’ option. Now enter a value for the amount of space to shrink in MB. If your Windows 8 partition contains a lot of data you want to consider a smaller partition size for Windows 7.
  • Click the ‘Shrink’ button to start the process. After the shrink operation has completed you will see ‘X’ GB in “Unallocated” space.
  • Create a “Primary” partition and format it: Right click on the unallocated space and select ‘New Simple Volume’.
  • In the “Format Partition” window,  you want an ‘NTFS’ partition, leave the allocation unit size set to ‘Default’ and either choose a ‘Quick’ or ‘Full’ format. I prefer a full format (takes longer to complete, but it’s safer). The ‘Perform a quick format’ option is unchecked. Click ‘Next’ to continue.
  • Change the volume name to Windows 7 (or Windows 7 Pro or Windows 7 depending on the version that you are installing). This will make it easy for you to select the proper location on the drive during the Windows 7 installation process.

Then, insert the Windows 7 disk and restart the computer. If you are trying to install Windows 32 bit version, you would need to navigate to BIOS screen and change the mode from UEFI to LEGACY. If it’s Windows 7 62 bit, it would automatically take up UEFI mode. Also, you would need to make sure that the primary boot drive is changed to DVD in place of hard drive. Refer to the link at HP website to do so.

Disclaimer: Modifying BIOS/ complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.

Restart the computer. Windows will start booting from the DVD. When you see the prompt to ‘Press any key” to boot from the DVD, do so. Windows 7 will start to boot up and you will see the following progress bar “Windows Is Loading Files” followed by the “Starting Windows” logo. Refer to the link:

Refer to the section “Using the Custom installation option and formatting the hard disk”

http://windows.microsoft.com/en-ca/windows/installing-reinstalling-windows#1TC=windows-7

Please let us know if you need more assistance on this issue.

ethan b. say:
Hey Aric,

This is a recovery partition brought into view in 1803. You cannot remove, but only you must remove its mapping of its respective drive letter, which is D. Once you do this, it brings it back to its intended visibility, being hidden and out of sight. Once you do this, you won't be pestered with low disk space notifications anymore. You will need to remove the drive letter and mount point in DiskPart in order to do this.

Hit Windows key + R together to go into Run. Type in the following: diskpart

Enter the following in the new DiskPart window:

select volume D
remove letter=D

Does not work after a reboot? Try entering this instead in an administrator session of Command Prompt: mountvol D: /D

john ado say:

Hi,

Do you have important files on your device? Removing the password on your device allows a person to access all your data. To protect your privacy, you may backup your files. Then, create a new local user account and delete your old profile.

To create a new user, go to PC Settings > Users > Other users > Add a user > Sign in without a Microsoft Account. Make sure that the new user is an administrator. To delete your old profile, go to Control Panel > User Accounts and Family Safety > User Accounts > Manage another account > Click your account > Delete the account.

Let us know how it goes.
