    The bit of code below gets a string to the immediate window, but it doesn't seem to execute.  Maybe you can play around with it a little and get something to work.  I'm willing to be that you won't be able to get anything in the Immediate window to execute while your VBA code is executing, since VBA is single threaded.

    Eric

     

    Option Explicit

    Sub testit()
    Dim myVBE As VBProject
    Dim tStr As String
    '
    Set myVBE = ThisWorkbook.VBProject
    With myVBE
        tStr = "?MyVBE.VBE.Windows(""Immediate"").Caption" & vbCrLf
        myVBE.VBE.Windows("Immediate").SetFocus
        Application.SendKeys (tStr)
    End With
    End Sub

    This solution article describes how customers can setup their phones to sync their Hotmail inbox, calendar and contacts using Exchange ActiveSync (EAS).



    EAS keeps everything in sync between your phone and Hotmail, so whatever you do on your phone, like delete an email, add an appointment, or update one of your contacts with a new number, will also be reflected on the web, and vice-versa. If you use an email client on your PC that already syncs with Hotmail, like Outlook with the Outlook Connector or Windows Live Mail, what you do on your phone will show up there as well, delivering a seamless experience for managing your stuff between your PC email client, your browser, and your phone. 



    Supported Devices

    • The following devices have been tested against Hotmail ActiveSync
    • Windows Mobile 6.1
    • Windows Phone 6.5, 7
    • iPhone, iPod Touch, iPad
    • Nokia E-series, S-series, N-series running Mail for Exchange application
    • Android 2.2 and later

    • More devices will be added to the above list over time. 



    • Users are welcome to try with other devices but support may be limited in the near term.

     Quick Steps

    • If your device supports only one profile of ActiveSync, delete the existing ActiveSync profile on your device if you already use it



    • Enter the following: 

    URL - m.hotmail.com 

    Username - Enter full name with domain info, e.g. *** Email address is removed for privacy ***

    Domain - Leave blank (see Known Issues for some Nokia devices)

    SSL - Enabled 

    Certificate - Accept the SSL certificate when prompted

    Mail, Contacts, Calendar, Tasks - All can be enabled. 



    iPhone Setup

    Android Setup

    Nokia Setup – Mail for Exchange 



    The following steps are for the Mail for Exchange application on most Nokia smartphones. If the existing ActiveSync profile is already in use, make sure to back up the data on the device before configuring the ActiveSync profile for Hotmail access.



    Click here

    to view the video on how to setup Hotmail on your Nokia.



    Here are the step-by-step instructions:

    1. Click on “Mail for Exchange” icon in the Application menu.

    2. Click on “Sync mode”. You may need to delete the existing profile if already used to configure it for Hotmail access.

    3. Enter your full Hotmail email address in Email address and Username fields.

    4. Enter m.hotmail.com in the Exchange server field.

    5. Select the Initial sync option for calendar, e.g. “Delete items on phone”.

    6. Select the Initial sync option for tasks, e.g. “Delete items on phone”.

    7. Select the Initial sync option for contacts, e.g. “Delete items on phone”.

    8. Configure the Sync mode to be either push or pull at a specified interval.

    Windows Phone 6.5 and Windows Mobile 6.1 Setup

    The steps are identical between Windows Mobile 6.1 and Windows Phone 6.5. If deleting an existing profile, make sure to back up the data on the device before configuring the ActiveSync profile for Hotmail access. 

    1. Click on “ActiveSync” under Start menu or under Programs.

    2. Click on the link above to configure the ActiveSync profile for Hotmail access. You may need to delete the existing profile if it is already in use.

    3. Disable automatic detection of Exchange server settings.

    4. Enter your full Hotmail/Live address in User name field. Enter password. Leave Domain blank.

    5. Enter m.hotmail.com in the Server address field.

    6. Select the data to sync. Hotmail supports all 4 data types via ActiveSync.

    7. The device will connect to Hotmail ActiveSync and perform an initial sync for the selected data.
