OK, read over all the Clean Reinstall Windows 7 steps and then start at  Step 5 for the actual install. 

The error you're getting pertains to the Windows Boot Manager which is how a UEFI install is booted.  To avoid it you must boot the installation DVD as a UEFI device.  To do that put the DVD in the drive, hold the power button to power off.  Then press the power button and immediately start tapping the F12 key (for Dell) which should bring up a Boot menu.  If you're not tapping soon enough or vigorously enough, you may need to power off and on again to start over.  You need to get the F12 key to bring up a Boot menu. 

Now this is very important:  When you get the Boot menu you are going to look for a UEFI DVD drive.  There should be a regular DVD drive and a UEFI DVD drive.  TO give you an idea of the difference look at the image below: 

Yours will look different but you want the UEFI or EFI DVD drive so you'll have to figure out which one that is.  If there isn't one then look for any other UEFI boot items and report back what you found.
