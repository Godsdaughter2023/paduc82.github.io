"Would all cells in rows to be deleted have to be unlocked??"

 

That's pretty much the answer - or maybe I should say the answer to that is Yes.  Seems to make the whole concept of "allow deleting rows" rather useless if you have formulas or captions in the row(s) that you wish to enable people to delete.

 

Depending on the whole layout of things, you may be able to simply get things set up to work in protected mode:  Select all of the cells on the sheet (select the sheet and press [Ctrl]+[A]) and start by unlocking all of them.  Then start selecting the ones that must remain locked (you can do this a few at a time in groups) and lock them so that they are untouchable once you apply protection to the worksheet.

Sub DeleteSelectedRows()
  ActiveSheet.Unprotect Password:="123"
  Selection.EntireRow.Delete
  ActiveSheet.Protect Password:="123"
End Sub

Dealing with AutoFilter isn't quite so easy: again all of the cells that may change based on the choices made have to be unlocked if the password has been applied.

Importing values from sheet 1 to sheet 2, deleting sheet 1 values and - Microsoft Community

jtran92 say:

Importing values from sheet 1 to sheet 2, deleting sheet 1 values and sheet 2 will retain?

So I tried using =INDIRECT() and was able to import data from sheet 1 to sheet 2 and it worked, but I want to be able to clear out data from sheet 1 and get sheet 2 to retain the values and keep adding to it when sheet 1 is populated with new data. I'm not sure if this is possible but any advice is appreciated.

Sheet 1 is an input form and sheet2 is a database.

jllatham say:

The rule we have to keep in mind:  An Excel cell may hold either a formula or a value, not both.

So right now your cells on Sheet 2 have formulas in them (the INDIRECT() formula).  Once you delete the original data on Sheet 1 you will either show all 0s (zeros) or if you deleted actual rows, you might end up with #REF! errors.

What you need to do before getting rid of the data on Sheet 1 is to turn the formulas on Sheet 2 into values.  You can do this by following this process:

Select Sheet 2.

Select the cells on Sheet 2 that have the data echoed from Sheet 1 by way of the =INDIRECT() formula.

Use Edit --> Copy  (or keyboard shortcut [Ctrl]+[C]) to copy them to the clipboard.

Without doing anything else, still leaving those cells selected, then you

Use Edit --> Paste Special with the Values option selected.

That gets rid of the formulas and leaves their results behind.

Now you can go back to Sheet 1 and clean it up and the other information will remain on Sheet 2.

A couple of things to pre-plan -- first make sure you've got some INDIRECT() formulas set up on Sheet 2 ready to grab the next information batch you toss onto Sheet 1.  Be sure you don't delete the data on Sheet 1 before you've used the Edit --> Paste Special w/Values option process.

Hope you find this useful.
