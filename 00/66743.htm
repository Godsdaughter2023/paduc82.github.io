PC screen only - The second screen is ignored and only the PC’s primary display is used.
Duplicate - The second screen mirrors, the primary display.
Extend - The PC display is extended across the two screens and allows you to have a desktop that spans two screens or to use Windows UI on one screen and the desktop on the other.
Second screen only - The main PC display will be disabled and only the secondary screen is used.

Indexing Options>Modify

gerry c j cornell say:

Palcouk

 

The Modify option allows you to change which Locations are indexed  and which are not. Files and the contents of files are indexed. I only use Windows Search to look for files. It occurred to me, having realised that contents are indexed, that for me that this a complete waste of time. A search with only files indexed would most likely considerably reduce time spent searching. Coincidentally many users complain about the time Windows Search takes.

 

To exclude a file from indexing right click on the files in Windows Explorer and select Properties. On the the line for Attributes click on the Advanced button and uncheck the box to exclude Contents being indexed. The problem is that every file has the box checked so I am looking for a way to reverse the default.

 

Another snippet of information I discovered earlier before starting this thread. If you wish to delete a 2 gb file using Shift+Del uncheck the box before indexing contents and the deletion takes a significantly shorter time.

 

In answering your message I seem to have resolved my original question. Select the properties of the folder containing the files. Click on Advanced, uncheck the box before indexing contents, click on OK and Apply and a prompt asks whether the change is to apply to the Folder or to folder, sub-folders and files. Select the second option and the contents of files in the folder cease to be indexed.
