suzanne s. barnhill say:

What you want is a decimal tab stop, but you don't need a tab character. When you set a decimal tab stop, Word automatically aligns cell contents on the decimal point or the place where one would be. In principal, this means that it aligns text on the first non-numeric character (aside from the thousands separator) following a series of numbers. In practice, this means that you can follow numbers with a closing parenthesis, a symbol or letter used as a footnote reference, a decimal point, or any other non-numeric character.

In order for this to work, you must have the cell contents left-aligned (or justified). If you use Center or Right alignment, the decimal tab is ineffective. The decimal tab stop is the only one that behaves this way; to insert a tab character in a table cell to advance to any other kind of tab stop, press Ctrl+Tab.

For more on this, see http://wordfaqs.mvps.org/NumberAlignment2007.htm#DecimalTabs.
