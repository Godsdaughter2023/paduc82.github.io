ossiemac say:

The following VBA code returns the date and time of the last save of the workbook where the VBA code is located.

Dim dteFileSave As Date

dteFileSave = FileDateTime(ThisWorkbook.Name)
MsgBox dteFileSave

You can use a string with the full path and filename enclosed in double quotes in lieu of ThisWorkbook.Name to find the last Save Date and Time of a closed file.

Dim dteFileSave As Date
dteFileSave = FileDateTime("C:\Users\User\Documents\FileName.xlsx")
MsgBox dteFileSave

For an open workbook, try something like:

   MsgBox ActiveWorkbook.BuiltinDocumentProperties(12).Value

For a closed file try the following function: 

'=========>>
Public Function Last_Saved(sFileFullName) As Date
    Dim oFile As Object

    Set oFile = CreateObject("Scripting.FileSystemObject")
    Last_Saved = oFile.GetFile(sFileFullName).Datelastmodified
    Set oFile = Nothing
End Function
'<<=========

'=========>>
Public Sub TestIt()
    MsgBox Last_Saved("C:\Users\NDJ\Norman20151218.xlsm")
