kerry_brown say: Some BIOS's are quite funky when it comes to booting from USB devices. A few of things to try: Change each USB related BIOS setting one at a time, rebooting and testing after each change. I do mean each and every BIOS entry that mentions USB. Try all the settings for each entry. Use the disk managment console to check if there are any Active partitions on the USB drives. If there are, remove the Active attribute. Be careful not to remove this from any internal drives as the system won't boot if you do that. Remove all other USB devices including disconnecting any internal card readers, printers, cameras, scanners, phones, etc. With one external drive and no ther USB devices connected see if the computer boots OK.
