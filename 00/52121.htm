To repair Mail.MsMessageStore file, please refer to the steps below:

 

1. Restart the computer (this is to ensure that Live Mail is fully shut down, and that none of it's components are in use when the computer starts)

2. Click Start, type CMD but do not press enter

3. You will see cmd.exe in the search results, right click on it and select Run as Administrator - type your admin password if asked, and if UAC pops up asking premission, allow it

4. You should now see the command prompt, with the following displayed:

Microsoft Windows [Version 6.1.7600]

Copyright (c) 2009 Microsoft Corporation. All rights reserved.

C:\Windows\system32>

5. Copy the following (Ctrl+C)

esentutl.exe -d "%userprofile%\AppData\Local\Microsoft\Windows Live Mail\Mail.MSMessageStore"

Note: the "" are necessary

6. Right click in the command prompt Window, and select Paste, so you now see the following:

Microsoft Windows [Version 6.1.7600]

Copyright (c) 2009 Microsoft Corporation. All rights reserved.

8. Once done, you can close the command prompt, and restart the PC again

First of all, by posting a question here you are not addressing "Microsoft."  By far the majority of people providing useful help in the Microsoft Community are not Microsoft employees but are volunteers willing to share their experience with Microsoft products in order to help others.

Second, Microsoft has decided that because Windows 10 will be compatible with everything (snark), it no longer needs to provide a "Compatibility Center."  See http://answers.microsoft.com/en-us/windows/forum/windows_10-other_settings/windows-10-compatibility-center-offline-and/c7577bb1-099b-4393-a86d-2d388d7ed265?msgId=459272eb-5534-4b7e-84db-06b65cfd94b9

Third, "Intel said the appropriate drivers were sent to Microsoft but Microsoft has not released them" is nonsense.  It's the typical, "It's not my fault, it's his" response that you get any time there are multiple vendors involved.  You can get drivers for Intel products directly from Intel if you want.  Just be aware that for OEM computers like Dell, the computer manufacturer may have used customized hardware for which it has customized the "standard" drivers.  That's why it's generally better to get drivers from the computer manufacturer's web site rather than from the site of the hardware component manufacturer.

    • added or removed software (what?)
