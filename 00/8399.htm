a. user say:

Re: convert the first four digits to a date

You can use a  UDF (user designed function) to accomplish this.
Copy and paste the following VBA code into a standard module.
'---
Public Function ExtractDate(ByRef lngDateNumber As Variant) As Variant
 Dim lngDay As Long
 Dim lngYear As Long

 If Len(lngDateNumber) < 4 Or Not IsNumeric(Left(lngDateNumber, 4)) Then
   ExtractDate = "Invalid Serial Number"
   Exit Function
 End If
'Invalid after the year 2019
 lngYear = "201" & Left(lngDateNumber, 1)
'Get digits 2 to 4
 lngDay = Mid(lngDateNumber, 2, 3)
 If lngDay > 365 Or lngDay < 1 Then
   ExtractDate = "Invalid Serial Number"
   Exit Function
 End If
 ExtractDate = DateSerial(lngYear, 1, lngDay)
End Function
'---

Use it like any normal Excel function...
Enter in a cell... = ExtractDate(E5)

E5 is the cell location containing the date/plant code.

The cell with the function must be formatted as a date.

john korchok say:
Outline tab of the left-hand slide list, select all text, click on Review>Language>Set Proofing Language and choose the language.

If you need to update the language on text boxes and notes pages, here is a macro that will do that for you. You can create copies of this with the LanguageID set to msoLanguageIDDutch and msoLanguageIDEnglishUK:

                            

Sub ChangeLanguagetoUSEng()
  'Declare variables
  Dim SlideCount&, ShapeCount&
  Dim x As Integer, y As Integer

  'Count the slides in the presentation
  SlideCount& = ActivePresentation.Slides.Count

  'Loop through the slides
  For x = 1 To SlideCount&

    'Count the shapes on each slide
    ShapeCount& = ActivePresentation.Slides(x).Shapes.Count

    'Loop through each shape
    For y = 1 To ShapeCount&

      'Check if the shape has a text frame
      If ActivePresentation.Slides(x).Shapes(y).HasTextFrame Then

        'If it does, set the language
        ActivePresentation.Slides(x).Shapes(y).TextFrame.TextRange.LanguageID = msoLanguageIDEnglishUS
      End If
    Next y

    'Count the shapes on the Notes page for the slide
    ShapeCount& = ActivePresentation.Slides(x).NotesPage.Shapes.Count

    'Repeat the steps above to set the language
    For y = 1 To ShapeCount&
      If ActivePresentation.Slides(x).NotesPage.Shapes(y).HasTextFrame Then
        ActivePresentation.Slides(x).NotesPage.Shapes(y).TextFrame.TextRange.LanguageID = msoLanguageIDEnglishUS
      End If
    Next y
  Next x
End Sub

  2. In the Run window, type:  Type %localappdata%\Microsoft\OneDrive\onedrive.exe /reset
