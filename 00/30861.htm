Thank you for the information. Follow these methods to resolve the errors you're currently encountering:

Methods in resolving kernel_data_inpage_error:

Method 1: Retry running Chkdsk /f /r:

One of the reasons the kernel_data_inpage_error error shows up is because of bad sectors on your Hard Drive. Chkdsk is a utility system tool that repairs bad sectors found on your Hard Drive. Follow these steps to run Chkdsk:

  1. Press Windows key + X on your keyboard and choose Command Prompt (Admin).
  2. Type this on the window that would show: CHKDSK /R /F
  3. Restart and check if the issue is resolved.

Method 2: Scan for malware or virus:

We recommend running a virus scan to see if there are any virus or malwares causing the issue as this is also a possible reason for the error code. We recommend using Windows Defender to scan. Simply press Windows key on your keyboard or press Start and search for and open Windows Defender and click Scan. If you already have a Antivirus software, then run a scan using that.

Method 3: Troubleshoot hardware drivers:
Another possible cause of the kernel_data_inpage_error is a misbehaving driver. A driver is used by Windows to communicate with the hardware that you use on your PC. Follow this article for instructions on how to automatically get the latest drivers or manually get them yourself.
Note: CRITICAL_PROCESS_DIED error is 90% caused by a misbehaving driver. Following this method can resolve this error.

Method in resolving Checking media: [fail] error:

Follow SciTechApple's and MathewFrancis' instructions on this thread to resolve the error message you're receiving when starting up your laptop. You can find this thread helpful as they have the same device brand as yours.

Update us of the result of the methods we've provided.

A lot of variables are involved when you connect to a network: cables, network adapters, drivers, and routers. So troubleshooting network connection problems can be challenging. Some common problems are:

  • A wireless network adapter switch is turned off

  • Airplane mode is turned on

  • Disconnected or improperly connected cables

  • Incorrect WEP, WPA, or WPA2 security keys or passphrases

  • Mismatched wireless standards

  • Missing updates

  • Incorrect network connection settings

  • Damaged or incompatible drivers

  • Hardware or software problems

It may also happen that if you try to connect internet with a VPN connection and it doesn't connect because thelocal computer does not support encryption. You may try the following steps:

  1. Open Connect to a Network by clicking the network icon in the notification area.

  2. Right-click the VPN connection, and then click Properties.

  3. Click the Security tab.

  4. Under Data encryption, select Maximum strength encryption (disconnect if server declines), and then click OK.

  5. Open Connect to a Network by clicking the network icon in the notification area.

  6. Click the VPN connection, and then click Connect to try connecting again.

How-to solve Windows Store starting issues. - Microsoft Community

yuri pustjens say:

How-to solve Windows Store starting issues.

Technical Level : Basic

Summary

There been many questions on these forums how to solve the Windows Store not starting issue and most of them occur after upgrade from Windows 8 to Windows 8.1.


There are several general methods of solving this, this guide will help you through the most given solutions. After following this guide,  if you are still having issues, then please make a new support request and include the link to this wiki saying you have already tried these steps.


Before continuing with this wiki, ensure that your system is fully up-to-date with all Windows Updates.


Details

Method 1: Microsoft Store Diagnostic Program

  1. Download the tool: http://go.microsoft.com/fwlink/p/?LinkId=268423
  2. Start the downloaded file and click on Next



  3. Follow the instructions given by the tool.
  4. After running the tool succesfully, check whether the store works. 

Method 2: Run WSReset.exe

  1. Open Modern UI (Start)
  2. Typ Wsreset.exe



  3. Hit enter
  4. The Windows Store will open and flash a couple of times.
  5. The Windows Store should now automatically appear and after a few moments if it does not, then try to start it manually.

Method 3: Register the Store trough Command Prompt

See all steps executed in this youtube video: https://www.youtube.com/watch?v=fj4viu2xwyk

  1. Open  Modern UI (Start)and typ CMD
  2. Right click on Command Prompt and choose Run as Administrator


  3. In command prompt enter the following command:

    powershell -ExecutionPolicy Unrestricted Add-AppxPackage -DisableDevelopmentMode -Register $Env:SystemRoot\WinStore\AppxManifest.xml




  4. Hit enter, the blackscreen will appear still for a couple of seconds. After a couple of second a new line saying C:\.... will appear.
  5. Close Command Prompt
  6. Try to start the Windows Store again.

Method 4: Scanning for corrupted system files

Did none of these methods work? Then try to run SFC and DISM to solve your issues: Troubleshooting Component Store / System Files Corruption
