How to delete a page range? - Microsoft Community

jim-chicago say:

How to delete a page range?

Have a long doc in Word and want to delete a range of pages.  How to do?  2011 Word for Mac.

Jim in Chicago

bob jones aka: cybertaz say:

One of the most confounding concepts to wrap one's head around is that there is no such element as a page in the structure of a Word document :-) What you have is a body of text comprising 1 or more Sections with each Section containing 1 or more Paragraphs. Delete the text & the euphemistic pages used to display it will evaporate.

Before deleting an extensive passage, though, it's a good idea to have the nonprinting characters [¶] displayed in order to better tell exactly what you're deleting. The Paragraph marks & Section Breaks carry most of the layout control of the document, so its best to know where they are in order to determine whether they should be deleted. E.g., if the passage spans a Section Break you may not want to delete the Section Break, itself. Instead, separately delete the text that precedes it & the text that follows it in order to preserve the Section Break. Otherwise there may be some surprises when the deletion occurs... If so, just use Undo, then examine what is within the deleted range that accounts for the change.

The simplest method for deleting an extensive range of text is to:

  1. Select the first paragraph to be deleted by Triple-clicking it or Double-Clicking in the left margin adjacent to that paragraph
  2. Scroll as necessary to view the last paragraph to be deleted
  3. Hold the Shift key while clicking in that paragraph
  4. Press the delete key

gmteacher say:

High CPU temps would certainly indicate excessive CPU Utilization to me.

I'd at least run Task Manager and keep an eye on CPU usage. (Do a start menu search for Task Manager)

Additionally, if it's really high, expand Task Manager and look at running processes sorted by the CPU Utilization column and see what's racing the clock on you.

Lastly, I'd try a good CPU monitor Utility to see what's going on with %, temps, and fans. I like CPUID HW Monitor, and it's free and helps me solve these kinds of mysteries.

As you can see, my box is running cool as a cucumber, but CPU Utilization is typically below 20%.

Typically, though, when I first boot up, I can see all the housekeeping tasks sucking up CPU cycles like crazy for about the first 5 to 7 minutes until everyone has done their homework, checked themselves for updates, and run the typical startup diagnostics. During that time, the CPU does get a tad warmer, but never above 120-130°F.

Disclaimer - I have no connection to any of the software products that I have mentioned in this post. I'm just an average user with an interest in what's going on under the hood.

noahsheng_msft4652 say:

hi giacomo,

it is not feasible to forward all of your email for the last year to your gmail account and retain the original email via owa.

for a workaround, you could consider putting your last year email to new folder, then export your last year email, in the end, import your file in gmail via outlook client.

take the following steps:
1. create a new folder.
2. click  rules, then click create rule in inbox mail.
 
3. click advance options

 
4. click received in a specific date span, then set date received. then click next.

 


5. click move it to send items folder, and choose your new folder. then click next.

 

6. click run this rule now on messages already in “sent items”. then click finish.

7. export your new folder, please refer to the following article: https://support.office.com/en-us/article/export-or-back-up-messages-calendar-tasks-and-contacts-d791fe88-5c67-44c5-a3f6-a484508082a1


8. configure the gmail account to outlook and then import the pst. file, please refer to the following article:  https://support.office.com/en-us/article/import-gmail-to-outlook-20fdb8f2-fed8-4b14-baf0-bf04b9c44bf7


if you encounter any question, please feel free to contact us.

regards,
noah
