My RAM must be installed in pairs, the slots are color-coded to make it easy, so I'm certain I have the right sticks in the right slots.  I've swapped slots with all the sticks, installed each pair separately, every combination you can think of, with the same results: The system reports the correct amount of installed RAM, but always has everything over 3GB as "Hardware reserved".

my scan results:  http://www.crucial.com/usa/en/scanview/4C4FDF4631D926C3

Yes, my sys is ancient, but still going strong =)  My biggest issue, other than this, is running out of HDD space.

I'm running Win 8.1 Pro with Media Center, Intel Core 2 Quad Q6600 @2.4 ghz, 64 bit OS.

I've done the check on msconfig/boot/advanced settings maxmem

I figured it out!

under my BIOS there is an option labeled "Memory Remap Feature" which is by default "Disabled."  The description for this feature provided (while in the BIOS) is "Allows remapping of overlapped PCI memory above the total physical memory"  I enabled this, and now the system reports 6.0 GB installed, 5.1 GB in use or available, and 0.9 GB Hardware reserved.   Makes no sense to me, since I have 6GB physically installed, but as long as it works, who cares?

I'm leaving this up for anyone else who might have a similar issue and odd terminology in the BIOS.
