john mcghie say:

Hi Kristen:

You need to avoid confusing "fields" with "bookmarks".  In Word, they are very different things.

To "move" a bookmark, you simply write it somewhere else.  Since a bookmark name must be unique within a document, when you re-write it, Word will update the location of the bookmark.  "Bookmarks" in Word are a table at the bottom of the document, containing the name of the bookmark (case-sensitive) and the byte offset of the bookmark range.

Fields are a different matter.  Fields do not have to be unique: there can be multiple instances of a field in a document.  To move a field, you need to locate it, cut it, and paste it somewhere else.

I am confused by your reference to "parent" fields.  In Word, none of the fields have "parents". Perhaps if you provide more explanation, I can help further?

Hope this helps

Hi,

A simple way is to use "CommandBar" and "CommandBarControl" object.

Please refer to the specific code:
In ThisDocument:

Option Explicit

Private Sub TextBox1_MouseUp(ByVal Button As Integer, ByVal Shift As Integer, ByVal x As Single, ByVal y As Single)
    '╱＊If right-click.＊╱
    If Button = 2 Then
        CreatePopupMenu 'Here can you see a popup menu.
    End If
End Sub

Sub CreatePopupMenu()
    Dim cmdbar As CommandBar
    Dim cmdBarCtrl(2) As CommandBarControl
   
    '╱＊First : add a CommandBar.＊╱
    Set cmdbar = CommandBars.Add("cmdMenu", msoBarPopup)
   
    '╱＊Second: add three buttons onto the added CommandBar.＊╱
    With cmdbar.Controls
        Set cmdBarCtrl(0) = .Add(msoControlButton)
        Set cmdBarCtrl(1) = .Add(msoControlButton)
        Set cmdBarCtrl(2) = .Add(msoControlButton)
    End With
   
    '╱＊Third: set text of each of the three added buttons.＊╱
    cmdBarCtrl(0).Caption = "&Copy"
    cmdBarCtrl(1).Caption = "&Paste"
    cmdBarCtrl(2).Caption = "&Delete"
   
    '╱＊No selected text in textbox1.＊╱
    If TextBox1.SelText = "" Then
        '╱＊if there's no text in clipboard.＊╱
        If GetClipboardText = "" Then
            cmdBarCtrl(0).Enabled = False
            cmdBarCtrl(1).Enabled = False
            cmdBarCtrl(2).Enabled = False
        '╱＊if there's any text in clipboard.＊╱
        Else
            cmdBarCtrl(0).Enabled = False
            cmdBarCtrl(1).Enabled = True
            cmdBarCtrl(2).Enabled = False
        End If
   
    '╱＊Any selected text in textbox1.＊╱
    Else
        '╱＊No text in clipboard.＊╱
        If GetClipboardText = "" Then
            cmdBarCtrl(0).Enabled = True
            cmdBarCtrl(1).Enabled = False
            cmdBarCtrl(2).Enabled = True
        '╱＊Any text in clipboard.＊╱
        Else
            cmdBarCtrl(0).Enabled = True
            cmdBarCtrl(1).Enabled = True
            cmdBarCtrl(2).Enabled = True
        End If
    End If
   
    '╱＊Give an action of each of the three added buttons.＊╱
    cmdBarCtrl(0).OnAction = "CmdCopy"      'Action to copy.
    cmdBarCtrl(1).OnAction = "CmdPaste"     'Action to paste.
    cmdBarCtrl(2).OnAction = "CmdDelete"    'Action to delete
   
    cmdbar.ShowPopup    'Popup the menu.
    cmdbar.Delete   'Delete the menu.
End Sub


In basMacro(just a module name):
Option Explicit

Public Sub CmdPaste()
    SendKeys "^v"   'Send a shortcut key: Ctrl+V.

    SendKeys "{DEL}" 'Send the Del key.
End Sub

'#######################｛SetClipboardText｝#######################
'【Function description】Copying text to the Clipboard。
'【Parameter meaning】strText -->> The text that you want to copy.
'##################################################################
Public Sub SetClipboardText(ByVal strText As String)
    Dim myCopy As New DataObject
    myCopy.SetText strText
    myCopy.PutInClipboard
    Set myCopy = Nothing
End Sub

'##############｛GetClipboardText｝##############
'【Function description】Get text from clipboard.
'################################################
Public Function GetClipboardText() As String
    On Error Resume Next
    Dim myCopy As New DataObject
    myCopy.GetFromClipboard
    GetClipboardText = myCopy.GetText
    Set myCopy = Nothing
End Function

In this way, we will find that the popup menu according to user operations and the contents of the Clipboard intelligently.

Unfortunately, I'm not really sure what all could interfere, but I do recall having a similar problem about 8 months ago. I tried everything I could think of, and never get it working properly. ="Joe Manager   0/10  [0%] 0/48 [0%] " & TEXT(N("x"),"")

 

The first part of the string was replaced every time by my VBA, and the N("x") was always tacked on the end to help me identify the target cells, since the visible part of the text string was constantly changing.
