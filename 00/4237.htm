Lookup - Microsoft Community

mike 609 say:

Lookup

in this formula =lookup(1E+100,G3:R3)-F3 what does the function1E+100 do. Also, is 1E+100 part of a bunch of other option or codes?

Thanks

Mike609

bernie deitrick say:

1E+100 is just a really big number: the formula returns the right-most numeric value in the range G3:R3. 

An alternative formula would be

=INDEX(G3:R3,1,COUNTA(G3:R3))

or just

=INDEX(G3:R3,COUNTA(G3:R3))

which can return text or numbers, but requires that no cells be skipped when filling from G3.

If you need to return the right-most text with cells skipped, then you could use this array formula, entered using Ctrl-Shift-Enter:

=INDEX(3:3,MAX(IF(G3:R3<>"",COLUMN(G3:R3))))
