a. user say:

Hi Ron,

The specified registry key contains zone mappings under the enhanced security configuration. Each item posted under this Windows registry component is a web site domain. Web addresses added to the list are either intranet zones or trusted site zones. Deleting the entire registry folder component will remove your capability to add web sites to the trusted zones which may affect your browsing experience when loading sites that doesn't pass the security parameters of your device.

We'd recommend deleting only the items within the component folder to make sure that the feature to add trusted sites as well as browsing the already tagged sites will not be affected. To delete entries in a component folder, you can perform the following:

  1. Search regedit using Cortana.
  2. Right-click on regedit, and then select Run as administrator.
  3. Locate HKEY_USERS\.DEFAULT\Software\Microsoft\Windows\CurrentVersion\Internet Settings\ZoneMap\EscDomains
  4. Right-click on an item that you'd like to delete, and then select Delete.
