suzy-appsforoffice say:

SOLUTION - RESOLVED

I'm working on a large template project that is using Word 2007 and I have just noticed this same annoying behavior on my client-site PC - I have a starting set of 85 templates and copying content from one template to another is converting all styles to Normal!

These are templates that have the same underlying style architecture.  And we may have to do updates, rebuild the style architecture and copy the old content over for small tweaks - this was going to be a disaster.

Luckily I have my own PC with me and checked my Word 2007 and 2010 environments - didn't have the same behaviour - that's a relief!

I then compared the Options settings with my client PC settings in MS Word 2007.

And this is the setting that did it:

1. Go to: Options > Advanced > Cut, Copy and Paste

2.  Make sure that Use smart cut and paste is ticked.  (it's the last checkbox in this section)

3. Click the Settings button next to this option

4. Make sure that Smart Style Behaviour is checked (it should be checked as a default).

In my client Word environment I had inadvertently unchecked the Use Smart Cut and Paste option.

Glad I can breathe again!

Hope this helps.

amit tandon say:

Try below:

=SUMPRODUCT(($B$1:$R$32=$T1)*($A$1:$A$32))

This will keep B1:R32 & A1:A32 constant, while T1 will become T2 if copied downwards.

Use $ sign to keep a range constant ex. $B1 will keep column B constant, B$1 will keep row 1 constant and $B$1 will keep the cell B1 constant, irrespective of formula being moved up/down or left/right.

amit tandon say:

Enter below formula as an array formula (Ctrl+Shift+Enter) in cell F4:

=SUM(IF((ROW(A1:A32)-ROW(A1)+1)=(B1:D32=F2)*(ROW(B1:D32)-ROW(B1)+1),A1:A32))

The formula will return the sum from column A, for matching row values of F2 in the range "B1:D32".

Please note that in the above formula if the number 12 appears twice in a particular row then the column A value is added twice, ex. if 12 also appears in C8 in addition to D8 then the formula will return -5.

* Alternatively, you may also use this shorter formula:

=SUMPRODUCT((B1:D32=F2)*(A1:A32))
