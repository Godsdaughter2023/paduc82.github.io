bobwald say:

Excel 2007 - How to set the default OPEN position of a textbox vertical scroll bar

 

Hello to all

I have a Excel 2007 UserForm containing two (2) TextBoxes with vertical scroll bars. 

When the UserForm opens the scroll bar in textbox 1 is in the desired top position.

However, the scroll bar in TextBox 2 opens in the undesired bottom position.

 

My question is – How to get TextBox 2 to open in the top position?

 

Thanks for any help

BobWald

ossiemac say:

In the UserForm_Initialize sub set the SelStart property to zero like the following and it will position the cursor at the top of the text box.

Generally the textbox that is the default first tab index (zero)  is the one that sets the cursor to the end of the text and I suggest that your textbox 2 is the one containing the cursor and is possibly the first tab stop when the userform shows.

You can set the tab indexes in the control Properties. Select properties for the control and ensure Tabstop is True and then set the TabIndex starting at zero for the first position and then select the next required control and set to 1 etc in the order that you want to tab through the controls on the Userform.

Following example of code in the Userform Initialize event to set the cursor to the start of the text.

Private Sub UserForm_Initialize()
    Me.TextBox1.SelStart = 0
    Me.TextBox2.SelStart = 0
End Sub

ossiemac say:

I should have also told you that the code can go in the Userform Activate event like the following.

Private Sub UserForm_Activate()
    Me.TextBox1.SelStart = 0
    Me.TextBox2.SelStart = 0
End Sub

If you Hide a Userform because you do not want to lose current data on it but want to show it again later then the Initialize event does not run but the Activate Event always runs whether the Useform is hidden or unloaded and then re-shown.

Sometimes the default internet settings sent by your network provider might not work properly or might lead to balance deduction.

this method solves the problem where only default browser connects to the internet but apps don't and also helps the users of Series 40 phones with Wi-Fi where the default browser and apps connects to network internet instead of Wi-Fi, leading to balance deduction.

You can rename the account name to any name you like.

2.
Now press back and select options>add new>access point.

3.
Now in settings>configuration>turn on configuration>(SIM1/SIM2)>web, select the setting which you have created as in step 1.

4.

Windows Defender is designed to function in a passive mode, if the "Periodic Scanning"

button is set to "on". What that means is, Windows Defender can be used for scans only.

Your Vipre program will supply your "Real Time protection.

The periodic scanning button is located on Windows Defender, "Virus & Threat Protection"

screen. It will only be present, once you have installed Vipre.

Windows Defender in passive mode, provides you with the option to run its Offline

Scanner, to supplement your Vipre program. This is desirable. Since it is Offline, it scans

for viruses without your Windows program being loaded. That allows it to do its job,

without interference from any malware that might be present.

There is one caveat;  Since the W10 v1803 release, on SOME PCs, the Periodic Scanning

Button appears to have lost its function. In that case, you will not be able to use  

Windows Defender at all. And any attempt you make to

Since the installation of Vipre deactivated it. And it cannot become passive, without

a functional periodic scanning button. Vipre will be your only security program.
