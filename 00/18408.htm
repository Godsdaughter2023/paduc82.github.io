mike h.. say:

Hi,

 

The macro recorder is fine as far as it goes but as you will see in your code it constantly refers to Activesheet and Selection and we should avoid that.

 

Also in your exclusion loop you need AND and not OR

 

Here I've shown the methods in the code by referring to WS and the column instead of selection.

 

Sub cleanupfile()
Dim ws As Worksheet
For Each ws In ActiveWorkbook.Worksheets
If ws.Name <> "Summary" And ws.Name <> "itemaster" And ws.Name <> "RevisionHist" Then  'sheets I am excluding
ws.Columns(2).ColumnWidth = 2 ' set column width to 2
ws.Columns(2).EntireColumn.Insert 'insert column for top level pn
ws.Columns(2).EntireColumn.Insert 'insert column for customer name
ws.Columns(2).EntireColumn.Insert 'insert column for customer program quantity
ws.Columns(5).ColumnWidth = 10
ws.Columns(6).ColumnWidth = 4
ws.Columns(7).ColumnWidth = 6
End If
Next ws
 End Sub

gcdem say:

To Pin a file to the taskbar:

Open File Explorer (aka Documents) > select the file you want to pin by left clicking your mouse > keep your finger on the left mouse button & drag the file to an empty part of the taskbar > you will see a prompt to "Pin to x 2013" (x = Excel, Word, etc...)" > Release your finger from the left mouse button > The Office Program Icon should be on the taskbar & the file will show to be pinned to that icon.

If you want to pin a new file to an existing Office Program Icon pinned to the taskbar:

Repeat above steps (Open File Explorer & select the file) > drag the file until it hovers over the Correct Office Program icon > you will see a prompt to "Pin to x 2013" (x = Excel, Word, etc...) > Release your finger from the left mouse button > the file will show to be pinned to the Office Program Icon. Repeat if you want to pin multiple files.

If you have an Office Icon already pinned to the dashboard, you don't need to delete it as others are suggesting.

MS should add this instruction to all the Office 2013 programs' help section FAQs because pinning files is widely used. Ultimately, MS could make this much more intuitive so that you can right click and one of the options is "Pin Current File to taskbar". Also, the list of pin files need to default to alphabetical or MS should all Pins to be moved up or down.

Yes, Excel is a bit stubborn when it comes to seeing dates. If the format of the cell is anything but "Text" and you enter anything that looks like a date, Excel will convert the value to  date and change the cell's format to one of the date formats. The only way to keep the value as entered and keep the selected cell formatting is to use the "Text" format.
