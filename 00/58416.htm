Hi

Here is what you should be seeing.

Moving a File changes the Source and Destination folder Modification Date and Time to Now.

 

Copying a file changes the Destination folder Modification time to now.

 

Renaming a file changes the current folder Modification date.

 

Saving a file changes the destination folder modification date to now.

 

Deleting a file changes the current folder modification date to now.

This should work on Internal as well as External drives.


Make sure that both drives are formatted with the NTFS file system.

  1. Click Start > Settings > System.
