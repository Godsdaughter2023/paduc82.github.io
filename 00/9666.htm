a. user say:

Hi, El.pd.win,

What I'd advise - 

1. If the account is still  able to obtain a Warranty should anything go wrong, have your Father contact the Manufacturer's website and have everything transferred to your name in case the warranty is still good.

2.You can change accounts where your's would be the Administrator account, not your Father's.

3. To Remove your Fathers account 

Assuming you're using Windows 10

Right click on Start > Select PowerShell (Admin)

Type Net User > Press Enter

This will show you the accounts on the system. You must be the Administrator account to remove his account.

Write down the account User Name you wish to delete (advise backing up his files before removing the account.))

Next, Type net user (his account name) /delete > Press Enter

The account will be removed from the system.

doug robbins - mvp office apps & services (word) say:

A work around is to select the cross references and use Ctrl+F11 to lock them before printing.  If required, they can later be unlocked by using Ctrl+Shift+F11

 

A macro containing the following code could be used to do that

 

Dim i As Long
With ActiveDocument
    For i = 1 To .Fields.Count
        If .Fields(i).Type = wdFieldRef Then
            If .Fields(i).Locked = False Then
                .Fields(i).Update
                .Fields(i).Locked = True
            Else
                .Fields(i).Locked = False
            End If
        End If
    Next i
End With
