Why is Windows 10 Search So Bad At It's Job? - Microsoft Community

ginkner say:

Why is Windows 10 Search So Bad At It's Job?

I've been using Windows 10 for a while now. For the most part, I like it. There have been some good additions like virtual desktops, the merged Win8 start screen and Traditional windows start menu, and it's got some nifty internal stuff going on. The most obvious flaw is the search feature.


I get it. It would be nice to have a search bar that let's you type stuff in and search your apps, your files, and the internet all at once. In theory it's an interesting tool. "Cortana", however, has some glaring issues that make it basically useless for everyday use. If going into my "All Apps" section to find what I need is easier than searching, you've failed.

Issue 1: Incremental Results
Again, this seems like a cool feature: change the results on the fly as I type in what I want to search for, then stream in results as they come in rather than posting a progress bar. Seems like it would be neat. It's not though, since there isn't any way for me to be sure that the search is complete, and incoming results change the position of the currently displayed results. That means if I've already found the app I want and am trying to click on it, it can suddenly be replaced with a completely separate app, or file, or website. This defeats the only practical benefit of HAVING streaming results in the first place.  Add to that a system that gives you just enough time to refresh after typing additional search words to move your mouse to a result that's about to disappear, and you've got yourself a frustrating and useless feature.

Issue 2: Failure to detect exact results
This is a big one. There are a large number of times where "Cortana" will fail to detect exact matches in the start menu, instead suggesting obscure system/program files. This is the MINIMUM a search feature on the start menu should do. This alone gets the system an F.

Issue 3: Bing and Edge
I don't know why Microsoft thought they needed a web search engine, but that's fine. Forcing me to use it is NOT fine. It is the furthest thing from fine I can imagine. Forcing users into a single web search provider is anti-competitive, pointless, and cripples the usefulness of the feature to the point where it's basically useless. Add in the fact that it launches Edge instead of the default browser, and you've turned your search feature into an invasive parasite.

Issue 4: Greedy Indexing
Indexing the file system can be a lot of work. USUALLY this runs when the computer is idle, but I've had many instances where the Indexing Service decides to run in the middle of a hard drive speed test, or during a game, ruining both. Some other new services also seem to have a hard time detecting whether or not the computer is being used.

These things need to be fixed. I'm sure people have other complaints, as do I, but these are the major ones that make the current system a mess to use. What do other people think? What other improvements could be made?

Hi,

There are many possible causes ranging from calibration to hardware or Firmware. Some issues can be rectified by installing the latest updates on your device. If the issue still persists, you may try a different approach to resolving this issue by following the steps provided below:

  1. Clean the Screen. Use a dry cloth or screen cleaner to clear the screen of any debris, or smudges that may be impacting touch functionality.
  2. Restart the Surface. Restarting the Surface refreshes the drivers for the device. If this recovers touch functionality, make sure you tries this step in the future.
  3. Perform a Two Button Shutdown. With the device turned on, press and hold the Volume up & Power Buttons for 15 seconds, or until the device shuts off. If the device boots to UEFI, go to Exit and Restart the device.
  4. Disable and enable HID-Compliant Touch Screen Driver in Device Manager.
    1. Open Device Manager.
    2. Go to Human Interface Devices.
    3. Look for HID-Compliant Touch Screen.
    4. Right click and then choose Disable.
    5. After disabling, right click on the driver and choose Enable.
      • Note: There may be two HID-Compliant Touch drivers. Disable both before enabling
  5. Disable and enable HID-Compliant Touch Screen Driver in Device Manager.
    1. Open Device Manager.
    2. Go to Human Interface Devices.
    3. Look for HID-Compliant Touch Screen.
    4. Right click and choose Uninstall.
    5. After uninstalling, restart Surface.
    6. After the first restart, restart once more, for a total of two restarts.
  6. Reset Touch Screen Calibration
    1. Go to start, and type in Calibrate.
    2. Locate Calibrate the screen for pen or touch input, and then select it.
    3. Click on Reset.
    4. Click the Yes button to confirm reset, and follow on screen instructions.
    5. If the reset button is greyed out, continue to Step 7.
  7. Check for electromagnetic interference. Sometimes, other electronic devices can negatively influence touch screen performance. To test this, move to a location that has minimal electronic devices in the environment and test the touch screen.

Should you need further assistance, feel free to get back to us.

lemp say:

It sounds as if you have misunderstood some advice.  BIOS updates (a) are not specific to any particular version of Windows and (b) generally are not installed "in Windows" but either directly from the existing BIOS or by using a bootable CD or USB flash drive.

Some manufacturers do provide Windows-based flashing tools, which you run on the Windows desktop to flash your BIOS and then reboot.  Even the manufacturers who provide these tools usuaull caution against using them.

Flashing your BIOS from within Windows can result in more problems.  All that software running in the background—including security programs that may interfere with writing to the computer’s BIOS—can cause the process to fail and corrupt your BIOS. Any system crashes or freezes might also result in a corrupted BIOS. It’s better to be safe than sorry, so use a BIOS-based flashing tool or boot to a minimal DOS environment to flash your BIOS.

The file extension 211 probably refers to a version number.  You'll need to use some sort of BIOS flashing tool.  Generally, this will be described in a README.txt file that accompanies the BIOS update file.

You need to be quite careful when updating the BIOS.  If you do things wrong, you can render your computer inoperative (this is called "bricking" your computer; in other words, making your computer about as useful as a brick).

steve cochran say:
First of all Norton and Windows Mail don't work very well together.  Seewww.oehelp.com/OETips.aspx#3
 
Second, most ISPs limit the size of messages that you can send, so even if you were to be able to overcome Norton's restrictions, the message would still get kicked back by the recipient's email provider.
 
So if you really need to send such a large email, and you are sure the recipient won't curse you, then you can go to Tools | Accounts | Mail | Properties | Advanced and check the box to break apart messages larger than a certain size (try 1 megabyte), and then the message will be sent in pieces that will combine themselves at the recipient's end, if their email program is compatible with such messages.
