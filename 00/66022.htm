wunders say:

Here is the breakdown:

  

 cacls F:    <- Change permissions on drive F:

   /E           <- Edit permissions (as opposed to Replacing Permissions)

   /T           <- Recurse down through existing directory structures

   /C           <- Continue even if you get an error

   /G Everyone:F    <- Grant  Everyone  Full access

 

Ref:  < http://technet.microsoft.com/en-us/library/bb490872.aspx >

 

Note that if permissions have been specifically denied to a user or group, they will likely continue to be denied as a result of the "/E" flag.  Denials always take precedence over Grants.

Login Information

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin

UPnP

  1. Open a web browser on a computer.
  2. Type the gateway address “192.168.1.1” into the web address bar.
  3. When prompted for a username and password type in “admin” for both the username and the password.
  4. On the left hand side of the web page select the “Advanced Settings” option.
  5. Select the “USB Application” subcategory below.
  6. Select the “Miscellaneous Setting” tab on the top.
  7. Next to “Enable UPnP?” select the “Yes” option.
  8. Click the “Apply” button on the bottom of the page.
