Hi Jm0x,

I would like to explain that it is different between Outlook, Office 365 and SharePoint (Including OneDrive).

You can set the timezone when you access Outlook first time.

About Office 365, you can change the timazone for Office 365 (Click Settings>select Language and time zone). It is set for the user who sign into Office 365 Portal.

You need to set timezone for each site.
