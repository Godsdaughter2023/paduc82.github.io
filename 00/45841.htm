vinod n say:

Hello Erwin,

Welcome to Microsoft Vista Answers Forum!

 

If your Guide listings in Media Center are affected by the transition, here are some things you can try:

• Set up your TV signal again. If you're receiving Guide listings for analog TV channels, there's a chance that they won't show up properly after the transition. Setting up your TV signal again might fix the problem.

• Make sure you have the most recent updates. You might need to install recent Windows updates before the Guide listings in Media Center will appear correctly.

 

 • Remove channels that don't work from the Guide. As part of the transition, you might see additional channels in the Guide listings in Media Center that you can't access. If this is the case, it's best to remove these channels manually to avoid problems with scheduled recordings.

 To add a missing analog TV channel:

1.  On the start screen, scroll to Tasks, click settings, click TV, click Guide, and then click Add Missing Channels.

 2.  Click Add Channel.

 3.  Type the name of the missing channel, and then click Next.

 4.  Type the number of the missing channel, and then click Next.

 5.  If prompted, type the physical number of the missing channel, and then click Add.

 6.  Click Done.

 To add a missing digital TV channel:

1.  On the start screen, scroll to Tasks, click settings, click TV, click Guide, and then click Add Missing Channels.

 2.  Click Add DTV Channel.

 3.  Type the name of the missing channel, and then click Next.

 4.  Type the number of the missing channel, and then click Next.

 5.  If prompted, type the physical number of the missing channel, and then click Add.

 6.  Click Done.
