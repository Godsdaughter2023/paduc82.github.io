rik_uk say:

Perhaps the simplest way will be to do the following to assign the macro to a shortcut key combination...

In the bottom left of the excel screen, in the status bar, is an Icon to record macros. Click this to begin recording a macro.

The Record Macro window will pop-up. Change the macro name to  SaveTabsAsText , type q  in the short cut key field,
make sure the drop down says  This Workbook , then click OK.
Press the stop icon in the bottom left of the screen...

From the View Ribbon click the icon above Macros (or press Alt+F8) and the Macro Window will pop-up.
Select the Macro you just created and click Edit. This opens the Macro Editor.
Click the cursor to the begining of the line which has End Sub and press enter to create a blank line. move the cursor to the new blank line.
Then Copy and Paste the code below in this line

Dim varSht As Variant, strFileNm As String
Application.ScreenUpdating = False
    strFileNm = ActiveWorkbook.FullName
    For Each varSht In ActiveWorkbook.Worksheets
        varSht.Copy
        ActiveWorkbook.SaveAs Filename:=Left(strFileNm, Len(strFileNm) - 4) & " " & varSht.Name & ".txt", FileFormat:=xlText
        ActiveWorkbook.Close False
    Next
Application.ScreenUpdating = True

Close the Editor window and save the file. Now when you press Ctrl+q the macro will run.

Hi fumei2,

Welcome to the Answers groups - glad some of these old unanswered questions still attract some attention!

{ USERNAME } inserts the user name that's defined in Office, which is logically distinct from the one held in the Environment (typically set up by Windows according to the login name of the user).
