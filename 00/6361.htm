Again, was it stored in your normal template? That is the default. If not, what was the name of the template in which it was stored? You need to look for that template. With Vista and Windows 7, ActiveSync has since been replaced with WMDC, so that explains why you can't use ActiveSync anymore. 

If Win7 is not seeing/detecting your device, then I would suggest checking to make sure that it has properly detected your USB drivers.  It may be that your USB is not working because the appropriate drivers have not been loaded.   So the obvious first question is: "Does your USB work when you plug other USB devices in (like a USB mouse, USB keyboard, USB Hub, etc)?"

how do i add time codes to a document - Microsoft Community

cynthiaswindle say:

how do i add time codes to a document

I need to add a time code every 30-40 seconds in a document as I type. 
suzanne s. barnhill say:

I can think of two ways to do this. Neither is perfect, and Jay may be able to provide a macro that will work better, but here they are:

Method 1

  1. On the Insert tab, in the Text group, click Date & Time.
  2. In the Date and Time dialog, select the time format you want to use.
  3. Click the button for Set As Default.
  4. Whenever you press Alt+Shift+D, the date/time will be inserted in the format you selected.
  5. Very important! The time will be inserted as a field that will update automatically, so you must immediately select it and press Ctrl+Shift+F9 to unlink it (convert it to ordinary text).

Method 2

  1. On the Insert tab, in the Text group, click Date & Time.
  2. In the Date and Time dialog, select the time format you want to use.
  3. Check the box for "Update automatically."
  4. Click OK to insert the TIME field.
  5. Select the field in the document and press Alt+F3.
  6. In the Create New Building Block dialog, type a name for the AutoText entry you are creating. You will want this to be something easy to type but not necessarily a real word, such as "Time." Since it's a time code, something like "tico" would work.
  7. Make sure that AutoText is selected as the gallery and General as the category (they should be by default, along with "Insert content only").
  8. Click OK.
  9. Now whenever you type "tico" (or the first four letters of whatever name you chose), you'll get an AutoComplete tip suggesting this AutoText entry. Press Enter to insert it.
  10. Very important! The time will be inserted as a field that will update automatically, so you must immediately select it and press Ctrl+Shift+F9 to unlink it (convert it to ordinary text).
