    Try the horizontal scrollbar: they may be scrolled to the right.

    Alternatively, they may be hidden or zero-width.

    If you can't solve it any other way another alternative is to set the form's Default View to Continuous Form. Cut each label (Ctrl+X) and paste into the Form Header section (side-by-side.) Place the text boxes beside each other in the Detail section. Then shrink the height of the Detail section so it is only one text box high.

    This gives you something that looks a bit like a datasheet, but with several advantages:

    • Hidden/zero-width fields is not an issue.
    • You can use the Form Header and Form Footer.
    • You can use command buttons.
