suzanne s. barnhill say:

From your description, this is a protected form. It can be locked and unlocked in Word 2007 in pretty much the same way as in earlier versions once you know the secret handshake.

 

  1. Start by enabling the Developer tab. If you don't see a tab labeled "Developer" at the right end of the Ribbon, then go to Office Button | Word Options | Popular and check the box for "Show Developer tab in the Ribbon." Click OK.
  2. Select the Developer tab. You'll see that it has a group labeled Protect.
  3. In that group, click Protect Document. Don't be misled by the fact that "Restrict Permission" shows "Unrestricted Access." This is a different kind of protection. Instead, click on "Restrict Formatting and Editing."
  4. This opens a task pane that should be showing only "Stop Protection." Click that button.
  5. You'll see that the task pane now contains the controls to protect a document for forms. When you first create a form, you may need to go through this task bar to apply protection initially, but what you really want is a simple Lock/Unlock button.
  6. You'll see that there is another group labeled Controls. The bottom right button in the left portion of this group is label toolbar in previous version except the Lock button!
  7. Don't despair! Go to Office Button | Word Options | Customize, select "Commands Not in the Ribbon," and scroll down to the Lock command (with the familiar padlock icon). Click Add>> to add this to your Quick Access toolbar. You'll now be able to lock and unlock forms quickly and easily.
