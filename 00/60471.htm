Error code 0x80131500 can either prevents people from downloading preferred apps or prevents Windows Store from launching. No worries because you can follow the troubleshooting below to get rid of getting the said error.

Method 1: Reset Store cache

1. Press Windows Key + R to open the Run dialog.
2. Enter wsreset.exe and press Enter or click OK.
3. After resetting the cache try to start Windows Store again.

Method 2: Change your regional settings

1. Press Windows Key + S, enter the region and select Region from the list of results.
2. When the Region window opens go to Location tab and change the Home location.
3. After you’re done click Apply and OK to save changes.
4. Restart your computer and check if the issue still persists.

Method 3: Create a new user account

1. Open the Settings app and go to Accounts > Family & other users.
2. Scroll down to the Other user's section and click Add someone else to this PC button.
3. Select I don’t have this person’s sign-in information.
4. Choose Add a user without a Microsoft account.
5. Enter the username and password for a new user. Click Next.
6. After you’re done switch to the newly created account and check if everything works.

DMZ

Netgear CG3100D

Login Information:

  • Default Gateway - 192.168.0.1
  • Username – admin
  • Password – password

DMZ

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select DMZ Host from under the Advanced section on the left side.
  4. Check the Default DMZ Server box and enter the IP Address of your Xbox console.
  5. Click the Apply button at the bottom of the page.

Disable DMZ

  1. Login to the router by entering 192.168.0.1 in to the address bar.
  2. Enter the username “admin” and the password “password”.
  3. Select DMZ Host from under the Advanced section on the left side.
  4. Uncheck the Default DMZ Server box.
  5. Click the Apply button at the bottom of the page.
