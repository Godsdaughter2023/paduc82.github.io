The above quoted section of your battery report is troubling. The Design Capacity and Last Full Charge Capacity for that battery is at 0 (zero). This means that there are no more charging cycles left in the battery.

 

The 2 most common causes of this condition are the following.

 

1. The battery is old and has reached it's End of Life. Typically a Li ion battery will last between 1 and 3 years.

 

2. The battery has been completely discharged too many times and the circuit that is designed to prevent the battery from completely discharging has failed. Unlike the older NiCad batteries, a Li ion battery does not have a memory circuit and completely discharging one can permanently damage the charging circuit.

 

In either case, if the system is only 6 months old, you should be able to use the warranty service to get a new battery.

Hi Nicholas

The reason this error occurs is because the Service that is running is not properly handled by the system.

See this article for an explanation.

Event ID: 1530 may be logged in the Application log on a Windows 7-based or Windows Vista-based client computer:
http://support.microsoft.com/kb/947238

The service involved in your case is the Windows Live ID Sign-in Assistant service. This service is used when you sign-in to a website that requires a Live ID authentication, such as these Microsoft Answers Forums.

Open Control Panel/Administrative Tools/Services.

Scroll down to the Windows Live ID Sign-in Assistant service and double click that service to open settings dialog box.

Make sure that the Startup Type: is set to Automatic. 

 

This is a common warning message for some services and can usually be safely ignored, unless you are experiencing problems with this specific component (Windows Live ID).
