guessed say:

Solving file size issues requires you to identify where in the file the size is coming from and then addressing those elements. To work out what is driving the file size up do the following.

1. Make a copy of the document and rename it so it ends with the suffix .zip (you need to see the file extensions to do this properly)

2. Open the zip file and look around for the files with a large compressed size. This is most likely to be in the Word/Media folder

If the issue is in the media folder, you can copy all these files into a new (uncompressed) folder somewhere on your hard disk and inspect the graphics using a suitable graphics viewer. Perhaps there is something you can do to replace those particular graphics without altering the document appearance.

andreas killer say:

Am 19.10.2010 19:20, schrieb Cluckers:

So I need to go down the line and remove the lead ", " if there is one.

Click in a cell in that column and execute the macro below.

Andreas.

Sub RemoveCommaAndBlanks()
  Dim Data, I As Long
  'read in the data
  Data = Intersect(ActiveCell.EntireColumn, ActiveSheet.UsedRange)
  For I = 1 To UBound(Data)
    Do
      'remove any leading commas and blanks
      Select Case Left$(Data(I, 1), 1)
        Case ",", " "
          Data(I, 1) = Mid$(Data(I, 1), 2)
        Case Else
          Exit Do
      End Select
    Loop
  Next
  'write data back
  Intersect(ActiveCell.EntireColumn, ActiveSheet.UsedRange) = Data
End Sub

rick rothstein, mvp - excel say:

The following will remove all the leading comma/space characters for you all at once (no need to run a second loop). Change the column designation in the For..Each statement to match the column you want to process...

Sub RemoveLeadingCommasFromColumn()
  Dim Cell As Range, CellVal As String
  Application.ScreenUpdating = False
  For Each Cell In Intersect(Columns("C"), ActiveSheet.UsedRange)
    CellVal = Cell.Value
    If Len(CellVal) Then Cell.Value = Mid(CellVal, InStr(CellVal, Left(LTrim(Replace(CellVal, ",", " ")), 1)))
  Next
  Application.ScreenUpdating = True
End Sub
