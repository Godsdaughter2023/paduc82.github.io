scott .. b say:

Not sure if it is confined to IE either.  MAC is an entirely different animal. (Windows8 PC runs 64-bit).   Probably should post on a Mac forum, but just in case it could be an IE problem, some suggestions.....

 

1)  When you get a blue screen, are there any messages on it (e.g. error codes, device anomalies, etc) or does it just flash by momentarily and you are unable to see much?

 

2)  Sometimes an intermittent  blue screen can be caused by something as simple as a loose USB cord, but mostly the cause can be traced to an improper device installation that results in corrupted system files.  Have you installed any hardware recently,  such as a printer, scanner, Bluetooth keyboard/mouse, digital camera dangle, etc.   If so, try uninstalling it, checking for the blue screen, and then reinstalling it.

 

3) Have you installed any RAM sticks.  If so, and they are not Apple RAM sticks, that will cause blue screen "looping". 

 

4)  The intermittent blue screen flashing is also caused by your book logging in and then logging back out unpredictably.  Check your user login and/or maybe try creating another user profile.

 

5)  There is also something called a "Blue Screen Virus" that emulates system failure in an attempt to have the recipient logon to a fixit screen. To check for this, you should boot up in "Safe Mode" and THEN run a full virus scan, deleting any malicious files. 

 

6)  If the blue screen circumstance has just started occurring, you might want to go back to a restore point, but it is usually smarter to analyze just what YOU have done since your last restore point and try to isolate it separately, e.g. uninstall a program that didn't quite work as advertised.

joeu2004 say:

Try:

=IF(RIGHT(ROUNDDOWN(A1,3), 1) <= "5", ROUNDDOWN(A1,2), ROUNDUP(A1,2))

Note the double-quotes around "5", since RIGHT returns text and all text is greater than any number (e.g. "5">6 is TRUE).

That seems to work even if A1 has less precision (e.g. 96.000).
