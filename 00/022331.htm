steve rindsberg say:

PowerPoint, through version 2007, used JavaScript in the HTML it created so that it could preserve or duplicate as much of the native PPT functionality as possible (HTML alone doesn't take you very far in that direction).

There are third-party add-ins and programs that convert PowerPoint to HTML in various ways, even from later versions of PPT that don't any longer allow saving as HTML directly.  Perhaps one of those might give you the flexibility to create HTML then edit the resulting HTML to add your own Javascript programming to the mix.  

I have one, PPTools PPT2HTML that's pretty flexible in this regard;  the free demo doesn't expire and is fully functional.  

VBA is not exactly like VBScript is not exactly like VB is not exactly like VB.NET but there are quite a few similarities.  Knowing any one of the first three would give you a start on the other two.  VB.NET's a different animal.  And for that matter you can include VBScript in HTML as well (though it's not going to be as widely supported by browsers as Javascript is.)

Clean up duplicate contacts at https://people.live.com

You can combine duplicate contacts into a single one with all the contact info.

Use “Clean up contacts”

  1. In People, click Manage, then click Clean up contacts.

  2. Within each contact set, clear the check box for any contact you don’t want in the set.

  3. Repeat step 2 for all contact sets, and then click Clean up.

Search and combine manually

  1. In the People search field, type the name of the contact you’re looking for.

  2. In the search result, select the contacts you want to combine, and then click Link.

Undo a combined contact

When a new combined contact is created, the original duplicate contacts are then deleted. If you accidentally combine two or more contacts that are actually different people, here’s how to restore them.

  1. In People, click Manage, and then Restore deleted contacts.

  2. Select the original contacts you want to restore, then click Restore.

  3. Click the incorrectly-combined contact, click Delete, then click Delete again.
