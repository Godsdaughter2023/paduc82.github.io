charles kenyon say:

Not as a "Watermark," per se. That is, not by using Page Layout > Watermark.

Watermarks are image files, usually faded as semi-transparent, behind text. They are in the header/footer level.

Simply create your image to be the appropriate placement/size/rotation/transparency for the half where you want it and put that in the Header.

Chances are that your Tent Card template uses a Table for layout but it might be using Text Boxes or Frames.  That makes no difference so long as you are putting your image, set to be floating behind text, in your header. You want to be storing your image in the header of the template.

Do you have other images stored on this drive or is there only one (your system) system image?

If there is only a older same system image stored, it will be overwritten(1), if there are other images, your current image will be added. In the latter case you'll need enough free space to store your boot disk image, that is the sum of all partitions on this drive!

Edit: overwritten is not the correct word, only modified files will be overwritten, new files will be added and deleted files will be removed from the old image.

Try:

'=========>>
Option Explicit

'--------->>
Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
    Const vbOrange As Long = 49407
    Const myCells As String = " E31:BP45"

    If Not Intersect(Target, Me.Range(myCells)) Is Nothing Then
        Cancel = True
        With Target.Interior
            Select Case True
            Case .ColorIndex = xlNone
                .Color = vbOrange
            Case .Color = vbOrange
                .Color = vbBlue
            Case .Color = vbBlue
                .Color = vbGreen
            Case .Color = vbGreen
                .ColorIndex = xlNone
            End Select
        End With
    End If
End Sub
