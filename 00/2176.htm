Use = instead of >=

Or use this shorter formula:

=MATCH(E3,{"Strongly Disagree","Disagree","Neither Agree Nor Disagree","Agree","Strongly Agree"},0)

bernard liengme say:

To test if a cell has a specific text value we need to use just = not >=

I suspect you want number answers so remove quotes around the numbers

=IF(E3="Strongly Disagree",1, IF(E3="Disagree",2, IF(E3="Neither Agree Nor Disagree",3, IF(E3="Agree",4, IF(E3="Strongly Agree",5)))))

But you may prefer

=MATCH(E3,{"Strongly Disagree","Disagree","Neither Agree nor Disagree","Agree","Strongly Agree"},0)

or even

=LOOKUP(E3,{"Agree","Disagree","Neither agree nor Disagree","Strongly Agree","Strongly Disagree"},{4,2,3,5,1})

Note the first array must be in alphabetical order
