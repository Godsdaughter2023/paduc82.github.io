    shnickins say:

    multiply and transpose

    If I have a table of data.... 4 by 4 and want to multiply that by 4 different values (for relative weights) and then multiply it by the transposed 4 values of the relative weights... how would I do that?  Data is in B5:E8 and A1:A4.  Thanks
    tasos_k say: Hi,
    the  table will be 4 by 4.
    Extract data in range A10:D13

    in cell A10 write:
    =MMULT(A1:D1,E$5:E$8) and copy down up to A13

    in B10:
    =MMULT(A1:D1,F$5:F$8)...up to B13

    in C10:
    =MMULT(A1:D1,G$5:G$8)...C13

    in D10:
    MMULT(A1:D1,H$5:H$8)...D13

    you don't need 'transpose'.
     
    [Edit]
    or
    in cell A10 write:
    =MMULT($A1:$D1,E$5:E$8)
    select cell A10 and copy on the right up to D10
     
    select A10:D10 and copy down up to row13.

    XXXXXXXXXXXXXXXXXXXXXXXXXX

     
    symbolic:
    MMULT=([Range row from table A],[Range column from table B])

    number of columns in table A = number of rows in table B

    e.g.
    dimension table A: [3 x 4] ( 3 rows by 4 columns)
    dimension table B: [4 x 2]
    dimension table A x B [3 x 2]

    dimension table A: [3 x 4]
    dimension table B: [4 x 3]
    dimension table A x B: [3 x 3]

    dimension table A: [1 x 4]
    dimension table B: [4 x 1]
    dimension table A x B: [1 x 1] (one cell)

    dimension table A: [4 x 1]
    dimension table B: [1 x 4]
    dimension table A x B: [4 x 4]

    How Many To, Cc and Bcc Recipients are Allowed Per Message

    Again, suspicious activity can lead to a temporarily lower limit (as low as 10 (ten) recipients).

    If you’ve been notified that you have hit some limit on a Microsoft product, it’s almost certainly part of an effort to reduce spam.
    You can also increase your limit simply by continuing to use your email normally
    It only takes a few days of normal activity to increase the limits—that gives us enough time to detect automated spam machines and accounts that have been hacked.
    We’re constantly working to make things worse for spammers and better for real people.

    The send mail limit depends on your reputation within Outlook.com
    So, there are stricter limitations on new accounts or accounts that have engaged in activity that is representative of abusers sending spam.

    If you are a new user, a low sending quota is a temporary restriction and is upgraded to the maximum limit as soon as the user establishes credibility in the system.
    If you are an established user, recent account activity may have sent us a false alert.
