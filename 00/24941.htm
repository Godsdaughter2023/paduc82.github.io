marvin ma msft say:

Hi RickFlashIT,

Based on my knowledge, if you'd like to migrate the users from source Office 365 tenant to the target, you need to migrate all users' mailbox data of the source tenant to the target. And before you migrate, you need extra licenses to create the same amount of active mailboxes in target tenant for preparation of migration. So to your question, the answer is yes because it's not to merge licenses between two tenants, but is to migrate data from one tenant to another. For your reference, see How to migrate mailboxes from one Office 365 tenant to another. Besides, since there're few users in these two companies, you can use network upload to import the PST files to the target tenant.

Let me know if you have further questions on this issue.
