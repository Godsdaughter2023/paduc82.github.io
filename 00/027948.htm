See also Automated Boilerplate Using Microsoft Word

Your boss has assigned you a big task. There are a number of ways to go about it. Start by getting all of the clauses that your boss thinks could be used. Organize them. Then you can save those as AutoText entries in a Template for the contract. That AutoText can then be inserted by typing the name of the entry and pressing F3, or using an AutoTextList field, or by using a userform checklist.

You will probably also want to have some fill-in-the-blank areas where information is typed in and then copied to other parts of the document. Repeating Data (Populating Multiple Like Fields) The components for that can be a part of your AutoText entries.

As I said, this is a big project. I would expect to devote a minimum of two weeks to it if you are talking about any kind of comprehensive contract. Learning as you go will extend this time. Make sure your boss knows what he/she is asking you to do. It is probably worth spending the time and will take less time the second time you do it.

A two-page contract with three or four possible clauses will take considerably less time, but doing it with a userform will still take a lot of time for learning the first time.

http://www.vbaexpress.com/forum/showthread.php?57631-save-selected-emails-on-my-hard-drive-with-many-option&highlight=Save+messages

That code is for saving a selection of messages. For a folder you also need

Sub SaveFolderMessages()
'Graham Mayor - http://www.gmayor.com - Last updated - 04 Nov 2017
Dim olFolder As Folder
Dim olItem As MailItem
Dim iCount As Integer
    On Error GoTo NextItem
    Set olFolder = Session.PickFolder
    For iCount = 1 To olFolder.Items.Count
        Set olItem = olFolder.Items(iCount)
        If olItem.Class = OlObjectClass.olMail Then
            SaveItem olItem
        End If
NextItem:
    Next iCount
lbl_Exit:
    Set olItem = Nothing
    Set olFolder = Nothing
    MsgBox "Process finished"
    Exit Sub
End Sub

mike middleton say: Regarding "how do I reinstall Solver?"

To uninstall, Exit Excel, Start > Control Panel > Programs and Features > Microsoft Office ... 2010 > Change > Add or Remove Features > Microsoft Excel > Add-ins > Solver > Not Available > Continue

To verify, start Excel, respond to message about missing Solver file, choose File > Options > Add-Ins > Manage Excel Add-ins > Go > select Solver from Add-Ins Available > respond to message.

To reinstall, the same steps as above, i.e., Exit Excel, Start > Control Panel > etc., except for Solver > Run from my computer > Continue

-  Mike Middleton, www.MikeMiddleton.com
