Often times it comes up as to whether or not "LED notifications" are supported on the Windows Phone platform.   The response to this is straightforward and can be answered in two bullet points:

  • Win10 mobile does and will support LED notifications
  • Device manufacturers (OEMs) need to build devices that support this functionality in order to take advantage of it

bernd p. say:

Closing the notebook just closes the file from within OneNote.

That's misleading.

A Onenote notebook is a folder at the Windows file system level. That folder contains *.one files for the sections in the notebook. At least one. These are the files that are closed.

If you want to reopen a notebook you have to navigate to a folder of the Windows file system with the name of your notebook.

This unusual concept in Onenote (which I not critizize) is a permanent source of confusion for the unexperienced user.
