Hi John:

I've used Quicken for about 15 years now.  Intuit puts out a new edition every year.

My experience is that it is not worth upgrading every year, but that it is worthwhile to upgrade to the newest Quicken edition once you have done two Operating System updates.  In other words, if you bought your present copy of Quicken back when XP was the current OS, about 4 OS updates have taken place since then (XP -> Vista -> Windows 7-> Windows 8).  The release of Windows 10 will be the fifth OS update.

The newer versions of Quicken can continue working with files that you have created using older versions of Quicken.  But, generally speaking, once you switch over to a newer version of Quicken, it updates the file structure, which means you can't go back to opening that file with an older Quicken version.

All things considered, I suggest you go buy the current version of Quicken now (while you are still running Windows 7), install it, get up and running with the new Quicken version first, then do the OS upgrade to Windows 10 after you have completed the Quicken update and everything is running OK with your finances.
