Even if you don't want to compress them less than 300 ppi, you can and should compress them...or should I say, resize them.

If you insert the image and resize the picture box to less than 100% scale then you can reduce the file size significantly by resampling the image and reducing the dimensions of the images. Leave the target output the same, at commercial which is 300 ppi, but under the compression options mark choose to delete cropped areas of the pictures and to resample. Save a copy of your original Pub file and test it on that and then check to confirm that you are not going to mess up the final resolution. You will find you can reduce the file size of your Pub file to a fraction of what it is now and probably fix the problems you are having with the larger than needed picture files.

I doubt if this will create a problem but if it does then resample and optimize your images in a third party image editor before you insert them. In my opinion this is the best approach so that you are not relying on the built-in image editing capability of Publisher. Private Declare Function GetSysColor Lib "user32" (ByVal nIndex As Long) As Long

Sub TestChooseColor()
  Dim strColor As String
  Dim lngRGB As Long
  Dim bytR As Byte
  Dim bytG As Byte
  Dim bytB As Byte

  lngRGB = ColorDialog()

  'Überprüfen, ob Fehlerwert zurückgeliefert wurde
  If lngRGB = &HFFFFFFFF Then Exit Sub

  'In einen Hexstring mit den letzten 6 Stellen umwandeln
  strColor = String(6 - Len(Hex(lngRGB)), _
      Asc("0")) & Hex(lngRGB)

  'Die einzelnen Farbanteile extrahieren
  bytR = CByte("&H" & Right(strColor, 2))
  bytG = CByte("&H" & Mid(strColor, 3, 2))
  bytB = CByte("&H" & Left(strColor, 2))

  MsgBox "RGB-Farbe = " & lngRGB & " (" & strColor & ")" & _
      vbCrLf & "Rotanteil = " & bytR & _
      vbCrLf & "Grünanteil = " & bytG & _
      vbCrLf & "Blauanteil = " & bytB
End Sub

Public Function ColorDialog() As Long
  Dim udtChoosecolor As CHOOSECOLOR
  Dim alngCustomColors(1 To 16) As Long
  Dim I

  ' Funktion mit Fehlerwert vorbelegen
  ColorDialog = &HFFFFFFFF

  'Die benutzerdefinierten Farben vorbelegen
  alngCustomColors(1) = RGB(255, 0, 0)    'Rot
  alngCustomColors(2) = RGB(0, 255, 0)    'Grün
  alngCustomColors(3) = RGB(0, 0, 255)    'Blau
  alngCustomColors(4) = RGB(255, 255, 255)    'Weiß
  alngCustomColors(5) = RGB(0, 0, 0)    'Schwarz
  alngCustomColors(6) = RGB(255, 0, 0)    'Rot
  alngCustomColors(7) = RGB(0, 255, 0)    'Grün
  alngCustomColors(8) = RGB(0, 0, 255)    'Blau
  alngCustomColors(9) = RGB(255, 255, 255)    'Weiß
  alngCustomColors(10) = RGB(0, 0, 0)    'Schwarz
  alngCustomColors(11) = RGB(255, 0, 0)    'Rot
  alngCustomColors(12) = RGB(0, 255, 0)    'Grün
  alngCustomColors(13) = RGB(0, 0, 255)    'Blau
  alngCustomColors(14) = RGB(255, 255, 255)    'Weiß
  alngCustomColors(15) = RGB(0, 0, 0)    'Schwarz
  alngCustomColors(16) = RGB(255, 0, 0)    'Rot
 
  For I = 1 To 16
    alngCustomColors(I) = GetSysColor(I + 2)
  Next

  ' Zeiger als Wert auf die benutzerdefinierten Farben
  udtChoosecolor.lpCustColors = VarPtr(alngCustomColors(1))

  ' Größe der Struktur in Bytes
  udtChoosecolor.lStructSize = Len(udtChoosecolor)

  'Den Dialog aufrufen
  If CHOOSEMYCOLOR(udtChoosecolor) <> 0 Then
    ' Den gewählten RGB Wert zurückgeben
    ColorDialog = udtChoosecolor.rgbResult
  End If
End Function

IM should work over wifi. To switch off wifi, open Comm Manager in Start - Programs. Locate the wifi icon and tap to disable.

 

Your device is wifi-enabled and can access the internet when within range of a wireless network or hotspot.
