Why use a Network Diagram view for this?  I think a Timeline view would work way better.  To use a Timeline view, try the following:

  1. Apply the Gantt with Timeline view.
  2. In the Data section of the View ribbon, click the Outline pick list button and select the Level 1 item to display only the highest-level summary tasks.  I think these are the major summary tasks about which you should report.

I have learned the hard way not to save / close the workbook in the BeforeClose and BeforeSave procedures.  I interpret those event procedures as occuring after Excel has already started its own internal process to close/save the workbook and trying to sneak around Excel is not a good idea.

Specifically, the suggestions to use the beforeclose procedure to close the workbook without saving changes crashes Excel 2010!

A lot simpler is to cooperate with Excel.  The code below allows the consumer to close a workbook w/o any prompt about an unsaved workbook.  It also lets the consumer save the workbook as another name but not overwrite the file on disk.

The real challenge is how does one save a workbook after the beforesave code in put in it?  After all, it won't allow the workbook to be saved! {veg}

Option Explicit

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    ThisWorkbook.Saved = True
    End Sub

Private Sub Workbook_BeforeSave(ByVal SaveAsUI As Boolean, Cancel As Boolean)
    Cancel = Not SaveAsUI
    If Cancel Then MsgBox Prompt:="Sorry, you may not save your changes.", Buttons:=0
    End Sub

There is no provision in Word to simply scale the document content to fit a different paper size. As Daniel described, you can select whatever paper size you wish, but the content & layout will have to be adjusted manually. daniel g. grau say:
In the main menu bar, select File > Page Setup and choose the proper sheet size.

To change the point size manually, select the whole content (Select All or Command A) and apply a smaller font size to the complete document, or select portions of text as needed.

To do it through styles (as long as you have not manually changed the sizes), in the main menu bar select Format > Style, then click on Normal, Modify and change the font size. If you are using other styles, you may need to change them as well (this is not always so because styles can be based on other styles and the font-size changes may already have propagated).

If you need to change the margins, select Format > Document > Margins and redefine them.
