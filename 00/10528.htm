cmdrkeene say:

It depends on your system manufacturer.  The computer company that puts together the laptop gets to decide what components to use, and which touchpad supplier they bought the part from.  That other (4th party) supplier is the company that built the touchpad control driver software, so it's up to them to determine if they allow customization of the scrolling inertia function, or even if they have customizable options of any kind.

In short: Microsoft doesn't build touchpads or the software for them.  Neither do most computer manufacturers like Dell or HP.  They just build it.  Depending on who they used to supply the parts, they may or may not have the ability to change what you want to change.

The only way to know is to contact the system manufacturer and inquire, or check the owner's manual.

bob jones aka: cybertaz say:

Yes, the Ribbon in your screen shot is that of a Windows version. The tool appears as a singular Insert button because the window shown is not sized sufficiently wide to allow the group of tools to expand... It's one of those that expands/condenses based on available space. If the window were wider the singular tool would grow to display as a Group containing the 3 categories of controls tools: Content Controls, ActiveX Controls, Legacy Form Fields.

Since Mac Office provides only Legacy Form Fields there is only the one small group to be displayed. The window typically is wide enough to display the entire group, but will condense if the width of the window is narrowed significantly.

The Shrink to Fit command is available on Word 2002, 2003 and 2007, how do I find this command in Word 2010?
tammy.m say:

In Word 2010, Shrink to Fit command is known as Shrink One Page. It is not available on the ribbon or the quick access toolbar. However, you can customize the ribbon and add the command “Shrink One Page” to a new or existing group.

 

In Word 2010, you can add the Shrink to fit option by adding the Shrink One Page command to the Customize Ribbon.   To do this, follow these steps:

  1. Start Word.
  2. Click the File tab, click Options, and then click Customize Ribbon.
  3. In the Choose commands from drop-down list, select All Commands.
  4. Search for Shrink One Page, and then click it.
  5. Under Customize the ribbon, click the custom group where you want to add the command. The command can only be added to custom groups. To add a custom group, click the tab where you want the group to appear, and then click New Group.
  6. If you prefer to add the new group to a new tab, click the New Tab button which automatically creates a new group to put the command in.
  7. The new group is created at the end of the selected tab. You can change the new group name by clicking the Rename button.
  8. Click Add button, and then click OK.

 

To see detailed information on how to add this command, please go to this Microsoft knowledge base article:

2664217 –Shrink to Fit Option in Word 2010
