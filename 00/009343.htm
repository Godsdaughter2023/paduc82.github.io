jllatham say:

That's actually easy to do with the code I posted before by changing .CountA to .Sum  Sum treats empty cells and cells with text in them as 0 so you only get a non-zero result if there is a non-zero numeric value in a cell.  I took your sample workbook and changed the $4.00 entry in column E to become $0.00 and then columns D+J, E+K became hidden.

Sub Review()
  Dim FinalRow As Long
  FinalRow = Range("A" & Rows.Count).End(xlUp).Row
 
  If Application.WorksheetFunction.Sum(Range("C2:C" & FinalRow)) = 0 Then
    Columns("C").EntireColumn.Hidden = True
    Columns("I").EntireColumn.Hidden = True
  End If
  If Application.WorksheetFunction.Sum(Range("D2:D" & FinalRow)) = 0 Then
    Columns("D").EntireColumn.Hidden = True
    Columns("J").EntireColumn.Hidden = True
  End If
  If Application.WorksheetFunction.Sum(Range("E2:E" & FinalRow)) = 0 Then
    Columns("E").EntireColumn.Hidden = True
    Columns("K").EntireColumn.Hidden = True
  End If
  If Application.WorksheetFunction.Sum(Range("F2:F" & FinalRow)) = 0 Then
    Columns("F").EntireColumn.Hidden = True
    Columns("L").EntireColumn.Hidden = True
  End If
End Sub

jllatham say:

We could have a pre-processor routine to check the contents of any of those columns and if it finds a value less than zero, pop up a message and tell where the problem is and simply exit the process without doing anything else.  In point of fact, we could use an idea from Norman David Jones to help with that.  Off to figure it out, I'll edit this post if/when I come up with something worthwhile.

[EDIT]  Ok new code added to do the pre-testing for negative values in C, D, E or F used range:

Sub Review()
'note that the 'data sheet' must be selected when
'this process is run.

  Dim testRange As Range
  Dim anyCell As Range
  Dim colPtr As Integer
 
  FinalRow = Range("A" & Rows.Count).End(xlUp).Row
'added "pre-processing" to ensure no negative values in the 4 primary columns
'this checks columns C, D, E and F
  For colPtr = Range("C" & 1).Column To Range("F" & 1).Column
    Set testRange = Range(Cells(2, colPtr), Cells(FinalRow, colPtr))
    If Application.WorksheetFunction.Min(testRange) < 0 Then
      'there is a negative value - identify the first one
      For Each anyCell In testRange
        If anyCell < 0 Then
          MsgBox "Negative value at cell: " & anyCell.Address, _
           vbOKOnly + vbCritical, "Processing Halted"
          'go to the problem cell
          anyCell.Select
          Set testRange = Nothing
          Exit Sub
        End If
      Next
    End If
  Next ' end colPtr loop
 
  Application.ScreenUpdating = False
  If Application.WorksheetFunction.Sum(testRange) = 0 Then
    Columns("C").EntireColumn.Hidden = True
    Columns("I").EntireColumn.Hidden = True
  Else
    'bring them back into view if they were previously hidden
    Columns("C").EntireColumn.Hidden = False
    Columns("I").EntireColumn.Hidden = False
  End If
  If Application.WorksheetFunction.Sum(Range("D2:D" & FinalRow)) = 0 Then
    Columns("D").EntireColumn.Hidden = True
    Columns("J").EntireColumn.Hidden = True
  Else
    'bring them back into view if they were previously hidden
    Columns("D").EntireColumn.Hidden = False
    Columns("J").EntireColumn.Hidden = False
  End If
  If Application.WorksheetFunction.Sum(Range("E2:E" & FinalRow)) = 0 Then
    Columns("E").EntireColumn.Hidden = True
    Columns("K").EntireColumn.Hidden = True
  Else
    'bring them back into view if they were previously hidden
    Columns("E").EntireColumn.Hidden = False
    Columns("K").EntireColumn.Hidden = False
  End If
  If Application.WorksheetFunction.Sum(Range("F2:F" & FinalRow)) = 0 Then
    Columns("F").EntireColumn.Hidden = True
    Columns("L").EntireColumn.Hidden = True
  Else
    'bring them back into view if they were previously hidden
    Columns("F").EntireColumn.Hidden = False
    Columns("L").EntireColumn.Hidden = False
  End If
End Sub

mistert---01 say: Hi fabiocer

Long-pressing # key toggles between Silent and General profile provided that your operator doesn't use this key for selecting the alternative line. Orange use the key for alternative line switching, so the shortcut to Silent profile isn't available. Several other operators do the same. In those cases, the quickest way of selecting Silent profile is by short-pressing the power key and selecting it from the list.

There's no point in taking it to a service point, as they won't be able to do anything for you... this is the way the phone was designed for these operators.

Btw - "long-press" in this context means approx 0.5 seconds.
