Hi Sara,

The name of the feature you are referring to is called Web Query. Unlike the Windows version of Excel, In Excel on the Mac you can’t get to the Web Query feature using the Ribbon. In both Mac and Windows Excel it is not obvious that the name of the feature you are referring to is Web Query. 

The Web Query feature does not involve ODBC or external drivers. There’s nothing extra to buy.

Excel comes with a built-in web query that might be what you are looking for regarding currency exchange rates. These easy steps will demonstrate the web query. Your Mac must have a live internet connection in order for this query to work:
  1. In Excel, start with a new worksheet
  2. On the Data menu choose Get External Data > Run Saved Query
  3. The Choose A Query dialog opens
  4. Select MSN MoneyCentral Currencies and then click the Get Data button
  5. The Returning External Data to Microsoft Excel dialog opens
  6. Click the OK button
Excel will then fetch up to the minute currency exchange rates from Money Central.

Ok, I have actually answered this one before.  But I'll put this out there again.

As the WM Client is NO MORE than a "Sync" software with E-Mail client Tendencies.  This means that if it is on the server, it will be on your phone.  There are 3 things you could do to fix this.

1.  On your PC, Turn off "Delete Messages from server after download"
- This way, the files would always be there and therefore, never "disappear"
- Downside: most E-Mail servers have a Storage limit and is possible to reach this, depending on your daily load

2. Set up a Back up E-mail  and forward to this server
- I would Suggest Gmail (Google Mail). They have insane amounts of storage for each of their members. I know, I have one.
- DO NOT set up your PC E-Mail Client to Download Messages from the Back-Up E-Mail. Instead access ONLY from a web browser.
- Most E-Mail Clients have either "Rules" or "Filters" that you can set up. Set up a Rule/Filter that forwards the wanted E-Mails to the Back Up E-Mail. By "Wanted" I mean either ALL or only those from a specified domain or e-mail, etc.
- Set up your WM E-Mail Client to Download from the Back up E-Mail. Be sure to Tell it to Downlaod ALL Messages.
- As you are not Downloading/deleting it from the Back-Up Server, they will always be there.
- The only Downside, though small, is that it may be an extra 10-15 minutes before it show up in your Back-up email, after you download it from the Initial Server. As you have do d/l it, the rules/filters process the emails and resend them, it has to find it's way to the new server, all the while, being scanned by various servers for viruses, etc.
3. SYNC
- This is probably the simpilest, but is also, sometimes a pain.
- Open your WM Mail Client. It should open on the "Select an Account" page (listing "Text Messaging" "OutLook E-Mail" "<Whatever Mail Server you have set up>" and "Setup E-Mail")
- Select [Menu] -> [Options]
- Hightlight /press "OutLook E-mail"
- Set the Features the way you want (Download the Past: "ALL", Message Format, Size Limit, etc)
- Sync your WM Device to your computer.
- This way ALL MESSAGES that have downloaded to your Microsoft Outlook will be SYNCed into your phone.
- Downside: If you only want, for instance, your work e-mails, and you download them and your personnal E-mails to that computer, you get THEM ALL. Also, after SYNCing, all messages Deleted off of the computer you sync with, will be deleted off of your WM Device as well.


I hope one of these solutions help you....
