The only way to avoid reinstalling programs is to image the drive. Connect the new hard drive as a slave (if IDE), then partition the new hard drive so it has two partitions (or more if you want) - the last partition being a little bigger then the amount of

How to upgrade Windows xp 32 bit to Windows 8 64 bit or How to create - Microsoft Community

tom264 say:

How to upgrade Windows xp 32 bit to Windows 8 64 bit or How to create media for ISO install

I thought I would just post the information in a new thread since the actual answers to this question are sometimes buried in numerous pages of things that didn't work or other me too comments.  

First, a few points: 

1) If you want to upgrade to Windows 8 using the current installer download from MS, you will automatically download the version of Windows 8 that corresponds to the version of Windows on the machine you are downloading from.  That means if you are downloading onto a 32 bit machine, you will get 32 bit Win 8, if it's 64 bit machine, you will get Win 8 64 bit.

2) The option to create media does not appear to work in Windows xp (for me this proved true for windows xp 32 bit and windows pro 64 bit)

3) If you want to upgrade from a 32 bit version to a 64 bit version you must do this from a bootable DVD or USB, it cannot be done from withing windows using the installer.

If you have either of these problems this is what you can do to get Windows 8 working properly and also create your bootable media.  Note that I did this all starting from a 32 bit version of Win XP running SP3.   If you have access to a windows 7 64 bit machine you can download using the installer from MS and should have the option to create media on DVD or USB and simply use that to install Win 8 64 bit on the machine you are trying to upgrade.  
1) From Windows XP, download the Windows 8 Release Preview 64 bit ISO from a 3rd party site (no longer available direct from MS).  This can be found in numerous places such as http://www.ijailbreak.com/microsoft/windows-8-release-preview-32-bit-64-bit-download/ 
2) Download Windows 7 DVD/USB download tool available from MS.  Use that software to create your bootable media/ISO file for the Windows 8 RP.  You may get an error that the files were copied successfully but could not make the device bootable.  In that case, download 32-bit bootsect and follow the instructions found here http://social.technet.microsoft.com/Forums/en-US/w8itproinstall/thread/ffdb9465-8f63-4304-8af4-38a17ce023fb
3) Reboot your machine and install Win 8 RP from the disc/USB.  You now have a working version of Win 8 64 bit.  
4) Use the upgrade assistant to download your official version of Windows 8 pro.  You will get the 64 bit version although it won't tell you anywhere which version you are getting.  

Once you've completed the download you should have the option now to create media.  You can either install directly or create media for later use depending on your preference.  For me I made a bootable USB and formatted my hard drive from the custom install option to get rid of any remnants of Win XP and Win 8 RP.  Hope this helps to put all the info in one place.  MS is also supposedly working on an update to the Upgrade Assistant which will allow you to do all of this without installing Win 8 RP, but for now this is the only option other than buying the retail version or buying the DVDs for an extra $15.
