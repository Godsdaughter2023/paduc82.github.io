If your computer is having problems viewing computers or shared files in your HomeGroup, try using the HomeGroup troubleshooter to fix the problem. 1. Open Control Panel by right click on Start Button.

1. Open Control Panel by right click on Start Button.

2. Click on Network and Sharing center and select HomeGroup option at the bottom of the left panel.

3. Select Change advanced sharing settings... option and select Private (current profile).

4. Select the option Turn on network discovery and check the option Turn on automatic setup of network

    connected-devices under Network discovery.

5. Click on Save changes and check if the issue is resolved, if not restart the system.
