OK, you're breaking all sorts of rules there... Selecting left right and centre.... Goto this, goto that....   not good ways to make a macro run quickly.  And as for making it easy to read... what's with all the double negatives in the If statements??  Don't worry, we all start somewhere.

This is completely not tested, and I fully expect someone to come up with an even more efficient way of doing this, but my first step was just to try and figure out what your macro does!  Please try this and let us know if it does the same as what your macro does.

 

Option Explicit

Sub Both3_Remove_Blank_Rows()
    Dim EndRow As Long, EndTop As Long
    Dim r As Range, c As Range
   
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual
   
    EndRow = Cells(ActiveSheet.Rows.Count, 14).End(xlUp).Row
    Do Until EndRow < 2
        If Cells(EndRow, 16).Value = "" Then
            EndTop = EndRow
            Do While Cells(EndTop, 16).Value = "" And EndTop > 1
                EndTop = EndTop - 1
            Loop
            Rows(EndTop + 1 & ":" & EndRow).Delete Shift:=xlUp
            EndRow = EndTop
        End If
        
         'No_Find:
        Set r = Range(Cells(EndRow, 1), Cells(EndRow, 15))
        If Application.WorksheetFunction.CountA(r) = 0 Then 'All cells are blank
            r.EntireRow.Delete
        Else

                r.EntireRow.Delete

        End If
        
         'No_Find3:
        EndRow = EndRow - 1
    Loop
    Application.Calculation = xlCalculationAutomatic
    Application.ScreenUpdating = True
End Sub

 

You should very rarely have to Select any cells to do things in a macro.  This is the biggest thing that can slow down code.

Use "Option Explicit" at the top of all your modules.  It forces you to declare all your variables.  This is a good thing.

I left the "No_Find" labels in as comments so you can see which bits correspond to which bits in your code.

ronrosenfeld say:

Your routine seems complicated, and I have not puzzled through all the logic.  But, if all you want to do is delete rows that are entirely blank, try:

===========================
Option Explicit
Sub DelBlnkRws()
    Dim r As Range
    Dim rw As Range
    Dim rBlankRows As Range
Set r = ActiveSheet.UsedRange
Set r = r.Columns(1).SpecialCells(xlCellTypeBlanks)

For Each rw In r
    With rw.EntireRow
    If WorksheetFunction.CountA(.Cells) = 0 Then
        If Not rBlankRows Is Nothing Then
            Set rBlankRows = Union(rBlankRows, .Cells)
        Else
            Set rBlankRows = .Cells
        End If
    End If
    End With
Next rw

If Not rBlankRows Is Nothing Then
    rBlankRows.Delete (xlShiftUp)
End If

   
End Sub
==================================

If you want to also delete other rows which may not be entirely blank, you can add a clause (OR) to the IF statement where we set up the rBlankRows range.

In the routine above:

  • UsedRange determines maximum area to check
  • BlankCells in Column A narrows that range down further
  • COUNTA to Check each row where Column A is blank to see if there is any content.
        UNION each blank row to create rBlankRows
  • Delete rBlankRows (data below moves up)
