I had this question too, but none of the answers they gave are correct for Windows 10 Home. So I decided to solve this myself. From what I can tell Microsoft forces you to use Windows Defender unless you have another Antivirus program in the Home Edition.

To stop Windows Defender and keep it stopped you'll have to use a workaround by setting up a Scheduled Task to run a Powershell script. If you're a more advanced user, apologies for the over explanation. 

  1. Click Start and type Task Scheduler
  2. Click on Task Scheduler Library on the left. 
  3. Click Create Task on the right
  4. Give it a name like, "Shut Off WinDefend" 
  5. Check Run with highest privileges and Hidden. 
  6. Select Run whether user is logged on or not
  7. Go to the Triggers Tab and press New
  8. Change the Drop Down to say "At log on" 
  9. In Advanced Settings below choose Repeat task every 1 hour for a duration of 1 day and then press OK. 
  10. Go to Actions and press New...
  11. Leave the drop down on Start a program and paste in: %SystemRoot%\system32\WindowsPowerShell\v1.0\powershell.exe.
  12. In the Add Arguments (optional) line add:  Set-MpPreference -DisableRealtimeMonitoring $true
  13. Press OK at the New Action window then again at the Create Task window
  14. When/if prompted enter your Wi

    You are done. Next time you log in your computer will run PowerShell hidden in the background as your user and then execute the command to shut off Windows Defender. It then reruns that command every 1 hour to make sure it stays off.

    All that is left is to make sure the task works. So first make sure Windows Defender is ON. Now in Task Scheduler right click your task and choose run. After a second or two Windows Defender should shut off and stay off for good. 

    Hope this helps!
