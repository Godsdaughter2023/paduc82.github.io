roady [mvp] say:

Due to various changes in the ost-format between these versions (like quite a reduction in size), the ost-file indeed needs to be reached to take advantage of this.

To prevent Outlook from doing this and force it to use the old format, you can set the following Policy setting;

User Configuration-> Administrative Templates-> Microsoft Outlook 2016-> Account Settings-> Exchange-> enable: Do not create new OST file on upgrade

For more info about using Group Policies see;

https://www.howto-outlook.com/howto/policies.htm
