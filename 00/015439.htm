Assuming that the 4 digit time is in cell A2. Insert a helper column beside column A (will become new column B).

  • Use Number format to format the new column B in the required time format.
  • Insert the following formula in the new cell B2.

         =TIMEVALUE(LEFT(TEXT(A2,"0000"),2) & ":"&RIGHT(TEXT(A2,"0000"),2))

  • Copy the formula down for the length of the data in column A
  • Select all of column B and Copy -> PasteSpecial -> Values (Ensure you do this before next step)

You do not need to disable the options afterwards, once they are enabled during any given session (while you have that workbook open) they remain enabled and you can't disable them.  However, if your security settings are normal you will be prompted to enable macros everytime you open the workbook.

Hi,

When you click the first Option button it opens a window which may have many security warnings, this is to let you decide which if any you want to enable.  If you file has Links to other files (formulas that reference another workbook) you will be warned, if you have Macros you will be warned, and if you are using something like one of the Data Connections on the Data tab, such as being connected to Access, SQL Server, a CUBE you will be warned.
