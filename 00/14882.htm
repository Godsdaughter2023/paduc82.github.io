jllatham say:

Instructions for inserting the code are much the same as before: 

To put this code into a workbook for testing, open the workbook, choose a sheet in it, RIGHT-click on the sheet's name tab and choose [View Code] from the list that pops up.  Copy the code below and paste it into the code module presented to you.  Just delete the previous code in that module and replace it with the code below.

Here is new code that should work with your worksheet layout.  Remember, replace the previous code I provided with this:

Private Sub Worksheet_Change(ByVal Target As Range)
  Const keyPhrase = "INSPECTOR'S NAME" ' all uppercase for a purpose
  Dim LC As Integer ' Loop Counter
  Dim validatedCells(1 To 7) As String ' hold addresses of data validated cells
  Dim datedCells(1 To 7) As String ' will hold corresponding cells for the date
  'fill the validatedCells() and datedCells() addresses into the array
  validatedCells(1) = "$C$16"
    datedCells(1) = "$K$16"
  validatedCells(2) = "$C$23"
    datedCells(2) = "$G$23"
  validatedCells(3) = "$C$29"
    datedCells(3) = "$G$29"
  validatedCells(4) = "$O$31"
    datedCells(4) = "$O$32"
  validatedCells(5) = "$C$33"
    datedCells(5) = "$G34"
  validatedCells(6) = "$H$36"
    datedCells(6) = "$N$36"
  validatedCells(7) = "$C$45"
    datedCells(7) = "$P$45"
  If Target.Cells.Count > 1 Then
    Exit Sub
  End If
  For LC = LBound(validatedCells) To UBound(validatedCells)
    If Target.Address = validatedCells(LC) Then
      If IsEmpty(Target) Then
        'clear date because they cleared name entry
        Application.EnableEvents = False
        '  If you want to leave a date in the cells
        '  once it is entered even if someone deletes or
        '  changes the NAME entries, then
        '  just delete this next line of code.
        Range(datedCells(LC)).ClearContents
        'and done
        Application.EnableEvents = True
        Exit Sub
      Else
        'something in the cell, is it a name or
        'is it our keyPhrase"?
        'if not keyPhrase then put date in proper cell
        If UCase(Trim(Target)) <> keyPhrase Then
          'presumed to be a name, enter a date
          Application.EnableEvents = False
          Range(datedCells(LC)) = Now()
          'and we are done
          Application.EnableEvents = True
          Exit Sub
        End If
      End If
    End If
  Next
End Sub

How to combine multiple sheets in a Workbook into 1 sheet - Microsoft Community

trinaz say:

How to combine multiple sheets in a Workbook into 1 sheet

I  have a workbook with about 40 separate sheets...is there a way or an app that will combine all 40 "sheets" into one sheet in one operation into the first sheet in the workbook ?

Would like to have some type of divider between the rows of the combined sheets ?

 Don't want to combine separate workbooks....but multiple "sheets" in the same workbook

Thansk....TRinaz

tintombstone say:

THis should do the trick.  It assumes that the data on each sheet starts in cell A1, change as required

 

Sub CombineSheets()
Dim sCount As Integer
Dim oSheet As Worksheet

    For Each oSheet In ActiveWorkbook.Worksheets
        If oSheet.Name = "Combined Data" Then
            Application.DisplayAlerts = False
            oSheet.Delete
            Application.DisplayAlerts = True
            Exit For
        End If
    Next oSheet

    ActiveWorkbook.Worksheets.Add Sheets(1)
    ActiveSheet.Name = "Combined Data"
   
    For sCount = 2 To Sheets.Count
        If Sheets(sCount).Range("A1").CurrentRegion.Rows.Count > 1 Then
            Sheets(sCount).Range("A1").CurrentRegion.Copy
            ActiveCell.Value = Sheets(sCount).Name & " Data"
            ActiveCell.Offset(1).Select
            ActiveSheet.Paste
            ActiveCell.Offset(ActiveCell.CurrentRegion.Rows.Count + 2).Select
        End If
    Next sCount

End Sub

suzanne s. barnhill say:

If your printer can print ledger-size paper, then it might be more satisfactory to use the "2 pages per sheet" setting in Page Setup (on the Margins tab, under "Multiple pages"). This will allow you to set up the document as Letter-sized pages (with the paper size set to 11" x 17") and work with it transparently. When you print, two pages will print on a single sheet.

US paper sizes, unlike European A sizes, don't scale well because they don't have the same aspect ratio. For this reason, you'll get better results using "2 pages per sheet" in Page Setup rather than in the Print dialog.
