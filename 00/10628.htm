Hi,

 

Try this, note I'm not looking at the dates but simply summing columns B&C based upon Col A

 

=SUMPRODUCT((A2:A20="Marketing")*(B2:C20))

 

Sorry no double unary(--) it isn't necessary but what it does is coerce TRUE or FALSE in 1 or Zero. To see it in action try this

 

Put TRUE in A1

=--A1

 

and you'll see it returns 1. We sometimes need this because SUMPRODUCT like to work with numbers and not text.
