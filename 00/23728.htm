ken sheridan say:
I need to create a Query that selects events for a given client that have not happened since 7/1/2012 but has occurred between 3/1/2013 and 3/31/2013.  

Don't you just need this:

SELECT *
FROM ProvidedServices AS PS1
WHERE PS1.ServiceDate BETWEEN #2013-03-01# AND #2013-03-31# 
AND NOT EXISTS
     (SELECT *
      FROM ProvidedServices AS PS2
      WHERE PS2.FamilyID = PS1.FamilyID
      AND PS2.ServiceDate BETWEEN #2013-07-01# AND #2013-02-28# );

NB:  all ServiceDate values must be integral, i.e not having a non-zero time of day.  Enforce this with a validation rule in the table design:

DateValue([ServiceDate])
