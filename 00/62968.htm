CG Omega is an Agfa font and was available in TrueType and as a PostScript Type 1 font. One possible source was with printer driver software for certain HP Laserjets.

 

If in TrueType format (font file has a ttf extension), there will be at least eight files, one each for regular, italic, bold, bold italic.

 

Create a folder/directory off the root of one of the hard drives, say c:\xxxfonts. Copy the ttf files to that directory. When the copy completes, right mouse click on the file name. A context menu will open. Select Install from the menu.

 

If in PostScript Type 1 format, there might be two files for each style (regular, italic, bold, bold italic), one with a pfb extension, the other pfm.
