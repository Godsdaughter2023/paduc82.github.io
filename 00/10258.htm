Sub FilterByColor()
    Application.ScreenUpdating = False
    Dim cel As Range, rng As Range
    Set rng = Sheet1.Range("A1", Sheet1.Range("A65536").End(xlUp).Offset(, 1))
    For Each cel In rng

      cel.EntireRow.Hidden = True
      End If
   
    Next cel

End Sub

Finding Colors

The downloadable module contains a function named RangeOfColor that will return a Range object consisting of the cells in an input range that have a font or fill color index equal to the specified color index. The function declaration is:

Function RangeOfColor(TestRange As Range, _ ColorIndex As Long, Optional OfText As Boolean = False) As Range

You can use this function to get a range of cells with a red fill color. For example,

Sub AAA() Dim R As Range Dim RR As Range Set RR = RangeOfColor(TestRange:=Range("A1:F20"), _ If Not RR Is Nothing Then For Each R In RR Next R Else Debug.Print "*** NO CELLS FOUND" End If End Sub

This will print to the VBA Immediate window the address of those cells in the range A1:F20 that have a red fill color.

suzanne s. barnhill say:
Text direction may be changed in tables, text boxes, and frames. When the insertion point is in a table or text box, the Text Direction command can be found on the corresponding contextual Ribbon tab (Table Tools | Layout or Text Box Tools | Format). To change text direction in a frame, you'll have to add a Text Direction button to the Quick Access Toolbar. Under All Commands, there are three listed. The first is just a button that cycles through the three available directions; the second and third produce a "gallery" and a "dialog" displaying the three options; they look different but work the same.
