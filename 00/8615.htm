    What you are showing is the old OSK.exe NOT the touch keyboard. Note the absence of icons from the Segoe MDL2 Assets font - for the OP that keyboard is in the System32 folder rather than assessable from the Task bar keyboard icon. I have pinned this to my Start so I can access a capable enough keyboard for a tablet without  keyboard.

    What the OP is probably taking about is this keyboard:

    This is the new and too small/non-resizable word flow touch keyboard.

    This change is rendering keyboard-less operation of tablets largely impossible and must urgently be fixed.

    The standard keyboard option setting in the Setting app has been lost (hopefully only temporarily). This option is ESSENTIAL for some languages (eg in Myanmar you cannot type all the letters from the basic touch layout). The above adds word flow (hurray - finally) but is too small and not resizable and, so, is only usable as a swipe style keyboard. Some languages are restricted to only the basic keyboard and no standard or word flow. Other languages only have the word flow and no standard or basic. You can trick these to show the basic by installing a keyboard that has no word flow keyboard yet (eg English International) and swapping between the languages on the task bar and the settings on the touch keyboards (behind with the keyboard and gear) until you get both the basic and word flow layouts (but still no standard layout) as an option in English US.

    The problem is three-fold:

    1.      The standard keyboard is missing and is no longer selectable as an option in the Settings App. This keyboard is ESSENTIAL for tablet use without a keyboard especially if you are using another script. I need to use Myanmarsa. The basic Myanmar keyboard does not allow you to type all the letters of this alphabet.

    2.      You sometimes have to trick the system to show both the basic and word flow keyboard. Initially I was only offered the basic for Myanmar, the basic and handwriting for English International and just the wordflow and handwriting for English US / Australian. Only after selecting International and then switching to US whilst the keyboard was displayed did I get Basic and Word flow for English.

    3.      The Myanmar Windows 10 Mobile keyboard was not ported over to Windows 10 Desktop. The phone keyboard does not support word flow (even though the English ones do). However the phone keyboard does allow visual order input. There are two input orders in Myanmar. There is Alphabetic order (or Unicode order) where to type the မြ (My) in Myanmar is typed မ (Ma) ြ(Ya-Yit). There is also Visual order (inherited from typewriters) where the Ya-Yit is typed before the Ma. It would have been nice to have the phone keyboard for Myanmar ported to the desktop so you could choose between the two orders – Andriod allows you select which order.
