1.  If you can receive but not send then you have a bad SMTP configuration

Check your SW version by entering *#0000#

2) Helpful Hints on Energy Management

a) If you do not care about WCDMA then switch from DUAL to GSM

b) Make sure your WLAN "show availability" in Tools/Connection is set to "never"

c) Do not leave Bluetooth On unless you are using it for an accessory

d) Do not set the HDD as your "memory in use" for messaging (SMS, MMS). HDD is not optimised for mesaging

3) Helpful hints with the HDD and other things

a) Do not set the HDD as your "memory in use" for messaging (SMS, MMS). In addition to the EM issue above - its slow (HDD buffering is optimized for Video, Music playback), you have to switch back to the phone memory each time you want to use mass storage to connect to PC. In short why keep a bunch of messges on you phone anyhow?

b) If you get a situation that the phone boots up and cant see the HDD - check your Battery cover with a paper clip over the little "box" on the inside. Jasmir stated he had a phone that was missing the magnet there which the device uses to see if the battery cover in on. Its rare but obviously someone missed something. Nokia will just give you a new battery cover no worries.

c) Dont save Themes on the HDD. People have said it slows down navigating through folders since there us some kind of interaction with the HDD

d) In general if you can - save the applications (SISx files) you download/buy on the phone memory then that is the best thing to do (better energy management, speed etc). My phone came with over 14 Mbytes of memory so that's alot of space for applications. I like to keep the HDD for what it was optimized for - Music and Videos.

e) Be patient with Gallery ehen you have loaded up a bunch of new songs and videos. Unlike file manager the Gallery has to create some type of database the first time it opens after a big HDD update or phone memory reset. Its one thing about the N91 i must admit annoys me.

f) Afraid about dropping it - There was a link to some russian site (I cant find it!) that stated Nokia drops the N91 over 300 times on cement to test the HDD

g) SMS starting to get sluggish? - I found this in an N80 forum. Enter *#7370# (lock codee 12345) and this cleans up the phone memory BEFORE you do this backup your phone memory using PC suite so you can still play the DRM stuff (Live8! or any ringtone you bought etc)

h) Phone turning off? - I found this on the 7380 forum. Try the same thing as SMS above or *#7780#. I did not have this problem but some of the people using 1.00 said they did

i) Back up your HDD data regularly. If you ran into data problems and have to re-format the HDD you will not lose as much data. I recommend using Mass storage to copy all the contents.

j) Don't take out the battery when the phone is ON.

k) When you want to disconnect USB mode on the N91 do so properly. If you just remove the cable from the phone there could be data that wasn't written and thus could create corruption. If there is no Disconnect option in the softkeys, use the Red key to end the connection. In XP there is an icon on the tray to disconnect the USB device.

l) Remember its a S60 3rd edition phone so old .SIS files will not work. Make sure before you download/buy that not only is it compatible for S60 3rd edition but that vefiry it is for the N91 as well just to be sure

m) "Out of Memory" - This happens sometimes on some web sites using that Internet Browser that comes with the phone. Its not the HDD but rather the RAM in the phone (cache). Make sure you have all applications closed (note "standby" and Music player cannot be closed) to free up memory. The Services (WAP) browser does not have this problem and I bought the Opera browser and it works great (installed on phone memory by the way).


n) Beware in copying installed applications (eg the actual folders and files not the SISx file) from an MMC/SD to the HDD. The N91 on start up may see these as incomplete installation and will start to install them. Its annoying and on the older sw it lead to not being able to format the HDD or connect to the PC in Mass storage. Use the proper installation of SISx files!

0) Some people using the older 1.00 software were having crashes and when they turned off the keypad tones (in profile) they stopped.
