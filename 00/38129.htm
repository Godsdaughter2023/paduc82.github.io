Microsoft Register Server is the command used to register DLLs (Dynamic Link Libraries) in Windows.

The command is regsvr32 with the full path being C:\Windows\System32\regsvr32.exe

Click 'More details' in the UAC prompt and if the path is the same you may approve it.
