Move Users directory - Microsoft Community

andufer say:

Move Users directory

Hi,
my name is Andrea. It's possible to move the directory Users to another disk or partition?
If yes, how?

Thanks a lot.
Andrea
_null____ say: Hi Andrea,

While it is an unsupported procedure, it is possible to move your Users directory to another disk or partition. However, in doing so, you will have to create new profiles which will be pointed to the new location, as it is a real pain to try to move the current profiles over (and doesn't always work).

Your best bet would be to perform a clean installation using an autounattend.xml file created by the Windows System Image Manager (found as part of the Windows AIK, available at http://www.microsoft.com/downloads/details.aspx?familyid=94BB6E34-D890-4932-81A5-5B50C657DE08&displaylang=en). A sample autounattend.xml file to perform this would look like this:

<?xml version="1.0" encoding="utf-8"?>
<unattend xmlns="urn:schemas-microsoft-com:unattend">
<settings pass="oobeSystem">
<component name="Microsoft-Windows-Shell-Setup"
publicKeyToken="31bf3856ad364e35" language="neutral" versionScope="nonSxS"
processorArchitecture="x86">
<FolderLocations>
<ProfilesDirectory>X:\Path</ProfilesDirectory>
</FolderLocations>
</component>
</settings>
</unattend>

(where X is the volume you wish to store your profiles and Path is the name of the folder you wish to use). If you're using 64-bit Windows, substitute "x86" for "amd64". Or, you can read the Windows System Image Manager documentation (which is included with the Windows AIK, or can be viewed on Microsoft Technet at http://technet.microsoft.com/en-us/library/cc749528(WS.10).aspx) for more settings and create your own autounattend.xml file.

Once you have an autounattend.xml file ready, simply place it at the root of a USB flash drive, place the Windows Vista installation DVD-ROM in your computer, then restart your computer and boot from the DVD-ROM (there may be a special hot key that allows you to display a boot menu during the BIOS initialization process, for example on Dell computers it is F12). The installation should load and automatically detect the autounattend.xml file.

If you are not interested in formattting the system to perform this procedure, you could use the Windows registry editor and a bit of trickery to move things to a new drive or partition, however as I said previously this isn't a supported procedure, and by moving the entire Users directory you will not be able to perform any upgrade installations of Windows in the future. For more information on moving the Users folder, please see http://www.vistax64.com/vista-installation-setup/24830-move-users-folder-once-all.html.

Hope this helps and have a great day! :)
