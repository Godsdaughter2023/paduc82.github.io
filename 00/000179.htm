The Send to Mail Receipt command is available in Word, Excel and Publisher applications. You need to add the command from All Commands to the QAT (Quick Access Toolbar) in these applications. To do so follow these steps: 

  1. Click on the Small drop-down chevron button in the Quick Access Toolbar on the top left corner of the Word, Excel and Publisher applications.

  2. On the drop-down menu click on More Commands.

  3. Click on the drop-down button which shows Popular Commands and select the option All Commands.

  4. Scroll down the All Commands list and select Send to Mail Recipient and click on Add.

  5. If required move it up the list.

  6. Save the selection.

  7. The option appears in the top left corner of the application in the QAT.
