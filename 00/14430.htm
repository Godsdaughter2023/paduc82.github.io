a. user say:

Hi Harold,

Thank you for your interest in Windows 10 Technical preview.

I suggest you to follow the methods below and check if it fixes the issue.

Method 1: Set Power plan Settings

  1. Press Windows key + X.
  2. Click on Power Options.
  3. Click on Require a password on wake up from the left top.
  4.  Click on Change settings that are unavailable and then select Don’t require a password and click on save settings.
  5. Now click on the Power plan and then click on change plan settings.
  6. Click on change Advanced Power settings.
  7. Click on Change settings that are currently unavailable and then change the setting to No under require a password on wake up.
  8. Click Apply and Ok.
  9. Restart the computer.
  10. Check if issue still persists.

Method 2: Run Power troubleshooter

The power troubleshooter can automatically detect and fix any incorrect power settings. Refer to these steps to run this troubleshooter:

  1. Press Windows key +W, then type Troubleshooting.
  2. Click on Troubleshooting and select the view all option on the left pane.
  3. Click on Power troubleshooter and follow the prompts.

Method 3

I suggest you to restore the power plans to default and check if it helps.

Step 1: Reset and restore the power plans to default:

  1. Press Windows key + X, click on Command Prompt (Admin).
  2. In the command prompt, type powercfg –restoredefaultschemes and hit Enter.
  3. Exit command prompt.

Note: This would reset the power plan settings to default. Any customized power plans are would be removed.

Step 2


Once the power plan is reset to default, you may create a new power plan and customize the same according to your needs and check if issue persists.

Hope this information is helpful. Please do let us know if you need further assistance, we’ll be glad to assist you.

Calculating an average in SharePoint with an if statement - Microsoft Community

a. user say:

Calculating an average in SharePoint with an if statement

I would like to add a formula that calculates the average of C1,C2,C3 columns, if C4=Engineering.  Any help or guidance would be greatly appreciated.
tim liu msft say:

Hi Greg,

You could check the following info for your reference:

1. Click Setting gear at right top>Add an app>Custom List.

2. Click Add column button to create C1,C2,C3 columns as Number type.

3. Add C4 column as Choice type. Input the choices. One of them is "Engineering".

4. Add column>More>select Calculated type>give it a name Average.

5. Input the formula =IF([C4]="Engineering",AVERAGE([C1],[C2],[C3]),"")>configure other settings>OK

6. Now when you add items to that list, and select Engineering in C4, Average column will automatically display the average result.
