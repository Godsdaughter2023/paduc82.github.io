Timer Displayed

john w. vinson say:

You can do this using the form's Timer event. Negative times are a bit tricky but can be handled.

 

Put two textboxes on the form, one for the current count (txtTimer) and one for the minus sign (txtMinus). You'll need some event - a command button likely - to start the countdown; its code could be something like:

 

Private Sub cmdStartTimer_Click()

Me.TimerInterval = 1000

Me!txtMinus = Null

Me!txtTimer = TimeSerial(0,20,0) ' twenty minutes

End Sub

 

This will set the form's TimerInterval property to 1000 - that's a thousand milliseconds, one second. In the form's On Timer event click the ... icon and choose Code Builder and put code like:

 

Private Sub Form_Timer()

If Me.txtTimer > #00:00:00# Then

  Me.txtTimer = DateAdd("s", -1, Me.txtTimer) ' subtract one second from the current timer

Else

  Me.txtMinus = "-" ' flag that we're now counting up in negative time

  Me.txtTimer = DateAdd("s", 1, Me.txtTimer) ' count up since DateTime values can't be negative

End If

End Sub

 

You'll need some event to "stop the clock" - set txtTimer and txtMinus to NULL and the form's TimerInterval property to 0 to keep the Timer event from firing until it's restarted.
