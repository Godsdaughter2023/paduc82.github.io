albert d. kallal say:

Thanks for following up.

 

I should point out that the location you note is typically for the office 365 version, not the “boxed” or so called msi versions (so we talking about the non subscription based version of office). The “term” for these types of installs are called click to run (CTR).

 

The “boxed” edition typically has the same path name as you outline, but without the “\root” node.

 

I should also point out that the path name you outlined is for the x64 bit version of office. Most if not all office installs are STILL x32 bits – and x32 version is still the recommended setup even for x64 bit computers running a x64 bit version of windows 7-10 etc.

 

So as a result, you will typical find the path name to Access or word for the CTR version of office 2016 here: 

 

C:\Program Files (x86)\Microsoft Office\root\Office16\

 

So winword.exe, or msaccess.exe will be found in the above. Note the “x86” in part of the above path name that denotes your running the x32 bit version of office. Thus your example is likely for x64 bit version of office.

 

If you running a NON office 365 version, and are using the “boxed” off the shelf edition of office that you purchased (as opposed to a subscription), then as noted, the “root” will not likely exist in your path name and the path name will thus be:

 

 

C:\Program Files (x86)\Microsoft Office\Office16\

 

So the x32 version of office installs to a different location then the x64 bit version.

 

And also keep in mind that for both x32/x64 CTR versions, then you typically find the “\root” folder in the path name.

Migrate Apple Mail To Outlook For Mac

Hello,.

Anyone can help Me?

I have situation must migrate all email, attachment, etc from Apple Mail to Outlook For Mac. It's possible without using 3rd party tools?

I had try used free tools ( emlxconvert) to do following:

  • In Mail.app, make sure your email is in a mailbox (folder) in the On My Mac section of mailboxes.
  • Then you download a small app called emlxconvert located here: emlx to mbox
  • Your Mail.app mbox folders will be in /Users/<your user name>/Library/Mail/V2/Mailboxes
  • Go to that directory in Finder then drill down through the sub-directories until you get to the Messages directory that contains all the <file name>.elmx files.
  • Select all the .elmx files and drag them in to the elmxconvert window and click Save Mbox...
  • After choosing a destination for the .mbox file the email messages are saved as a single .mbox file that can now be imported in to Outlook 2011
  • In Outlook 2011, choose to import from a text file and select mbox to import from.
  • After you select the .mbox file you created, Outlook then will import all your old email in to

    But with that way i can't migrate my attachment to Outlook for Mac (cmiiw)

    Very appreciated if anyone can give me solution for this,. :)

    Thanks,.

Hello
