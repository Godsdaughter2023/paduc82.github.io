Word creates labels as tables. While it is possible to make minor adjustments to cell dimensions, provided the label formats have the same number of labels per page, it is simply not practical to convert existing tables into new tables and transfer all the

From your description it seems that you are producing full pages of labels with the same content in each label. You then create a label merge document http://www.gmayor.com/mail_merge_labels_with_word_xp.htm or http://www.gmayor.com/merge_labels_with_word_2007.htm  and having propagated the labels, change the document type to 'Letter' and remove the Next record fields before merging to a new document.
