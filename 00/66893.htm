Hi Sargan_499

The best way and possible only way to install onto an SSD is to clean install Windows 10 onto the SSD. That way you take full advantage of the speed boost from your new SSD.

You must disconnect the HDD temporarily when installing Windows 10 onto the SSD. It is not well reported, if you leave the HDD connected, the boot configuration files and/or page file can end up on the HDD instead of the SSD

You can connect the HDD up again once the installation completes.

One other thing, when you are in BIOS changing the Boot order, make sure SATA Mode is set to AHCI

a ruc say:

Issue with maximizing windows.

Ok so I've been using Win 7 Ultimate since it's release without any issues. Then out of nowhere my windows no longer maximize to the correct size. When I hit the maximize button there is about 1/5th of the desktop still viewable on the right hand side. I can click maximize or drag the window to the top of the screen and get the same result. Also when dragging a window to the right to have it snap to the side the mouse stops at the same point as the maximized windows and sizes as if that is the edge of the screen. I cannot drag desktop icons to this part of the screen either. When I try they simple pop back to the other side of this "edge." However my taskbar still extends the full length of the screen and I can manually size the windows to have them fill the whole screen, and full-screen games still take up the whole screen. I thought the problem might have been my GPU so I replaced and upgraded to a more current card that has more support for Win 7 but that didn't work either. Here are some pics to show you what is going on: The first one is a maximized window. The 2nd one is a window snapping to the right side of the screen. You can see the circle nova where the mouse should be (not captured by Print Screen). Pic 1 & Pic 2

Problem solved.

A O H3LL (AOL) Instant Messenger was the culprit. I don't know what it did but even when the program was closed it was still reserving that space. Weird but problem solved. Thanks anyways everyone.
