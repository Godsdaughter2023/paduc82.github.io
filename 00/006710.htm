Data Transfer from one Workbook to Another workbook. - Microsoft Community

jimbryant1340 say:

Data Transfer from one Workbook to Another workbook.

If I am in one workbook and want to reference a cell in another workbook. I actually want to copy the data in the other workbook to the workbook I am presently in.  What formula do I use to do this transfer?
tom urtis say:

Open both workbooks.

Select the destination workbook (the workbook where you want to have the data copied to). Let's say your destination workbook is named Book2.

While in that selected Book2 workbook, select the worksheet and the cell in that worksheet where you want the data copied to. Let's say the destination worksheet is named Sheet3 and the destination cell is J5.

While on that destination cell J5, hit the = key on your keyboard.

Activate the source workbook. Let's say the source workbook is named Book1.

In the source workbook, navigate to the source worksheet and cell where the original data is. Let's say it is on Sheet1 in cell A1.

Simply select cell A1 in Sheet1 of that source workbook, and hit the Enter key. You will automatically be taken back to Book2 Sheet3 cell J5 where you will see the data in that cell which is still originally existing in cell A1 of Sheet1 of  Book1.

Now, what you will actually have is a link formula in cell J5. If you want that to stay a linked formula, it would be because you expect that cell A1 of Sheet1 of Book1 will change at some point, and you want cell J5 of Sheet3 of Book2 to always show the same to-be-changed value. But if you think the original cell in Book1 will remain constant and you don't want to keep being reminded every time you re-open Book2 to refresh the link, you can make cell J5 a static value as well. Simply select cell J5, and press F2 then F9 then Enter.

suat.cini say:

Hi.

Microsoft Media Creation Tool uses NTFS to format the USB stick in order to create bootable Windows Install Disk.

USB must be formatted as FAT32 to be able to install on UEFI disk.

So what you can do now is, copy all the files on the USB stick to another place. Format the USB stick as FAT32 and then you may create a UEFI bootable disk by running the commands below. These commands will create a bootable USB stick for UEFI purposes only.

Open a command prompt with admin rights and type in

diskpart
list disk
select disk #  ( where # is the disk number of USB stick )
clean
create partition primary
select partition 1
active
format fs=fat32 quick label="WIN10_SETUP"
assign

exit

Now copy all the files back on to the USB stick. You can boot to this disk and install Windows 10 on UEFI disk.

Hope this helps.
