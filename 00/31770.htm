Hi Bob McGee MD,

 

This error is usually occurs with photo mail and the user is not using a Windows Live or Hotmail account. Since photo mail is used with SkyDrive, it doesn't know where to put the pictures and throws an error. I recommend that you delete the email that is stuck in the Mailbox, you may need to work offline for this to work and resume normal function after that. 

 

To troubleshoot the issue, please check the following:

1. Incorrect POP settings (usually the SSL option for SMTP) – check with provider if it is required or not.
2. Firewall blocking ports (usually port 25) – could be 3rd party since Windows Firewall generally allows this port through, unless specifically told not to do so.

For added information, please visit the link below:

How to configure/re-configure a non-Windows Live Hotmail account in Windows Live Mail

 

As a workaround, please follow these steps to send an email with photo attachment:

1. Open Windows Live Mail.
2. Click Email message.
3. When the message compose window opens, choose Photo album.
4. Click on an image to select the album.
5. On the Format tab, click the Paperclip in the Album styles section.

Touchscreen Not Working On Dell XPS After Windows 8.1 Update?

I've never had a problem with Windows 8 and I've become very reliant on my touchscreen for many things including internet browsing.


How can this be fixed??

garryadams say:

This is actually a power setting.  It drove me bonkers until I figured it out on my XPS 15.  Here's what you need to do:

  1. Open Device Manager (right click on "This PC" and select Manage, then go to Device Manager from the tree)
  2. Expand Human Interface Devices
  3. There should be a "USB Input Device" listed.  This is your touchscreen interface.  Right click it and select "Properties".  
  4. Go to the "Power Management" Tab.  
  5. There is a box for "Allow the computer to turn off this device to save power".  UN-Check this option.  
  6. Click "OK" at the bottom.  
  7. Your screen should refresh and your touchscreen again works.  

It's just a power management issue.  I don't know why this particular model gets shut off like that, but it fixes it every time on this model.
