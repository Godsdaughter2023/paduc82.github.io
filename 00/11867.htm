jeff toole say:

The update will start automatically only if you are 24 hours out of date.  If you want to stop this, you should ensure you are never 24 hours out of date. 

If you have a time when you know you can schedule a signature update, you should create a scheduled task to do the update at that time.  Have the scheduled task call "MpCmdRun.exe -SignatureUpdate".

If this solution is not acceptable, you can contact support for assistance with changing the configuration "Signature Updates\SignatureUpdateInterval" which determines how out of date your signature can be before an update is done automatically.
