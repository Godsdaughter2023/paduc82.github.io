Inserted row does not have same format as row above - Microsoft Community

trippap say:

Inserted row does not have same format as row above

For years, I've been teaching students that an inserted row receives the same format as the row above. I've just recently noticed that is not always true. The difference I've found concerns borders. If the cell above has some border, and the row below has the same border, then the inserted row retains that common border. But if there is a difference between the borders, on a border-by-border basis, between the row above and one directly below, the inserted row does not inherit that border characteristic. I've attached two snips that illustrate this behavior. The first shows row three having different top border in cells A3:C3. Row 4 has no border in A4 and the same blue top border on B4:C4. I then inserted a row between row 3 and row 4, and the resulting row only has a top border on cell C4, because both C3 & C4, before the insert, had the same blue top border. Has Excel always worked like this? This example is from Excel 2013.

      

w. clarkmetsker say:

Hi Trippap,

This may have changed in Excel 2013.  If you format a border like you did in the top border of row 3, and then insert a row below it, that row will not have the same formatting as row 3.  It will have the default borders which is no borders.  If you create two rows with the same borders, and insert a row between them, then the new row will have the same borders as the two rows it was inserted between.  However, if anything is different between the borders on the cells in the top and bottom rows, Excel will default to no borders.

trippap say: I checked Excel 2010, and that's the way it works also. I think the change likely came between the 2007 and 2010 versions. I find it interesting that the insert row algorithm is so precise about only copying borders if the borders on the two adjacent rows are exactly the same, but styles do not reproduce row height. IMHO, adding the row height property to custom style definitions would be a very positive enhancement.
