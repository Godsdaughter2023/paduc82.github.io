xbr dionysus say:

How to Find Firmware Version

Login Information

  • Gateway – 192.168.0.1
  • Username – admin
  • Password – admin
How to Find Firmware Version
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Click on the Home button in the top right. (The icon looks like a house)
  5. The firmware is located on this page labeled Application Version.

 

Firmware Update

  1. Plug a computer into the LAN port on the back of the device.
  2. Open a web browser.
  3. Go to the Engenius support page and find the newest version of firmware for your router.
  4. Download it and save it to the desktop.
  5. Log into the router by entering the default gateway (192.168.0.1) into the address bar of the internet browser.
  6. If prompted to login enter “admin” as the username and “admin” for password.
  7. Click on the Advanced Networking Setup button in the top right. (The icon looks like a person with a graduation cap.)
  8. Select Tools on the menu on the left.
  9. Select Firmware from the submenu.
  10. Select the Browse button.
  11. Find the file you just downloaded and open it.
  12. Click the Apply button.
  13. Wait for the router to reboot.
