I don't know what your << and >> are supposed to be -- the chevrons « and » are used for merge fields but nothing else -- but the way to access the value of a document property is to insert a DOCPROPERTY field. { IF { DOCPROPERTY Matter.CustomField.Application } = "vts" "vts" "(optional add-on)" }
