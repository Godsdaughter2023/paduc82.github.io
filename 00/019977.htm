graham mayor say:

You can do it with a script, however how to proceed depends on whether you want to add comment or just forward it. The following assumes you just want to forward it.

Copy all the code to an Outlook VBA module (the procedure is virtually identical to that shown at http://www.gmayor.com/installing_macro.htm)

I have also included a macro to test it.  You will need to enter the account name and recipient name where indicated in bold.

Select a message and run the Test macro. If you are happy address the items in italics and run the main macro from a rule.

Option Explicit
Sub ForwardUsingAccount(oItem As Outlook.MailItem)
Dim oAccount As Outlook.Account
Dim olMail As Outlook.MailItem
Const strAcc As String = "*** Email address is removed for privacy ***"        'the account you want to send from
    For Each oAccount In Application.Session.Accounts
        If oAccount.DisplayName = strAcc Then
            Set olMail = oItem.Forward
            With olMail
                If oItem.BodyFormat = olFormatHTML Then
                    .BodyFormat = olFormatHTML
                    .HTMLBody = oItem.HTMLBody
                Else
                    .BodyFormat = olFormatPlain
                    .Body = oItem.Body
                End If
                .To = "*** Email address is removed for privacy ***"        'The address you want to forward to
                .SendUsingAccount = oAccount
               .Display        'remove line after testing
                '.sEnd 'remove apostrophe after testing
            End With
            Exit For
        End If
    Next oAccount
lbl_Exit:
    Set oAccount = Nothing
    Set olMail = Nothing
    Exit Sub
End Sub

Sub TestMacro()
Dim olMsg As MailItem
    On Error Resume Next
    Set olMsg = ActiveExplorer.Selection.Item(1)
    ForwardUsingAccount olMsg
lbl_Exit:
    Exit Sub
End Sub
