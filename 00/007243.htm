richard ji msft support say:

Hi Hakan,

Yes, there is indeed other options to bypass the SCP problem: modify the register key value on the machine that is running Outlook client.
1.Start Registry Editor.
a. In Windows 10 and Windows 8, press the Windows Key + R to open a Run dialog box. Type regedit.exe and then pressOK.
b. In Windows 7, click Start , type regedit in the Search programs and files box, and then press Enter.
c. In Windows Vista, click Start, type regedit in the Start Search box, and then press Enter.

2.Locate and then select the following registry subkey:
Set the value “HKEY_CURRENT_USER\Software\Microsoft\Office\x.0\Outlook\AutoDiscover\ExcludeSCPLookup” to 1
NOTE: x.0 in the above registry path corresponds to the Outlook version (16.0 = Outlook 2016, 15.0 = Outlook 2013, 14.0 = Outlook 2010, 12.0 = Outlook 2007).

If issue persists after trying the above steps, feel free to let us know. We’re always here to assist you.
