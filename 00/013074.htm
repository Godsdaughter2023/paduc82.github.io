joannegrit say:

PowerPoint Office 2011 for Mac - Edit to multiple ppt files

Is there any ay to open several PowerPoint files, then find and replace on all open files?
rich michaels - mvp (greatcirclelearning.com) say:
The first important question that needs to be asked is, "What specifically are you trying to find and replace?"

If, in a number of PowerPoint files, you are trying to find a certain string of text and replace it with a different string of text then you really don't need a macro. Just open ALL of the PowerPoint files and then with the last opened and currently active file, select Edit > Find > Advanced and set the specific find/replace data you need. Then click Replace All. When done with the first file, save and close and go to the next file and perform the same Replace All, followed by Save and Close. Repeat with all open files.

Don't close the Find/Replace dialog until you've scanned all of the files.
