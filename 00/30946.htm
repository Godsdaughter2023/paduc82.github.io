davidmcandy say:
To decode 0x80070422. The 0x means it's a hexadecimal number, the 8 means error, the first 7 means it a windows error, and the rest of the number, 422, is the actual Windows error.
 
To look up the error we need it in decimal format. Start Calculator (Start - All Programs - Accessories - Calculator) and choose View menu - Scientific, then View menu - Hex. Enter 422. Then View menu -  Decimal. It will say 1058.
 
Start a Command Prompt (Start - All Programs - Accessories - Command Prompt) and type
 
net helpmsg 1058
 
and it will say
 

The service cannot be started, either because it is disabled or because it has no enabled devices associated with it.

 

See if the service associated with your program is set to autostart.

 

Click Start - Control Panel (and select Classic view in the left hand pane) choose Administrative Tools then Services. Find the service that is part of your program (there might be more than one). Right click it - Properties - and change Startup to Automatic. Then click Start.
