jay freedman say:

There is probably no relation between the macros and the loss of the
shortcuts, unless the macro was programmed to change keybindings --
and there is no way to record that.

You can put back the shortcuts the same way you would create new ones,
through the Keyboard Shortcuts dialog:

- Click the Office button, click Word Options, click the Customize tab
of the options dialog, and click the Keyboard Shortcuts Customize
button at the bottom.
- Click "Office Menu" in the Categories list. Then click FileSave in
the Commsnds list. Click in the "Press new shortcut key" box and type
Ctrl+S. Then click the Assign button at the bottom left.

- Click "Home Tab" in the Categories list. Then click EditCopy in the
Commands list. Click in the "Press new shortcut key" box and type
Ctrl+C. Then click the Assign button at the bottom left.

- Click the Close button.

How to insert a picture in the header or the footer
To insert a picture into the header or the footer of a worksheet, use one of the following methods.
Method 1: Microsoft Office Excel 2007
Start Excel.
Click the Insert tab.
Click Header & Footer in the Text group.
Click the area inside the header or footer where you want to add the picture.
On the Design tab, click Picture under Header & Footer Tools in the Header & Footer Elements group.
Locate and select the picture that you want to insert, and then click Insert. The text &[Picture] will appear where the picture is positioned.
Method 2: Microsoft Office Excel 2003
Start Excel.
On the Viewmenu, click Head and Footer.
Click Custom Header or Custom Footer (depending on where you want to add the picture).
Click anywhere in the Left section box, the Center section box, or the Right section box.
On the row of buttons in the Header or the Footer dialog boxes, click Insert Picture, and then find the graphic that you want to insert.
Double-click the graphic.
Click OK two times.
How to improve the appearance of a worksheet that contains a picture in the header or in the footer
Use one of the following methods to improve the appearance of the worksheet and to make sure that all the data is displayed:
Increase the header margin by dragging the margin indicators that appear on the screen.
Change the size of the picture.
How to change the size of the picture
To change the size of the picture, use one of the following methods:
Method 1: Microsoft Office Excel 2007
Click the header or the footer that contains the picture that you want to adjust.
Click any location inside the &[Picture] text.
Click the Design tab under Header & Footer Tools.
Click Format Picture in the Header & Footer Elements group.
Type .5in the Height box.

Note Because both the Lock aspect ratio and the Relative to original picture size check boxes are selected, the width changes in proportion to the change that you make in the Height box.
Click OK to close the Format Picture dialog box.
Method 2: Microsoft Office Excel 2003
On the Viewmenu, click Head and Footer.
Click Custom Header or Custom Footer.
Click any where in the &[Picture] text.
Click the Format Picture button.
Click to clear the Height check box, and then type .5in the Height box.

Because both the Lock aspect ratio and the Relative to original picture size check boxes are selected, the width changes in proportion to the change that you make in the Height box.
Click OK three times.
Note If you decide to move the picture to another section of the header or the footer after you have positioned and formatted it, you can copy and paste the code &[Picture], but you must select the picture again and reformat it if you do so.

trevor rabey say:

Where you say "I can load level", what exactly do you mean by that?

If you only have one task with a resource assigned at 100%, and max units is 100%, then there is no over-allocation, so no reason to level, and no change occurs if you do.

MSP will not change the assignment of the resource just because you change the max units.

The reason is that the software cannot know that is what you want to do, having already been told that you want the resource assigned at 100% (of availability).

The default assignment is always max units if max units is less than 100%, and 100% if max units is greater than 100%. If you set max units to 50% before you assign the resource, the assignment will be 50%.

Max units is just a threshold setting so MSP can know if the resource is over-allocated (or not).

Max units plays no part in the Work = effort x duration calculation involving the task type and effort driven settings. Whatever the task type and effort driven combination, if you change work, or effort (ie assignment units), or duration you will see a change in one of the other two variables but not both.

But you are not changing any of those three variable, only max units, but max units is not involved in W = E x D.

The reason that the over-allocation cannot be resolved is because there is only one task with a resource assigned at more than its max units, so MSP has no tasks to delay, which is all levelling does.

Levelling does not change assignments.

If I have any of this incorrect, we can be sure that John, Julie or Rod will be onto it straightaway.

Hope this helps.
