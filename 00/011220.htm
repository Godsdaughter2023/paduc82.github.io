amit tandon say:

Using Countif for Text with a "<" sign is not correct, so that =COUNTIF(B2:B24,"<1 Year") will count text like "0 Year", "001 Year", and so on (starting with zero or text like "1 Blue" which precedes "1 Year").

When Countif is used with text it is usually like: =COUNTIF(B2:B24,"1 Year") or =COUNTIF(B2:B24,"Blue") where it returns a Count of all Text which is "1 Year" or "Blue".

Else you may use: =COUNTIF(B2:B24,"<1") which will count all values less than 1 ie. 0, -1, etc.
