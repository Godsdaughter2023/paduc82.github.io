This leads me to belive that data transfer mode merely turns your phone into a card reader and gives the computer or whatever else you're connected to exclusive acces to the memory card. During this time you phone is functioning as if the memory card isn't inserted.

 

PC suite mode on the other hand is more of an intergration between your phone and PC and lets them function together by allowing the PC some "control" of the phone.

 

On both my phones, only the memory card can be accessed when connected via data transfer so I think this is how it's meant to be

chanchan05 say:

Well, the short answer is you can't see phone memory using mass storage. This is simply because in mass storage mode, your phone acts like a card reader for your memory card. The only way you could see the phone memory is through PC suite.

 

The phone classifies the internal memory as C: and the card as E:, just like my computer on a PC would. And E: is designated as mass memory.
