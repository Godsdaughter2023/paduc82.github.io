teecarter say:

THe newer version of the files should sync to the Laptop B because they have a newer timestamp. Where there is a conflict that can't be automatically figured out, both versions of the files will sync and you'll end with two sets of files (with the name of the computer at the end of one set of files), like this:

  • File A
  • File B
  • File A-Laptop B
  • File B-Laptop B

The file C shouldn't be deleted from OneDrive or Laptop A because of the timestamps on the file and the last sync date on Laptop B.

craiglong say:

My guess is if you delete file C on one PC then OneDrive remembers that you want file C deleted everywhere. 
If you add a file to the OneDrive then you want it everywhere in OneDrive or a link to the file everywhere.  Don't wait too long to get everything in sync otherwise the backup PC will not be current.  

Files On-Demand are not backed up files.

If you know that Laptop B does not have any current data, you can turn it on with Flight mode enabled, unlink your OneDrive account from the OneDrive app, then rename the OneDrive folder on the Laptop B (e.g. C:\Users\yourusername\OneDrive.old).
