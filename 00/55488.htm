cmdrkeene say:
That location is where the system redirects file saves when a program tries to save a file to a secured system location (like Program Files, or the Windows or System32 folders).  It does this for security and reliability.  Saving files into protected locations can cause the system to become unstable.  If the file being saved was malicious, it could infect your computer.

To prevent it from happening, I'd recommend choosing the location when you save your file (use "Save As..." if available) and then choose one of your user folders, like your Documents or Music folders.  After you choose the location once, most applications will remember your choice for future saves.
