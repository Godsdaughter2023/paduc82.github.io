    merge two field sources in one pivot table column - Microsoft Community

    nsgma say:

    merge two field sources in one pivot table column

    On my source table I have a column for project, a column for phase and a column for the phase status.

    On my Pivot table, I would like to have the phase and the phase status in a cell.

     

    As in

    -Project One

       Phase one (Approved)

       Phase two (Submitted)

       Phase three (Waiting)

     

    I know I can create an extra column where I concatenate the other tow columns and use that as the source for the Pivot, but that original table will receive the input from many people and I would like to keep the input of data there with a minimum of complication (e.g. please don't forget to copy also the formula from the cell above on that particular column after inserting another row).

     

    Thank you for any ideas.

    N

    butch pedorcik say:

    nsgma,

     

    I can list some instructions if you are interested in trying.

     

    In a new Excel document:

    1. From the Data tab, select the From Other Sources button and pick From Data Connection Wizard
    2. Select ODBC DSN and click Next
    3. Select Excel Files and click Next
    4. Now navigate to your source file and click OK
    5. My source file has one sheet of source data so I simply hit Next at the Select Datbase and Table window
    6. Again I left default data on the Save Data Connection File and Finish window and clicked Finish
    7. On the Import Data window select PivotTable Report and click OK

    The resulting Pivot Table created by the wizard in my case has a Command Text of:

     

    SELECT * FROM `C:\DOCUMENTS AND SETTINGS\BPEDORCIK.MYDOMAIN\Desktop\Source Data.xlsx`.`Sheet1$MyData`

     

    I modified it to be:

     

    SELECT Project, (Phase + ' ' + Status) as Combined
    FROM `C:\DOCUMENTS AND SETTINGS\BPEDORCIK.MYDOMAIN\Desktop\Source Data.xlsx`.`Sheet1$`

     

    You'll notice the change of discretely selecting individual columns instead of * thus allowing a dynamically created concatenated column.  I also removed the MyData portion to have the pivot table look at all of Sheet1 in the event that I add any records that may end up outside of the MyData range.

     

    If you have a problem at step 5 or 6 like data source contains no visible tables, you will just need to define a name for some range of data in your source file.  This can be done in your source data by right-clicking and selecting Define Name...  In my example I defined a range named MyData that I eventually removed from my command text to expand to the full sheet.

    john korchok say:

    Display as icon is not a method to make the command visible as an icon. Instead, it is enabled when you're pasting something like an Excel Workbook that you want to display in a presentation as an icon instead of showing the workbook contents on the slide.

    Here's how to add Paste as Bitmap to the Quick Access Toolbar:

    1. At the right end of the QAT, click on the icon that looks like a dash with a down-pointing arrowhead below. and choose More Commands. The PowerPoint Options dialog opens to the QAT Customization pane.
    2. Under Choose commands from, change Popular Commands to All Commands.
    3. Scroll down to Paste as Bitmap and select it.
    4. Click on the Add>> button between the lists. The command is added to the right-hand list. OK out. The icon is added to the right end of the QAT.

    In addition to clicking on the QAT icon, you can now active it with a keyboard command. Count the number of icons in the QAT. If the one you just added is 5th from the left, you can run the command with Alt + 5.

    john korchok say:
    • Don't allow hanging punctuation: Word>Preferences>Compatibility
    • Font Alignment: Baseline (or Font Alignment: Center), Don't adjust space between Latin and Asian text. Don't adjust space between Asian text and numbers: These normally only appear in styles after they have been created on a computer that has Asian language support added.
    • Style: Linked, Hidden: You can't hide styles in Word 2011. On Word 2007, 2010 and 2013, there is a Styles Management dialog that allows you to hide styles and they will remain hidden on a Mac.
    • You can access all the style options from Word's Format>Style menu command. Select a style, then click on the Modify button. Fro the option Don’t add space between paragraphs of the same style, change the Format dropdown to Paragraph and check that option on the Paragraph dialog.

    chrismenard_7 say:
    Hello Nicholas. PowerPoint for Office 2016 and even previous versions have some limitations compared to Word with bullets. PPT will not support:

    1) Decimal numbered lists (1.1, 1.2, and so on).
    2) Defining new number formats (you must choose from the default set of styles offered on the Numbered tab in the Bullets and Numbering box).
    3) Applying bold, italic, or underline formatting to bullets or numbers (any formatting will be applied to the entire selected line or list).
    4) Nested lists (you can press Tab or click Increase List Level to create the same effect, but a new indented bullet or number style is not set automatically by PowerPoint).
