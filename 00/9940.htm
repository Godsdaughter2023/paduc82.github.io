bob jones aka: cybertaz say:
Unless you want to remove all user-defined AutoText Entries your only option would be to write a macro/VBA procedure. I'm not sure that would be any quicker than deleting the unwanted entries individually, though. It would have to be a rather complex bit of code to determine what to remove & what to retain.

I'd suggest using Tools> Templates & Add-Ins - Organizer - AutoText, where you can select multiple items by selecting as many items as you wish, then hit the Delete key once. Use:

Use Command+Click to select multiple non-consecutive items.
