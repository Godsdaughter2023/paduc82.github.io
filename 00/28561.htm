In the Find What box with wildcards turned on, use this expression to represent 4 or more spaces:

[ ]{4,}

There is a space character (from the spacebar) between the two square brackets. The {4,} using curly brackets represents "4 or more of the character matched by the preceding part of the expression".

Graham Mayor has an extensive reference page on wildcard searches at http://www.gmayor.com/replace_using_wildcards.htm.
