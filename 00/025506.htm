joeu2004 say:

Starbux75 wrote:

How do I make certain my formula results stay in General format all the time (Like in Excel 2003), even if they are linked to cells in currency format?

 

You cannot.

 

The solution is assign a format other than General, for example Number with 2 decimal places.

 

General format is poor choice anyway, especially for financial calculations, because of the variable number of decimal places (which depends on the column width) and possibility of seeing Scientific notation of the form 1.23E-45.

 

This was also a problem in Excel 2003; for example, PMT.  Drives me up the wall, too.

 

The reason you might think that it is more of problem in Excel 2007 and later is because the financial functions that were part of the ATP in Excel 2003 are now native Excel functions like PMT.

 

PS:  Instead of making "certain my formula results stay in General format all the time", you might consider changing the Normal style so that cells default to Number with 2 decimal places.  Note:  I have tried this myself.  I don't know that it will apply to the financial functions.  And there might be good reasons not to do this anyway.  I just offer it as food for thought.

daniel g. grau say:
Hello, Athlea.

I used to have naming problems with older Word versions, which did not admit a slew of characters, neither in the file name nor in the containing folder, or the folder containing that one, an so on, up the chain and all the way to the disk name (a.k.a. file path). One poster here had a disk named something like "Mac/John" and could not save anywhere from Word, so make sure there's no slash in the disk name.

The latest incarnation of Word is 14.1.3 (see About Word). In my experience, the problem of "invalid characters" now only appears in file names that contain forward slashes. Here, Word 2011 flags no errors otherwise, not even if the folder name contains a slash. Thus, only two characters are prohibited: the colon in files or folders (Finder's imposition) and the slash in Word file names. 

To play it safe for now, open TextEdit (it's in your Dock, but you'll also find it in your Applications folder), paste the text in a blank document and Save it from within TextEdit.

  • Can you save any document at all from within Word? New documents, changes to old documents?

It may be a temporary glitch that you can cure by restarting Word or rebooting the Mac, but first paste the text somewhere else, as indicated.

If restarting/rebooting fails to fix it:

  • Quit Word and launch it while you keep the Shift key pressed. Then test and post back.
