Your PC might have come with a recovery image that’s used to refresh or reset your PC. The recovery image is stored on a dedicated recovery partition on your PC, and is typically 3 to 6 GB in size. To save space on your PC, you can delete the recovery image from your PC and use a recovery drive instead. Windows 8.1 includes a built-in tool to create a USB recovery drive. When you open a file that is garbled, then click 'Save As', what encoding is displayed in the Save As dialog? That will tell you what encoding Notepad 'thought' was correct.

You might also try opening one of the files using the '/w' switch in Notepad (from Run dialog or Command):

notepad /w 'filename.txt'

Keith

keith a. miller say:

If the '/w' switch doesn't work, PowerShell should be able to help.

PowerShell: Get-Content

Test 'Get-Content' with varying encoding parameters to see which displays your content correctly. For example:

Get-Content 'myfile.txt' -encoding utf8

The link above gives all valid values for '-encoding'

Once you've found the right way to read the files, you can use set-content to create a new file:

get-content 'myfile.txt' -encoding utf8 | set-content 'newfile.txt' -encoding utf8

If you find a combination that works, we can script the processing of multiple files.
