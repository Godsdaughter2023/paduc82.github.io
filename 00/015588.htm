After an account becomes inactive, all messages, folders, and contacts are deleted. Incoming messages will be sent back to the sender as undeliverable. Your account name is still reserved.

If you're using a third party email client to access your Microsoft account, please make sure that you have selected “Always leave a copy of the message on the server” option to prevent this incident in the future.

Hey Happness,

 

If you reboot into Safe Mode and your Internet Explorer works correctly, this means that a third party program/service is interfering with Internet Explorer's ability to turn off its pop-up blocker. You can find which program by:

·         Click the Start Orb

·         Search msconfig

·         Click Continue on the User Account Control (If prompted)

·         Click the Services tab

·         Check the Hide all Microsoft services box

·         Click Disable all

·         Click OK

·         Click Restart Now

 

When you are booting, don’t enter Safe Mode again. With the non-Microsoft services disabled this is what is called a called a clean boot. Once your computer is back up and running test to make sure everything is still operating correctly. If you are good to go, then you just have to isolate which service was causing the problem.

·         Click the Start Orb

·         Search msconfig

·         Click Continue on the User Account Control (If prompted)

·         Click the Services tab

·         Check the Hide all Microsoft services box

·         Check (Enable) the top half of the service boxes

·         Click OK

·         Click Restart Now

 

When you are back up and running, test to see if Pop-up Blocker is still operating correctly. If everything still works right, then the services you enabled are good to stay. Repeat the enabling process for the services and continue this process until the problem returns. Then you will need to test the last batch of services enabled one by one to identify which service needs to be disabled/uninstalled/updated to avoid further problems.
