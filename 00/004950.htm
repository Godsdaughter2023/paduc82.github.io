billaboard say:

If the above 2 replies are confusing, this is what I was going to post (but the laptop was upgrading to the latest build when I came to type it in).

Get into the bios by hitting whatever key it says on the very first screen you see. If it's the F2 key, keep hitting it repeatedly as soon as you can, before Windows starts.

You should then be in the bios screen. The mouse won't work - you have to navigate with the arrow keys and use keys like the return key and probably function keys to select and alter things.

You then need to poke around until you find the boot order menu.

You need to set this up so that the DVD drive is first (at the top) and the hard disk second or third, depending if you ever want to boot from usb etc.

You then save any alterations, make sure the right recovery DVD is in the drive and restart the machine. It should then re-install the original operating system.

There might also be an option shown when you first start the machine to hit something like F12, which might or might not provide an option to boot from DVD.

Good luck.

don cuthbert say:

From Administrator Command Prompt, try running cleanmgr /sageset and select every item. Then run cleanmgr /sagerun to clear all of the pre-selected items.

Then prepare a clean boot.  Run msconfig to disable all non-Microsoft Services and Task Manager to disable all Startup items, then Restart prior to the next upgrade attempt.

Also disconnect every peripheral and USB device aside from keyboard and mouse.  Sometimes disabling all network adapters, once the device is ready for the first Restart during an upgrade, also helps.

A non-standard approach is to create your own 17101 ISO file, using the user scripts or utilities that have been developed and are available online.  These take the downloaded and processed Windows Update files and assemble them into a single ISO file, from which you can run setup.exe to upgrade.

You can disable the OneNote add-in on your Outlook app by following the instructions below:

  1. Open your Outlook app.
