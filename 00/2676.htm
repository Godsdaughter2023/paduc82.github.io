Read Receipt or Email tracking feature is currently not available in Hotmail or in Outlook.com. Read receipt is a response from an e-mail recipient that indicates the message was opened. The read receipt has to be requested by the sender, and the recipient may or may not comply. Although it is called a "read" receipt," there is no guarantee the person actually read any of the text. The message could have been opened and immediately trashed. E-mail client programs such as Windows Live Mail support read receipt requests and responses, while Web-based e-mail does not.

graham mayor say:

Isn't that the whole point of setting dates for tasks - that you can see when they are overdue, rather than just put them off? However, what you ask is possible. The following macro will identify overdue tasks and provide the option to reset the due date to today.

Sub Change_Task_Date()
Dim olApp As Outlook.Application
Dim olNs As Outlook.NameSpace
Dim Fldr As MAPIFolder
Dim olTask As Outlook.TaskItem
Dim sChange As String

Set olApp = New Outlook.Application
Set olNs = olApp.GetNamespace("MAPI")
Set Fldr = olNs.GetDefaultFolder(olFolderTasks)
For Each olTask In Fldr.Items
    If Format(olTask.DueDate, "YYYYMMDD") < Format(Date, "YYYYMMdd") _
    And olTask.Complete = False Then
        sChange = MsgBox("Change date of " & olTask.Subject & "?", vbYesNoCancel, "Task Overdue")
        Select Case sChange
            Case Is = vbYes
                olTask.DueDate = Format(Date, "Short Date")
                olTask.Save
            Case Is = vbCancel
                Exit For
                GoTo Cleanup
            Case Else
        End Select
    End If
Next olTask
Cleanup:
Set olTask = Nothing
Set Fldr = Nothing
Set olNs = Nothing
Set olApp = Nothing
End Sub
