Re: find duplicate names vs. new sheet names

Not willing to quit yet...
The sheet names from your picture don't appear to add anything of value.
The following code replaces the existing sheet tab name with the text from Cell B1.
It adds a numeric suffix to the sheet name, if it a duplicate.

'---
Sub UpdateSheetNamesToB1()
'Jim Cone - Portland, Oregon USA - April 2016
 Dim rngLocation As Excel.Range
 Dim ws As Excel.Worksheet
 Dim N As Long
 
 For Each ws In ThisWorkbook.Worksheets
   Set rngLocation = ws.Range("B1")           '<<< Adjust as needed
   If IsEmpty(rngLocation.Value) Then
     MsgBox "No name available for worksheet " & ws.Name & "   ", _
             vbExclamation, "Auto Sheet Name"
     Exit Sub
   End If
   N = 1
   On Error Resume Next
   ws.Name = Left(rngLocation.Value, 29)
   If Err.Number <> 0 Then
     Do
       Err.Clear
       N = N + 1
       If N > 99 Then Exit Do
       ws.Name = Left(rngLocation.Value, 29) & Format$(N, " 00")
     Loop Until Err.Number < 1
     N = 0
   End If
 Next

End Sub

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oFld As Word.Field

For Each oFld In ActiveDocument.Fields
   If Selection.Range.InRange(oFld.Result) Then

      oFld.Updat

      Exit For

  End If
Next oFld
End Sub

mbizup say:

As far as I know, you cannot conditionally format a PivotTable.

 

However a possible work around is to use a Pivot Table's queried data as the recordsource of a datasheet or continuous form.

For example, the data for your scenario could be collected in a query similar to this:

 

TRANSFORM Sum(Occupancy) AS total
SELECT RoomNo
FROM tblOccupancy
GROUP BY RoomNo
PIVOT DateOfStay

 

If you show that in a report, datasheet or continuous form view, you can conditionally format the display.

 

That may be useful for creating a one-time report if that meets your needs.  The catch is, of course that there is a  potential for a varying number of columns -- so applying that iin a seamless way would be difficult if even feasible (you could possibly write code to design a form on the fly based on your pivot table data).
