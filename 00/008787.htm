Bruce

There is no automatic way of doing what you are asking. Word styles have only one possible 'special case' spacing override which is to remove spacing between paragraphs with the same style.

Hello,

if hiding data in the source grid is not an option, you can create dynamic range names that ringfence only the current data values. Then plug those range names into the chart, instead of direct references. 

For example, create these two range names:

chtLabels =OFFSET(Sheet1!$A$2,0,0,COUNT(Sheet1!$B:$B),1)
chtValues =OFFSET(chtLabels,0,1)

The chtLabels range will only cover the rows with data in column B. The chtValues range will be offset by one column.
