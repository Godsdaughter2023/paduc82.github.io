Overview - Microsoft Community

xb1r apollo say:

Overview

Comtrend CT-5621T

Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - admin |   root   |   user  |   user   | support
  • Password -  admin | 12345 | 12345 |   user   | support

MAC Filtering

 *only available when bridging is configured*

  1. Open a web browser.
  2. Enter http://192.168.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. From the menu on the left, select Advanced setup.
  5.  In sub-menu that populates select Security.
  6. Select MAC filtering from the sub-menu here.
  7. Click add/remove to configure filter.
  8. An new screen will appear, specify the condition for the filters.
  9. Enter MAC address to be filtered.
  10. Select Save/Apply.

DHCP Reservation

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. *This option is unavailable*

UPnP

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link).
  3. When prompted enter the admin for the username and admin for the password if the gateway asks.
  4. From the menu on the left, select Advanced Setup.
  5. Select LAN from the sub-menu on the left.
  6. Toggle "Enable/Disable" UPnP Protocol by selecting/deselecting check box.
  7. Select Save/Reboot.

DMZ

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. From the menu on the left, select Advanced Setup.
  5. Select NAT from the sub-menu on the left.
  6. Select DMZ Host on the next sub-menu.
  7. Enter the IP Address of the Xbox.
  8. Select Save/Apply.

 Disabling DMZ

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. From the menu on the left, select Advanced Setup.
  5. Select NAT from the sub-menu on the left.
  6. Select DMZ Host on the next sub-menu.
  7. Clear the IP Address field
  8. Select Save/Apply

Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter the admin for the username and admin for the password if the gateway asks.
  4. From the menu on the left, select Advanced Setup.
  5. Select NAT from the sub-menu on the left.
  6. Select Virtual Servers on the next sub-menu.
  7. Select the Add button.
  8. Select Custom Service and input the name "Xbox".
  9. Enter the IP Address of the Xbox in "Server IP Address".
  10. Input 3074 for the External Port Start/End and Internal Port Start/End.
  11. Select TCP/UDP for the Protocol.
  12. Select Apply/Save.

Disabling Port Forwarding

  1. Open a web browser.
  2. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter the admin for the username and admin for the password if the gateway asks.
  4. From the menu on the left, select Advanced Setup.
  5. Select NAT from the sub-menu on the left.
  6. Select Virtual Servers on the next sub-menu.
  7. Place a check in each port table to be removed
  8. Select remove.
  9. Select Apply/Save

Firewall

There are no additional Firewall features on this device.

Updating Firmware

Firmware updates are installed automatically by the Internet Service Provider

Finding the Firmware Version

  1. Open a web browser.
  2. Enter http://192.168.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. From the menu on the left, select Device Info
  5. The Firmware can be found on the following page labeled as "Software Version".

Restore Defaults

Through firmware

  1. Plug the computer into a LAN port via an Ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link)
  4. When prompted enter the admin for the username and admin for the password if the gateway asks.
  5. From the menu on the left, select Management.
  6. Select Settings from the sub-menu on the left.
  7. Select Restore Defaults from the next sub-menu.
  8. Select the Restore Default Settings button.

Through Hardware

  1. Locate the reset button on the back of the device.
  2. Using a toothpick or ballpoint pen press and hold the reset button for 30 seconds.
  3. Wait for the device to restart.

Bridging Guide

*Bridge mode should only be used if you have a router to plug into the device.*

*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  1. Open a web browser.
  2. Enter http://192.168.1 or http://192.168.1.254 into the address bar. (Or click the link)
  3. When prompted enter admin for the user name and admin for the password.
  4. From the menu on the left, select Quick Setup, click next.
  5. Enter the PVC index, click next.
  6. Enter the PPPoE information provided by your ISP, click next.
  7. Select the Bridging radio button, click next.
  8. Select "Enable Bridge Service".
  9. Enter service name, click next.
  10. Enter the IP Address and Subnetmask of the LAN interface. Click next. (The IP address will show as 192.168.1.1 by default)
  11. In the next screen select Save/Reboot.

Multiple Console Support

We have confirmed that this network device is unable to support more than one console with an Open NAT. 

Known Issues

**The CT-5621T is a wired  ADSL router with one USB port, four Ethernet ports and “NO” wireless broadcast capability.**

**If you are only able to log in with the user account you may not be able to
make any changes.** 

 If this is the case, navigate to the change password page (http://192.168.1.1/password.html). There you should see the passwords for the admin
and support usernames printed in plain text.

Hi Leigh,

You may change the name the start up name on your computer by creating a new profile. To create a new profile, kindly follow these steps:

  1.  Log in with Admin rights and go to Start>Control Panel>Classic View>User Accounts.
  2. Select Create new user (This will be the name you want to use ) and choose Administrator for the account type and assign password.
  3. Logoff and then logon as the new user (This will allow Windows to create the profile for your new name.)

  7. Browse to your new profile and click OK and let Windows copy the profile for you.
