Table of content, figures, and tables. - Microsoft Community

a. user say:

Table of content, figures, and tables.

I have been using Office 2016 for three weeks.  I think my current issue did exist in 2013 as well.

When I add an attachment to my Word 2016 document, and add a caption to that attachment (drawing or table), in the Table of Contents, / Figures / Tables, if I do not put a paragraph mark between the displayed attachment and the caption, the drawing itself appears in the body o the TOC / TOF / TOT.  If I add the paragraph mark, The Tables are fine (without the drawing).

jay freedman say:

That's the way Word works -- in Word 2016 and all previous versions.

The Table of Figures and Table of Tables are both created by TOC fields whose field code includes a \c switch. The Table of Figures field contains \c "Figure", and the Table of Tables field contains \c "Table". That switch tells the field to grab every paragraph that includes a SEQ field with the matching identifier, either SEQ Figure or SEQ Table.

If there is no paragraph mark between the picture or table and its caption, then they're both part of the same paragraph. The TOF or TOT will grab the entire paragraph, leading to the behavior you see.

The "fix" (i.e., the way you're supposed to work) is to include a paragraph mark between the picture/table and the caption, placing them in separate paragraphs.

andre da costa say:

What does Windows as a Service entail?

Since the introduction of Windows Update as part of Windows, Microsoft has delivered updates as way to keep Windows updated and secure. This has always been the way the company keeps commercial versions of Windows on the market current.

With Windows 10, Microsoft is making significant changes to this model by delivering new methods of servicing Windows. Taking into account we now live in a mobile, constantly connected world, where threats happen all the time. Part of the solution to this is to keep users current by providing the latest updates as soon they are available in addition to delivering features as soon as they ready too.

Microsoft will do this using three types of service branches for Windows 10:

  • Security Updates, Features and Fixes are automatically applied.
  • There is no option to delay or customize these updates.
  • Windows 10 Home
  • CBB includes the requirements of the Current Branch, but also provides the option of customizing when and which Security Updates, Features and Fixes are applied, similar to how Windows Update works today in current versions of Windows. 
  • Updates cannot be deferred indefinitely.
  • Windows Updates can be managed using enterprise management tools such as Windows Update for Business and/ WSUS.
  • Windows 10 Pro
  • Windows 10 Enterprise
  • Windows 10 Education
  • Only available to volume license customers running Windows 10 Enterprise or Education.
  • Flexible options for managing Windows Updates.
  • Windows Updates can be customized to only accept Security Fixes.
  • Windows Updates can be managed using enterprise management tools such as Windows Update for Business and/ WSUS.
  • Windows 10 Enterprise

Frequently Asked Questions: Windows 10

graham mayor say:
In addition to Doug's suggestion, It could be attributable to a bug that sometimes seems to have Word misreading temporary files related to the printer currently in use.

That being the case you can usually 'unscramble' the display from File=>Print and switch to the printer properties or to another printer then your normal one, or close Word and delete all files found on Start=>Search and looking for ~$*.*;*.tmp as the name string then restarting Word." See also http://www.gmayor.com/what_to_do_when_word_crashes.htm  

Never ever save to removable media. It is a constant source of document corruption. Always copy to and from removable media and work from the hard drive. You may find http://www.gmayor.com/SaveInTwoPlacesAddIn.htm a useful aid to this.
