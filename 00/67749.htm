I was able to fix all of these problems through the control panel.

To fix the edge swipe (try this if the windows charms appear when you don't want them to):

Control Panel- Mouse Properties- Elan- Options - One Finger (tab) - Edge Swipe - untick 'enable'

To fix the zooming in and out function (note that this is just for the touchpad and you will still be able to zoom in and out using the touch screen):

Control Panel- Mouse Properties- Elan- Options- Multi-finger (tab)- Zooming- untick 'enable'

To fix the bottom-up scrolling:

Control Panel- Mouse Properties- Elan- Options- Multi-finger (tab)- Scrolling- untick 'reverse'
