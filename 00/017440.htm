glenprouty say:

Hi Debbie,

You have received good advice, that may have solved your problem already.

If it has not, you might try clearing your Windows Defender cache. Sometimes it

becomes corrupted, and causes Windows Defender to not update.

The procedure is simple, and you perform it from Command Prompt (admin).

Get into Command Prompt (admin) by hitting the Start key and type CMD in the search

box. Command Prompt will show in the pop up.  Select "Run as administrator" and you will

be taken to the Command Prompt panel.

Type the following, one at a time, hitting <enter> after each.

CD\                           <enter>

CD program files\windows defender     <enter>

MpCmdRun -removedefinitions -all       <enter>   (there is a space before each -)

Exit                             <enter>                 (To leave Command Prompt)

Restart your PC

And "check for updates"

If corruption was your problem, the updates should download.

Good Luck,  Glen
