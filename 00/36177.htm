gerry c j cornell say:

Both keyboard and mouse are USB wired

We still need to clarify what type of keyboard and mouse you have. A wireless keyboard and mouse has no wired connection from either device to the computer. There are two common types . One type connects each device by wireless to a transponder which is then connected by wire to a USB port or indirectly through a USB hub. A second type has both devices connecting by wireless to a dongle which is plugged into a USB port. Before the introduction of wireless keyboard and mouse most users had PS/ 2 devices wired back to special non-USB ports, one for each device.

From what you seem to be saying you most likely have a wireless keyboard and mouse which connect by wireless to a transponder, which is wired back to a USB hub and then to a USB port. This type of keyboard and mouse is prone to the type of connection problem you have. There are several possible causes of one or both failing to connect. Some suggestions:

1. Try replacing the batteries in both the keyboard and the mouse.

2. Try pressing the reset button on one or both devices for up to 10 seconds to re-establish the connection. The reset buttons are on the underside and are often quite small and recessed to avoid accidental resetting.

3. The transponder has a button to refresh the wireless connection. Make sure there is a green light on the transponder as no green light means the transponder has no power / is not working. When you press the button on the transponder the green light should flash.

4. USB devices can be temperamental. Changing USB ports and connecting directly to a USB port and not to a USB hub or connecting to another USB port can re-establish connections.

5. Getting function keys on some keyboards to work can be difficult because they have an F lock key which needs depressing before any function key will work. It means you cannot use a function key until after the computer has booted. No good if you want to do something before the boot completes like access safe mode.

6. When all else fails the simplest solution ifis to buy or borrow a wired PS/2 keyboard and mouse providing you have a suitable port on the computer to connect the devices to. A PS/2 device is a more certain means of entering passwords and using function keys which your wireless devices may not be providing.
