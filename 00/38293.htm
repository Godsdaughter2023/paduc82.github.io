a. user say:

In general, upgrading an operating system is a crapshoot. That's why the experts prefer a clean install.

The first step before upgrading is to find out whether all your computer's hardware is supported on Windows 10. If any of your hardware is not supported, and it can't be replaced with hardware that is supported, Windows 10 won't work properly on your computer.

The next step before upgrading is to find out whether all your software applications are supported on Windows 10. Software versions that are not supported should be updated to versions that are supported. Software that is not supported won't work properly on Windows 10, and if you're unlucky they'll cause problems for Windows 10 itself.

The third step is to solve any problems that exists with your existing version of Windows. Unsolved problems will just migrate to Windows 10, where they will probably cause even more problems.

The last step before upgrading is to ensure that your existing version of Windows has a supported configuration. The Windows installer (which is known as Setup) doesn't know how to deal with a Windows configuration that isn't supported by Microsoft. When Setup encounters unsupported configurations the results are usually bad. The great advantage of a clean install is that you get a fresh start.

When an upgrade has multiple unrelated problems, it's usually the result of the above causes.

The paint program should be in the system32 subfolder of the Windows folder.

 

You should be able to invoke it by...

  • Click Start
  • Click Run
  • Type mspaint and hit enter

If the shortcut isn't there you can create one, the target needs to be %SystemRoot%\system32\mspaint.exe

 

If paint does not reside in the system32 folder you can get installed by Control Panel-Add or Remove Programs-Add/Remove Windows Components and then looking for "Details.." (click button) for "Accessories and Utilities" and then "Accesories". Look for Paint and tick the box, click OK twice and then Next>.

 

If the box is ticked, you can try unticking to uninstall and the reticking to install again. This should bring back the start menu shortcut too.

 

You may need your Windows installation disk to go through (re-)installation.

lemp say:

How do you locate the file that you want to open?  Do you use an address in any one of the following formats:

  • \\Computer.childdomain.domain.com \Share
  • http://computer.childdomain.domain.com
  • \\157.54.100.101\share
  • file://157.54.100.101/share
  • http://157.54.100.101

If so, there is a problem that is addressed in the following article: http://support.microsoft.com/kb/303650/en-us

Given your stated level of expertise, I suggest that you show that article to whoever is responsible for the network to which your computer is connected.

Or, simply copy the file to your computer:  browse to it using Windows Explorer and then click and drag it to your desktop.  This process, however, creates a new copy of the file, so that if anyone looks at the file in its original location, they won't see any changes you've made to the file you copied to your own computer.  To get around that, you'd have to copy the file back to its original location and answer "Yes" to the question that the file exists do you want to replace it.
