a. user say:

Hello,

Thank  you for posting your query in Microsoft Community.

EXX files are Data Files primarily associated with IBM LinkWay MsgPut Intermediate File. Additional types of files may also be using the EXX file extension.

The fastest and easiest way to open your EXX file is to double-click it. This allows the intelligence of Windows to decide the correct software application to open your EXX file. In the case that your EXX file doesn't open, it is highly likely that you do not have the correct software application installed on your PC to view or edit EXX file extensions.

If you want to open a .exx file on your computer, you just need to have the appropriate program installed. If the .exx association isn't set correctly, you may receive the following error message:

"Windows can't open this file"

To open this file, Windows needs to know what program you want to use to open it. Windows can go online to look it up automatically, or you can manually select from a list of programs that are installed on your computer.

Other possible reasons why your .EXX file does not want to open

In addition to the problem described above, i.e. the lack of the appropriate application to support the .EXX file, there may be several reasons that cause problems with opening and operation of the data format:

1. Incorrect version of the installed application

A quite common problem is the lack of application in the appropriate version. Even if you have an application, with the help of which you should open the .EXX file, you may have a problem with it because of its version. Perhaps in previous versions the application has not yet supported the .EXX file, so download the update and check again.

2. Incorrect .EXX file association with the program

If you have already installed the appropriate application, and it is the correct version, the file may not open correctly, because the operating system does not have information that this program supports the file. So you need to "show" which program the computer should use. To do this:

  • Right-click a file with the extension whose association you want to change, and then click Open With.
  • In the Open With dialog box, click the program whith which you want the file to open, or click Browse to locate the program that you want.
  • Select the Always use the selected program to open this kind of file check box.

3. The file is corrupt

It also happens that the file you want to open is corrupt and therefore there is no such possibility. In this situation, you should find its different version, or ask a person from whom you got, to send it again.

Hope this information is helpful. Please feel free to reply in case you face any other issues with Windows in future.
