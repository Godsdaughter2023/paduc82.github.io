Do you mean that you’re receiving all the emails in Outlook on a desktop computer, but the emails are not being received in Outlook on another laptop/mobile device?

If yes, did you ensure that the option to leave messages on the server is selected in Outlook?

 

If it’s a POP account, you need to select the option to leave a copy of the messages on the server, so that you can access all your emails from multiple machines. This option has to be set on both the machines.
