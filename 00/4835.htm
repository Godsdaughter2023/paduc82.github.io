stefan blom say:

A Quick Style set is simply the set of styles available in the Styles group (on the Home tab of Word 2007 and 2010). Choosing a different style set means that you replace the set of styles currently displayed in that group. In other words, the procedure is basically the same as copying styles from a template into the current document, which you can do manually via the Organizer dialog box. Here's the catch: Copying numbered styles is a bit unpredictable in Word, and it has been for several versions now. So this is an old issue/bug.

If I remember correctly, problems with style sets including numbered styles have been discussed before in this particular forum. I believe we agreed that it simply isn't 100% reliable.

What you should do instead is create numbering or bullets as discussed in the article at http://www.shaunakelly.com/word/numbering/numbering20072010.html, and save the file as a template. Then create a new document based on the template and bring in your existing content via the Insert File dialog box (Insert tab | Object | Text From File).
