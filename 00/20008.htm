ossiemac say:

The point made by fshaikh7 will not stop the code working. However, I always include the loop variable because if there are nested loops then it makes it easy to find the various start and end of loops.

Am I understanding correctly in that the ranges to be tested are NOT the same on each worksheet? If so, then the following code. Note that the UDF (User Defined Function) can go in ThisWorkbook module with the Workbook_BeforeClose code or it can be placed in a standard module.

You need a case statement for each worksheet name that requires testing together with the range to be tested. Because the Select Case comparison is sensitive to upper and lower case you will see that the ws.Name is combined with the Lcase function and then each sheet name at the case statements must be fully lower case.

Note some comments in code edited since initial posting. Had the "(Must be in lower case)" against ranges instead of sheet names.

Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Dim ws As Worksheet
    Dim rngToTest As Range
    Dim rngInvalid As Range
   
    For Each ws In Worksheets
       
        Set rngInvalid = Nothing
        Set rngToTest = Nothing
        'The comparison is case sensitive so express name as lower case
        'and then in Case statements enter the names in lower case
        Select Case LCase(ws.Name)
            Case "sheet1"       'Edit to a sheet name to be tested (Must be in lower case)
                Set rngToTest = ws.Range("D12:D16") 'Edit the range to be tested
                Set rngInvalid = ValidCheck(rngToTest)
               
           
            Case "sheet2"       'Edit to a sheet name to be tested (Must be in lower case)
                Set rngToTest = ws.Range("E12:E16") 'Edit the range to be tested
                Set rngInvalid = ValidCheck(rngToTest)
               
            Case "sheet3"       'Edit to a sheet name to be tested (Must be in lower case)
                Set rngToTest = ws.Range("F12:F16") 'Edit the range to be tested
                Set rngInvalid = ValidCheck(rngToTest)
               
            'Add a case statement with code as above for each worksheet that requires testing
            'and edit the range to the range that requires testing
        End Select
       
        If Not rngInvalid Is Nothing Then   'Not nothing then is something so invalid range found
            Cancel = True
            Application.Goto rngInvalid     'Will go to the worksheet and cell
            MsgBox "Missing Answer choice - Response required. Thank you!", vbInformation, "Selection not made"
            Exit Sub
        End If
       
    Next ws
End Sub

Function ValidCheck(rng As Range) As Range
    Dim cel As Range
   
    Set ValidCheck = Nothing        'Initialize to nothing
   
    For Each cel In rng
       If cel.Value = "" Then
           Set ValidCheck = cel
           Exit Function
       End If
    Next cel
End Function
