bob jones aka: cybertaz say:

Normally, that message only appears when attempting to enter a formula that contains the circular reference, so the problem should be with the formula being entered into that particular cell.

In Excel 2011 you may see a blue dot in a cell containing the circular reference. If not, click the Recalculate Sheet or Recalculate All button on the Formulas tab then look again. Depending on the nature of the circular reference, blue Tracer arrows may appear as well.

Another visual indicator is that cells containing a circular reference typically will return a 0 result even if the dot isn't displayed.

Also, take a look at the Status Bar. If the circular reference is on that same sheet you should see the word Circular: followed by the cell reference of the "first" circular reference in the sheet. If the circular reference is on a different sheet only the word Circular will be shown.

ronbarker say:

Paul

I have no idea how Office Pro Plus could have got onto your computer!

You can backup your outlook your PST files. (If fact you can backup your OST files as well, which will be converted into PST files. Not saying you should bother though).

"Will the PST file be saved so that on re-install I get full function back?"  Yes, but you will need to import the PST files from where exported them to back them up.

The main folder to be concerned about is your 'Personal Folders' as it contains your calendar and contact etc.  

First of all open Outlook>Click on the three ... at the bottom>Choose>Folders>This will open all your folder so that you can see where your calendar and contacts are.

Then click on>File>Open & Export>Import/ Export>Export to File>Next>Outlook data file (pst)>Next>Make sure the folders are closed and that include subfolders is checked>Next>Browse to where you want  to store you PST backup>Finish>Confirm>That is it.  (If you open the destination folder whilst you are doing this you will be able to see the PST appear. 

When you do this for the second PST to be exported be careful as Outlook will use the same name so you need to change it.  What I mean is when you get to the browse stage if you have already exported your 'Personal Folders' even though you are now exporting another PST Folder you will need to change the name in the Browse window because it will still be 'Personal Folders'

Please come back if I have confused you.
