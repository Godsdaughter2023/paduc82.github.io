pa bear - ms mvp say:

If the behavior persists, I'm afraid you're looking at a Reset. In hopes of avoiding same, run Disk Cleanup once more, this time do NOT check/enable the "Clean up system files" option. Once you start Disk Cleanup, walk away from the computer and don't come back for 45 minutes to an hour.

Assuming the run has completed by then, run it again, this time with the "Clean up system files" option enabled. Once you start Disk Cleanup again, walk away from the computer and don't come back for another 45 minutes to an hour.

  • You could let Disk Cleanup run overnight, if you want.

After the second run, tell me how much free space your C: drive has now.

[Personally, I don't think you need to run Disk Cleanup at all since only 12% of the drive is Used space. On my 700 GB drive, ~20% is Used space. As my mentor told me year ago, "Space is there to be used. Don't be concerned about Free space unless & until you're having a problem which you're certain is being caused by not enough Free space." Hasn't happened yet and I'm on my fourth computer.]
