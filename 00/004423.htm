graham mayor say:

Or format the text as white coloured to retain the space (white text on a white background is hidden).

You could create a white coloured style of each type used that you wish to hide and switch them at print time using a macro e.g. the following macro will format text in the normal style with the normal_white style, print the document then change it back again. Change the names to suit the styles in the document and your added white versions of the styles. Add more conditions to each of the four loops to match the style names you want to hide.

Dim oStory As Range
Dim oPara As Paragraph
'Swap style
For Each oStory In ActiveDocument.StoryRanges
    With oStory
        For Each oPara In .Paragraphs
            If oPara.Style = "Normal" Then oPara.Style = "Normal_White"
        Next oPara
    End With
    If oStory.StoryType <> wdMainTextStory Then
        While Not (oStory.NextStoryRange Is Nothing)
            Set oStory = oStory.NextStoryRange
            With oStory
                For Each oPara In .Paragraphs
                    If oPara.Style = "Normal" Then oPara.Style = "Normal_White"
                Next oPara
            End With
        Wend
    End If
Next oStory
'Print the document
ActiveDocument.PrintOut
'And swap back again
For Each oStory In ActiveDocument.StoryRanges
    With oStory
        For Each oPara In .Paragraphs
            If oPara.Style = "Normal_White" Then oPara.Style = "Normal"
        Next oPara
    End With
    If oStory.StoryType <> wdMainTextStory Then
        While Not (oStory.NextStoryRange Is Nothing)
            Set oStory = oStory.NextStoryRange
            With oStory
                For Each oPara In .Paragraphs
                    If oPara.Style = "Normal_White" Then oPara.Style = "Normal"
                Next oPara
            End With
        Wend
    End If
Next oStory
Set oStory = Nothing

ken sheridan say: A couple of options spring to mind:

1.  Include an unlinked subform in the current form, in its footer if the current form is in continuous forms view, to show the attachment.  The subform would have RecordSource property of:

SELECT DocID, Attachments
FROM tblDocuments
WHERE DocID = 1;

2.  Instead of placing the form as a subform in the current form add a command button to the current form to open it in dialogue mode with code along these lines:

DoCmd.OpenForm "frmAttachment1", WindowMode:=acDialog

Whichever option you go for set the frmAttachment1 form's AllowAdditions property to False (No).
