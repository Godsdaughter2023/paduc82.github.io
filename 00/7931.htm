There are two services each of them controlling a kernel mode filter driver.

The services are wdnissvc and windefend respectively coupled with the wdnisdrv and wdfilter filter driver.

Wdnissvc/wdnisdrv is the so called real-time malware protection core while the other pair is the File System malware protection core. These pairs are not new, they exist since the introduction of Windows defender.

Above these services sits a UI mode application, or better, a number of applications. They are just a user interface to control and configure the services. Currently, only the Real time service can be controlled (disabled/enabled). The other service (and it's driver) can only be stopped/started by the system and is always running.

Currently (14986) we see three UI interfaces:

1- Settings / Defender

2- Windows Defender App (a UWP App).

3- Windows Defender User Interface (Win32 application).

1 is the current Windows Defender UI, 2 is the 'new' App (still under development), 3 is the 'old' win32 application.

1. is the most functional in the sense that you can disable/enable the real-time protection, but watch out for the following bug.

Once real-time protection is turned off (in settings), you'll have to start the service manually, manually turning the real-time protection back on, doesn't start the service (and it's driver).

 DWORD dwEditFlags;
 ULONG cb = sizeof(dwEditFlags);
 hr = passoc->GetData(NULL, ASSOCDATA_EDITFLAGS, NULL, &dwEditFlags, &cb);

Excel VBA Add text to worksheet name

HI

I Have this macro from a previous question.  It opens workbooks in a folder (about 200), copy paste values the first sheet and then delete's all the sheets form 2 to how ever many there are.  I would like to add " 2015" to the first sheet's name.

Sub Chop_Sheets()

    Dim i As Long, strFolder As String, strFile As String, wbk As Workbook

    With Application.FileDialog(4) ' msoFileDialogFolderPicker

        ' Show dialog

        If .Show Then

            ' Assign selected folder to variable

            strFolder = .SelectedItems(1)

        Else

            Beep

            Exit Sub

        End If

    End With

    ' Append \ if necessary

    If Right(strFolder, 1) <> "\" Then

        strFolder = strFolder & "\"

    End If

    On Error GoTo ErrHandler

    ' Turn off screen updating and event handling

    Application.ScreenUpdating = False

    Application.EnableEvents = False

    Application.DisplayAlerts = False

    ' Get first file name in folder

    strFile = Dir(strFolder & "*.xls*")

    ' Loop

    Do While strFile <> ""

        ' Open workbook

        Set wbk = Workbooks.Open(Filename:=strFolder & strFile)

        ' Loop through sheets except first one

        With wbk

            .Sheets(1).UsedRange.Cells = .Sheets(1).UsedRange.Cells.Value

            Do While .Sheets.Count > 1

                .Sheets(2).Delete

            Loop

            ' Close and save workbook

            .Close SaveChanges:=True

        End With

        ' Get next file name

        strFile = Dir

    Loop

ExitHandler:

    ' Turn on screen updating and event handling

    Application.EnableEvents = True

    Application.ScreenUpdating = True

    Application.DisplayAlerts = True

    Exit Sub

ErrHandler:

    MsgBox Err.Description, vbExclamation

    Resume ExitHandler

End Sub

macropod say:

There are multiple ways of doing this.

One way is to use IF fields to conditionally output the content of the various letter.

Alternatively, you could use INCLUDETEXT fields in conjunction with IF tests to determine which document(s) should be generated for a given record.

Yet another way is to combine all the source documents into one, then bookmark the relevant ranges for each, and use the INCLUDETEXT field's bookmark switch to control the process.

For examples of all three approaches, see: http://www.msofficeforums.com/mail-merge/15997-merge-word-documents-using-mailmerge-field.html
