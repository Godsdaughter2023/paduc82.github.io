Temporary files are automatically deleted after exiting an application or a game.   2. Select System. 

  3. Select Storage. 

  4. Select This PC.

  6. Select Temporary files. 

  7. Click Remove files. 

  8. Restart your device so that the changes can take effect.  

  1. Launch the game and check if the same issue occurs.
