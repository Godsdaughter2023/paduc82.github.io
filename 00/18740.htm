Since the SCL value is 5, it’s an expected behavior that the message is marked as spam.

Please refer to:

And as sender’s IP is on the public block list, the message has a high SCL value.

 

About “The anti spam settings are probably default, and the SCL is set to 7.”, this setting is for bulk emails, not for normal emails. And it’s BCL, not SCL. 1 marks the most bulk emails as spam and 9 allows the most bulk email to be delivered. You can refer to:

https://technet.microsoft.com/en-us/library/dn759623(v=exchg.150).aspx

 

 

To let the messages which has been marked as spam (SCL is 5 or higher) to be delivered to the intended recipients, you can use the action Prepend subject line with text, then the message will be delivered to the intended recipients but prepends the subject line with the text that you specify in the Prefix subject line with this text input box.

How To Recreate This Slide - Microsoft Community

mrhuddman say:

How To Recreate This Slide

I would really like to recreate this effect in PowerPoint 2016 - anyone have an idea on how to proceed?

john korchok say:

This was likely created in PhotoShop or a similar image editor, so you can only get an approximate likeness using PowerPoint's tools.

Place a black and white photo as the background, then draw a circle in the middle. Change the circle Line to No Line, then set the fill to a light blue and transparency to about 50%. (PowerPoint will lighten the shadows and darken the highlights, unlike your illustration).

Right-click on the circle and choose Edit Text. Type in your text. To get the close line spacing shown, select the second line, change the paragraph spacing to Multiple and set it around 80%. Then experiment to get exactly the spacing you want.

With the text still selected, right-click on it and choose Format text... In the Format text dialog, select Text Fill and set the color and transparency of the text. The final will have less contrast in the transparent areas than your example.
