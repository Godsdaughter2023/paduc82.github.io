The only way to get all five back is to allow all five of your previous switches to "age off" your account. For example, let's say you changed it in May, July, August, October and December of 2018. If you try to change it now, you'll get the message that you can't change it again until May 2019. On the date show, you'll get ONE switch back, because that switch made in May 2018 drops off the account. If you were to immediately change it again, you wouldn't be able to do it again until July 2019, when that July 2018 change drops off.

Again, to reiterate what other Ambassadors have said, there are VERY few reasons that Home Xbox should be changed, such as when you get a new console to replace an older one. It was never designed to be passed around to other users, and the Xbox Live Code of Conduct specifically prohibits users from exchanging account info with each other (that is, you can't tell someone your Microsoft account address and password).

a. user say:

Hi Mau,

This is a bit of a conundrum. You are correct in your research that Windows has a limit on path length. This limit includes the full path string, starting from the drive letter, going through the directory tree to where the file is located, and then ending with the file name and its extension. That whole thing needs to be less than or equal to 260 characters.

Have you tried copying the files to the root folder (C:\) of your hard drive? You can also try creating a folder on your computer with a short name such as "A" (C:\A\). This should lessen the path length on the destination drive.

If the issue persists and to further assist you with this problem, we would like to know the following information:

  • How long is the original full path on the DVD?
  • Have you tried copying the files using Robocopy?
  • Which build and edition of Windows are you currently using? (Home, Pro, Enterprise, etc.)
