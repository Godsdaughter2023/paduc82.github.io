  2. Open a second Explorer window at your Documents folder.
    [Tip: click inside one window and press Windows key  + right arrow to snap it to the right-hand half of the screen. Click inside the other one and press Windows key + left arrow. You then have the two windows side by side]
  3. In Documents, create a new folder called WLMail Temp
  4. Find the folders that contain messages in the store folder and drag them across to the WLMail Temp folder in the other window.
  5. Shuffle these folders around into the simplest hierarchy you can manage with (to keep the file paths as short as possible). If possible, keep the folder names under 15 characters. If they're any longer, you risk losing all but the first 11 characters.
  6. Close down both Explorer windows.

 

We'll now try and reproduce this new folder structure in Windows Live Mail.

  1. Disconnect from the Internet.
  2. In Windows Live Mail, delete each folder below the system folders (Drafts, Sent items, Deleted items) under Storage folders.
    You can make this a bit easier by using Shift-Delete to remove a folder. This is the     permanently delete function, so the program doesn't have to re-index the folder contents to record the fact that it's been moved to a different folder (Deleted items).
  3. When all the folders have gone, open Task Manager (Ctrl-Shift-Escape) and watch wlmail.exe on the Processes tab. When it's obviously slowed down to tickover, close down Windows Live Mail and wait until wlmail.exe disappears from the Processes list.
  4. Re-launch Windows Live Mail and again wait until it's settled down before doing anything else.
  5. Press Alt-F I to launch the Import messages wizard. Select Windows Live Mail as the format and select the Documents\WLMail Temp folder to import from.
  6. Be patient! It may take a long time, so leave the program alone meanwhile. When it's finished and the program has settled down, close it down and wait for the program to do its housekeeping.
  7. When wlmail.exe is no longer there on the Task Manager processes tab, re-launch the program and examine Imported folder under Storage folders.
