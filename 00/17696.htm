Automatically add word count of MSOffice document to cell in the Excel - Microsoft Community

aservant7 say:

Automatically add word count of MSOffice document to cell in the Excel Spreadsheet

How do I automatically add the word count of an MSOffice document to a cell in the Excel Spreadsheet?  I would like it to automatically update the word count of the listed documents every time the spreadsheet is opened.  I am thinking there is probably some way to do that with a call to the MSWord object, but I can't find out how.
rich007a say:

Hi,

From Excel hit Alt+F11 and go to the ThisWorkbook module for your Excel file.

Paste this code:

Option Explicit
Private Sub Workbook_Open()
' This procedure requires a reference be set to
' Microsoft Word 12.0 Object Library
' via VBE > Tools > References.
'
    Dim WD As Word.Application
    Dim wdDoc As Word.Document
    Dim strPath As String
    Dim bWeStartedWord As Boolean
    Dim rngDocList As Range, c As Range
   
    '---------------------------------------------------
     'Input range and path:
    strPath = "C:\temp\"     'include trailing backslash
    Set rngDocList = Sheets("Sheet1").Range("A1:A2") 'Adjust as required
    '---------------------------------------------------
   
     'Get Word if it's running:
    On Error Resume Next
    Set WD = GetObject(, "Word.Application")
    If Err <> 0 Then
        On Error GoTo ExitPoint
         'Word wasn't running, start it from code:
        Set WD = CreateObject("Word.Application")
        bWeStartedWord = True
    End If
    On Error GoTo ExitPoint
   
     'Loop through each cell, open Word doc and get wordcount:
    WD.DisplayAlerts = wdAlertsNone
    For Each c In rngDocList
        Set wdDoc = WD.Documents.Open(strPath & c.Value)
        c.Offset(0, 1).Value = wdDoc.ComputeStatistics(wdStatisticWords)
        wdDoc.Close
    Next c
    WD.DisplayAlerts = wdAlertsAll
   
ExitPoint:
    On Error Resume Next
    If bWeStartedWord Then WD.Quit
    Set WD = Nothing
End Sub

 

Note you have to add a reference to "Microsoft Word 12.0 Object Library" via VBE > Tools > References.

john korchok say:

There are several ways to do this, but perhaps the simplest to set up is to use Built-in Document Property Content Controls. Greg Maxey has a comprehensive tutorial on this subject here: Repeating Data The Document Property Content Control solution is about 3/4 the way down the page.

There are 15 Document Property controls, which should be enough for the repeating elements in a legal form. Here are the steps:

  1. In Word, with your form open, click where you want to enter an item, like the client name.
  2. Use Insert>Text>Quick Parts>Document Property and choose one to insert.
    They come with predefined names, like Abstract or Author, which it is possible to customize. For now, just insert one of them so you can see how the process works.
  3. Copy the control you just inserted.
  4. Move to the next place in the form where the same data is to appear and paste a copy of the control. Repeat for all instance of that datum in the form.

Now go back to the first control, enter the information and press the Tab key. All controls update.

Greg's page has much more information about how to rename the controls and how to use more advanced controls, but this will get you started.

br. codyjorgensen say:

Type cover won't work.

The on-screen keyboard will give you an option to show a 10-key on the far right, but from my experience, this did not work either.

The alt-codes are mapped to the 10-key series of number buttons (the right block of numbers on a big keyboard, not the top numbers above the letters).

Essentially there are two solutions:

  1. When you need to enter in special chars with alt codes, you can use a full size keyboard, just plug it into your USB port, and the alt codes will work.
  2. Find a Windows key-mapper. A key mapper program (there are many), will allow you to change, or remap what your keys are. So, you can go into the mapper program, and change your numbers above your letters (the only number series on the Type keyboard), to the 10-key numbers on the right. The key mapper should show you a picture layout of a fullsize keyboard. You simply click it and tell it to assign the 10-key #1 to the #1 above the letter Q... and so on, for all numbers 0-9. Then, the alt should work, because when you hit the top numbers, you're actually hitting the 10 key numbers on the right (that technically don't exist on the Type cover).

Personally, I worked around the alt codes, I don't use them anymore. But if I did, I found it a lot easier to just have a full size keyboard to plug in when I needed it. It's cumbersome, and obviously not very mobile... but it works.

Good luck to you!

I realize this is an old question, but here's a full answer, not just a workaround: Hold down Alt on your Type Cover while using the number pad of the On-Screen Keyboard (not the Touch Keyboard).

If you'd like the step-by-step,

  1. Press Start and type Keyboard. Then click "On-Screen Keyboard" in the search results. (This brings up the Accessibility keyboard, NOT the Touch Keyboard that there may be a button for on your taskbar.)
  2. Press the "Options" button along the bottom of your On-Screen Keyboard. Ensure the checkbox for "Turn on numeric keypad" is checked, and press OK.
  3. On your On-Screen Keyboard, make sure NumLock (bottom-right) is on. When it is, you'll see a grid of number keys instead of arrow keys above it.
  4. To enter an Alt+Numpad code, hold down Alt on your Type Cover (your physical keyboard) while entering numbers on the keypad of the On-Screen Keyboard. It won't work to use the Alt key of the On-Screen Keyboard, as Alt holds itself down only long enough to
