z1p say: Hi CSKUNK,

    
     A program can show up in multiple places in your start menu and I'm not sure which place it is missing from.  If your short cuts are missing from under "All Programs", then as DanNe7 said you can manually re-create those entries by create shortcuts and moving them to the start menu.

If you are talking about the few programs that show up in the Start Menu above All Programs, those are added and removed automatically by Windows based on how often you use the programs. 


The 3rd place place program shortcuts show up in the start menu is at the top.  The programs at the top are 'pinned' there and could be thought of as your favorites.  Usually your INTERNET and EMAIL program show up there, but if you want to add a few of your own personal favorites you can.  All you have to do is find the program you want added, right-click on it, and select 'Pin to Start Menu'.  If you ever want to remove an item from the 'pinned' programs, just right-click on it and unpin it.
