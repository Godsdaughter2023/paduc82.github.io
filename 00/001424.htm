The ~$ at the beginning of the filename indicates that this is the "owner" or "lock" file for a template that has probably been deleted (see http://support.microsoft.com/?kbid=211632 for more about this type of temporary file).

The likelihood is that the original file was some sort of add-in that has been incompletely removed, leaving only the owner file. It is probably in Word's Startup folder, but since you're using the Word Web App, it may work differently. In any case, you need to find ~$Wiz12s.dotm (search for it in Windows Explorer if necessary) and delete it.

daniel g. grau say:

This would be the standard procedure to set a document's language and make it extensive to the parent template:

  1. Open any of the problem documents (every document is based on a template).
  2. Language is defined in the document's styles, so access Format > Style… in the main menu at the very top of the screen and highlight a style (start with the Normal paragraph style).
  3. Click on "Modify" and select "Language" in the dropdown menu at the bottom left of the newly opened Modify Style dialog.
  4. Select your language and make sure there's no checkmark next to "Do not check spelling or grammar."
  5. Click on the "Default" button to propagate the language change back to the original template (if this is not available, it will be grayed out).
  6. Click OK twice to exit that style's language dialog and its Modify Style dialog, but do not close the main Styles… dialog.
  7. Select "Styles in use" in the dropdown menu at the left of the Styles dialog and sequentially highlight all styles, making sure they have the proper language. Many will, since they will be based on the Normal style. You can check it in the style's summary description and/or access the style's definition through the "Modify" button. To enter a change, do steps 3-6.

That should take care of the theoretical aspect. I say this because even if a style defines a language, you can override it by selecting text and manually applying a different language. To ensure against that, do Edit > Select All (or Command A), then do step 4.

If the above does not work, try changing the template by opening it via Word's File > Open menu (otherwise you'd be opening a document based on that template). Do steps 2-5 to properly define the styles' language at least in the Normal paragraph style. For good measure, manually apply the language to the main text.

grschinon say: A UK-bought 2720 fold will not work in North America.

The 2720 fold is a dual-band GSM phone. Those sold in the UK and pretty much everywhere else in the world except North America operate on the 900 and 1800 MHz bands. North American GSM networks operate on the 850 and 1900 MHz bands.
