That process should be:

1. Left click the Start icon

3. Type this command in the window:

sfc /scannow

Note that there is a space between "sfc" and "/scannow"

6. Restart your computer and then open Outlook.

Place your cursor in the EXPERIENCE paragraph (but do not select any text), then select Format > Borders and Shadings and insert a "line below" the paragraph (use the "Apply to: Paragraph" dropdown). You can also use the table borders feature (the Ribbon's Table > Borders).

If you are using Outlook.com in Outlook desktop (2007 and newer), the account should be configured as Exchange - you'll have full access to the junk folder, the junk filters.
