MSE has “real-time” protection; so there’s no need to run a scan every hour. You can run a startup scan by dropping this batch file into your Startup folder. Copy and paste these lines into a Notepad file. Name the file something like “Quick Scan.bat” (the .bat extension is the critical factor) and save it to your startup folder (C:\Users\<account name>\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup):


 


@echo off


Timeout /t 20


cd C:\Program Files\Microsoft Security Client


msseces.exe -QuickScan -Hide


 


The “-Hide” parameter will cause the scan itself to run minimized, but the command prompt window will be displayed during the timeout countdown interval.
