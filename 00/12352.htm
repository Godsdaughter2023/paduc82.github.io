roady [mvp] say:

There is no difference in how pictures within the message body and within the signature are being treated.

DPI settings unfortunately always play a role. So if you are using higher DPI settings, the image will get changed to that.

The most fail proof way would be to place the picture on a web server and link to that location in your signature. Also make sure that Outlook is set not to download and include pictures from the Internet upon sending. Any size that you've then set in your HTML for the image will remain and the recipient will download your uncompressed image.

You must most likely create a Registry value for this.
