=================================
WAV files can be used to create an Audio CD which
should play in any Standard CD Player. CDs are
usually 70 or 80 minutes so it will take many for
a whole book.

If you have a CD player that can play MP3 files...you
could convert the WAV to MP3 and then you could
create a Data CD which would hold a much longer
file.

Audio/Music CDs are based on Duration and Data
CDs are based on File Size.

An alternative would be to copy the file to a Flash
Drive or Memory Card and the file could be played
on most any computer. In that case...converting the
WAV file to the WMA format might be a good idea
because the file size would be smaller.

k-ring say:

UPDATE!

So after half a work day on support chat with Microsoft staff, including sharing access to my computer, we figured out the problem.

Background:

- I linked my Skype account to my Microsoft Live-account.

- I used my gmail account to create my Microsoft-account upon upgrading to Windows 10.

- I connected my accounts on a Samsung Galaxy S7 Edge. I don't remember which, but might be phone, gmail and skype contacts all where connected.

- Contacts where then cross-referenced between these sources.

- A lot of duplicates were created.

- In Outlook online, under the Peoples-tab (contacts), there is a function called "Clean up contacts". This is an automatic function which allows Outlook to search through your contacts and try to find similar entries and "Link" them in Outlook Peoples. It allows you too go through the list and uncheck mistakes before accepting.

- This "Clean up"-function worked fine to begin with, however the only thing I was told was that there was an error made in a patch server side about two-three weeks ago. 

- The server error led to wrongly "linked" contacts/people. I would guess in the vicinity of 100 contacts in my list got weird links to other entries, mostly from people or businesses I have emailed at some time in the past. 

Problem:

- Skype has a Display name for each contact, which is editable and can be used as a nickname. Only you can see your display name for your contacts.

- Somehow Outlook People "push" a Display name to Skype for each of your contacts (if you have them connected, I guess). With this server error, Outlook would push a wrong name to many contacts. Even though they retainted all their original info, the wrongly added info sometimes took precedence, thus making my friends Display name e.g. "Blizzard Entertainment".

The fix:

- Contact live chat for Outlook. Outlook has to see if the error has been fixed server side for your account, and there is nothing Skype can do about it.

- Go through all your contacts and remove wrong links. Has to be done 1 by 1. This might not be necessary, but there is no promise that Outlook will be able to revert all changes made.
