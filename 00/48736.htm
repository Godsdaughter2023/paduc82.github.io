This gets into art and will take some practice and patience.
Unfortunately, Paint does not have the tools you need for
this task.

Using a solid color is not the best method because you lose
the texture of the skin.

A better tool to use is called a Clone Brush/Stamp. With a
Clone Brush you can copy small portions of the photo that
are the correct shade and paste them where you need them.

Also a brightness tool can help to lighten dark areas.

1) Two possibilities...one freeware and one retail.

The freeware PhotoScape has these tools and may be worth
experimenting with.

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

PhotoScape
http://www.filehippo.com/download_photoscape/
(NT/2000/XP/Vista/7)
(the file to download is  :  PhotoScape_V3.6.4.exe)
***If you do not want to install Google Chrome...be
sure to uncheck the boxes on the install screen***

PhotoScape Help (Tutorials)
http://www.photoscape.org/ps/main/help.php

Effect Brush Video
http://help.photoscape.org/help.php?id=editor_tools#effectbrush

Photoscape Clone Stamp Video
http://help.photoscape.org/help.php?id=editor_tools#clonestamp

When you open the program you will see a circle-
wheel of choices...choose...Editor...

When the Editor opens...find the photo you wish
to edit in the left pane file tree and double left
click it to open it in the editor.

I think I would try the Brighten too first and then
blend with the Clone Stamp.

For Brightnen tool...go to...Tools / Effect Brush /
Brighten)

For Clone Stamp go to...Tools tab / Clone Stamp.

Be aware when using the brighten tool...you
must hold the mouse button and drag....if you
release the mouse button and re-press it the
intensity of the effect will be increased.

The size of the stamp can be adjusted by clicking
the three dots or use the scroll wheel on your mouse.

Left click a portion of the image that will blend with
the area surrounding the unwanted object and then
left click on top of the unwanted object.

So...the method is to...copy/paste/reset...copy/paste/
reset...until the problem area blends with the
surrounding area.

Press your Escape key to reset the tool and if you are
unhappy with the edit...left click the Undo button at
the bottom right.

This will take some practice but you will find that you
cab do some amazing editing with a clone brush.

When you are happy with the edit...go to...Save / Save As...
enter a new name for the edited version and choose a save
location. Click the Save button to complete the save.
