Can you password protect a folder? - Microsoft Community

eve290 say:

Can you password protect a folder?

Does windows xp provide password protection for folders?  Does any window environment offer that?

wunders say:

Yes an No.  Natively, windows provides password protected zip folders:

"How to create and use compressed (zipped) folders in Windows XP"
  < http://support.microsoft.com/kb/306531 >

But I don't think this may be what you're looking for.  Windows XP Pro has the Encrypting File System where your login/password provides the credentials to access files in designated folders.  It is difficult/awkward to share such folders and unless you know enough to backup your certificate, you will most likely lose all your data with the next system crash...

"Best practices for the Encrypting File System"
  < http://support.microsoft.com/kb/223316 >

Otherwise you are looking at 3rd party solutions.  One of my favorite is the freeware "Truecrypt".  It works a bit differently in that you create a file that you can mount as a disk by double-clicking on it and providing a password.  Once mounted, you access it by any application just like it were another disk.  Dismount it and it exists as an encrypted file on your system.  It's like inserting a thumb drive, providing a password, using it, then taking the thumb drive out.  IMHO, one of the best freeware out there.

Truecrypt:  < http://www.truecrypt.org >

HTH,
  JW

naman r say:

Hi Pergonalpeg,

Welcome to Microsoft Vista answers Forum!

This might be an issue if the microphone is not setup on your computer properly, I suggest that you follow the steps in the link which will help you to setup microphone:

  1. Make sure the microphone and speakers are firmly connected to your computer.
  2. Open Speech Recognition Options by clicking the Start, click on Control Panel, click Sound, and then click on Recording Tab, now you have to see the Microphone listed there, select it and configure it.
  3. Follow the instructions in the wizard.

Set up microphone: http://www.microsoft.com/enable/training/windowsvista/srmic.aspx

Set up your microphone for Speech Recognition: http://windows.microsoft.com/en-US/windows-vista/Set-up-your-microphone-for-Speech-Recognition

Now check if you are able to use Windows Movie Maker and check If the issue is resolved or not.

You may also check for the Mixer settings to the bottom right corner on the computer.

If you do not have in-built Microphone, you may have to use the external Microphone.
