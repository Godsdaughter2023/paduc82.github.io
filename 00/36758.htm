Yeah, I figured you probably did, but I posted links for others in case they run into this age-old problem.

Nero's ASPI32 is something I used to distribute in one of my own apps to let it continue working on Windows Vista, so it's what I recommend and given the app in question that you were asking about was Nero CD DAE, it only made sense to use theirs. How much RAM does your system have?

In my experience, Windows 10 32bit requires a minimum of 2GB RAM and Windows 10 requires a minimum of 4GB RAM.

This is slightly more than recommended, but it is my experience that these are the minimums for a stable system . . .

Are you running many background tasks/processes?

To minimise startup processes and free up RAM, try this:

  1. Press Ctrl + Shift + Esc keys together to open Task Manager
  2. On the Startup Tab look for anything that might not need to be run at startup, like software updaters
  3. If you find anything, select it and click 'Disable'

  1. Press Windows + R keys together to open the Run Dialog
  2. Type msconfig in the run dialog and hit enter
  3. Click on the Services Tab of the resulting dialog
  4. Check the box marked 'Hide all Microsoft Services'
  5. Again look for anything in the remaining processes that might not need to be run at startup
  6. If you find it, uncheck it
  7. Click Apply and Reboot

PLEASE NOTE: do not disable anything you are unsure of as this can cause your system to be unstable!
