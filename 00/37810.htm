The main problem is you are adding only one stick and this will make your RAM run in
single channel mode vs dual channel mode which will be less efficient.
Best to get two more sticks, but RAM is cheap these days, so you could buy an 8GB kit for
about $15 more than 4GB.

You should at least use RAM that has the same voltage, speed and latency listed on the label
ie - DDR3 / 1.5V / 1600MHz / 9.9.9.24

The two options below are different, and without knowing what your particular AMD processor/motherboard combination supports, they are only examples.

No RAM mfg will guarantee stability with miss-matched RAM, but apparently a lot of people
get away with it.
-Since moving the memory controller from the motherboard to the cpu a few years ago RAM has been less of a problem in this regard, but being one who does not like to challenge Murphy's Law,
I prefer not taking chances and only use RAM from a factory matched kit of the same brand that
matches the original pair.
