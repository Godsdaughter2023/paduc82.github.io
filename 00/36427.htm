darleen pen say:

Hi,

The $Windows.-BT folder is associated with the Windows 10 upgrade process. It contains the previous Windows installation on your system. These files are used to downgrade to a previous version or a previous build of Windows 10. 

Note: If you'll delete the $WINDOWS.-BT folder, you won’t be able to downgrade to the previous build or previous version of Windows your PC had installed. The option to roll your PC back will not be available. 

Follow these steps to delete the $WINDOWS.-BT folder:

  1. Open Disk Cleanup by searching it using Cortana Search.
  2. Under Disk Cleanup tab, check the Previous Windows installation(s) and Temporary Windows installation files.
  3. Click on the OK button to remove the files.

If you have other concerns, feel free to reach out to us.
