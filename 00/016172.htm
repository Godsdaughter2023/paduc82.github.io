swischner say:

Yes, it makes perfect sense to me.

As OneNote itself does not offer any option to sort sections, your idea to install and use Onetastic has been the absolutely correct way, but...

as you noticed, the Sort mechanism in Onetastics macro language does not know date formats. So it does not recognize months or years. It merely sorts character strings. 

As far as I can see there are 3 options. Two of them require you to learn the macro language.

1. Modify the macro to manupulate the string sort. There is an example at https://getonetastic.com/blog-sort under "Custom sort orders", but it is using the old macro language (up to Onetastic 2.x). Omer Atay changed the whole language, so you have to understand how his example works and learn the new language.

2. Write a new macro that changes all section names from "MM-DD-YY" to "YY-MM-DD". That should make the standard sort of Onetastic produce the order that you want. Again you'd need to learn to program in Onetastics language.

3. Alter all section names by hand as suggested in 2. No programming is needed here but as you mentioned hundreds of sections you may want to free a weekend for this.

4. Under Step 1, type in on the third box the email address.
5. Under Step 2, choose Delete these messages—don't even put them in my Deleted folder.
6. Click Save at the bottom of the page.

If the Exclusive level was set, messages from senders not on your Contacts or Safe senders lists will be sent to the Junk folder.
