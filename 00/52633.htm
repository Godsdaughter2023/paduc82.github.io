First of all you need to be sure that the Windows ISO you have is the same Edition (Home, Pro, etc.) and Architecture (32 or 64-bit) as originally came on the system - or the new installation will not activate/authenticate.

Then assuming the previous ideas help you verify the validity of the downloaded ISO you need to describe what "can't seem to get it to work" means.

When you have an ISO that you want to use as an installation you just can't copy it to a CD/DVD or USB thumb drive and expect it to work - you have to use a program that knows how to create a bootable media like a CD/DVD or USB thumb drive and there are free programs for that.

Once you have your bootable media you need to adjust the target system so that when it restarts it will not look at the look to the hard drive as the first boot device but instead looks to the bootable media you created as the first boot device and your Dell User manual will tell you how to do that.  With the Service Tag you can find out how on the Dell Support site. 

Here are some ideas how to make some bootable media from an ISO file:

The free third party program called ImgBurn is popular for creating bootable Windows 7 CD/DVD from the ISO file you are going to download.

The problem with some free third party programs is that during their installation they want to install other things (McAfee, Avira, Avast, etc.) and make adjustments to things like your browser home page and you don't want any of that.  Always pay attention during the installations and choose to Customize if you see that offered, uncheck anything you don't want.  Sometimes they change things around so pay attention...

http://www.imgburn.com/index.php?act=download

When you are done you can uninstall ImgBurn.

If you want to make a bootable USB thumb drive I suggest Rufus for that and you can download it from here:

http://rufus.akeo.ie/
