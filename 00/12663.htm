Connection Reset or Page cannot be displayed - FIX SOLUTION HERE - Microsoft Community

jw_72 say:

Connection Reset or Page cannot be displayed - FIX SOLUTION HERE

Lately people have been reporting unable to access the login.live.com or outlook.com websites no matter what browser they used and despite following the 'standard procedures' of clearing browser cache files and cookies.
I know...since I had the same problem since the 1st of January.

My laptop, personal PC and my wife's PC all had the same problem. My Samsung Galaxy and my wife's iPhone...no problems...so it was not my internet connection or router configuration.

I analyzed what the three PCs had in common - Bitdefender Total Security 2014. I tested 'repairing' the BF TS 2014 install on my laptop to no avail...the issue persisted. So I started looking around at the default configuration and settings. The solution is the following -

Open the Bitdefender Total Security 2014 (might also affect 2015, don't know) -

Click on the Configuration button (upper right-hand) of the PRIVACY setting button.

Under the AntiPhishing Tab -

Find the option for Scan SSL...make sure the slider is in the OFF position.

You are now good to go with outlook.

Happy New Year to all!

How to select multiple option button - Microsoft Community

tuan pham 1973 say:

How to select multiple option button

In my form, I have many option Groups. Each group consist of 3 option button with label : AND, OR and NO FILTER. I use these option button to set the filter condition. How can I reset the selection, so that only button "NO FILTER" are checked

Thanks

hansv mvp say:

The TypeName function returns the type of a control as a string. So if ctl is a text box, TypeName(ctl) will return "TextBox", and if ctl is an option button, it will return "OptionButton".

The loop

    For Each ctl In Me.Controls
        If TypeName(ctl) = "OptionGroup" Then
            ctl.Value = 3
        End If
    Next ctl

loops through all controls on the form, and checks the TypeName of each control. Only if it is an OptionGroup, the value is set. All other controls are ignored.

So you don't have to set the Tag or do anything else.

Office 365 file compatibility between Mac, Windows - Microsoft Community

mandoboy say:

Office 365 file compatibility between Mac, Windows

If I create, say, a Word document in Office 365 on a Mac, (using the latest Mac operating system) and someone opens the document in a recent version (within the last few years) of Word for Windows, will everything look the same? And vice versa? 


john korchok say:

Not necessarily. Different versions have changes to page layout properties. As one example, tables in Word 2010/2011 will hang out into the left margin by the amount of horizontal padding applied to the cells. In other words, the left edge of text inside a table will align with text outside the table.  In newer versions, the left table edge aligns with the left margin, so text inside is slightly indented relative to text outside.

If you're shifting documents between 2019 for Windows and 2019 for Mac, as long as you have the same fonts on both, the appearance should be very similar. Mismatching fonts or fonts installed on one computer and not the other is the most common source of major formatting changes.

bob jones aka: cybertaz say:

It isn't so much a matter of "file compatibility" as it is a matter of environmental similarity pertaining to hardware as well as software. There are recommended procedures to follow to insure as much as possible that documents will display as intended on any given computer but even following those guidelines provides no guarantee. However, a document created on a specific computer may appear differently when opened on any other system.

Additional to what John mentioned, the printer/driver can make a noticeable difference as can the graphics components of the computer.

If appearance is of paramount importance send a PDF along with or in place of a Word documents. Universally accurate & consistent display on all platforms is what PDF was created for.
