    ksb1941 say:

    Stop UAC from popping up for a specific program.

    How can I eliminate the UAC popup for a specific program which I trust (and use regularly) while keeping for other unknown publishers? 
    srimadhwa b say:

    Hi,

    Thank you for posting your question in the Microsoft Community Forums.

    In the UAC, there is no option to disable pop up for a specific program as it has been structured in this way. User Account Control (UAC) notifies you before changes are made to your PC that require administrator-level permission. By default, UAC is set to notify you whenever apps try to make changes to your PC.

    User Account Control (UAC) is designed not only for preventing automatic malware executions in PCs, but also to prevent temp users from doing wrong things (e.g. turn off security features which they should not). The UAC security feature doesn’t have a “list” and a “black-list”, were trusted programs can be added. The UAC feature is built-in deep inside the Windows Kernel and the Windows Account System.

    Feel free to post your query here on Windows Forums, we would be happy to assist you.
