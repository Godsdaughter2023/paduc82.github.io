a. user say:

Hi,

 

Thank you for choosing Windows 8 and welcome to Microsoft Community Forums.

 

I see that you received the error message: Windows has stopped this device because it has reported problems (Code 43).

 

Let us work together to resolve this issue.

 

However, please answer the following questions to clarify a few doubts.

 

1.            What is the make and model of the computer/DVD-ROM drive?

2.            Is the driver recognized in Device Manager?

3.            When exactly do you get the error message?

 

Follow the steps given below to check the status of the device in Device Manager and to resolve basic issues.

 

a.     Press Windows key + X and select Device Manager.

b.     Expand CD/DVD ROM Drive and right-click on the drive.

c.      Select Properties and check if the device is working properly under Device Status.

 

If there is a yellow triangle with an exclamation mark in it, then it indicates that there are some issues with it.

 

When you get this error with CD/DVD ROM drive, it’s because one of the drivers controlling the device notified the operating system that the device failed in some manner. This can be because of corruption in one of the loaded drivers and the driver cache just needs to be flushed.

 

Method 1: But before that, run the hardware troubleshooter just to make sure that there aren’t any other issues.

 

You might be able to solve some problems by running a troubleshooter that will automatically fix some common issues with Hardware Performance.

 

Follow the steps to run the troubleshooter:

a.     Press Windows key + C on your keyboard to show the Charms bar.

b.     Type Troubleshooting and click Troubleshooting under Settings.

c.      Now type Hardware Troubleshooter in the Search option.

d.     Click on Hardware troubleshooter to run the troubleshooter.

 

Method 2: Un-install and re-install the CD/DVD-ROM drivers.

 

Follow the steps below to un-install the CD/DVD-ROM drivers.

 

a.     Press Windows key + X and click on Device Manager.

b.     Search for CD/DVD-ROM drivers.

c.      Right-click on the CD/DVD-ROM and click on Un-install.

 

Restarting the computer will re-install the drivers automatically. Check if the issue is resolved.

 

Contact the manufacturer of the computer and download the latest drivers for CD/DVD-ROM.

 

If the drivers aren’t compatible, then refer to the following link to install them in compatibility mode.

 

Make older programs compatible with this version of Windows

http://windows.microsoft.com/en-US/windows-8/older-programs-compatible-version-windows

 

Further reference:

 

Code 43: Windows has stopped this device because it has reported problems

http://technet.microsoft.com/en-us/library/cc725873(v=WS.10).aspx

 

Please get back to us with the results.

 

If you have any further questions about Windows, write to us anytime.

 

We are happy to help.
