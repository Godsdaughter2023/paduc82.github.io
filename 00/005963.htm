brian tillman [outlook mvp 2007-2019] say:

With POP accounts, one way I can see accomplishing what you want is for each person to have either his or her own Windows usernames (which will mean separate mail profiles for each person) or, perhaps, individual mail profiles under the same Windows username.  With separate mail profiles, each person's iPAQ can have a partnership with the individual's mail profile, so that synching is with each person's default calendar, contacts, etc.  The profile gets chosen at the time Outlook starts or the iPAQ gets cradled.  If the PSTs (Personal Folders files) get put in a folder that's common to all Windows users, if using separate usernames for each individual, then each profile can reference the other profiles' default PSTs as secondary PSTs and the calendars displayed side-by-side or in Outlook's overlay view.

Only one person at a time can use Outlook and Outlook must be closed and restarted to switch profiles.  If using separate Windows usernames, Fast User Switching can be used to move from one person to the other even if each username has Outlook open.

There are other methods that can be used to share calendars as well.  See Sharing Microsoft Outlook on One PC and Sharing Microsoft Outlook Calendar and Contacts.

graham mayor say:

Now having tried the suggestions, I find that entering anything within the brackets repeats on each successive page. So that's not the cure.


That's the whole point of the header footer. Whatever you enter into it is repeated in all the other matching header footers. If you insert a page number using the built-in option, then a page field is inserted. Press ALT+F9 to toggle the display and see for yourself. Using CTRL+F9 to enter the field brackets is just a simple way of entering fields manually.

Why several steps? Because users have their own ideas how to layout headers and footers and it is not possible to provide a one step fits all solution.

If you need the same personal layout for a variety of documents, then save the empty document with your choice of header footer as a template and create the new documents from that template.
