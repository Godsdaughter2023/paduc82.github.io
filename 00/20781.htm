Can I use vlookup and set the criteria to the worksheet name?  =vlookup("current worksheet name" (i.e. Jan,Feb...Dec),Overview!C36:G47,2,false)

t. valko say:

=VLOOKUP(INDIRECT("'"&MID(CELL("filename"),FIND("]",CELL("filename"))+1,256)&" Overview'!$C$36:$G$47"),2,false)

I also added absolute referencing to your range as in $C$36:$G$47.

When you do this:

INDIRECT("C36:G47")

The range is automatically evaluated as an absolute range and because the range is quoted it's a TEXT string and will NEVER change if the formula is copied or moved.

What version of Excel are you using? Try it without the $ signs then run the formula through the Formula Auditing>Evaluate Formula command and you'll see what I mean.

If there is no cell reference in the sheet name expression then the expression will return the name of the sheet that was last calculated. It doesn't necessarily have to be the active sheet.

Try this experiment in an already existing file.

On Sheet1 A1 enter your sheet name formula:

=MID(CELL("filename"),FIND("]",CELL("filename"))+1,256)

On Sheet2 enter XXX in some cell.

Navigate back to Sheet1. What sheet name appears in cell A1?

Using a cell reference "locks" the formula to the sheet the formula is entered on.

=MID(CELL("filename",A1),FIND("]",CELL("filename",A1))+1,256)

jlfrnc say:

Part 1: Export from Outlook for Mac

1.Open Outlook for Mac and click on Contacts.

2.From the File menu, choose Export.

3.A box pops up, asking you “What do you want to export?” Items that are in the category" choose the category which customer wanted to export.

4. A Save box appears. In the “Save As” field, put the name you want to give your file. Under “Where” choose your desktop. Then click the Save button.

5.Outlook then exports your contacts. When the export is complete, click the Done button.

 

6. Create a new Identity in Outlook and import the .olm file.

 

7. Then, EXPORT your contacts as a tab-delimited file.

8. File will be exported to your chosen folder as .txt file.

 

Part 2: Import to Excel

1. Open Excel.

2. From the File menu, choose Open…

3. When the Open box appears, go to the bottom of the box and under “Enable” make sure it’s set to “All Readable Documents.” then in the top part of the window, find the .txt file you exported from Outlook to your desktop, and choose it. Then click the Open button.

4. The Text Import Wizard opens up. Choose Delimited and click Finish. Your contacts open up in an Excel spreadsheet.

5. Save it as Excel by going to the File menu and choosing Save As; under Format choose Excel Workbook.
