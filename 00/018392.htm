tasos_k say:

Hi,

try this sample

1) open a new workbook and save as with extension .xlsm (macros enabled)

 

2) in col A  In range A1:A40

write

1

2

3

.

 

40

 

3) press Alt+F11 to open Visual Basic

select Insert > Module

and paste the code below on the right side

(this code creates a drop down list in active cell)

 

Sub macro01()
If MsgBox("select a cell", vbOKCancel, "Add Drop Down List") = vbCancel Then Exit Sub
Dim rr As Range
Set rr = Selection
Set r = ActiveSheet.DropDowns.Add(rr.Left, rr.Top, rr.Width, rr.Height)
r.ListFillRange = "$A$1:$A$40"
End Sub

 

4) press Alt+Q to close Visual basic

 

5) right click on DropDown > Format Control > Control, Drop Down lines write....  40

also (if you want) in Cell Link select a cell that returns drop down selection

press OK

 

Note

this drop down is an ActiveX Control

the 'classic' drop down is.... Data > Validation > Settings > Allow.... List > Source > ect..
