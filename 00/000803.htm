I'll never understand why anyone would use DAO to create a table when DDL is simple and quick.

For one thing, Key is a reserved word. The creation of the table is failing because of that. Change it to MyKey or something equally safe.

    Dim strSQL As String
    Dim db As DAO.Database
    Dim tdf As DAO.TableDef
    Dim fld As DAO.Field
    Const conPropName = "AllowZeroLength"
    
    'Use ADO to create table
    strSQL = "CREATE TABLE MyTable(" _
        & "MyKey int IDENTITY(1,1) Primary Key," _
        & "Data varchar(25)" _
        & ")"
    
    CurrentProject.Connection.Execute strSQL
    
    Set db = CurrentDb
    Set tdf = db.TableDefs("MyTable")
    Set fld = tdf.Fields("Data")
    fld.Properties(conPropName) = True
    
    Set fld = Nothing
    Set tdf = Nothing
    Set db = Nothing

Sub get_bckg()
Dim osld As Slide
Dim i As Integer
Dim R As Integer
Dim G As Integer
Dim B As Integer
Dim strReport As String
Set osld = ActiveWindow.View.Slide
With osld.Background.Fill.GradientStops

R = &HFF& And .Item(i).Color
G = (&HFF00& And .Item(i).Color) \ 256
B = (&HFF0000 And .Item(i).Color) \ 65536
strReport = strReport & Round(.Item(i).Position * 100) & "%" & _
">>" & "RGB( " & R & "," & G & "," & B & ")" & vbCrLf
Next i
End With
MsgBox strReport
End Sub

markmzz say:

Another way.

Lets go:

1) Initially, save as your workbook with the macros enabled type (xlsm - for Excel 2007-).

2) Now, create two more worksheets (in addition to the sheets of the day: 16-Feb, 20-Feb, etc.). One named Main (positioned at the beginning of the lists of sheets ) and another with the name of ListOfSheets (positioned at the end of the list of sheets).

3) Then, create the two names below:

Sheets - Refers to: =MID(GET.WORKBOOK(1),FIND("]",GET.WORKBOOK(1))+1,31)

MySheets - Refers to: =ListOfSheets!$A$2:INDEX(ListOfSheets!$A:$A,MATCH("?*",ListOfSheets!$A:$A,-1))

4) Then enter the following formula in cell A2 of sheet ListOfSheets and copy down until the cell A256:

=IF(ROWS(A$2:A2)>COUNTA(Sheets),"",INDEX(Sheets,ROWS(A$2:A2)))

5) Finally, type the following formulas in cell A3 and B3 of the Main worksheet:

=SUMPRODUCT(COUNTIF(INDIRECT("'"&INDEX(MySheets,MATCH("16-Feb",MySheets,0)):

INDEX(MySheets,MATCH("12-Mar",MySheets,0))&"'!A:J"),A2))

=SUMPRODUCT(COUNTIF(INDIRECT("'"&INDEX(MySheets,MATCH("16-Feb",MySheets,0)):

INDEX(MySheets,MATCH("12-Mar",MySheets,0))&"'!L:L"),"email"))

I hope this helps.
