Hi RobertPreviti,

 

·         Do you know if recovery console is installed on your computer?

·         Do you have the Windows XP CD?

·         Did you check for safe mode?

 

It might be difficult to recover the Operating System if there is no optical drive/XP CD or if you do not have recovery console installed. Check to see if the following helps.

 

Method 1:

Step 1: Boot into recovery console. (You will find this option only if it is installed on the computer)

 

a.        Restart the computer and keep tapping the down arrow key on the keyboard.

b.       Select “Recovery Console” and press Enter.

 

Note: Similarly, check if you can boot into safe mode. Restart the computer and keep tapping F8 key on the keyboard.

 

Step 2: Perform a disk check from recovery console. To do so, follow these steps.

 

a.        Select the Windows Installation as prompted and press Enter.

b.       Type “chkdsk /p” (without quotes) and press enter. See if it finds any errors. If yes, then type the next command.

c.        Type “chkdsk /r” (without quotes) and press enter.

 

Important: While performing chkdsk on the hard drive if any bad sectors are found on the hard drive when chkdsk tries to repair that sector if any data available on that might be lost. 

 

Method 2: Rebuild the boot.ini file. To do so, follow these steps.

 

a.        At the command prompt, type “bootcfg /rebuild” (without quotes), and then press ENTER. This command scans the hard disks of the computer and then displays the results.

b.       Follow the instructions that appear on the screen to add the Windows installations to the Boot.ini file

 

         i.            When you receive a message that is similar to the following message, press Y:

Total Identified Windows Installs: 1

[1] C:\Windows
Add installation to boot list? (Yes/No/All)

 

        ii.            You receive a message that is similar to the following message:

Enter Load Identifier

 

This is the name of the operating system. When you receive this message, type the name of your operating system, and then press ENTER. This is either Microsoft Windows XP Professional or Microsoft Windows XP Home Edition.

 

      iii.            You receive a message that is similar to the following:

Enter OS Load options

 

When you receive this message, type “/fastdetect” (without quotes), and then press ENTER.

 

Note The instructions that appear on your screen may be different, depending on the configuration of your computer.

 

c.        Type “exit” (without quotes), and then press ENTER to quit Recovery Console. Your computer restarts, and the updated boot list appears when you receive the "Please select the operating system to start" message.

 

If Recovery Console is not installed on your computer then, you may have to get in touch with your computer manufacturer.
