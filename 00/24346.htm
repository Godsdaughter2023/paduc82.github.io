diane poremsky mvp (slipstick.com) say:

>>  have two folders that I did not create but I guess are standard? "spam" and "junk."

Not standard for outlook, but not unusual either. Spam is created either by a 3rd party antispam filter or, if imap, the server created it.  If the folder is just called 'Junk', the same would apply - something besides outlook created it.

Junk E-mail and Junk Email are outlook folders (MS removed the hyphen from the folder name a couple of years ago). You should only have one or the other, but I have seen a few cases when Outlook creates the hyphen-less folder when Junk E-mail already exists. 

>> So now I'm wondering if there's a way to force all questionable email to the Junk folder and not to a spam folder.

It depends on what is using the spam folder and if it will let you choose a different folder.
