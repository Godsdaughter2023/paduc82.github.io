    toni elias say:

    1- Stop the update service. Run these commands in command prompt (Admin):

    net stop bits

    net stop wuauserv

    2- Clear the windows update cache. Delete everything in:

    C:\Windows\SoftwareDistribution

    3- Start the update service. Run these commands in command prompt (Admin):

    net start wuauserv

    net start bits

    4- Try to check for updates as usual.
