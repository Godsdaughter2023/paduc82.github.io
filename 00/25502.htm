As the first tie breaker, you need to use the head to head results, and then it gets more complicated from there. Basically, that means you need to keep the entire schedule with results and types of scoring to be able to calculate all the tie breakers. I am not 100% sure I understand what you are doing but first of all, if you want to make changes go back to your original Publisher file, and not try to modify the PDF file you produced from that Publisher file. Yes I understand that Word has limited capability of opening and editing a PDF file but I think that if that PDF file was produced by another program such as Publisher you are going to always have layout and formatting issues. My suggestion is whether you use Publisher or Word that you save the original document and go back to it to make any changes and then create a new PDF file. Don't try to edit the PDF file.

You can open or insert a Word file within Publisher by just opening the Word file within Publisher or creating a blank page > Insert File and browse to your Word file. Publisher will create as many pages and linked text boxes as needed. You can also insert the file within a text box you create. You can also Edit Story in Microsoft Word - that Word file in Word, save the changes and it will be reflected in the Publisher file.
