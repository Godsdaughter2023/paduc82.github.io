    The formula below will do that for you . . .

    =IF(ISERROR(MATCH(E3,A1:A6,0)),"NO","YES")

    Use VLOOKUP to get the value you want. Try this in I23:

    =IF(IFERROR(VLOOKUP(H23,J23:J27,1,FALSE),"No") = H23,"Yes", "No")

    It will show Yes if there is a match, and No if there isn't one.

    How do I concatenate a lot of fields easily? - Microsoft Community

    j2theflo say:

    How do I concatenate a lot of fields easily?

    How do I concatenate a lot of fields easily?

     

    For Ex.

     =A1&";"&A2&";"&...A150 

     

    Can I create a macro to do this? Or some code i can use? i need to do this 50 times :/

    rick rothstein, mvp - excel say:

    One more way...

    Function ConCat(Delimiter As Variant, ParamArray CellRanges() As Variant) As String
        Dim Cell As Range, Area As Variant
        If IsMissing(Delimiter) Then Delimiter = ""
        For Each Area In CellRanges
            If TypeName(Area) = "Range" Then
                For Each Cell In Area
                    If Len(Cell.Value) Then ConCat = ConCat & Delimiter & Cell.Value
                Next
            Else
                ConCat = ConCat & Delimiter & Area
            End If
        Next
        ConCat = Mid(ConCat, Len(Delimiter) + 1)
    End Function

    Here is an example of its use...

    =ConCat("-",A1:A3,C1,"HELLO",D1:D2)

    Note that you can freely mix individual cells, cell ranges and text constants in any order... the first argument is the delimiter you want to put between the items (use the empty string "" if you don't want a delimiter).
