  • If you're getting from one page to another by clicking a hyperlink, there is a Back command just like the one in your browser. If you want a button, you can add the Back command to the Quick Access Toolbar: go to File > Options > Quick Access Toolbar, change the "Choose commands from" dropdown to All Commands, click on the Back command in the left list, and click the Add button between the lists; then click OK.
  • If you made an edit (insert or delete characters or graphics, or format text) before going to another page, the GoBack command (Shift+F5) will return you to the most recent edit point, and one or two additional presses of the same keys will go to the edits you made before that (it cycles through the three most recent ones).

However, if all you've done is scroll from one place to another without making edits or using a hyperlink, there is no command that will return to the previous place.
