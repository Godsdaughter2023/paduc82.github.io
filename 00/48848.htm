Hi Johannes,

It is possible that your Surface device is experiencing phantom touches issue. This may happen when there is a problem with the touch screen calibration of the device. You can download the hotfix 3165497 from this article.

If downloading the hotfix did not work,  it is also possible that the issue may be caused by physical damage, installed software, or nearby electromagnetic interference.

Here a some solutions you can try to help you resolve the issue.

Change the power settings.

  1. Press the Windows Key + S and type Power. Select Power Options from the menu.
  2. Locate your current power plan and click Change plan settings.
  3. Click Change advanced power settings.
  4. Find When I close the lid and Plugged in options and set them to Do nothing.
  5. Click Apply and OK to save changes.

  1. Select the search box in the taskbar, enter device manager, and select Device Manager in the results.
  2. Expand Keyboards.
  3. Right-click Surface Type Cover Filter Device.

  5. Restart your Surface. It'll automatically download and install the Surface Cover driver the next time you connect to the Internet.
