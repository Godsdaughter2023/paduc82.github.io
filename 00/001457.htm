Setting character limits in Word 2013? - Microsoft Community

elizabethhall_552 say:

Setting character limits in Word 2013?

I have learned that setting character limitations in tables or text boxes is not possible in Word 2016? Does anyone know if it is possible in 2013, and if so, how do you do so?
jay freedman say:

It is not possible to limit characters in tables or text boxes in any version of Word.

If you insert legacy text form fields, you can open their Properties dialogs and set maximum characters there. However, this will require restricting editing to filling in the fields; the rest of the document won't let you select, edit, or format any text.

In any table, you can go to the Table Properties dialog, click the Row tab, and set an exact row height. Users will then be able to type as much as they like into a cell, but the excess will disappear below the bottom of the cell.

Text boxes cannot be prevented from expanding as more text is typed into them.
