irfan h say:

Hello JCPHX,

1.     Do you see your favorites showing up in the Users folder?

2.     Did you make any changes to the computer (hardware or software) before the Favorites disappeared in the Internet Explorer?

I suggest that you try the below methods to fix this issue:

Method 1:

I suggest that you move the favorites files from the ‘Documents’ folder to the ‘favorites folder’. You can copy the files that you see in the documents folder to the location ‘C:\Documents and Settings\{username}\Favorites’ and check if they reappear in the Internet Explorer.

Method 2:

If you find the favorites folder and still can’t open them with the Internet Explorer, check if the registry path is set to default:

a.      Click on start, type ‘regedit’ and select registry editor.

b.      Navigate to the registry:

 [HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders]

Check if the value is set to:

"Favorites"="C:\Documents and Settings\{username}\Favorites"

Note: This should be same location as where you have the Favorites folder saved.

Also type:

In the Start Menu's Search box, or in the Run command, enter: shell:Favorites

The folder should be recreated, if not immediately then after a restart.

If the Favorites Menu is Missing or Greyed Out, try the below steps:

a.     Click on start, type ‘regedit’ and select registry editor.

b.     Navigate to the registry:

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Internet Explorer]

[HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\policies\Internet Explorer]

Go to these Registry keys and, delete the value "NoFavorites" if found.
