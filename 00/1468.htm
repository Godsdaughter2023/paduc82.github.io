the form has 17 fields when the focus changes from one input field, e.g. tab or enter,to another (ongotfocus) I want to change the background color of the input field, when the field loses focus (onlostfocus) i want to change it back to white.

djsteele, mvp say:
Add a function like the following to the module associated with the form in question:

Function SetActiveControlColour()
Dim ctl As Control

    For Each ctl In Me.Controls
        ctl.BackColor = vbWhite
    Next ctl
    Me.ActiveControl.BackColor = vbRed

End Function


Set the On Got Focus property of each of the controls of interest to =SetActiveControlColour() (make sure to include the equal sign and parentheses)

marshall barton say:

For text and combo boxes, you can use Conditional Formatting's Field Has Focus option.

 

For other controls where CF is not available, but do have the BackColor property (e.g. list box), use a function in the OnGotFocus property and the OnLostFocus property to set or reset the BackColor :

 

Function SetResetHighlight(color As Long)

   Me.ActiveControl.BackColor = color
End Function 

 

You'll have to find another way to highlight controls that do not have the BackColor property (e.g. buttons, check boxes, etc).   The way that I have used is to place a slightly larger rectangle control (named the same as the control with a "X" as a suffix) behind the control and use a similar function to set/reset the rectangle's BackColor:

 

Function SetResetHighlightX(color As Long)

      Me(ActiveControl.Name & "X").BackColor = color
End Function

 

Don't forget that you can select a bunch of controls by using Shift+Click and set a common property for all of them in one shot.

    Sub DeleteOldMessages()
