using the clipboard feature

don patterson say:
The clipboard isn't a program you can actually access and play with. It's a built-in windows component that works transparently. When you copy or cut, the info is put onto this clipboard. When you paste, the information that's on the clipboard is put into whatever it is you're working on. For example, you could copy some text from a web site (highlite the text and Press CTRL-C will place it on the clipboard). You could then paste that text into a word processing program by pressing CTRL-V. You could have a graphic on the clipboard and paste it into a graphics program like MS Paint.
