ossiemac say:

Hello Ken,

You need to print each sheet individually. However, this can be done with VBA code. Below I have provided 3 options as separate routines.

The first Sub PrintAllShts will print all sheets in the workbook

The second Sub PrintSpecificShts will print specific sheets that you have named in the Select Case statement. Note that the sheet names need to be edited with your sheet names and they are separated by a comma. Note also that the sheet name text is case sensitive so you need to use the same upper and lower case as per your sheet names.

The third Sub PrintSelectedShts will print out the selected (grouped) sheets if you prefer to select the sheets before printing.

All 3 examples currently have PrintPreview coded for testing. For production, comment out these lines with a leading single quote and remove the comment single quote from the Printout lines.

  • Alt and F11 will open the VBA editor
  • Select menu item Insert -> Module
  • Copy the required code option from below and paste into the editor.
  • Save the workbook as macro Enabled.
  • Enable macros. See help for how to do this. (Option to "Disallow all with notification" should be OK.)
  • If you need further guidelines to install the code then let me know.

Option 1:

Sub PrintAllShts()
    Dim ws As Worksheet
   
    For Each ws In Worksheets
        ws.PrintPreview         'For testing. Uncomment and comment out next line
        'ws.PrintOut Copies:=1   'Comment out if using PrintPreview for testing
    Next ws
   
End Sub

Option 2:

Sub PrintSpecificShts()
    Dim ws As Worksheet
   
    For Each ws In Worksheets
        Select Case ws.Name
            Case "Sheet1", "Sheet2", "Sheet4"       'Note Uppper and Lower case sensitive
                ws.PrintPreview         'For testing. Uncomment and comment out next line
                'ws.PrintOut Copies:=1   'Comment out if using PrintPreview for testing
        End Select
    Next ws
   
End Sub

Option 3:

Sub PrintSelectedShts()
    Dim shts As Sheets
    Dim ws As Worksheet
   
    Set shts = ActiveWindow.SelectedSheets
   
    For Each ws In shts
        ws.PrintPreview         'For testing. Uncomment and comment out next line
        'ws.PrintOut Copies:=1   'Comment out if using PrintPreview for testing
    Next ws
       
End Sub
