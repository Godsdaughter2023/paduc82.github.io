In answer to your basic question, they have no effect on your computer performance.

It was a problem with early version of Windows where orphaned entries could produce delays in the boot up process waiting for responses

Since the introduction of Windows Vista and improving with each release the way in which the Registry is processed was changed meaning these type of invalid lines were ignored.

This type of error tends to be a major part of registry cleanup program reports and it is strongly recommended not to use these type of utilities because they serve little purpose and are potentially damaging.
