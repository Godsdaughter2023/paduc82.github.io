Did a little more digging and figured out how to do it. Works fine now. Here are the steps:

First, select the entire command "get-appxpackage *microsoft.office.onenote* | remove-appxpackage" inside of the quotes; This is what you will select:

get-appxpackage *microsoft.office.onenote* | remove-appxpackage

and press Ctrl+ C to copy the command into the clipboard.

Then, open PowerShell in admin mode. PowerShell lives in Start Menu Button | All Apps | Windows System | Windows PowerShell. Right-click on PowerShell and select "Run as administrator" (you can also just type in "powershell" in search/Cortana field on the task bar, just don't forget to right click on it to "run as administrator").

Say "Yes" to the User Account Control prompt if that comes up.

After PowerShell opens, press Ctrl+V (right-clicking in order to paste has no effect here) and the command you copied in this post will execute.

Check apps list. OneNote should be uninstalled at that point.

Go to the store and install the app again.

Hi,

 

The desktop.ini file is a hidden file used to customize and adjust settings for the Windows folders that contain the file.

 

This file may have settings associated with the folder that it contains and deleting this file will change these settings back to default. For example, if the folder containing this file has a different icon and you delete this file the default folder icon will be re-enabled.

 

I would suggest you to check if you have enabled to see the hidden files. If so, you can disable the option and the file will be hidden.

a. Open Windows Explorer (Winkey + E).

b. Click on Organize.

c. Click on Folder and search options.

d. Under View tab, select “Don’t show hidden files, folders or drives”.

e. Click Apply and OK.
