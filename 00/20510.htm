scottgem say:
If you are storing a Date/Time value as part of the record, then there is no need also store the day of the week since that can be extracted from the date. The Weekday() function will return the day of the week as a number. Karl's Format() suggestion will display the name of the day.

If you want to validate the data entry to insure that the date is not a weekend, then you can use the Weekday function to do that.
