Hi Afreen,

In PowerPoint 2011 you can create custom toolbars manually but unfortunately you can't add macros to the toolbar as you can in Excel or Word. A huge design flaw in PowerPoint 2011.

Here is a macro that you will need to modify for your particular situation. It will add a custom toolbar and buttons that reference macros in your solution.

Hope it helps

Sub AddCustomToolbar()
    'Written by Richard V. Michaels, Mac MVP, October 2013
    'Great Circle Learning (GCL), http://www.greatcirclelearning.com
    'GCL provides custom and off-the-shelf add-in apps for Microsoft Office
    Dim cmdBar As CommandBar, cmdButton As CommandBarButton
    Dim bars As CommandBars, barMenu As CommandBarControl
    Dim bN As String
    
    bN = "GCL Custom 1"
    Set bars = Application.CommandBars
    
    For Each cmdBar In bars
        If cmdBar.Name = bN Then cmdBar.Delete
    Next
   
    bars.Add (bN)
    With bars(bN)
        .Visible = True
        .Controls.Add (msoControlButton)
        Set cmdButton = .Controls(1)
        cmdButton.style = msoButtonIconAndCaption
        cmdButton.FaceId = 1192
        .Controls(1).Caption = "Insert Equation"
        .Controls(1).Tag = "Equation"
        .Controls(1).TooltipText = "Inserts the Legacy Equation Editor"
        .Controls(1).OnAction = "GCL_RibbonClk"
    End With
    
End Sub
