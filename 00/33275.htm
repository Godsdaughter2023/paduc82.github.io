Sorry for my bad English.

It seems that there are problems during installation of this cumulative update. Many users with 64-bit computers are having problems. Uninstall (temporarily) the antivirus and restart the system. Then checks to see if the problem is solved and the update KB3093266 is correctly installed.

If you still do not get it, install it manually. To do this, download the update package from the official Microsoft download page:

http://download.windowsupdate.com/c/msdownload/update/software/crup/2015/09/windows10.0-kb3093266-x64_9b2f96010adf1b547157c79c8ad52bdb26815e0b.cab (493 KB)

Then, open a command prompt with administrative privileges and run the following command:

DISM /Online /Add-Package /PackagePath:FOLDER\windows10.0-kb3093266-x64_9b2f96010adf1b547157c79c8ad52bdb26815e0b.cab

Replaces "FOLDER" the route location where you downloaded the package (for example, if you have downloaded on the desktop, the path would be c:\users\user\desktop, where User is the username).

Press Enter, and wait until the package is installed, it may take several minutes until the message that the installation was successful appears. Press Y and the system will reboot automatically.
