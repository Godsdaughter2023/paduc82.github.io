raywood1 say:

How to Flow Text Around a Text Box or an Image in a Footnote

I'm using Word 2010. I have a long footnote. I want to put an image in the footnote. Normally (as far as I can tell) images can only go into a footnote inline. That is, the footnote's text won't flow around an image in a footnote. I would like for the footnote's text to flow around the image. I would be happy to put the image inside a text box in the footnote, but that doesn't seem to help.

Another possibility, useful in some of my footnotes, would be to send the image back of the footnote text, if possible.

jay freedman say:

There is no way to wrap footnote text around an image. As you'll have seen, when an image is inserted in-line in a footnote, the text wrapping controls on the right-click menu and on the Picture Tools ribbon are disabled. The reason for this is that footnotes and endnotes are not in the same layer as regular text, and the layout engine doesn't have code to perform text wrapping for footnote text.

There is a way to place an image behind footnote text. You must insert the image in the regular text on the same page and change its wrapping to Behind Text. Then you can drag it to the position in the footnote area. This should be done only after all other editing of the document has been completed, so the regular text and the image won't shift with respect to the footnote.
