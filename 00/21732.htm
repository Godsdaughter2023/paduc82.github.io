graham mayor say:
What I think you want is as follows, which will reproduce the values from the 14 cells in "Data entry" in the next row of the "Data" sheet.

Sub StartCopy()
Dim wksFrom As Worksheet
Dim wksTo As Worksheet
Dim iNextRow As Long

Set wksFrom = ActiveWorkbook.Sheets("Data entry")
Set wksTo = ActiveWorkbook.Sheets("Data")

'Find the next empty row
iNextRow = wksTo.Range("A" & wksTo.Rows.Count).End(xlUp).Row
If wksTo.Range("A1") <> "" Then iNextRow = iNextRow + 1

'Copy the values in the cells to their corresponding cells in the next row of the other sheet.
wksTo.Cells(iNextRow, "A") = wksFrom.Range("H10")
wksTo.Cells(iNextRow, "C") = wksFrom.Range("H11")
wksTo.Cells(iNextRow, "B") = wksFrom.Range("H12")
wksTo.Cells(iNextRow, "D") = wksFrom.Range("H13")
wksTo.Cells(iNextRow, "K") = wksFrom.Range("H17")
wksTo.Cells(iNextRow, "H") = wksFrom.Range("H18")
wksTo.Cells(iNextRow, "I") = wksFrom.Range("H20")
wksTo.Cells(iNextRow, "J") = wksFrom.Range("H21")
wksTo.Cells(iNextRow, "F") = wksFrom.Range("H22")
wksTo.Cells(iNextRow, "M") = wksFrom.Range("H23")
wksTo.Cells(iNextRow, "O") = wksFrom.Range("H27")
wksTo.Cells(iNextRow, "Q") = wksFrom.Range("H28")
wksTo.Cells(iNextRow, "L") = wksFrom.Range("H30")
wksTo.Cells(iNextRow, "P") = wksFrom.Range("H34")

Set wksFrom = Nothing
Set wksTo = Nothing
End Sub

A SIM card isn't needed to update the phone. When an update is ready to be installed on your phone you will get a notification on the phone if it is connected to the internet (a WiFi connection is fine) and if you've got the options on the phone set to show a notification when an update is ready for the phone.

 

If you plug your phone into your computer and check for an update via the Zune software, you'll also see a notification there when an update is ready to be installed on your phone. Since you checked and the Zune software did not indicate that an update is ready to be installed on your phone then there isn't an update for your phone.

 

If your phone is branded to a particular carrier you will get updates only after the carrier releases an update for your phone. (the make & model)...even if the phone is not locked to that carrier or even if you are not using the phone with that carrier.

this did it for me

I edited the registry values below as suggested here (https://forums.mydigitallife.info/threads/72329-DISCUSSION-Windows-10-Insider-Preview-Build-14986-(PC)?p=1299095&viewfull=1#post1299095) and the installation is going fine.
