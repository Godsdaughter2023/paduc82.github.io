jllatham say:

While I'd be more than happy to come up with a macro, you can do this with a formula.  Put this into your total column:

=SUM(INDIRECT(ADDRESS(ROW(),2)&":"&ADDRESS(ROW(),COLUMN()-1)))

and fill it on down as far as you need to.  The first ADDRESS() formula will always return B# as an address, where # is the row the formula is on.  The second ADDRESS() formula will always return the address of the cell on the same row 1 column to the left of where this whole formula is entered.  So...

With wk1 through wk3 in use and the formula in E2, it would be the same as =SUM(B2:D2).  Then when you insert a new column for wk4, it will be the same as =SUM(B2:E2).

A table HAS NO ORDER.

 

It should be viewed as an unordered "heap" of records.

 

If you want to see the records in some particular order on a Form, simply base the Form on a Query - not directly on a table - and specify a sort order in the Order By clause of the query; to do so in the query grid, just select Ascending on the Order By field under [File_Name].

 

The append query is not only unnecessary, as you have seen it won't even work!

I am not sure where you are looking for the Add to Document button, but it is definitely on the Envelopes tab of the Envelopes and Labels dialog

http://www.gmayor.com/EnvelopesAndLabels.htm

Also, below are the instructions from the Word Help File from earlier versions of Word for adding a logo to the return address facility. 

They still work except that to save the logo as an autotext entry, go to the Insert tab of the ribbon and from the Quick Parts dropdown, select Save selection to Quick Part Gallery, and in the Create New Building Block dialog that opens, insert the EnvelopeExtra1 in the Name control and change the Gallery to Autotext and then click on OK

To automatically include a graphic logo on all envelopes you print, you can
save the graphic, including its position on the envelope and formatting, as
an AutoText entry named EnvelopeExtra1.

1 In a graphics program (such as Microsoft Paint), create the graphic logo
you want. You can also use any existing graphic.
2 In Word, create the text portion of the return address.
3 Click where you want to insert the graphic logo.
4 On the Insert menu in Word, point to Picture, and then click From File.
5 Click the file that contains the graphic logo you want to use, and then
click Insert.
6 Click the graphic logo.

7 On the Insert menu, point to AutoText, and then click New.
8 In the Please name your AutoText entry box, type EnvelopeExtra1, and then
click OK.

Word inserts the graphic logo defined by EnvelopeExtra1 automatically each
time it prints an envelope.

Notes

· You can define up to two EnvelopeExtra AutoText entries. If, for example,
you want Word to automatically insert additional text such as promotional
text each time you print the envelope, you can create another AutoText entry
named EnvelopeExtra2. When you include both AutoText entries in a template,
then all envelopes created with that template will contain both entries. If
you don't want to include these entries, use a different template, or delete
the AutoText entries. 
· If you don't want to print the return address (shown in the Return address
box) on an envelope, make sure that the Omit check box is selected on the
Envelopes tab in the Envelopes and Labels dialog box. When the Omit check
box is selected, Word still includes the EnvelopeExtra AutoText entries when
it prints the envelope.
· You can also replace the default return address with a graphic that
already includes a return address or with just a graphic. To do this, click
the User Information tab (Tools menu, Options command). In the Mailing
address box, delete all the text and replace it with a single space.

If you have not entered a large amount of data (under 255 characters), you could create a custom table with the task name and the notes field.  Print that table.
