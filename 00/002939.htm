HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\16.0\Excel\Options

Sub PasteWithSourceFormatting()
Dim oSource As Presentation

oSource.Slides(1).Copy
'To paste the slide, first activate the target presentation
Presentations(1).Windows(1).Activate

oSource.Close
Set oSource = Nothing
End Sub

Even when you choose to delete your out-of-date security info, MS don't fully remove it for 30 days.

But the waiting period doesn't mean you're locked out of your Microsoft account.

The “Recent activity” page shows info about the activity in your Microsoft account, within the last 30 days.

jeeped say:
Try double-clicking a section of the title bar. This should bring the Word application window to full screen mode. Double-clicking a second time should toggle the window size back to the original.

To resize any application window without the mouse press Alt+<spacebar>, then tap S. This will put you in sizing mode, noted by the change in mouse icon. Use the arrow keys to adust both vertical and horizontal metrics then press <enter> when you are satisfied with the results.

Word should remember the last size and position of its application window when it is closed. Restarting Word should bring up a new instance of Word in the same size and position that the last one was closed as.

sp-vba-guy say:

Macros stripped from emailed workbook

I have an Excel 2010 workbook with a large number of worksheets.  On the first sheet I created some Command buttons, and behind each button is an on-click macros to hide/reveal and navigate to various worksheets.  When I email this file manually from Excel (File...Save & Send...Send as Attachment), the recipient receives the file and it includes these navigation buttons as expected.  When I email this file to a group of people using the macro listed described below, the file arrives but is missing the navigation buttons and the macros behind them.

Private Sub zzMailPrep()
    ThisWorkbook.Save
    
    Dim OutApp As Object
    Dim OutMail As Object

    Set OutApp = CreateObject("Outlook.Application")
    Set OutMail = OutApp.CreateItem(0)
   
    Dim TOLIST As Range
    Range("A2").Select
    Range(Selection, Selection.End(xlDown)).Select
    Set TOLIST = Selection
   
    Dim CCLIST As Range
    Range("B2").Select
    Range(Selection, Selection.End(xlDown)).Select
    Set CCLIST = Selection
   
    Dim MAILTO As String
    For Each Cell In TOLIST
    MAILTO = MAILTO & ";" & Cell.Value
    Next
   
    Dim CCTO As String
    For Each Cell In CCLIST
    CCTO = CCTO & ";" & Cell.Value
    Next

    
    With OutMail
        .To = MAILTO
        .CC = CCTO
        .Subject = ActiveWorkbook.Name
        .body = ""
        .Attachments.Add ActiveWorkbook.FullName
        .display

    End With

    Set OutMail = Nothing
    Set OutApp = Nothing

End Sub
