Like you, I use MS Security Essentials.  I have seen it recommended by many people in this forum and it seems okay to me.  I have Malwarebytes* for additional manual scanning.  I also have Windows Defender Offline* on a USB stick for rootkit hunting. [*  I normally update then run these monthly just to counter the possibility that something has found a way of hiding from Security Essentials but I accept that I ought to make the effort to do this weekly or at least before every online banking session.]


The best place to go for relevant advice is the forum section Virus and Malware. 


Here is what I recommend for virus cleaning.


0  Run malware scanning:-


0.1  If your routine scanner tells you there is an infection


0.2  If, during a browser session, you get a popup for one of those fake AV scanners & you have then closed the popup or a browser window conventionally.  The recommended action is to

0.2.1  Leave the popup & the browser as they are without touching anything.

0.2.2  Open Task manager's Application tab then close every browser entry in the list as well as your other open applications.

0.2.3  Open Task manager's Processes tab then close every iexplore.exe entry in the list [or the equivalent entry for other browser(s) in use].

0.2.4  Shut down the PC then start it again. 

0.2.5  Go to Control Panel, [View by - Small icons, if necessary], Internet Options and delete all temporary internet files & all cookies. 

0.2.6  Finally, run malware scans to confirm that you have escaped infection.


0.3  If there is an unusual occurrence on the system that has no other logical interpretation.  You can either rely on your own knowledge & experience for this decision or you can look around this whole forum [including but not limited to the Virus and Malware forum section] for the interpretations given by others for similar symptoms.  Since malware can take many forms, there can often be no definitive guidance on distinguishing a malware infection from a technical fault other than experience.


1  I recommend running several checks.  I suggest multiple scanners because some malware can hide itself from some particular scanners some of the time or even disrupt scanner installation but it would be more difficult for malware to survive every scan from an array of tools.


1.1  If you find anything during the set of scans recommended below then follow the removal advice that the scanner gives you and then get a second opinion by checking in the MS Community forum Virus and Malware as well - you will find excellent malware checking & removal advice there. 


1.2  Continue scanning after cleaning out all malware until you get a full set of clear results from every scanner used. 


1.3  Regard every one of your drives as suspect until they have passed scanning - you will probably want to focus on your main OS/data drive[s] at first but make a list of all your other drives so you can also work your way through them when the system itself is clean or you risk allowing the malware to sneak back onto your system. 


1.4  If every scanner reports that the system is already clean then review the infection symptoms again & consider the possibility that the system has a technical defect that is not related to malware.  "Sounds like malware, run AV scans" might now be as common a suggestion as "Reboot & try again".

 

2  Update your installed anti-malware utility and do a “full” scan using it [you may well have set it up to do only “quick” scans normally].  Some can be run in Safe Mode to detect malware that can hide itself during normal operations but not all anti-malware utilities run properly in Safe Mode & you’d have to check the Help file or their website to find out.  MS Security Essentials is one of those that can usefully be run in Safe Mode.

 

3  Scan with the onboard MS Malicious Software Removal Tool [MRT].  Go to the Start menu search box, enter mrt then right-click on the shortcut offered to you & select Run as Admin.  This tool is updated as part of Windows Update so your onboard version should already be current.  Run a full scan with it.

 

4  Download the Microsoft Safety Scanner and run a full scan with it.

 

5  Download the free version of Malwarebytes, install it & update it [for both of which you will need Admin approval] and run a full scan using it.  Note that this free version of Malwarebytes can exist alongside your installed anti-malware application [your Real-Time scanner] so it is always useful to retain it for use as an onboard second opinion. Do not run Malwarebytes in Safe Mode as too few Malwarebytes components load for it to give useful results.  If you have a failed or suspect installation of Malwarebytes then clean it up before trying again - Use mbam-clean.exe to completely remove Malwarebytes Anti-Malware.  If you have difficulty installing Malwarebytes because of the malware infection then  Use Malwarebytes Chameleon to install Malwarebytes Anti-Malware on an already infected system [Malwarebytes Chameleon is a component of Malwarebytes].

 

6  Make a Windows Defender Offline boot disk on USB [the initial download file is a small one that you then use to download the main installation file], update it, boot into it & run a full scan using it.  You will also need Admin permission to do this but you can use another PC to do this task.  Windows Defender Offline can be particularly useful for removing a type of malware called a Rootkit. 


7  Download SuperAntiSpyware Portable Scanner Personal Edition, install it [there is no need to update it] and run a “complete” scan.  You can save it to a USB / DVD / CD in order to run it from any PC.  You would need to download a new copy if you wanted to use it again in the future as its definitions were only current when it was downloaded.  You might notice that the term portable is used incorrectly in the software name as the software has to be installed to run.

 

8  A PC that has been cleaned of malware should then be restored from an image & all user files ought also to be restored from your backups.  The malware cleaning should only be regarded as ensuring that no malware can slip into the restored system.  The external drive[s] containing the image & the file backup should be scanned before use.  Whilst preparing for the image restoration, if you have to rescue updated [i.e. not yet backed up] user files from the main drive after the cleaning then scan the whole of the drive you save them on before putting these files back onto the PC. 

 

9  If you don’t make system images then the drastic tasks of restoring-to-factory or doing a new installation of Windows, reinstalling your applications as well as backing up then restoring user files ought to be undertaken.  If you have to rescue updated user files after the cleaning then scan them before putting them back onto the PC.  If you have to rescue updated [i.e. not yet backed up] user files from the main drive after the cleaning then scan the whole of the drive you save them on before putting these files back onto the PC. 

 

10  If you are not going to restore the complete system after cleaning all malware then at least consider running a Repair Install [also called an InPlace upgrade].


10.1  To run the Repair install, use the illustrated procedure* at Repair Install - Windows 7 Forums and also have a look at the MS summary at How to Perform an In-Place Upgrade. 

[*  But when the Repair install has all but finished & prompts you for your Product Key do not enter it, instead clear the box for Automatically activate Windows when I’m online then click on Next.]


10.2  After completing the Repair install, there is some tidying up to do.

10.2.1  Check your activation status at Control panel, [View by - Small icons, if necessary], System.  It is shown as the second-to-last entry near the bottom of the window.  You will probably find that its status has not been affected by the Repair install.  If it is no longer activated then try to activate it.  If, for Windows 7 preinstalled by the PC maker,  a first attempt to re-activate fails then use the Product Key on your CoA label [these systems are initially self-activated by a different key that the PC maker has installed in the system BIOS but that is not valid for manual use].  If it still cannot activate then phone the Activation Helpline - enter slui 4 in the Start menu search box to get the phone number.  When you call , press no buttons at all - stay on the line and a human operator will eventually answer so you can explain the situation.

10.2.2  Check drivers, applications & Windows settings as they might need to be re-installed or reset.

10.2.3  Check & adjust settings for your networks.

10.2.4  Re-enable your malware protection if you had to disable it to run Repair install.

10.2.5  Make a new system image.


11 If you are not going to restore the complete system or run a Repair Install after cleaning all malware then at least follow up the cleaning with full ChkDsk scanning & by running SFC.  These checks are intended to help correct any damage done to your system by the malware, they are not anti-malware steps themselves.

Hi Christine,

Double-click that and set its startup type to Automatic

Save that and close the Services app

That's It!
