john - project say:

Mucko72,

You're welcome and thanks for the feedback.

Perhaps the best way to demonstrate is with an example. I created a small test file with a few tasks.

Then I set up three custom Gantt Bar Styles. I used purple, orange and gray instead of your colors since the normal default bar style is blue and I wanted something that would more readily show.

Then I customized the Text2 field with a Lookup table. That is more robust than manually entering test strings which is prone to human error.

This is the result after running the below macro code on the file

Hope this helps.

By the way, if you want to learn more about Project VBA, I highly recommend Rod Gill's book on the subject. You can get more info at:

Option Compare Text
Option Explicit
Sub ChkTxt2()
Dim t As Task
'This assumes custom Gantt Bar Styles have already been created
'   with Flag1=Blue, Flag2=Navy and Flag3=Gray
'Do a little housekeeping before starting
'   expand file and clear Text22 and flags (if set from previous run)
OutlineShowAllTasks
For Each t In ActiveProject.Tasks
    t.Text22 = "": t.Flag1 = False: t.Flag2 = False: t.Flag3 = False
Next t
'Create custom filter to isolate those tasks with an entry in
'   Text2 for faster processing
FilterEdit Name:="Txt2Chk", create:=True, TaskFilter:=True, overwriteexisting:=True, FieldName:="Text2", _
    test:="does not contain", Value:="", ShowInMenu:=False, showsummarytasks:=False
FilterApply Name:="Txt2Chk"
'Create object for selected tasks and set parameters
SelectTaskColumn
For Each t In ActiveSelection.Tasks
    t.Text22 = t.Text2
    Select Case t.Text2
        Case "Wksp Fitter": t.Flag1 = True
        Case "Service Crew": t.Flag2 = True
        Case "Wksp S/Man": t.Flag3 = True
    End Select
Next t
FilterApply Name:="all tasks"
End Sub
