Hi n666

There is no way to set this up the way you are describing.

The Windows Explorer Favorites can only display Shortcuts to actual objects (Folders) in the file system. Similar to the IE Favorites that can only display shortcuts to websites.

When you select a shortcut in the WE Favorites list, the actual folder will be opened and displayed. This is simply a way to save time and the need to drill down into a deep folder heirarchy to open a particular folder.

You can create a parent folder and sub-folders for your project and add the parent folder to the Favorites list.

All of the shortcuts that are displayed in the Favorites menu are stored in the C:\Users\your name\Links folder.

Hope this helps.

Hi, this error as you can see indicates some sort of storage space problem or even a failed or corrupted storage device. On the console go to settings/ system/ storage, does your storage device show as unformatted, if you select the storage device do all the files show a size of 0Kb. Those are indications of a storage device issue.

-

Should the storage device show either of the above symptoms then a format is really the only option. To format a storage device, go to settings/ system/ storage, highlight the storage device in the list and press Y on the controller, you will see the format option, select that and follow the onscreen instructions.

-

In some cases it could just be a failed update, to try and fix a failed update, go to settings/ system /storage, highlight your storage device and press the following buttons in sequence on the controller. LB RB X LB RB X select yes on the next screen, the console will restart and attempt to update again.

-

There is also the option to try updating using a USB flash drive or CD, this process is described in the link below along with some other things to try.

-

http://forums.xbox.com/xbox_forums/xbox_support/xbox_360_support/f/7/t/166045.aspx

-

The worst case scenario is your storage device has failed, this would mean arranging a repair.

-

http://support.xbox.com/xbox-360/repair/repair-process
