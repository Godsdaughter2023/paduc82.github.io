    You can set up your account's caller ID to let your friends, family and business contacts know you're calling them. When you have set up caller identification, your Skype Number (as long as it is supported) is displayed when you call mobiles and landlines from Skype.

    Follow the steps below to set up caller identification using your Skype Number:

    • Sign in to your account.
    • In the Manage features section, click Caller ID.
    • Click select a pre-saved number.
    • From the bulleted list displayed, select the Skype Number that you want to use as your caller identification.
    • Click Save Settings. Your caller identification is now set and ready to go.

    Note: Caller identification using your Skype Number is available in: USA, UK, Chile, Denmark, Estonia, Poland, Sweden and Switzerland. Please refer to this link for additional details. What is caller identification and how do I set it up?
