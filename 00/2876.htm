kelwynhough say:

Excel - Duplicate Values

I have a spreadsheet list First Name and Surname.

First Name Family Name
Mabel Allchin
Lucy Johnson

Is it possible to highlight the row only when both the First and Family name match on a row



hansv mvp say:

Let's say First Name is in column A and Family Name in column B, with data starting in row 2.

Select the range; I'll assume that A2 is the active cell within the selection.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula

=COUNTIFS($A:$A,$A2,$B:$B,$B2)>1

Click Format...

Activate the Fill tab.

Select a highlight color.

Click OK, then click OK again.

Here's my scenario.

f z z say: Excel interacts with the Windows clipboard (never used Macs, so have no idea about Mac Excel) differently than any other Windows application either from MSFT or any other source.

Excel itself clears any objects it puts into the clipboard when it inserts new worksheets (as well as just before performing various other operations), so it appears Access may be allowing Excel to treat Access objects in the clipboard the same as Excel treats its own.

Bad news: this is intentional behavior for Excel. Excel has been this way since the first Windows version in 1987, so it's extremely unlikely this would ever change. There's no way to turn this off. You're going to have to get into the habit of performing these steps 2-1-3.
