Prefix Zeros dropping off when record is created in MS Access 2010 - Microsoft Community

racheltalmo say:

Prefix Zeros dropping off when record is created in MS Access 2010

I have a database which requires us too assign a unique control ID of 5 numeric s, we're rolling over from 99999 too 00101 and the database doesn't recognize the prefix of the 3 zeros, is there a setting to recognize or should I start with 10101? 
ken sheridan say: You have two options:

1.  Use a column of text data type.  Leading zeros can then be stored as part of the string.

2.  Use a column of long integer number data type, which is what I assume you have at present, and format the value as "00000".  This will show the values with leading zeros, but the actual values stored in the column will be numbers.

You'll sometimes see it said that a value composed solely of numeric characters should only be stored as a number data type if mathematical operations are to be done on the values.  I don't go along with that entirely, however.  Account must be taken of the semantics, and I would take the view that if a value has an ordinal or cardinal significance, then it's a number, and is appropriately stored as a number data type regardless of whether any mathematical operations will be done on the values.  If one the other hand the value is simply a string of characters which happen to be numeric digits, but have no ordinal or cardinal significance, then a text data type is more appropriate.  Often such values are actually encoding systems, a US zip code is a case in point, rather than numbers as such.

These are the following reasons why emails are missing from your mailbox:
Your account has been marked as inactive

http://windows.microsoft.com/en-US/hotmail/how-recover-deleted-messages

If you have configured your email address on an email client, please  set your option to leave messages on the server.
http://windows.microsoft.com/en-US/hotmail/send-receive-email-from-mail-client
(Refer to Prevent losing email in Hotmail when you've set up POP3)

To check, your email rules please follow these steps
(for Hotmail)
1. Sign in your Hotmail account and go to your inbox.
2. Move your cursor over the Folders link on the left pane.
3. Click on the Gear icon, and then click Manage Rules
4. Check for message rule that cleans up your messages and delete it.

(for Outlook)
Gear on the right>more mail settings>customizing Outlook> managing rules....
