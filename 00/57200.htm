How to Add symbols to existing image.

kosh vorlon say:

Hi Susieque,

 

I just did it using Paint and the Character Map.  Here's how, in a nutshell:

 

1.  Open your logo in Paint and make it the size you want.  I used a .jpg file, though I suspect it should work with any type Paint can handle (actually, I used my Avatar - LOL!).

 

2. Now in Paint, open a Text Box big enough to accommodate the size you expect for the

Â® where you want it to be.

 

3. Now, open Character Map and find the symbol.  I used Arial to be simple about it.  Click on it and then click copy.

 

4.  Go back to Paint and go to the text box and inside the box hit CTRL-V to paste the Symbol.  When I did it, it was microscopic.

 

5.  Go to View and check Text Toolbar.

 

6.  Make sure you're still in Textbox mode.  Click to select the symbol inside the text box.  The Text Toolbar will either be there already or will show when you go to Textbox mode or select the symbol.  Mine was set at 8.  I changed it to 72 (or whatever size you want).

 

7 After leaving Textbox mode, the symbol showed just where I wanted it.  I did not try this on top of the image but just beside it.  Since it was already in the new image, I saved it.  It became part of the new image.

 

8.  Insert it where you want.  No cropping should be needed since it's exactly what you want.  Resizing will simply resize it all just like any other image so no cropping - just simply grabbing the image and changing it in the document or whatever you use to design the webpage - selective zooming if that feature exists in the program.  It shouldn't be any different than any other image.  If you need to resize it elsewhere, just bring it back as is to Paint and resize the entire thing there based on pixels and save it when you have the size you want and save the result as a new version and then paste the result where you need it.

 

That's it.

 

I hope this helps and works for you.  There are other options, but I tried to stick with what I knew you had so you wouldn't need more software.

 

Good luck!

 

Kosh

13. Enter your Xbox login information

16. Select Settings

17. Select Your Account

I cannot tell from your question whether or not you have tried to reset the power plan to default.  This can be useful or it can have no useful effect, it is merely a quick initial response that ought to clear the decks.  You'll see the option to do so in the power options dialogs:

You can also check through the Advanced power settings to see if there are any apparently contradictory entries there.  There will probably not be.

Check your display driver [Device manager, Display, Update driver] as problems with that can cause baffling symptoms.

You could try to understand the problem better by using PowerCfg to check just what power plan settings the system actually has set on it [as opposed to what power plan settings you have told the user interface to set on it].  In an elevated command prompt, enter

PowerCfg -qh >"C:\Users\YourUserName\Desktop\PowerPlanProperties-Balanced.txt"

[choose any convenient folder & filename]

I paste the output into Excel to look through the very long output file. 

It might look daunting at first but is just a large number of repetitions of a simple pattern -

    Power Setting GUID: 245d8541-3943-4422-b025-13a784f679b7  (Power plan type)
      Possible Setting Index: 000
      Possible Setting Friendly Name: Power saver
      Possible Setting Index: 001
      Possible Setting Friendly Name: High performance
      Possible Setting Index: 002
      Possible Setting Friendly Name: Balanced
    Current AC Power Setting Index: 0x00000002
    Current DC Power Setting Index: 0x00000002

The initial GUID entries for each group are the reference number[s] for that item

The last two lines tell you what the settings are for AC & DC. 

The intervening lines tell you what units are used for the setting and, where relevant, what the range of possible settings is.  I chose a simple example above but the units could be %, seconds, Yes/No or many other things.

As you are investigating the computer going to sleep, search for the word sleep in the output file.

You'll find an entry like this one from my computer:

    Power Setting GUID: 29f6c1db-86da-48c5-9fdb-f2b67b1f44da  (Sleep after)
      GUID Alias: STANDBYIDLE
      Minimum Possible Setting: 0x00000000
      Maximum Possible Setting: 0xffffffff
      Possible Settings increment: 0x00000001
      Possible Settings units: Seconds
    Current AC Power Setting Index: 0x00000000
    Current DC Power Setting Index: 0x00000000

The last two entries tell you what sleep delay setting the system believes it has been told to use.  If, as you describe, the computer is going to sleep after 1 minute then you will see the entry 60 [well, 3C as it's hex].  That confirms for you that the system thinks that is what it has been told to do. 

If this is not what was set [Power options, {Balanced, or whatever plan you are investigating}, Change plan settings, Change advanced power settings, Sleep, Sleep after] then you can focus your investigation on a fault existing in the Power options UI.  I'd say that your initial response ought to be to  boot into Safe mode and see if the fault persists - if it does then this result would indicate that it is Windows itself at fault so run SFC / scannow in an Admin command window.  If this does not fix it then make an installation disk [see
