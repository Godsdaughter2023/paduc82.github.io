charles kenyon say:

Hi,

This is most likely a proofing language problem. The "proofing language" is a key to several Word features including spelling and grammar and AutoCorrect. It is not an application-wide setting nor even a document setting; it is set at the character level! To see if this is the source of your problem:
•    Select (or create and select) a mis-marked word.
•    Then under the Review tab, click on the "Language" button.
•    Pick "Select Proofing Language" from the short menu.
•    Look at the language set in the dialog .
•   If it is wrong, select the correct language.
•    Then OK.
Does this make a difference?

If it does make a difference, then Ctrl+A to select all text in your document and repeat the language selection.
If the problem occurs regularly, it is probably a problem with one or more Style definitions. If this is the problem, write back for more help.

This is a character(font)-level formatting in Word. When  you paste text, it will carry language formatting with it.

You may want to look at Suzanne Barnhill's article on the spelling checker. It is the definitive article. http://wordfaqs.ssbarnhill.com/MasterSpellCheck.htm

    j.e. mcgimpsey say: ⌘-t will cycle between absolute, relative, and mixed addressing in Edit mode (while editing the cell). It will not change addressing mode just with the cell selected.

    You can certainly enter the formula in absolute mode directly by adding the dollar signs:

       =$C$1 + $D$1

    Also, if you copy the formula while editing the cell, then select your target, and either double-click the cell or click the formula bar to enter Edit mode, when you paste, you'll get an exact copy, regardless of addressing mode.
