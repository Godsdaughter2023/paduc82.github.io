greg maxey say:

Put the cursor between the brackets:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oRng As Word.Range
Set oRng = Selection.Range
oRng.MoveStartUntil CSet:="[", Count:=wdBackward
oRng.Start = oRng.Start - 1
oRng.MoveEndUntil CSet:="]", Count:=wdForward
oRng.End = oRng.End + 1

oRng.Select
End Sub

is it possible to add cross references to a SmartArt org chart? - Microsoft Community

saramorin say:

is it possible to add cross references to a SmartArt org chart?

I am trying to add cross references to an org chart created with SmartArt - can you please assist? It doesn't seem to be working by using Ctrl F9 or creating with the references tab. Thanks!

jay freedman say:

It doesn't appear to be possible to put anything other than plain text into a SmartArt label.

As an attempted workaround, I inserted a cross-reference in the body text, cut it to the clipboard, and pasted into the Text panel of the SmartArt. The current value of the cross-reference appeared there, but it wouldn't update when the referenced text was changed -- it was just plain text.

An effort-intensive workaround, if you really need updatable cross-references in the SmartArt, is to put them into text boxes that float on top the SmartArt. The text box needs to be set to no fill color and no outline, and the text inside needs to be formatted to match the rest of the labels. If/when the target of the cross-reference changes, you'll have to update the cross-reference manually -- either click on it and press F9, or use a macro like this to update all of the text boxes at once:

Sub UpdateTextboxRefs()
    Dim shp As Shape
    
    For Each shp In ActiveDocument.Shapes
        If shp.Type = msoTextBox Then
            shp.TextFrame.TextRange.Fields.Update
        End If
    Next shp
End Sub

Even when you choose to delete your out-of-date security info, MS don't fully remove it for 30 days. This waiting period helps to deter hackers from updating your security info and gaining access to your account without your knowledge.
