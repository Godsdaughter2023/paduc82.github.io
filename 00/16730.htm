ossiemac say:

@Rick. Your code deletes all rows including column headers. Think you meant .Replace "Mgr", "=" & """Mgr""", xlWhole but need to address the column headers also. I like the idea behind it.

Alternative method. Apply AutoFilter and then select the rows you want to keep instead of using an InputBox.

Then run the macro below which deletes all of the hidden rows.

See comments in code for options to either just ShowAllData or remove AutoFilter completely after deleting hidden rows.

Has the advantage of being able to accurately select the values to keep from the dropdowns instead of using an InputBox where errors can be made entering data.

Note: Really only suitable for xl2007 and later. Earlier versions do not have option of multiple selections in AutoFilter.

Sub DeleteHiddenRows()
  Dim i
  Dim lngLastRow As Long
   
  With ActiveSheet
    If .FilterMode Then
      lngLastRow = .AutoFilter.Range.Rows.Count
     
      'Must work backwards when deleting rows
      For i = lngLastRow To 1 Step -1
        If .Rows(i).Hidden Then
          Rows(i).Delete
        End If
      Next i
     
      .ShowAllData  'Just resets the dropdown. Doesn't really do anything meaningful.
      
      'Alternatively remove AutoFilter completely
      '.AutoFilterMode = False
     
    Else
      MsgBox "No filters set therefore no rows to delete." _
          & vbCrLf & "Processing terminated."
      Exit Sub
    End If
  End With
End Sub
