frederik long say:
In Windows Explorer, I can see a top level directory named after my old computer.
Windows "knows" that the folder G:\OldComputer is a backup folder. When you now launch Windows Explorer and double-click this folder then you get a prompt to restore your files.

justinwthomas say: Unfortunately, I received no such prompt.

frederik long say:
Unfortunately, I received no such prompt.
Windows looks for the hidden file J:\OldComputer\Desktop.ini with the following contents:

[.ShellClassInfo]
DirectoryClass=WindowsBackupFolderOptions
IconResource=%SystemRoot%\system32\sdclt.exe,-5501

The presence of this file tells Windows that OldComputer is a backup folder. You now need to ask yourself why the file is missing, unreadable or has the wrong contents.
