Column A contain PAN(AAAAA3456B)

andreas killer say:

Copy the code below into a regular module, save the file into the same directory as the PDF files and execute.

Warning! Be sure to have a backup copy of that directory! There is no error checking in the code below!

Andreas.

Sub Test()
  Dim R As Range
  Dim OldName As String, NewName As String
 
  'Visit each used cell in column A
  For Each R In Range("A1", Range("A" & Rows.Count).End(xlUp))
    'Search for the old file
    OldName = Dir(ThisWorkbook.Path & "\" & R.Value & "*.pdf")
    'While found
    Do While OldName <> ""
      'Prepare the new name
      NewName = R.Offset(, 1) & " " & OldName
      'Rename the file
      Name OldName As NewName
      'Search again
      OldName = Dir(ThisWorkbook.Path & "\" & R.Value & "*.pdf")
    Loop
  Next
End Sub
