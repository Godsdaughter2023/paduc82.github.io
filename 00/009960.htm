diane poremsky mvp say:
Hi spete,
What type of email account do you have configured in Outlook? POP, IMAP, Outlook.com, or Microsoft Exchange?

If you use Outlook .com or IMAP, the mail is stored on the server and would resync when you add the account back. If you use POP or a PST file, it would be on the hard drive. If you reformatted before reinstalling Windows and did not have a backup copy of your data, the folders were lost.

spete58 say:

its showing IMAP but not sure if I was using POP or PST in the past.

diane poremsky mvp say:
If the account was created in an older version of outlook, it may have been a pop account. Outlook 2010 and older defaulted to using POP, newer versions always use IMAP if it is supported.

If the mail is not in the imap account, it probably was a pop account with a pst file. If you did not use the Windows option to preserve your data, the pst would be lost unless you had a backup copy. Sorry.

I was using Outlook 2010 so is there any way of getting the folders back with the IMAP?

diane poremsky mvp say:
If you use POP with outlook 2010, you need the pst file you were using with it to get the folders back. They would not have uploaded to the IMAP server.
