john - project say:

Shahid,

That's because the consultants are not fixed cost per the Project definition of fixed cost.

I'm assuming the consultants charge a flat rate per meeting or perhaps per day. On the Resource Sheet, instead of entering a normal hourly rate for each consultant, enter a Cost/Use. Then when assigning those consultants to a meeting task, their "use" is 1 meeting regardless of how many hours (or days) that meeting spans. If their rate is per day, then for each day they will be working, assign that as a "use". davidf2 say:

Publisher is a DTP and gives you the tools to build and layout just about any grid or 'graph paper' type page layout that you want but you will have to do the work. There is no automagic way of doing the things you ask.

Reference: Understanding the Layout Guides dialog box in Publisher: https://support.office.com/en-NZ/article/Understanding-the-Layout-Guides-dialog-box-in-Publisher-bb583994-a9fa-47d3-a29a-cc14d18b7094

Scroll down to the Grid Guides tab. Note that you can choose the number of columns and rows and the spacing to create any grid size you want.

Once you have the grid created you can insert lines, dots or what ever type of design element on the page that you want and use the guide lines to help you layout your page.

For lines go to Insert > Shapes > or Home > Shapes and choose the line. Click where you want to start drawing the line and if you hold down the shift key you will find it easier to get a perfectly horizontal or vertical line. Then right click the line > format autoshape and you can choose the color and weight of the line and a dashed version if you prefer.

Then once you get your line styled the way you want it go to Page design > Layout group and check Align to Guides and Objects. Now you can right click, drag the line to the next guide line and it will 'snap to' that guide line > Copy Here, repeat as many times as necessary and create your grid quickly. You will also find it easier to get other design elements you might want to insert, to align to your guides if you use this 'snap to' approach.

If you don't want to use the guide lines approach then go to View > and check Rulers and you can use it to align your lines.

I have also found it helpful sometimes to create a specific sized 'spacer' text box filled with a color and use it with 'snap to' to align elements in relation to each other. I just drag it and let it snap to a particular element and then drag the second element until it snaps to the spacer box and then move the space box to the scratch area until I need it again.

You can also insert a table in Publisher but you won't get as many options to customize the look of the lines etc. that you will if you manually create the grid.

If you can create the type of page you want in Excel, then use Excel. Sorry but you will not find all features of Excel, Word or the other Office programs in Publisher. You might be able to copy and paste some of the elements you created in Excel or Word to your Publisher page.

Regarding you couldn't sync files someone shared with you, I'd like to confirm whether you sync these files in Shared With Me section of your OneDrive for Business

If so, I'd like to explain that we can't sync a single file and we need to sync the container of the file using OneDrive for Business sync client. The container can be either a folder or a library. However, the Shared with Me section is just a view that lists all items Shared with you but not an Actual SharePoint library.

Therefore, when someone shares an item with you, there are two scenarios:

1.       if someone shares a folder with you, you can go to that folder and then directly sync all items in it down using             OneDrive for Business sync client.

2.       If someone shares a single file with you, we can't sync it directly in Shared with me section. But we can take the             following steps to sync it down using OneDrive for Business sync client:

          a.  Open your OneDrive for Business library in a browser, then click Return to classic OneDrive on the                              lower-left corner

          b.  Under the classic interface, click the ellipsis next to the file name, then choose View Library.

          c.   You can see the Sync button on the ribbon to sync the library down with Edit permission.

Besides, the files or folders under a library someone else shared with you will not appear under the Shared with me section of your OneDrive for Business. You can browse to that library directly to sync files under that library.
