How to display data that are outside of the current axis range?

Does anyone know how to display data that are outside of y axis range in excel? i have data that range from -0,2 to 1,2 and i would like to plot a graph with y axis having minimum=0 and maximum=1 but to still show the data that fall outside the axis range.

Thanks you for any advice.

bernard liengme say:

Let us say your x-values are in column A and the y-values in B with headings

In C2 enter =IF(B2<0, 0, IF(B2>1.2, 1,2, B2))

Drag this down to process all y-values

Select all the entries in column A; hold down CTRL and drag over column C; now make the chart

Your low values will plat as 0 and the high values as 1.2

Alternative: in C2 use =IF(B2<0,0,NA() and in D2 use =IF(B2>1.2,1.2, NA())

Drag the two down to the last row

Make a chart of all the data

Now your low and high values will have their own colours.

somasundaram pandiaraj say:

As a security measure Windows 7 does not run applications as an administrator by default. To run Word 2007 as an Administrator follow the steps below:

 

Click on Start > All Programs > Accessories > Run

Copy and paste the following text in the Open field and then press Enter:

 

%ProgramFiles%\Microsoft Office\Office 12

 

Locate Winword (.exe) on the current folder then right-click on it and select ‘Run as Administrator’

 

However to remove the template(s) you may also try the steps given below:

 

1)    Click on Start > All Programs > Accessories > Run

2)    Copy and paste the following texts one at a time in the Open field and then press Enter

 

a.      %ProgramFiles%\Microsoft Office\Office12\Startup (For a 32-bit Windows)

 

                                                       Or

 

               %ProgramFiles(x86)%\Microsoft Office\Office12\Startup (For a 64-bit Windows)

 

 

b.     %Appdata%\Microsoft\Word\Startup

 

 

3)    Delete all the files from the above locations

 

 

If the issue still persists then remove ‘Normal.dotm’ from the following location:

 

Note: Ensure to backup the ‘Normal.dotm’ if you have customized it.

 

%Appdata%\Microsoft\Templates
