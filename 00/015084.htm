rob van der burg say:

I had the same problem and it seemed that one of my folders was in a sort of infinite loop. You can see this when you go to your file explorer and refresh it.

The file name will change constantly: <Filename>_<Machine Name_<number which changes constantly>

After pausing OneDrive, Rebooting, Renaming the files, Stopping OneDrive and a few other tries I was able to fix it by cleaning the OneDrive Cache.

How does that work?

  • Open a command prompt by going to start and type "command" or "cmd" if you want to show off :-).
  • In the command box type or copy the following command:
  • %localappdata%\Microsoft\OneDrive\onedrive.exe /reset

This will clean the cache of your client or clients if you have both business and personal OneDrive.

The OneDrive icon should reappear after a few minutes, if not you can also go to start, type "onedrive" and select it

In my case, when all was done, I had to rename the folder mentioned above which had a temporary filename, something with "file~name-<machine_name>-<number>" (notice the "~" in the filename)

Your files should not be affected, you can always check this by going online to https://onedrive.live.com. and see the source of your files.

As usual before you do anything make sure you feel save about what you do!

cmdrkeene say:
It's sharing code and programming with other office applications and the operating system.  It's shared system resources, not shared user data.

What I believe you're asking is you want to sync your phone contacts with Outlook, without them touching Google's servers.  You're in luck because you already use Exchange, which can already seamlessly sync contacts, calendar items, and to-do (task) list items in addition to email.  This is typically configured on the settings on your phone in the same place where you set how often it downloads your mail, but it varies by phone.

One concern may be this: on many phones, when you create a contact, it will ask if the contact should be stored on the phone or in Exchange.  If your contacts were already created on your phone before your phone joined the Exchange server, it may be difficult (likely not possible) to seamlessly sync them to your Outlook.  In this case you'd have to check to see if your phone has an export option from the contacts list, which can be used to export to a single export file (usually a CSV file) that can be imported into your Outlook to get the process rolling.
