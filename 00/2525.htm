andreas killer say:
What my question is from among the 750 duplicates, in col-b, which 2 cells having the same duplicate part numbers and how to group the entire 750  duplicates to delete  any one duplicate.
I guess you mean which cells in column C has duplicates of the 750 duplicates in column B!?

Run the macro below and look if the selected cells are the cells that you are looking for.

Andreas.

Option Explicit

Sub Main()
  Dim All As Range, R As Range
  Dim Duplicates(), Items()
  Dim i As Long, j As Long
  Dim Second As Boolean
 
  'Get all duplicates in column B
  Duplicates = GetDuplicates(Range(Range("B1"), Range("B" & Rows.Count).End(xlUp)))
  For i = 0 To UBound(Duplicates)
    'Get all duplicates in column C for this duplicates of column B
    Items = GetDuplicates(Intersect(Duplicates(i).EntireRow, Columns("C")))
    For j = 0 To UBound(Items)
      'Combine all duplicates, except the first one
      Second = False
      For Each R In Items(j)
        If Not Second Then
          Second = True
        Else
          If All Is Nothing Then Set All = R Else Set All = Union(All, R)
        End If
      Next
    Next
  Next
  'Show all duplicates if any
  If Not All Is Nothing Then
    'Change .Select to .Delete to delete the lines
    All.EntireRow.Select
  End If
End Sub

Function GetDuplicates(R As Range, _
    Optional ByVal Compare As VbCompareMethod = vbTextCompare) As Variant
  'Return all duplicates in R as array of ranges
  Dim C As Range
  Dim i As Long, j As Long
  Dim Item(), Result()
  Dim Dict As Object 'Dictionary
  Set Dict = CreateObject("Scripting.Dictionary")
  Dict.CompareMode = Compare
  For Each C In R
    If Dict.Exists(C.Value) Then
      Item = Dict(C.Value)
      If Item(0) = 1 Then i = i + 1
      Item(0) = Item(0) + 1
      Set Item(1) = Union(Item(1), C)
      Dict(C.Value) = Item
    Else
      Dict.Add C.Value, Array(1, C)
    End If
  Next
  If i = 0 Then
    GetDuplicates = Array()
  Else
    ReDim Result(0 To i - 1)
    Item = Dict.Items
    For i = 0 To UBound(Item)
      If Item(i)(0) > 1 Then
        Set Result(j) = Item(i)(1)
        j = j + 1
      End If
    Next
    GetDuplicates = Result
  End If
End Function

The DOM Storage includes two related mechanisms for persisting client-side data in a secure manner using the Document Object Model (DOM), sessionStorage and localStorage. These objects were introduced in Internet Explorer 8.DOM Storage is often compared to HTTP cookies. Like cookies, Web developers can store per-session or domain-specific data as name/value pairs on the client using DOM Storage. However, unlike cookies, DOM Storage makes it easier to control how information stored by one window is visible to another.

More information refer to the link “Introduction to DOM Storage” http://msdn.microsoft.com/en-us/library/cc197062(VS.85).aspx

3.To clear AutoComplete or web address history, click the General tab. Under Browsing History, click Delete. In the Delete Browsing History dialog box, select the Form data and Passwords check boxes.
4.To clear web address entries, follow the steps for Clear History below.
