The "line spacing" setting applies only to the space between two lines that are part of the same paragraph. When you press Enter, the next line is in a different paragraph, so the setting that applies there is usually the Space After setting of the first line. (If the second line is in a different style, the space might be the Space Before setting of the second line.)

To remove that space, select all the lines of an address except the last one, and apply the No Spacing style. (If that style doesn't appear in the Home tab, press Ctrl+Shift+S to open the Apply Style dialog box, and start typing No Spacing; when the name appears, click the Apply button.)

I already have a worksheet change event for the Lookup sheet that forces the DESC field to clear old info whenever a new id # is entered , so I'm not sure if code would need to be added to that or if I would need a selection type of event.

Thanks for your help!

edwin tam say:

I assume your "drop down list" is a validation list in a cell.

Assume the address of your "list" is D3.

You can place a macro like this into your sheet (Sheet, not the module)

Private Sub Worksheet_Change(ByVal Target As Range)
If Target.Address = Me.Range("D3").Address Then
    Me.UsedRange.Rows.AutoFit
End If
End Sub

This macro fires when some change is made to the worksheet. First, it checks whether the change is made to D3 (your list). If yes, autofit the rows.
