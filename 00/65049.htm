vijay b say:

Hi,

Thank you for posting in Microsoft community! You've reached the right forum for your question.

For a media file to be written onto a disc, along with the duration of the file it also checks for the size of the file on disc.

To check if the media file you are trying to burn is within the supported size,

a. Browse to the location of the file.
b. Right click on the file and select properties.
c. Look for the size on disk.

For a CD, the size should not exceed 700MB and for a DVD, the size should not exceed 4.7GB

Refer the section which talks about “What are the differences between the various kinds of CD and DVD discs”.

http://windows.microsoft.com/en-US/windows7/Which-CD-or-DVD-format-should-I-use


If it is a single file, then you cannot split it during the burning process. In such cases, media player will tell that there is no sufficient space in the disc and the burning process would not be successful.


If it is a playlist or set of multiple files, then it would ask for the second disc automatically when the first disc is full.


Have a look at the below article for more information of burning a CD/DVD using media player:

http://windows.microsoft.com/en-US/windows7/Burn-a-CD-or-DVD-in-Windows-Media-Player-frequently-asked-questions
