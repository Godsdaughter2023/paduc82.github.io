How to export contacts from outlook.com, and import them to outlook - Microsoft Community

nathan roberts (sunknath) say:

How to export contacts from outlook.com, and import them to outlook 2007

Technical Level : Basic

Summary

If you have been trying to work out how to export contacts from Outlok.com, and import them onto Outlook 2007, desktop app, but have had not luck, or just don't know what to do or where to begin. Then this article should help you get sorted out!

I will keep the article short and snappy, so you can quickly move on, and enjoy your day. I will also go though all of the steps, and explain the best I can for you all. 


Details

To begin, you will first need to export your contacts from your Outlook.com account. To do this, please follow these instructions below:

  1. Login to your Outlook.com account.
  2. Click the down-arrow at the upper-left corner of your screen.
  3. Choose People.
  4. Click "Manage" in the menu bar. 
  5. Choose "Export for Outlook.com and other services".
  6. It automatically opens the file in Excel

Once you have got all of your contacts in the Excel file, you are good to go for the next step!

Format the contacts within Excel

Before we can import all of your contacts into Outlook 2007, we will need to make sure you have correctly formatted it before trying. Otherwise, you may start to see errors, or it may not be imported correctly. 

To format your file, you will need to follow these steps:

  1. Open the file in Excel.
  2. Click on the header for column A to highlight the entire column.
  3. Click "Data" in the top menu.
  4. Click the Text to Columns button.
  5. Select "Delimited".
  6. Click Next.
  7. Put a check mark by "Comma".
  8. Uncheck "Tab".
  9. Set “Text qualifier” to {none}. If you don’t, excel will eliminate all international suffixes from phone numbers (excel can’t have 0’s in the beginning of a number), and it will treat all international numbers that begin with a + in a bizarre manner. You don’t want to have to sort all this mess out.
  10. Save the file as “other formats”, the default is CSV (Comma delimited).
  11. In the “save as” dialog box you’ll be prompted with compatibility issues (“… [file name] may contain features that are not compatible with CSV (Comma delimited). Do you want to keep the workbook in this format?”).
  12. Click yes and proceed.

Treating the file for compatibility issues

For an unknown reason, once saved by excel 2007, the *.csv file will contain semicolons (;) instead of commas (,) as a separator.

This will happen every time you save the *.csv file with excel 2007.

You need to replace every semicolon (;)  with a comma (,), or else Outlook won’t be able to properly map the destination fields when you import the contacts. Also, since we unchecked the option to use quotation marks (“) as a text qualifier in excel initially, the *.csv file will keep all information that was in the cells between quotation marks. You need to eliminate them before importing to outlook 2007, unless you want all the quotation marks to go into your contacts.

To replace all semicolons with commas, you will need to open this in Notepad. This is so you can make changes to the *.csv file.

To do this, you will need to follow these instructions: 

  1. Open notepad
  2. Open the *.csv file. (Remember to check “all files” in the open dialog box, so notepad can see the *.csv file)
  3. Choose Edit then Replace. 
  4. Replace all semicolons ( ; ) with  commas ( , )
  5. Then  replace all quotation mark symbols (“) with nothing (don’t fill in the field). Quotation marks will disappear.
  6. Choose File then Save. (Do not use “File & Save as” or it will be converted to a *.txt file and you will need to convert the file extension before importing)

The last step- Importing your contacts into Outlook 2007

(Note: make sure the contacts file is in *.csv file format. If not, change the extension.)

Now that you have saved the *.csv file on your computer, with all of your contacts, and have removed the replaced the semicolons with commas.(And removed the quotation marks). You are now ready to import your contacts on Outlook 2007. 

To do this, please:

  1. Choose File then Import and Export.
  2. On the Import and Export Wizard window, choose “Import from another program or file” and click Next.
  3. Select “Comma Separated Values (Windows)” and click Next. Works just as well if you select “Comma Separated Values (DOS)”.
  4. Browse to the location where you saved the *.csv file, select it, select the option of your choice for handling duplicates and click Next.
  5. Select the Contacts folder as the destination and click Next.
  6. Click Finish to complete the process.
