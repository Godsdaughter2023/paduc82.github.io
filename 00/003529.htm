When I installed Voice Command to my HTC Touch Diamond2, an icon did not show up in the Programs screen of my phone, and I was not able to choose Voice Command when I attempted to assign it to a programmable button on my phone.  So you may need to create a shortcut for Voice Command and place it in the \windows\start menu\programs folder before you'll be able to assign Voice Command to one of the programmable buttons on your phone.  See this article to learn more about creating shortcuts on a windows mobile device.

Just so you know, that this was a great solution.  I also suddenly realised that you can specify a background image by going to Page Layout -> Page Color -> Fill Effects -> Picture and select a picture that way to actually become a set background image.

Instead, I was just using Insert Picture method and have it stretch 100%.  This is not the way to do it.

judith_c. say:

Hi Carolyn Wilson,

 

It is possible that a third-party pop-up blocker blocked the link or Hotmail deactivated the links.

 

The pop-up blocker feature prevents pop-up windows from opening when users click hyperlinks in email messages. To enable pop-up windows to open, please follow the steps below:

 

Windows XP:

 

a. Click the Internet Explorer information bar, and then select one of the following options:

o Temporarily allow pop-ups: This option opens the blocked pop-up window. Additional pop-up windows will not open when you click hyperlinks. 

o Always allow pop-ups from this site: This option adds the hyperlink that you clicked to a list of safe websites, and then opens the blocked pop-up window and any additional pop-up windows on the current site. 

 

To access the pop-up blocker settings, ask the user to perform the following steps:

a. Start Internet Explorer. 

b. On the Tools menu, click Internet Options. 

c. On the Privacy tab, click Settings. 

d. Under Address of website to allow, add the following sites to the list of websites to allow pop-ups:

• live.com 

• msn.com 

• hotmail.com 

• passport.com 

 

Windows Vista:

 

a. Start Internet Explorer. 

b. On the Tools menu, click Internet Options. 

c. On the Privacy tab, click Settings. 

d. Add the following sites to the list of websites to allow pop-ups:

• live.com 

• msn.com 

• hotmail.com 

• passport.com 

 

After adding the list of websites to allow pop-up, follow the steps below:

 

a. Click the Internet Explorer information bar, and then select one of the following options:

o Temporarily allow pop-ups: This option opens the blocked pop-up window. Additional pop-up windows will not open when you click hyperlinks. 

o Always allow pop-ups from this site: This option adds the hyperlink that you clicked to a list of safe websites, and then opens the blocked pop-up window and any additional pop-up windows on the current site. 

 

If you have software from a third-party that blocks pop-up windows, you may receive a warning message about a pop-up window, or nothing happens when you click the hyperlinks in a message. You need to contact the manufacturer or check the documents of the third-party software's list of safe websites.

 

If you receive the following message at the upper part of the email message, indicating the hyperlink and images in the email message have been disabled.

 

Ex.

“Pictures and links in this message have been blocked for your privacy. Show Content.”

 

You need to click "Show content" link to enable the hyperlinks or images on the email message.

The simplest way is the Underscore character.

You get it by typing an "uppercase hyphen", typically to the right of the 0 key on the number row at the top of keyboard.

suzanne s. barnhill say:

Several methods of creating lines have been mentioned. Others are described at http://wordmvp.com/FAQs/TblsFldsFms/LinesInForms.htm

My preferred method for creating such lines is an underlined tab character, with a tab stop set such that all the lines end at the same point (this is very difficult to achieve with underline characters in a proportional font).

To answer your initial question, however, spaces on their own are not underlined unless you:

  1. Enable the option to "Draw underline on trailing spaces (File | Options | Advanced: Layout options), or
  2. End the series of spaces with a nonbreaking space (Ctrl+Shift+Spacebar).
