jeeped say:

While I applaud your approach at preserving the data and attempting proper procedure when using removeable media, there would appear to be a couple of inherent problems.

First, in order to eject removeable media or prepare for safe removal, you need to know the drive letter. Obviously this can change from computer to computer depending upon hardware configuration. Another thumb drive, flash card, external HDD or even a software mounted ISO file could throw a wrench into the works. I suppose that a cyclic routine could be written to loop through the drive letters, but this will take time.

Second, the Removable Storage Manager Reference or RSM API has apparently changed significantly between XP/Vista and Windows 7, so built-in function calls may not work. You are only needing the most basic of calls, e.g VBA equivilent to EjectDiskFromSADrive, so there may be some transparency.

You need to create a query that joins your all three tables \. Join Sales and Purchase to Product on ProductID.
