No, I did not receive any error messages during or after the upgrade.

While problems can certainly occur, there are two things that will absolutely not happen when upgrading Office (Outlook) on the same computer

1) selective deletion of folders inside a PST file

2) selective removal of a PST file from the profile. Anything is possible in terms of Outlook not finding a PST file configured in the Outlook profile or having issues opening it etc BUT that would always result in some kind of error message when Outlook is opened using the profile.

The upgrade process on the same machine would have copied & used the Outlook profile(s) from OL'2007 so taking the above 2 items into account, the only question left to ask is (assuming you have only been using one Outlook profile in OL'2007)

Do you have have any email folders in your Navigation Pane with a + (plus) sign beside it that hasn't been expanded. Beyond that, only other suggestion would be to search your computer for all "*.pst" files and see if you find any outside of the one currently being used in Outlook. The way to determine which PST file is being used currently is by going to

File --> Account Settings --> Email tab --> highlight the POP account --> go down to the <Change Folder> button --> there you'll see the full path/file name of the PST file being used for the account

Clicking on the Data Files tab will list all data files configured in the profile. From your description - there should only be one.
