Affichage de l'erreur 80151011 lorsque vous tentez de vous connecter au Xbox LIVE

Problème

Le message et code d'erreur suivants s'affichent lorsque vous tentez de vous connecter au Xbox LIVE :

Impossible de télécharger les profils Xbox LIVE pour le moment. Réessayez ultérieurement.  

Code d'état 80151011

Ce message peut indiquer un problème avec votre profil.


Solution 1 : avez-vous récemment activé la vérification en deux étapes ?

Si vous avez récemment activé la vérification en deux étapes, vous risquez d'ignorer comment vous connecter à des applications et des appareils qui n'acceptent pas les codes de sécurité. Certaines applications (telles que les application de messagerie de certains smartphones) et certains appareils (tels que la console Xbox 360) ne peuvent pas vous inviter à entrer une vérification de sécurité lorsque vous tentez de vous connecter. Si une application ou un appareil indique que votre mot de passe est incorrect, vous devez créer un mot de passe d'application distinct pour vous connecter. Une fois connecté à l'aide de votre mot de passe d'application, vous pouvez commencer à utiliser l'application ou l'appareil. Vous vous servirez de ce nouveau mot de passe d'application à la place du mot de passe de votre compte Microsoft pour vous connecter à votre appareil.

Vous devrez créer et entrer un mot de passe d'application pour chaque application ou appareil qui ne prend pas en charge les demandes de vérification de sécurité.

1.Connectez-vous à votre compte Microsoft.

2.Sous Mot de passe et informations de sécurité, appuyez ou cliquez sur Modifier les informations de sécurité.

Si vous êtes invité à fournir une vérification de sécurité, entrez-la, puis appuyez ou cliquez sur Envoyer.

3.Sous Mots de passe d'application, appuyez ou cliquez sur Créer un mot de passe d'application.

Un nouveau mot de passe d'application est généré et s'affiche à l'écran.

4.Lorsque vous êtes invité à fournir le mot de passe de votre compte Microsoft sur la console Xbox, entrez le mot de passe d'application que vous avez créé en ligne.

Une fois connecté à l'aide de votre mot de passe d'application, vous pouvez commencer à utiliser l'application ou l'appareil. Vous vous servirez de ce nouveau mot de passe d'application au lieu du mot de passe de votre compte Microsoft pour vous connecter à votre appareil.

 

 

Solution 2 : vérifier les informations de sécurité de votre compte Microsoft

Effectuez les étapes suivantes :

1.Accédez à Compte Microsoft. Si vous avez oublié votre mot de passe ou s'il ne fonctionne pas, consultez Redéfinition de votre mot de passe.

2.Cliquez sur Modifier les informations de sécurité.

3.Vérifiez vos informations de sécurité. Par exemple, vérifiez les informations sous Numéro de téléphone, Autre adresse e-mail et Question de sécurité.

4.Déconnectez-vous de votre compte Microsoft, puis reconnectez-vous.

5.Réessayez de vous connecter au Xbox LIVE.

 

 

Solution 3 : vider le cache système

1.Appuyez sur la touche Xbox Guide de la manette.

2.Accédez à paramètres, puis sélectionnez Paramètres système.

3.Sélectionnez Stockage ou Mémoire.

4.Mettez en surbrillance un périphérique de stockage, puis appuyez sur la touche Y de votre manette. (Le choix du périphérique de stockage sélectionné n'a aucune importance ; le vidage du cache s'applique à chacun d'entre eux.)

5.Sélectionnez Vider le cache système.

6.Lorsque vous êtes invité à confirmer la maintenance du périphérique de stockage, sélectionnez Oui.

 

 

Solution 4 : déplacer et retélécharger votre profil Xbox LIVE

Remarque Si vous voulez utiliser une clé USB ou un disque dur USB pour effectuer les étapes suivantes, vous devez formater la clé ou le disque dur USB comme espace de stockage Xbox. (Consultez Support technique des clés USB pour la console Xbox 360.)

1.Déplacez votre profil sur une clé USB ou un disque dur USB. Pour cela, procédez comme suit : a.Raccordez la clé USB à la console.

b.Accédez à paramètres, puis sélectionnez Stockage.

c.Sélectionnez Tous les périphériques.

d.Sélectionnez Profils, puis votre profil de joueur.

e.Sélectionnez Déplacer.

f.Sélectionnez la carte mémoire sur laquelle vous voulez déplacer votre profil.

Remarque Une carte mémoire peut être une clé USB, un disque dur USB ou le disque dur Xbox 360. Veillez à déplacer le profil sur une clé USB ou un disque dur USB.

g.Retirez la carte mémoire de la console.

2.Retéléchargez votre profil. Pour plus d'informations, consultez Télécharger, déplacer ou supprimer votre profil Xbox LIVE.

 

 

Solution 5 : réinitialiser la console Xbox 360

Pour réinitialiser la console Xbox 360, procédez comme suit :

1.Appuyez sur la touche Xbox Guide, puis sélectionnez Arrêter la console.

2.Mettez la console Xbox 360 sous tension.

Remarque Si vous utilisez une manette sans fil Xbox 360, vous pouvez allumer la console en utilisant la touche Xbox Guide de la manette.

 

 

Solution 6 : supprimer et retélécharger les mises à jour système

Pour supprimer, puis retélécharger les mises à jour système, procédez comme suit.

Remarque Les étapes suivantes suppriment les mises à jour système installées sur la console Xbox.

1.Si nécessaire, suivez les étapes figurant dans la section Déplacer et retélécharger votre profil Xbox LIVE (solution 3, ci-dessus) pour déplacer votre profil Xbox LIVE.

2.Appuyez sur la touche Xbox Guide, puis sélectionnez Accueil Xbox.

3.Sélectionnez paramètres, puis Système.

4.Sélectionnez Stockage.

5.Mettez en surbrillance votre périphérique de stockage (mais n'appuyez pas sur la touche A pour le sélectionner).

6.Appuyez sur les touches suivantes : a.Gâchette haute gauche

b.Gâchette haute droite

c.Touche X

d.Gâchette haute gauche

e.Gâchette haute droite

f.Touche X

7.Dans l'écran Supprimer la mise à jour système, sélectionnez Oui.

8.Après le redémarrage de la console Xbox, sélectionnez OK pour télécharger et installer la mise à jour système.

9.Une fois la mise à jour système installée, retéléchargez votre profil Xbox LIVE et essayez de vous connecter au Xbox LIVE.

 

 

N'hésitez pas à revenir si le  problème persiste ou si aucune solutlion n'as pu vous aider ;)
