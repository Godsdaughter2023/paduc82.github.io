Hello Wayne,
Thank you for posting. As described, the ‘Show this folder as an e-mail Address Book’ option is grayed out in Outlook 2007.
 
The workaround is to create new profile in Outlook, as one of the causes could be of profile corruption.
You'll not lose any email just by creating a new profile.
So you may follow the steps mentioned below on how to create new profile and configure email account:
 
Step 1: Ensure to make a note of account settings of the email account configured in the existing Outlook profile.
 
Step 2: Create new profile in Outlook and choose to set as default when opening Outlook.
http://support.microsoft.com/kb/829918
 
Step 3: Once done configure an email account in Outlook has made a note of account setting.
 
Refer the article on how to ‘Add or remove an e-mail account’ in Outlook 2007:
http://office.microsoft.com/en-us/outlook-help/add-or-remove-an-e-mail-account-HA001231634.aspx
 
Step 4: Once setup email account, ensure to open your PST file which was set in old profile so that mails get deliver to this data file.
 
To open data file in Outlook 2007, refer the section ‘Open a .pst file’ in the article below:
http://office.microsoft.com/en-us/outlook-help/open-change-or-close-outlook-data-files-HA001232805.aspx
 
 
Step 5: To set new mails to deliver to data file:
Open Outlook > Tools > Account Settings > click on ‘Email’ tab > highlight the email address > Click on change folder at the bottom > then select the appropriate .pst file which you want your mails to be received.

ossiemac say:

First make a backup of your workbook with the backup code. (This is always advisable)

Insert a short cut on the Desktop to open the workbook as follows:

  • Navigate to the file in Windows Explorer
  • Right click the file
  • Select Send to -> Desktop (create shortcut)
  • Close or minimize Windows Explorer
  • Find the Icon on the desktop and click and drag to preferred location
  • You can right click the icon and rename it if you so desire.

  • Next open the workbook
  • Open the VBA editor
  • Double click ThisWorkbook in the Project Explorer (Left column)
  • Insert code as per the example below as the Workbook Open event to auto run when the workbook opens. (Note you cannot change the sub name or it does not work)
  • Save the workbook.
  • If you need to open the workbook without the code running then open Excel first and then hold the Shift key down while you open the workbook and it will suppress the workbook open event. (This will be required if you want to edit the workbook and/or code because otherwise it will auto close after running the backup code)

Private Sub Workbook_Open()
    'In place of the MsgBox insert your code to perform the required backup. You can if you wish use a call to the sub that you already run.
    MsgBox "Workbook Open event running"
   
    'Last line of the Workbook_Open sub to close the workbook again.
    ThisWorkbook.Close SaveChanges:=False
End Sub
