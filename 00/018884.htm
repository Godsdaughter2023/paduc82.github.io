maxwell joseph say:

Saving as “Filtered’ HTML will remove any Office specific tags and make changes to the file to help insure browser compatibility. Beyond stripping tags, saving as Filtered HTML also will reduce images to screen resolution. This is the same behavior as in Word 2007.

 

There are two options to create HTML documents with higher image resolution. Either you can save the file as a web page without filtering, or else replace the lower resolution images in the files folder for your filtered web with a copy of the higher resolution original. You would also need to rename the image to match the name created by Word when saving a filtered web.
