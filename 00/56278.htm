Hi StrawNic,

 

Thank you so much for posting your query on this forum. You have reached the right place and I will try my best to assist you with this issue.

 

This error usually happens when the path of the directory into which the installation files of the program is downloaded, contains a space. For example, in your case, you are downloading the setup file of the “League of Legends” game. Let us assume that you are downloading the file into a folder “Downloads”. Now, if this folder is in the “Temp” folder (which is again inside your user account folder) and your Windows username contains a space, then the path to the “Download” folder will contain a space. Thus, the Win32 Cabinet Self-Extractor tool cannot handle the DBCS characters in the user name and that results to this error.

 

To fix this issue, please follow these steps:

 

a) Press “Windows Logo” + “R” keys on the keyboard, type “sysdm.cpl” in the “Run” command window and then press “Enter” to open the “System Properties” on the computer.

b) On that window, click on the “Advanced” tab and then click on the “Environment variables” button.

c) Under the section “User variables”, click on “TEMP”, and then click on “Edit”.

d) In the “Variable value” box, type “C:\Temp”, and then click on “OK” twice.

e) Click on “Apply” and then “OK”.

f) Now, follow the same steps, from the step c) to e), for the “TMP” environment variable as well.

g) Also, please “Confirm” that “C:\Temp” folder exists, if not create one.

h) Now, restart the computer

i) Start the download and installation process of the “League of Legends” game again and check if that resolves the issue.

 

Hope this resolves the issue. Please get back to us with the result.
