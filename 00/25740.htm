Try this code:

Private Sub Worksheet_Change(ByVal Target As Range)
   If Intersect(Columns("A"), Target) Is Nothing Then Exit Sub
   
   Select Case UCase(Target)
      Case "": Target.Offset(0, 1).ClearContents: Target.Offset(0, 2).ClearContents
      Case "OPENED"
         With Target.Offset(0, 1)
            .NumberFormat = "dd mmm yyyy hh:mm:ss"
            .Value = Now
         End With
      Case "FIXED"
          With Target.Offset(0, 2)
            .NumberFormat = "dd mmm yyyy hh:mm:ss"
            .Value = Now
         End With
      Case Else
   End Select
End Sub

One thing that would need attention is that Office 2013 programs by default use different file extensions than the Office 2003 programs. Where Word 2003 saves *.doc files, Word 2013 saves *.docx files (or *.docm files if they must contain macros). Similarly, Excel 2003 saves *.xls files, but Excel 2013 saves *.xlsx (or *.xlsm) files.

If your macros contain Open and/or SaveAs commands, you'll have to modify them to handle the new file extensions, or else decide to keep everything in Office 97-2003 (Compatibility Mode) format.

On start, Edge is preloading your Edge Start and Edge New Tab pages. - You can stop this from happening by using Group Policy "Prevent Microsoft Edge from starting and loading the Start and New Tab page at Windows startup and each time Microsoft Edge is closed".

Here are the corresponding Registry settings:

- current user

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Policies\Microsoft\MicrosoftEdge\TabPreloader]
"PreventTabPreloading"=dword:00000001

- local machine

Windows Registry Editor Version 5.00

[HKEY_LOCAL_MACHINE\Software\Policies\Microsoft\MicrosoftEdge\TabPreloader]

"PreventTabPreloading"=dword:00000001
