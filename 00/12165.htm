You can only change you display name in Skype. Custom Skype names are no longer available. In Formula Builder mode, formulas are treated as array formulas. Hence, it shows correct result. Hence, you will need to enter it as array formula - Take F2 in cell to put the cell in Edit mode and then press CTRL+SHIFT+RETURN
