hansv mvp say:

Here you go:

Sub ColorWord()
    Const strWord = "above"
    Dim cel As Range
    Dim strAddress As String
    Dim lngPos As Long
    Application.ScreenUpdating = False
    With Cells
        Set cel = .Find(What:=strWord, LookIn:=xlValues, LookAt:=xlPart, MatchCase:=False)
        If Not cel Is Nothing Then
            strAddress = cel.Address
            Do
                lngPos = InStr(1, cel.Value, strWord, vbTextCompare)
                cel.Characters(lngPos, Len(strWord)).Font.Color = vbGreen
                Set cel = .FindNext(After:=cel)
            Loop Until cel.Address = strAddress
        End If
    End With
    Application.ScreenUpdating = True
End Sub

jeeped say:

You cannot change part of a result returned from a formula and the reason for not searching blanks is self-explanatory so you could look through the .SpecialCells(xlCellTypeConstants) in the .UsedRange. That should cut down the search area appreciably.

Sub All_Vegan()
    Dim x As Long, c As Range, s As String
    s = "above"
    For Each c In ActiveSheet.UsedRange.SpecialCells(xlCellTypeConstants, 23)
        x = InStr(1, c.Text, s, vbTextCompare)
        If CBool(x) Then c.Characters(x, Len(s)).Font.Color = vbGreen
    Next c
End Sub

shane devenshire say:

Replace A:A with $A:$A and see if that helps, but I think that is only a part of the problem.  Your formula starts from and includes A4 but in you message you say the pivot table occupies columns 10-16 which means J:P which would be an offset of 6 from J to P, not from A.  Next, are you really checking column A for the word Grand Total?  Its outside the pivot table range so if it is actually typed into column A its not moving as the pivot table resizes.  My guess is you are looking for the word Grand Total in the first column of the pivot table, column 10, column J?

As another considerations - grand totals can be turned off which would cause the formula to fail.  Here is the a modified version of the formula:

='View this tab'!$J$4:OFFSET('View this tab'!$J$4,MATCH("Grand Total",'View this tab'!$J:$J,0)-5,6)

I have also shown code at your other post but you would need to adjust it for J4 also.
