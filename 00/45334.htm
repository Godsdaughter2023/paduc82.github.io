Sure. You can create a desktop shortcut with the following target:

explorer shell:desktop

Or you can even implement it in the right-click menu. So when you right-click the desktop you get the "Explore here" option. Here is the REG file for it.

Windows Registry Editor Version 5.00

[HKEY_CLASSES_ROOT\DesktopBackground\Shell\&Explore]
@="Explore Here"

[HKEY_CLASSES_ROOT\DesktopBackground\Shell\&Explore\command]
@="explorer.exe shell:desktop"

Copy the above contents (the lines in bold letters) to Notepad, and save with .REG extension.

Double-click the file to run it.
