Microsoft KB130510

Superuser - Explorer command line switches?

For Windows Vista this should be enough:

The separate switch is required to start explorer.exe as a separate process. Otherwise the 64-bit version is started anyway, even though the 32-bit version explorer.exe is stored at syswow64.
