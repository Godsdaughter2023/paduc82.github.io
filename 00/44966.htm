devlin sco say:

Hello CsIP,

You can still export or import a user account's custom file association using the Registry Editor if the user account doesn't have administrative permission.

  1. Click the Start button, type regedit, and then hit Enter.
  2. Navigate to the path below.
    • HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FileExts
  3. Right-click the FileExts key and select Export.
  4. Select the save location and enter the desired name for the Registration File (.reg).
  5. Click Save to complete.

  • To import the file association, right-click the .reg file, select Merge, and then choose Yes to complete the process.

Important: This section, method, or task contains steps that tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly.
