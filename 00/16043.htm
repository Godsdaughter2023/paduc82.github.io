    This causes the 2 series problem.

    If you create a scatter chart on your data and then set that as default you should then be able to generate a new chart from a selected range using ALT+F1 approach.

    Now take that chart and customize the formatting as save as template. This template can then be used to format new charts. This approach will not allow for chart creation using a single action but  will keep it to a minimum.

    Adding text above the Y data also helps Excel determine the 2 areas of data should be treated as xy data rather than 2 sets of y data.

    However, VBA auto-shortens these to (16-bit?):

         MyArray = Array(&HFF&, &H80FF&, &HC0C0&, &HC000)

    That is not correct. VBA removes only the leading zeros, not the type sign "&" at the end.

    MyArray = Array(&HFF&, &H80FF&, &HC0C0&, &HC000&)

    Means it contains the values 255, 33023, 49344, 49152 which are unsigned Integers.

    MyArray = Array(&HFF, &H80FF, &HC0C0, &HC000)

    Means it contains the values 255, -32513, -16192, -16384 which are signed Integers.
