Volume I - I would like to page number with the Volume Number with the Roman Numeral I-(page number). I would like the I-1, I-2, etc. to follow in the Table of Contents as well.

suzanne s. barnhill say:

Does the volume number appear in a heading in the text? If so, you can use the "Include chapter number feature" (https://wordmvp.com/FAQs/Numbering/ChapterNumber.htm). If not, there are two approaches you can take:

  1. If the document contains more than one volume, and there is a heading (in one of Word's built-in heading styles) that contains the volume title, you can apply I, II, III numbering to that heading level and format it as Hidden, with no punctuation and "Nothing" for "Follow number with." Even though the numbering is Hidden, you will be able to use it as the "chapter number" in your page numbers.
  2. If the document is a single volume, select an unused heading level and apply I, II, III numbering to that level. Insert one paragraph in that style at the beginning of the document and format the entire paragraph as Hidden. Then use that level for the "chapter number."
