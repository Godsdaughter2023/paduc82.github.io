If you have created this answer sheet using character U25CB (or Wingdings 2 character 153), then what you really want to do is swap in character U25CF (or Wingdings 2 character 152). If the image is "Inline with text" and you set the paragraph that contains the image to Hidden, the image will also be rendered invisible.

You can't do that with floating (text-wrapped) images.

Why can't the valid product key be recognized on my computer?

don cuthbert say:

Windows 10 free upgrade does not activate with any product key.  It activates only by upgrade from an activated, qualifying Windows 10, 8 or 7 OS.  The Windows 7 product key will only activate Windows 7.  If upgrading to Windows 10, you must not enter any product key if asked for one.  Skip these screens and choose to keep Windows settings and files to ensure an upgrade is done and not a clean install.  Setup.exe for the Windows 10 install has to be run from within your activated Windows 7 or use Upgrade this PC now if running the Media Creation Tool.

http://windows.microsoft.com/en-us/windows-10/media-creation-tool-install

donginsberg say:

Thanks for your help

 Can't get the windows 10 upgrade icon on the computer. I have a second identical machine that I did the win 7 upgrade to and it successfully did the windows 10 upgrade. How do I get this first machine to upgrade to Windows 10

Thanks in advance for your help

Don Ginsberg 

don cuthbert say:

Don't bother reserving Windows 10.  It can be difficult to get the icon to show up for some and even then you have no way to know when your upgrade will finally be ready to do.

Just create the matching Edition of Windows 10 install media (Pro or Home) and use that to upgrade from 7 to 10, or choose "Upgrade this PC now" from the Media Creation Tool that I linked above.  The easiest way is to use the Windows USB/DVD Download Tool to create a USB with Windows 10 on it and run setup.exe from within your activated Windows 7.  A 4GB USB flash drive is big enough and reads faster than a DVD.  Choose to keep Windows settings and do not enter any product key when asked for one.
