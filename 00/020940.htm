Try 

Private Sub Document_New()
  Dim oFrm As UserForm1
  Set oFrm = New UserForm1
  oFrm.Show
  Unload oFrm
  Set oFrm = Nothing
lbl_Exit:
  Exit Sub

I created the following VBA for txtYear: Private Sub txtYear_Change() Worksheets("BB Data").Range("I2").Select Do Until ActiveCell.Value = "" If ActiveCell.Offset(0, -2).Value = txtYear.Value Then cboModel.AddItem ActiveCell.Value ActiveCell.Offset(1, 0).Select Loop End Sub  I cannot get cboModel to populate.     mike h.. say:

    Mike:

    Thank you!

    Now, if I want my year to be between 2005 and 2014, am I better off doing a combo box with those values or do I do an If Statement for those years? Which is cleaner? Any thoughts on how I do this?

    Chad

    Hi,

    Once again there are many ways you could do this. For example:-

    1. A start year textbox and another end year textbox and populate all the years in between. Note this would almost certainly require a button to do this.

    2. Simply enter a start year as you are now and populate with all years from that year to present. The changed code below does that.

    3. If the code below isn't what you want then please elaborate.

    Private Sub txtYear_Change()
    If Len(txtYear.Text) < 4 Then Exit Sub
    Dim Ws As Worksheet, LastRow As Long
    Dim TheYear As Long, x As Long
    'This is the worksheet with the data on
    Set Ws = Sheets("Sheet1")
    LastRow = Ws.Cells(Cells.Rows.Count, "I").End(xlUp).Row
        TheYear = CLng(txtYear.Text)
        For x = 2 To LastRow
            If Ws.Cells(x, "I") >= TheYear Then
            cboModel.AddItem Ws.Cells(x, "I").Offset(, 2)
            End If
        Next
    End Sub
