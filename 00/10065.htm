TOC Styles right indent - Microsoft Community

trish vrooman say:

TOC Styles right indent

How can you set the text of the TOC Style right indent without the page number changing indents also. Whenever I add a right-indent to paragraph of a TOC Style it also indents the page number.
stefan blom say:
By default, Word uses a right-aligned tab stop for the page number. You can add it to the right of the right indent as follows: Click in the relevant TOC style. Press Ctrl+Shift+S; in the Apply Styles pane, verify that the correct style name is displayed. Click Modify. In the dialog box, click Format, Tabs. Specify the position for the tab stop. Click Set. Click OK to close dialog boxes.

Note that tab stops to the right of a right indent won't work properly unless you install a hotfix released earlier this year. See http://support.microsoft.com/kb/2817342. (Warning: You should NOT install the September update, which causes trouble.)
