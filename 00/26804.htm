If you KNOW the first 10 Sequence values in each WAD_Route are 10, 20 ... 100, then the query is trivial:
SELECT SERVICE_LOCATION.WAD_Route, SERVICE_LOCATION.Sequence
FROM SERVICE_LOCATION
GROUP BY SERVICE_LOCATION.WAD_Route, SERVICE_LOCATION.Sequence

SELECT SERVICE_LOCATION.WAD_Route, SERVICE_LOCATION.Sequence
