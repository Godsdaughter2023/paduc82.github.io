andre da costa say:
Yes, it does create a 200 MB recovery partition:
This is for the Windows Recovery Environment (WinRE), which also shipped with Windows Vista only it was available from the Windows Vista installation disc only, although some OEMs made it available on a separate partition on their factory installations. In Windows 7, this feature is installed on all computers, and is stored in the 200MB partition if you install using a single partition scheme. However, if you install on a drive that has multiple partitions, the recovery environment is stored in a folder in the root of your installation drive, so if you install to a partition with a label of C:\ it will be located at C:\Recovery.

To access the System Recovery Environment in Windows 7, simply boot your PC, just before the system loads the Windows operating system; hit the [F8] Function 8 key on your keyboard which will launch the Advanced Boot Options menu. There you will see a new option 'Repair Your Computer', select this option and hit 'Enter' on your keyboard.

darian tab say:

Hi Zanie,

If pictures are inserted as attachments in an email message, you can easily save them with the Save Attachment feature. But when the  pictures are embedded in the message body, we recommend following this steps to save embedded pictures as JPEG / PNG / GIF / TIF / BMP format from an email message in Outlook:

  1. Preview the email message with the pictures that you will save in the Reading Pane.
  2. Right click the picture that you will save, and then click the Save as Picture in the right-clicking menu.
  3. In the popping up File Save dialog box, open a folder that you will save the picture in.
  4. Enter a name for the picture in the File name: box.
  5. Click the Save as type: box, and select one type that you will save the picture as.
  6. Click the Save button.

Note:

  • To save pictures as JPEG format, please select the JPEG File Interchange Format in the Save as type: box.

Let us know if you need any other assistance.

halima s say:

Hi aliloga,

• Purchase an SD card reader if your computer does not have a card slot.

• You may also use the data cable or Bluetooth capability if available with the phone to transfer the pictures etc. To know more about it refer the phone manual or contact the phone maker for more information.

• Plug your SD card into the SD/MMC/SM/MS/MS-PRO card slot on your computer if it has one. Most new computers include a built in card reader. Connect the card reader to the computer and insert the SD card into the card slot if your computer does not have a card slot. Your computer recognizes the new device.

• Click the Windows "Start" button and click “Computer." You should see your card listed as a removal disk (hard drive).

• Double-click the card to open its window. Locate the song you want to download from the location (folder) it's saved on your computer and right-click it. Click "Copy."

• Go back to the card window and right-click an empty spot. Click "Paste" to download the song to your SD card. Repeat this same process to download additional songs.

• Close the card window once you are done and click the "Safely remove hardware" icon on the system tray.

• Click your memory card on the safely remove hardware window and click "Stop." Disconnect your SD card from the computer when prompted to do so.


Hope this helps!
