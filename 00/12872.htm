jim gordon mvp say:

Phillip correctly pointed you to the Page Setup options.

In those options, you might try adjusting the setting for Format For to Any Printer instead of a particular brand and model. Experiment with the scaling. You may find the Fit to x pages wide by y pages tall setting useful.

Formatting is also affected by the fonts selected and the version of those fonts available on your Mac. Even common fonts like Times New Roman and Helvetica undergo version changes over time. If the version you are now using is different from the version of the font that was used when the workbook was made and the fonts selected, Excel substitutes fonts. Try changing the fonts to different fonts. You can use Font Book application or other font management tools to activate and deactivate versions of fonts on your Mac.

There is a fundamental change when moving from (.xls) and other older formats to (.xlsx) and other current file formats. When you open an old file format in a current version of Excel, it goes through a translator and doesn't always look the same as it did in old versions of Excel. I recommend you save old files in the appropriate new file format. These new formats work with Excel 2007 and later for both Mac and PC.

wunders say: You may have an Add-on or Extension that is causing this problem.  See if this problem exists if you start your browser without add-ons (I'm assuming that you're using Internet Explorer):

    Start -> Programs -> Accessories -> System Tools -> Internet Explorer (No Addons)

If you do this and your problem goes away, then you need to start IE normally and go to Tools -> Addons and disable selected extensions and plug-ins and restart IE until you find the one causing your problem.

See "Troubleshooting Addons" section of this KB article:
   < http://support.microsoft.com/kb/883256 >
