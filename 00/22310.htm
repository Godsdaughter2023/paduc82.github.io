Let's say however that you're in a bind and need to do some chicanery.
Here we go:
1. SkyDrive menu app -> Preferences -> About -> Unlink Skydrive
2. Once the SkyDrive folder has been unlinked, find it and rename it to something sensible (like "Old_SkyDrive")
3. SkyDrive menu app -> Quit SkyDrive
4. Find the SkyDrive.app (in the /Applications folder) and launch it.
5. Log-in to SkyDrive using the new account.
6. Go through the rigmarole of picking a folder location, etc.

That's it. The result of this evolution will be a functioning SkyDrive installation being synchronized to your new account and an "abandoned" set of old SkyDrive files that can be dispositioned as you please.

graham mayor say:
The short answer is no, but if you merely wanted to print such marks, you could cobble something together in vba that might work for you. e.g. as follows. The first macro adds a red coloured marker before each tab and formats it as hidden. You can then toggle the hidden view and print hidden text with the second macro. However note that some code when copied will have the white space created with spaces and this will not work for that.


Sub Macro1()
Dim oRng As Range
Set oRng = ActiveDocument.Range
With oRng.Find
    Do While .Execute(FindText:=Chr(9))
        oRng.Text = "|" & Chr(9)
        oRng.Characters(1).Font.ColorIndex = wdRed
        oRng.Characters(1).Font.Hidden = True
        oRng.Collapse wdCollapseEnd
    Loop
End With
ActiveWindow.View.ShowHiddenText = True
Options.PrintHiddenText = True
End Sub

Sub Macro2()
    With ActiveWindow.View
    .ShowHiddenText = Not .ShowHiddenText
    .ShowAll = False
    End With
With Options
    .PrintHiddenText = Not .PrintHiddenText
End With
End Sub
