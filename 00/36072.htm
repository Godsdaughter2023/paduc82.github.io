define default recording time in windows sound recorder

afzal taher say:

Hi,

The default recording time in Sound Recorder is 60 seconds. There is a workaround to increase the maximum recording time. Follow the steps mentioned below:

Step 1:
a. Start Sound recorder utility.
b. Click Record, and then let Sound Recorder run for 60 seconds.
c. After Sound Recorder stops recording, click Save As on the File menu, and name the file Blank.wav.
d. To increase the maximum recording time in Sound Recorder, click Insert File on the Edit menu, and then insert the Blank.wav file that you saved in step 3. When you do this,
the maximum recording time is increased by 60 seconds. You can repeat this step once for each additional minute of recording time that you want to add. For example, for a 5-minute
recording time, insert the Blank.wav file five times.

Step 2:
To make Sound Recorder use the extended length of recording time automatically:
a. Right-click Start, and then click Open.
b. Double-click the Programs folder.
c. Double-click the Accessories folder.
d. Double-click the Multimedia folder or the Entertainment folder.
e. Right -click Sound Recorder, and then click Properties.
f. Click the Shortcut tab.
g. In the Target box, press the END key, press the SPACEBAR, and then type the location of the Blank.wav file.

For example, if the target is "C:\Windows\sndrec32.exe", and the location of the Blank.wav file isC:\Blank.wav, the new line in the Target box is "C:\Windows\sndrec32.exe C:\Blank.wav"

The next time you start Sound Recorder, the Blank.wav file opens automatically. After you record your new .wav file, you can click Delete After Current Position on the Edit menu to
remove the extra blank space in the file.

NOTE: When you finish recording and editing your new .wav file, save the file with a new file name.

Hi Nikta SF

 

That 'Out of memory' error is referring to the Installed Memory (RAM) in the computer, and not the disk space left on the Hard Drive. Removing files will not help.

 

Memory is used each time you start a program. When you exit the program, the memory is returned to the computer.

 

One of the most common reason for low memory errors is the programs that are set to start when you first turn on the computer, since the memory used by these programs is never returned to the computer.

 

You can easily check all of the programs that automatically start when you turn on the computer.

 

Go to the Desktop, right click the Taskbar and select the Task Manager.

 

In the Task manager, select the Startup Tab. This will show a list of programs/apps.

 

If you see any programs that do not need to start automatically, right click these apps and select the Disable option. You can always start these apps when you need to use them by clicking the Tile on the Start Screen or All Apps screen.

 

You can check the amount of RAM that is being used by opening the Resource Monitor.
 
Press the WinKey+R to display the Run dialog.

Type  resmon  and click OK.
 
This will display the Resource Monitor.
 
The graphic in the Physical Memory section will show how your RAM is being used.
 
Hover the mouse cursor over any item to see a description.

 


Let us know if you have more questions.
