For UEFI based systems

If your computer is UEFI based, these are normally systems that come pre-loaded with Windows 8 or later, you will need to prepare the ISO file for such a configuration or you will receive an error message during setup. The thumbdrive needs to be formatted as FAT32 and use the GPT partitioning scheme. To do this, you need to use Rufus, a small tool you can download for free.

  • Select ISO Image
  • Point to the Windows 10 ISO file

  • Choose FAT32 NOT NTFS as the File system
  • Make sure your USB thumbdrive in the Device list box
  • Click Start
  • Close when complete

anoop k m say:

Hello,

 

Thanks for asking the Microsoft Community.

 

By default, the Works Word Processor doesn’t have an option to print double-sided. You may have used the printer specific properties to print as double-sided earlier.

 

When you upgrade the operating system, these printer settings may also change accordingly. In Works, if you cannot find the double-sided option when you go to File > Print > Properties, then you may update the printer driver and check if that helps. You may have to contact the printer manufacturer for further help regarding the driver update.
