      brian tillman [outlook mvp 2007-2019] say:

      The presence of the POP folder means that you are accessing your mailbox not by using a web browser like Internet Explorer, Chrome, Firefox, or Safari, but instead are using a program like Windows Live Mail, Apple Mail, or some app on a mobile device.  You are making a connection using a mail protocol named "POP3".  This protocol requests the mailbox to send all recent, new messages, then it sends a command to the mailbox to delete the messages that were sent.  Outlook.com doesn't obey that command.  Instead, it moves the messages from the Inbox folder to the POP folder.

      The POP folder may be a subfolder of the Deleted folder.  Examine the list of folders on the left side of the screen after connecting to http://mail.live.com/ and logging in with your Hotmail/Outlook.om/Microsoft address and password.  You should see the folders and Deleted may have a ">" to its left.  Click that and you should then see the POP folder.

      To stop using the POP folder, you must stop access the mailbox using a POP3 client.  Only you know what client you might be using.  You are the one who must have set it up, so only you can change it.

      jeeped say:

      A4= TRIM(A1) Returning the same as above, without removing additional spaces. ... Also tried =TRIM(CLEAN(A3))


      It would appear that you have non-breakingspaces (e.g. CHAR(160)) within that text string, probably from copying a web page's content. Non-breaking spaces are not removed by either TRIM() or CLEAN(). Try the following,

      =TRIM(CLEAN(SUBSTITUTE(A1,CHAR(160),CHAR(32))))

      ... which changes the non-breaking spaces to spaces before processing the rest of the text string.
