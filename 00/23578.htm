andreas killer say:
Sub MergeSheets()
  'Merge all data from all sheet into a new sheet (formulas are removed)
  Dim DestSheet As Worksheet
  Dim i As Integer
  Application.ScreenUpdating = False
  Set DestSheet = Worksheets.Add(Before:=Sheets(1))
  With DestSheet
    Worksheets(2).UsedRange.Copy
    .Cells(1, 1).PasteSpecial xlPasteValuesAndNumberFormats
    For i = 3 To Worksheets.Count
      Worksheets(i).UsedRange.Offset(1).Copy
      .Cells(.UsedRange.Count + 1, 1).PasteSpecial xlPasteValuesAndNumberFormats
    Next
    .Cells(1, 1).Select
  End With
  Application.CutCopyMode = False
  Application.ScreenUpdating = True
End Sub

Presentation.ExportAsFixedFormat Method

Syntax

Parameters

Name Required/Optional Data Type Description

Intent Optional PpFixedFormatIntent The purpose of the export.
