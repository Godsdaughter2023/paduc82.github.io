Find the settings for your network and do the following. NOTE - THESE INSTRUCTIONS ARE FOR O2 ONLY - MAKE SURE YOU GET THE RIGHT SETTINGS FROM THE LINK FOR YOUR NETWORK

Packet data access pt. : payandgo.o2.co.uk

Network Type: IPv4

Authentication type: Normal

User name: payandgo

Password: password

 

Save all that and come back to your settings menu.
