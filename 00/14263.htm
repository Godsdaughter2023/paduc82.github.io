What should the "limit CPU usage during scan" be set at? - Microsoft Community

rustyaf say:

What should the "limit CPU usage during scan" be set at?

What should the "limit CPU usage during scan" be set at?
kosh vorlon say:

Hi Rusty,

 

That depends on when you do them, what else is happening at the time, whether or not you are actively using the computer when the scan runs, your available resources, and how much you really care about how long the scan takes.

 

I have mine set at the default of 20% and run scans in the background while I'm otherwise working like normal.  It interferes very little with what I do and I don't really notice any difference in system performance.  But I do need to watch myself and not get carried away with 40-50 open IE windows at the same time along with all the other programs running or it will slow things down a lot even at 20% - but that's just learning to overcome what is actually a bad habit of not closing windows or tabs when I'm done with them.  In essence, this works just fine for me.

 

If you prefer to run scans at night when nothing else is running and you aren't using the computer yourself, you can set the throttle higher to 50-60% or even more since there will be nothing affected by it using that much of the CPU.  Then the scans will go quite faster.

 

Those are sort of the extremes.  If you want, you can experiment with it to see what is right for when you do scans and what else is occurring at the same time.  If you set it to run in the background while working and notice the system getting sluggish, then change the setting to a lower percentage (though that will not change the scan in progress but will apply to the next one done).  You can also tackle that problem by changing your startup programs to remove things you don't really need and closing programs you aren't using and that too may bring the system performance back in line too.  It's a matter of finding the right balance for your situation.  Start with what I suggested above and then adjust a bit each scan to see the effect until you find the spot that suits you.  THAT is then what it should be set at for you.

 

Remember to consider your power saver settings as only a scan running will be perceived as the computer being idle and will activate set power saver actions like sleep or hibernation or shutdown unless you change them to the power saver action is longer than the scan so the scan has time to finish (and give it some good grace period just in case or the scan will cancel and everything already done will be as if it never happened - it does not pause to be restarted when the computer is restarted or awakened - I personally leave it on never just to be sure).

 

I hope this helps.

 

Good luck!
