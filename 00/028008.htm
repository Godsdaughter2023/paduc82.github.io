This was asked before and I know of two ways of performing that trick: AutoText and OLE, but both approaches are not self-contained to the document being edited.  This is a problem:

- OLE is a problem requiring either a central document or it breaks when renaming the file

There are actually numerous fields that are repeated, so copying/pasting is tedious and introduces a big potential for human error. - Linked text as OLE (paste special -> link) work, even inside the same document, BUT this breaks down if you rename the document, so you are back to a central file, and a file management problem.
