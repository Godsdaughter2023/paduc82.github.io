joeu2004 say:

agrivated2 wrote:

I do invoices from a Microsoft template.  I have noticed that the sum is not always correct.  Sometimes it leaves cells out of sum or auto sum.

The most common reason for whole amounts being omitting is that omitted values are text, not numeric, despite appearances and formats.  If A1 is omitted from the sum, does =ISTEXT(A1) return TRUE?

If so, the remedy is to ensure that A1 is numeric.  The solution depends on details that you have not provided.  In particular, why is A1 text?

Some possible reasons:

1. The cell was formatted as Text when you entered the data.  Even if you change to a numeric format (e.g. Currency) later, that does not change the type of the data immediately.  The data must be "re-entered", for example by selecting each errant cell and pressing F2, then Enter.

2. The cell value was copy-and-pasted from another application.  Consequently, it might include spaces and non-breaking spaces.  The latter come from web pages, typically.  In some case, you can use the Text To Columns feature to convert the errant cells to numeric.   More generally, enter the following formula into a parallel column:

=TRIM(SUBSTITUTE(A1, CHAR(160), ""))

Then copy the parallel column and use paste-value to overwrite the original data.

If neither works, it would be prudent for you to upload an example Excel file (redacted of personal identifying data) that demonstrates the problem to a file-sharing website, then post the public/share URL in a response here.

hansv mvp say:

Create a query based on the table in design view.

Add Item, Tran Type and Tran Date.

Click Totals on the Design tab of the ribbon.

Leave the Total: option for Item and Tran Type set to Group By, and set that for Tran Date to Max.

 

If other fields are involved, you can create a query with the following SQL

 

SELECT *
FROM [MyTable]
WHERE [Tran Date] =
    (SELECT Max(T.[Tran Date]
     FROM [MyTable] AS T
     WHERE T.[Item] = [MyTable].[Item] AND T.[Tran Type]=[MyTable].[Tran Type])

 

where MyTable is the name of the table.
