I tried that and it also didn't work in that the text flipped back upright when it converted to PDF - the same as on the printer. 

I finally got it to print with the upside down text though. For the benefit of others searching here's what worked:

As noted I had text in text box I'd created and I wanted to print it upside down so I could make a fold-up booklet thing.

Initially I was flipping it using the rotate function under the page layout tab. This was flipping the text like I wanted but then when I printed it would always flip back.

To make the text flip and stay that way I had to click inside the text box to get the the Drawing Tools/ Format tab. Then I opened the shape Effects tab, then the 3D Rotation tab. Under that, the first option in the Perspective section made it flip 180 and when I printed, it printed 180 too.

Unfortunately, Safe Mode has been broken this way for at least the last three builds.  The pre-preemptive solution is keep a shortcut to MSCONFIG on your Desktop so you can run that to get back to normal mode.  Of course, that's of no use to you at this point.  In my case, I was lucky enough to be able to right-click on the taskbar, select Task Manager, go to File, select Run Task and then run (or search for) C:\Windows\System32\msconfig.exe.  Since you have access to the command prompt, I assume you could run the same thing from there.

EDIT:  Also, since you mentioned trying to run "sfc /scannow", I should have been more complete:  that's broken, too (and also for multiple builds).  You can get it going by using DISM and an ISO.  But, since the ISOs don't come out until a build hits the slow ring, you'll have to build it using ESD images.  The thread for getting sfc to work with DISM is here:

https://answers.microsoft.com/en-us/insider/forum/insider_wintp-insider_install/how-to-repair-your-system-files-using-dism-and-sfc/1021b42a-09ff-41a0-a95a-48f3ee3ae699

and the thread for building an ISO from the ESD images is here:

https://answers.microsoft.com/en-us/insider/forum/insider_wintp-insider_install/instructions-create-a-bootable-iso-file-using-esd/1d64c65c-f619-4193-9bce-d6ed5d3b66da
