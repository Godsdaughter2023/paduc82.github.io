doug robbins - mvp office apps & services (word) say:

Use the following:

Dim strFind, strReplace
strFind = ""  'Insert text to be found inside quotes
strReplace = ""  'Insert replacement text inside quotes or leave as is to delete found text.
Selection.HomeKey wdStory
Selection.Find.ClearFormatting
With Selection.Find
    Do While .Execute(FindText:=strFind, MatchWildcards:=False, Forward:=True, Wrap:=wdFindStop) = True
        With Selection
            If .Range.End < ActiveDocument.Bookmarks("Bibliography").Range.Start Or .Range.Start > ActiveDocument.Bookmarks("Bibliography").Range.End Then
                .Range.Text = strReplace
            Else
                .Collapse wdCollapseEnd
            End If
        End With
    Loop
End With

hansv mvp say:

The problem here is that the text in column is too long. VLOOKUP and MATCH don't work if the values to be searched are more than 255 characters long. The same goes for the SEARCH and FIND functions.

You could use this VBA function:

Function LongLookup(lookup_value As Variant, table_array As Range, column As Long)
    Dim cel As Range
    For Each cel In table_array.Columns(1).Cells
        If cel.Value Like lookup_value Then
            LongLookup = cel.Offset(0, column - 1).Value
            Exit For
        End If
    Next cel
End Function

Please use it with limited ranges only - if you specify entire columns, and if the value is not found, the function could take a LONG time.

Use like this:

=LongLookup("*"&J5&"*",Schedule!$A1:$C50,3)
