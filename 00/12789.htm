macropod say:

Ordinarily, a Table of Contents inserted via References|Table of Contents will automatically pick up headings formatted with Heading Styles 'Heading 1', 'Heading 2' and 'Heading 3'. Refreshing the Table of Contents (e.g. by clicking in it and pressing F9) should give you the option to update the contents. If you toggle Word's field code display 'on' (e.g. by pressing Alt-F9), the Table of Contents display will change to { TOC \o "1-3" \h \z \u }. Changing the 3 to 4 will expand the scope to include 'Heading 4'.

An Alternative method of creating a Table of Contents is to select the position where you want it to appear, press Ctrl-F9 to create a pair of field braces, then type TOC between them, thus: { TOC }, then press F9 to update. That will generate a Table of Contents that automatically picks up all 9 Heading Styles.
