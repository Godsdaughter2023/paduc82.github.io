buzzlight say: Hi folks,

I had a similar problem with Vista machine (64-bit home ultimate). I had successfully run automated backups for over a year. Suddenly, backups started failing with the following error code:

"The file or directory is corrupted and unreadable 0x80070570"

I thought it might be my external drive - so I tried another one - same error. After searching around various threads, I tried various fixes (similar to the ones above) but none of them worked.

I DID manage to find the problem however. The issue is not with the disk being written to - it is the disk being backed up that is the problem. If you're backing up your main drive (C: for example), try running an error-check on that drive (right click on your home drive, the one you're trying to back up, pick "properties," then go to "tools" - click "check now" under error checking - and check both boxes). Since this is the active drive, you will need to shutdown and reboot - the system will then automatically check the file system and also windows files during the next startup.

After doing this, my problem cleared up. Apparently the backup software was crashing when trying to read either a corrupted file or sector from the hard drive it was backing up.

fredtheweasel say:

In the mean time I figured out that this is. 

If you have a desktop background of photos that change (slideshow), Microsoft windows 10 dynamically alters the color scheme to match the color palette of the background. I guess it seemed a good idea at the time but it's very confusing.

They simplify the colors to simple primaries or secondary colors. For example, some of my photo backgrounds are desert scenes - pastel orange colors. When those display, MS changes the start buttons to primary red (ie what's the closest primary color to orange? Red! It tints the buttons at 100% saturation so there are no pastel effects at all). If the photo is dark or has a lot of sky I get blue buttons. I just changed from a dark photo to one with lots of orange life-jackets and the buttons changed from blue to bright red.

In other words it's not a bug but a deliberately designed but misguided feature.
