yves dhondt say:

The workbook name is only used if the selection remains active in the first workbook.

So in workbook one, copy the formula from the formula bar, then press ESC to prevent anything else from being taken into account. In workbook two, go to the sheet, select the cells where you want the result to be put, do a paste in the formula bar, and press CTRL+SHIFT+ENTER again.

If you really have that many files all with the exact same layout (same sheet names, same number of rows and columns, ...), you are probably better off writing a small macro doing all the transposing in all the files at once.
