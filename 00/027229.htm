Option Compare Database
Private Declare Function apiGetUserName Lib "advapi32.dll" Alias _
"GetUserNameA" (ByVal lpBuffer As String, nSize As Long) As Long
Private Declare Function apiGetComputerName Lib "kernel32" Alias _
"GetComputerNameA" (ByVal lpBuffer As String, nSize As Long) As Long

Function fOSUserName() As String
'Returns the network login name
    Dim lnglen As Long, lngX As Long
    Dim strUserName As String
    strUserName = String$(254, 0)
    lnglen = 255
    lngX = apiGetUserName(strUserName, lnglen)
    If lngX <> 0 Then
        fOSUserName = Left$(strUserName, lnglen - 1)
    Else
        fOSUserName = ""
    End If

End Function

Desktop shortcut Favourcons

don varnau say:

Since reloading  my windows 8.1, most of my desktop shortcut logos have been replaced by the IE favourcon

Hi,

Favicons are stored in the Temporary Internet Files folder. Reloading Windows cleared out this folder and deleted your Favicons.

Favicons will be re-downloaded on subsequent visits to the websites.

In the future you should be able to preserve Favicons by:

1. Unchecking "Delete browsing history on exit" at Internet Options> General tab> Browsing History...

and

2. Not clearing the History or Temporary Internet Files

or

3. On the Delete browsing history screen, check "Preserve Favorites website data"

and make sure that sites, whose Favicons you want to preserve are in Favorites.

Then when you clear History and/or Temporary Internet Files most (if not all) Favicons should be preserved.

Don
