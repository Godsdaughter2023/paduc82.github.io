On Mon, 15 Nov 2010 21:18:42 +0000, nifty nev wrote:

I have just set up on my new laptop windows 7,and when I try to print a doc. my canon 3 in 1 trys to fax it,even though all settings look correct

Faxing is treated as printing in Windows. Your fax device is seen as a
printer, and very likely your problem is that the fax "printer" is set
up as the default printer.

Either change the default from the fax "printer" to the real printer
or when you print, on the print window choose the real printer instead
of accepting the default.

yaqub k say:

Hi,

A Code 10 error is generated in Device Manager in one of the following situations:

  • Device Manager can't start the device.
  • One of the drivers that the device needs does not start.

Device Manager has insufficient information to recognize the error that is bubbled up by the device driver.
Note In this situation, the Code 10 error can be a very generic message that indicates some kind of unspecified driver or device problem.

What is the make and model of your computer?

I would suggest you to follow the steps mentioned in the following link:

Fix: "This device cannot start" Code 10 error in Device Manager in Windows

http://support.microsoft.com/kb/943104/en-us

Note: Make sure you run the fixit.

Also, update the latest drivers for Bluetooth from computer manufacturer’s website.
