Hello all,

I've suffered this problem intermittently for years, and now hypothesise that it is actually one type of problem that appears to be another type of problem.

What it appears to be is a problem with the email account / profile that necessitates regular re-authentication.

What it really is is a problem with ones external network connectivity and/or ones local wi-fi connectivity, such that the connection drops, and then when it reactivates Outlook does not realise it has already authenticated and asks the account to re-authenticate i.e. the pop-up box appears demanding "Enter your user name and password ...".

It can readily be proven that it is not a problem with the email account / profile -

In Accounts Settings, select the problem account, click on Change, then without changing anything click on Test Account Settings ...

This will send a Test Message to your Inbox.
(If the pop-up box appears during the test, repeat the exercise until you have sent a test message without re-authenticating, just to prove that your current settings do work with no change.)

The fact that the Test Message appears in your Inbox demonstrates that there is nothing wrong with the email account / profile, because the test could not work even once if there were.

So now how do you actually fix the problem?

In Accounts Settings, select the problem account, click on Change, then click on More Settings, click on Advanced tab, then on the Server Timeouts setting

My version of Outlook had default 1 minute which I increased to 5 minutes.

Since I made this change, the pop-up box which had been appearing very regularly has not reappeared.

I'd appreciate your help to test this hypothesis - please make a similar change yourself, and then report your results here in due course.

Good luck!
