Open the workbook of interest and try as follows:


Alt-F11 to open the VBA editor

Ctrl+R to access the Project Explorer windowPaste the following code:

'=========>>
Option Explicit

'--------->>
Private Sub Workbook_Open()
    Dim Nme As Name
    Dim iYear As Long

    Set Nme = ThisWorkbook.Names.Add(Name:="myYear", _
                                     RefersTo:=Year(Date))
    iYear = Year(Date)
    Date = DateSerial(iYear + 1, Month(Date), Day(Date))
End Sub

'--------->>
Private Sub Workbook_BeforeClose(Cancel As Boolean)
    Dim iYear As Long

    iYear = Evaluate(ThisWorkbook.Names("myYear").RefersTo)
    Date = DateSerial(iYear + 1, Month(Date), Day(Date))
End Sub

'--------->>
Private Sub Workbook_Activate()
    Dim iYear As Long

    iYear = Evaluate(ThisWorkbook.Names("myYear").RefersTo)
    Date = DateSerial(iYear + 1, Month(Date), Day(Date))
End Sub

'--------->>
Private Sub Workbook_Deactivate()
    Dim iYear As Long
    iYear = Evaluate(ThisWorkbook.Names("myYear").RefersTo)
    Date = DateSerial(iYear, Month(Date), Day(Date))
End Sub
'<<=========

Place the cursor anywhere inside the Workbook_Open procedure code and press the F5 function key. Save the workbook as a macro enabled xlsm file.
