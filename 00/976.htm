tom van stiphout say:

You are on the right track. You need to concatenate a sql statement to set as the RecordSource of the subform.

dim sql as string

sql = "SELECT * FROM Query WHERE Query.Firstname = '" & cboName.value  & "' OR Query.LastName = '" & cboName.value & "'"

me.mysubform.form.recordsource = sql

Note how the sql statement is concatenated with & from several fragments.

Also how string values are wrapped in single-quotes.
