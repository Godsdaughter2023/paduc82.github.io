Add your Outlook.com/Live.com/Hotmail.com/MSN.com email address(es) to either Outlook 2010 or Live Mail 2011 (Outlook is much preferred if you have access to it)

  • If you're copying from a PST file, open the PST file in Outlook 

There are lots of things you can do.  The most painstaking way and nearly 100% reliable way is:

  • If you're just copying/moving a few messages, copy your messages from folder to folder and click the Send/Receive button to force a sync
  • If you want to copy entire folders
    1. Create a new empty folder in the destination account,
    2. Select-all on the messages in the source folder and copy them to each destination folder
    3. Click Send/Receive
    4. Via the web client, login to your account and check that all of the messages have synch'd by checking the message count
    5. Repeat with the next folder

If you have a lot of folders to copy, the above method can take forever and it's tedious.  There are two other, faster, ways to get the work done.  First, login to the Outlook.com web client so you can see what progress has been made on the server (note you have to refresh to see changes):

  1. Try to copy whole folders, including all of the messages in that folder, into the destination account
    • This can hang if you do more than one folder at a time, so click send/receive after each folder copy and wait for the synch to complete
    • Do NOT do this if a folder has subfolders.  In this case, the failure rate is very high, use method 2
  2. By hand, create the empty folder structure in the destination account, then once complete, copy the messages folder-by-folder. 
    • It's best to login to the Outlook.com web client to create the folder structure
      • Either that or you have to hit Send/Receive after the creation of each new folder in the Outlook/Live Mail client, which is slow and tedious (creating 2 or more folders. then hitting send/receive,
    • You can queue up several copy operations this way.  Select-all to highlight all the messages in a folder, copy them to the destination folder, and repeat for successive folders, then click Send/Receive
      • Be careful here, getting greedy and queueing up too many copies will result in the synch from the Mail client to Outlook.com account to hang
      • Just do 3-4 folders worth of mail at a time is best.  I have had luck doing this for up to 8 folders, but it's unreliable. 

If the sync between the Outlook/Live Mail client and the Outlook.com server gets hung there are two ways to fix it:

  1. Look on the web client and find the folder that was in-progress when the copy hung (hint: look at the message count in the web folder and compare to message count in the mail client to know)
    • Delete that folder in the web client
    • Delete that folder in the mail client
    • Hit Send/Receive
    • Start over creating the empty folder and copying it's messages
    • Sometimes the mail client's local cache file gets corrupted and you cannot get the new folder to synch with the server, move to step two
  2. Close the Outlook/Live Mail client and delete the OST (or equivalent Live Mail file type) local cache file on your PC.  Open the mail client and let it rebuild the OST file by downloading folders and mail from the server.  Once that process is completed, resume your upload copying.

To find out which cells would be affected, check the Applies to column:

For example, =$K$32:$P$34,$K$30:$S$30 from the top rule are two sets of cells where the formula is applied. K32 is the starting point of the first selection, and this selection continues up to P34, which is the value after the colon. The dollar signs denote that these cell locations are anchored or absolute, so in case the formula is copied to a different place, the cell location will stay the same. Once you check these cell locations, you'll know specifically where each formula is being used. Another way to do it is by clicking the up arrow icons, circled in red above.

If you want to scroll through the document while checking the cells where the conditional formatting is applied, use the scroll bars on the bottom or the side of the screen or the scroll button of your mouse instead of the arrow keys or the navigation keys.
