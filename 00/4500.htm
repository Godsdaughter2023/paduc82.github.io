Unlink previous pages - Microsoft Community

mariomcdaniel say:

Unlink previous pages

I submitting a document that page numbering are to be in the upper hand corner except on pages where new chapters begin. On these, it will be centered atr the bottom.
jay freedman say:
The way Word usually handles this is to place a Next Page section break at the beginning of each chapter, and to turn on "Different first page" for headers and footers.

Each section has up to three headers and three footers:
  • The "main" header and footer are those that appear on every page if both "Different first page" and "Different odd and even" are turned off. They also appear on pages after the first page of the section if "Different first page" is turned on, and only on the odd-numbered pages if "Different odd and even" is turned on.
  • The "first page" header and footer appear only on the first page of each section if "Different first page" is turned on.
  • The "even page" header and footer appear on even-numbered pages if "Different odd and even" is turned on.


All three of these sets are independent, and they may have text content even if they aren't shown.


To set this up for your document, first make sure that there is a Next Page section break (and no manual page break) between the end of each chapter and the start of the next chapter.


Double-click in the header area (or right-click there and click Edit Header). This opens the Header & Footer Tools tab on the ribbon.

I assume you already have a page number at the right end of the header. If that isn't true yet, press the Tab key twice to get to the right end, and press Alt+Shift+P to insert a page number.

In the Options group on that ribbon, check the box for Different First Page. The blue tag on the header will say "First Page Header" and the header will be empty (which you want it to be).

Now click the Go To Footer button on the ribbon.

In the footer, press the Tab key once to go to the center of the footer, and press Alt+Shift+P to insert a page number there.

Finally, click the Close Header & Footer button on the ribbon, or double-click in the main text. Scroll through the document to verify that the page numbers are correctly positioned.

sitnsidewayz say: Yes, I have a Matrix Pro and I have done this. There are two ways to do it. The easy way and the technical way. With the easy way, it will add the sound files to the alerts lists, but will also add them to the ringtone list. The technical way will only add them to the alerts lists, but requires using a registry editor for editing the registry. 

This is the easy way: Take the file, can be either mp3, wma, or wav, and put it in this Windows folder, Start>My Stuff>Up>Application Data>Sounds. Since it is just an alert sound, it's best to keep the file small and short.

The techinical way: Place the file in the Windows directory, Start>My Stuff>Up>Windows. Then using the registry editor, either on board or thru a PC, go to HKCU>ControlPanel>Sounds>AllSounds and edit the value to show the file name in the list, including the file extension. It must be the exact file name or it won't show up. You can also remove any existing ones you don't want when editing the value.

These changes that Word makes on the fly do not affect its use of the datasource with mailmerge itself.

While some reformatting might be required, converting footnotes to endnotes at the end of each section is a matter of opening the Footnote and Endnote dialog (using the dialog launcher in the Footnotes group on the References tab, clicking Convert... | Convert all footnotes to endnotes | OK and then setting the position to "End of section."

Word does its dead-level best to keep footnotes (or at least the beginning of a footnote) on the same page as the footnote reference. Anything else is counter to publishing convention, and a bug that resulted in footnotes on a different page is addressed in the article at http://word.mvps.org/FAQs/Formatting/FootnoteOnDiffPage.htm. If you don't want the note itself split, then you can format the Footnote Text style as "Keep lines together." If you don't want the paragraph containing the reference split, you can format it as KLT. But prepare yourself for some very short pages; see http://wordfaqs.mvps.org/BottomLine.htm.
