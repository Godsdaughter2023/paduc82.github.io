don cuthbert say:

Boot with Windows 10 install media.  Use SHIFT-F10 from the first install screen to get a Command Prompt.

Run Diskpart and in diskpart list volume to see what drive letters are currently given to your Windows 10 and Windows 7 partitions.  Type exit to end diskpart.  In Diskpart, if your 2 Windows partitions do not both have a drive letter, use select volume # to select the volume without a letter and then enter assign to give it a letter.  Use whatever volume number in place of the #.  It doesn't matter what letters they have at this time, as long as you know which one is which.

If Windows 10 is D: and Windows 7 is E:, try using these commands:

bcdboot E:\Windows

bcdboot D:\Windows

Use whatever letters are for your Windows partitions instead of E: and D:.  Use the Windows 7 letter first, then the Windows 10 one so that Windows 10 becomes the default.
 

If you now type bcdedit, you should see a couple of entries from your BCD.  The default should point to Windows 10 and another one should point to Windows 7.  Now try restarting the computer and you should have a dual-boot menu, with Windows 10 as first choice.  If you choose to boot to Windows 7 it should show a text-style boot menu.

This will change the Lockscreen timeout settings

Option 1:
Right click on Start > Select PowerShell (Admin)
copy/paste the following command into the window then press Enter.

powercfg -change -monitor-timeout-ac 20

20 indicates the screen will timeout after 20 minutes. Set the number for what you wish it to do.

Open Control Panel > Power Options > Choose when to turn off the display
You will see beneath the Plugged in option your setting has changed to the time you commanded Power Cofiguration to Timeout. The screen should darken and stay darkened after that period.

Option 2:
This entails editing Windows Registry. Ensure you create a backup of Windows Registry before making the following changes.
https://support.microsoft.com/en-us/help/322756...

Press Winkey + R to open Run box
Type Regedit > Press Ok or Enter
Navigate to the following key

HKEYLOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\7516b95f-f776-4464-8c53-06167f40cc99\8EC4B3A5-6868-48c2-BE75-4F3044BE88A7
On the right side, double-click the Attributes DWORD.
Change the value from 1 to 2.
Click OK.

Open Control Panel > Power Options
Select "Change Plan Settings"
Click "Change Advanced Power Settings"
In Advanced Power Settings > Expand Display by clicking the + next to it.
Change the default time to your choosing.
Click Apply, Ok
Reboot the computer.
