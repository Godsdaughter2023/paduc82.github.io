Hi, merityoussef. The error message you are getting denotes failure while initializing the Download Manager. The resolution for this is to reset the phone to clear and recreate its cache. If you're using a Windows Phone 8 or 8.1, go to Settings > about > Reset your phone. As this will wipe all your data, we recommend creating a backup file to avoid any data loss: http://www.windowsphone.com/en-us/how-to/wp8/settings-and-personalization/back-up-my-stuff. Hope this helps.
