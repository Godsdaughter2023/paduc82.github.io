Sorry, Jack,

But your question is a bit vague. In Access terms what you are calling a "drop down" list is called a combobox. A combobox is a form control that allows you to select for a list of items. There is a combobox wizard in the Control Toolbox (on the Form design ribbon) that can walk you through creating a combobox.

Your use of the word parameter suggest that you are trying to provide criteria to a query. Access has the ability to prompt the user to enter a parameter when the query runs. This is called a "parameter prompt". However, most Access developers don't use that since it is very limiting. Instead they will create a form, use a combobox control on the form to select a value, then reference that selected value using the syntax:

=Forms!formname!controlname

Without more specifics we can't give you step by step. I'm sure if you search on the phrase creating a combobox in Google or Bing, you will find many tutorials to lead you through the process if what I have said is not enough to get you going.

Hi Padda

Here a solution to your question.

To achieve that we need to use a macro.

** The macro will colorindex all sequences in a row.

***Also will type the sequences in the nearby columns

Please check pictures below

Before running the macro. You must select first cell of the sequence line(row). In this case A1

and the code automatically will evaluate each row

After running the macro

This is the macro

*******************************************************************************

Sub ColorSequence()


Dim SeqRng As Range

Dim LRow As Integer

Dim LCol As Integer

Dim Q As Integer

Dim QNext As Integer

Dim Cnt As Integer


If ActiveCell.Value = "" Then

   Exit Sub

End If


StRow = ActiveCell.Row

StCol = ActiveCell.Column

LRow = ActiveCell.End(xlDown).Row


For x = StRow To LRow


LCol = Cells(x, StCol).End(xlToRight).Column


    Set SeqRng = Range(Cells(x, StCol), Cells(x, LCol))

                     Cnt = SeqRng.Cells.Count

  

   Z = 43

   Q = Application.WorksheetFunction.Large(SeqRng, Cnt)

   MySeq = Q

 Do Until Cnt = 1

     

  Q = Application.WorksheetFunction.Large(SeqRng, Cnt)

  QNext = Application.WorksheetFunction.Large(SeqRng, Cnt - 1)

   

      If QNext = Q Then

          GoTo Jp

      End If

   

      If QNext = Q + 1 Then

         SeqRng.Find(What:=Q, LookAt:=xlWhole).Interior.ColorIndex = Z

         SeqRng.Find(What:=QNext, LookAt:=xlWhole).Interior.ColorIndex = Z

         MySeq = MySeq & "," & QNext

      End If



             If QNext > Q + 1 Or QNext = Application.WorksheetFunction.Large(SeqRng, 1) Then

                    Select Case Len(MySeq)

                           Case Is <= 2

                           MySeq = QNext

                           Case Is > 2

                           Cells(x, StCol).End(xlToRight).Offset(0, 1).Value = "Sequence (" & MySeq & ")"

                           MySeq = QNext

                           Z = Z + 1

                     End Select

               End If


      

Jp:

    

      Cnt = Cnt - 1

      

     Loop

    

Next x


End Sub
