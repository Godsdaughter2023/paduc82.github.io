jim gordon mvp say:
Silverlight is a small web browser plug-in that is just a single file. All you have to do is drag it to the trash, and then empty the trash.  Here are the instructions:

http://www.microsoft.com/getsilverlight/get-started/install/removing-silverlight-mac.aspx

Silverlight is included with Microsoft Office to make using SkyDrive faster and easier, but SkyDrive will work without Silverlight using standard web browser interface elements, too, so you won't lose this particular functionality by trashing Silverlight.

Netflix uses Silverlight to stream movies to your Mac. If you are streaming movies with Netflix, don't trash Silverlight. Some other web pages may not work at all or as well without Silverlight.
