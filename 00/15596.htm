Set English (Australia) as default language for spell check - Microsoft Community

michael mackie say:

Set English (Australia) as default language for spell check etc.

I have followed prompts to set English (Australia) as default language in Word docs but, whatever I try, computer always reverts to English (United States). How do I fix it permenantly?
doug robbins - mvp office apps & services (word) say:

Go to the Windows Control Panel and click on "Change keyboards or other input methods" in the "Clock, Language and Region" section and then on the "Keyboards and Languages" tab, click on "Change keyboards" and then on the "General" tab of the "Text Services and Input Languages" dialog, select "English (Australia) - US" as the Default input language.

In Word, you may then need to go to the "Review" tab of the ribbon and click on the "Set Language" in the "Proofing" section of the ribbon and then select "English (Australia)" and click on the "Default..." button and then on "Yes" when asked: "Do you want to change the default language to English (Australia)?"
-- Hope this helps.
