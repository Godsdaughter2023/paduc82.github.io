bob jones aka: cybertaz say:
Printer Presets are a feature of the print subsystem, they are not "in" Word.

To delete a custom Preset, in any program go to File> Print, select the Preset from the dropdown list then open the list again & select Delete.

To modify a custom preset, select it from the list, change whatever settings you want to modify then save again with the same name to overwrite the previous settings.
