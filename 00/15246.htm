I have duplicte personal folders in outlook 2003. How do I remove - Microsoft Community

moncarm say:

I have duplicte personal folders in outlook 2003. How do I remove one?

I recently got a new comuter and transfered files along with Microsoft Office 20003. I can not retrieve old outlook file and now have duplicate personal files along with related info.

 

russ valentine say:

It is very common to get a duplicate or "ghost" Personal Folders file if you do not migrate your previous Outlook Data File correctly to Outlook 2003. Importing a PST file is a common cause of this. Once you do, your Outlook profile will be corrupt and you must start over with a new profile.

First you create a new profile:
http://support.microsoft.com/default.aspx?scid=kb;en-us;829918&Product=out2003
Then you configure this profile to use your previous PST file as its default instead of using the new one it creates:
http://office.microsoft.com/en-us/assistance/HA010771141033.aspx
Just make sure you place your previous PST file somewhere _other_ than the default location so you don't run the risk of overwriting the PST file that Outlook creates.

john mcghie say:
Hi Patrick:

Word 2011 is a level behind Word 2007.  It does not have the new Equation Editor built in to Word 2007.  So it cannot read the fractions created by Word 2007/10.

You "could" convert the equations to "pictures" in Word 2007.  Cut the equation, then Past Special>As Picture.  It will be equally visible in both versions of Word.

Sadly, it will no longer be editable in either.

If your work requires a lot of equations, you would be better advised to run a virtual machine in something such as Parallels Desktop, and load a copy of Windows 7 and a copy of Word 2010 in that.

I realise this is an expensive solution, but it's the one I use, because it guarantees there will be no little "issues" between the two platforms.

Word 2011 is exactly "compatible" with Word on the PC, but it does not have many of the advanced functions, such as the new equation editor.  There will be a new version of Office for Mac, probably next year.  We could guess that the new version of Office for Mac will have some of the functions that were left out of Office 2011, although likely not all of them.

Hope this helps
