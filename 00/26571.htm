graham mayor say:
You can do a surprising amount with vba, and you can run vba from a script in a rule, so the following script should work

Sub OptReply(olItem As MailItem)
Dim olOutMail As MailItem
Dim TestDate As Date
Const strMessage As String = "I regret I am out of my office today. I will respond on my return."
    Set olItem = ActiveExplorer.Selection.Item(1)
    TestDate = CDate(olItem.ReceivedTime)
    Select Case Format(TestDate, "dddd")
        Case "Tuesday", "Thursday", "Friday"
            With olOutMail
                .To = olItem.SenderEmailAddress
                .Subject = "RE: " & olItem.Subject
                .Body = strMessage
                .Display        'Change to .Send after testing
            End With
        Case Else
    End Select
End Sub

You will of course have to keep the PC switched on with Outlook running and set to automatically send and receive. Run the rule on all messages.
