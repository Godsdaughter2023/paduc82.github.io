Hey there poison ampoule and TwistedPixkle18,


If you accidentally unmapped the "A" button on the controller the way to fix it is to do the following. You will need a second controller to do this, it's not possible to correct the issue with the only controller.


1. Connect the controller with the problem to the console.

2. Connect a 2nd controller to the console.

3. Using the 2nd controller navigate to Settings>Kinect & devices > Devices & accessories.


You should see both controllers listed in the accessories screen here.


4. Using the 2nd controller Highlight and select "Configure" on the problem controller.

5. Adjust the button mapping so that the buttons will be configured as you prefer.


And that should do it, you can then disconnect the 2nd controller and navigate as normal.
