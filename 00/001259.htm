julie fka profjulie say:

I would be interested in reviewing the documentation you reference in your statement that "After what I have heard all 3G chips contain a GPS module. But the manufacturer does in many cases not enable the GPS. So a GPS program finds the GPS but the GPS doesn't find any satelites."  Please share some links.

 

Whether or not a device has GPS or not has nohing to do with the SIM card or the type of SIM card that is used.  The GPS radio is a piece of hardware.  If the mobile device includes an installed GPS radio, then you should be able to use it, even if no SIM card is installed in the device.

 

3G refers to the type of data connection that could be achieved through the cellphone network to which you subscribe...if your cell provider does not provide a 3G network then the best data connection you might be able to find on a 3G enabled phone is Edge.

 

You can confirm that your mobile device includes a GPS radio by tapping Start > Settings > System > Asset Viewer (assuming the device offers that option).  Then look for the GPS tab.  Tapping the tab will probably show that GPS is present but disabled.

 

There are some required conditions that must be satisfied before you can use the GPS on your device. 

 

Firstly, you should confirm that the correct COM port is assigned to the GPS radio.  The documentation that came with your device should tell you what COM port is used by the GPS radio.  (I don't know what the correct COM port is for the Omnia.)

 

Once you know the correct COM port that GPS uses, you should review the GPS settings on the device to confirm they reference the correct COM number:  tap Start > Settings > System tab > External GPS icon  > Hardware tab > change the GPS hardware port setting, if necessary, to match the port that is used by the GPS radio in your device. (NOTE: if you can not find a GPS icon on the System page, then this option is hidden.  You can unhide this option by making a simple registry edit, described in the article linked below)

 

The GPS radio is enabled when you use GPS software on the device.  Whether you install GPS software to the device or use something like Google Maps or Windows Live, you will probably also need to open an option within the software used and reference the GPS port that your device uses.

 

The very first time you activate the GPS radio (by opening a properly configured GPS software on the device), it may take several minutes to fix on the satellites.  It is best to activate the radio outside under a clear view of the sky - just take the device outside and leave it alone for several minutes.
