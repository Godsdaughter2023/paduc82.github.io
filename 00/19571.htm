Hi Tonya,

In the example given, the OP wanted:

Phone: 08 1111 1111
Fax: 08 2222 2222
Email: *** Email address is removed for privacy ***

to be output when:

Roundhouse Nurseries

was selected.

Content control dropdowns accept inputs for both Display Name and Value. To achieve this, for the example given, the:

• Display Name is input as - Roundhouse Nurseries

• Value is input as - Phone: 08 1111 1111|Fax: 08 2222 2222|Email: *** Email address is removed for privacy ***

Note how the value input uses | characters as a delimiter. That's because the Value doesn't accept paragraph breaks or line breaks.

The content control dropdown was given the title 'Client' and the text content control where the output is to be displayed was given the title 'ClientDetails'. The ContentControlOnExit macro simply reads the Value that corresponds with whatever the Display Name is, then replaces the | characters with manual line breaks and outputs the result in the 'ClientDetails' content control. In the sample macro, the 'ClientDetails' content control is simply referred to by:

ActiveDocument.ContentControls(2)

If you can't rely on it being the second content control in the document, it would be better to refer to it via:

ActiveDocument.SelectContentControlsByTitle("ClientDetails")(1)
