mikey butcher say:

Windows 10 'Input Out of Range Error' solved on my pc.

I had this issue for a few days and did not find anyway of a resolution. I finally started to troubleshoot myself, since nothing online would help. I have a three monitor setup. 

Steps:

1. Right click on desktop background. 

2. Click on display settings.

3. Click on Advanced display settings. 

My desktop resolution and active signal resolution were different. The active signal resolution was larger than the active signal resolution.

4. Click on Display adapter properties for Display 1, 2, etc.

5. On Adapter Tab click on List All Mode.

My input out range mode said to change to 1680 x 1050. 

6.  Click on a range lower than what the the monitor resolution is.

7. Click OK and then click apply.

8. Click apply changes.

9. On Adapter Tab click on List All Mode.

10. Select the resolution the monitor is capable of handling. 

11. Click OK and then click apply.

12. Click apply changes.

Now the input out of range should be removed from the screen. It worked for me and hopefully it works for you.

ronnie vernon say:

Hi

 

The C:\Documents and Settings folder is what's called a Junction, it is not a real folder. This folder was used in previous versions (XP) to hold all of the users files. This location was changed in Windows 7 to  C:\Users.

 

A Junction is used to redirect a program to a different location for backward compatibility purposes. When a program that was developed for Windows XP  is installed in Windows 7, it may be hard coded to look for the Documents and Settings folder. When it tries to access the Documents and Settings Junction, it will be automatically re-directed to the C:\Users folder.

 

This Junction allows the older program to install and work. This backward compatibility is the only reason that the Documents and Settings Junction is present on the system. There are many other Junction points in Windows 7 and these can be recognized by the Shortcut Arrow or Padlock overlay on the Junction icon.

 

All of these Junctions have the permissions set to Everyone = Access Denied. This is done to protect them from accidental deletion or permission changes. You should not change any of these properties, because doing so can cause some programs to quit working or prevent some programs from installing.

 

There are many other Junctions in Windows 7 and these can be recognized by the Shortcut Arrow that is present on the folder icon. See my post in the following thread for the instructions on how to create a text file that shows all of the Junctions on Windows 7 and the target folder that they point to. This will show you how to easily recognize these Junctions.
