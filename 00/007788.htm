vincentchoy say:

Its really up to you. Both provide different layers of protection.

EOP is your Microsoft's version of your typical Mail Gateway Anti-Virus.

ATP is Microsoft's version of what is commonly know in the the Anti-Virus world as Anti-Threat Protection that deal with more sophisticated attacks. Ideally you should have both if budget allows.

john korchok say:

The most common cause of this is that you copied and pasted placeholders when you were designing the custom layouts. When you do that, all the placeholders have the same id number and PowerPoint can't tell which placeholder uses which content. I cover this issue in this article, please scroll down to the Custom Slide Layout section: PowerPoint Construction - Brandwares Best Practices

However, even if you don't copy and paste, and the layouts have different numbers of placeholders and you switch back and forth, you'll still confuse PowerPoint and see this effect.

john sr wilson say:
Text can jump between placeholders even if they have the correct idx numbers and the same number of placeholders. You can see this easily if you open a comparison layout in a standard Office Template. Write in ONLY the right hand placeholder and then just re-apply the same layout. The text will jump.
