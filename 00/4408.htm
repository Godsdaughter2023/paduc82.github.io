Simulating a press of a button on a ribbon may be the best you can do in Outlook.

Option Explicit

Private Sub ChooseForm_ExecuteMso()
   
    Dim objExpl As Explorer
  
    Set objExpl = ActiveExplorer

    If Not objExpl Is Nothing Then                            
        objExpl.CommandBars.ExecuteMso ("ChooseForm")
    End If

    Set objExpl = Nothing

End Sub

Hovering over the icon when choosing a button for a ribbon gives you the text such as "ChooseForm".

It's all just a mess! The best solution is to completely stay away from computers that use display scaling.

maxhist say:
If you're using the BetterTouchTool, here's an alternative method for pinch-to-zoom (example for Word 2011 Mac):

1. In Word 2011, go to Tools ---> Customize Key.board... 
2. Select the 'View' category on the left side
3. In the 'commands' list, look for ViewZoomIn and select it.
4. Define a keyboard shortcut. I use the combination of Command, Option, +.
6. Click on 'Assign'
7. Repeat the same operation for 'ViewZoomOut.' I use the combination of Command, Option, -.

In the BetterTouchTool, you can assign "pinch in" and "pinch out" to the same keyboard shortcuts that you just created in Word.

There you have it! Pinch to zoom working in Word 2011. Unfortunately, you'll have to repeat this for Excel, PowerPoint, Outlook, etc.
