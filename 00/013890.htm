chip pearson say:

Use the INDIRECT function:

=SUM(INDIRECT("D5:D1000"))

The parameter to INDIRECT is a quoted text string, not an actual range reference, so Excel won't change it as you insert/delete rows/columns. It will always point to D5:D1000.
