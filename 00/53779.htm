    Thank you for posting your query in Microsoft Community forums.

     

    I understand that you facing issue in fast user switching in Windows 8.

     

    I would suggest you to disable and then enable Fast user accounts and check if the issue persists.

     

    Also follow the steps given below to disable and enable Fast user accounts. 

     

    a)    The Start Screen in Windows 8 displays your name along with the account picture at the top right corner. Clicking it opens up a menu containing options to Change account picture, Lock and Sign out from the current account, or access any other account existing on the computer just by clicking its name. Once you select another user account, you will be automatically logged off from the current one and taken to the login page of the selected account.

     

    b)    To begin, make sure that you have logged in as Administrator or have required administrative rights to change local group policies. First, go to Start Screen, type gpedit.msc, select Apps from right sidebar, and then click gpedit.msc in main window. This will open Local Group Policy Editor. Now, go to the following location and look for Logon in the left pane.

     

    c)    Local Computer Policy/Computer Configuration/Administrative Templates/System/Logon

     

    d)    Just double-click Hide Entry Points for Fast User Switching to bring up a dialog box to change Fast User Switching policy setting.

     

    e)    At the top, there are Not Configured, Enabled and Disabled options available. Selecting each setting will let you read its affect in Help section. To disable Hide Entry Points for Fast User Switching, just select Enabled from the list and click OK.

     

    f)     Once done, you will have to enforce the change made to Fast User Switching policy setting. To do so, close the Local Group Policy Editor and open the Run dialog box. Enter the command “gpupdate /force“ and click OK. The policy will be updated and applied on all user accounts.

     

    g)    Fast User Switching will now be disabled in Windows 8. You will notice that User Tile menu (accessible from Start Screen) doesn’t show other user accounts.

     

    h)    To re-enable Fast User Switching, just choose Not Configured in its policy setting dialog and apply the changes via gpupdate /force command.


     
    I would also suggest you to stop and restart Fast User Switching Compatibility in services
