bossdweeb say:

"so I will get the fastest 4gb ddr3 ram I can get"

Get the same speed as what you already have. There are several reasons for this, but

first, is your motherboard's memory controller (or possibly the cpu's. I'm not sure if your cpu

is new enough to have the memory controller built in) cannot handle any speed RAM you whack into it.

If you try to force a speed more than one level higher than what the motherboard BIOS defaults to

(maybe not even that) you will very likely get BSOD's ( higher speeds can require over-clocking the system, and you really don't want to go there with this machine).

Second you will not see any real world difference, even if the machine can use a higher speed.

"trying to run windows and the game both at the same time"

Yes, an unavoidable scenario, but do avoid multi-tasking (close browsers, etc.), and having too much stuff running in the background - eg. Disable unnecessary Starup programs via -

System Configuration, ( msconfig > Startup).

"and really the game runs fine at 50 fps...and the game spikes like 10 fps down every 10 sec so"

A 10 fps drop should not be serious. Doping below 25fps could cause serious lag though.
I can't offer suggestions as to what RAM your system can run on stably. Compaq support may help.

Or go with a new 4GB kit of the same brand, speed, latency, and voltage as what you already have.
