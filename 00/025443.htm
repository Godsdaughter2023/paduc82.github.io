Unfortunately, the answer is: "you don't/can't". Win10 Mail/People/Calendar are completely separate and distinct from Outlook and the two <email clients> do not sync with each other.

The Play Narrations and Use Timings features cannot be permanently disabled in PowerPoint to be only enabled when those features need to be used. By default these features are always enabled/selected in PowerPoint.
