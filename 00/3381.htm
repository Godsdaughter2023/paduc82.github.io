If you have hidden and system files displayed in Windows, it is normal for a number of temporary files to "disappear" when you close your document (see https://support.microsoft.com/en-us/help/211632/description-of-how-word-creates-temporary-files).

However, you obviously shouldn't be losing your original document. Have you tried searching for it? In particular, if you clicked an item on the list of recent files in Word, the original file may have been moved, which is why searching for it would be a good idea.

Note that if you have been editing an e-mail attachment which you opened directly from an e-mail, you will have to save it under a new name for the changes to be retained.

Except for the cases mentioned above, documents shouldn't get lost, unless your local drive is damaged.
