Well, from your error message, "something" has deleted a specific file:

Dyld Error Message:

The file MicrosoftComponentPlugin should be in /Applications/Microsoft Office 2008/Office.  If it is not, or it is at the wrong version, that will cause the problem.

First, look in the /Applications/Microsoft Office 2008/Office folder to see if you can see a file named "MicrosoftComponentPlugin".  If you cannot, then see if you can find it in the Trash.  If you can't, or if the file IS there, then you need to do a Remove and Re-install followed by an update.

To do that:

1)  Check that you have your original Office CD AND your Product Key available.  If you have not, go no further: because without the key you will not be able to re-install.

2)  Quit all Microsoft applications

3)  Go to /Applications/Microsoft Office 2008/Additional Tools/Remove Office and run the "Remove Office" utility.

4)  Empty the Trash and Restart the computer.

5)  Re-install Office.

5)  In any of the applications, go to Help>Check for Updates and run that as many times as it takes to tell you you are fully up-to-date.

Do NOT attempt to run any Office applications until you have updated, otherwise you may get problems.

Hope this helps
