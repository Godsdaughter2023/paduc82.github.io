joeu2004 say: I wrote:
If that is too difficult to do for "the whole spreadsheet", we could provide a macro.  Is that what you want?

Here is one implementation.  To create the macro:

1. In Excel, press alt+F11 to open the VBA window.
2. In VBA, click on Insert, then Module to open a VBA editor pane.
3. Copy the macro below, and paste it into the VBA editor pane.
4. You may now close the VBA window.

To use the macro in Excel, press alt+F8, select the macro, and click on Run.

Execute the macro for each worksheet.  Alternatively, we could modify the macro to iterate over all worksheets.


-----

Option Explicit

Sub doit()
Dim cell As Range, oldCalc As Variant
With Application
    oldCalc = .Calculation
    .ScreenUpdating = False
    .Calculation = xlCalculationManual
    .EnableEvents = False
End With
For Each cell In ActiveCell.SpecialCells(xlCellTypeConstants, xlTextValues)
    If InStr(cell, Chr(127)) > 0 Then
        cell = Trim(Replace(cell, Chr(127), " "))
    End If
Next
With Application
    .EnableEvents = True
    oldCalc = .Calculation
    .ScreenUpdating = True
End With
End Sub
