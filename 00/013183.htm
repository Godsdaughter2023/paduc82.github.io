andreas killer say:

Am 28.08.2010 02:12, schrieb Jastrick:

I've been using a macro on one of my workbooks that allows me to paste onto the visible cells only.

This problem can be very easily solved if you just copy the visible cells.

1. Select your range with or without hidden cells
2. Start this macro:

Public Sub CopyVisibleCells()
  'Copy only cells!
  If Not TypeOf Selection Is Range Then Exit Sub
  Selection.SpecialCells(xlCellTypeVisible).Copy
End Sub

3. Go to the destination cell
4. Press Ctrl-V (or right click the cell and select insert)
