Word Tables / VBA: Selecting the Range of the First Column - Microsoft Community

l.a. lawyer say:

Word Tables / VBA: Selecting the Range of the First Column Only

I need to extract the text (range) of each row of the first column only of the last table in the document.  The number of rows is variable.

Also, I need to have some way of identifying the text of each row separately (since the text of each row will be used for a separate page).

How is that done?

TIA

jay freedman say:

Here are two very simple demonstrations, using a message box instead of whatever you plan to do with the text.

    

Sub x()
    Dim rownum As Long

    With ActiveDocument.Tables(ActiveDocument.Tables.Count)
        For rownum = 1 To .Rows.Count
            MsgBox rownum & vbCr & .Rows(rownum).Cells(1).Range.Text
        Next rownum
    End With
End Sub


Sub y()
    Dim rownum As Long
    Dim col As Column
    With ActiveDocument.Tables(ActiveDocument.Tables.Count)
        Set col = .Columns(1)
        For rownum = 1 To col.Cells.Count
            MsgBox rownum & vbCr & col.Cells(rownum).Range.Text
        Next rownum
    End With
End Sub

The first one, which is preferable because it's simpler, just loops through the table's rows and picks out .Cells(1) of each row. The second one gets the table's first column in a Column object, and then loops through all the cells in that column. The results of both macros are identical.

As a practical matter, you'll want to remove the last character of the cell's .Range.Text, because that character is the cell marker. You can use the Replace function to replace Chr(7) with "" (the empty string).

I don't know what to do with your second paragraph. What do you mean by "identifying the text of each row separately"? Do you just want to insert a page break after each row's text?
