It depends on how Windows 10 was originally configured on your computer.

Open a command prompt by clicking start-->run, then type CMD in the run box and click "OK" or press the enter key. At the command prompt, copy and paste:

rundll32.exe advpack.dll,LaunchINFSection %Windir%\Inf\sr.inf

To address the registry restore question, you would not be able to restore the registry unless you had first made a copy of the registry at some point. It may however cause some application, that you installed at some point AFTER the date of your registry backup, to fail in it's performance. This would of course be because the registry entries required to run that application would be missing after the restoration of a registry copy dated prior to the installation of that software.

lemp say:

Danger, Will Robinson!

Before you delete anything, make sure that you know what is on your D: drive and what you propose to delete.

Although it usually is hidden and has no drive letter, sometimes the partition used by your computer manufacturer to save the disk image used to "restore to factory conditions" does have a drive letter.  Because these "recovery partitions" are only intended to be used for recovery and not for user storage purposes, they are by design almost full.  If some program mistakenly begins to save files in the recovery partition, the partition can reach the "alert" condition very quickly.

Do as Frederik Long suggests and use http://www.jam-software.com/treesize_free/ to see what's there (it's usually best to run TreeSize Free "as administrator" by right-clicking on the shortcut). 

If you can't determine what might be safe to delete, just ask here, preferably including a screenshot of the TreeSize output (https://answers.microsoft.com/en-us/feedback/forum/fdbk_commsite-fdbk_getstart/how-to-include-a-screenshot-in-your-post/2594b08e-32a3-476a-85a6-b021181be7e4)

Step 2: 

  • How to: Perform a clean install of Windows 10

In BIOS, select the Windows bit version you are going to install:

  1. Press F2 during boot to enter BIOS Setup.
  2. Go to the Configuration menu.
  3. For the Select Operating System setting, choose either Windows 64-bit or Windows 32-bit.
  4. Press F10 to save and exit BIOS Setup.
