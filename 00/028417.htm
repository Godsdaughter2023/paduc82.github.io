Hi Vince,

This is a known issue for the Microsoft Office for the Mac team. 

HD images are being displayed in low res on the HD display. In Office 2011, Microsoft has fixed the text display issue that occurred when the Retina (HD) displays were introduced. The display issue for graphics will not be fixed until the next full release of Office due out sometime next year.

A workaround is to zoom in to around 120% or greater and the blurriness is greatly reduced... at least to my eyes.

suzanne s. barnhill say:

Yes, every time you change the number of columns, you will have a Continuous section break. If you have not already opted to display the section number in the status bar, you should do so (right-click on the status bar and click on Section). This will help you keep track of where you are. Formatting page numbering in "landlocked" sections is much easier in Word 2007 and above than in earlier versions because you can access the Page Number Format dialog from anywhere in the document using Insert | Header & Footer | Page Number | Format Page Numbers...

If you insert an Odd Page section break and the previous section ends on an odd page, Word will insert a completely blank even page. You will not be able to see this in Print Layout view, but you can see it in Print Preview if you display two or more pages (this will also allow you to see facing pages). The preview on the Print tab of the Backstage won't display just two pages; if you reduce the zoom enough to show two, it will show four. But you can display the classic Print Preview (which uses the entire screen) by adding Print Preview Edit Mode to the Quick Access Toolbar.

If you have enabled Different First Page in more than one section, the First Page Headers will by default be linked; you can unlink them if necessary or, if you don't need a First Page Header in a given section, you can turn off Different First Page for that section. The First Page Header and primary Header are entirely separate; you cannot link them. Each header or footer links only to the header or footer of the same type (First Page, Odd (primary), or Even).

From your description of the problems you're having, I doubt that a 3-4 page sample would be representative, but you can send it to ssbarnhill at gmail dot com.
