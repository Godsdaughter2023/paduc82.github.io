What is MSE 3.0? - Microsoft Community

mayanksingh_781 say:

What is MSE 3.0?

The Beta release is 4.0. The stable release is 2.1. Then what is MSE 3.0? Why has it not been released?

kosh vorlon say:

Hi Mayank,

 

MSE 3 is the exact same thing as MSE 4 and both are simply names given (right or wrong and AFAIK not officially) to what is the current MSE Beta being tested.  It started out being called MSE 3 (since the current version is 2.1.1116.0), but when the Beta came out we were surprised to find the version listed as 4.0.1111.0.  So some still call it MSE 3 as being the next number and what it was called before it was actually released and some now call it MSE 4 because of the version number assigned.  I've no clue why they went from 2.x to 4.x and skipped 3.x, but it really doesn't matter.

 

Because of the confusion, I've personally started calling it "the current MSE Beta" and avoiding use of numbers (or adding the 4.0.1111.0 part which while not so important yet, may become more important if we start getting different versions of the Beta all out there at the same time).

 

I hope this helps.

 

Good luck!
