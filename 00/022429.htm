HiKristofer Gafvert,

This issue can occur for any of the following reasons:

·         The content on a Web page is only partially downloaded (for example, if you click Stop while a page is still being opened, or if you navigate to another page before a page is completely downloaded).

·         Some of the files in the Temporary Internet Files folder are hidden, used by other processes, or otherwise protected files.

To work around this problem, move your Temporary Internet Files folder to another location, delete the old folder, and then (if you want to) move your Temporary Internet Files folder back to the original location:

a.    Create a new folder in the Explorer.

b.    In Internet Explorer, click  Internet Options on the Tools menu.

c.    On the  General tab, click Settings under Browsing History.

d.    In the  Settings dialog box, click Move Folder in theTemporary Internet Files and History Settings.

e.    Click the folder that you created in stepa, and then click OK.

f.     If you are prompted to log off, click  Yes and then log on to Windows again. If you are not prompted to log off, click OK to close all of the dialog boxes.

g.    In the Explorer window, delete the old Temporary Internet Files folder, and then create a new folder in the same location with the same name.

h.    Repeat steps  'b' through 'f' again, but in step 'e', click to select the new folder that you created in step  'g'.

NOTE: You must repeat this process when logged on as each user of the computer.

white margins with overlapping pictures and text - how to remove?

Hey with Office 2013 they introduces a new white margin around text which is lying over a picture (in the background with layout options).

I wan`t to have it seamlessly overlapping, without any extra margin. For example: A picture with something written over it, seen as being together.

Thank you for your help!
jay freedman say:
It seems that you have white background shading applied to the text. The default is "No Color", which is transparent. If you ever applied white shading to any text in the past, or modified the Normal style, this may have been transferred to your Normal.dotm template.

Click in the paragraph of text that passes over the picture. On the Home tab, in the Paragraph group, click the down arrow on the Shading button (the one that looks like a paint can) and choose No Color in the dropdown.
