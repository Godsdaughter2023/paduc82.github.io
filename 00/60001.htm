Configure this router as a wireless access point only, remove any internet connection details and disable DHCP on it, give it a static ip address in the same subnet as the gateway. The gateway will probably have an ip address of 192.168.x.y, so your router should have a static ip address of 192.168.x.z, where z does not clash with either the gateway ip address or the ip address of either of your PCs. Make z as big a number as you can without clashing (max 254). Make a note of the access point's SSID and set up some form of encryption for wireless connections (WEP/WPA) and make a note of the encryption key too.

 

Having set up the Edimax as an access point, disconnect it from the PC and reconnect the PC to the Hotbox (reset TCP/IP to automatic if you had to alter settings). Connect the Edimax to the Hotbox with an Ethernet cable.

 

Connect any devices that need wireless connection to access the internet or your internal network (e.g. smartphone/printer/television/laptop etc.) to the SSID of your router using the encryption key that you set up.

 

If you ever get notifications of ip address conflicts, power off all devices (routers, access points, PCs, notebooks, smartphones, printers, televisions etc.), then power up in this order...

Hotbox (wait for stable indicator lights)

Edimax (again, wait for stable indicator lights)

Other devices in whatever order you want

 

Let me know what ip addresses the Hotbox gives to the PCs, what your gateway's ip address is, what the ip adresses of the PCs are and also what ip address you assigned to the Edimax.
