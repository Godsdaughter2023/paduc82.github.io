¡firedog say:
Problems, especially intermittent ones, with Yahoo accounts are often caused by minor corruptions in email messages at the server. Log into the account in webmail and start by clearing out the trash and spam folders. Then move all the messages in the inbox to a user-created temporary folder. Now conduct a send/receive in Windows Live Mail - you should see a No new email messages status message. Then move half of the messages back into the inbox and do send/receive again. Rinse and repeat, halving the number of messages moved each time until the error recurs, so that you can identify the message(s) that are causing problems and deal with them in webmail.

ghlnknjc say:

Is it faster to run RAM in dual channel mode?

I would like to add more ram up to at least 8 gig. What will work better performance wise, adding one more stick of 2 gig making all dual channel or putting in just two 2x4 gig sticks in the dual channel mode. THANKS  
original title: Adding System Ram
j w stuart say:

First install and run CPU-ID (CPUZ): http://www.cpuid.com/cpuz.php
Shows the clock speed of the CPU under various load conditions
(when using Intel's EIST speed step technology).
Note #1:
CPU-ID has two tabs -  'Memory' tab which show the actual memory speed
and the 'SPD' tab shows the rated speeds for each memory slot that is populated.
Note #2:
Compair the two values, the actual memory speed should not exceed the rated speed of your memory.

 

Use the results as to what to buy. Memory is usually installed in pairs (Slots 1 and 3, and the second pair in slots 2 and 4 with slot 4 being the furthest away from the CPU.).

 

What I would do is remove the AC power cord, wait 5 minutes and then carefully remove a three memory sticks, keeping each stick in the same order as the slot it was removed from. Check the part number (in detail) of all the sticks.

 

1) If they match then just purchase 1 more additional stick of the same part number.

2) If they do not match then go for the "two 2x4 gig sticks"  option, choosing a performance spec. that meets or exceeds what you currently have. The operating voltage of the replacement sticks must be the same as what you currently have.

 

Now with AC power cord still removed carfully insert each stick in the order it was removed, make certain the latch at each end is locked / inserted fully into the small slot. Run your finger over the top of all 3 sticks, if one feels like its higher that the others then its not fully inserted and locked.

Then connect AC power and reboot. Once your new memory arrives follow the same proceedure for removing AC power and inserting the new stick or sticks.
