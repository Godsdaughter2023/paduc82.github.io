Remove space resembling characters from a range of cells - Microsoft Community

ashish mathur say:

Remove space resembling characters from a range of cells

Summary
When you copy data from External Programs e.g. body of Outlook e-mail message or from a table on a webpage, then some space resembling characters become part of the copied text. These space resembling characters prevent you from working with mathematical calculations on such a range. Surprisingly, functions such as TRIM() and CLEAN() and tools such as DATA > TEXT TO COLUMNS and FIND AND REPLACE (Find for a space) do not remove such space looking characters either.
Details

To remove these space resembling characters from a range of cells, try this

 

1. Select any one cell of the copied range, press F2 and copy one space resembling character

2. Select the range of cells and press Ctrl+H

3. In the Find What box, press Ctrl+V

4. Ensure that the With box is empty

5. Before clicking on Replace All, ensure that the two checkboxes in the Options section are unchecked

6. Click on Replace All

 

This should get rid of space resembling characters from the range of cells.

 

To solve this problem with a formula, you may try this

 

1. Suppose data is in range A2:A10

2. In cell B2, enter this formula and copy down

 

=SUBSTITUTE(CLEAN(TRIM(A2)),CHAR(160),"")

 

If the steps above do not solve your problem, then try this formula in cell B2 and copy down

 

=SUBSTITUTE(CLEAN(TRIM(A2)),CHAR(13),"")

john mcghie say:
If you select all the objects on each slide and "Group" them, PowerPoint will express them all on the same layer.  Technically, that's not completely "flat" but it should be close enough.

Otherwise, you do need a graphics application to do this properly.  GIMP would be a great choice (it's free...).  PhotoShop would be overkill (and it's expensive...).  You could Save As out of PowerPoint and choose a format of PNG, TIFF or JPEG.  That will turn each slide into a picture, and it will be completely flat. PNG will preserve detail and be small.  TIFF will preserve 'everything' (it's lossless) but not so small (maybe 20 times the size of PNG.  JPEG is good for photos, but a disaster for this that need detail, such as text.

My first question, though, is "WHY??" Why do you need to flatten it? Why would it matter whether there are layers in the slides or not?

I would also be very likely to simply send them the PowerPoint file.  "Here you are, print it: that's what I am paying you for!"  If they can't do it, find someone else :-)

jeanette cunningham say:

Here's the code, after testing that the code works for you, then I discuss where to put the code.

-------------------

Private Function RefreshExcel()
Dim appexcel As Object
Set appexcel = CreateObject("Excel.Application")
appexcel.workbooks.Open "C:\Users\Jeanette\Documents\Testing.xlsx"
appexcel.activeworkbook.refreshall
Set appexcel = Nothing
End Function
-------------------

Note: change the name of the excel file  and the full path to the file, to suit your database.

 

My file was called Testing.xlsx

The full path to the file was C:\Users\Jeanette\Documents\

 

You can test the code by putting a button on your form and getting the button to run the code called RefreshExcel.

After testing that the code does refresh the excel file, then you can call the code before you run the macro that emails the report.

 

If you are not using a form to send the report, you probably need to make the routine called

RefreshExcel into a public function and put it in a module (not a module behind a form or report).

Then you call that function from a macro before you run the macro that emails the report.

 

To make it a public function, change the word "Private" to "Public" on the first line of the function.

robyork say: Mike,

I've been tinkering with it this afternoon.  There are are 2 ways to send a Word/Excel/PPT doc in RT.

1.  From Word/Excel/PPT, save the document to the hard drive of your Surface.  Make sure you save it into the "Documents" folder.  Next, go back to the home Metro screen (the one with all the tiles) and open the email app (you should have an  online email account set up with Microsoft).  Click the "+" at the top right of the screen to create a new email. Address it, then swipe from the bottom bezel up to show options and click/touch "attachments". You may have to touch "Files" then "Documents" to get to the documents folder. You should see your file, touch it, then touch "attach" on the lower right.  Whaalaaa!

2.  The second method is to save it to Microsoft's cloud called "Skydrive" and share it with others.  You can save it directly to Skydrive from Word/Excel/PPT by clicking the File tab, then Share, then Skydrive.  Your contact must be in the Skydrive share.

Hope this helps.
