Counting the number of tabs or sheets in a workbook - Microsoft Community

john illinois say:

Counting the number of tabs or sheets in a workbook

How can I find out how many tabs or worksheets I have in my workbook?
shane devenshire say:

Where is it that you want to put it - the box?  If you mean cell

Sub HowMany()
    Sheets("Sheet1").Range("A1") = Worksheets.Count
End Sub

A macro is not a formula.  You would press Alt+F11, and choose Insert, Module and then paste the above code into the code sheet on the right.  This formula would put the number of sheets in the workbook into cell A1 of sheet1, you can change those. 

I figured you were using this in a macro, maybe it would be better to use this

Function Sheets()
    Sheets = Worksheets.Count
End Function

Then you can enter the formula

 =Sheets()

In any sheet of the workbook.  The code for this is added just as in the previous discussion.
