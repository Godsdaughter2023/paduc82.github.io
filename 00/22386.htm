ken sheridan say: You only need one form and one query.  The form's RecordSource property is a query which returns all categories of orders.  You can then open the form filtered to the relevant category with code in the Click event procedure of each of the buttons.  The code in the 'Current' button's event procedure for instance would be along these lines:

    Const FORM_NAME = "NameOfYourFormGoesHere"
    Const CRITERION = "OrderCategory = ""Current"""

    DoCmd.OpenForm FORM_NAME, WhereCondition:=CRITERION

If you are unfamiliar with entering code into a form's, report's, report section's  or control's event procedures, this is how it's done in form or report design view:

1.  Select the form, report, section or control as appropriate and open its properties sheet if it's not already open.

2.  Select the relevant event property and select the 'build' button (the one on the right with 3 dots).

3.  Select Code Builder in the dialogue and click OK.  This step won't be necessary if you've set up Access to use event procedures by default.

4.  The VBA editor window will open at the event procedure with the first and last lines already in place.  Enter or paste in the code as new lines between these.

Syllable Counting - Microsoft Community

druid9 say:

Syllable Counting

Does MS Word have a syllable counter?  Or is there any add-on software for Word that specifically counts syllables?

shashirekha say:

The syllable count in word is not displayed directly, but with the Flesch Reading Ease test we can determine the average syllables used in the document.

The formula for the Flesch Reading Ease score is:

206.835 – (1.015 x ASL) – (84.6 x ASW)

Where:

ASL = average sentence length (the number of words divided by the number of sentences)

ASW = average number of syllables per word (the number of syllables divided by the number of words)

For more information on this click on the link below:-

http://office.microsoft.com/en-us/word-help/test-your-document-s-readability-HP010148506.aspx?redir=0

Sub GLMacro_AK()
  Dim All As Range, R As Range
  Dim Account As Range, Debit As Range, Credit As Range, Balance As Range
  Dim FirstAddress As String, MyFormula As String

  'Screen off, runs faster
  Application.ScreenUpdating = False

  'Get all data
  Set All = Range("A1").CurrentRegion
  'Remove previous results (if any)
  All.RemoveSubtotal

  'In the first row...
  With All.Rows(1)
    'Find the headings for our needs
    Set Account = .Find("Account", LookIn:=xlValues, LookAt:=xlWhole)
    Set Debit = .Find("Debit")
    Set Credit = .Find("Credit")
    Set Balance = .Find("Balance")

    'Be sure they exists
    If Account Is Nothing Then
      MsgBox "Heading 'Account' not found"
      Exit Sub
    End If
    If Debit Is Nothing Then
      MsgBox "Heading 'Debit' not found"
      Exit Sub
    End If
    If Credit Is Nothing Then
      MsgBox "Heading 'Credit' not found"
      Exit Sub
    End If
    If Balance Is Nothing Then
      'Add it on the right side
      Set Balance = All.Cells(1, All.Columns.Count + 1)
    Else
      'Clear previous formulas
      Balance.EntireColumn.ClearContents
    End If
    Balance = "Balance"
  End With

  'Sort by Account
  All.Sort Account, xlAscending, Header:=xlYes
  'Add subtotals
  All.Subtotal Account.Column, xlSum, Array(Debit.Column, Credit.Column), _
    True, True, xlSummaryBelow

  'Add the Balance formulas on each row with a SUBTOTAL formula
  Set R = Debit.EntireColumn.Find("Subtotal", LookIn:=xlFormulas, LookAt:=xlPart)
  FirstAddress = R.Address
  Do
    'Create the formula
    MyFormula = "=Debit-Credit"
    MyFormula = Replace(MyFormula, "Debit", R.Address(0, 0))
    MyFormula = Replace(MyFormula, "Credit", _
      Intersect(Credit.EntireColumn, R.EntireRow).Address(0, 0))
    'Write it into the Balance column
    Intersect(Balance.EntireColumn, R.EntireRow).Formula = MyFormula
    'Find next SUBTOTAL
    Set R = Debit.EntireColumn.FindNext(R)
  Loop Until FirstAddress = R.Address

  'Grab the whole range again (Remember: we've added some rows/columns)
  Set All = All.CurrentRegion

  'Make some formattings
  With All.Rows(1)
    With .Interior
      .ColorIndex = 49
      .Pattern = xlSolid
    End With
    With .Font
      .ColorIndex = 2
      .Bold = True
    End With
  End With
  All.EntireColumn.AutoFit
  Debit.EntireColumn.Style = "Comma"
  Credit.EntireColumn.Style = "Comma"
  Balance.EntireColumn.Style = "Comma"

  'Reset the last cell
  '  Note: SUBTOTAL moves this cell down if applied more than one time
  ActiveSheet.UsedRange
  'Done, show the result
  Application.ScreenUpdating = True
End Sub
