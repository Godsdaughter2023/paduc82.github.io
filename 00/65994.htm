Hello Anubh,

Thank you for posting your concern in Microsoft Community and welcome to the Windows 10 Family.

The error SYSTEM_THREAD_EXCEPTION_NOT_HANDLED can be a result of incorrectly configured, old, or corrupted device drivers. As well it can be due to corruption in Windows registry from a recent software change.

I suggest you to deploy Startup Repair which automates common diagnostic and repair tasks of even unbootable Windows installations. If the computer fails over into Windows RE because of a detected boot failure, Startup Repair launches automatically. If however, Automatic repair fails over Windows RE too, Startup Repair can also be launched as a manual recovery tool from a Windows RE installation media.

Automatic Repair using an installation media

  1. Create Windows 10 ISO media on USB from another computer using the below link:

    http://www.microsoft.com/en-us/software-download/windows10

  2. Insert the USB media and boot Windows from it.

  3. In the ‘Windows Setup’ Page select the ‘language to install’, ‘Time and currency format’ and ‘keyboard or input method’ and click on ‘Next’.

  4. Click on ‘Repair your computer’ and select ‘Troubleshoot’.

  5. Click on ‘Advanced options’ and select ‘Automatic Repair’ and select the operating system.

Additionally, you may also refer to the below Microsoft link on Troubleshoot blue screen errors.

http://windows.microsoft.com/en-us/windows-10/troubleshoot-blue-screen-errors
