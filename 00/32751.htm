Logitech controllers and wheels use the Logitech Profiler, v5.10.127 software.

http://support.logitech.com/support/en_us/search#q=lgs%20510&t=Products&scope=d

-  Go to Device Manager and locate the G27 wheel (or the 'unrecognised' device) in the

- Click on Update driver and select the Browse my computer option.

- Browse to your location for the XBOX software and select.

- Install the driver, and maybe reboot (I don't recall if I had to reboot when I had this

Disabling CEIP Customer Experience Improvement Program

1. Click start and type gpedit.msc on search bar or on RUN command
