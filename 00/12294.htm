How to install Zune Software on Windows Server 2012

With WP8 Microsoft drops their early adopters as useless **** (even newly released 'Windows Phone' application for Windows Store doesn't support our 'pre-historic' devices). So let's help each other...

If you have Windows Server 2012 and wish to connect or update your WP7 device, you can do the following:

  1. In Server Manager's Add Roles Features Wizard check if .Netframework 3.5 Features and Desktop Experience are installed. It should be!
  2. Download full ZunePackage.exe from official download page (version 4.8 is approx. 270 Mb)
  3. Unzip ZunePackage.exe as an archive file (by 7-zip or other tool) to specific folder (let's name it Zune).
  4. In Zune\x64\packages\ subfolder find zune-x64.msi, zunewmdu-x64.msi (and maybe zune-***.msi for your native language) .
  5. Run zunewmdu-x64.msi to install Windows Phone 7 devices updater.
  6. Run zune-x64.msi to install core components of Zune software.
  7. (optional) Run zune-***.msi to obtain your native language in Zune. Note, it should be server's GUI language too.

That's all, folks! ©

Can I convert a series of recurring events to individually separate - Microsoft Community

pierre dutronc say:

Can I convert a series of recurring events to individually separate events ?

Hello,

EDIT: When I write "event" below, I mean events and appointments.

I like the recurring events in outlook but when syncing to Android calendar the lest occurence is not respected and the event keeps on repeating endlessly. And when syncing the next time it will make a mess of the Outlook calendar on the computer as well. The only solution I see to this is to stop using recurrent events in Outlook.


So, is there a way for me to break the chains in Outlook while retaining all information in the events? 

To cut up every series of recurring events into its constituent parts, and make single events out of all the recurring events? 


Best regards!


brian tillman [outlook mvp 2007-2019] say:

Yes, you can do this.  Create a new folder containing calendar items.  Display your main Calendar folder where the recurring event lies in a table view like List or By Category.  Locate the event you want and copy it to the new calendar folder you created.  Now click File>Import and Export>Export to a file.  Export the contents of the created calendar folder (which will be just the recurring event) to a CSV file.  You'll have to give Outlook a date range for events whose recurrence pattern has no end date because you cannot export an infinite number of events.  Your exported file will contain every recurrence as a separate entry, including any exceptions you made to the pattern and any notes you recorded in the notes fields of the individual recurrences.

Now, return to the main Calendar, delete the original recurring event, and import the file you just created.  The original event will now be the series of individual events you requested.  Of course, if your recurrence pattern is still in effect and there are future recurrences, instead of deleting the original event, you can change the start date to the next recurrence of the event or the date you chose as the end date when you exported so that the past recurrences are all individual entries but your future recurrences are calculated from the recurrence pattern.
