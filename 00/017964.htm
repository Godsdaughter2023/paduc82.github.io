hansv mvp say:

Make a backup copy, just in case.

Create a query in design view based on the table.

Add the yyyymmdd columns to the query grid.

Change the query to an update query.

In the Update to row for each column, enter an expression like the one below, substituting the field name:

 

IIf(IsNull([MyText]), Null, Left([MyText],4) & "/" & Mid([MyText],5,2) & "/" & Right([MyText],2))

 

Run the query once.

Open the table in design view.

Change the field type of the columns to Date/Time.

dnyberg2 say:

Automatically place markers on the peaks of a spreadsheet

I routinely make measurements of things in a lab setting and produce low complexity X Y Scatter graphs.  The graphs always have about 20 peaks of interest.  Is there any way to automatically detect these peak values and place a marker on each peak along with the X & Y values without having to do this by hand for each one?  Thanks.
pbezucha say:

I see that the number of conditions for identifying your maxima inflates. I will leave the whole selection with eventual algorithms to you. Glad. I don’t doubt you will manage.

Nevertheless I have another rabbit out of hat. An invaluable one, at least I haven’t met such an access before. If you be able to mark all the maxima according to your choice by ‘m’ (or any fixed character that must be included in the macro as C-constant) in the column just adjacent to the y-column (remaining cells empty) then AddLabels should do the job for you.

 

Option Explicit

 

Sub AddLabels()

 

'Sub adds labels y/x to the  s e l e c t e d  series in xy-chart

'to the points that are marked by 'm' in the auxiliary column next to y-column

'(remaining cells in auxiliary column empty!)

 

Dim RngLabels As Range, Labels As DataLabels, Ser As Series, I As Long, N As Long, _

  StrSer As String

Const C As String = "m"

If TypeName(Selection) = "Series" Then

  Set Ser = Selection

  StrSer = Ser.Formula

  StrSer = Left(StrSer, InStrRev(StrSer, ",") - 1)

  StrSer = Right(StrSer, Len(StrSer) - InStrRev(StrSer, ","))

  Set RngLabels = Range(StrSer).Offset(, 1)

  Ser.HasDataLabels = True

  With Ser.DataLabels

    .Position = xlLabelPositionAbove

    .Orientation = xlHorizontal

  End With

  N = RngLabels.Count

  Application.ScreenUpdating = False

  For I = 1 To N

    If RngLabels(I) = C Then

      Ser.Points(I).DataLabel.Text = _

        CStr(Ser.Values(I)) & vbCr & CStr(Ser.XValues(I))

    Else

      Ser.Points(I).DataLabel.Text = ""

    End If

  Next I

Else

  MsgBox "No chart series has been selected"

End If

  Application.ScreenUpdating = True

Exit Sub

ErrExit:
