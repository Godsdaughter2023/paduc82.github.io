There are two product keys for an Office 2013/2016 installation.

allanlu_msft say:

Hi Jan,

We can open the AutoCAD (*.dwg) files in the OneDrive for Business sync client in our local folder. But we cannot open them in the OneDrive for Business from the browser.

1. Is it feasible to open AutoCAD (*.dwg) files directly (without downloading it first) from OneDrive for Business (from the browser)?
It’s not feasible to open the AutoCAD (*.dwg) files in the OneDrive for Business in the browser. When we choose to open it, it will download the file to the local.

2. Is it feasible to open AutoCAD (*.dwg) files directly (without downloading it first) using OneDrive for Business Next Generation Sync Client?
The OneDrive for Business Next Generation Sync Client is a desktop client and we can open AutoCAD (*.dwg) files stored in the client.

3. When we open a file (with method 1 and/or 2), is it feasible to click "Save" and have modifications uploaded to OneDrive automatically or do we have to save the file on our desktop and the upload it again?
If we open the file from the OneDrive for Business from the browser, as I mentioned above, it actually download the file to the local. And via this method, we need to save the file on our desktop and upload it again. If we open the file from the OneDrive for Business sync client, it will sync to the cloud automatically after we modify the file.

Feel free to let us know if you need further assistance.

Allan

OneNote 2013 - Ink Sizes and Standard Paper Sizes

Before my wall of text, I'll link some photos to illustrate what I'll be talking about.

Close Zoom, Medium Zoom, Far Zoom

What I have done:

Inside OneNote I made a test page, selected standard page size A4.  

I made a series of quick tests with different line sizes and different ink sizes (first three line sizes, and first three standard pen sizes).  I also did a small pressure sensitivity test to see how large the light vs heavy pressure would look.

I printed the page on a standard A4 size piece of paper.

I took my 0.5mm and 0.7mm pencils used in my engineering drafting class and made straight lines to illustrate the "working" size of the leads.

Finally, I took a ruler and measured what was supposed to be the 0.5mm pen inside of OneNote at full pressure.

What I found:

0.25mm pen size does not have pressure sensitivity.

0.35mm and 0.5mm pen size at full pressure are much greater than the indicated size.

0.35mm and 0.5mm pen size can produce a thinner line at low sensitivity than the 0.25mm at any/all sensitivity.

Discussion:

The first question I think coming my way is "are the pens the correct size when pressure sensitivity is turned off?"  Good question, they might.  My rebuttle is: does a pen/pencil have its maximum width when pressing lightly or heavily?  The pen/pencil should have maximum width when pressing the hardest, not the softest.

I have no idea what really goes on in the background of this software, and this issue may be harder to solve than I suspect.  But my main issue is this: if the pen sizes do not match even the standard paper sizes provided, why list the pen sizes at all?

P.S.  Please, please, please, if I am missing something, please tell me.  I honestly and sincerely hope I am just wrong and am missing.

How to Create macro to auto change color of a cell base on value - Microsoft Community

iyf2 say:

How to Create macro to auto change color of a cell base on value

"If cell is <50 change color to some yellow & if cell is >=100 change to another color."

How to simple write a VBA/Macro for auto coloring base on the value I place in field b1:b50. 

Cannot use Conditional Formatting. 

Items Cost
Client Site Taxi Fares 50
Stationery 100
Office Sundries 150
Coffee with Clients/Candidates 200

ossiemac say:

Right click on the worksheet tab name and select View code and then copy and paste the code below into the VBA Editor.

The particular event will run the code each time a change is made to any of the cells in the specified range.

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim myRange As Range
    Dim cell As Range
    Set myRange = Range("A1:A50")
    'Next line Not nothing then is something so Target is within range
    If Not Intersect(Target, myRange) Is Nothing Then
        For Each cell In myRange
            If cell.Value = "" Then GoTo SkipFormat
            If cell.Value < 10 Then cell.Interior.ColorIndex = 6
            If cell.Value < 5 Then cell.Interior.ColorIndex = 3
SkipFormat:
        Next
    End If
End Sub

More info of interest:

You could also use a Selection_Change event so that every time you select something different on the worksheet, the code will run.

To find the available Event Subs, after you select the worksheet tab name and right click and select View Code, you will see 2 dropdowns at the top of the VBA editor that default to "(General)" and "(Declarations)". Select "Worksheet" from the first drop down and then select the required event from the second drop down and the sub name and end sub will appear in the editor.

Note that these sub names must not be changed or the events will not work. Also after adding an event sub, the dropdowns at the top will default to the sub within which the cursor is positioned.

Look for Help with using event VBA code.
