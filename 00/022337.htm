INSERTTEXT Filename {DOCPROERTY ... }

I have useful information saved in Document Properties fields of Word 2010 documents.

jay freedman say:
> My goal was to pull in properties from multiple other documents. Can this be done using the recommended add-in plus some extra know-how?

That's a horse of a completely different color. DocProperty fields in Word have no way to say "give me a value from another document". It will take some considerable macro programming.

Greg Maxey's add-in, at least as it exists now, will not give you access to properties from other documents. I know Greg likes a challenge, so I'll suggest it to him as an enhancement or maybe as a separate project, but whether he's interested is up to him.

In any case, it will take considerably more than "some extra know-how". The first task would be to define the way you want the program to work, in extremely fine-grained detail. For example, do you want to pull properties from all the documents in a given folder, or to specify for each DocProperty field which file and which property it should use? Where do you want to store the information needed to update the fields (it can't be stored in the DocProperty fields themselves), and when and how do you want to trigger an update? There will be many other such questions.

In the end, a Word add-in may turn out to be the wrong approach anyway. I suspect that what you really need is a database (or at least an Excel worksheet being misused as a database) to contain all the values. Then the Word documents would not contain the values as document properties, but would pull the values from the database when needed. If you want a summary of all the values, that could be provided by a database report (which can be formatted as a Word document).

jay freedman say:
Yes, it is possible to insert the values of both built-in and custom document properties into the text of a document.

The simplest way, without having to learn about fields and other arcane details, is to download and install the add-in from http://gregmaxey.mvps.org/word_tip_pages/cc_var_bm_doc_prop_tools_addin.html. It will let you pick the property names from a list and insert their values with a button click.
