echo s say:

If the graph was created in Excel and you pasted it into PPT, PPT creates a link to it. When you send the file to someone, the Excel file isn't available so the user will get a "not available" message if they try to edit the chart.

 

You might also check in File | Info and see if there is an option to edit links to files. You may be able to break the link there.

 

Note that even if you do either of these things, the  user will still be able to add a data table to the chart and that data table will still show the end point data that PPT uses to generate the chart. I suspect you may have added the Exchange Server service to your Outlook profile by mistake.  Unless you know for a fact that you have an email account on an Exchange Server, with Outlook closed, go to Control Panel > Mail > Email accounts, select the Exchange Server account and Remove it, then add a new account of the correct type.
