john w. vinson say:

First off, the form's AfterUpdate event is the wrong place to run an Update - you'ld get an infinite recursion, where the AfterUpdate event triggers an Update, which triggers the AfterUpdate event, which... Use the BeforeUpdate event instead.

Secondly, your test for NULL is incorrect. NULL is a funny beast - it means "undefined, unknown, uninitialized". As such nothing is equal to NULL, nor is anythingunequal to NULL - any comparison to NULL returns NULL. The proper syntax would be

Private Sub Form_BeforeUpdate(Cancel as Integer)
If Not IsNull(Me!Days_Schdl_Start) And IsNull(Me!Days_Schdl_End) Then
    Me!Days_Schdl_End = Me!Days_Schdl_Start
End If
End Sub
