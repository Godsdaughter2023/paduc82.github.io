I have a solution that works for me :-)

1, Start Outlook, and the main window will not open

2, Start Activity Monitor and kill the Microsoft Database Daemon

3, Go back to Outlook, and now the main window will open

4, Put the text "Microsoft Database Daemon" in the Spotlight, find it and start it

/Applications/Microsoft Office 2011/Office/Microsoft Database Daemon.app


Now you will have Outlook working until next time you have to restart the Mac
