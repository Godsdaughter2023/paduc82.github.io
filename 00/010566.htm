How you do this depends on when you want the updating to happen. This bit of code is attached to a command button on the form and returns the name of the selected option button, you can then populate your cell based on that

Private Sub CommandButton1_Click()
If Station.ActiveControl.Value Then

  Select Case Station.ActiveControl.Name
    Case Is = "OB1"
        Range("A2") = 1
    Case Is = "OB2"
        Range("A2") = 2
    Case Is = "OB3"
        Range("A2") = 3
    Case Is = "OB4"
        Range("A2") = 4
  End Select

 Else
  MsgBox "None selected"
 End If
End Sub

Range("A2").Value = -(OB1.Value + 2 * OB2.Value + 3 * OB3.Value + 4 * OB4.Value)
