export LANG="zh_TW.UTF-8"

# Connect to DB
my $db = DBI->connect("dbi:Oracle:$DBNAME", $DBUSER, $DBPWD) or die "Can't connect to Oracle database: $DBI::errstr\n";

(Type => 'text/plain; charset="utf-8"', Encoding => "quoted-printable", Data => $body)

While you can set any folder as default for the Address Book (and when you click To), that doesn't apply to the Contacts/People module.

Outlook *should* remember the last contacts folder used though.  When it doesn't remember the last used folder, either the navigation pane is corrupt or the folder is loading slow when you first start Outlook - if the selected folder is not available, Outlook drops to the default.  To reset the navigation pane and start with a fresh one, close outlook and restart it using the /resetnavpane switch - this will wipe out Mail favorites and reset the navigation panes. 

Open outlook with the /resetnavpane switch: Close Outlook. Press Windows key + R to open the run command and type or paste

outlook.exe /resetnavpane 

Then press Enter (or OK button) to restart Outlook. Note: Make sure there is a space between the words.

Crystal

suzanne s. barnhill say:
  1. On the Labels tab of Word's Envelopes and Labels dialog, click Options...
  2. In the Label Options dialog, select the Avery label that is the closest to your Uline layout.
  3. Then click New Label... and, in the Label Details dialog, change the settings to match your labels.
  4. Give the labels a new name and click OK to save them. They'll be saved in Other/Custom.
  5. With this label selected, you should easily be able to create a full sheet of the same label.

You can also use New Document to create a blank sheet of labels. Fill in the first label (they're set up as a table, so make sure you have table gridlines displayed so you can see the label boundaries) and, when you have it just the way you want it, then select it and Copy. Select the whole table and Paste. You can then save the document for reuse if desired.
