hansv mvp say:

You can create a main form based on tblGeneralAppInfo and a subform based on tblApplication (or a query based on tblApplication and tblTenants); the subform should be linked to the main form by setting its Link Master Fields and Link Child Fields properties to lngGeneralAppID.
