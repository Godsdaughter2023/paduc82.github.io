graham mayor say:
No! Safe mode merely suggests a way forward. The issue probably relates to the font and language at the cursor when you begin to type. Word is a style based application. Every paragraph (every character even) is associated with a style. The style at the cursor position is dictated by the document template. By default Word uses the normal template and the language settings of this template are determined by the Regional settings in Windows.

When you type in more than one language Word has to determine which language you are using in order to use the characteristics of that language. Word can detect the language based on what you type, but it is somewhat hit or miss. Much better to turn off the automatic language detection and setup input languages associated with the languages you use and then switch to those languages using keyboard shortcuts.

By default the keyboard shortcut to switch between Windows input languages is LEFT Alt + Shift. The current language is then indicated in the Windows task bar. You can then switch languages at will e.g
abcde (change language) αβψδε (change language) abcde

If you want to type some documents in (say) English and some in Croatian, then create a separate template for the language you use least, pre-configured with the styles you wish to use for that language. Then create documents from that template rather than the normal template. In fact it is good practice to use the normal template only for ad hoc documents and create document templates for your particular document requirements.

brian tillman [outlook mvp 2007-2019] say:
You can't decline an invitation part way through the recurrence pattern.  What you can do, though, is create a new calendar folder (named something other than "Calendar", of course), change the view of your main Calendar to a table view like Events, copy the event to the new calendar folder you created, then export that calendar folder (File>Import and Export>Export to a file) to a csv (comma-separated values) file.  You will be asked to select a date range for the export.  Select the start date of the recurrence pattern and specify an end date of when you wish to end attendance.  This will create a file containing one entry for every recurrence you've attended or wish to attend, including any exceptions, such as one occurrence being moved, and any notes you may have added to a particular occurrence entry.  You can then delete the event from your main Calendar, which will ask if you want to inform the organizer.  Do that.  Then import the file you export

princemarvin say:

You are right alfdereste, you will not find those apps on the application manager. No need to wait for the next firmware to reinstall those apps on "e:". What you have to do is simply as what radical24 said, change the setting on Ovi client on your phone to install apps on "e:". Then simply, go the Ovi store using the client, search for the apps, such as Bloomberg, AccuWeather, facebook, AP News & all the rest. Then simply download them & over install them on "e:". By this you will clear reasonably good space on "c:" & end up between 47MB to 51MB depending on the apps of your phone. After the reinstall, you will then find these apps under the applications manager.

You can also hard rest your N97 using the steps mentioned on this forum, but to me, you do not have to do that now since you may lose some apps forever, like Guitar Rock Tour, Rueters Slideshow, & LiveJournal. Those you cannot find anywhere at least for now...

Good luck.
