I have an issue with Remote Desktop. - Microsoft Community

shreyhanda say:

I have an issue with Remote Desktop.

I have the android app for the remote desktop. However, whenever I try to connect, it always disconnects me, fails and says "Connection Error" and in the dialog box it says"Operation Canceled". How can I resolve this ?

a. user say:

Hi Shrey,

 

Which Android app are you using?

 

I suggest you to try the following steps and check if it helps.

 

Method 1:

 

Enable remote connections on your PC

 

a.    Press Windows key + Pause\Break key on the computer.

b.    Select “ Remote desktop”.

c.    Under Remote Desktop, make sure that you allow remote connections to the computer and choose Network Level Authentication. If you have problems connecting, you can try disabling Network Level Authentication. You can also restrict connections to specific users by clicking on the Select Users button.

 

Method 2: Connect to your PC with an Android device

 

 

a.    Make sure when you launch the RD client, that you're on the Remote Desktops tab. From there, tap on the "+" to add a new connection.

b.    Enter a name for the connection, the host name or IP address of the remote PC, and the log-in credentials if you want them saved. If you leave the credentials blank, you'll be prompted for them at the start of the connection. Tap "Done" to save the remote desktop connection.

c.    Tap on the connection you just created to connect. If you get an untrusted certificate prompt, select "Trust Once" or "Trust Always," if you trust the remote PC. You should then see your remote PC's desktop.

 

Method 3:

 

If the issue persists, I suggest you to start the services like “Remote Desktop Connection Manager” and “ Remote Access Connection Manager”.

 

To do so:

 

a.    Press Windows key + R and type services.msc.

b.     Right click on remote desktop to connection manager and select the option “Start”. If it’s already started “restart” the service. Do the same for Remote Access Connection Manager also.

 

 

You can refer to the following Microsoft article for more information.

 

Remote Desktop app for Windows: FAQ

 

http://windows.microsoft.com/is-is/windows-8/remote-desktop-app-faq

 

Hope this helps.

 

If the issue persists, I suggest you to post back with the required information and we will be happy to help you.

andre da costa say:

Can I perform a clean install using the Free upgrade?

No, it will require that you are running a previous qualifying version and start the upgrade from within the qualifying version. You can initiate a clean install after completing the Upgrade.

Please be aware that you cannot use the free upgrade offer to perform a clean install on first attempt. You must first upgrade from the qualifying version of Windows 7 or Windows 8/8.1 (whether you do it through Windows Update or using the .ISO file). Ensure the upgrade is completed successfully and then ensure that it is activated. You can then proceed to do a clean install by using recovery media or using the Reset function in Start > Settings > Update & Security > Recovery > Reset this PC (Get Started).

See: 

How to: Perform a clean install using Reset this PC in Windows 10

How to: Create a Recovery Drive for reinstalling Windows 10
