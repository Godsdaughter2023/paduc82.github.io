One possible solution is to generate a MonthView control on a UserForm and show that form when the desired cell is selected.

Here is a sample file how this can be done, I also add the description below, in case someone can not or don't like to download the file.
http://dl.dropbox.com/u/35239054/DatePickerAlternative1.xlsm

Andreas.


Create a userform, name it ufDatePicker, add a monthview control and two command buttons (the positions doesn't really matter, the code moves the controls)

Add this code to the form

--- schnipp ---
'Version 1.0
'19.01.2013
'Andreas Killer
Option Explicit

Public Value As Date

Private Sub CommandButton1_Click()
  'Ok, return the current date
  Value = Me.MonthView1.Value
  Me.Hide
End Sub

Private Sub CommandButton2_Click()
  'Cancel, return zero
  Me.Hide
End Sub

Private Sub MonthView1_DateDblClick(ByVal DateDblClicked As Date)
  CommandButton1_Click
End Sub

Private Sub UserForm_Initialize()
  'Move the MonthView to the upper left edge
  Me.MonthView1.Left = 0
  Me.MonthView1.Top = 0
 
  With Me.CommandButton1
    'ENTER = default for Ok
    .Default = True
    'Move the button below the MonthView
    .Top = Me.MonthView1.Height + 3
  End With
 
  With Me.CommandButton2
    'ESC = default for Cancel
    .Cancel = True
    'Move the button below the MonthView
    .Top = Me.MonthView1.Height + 3
  End With

  'Resize the form and show the MonthView only
  Me.Width = Me.Width - Me.InsideWidth + Me.MonthView1.Width
  Me.Height = Me.Height - Me.InsideHeight + Me.MonthView1.Height
End Sub
--- schnapp ---

Add a normal module and add this code

--- schnipp ---
'Version 1.0
'19.01.2013
'Andreas Killer
Option Explicit

Function DatePicker(Optional ByVal Caption As String, _
    Optional ByVal Default As Date) As Date
  Load ufDatePicker
  With ufDatePicker
    'Store the caption
    .Caption = Caption
    'Set the default date if any
    If Default > 0 Then
      .MonthView1.Value = Default
      .Value = Default
    End If
    'Show it, this macro stops until the form is closed
    .Show
    'Return the date
    DatePicker = .Value
  End With
  Unload ufDatePicker
End Function
--- schnapp ---


Call the DatePicker within the Selection_Change event routine as this sample shows

'Version 1.0
'19.01.2013
'Andreas Killer
Option Explicit

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
  Select Case Target.Address(0, 0)
    Case "B2"
      'Choose a date
      Target = DatePicker("Choose a date", Target)
      'Go one cell below
      Application.EnableEvents = False
      Target.Offset(1).Select
      Application.EnableEvents = True
  End Select
End Sub

byhamc say:

There are a couple of ways to tackle this.

One is to use VBA, but, unless you're quite confident, this can be a daunting task.

The other is to introduce a further step in your scenario above.

So, keep C23 as the Yes / No Option. Then, in another cell, say D23 (I'm not sure of the layout of your spreadsheet), place the IF formula you have above, but replace the "" with wording to the effect of "Pick from list below"

Then in C24, have your data validation A,B,C,D.

Again, depending on exactly what you're trying to achieve, you can use conditional formatting to essentially 'hide' the detail in C24 if you pick No (set font colour to the same as background colour). And if you're stringing other formulae off of the A,B,C,D output in C24, simply re-direct the formulae to state if C23 = No, then output = "", otherwise pick value from C24:

=IF(C23 = "No","",C24)
