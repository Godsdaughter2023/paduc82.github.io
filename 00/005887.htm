Hi,

Selecting cells to read them is the least efficient way of doing things and is bound; sooner rather than later, to make your code crash.

Try it like this, no selecting anything and using fully qualified ranges.

Private Sub UserForm_Initialize()
Dim i As Long, ws1 As Worksheet
Dim c As Range, MyRange As Range
     For i = 2005 To 2014
         Me.CboYear.AddItem i
     Next
Set ws1 = Worksheets("Drop Down Lists (Chad)")
LastRow = ws1.Cells(Rows.Count, "D").End(xlUp).Row
Set MyRange = ws1.Range("d2:D" & LastRow)
For Each c In MyRange
    Cbocolor.AddItem c.Value
Next
Cbocolor.ListIndex = 0
Set ws1 = Worksheets("Dealer Info")
LastRow = ws1.Cells(Rows.Count, "A").End(xlUp).Row
Set MyRange = ws1.Range("A1:A" & LastRow)
For Each c In MyRange
    cboDealership.AddItem c.Value
Next
    cboDealership.ListIndex = 0
End Sub
