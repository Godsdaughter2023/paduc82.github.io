What does it mean if I want to "Run or Save" a file I am - Microsoft Community

carole777 say:

What does it mean if I want to "Run or Save" a file I am trying to download?

What does it mean if I want to "Run or Save" a file I am trying to download?

 

steve cochran say:

Run means to run the file (it is likely an exe or similar that will run a program).  Save means to save the file and you can click on it to run it later.  It is always best to save the file first (in a location you can easily find) and then scan it for potential viruses first before running.  Make sure you know what you are clicking on and downloading, as some things will immediately infect your computer -- often to the point of making it unusable.
