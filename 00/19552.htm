Getting the hyperlink to look like a hyperlink rather than ordinary text is much tougher because Word is set up to not have its automatic TOC hyperlinks be blue and underlined. Too many people found having the TOC look like that to be ugly and distracting.

suzanne s. barnhill say:

Press Alt+F9 to view the TOC field code. If it includes the \h switch, then the entries are already hyperlinked by default. When you mouse over one, you will get the "Ctrl+click to follow link" screentip. If the code doesn't include the \h switch, you can add it, but even when full entries are not hyperlinked, the page numbers always are.

As Charles said, they are not formatted as hyperlinks even though they will display the Hyperlink character style as the style applied. Ironically, if you unlink the TOC, the unlinked entries, still in Hyperlink style, will display blue and underlined (or whatever formatting is defined for the Hyperlink style).

how do you truncate a date field (DDMMYYYY) into a YY only - Microsoft Community

paul lovell say:

how do you truncate a date field (DDMMYYYY) into a YY only format that can be mail merged and printed as YY

Have a list of people and their dates of birth (DDMMYYYY) in a Excel 10 Spreadsheet, I want to show only YY against their names and also be able to mail merge the data into another document and print. I have used format and special with YY and it does show correctly in the Spreadsheet but when you Mail Merge and Print the whole date appears (DDMMYYYY).
hansv mvp say:

One option is to right-click the date of birth merge field in the Word document and select Toggle Field Codes from the context menu. You'll see something like

{ MERGEFIELD DateOfBirth }

or

{ MERGEFIELD DateOfBirth \* MERGEFORMAT }

 

Make it look like this:

{ MERGEFIELD DateOfBirth \@ "yy" }

Press F9 to hide the field code and update the result.

 

Another option is to add a column to the Excel sheet. Let's say the dates of birth are in D2 and down.

In another column, enter YearOfBirth in row 1 and the formula =TEXT(D2,"yy") in row 2. Fill down from row 2 as far as necessary.

Use YearOfBirth in the mailmerge.

My table:

Physician Name FTE Status Specialty # of Visits in 2016 Percentile
Dr. John Doe 0.90 Neurology 1230 ???

Table of percentile data I'm referencing:

Specialty 10 %tile 11 %tile 12%tile 13%tile 14%tile 15%tile ...and so on
Neurology 1235.40 1251.94 1292.48 1326.85 1360.26 1391.35 ...and so on

We use the closest-to-without-going-over rule, so in this example Dr. John Doe would be in the 14th %tile because 1,360.26 * 0.90(FTE status) = 1,224.23 which is the highest number his 1,230 visits exceeds (15th %tile would require 1,252.26 visits).
