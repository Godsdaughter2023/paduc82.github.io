graham mayor say:

You can only insert a PDF as an object. Inserted objects will shrink to fit the page width between the margins. If that is less than the size of the PDF the object will be scaled to fit. It is that scaling that reduces the resolution. You may be able to re-scale the object to 100% to improve the resolution, or convert to an image as suggested by Doug. If the PDF is more than one page you will have to split it in any case, so that would be the way forward.

    as titlle
    daniel g. grau say:
    1) Set the text in a two-column table (Minutes to the left, Actions to the right). Judicious use of rows will let you align Actions with specific Minutes.

    2) Less accurate, but equally viable: define two columns in your section (Format > Columns), which can have uneven sizes. Do Insert > Break > Column break to switch from Minutes to Actions.

    Most people just add Returns to horizontally align the text in both columns, but this soon becomes inaccurate if you use different fonts and sizes, or if the line-spacing and paragraph inter-spacing are not set just right. Tables is the best choice.

    rich michaels - mvp (greatcirclelearning.com) say:

    Try using a 2 column table and adjust the width of each column as fits your need.

    The nice thing about using a table is you'll be able to line up each action item to the appropriate comment.
