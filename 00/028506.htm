Error code 0x80004005-0x0004b9-0x000501 solution - Microsoft Community

l.something say:

Error code 0x80004005-0x0004b9-0x000501 solution

Outlook 2016 was not sending or receiving email for recently created Outlook.com email.

Error returned was:  

Error code 0x80004005-0x0004b9-0x000501  
  

 Couldnt find a solution and was irritated that the 4 or so technet forums that I visisted did not provide a clear solution and even worse the threads were locked so even after I resolved the issue, I could not reply my findings.  ****? 

So here I am creating this discussion so another might find it before uninstalling or being redirected to some other technet support thread.

Although its mentioned that the error is network related, in my case it was due to a new outlook.com account that was created but never logged into via the web browser.

Upon logging into outlook.com with the affected account I was instructed to chose a timezone and a few other personal questions. Immediately after that was done, Outlook 2016 was then able to send and receive emails.

Other reasons you may get this error would be due to a scheduled password reset or account lockout threshold was triggered by too many incorrect login attempts.

When in doubt, whether it be O365 or Outlook/Hotmail the first place to check is logging into the account from a web browser.

Hope this helps someone in the future.
