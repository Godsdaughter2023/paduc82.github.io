dirk goldgar say: A text, memo, or hyperlink field will contain Null unless it either has a default value or something has been entered in it.  If there is a default value, that default value may be a zero-length string, but the user would have to have set it to be so.  It is also possible for a user to enter a zero-length string in a text field, though there's a trick to it -- just clearing the value displayed in the datasheet won't do it; you have to enter "" in the field.  Or an action query or import process can put a zero-length string in the field.

ben m. schorr -bschorr say:

If it's an OEM version then the answer is no.  The OEM license is locked to the machine the software came with and can't be transferred to a new machine.  If you're planning to sell the whole machine, with Office, then that's fine, but you can't sell an OEM license of Office without the machine it's tied to.
