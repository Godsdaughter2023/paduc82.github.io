cliff gu msft say:

Hi Chau,

 

Sorry for my misinformation in my post to make you a bit confused. So I’d like to make a summary to all your questions to help you in moving forward.

For external sharing, there are two options for you to move forward, Anyone and Specific people. For the Anyone option, the shared external user can open the shared file directly. For the Specific people option, the external user need the verification code to open the shared file.

 

Now let me reply to your questions.

1. Can i share to e-mail which is not of microsoft account or Office 365 account?

The answer is Yes. You can share a file with an external user who doesn’t use a Microsoft account or a work and school account such as a Gmail account.

2. Can i share to e-mail which he is not need sign-on and just click to link (file) on received e-mail and open it?

The answer is Yes. Based on your requirement, I suggest you use the Anyone option. You can achieve it via clicking on the file in the modern OneDrive interface>Anyone with the link can view and edit>type the external user’s email address>Send.

3. Can he/she update file after modify? How to share that way?

The external user who are granted the Edit permission can modify and update the shared files.

4. I sent it to Mr. A, if Mr.A forwards that mail to another person (Mr.B),

is Mr. B able open that file by click on that link as Mr. A?

There are two scenarios for the Anyone option and the Specific people option.

a. The Anyone option:
