Assuming that the list fill range of the dropdown is on the same sheet:

 

Sub DropDown1_Change()
    Dim ddn As DropDown
    Dim rng As Range

    Set rng = ActiveSheet.Range(ddn.ListFillRange)
    MsgBox rng.Cells(ddn.Value).Value

Hi,

We highly suggest you block the exact domain you are receiving unwanted messages from. Follow these steps: 

  1. Click the settings gear icon on the web toolbar.
  2. Select Options in the menu that has appeared.
  3. Open the Mail > Junk email > Blocked senders category.
  4. Type the address you want to block in the box "Enter a sender or domain here".
  5.  Click the + sign.

Please remember, to block mail from all addresses at a domain, enter the domain name—this is what follows the '@' sign in an email address. 

Once you add "sampledomain.com" to the list, it will, for instance, block messages from "she@sampledomain.com" as well as "he@sampledomain.com" and other addresses ending in "@sampledomain.com".

Note: You have to block sub-domains if you need to; "sampledomain.com" will not block messages from "ex@loc.sampledomain.com".

Aside from the steps mentioned above, you may also try these recommendations:

  • Change the level of protection in the Junk Email Filter
  • Manage email messages by using rules
  • Block or allow (junk email settings)
  • Get help with Outlook.com
