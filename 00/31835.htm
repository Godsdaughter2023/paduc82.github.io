  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Cryptography\Defaults\Provider\Schlumberger
  • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Cryptography|Defaults\Provider\Infineon

Perform the following steps to achieve this:

 

a.    Go to the Start menu and click Run.

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Cryptography|Defaults\Provider

j.     Right-click on the new string value you created, click Rename and then rename the value to Image Path.

k.    Now, right-click the string value, Image Path, click Modify, and in the box under Value data write C:\WINDOWS\system32\slbcsp.dll.

l.     Click OK.

m.  Next, select the Infineon SICRYPT Base Smart Card CSP key.

n.    Repeat the step i and j.

o.    Next, right-click the string value, Image Path, click Modify and in the box under Value data write C:\WINDOWS\system32\sccbase.dll.

u.    Click OK.

v.    Next, select the Infineon SICRYPT Base Smart Card CSP key in the left-pane of Registry Editor window.

w.   Repeat the steps r, s, and t.
