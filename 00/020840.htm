cecelang say:

I found a makeshift solution to this problem.

Apparently, the export to mp4 feature DOES NOT RECOGNIZE the Transition settings.

What it does when you export to mp4, is it automatically loads the second slide, bypassing the timing you created for the transition between slides.

So to solve this problem, I added an INVISIBLE TEXT BOX to EVERY slide and made it FADE (animation) with a timing delay equivalent to the timing in the transition between slides.  This replaces the transition timing.

Presto, the resulting mp4 is now in sync with my music background!

Create a named range "ListName" and reference that for the Dropdown  (=ListName). For the name's definition, use

=OFFSET(B6,0,0,COUNTA(B6:B1000),1)

Then you dropdown will show those values in your range, as long as you fill in starting from the top with no blanks. Another solution is required if you have blanks within your list.
