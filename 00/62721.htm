bossdweeb say:

d3dx9_43.dll is one of the most recent DX files ( if you can call 2010 recent ) and if you have never

'updated' DirectX  running the DirectX Web Installer will install any 'missing' DX files.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

Should you get a message after running the Web Installer that a 'newer or equvalent version' is installed, the

problem may be a corrupt d3dx9_43.dll file.

In this case you can delete this file from System 32 and run the Web Installer again and it will replace the

file with a new copy of d3dx9_43.dll  ( the Web Installer does not overwrite existing files, so the suspect file must

be deleted. )

- A lot of people still suggest installing DX 9.0c when there is an issue with DX9.

This installer is outdated, and does not have all DX files.

On those rare occasions when it is necessary to reinstall DX, generally only if there is an unknown corrupt DX file,

use the DirectX Redist (June 2010), as this has all of the DX9 file that 9.0c had and more (including DX10 and 11).

This installer will overwrite all DX files, as well as install any 'missing' files,  and fix any corrupt files in the process.

Download Details - Microsoft Download Center - DirectX Redist (June 2010)

BCCode: 4E   0x0000004E

Bug Check 0x4E: PFN_LIST_CORRUPT

The PFN_LIST_CORRUPT bug check has a value of 0x0000004E. This indicates that the page frame number (PFN) list is corrupted.

Important Information If You Have Received a STOP Code

If you have received a blue screen error, or stop code, the computer has shut down abruptly to protect itself from data loss. A hardware device, its driver, or related software might have caused this error. If your copy of Windows came with your computer, call the manufacturer of your computer. If you purchased Windows separately from your computer, Microsoft provides support. To find contact information for Microsoft or your computer manufacturerContact Support.

If you have experience with computers and want to try to recover from this error, follow the steps provided in the Microsoft articleResolving STOP (Blue Screen) Errors in Windows.

The following actions might prevent an error like this from happening again:

Download and install updates and device drivers for your computer from Windows Update.
Scan your computer for computer viruses.
Check your hard disk for errors.

Cause

This error is typically caused by a driver passing a bad memory descriptor list. For example, the driver might have called MmUnlockPages twice with the same list.

angelcarreno say: Hi,

My name is Angel. I am an Independent Advisor. Thank you for posting in Microsoft Community.

By default, Windows will use the index when searching to give you faster search results. The search index only includes your selected locations. These locations can be filtered for what file types, file properties, and file contents you want yo be indexed. Here are the steps that you can follow to rebuild search index in Windows 10:


1. Open the Control Panel (icons view), click/tap on the Indexing Options icon, and close the Control Panel.
2. Click/tap on the Advanced button.
3. In the Index Settings tab, click/tap on the Rebuild button under Troubleshooting.
4. Click/tap on OK to confirm.
5. Indexing will now be in progress.
6. When indexing has completed, click/tap on Close.
