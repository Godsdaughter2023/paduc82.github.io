Sub Draft()

Dim oSection As Section
For Each oSection In ActiveDocument.Sections

Next oSection
End Sub

If you put the XML file there, the theme color set will show up in the colors dropdown on the Design tab, but it won't be built into the template. rdrewd2 say:

Hi Dale,

Media Center has been removed from Windows 10, Media Player is still included, although I've heard from some folks that theirs was disabled and had to be restarted from the control panel.  It should be in your all apps list under the heading Windows Accessories, or you should be able to just type media player into your Cortana/Windows search box and hit enter to bring it up.  If neither of those works, type Windows features into your search box and in the results click Turn Windows features on or off.  In that list, click the + next to Media Features, and Windows Media Player should appear right below it.  Put a check in it's box and restart; if the box is already checked, uncheck it, restart, go back and check it, and restart again.  That should enable your player if it's currently disabled.  In regards to saving music to your PC, you don't normally need Media Player for importing, but where are you trying to import your music from (another PC, an external drive, online, CD collection, etc)?

daleday say: Doesn't solve my problem. The program is there and has music and videos from my Win8.1 version. But can't figure out how to add music videos to it.

rdrewd2 say: Without knowing where you're trying to import from I can't be of much help; this depends on the source, codec, and file extension of the video.  In general, you can just import most videos into your user videos folder and Media Player will find and play them, but again this is source and format dependent.  If you're wanting to download youtube music videos, you can just use any of the .mp4 downloaders available from the Windows Store. If you're downloading videos from iTunes, those will be DRM-protected to be played exclusively with the iTunes player and stored within the iTunes media folder, i.e. they won't appear in your videos folder and Windows Media Player won't recognize the format.  If you're wanting to import a live concert DVD, some will allow straight import from the DVD drive to the videos folder, some won't; but many of those come with digital-download entitlement certificates which allow a copy to be downloaded off the internet.  But again, without knowing what and from where, I can't really give any specific advice.
