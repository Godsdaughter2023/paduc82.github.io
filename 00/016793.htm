vincenzo di russo say:

Try this workaround.
1. Close IE9.
2. Go to Internet Options -> Content Tab -> AutoComplete:
- check "Address Bar" under "Use AutoComplete for".
- uncheck "Browsing history", "Favorites", "Feeds" and "Use Windows Search for better results" -> OK.
3. Start Internet Explorer 9.



Now your Address bar drop-down list will display the last 25 websites you typed in.
