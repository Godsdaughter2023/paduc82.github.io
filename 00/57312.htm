shinmila h say:

Hi jv59,

Welcome to Microsoft Answers.

Since Group Policy Editor is not available in Home Premium version of Windows 7, the other way of disabling it is through registry editor.

Important: This article contains information about how to modify the registry. Make sure that you back up the registry before you modify it. Make sure that you know how to restore the registry if a problem occurs. For more information about how to back up, restore, and modify the registry, click the following article number to view the article in the Microsoft Knowledge Base:

Back up the registry

http://windows.microsoft.com/en-US/windows7/Back-up-the-registry

 

1)       Click Start, type regedit.exe in the Start Search or Run dialog box and press Enter.

2)       Navigate to HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ Policies \ System

3)       Create a DWORD (32-bit) Value named HideFastUserSwitching by right clicking on the right side of the editor and selecting New >  DWORD(32bit) Value.

4)       Set the Value data for HideFastUserSwitching to 0 to disable fast user switching.

5)       In future if you want to enable fast user switching, set the value data for HideFastUserSwitching to 1.

Having done so, log off to see the changes. Fast User Switching should be disable after performing the steps above.

cmdrkeene say:

Well it's not really the resolution per se, which is fixed at the 3200x1800.  The issue you're familiar with is the DPI scaling that comes when you use anything other than the 100% size.  Some programs (less as time goes on) don't know how to stretch out correctly and will show up very small, even if you set your scale to be 200% (which should double the size of the program's interface, obviously).  Other programs respond to the scale, but their resources (literally the internal image files that make up the program's interface) aren't available in higher sizes, so the program's interface looks blurry or stretched.

That's not the issue at play here.  Instead, it's just the different scales being used.  You have your desktop set to 200%, but the modern interface set at default (100%), so the same image will appear twice the size when viewed from a desktop program.

The 'solution' if you want the same size everwhere, is to use the same scale everywhere.  So either set:

[Desktop = 100%] and [Modern = Default]

or

[Desktop = 125%] and [Modern = Medium]

or

[Desktop = 150%] and [Modern = Larger]

In any case, as long as they are set to their corresponding equal size, everything on them will appear equal sized regardless of which interface you are using.

Also yes, I can confirm the same behavior here on the Surface 2 and actually on the Dell Venue tablet as well.
