If you only have two windows (documents, more or less) open, you could try using the following Applescript instead (I think the above page probably tells you what you need to know to install it).

tell application "Microsoft Word"

set theWindows to every window

set theWindowCount to the number of items in theWindows

if theWindowCount = 2 then

arrange windows arrange style tiled

set window1 to item 1 of theWindows

set window2 to item 2 of theWindows

set theWidth to ((the (width of window1) / 2) as integer)

set theHeight to ((the (height of window1) * 2) as integer)

set the width of window1 to theWidth

set the height of window1 to theHeight

set the width of window2 to theWidth

set the height of window2 to theHeight

set the left position of window2 to the (left position of window1) + theWidth

if the top of window1 < the top of window2 then

set the top of window2 to the (top of window1)

else

set the top of window1 to the (top of window2)

end if

end if

end tell
