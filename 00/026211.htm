Private Sub Worksheet_Change(ByVal Target As Range)
    Dim rngS As Range
    Dim blnT As Boolean
    Dim i As Long
    Dim lngRow As Long
    blnT = True
    If Target.Count = 1 Then
        lngRow = Target.Row
        If IsEmpty(Me.Cells(lngRow, 7)) Then
        Select Case Target.Column
            Case 2, 4, 6
                With Me
                    For i = 1 To 3

                            blnT = False
                            Exit For
                        End If
                    Next

                End With

If you want to turn on iteration in the Calculation section (Enable Iterative Calculation) under options, and change Maximum Iterations: to 1.  then  you could put in a formula that refers to itself.

B3: =B3+B2

that will increment everytime there is a calculate event so you would probably want to go to manual calculation and hit F9 when you wanted B3 to increment.   The formula could certainly be more sophisticated.  It seems more prone to becoming corrupted and so forth.   I thing it would be better to use another sheet and put in each value of B2.  Then you could just sum those up.  You would also have an audit trail and certainly more confidence in the value in B3

B3: =sum(Audit!B:B)  

for example.
