how to requarantine a restored virus - Microsoft Community

gsmps say:

how to requarantine a restored virus

i have restored a virus by mistake and than the antivirus is not detecting it how can i fix it?

thank you

greginmich say:

Restoring a file from the Quarantine automatically registers a temporary path exclusion (in a "TemporaryPaths" registry key that’s invisible to the MSE interface). The restored file will no longer be monitored by MSE; and since MSE’s automatic exclusion for the restored file doesn’t show up in the “Excluded files and locations” settings or in the “Allowed items” list, there’s no easy way to immediately re-enable detection for the file – but when the automatic exclusion expires (after a couple of days), the ability to detect the file will be re-enabled, and running a Full Scan will return the file to the Quarantine.

 

http://answers.microsoft.com/en-us/protect/forum/protect_scanning/mse-beta-false-positives/601b6136-9433-4482-881c-68fbea546a29

 

Deleting the registry value for the restored file in the TemporaryPaths key is a possible solution, but it might be a little less confusing (and a little safer) if restoring a file from the Quarantine just automatically created a path exclusion in the main interface.

 

 

GreginMich

This formula works great to sum values of C3 cell for all months of Full year.

However, there is a risk involved...

If in your workbook, you have some additional worksheet having a 3 character name ( for e.g. say XYZ ) and if the cell C3 in this worksheet also has some value, then this 3-D formula will not be able to distinguish between the worksheets named for a month of a year.   1. Disable service click-to-run (in services - set disabled).
  2. Restart computer (click-to-run will be not loaded).

  6. Restart your computer.

jim gordon mvp say:

In my case I have users opening workbooks on a variety of monitors and laptops, so I let the user decide what zoom level they like, but I suggest 110. I also set the workbook window to fill the screen. Maybe this will work for you:

maximize & set zoom, allow Mac users to choose
    If Application.OperatingSystem Like "*Mac*" Then
        With ActiveWindow
            .WindowState = xlMaximized
            .Top = 1
            .Left = 1
            .Height = Application.UsableHeight
            .Width = Application.UsableWidth
        End With
        With Application
        'Allow user to choose zoom
            .Dialogs(xlDialogZoom).Show 110
            Let ZoomSetting = ActiveWindow.Zoom
        End With
    Else
    'The zoom dialog is not available in Excel for Windows so don't do anything here
    End If

Sub ALL_IN_ONE()
Dim WB, myWB As Workbook
Dim myWS As Worksheet
Dim FolderPath As String, FilesPath As String
Dim v As Variant
Dim N As Long, LRow As Long, r As Long, X As Long
Dim rTo As Range
Application.ScreenUpdating = False
On Error Resume Next
Set myWB = ThisWorkbook
Set myWS = myWB.Worksheets(1)

r = myWS.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row

If r = 1 Then r = 2

myWS.Rows("2:" & r).ClearContents

FolderPath = "C:\ your folder\" '  <<< change path
FilesPath = Dir(FolderPath & "*.xls*")
Set rTo = myWS.Range("A2")
N = 0
Do Until FilesPath = ""
N = N + 1
ReDim v(N)
v(N) = FilesPath
FilesPath = Dir()
If N = 0 Then Exit Sub
Set WB = Workbooks.Open(FolderPath & v(N))
For X = 1 To WB.Worksheets.Count
r = WB.Worksheets(X).Cells.Find(What:="*", SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row
WB.Worksheets(X).Rows("2:" & r).Copy
rTo.PasteSpecial xlPasteValues
LRow = myWS.Cells.Find(What:="*", SearchOrder:=xlByRows, _
SearchDirection:=xlPrevious).Row
Set rTo = myWS.Cells(LRow + 1, 1)
Next X
Application.CutCopyMode = False
WB.Close False
Loop
On Error GoTo 0
Application.ScreenUpdating = True
End Sub
