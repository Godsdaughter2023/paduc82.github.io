jeeped say:

... { IF {MERGEFIELD udExhibitA } = "True" "{ INCLUDETEXT C:\\...\\..docx }" "" } ...


I believe you are confusing the boolean value True with a text word "True" which will always resolve to False when directly compared (e.g. they are not the same thing). You might also want to get rid of some of the quotes so the field code within the quotes is not interpreted literally. I did a quick mockup with an Excel worksheet supplying the merge fields as true boolean True/False values and this worked for me,

{ IF { MERGEFIELD udExhibitA } = True { INCLUDETEXT  "c:\\temp\\my test.docx"  \* MERGEFORMAT } "" }

shyam pillai say:

You can use this routine to arrange the shapes in a circle.

 

Sub Test()
Call AlignShapesInCircle(720 / 2, 540 / 2, 100, ActiveWindow.Selection.ShapeRange)
End Sub

Function AlignShapesInCircle(x As Single, y As Single, r As Single, shprng As ShapeRange)
'x,y    = center point of the circle
'r      = radius of the circle
'shprng = the shape selection that needs to be arranged

Dim angle As Single
Dim currentangle As Single
Dim x1 As Single
Dim y1 As Single
Dim i As Integer

currentangle = 0
angle = 360 / shprng.count
For currentangle = 0 To 359 Step angle
    i = i + 1
    x1 = r * Cos(D2R(currentangle))
    y1 = r * Sin(D2R(currentangle))
    shprng(i).Left = x + x1
    shprng(i).Top = y + y1
Next
End Function

Function D2R(Degrees) As Double
    D2R = Degrees / 57.2957795130823
End Function
 
Function R2D(Radians) As Double
    R2D = 57.2957795130823 * Radians
End Function

Battery control is provided by Power options.  You can fine tune that control using the Windows 10 utility PowerCfg.

powercfg -energy >D:\Desktop\EnergyReport.txt

powercfg /batteryreport /output D:\Desktop\BatteryReport.html /Duration 1
