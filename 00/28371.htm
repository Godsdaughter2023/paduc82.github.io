I fought with this for a long time and finally read a blog post with a "trick" that helped - it's still a workaround but I'll share it 'cause it was the only thing that let me finish my work:

In my case, I was opening a doc created in 2010 and copy/pasting sections into a new 2013 document. I would then add new images to the 2013 file and edit the text. The new 2013 doc would bloat out of control.

Take the bloated file and "Save As" web page file type
Create a folder for the temporary .html file (Word will create a sub-folder for the images)
Close the file. (I would also re-start Word for good luck)
Open the temporary .html file in Word
"Save As" .docx in the same temporary folder (saving in the same folder may not be necessary, but I had some trouble with images sometimes if I didn't)

Once it's a happy .docx file again you can move it or save as to another folder. Doing this process would create a clean 2013 file a quarter or LESS the size of the original bloated size. And remarkably, it didn't disturb the layout very much at all.

It was maddening to see essentially the same content in two documents and have the one created in 2013 be 5-6x the file size of the one created in 2010! Definitely a 2013 issue.

You need to insert a section break at the point of renumbering, because numbers are sequential within sections.

Once you do that, select any footnote reference in the main text flow of the new section, select Insert > Footnote and change the "Numbering" dropdown menu to "Restart each section." Then, click on Apply.
