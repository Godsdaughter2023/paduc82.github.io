bob jones aka: cybertaz say:

This most likely is not a corrupt Word Preferences file. In newer releases of macOS, Scroll Bar behavior primarily is controlled by System Preferences settings. IMO it's one of the dumber moves Apple has made in recent years, but that's off-topic :-) The connecting point is that the usurpation of those UI functions by the OS is what disables the check boxes in the program.

First go to > System Preferences -General, then change your preference for how to Show scroll bars. FWIW, I prefer to use the Always option.

Regardless of those settings, though, each Scroll Bar is contextual - it will appear only if some portion of the document extends beyond that respective dimension of the window. Keep in mind these 3 points;

  • The Horizontal Scroll Bar only will display if the document is Zoomed to a width that doesn't fit within the document window, &
  • The Vertical Scroll Bar will not display for a 1 page document if the entire height of the page fits within the document window, &
  • Neither the Vertical nor the Horizontal Scroll Bar will display even in a multi-page document if the Zoom is set to fit all pages within the document window.
