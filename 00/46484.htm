If the M2 is not showing up in Bios, it may have failed...

If your machine is running Legacy Bios the M2 may not be functional at all...

That is something that varies a lot between M\B models and manufacturer's..

If you attach additional drives it can mess up your Boot Order to the point you need to change the settings in your Bios...

3. Close the dialogue boxes as it requested.
4. Open up the Properties again and click on Security.  Select your User from the list and click Edit.  From there select Full Control.

This should give you full control over deleting the original file for WMP and easily let you copy the other file into Programs (x86).

mouneshwar r say:

Hi Jodi B,

 

It depends on how Service Pack 3 was installed on your computer. If Service Pack 3 was installed through Windows Update or it was installed along with the Windows XP operating system (i.e. Windows XP disc integrated with Service Pack 3).

 

If you got Service Pack 3 installed through Windows Update, then you can uninstall Service Pack 3. Once you uninstall Service Pack 3 from your computer, it will be restored to pre-service pack files. In other words, the updates that were included with Service Pack 3 and post Service Pack 3 will be uninstalled from the computer. Once you apply the update SP37394.exe, you’ll have to re-install Service Pack 3 and other updates post Service Pack 3 installation.

 

How to remove Windows XP Service Pack 3 from your computer
