Sorry, Gord. I had in mind only an alternative answer to your question. Petr

We shouldn’t forget the case of intermittent highlighting. The example macro would toggle the pertinent row shade:

Option Explicit

Sub HighlightRowIfXEq0()

Dim I As Long, C As Range

Const LightGrey As Long = 15, Colorless As Long = -4142

I = 1

Do

  I = I + 1

  Set C = Cells(I, "X")

  If IsEmpty(C) Then Exit Do

  If C.Value = 0 Then

    With Rows(I).EntireRow.Interior

      If .ColorIndex = LightGrey Then

        .ColorIndex = Colorless

      ElseIf .ColorIndex = Colorless Then

        .ColorIndex = LightGrey

      End If

    End With

  End If

Loop

End Sub

macropod say:

The problem you're having is that Word interprets the hyphen as a minus sign, then tries to do math on the field. The following field code correctly formats 5-digit and 5+4-digit Zip Code data, even where the source data may have incorrect formatting such as a hyphen followed by trailing 0s after plain 5-digit Zip Codes (eg because a db only accepts Zip Codes in the 5+4-digit format) or is formatted as a 5-digit or 9-digit string without hyphens. A data field named ‘ZipCode’ is assumed.
{QUOTE
{SET Zip {MERGEFIELD ZipCode}}
{IF{Zip}> 99999 {SET Zip {Zip \# "00000'-'0000"}}}
"{=-{=-{Zip}-Zip}/2 \# 00000;;}{=-({Zip}*(-1)-Zip)/2 \# ;-0000;}"}

 

Note: The field brace pairs (ie '{ }') for the above example are created in the body of the document via Ctrl-F9 (Cmd-F9 on a Mac) - you can't simply type them or copy & paste them from this message.

As per the description provided, I understand that you want to know how to change default contact's folder in Outlook.

Do the following to change the contact folder order and see if it helps:

1) Open Outlook > click on Address Book > Tools> Options > click on Custom > now select the Contacts folder which contains the contact information and click on the Above arrow.

2) Click on the drop down arrow for the option When opening the address book, show this address list first > select the Contacts folder with data >OK.

3) Now close Address book, restart Outlook and go to contacts and verify the result.
