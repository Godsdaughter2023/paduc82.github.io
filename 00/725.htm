macropod say:
Linked Excel objects cannot span page boundaries in Word. If you need to display more rows than will fit on a single page, you will need to use a different link format or use separate links (and ranges) for the links required for each page.

john korchok say:

You can expand the size of the named range in Excel, then Word will add it to the section that is displayed:

  1. With your Excel source open, click on FORMULAS>Name Manager.
  2. Select the named range to be modified, then click on the Edit button.
  3. In the Refers to field, change the last number to include the new rows. OK out. The additional rows are added to the Word display.

If you are using a conventional numbered range instead of a named range, you can edit the Word field that displays the Excel data to add rows:

  1. Select the entire range in Word.
  2. Press Shift + F9. The table changes to a field code something like this:
    { LINK Excel.Sheet.12 "Book1" "Sheet1!R1C1:R2C7" \a \f 4 \r }
  3. To add one row to this, you would change R2C7 to R3C7.
  4. Right-click on the field and choose Update Field.
  5. Press Shift + F9 to return to displaying the field as a table. It now has one additional row.

which typically happens when the range expands beyond the bottom of the page in Word.
