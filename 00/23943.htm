graham mayor say:
The code
If InStr(1, LCase(.Subject), "pla ") > 0 Then
converts the subject text to lower case before testing for pla, so it will find pla and PLA.

If you want a more belt and braces approach then

Sub MovePLA(Item As Outlook.MailItem)
Dim vSubject As Variant
Dim i As Long
    With Item
        If InStr(1, .Subject, Chr(32)) > 0 Then 'Check for more than one word in the subject
            vSubject = Split(.Subject, Chr(32)) 'split the subject at the space(s)
            For i = 0 To UBound(vSubject) 'test each word
                If LCase(vSubject(i)) = "pla" Then 'if the lower case subject contains pla
                    Item.Move Session.GetDefaultFolder(olFolderInbox).folders("Foldername") 'move the message to the indicated folder
                    Exit For 'and stop looking
                End If
            Next i
        Else 'the subject has only one word
            If LCase(.Subject) = "pla" Then 'if that word in lower case is pla
                Item.Move Session.GetDefaultFolder(olFolderInbox).folders("Foldername") 'move the message
            End If
        End If
    End With
End Sub

No space with "pla" this time will find PLA or pla as a whole word in the subject.
