Exchange accounts initiate a sync with the Exchange server once every minute. However, that sync can extend beyond a minute if Outlook has a large Inbox that is has to verify.

How large (how many messages) is your Inbox? I recommend keeping no more than a few hundred at most and have no sub-folders within it. The Inbox syncs more frequently than any other folder and a very large Inbox can cause sync issues.

Hi all, seeing this same problem with Outlook 2010 in an exchange 2007 environment. Unfortunately can only be done by disabling an add-in called Microsoft Exchange Add-in.

  1. On the File tab click Options.
  2. In the Outlook Options dialog box, click Add-Ins.
  3. In the Add-ins section of the Outlook Options dialog box, click Go.

To make this function work you in any of your workbooks you must create a module in your personal

This is the Function created to meet your purpose.

******************************************************************************

Public Function CountYellowCells(Trg As Range) As Long

Application.EnableEvents = False

On Error Resume Next

i = 0

For Each cell In Trg.Cells


If cell.Interior.Color = RGB(255, 255, 0) And cell.Font.Bold = True Then

i = i + 1

End If

Next cell

Application.ScreenUpdating = False


CountYellowCells = i

Application.EnableEvents = True


End Function
