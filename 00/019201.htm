Excel 'RANDBETWEEN' formula - Microsoft Community

rongittings say:

Excel 'RANDBETWEEN' formula

Using the 'RANDBETWEEN' formula what are the odds of the same number being selected twice? If the numbers were between 1 to 5,000, for example, would the formula select each number once before it selected a number for the second time?

v_agarwal say:
Its the algorithms used by system, which generates random numbers.
There may be chances that same number can be repeated.

joeu2004 say: Ron wrote:
Using the 'RANDBETWEEN' formula what are the odds of the same number being selected twice? If the numbers were between 1 to 5,000, for example, would the formula select each number once before it selected a number for the second time?

To answer the last question first, no, if you are using simply =RANDBETWEEN(1,5000).  But there are algorithms and methods for ensuring a random selection from 1 to 5000 without replacement.

As for the odds that RANDBETWEEN(1,5000) will repeat itself, I believe it is determined by the binomial distribution.  The odds depend on the number of samples; also on whether you mean "exactly" twice or "at least" twice; and also on whether you mean a particular number "twice" or any number "twice".

For a "particular number", I believe we can use the Excel BINOMDIST function.  Use =BINOMDIST(2,n,1/5000,0) for "exactly twice" in n samples.  Use =1-BINOMDIST(1,n,1/5000,1) for "at least twice".

For "any number" "at least twice", give http://en.wikipedia.org/wiki/Birthday_problem a read.  It is not as clear as I would like, but it might give you some insight into the problem.  I believe the answer is:  =1-PERMUT(5000,n)/5000^n.

Caveat:  The accuracy of that formula might depend on binary arithmetic limitations (TBD).

jeeped say:
Oddly enough, this printing capability seems to have been revoked since Excel 2007. While you could manually set the Print Area to exclude certain pages, you can also imitate the process with one of the macros below.

Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module (e.g. Alt+I,M). Paste the following into the new pane titled something like Book1 - Module1 (Code).

Sub Print_Odd_Pages()
    Dim lAllPages As Long, lStartAt As Long, iPage As Integer
    lAllPages = Application.ExecuteExcel4Macro("GET.DOCUMENT(50)")
    lStartAt = 1
    For iPage = lStartAt To lAllPages Step 2
        ActiveSheet.PrintOut From:=iPage, to:=iPage, Copies:=1, Collate:=True
    Next
End Sub

Sub Print_Even_Pages()
    Dim lAllPages As Long, lStartAt As Long, iPage As Integer
    lAllPages = Application.ExecuteExcel4Macro("GET.DOCUMENT(50)")
    lStartAt = 2
    For iPage = lStartAt To lAllPages Step 2
        ActiveSheet.PrintOut From:=iPage, to:=iPage, Copies:=1, Collate:=True
    Next
End Sub

Sub Print_Some_Pages()
    Dim v As Long, lAllPages As Long, vPages As Variant, iPage As Integer
    lAllPages = Application.ExecuteExcel4Macro("GET.DOCUMENT(50)")
    vPages = Array(1, 2, 3, 6, 7, 8, 12)  'change this to match your print job
    For v = LBound(vPages) To UBound(vPages)
        If vPages(v) <= lAllPages Then
            iPage = vPages(v)
            ActiveSheet.PrintOut From:=iPage, to:=iPage, Copies:=1, Collate:=True
        End If
    Next
End Sub

The first two are pretty self-explanatory from their titles. The last uses an array of numbers (3rd line) to print pages that follow no particular pattern. My example prints pages 1-3, 6-8 & 12. You would need to manually edit this for your own purposes. Tap Alt+Q to return to your worksheet then Alt+F8 to open the Macros dialog and Run the macro.

Sub ChoosePagesToPrint()
  Dim v As Long, lAllPages As Long, vPages() As Variant, iPage As Integer
  Dim userChoices As String

  userChoices = InputBox("Enter pages to print separated by commas: ", "Pages to Print", "")
  If userChoices = "" Then
    Exit Sub
  End If
  ReDim vPages(1 To 1)
 
  userChoices = userChoices & ","
  Do While Len(userChoices) > 1
    MsgBox Val(Left(userChoices, InStr(userChoices, ",") - 1))
    vPages(UBound(vPages)) = Left(userChoices, InStr(userChoices, ",") - 1)
    userChoices = Right(userChoices, Len(userChoices) - InStr(userChoices, ","))
    ReDim Preserve vPages(1 To UBound(vPages) + 1)
  Loop
  If UBound(vPages) > 1 Then
    'delete empty one
    ReDim Preserve vPages(1 To UBound(vPages) - 1)
  End If
   
  lAllPages = Application.ExecuteExcel4Macro("GET.DOCUMENT(50)")
  For v = LBound(vPages) To UBound(vPages)
    If Val(vPages(v)) > 0 And Val(vPages(v)) <= lAllPages Then
      iPage = Val(vPages(v))
      ActiveSheet.PrintOut From:=iPage, To:=iPage, Copies:=1, Collate:=True
    End If
  Next
End Sub
