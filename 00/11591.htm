jay freedman say:

You need two STYLEREF fields in the header, one to display the most recent Heading 1 paragraph (the chapter title) and the other to display the most recent Heading 2 style. If you want the words "Chapter" and "Section" and they aren't part of the paragraphs in the text, just type them before the corresponding fields.

If you click the Insert > Page Number > Top of Page command to insert a page number in the header, it will remove anything else you've already placed in the header. Instead, you can position the cursor (using a tab if needed) where the page number should go, and use Insert > Page Number > Current Position. If you just want a plain number, don't use the ribbon at all; just position the cursor and press Alt+Shift+P to insert a {PAGE} field.

jfread say:

Running Excel 2008 for Mac, I made a drop down list using the (Data, validation, allow "list")

Running Excel 2008 for Mac, I made a drop down list using the (Data, validation, allow "list").My problem is I gave my group of cells the wrong name when I dragged over them and now I can't rename them. When i try to type in the proper name it simply reverts to original. Thanks
bob jones aka: cybertaz say:
Unfortunately, renaming a range is a little more inconvenient than it should be :-)

  1. Go to Insert> Name> Define
  2. Select the range you want to rename
  3. Type the new name into the editable field (in place of the current name)
  4. Click the Add button
  5. Select & Delete the original in the list
  6. Click OK to close the dialog

Alternatively - and perhaps a bit faster - you can just create a new range using a different range name for the same group of cells, then go into the Define dialog & delete the original.
