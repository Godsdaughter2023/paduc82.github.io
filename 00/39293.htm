OK. Run regedit and navigate to  

 

HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows Live Mail\Dont Show Dialogs
 

Double-click on the value Show Block Sender Warning in the right-hand pane and change its value data to 0.
