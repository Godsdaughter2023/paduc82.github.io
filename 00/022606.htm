doug robbins - mvp office apps & services (word) say:

Access the Properties for the content control and give it a title.  Here, I used Hide as the title.

With the checkbox and the text selected, assign a bookmark to the selection - here I used RangetoHide

Then, in the ThisDocument object for the document in the Visual Basic Editor, insert the following code:

Private Sub Document_ContentControlOnExit(ByVal ContentControl As ContentControl, Cancel As Boolean)

Dim cc As ContentControl

For Each cc In ActiveDocument.ContentControls

    If cc.Title = "Hide" Then

        If cc.Checked = True Then

            ActiveDocument.Bookmarks("RangetoHide").Range.Font.Hidden = True

        Else

            ActiveDocument.Bookmarks("RangetoHide").Range.Font.Hidden = False

        End If

        Exit Sub

    End If

Next

End Sub

When you exit from the checked checkbox, the range of the bookmark is formatted as hidden (see the dotted line)

but, when it is unchecked there is no dotted line, indicating that the range is not formatted as hidden

Unless you were surfing the internet via an independent data connection (through 3G, Edge, or GPRS connection), you were using the ActiveSync internet passthrough connection to surf the web.  The ActiveSync internet passthrough connection actually uses the computer's internet connection.
