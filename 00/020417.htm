rich michaels - mvp (greatcirclelearning.com) say:

I'm going to guess that you updated your Operating System software and if so here's a list of basic maintenance actions to perform, which should clear up the problem. Do them in the order listed.

  1. Restart your Mac
  2. From the Applications/Utilities folder run the Disk Utilities.app program and Repair Permissions on your primary drive.
  3. Restart your Mac
  4. From the Applications folder run the Font Book.app program. Then from the Edit menu run the "Look for Enabled Duplications" function. Trash any duplicates it finds. Then from the File menu and with All Fonts Selected run the Validate Fonts function. Clean up all problems found. Use the Font Book Help feature to answer any questions about what you are doing.
  5. Empty the Trash and then "cold" start your computer, meaning a full power down and reboot.
  6. Hold down the Shift Key while starting PowerPoint and keep holding it down until the blank presentation appears. This is called a "safe" start and bypasses many normal functions.
  7. If during the safe start any of the files that keep opening appear or are recovered, close them. Keep closing them until all you have left on your display is the blank presentation document.
  8. Quit PowerPoint completely and restart your system.
  9. Now open PowerPoint as you normally do and see if everything is working correctly again.

Hope this helps

Office 365 2016 Outlook indexing resolved - Microsoft Community

edwineubank say:

Office 365 2016 Outlook indexing resolved

FYI, I had an Outlook indexing issue after upgrading to Office 365 2016. When searching the message appeared, Outlook is indexing your items, _________remaining to be indexed. After a day it became apparent that the indexing was not working, as the number remaining to be indexed did not change. With a 365 Tech's assistance, the following was the solution: the Windows indexing is an add-on for Outlook that may be turned on or off. Go to Ribbon, select File, Options, Add-ins, and at the bottom of the list will be "Windows Search Email Indexer". If off, turn it on to resume indexing.
