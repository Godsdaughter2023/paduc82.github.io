ethan b. say:
Hey Al,

Unbeknownst to me, Microsoft makes this difficult or arguably impossible to disable updates as a whole. The best you can do (and this has worked for me although it doesn't disable updates as a whole) is that on Windows 10 Pro, you can use the Local Group Policy Editor to make sure Windows 10 does not automatically download and install updates, and that it has to notify you first and ask you before it downloads.

To do this, hit Windows key + R together to go to Run, then without quotes, type in "gpedit.msc".

You should now be in the Group Policy Editor.

Navigate in the left pane to Local Computer Policy > Computer Configuration > Administrative Templates > Windows Components > Windows Update.

You should see a bunch of Windows Update related tasks. Find "Allow updates to be downloaded automatically over metered connections" and disable it. Now find "Configure Automatic Updates" and toggle it as "Enabled". Now under "Configure automatic updating", select "2 - Notify for download and auto install". This, as implied, does automatically install the updates but you can at least stop it in its tracks.

Remember that metered connections policy I told you to make changes to? Hit Windows key + I together, go into Network & Internet, then go into your applicable network connection (Wi-Fi or Ethernet) on the left pane, then on the right side, click your network icon (the one with the label "Connected". See under "Metered connection"? Toggle "Set as metered connection" on. If you're confused about the process, Microsoft has a support article here (helps especially if you're under Wi-Fi): https://support.microsoft.com/en-us/help/402845...

Unfortunately, there is no cinch way to disable Windows Updates entirely. Microsoft made this arguably impossible. Hope that at least clears it up.

Bug_Check 124 is a Fatal Hardware Error

This is usually heat related, defective hardware, memory or even processor though it is "possible"
that it is driver related (rare). Remove ALL power and reseat all cards, memory and cables (on
both ends) - actually remove and replace - do not just snug. Make sure the system has adequate
ventilation - add a small fan blowing into intakes. Carefully inspect the motherboard and other
cards for damage. Pay attention to the fatal hardware error implication.
 
BCCode: 124   0x00000124

The WHEA_UNCORRECTABLE_ERROR bug check has a value of 0x00000124. This bug check indicates
that a fatal hardware error has occurred. This bug check uses the error data that is provided by the
Windows Hardware Error Architecture (WHEA).

the troubleshooter as needed. In rare cases the antivirus/antispyware/security programs can
cause this error.

issues - also check in their forums.
