To check, copy & paste the following PowerShell code, which will return the number of saved views in the registry:

((gp "HKCU:\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\BagMRU").Nodeslots).count

If it's reached 5000, you should delete these two keys:

"HKCU:\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\BagMRU"

"HKCU:\SOFTWARE\Classes\Local Settings\Software\Microsoft\Windows\Shell\Bags"

This will delete the individual saved views for all folders but not the customized templates created by 'Apply to folders'.

If you're interested in more advanced customization of the templates used by library folders, I've been working on that with someone over on TenForums:

https://www.tenforums.com/general-support/129966-how-apply-view-settings-library-subfolders.html

I'd be happy to work with you if you want.
