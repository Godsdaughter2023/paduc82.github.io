Connect the recovery drive and boot from it.

 

When booting from the Windows 8 Recovery Drive (USB flash drive or an optical disc); the Windows logo is displayed on a black screen for few moments; in the background, the computer is actually booting into the Windows Recovery Environment (more simply called, Windows RE).
Windows RE is based on Windows PE (Windows Preinstallation Environment)

Once Windows RE is operating, a prompt screen will appear requesting the user to choose a keyboard layout; for instance, one might select US layout.However, should the desired layout not be visible; then, select, 
See more keyboard layouts.
(There are a total of 16 pages of keyboard layouts, alphabetically arranged.)

This next screen allows three important options:


 
Click the option Troubleshoot: This is the option to Refresh; Reset the computer; or, use the Advanced Tools on the Recovery Drive.
 

Click the option Reset the PC: This option will Reset the PC, and at the same time delete all files.
Windows RE will find and backup all data, settings, and apps; it will then install a fresh copy of Windows and restore all of the data, settings, and apps.
When the PC restarts; Login with the existing username and password, and it will find all the installed data.
However, any desktop applications that have been installed will not be saved or restored; the reasoning is that it is possible that a recently installed desktop application could be the cause of the instability.
Windows RE will create a list of those applications that were not saved or restored, so that a decision can be made whether to reinstall them.

C:\Windows\System32\LogiLDA.dll - Microsoft Community

munirkarkar say:

C:\Windows\System32\LogiLDA.dll

How can I fix C:\Windows\System32\LogiLDA.dll

davem121 say:
Hi Munir, this is caused by Logitech Download Assistant . . .

Open Control Panel - Programs and Features

If you find Logitech Download Assistant, uninstall it

Press Ctrl + Shift + Esc to open Task Manager
Click on the Startup Tab
See if Logitech Download Assistant, is listed there
If it is, select it and click 'Disable'
Close Task Manager

Click your Start Button, type msconfig and hit Enter
When the System Configuration dialog opens, click on the Services Tab
Check the box marked 'Hide Microsoft Services'
In the remaining list look for Logitech Download Assistant,, uncheck it if you find it
Click Apply and OK

Click your Start Button, type Regedit and hit Enter to open the Registry Editor
In the left Pane, navigate to:
HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run

Find Logitech Download Assistant, then right click and choose delete

Restart your System and this annoyance will be gone . . .
