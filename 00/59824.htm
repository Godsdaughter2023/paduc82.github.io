kasparww say:

If you are loading the game from original CD's onto W7 or W8.x system then it's likely an issue with SECDRV.sys being disabled in W8 and patched to disabled in W7 by a MS update for security reasons. For W10, games dependent on secdrv.sys service are a bit more complicated to run.

Open notepad and copy the below using the drive letters and folder path where you have the game installed (i.e. I have it installed on drive f: in folder Quake 4 but your installation path\folder may be different so change them where appropriate).

sc config secdrv start= demand
sc start secdrv
f:
cd f:\Quake 4\
Quake4.exe
sc stop secdrv
sc config secdrv start= disabled
exit

Save the file as Quake 4.bat in the Quake 4 folder where the .exe is. Right click on the Quake 4.bat file and create a shortcut to the desktop then rclick on the shortcut and select properties. Click Advanced on the Shortcut tab and select Run as Administrator.

When you click on the shortcut the game should now launch for you. The instructions in the Quake 4.bat file enables the secdrv.sys service and then starts it. After you play the game and exit it the secdrv.sys service will be stopped and then disabled to retain your system security.

Hope this helps.

P.S. This also works for any other older games you may have on CD that were issued with the secdrv security. I used it for Undying as well, just change the drive letter, folder and executable names as required.
