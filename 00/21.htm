Probably too late, but I'm posting this as a reference to other people who have this problem.

I faced the same issue with Excel 2013... After extensive troubleshooting, this is what fixed my problem:

1. Open the Registry Editor (Open Run, type regedit, hit enter!)

2. Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\15.0 and delete the whole Excel branch. 

3. Look for the User Settings in the same place, expand it and delete all keys that start with Excel

4. Navigate to HKEY_CURRENT_USER\Software\Microsoft\Office\15.0 and delete the whole Excel branch and repeat step 3 for this area.

5. Now open Excel, it should launch Windows Installer and re-configure excel.

chrizk say:

I replied to Nayana as her response has been found most helpful, but then realised that replies are not very obvious, so here is the information for everyone...

If you inadvertently delete something, you may not know 'what' you have deleted, only that know you hit delete by mistake.  The problem with the default view of Deleted Items, is that you cannot immediately see what you have just deleted, as the default is to view by the calendar event, date received etc.  You may find it useful to add the Modified attribute to the view.

in regard to deleted items, firstly select Deleted Items from the Mail Items view.  Right click on any column heading (FROM, To, SUBJECT...) and select Field Chooser from the drop down menu.  With the mouse button continually pressed down, 'drag' Modified to the location on the column, eg next to RECEIVED, then let go of the mouse button.  This should result in the field being 'dropped' into place.

NB You may need to change the drop down menu within Field Chooser to All Mail Fields or All Date/Time fields to locate the Modified field.

You can then select the column heading to reorder (sort) by the Modified date, easily identifying anything you have just deleted at the top (note, if you click twice, it will reverse the sort order {old to new}, click again to reverse again).

If you don't want Modified to always be displayed in this view, you can right click the Modified column heading and choose Remove this Column.
