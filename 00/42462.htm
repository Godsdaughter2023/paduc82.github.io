After reading through the thread, there appears to be some confusion regarding Wireless Networking technologies and configurations using Windows 7 and I would like to spend some time to try and clear things up with some basic troubleshooting / best practices for Wireless Network related issues.

1. First try and power off your modem, router and desktop/laptop. Then power on the modem, then the router, and then finally the desktop/laptop. Please do this in that order, allowing 60 seconds in between each device to enable the device to be fully booted and able to accept connections.

2. Update the drivers for your wireless network adapter via Windows Update, or by using the website for either the company you bought your PC/Laptop from or the manufacturer of your networking device.

3. Ensure that your router is set to broadcast it’s network name (SSID), ensure that you are connecting to the correct wireless network (SSID) and that you are using the correct WEP/WPA passkey on both devices.

a. Please use “Windows Connect Now” or “Wi-Fi Protected Setup” if available on your wireless router to ensure that the connection is setup correctly.

4. If you are trying to use WPA or WPA2, please ensure that both your network adapter and wireless router support this protocol. If you are having issues detecting your wireless network when using WPA or WPA2, try configuring your router to use WEP instead and see if you can connect.

a. Please note that the change between WEP/WPA/WPA2 will need to be made on both your computer and wireless router.

b. If you are only able to connect using WEP, then your networking hardware may need a firmware upgrade. If you are not comfortable doing this yourself, please call your hardware/router manufacturer for help in doing this.

5. If you are having a problem with your wireless connection after resuming from hibernate or sleep, or are having intermittent connectivity issues, try to adjust your Power Management profile to change your Wireless Adapter Settings to Maximum Performance.

a. To do this click “Change Plan Settings” next to your Power Profile, and choose “Wireless Adapter Settings” and set it to Maximum Performance. You should do this with all of your power profiles. Additional guidance is available here: http://windows.microsoft.com/en-us/windows7/products/features/power-management

6. Call the support line for your router manufacturer. It is likely that neither Windows nor the router are having problems, it is simply an issue of getting the configuration settings on both set up to work correctly together. The router manufacturer will be the best resource to know how to configure their device. Most of the contact numbers are listed here: http://support.microsoft.com/gp/vendors/en-us

Windows Media Player Keeps Changing My Artwork! - Microsoft Community

a. user say:

Windows Media Player Keeps Changing My Artwork!

I have a 2000 CD collection ripped in WMA lossless. After ripping a ripping session, I would spend considerable time cleaning it all up. You know editing artwork, album titles, track names, etc. Then WMP willy-nilly follows me around and screws up most of my changes.

You can uncheck "Retrieve Additional Information from the Internet" but WMP will still run around trying to screw up some of your artwork. If WMP can't find any info on an album online, or maybe, if you changed it manually, the metadata ends up getting set to all zeros, like this, AlbumArt_{00000000-0000-0000-0000-000000000000}_Large.jpg. These files are hidden in the album directory, BTW. Then WMP will take a random one of those from somewhere and copy it everywhere else with {00000000-0000-0000-0000-000000000000} metadata totally screwing up hours of work. This was really **** me off! There is no way to fix or stop it!

See more here for more here:  Link

Anyway, only solution, for me at least, was to go out and buy MediaMonkey. I pointed WMP to a complete different directory and never use it anymore. Never let WMP touch your library again or it will go around changing these things on you again. If you insist trying, make a good backup first. I suspect the W8 Music App is the same way since it appears to be tied into the WMP API like Media Center is. So don't let it touch your library either (without a backup) because it will probably do the same thing!

With MediaMonkey you can edit and change things to your heart's delight without it messing with you. Their is a learning curve to MediaMonkey and you have to make some setting changes before you do an "Auto-Organize". But once you get use to it you will never go back to WMP. Start small with MediaMonkey, like 10 CDs, make a backup, then experiment to learn everything and tweak various settings. I tweaked my MM settings to keep the WMP album format since that was what I liked best.

If Microsoft ever fixes this issue, or gives us an option to turn this "feature" off, then WMP would be a fine program that I wouldn't mind using! But as it is now, it is completely unusable for me. Only reason I revisited this is I was considering trying the W8 Music App and kind of forgot about the artwork issue with {00000000-0000-0000-0000-000000000000} metadata.

It takes me like all night to copy my library to a separate drive for a backup. I got better things to do than be a guinea pig copying it back and forth experimenting.

a. user say:

Hi Bill,

 

I would suggest you to refer to the following steps and delete the drivers and check.

a. Go to this site:sentinelcustomer.safenet-inc.com/sentineldownloads/

b. Download: "Sentinel HASP/LDK - Command Line Run-time Installer".

c. Unzip the files.

d. In an elevated command prompt (run cmd.exe as an administrator) navigate to the location where you unzipped haspdinst.exe.

e. Type the command haspdinst.exe –remove

f. When it finishes successfully, type the command haspdinst.exe –purge

g. When this finishes successfully, the HASP drivers should now be uninstalled. Re-boot and try the upgrade again.

 

Let us know about the status of the issue. We would be happy to help you further.

johnpersichilli say: Thank You! For the great work on helping the rest of us to get MS 8.1 installed. For those novices (myself included) you need how to type at the command line.

Once you have your command line you need to point to the folder where you downloaded and unzipped the Sentinel HASP/LDK. To do this you need to use the command CD which stands for Change Directory. and type the complete address to that folder (directory). So for example:

CD C:\John\Downloads\Removeit\Sentinel HASP\LDK

The address must be exact otherwise you will see an error message in the command prompt. That's o.k. as you can just keep trying until you get it right. I had to remember how to change the directory and use the command prompt to get to the correct folder (directory).

One important note: Remember the name of the folder (directory) where you downloaded the above software.
