susan_s. say:

Hello,

The Skype Name (username) of the account cannot be changed or edited once it is created. Also, creating a custom Skype Name is no longer available since Skype migrated to Microsoft last October 2016.

 

As the account creation process has been shortened, users will only have the option to create an account using either a phone number or an email address. These types of account still do have a Skype username as a unique identifier, however they are system-generated.

 

The username will show as live:firstpartofemail@ for email address-created accounts, while it will show as live:randomalphanumericcharacters for accounts created using a phone number.

 

What we can only modify is your display name in Skype. To change your display name in Skype: 

  1. Sign in to the Skype website.
  2. On the left pane, click on Edit profile.
  3. Edit your First Name and/or Last Name.
  4. Click on Save.

Note: Make sure to log out from the Skype app before changing your display name on the website. This is to sync the changes in your profile.

jllatham say:

The most likely cause of this that I can think of is that your Macro Security setting has been changed to either High or Very High.  Each application (Word, Excel, Access) has its own setting, so if this is the reason, you'll have to check both Excel and Access.  But it seems odd that both have become reset somehow at apparently the same time?

To check the setting in Excel, open the workbook and go to

Tools --> Macros --> Security and examine the settings on the [Security Level] tab.  If it is at High or Very High, try setting it to Medium (I strongly do not recommend Low!!) and OK out of the dialog.  Close the workbook. And close Excel.  The changed settings do not take effect until you close and reopen Excel.  Then open it up, open the workbook and see if your macros work again.  With things set to Medium, you should get a warning and be asked if you wish to Enable or Disable macros, naturally to use them you need to Enable them.

Just like with fight club, you don't talk about "going global" ;)

 

the right question to ask is how do I flash my phone.. and the answer will include all the information you need!

 

Here we go:

 

1) Download the flashing tool from Nokia's website:

http://tablets-dev.nokia.com/maemo-dev-env-downloads.php

If you're using windows you want to download and install

maemo_flasher-3.5_2.5.2.2.exe

 

2) Download the firmware from Nokia's official website:

http://tablets-dev.nokia.com/nokia_N900.php

 

2a) Download the EMMC firmware.

Download the latest EMMC firmware file:

RX-51_2009SE_10.2010.13-2.VANILLA_PR_EMMC_MR0_ARM.bin

When you've downloaded it, rename the file to "emmc"

 

2b) Download the operating system firmware

There are 5 versions available depending on your region. Not all regions are available (HongKong and Vodafone UK)

 

Download the correct file:

USA:

    RX-51_2009SE_10.2010.19-1.002_PR_COMBINED_002_ARM.bin

 

Middle East / North Africa:

   RX-51_2009SE_10.2010.19-1.003_PR_COMBINED_003_ARM.bin

 

India:

   RX-51_2009SE_10.2010.19-1.004_PR_COMBINED_004_ARM.bin


UK:

   RX-51_2009SE_10.2010.19-1.203.1_PR_COMBINED_203_ARM.bin


Rest of world excluding Hong Kong and Vodafone UK:

   RX-51_2009SE_10.2010.19-1_PR_COMBINED_MR0_ARM.bin

 

Once you have downloaded this file, rename it to "os"

 

3) Move the firmware

Run a cmd window (start, run, cmd) and run these two commands:

 

cd \program files\maemo\flasher-3.5

start .

 

A windows explorer window will pop up. Drag the os.bin and emmc.bin files you just downloaded into this new window. then close it

 

4) Flash the phone

THIS WILL COMPLETELY WIPE YOUR PHONE CLEAN!!!

BACKUP to a micro sd card first (or backup to the internal storage, then copy the "backups" folder off your phone and onto your PC)

 

Back in the cmd window run this command

 

flasher-3.5 -F os.bin -f

 

Flasher will sit there waiting for suitable device...

 

Turn your phone off

Press and hold the U button on the keyboard and keep it pressed

plug the phone into your PC via USB

keep the U button pressed until flasher wakes up and starts doing stuff on the PC

 

Wait for flasher to finish (it was say completed successfully)

 

Now, without touch your phone again, run this command in the same cmd window

 

flasher-3.5 -F emmc.bin -f -R

 

Flasher should start flashing your phone's emmc again straight away.. If that command also finishes correctly (again it will say command completed successfully) you can unplug your phone and if necessary, turn it on.. and you should be done
