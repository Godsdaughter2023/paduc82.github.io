tom ogilvy say:

Gordon's function counts all the comments on the sheet. 

If you need a count at the bottom of more than one column where it counts columns in that column then you could modify the UDF to count comments in the column containing the formula:

Function CommentCount() As Long
  Dim C As Comment
  Application.Volatile
  Set r = Application.Caller
  For Each C In r.Parent.Comments
    If C.Parent.Column = r.Column Then CommentCount = CommentCount + 1
  Next
End Function

usage is still  =CommentCount()
