Hi chagnontwins Let us know if you are getting any error messages related to DEP.  Data Execution Prevention is a set of hardware and software technologies that perform additional checks on memory to help protect against malicious code exploits.

Hardware-enforced DEP marks all memory locations in a process as non-executable unless the location explicitly contains executable code.
