2007 & 2010-2013 USE DIFFERENT QAT CONFIG FILES

2010 AND 2013

SAVE CUSTOM QAT TO FILE
Normally, the QAT configuration is saved in a configuration file. But, you also have the option of saving the QAT with a document. So a document can have it’s own custom QAT

In the Word Options > Customize the Quick Access Toolbar dialog, in a drop down you have the option of saving it to the default file or to the current file.
That means, you could even save it to the NORMAL.DOTM.

ossiemac say:

Copy the data in the Master worksheet that is to appear in one of the other sheets and then change to the other sheet and select the first cell of the destination and then PasteSpecial and click PasteLink button in the dialog box.

The new cross-reference is not formatted as a hyperlink, but if I hover over it, I see a popup  "Current document: CTRL+click to follow link."

That is normal.  If you uncheck the Insert as hyperlink box in the Cross-reference dialog, the Ctrl+click screen tip will not appear when you hover over the cross reference.

If you want a "real" hyperlink, then that is what you must insert, via the Insert>Hyperlink facility.
