Hi Hazel,

Thank you for posting your query on Microsoft Community.

In Windows Live Mail, to determine or change the email storage location you can follow the steps below:

  1. Sign into your Windows Live Mail.
  2. Press Alt+F once you're logged in.
  3. Click on Options and select Mail.
  4. Click on the tab Advanced .
  5. Under the Advanced tab click on Maintenance .
  6. And select Store Folder ,this will open a small dialog box that shows the path to the current mail store folder.
  7. Click on the Change button and you will now be presented with a Browse dialog box where you can select the new location for the mail store.
  8. Browse to an existing location that you wish to move your mail to or create a new folder. Once you are done, click on the OK button. You will now be shown the Store Location dialog box with the new path that you selected.
  9. To have Windows Live Mail use this new folder, click on the OK button. Once you click on the OK button, you will be shown a prompt that states you need to restart Windows Live Mail for these changes to go into effect. Click on the OK button in this prompt and then close Windows Live Mail.

Once you close Windows Live Mail, the program will automatically copy all your existing mail and mail folders to the new location. Once you close Windows Live Mail, the program will automatically copy all your existing mail and mail folders to the new location.

Note: You may have to make hidden files and folder viewable to be able to see the folders and files.

You'll need to upload the notebook to SkyDrive, then share that notebook from SkyDrive with others.

That's a long-standing bug. You can't get a full-page gradient or picture fill by using the page background. Instead:

  • Open the header.
  • Insert a rectangle shape from the Drawing toolbar. Drag its edges to the edges of the page (although both this method and the page background method will leave a white border all the way around the page, which is a limitation of most inkjet printers).
  • On the Drawing toolbar, click the Fill button and choose the same gradient fill.


The result looks the same in the document view and the Print Preview, but the rectangle fill doesn't form stripes or tiles when printed the way the page background does.

I have a spreadsheet containing 1500 columns. I would like to combine - Microsoft Community

hometraderinc say:

I have a spreadsheet containing 1500 columns. I would like to combine these into one single column. How?

I have a spreadsheet that contains 1500+ columns and I would like to combine them into one single column, without copy/paste.  Any suggestions?  excel or access is fine.
jeeped say:
Sub mcrCollectColumns()
    Dim i As Long, ws1 As Worksheet, ws2 As Worksheet
    Application.ScreenUpdating = False
    Set ws1 = ActiveSheet
    Sheets.Add after:=Sheets(Sheets.Count)
    Set ws2 = Sheets(Sheets.Count)
    ws2.Name = "All_Columns"
    With ws1
      For i = 1 To .Cells(2, Columns.Count).End(xlToLeft).Column
        .Cells(2, i).Resize(.Cells(Rows.Count, i).End(xlUp).Row - 1, 1).Copy _
          Destination:=ws2.Cells(Rows.Count, 1).End(xlUp).Offset(1, 0)
      Next i
    End With
    ws2.Rows(1).Delete
    Application.ScreenUpdating = True
    Set ws2 = Nothing
    Set ws1 = Nothing
End Sub

Sent items in outbox why? - Microsoft Community

sanford_25 say:

Sent items in outbox why?

When using Outlook Express my Sent items are now showing up in outbox but not in Sent items, why?
pa bear - ms mvp say:

Variation on Bruce's theme...

1a. Move any messages that you want to keep out of Sent Items and Deleted Items folders and into other local OE folders you've created for archiving (saving) such messages.

1b. Move 99% of your messages in your Inbox folder to other local OE folders, too.

2a. Write down the location of your identity's store (http://www.insideoe.com/files/store.htm#storemain).

2b. Enable 'Show Hidden Files and Folders' via Start | Control Panel | Folder Options | View (cf. http://www.bleepingcomputer.com/tutorials/tutorial62.html).

3. Close Outlook Express.

3b. Are you sure you closed Outlook Express?

4. In Windows Explorer, navigate to your store folder, find & delete the files Outbox.dbx, Sent Items.dbx, and Deleted Items.dbx.

====================

To avoid such problems in the future, adhere to the following caveats:

- Don't use Inbox or Sent Items folders to archive messages.  Move them to local folders created for this purpose.

- Empty Deleted Items folder daily.

- Frequently perform a manual compact of all OE folders while "working offline".  More at http://www.insideoe.com/files/maintain.htm

- Do not cancel Automatic Compacting, should it occur, and do not attempt to close OE via Task Manager or shutdown your machine if Automatic Compacting is taking place.

- Disable email scanning by your anti-virus application.  It can cause corruption (i.e., very problem you're having and/or loss of messages) and provides no additional protection:

    Why you don't need your anti-virus to scan your email 
    http://thundercloud.net/infoave/tutorials/email-scanning/index.htm
