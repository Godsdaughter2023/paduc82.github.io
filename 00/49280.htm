rohit siddegowda say:

Hi,

 

1.    Were there any changes or updates made on the computer?

2.    What is the complete and exact error message?

 

Method 1:

I would suggest you to restart your computer in clean boot mode and check if the issue persists.
How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7
http://support.microsoft.com/kb/929135

Also, see the section on how to return your computer to a Normal startup mode by following the steps under “Reset the computer to start as usual”.

 

Method 2:

I would suggest you to restart the print spooler service manually and check if it works.

To change or restart the Print Spooler service

You must be logged on as an administrator to perform these steps.

1.    Open Administrative Tools by clicking the Start button, clicking Control Panel, clicking System and Maintenance, and then clicking Administrative Tools.

2.    Double-click Services. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

3.    Right-click the Print Spooler service, and then click Properties.

4.    On the General tab, next to Startup type, make sure that Automatic is selected.

5.    If the service is not already running, under Service status, click Start, and then click OK.

Note: If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

 

Method 3:

If the issue persists then, you may try creating a new user profile and check if the issue persists.

To create a new profile, you must first create a new user account. When the account is created, a profile will also be created.

Refer: 

Create a user profile.
http://windows.microsoft.com/en-US/windows7/Create-a-user-account

 

Now check for the issue.
 
If the new user profile resolves the issue, you can fix the corrupt profile and re-install the applications and programs.
