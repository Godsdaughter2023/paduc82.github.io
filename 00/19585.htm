Count Excel cells with italic font - Microsoft Community

mikebadgley say:

Count Excel cells with italic font

I have a spreadsheet where several cells are italic to represent certain characteristics, then I manually count the italic cells and enter at the bottom for use in another calculation. Is there a way to automate the counting of the italic cells in a range?
chip pearson say:

You need to use a VBA function.

Function CountItalics(RR As Range) As Long
    Dim R As Range
    Dim L As Long
    For Each R In RR.Cells
        If R.Font.Italic = True Then
            L = L + 1
        End If
    Next R
    CountItalics = L
End Function

Then, call this from a worksheet cell with something like

=CountItalics(A1:A10)

This will return the number of cells in range A1:A10 that have an italic font.

 

 

bernd pl say: Hello,

A user defined function (macro) can help you. Press ALT + F11, insert a new module and enter (copy & paste):

Function CountItalics(r As Range) As Long
    Dim rI As Range
    For Each rI In r
        CountItalics = CountItalics - rI.Font.Italic
    Next rI
End Function
