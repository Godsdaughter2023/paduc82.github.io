  14. Create the footer as you like
  15. Click Ok to close the Page Setup window when you're done
  16. Press ESC to close the Print Preview
  17. Select Book2, this will un-group all workbooks!

bernd p. say:

Short answer: NO

Longer answer:

A local OneNote notebook is implemented as a folder (Windows file system) with the notebook's name. This folder usually contains one or more .one files, the so called sections.

If you store other files in this folder, OneNote will ignore them. So far so good.

The problems start when the folder has subfolders. OneNote will interpret them as so called section groups (not surprisingly used for grouping some more sections) and show them in its GUI. If these subfolders contain other files they will be ignored. A superfluous .onetoc2 file will be created, down the whole subtree with sub-subfolders ....

Maybe there are more side effects.

So I suggest not to mix your notebooks with your existing tree.

Bernd
