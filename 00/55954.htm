¡firedog say:

Messages in the deleted file are removed very quickly.

Windows Live Mail doesn't remove messages from the Deleted items* folder. If the account concerned is one that synchronizes with the server, then there may be settings or policies there that determine how long to keep deleted messages. Synchronizing accounts are set up to use either IMAP or HTTP**, not POP3. For example, messages are kept in the Deleted folder at Outlook.com for five days. They should still available to be recovered for another five days after that. However, when a message has been removed from the Deleted folder, it will also be removed from the Deleted items folder in Windows Live Mail at the next sync.

If you want to remove messages from the inbox but keep them for longer, don't delete them; move them to an archive folder instead.

Repairing the installation will have no effect on this.

*   The folder may have a different name if it's an IMAP account.

**  If you're not sure what type of account you have, right-click on an account name in the folder pane and select Properties. The type of account is given at the top of the Server(s) tab. The properties dialogue for an HTTP account has three tabs, for a POP3 account four and for an IMAP4 account five.

andre for directly say:
Open Start, type: CMD
Right click CMD
Click Run as administrator

At the command prompt, type:

netsh int ip reset

Hit Enter
Exit the prompt then restart

Open Start > Settings > Update & security > Troubleshoot
Scroll down
Click Network adapters
Click Run the Troubleshooter

When complete, restart to see if the problem is resolved.

If that does not work

Press Windows key + X
Click Device Manager
Expand Network Adapters
Select the Network card listed,
Select “Update Driver”
This should give you a choice of drivers - select the oldest one
Wait a few minutes while it installs the old driver

or

press Windows key + X
Click Device Manager
Expand Network adapters
Right click your adapter
Click Uninstall


Exit Device Manager, restart. Windows 10 should detect the network adapter then reinstall it. Check if you can connect and browse.

also try:

Open Start > Settings > Network & Internet > Status


Scroll to the bottom then click Network reset.

Click Start > Settings > Network & Internet > Wi-Fi > Manage known networks > select the Cloud wireless network
Click Forget

sumit (independent advisor) say:
Activation depends on what type of license you have of Windows 7/8/10.

If Windows 7/8/10 came preinstalled with the computer, that is called an OEM license. For an OEM license, the license dies as soon as Motherboard dies. So essentially it is non-transferable. If that is the case you need to get a new license.

However, this is not same for retail licenses i.e. you purchased the license through Microsoft store. That does carry transfer rights. If that is the case, please see the below steps:
(Not applicable to you)
What you need to do is to grab the key using any Software like Showkey Plus, uninstall that key from older PC and use it on another computer.

Download Showkey Plus:
www.thewindowsclub.com/showkeyplus-product-key-finder

Uninstall the key
https://winaero.com/blog/how-to-deactivate-wind...

Use the key by Showkey Plus on the newer computer.
(Not applicable to you ends)
Alternatively, there are two more options available other than this(one if you have already run activation troubleshooter)

Contact Microsoft using the phone numbers below or use the chat support link(weekdays only)
https://support.microsoft.com/en-in/help/405170...

https://partner.support.services.microsoft.com/...

Use the Activation troubleshooter on the new PC signed in with same MS account.
Go to Settings>Update and security>Activation and run the troubleshooter.
Select I recently changed hardware on this device.
Please let us know if you need any assistance.
