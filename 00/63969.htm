lukesnyder say:

Here's what worked for me and probably what you need to do:

Open an elevated command prompt (press windows key, type cmd right click the application and run as administrator)

Each of these commands will take some time, but feel free to use your system as normal while they are running.

run the command sfc/ scannow

regardless of what corruption is detected/ undetected, fixed/ not fixed-- make sure youre connected to the internet and immediately run the following commands one after the other:

Dism /Online /Cleanup-Image /ScanHealth
Dism /Online /Cleanup-Image /CheckHealth

You may retest the original issue at this point but I recommend moving straight to the final step; insert your windows installation media. Don't run anything from the media, but make sure it is accessible and run the online image repair command replacing everything after 'Source:' with the location of your media. If you're using a disk, for example, and the drive is labeled 'D:' then replace 'C: \test' etc. with 'D:' and run the command:

Dism /Image:C:\offline /Cleanup-Image /RestoreHealth /Source:c:\test\mount\windows

If you're using a disk in drive D: the command will look like this:

Dism /Image:C:\offline /Cleanup-Image /RestoreHealth /Source:D:

After repairing your image, run the automatic troubleshooter for windows update again and check for updates.

After performing these steps I was able to install all updates including the ones which failed previously. I really hope this helps and sincerely apologize if it does not.

I'm not a software engineer, IT professional, tech support specialist or a Microsoft employee, but this is what worked for me after a lot of troubleshooting.

You may refer to this online resource for more info on repairing a windows image: http://technet.microsoft.com/en-us/library/hh824869.aspx
