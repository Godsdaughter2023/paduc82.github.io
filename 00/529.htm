doug robbins - mvp office apps & services (word) say:

You should use the Sort facility to sort the Master sheet by the Sales rep, then the Class 1 Day and then the Class 1 Time and then the Class 2 Day and then the Class 2 Time

Then you could select the block for each Sales rep and copy it to the appropriate sheet, or, it would be possible to create a macro to do that.

Assuming that the Sales Rep names are in the first column, the following code should do it

Dim strRep As String
Dim shtRep As Worksheet
Dim i As Long, j As Long, n As Long
With Sheets("Master").Range("A1")
    strRep = .Offset(1, 0)
    Set Repsheet = Sheets.Add
    Repsheet.Name = strRep
    For j = 0 To .CurrentRegion.Columns.Count - 1
        Repsheet.Range("A1").Offset(0, j) = .Offset(0, j)
    Next j
    n = 1
    For i = 1 To .CurrentRegion.Rows.Count - 1
ResumeHere:
        For j = 0 To .CurrentRegion.Columns.Count - 1
            Repsheet.Range("A1").Offset(n, j) = .Offset(i, j)
        Next j
        If .Offset(i + 1, 0) = strRep Then
            n = n + 1
        Else
            strRep = .Offset(i + 1, 0)
            GoTo NewSheet
        End If
    Next i
NewSheet:
    Set Repsheet = Sheets.Add
    Repsheet.Name = strRep
    For j = 0 To .CurrentRegion.Columns.Count - 1
        Repsheet.Range("A1").Offset(0, j) = .Offset(0, j)
    Next j
    n = 1
    GoTo ResumeHere
End With
