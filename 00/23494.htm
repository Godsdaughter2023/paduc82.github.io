suzanne s. barnhill say:

You can't frame the picture and caption together unless they are both In Line with Text. What Stefan was getting at was that if the picture is wrapped (has a wrapping style other than In Line with Text), the caption will automatically be placed in a text box (also wrapped), and you can't have a footnote in the wrapped text box. But if you change the wrapping of the picture to In Line with Text, then the caption will be inserted as a normal text paragraph, and you can select the two and put them in a frame. Although you can wrap text around a frame, the text inside is considered "inline," so you can insert a footnote. But unless you need to wrap text around the picture and caption, it is better to just leave them inline.

If you do need to use a frame, there are two ways to insert it. If you have the Developer tab displayed (Office Button | Word Options | Popular: Show Developer tab in the Ribbon), the Insert Frame command is found in the Legacy Forms palette that you can open from the Legacy Tools button in the Controls group. If you find you use frames a lot, you'll want to add it the button to the Quick Access Toolbar. Look for the Insert Horizontal Frame command under All Commands in the Customize the Quick Access Toolbar dialog.
