rob koch say:

Typically what Malwarebytes Anti-Malware is finding are items called Potentially Unwanted Software or PUPs.  As the following Malwarebytes Labs Blog article discusses, they had decided a few years ago (in 2013) to increase how aggressive they were at detecting and removing these.

Malwarebytes gets tougher on PUPs - Malwarebytes Labs

This is one of the more obvious differences between the criteria Malwarebytes and Microsoft security applications use when detecting malware.  Here's the equivalent criteria that Microsoft uses regarding detection and removal for comparison.

Evaluation criteria for malware and unwanted software - Windows Defender Security Intelligence

So though you are correct that Microsoft detects less items than Malwarebytes Anti-Malware, that's actually a conscious decision, in part due to the fact that Microsoft's security products are the default security software included with Windows 8.x and 10.  For this reason Microsoft must inherently be more conservative in their approach in order to avoid detecting items that some of their customers might prefer to use, regardless of whether these programs may not be desirable to a few of their other customers.

This is where most such confusion between different security products occurs today, since differences in detection criteria will inherently cause certain items that aren't detected by one product based on criteria to be found by others using a different set of criteria.

As someone who is very strict in my choices of software to be installed on my systems, I've almost never had anything that's been detected by Malwarebytes AM, since I so rarely install non-major name software that I'm unlikely to receive any of the PUPs that some other consumers who are less discerning in their choices find they've installed, often unintentionally as tag-along software attached to supposedly free offerings.

Rob

The security community is in a constant state of change as new infections appear and it takes time for new malware to be reported, samples collected, analyzed, and tested by anti-virus/anti-malware researchers before they can add a new threat to database definitions. Malware writers have the advantage since no matter how hard security vendors attempt to stay on top of new threats, there is always a short time-frame in which a new malicious file goes undetected and can infect a computer without detection. Why not use OptionButtons instead of CheckBoxes? They function the way you describe automatically.
