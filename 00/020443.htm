I wrote the If Then part above. What's left is to set the variable moveMe before you run the If Then part. moveMe is going to look at the Excel file and be set to whatever is in the cell in question. It should go something like this:

Sub MoveIfNecessary()
    Dim oXLApp As Object
    Dim oWb As Object
    Dim row As Long
    
    Set oXLApp = CreateObject("Excel.Application")
    'On a Mac change \ to : in the following line
    Set oWb = oXLApp.Workbooks.Open(ActivePresentation.Path & "\" & "YourFile.xlsx")
    moveMe = oWb.Worksheets(1).Range("A1") 'Change A1 to the cell with the goods
    If moveMe = 1 Then
        ActivePresentation.SlideshowWindow.View.Slide.Shapes("Apples").Top = 100
        ActivePresentation.SlideshowWindow.View.Slide.Shapes("Apples").Left= 200
    End If
End Sub
