Wizard for content control - Microsoft Community

zxrichie say:

Wizard for content control

Hi, i am looking at revamping my Templates written in VBA by using content controls for the information, these content controls will replace the bookmarks which were getting deleted in a document by the user, Now, i've looked at CC and they seem to fit my criteria, however, is there a way to have a "wizard" pop up when the document is created asking for some specific information that will initially fill out the Content Control boxes.

Regards

Richard

beth melton say:

If you previously used a UserForm (VBA form) then that's still your 'wizard'. It will work similar to before but to populate your content controls you'll need to loop through the collection of content controls, check the Tag property, and insert the content accordingly. Here's a code snippet: (If you need more help with the below let us know. For example I didn't add the variable declarations. )

      For Each cc In fDoc.Range.ContentControls 
       With frm 
        Select Case cc.Tag
         Case "Office"
            cc.Range.Text = .txtOffice
         Case "Address1"
            cc.Range.Text = .txtAddress1
         Case "Phone"
            cc.Range.Text = .txtPhone
         End Select
       End With
     Next cc

However, depending on what you need, you can use the placeholder text for the content control and use it to prompt the user for information. If you need to protect all content, other than the content controls, insert the content controls where needed, and select the document, insert a rich text content control, and then in the properties of the newly inserted rich text content control, set "Content control cannot be edited". That way the user can only fill in the content controls and not change anything else.

Note that the user can change the properties and edit the document if they want. I use this method for preventing accidental changes.

~Beth Melton Microsoft Office MVP

How to Format Nokia N72, N73, N81, N95, NSeries Mobile Phones.

Method 1: Three Finger Reset for Nokia Phone

1. Switch off your Nokia phone.

2. Hold down the following three keys, Green (call answer button) * Key, and Number ‘3? key.

3. While holding these buttons, press the power button and switch on the phone.

4. Wait for message ‘Formatting’ to appear on the screen and release all keys once you see that message.

Method 2: Hard Format of Nokia Mobile Phone

1. Switch off your Nokia phone

2. Now Press following key sequence *#7370#

3. Enter Default lock code is 12345.

4. While holding these buttons, press the power button and switch on the phone.

5. Wait for message ‘Formatting’ to appear on the screen and release all keys once you see that message.

Method 3: Soft Reset of Nokia Mobile Phone

1. Switch off your Nokia phone

2. Now Press following key sequence *#7780#

3. Enter Default lock code is 12345.

4. While holding these buttons, press the power button and switch on the phone.

5. Wait for message ‘Formatting’ to appear on the screen and release all keys once you see that message.

Please Note: The Above Nokia Phone Formatting method will erase all your Phone data. Sodo backup before proceeding for Formatting
