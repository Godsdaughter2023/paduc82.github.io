bob jones aka: cybertaz say:

Well, there is no "Circle" shape. There is an Oval [which looks like a filled circle] & a Donut [which looks like a hollow ring]. From your description I have the impression you're referring to the Donut.

When you select the shape, note the yellow diamond that appears on the left inner edge of the band... Drag the diamond left/right to increase or decrease the width of the band. There is no value box into which you can enter a number for that property.

Also, that shape actually consists of 2 lines (inner & outer) which cannot be formatted individually, with a Fill sandwiched between them. IOW, the Donut is the shape, so to change its color you need to select the Fill Color as well as the Line Color... As with most other shapes, Fill & Line are separate properties.

Depending on the specific effect you're after, it might be better to use the Oval tool... either click to create the shape or hold shift while dragging in order to maintain the symmetry. (After-the-fact, hold shift while dragging a corner handle.) Then choose No Fill as the Fill Color & set the Line properties to what you prefer.

Find and Replace across multiple text boxes in a single Word file - Microsoft Community

layeredtext say:

Find and Replace across multiple text boxes in a single Word file

Can one Find and Replace across multiple text boxes in a single Word file?
john mcghie say:

Arshad:

I think that wins the prize for this week's least informative answer.  Of course it SHOULD work.  Now: what are the circumstances under which it DOESN'T?


Layered Text:

You need to know that text boxes, despite their name, are actually GRAPHICS objects.  They are little "drawings" that can also contain text.

A Word document is like a string of sausages: the text is in one of the sausages and various other things are in other sausages.  Such as graphics: they are all at the end of the string, in one of the last sausages.  The position in which an object appears in the document bears no relationship to where it is stored.

Normally this doesn't matter in a Find Replace, because normally you do the whole document.  But if you are finding within a selection, and that selection does not contain the text boxes, or some of them, then the text boxes will not be searched.

As a rule of thumb, try to make sure the document has no selection.  If for some reason you need the selection (to search in a constrained area) then it is likely the text boxes will not be searched.

Hope this helps

scottgem say:

I have two problems:

You probably have more than those two problems. 

First: "I am trying to copy one or more rows from excel to access"

This indicates a structural problem. Access is not a spreadsheet and should not be used as one. A table of data that works fine in Excel is generally unsuitable for use in a relational database. So if you can copy and paste from Excel to Access, the odds are strong that your Access tables are not properly designed.

Rather than trying to fix the problem you think you have, you should be working on fixing the problem in your design. 

To copy data from Excel to Access, you should be linking to your Excel table, then running a series of Append queries that will take the Excel data and place that data into normalized tables within your Access database.

diane poremsky mvp (slipstick.com) say:

A profile is a collection of all of your accounts, a pst is a data file for all of your email. You can have more than one of each. :)  A new pst is just a new data file, it does not create a new profile. 

If you have two accounts in your profile and add a pst file for each POP3 account then set the POP3 accounts to use their own pst (in File, Account Settings, select account, click Change Folder),the mail will be kept separate. Exchange server, Outlook.com, and IMAP accounts always using their own data files. 

palcouk say:

A little typo; adding a new data file does not create a new/seperate Profile

The new data file will appear as you currently see in the folder tree view pane.

If you wish to create a new Profile, with Outlook closed use the mail applet in the control panel to add a new profile and manually add your mail accoun(s)

You can set this new Profile as the default or 'prompt for Profile when you start OL

NB You cannot switch Profiles within Outlook, you have to close Outlook, then restart to select the required Profile.

You can use two data files within Outlook and if you have two mail accounts, in the account settings designate the data file to be used by the individual mail account.
