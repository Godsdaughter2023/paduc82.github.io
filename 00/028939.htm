ossiemac say:

You can try the following.

  1. If the workbook is open then Save and close it.
  2. Close Excel.
  3. Backup your workbook as follows.
  4. Create a new folder in windows explorer
  5. While still in windows explorer, select the existing workbook and copy it.
  6. Change to the new folder and paste it.
  7. Using the above procedure, the workbook name does not change and if there are any links etc then they will not change. Do not open the backup workbook because we do not want it to attempt to update links etc.
  8. Now open Excel.
  9. Open the original workbook.
  10. Open the VBA editor. (Alt and F11 opens the VBA editor.)
  11. If you can see VBA code in the large area on the right then go to step 17 
  12. If can't see VBA code, look in the Project Explorer (left column) and find Userform1.
  13. If you can't see Userform1 but can see Forms then click on the plus sign beside it to expand it and you should see Userform1.
  14. If you can't expand the names in Project explorer because it asks for a password then unless you know the password, you cannot do any more so refer back to the developer of the project.
  15. Right click on Userform1.
  16. Select View code and you should now see the VBA code.
  17. If you can see the VBA code then select menu item Edit -> Replace...
  18. Enter "Sheet1" (including the double quotes) in the Find What: field.
  19. Enter "Menu" (including the double quotes) in the Replace With: field.
  20. In the bottom left corner select the optio (Important)
  21. Click Replace All.
  22. Close the VBA editor. (Cross with red background top right)
  23. If you have not already changed name of Sheet1 to Menu then do so.
  24. Save the workbook and then test.

You can repeat the above to change the name of any other worksheets in the workbook. If when doing this it tells you it did not find a worksheet then do not worry because that worksheet might not be named in the code and it will not affect it.

If the above does not work or causes problems then in windows explorer, you can delete the workbook and then copy the backup from the new folder back into the folder where you usually have the workbook.

If you can't get it to work then I doubt that we can help without a copy of the workbook and information on what the existing worksheet names are to be changed to.

Guidelines to post a workbook on OneDrive

  1. Zip your workbooks. I prefer that you do not just save to OneDrive. (To Zip a file, in Windows Explorer  Right click on the selected file and select Send to -> Compressed (zipped) folder.)
  2. Go to this link.  https://onedrive.live.com
  3. Use the same login Id and Password that you use for this forum.
  4. Select Upload in the blue bar across the top and browse to the zipped folder to be uploaded and select Open (or just double click). (Be patient and give it time to display the file after initially seeing the popup indicating it is done.)
  5. Right click the file on OneDrive and select Share.
  6. Do NOT fill in the form; "Select Get a Link" on the left side.
  7. Click the button "Create a Link"
  8. Click in the box where the link is created and it will highlight.
  9. Copy the link and paste into your reply on this forum.
