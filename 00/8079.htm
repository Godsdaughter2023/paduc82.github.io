line break - Microsoft Community

rich b87 say:

line break

I have data that is exported from a PLM software into Excel.  Some of the cells uses the Line Break feature. Is there some code that I can use in a macro to move the data to another cell?
bernie deitrick say:

Try

Sub Test()

    Range("D:D").TextToColumns Destination:=Range("G:G"), DataType:=xlDelimited, Other:=True, OtherChar:="" & Chr(10) & ""

End Sub

Note that this will leave the values in column D - if you want to remove that column, use

Sub Test2()
    Range("D:D").TextToColumns Destination:=Range("G:G"), DataType:=xlDelimited, Other:=True, OtherChar:="" & Chr(10) & ""
    Range("D:D").Delete
End Sub
