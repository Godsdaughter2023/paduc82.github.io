ron sommer mvp 2009-2017 say:

Windows Contacts (Start, User, Contacts) are not used by WLMail.

Open WLMail, click on the Contacts icon (it looks like an address book) at the bottom of the folder list.
In WLMail, Ctrl + 3 will also open Contacts.
Click on Category.  Enter a name and select the contacts.

Note: contacts in a Category must be listed in all Contacts.
A group (which cannot be used by WLMail) can have names that were only listed in the group.

When you enter a Category name in an address field, a plus sign appears in front of the Category name.
Clicking on the plus sign makes all of the Category names appear in the address field.
You can highlight and delete names.  Only the remaining addresses will receive the email.  This does not remove the names from the Category.

Your computer is acting as if it doesn't have a wireless network adapter.

Although when you restored the computer to its factory state all of the device drivers should have been installed, perhaps something went wrong. 

Also, of course, when you restored the computer, you reinstalled all of the bloatware that HP originally included with the computer and removed all of the Windows updates.  You'll need to visit Windows Update (as soon as you get the network connection issue fixed).

If any free trial software was included, even though the software was reinstalled by the restore process, the free trial will have expired.  If any of this trial software was an antivirus product (e.g., Norton, McAfee), you should remove it using the appropriate removal tool from the antivirus software's publisher.  For convenience, a list of these tools, with download links, is available here --> http://support.eset.com/kb146/?locale=en_US

Take a look at Device Manager again.  Is there a category for "Other devices"?  For example:

If there is nothing in Device Manager, you can also take a look using the System Information utility, but I don't think the results will be different.

I assume that you're using a different computer to post your questions/replies here, so please do the following:

  • Click the Start Orb, type msinfo32.exe in the Start box, and press Enter
  • Once the System Information window has populated, click Edit > Select All, then click Edit > Copy
  • Open Notepad and Paste the data from System Information into Notepad
  • Back in the System Information window, expand "Components" by clicking the + to its left, then expand "Network" and then select "Adapter
  • Click Edit > Select All, then click Edit > Copy
  • Paste the data into Notepad below the previous data
  • Back in the System Information window, click "Problem Devices"
  • Click Edit > Select All, then click Edit > Copy
  • Paste the data into Notepad below the previous data
  • Save the Notepad document; copy it to a USB flash drive; bring the flash drive to the computer you're using to post here, and copy/paste the information from the Notepad file into your Reply

If your wireless network adapter is not showing up in Device Manager as an "Other device" and is not showing up in System Information, I can think of only two reasons why this might happen.

  1. If your wireless network adapter is integrated into the motherboard, there may be a BIOS setting that needs to be changed to enable the adapter
  2. If your wireless network adapter is separate circuit board (fairly common), it may have become loose enough in its socket to be disabled; you would have to open up the computer's case and re-seat the board.
