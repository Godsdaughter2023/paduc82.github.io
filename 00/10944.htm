daniel g. grau say:
The backslash is not available in the Spanish ISO layout, but it is in the plain Spanish layout. The other characters you mention are available in both layouts, but in different positions. Do this:

Add the Keyboard Viewer to the main menu
  • Go to System Preferences
  • In the first row, select Language & Text
  • Click on the Input Sources tab at the top
  • Enable Show input menu in menu bar, at the very bottom

This will add a new menu next to the clock, at the top right of the menu bar. Click on it and select Show Keyboard Viewer. By pressing the various modifier keys (Shift, Option, Option + Shift) you will see all available characters. Keys marked in orange indicate key that you need to press and follow by a second key, such as the accented vowels.



There is a setting in Word that will apply the matching language as you switch keyboards (Word > Preferences > Edit > Match font with keyboard). This is not as convenient as is sounds, as one would not change keyboards to switch between English and Spanish.
