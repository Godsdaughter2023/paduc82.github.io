rob koch say:

As CassofShadows correctly guessed, Steve is referring to Widows XP itself, since it has received almost no security update support since that ended in early April 2014.  This lack of critical security updates means that no security program can truly protect Windows XP any longer and those that claim to are really providing a disservice by stating they can.

The problem is that though an antivirus only provides protection from malware which are known and so recognized and blocked, true vulnerability protection is provided by the operating system and security updates that patch any newly discovered vulnerabilities (programming flaws) found during the support lifecycle of the operating system.  Once those critical security updates cease to be provided, the first few vulnerabilities which are discovered in other versions of the same operating system can be used by attackers to produce new malware which exploit them, with many of these new malware working on the older os version as well due to similar or identical flaws.

There are also differences between each operating system version which improve its inherent security, especially in a case such as Windows XP which hadn't received such significant improvements since Service Pack 2 in August 2004.  With the extension of security update support by Microsoft to nearly 10 years beyond that point, the true security posture of Windows XP had badly decayed well before the end of support in April 2014, resulting in most knowledgeable users having upgraded or replaced it long before that time, in my own case over 3 years previous.

At this point using Windows XP is risky, especially when accessing the Internet since many malware exist against which it provides no vulnerability protection.  So a Windows XP system shouldn't be used to browse any websites which aren't completely trusted and even then will likely become infected if one of these sites becomes compromised itself.

It's extremely rare that antivirus or even most antimalware applications provide additional protection against vulnerabilities, since that's the operating system's job, so the risk of infection increases exponentially as new vulnerabilities are discovered.  We know that at least several and possibly dozens of such new vulnerabilities now exist within Windows XP, so declaring that any security program truly provides protection at this point is often inaccurate and thus you shouldn't assume there is any.

If you must use XP, stay offline whenever possible and only browse the fewest possible trusted websites when it's an absolute must.  If possible, move any critical applications which contain either personal data or other non-public information to a different PC with a supported operating system and then isolate the Windows XP system on its own dedicated network with no attachment to others, to avoid its use as a launching platform to attack other systems when it becomes compromised.

In case the above doesn't make this obvious, get rid of that Windows XP system as soon as possible, it's not worth the high risk of compromise which it creates in order to run the handful of outdated and so inherently unsupported applications it contains.  The true cost of such a system is much higher than anyone ever understands, since the potential for complete compromise of the entire network environment in which it resides is radically increased each time it is turned on.

Rob

Windows 10 and Cannon Pixma printer - Microsoft Community

alan hosey say:

Windows 10 and Cannon Pixma printer

I have a Lenovo laptop that was running Windows 8 with a wireless network printer Canon Pixma MG6120 with no problems. I upgraded to Windows 10 and the printer doesnt work anymore from the upgraded laptop but does work still on other PC's in the house still on Windows 8 and XP also.

Using the laptop with Windows 10 on the same network I cannot discover the printer anymore and the Canon website does not have any Windows 10 print drivers for any of their printers

Any suggestions please

gmteacher say:

Try downloading the windows 8 / 8.1 printer drivers from Canon and then attempt to install them in Win 10 using "Compatibility Mode." There are numerous step by step instructions available if you are not familiar with Compatibility Mode.

Also, I had a similar problem with some devices on my Vaio laptop, and contacted Sony. They advised me that they were developing a new set of drivers to make their hardware work with Win 10, but that the package would not be ready until as late as November. Of course, Sony actually just sold Vaio and their entire Desktop / Laptop business to an outside company, Japan Industrial Partners, and announced that Sony was leaving the computer arena, so I am crossing my fingers.

So, you may want to look to Lenovo also for an update that will restore your Canon's functionality.
