doug robbins - mvp office apps & services (word) say:

You will need to use a macro such as

 

Dim i As Long, j As Single
With ActiveDocument
    For i = 1 To .InlineShapes.Count
        With .InlineShapes(1)
            .ScaleHeight = 100
            .ScaleWidth = 100
            j = InchesToPoints(2) / .Height
            .Height = InchesToPoints(2)
            .Width = .Width * j
        End With
    Next i
End With

Replace the (2) with the desired height.

jay freedman say:
The problem is a simple typo: In the line

       With .InlineShapes(1)

change the digit 1 to the letter i.

The original macro makes the changes only on the first picture -- that is, .InlineShapes(1) -- as many times as there are pictures in the document. Changing that to .InlineShapes(i) will make the changes to the first picture, then the second, .... until it reaches the last picture.

doug robbins - mvp office apps & services (word) say:

I am not sure what you mean by "resized to 7.5", but with appropriate modifications, the following should process the first inline shape in each section of a document.

Sub resize()
'
Dim i As Long, j As Single
With ActiveDocument
    For i = 1 To .Sections.count
         With .Sections(i).Range
            If .InlineShapes.count > 0 Then
                With .InlineShapes(1)
                    .ScaleHeight = 100
                    .ScaleWidth = 100
                    j = InchesToPoints(12.6) / .Height
                    .Height = InchesToPoints(12.6)
                    .Width = .Width * j
                End With
            End If
        End With
    Next i
End With
End Sub

Try...

=SUMPRODUCT(($A$2:$A$100=$A3)*($B$2:$C$100))

Alternatively, try the following more efficient formula that needs to be confirmed with CONTROL+SHIFT+ENTER...

=SUM(IF($A$2:$A$100=$A3,$B$2:$C$100))

Note that array formulas are resource intensive, hence the use of fixed range references.
