Dim c As Range, TopRange As Range, Top As Long
If Not Intersect(Target, Range("A1:B10")) Is Nothing Then

On Error Resume Next
For Each c In Range("B1:B10")

    End If
Next

The beta version will work with Windows XP, Windows Vista, Windows 7 ,Windows 8 and Windows 8.1.
