Here's my script:

$SysEvent = Get-Eventlog -Logname system -Newest 2000 -ComputerName YOUR_SERVER_NAME
$SysError = $SysEvent | Where {$_.entryType -Match "Error" -or $_.entryType -Match "Critical"}
$SysError | Sort-Object EventID -Unique | Format-Table EventID, Source, Message -auto

Hi Graham, this is a known issue in the 1803 Update,

It is the recovery drive, you are not meant to see it, it is not meant to have a drive letter, remove the drive letter and everything will return to normal:

Run this command and hit Enter

diskpart

In the Second Command window, run this command and hit Enter

select volume H

Run this command and hit Enter

remove letter=H

Close Command Prompt

That's It!

To set the Time Zone automatically in Windows 10, you can follow these steps:

  1. Click the Start button and open the Start Menu.
  2. Go to Settings and select Time & Language.
  3. In the left pane, select Date & Time. You can set the time to adjust automatically or change it manually.
  4. In the right pane, you will see the setting Set time zone automatically. You can toggle the slider button to On.

If you need to bring up the Xbox Startup Troubleshooter again, follow these steps:
