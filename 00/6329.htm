rohn007 say:

I don't understand why you want to print PDFs to XPS. 

XPS is MS's clone of Adobe's PDF file format.

In theory they are both simply static "photocopies" of text that you are not supposed to be able to edit. Later, (MUCH LATER) Adobe created a version that stored text in a separate "layer" so that it could be copied and extracted.

If you want to extract text from PDFs you have 2 options.

First you can try opening it in Word 2016.  If the PDF is the right format, Word will be able to extract the text.  If not, the PDF will appear as an image you can't edit.

In that case you need to find a tool that can "OCR" (Optical Character Recognition) a PDF.  "OCR" is a technique used in programs to recognize and extract characters from images. If you have a good image, not a crumpled up photocopy, OCR can be good. But you ALWAYS have to review the text. Sometimes OCR cannot recognize everything correctly.
