Hi

 

You can create a Start Screen Tile that will Lock the computer.

 

 

Perform the following.

Note: In the following steps, the AppData folder is normally hidden. If you do not see the folder, select the View Tab, click Folder Options. Click the View tab and select the Show Hidden folders option.

1. Open File Explorer and navigate to the following folder.

C:\Users\your name\AppData\Roaming\Microsoft\Windows\Start Menu\Programs

2. Right click in the Programs folder and select New/Shortcut.

3. In the Location box, copy/paste the following command.

rundll32.exe user32.dll, LockWorkStation

4. Click Next, type Lock for the name and click Finish.

Go to the Start Screen and you should see the Lock shortcut Tile that you can use to Lock the computer.

 

You can also Right Click this shortcut and select the Pin to Taskbar if you wish.

iamveryshai say:

Hi Chance!

Try following these steps:

HOW TO REMOVE OTHER ACCOUNT PICTURES OF USERS

 

1. Press the Windows key + R to open the Run window

2. Type in %appdata%

3. Open this file folders: Microsoft > Windows > AccountPictures

4. On the last folder, it will contain all the pictures that you have used for your account. Select the pictures that you would like to delete.

 

HOW TO GET THE DEFAULT USER ACCOUNT PICTURE

 

1. Press the Windows key + R to open the Run window

2. Type in %programdata%

3. Open this file folders: Microsoft > User Account Pictures

4. Select the picture that you needed then Copy (Ctrl + C) and Paste (Ctrl + V) somewhere on your PC such as the Desktop
