On the Page Layout tab of the ribbon, click the Columns button and choose More Columns. In the dialog, click the button labeled Two, check the box for "Line between", and set the first dropdown in the Spacing column to 0. Click OK.

Now insert the table in the left column. Enter the column headings (Name and Address) in the top row; then go to the Table Tools | Layout tab and click the Repeat Header Rows button. karl timmermans [outlook mvp 2012-2018] say:

The fields that get exported are identified in the field map so to get back to some basic items:

1) - To ensure that every field is included, you would need to review what fields appear in the export field map. The only way to do that is to click on the "Map Custom Fields" button. The label for that button is a misnomer since it has absolutely nothing to do with user-defined fields - just brings you to the import/export field map. The "Map Custom Fields" in this case simply refers to the fact that it lets you select specific fields from the standard list - no more - no less (this is true whether importing or exporting)

2) - To ensure that all fields are included for your export - after clicking on the "Map Custom Fields" button - then click on the "Default Map" on field map page. That will ensure that every field is selected. If you don't do that, then whatever fields were included in any previous export - will appear which may or may not necessarily be the complete list.

If after that, there is missing information - be specific as to "exactly" what field(s) you are referring to.

Automatic Maintenance combines all the background processes, including the third part software and apps to include themselves in the process.

Automatic maintenance will have impact on Disk Fragmentation, Running Anti-Virus Scan, and Cleaning up temporary files, running Windows Update Manually, security scans etc.

Please follow the steps to set automatic system maintenance.

  1. Press Windows Key + x and select Control Panel and select System and Security
  2. Next select system and maintenance.
  3. Expand Maintenance and click on Change maintenance settings.
  4. Check the option which says allow computer to wake up my computer at the scheduled time.
  5. Set the Automatic maintenance Schedule time to a time when the computer is powered off / not in use.

If you want to stop a maintenance, you can follow the below steps.

  1. Press Windows Key + x and select Control Panel and select System and Security
  2. Next select system and maintenance.
  3. Expand Maintenance and click on stop maintenance.
