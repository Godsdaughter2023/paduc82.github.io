a. user say:

Hi,

An outdated driver or corrupted system file is one of the reasons why this issue occurs. To troubleshoot the issue follow the steps below:

  1. Make sure that your printer is plugged into the power supply and turned on. Check the USB connection (for wired printers) or the wireless connection (for wireless printers).
  2. Download and run the printing troubleshooter. This troubleshooter will help us to identify the cause of the issue and fix it at the same time.
  3. Update printer driver.

  a. Click Start, and then select Device Manager.

  b. Expand Printers, find your printer in the list, right-click (or press and hold) it, then select Update Driver.

If the issue still persists, we suggest the following:

Clear spooler files and restart the spooler service.

  1. In the search box on the taskbar, type Services, then select the search result of the same name.
  2. In the Services list, double-tap (or double-click) Print Spooler.
  3. Select Stop, then click OK.
  4. In the search box on the taskbar, type %WINDIR%\system32\spool\printers, select the file folder in the search results, and then delete all files in the folder.
  5. Search for and open Services again
  6. In the Services list, double-tab (or double-click) Print Spooler.
  7. Select Start, and in the Startup Type list, make sure that Automatic is selected, and then select OK.

Download and run the following easy fix solution:

Troubleshoot problems with devices and printers.

Find and fix problems with printing.
