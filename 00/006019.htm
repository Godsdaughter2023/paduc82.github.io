doug robbins - mvp office apps & services (word) say:

Use the following to set a document variable varCoverPage = True when a cover page building block is inserted.   Then if the user runs the code again, the coverpage that was inserted by it will be deleted before another one is inserted.

On Error GoTo InsertCoverPage
If ActiveDocument.Variables("varCoverPage").Value = True Then
    With Selection
        .HomeKey wdStory
        .Bookmarks("\page").Range.Delete
    End With
End If
GoTo InsertCoverPage

InsertCoverPage:
    ActiveDocument.Variables("varCoverPage").Value = True
   'Code to Insert CoverPage Building Block

-- Hope this helps.
