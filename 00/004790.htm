word 2010 - table border outside margin

hi
I have a standard template, margins set to T2cm, B2cm, L2cm, R1.5cm.

When I insert a table the borders print outside the margins instead of aligning with the "non table" text within the margins.

I have the table properties set to 100% and align to window.

How do I set it so that the borders print aligned to the margins and not outside the margin area?

thanks
Sue

jay freedman say:

Word by default wants to align the text inside the leftmost and rightmost cells with the outside text margins. That puts the table boundaries outside the margins.

If you want to align the table borders with the outside text margins:

- On the Table Tools > Layout tab of the ribbon, click the Cell Margins button. Change the left and right cell margins from the default 0.19 cm to 0.

- The text inside each cell of the table will now butt up against the cell's borders. To restore the inside clearance, select the whole table; click the dialog launcher (the little arrow button in the
lower right corner) of the Paragraph group on the Home ribbon; and set the left and right indents to 0.19 cm.
