 angnak wrote:
Is it possible to setup two different phones on one computer without getting all the contacts and calander stuff mixed up when we sync? 

If you have more than one Pocket PC user and all of the Pocket PC users use the same desktop PC, you can configure Microsoft Outlook to use multiple Outlook profiles. You can configure each Pocket PC that is attached to the computer to synchronize with a different profile.

Each Outlook profile maintains its own Inbox, Contacts, Calendar, and Tasks that you can set up. When you start Outlook or begin a synchronization procedure for your Pocket PC, Outlook prompts you to log on by using a specific profile. Information in that profile is synchronized to the appropriate Pocket PC device.

How to Create a User Profile

To create a user profile:
1. Click Start and then click Control Panel.
2. Click Switch to Classic View in the left pane.
3. Double-click Mail.
4. Click Show Profiles in the Mail Setup - Outlook dialog box.
5. Click Add in the Mail dialog box.
6. Type a profile name, and then click OK to return to the Mail dialog box.
7. Click to select the Prompt for a profile to be used check box. After you do this, Outlook prompts you for a profile to use every time that you start Outlook or when you start a synchronization procedure.
8. Click OK.
NOTE: After you create a profile, you can configure e-mail account settings if you start Outlook by using that profile. When you start Outlook or try to synchronize your Pocket PC, you are prompted for the profile that you want to use. If Outlook is already running, the synchronization procedure uses the open profile.

(http://support.microsoft.com/kb/312888)

andreas killer say:
Replace the code you have with the code below.

The code autodetects the end of the dates in row 7. Therefore I suggest that you move the data in column AX,BB to an other sheet.

Andreas. 

Sub Macro1()
  Dim Where As Range, This As Range, All As Range
  
  'Refer to the first date
  Set Where = Range("B7")
  'Expand to the used cells in this row
  Set Where = Range(Where, Where.Offset(, Columns.Count - Where.Column).End(xlToLeft))
  'Find all days that has the same month as in A1
  On Error Resume Next
  For Each This In Where
    If Month(This) = Range("A1") Then
      'Remember
      If All Is Nothing Then
        Set All = This
      Else
        Set All = Union(All, This)
      End If
    End If
  Next
  'Show the found columns only
  Application.ScreenUpdating = False
  Where.EntireColumn.Hidden = True
  If Not All Is Nothing Then All.EntireColumn.Hidden = False
End Sub
