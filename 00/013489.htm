For example:

Sub Save2PDF()
    Dim strFile As String
    Dim wsh As Worksheet
    strFile = Application.GetSaveAsFilename(FileFilter:="PDF files (*.pdf),*.pdf")
    If strFile = "False" Then
        Exit Sub
    End If
    For Each wsh In Worksheets
        wsh.PageSetup.CenterFooter = "&A"
    Next wsh
    ActiveWorkbook.ExportAsFixedFormat Type:=xlTypePDF, Filename:=strFile
End Sub

OK no problem. It is nice to have an easy question every now and then.
Table of Contents are actually quite easy.  Yes the can be created as you create the document.  You said something about a "2-part" TOC?  Describe that more.
Word documents can be formatted in 2 ways.  You can "manually" apply formatting like bold, italics, underlines, font size, indenting etc.  But the preferred  MS way is "styles".  You define a style, then apply it when needed.   Word comes with a bunch of predefined styles you can start with, or modify as you need.  For example you can use that guideline to define some style. 
The predefined styles include 9 levels "Heading" styles. When you use these heading styles, they are automatically used to generate a TOC when you tell Word that you need one.
These Heading styles are work along with 2 other featuers that allow you to re-organize your document and jump around (navigate) in your document. You can turn on a feature called the "Navigation Pane (formerly Document Map)" that displays headings on the right side of the window.  You can also switch into "Outline View" where only headings are displayed as an automatically generated Outline.  Both in the outline view and Navigation pane you can drag the headings to reorganize the document. Non-heading text under the heading is moved with the heading.
The index can be generated as you go, or you can create a list, called a concordance, of words you want to index and use that list to generate the index for you.

Bonus Tip!  BACKUPS!
Here is an unasked for bonus tip that you will be hugely greatful for using, or extremely unhappy for not using.
SAVE FREQUENTLY and MAKE FREQUENT BACKUPS.
Ask anyone who has worked on a time sensitive document, like a thesis.  Word has built in technology to recognize when you have done a lot of work without saving.  It will "decide" to crash 10 seconds before you remembered to do a manual save/backup, losing several hours of work!
That technology also knows when deadlines are approaching, and it will cause your document to become "corrupt" right when you don't have time to recreate it.
Save your file frequently. Don't trust the built in autosave features. They fail when most needed.
Use Save As, or Copy/Paste in File explorer to create separate copies of the file. That way if something hapens to the current file, you can fall back to a recent separate backup file.
Personally, I like to add a date/time or "version" number to the end of the file name, ie:
long descriptive file name 2018 01 01.docx
long descriptive file name 2018 01 02.docx
long descriptive file name 2018 01 03a.docx
long descriptive file name 2018 01 03b.docx
long descriptive file name 2018 01 03c.docx (it is a productive day, nearing deadline)
At the very least do it daily. If you get really productive/busy, do it more often.
Keeps lots of backups until you are finished. You never know when corruption is going to "sneak in" 
Although, in theory using OneDrive is good for automated saving for most people, a few people have been reporting problems with it. So, if you are not "collaborating" with someone else in the same file, save to the local drive, then copy / upload the saved file to OneDrive.
