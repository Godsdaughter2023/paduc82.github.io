What are the System Protection settings for available drives? - Microsoft Community

a. user say:

What are the System Protection settings for available drives?

There are (2) available drives listed in the System Protection tab: "RECOVERY" and OS(C:)(System). The "RECOVERY" drive was "Off" when I received my computer. Should the "RECOVERY" drive be "On" also?
j w stuart say:

You have a single drive with two partitions. The 'Recovery' partition is only used to restore your computer back to its "out of the box" state that Windows was in when you first turned on the computer. Normally there is no need to have system protection ON for the recovery partition, as you should not use or modify the contents of this partition, leave it untouched.

Check your owner's manual on how to use the recovery partition should you ever have the need. Be aware that recovering Windows using this partition will erase all your personal data and applications you have installed.

thahaseena m say:

Hi,

 

System protection is a feature that regularly creates and saves information about your computer's system files and settings. System protection also saves previous versions of files that you've modified. It saves these files in restore points, which are created just before significant system events, such as the installation of a program or device driver.

 

System protection is automatically on for the drive thatWindows is installed on. System protection can only be turned on for drives that are formatted using the NTFS file system.

 

The other drive that is present in the list will be another partition on the system. You can turn on the system protection for the drive so that the restore points will be created for that drive also.

¡firedog say:

With difficulty, I'm afraid.

Outlook 2016 can open .eml files, and you can save an opened message in Outlook by using the Move to folder function. However, this only works for one file at a time, so it could get really tedious if you have a lot of messages.

 

The best way, which may or may not work for you, is to install Outlook on the source machine, then use the Export > Exchange function to transfer mail messages to the active Outlook profile. You can then transfer the resulting .pst file to the target computer and open it in Outlook 2016.

The alternative is to use one of the many .eml > .pst converters available on the web. If you go this route, please be very careful; there are a lot of unscrupulous characters out there looking to make money out of people in your situation, without necessarily solving your problem.

A final solution, of course, is to install Windows Live Mail 2012 on the target machine and continue using it as before. This would suit if you have no other particular reason for using Outlook.
