There is no royal road to indexing. It may be that Word can create an automatic concordance, but that is not a good way to generate an index. See http://word.mvps.org/faqs/formatting/Createindex.htm. Word has never been able to generate an index with hyperlinked entries.

Unfortunately, in order to sort the TOC alphabetically (and keep it sorted), you'd have to unlink it.

 

There's a reason so few books have indexes at all and so many that do have indexes don't have good ones. Indexing is a job for a professional, and, although the author should theoretically know what terms are important enough to be in the index, often even the author is ill-equipped to create the index.

jay freedman say:
If you're viewing your pages in Firefox, Chrome, Safari, or some browser other than Internet Explorer, you won't see popups. This happens because Word inserts the alt text in the HTML as an alt= attribute instead of a title= attribute. The alt attribute should appear only when the browser doesn't support graphics, while the title attribute should show a popup. This is actually a violation of the HTML specification, which only IE ignores. (Reference: http://www.456bereastreet.com/archive/200412/the_alt_and_title_attributes)

If that isn't the cause, perhaps the modified files weren't saved to the correct location on the web site and you're still seeing the old pages; or maybe you just need to empty your browser's cache and force it to download the new files from the server.

If you still can't get Word's built-in alt text generation to work, here's how to do it manually (which will be a bit more tedious):

I'd recommend as a first step that you open each page in Word, click File > Save As, and set the "Save as type" dropdown to "Web Page, Filtered (*.htm, *.html)" instead of just "Web Page (*.htm, *.html)". Save that format to the same file name as before. This will remove all the metadata code that makes the HTML so hard to deal with. All you're losing with that is the capability to "round-trip" the document from Word to HTML and back to Word with perfect formatting fidelity -- not something that most people need.

After doing that, open each HTML file in Notepad or another text editor, and wherever you find

alt="...."

add the text of the popup you want, like

title="this is my popup text"

It may or may not be the same as the alt text. The final result should look something like

<img width=336 height=265 id="Picture 1" src="Doc1_files/image001.jpg" alt="A picture of something" title="this is my popup text">

for each picture in each page.
