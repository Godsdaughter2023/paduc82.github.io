Sub Degree_Symbol()
    For Each ocell In Selection
        If Application.IsNumber(ocell) Then

    Next ocell
End Sub

johnvsy say: If the presentation is shifted to PowerPoint can it be turned into a video presentation that I can add audio and then upload to YouTube?  Is that possible?

jay freedman say:
If the presentation is shifted to PowerPoint can it be turned into a video presentation that I can add audio and then upload to YouTube?  Is that possible?

Bearing in mind that I haven't personally done this, I'm sure it's possible.

First you need to know what version of Office you have, and what formats it can save in.

  • PowerPoint 2007 doesn't save to any video format. You would need a third-party program or add-in, such as SnagIt or Camtasia (both from TechSmith).
  • PowerPoint 2010 can save a slide show in Windows Media Video (WMV) format, and you can upload that to YouTube or first convert it to MP4 (see http://www.google.com/search?q=convert+wmv+to+mp4).
  • PowerPoint 2013 can save directly to MP4 format.

The audio is a problem, though. Although PowerPoint will save a link to an audio file in a .pptx file, it doesn't embed audio the way it does with graphics; and if you save the file as .mp4 it won't even keep the link. If you have a lot of slide shows to create and upload, take a look at Moyea PPT to Video Converter. There might be less expensive tools to do this, but I don't know of any -- you should probably post this as a separate question in the PowerPoint forum.

ken sheridan say: Using the EXISTS predicate, which should be the more efficient of the two options:

DELETE *
FROM tblProformaExcel
WHERE EXISTS
    (SELECT *
     FROM qryProformaExcel5
     WHERE qryProformaExcel5.Autonumber = tblProformaExcel.Autonumber);

Using the less efficient IN operator:

DELETE *
FROM tblProformaExcel
WHERE Autonumber IN
    (SELECT Autonumber
     FROM qryProformaExcel5);

Both should do the same, but on the other hand, so should the original SQL statement I posted.
