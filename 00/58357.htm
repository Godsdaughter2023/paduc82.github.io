The primary reason for this distortion is caused by the difference in size between the image and your systems display ‘resolution’ or screen size, for example, the size of the image that you chose as your wallpaper is 300 x 200 and your screen resolution is set at 800 x 600, than Windows is going to ‘blow up’ the image by a factor of 2.67 for the width x 3 for the height.
This will cause two changes to the image. First, when you increase the size of an image, it becomes very ‘pixilated’ or blurry because there are not enough ‘dots’ or ‘pixels’ to fill the larger space.
The easiest thing to do is to allow the image to be displayed in its original size, which will not allow it to cover the entire background, but the image will be clean.

a. Right-click anywhere on the background, then select Properties. This will open a ‘Display Properties’ dialog box with several tabs across the top.
 
b. Select the ‘Desktop’ tab, then look towards the lower right side of the box for the ‘Position:’ setting.
 
c. It should currently say ‘Stretch’ which can be changed to either ‘Center’ or ‘Tile.’
 
d. ‘Center’ will place the image in the center of the screen at its original size and ‘Tile’ will repeat the image to cover the entire background.
 
e. If the image appears very small when you select ‘Center,’ then it’s true resolution is very low and may not be a good candidate as a Wallpaper image.

considerate_guy say:

I'm always surprised by the surface knowledge that Microsoft Support personnel seem to have on their own products while not having a very deep knowledge on them. I'm not part of Microsoft, just a fellow consumer/user of their products who has a lot of familiarity with programs like Movie Maker, more so apparently than their own support personnel.

Windows 8.1 ProPack with Media Center wouldn't be able to help you with being able to create video DVDs. The ProPack adds the ability to play DVDs. It adds the licensed MPEG-2 codecs that let you decode video on a video DVD. But it does nothing to allow you to burn video DVDs. I'm sorry for the misinformation you were given.

To give you some history -- in Windows Vista and Windows 7, the ability to play DVDs and to burn DVDs was included. Microsoft licensed the MPEG-2 codecs that allowed both viewing and burning. The viewing was done in Windows Media Player. The burning was done in a program called Windows DVD Maker. In Windows 8/8.1 both the codecs and the program are gone.

Movie Maker (both the older and the newer version), if it is running on a system with Windows DVD Maker (Vista or 7) would then offer you the option to burn a DVD. It would pass the project file to the program and then let you create a DVD with menus, etc.

Much like Apple dropped support for burning via their iDVD program, Microsoft has stopped providing the Windows DVD Maker program. Furthermore, there is no way to get the program from Windows Vista/7 onto a Windows 8/8.1 machine. Essentially it is no longer provided/supported/available.

Now, before I go into possible workarounds, can I also ask you why you might need the ability to burn a video DVD? The video DVD standard came from the era of analog TVs and is stuck at a maximum standard definition resolution of 480p (NTSC). It's not really a future-proof format any more. Personally, I save all my videos to a computer with copies on a network device. I can stream from either of these to any HDTV in the house via DLNA because they are connected to the same network. Furthermore, I make regular backups of videos (and photos) to an external hard drive which is then stored offsite. For me, that gets me the same functionality I might get from an optical DVD disk. If the concern is sharing with family and friends, I would stick with online sharing via YouTube, Vimeo, OneDrive, etc. You can also password protect videos to keep things private.

If you have someone that needs to be able to view a movie and they are behind technology-wise, some other options are to burn a video file (.mp4 format) to a DVD just as a data file. That can then be played in a Blu-ray player and will be in HD format to boot.

Okay,  if none of that is relevant and you still need a shiny disc in the legacy SD video DVD format, the only real options at this point are to look for 3rd party applications that can burn a video DVD. A few that I've seen used are Freemake Video Converter, DVD Styler and DVD Flick. Just be careful with installing any free programs like these because they often hide additional programs during the install. Always choose a custom install so that you can uncheck anything the program tries to install in addition to the program.

I hope that helps. Let me know if you have any further questions. And again I'm sorry for the incomplete information you received previously.
