shane devenshire say:

Hi,

The Word object model and the Outlook object model are not the same.  Maybe the easiest way  to understand this is to consider that you are in Word and you want to press Send/Recieve (a standard command in Outlook) but its not there??  Why, because that is what Outlook does but not Word.  Now try to open the Inbox in Word - not going to happen because Word doesn't do Inbox's.

You want some of the feature from Word to be available in Outlook but if they all were then there would not be an outlook program.  But how many people need Bibliographies and Footnotes in there email?

For a macro that runs in Word to run exactly the same in Outlook would require that both programs have the same object model - which means all their commands must be exactly the same.  From the discussion above you can see why this is not the case.

It is not to say that you can't copy code from Word to Outllook but most likely you will need to modify it.

jllatham say:

Make a copy of your workbook to test this with.  Try this code instead of what you have.  You need to have the worksheet you want to work with selected, just as you do now.  This code will ignore what someone may have selected and will work with entries in column K beginning at row 2 on down to 1 cell past the last used row in column K

 

Sub FillCol_K_Blanks()
  Dim Cell As Range
  Dim codeSelectedRange As Range
 
  'this will work with all the cells beginning
  'at K2 on down to the last row in column K
  'with anything in it PLUS 1 blank row below that.
  Set codeSelectedRange = _
   ActiveSheet.Range("K2:" & _
    ActiveSheet.Range("K" & _
    Rows.Count).End(xlUp).Offset(1, 0).Address)
 
  For Each Cell In codeSelectedRange
    If Cell.Value = "" Then
      Cell.Value = Cell.Offset(-1, 0).Value
    End If
  Next Cell
  'good housekeeping
  Set codeSelectedRange = Nothing
End Sub

You can use Intersect() to pare the full column selection down to what is actually used.

Sub FillSelectionBlanksBelow()
    Dim Cell As Range
    For Each Cell In Intersect(Selection, ActiveSheet.UsedRange)
        If Cell.Value = "" Then Cell.Value = Cell.Offset(-1, 0).Value
    Next Cell
End Sub

teylyn say:

Hello,

what do you mean with "I'm using Office 365"? If you have an Office 365 license, you still have different options for working with Excel, i.e. Excel for desktop, Excel online, etc. So, what are you using?

A VBA macro will be the way to go, but that will not work in Excel online, that's why I'm asking what you are using. 

The following code is a starting point. It will look at column A in Sheet1 and will create a new sheet for each row of data it finds in column A of "Sheet1" and will copy that into a new sheet. This code needs to go into a Code Module in the VB Editor. You can adjust the reference to "Sheet1" to your sheet name. 

Sub SplitRowsToSheets()
Dim cel As Range
Dim lastRow As Long, i As Long
Dim sh As Worksheet

Set sh = ThisWorkbook.Worksheets("Sheet1")

lastRow = sh.Cells(Rows.Count, 1).End(xlUp).Row

For i = 1 To lastRow
    sh.Rows(i).Copy
    Sheets.Add After:=ActiveSheet
    ActiveSheet.Paste
Next i
End Sub

Let me know how you get on with this.

And on the opposite end, if an email address is removed from a distribution list, is there a way for that calendar event to be removed from that employee's calendar automatically? 

It gets very convoluted when trying to figure out who has been added since the quarterly meeting was created and who is missing now that we are getting close to the date.

nemanja pantelic msft say:

Hi Erin,

Acording to my tests, there is no an option for the automatic update you asked for. The best workaround solutions for the questions would be:

  • After adding new contacts to the distribution list, find the meeting in your Sent items folder and forward it by clicking on Respond> Forward. Re add the group in To field and send the meeting again. This way, you will send the meeting invite to all (old and new) users.
  • For removing, after you send new invite, removed employs will not be included. So the best way would be to contact all of them and ask to remove the meeting from the calendar.
