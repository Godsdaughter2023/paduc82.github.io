wazzi_3 say:

Hi Vivek,

Thank you for your post on Yammer.

Please find the below information on your queries.

Q) Yammer Logs – can we get Logs to verify User activity or Group activity

A) The Yammer Analytics Dashboard provides the information on the Yammer activity, however it has been deprecated since 14th May 2015. Also you can check the link for the same https://support.microsoft.com/en-us/kb/2840526.

Q) Announcement – can we make an announcement administrator who can only make announcement to "all company"(currently only -admin and verified admin can do this).

A) Yes, only admin and verified admin can do the announcement. However we cannot make a admin for announcement only.

Q) We have observed Any user can deactivate other user profile and NO there is no logs to trace it.

A) When a user has been deactivated by another user, you can follow the below step:

  • Once you are logged into Yammer as a verified admin
  • Click on the gear symbol on the bottom left
  • Then click on Network admin, then click on Remove users
  • It will give you the list of deactivated user and also give the name of the user who has deactivated it.

Q) Unable to delete private messages and conversations & Verified admin cannot perform certain actions like deleting messages, log verification, users activity.

A) A verified admin can delete and view the private messages and conversation.

You can follow the below steps.

  • Once you are logged into Yammer as a verified admin
  • Click on the gear symbol on the bottom left
  • Click on content mode, then click on Private mode and then save
  • After which you can join any group and you can delete or view messages or conversation.

Also check the support link for the Private content Mode:

https://support.office.com/en-us/article/Monitor-private-content-in-Yammer-Verified-Admins-b5271fe7-62aa-4dc4-a1d7-b2b0bd23edae?ui=en-US&rs=en-US&ad=US

If a problem affects multiple user accounts on a specific computer, that's usually when the Windows installation or the computer itself is problematic. You can try to create another user account on the computer to see if the issue is specific to the existing one. To do so, follow these steps:

  1. Right-click on the Windows icon on your home screen.
  2. Choose Command Prompt (Admin).
  3. Click Yes if you get a prompt window.
  4. Type net user test /add.
  5. Press Enter.
  6. Sign out from your user account or restart the computer.
  7. Sign in to the new user account (test).
  8. See if you will be able to use a Microsoft account as the sign-in option using the test account.

If you are still unable to add your Microsoft account on the new user account, check if updating your current Windows 10 build will take care of the problem.

To remove the test account we created, refer to these steps:

  1. Sign in to your real user account.
  2. Right-click on the Windows icon on your home screen.
  3. Choose Command Prompt (Admin).
  4. Click Yes if you get a prompt window.
  5. Type net user test /delete.
  6. Press Enter.
