muddasirshaikh say:

IFERROR works in the following manner:

IFERROR(<Here your formula goes>,<Here the action you want to perform or value you want to show in case of error>)

In your case, you can write the formula:

=IFERROR((I20-F19)/F19,"No value in one or both cells")

or you can use the following formula as well

=IFERROR((I20/F19-1,"No value in one or both cells")

The message "No value in one or both cells" can be replaced with any value like "N/A" or even left blank using ""

ActiveSync Error Code 85020006 - Microsoft Community

schmoov say:

ActiveSync Error Code 85020006

Why do I get this error during synchronization?

schmoov say:

Possible Cause:

The Connection Manager could not find a path to the destination.

- OR -

Unable to connect. Verify your dialup or proxy settings are correct, and try again


Possible Resolution:

1. Verify server name is entered correct.

2. There is no connection available for the device to communicate with the server.

3. The user has more than 372 top level folders in their mailbox and is using a Windows Mobile 2002-based device.


User Action:

1. Verify the connectivity settings on your device.

andreas killer say:
Am aware that the details vary depending on the file type.
Okay, then read this MSDN article and try the function below.

Andreas.

Option Explicit

Sub Test()
  Dim i As Integer
  For i = -1 To 4
    Debug.Print i, GetDetailsOf("c:\windows", i)
  Next
End Sub

Function GetDetailsOf(FName As String, Index As Integer) As String
  'Liefert spezifische Dateieigenschaften (Author, Titel, Thema, etc.)
  Static objShell As Object, fso As Object
  Dim objFolder As Object, objFile As Object
  On Error GoTo ExitPoint
  If objShell Is Nothing Then
    Set objShell = CreateObject("Shell.Application")
    Set fso = CreateObject("Scripting.FileSystemObject")
  End If
  Set objFolder = objShell.Namespace(fso.GetParentFolderName(FName) & Chr(0))
  Set objFile = objFolder.ParseName(fso.GetFileName(FName) & Chr(0))
  GetDetailsOf = objFolder.GetDetailsOf(objFile, Index)
ExitPoint:
End Function
