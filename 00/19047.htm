VBA to change font type on current active slide - Microsoft Community

jenloi say:

VBA to change font type on current active slide

Need help with using VBA to change font type on the active slide.  For example, the font on the active slide is "Arial", and I want to change to "Tahoma" - just for this active slide; not on any other slide.

john sr wilson say:

Try:

Sub To_Tahoma()

Dim osld As Slide

Dim oshp As Shape

Set osld = ActiveWindow.Selection.SlideRange(1)

For Each oshp In osld.Shapes

If oshp.HasTextFrame Then

oshp.TextFrame.TextRange.Font.Name = "Tahoma"

End If

Next

End Sub

NOTE this will not change tables, groups. smart art you would need to write much more complex code if that is needed.
