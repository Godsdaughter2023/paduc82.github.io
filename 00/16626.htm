ken sheridan say: When you say 'combined 2 tables' do you mean that you moved the data from one into the other, making the first redundant?  If so, and you want the form to show the data from the table into which you moved the data, then create a new query, solely on that table, returning all the columns (fields), for which you can use the asterisk character, and sorting the rows in whatever order you want them to show in the form.  With a Contacts table for instance you might want to sort first by LastName and then by FirstName, so the query would be:

SELECT *
FROM [Contacts]
ORDER BY [LastName], [FirstName];

The form should then have the name of the query as its RecordSource property, and each bound control in the form should have the name of one of the columns as its ControlSource property.
