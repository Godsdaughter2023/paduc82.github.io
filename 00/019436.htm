ron coderre say:

Since you are using Excel 2010, you have Slicers available. Those are little windows that control one or more pivot tables by applying the same filter to all pivot tables that are connected to them.

 

Try this:

• Select a pivot table

• PivotTable_Tools.Options.Insert_Slicer.Insert_Slicer

...Select a field to use as a filter...for example: Weekday

...Click: OK

That will create the basic Slicer.

 

• Right-click on the slicer to view the context menu

...Select: PivotTable connections

...Check each PivotTable that you want controlled by that Slicer.

...Click: OK

 

Now, whenever you change the Slicer filter, all connected PivotTables will be impacted.

 

Is that something you can work with?
