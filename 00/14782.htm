Perhaps you have the wrong concept. Customize Keyboard is used to define shortcuts to Word commands. Therefore, the Press new keyboard shortcut field only admits certain key combinations (with one key being Command or Control).

What is it that you are trying to accomplish?

bob jones aka: cybertaz say:
As Daniel suggests, not every key combination can be used... Some are owned by the OS, some are hard coded into the program. There just are too many variables to list here what can/cannot be done, though.

If you provide specifics about;

  • What command you're attempting to assign a shortcut to
  • What combinations you're attempting to use

...someone is better able to assist. Without those details it's impossible to tell whether what you're attempting can be used. You also need to include the present update levels of your OS X & Office software.

3 very basic points:
  1. Once you select the command you need to click in or tab to the Press new keyboard shortcut field... You can't just 'select & type'
  2. Do not attempt to type the names of the keys [e.g., C-o-m-m-a-n-d]. Press the actual combination as you normally would use it
  3. Use of the Option key is highly restricted. Attempting to use it will invalidate almost any combination you're trying to use.
