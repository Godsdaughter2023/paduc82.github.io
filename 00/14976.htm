mateusml say: This error is called the "FDE_DOWNLOAD_SIZE_LIMIT_EXCEEDED". For both feeds and enclosures, the default size limit is 15 MB. To increase, you have to change one or two registry values. Just open your registry editor, run "C:\Windows\regedit.exe". Look for HKEY_CURRENT_USER > SOFTWARE > Microsoft > Feeds. Inside you have two DWORDs: MaxFeedSize and MaxEnclosureSize. If they are not there, look at HKEY_LOCAL_MACHINE > SOFTWARE > Microsoft > Feeds. Didn’t find it yet, so you can create them (on both folders it will work ok). You need to set a value for the DWORDs. E.g.: If I want to download 500MB video files, I would define both MaxFeedSize and MaxEnclosureSize = 512000 (500 * 1024). For reference look at: http://msdn.microsoft.com/en-us/library/windows/desktop/ms686401(v=vs.85).aspx.

I hope this helped you.

rob koch say:

Most likely you have at least one key on your keyboard sticking, possibly a shift key based on the inserted text.

People ask questions like this quite often, though usually the sticking key is simply something like a "1", which simply repeats quickly hundreds of times and fills the page.

A sticking Alt, Shift or Ctrl key modifies whatever other keys you type and can vary in its effect when in an application or Windows itself, which causes the inconsistent behavior.

If you have a desktop keyboard with a USB connector available, try connecting it to the laptop USB and see whether it works properly.  If so the problem is almost guaranteed to be the laptop keyboard.

You can also try tapping each of the Shift, Ctrl and Alt keys a few times to see if that releases the stuck key at least temporarily.  If the problem is something damaged within the key or failed electronics, this may not work though.
