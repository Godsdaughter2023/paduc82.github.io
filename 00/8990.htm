Hello

The new Skype for Windows 10 no longer supports desktop API which connects third party apps. Therefore, it's not possible to use Pamela with the new Skype version.

Hi folks,

I just did the Windows 10 upgrade from Win 7 on a customers computer. The machine was cluttered with malware, spyware and junkware which necessitated a major cleanup before the upgrade. I utilize a combination of Spybot, Malwarebytes and MSE also on my own machines as well as customers.

After the upgrade, I experienced the same issue with Windows Defender not being able to be turned on. It is indeed because of Spybot 2.5 but there is a ridiculously easy fix. It seems that when Spybot 2.5 installs, it integrates itself automatically into Windows 10 "Security Center Service" which will turn off Windows Defender. You will need to go into Spybots settings: under the "System Services" tab the "Security Center Service" option is first. Uncheck " Active after every reboot" and move over to the drop down menu to the right and choose "Stop".

You will now be able to go to Windows Defender in the Control Panel and it will start up and run as normal. Keep in mind you will still be fully immunized and internet protected by Spybot, it is just no longer integrated into Windows Security Center. You now will have, as before, protection from Defender(formally MSE), Malwarebytes and Spybot.
