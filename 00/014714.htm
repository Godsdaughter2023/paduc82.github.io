On and off for some time now I have had intermittent problems updating the virus definitions on MSE which gets stuck on Search, and with the green bar stopping very approximately above the end of the underlying word "minimize". Originally it would eventually complete the download and installation, nowadays it just ends after an hour or more with a "not connected to the internet" error (although the internet connection is fine, of course).

What I've successfully done now on both my machines is as follows:

1. Leave the updating stuck as above for about 5 or 10 minutes.

2. Open the task manager and end the application entry for MSE, the window for MSE will then close.

3. Go to the start menu and activate MSE. The window will now re-open.

4. Click again on "Update Definitions" - the update will now download instantly and install.

It would seem therefore that this problem is caused not by the program searching for, and failing to find, the update, but rather it finds the update but gets stuck at the commencement of the download. I stress that there is no need to reboot the computer.

I must stress that I have only tried this once on both machines, but as it was successful in each case I was keen to get this solution before the community in order that others could try it and speed up any confirmation or rejection of its effectiveness.  I hope it works for all who have this problem. If it does, and my diagnosis of the cause is correct, then it may not prove to be necessary to leave it for 5 or 10 minutes under stage (1) above as moving to stage (2) as soon as it appears stuck may still be effective. That's something I'll continue to test.

Actually, this is a pretty good idea. As Bob said, a macro might be able to do what you want. The macro would need to look for a worksheet event when a cell is selected. It's a bit of an advanced project for a VBA beginner, but this is a very good idea so I hope you pursue it.

Since I don't know your experience level with VBA, I'll start with assuming that you have at least some familiarity with VBA. Using VBA is something I think just about every user of Microsoft Office should be able to do to get the full value of the suite.

The macro recorder can help you with the shading options. For example, to shade a column and row your code could be as simple as

Sub Macro3()

'

' Macro3 Macro

'

'

    Columns("D:D").Select

    ExecuteExcel4Macro "PATTERNS(1,0,14350563,TRUE,2,3,0,0)"

    Rows("7:7").Select

    ExecuteExcel4Macro "PATTERNS(,0,14350563,TRUE,2,3,0,0)"

End Sub

You could insert the hyperlink (and any other content) into a Rich Text content control, then set the content control's properties to 'Content control cannot be deleted' and 'Contents cannot be edited'.
