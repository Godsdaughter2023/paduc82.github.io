wildrosend say:

Help with macro to resize all comments in the workbook

I'd would like to have a macro that will find all comments on all sheets within the workbook and resize the comment boxes to a particular width, and adjust the height of the comment box to fit the text without being too long or too short.

I have used the following code but this is only applicable to the active sheet, and I've noticed that the height of the comment box can be very long in some cases. Is there any suggestions to modify this code.

Many thanks in advance for your help - Tom

Sub Comments_AutoSize()
'posted by Dana DeLouis  2000-09-16

  Dim MyComments As Comment
  Dim lArea As Long
  For Each MyComments In ActiveSheet.Comments
    With MyComments
      .Shape.TextFrame.AutoSize = True
      If .Shape.Width > 300 Then
        lArea = .Shape.Width * .Shape.Height
        .Shape.Width = 250
        ' An adjustment factor of 1.1 seems to work ok.
        .Shape.Height = (lArea / 200) * 1.1
        End If
    End With
  Next ' comment
End Sub
