bob jones aka: cybertaz say:

Not to dismiss the suggestions offered by John & Daniel, but the fact is that what you have in mind really is not a simple task in a word processing program. It would be handled better by a page layout or Desktop Publishing program rather than a word processor.

With a page layout program you can define Master Pages which can be inserted as necessary anywhere in a publication. Each Master is an independent element which carries its own page layout attributes. Formatting & column definition can be applied to the containers (i.e., text boxes) on those Masters which hold the text.

Word processors do not work that way - there is no 'page' component to be defined, despite the facts that the terminology used & the program being designed to make it look as though the document is comprised of pages give that impression. Word processing programs are a flow of text from start to finish. Changes to layout [orientation, margins, columns, etc.] have to be defined by Section Breaks that alter the text flow. Formatting [font, font size, line spacing, etc.] has nothing to do with Sections, though. Character & Paragraph formatting are applied by the use of Styles which can be applied to the text in any Section.

If this must be done using Word, what Daniel detailed is a usable approach. Just keep in mind that the Section Breaks are not fixed in place... IOW, as content is added to Section 1 all sections beyond that move down to accommodate the expansion of that section. Accordingly, the content for each 'page' as you call it would have to be entered in the appropriate Section - content will not flow from one predefined Section into the next.

In addition to other considerations, documents based on such a template also are subject to inclusion of blank pages dependent on how the Section Breaks flow as content is added. For example, if the Section Break for Section 2 gets pushed to the top of the 2nd page, page 2 will be laid out the same as the 1st page, it will be blank & the 2nd Section will start on the 3rd page.

In a nutshell, anyone using the template will have to be well educated in the use of Word as well as the design of the template. Even a well done job won't provide a simple "point 'n' click" result :-)

daniel g. grau say:

Create a blank document and Save it as .dotm (Macro-Enabled Document).

Define the parameters or the first half of your document:

  1. Use Format > Document to define the margins.
  2. Use Page Setup (at the bottom of the above dialog) to define the page/paper size.
  3. Exit the Format > Document dialog.
  4. Access the Header and Footer and type any text you need repeated on every page. You may also want to add an automatic page number and/or unlink them.
  5. Access Format > Columns and define the number of columns.
  6. Go to the end of the document, then do Insert > Break > Section Break (New Page).

Define the parameters or the second half of your document:
Go to the end of the document, then do Insert > Break > Section Break (New Page). Repeat steps 1-6.

Define fonts, styles, etc.:

Access the various paragraphs and apply the styles, fonts, colors, etc. that you need.

Create multiple pages:

Do Select All (Command A) followed by Copy (Command C).

Go to the end of the document and Paste (Command V) as many copies as you need.

It all started several years ago with an email virus that continuously sent out nonsensical and weird emails, at random times, to all of my contacts.  Many were insignificant, but some of them ended up having damaging results…those were the ones sent to my co-workers.  They thought I was going crazy, and I sure can’t blame them!  Most of my friends and family felt the same way, but they knew me too well…

Next development:   about two weeks ago I unknowingly “blocked” my OWN email address (sometimes I send email to myself, mostly reminders, or maybe blind-copy myself on an important memo).  When I didn’t receive my emails the next day, I looked in the Junk Inbox and found them there.  That’s also the same day I discovered there seemed to no longer be a way to access my Blocked Senders List.  Can you even imagine my panic?

From that time onwards I monitored my Junk email even more closely.  Some of my Blocked Senders are now sending emails to me using MY OWN EMAIL ADDRESS as their return address.  For some reason the emails are going to my “Junk Inbox”, which I don’t really understand.  However, I’m perfectly happy to accept it and to consider myself extremely lucky that they are…for now!

The new emails received today had their Message Sources BLOCKED so I couldn’t look at them at all!  Guess they’re getting “even smarter”, eh?

Please, does anybody out there know how I can stop this invasion of my personal space?
