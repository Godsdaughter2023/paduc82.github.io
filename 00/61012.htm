vernbauch say:

I am trying to copy picture files from C: to D: drives. How do I do it

I need to increase the space on my C: drive and would like to copy picture files to the D: drive and delete them from the C: drive.  I can not find a solution on how to do this.  DOS was easy but there seems to be no program for this.
lemp say:
Two ways:

  • Open one Windows Explorer window and navigate to the directory that has the picture files
  • Open a second Windows Explorer window and navigate to the D:\ drive (or to a new folder that you might want to create on D: by using File > New > Folder).
  • Drag and drop into the 2nd window.  See http://windows.microsoft.com/en-US/windows-vista/Moving-and-copying-files-by-using-the-drag-and-drop-method  You can do this one at a time or select multiple files by using the CTRL key or SHIFT key.  If you use the right mouse button instead of the left while you're dragging, you'll get a menu that allows you to select whether to MOVE or COPY (better practice is to COPY then DELETE after you're sure the copy did what you wanted; MOVE automatically deletes the source files after they are copied to the destination).
Or

  • Open a Windows Explorer window and navigate to the directory that has the picture files.
  • If the left pane of the Window does not show a list of common tasks, make sure that neither the "Search" button nor the "Folders" button in the Explorer toolbar is selected.
  • Select the files to be copied/moved, using CTRL and/or SHIFT as desired
  • Under "File and Folder Tasks", click "Copy this file"
  • Browse to the destination folder and click "Copy"
