Method 1:

This issue occurs because the USB 3.0 device is not fully inserted into the port.

The USB 3.0 port supports USB 1.0, 2.0, and 3.0 devices. The first layer of pins inside of the USB connector are for USB 1.0 and 2.0 ports. f)       Restart the computer, and then reinstall the USB controllers.

g)      Plug in the removable USB storage device, and then test to make sure that the issue is resolved.

For more information about strong passwords, see Tips for creating strong passwords and passphrases.


If you want to remove your password:

Remove Windows User Password

1. Open User Accounts by clicking the Start button, clicking Control Panel, clicking User Accounts and Family Safety, and then clicking User Accounts.

2. In "Make changes to your user account" of the User Accounts window click Remove your password.
3. Enter your password
4. Click Remove Password to confirm.
