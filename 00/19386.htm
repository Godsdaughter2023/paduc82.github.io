hansv mvp say:

Let's say you have data in Sheet1!A2:A100 and in Sheet2!B2:B100.

Select the first range, i.e. Sheet1!A2:A100.

On the Home tab of the ribbon, click Conditional Formatting > New Rule...

Select 'Use a formula to determine which cells to format'.

Enter the formula

=ISERROR(MATCH(A2,Sheet2!$B$2:$B$100,0))

using the correct address of the second range.

Click Format...

Activate the Fill tab.

Select a highlight color.

Click OK twice.

You can do the same for the other range, adjusting the formula of course.

brian tillman [outlook mvp 2007-2019] say:

Sounds like one of the following: 1) you are not correctly specifying the outgoing server, 2) you are not properly authenticating to the outgoing server, 3) you are using the wrong port.

Are you trying to access your Hotmail mailbox using a POP account?  If so, here are the settings:

 

  • POP (incoming) server: pop3.live.com
  • POP port: 995
  • POP SSL required: yes
  • POP user name: your complete Windows Live Hotmail address (including "@hotmail.com", "@live.com", etc.)
  • POP password: your Windows Live Hotmail password
  • SMTP (outgoing) server: smtp.live.com
  • SMTP port: 25
    • If you run into problems sending mail, try "587" for the SMTP port instead.
  • SMTP SSL required: yes
  • SMTP authentication: yes
  • SMTP user name: your full Windows Live Hotmail email address
  • SMTP password: your Windows Live Hotmail password

For the last two items, you can also substitute the "use the same values as the incoming server" option.  If you are using POP access, you will not be able to see the contents of any Hotmail folder except Inbox.  POP doesn't understand folders.

    Hi Nathalie,

    If you merely want a hyperlink, then only copy the URL of the document and paste that into Sway.

    If you want to embed it in Sway and host it from another site, get the iframe code of the document and paste that into an Embed card in Sway.

    Let me know if you mean something else and I'll do my best to help.

    nathalie kennedy say:

    hi Jason,

    No, what I am after is to be able to link to pdf or word documents (or ppt as well) as attachments to the newsletter.     The documents we need to attach to newsletters are usually emailed to us and are stored on my desktop or my email folders.

    thanks

    jason (office engineer) say:

    When you send an email with an attachment, the attached file is actually sent along with the email message to the recipients.  Sway is different in that it exists on the web similar to a web page and doesn't have the ability to hold the contents of a PDF or Word document in that form.  You can either insert the contents of those files into your Sway, which it sounds like is what happened when you first posted and wasn't what you wanted, or you can link to/embed the file so that it still displays as a PDF or Word file.  When doing the latter, that file needs to live somewhere on the Internet or your organization's network so that other people can access it.  If you want to have it embedded so it shows up within a smaller window within the Sway, uploading it to OneDrive is the easiest.  If you merely want to link to it, then you can really upload it anywhere and then copy the URL to it and paste it into your Sway.

    Hope this helps!

    Jason
