danielco say:

Suppose the addresses are pasted cell A1 :

Sub test()

Dim txt As String, Ctr As Long

Application.ScreenUpdating = False

With Sheets("Sheet1")

txt = Replace(.[A1], " ", "")

For Each Item In Split(txt, ";")

    Ctr = Ctr + 1

    .Hyperlinks.Add .Cells(Ctr, 1), "mailto:" & Item, _

    TextToDisplay:=Item

Next Item

End With

Application.ScreenUpdating = True

End Sub

As I mentioned in my reply to the message you sent there is no reliable way to do this.

You are trying to change the write password which is not the same and in any case if there was a write password you wouldn't have an Activepresentation would you?
