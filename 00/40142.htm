"How to Refresh Windows 8 and Windows 8.1"

  • The PC boots into Windows RE (Recovery Environment).
  • Windows RE scans the hard drive for your data, settings, and apps, and puts them aside (on the same drive).
  • Windows RE installs a fresh copy of Windows.
  • Windows RE restores the data, settings, and apps it has set aside into the newly installed copy of Windows.
  • The PC restarts into the newly installed copy of Windows.

"How to Reset Windows 8 and Windows 8.1"

"When you reset your PC, the drive will be completely reset, and all your apps and files will be removed from it. Windows 8 will be reinstalled, and you'll need to change any settings and reinstall any apps that you want to get back after resetting. If you know if any files that you don't want to lose, then you should back those files up before starting the reset"

I read that the solution was to remove c:\windows\temp\WUSB*.gif, reboot, which I did, but I had the same problem.

To really fix it without trying a bunch of things (I already tired them):

Stop the WUSB... Service and change the Properties from Automatic to Manual.
(Start, Settings, Contort Panel, Administrative Tools, Services)
Find the WUSB... Service, Stop it and set the Properties to Manual.

Delete c:\windows\temp\WUSB*.gif (4 files I think)

Restart the system (with no WUSB service running since it was set to manual.

Wait a few minutes for things to settle down, but no connection to the wireless network was noted.

Go back into Services and Start the WUSB service manually.

Four (or maybe five) WUSB*.gif files then appear in c:\windows\temp but there is no complaining.

The Linksys icon then reappeared in the Notification Area and you are connected to the Internet.

Let things settle down for a few minutes and test the Internet connection.

Go back into Services and change the service from Manual to Automatic so it would start on the next reboot.

Reboot and resolve any remaining issues.

Make the Linksys display icon in the Notification Area as Always hide (optional).
