Hi Kona

I think you will be very happy with Office 2019. There are a LOT of new features, plus with a subscription you get access to many new applications using the Office portal.

Microsoft User Data folder(s)

Hello mslcfl,

When you start the Zune software for the first time, the software automatically adds music, videos, pictures, and podcasts on the computer to the Zune collection. Then, you can set up the Zune software to monitor the folders in which you store your media. If you add or delete something in these folders, the Zune software updates your collection accordingly.

To prevent the Zune software from automatically adding these files to your collection, follow these steps:

1. Start the Zune software. 
2. Click Settings , click S oftware , and then select Collection . 
3. Under "Windows 7 Libraries," click one of the Manage  buttons under the monitored folders. 
4. When you are prompted, select a folder and then click the  R emove button to stop the Zune software from adding files to your collection. 
5. If the What about the content ?  dialog box appears, select whether you want to remove the folder and its content from your collection or you want to remove the folder but keep its content. 
6. Click OK. 

To remove music, videos, pictures, podcasts, and Zune devices from your collection:

1. Start the Zune software. 
2. Click Collection , and then click one of the following:
• Music
• Videos 
• Pictures 
• Podcasts 
• Channels
3. Right-click the item that you want to remove, and then click Delete.
4. Select the option that you want to use, as follows:
• Keep file(s) but remove from collection
The item remains on the computer, but the item is removed from your collection.
• Delete file(s) and remove from collection
The Item is deleted from the computer, and the item is removed from your collection.
5. Click OK.

Please let me know how you get on. Thank you for posting!

Basically if you have a date in a document property displayed in your document it will not display correctly if you have daylight saving enabled and you are in the daylight saving period and the date is not in the daylight saving period.

e.g.

Current date 9/4/18 (in the daylight saving period)

With automatic daylight saving enabled

date in property 31/12/17 (outside the daylight saving period - same effect if you go forward with the date too - e.g. to 31/12/18)

date displayed 30/12/17

date in property 31/3/18 (in the daylight saving period)

date displayed 31/3/18

Now automatic daylight saving disabled

date in property 31/12/17 (outside the daylight saving period)

date displayed 31/12/17

date in property 31/3/18 (in the daylight saving period)

date displayed 31/3/18

Note turning daylight saving off (only with the document open(?)) will change the value stored in your document property!!

e.g.

Current date 9/4/18 (in the daylight saving period)

With automatic daylight saving enabled

date in property 31/3/18 (in the daylight saving period)

date displayed 31/3/18

Turn off daylight saving

date displayed 30/3/18

date in property 30/3/18
