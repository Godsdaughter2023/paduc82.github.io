System Interrupts are a mechanism by which the hardware (or a background device) can notify the processor that something requires attention, could be USB transfer, disk transfer something like that; nothing to worry about.

Even simply moving your mouse activates an interrupt, the Operating System see's the interrupt which tells it that the mouse is being used and it must divert it's attention to what the mouse is doing.

You might experience performance issues in Windows for a number of reasons. Some common issues that can cause these problems are:

  • Due to Software conflicts or Hardware problems.
  • If the drivers are corrupted or incompatbile with Windows.
  • Multiple startup programs running in background.
  • If your computer is infected by malware.

I suggest you to follow the troubleshooting methods listed below and check if it resolves the issue.

Method 1: Run the System Maintenance troubleshooter

Windows 10 has a built-in troubleshooter to check and fix issues with system performance and maintenance. Follow below steps:

1. Right click on Start button, select Control Panel.
2. Change the view by option on the top right to Large icons.
3. Click on Troubleshooting and click on the view all option on the left panel.
4. Run the System Maintenance troubleshooter.

Method 2: Disable the unnecessary Startup items

Make sure to kill processes that are not needed and disable the unnecessary Startup items via Startup in Task Manager.

1. Right click on Start and select Task Manager.

2. Click on Startup tab, right click on the process and click on Disable.

3. Restart the system and check if the issue is resolved.

Method 3: Update the Drivers

Most hardware interfaces require driver software to work properly. If you recently upgraded from one version of Windows to another, power outages, viruses, or other computer problems, it's possible that the drivers have become corrupted. Downloading and installing the latest driver can resolve these types of problems.

How to update drivers
