Sometimes failed print jobs get stuck in the print queue and just won't delete the way you would like...  so you have to do it yourself.

Here's how:

Sometimes when print jobs fail they clog up the print queue and they might not show up when you check to see what's printing.

In that case you need to delete the clogged up print jobs manually but first you have to stop the Print Spooler Service.

Here's two ways to do it:

You can open a Command Prompt as Administrator and do the work from there:

Then from the Command Prompt Window enter the following commands one at a time pressing the Enter key after each:

net   stop   spooler

del   /f/q   %systemroot%\system32\spool\printers

net   start   spooler 

exit

Any failed print jobs will need to be resubmitted.

If that makes you nervous here is another way to do the same thing:

Click the Start button/orb and in the box enter:

services.msc

Choose to Run as administrator:

With the Services dialog open find the Print Spooler Service, right click and choose to Stop it:

Close the Services applet.

Now that the Print Spooler Service is stopped you can delete any clogged up print jobs.

Open Windows Explorer and navigate to this folder, select all the files in the folder and choose to delete them:

Close all the windows and dialog boxes and choose to restart your system which will automatically restart the Print Spooler Service.

Any failed print jobs will need to be resubmitted.
