You have a very complex configuration.

If you have non-essential external devices attached to your computer during installation this can interfare with the setup engine and prevent the installation from completing because Windows 10 is trying to detect them. ActiveSync Error Code 86000209 - Microsoft Community

schmoov say:

ActiveSync Error Code 86000209

Why do I get this error during synchronization?

schmoov say:

Possible Cause:

The error code has to do with the sync key of the folder hierarchy. When you sync a number is exchanged between the device and data end. If you make say 10 changes between 10 different sync's than the device will send 10 back. What is happening is the device is sending back a number that is way off, say 4.


Possible Resolution:

The most common cause of this is resetting the device without adequately flushing the flash file system's RAM cache.

To avoid this in the future, press the suspend button before resetting the device. When the device goes into hibernate mode the cache is flushed. (you may want to wait a few seconds after hitting the button to allow the cache to be written).

With the smartphone, always turn off the phone, and never yank the battery. The ram cache is flushed to flash during the power off process.

User Action:

Try unselecting mail for sync. Then re-enabling it.


Thanks to Chris De Herrera for the source information in this answer: http://www.pocketpcfaq.com/faqs/activesync/exchange_errors.php

SumBoldIF formula - Microsoft Community

katrinab13 say:

SumBoldIF formula

Hi,

I have made a SumBold formula to sum all bold numbers in a set range (all of column B) using the below Visual Basic formula:

<pre>Function SumBold(rng As Range)
Dim rCell As Range
Application.Volatile
SumBold = 0
For Each rCell In Rng
If rCell.Font.Bold Then _
SumBold = SumBold + rCell
Next
End Function</pre>

Now I would like to sum all bold numbers in the set range (all of column B) IF column A has specific text (ex: "broker"). Without the bold condition, this would look like "=SUMIF($A:$A,"broker",$B:$B)" so I should be able to create a SumBoldIF formula using the same idea. What is the function I would use in Visual Basic? Thank you!

bernie deitrick say: =SumBoldIf(A1:A20,"broker",B1:B20)

Function SumBoldIf(rng1 As Range, varCnd As Variant, rng2 As Range) As Variant
    Dim lngR As Long
    Application.Volatile

    SumBoldIf = 0

    If rng1.Rows.Count <> rng2.Rows.Count Then SumBoldIf = "Range Error"

    For lngR = 1 To rng1.Cells.Count
        If (UCase(rng1.Cells(lngR).Value) = UCase(varCnd)) And rng2.Cells(lngR).Font.Bold Then
            SumBoldIf = SumBoldIf + rng2(lngR).Value
        End If
    Next lngR
End Function


Note that this does not expect an exact match: "Broker" will equal "broker" - if you want it to be case sensitive then take out the UCase functions
