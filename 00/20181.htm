suzanne s. barnhill say:

The method of inserting page numbers in Word 2013 is not fundamentally different from that in any other version. You should be able to insert a simple page number anywhere in a document (including the header or footer) using the shortcut Alt+Shift+P. You can then use Format Page Numbers to change the number style, restart numbering, etc.

For pages on which you wish to omit page numbering, you have two options:

  1. If the page that should not display a page number is the first page of a section, you can check the box for Different First Page on the contextual Header & Footer Tools | Design tab and then leave the First Page Header/Footer empty.
  2. If the page that should not display a page number is not the first page of a section, or if there are several pages to be left unnumbered, you will have to set those pages apart in a separate section with the header/footer unlinked from previous and following sections and leave it empty.

I have tried to avoid using the term "unnumbered" because these pages are not, strictly speaking "unnumbered." Every page in the document should be counted in the numbering even if it does not display the page number. It is best if the cover page is page 1, but if you really want to start numbering on the page after the cover, you can set numbering to start at 0 (provided you're using arabic numerals; roman numerals do not permit this).
