This is a deliberate default, intended to prevent the embarrassment that could result if you do not know that a document contains tracked changes. You can change the default, but only do so if you're absolutely certain it won't cause problems.

Click Trust Center.

Click Trust Center Settings...

Click Privacy

Clear the check box "Make hidden markup visible when opening or saving".

Click OK twice.
