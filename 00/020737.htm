If you had not given any alternate security info, unfortunately the old account is a lost cause. Microsoft had to tighten the recovery of Microsoft accounts due to attempted thefts from bad people stealing info from people, then using that info to attempt to get into the account.

entegy say:

Log into account.live.com/p with your new Microsoft account. This is the account security page. Add your phone number as security info here.
