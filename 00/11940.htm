1. Open your Internet Explorer.

6. Click Apply and Ok.
Close your browser and start a new session.

Sorry about the Luddite thing, but I can give you the steps for saving a GIF file from IrfanView.

Open the JPG version of the picture in IrfanView. Click File and choose Save As from the menu. In the dialog, open the "Save as type" dropdown and choose GIF.

You should also see a smaller dialog called "JPEG/GIF save options". If it isn't there, check the "Show options dialog" box at the bottom of the Save As dialog.

In the "JPEG/GIF save options" dialog, check the box for "Save transparent color" and click the option button for "Choose transparent color during saving".

In the Save As dialog, give a file name (and, if needed, a different folder) and click the Save button. When you do that, the preview will appear (the title says "Choose transparent color"). Use the mouse cursor to click somewhere in the picture that has the color you want to make transparent. The preview will immediately disappear.

The one thing that may be a bit confusing is that once this is done, IrfanView still shows you the original JPG picture. (In most programs, if you do a Save As, the original file closes and the new one opens.) If you want to see the GIF version, you'll have to open that separately.
