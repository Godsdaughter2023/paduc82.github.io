brian tillman [outlook mvp 2007-2019] say:

Do you have a digital signature (certificate)?  This is not a signature you might put at the end of a mail message that's equivalent to signing your name on a letter.  It is an attachment that can be used to prove electronically that you are who you say you are.  It can also be used to exchange encrypted messages with people.

Examine the header lines of the message as they appear in the Reading Pane or in the open message.  Do you see a little red ribbon symbol at the right end of a heaer that says "Signed by:" followed by the sender's address?  If so, this means your sender included a digital certificate to prove he is who he claims to be.  Outlook can be configured to digitally sign any reply you make to a message that was digitally signed.  If you've configured Outlook this way but do not have a digital signature, then you can see the error you see.

Does this happen only when you are replying or does it happen when you create and try to send a new message?  Click Tools>Options>Security.  You should see four check boxes at the top of that tab.  Are any checked?  If so, which ones?

    jay freedman say:

    If your registry doesn't contain the key HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\Word\Options, then your Office installation isn't correct. The similar key in the HKEY_LOCAL_MACHINE hive (without the Options subkey) is not meant to contain Word's options; the program will never look there for them. If you look again and still don't see the keys under HKEY_CURRENT_USER, try repairing the Office installation.

    An alternative is to avoid using the Start screen or the File > New screen at all. Instead, add the "New Document or Template" button to the Quick Access Toolbar. That will open the old-style New dialog box, which displays only templates in the %appdata%\Microsoft\Templates folder and any subfolders you create there. Also, in the Word Options dialog's General page, clear the check box for "Show the Start screen when this application starts".
