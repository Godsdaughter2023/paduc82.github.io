tryx3 say:

You need to create, for use as an equivalent to the Windows 7 System repair disk, either

1.1     A Win10 installation disk [on USB or DVD - 4GB for 32 bit, a bit over 4GB for 64 bit], or

1.2     A Win10 Recovery disk [on USB or DVD, 512MB]

2     You also need to make a system image on an external drive.

You can then, if all else fails, boot from the installation / Recovery disk to restore the image onto your HDD.

You made an installation disk [1.1] to install Win10 in the first place.  You can make a Recovery Disk [1.2] by going through Control panel, [View by - Small icons, if necessary], Recovery.  If you had a PC maker's recovery partition for Windows 10 {which nobody can have yet} then it would attempt to put that on the same drive as  the recovery tools so you need to uncheck that option on the second dialog of the procedure.

You can make a System image by going through Control panel, [View by - Small icons, if necessary], File history - you'll see the link in the bottom left corner.

Please note that both the installation disk and the Recovery Disk will take you into an almost-but-quite-not-quite-the-same Windows Recovery Environment as pressing F8 during bootup, just before Windows starts loading.  The advantage of using an installation disk or Recovery Disk is that you will be able to restore an image even when the HDD is completely corrupted or gets replaced. 

You must always test an  installation disk or Recovery Disk so you can be sure that they will work in an emergency.  I test them by booting up with them then working my way through the menus to reach the Troubleshooting, Advanced options and just opening a Command prompt.

greginmich say:

There isn’t any Local Group Policy Editor in the Home Version of Windows 10, so you might need to use the Group Policy registry switch in order to turn on Windows Defender. You can toggle Defender on or off directly (and very quickly) by running the appropriate REG command –

Turn Off Windows Defender:

REG ADD "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware /t REG_DWORD /d 1 /f

Turn On Windows Defender:

REG DELETE "HKLM\SOFTWARE\Policies\Microsoft\Windows Defender" /v DisableAntiSpyware

If that’s too simple for your liking, then you can download a REG file that will perform the identical registry operation:

https://www.tenforums.com/tutorials/5918-windows-defender-turn-off-windows-10-a.html   

The Ten Forums Turn_Off_Windows_Defender.reg file is just an exported copy of the text in the modified registry key that can be merged once it's given a .reg extension. This is all there is to it:

Windows Registry Editor Version 5.00

; Created by: Shawn Brink

; http://www.tenforums.com

; Tutorial: http://www.tenforums.com/tutorials/5918-windows-defender-turn-off-windows-10-a.html

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender]

"DisableAntiSpyware"=dword:00000001

And the Turn_On_Windows_Defender.reg file just clears that registry entry:

Windows Registry Editor Version 5.00

; Created by: Shawn Brink

; http://www.tenforums.com

; Tutorial: http://www.tenforums.com/tutorials/5918-windows-defender-turn-off-windows-10-a.html

[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender]

"DisableAntiSpyware"=-
