devlin sco say:

Hi Stanis,

An unknown background program is possibly using the device. You can perform a clean boot to determine which program is causing the error message to appear. A clean boot is performed to start Windows by using a minimal set of drivers and startup programs. This helps eliminate software conflicts that occur when you use your input devices. You will need administrative permission to perform this isolation method.

  1. Click the Start button, type System Configuration and then press Enter.
  2. If prompted, choose Yes or enter the administrator password.
  3. On the Services tab of the System Configuration window, select the Hide all Microsoft services check box, and then click Disable all.
  4. On the Startup tab of the System Configuration window, click Open Task Manager.
  5. On the Startup tab in Task Manager, for each startup item, select the item and then click Disable.
  6. Close Task Manager.
  7. On the Startup tab of the System Configuration dialog box, tap or click OK, and then restart the computer.

For more details about performing a clean boot in Windows 10, please watch our YouTube video
