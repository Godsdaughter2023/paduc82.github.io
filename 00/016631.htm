I can not agree, the END key feature (press END, then an arrow key for the direction) is in Excel since version 5.0!?

Ctrl+Arrow Key moves to the edge of the current data region in a worksheet. Have a look here for more shortcuts:

https://support.office.com/en-us/article/Excel-keyboard-shortcuts-and-function-keys-for-Windows-1798d9d5-842a-42b8-9c99-9b7213f0040f?omkt=en-US&ui=en-US&rs=en-US&ad=US#ID0EABDAAA=2013

Anyway, you can overwrite an existing shortcut with a macro, copy this into a regular module in your PERSONAL.XLSB file:

Sub MyEndKey()
  ActiveCell.Offset(, Columns.Count - ActiveCell.Column).End(xlToLeft).Select
End Sub

And copy this into the code module "ThisWorkbook" in your PERSONAL.XLSB file:

Private Sub Workbook_Open()
  Application.OnKey "{END}", "MyEndKey"
End Sub

Close and reopen Excel.

  • Go to Page Design - Change Template
  • Click on Home, then Calendars
  • Choose a random calendar template (doesn't matter which one). Timeframe stays at 'One month per page'.

Customize the sort option within the function.

Public Function AddNewShape(r As Range)

'

Const loName As String = "tbl_01"

Const shpName As String = "Rectangle_01"

'

Dim lo As ListObject

Dim shp As Shape

Dim N  As Integer

'

'1) Delete old shape if exists

N = ActiveSheet.Shapes.Count

If N > 0 Then

For Each shp In ActiveSheet.Shapes

If shp.Name = shpName Then shp.Delete

Next

End If

'

'2) Sort Table

Set lo = ActiveSheet.ListObjects(1)

lo.Name = loName

Set r = lo.ListColumns(1).DataBodyRange

With lo

.Sort.SortFields.Clear

.Sort.SortFields.Add _

Key:=Range(loName & "[data]"), SortOn:=xlSortOnValues, Order:=xlAscending, DataOption:=xlSortNormal

With .Sort

.Header = xlYes

.MatchCase = False

.Orientation = xlTopToBottom

.SortMethod = xlPinYin

.Apply

End With

End With

'

'3) Add Rectangle

Set shp = ActiveSheet.Shapes.AddShape(msoShapeRectangle, 400, 50, r.Item(1).Value, r.Item(2).Value)

With shp

.Name = shpName

.Line.Visible = False

.Fill.ForeColor.RGB = RGB(Int(Rnd() * 255), Int(Rnd() * 255), Int(Rnd() * 255))

End With

'

AddNewShape = "AddNewShapeFunction"

End Function

Hi Heimo,

Thank you for posting your query on Microsoft Community. Let me assist you.

As you are getting an error when you try to use the elevated command prompt as it would require Admin access. As you can’t access that on your system, I suggest you to go to Windows Recovery Environment and use the Command Prompt there, which is elevated and enable the hidden admin on the system to check if it helps.

Follow the below steps:-

1)    Press Shift and Restart from the power icon (together)

2)    Select Troubleshoot.

3)    Go to Advanced Options.

4)    Select Command Prompt.

5)    Type "net user Administrator /active:yes"

6)    Hit Enter.

A user account with the name Admin will be created. Log in with that account and you will be able to access Admin privileges. You can also create more Administrator accounts with that account.

Further, Run an Startup Repair on the system

This is an alternate way to boot the PC and run recovery tools when something goes wrong with the Operating system.

Please follow the steps given below. Insert the Windows 10 installation DVD in the DVD ROM and restart your computer.

a) After you boot your computer using Windows DVD or System Repair Disc, a black screen appears with gray text "Press any key to boot from CD or DVD". Press any key.
b) Select the correct time and Keyboard type.
c) Click Repair your computer in the lower left corner.
d) Select Troubleshoot from Choose an option screen.
e) Click Advanced options in Troubleshoot screen.
f) Click Startup Repair from Advanced Options screen.

g) Windows will then look for problems on hard drive(s) and verify that all required files are intact.

You can use the USB drive if it has the Windows 10 image.
