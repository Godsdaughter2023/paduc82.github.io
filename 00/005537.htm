Hi Cindy,

Sub ApplyAllHeading1()
    Dim doc As Word.Document
    Dim rng As Word.Range
    Set doc = ActiveDocument
    Set rng = doc.Content
    With rng.Find
        .ClearFormatting
        .Replacement.ClearFormatting
        .Format = True
        .Forward = True
        .Text = "Purpose"
        .MatchCase = False
        .MatchWholeWord = True
        .Replacement.Text = "PURPOSE"
        .Replacement.Style = doc.Styles("Heading 1").NameLocal
        .Wrap = wdFindStop
        .Execute wdReplaceOne
    End With
End Sub

rich michaels - mvp (greatcirclelearning.com) say:

Hi Cindy,

If you want every occurrence of "purpose" changed to "PURPOSE" with as Heading 1 style. regardless of current case setting, current style, or location in the document, then this will work.

Sub ApplyAllHeading1()
    Dim doc As Word.Document
    Dim rng As Word.Range
    Set doc = ActiveDocument
    Set rng = doc.Content
    With rng.Find
        .ClearFormatting
        .Replacement.ClearFormatting
        .Format = True
        .Forward = True
        .Text = "Purpose"
        .MatchCase = False
        .Replacement.Text = "PURPOSE"
        .Replacement.Style = doc.Styles("Heading 1").NameLocal
        .Wrap = wdFindContinue
        .Execute Replace:=wdReplaceAll
    End With
End Sub
 

Edit ... If you only want to match whole words so that you don't get "Purposeful" altered, then add 

        .MatchWholeWord = True

into the With statement.

Integer and Long Integer are used when there is no need for decimals in calculations. In other words whole numbers. The difference is that Long Integers can hold a larger number.

Currency is used for Currency calculations. It automatically shows the currency sign according to regional settings. It is accurate to 4 decimal places.

Single and double are for floating point calculations. The difference is in the accuracy and size of the numbers stored.

Decimal is for larger numbers that require decimals.
