tom ogilvy say:

CatherineYKF,

If I have a tab delimited file that looks like this:

file1.txt

Header1    Header2    Header3    Header4
G    6    H    30
Z    4    G    26
H    6    J    64
G    4    Z    8

(entries are separated by tab characters).

I then open it in excel using the text import wizard and choose delimited with tab as the delimiter, then I will get these entries in 5 columns and 5 rows.   If I then save the file as a CSV file it will look like this in notepad:

Header1,Header2,Header3,Header4
G,6,H,30
Z,4,G,26
H,6,J,64
G,4,Z,8

The code below will do that for each .txt file in the folder   "C:\Data\Files\"   making a copy of the file and replacing the .txt extension with .csv 

Sub ConvertFiles()
Dim sName As String, sPath As String
sPath = "C:\Data\Files\"  '<== change to reflect your folder.  Make sure it ends with a "\" character
sName = Dir(sPath)
Do While sName <> ""
  If LCase(Right(sName, 3)) = "txt" Then
     Workbooks.OpenText Filename:=sPath & sName, _
        Origin:=437, _
        StartRow:=1, _
        DataType:=xlDelimited, _
        TextQualifier:=xlDoubleQuote, _
        ConsecutiveDelimiter:=False, _
        Tab:=True, _
        Semicolon:=False, _
        Comma:=False, _
        Space:=False, _
        Other:=False, _
        TrailingMinusNumbers:=True
     ActiveWorkbook.SaveAs Filename:=sPath & Left(sName, Len(sName) - 4) & ".csv", _
          FileFormat:=xlCSV, _
          CreateBackup:=False
     ActiveWorkbook.Close SaveChanges:=False
  End If
  sName = Dir()
Loop
End Sub

--

If that isn't what you want, then it just sounds like you want to rename your .txt file with a .csv extension and not change the contents to be what is expected in a true .csv file (US English).

Hi

I've been tracking complaints about Excel speed. This is what has been reported on the topic so far:

  • Copy (Cmd+C) causes spinning beach ball delay for 3-4 - Microsoft Community (HandOff issue)
  • Another user reported the problem goes away in a new Mac OS user account. A corrupt user account can cause Excel to be slow
  • Zoom – set zoom to 100% or less
  • Another user reported the problem goes away when menu manager is turned off. The name of the manager was not disclosed. The general case is that a 3rd party application intercepts keyboard and mouse input, performs actions, and then passes the keyboard and mouse activity on to Excel resulting overhead and delay.
  • A different user reported uninstalling QuicKeys input manager fixed the problem.
  • Page Layout view is slow. Use Normal View, instead
  • Another user reported that inserting pictures caused Excel to be slow and/or hang

There does not seem to be one thing in particular going on that causes slowness in Excel. You can use the above list to rule things out, but there may be other causes.
