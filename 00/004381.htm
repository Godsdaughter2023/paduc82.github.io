Excel VBA DO UNTIL LOOP to copy ranges into master worksheet - Microsoft Community

dawn b say:

Excel VBA DO UNTIL LOOP to copy ranges into master worksheet

I'm working on an Excel 2007 workbook with individual worksheets to track project hours by employee. The structure of each worksheet is the same including the data range, for example, C5:G25 but the number of worksheets will vary as employees leave or are hired. I'm looking for help, links, or examples with a DO UNTIL LOOP to:

  1. Copy a specified range from an employee worksheet into a master worksheet
  2. Move to next available position in the master worksheet for next range
  3. Move to the next worksheet and repeat for each employee

Don't want to use range names as worksheets will be added, moved, and deleted. Data from master worksheet will be used by multiple PivotTables to analyze workload, etc.

Client is comfortable making a simple edit to update the count # as needed.

Thanks in advance for any direction and resources.

Dawn

ossiemac say:

The following is an example of copying a fixed range from each sheet to a sheet named Master. The sheet named Master is excluded from the copy with the If/End If. However it might need modification; one being if there is more than one sheet that is not to be copied then Or statement needs to be added in the If/End If section.

Should give you the idea of how to get started. If the Master sheet is not named "Master" then edit the code to your sheet name.

I have copied the range C5:G25 as per your example and pasted it into the matching columns of Master. If not to be in matching columns then just edit the "C" (Column Id) in the set wsDest line.

Sub CopyToMaster()
  Dim ws As Worksheet
  Dim wsDest As Range   'Destination cell
 
  For Each ws In Worksheets
    If ws.Name <> "Master" Then   'Don't copy Master sheet
      'Find next availabe cell in column C of Master and assign to variable
      With Sheets("Master")
        Set wsDest = .Cells(.Rows.Count, "C").End(xlUp).Offset(1, 0)
      End With
      'Copy the range and paste into next availabe cell in Master
      ws.Range("C5:G25").Copy Destination:=wsDest
    End If
  Next ws
End Sub

hal hostetler - mvp-outlook say:

.OST files are not sharable, if that's what your asking, .OST files are volatile mirror image of your online Exchange server mailbox complete with all folders, files, contacts, calendar, etc..  Its sole propose is to give your something to work with when offline and unconnected.  It can be completely deleted and Outlook will rebuild it for you the next time it starts.  As to the Exchange Server, itself, it can be no older than Exchange 2010 since this is the oldest version that Outlook 2016 will work with.  Also, Outlook 2016 cannot be setup manually to work with the Exchange Server, It requires Auto Account Config and the Exchange Server must have AutoDiscover records associated with it.  Please see
