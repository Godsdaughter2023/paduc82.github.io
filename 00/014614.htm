Regarding “…the length of time items remain in the Deleted Items folder is extended to indefinitely…” in the article, which means that we have the solution to extend the length of retention time to indefinitely, not directly modifying the value of RetainDeletedItemsFor parameter  

By default, An Exchange Online mailbox is configured to retain deleted items for 14 days. You can use the Exchange Management Shell to change this setting to a maximum of 30 days. If you need to retain deleted items for longer than 30 days, you can place the mailbox on In-Place Hold or Litigation Hold.
For your requirement, if you want to retain deleted items 180 days for all users, you may use these PowerShell commands below:
Get-Mailbox -ResultSize Unlimited -Filter {RecipientTypeDetails -eq "UserMailbox"} | Set-Mailbox -LitigationHoldEnabled $true -LitigationHoldDuration 180

Note: if you want to place more than 500 users on In-Place Hold, please use the Exchange Management Shell. See New-MailboxSearch.
For example, if you want to place 10000 user’s mailbox on In-Place Hold, you may create a DG (distribution group) testDG and run these commands.
New-MailboxSearch -Name "Hold-ProjectX" -SourceMailboxes testDG -InPlaceHoldEnabled $true
