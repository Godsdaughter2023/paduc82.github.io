andre for directly say:
Hi Power

My name is Andre Da Costa; an Independent Consultant, Windows Insider MVP and Windows & Devices for IT MVP. I'm here to help you with your problem.

Option 1:

During OOBE, press Shift+F10 to bring up the command prompt.

To change IP and default gateway:

netsh int ip set address "local area connection" static 192.168.0.101 255.255.255.0 192.168.0.254 1

To change DNS:

netsh int ip set dns "local area connection" static 192.168.0.254 primary

Option 2:

Complete setup, after reaching the desktop, click Start > Settings > Network & Internet > Status > Network and Sharing Center

Click Change adapter settings
Right click the network connection
Click Properties

    bill smithers say:

    The idea of the recycle bin is a place to keep deleted files until you have had sufficient time to decide whether deleting those files was a mistake or the correct decision.

    If you come to the conclusion that you definitely do not want those files, then you permanently delete them by emptying the recycle bin.

    But the fact is that those "deleted" files still exist until they are overwritten by other files.

    This is an interesting article about it:

    Why Deleted Files Can Be Recovered, and How You Can Prevent It

    When you delete a file, Windows removes the pointer and marks the sectors containing the file’s data as available. From the file system’s point of view, the file is no longer present on your hard drive and the sectors containing its data are considered free space.

    However, until Windows actually writes new data over the sectors containing the contents of the file, the file is still recoverable. A file recovery program can scan a hard drive for these deleted files and restore them. If the file has been partially overwritten, the file recovery program can only recover part of the data.
