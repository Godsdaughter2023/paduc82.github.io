bob jones aka: cybertaz say:

Restating what Jim explained... The term 'docked' now has 2 separate meanings :-) It can pertain to anchoring a toolbar within the document window or it can pertain to anchoring a toolbar to the edge of the screen outside the document window.

In Office 2011 only the built-in Standard & Formatting toolbars can be docked within the document window. The Standard toolbar can be displayed in only that way - it isn't allowed to float, whereas the Formatting toolbar can have its Docked property turned off.

Toolbars you create can be floating, only - they cannot be docked within a document window. However, even though it may seem contradictory terminology, Floating toolbars can be 'docked' [i.e., anchored] to the edge of the screen outside of the document window.

It's also worth noting that the toolbars docked within the document window are replicated for each open document, but there is only one instance of those which float.

Word 2010 & 2013 Fonts - Microsoft Community

stace6 say:

Word 2010 & 2013 Fonts

The font Calibri Light works great on word 2013 but does not appear on word 2010.

Will it be possible to add this font to machines with word 2010 installed?

jay freedman say:

It's just a matter of copying the two font files from the machine with Office 2013 and installing them on the machines with Office 2010.

  • On the Office 2013 machine, use Windows Explorer to open the folder C:\Windows\Fonts\Calibri.
  • Copy the Calibri Light file and the Calibri Light Italic file to a USB memory stick, a network folder, or some other transfer medium. (Windows Explorer does its best to hide the actual file names, which are CalibriL.ttf and CalibriLI.ttf.)
  • On the Office 2010 machines, right-click one of the files and click Install. Repeat for the other file.

Once the fonts are installed, they'll be available in all programs that use fonts.

Windows Defender on Windows 8 integrates a more robust version of Windows Defender (and uses that name) for its anti-virus and anti-malware protection. Windows 8 Defender will remain disabled until the third party anti-virus has been completely uninstalled and then Windows 8 Defender needs to be activated if you choose to use it.

bill manville say:

So there are 2 possibilities:

- the file does contain a virus - in which case you might not please your boss by passing it on to him
- the file does not contain a virus but your virus checker thinks it does

Either way, I would let the virus checker check my files and take note of what it says.

If you are convinced the file is clean then find another way to give it to your boss:
- put it on a server you can both see
- put it on a website you can both see (e.g. Skydrive)
- zip it and email the zip file if your checker will allow
- disable your virus checker and send it
