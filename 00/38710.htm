I'm Greg, a volunteer installation specialist and 8 year Windows MVP, here to help you.

Did this begin after updating to Fall Creators Version 1709? Give it more time to mature.

There is also a new setting in Version 1709 that restricts folder access which is supposed to be set at Off by default but may have been turned on by an overly aggressive bloatware anti-virus.
