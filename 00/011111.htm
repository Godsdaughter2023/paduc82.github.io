Lets clarify some terminology, so we are on the same page.

Download: copy file from an outside source, like the internet or USB drive, to your local computer, typically saving to your hard drive. But a “download” could also be simply viewed. For example, your browser “downloads” all of the files/websites you view.

 

Install: make a program usable on a computer. Most Windows programs require a special setup program / file that copies the program files to various locations, creates registry entries and creates shortcuts in the start menu or desktop.

 

Activation / Registration : Activations is a MS process that proves to MS that you have a legal license which is called a "Product Key".  Many other companies call this "Registration". Actually, MS also uses the term Registration for the old MS Works suite.

 

Product Key: is a UNIQUE 25 character, written in 5 groups of 5, code. The PK is your “proof of purchase”. Protect your PK! Keep records of it If you “lose” your PK, you have lost your copy of Office!  Note: for MS Works, they call it the “Product ID”. Other makers often call it the “Registration Number”

 

 

Registration: MS Registration gives you no real additional benefit. Is a source of propaganda / sales e-mail from MS. 

 

Product ID: In Office applications you can see the Product ID in the Help About dialog. The unique Product ID is “one-way” generated during the Activation process.

 

So the Office installation process is:

·      You "download" the "installation" file from the online store.

·      During the “download” you “Save to HD”.

                  OR

       insert the installation disk into the CD player


·      then you run the "installation" / Setup.exe file to setup Office on your computer.

·      Depending on your version of Windows, you may have to use “Run As Administrator” to start the installation process to get all of the required access rights.

·      Part of the installation process includes the option to "Activate" the Office installation using the "Product Key". If you do not “Activate” during the install, your Office installation will be in a time limited “trial mode” until it is activated

After the installation is done, save the installation file to someplace safe: external HD(s), CD/DVD(s) (more than one copy is a good idea) You want to keep the installation file safe so you don’t have to depend downloads in future. For example, people with Office 2003 can’t download now.

rohn007 say:
OK, you bought an online download. That means you most LIKELY have a "Product Key Card" / "PKC" type license. It would have probably cost around US$100-120.  There is a small chance you got a "Retail" / "FPP" type license that would have cost around US$150-160.

The point is if you have a PKC type license you cannot transfer it to a new computer. PKC licenses are "disposable". They are permanently tied to the first hardware they are installed on.  When the hardware dies, so does the license. When you sell the computer, Office has to go with it.

If you don't remember how much you spent, or are not completely sure which type of license you have (if it is not explicitly stated on your confirmation email), you should call the phone activation center.  Have your Office Product ID from the installed computer as well as the Product Key handy. When you call in, don't reply to the initial prompts, wait until you are offered the option of talking to a live "service" rep. Ask them to tell you what type of license the key is associated with.

Once you confirm you have a retail license, we can continue with the steps you have to take. In short form the steps are:

  • check your browser downloads. CTL J works in IE and FireFox
  • the end of the file name may look SOMETHING like this  X17-75161.exe or you recognize the download site
  • If you find it, Run it to start the installation

Synchronizing with Exchange Server (SBS 2003) - Microsoft Community

wdpc say:

Synchronizing with Exchange Server (SBS 2003)

I have attempted to sychronize with my Exchange Server with my PPC running WM 5.0 without success.  Can anyone provide me with step by step detailed instructions (in "layman's" English please) that will allow me to do this.  I am able to configure the Server (me thinks) but can never successfully synchronize.  Any words of wisdom would be appreciated.

elrendhel say:

Greetings WDPC!  Welcome to the Windows Mobile forums!

 

Here are the steps that I use to setup corporate clients.  This assumes that the Exchange Server is setup correctly:

 

  1. On your PocketPC, please go to Start > ActiveSync.
  2. On the Tools menu, please select Options.
  3. On the Server Tab,
    • place your cursor in the "Sync with this server" box,
    • then enter the correct FQDN (Fully Qualified Domain Name) or IP address for your Exchange Server.
    • If the server uses SSL, please make sure you select the SSL check-box.
    • Click on the Options button.
      • Enter your credentials (username, password, domain name), and select Save Password.
      • In the Rules tab, please make your selections as to:
        • the frequency of ActiveSync sessions,
        • what process to observe in case of a item conflict,
        • the level of logging.
  4. In the "Sync these items box, select the items you wish to synchronize with the server:
    • Calendar
    • Contacts
    • Inbox
    • NOTE: The Calendar and Inbox choices have additional Settings which can be set: the size of e-mails to download, whether to download the whole body or just headers, etc).
  5. In the Mobile Schedule tab make your selections as to the frequency of the ActiveSync operations, and select which days and what hours are considered as "Peak Times". When done click Ok.
  6. Now press the Sync button on the main ActiveSync page on your PPC.

If you programmed everything else correctly, you should get a confirmation that your PocketPC is actually synchronizing with your Exchange Server.

jim gordon mvp say:

Make sure your Mac has a working internet connection before checking for updates. Install Mac OS X updates followed by Office updates.

 

Install Mac OS X updates first:

To get Mac OS updates, go to the App store on your dock, and then click the Updates button to reveal available updates.

 

Then Install Office Updates

To get the update:

  1. Open any Office application
  2. Go to the Help menu and choose Check for Updates
  3. Let AutoUpdate check for and install updates
  4. Restart your Mac

 

If you have any add-ins such as WebEx, Mendelay, EndNote, Zotero, TypeIt4me, etc it is essential that you install the latest updates for your add-ins.
