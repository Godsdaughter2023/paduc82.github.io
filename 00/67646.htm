This is usually the case with a Office update, it will reset Outlook as the default email handler and assign it all available defaults.  You'll need to go back to 'Default Programs|Set your default programs' and reset Thunderbird as default and that it has all available defaults assigned to it.  It's usually a good idea to also set Thunderbird as default under 'Default Programs|Set program access and computer defaults' after finishing up on the 'Set your default programs' tab per Ramesh's post.
