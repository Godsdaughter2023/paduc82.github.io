If you want dates without the Sat and Sun then enter the start date in the first cell (In your example in cell A1 enter 1/10/2017)

=NOT(WORKDAY(A1-1,1)=A1)

To return True for workdays and False for Sat and Sun use the following

=WORKDAY(A1-1,1)=A1
