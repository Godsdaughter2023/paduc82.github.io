jess bor say:

Hi Kossiossi,

SuperFetch can help decrease boot time and make programs more efficiently. However, this has been identified as a potential cause of disk performance issues in Windows 10. To stop this service follow these steps:

1. Press the Windows key + X, then choose Command Prompt (Admin).
2. Click Yes at the User Account Control prompt.
3. Once the command prompt appear, type in the following command:

net.exe stop superfetch

4. Click Enter.
5. Check your computer’s disk usage: Type task manager in the Windows search bar and select Task Manager, then in the Processes tab, look at the % at the top of the Disk column. If it’s no longer 100%, issue is resolved.

If issue is not resolved, you may consider the next method.

Perform a disk check by following these steps:

1. Press Windows key + X, then choose Command Prompt (Admin).
2. Click Yes at the User Account Control prompt.
3. Once the Command prompt shows, type the following command:

chkdsk.exe /f /r

4. Click Enter, then type Y to confirm that you would like to perform the disk check the next time you restart your computer. Make sure to close all of your applications first.

Note: Disk check will start the next time you boot your PC and this may take some time to complete (could be a day for some). If, you don’t have time to wait for the disk check to complete, you can skip it. You’ll need to reschedule it.

5. After completing the disk check, check your computer’s disk usage: Type task manager in the Windows search bar and select Task Manager, then in the Processes tab, look at the % at the top of the Disk column. If it’s no longer 100%, issue is resolved.

Let us know the result for us to help you further.

Windows 8 64 bit is more secure too, malicious code cannot easily infiltrate it, drivers are more reliable since they must be signed before they can work with 64 bit Windows 8.
