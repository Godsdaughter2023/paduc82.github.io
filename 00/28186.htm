rob koch say:

Though this gets blamed on Microsoft Security Essentials because people don't understand how the updating actually works, it's really a problem with how an individual has their Windows Automatic Update configured in Windows 7.

Those postings are correct that something has changed though, since MSE first had this problem when it actually began including the ability for it to update via Windows Update as well several years ago and some users began to complain about the update icon being displayed multiple times daily.

At the time this was due to the fact that Microsoft had changed the MSE definitions update packages from optional to a much higher priority similar to critical, which caused them to display rather than install silently when MSE itself requested them as it always had.  This issue only presented for those using the "Notify" settings option in Windows Automatic Update, since anyone allowing automatic updates to occur without notification didn't see this and eventually Microsoft made additional changes that got rid of the problem.

What's happened this time is something similar, though slightly different in that it relates to how recommended updates are configured.  As I recall from another thread like this, if the setting to “Give me recommended updates the same way I receive important updates” is selected, this triggers the updates waiting to install flag the same way it did back when the earlier changes were made.

I don't have a Windows 7 system available anymore to confirm this, but this implies that the MSE definitions update packages have likely been changed to recommended rather than whatever they had been for several years.  Though on the surface this sounds reasonable, it's the complex interrelationships between the built-in MSE update system and the several Windows Update option settings that combine to create these types of issues.

No one is trying to force anyone into a particular update configuration, though this does indicate that some within Microsoft's security arm assume that sensible people have configured their system's updates to be completely automatic.

However, the paranoid who feel they need to control everything still sometimes choose to perform updates manually.  This is the innocent, but relatively obvious clash between those two opposing assumptions.

Though it may not seem the optimum solution, the easiest way around this current problem is to simply turn off the setting to “Give me recommended updates the same way I receive important updates”, though that means you'll need to review the optional update screen within Windows Update to select any recommended updates manually.  However, if you're a control freak who wants to inspect everything manually, which is what enabling the "Notify" setting implies, then isn't this what you'd typically want to do anyway?

In any case, this situation will only continue until January 14, 2020, since after that date Windows 7 ends extended supportfor either critical security updates and likely MSE definitions as well, though the latter have been extended for a few months when past Windows versions reached their End-of-Life.

I personally wouldn't expect this situation to change, since support for Windows 7 has already been deprecated to some extent in these forums and all true effort is being placed in Windows 10, which is the only full version of Windows still in the mainstream support phase.

Rob

Like this. Change the array arrColumns as needed.


Sub CopyFromWorksheets()
    Dim arrColumns As Variant 'Array of columns
    Dim wrk As Workbook 'Workbook object - Always good to work with object variables
    Dim sht As Worksheet 'Object for handling worksheets in loop
    Dim trg As Worksheet 'Master Worksheet
    Dim rng As Range 'Range object
    Dim col As Variant 'Column
    Dim c As Long
    Dim r As Long
    Dim m As Long
 
    ' Specify the columns
    arrColumns = Array("A", "C", "F", "H", "W", "Z", "AA", "AE", "AK")
 
    Set wrk = ActiveWorkbook 'Working in active workbook
    For Each sht In wrk.Worksheets
        If sht.Name = "Master" Then
            MsgBox "There is a worksheet called as 'Master'." & vbCrLf & _
            "Please remove or rename this worksheet since 'Master' would be" & _
            "the name of the result worksheet of this process.", vbOKOnly + vbExclamation, "Error"
            Exit Sub
        End If
    Next sht
 
    'We don't want screen updating
    Application.ScreenUpdating = False
 
    'Add new worksheet as the last worksheet
    Set trg = wrk.Worksheets.Add(After:=wrk.Worksheets(wrk.Worksheets.Count))
    'Rename the new worksheet
    trg.Name = "Master"
    'Get column headers from the first worksheet
    Set sht = wrk.Worksheets(1)
    c = 0
    For Each col In arrColumns
        c = c + 1
        With trg.Cells(1, c)
            'Copy header
            .Value = sht.Cells(1, col).Value
            'Set font as bold
            .Font.Bold = True
        End With
    Next col
 
    'Loop through sheets
    r = 2
    For Each sht In wrk.Worksheets
        'If worksheet in loop is the last one, stop execution (it is Master worksheet)
        If sht.Index = wrk.Worksheets.Count Then
            Exit For
        End If
        m = sht.Cells.Find(What:="*", SearchOrder:=xlByRows, SearchDirection:=xlPrevious).Row
        'Loop through columns
        c = 0
        For Each col In arrColumns
            c = c + 1
            'Data range in worksheet - starts from second row
            Set rng = sht.Cells(2, col).Resize(m - 1)
            'Put data into the Master worksheet
            trg.Cells(r, c).Resize(m - 1).Value = rng.Value
        Next col
        r = r + m - 1
    Next sht
    'Fit the columns in Master worksheet
    trg.Columns.AutoFit
 
    'Screen updating should be activated
    Application.ScreenUpdating = True
End Sub

Hi,

Actually, there is a way to generate PDF files including hyperlinks with Word 2016 for Mac :

1 - Insert hyperlinks in your document,

2 - Go to "Save as...", choose PDF and check the box "Best for electronic distribution and accessibility" instead of "Best for printing",

3 - Save. It will take a while as it uses Microsoft online services but then the generated PDF file includes hyperlinks!



Let me know if that helps.
