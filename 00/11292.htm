macropod say:

You could use:

In your «Section_Name» {=«Very_Helpful»/«Total Respondents»*100 \# 0.00%"} of respondents indicated Very Helpful.

 

Note: The field brace pairs (ie '{ }') for the above example are created in the body of the document via Ctrl-F9 (Cmd-F9 on a Mac) - you can't simply type them or copy & paste them from this message. Likewise, you can't type or copy & paste the chevrons (ie '« »') - they're part of the actual mergefields, which you can insert from the mailmerge toolbar.

 

Since you apparently have multiple records per person, you may not want to send them a separate email for each record. In that case, you can use Word's Catalogue/Directory Mailmerge facility (the terminology depends on the Word version). To see how to do so with any mailmerge data source supported by Word, check out my Microsoft Word Catalogue/Directory Mailmerge Tutorial at:

http://windowssecrets.com/forums/showthread.php/154370-Microsoft-Word-Catalogue-Directory-Mailmerge-Tutorial

or

http://www.gmayor.com/Zips/Catalogue%20Mailmerge.zip

The tutorial covers everything from list creation to the insertion & calculation of values in multi-record tables in letters. Do read the tutorial before trying to use the mailmerge document included with it.
Alternatively, you may want to try one of the Many-to-One Mail Merge add-ins, from:
Graham Mayor at http://www.gmayor.com/ManyToOne.htm; or
Doug Robbins at https://skydrive.live.com/?cid=5aedcb43615e886b#!/?cid=5AEDCB43615E886B!cid=5AEDCB43615E886B&id=5AEDCB43615E886B%21566

doug robbins - mvp office apps & services (word) say:

Use the following field construction

 

{ = { MERGEFIELD "Very Helpful" }/{ MERGEFIELD "Total Respondents" } * 100 \# "0.00%" }

 

You must use Ctrl+F9 to insert each pair of field delimiters { }

 

See fellow MVP Macropod's Word Field Maths Tutorial at:

http://www.gmayor.com/downloads.htm#Third_party

Macro to link text boxes in microsoft word

I know how to link two textboxes together so that text will flow from one to the next.  I would like to do this using a macro.  Does anyone have appropriate code for this?  I am referring to Microsoft Word.

greg maxey say: Something like this:

Sub ScratchMacro()
Dim oShp1 As Shape, oShp2 As Shape
Set oShp1 = ActiveDocument.Shapes(1)
If oShp1.TextFrame.Overflowing = True Then
 Set oShp2 = ActiveDocument.Shapes. _
   AddTextbox(msoTextOrientationHorizontal, 72, 72, 100, 200)
 oShp1.TextFrame.Next = oShp2.TextFrame
End If
End Sub
