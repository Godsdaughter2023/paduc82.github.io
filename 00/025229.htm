Facbook created accounts were discontinued in January 2018. All other accounts were migrated over. Even those accounts do not retain any cloud data beyond mid April 2017. Due to Facebook created accounts being retired, with no ability to sign into them, even data in that range is inaccessible. You should be able to still pull that information from your local history file (main.db) located in a facebook#id  folder in your %appdata%/Skype folder (in Windows). OK, a bit of digging yields the following conclusion: it's a long-standing bug, supposedly fixed in the various Word 2008 updates, but not quite.

  • Copy equations from old backups. This seems to work "mostly" (could be so, if the AutoRecover corruption is real).
  • Save equations separately and link to them in the main document.
  • Some users blame the .docx format, but does not seem likely.
