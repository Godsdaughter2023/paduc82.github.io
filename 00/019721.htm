Backups

Final piece of advice. For a thesis,

A common problem we see on this forum around the end of school term (ie December and April) is a plea for help because a valuable file, thesis or major end of term assignment, has become corrupt and the person is on a very short deadline to submit it.  Yikes.

The solution, or at least a major workaround, is to make lots of incremental backup copies of the file.  If you use OneDrive, then this is done partly for you. But that is NOT enough!  You should keep at least some copies on one or even 2 different media, ie your local hard drive AND a USB drive.  At the very least, in this sort of situation I keep "end of day" backup copies of the file on separate media. I add a datetime stamp to the end of the file name ie

Filename1 190101 2400.docx

Filename1 190102 2400.docx

Actually I use macros that constantly do timed backups for me in 5-10 minute intervals. This is separate from the autobackup feature in Word. The macro automagically does a SAVEAS with new file name every x minutes.  Yes, this quickly uses up a lot of disk space with large documents. So I have to go back and "thin" the saved files periodically.

The point is, I like to have lots of separate backup file versions so that I can fall back to the last good file copy, even if that happens to be several generations back.

Side-by-side printing on same page - Microsoft Community

coryschneider say:

Side-by-side printing on same page

I want to have 2 multipage documents printed on the same pages side-by-side.  Using columns doesn't work because the first document appears (in 2 columns) and is then followed by the second.

doug robbins - mvp office apps & services (word) say:

I have not tested this macro, but if you insert the relevant path and filenames of the 2 documents into the Set DocA and Set DocB lines, it should when you run it, create a new document in which the pages are taken alternately from those in DocA and DocB. 

 

When you then print that document two pages to a sheet, you should then have what you want.

 

Dim DocA As Document, DocB As Document, DocTarget As Document
Dim rngA As Range, rngB As Range
Dim i As Long
Set DocA = Documents.Open("Path\Filename of the first document")
Set DocB = Documents.Open("Path\Filename of the secong document")
Set DocTarget = Documents.Add
For i = 1 To DocA.BuiltInDocumentProperties(wdPropertyPages)
    Set rngA = DocA.Bookmarks("\page").Range
    DocTarget.Range.InsertAfter rngA
    rngA.Cut
    Set rngB = DocB.Bookmarks("\page").Range
    DocTarget.Range.InsertAfter rngB
    rngB.Cut
Next i
DocA.Close wdDoNotSaveChanges
DocB.Close wdDoNotSaveChanges
