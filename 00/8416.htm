jay freedman say:

The fact that VBA in Office 2013 is generally much slower than in previous versions has been discussed many times. The most we've heard from Microsoft is "we'll look into it".

In this case, setting a Range object equal to ActiveDocument.Range and using that object's Find method, with .Find.Style = ActiveDocument.Styles("Heading 1"), will run much faster in any version than will looping through all the paragraphs. As an example:

Sub formatAppendices(doc As Document)
    Dim rg As Range
    Set rg = doc.Range
    With rg.Find
        .Style = ActiveDocument.Styles("Heading 1")
        While .Execute
            If LCase(Left$(rg.Text, 9)) = "appendix " Then
                rg.ListFormat.RemoveNumbers NumberType:=wdNumberParagraph
            End If
        Wend
    End With
End Sub
