cmdrkeene say:

I think the last time the real foundation of Windows was revolutionarily changed in a major way, was Windows NT.  That's when the big shift happened.

Although you can see from the version numbers (which are themselves mostly marketing) that there were massive changes a few other times, just not as impactful as changing the kernel.

Windows 3.1, 95, 98 all used the old DOS underpinnings

Windows NT = Windows 4.0 (NT 4.0)

Windows 2000 = Windows 5.0

Windows XP = Windows 5.1 (only minor cosmetic changes)

Windows Vista = Windows 6.0 (major changes, which is why so many programs broke)

Windows 7 = Windows 6.1 (only cosmetic changes again)

Windows 8 = Windows 6.2 (more UI changes, some under the hood, but still same kernel)

Windows 8.1 = Windows 6.3 (more tweaks but still based on a significantly-updated yet still NT foundation)

Windows 10... still built from the NT legacy, but so much of the foundation has been updated, modernized, and made smaller that it really just doesn't resemble its heritage much at all.

Hi William, the only way provided to us to go from the 32bit version of Windows to the 64bit version, is a clean install of Windows 10 . . .

That will mean you will have to backup all your data, clean install Windows 10, install a full set of device drivers, restore your data and re-install all 3rd party applications . . .


You can reinstall Windows 10 at any time and it will not cost you anything !

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

from what I have read ,, if you use diskpart or other formatting software but leave this space of hdd unallocated windows 7 up to 8.1 creates the system reserved partition,, if deleted system will not boot.. the system reserved holds the boot information and possibly the bitlocker key if you have pro.. among other things it holds some startup programs to help facilitate a faster booting os.. on the other hand if you format the hdd and create a single partition taking up the space of drive windows 7 thru 8.1 installs the boot info and other thing meant for the system reserved onto the main partition along with os.. so to answer your question is it normal id have to say yes,, depending on how the partition is set up to begin with...

so if windows xp is installed and you upgrade to windows8.1 being that theres no system reserved partition already itll use the entire partition just like xp

now if windows 7 is installed with a system reserved partition already then windows 8.1 will occupy the same system reserved partition but might fail if its not big enuf... windows 7 system reserved is only 100 mb while windows 8.1 requires 350 mb.. in that case you might need to use third party partition manager to move free space from main partition to system reserved..

If you want to see only a selected kind of notifications, you can use Notifications Assistant. To do so:

1. Press WINDOWS + I
2. Click on "System"
3. Click on "Notifications Assistant"
4. Click on "Only priority"
5. Click on "Customize your priority list"
6. Select the apps you want to receive notifications from

a. user say:

I don't think is possible with GPO, if you disabled notification center.
But you could let only Windows notifications active using Microsoft Windows Settings:

1. Press WINDOWS + I
2. Click on "System"
3. Click on "Actions and Notifications"
4. Switch off "Get notifications from apps and other senders"

This will block every third party notifications and will only show System notifications.
Obviously yo need to enable from GPO the Notifications Center.
