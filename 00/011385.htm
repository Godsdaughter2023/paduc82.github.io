Listing installed Fonts in Word2016 alphabetically

Having downloaded a macro from the Microsoft website to produce a Word document which lists installed Fonts, I find it does not list them alphabetically.

The following macro, which I obtained many years ago from where I can't remember, does produce a document in alphabetical order.

Maybe you could pass it on to any interested parties.

Ian

Sub IansFontMacro()
'
' IansFontMacro Macro
'
'
Dim MyFontList As Variant

Documents.Add Template:="Normal"

ActiveDocument.Paragraphs.Alignment = wdAlignParagraphLeft

Application.ScreenUpdating = False

For Each MyFontList In FontNames
    With Selection
        .Font.Name = "times new roman"
        .TypeText Text:=Chr(11)
        .Font.Size = 12
        .TypeText MyFontList + Chr(11)
        .Font.Name = MyFontList
        .TypeText "ABCDEFGHIJKLMNOPQRSTUVWXYZ" + Chr(11)
        .TypeText "abcdefghijklmnopqrstuvwxyz" + Chr(11)
        .TypeText "0123456789?£@%&()[]{}*_-=+/<>~""" + Chr(11)
        .InsertParagraphAfter
        .MoveDown Unit:=wdParagraph, Count:=1, Extend:=wdMove
    End With
Next MyFontList

Selection.WholeStory
Selection.Sort
Selection.HomeKey Unit:=wdStory

Application.ScreenUpdating = True

End Sub

Hello,

For an account created using an email address, this will be in the form of live:firstpartofemail and the Skype Name to be provided is your exact email address.

For an account created using a phone number, this will be in the form of live:alphanumericcharacters and the Skype Name to be provided is your exact mobile number, in this format: plus sign + country code + mobile number.

Application.ScreenUpdating = False
Dim Shp As Shape
On Error Resume Next

    'Set Shp = ActiveDocument.InlineShapes(ActiveDocument.InlineShapes.Count).ConvertToShape
     Set Shp = Selection.InlineShapes(1)
     If Not Shp Is Nothing Then
          Set Shp = Selection.InlineShapes(1).ConvertToShape
     Else
          Set Shp = Selection.ShapeRange.Item(1)
     End If

    If Not Shp Is Nothing Then

    End If
