    Restricting Column Width in Microsoft Word - Microsoft Community

    murugesanshanmugam say:

    Restricting Column Width in Microsoft Word

    I would like to restrict all the column widths in a Microsoft Word table in such a way that even if I try to increase the column size using a keyboard mouse by mistake. 

    I do know

    Table Properties > Preferred width > Exactly

    and 

    Autofit > Fixed column width options.

    I am looking for an option to freeze the column sizes so that it can't be resized knowingly or unknowingly.

    Please let me know if it is possible and how? (I am using Microsoft Word 2013)

    jay freedman say:

    There is no such option in Word.

    If you mistakenly change a column width and you notice it immediately, you can use the Undo command (on the Quick Access Toolbar or by Ctrl+Z) to restore the previous width.

    If that isn't sufficient, I suggest that you set up an empty table with the desired columns and save it as a building block (use the Quick Parts button on the Home tab, or press Alt+F3) in the template used for your document. In case you alter a table later and want to return to the original widths, insert the building block table; copy/paste the contents from the altered table, and then delete the altered table.

    An alternative would be to write a macro that sets the width of each column in the table to the desired size. You can run that macro whenever you notice that the widths changed.
