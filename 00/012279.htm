So I'm looking for a VBA code that will remove the special characters "." and replace with a space between the words if needed and also remove the numeric values at the end. 

James.Borwn - James Brown

James.Brown1 - James Brown

James.Brown01 - James Brown

I was able to find a code that will remove the unwanted characters but I haven't been able to figure out a way to add a space where needed.

The Code  

From Form

GarbageInExtra(Environ("username")

From Module:

Function GarbageInExtra(pvarText As Variant)

'gets rid of everything except numerals

    Dim intChars As Integer
    Dim varGarbageOut As Variant
    Dim strJunkChars As String
    strJunkChars = " !@#$%.,<>?/\|][{};:^&*()_-+='~`0123456789"
    Dim intChar As Integer
    If IsNull(pvarText) Then
        varGarbageOut = Null
     Else
        intChars = Len(pvarText)
        For intChar = 1 To intChars
            If InStr(strJunkChars, Mid(pvarText, intChar, 1)) = 0 Then
                varGarbageOut = varGarbageOut & Mid(pvarText, intChar, 1)
            End If
        Next
    End If
    GarbageInExtra = varGarbageOut
End Function

    Dim strJunkChars          As String
    strJunkChars = " !@#$%.,<>?/\|][{};:^&*()_-+='~`0123456789"
    Dim intChar               As Integer
    If IsNull(pvarText) Then
        varGarbageOut = Null
    Else
        intChars = Len(pvarText)
        For intChar = 1 To intChars
            If InStr(strJunkChars, Mid(pvarText, intChar, 1)) = 0 Then
                varGarbageOut = varGarbageOut & Mid(pvarText, intChar, 1)
            Else
                varGarbageOut = varGarbageOut & " "
            End If
        Next
    End If
    'Option 1
    GarbageInExtra = varGarbageOut 'Will leave leading and trailing spaces
    'Option 2
    GarbageInExtra = Trim(varGarbageOut) 'Will remove leading and trailing spaces
End Function
