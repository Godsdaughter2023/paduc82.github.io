rich007a say:

Hi,
In Sheet3 D1, enter:
=MAX(IFERROR(MATCH(1E+306,Sheet1!$A:$A,1),0),IFERROR(MATCH("*",Sheet1!$A:$A,-1),0))
and hit Enter.

 

In Sheet3 D2, enter:
=MAX(IFERROR(MATCH(1E+306,Sheet2!$A:$A,1),0),IFERROR(MATCH("*",Sheet2!$A:$A,-1),0))
and hit Enter.

 

In Sheet3 A1, enter:
=IF(ROW()<=$D$1+$D$2,IF(ROW()<=$D$1,INDEX(Sheet1!A:A,ROW()),INDEX(Sheet2!A:A,ROW()-$D$1)),"")
and hit Enter.

 

Drag the formula in A1 down as far as you think necessary.

 

Cheers
Rich

rich007a say:

Hi,
To do this in one formula with no helper cells or columns is going to be very messy.

 

I would make the two lists that you want to combine in some helper columns.
Have a read of http://office.microsoft.com/en-gb/excel-help/how-to-look-up-a-value-in-a-list-and-return-multiple-corresponding-values-HA001226038.aspx to learn how to return only the cells which are marked with "Y".

 

I'd then use my method above, with the two helper cells which determine the row at which the data stops on each sheet.

 

I've found some better formulae to use for those cells, I've updated my previous post accordingly, please re-read it.

 

You can put all your helper cells on a separate sheet, and if you must, you could make it hidden.


Or, you could just do it with a macro:
Press Alt+F11 to open the Visual Basic Editor (VBE).
Then do Insert > Module.
Paste the code below in the right of the VBE.
Close the VBE and switch back to Excel.
In Excel do View tab > Macros > select "DoIt" > Run

 

 

 

Sub DoIt()
     'Clear data from Sheet3 col A:
    Worksheets("Sheet3").Range("A:A").ClearContents
    Worksheets("Sheet3").Range("A1").Value = "Combined List"
   
     'Copy the Y data from Sheets 1 and 2:
    Call CopyYData(Worksheets("Sheet1"))
    Call CopyYData(Worksheets("Sheet2"))
End Sub


Sub CopyYData(ws As Worksheet)
    Dim wsDest As Worksheet
    Dim rng As Range, c As Range
    Dim lnLastRow As Long
   
    Set wsDest = Worksheets("Sheet3")
   
    lnLastRow = ws.Range("A" & ws.Rows.Count).End(xlUp).Row
    Set rng = ws.Range("A2:A" & lnLastRow)
    For Each c In rng
        If UCase(ws.Range("K" & c.Row).Value) = "Y" Then
            wsDest.Range("A" & wsDest.Rows.Count).End(xlUp).Offset(1, 0).Value = c.Value
        End If
    Next c
End Sub

 

 

 

That could be configured so it is triggerred automatically when a cell in either Sheet1 or Sheet2 is changed.  Let me know if you want to know how to do that.

Rich
