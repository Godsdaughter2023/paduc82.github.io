How to use the Relative Standard Deviation formula on Excel 2008 - Microsoft Community

rebelturtle9 say:

How to use the Relative Standard Deviation formula on Excel 2008

Original Title: "relative standard deviation on excel"
I am using Excel from Microsoft Office 2008 on my mac and I can't figure out how to do the relative standard deviation formula (RSD). I know there is a way to do it on a PC with an older version of Microsoft Office and I found shortcut for standard deviation (STDEV), but I can't find one for relative standard deviation (RSD).  Please Help.
j.e. mcgimpsey say: RSD  is just the sample standard deviation divided by the expected value of the sample, so 

    RSD = STDEV(rng)/AVERAGE(rng)

formatted as a percentage or 

RSD = 100 * STDEV(rng)/AVEREAGE(rng)

when formatted as a number.

I don't remember any version of XL actually combining the two into a single function - there's hardly any point...

Note that in XL2011, STDEV() is deprecated, so the preferred formula is

   =STDEV.S(rng)/AVERAGE(rng)

but the first will still work.

Apostrophe - Microsoft Community

gary's student say:

Apostrophe

If I type an apostrophe, I get a character with code 145 rather than 39.  How do I fix this?
suzanne s. barnhill say:

Although character 39 is identified by Unicode as an apostrophe, in typeset-quality text, you should be using character U2019 (ASCII 146) instead, even though it is identified as a "right single quote." If you have reason to prefer a straight apostrophe (in code, for example), then Undo will work for specific instances.

However, since you refer to getting ASCII 145 instead, I suspect that where you are wanting the apostrophe is as a leading character in words such as 'tis. In that case, you can use Word's built-in keyboard shortcut for an apostrophe/right single quote, which is Ctrl+', '. This will produce the character you are looking for, I think.

For other shortcut keys to force the correct quotes (which are often incorrect in other situations as well), see the Special  Characters tab of the Symbol dialog. More at http://sbarnhill.mvps.org/WordFAQs/InsertSpecChars.htm.

pamcaswell say: After typing the  apostrophe, immediately press Ctrl+z to undo the autocorrection.   If you  do not want Word to make curly ( or "smart") quotes at all, go to  Word options > Proofing > AutoCorrect Options > AutoFormat tab and clear the  box before "Straight Quotes" with "smart quotes".

trixolina say:

a-gps uses a bit of data to ask a server where to find the closest satellites. It tells the phone where to look for them and thus makes positioning faster.

 

GPS on its own will take a long time to get an initial fix, but after the initial fix, it should be faster. Depending on your hardware, it can take anywhere from 5 to 20 minutes.

 

The new wifi positioning also uses a small amount of data to ask a server where the wifi nodes around you are located so that it can tell you accurately where you are. The locations of the wifi nodes are not communicated from the modem to the phone. The locations are kept in a dtabase that has to be queried.

 

Networked positioning uses data, but I haven't yet learned exactly what it does.

 

Putting maps in offline mode dramatically reduces your data use, but keeps a-gps on so that the application will still work and you don't wait 15 minutes for a fix. To be completely data-free, the best is to put the phone itself offline.

ozrob---01 say:

A-GPS: This uses information from cell network to get a rough idea of where you are. It then uses this information to figure out which satellites to look for. Result is a faster satellite lock. So A-GPS uses a few kB of

 

Network based: This uses information from the cell network to plot the current position when GPS satellite signal is not available, ie indoors, in a tunnel, etc. Position info derived from the cell network is not as accurate as that from GPS satellites, but serves as a stopgap until GPS signal can be re-acquired. Network based positioning used your data connection EACH TIME the satellite lock is lost. So the amount of data used is dependent on the quality of access you have to the GPS satellites. If you don't lose the connection to the GPS satellites, then you won't use any data.

 

The ‘offline’ option within the Nokia Maps app only refers to street, POI, etc searches, NOT to A-GPS, Network based positioning or to connecting with the GPS satellites.

 

If you want to navigate without using ANY data connection, set A-GPS off AND set Network based positioning off AND set the Internet option in Nokia Maps to Offline. Do this and you will not use any of your data connection allowance when navigating. There is no need to put the phone into flight mode.

I'm unsure as to what your purpose is in using this shutdown command, but if you're trying to create a quick shutdown method then it is not necessary to do so using a batch file. Assuming you are using Windows 7 or later, the proper syntax (depending on what it is you want to do) would be as follows:

hibernate:

Click next, enter the appropriate name of the shortcut type and then click finish. You can of course add to or change the commands as needed.

hanstand say:

I actually managed to work around it by using the following Macro:

Sub Macro4()
'
' Macro4 Macro
'
' Keyboard Shortcut: Ctrl+q
'
For Each objCht In ActiveSheet.ChartObjects
With objCht.Chart
                .PlotArea.Width = 125
                .PlotArea.Height = 125
                .PlotArea.Left = 30
                .PlotArea.Top = 100
End With
Next
End Sub
