Hi Wendell,

If Safe boot is selected in Boot options, your PC will always start in Safe Mode. Here's how to disable it:

  1. Open Run (Win+R).
  2. Type msconfig, and then click OK.
  3. In the System Configuration window, go to the Boot tab.
  4. Under Boot options, uncheck Safe boot, click Apply, and then OK.
  5. Restart your PC.
