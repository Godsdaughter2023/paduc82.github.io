hansv mvp say:

Create the following custom function in a VBA module:

 

Function GetR1C1(Addr As String) As String
    GetR1C1 = Range(Addr).Address(True, True, xlR1C1)
End Function

 

Use like this:

 

=GetR1C1(A1)

 

If you store the function in a module in your personal macro workbook Personal.xlsb, use

 

=Personal.xlsb!GetR1C1(A1)

Your phone looks at your phone memory for contacts. If you didn't copy the contacts to your phone memory, as far as it is concerned, there are no phone numbers.
