andreas killer say:
Hi Johannes,

You've taken a step in the right direction, I want you to go two/tree steps further.

The 1st is not to use Range("E2"), because E2 of where / in which sheet?
In Excel we can give cells a name.
https://support.office.com/en-us/article/define...

In short: Select any cell you want and write "Limit" into the name box, press Enter.
After that we can use Range("Limit") in the code.

The 2nd is difficult to explain / understand.

In short:
When you apply a filter on a column that contains numbers, the criteria must be numbers also.... sounds silly, but isn't.

The details:
The background is that a date and/or time in Excel is also a number. If you read something from a cell using Range("Whatever") the default property .Value is used. If the cell contains a time, this property returns a data type Date. And when you add this type to a string, the string looks like the contents of the cell, e.g.: ">=12:34".

Are you still there? :-)

The problem is: You can apply that to the Autofilter, but that filters out all rows!
The solution is to use a criteria like ">=0.523611111111111" instead and it works.

Most people struggle on this part and use CDbl to convert the Date into a number, a simpler solution is to use the .Value2 property instead, this property returns also numbers for dates/times and currencies. See code at the bottom.

So if you write a time e.g. "1:00" into the Limit cell, you can also filter the time columns. That's a little unattractive, because Excel changes the number format of the cell automatically.

The 3rd step would be to detect if the selected cell contains a date/time and if so, use a different input cell, e.g. "LimitTime". Not hard to implement: 

    If IsDate(ActiveCell.Text) Then
      Low = ActiveCell.Value2 - Range("LimitTime").Value2
      High = ActiveCell.Value2 + Range("LimitTime").Value2
    Else
      Low = ActiveCell.Value2 - Range("Limit").Value2
      High = ActiveCell.Value2 + Range("Limit").Value2
    End If

Andreas. 


Sub Test()
  Dim LO As ListObject
  Dim Index As Long
  Dim Low As Variant, High As Variant
  
  'Get the table from the current cell
  Set LO = ActiveCell.ListObject
  'Do nothing if no table is selected
  If LO Is Nothing Then Exit Sub
  
  With LO
    'Remove the filter if no data cell is selected
    If Intersect(ActiveCell, .DataBodyRange) Is Nothing Then
      .AutoFilter.ShowAllData
      Exit Sub
    End If
    'Calculate the settings
    Index = ActiveCell.Column - .Range.Column + 1
    Low = ActiveCell.Value2 - Range("Limit").Value2
    High = ActiveCell.Value2 + Range("Limit").Value2
    'Set the filter for this column
    .Range.AutoFilter Index, ">=" & Str(Low), xlAnd, "<=" & Str(High)
  End With
End Sub
