etc.

 

How do I count the unique Client IDs who are COMPLETE for all 4 categories? Or INCOMPLETE, i.e., not complete in all 4 categories? In this case I would have a count of 1 as being COMPLETE and a count of 1 as INCOMPLETE.

 

Thank you for your help!

marshall barton say:

Try using:

SELECT DISTINCT  IIf(compcnt Is Null, "Incomplete", "Complete"), As Status
                                 IIf(compcnt Is Null, Count(*) - compcnt, compcnt) As ClientCnt
FROM yourtable LEFT JOIN (SELECT X.[Client ID], Count(*) As compcnt
                                               FROM yourtable As X
                                               WHERE X.COM = "COMPLETE"
                                               GROUP BY X.[Client ID] HAVING Count(*) = 4) As Y
                            ON yourtable.[Client ID] = Y.[Client ID]

glenprouty say:

Hi Margotr,

Any third party security program that includes real-time antivirus protection, can

run it in concert with Windows Defender. You must make sure that Windows

Defender is "passive". You do that by setting to "ON", the periodic scanning

button in Windows Defender. In that state, MalwareBytes is supplying your

"real-time" protection, and Windows Defender is available for scans only.

If you find the "Periodic Scanning" button at the bottom of the "Virus &

Threat Protection" screen in Windows Defender, Set to "ON". Then WD

will remain updated, as well as supplement your real-time anti-virus with

its Offline Scan capability. When it is not on, any attempt to update or

access Windows Defender is met with undesirable consequences.

mike h.. say:
Thanx! Almost perfect!... Can we exclude blank cells (on Difference)? 

Hi,

I assume you mean blank cells in Column E, If so try this

Sub Difference()
Dim Lastrow As Long, MyRange As Range
Dim LastCol As Long, ChkRange As Range
Dim MyString As String, c As Range
Dim Thetext As String
Thetext = " is not a valid entry."
Lastrow = Cells(Cells.Rows.Count, "E").End(xlUp).Row
LastCol = ActiveSheet.Cells(5, Columns.Count).End(xlToLeft).Column
Set ChkRange = Range("M5").Resize(, LastCol - 12)
ChkRange.Select
Set MyRange = Range("E6:E" & Lastrow)
For Each c In MyRange
   If Len(c.Value) > 0 Then
    If IsError(Application.Match(c.Value, ChkRange, 0)) Then
    MyString = MyString & c.Value & Thetext & Chr(10)
    End If
  End If
Next
If Len(MyString) > 0 Then
MsgBox MyString
End If
End Sub
