Hi 

When I open a new doc in Word, it says in paragraph format 0 pt spacing before and after...I assume that's the default?

when I open a few existing documents and look in paragraph format it says "Auto" for spacing before and "Auto" for spacing after

can you please explain what "Auto" setting means?

cliff gu msft say:

Hi Craig,

 

It seems that you set the 0 pt spacing before and after for a normal template and set Auto spacing before and after for some existing files.

 

If the paragraph spacing is set to Auto and no layout option defaults have been changed, then auto uses a default html-style spacing of 0em before and 1em after the paragraph.

MS Query Wizard in Excel - Microsoft Community

utherpendragon say:

MS Query Wizard in Excel

Is there an equivalent to the MS Query Wizard that works within the workbook?  It appears that the query wizard is designed to import external data.  I want to query data within the workbook and output to another range in the same workbook.
ron coderre say:

You can use MS Query to consolidate Excel ranges from multiple wkbks/wkshts or to query data from the active workbook (Just save it first so Excel can find it):

This example uses a named range in the current workbook.

Assumptions:
The data in the range is structured like a table:
--->Col headings (Dept, PartNum, Desc, Price)

The data is in a named range.
--->Example: MyProductData

(Note: MS Query may display warnings about it's ability to show the query
...ignore them and proceed.)

Starting with an empty worksheet:
1)Select the cell where you want the consolidated data to start

2)Data.From_other_Sources.From_Microsoft_Query
or...for XL2003: Data.Import_External_Data.New_Database_Query

• Databases: Excel Files

Browse to your file, pick the data range to import.
--->Accept defaults until the next step.

At The last screen select the: Return the data to Excel.

Once that is done....to get the latest data just:
Right Click in the data range
...<Refresh Data>

You can edit the query SQL at any time to
add/remove data sources and/or fields.

Is that something you can work with?
