No, you cannot avoid it because it's document corruption and you can't really say at what point that happened. Could be anything. When you inserted a picture, when you moved text, when you saved or changed format…

However, you can ameliorate it if you keep backups (Finder copies), and in particular if you enable the backup option in Word's preferences. david marcovitz say: The Word documents need to be in the exact right format to get translated into PowerPoint. Basically, the titles need to be in the style Heading 1, and the content for each slide needs to be in style Heading 2. It's not much, but when you import a file into PowerPoint in this format, it will dump the text into the appropriate placeholders.
