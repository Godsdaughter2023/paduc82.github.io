daydr3am3r say:

DirectShow it's a multimedia framework, designed by Microsoft, to perform various operations with media files or streams.

DirectShow Filters are, as the name suggests, the actual codecs. Each filter provides support for a certain format and/or task.

In some cases. these filters get corrupted or some filters get in conflict with other filters, resulting in problems like the ones you have.

That is why it's better to make sure there are no other codecs installed when you install new ones.


As for those broken files, it's not Windows's fault. Installing too any codecs will cause this on any other system.

Method 1: If you have not shared anything on the network then you may disable the windows media player network sharing service.

a)    Click Start, type "Services.msc" (without the quotation marks) in the open box and click OK.

c)    Click Stop and click OK.

Check if the CPU usage is high.

Step 1: Uninstalling the Windows Media Player:

a)    Go to Start and in the search type "Turn Windows features On or Off".

b)    Click on "Turn Windows features On or Off".

c)    Browse to the Media Features and uncheck the mark in front of Windows Media Player.

d)    Restart the computer,

Step 2: Reinstalling the Windows Media Player:

a)    Click on ‘Start’ and in the search type "Turn Windows features On or Off".

b)    Click on "Turn Windows features On or Off".

c)    Browse to the Media Features and place a check mark in front of Windows Media Player.

d)    Restart the Computer.
