rohn007 say:
Setting Power Management to Performance primarily impacts the CPU. It tells Windows to run the fastest it can, making the CPU run hotter, and maybe the GPU run somewhat hotter to keep up with the CPU.

https://nvidia.custhelp.com/app/answers/detail/...

The other settings you identified are GPU specific.
Open GL - Your setting is directing OPEN GL work to the GPU all the time, making it run harder / hotter. If you leave it auto, sometimes it will go to the GPU and sometimes to the graphics built in to the CPU.

Anti-aliasing - leave it OFF. That option makes the GPU work LESS Hard, the image is slighly more "jagged".

Texture Filtering - You want to turn it OFF, I'm just not sure which of the 2 options that is on your card. I think it is "performance". When you turn it on the GPU works harder to display more "focused" / "crisper" images.
