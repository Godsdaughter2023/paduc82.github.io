GROUP BY [KMS RUN BETWEEN TWO DATES].vehno, VEHICLES.make, VEHICLES.attachedto

The "FRM VEH MAIN TO UPDATE EXP" is the master form and "KMS RUN BETWEEN TWO DATES" is the sub form programmed to show the latest entries only and date entries are also made through the sub form corrosponding to the vehicle no displayed in the main form.

spot the error for me please. Thanx

karl donaubauer say:

Hi,

 

crosstab queries can't handle form references except they are explicitely declared as parameters. Read e.g. http://support.microsoft.com/kb/209778. This not only applies to references directly in the crosstab query but also in queries that are used by it.

 

So the error message means that you use Forms![FRM VEH MAIN TO UPDATE EXP]!vehno somewhere (usually as a criteria) in a query that is used by your crosstab query, i.e. either in [KMS RUN BETWEEN TWO DATES] or VEHIC or in a source of these two.

 

Find where it is used and declare it as parameter.

Colour a cell with an IF formula - Microsoft Community

(neil88) say:

Colour a cell with an IF formula

How do I colour a cell with an IF formula?

I am thinking =IF(E5>90,"Green","Red")  

The formula is written in the blank cell E6

Basically the cell E6 next to the number in E5 will turn green or red.

How can i do this?

faysal farooqui say:

For Red Color:

- Click on E6 Cell to select it.

- Then apply a conditional formatting: Home | Conditional Formatting | Highlights Cells Rules | Equal To...

- Type Red in Equal To dialog box and choose a custom Fill color formatting Red.

For Green Color:

- Again you need to make sure that E6 cell is selected and then go to: Home | Conditional Formatting | Highlights Cells Rules | Equal To...

- Type Green in Equal To dialog box and choose a custom Fill color formatting Green.

Now if the E5 cell is Greater Than 90, then your formula will return a value Green and Equal To Green, the conditional formatting will fill the cell Green as background.
