I joined the list and got a good pointer straight away - use Applescript to set the preference instead of mucking around with plistbuddy and com.apple.LaunchServices.plist.  So I created the following Applescript:


tell application "Microsoft Outlook"

system default everything application to yes

end tell

tell application "Microsoft Outlook" to quit

end


which I later reduced to the following lines of code in a shell script:


osascript -e 'tell application "Microsoft Outlook" to set system default everything application to yes'

osascript -e 'tell application "Microsoft Outlook" to quit'
