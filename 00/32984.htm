PCI Modem in XP Mode - Microsoft Community

oldredgreenblue say:

PCI Modem in XP Mode

I have an Agere Win Modem (a PCI card fax modem) in my computer, which Windows 7 Professional 64-bit has correctly recognized and successfully installed.  However, the same can't be said within XP Mode.  If I install a standard 56K modem, it shows the modem as external and doesn't respond.  If I use the driver for the modem that's correct for Windows XP, I can make XP Mode install it and identify it correctly, but then in modem properties, it's shown as "nonfunctional," which it is.  Is it even possible to use an internal fax modem in XP Mode?  (I have an old fax program that I'd like to use, WinFax Basic Edition 10, which is incompatible with Windows 7 64-bit but appears to be compatible with XP Mode.)

Thank you.
oldredgreenblue say: A user in the PCWorld forums has come up with a solution to this issue (http://forums.pcworld.com/index.php?/topic/75693-dial-up-modems-in-xp-mode/page__hl__modem__fromsearch__1 ).  A modem installed in Windows 7 can be accessed in XP Mode by first adjusting the com port settings for XP Mode.  This is done through the Settings dialog box , which can be found by either opening Windows Virtual PC (not XP Mode itself), selecting Windows XP Mode.vmcx, and clicking on Settings on the menu bar near the top of the screen or, when XP Mode is running, clicking on the resize button to take XP Mode out of full -screen mode, upon which a menu bar that includes a Tools drop-down menu, on which the Settings option can be found.

 

Within settings, select COM2 (or whatever port is listed as the physical serial port—I don’t know if it’s possible for that to be other than COM2), and set the physical serial port to COM3 (or whichever com port your modem is attached to in Windows 7).  Also, check the box that says “Wait for modem command to open port.”  Now go into XP Mode control panel and add the modem.  My experience is that the applet will not be able to automatically detect the modem, so you will need to select it manually.  From the list offered by Windows XP Mode, simply select the standard modem of the correct speed, e.g ., 56 K, and complete the process of adding the modem.  You should now be to select the modem in your fax or other modem-using program.  Note that the modem will appear in XP Mode as being attached to COM2, not its actual port in Windows 7. 

Both the PCWorld forms user and I have been able to do this and successfully fax with WinFax 8 and WinFax 10, respectively.  XP Mode is indeed a practical and useful tool.

Touchpad scrolling and gestures not working in Asus V56VM laptop after Windows 10 Version 1511 update? Check this out.

After I upgraded to the November Update of Windows 10 (Version 1511), I noticed that my laptop Asus N56VM's touchpad wasn't working properly. Scrolling didn't work anywhere, and the three finger gestures weren't working either.

I went to "Apps and Settings" in the new Settings app, and uninstalled "Asus Smart Gesture" from there. After that, everything has started properly again. Just wanted to throw this information out to anyone having the same problem.

P.S: Before I found that this works, I had tried a couple of other steps. First, I uninstalled Asus Smart Gesture as mentioned above, but after that I went to the Asus website and downloaded the touchpad drivers for my laptop from there, and installed them. I tried both Windows 7 and Windows 8 64-bit drivers, and neither worked. Only after that I found that just installing Asus Smart Gesture solves the problem.
