Step2: Now, I am trying to figure out the VBA code to where I can select different sheets fromWorkbookB instead of just one sheet & then import all these selected sheets into WorkbookA at the same time.  The VBA code would accomplish this by reflecting all the sheets of WorkbookB in a "drop down selection box" & then I can check off which sheets to import intoWorkbookA.

Can you please advise on how to do this?

Thanks! 

bill manville say:

I guess you would use a UserForm containing a multi-select list box which you would populate with the sheet names in WorkbookB

You would then work through the Selected items in the list box.
You could copy the sheets one at a time, but that risks leaving links behind.
Better would be to select the group of sheets and copy them as a group; something like this:

Dim iItem As Integer
Dim bFirst As Boolean
bFirst = True
With Me.ListBox1
  For iItem =  1 To .ListCount
    If .Selected(iItem-1) Then
      Worksheets(.List(iItem-1)).Select(Replace:=bFirst)
      bFirst = False
    End If
  Next
End With
If bFirst Then
  msgbox "No sheets selected!"
Else
  ActiveWindow.SelectedSheets.Copy Before:=Workbooks("WorkbookA.xls").Sheets(1)
End If

If you need more help, come back and tell us where you are stuck
