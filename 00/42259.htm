Hi Allanon04,

Many programs refuse to run, or they run with limitations (for example, they might not display all settings, or they might not save settings you make). On those occasions you do need to use such a program with full capabilities.

To perform this procedure, you must be logged on as a local administrator or provide the credentials of a member of the local Administrators group.

To run an application one time with a full administrator access token

1.    Locate the program icon or a shortcut in Windows Explorer.

2.    Right-click the program icon or shortcut, and then click Run as administrator.

3.    When the UAC message is displayed, do one of the following:

o    If you are logged on as a standard user, or if UAC is configured to always require credentials, enter the appropriate administrative credentials, and then click OK.

o    If you are logged on as an administrator and UAC is not configured to always require credentials, click Yes to start the application.
