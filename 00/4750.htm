If we all upgrade to windows 10, what happens to our windows 7/8/8.1 Key after the update to windows 10?

In the case of Windows 10 Preview, it is pre-release software which will expire. In the case of Windows 10 final version:

15. UPGRADES. To use upgrade software, you must first be licensed for the software that is eligible for the upgrade.

Can we use our previous key to reinstall windows 7 or 8 or 8.1 after it was use as an upgrade of windows 10?

Yes, or use the rollback function:

How to recover / restore your previous version of Windows using the Rollback feature

No use Internet Activation or Phone Activation if you encounter any problems.

      All sounds good.  We have an knowledgeable user with problems and lots of knowledge on this side.

      Whenever I come across a dead O/S on a working PC , the options are somewhat limited.  This is how a PC built in the garage comes to life.  A load from BIOS operates without Drive C.

      With the BIOS Load order change, the load swings right to the bootable USB and wipes Drive C.  I always go default on partitions and the data is transferred from the USB to Drive C.  You will be asked to remove the USB.  At this point I go back into BIOS and change the order back to Drive C.  In actual practice, if no USB, the load is likely a default to Drive C but BIOS should be changed anyway.

      The install then takes place from Drive C.

      Just as way of info, one needs to pull the proper edition from the Media Creation Tool which will have Home and Pro.  I am a Pro guy but the default install could be Home.  If in doubt, I enter in a generic Pro key to pull the Pro edition.  This is not an activation key.  If we are talking Home here, you will likely default to Home.

      I have used BIOS on several occasions when I fry W10 or a bad O/S comes to me. 

      The other option if there is a good public built in the house is a simple clone which will  give you a copy of the good PC.   I have a bunch of 2 1/2  drives on the shelf, a bunch of HP notebooks around and a $10 harness from 2 1/2 to USB.  I also have access to a lot of software so I use Acronis.  This may be a "Buy" program but likely lots of free ones out there.  The clone avoids app and file lost as long as the 2 PCs are close in purpose and the good PC has the important data.

      A few choices out there.
