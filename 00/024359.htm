Follow the Steps below to find the SCANPST tool to repair your PST file in outlook:

 

STEP 1: Unhide the Files and Folders and File extensions:

 

Start >My Computer >Organize >Folder and Search Options > View Tab >Check Show Hidden Folders and Files and uncheck “Hide Extensions for Known File Types”.

 

Location for Scanpst.exe:

 

Scanpst for Outlook 2007 in Windows 7 and Windows Vista

64 bit: Start >My Computer > Program files(x86) >Microsoft Office >Office 12 >ScanPST.exe

32 bit: Start >My Computer > Program files>Microsoft Office >Office 12 >ScanPST.exe

 

Scanpst for Outlook 2007 in Windows XP.

Start >My Computer > Program files>Microsoft Office >Office 12 >ScanPST.exe

 

STEP 2 : Browse to the location and repair the pst file:

 

 The error message shows the location of the outlook.pst file, which has to be repaired. In the scanpst tool, Click on browse and go to the location show in the error message and click on repair.

 

 Run the tool, and browse to the following location to choose the Outlook data file which you want to repair.

“C:\Users\<Account Name>\AppData\Local\Microsoft\Outlook\Outlook.pst.”

 Click Start, which will start the repair process.

 

 Note:  The repair might take a long time depending on the size of Data file.

 After the scanning is complete you will be prompted to click repair. Click repair and wait for the repair to complete and then click ok.

Try opening Outlook to check if the issue is fixed.

roubo say:

Hello there julob!

I know it's 6 years later but the problem still persists. May this be helpful to someone else in the future.

So, I have learned a workaround which prevents you from having to manually go back and change the proofing language of your paragraph every time you decide to switch your keyboard layout.

  1. You are typing a sentence in one language.
  2. You decide to switch your keyboard layout (via the language bar or a keyboard shortcut).
  3. You observe (in the status bar, for example) that Word has started to anticipate a different language to proof.
  4. MOVE YOUR CARET BACK AND FORTH USING YOUR ARROW KEYS ←→ by one character only, before you type anything else.
  5. Observe that Word has forgotten the newly assumed language and continues to proof the original language.
  6. Profit.

You will soon learn this as muscle memory and will do it automatically. It's kind of ugly but kind of elegant because you don't have to lift your hands from the keyboard or perform crazy shortcut gymnastics.

I hope this helps.
