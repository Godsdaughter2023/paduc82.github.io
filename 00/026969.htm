jay freedman say:

There is no simple, field-based method to prompt for a value to put into a document property. There are two macro-based methods; the choice depends on how much control you want over the users' entries.

For the simpler method, open the template and then open the macro editor (Alt+F11). In the Project pane at the upper left, expand the project that contains the template's name, expand the Microsoft Word Objects folder under that, and double-click the ThisDocument icon. Copy and paste the following code into the large empty area on the right.

Private Sub Document_New()
    Dim strValue As String
    strValue = InputBox("Enter a value for 'myproperty':", "myproperty", " ")
    
    ' the value will be an empty string, "", if the user cancels
    ' or deletes the default space; fix that
    If strValue = "" Then strValue = " "
    
    ActiveDocument.CustomDocumentProperties("myproperty").Value = strValue
    
    ActiveDocument.Fields.Update
End Sub

Save the template as a Word Macro-Enabled Template (*.dotm) file. (A regular template, *.dotx, cannot contain any macro code.) The template must be stored in a trusted location, such as %appdata%\Microsoft\Templates, to allow the macro to run -- and the same is true for any other user who wants to base documents on it.

When the user creates a new document based on the template, the Document_New macro runs automatically. It displays this dialog box:

If the user enters a value and clicks OK (or presses Enter), the value will be stored in the document variable and the document's fields will be updated so the DOCPROPERTY field will display the value.

The macro can be expanded to ensure that the entered value is valid, according to whatever rules you want.

The second method involves creating a userform, which is more sophisticated. For example, it could collect several pieces of information in one form, rather than making the user complete several dialogs of the type shown above; and it could include date pickers, list boxes, and other entry mechanisms as well as a plain text entry box.
