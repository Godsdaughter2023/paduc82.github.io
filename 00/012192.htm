The SupportMultipleDomain switch is used with New-MsolFederatedDomain, Convert-MSOLDomainToFederated, and Update-MSOLFederatedDomain when we need to let one AD FS server authenticate multiple domains. A CSP global admin account is permitted to configure cloud services but not the local services.

There seems to be a discrepancy in your description of the problem. You have posted your inquiry as an Outlook 2013 question but the path you have referred to is an Outlook 2010 solution (office14). At a minimum, it should be in a path identified by office15.

The path for ScanPst.exe on an 64-bit Office 2013 program on a 64-bit Windows operating system is typically,

C:\Program Files\Microsoft Office\Office15

For a 32-bit Outlook on a 64-bit Windows operating system, the location changes to,

C:\Program Files (x86)\Microsoft Office\Office15

If you are running a Click-to-Run Office installation then it should be in,

C:\Program Files\Microsoft Office 15\root\office15

A more complete list of the full paths for various Office/Outlook installations ca be found by clicking through the following link,.

Inbox Repair Tool (scanpst.exe)
