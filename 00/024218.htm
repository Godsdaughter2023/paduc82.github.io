Let's say that the path and filename for your images are located in the corresponding cells one column to the right of the series values, replace...

   For Counter = 1 To Range(xVals).Cells.Count
     ActiveChart.SeriesCollection(1).Points(Counter).HasDataLabel = _
         True
      ActiveChart.SeriesCollection(1).Points(Counter).DataLabel.Text = _
         Range(xVals).Cells(Counter, 1).Offset(0, -1).Value
   Next Counter

with

    For Counter = 1 To Range(xVals).Cells.Count
        With ActiveChart.SeriesCollection(1).Points(Counter)
            .HasDataLabel = True
            With .DataLabel.Format.Fill
                .UserPicture Range(xVals).Cells(Counter, 1).Offset(0, 1).Value
                .Visible = msoTrue
            End With
        End With
    Next Counter

Application.EnableEvents = False

 

Close the Immediate window (Cross at top right of Immediate window)

 

Close VBA Editor and continue with the following.

 

'******************************************************************

Turn off Automatic calculation. (File -> Options -> Formulas -> Set Workbook calculation to Manual. (Ensure you click OK; not just close the window which is same as Cancel.)

Find the column after the last used column on the worksheet and select it.

Hold Ctrl/Shift and press the right arrow to select all of the unused columns to the right of your data to the last worksheet column.

In the Home ribbon in the Editing block at the far right click Clear and select Clear All from the drop down. (Do not use the Clear Contents from the dropdown you get with Right click.)

Find the row after the last used row and select it

Hold Ctrl/Shift and press the down arrow to select all of the unused rows below your data to the last worksheet row.

Clear All as was done for column. (Remember: must be Clear All not just Clear Contents.)

Repeat from the Asterisk line above for all worksheets.

Save the workbook and see if it has made a difference to the workbook size.

 

Turn Auto Calculation back on

If you actually have formulas copied down to the bottom of the worksheet then the Clear All is going to take them out. Clearing the rows below the actual data does not affect the formula. The particular function does not seem to have any affect on the speed of the system by addressing the entire column. I am fairly sure that the code behind the function recognizes where the data ends.

 

In my experience, array formulas that address entire columns rather than just the required range slow the system down because they attempt to process every cell. It can be a lot more efficient to address the number of rows likely to be required (plus some) because although there are a million plus rows available, is it likely you will ever use them all.

 

I should have mentioned in my previous post that you should always back up your workbook before attempting to Clear All from the unused area.

Unfortunately any reference to another worksheet references back to the source workbook and the following is the procedure to copy and get rid of the reference back to the source workbook.

Ensure that backup your workbook first in case you have problems.

Open a new workbook

Name each worksheet to match the worksheets in the source workbook. It is essential that you get the worksheet names correct otherwise you will have problems when removing the reference to the source workbook. (Explained later how to overcome this problem if it occurs.)

Select just the used range of the worksheet and copy. (Do NOT select all cells in the worksheet)

Select the appropriate worksheet in the new workbook.

Select the first cell of the destination and paste. (First cell must match first cell address of source.)

Repeat for each worksheet.

To remove references to the source workbook from the formulas.

Select the first worksheet in the new workbook.

Find and select any SINGLE cell containing a reference to the source workbook. (It will only be in cells with formulas that reference other worksheets.)

In the formula editing bar at the top, within the formula, select/highlight the source workbook name only including the square brackets.

Right click and copy (or Press Ctrl/C to copy)

Press Esc to leave the formula bar. (Must do this before next step)

Select Find and Select -> Replace

Click in the Replace what field.

Right click and Paste (or Ctrl/V to paste)

Ensure that the Find what field only contains the workbook name enclosed in square brackets. If you have any other characters (including single quotes) before or after the square brackets then delete them. If you have not got the square bracket at each end of the workbook name then insert them.

Leave the Replace with field blank. (We are replacing the workbook name with nothing.)

Select options and ensure that Lookin is set to Formulas and that Match entire cell contents is NOT checked.

Click Replace all.

If you get a dialog box requesting that you select a file then it is because it cannot find the worksheet name in the current workbook. You probably incorrectly spelt the name of the worksheet when you created them in the new workbook. Exit and use undo to return the formula to its previous state and correct the spelling of the workhseet name in the new workbook and try again.

Replace for each worksheet in turn. You should be able to simply select Find and Select -> Replace and it should have retained the string to replace without having to repeat the copy and paste.

Save the workbook and see if it makes a difference.
