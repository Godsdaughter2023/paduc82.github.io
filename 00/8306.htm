Type the first two equations in separate equation containers, just as you show them here. Then select both lines, right-click anywhere in the selected area, and choose "Align at =".

suzanne s. barnhill say:

If you are using Word 2007 or above, the default line spacing for paragraphs is 1.15 lines, and the default Spacing After is 10 points. If you don't want this spacing, you can change the settings in a variety of ways:

  • The Line and Paragraph Spacing dropdown in the Paragraph group on the Home tab allows you to select Single (1.0) spacing and remove the Spacing After for a given paragraph (and subsequent ones).
  • There is also a No Spacing paragraph style in the Quick Styles Gallery on the Home tab that you can apply to a given paragraph or selected paragraphs.
  • Or you can use Change Styles in the Styles group on the Home tab to select the Word 2003 style set, which has Single spacing and no Spacing After. If you select this style set, you can then use Set as Default on the Change Styles menu to set is as the default for a given document.

If you want to make the change for all new documents, however, your best bet is to do this through the document defaults.

  1. Open the Styles pane by clicking the dialog launcher in the bottom right corner of the Styles group on the Home tab or using the keyboard shortcut Alt+Ctrl+Shift+S.
  2. In the bottom left corner of the Styles pane, click Manage Styles (third button from the left).
  3. In the Manage Styles dialog, select the Set Defaults tab.
  4. Change the paragraph spacing as desired.
  5. Select the radio button for "New documents based on this template."
  6. Click OK.
  7. If you are prompted to save the Normal template when you quit Word, say Yes.

Previous settings that Outlook left behind had to be deleted with the following simple steps:

  1. Close Outlook
  2. Open Terminal (click the Spotlight search glass in the top right corner and type terminal)
  3. Type: defaults delete com.microsoft.Outlook (to delete the old preferences)
  4. Type: killall cfprefsd (to kill the cached prefs)
  5. Launch Outlook

Hi.  This ideas appends the second array onto the first.

There's a lot of different ways to do this.   This is just one of many.

Note that in your function, you never returned a value by using something like the following at the end of your Function:

Function MyMerge(RngIncome As Range, RngExpenditure As Range) As Variant
    Dim V1(), V2(), UBc, UBr
    Dim P, R, C
   
    '// Transpose in order to expand the # or Rows.
    '// Swap the Rows/Column reference
    V1 = WorksheetFunction.Transpose(RngIncome)
    V2 = WorksheetFunction.Transpose(RngExpenditure)
    UBc = UBound(V2, 1) '# Columns
    UBr = UBound(V2, 2) '# Rows
   
    '// Test that the # of columns are equal
    If UBc <> UBound(V1, 1) Then
        MsgBox "Number of Columns not Equal"
        Exit Function
    End If
   
    P = UBound(V1, 2) + 1
    ReDim Preserve V1(1 To UBc, 1 To UBound(V1, 2) + UBr - 1)
   
    '// Append RngExpenditure to end of RngIncome
    '// Starting with 2nd Row
    For R = 2 To UBr
        For C = 1 To UBc
            V1(C, P) = V2(C, R)
        Next C
        P = P + 1
    Next R
    MyMerge = WorksheetFunction.Transpose(V1)
End Function
