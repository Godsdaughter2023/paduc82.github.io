teylyn say:
No, you did not sound rude. I was offline and did not see your reply.

You can create a column chart with three columns from the current layout, but it would be a lot easier to create a table with all the chart data in one block, like in the screenshot below. Cell J2 refers to cell A42, cell K2 has the formula =C42, cell L2 has =E42. Then you can select the cells marked with a red border and insert a column chart like in the screenshot.



The color coding is a little more complex, especially if you want the colors to change dynamically. To build conditional formatting in charts, one technique is to use one series for each color. You need a data table with a row for each color. The data in the table is calculated with a formula. If the data meets the condition of a specific cell, it will be placed in that cell, otherwise the cell shows a zero. Then you build a stacked column chart with all the data. The zero values will not display.

In your example, you could build a data table like in the screenshot below. First count the numbers that contribute to each average with a Count() function. In J22 I have =COUNT(A2:A39) and the two cells to the right have similar formulas for the ranges in C and E.

In H24 and down are manually entered numbers to set the thresholds for each color band.

The formula in J24 has been adjusted to refer to columns C and E in the neighbouring cells and then copied down to row 29.

J24 =IF(AND(J$22>=$H24,J$22<$H25),$A$42,0)
K24 =IF(AND(K$22>=$H24,K$22<$H25),$C$42,0)
L24 =IF(AND(L$22>=$H24,L$22<$H25),$E$42,0)

The range outlined in red is the source for a stacked column chart with the series in rows. 

The easiest way to format all the different series with the correct colors is to enter some dummy data into the table, so that all series have at least one value greater than 1. Then format each series color, and finally put the formula back in.



Instead of color coding, you could also use a Bubble chart, where the size of the bubble represents the numbers that contribute to the average. That would be a lot easier to build.

john korchok say:

With trickery! Create a duplicate of the shape you are swiveling, and place it below the shape that is swivelling. Give the top shape a swivel, then click on Add Animation and give the same shape a Disappear animation. Set this to start With Previous and give it a delay of about 1 second. Now set the duplicate shape to Appear with the same delay and also starting With Previous.

You'll have to play with the timing to get it just right, but it should look like one object that swivels once before stopping.
