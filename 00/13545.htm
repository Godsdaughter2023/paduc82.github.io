stefan blom say:

The Set Defaults tab defines the so-called document defaults for a Word document (or template). Think of these as basic settings that apply to the document as a whole. When you modify a paragraph style, the changes are at a more specific level.

Microsoft actually recommends changing the document defaults rather than modifying the Normal style.

Document defaults have been around in Word for a long time, but users haven't been allowed to customize them until Word 2007 and later.
