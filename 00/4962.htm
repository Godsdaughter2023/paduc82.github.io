By default, all SharePoint site collections that are part of an Office 365 Group have the sharing setting set toAllow sharing only with the external users that already exist in your organization’s directory. That's why Anyone is grayed out in your screenshot above.

To change this setting, we can use the Set-SPOSite Windows PowerShell cmdlet. Below are the steps:
