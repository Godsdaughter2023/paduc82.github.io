jllatham say:

One question I have is which version of Windows are you using?

Typically, with Windows 7, all you should have to do is to insert a blank CD into a disk-writing capable drive and you'll usually get a message asking if you want to write files to it and which will continue on, if you say yes, to let you browse for files to burn to it. 

Even if you don't get that message after inserting the CD into the drive, if it has a writable-CD in the drive you should be able to simply drag and drop files from their source folder onto the CD drive and they'll get queued up for writing.  Eventually you'll be able to right-click on the CD drive and choose "Burn Files" and they'll be written to the disk.

If none of this kind of thing works, as Keith Ruck said, probably best asked in the general OS forum.  The will also want to know the exact wording of the error message you are getting.  That will be a BIG clue.  Usually the "operation failed" message will give some kind of indication of why it failed (but then it might not).
