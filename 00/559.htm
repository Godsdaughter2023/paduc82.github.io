List once for every occurrence in A2:AZ25.

mike h.. say:

Hi,

 

To prevent this formula becoming huge we can use a named range.

 

Select all the cells in this table of names, right click, Name a range, Ensure the 'Scope' is set to workbook and call the range tbl

 

OK.

 

Now go to this other sheet and in A1 enter a header (say) Unique, Now put this ARRAY formula in A2. ARRAY enter it; see below on how to do that, and drag down until it starts producing blank cells.

 

=IFERROR(INDEX(Tbl, MIN(IF(COUNTIF($A$1:A1, Tbl)=0, ROW(Tbl)-MIN(ROW(Tbl))+1)), MATCH(0, COUNTIF($A$1:A1, INDEX(Tbl, MIN(IF(COUNTIF($A$1:A1, Tbl)=0, ROW(Tbl)-MIN(ROW(Tbl))+1)), , 1)), 0), 1),"")

 

Now against this new list you can do your countif in b2 and drag down

 

=COUNTIF(Tbl,A2)

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

You can use COUNTIFS if you need multiple criteria.

Let's say gender is in A2:A100 and size in B2:B100.

To count the number of rows for male + small, you can use

=COUNTIFS($A$2:$A$100,"male",$B$2:$B$100,"small")

etc.

rick rothstein, mvp - excel say:

You cannot do what you want with Cell Formatting; however, there is a VB solution available. Right-click the name tab at the bottom of the worksheet you want to have this functionality, select "View Code" from the popup menu that appears and copy/paste the following into the code window that opened up...

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim Fraction As String
  Const COL As String = "A"
  If Not Intersect(Target, Columns(COL)) Is Nothing Then
    If Len(Target.Value) = 0 Then
      Target.NumberFormat = "General"
    Else
      Target.NumberFormat = Trim(Format(String(Len(Int(Target.Value)) - _
                                 1, "#"), "@@@ @@@ @@@ @@@ @@0"))
      Fraction = Mid(Target.Value, InStr(Target.Value & ".", "."))
      If Len(Fraction) > 1 Then
        If Target.NumberFormat = "General" Then Target.NumberFormat = "0"
        Target.NumberFormat = Target.NumberFormat & Trim(Format(String( _
                             Len(Target.Value) - InStr(Target.Value & _
                             ".", ".") - 1, "#"), "!.0@@ @@@ @@@ @@@ @@@"))
      End If
    End If
  End If
End Sub

You have to make one minor change in the above code... find the Const statement at the beginning of the code and change the "A" to the column letter where you will be entering your numbers (make sure to include quote marks around it). Okay, that's it. Go back to your worksheet and type numbers into the column you specified and the blanks will be inserted automatically. This feature will work for any valid number that you can enter into an Excel cell. Note that if you have decimal values as part of a floating point number entry, the decimal portion of the number will also have spaces inserted to form groups of three as well (that is the convention as I am familiar with it).
