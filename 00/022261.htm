It is possible to do this with VBA as per the code below. Don't know what your expertise is with VBA but if you need more help with it then please get back to me but please understand that I cannot teach you VBA in one easy lesson.

 

You will need to create a table with all of the picture names and the costs associated with them and the code uses Vlookup to get the costs. I have used another worksheet for table. The code is written so that the table can be of variable length and the range variable assigned to it picks up from Cell A2 (Assumes column headers) to the last used cell in column B.)

 

Sub IdentifyPictures()
    Dim wsPictures As Worksheet
    Dim wsLookup As Worksheet
    Dim shpPic As Shape
    Dim rngTableArray As Range
   
   
    'Edit "Pictures in next line to your worksheet name"
    Set wsPictures = Worksheets("Pictures")
   
    'Edit "Lookup" in next line to your worksheet with values table
    Set wsLookup = Worksheets("Lookup")
   
    'Assign the lookup values table to a range variable _
    ' (for use as Table Array in Vlookup)
    With wsLookup
        Set rngTableArray = .Range(.Cells(2, "A"), _
            .Cells(.Rows.Count, "B").End(xlUp))
    End With
   
    With wsPictures
        For Each shpPic In .Shapes
            'Identify the pictures within the required range
            If Not Intersect(shpPic.TopLeftCell, _
                    .Range("C7:C26")) Is Nothing Then
                'MsgBox shpPic.Name      'Used during testing and left commented out if required.
                'Add the cost of the picture to the value in cell E10
                .Range("E10").Value = .Range("E10").Value + _
                        WorksheetFunction.VLookup(shpPic.Name, _
                        rngTableArray, 2, False)
            End If
        Next shpPic
    End With
End Sub

Yes! That is the way Excel works. However, you can rename the pictures to a more meaningful name if you wish. When you click on a picture you will see the name (Picture 1) at the very left of the formula bar (to the left of the grey Insert Function icon). Edit the name and press Enter. You might also find the routine below helpful to create the list of the picture names. Tip for the following line of code:-

 

If shpPic.Type = msoPicture Then

 

If you want to see the other shape types that can be identified then highlight the = sign and then re-type the = sign over itself and the VBA intellisense will kick in and list them. Further information on the shape types at the following link.

http://msdn.microsoft.com/en-us/library/office/aa432678(v=office.12).aspx

 

Sub CreatePictureList()
    Dim wsPictures As Worksheet
    Dim wsLookup As Worksheet
    Dim shpPic As Shape
   
    'Edit "Pictures in next line to your worksheet name"
    Set wsPictures = Worksheets("Pictures")
   
    'Edit "Lookup" in next line to your worksheet with values table
    Set wsLookup = Worksheets("Lookup")
   
    With wsLookup
        For Each shpPic In wsPictures.Shapes
            If shpPic.Type = msoPicture Then
                .Cells(.Rows.Count, "A").End(xlUp).Offset(1, 0) = shpPic.Name
            End If
        Next shpPic
    End With

End Sub

Kelvin provides an Exclude Dictionary to take out the Americanisms.  Getting the Exclude Dictionary working in Word for Mac 2011 is a real chore.  Matthew Stevens developed the following procedure for making it work:


1. Open a blank document and type up your list of words to flag.

2. Go to Preferences and click on Save. In the pull-down menu against "Save Word files as:", select "Speller Exclude Dictionary (.dic)". This file format is not available in the normal "Save As" dialogue box (and there is no explanation of why). Click on OK to exit.

3. Now save your document to the MS Word dictionary location on your computer (which depends on the settings you chose). If you have not moved it, it will be in /Users/<your login>/Library/Preferences/Microsoft. 

    Call it whatever you like (mine is "English Exclude Dictionary.dic", which follows the naming convention).

4. Go back to Preferences: Save and change the default file format back to what you had before.

5. Still in Preferences, click on Spelling and Grammar. Click on the "Dictionaries" button. In the new window that opens, click on "Add". In the file navigation box that opens, select the exclusion dictionary you just saved. Click on OK and exit.

6. Quit MS Word.

The second problem is that when MS Word creates a user dictionary, it sabotages itself by saving the file in the wrong format. The result is that your carefully constructed dictionary is ignored, even though it's selected.

This is not something that can be changed in MS Word. Instead, you need a heavy-duty text editor. I use TextWrangler (free download from http://www.barebones.com/products/textwrangler/).

1. Locate the faulty Exclude dictionary file. (This set of steps is necessary for the Speller Exclude Dictionary.)

2. Open it in your text editor.

3. In TextWrangler, at the bottom of the screen, is a choice of file encoding formats. (I know nothing about any other program. This is the sort of thing I'd expect to find in the Save As dialogue box.) MS Word has saved the file in "Unicode (UTF-8)" format. It needs to be saved in "Unicode (UTF-16 Little-Endian)" format. (The "Windows (CRLF)" setting is correct.) Save it and close.

4. Now start MS Word again. You should find (as I did) that previously ignored dictionaries are now recognised (abstruse spellings are now accepted) and excluded words (such as pubic) are now marked as incorrect.

  1. Download the Microsoft Outlook app or search for Outlook in the app store with your iOS device.
  2. After the app has installed, tap Get Started.
  3. Next, select the account you would like to add.
  4. Sign in to the account and the Microsoft Outlook app will begin to sync with the account.

If you’re unable to use the app for any reason, you can still set up Outlook.com on your iOS device.

  1. Tap Settings, and then tap Mail, Contacts, Calendar.
  2. Tap Add Account in the Accounts page.
  3. Select Hotmail.
  4. Enter your Outlook.com address and password.
  5. Select the fields that you want to sync. Tap Save.

  1. Tap Settings, and then tap Mail, Contacts, Calendar.
  2. Tap Add Account in the Accounts page.
  3. Select Other.
  4. Tap Add Mail Account.
  5. Provide your Name, Email and, Password.
  6. Tap Next.
  7. Select IMAP.
  8. In the INCOMING MAIL SERVER, enter the Host Name as: imap-mail.outlook.com. Port should be 143.
  9. Provide your email address in the User Name field, and the email password in the Password field.
  10. In the OUTGOING MAIL SERVER, enter the Host Name as: smtp-mail.outlook.com. Port should be 587.
  11. Again, provide your email address in the User Name field, and the email password in the Password field.
  12. Tap Save.

hal hostetler - mvp-outlook say:
Sadly, you can thank the developers of the POP email protocol for that sync failure.  The POP protocol syncs NOTHING at all.  If you have your email syncing between Outlook and your Android and you are using POP, the only way this is possible is by "Fudging" both email clients to not delete email as it is downloaded, which is how the protocol is designed.  To get Contacts and Calendars to sync, you need to add a email account that supports this form of syncing.  If you have a Live account, it will handle syncing these items for you.  So, the short solution is two email accounts, the POP account to handle the email and a Live account to handle the rest.  Or, you could just move email duties to the Live account and let it handle everything.
