Hi all,

Well, with help from a nice chap at Microsoft Support the issue now seems resolved.

For the sync to work the PST files need to be in the correct folder - This may seem obvious but for some reason my system was continually putting the PST files in two different locations.

The correct location (in Windows 7) is user name\AppData\Local\Microsoft\Outlook - In my system setup a folder was created in my documents called Outlook Files.

If a pst file is in the wrong location, close Outlook  then make a copy of the PST and put it somewhere safe, then cut and paste the PST file(s) to the correct location (as above. Start Outlook and it will tell you the PST(s) are missing and then opens a window which allows yout to direct Outlook to the new location. Press OK to the dialogue box (you get another dialogue box which you press OK to as well) and Outlook should open and sync should be possible.

People, this instructions worked for me and fixed my problem with MMS messages. Just be sure which config file (.xml) file to choose. It must be the one of your service provider. This was posted by Ben, and thank you Gary.

MFG is a hidden admin app on the phone.  Here are the steps -- !!! use at your own risk !!!:

  1. Go to the phone app
  2. Dial ##634# and click Send
  3. When prompted for a password type 277634#*#
  4. You should now be at "Factory Menu"
  5. Choose "Engineer Menu"
  6. Choose "Other Setting"
  7. Verify that "Set ADC" is set to Enable
  8. From the "Other Setting" page, choose "Set network profile"
  9. You should now be on the "Set network profile" page and in the bottom half of the window you should see a list of *.xml files.  These appear to be the network configuration files. 
  10. Look at the filenames for the config files and take your best guess at which one to choose for your provider.  I'm on AT&T and I fixed the MMS problem by choosing "_PROV_ATT_US_310_410.xml"

After you choose that *.xml profile, your MMS should be working.
