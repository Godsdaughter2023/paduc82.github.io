Outlook for Windows is the only application that supports using PST
files. No other email application (Mac, Windows or UNIX) supports using
PST files.
 
Outlook for Mac can only import Outlook for Windows PST files. All

file under "On My Computer".
 
The import process does have some problems that Microsoft has not yet
addressed. For example, if you have mail stored at the top level of your
PST file and not inside a folder then those messages will not import.
Also, some messages with PDF attachments do not properly import the
attachments.

Another sample that may be more familiar looking to you - although the method that f z z offered is probably more efficient.  The Application.ScreenUpdating=False statement accomplishes 2 things: keeps the screen from flickering as you choose and work with other sheets, and it definitely speeds up processing speed.

Sub ReturnHome()
  Dim whereWasIAs String

  whereWasI= ActiveSheet.Name
  Application.ScreenUpdating = False ' keep screen from flickering
  'do more stuff, visit other sheets, etc.
  ' when finished then
  Worksheets(whereWasI).Select
End Sub

Sub foo()
  Dim oaws As Worksheet

  Set oaws = ActiveSheet

  With ActiveWindow.SelectedSheets
    MsgBox Title:="test", prompt:=.Count & " worksheets selected"
    .Item(1 + Rnd * .Count).Activate
    MsgBox Title:="test", prompt:=ActiveSheet.Name
  End With

  oaws.Select

End Sub

12/06/2016 - Update 3:  

I think I found a solution, at least in build 14361 (14366 and 14367 too).

  • Open regedit
  • Go to the following Registry key:
    HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DataCollection
    Create a new a 32-bit DWORD value named AllowTelemetry and set it to 0.
  • Go to Services.msc, Disable the following services:

Connected User Experiences and Telemetry

dmwappushservice

  • Restart PC
  • Create a shortcut from lol.launcher.admin.exe to desktop
  • Rename it to "League of Legends"
  • Execute

No BugSplat or Maestro Error since I did it. 

Just one thing: when Microsoft release a new insider build, you need to activate those services again and delete the Value that you created on Regedit and restart PC to be able find the build in Windows Update and also to post feedback in the Feedback Hub.

26/06/2016 - Update 4

No more problems with latest builds (14371 and 14372) even without these steps.
