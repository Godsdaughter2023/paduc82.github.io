But, you can resize the graphic yourself to solve the problem. 

1) Navigate to this location where the Edge icons are stored:

%USERPROFILE%\AppData\Local\Packages\Microsoft.MicrosoftEdge_8wekyb3d8bbwe\AC\MicrosoftEdge\User\Default\DataStore\Data\nouser1\120712-0049\Favorites

3) Sorting by "Dimension" column will allow you find the one you want. 

4) Then you can open that file with Paint (or any other graphic/photo app you might have).

5) Next, just resize the graphic to 24x24 pixels and save.

kevinmontague say:

I just bought a HP Smartscreen computer this week with Windows 8 and I'm having the same issues. Fortunately there are controls that handle the touchpad issues. Even if you own another computer with an overly sensitive touch pad, this information may help you going in the right direction.

http://h10025.www1.hp.com/ewfrf/wc/document?docname=c03523373&tmp_task=solveCategory&cc=us&dlc=en&lc=en&product=6875730#N1771

I was able to activate the HP touchpad icon into my list of memory resident icons which allows me to directly access the touchpad controls when I right click on the icon in my tray. All I did was go into Control Panel, and click on "mouse". Up came the mouse appliations window. Then I moved to the last tab labeled "Device Settings". From there you can activate the touchscreen icon into your tray by selecting "static tray icon in taskbar". Then I went to that newly generated icon, right clicked on it and selected "touchpad properties". Then, all I did was deselect the box "Edge Swipes Overview" and that got rid of the problem with the touch pad. I can still use the pad, but it is now not hypersensitive when I'm typing on the keyboard, which is great.

If you don't own a current computer, see if this helps you track it down on your computer, even if it is not an HP computer. I also own a 3 year old Gateway computer and find I can not find a applications window for the Touch pad. I noticed this on another computer my cousin owned, an MSI, which didn't have controls for the touch pad. So I'll understand if you have trouble finding controls. At least the new HP Touchscreen units have extensive controls for the touch pad.

At least this will give you some food for thought on how to track down the problem and fix it. Point is, you should have a touchpad control feature on your laptop if you have a touchpad and you should be able to adjust it's settings. Contact the manufacturer if you are having trouble finding the feature. They can't install a touch pad without controls you'd think. Good luck.
