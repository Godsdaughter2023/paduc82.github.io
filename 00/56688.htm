rdpclip is used to allow use of the clipboard in Remote Desktop sessions. As you can't remote desktop to a Windows 7 Home system, it isn't shipped with those systems.

mike hall... always say:

The solution - Find the hidden devices and uninstall them.

This is how you get them to show..

  1. Right-click on My Computer
  2. Select Properties from the context menu
  3. Click Advanced tab
  4. Click Environment Variables
  5. Click New
  6. In the variable name put devmgr_show_nonpresent_devices
  7. In the variable value put 1
  8. Click OK to exit

Open up Device Manager, click on VIEW and select show hidden devices.

Open up the keyboard section, and uninstall anything that is greyed. You will need to check the box to completely uninstall the drivers.

Then install the downloaded PS/2 driver that you already have..
