joeu2004 say: TBoyd1120 wrote:

Thanks for the suggestions - I had already tried both of those and neither work.  When the data was extracted from another system it populates upon export to Excel it displays in the cell as 1:12 (one minute and 12 seconds); however, when you "click" on the cell and look at the formulat bar it displays as 1:12:00 AM.  Not sure how to get rid of the AM/PM.  Everything I have is only in minutes and seconds - no hours.  Any other ideas?

That is right:  the Formula Bar has its own formatting rules.  It always displays AM/PM.  BTW, it also fails to display fractional seconds.  Nothing can be done about that.

However, be careful with entering 1min 12sec as 1:12.  Excel interprets that as 1hr 12min, even if the cell is formatted as Custom mm:ss.  That is by evidenced the fact that you see 1:12:00 AM in the Formula Bar.

You must enter 1min 12sec as either 0:1:12 or 1:12.0

If you must enter 1min 12sec as 1:12 (i.e. that is how it appears when "extracted"), you will need to use a helper cell to convert 1hr 12min to 1min 12sec.  One way:  enter =A1/60.

Sometimes Internet Service Provider’s (ISP’s) e-mail account only work when you are using the ISP's own network. This is for protection against outside spammers trying to use the ISP e-mail gateway. Outgoing emails need to be authorized by the ISP for POP-SMTP clients.

When you are connected to public wireless network you may have to change the outgoing server settings. You may contact the ISP (public wireless network ISP) for their Outgoing server settings, change the settings appropriately in Outlook and check if it works fine.

 

To change the server settings in Outlook, click on File< Account Settings< double click on the email account and change the server settings appropriately.

At home you have the same internet service provider (ISP) and email service provider (ESP) hence the server settings set in Outlook will work fine, and Outlook will be able to send and receive emails.

But when you are connected to a different ISP (in your case Public wireless network) you will be able to work on internet, access all the internet sites but you won’t be able to send emails, because every email sent from Outlook will be sent through the outgoing SMTP server. Since you are connected to a different internet connection you will be able to receive email as the incoming server will be the same but won’t be able to send emails as the Outgoing server will differ.
