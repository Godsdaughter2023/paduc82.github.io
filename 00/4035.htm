john korchok say:

Your topic is different than the thread in which you posted, so I moved it to be a new question.

Your Normal.dot is now Normal.dotm. You wrote "it doesn't work anymore", but didn't post any details of exactly what steps you've or what the results were.

Installing updates normally does not make any change to Normal.dotm, so I'm not clear why you associate updates with having to create a new Normal.dotm.

It's quite easy to extract all the information from Normal.dotm to a separate file. This is a good idea to create backups of your template contents as well:

  1. Create a new blank document, then save it as a macro-enabled template. Leave it open.
  2. In Word, choose Tools>Templates and Add-Ins.
  3. Click on the Organizer button.
  4. There are 2 lists, with your new template on the left and Normal.dotm on the right. In addition, there are 4 panes for Styles, AutoText, Toolbars and Macro Project Items.
  5. Start with the styles pane. Select all the styles you want to copy on the right side, then click the Copy button.
  6. Repeat step 5 for the AutoText, Toolbars and Macro Project Items.
  7. Close the dialog and save your new .dotm file.

davidf2 say:

Think about it. A Publisher template is a publication made up of various design elements including text boxes. If you fill the text box and then save as a template then when you bring up the template again that text box is going to reflect the state it was in when it was created. If you don't want a text box to be filled with text, then don't save it with the text. If you don't want a picture holder to have an actual picture, then don't save as a template with the picture inserted. In other words leave the variable elements 'empty' and save as a template again.

Note go to View > check Boundaries. For some reason MSFT decided to uncheck boundaries as the default which means you cannot see an empty text box unless you happen to mouseover it.

a. user say:

Hi Jeffrey,

Thank you for posting your query on Microsoft Community.

Try this to restore down your Microsoft Edge Window:

1. Maximize the Window
2. Restore Down
3. Manually enlarge the window by dragging the left and right edges to fill the screen
4. Maximize

You can also use the following keys:

To Maximize: Windows key + Up Arrow

Close the app: Alt+F4

Snap the app to the left side of the window: Windows Key+Left Arrow.

Snap the app to the right side of the window: Windows Key+Right Arrow.

To Minimize: Windows key + Down Arrow

Hope this helps. Please respond if you have further related queries.

Regards,

Jesinta Rozario
