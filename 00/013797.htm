aravinda balakrishna say:

It is not possible to have the date sequence in chart area. Plot area only plots numbers that is why dates are created to numbers.

Since you are plotting dates, the dates are being converted to numerical values and these values are being plotted. If you want the date formatting on the axis, then the first column of your chart data should be the dates. This will allow you to treat the axis as a date instead of a number.

 

For example, create a new data range, with dates in the first column, and some random numbers in the second column. Select this data range and insert a chart. Now, if you select the date axis and format the axis, you will see that the data is automatically being treated as a date and give you options to set the axis in days/months/years.

 

Unfortunately, as the other parameter in  your data is plain text and not numbers, it is not possible to plot it if you have dates in the first column.
