I am trying to retain all of the tags and detail I have added to thousands of pictures and videos done via Windows Live Photo Gallery when moving to a new PC.  Cna somene clarify what each of the files contains in the User > AppData > Local > Microsoft > Windows Live Photo Gallery folder?

 

SqmApi folder

Pictures.pd6


Providing that your photos are in JPEG format, then People tags, Captions, Descriptive tags, geotags and Ratings will be stored as metadata in the files themselves, so that information travels with the files.

However, if your photos are in RAW format, then Windows Live Photo Gallery will NOT store any of this information as metadata in the files, but only in its local database (Pictures.pd6) on the computer where you added the tags and/or captions.

Note that Windows Live Photo Gallery also has the ability to mark photos with "Flags". These Flags are never stored as metadata in any image files, but only in the local database.

When it comes to video files, it depends on the video format as to whether WLPG will store tags in the files themselves, or just in the local database. I'm sorry, I don't know which formats can store metadata. I suggest that you do a test: copy some of your video files to another PC, and see if the WLPG running on that PC will display the tags. If it does, then the files themselves contain the metadata, which is then read in by the WLPG running on that PC.

As far as I'm aware, the FaceExemplars.ed1 file contains the example faces displayed in the WLPG Ribbon, while FaceThumbs.fd1 file contains information on the faces found in your photo collection.

NavTreeState.dat stores the current state of the Navigation Pane on the left of WLPG, so that when you start up WLPG, it returns to where you were in your previous session with WLPG.

I don't know the purpose of the SqmApi folder - I suspect it holds your credentials for signing in to online services for Publishing your photos online.

To be honest, I personally wouldn't copy across any of these files to a new PC. Providing your photos and video files contain metadata, then I would simply let the WLPG on your new PC read in the metadata from your copied photos and videos and recreate its own version of the database. Copying across the database files to a new PC is not guaranteed to work, as far as I'm aware. But I could be wrong about that. It would be good to get the definitive answer from someone in Microsoft on this issue.
