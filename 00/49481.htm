wunders say:

Generally, the way to enter Safe Mode is to continually tap the F8 key at a rate of about two taps per second as the system boots.  If your system is an older system that originally came with a PS/2 keyboard (small round connector, usually orange), and you've replaced it with a USB keyboard and your BIOS does not support USB (Windows supports USB after it boots, though), then you may need to switch back to a PS/2 keyboard to do this.

 

Another method to start in Safe Mode is to boot from a bootable CD (such as Hirens) and edit the C:\boot.ini file to add " /safeboot:minimal " at the end of the line under [operating systems].

 

Differences between XP [Home] and XP Pro...  XP Pro is more geared toward a professional environment where access control and automatic update and policy installations are desired. Most differences are network-related. A few are listed as follows: 

XP Pro allows you to turn off "Simple File Sharing" to allow users to individually authenticate when connecting over the network to the computer.  XP Home enforces Simple File Sharing such that all network connections always authenticate as "Guest" user. 

XP Pro can have a "Security" tab under File and Folder Properties that allow file access permissions to differ per logged-in user.  XP Home can do this but only from the command prompt window using the "cacls" command (or as an administrator in Safe Mode).

XP Pro allows 10 concurrent network connections, Home only allows 5.

XP Pro has a "Group Policy Editor", XP Home does not.

XP Pro can join Domains;  XP Home can't (Workgroups only).

  JW
