lemp say:

"Clear cache" or "delete cache" or "clean cache" generally means to delete the files that your web browser has saved that make it work a bit faster to display pages when you re-visit a particular web site.

The specific instructions vary depending on the web browser you are using.

If you have Internet Explorer version 11 (the current default web browser for Windows 7):

  1. With Internet Explorer open, click the "gear" icon at the top right of the window (or press ALT+X) and then click "Internet Options"
  2. Near the bottom of the "General" tab, in the "Browsing history" section, click the "Delete" button
  3. Make the selections you want; you'll probably want to check the top option ("Preserve favorites website data") and UNcheck the "Passwords" option (otherwise, your saved passwords will be deleted).  To clear the "cache," check the box next to "Temporary Internet files and website files"
  4. Click the "Delete" button

If you have Firefox:

  1. Click the Firefox menu icon (the 3 horizontal bars at the top right) and then click "Options"
  2. Click "Advanced" in left menu
  3. Click the "Clear now" button to the right of "Cached web content"

If you have Google Chrome:

  1. Click the Google Chrom menu icon (the 3 vertical dots at the top right) and then click "Settings"
  2. Click "Advanced" at the bottom of the page and continue scrolling down
  3. Click "Clear browsing data"
  4. Make the selections you want (see comments above)
  5. After you've selected/unselected the desired options, click "Clear browsing data"

If a problem does not reappear when you start in safe mode, you can eliminate the default settings and basic device drivers as possible causes. If a recently installed program, device, or driver prevents Windows from running correctly, you can start your computer in safe mode and then remove the program that is causing the problem.
