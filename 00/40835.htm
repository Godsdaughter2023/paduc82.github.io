You could borrow a Windows 7 System repair disk from someone else - it must match your type of system [32 or 64 bit] but it does not have to be the same Windows 7 Edition [Home Premium etc].

1  You might still be able to download System repair disk ISOs from

Windows 7 System Repair Disc 64-bit

or

Windows 7 System Repair Disc 32-bit

To check whether you have a 32 or 64 bit Windows, go to Control panel, [View by - Small icons, if necessary], System.  The type of system is identified in about the middle of the screen.

There are a host of tools that can then create a bootable System repair disk

WinUSB Maker - Josh Cell Softwares v2.0beta

Download WinUSB-Maker from SourceForge.net - v1.8

If these sites do not work any more then search online for Windows 7 System repair disk ISO to see if you can get them from another site.  Scan the files with at least one anti-malware application before using them.

*  MS are closing down many of these sources as they are distributing MS software without permission.  So if you are ever going to download these ISOs, do it now.  I last used the links in July 2013.


2  After you make the disk, test it:-

2.1  Boot from it, then

2.2  Select keyboard & language,

2.3  Select Use recovery tools...* [and note the drive letter being used for the Windows drive** before clicking on Next] then

2.4  Select Command prompt or just restart the system from that menu to get back into Windows itself. 

The disk has now proved that it can boot correctly - there are some errors with System repair disks that only become apparent when you boot from them [nothing seems to be wrong with them when you make them].

*  This is not part of the test but you'll see in this dialog another option at the bottom of the Windows - Restore from an image.  This is what you will select if you are ever using the disk to restore from a system image.

** This is not part of the test but you would need to know this if you then tried looking around while you were in the Command prompt.  These disks apply rules that are different to those of Windows itself so your C:\ drive might be D:\ [or anything except X:\] but will revert to C:\ when you later reboot into Windows normally.
