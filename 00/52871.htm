To increase security and performance, Windows 10 in S mode runs only apps from the Microsoft Store.

If you want to install an app that isn't available in the Microsoft Store, you'll need to permanently switch out of S mode. There's no charge to switch out of S mode. Please follow the steps below.

  1. On your PC running Windows 10 in S mode, open Settings  > Update & Security > Activation.

If the system says it's taken, then it's in use somewhere on Microsoft's servers. The tag system is in use on other Microsoft services, so even if it's not in use on an Xbox, it may be in use somewhere else. The system also doesn't see things like spaces as making a gamertag unique. So if you were to look up "gamertag" it may not show up even though it's actually in use as "gamer tag". There's no way to take a gamertag if it's unavailable. If it's in use, the original owner would have to change the name or if they do not sign into the account for 5 years (starting in 2016) it will be freed up.

Improper mic configuration or outdated driver are some of the factors that can prevent the recording device from working properly. To isolate your concern, we suggest that you try the following methods:

  • Ensure that mic is set as default device to rule out improper settings issues.
    1. Right-click on the speaker icon in the notification area.
    2. Select Recording devices.
    3. Under the Recording tab, click the Set Default button.
    4. Click OK.

  • Run the Recording Audio troubleshooter to identify and fix and error.

    1. Type Troubleshooting in the search box and press Enter.
    2. Select HardwareandSound.
    3. Select RecordingAudio.
    4. Follow the on-screen instructions to complete the diagnostic.

  • Update device drivers to ensure that your device is running on it's optimal level. To do this follow the steps provided on this link on how to update driver.
    Note: The recording device driver is under Audio inputs and outputs.
