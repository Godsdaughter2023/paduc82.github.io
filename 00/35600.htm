Hi,

 

When you upgrade from Windows Vista to Windows 8 you can keep Windows Settings and Personal. Refer to the question 'What can I keep when I upgrade?' in the following article:

http://windows.microsoft.com/en-US/windows-8/upgrade-to-windows-8

 

As for your data, check if you can find a folder 'Windows.old' in C drive (Where 'C' is the operating system drive). If yes, then refer to the following article which explains: How to retrieve files from the Windows.old folder?

 

Generally outlook 2003 saves the all data in '.pst' file which will be stored in the following location

in Windows Vista: 'C:\Users\<username>\Local Settings\Application Data\Microsoft\Outlook'. In Windows 8 if you find Windows.old in C drive; navigate to the following location:'C:\Windows.old\Users\<user name>\AppData\Local\Microsoft\Outlook' and you will find the  file. Note that the appdata folder could be hidden, to view the folders and its contents enable the following options:

a) In 'Windows explorer' click on ‘Organize’ at the top left and select ‘Folder and Search Option’.

b)    Click on the ‘View’ tab and under ‘Advanced Settings’; Check the radio button ‘Show hidden files, folders and drives’ & Uncheck ‘Hide protected operating system files’.

To create a USB flash drive

  1. Click USB flash drive.

  2. Plug the USB flash drive into a USB port on your PC.

To create a DVD

    Hi,

    Thank you for writing to Microsoft Community Forums.

    1. May I know the make and the model of the computer?
    2. Are you aware of any changes made on your computer before this issue occurred?
    3. May I know the Windows 10 version are you using?

    You can press the Windows logo key + R, type winver in theOpenbox, and then select OK to find out which version of Windows your device is running.

    I suggest you to try the following methods and check:

    Method 1: Remove password using Command Prompt.

    1. In the search box on the taskbar, type cmd in the and then right-click or press and hold Command Prompt.

    2. Tap or click Run as administrator.

    3. To disable the sign-in option while your device is running on battery, type the following command and press Enter:

      powercfg /SETDCVALUEINDEX SCHEME_CURRENT SUB_NONE CONSOLELOCK 0

    4. If you want to disable the sign-in option while your device is plugged in, type the following command and press Enter:

      powercfg /SETACVALUEINDEX SCHEME_CURRENT SUB_NONE CONSOLELOCK 0

    Method 2: Skip password prompt after sleep using Settings.

    1. Select the Start, then select Settings

    2. Click on Accounts.

    3. Click on Sign-in options.

    4. Under "Require sign-in," choose Never from the drop-down menu to complete the task.

    Once you completed the steps, you'll no longer be required to enter a password after waking up Windows 10 from sleep.

    Method 3(For Windows 10 Pro PC): Disable Require Sign-in on Wakeup for All Users in Local Group Policy Editor.

    1. In the search box on the taskbar, type gpedit.msc and then select gpedit.msc.

    2. Open the Local Group Policy Editor.

    3. In the left pane of Local Group Policy Editor, navigate to the location: Computer Configuration\Administrative Templates\System\Power Management\Sleep Settings

    4. Double-click the policy you want to enforce: Require a password when a computer wakes (on battery) or require a password when a computer wakes (Plugged in).

    5. Check the Disable option in the top-left corner.

    6. Click Apply.

    7. Click OK to complete the task.

    Reboot the PC to take effect

    Hope it helps.
