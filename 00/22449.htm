lambchop123 say:

Words breaking to another line

I use Office 2010, academic version. 

When I type a paragraph in word, the last word of a line breaks.  e.g.

I have done everything I can to finish my hom

ework.  

 

The word "homework" break itself.  I tried to use "del" , "backspace"  to put the word together, but it seems the word is still in one word but it just breaks itself down into two lines.  

 

Any idea?

thanks

 

suzanne s. barnhill say:
This is characteristic of a damaged font or a corrupt document. The former is less common, but you can test it by switching to a different font to see if that solves the problem. If a bit in the font file gets switched so that Word thinks the font is a "decorative" font (that is, a symbol font, like Wingdings or Symbol), then it can break the word anywhere.
 
If the font is not the issue, then likely the document itself is the problem. There are tricks that can be used to deal with corrupt documents at http://word.mvps.org/FAQs/AppErrors/CorruptDoc.htm, but some of these cannot be used for the Word 2007/2010 XML format. So your first line of attack would be to save and close the document, then reopen it using the Open and Repair option.
