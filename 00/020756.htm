tom ogilvy say:

assume date is in A1 in YYYY-MM-DD  format

 

in B1

 

=DATE(LEFT(A1,4),MID(A1,6,2),RIGHT(A1,2))

 

format cell B1 as you wish it to appear (mm/dd/yyyy)

 

or you can do

 

=Text(DATE(LEFT(A1,4),MID(A1,6,2),RIGHT(A1,2)),"mm/dd/yyyy")

 

if you want a string value in cell B1

 

you can then drag fill either of the formula down the column.

fedbil100 say:

Online Banking

rob koch say:

Billy,

Moderators manage the forums, they aren't necessarily any more knowledgeable than any other forum member, though they do have special privileges that allow them to modify, delete or edit any post or thread.

Igor has already provided an answer, but it doesn't fully explain the true design behind what these specific products call "online banking" or payment system protection, so I'll cover that here with references to those products own pages describing how they supposedly perform this.  I'll also try to explain why all of this is mostly just marketing and how Windows Defender and most other security programs already do most of these same things for all websites, not just banking or payments.

First, both Bitdefender Safepay and Kaspersky Safe Money are basically nothing more than a specially-designed "protected" web browser built into the security product.  So rather than using Windows Edge or another 3rd-party browser, the security product instead detects the fact that a URL is one from a list that indicates this is a banking or other payments website and either offers to or automatically switches you into their own browser instead.

Though this "protected" browser is assumed by the security vendor to be better than other browsers, in truth it's just a simpler browser that doesn't have all the features these others do, so it's inherently less likely to be compromised by things like malicious plug-ins or other add-ons which these more limited browsers don't support.  That doesn't really make these browsers "better", just too simple to contain most types of security risks or vulnerabilities that the more capable Edge and other browsers may possibly contain.

In truth though, these limited browsers aren't perfect either and are far less well tested by 3rd-parties that test the Microsoft and other major browsers, so you are assuming that the limited capabilities and internal testing done by the security vendor themselves is sufficient to better protect you from malicious threats, which may or may not be true.

As an example of what they do, here's the Kaspersky short list from this What is Kaspersky Safe Money and How is it Useful article, with my own description of how Windows or the Edge browser does these same things for any HTTPS [secure] website you visit.

1. Trusted Site Database is an important factor of Safe Money, and the database is used as a reference. If there is no customization made from user, the URL is cross-checked with the database before proceeding. If two URLs match, the Safe Money browser will be launched. There’s another perk of escaping from phishing, because the site is opened through the Kaspersky database.

Microsoft Edge doesn't use a list or special browser, it just assumes that any secure website requres the same HTTPS SSL (Secure Sockets Layer) encrypted communications protection, as well as anti-phishing (via SmartScreen Filter) and other website protections as any other.  No need for special handling, that's just what these protections always do.

2. Your Connection is also checked for trustworthiness before proceeding. That is, several factors such as site’s certificate will be analyzed by Safe Money. If there are some issues with site’s in-bound security, you will be blocked.

That seems like a nice added feature, since it lets the user know by completely blocking their access to the website, which some less capable users with feeble minds might need.  However, in truth, all web browsers including Edge have included this capability for decades, but instead typically display a small secure "padlock" icon in the address bar to indicate that a website is secured with SSL.  For example, just click the padlock icon for this website above to see that information for the answers.microsoft.com secure website these Community forums reside on.

3. PC Environment is also taken into consideration by Safe Money! The OS will be scanned for possible threats, errors and vulnerabilities. It’s done so carefully that nothing would interfere the functioning of Safe Money.

"OS will be scanned for threats, errors (huh? what errors?, not sure what this means) and vulnerabilities", isn't that exactly what Windows Defender and in fact any modern security program is doing all the time?  So what's different about that?

So as you can see, though the isolated and limited protected browser and lists of banking URLs, along with other things like displaying or even blocking a website based on its secure SSL status seem like something special, they're really just slightly modified ways of doing the same things that most any modern browser or Windows does already.

Thus, what these special "online banking" features truly are is mostly nothing more than marketing, displayed with smoke and mirros in a way that unknowledgeable users might think they're something extra special, so aiding in selling the security vendor's products.

Now that's not to say that Microsoft couldn't improve and simplify the way that some of these features work, such as making the fact that a website's SSL certificate isn't current or correct much more obvious.  But in fact they do to some extent now, since they'll pop up a warning message if so and require the user to go through some hoops to bypass this message befoe gaining access to the site.  Most people just don't run into such sites, so they aren't aware the Edge browser already does this in a different way.

Another thing to note is that on occassion these protected browsers don't actually work properly with secure websites, in some cases even because the bank itself doesn't feel the way these products work is actually "safe" for their customers.  So in those cases the products are actually a negative for the safe, simple use of online banking for the customers affected.

So what you think of these online banking features to some extent depends on how well you understand that Windows security already works and whether you'd know what to do if any of these security messages were displayed.

To me, these online banking features are just marketing and add little to no true security, though they may make some of it more obvious for some less knowledgeable consumers.
