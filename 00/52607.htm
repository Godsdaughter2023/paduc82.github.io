The reason you are seeing more 'Service host' entries in Task Manager is because prior to Creator Update - you would have only seen 5-10 'Syshost' entries but each one contained several (up to maybe 20)  'Service host) entries.

Now instead of 'Service host's'  being grouped together, each 'Service host' is listed individually, this supposed to make it easier for tracing information / problems with 'Service host' entries - you may have any amount of entries listed depending on haw many services you have running.

It is not likely they are using as much memory as you may suspect, each service uses on ave.  approx. 6.0MB of memory - so if you have say 100 services listed they are only using maybe 600MB of memory, which is small compared to other processes / apps - such as your 'Antimalware Service Executable' which is using 1472MB of memory.
