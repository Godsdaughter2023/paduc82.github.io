jeanette cunningham say: Here's code that works for me, used in the mouse down event of the hyperlink text box - the text box in my code is called 'UrLink'.

Private Sub UrLink_MouseDown(Button As Integer, Shift As Integer, X As Single, Y As Single)
On Error GoTo Err_Handler
    If Button = acRightButton Then
        DoCmd.RunCommand acCmdEditHyperlink
    End If

Exit_Handler:
    Exit Sub
Err_Handler:
    Debug.Print Err.Number, Err.Description, "UrLink_MouseDown", Now
    Resume Exit_Handler

End Sub

tom van stiphout say:

CF cannot do this directly, but you could add a column to your query that does this, then use CF to look at that field to decide what to do.

The new column would be created with another instance of the table, joining on appropriate fields, with one field having criteria of Max less than the main table. That would get you the previous record. Then check if it fits the criteria.

Previous record queries are not uncommon. You should be able to find help online, e.g. http://stackoverflow.com/questions/1067672/how-to-select-previous-row-value
