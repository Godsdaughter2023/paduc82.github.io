As well as what JohnK said:

if you resize ONE pic as you need, select it and run this code it might do what you need.

Sub resizer()
Dim osld As Slide
Dim oshp As Shape
Dim sngL As Single
Dim sngT As Single
Dim sngH As Single
Dim sngW As Single
'select one picture AFTER resizing it
err.Clear
On Error GoTo err
With ActiveWindow.Selection.ShapeRange(1)

.LockAspectRatio = False
sngL = .Left
sngT = .Top
sngH = .Height
sngW = .Width
End With
For Each osld In ActivePresentation.Slides
For Each oshp In osld.Shapes
If isPicture(oshp) Then
oshp.Left = sngL
oshp.Top = sngT
oshp.Width = sngW
oshp.Height = sngH
Exit For
End If
Next oshp
Next osld
Exit Sub
err:
MsgBox "Did you select the picture?"
End Sub

Function isPicture(oshp As Shape) As Boolean
If oshp.Type = msoPicture Or msoLinkedPicture Then
isPicture = True
Exit Function
End If
If oshp.Type = msoPlaceholder Then
If oshp.PlaceholderFormat.ContainedType = msoPicture Or _
oshp.PlaceholderFormat.ContainedType = msoLinkedPicture Then
isPicture = True
Exit Function
End If
End If
End Function

Hello John

A few house-keeping things

#1 - If you still have access to the system where '2003 is installed (where the PST file) is from, then the first thing you should do is run ScanPST on that file on that system to eliminate any potential errors in the PST file itself

#2 - You really don't want to import your data into "This Computer Only" folders. Instead simply open the PST file in Outlook and set it as the default data file for the profile. The "default data file" is where the default contact/calendar folders are stored. The reasons you don't want to have the OST file for the IMAP account set as the default data file are

a) if the IMAP account is ever removed from the profile, the OST file is also deleted from the system and the data in the "This Computer Only" folders will be lost
b) similarly, if Outlook cannot find/open the .ost file for the account, it will create a new one and sync the email data with the server. The "This Computer Only" folders will be empty

To add the PST file to the profile.......
Go to File --> Open & Export --> Open Outlook Data FIle --> Select the PST file

To set the PST file as the "default data file"..............
Go to FIle --> Account Settings --> Data Files tab --> highlight the PST file --> set it as the default

3) You should also confirm whether this PST file from '2003 is an ANSI or UNICODE PST data file. Once it's added to the profile, highlight the <Root> of the PST file (level above the <Inbox>) --> right-click --> select Data FIle properties --> click on the <Advanced> button --> if the <format> field contains "97-2002" - it is an ANSI file otherwise it's a UNICODE file.

If it's an ANSI file, would suggest exporting the contact /calendar folders to a new PST file and then
a) add the PST file just created from the export to Outlook
b) set the newly added PST file as the default data file
c) highlight the old PST file --> <remove>

Hi SeanMcCandless and welcome to the Skype Community! 

You can add an alias to your account and change your sign in preference. This is a great alternative if you want to manage several email addresses conveniently from a single Microsoft account. Follow the suggested steps below on how to change your sign in preference. 

  • Sign in to Microsoft account website.
  • Click Your Info.
  • Select Manage how you sign in to Microsoft. 
  • Click Change sign-in preferences.

Note: Changing your sign in preference will only change how you sign in to your account, but this will not change the email address that was used to initially create the account nor your Skype Name. 


For additional details, please refer to this FAQ from the Microsoft support site. Manage aliases on your Microsoft account 

Hope this helps! If you need additional information, let us know so we can assist you further.
