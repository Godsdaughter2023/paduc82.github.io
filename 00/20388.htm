Lock columns to prevent changing the width

Is there a way to lock the columns of the full spreasdsheet to prevent someone from changing the width of the columns. They should still be able to modify the height, add/remove text, etc.

hansv mvp say:

First, select all cells and clear the Locked check box in the Protection tab of the Format Cells dialog (Ctrl+1).

Then, click Protect Sheet on the Review tab of the ribbon. Tick the check boxes for everything you want the user to be able to do, except obviously "Format columns".

Click OK; a password is optional.
