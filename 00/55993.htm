6.    If there is no "Other Devices" category, then in the "Ports (COM & LPT)" category you should see a device called "Prolific USB-to-Serial Comm Port" OR "ST Microelectronics Virtual COM Port" with a COM value in brackets next to it. RIGHT click this and click Update Driver

8.    In this window, again click the bottom option (it will say something about not browsing the computer but that you will pick it from a list) and click Next
9.    In this window you should see a white box area with a couple of renditions of the same thing...except they have different dates in brackets next to them. Choose the oldest date by clicking it to highlight it
10.    Click Next and it should install the driver accordingly.

For other hardware obviously the process would differ with regards to what you device you are working with and what to look for in Device Manager but the update process stays the same.

MS services in Stopped status in Config - Microsoft Community

sjones21458 say:

MS services in Stopped status in Config

I have 38 MS services in stopped status in Config, is that unusual?

a. user say:

Nope.  I have 68 showing as 'Stopped' in msconfig.  But my configuration is very lean and I also don't have any item listing in the msconfig Startup tab (zero).  That works for me, but it might not work for you.

There are lots of Services that may be stopped or completely disabled depending on your system configuration.

There are also many XP Services that get installed as an Automatic startup item that you might not need to be starting at all (again - depending on your configuration).  There are some Services that are configured to start Automatically that you might not need that if running, leave your system vulnerable to outside threats.

If you check the box 'Hide All Microsoft Service' you will see other Services that may have been installed by third party applications you have installed.

If you want to learn more about the XP Services and which ones you need or might not need, a good place to start looking is here:

http://www.blackviper.com/service-configurations/black-vipers-windows-xp-x86-32-bit-service-pack-3-service-configurations/

Using a search engine like Google can also help you decide which Services you need to have running, starting or disabled - or ask here if you are not sure.

If you think you are going to be messing around with your Services, it is prudent to research them first and if you are going to start changing things, only do one or two at a time remembering and keeping track of what adjustments you make so you can undo the adjustments if something goes wrong.

It is also possible to render your system unbootable if you disable certain Services so it is also prudent before you start to have a plan for what you are going to do if your experimentation or tinkering leaves your system in an unbootable state - how are you going to undo things if your system will not boot?

It would be a good idea to make a manual System Restore point and registry backup before starting too.  Always make sure you have a way to recover in case you get into a jam.

Many Services 'depend' on other Service to be running for them to start so if you disable one thing thinking you don't need it, that might prevent other Services that you do need from starting.

Example:  There are some Services you might think you don't need and might disable (like Task Scheduler) but just because you might not have any Scheduled Tasks, it is not a good idea to disable that Service since other XP functions depend on that to work properly.

Example:  Much MS Answers advice says to set things like the Help and Support Service to 'Automatic' start, but I always set that to 'Manual' so it will only start when it needs to start.  There is no need for the H&S Service (and others like it) to be starting automatically every time your system boots and running all day with nothing to do except consume memory and CPU cycles that you would probably prefer to go towards something you really do need.  All that stuff slows you down

You might be able to disable many 'Updater' Services like Google, Flash, Skype and just update those things when you need to update them or they will tell you they need an update when you run them.

I also always disable the Java Quick Starter Service. and the associated jusched Startup item.

You can shave many seconds off your boot time without these things starting every time you reboot your system and running all day with nothing to do.

Of course the only way to really see the effect is to have a good way to accurately measure your startup time before and after adjustments instead of just saying "Well, it seems faster" or "It might be faster".   Accurately measuring to the 100th of a second also eliminates things like "It takes my system about 40 seconds to boot (or 5 minutes)".  Using software tools, there is no "about" to it.

There is free and easy to use software to help you do all that of course, such as:

http://www.greatis.com/utilities/bootlogxp/

how to pair your microsoft bluetooth mobile keyboard 6000 if you are - Microsoft Community

scott wagnon say:

how to pair your microsoft bluetooth mobile keyboard 6000 if you are unable to pair

the keyboard like any other device is considered one device. but unlike other devices, this keyboard is unique, in that is has TWO parts working as ONE device.

when a device goes in to discovery mode, it does not interfere with other bluetooth devices that are currently connected, so while the devices may be on and connected, they do not interfere with other devices that are in discovery mode.

HOWEVER when you attempt to get your PC to discover the keyboard device something different is happening and that is interference. but from what? YOUR NUMBER PAD IS POWERED ON AND THEREFORE INTERFERING WITH THE KEYBOARD WHEN IT IS IN DISCOVERY MODE. the number pad is on but because is part of ONE device, it is creating a sort of confusion for the pc or mac (older software). the pc is confused about your keyboard because is getting mixed signals? is it wanting to connect or not? so it refuses to accept the disovery mode, because the number pad (which is the same device as the keyboad) is saying something different than the keyboard.

SO THE SOLUTION IS SO SO SIMPLE.

before you start, MAKE SURE THAT EITHER THE KEYBOARD OR THE NUMBER PAD IS POWERED OFF BEFORE YOU GO INTO DISCOVERY.  so if you are connecting the number pad, TURN OFF THE KEYBOARD. if you are connecting the keyboard, TURN OFF THE NUMBER PAD.

once you are in the add device box, press the discovery button a couple of times to make sure that the pc or older mac is throughly connected with the keyboard or number pad. once you see the specific name and not the generic title that the pc is issuing, and you also see an exact image of the device in the box, select and press next.

once you are done with one part of the device. repeat this. NOW TURN OFF THE OTHER PART THAT YOU JUST CONNECTED, AND NOW TURN ON THE OTHER PART THAT YOU HAD TURNED OFF. and repeat this process once more.

there you go. that is your complete solution.
