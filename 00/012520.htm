Aside from making a backup being an excellent first step, when you move your contacts, follow the these steps

#1 - Highlight the source folder containing the contacts you want to move (Contacts - Prism MD)
#2 - Press CTRL+A to highlight all the contacts
#3 - Right-click on the highlighted contacts
#4 - Select <Move> --> select <Copy to Folder) instead of <Other Folder>

A section in Word means the whole document unless someone has added one or more section breaks (Page Layout tab | Breaks). In a single section, you can have up to three different headers, and three different footers; you accomplish that by selecting "Different first page" and "Different odd and even" on the Header & Footer Tools Design tab.

cbarnhorst say:

Will I be able to upgrade to the new Build on the 29th on my Build 10240 (which is Activated and has the Fast/Slow options available)? I intend remaining on the Insider Programme.

Yes.

 will I be able to upgrade on the 8.1?

Yes.  The two are not related.  Each is upgradeable on its own merit.

Will I then be able to clean install the new Build but on the other SSD?

If you upgrade the Build 10240 copy you will be able to perform a clean install either with the .iso or simply by using the Reset This PC tool in Settings/Recovery.  

Same answer if you upgrade 8.1.  

Each drive has its own upgrade path.  What you do with one doesn't concern the other.

Only if you clean install on a drive that has not had a Windows 10 Free Upgrade license registered, will you need to purchase a retail copy of Windows 10 in order to have a licensible product key. 

It sounds to me like you are good to go just as you are.
