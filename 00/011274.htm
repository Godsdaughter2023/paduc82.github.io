julie sheets say:

Is each "chunk" of task a mini-project in itself?  In other words is Increment #1 tied via predecessor/successor to Increment #2?

If Increment #1 ends without relationship to Increment #2, then you will want to turn on the option "Display multiple critical paths" to create the critical path within each path.

As far as "accurate" critical path, that will be changed with constraints and deadlines.  If I recall Project 2003 correctly, if you set a deadline or constraint that was earlier than the currently scheduled date, it would build slack into the schedule.  If Total Slack is greater than zero, the task is not critical.
