a. user say:

what is dos mode

Originally, Windows ran 'on top' of MS-DOS: First the computer started MS-DOS, then it loaded Windows. Sometimes, if you needed to get something done that Windows couldn't do on its own, you could work with MS-DOS directly. That was known as 'dropping to DOS mode' and it meant working with MS-DOS directly, even though Windows was running.

Running one operating system on top of another was not the best way to operate a computer. It was frequently unstable and so Windows crashed often.

But that was long ago. Beginning with WindowsNT, Windows became a complete operating system that did not need MS-DOS. Thus, ever since NT, there has been no DOS mode.

Many people confuse PowerShell with DOS mode. Just because you can run commands with PowerShell, that doesn't mean it's DOS mode. The real DOS mode hasn't existed since WindowsNT.
