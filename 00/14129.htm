ken sheridan say: Where you want to return rows from one table regardless of whether there is a match in another table an OUTER JOIN is used.  This can be a LEFT or RIGHT OUTER JOIN depending on which side of the join you wish to return both the matched and unmatched rows.  At first sight my guess would be that PricesDisc will be the table without the matching rows, so try changing INNER JOIN PricesDisc to LEFT JOIN PricesDisc to start with (the OUTER can be omitted).
