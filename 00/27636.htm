bob jones aka: cybertaz say:

When you define a Style there is a dropdown list of Style Types which contains 5 items:

  • Paragraph
  • Character
  • Linked (paragraph and character)
  • Table
  • List

The default is Paragraph, so if you don't specify otherwise the Style is defined as a Paragraph Style. It sounds like that parameter was left at the default.

As Stefan indicated, there is no separate pane for Character Styles. The Styles pane lists both Character & Paragraph styles: Paragraph Styles are indicated by a pilcrow [ ¶ ] to the right of the style name; Character Styles display the underscored l/c letter a [ a ]. (Linked Styles display both symbols [

Even if only 1 character is selected when applying a Paragraph Style the style will affect the entire paragraph.

However, unless you have a specific reason for doing so there is no need for a Style in order to apply Subscript or Superscript. In addition to those attributes being available in the Font group on the Home tab:

For existing text, select the text then use the built-in keyboard shortcut;

  • Command+Shift+〓 for Superscript
  • Command+〓 for Subscript

While typing, execute the appropriate shortcut first, type the character(s), then use the shortcut again to toggle the attribute off.

gord dibben say:

VBA Worksheet Event code to color activecell or cells when selected.

Right-click on sheet tab and "View Code" to access the sheet module.

Paste code in that module.  Alt + q to return to sheet view.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
'highlighter
'Mike H  does not destroy existing formats
        Cells.FormatConditions.Delete
        With Target
            .FormatConditions.Add Type:=xlExpression, Formula1:="TRUE"
            With .FormatConditions(1)
                .Interior.ColorIndex = 8
            End With
        End With
End Sub
