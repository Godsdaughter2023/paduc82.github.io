boblarson say:

Paste this into a new, standard module:

' ######### CODE START ##########

Private dteStart As Date

Private dteEnd As Date

Private strLocation As String

 

Function StartLog(strFileLocation As String)

    dteStart = Now

    strLocation = strFileLocation

End Function

 

Function LogTime(strQueryName As String)

Dim iFileNum As Integer

Dim strFileName As String

Dim lngElapsedTime As Long

   dteEnd = Now

strFileName = strLocation

iFileNum = FreeFile

' Opens the text file to write to

Open strFileName For Append As iFileNum

' gets the elapsed time

   lngElapsedTime = DateDiff("s", dteStart, dteEnd)

      Print #iFileNum, Now & " - " & strQueryName & " took " & lngElapsedTime & " seconds to run."

      Close iFileNum

   ' reset variables

 End Function

 ' ########### CODE END ##############

and then you would use the Action RUNCODE in your macro just before and after each query.  The first would be

ACTION: RUNCODE

ARGUMENT: StartLog("YourFileNameAndPathHere")

 

(so for example if you wanted the log named QueryLog.txt and you wanted it in your C:\Temp\ folder, you would use:

StartLog("C:\Temp\QueryLog.txt")

 

 

And then AFTER the query you would use

ACTION: RUNCODE

ARGUMENT: LogTime("NameOfYourQueryInQuotes")

 

And then it will create a text file which has the information in there for you.

#4d - Right-click on the select items --> select <Move> --> Select <Other Folder> --> select the <Inbox> in AC2 --> press OK

hansv mvp say:

Create the following macro in your personal macro workbook Personal.xlsb (see http://www.rondebruin.nl/personal.htm if you don't have a personal macro workbook or don't know what it is):

 

Sub ExportChart()
  Dim cht As Chart
  Dim strFile As String
  On Error Resume Next
  Set cht = ActiveChart
  If cht Is Nothing Then
    MsgBox "Please select a chart or chart sheet and try again.", vbExclamation
    Exit Sub
  End If
  On Error GoTo 0
  strFile = Application.GetSaveAsFilename(FileFilter:="PNG files (*.png),*.png")
  If strFile = "False" Then
    MsgBox "You didn't specify a filename.", vbExclamation
    Exit Sub
  End If
  cht.Export Filename:=strFile, FilterName:="PNG"
  Exit Sub
ErrHandler:
  MsgBox Err.Description, vbExclamation
End Sub

 

Switch back to Excel.

Customize the Quick Access Toolbar.

Select Macros from the "Choose commands from" dropdown.

The above macro will be listed on the left hand side as PERSONAL.XLSB!ExportChart.

Select it and click Add>>.

Select the macro in the list on the right hand side.

Click Modify... to change the name and icon.
