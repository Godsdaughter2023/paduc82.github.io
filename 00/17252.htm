Word is not the appropriate application for such things, a better way is to use Excel.

Within Excel you can separate the ASCII part and the number part from the filename 2 helper cells and then sort the table based on those 2 criteria.

macropod say:

Not that complicated, really:

Sub Demo()
Dim StrTxt As String, i As Long, ArrTxt() As String

'Build an input string:

For i = 20 To 1 Step -1
  StrTxt = StrTxt & vbCr & "Text " & i
Next

'Display the input string:

MsgBox StrTxt

'Parse the Input string and load it into an array,

'with the numbers formatted:

For i = 1 To 20
  ReDim Preserve ArrTxt(i)
  ArrTxt(i) = Split(Split(StrTxt, vbCr)(i), " ")(0) & " " & _
    Format(Split(Split(StrTxt, vbCr)(i), " ")(1), "00")
Next
StrTxt = ""

'Sort the input String:

WordBasic.SortArray ArrTxt(), 0, 1, UBound(ArrTxt()), 1

'Build an ouput string, re-parsing to remove the number formatting:

For i = 1 To 20
  StrTxt = StrTxt & vbCr & Split(ArrTxt(i), " ")(0) & " " & _
    CLng(Split(ArrTxt(i), " ")(1))
Next

'Display the output string:

MsgBox StrTxt
End Sub

It's not so much the older version of outlook but how the calendar was shared and how it was opened.
