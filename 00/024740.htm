How do I change the spacing? - Microsoft Community

How do I change the spacing?

How do I change from double spacing to single spacing?

[Original title: Microsoft Word]

jay freedman say:

If this is just for a single part of a single document:

  • Select all the text that you want to change to single spacing.
  • Click the Line and Paragraph Spacing button (in the Paragraph group on the Home ribbon )
  • In the menu that opens, click the 1.0 item.

If you want to change the default for all new blank documents:

  • In the Styles group of the Home ribbon, right-click the Normal style and choose Modify in the menu.
  • In the Modify Style dialog, click the "New documents based on this template" option at the bottom. (Don't click OK yet!)

  • Click the Format button at the bottom left corner and click Paragraph in the menu.
  • In the Paragraph dialog, change the Line Spacing dropdown from Double to Single.
  • Click OK in both dialogs.
  • If any text with the Normal style is still double-spaced, select it and press Ctrl+Q to clear the direct (non-style) paragraph formatting.
