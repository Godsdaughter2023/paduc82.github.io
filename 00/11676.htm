Need macro to scroll to a row based on search result - Microsoft Community

jon cohen say:

Need macro to scroll to a row based on search result

I have a large database in an Excel worksheet, with about a dozen columns and several hundred rows.  The data is sorted alphabetically, based on the content of the first column.  The top three rows are column labels and I have used "freeze pane" to keep the column labels always on top.

What I want to be able to do is to enter a few characters in an unused cell (in the top frozen rows) and run a macro that will search the first column and scroll the row to the top that contains content that is the first to be greater than the contents of the search cell.  Thus, if I have some rows with the following in the first column:

  • Emma
  • Peter
  • Sam
  • Sue
  • Tim
  • Vanessa

And if I put "SO" in the search cell, then executing the macro would put the row for "Sue" scrolled up to the top of the worksheet.  If I put "SO" in Cell O2, then "=MATCH(O2,A4:A669)+3" returns the number of the row that best matches the contents of O2; the "+3" is needed to skip past the 3 frozen rows.  So, in VBasic, how do I get to scroll down to the row specified in the "MATCH" function?

Any hints for the Visual Basic code for this macro?

hansv mvp say:

+3 would take you to the last cell with a value smaller than or equal to the search value. +4 will take you to the first value greater than the search value.

Right-click the sheet tab.

Select View Code from the context menu.

Copy the following code into the module that appears:

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim r As Long
    If Not Intersect(Range("O2"), Target) Is Nothing Then
        If Range("O2").Value <> "" Then
            On Error Resume Next
            r = Application.Match(Range("O2").Value, Range("A4:A669"))
            If r > 0 Then
                Application.Goto Range("A" & (r + 3)), True
            End If
        End If
    End If
End Sub

Change 3 to 4 if required.
