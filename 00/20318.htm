1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
5) Save the file as a macro-enabled .xlsm file.

The code should warn the user to not paste from the clipboard anytime that a cell with data validation is chosen (if they are in copy/paste mode) - or you can clear the clipboard.

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    Dim strV As String

    On Error GoTo noValidation

    strV = Target.Validation.Formula1
    If Application.CutCopyMode Then MsgBox "Don't paste on cell " & Target.Address & "!!!!"

    'Optionally, clear the clipboard
    'Application.CutCopyMode = False

noValidation:
    On Error GoTo 0
End Sub
