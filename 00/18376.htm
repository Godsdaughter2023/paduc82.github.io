hansv mvp say:

Option 1:

Hold down Alt and type 130 on the numeric keypad, then release Alt.

Option 2:

Hold down Alt and type 0233 on the numeric keypad, then release Alt.

Option 3:

Press the Windows key or click the Start button.

Type character and select Character Map from the search results.

Locate é in the list of characters, and select it.

Click Select, then click Copy.

Switch to Access and paste the character where you want it.

hans lu msft say:

Hi Carolhill1,

 

Thanks for the clarify. The attachments of original email would be removed automatically in the replied email, this is by design. But it is an issue if the attachments would be removed from the original email. 

And after the attachments disappear from the original email, Outlook for IOS would sync this action to server. That's why the attachments disappear from laptop as well.

 

Given this situation, it is recommended you to contact the in-app support of Outlook for IOS by clicking setting icon> Help & Feedback> Contact support where is a dedicated support channel for Outlook for IOS issues.
