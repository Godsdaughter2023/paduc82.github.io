The audio file might still be in the "OneNoteOfflineCache_Files" folder - this folder will be next to the "Audio Cache" folder.  In Vista/7/8 the path should be:

%userprofile%\AppData\Local\Microsoft\OneNote\14.0\OneNoteOfflineCache_Files

 

Or in XP:

%userprofile%\Local Settings\Application Data\Microsoft\OneNote\14.0\OneNoteOfflineCache_Files

 

Sort these files by Date Modified: hopefully your audio file will be near the top of the list with a .wma file extension.  Copy and Paste this back into OneNote.

SELECT Orders.StudentID, Orders.OrderDate,
