I expect to see Messenger disappear altogether, at least in the US, in the next year or so. Microsoft doesn't plan to provide the Rooms feature in windows 10, so unfortunately for those of us who use it now, it is being discontinued.

Many of the Rooms features are redundant to existing features..so I've already made a few changes on my phone so I can create and use my own "virtual" family room on the phone once the Rooms feature is discontinued:

  1. Creating & sharing a calendar, as needed, with those in my desired room - if I have to create a new shared calendar I will need to do that.
  2. Created a Skype group chat thread that includes the same set of room contacts. I created this group earlier this year and have gotten all my Room contacts shifted over to Skype already
  3. Created and shared a photo album for my room contacts - see the article I posted to my blog called "Create a virtual photo album via the OneDrive app"
  4. Creating a shared OneNote notebook, if needed, for my room contacts
  5. Creating a shared documents folder on OneDrive, if needed, for my room contacts
  6. Creating a group in the phone's People hub that includes my room contacts

For my existing Room, I've pinned all of these things to the phone's home screen except the calendar and Skype Group Chat thread - Skype is pinned as a separate tile on the home screen & any new messages to this thread will show up as notifications on that tile. Then I moved all these pinned items into a folder on the home screen. The only things I had to create for my existing Room are the Skype Group Chat thread, the People Hub group and the photo album, I pinned the currently used documents folder and OneNote Notebook so didn't create new ones for my existing room. So now I have one tile on my home screen, with all these family room items in it. It is not as elegant as the current Rooms feature, but still very useful.
