considerate_guy say:

There was a program that was part of Windows 7 called Windows DVD Maker that would allow you to author a video DVD. If Movie Maker was running on Windows 7 it would detect the presence of that program and then offer the "Burn to DVD" option in the menus. It would then pass the project file off to Windows DVD Maker to author the video DVD.

However, that program has not been included in later versions of Windows so if you have Windows 8, 8.1 or 10, you'll have to find other options.

Notice: The video DVD standard was created in the late 1990s before the era of HDTVs so it has a maximum resolution of 480p. It's really considered a "legacy" format and neither Microsoft nor Apple has direct support of authoring the video DVD format anymore. For example, Apple dropped iDVD from its iLife suite back in 2011.

There are a few things you can do:

  1. You can find 3rd-party software for burning a video DVD. You'll first have to save your project as a movie (File > Save movie) and then use the software to down-rez the video to 480p and the older MPEG-2 format needed on a video DVD.
  2. You can save your movie (File > Save movie) and then just burn the resulting .mp4 direction onto an optical disc (even a DVD as a *data* DVD). That movie is then playable on another PC or in a *Blu-ray* player. As a side benefit, the resolution will be HD rather than SD.
  3. You can always just share the resulting movie on video sharing sites like YouTube, Vimeo, etc. Or you can copy the .mp4 file onto a USB drive and give that to someone that wants to see your movie. Or if you want to keep a library of movies, you could put them on a larger capacity external hard drive.

If you upgrade your Win7 to WIN8 upgrade, the license for 8 is tied to the win7, which you cannot move or use ---while this win8 upgrade is on your system.

You do have the right to revert back to your Windows 7 on this pc, but you would have to wipe/delete the windows partition before doing a clean install of your Windows 7.

mark ole say:

Hi,

Windows updates and service packs may fail to install if there are corruption errors. For example, an update might not install if a system file is damaged. The DISM or System Update Readiness tool may help you to fix some Windows corruption errors.

To resolve this problem, use the inbox Deployment Image Servicing and Management (DISM) tool. Then, install the Windows update or service pack again.

  1. Open an elevated command prompt. To do this, swipe in from the right edge of the screen, and then tap Search. Or, if you are using a mouse, point to the lower-right corner of the screen, and then click Search. Type Command Prompt in the Search box, right-click Command Prompt, and then click Run as administrator. If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.
  2. Type the following command, and then press Enter.
    It may take several minutes for the command operation to be completed.
    DISM.exe /Online /Cleanup-image /Restorehealth
    Important When you run this command, DISM uses Windows Update to provide the files that are required to fix corruptions. However, if your Windows Update client is already broken, use a running Windows installation as the repair source, or use a Windows side-by-side folder from a network share or from a removable media, such as the Windows DVD, as the source of the files. To do this, run the following command instead:
    DISM.exe /Online /Cleanup-Image /RestoreHealth /Source:C:\RepairSource\Windows /LimitAccess
    Note Replace the C:\RepairSource\Windows placeholder with the location of your repair source. For more information about using the DISM tool to repair Windows, reference Repair a Windows Image.
  3. Type the following command and press Enter.
    It may take several minutes for the command operation to be completed.
    sfc /scannow
  4. Close the command prompt, and then run Windows Update again.

Please update us with the results.
