Do you mean that the equation appears like this?

That usually means that the paragraph that contains the equation has its line spacing set to an Exact amount that's less than the height of the equation. (It doesn't have to be an equation -- it could be an inline graphic or text with a large font size.)

Open the Paragraph dialog and check the line spacing:

If it's set to Exactly, change it to At Least. That will let Word increase the line height to accommodate the larger object.

If a specific style is applied to all the paragraphs that contain equations, change the line spacing in the style's Modify dialog.

rohn007 say:
I'm sorry but i'm not very familiar with both these terms. What is NORMAL.DOT and QAT, and what does this mean for my problem?
No problem, you are learning, and that is what this site is set up to do, share knowledge ...

NORMAL.DOTM is the core template file that Word uses to store many of the default settings used to create new documents.  WAAAAY back, in the 80's, before Windows, NORMAL.DOT held all of the setup info.  When Word was Windowized around in 1991, some of the settings were moved into the registry, and other other files.  Key settings that you described, style definitions (calibri to times roman) are still in Normal.DOTM  (The "M" is part of the relatively new "x-file' format that was created for Office 2007).

The QAT, Quick Access Toolbar, is the remaining ribbon element that users can completely customize.  By default it displays in the upper left corner of the Ribbon.  The customizations used to create the QAT are stored in settings files

So, it was my assumption that when MS said that settings were "portable" in Office 2013 and Skydrive, I assumed they meant that those files would be copied up to skydrive and used to carry your personal customization to other machines that you log in to.  From what you've said, that does not appear to be happening.

Right now I have no need for 365 so I'm using a local install of Office 2013, so I cannot check that those assumptions are right, or apparently wrong based on your observation.  Hopefully someone else with hands on experience  in 365 will be able to confirm my assumption or your observation of what is happening and point how to make my assumption correct (if possible).

jon2_3 say:

Old thread. In case others are wondering.

1. Styles exist in NORMAL.DOTX. Take this file from "C:\Users\USERNAME\AppData\Roaming\Microsoft\Templates" and send it to your home, and replace the one there (keep a backup).  Load Word and you should have all your styles from work.

You can also locate this file on a shared drive, etc. and then point Word to that location via as a previous user pointed out.  FILE->OPTIONS->ADVANCED-> look for the button marked "File locations" and in the next screen you can set the location of various files.

2. Ribbon customization and keyboard customizations can be exported from Word through FILE-> OPTIONS-> CUSTOMIZE, then use the IMPORT/EXPORT button to export these settings to a file. Send the file home. Follow the same instructions to IMPORT that file into the receiving computer.

Voila.
