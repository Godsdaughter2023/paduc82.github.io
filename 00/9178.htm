suzanne s. barnhill say:

Another approach, which maintains the text as editable:

  1. Select the text box.
  2. Right-click on the edge and choose Format Shape.
  3. In the dialog or task pane, select 3-D Rotation.
  4. Set X Rotation to 180°.
  5. For some reason this results in a gray fill in a previously white text box, so select Fill in the dialog or the paint bucket in the task pane and choose No Fill.

doug robbins - mvp office apps & services (word) say:
You will need to copy the text and then in another position in the document use Paste>Special to paste it as a picture.  When you do that and have the picture selected, the Picture Tools>Format tab of the ribbon will appear and from the Rotate dropdown in the Arrange section of the ribbon, select Flip Horizontal.
