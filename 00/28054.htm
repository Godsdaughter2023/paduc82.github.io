a. user say:

Hi Krizle,

Thank you for posting your concern in Microsoft Community. I understand that you want to set your Windows Live Mail to notify you first before downloading any large emails/attachments. Let me try to assist you regarding this matter.

In Windows Live Mail, you can create an email rule wherein an email with an attachment or a message with an attachment of a specific size will not be downloaded from the server. To do this, please follow the steps below:
  1. Open Windows Live Mail.
  2. Click the Folders tab on the upper part of the window and select Message rules.
  3. Under Select one or more conditions:, put a check beside Where the message size is more than the size and Where the message has an attachment.
  4. Under Select one or more actions:, select Do not Download it from the server.
  5. Under To edit this description, click the underlined words, click size and enter the size of the message or attachment.
  6. Enter a name for the email rule.
  7. Click Save rule.
Note: You can't use rules for IMAP or HTTP email accounts like Outlook.com.

Let us know if you need further assistance.
