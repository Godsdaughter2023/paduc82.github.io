Task Manager (and perfmon) report ~50% CPU but processes list says System Idle Process 96%

For the last week or so I've noticed (but it may have been happening longer) CPU utilisation being reported as approx 50% on both CPUs, continuously with no open applications. It's not a flat line, it bounces between 40% and 50%. However, the Processes list (with show all users) reports the System Idle Process as 95-99% so I can't find what's causing it.

Have also tried using performance monitor with the same results.

Q. Under what circumstances can this discrepancy be caused? Driver issue? Virus?

I run a virus scanner (AVG) and am all up to date with Windows updates and haven't made any significant changes to the system recently so at a loss to explain it. If it was simply being misreported then I could perhaps live with it but when running some applications I get the "Your computer is running slow" dialogue appear.

I've shut down just about every service and process I can, although if it were one of these then I guess Task Manager would be reporting the CPU usage against it anyway.

The system is due for a re-install but I'd like to understand what is causing this so I don't get a repeat.

Any insights gratefully received.
