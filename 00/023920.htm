gord dibben say:

What do you mean by stop every 5 lines?

Do you want to resize the window so you see just 5 rows at a time on the screen?

You could do something like that with some code.

Add this code to a module then run it to see if it looks like something you may use.

Sub slideshow()
    Dim i As Long
    Dim Lastrow As Integer
    Lastrow = ActiveSheet.Cells(Rows.Count, 1).End(xlUp).Row
    With Application
           .WindowState = xlNormal
           .Top = 200
           .Left = 400
           .Width = 800
            .Height = 215
     End With
    For i = 1 To Lastrow / 5
        Application.Wait (Now + TimeValue("0:00:5"))
        ActiveWindow.SmallScroll down:=5
    Next i
    Application.Goto Range("A1").EntireRow, True
    Range("A1").Select
End Sub
