teylyn say:

Hello,

you can use the text function with the letters d, m, and y for day, month and year, and still have the date displayed in a different language if you use the LCID identifier of the language. Consider the following screenshot:

The formula in all cells in column F is the same, starting in cell F1 and copied down:

=TEXT($A$1,"[$-"&D1&"]dddd mmmm, yyyy")

See how the LCID is dynamically pulled into the formula according to the value in column D. You could let your user select which language they want to see. Depending on the user choice, set a specific cell to the looked up LCID and reference that cell in all the Text functions of the spreadsheet. 

You can find a list of all LCIDs here: https://excel.tips.net/T003299_Specifying_a_Language_for_the_TEXT_Function.html

Does that work for you?

Alias is just an email address for receiving emails. It can’t be used to send emails.
