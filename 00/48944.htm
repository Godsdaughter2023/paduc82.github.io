a. user say:

how do I reset my laptop?

Depends on what kind of reset you want to do. There are several:

A factory reset restores your computer to the way it was when it left the factory - ergo the name. The procedure comes from the computer manufacturer.

A remove everything reset removes everything - ergo the name - and reinstalls a recent version of Windows only. The procedure comes from Microsoft.

A keep my files reset removes everything, reinstalls a recent version of Windows only and restores your personal files for you - ergo the name. That's also a Microsoft procedure.

A repair install reinstalls Windows over itself, to hopefully fix problems caused by damaged or missing files or messed up Windows settings (with which I am extensively familiar, unfortunately.)

A clean install blows everything away and lets you start over from scratch. This is by far the best way to install Windows, but it's a lot of work.

So ... what is your pleasure?

FYI: Reinstalling software doesn't fix problems with hardware. Just saying.
