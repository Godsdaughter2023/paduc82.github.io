bob jones aka: cybertaz say:

First of all, your version of Office 2916 is nearly a year behind on updates. Version 16.16 is the current build, so I'd suggest updating your software before chasing any symptoms :-)

However, updating will not change anything in this regard. If it truly is a Linked object it must have been created using a Windows version of PowerPoint. PPT for Mac does not support Linking to external files other than media file types. Pursuant to that, it provides no means of updating or revising Linked objects.

It sounds like it's actually an Embedded object, though. If that's the case, it & the supporting data can be edited. Also, a different data range in the same embedded workbook can be specified, but the chart cannot be "pointed" to a different Excel file.

One approach would be to copy the data from the new source & paste it into the embedded Excel file in place of the current data.

Another approach would be to paste the data into a different range of the embedded workbook then specify that range as the data source.

Also, of course, you could delete the existing Chart then create another one based on the new source file.

So lets say I start with this: 

Jack and Jill went up the hill 


And I want to turn it into this:
{Jack and Jill} went up the {hill} 

Is there any way to use word to actually do that? 

hansv mvp say:

You could do the following:

Step 1:

  • Press Ctrl+H to activate the Replace dialog.
  • Click More>> to expand the dialog.
  • Click in the 'Find what' box. Leave it blank, but click Format > Highlight.
  • Click in the 'Replace with' and enter  {^&}
  • ^& is the code for 'whichever text you found'.
  • Click 'Replace All'.
  • Close the Replace dialog.

Step 2:

  • Select the entire document (Ctrl+A)
  • Remove highlighting by selecting Highlight > No Color.

curious writer say: Thank you for those instructions, which imported the text just fine.  However, my Reference List now has TWO of several cited references because there were references that were duplicated in the two papers.  Do you have any advice on how to clean up (eliminate) the duplicates?

macropod say:

Had your two sets of references been created from the same bibliography entry in the 'Manage Sources' dialogue, you wouldn't have any apparent duplicates. Since Word adds any new  bibliography entry to the 'Master List' there, all you needed to do for the second one was to copy the entry from the 'Master List' into whatever document you wanted to add it to. The only reason you have apparent duplicates is because the bibliography entries differ in some way.

A clean-up requires:

• choosing the correct bibliography entry in Word's 'Master List' (assuming both are there), then deleting the other.

• re-inserting citation references apparent duplicates in the combined document, pointing them to the to correct bibliography entry, then deleting the corresponding citation reference to the to incorrect bibliography entry; and

• deleting the incorrect bibliography entry from the combined document.

Note: You cannot delete a bibliography entry from a document while there is a citation reference to it, which is indicated by a tick against the 'Current List' entry in the 'Manage Sources' dialogue
