What I have found works:

1) go to the bottom most corner of the document, with the scroll bar all the way at the bottom. Click and drag down and right for a few seconds, essentially creating even more space in your doc below and to the right.

2) start from where you ended that scroll (in the far right bottom corner) and do the opposite. Click and drag all the way to the left and up for double the time you dragged down.

3) press delete.

4) click and drag the scroll bar up a few times

Filter Excel sheet to show most recent record for an item - Microsoft Community

djjjblawson say:

Filter Excel sheet to show most recent record for an item

Hi, I'm trying to create a report of the most recent record for a customer.  Would appreciate a example spreadsheet of how this is done.

Thanks

lz. say:

Hi

Looks like a never ending story ;-) Query updated to dynamically take into account any other column(s) whatever there name is. Mandatory column names: [Id], [Group], [Transaction], [Date]. And before you ask ;-) the ouput is sorted by [Group] then by [Date]

Query code:

    Source = Excel.CurrentWorkbook(){[Name="Table1"]}[Content],
    TypeChanged = Table.TransformColumnTypes(Source,{
        {"Id", type text}, {"Group", type text}, {"Transaction", type text}, {"Date", type date}}
    ),
    GroupedRows = Table.Group(TypeChanged, {"Id", "Group"},
        {{"GroupedId", each
            let
                SortedDate = Table.Sort(_, {{"Date", Order.Descending}}),
                FirstRecordSelected = Table.FirstN(SortedDate, 1)
            in
                FirstRecordSelected,
                type table
        }}
    ),
    ColumnsToExpand = List.RemoveMatchingItems(Table.ColumnNames(GroupedRows[GroupedId]{0}), {"Id","Group"}),
    ExpandedGroupedId = Table.ExpandTableColumn(GroupedRows, "GroupedId", ColumnsToExpand),
    TypeChanged1 = Table.TransformColumnTypes(ExpandedGroupedId,{{"Date", type date}}),
    SortedRows = Table.Sort(TypeChanged1,{{"Group", Order.Ascending}, {"Date", Order.Ascending}})
in
    SortedRows

lz. say:

In this workbook you have both options. To ensure you get the same records from the Formula and the Query I revised the later

Instead of:

            let
                SortedDate = Table.Sort(_, {{"Date", Order.Descending}}),
                FirstRecordSelected = Table.FirstN(SortedDate, 1)
            in
                FirstRecordSelected,
                type table
        }}
    ),

it's now:

            let
                SortedDate = Table.Sort(_, {{"Date", Order.Ascending}}),
                LastRecordSelected = Table.LastN(SortedDate, 1)
            in
                LastRecordSelected,
                type table
        }}
    ),

Hope this makes sense
