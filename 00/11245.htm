Hello Denton,

Thank you for posting your query in Microsoft Community.

Do you get any error message when you try to print?

This issue may occur due to outdated or corrupted printer drivers, if you do not have printer permissions assigned, if Protected Mode is enabled for the Internet security zone of the web page or if there is an issue with temporary folders on your user account and the Low folder under the %TEMP% location has been deleted.
So, I would suggest you to try the following methods and check if the issue persists:


Method 1:

Try printing a webpage running Internet Explorer as an Administrator.

To start Internet Explorer as an Administrator, follow these steps:

a) Close all Internet Explorer windows.

b) Click Start, click All Programs, right-click Internet Explorer and select Run As Administrator.

c) Browse to a website and try test printing a page while running as the Administrator.

If the issue persists, try next method.

Method 2:

Try printing after disabling Protected Mode.

Try printing a webpage after disabling Protected Mode, to do this, follow these steps:

a) Click the Tools icon in Internet Explorer.

b) Click the Security tab and uncheck the checkbox beside Enable Protected Mode (requires restarting Internet Explorer).

c) Click Apply, and then click OK.

d) Close all open Internet Explorer windows, and then restart Internet Explorer.
e) Browse to a website and try test printing a page while running as the Administrator.

If the issue persists, try next method.

Method 3:

Set IE as default browser. To do so, try the following steps:

a. Click Start, type Default Programs, and hit ENTER.
b. Click Default Programs in the search results.
c. Click Set your Default Programs.
d. Choose Internet Explorer from the list.
e. Select Set this program as default, and then click OK.

You may refer to the link for additional information.

Fix printer problems
http://windows.microsoft.com/en-us/windows/printer-problems-in-windows-help#fix-printer-problems=windows-7&v1h=win8tab1&v2h=win7tab1&v3h=winvistatab1&v4h=winxptab1

 

Method 4:

Re-create the missing folder

1. Close all the instances of Internet Explorer.
2. Click Start, type %Temp% in the Start Search box, and then click the Temp folder in the list.
3. If you are running Windows Vista, click New Folder on the Organize menu.

If you are running Windows 7, click New Folder on the main menu.
4. Type Low for the folder name, and then press ENTER.
5. Close the Windows Explorer window, start Internet Explorer, and then print or print preview a Web page.

 

Method 5:

Reset low integrity level on the Low folder

1. Click Start, click All Programs, and then click Accessories.
2. Right-click Command Prompt, and then click Run as administrator.
3. Type following command and press ENTER:
ICACLS "%userprofile%\AppData\Local\Temp\Low" /setintegritylevel (OI)(CI)low

4. Restart the computer.
5. Print or print preview a Web page in Internet Explorer.

 


Hope this information helps. Please post back for further assistance.

jay freedman say:

The behavior you describe is what happens when the document is in Word 97-2003 (*.doc) format. You should also see a menu bar at the top of the window instead of the ribbon, and you'll find the chart type and chart options commands on the Chart menu.

To get the native Word 2013 behavior where the dialog appears first, you need to update the document by clicking File and then clicking the Convert button, and save as Word Document (*.docx) format.
