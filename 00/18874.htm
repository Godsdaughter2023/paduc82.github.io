Hi Craig,

Thank you for your interest in Windows 10.

I suggest you to follow the methods below and check if it fixes the issue.


Method 1: Run Hardware and Devices Troubleshooter

Follow these steps to run Hardware and Devices troubleshooter:

  1. Press Windows key + X and click on Control Panel.

  2. Select Troubleshooting.

  3. Select View all on the top left corner.

  4. Click Hardware and Devices.

  5. Follow the on-screen instruction to run the troubleshooter.

Method 2: Uninstall and reinstall the Sound driver

I suggest you to try uninstall the Sound drivers check if it works for you.

Follow the steps to uninstall the Sound drivers:

  1. Press Windows Key + X key and click Device Manager.

  2. Expand ‘Sound, Video and Game Controllers.

  3. Right-click on the Audio devices listed and click on ‘Uninstall’.

  4. Select the option to Delete driver software.

  5. Restart the computer after its uninstalled.

I need to archive all emails that are older than 6 months

tobytu_msft say:

hi rory,

thanks for clarifying your situation.

if you want to move all emails which are older than 6 months to the archive mailbox, i suggest you use the retention policy because the sweep rule applies to single emails. please refer to the following steps:

firstly, please let the administrator create a related retention tag for the retention policy which applies to you.

tips: the end user is not able to do this. if you are not an administrator, please contact him. 

there are three types of retention tags. if the administrator creates a “default” retention tag, as this retention tag applies to an entire mailbox, the end user doesn’t need to do anything.

however, it will also affect other users in your organization who are using the same retention policy as yours.

if you just want it to be used for your mailbox, i recommend you let the administrator create a “personal” tag.

then you can follow the steps below to add this retention tag to your mailbox. sign in to owa > options > retention policies > add > select the related tag > add > save

finally, we can apply this retention policy to the specific folder.

right click a folder > assign policy

please let me know if you run into any errors.

best regards,

toby
