How can I make or obtain a list of these hand combinations in an excel file? I'm working on a promotion and I need to get the full list of possible combinations so I can randomly issue them on a printed ticket.


Try the VBA procedure below.

To enter the VBA code, select the worksheet tab, right-click and click on View Code.  That will open the VBA window.  Copy the VBA code below (ctrl+C) and paste (ctrl+V) into the VBA window.  You can now close the VBA window.

To execute the VBA code from Excel, press alt+F8, select the VBA procedure name and click on Run.

-----

Option Explicit

Sub gen3CardHands()
Dim i As Long, j As Long, k As Long
Dim nd As Long, nc As Long
Dim face(1 To 13) As String
Dim deck(1 To 52) As String
Dim hand(1 To 22100, 1 To 3) As String
Dim suit, card

Application.ScreenUpdating = False

' generate 52-card deck
For i = 2 To 10: face(i) = i: Next
face(1) = "A"
face(11) = "J"
face(12) = "Q"
face(13) = "K"
nd = 0
For Each suit In Array("S", "H", "D", "C")
    For Each card In face
        nd = nd + 1
        deck(nd) = card & suit
    Next
Next

' generate 3-card hands
nc = 0
For i = 1 To nd - 2: For j = i + 1 To nd - 1: For k = j + 1 To nd
    nc = nc + 1
    hand(nc, 1) = deck(i)
    hand(nc, 2) = deck(j)
    hand(nc, 3) = deck(k)
Next: Next: Next

' write hands to column A and sort randomly
Range("a1").Resize(nc, 3) = hand
Range("d1").Resize(nc).Formula = "=rand()"
With Range("a1").Worksheet.Sort
    .SortFields.Clear
    .SortFields.Add Key:=Range("d1").Resize(nc), _
        SortOn:=xlSortOnValues, Order:=xlAscending, _
        DataOption:=xlSortNormal
    .SetRange Range("a1:d1").Resize(nc)
    .Header = xlNo
    .MatchCase = False
    .Orientation = xlTopToBottom
    .SortMethod = xlPinYin
    .Apply
End With
Columns("d:d").Clear

Application.ScreenUpdating = True
End Sub

chip pearson say:

By default, all cells are locked, but the lock does not take effect until you protect the worksheet. If you want to use VBA to change a locked sheet, protect the sheet using the UserInterfaceOnly option. For example, in your workbook open procedure, use code like

 

Private Sub Workbook_Open()
 Worksheets(1).Protect UserInterfaceOnly:=True
 With Range("CH9")
     If .Value = "" Then
         .Value = Evaluate("randbetween(0, 9999999999)")
         .NumberFormat = String(10, "0")
     End If
 End With
 End Sub

When UserInterfaceOnly is used, the workbook is protected from modificaiton by the user (keyboard and mouse, basically), but VBA is free to modify any cell, regardless of the protection mode. The UserInterfaceOnly flag is not preserved when the workbook is closed and reopened, so you should put the code in Workbook_Open so it executes whenever the workbook is opened.  There is no menu item or button to set the UserInterfaceOnly flag -- it must be set via VBA.

Example

gord dibben say:

You have spaces in some of the "blank" cells you are concatenating.

i.e.    =LEN(S3)   returns  1  even though S3 looks blank.

If you TRIM those feeder cells you will eliminate the spaces.

Run this macro on columns S, T, V, W

Sub TrimALL()
   'David McRitchie 2000-07-03 mod 2000-08-16 join.htm
   Application.ScreenUpdating = False
   Application.Calculation = xlCalculationManual
   Dim Cell As Range
   'Also Treat CHR 0160, as a space (CHR 032)
   Selection.Replace What:=Chr(160), replacement:=Chr(32), _
     Lookat:=xlPart, SearchOrder:=xlByRows, MatchCase:=False
   'Trim in Excel removes extra internal spaces, VBA does not
   On Error Resume Next   'in case no text cells in selection
   For Each Cell In Intersect(Selection, _
      Selection.SpecialCells(xlConstants, xlTextValues))
     Cell.Value = Application.Trim(Cell.Value)
   Next Cell
   On Error GoTo 0
   Application.Calculation = xlCalculationAutomatic
   Application.ScreenUpdating = True
End Sub
