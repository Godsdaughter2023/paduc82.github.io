cédricmi say:

You need to use bcdedit

 

First on the Metro interface select show all applications the select the command prompt with a right click and on the menu bar select run as administrator.

 

Then type bcdedit

 

Then look carefully at the list and copy the identifier of the entry you want to delete.

Then type bcdedit /delete {boot loader identifier}

where {boot loader identifier} is the identifier of the entry you want to delete

 

NB: to copy in the command prompt right click, mark, then select the text as you would Under a gui. to paste right click paste

gokul t say:

Hi daniellenapier,

 

When you import pictures from a digital camera, it imports all the files and the files will be stored in the My Pictures folder in Windows.

 

Refer the section To import media from a digital camera or other removable device in the following article:

 

Add media files to Windows Media Center

http://windows.microsoft.com/en-us/windows7/Add-media-files-to-Windows-Media-Center

 

As a work around, if you want to add only selected pictures, you may add them manually to My Pictures folder to see them in Media Center.

 

Also refer the following articles:

 

How do I get pictures from my camera to my computer?

http://windows.microsoft.com/en-US/windows7/How-do-I-get-pictures-from-my-camera-to-my-computer

 

Change settings for importing pictures and videos

http://windows.microsoft.com/en-us/windows7/Change-settings-for-importing-pictures-and-videos

 

Hope this information helps.
