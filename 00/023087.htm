Excel auto roundoff long numbers in the .csv file

I have a .csv file which contains many numeric values of 19 digits. Whenever I open the file in excel, the 19 digit numbers are already rounded off to 15 digits....

I want to avoid this automatic rounding off of numbers, what should I do?

zorvek say:

Excel automatically interprets all values read from a text file as a number, a date, or as text if not a number or date. This occurs when opening a text file with Excel such as when using the File->Open menu command or instructing File Explorer to open the text file with Excel. To prevent Excel from interpreting the imported values, create a new empty worksheet, select all of the cells in the worksheet and set their formatting to text (choose Format->Cells, navigate to the Number tab, and choose Text,) select cell A1, and then import the data using the menu command Data->Import External Data->Import Data.

This will result in a data query being defined for that worksheet. While the data query does have some advantages such as a refresh option, it may not be required and might get in the way of downstream activity. To remove the query (but retain the data just imported,) right-click on any imported call and select "Data Range Properties", uncheck "Save query definition", click OK on the warning prompt, click OK on the "External Data Range Properties" dialog.

To only prevent specific columns from being interpreted, choose only those columns when assigning the Text formatting option.

Kevin
