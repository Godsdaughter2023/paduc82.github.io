2) Look for a folder called $WINDOWS.~BT

3) Rt click on folder and rename to OLD_$WINDOWS.~BT (it won't delete without admin perms, so you'll need to clean it later.

4) Open Task Manager (rt click on Windows Start Bar, select Task manager)

5) Find and highlight "Windows 8 Setup (2)" or "Windows 8 Setup"

6) Choose End task

Result:

Trying running setup again, and see if it works now.   If yes, when you are all done, remember to go back to your C: drive and delete the OLD_$WINDOWS.~BT directory
