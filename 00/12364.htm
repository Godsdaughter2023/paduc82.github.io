Hi Kat,

 

Since the main issue is about the upload failed, we'll focus on this error message first.

 

Please try clearing the upload center office document cache.

 

1. Perform a clean boot: This step is necessary to make sure the cache files don't get loaded during startup. If they did, they would be locked and you couldn't delete them.

 

2. Delete cache files: Often, this step fixes the problem. Office typically repairs the document cache files if it detects problems. If the cache files are corrupt and irreparable, deleting them allows Office to repair itself by replacing the corrupt files.

 

3. Reboot and sync: This step is when Office repairs the cache. After the cache is repaired, a sync operation should clear up any "stuck" files.

 

For more information, you can refer to this article: Delete your Office Document Cache.

ahzanti c say:

Realtech audio driver issue

When I updated to 10041 on the evening of the 18th, I had no sound like many other insiders where having.  I was able to resolve my issue because my update was timing out of the driver install, [Error code:0x000005B4 it ended with that].  The answer I found was on an old windows page.  "Press windows key and the R key at the same time.  Type in the box 'gpedit.msc' and push enter.  Then under Local Computer Policy >Administrative Templates >System >Device Installation double click on the settings window 'Configure device installation time-out'.  A new window will popup set it to Enabled and in the Device Installation Timeout (in seconds) window type 1800 and click apply.  Don't close that window yet, try your update again first.  For me that caused it to install.  If yours installs fine with that then set the 'Configure device installation time-out' window click Not Configured.  and click ok.  Then close all the windows and Restart if nessary.

Hope this helps someone.
