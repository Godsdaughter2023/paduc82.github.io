The Drive is also visible in Windows Explorer with "Properties" showing 46MB free space.  This is within the range for generating the nuisance alarm for this size "Disk".

I have tried to clean up the drive and reduce the number of restore points without success.

Disk 0 partition 2 - EFI System Partition - 99 MB NTFS

C: - Primary Partition - 238 GB NTFS

N: - OEM Partition - 450 MB NTFS

How can I get rid of this drive or the nuisance alarm without going into the registry and disabling the "out of disk space" warning all-together?

davem121 say:
Hi Clive, this is a known bug in the 1803 Update,

It is the recovery drive, you are not meant to see it, it is not meant to have a drive letter, remove the drive letter and everything will return to normal:

Click your Start Button, type cmd, then right click Command Prompt and choose 'Run as Administrator'

Run this command and hit Enter

diskpart

Run this command and hit Enter

list volume

Note down the letter associated with that new drive

Run these commands one at a time and hit Enter (replace X with the correct drive letter)

select volume X

remove letter=X

Close Command Prompt and restart your PC

sumit (volunteer moderator) say:
This is one of the known issues with the update as Recovery partition should be hidden and is accidentally assigned a drive letter.
Right-click Start>Command Prompt(admin)
Type in the following:
diskpart
list volume
select volume 3
Replace 3 with the number of the volume which is assigned a drive letter. (In hundreds of MBs)
remove letter="G"
Replace G with the Recovery drive letter.

Don't worry, removing a drive letter would not cause any harm to your device.
