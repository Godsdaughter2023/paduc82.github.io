"Sticks and stones..." and all that. Words have no power unless you give it to them.

Unfortunately with email systems designed as they are today, unless you control the mail server, there's no way to block the delivery of junk. Try the following  UDF (User Defined Function):

  • Alt+F11 to open the VBA editor
  • Alt+IM to insert a new code module
  • In the new module, paste the following code 

'=========>>
Option Explicit

'--------->>
Public Function CheckedCount() As Long
    Dim cBox As CheckBox
    Dim i As Long, iCtr As Long
    

    Application.Volatile

    For Each cBox In ActiveSheet.CheckBoxes
        If cBox.Value = 1 Then
            iCtr = iCtr + 1
        End If
    Next cBox
    CheckedCount = iCtr
End Function
'<<=========

  • Alt+Q to close the VBA editor and return to Excel
  • Save the file as a macro-enabled workbook with an xlsm extension

Now the UDF may be used on your worksheet like a native Excel function:

Note that the value returned by the function will not change when new checkboxes are checked or unchecked; the function will update at each calculation operation.
