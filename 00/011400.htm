go to a record on subform

regard

marshall barton say:

A couple of ways of doing that.  The simplest way is to set the subform CONTROL's LinkMaster property to Txt1 and the LinkChild property to the name of the number field in the subform's record source table/query (not the control in the subform).

 

If that's too automatic for your needs, then you can use VBA code in Txt1's AfterUpdate (or a command button's Click) event:

 

With Me.sub1.Form.RecordsetClone

   .FindFirst "numberfield=" & Me.Txt1

   If .NoMatch Then
      Beep
      Exit Sub
   End If
   Me.sub1.Form.Bookmark = .Bookmark

End With

 

and, if you want to focus to move to the found record in the subform, add this line after the above:

 

Me. sub1.Setfocus
