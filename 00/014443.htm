If you’re still running into issues after installing the Insider Fast update per my previous instructions, please complete the following steps and let us know what you experience:

  1. Close Word.
  2. Click the magnifying glass icon in the top right corner of your screen, type Terminal, and press Return.
  3. In the Terminal window that opens, type the following and press Return: defaults write com.microsoft.Word ForceTempFilesInAppContainer 1
  4. To ensure that this setting worked,
    1. Open a Finder window, and press Command+Shift+G.
    2. Type the following path:

      ~/Library/Containers/com.microsoft.Word/Data/Library/Application Support/Microsoft/Temp

    3. Press Return. Upon launching Word, this folder will be created if it doesn't exist, and Word will create a temp file inside it. Quitting Word will then cause this temp file to removed, which tells us that the new setting has taken effect.

talking moose say:

The message you’re receiving is not coming from Outlook. It’s coming from your mail server and Outlook is simply displaying it to you.

First, keep in mind that computers and phones all count as connections and if you’re using IMAP then connections are probably happening without you realizing it.

In Outlook choose Outlook menu >  Preferences > Accounts > your IMAP account > Advanced.

  • UNcheck the option to use IMAP IDLE.
  • Set the sync for all IMAP folders to 10 minutes or higher.
