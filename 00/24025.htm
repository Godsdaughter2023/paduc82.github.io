rick rothstein, mvp - excel say:

If you can make use of VB code, here are two macros that will do what you have asked for (they import the data directly into the cells... nothing else is needed)...

Sub SingleColumnImport_PreserveRows()
  Dim FileNum As Long, TotalFile As String, PathAndFileName, Data() As String
  PathAndFileName = "c:\temp\data.txt"
  FileNum = FreeFile
  Open PathAndFileName For Binary As #FileNum
    TotalFile = Space(LOF(FileNum))
    Get #FileNum, , TotalFile
  Close #FileNum
  Data = Split(TotalFile, vbNewLine)
  Range("A1").Resize(UBound(Data) + 1) = WorksheetFunction.Transpose(Data)
End Sub

Sub SingleColumnImport_ExtendRows()
  Dim FileNum As Long, TotalFile As String, PathAndFileName, Data() As String
  PathAndFileName = "c:\temp\data.txt"
  FileNum = FreeFile
  Open PathAndFileName For Binary As #FileNum
    TotalFile = Space(LOF(FileNum))
    Get #FileNum, , TotalFile
  Close #FileNum
  TotalFile = Replace(TotalFile, ",", vbNewLine)
  Data = Split(TotalFile, vbNewLine)
  Range("A1").Resize(UBound(Data) + 1) = WorksheetFunction.Transpose(Data)
End Sub

The first imports the file preserving the existing row count yielding 100 rows where each cell contains the comma delimited list for that row. The second imports the file where each value is placed in its own cell. You would run this macro from a blank worksheet as it will always place the first item in cell A1, the second item in A2 and so on down the column. Oh, and don't forget to change my example path and file name (c:\temp\data.txt) assigned to the PathAndFileName variable at the beginning of each code procedure to your actual path to the file you want to process.

If you are new to macros, they are easy to install and use. To install it, simply press ALT+F11 to go into the VB editor and, once there, click Insert/Module on its menu bar, then copy/paste the above code into the code window that just opened up. That's it.... you are done. To use the macro, go back to the worksheet with your data on it and press ALT+F8, select the macro name from the list that appears and click the Run button. The macro will execute and perform the action(s) you asked for.
