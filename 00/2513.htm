I'm a bit confused as you supply customeremail  as an input variable to the EmailPDF function, yet you the proceed to define it in the function itself.  Could you explain why?

customeremail = customeremail & rs!dsaemailaddress & ";"

is the proper approach, in the sense that it will loop through the recordset and concatenate the e-mail addresses together so you can send out your e-mail to multiple e-mail addresses in one shot.  Now if it is including too many addresses this means that the qrysecurityauditcontacts query is not returning a properly filtered listing.  You'd need to look into restricting the query further so it only returns those addresses you are after.
