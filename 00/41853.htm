"I'm not too sure how to update drivers either. Apart from the common update driver disks that are popups on different sites. "

It is essential to install all new drivers for the graphcis  and motherboard 'chipset' with a clean
Windows install.
The various driver finder (along with registry 'cleaners' and system optimising) type programs are
not at all necessary.
- Windows installs a number of drivers, but on a clean install you should not have to worry about these.
If one of these becomes corrupt the easiest way to fix it is normally via Device Manager.
It's just a matter of finding the device in question and un-installing the driver for that device.
When you re-boot the machine Windows will reinstall the driver.
- You can also update these drivers via Device Manager, but that is not often necessary.

- However,  it is best to update graphics and the  'chipset'
(various drivers that run functions on the motherboard, such as LAN and audio) drivers
by downloading them  from one of two places;

- If using a laptop or OEM PC (DELL, HP, etc.) it is generally best to download drivers for the model
of your machine from the mfg's web site (older machines will sometimes not have recent driver updates
in which case you can try downloading from elsewhere, but with no guarantee of stability).

- When using a 'custom' built machine ( machines assembled by the user or small shop) it is best to
get the motherboard drivers from the motherboard mfg's web site.
Dedicated graphics cards are the same. Use either the site for the cards mfg, or the OEM site.
- Intel and AMD Integrated graphics  which are part of the central processor are not so critical
and can be updated via Windows Updates.
Do not allow Windows Updates to install drivers for dedicated graphics cards or Integrated graphics
in the form of a chip on the motherboard.

- After downloading the drivers uninstall the current drivers via Programs and Features, and install
the new drivers after re-booting the machine.
A lot of hardware also has it's own driver updater software included in the driver installer, but I still
prefer manual install. I  don't trust the automated versions just yet [ control freak ;)  ].
