Another better method would be to delete everything on the 32GB SSD, select the empty space, and start over completely fresh. This is what a lot of people recommend...

a. user say:

Hi,

Thank you for posting your query on Microsoft Community.

I appreciate you for providing details about the issue and we will assist you resolve this issue.

I suggest you to try the following steps and check if it works for you:

Step 1:

You may have to create a new user account with administrator type and delete the old user account. After creating a new user account with administrator type. Check if the present account is a Microsoft account or a local account. If it’s a Microsoft account then convert it to local account.

Note: You can only delete another account provided your account has administrator privileges.
All apps you installed on the old account will also be deleted, but you can decide to keep the personal files.

To convert Microsoft account to local account follow the below steps:

  1. From the Desktop, Press Windows Key + R.
  2. Type netplwiz and click on OK.
  3. Select the Microsoft account and click on Properties button.
  4. Click on Group Membership tab and change the option to Standard User.
  5. Click on Apply and OK.

After performing above step follow the below step.

Step 2:

Try the below steps to delete user account after converting from Microsoft account to local account.

  1. Login to your default account.
  2. Click on the Windows logo and type User Accounts.
  3. Now, select Settings and then select the User Accounts icon.
  4. Click on Manage another account and then select the user account that you want to delete.
  5. Click on delete the account.
  6. Depending on whether you want to Keep the files or delete the files choose the appropriate option.
  7. You can delete the other accounts similarly.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.

Thank you.
