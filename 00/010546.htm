This happens because some named ranges yield errors.  This in turn arises because some worksheets have been simply "Moved or copied" from other workbooks which in turn had named ranges.  Try this

2. Open Excel and then open the workbook

3. Press Ctrl+F3 and filter the named ranges on errors

4. Delete all erroneous named ranges

5. Click on OK

6. Save the workbook

Now when copying worksheets, you should not face that error.

Hope this helps.
