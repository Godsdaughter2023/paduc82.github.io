john korchok say:

Word for Windows has 3 different form controls that can be inserted:

  1. Legacy Form Fields. These can be created on both the Mac and Windows versions and work equally well on both OSs.
  2. ActiveX Controls. These are Windows only and don't work on a Mac at all.
  3. Content Controls. These work in Word 2016 and 2019 for Mac, as well as the Windows versions, but can only be created on a Mac by using a VBA macro.

Most likely the form designer used an ActiveX control for the dropdown. Those just aren't going to work on your computer.
