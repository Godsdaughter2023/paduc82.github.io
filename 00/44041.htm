a. user say:

If you are seeing an error message, what does it say exactly and when do you see it, and as usual we don't know anything about your system so that leads to guessing.

Here is a pretty popular example of a Windows Installer error message that might be what you might see  when trying to install a program or clicking the "Change" or "Remove" button in Add or Remove Programs on an already installed application:

Windows Installer may be corrupted or your computer is in safe mode

or perhaps this message:

The Windows Installer Service could not be accessed.
This can occur if you are running Windows in safe mode, or if the Windows Installer is not correctly installed. Contact your support personnel for assistance.

Instead of contacting your support personnel for assistance,  do this:

Click Start, Run and in the box enter:

services.msc

Click OK to open the Services applet.

Look in the Services applet to see if the Windows Installer Service is even listed and you will find that it is probably not listed (but it should be listed).

If the Windows Installer Service is listed, make sure it is not Disabled and if it is Disabled, right click the Windows Installer Service, Properties and change the Startup type to Manual.  You don't have to set it to Automatic as it will start on it's own when it needs to start and you don't need it starting every time you restart your system and running all day when there is nothing for it to do.

The Service will start when it needs to start if it is able to start.  Click OK enough times to save your adjustments and close the Services applet, restart your system to be sure the changes "stick" and see how things look now.

If the Windows Installer Service is completely missing from Services, you need to add it as follows:

Click Start, Run and in the box enter these commands one at a time clicking the OK button for each:

msiexec   /unregister    (or you can enter:   msiexec   /unreg)
msiexec   /regserver

Click OK after each command   (there will be no message about what happened)

Check in the Services applet again to see if the Windows Installer Service is now listed and the Startup type is set to Manual.

Now verify the version is your Windows Installer:

To find what this is, click Start, Run and in the box enter:

msiexec

Click OK and the MS Installer usage and help message will appear.

If you have the current version installed, you should see a message like this in the top line:

Windows ® Installer. V 4.5.6001.22159

If that is not the version number you see, tell us what you do see for the version number.

If your installer is NOT version 4.5, you need to download and install version 4.5 it from here:

http://support.microsoft.com/kb/942288/en-us

After installing version 4.5, reboot your system to be sure the changes "stick".

Then check your Services again to be sure the Windows Installer Service is listed, and check your installer version again to be sure it is version 4.5.

Then test your original failure condition and report your results

samhrutha g say:

Hi James Vester,

 

Install Microsoft Flight Simulator by using a flat folder

 

To install Microsoft Flight Simulator by using a flat folder, follow these steps:

i.           Boot the computer in clean boot. Below link will help you in booting the computer in clean boot: http://support.microsoft.com/kb/929135

ii.         Save a copy of the Microsoft Flight Simulator to your computer's hard disk. To do this, follow these steps:

iii.       Start Microsoft Windows Explorer. To do this, right-click on Start, and then click Explore.

iv.        Create a new folder at the root of a hard disk volume, and then name it "Flatfile" (for example, C:\Flatfile).

v.          Locate the drive that contains the Microsoft Flight Simulator CD-ROM, and then copy all the contents of the Microsoft Flight Simulator CD-ROM into the C:\Flatfile folder. To do this, follow these steps:

i.                    In Windows Explorer, click the CD-ROM icon (or DVD-ROM icon) that contains the Microsoft Flight Simulator CD-ROM.

ii.                  On the Edit menu, click Select All.

iii.                Paste all the files into the new folder. To do this, drag the selected items on the right side of the Explorer window into the C:\Flatfile folder on the left.

 

Note If you receive the "This Folder contains "X" hidden files" error message, follow these steps:

i.           On the Tools menu, click Folder Options.

ii.         Click the View tab.

iii.       Turn on Show Hidden Files and Folders.

iv.        Turn off Hide Protected Operating System Files.

v.          Click Apply, and then click Ok.

 

Note: If there are errors during the copy process, this can indicate problems with the CD-ROM (media), a dirty CD-ROM drive, or a problem with your hard disk.

 

Restart your computer the way that you typically do, and then install Microsoft Flight Simulator from the C:\Flatfile folder onto your hard disk.

 

Make sure you reset the computer to normal mode (step 7 from the above link).
