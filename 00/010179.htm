e.j. gun say:

If iConCtr represents the total number of columns in the range you want to define, and iConCtr*3 represents the total number of rows, and the first cell in the range is "B3", then:

  Set rControl = ActiveSheet.Range(ActiveSheet.Cells(3, 2), ActiveSheet.Cells(2 + iConCtr * 3, 1 + iConCtr))

should work.

If iConCtr represents the actual column number (not the number of columns) and iConCtr*3 represents the actual row number, then use:

    Set rControl = ActiveSheet.Range(ActiveSheet.Cells(3, 2), ActiveSheet.Cells(iConCtr * 3, iConCtr))

jay freedman say:

The Align button in the Arrange group on the far right of the Layout ribbon is not for text; it's for floating objects such as pictures and text boxes.

To align text in any "ribbon" version of Word, click the Layout tab and then click the dialog launcher in the Page Setup group (the little button indicated in the picture below).* In the Page Setup dialog, you'll find the vertical alignment in the same place you're used to, on the Layout tab of the dialog.

*You can also open the Page Setup dialog by double-clicking in the top half of the horizontal ruler, but if the cursor is just a hair too low it inserts a tab stop instead. The launcher button is more reliable.
