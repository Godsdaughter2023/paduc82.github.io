ken sheridan say: One method for this requires the string expression to be a Boolean expression, which is assigned to the WhereCondition argument of the OpenReport method.  The expression applies the IN operator to the value list.  Assuming that the column on which the report is to be filtered is LotNumber, and this is a number data type:

     strWhere = "LotNumber IN(" & Left(strWhere, Len(strWhere) - 1) & ")"

    DoCmd.OpenReport "LoadPickReportBYlot", _
        View:=acViewPreview, _
        WhereCondition:=strWhere

    DoCmd.Close acForm, "LOADLotPickReportBYlot"

If the LotNumber column is of text data type, you need to wrap each value in literal quotes when building the expression:

    strWhere = strWhere & """" & ctl.ItemData(varItem) & ""","

You'll find an examples of the use of controls in a dialogue form for restricting a report's output, including the use of a multi-select list box, in DatabaseBasics.zip in my public databases folder at:

https://onedrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

Note that if you are using an earlier version of Access you might find that the colour of some form objects such as buttons shows incorrectly and you will need to  amend the form design accordingly.  

If you have difficulty opening the link, copy the link (NB, not the link location) and paste it into your browser's address bar.

In this little demo file, the section on 'retrieving data from the database' includes the method which I think you are attempting, which is to reference the value list in a text box (hidden in my case) in the dialogue form as a parameter.  However, the IN operator cannot reference a value list as a parameter in a query, but Microsoft has published InParam and GetToken functions which simulate this.  My demo uses these functions.  The code to assign the value list to the text box and open the report is:

    Dim varItem As Variant
    Dim strContactIDList As String
    Dim ctrl As Control
    
    Set ctrl = Me.lstContacts
    
    If ctrl.ItemsSelected.Count > 0 Then
        ' if at least one item has ben selected in the list box
        ' loop through the control's ItemsSelected collection
        ' and build a value list of ContactID values
        For Each varItem In ctrl.ItemsSelected
            strContactIDList = strContactIDList & "," & ctrl.ItemData(varItem)
        Next varItem
        
        ' remove leading comma
        strContactIDList = Mid(strContactIDList, 2)
        ' assign value list to hidden text box control
        Me.txtContactIDList = strContactIDList
        
        DoCmd.OpenReport "rptContacts_1", View:=acViewPreview
    Else
        MsgBox "No contacts selected", vbInformation, "Warning"
    End If

Note that the value list in this case does not need to wrap text values in literal quotes characters.

In my demo the functions are in the basInParam module, and the report shows in its header how the InParam function is called.  The GetToken function is called by the InParam function.
