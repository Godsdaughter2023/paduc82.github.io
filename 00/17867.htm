3. Compact again

You might also try a Decompile. To do that, create a Shortcut with a Target like this:

"full path to msaccess.exe" /decompile

Run that shortcut, then select your database and Access will decompile it. Access won't alert you when it finishes, but give it a few moments and the decompile should be finished.

After decompiling, run the 3 steps above again.

You might also consider creating a new, blank database and importing everything into that new database. You can use the External Data - Import & Link - Access ribbon item to do so.
