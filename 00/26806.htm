jackpot. say:

Try the below

1. Select the data range in ColA (say A1:A100).
2. Press F5. From 'Goto window'>Special> from Options select 'Blanks' and hit OK
3. This will select all blanks.
4. Now press = (equal sign)
5. Then press Up Arrow to reference the cell just above
6. Now press Ctrl and Enter key together

OR

1. Select an unused cell out of the data range (say E2)
2. Enter the formula in cell E2 which references the top cell (=E1)
3. Copy the cell (not the formula)
4. Keeping the copy; select the data range in ColA (say A1:A100).
5. Press F5. From Goto window>Special> from options select 'Blanks'
6. This will select all blanks. Now Right click>PasteSpecial>Formulas>OK will
fill in with the data you need.
7. Once done you can select col A > Copy >Paste Special>Values to turn
formulas to actual values

OR

If you want this as amacro; try the below. If you are new to macros..

--Set the Security level to low/medium in (Tools|Macro|Security).
--From workbook launch VBE using short-key Alt+F11.
--From menu 'Insert' a module and paste the below code.
--Get back to Workbook.
--Run macro from Tools|Macro|Run <selected macro()>

Sub MyMacro()
Dim lngRow As Long
For lngRow = 2 To Cells(Rows.Count, "B").End(xlUp).Row
If Trim(Range("A" & lngRow)) = "" And Trim(Range("A" & lngRow + 1)) = "" Then _
Range("A" & lngRow) = Range("A" & lngRow - 1)
Next
End Sub

sunil chopra (sunny) say:

I had the same where one of my PST which was not my current default PST had "Inbox", "Sent", "Drafts" folders etc. and I wanted to get rid of it but option to delete was greyed out.So after lot of searching and hit and trial methods this is THE ONLY FIX iI found.

Problem : "Inbox", "Sent", "Drafts" folders in non-default PST and not able to get rid of those

Cause : At any point if that PST has been made default then Outlook will make these folders which cannot be removed.

Resolution : 1. In Outlook Goto "New Items" -> More Items-> and select "Outlook Data file". Select a place to save the new outlook data file (PST) you want to save. (Don't select desktop. You may select My Documents folder). 

2. Now start moving all the folders of old PST to this new created PST and do not move Inbox,Sent,Drafts folders etc.

3. Once you are done moving all the folders (except "Inbox", "Sent", "Drafts" folders etc.) Close the old PST.

Done !
