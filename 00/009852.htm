rann sanders say:

Hello,

There are a couple of different ways you could come at this.  In the past when I've had this problem, sometimes I would just completely remove Office, reboot, and reinstall.  And sometimes even that didn't fix the issue.

Here are the recommended troubleshooting steps, let me know if you've already gone through these processes.

Clean Your Outlook Profile

To clean your Outlook profile:

  1. Close Microsoft Outlook if it is open
  2. Press Windows Key + R to open the Run dialog box
  3. In the Run dialog, type Outlook /cleanprofile
  4. Click OK or press Enter.

Start Microsoft Outlook and see if the problem is solved. If not, you will need to see if any add-ons are creating the problem.

Start Outlook in Safe Mode

  1. Close Microsoft if it is open
  2. Press Windows Key + R to open Run Dialog box
  3. In Run Dialog box, type Outlook /safe
  4. This will start Outlook in safe mode where any third party add-ons are disabled.
  5. Check and see if your Microsoft Outlook is working properly

If MS Outlook works properly in safe mode, it means some add-on is creating causing Outlook to close. Now you need to isolate and remove the add-ons that might be causing Outlook to crash. To do this:

  1. Open the File menu (Office 2010) or Office menu (Office 2007).
  2. Click on options
  3. Click Add-ins
  4. Towards the bottom of the window you are presented with, select COM and click Go.
  5. You will be now presented with a list of add-ons in Microsoft Outlook
  6. Disable all add-ons (remove the check mark).
  7. Enable one add on at a time (check the checkbox against each one, one at a time) and run Microsoft Outlook to see if works properly

When you identify the issue-causing Add-in, you would disable and/or uninstall it.

Requirement

Suppose I am using English (United Kingdom) as my proofing language and would like to add a two-word phrase like Data Center,say from a company's product whose spelling I cannot alter, into a custom dictionary as a proper noun. How would I go about doing this without spellings like center or Center appearing on their own being treated as correct?

Prior Knowledge

I already know how to create a custom dictionary via File > Options > Proofing > Custom Dictionaries... and set the dictionary to the correct language. I am able to add, say Center, into the dictionary and have the red underlines disappear, but adding the full phrase "Data Center" does not work.

stefan blom say:
In the body of the document, type in your two-word term and select it. Apply "Do not check spelling or grammar" via Review > Language > Set Proofing Language and click OK to close the dialog box. While still leaving the two-word term selected, display the AutoCorrect dialog box (File > Options > Proofing > AutoCorrect Options) and go to the AutoCorrect tab if it isn't already displaying. Select the "formatted text" radio button, type in a trigger in the "Replace" box and confirm by clicking the Add button. Close dialog boxes.
