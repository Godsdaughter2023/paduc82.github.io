How to set language for Mac Word 2016 - Microsoft Community

s judith z say:

How to set language for Mac Word 2016

There seems to be no way to set languages and dictionaries. Kindly advise!
bob jones aka: cybertaz say:

It sounds like you're referring to a custom toolbar button, which you cannot do in Office 2016. However, for what you're asking to do there's no customization needed.

In Word 2016 there is a button on the Status Bar which displays the language currently being used. (If it isn't displayed, right-click the Status Bar & select Language.) You can then use it to switch from one language to another... You can't 'toggle' back & forth directly between 2 languages so it may not be quite as convenient but it should help.

As an alternative, you might consider creating Styles which are identical except for the Language attribute. You can assign a keystroke to each or just apply them from the Styles pane. In fact, you may only need to create one to apply Italian & just switch to a different existing Style when you want to revert to English.
