Hi Michal,

Thanks for posting the query on Microsoft Community.

Am sorry to know that you are not able to find exact RAM usage in the computer. You can find out, how much memory is being used by RAM by going to the, Resource Monitor to check memory in real time.

  1. Go to Search and type Run
  2. Click on Run. Then type Resmon.exe and click ok.
  3. Then you find memory tab. Click on Memory option. You will find % of RAM used.

It could be showing incorrect percentage of Ram used, due to virus or 3rd party applications or any software changes on your computer.

Follow the steps mentioned below.  

  • Press “Windows Key + C” to open Charms Bar.
  • Select Search option.
  • Select “Settings” then type “Troubleshooter” without quotes in the Search box and hit enter.
  • Select “System and Security”.
  • Select “System Maintenance” under System options.
  • Follow the on screen instructions to run the troubleshooter.

If it is still not showing the exact RAM used, do let us know.
