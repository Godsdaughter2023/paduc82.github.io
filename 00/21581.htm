rohn007 say:

Yes it can be done, it will take a little work.

At the end of the first document, insert a next page section break. Whether you make it "next page" an "odd number" page break is up to you. Do you want to the second document to print with the pages on the same side of the the paper?

Do you want the page numbering to restart for the second document?  Go into the header/footer of the second section. Go to "Header & Footer Tools" contextual tab > Header & Footer section > Page number drop down > Page Number Format command.  In the dialog you can set Page Number to start at 1 again.

To separate the entries in the 2 TOCs you have 2 options. 

You could duplicate the heading styles, creating a duplicate set of heading style for the second document. Then you make sure to specify those heading names when you setup the second TOC.

make sure the levels are set correctly in the second set

Changing Levels And Formatting http://blogs.office.com/b/microsoft-word/archive/2010/07/14/word-of-the-week-toc-changing-levels-and-formatting.aspx

http://wordribbon.tips.net/T012660_Creating_a_Table_of_Contents_from_TOC_Fields.html

Then you can select the second half and do a series of Find and Replaces to change the heading styles applied in it.  Then you generate a custom TOC field second set of heading styles

Creating a TOC that Includes Specific Styles / Including a Specific Style in a TOC
http://wordribbon.tips.net/T007123_Including_a_Specific_Style_in_a_TOC.html

Using Multiple Tables of Contents http://wordribbon.tips.net/T010080_Using_Multiple_Tables_of_Contents.html
Adding multiple tables of contents is a must for some types of document design. Here’s a great overview of how you can add all the tables you need.


The second option is to use "TC" field codes to define the heading entries in the second document and specify using TC fields when you create the TOC. This will take more effort.

Using TC Fields for Notes http://word.tips.net/T001400_Using_TC_Fields_for_Notes.html
The TC field is normally used in constructing manual Tables of Contents. The way the field works, however, makes it a natural for pulling together a handy “to do list” for a document.

TOC (TABLE OF CONTENTS) FIELD http://office.microsoft.com/en-us/word-help/field-codes-toc-table-of-contents-field-HP005186201.aspx
Builds a table of contents. The TOC field collects entries for a table of contents using heading levels, specified styles, or entries specified by TC (Table of Contents Entry) fields. Microsoft Word inserts the TOC field when you use the Index and Tables command from the Reference submenu on the Insert menu.
