gord dibben say:

Two macros for protect and unprotect all sheets are enclosed.

These would go into your Personal Macro Workbook only, not into the workbook you send.

 

To learn how to create a Personal Macro Workbook see Excel Help.

To learn how to add a button to QAT see Debra Dalgleish's site.

 

http://www.contextures.on.ca/excel-quick-access-toolbar-customize.html#Add

 

Sub ProtectAllSheets()
    Application.ScreenUpdating = False
    Dim n As Single
    For n = 1 To Sheets.Count
        With Sheets(n)
          .EnableSelection = xlUnlockedCells
          .Protect Password:="justme"
        End With
    Next n
    Application.ScreenUpdating = True
End Sub

 

Sub UnprotectAllSheets()
    Application.ScreenUpdating = False
    Dim n As Single
    For n = 1 To Sheets.Count
        With Sheets(n)
          .Unprotect Password:="justme"
        End With
    Next n
    Application.ScreenUpdating = True
End Sub

steve rindsberg say:

In addition to Bob's questions, check this as well:  how many links are there in the entire presentation, and how long are the titles of the slides you're linking to?

PowerPoint reserves a limited amount of shared space for links and other things.  If the amount of data used by the links exceeds this, links will break or go to the wrong slides.  The slide title becomes part of the link data that PPT stores, so slides with long titles will eat up available storage faster than slides with short titles.

brian tillman [outlook mvp 2007-2019] say:
Sounds like you imported the data, not opened the data file. Opening the data file (File>Open & Export>Open Outlook Data File) will always place it as a separate set of folders in the Navigation Pane. It will not intermingle.

If, in fact, the messages are intermingled, use Instant Search to look for "To:theotherperson" or "From:theotherperson" to isolate the messages. You might also try adding the "Email Account" field to the column header using Field Chooser to see if those messages show a different account than yours. If so, you can sort or group by that field to gather all of the other person's messages and give you a chance to delete them.
