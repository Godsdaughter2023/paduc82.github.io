It would be nice to live in a world where all our dreams come true. Unfortunately, in the real world, we have to live with the limitations designed into the software.

You can use Paste Special and paste as a link, then the Excel file will be linked, not embedded. If you do so, then editing the date in Excel will update the information in PowerPoint. Please check the timezone and the daylights savings time setting on your device and the server.  These must be correct in order to ensure that the calendar events will be stored correctly.

This occurs because Windows Mobile and Exchange use the offset from GMT for the timezone to store the calendar events.  If the offsets are wrong due to the wrong DST setting then you will see differences in the calendars by one hour.
