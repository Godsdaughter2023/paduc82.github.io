Hi,

Some controversy about the difference, possibly powered by Nvidia and AMD who both produce
competing chipsets and high-end video cards both of which are threatened by the i7 and X58
chipset combinations.

Its all about architecture.

i7 = 4 cores on 1 die                       core 2 duo 2 cores on 1 die, 2 dies put together

Difference is speed and the 4 cores can produce hyperthreading which equal 8 virtual cores.

Also the i7 has Quick Path InterConnect which replaces Front Side Bus. And it has an On Die
Memory Controller which allows much faster memory access that older models which had an
external memory controller.
