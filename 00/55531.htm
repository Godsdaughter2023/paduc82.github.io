alex truong say:

When you install the new version, your Movie Maker project files and associated photo and video files that you used in your projects will not be touched, moved or modified.

 

When you open your Movie Maker 2011 project file with Movie Maker 2012, everything should be the same and should just work as long as you haven't moved any photo or video files from their original locations.

 

Note that after you save a project using Movie Maker 2012, the 2012 project file can no longer be opened with the older Movie Maker 2011 since the project file will be in the newer 2012 format.

  1. Click the Start Button on the Desktop.
  2. In the Search field type Date and Time and press enter.
  3. Verify the date and time is set correctly

Retype the Product Key:

  1. Click the Start Button on the Desktop.
  2. Select Computer.
  3. Select System Properties on the Taskbar.
  4. At the bottom click Change product key and retype your Product Key.

Phone Activation:

  1. Click the Start Button on the Desktop.
  2. In the Search field type SLUI 4.exe and press enter (Note, there is a space between “I” and “4”)
  3. Select your country.
  4. Call the phone number listed and follow the automated prompts to activate Windows.
