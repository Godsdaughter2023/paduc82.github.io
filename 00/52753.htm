Bug check 0x50 usually occurs after the installation of faulty hardware or in the event of failure of installed hardware (usually related to defective RAM, be it main memory, L2 RAM cache, or video RAM). Another common cause is the installation of a faulty system service. Antivirus software can also trigger this error, as can a corrupted NTFS volume.

Follow the methods below and check if it works.
 
Method 1:  If hardware has been added to the system recently, remove it to see if the error recurs. If existing hardware has failed, remove or replace the faulty component.
 

Run the hardware and devices troubleshooter and check if it helps:

It checks for common issues and makes sure that any new device or hardware attached to your computer was installed correctly.

http://windows.microsoft.com/en-us/windows7/Open-the-Hardware-and-Devices-troubleshooter

Method 2: If the error occurs during system startup, restart your computer, and press F8 at the character-mode menu that displays the operating system choices. At the resulting Windows Advanced Options menu, choose the Last Known Good Configuration option.

Using Last Known Good Configuration
http://windows.microsoft.com/en-us/windows/using-last-known-good-configuration#1TC=windows-7
 
Check your hard disk for errors in Windows 7
http://support.microsoft.com/kb/2641432

Media Center guide download error code 14 caused by BladeRunner Pro

Issue

Installing or uninstalling BladeRunner Pro can cause Media Center guide download to fail with error code 14.

Workaround

BladeRunner Pro changes some guide download registry values, which can cause download errors in certain cases.  If you currently have BladeRunner Pro installed or recently uninstalled and are encountering guide download error code 14, the following steps may help resolve it:

1.    Click on Start; in the start search type regedit and hit enter.
2.    Rename the registry sub-hive located at HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Media Center\Service\EPG
3.    Launch Media Center
4.    Go to Settings | TV | Guide and choose Setup Up Guide Listings
5.    Reconfigure guide settings - this will cause Media Center to recreate the registry sub-hive that was renamed in step 1 above

Important: Steps above tell you how to modify the registry. However, serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully.
