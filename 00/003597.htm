Win 8, 8.1 and 10 have a new feature called Fast Startup. It can sometimes cause problems like you mention as well as others.

cupajo say:

How to make worksheet forget external link values when links are broken

I have a worksheet that contains external links to another file on a network server, and I want the worksheet with the references to "forget" the external link values when the user closes the worksheet.  Then if the file is reopened offline, I want those external link values to return a #REF.  Instead, what I see are the previously updated values...even though I've unchecked the "Save External Link Values" box.  Please help - thanks!

ashish mathur say:

Hi,

 

In all cells of the file (where you want the #REF error to appear), add the DUMIF() function to the existing formula.  Therefore, say, there is a formula =[Book2]Sheet1!$C$3 in Book1.

 

To this just add the SUMIF() function

 

=[Book2]Sheet1!$C$3+SUMIF([Book2]Sheet1!$C$12:C15,D22,[Book2]Sheet1!$D$12:D15)

 

[Book2]Sheet1!$C$12:C15 and [Book2]Sheet1!$D$12:D15 should be blank so that the result is unaffected (when Book2 is open)

 

Since the SUMIF() function does not work if the source workbook is closed, you will see errors in cells which have the SUMIF() function.
