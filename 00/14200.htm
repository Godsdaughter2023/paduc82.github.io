Hello,

Your Outlook is connected to the Telus server, and there you are receiving emails from that domain.

As it says, they do not have connections between the Telus server and Office 365. So the emails that are received in Telus can not go to Office 365.

The emails of a domain go to where the DNS record of type MX is pointing. Apparently the MX records are pointing to the Telus server and that's why the emails from your domain never reach Office 365.

andreas killer say:
C2: {=MATCH(C1,ROUND($A$2-A3:A11,2),0)+2}

If you apply {=$A$2-A3:A11} into B3:B11 you get the same values as in your helper column.
Comparing decimal values is always critical due to the IEEE 754 data format, therefor we have to round the result to cut off unnecessary decimal places.
Now we can use MATCH to find the number in that array, but as the array start at row 3 we have to add 2 to calculate the real row number in the sheet.

Andreas.

Excel VBA code to remove unwanted characters

I would like VBA code to remove unwanted characters. I will provide a list of WANTED characters and the code will delete all other characters thru the entire worksheet.

perhaps I can my wanted characters in the .patern statement (just advise on how they are to be entered)

Thank you,

hansv mvp say:

Try this:

Sub StripCharacters()
    Dim RE As Object
    Dim rng As Range
    Set RE = CreateObject("VBScript.RegExp")
    With RE
        .Global = True
        ' Place the WANTED characters between [^ and ]
        .Pattern = "[^a-zA-Z0-9]"
        For Each rng In ActiveSheet.UsedRange.SpecialCells(xlCellTypeConstants)
            rng.Value = .Replace(rng.Value, "")
        Next rng
    End With
End Sub

In this example only letters and digits are kept.
