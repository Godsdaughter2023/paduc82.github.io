' Here is my problem. I want to put the data in Cells A1 to A1000 in this example. I use the CVar to convert the range string to a Variant (that is what

' excel wants to have for the range designators) and try to insert my array contents to the range.  

ActiveSheet.Range(CVar(strRange)) = dynName

' Excel accepts this apparently (there are no error messages) but the range stays blank in the worksheet.  Nothing I have tried works.

hansv mvp say:

You should transpose the array - Excel treats a one-dimensional array as a row, not as a column.

Furthermore, since strText begins with a semi-colon, the first item of the array is empty.

Here is a more efficient version that avoids looping:

Public Sub GetData()
    Dim strFile As String, strText As String, strRange As String, x As Long
    Dim dynName As Variant
    strFile = "C:\Data.txt" ' File path and name
    Open strFile For Input As #1 'Open the file for reading.
    strText = Input(LOF(1), 1) ' Read the contents into a string
    Close #1 'Close the file as we are finished with it
    dynName = Split(strText, vbCrLf)  'Insert the data, separating at line breaks
    x = UBound(dynName) + 1 ' Get the number of rows
    strRange = "A1:A" & x  'Range address
    ActiveSheet.Range(strRange) = Application.Transpose(dynName) ' Fill the range
End Sub

gina whipp say:

Let's try this...

Function fCloseExcel()
On Error GoTo handler
 
    Dim objExcel As Object
    Dim xlWorkbook As Object
 
    Set objExcel = GetObject(, "Excel.Application")
   
    Do Until objExcel Is Nothing
        For Each xlWorkbook In objExcel.Workbooks
            objExcel.DisplayAlerts = False
            xlWorkbook.Save
            xlWorkbook.Close
            objExcel.ActiveWorkbook.Saved = True
            objExcel.ActiveWorkbook.Close True
            objExcel.DisplayAlerts = True
        Next
            objExcel.Quit
            Set objExcel = Nothing
   Loop

  
handler:
    objExcel.Quit
    Set objExcel = Nothing
    If Err <> 429 Then
        MsgBox Err.Description, vbInformation
    End If
End Function

Hopefully, this does it.
