john_barnett say:

Hello Telegram_Sam,

 

My apologies for giving you inaccurate facts. looks like I've been using Windows 8 for too long:-) The options I referred to on the dual boot menu are not available for the Windows 7 and below boot menu so we can't use that to change the default operating system. However, all is not lost.

Proceed as follows:

 

1. In Windows 7 open Windows Explorer and Right click on Computer.

2. From the drop down menu, Click on the Properties option.

3. The "View basic information about your computer" window will now open. Look on the left hand side of this window and you will see a list of options. Click on Advanced System Settings.

4. The System Properties window will now open with the Advanced tab visible.

5. On the Advanced Tab, click on the Settings button in the Startup and Recovery section.

6. In the Startup and Recovery Window look for the section market "Default Operating system" at the top of the window.

7. Click the Arrow on the right hand side of this box and, from the drop down list, change the default operating system to XP.

8. Now click OK, followed by OK in the System Properties Window.

9. Finally close the System Window.

10. Now when you reboot XP will be the default Operating system.

 

Incidentally you can, from the same Startup and Recovery Window change the time allowed for the menu to remain on screen.

 

Hope this helps.

If you've already installed Toolbar but it's not displaying, try the troubleshooting steps below to resolve the issue. If a particular step fixes the problem, no need to try subsequent steps. Having trouble installing Toolbar?

Toolbar for Internet Explorer
Step 1: Enable add-ons
  1. Click the Internet Explorer Tools menu, or press Alt-T on your keyboard.
  2. Select Manage Add-ons.
  3. If Google Toolbar, Google Toolbar Helper, or Google Toolbar Notifier BHO is disabled, select each add-on and click Enable.
  4. Click Close.
Step 2: Enable the Toolbar in the "View" menu
  1. Click the Internet Explorer View menu, or press Alt-V on your keyboard.
  2. Select Toolbars.
  3. If Google Toolbar doesn't have a checkmark beside it, select Google Toolbar.
Step 3: Reset Internet Explorer settings
  1. Click the Internet Explorer Tools menu, or press Alt-T on your keyboard.
  2. Select Internet Options.
  3. On the Advanced tab, click Reset.
  4. Select the "Delete personal settings" checkbox.
  5. Click Reset.
  6. If you still don't see the Toolbar, try the steps above again to re-enable add-ons and the Toolbar.

Hi Peter

 

In the Windows 8 Pro version there is a Group Policy that disables the Lock Screen by making a change to the registry. In Windows 8 core, you can implement the same change by editing the registry directly.

 

Press the WinKey+R to display the Run dialog, type  regedit.exe  and click OK to open the registry editor.

 

Navigate to the following registry key.

 

HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows

 

Right click the Windows key and select New/Key. Rename the new key  Personalization.

 

Right Click the Personalization key and select New/DWORD (32-bit) Value.

 

Name the new value NoLockScreen  (Case sensitive). Right click the NoLockScreen value and select Modify.

 

In the Value data box, type the number 1  and click OK.

 

Exit the registry editor and restart the computer to see the change.
