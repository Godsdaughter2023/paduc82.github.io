ron sommer mvp 2009-2017 say:

One or several of your devices/programs are using pop3 protocol to access your emails on outlook.com.  The default procedure when accessing emails using pop3 protocol is to delete messages from the server after the emails are downloaded.  Instead of deleting the messages, outlook.com moves the emails to the POP folder.

Check each device/program to make sure the Leave a copy settings is selected.
