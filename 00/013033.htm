suzanne s. barnhill say:
Where do you have the option checked and when does Word fail? Because there are two settings and two scenarios for the formatting. Go to Office Button | Word Options | Proofing: Autocorrect Options, and you will see that there are separate tabs for AutoFormat and AutoFormat As You Type.
  1. Check the option on the AutoFormat As You Type tab to have links converted as you type them.
  2. Check the option on the AutoFormat tab so that you can explicitly run AutoFormat on a document with existing URLs/email addresses and have them converted.
In addition, if you have the Insert Hyperlink button on a toolbar, you can convert an existing link by selecting it and clicking the button.
