Just to get "terminology" out of the way

Re: #1 & 2 - "Contact Groups" has an explicit meaning when referring to an item in an Outlook contact folder contained in a PST file. "Contact Groups" are not exported when exporting a contact folder to a CSV file

A "Contact List" in Outlook.com parlance refers to the equivalent of a "Contact Group" in Outlook desktop program (part of the Office suite).

You don't export an "Address List" in Outlook (desktop) but a "contact folder" to a CSV file.

The above might sound like nit-picking but each of those items "Contact Group" / "Contact List" / "AddressList" has a specific meaning which can cause confusion when describing a given situation

Re: #3 - You can't import a PST file into an Outlook.com account via the browser import process so that can be taken out of the equation altogether
