Ctrl+X not working properly - Microsoft Community

maryann149 say:

Ctrl+X not working properly

When I try to use Ctrl+X the entire Excel sheet gets selected. 

PS: I use an Excel 2007. How do I fix this?

bill manville say:

First see if this happens in Safe Mode.

Hold down ctrl while launching Excel.

If Ctrl+X then behaves normally, the bad behaviour is due to an add-in or a workbook in an XLSTART folder (as these are not opened in Safe Mode).

While still in Safe mode, open each file in your XLSTART folders one by one and check if the bad behaviour returns - if so that file is the culprit 

The XLSTART folders can be found by typing each of the following. followed by the Enter key, in the visual basic editor's immediate window (Alt+F11, Ctrl+G):

?Application.StartupPath
?Application.AltStartupPath
?Application.LibraryPath & "\XLSTART"

If it was not a file in XLSTART that was responsible, disable all add-ins (Office button > Add-ins) and restart Excel normally.

Then you can enable the add-ins one by one until the bad behaviour returns and the last one you enabled will be the culprit.

If Ctrl+X still misbehaved in safe mode then maybe the workbook you opened when testing is the culprit.

Alt+F11 to the VB editor and Ctrl+G to the immediate window. Type
Application.OnKey "^X"

and press the Enter key

If Ctrl+X then works normally, there is a macro somewhere in your workbook that is assigned to Ctrl+X - this may be done using Application.OnKey or it may be a shortcut key associated with a macro.

Mail merge - address labels - how to skip address2 if blank and use - Microsoft Community

tomasss1982 say:

Mail merge - address labels - how to skip address2 if blank and use address3

Hi all

I have got a problem with mail merge. I want to create address labels. I'm importing values from SQL database.

I got something like this ready ->

 

«client_firstname» «client_surname»

«client_address1» «client_address2 »

«client_address3»

«client_address_city»

«client_address_postcode»

Where client_address1 = house nr , client_address2 = house name, client_address3 = street name

 

I want mail merge to skip «client_address2» when is "" and use«client_address3» instead and then to skip «client_address3» if it was used like that. It might be simple...I don't know....I tried to use IF command but with poor results :(

Pls help

suzanne s. barnhill say:
The secret to avoiding blank lines for skipped fields is to include the line break or paragraph break with the merge field in the IF field. Your
situation is just a little more complex. I think it will work with something like this:

{ IF { MERGEFIELD client_address2  } = "" { MERGEFIELD client_address3 } { MERGEFIELD client_address2 } }
{ IF { MERGEFIELD client_address2  } = "" "" "{ MERGEFIELD client_address3 }
" }

In other words, in the first line (IF field), if the client_address2 field is blank, Word will print client_address3; if client_address2 contains data, that data will be printed. In either case, the line will end with a line break or paragraph break.

In the second line (IF field), if the client_address2 field is blank, then nothing will be printed because client_address3 will already have been printed on the previous line. If client_address2 contains data, then client_address3 will be printed on this line, followed by a line or paragraph break, but because the break is inside the FalseText quotes, it will be used only if client_address3 is printed.

How do i extract emails on multiple sheets in excel - Microsoft Community

eric_993333 say:

How do i extract emails on multiple sheets in excel

I have an excel file with 200 plus sheets, in the box D8 there is one email per sheet, how do i go about extracting and copying all the emails so i can send a mass email in outlook? 
jeeped say:

You are going to want to apply a macro that will cycle through the 200 worksheets and collect the email addresses to somewhere. There are some vagaries in your description but I will try to address them.

Tap Alt+F11 and when the VBE opens, immediately use the pull-down menus to Insert, Module (aka Alt+I, M). Paste the following into the new pane titled something like Book1 - Module1 (Code),

Sub aggregate_emails()
    Dim w As Long
    With Sheets.Add(before:=Sheets(1))
        .Cells(1, 1) = "emails"
        For w = 2 To Sheets.Count
            .Cells(Rows.Count, 1).End(xlUp).Offset(1, 0) = Sheets(w).Cells(8, 4).Value
        Next w
    End With
End Sub

Tap Alt+Q to return to your worksheet. Tap Alt+F8 to open the Macros dialog and Run the macro.
