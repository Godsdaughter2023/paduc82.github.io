patc---01 say: I guess NMM "claimed" (ie: changed the file associations) for the various file types when you installed it.

You need to change the associations - Windows default program - for each file type. There are a couple of ways you can do this.

In Windows Explorer, to change the default for MP3s to Windows Media Player:

- open Windows Explorer
- right click on a an MP3 file
- from the pop-up menu select "Open with" and then "Choose program .."

You'll then see a list of applications.

- Select Windows Media Player
- click the "Alwayse use the selected program to open this kind of file" box at the bottom of the list

You'll need to repeat this process for each affected file type.

Alternatively, open your favourite media player and somewhere in the options there will be a list of file associations where you can "re-claim" the file types you want to use it for.

Usually, when you install media players there's an option during the installation about allowing the media player to change the file associations: generally speaking you should say No.

Calculate date three working days from today - Microsoft Community

jamescowell say:

Calculate date three working days from today

I need to write a formula to calculate the date three working days (mon-fri) from the current date.

 

I am using excel 2003

 

Can anyone help?

 

Cheers

 

hansv mvp say:

Try this custom function:

 

Function MyWorkday(Start_Date As Date, Days As Long, _
        Optional Holidays) As Date
    ' Similar to the WORKDAY function in Analysis ToolPak
    Dim D As Date
    Dim n As Long
    Dim varItm As Variant
    If Days = 0 Then
        MyWorkday = Start_Date
    Else
        D = Start_Date
        Do
            D = D + 1
            If Weekday(D, vbMonday) < 6 Then
                n = n + 1
                If Not IsMissing(Holidays) Then
                    For Each varItm In Holidays
                        If D = varItm Then
                            n = n - 1
                            Exit For
                        End If
                    Next varItm
                End If
            End If
        Loop Until n = Days
        MyWorkday = D
    End If
End Function

 

Copy it into a module in the workbook.

Use like this:

=MyWorkday(TODAY(),3)

Then you aren't completely clearing the TIF.   Do it this way:   Ctrl-Shift-Delete, then check only Temporary Internet Files.  In particular, do not leave Preserve Favorites... checked.   That will delete some favicons but we assume that they can be regenerated when needed.   Alternatively, I suppose you could try using the TIF Viewer (Alt-T O, Alt-S V)  to try to find and delete the particular cached file which is being checked.   Unfortunately, since it is likely to be cached in the Low TIF you may not see it that way.  A search of the TIF and its subfolders may find it but if you deleted it then the TIF's index.dat would not be synchronized.   Unfortunately, the header info is likely to be stored in the index.dat entry so the cache-checking might still happen.   Still, the data itself would be deleted, so it would be interesting to see what the actual result would be.   Caveat emptor!   ; )
