.xlsb is a file extension that indicates the file is saved in binary (not plain text) format

.xlsx is a file extension that indicates the file is saved in plain text (zipped)

.xltx is a file extension that indicates the file is saved in plain text (zipped) and is a template

Choosing Excel File Formats in Office 2011 for Mac - dummies

jim gordon mvp say:

Hi

The point I was trying to make is the .xlsb is a regular Excel file format. These workbooks can be saved in the same directory as any other Excel workbook of type .xlsx or .xls

The only difference between workbooks saved in .xlsx format and .xlsb format is the internal structure of the file.

.xlsx internally is plain text

.xlsb internally is binary (all zeroes and ones)

.xlsb is not a template file format, so you wouldn't put them into the Templates folder, but you can put them wherever you would put any other workbook. You can use your Documents folder or Desktop. These are the two most common locations, but you can put them on a network drive, OneDrive, DropBox or other location.

The binary format is best for large workbooks.

If backup is turned on under Settings, you can restore your apps and settings after performing a hard reset (as long as you initialize the phone with the same Microsoft Account).

Note that resetting your device will not remove any OS or firmware updates. You will remain on the same version of Windows Phone after the reset.

You can reset your phone under Settings: Settings > about > reset your phone.
