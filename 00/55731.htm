    delpinsky say:

    I SOLVED!!!

    Guys now I'm able to use my Iomega Prestige 3.0 1Tb portable plugged into a USB 3.0 port in write mode without Windows getting stuck when copying files to the external hard disk!

    Lenovo (acquired Iomega) guys in the forum were no able to help but they pointed out the issue could reside in the the default USB 3.0 controller drivers of Windows 8.

    So I googled (sorry, no Bing) once again and I found this blog:

    http://plugable.com/2012/12/01/windows-8-and-intel-usb-3-0-host-controllers

    The solution is to disable Microsoft eXtensible Host Controller Driver and USB stack, and install Intel(R) USB 3.0 eXtensible Host Controller, with a couple of modified usb .inf files to overwrite into the Win7 64bit folder of the Intel eXtensible Host Controller Driver.

    Now guys I can finally exploit all the advantages of the writing speed of USB 3.0 for my Iomega Prestige portable hard disk.
