If you created this project in Movie Maker before burning in DVD Maker...
some users report that Publishing the project in the DV-AVI format may
eliminate the appearance of the Green Bar. Once Published...you can
import the .avi file into DVD Maker.

If you want several Scenes (chapters) on the DVD...just create several
different projects in Movie Maker...Publish each one in the DV-AVI format
to a folder on your hard drive and then import the collection of movie clips
into DVD Maker.
