Private Sub cmdRefresh_Click()

Dim olApp As Object
Dim nsMAPI As NameSpace
Dim ContractorFolder As MAPIFolder
Dim i As Integer

Dim Trade As Variant

      DoCmd.RunSQL strSQL

Based on your description, you would like to paste cell's value only on your worksheet using the shortcut keys Ctrl +V, please following the steps below to achieve it with VBA code:

1. Hold down the ALT + F11 keys to open the Microsoft Visual Basic for Applications window.

2. Click Insert > Module, and paste the following code in the Module Window.

VBA code: Set paste values as default paste when using Ctrl + V

Sub PasteasValue()
Selection.PasteSpecial Paste:=xlPasteValues
End Sub 

3. Then save the workbook as Excel Macro-Enabled Template (*.xltm) file type and close the code, and press Alt + F8 keys to open the Macro dialog box.

4. In the Macro dialog, choose the used macro name PasteasValue and Macros in the workbook you just saved, and then click Options button, see screenshot, see screenshot:

5. Then in the Macro Options dialog box, under the Shortcut key section, please enter v, click OK and then close the Macro dialog.

Now when users open the workbook and enable Macro, they will only paste cell’s value using the shortcut keys Ctrl + V.
