PMFJI & perhaps I'm oversimplifying, but ... :-)

As you've discovered, a Chart Placeholder stipulates nothing more than the size & location for a Chart to be inserted on the slide. It contains a button that merely activates the Charts tab of the Ribbon. From there it's up to the user to select the type & style of chart s/he wants to insert. Nor can you [easily] change what you get by using any of the built-in chart types. I think we're all on the same page to this point.

If the user then selects your custom Chart Type from the Other category - either manually or programmatically - as John suggested, that's the type of chart that will be inserted. It should include all of your customized styling, including the lack of Shadow if that's how the template is designed. The only 'catch' is that the user needs to select your creation rather than selecting the built-in Stacked Bar from the Bar category. IOW, even though you can't assign a default to the Placeholder the user ultimately does decide what goes into it.
