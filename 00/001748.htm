bob jones aka: cybertaz say:
The instructions you've been given are based on the assumption that the files have been saved in the default folder for user Templates... files saved with the .xlt, .xlts, or .xltm extension. However, template files can be saved in any location the user chooses.

No offense intended, but I get the impression that you actually may be referring to regular Excel workbook files rather than templates in the first place. Likewise, they are stored in whatever folder was selected when the files were first saved or moved to. Only you know where you may have saved the files.

The bottom line is that in OS X [unlike the Windows OS] you cannot delete files from within the Open or Save As dialogs. You must use Finder to navigate to where the files are stored & delete them from there. If you aren't sure where the files are, use the OS X Spotlight/Find feature in a Finder window to locate them. If necessary, see Mac Help on using Spotlight to locate files. Once you find them,
  • drag them to the Trash, or
  • select them & use the Command+delete keystroke, or
  • right-click them & use the Move to Trash command from the contextual menu.
