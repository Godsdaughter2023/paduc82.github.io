Ok, I usually don't reply and attempt to help but since I spent so much time researching this issue after experiencing the very same issue here it goes:

This solution worked for me on Windows 7 it has not been tested on Windows 8 but it worked for me after I tried all of the same steps as the original poster to no avail.

The problem is the Software Protection service and it's inability to start as you found early on.

Step 1. Open an administrative command prompt.

Step 2. Copy and paste the following command into your command prompt and hit enter. This resets the permissions on the service correctly:

icacls %windir%\ServiceProfiles\NetworkService\AppData\Roaming\Microsoft\SoftwareProtectionPlatform /grant "BUILTIN\Administrators:(OI)(CI)(F)" "NT AUTHORITY\SYSTEM:(OI)(CI)(F)" "NT Service\sppsvc:(OI)(CI)(R,W,D)" "NETWORK SERVICE:(OI)(CI)(F)" 

Step 3. Open Windows Explorer and navigate to %windir%\System32

Step 4. Make sure that you can see hidden and system files and folders.

Step 5. Delete any Files/Folders with the following name (You can move them to an archive folder instead of deleting them if you want to be safe):

7B296FB0-376B-497e-B012-9C450E1B7327-5P-0.C7483456-A289-439d-8115-601632D005A0 

Step 6. In your Windows Explorer window navigate to:

%windir%\ServiceProfiles\NetworkService\AppData\Roaming\Microsoft\SoftwareProtectionPlatform\

Step 7. Rename tokens.dat to tokens.bak

Step 8. Try to start the Software Protection service. It should start successfully now. If it does then restart the computer and try activating Windows again.
