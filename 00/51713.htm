marilyn o say:

1) Go to your User Accounts and see if there are any unfamiliar accounts that are Administrator. If so, delete the accounts. Make sure you have an Administrator account that you have created and change the password so you are the only one who knows the password.
Then delete any accounts that you are not familiar with or don't want on your computer. Do not delete a Network Service account or Local Service account.
Do not delete Administrator, or System or Guest account. However, you can disable the Guest account.

2) Change your user account password every few days until you know that you are no longer being compromised.

3) Right click on the taskbar and open up Task Manager. Go to the users tab. There you can see if there are any other users connected to your computer. If there are users you are not familiar with, highlight the account and disconnect the user.

4) Get your computer secure before you go out on the internet. Only connect to internet when needed. You can turn off your router if you are not on your computer.
If you are using wireless, be sure you are using the most secure authentication type.
Change passwords on routers as most ship with a password. Change the router password so no one else has the password. Check with router documentation on how to change your password.

5) Make sure remote desktop is not enabled on your computer. If you have a user name that you don't know in the Remote Desktop users box, highlight and select Remove. Uncheck, “Allow users to connect remotely to this computer”.  With unchecking this setting your nephew should not be able to connect remotely to your computer.

You cannot use Remote Desktop Connection to connect to computers running Windows XP Home Edition.


Hope this helps to make your computer more secure.
