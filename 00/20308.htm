I think you misunderstood the article. The entire purpose of the Free ISO Burner program is to extract the contents of the input ISO file and write the contents onto a blank DVD.     If you're stuck on the Welcome screen (as opposed to the sign-in screen), press [Ctrl]+[Alt]+[Delete].  The screen should change to a screen with options like Switch user and Task Manager.  Click [Cancel], and you should be returned to the desktop where you will be able to see whatever message is hanging the logon process.  Closing the message should allow your icons and taskbar to load.

    If you're returned to a blank screen with no message, press [Ctrl]+[Alt]+[Delete] again and try the Task Manager option.  If you can start Task Manager:

  1. If needed, click the More details button at the lower left corner to expand Task Manager.
  2. Click File -> Run new task.
  3. Type "cmd" into the textbox.
  4. Tick Create this task with administrative privileges, and click [OK].
  5. Type "explorer" into the command window and press [Enter].
  6. If that doesn't open the desktop, type "sfc /scannow" and press [Enter].
  7. When it is done scanning, note whether it said there was irreparable damage, repaired damage, or that verification completed successfully.
  8. Type "chkdsk C: /f /v /x" and press [Enter].
  9. Confirm that you want to schedule a disk check for the next boot.
  10. Type "shutdown /r /t 0" to reboot your computer.
  11. Let the disk check run (shouldn't take more than 15 minutes), and see if your computer works correctly afterwards.
