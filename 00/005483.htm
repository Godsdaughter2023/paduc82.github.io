Running a macro containing the following code will insert a combobox content control containing the names of the bookmarks in the documentat the location of the selection:

Dim objCC As ContentControl

Dim i As Long
With ActiveDocument
    Set objCC = .ContentControls.Add(wdContentControlComboBox)
    objCC.Title = "Select"
    If objCC.ShowingPlaceholderText Then _
        objCC.SetPlaceholderText , , "Select an item"
    For i = 1 To .Range.Bookmarks.Count
        objCC.DropdownListEntries.Add .Bookmarks(i).Name
    Next i
End With

We can make some assumptions for any leg if we need to: assume a fixed direction (due East/West/North/South) and then calculate ... heck, as you said, Trig could work quite well for this:  it's simple triangulation: you have 1 angle and 2 sides (always same length!); what is the length of the 3rd side?  So we are working with our basic Isoscolese triangle.  Now this won't account for the curvature of the earth, but for distances out to 'only' 500 miles, we just won't worry about it for now.

=2*$A2*COS(RADIANS(180-((B$1/2)+90)))

The result being how far from your destination point you are in same units as in column A.

jllatham say:

A better formula for B2 that allows your table to run all the way out to 360 degrees.  For any angles greater than 180, it simply flops it over, so that if your angle is 181 (for example), then it is calculated as 179, and if it were 359, calculates it as simply 1 degree deviation.  Handles the 'special' case of 180 where your distance from the intended point would be 2*original distance.

=IF(B$1=180,2*$A2,2*$A2*COS(RADIANS(180-(((360-ABS(((B$1>180)*360)-B$1))/2)+90))))
