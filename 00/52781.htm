tryx3 say:

Raphael,

Your task is not an upgrade as such.  You need to do a full installation of Windows 10.  That will involve backing up all your own files first as the hard drive will be completely overwritten.  You will have to install all your applications afterwards as well.

The error message you are getting is the response you would get if you were trying to run the installation from within Windows XP rather than by booting from the installation disk.  You cannot do it that way.

Create a Windows 10 installation disk [on a USB stick] then boot from it.  I have not heard of anybody being able to install from an external hard drive instead [Windows can tell the difference between them].  You can find recommended procedures in -

Clean Install Windows 10 - TenForums

Clean Install Windows 10 - WiKi

People have had quite different experiences with Windows 10 on computers that originally came with XP. 

  • You can find out about any serious compatibility issues with the advice of Windows 7, 8.1, 10 compatibility testing for Windows XP, Vista - WiKi.
  • You will have to experiment with different drivers to find ones that work with your old hardware [see Drivers - Where to get hardware drivers].  You might find suitable ones automatically as Windows will try its best for you during installation.  You might never find suitable drivers for some hardware.
  • You might find after all your hard work that its performance is unsatisfactory.

Make a system image beforehand so you can at least get back to where you started.  Many people praise the system imaging utility Macrium Reflect [free edition].

My own experience of using Windows 10 on an old XP is pretty benign. 

  • I cannot use Task scheduler to wake the computer up despite all the correct settings having been made [and despite having had no such problems with XP]. 
  • I cannot use the Windows PowerCfg utility to create "battery reports" but that is no great loss. 
  • Everything else works but much more slowly than it did with XP as Windows 10 makes far greater demands on the computer's hardware than XP did.
  • I was in touch with another user of the same computer model but he suffered such serious problems with Windows 10 that he had to give up the idea completely.

Denis

tryx3 say:

Raphael,

Thanks but I returned to this thread to add a warning I had omitted before.

If your computer is compatible with Windows 10 today, it might not be compatible with the next version [the current one is Ver 1703 from March 2017; Ver 1709 is expected in September]. 

  • If you do not update to the next version, you will stop getting all updates after a year or so. 
  • For example, the original July 2015 version of Windows 10 has recently had support withdrawn.
  • If any users were unable to update to later versions then they would soon be in the same predicament as XP users, they will not be able to use their computers online with any degree of safety.
  • Once hardware OEMs stop supporting their hardware, with updated drivers to match the requirements of updated Windows 10 versions, then the writing is on the wall - obsolescence looms.
  • For example, users of old computers with a Broadcom 440x Ethernet card found that they could not install Ver 1703.  MS have said they will address that in a future update but it was still a stark reminder of the reality of end-of-support for users of old computers such as ours.  {My BIOS had a setting that allowed me to disable the Broadcom 440x card so I was then able to go ahead with this update}.

If your computer is ever incompatible with a newer Windows 10 version -

  • You should start to keep an eye on Windows 10 update history.  This has a list of "Cumulative updates" for each version so you can see when they stop being issued.
  • You should also start to keep an eye on updates for Windows defender [strictly speaking, I should now refer to Windows defender security centre] to see if its malware definitions updates are still being issued.

I thought I ought to add this warning for you but I have no way of knowing if you will face any problems upon installation or even in the immediate future. 

I remain delighted to have a new lease of life for a 12 year old computer that I originally retired in 2010 but that still has many useful functions to perform for me. 

  • I have been running Windows 10 on it successfully since the start of 2015 [the pre-release "Insider preview" period]. 
  • I can be pretty confident that I will get another year's Windows use out of this 12 year old computer unless it blows up in the meantime.
  •  Once you've reached twelve years old, every day is a bonus.

Best of luck,

Denis
