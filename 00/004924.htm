Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If ActiveCell.Row < 29 Then

End Sub

If you select a cell within Rows 1-28, the scroll area is locked to rows 1:29.

To select a cell > Row 29, first select a cell "In" row 29, then select something > row 29.

Or, use the down arrow to move out of the area.

Yes you can pick that numbering style from the "multilevel numbering" button on the Home tab > Paragraph group.

The numbering level is controlled by tabs. Add a tab to move down a level, delete a tab to move up a level.

Why are you concerned about "losing bullets and formatting"? There may be something else going on.

Actually, if your kids can use word, it should be easy enough to teach them how to use outline numbering.

If they are old enough you may want to consider showing them the Outline view. a. user say:

My home computer currently is running Windows mail and Outlook simultaneously to the same email address.  Some messages go to Outlook, some to Windows mail and some to both.  I want to use windows mail only and disable Outlook, but still maintain my Outlook mailbox history /contacts, etc.  What is the best way to accomplish this?

The reason why you had "  Some messages go to Outlook, some to Windows mail ... " is because youhave not done the following :
Open Windows Mail > Tools > Accounts > your account > Properties > Advanced > check " Leave a copy of messages on server " > OK
Open Outlook > Tools > Account Settings > Change > More Settings > Advanced >check " Leave a copy of messages on server " > OK
If you did, you would find same e-mail in both Inboxes of Windows Mail and Outlook.

You really don't have to " disable " outlook. Just don't use it.
Go to Windows Mail, Advanced tab ( stated above ), and UN-check " Leave a copy of messages on server " > OK.
The reason of doing the Advanced - " UN-check " routine is that if you happen to open Outlook, it won't start downloading e-mails which you already received in Windows Mail.

tod beane say:

I was able to resolve this problem for myself today. Below is the process I followed, perhaps it will work for you too.

Summary:

Locate the problem OneNote file on your hard drive, make a copy of it, and drag the copy into OneNote.

Background:

Following a laptop failure, my files were backed up to a server, and restored to a new laptop.  Most of my OneNote tabs and pages opened correctly, but one Section (tab) within a restored Notebook and Section Group did not. Instead, the same error message reported here appeared ("This section isn't available yet. It was added from another device. You'll be able to use it when the device syncs."). Perhaps file corruptions look like sync errors to OneNote.

Detailed steps (Windows 8.1, OneNote 2013):

  1. Open OneNote to the problem location, where the error appears
  2. Open My Computer / This PC - keep this window visible next to your OneNote window
  3. Under c:\Users\YourUserName\Documents\ look for a folder called "OneNote Notebooks"
  4. Below that you should see folders for each Notebook, which may contain sub-folders for each Section Group, and "*.one" files for each Section (which contains all of the actual Pages and Sub-Pages)
  5. Locate the *.one file for the problem section, e.g. Section-Name.one, then copy-paste it in the same folder (I renamed my copy to Section-Name-Copy.one)
  6. Drag the copy directly onto the OneNote content area.  It should create a new tab (Section) for you called Section-Name-Copy, containing all of your missing Pages.
  7. You can now safely delete the corrupt Section / Tab in OneNote, and rename the copy that you added

Good luck! Regards, Tod

p.s. after I deleted the corrupt section, I found it listed under "Misplaced Sections" in the list of Notebooks. I can't say whether it was there before. 

Procedures to work with misplaced sections are in OneNote online help. You can also find instructions here:  https://support.office.com/en-US/Article/Resolve-misplaced-sections-in-OneNote-32768b56-bc33-42fd-926e-7fccb1af3aa5?ui=en-US&rs=en-001&ad=US
