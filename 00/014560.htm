tushar mehta say:

I'm trying to add only the numbers in cells in the column which are shaded gray.  Would I do this using a SUMIF function?  If so, how would I designate the gray shading format within the function?



If there is a logical basis for coloring the cells enter that criterion/criteria in some column so that the cells contain either a TRUE or a FALSE.

Now, suppose your original column is A and the criterion color is B and you have data in rows 1:1000.

Then, use the unverified formula =SUMPRODUCT(A1:A1000,--(B1:B1000))

If there is no logical basis that you can add as a formula in a criterion column, consider adapting Chip Pearson's http://www.cpearson.com/excel/colors.aspx  In particular see the SumColor function.
