In the button click event in the MS Access form, you can write a SQL statement which join these three table(User Info,Login Info,Website) according to

following code snippet.

 

Dim rs As DAO.Recordset

Dim db As Database

Dim strSQL as String

Dim userID as String

Dim address as String

 

userID=CStr(Me.YourLoginIDText.Text)

Set db=CurrentDB

If rs.NoMatch Then

        MsgBox "User not found"

    address=CStr(rs![WebsiteAddress])

rst.Close

Sub looptest()
    Dim rng As Range
    Dim cCell As Range
    Dim i As Long
    Dim xlCalc As XlCalculation

    With Application
        .ScreenUpdating = False
        xlCalc = .Calculation
        .Calculation = xlCalculationManual
        .EnableEvents = False
        .DisplayAlerts = False
    End With
  
    Set rng = Range("A1:A400000")

    i = 1

    Range("C1") = Now()
    For Each cCell In rng
        cCell.Value = i
        i = i + 1
    Next cCell

    Range("C2") = Now()

    With Application
        .ScreenUpdating = True
        .EnableEvents = True
        .DisplayAlerts = True
        .Calculation = xlCalc
    End With

How do I add a Birthday Calendar in Outlook? - Microsoft Community

apennyforyourthoughts say:

How do I add a Birthday Calendar in Outlook?

Hello there! I migrated my Outlook .pst file to a new computer, and have lots of Outlook Contacts. However, I lost my Birthday Calendar. How do I create a new Birthday Calendar, based on my current Outlook Contacts? Many thanks!
karl timmermans [outlook mvp 2012-2018] say:

Going to presume that the question you are really asking is "How do I add Birthday's from the contacts I migrated to my calendar?" (there is no specific "Birthday calendar")

All comes down to what you mean by/how you "migrated my Outlook.pst" file.

<IF> you simply opened Outlook .pst file on the new computer and it was the "default" from the old computer - then all Birthday's will already exist in the calendar folder contained in that PST file.

<IF> you <imported> your contact data, you will need to re-add them by opening each contact and doing any one of several things such as (in each case, open the contact item)

1) Click on the FullName button - do nothing else but click OK and then save the contact OR

2) Change the Birthday value (set to none - then re-enter the Birthday value) - save the contact

If you have a lot of contacts, the you may be interested in the Birthday functions in ContactGenie Toolkit which has <bulk> functions

http://www.contactgenie.com/toolkit_4_outlook.htm

For some background info - you may find the following articles of interest

Birthday / Anniversary calendar entries via Outlook UI
http://www.contactgenie.info/birthday-anniversary-calendar-entries-via-outlook-ui

Importing Outlook Contact Birthday / Anniversary date values

Understanding Default calendar folder for new Outlook Birthday / Anniversary dates
http://www.contactgenie.info/understanding-default-calendar-folder-for-new-outlook-birthday-anniversary-dates
