I have assumed here you have a 'Data Validation' dropdown and in this case it's in e1. Put you button on the sheet and attach this code. You can extend with additional cases as required and the sub you call doesn't have to be of the same name as the dropdown.

 

 

Private Sub CommandButton1_Click()
Select Case Range("E1")
    Case Is = "One"
        Call one
    Case Is = "Two"
        Call two
    Case Is = "Three"
        Call three
    Case Else
    Exit Sub
    End Select
End Sub
