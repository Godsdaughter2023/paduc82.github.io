jay freedman say:
Right-click the pinned template and look at the list of commands. If the one that's bold is Open instead of New, that indicates an incorrect registry entry. If you right-click any template in the Windows Explorer, you'll probably see the same bold Open.

Also, if you have both .dot and .dotx templates, check the commands on the right-click menus for both kinds of files.

If you see the bold Open command, follow these steps:

  • In the search box on the Start menu, type cmd.  When cmd.exe appears at the top of the Start menu, press Ctrl+Shift+Enter. In the User Access Control prompt, click Yes to allow the program to run. This will open the Command Prompt window with administrator privileges (see Method 2 in http://www.bleepingcomputer.com/tutorials/windows-elevated-command-prompt for illustrations).
  • To fix the problem for .dotx and .dotm templates, copy the following line to the clipboard:

             reg add HKLM\SOFTWARE\Classes\Word.Template.12\shell /ve /d New

  • In the Command Prompt window, right-click and choose Paste. Then press Enter. 
  • You'll see the response

              Value  exists, overwrite(Yes/No)?

  • Type the letter Y and press Enter. You should see the response

              The operation completed successfully.

  • If the Open command is also the bold command for .dot templates, repeat the previous steps but copy the following line instead:

               reg add HKLM\SOFTWARE\Classes\Word.Template.8\shell /ve /d New

  • Close the Command Prompt window and check out your pinned template.

        vijay a. verma say:
        This is not possible through built-in function supplied by Excel. But it can be done through custom VBA function.

        1. Open your workbook and ALT+F11
        2. Locate your Workbook name in Project Explorer Window
        3. Right click on your workbook name > Insert > Module
        4. Copy paste the Macro code given
        5. Save your file as .xlsm

        You can use the formula like
        =SumDU(A1:A10)

        Function SumDU(Rng As Range)
        Dim Cell As Range
        For Each Cell In Rng
        If Cell.Font.Underline = xlUnderlineStyleDouble Then
        SumDU = SumDU + Cell
        End If
        Next Cell
        End Function
