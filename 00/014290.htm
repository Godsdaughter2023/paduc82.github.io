Workaround suggestion:

It is a shame that it does not work the way you would expect. I am a big fan of the StyleRef Field.  This is apparently one of those places where you want to disable the field.

suzanne s. barnhill say:
Try this: At the end of the main document, before the endnotes, insert a Next Page section break. Insert page breaks so that you have three pages in the section (this will allow you to access the First Page, Even Page, and Odd Page headers). Unlink all three of those headers from the previous section and delete the header content. Then delete the two extra pages. With the insertion point in the first page of the section, go to the Layout tab of Page Setup and change the section start type to Continuous. This should reduce the section to the bottom of the last page of document text, with the endnotes (presumably) starting in a new section on the next page. The endnote section should inherit the empty headers from the previous section.
