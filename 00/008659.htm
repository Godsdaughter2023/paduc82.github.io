jay freedman say:
I don't see Word 2007 replacing tabs with spaces in SEQ field codes, so I'm not sure of the origin of your problem.

Be that as it may, you should be able to replace the field codes without specifically searching for the fields. The following works for me:

  • Press Alt+F9 to display all the field codes in the document.
  • Copy the SEQ Figure \#"'Figure '0'. '" from any SEQ field code. Avoid selecting the field markers { } and get just the code.
  • In the Replace dialog, paste that field code into the Find What box. Paste it again into the Replace With box, then delete the offending space and type in ^t in its place.
  • Click the Replace All button, and close the dialog.
  • Press Alt+F9 to collapse all the field codes. Press Ctrl+A to select the whole document, and press F9 to update all fields.


You should now find tabs in the Figure SEQ fields.

a. user say:

Hi Tonny,

I am solving this issue by VBA code. Please follow the steps:

  • Open your Excel sheet which has URLs.
  • Press ALT+F11 to open VBA.
  • Go to Insert->Module
  • Copy the following code into the module:
Sub ExtractHL()
Dim HL As Hyperlink
For Each HL In ActiveSheet.Hyperlinks
HL.Range.Offset(0, 1).Value = HL.Address
Next
End Sub

  • Save it.
  • Now select the module and Click on RUN option.

Good Luck!
