Hi,

You can use this to count the unique values in a filtered list. Note that I haven't used table referencing because to formula gets long. In this b2:B7 is the column excluding the header we're counting.

Note these are bot ARRAY formula. See below for how to enter them

=SUM(IF(FREQUENCY(IF(SUBTOTAL(3,OFFSET(B2,ROW(B2:B7)-ROW(B2),,1)), IF(B2:B7<>"",MATCH("~"&B2:B7,B2:B7&"",0))),ROW(B2:B7)-ROW(B2)+1),1))

If you want to use table referencing this is it. Note that B2 refers to the topmost cell (Not the header) in the column we're trying to count.

=SUM(IF(FREQUENCY(IF(SUBTOTAL(3,OFFSET(B2,ROW(Table1[Name of Volunteer])-ROW(B2),,1)), IF(Table1[Name of Volunteer]<>"",MATCH("~"&Table1[Name of Volunteer],Table1[Name of Volunteer]&"",0))),ROW(Table1[Name of Volunteer])-ROW(B2)+1),1))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

david a matson say:

Greetings newnews.  Welcome to the Windows Phone forums!

The IME (Input Method Editor) is merely the gateway interface to the various keyboard SIPs (Software Interface Panel) like Swype, etc.   If it takes 10 seconds to load, then you likely have an installed SIP that is simply too big, or you could also have too many SIPs loaded into your device.  Bear in mind that the more SIPs that you have loaded, the longer it will take for the initial IME interface to load and provide you with the various options.

Please consider uninstalling SIPs other than your primary SIP to see if that affects the loadtime of your main IME interface.

Most likely, this will become an equation of Form (quick load) or Function (many choices), where you will have to chose one over the other in order to be happy.

suzanne s. barnhill say:

There is not a separate header for each page. Headers are section properties, and, although it is possible to have up to three different headers per section, if you have enabled only "Different First Page" and not "Different Odd and Even," then you have only two: the First Page Header and the primary Header. If you insert the Page X of Y fields in those two headers, then the one in the Header should repeat on every page after the second.

If the page number is not incrementing, perhaps you have typed the number as ordinary text instead of inserting a PAGE field. You can insert this field anywhere in a document by pressing Alt+Shift+P.
