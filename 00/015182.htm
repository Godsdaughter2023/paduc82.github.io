Here's how I ended up resolving the issue:

- I unchecked the “Show Hidden files, folders and drives” box, as well as the “Hide protected operating system files” box

- Navigated here:  C:\Users\*User name*\AppData\Local\Microsoft\Outlook\RoamCache

- There were 2 “Stream_autocomplete….” Files; one with a current time stamp, and one from the prior week, which was much larger in size.

- Copied the prior week’s (larger) file and saved it (Temp folder) as backup

- On the file with the current timestamp, right clicked > selected rename and copied the name. Then renamed that file to “old” (and removed the .dat file ext).

- On the prior week’s file (the larger file) right clicked > rename and pasted name of the current file.

- Closed/relaunched outlook and the auto complete entries were restored.
