zigzag3143 (ms -mvp) say:
Both Hotmail & Outlook.com no longer support read receipts.  Window Live Mail does however.

The read receipt feature is currently not available.  To have this feature, you may use Windows Live Mail (email client). Please refer to the steps and links below on how to configure your account in Windows Live Mail to start using the read receipt feature:

1. Download and install Windows Live Mail.
2. Start Windows Live Mail and sign in using your Hotmail credentials.
3. How do I add an account to Windows Live Mail.

 

To enable this feature in Windows Live Mail client 2011, please refer below:

1. Open Windows Live Mail client.
2. Press Alt + F, O, M.
3. Click on Receipts tab.
4. Under Requesting Read Receipts, check the checkbox Request a read receipt for all sent messages.
5. Under Returning Read Receipts, click the radio button: Notify me for each read receipt request if you want to return read receipts.
6. Click Apply then OK.
