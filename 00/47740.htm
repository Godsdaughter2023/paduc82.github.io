hassanb say:

Hello,

Please try to reset the consoles display settings while the component cables are plugged in by following the directions below:

  1. Remove any discs from the disc tray.
  2. Turn off the console.
  3. Turn on the console.

    Note: If you are using an Xbox 360 Wireless Controller, turn on the console by using the Guide button on the controller. Make sure that you use the player 1 controller when you do this. The player 1 controller will have the upper-left light illuminated.
     

  4. As the console starts, on the controller, press and hold Y while pulling the right trigger. The display settings are reset to the default settings, and then the console automatically restarts.
     

If this doesn't work please try to do the following:

Try a new set of component cables.
Try to use another TV with component (if available) with the old pair of cables and a new pair of cables.

Sound but no picture?

Try the following:
 

  • Make sure that your TV is on.
  • Remove any devices, such as a receiver, placed between the console and the TV.
  • If your TV has an eco/energy saving mode, turn off the video options.
     

More information here.

If this does not work then it's likely that there is something wrong with the component input of the XBOX 360 and it would need to be repaired. This can be done by going to the Microsoft repair site here. Assuming your console is not in warranty the charge is $99. However, I suggest that you put that money towards a new console if component is that important to you.
