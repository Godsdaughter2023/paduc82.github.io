frederik long say:

Creating a system image on DVDs is strongly discouraged. You need far too many disks and there is a substantial risk that the whole image becomes useless because one of the DVDs is flawed. Much better to use an external USB disk. You can get a 500 GByte disk for around $50. Its capacity would be equivalent to 100 DVDs . . .

When you create an image and store it on a hard disk then you get a .VHD file (Virtual Hard Disk) that you can "mount" as a separate drive, e.g. drive Z:. I do not know if this applies to DVDs too. The process requires a fair amount of skills at a maintenance level. If you do not have these skills then your best bet is to ask a friend to help you.

frederik long say:

The information or pictures are scattered over each disk so no information is complete , am I on the right track

Not exactly. A .vhd file is an image of a whole partition. Whether Windows is capable of mounting an image when spread over numerous DVD I have no idea. Here is how it would work if the file was on a hard disk in one piece and if it was complete:

  1. Use Windows Explorer to locate the .vhd (Virtual Hard Disk) file in the folder \WindowsImageBackup on your external disk (or first DVD). Note the .vhd extension.
  2. Launch the Disk Manager (diskmgmt.msc).
  3. Click Action, then Attach Virtual Hard Disk.
  4. Use the Browse button to locate the file you found in Step 1 above, then click OK.
  5. Note the new drive letter that Windows assigned. It could be G:. If no drive letter was assigned, right-click the new disk and give it a drive letter.
  6. Use Windows Explorer to open drive G: and retrieve your files.
  7. When finished, close Windows Explorer.
  8. Go back to the Disk Manager.Right-click the new disk on the left side of the panel where it says "Basic" and "Online".
  9. Click "Detach VHD".
