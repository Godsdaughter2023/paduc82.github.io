don varnau say:

Hi,

 

Yes, with IE9, 32-bit IE9 will be the default.

 

You don't necessarily have 64-bit IE9 set as default, but the IE icon/shortcut you're using has 64-bit IE9 as its target.

 

Right-click the IE icon> Properties> Shortcut tab and change Target to  "C:\Program Files\Internet Explorer (x86)\iexplore.exe"

 

Or, browse to [C:\Program Files (x86)\Internet Explorer\iexplore.exe] and create a shortcut to iexplore.exe. Place that shortcut on the Taskbar and create another for the Start Menu.
