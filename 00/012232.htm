You cannot convert a .jpg file to a .doc file.  However, you may be able to insert the jpg as a picture behind the text in a document so that you can type your letter in front of it.  It will probably be best to insert it into the First Page Header of the document.  Then it would be best to save the file as a template so that you can use File>New and select that template as the basis of the document that you want to create so that you only have to insert the picture the one time.

suzanne s. barnhill say:

I am often called upon to create templates with images for the letterhead and footer. What Doug suggests is the necessary procedure.

  1. Access the header by double-clicking in the header area.
  2. On the contextual Header & Footer Tools | Design tab, in the Options group, check the box for Different First Page. You will now be in the First Page Header.
  3. Use Picture in the Insert group to insert your letterhead graphic.
  4. On the contextual Picture Tools | Format tab, in the Arrange group, click Wrap Text and choose Behind Text.
  5. If your picture was originally inserted In Line With Text, you will need to resize it. Assuming it is full-page-width, use the control in the Size group (or drag a corner) to restore it to its natural width.
  6. In the Arrange group, click Position and choose More Layout Options.
  7. On the Position tab of the Layout dialog, set the horizontal and vertical alignment to "Left relative to Page" and "Top relative to Page."
  8. Because your image is Behind Text, it will not push the document body down, so you will need to add some space to your First Page Header to clear the letterhead (this will not affect the margin on subsequent pages). To do this, right-click in the default (empty) paragraph and choose Paragraph... In the Paragraph dialog, add some Spacing Before. You can type this into the box in inches or centimeters using a number followed by ", in, or cm; Word will convert it to points. This may require some trial and error.
  9. When you have completed setup of the header, move the insertion point to the footer and repeat the steps for the footer image.
  10. When you have finished, save the document as a template. For more see, http://wordfaqs.mvps.org/LetterheadRibbon.htm.

tatiana33tatiana say:

What is the equation that Excel uses for the PPMT() and IRR() functions?

joeu2004 say:

Tatiana wrote:

I am searching for formulas of IRR() and PPMT() excel function, but i haven't found a certain answer. 

The formula for PPMT(rate,per,nper,pv,fv,type) could be written as:

=FV(R1,P1,PMT(R1,N1,PV1,FV1,T1),PV1,T1) - FV(R1,P1-1,PMT(R1,N1,PV1,FV1,T1),PV1,T1)

where R1 is "rate", P1 is "per", N1 is "nper", PV1 is "pv", FV1 is "fv", and T1 is "type".

See the PV function help page for the mathematical formula that can be used to derive FV and PMT.  They are:

PMT = -(fv + pv*(1+rate)) * rate / ((1+rate)^nper - 1) / (1 + rate*type)

FV = -pv*(1+rate) - pmt*(1 + rate*type)*((1+rate)^nper - 1)/rate

-----

There is no direct formula for IRR(values,guess).  The IRR is the discount rate that causes the NPV to be zero, which must be derived iteratively.

See the NPV function help page for its mathematical formula.  The formula for NPV(rate,values) could be written as (in NPV1, for example):

=SUMPRODUCT(CF1:CF10/(1+R1)^(ROW(CF1:CF10)-ROW(CF1)+1))

where CF1:CF10 is "values", and R1 is "rate" (IRR), which is initially set to "guess" (default: 10%).

You could use Solver to vary ("by changing") R1 until NPV1 ("objective") is zero.

According to http://support.microsoft.com/kb/214105/en-us, the IRR function uses the Newton-Raphson method to derive "rate".  See http://en.wikipedia.org/wiki/Newton's_method.
