techfreak_ say:

Any older history logs would be contained in %appdata%\Skype in a main.db file located in a Skype name folder of any computer that ran classic Skype (7.x). You can simply search for main.db on your older hard drive and make a copy of the folder it is contained in. If the history is not damaged, it can contain the history you have had on Skype since you used it on your old computer.

To view or search this history:

1. You can use third party software like SkypeLogView(er) [nirsoft.net] to view the data contained in the main.db file.

2. Copy that folder onto your new computer and use the export feature in Skype for Windows Desktop 8.x to export that older history to a searchable .html file. You might have to recreate a Skype folder in the %appdata% folder on your new computer and copy the Skype name folder, so that the desktop software can detect that older history.

The ~2 year old history in the cloud can be exported and viewed in a web browser with the Skype Parser viewer.

1. Request an export of your ~2 year old history.

2. Wait a period of time like 20 minutes to an hour. Return to the export page and download your export request to your hard drive. This request will remain in your account for 30 days.

3. Download and use the Skype Parser viewer.

4. Unpack the exported file packed in .tar format. and the parser packed in .zip format to a location on your hard drive.

5. Double click on the index.html file from the unpacked Skype Parser viewer to view it in your default web browser.

6. On that web page, click on choose file and locate the messages.json file from your unpacked .tar file of your ~2 year old history.

7. On that same web page, press Load.

8. Browse through your ~2 year old history in  your web browser.

Skype FAQ on this process:

How do I export my Skype files and chat history?
