The key reason for PDFs is to make the text not editable. To make the page display constant on all machines.

You have a few options.

If person needs to be able to update documents in a folder, then grant them access rights to upload to that sharepoint folder. 

If you don't want to do that, you have to come up with some sort of "Change control process". Some way for people to request, suggest, make changes that then have to be approved by a designated "owner" for the file.

Save a document (I called it "New Blank Flowchart.vsd") with all the default setup you'd like. Then put that file as the command line parameter to open Visio.  You can do that by right clicking on the icon used to start Visio (even via the Start menu item) and clicking properties.

Normally:

"C:\Program Files (x86)\Microsoft Office\Office12\VISIO.EXE"

Change to:

"C:\Program Files (x86)\Microsoft Office\Office12\VISIO.EXE" ...\Visio2007\New Blank Flowchart.vsd

of course you need to put your own complete path to the file (unless you place the file in your normal drawings file location in the "tools\options\advanced\file path" menu choice)
