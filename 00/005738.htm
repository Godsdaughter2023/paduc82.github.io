doug robbins - mvp office apps & services (word) say:

A macro containing the following code will delete a space before a paragraph mark (¶) in footnotes, but unfortunately, it will also delete any direct formatted that has been applied to the footnote text.

Dim i As Long
Dim rng As Range
With ActiveDocument
    For i = 1 To .Footnotes.Count
        With .Footnotes(i)
            Set rng = .Range.Duplicate
            rng.Collapse wdCollapseEnd
            rng.Start = rng.End - 1
            If Asc(rng.Text) = 32 Then
                .Range.Text = Left(.Range.Text, Len(.Range.Text) - 1)
            End If
        End With
    Next i
End With
