suzanne s. barnhill say:

Word 2013 contains a built-in template for this size of labels (Avery 3652). Proceed as follows:

  1. Select the Mailings tab.
  2. In the Create group, click Labels.
  3. In the Labels dialog, click Options...
  4. In the Label Options dialog, select "Avery A4/A5" as the label vendor.
  5. In the "Product number" list, scroll down to 3652. (If you click Details..., you will see that this is a 3x7 label setup.)
  6. Click OK.
  7. Back in the Labels dialog, click New Document. This will give you a sheet of labels.

Note that labels are set up as a table; when you click on the page, the contextual Table Tools will become available. If you don't see cell boundaries, select the Layout tab and, in the Table group, click on View Gridlines to turn it on. These gridlines do not print.

 

Once your template is set up, you should not need to adjust the margins or column widths. You can type content in the cells just as you would in any table, using Tab to go from one label to the next.

 

If you are wanting to create a template for a different size label, start with whatever is the closest layout to what you want, then click New Label... and set the margins, pitch, label height and width as required. When you save this label definition, it will be listed under "Other/Custom."

suzanne s. barnhill say:

That is the way these labels are laid out. If you look at the Details, you'll see that the top and side margins are set to 0". These labels are described as "Copier and Multifunction Labels." It is typical for copier labels to have no margins; they're not really designed for desktop printers.

 

What you can do is, on the Layout tab of the Table Tools, click Properties. On the Table tab of the Table Properties dialog, click Options... Set the default cell margins to an amount that will provide the required top and side margins for your labels. For a laser printer, this may be less than 0.2", but for an inkjet will probably be more (especially at the bottom). If you don't want all the labels to have the same top margin, you can adjust individual cell margins for selected cells by clicking Options... on the Cell tab and clearing the check box for "Same as the whole table."

graham mayor say:
There are many pre-programmed label templates in Word, both Avery formats and others. You need to input the correct label number for the format that you are using. The label format should then be correct. If it is not correct then there are several reasons why that should be so:

1. You have added a header/footer to the normal template (rename the normal template)
2. The page size set in the printer properties doesn't match the page size of the label sheet.
3. You have a scaling option set in the zoom section of the print dialog
4. Your printer has poor paper handling. Some printers will slip when faced with label stock and thus are incapable of feeding the label correctly. Cleaning the rollers in the print path may help.

3. Is the most commonly incorrect setting.

80 per sheet label formats from Avery include -
15267, 15667, 18167, 18667, 28667, 48267, 48467, 48867, 5167, 5267, 5667, 5967, 8167, 8667, 8867, 8927
