yangzheng_msft6300 say:

Hi Runa,

I understand that you want to completely disable clutter feature. There is no option to do this directly in Office 365 Admin Portal. However, you have two other ways to do this.
If you want to let end users disable this feature themselves, you can let then disable it in Office 365 OWA (Outlook Web App).
1. Login into Office 365 OWA.
2. Click the Settings Menu gear on the top right and select Options from the drop down.
3. Mail->Clutter, turn off the check box Separate items identified as clutter.

If you want to disable this feature for all users in your tenant as an admin, you can do this with PowerShell.
1. Connect to Exchange Online using PowerShell with an admin account.
2. Run the command below.
Get-Mailbox | Set-Clutter -Enable $false

After turning off the clutter feature, the clutter folder will still exist but no emails will enter that folder.

Regards,
Yang

lancecai_msft2205 say:

Hi Robert,

Yes, the PowerShell command disables the Clutter for existing Exchange Online mailboxes.

If a new account is created, which means a new Exchange Online mailbox is created, you will need to run the PowerShell command

“Set-Clutter -Identity "NewUser" -Enable $false” for disabling the Clutter for the new one.

Rolling Moving Average in Excel

I am looking for help on a rolling moving average in Excel.  A few specifics about my setup:

 

1.  Data is entered on a daily basis comparing the actual number with the forecast number.  The difference is placed starting in cell F2

2.  The oldest data starts at the top with the newest data at the bottom of the row.  As an example, if the range is F2:F1300, cell F1300 would have the newest data

3.  All cells in the display Column A have the following formula in it: =IF(A2>0,B2-C2," ").  The formula was entered to save me the time of having to copy and paste the formula in each day.  If there is no data, the cell in Column F will display a blank

4.  I would like to create a 5 day rolling average for the data in Column F.  As an example, there would be one cell in Excel that calculates the moving average number.  Each time a new data point is entered and the cell displays a numerical value, Excel will take the average for the last 5 entries and display the average.  The output for the 5 day rolling average could be in any cell, say H2, as an example.  This would be the only cell to show the rolling moving average.

5.  I have not been able to get any moving average formulas to work.  I suspect that the entry I am using, the " " for blank data, may be used in calculating the formula

 

Any help would be greatly appreciated.

 

Jeffrey

mike h.. say:

CharAbeuh,

 

Thank for your formula and approach.  The formula produces the same results.  What variables in the formula would I change if I wanted to have a rolling average for the last X days.  As an example, if I wanted to go from the lsat 5 days to the last 10 days, what variable(s) wouild I change?


To change that formula to that last 10 instead of 5 you must change the 5 (in 2 places) to a 10

 

 

=SUMPRODUCT( (ROW(AF3:AF236)>= LARGE(IF(ISNUMBER(AF3:AF236),ROW(AF3:AF236),""),10) ) *ISNUMBER(AF3:AF236),AF3:AF236 ) / 10

 

If you want you can hold the row number in a cell and here I've uased AJ1

 

=SUMPRODUCT( (ROW(AF3:AF236)>= LARGE(IF(ISNUMBER(AF3:AF236),ROW(AF3:AF236),""),AJ1) ) *ISNUMBER(AF3:AF236),AF3:AF236 ) / AJ1

 

Now you can ARRAY enter the formula and simply change the number In AJ1 for the number of rows you want.
