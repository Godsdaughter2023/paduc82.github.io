Using Pub 2010 if you draw an empty text box on the first page of a Publisher publication > Insert > Text group > insert file and browse to your Word file Publisher will convert the Word file, import it and automagically add as many pages as necessary to accommodate the Word file. All the text boxes will be linked and flow from page to page.

This also works if you want to add another Word file to a publication that already is 150 pages.

If you just try to paste a multiple page Word doc into a single text box then yes, you will have to manually link the overflow text into additional text boxes on additional pages. I can explain that in more detail if you want.

If Publisher freezes or crashes during the process it may be that your images are too high resolution. In that case compress all your images before you try to insert/import the Word doc. In general the newer version of Publisher tend to choke on large image files so compress them to no more than 300 dpi.

Another tip in case you did not know it, click in a text box > Text Box Tools/format tab/ribbon and right click on an empty space to the far right of the options > Customize the Ribbon > Choose commands > All commands and scroll down until you find 'Edit Story in Microsoft Word' and add it to the ribbon. You can put it anywhere but I find it makes sense on the text box tools tab ribbon. Now you can click within those linked text boxes > Edit Story... and the Word file that you just imported will open in Word for editing. Close Word and the changes are reflected in Publisher.

goirish1978 say:

Is there a way to change default settings of pivot table display to hide "show contextual tooltips"?

I know to go to options/display/--->uncheck "show contextual tooltips" - looking for a way to change the default setting so this box is unchecked instead of having to do every time I open a pivot table. 

shane devenshire say:

Hi,

To do what you want you need to consider a macro solutions.  Here is an example:

Private Sub Workbook_SheetPivotTableUpdate(ByVal Sh As Object, ByVal Target As PivotTable)
     Dim myTable As String
     myTable = Target
     ActiveSheet.PivotTables(myTable).DisplayContextTooltips = False
End Sub

1. To add this code to your file, press Alt+F11,
2.  In the VBAProject window, top left side, find the thisWorkbook object  under your file name and double click it.
3.  Paste in or type the code above.
