You don't need any code to do that.

 

Try using the text box expression:

 

  =IIf(fEstimatesBaseSumAdd.Form.Recordset.RecordCount>0, fEstimatesBaseSumAdd.Form!MatTot, 0)

 

or, if that doesn't do it, the less specific expression:

 

  =IIf(IsError(fEstimatesBaseSumAdd.Form!MatTot), 0, fEstimatesBaseSumAdd.Form!MatTot)
