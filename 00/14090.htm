Excel VBA to loop - copy and paste to different sheet

Hi,

I have a file with hundreds of worksheets. I have a Master Sheet on the first page with a list of a bunch of names organized in one column. I need to copy and paste each name into A1 of every sheet.

For example, the first name on the Master Sheet goes onto A1 of Sheet 1.  The second name on the Master Sheet goes onto A1 of Sheet 2, the third name onto A1 of Sheet 3, and so on.

Can you make a code to make this automated? Thanks!

sheeloo say:

Sub copyToAllSheets()

Dim inWS As Worksheet
Dim ws As Worksheet

Dim i As Long

Set inWS = Worksheets("Master Sheet")

i = 1
For Each ws In Worksheets
    If ws.Name <> "Master Sheet" Then
        ws.Range("A1") = inWS.Cells(i, 1)
        i = i + 1
    End If
Next ws

End Sub

Your master sheet should be named 'Master Sheet' or you can update the code to match your name.
