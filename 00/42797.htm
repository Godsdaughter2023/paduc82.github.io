a.       Click on Start and then Run.

d.      Confirm that you want to delete the files by clicking Yes on the Confirm Multiple File Delete window that opens.

f.        Now that all of the files and folders are selected, hit your Delete key or choose File and then Delete from the menu.

g.       Confirm that you want to delete the files by clicking Yes on the Confirm Multiple File Delete window that opens.

h.      After all of the files have been deleted you can close the window and empty your Recycle Bin, permanently removing the files from your PC.

For Code 28 you need to Uninstall and reinstall the driver
 
In order to troubleshoot this issue you may goahead and uninstall the drivers manually and then reinstall the drivers and check for the issue.
Uninstall and reinstall the driver for the hard drive from device manager. To do that follow the below provided steps:
 
i.     Click Start, type devmgmt.msc in search box and hit enter.
ii.    On the “Device Manager” window, Select the device
iii.   Right Click on the device and click uninstall
iv.   Restart the computer once the driver is uninstalled.
v.    Install the latest drivers from the manufacturer and check if it works
 
Also you may follow the link provided below for further information.
http://windows.microsoft.com/en-us/windows7/Update-a-driver-for-hardware-that-isnt-working-properly
 
If you could post back the details( make and model ) of your laptop, then it will be helpful for us to send you the details of the drivers present for your laptop.
 
As a workaround in solving the problem you can follow the steps provided below.
 
1. Click Start, type Sound in the Start Search box.
2. Click Sound in the search results at the top of the Start menu.
3. Click the Playback tab. Notice the check mark next to the default device. For most users, the default device is Speakers. Make sure that your audio settings are configured as expected.
Follow these steps:
 
On the Playback tab, select the playback device, click Properties, click the Advanced tab, and make sure that the Default Format is set to the value that you expect. When you finish, close this window to return to the previous window.
Make sure that the default sound device is set correctly for the sound device that you are using. To set the default device, follow these steps:
 
On the Playback tab, select the playback device, click Configure, and then make sure the configurations are set correctly.
If the sound settings are set correctly, and your speakers are already set as the default device.
Select the sound device that you want as the default sound device, and then click Set Default.
Click OK to save your settings and to close the Sound dialog box.
 
If the above provided solution doesn’t resolve the issue then you need to update the latest audio drivers from the manufacturer’s website and check for the issue.
 
Also you need to check whether the Windows Audio Service are up and running in the services.
To do that follow the steps provided below.
 
Click start.
Type in services.msc in the search field and hit enter.
Right click on the service
If the service is stopped then Start the service
To start the service right click and select Properties
And in the Start-up type select Automatic
 
If the above steps doesn’t resolve the issue then you need to follow the steps provided below.
 
Click on start and type in troubleshooting and hit enter.
Now under hardware and sound  click on troubleshoot audio playback issue.
Now follow the wizard to trouble shoot the issue.

jessen p say:

Hi Austin Monroe,

Welcome to Microsoft Answers!!!

Adding to the inforamtion by Jetta48, you may follow the methods below to enable the thumbnail view;

Step 1:

1) Click the Organize button on any folder, and choose Folder and Search Options from the menu.

 

2) Click the View tab, and then uncheck the “Always show icons, never thumbnails checkbox”.

(The option is checked in the above figure; make sure you uncheck the box)

Step 2:

To view thumbnails of your pictures
 
1. Open the Pictures library by clicking the Start button, and then clicking Pictures.

2. In the left pane, click the folder that contains the pictures you want to view.

3. Click the arrow next to the Views button, and then move the slider to change the size of the thumbnails.

Access the link below for more information on thumbnail in Windows 7.
