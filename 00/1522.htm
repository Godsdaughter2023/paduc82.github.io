jay freedman say:

Look at the folder C:\Program Files (x86)\Microsoft Office\root\Office16\PROOF, where you should find a series of files whose names start with "msth" (two files for each language installed on your computer). The one with the extension .LEX is the thesaurus, and the one with the extension .DLL is the code that makes it usable in Word.

If the files are there, then the problem is probably an incorrect set of registry values that should identify the DLL. That can be fixed by running a "quick" repair of Office. If they're missing, the online repair should install them.
