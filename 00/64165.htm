john ray_m. say:

Hi IeszerEscalante,

 

The cause of the black borders in your movie is due to the dimensions (Width and Height) of the video that you used not fitting the aspect ratio/s (4:3 and 16:9) required by Windows Live Movie Maker 2011. The program normally puts these "black borders" in order to compensate for the missing dimensions of the video used with respect to the aspect ratio that you are applying for the movie.

 

Since the file is an anime video, we would suggest finding a third party application that would help you adjust the file's dimensions respectively to fit the aspect ratio being asked for by Windows Live Movie Maker. You may also use online aspect ratio calculators in order to get the correct dimensions for the video in order to correctly fit it in the preview screen as well as output displays once the project file has been rendered as a movie.

 

  • Example: for Digital Videos - in order to get a 4:3 aspect ratio in Windows Live Movie Maker, the dimensions should be Width = 720 pixels and Height = 540 Pixels while for a 16:9 aspect ratio movie the dimensions should be Width =720 Pixels Height = 405 Pixels. Again, these numbers are for Digital videos and the case may be different if you were to use an "Animated Video".

 

Hope this gives you an insight of the issue at hand.

 

 

Thanks!

720 x 480 video has a native aspect ratio of 3:2 so there need to be flags in the file to tell an editor or playback program whether that is 4:3 video or 16:9 video.  Somehow those flags are either set wrong or are being interpreted incorrectly by Movie Maker.
