ISO Week Numbering with Start of Year not set to Jan 01. - Microsoft Community

kala sha jahan say:

ISO Week Numbering with Start of Year not set to Jan 01.

Is there a way to use an ISO Week Numbering system but not have Jan 01 as the start of the year?  It would be somewhat a combination of Chip Pearson's (Week Numbers in Excel) Week Number From Date + Week Number ISO.

 

Issue:

We have our fiscal year end on August 31 every year, so our new year starts Sept 1.  We number our weeks starting September and use stats for comparison purposes year over year, but I'd like to have a set standard determined by a function using the ISO Standard to eliminate issues as to what week to compare the data to

 

Criteria or Needs:
The first day of the week is specified by the International Standards Organization (ISO) definition (Monday to Sunday)

  • The first week is a specific date, and always begins on September 1 of the year, but Week 1 begins on the first Monday after the first Thursday of the year..., so:
  • In 2013, August 26th to September 1st would have been Week 53 AND September 2nd to September 8th would be Week 1.  In reality, there would be several Week 53 for several years in a row

     

    Is this possible with a function?

     

    KSJahan

     

     

    • hansv mvp say:

    According to my calculations, Aug 26, 2013 to Sep 1, 2013 was week 52, not 53.

    Try this custom VBA function:

     

    Public Function IsoWeekNumberExt(d1 As Date, m As Long) As Integer
        Dim y As Long
        Dim n As Long
        Dim d2 As Date
        d2 = d1 - Weekday(d1 - 1) + 4
        y = Year(d2)
        n = Month(d2)
        If n < m Then
            y = y - 1
        End If
        d2 = DateSerial(y, m, 3)
        IsoWeekNumberExt = Int((d1 - d2 + Weekday(d2) + 5) / 7)
    End Function

     

    With a date in cell A1, use the formula

     

    =IsoWeekNumberExt(A1,9)

     

    to find the week number with September (month #9) as starting month.

    hansv mvp say:

    And if you wish a non-VBA solution:

     

    =INT((A1-(DATE(YEAR(A1-WEEKDAY(A1-1)+4)-(MONTH(A1-WEEKDAY(A1-1)+4)<9),9,3))+WEEKDAY(DATE(YEAR(A1-WEEKDAY(A1-1)+4)-(MONTH(A1-WEEKDAY(A1-1)+4)<9),9,3))+5)/7)
