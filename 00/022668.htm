OneNote Options - Save & Backup - Microsoft Community

randy minder say:

OneNote Options - Save & Backup

I would like to modify where OneNote 2010 saves all its Notebooks. However, I am baffled by how the Default Notebook Location, Backup Folder and Unfiled Notes Section works.

I have four different notebooks and I would like to save them as follows:

 

Notebook1:  C:\OneNote2010\Notebook1

Notebook2:  C:\OneNote2010\Notebook2

Notebook3:  C:\OneNote2010\Notebook3

Notebook4:  C:\OneNote2010\Notebook4

Is this possible? When I try to set the Default Notebook Location to one of these locations I get errors, even though the folders exist.

 

Thanks

 

 
callkathy say:

I don't know that what you want to change is the default notebook location.

The default notebook location is the folder you want to hold all new notebooks by default. In your case, you want to set it to C:\OneNote2010

The backup folder is where you want OneNote to put the automatic back up copies of your notes. OneNote runs this back up automatically on a schedule you can adjust. I would set that folder to be some drive and notebook combination that isn't on your main drive.

The Unfiled Notes section points to a section where you want to put things that don't go anywhere else. In your case, you probably want to set it to the same folder as you are using for your default location. The difference here is that you are creating a section in that folder, not telling OneNote to put things in the folder.

What I would do is:

  • Go to File--> Options--> Save & Backup
  • Click Default Notebook Location in the list at the top of the screen.
  • Click Modify
  • Change the location to C:\OneNote2010 and click select
  • Click Unfiled Notes Section in the list at the top of your screen
  • Click Modify
  • Change the location to C:\OneNote2010 (or verify that it is already set there) and click Open
  • Click Backup Folder in the list at the top of your screeo
  • Click Modify
  • Navigate to where you want your backups put and click select

You will then be set the way you want. As one final step, I recommend you click Back Up All Notebooks Now. (This way you know the backups are set correctly and you have a valid backup for your notes.)

Hope you find this helpful!
