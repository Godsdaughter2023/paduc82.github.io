  I began to think of iterative change:
  Selection.ColumnWidth = ...

P. S. Thank you in advance

andreas killer say:
Quite accurately, this is not possible, because there are 2 problems, but we can use a trick to get nearly the same.

If we want to get the exact size of a paper, we have to use the windows API... a lot of code. But for the most common papersizes it's enough to convert the PaperSize constant of the PageSetup object to a width.

The Pagesetup object also let us know how large the margins are, we can subtract them from the width.

But we can not use the whole width for printing, every printer has a unprintable range.

Set the left margin to 0, make a border around cell A1 and switch to the print preview. As you see there is a slight margin on the left side.

I have no idea how to determine the size of this margin and that's the first problem, we have to guess how large it is.

The second problem is that the width of a column on the screen must fit the pixel resolution, so if we set the width to 3.1415926 Excel scales the width to 3.14 (or whatever). And your printer has a different resolution as your screen.

So we can use a loop only to set the width approximately, not more.

After we have done all this, we can tell Excel to scale the output to fit the width and ignore the height.

See the code below for an example.

Andreas.


Option Explicit

Sub ScaleSelectionToFitPaperSize()
  Dim PaperWidth As Double, RangeWidth As Double, SelWidth As Double
  Dim R As Range

  With ActiveSheet.PageSetup
    'Get the width of the paper in centimeters
    PaperWidth = GetPaperSize(.PaperSize)(0) / 10
    'Subtract the unprintable margin
    PaperWidth = PaperWidth - 0.9
    'Scale to points
    PaperWidth = PaperWidth * Application.CentimetersToPoints(1)
    'Subtract the margins
    PaperWidth = PaperWidth - (.LeftMargin + .RightMargin)
  End With

  'Calculate the width for each column
  RangeWidth = PaperWidth / Selection.Columns.Count
 
  'Install an errorhandler, maybe we struggle into an endless loop
  Application.EnableCancelKey = xlErrorHandler
  On Error GoTo ErrorHandler

  'Scale the columns until we fit
  Do
    SelWidth = Selection.Width
    For Each R In Selection
      R.ColumnWidth = R.ColumnWidth / R.Width * RangeWidth
    Next
  Loop Until SelWidth = Selection.Width
 
  'Let Excel scale the print to fit the width
  With ActiveSheet.PageSetup
    .FitToPagesWide = 1
    .FitToPagesTall = False
  End With
ErrorHandler:
End Sub
