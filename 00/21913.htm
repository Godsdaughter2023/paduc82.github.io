john korchok say:
Word for Mac lacks many settings that the Windows version has. There is no Trust Center, and thus no Trust Center Settings, where the Windows Privacy options are found.

john korchok say:

This macro (originally from Doug Robbins) works in Word for both Mac and Windows:

Sub AutoOpen()
  With ActiveDocument
    If .ProtectionType = wdAllowOnlyRevisions Then
      .Unprotect
    End If
    .TrackRevisions = False
    .ShowRevisions = False
  End With
End Sub

Add it to your Normal.dotm file to display all documents with no markup.
