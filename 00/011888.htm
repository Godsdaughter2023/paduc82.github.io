messages sent from outlook (linked to yahoo mail) do not show - Microsoft Community

ramesh2011 say:

messages sent from outlook (linked to yahoo mail) do not show up in "Sent" folder in yahoo webmail - how to fix?

I have set up outlook 2010 & successfully linked yahoo mail & gmail to it.  However when I send messages from outlokk (with yahoo mail) these messages do not appear in the sent message folder when I then access yahoo mail via the web.  Vice versa also - ie if I use a browser to access yahoo mail & send a message - it does not show up later in outlook - how to fix?
brian tillman [outlook mvp 2007-2019] say:

If you are accessing the Yahoo mailbox using a POP account, there is no way to achieve what you want.  POP mail has no concept of folders and it's a one way path from the Inbox on the server to the Inbox on the client.  Outbound mail is SMTP and never stays on the server, but gets routed to the next hop in the path between the mail client and the recipient.

If you are accessing Yahoo via IMAP, you can tell Outlook to store your outgoing messages in the Sent Items folder on the server by opening your account settings (Tools>Account Settings), selecting your account, clicking Change, then More Settings, selecting the Folders tab and choosing the server's Sent Items folder.  See this for pictures: http://www.howtogeek.com/howto/microsoft-office/use-gmail-imap-in-microsoft-outlook-2007/.  It's for gmail, but any IMAP account wll work the same way.
