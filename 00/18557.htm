daniel pineault say:

Your primary issue is the fact that you do not use the criteria part of the DLookUp().   So in fact, you will always pull the same value every time.

You need to specify something along the lines of:

UserName = Environ("USERNAME")

UserId = DLookup("[USERID]", "tblUsers", "[username]='" & UserName & "'")

UserLevel = DLookup("UserLevel", "tblUsers", [username]='" & UserName & "'")

...

That said, normally you use Environ("USERNAME") (or an api that does the same - then Environ can be hacked so i prefer the api approach) to get the network username, with that you get that user's userid within your db, then you use the userid to determine and apply their rights and if the user is not located, bit then put off your application.

jim thomlinson say:

Sub test()
Dim rng As Range
Dim wks As Worksheet

For Each rng In Sheets("menu").Range("B1:B30")
    If Trim(rng.Value) <> "" Then
        On Error Resume Next
        Set wks = Nothing
        Set wks = Sheets(rng.Value)
        On Error GoTo 0
        If wks Is Nothing Then
            MsgBox "Sheet " & rng.Value & " does not exist"
        Else
            wks.PrintOut
        End If
    End If
Next rng
End Sub
