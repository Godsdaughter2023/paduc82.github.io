ronrosenfeld say: It would be much simpler if you specified that the time be entered in a manner that Excel will interpret as a time.  Since that is not the case, one way is to convert it to time, subtract, then output it in your specified format.

Assuming the times will always be on the same day, and that 1st time is in A1, 2nd time in A2:

=HOUR(TEXT(A2,"00\:00")-TEXT(A1,"00\:00")) & " hours " & MINUTE(TEXT(A2,"00\:00")-TEXT(A1,"00\:00")) & " minutes"

If your time separator is not the colon, you could use:

=SUBSTITUTE(TEXT(DOLLARFR(DOLLARDE(A2/100,60)-DOLLARDE(A1/100,60),60),"0.00"),"."," hours ") & " minutes"


However, if you are going to have a bunch of these, and you want to add them up, it will be much simpler to input the times as:

7:30
15:45

(or input them as you have above, and use the TEXT formula to convert them to "Excel times" in a helper column; then do your math).

Then the result is merely A2-A1, formatted to show as 8:15

bob jones aka: cybertaz say:
The default location for Office 2008 Custom Dictionaries is;

~:UserName:Library:Preferences:Microsoft:Office 2008

But custom dictionary files can actually be stored anywhere you choose. [Note: If using OS X 10.7 or later the user's Library folder is a Hidden folder.]

The easiest way to make a backup, though, is to go to Word> Preferences> Spelling & Grammar then:
  1. Click the Dictionaries button
  2. Select the Custom Dictionary you want to backup
  3. Click the Edit button
  4. The file will open in Word
  5. Use Save As to save a copy in the location of your choice
  6. Be sure to select Speller Custom Dictionary (.dic) as the file type (You can save it as a Text File .txt if you prefer & still be able to use it later.)
