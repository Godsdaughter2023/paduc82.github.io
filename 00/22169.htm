joeu2004 say:

Your interpretation is correct:  if the second parameter for COUNTIF can be interpreted as numeric (includes dates), it is treated as numeric, as are any values found in the first-parameter range.

Work-around:  COUNTIF(A1:A2, H1&"*") .

PS.... But I would use SUMPRODUCT.  Even though it is less efficient, I feel more comfortable knowing that it is matching exactly what I expect.  Of course, H1&"*" might match any longer string of digits, too.

But yes, in this scenario, as Joeu pointed SUMPRODUCT is better.

Hi David,

 

What happens when you try to enable the cookies?

 

A cookie is a piece of text that a Web server can store on a user’s hard disk. When the same website is browsed again by the user in the future, the data stored in the cookie can be retrieved by the website to list the previous activities of the user.

 

Method 1:

 

Try to enable cookies by following the suggestion provided:


a. Open the Internet Explorer by clicking Internet Explorer tile on the Start Screen.
b. Next, Internet Explorer opens. Click on the Tools and select Internet Options.
c. Next,click on the Privacy tab.
d. Next, click on the Advanced.
e. Next, check / tick ‘Override automatic cookie handling’. Here you can Allow or Block Cookies.

 

Recommendations: Block third-party cookies, Accept first-party cookies and always allow session cookies.

 

You can also refer to the suggestion by Scott .. B from the following link and check if it helps.

 

http://answers.microsoft.com/en-us/ie/forum/ie10-windows_8/how-do-i-enable-cookies-on-windows-8/d8129d77-d048-4248-97a6-98fbf154acc4

(Note: This applies to Internet Explorer 11 also.)

 

Method 2:

 

If the issue persists, I suggest you to save the emails as HTML format and check if it helps. You can try the following steps to do so.

 

  1. Open the website in desktop version of Internet Explorer.
  2. Click ok “File”.
  3. Click on “SaveAs”.
  4. While saving select the “Save as type” as “webpage, complete(*.htm,*.html)”.
  5. Try to open the email after that and check if it helps.

 

Hope this helps.

To use the Reset Internet Explorer Settings feature from Control Panel, follow these steps:
