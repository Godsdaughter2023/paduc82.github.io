The location of the Automatic Maintenance settings actually did change in Windows 10:

So this is where you have to go now to change the schedule for the Windows Defender Scheduled Scan task: Right-click on the Start button and choose Control Panel > Security and Maintenance > Maintenance > Change maintenance settings.

It’s been 3 years now, and there are still way too many people (including Microsoft documentation authors) who just can’t quite seem to get a grip on how this works. The fact that the controls for the scheduled scan went missing in the new Windows Defender user interface doesn’t mean that the feature was dropped and that you’ve been left to your own devices – it means that the scheduled scan was reengineered so that it now runs reliably without any user configuration. Using IF Function in Word - Microsoft Community

kslyons say:

Using IF Function in Word

I am trying to create an IF function in Word to add a header and footer if I enter paperless in the ASK field. Is this possible?
macropod say:

An ASK field simply creates a bookmark. To use one to create conditional output, you could use a field coded as:

{QUOTE{ASK Paperless "Paperless output: Yes/No."}{IF{REF Paperless \* Caps}= "Y*" "Content for 'Yes'" "Content for 'No'"}}

A field constructed that way might be used in the header. For the footer, you'd use just:

{IF{REF Paperless \* Caps}= "Y*" "Content for 'Yes'" "Content for 'No'"}

With such a pair of field constructions, the header would update immediately, according to the Yes/No response (only 'Y' or 'y' is tested, so that's all one actually needs to supply; anything other than something beginning with 'Y' or 'y' is interpreted as a 'no'). That is achieved via the QUOTE field. Since you want to update the footer as well, and you can't incorporate that into the QUOTE field, the footer would have to be updated separately (e.g. via Print Preview, with Word's 'update fields before printing' option checked).

Note: The field brace pairs (i.e. '{ }') for the above examples are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via any of the standard Word dialogues. The spaces represented in the field constructions are all required.
