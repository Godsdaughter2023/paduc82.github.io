ken sheridan say: It might not make any difference to the problem of the extra row in Excel, but your query has a HAVING clause when it should have a WHERE clause.  Also your ORDER BY clause seems illogical.  At present it is sorting alphabetically, so the rows are not being returned in chronological order.  Change it as follows:

SELECT EmpInfoT.Shift, Format([ProdDate],"mmm/yyyy") AS [Month], Avg(EmpProdT.Attainment) AS AvgOfAttainment
FROM EmpInfoT INNER JOIN EmpProdT ON EmpInfoT.[Emp#] = EmpProdT.[Emp#]
WHERE EmpInfoT.Shift = "6"
GROUP BY EmpInfoT.Shift, Format([ProdDate],"mmm/yyyy"),Format([ProdDate],"yyyymm")
ORDER BY Format([ProdDate],"yyyymm");
