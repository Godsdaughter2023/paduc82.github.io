Please look closely the output of USB viewer. The mass storage device is showing up in a port that is  not marked SS. Or it could also be that the OS knows that this is bad USB3 device and forces it to be 2.0 device. Post the VID/PID of the device.

lemp say:

When your computer runs well in Safe Mode but poorly in normal mode, that means that something that runs in normal mode only is causing the problem.  You can determine what that is by repeated experimentation.  This is a bit tedious, but it should enable you to get to the root of the problem.

Start by disabling all non-Microsoft services:

  1. Click Start, type msconfig into the Search box, and press Enter
  2. Select the Services tab
  3. Check the box at the lower left to "Hide all Microsoft services"
  4. Click "Disable all"
  5. Click OK
  6. Restart (not in Safe Mode).

If the problem is solved, then the cause is one of those non-Microsoft services.  Narrow it down by returning to msconfig and check (enable) half of the non-Microsoft services.  Restart.  If the problem is still solved, then the cause is one of the non-Microsoft services that is still disabled.  If the problem recurs, then the cause is one of the non-Microsoft services that you just re-enabled.  Repeat this method until you determine the service that cause the problem.

If the problem is not solved when all non-Microsoft services were disabled, then the cause is not a service.  Return to msconfig and re-enable all services, including all non-Microsoft services.  Then select the Startup tab.  Disable half of the items and restart.  Repeat this method until you determine which item causes the problem.
