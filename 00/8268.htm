By the way, rename the Insider partition "Insider" now so you don't wind up with two volumes named Windows 10 on the dual boot menu.  You can download EasyBCD2.2 and install it in either partition.  It will let you easily rename partitions, set boot delay, change default boot loader and cook eggs.  It will survive the upgrade to 10 so go ahead and put it in 7 now.  It is free and I put it in both installations on dual boot setups.
