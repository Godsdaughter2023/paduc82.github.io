Suggestions for improvements - Microsoft Community

sbrax say:

Suggestions for improvements

I am a researcher and have tried to improve my use of OneNote and it lacks a few basic functions that would be really handy.

1. Combined search function

I like to tag things based on what they are, for instance a "definition". Then, I would like to search for definitions using combined queries, like like "services (word) + concept (tag)". I would then get a list of all definitions for services I have gathered from literature. I cannot believe this is not possible in OneNote. This would allow researchers and students to create a wiki of their literature notes.

2. Catalogs and overview pages

I would like to add organized catalogs or overviews to my notes. The folders can get really packed. It is typical that a page may be logical to put in one folder but also relevant in another one - assuming we like to reuse the information we collect. Making organized catalogs would be nice. Maybe a MindManager map as an overview page for a folder. Consider this as going through your notebook and sorting things the way you like to organize them for a specific purpose, and not starting all research from zero when you begin a new project.

3. Linking and tagging to a phrase

Using tags I noted the tag is always for a paragraph. A more sophisticated use of the tags function requires that I could tag specific words. Now the search pane shows the beginning of paragraph, so if I search for definitions I can only find what I want if the word started the tagged sentence.

4. Highlight colors

The highlight colors are too dark - if you use them you cannot see the text. Only a fraction of the palette is actually useful. Pastel colors please.

5. Handwriting

The assumption is that people want to convert their handwriting into printed letters. I would like to see a handwriting style option available to make my e-notes to look more human.
