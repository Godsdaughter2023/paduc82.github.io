Conditional formatting based on whether cell has a formula or not - Microsoft Community

jimsprime say:

Conditional formatting based on whether cell has a formula or not

I was goung to ask the question how can one determine the type of content in a cell ? .   So, if I understand answers to similar questions correctly, there is no built in way to determine what a cell contains - number, formula, text.....  Correct?  You have to create your own code.

hansv mvp say:

Excel 2013 and later have a new function ISFORMULA.

=ISFORMULA(A1) returns TRUE if A1 contains a formula, FALSE if not.

In Excel 2010, you can create a custom function for this purpose:

Function IsAFormula(rng As Range)
    IsAFormula = rng.HasFormula
End Function

Use like this in a cell formula or in a conditional formatting formula:

=IsAFormula(A1)

All versions of Excel have functions ISNUMBER and ISTEXT. You can combine this  with IsAFormula:

=AND(NOT(IsAFormula(A1)),ISNUMBER(A1))

returns TRUE if A1 contains a constant number (not a formula that returns a number)

=AND(NOT(IsAFormula(A1)),ISTEXT(A1))

returns TRUE if A1 contains a constant text string (not a formula that returns text)

Hi Mike,
If you use AutoArchive, the mail is moved into a pst file and it is only available on the computer where the pst is. ('m pretty sure Autoarchive doesn't work with shared mailboxes but i can't verify because auto archive is disabled in my accounts as we use online archives.)
