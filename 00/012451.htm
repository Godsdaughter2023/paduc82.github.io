shane devenshire say:

You probably will need a macro because you want it to automatically update.  Since you did not show us what the pivot table looks like it is hard to know what to do.  Below I suggest you explore another possibility, which also may not suit your needs:

A:  What you have above look exactly like a pivot table, except maybe your pivot table is in Compact form.  If you want to allow the flat table to update when the pivot table updates, and if pivot table looks like the above say with Jan in C1 and Group1 in A3 try this and see if you get what you want

1. With the cursor in the pivot table choose PivotTable Tools, open Options drop down and make sure Generate GetPivotData is on.

2.  Move to a cell where you want the above "flat table" and type = and then click on A3.  This will build a simple formula =A3.  Repeat this for all the row and column labels.

3. Next move to the cell where 1000 for Jan would be located in the "flat table" and type = and click on C3 (1000) and press enter.  You will get a formula like

=GETPIVOTDATA("Name",$J$1,"Name","Name4","Location","Location2")

Of course it won't be the same.  Everything in this formula is hard coded but the can be set to cell references.  You will need to experiment with the GETPIVOTDATA formula and see if this kind of setup will suit your needs.

B:  One suggestion has been to double-click the bottom right corner of the pivot table - the grand total of rows and columns which will output a copy of the raw data to a new sheet.  If you can do that (it depends on where your source is located) then you can design formulas to create any desired layout.  It could be a lot of work.

C:  You can select the entire pivot table and copy it, move to a new location and use Paste, Paste as Value to create a static copy of the pivot table. 

 

If this answer solves your problem, please check, Mark as Answered. If this answer helps, please click the Vote as Helpful button. Cheers Shane Devenshire
