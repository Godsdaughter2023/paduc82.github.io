Can I use a drop down list box where you can choose more than one - Microsoft Community

teachinginmn say:

Can I use a drop down list box where you can choose more than one choice

I am making a fillable form and need to have a drop list box where you can pick more than one choice to be listed.  Is this possible?  How do you do it?
jay freedman say:
In Word, it isn't possible with any of the objects (form field, ActiveX control, or content control) that you can put into the body of a document. The only place you can get multiple selection is in a list box placed on a userform (a custom dialog box).

For information about programming a userform, read http://gregmaxey.mvps.org/Create_and_employ_a_UserForm.htm.

When you put a list box on the userform, go to its Properties panel and set the MultiSelect property to either fmMultiSelectMulti or fmMultiSelectExtended. (The "extended" setting acts more like what most people expect, where you can use the Shift and Ctrl keys to add and remove selections.)

The other part you'll need to know about is how to determine which items in the list were selected by the user. In the OK button procedure (usually) this involves writing a loop that steps through the entire list, checking the .Selected property of each item and writing that item back to the document only if it was selected. As a very simple example,

Private Sub CommandButton1_Click()
   Dim i As Integer
   Dim result As String
   For i = 0 To ListBox1.ListCount - 1
      If ListBox1.Selected(i) Then
        result = result & ListBox1.List(i) & vbCr
      End If
   Next
   ' write result back to document
End Sub

jeeped say:
You need to prefix the asterisk with a tilde (e.g. ~) character to force Excel to explicitly look for an asterisk and not to treat the asterisk as a wildcard.

rng = Range("A:A").Find(What:="~*TEST", After:=Range("A:A").Cells( Range("A:A").Cells.Count), _
                                    LookIn:=xlValues, LookAt:=xlPart, MatchCase:=False)

This is essentially the same syntax adjustment you would make with a Ctrl+F Find within an Excel worksheet.
