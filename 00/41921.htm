Hello,

Windows 10 has a feature that is called File History to back up your data, it must be enabled to maintain a backup of all the files in their libraries. File History only works per user, since creating, modifying and deleting files can only be saved in the file history of a certain user account. According to your description, your NAS drive seemed to be using this function already.

We understand that you want to backup files from all your user accounts to a specific Admin account. However, merging the backup files on a specific user account is not yet available in Windows 10.
