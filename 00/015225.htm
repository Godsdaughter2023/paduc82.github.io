Hi Breno,

You can use the following Office 365 PowerShell command:

Get-MsolUser | Where-Object {$_.isLicensed -eq $true -and $_.Licenses[0].ServiceStatus[8].ProvisioningStatus -ne "Disabled"}

Note: Before that, please make sure you have installed the required software.

The value of the Licenses[0].ServiceStatus[8].ProvisioningStatus property is not equal to (-ne) Disabled, either PendingInput or Success.

Licenses[0] would represent the first licensing plan, and Licenses[1] would represent the second licensing plan.

ServiceStatus[8] represents the index number for SharePoint Online in Office 365 Enterprise E3 plan. (Updated on 5/18/2016)

PowerPivot / Data Modelling
Power Pivot is a data modeling technology that lets you create data models, establish relationships, and create calculations. With Power Pivot you can work with large data sets, build extensive relationships, and create complex (or simple) calculations, all in a high-performance environment, and all within the familiar experience of Excel. In a nutshell, it allows you to use multiple data sources. Then you could import, merge and perform analysis on the resulting data.  Excel can “natively” only store 1 million rows of data.         Application.DisplayAlerts = True

       Set N = Workbooks.Add(xlWBATWorksheet)

        Ns.Name = MySheet.Name 'AlphaNumericOnly(UListValue.Value)
        Cells.EntireColumn.AutoFit
        N.SaveAs Filename:=Application.ThisWorkbook.Path & "\" & AlphaNumericOnly(UListValue.Value) ', FileFormat:=xlExcel12 _
        , CreateBackup:=False
        N.Close False

You should be able to bulk convert the values back to true numbers is you select all of column A and use Text to Columns from the Data tab's Data Tools group.
