jim gordon mvp say:

The problem is that Word is not a page layout program and doesn't really have the concept of a page. Instead, Word is a word processing program where text flows within margins and around (or over or under or through) objects within the text.

There are two primary reasons for page layouts not being identical from one computer  to another, regardless of whether the computers are Macs or PCs.

The first is the Page Setup settings. When you open a document, Word checks those. If a specific brand and model printer has been chosen and that brand and model printer is not the default on the system you opened the document on, then Word re-flows all the text. This can be mitigated somewhat by using Page Setup and format the document for Any Printer.

The other common problem is font substitution. If any font within the document is not available on the current machine, Word will substitute a font it thinks might be close to the requested font. Yes, everyone might have Arial, but do you have the same brand, model and revision number of Arial? If not, there will be differences in how the document is displayed.
