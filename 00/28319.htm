tasos_k say: Hi,

Let's say that in Sheet1 (List1), the emails are in column C,

and in Sheet2 (List2) are in column A.

Do you want to delete the emails (entire rows) in Sheet2,

if  there are in sheet1 also?

if so, try this..


1) Pres ALT+F11 to open VB editor.

2) Insert > module and paste the code below on the right

3) Close VB editor.


Sub abcd()
Dim ws1 As Worksheet, ws2 As Worksheet
Dim str1 As String, str2 As String
Dim r1 As Long, r2 As Long
Set ws1 = Sheets("Sheet1")
Set ws2 = Sheets("Sheet2")
str1 = "C"
str2 = "A"
r1 = ws1.Cells(Rows.Count, str1).End(xlUp).Row
r2 = ws2.Cells(Rows.Count, str2).End(xlUp).Row
For X = 1 To r1
For Each r In ws2.Range(str2 & "1:" & str2 & r2)
If r.Value = ws1.Cells(X, str1).Value Then ws2.Rows(r.Row).Delete
Next r
Next X
End Sub


4) To run the macro, press ALT+F8, 

5) select <abcd> from the list and click the run button.


(change columns A and B and Sheets1,2 as needed)

rgeoffhunt say:

Discontinuity in BESSELK(x,0) at argument x=2

I used the Excel 2010 modified Bessel function of the second kind and zero order BESSELK(x,0) as part of a back-check that a solution to a heat conductivity problem correctly solved the original equation (i.e. the equation balanced). The balance was good except in the neighbourhood of a point which turned out to correspond to x=2. There appears to be a discontinuity in BESSELK(x,0) at x=2. This can be seen by plotting BESSELK(x,0) against x for values of x at intervals of (say) 1.0E-8 between (say) (2.0-1.0E-7) and (2.0+1.0E-7). There is a clear step at x=2.0. BESSELK changes by -1.399E-09 for each 1.0E-8 increment in x in this region, except for the increment between x=2.0 and x=2.0+1.0E-8 where BESSELK changes by -8.976E-09.

I understand that this is a very small error relative to the value of BESSELK(x,0) at this point (about 0.1139) but it does lead to trouble in calculating differentials of BESSELK with respect to x.

Is this already a well-known problem? Are there any plans to fix it?

My check required only BESSELI(x,0) and BESSELK(x,0) so I have not looked for discontinuities in the BESSELK function for orders other than 0, or in other members of the BESSELx family!
