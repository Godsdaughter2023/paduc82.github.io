mauricedurbin say:

Taskbar and Start Menu settings – a message to Microsoft

     

You have been tinkering with the Taskbar and Start Menu for many weeks, no doubt influenced by the many and varied requests from Insiders, but for each new tweak you seem to add a new issue. For example you have introduced the choice of Search Icon, Search Box or Search Disabled on the taskbar (GOOD – that’s been asked for!), but at the same time made the Start Menu depend on the Taskbar for its Search capability forgetting that if the Taskbar has search disabled then so will the Start Menu… NOT what is needed!

 

I have seen many people comment adversely about the Start Menu; the traditionalists (who love Windows 7) hate its modern attributes, the modernists (the increasing number of Windows 8.1 fans) hate its ‘classic’ attributes; they see the left hand side as a waste of valuable screen space, and the ‘All Apps’ format as old fashioned and cumbersome.

 

Far from being ‘all things to all men’ the start menu divides opinion as to why it is bad while uniting people in thinking it is not right. Simply prettying it up won’t fix the problem. The biggest problem is that what one person demands another person rejects; you can only solve that by allowing more customisation options.

 

Isn’t it time to take stock and start again with some clear objectives?

 

Here’s some thoughts based on what I have seen people asking for.

 

  1. Allow a choice of Search Icon, Search Box or None on the Taskbar.           DONE that, keep it!

  2. Place a Search Icon within the Start Menu area; don’t rely on the Taskbar to provide it.

  3. Allow a choice of where it sits

    • Offer Top Left/Top Right/Bottom Left/Bottom Right

    • You also need to consider whether to treat User Name, Search and Power Button as a block or to allow their positions to be selected individually – I would treat them as a block at the top right like Windows 8.1, but others will have other ideas, I’m sure!

  4. Allow a choice to show (or not) the Start Menu list items (those currently on the left)

    • If the choice is NOT to show the list items, then that frees up valuable screen space to allow the App tiles to start at the left hand side of the screen

    • You might also extend the ability to select what is shown (or not) beyond Recent Programs and Recent Items to File Explorer, Documents, and Settings.

    • Are there other things which should be included?

  5. Allow a choice to show (or not) the Start Menu App tiles (to some extent that is already there because we can delete them individually, but include a global setting)

  6. Default to show both list and tiles, which gives us the hybrid we have now, satisfying those who like that layout.

  7. Allow a choice between a Vertical list of All Apps (like now) or a Full Screen presentation of them (like Windows 8.1).

  8. Use the same Taskbar settings (Auto hide, Icon size, Location and so on) for the Start Menu Screen and the Desktop; don’t override them to try and fudge a fix for some other issue.

  3. Under Writing email, select Saving sent messages.
  4. Choose Save all sent messages in the Sent items folder.
  5. Click Save.
