shane devenshire say:

Hi,

Here is a vba approach

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim Insect As Range
    Set Insect = Application.Intersect(Target, Me.Range("B1:Z10"))
    If Not Insect Is Nothing Then
        Cells(Target.Row, 1) = "Ö"
        Cells.Font.Name = "Symbol"
    End If
End Sub

The above code will reflect a entered change made by entering or deleting something in the range B1:Z10.  What it will not do is add a check if a formula in that range is updated by a change elsewhere.

Also note that the Change event does not respond to things like changing the font to bold or coloring the cell.  That is why I suggested track changes.

If this answer helps, please click the Vote as Helpful button. Cheers, Shane Devenshire
