ivan fung say: Thank you for all reply

I have recently found the solution when I found that ALL OFFICE 2010 SOFTWARE suffers different kinds of problem.

I am here to summarize it:


Symptoms:
- Powerpoint: Some images cannot be displayed in normal view
- In protected view, all images are displayed as usual
- In normal view, error message: 'Image cannot currently be displayed' are found in SOME IMAGES ONLY

- Word: When opening it, error message: word could not create the work file
- Similar images problems suffered

- Other software: similar images problems suffer


Solution:
1. Some registry values has been changed, in my case: Temporary Internet Files
2. Fixed by the following procedure: 'Start' -> 'Run' -> 'regedit' -> found 'HKEY_CURRENT_USER \Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders' -> 'Cache' -> change the value back to '%USERPROFILE%\AppData\Local\Microsoft\Windows\Temporary Internet Files'
3. Probably, All data under this location of registry should have data starting with %USERPROFILE% except the 'default' line
- I just guess: Due to some reasons, these data change to other things which can make OFFICE not work smoothly and normally

bill manville say:

I just tested my proposal and found that it did work, but that the object's width had also been reduced so it remain virtually invisible.  The amended version is:

Sub RecoverShapes()
   Dim S As Shape
   Range("A1").CurrentRegion.AutoFilter  ' remove filter
   For Each S In ActiveSheet.Shapes
     If S.Height = 0 Then
        S.LockAspectRatio = False
        S.Height = S.TopLeftCell.Height
     End If
     If S.Width = 0 Then
        S.Width = S.TopLeftCell.Width
     End If
   Next
End Sub

If your filtered range does not start at A1 you will need to change the Range(... line

If you know what height/width the objects should be then you can set those explicitly
