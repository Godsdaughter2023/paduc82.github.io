Hi Jet0206,

I would suggest you to manually assign the drive letters by getting into Disk Management.

When you visit Disk Management, then you’ll see the original drive letters assigned to your Card reader.

But before you do that, let us try to uninstall your USB host controllers by following the steps below. Then we’ll re-install the drivers for the USB so that it would treat your USB card reader as a new device and windows would assign a drive letter automatically.

1. Click Start, and then click Run.

Note If you are running Windows Vista, click Start, and then use the Start Search box.
2. Type devmgmt.msc, and then click OK. Device Manager opens.
3. Expand Universal Serial Bus controllers.

Note You might have to scroll down the list to find this item.
4. Right-click the first USB controller under Universal Serial Bus controllers, and then click Uninstall to remove it.
5. Repeat step 4 for each USB controller that is listed under Universal Serial Bus controllers.
6. Restart the computer. After the computer starts, Windows will automatically scan for hardware changes and reinstall all the USB controllers that you uninstalled.

Now plug in your Card readers and check if the original drive letters are assigned to it or not.

If the above method did not work, then try assigning the drive letter manually as mentioned in the windows help link below.

Change, add, or remove a drive letter
http://windows.microsoft.com/en-US/windows-vista/Change-add-or-remove-a-drive-letter
