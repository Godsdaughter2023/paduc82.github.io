PDF is a format which a program like PPT uses an external engine to generate, it isn't a working format. PPT cannot open a PDF, it has to be opened for viewing by a PDF reader such as Preview or Adobe Reader.

However, single pages of a PDF can be inserted into a PPT presentation in the same way as pictures.
