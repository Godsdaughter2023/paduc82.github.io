• If you want to manually enter a specific number of seconds to buffer the content, select Buffer, and then type the number of seconds that you want.

  • RTSP/TCP

  • HTTP

 d. Click Configure.
 e. Click use proxy settings of the web browser.
 f. Click OK, Click Apply, and then click OK to close the options dialog box.

a. In Windows Media Player, on the Tools menu click options.
b. Under Performance tab, select the connection speed.

c. Select the appropriate speed and then click OK.

Thanks for the follow up. To further troubleshoot this error code in adding your account, we suggest that you uninstall and reinstall the Mail app.

To install the Mail app in Windows 10, follow these steps:

  1. Type PowerShell on the search engine.
  2. Right-click on Windows PowerShell then choose Run as administrator.
  3. Click Yes if you are prompted for access.
  4. Type this command in the PowerShell window: Get-AppxPackage *microsoft.windowscommunicationsapps* | Remove-AppxPackage and press Enter.
  5. Wait for the command to complete.
  6. Restart your PC.
To reinstall the Mail app, proceed to these steps:
 
  1. Launch the Store app.
  2. Once the Store app is launched, type mail in the search box and then click Mail and Calendar to open its page.
  3. Select the Install button.
  4. Try to add your account again.
Note: Reinstalling the Mail app might remove any saved mails and credentials in the app. You might need to add your email accounts all over again. We advise you create a manual restore point before attempting to reinstall the Mail app.Uninstalling the Mail app will also uninstall the Calendar app as both are packaged together as Windows communication apps. Likewise, when you install the Mail app, the Calendar app will also be installed.
