Genuine Windows Activation message [SOLVED] / Windows - Microsoft Community

kylesanders_996 say:

Genuine Windows Activation message [SOLVED] / Windows Update 'service not running' [SOLVED]

Microsoft was unable to solve the 2 issues I had. My brother, who is quite proficient in computers, was able to resolve the issues. Listed below are the 2 issues I was having and our solution for them.

1) Since rebuilding a former Vista-installed Dell 1545 in 2014 using Windows7, I have always received a Genuine Windows Activation message, that it wasn't able to detect licensed software. I knew this was incorrect as I had purchased the OS through WalMart, and correctly installed it myself. In addition, System and Windows Software Licensing Management Tool, slmgr.vbs displayed a genuine and activated OS. Apart from a timed message window pop-up asking me to activate Windows, other tools were affected or would not run such as Troubleshooter and Network Diagnostics, as well as some automatic update features.

Solution to 1) Removing Windows Update KB971033 from the recent updates list solved this issue immediately.



2) In attempting to resolve the above issue (which again, sometimes prevented updates and drivers from auto-installing) I would have to manually update drivers and such. I unintentionally installed and updated the Intel Matrix Storage Manager, which is what originally came on the computer. Although, Intel no longer supports this software, and it was preventing the current iteration of Intel chipset software (Intel Rapid Storage Technologies) from performing. This in-turn was not allowing Windows Update to function, claiming the 'service not running', although in Services, Windows Update and the Background Intelligence Transfer Service were in-fact started by the system and running. (Deleting SoftwareDistribution files while these services were stopped did not fix this issue).

Solution to 2) Removed/Uninstalled Matrix Storage Manager, and the current RST chipset software was able to function, allowing Windows Update to work again.

Method 1: Use the hardware troubleshooter to fix the issue

The Hardware Troubleshooter is an automated tool which will check the hardware connected to the computer for any known issues with it and provides the details and on how to fix them. Follow these steps and run the troubleshooter.

Follow these steps:

  1. Press Windows key to get the Start menu.
  2. Type Troubleshooting in the search box right up corner and hit enter.
  3. Click on Troubleshooting tab.
  4. Click on view all option on the upper left corner.
  5. Select the Hardware and Device options from the list.
  6. Click Next to run the Hardware and Device Troubleshooter.

If the issue persists, follow method 2.

Method 3: Update the Bluetooth drivers.

Step 1: Check for the latest driver update through Windows Updates.

  1. At the Start screen, type Windows Update.
  2. Click or tap Settings on the right side of the screen, and then select Install optional updates from the left results.
  3. When the Windows Update control panel opens, click on “Check for Updates” on the top left corner to receive a fresh list of updates.

Incompatible or not properly installed graphics card drivers are the most common cause of this issue.

Step 2: Follow the steps below to update the sound drivers.

  1. Press Windows + X key on the keyboard.
  2. Go to device manager.
  3. Right-click your Bluetooth devices.
  4. Click Update Driver Software and then follow the steps in the wizard that appears.

Restart the computer and check.

Step 3: Check in compatibility mode.

Download the Bluetooth drivers from the manufacture's website for Windows 10, if not available.

Follow these steps to install the drivers in compatibility mode and check if that helps.

  1. Download the driver from the manufacturer’s website.
  2. Right-click on the driver setup file and click on ‘properties’.
