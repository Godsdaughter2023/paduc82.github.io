vincentlavallee say:

It appears to me that none of these answers hit the point nor are really correct. There are real limits to the number of internal hard drives, which depends on the mother board, and if additional hard drive PCI cards are installed. Basically, most motherboards can handle ONLY 4 SATA drives, which is often 2 DVD drives and 2 hard drives. Or you can have 3 hard drives and one DVD drive. Some modern motherboards have 5 and some even have 6 SATA headers (where you connect the hard drive cables), so in these cases you can have either 4 internal hard drives and one DVD or 5 and one, or 4 hard drives and 2 DVD drives.

So, it has nothing to do with drive letters nor the space in your case, but much more with the system and the motherboard. With SCSI, it is a chain where each device (hard drive and other types) have an input connector and an output, so these can be chained together. With current SCSI controllers, you can have a lot more drives since they are in a chain and ALL connected to one header on the motherboard.

In addition, there may be some restirctions (I cannot find this info online yet) in the OS that limits the number of hard drives as well.With Windows, there are differences in limits allowed between Home Premium, Professional, and Ultimate, as is clearly the case with max memory recognized (Home premium can only see up to 16 GB, and the other two OS's can see at least 32GB of memory).

I will post more infor here when I learn more.

Each device has to be upgraded separately. One does not affect the other.

The KB3035583 that he is referring to is the 'Get Windows 10 App'.  However removing the update and therefore the App does not mean that Windows Update won't try to Upgrade to Windows 10 anyway. The App is just a front for Windows Update.

So if you don't want to upgrade your Desktop, uninstall KB3035583 but when you do updates for the Desktop, check that Windows Upgrade isn't set to download and install the Windows 10 Upgrade - which will show as an Optional update.

If you forget and leave the Windows 10 upgrade ticked in Windows Update it will download to your Desktop, but you can still cancel the installation, but you will in all likelihood be prompted about it from time to time.

PS. After you remove KB3035583 it will reappear in the update list again. I'm pretty sure that first time round the update was not an optional. If they haven't changed that, there is a chance it will simply download and install when Windows Update is run.

Easiest solution to that and Windows 10 Upgrade would be to make them hidden updates.
