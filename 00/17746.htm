hansv mvp say:
  1. Select the rows that you want to color, for example rows 2 to 50.
  2. In the following, I'll assume that the active cell within the selection is in row 2.
  3. On the Home tab of the ribbon, click Conditional Formatting > Manage Rules...
  4. Click New Rule...
  5. Select 'Use a formula to determine which cells to format'.
  6. Enter the formula   =$K2="Assigned"
  7. Here, 2 is the row number of the active cell and K is the column containing the dropdown menus.
  8. Click Format...
  9. Activate the Fill tab and specify the desired color.
  10. Click OK twice to return to the Conditional Formatting Rules Manager.
  11. Repeat steps 4 through 10 for each option in the dropdown menus, specifying the option in step 6 and choosing a different color in step 9.
  12. Click OK when you're done.

a. user say:

Microsoft has had to update the RTF standard for every new version of Word (more or less) and it is far more complex now than it was in Word 2000. 

So unfortunately the simplest approach is likely to be to ensure that you still have the version of Word that used to be used.

Other programs can also save in RTF format, but they may not understand all the RTF that is being used, and they may throw some of it away. But for example Windows Wordpad can save as RTF (although I suspect that the version in Windows 7 and possibly even Vista may create the more complex type of RTF. Open source software such as OpenOffice may be able to do it.

Otherwise, for small changes, a programmer who spent a bit of time learning RTF could certainly make small changes by opening the RTF as a plain text file (e.g. in Notepad). Making larger changes (e.g. that reflect a substantial change in the document content) would be much more difficult.

You might be better off trying to find a better editable format - for example, Word 2003 XML can be saved from Word 2003, 2007 and 2010. It's certainly not simple, but it's probably a bit easier to maintain than RTF. How long it will be available as a format that Word can work with, I cannot tell. For Word 2007 and later there is the rather more complex single-file "Flat OPC" XML format, which is like a single-file version of a .docx

(although if you have C code that expects RTF, using XML is obviously not going to work)
