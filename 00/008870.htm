jim gordon mvp say:

I do not concur with Phillip's suggestion that uninstall/reinstall will fix this.

When you booted to safe mode, the font cache was emptied, your file system was repaired, other apps that normally start up when you boot your Mac are not allowed to run, and the font set is minimized.

I don't think uninstall/reinstall will change anything.

If you have any 3rd party applications (especially WebEx, Zotero and Mendelay), be sure they are all up to date.

If all add-ins are up to date, the next place to check is fonts. Run a font utility (Font Book comes on your Mac). Use it to validate ALL your fonts. Delete old versions of duplicate fonts. Trash any font that doesn't come up as perfect. Then restart your Mac, empty the trash, and things should be OK.
