Hi G,

Both Microsoft Office Pivot Table 10.0 and Microsoft Office Pivot Table 11.0 Active X controls rely on Office Web Components which were deprecated beginning in Office 2007. If you need these controls, you will have to install the Office Web Components either from an earlier version of Office or they can be downloaded.

To see if you have the Office Web Components installed on your machine, you can search for either owc10.dll or owc11.dll.

Here are links to the downloads:

The Office 2003 Office Web Component Add-in version (OWC11) is available from
http://www.microsoft.com/downloads/en/details.aspx?FamilyID=7287252c-402e-4f72-97a5-e0fd290d4b76&displaylang=en

The Office XP Office Web Components (OWC10) are available from
http://www.microsoft.com/downloads/en/details.aspx?FamilyID=982B0359-0A86-4FB2-A7EE-5F3A499515DD&displaylang=en

However, because these controls are based on deprecated features, I would caution you on using them for any newly designed databases. I believe they are available mainly for backwards compatbility at this point.

Hope this helps,

Kathy
