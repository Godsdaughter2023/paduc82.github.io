You can simply move the Outlook Data File (.pst) to Local Disk (D:) and then open it up in Outlook again. Here's how:

  1. Exit Outlook.
  2. Click the Start button, type File Explorer and then press Enter.
  3. Locate the Outlook Data File (.pst) and move it to Local Disk (D:).

  5. An error message will appear notifying you that an Outlook Data File (.pst) could not be found.

rohn007 say:

Right basic idea, but it is almost impossible to do with normal text.

Instead I used the Insert tab >Shapes drop down to create a compound shape. Using a "Drawing Canvas" allows me to group separate shapes so that they are treated as a single unit by word

  1. in the Shapes drop down select "New Drawing Canvas" (this is key)
  2. in shapes drop down, insert a rectangle (orange border)
  3. double click inside of it type your letter
  4. select the letter set font attributes: Font size, color, font set
  5. Click on the shape to display Drawing Tools Format Conditional tab
  6. Alter the shape attributes, ie Shape Fill = none, Shape outline = none (I left them on and colored just for this example so you can see what I'm doing more easily)
  7. in shapes dropdown insert another rectangle (blue border)
  8. set shape attributes again
  9. double click inside the shape
  10. type the rest of the word
  11. select the letters
  12. set the font attributes
