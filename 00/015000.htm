In essence what you need to do is find the MIN matching the criteria on each of the three sheets, and then find the MIN of the MINs!

Like so:

=MIN(MIN(IF(Sheet1!$A$2:$A$10=$A2,Sheet1!$B$2:$B$10)),MIN(IF(Sheet2!$A$2:$A$10=$A2,Sheet2!$B$2:$B$10)),MIN(IF(Sheet3!$A$2:$A$10=$A2,Sheet3!$B$2:$B$10)))

This is an array formula, so you have to enter it using CTRL+SHIFT+ENTER. If at any time you edit it, you must always press CTRL+SHIFT+ENTER to re-enter it or it won't work properly.

Explanation:

MIN(IF(Sheet1!$A$2:$A$10=$A2,Sheet1!$B$2:$B$10)) finds the minimum value in column B on Sheet1 where column A matches the value in A2 on your summary sheet (i.e. Alpha).

This is done two more times for the other two sheets.

The outer MIN finds the minimum of those 3 results.
