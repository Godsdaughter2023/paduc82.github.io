a. user say:

DLC is licensed to two things. The account used to buy it and the console it was first download on. Typically DLC can be shared between accounts but there are of course a few exceptions, most notably Avatar items.

The console license lets any account on the console use the DLC, online or offline.

With the account license that lets you use the DLC on any console as long as you are connected to Live. This can also be used to temporarily allow others to access the DLC but as noted above, the purchasing account must also be signed into Live.

The license transfer can now be done solely on the console. Sign in with the purchasing account, press the guide button on your controller and press right until you reach the Settings Blade. Here pick the Account Management option.

You'll be taken to a new screen and on the first panel here the last option will be License Transfer, so pick that. Now in a couple of button presses the console license will be transferred to the new console, however this is not the end of the process.

The license must be refreshed and that means setting the content to be downloaded again. If the DLC is already on your hard drive then all that ill be grabbed from Live is the new license file, so it will be an incredibly quick process.

Now simply repeat the above for any other accounts that have DLC you want to belong to the new console. Remember though that the license transfer can only be done once every four months.

In a case like this, I'm only thinking of two possible things that could prevent Fallout 4 from loading:

  1. Corrupted game cache?
  2. Dying hard drive inside Xbox One.

When playing a game, the game automatically sets aside some "reserved space" (or cache) in the Xbox so some things could efficiently load faster while playing. Sometimes, the cache can randomly get corrupted and it could prevent the game from loading properly.

  • From the main menu, go to "My Games & Apps"
  • Find "Fallout 4" in your library of games.
  • Go to "Manage Game"
  • On the left side, go down to "Saved Data".
  • Delete the item that says "Reserved Space". It might be up to 2GB in size (at least for me). This is just the temporary game cache. (The item with your gamertag is your game saves)
  • Attempt to run Fallout 4 now.

If Fallout 4 still doesn't work after deleting the cache, then this might indicate a problem with the hard drive.

Vista did have problems connecting to Unix/Linux computers and devices and Vista did require the LAN Manager authentication to be changed to work with Samba. Win7, however, does not.
