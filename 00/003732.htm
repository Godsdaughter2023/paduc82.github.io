bogeymanin say:

This is for Excel 2016:

1. Deleted old sheets that contained connections to quoted data.

2. Opened a new blank sheet.

3. Used the commands: Data>Get Data>From Other Sources>From Web.

4. Pasted the string (http://quotes.fidelity.com/webxpress/get_quote?QUOTE_TYPE=R&SID_VALUE_ID=FPURX&submit=Quote) into the resulting dialog box.

5. Clicked the OK button. This created a dialog box from which I could choose a table to import.

5. Clicked through the tables and chose the one containing the mutual fund NAV or closing stock price.

6. Clicked the Load button. This created a new sheet with the selected data for that mutual fund or stock.

7. Repeated 3-6 for all mutual funds and stocks in my portfolio, but in each case replacing the FPURX in the string with the appropriate stock or mutual fund identifier.

8. Renamed all the new sheets appropriately.

9. Relinked the cells on the main sheet with the appropriate cell on the newly created sheets. 

10. Used the Refresh All command.

There may be simpler ways to do it, but this worked for me. When I originally created this file I used data from MS Money, but a few years ago that site quit providing that info, so I had to go elsewhere. I heard you can still get it from Yahoo!, too.

amit tandon say:

To get numerical value of a month, enter either of below formulas in cell B1:

=TEXT(A1&" 1","m")+0

or

=TEXT(A1&" 1","m")+0

To convert it back to a month in text:

=TEXT(B1*29,"mmm")
