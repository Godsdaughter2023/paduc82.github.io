What is your Windows resolution set to?

norman david jones say:

Hi Alex,

As indicated by Ketel, the the font size of a drop-down list that is created using data validation cannot be modified.

As a workaround, try the following event code which increases the zoom rate when a data validation dropdown cell is selected. If, subsequently,  a non dv dropdown cell is selected, the zoom rate is reduced.

  • Right-click the sheet tab
  • Select the View Code option from the resultant context menu
  • Paste the following event code:

'=========>>
Option Explicit

'--------->>
Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    Dim Rng As Range
    Const IncreasedZoomPercentage = "200"                '<<=== Change to taste
    Const NormalZoomPercentage As Long = "100"     '<<=== Change to taste

    On Error Resume Next
    Application.EnableEvents = False
    Set Rng = Me.UsedRange.SpecialCells(xlCellTypeAllValidation)
    Application.EnableEvents = True
    On Error GoTo 0

    With ActiveWindow
        If Not Intersect(Target, Rng) Is Nothing Then
            If Target.Validation.Type = xlValidateList Then
                .Zoom = IncreasedZoomPercentage
            End If
        Else
            .Zoom = NormalZoomPercentage
        End If
    End With
End Sub
'<<=========

  • Alt+Q to close the VBA editor and return to Excel

===

Regards,

Norman

norman david jones say:

Hi Alex,

Thank you Norman, but is it possible to make this vba code apply to every excel file I open?

I often receive client projects that include a lot of data with dropdowns.

You will need to use application events.

In the ThisWorkbook module of your PERSONAL.XLSB file, paste the following code:

'=========>>
Option Explicit
Private WithEvents App As Application

'--------->>
Private Sub Workbook_Open()
    Set App = Application
End Sub

'--------->>
Private Sub App_SheetSelectionChange(ByVal Sh As Object, ByVal Target As Range)
Dim Rng As Range
    Dim bZoom As Boolean
    Const IncreasedZoomPercentage = "200"                '<<=== Change to suit
    Const NormalZoomPercentage As Long = "100"     '<<=== Change to suit

    If Target.Cells.Count > 1 Then
        GoTo XIT
    End If

    On Error Resume Next
    Application.EnableEvents = False
    Set Rng = Sh.UsedRange.SpecialCells(xlCellTypeAllValidation)
    Application.EnableEvents = True
    On Error GoTo 0

    If Not Rng Is Nothing Then
        If Not Intersect(Target, Rng) Is Nothing Then
            If Target.Validation.Type = xlValidateList Then
                bZoom = True
            End If
        End If
    End If

XIT:
    With ActiveWindow
        If bZoom Then
            .Zoom = IncreasedZoomPercentage
        Else
            .Zoom = NormalZoomPercentage
        End If
    End With

End Sub
'<<=========

Save the changes to your PERSONAL.XLSB file. Then, to initiate the App variable, close and reopen Excel - or simply place the cursor inside the Workbook_Open procedure and hit the F5 key. Now, the zoom will be applied to any list-based data validation in any workbook.

For a detailed primer on the use of application events, see Chip Pearson at:

             http://www.cpearson.com/Excel/AppEvent.aspx

===

Regards,

Norman
