I'll try answering your question for the 2nd time.  PS: NEVER press the Backspace button using MS' 'improved' interface or you'll loose everything you've written.  Now that I'm done venting, let's get on with it!

The basic principle is very simple.  That said you can't do it directly with a macro.  You'll need to create some code.  Then if you wish you can run it using a macro.

So what do we need to do what you want:
1. Means to get a directory listing of files
2. Means to copy a file
3. Means to delete a file

(Yes there are other methods, but I will focus on a file level approach...)

For 1. See: http://www.devhut.net/index.php?lang=en&pid=0000000027#GetDirListing
For 2. See: http://www.devhut.net/index.php?lang=en&pid=0000000027#CopyFile
For 3. See VBA Help file on subject: Kill Statement

Now that we have the basic elements we simply need to weave them into a useful conhesive function or two.  How?  Try something along the lines of (untested air code)

Start off by creating a new module and add in the CopyFile Function
Then below, we'll start creating a new version of the GetDirListing Function but we'll call it MoveDir()

Function MoveDir(strPath As String, Optional strFilter As String)
' Input Variables:
' ~~~~~~~~~~~~~~~~
' strPath = full path include trailing \ ie:"c:\windows\"
' strFilter = extension of files ie:"pdf".  if you want to return
'             a complete listing of all the files enter a value of
'             "*" as the strFilter
'
Dim MyFile As String
Dim MyFolder As String
Dim bCopySuccess As Boolean
Dim sSourceFile As String
Dim sDestinationFolder As String
Dim sDestinationFile As String

    MyFolder = strPath
   
    'Add the trailing \ if it was omitted
    If Right(MyFolder, 1) <> "\" Then MyFolder = MyFolder & "\"
    'Modify the strFilter to include all files if omitted in the function
    'call
    If strFilter = "" Then strFilter = "*"
   
    sDestinationFolder = "Your Path Where To Move The Files To"
    If Right(sDestinationFolder, 1) <> "\" Then sDestinationFolder = sDestinationFolder & "\"
   
    'Loop through all the files in the directory by using Dir$ function
    MyFile = Dir$(MyFolder & "*." & strFilter, vbNormal + vbHidden + vbSystem)
    Do While MyFile <> ""
        sSourceFile = MyFolder & MyFile
        sDestinationFile = sDestinationFolder & MyFile
        bCopySuccess = CopyFile(sSourceFile, sDestinationFile)  'Copy the file
        If bCopySuccess = True Then 'See if the file copy was successful
          Kill SourceFile 'Now we delete the original copy
        End If
        MyFile = Dir$ 'Goto the next file in the directory listing
    Loop

End Function

As always you need to add proper error handling...  But this is the jist of it.
