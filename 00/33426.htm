a. user say:

Hi Andrew,

 

Thanks for posting on Microsoft Community. We regret the inconvenience caused and will assist you in resolving the issue.

 

The above issue might occur if the server of the website is down or if the Windows Media Player application settings are not set appropriately.

 

Sometimes website servers may be down due to various network issues. If that is the case, checking for connectivity of the website from time to time will help in confirming the server down of the website.

 

If not, follow the methods below and check if the issue is resolved.

 

Method 1: Check if Work offline is ticked in Windows media player.

 

  1. Open Windows Media Player.
  2. Click on File menu.
  3. Check if Work offline is ticked. If yes, then disable Work offline option.

 

Method 2: Check the privacy settings in Windows media player.

 

  1. Open Windows media player.
  2. Click on Tools and then again on Options.
  3. Switch to Privacy tab and check whether all the privacy settings is checked.

 

Method 3: Check the proxy server settings.

 

  1. Open Windows media player.
  2. Click on Tools and then again on Options.
  3. Switch to Network tab.
  4. Make sure that use the proxy settings of the web browser is checked.

 

Write to us with the status of the issue on the same post for further assistance. Your reply is most important for us to ensure we assist you accordingly.

 

Thank you.

If you're still having issues with album information, try doing the following.

Then check if you have the following line:
