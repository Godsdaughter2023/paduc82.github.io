thetao---01 say:

Nokia calls their Vibrate profile "Pager".  Have a look at Control Panel -> Profiles and compare the configuration of the Normal vs. the Pager profiles (Options -> Customize).  What makes Pager vibrate the phone is having "Vibrating alert" set to "On".  So you could make Normal both ring and vibrate by toggling on its "Vibrating alert" setting.  You could also make a brand new Profile and call it "Vibrate" if you like.

 

If Pager doesn't seem to work, try toggling one of its settings on and off ("Warning Tones" worked for me).  Could be a little bug there.

 

FYI, I found something that seemed like a real bug.  On my E73, in order to have the optical Navi key vibrate, I must have both "Vibrating alert" and Control Panel -> Settings -> General -> Personalization -> Optical Navi Key -> Vibration set to On.  That is, to make the Navi key vibrate, the phone must also always vibrate on incoming calls/messages.  To me, these functions should have nothing to do with each other. :smileysad:

Actually you can override Ctrl+V behavior in VBA

Alt+F11 > Insert Module > Copy and paste the below code

Sub pasteVal()
ActiveSheet.PasteSpecial Format:="Unicode Text", Link:=False, _
        DisplayAsIcon:=False 
End Sub

This time copy the below code in "ThisWorkBook" instead of new Module

Private Sub Workbook_Open()
    Application.OnKey "^v", "pasteVal"
End Sub

Now you can use Ctrl+V to paste as Unicode
