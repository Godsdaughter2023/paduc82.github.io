karl timmermans (outlook mvp 2012-2018) say:
Hello Robert

>>
Is there any reason to use IMAP on new LP versus using POP.
<<

There's no single "right answer" - depends on your requirements which only you can determine contrary to what some may say.

General Overview

#1 - POP account only downloads the <Inbox> data in a one-time one-way process. Nothing is sync'd with the server Everything is stored locally on the computer where the account is configured.

#2 - IMAP account syncs all email folders in a bi-directional process or to put it another way, what is add/changed/deleted on the server is reflected on your PC and vice-versa. If you have multiple devices that you want to have access to your email folders then IMAP is the way to go

I have a background article on the subject in case it's f interest

Understanding which is better for you – POP3 or Imap
https://www.contactgenie.info/understanding-pop...

Re: Importing problem

#3 - Personally wouldn't use the Import process to get the PST file data into your IMAP account if that's the way you want to go long-term. Would add the PST file to the Outlook profile and deal with the individual folders.

Rather then make this an extended response covering every conceivable option, will wait to provide specific steps once you confirm whether you want to revert the IMAP account to POP3 or continue on with the current IMAP account.

karl timmermans (outlook mvp 2012-2018) say:
Certainly your call but IF you opt to use the account as a POP3 account - keep the following in mind regardless of what any tech tells you

karl timmermans (outlook mvp 2012-2018) say:
Short answer: Yes

Assuming you plan on configuring the email account as a POP3 account, then what you refer to as "identical on the new as on the old laptop" is nothing more them a copy of the old pst file which I'm assuming is the file that was copied from the old laptop to "import into the IMAP account".

Some other background trivia - when an email account is newly configured by simply entering the email address/password, Outlook automatically configures it as an IMAP account by default (exception being for Outlook.com/Exchange accounts). Going to guess that the tech didn't ask you if you wanted to keep the account as a POP3 account which requires manual configuration.

During the manual configuration of a POP3 account, there is an explicit option to use an existing PST file (as shown in the attached screenshot). If you use the PST file from the old laptop - you're good to go once the account is configured - as I said, nothing to import or otherwise worry about.
