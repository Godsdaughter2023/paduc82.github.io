andreas killer say:

I assume the exported date in the cell is not a date, it is a string. If so you can get the parts:

From fixed positions as in row 2:

D2:  =MID(A2,4,2)
E2:  =LEFT(A2,2)
F2:  =RIGHT(A2,4)
G2:  =DATE(F2,E2,D2)

resp. combined

G2:  =DATE(RIGHT(A2,4),LEFT(A2,2),MID(A2,4,2))

You can also search for the /, get the position and then parts. That is useful if you have dates as in row 4 or 5:

B3:  =FIND("/",A3)
C3:  =FIND("/",A3,B3+1)
D3:  =MID(A3,B3+1,C3-B3-1)
E3:  =LEFT(A3,B3-1)
F3:  =MID(A3,C3+1,4)
G3:  =DATE(F3,E3,D3)

If the year has only 2 digits, it is interpreted as 1900, so if you want to convert it to 2000 use

G5:  =DATE(F5+IF(VALUE(F5)<100,2000,0),E5,D5)
