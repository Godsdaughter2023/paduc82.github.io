cmdrkeene say:

There's no one removal process, and no way to notify the Microsoft server that you've removed it (as far as the server knows, your Mac was just turned off for a long time).  But the removal still serves two purposes:

  1. Legal requirement based on license terms
  2. So it doesn't call the server later and cause Microsoft to indicate that the code has been used on more than 1 PC

As far as meeting the requirements for subsequent activation on another PC, "removal" could just mean you never boot it up ever again. But it's up to you how far you take that: erasing it, recycling the parts, re-installing something else in it's place (or not), all your choice. Just don't boot up Windows on it.

--

As for activation of the same copy later, it will probably succeed without question (it usually does until the same key gets used many times, and a single transfer by an owner like you doesn't usually flag it).  But even if it fails the automatic activation, you can call the toll-free number and get it activated an unlimited number of times -- as long as you aren't using it on more than 1 PC simultaneously, they don't care.
