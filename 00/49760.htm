a. user say:

Hi Maine,

Windows Live Photo Gallery tagging feature can use people tags to identify contacts that appear in their photos. If you want to just add a people tag to a photo without identifying each person in a photo, select the People tag option on the Home tab or in the Info pane. You can also change or remove the people tag.

To change the person who is tagged in a photo, perform the following steps:

1. Select a photo for which you want to change the people tag.
2. In the information pane, click the arrow next to the people tag you want to edit, and then click Tag as someone else.
3. In the Tag someone list, select another tag from the list or type a new name for the people tag, and then press Enter. This will replace the previous tag with the new one on that particular photo only.

To remove or rename tag:

1. Select a photo for which you want to change the people tag.

2. In the information pane, click the arrow next to the people tag you want to edit, and then click Remove tag.

3. To rename the tagged contact, click Rename contact.

4. A windows will pop-up where you can change the first and last name of the contact.

Please inform us if the issue still persists or if you have further questions/concerns, we will gladly help you with it.

a. user say:

Hi,

If you want to delete all the people tags in Windows Live Photo Gallery, please follow the steps below:

1. Launch Windows Live Photo Gallery.

2. Make sure all photos are included in the client.

3. Go to View tab and enable Tag and Caption pane.

4. Switch to Find tab.

5. Click the drop-down arrow in the People area.

6. Select All tagged or detected.

7. CTRL + A to select all photos.

8. Click the drop-down arrow beside the contact name below People tags in the right -side of the client.

9. Click Remove tag.

You have to do this to all people tags to completely delete the tags.
