Auto-Indent

In the new document I've created, it won't do it. I've checked to make sure that the word options are set for auto-formatting. I have no idea why it won't do it. Anyone else come across this?
jay freedman say:
The indenting has nothing to do with "auto-formatting". It's part of the definition of a paragraph style.

You should read http://www.shaunakelly.com/word/styles/stylesms.html for a description of what's in paragraph styles and how to use them and change them.

When you create a new document, Word builds the document from the Normal.dotm template. The first (and usually the only) paragraph in the new document has whatever style exists in the first (and usually only) paragraph in the template. Usually that style is called Normal (not to be confused with the Normal.dotm template). Each time you press Enter, the new paragraph has the same style as the previous paragraph unless you select a different style.

It sounds like you used to have a 0.5" first-line indent in the Normal style of your Normal.dotm template. (The default when Word is first installed is to have no first-line indent, so this must have been changed at some point.) Now that style has no first-line indent (possibly because the modified Normal.dotm template was deleted, and Word recreated it from the program default).

To get the indent back again, modify the Normal style (right-click it in the Home ribbon and click Modify), and click the option in the dialog for "New documents based on this template". Click the Format button in the Modify dialog and choose Paragraph; open the dropdown under "Special" and choose "First line"; and click OK in each dialog. When you next close Word, you may be asked whether to save changes to the Normal.dotm template, and you must say Yes to keep the modified style.
