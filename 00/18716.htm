Hi Danielle,

We cannot set specific users to create teams directly. 

By default, all users with a mailbox in Exchange Online have permissions to create Office 365 groups and therefore a team within Microsoft Teams. Thus, if we disable the permission of the user to create Office 365 group, the permission of Teams creation would be disabled as well.
