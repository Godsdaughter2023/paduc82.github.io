considerate_guy say:

I figured it out with a little research on the web. I'm not sure how long this will last, but for now you can reenable the Apps functionality in your Zune HD software as follows:

In the registry, navigate to:

HKEY_CURRENT_USER\Software\Microsoft\Zune

Create a new key called FeaturesOverride and now navigate to:

HKEY_CURRENT_USER\Software\Microsoft\Zune\FeaturesOverride

Add a new DWORD and call it Games

Set the value of Games to a non-zero value (e.g. 1)

When you restart your Zune software, you'll see the Apps options in the Marketplace, in your Collection and on your Device.  You can apparently still access the Marketplace, so if you missed downloading some apps by the 8/31/2012 deadline, go get them while you can.

And again, I'm not sure how long this might last.  It's possible that an update to the Zune software will completely remove the Apps and the ability to override and restore the feature.  I hope not, because I purchased my Zune HD with the promise that it would support apps and more, so I would feel completely cheated if they killed them entirely.
