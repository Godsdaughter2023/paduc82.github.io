On the other hand, perhaps you want something in particular to display. Even DoEvents may not slow things down enough to let the screen fully re-draw. I use this module to slow things down even more:

Sub KillSomeTime()

Dim PauseTime, Start
PauseTime = 1  ' Set duration in seconds
Start = Timer   ' Set start time.
Do While Timer < Start + PauseTime
DoEvents   ' Yield to other processes.
Loop

End Sub

markinthebox say:

I think you may not be understanding what I meant.  Let me try and explain this a different way:

The memory that Zune reports as "reserved space" is memory that will be used by your phone for applications you download (and their data), pictures and videos you take with the phone, etc.  The phone will use as much memory as it needs for these things (until memory is completely full), no matter what setting you have in Zune.

The Zune setting, on the other hand, just tells the Zune software that it must leaveat least the selected amount of memory free on your device when it syncs your collection.  Without this setting, if you tell Zune to sync your entire collection and your collection is large, it would fill all available memory and your phone would no longer have any memory left for its own functions.

To answer your follow-up question, I am not aware of a way currently to see a more detailed breakdown of how the reserved space is currently being used.

Meanwhile solved for me - after I remembered the ancient domain on which I created that live ID for the Windows Home Server Beta (passport.com) it was easy.

Better than relying on the brain would be however being able to enter the domain name somewhere and get the associated ID shown in the same way, as the security code request presents the email address.

If the system still exists, you can find the account in the registry under

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows Server\Domain Manager\ActiveConfiguration

While it is clear text in some older versions of Server Essentials, or Windows Home Server 2011, in Windows Server 2016 the Account Name is converted to binary.
