considerate_guy say:

Did you save your completed project as a movie?  (File > Save Movie)

 

If so, then you have a .mp4 or .wmv file that is your movie and it shouldn't loop.  If you are seeing looping, it would be a setting in your media playback software.  For example, if you are using Windows Media Player, make sure you haven't turned on auto-looping.  You can toggle the setting by clicking the loop icon, or just pressing Ctrl+T while in Windows Media Player.

 

Alternatively, if you have made a DVD, there's an option to play once or to loop.  Be sure under Options, when creating the DVD that you set it to not loop back to the beginning.

considerate_guy say:

Following on to your question, I tested the full screen preview within Movie Maker.  It seems like this must be the feature you are using to preview your project.  And in that mode, playback will loop back to the beginning.  I don't believe there is any setting that would stop that from happening.  The best I can suggest is that you submit this suggestion to feedback.live.com.

 

The preview is really just intended to give you a full screen preview of your project as you are editing.  It really wasn't intended to be the way you would show off your final movie.  Instead, to finalize your slideshow/movie, you should use the 'Save Movie' option I mentioned earlier.  That creates a stand-alone movie file which you can view (full screen if you like), upload to various sites, share with friends, etc.
