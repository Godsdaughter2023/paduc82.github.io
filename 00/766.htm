how do I keep from getting junk mail?

my junk box gets more than my inbox. I have a thousand in my block email box. how do I keep from getting junk mail?

brian tillman [outlook mvp 2007-2019] say:
You cannot prevent junk from arriving.  Only the server admins can prevent that.  The tool you have is the junk mail filter level (i.e., "Standard" or "Exclusive").  Junk mail is SUPPOSED to be delivered to the Junk folder.  That's its entire reason for existng.  From time to tim, just review the subject lines of the messages there to make sure a message hasn't been improperly delivered to Junk, but otherwise, forget about the messages there.  They'll be automatically deleted after a period of time.

Using your Blocked Sender list as a means of controlling junk mail is nearly pointless because junk mail senders don't use the same address over and over.  They might use a particular address a few times, but thet change them frequently and once they do, any entry you made in the Block Sender list is worthless.

sergent---01 say:

Hi guys,

 

When your phone notifies you if the phone memory is getting low. To free phone memory you have the following options:

  • Move data that you want to keep to an alternative memory (mass memory or memory card, if available) or to a compatible computer.
  • Remove data you no longer need.

Use File Manager or the respective application in your phone to remove data or to move data to alternative memory as described in the user guide. Typically you first mark files to be removed or moved and then you select the corresponding option (Delete or Move to folder).

Use Nokia Suite to move data to a compatible computer.

In order to free phone memory, you could remove or move the following:

  • Messages from Inbox, Drafts and Sent folders in Messaging
  • Retrieved e-mail messages from the phone memory
  • Saved Web pages. Clear the web browser cache to free some phone memory (e.g. select Options > Clear privacy data > Cache).
  • Saved images, videos, or sound files (in Gallery, Photos, Videos or Music player)
  • Contact information (in Contacts)
  • Calendar notes (in Calendar)
  • Downloaded application packages
  • If there are applications which are not needed any more, uninstall them with Application Manager
  • Any other data that you no longer need

If you are deleting multiple items and any of the following notes are shown: Not enough memory to perform operation. Delete some data first or Memory low. Delete some data, try deleting items one by one (starting from the smallest item). After deleting data, restart (power off/on) your phone.

Also Use the memory card or mass memory instead of phone memory in the Camera or other applicable applications in case the selection is available (e.g. via Options > Settings > Memory in use).

 

Hope this helps, let us know how you get on,

Hi Bryan2010,

Thanks for contacting Microsoft Support.  You've got some concerns with cloud matching in the Xbox Music App and to your Windows Phone 8.  Let's go over a few things regarding matching and syncing between your Windows 8 devices.

Having close to 13,000 songs is quite a large library, but that shouldn't affect cloud syncing.  The basic premise behind matching in Xbox Music is "Is this song available in our marketplace?".  This goes further in that, can the Xbox Music Marketplace read the metadata currently on the music file and match that to what is in the marketplace? 

We then get into the issue with licensing.  While Google can match the majority of your collection to their cloud services, those matching licenses, which allow for music to be streamed may not be available for Xbox Music.  Albums and artists don't actually denote which of their tracks may or may not be available for streaming, this can mean that both Xbox Music subscription downloads, and even purchases may not be available for streaming or cloud matching.

As for your playlists not syncing to your phone, there are a few things I would like to ask.

  • Do the playlists themselves show up on the phone at all?
  • Are you able to view these playlists on the http://music.xbox.com website and do they play without issues?
  • If the playlists themselves are showing up on the phone, but are just empty, one of the reasons for this can be that music was added to this playlist directly from the marketplace to a playlist. 
  • What is the make and model of your phone?  Also, what is the complete OS version number?  (found under Settings>About>More Info)

  • On your Windows 8.1 PC, open up the Xbox Music App.  Then go to Settings>Preferences.  We're looking for two things; "When you add songs from Xbox Mus features turned on?
