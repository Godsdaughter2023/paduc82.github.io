Hi,

 

That's the way SUM works and isn't a bug, SUM will ignore text cells. Adding then up using

 

=A1+A2+A3 cooerces the text values into numeric values so it returns the correct number.

 

To demonstrate this thy this put a text number in a1 and enter this formula

 

=ISNUMBER(A1)

 

The formula will return FALSE. Now try this formula

 

=ISNUMBER(A1*1)

 

multiplying the number by 1 makes it return true.

 

Now I have no idea how the SUM function works and would be deeply suspicious of anyone who says they do but clearly it doesn't coerce text values to numeric, it does exactly what it says in help, it ignores them.
