Open Form with Command Button - Microsoft Community

lostfan789 say:

Open Form with Command Button

I have a command button on a form (frmSearchPO) that will open up a form (frmLDC) to a particular PO number. It works fine but I don't want frmLDC to be filtered just to that number. I want the command just to go to that particular record but still have the other records available for viewing. Any ideas? Right now, the command button is using a macro to do this.
boblarson say:

Private Sub OK_Click()
Dim rs As Object
Dim lngBookmark As Long

'set a variable to the current record
lngBookmark = Me.LDCPONum
'open the new form
DoCmd.OpenForm "frmLDC"

'take it to the selected record
Set rs = Forms!frmLDC.RecordsetClone
rs.FindFirst "LDCPONum = " & lngBookmark
Forms!frmLDC.Bookmark = rs.Bookmark

Set rs = Nothing
End Sub

So I'm getting a "Type Mismatch" error.

Sorry, I think Paul didn't quite set it up the best there now that I'm looking at it.  He named lngBookmark with the word bookmark in it and isn't really using that for the bookmark.

 

So your revision SHOULD be something like this:

Private Sub OK_Click()
Dim rs As Object
Dim strFindValue As String

'set a variable to the current record
strFindValue = Me.LDCPONum
'open the new form
DoCmd.OpenForm "frmLDC"

'take it to the selected record
Set rs = Forms!frmLDC.RecordsetClone
rs.FindFirst "LDCPONum = " & Chr(34) & strFindValue & Chr(34)
Forms!frmLDC.Bookmark = rs.Bookmark

Set rs = Nothing
End Sub

Since LDCPONum is really text then you want to use the CHR(34) which are double quotes.
