matthewprice_981 say:

Thanks for the response Nayana,

This is now resolved.

Setting the reg key [HKEY_CURRENT_USER\software\policies\microsoft\office\15.0\common\signin] signinoptions to a value of 3 disables 'connected services' in Word | File | Account whilst leaving clipart 'Online Pictures' able to connect to the internet (as well as other internet enabled buttons on the ribbon).

The useonlinecontent key needs to be set to 3 (Allow Office to connect to the internet).

The first thing to try is to go to settings > Wi-Fi > turn the connection off > back out to the start screen > go back to Wi-Fi settings > turn back ON. 

 

If networks are still not displaying, try to soft reset the phone.  The easiest way to do this would be to remove the battery for about 2 minutes, replace and restart the phone.  Once you have done this go to Wi-Fi settings to see if any networks are being displayed.

 

Next, try adding a network manually.  More information on adding networks to your phone manually can be found here, http://www.windowsphone.com/en-us/how-to/wp7/start/connect-to-a-wi-fi-network.

  • Go to the bottom of the screen and tap the advanced button.

  • Input the network name in the network name box then tap add.
  • Enter the password in the password box.
