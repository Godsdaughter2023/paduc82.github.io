The vertical borders will be whatever the b/g color of the slide is, so while there's no direct way to control this, there's a pretty quick and simple trick, assuming you don't have loads of differently formatted slides:

Give the slides a black background, then go to the slide master and insert a rectangle, make it fill the slide, send to back and format it to match whatever the original fill of the slides was.  Now the slides all look the way they originally did but there's a hidden black b/g behind the "fake" b/g (the new rectangle on the master).

Now when you resize, you get black borders.

If the slides have lots of different formatted backgrounds, it gets "workier" ... you'd have to do this for each slide rather than relying on the masters to do the global job for you.

Windows 10: Windows Defender will not turn on *SOLVED* - Microsoft Community

technology guru say:

Windows 10: Windows Defender will not turn on *SOLVED*

I had recently disabled windows defender on windows 10 and when I did I know want to enable it again but when I open action center and press Turn On Now it does nothing. I have pressed it multiple times and with no result.

-------------------------------------------------------------------------------------------------------------------

Fix:

1. press Windows Key + R

2. type regedit

3. locate the following path HKEY_LOCAL_MACHINE\SOFTWARE\MICROSOFT\WINDOWS DEFENDER\

4. Right Click this file and select permissions 

5. press Advanced

6. where it says owner press change

7. change the owner to your username

8. Now press apply then ok

9. select Full Control from the permissions menu

(click the Widnows Defender folder)

10. Find the value DisableAntiSpyware and DisableAntiVirus and set these values to 0

(explaination: double click each and where the 1 is put 0 and press ok)

11. Open services.msc

12. find Windows Defender

13. there should be 2 services: Windows Defender Service and Windows Defnder Netwrok

14. right click each and press start

15. after you have started both restart your computer

16. Run Windows Defender

I Hope this Helps!
