hi, how can i turn off the microsoft gs wavetable synth? i want to use my midi keyboard with fl studio 10 and finale 2011. theplectrum say: In both FL Studio and Finale you'll need to use what is called a soft-synth and play the input from your midi keyboard into the output of the soft-synth.

I don't have either of these programmes, i use Sonar, but i would imagine the process is very similar.

Go into the Audio/Midi Options of both software, making sure your keyboard is ON and recognised as the default midi input within W7 (via the mixer)

Open up on of the two software and go into the Midi Options. Select your keyboard as the input device and also de-select GS Wavetable if listed.

Open up a project and insert a soft-synth (RTM about midi operation if in doubt). This should open a midi and an audio track. Select again your keyboard as the input in the midi track, and your soft-synth to output to.

In the audio track, select the midi track (or Omni) as input, and your audio device as output. If in doubt then RTM.

Theabove s the basic way in any DAW to use a keyboard within it, via a soft-synth, by passing the aweful GS Wavetable..

Once you've done it for one software you'll need to do it for the other. I would advise you strongly to save the set up as a template, if possible, so you won't have to go through the set up rigmarole again.
