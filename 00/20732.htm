jeeped say:

The following macro will give you all variations of the arrays you've described.

Sub mcr_All_Permutations()
    Dim i As Long, j As Long, k As Long, l As Long, m As Long
    Dim v1 As Variant, v2 As Variant, v3 As Variant, v4 As Variant
    v1 = Array("bird", "cat")
    v2 = Array("dog", "emu", "fox")
    v3 = Array("goat", "hen")
    v4 = Array("iguana", "jackal", "kangaroo")
    With Sheets(2)
        m = 1
        .Cells(1, 1).CurrentRegion.ClearContents
        .Cells(1, 1) = "data set"
        For i = LBound(v1) To UBound(v1)
            For j = LBound(v2) To UBound(v2)
                For k = LBound(v3) To UBound(v3)
                    For l = LBound(v4) To UBound(v4)
                        .Cells(Rows.Count, 1).End(xlUp).Offset(1, 0) = _
                          Left(v1(i), 1) & Left(v2(j), 1) & Left(v3(k), 1) & Left(v4(l), 1)
                        .Cells(Rows.Count, 1).End(xlUp).Offset(0, 1) = m: m = m + 1
                    Next l
                Next k
            Next j
        Next i
    End With
End Sub

Note that the target worksheet is the second worksheet in your workbook. Change the seventh line if you want the results somewhere else. Once you have this table, you should be able to use a VLOOKUP() on the concatenated first initials to get an ordinal number.

I'm sure the microphone won't really hear 48 kHz, and probably won't even hear 16 kHz. Most people can't hear 16 kHz sounds unless they're in their teens or early 20's. So there's really no cost to leaving it in the 16 kHz mode.

How to write NULL into csv from Excel - Microsoft Community

mmocom say:

How to write NULL into csv from Excel

How do I write NULL for each blank field into a csv file when I export data from Excel?   I have rows of data in Excel and there are rows where a given column might be empty.  For each column where there is no data specified, I want to write NULL in the CSV.  Is there a way to do that within Excel?

 

hansv mvp say:

Perhaps you can adapt the following macro to suit your purpose:

 

Sub Null2CSV()
    Dim wsh As Worksheet
    Dim strFile As String
    Dim rng As Range
    Set wsh = ActiveSheet
    Set rng = wsh.UsedRange.SpecialCells(xlCellTypeLastCell)
    Set rng = wsh.Range(wsh.Range("A1"), rng)
    On Error Resume Next
    rng.SpecialCells(xlCellTypeBlanks).Value = "NULL"
    On Error GoTo 0
    strFile = "MyFile.csv"
    wsh.SaveAs Filename:=strFile, FileFormat:=xlCSV
End Sub

If you have tasks which are not truly, ruly, predecessors, then don't link them. I know it is tempting, but it is a poor modeling practice because it is not a good representation of your project.
