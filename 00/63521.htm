When Windows is usable in Safe Mode but not in Normal Mode then the cause is in most cases a rogue program or service. Using msconfig.exe to disable all startup programs and all non-Microsoft services will usually identify the culprit. In your case this does not appear to be the case. The next suspect is a misbehaving driver, e.g. a display driver or NIC driver. You can disable all non-essential drivers in Safe Mode via the Control Panel / System / Hardware Manager.

Note also that a wallpaper screen without shortcuts is caused by Windows Explorer not running. You can force it to run like so:

  • Press Ctrl+Shift+Esc to launch the Task Manager.
  • Click the Applications tab.
  • Click New Task.
  • Type Explorer.exe and press Enter.

Full version (Retail):

- Includes transfer rights to another computer.

- Doesn't require a previous qualifying version of Windows.

- Expensive

Upgrade version (Retail):

- Includes transfer rights to another computer.

- require a previous qualifying version of Windows.

- Expensive, but cheaper than full version

OEM System Builder version:

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM licenses are tied to the very first computer you install and activate it on

- OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

- OEM versions cannot be used to directly upgrade from an older Windows operating system

There is nothing wrong with using it too, especially since all future releases of Windows will be full versions anyway.

You can find OEM System Builder software from dozens of online merchants. The current price for OEM Windows 7 Professional at Newegg, for example, is $140. In the past, Microsoft has been remarkably inconsistent in its advice to customers about whether this practice is allowed. (See "Is it OK to use OEM Windows on your own PC? Don't ask Microsoft.")
