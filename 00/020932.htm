shane devenshire say:

Well the easiest solutions are 1. to put the buttons at the top of the sheet and apply Free Panes.  2. attach the buttons to the QAT or a custom tab on the ribbon.

And the hard way is to call a windows DLL to detect the screen resolution on the system and modify the display accordingly.  One example from the internet:

There's no property within the VBA object model that will give you the
screen dimensions. After all, the object model used by an Office application
isn't dealing with the "screen" (the entire area available on the monitor)
-- it's dealing with some window ON the screen.

You will need to use functions from the Win-32 API. You can use the Declare
statement to make the functions available to your VBA code, and then call
the functions to retrieve that information.

For example:
..
Option Explicit
Private Declare Function GetSystemMetrics Lib "user32" (ByVal whichMetric As
Long) As Long
Const SM_CXSCREEN As Long = 0
Const SM_CYSCREEN As Long = 1
Public Sub GetScreenDimensions()
..Dim myDimension As Long
..
..myDimension = GetSystemMetrics(SM_CXSCREEN)
..Debug.Print "The screen is " & myDimension & " pixels wide"
..myDimension = GetSystemMetrics(SM_CYSCREEN)
..Debug.Print "The screen is " & myDimension & " pixels high"
End Sub

That's because the feature is primarily for use in lists to which the Table structure has been applied. In an actual Table the Remove Duplicates command will pick up the literal column headings. It can be applied to a simple list, but it uses the generic column labels because the first row technically is not recognized as being a Header Row.

If you convert the list to a Table you can click anywhere in the Table & when you select the Remove Duplicates command (also available on the Table contextual tab) your column headings will be shown in the dialog window.

Private Sub Merge_Click()
Application.DisplayAlerts = False
Dim applicationDocName As String

applicationDocName = ActiveWorkbook.Name

Workbooks.Add

mergeDocumentName = ActiveWorkbook.Name

For i = 1 To ActiveWorkbook.Sheets.Count - 1
     ActiveSheet.Delete

ActiveSheet.Name = "End"

        Workbooks.Open (ListBox1.List(i))
        Dim workbookName As String
        workbookName = ActiveWorkbook.Name
        ActiveWorkbook.Sheets(1).Select
        ActiveSheet.Copy Before:=Workbooks(mergeDocumentName).Sheets(i + 1)
        Dim sheetName As String

        ActiveSheet.Name = sheetName
        Workbooks(workbookName).Close
    Next
    Unload UserForm1

End Sub
