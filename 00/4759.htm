Open your template notebook and then Export (File, Export) it using the .onepkg format into one file. Close it. david in sydney say:

and if you drag that template file (from the usual Windows Explorer view) to the tray icon for OneNote, it will become a pinned item: hold or right click the tray icon and then you can select the template file and away you go ...
