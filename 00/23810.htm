john korchok say:

You can set the text size of placeholders to 16 point, but if you paste in a lot of text and fill the placeholder, by default the text will shrink. To change this:

  1. Select the placeholder on the slide layout or master.
  2. On the Drawing Tools Format tab, click on the dialog opener in the lower right corner of the Shape Styles or Size group. The Format Shape pane opens at the side of your screen.
  3. In the Format Shape pane, click on Text Options, then click on the Text Box icon.
  4. Check Do not Autofit.
  5. Repeat for all placeholders. Setting this on the slide master first should set it on most of your layouts as well.

When pasting, if you click on the bottom half of the paste button, you'll see several clipboard icons. Use Destination Theme will use the theme of your presentation. Use this when pasting from Word. The Plain Text paste will be more useful for pasting blocks of text.

If you choose Paste Special, you'll see the format options for the current clipboard content. This list will vary depending on the source of the text.

While you can't set the default paste format in PowerPoint like you can in Word, you can add a variety of Paste buttons to the Quick Access Toolbar:

  1. In PowerPoint, choose File>Options>Quick Access Toolbar.
  2. Change Choose Commands from: to All Commands.
  3. Choose from 21 different Paste buttons. For your purpose, the most likely candidate is Paste and Keep Text Only. Select it and click on the Add button.
  4. OK out. The new icon is added to the QAT.
  5. These icons have keyboard shortcuts. The 4th icon from the left will be triggered by Alt + 4. Alt + 5 will fire the 5th icon, etc.

diane poremsky mvp say:
You can move the domain to a new hosting- GoDaddy used to include some free mailbox with the domain registration, I dont know if they still do. Otherwise, you will need to either use forwarding (should be free, can forward to the user's current Outlook.com accounts) or pay a small amount for hosting email - if you sign up for a cheap web host package (HostGator etc) somewhere, you should get email accounts - look for one in your price range that offers unlimited email.

To move the domain and use Forwarding, each user needs to an add outlook.com alias to their account. Then move the domain or configure forwarding for email, update DNS and forward to the outlook.com aliases.

As long as the user log into their outlook.com account once a year, they wont need to export their mail to a pst (but may want to anyway, as a backup.)
