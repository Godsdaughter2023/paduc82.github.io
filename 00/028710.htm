graham mayor say:
There are three settings that affect the vertical spacing of paragraph text.

1. Line spacing - this affects the spacing WITHIN a paragraph
2. Space Before - this affects the spacing BEFORE a paragraph
3. Space After - this affects the spacing AFTER a paragraph.

You begin a new paragraph each time you press the Enter key so line spacing is not going to change the address line spacing.

You have two choices. Either use Shift+Enter to create a line break within a paragraph or set Space Before/Space After to '0'.
