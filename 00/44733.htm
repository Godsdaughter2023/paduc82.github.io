    My printer will not print until I clear the print job. - Microsoft Community

    michael lee_905 say:

    My printer will not print until I clear the print job.

    Hello my printer will not print I go and click on print and it shows that it is printing but never prints then  and then it will not even do that until I clear the print job
    lemp say:
    Usually, when a wireless printer will not print, the reason is that the IP address that was originally assigned to the printer by your wireless router has changed.  This can easily happen if, while the printer is turned off, any other device is connected to the wireless network.  The new device gets assigned the IP address formerly assigned to the printer and Windows does not update its configuration with the printer is turned back on and assigned a new IP address.

    The following looks a lot more complicated than it is.  Really.  Read through it before you give up.  The User Guide for your printer is HERE and the User Guide for your router is HERE.  All of the following is described in those two manuals.

    To check for the above situation, do the following.

    1. On the printer, press Home.

    2. Press (down arrow) to select Network Settings, then press OK.

    3. Press (down arrow) to select View Network Configuration, then press OK.

    4. Make a note of the IP address.  It should look like 192.168.0.x (where x is between 2 and 254)
    5. On your computer, go to Start > Devices and Printers
    6. Right-click on the icon for your Kodak printer and select "Printer properties"
    7. Click the "Ports" tab
    8. Click the "Configure Port" button
    9. Look at the box next to "Printer name or IP address."  It should have the same value as the IP address that you noted in Step 4.  It probably isn't the same, so make it so.
    10. OK your way out and you should be able to print now.


    If that was in fact your problem, it may happen again.  The way to prevent that is to configure the printer to use a "static" or permanent IP address.  First, however, you have to know what to choose for the static IP address.


    1. Open a web browser (e.g., Internet Explorer) and go to the following address:  192.168.0.1
    2. Click the "Login" link.
    3. Unless you have changed the defaults, use "user" (without quotes) for both the User name and password
    4. At the top of the page, you should see a "Menu Bar" with the following:  Modem, Gateway, Wireless, VPN, Routing, Parental Control, Firewall, Tools.  Select "Gateway"
    5. At the Gateway menu, click "Information" in the "Menu Tree" on the left.  Among other things, you should see an item called "Private IP Range."  The manual shows this as
          192.168.0.10 through 192.168.0.15
      but I don't know whether that's the default or just an example.  In any case, look at the last segment of those addresses.  As long as the first one is greater than 2 and the last one is smaller than 254, you don't have to make any changes.  You can select a static IP address for your printer that is greater than 1 and less than the lower limit or greater than the upper limit but lower than 255.   In other words, for the example shown, you could select 192.168.0.5 or 192.168.0.50.
    6. In order to keep these directions from getting even more complicated, I'm going to assume that you're able to select a static IP address where the last segment is less than 255 and that also is not within the "Private IP Range."  If that assumption is false, let me know and I'll give further directions.
    7. Now that you have chosen a static IP address, you have to configure it both in the computer and in the printer.
    8. While you're looking at the Gateway Information page, make a note of the address shown for DNS in the "Internet Settings" area.

    To set up your IP Address on the printer:


    1. Press Home.
    2. Press [down arrow] to select Network Settings, then press OK.
    3. Press [down arrow] to select Advanced IP Address Setup, then press OK.
    4. Press OK again to continue.
    5. Select Manual (Static), then press OK:
    6. You will be prompted to enter an IP Address, a Subnet Mask, a Gateway Address, and a DNS Server address.  Press OK after each entry. Use the navigation buttons to enter each digit of the numbers.


    IMPORTANT:  This Kodak printer requires that each segment (the parts between the periods) have 3 digits.  You have to add leading zeros where necessary.  For example, if you selected 192.168.0.50 as the static IP address you want to use, you would enter 192.168.000.050


    For the Subnet Mask, enter 255.255.255.000

    For the Gateway Address, enter 192.168.000.001

    For the DNS Server, enter  either the IP address you noted above from the router's Gateway Information page (being sure to insert leading zeros as needed) or 192.168.000.001


    To set up the static IP address on your computer, use the same method as originally described above:

    1. On your computer, go to Start > Devices and Printers
    2. Right-click on the icon for your Kodak printer and select "Printer properties"
    3. Click the "Ports" tab
    4. Click the "Configure Port" button
    5. Enter your chosen static IP address in the box next to "Printer name or IP address." 
    6. OK your way out and you should be able to print now.
