This is a recovery partition brought into view in 1803. You cannot remove, but only you must remove its mapping of its respective drive letter, which is J. greg carmack - mvp 2010-2020 say:
Hi Michael. I'm Greg, an installation specialist and 8 year Windows MVP, here to help you.

There is an issue where Version 1803 adds a drive letter to the Recovery partition that should be hidden because it's only used for bootable repair tools that in previous edition of Windows were on C drive. Then it reports that it is full.

Since the partition is to meant to be hidden it can't have drive letter removed in Disk Mgmt like other partitions. So:

1. Select the Start button, type cmd and then right-click Command Prompt in the search results and select Run as administrator.

2. In the Administrator: Command Prompt window, type diskpart and press Enter.

3. Type list volume and press Enter.

4. Make note of the volume number for the drive letter you want to remove. For example, if the drive letter you want to remove is Drive “D”, note the volume number for Drive D.

5. Type the following command, and press Enter.

select volume <volume number>

When typing this command, in place of <volume number>, substitute the volume number you made note of in Step 4. In the example above, if the volume number for drive G is 5, you would type: select volume 5

6. Type the following command and press Enter.

remove letter=<drive letter>

When typing this command, in place of <drive letter>, substitute the drive letter you want to remove. In this example, you would type: remove letter=G

7. Close the Command Prompt window.

I hope this helps. Feel free to ask back any questions and let us know how it goes. I will keep working with you until it's resolved.

Color problems with Games on Windows 7 and Vista - Microsoft Community

littledevil501 say:

Color problems with Games on Windows 7 and Vista

Hello,

 

When playing Age of Empires or other legacy games I the colors are not displayed correctly, game pieces are not the right colors. Is there a fix for this?

 

Thanks

bobbymi say:

Hello LD,

 

You can see color problems with games installed on Windows if the game is not compatible with your version of Windows, with the Windows Aero Themes you may be using, older video cards, video card driver, DirectX version, or even other processes running on the PC. This is usually the case with legacy games.  Try the following methods to fix the issue:

 

Method 1: Set the game to run in Compatibility mode if it is not compatible with Windows 7

 

1.       Browse to the executable (.exe) for the game

2.       Right-click on the .exe file

3.       Under Compatibility mode, select the option to Run this program in compatibility mode for:

4.       From the drop down, set the operating system to the version of Windows that the game was designed for.

5.       Click OK and then test the game

 

Method 2: Disable the visual themes

 

1.       Browse to the executable (.exe) for the game

2.       Right-click on the .exe file

3.       Under Compatibility mode, select the option to Run this program in compatibility mode for: to your current version operating system, (Example: Windows 7)

4.       From the drop down, set the operating system to the version of Windows that the game was designed for.

5.       Click the option under Settings to Run in 256 colors, Disable visual Themes and Run this program as an administrator

6.       Click OK and then test the game

 

Method 3: End the Explorer.exe process

 

1.       Press Ctrl + Alt +Delete and then click Start Task Manager

2.       Click the Process tab and locate explorer.exe in the list of running processes

3.       Click to select explorer.exe and then click the End Process button

4.       Use the Alt + Tab keys to go back to the game

 

Note: This will temporarily remove the task bar and close any open folders

 

To get the desktop back, go back to the Task Manager, click the Applications tab, click the New Task button and type explorer.exe in the Open box and click OK.

 

Method 4: For Age of Empires 2, open the resolution changer window before starting the game

1.       Click Start and type Resolution in the Search box, click Adjust screen resolution

2.       Start Age of Emperors 2

 

Method 5: Create a batch file to start Age of Empires 2

1.       Open Notepad

2.       Paste the following text into Notepad

taskkill /F /IM Explorer.exe
age2_x1.exe

Start explorer.exe

3.       Save the file as age2.cmd and put it in the same folder as age2_x1.exe

 

Note: To find where age2_x1.exe is located right-click the link you use to start the game and click Open file location.

Note: This may work for other legacy games that have similar issues with colors, just use the .exe for that game.

 

For Age of Empires the age of kings, use this text for the batch file text and put it in the folder with empires2.exe

taskkill /F /IM Explorer.exe

empires2.exe

Start explorer.exe

 

Hope this helps

 

Bobby
