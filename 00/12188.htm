Even though the owner account is deleted, one of the co-owners with the global admin role can add himself as the secondary site administrator to the deleted user’s personal site via SharePoint admin center>user profiles>Manage User Profile>Manage site collection owners before the owner’s account is deleted. What you can do is create a partition from within Windows 8.1 Pro on your Surface Pro

Wait while Windows 8.1 checks for available disk space.

greginmich say:

If you don’t want to wait for the update included with the scheduled scan – or the rather unpredictable automatic daily update, you can have MSE check for updates every time the PC is turned on with a startup batch file. Copy and paste the lines to a Notepad file; give the file a .bat extension; and save it to your Startup folder.

 

Command line signature update:

 

@echo off

timeout /t 200

Cd C:\Program Files\Microsoft Security Client

MpCmdRun.exe -SignatureUpdate

Pause

 

Main interface signature update:

 

@echo off

timeout /t 200

“C:\Program Files\Microsoft Security Client\msseces.exe” -Update

 

Adjust the timeout (in seconds) as required.

stefan blom say:

In Word 2007 (and 2010), you will have to set this up with multilevel (outline) numbering; then you can add static text but omit the actual number.

Here's how to do it: On the Home tab, click Multilevel List, and then click Define New Multilevel List. In the dialog box, clear the numbering by deleting it in the "Enter formatting for number" box, and then type in the static text that you want. Click the More button (if necessary) to see the entire dialog. In the "Link level to style" box, select the style you want. When you are done, click OK. Apply "numbering" by applying the attached paragraph style to text.

If you want to edit your list, place the insertion point in the first list item, and then click Multilevel List | Define New Multilevel List again.
