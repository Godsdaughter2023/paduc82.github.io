We apologize for the delay in our response. To organize the files into several folders, you have to create specific Month folders and group the photos by Month taken. Please follow these steps:

 

1. On Photo Gallery, go to the folder where the photos are contained.

2. Right-click on an empty part where the pictures, click Group by -> Month taken. The photos should be grouped per month.

3. Create a new folder by right-clicking the folder on the left pane, then clicking Create new folder. Repeat this step for the months that you have for the photos.

4. Go again to the root folder where the photos are located. Click on the month so that all photos for that month will be selected.

5. Press Ctrl+X on your keyboard to cut the photos.

6. Go to the folder for that month, then press Ctrl+V to paste it.

7. Repeat steps 4-6 for all the photos so it can be organized to each folder.
