lemp say:

It's OK to connect to the Internet right after Windows 7 installation ... as long as you only go to known secure websites.  These include the Microsoft site for direct download of SP1, your computer manufacturer's website for downloading drivers, and an A/V manufacturer's web site for downloading the installer for an antivirus program.

Note, however, that all 3 of the above (Windows 7 sp1, necessary drivers, and an a/v installer) should preferably be downloaded before you start the whole Windows install process and either burned to a CD or saved on a USB flash drive that is known to be not infected (i.e., scan it first with good a/v and antimalware scanners).

Going to Windows Update after you've installed Windows 7 (either before or after installing SP1) is safe.
