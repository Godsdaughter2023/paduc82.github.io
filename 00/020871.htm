How do I copy cells to keep the row constant but change the column - Microsoft Community

pc_907 say:

How do I copy cells to keep the row constant but change the column

How do I copy cells to keep the row constant but change the column?
jeeped say:
if you have a formula in Y1 to SUM() the cells in the A1:A10 range, then the formula would be something like,

=SUM(A1$:A$10)

If you copy that formula from Y1 to Z5 then it will become,

=SUM(B1$:B$10)

The $ locks the row as absolute while leaving the column relative. In this manner the rows stay the same but the columns will shift according to the new location.

Switch between relative, absolute, and mixed references

chip pearson say:

Use the $ character before the part of the range reference that you don't want Excel to change. E.g.,

 

=A$10

 

Here, if you copy to the right, Excel will change the A to a B to a C and so on. If you copy down, Excel will not modify the 10 because it is prefixed by a $ character.

bonnienatalie say:

Can I transfer (copy) my emails onto a CD or DVD?

Hi. I want to copy/transfer many of my Outlook emails onto a DVD so I can keep them and then delete them from Outlook. I don't know how to do that or how to find that information.  Can anyone help me with this ?  Thanks!
diane poremsky mvp (slipstick.com) say:

Converting email: you can use File, Save as but that is one message at a time. You'll need to use a 3rd party program to process an entire folder or message store - the programs preserve folder structure so the folders in windows explorer will be the same as in outlook. A list of programs is at http://www.slipstick.com/addins/housekeeping.asp. Or you can use the Acrobat trial to make the package. You can test it - you won't be deleting messages from the pst and can try various methods.

You can drag as many messages to Windows explorer as you want - i'd limit it to about 100 at a time though, just because it might be slow if you do too many at a time. Selecting one item, holding shift then clicking on the last should select all. If you want to select the entire folder contents, select one message and Ctrl+A to select all. (This may not work in the yahoo email client - it will work in outlook.)
