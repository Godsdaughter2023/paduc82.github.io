Prior to OL'2013 - the default contact/calendar folders were stored in a separate PST file which would have been set to be the "default" data file for the profile (this is based on there only being one email account being configured for the sake of this response). Starting with OL'2013 - the contact/calendar folders were included in the .OST file for the account with the added description "This Computer Only" (a seriously bad decision). which means that if the account is deleted from the profile OR the .ost file gets separated from the email account/profile used to create it - you will lose all data in any folder with the designation "This computer only".

The "default" data file defined for the profile has no impact on the operation of Outlook other then changing where the default contact/calendar folders reside. and once set to a PST file, means you won't lose the data in those folders since .pst files never get deleted and can always be opened/read on their own in any Outlook version.

Ignoring all else, changing the default contact folder to a .pst file is an extremely prudent choice but provides the added benefit of enabling categories

HOW I USE THE LEN FUNCTION - Microsoft Community

maks10101 say:

HOW I USE THE LEN FUNCTION

HOW I GET THE VALUE IN CELL "A1"=1 AND CELL "B1"=.0  WHEN A CELL "D1"CONTAINS VALE 1.0 IN NUMBER FORMAT
sheeloo say:

If you type 1.0 in D1, Excel will store it as 1 (ignoring the 0 after decimal) so LEN(D1) will give you 1 and your formula will not work...

 

You can try

IN A1, =IF(LEN(D1)<=3,LEFT(D1,1),"")
or

IN A1, =IF(LEN(D1)<4,LEFT(D1,1),"")

 

Pl. note that if D1 has any number from 1.01 to 1.09 you will still get 1 in A1.

Since you are not sure whether D1 will contain 1 or 1.0 you will have to use the AND operator to test it for length and equality with 1 like this

=IF(AND(LEN(D1)<=3,D1=1),1,"")

 

Let me know how it goes.
