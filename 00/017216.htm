I agree with Le Boule that WDO may not be compatible with Windows 8.1 yet.

 

However, be sure that you are using the correct architecture. To scan a 64 bit version of Windows, you need to install the 64 bit download.

When you download Windows Defender Offline, you select the architecture for the PC you are going to scan. So, on a machine running 32 bit Windows, you select the 64 bit version of Windows Defender Offline. The Wizard will create the bootable media for your target architecture -- x64.

 

This is contrary to what downloading 32 bit vs. 64 bit means. The WDO download that runs to create the boot media can run on either architecture. The critical part is that the architecture of the bootable media created matches the OS of the PC to be scanned.
