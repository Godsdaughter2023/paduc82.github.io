Going from Vista back to XP - Microsoft Community

susie1951 say:

Going from Vista back to XP

I am starting a at home business and the software requires I have XP, not Vista.  My lap top has Vista. Can I go from Vista back to XP?
malkeleah say:

I'm assuming that you didn't upgrade to Vista from XP and that the laptop came with Vista preinstalled.

General information about replacing Vista with XP:

On an OEM (HP, Sony, etc.) computer:

1. Go to the OEM's website and look for XP drivers for your specific model computer. If there are no XP drivers, then you can't install XP. End of story. If there are drivers, download them and store on a CD-R or USB thumbdrive; you'll need them after you install XP.

2. Check with the OEM - either from their tech support website or by calling them - to see if you will void your warranty if you do this. If you will void the warranty, you make the decision.

3. If the OEM does support XP on the machine, call them and see if you can have downgrade rights and have them send you an XP restore disk. This will be far the easiest and best way of getting XP on the machine.

4. If XP is supported on the machine but the OEM doesn't have an XP restore disk for you, understand that you'll need to purchase a retail copy of XP from your favorite online or brick/mortar store.

5. Also understand that you will need to do a clean install of XP so if you have any data you want, back it up first.

6. If none of the above is applicable to you because you can't run XP on that machine (see Item #1 above), return the computer and purchase one running XP instead.

An alternative would be to install virtualization software on your Vista machine such as MS Virtual PC , VMware Workstation , or Sun's VirtualBox . You will need to have a full retail version of XP to install in the virtual machine which is not in use on any other computer.

bossdweeb say:

The System Restore points are somewhere deep in Windows System, which is normally, and best left hidden.

Clear the tic box next to  'Hide protected operating system files (Recommended)' in...

System>File Explorer Options>View tab to access this part of Windows.

- I don't know the actual location, but you are not likely to be able to use the System Restore files from within Windows if you do find them.

You may be able to find a 3rd party software that can restore from that location though ?

"added restore points since I was doing some updates over the weekend"

The version updates overwrite a lot of Windows, so they wipe the System Restore points, and generally disable System Restore

as well. The pre update restore points could cause havoc with the 'new' Win10 version.

Cumulative & security updates... I don't recall any of these wiping my restore points, and System Restore can generally be

used to roll these updates back if necessary, but there may be exceptions ?

.

Hi Frank

Large Windows Updates will delete all your Restore Points to maintain System Stability.

System Restore Points are Stored in the System Volume Information folder on your C Drive, they are however, totally inaccessible to the user

Since Windows Automatic Updates was introduced in Windows 10, System Restore is basically a defunct function n Windows 10, unless you manually create Restore Points after every major Windows Update
