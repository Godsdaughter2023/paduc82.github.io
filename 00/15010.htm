Hi AutomateNZ,

 

I could suggest you one more solution, but I have to mention that Zendesk is out of my scope, so you should check if the suggestion is legally possible.

 

If you need to send auto reply to all customers, instead send it from Zendesk, set an autoreply in Outlook. So when customer send a message to Shared mailbox or to Agent, the answer will be sent automatically. Now, we have to set a way to forward messages intended for Zendesk to Shared Mailbox. Here we could make an inbox rule to forward message and it would be applied if the message has some words in Subject. For example, all messages with “Support” in Subject will be sent to Shared mailbox. Now, for all messages from clients and agents which come to Shared mailbox, we could set a forwarding rule from Exchange Online Center (mail flow rule) which will send the messages to Zendesk.
