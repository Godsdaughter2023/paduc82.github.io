Activation in Windows 10

Activating after reinstalling Windows 10

How you activate Windows 10 after reinstalling it depends on how Windows 10 was originally installed on your device.

　

If you activated a free upgrade to Windows 10 or bought and activated Windows 10 from the Windows Store, you have a digital entitlement for your device. This means you can reinstall the same edition of Windows 10 that your device has a digital entitlement for without entering a product key. During reinstallation, if you’re asked to enter a product key, select Skip. Windows 10 will automatically activate online after the installation is complete.

　

If you previously installed and activated Windows 10 using a product key, you’ll need to enter the product key during reinstallation.

a. user say:

Hi,

Thank you for contacting Microsoft Community.

I suggest you to follow the below methods and check if it helps.

Method 1: I suggest you to run System file checker. System File checker (SFC) scan is done to check if there are any corrupted system files that could be causing this issue. Kindly follow the below given steps: 

  1. Press Windows key + X, select Command prompt (Admin) to bring up elevated Command prompt.
  2. In Command prompt type sfc/scannow and press enter.
  3. Restart the computer.

Method 2: If no corrupted system files were found in the previous state, then I suggest you to try the following steps to employ the DISM command lines: 

  1. Open command prompt by pressing Windows key + X and selecting “Command prompt admin”.
  2. In command prompt, type the following commands and press Enter after each command:
    • DISM.exe /Online /Cleanup-image /Scanhealth
    • DISM.exe /Online /Cleanup-image /Cleanupimage
    • DISM.exe /Online /Cleanup-image /Restorehealth
  3. Close the command prompt and restart the PC and check if it works.

Hope this information was helpful and do let us know if you need further assistance. We will be glad to assist.

Start button > Control Panel > classic view > Regional and Languages > Keyboard and Languages > Change Keyboards > under Installed services, scroll to find ZH Chinese (Traditional, Hong Kong S.A.R.) > click that > click ADD > scroll to find Chinese (Traditional, Hong Kong S.A.R.), click the +sign in front >click open Keyboard > put checkmark on whichever one you want > click OK when done.
