bob jones aka: cybertaz say:

I'm at a loss to understand why you can't "work on this spreadsheet now"?...

The Cmd+Click only is necessary if you want to have the very first or very last sheet tab brought directly into the viewing area. IOW, "just pressing on the arrows" buttons, alone, continues to scroll the tabs, but does so only 1 tab at a time. The Cmd+Click is a convenience option which is quite useful if there is a significant number of sheets in the workbook. The Cmd+Click technique amounts to nothing more than holding the Command key [ ⌘ ] while executing a regular left click.

You still can use only the arrow buttons... It's just that they can be difficult to get to if there isn't enough space between the bottom edge of the window & the Dock or the display screen. You might try reducing the height of the Excel window a bit.

Note that you also can right-click the arrow button device to produce a list of sheets by name. Select the sheet name to go directly to that sheet.
