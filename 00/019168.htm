stanleygooch say:

Date macro Word 2016

I want to create/modify a date macro in Word 2016 to insert the date in the following format:

Monday, 1st May 2016

andreas killer say:
Sub Test()
  Selection.TypeText _
    Format(Now, "dddd, ") & _
    EnumerateOrdinal(Day(Now)) & _
    Format(Now, " mmmm yyyy")
End Sub

Function EnumerateOrdinal(ByVal Number As Long) As String
  '1st,2nd,3rd,4th,5th,6th,7th,8th,9th,10th,11th,12th,...
  EnumerateOrdinal = Number & Mid$("thstndrdthththththth", 1 - 2 * _
    (Abs(Number) Mod 10) * (Abs(Abs(Number) Mod 100 - 12) > 1), 2)
End Function

greg maxey say:

Interesting function Andreas.

Just for anyone interested, one way to make the numerical date ordinals superscript:

Sub TestII()
Dim oRng As Word.Range
Dim oDate As Date
  oDate = InputBox("Enter date", "DATE", CStr(Date))
  Set oRng = Selection.Range
  With oRng
    .Text = Format(oDate, "dddd, ")
    .Collapse wdCollapseEnd
    .Text = EnumerateOrdinal(Day(oDate))
    Do While IsNumeric(.Characters.First)
      .MoveStart wdCharacter, 1
    Loop
    .Font.Superscript = True
    .Collapse wdCollapseEnd
    .Font.Superscript = False
    .Text = Format(oDate, " mmmm yyyy")
    .Font.Superscript = False
  End With
End Sub
