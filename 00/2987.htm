You don't need Norton any more than you need most of the other addins.

 

Business Connectivity Services AddIn <== not necessary for most users, unless you use  sharepoint.

 

iTunes Outlook AddIn <== not needed if you don't sync apple device with outlook

 

Microsoft Exchange Add-In <== not needed if you don't have an exchange mailbox

 

Microsoft Outloook Social Connector <== this adds the people pane and facebook/linkedin integration

 

Microsoft SharePoint Server Colleague Import Add-In <== not needed if you don't use sharepoint

 

Microsoft SharePoint Workspace Proxy for Outlook Add-In <== not needed if you don't use sharepoint

 

Norton AntiSpam Outlook PlugIn <== if outlook's spam filter (set on high) meets your needs, this is not necessary

 

OneNote Notes about Outlook Items <== do you use onenote?

 

Outlook Change Notifer <== companion to the iTunes addin. Not needed if you don't sync with an apple device

Microsoft VBA for Outlook Add-in <== adds VBA support. Necessary if you ever want to use macros.

Personally, the Norton AntiSpam Outlook Plugin would be the first one I'd kill but then again, I wouldn't let any Norton product near my computer at all ;-)

 

The iTunes Outlook AddIn and the Outlook Change Notifier are both from Apple. If you do not sync Outlook with any iPod, iPhone or iPad, you won't need it.

 

The WinZip and Norton ones you know already.

 

The rest are default add-ins that ship with Outlook. They are designed not slow down Outlook in any way if you do not need them. However, you are free to disable them as with POP3 in a home situation, you won't have a direct for most of them.

  • OneNote Notes about Outlook Items
    In case you also use OneNote to save additional notes and have them linked with Outlook or create Task items in OneNote. For instance: http://www.msoutlook.info/question/721
  • Microsoft VBA for Outlook Add-in
    In case you want to use macros. It will only load when it is needed (which is why it is inactive now).

neha singh msft say:

Hi DanWtheITguy,

 

Three types of admins with the following roles can receive the password reset email:

 

  1. Password reset admin
  2. User management admin
  3. Global admin

 

If there are all the three kinds to admins available in your organization, then only Password reset admin will receive the password reset email.

 

If User management admin and Global admin are available, then User management admin will receive the email.

 

And if, only admin in your organization is the Global admin, then Global admin will receive the password reset email.

 

Hence, by priority Password reset admin is the first person to receive the email then User management admin and then comes global admin.

 

If you want to receive the password reset email, then please make sure there no User management admins and Password admins in your organization.

 

Let me know if you have any question.

Neha
