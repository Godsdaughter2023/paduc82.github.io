Yes there is. All you need is a USB stick with a storage capacity of 8GB or more and another Windows PC that's working running Windows 7 or higher.

This page is the homepage for the media creation tool. Click "Download tool now" under Create Windows 10 installation media.

In the tool, You can pick any language or bitness you want. 64-bit is suggested these days though. For edition, you may only see "Windows 10" but this will give you both Windows 10 Home and Pro options during install.

The most important thing is that you install the same edition you had before. If you had Home, install Home. If you had Pro, install Pro. If you are asked for a product key, you can click "I don't have one" and Windows will activate automatically once it's online later.

If you have changed certain hardware, such as the motherboard or network card, then Windows cannot automatically activate and you will need a product key. Changing a hard drive will not change the hardware ID used to determine if this PC was previously activated.

  • Nero software by Nero Inc
  • Roxio Creator software by Sonic Solutions
