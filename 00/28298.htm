teylyn say:

Hello,

Vlookup goes like this: =Vlookup(lookup value, lookup table, table column, match type)

The #N/A error means that Vlookup cannot find the value you are looking for.  Some reasons for this include:

- wrong data type. Your Vlookup formula has a number as the first argument, but the lookup table has text in the first column (or the other way around). Check that the data type is the same for the lookup value and the first column of the lookup table.

- spaces before or after the lookup value. Make sure that there are no space characters leading or trailing either the lookup value or the values in the first column of the lookup table. 

- if the fourth parameter is 1 or TRUE or has been omitted, the lookup table must be sorted ascending by the first column. Otherwise there may be wrong results, including false errors for #N/A

Check these three things and if the Vlookup still produces an error, post a screenshot with your data and your exact formula.

julie fka profjulie say: Yes! It is possible to keep your three Outlooks synchronized with the same Live account without running into problems!

To keep all devices and computers in sync with the same Windows Live account, all you need to do is make sure you've got the Outlook Hotmail Connector installed and configured for the same Windows Live account on both computers.  Then changes you make to Computer A will sync up to your Windows Live Account and, in turn the changes will be synced to Computer B and your phone.  If you make changes to Computer B or your phone, the changes will be synced to your Windows Live account and in turn to the other devices.

Simultaneous syncing isn't exactly how it occurs, but syncing is pretty fast to the phone.  For the computers, you can press the Send/Receive option or press the F9 button to initiate a manual sync.  Check the Send/Receive options for your Windows Live account in Outlook to make sure you've got them set up to accommodate your needs (syncing to and from Windows Live occurs whenever a Send/Receive command is issued, either manually or automatically in whatever time intervals you've set for your Windows Live account).

julie fka profjulie say:

You can sync contact and appointment information from any Windows Live, Google, or Exchange Server accounts you use and you are not limited to only syncing to one of them.  I don't use Gmail for much, so syncing to Gmail is not important to me, but I currently sync 3 different windows live accounts to my WP7 phone.

I still use Outlook on my computer to manage the bulk of my appointments and contacts and I use the Outlook Hotmail Connector to sync the data from Outlook to my Windows Live account.  Once the data is on my Windows Live account it syncs to my phone.
