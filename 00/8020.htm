ericyang_msft6286 say:

Hi Saab,

Try this command:

$Users = Get-MsolUser -all | Where-Object ({$_.Licenses | Where-Object {$_.AccountSkuId -eq "<TenantName>: EXCHANGEARCHIVE"}).Length -gt 0 }

$Users | Export-Csv -path <Path>

Replace <TenantName> and <Path> with your custom value. EXCHANGEARCHIVE is the SKU ID of Exchange Online Archiving license. You may need to first check the SKU ID in your Office 365 tenant using Get-MsolAccountSku. If the SKU ID of the Exchange Online Archiving license is different from this value, replace it with actual value.
