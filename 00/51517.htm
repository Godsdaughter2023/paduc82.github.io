There are several methods you can use to add/edit files to your Player Library:
 
• Monitoring folders.  When you start the Player for the first time, it automatically searches certain default folders included in the Music, Pictures, Videos, and Recorded TV libraries on your computer. If you ever add or remove files from these media libraries, the Player will automatically update its available media files. You can also include new folders in your Windows libraries from other locations on your computer or from removable storage devices.
 
• Add media files that you play. When you play a media file on your computer or from a removable storage device, the file is automatically included in your Player Library so you can access it directly from the Player. And if you have access to remote media files stored on other computers on your network, you can change the Player's settings to add these remote files to your Player Library when you play them. The Player doesn't automatically add a file that you play from removable media, such as a CD or DVD.

Add items to the Windows Media Player Library

To add or edit media information in Windows Media Player
 
You might have items in your Windows Media Player Library that have missing or incorrect media information, such as song titles, track numbers, artist names, album titles, or genres. Incorrect information might have been added when you ripped the tracks from the CD or because you used another program to rip your CDs. Whatever the reason, the Player relies on media information to organize the items in your Player Library, so it's beneficial for all items in your Player Library to have accurate and complete media information.

Hi Parameswara,

Note: Using Third Party Software, including hardware drivers can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the use of Third Party Software can be solved. Using Third Party Software is at your own risk.


For Windows Live Mail Calendar, refer to the following Microsoft article.

 

Looking for Windows Calendar?

 

To create a message using Task Scheduler, refer to the following steps.

 

Open the Start Menu, type taskschd.msc, and press Enter.

If prompted by UAC, then click on Yes.

In the left pane, click on Task Scheduler Library folder to select it, and then click on Create Task in the right pane.

In the General tab, type in a Name and Description for this task to help let you knows what it's for.

If you wanted to change which user this message reminder will display for other than the default of your user account, then click on the Change User or Group button and select a user or group.
NOTE: The steps are numbered in the order to follow for how to select a different user or group. The Users group would be for all users.

Click on the Triggers tab, and click on the new button.

If not already selected, then select on a schedule.
NOTE: You could select what event you would like or works better for you instead, but I'm just using this one as an example. If you select a different option, then you will have different setting to set instead of the ones in steps GA-7B below.
A) Select how often you want the message reminder to display.
B) Depending on what you selected in step gA, select and set when (ex: date and time) you want the message reminder to display.
C) When finished, click on OK.

Click on the Actions tab, and click on the new button.

In the Action drop down menu, select Display a message.

Type in a Title and Message that you want displayed in the message reminder, and click on OK.

Click on the Conditions tab, uncheck the Stop if the computer switches to batter power box, and uncheck the Start the task only if the computer is on AC power box.

Click on the Settings tab, check the Run task as soon as possible after a scheduled start is missed box, and check the If the task fails, restart every: box. 

Click on OK.

You will now see this task listed in the Task Scheduler Library folder.
NOTE: You will always be able to edit or delete this task.

Close Task Scheduler. You're message reminder will now display for the user or group and when you have it set for in this task.

 

 

Hope the information provided is helpful.

There are no standard folders called Sent or Trash in Windows Live Mail,  so perhaps this is an IMAP account using the names of the folders on the IMAP server. If so, visit the account in webmail and remove the unwanted folders there.

¡firedog say:

Thanks. I understand that these are folders you created under the account. There's nothing wrong with that, of course, and there's not much difference in behaviour between folders created under a POP3 account and ones created under Storage folders. There is a world of difference, however, with folders created under an IMAP account, which is why I wondered whether you hadn't actually given the names of IMAP folders.

I take it that you have now created the folders you want under Storage folders and moved messages into them from the folders under the POP3 account. Once a folder is empty, you should be able to click on it in the folder pane and then press Shift-Delete on the keyboard to delete it.
