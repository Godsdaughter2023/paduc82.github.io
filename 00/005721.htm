julie fka profjulie say: First backup the contacts on your mobile device just in case the sync does not work as expected.  You can do this by copying the PIM.VOL file (found in the root folder of the device).

rsrc say:

Count only cells within a range with red text from condition formatting; using office 2016 for Mac

Range is C4 to E??? and growing. text colours are Automatic black, blue, green & red.

Have tried the following Function and Formula by Mike H but it appears to just counts the number of cells within a test range D4 to D26 (23 cells) containing 6 known blue text coloured cells.

The formula gives an answer of 23 not 6.

Below is a copy the Formula and Function copied from "Mike H." Microsoft Community Post modified to suit test range.

Does the Function have to be changed to suit Excel on a Mac or the Formula?

     =CountColour(D4:D26,D8)         In Test Range Cell D8 has blue text.

Function CountColour(rng As Range, clr As Range)
Application.Volatile
Dim c As Range
For Each c In rng
    If c.Font.Color = clr.Font.Color Then
        CountColour = CountColour + 1
    End If
Next
End Function
