Try the below methods and see if it works for you.

Method 1:

Run the hardware and device troubleshooter.

http://windows.microsoft.com/en-US/windows7/Open-the-Hardware-and-Devices-troubleshooter

Method 2:

Check if the Bluetooth service is running.

a) Press Windows Key  + R at the same time, type services.msc in run box and press Enter.

b) Search for Bluetooth support service.

c) Check if it is started, if not Right click on the service and click on Start.

Method 3:

Uninstall and re-install the drivers for your Bluetooth device and then check for the issue.

Follow the steps to uninstall and reinstall the Bluetooth device drivers from device manager:

  • Click Start, then type ‘Device Manager’ (without quotes) in the Start search box and press Enter.

Note: If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

  • Locate the Bluetooth device by expanding the categories.
  • Now right-click on the Bluetooth device driver and click on Uninstall.
  • Restart the computer. This will let the computer automatically detect the device drivers on restart of the computer.

Note: If there is yellow question marks in device manger, please repeat the steps from step c.

Try to refer the steps provided in the following Microsoft article and see if it helps.

Connect to Bluetooth and other wireless or network devices
http://windows.microsoft.com/en-US/windows7/Connect-to-Bluetooth-and-other-wireless-or-network-devices
