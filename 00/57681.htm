No personal recommendations on recovery tools.

I'm not sure what kind of an external hard drive that you have.  If it's a USB drive powered from the USB itself, then it is unlikely to be a power supply problem.  If you have an external power supply that plugs into the external disk drive, then usually there is an indicator light on the drive that may indicate if power is present.  You can also put a meter across the power plug. 

Also consider the possibility that sometimes the bearings on the disk go bad and it stops spinning (or slows down significantly).  If that's the case, some people have luck cooling their disk drive in the freezer overnight which shrinks the bearings and can allow the drive to start spinning again for a short while until it warms up again.  If this is the case, you're looking at purchasing a new disk drive.

If you think the drive is spinning properly, you might try booting your machine to a free Live Linux CD (such as Knoppix ).  Since Linux uses a completely different set of drivers and is fairly resistant to crashes, you might get a different result.  Couldn't hurt to try.  To create a Live Linux CD, just download the ".iso" file and use your CD Burning program's "Burn Image" feature to create the bootable CD.

vijay a. verma say:
Assuming your video is saved, try following. If your video is not saved, don't try step 2 and 3.

1. Run Windows Store Apps Troubleshooter
Type troubleshoot in Windows Start Search box > Click Troubleshoot > Scroll down to bottom and click Windows Store Apps > Click Run the Troubleshooter and follow on-screen directions.

2. Repair / Reset Photos App
Windows Key+i> Apps > Locate Photos by Scrolling Down > Click Advanced Options > Click Repair
If Repair doesn't work, click Reset

3. If Reset doesn't work, then you should uninstall using Powershell and reinstall the Photos App
Windows+X > Click Windows Powershell (Admin) > Copy the below command and right click in Powershell to paste it

get-appxpackage Microsoft.Windows.Photos | remove-appxpackage

Now, you need to reinstall it
Use below Powershell command to install

Get-AppxPackage -allusers Microsoft.Windows.Photos | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”

(If you encounter problem installing it, you can always install it from Microsoft Store)
