Hello Killer,

Thank you for bringing this to our attention. Before going into anything to extensive, Like port forwarding, I recommend trying out the Alternate Port Selection settings on the Xbox one.

  1. Press the Xbox button to open the guide.
  2. Select Settings.
  3. Select All Settings.
  4. Select Network.
  5. Select Network settings.
  6. Select Advanced Settings
  7. Select Alternate port selection
  8. Select Manual

From here on out it will be trial and error. You would select a port from the drop down list and then Test your NAT type from network settings until you find a port that leaves you on open NAT.
