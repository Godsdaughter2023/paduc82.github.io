Option 1:

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell H2 and copy down - this extracts uniques per month (full month name) specified in cell G2:

=IFERROR(INDEX($A$2:$A$50, MATCH(0, NOT($G$2=TEXT($D$2:$D$50,"mmmm"))+COUNTIF($H$1:$H1, $A$2:$A$50&""), 0)), "")

=IFERROR(INDEX($A$2:$A$50, MATCH(0, NOT(MONTH($E$2)=MONTH($D$2:$D$50))+COUNTIF($F$1:$F1, $A$2:$A$50&""), 0)), "")

Right-click the sheet tab.

Select View Code from the context menu.

Enter or copy the following code into the worksheet module:


Private Sub Worksheet_Change(ByVal Target As Range)
    On Error GoTo ErrHandler
    If Not Intersect(Range("A1"), Target) Is Nothing Then
        Application.EnableEvents = False
        Me.Unprotect
        Range("B2").Locked = (LCase(Range("A1").Value) <> "yes")
    End If
ExitHandler:
    Me.Protect
    Application.EnableEvents = True
    Exit Sub
ErrHandler:
    MsgBox Err.Description, vbExclamation
    Resume ExitHandler
End Sub
