  • Set Feedback & Diagnostics to FULL
  • Verify MS account

2016 Office - stop automatic 2nd line indent - Microsoft Community

tish onsager say:

2016 Office - stop automatic 2nd line indent

How do I stop my Word document from automatically indenting on the 2nd line in Office 2016?
suzanne s. barnhill say:

Is it just the second line or every line after the first in a paragraph? If the latter, it is a "hanging indent." You can remove this either in the Paragraph dialog or by dragging the Hanging Indent marker on the horizontal ruler. If the former, see http://word.mvps.org/FAQs/Formatting/UnaccountablyIndented.htm.

Is this affecting a single paragraph or every paragraph in a document? If the latter, you can use Ctrl+A to select the entire document and then remove the indent.

Does this affect the default empty paragraph in new documents? If so, you will need to correct the problem in the Normal template. To do this, open the Paragraph dialog (using the dialog launcher in the Paragraph group on the Home tab or by right-clicking in the text and choosing Paragraph...), set Special to (none), and click Set As Default. In the ensuing dialog, select the radio button for "All documents based on the Normal.dotm template?"
