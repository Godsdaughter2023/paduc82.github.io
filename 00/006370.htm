john_sur say:

Hi,

I've had this problem with importing data that varies in size.  For example, if I imported lets say 100 lines in the first import I would have a file size of 20k.  The second data import would have 500 lines of data.  File size would jump to 40k.  The third data import would have 100 lines again.  I would check the file size and it would still be 40k, exactly the same size as the second import.  What I found out was that if I selected Clear All from the Clear option on the ribbon and saved the workbook, it would release any space that was not being used.  In other words, cells that no longer had data in them.  Before you do the clear make sure that all of the cells that have data in them and that need to be deleted/cleared is selected.  So in your code you might want to use Selection.Clear instead of Selection.ClearContents.  I'm assuming that you save the workbook after making changes.  The clear works only if the workbook is saved.  It has always worked for me.  Just a thought.

gerald mal say:

Hi Mukesh,

If you can sign into the site above but cannot get the account to sync on the phone, there are a few things that could be causing this issue.  Try/check the following and see if the account sync issue can be corrected.

  • While signed into the site above, go to "Security Info."  Once you have retrieved the security code, look over this page for the option to create an app password.  If you are able to do this, please do so and then go to your phone > tap settings > emails + accounts > Microsoft account/Windows Live > Password > enter the app password you created.
  • If you do not see the option to create an app password, the next thing to do is go to the phone > tap settings > time + date > turn off automatic settings > set the time, date and time zone manually.  Once you do this, manually sync the account by tapping and holding it and selecting sync.
  • Finally, if the issue continues, go to settings > emails + accounts > go to the server name field and enter each of m.hotmail.com, blu-m.hotmail.com, bay-m.hotmail.com and dub-m.hotmail.com.  After you enter each, sync the account to see if anything changes.

If the issue continues after trying the other suggestions in this thread and mine above, provide the following details and we will gladly provide the next steps to try:

  • Make and model of your phone?
  • The exact error code, please include the word for word message that is included with this code.
  • The current OS on your phone, you can find this by tapping settings > about > more info.
