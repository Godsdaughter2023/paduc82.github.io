    kaushalls say:

    The issue is with your GPRS/WAP settings. First, check if there are any settings defined for WAP services for your operator under Settings > Connectivity > Destinations > WAP Services.

     

    If there is a setting for your operator/service provider under WAP Services, try using it when FOTA prompts you to select a connection instead of selecting the option under "Internet"

     

    If you have no access points defined under WAP Services, You will have to contact your Operator/Service Provider for the correct WAP Settings and create a new Access Point under WAP Services. You can also try Nokia's Settings service (http://europe.nokia.com/A4144889). Select Advanced Settings >  Select 5800 XM as Phone model > Select WAP/GPRS as the Service and follow the rest of the instructions and save it once you receive it.

     

    Once the WAP access point has been setup, check for updates using FOTA, it will then prompt you to select a connection, then select the option under WAP Services to use for FOTA
