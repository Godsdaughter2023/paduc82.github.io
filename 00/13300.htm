Hi Laxminarayana,

Here are the answers to your questions:

The "Get-Distributiongroup” cmdlet retrieves groups with group type as “Distribution List” and “Mail-enabled Security Group”. Is there any way to retrieve Groups of other types (i.e. “Security Group” and “Office 365 Group”)?

--The Get-MsolGroup command

Is it possible to manage these groups (status as ‘Synced with active directory’) using Powershell cmdlet?

--No, you need to make the change in your local AD since they are synced with AAD Connect.

Is there any way to filter the output of “Get-MsolGroup” cmdlet so that I can have only the groups with status as “In Cloud”?

--Yes, use the command to get In Cloud only groups: get-msolgroup | where {$_.lastdirsynctime -eq $null}

--We cannot tell from the command. Use this command in the article to get Office 365 group: https://support.office.com/en-us/article/Manage-Office-365-Groups-with-PowerShell-aeb669aa-1770-4537-9de2-a82ac11b0540
