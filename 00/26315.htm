cody c. say:

Hi everyone!

 

Thanks to everyone who sent installation logs for this issue.  There are two separate 14.2.4 installation issues that have shown up in this thread.

 

The first issue (and most common in this thread) stems from re-installing on top of existing installations. In general, don’t do this. Re-installing on top of existing installations can re-introduce files that should not be present. If you need to re-install, please be sure to remove Office 2011 as described on the following links:

 

Video | Uninstall Office 2011

Step-by-Step Instructions | How to completely remove Office for Mac 2011

 

One workaround for the 14.2.4 update that has helped some users is to temporarily move or rename the /Library/Automator folder. You can do this by following the steps below:

 

  1. With Finder open, choose Go > Go to Folder...

  2. Type in the following path and then click Go:
    /Library

  3. Find the Automator folder, select it, and then click the name of the folder again to Rename the folder. Name the folder Automator_Old

    Note    :     OS X may prompt you for your password. This is normal.

  4. Attempt to install the 14.2.4 update again.

  5. If the update installs correctly, rename the Automator_Old folder back to its original name: Automator

 

If that solution does not work and you know you have re-installed Office 2011 on top of an existing Office 2011 installation, you’ll need to remove Office 2011 properly and re-install. 

 

The second issue was some users changed the size of the /tmp/ directory on their machine which was too small for the update to fully install. Re-sizing the /tmp/ directory resolved the install issue for these users.


If you're still experiencing issues installing the 14.2.4 update, please create your own thread by choosing Ask a question... at the top of this page. Be sure to include details about which solutions you've already tried.

rohn007 say:

It is relatively easy to insert hyperlinks to locations inside of a document.   

As Paul has pointed out, a lot of what you want to do can be achieved using the default "Heading" styles in Word.  Actually, heading styles are probably my favorite feature in Word.

There a 9 levels of heading styles you can apply to use to organize your document.  Once you have applied Headings, you have several options for using them to jump around the document.

First, as you suggested, you can AUTOMATICALLY generate a TOC that includes hyper links to the headings in the document. But that is the "old school" way of using them.

Second, you can turn on the "Document Map".  This displays the headings in a "pane" on the right side of the window. You can click on the appropriate heading in the Map and the document will display the appropriate location.  You can right click in the document map and select how many levels of headings you display.

Third, you can use the Outline View. When you turn on Outline View, only headings are displayed. This compresses the document view for faster moving about in the document.  You can also click and drag headings to new locations in the document, taking all subordinate text and headings to the new location.

Hyperlinks from Headings to the TOC http://wordribbon.tips.net/T013059_Hyperlinks_from_Headings_to_the_TOC.html
A table of contents is a great way to help organize lengthy documents. In a default TOC, you can use each entry as a hyperlink to the headings in the document. Getting back to the TOC, though, can take a bit of work.

How to use the Document Map in Microsoft Word
The Document Map shows an overview of your Word document. It lets you navigate rapidly. It’s most useful when you’re working on a big document. Here’s how.
