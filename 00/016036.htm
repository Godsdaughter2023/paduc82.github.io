don cuthbert say:

It is normal and expected for Windows 10 on a new device not to activate without purchasing it, unless you can transfer your previous digital license to the new device. However, other than Personalization settings, Windows 10 is very useable without being activated and never seems to quit working like previous Windows versions did. Even though it is the same SSD and has not been reinstalled, it is now in a different device, so loses activation.

If the old PC was activated, the Activation page in Settings would still have shown "Windows is activated with a digital license". If you logged into Windows with a Microsoft Account, it adds the phrase "linked to your Microsoft account". Even if you had never logged into that PC with a Microsoft Account, it would still have a digital license for Windows 10. The license is on the specific device and allows for activation of a clean install of Windows 10. Once you connect to the Internet, the device's hardware is recognized by the activation servers as licensed for that specific Edition of Windows 10 (Home, Pro, Single Language, etc.) and is auto-activated, even if you do not use a Microsoft Account to login.

The auto-activation really has nothing to do with your Microsoft Account. It can be "linked to" it though, which helps if you change hardware and lose activation, by enabling you to reassign the license to the new hardware, such as when replacing a motherboard.

Do you see any reference to the previous PC in your Microsoft account list of devices at the link I posted above? I presume you would use the same Microsoft Account that you log into this forum with, since it shows "Insider", unless you use another account.

You can still launch the Activation Troubleshooter as I described above, after launching Command Prompt as Administrator. But the only chance of transferring the old digital license is through the Microsoft account record. Only stand-alone Windows software has a transferable license, so not a license that began as OEM Windows 7 or 8.

don cuthbert say:

Once a device has a Digital License for Windows 10 Pro, it will auto-activate Windows 10 Pro after a clean install, whether you use the public build or an Insider build. The license is for that specific device, so any other device will not activate, even if you use the same Microsoft Account to login to it. If a device does have a digital license, it will auto-activate even if you do not login with your Microsoft Account.

Stand-alone software includes the right to transfer to another device, even if it belongs to someone else, or if it has been upgraded to a new Windows version. However, most PCs are sold with an OEM Windows installed, which does not have transfer rights. So if the previous PC had been upgraded from OEM Windows 7 or 8, it is not transferable to another PC.

The Activation Troubleshooter is meant to restore activation if you change a major component, such as the motherboard, so that it looks like a different device to the Microsoft activation servers. It will not work between different types of device, such as between a laptop and desktop. I would think that there may be some chance to transfer to a completely new PC using the Troubleshooter, but not if the original Windows 7 or 8 that was upgraded was not purchased as Stand-alone software.

C:\ProgramData\Microsoft\OfficeSoftwareProtectionPlatform\Tokens.DAT

C:\Windows\ServiceProfiles\NetworkService\AppData\Roaming\Microsoft\SoftwareLicensing\Tokens.data

C:\Windows.Old\ProgramData\Microsoft\OfficeSoftwareProtectionPlatform\Tokens.DAT (if you did an in-place Windows Upgrade)

After you extract the Product Key, make sure to write it down and keep it in a safe place. Burning a copy of the installation file to a CD would also be a good idea if you don’t have the installation disk.

 

NOTE: OEM (factory install and activated) licenses use a different key than the one reported.  Only the factory can use the PK from the original factory activation. You would have been provided with a separate piece of paper with a Product key for reactivation.
