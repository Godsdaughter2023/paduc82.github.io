shekhar s say:
Hi,
 
 
Check if your computer supports one of the hibernation options. To do this, go to the start menu and type Command in the search box.
In the results, right click and select “Run as Administrator”. When the command prompt opens up, type
 
powercfg -a
 
which shows you all of the available types of hibernation that your hardware supports.
 
If your hardware does support hibernation but still not working, either the hibernation option has been disabled or the Hibernation File Cleaner has been deleted by the Disk Cleanup Utility. Both these options are easily fixed using the powercfg utility again. To turn hibernation back on, type powercfg -h on.
 
Note If the computer does not support the hibernation feature, you cannot enable the feature from the Windows.
I would recommend you to check with the BIOS manufacturer or computer manufacturer to verify if BIOS is required to be updated.
 
Important:
Modifying BIOS/complementary metal oxide semiconductor (CMOS) settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of BIOS/CMOS settings can be solved. Modifications of the settings are at your own risk.
