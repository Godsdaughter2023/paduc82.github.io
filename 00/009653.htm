graham mayor say:

These are document/template lock files. Every time a document is opened or a template accessed a (normally hidden) lock file is created in the same folder as the document/template. They should be removed whenever the document is closed, but they can remain if Word closes abnormally. See http://www.gmayor.com/what_to_do_when_word_crashes.htm
