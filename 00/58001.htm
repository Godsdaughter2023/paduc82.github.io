a. Have you turned off the User Account Control (UAC) settings, by chance?

This could be possibly due to turning off UAC. Once UAC is turned off, Once UAC is off, people that log on as administrator will always have the permissions of an administrator.

So, running the program directly would be equivalent to running as administrator.

User Account Control (UAC) is a feature in Windows that can help you stay in control of your computer by informing you when a program makes a change that requires administrator-level permission.

Turn on UAC and check if the issue reappears.

Refer the link below for details on UAC settings:

What are User Account Control settings?
http://windows.microsoft.com/en-us/windows7/What-are-User-Account-Control-settings

Note: Microsoft does not recommend its users turning off UAC settings for security reasons.

If you have only one account with administrative privileges, it is not possible to change the same account to a standard user account directly.

To change it into a standard user account, you’d have to first create another account with administrative privileges and log in with that particular account to change your current account into standard user account.

Windows requires at least one administrator account on a computer. If you have only one account on your computer, you can't change it to a standard account.
