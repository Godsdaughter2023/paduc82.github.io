Insert cover page moving all pages and headers down - Microsoft Community

michaelglennwilliams say:

Insert cover page moving all pages and headers down

Hi,

I have a doc with a header on each page. All are marked as "not linked to previous" because the header for each page is unique to the text on that page. 

When I try to insert a cover page, the cover page replaces page 1, so the header from page 1 is no on the cover page, the header from page 2 is now on page 1, etc.

I want the headers to "follow the pages they are on" and for word not to count the cover page as a page.

Is there a way to do this?

I've tried adding a page break on the first page, but again the headers do not follow their pages.

tina zhou msft say:

Hi Michael,

Please make sure you perform the following steps to insert page breaks and headers, and then a cover page (you may create a new document to check the outcome) :

1. Click Layout > Breaks > Next Page.

2. Double click the new page's header > click Link to Previous to unselect it > change the header in the new page.


3. Click Insert > Cover Page.


If the issue still occurs, please provide the following information to help troubleshoot:

1. Double click the header on a page, capture a screenshot of the first page, the second page and the third page.

2. Can you get the expected results if you create a new document with my steps?

3. Your Mac OS version (e.g. 10.12.4) and Word version. For Mac OS version, from the Apple menu, choose About This Mac. The version number appears beneath “macOS” or “OS X.” For your Word version, choose Word in the top menu bar > About Word and please capture a screenshot of the window.

Thanks,

Tina
