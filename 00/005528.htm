In Excel, how to run a macro when a specific cell value automatically - Microsoft Community

jamilvalente say:

In Excel, how to run a macro when a specific cell value automatically becomes greater than 100?

In an Excel spreadsheet automatically fed data via RTD, when cell A1 become greater than 100, run a specific macro. How can I do this?
andreas killer say:
IMHO the Change event will not work for you, because that event only fires if you manually write the value into the cell.

But you can use the Calculate event for that, this event fires every time when a formula is calculated.

Right-click on the sheet tab
Choose "View Code"
Paste this code 

Private Sub Worksheet_Calculate()
  If Range("A1") >= 100 Then
    MyMacro
  End If
End Sub
Sub MyMacro()
  MsgBox "Hello World"
  'Reset A1, otherwise we're called again at the next calculation
  Range("A1") = 0
End Sub

bob jones aka: cybertaz say:
No offense, but changing margins to force content to print in a certain location based on one paper size then printing on different size stock can be guesswork at best :-) This is especially true when there is a large difference between the document specs & the actual paper size.

IMO, the best approach is to specify the actual Paper Size for the Word document itself: Format> Document> Page Setup> Paper Size. If there is a 4x6 dimension listed for your printer, select it there. If not, use the Manage Custom Sizes dialog to create a 4x6 custom size... but not all printers will accommodate every size. You might want to consult the documentation for the printer to determine what [if any] custom size stock is supported. I'm afraid I can't tell from your message whether you've done so or not, but it doesn't sound like it.

Once you specify the 4x6 size there, set the margins accordingly. When you print, the settings should carry over with no further adjustment needed, but it wouldn't hurt to go to Paper Handling in the Print dialog's Copies & Pages menu to confirm that the correct Paper Size is indicated.

It's also possible that the printer requires that certain sizes of stock be fed in specific ways. If the stock is fed differently some of the content is bound to miss the intended target.

Another consideration is updates: Office 2011 should be at version 14.3.1 & it might be a good idea to check the printer mfr's web site for driver updates.

hansv mvp say:

Select File | Options.

Click Display.

If the check box "Show white space between pages in Print Layout view" is clear, tick it.

Click OK.

 

Alternatively, move the mouse pointer to the top of the document window, just below the ribbon, until the mouse pointer becomes a symbol of two pages with up/down arrows. Then double-click.

Gradient background on Word 2016 for Mac

I'm trying to apply a gradient fill effect to a document in Word. I'm using Word 2016 on a mac and I can't for the life of me figure out how to do this. When I open the "Page Color" icon it doesn't give me any options for fill effects. Any help would really be appreciated.

bob jones aka: cybertaz say:

The Fill Effects option (which included Gradient among others) for Page Color was dropped in the Mac Word 2016 release. Only solid fills are available. You can express your opinion by clicking the ☺︎ at the right end of the Ribbon in Word.

You can improvise by drawing a full page rectangle in the Header/Footer layer & applying a gradient fill to it. If you have any content in the Header or Footer apply Behind Text Text Wrapping to the rectangle.
