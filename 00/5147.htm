zack barresse say: all data where no cell in column "EmploymentStatus" shall equal "Terminated".

                                                            

Private Sub UserForm_Initialize()

    Dim Table As ListObject
    Dim EmploymentStatus As ListColumn
    Dim ExcludeCount As Long
    Dim Index As Long
    Dim IncludedValues As Variant
    Dim ExcludedValues As Variant

    Const ExcludeStatus As String = "Terminated"

    Set Table = Worksheets("LBEmployee").ListObjects("tTable01")
    If Table.DataBodyRange Is Nothing Then Exit Sub
    Set EmploymentStatus = Table.ListColumns("EmploymentStatus")

    IncludedValues = Table.DataBodyRange.Value

    For Index = LBound(IncludedValues, 1) To UBound(IncludedValues, 1)

        If IncludedValues(Index, EmploymentStatus.Index) <> ExcludeStatus Then

            ExcludeCount = ExcludeCount + 1
            If IsEmpty(ExcludedValues) Then
                ReDim ExcludedValues(1 To 5, 1 To 1)
            Else
                ReDim Preserve ExcludedValues(1 To 5, 1 To ExcludeCount)
            End If
            ExcludedValues(1, ExcludeCount) = IncludedValues(Index, 1)
            ExcludedValues(2, ExcludeCount) = IncludedValues(Index, 2)
            ExcludedValues(3, ExcludeCount) = IncludedValues(Index, 3)
            ExcludedValues(4, ExcludeCount) = IncludedValues(Index, 4)
            ExcludedValues(5, ExcludeCount) = IncludedValues(Index, 5)

        End If

    Next Index

    If Not IsEmpty(ExcludedValues) Then
        Me.ListBox1.List = Application.Transpose(ExcludedValues)
    End If

End Sub

t. valko say:

This is a follow up question, based on a question that I asked this morning. OK, enough preamble.

How can I sum a filtered list with a condition and a wildcard?

 

Here’s the scenario:

Filter items in ColumnA, starting in Row13; sum values in ColumnM, based on matches in ColumnT.

=SUMPRODUCT(SUBTOTAL(3,OFFSET(A13:A240,ROW(A13:A240)-ROW(A13),0,1)),--(T13:T240="MTM*"),M13:M240)

 

This function is giving me $0. I’d expect to see $12,702,781.24.

What am I doing wrong?


SUMPRODUCT doesn't work with wildcards.

In SUMPRODUCT, MTM* is the literal text string MTM(asterisk).

You can replace:

--(T13:T240="MTM*")

With either:

--(LEFT(T13:T240,3)="MTM")

--(ISNUMBER(SEARCH("MTM",T13:T240)))

Note that with the SEARCH method, it will return TRUE/1 if MTM is found *anywhere* within the cell.

bob jones aka: cybertaz say:

I'm not sure why you're being bounced around from updates to fonts to Outlook... None of which have anything to do with the issue :-)

The fundamental problem is that Office 2016 has broken the capability of AutoTextList fields. More precisely, AutoTextList fields rely on customization of contextual menus & Office 2016 has done away with the internal machinery that supports any type of menu customization. The field is being recognized as existing but the contextual menu cannot be populated. That's why you're getting the restricted popup but without the content you're expecting to see.

I don't know of any means within the program to serve as an alternative until such time as customization capabilities are expanded. Perhaps you'll then be able to add controls to the Ribbon or contextual menu customization may be implemented. Resetting the Device

Sometimes there is a need to reset your Pocket PC. Resetting the
device is known as performing a “soft” or “hard” reset.

Performing a Soft Reset

A soft reset stops all running applications, but does not erase any
programs or saved data. Be sure to save any unsaved data within
a running application before performing a soft reset.
Perform a soft reset to:
■ Stop all running applications
■ Install a new application
■ Restart the Pocket PC if it stops responding

To perform a soft reset:

1. Locate the recessed Reset button on the bottom of your iPAQ
Pocket PC.
2. Use the stylus to lightly press the Reset button.
The Pocket PC restarts and displays the Today screen.


Performing a Hard Reset

Perform a hard reset only when you want to clear all
user-installed settings, applications, and data from RAM
(Random Access Memory), where data and applications you have
loaded are stored.

and programs not saved in iPAQ File Store.

To perform a hard reset:

1. Press and hold down the Calendar and iTask buttons.
2. While holding down these buttons, use the stylus to
lightly press the Reset button on the bottom of the iPAQ
Pocket PC for about two seconds.



3. When the Pocket PC screen begins to fade, release the
Calendar and iTask buttons first, and then remove the stylus
from the Reset button.

and iTask buttons for more than two seconds, the battery
disconnects. To restart the device, either plug the device into the
AC Adapter or press the Reset button again.
After performing a hard reset, you may need to reinstall
applications stored in iPAQ File Store to restore shortcuts and full
functionality.

4. The Pocket PC resets and powers on.

To return your iPAQ Pocket PC to factory settings after
performing a hard reset:

1. Tap Start > Programs > File Explorer > My Device.
2. Tap the iPAQ File Store folder to open it.
3. Tap Edit > Select All.
4. Tap and hold the selected files, then tap Delete.
5. Tap Yes to delete all file in iPAQ File Store.
