cmdrkeene say:

Hotmail is not changing your file names... you're just accidentally downloading the wrong file.  .ASHX files are web site programming files (ASP.net Web Handler File).  What's happening is that you are probably right-clicking on the attachment link, then choosing to "save target as".

Instead, you need to click the link.  This will direct you to the .ashx file, which runs a server process and then in turn delivers the file to you, which will be the correct format.

The reason you may be seeing partial success when you download the wrong file is that Word (or whatever other program) is trying to open the file and is following along, sending the query back up to the server and getting the resulting file.
