Disk Defragmenter error about Chkdsk.

Trying to run Disk Defrag, when I hit Analyze or Defragmenter I receive an error window that says: "Disk Defragmenter has detected that Chkdsk is scheduled to run on the volume (C:). Please run Chkdsk /f." I have ran the Chkdsk program, restarted, but still receive the same message. Can anyone help??????

I am running Windows XP.

ann marie scrimshaw say:

Hi rooneytune.  You have posted in the Windows 7 Forum however I will try to help anyway.  I am sure a Moderator will move this topic when it is reviewed.

You may have what is known as a "dirty bit" set on your volume.  This can happen for various reasons, one being an improper shutdown.

Lets find out.  Go to Start > Run and type:

cmd.exe

and click ok.  A command prompt will open.  Type the below command after the prompt and hit enter:

fsutil dirty query c:

Does it report a dirty bit?

If so, type the below command and hit Enter:

chkntfs /x c:

and reboot.  This will stop chkdsk from running when you next restart.

Once you have rebooted, open a command prompt again and type the below command and hit Enter:

chkdsk c: /f /r

reboot again and chkdsk should run all 5 stages and hopefully clear the dirty bit.

If a dirty bit is not reported, just try the last step I posted.  Good luck.
