Critical Path in Project Online - Microsoft Community

mamot say:

Critical Path in Project Online

How do I show critical path for project online?
dale howard [mvp] say:

MomoT --

In Microsoft Project, open a project and apply the Gantt Chart view.  Click the Format tab to display the Format ribbon.  In the Bar Styles section of the Format ribbon, select the Critical Tasks checkbox.  Critical tasks will be formatted with red Gantt bars, while non-Critical tasks will be formatted with blue Gantt bars.

In the Project Center page in PWA, click the name of a project.  In the drill-down menu at the top of the Quick Launch menu, click the Schedule link.  Click the Task tab to display the Task ribbon.  In the Data section of the Task ribbon, click the View pick list button and select the Tasks Tracking view.

Hope this helps.
