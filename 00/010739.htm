dirk goldgar say:
However if I want to include these bracket symbols for example: [ ] then it do not provide any result. Same goes for - " ' these symbols too. How can I include this symbol in the above equation?


Now it gets more complicated.  The characters - and ' were in the original set you specified, so the original expression should work fine for them.  The double-quote character (") can easily be added to that set, but it has to be doubled up to allow it to be specified within a quoted string:

 

    WHERE YourColumn Not Like "*[!-A-Z0-9 \/'&""]*"

 

However, the bracket characters [ and ] are a problem, because they can't be used inside the bracketed list.  I think if you need to accept those characters, we need to remove them from the string to be tested before using the Like operator.  Try something like this:

 

    WHERE Replace(Replace(Nz(YourColumn), "[", ""), "]", "") Not Like "*[!-A-Z0-9 \/'&""]*"

 

Note two things:

 

(1)  The use of the VBA Replace function will inevitably make this expression much slower to evaluate, so that will have an impact on the query performance.

 

(2) Given that this is going to be relatively inefficient anyway, it would be better to write your own function to validate the string, and just call that from your query.  The meaning would be clearer.

'Select VBA Menu Item Tools -> References ->
'Check the box against Mictrosoft PowerPoint nn.n Object Library and OK.
'(Where nn.n is the version of Office)

Sub CreatePDFfiles()

    Dim PPapp As PowerPoint.Application
    Dim PPPres As PowerPoint.Presentation
    Dim invpath As String
    Dim cor_file_name As String
    Dim DestinationPPT As String
   
    Sheets("printing").Select
   
    invpath = Range("g3")       'g3 = "C:\Users\UserName\Documents\Power Point\"
    cor_file_name = Range("i8")       'i8 = "My pdf file name"
    DestinationPPT = "C:\Users\UserName\Documents\Power Point\Test Save PP.pptx"
   
   'Initialize PowerPoint Object Library

    'to alleviate multiple instances of PowerPoint, best to attempt GetObject first

    'and if not already open then use CreateObject

   

    On Error Resume Next       'Will error on GetObject if not already open
    Set PPapp = GetObject(, "PowerPoint.Application")
    On Error GoTo 0
   
    If PPapp Is Nothing Then   'If Nothing then GetObject failed because PowerPoint not already open.
        Set PPapp = CreateObject("PowerPoint.Application")
    End If
   
    PPapp.Visible = True
   
    Set PPPres = PPapp.Presentations.Open(Filename:=DestinationPPT)
   
    PPPres.ExportAsFixedFormat invpath & cor_file_name & ".pdf", ppFixedFormatTypePDF, ppFixedFormatIntentPrint
   
    PPPres.Close
   
    PPapp.Quit
   
    DoEvents    'Can error on next code line if not finished Quitting before setting variables to Nothing
   
   'Cleanup
    Set PPPres = Nothing
    Set PPapp = Nothing
   
End Sub
