petrib say: If you use Bluetooth pairing, the passcode has to match on both devices.

If you are pairing with a PC, pick whatever numbers you want, and use the same numbers on both devices.

If you are pairing with a Bluetooth headset or some other Bluetooth device without a display or keypad where to enter a passcode, then the device has a fixed passcoce. The fixed passcode can be always the same for all similar devices (same model) from the same manufacturer, or it can vary.

If it is fixed, then it is probably 0000 or 1234 or something else that that the device's User Guide/Manual has printed.

If it is changing, then it is also provided in the User Guide or a separate leaflet, or sticker ot something. It could also be be the beginning or ending part of the devices serial number.

The Bluetooth pairing passcode is only needed once when the pairing happens (it is used to generate encryption/decryption keys for the communications between the devices, and then they are no longer needed/used). So, if you're pairing with a PC, you don't need to write down the passcode for future use; you just need to remember it long enough to enter it once on both devices.

I’d like to explain that only the onmicrosoft.com addresses cannot be removed. All addresses on your custom domains are always manageable by an administrator.
