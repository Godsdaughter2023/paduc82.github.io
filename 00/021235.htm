Hello SpeedyRice,

Welcome to Skype Community forum!

If your microphone is not working in Skype but works on other application and Windows audio set up, you may try these steps:

Completely uninstall Skype

  1. Quit Skype.
  2. Press Windows key and R on your keyboard. Type appwiz.cpl in the Run dialog box and click OK.
  3. Search Skype and uninstall it.
  4. Press Windows key and R on your keyboard again and type %appdata% and click OK.
  5. Open the Skype folder and delete the shared.xml folder.
  6. On the same folder, look for your Skype name and double click it then delete config folder.
  7. Restart your device.
  8. Download and install the latest version of Skype.
  9. Sign in to your account and try to make a voice call or the echo test service.

If your microphone still won't work, try to plug in a different microphone and check if that will work. If it works, it is possible that the other microphone is not compatible with Skype.
