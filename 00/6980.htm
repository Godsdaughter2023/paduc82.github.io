Opening & editing pdf document in word - Microsoft Community

felicity baker say:

Opening & editing pdf document in word

Hi

Is it possible to open a pdf document and edit it in word. If so how?

Thanks

suzanne s. barnhill say:

If you have Word 2013 or above, you have the feature Microsoft calls "PDF reflow." This means that you can open and edit some PDF files.

  • Any PDF created by using Save As PDF (or its equivalent) in Word can be opened by Word.
  • PDFs created by scanning a document can be opened if they were scanned as editable text.
  • By default, most scanners will create only an image file (usually a JPEG), which is just a picture of the document, not editable text. In order to open and edit this in Word, you would have to use OCR (Optical Character Reader) software to convert the picture to editable text. Some scanners come with at least a "lite" version of some OCR software, and you can also use OneNote and Snagit to convert an image to text (though not very well).

If a document is primarily text in a single column, Word will do a pretty good job of rendering the PDF. If it contains wrapped graphics, multiple columns, tables, frames/text boxes, or other such complications, what you get will probably be a mess.
