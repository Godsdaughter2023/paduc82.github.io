I appreciate your efforts taken to resolve the issue. However, there could be several reasons why you are unable to see your router on Windows 8.1 system. The firewalls (Windows or third party) generally obstruct the network connection, even after being disabled. It is, therefore, sometimes suggested to completely remove the security software with the help of its removal tool to resolve the connection-related issue. However, you should reinstall/update the security software after troubleshooting to reinstate system's security.

Other factors include a third party software conflict hindering the connectivity, corrupted wireless network adapters or misconfigured Windows settings for network discovery.

In order to troubleshoot the exact cause of the issue, I would suggest you to try the following methods and check if the issue persists.

Method 1:

Run the network adapter troubleshooter to diagnose possible causes with the wireless network adapter and check if it helps.

a. Press “Windows key + W”.
b. Type “Troubleshooting” without quotes in the Search box and hit enter.
c. Select “Network and Internet” from the Troubleshooter Window.
d. Select “Network Adapter” under Devices option.
e. Follow the On Screen instructions.

If the issue still persists, refer to the additional steps in the following article:

How do I fix network adapter problems?

If the issue still exists, try the next method.

Method 2: 

I would suggest you to change to the following options for Windows 8.1 system on the network in the advanced sharing settings (Right click on the network icon in the taskbar and click Network & Sharing Center > Change Advanced Sharing settings):

- Turn on Network Discovery

- Turn on file and printer sharing

- Turn on sharing so anyone with network access can read and write files in the public folders

- Turn off password protected sharing

Now, try to map the USB drive following these steps:

  1. Press Windows key + E.
