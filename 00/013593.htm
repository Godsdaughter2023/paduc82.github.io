Hi,

OK I thought of a way. This ARRAY formula, see below for how to enter it, will return the maximum date in col H if there are only 2 dates for a particular reference month but if there are 3 it will return the middle date.

=LARGE(IF(MONTH(H10:H26)=MONTH(E15),IF(YEAR(H10:H26)=YEAR(E15),H10:H26)),SUMPRODUCT(--(TEXT(H10:H26,"MMYYYY")=TEXT(E15,"MMYYYY")))-1)

We can now put that into 2 SUMIF formula to sum before that date and >= to that date. This sums column A < the date

=SUMIF(B10:B26,"<"&LARGE(IF(MONTH(H10:H26)=MONTH(E15),IF(YEAR(H10:H26)=YEAR(E15),H10:H26)),SUMPRODUCT(--(TEXT(H10:H26,"MMYYYY")=TEXT(E15,"MMYYYY")))-1),A10:A26)

and this for >= to the date

=SUMIF(B10:B26,">="&LARGE(IF(MONTH(H10:H26)=MONTH(E15),IF(YEAR(H10:H26)=YEAR(E15),H10:H26)),SUMPRODUCT(--(TEXT(H10:H26,"MMYYYY")=TEXT(E15,"MMYYYY")))-1),A10:A26)
