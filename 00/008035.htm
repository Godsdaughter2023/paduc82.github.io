tychicusthefriendly say:

To dual boot you first have to create a separate partition.  For the Windows 8 preview you needed at least 20 GB so to be safe on this install I made a partition of 30 GB to give me some head room. 

Create a partition...

  1. Go into Disk Manager, right clicking the drive you wish to shrink and selecting Shrink Volume.
  2. Enter in how much you want to shrink the volume by in MB. For me, when I did 30 GB it worked out to 30720 MB.
  3. Click okay and you should have a new partition that says 'unallocated something rather'

Now you have to create a 'disk' with the .iso image and make it bootable.  I have Windows 8.1 so I just right clicked the .iso file and selected 'Burn to Disk'.  Unfortunately it is not that easy in Windows 7 so you will have to do a little digging on the Internet to see how you create a bootable disk from a .iso image.  I do know that you do not have to pay for a program to burn an .iso image.  People will try and get you to buy Nero or some other burning software and it is not needed. Unfortunately I don't remember how. 

Once you have a bootable 'disk' either put it in your optical drive or a usb plug and restart your computer.  When you restart your computer you might have to go into the BIOS to change the boot order.

When your computer boots from the 'disk' and you are presented with the install page select costume.  When it asks what drive to install on select the partition you created.  BE CAREFUL you do not select a different drive other wise you will write over your data.

To get rid of Windows 10/the partition go back into Disk Manager, right click the partition you want to delete and select delete.  Then you right click the main partition on that drive and select Expand.  Now you just have to make sure you expand it the same amount you shrunk it. After that it should be as if nothing happened.

There are two different ways to create a hyperlink in Excel. 

- click Insert > Hyperlink and navigate to the saved html file on the hard drive. This option can be tedious and time-consuming and may not be suitable for your needs. The hyperlink applies to the cell and you can type in any text you like

- use the Hyperlink() function in a formula. If all files are in the folder C:/Desktop/saveddesigns, and they all end with .html, then you can just create a list of the file names in column A, like

savedpage1

savedpage2

savedpage3

etc. In another column, let's say column B, you can then use

=HYPERLINK("C:/Desktop/saveddesigns"&A1&".html","click to open")

and copy down.

That date sounds suspiciously like one the PC might revert to when the battery for the system clock device fails and the time is automatically reset by the BIOS to the base date defined within its programming.

I'd be checking out the system clock itself and noting whether such a thing might have occurred either due to a failing battery or possibly a transient error situation where the clock was temporarily scrambled and reset as a result.
