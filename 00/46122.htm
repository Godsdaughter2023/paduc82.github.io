mann_manohar say:

Hello Qosaikadadha,

Welcome to Microsoft Community.

As per the description I understand that your printer continues to print the same test page when the command is given for the fresh pint.

I would appreciate if you can provide us the following information to help us understand the issue better.

  1. What is the make and model of your printer?
  2. Does it show the fresh page in the print preview?
  3. Have you checked connecting the printer to a different computer? Does the problem persists with other computer as well?

Sometimes when you print it may get stuck in the waiting queue. This can be due to multiple issues such as a delay in communication or an error. In the end the print job may print, however, there are cases where the print job remains and since it is at the top of the list other documents will no longer print.

Each time you hit the “Print” button Windows saves that as a print job. Thus, hitting the “print” button five times causes Windows to saves that as five print separate print jobs.

There is a way to manually clear the print queue so that you can make sure no print jobs are keeping other jobs from printing. The video below will show you how to manually clear the print queue.

Method 1: Stop print spooler services and start again.

Step 1: Stop the print spooler service:

  1. Press Window Key + R.
  2. In the search box type in service.msc.
  3. Look for the print spooler in the services running.
  4. Right click on print spooler and choose stop from the context menu.
  5. Restart the computer.

Step 2: Cancel the all pending print jobs:

  1. Go to Control Panel, Printer, double click your printer.
    You should be able to see the print queue with the pending print jobs.
  2. Click Cancel All Documents
  3. Wait for the printer to respond and now restart the printer and check.

Step 3: Start the print spooler service:

  1. Press Window Key + R.
  2. In the search box type in service.msc.
  3. Look for the print spooler in the services running.
  4. Right click on print spooler and choose start from the context menu.

Method 2: Turn off the printer and Turn-on.

I would recommend you to manually turn off the printer and turn on from the power button.
