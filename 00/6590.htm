always learning... say:

jay freedman say:
You can format the display of fields by adding "switches" to the field codes. Graham Mayor has a comprehensive page about field switches at http://www.gmayor.com/formatting_word_fields.htm. For formatting any kind of date field, look at the topic "Date-Time Picture (\@) field switch" near the bottom of the page. For example, if you want the field to show a date like "May 30, 2012" then your field code should look like
        { savedate \@ "MMMM dd, yyyy" }
while for a display like "2012/05/30" the code should be
        { savedate \@ " yyyy/MM/dd" }

If the result is still showing as zeros, select the field and press F9, or right-click it and choose Update Field. If that still doesn't change it, try closing and reopening the document.
