tom-s say:

Hi Catherine,

I did some trials and noticed (like RonRosenfeld also mentioned) that when the end date falls on a weekend then the formula needs amending to NETWORKDAYS-3 rather than NETWORKDAYS-4. I didn't test that out thoroughly but sufficient to believe it valid.

The formula I gave simply amended your own, adding an IF function to calculate what the subtraction should be (3 or 4), based on whether the end date falls on a weekend.

Using WEEKDAY(date,2) returns Mon-Fri as day numbers 1-5, so the IF part of the formula translates as: IF end_date is 1,2,3,4 or 5 (Mon-Fri) then subtract 4, else subtract 3.

Not sure if that helps. Let me know if you need more.

 

*** EDIT ***

I have assumed that what you want your formula to achieve, in words, is this:

For any reference date (15 or last day of month), all days up to but not including the 4th workday after the reference date attract a value of zero, and then each successive workday attracts a value increasing by 1, with sucessive non-work days attracting a value increasing by zero .

 

Here's an alternative way of expressing the formula:

=IF(NETWORKDAYS(WORKDAY(_JAN31,4),E2)<1,0,NETWORKDAYS(WORKDAY(_JAN31,4),E2))

vijay a. verma say:
Below is the Macro -

1. This macro will print all mails in a given folder. Pages from 1 to 2.
2. You can create a blank folder and copy (not move) the mails for whom you want to take batch printouts. Later on, you can delete mails from this folder.
3. Open Outlook > ALT+F11 to open VB Editor
4. Copy and paste the below code.
5. Whenever you want to print, open the folder where you copied the mails and select a mail.
6. Press ALT+F8 to open macro window and run the macro named PrintMultiMails
7. First test it for 2 to 3 mails in that folder. 

Sub PrintMultiMails()
    Dim olItems As Outlook.Items
    Dim olItem As Outlook.MailItem
    Set olItems = Application.Session.GetDefaultFolder(olFolderInbox).Items
    For Each olItem In olItems
        Call PrintFirstSecondPages
    Next olItem
    Set olItem = Nothing
    Set olItems = Nothing
End Sub
Sub PrintFirstSecondPages()
    SendKeys "%F"
    SendKeys "p"
    SendKeys "r"
    SendKeys "{TAB}"
    SendKeys "{TAB}"
    SendKeys "{TAB}"
    SendKeys "1-2"
    SendKeys "{ENTER}"
End Sub
