macropod say:

After spending an hour or so carving up my footer i have now hit a snag, part of the footer is a merge field which is longer than the max char length and so is truncated.

I do not have control over the merge fields themselves and cannot create new ones to seperate the resulting info in segments.

Other than that, this solution worked perfectly.

Many thanks to all.


Hi PlotGuru,

 

The character limit for a paicture switch is 63. You can get around this by embedding the expressions within the switch in another field. Typically, you might use a QUOTE field for this. Thus:

{={NUMPAGES}-{PAGE} \# "{QUOTE "A very long text string for True cases"};;{QUOTE "A very long text string for False cases"}"}

Yoy can include additional fields (eg MERGEFIELDS) within the QUOTE fields. With this approach, you could include a whole page of text in the field, including pictures etc. in the switch expressions.

I have sent you back the ListaArchivos workbook, saved as a Macro Enabled Workbook containing the following macro that will create documents like the one that you sent to me

Sub Merge()

Dim wdApp As Object

Dim wdDoc As Object

Dim wdRange As Object

Dim bstartApp As Boolean

Dim strPath As String

strPath = ActiveWorkbook.Path

Dim strArchive As String

Dim xlSource As Workbook

Dim xlSheet As Worksheet

Dim i As Long

Set wdApp = GetObject(, "Word.Application")

If Err Then

    bstartApp = True

    Set wdApp = CreateObject("Word.Application")

End If

On Error GoTo 0

With Sheets("Hoja1").Range("A1")

    For i = 1 To .CurrentRegion.Rows.Count - 1

        strArchive = .Offset(i, 0)

        Set xlSource = Workbooks.Open(strPath & "\" & strArchive)

        xlSource.Sheets("Hoja4").Range("A3").CurrentRegion.Copy

        Set wdDoc = wdApp.Documents.Add

        With wdDoc

            .Range.Text = "Esta es una prueba de correspondencia " & strPath & "\" & strArchive & vbCr

            Set wdRange = .Range

            wdRange.collapse wdCollapseEnd

            wdRange.Paste

            Set wdRange = .Range

            wdRange.collapse wdCollapseEnd

            wdRange.InsertAfter vbCr & "Aquí termina"

            .SaveAs strPath & "\" & Left(strArchive, InStr(strArchive, ".") - 1)

            .Close

        End With

        xlSource.Close SaveChanges = False

    Next i

End With

If bstartApp = True Then

    wdApp.Quit

End If

Set wdApp = Nothing

End Sub
