doug robbins - mvp office apps & services (word) say:

You can move the First Line Indent Marker independently of the Hanging/Left Indent Marker, but you cannot move the Hanging and Left Indent Markers independently of each other.

If you click on the square portion of the Hanging/Left Indent Marker, and move them, the First Line Indent Marker will also move by the same amount and in the same direction.

If you click on the upper triangular portion of the Hanging/Left Indent Marker and move them, the First Line Indent Marker will stay in its position and not be moved.

jay freedman say:
Go to File > Options > Advanced, scroll down to the Display section, and check the box for "Show all windows in the Taskbar". Now each document will open in a separate window, and you can drag each window to a separate monitor.

When that box is unchecked, all documents open in the same window (as child windows). With that setting, it's still possible to have two documents on separate monitors, but the procedure is different. First, you have to have Windows configured to have the monitors as "extended" (behaving like one big desktop). With the Word window neither maximized nor minimized, drag the sides so that half of the window appears on one monitor and the other half on the second monitor. In the View ribbon, click the View Side By Side button, and the two open documents will be distributed to the separate monitors. A bonus in this case is that you can use synchronous scrolling (or turn it off in the View ribbon).

Hello,

Thank you for your post.

In Yammer there are two type of group Internal and External groups.

You cannot restrict the users to create the internal group but you can restrict all the users even admins to create the External groups.

To stop the creation of external group you can follow below steps:-

First create a Rule in Exchange Online admin centre by following below steps.

  • Login to Exchange online admin centre.
  • Navigate to Exchange Admin centre> Rules
  • Click on + 
  • Click on "create new rule"
  • Click on "More option"
  • We want to disable the email with sender address as "*@yammer.com " and the recipient located outside the organization.
  • "Do the following"  > Deliver the message to hosted Quarantine.
  • Click on Save.

Note: This will disable only the external messages from Yammer and will not affect other external messages

Once the Exchange online rule is configured we can configure Yammer to user the Exchange Transport rule ETR to block external messaging.

  • Please follow below steps to enable "Exchange Transport in Yammer"

  1. Please Login to Yammer as Global or verified Administrator
  2. Click on the gear icon on left and click on Network admin
  3. Click on security settings.
  4. Check Mark on "Enforce your Exchange Online Exchange Transport Rules (ETRs) in Yammer"
  5. Click on Save.

https://support.office.com/en-us/article/Control-external-messaging-in-a-Yammer-network-with-Exchange-Transport-Rules-f8fd6403-c8f3-4307-9230-65304d6000d9

  • Once the Exchange Online Transport rule is created you can check after 30 Minutes and see if you are able to create an external group from yammer. If you are not able to create an external group from yammer than the rule is successfully configured.

If this doesn't work then kindly create a service request with Yammer support to stop users to create External group.
