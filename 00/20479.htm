joeu2004 say: pikejohn wrote:
Subject:  When I try and enter 1.e3 in a cell,it always goes to the scientific mode when I want it to stay in the general mode.

Are you saying that you would like to see 1000 instead of either "1.e3" or 1.00E+03, and you want the value to continue to be interpreted as a number, not text?

If so, I'm afraid that you must format the cell as Number.  Unfortunately, that means you must select a fixed number of decimal places, which will be the same (unless you change it manually) even for cells that require greater precision to display meaningful digits.

General format tells Excel to apply whatever heuristics it wishes to decide how to display the value.  We have no control over it.  Apparently, Excel chooses to display 1.e3 in Scientific format even if a simpler form (e.g. 1000) would fit in the column width.  That's just the nature of the beast.

Arguably, alternatively you could create an event macro that would alter Excel's choice of format.  Here's a bare-bones example to demonstrate the principle:

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
If Target.Count = 1 Then
    If IsNumeric(Target) And InStrRev(Target.NumberFormat, "E+") > 0 Then
        Application.EnableEvents = False
        Target.NumberFormat = "General"
        Application.EnableEvents = True
    End If
End If
End Sub

milos trbojevic say:

use VBA code

Sub SendAllDrafts()
' Send the messages in the Drafts folder (ignore any subfolders)
If MsgBox("Are you sure you want to send ALL the items in your default Drafts folder?", _
    vbQuestion + vbYesNo) <> vbYes Then Exit Sub
Dim fldDraft As MAPIFolder, msg As Outlook.MailItem, intCount As Integer
Set fldDraft = Outlook.GetNamespace("MAPI").GetDefaultFolder(olFolderDrafts)
intCount = 0
Do While fldDraft.Items.Count > 0
    Set msg = fldDraft.Items(1)
    msg.Send
    intCount = intCount + 1
Loop
If Not (msg Is Nothing) Then Set msg = Nothing
Set fldDraft = Nothing
MsgBox intCount & " messages sent", vbInformation + vbOKOnly
End Sub
