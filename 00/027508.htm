Problem with Excel Chart - Microsoft Community

nancy jayne say:

Problem with Excel Chart

In an Excel 2010 workbook, I have a data table on a worksheet and the chart of that data on another worksheet.  I want that chart to remain unchanged, but I must filter the data table to show only a portion of it.  How can I get the chart to remain unchanged when I change the data table?

Protecting the sheet doesn't keep if from changing as the data changes.  Copying the chart to another worksheet doesn't protect the chart.  The chart remains the same until I save it or print preview it, then the chart disappears off the page.

I have spent hours on this project and have completed the poject a total of six times and I still cannot get it right.

 

Thanks for any help!

biddu say: Hi,

There is a simple solution to this.

You can duplicate the sheet containing the data table by rightclicking on the sheet name -> Move or copy..  -> Check create a copy and click OK.

Now link your chart with this table by right clicking on the chart -> Select Data -> Select the entire table on the duplicate sheet.

After linking the data, hide the sheet by right clicking on the duplicate sheet name ou and click on Hide.

Now, you will be able to filter the data on the original data table without risking any changes to the chart.

Hope this helps!

rich007a say:

An alternative / possible improvement to this excellent idea from Biddu, is to do this:

  • Insert a new worksheet.
  • Copy the table of data fromt the original data sheet.
  • Go to the blank sheet and Paste Special As Links.
  • Now plot from the new table (link your chart with this new table by right clicking on the chart -> Select Data -> Data Range tab -> Select the entire table on the duplicate sheet).

 

Now everything references one table (the original one) so if any of the data is changed, everything else is changed, but if you hide rows of data (which is essentially what filtering does), the new sheet you made still references the hidden cells and so your plot still shows all the data.

Hope that helps.

Cheers
Rich
