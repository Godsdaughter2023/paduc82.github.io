The confusion comes because you need to use the same table as both parent and child, and you want to return only the distinct (unique) LearnerID values.  Try this (after adjusting field names as appropriate):

    SELECT DISTINCT Endorsements.LearnerID, ConcatRelated("Code","Endorsements","LearnerID=""" & [LearnerID] & """") AS CodeList
FROM Endorsements;

Note that the above assumes that your LearnerID field really is text, as your example implied,

akash_nath say:

Here is modified code:

This will add a new worksheet called PT Directory and save all pivot table details in Column A

Sub qqaazz()

ThisWorkbook.Worksheets.Add
ActiveSheet.Name = "PT Directory"

SRow = 1

For Each sh In Sheets
    s = "sheet:  " & sh.Name
    s1 = ""
    For Each pt In sh.PivotTables
        If s1 = "" Then
            s1 = "pt address: " & pt.TableRange1.Address & vbCr & _
            "pt name: " & pt.Name & "; " & "pt source data: " & pt.SourceData
        Else
            s1 = s1 & vbCr & "pt address: " & pt.TableRange1.Address & vbCr & _
            "pt name: " & pt.Name & "; " & "pt source data: " & pt.SourceData
        End If
    Next
    If s1 <> "" Then
        Worksheets("PT Directory").Range("A" & SRow).Value = s & "; " & s1
    Else
        Worksheets("PT Directory").Range("A" & SRow).Value s & "; " & "no pivot tables"
    End If
    SRow = SRow + 1
Next

End Sub

Display Values in Cell Formula instead of References - Microsoft Community

kevinpc06 say:

Display Values in Cell Formula instead of References

Hello,

Is it possible to display values in cell formula instead of references?  It would be helpful when doing research in spreadsheets.

bernie deitrick say:

The Evaluate Formula feature is sometimes not specific enough. You can do the same manually.

Press F2, click on the formula entry area, or double click on the cell to enter edit mode. Then select the cell reference(s) and press F9. You can select any part of the formula that results in a returned value to see what that value is, very helpful for troubleshooting formulas. Just make sure that you use ESC instead of Enter so that you don't permanently change the formula when you want to stop, or press Ctrl-Z if you accidentally press Enter.

For example, with the bolded part of the previous line selected prior to pressing F9, the next line will appear (Note:  This formula is array entered, but the { and  } disappear when in edit mode)

=SUM(IF(A2:A5="Bernie",B2:B5)) 

=SUM(IF({"Bernie";"Not Bernie";"Bernie";"Not Bernie"}="Bernie",B2:B5))

=SUM(IF({"Bernie";"Not Bernie";"Bernie";"Not Bernie"}="Bernie",{1;2;3;4}))

=SUM(IF({TRUE;FALSE;TRUE;FALSE},{1;2;3;4}))

=SUM({1;FALSE;3;FALSE})

4

Note that if you don't select a calculable sub-part before pressing F9, you will get a message "The formula you typed contains an error."
