john korchok say:

Here's how to reassign a key combination that's already taken:

  1. In Word, choose Tools>Customize Keyboard.
  2. In the Press new keyboard shortcut field, press the key combination that is already assigned. Then check which command has it in the area just below that field. In the case of F4, I get Currently assigned to: EditPaste.
  3. Find the command that was named. EditPaste is under the Edit category, but if it's a more obscure command, you might have to change Categories to All Commands.
  4. Click on the command name in the right-hand column. Check the Current keys: field to see what shortcuts are associated with that command.
  5. Select the shortcut you want to use, then click on the Remove button.
  6. Now you can assign your preferred command to the shortcut you removed and it will work as expected.

FAQ: Microsoft Interactive Classroom add-ins for PowerPoint and - Microsoft Community

rich porter - microsoft say:

FAQ: Microsoft Interactive Classroom add-ins for PowerPoint and OneNote

Microsoft Interactive Classroom add-ins for PowerPoint and OneNote enhance classroom presentations by increasing student participation, helping teachers assess student comprehension, and enabling students to personalize the information presented.

During a presentation, teachers can: 

  • Poll students with multiple choice, true/false, or yes/no questions.
  • Distribute the lesson to students with OneNote
  • Include real-time ink and text annotations.

Students can respond to poll questions through their individual OneNote notebooks or hand-held clickers, and the results display in the presentation. When the presentation is over, students keep the annotated presentation and notes so they can keep track of what they need to learn.

 

Q: How much does it cost?

A: Microsoft Interactive Classroom is available as a free download, found here.

 

Q: How can I get support for the Interactive Classroom Add-in?

A: Questions about using the Interactive Classroom should be asked on the Interactive Classroom discussion board on Microsoft Education Labs.

 

Q: Which versions of PowerPoint or OneNote work with the Microsoft Interactive Classroom Add-in?

A:The Microsoft Interactive Classroom Add-in is compatible with Microsoft Office PowerPoint 2007, Microsoft Office OneNote 2007, Microsoft PowerPoint 2010, and Microsoft OneNote 2010. A full list of requirements is available on the download page.

 

Q: Can I install the Microsoft Interactive Classroom Add-in if I have a previous version of the Interactive Classroom add-in already installed?

A:If a pre-release version of Interactive Classroom is installed on your computer, that version must be uninstalled before you install this release.

 

Q: How do students connect to a shared session?

A: Students can connect to a shared session by clicking the Join Session icon in OneNote 2007 or by clicking Join Session on the Academic tab in OneNote 2010.

 

Q: How do students know which session to connect to?

A: Students see a Join Session dialog when connected to the school network. The dialog contains the title of the presentation and the presenter’s computer name. The presenter can require an access key, which the student must enter to connect to the session.

 

Q: How does the student see the presenter’s slides?

A: When a student joins a session, each slide in the PowerPoint presentation is converted to an image and automatically added as a page in the student’s OneNote notebook.

 

Q: Where do I find the add-in after I install it?

A: The add-in appears as the Academic tab in PowerPoint and OneNote.

 

Q: I’ve installed the add-in but don’t see an Academic tab in PowerPoint or OneNote. Where is it?

A: Add-ins can be active or inactive in PowerPoint and OneNote. If the Interactive Classroom add-in is inactive, it will not appear in PowerPoint or OneNote after it is installed. To activate the Interactive Classroom add-in, do the following:

1.   On the File menu, click Options.

2.   Click Add-Ins, and then click Go next to the Manage dropdown.

3.   Select the Interactive Classroom checkbox, and then click  OK.

2. I can't rename a directory, as when I click on the directory it opens to allow the name change then immediately closes

jessen p say:

Hi Christian,

Thank you for posting the query in Microsoft Community.

1. Go to TaskManager 
2. Click CPU% tab to sort the highest usage task to top
3. Find the process with name *spool* or *spooler* in it

4. Kill it (end process)

Reason: If you have tried to print a page/selection on a browser (especially Chrome) and due to some reason if the printer is not connected or having connection issues, you might have tried to print more than once. This print process/spool usually do not show in the systray. No matter how many times you reboot, it may not go away. On every restart, it picks the spool again causing flickering (whether the printer connection is fixed or not). So try the procedure above, this should fix the problem.

Do get back to us for any issues pertaining in the future. We will be happy to help.
