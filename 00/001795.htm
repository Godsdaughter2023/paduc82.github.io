Creating a set number of labels per record with mail merge

I am attempting to create labels using mail merge. When I link to the excel sheet and select the reccords for the labels it either makes 1 label or a full sheet. I need to be able to make multiples of each record ( example: 3 labels of the first record, 6 of the second, 4 of the third and so on.) Is there a way to use NEXTIF or is it best to multiply the records in excel before it is merged into word?
doug robbins - mvp office apps & services (word) say:

You could selectively remove the «Next Record» field from the mail merge main document, but that is only going to work if the merge does not produce more than one sheet of labels.

If you will be wanting to produce more labels than will fit on one sheet, then multiplying the records in Excel is the thing to do.

Here is the code for an Excel macro that will insert the required number of copies of each record into the data source if the number of labels required for each is contained in one of the fields (columns) in the data source:

Dim i As Long, j As Long, k As Long, m As Long
With Worksheets(1)
   i = InputBox("Enter the number of the column that contains the quantities.")
   For j = .Range("A1").CurrentRegion.Rows.Count To 2 Step -1
       For k = 1 To .Range("a1").Offset(j - 1, i - 1).Value - 1
           .Cells(j, 1).EntireRow.Insert
           For m = 1 To .Range("A1").CurrentRegion.Columns.Count
               .Cells(j, m).Value = .Cells(j + k, m).Value
           Next m
       Next k
   Next j
End With
 -- Hope this helps.
