I suspect the error is related to the new Windows build that's running.  The error may be what's expected after installing a new OS version.  I don't know.  More importantly, I don't know what to do.  I don't know how to tell DISM to find the correct image.  And neither does free support at Microsoft Answer Desk, which has never heard of DISM and thinks I should reset Windows. I don't understand most of the TechNet articles well enough to trust myself. 

jaygourley say:

There is a simpler solution than the one above proposed by Ruth Buruga.  It's on Microsoft TechNet at https://social.technet.microsoft.com/Forums/en-US/8c57af52-0964-45c8-99e9-505e56e796ec/dism-online-cannot-find-image-what-do-i-do?forum=win10itprogeneral.  Specifically look at the clear and simple answer from "david hk129" for an explanation.  I started both this thread and that one.

Briefly, the issue is the result of updating to a new Windows build through Windows Update.  Windows Update installs the new build by downloading a .esd image.  But DISM needs a .iso image.

So first download the .iso image just as Ruth Buruga suggested.  Instead of creating a thumb drive, just mount the downloaded .iso image and then run dism.exe successfully by pointing it to the mounted image with the following command in a command prompt run as an administrator using your mounted drive letter where the "W" is.

Dism /Online /Cleanup-Image /RestoreHealth /Source:wim:W:\sources\install.wim:1 /limitaccess

If that runs successfully, you can then run sfc /scannow to repair the corrupted files.

royalcoach{2280} say:

Additional unidentified public network listed on my active networks.

Today I noticed that there is an additional network listed on my active networks. My home network which is my office- and today an unidentified public network appears below it. What is this and how do I remove it? or rather, how do I find out what or where it came from? thank you.

Lisa Lou

a. user say:

Hi,

Thank you for contacting Microsoft Community.

I would suggest you to refer to the mentioned steps and check if it helps:

1)  Use your mouse to click the Start menu and select the Control Panel.

2)  Click the "Network and Internet" button once the Control Panel loads and select the Network and Sharing Center.

3)  Click the "Manage network connections" tab.

4)  Look for the old network connections you wish to remove in the scroll box. Depending on how many connections you currently have in Windows Vista it is possible you'll need to scroll a bit to find the one you wish to remove.

5)  Click on the old connection you wish to remove.

6)  Click the "Remove" button.

7)  Confirm your selection by pressing "Yes" when asked whether you are sure you wish to remove the connection. You may also elect to press "Cancel" or "No" to return you to the previous screen and prevent the connection's removal.

8)  Note that you can also change network connections through the network wizard. Simply click Start, then "Connect To" and "Show all connections."

9)  Select the network connection you wish to change by right-clicking on it and selecting the option you wish. You can choose to "Disable" the connection, which would have roughly the same effect as completely removing it, rendering it inactive.

10) Press the "Delete" key after clicking on a network connection in the Network Connection window. This will completely delete the connection in the same way that removing it from the Control Panel would.

 

Hope this helps. Feel free to contact us again for any further assistance.
