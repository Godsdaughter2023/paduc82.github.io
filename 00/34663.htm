When you pin a program to the Taskbar a shortcut for that program is created and the shortcut is 'Locked' on the Taskbar. This prevents any changes to that shortcut. This has been true for the last several versions of Windows.

As a work around, you can create a shortcut for the program on the Desktop, Change the icon for that shortcut and then pin it to the Taskbar. The custom icon will remain.

A. Generally, an end user can upgrade or replace all of the hardware components on a computer—except the motherboard—and still retain the licence for the original Microsoft OEM operating system software. How to find/fix lost Minecraft Windows 10 Edition worlds

If you have lost your world (meaning when you open the game your world is missing) this may be helpful. THIS IS FOR WINDOWS 10 EDITION ONLY

First off open Minecraft and make a new world (give the world the same settings as you old one, as the settings will change after the process). Save and close and save the world, then close Minecraft.

Then, open files explorer. go: This PC/Local Disk (C:)/Users/User/AppData/Local/Packages/Microsoft.MinecraftUWP_8wekyb3d8bbwe/LocalState/games/com.mojang/minecraftWorlds
