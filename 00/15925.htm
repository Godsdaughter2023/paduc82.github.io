How to insert a footnote - Microsoft Community

janethomas(2) say:

How to insert a footnote

I want to insert a footnote.  Not  citation, just a comment.  Can you tell me how to do it in Word 365?  Also, I cannot find a help button.  Does it have one, and if so, where?
suzanne s. barnhill say:

To insert a footnote, press Alt+Ctrl+F to insert one using the default settings, or select the References tab and, in the Footnotes group, click Insert Footnote. This will still use the default settings, but if you want to change anything (for example, to use an asterisk instead of a number), click the dialog launcher (small arrow) in the bottom right corner of the Footnotes group to open the Footnote and Endnote dialog and select your desired settings.

The Help button is the question mark in the top right corner but F1 opens Help in any Windows application.

suzanne s. barnhill say:

Open the Footnote and Endnote dialog as described above and make sure that the "Start at" number is not set to 2.

If you find that "Start at" is set to 1 and you're still getting 2 for the first footnote, then there must be a previous footnote somewhere that is hidden. You can search for it.

  1. Press Ctrl+Home to go to the beginning of your document.
  2. Display Hidden text by clicking the Show/Hide ¶ button in the Paragraph group on the Home tab (or press Ctrl+*).
  3. Press Ctrl+H to open the Replace dialog and select the Find tab.
  4. In the "Find what" box, type ^f (the code for a footnote reference mark).
  5. Click Find Next.

If Find places the insertion point somewhere other than the footnote you inserted, even if there isn't a visible footnote reference, select a character or two on either side of the insertion point and Delete. Then retype the deleted characters. If there was a phantom footnote reference there, it will have been removed.

If that doesn't solve the problem, there may be a phantom footnote rather than a reference in the text. Switch to Draft view (View | Views | Draft), then switch back to the References tab and click Show Notes in the Footnotes group. Make sure there's no content before the note you inserted. You can also use Find to search for a footnote reference mark in the footnotes pane.

frank fei msft say:

Hi 1957sdp,

When you open a file in edit mode, you get exclusive editing capability in the file, and no other person can then open the file to edit. They can only view the spreadsheet in read-only mode. Even if you are the only person that uses the workbook, this can happen when you open a file in edit mode and leave it open on your work machine, for example, which means you are locked out of the file for editing on another computer at home or elsewhere.

Changing the behavior to open workbooks by default in read-only mode helps prevent these scenarios.

Besides, as I mentioned above, for your requirement " have a choice to open as read or edit", please submit feedback to our related team on UserVoice website.

You can check to see if Norton is the problem by downloading and running the Norton Removal Tool. (Make sure to have your product key saved/stored somewhere so that you can re-install if you want.)

 

You can check the other third party programs you have on your machine by doing a clean boot.

·         Click the Start Orb

·         Search msconfig

·         Click Continue on the User Account Control (If prompted)

·         Click the Services tab

·         Check the Hide all Microsoft services box

·         Click Disable all

·         Click OK

·         Click Restart Now

 

Once you have finished restarting, try to replicate the crash you were previously getting. If you aren’t able to replicate the crash then you just have to find which service was causing Windows Explorer to have problems.

·         Search msconfig

·         Click Continue on the User Account Control (If prompted)

·         Click the Services tab

·         Check the Hide all Microsoft services box

·         Check (Enable) the top half of the service boxes

·         Click OK

·         Click Restart Now

 

When the computer finishes restarting, make sure that the error is occurring again. If you are error free then the services you enabled are good to stay. Repeat the enabling process for half the remaining disabled services and continue this process until the problem returns. Then you will need to test the last batch of services that were enabled one by one to find which causes Windows Explorer to crash.
