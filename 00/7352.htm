Duplex Printing problem - in printing a two page Word document in - Microsoft Community

jwebber269@aol.com say:

Duplex Printing problem - in printing a two page Word document in landscape mode, the second page comes in upside down

Original title: Duplex Printing problem

in printing a two page Word document in landscape mode, the second page comes in upside down.  How do I fix this?

graham mayor say:
You should be able to set this in the print driver. Normally with portrait mode you would set horizontal duplex. If your document is set to landscape orientation then horizontal duplex still turns the page around the length of the paper so the second page is inverted. You need to change to vertical duplex.

In a document with mixed portrait and landscape pages, it might be simpler to print the portrait and landscape pages separately, though if your printer responds to PCL commands you may be able to achieve the correct duplex mode using print fields.

 A PRINT field is simply a type of field that allows you to send instructions directly to the printer.

There are 3 possible data values for duplex setting:  PRINT 27"&l0S" simplex PRINT 27"&l1S" vertical duplex . PRINT 27"&l2S"  horizontal duplex (bookformat)

The codes after PRINT are 27 (escape), double quote, ampersand, lowercase L, 0, 1 or 2, uppercase S and finally terminated with a double quote.

Thus to switch to vertical duplex you would insert the following field :
{ PRINT 27"&l1S" }
at the start of the landscape section, then
{ PRINT 27"&l2S" } at the start of the next portrait section.
and
{ PRINT 27"&l0S" }
to print on only one side of the paper.

suzanne s. barnhill say:
If the entire document is in landscape orientation, then you need to change a setting in your printer Properties. Depending on the printer model, it may be expressed as "Flip pages up," "Flip on short side," "Tablet," or the like.

try this...

Sub rename_sht()
For Each sh In Sheets
If sh.Name Like "*Purch" Then
sh.Name = Format(Date, "mmdd") & Space(1) & "Purch"
End If
Next
End Sub
