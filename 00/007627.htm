Attachments are not permitted in this forum. You can upload a file to OneDrive, make it public, and post a link here, or you can post a screen shot here using the Insert image button on the editor toolbar.

To get full control of bullets, you should define and use a bulleted style (see https://shaunakelly.com/word/bullets/controlbullets20072010.html), but there are some shortcuts:

  1. Go to File | Options | Proofing | AutoCorrect Options...
  2. Select the AutoFormat As You Type tab.
  3. Clear the check boxes for "Automatic bulleted lists" and "Set left- and first-indent with tabs and backspaces."

This should allow you to press tab and insert a hyphen without triggering any of the AutoFormat actions.

To use a bulleted style, you can modify one of Word's built-in styles. Pressing Alt+Shift+L will apply the List Bullet style, which has a round bullet at the left margin. You could modify that style to have a 1" indent, or you could use the List Bullet 5 style, which already has that indent. Since you would still need to change the bullet character, however (if you really want a hyphen), you might as well modify the List Bullet style. To do this, you will need to use the Define New Multilevel List dialog to specify the bullet to use and  the position of the bullet and the hanging indent, then link this list level with the desired style. suzanne s. barnhill say:

It appears that the problem is occurring when you update the TOC, which suggests that the problem is with the TOC styles. FWIW, I find that a standard, old-fashioned TOC field, inserted via References | Table of Contents | Table of Contents | Insert Table of Contents, is easier to work with and less prone to error than one of the "Automatic" TOCs in the gallery, which are inserted in a content control. You do have to provide your own "Table of Contents" or "Contents" title, but that's easy enough.

 

 

If you don't need the entries to be hyperlinked, you can can clear the check box for "Use hyperlinks instead of page numbers." Page numbers will still be hyperlinked, but it will be easier to modify TOC styles because, when you click in an entry, the style shown will be TOC 1 or TOC 2 instead of Hyperlink. TOC styles are set to update automatically, so any change you make to an entry at a given level should be propagated to the other entries and saved in the style. If this isn't happening, then something else is wrong.
