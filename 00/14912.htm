Moreover, about the question "is there a way where we could create a resource mailbox or shared folder and import this calendar to the resource mailbox and than share this with everyone", the answer is yes. You can import this .pst file to a shared mailbox and all members of this shared mailbox will seethe imported information on their side.

      Hi Rebecca,

      The "proofing language" is a key to several Word features including spelling and grammar and AutoCorrect. It is not an application-wide setting nor even a document setting; it is set at the character level!

      You may want to look at Suzanne Barnhill's article on the spelling checker. It is the definitive article. http://wordfaqs.ssbarnhill.com/MasterSpellCheck...

      Sub StyleSpanish()
'   Written 10 November 2018
'   Charles Kenyon
'   Intended to set all styles to Spanish, proofing, not automatically update
'
    Dim aStyle As Style
    On Error Resume Next ' Some styles have no language attribute and will give an error
    For Each aStyle In ActiveDocument.Styles
'        aStyle.AutomaticallyUpdate = False ' probably do not want this for TOC styles - future work
        aStyle.LanguageID = wdSpanish
        ' For variations of English, see https://docs.microsoft.com/en-us/office/vba/api...
        aStyle.NoProofing = False   ' also turn on spelling and grammar checking
    Next 'aStyle
    ActiveDocument.UpdateStylesOnOpen = False ' For information on using this line, see:
'       http://www.shaunakelly.com/word/sharing/willmyf...
    On Error GoTo 0
End Sub
