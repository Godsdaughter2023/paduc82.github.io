lemp say:
The search function of Windows works equally well for flash drives (or any other storage device connected to the computer).

The flash drive is considered to be "removable storage" and is not a "Local Hard Drive."  If you want to search on the flash drive, you have to set the "Look in" option in Windows search to the drive letter assigned to the flash drive.  If you're not sure what that is, open My Computer and you'll see the flash drive there with its drive letter assignment.

If you can't find the file on the flash drive, your description ("the icons jumped around") suggests that it might be on the hard drive somewhere.  In this case, set the "Look in" option to "Local Hard Drives."  If the computer has a large hard drive (or more than one), the search may take some time .... but if the file is there (didn't get deleted by mistake), it will be found.

Online search for drivers via Windows Update for all devices might not give you the desired results, as Windows Update will have the drivers for the devices that have passed the verification in Windows Hardware Quality labs (WHQL) or if the device manufacturer is partners with Microsoft.
