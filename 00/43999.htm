From your description of how you are installing it, I'm not quite sure but think you're doing it wrong. Here are the steps to install Windows using Boot Camp:

2. Use the 32GB choice for the partition size for XP, 40GB (or more if desired) for Vista.

3. Put your disc in, the Mac will restart, and the Windows XP installation routine will begin. If for some reason it doesn't, restart again and hold down the Option key so you can choose to start from the XP disc. If you get a prompt to "Press any key to boot from the CD", do so.

Use Boot Camp Assistant from within Windows to choose which OS you want as the default. Reboot whenever Windows tells you to. Do not install drivers from Windows Update; you want to use the Boot Camp Drivers.

And normally that's all there is to it.

lemp say:
Many newer Bluetooth devices use Bluetooth 4.x, also called Bluetooth LE (low energy) and Windows 7 does not natively support this version of Bluetooth.  This may (or may not) be your problem.

Open Device Manager on your computer (click Start, type Device Manager in the Search box, and press Enter)

Double-click the Bluetooth category to expand it

Right-click your Bluetooth adapter (receiver, transceiver, etc.) and select Properties

Select the "Advanced" tab

Check the LMP version

  • LMP 0.x - Bluetooth 1.0b
  • LMP 1.x - Bluetooth 1.1
  • LMP 2.x - Bluetooth 1.2
  • LMP 3.x - Bluetooth 2.0 + EDR
  • LMP 4.x - Bluetooth 2.1 + EDR
  • LMP 5.x - Bluetooth 3.0 + HS
  • LMP 6.x - Bluetooth 4.0
  • LMP 7.x - Bluetooth 4.1
  • LMP ​8.x - ​Bluetooth 4.2
  • LMP 9.x - Bluetooth 5.0

If you don't have LMP 6.x or better and the speaker uses Bluetooth 4, that's your problem.

If that is the case for you, the solution is to buy an inexpensive Bluetooth USB transceiver that will work with Windows 7.  For example: https://www.newegg.com/Product/ProductList.aspx?Submit=ENE&N=100158097%20600315449&IsNodeId=1&Description=bluetooth&bop=And&PageSize=96&order=RATING

If you do get a Bluetooth transceiver, disable the onboard Bluetooth in your computer.
