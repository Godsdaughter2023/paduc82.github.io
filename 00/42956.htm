jedediah say: The HM3500 does work with Windows 7! I have a similar HP model PC and a generic $4 USB Bluetooth adapter for my computer. I've used it for listening to music or when I play WoW. Works great but has a short battery life.

1. Unplug the Targus adapter and uninstall all BT drivers from your PC.
2. Re-install this driver from Targus http://cdn.targus.com/web/us/downloads/acb10_v6_0_1_6300_vista_only.exe
3. Plugin BT adapter.
4. Connect Windows to headset and make sure that the passkey is enabled. PW is  0000
5. R click on speaker on lower RH tray and select  >Adjust Audio Properties>Audio>set DEFAULT DEVICE to the headset.
6. Checking "Use only default devices" might help too.
7. Check the settings of any program that you are using and set the 'audio out' to your BT headset. The settings can usually be found under Preferences or Options menu.
