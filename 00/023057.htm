Replace Text Macro in PPT - Microsoft Community

cummingsjp say:

Replace Text Macro in PPT

This is sort of the inverse of kt.mcdaniel's post in early 2010.

 

I have a series of 10 slides with distinct titles.  I want to replace each title with the same string (i.e. each slide title will be the same).

I ended up with

Sub ReplaceText()
    Dim oSld As Slide
    Dim oShp As Shape
    Dim oTxtRng As TextRange
    Dim oTmpRng As TextRange
    Dim strWhatReplace As String, strReplaceText As String
    Dim WhatReplace(10) As String
    
    WhatReplace(1) = "Title 1"
    WhatReplace(2) = "Title 2"
    WhatReplace(3) = "Title 3"
    WhatReplace(4) = "Title 4"
    WhatReplace(5) = "Title 5"
    WhatReplace(6) = "Title 6"
    WhatReplace(7) = "Title 7"
    WhatReplace(8) = "Title 8"
    WhatReplace(9) = "Title 9"
    WhatReplace(10) = "Title 10"

   
     ' write change text
    strReplaceText = "Same Title for Each Slide"
    
     ' go during each slides
    For Each oSld In ActivePresentation.Slides
         ' go during each shapes and textRanges
        For i = 1 To 10
           
     ' write find text
    strWhatReplace = WhatReplace(i)
    ActivePresentation.Slides(i).Select
       
       
        For Each oShp In oSld.Shapes
             ' replace in TextFrame
            Set oTxtRng = oShp.TextFrame.TextRange
            Set oTmpRng = oTxtRng.Replace( _
            FindWhat:=strWhatReplace, _
            Replacewhat:=strReplaceText, _
            WholeWords:=False)
            
            Do While Not oTmpRng Is Nothing
                
                Set oTxtRng = oTxtRng.Characters _
                (oTmpRng.Start + oTmpRng.Length, oTxtRng.Length)
                Set oTmpRng = oTxtRng.Replace( _
                FindWhat:=strWhatReplace, _
                Replacewhat:=strReplaceText, _
                WholeWords:=False)
            Loop
        Next oShp
        Next i
    Next oSld
End Sub

 

which goes through all 10 slides, then goes back to slide 1, replaces the title with the correct text, and then errors.

 

I need to do this to about 20 different presentations, which is why I thought I'd write a macro instead of brute-forcing it.  I'm relatively experienced with EXCEL macros, but PPT macros are kicking my butt.

 

Thanks in advance for any assistance you can provide.

 

JPC


shyam pillai say:

Hi,

If the idea is to replace all slide titles (not all text) and all the slide titles are in the title placeholder then you can use this macro.

Sub ReplaceAllTitles()
Const strReplaceText = "Same Title for Each Slide"
Dim oSld As Slide

For Each oSld In ActivePresentation.Slides
    If oSld.Shapes.HasTitle Then
        oSld.Shapes.Title.TextFrame.TextRange.Text = strReplaceText
    End If
Next

End Sub
