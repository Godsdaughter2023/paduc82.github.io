cmdrkeene say:

There's really no such thing as "THE" temporary folder.  There's quite a many, and any program can arbitrarily decide to use something else.

Most programs use a standard of %localappdata%\temp, but they aren't required to.  The person who built the program might have decided to use %appdata%\local, or even something else instead. Browsers often use specialized places for their temp files.

So short answer: it depends what program you are caring about.

To delete all the Windows Explorer searches, go to the Windows Orb (Start), type regedit, press Enter and in the left pane navigate to HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\WordWheelQuery. It is prudent to make a backup of the registry or create a system restore point before making any registry changes.
