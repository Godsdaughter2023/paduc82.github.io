This depends on the Operating System (XP, Vista, Windows 7), but it is usualy set so that the laptop will "Sleep" which is a low power, but still on, mode that minimises the start time when you open the lid. When you do open the lid, it should start in under 5 seconds, depending on the speed of you laptop.

Once it has been closed for a while your laptop will probably then "Hybernate" which it then copies the contents of its memory to the hard disk and turns OFF. This will allow you st start from where you left off, but will take a little longer than with "Sleep". When you open the list, you will probably need to push the power button, it will "resume" in under 20 seconds, again depending on the speed and amount of memory you have...
