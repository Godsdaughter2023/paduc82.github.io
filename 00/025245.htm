macropod say:
You can format the Heading 3 Style with the 'Keep together' and 'Keep with Next' attribute to keep it all on the same pade as whatever paragraph follows it (regardless of that paragraph's Style). You could also format the Normal Style as 'Keep together', but I would advise against it as every other Style you use is liable to inherit the same attribute from it; you should create your own Style and use that instead. Don't format that Style with the 'Keep with Next' attribute, though, as doing so becomes self-defeating once you go beyond one page.
