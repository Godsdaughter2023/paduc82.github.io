net start wuauserv

I found this very simple fix which sorted all my problems with the mic not working on Skype on my MacBookPro

Go into applications, select utilities and then select terminal 

Type in      sudo killall coreaudiod

Then type in your administrator's password when prompted

Core Audio is your Mac's basic sound system and sometimes there are problems with it. By using this fix you will not lose anything, you will simply be kickstarting your audio system again
