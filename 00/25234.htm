Usually you are only given that option when you have already cancelled the subscription. It is important to note that the subscription has a life cycle so if your subscription renews on the 23rd of each month and you cancelled on the 24th or even the 23rd of a given month, then the subscription for the coming month would have been already renewed and you would be cancelling the subscription on its consecutive renewal.

 

Please do the following;

Open the Zune software and login with your account (if you were already logged in then log out and then log back in), then go to Settings > Account and check under Zune music pass what the status of your Zune pass is. If the status is "Renews on A Date" then there might be an issue with your account and you might have to contact support. Else, if the status is "Expires on A Date" then you have already cancelled the subscription.

 

The Zune pass subscription fee is usually not applied on the date of the renewal but a few days later, and that might be why you see a charge on this month after having cancelled the subscription (if you did).

steve rindsberg say:

Choose Insert | Object | Create from File

Browse to the file you want to insert.

Put a check next to "Display as icon" and then select an icon you like better from the ones presented (or browse and choose a different icon file; I'm not sure if you can choose anything but ICO files, but possible you can pick JPGs as well.

But I think you might like this approach better:

First create the object:

Insert | Object | Create from File | Browse to the file and don't click display as icon.

Insert the picture that you want to use instead.

Select the inserted object

On the Animations tab, choose Add Animation | OLE Action Verb | Open

Then in the Advanced Animation group choose Trigger, On Click Of and pick the name of the picture you inserted.

Now drag the inserted object off the slide if you wish.

When you run the show, clicking the picture will now open the inserted object.
