rnardone say:

So I found a possible solution:

In REGEDIT go to:

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Power\PowerSettings\238C9FA8-0AAD-41ED-83F4-97BE242C8F20\7bc4a2f9-d8fc-4469-b07b-33eb785aaca0

At the location there is a DWORD named "Attributes". Modify the value of "Attributes" from 1 to 2. A value of 2 will make "System unattended sleep timeout" appear in the advanced power settings under "sleep." Notice the default value is 2 minutes for both battery and plugged in.

Change the values to whatever you want them set to. I made mine match the "Sleep after" settings. Some people have reported that the settings will return to the defaults (2 minutes) occasionally. I have not experienced that yet but I will update this post if it does. I suspect they may have changed the power plan and forgot to change the defaults. Your changes will not carry across plans.

Basically what is happening is your PC is being woke up in an unattended state so it defaults to the "System unattended sleep timeout" settings instead of the "Sleep after" settings. An unattended state is usually when the PC wakes of to do a task like Windows Updates or a wake on LAN (WoL) event. I'm not sure why the "System unattended sleep timeout" setting is used when I wake the PC with mouse movement, but the steps above seem to have solved my issue.
