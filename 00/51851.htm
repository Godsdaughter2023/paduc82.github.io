Tips to resolve problems with Windows Update (BITS), Base - Microsoft Community

coder_2048 say:

Tips to resolve problems with Windows Update (BITS), Base Filtering Engine (BFE), and Firewall (MpsSvc), and HomeGroup (FdResPub)

Hi, I had several problems that may have been caused by a virus and I was able to resolve them after quite a bit of digging so I wanted to share some information.

All the fixes were made with registry settings.

 

1. Unable to use Windows Update Successfully

    Try fixing the Background Intelligent Transfer Service ((BITS) registry.

http://answers.microsoft.com/en-us/windows/forum/windows_7-windows_update/windows-update-wont-work-error-code-80246008/df067579-7c6f-4a2f-922a-7bea993cf618

 

2. Unable to start Base Filtering Engine Service

 

    Grant permissions to access a registry key.

 

    Key Name: HKLM\SYSTEM\CurrentControlSet\services\BFE\Parameters\Policy

 

    Names in Users and Groups, Permissions:

    NT Service\BFE service, Read

    LOCAL SERVICE, Full Control

    NETWORK SERVICE, Read

 

3. Unable to Start Firewall Service (MpsSvc)

 

    Grant permission to access a registry key.

 

    Key Name: HKLM\SYSTEM\CurrentControlSet\services\SharedAccess

 

    Name in Users and Groups, Permission:

    NT Service\MpsSvc, Full Control

 

4. Unable to access Homegroup Setup

 

    Function Discovery Resource Publication (FdResPub)
    Access Denied, Error Code 80070005 upon starting service

 

    Grant permission to access a registry key.

 

    Key Name: HKLM\SYSTEM\CurrentControlSet\services\SharedAccess

 

    Name in Users and Groups, Permission:

    NT Service\FdResPub, Full Control

 

 

 

I suggest fixing all of the above in the order listed because some depend on the other being fixed first.

 

Best Wishes,

Coder_2048
