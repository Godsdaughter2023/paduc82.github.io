How to turn off "USB device not recognized" notification - Microsoft Community

lesley_smith say:

How to turn off "USB device not recognized" notification (Windows 7)

So, I came on here looking for this answer, found a thread on the discussion but no answer on how to disable the notification.  I ended up playing around on the computer myself and figured it out.  So, of course I went back to the thread to post how I did it...and the thread is locked.  So, in case this hasn't been answered anywhere else - here's how I did it.

NOTE:  this is how to turn off the notification - not how to fix the problem.  My problem was the annoying notification pop up happening because every time I moved my usb cord (charging my phone) this pops up a million times.  Obviously the connection is not perfect, however the cord/charging works fine so instead of replacing a cord that is working, I just needed to disable the notification.

For Windows 7

Notification pop up

1.  click on tool icon in the notification pop up (will now show "Windows Explorer" as top icon in "Notification Area Icons"

2. in drop down, change to "Hide icon and notifications"

3. Okay/Apply

Notification sound

1. click Windows key

2. click "Control Panel"

3. in search bar type "sound"

4. Under Sound click "change system sounds"

5. scroll to "Device connect" and "Device disconnect" and use drop downs to change sound to "none" for both

6. click "Apply"

Hope this helps :)

Try running the Network and Bluetooth troubleshooter.

  1. Go to Settings > Update & Security >Troubleshoot

  2. Click on ‘Internet Connections’

  3. Now click on ‘Identify and repair network problems’.

  4. After the network troubleshooter completes click on ‘Bluetooth’.

  5. Run the troubleshooter and it will fix issue related to Bluetooth.

This will try to fix the connectivity issues with the internet and Bluetooth.

Method 2:

Try updating the drivers through device manager.

  1. Press Windows key + R. This will open Run. Alternatively, you can go to Start and search for ‘Run’.

  2. In Run dialog box, type ‘devmgmt.msc’ and hit enter.

  3. Go to ‘Network Adapters’ and expand them.

  4. Right click on the ‘WIFI’ network adapter, and select ‘Update Drivers’.

  5. Click on ‘Search automatically for updated driver software.

In the same way you can update the ‘Bluetooth’ drivers.

  1. Go to ‘Bluetooth’ and expand it.

  2. Right click on the ‘Bluetooth’ drivers, and select ‘Update Drivers’.

  3. Click on ‘Search automatically for updated driver software.

  4. Restart the PC after the updating of the drivers completes.

This will update the drivers of WIFI and Bluetooth.

Once the drivers are updated check, if you can use both WIFI and Bluetooth together.
