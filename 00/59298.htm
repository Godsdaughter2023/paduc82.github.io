c:\users\your_user_name> path = %path%;c:\Windows\SysWow64;
[Disclaimer I didn't test this myself: The instructions below are for a 64bit Windows 10 installation. If you are on a 32bit Windows box, change SysWow64 to System32 (or whichever path where Windows stores those files)]

If this fixed your issue, you may want to add this info permanently to your PATH variable. To do so,  you need to update the PATH System Environment Variable. Here is how you do it.

Step 1: Open System Properties (Search for environ on your search box to directly open the System Properties with the Advanced tab selected)

Step 2: Open the Advanced Tab, if it is not already opened. Click on the Environment Variables Button.

Step 3: In Environment Variables select PATH and then click on the Edit button

Step 4: Create a New entry and add C:\Windows\SysWow64\ and then click OK

Now open a new CMD window and see if your DOS commands are back.
