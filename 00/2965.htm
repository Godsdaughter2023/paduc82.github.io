ayushman say:

Hi,

You will have to create four tables in excel.

1. Paper - Put all paper items under it and give the range name Paper.

2. Cardboard - Put all cardboard items under it and give the range name Cardboard.

3. Cloth - Put all cloth items under it and give the range name Cloth.

4. Paper & Cardboard - Put all paper and cardboard items under it and give the range name PaperCardboard. (Please note no space)

Now,

Suppose go to cell A1 >> then click Data Validation >> List >> Put all these four name.

It will create a drop down in cell A1.

Suppose go to cell B1 >> then click Data Validation >> List >> write the below formula :

=INDIRECT(A1)

THATS ALL!

Now, select "PaperCardboard" from drop down in A1 and the magic in B1.

You can turn off the recording tab (in PowerPoint) via the registry

HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\PowerPoint\Options\RecordingRibbonTabState 

(set to 0)

BUT this is probably not what you mean and also easy to override.

If you just want to disable the screen recording button you will probably need to write a small ppam AddIn that disables it in the XML.

stephen boots say:

Sorry for the delayed reply.

 

My recommendation is that you skip the mpcmdrun task and simply create a local task on each PC to execute the downloaded full definitions and engine. When you execute the downloaded file, it installs the whole shebang.

 

On a single machine, create a task to download the full engine/database daily prior to the execution of the local tasks.

 

With the exception of the full database/engine refresh that happens monthly, BITS and the small download of daily updates really should not present an undue burden on your Internet connection as all machines will not be checking for an downloading updates concurrently, Note that Windows Updates will also download in the same manner, using the same pipeline. If you need to stop that, you may wish to consider a WSUS setup with local machines obtaining Windows and other updates from the local server. MSE can also obtain updates this way.
