rohn007 say:

Yes you can.  I have Office 2003, 2007, 2010 and 2013 all installed in Win 7 at the same time.

The "official" answer is no you cannot, but that is not correct.

There are a couple of considerations to be aware of:

  • Windows requires you pick one version or the other to be the "Windows Default" for file associations that load application when you double click on a file
  • You can only have one copy of Outlook installed and working (although in theory this is no longer true with Office 365, I have no experience with making outlook work in both versions.
  • You will have to perform a windows "registry hack" to make this work.

Here are the instructions I used to make it work for me:

Dual Install - Working with Multiple Versions of Office

https://office.microsoft.com/en-us/word-help/keep-earlier-versions-of-office-programs-when-installing-office-2010-HA102597134.aspx

This tip describes the specific steps and choices during the 2010 custom install. After you have the multiple versions installed you still have some additional steps to do. They are described in this tip:

 

http://pubs.logicalexpressions.com/pub0009/LPMArticle.asp?ID=762

With multiple versions installed, there is one issue you can’t avoid, Default Windows File Associations. Only one app can be used to open the files by double clicking in Windows Explorer. You have to pick which one.

FYI: NoReReg key works with Office 2010 / 2013 installed on Windows 7 and 8 (I tested it myself Rohn007)

Note: in user accounts, when you log in to do the NoReReg hacks, DO NOT start REGEDIT using “Run As Admin”. You will be editing the wrong “Current User” key <grin>

Manual method:

Before you close up shop and start working on this reg fix, open the version of Word that you will want to be your default. In other words, say you have two versions of Word on your system. You want to use 2007, but you want 2003 to be the version that opens by default when you click on a DOC file. Open Word 2003 before you insert the fix below. This will ensure that Word 2003 is the last version opened and it will, if necessary, reset the registry to that version. (More on getting around this default in the next section.)

  • Now make sure Word (office apps) is closed
  • Start regedit  <Win  R> and type REGEDIT
  • Note that if this doesn’t do anything, you’re using a system where the administrator has taken away your ability to edit the registry so you don’t mess it up!
  • Once within the registry, click the + (plus sign) to expand the /HKEY_CURRENT_USER/ folder.
  • Scroll down and locate the /Software/ folder and expand that one. We have a long journey ahead, so hang in there!
  • Once within the Software folder, scroll down that branch of the folder tree until you find the folder for /Microsoft/. Click the + to expand that one.
  • Now scroll down to locate the /Office/ folder.
  • There you’ll see the numbered versions of the Office versions you have:
  • 2013 is 15.0
  • 2010 is 14.0
  • 2007 is 12.0
  • 2003 is 11.0
  • 2002 is 10.0e.
  • This fix can be used for all recent versions. Or at least, I know it works as far back as 2000. I can’t confirm anything earlier. But if you’re still using a version older than 2000, speed is obviously not an issue for you!<grin> Expand the folder for the version you want to speed up. (Note, this speed up fix will only help you if you are running multiple versions. So if you only have one version of Office on your system. Don’t bother with this setting, as it won’t do anything for you.)
  • Move to the /Word/ folder and expand that. Almost there!
  • Okay, locate the /Options/ folder and select that one. This is the folder where you’ll be adding a new registry key.
  • With the Options folder selected in the route we took above, click Edit > New > DWORD Value
  • This will insert a new registry key into the Options folder. It will be highlighted in edit mode so you can go right ahead and start typing the name of this key: NoReReg
  • Hit the Enter key after typing the name to have the name accepted.
  • Now double click this new key to open it. The Value data will read 0 (zero) by default. Change that setting to the number 1
  • click OK to accept that value.

Then close the registry. You did it!

So what is it that you’ve done? Well, you went into the Word registry settings for the version of Word you selected and you set a new registry key. The key is an option that stops Word from reregistering itself if another version is the current default. The value of 1 means Yes. So the NoRereg (no reregistering) is set to Yes...i.e., don’t do it! This will cause Word to ignore the default version and just open, without taking several minutes to go through the reregistering hassle.

Get date from year, weeknumber + weekday?

Is this possible?

I would like to create a formula that takes a given year in one cell, the week number (in ISO) from another cell, and the name of the weekday (e.g. Monday) from a third cell, and then get the date in a forth cell?

Is it even doable?

ronrosenfeld say:
On Tue, 21 Dec 2010 20:14:07 +0000, jdb264c wrote:
 
>
>
>Is this possible?
>
>I would like to create a formula that takes a given year in one cell, the week number (in ISO) from another cell, and the name of the weekday (e.g. Monday) from a third cell, and then get the date in a forth cell?
>
>Is it even doable?
 
Try:
 
=DATE(Yr,1,5)-WEEKDAY(DATE(Yr,1,3))+(WeekNum-1)*7+SEARCH(LEFT(WeekDy,3),"montuewedthufrisatsun")/3
 
The first part of the formula should return the first Monday that precedes the first Thursday in the year; in other words, Day 1 of Week 1 (which can sometimes be in the previous year).

a. user say:

Hello,

 

I am sorry for the late reply. I can see the convenience of having a sound notification when new messages arrived in your inbox. However, as of this time, sound notification on Outlook web based account is not available. Outlook does not have the option to setup this feature. You can post on our feedback site regarding this concern. Please click this link to visit our feedback site.

 

I have a suggestion, if you are going to use an email client like Live Mail, there is a feature there where in you can setup sound notification when new emails arrive. To download Live Mail, please click here.

For the steps on how to add your email account in Live Mail, please refer to the solution article below:

 

How do I add an account to Windows Live Mail?

 

Lastly, here are the steps on how to setup sound notification in Live Mail:

1. Pull up your Live Mail.
2. Press Alt + F.
3. Choose Options then Mail.
4. Click General tab
5. Under Send / Receive Messages
6. Check, Play sounds when new messages arrive.
7. Click Apply, then OK.

 

Now, if you are looking for this feature in Mac mail client, I am not sure if they have this feature. You need to ask them. If you have further questions, please get back to us.
