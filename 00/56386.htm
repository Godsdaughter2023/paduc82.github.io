¡firedog say:

It rather depends what the recipient intends to do with it.

If you want to share all the data for a number of your contacts, you could attach a vCard to your message for each one. The recipient will probably be able to add them to his own contacts list simply by double-clicking on them. To do this,

  1. Switch to the contacts window (Ctrl-3)
  2. Open a new mail message (Alt-F N)
  3. Arrange the windows so you can see both at once.
  4. Select the contacts whose data you want to share (e.g. by Ctrl-click)
  5. Click and drag from the contacts list into the Subject box of the message window.

If you only want to send a list of names and email addresses,

  1. Open a new mail message.
  2. Click To..., then Ctrl-click to select the contacts whose name and address you want to share
  3. Click To-> and OK 
  4. Press Ctrl-A X to select all the entries in the To... box and cut them
  5. Click inside the message body and press Ctrl-V

This will produce a semicolon separated list in the format name1<address1>;name2<address2>;... which can easily be transformed into a list like
name1   address1
name2   address2

using Find and replace in a text editing program like Word or Notepad.
