Arguments in AVERAGE function

Does anyone know how to increase the "arguments" in the AVERAGE function?  I have a list of numbers that varies and it seems that I can only submit 32 numbers or up to 155 characters.  I am sure that there is SOME way to increase the number of "arguments" i can use in an AVERAGE function.
t. valko say:
Does anyone know how to increase the "arguments" in the AVERAGE function?  I have a list of numbers that varies and it seems that I can only submit 32 numbers or up to 155 characters.  I am sure that there is SOME way to increase the number of "arguments" i can use in an AVERAGE function.


You should've posted the formula you're trying to use.

In Excel 2007 (and later) the AVERAGE function can take up to 255 arguments. You should NEVER run out of arguments!

=AVERAGE((A1,D2,E3),(B27:D27),X10)

That formula contains 3 arguments. Each of the inner (...) is ONE argument. So:

  • Argument 1 = (A1,D2,E3)
  • Argument 2 = (B27:D27)
  • Argument 3 = X10

Hi Asharaf,

1. Can you please help the storage limits for SP & one drive please rather giving article, please give clear details please: 

SharePoint storage is 1 TB per organization plus 10 GB per license purchased. (When you have an Office 365 subscription and a SharePoint Online standalone plan, the storage amounts are added. ) Up to 25 TB per site collection or group. (SharePoint Online administrators can set storage limits for site collections and sites. ) For your reference: SharePoint Online Limits, Add storage space for your subscription.

2.If a user assigned with SP PLAN1 OR 2, is the one drive storage & share point storage different? or it will be the same ? 

For the storage limits about OneDrive: OneDrive for Business Plan 1: 1 TB of OneDrive storage per user. OneDrive for Business Plan 2: Unlimited personal cloud storage.

Different SharePoint Online plans have the same storage limits. They support different features. For your reference: SharePoint Online Service  Description. SharePoint plan doesn't have an influence on OneDrive storage. You can get more info about OneDrive storage via this article: OneDrive for Business Service Description.

3.When a site is created by the user from SP, do he has access to delete the site from client end?

Yes. The site creator can delete the site from the site. You can get more info from these links: Delete a SharePoint site or subsite, Delete a site.
