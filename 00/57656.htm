lemp say:
Although you didn't answer this question, I'm pretty certain that the error message you see is displayed on the 1135n rather than on your computer.  Please confirm.

Unfortunately, the User Guide for the Dell 1135 does not list this error message.  At least one other fax machine uses error code D.0.7 to indicate an error when dialing (that is, when a fax is being sent from your machine, rather than being received).  Your original question, however, indicates that you get this message when receiving a fax from an outside machine.

Do you get the error with faxes sent from more than one source?

If you get the error message only with faxes with from one outside fax machine, try asking the sender to use a slower speed.  Or ask the sender to turn on Error Correcting Mode (ECM), if available on his machine.  If the sender turns on ECM, you should turn it on for your 1135 as well.  See the User Guide.

If you get the error message with faxes sent from several outside machines, the problem likely is with your telephone line.  If you have Verizon DSL rather than Verizon FiOS for your Internet, you should have DSL filters on all phone lines, including the line to the 1135.

If you pick up a regular telephone, do you hear any static on the line?

You may need to have a Verizon tech come out and check the condition of your lines, but before you go to the expense of a Verizon service call, you might want to ask your question in the Dell Printers Forum (click "Join" at the upper right of the page to create a login identity).


lemp say:
You couldn't find Windows Fax and Scan because that is a Vista or Windows 7 application and -- because you posted in a Windows XP forum -- you probably have some version of Windows XP.

It makes it more likely that you'll get help if you specify the version of Windows you have, including service pack, in your original question.  For details, right-click My Computer and select Properties.

Whether you have Windows 7 Fax and Scan or the XP version of fax support really isn't relevant because your Dell 1135N should be functioning as a stand-alone fax machine.  It does not use Windows fax support.

  • Where do you see the error message?  Is it on your computer or on the Control Panel display on the Dell 1135N?  What is the exact complete text of the error message?
  • What kind of telephone line did you connect to the 1135 printer -- (a) standard analog home telephone line, (b) digital telephone line (usually found only in business offices), (c) Internet telephone line (e.g., Vonage, MagicJack)?

  • If you use DSL to connect to the Internet, have you installed filters correctly on the telephone lines connected to the printer and all other telephone devices?

Dell's Fax and Scan Manager software should have been installed when you installed the drivers for your printer.  If you click Start > All Programs, can you find and start the Dell Fax and Scan Manager? You should be able to use this (as well as the front panel controls on the 1135n) to configure the fax settings of the machine.  You probably will want to set Receive Mode = Fax, Ring to Answer = 1, Secure Receive = Off, Default Mode = Fax.


You can configure the 1135n to use this software to save incoming faxes as files on your computer ... but that's not what it does by default.  Did you configure the 1135n to "forward" received faxes to your computer (you have to make settings for this both on the printer's control panel AND in the Dell Fax and Scan Manager)?
