OK, I didn't understand.  What you need is a sheet offset button that is written in Visual Basic.
I suggest placing the Offset value in sheet1.  This is how I did sheet 1.

The formula bar shows the formula in A2 of sheet 1.  The offset value is in B2 so we make this reference absolute.  The second reference is the cell who's value we want to return.  The show the vaues of another sheet we just change the sheet offset value in B2.

The User Defined Function is:

==================================

Function SHEETOFFSET(Offset, ref)
'Returns the cell contents at Ref, in the sheet offset
    SHEETOFFSET = Sheets(Application.Caller.Parent.Index + Offset) _
    .Range(ref.Address)
End Function

===================================

Note: this function will produce errors if the workbook contains Chart sheets.

===========================

Excel Trendline Question - Microsoft Community

ta911966 say:

Excel Trendline Question

I did a trend line in excel for a data set.  The trendline was exponential and had an equation of 4E+06e^-.025x.  I wanted to generate the data points for this equation.  How do I write an equation in excel for this?
joeu2004 say: TA wrote:
I did a trend line in excel for a data set.  The trendline was exponential and had an equation of 4E+06e^-.025x.  I wanted to generate the data points for this equation.  How do I write an equation in excel for this?

Two ways:  the low-tech quickie way, and the "right" way.

Suppose your data (x) are in A1:A100.  Ostensibly, you could enter the following formula into B1 and copy down:

=4E06*exp(-0.25*A1)

However, that will likely be inaccurate.  Before you do that it, it would be prudent to display the trendline formula with greater precision.  For example, 4E+06 might be 3.789E+06 or 4.456E+06.  Similarly, -0.25 might be -0.25456 or -0.254789.

One way to change the number format of the trendline is to select the box around the trendline formula, right-click and click Format Data Labels, then click the Number tab and choose a format with more decimal places.  Then substitute the more accurate constants in the formula above.

For the "right" way, see http://spreadsheetpage.com/index.php/tip/chart_trendline_formulas.

If your original data are in A8:A13 (x) and B8:B13 (y), you might enter the following formulas:

C1:  =EXP(INDEX(LINEST(LN(B8:B13),A8:A13),1,2))
D1:  =INDEX(LINEST(LN(B8:B13),A8:A13),1)

Note that those do not have to be array-entered; just press Enter.

Then, if your additional data (x) are in A14:A100, put the following formula into B14 and copy down:

=$D$1*exp(B14*$C$1)

If you want to understand the formulas in C1 and D1, start by reading the Help pages for LINEST, INDEX, EXP and LN.  If you have further questions about those Help page descriptions, feel free to ask.

Deleting Office credentials via Credential Manager didn't work for me, but what did work for us was adding the SharePoint URL to Trusted Sites in Internet Explorer, which allowed the laptop login credential to pass through, eliminating the explicit password queries.
