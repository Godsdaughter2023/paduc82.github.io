Comparing two columns of data - Microsoft Community

bfrost1 say:

Comparing two columns of data

Hello,

I am trying to compare two columns of data. Column 1 contains a list of many thousands of number codes, column 2 contains a few hundred. If the number code exists in column 1 and 2 then it is category A, if it does not it is category B. Any ideas? I've tried Vlookup, IF, ISERROR, and others but I keep getting NAME or failed comparisons when I run it. Thanks!

teylyn say:

Hello,

try to match the value from column B in column A. If there is a match, the MATCH function will return the row number. If there is no match, the MATCH function will return an error. So you can put the MATCH function inside an IF statement that checks if it returns a number

=if(isnumber(Match(B2,$A$1:$A$10000,0)),"Category A", "Category B")

Or you could simply count how many times the value in column B appears in column A. If it is not there, it will be a zero. If the value is present, it will be counted, so at least be a 1. In Excel 0 can mean FALSE and any number greater than zero can mean TRUE in an IF statement. So you can build this:

=if(Countif(A:A,B1),"Category A", "Category B")

Note how we don't need to compare the result of the Countif with anything, because if the count is 0 it means "False" automatically, and any number greater than 0 will mean "True".
