larry yu msft say:

Hi Graham,

Adding the Planner tab to Microsoft Teams means we can create new tasks in Planner (tasks.office.com) through Microsoft Teams, the planner tab and Planner aren’t the same thing. In other words, add the planner tab means create a new plan in Planner. I’d like to explain how this works as below:

1. When user A creates a new team in Microsoft Teams, it will create a plan in the All plans section in user A’s Planner.

2. If user A creates a new task in the planner tab and assign it to himself, this task will also be created in user A’s Planner > My tasks. But if the task has been assigned to another user such as user B in the team, it will appear in user B’s Planner.

3. If user A create a new bucket in the planner tab, it won’t be created in user A’s Planner because this isn’t a plan or a task. But tasks created in the new bucket will be created in user A’s Planner > My tasks.

This also explained why tasks or buckets created in Planner won’t be created in the planner tab in Microsoft Teams.

If you have created a new task in the planner tab and assigned it to yourself but cannot see it in your Planner > My tasks, we need to collect some information to better understand your situation:
1. The detailed steps about how you create the new task in the planner tab.

2. Will it appears in Planner > My tasks if you create a brand-new task in the planner tab?

Never open unsolicited email attachments.

Practice safe web browsing.
