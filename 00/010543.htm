Hi,

try this...

right click on getData tab, select View Code and paste in..

Private Sub Worksheet_Change(ByVal Target As Range)
Const myCol As String = "Q" '<<< week in column Q
Dim ws2 As Worksheet
Set ws2 = Sheets("ReportData") '<< target sheet
If Not Intersect(Target, Range("B4")) Is Nothing Then
Application.ScreenUpdating = False
ActiveSheet.AutoFilterMode = False
Range(myCol & "5:" & myCol & "350006").AutoFilter Field:=1, Criteria1:=Target.Value & "*"
Range(Range("C6"), Cells(350006, myCol)).Copy
ws2.Range("C3").PasteSpecial xlValues
Application.CutCopyMode = False
ActiveSheet.AutoFilterMode = False
Application.ScreenUpdating = True
End If
End Sub

hi ravikaran patel,

for your questions:
1. all the files are shared with edit permission to respective team members and they edit them on regular basis, how can i come to know which file was updated and under which folder is it. (i am the owner of those files)
we can see the updated file via recent option in onedrive. we can see the people who modified the file and the modified time. however, there isn’t the information that the file under which folder. we can see the information by a workaround i will mention in question 2.

2. the folders do not show the last modified time or date, it only shows the time when it was created, i want to get some kind of notification that will help me with details on which file was modified and by whom.
yes, by default, it only shows the time when the folder was created. only when we get into the folder, we can see the detailed information about the change for the folder.

in addition, we have a workaround to see the changes for the file among the whole onedrive document library.
1. go to site settings > storage metrics under site collection administration.
2. choose our onedrive document library (documents) > click last modified, then we can see the folders and files which has been modified.


3. we can also get the detailed information about who and when the people has modified this file by click version history.


 
i understand it may be inconvenient for us to check this every time and it will be nice if we can get a report for this feature. given the situation, i would suggest you submit feedback to our related team via uservoice as this the best way to improve the product. many features have been designed or upgraded based on our customers' constructive comments.
