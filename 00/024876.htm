Office for Mac document recovery - Microsoft Community

paulsmithtc say:

Office for Mac document recovery

I have been working on a Word document for a few days now and when I tried to paste something from a different application, I got the revolving beach ball... I can't work on any other Word documents that are also open.  I think my only option is to force close the application.  Does Mac have an autosave function, or should I wait it out?
The other Office applications work fine.

Thanks

john mcghie say:

Just to answer some of your questions:

*  Word does not have an autosave function.

*  It does have an "auto recover" function.  The difference is that autorecover does not save the document, it saves only the changes to the document.  Which means that if the main document corrupts, Word has nothing to which to apply its stored changes, so it is unable to recover anything at all.

*  There is no point in waiting longer than 20 seconds for a "beachball".  The beachball will appear only when Word locks up, it will never clear spontaneously.  Go ahead and force-quite Word.

*  If the main document is intact, when Word re-starts it will present you with the recovered version of any documents that were open when it failed.

If no recovered documents appear, you then need to restore your documents from Time Machine.

Rich has suggested one operating procedure that's very important.

Another is to turn on "Always create backup" in Word>Preferences>Save.  If  that had been turned on, a copy of your document would appear in the same folder every time you manually save the document.  That is a full, complete document, so the maximum you can lose is the text created since your last manual save.

It is also important to turn off Track Changes if you are going to to a lot of work on a document: track changes will eventually produce the symptoms you have.  If you need change marking, you can apply it after you finish by using Compare Documents.

Sorry you lost your changes, but if you do these things, you won't lose them again :-)

Cheers
