delete the first and last records in a table using VBA - Microsoft Community

magicdds say:

delete the first and last records in a table using VBA

I would like to use VBA code to delete the first and last records of a table. How would I do that?

 

Thanks

ken sheridan say:
Instead of a time and date stamp, can I use a field called "ID" that is an autonumber and consider the record with the minimum ID to be the first record in the table and the record with the maximum ID to be the last record in the table?


Probably, but not with 100% confidence.  An autonumber is guaranteed to provide distinct values only, not necessarily sequential ones.  The code would be along these lines:

    Dim strSQL As String
  
    strSQL = _
        "DELETE * " & _
        "FROM YourTable " & _
        "WHERE ID = " & _
            "(SELECT MAX(ID) " & _
             "FROM YourTable) " & _
         "OR ID = " & _
            "(SELECT MIN(ID) " & _
             "FROM YourTable)"

    CurrentDb.Execute strSQL, dbFailOnError

A date/time stamp column would be more reliable.  Or you could use a custom generated sequential number as the key, using the method demonstrated by Roger Carlson at:

http://www.rogersaccesslibrary.com/forum/topic395.html

First, I have no idea if you've ever used VBA or made a macro before. If you're new to VBA, here are two links that might be helpful to you:

Display and Navigate the VBE Interface in Office 2011 for Mac

How to Code with the VBE in Office 2011 for Mac

This example will paste whatever's on the clipboard as a value (without pasting formatting) into the selected cell.

Now for a step by step. I suggest doing this in a new, blank workbook.

1. Type something in a cell (a formula, number, or date)

2. Select the cell, and use Command-C to copy the cell

3. From the Tools menu choose Macro > Record New Macro

This dialog will display

4. Fill in the blanks.

Give the macro a Name (must be alphanumeric without spaces)

Assign a ShortcutKey

Store the macro in your Personal Macro Workbook so that it will always be available to you on your Mac in Excel

Give the macro a description

5. Click OK

6. Select an empty cell

7. From the Edit menu choose Paste Special

8. In the Paste Special dialog (see Phillips answer) choose Values then click OK

9. From the Tools menu choose Macros > Stop Recording.

Your Macro is almost ready.

Go to the Visual Basic Editor and trim the excess code so that all the macro does is the paste special operation. Edit your code to read as follows

Sub PasteValues()
'
' PasteValues Macro
' Paste Special Values
    ActiveCell.Select
    Selection.PasteSpecial Paste:=xlValues
    ActiveCell.Activate
End Sub

Then follow these steps using paste special > formulas to make a second shortcut.
