I predict problems if you try to replace the Excel 2007 with Excel 2010.  All of Office has some shared components, and some of those associated with Excel 2010 might cause problems if any of the other Office 2007 applications need to access them.

Proceed with caution: you may be able to install Excel 2010 in addition to Excel 2007 on the system.  Watch as you get into the install process and if it offers you an opportunity to "replace or install in addition to" type of option, take the one that appears to install the separate copy of Excel 2010.  If you don't see that option by the time you get to the "complete rest of installation" phase, then I'd probably cancel out.
