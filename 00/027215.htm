suzanne s. barnhill say:

If you use legacy dropdown field, you will need to protect the document for forms, and this will probably not fit with your overall plan if you've started off using content controls, but you can learn more about form fields from the series of tutorials by Dian Chapman linked from http://word.mvps.org/FAQs/Customization/FillinTheBlanks.htm.

 

The most efficient way to create the IF fields is manually.

  1. Press Ctrl+F9 to insert a field.
  2. Between the field delimiters (which look like braces), type IF, then the name of the bookmark you've applied to your dropdown field.
  3. Select that name and press Ctrl+F9 to convert it to a field.
  4. Then type one of the company names you've used as selections in the dropdown field. If it's more than one word, you'll need to enclose the term in quotation marks.
  5. Then, in quotation marks, type or paste the information you want inserted if that company is selected. This can include line or paragraph breaks (which is why pasting is often less confusing).
  6. When you've completed typing the first field, start another immediately after it (with no space between) and continue till you've done them all.
  7. Select all the fields and press F9 to update them. This will result in making them all disappear.
  8. Protect the form.
  9. Select a company and Tab out of the field; you should see the relevant information where you inserted the IF &n

    It looks like you are doing everything right.  Ignore what you see in the Formula Bar.  It has its own limited formatting rules.  For example, the Formula Bar does not show fractional seconds, even if the cell is formatted to do so, which can make it very frustrating to edit data entry with fractional seconds.

     

    Rob wrote:

    How can I find an average time? 

     

    Just use AVERAGE as you normally would.  But if you display it in a cell, format the cell as Custom [m]:ss.000 or [h]:mm:ss.000, depending on how you want to display it.  The format [m] and [h] is useful for displaying minutes and hours that might exceed 59.

    Note:  You can use ss.0, ss.00 or ss.000.  But ss.000 is the most precision that Excel will format.

     

    Rob wrote:

    Additionally I have another field that shows idle time and that displays like:  170:38:00[.]  This would be 170h:38m:0s

     

    Format as Custom [h]:mm:ss.

     

    Rob wrote:

    These fields represent the average speed of answer for a call center.

     

    Caveat:  the average of averages is often not the same as the average of the total.

    They are the same only if all of the individual averages have the same divisors; that is, they are the average of the same number of calls for each call center.

    It might be the best you can do if you have no further information.

    But if you have the number of calls for each call center, the following computes the true average, assuming the average for each call center is in A1:A20 and the number of calls for each call center is in B1:B20:

    =SUMPRODUCT(A1:A20,B1:B20) / SUM(B1:B20)

    formatted as Custom [m]:ss.000 or [h]:mm:ss.000 as you wish.

    Use of layout view has ZERO and NOTHING to do with hiding a form. Might as well talk about your pet dog – simple zero and no relationship to hiding a form.

    Layout view is an option for DEVELOPERS to change and modify the form WHILE looking at live data – it would make zero sense to ask anything about layout view in the context of attempting to hide a form.


    So you are most confusing by asking about hiding a form and in that context talking about layout view.

You then let the users edit data, and then when done they hit a button to CLOSE the form. When you close the form you thus return back to the main form where the user can then can hit a button to launch more forms to edit more data.

    So talking about layout view and that of hiding forms makes no sense and explains why you are having difficulty finding an answer to your question.
