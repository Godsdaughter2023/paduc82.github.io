rick rothstein, mvp - excel say:

CODE CORRECTION
==========================

Cal (cocoknots) contacted me via email to tell me that some of the numbers were not being converted correctly. The problem was a line of code I left in when I was trying to handle the "Excel making a dashed number into a date" problem. Since Cal removed that requirement to try and correct the column, the code line I left in (which had an unintended side effect that I did not notice) is no longer needed. Here is the corrected code...

Sub ConvertCommodityQuote()
  Dim Cell As Range, CellValue As String, TrailingLetter As String, Parts() As String
  With Worksheets("Data")
    For Each Cell In Union(.Range("C3:C17"), .Range("E3:E17"), .Range("G3:K17"))
      TrailingLetter = ""
      If Len(Cell.Value) Then
        CellValue = Cell.Value
        If Right(CellValue, 1) Like "[!0-9]" Then
          TrailingLetter = Right(CellValue, 1)
          CellValue = Left(CellValue, Len(CellValue) - 1)
        End If
        Parts = Split(CellValue & "-", "-")
        Cell.Value = CStr(Parts(0) + Val(Parts(1)) / 8) & TrailingLetter
      End If
    Next
  End With
End Sub
