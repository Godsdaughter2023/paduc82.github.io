No.  It's the .vob file(s) that are played.  The .ifo file is for storing information about the format of the disc, such as chapter begin/end data, and menu information.  The .bup file is not a "legacy backup file," it's an exact copy of the .vob files and can be used in the event that the .vob files are corrupted.   Another Directly fail.
