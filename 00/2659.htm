scott .. b say:

In Windows 8, there are no longer two separate versions of the browser in the start menu as was the case in Windows 7.  However, the browser still retains both its 32-bit and 64-bit modes.

 

 To choose which mode to use, you will have to take a different approach. Enabling/disabling it is associated with security zones, just the way the security settings and Protected Mode are. This means you can set 32-bit mode for the Internet zone while using 64-bit mode for your local intranet, or you could use 32-bit mode for trusted sites for maximum compatibility and 64-bit mode for untrusted sites for maximum security.

 

 

EXAMPLE:
Notice that the "about IE10" is the same whether you open iexplorer.exe from the "Program Files" or "Program Files (x86)" location.

 


1. Open Internet Options.


2. Click on the Advanced tab, scroll down to the Security section. Turn on "Enable Enhanced Protected Mode" for 64-bit IE10 in Windows 8.

 


 3. Open Security tab .

 If not already checked, select Enable Protected Mode box for the Internet zone. (this box is checked by default).

 


Click APPLY.....OK. This sets the IE10 64-bit browser for the Security Zone..

 

4 To set the 32-bit browser for the Security Zone, disable "Enhanced Protected Mode"

5. Restart the computer for the changes to take effect..

 

 

Once you have chosen your preferred settings for all the security zones, simply restart Internet Explorer. You will now be able to browse the zones in the corresponding 32-bit or 64-bit modes. This new method seems to be a welcome change considering it removes the necessity of having two versions of the same browser installed, while still granting access to both versions.

In Word, language is a character property, meaning that it can change at the insertion point; in particular, this happens a lot unless you turn off "Detect language automaticallly" at Review tab | Language | Set Proofing Language.

More importantly, make sure that the user has the correct input language (keyboard layout); the input language may also change unexpectedly. It's a good idea to uninstall input languages that aren't needed.
