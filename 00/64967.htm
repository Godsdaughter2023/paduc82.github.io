Security applications -- particularly those from McAfee and Norton -- may not be fully removed by using Control Panel > Add or Remove Programs.

For McAfee, see http://service.mcafee.com/FAQDocument.aspx?lc=1033&id=TS100507 which explains how to use the McAfee Consumer Products Removal tool (which you get from the link in that article).
