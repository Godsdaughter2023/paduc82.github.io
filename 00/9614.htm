conditional formating formula for icon set - Microsoft Community

washingtonsquarepk say:

conditional formating formula for icon set

Hello

I want to automate a conditional formatted cell using the icon sets

 

if I7 is > than H7 then green arrow up

if I7 is < than H7 then red arrow down 

 

Is this possible?

thanks

 

paulcordts say: Step 1 
Create a couple of custom number formats like
0 "▲" and 0 "▼". (Use another character for the triangles if you want, if you need more precision use 0.000 "▼")

Step 2 
Apply Conditional formatting to column J
I like to start with uppermost cell (I will use  J2 for this example)

Condition 1 For values less than 10
Select Cell J2 and the select Conditional Formatting - New Rule
For Rule Type select Use a formula.
For the formula type =I2-h2<10
Click the format button and pick the following 2 options
For Number pick the custom format you created 0 "▼"
For Font select the color red

Condition 2 For values less than 10
Select Cell J2 and the select Conditional Formatting - New Rule
For Rule Type select Use a formula.
For the formula type =I2-h2>10
Click the format button and pick the following 2 options
For Number pick the custom format you created 0 " ▲ "
For Font select the color green

You can then copy this down column J

A couple of notes.

First you did not indicate any formatting if H-I=10, so this will not have an arrow or a color.
Second If your formula in column J is already the difference between H and I (ie =I2-H2), you can simplify the formatting by using the value of the cell in J instead of having to use a formula.

Third I used the character map (in Accessories to find the ▲ character and copy to the clipboard.  You need to use a "standard" you can find so that you can paste this into the custom format.
