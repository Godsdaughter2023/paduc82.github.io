talking moose say:
Outlook for Mac synchronizes messages with your Exchange server. If you make big changes in Outlook such as deleting hundreds of messages with attachments then those changes will need to sync with the server.

First, are you dealing with a lot of large message attachments (sending or receiving)? If so, find another solution for transferring these files. Consider using DropBox to store the files and then just providing links in your messages.

Next, you can set your Exchange account to headers-only mode by going to Outlook menu -->  Preferences... --> Accounts --> <your account> --> Advanced... button --> Server tab and selecting Download headers only.

Empty your Deleted Items folder. Many folks delete messages without emptying the folder. The messages still exist and will still synchronize with the server.

Finally and most importantly, do not store more than a few hundred messages in your Inbox and do not store folders inside your Inbox. This folder synchronizes more than any other folder and should be kept as empty as possible. If Outlook is busy trying to keep thousands and thousands of messages in sync with the server then it will generate a lot of traffic and perform poorly.

Hope this helps!

gord dibben say:

Have I got this right?

You want every sheet but yesterday's locked when you open the workbook?

Copy/paste this code to Thisworkbook module.

Private Sub Workbook_Open()
    Dim ws As Worksheet
    For Each ws In ThisWorkbook.Worksheets
        ws.Protect
        If ws.Range("A1").Value = Date - 1 Then
            ws.Unprotect
        End If
    Next
End Sub

nadiasz say:

Hello,

You can use this procedure:

Public Sub CustomerNet(iStart As Integer, iEnd As Integer, CusID As Integer)
    Dim db As dao.Database
    Dim rs As dao.Recordset
    Dim i As Integer
   
    Set db = CurrentDb
    Set rs = db.OpenRecordset("tblCustNet")
   
    For i = iStart To iEnd
         rs.AddNew
         rs.Fields("ipID") = i
         rs.Fields("ipADDRESS") = "192.168.1." & str(i)
         rs.Fields("CustomerID") = CusID
         rs.Update
    Next
   
    Set rs = Nothing
    Set db = Nothing

End Sub

     CustomerNet 1,254,1
