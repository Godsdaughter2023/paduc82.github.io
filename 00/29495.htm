General

For the latest information on The Sims and The Sims Deluxe Edition visit: http://www.thesims.com. You can visit fan sites and find message boards, FAQs (Frequently Asked Questions), walk-throughs, and hints on the site. Visit the chat area of our site to chat with other players. 


Performance Tips
----------------
-In general, the system is at full capacity when running the simulation at full speed, all graphics enabled, and sounds and music playing. To improve performance, turn graphics options down or off, turn sounds and/or music off, and use lower screen resolution settings. These settings are found by clicking the "Options" button in the Control Panel. You can change the graphics options by clicking the "Graphics Options" button that looks like a monitor. NOTE: The option to turn Anti-Alias off is no longer available or needed. You can change the sound options by clicking the "Sound Options" button that looks like a speaker. You can use the shortcut in the start menu to run at 800x600 rather than 1024x768.
-Running the game with the "Roof" option on can affect system performance. We recommend players use the "Walls Cutaway" option, located in the Control Panel above the time indicator.
-Disabling the "Export HTML" function in the "Play Options" menu can speed up saving of the game. We recommend this for players who will not be using the web page features of The Sims. 
-We recommend that you don't frequently Alt+Tab in and out of the game, especially when the game is saving or loading.


System Requirements
-------------------
- Windows(r) 95, 98, or XP (Windows(r) NT and Windows(r) 2000 are not supported)
- 233 Mhz or faster Intel(r) Pentium II(r) processor 
- 64 MB RAM (128 MB required if using Windows XP or other expansion packs for The Sims)
- 2MB PCI or AGP video card capable of High Color (16-bit Color), 800x600 resolution display. It must have a DirectX 7 or later compatible driver.
- 4x CD-ROM drive using 32-bit Windows(r) 95/98 driver 
- 1.4 GB free hard disk space (additional space required for DirectX 8.0 installation and Windows(r) swap file)
- DirectX 7.0 (or higher) compatible sound card
- A keyboard, mouse, monitor and a power source.

When playing The Sims Deluxe Edition on Windows(r) XP, be sure to use an account that is a member of the Administrators or Power Users groups.

We only support one copy of the game running on Windows(r) XP, even with multiple users logged in.


Video Problems
--------------
-The Diamond Fire GL 1000 Pro, 3DLabs Permedia 2, and Symetric Glyder-Max 2 video cards did not pass our compatibility tests for Windows(r) 95/98. 
-The ATI Rage Fury Maxx, Cirrus Logic 5464 Reference Board, Diamond Stealth II G460, Intel i740, REAL 3D Starfighter, Hercules Terminator Beast, Matrox Mystique 220 MGA-1064SG, and S3 Savage 3D video cards did not pass our compatibility tests for Windows(r) XP. 
-Although 3D acceleration won't improve game performance, newer 2D/3D video cards tend to have better 2D support than older 2D cards.
-If The Sims screen appears fuzzy after switching between applications by pressing Alt+Tab, this is likely a result of outdated video drivers. 
-If you are still experiencing problems, try updating your video card drivers.

Diagnostic tools:
If you are still having problems with video, try running the game from a shortcut with some of these command line options:
-w (forces game into windowed mode)
-r1024x768 (forces 1024x768 screen size)
-skip_intro (skips the intro movie)
-skip_verify (This skips verification for OS, DirectX version, a compatible graphics card, the game already running, not enough memory/hard disk space, and a Pentium processor or better; it will try to set 16-bit color in windowed mode if it is not set to 16-bit color. Use this if the game will not start at all. It provides a possible solution for some laptop and audio/video workstation problems. Make sure you run 800x600 mode and use 16 bit or higher color with this setting.) 

To create a shortcut, right click on Sims.exe and choose "Create Shortcut". To edit the shortcut, right click on the shortcut and select "Properties." At the end of the line called "Target", add a space after the existing text and enter your command line option (such as -skip_intro).

When running the game with video cards using 2MB of RAM, the buy/build mode menu in the lower left hand corner flickers as you pan around the screen.


Sound Problems
--------------
Diagnostic tools: 
If you are having problems with sound, try running the game from a shortcut with the "-nosound" command line option. This will run the game with no sound or music. Try this only if you need to determine if your sound card is causing problems with the game.

Turning the sound effects, voice, and music all the way down in the game will effectively turn all the sound off. These options are located in the Control Panel under "Options" and then "Sound Options". 

Music only works from mapped drives (e.g., g:\), not "universal naming convention" (UNC) devices (e.g.,\\myserver).

No Sound with USB Speakers:  
-DirectSound does not work properly on some systems with USB speakers. 
-If you have USB speakers and are not getting any sound then you need to turn off sound acceleration on the Multimedia control panel.

Windows(r) 2000
---------------
The Sims Deluxe Edition does not support Windows(r) 2000. However, it does run on many Windows(r) 2000 systems. One way to avoid some of the compatibility issues is to install and play the game (and any expansion packs) as an Administrator.

Both the installer and the game set registry entries under the LOCAL_MACHINE tree. This registry tree is protected in Windows(r) 2000 so that only Administrators can modify it. Some game features expect to see certain registry entries there in order to detect what game version you have, what expansion packs are installed, and other important details.

If you do not install The Sims Deluxe Edition or any of The Sims expansion packs as an Administrator under Windows(r) 2000, you may get unsatisfactory results.

Regardless of your results, The Sims Deluxe Edition does not support Windows(r) 2000. EA tech support cannot solve Windows(r) 2000 problems with The Sims Deluxe Edition.

Importing Families
------------------
Family Importing in The Sims Deluxe Edition
You will notice a button on the neighborhood screen that looks like a house with a plus sign. This is the family importing button. Instead of automatically prompting you to import a house as soon as it is in the import folder, The Sims Deluxe Edition gives you the option to import families you want into the neighborhood you specify. Simply change to your desired neighborhood with the neighborhood arrow buttons, and then click the import family button to import your downloaded families. A dialog box will appear, prompting you to import the families that you have downloaded.

The Import Family button will only function if you have a family to import in your userdata import folder. There is only one import folder, but houses in it can be imported to any neighborhood you wish. When houses are downloaded from The Sims Exchange, they are first downloaded to the Import folder until the player imports them into a neighborhood. In a typical install, the Import folder can be found in C:\Program Files\Maxis\The Sims\UserData\Import. Likewise, when a house is saved, it will appear in the Export folder C:\Program Files\Maxis\The Sims\UserData\Export. (These are automatic functions that the player does not need to manage.)

Foreign language versions of The Sims Deluxe Edition will NOT have the ability to upload or download families.

If you do not have The Sims installed on your computer, you must install The Sims Deluxe Edition before installing any expansion packs. NOTE: The Sims Deluxe Edition includes Livin' Large, so there is no need to install Livin' Large after you have installed The Sims Deluxe Edition. 

If you already have The Sims (and any expansion packs) installed on your computer, you can install The Sims Deluxe Edition over these as an expansion pack.

Please click OK.  The game will run The Sims Deluxe Edition's installer to update the game files as needed.


Last Resort: Reinstalling
-------------------------
If everything above fails, and you're still having problems with the game, here's what you do:

1) Move ALL your UserData directories (your neighborhoods) somewhere safe, like your desktop.
2) In the Start menu, under The Sims, choose Uninstall The Sims. Make sure that you have tried to launch the game at least once prior to uninstalling, so that any maintenance installer files get copied to your hard drive. 
3) When the uninstall process is complete, delete all remaining files and folders in the location where you had previously installed The Sims (by default, this is C:\Program Files\Maxis\The Sims\).
4) Install The Sims Deluxe Edition.
5) Install House Party if applicable. 
6) Install Hot Date if applicable. NOTE: you may occasionally receive a request to insert "The Sims Livin' Large CD" when installing Hot Date. If this happens, insert CD #1 for The Sims Deluxe Edition and click "Next".  
7) Install Vacation if applicable.
8) Move your backed-up neighborhood directories from Step 1 back into your "The Sims" folder. Click yes to overwrite if Windows(r) asks to overwrite a file.
9) Put CD #1 for The Sims Deluxe Edition back into your CD-ROM drive.
10) Play! You've earned it. 
When you launch the game, you will get a dialog that says "It has come to our attention that you have installed another Sims expansion pack after The Sims Deluxe Edition.  For this to work properly, The Sims Deluxe Edition installer will now update your installation.  Select OK to proceed." Please click OK.  The game will run The Sims Deluxe Edition's installer to update the game files as needed.

Neighborhoods
-------------
The Sims Deluxe Edition supports multiple neighborhoods. You can select new neighborhoods by using the arrow buttons in the upper left portion of the neighborhood view screen. The first five (5) neighborhoods are created by The Sims Deluxe Edition. If you are installing The Sims Deluxe Edition over a current installation of The Sims, whether alone or with other expansion packs, your current neighborhoods should not get overwritten with a normal install process. The first neighborhood, Neighborhood 1, is reserved for the existing neighborhood players have created since The Sims. Neighborhoods 2,3,4 and 5 are created by Livin' Large and Deluxe Edition. Neighborhoods 6, 7, and 8 are created by House Party, Hot Date, and Vacation. It is possible for users to have even more neighborhoods in The Sims, with any of the expansion packs installed.  Here's how:
- Make a copy of the TemplateUserData folder.  Place it in the same directory as the other UserData folders.
- Rename this copy to UserDataXX, where XX is an unused number (check the existing UserData folder numbers) up to 99. We do not recommend you use neighborhoods 2-5 after you have installed Livin' Large or Deluxe Edition, or neighborhoods 6-8 after you have installed House Party, Hot Date, or Vacation. See the "Last Resort: Reinstalling" section for more details. 
- You can do this before you run the game, or while you are alt-tabbed away from the game.  Either way, the game should pick up the fact that you added a new neighborhood.

Here's an example:
- You're running the game, and you decide you want to add a new neighborhood to hold some imported houses you just found on The Sims Exchange at www.thesims.com.
- You alt-tab to the Windows(r) desktop, find the installation folder for The Sims, and make a copy of TemplateUserData.  Rename this folder to UserData23.
- Alt-tab back to the game.  As you scroll through the different neighborhoods, "23" will show up.  This refers to your new UserData23 directory.

As I suspected, the weak link there is your integrated graphics - the Intel GMA910GL just doesn't have the juice (nor does it have the WHQL-certified WDDM drivers needed) to properly support DirectX (nor a few others things) fully in hardware (not on Vista, nor Win7.)  Even the GMA950 chip had issues in Media Center (going all the way back to Media Center 2005); the minimum that actually really worked - ever - was the GMA x3000. 
In other words, you're gonna need a graphics card.  But, you need not spend a lot of money - a decent PCIe card (assuming that you have a PCIe x16 slot) can be had for <$50.

AGP slot:  http://en.wikipedia.org/wiki/File:AGP_slot.jpg
In both photos, the mounting bracket is on your left.
