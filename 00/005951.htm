the top), scroll down to the bottom of the screen. If you have another anti-

virus, there will be a "Periodic Scanning" button present. Setting this button

to on places Windows Defender in mode and allows you to scan

with Windows Defender as you desire. It is not offering "Real Time" virus

protection. Your other program is doing that, but you now have the option

to scan with Windows Defender, if you wish.

resolve virus problems, since it runs without Windows being loaded. That

eliminates interference from the virus itself during the scan. The scan can

be accessed by clicking advanced

screen. Then follow the obvious options that follow. The offline program

will reboot your machine, run the program, then reboot again. Any virus

that is found will be quarantined. If any were found, you can see the results

in Threat History / Quarantined threats.

1.  Change the RecordSource property of the report to:

SELECT * FROM [EAC 1] WHERE ID = Forms![Eac 1]!ID;

This will restrict the report to the current record selected in the form.

2.  In the Command159 button's embedded macro insert a row before the current action which calls the RunCommand action with the SaveRecord command.  This will ensure the record is saved before outputting the report as a PDF file.  Otherwise any unsaved changes in the form would not be reflected in the report.
