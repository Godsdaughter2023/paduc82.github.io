e.j. gun say:

Here is some very simple VBA (which will be inside your Excel file, not your Word file) that will read all the paragraphs in a Word document and put them into the active worksheet of your Excel document.  Test it out to see if it does what you want.  You might need to modify it to suit your needs, but it will give you the idea.

 

Note:  this code assumes you have the Word document already open and that it is the active document.

 

Eric

 

Option Explicit

 

Sub GetWordParagraphs()
    Dim i As Long
    Dim wdObj As Word.Application
    Dim wdDoc As Word.Document
    Dim wdPara As Word.Paragraph
'
    Set wdObj = GetObject(, "Word.Application")
    Set wdDoc = wdObj.ActiveDocument
'
    i = 0
    For Each wdPara In wdDoc.Paragraphs
        i = i + 1

' The next line replaces the Word line breaks with "ALT ENTER" Excel line breaks
        ActiveSheet.Cells(i, 1) = Replace(wdPara.Range.Text, Chr(11), Chr(10))
    Next wdPara
End Sub
