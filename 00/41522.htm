A) If the hard drives are not encrypted, you may try performing a system restore from the Windows Recovery Environment.

i) Boot the computer from the Windows installation disc

iii)  Now select your install Language, Time and Currency Format, plus keyboard type, once you are happy with the settings, Click on "NEXT" to continue.
iv) From the next screen click on "Repair Your Computer" to continue. Windows will now search for Windows Installations.
v) From the next screen select the operating system on which you wish to do a repair. Select it, then press "NEXT" to continue
vi) You are now at the "Recovery Options" menu.
vii) From the "Recovery Options" menu select "SYSTEM RESTORE".
viii) The "Restore files and settings" menu will appear, click "NEXT" to continue.
Now from a selection of "Restore points" select one which you wish to restore too. Then click on "NEXT" to continue. You are now given an option to restore other disks along with the operating system drive. If you have no problems with them then just restore the main drive.
 
ix) Now you are asked to confirm the restore point. Click "FINISH" to continue, or "BACK" to select another drive or restore point. You are now asked one final time to confirm that you wish to continue. Click "YES" to continue or "NO" to select another drive or restore point.

x) "System Restore" will now initialize.

System Restore will now restore files, Finalize the restore.
Once done, it will tell you if it has been successful or not, if so click "RESTART" to restart the computer, if not then try a different date.
Once rebooted and you get to the desktop, you will get a confirmation of the restores success. Click "CLOSE" to continue.
