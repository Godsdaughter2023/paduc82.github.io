This normally happens when the current user profile's cursor/ touchpad settings are corrupt.

To transfer the profile settings, follow the suggestions below:

a.       Log on as a user other than the new user you just created or the user that you want to copy files from.

b.      Press Windows key and x together and select File Explorer.

c.       Click/tap the View tab, and then Click/tap Hidden items.

d.      Now navigate to C:\Users\Old user\

e.      Copy all the files, except these three files NtUser.dat, NtUser.ini, NtUser.log.

f.       Navigate to C:\Users\New user\

g.      Paste all the files here.

h.      Restart the computer and login to New user and check.

i.       Once you are confident that all the settings and data are transferred to the new user account, you may delete old user account.

c) In the System Recovery Options dialog box, click Command Prompt.

d) Type attrib c:\boot\bcd (file:///c:/boot/bcd) -h -r -s and hit Enter. Again from command prompt type Bootrec /RebuildBcd, and then press ENTER.

o If the Bootrec.exe tool runs successfully, it presents you with an installation path of a Windows directory. To add the entry to the BCD store, type Yes. A confirmation message appears that indicates the entry was added successfully.

bcdedit /enum all

In the Windows Boot Loader section of the output from this command, note the GUID that is listed for resumeobject. You will use this GUID later.

bcdedit -create {bootmgr} -d "Description"

Note Descriptionrepresents the description for the new entry.

vincent wansink say:

A Microsoft account is online, which means you are logging into your computer with your Microsoft email - outlook.com, hotmail.com, live.com, etc.  (or another email account that you've designated as your username for your Microsoft account) so that within Windows you will have access to all of your online account information such as your email, calendar, onedrive, etc. without having to separately login to those things.  

With a local account you can either login with a user name/password that is stored on your computer, or you can set it to unlock without having to log in.  

One way to tell what account type you are using is to go to www.outlook.com.  If it asks you to log in, then you are using a local account.  If you are automatically logged in (and it displays your email inbox) then you are logged into your computer with your Microsoft account.
