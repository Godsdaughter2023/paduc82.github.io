DMZ stands for Demilitarized Zone and it is used to place a PC or consul outside of any firewall meaning it has no protection which is fine for the Xbox.

To do this first you need to give you Xbox a static IP Address.

1) Go to Settings – Network – Network Setting

2) Select Advanced Settings and take note of the IP Address, Subnet Mask Gateway and both Primary and Secondary DNS

3) Select IP Settings – Manual

4) Enter an IP Address which should be something like 192.168.x.2 where x is the third number of the IP Address that you wrote down and the forth is any number in the between 1 – 255 and select enter (1 is normally used by the modem)

5) Enter the Subnet Mask from step 2 and select enter

6) Enter the Gateway from step 2 and select enter

7) Enter the Primary DNS from step 2 and select enter

8) Enter the Secondary DNS from step 2 and select enter

9) Once this is done it will check your connection and if all is well hit B to save the settings

10) Using a web browser go to your routers configuration settings and enable DMZ and enter the IP Address you just assigned to your Xbox this should then place it outside the firewall.

taidara say:

A Teredo server is listening on network port 3544 (UDP) and the FRITZ!Box is, by default, configured to block port 3544 for security reasons.

So with this in mind you would have to login to your FRITZ!Box web utility interface and disable the option: "Teredo filter enabled".

For this to work, you will have to take more sure "Advanced View" is enabled on the user interface of your FRITZ!Box.

- Click "Filters" in the "Internet" menu.

- Click on the "Lists" tab.

- Disable the option "Teredo filter enabled".

- Click "Apply" to save your settings.

I would also make sure IPv6 is completely disabled for all interfaces on your FRITZ!Box.

At this point, make sure only UPnP (Universal Plug and Play) is enabled, as the feature to forwarding (automatically) network ports used by Xbox Live.

Don't forget to "Apply" your configuration and reboot your gateway afterwards.

It also creates a Log file with date and time, so that there is a permanent log of what was copied, when.

My user folder is called "john-" so that need changing to your own user name.

contents of Filebak.bat;

dir D:\
Dir D:\ >>  C:\Users\john-\Documents\XDocsLog.txt
rem This copies the DOCS folder. Ctrl C to exit. (40secs for 40Gig 3.6Hz I7 Win10) 
PAUSE
Date /T >> C:\Users\john-\Documents\XDocsLog.txt
Time /T >> C:\Users\john-\Documents\XDocsLog.txt
xcopy C:\Users\john-\Documents D:\Docs /s/d/y >> C:\Users\john-\Documents\XDocsLog.txt
more < C:\Users\john-\Documents\XDocsLog.txt 
PAUSE
Exit

To customise;Right click the .bat file to edit, and customise the folder names of source Xcopy copies from, and destination to where copied to.

">>" redirects output to a Logfile. and "more" outputs that Logfile to the screen with information as to how many new, and changed files copied.
