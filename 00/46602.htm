I can only assume you are referring to Skype for Business instead of Skype. There are two very different programs. Issues with Skype for Business should be addressed in the Office forums in the Skype for Business sub category. In any case, I will cover them both:

Standard Skype:

If you are running Windows 10, Skype comes with the OS in App form. Windows 10 UWP Apps are removed from Apps & Features instead of Programs & Features. You can also right click on an App shortcut and select the option to uninstall it. Every other version of Skype in any supported Windows can be removed from Programs & Features or Add/Remove Programs if the install is not corrupted.

Skype for Business:

Since Skype for Business is part of Office packaged in a click-to-run installer, you have to uninstall Office in order to remove Skype for Business. The alternative to that is manually deleting shortcuts to Skype for Business or its files. Older non click-to-run installers provided you with the option to remove individual components after they were installed. This is why you will see what appears to be conflicting information and screenshots.

Another option for admins, though users can do this as well, is reinstalling Office using the Office deployment tool [askvg.com] to explicitly omit Skype for Business from the installer. If you are using a 365 variant of Office, you would need the Office 365 variant of the deployment tool using the same guidance. This solution still requires you to uninstall Office first.

However, having tried almost everything I could think of to fix the problems listed, I chased down this error and found an answer on:

https://support_microsoft.com/en-us/kb/929833

This suggested running the System File Checker (sfc).

Having done this it showed that there were several corrupt files associated with upgrade KB3081424 which it was able to fix.  

So far, things appear to be working.   Task Scheduler is running again and most of the things I tried in a quick check also worked properly, so this might provide some help to others.
