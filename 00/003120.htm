john korchok say:

First, try and Office Repair:

  1. In Windows, click on the Start orb, then on Control Panel.
  2. Select Programs, then Programs and Features.
  3. If you bought Publisher separately from Office, right-click on Microsoft Publisher 2013 and choose Change. If Publisher came as part of Microsoft Office, then it won't show in the list. In this case, right-click on Microsoft Office Professional 2013 and choose Change.

  5. Allow the process to complete, reboot Windows and test Publisher to see if you can enter the Croatian characters.
