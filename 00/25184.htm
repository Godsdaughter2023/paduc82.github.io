Function RangetoHTML(rng As Range)
' Changed by Ron de Bruin 28-DEC-2006
' Working in Office 2000-2007
    Dim fso As Object
    Dim ts As Object
    Dim TempFile As String
    Dim TempWB As Workbook
    Dim Stringbodyup As String
    Dim Stringbodydown As String
    Dim Cell As Range
   
    For Each Cell In Range("AJ154", "AL155:AL156")
        Stringbodyup = Stringbodyup & Cell.Value & vbNewLine
    Next
   
    For Each Cell In Range("AJ171:AL181")
        Stringbodydown = Stringbodydown & Cell.Value & vbNewLine
    Next
    TempFile = Environ$("temp") & "/" & Format(Now, "ddmmyy hhmm") & ".htm"    HANGs HERE
 
    'Copy the range and create a new workbook to past the data in
    rng.Copy
    Set TempWB = Workbooks.Add(1)
   
    With TempWB.Sheets(1)
        .Cells(1).PasteSpecial Paste:=8
        .Cells(1).PasteSpecial xlPasteValues, , False, False
        .Cells(1).PasteSpecial xlPasteFormats, , False, False
        .Columns(1)("A:C").EntireColumn.AutoFit
        .Cells(1).Select
        Application.CutCopyMode = False
        On Error Resume Next
        .DrawingObjects.Visible = True
        .DrawingObjects.Delete
        On Error GoTo 0
   
    End With
 
    'Publish the sheet to a htm file
    With TempWB.PublishObjects.Add( _
         SourceType:=xlSourceRange, _
         FileName:=TempFile, _
         Sheet:=TempWB.Sheets(1).Name, _
         Source:=TempWB.Sheets(1).UsedRange.Address, _
         HtmlType:=xlHtmlStatic)
        .Publish (True)
    End With
 
    'Read all data from the htm file into RangetoHTML
    Set fso = CreateObject("Scripting.FileSystemObject")
    Set ts = fso.GetFile(TempFile).OpenAsTextStream(1, -2)
    RangetoHTML = ts.ReadAll
    ts.Close
    RangetoHTML = Replace(RangetoHTML, "align=center x:publishsource=", _
                          "align=left x:publishsource=")
 
    'Close TempWB
    TempWB.Close savechanges:=False
 
    'Delete the htm file we used in this function
    Kill TempFile
 
    Set ts = Nothing
    Set fso = Nothing
    Set TempWB = Nothing
End Function

Sway image lock

When using Sway is there anyway of locking the view of the image to look like the second image instead of the first?

jason (office engineer) say:

Hi David,

Choose the exact layout, positioning, or sizing of content is not supported in Sway.  The two layouts above depend upon the style you have chosen, the dimensions of the image, the length of the text, and the dimensions of the screen it is viewed upon.  For instance if you viewed that Sway in a window that was taller than it was wide or vice versa, you'd get different layouts.  The reason for this is that Sway attempts to provide a layout optimized for the device/screen that it is viewed upon and in order to be able to do that, Sway needs to have the flexibility to reposition content so it is not something available to the author.  You could try some different things like grouping or even making the text a caption on the image, but you won't be able to lock it down exactly as you have asked for.

    Replace First space - Microsoft Community

    lkatz2 say:

    Replace First space

    Is there anyway to replace only the first space after the first tab in that row with comma in microsoft word 2010
    graham mayor say:
    The following looks at the first 'word' in the paragraphs. Word treats a tab character as a 'word', so if the first 'word' is not a tab character (as in your first example), this version adds a tab character to the start of the second 'word'

    Dim oPara As Paragraph
    Dim oRng As Range
    For Each oPara In ActiveDocument.Paragraphs
        If oPara.Range.Words.Count > 1 Then
            If oPara.Range.Words(1).Text <> Chr(9) Then
                Set oRng = oPara.Range.Words(2)
                oRng.InsertBefore Chr(9)
            End If
        End If
    Next oPara
