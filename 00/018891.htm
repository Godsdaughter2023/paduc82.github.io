Excel formulas - adjusting values based of another value - Microsoft Community

shane humphries say:

Excel formulas - adjusting values based of another value

Hi all

I’m creating a database of foods I eat and would like to know if anyone knows of a way that I can have the figures from the green cells adjust accordingly when I modify the figure from the ‘Serving size’ cell?

For example: If I changed the Bananas, raw serving size to 60 all the green cells for that row would half. Just as if I increased the Bananas, raw serving size to 140 all the green cells would adjust accordingly. Of course I do not want any cells with the value of 0 to drop into the negatives.

Thanks in advance.

myall_blues say:

You could put a formula in each green cell that represents the ratio of whatever unit (calories, carbs etc) to the serving size.

So for example for bananas, the calories is 91 per 120 grams. So if serving size is in column C, bananas in row 3, then place a formula in E3 (the calories column) as follows:

=91/120*C3 (or if you prefer =0.7583*C3)

Adjusting the serving size value in C3 will then adjust the corresponding calories. Similarly for the other green columns.

For the zero values, because the numerator in the ratio is zero they will remain zero (e.g cholesterol for bananas =0/120*C3, or just leave them as zero value)

Regards

Murray

teylyn say:

Hello, 

may I suggest a different approach. 

Instead of logging the nutritional values based on serving size, log them based on 100 grams (all food info tables have that measure, anyway), so it's easy to get by. This should be done in a static table that is used for looking up the nutritional values.

Then you can build you daily log and enter the serving size of that serving in grams. Use lookup formulas to find the nutritional value from your 100 grams based table, and calculate the nutritional value in relation to the serving size. 

So, if 100g of food has 80 calories, a 120 serving of that food will be =([100gCals]/100)*[ServingSize]

Does that make sense?

To illustrate: the red table in the screenshot is called FoodList. The formula in the blue table looks up the 100gCals value, divides it by 100 and then multiplies the result with the serving size in the blue table. 

Now you can weigh your bananas and enter the serving size of each banana, and each banana can be a different serving size. But you don't change the nutritional values with each change of serving size.
