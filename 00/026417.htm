what does "limit CPU usage mean"? - Microsoft Community

kittychavez say:

what does "limit CPU usage mean"?

what does "limit CPU usage mean"?
kosh vorlon say:

Hi Kitty,

 

Scans can take up a great deal of CPU resources.  The option to limit CPU usage allows you to control how much is used by MSE during scans.  If you keep it around the 20% default, then when running quick scans and sometimes even full scans (after the first one), you will most likely be able to work on the computer as normal while the scan is running in the background.  If you keep the computer on 24/7 and do scans at night while sleeping, you can increase the usage to 50-60% or even higher since it won't disturb what you're doing as you'll be asleep while it occurs.  I run mine in the middle of the day while working and have it set at 20% and I hardly notice any difference unless I try to do some resource-intensive tasks or too many things (like have 30 open IE windows at the same time which sometimes I don't pay enough attention to) and then it will begin to show signs of getting sluggish, but all I need to do is close things I really aren't using and all is fine again.  I also get to see the scan in progress if I want and review the results when it is finished.  So in essence it's how to control how much of your CPU is dedicated to MSE scanning so you can either keep it low so it doesn't have too much of an effect while you are working (but will take longer to finish), or keep it high so it will finish faster (hopefully before you awaken if started at night).

 

The thing about scans is the computer needs to be on and connected to the internet if configured properly.  MSE will not wake the computer from shutdown or sleep or hibernation to do the scan even if scheduled.  MSE will not prevent the computer from going into sleep, hibernation, or shutdown due to power saving settings (and thus cancelling the scan which will not save where it was or begin again the next time you start - it will wait for the next scheduled scan and in the meantime enough time may pass that you get a warning from MSE that a scan is needed because it's been too long since the last one).  That's why controlling CPU usage allows you some control over how much resources it uses and thereby how long it takes to finish.

 

I hope this helps.

 

Good luck!
