Nathan's code will work for you.  What you have to realize is that when using code for opening a form, it has a certain default.  If you type in DoCmd.OpenForm and then a space you will see the arguments for it.  And if you notice it will show acFormView = acNormal  as the default.  So if you don't use an argument that specifically is acFormDS then it will use the default which is a normal single view form view.
