hula_baloo say:

 You posted this in a Windows 7 forum, but Windows 7 will not install in a FAT32 partition.  Please explain.  Assuming you want to convert other than a Windows 7 partition, here's the procedure.

To convert a hard disk or partition to NTFS format

  1. Close any open programs running on the partition or logical drive to be converted.

  2. Click the Start button , click All Programs, click Accessories, right-click Command Prompt, and then click Run as administrator. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

  3. At the command prompt, type convert drive_letter: /fs:ntfs, where drive_letter is the letter of the drive you want to convert, and then press Enter. For example, convert E: /fs:ntfs would convert drive E to the NTFS format.

  4. Type the name of the volume you want to convert, and then press Enter. You must use the existing name of the volume or the conversion will be canceled. You can view all available volumes in the Computer folder under Hard Disk Drives.

I'm sorry to tell you that the built on sound card which is UDA038_build03L never had Vista drivers made for it. What that means to you is that the sound card will not work in Windows 7. You only have a few options, the first option is to buy a new pci sound card with vista or higher drivers, second you can try using XP compatibility mode, and last thing you can try is buying a new Motherboard that is Vista / Windows 7 compatible. I was looking over your board which can be found here it was designed for Windows XP which means it wasn't intended for Vista or Windows 7.

Now if you intend to continue using this board then options one and two would be in your best interest to try. The first option is a better bet to work then option two for the drivers that were made for XP more then likely will not work. If you would like to try compatibility mode I would first download the latest driver from Asrock for you board which you can find here. Then right click the file and go to properties, next select the tab compatibility and select the check box to select XP SP3, and then push apply then ok to close. Now right click the file again and select run as administrator push yes and follow the instructions to install the driver. If the driver doesn't work then make sure to uninstall it from your system.
