Footnote superscript in text - Microsoft Community

marcraboy say:

Footnote superscript in text

Whenever I insert a footnote I need to manually make the number in the text appear as superscript. How can I do this automatically?
hansv mvp say:
  1. Select the footnote number.
  2. Click the little arrow in the lower right hand cornder of the Styles group on the Home tab of the ribbon to display the Styles task pane.
  3. Click the Style Inspector button at the bottom of the task pane (it's the second button).
  4. You should see "Footnote Reference" in the "Text level formatting" box.
  5. Click the dropdown arrow to the right and select Modify... from the dropdown menu (you can also right-click the style and select Modify... from the context menu).
  6. In the Modify Style dialog, click Format > Font...
  7. Tick the check box Superscript.
  8. Click OK to return to the Modify Style dialog.
  9. If you want to apply the change to all new documents, select the radio button "New documents based on this template".
  10. Click OK.

Deleting the Word Data key in the Registry is a common fix for a lot of Word problems.
