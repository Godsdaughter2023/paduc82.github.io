If it's about the specific objects/methods/properties you need to be looking at, that depends on what type of form fields you have to deal with. In Word 2010, there are three (at least!):
 a. the newest sort (Content Controls). These are represented by the Document.ContentControls collection and the Document.ContentControl object
 b. the older sort ("legacy form fields"). You can iterate these using the Document.FormFields collection and modify values using the Document.FormField object. Each type of field (text, dropdown, checkbox) has a separate subobject with its own properties.
 c. the "ActiveX" controls. I doubt you have to deal with them.

Documents can have all three types (and a mix of (a) and (b) is not unusual). greginmich say:

Manual scans are easy now that the icon is always there; and it’s also relatively easy to schedule a daily Quick Scan that runs in the Defender UI (see the link in my previous reply). The directions provided here by SpiritX MS MVP should also work in most cases, although the daily Automatic Maintenance Quick Scan might be a little disconcerting when it runs in a “Command Prompt” window with an only-on-idle priority (there are ways to fix that).

Also, I think the PowerShell command for setting the signature update interval is working correctly at this point (some of the PowerShell commands appear to be either nonfunctional or dysfunctional), if you're looking for a way to increase the update frequency:

-SignatureUpdateInterval<UInt32>

Specifies the interval, in hours, at which to check for definition updates. The acceptable values for this parameter are: integers from 1 through 24. If you do not specify a value for this parameter, Windows Defender checks at the default interval. You can use this parameter instead of the SignatureScheduleDay parameter and SignatureScheduleTime parameter.

https://technet.microsoft.com/en-us/library/dn433291.aspx?f=255&MSPPError=-2147217396

To set the update interval to 4 hours, for example, you would run this command at the elevated PowerShell prompt (copy, paste, and enter):

Set-MpPreference -SignatureUpdateInterval 4
