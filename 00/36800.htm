Go to Power Options

Go to Change Plan Settings (under whatever plan you are using)

Computer management>>disk management

If there is no EFI partition, then no UEFI

And there is no advantage using UEFI except for drive support over 2TB in size. Most people will never fill up a 2TB drive to begin with.

BIOS installation is the best by far. There are so many issues with PC's/Laptops with UEFI.

And you get all the same start up recovery options on a Legacy BIOS install.

BIOS:

MSI partition (Hidden)

WRE partition (NTFS)

C:\ OS partition (NTFS)

and any extra partitions.

UEFI:

MSI partition (Hidden)

EFI (FAT..)

WRE partition (NTFS)

C:\ OS partition (NTFS)

and any extra partitions.
