I have also two question.

1) Is windows defender also work as a internet security?

2) Is it safe to use internet banking in my computer where only windows defender is installed? I have no other internet security installed in my computer.

-Soumen

Though in the strict sense of what some Internet Security suites contain today, Windows Defender alone doesn't necessarily contain all of the features that some of these do such as identity theft protection for example, the commonly included components required to provide internet security do exist within Windows Defender as well as the other Microsoft provided applications for Windows 8.

This is an area where there is much confusion, since many people assume that the term internet security implies that these add-on tools like identity theft are included, when in truth these are simply modular optional add-ons that some vendors use to try and increase the value of their product offering.

Microsoft on the other hand has chosen to keep the entire protection within Windows modular and application specific, with the necessary protection for specific applications contained within them and the common components such as network and file system (malware) protection contained in the Windows Defender or 3rd-party security application.  In the case of Windows 8 these modules are far more defined and regulated than in the past since there are specific requirements in order for the security application to provide compliant drivers for Early Launch Anti-Malware (ELAM), Secure Boot, the core anti-virus and related anti-malware detection, blocking and removal.

In the specific case of Windows Defender the Network Inspection System is an additional component that provides network level protection for known vulnerabilities.  This along with various other anti-malware modules are all considered part of Defender, even though some of these operate somewhat independently and simply use the Defender GUI for display or configuration.

On the other hand, security for Internet Explorer for example is mostly self-contained within that application, though for some operations it may pass control temporarily to the installed anti-malware application (Defender or 3rd-party) as a final sanity check once say a file download has been completed.

In a unique reversal of this trend, the SmartScreen Filter included in Internet Explorer 10/11 has also been integrated directly into the core networking components of Windows 8 such that file downloads and some other types of Internet access performed by any application can now be examined by that protection module.

So the term internet security has never really been well defined and with the significant changes in the design of Windows 8, it's no longer possible for 3rd-party security applications to completely bypass Windows control of the network and other data streams, so fundamentally all of these are now operating the same in the background regardless of what they may claim in their GUI or advertising.

So yes Windows Defender provides internet security, it simply doesn't include some popular, though often of questionable value add-on modules.
