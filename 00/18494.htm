Any fonts with joined up writing?? - Microsoft Community

plonky say:

Any fonts with joined up writing??

Hi.

I need a font (preferably with bold and italic options) that is like handwriting and all the letters must join up, even if they are in capitals. The style is not that important, but all letters must join together. Any suggestions would be greatly appreciated.

suzanne s. barnhill say:

You will not find a handwriting font with bold and italic variants because "script" is considered "italic" already. What you are looking for is a "flowing script." There are many such, but not all are supplied with Office 2003. Among those that are are Brush Script (not really a "handwriting font"), Edwardian Script ITC, French Script, Kunstler Script, Lucida Handwriting Italic, Mistral, Palace Script, Rage Italic, Script MT Bold, and Vladimir Script. You can see a list and samples of all fonts supplied with Office Professional 2003 at http://www.microsoft.com/typography/fonts/product.aspx?PID=143.

 

For other free script and handwriting fonts, see http://www.dafont.com/.

 

Note that probably no script font will join capitals because All Caps in generally a no-no in handwriting fonts.
