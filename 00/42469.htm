There are probably simpler methods but this one certainly works: Create a dummy account, then cause it to be hidden from the logon screen. The following batch file will do it for you when executed under elevated administrative privileges. You must, of course, disable the usual auto-logon setting via this command:

control userpasswords2

And here is the batch file. Use copy & paste - do not retype it! Note that the line starting with
echo >>%RegFile%" is a very long line that ends with "UserList]". Do not break it up!

@echo off
set RegFile="%temp%\reg.reg"
echo >%RegFile%  REGEDIT4
echo >>%RegFile% [HKey_Local_Machine\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Winlogon\SpecialAccounts\UserList]
echo >>%RegFile% "DummyUser"=dword:00000000
regedit %RegFile%
net user DummyUser supercalifrag /add

how to edit out background noises? - Microsoft Community

jacquelinechambers say:

how to edit out background noises?

In Windows Movie Maker, how are background noises edited out?
considerate_guy say:

It depends on whether there is other sound that you want and if that sound is on the same track or on the added music track. There is sound that is associated with your video and sound that you add (like music).

On the Video Tools | Edit tab (which apepars when you select a clip), you can change the Video Volume. You can use that to reduce the volume of your original clip, all the way down to 0, if you like.  This would work well if you wanted to remove all the audio from the video clip and only hear the music.

That's how you would do it for a single clip, or several clips.  If instead you want to do this for all your clips (only have the added audio), you can use Audio Mix on the Project tab.

These routines work if you want to remove the audio completely.  However, if you have audio that has background noise (birds, cars, wind noise, etc.) and foreground noise (someone speaking to the camera, for example) that's not something that Movie Maker can separate.  All the sound is on one track and Movie Maker can't distinguish background from foreground noise.

For that you'd need specialized software that might be able to adjust volume based on frequency trying to isolate the desired frequencies (say voice) from undesired frequencies (car honking, background hum, etc.)

marc cai say:

Hello Rhonda,


We appreciate you being part of the community. To answer your query about Indexing, it helps the system identify the file location based on its details. Therefore, we may think that the file indexing is innocent in the issue.

Since the files are removed from the system, we are wondering what makes you think that the files are still in the same folder? If the files are deleted, they're normally move to the Recycle bin. If you're using OneDrive and there is where you put the files, the system could have synced the files from the online storage.

One of the best ways to free up storage spaces is to use Disk cleanup:


  1. Open Disk Cleanup by clicking the Start button . In the search box, type Disk Cleanup, and then, in the list of results, select Disk Cleanup.
  2. If prompted, select the drive that you want to clean up, and then select OK.
  3. In the Disk Cleanup dialog box in the Description section, select Clean up system files.
  4. If prompted, select the drive that you want to clean up, and then select OK.
  5. In the Disk Cleanup dialog box, on the Disk Cleanup tab, select the check boxes for the file types that you want to delete, and then select OK. To remove data from a previous installation of Windows, select the Previous Windows installation(s) check box.
  6. In the message that appears, select Delete files.
  7. Go back to Computer in File Explorer, select the drive you cleaned up, and then select Refresh. If you need to free more space (the drive is still marked red), go to the next tip.

Feel free to chime back when you need further info.
