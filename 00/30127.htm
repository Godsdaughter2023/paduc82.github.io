joelsbraun say:

Hi,

Bitlocker To Go Reader is not a virus, it's a program that allows users to read flash drives that have been encrypted by a version of Windows which integrates Bitlocker to go drive encryption (Windows 7 Ultimate/Enterprise).

On Windows XP and Windows Vista, a flash drive that has been encrypted by bitlocker cannot be read without the assistance of the Bitlocker To Go reader. It's possible that your friend had a Bitlocker encrypted drive and as such had to install the reader.

For more information on Bitlocker and how it works, see: http://windows.microsoft.com/en-US/windows7/products/features/bitlocker
