Hello,

I know it is frustrating and I can imagine the inconvenience you are experiencing. I will certainly help you with this issue.

Windows 8.1 introduces a new feature that allows you schedule and run Automatic Maintenance on your computer. The task when run, will carry out task like security updating and scanning, Windows software updates, Disk Defragmentation, Disk Volume Errors, System Diagnostics, etc and that's the reason you have this message when your compute is idle.

I would suggest you to run the Automatic Maintenance completely and check if it helps.

Windows Automatic Maintenance minimizes impact to the PC by scheduling maintenance only when the PC is on and idle.

Automatic Maintenance in Windows 8.1

  1. Press Windows + X key on the keyboard

  2. Click on Control Panel and navigate to Action Center.

  3. Under Maintenance, click on the small arrow on the right side.

  4. This will expose Automatic Maintenance.

  5. If you find that this is not being monitored, you may click on Turn on messages about Automatic maintenance.

This will let Action Center monitor Automatic Maintenance and give you any messages via the Action Center icon in your taskbar.

If you wish to disable these messages, you can also do so by unchecking the option under.

  1. Press Windows + X key on the keyboard

  2. Click on Control Panel and navigate to Action Center.

  3. Go to Change Action Center settings

  4. Click on Maintenance messages

  5. By default, Automatic Maintenance runs in the fully Automatic Mode

If for some reason you wish to start the Maintenance task right away, you can do so by clicking on the Run Maintenance button. This will allow you to run this task at unscheduled times. This is referred to as the User Initiated Mode.

To stop the Maintenance job, you can simply move the mouse cursor. If it does not help, then you may click on the Stop maintenance button anytime. The task will be stopped in a few seconds.

When Automatic Maintenance is being run, you will see its indication in the taskbar icon.

To Disable Automatic Maintenance in Windows 8.1

  1. Click the Start button.
  2. Click Control Panel .
  3. Click System and Security .
  4. Click Administrative Tools .
  5. Double-click Task Scheduler .
  6. Right-click on Idle Maintenance, Maintenance Configurator and Regular Maintenance and select Disable.

More information: http://msdn.microsoft.com/en-us/library/windows/desktop/jj835985(v=vs.85).aspx
