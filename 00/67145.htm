You would not run the compatibility trouble shooter for the operating system it IS running on. The objest of the utility is to try and give the program the environment it was running in successfully with a previous O/S but within Windows 10. What is RAM and how does it work?

Let's start with the most basic elements of the question. RAM is where data is stored before processing. RAM stands for Random Access Memory, and it consists of dynamic memory chips that can be written and rewritten with data very quickly. Unlike your hard drive, however, the memory used for RAM is also volatile, meaning that it only holds onto that data while the chip is powered, so it's not designed to hold information long term.

Essentially, RAM is the memory that actually lets you work with the data needed to run programs and open files. Whenever your computer loads a program or opens a file, it opens up that data in RAM. Even your operating system uses RAM when it's running. If you want to run a program, it pulls the data from long-term storage on your hard drive to short-term storage in RAM, where it can be accessed quickly enough for smooth operation.

andre da costa say:

1. If my hardware is corrupt, and I have to replace it, will Microsoft let me activate my Windows on the newly installed motherboard?

If its a retail full Windows 10 license, yes or it was upgraded from a retail Windows 7 or Windows 8 license.

If yes, for how many times is this process activation process allowed. Say, somebody had a motherboard failure the second time. Will he be allowed to activate again with the same license?

Yes, but I would be careful of not exceeding 5 activation's, this raises red flags and Microsoft will block the key permanently.

2. Is there a yearly limit to the license, such as 3 years or 5 years, etc? Or, one can keep using the same license until the end of support of the Windows version in question?


You can use it as long as you wish, even when Microsoft no longer supports it

3. I have a Windows PC, i.e., it came pre-installed with Windows. Microsoft says that there is not going to be a Windows 11; there will only be periodic major updates to the Windows 10.  So, a user with one Windows 10 license doesn't ever have to purchase a Windows license again? 

It depends, if the device is still working 10 years from now and the manufacturer still supports it, yes. Lifetime support is dependent on vendor support. If the brand no longer provides updated drivers or general support, then Microsoft has all right to end support for Windows 10 on that specific model.
