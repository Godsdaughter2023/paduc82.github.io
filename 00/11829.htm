SELECT doc_tblObjects.Name AS TableName, doc_tblObjects_1.Name AS FieldName, doc_tblObjects_1.Extra2 AS FieldType, doc_tblObjects_1.Extra3 AS FieldSize

WHERE (((doc_tblObjects_1.TypeID)=11));

No problem.

When you start a program, Office checks to see if that version of Office is default. When you have more than one version, that check will trigger the "configuring" process.

There is a simple fix for this. You have to do a registry hack to add NoReReg registry keys.

Dual Install - Working with Multiple Versions of Office PRE 2016! – NoReReg
http://answers.microsoft.com/en-us/office/wiki/...

https://office.microsoft.com/en-us/word-help/ke...
This tip describes the specific steps and choices during the 2010 custom install. After you have the multiple versions installed you still have some additional steps to do. They are described in this tip:
https://web.archive.org/web/20130603071131/http...
With multiple versions installed, there is one issue you can’t avoid, Default Windows File Associations. Only one app can be used to open the files by double clicking in Windows Explorer. You have to pick which one.
FYI: NoReReg key works with Office 2010 / 2013 installed on Windows 7 and 8 (I tested it myself Rohn007)
Note: in user accounts, when you log in to do the NoReReg hacks, DO NOT start REGEDIT using “Run As Admin”. You will be editing the wrong “Current User” key <grin>

OFFICE 2010 CONFIGURES EACH TIME YOU START
http://support.microsoft.com/kb/2528748
In Method 1, this tip has a Fix it, 50780, that applies the required Registry fix.

You have to add the "NoReReg" key for each application in the Office bundle. Office 2010 is V14 / 14.0 and 2016 is V16 / 16.0.

You can reverse sync pictures from zune to PC, but that should be your last resort.  Zune software transcodes high resolution pictures to a lower resolution to sync on the zune...  saves a lot of space, and you can't tell on the small screen.   You need to copy the pics from your old PC to the new PC... any way you like, such as on an external drive, on a USB memory stick, over a home network... 
If you did this and it's just that they don't show up in the zune collection, then go into zune software->settings->software->collection and change the monitored libraries (win7 and vista) or folders (XP) to the correct location for the pics and the music.  Or move them on the PC to the libraries monitored by zune.

If you do have to reverse sync the pics or music:  connect the zune, click zune, then find the content.  select a bunch and drag it down to the PC icon at the lower left.
