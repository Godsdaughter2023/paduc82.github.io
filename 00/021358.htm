If you have recently clean installed Windows 10 and cannot download - Microsoft Community

a. user say:

If you have recently clean installed Windows 10 and cannot download the latest Insider Build, try this.

This problem has been affecting me and a number of other users for quite some time.

First make sure your OS is activated, and that you are on the Fast Ring.

Here is the workaround that worked for me (and also for other people, apparently):

Open the command prompt as administrator and type:

"net stop wuauserv";

Next, navigate to 'C:\Windows' and delete the entire 'SoftwareDistribution' folder completely.

Back in the elevated CMD, type: 

"net start wuauserv".

Now leave the PC on as long as possible...

A few hours later, the insider build should show up and install flawlessly.

Thanks for the users who helped me with this.

And i hope this helps anyone with the same problem.
