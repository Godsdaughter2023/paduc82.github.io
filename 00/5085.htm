Solution for 'no gateway reply' error for N95 and E90 - Microsoft Community

joshidinesh say:

Solution for 'no gateway reply' error for N95 and E90

I had the 'No gateway reply'error, I tried following which is basic network stuff and it worked.

Initially I was getting IP address by DHCP as below.

IP address : 10.30.100.208
Subnet Mask: 255.255.255.255
Gateway: 10.30.100.1

Above setup works, if you have windows computer, as it will reach to gateway by broadcasting method. In this case IP address and Gateway both are not in the same subnet. so
If its Symbian or other OS who doesn't work on broadcast will not work and throw 'No gateway reply' error.

I tried below:

IP Address: 10.30.100.208
Subnet Mask: 255.255.255.0
Gateway: 10.30.100.1

Try setting IP address manually in
Tools-->Setting-->Connection-->access point-->Select or Accesspoint-->Advance setting and enter all details manually for that WLAN access points.

I have tested on N95 and E90 working on both.

All the best.
