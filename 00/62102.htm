Hi,
Thank you for posting your query in Microsoft Community.

From the description I understand that you are receiving the error message regarding IP conflict error - “Another computer on this network has the same IP address as this computer."

An IP address conflict occurs when two computers on a LAN (local area network) or the Internet have been assigned the same IP address. IP conflicts between two computers normally render either one or both of them unusable for network operations. Windows then notifies one of the clients about this, and requires that the administrator make some changes on the network settings. This can be an issue with the DHCP on the router.

I would be suggesting you these below methods and check if it helps in resolving your issue.

Method 1
1. Click Start Menu and in the Instant Search type “Command Prompt” then right-click and left-click Run as   administrator.
2. Type the first command ipconfig /release and press Enter.
3. Type then the second command ipconfig /renew and press Enter.

Explanation:
The fist command will send a DHCPRELEASE message to the DHCP server to release the current
DHCP configuration and discard the IP address configuration for either all adapters, the second
command Renews DHCP configuration for all adapters.

Troubleshoot blue screen errors <-- read this link
http://windows.microsoft.com/en-us/windows-10/t...

By far the most common causes of BSOD with new home builds are hardware issues or BIOS, chipset, major driver issues. A lot of different Bug_Check also points to hardware. A real problem with any hardware issue in home builds is the lack of known good components to swap in and out.

Remove ALL power then reseat the cards, memory, and cables (on both ends when possible) - actually remove and replace - do not just snug.
New builds often have cards that loosen with time or during the build process. <-- please do not fail to do this as I see it over and over.

Check with the MotherBoard Maker for updated BIOS and chipset drivers. Also for any onboard device driver updates. Do the same at the individual device makers for updated firmware and device drivers (such as Samsung for the SSD and Nvidia).
