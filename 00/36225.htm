cody c. say:

Hi everyone!

Just wanted to clear up some of the confusion.

Marilyn's response did suggest removing WMC from startup, which the OP had not mentioned in their first post. Since the OP didn't return for over 7 days,  it is assumed these steps resolved the issue and is then marked as answer. 

Since many of you have tried those steps and they didn't help, I've unmarked the post.

As for a solution, many of you are seeking a way to simply disable Windows Media Center entirely. I should note that if you're using Windows 7 you can quickly and easily disable this feature using "Turn Windows features on or off". But since most (if not all) of you are using Vista, I'll press on.

There are several ways to skin this cat:

A. Disable the Windows Media Center Services
  1. Click Start, type services.msc and pressEnter
  2. Find Windows Media Center Extender Service and double click it
  3. On the General tab, change Startup Type to Disabled and click OK to close that window
  4. Repeat steps 2 and 3 for Windows Media Center Receiver Service,Windows Media Center Scheduler Service andWindows Media Center Service Launcher
  5. Close the Services window and reboot

B. Disable Windows Media Center via Group Policy (Note: This is not an option for Windows Vista Home Premium users)
  1. Click Start, type gpedit.msc and press Enter
  2. Under Computer Configuration, navigate to Administrative Templates > Windows Components > Windows Media Center
  3. In the right pane, double click the only policy (Do not allow Windows Media Center to run) 
  4. Select Enabled (sounds weird but this will indeed prevent WMC from running) and clickOKto close that Window
  5. Close the Local Group Policy Editor and reboot

C. Disable Windows Media Center via the registry
  1. Click Start, type regedit and press Enter
  2. Navigate to HKEY_LOCAL_MACHINE > SOFTWARE > Policies > Microsoft
  3. Right click on Microsoft and choose New >Key
  4. Name the key WindowsMediaCenter and press Enter
  5. Right click on the key you just created and choose New > DWORD (32-bit) Value
  6. Name the DWORD MediaCenter and press Enter

    You really only need to perform one of the above fixes for Windows Media Center to be disabled.

    Let me know if the above helps resolve your issue!
