alex_ncfc say:

Thanks very much for the reply, Neha.

This is now solved - 

When I first encrypted the MicroSD card in Windows 10 Pro, I did so via the Command Line, which invoked a dialog box giving me the % of the encryption. I thought this was the only way to enable BitLocker encryption and thought this was all there is to it.

As it turns out, if I'd gone through the Control Panel applet, it'd have started a wizard, and one of the questions was whether to enable "compatibility mode", as the other mode was for Windows 10 1511 build only and this is the mode that enabling BitLocker from the CMD prompt uses. Formatting the drive to NTFS and enabling this compatibility mode through the GUI wizard means my MicroSD card is now functional in Windows 8.1 (or indeed any Windows with BitLocker)

a. user say:

Hi James,

Let me help you with the Windows features such as System Repair Disc, System Image, File History, Recovery Drive.

File History: This feature is more limited, and it only backs up the files located in your user account’s libraries. You’ll need to make libraries appear in File Explorer and add folders to these libraries or create new libraries if you want to back up files stored in other folders.

Once you’re done, Windows can automatically create backup copies of your files. You can use it to restore all your files if you ever lose them, restore a single file, or roll back a file to an older version.

System Image: This allows you to create an entire image of your current Windows system, including your operating system, installed programs, settings, and user files. This backup can be stored on an external hard drive, on several DVDs, or on a network location. If you ever need to get your computer back to a specific state, you can restore this backup.

System Repair Disc/ Recovery Drive: Mostly used when the operating system is in no boot situation.

Windows 10 allows you to create a recovery drive (USB) or system repair disc (CD or DVD) that can be used to troubleshoot and restore your computer. Each type of recovery media gives you access to Windows’ advanced startup options.

You can also copy your computer’s recovery partition to the drive, creating a backup copy. After you create the backup copy, Windows will allow you to remove the recovery partition from your computer, freeing up gigabytes of space.

You may try to enable the default administrator account first.

2.  Now type the following command: net user administrator /active:yes
