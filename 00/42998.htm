How to keep computer always Safe,Secure and Protected? - Microsoft Community

sexy-sassy say:

How to keep computer always Safe,Secure and Protected?

Original Title:I NEED HELP WITH COMPUTERS


I'm new to computers and I want to make sure I'm "ALWAYS" Safe, Secure and Protected. What pointers can you give someone like me? 

george222(2) say:

There is some general information starting at this page - http://www.microsoft.com/en-gb/security/default.aspx


Typically:

1 Keep your PC patched and upto date. This includes Microsoft updates via Windows update and any programs you use like your browser and items like its flash plugin etc

2 Run a supported Anti Virus real time scanner ( only run one ) and keep it up to date

3 Run occasional scans with a back-up on-demand Malware checker like free Malwarebytes ( there are others to try ). Keep it up to date and run occasional scans with it, esp before making a backup of your PC/data.

4 Make sure your firewall is running. Test it here - https://www.grc.com/x/ne.dll?bh0bkyd2 at 'shields up'.

5 Backup regularly, esp your data - documents, photos etc to a different Hard Drive, so if something does go wrong, you have a backup copy safe and sound.

6 Only install software from genuine safe sites. Resist the temptation to install 'freebies' from any old website or download source. They may have malware attached.

7 Use decent passwords not names or dictionary words. Use upper and lower case letters and numbers, and if possible punctuation marks.

8 When surfing or emailing, don't give away personal information "willy nilly". Only provide it to people you trust and who really need it. This includes things like email addresses, phone numbers, real home address, bank details etc

9 Configure your user account to be at user level only. Have an admin account and make sure you remember its password ( you will need it ), but do not use it for your day to day work.

10 Be careful with some email you may receive or websites you may visit. Don't just click a link because it provides one to click. i.e. If you receive an email from a bank with a link to its supposed website, unless you are certain its OK, check it out before clicking.



I'm sure there are others I've not listed, so carry on doing what you have started and research, or if you are not certain ask more questions.

lemp say:

There are two issues here:  physically moving the program and licensing the new computer to use the program.

Classically, the only way to move a complex program like Microsoft Word was to install it on the new computer using the original installation medium (e.g., a DVD or a downloaded installer file) and the required product key (license).

Whether you also UNinstalled Word from the original computer was part of the licensing issue.  The standard end user license agreement for Office 2003 permitted you to:

(a) install and use a copy of the Software on one personal computer or other device; and
(b) install an additional copy of the Software on a second, portable device for the exclusive use of the primary user of the first copy of the Software.

Thus, assuming that the "other" computer in your question is going to be your new primary computer, you would be obligated to uninstall Word from the old computer in order to comply with the license terms.  You should do so only after satisfying yourself that the new installation is working properly.

More recently, several commercial (i.e., not free) programs have been advertised as being able to move application programs (like Word) from one computer to another.  I have not used these programs and offer no opinion about them.   It's not at all clear that these programs will work for Microsoft Word.  Before you buy, you should ask the company specifically if Word 2003 is one of the application programs that can be moved.

PCMover by Laplink says that its "Home" and "Professional" versions can move applications.  They even suggest that the moved application will be properly licensed.  I don't know if this implies that the copy on the old computer will be uninstalled.  Typically, "move" implies a copy followed by a delete, so perhaps the original is uninstalled.  See http://web.laplink.com/pcmover_feature_overview/

EaseUS PCTrans is a relatively new program.  It too says that it can move applications and offers a free edition.  Unlike PCMover, the PCTrans website says that "Applications may require re-activating on the new PC/OS."  See https://www.easeus.com/free-pc-transfer-software/

a. user say:

Hi,

Thank you for posting your query in Microsoft Community.

I appreciate you for providing details about the issue and effort towards resolving it.

“Windows Sockets registry entries required for network connectivity are missing “error might be due to the registry entries getting corrupted or missing. The missing or corruption of Windows Socket API (winsock) inside the windows registry is the most common reason behind this error.

I would suggest you to try the Methods below and check.

Method 1: Check the following services.

1. Press Windows Key + R.
2. Type "services.msc" in the box. Check if these services are enabled:
              - TCP/IP NetBIOS Helper service
              - Wireless lan
              - DHCP
              - DNS Client
              - Function Discovery Resource Publication
              - SSDP Discovery
              - UPnP Device Host
 
To start the service and set it to automatic, follow these steps:
1. Right-click each of the services listed above and click on Properties.
2. Click the General tab, and then, next to Startup type, select Automatic.
3. Click Apply, and then click Start.

Method 2: Follow the steps below and check if it helps.

1) Start > Run > type "regedit" > hit enter.

2) Locate the following subkey:

           HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services

3) Right click and delete the entire Winsock key and the Winsock2 keys.

4) Close regedit. 

Now, re-install TCP/IP overtop itself (it's a core component and can't be uninstalled, so we improvise..)

1) Login as admin. In control panel open up, "Network and Sharing Center"

2) Right click "Local Area Connection", and then click "Properties"

3) Under the This Component Uses the Following Items dialog box, click Install

4) Click Protocol, and then click Add

5) Click Have Disk

6) In the Copy Manufacture's files from box, type C:\windows\inf and then click OK.

7) In the list of protocols, click Internet Protocol (TCP/IP) and then click OK. Restart computer.
