suzanne s. barnhill say:

I would not have thought it possible, but you actually can do this with either heading text or text with an outline level other than Body Text in the middle of a paragraph. You have to create three separate paragraphs (Body Text, Heading x/Level x, Body Text) and then insert style separators in the first two. The one with the Level 1 outline level will appear in the Navigation pane.

There is no way to change the default action of the Add Text button or the Level buttons in Outline view, but you can create a style that is formatted just like your Body Text style except for the outline level and add it to the Quick Styles gallery so that it's one click to apply. A style separator is inserted with Alt+Ctrl+Enter.
