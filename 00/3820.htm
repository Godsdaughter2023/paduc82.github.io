echo s say:

I would guess you're running into an issue with files saved using the PPT 95 format.

PowerPoint 2010 and 2007 won't open files that were saved as PPT 95 format. This wouldn't be an issue except that PowerPoint 2003 had an option to Save as "97-2003 AND 95" format. It's that "and 95" part that's the killer.

As far as I know, there's no way to force PPT 2010 or 2007 to open these files. You need to access an earlier version of PowerPoint -- 2003, 2002 (aka XP), 2000, 97 or even 95 -- to open them. If you've got access to a system with Office 2003 SP3 on it, you'll have to do some registry hacking in order for it to open the files, too:http://support.microsoft.com/kb/938810/en-us

vijay a. verma say:
1. First see whether Powerpoint is installed or not. To check this type PowerPoint in Windows search and see whether it comes up or not. If yes, it can be pinned by right clicking on it.

2. Another way to check it - type appwiz.cpl in Windows Start Search Box > Click appwiz.cpl > See whether you have your Office listed here or not.

3. Start Excel by double clicking any Excel file > CTRL+SHIFT+ESC to open Task Manager > In Processes tab, right click on Excel > Open File Location > See whether Powerpoint.exe is there or not. If it is there, right click > Create shortcut and pin the shortcut

4. Try repairing your Office installation. First try Quick Repair (takes 5 - 10 mins) , if that doesn't help then try Online Repair (Takes more than 30 mins but less than 1 hour in general)
