kenneth gar say:

Hi Aaron,

Thank you for bringing this up. If your on-screen keyboard keeps on appearing while on tablet mode, this means that your on-screen keyboard is enabled. To turn off the on-screen keyboard on your Surface, select the Start button, and then go to Settings > Ease of Access > Keyboard > Turns on the On-Screen Keyboard > Toggle right to turn off. 

Also, to make sure that your Surface is performing at its best, we recommend that you install the latest firmware and software updates available. To check for updates, select the Start button, and then go to Settings > Update & security > Windows Update, and select Check for updates.  

If you continue to have this issue even after performing the above steps, please reply to this thread. The system will notify me so I can get back to you and provide you more answers.

Regards.
