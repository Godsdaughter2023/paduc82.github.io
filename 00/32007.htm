how much amount of disk space should be reserved for system restore in all the drives, specially the drive where Program Files exist?

how much amount of disk space should be reserved for system restore in all the drives, specially the drive where Program Files exist?

waseemulla shariff say:
Hi,
 
To store restore points, you need at least 300 megabytes (MB) of free space on each hard disk that has System Protection turned on. System Restore might use up to 15 percent of the space on each disk. As the amount of space fills up with restore points, System Restore will delete older restore points to make room for new ones.
 
System Restore will not run on disks smaller than 1 gigabyte (GB).
 
You can always change the size of the space reserved for system restore on the drives. To do this, try these steps.
1. Click Start, right click Computer and click Properties.
2. Click System Protection.
3. Select the drive that you want to change the system restores reserved space and click on Configure button.
4. Move the slider to increase the space.
5. Apply the settings.
 
For more information on "How much disk space does System Restore require?” check this link:
http://windows.microsoft.com/en-US/windows-vista/How-much-disk-space-does-System-Restore-require
 
Hope this helps.
