¡firedog say:
The application was unable to initialize the store directory

This sounds like a fairly serious corruption of the store folder, which stores account details, account folders and storage folders with saved messages, calendars, proofing tools and custom dictionaries and a few other things besides. You may be able to force the program to create a new, 'empty' store folder by renaming the old one. If this is successful and the program starts, you can then import messages from the old one. You probably won't be able to rescue the default calendar, and you'll have to set up mail and news accounts from scratch. Contacts and RSS feeds should survive.

  1. In Control Panel, open Folder Options and on the View tab, select Show hidden files, folders and drives. While you're there, deselect Hide extensions for known file types.
  2. Open Windows Explorer (Windows key + E) and navigate to your store folder, by default
    C:\Users\{user}\AppData\Local\Microsoft\Windows Live Mail
    where {user} is the Windows user account name.
  3. Right-click on this folder name and Rename it, to (say) Windows Dead Mail.
  4. Launch Windows Live Mail.

Any luck?

swingroll say:

Arrow keys not working in Microsoft Photos app

I also tried to troubleshoot but had no luck. The Photos app is up-to-date. Since I cannot uninstall the app, I reset the app instead. However, still no results.

How do I fix this?

vijay a. verma say:
Try following and check after each step (Note - None of the steps below will delete any data / picture. All your data / pictures will be preserved)

1. Re-register All Store apps
Right click on Windows Start > Right click on Windows Powershell (Admin) > Copy the following from below and right click in Powershell to paste > Enter > Restart your computer

Get-AppXPackage -AllUsers | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}

2. Reset Photos App - Skip this as already done
Windows+X > Settings > Apps > Apps & Features > Locate Photos by Scrolling Down > Click Advanced Options > Click Reset
Go to Microsoft Store > Click 3 horizontal dots on top right > Downloads and Updates > Check if there is any update for this app > If yes, update it.

3. If Reset doesn't work, then you should uninstall using Powershell and reinstall the Photos App
Windows+X > Click Windows Powershell (Admin) > Copy the below command and right click in Powershell to paste it

get-appxpackage Microsoft.Windows.Photos | remove-appxpackage

Now, you need to reinstall it
Use below Powershell command to install

Get-AppxPackage -allusers Microsoft.Windows.Photos | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register “$($_.InstallLocation)\AppXManifest.xml”}

4. Reset the Database of Photos App
Copy following and paste in Windows Start Search box
%LocalAppData%\Packages\
And now locate Microsoft.Windows.Photos_........folder
Make a copy of Microsoft.Windows.Photos_........folder (for the sake of backup) and then Delete everything from inside the folder

Source - https://answers.microsoft.com/thread/e8be2a12-f...
