a. user say:

Hi,

My apologies for the delayed response.

Sometime corrupted User Profile will causes this kind of issues. In this case, you will need to create a new profile, and then copy the files from the existing profile to the new one.

Follow the below steps to copy the files from the existing profile.

1. Log on as a user other than the new user you just created or the user that you want to copy files from.

2. Right click on Start and select Control Panel > File Explorer Options.

3. In File Explorer options, click on View Tab, and then click Show hidden files, folders, and drives.

4. Clear the Hide protected operating system files check box, click Yes to confirm, and then click OK.

5. Locate the C:\Documents and Settings\Old_Username folder, where C is the drive on which Windows is

    installed, and Old_Username is the name of the profile you want to copy user data from.

6. Press and hold down the CTRL key while you click each file and subfolder in this folder, except the following files:

  • Ntuser.dat
  • Ntuser.dat.log
  • Ntuser.ini

7. On the Edit menu, click Copy. If you don't see the Edit menu, press Alt.

8. Locate the C:\Documents and Settings\New_Username folder, where C is the drive on which Windows 10

    is installed, and New_Username is the name of the user profile that you created in the "Create a New User

    Profile" section.

9. On the Edit menu, click Paste.

10. Log off the computer, and then log on as the new user.

Disclaimer: You must import your e-mail messages and addresses, any data to the new user profile before you delete the old profile.

diego rios_06 say: If the files are move to OneDrive and they didn't sync, please following procedures below:

Log into OneDrive on your computer and run the following commands to update OneDrive:

Press the Windows key + R

In the Run window, enter:
%localappdata%\Microsoft\OneDrive\onedrive.exe /reset

Wait 2 minutes and then execute the following command:

%localappdata%\Microsoft\OneDrive\update

Click OK.

Make sure the OneDrive icon in the notification area of the taskbar disappears and then re-appears after a minute or two.

If the OneDrive icon doesn’t re-appear after a few minutes, open the Run window again and enter:

%localappdata%\Microsoft\OneDrive\onedrive.exe
