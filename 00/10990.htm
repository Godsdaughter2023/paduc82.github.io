AutoFilter a table based on a cell value - Microsoft Community

full-count say:

AutoFilter a table based on a cell value

I have a table in a worksheet; C5:Q350000.  Row 5 of the table is the header row.  I have an autofilter included in the table in C5:Q5.  I would like to filter column Q of the table based on the value of a cell which is outside of the table, cell Q4.  I'm not sure how to write the code for this.  Thanks!

andreas killer say:

Add the code below to the code module of the sheet. You can write a value in any cell above the autofilter (row 4 in your case) to filter the corresponding column below.

Andreas.

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim R As Range
  'Be sure that we have an autofilter
  If Not Me.AutoFilterMode Then Exit Sub
  'Work only when a cell above the filter is changed
  Set Target = Intersect(Me.AutoFilter.Range.Rows(1).Offset(-1), Target)
  If Target Is Nothing Then Exit Sub
  'Set each
  For Each R In Target
    If IsEmpty(R) Then
      'Remove the filter
      Me.AutoFilter.Range.AutoFilter _
        Field:=R.Column - Me.AutoFilter.Range.Column + 1
    Else
      'Filter this
      Me.AutoFilter.Range.AutoFilter _
        Field:=R.Column - Me.AutoFilter.Range.Column + 1, _
        Criteria1:=R.Value
    End If
  Next
End Sub

The reason is that the Change event is not triggered when a formula is updated. In that case the Calculate event is triggered. You can play around with that event, but it is not so easy, because the Calculate event did not provide any information which cell is updated.

Andreas.

Private Sub Worksheet_Change(ByVal Target As Range)

   if Target.Address(0,0) = "K4" then Sheets("Whatever").Range("P4") = Target

End Sub

You can try the following function but it may filter out more than you require such as dates with words in them.

 

Public Function IsValidDate(ByVal vntThisDate As Variant) As Integer

    Dim vntTemp As Variant

   

    If IsDate(vntThisDate) Then

        On Error Resume Next

        vntTemp = Eval(vntThisDate)

        If Err.Number = 0 Then IsValidDate = Not 0

        Err.Clear

    End If

   

End Function
