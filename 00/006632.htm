karl timmermans (outlook mvp 2012-2018) say:
>>
how can i transfer contacts from a .pst file to an .ost file on new computer?
<<

#1 - In essence, what you DON'T want to do is store your contact/calendar data in an .ost file associated with an IMAP account. (Additional explanation for that included later on in the response)

#2 - Assuming that both your accounts were also configured as IMAP on your old computer and you only have PST files, means that you were using Outlook '2010 (or earlier). An IMAP account in pre-'OL2013 versions used 2 PST files. The first for the email folders that get sync'd with the server and the second for contact/calendar data which would have been set as the default data file for the profile. To be more specific, if you had 2 IMAP files, there would have been 3 PST files (2 for email folders and 1 for the default contact/calendar folders).

#3 - To get back to your question - strongly suggest that you copy the PST file from your old computer to your new computer (with Outlook closed when you make that copy). The way to do that would be - from Outlook menu, go to
File --> Open & Export --> Open Outlook data file

#4 - When #3 is complete, then go to
File --> Account Settings --> Data FIles tab --> select the PST file just added and set it as the default data file for the profile (this will be where the default contact/calendar folders will be located once the file is set as the default

#5 - So why don't you want to store your contacts in an OST file associated with an IMAP account?

a) Contact/calendar folders will be stored in folders with the added words "This Computer Only" in the folder description - and it means just that. That data is stored locally only and not sync'd with the server

b) If you remove the IMAP account from the profile, the associated OST file will also be summarily deleted from the system. If you haven't backed up (exported) your contact/calendar data to a PST file, then that data will be lost

c) If Outlook can't find/open the OST file for the account, it will recreate the file and the "This Computer Only" folders will be empty (as in data lost)

d) If you decide to make a copy of the OST file as a "backup" - be aware that only the original Outlook profile/email account used to create the file can open/read the contents otherwise you'll need a 3rd party OST2PST conversion utility (which are not free)
