Change Wireless Mode

Hi SG,

 

A possible cause is there is a compatibility issue between Skype for Business and your device. Please try to update Skype for Business to the latest version to check if the issue can be solved. Open Word>File>Account, choose Update options>update now. Or you can use the download link in the following article to manually download the latest version (16.0.4690.1000): https://blogs.technet.microsoft.com/uclobby/2016/07/29/skype-for-business-2016-update-list/

 

If the issue still persists in the latest version, another method you can try is Windows clean boot. It will disable all other apps running on your laptop except Skype for Business, system and other required basic services. Then you can try to reproduce the issue again, if it is gone, the issue can be narrow down to software conflicts rather than a compatibility issue. Please do follow the every detailed step in the following article to perform a clean boot: https://support.microsoft.com/en-sg/help/929135/how-to-perform-a-clean-boot-in-windows
