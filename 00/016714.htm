You might get this error message if some of the Diagnostic services are disabled or stopped. The three diagnostic services that are responsible are 
1). Diagnostic policy
2). Diagnostic Service
3). Diagnostic system host.
 
Make sure these services are started and set to automatic. If you are having any issues trying to start these services, check their dependency services and make sure they are started and set to automatic as well.
 
This issue may occur when the account Trusted installer is missing the permissions for the key below:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\DPS\Parameters
Permission needed: Full Control, Read

The biggest benefit is, you don't lose your production installation of Windows and you can reboot into it at anytime.
