In Excel 2011, is it possible to perform the function available in Word whereby the relevant date the file was last modified can be inserted within a Header or Footer?

bobgreenblatt say: Well, I'm not sure you can get this exactly. First you will have to write a VBA routine to do it. Second, all you can get from document properties is the last saved date and time, but the document may not have been changed, just saved. You will need to trap the save event and see if the document has been changed. If so, put the current data and/or time into the worksheet header. This may be further complicated by the fact that if the workbook has more than one worksheet, you may not be able to tell which sheet got changed, only if anything in the document got changed, so it may be hard to tell which worksheet header to modify. And, lastly, if the workbook has any volatile functions, like NOW() for example, or if it gets recalc'd without any changes the document will probably be marked internally as changed even though you never altered any content.

boriana say:

Hi, you can use NETWORKDAYS, but this function is in add-ins in excel 2003.

1. Tools

2. Add - ins

3. Check Analysis ToolPac (first check box)

4. Now you can find NETWORKDAYS in Data and Time category in Insert Function dialog box.

 

Hope I help you:)

 

jllatham say:

Once you have the Analysis Tool Pac active, you can use a formula like this to calculate the number of workdays left in the month with a formula like this:

=NETWORKDAYS(Now(),"31/08/12")

Eventually it will return a negative number (on 01/09/12 and after).  So you need to be prepared for that.  One way to do that would be with this formula:

=IF(NETWORKDAYS(Now(),"31/08/12") < 1, 0, NETWORKDAYS(Now(),"31/08/12"))

Also, if you set it up before the first of the month, you may want to limit it to the number of hours available in that month, and you could do that in this fashion:

=MIN(NETWORKDAYS(Now(),"31/08/12"),NETWORKDAYS("01/08/12","31/08/12"))

and finally, you can combine them both into one long formula:

=IF(NETWORKDAYS(Now(),"31/08/12") < 1, 0, MIN(NETWORKDAYS(Now(),"31/08/12"),NETWORKDAYS("01/08/12","31/08/12")))

 

Here is a page that shows one way to use NETWORKDAYS() with values in other cells to determine what dates to use.  By putting the start/end dates into other cells and referencing them, you can keep from having to actually edit the formula each month.  Also, this page shows how to use it with a list of holidays so that they are not counted as available hours - although in the example it is just a list of 1 day.

http://www.dailydoseofexcel.com/archives/2004/07/19/networkdays/
