Customize desktop icons differently on each user account with windows - Microsoft Community

pdepappe say:

Customize desktop icons differently on each user account with windows 7?

Can I customize desktop icons differently on each user account on my computer with windows 7? If I delete desktop icons from my sons desktop under his user account, they disappear from mine as well.
frederik long say: Yes, you can!

Desktop shortcuts are kept in one of two places:
a) In c:\Users\Public\Desktop. These shortcuts are shared and are visible to ever user.
b) In c:\Users\UserName\Desktop. These shortcuts are not shared and are visible to UserName only.

When you right-click a shortcut, then click General then you can see where this particular shortcut is stored.

To make a shortcut from group a) visible to yourself only, do this:
1. Drag the shortcut to the side with your right mouse button.
2. Select "Copy". You will now have two seemingly identical shortcuts.
3. Delete the original shortcut.
