oliver wu msft say:

Hi Admin-Cloudwebs,

 

I am glad to help you here. If you are an admin of office 365 accounts and want to block autoforwarding for some specific users, you can use power shell to accomplish this feature:

  1. Please use your admin account to connect to Exchange online Power Shell, you can refer to  Connect to Exchange Online PowerShell.
  2. Please run following cmdlets sequentially:

 

  1. New-ManagementRole MyBaseOptions-DisableForwarding -Parent MyBaseOptions
  2. Set-ManagementRoleEntry MyBaseOptions-DisableForwarding\Set-Mailbox -RemoveParameter -Parameters DeliverToMailboxAndForward,ForwardingAddress,ForwardingSmtpAddress
  3. New-RoleAssignmentPolicy -Name DisabledForwardingRoleAssignmentPolicy -Roles MyBaseOptions-DisableForwarding,MyContactInformation,MyRetentionPolicies,MyMailSubscriptions,MyTextMessaging,MyVoiceMail,MyDistributionGroupMembership,MyDistributionGroups, MyProfileInformation
  4. Set-Mailbox –Identity user@domain.com -RoleAssignmentPolicy DisabledForwardingRoleAssignmentPolicy         Note: (Give it some time to replicate after the change)
  5. Get-Mailbox -ResultSize Unlimited -Filter {(RecipientTypeDetails -ne "DiscoveryMailbox") -and ((ForwardingSmtpAddress -ne $null) -or (ForwardingAddress -ne $null))} | Select Identity | Export-Csv c:\ForwardingSetBefore.csv –append
  6. Get-Mailbox -filter {(RecipientTypeDetails -ne "DiscoveryMailbox") -and ((ForwardingSmtpAddress -ne $null) -or (ForwardingAddress -ne $null))} | Set-Mailbox -ForwardingSmtpAddress $null -ForwardingAddress $null

 

You can run these commands to disable the auto-forwarding feature for some specific users in your domain, and please note the sixth command, you should replace

gina whipp say:

Then this should do it (untested)...

Function streMailDistributionList(lngDistributionID) As String

    Dim rs As Recordset
    Dim strDistributionRecipients As String
    Dim strSQL As String
   
    strSQL = "SELECT tbleMailRecipients.erID, tbleMailRecipients.erReportID, ([apLastName]+', ') & [apFirstName], tblAssociateProfile.apeMailAddress " & _
                "FROM tblAssociateProfile INNER JOIN tbleMailRecipients ON tblAssociateProfile.apAssociateID = tbleMailRecipients.erAssociateID " & _
                    "WHERE tbleMailRecipients.erDistributionID= " & lngDistributionID

          
        If rs.RecordCount > 0 Then
            rs.MoveFirst
            Do
                If Not IsNull(rs!apeMailAddress) Then
                    strDistributionRecipients = strDistributionRecipients & rs!apeMailAddress & ";"
                    'Debug.Print strDistributionRecipients
                    rs.MoveNext
                Else
                    rs.MoveNext
                End If
            Loop Until rs.EOF
        End If
       
    streMailDistributionList = strDistributionRecipients
   
End Function

Remember to rename the Objects and the fields to match what is in your database.
