Hi Victor,

Thank you for the update.

Let's try to uninstall IE10 from command prompt.

a. Click "Start," and then click in the Search box.

b. Type “cmd” (without the quotations), right click Command Prompt and click Run as Administrator

c. Copy the following command exactly as shown to your clipboard by highlighting it and pressing "Ctrl-C":

FORFILES /P %WINDIR%\servicing\Packages /M Microsoft-Windows-InternetExplorer-*10.*.mum /c "cmd /c echo Uninstalling package @fname && start /w pkgmgr /up:@fname /quiet /norestart"

d. Press Enter after pasting the command in Command Prompt.

e. You will receive a couple of “Access denied” error messages; click “OK” and disregard them. When the process finishes, Windows will return to the C:\Users\%Username% prompt.

f. Type “exit” (without the quotations) and press “Enter” to close the window.

g. Reboot your computer.

h. Log back in to Windows and open Internet Explorer. If the “Set up Internet Explorer 8” page appears, you’ve successfully uninstalled Internet Explorer 10.

Hope this works.

stefan blom say:

To control the Text Wrapping options, you need to work with the WrapFormat object, specifically by setting the Type property to wdWrapSquare as follows

Sub MacroExampleShape()
Dim t As Shape

If ActiveDocument.Shapes.Count > 0 Then

Set t = ActiveDocument.Shapes(1)

t.WrapFormat.Type = wdWrapSquare

End If

End Sub

The above example manipulates the first Shape object in the document, if such an object can be found.

If you want to access the first Shape in the selection, you can use this instead:

Sub MacroExampleShape2()
Dim t As Shape

If Selection.ShapeRange.Count > 1 Then

Set t = Selection.ShapeRange(1)

t.WrapFormat.Type = wdWrapSquare

End If

End Sub
