On Sheet1 use a rank formula to get the ranks.

Assuming the percentages are in the range D2:D6 enter this formula in E2 and copy down to E6:

=RANK(D2,D$2:D$6)+COUNTIF(D$2:D2,D2)-1

Then, on Sheet2 to list the names based on the rank...

Entered on Sheet2 in B2 and copied down to B6:

=INDEX(Sheet1!A$2:A$6,MATCH(ROWS(B$2:B2),Sheet1!E$2:E$6,0))

For the percentages, entered on Sheet2 in E2 and copied down to E6:

=INDEX(Sheet1!D$2:D$6,MATCH(ROWS(E$2:E2),Sheet1!E$2:E$6,0))

azrael's kiss say:

Excuse me, but I'm not really sure what problem you are having, so I'll give ya some general information about Remote Assistance/Remote Desktop Connection.  

On the system you want to allow Remote Assistance, right click the Start button and click System.  Click Advanced system settings.  Click the Remote tab.  Check "Allow Remote Assistance connections on this computer" to activate.  Click Advanced to change options.

On the system you want to access a remote system from, click the start button.  Click All apps.  Click Windows Accessories.  Click Remote Desktop Connection.  You will need to enter the name of the system you are try to take control of and you will need to enter login credentials when you connect.  The user you select to access as will need to be a user on the remote system and that user will have to have permission to access the system remotely.
