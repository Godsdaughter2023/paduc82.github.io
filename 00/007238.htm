brian tillman [outlook mvp 2007-2019] say:

Is it a public mail provider like Yahoo or Google? Do they allow IMAP connections? Do you have Thunderbird configured to leave messages on the server?

If you leave message on the server, then it will be fairly easy to configure Outlook to access your mailbox by either a POP account or an IMAP account. A POP account will download all of the messages in the mailbox's Inbox on the server to Outlook's Inbox, but if the server has any other folders, like a Sent folder, you won't get those messages. If you connect using IMAP, then Outlook will see all the mail folders in the mailbox on the server and, depending on how the server treats outgoing mail, your sent items mail show up.

If you don't leave messages on the server but your server supports IMAP, you can configure Thunderbird to access the mailbox via IMAP which will allow you to move all the local messages back to the server. You can then configure Outlook to access the mailbox and get the messages you put back on the server.

Even if your current mail provider doesn't support IMAP, you can create an account on any free service like Yahoo, Google, or Outlook.com that does support IMAP, connect Thunderbird to that mailbox, upload your messages, then connect Outlook to that same mailbox to download the messages again into Outlook.

Another way to skin this cat would be to export the messages from Thunderbird and import into Outlook using the steps outlined here: http://smallbusiness.chron.com/export-thunderbird-email-ms-outlook-55545.html.

You can also find plenty of other tools mentioned in this thread: https://social.technet.microsoft.com/Forums/en-US/b0e2d624-b4bb-46b3-870c-553280fbef2e/how-to-convert-thunderbird-to-outlook-in-simple-method

By default, the email address or phone number that you use to sign up for your Microsoft account is considered the primary alias. Your primary alias shows up as the main account name when you view your profile information. This can be changed at any time. Any other aliases are considered secondary aliases. If you want a specific alias to show as the email sender, you can set your secondary alias as your primary email address. To do this, here's how:

  1. Sign in to the Microsoft account website.

  2. Select Your info.

  3. Select Manage how you sign in to Microsoft.

  4. Select Make primary next to the alias you want as your primary.

Populate field with current Date with Checkbox - Microsoft Community

brenda_l say:

Populate field with current Date with Checkbox

I have created a Word Form in 2007.  I would like to mention that I would like to maintain compatibility with 2003 version.

What i need is this:

My form is a checklist with checkboxes, dropdowns, and text fields.  I want the user to be able to check a box (indicating task is complete) and have a text field next to it populate with the current date.  The idea is to save time for data entry purposes.  The date must remain static so that if a user saves/closes then reopens the form the date remains the same.  If a user unchecks then rechecks the box on a later date however the date should update accordingly to reflect the date the box is being checked.  Make sense?  I know it is possible to populate fields based on drop down options, etc. but I haven't found a solution to the dilema I face anywhere online..

 

thanks!

Brenda

doug robbins - mvp office apps & services (word) say:

Assuming that your checkboxes have the default bookmark names of Check1, Check2, etc.  change the bookmark names of the formfields in which you want the date to appear to the corresponding Date1, Date2, etc (and you may also want to uncheck the box for Fill in enabled).  Then set a macro containing the following code to be run from each of the checkbox formfields:

Dim i As Long
i = Mid(Selection.FormFields(1).Name, 6)
With ActiveDocument
    If .FormFields("Check" & i).CheckBox.Value = True Then
        .FormFields("Date" & i).result = Format(Date, "dd/MM/yyyy")
    Else
        .FormFields("Date" & i).result = "     "
    End If
End With


-- Hope this helps.
