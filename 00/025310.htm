jay freedman say:

The reason for the Table Properties, and for the box in the upper left corner, is that the heading (and possibly some of the text below the heading) is in a table. You can't just remove the Table Properties, but you can remove the table while keeping the text that it contains.

When the cursor is in the heading, notice that the Table Tools | Layout ribbon appears (you can see it in the screen shot that you posted). On the right end of that ribbon, just to the left of the Formula button, is the Convert to Text button. Click that button, and click OK in the dialog box that opens (it should be set to separate the text in the table's columns with tab characters).

This step may change some of the formatting, depending on what is inside the table and how it's aligned. It would be a good idea to start with a copy of the original document, so you can compare them afterward and fix any unwanted changes.

How to create a graph which allows comparison of daily data for - Microsoft Community

gareth_ww say:

How to create a graph which allows comparison of daily data for multiple years which takes into account leap years

trying to compare daily data for a given year e.g. 2008, 2009.

 

however when a leap year comes into effect how can the graph allow the extra day to be skipped in non leap years

rich007a say:

A slightly different interpretation of your need:

Do you want to have a plot with 1-Jan to 31-Dec on the x axis, and be able to plot daily data from several years?

If so, you need to add a helper column next to your dates and calculate the day number of each day.

If your dates are in col A, and start in A1, use this in B1:

=A1-DATE(YEAR(A1),1,1)+1

and drag down.

Now plot using this column as the x axis.

Once plotted, format the x axis; numbers as custom: dd/mmm
and set min value to 1, max value to 366, major unit to 7.
Under Alignment, set text direction to Rotate all text 270deg.

Unfortunately, a week (7 days) is the only unit we can use to get sensible labels, as there is no "month" unit in Excel.

Hope that helps.

Cheers
Rich

rich007a say:

And if you want 1st March each year to be day 61 (i.e. no day 60 in non-leap years), you could use this formula in your helper column:

=IF(AND(DATE(YEAR(A1),12,31)-DATE(YEAR(A1),1,1)+1=365,A1-DATE(YEAR(A1),1,1)+1>59),A1-DATE(YEAR(A1),1,1)+2,A1-DATE(YEAR(A1),1,1)+1)

This means you can compare data from 1st March 2008 (leap year) to data from 1st March 2009 (non-leap year)...

Cheers
Rich
