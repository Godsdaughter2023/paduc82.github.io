suzanne s. barnhill say:

There are several possible factors here, as those who have already replied have discerned.

  1. If you have used one of Word's built-in Header building blocks that does not include a page number, it may be difficult to figure out how to add a page number. If you have figured out how to add any text to a pre-made header, then you can add a page number, but you can't just type the number: you have to insert a PAGE field, and the easiest way to do that, as Doug and Charles have said, is to press Alt+Shift+P.
  2. If you have enabled either Different First Page or Different Odd and Even, you may have more than one header in the document; if the document contains more than one section, and the headers in the various sections are unlinked, then you are dealing with even more. You have to make sure you have a PAGE field in each header where you want one to appear (if you've enabled Different First Page, however, it's usually for the purpose of omitting a running head on the first page of a chapter). On the Header & Footer Tools | Design tab, in the Navigation group, there are buttons to allow you to skip from one header to another.
  3. The best way to create a simple running head for a document is to avoid the fancy (or even the plain) Header building blocks and create your own from scratch. Do this by double-clicking in the header area to open the Header pane. Then type directly into the header. By default, the Header style includes a center-aligned tab stop centered between the margins and a right-aligned tab stop at the right margin. These are "alignment tabs," which means that they adjust automatically if you change the margins, and using them facilitates placing text both left- and right-aligned. If you need only centered text, it's easier to center the paragraph rather than use a tab.

For more on headers and footers, see http://wordfaqs.ssbarnhill.com/headerfooter.htm.

Saving in Multiple Locations

If the drive letter assigned to your USB drive is always the same, you can use code like the following in the ThisWorkbook module.

Private Sub Workbook_BeforeClose(Cancel As Boolean)

    On Error Resume Next

    Application.EnableEvents = False

    ThisWorkbook.Save

    ThisWorkbook.SaveCopyAs "J:\" & ThisWorkbook.Name

    Application.EnableEvents = True

End Sub

Change the "J" to the appropriate drive letter.

If the drive letter may change but the volume label is always the same, use code like the following in the ThisWorkbook code module:

Private Sub Workbook_BeforeClose(Cancel As Boolean)

    Dim DriveLetter As String

    Dim VolumeName As String

    On Error Resume Next

    Application.EnableEvents = False

    ThisWorkbook.Save

    VolumeName = "YourVolumeName"

    DriveLetter = GetDriveLetter(VolumeName)

    If Len(DriveLetter) > 0 Then

        ThisWorkbook.SaveCopyAs DriveLetter & ":\" & ThisWorkbook.Name

    End If

    Application.EnableEvents = True

End Sub

WordTips are not designed to promote one method of doing something over another, but to provide alternative ideas that people can incorporate into their Word use, if they so desire.

With that being said, let me point out specifically why you should not use the FastSave feature.

  • Other word processors may not be able to open Word documents saved with the FastSave feature enabled. For instance, WordPerfect says it can open Word documents, yet some versions of WordPerfect cannot open FastSaved Word documents.
  • Third-party software designed to work with Word documents (such as some grammar checkers, document management programs, and desktop publishing software) will not work correctly with FastSaved Word documents.
  • Find File (either in Word or in Windows) may not work correctly with FastSaved Word documents.
  • FastSave results in more complex files being saved on disk. Any time you increase complexity, you run the risk of corrupting the files easier.
  • FastSaved files take up more disk space.
  • There have been reports of macro viruses ‚Äúliving on‚Äù in FastSaved documents, even though you thought you had them removed.

¬∑         The list could go on and on, but you get the idea. In short, unless you have a specific need to use FastSave (such as if you are only creating simple documents using Word 6 on a slow 286 machine), it is probably a good idea to turn it off. The FastSave feature is turned on by default in all versions of Word, even in the original Word 97. Only if you are using Word 97 SR-1 is it turned off by default.

As a side note, according to the Microsoft Knowledge Base there are several circumstances under which a FastSave is not done by Word, even if you have enabled the feature. These circumstances include the following:

  • When you save a document for the first time. (This makes sense, right?) This includes when you choose Save As to save under a new name or in a new location.
  • When your document is saved on a network server or remote volume. (Word only performs a FastSave to a local volume.)
  • When Word reaches the limit of how much information it can save using the FastSave option. In this case Word does a normal save, and then again starts using FastSave for future saves.

I went through the process of setting up a send/receive account for another address in Outlook.com.  Unfortunately, while it looks like you have control over the server name, port, and encryption for the incoming server, Outlook.com provides no ability to supply values for the outgoing server, port, encryption, or, most importantly, authentication.  Consequently, I believe that, just as would happen with the account if you were to try to use a PC-based mail client and did not specify "My outgoing server requires authentication" in order to connect to the SMTP server, Outlook.com is not authenticating to the server and, thus, is getting rejected, as is proper for soomeone trying to relay through the server.  This appears to be a design flaw in the Outlook.com handling of outgoing mail to other SMTP servers.

Here is one way to modify the Normal style: Create a new, blank document, for example by pressing Ctrl+N. In the Apply Styles pane (Ctrl+Shift+S), type in Normal and then click the Modify button. In the Modify Style dialog box, click the Format drop down, and click Paragraph. "Don't add space..." is on the Indents and Spacing tab. When you are done, click OK. In the Modify Style dialog, select "New documents based on this template" and then click OK. If/when you are prompted to save the Normal style, choose to do so.
