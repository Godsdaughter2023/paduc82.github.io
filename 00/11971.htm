multiplication in access - Microsoft Community

ali ajami 61 say:

multiplication in access

How can define one field of database by multiplication of two other fields?

Thanks

djsteele, mvp say:

While I believe it is possible in Access 2007, there's no logical reason why you'd need to do this.

As fellow Access MVP John Vinson likes to say "Storing calculated data generally accomplishes only three things: it wastes disk space, it wastes time (a disk fetch is much slower than almost any reasonable calculation), and it risks data validity, since once it's stored in a table either the Total or one of the fields that goes into the total may be changed, making the value WRONG."

Instead, create a query with a calculated field that multiplies the two values together, and use the query wherever you would have used the table.

djsteele, mvp say:

Create a query like normal. In an empty column in the grid, put something like [Field1] * [Field2] (replace Field1 and Field2 with the actual field names).

Assuming you want to give the calculated field a meaningful name, put that name in front, followed by a colon, like Product: [Field1] * [Field2]

Display pdf file names in my List box in a Form - Microsoft Community

rev12 say:

Display pdf file names in my List box in a Form

Hi experts

is there a way I can view my PDF filenames in my listbox in my form?

please anyone can help I try to search from the threads here but no luck.

Thanks in advance,

Rev12

daniel pineault say:

You can use the Dir Function to loop through the files and add them to your listbox.  Do not forget to switch your listbox to Value List Row Source Type and use the form's load event to populate everything.

Private Sub Form_Load()
    On Error GoTo Error_Handler
    Dim sPath                 As String
    Dim sFile                 As String

    sPath = "C:\Users\Daniel\Documents\"
    sFile = Dir(sPath & "*.pdf")
    Do While sFile <> vbNullString
        If sFile <> "." And sFile <> ".." Then
            Me.YourListBoxName.AddItem sFile
        End If
        sFile = Dir
    Loop

Error_Handler_Exit:
    On Error Resume Next
    Exit Sub

Error_Handler:
    MsgBox "The following error has occured" & vbCrLf & vbCrLf & _
           "Error Number: " & Err.Number & vbCrLf & _
           "Error Source: Form_Load" & vbCrLf & _
           "Error Description: " & Err.Description & _
           Switch(Erl = 0, "", Erl <> 0, vbCrLf & "Line No: " & Erl) _
           , vbOKOnly + vbCritical, "An Error has Occured!"
    Resume Error_Handler_Exit
End Sub

Also, this works for simple lists, but if there are to many files, you may first need to push the files name to a table and still use a table to populate the listbox.

shrink to fit for printing - Microsoft Community

kate jensen say:

shrink to fit for printing

In 'print' office 2007 had a button to shrink the doc to fit the page for printing.  this allowed the entire doc to printed on a page without losing any of the materialto margin.  where do I find this in  office 2010?  thank you!
jay freedman say:
There are two ways to get access to the Shrink button, both of them carefully hidden. :-/

For both ways, first do this:
  • Click File > Options and then either Quick Access Toolbar or Customize Ribbon (your choice of where to put the button).
  • If you chose to customize the ribbon, click Home in the right-hand list and then click the New Group button at the bottom. (You can't add anything to the built-in groups.) If you want, click the Rename button and change the name of the new group. Leave the new group selected.
  • At the top of the left-hand list, click the dropdown and choose All Commands.


Now, one way is to get back the old Print Preview that isn't the new File > Print page. The Shrink button will be on the ribbon in that view. To set that up:

  • In the command list, scroll down to Print Preview Edit Mode, click on it, and click the Add button between the lists. The new button should show up in the right-hand list, either at the end of the items on the Quick Access Toolbar or in the new group of the Home ribbon. (You can use the arrow buttons on the right to move the item up or down in the list.)


The other way is to put just the Shrink button on the ribbon or the Quick Access Toolbar:

  • In the command list, scroll down to Shrink One Page, click on it, and click the Add button.


When you're done, click OK. Try out your new button...

stefan blom say:
