Excel - Microsoft Community

amycrawford1 say:

Excel

when a cell containing a formula is copied and then pasted into a new cell what happens 

hansv mvp say:

The formula is copied to the new cell. How that works out depends on the cell references in the formula.

Here are some examples.

1) Let's say B1 contains the formula =A1

If you copy B1 and paste into B2, then B2 will contain the formula =A2

If you copy B1 and paste into C1, then C1 will contain the formula =B1

2) Let's say B1 contains the formula =A$1. The $ before the row number 1 "fixates" that number.

If you copy B1 and paste into B2, then B2 will contain the formula =A$1

If you copy B1 and paste into C1, then C1 will contain the formula =B$1

3) Let's say B1 contains the formula =$A1. The $ before the column letter A "fixates" that letter.

If you copy B1 and paste into B2, then B2 will contain the formula =$A2

If you copy B1 and paste into C1, then C1 will contain the formula =$A1

4) Let's say B1 contains the formula =$A$1. Both the column letter and the row number are fixated..

If you copy B1 and paste into B2, then B2 will contain the formula =$A$1

If you copy B1 and paste into C1, then C1 will contain the formula =$A$1
