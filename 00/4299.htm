1.) Scan your Document into Excel, then "fix" the field names (Column headers) to eliminate the spaces. It's Access that doesn't like the spaces.

2,3,4) First of all, I have doubled my typing speed by using one finger from BOTH hands ;) ... so yes, I don't mind "touching up" the data that this program sends to Excel. How much "touching up" generally depends on the layout of the original document. You may have to change the data type on some fields.

Secondly, the program has an "express" mode for conversion, where it makes decisions and does its best at preserving the layout. Again, if it's tabular, it usually works.
