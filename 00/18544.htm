Hi,

No there isn't, changing a cell colour doesn't make excel calculate. The best you can do is add the line in bold below and now if you change a cell colour and tap F9 then the function will calculate.

Function CountCcolor(range_data As Range, criteria As Range) As Long
Application.Volatile
Dim datax As Range
Dim xcolor As Long
xcolor = criteria.Interior.ColorIndex
For Each datax In range_data
    If datax.Interior.ColorIndex = xcolor And datax.RowHeight > 0 Then
        CountCcolor = CountCcolor + 1
    End If
Next datax
End Function
