daniel orman say:

I've found a work around:

Private Sub UserForm_Initialize()
    Application.ScreenUpdating = False
   
    WS_StoredData.Activate
End Sub

Private Sub UserForm_QueryClose(Cancel As Integer, CloseMode As Integer)
    WS_Operations.Activate

    Application.ScreenUpdating = True
End Sub

This makes the required sheet active and then hidden when it's closed. Does exactly what I need it to do.

don cuthbert say:

I have seen this a few times on my installs.  Going through the motions a few times eventually works.  Switch back to local account, then switch to MSA.  Start Insider Builds - Restart the computer as soon as it requests you to.  If you hit Fix and it instead Stops Insider Builds, go ahead and follow through to Stop them.  Again Restart as soon as asked without delaying the restart.  Then try Restarting insider Builds again, Restart as soon as asked, and you will probably be back in where you want to be.

I found that it is possible to upgrade to an activated Windows 10 Pro on a PC licensed for Windows 10 Home, but that only the Windows 10 Home will clean install (which is appropriate, based on the actual prior OS license).

So with a dual-boot setup, it is possible to remain on the Insider Program on one partition while having upgraded your actual OS license to the appropriate Windows 10 version on your main partition.  You can opt in or out on either one, but only the actual licensed OS will clean install (once your hardware has been registered as a Windows 10 device, with the version based on your prior OS).

This is really the same as those who were dual-booting Windows 7 or 8 with Insider Preview Builds.  Now you can dual-boot between your licensed Windows 10 and your Insider Preview Builds.  The activation servers seem to be on top of which one is supposed to be able to clean install.

EDIT: On my licensed Windows 10 Home devices that also have Windows 10 Pro activated on another partition, the Windows 10 Pro shows "permanently activated", even though I cannot clean install it.  However, I can clean install Windows 10 Home.
