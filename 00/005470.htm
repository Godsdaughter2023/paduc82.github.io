NOS Logs
The logging needs to be enabled first by editing the file debug_conf.txt with a text editor. The line FILELOG=FALSE must to be changed to FILELOG=TRUE. The file is located where Nokia Ovi Suite was installed (C:\Program Files\Nokia\Nokia Ovi Suite)

 

The logs (debug_log.txt and DalMeasurementFile2.log) will be written to the user's %TEMP% folder

In WinXP the folder is C:\Documents and Settings\[username]\Local Settings\Temp
In Vista/Windows 7 the folder is C:\Users\[username]\AppData\Local\Temp

 

Crash Dump files
In XP:
On application crash, you'll see the dialog asking whether you want to Debug or Close. Don't do either. Closing the dialog destroys the dump file before you have a chance to copy it.
- Click the "For more information about this error, click here" link in the crash dialog, and copy the "Error signature" contents. The relevant content can be selected with mouse, and copied. Paste the text to the report.
- Find the minidump file (usually C:\Documents and Settings\your_username\Local Settings\Temp\???.dmp). If there are multiple .dmp files, check the modification time the find the one that was created at the same time the application crashed.
- Start Task Manager and end process "dwwin.exe". The dump file is locked by this process.
- The dump file can now be copied and attached.

 

In Vista and Windows 7:
- Expand the "View problem details" in the crash dialog and copy the contents
- Open Task manager and find NokiaOviSuite.exe from the Processes list
- Right click over NokiaOviSuite.exe and select Create Dump File.
- The dump file can now be copied and attached.
