jeeped say:
BTW, if that data was copied from a web site, there is a good chance that the leading/trailing spaces are actually non-breaking spoces (i.e. ASCII 0×160) and not regular spaces (i.e. 0×032). Trim and Clean functions have no effect on non-breaking spaces. Likewise for Text to Columns. You will need to firs Find & Replace all non-breaking spaces with regular spaces. A non-breaking spaces can be generated by holding down one of the Alt keys and typing 0·1·6·0 on the numpad with Num Lock on then releasing the Alt key.

ashish mathur say:

Hi,

 

Try this

 

1. While on any one cell, select one space looking character (which is either before or after the number) and press Ctrl+C

2. Select the range of numbers and press Ctrl+H

3. Ensure that the Find What box is empty and press Ctrl+V there

4. Ensure that the Replace With box is empty and press Replace All

 

Does this work?

Since no one else has commented on your section question, here we go:

Word "hides" formatting information in various places, depending on the scope of that formatting.

So you can potentially find formatting information in :

  • each character (ie manually applied bold or special font)
  • each paragraph (ie style applied to the paragraph)
  • each section (ie margins, headings, page numbers)
  • document (ie styles associated with the document)
  • template (source of styles associated with document, and base formatting)
  • global templates (ie "global/system shortcut keys)

In a brand new blank document there is an implied (hidden) section break at the end of the document.

So when you want to change a section attribute you insert a section break. For example:

If you want to add a short section of "newspaper columns" on a page you insert a continuous section break before and after that section then turn on the "columns" feature

If you want to have different headings page numbers on the front pages of a book, ie roman number pages vs arabic numbers in the body, you insert a section break.

If your book style requires chapters start on an odd number page, you insert an "odd" section break instead of using a manual page break or a bunch of blank paragraph marks.  This has the advantage that you don't have to worry about the manual formatting as you make changes to the body text before.  If the new chapter naturally starts on an odd page there is no "extra" blank pages messing things up.

Is this happening in one particular file or all Excel files?

Try creating a new Excel file and check if the same issue occurs in the new file.

 

Open Excel in safe mode and check if it makes any difference. Click on Start > All Programs > Accessories > Run > type excel /safe > hit OK.

If it helps, an add-in might be causing problems. See the section about add-ins in this article:

http://support.microsoft.com/kb/919196

 

If it’s an issue with one file, you may try repairing the file using the Open & Repair option and see if it helps. Open Excel > click the Office button > click Open > browse & select the concerned Excel file >click the drop down menu and select Open & Repair at the bottom right corner.
