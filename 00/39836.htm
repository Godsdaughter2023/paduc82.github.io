zigzag3143 (ms -mvp) say:

CY

An FTW file is a Family Tree Maker File, developed by Ancestry.com.

charlieyoung say: Can anyone answer the two (2) questions I asked?

lemp say:

In order to answer your question properly, you first need to explain where you obtained the CD and what you think the file is.

You didn't mention what "Microsoft site" you went to that indicated that you should use "RegSERVO."  As near as I can determine, "RegSERVO" is a registry cleaner and should be avoided because it is likely to damage your system.  See Microsoft support policy for the use of registry cleaning utilities

There are reputable websites that will show you programs most commonly associated with various file name extensions, for example, http://filext.com/ and http://fileinfo.com/  Both of these sites indicate -- as ZigZag told you -- that FTW files are associated with Family Tree Maker.

Furthermore, the following site indicates that FTW files that were created by Family Tree Maker versions 1 - 4 can't be opened directly in newer versions of Family Tree Maker --> http://ftm.custhelp.com/app/answers/detail/a_id/3673/~/opening-old-and-unsupported-files-in-family-tree-maker

To specifically answer your two questions:

1.  No, it is not safe to download and run RegSERVO

 2.  The message "Windows cannot open this file" does NOT mean that "your system files are misconfigured or not properly defined."  It means that you haven't installed a program that recognizes a file with that extension.  You may -- but not likely -- have a program that will be able to open the file if you rename it to have a different extension, but the only way you could determine that would be to know what sort of file it is in the first place.

Windows Search cannot index the A: drive - Microsoft Community

itsacarol say:

Windows Search cannot index the A: drive

Do not assign the letter "A" to a drive partition, because Windows Search cannot index the A: drive. I did a Win7 install on a system with an SSD boot drive and a secondary conventional HD.

I made my data partition the A: drive, and moved my user folders to that partition. Windows Search couldn't find anything on that drive until I assigned a different drive letter to the partition.

a. user say:

The "A" drive is designated as the Floppy drive in all machines  - as is B. That's why your primary partition is the C drive.

Therefore of course it won't be indexed - particularly if your machine doesn't HAVE a floppy drive..

a. user say:

I know there are very few machines with floppy drives.

It's still the convention however that drives A and B are reserved for floppy drives and will be until there are no more floppy drives left.

If you connect a USB external floppy drive to your machine it becomes the A drive, NOT the D or E or F or any other letter.

Therefore it's not a good plan to designate any partition on an HDD as the A drive...
