deepa raj say:

Hi Ronald,

I will assist you with the query you have with Outlook.

 

Outlook 2010 uses Outlook connector in order to configure the Hotmail account as MAPI type, hence the settings cannot be carried over from the previous version of Outlook as Outlook 2013 uses Exchange Active Sync method to sync the information with the server.

 

Note: The information on the server and on Outlook remains and will synchronize the changes. No information is deleted automatically.

 

Remove the Hotmail account from Outlook 2013 and re-configure the Outlook with this account using the Exchange Active Sync to get all the data into Outlook 2013.
