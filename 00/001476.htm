Only the 'web safe fonts' are being shown for your web publication. Bimini is not a web safe font. If you use Bimini and the viewer of your site does not have Bimini installed on their computer, the browser will either subsitute in another font which will change your layout or will convert the text to an image which you don't want either.

bill mosca say:

John

Here is how an error trap usually looks. If an error occurs the user gets a message and the procedure exits. There is no debug or help choice.


Private Sub Command1_Click()

    On Error GoTo err_PROC
    
' check if the spreadsheet is named correctly and is where it should be
    If Dir("C:\Students.xls") = "" Then
        MsgBox ("The database can't find the Spreadsheet. Check that it is named correctly " _
                & "as Students.xls and is placed in folder C:\")
    Else
        'transfer the data in
        DoCmd.TransferSpreadsheet acImport, acSpreadsheetTypeExcel12, _
                                  "tblImport", "C:\Students.xls", True
        MsgBox ("Import Completed")
    End If
   

exit_PROC:
    On Error Resume Next
    Exit Sub

err_PROC:
    Dim strErrMsg As String
    Dim lngIcon As Long
    
        strErrMsg = "Error " & Err.Number & " (" & Err.Description & ") " _
                & "occurred in procedure Command1_Click of " _
                    & "VBA Document Module1"
        lngIcon = vbOKOnly + vbInformation

        MsgBox strErrMsg, lngIcon, "Error"

        Resume exit_PROC


End Sub
