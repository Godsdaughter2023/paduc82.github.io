karl timmermans (outlook mvp 2012-2018) say:
>>
I use a Gmail account for email (no Exchange server).
<<

Hello Barbara

If your GMail account was configured as an IMAP account, the underlying data file associated with the account would have been an .ost file which is where the Notes folder would have been stored. Unfortunately, an .ost file can only be opened/accessed by the original Outlook profile/email account used to create it. If you have the .ost file from the original install, then your only option is to use (purchase) a 3rd party OST2PST conversion utility. Just do a search for 'OST 2 PST" and you will have no shortage of results

If the account was configured as a POP3 account, it would have used a .PST file which is where the Notes folder would be located. If this is the case, the original PST file should also have been used when the account got configured again.
