Conditional Formatting based on performance in a rank spread sheet - Microsoft Community

cassandrawilliams17 say:

Conditional Formatting based on performance in a rank spread sheet

Hello! So I have an excel spreadsheet that shows top 50 performing items this week with another tab showing top 50 performers from last week. I want to conditional format the current week sheet to highlight a cell when one of the top performers is higher ranked than the week before. In the spreadsheet I have the item and the rank, I have tried doing a vlookup but haven't had any luck. Does anyone have any thoughts? Thanks

jeeped say:
Let's ay that the items for this week are listed in column A of Sheet1 and the ranking is in column B. The data layout is the same for last week on Sheet2. Both sheets have a column header row in row 1.

Select A2:B999 on Sheet1 and base your conditional formatting on the following formula,

=$B2<VLOOKUP($A2, Sheet2!$A$2:$B$999, 2, FALSE)

You can test that formula in an unused column to the right of your data in Sheet1. It should show TRUE for any rows where the ranking is higher (closer to 1) than last week.
