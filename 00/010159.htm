Format Painter in Filtered List - Microsoft Community

markseward617 say:

Format Painter in Filtered List

I'm having trouble copying and pasting formatting in a filtered list in Excel 2016. 

I set a conditional formatting in a cell in a filtered list and used Format Painter to paste the conditional formatting to visible cells in the same column. The only problem is the formatting is also being applied to other cells in the same column. 

Is there a way to only paste the conditional formatting to visible cells (without being applied to cells that are not visible)?

ossiemac say:

Try applying the conditional format a different way.

  1. Clear any existing conditional format from the range. (Select the range -> Select Conditional formatting -> Clear Rules -> Clear rules from selected cells) 
  2. Apply the AutoFilter as required.
  3. Select the entire visible range required for the Conditional format
  4. Select "Find & Select" icon (Far right of Home ribbon)
  5. Select "Go to Special"
  6. In the dialog box select "Visible cells only" and OK
  7. Now select Conditional formatting -> New rule.
  8. Apply the required Conditional formatting rule and the required format as if you are applying it for the first cell of the selected range only. (Excel will look after applying the conditional format to the remaining selected ranges)
  9. When finished if you select all on the filter you will see that the conditional format will only be applied to the cells that were visible while filtered. (You can test by editing one of the other cells in the column to match the filter condition and the conditional formatting should not be applied to it.
  10. If you select one of the cells with the conditional format applied and then select Conditional formatting -> Manage rules and you will see in the "Applies to" field that the non contiguous  ranges are separated by commas.

Having provided the above guidelines, I wonder why you need the conditional formatting to be only applied to the cells that were visible because if the conditional formatting was applied as per the autofilter condition then only the visible cells should contain the formatting anyway.

markseward617 say:

Ossie, many thanks, but it didn't quite work because the formatting for each cell in the filtered list was based on how the value of the first cell fit the rule for the conditional format instead of how the value of each cell in the list fit the rule. 

I ended up fixing the problem by showing all values (no filters applied) and setting multiple rules in the first cell and then using format painter to apply these rules to all the cells in the list. 

I'm not sure if this explanation adequately explains the fix, but I got it working and I thank you for your reply because it challenged me to think of a different solution.

It is difficult to provide an answer with conditional formatting without knowing the rules for the conditional format. I thought you had that sorted out when you said that the conditional formatting was simply applying to all cells instead of just the visible cells.  However, generally when applying conditional format, if all cells need to reference a particular cell for the rule then the cell is placed in a formula as absolute.

Guidelines to upload a workbook on OneDrive. (If you already use OneDrive and your process for saving to it is different then you can probably start at step 8 to get the link but please zip the file before uploading.)

  1. Zip your workbooks. Do not just save an unzipped workbook to OneDrive because the workbooks open with On-Line Excel and the limited functionality with the On-Line version causes problems.
  2. To Zip a file: In Windows Explorer Right click on the selected file and select Send to -> Compressed (zipped) folder).   6. Select Upload in the blue bar across the top and browse to the zipped folder to be uploaded.
  7. Select Open (or just double click). (Be patient and give it time to display the file after initially seeing the popup indicating it is done.)
  8. Right click the file on OneDrive and select Share.
  9. Select "Get a Link" from the popup menu.
  10. Click in the field displaying the link and it should highlight and then Copy and Paste the link into your reply on this forum. (I suggest that you avoid the "Copy" button on the "Get a link" screen because it introduces additional steps that are not required.)
