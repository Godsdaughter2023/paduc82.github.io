frederik long say:

Can I check the version of windows 7 (home or pro?) via BIOS?

You shouldn't ask yes/no questions unless you expect the answer to be either "yes" or "no" . . .

Anyway, the answer is No. The BIOS knows nothing about the installed operating system.

To find out the installed operating system you can boot the machine into Windows Repair Mode, then load the correct System registry hive with regedit.exe and examine this key:

HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProductName

Make sure not to load the hive on drive X:. It is not the real hive.

gts-nj say: There is a short window of opportunity to undo a file action (move, rename, copy type) in Windows Explorer or folder windows using the Edit Menu (or pressing Ctrl Z).  You would see undo rename as the option for example.  Only the most recent file action will be un-doable, so it may be too late.

It appears that you had multiple files selected when you did the rename. When you have multiple selections Windows will rename them using the name you enter and the bracketed sequential numbers.
