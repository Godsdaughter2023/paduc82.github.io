delixis wrote:
It is file format dependent.

The FAT size has nothing to do with it.

Cards of 4GB and higher are SDHC cards and SDHC is a technology sufficiently different from regular SD cards for them not to work in devices designed only for regular SD. SDHC cards only happen to be formatted using the FAT32 filesystem because straightforward FAT only allows volume sizes of up to 2GB. FAT32 over FAT is an effect, not the cause.

It is a generic message, disk space is only one of the possible causes. You have confirmed that general free disk space is not an issue.  So that means some other form of memory is running short. Most likely a Windows reserved block of memory that you have no way of even identifying.

Sometimes poorly written applications or addons (mini-apps) have "memory leaks". They use some memory but don't release it when they are done with it. So they keep using more and more memory until something "breaks", with the error message you got or something similar.

In this case I believe it is your Internet Explorer Cache. IE has a fixed size assigned to the cache. It appears you have reached that defined disk space limit and IE has not cleaned anything out. Thus you "ran out of disk space".

pranayan say:

NOKIA 6080

Anyone knows whether 6080 can be used as a modem?

shipnil---01 say: Hi,
To all those who are struggling to install Nokia 6080 as modem. Nokia has done a really poor job of instructing users about how to install it. After much fiddling, I was able to figure out.

I am assuming that you have installed the Nokia PC Suite.

Follow the process as described below.
1. Open the Nokia PC Suite from taskbar/ programs menu.
2. Open manage connections window from the PC Suite
3. Deselect the connections (DKU5, CA 42 etc). Remember there should be no option selected among all the connections. Now close the window.
4. Verify that you see a white phone like icon having a red dot. This means your phone is connected to pc through the interface cable but Nokia PC suite does not recognize it. This is must to install the phone as modem.
5. Now go to Control Panel>Phone and Modem Options. Go to Modem tabs in the window that comes up. Click on Add. A window will appear enquiring whether the windows should search for it. There will be one option at the bottom saying don't detect my modem, I'll select from the list. Select it. Click Next. You'll get a list of standard modems and another of Nokia modems. The Nokia Modems list does not have 6080 cable. But don't worry, select any phone model which has cable written next to it. Click on next. It will give a list of COM ports where you can install it. If you don't have any other device using COM ports, then only the port on which the CA42 or DKU 5 cable was installed will be listed there. Click on it and Click on Next/Finish. A window saying uncertified software will pop up. Click on continue anyway. Voila! your modem is installed.
6. Now you have to create a dial up connection for this modem
7. Go to Programs>Accessories>Connections>Network Connections.
8. On the left side you'll have Create a new connection. Click on it. Rest of things will be similar to creating a new dial up connection.

Hope this will help. I might answer any queries but only related to installation of modem and not about creating dial up connection etc.

Shipnil

mmmmartin---01 say: I SUCCEEDED!!!! I DON'T BELIEVE MY EYES!

I didn't use any NOKIA modem, but normal Standard windows modem 33600 bps, which is there by default. Clicked "Don't detect modem, I will choose from the list" and chose this standard one. Installed on the virtual COM port for the data cable. Then in control panels "Phone and modem options" (sorry for exact names, I am using localized Windows) I displayed properties of that modem. In the new window I chose card nr. 4 (something like "specify precisely" ) and into box "Special initialization commands" (sorry again) typed string ***&D2 +CGDCONT=1,"IP","internet"*** without stars, I am using *** instead of " symbol, which is part of the string. Between 2 and + is space. Then created normal dial-up network connection with dialing number *99# (* is *, not " ) with no login, no password, dialed and I WAS THERE!!! I suspected Nokia, that they lie about possibility to run internet.

How I found it: I used different nokia phone with Bluetooth and notebook. Some Bluetooth software dected the modem in the mobile and installed the modem and the dial up connection. I was shocked that it didn't use any Nokia modem from PC suite, but standard windows modem. I looked at the special initialization command and there was that previous one. Instead of ***AT+CGDCONT=1,"IP","internet"*** which is everywhere on the internet and in many forums. I tried it with cable and 6080 too, and succeeded. Strange, that no one can help, nor Nokia support.
