j.e. mcgimpsey say: First -  "VBAProject" is the default name for ALL workbooks. One should never use that name for a workbook project that you're referencing. 

Instead, rename the project in your macro workbook (select the workbook with the macro in the Projects window and change the (Name) property in the Properties window.

Select your non-macro workbook and choose Tools/References. Your project will then show up in the References dialog under the new name. Check the appropriate box, and save your non-macro workbook. Now when you open the non-macro workbook the macro workbook will also open (if it's not already).

IF this doesn't work, my best guess is that your non-macro workbook is corrupted. Migrate the info/project to a new workbook (or try round-tripping it to SkyDrive or WinXL)

devide a whole number by multiple numbers without leaving a remainder

I have 4600' and must divide it by multiples of 350' and 100'.  I cannot leave any "leftover" footage.  Is there a formula I can use? 
tasos_k say:

Hi,

try this approach, using vba code

 

open a new workbook and..

 

step1

Save As, your Workbook with extension .xlsm (macros enabled)

 

step2

1) press ALT+F11 to open Visual Basic

2) Insert > module and paste the code below on the right

 

Sub Macro_01()
Const num1 As Long = 100  '<<< number1, change
Const num2 As Long = 350  '<<< number2, change
Const myNum As Long = 4600  '<<< expected result, change
Dim N As Long, t As Long, i As Long
N = Int(myNum / num1)
Dim X As Double
Range("A:B").ClearContents
Cells(1, "A") = num1
Cells(1, "B") = num2
t = 1
For i = 0 To N
X = (myNum - i * num1) / num2
If X = Int(X) Or X = 0 Then
t = t + 1
Cells(t, "A") = i
Cells(t, "B") = X
End If
Next
If Cells(2, "A") = "" Then
MsgBox "no combination"
End If
End Sub

 

3) Press ALT+Q to Close Visual Basic
