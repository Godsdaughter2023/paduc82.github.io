rob koch say:

Defender was never the problem, it simply turns on if MSE doesn't, since it's the fact that MSE is operating that supresses Defender.  Since MSE wasn't operating properly, Defender took over just as it was designed to do if no other anti-malware application was operating on the system.

THough you might have eventually found the specific issue, performing an uninstall and then a reinstall is often a quicker and simpler way to accomplish the same goal.  Once MSE was operating properly, Defender was automaticlly disabled by NSE at startup as it is supposed to be.

So Defender running isn't blocking MSE from running, it simply indicates a problem of some sort with the installation and/or operation of MSE.

Hi,

Let me help you with the issue.

This behavior will occur if the following conditions are true:

â€¢Microsoft Excel is set to use the 1900 date system

 -and-

â€¢The value in the cell is a negative number (less than zero).

 -and-

â€¢The cell is formatted as a date or a time.

To display the number correctly, do either of the following:

â€¢Change the date system to the 1904 date system.

 -or-

â€¢Use the TEXT worksheet function to change the displayed result.

Steps to Change to the 1904 Date System

  1. Click File.
  2. Go to Options.
  3. Go to Advanced.
  4. In When calculating this workbook section, check Use 1904 date system.

Using the TEXT Function

You can use the TEXT function to change the display of the negative number. For example, if you are subtracting A2 from A1 and you want the results displayed as a time format, you can use a formula similar to the following:

=TEXT(ABS(A1-A2),"-h:mm:ss")

where A1 has a value smaller than A2.

The above formula will display the value as text. You may need to format the cell alignment to right justified.

WARNING: This is not an acceptable workaround if you need to use the result in another calculation.
