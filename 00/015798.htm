rajesh c. say:

As rightly suggested by Vijay, you can use VLOOKUP function as

=VLOOKUP(VALUE("TEXT"),TABLE ARRAY,COLUMN,FALSE)  by giving the text to search in double-quotes.

For example -

If you wish to search  the text     ABC1234-AEREOPLANE     in the table

then you have to type the formula as

=VLOOKUP("ABC1234-AEREOPLANE",TABLE-ARRAY-RANGE,COLUMN-TO-RETURN,FALSE-OR-ZERO)

Here, since the result ABC is in second column of the table range, the COLUMN-TO-RETURN will be 2.

And if the table array range is , say, B1:C20 then, your formula will become

=VLOOKUP("ABC1234-AEREOPLANE",$B$1:$C$20,2,FALSE)

Now, if , say for example, the text    ABC1234-AEREOPLANE   is in cell A2 and

instead of putting the   text in the formula (hardcoding), you wish to refer to the cell A2 then

the formula will become

=VLOOKUP(A2,TABLE-ARRAY-RANGE,COLUMN-TO-RETURN,FALSE-OR-ZERO)

OR

=VLOOKUP(A2,$B$1:$C$20,2,FALSE)

OR

=VLOOKUP(A2,$B$1:$C$20,2,0)

Hope this explains.
