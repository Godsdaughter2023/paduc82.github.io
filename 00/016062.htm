hansv mvp say:

You'll have to set it up for each worksheet individually:

- Activate a sheet.

- Select the appropriate range, or the entire sheet.

- On the Home tab of the ribbon, click Conditional Formatting > Highlight Cells Rules > Equal To...

- Enter the word Newton in the box.

- Select a format from the dropdown.

- Click OK.

- Repeat for the next sheet, etc.

If the number of sheets is very large, you could run a macro to set the conditional formatting:

Sub FormatAll()
    Dim wsh As Worksheet
    For Each wsh In Worksheets
        ' Change range as needed
        With wsh.Range("A1:K50").FormatConditions
            ' Optionally delete existing rules
            .Delete
            ' Create new rule
            With .Add(Type:=xlCellValue, Operator:=xlEqual, _
                    Formula1:="Newton")
                .Interior.Color = vbYellow
            End With
        End With
    Next wsh
End Sub
