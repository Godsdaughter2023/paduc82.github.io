_null____ say:

Mukul Modani,
It sounds like there may be an issue with the Registry entry for the "Send To" menu.

Important: This document involves modifying the Windows registry. Serious problems might occur if you modify the registry incorrectly. Therefore, make sure that you follow these steps carefully. For added protection, back up the registry before you modify it. Then, you can restore the registry if a problem occurs. For more information about how to back up and restore the registry, review Microsoft Knowledge Base article 332756 (How to back up and restore the registry in Windows).

Try the following steps and see if it resolves your issue:

  1. Click on Start, then type regedit.exe in the Start Search field. When regedit.exe is displayed, press Enter or click on its icon.
  2. Provide consent or the appropriate credentials to User Account Control if prompted.
  3. In the tree pane, navigate to HKEY_CLASSES_ROOT\AllFilesystemObjects\shellex\ContextMenuHandlers
  4. Open the Send To subkey. If it does not exist, create it.
  5. In the Send To subkey, you will see a (Default) string value. Ensure that it is set to {7BA4C740-9E81-11CF-99D3-00AA004AE837}.
  6. Log off and back on.

If you do not feel comfortable editing the registry, I have exported the correct values from the registry of a clean Windows Vista installation which you can import. Simply download the following ZIP file, extract the file contained inside, double-click on it and click on "Yes" to the prompts.

http://localhostr.com/files/5b4fdf/Send%20To%20Menu.zip

Hope that helps, have a great day!
