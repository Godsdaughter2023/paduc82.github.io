For my network of a WIn7x64 connected using a crossover cable to a WinXP, I found some useful advice to resolve this.  I think you are also connected with a crossover cable but you do not say so explicitly.

In the LAN card settings -

Local Area Connection, Properties, TCP/IP 4, Properties
set the Default Gateway for each PC to the IP address of the other PC

so for the 192.168.137.1 PC, set Default Gateway 192.168.137.2

and for the 192.168.137.2 PC, set Default Gateway 192.168.137.1

On both PCs, then go to
Advanced, un-check Automatic metric box, enter 999 in Interface metric box

 

This forces Win7 to recognise the network as a Work network permanently so you don't have to keep reselecting it.

I found this advice by Steve Winograd in the thread Networking XP and windows 7
