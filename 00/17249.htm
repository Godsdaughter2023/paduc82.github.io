(The text files are used in other applications must must remain as text files). I would like to use MS Access 2007 because I am familiar with it, but it does not seem to recognize the VB code that I know for writing to text files (StreamWriter object and writeLine).

How can append to a txt file in MS Access (with a loop that uses values from db in name of test files)?


StreamWriter and writeLine are VB.Net elements, and Access uses VBA, which is a pre-.Net version of the language.  VBA has built-in file I/O statements Open, Write, Print, and Close, as well as the FreeFile() function to get a file handle.  Alternatively, you can use objects from the Windows Scripting library to put an object-oriented gloss on it, at the minor cost of extra overhead.  I usually stick to the built-in I/O features.

For writing text files, you would want to use basic (no pun intended) statements like these:

    Dim intFileNo As Integer
    Dim strFileName As String
    Dim varMyValue As Variant

    varMyValue = 1234

    strFileName = "C:\My Documents\MyFile.txt"

    intFileNo = FreeFile()
    Open strFileName For Append As #intFileNo

    Print #intFileNo, "This is some text, and here's a value: "; varMyValue

    Close intFileNo

Dirk Goldgar, MS Access MVP

Convert multiple tables to text? - Microsoft Community

Convert multiple tables to text?

I have numerous Words documents in which the images were put inside single-cell tables. Is there an automated way to convert all these tables to text?

Thanks,

pjs
doug robbins - mvp office apps & services (word) say:

You could use a macro containing the following code to convert all of the tables in a single document

 

Dim i As Long
With ActiveDocument
    For i = .Tables.Count To 1 Step -1
        .Tables(i).ConvertToText
    Next i
End With

To convert all of the tables in all of the documents in a folder, you could modify the code in the

article "Find & ReplaceAll on a batch of documents in the same folder” at:

http://www.word.mvps.org/FAQs/MacrosVBA/BatchFR.htm

by substituting the above code for the replace routine in the code in that article.

greg maxey say:

Sub ScratchMacro()
'A basic Word macro coded by Greg Maxey
Dim oTbl As Word.Table
For Each oTbl In ActiveDocument.Tables
  If oTbl.Range.Cells.Count = 1 Then
    oTbl.ConvertToText
  End If
Next oTbl


End Sub
