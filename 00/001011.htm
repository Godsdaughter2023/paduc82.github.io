john w. vinson say:

How can I get these employees who don't have evaluation yet?

 


By using a LEFT JOIN to the evaluations table.

 

Select the join line in the query deisgn grid and choose the join type option 2 (or maybe 3): "Show all recrods in tblEmployees and matching records in tblEvaluation".

 

ken sheridan say: If this is for a report rather than a form then the query need not be updatable so you can use the MAX operator in the query much more efficiently than calling the DMax function.  The query would be along these lines:

SELECT tblEmployees.lngEmployeeID, FirstName, LastName,
MAX(dtmEvaluationDate) AS latestEvaluation
FROM tblEmployees LEFT JOIN tblEvaluation
ON tblEmployees.lngEmployeeID = tblEvaluation.lngEmployeeID
GROUP BY tblEmployees.lngEmployeeID, FirstName, LastName;

You can include any  other columns from tblEmployees in the SELECT clause, but they must also be included in the GROUP BY clause.

ken sheridan say:
I could include latest manager too.

Not quite so easily. Remember that the MAX function returns the highest value of a set of values.  With a date this is the latest date, so no problem.  With a manager's name, however, it would be the manager with the last name in the set sorted alphabetically, which won't necessarily be the latest manager. If for instance Manager is a column in tblEvaluation then the easiest way is to first create one query to return the latest date and manager for evaluated employees.  For this you's use a subquery to restrict the outer query to the latest date per employee:

SELECT tblEmployees.lngEmployeeID, dtmEvaluationDate, Manager
FROM tblEmployees INNER JOIN tblEvaluation ON tblEmployees.lngEmployeeID = tblEvaluation.lngEmployeeID
WHERE dtmEvaluationDate =
    (SELECT MAX(dtmEvaluationDate)
     FROM tblEvaluation
     WHERE tblEvaluation.lngEmployeeID =  tblEmployees.lngEmployeeID);

Save this as qryLatestEvaluations, and then join it to the employees table to also return the non-evaluated employees:

SELECT tblEmployees.lngEmployeeID, tblEmployees.FirstName, tblEmployees.LastName,
qryLatestEvaluations.dtmEvaluationDate AS LatestEvaluation,
qryLatestEvaluations.Manager
FROM tblEmployees LEFT JOIN qryLatestEvaluations
ON tblEmployees.lngEmployeeID = qryLatestEvaluations.lngEmployeeID;

There are 3 common protocols for synchronization of emails to mail clients. These are POP/SMTP, IMAP and EAS. The latter has the ability to synchronize not just the mail but the calendar and contacts as well.
