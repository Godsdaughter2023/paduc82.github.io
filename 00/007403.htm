  • the extraction should include not only the word itself with it's sentence, but also the timestamp of the very first sentece-fragment.

graham mayor say:

This one proved rather more testing than the other. The following may work for you, but it only extracts the one line with the entry from multi-line subtitles.

Dim oPara As Paragraph
Dim oDoc As Document
Dim oNewdoc As Document
Dim oRng As Range
Dim oText As Range
Dim i As Long
Dim vFindText As Variant
vFindText = Array("afoot", "appaling", "brink", "circuit", "coastline", _
"commencement", "completion", "considerate", "conveniently", "converge", _
"eligible", "evasive", "facilitating", "frightful", "hunch", "interstellar", _
"lucrative", "merit", "miscalculation", "peckish", "poised", "predicament", _
"protrusion ", "pursue", "resent", "resplendent", "sod", "sodding", "tenable", _
"tripe", "unabashedly", "unabated", "unequivocally", "warlike", "wretched", "zest")
Set oDoc = ActiveDocument
Set oNewdoc = Documents.Add
oDoc.Activate
For i = 0 To UBound(vFindText)
    With Selection
        .HomeKey wdStory
        With .Find
            Do While .Execute(vFindText(i))
                Set oRng = Selection.Range
                oNewdoc.Range.InsertAfter oRng.Text & ChrW(8212) & vbCr
                Set oText = oRng.Paragraphs(1).Previous.Range
                Do While InStr(1, oText, "-->") = 0
                    Set oText = oText.Paragraphs(1).Previous.Range
                Loop
                oNewdoc.Range.InsertAfter oText
                Set oText = oRng.Paragraphs(1).Range
                oNewdoc.Range.InsertAfter oText
            Loop
        End With
    End With
Next i
For i = 1 To oNewdoc.Paragraphs.Count
    oNewdoc.Paragraphs(i).Range = Replace(oNewdoc.Paragraphs(i).Range, "<i>", "")
    oNewdoc.Paragraphs(i).Range = Replace(oNewdoc.Paragraphs(i).Range, "</i>", "")
    If i Mod 3 = 1 Then
        oNewdoc.Paragraphs(i).Style = "normalB"
    Else
        oNewdoc.Paragraphs(i).Style = "gQuote"
    End If
Next i
oNewdoc.Activate

graham mayor say:

The macro is working from a list of words. The document is processed individually for each word in the order the words appear in the list. If the first word is 2/3 the way through the document and the second word 1/4 the way through, the first word timing will be later than the second word. You can only maintain the timing if searching for multiple instances of a single word. Otherwise you have to come up with a means of sorting by the resulting times. If you know the order the words appear then use a word list with the words arranged in that order. The sample document and word list appear to have the words in the correct order, so on that basis, convert the list to a single column table and save the document. Put the document name in the macro.

The following is not very pretty but should work. However I don't understand your reference about sentences. This sort of macro is complicated enough to program without entering that degree of complexity. Use the following as a start point:

Dim opara As Range
Dim oDoc As Document
Dim oWords As Document
Dim oTable As Table
Dim rFindText As Range
Dim oNewdoc As Document
Dim oRng As Range
Dim oText As Range
Dim i As Long, j As Long, k As Long
Dim sFname As String
On Error GoTo CleanUp
'Document name for words list comprising a single column table
sFname = "D:\My Documents\Test\Words.doc"
Set oDoc = ActiveDocument
oDoc.Range = Replace(oDoc.Range, "<i>", "")
oDoc.Range = Replace(oDoc.Range, "</i>", "")
Set oNewdoc = Documents.Add
Set oWords = Documents.Open(FileName:=sFname, Visible:=False)
Set oTable = oWords.Tables(1)
oDoc.Activate
For i = 1 To oTable.Rows.Count
    Set rFindText = oTable.Cell(i, 1).Range
    rFindText.End = rFindText.End - 1
    With Selection
        .HomeKey wdStory
        With .Find
            .ClearFormatting
            .Replacement.ClearFormatting
            .Replacement.Text = ""
            .MatchWholeWord = True
            Do While .Execute(rFindText)
                Set oRng = Selection.Range
                oNewdoc.Range.InsertAfter oRng.Text & Chr(32) & ChrW(8212) & Chr(32) & vbCr
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.Start = opara.Previous.Start
                opara.End = opara.Paragraphs(1).Range.End
                opara.Style = "normalB"
                Set oText = oRng.Paragraphs(1).Previous.Range
                Do While InStr(1, oText, "-->") = 0
                    Set oText = oText.Paragraphs(1).Previous.Range
                Loop
                oNewdoc.Range.InsertAfter oText
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.Start = opara.Previous.Start
                opara.End = opara.Paragraphs(1).Range.End
                opara.Style = "gQuote"
                Set oText = oRng.Paragraphs(1).Range
                oNewdoc.Range.InsertAfter oText
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.Start = opara.Previous.Start
                opara.End = opara.Paragraphs(1).Range.End
                opara.Style = "gQuote"
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.MoveStart wdParagraph, -1
                opara.End = opara.Paragraphs(1).Range.End
                j = Len(rFindText)
                k = InStr(1, opara, rFindText) - 1
                opara.Start = opara.Start + k
                opara.End = opara.Start + j
                opara.Font.Bold = True
            Loop
        End With
    End With
Next i
CleanUp:
sFname = oDoc.FullName
oDoc.Close wdDoNotSaveChanges
Documents.Open sFname
oNewdoc.Activate
oWords.Close wdDoNotSaveChanges
Set oDoc = Nothing
Set oNewdoc = Nothing
Set oWords = Nothing
Set rFindText = Nothing
Set oRng = Nothing
Set opara = Nothing

Below is the revised version to use a simple list document (one word per paragraph - no blanks) with the option to select the file

Dim opara As Range
Dim oDoc As Document
Dim oWords As Document
Dim rFindText As Range
Dim oNewdoc As Document
Dim oRng As Range
Dim oText As Range
Dim i As Long, j As Long, k As Long
Dim sFname As String
Dim fDialog As FileDialog
Set fDialog = Application.FileDialog(msoFileDialogFilePicker)
With fDialog
    .Title = "Select Words List and click OK"
    .AllowMultiSelect = False
    .InitialView = msoFileDialogViewList
    If .Show <> -1 Then
        MsgBox "Cancelled By User", , _
        "List Folder Contents"
        Exit Sub
    End If
    sFname = fDialog.SelectedItems.Item(1)
End With
On Error GoTo CleanUp
Set oDoc = ActiveDocument
oDoc.Range = Replace(oDoc.Range, "<i>", "")
oDoc.Range = Replace(oDoc.Range, "</i>", "")
Set oNewdoc = Documents.Add
Set oWords = Documents.Open(FileName:=sFname, Visible:=False)
oDoc.Activate
For i = 1 To oWords.Paragraphs.Count
    Set rFindText = oWords.Paragraphs(i).Range
    rFindText.End = rFindText.End - 1
    With Selection
        .HomeKey wdStory
        With .Find
            .ClearFormatting
            .Replacement.ClearFormatting
            .Replacement.Text = ""
            .MatchWholeWord = True
            Do While .Execute(rFindText)
                Set oRng = Selection.Range
                oNewdoc.Range.InsertAfter oRng.Text & Chr(32) & ChrW(8212) & Chr(32) & vbCr
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.Start = opara.Previous.Start
                opara.End = opara.Paragraphs(1).Range.End
                opara.Style = "normalB"
                Set oText = oRng.Paragraphs(1).Previous.Range
                Do While InStr(1, oText, "-->") = 0
                    Set oText = oText.Paragraphs(1).Previous.Range
                Loop
                oNewdoc.Range.InsertAfter oText
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.Start = opara.Previous.Start
                opara.End = opara.Paragraphs(1).Range.End
                opara.Style = "gQuote"
                Set oText = oRng.Paragraphs(1).Range
                oNewdoc.Range.InsertAfter oText
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.Start = opara.Previous.Start
                opara.End = opara.Paragraphs(1).Range.End
                opara.Style = "gQuote"
                Set opara = oNewdoc.Paragraphs.Last.Range
                opara.MoveStart wdParagraph, -1
                opara.End = opara.Paragraphs(1).Range.End
                j = Len(rFindText)
                k = InStr(1, opara, rFindText) - 1
                opara.Start = opara.Start + k
                opara.End = opara.Start + j
                opara.Font.Bold = True
            Loop
        End With
    End With
Next i
CleanUp:
sFname = oDoc.FullName
oDoc.Close wdDoNotSaveChanges
Documents.Open sFname
oNewdoc.Activate
oWords.Close wdDoNotSaveChanges
Set oDoc = Nothing
Set oNewdoc = Nothing
Set oWords = Nothing
Set rFindText = Nothing
Set oRng = Nothing
Set opara = Nothing

First, you'll have to add a section break (Insert>Break>Section Break (Next Page)) just before the page where you want the numbering to start. You may have already done this step. Then double-click on the header or footer on the page after the section break to open it. Then disconnect the header or footer by clicking on the Header & Footer>Link to Previous button. Finally, choose Insert>Page Numbers, select an alignment for the number and check Show number on first page (the reference is to the first page of the section). Click on the Format button, choose Start at and enter 1 in the field beside it. OK out to see the effect.
