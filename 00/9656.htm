jimmy rishe say:

It's possible to do this without code, but it's a little tricky.

First, use notepad to create an XML file with the following contents:

<Quarters>
 <Quarter />
 <Quarter />
 <Quarter />
 <Quarter />
 <Quarter />
 <Quarter />
 <Quarter />
 <Quarter />
</Quarters>

Then add this XML file as a secondary data connection.

Add a rule to the Quarter field as follows:

Condition: Quarter is equal to go

Action: Set Quarter to the formula:

concat("Q", 1 + (39 + floor((substring(today(), 6, 2) - 1) div 3) - count(preceding-sibling::Quarter)) mod 4, " ", substring(today(), 1, 4) + floor((substring(today(), 6, 2) - 4 - 3 * count(preceding-sibling::Quarter)) div 12))

Then add a Form Load rule that sets the Quarter field to the value go.

Use the Quarter repeating field as the source for your dropdown, and you're all set!
