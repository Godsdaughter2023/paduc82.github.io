I'd like to be able to back up this information in case of loss of phone, upgrade, or restoration after a firmware upgrade.

Allow me to hide (via a setting - i.e. I never want to see them unless I specifically ask) the entries for old / watched / deleted podcasts to allow easier access to those I still want to listen to.

Implement "smart" (i.e. rule-based) playlists within the Podcast app. e.g. let me set up a rule that dynamically includes all New podcasts that match a particular list of titles. Enhancement request 8:
Better recovery from "stalled" downloads

Ability to specify whether to download all / recent (within a given time period) or only the latest missing podcast when updating automatically. At the moment, anything other than the latest podcast must be downloaded manually.

Enhancement request 10:
Ability to specify times of day to update podcasts e.g.10pm and 6am, rather than intervals between updates (i.e. set updates to occur at times when the user expects to be within reach of the WiFi network which is configured as the access point for the Podcast app).
