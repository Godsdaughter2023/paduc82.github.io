For anyone who is curious, I managed to figure this out. Someone provided a solution on another thread:

  1. Convert the carriage returns to commas with  =SUBSTITUTE(A1,CHAR(10),",")
  2. Copy the result of above and paste special Value into another cell
  3. Use Text to Columns (Data tab, Data Tools group)

Hello AHAAdmin,

 

Thank you for bringing up this issue to our attention. I understand that you want to know a way to have your emails be automatically forwarded to your groups or to multiple contacts. Please be informed that this feature is not available in Outlook.com.

 

The email forwarding option is designed to have one email recipient only. You cannot add a group name or multiple email address in the box. As a workaround, I suggest that you configure the account in Windows Live Mail and set up a rule that will auto Forward your emails to your contacts. Here are the steps:

  1. Once you've configure the account, open Windows Live Mail.
  2. Click the Folders tab.
  3. Select Message rules.
  4. On the Email Rules tab, click New.
  5. Under the Select one or more conditions window, mark the Where the To line contains people.
  6. Under the Select one or more actions window, mark the Forward it to People.  
  7. In the To edit this condition, click the underlined words window, click the contains people link and then type your won email address and then hit Ok.
  8. Click the People link from the To edit this condition, click the underlined words window.
  9. Click Contacts.
  10. Locate and select the Group name that you ahve created in your People page, and then click the To box, then hit Ok.
  11. Then click Ok again.

I hope this helps. If you have clarifications, feel free to post back for further assistance.

 

Thank You.

how to enlarge & print image as poster 3 pages wide and 3 pages - Microsoft Community

bradmillard say:

how to enlarge & print image as poster 3 pages wide and 3 pages tall

how do I take an image thats on one page and enlarge it & print it on 3x3 page size poster?
bob jones aka: cybertaz say:
The only thing you can do from Word -- as well as most word processing programs -- would be to insert the image & scale it to fit the maximum page size Phillip indicated. Objects in a Word document cannot span multiple pages or generate a document that large.

In the Print dialog you can open the Copies & Pages dropdown, select Paper Handling then check the box for Scale to fit paper size & select from whatever paper sizes your printer offers in the Destination Paper Size list.

The type of image also makes a big difference... Keep in mind that if the resolution of the image is not sufficient the quality of the printed output will not be acceptable, though. Despite the impression we may get, you can't just 'stretch' most images to whatever size you wish without consequences.

As Phillip suggested, this type of project is best done in the appropriate type of graphic design software or a quality desktop publishing [Page Layout] program... or contracted to a commercial service.

Bob J.

gh541 say:

Table cell spacing

Hi everyone,

I"ve set up a table style in Word 2011 and have checked "Allow spacing between cells" with a 0.8mm default cell spacing. I've also set up alternate row shading. 

Because the rows in each table are coloured, this means that the spacing between the cells now creates a white border effect as you can see through the cells to the background colour of the page. The only problem with the cell spacing is that the "border" it creates varies in size, ie. I have to kind of adjust the cell height in all tables so that they're more or less the same border weights. Not sure if that's clear.

I know that I could achieve the border effect using table borders instead but I like the fact that you can place a table on a different background and the "border" (formed by the spacing) takes on the colour of the background it's on.

So my question is, is there a way to fix the problem with getting the cell spacing to be even across all rows of a table without my having to adjust it manually? The space after the paragraphs in the table are all the same so I don't think that could be contributing to the problem.

If anyone's experienced this and has any advice, I'd be grateful.

Thanks
john mcghie say:
As always, formatting will be "consistent" if the formatting is always done by Styles, and not, if it isn't.

When using a Table Style, the entire table must have the Paragraph Style of "Normal" applied.  If not, the paragraph style will override the Table Style.  So that's the first thing to check: that the entire table is in Normal style.

You should also check that the Normal style has no space before, after, or either side.  No space before or after, no margins left or right.  Again, it will override your Table Style if it has.

Then you need to check that the Cell Alignment in each cell is the same.  Generally the best result will come from setting the entire table to a cell alignment of "Top Left".  But whichever one you choose, it must be the same for each cell.

You should then make a final check that any manual overrides (such as cell height, row height, font, spacing, etc are removed.

When using Table Styles, for best results the formatting must be "All table style" or "No table style".  If you have a mixture, you will be chasing the result forever.
