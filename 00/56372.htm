jeeped say:
This is one of those times where the SFC /SCANNOW command should actually fix the problem.

Click Start or tap the Windows key and type cmd into the Search programs and files text box then tap Ctrl+Shift+Enter simultaneously. You should receive a UAC warning. Acknowledge it and provide administrative credentials is required. This will open a command prompt window with the elevated privileges you require to successfully run SFC.EXE (System File Checker). With the command window open, run the following command,

SFC /SCANNOW

If you're unsure on whether you want to attempt this, start by running the test command,

SFC /VERIFYONLY

This performs the same operation but reports problems only; it does not attempt repair.

This will cycle through your system and replace/repair any system files found wanting. The LogitDA.dll is one of these. Expect this process to take a little while and you will receive a system report message upon its completion.

c.       Close the elevated command prompt.

d.      Click on the sfcdetails.txt file that was just placed on your desktop to see the SFC scan details (ones with the [SR] tags) in the CBS.LOG.

e.      You can safely delete the sfcdetails.txt file afterwards if you like.

Check the box...Use advanced options...

Advanced Options button / Text tab /
Check the box...Add Text...

(You'll have to experiment with position and font
size and color to achieve the result you desire.)
(text size will have to be adjusted depending on
the size of the photos)

Open the drop window..."Insert a Variable"
choose...EXIF Date Time / Date And Time...
(in the white field you should  see    ($H1)
Left click...OK

On the Batch Convert tab...in the left field...
Left Click the square button "Select Source Folder"
Browse to and select the photos you wish to
append.

Left click the Add button to move the files to
the right field.

Choose an Output Format...

Choose an Output folder...

Click the Convert button...

This is much easier to do than it is to explain so
give it a try before you say "Good Grief...that's too
much work".
