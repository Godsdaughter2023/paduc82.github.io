j w stuart say:

There is a hidden 'Administrator' account
(the account name is: Administrator) in Windows.
The default password for this hidden account is normally blank (none).


For XP Home users:
Boot the computer into Safe Mode
Do this by pressing the F8 key (several times)
start pressing F8 immediately after the Power On Self Test
is finished, and until the Start menu appears.
This will get you to the correct menu window.


Once in Safe Mode,
you will see the normally hidden Administrator account
and any other accounts.
(Note: Accounts are displayed alphabetically).


Use the mouse or Up/down arrow keys to highlight
the 'Administrator' account.
The default password is a blank (no password required).


For XP Pro users:
You do not need to go into Safe Mode.
At the Welcome Screen, do/press the Ctrl-Alt-Del keys twice
to display the Windows logon box.
Type in "Administrator" (without the quotes),
by default the password is blank, no password need be entered.

How to change the wrong email address entered during installing windows 8

I used an non existant email acct when I set up win 8 Google will not let me create this account I need to change end of the address from .co.uk to .com how do I do this

Original Title: When creating my log on email i used .co,uk instead of .com how can i change this

a. user say:

Hi Howard,
Thank you for posting your query in Microsoft Community.


From your description, it appears that you had used a non existing email address while installing windows 8 and you want to change the email address.


To fix this issue you can follow these steps:
Switch to a Local Account, then switch back again to a Microsoft Account using the new email address.
a) Press “Windows + I “to open settings option.
b) Select Change Pc settings
c) Select User account
d) On the right side, it will show your current account with a option that says "Switch to a local account.
e) Select “switch to a local account” and follow the steps, it will enable you to be able to log into Windows as you do with all other versions of Windows


Switching back to a Microsoft Account using new address:


You can choose between whether it uses your Microsoft Account or a normal user account:
a) Press “Windows + I “to open settings option.
b) Select Change Pc settings
c) Select User account
d) On the right side, it will show your current account with a option that says Switch to a Microsoft account".
e) Once you have setup your Microsoft account (this requires a Windows Live ID (Hotmail or MSN, Outlook.com).
Enter the new email address and this will change the email address.
