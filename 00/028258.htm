greginmich say:

Whether I am so tired today, or simply I cannot find the place to initialize the scanning of a USB I just got.

“Scan with Windows Defender” was finally put back in the right-click context menu in the latest version of Windows 10. So now you can just open This PC (or File Explorer), right-click on the USB drive and choose “Scan with Windows Defender”.

You can also automate a command line scan of your thumb drive with an Event Viewer Task as described in this old thread: 

http://answers.microsoft.com/en-us/protect/forum/mse-protect_scanning/is-there-any-way-to-scan-pendrive-automatically/15ecf8bb-7039-4665-bd9c-078844affee6

In Windows 10, the log at Event Viewer/Applications and Services Logs/Microsoft/Windows/DriverFrameworks-UserMode/Operational is disabled by default – so you first have to enable the log. Then, if you want to see the scan result in the command prompt window, you'll have to have the task run this batch file (replace “J:\” with the path of your USB drive):

@ echo off

"C:\Program Files\Windows Defender\MpCmdRun.exe" -Scan -ScanType 3 -File J:\

Pause

stefan blom say:

You can create custom footnotes via the Footnote and Endnote dialog box, but note that you will then lose the automatic numbering.

If you want the slash only in the footnote area, you can do the following: On the View tab, click Draft. Click References | Show Notes; in the footnotes pane, select all notes with Ctrl+A and choose to replace ^f with ^&/. When asked if you want to search the rest of document, click no.

On the other hand, if all notes (both in the main body of the document and in the footnote area) should be followed by a slash, you can just stay in Print Layout view and choose to replace all ^f with ^&/.
