    AutoRecover can help Word recover a file that you were editing after a power failure or other problem that interrupts Word while you are editing the file. If Save AutoRecover Info option is enabled, the autorecovery file will be saved on the: ~/Library/Containers/com.microsoft.Word/Data/Library/Preferences/AutoRecovery/

     

    The save every box determines the interval you want in saving AutoRecover information. You stated that you have configured 10 minutes’ interval, the file recovered on the path above, is the last file saved by your AutoRecover option.
