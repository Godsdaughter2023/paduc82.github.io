cmdrkeene say:

Windows Defender you can disable from the checkbox inside the app's options (or it will also auto-disable if you install any other 3rd party product like AVG, Norton, McAfee, Trend Micro, or whatever else you like).

The Malicious Software Removal Tool (MSRT) is not an app on your system, it's a stand-alone executable file (I don't know if it's MSI or EXE as an extension, sorry), and is downloaded and run a single time during monthly Windows Updates, then it deletes itself.  Since it's delivered as a Windows Update, the way to block it would be to go into Windows Update, right-click it, and choose "Hide Update", but it will be back next month because it's a new tool completely the following month (re-compiled source code, not just re-offering the same original one).
