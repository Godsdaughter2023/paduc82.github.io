Error msg "This disk cannot be used to store a Windows complete PC - Microsoft Community

heather_604 say:

Error msg "This disk cannot be used to store a Windows complete PC backup image because it is not formatted with NTSF"

Original Title: What is NTSF?
When trying to do a complete back up I get the error msg "This disk cannot be used to store a Windows complete PC backup image because it is not formatted with NTSF".  My OS is Windows Vista Ultimate.

What is NTSF?

What type of disk to I use to do a complete backup?

chet814 say:

It's actually NTFS (New Technology File System) and is a type of file system used on hard drives by Windows.

 

The file system on your Windows Vista is probably FAT32. FAT32 has a file size limitation of 4GB (approx.) and since you're attempting a full backup the size of the backup file will be greater than 4GB, hence the error message you receive.

 

FAT32 can be converted to NTFS, see the article below:

 

Convert a hard disk or partition to NTFS format
 http://windows.microsoft.com/en-US/windows-vista/Convert-a-hard-disk-or-partition-to-NTFS-format

 

Prior to converting your file system, it is recommended that you make a backup of your personal files and data  to external media (CD/DVD, CD-RW/DVD-RW, external hard drive, etc.) just in case. I realize that you are attempting a full backup so it may sound strange suggesting you do a backup, but only of your personal data which should be a much smaller file or may be done in groups, etc.

 

BTW, storing a backup of your system or personal data on the same physical hard drive has little value, i.e., should the drive fail the originals are lost as well as the backups. Backups should be on at least a different physical hard drive and an external hard drive is recommended.

 

Regards,
