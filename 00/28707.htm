hansv mvp say:

Select B2:R10.

On the Home tab of the ribbon, select Conditional Formatting > New Rule...

Click on 'Format only cells that contain'.

Leave the first dropdown set to the default 'Cell Value'.

Select 'equal to' from the second dropdown.

Enter the following formula in the box next to it:

=MIN(B$2:B$10)

Click Format...

Activate the Fill tab.

Select green.

Click OK, then OK again.

To count the number of wins, I'd create a helper row, for example in row 12. Enter the following formula in B12:

=MIN(B$2:B$12)

Fill right to R12. Then, enter the following formula in row 2 in the Rounds Won column:

=SUMPRODUCT(--($B2:$R2=$B$12:$R$12))

Fill down to row 10.

ken sheridan say: You can return the total number of rows in a subquery, so to get the percentage you'd return the number of rows which meet the criteria divided by the value returned by the subquery, and multiplied by 100, e.g.

SELECT COUNT(*)/
    (SELECT COUNT(*)
      FROM YourTable)*100 AS Percentage
FROM YourTable
WHERE <your criteria expression>;

tom van stiphout say:

Welcome!

Thanks for taking up the challenge. Not everyone here is an English language native speaker but everyone tries to be.

"Serial Number" can mean many things. In its simplest incarnation it's a "Unique Number". In that case, an AutoNumber data type will suffice.

A more problematic one is "A sequential number without any gaps". That would be a challenge even for seasoned developers, because it would presumably include renumbering records after one is legitimately deleted. Anyway, search for "DMax+1" to see if it is a fit for you.

A more fundamental issue is: why do you need a serial number? Many applications survive without one.

bob jones aka: cybertaz say:
When you go to the Print dialog open the Copies & Pages list & select Microsoft Word. Open the Print What list, then choose Styles to print the list.

That produces an alphabetized list of all Styles available in the document -- including those that have not actually been applied. The attributes of each Style are included as well.
