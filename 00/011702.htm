steve rindsberg say:

You can add Tags to presentations, slides or shapes.  I'd consider using tags on your slides for this.

Select a single slide or a group of slides and use this to tag them:

Sub TagEm()

Dim oSl as Slide

For Each oSl in ActiveWindow.Selection.SlideRange

.Tags.Add "MyTag", "4"

' Change 4 to be any other value you might select and search for.

Next

End Sub

 

Then to hide/show slides, assuming again that you're looking for 4 as the value

Sub HideUnhide()

Dim oSl as Slide

For Each oSl in ActivePresentation.Slides

If oSl.Tags("MyTag" = "4" Then

oSl.SlideShowTransition.Hidden = msoFalse

Else

oSl.SlideShowTransition.Hidden = msoTrue

End If

Next

End Sub

terfar say:
One easy way is as follows...
 
Open Find and Replace (Ctrl+H)
In the Find What box enter :+space (that is type a colon followed by a space)
In the Replace With box enter tab (that is press the tab key)
Press Replace All
 
Now select All the text and use Convert Text to Table selecting Tab as the delimiter.
 
Now you should have a two column table with Speaker/Another Speaker/etc in the left column and the speech in the right column.
 
-- Terry Farrell - MSWord MVP

suzanne s. barnhill say:
You just need to set a hanging indent at the appropriate location. The best way to do this is through a dedicated style for the dialog. You can use the ruler to drag a hanging indent or set it specifically from within the Paragraph dialog.
 
By default, there is a tab stop at the hanging indent position, so you can press the Tab key after the character's name. For existing text, assuming you have inserted a space after the colon, you can use Replace (Ctrl+H) to replace the colon and space with a colon and a tab character.
  1. In the "Find what" box, type ": " (that is, a colon and a space, without the quotation marks)
  2. In the "Replace with" box, type ":^t" (without the quotation marks).
  3. Don't use Replace All unless you are sure there are no colons elsewhere in the text.
