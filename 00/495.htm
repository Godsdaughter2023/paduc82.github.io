bobgreenblatt say: OK, here's one way, suppose the column containing the values you want to filter on are in column A, starting in row 1. Then in a spare column to the right, in row 1 of that column, enter the formula:
=countif(a:a,a1)
Fill this formula down for the length of your list. Now it will contain counts of each item in the list. Next select the entire range of your list, including this extra column. Unfortunately is is right.

Photo Album is a very quick and easy way to CREATE an album but not at all flexible when it comes to updating.

In your particular case it would (maybe) be possible to write some simple code to track where the notes came from and then put them back after updating but it would depend on keeping the original order unchanged.

Make a copy first

INSERT Module and paste in

'------------------------------

Sub tagger()
Dim osld As Slide
For Each osld In ActivePresentation.Slides
osld.Tags.Add "SLIDE", CStr(osld.SlideIndex)
Next
End Sub

Sub putNotesBack()
Dim L As Long
Dim origSld As Slide
For L = ActivePresentation.Slides.Count To 2 Step -1
If ActivePresentation.Slides(L).Tags("SLIDE") <> "" Then
Set origSld = ActivePresentation.Slides(CLng(ActivePresentation.Slides(L).Tags("SLIDE")))
origSld.NotesPage.Shapes(2).TextFrame.TextRange = _
ActivePresentation.Slides(L).NotesPage.Shapes(2).TextFrame.TextRange
ActivePresentation.Slides(L).Delete
End If
Next L
End Sub

Text Field To Display In Upper Case - Microsoft Community

Text Field To Display In Upper Case

I have a Text field that I want to display so that all characters appear (ie will display) in upper case.

How do I do this please?

TIA for any replies.

john p spencer say:

Do you want to change the values PERMANENTLY?  If so, you can use an update query to change the data.

UPDATE YourTable

SET YourField = UCase(YourField)

To do that in design view, add your table, select your field into the list, select query Update.  Now in the UPDATE to line enter UCase([YourField]).  DO NOT FORGET the [] characters or Access will change the text to the literal string "YourField".

If you just want to change the display of the date on a form or in a report or even in datasheet view of a query, you can set the format property of the control (or field).

To do so, in design view select the control and enter the > sign.  If the field is a memo field this has a side effect of truncating what is displayed to 255 characters.

hansv mvp say:

You have no control over the way Excel displays numbers in the formula bar.

 

You can enter the values as strings in the formula:

 

=GEOMEAN("2.7e-2", "1.3e-3", A4, B$7)

 

Excel will convert the strings to numbers (if possible, of course).

 

Alternatively, store the values in cells, format the cells the way you want, and refer to those cells in your formula.
