xbr dionysus say:

Change Wireless Channel

Login Information

 

  • Gateway: 192.168.1.1
  • Username: admin
  • Password: admin or a custom password was created during setup.

Change Wireless Channel

 

  1. Connect a computer to the gateway with an Ethernet cable.
  2. Open a web browser on a computer.
  3. Type the address 192.168.1.1 into the web address bar.
  4. When prompted for the username type in admin, use the password created during the setup process. If you did not setup one then the use the default password admin.
  5. Once logged in click on the Wireless tab on the top of the page.
  6. Click on the Basic Wireless Settings option in the subcategory.
  7. Locate the Standard Channel drop-down box.
  8. Choose a channel
  9. Click the Save Settings button
