lemp say:

If it's been less than 30 days since you upgraded to Windows 10 and you haven't deleted the C:\Windows.old file, you can try the automatic "go back" tool provided in Windows 10.  From what I've seen, your chances of getting a satisfactorily-running Windows 7 (or 8.1) system are not too high.

If it's been more than 30 days, or if you deleted the windows.old file, or if you just want a stable installation of Windows 7 (or 8.1), you can do a clean install of the earlier version of Windows. 

There are a couple of problems with this approach, however.  You have to find an installation DVD/USB for the earlier version of Windows (which may be difficult if Windows was pre-installed on your computer) and you'll have to reinstall all of your programs and data (which you would back up before you begin doing anything) and then run Windows Update (which has its own problems, see Windows 7 Update Problems? Read this first. )

Or, if Windows was pre-installed and your installation of Windows 10 did not destroy it, you can use the manufacturer's "return to factory condition" option.  In this case, you'll have to first uninstall all of the bloatware that was pre-installed, especially any trial versions of antivirus software using the correct tool, and then reinstall all of your programs and data ) and run Windows Update.
