Hi Jon,

An outdated or incompatible network adapter driver may cause connection problems. We suggest that you check if an updated driver for your network adapter is available.

If Windows can't find a new driver for your network adapter, visit the PC manufacturer's website and download the latest network adapter driver from there. If your PC can't connect to the Internet, you'll need to download a driver on a different PC and save it to a USB flash drive, so you may manually install the driver on your PC. You'll need to know the PC manufacturer and model name or number.

To manually install the network adapter driver:

  • If you downloaded an executable file (.exe), just double-click the file to run it and install the drivers.
  • If you downloaded individual files, and at least one file has a .inf file name extension and another has a .sys extension, do the following:
    1. In the search box on the taskbar, type Device Manager, and then select Device Manager from the list of results.
    2. In Device Manager, select Network adapters > the network adapter's name. (If it's not listed there, check Other devices.)
    3. Right-click the network adapter, and then select Update Driver Software > Browse my computer for driver software.
    4. Select Browse > select the location where the driver files are stored > OK.
    5. Select Next > follow the steps to install the driver > Close.
    6. After you've updated the driver, select the Start button > Power > Restart if you're asked to restart, and see if that fixes the connection issue.

If you need further assistance, feel free to ask us.
