jay freedman say:

Word doesn't have any such field. Even if it did, it probably wouldn't automatically update.

You can make something like this work with a macro. There are various ways to do it, depending on what else is in the document. If the document uses legacy form fields and forms protection, then you would use a form field to hold the count; otherwise you could use a predefined bookmark or a content control. I'll show you how to use a content control.

First, at the location in the document (or its template) where you want the count to appear, insert a plain-text content control. Open its Properties dialog and enter a value in the Tag box. For this example, where I'll count the number of instances of the Heading 2 style, I'll use the Tag value H2count. (It could be any text you prefer, but be aware that finding a content control by its tag is case-sensitive.) You may also want to turn on Design Mode in the Controls group and change the placeholder text, then turn off Design Mode.

Open the macro editor and insert this code. Then change the name of the macro as desired, change the "H2count" to whatever tag you used (remember, it's case-sensitive), change the wdStyleHeading2 to the constant for the style you're counting (if it's a custom style, use its name in quotes instead), and comment out or remove the LockContents statements if you don't need them.

Sub CountH2instances()
   Dim rg As Range
   Dim ccs As ContentControls
   Dim cc As ContentControl
   Dim counter As Long
   
   Set ccs = ActiveDocument.SelectContentControlsByTag("H2count")
   If ccs.Count > 0 Then
      Set cc = ccs(1)
      cc.LockContents = False  ' to allow manual changes
      
      counter = 0
      
      Set rg = ActiveDocument.Range
      With rg.Find
         .Style = ActiveDocument.Styles(wdStyleHeading2)
         .Wrap = wdFindStop
         While .Execute
            counter = counter + 1
         Wend
      End With
      
      cc.Range.Text = CStr(counter)
      cc.LockContents = True  ' to prevent manual changes
   End If
End Sub

Finally, add a button to the Quick Access Toolbar  or a keyboard shortcut to run the macro to update the content control. You can do that as often as needed.

Note that if there is no content control in the current document with the specified Tag value (case-sensitive!), then nothing will happen when you run the macro.
