=IFERROR(INDEX($N$15:$N$999,SMALL(IF($A$15:$A$999=B$1,ROW($1:$984)),ROW(1:1))),"")

As an array formula, that require Ctrl+Shift+Enter rather than just Enter. once entered correctly, it can be filled right to M2 then B2:M2 can be filled down to B2:M11.

jeeped say:
Essentially your formula has been askling go look up this value, but only look in one cell when it should have been asking go look up this value in this range of cells. VLOOKUP() is written like this,

=VLOOKUP(<something to lookup>, <a bunch of cells to look it up in>, <column to return value from>, <exact or approximate>)

The formula in your Sheet1!B2 should have been,

=VLOOKUP(A2,Sheet2!$A$2:$A$28,1,FALSE)

Note that the second parameter is a range of cells and not a single cell. I still don't understand why you want to look up a value you already know the value of. As mentioned in my earlier posts, you would get more value out of something like MATCH() that at least returns the position the item is on within the range.
