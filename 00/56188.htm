As it’s .exe programs that you cannot run, it can be difficult to resolve. However, one of these should solve it:

1. In Windows Explorer, navigate to \Windows\System32, right-click on cmd.exe and choose Run as administrator. If that fails because it’s an exe program, use the program command.com, which is in the same folder. Now type assoc .exe=exefile and press Enter. You may have to restart the computer.

2. On another computer, open Notepad, type assoc .exe=exefile and save it on an external device as tempassoc.bat. Just double-click it to run it on the ‘infected’ machine.

3. Boot to Safe Mode (start tapping F8 when you switch on) and choose Safe Mode with Command Prompt. At the prompt type assoc .exe=exefile and press Enter. There will be no acknowledgement, just type exit and press Enter. Restart the computer.
