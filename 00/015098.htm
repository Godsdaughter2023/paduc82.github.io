Macro to export Comment numbers along with the Comments to a new - Microsoft Community

mageeg say:

Macro to export Comment numbers along with the Comments to a new document

I'm using Word 2007 on a PC.

I've been trying to figure out how I can end up with a text-only list of Comments in a separate Word file, rather than using the "Print list of markup" option in the Print dialogue box, since that gives me a bunch of graphics and colors I don't want. I just want the text. 

I found just the macro I want (below); however, instead of placing the commented text in Column 2, I want to place the Comment number in that column (e.g., [M2], [M103]). What would I have to change in the following macro to have it place the Comment number in Column 2 instead of the text that was selected when the Comment was inserted?

Lene Fredborg posted it in Jan 2007 in response to a similar query :
:
Maybe you can use the macro included below.
The macro creates a new document with a 4-column table which will be filled
with information about the comments found in the document that is active when
you start the macro. The following information will be inserted in the table:
Column 1: Page number where the comment is found
Column 2: The commented text (i.e. the text that was selected when the
comment was inserted) [I want to change this]
Column 3: The comment itself
Column 4: Comment author

You may want to adjust the table layout (this could be done by the macro too).


Sub CreateCommentsDoc()
Dim oDoc As Document
Dim oNewDoc As Document
Dim oTable As Table
Dim nCount As Long
Dim n As Long


Set oDoc = ActiveDocument
nCount = ActiveDocument.Comments.Count


'Create a new document for the comments
Set oNewDoc = Documents.Add
'Insert a 4-column table for the comments
With oNewDoc
.Content = ""
Set oTable = .Tables.Add _
(Range:=Selection.Range, _
numrows:=nCount + 1, _
NumColumns:=4)
End With


With oTable.Rows(1)
.Range.Font.Bold = True
.Cells(1).Range.Text = "Page"
.Cells(2).Range.Text = "Comment scope"
.Cells(3).Range.Text = "Comment text"
.Cells(4).Range.Text = "Author"
End With


'Get info from each comment from oDoc and insert in table
For n = 1 To nCount
With oTable.Rows(n + 1)
'Page number
.Cells(1).Range.Text = _
oDoc.Comments(n).Scope.Information(wdActiveEndPageNumber)
'The text marked by the comment
.Cells(2).Range.Text = oDoc.Comments(n).Scope
'The comment itself
.Cells(3).Range.Text = oDoc.Comments(n).Range.Text
'The comment author
.Cells(4).Range.Text = oDoc.Comments(n).Author
End With
Next n


oNewDoc.Activate
MsgBox "Finished creating comments document."


Set oDoc = Nothing
Set oNewDoc = Nothing
Set oTable = Nothing


End Sub
