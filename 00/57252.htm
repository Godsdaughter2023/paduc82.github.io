yaqub k say:

Hi,

 

It is not possible to show both the notification icons when you connect wired and wireless network at the same time in system tray.

 

When you connect a wired connection it will show the wired connection icon and if you connect to internet wirelessly it will show a wireless connection icon. Both will never appear at the same time.

However if you have connected both wired and wirelessly to internet it will show either of the two icons based on the priority you have set.

We can change the priority of Network Connections and make them follow a desired connection order. You can make Windows 8 use wired connection instead of Wireless connection. When you start your laptop, if there is a wireless connection available, your Windows laptop will connect to the Wi-Fi connection. Even if you connect to a wired connection, the usage will continue to be from the Wi-Fi connection. This is because, when more than one network connection is available, Windows uses the one with the lowest metric value.

To change the Network Connection Priority, Follow the steps.

a. Press Windows key + C.

b. Select Search and type network connections.

c.  Now click on the Alt key to make the Menu bar appear. Click on Advanced Settings. This will open its Properties box.

d.  Under the Adapters and Bindings tab, you will see the list of connection and their order in which they are accessed by Network Connections and other related Windows services. The default is Wi-Fi, Ethernet and then Remote Access. The Network Connections service manages objects in the Network and Dial-Up Connections folder, in which you can view both local area network and remote connections. Using the Up and Down arrows, you can change their order, and set it according to your priority. When you change the setting as shown above and make Ethernet as the first choice, your laptop will use the wired connection first by choice.

e. Click OK and Exit.

Your Windows PC will now follow this order of priority when detecting available network connections.
