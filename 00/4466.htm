pheasey say:

Change Case Menu Missing in Word 2003

Word 2003

Each time I click on Format - Change Case the menu window is NOT displayed.  I have several PC's and it is working fine on all except one.  On this troublesome one the function still works - i.e. if I select text and click Format - Change Case, Word will perform the task and change the case to UPPERCASE.  I have to repeat he steps to get it to go to lowercase, and again for Title Case etc.  However, what I want is for the menu window to open (as it does on all other copies) so that I can choose the case option myself.

jay freedman say:
There are actually two Change Case commands that could be on your menu. The one that should be there, the one that opens the dialog, says "Change Case..." with the three dots. The other one says "Change Case   Shift+F3", and it just cycles through the items in the dialog (except Title Case) one at a time. It sounds like you've somehow gotten the wrong one on the menu.

Click Tools > Customize. Open the Format menu and drag the Change Case command off it; drop it anywhere in the document area and it will disappear. Now, in the Customize dialog, click the Commands tab. Select "Format" in the Categories list. Then select "Change Case..." in the Commands list (it's near the bottom). Drag it to the Format menu and drop it below the Text Direction command. Close the Customize dialog.

By the way, the reason a reinstallation didn't fix the problem is that the menu customizations are stored in the Normal.dot template. Microsoft fixed things so that uninstalling Word or Office doesn't delete Normal.dot or other files that have changed since installation, and doesn't overwrite them during reinstallation, because you might want to keep the customizations. But that means many problems can't be fixed that way.
