bill mosca say:

Assuming all you have is the file path in the hyperlink, you can run an update query to change the folder path. A hyperlink with just a file path actually looks like this if it were text:
#FilePath#

Say you file is C:\My Folder\MyFile.pdf

You would need to capture just the file name and replace the folder path. a simple update query would work for this.

I used [NewFolder] as a parameter that will prompt the user for the new folder path.

UPDATE MyTable
SET MyLink = '#' & [NewFolder] & Replace(Mid([MyLink], InStrRev([MyLink], '\')),'#','') & '#'

 

What it is doing is getting the file name and concatenating the new folder onto the file name.

bill mosca say:

Ah, then there is a display name in the link. That makes it a bit more complicated. To see what the actual text is make a copy of the table (just in case) and change the hyperlink field to text. Save it and open the table in datasheet view. You'll see something like this:

UPDATE MyTable
SET MyLink = Replace(MyLink, 'C:\MyFolder', 'C:\MyNewFolder')
