greatemerald say: On Windows 7 32-bit, you can get it working, but it takes some doing. First of all, the D-Link website doesn't offer Windows 7 drivers, but offers drivers for pretty much every older Windows OS. They also don't offer an installer. So what you have to do in order to install a driver for an older OS is to go to the Device Manager, locate your DUB-E100 and press "Update drivers". From there, choose to update them manually, then tell it to search in the Windows 2003 x86 driver folder. Now you could install the Vista drivers, but they actually have serious issues - your internet connection will work, but you will get heavy data corruption, so any download would be a guaranteed failure, while webpages would display properly only after refreshing them several times. The Windows 2003 drivers work a lot better. I believe that the internet speed is slower with them, but at least you don't get data corruption. Also, in case you want to switch drivers, you have to uninstall the ones you have installed first, or the system will simply say that the drivers are already current. I also haven't tried the XP drivers yet.

Hi Anu,

 

Thank you so much for posting your query on this forum. You have reached the right place and I will try my best to assist you with this issue.

 

It seems like you have chosen incorrect settings while creating the hotspot on the computer. I would suggest you to delete the existing hotspot that you have already created and then recreate it again using Command Prompt. Please follow the steps given below:

 

a) Press “Windows Logo” + “R” keys on the keyboard, type “ncpa.cpl” in the “Run” Command box and press “Enter”.

b) When your network connections open, right-click on your Wireless network adapter and select “Properties” from the context menu.

c) Click on the “Sharing” tab and allow other devices to use your machine’s Internet by selecting the first option under the section “Internet Connection Sharing”. Uncheck the second option, if it is already enabled and the click on “OK”.

d) Now, go to the desktop interface, press “Windows Logo” + “X” keys on the keyboard and select “Command Prompt (Admin)” from that menu.

e) Now, the first thing we need to do is set up the wireless network using the “netsh” command.

f) Type the following command in the Command Prompt window and press “ENTER”.

 

netsh wlan set hostednetwork mode=allow ssid="(Enter the name/SSID of your network here)” key="(Enter the network password here)"

 

g) Now we are ready to start broadcasting our network. For that, enter the following command in the command prompt window and press “ENTER”.

 

netsh wlan start hostednetwork

 

h) You should be seeing a message “The hosted network has started” on the command prompt window.

i) To check if this is working, you may type the following command on the command prompt window and press “ENTER”. This command also gives you the information on clients which are connected to your wireless access point.

 

netsh wlan show hostednetwork

 

j) Now, you should be able to see the newly created access point on all the other devices and you may try connecting to it by using the passkey which you have used in the “Step f” and check if you are able to connect to the Internet.

 

Hope this resolves the issue. Please get back to us with the result.

Hi Franco

The Activation on the computer where the Free version of Windows 10 is being installed is completely automatic.

There are no Product Keys associated with the Free Windows 10 upgrade offer. The free upgrade uses a completely new form of Licensing and Activation.

When you upgrade through Windows Update a check is performed to make sure the current version of Windows on the computer is activated.

An inventory is taken of the hardware on the system and a unique ID number is created and uploaded to the Cloud. This ID is used to Activate Windows 10. This process is called a Digital Entitlement and is used make sure that you can use this version of Windows 10 for the lifetime of the computer where it is installed.

The first time you install the Free Upgrade, you must perform an Upgrade in order for the Entitlement to be properly created. After this is done, you can then re-install Windows 10 on the same computer as many times as you wish, including a Clean Install.
