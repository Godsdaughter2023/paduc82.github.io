Port Forwarding - Microsoft Community

xbr dionysus say:

Port Forwarding

Login Information

  • Gateway – 192.168.1.1
  • Username – admin
  • Password – admin
Port Forwarding
  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select Port Forwarding from the sub menu.
  6. Check the Enable Port Forwarding box.
  7. Enter the IP Address of the Xbox into the IP Address box.
  8. For Protocol select Both.
  9. In both Port Range boxes type 3074.
  10. In the Comments box type Xbox.
  11. Select Apply Changes.  

 

Disable Port Forwarding

  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.1.1) into the address bar of the internet browser.
  3. If prompted to login enter “admin” as the username and “admin” for password.
  4. Select Firewall from the menu on the left.
  5. Select Port Forwarding from the sub menu.
  6. Click the Select radio button next to the port you want to remove.
  7. Click the Delete Selected button.
  8. Click the Apply Changes button.
