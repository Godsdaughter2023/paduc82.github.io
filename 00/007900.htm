Hi Richard - I was looking at your earlier post; thank you for your positive comments about the community!

Now the one thing you never want to use in WMP is 'Update Album Info' - if you have altered anything it will change it back to whatever data was originally associated with that file. 'Find album info' is fine because you can search for a correct version in the database and select that (being a global database there can be varying versions of many mainstream albums).

When you edit information in WMP that information is retained in the properties/details tab of the file... Conversely if you alter information in the details tab that will appear in WMP. If I wish to significantly alter a track's details I use the properties - details tab and do this work away from WMP (usually on an external drive or in the Documents folder) so that there is no muddling of info.

To prevent files changing back all you should really need to do is - avoid 'Update info' and limit the control WMP exerts in the 'Organize-Options' Library and Privacy tabs - apply the settings along the lines I use by matching those shown below especially the red bits (then click 'Apply and OK') - Get back if any clarification required...
