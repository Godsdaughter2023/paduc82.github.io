jay freedman say:

You don't really want to "fill" the blank space with the number, as that would make each entire paragraph into one word, and it would no longer show the automatic line breaks properly. Instead, you need to add a number after each space. Also, in order to number the first word of each paragraph you need to add a number at the start of the document and after each paragraph mark. This macro will take care of that.

Sub NumberWords()
    Dim rg As Range
       
    ActiveWindow.View.ShowFieldCodes = False
       
    ' place a SEQ field at the start of the document
    Set rg = ActiveDocument.Range
    rg.Collapse wdCollapseStart
    ActiveDocument.Fields.Add Range:=rg, Type:=wdFieldSequence, _
        Text:="wd", PreserveFormatting:=False
    
    ' find each "white space" (incl. multiple spaces, tabs, nonbreak spaces)

    ' and insert the field after it
    Set rg = ActiveDocument.Range
    With rg.Find
        .Text = "^w"
        .Wrap = wdFindStop
        While .Execute
            rg.Collapse wdCollapseEnd
            ActiveDocument.Fields.Add Range:=rg, Type:=wdFieldSequence, _
                Text:="wd", PreserveFormatting:=False
            rg.Collapse wdCollapseEnd
        Wend
    End With
    
    ' find each paragraph mark and insert the field after it
    ' unless it's the last one in the document
    Set rg = ActiveDocument.Range
    With rg.Find
        .Text = "^p"
        .Wrap = wdFindStop
        Do
            .Execute
            rg.Collapse wdCollapseEnd
            If rg.End = ActiveDocument.Range.End - 1 Then Exit Do
            ActiveDocument.Fields.Add Range:=rg, Type:=wdFieldSequence, _
                Text:="wd", PreserveFormatting:=False
            rg.Collapse wdCollapseEnd
        Loop
    End With
    
    ' update all fields
    ActiveDocument.Fields.Update
End Sub

Here is another technique you can try:

Remove Cover page from page count

ie  For the second part of the “Page x of Y”

1.  In footer remove field representing “Y” page count

2.  Press Ctrl+F9.

3.  Inside the brackets type =

4.  The press Ctrl+F9 again & type NUMPAGES

5.  Then outside the next bracket type - 1    so that it looks like {= {NUMPAGES} -1 }

An automatic way to create a non-blank, even-numbered page at the end of a chapter, if the chapter would otherwise end on an odd page OR Force Chapter to start on an Odd Page

want new Chapters to always start on an odd page use a MOD function in a field.  Perhaps that will help.  The syntax is:

{ IF{ =MOD({ PAGE },2)}= 0 " " "[page break]This page is intentionally blank." }

where { and } are field bounding characters produced by pressing Ctrl + F9; and where [page break] stands for a hard page break - press Ctrl + Enter to insert it.  It cannot be a Section Break. 

Note    that there are three nested fields: hit Ctrl + F9 three times in a row to give you your three nested sets, then fill in the rest.

Note    the spacing carefully: the spaces or absence of spaces is critical. And note the final " } on the fourth line.

For more details, see an article by John McGhie at http://www.mvps.org/word/FAQs/TblsFldsFms/InsEvnPgEndSctn.htm
