bob jones aka: cybertaz say:

The direct answer to your question is that the Task Panes in Office 2016 cannot be disabled, but it can be avoided. However, the key question is: For what purpose are you double-clicking the Text Box?

If you're simply intending to format the Text Box, itself, only a single click is needed to select it. A single click will not elicit the Format Shape Task Pane.

If you actually are trying to edit content within a Text Box, Word 2011 works pretty much the same way: If the Text Box is not selected double-clicking it simply produces the Format contextual tab on the Ribbon without placing the insertion point or selecting any existing content. The main difference is that there is no Format Shape Task Pane in Word 2011. In either version, though, the Text Box first must be selected by a single click. The next click places the insertion point within the Text Box or text can be selected by a double-click.
