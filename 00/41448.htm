robert aldwinckle on forums say:

I looked at Windows Settings and did not find any setting related to double clicking.

It is not in the modern settings but is in the Desktop ones for Mouse Properties.  If you are like me, even though you specifically request  double-click speed  and see  Mouse Properties  open as a result, Mouse and Keyboard Center overrides the choice of tab.  The correct tab it should be opening in for that search request is Buttons.  Note that there is a slider in there for Double-click speed.  Use the folder beside it to check that your double-click speed is appropriate; e.g. make sure that by double-clicking on the test folder icon that you can reliably open and close it.

BTW some Mouse support packages, including Mouse and Keyboard Center provide button customization.  So, you could also check there to make sure that you have not reassigned a button inappropriately, or perhaps forgotten that you assigned a different button to be your double-click function; etc.   Otherwise, assuming you are using the button that you have left assigned as your primary single-click (AKA "left click") then you would use that for both the double-click speed setting and whatever it is that you are having trouble using a double-click on.

Also, note that there are other options which you may have set which could confuse things if you have forgotten about them.  For example, there is an option which allows a user to single-click when they would normally have to double-click.  In that case double-clicking would cancel the single-click request and thus result in the symptom that you are complaining about.
