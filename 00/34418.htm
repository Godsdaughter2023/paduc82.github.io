Only those tif files over a certain length (60+ - Microsoft Community

jh wilson say:

Only those tif files over a certain length (60+ chars): can't open, delete, view, change name or anything

Only those tif files over a certain length (60+ chars):
I can't open/view them with any app, can't delete, and can't even get the file name to allow me to try to shorten the name. I can't copy the files or even move them from their current folder.  What is going on? They are files that I saved last year. (I don't recall whether I ever tried to open them before.) My computer is an HP laptop, DV 6000 series, Media Center Edition Windows XP with ServicePak 3.
wunders say: There is a fundamental limitation in Windows that the total length of a filename, including the path, cannot exceed 255 characters.  It is possible that the total path to your .tif files is something long like:
  C:\Documents and Settings\Username\My Photos\MySummerVacationLastYear\BigFileName.tif
and the total number of characters in all of that exceeds 255.

There are a couple of ways that you can deal with this.  One is using the "SUBST" command to alias a drive letter to the folder containing your files.  To do this, bring up a command prompt window (Start -> Run -> "cmd")  and enter a command like (to use the above as an example):

   SUBST  M:  "C::\Documents and Settings\Username\My Photos\MySummerVacationLastYear"

Where "M:" is an unused drive letter and the rest is the path to the folder containing your files.  Now you can access your file using:  
   M:\BigFileName.tif
Which will have significantly fewer characters in the path than your original file and let you access them.

A second approach is to create a folder at the root level of your hard drive (e.g.  C:\Pics ) and then copy your files into this folder.  Not just any copy program can handle copying long paths (as you've discovered), but "Robocopy" (Robust Copy) can.  You can get Robocopy free from the Windows Server 2003 Resource Kit on the Microsoft Download website
   < http://www.microsoft.com/download/en/details.aspx?displaylang=en&id=17657 >
After you copy the file(s), it will have a much shorter path and be accessible.  Robocopy is a command-line program and must be run from a command-prompt window.
Alternatively, you might be able to move the folder containing these pictures into this root-level directory.

HTH,
   JW
