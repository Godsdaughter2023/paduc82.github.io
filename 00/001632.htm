carsenogenix say: I finally fixed the issue. Below is my solution...

In SkyDrive there should be three categories on the left side of the main page:
- Files
- Documents
- Photos

If I selected the "Files" category, there would be a "Documents" folder, "Pictures" folder, "Public" folder, plus various other folders that I created. (Note that there is a "Documents" category AND folder)

If I selected the "Documents" category, then the "Documents" folder would go missing.

What I discovered was that the "Documents" folder was located in the "Photos" category.

What I needed to do was create a new "Documents" folder in the "Documents" category. (I created one called "Documents 2", then moved everything from the original "Documents" folder to "Documents 2", then deleted the original "Documents" folder, then renamed the "Documents 2" folder to "Documents").

At this point my "Documents" folder was now located in the "Documents" category.

After all of this, in the "Documents" folder, I created a OneNote file called "Personal (Web)" and created a page in the file called "Unfiled Notes". I believe this is necessary in order for the app to work. See http://office.microsoft.com/client/helppreview14.aspx?AssetId=HA102743695&NS=MOB_ON_iOS for more info on this.

After ALL these changes the OneNote app FINALLY worked.

Hope this isn't completely confusing. Good luck!

·         Click Start

Done it!

First changed the cell contents to text by replacing = with $$$=FLOOR(
