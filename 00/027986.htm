mjainuec say:

Worked out solution:

 

I was facing the same problem and solved it without even formatting the phone.

 

The problem is with application faceblog which creates hidden and system files regularly and takeup the whole space.

These files are not visible with the regular file explorer given in N73. And even if you can somehow see these files, ou cannot simply delete them as they are currently being used by lifeblog.

So here is the procedure:

 

Download JbakTaskman, a task manager tool  and X-plore (a file explorer) to ur cellphone.

Open Jbaktaskman. Select show processes from the menu. and terminate the process LifeblogDBSERVER.exe

now you can delete those files

Open X-plore. Change its configuration to show hidden and system files as well. Open C:\data\lifeblog. Under some subfolders u might see a huge file of lifeblog (some db file) with around 30 MB size. Delete that now.

Its done.. ur phone memory is again freed now.

 

To prevent this problem from occuring again, open lifeblog->settings->item types-> uncheck all and then lifeblog->settings->Autoclean up, set all things to "Always".

 

Enjoy.

 

Note: There is another way to clear your phone memory. Just change the above lifeblog settings. But generally users encounter this problem when whole of their memory is full and lifeblog cant be opened at that stage and even if it is opened somehow, it kinda hangs on changing these settings.

jeeped say:
A quick macro would serve you best here even if you decide not to retain it for future use.

Tap Alt+F11 and when the VBA opens, immediately use the pull-down menus to Insert, Module. Paste the following into the new pane titled something like BooK1 - Module1 (Code),

Sub mcr_Fix_Case()
    Dim r As Range, s As String, c As Long
    For Each r In Selection
        s = vbNullString
        For c = 1 To Len(r.Value)
            s = s & IIf(Asc(Mid(r.Value, c, 1)) > 64 And _
                        Asc(Mid(r.Value, c, 1)) < 91, _
                        Chr(32), vbNullString) & Mid(r.Value, c, 1)
        Next c
        s = Trim(s)
        r = Left(s, 1) & Right(LCase(s), Len(s) - 1)
    Next r
End Sub

Tap Alt+Q to return to your worksheet. First select teh cells containing the run-on sentences then tap Alt+F8 to open the Macros dialog and Run the macro. It will convert every cell you selected from this,

IHaveSentencesThatAreAllRunTogetherAndEachNewWordIsACapitalLetter.

... to this,

I have sentences that are all run together and each new word is a capital letter.
