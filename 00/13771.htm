Keep in mind that the data type of B1 and J16 must agree.  That is, if B1 is a number and J16 is text your macro will fail.  If B1 is a legal date but J16 is a text entered date the macro will fail. 

 

So if B1 is 1 and J16 is 1

 

1=1

or

1<>1

 

Depending on whether 1 was entered as text or a number.

   You can use ISTEXT and ISNUMBER to test these.

or you can test =B1=J16

bob jones aka: cybertaz say:
By default, objects inserted into Word documents are inserted as In Line with Text into the paragraph where your insertion point is at the time of insertion. What you describe suggests that you simply have an empty paragraph, so that's where the image goes. As long as it's formatted as In Line it cannot be dragged to anywhere other than another paragraph, so if there are no other paragraphs on the 'page', guess what :-)

Positioning of the image can be done in several ways. One of the most accessible is the Wrapping section that appears in the Formatting Palette when the image is selected. The use of either form of Wrapping converts the image to a floating object rather than In Line. You can then drag the image to any location... the trick is making it stay there :-), especially if you intend to continue entering & revising text once the image is inserted.

Before you get too far along, though, you might want to do some studying on the use of objects in a Word document -- It is one of the more complex aspects of working in a text-based application & can be quite frustrating, even for an experienced veteran. I'm not saying that such a project can't be done in Word, but controlling the position of objects does take a great deal more understanding of how the program works than what most users realize -- primarily because Word works quite differently than how it appears.

You might do better to use Publishing Layout view for a document of this type, but there may well be limitations or constraints -- Not all features of the program can be used in Pub documents. Pub layout provides an object-oriented environment, but you're still dealing with a Word document substructure. It might be a good idea to check it out first using Word Help on Publishing Layout View which includes Videos & Tutorials.

For info re the use of graphics in standard Word documents I'd suggest you start Here:
