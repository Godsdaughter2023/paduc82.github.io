Alt and F11 to open the VBA editor

Select menu item Insert -> Module

Copy the code below and paste into the VBA editor.

=CountColors(B10:C34,B10)

where B10:C34 is the range in which you want to count the colors and B10 is a cell containing a color to be counted.

You can repeat the formula as above for the other colors but change B10 to another cell containing a different color.

Function CountColors(rngToSearch, rngToMatch) As Long

    Dim rngCel As Range
    Dim lngCount As Long

            CountColors = CountColors + 1
        End If
    Next rngCel
   
End Function

Function GetCellColor(xlRange As Range)
    Dim indRow, indColumn As Long
    Dim arResults()

    Application.Volatile

    If xlRange Is Nothing Then
        Set xlRange = Application.ThisCell
    End If

    If xlRange.Count > 1 Then
      ReDim arResults(1 To xlRange.Rows.Count, 1 To xlRange.Columns.Count)
       For indRow = 1 To xlRange.Rows.Count
         For indColumn = 1 To xlRange.Columns.Count
           arResults(indRow, indColumn) = xlRange(indRow, indColumn).Interior.Color
         Next
       Next
     GetCellColor = arResults
    Else
     GetCellColor = xlRange.Interior.Color
    End If
End Function

    Dim cntRes As Long

macropod say:

In arrangements like that, the a) & )b) text would not ordinarily use heading Styles, but another Style with listnumbering. Even so, if you want to apply a particular Style to such text, you could add the following code between the 'Next' and 'End With' in the first macro:

 
  With .Find
    .ClearFormatting
    .Replacement.ClearFormatting
    .Text = "^13^t[a-z]\)(*^13)"
    .Replacement.Text = ""
    .Forward = True
    .Wrap = wdFindStop
    .Format = True
    .MatchWildcards = True
    .Execute
  End With
  Do While .Find.Found = True
    .Start = .Start + 1
    .Collapse wdCollapseStart
    .MoveEndUntil " ", wdForward
    .End = .End + 1
    .Text = vbNullString
    .Style = "Heading 4"
    .Find.Execute
  Loop

Note: the above code assumes the 'a)', for example, is preceded by a tab character and that you'll use Heading 4. If you don't have a tab character, delete the '^t' and, if you want to use something other than "Heading 4", change that to whatever Style you want. As before, if you want to retain the original 'a)', etc, delete '.Text = vbNullString'.

cliff gu msft say:

Hi Max,

Please don’t enable the publishing feature to use the Save site as template feature to in the newly created site.

Based on your description, I’d like to provide some information for your reference.

The symptom of the issue:

After enabling the publishing feature in the SharePoint sites, the Save site as template option is missing in them.

The root cause of the issue:

SharePoint doesn’t support creating a template from a site where publishing or community features were enabled. This is because the publishing feature creates site elements that are not supported as part of a template, and these elements remain even when the feature is disabled. Although the Save site as template option may become available after you deactivate publishing features, it is still unsupported to create a template from a site that has ever had publishing features enabled. 

Solution:

In your environment, the sites have no way to be saved as template sites. You may re-create a site with the same content manually instead. As you have created new sites, please don’t enable the publishing feature to use the Save site as template feature in them.

For your reference: https://support.microsoft.com/en-sg/help/2492356/-save-site-as-template-option-isn-t-available-in-sharepoint-online-in

Thanks,

Cliff
