a. user say:

Hi Dan,

I will be glad to help you with the issue you have with the computer.

Using the built-in Diskpart tool that comes with Windows, you can check whether or not the drive or volume is set to read-only. If it is, you can then use diskpart to remove the read-only property.

1. First open a command prompt and type diskpart. You’ll be brought to the diskpart prompt.

2. Now type in list volume or list disk to see the list of disks and partitions on the computer.

3. Now you have to select a volume or disk to check whether it is read-only or not. If you want to select the C drive (boot drive), then I would type select volume 2.

4. Now type attributes volume to see the various attributes of the volume. One of them is read-only.

Finally, to remove the read-only flag on a drive or volume, type the following command after you have selected the appropriate volume or disk:

attributes volume clear readonly

attributes disk clear readonly
