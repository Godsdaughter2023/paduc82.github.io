jay freedman say:

It says it backs up everything every time. I've got about 1.5 TB of files and would want to keep at least a weeks worth of changes, ideally a month. Does that mean I need 10.5 TB of storage for my File History?

Only if you need to back up the whole 1.5 TB every time. If only a very small subset of that changes in a given week, and you know specific folders where those changes happen, then you can use the backup options page (from the More Options link in the Backup Settings page) to remove from the list any folders that don't change and add specific ones that do change.

Also, File History isn't the only backup you need. At longer intervals (probably weekly or every other week) you should make a full backup or an image backup on an external drive. The external drive should be disconnected from the computer between backups, to prevent lightning strikes or ransomware viruses from reaching it. Most backup software (such as Macrium Reflect) can be told to keep a certain number of previous backups and automatically delete the oldest one.

williamshiflet say:

Creating contact in Skype from my Mac contact list

How can I automatically create a Skype contact list from my Mac laptop contacts. Years ago with Skype it was automatic. can not figure it out. Thanks

a. user say:

Hi WilliamShiflet,

 

Welcome to the Skype Community Forum.

 

With the latest version of Skype for Mac, you have to manually create a list to organize your contacts. When you select a particular list, only contacts within that list will be displayed.

To create a new list:

  1. Sign in to Skype.
  2. In your contact list, find a person you want in a new list. Go to the sidebar and click Contacts, then in the list bar, click Skype. You may have to click >> to see all of your lists.
  3. Right-click the contact and select Add to List….

  1. In the window that opens, click Create List…, then enter a name for your list and press the return key.

  1. A new list appears in the list bar.

Remember: You can change the order of the lists in the list bar. Just click and drag the list to a new position.

You can also create a Favorites section in the sidebar. Go to a contact list, then right-click the contact and select Add to Favorites. This creates a Favorites section at the top of the sidebar, just under Contacts - if there wasn’t one already - and adds the selected contact or conversation. You can add as many contacts or conversations to the Favorites section as you like. They will always appear in the sidebar, so you can easily find them.

Please do not hesitate to reply to this thread if you have any additional questions or concerns.
