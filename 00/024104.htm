daniel g. grau say:

Unfortunately, the Ribbon is not customizable in Word 2011 for Mac. However, you can use to features that would yield a similar result (preview included). These are AutoCorrect and AutoText.

Start by going to your header and selecting whatever elements you want to be in your "header library." Then select Word > Preferences > AutoCorrect. Now you have two courses of action and both work similarly. What are their differences? An AutoCorrect entry needs to be typed in its entirety for Word to replace it. An AutoText entry needs to be typed partially, then Word will offer to insert it.


To use AutoCorrect

  1. Whatever elements you want to be in your "header library" should already be selected
  2. Select the AutoCorrect tab at the top of the dialog
  3. Make sure there is a checkmark at the top, next to "Automatically correct..."
  4. Likewise with "Replace text as you type"
  5. In the blank "Replace:" field, type a unique sequence (for instance, "h1\")
  6. Above the field to the right, select "Formatted text"
  7. Press the "Add" button

Now, back in your document, all you need to do is type h1\ to have Word paste that header. The dialog box you just used to define the AutoCorrect sequence will show a partial preview of the header.

To use AutoText

  1. Whatever elements you want to be in your "header library" should already be selected
  2. Select the AutoText tab at the top of the dialog
  3. Place a checkmark next to "Show AutoComplete tip..."
  4. In the blank field under "Enter AutoText entries here," type a unique sequence or name for the selected header
  5. Press the "Add" button

Now, as you type the header's name, Word should show you the option to insert it. You can also insert it via the main menu's Insert > AutoText > AutoText. As before, the large dialog box will show a preview.
