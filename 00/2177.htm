karl timmermans [outlook mvp 2012-2018] say:

#1 - Don't confuse the size of the PST file with the size constraints of a ANSI format PST file used in Outlook. "Copying" a file from one location to another via Windows Explorer is a completely separate issue and has no bearing on the PST file format type.

 

#2 - If you click on the <root> of your PST file in Outlook --> Properties --> Advanced -->

and the <Format> simply says "Personal Folders File" then it's the current Unicode format and this whole issue of whether to copy versus export is "moot". The old ANSI format which must be increased in size will include Outlook 97-2002 in the format description.

 

#3 - In your case, based on everything described, simply exporting your existing file <properly> should be all you need (properly means making sure that you also ensure to include "all sub-folders" etc). That creates a new PST file in the right format with all your data in it.

 

In short, for most things there generally is usually more than one way to get to the same destination and everyone has their own preferences. If you have a backup of your existing PST file - you can't do any damage so there is no need to over-complicate this. Pick an approach and just follow the steps. If have any questions/run into a problem - reference the "specific step"

 

#4 - As for the "Email tab" question - there are only two steps in the process, picking the email account you want a new destination PST file to apply to and selecting the Inbox folder from within that new PST file. Not sure what more info you are looking for. The line above the "Change Folder" button explains the purpose.

a. user say:

Hello Mohd Natiq,

 

Welcome to the forum. I know that you wish to set your account to not delete messages from Outlook.com server when deleting messages via a mobile. I will be glad to assist you with this question. Please follow the steps below to configure your account to not delete messages (when deleting messages on mobile):

 

1. Sign in to your account.
2. In the upper-right of the window, click the Options icon (gear icon).
3. Click More mail settings.

4. Under Managing your account, click POP and deleting downloaded message.
5. Choose Don't let another program delete messages from Outlook.

6. Click Save.

 

With the settings above, when you delete a message on your phone, it will not be deleted from Outlook.com server, the deleted messages will be moved to POP folder (on web mail account).
