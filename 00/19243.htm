gene g., sherman tx say: I had the same problem and the Shading function was grayed out when I selected the text.  You can in fact use the Shading function in the Paragraph section of the Home Ribbon.  The trick is that you can't include the end-of-line when you select the text that you want to modify. It looks like an extra space past the last character, or the paragraph mark if you turn on the "display symbols" by clicking the paragraph icon.  (Microsoft should fix this!)

Select the text only, click the Shading icon (bucket), select "No color".  Done.
