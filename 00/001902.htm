Command-line arguments for Access

Hello,

I'd like to know if there is a Microsoft Access command-line argument that would allow me to compact the database upon opening.

Thanks

hansv mvp say:

The command line switch /compact will start Access, compact the database, then quit. Example

"C:\Program Files\Microsoft Office\Office11\MSAccess.exe" "C:\Databases\MyDatabase.mdb" /compact

There is no command line switch that will compact the database and leave it open.

Alternatively, you can specify that the database will be compacted when it's closed, in the General tab of Tools | Options. This is a database-level setting.

neha singh msft say:

Hi Laura,

If you are using Onenote 2016, you can change the location of the notebook to sync with your Office 365 account. For this follow the steps below:

  1. Open the notebook in OneNote 2016 which you wish to move to other account.
  2. Right click the notebook in OneNote 2016 and select Properties>Change location.
  3. Once clicked on change location, it will ask you to confirm the new location. At this point, sign in to OneDrive for Business in browser.
  4. Copy the URL, the URL should be https://tenant-my.sharepoint.com/personal/yourname_tenant_onmicrosoft_com/_layouts/15/onedrive.aspx
  5. Remove _layouts/15/onedrive.aspx from the above URL and replace it with documents, the final URL will be https://tenant-my.sharepoint.com/personal/yourname_tenant_onmicrosoft_com/documents  
  6. Copy-paste the URL in change location window and select OK. Now, the notebook will start syncing to the new location.
