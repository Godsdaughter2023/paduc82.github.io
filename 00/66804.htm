Hello Charlotte,

Error code 0xc0000005 can be caused by several reasons. Device driver is probably the most common issue but this can have diverse causes like the inability to run programs as designed or inability to start programs.

Let us resolve this by running the Hardware Troubleshooter and see if it would do the trick. To do this:

  1. Type “Troubleshooting” in the Search bar and press Enter.
  2. Click on “View All” in the top left corner.
  3. This will display the list of troubleshooters which are built-in on the computer. You can click on Hardware and Devices from the list and you can run the troubleshooter.

While it's likely you already have the most recent driver, if you're having trouble with a device, updating your device driver would be the best course to take. This ensures that you'll have the latest fix for bugs and errors.

  1. Select the Start button, type Device Manager, and select it from the list of results.

  2. Expand one of the categories to find the name of your device, then right-click (or tap and hold) it, and select Update Driver. For graphics cards, expand the Display adapters category, right-click your graphics card and select Update Driver.

  3. Select Search automatically for updated driver software.

  4. If Windows doesn't find a new driver, you can try looking for one on the device manufacturer's website and follow their instructions.

If these steps don't work, try reinstalling the driver: Open Device Manager, right-click (or tap and hold) the name of the device, and select Uninstall. Restart your PC and Windows will attempt to reinstall the driver.

    The procedure:

    * Attach the ZIP archive to your post using the "paperclip" (file attachments) button.

    * Ensure that Automatically restart is unchecked.
    * Under the Write Debugging Information header select Small memory dump (256 kB) in the dropdown box (the 256kb varies).
    * Ensure that the Small Dump Directory is listed as %systemroot%\Minidump.
    * OK your way out.
