If the templates are located in the user's (or a workgroup (though that is not to be recommended)) template folder, that is a trusted location and therefore the macros in a particular template will be available for use in any document that is created from that template.

Note, the macros should be created in macro enabled templates (*.dotm) not in a macro enabled document (*.docm)

If you want the macros to be available for use in ALL documents, not just those created from a particular template, the template (*.dotm) containing those macros should be saved in each users Word Startup folder.

The reason that you should not use a central location is because of the difficulty that you will encounter if you need to revise the template(s) as a lock will be placed on it/them if any user has a document open that was created from the template.

Method 1: -

I’d suggest you to start Internet Explorer in No Add-ons mode and check if Internet Explorer is working fine.

Start IE with No Add-Ons

1.       Click on Start.

2.       Click on All Programs.

3.       Click on Accessories.

4.       Click on System Tools.

5.       Click on Internet Explorer (No Add-ons).

Start Internet Explorer and check if you get the same error message, if you do not get the error message one of the Add-on installed on your computer is causing this issue.

To locate the Problem Add-on

1.       Restart IE normally.

2.       Click on Tools.

3.       Click on Manage Add-ons.

4.       Disable the add-ons by clicking them one at a time to highlight them and then click on disable button.

5.       Re-enable the add-ons one by one and check with which Add-on you get this error message. 

6.       Disable the Add-on causing the problem.

 

Method 2: -

Please run online virus scan using Windows Live OneCare from the link mentioned below to scan for any viruses which might have causing this issue, from the link mentioned below.

·         http://onecare.live.com/site/en-us/default.htm

Please see How to get rid of malware to remove any malware and spyware programs causing Internet Explorer to shut down and restart unusually.

 

Method 3: - Perform reset of Internet Explorer settings.

By resetting Internet Explorer settings, you return Internet Explorer to the state it was in when it was first installed on your computer. Resetting Internet Explorer to its default settings does not delete your favorites, feeds, Web Slices, and a few other personalized settings.

To reset Internet Explorer settings

1.       Close any Internet Explorer or Windows Explorer windows that are currently open.

2.       Open Internet Explorer by clicking the Start button. In the search box, type Internet Explorer, and then, in the list of results, click Internet Explorer.

3.       Click the Tools button, and then click Internet Options.

4.       Click the Advanced tab, and then click Reset.

5.       Select the Delete personal settings check box if you would like to remove browsing history, search providers, Accelerators, home pages, and InPrivate Filtering data.

6.       In the Reset Internet Explorer Settings dialog box, click Reset.

7.       When Internet Explorer finishes restoring the settings, click Close, and then click OK.

8.       Close Internet Explorer.

Your changes will take effect the next time you open Internet Explorer.

Important: - Resetting Internet Explorer will delete all the stored passwords and history.
