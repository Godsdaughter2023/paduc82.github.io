marshall barton say:

Layout view is a design feature and should not be used to display data for users.  The Report View is for some kind of use in a web database and acts like a form, not like a report.

So, you need to use Print Preview or Normal.  In these views, subreports with no data display absolutely nothing as you are seeing.  A common design for displaying something when a subreport has no data is to put a text box. label, image or whatever under the subreport control and use the section's Format event to make one or the other visible/invisible:

   Me.subreportcontrol.Visible = Me.subreportcontrol.Report.HasData
   Me.whatever.Visible = Not Me.subreportcontrol.Report.HasData

Hi Ryan131,

You may follow the steps below to disable the preview of attachments in the Outlook client:
1. Open the Outlook client.
2. File->Options->Trust Center, click Trust Center Settings.
3. In Attachment Handling, select Turn off Attachment Preview.
