bernie deitrick say:

I have assumed that you are filling the cell with color rather than changing the font color.

Select a cell that has the color of interest and run this macro, and remember the number displayed.

Sub WhatColorIndex()

    MsgBox ActiveCell.DisplayFormat.Interior.ColorIndex

End Sub

Replace the 14 in this macro with that number, and replace the D2:Z100 with the cells that need counting, and the A2 with the cell that should receive the count, then run CountGreens

Sub CountGreens()

    CountCFColor Range("D2:Z100 "), 14, Range("A2")
End Sub

Sub CountCFColor(rng As Range, intColor As Integer, rngOut As Range)

    Dim r As Range

    Dim iCount As Integer

    For Each r In rng.Cells

        If r.DisplayFormat.Interior.ColorIndex = intColor Then iCount = iCount + 1

    Next r


    rngOut.Value = iCount

End Sub

Sub Inspections()

 

'Petr Bezucha, 2019

 

'Sub marks the dates of last inspections in the B-column (beginning with the 6th row) with

'background colors according to the term categories. The three category boundaries must be

'specified in a decreasing order, together with respective background colors,

'in the cells A2 (i.e. 60, red), A3 (53, yellow) and A4 (0, green - all optional)

'The date block in B-column must be continuous without any non-date value.

'The time decisive dating must be input in the cell B2 before launching the Sub,

'mostly through the worksheet function =TODAY().

'In the run, the pertinent three categories are counted up into cells B2, B3 and B4.

'Attention: The Sub must be launched manually after the change of any data.

 

Dim I As Long, R As Long, Term As Long, Limits(1 To 3) As Single, Colors(1 To 3) As Long

Dim Today As Date

Dim Terms As Range

Today = Range("B2").Value

Set Terms = Range("A3:A5")

For I = 1 To 3

  Limits(I) = Terms(I).Value

  Colors(I) = Terms(I).Interior.Color

  Cells(I + 2, 2).Clear

Next I

For R = 6 To Range("A6").End(xlDown).Row

  Term = Today - Cells(R, 2).Value

  For I = 1 To 3

    If Term >= Limits(I) Then

      Cells(R, 2).Interior.Color = Colors(I)

      Cells(I + 2, 2).Value = Cells(I + 2, 2).Value + 1

      Exit For

    End If

  Next I

Next R

End Sub

Hi Kevin,

If you are unable to receive emails in Outlook.com, the server might be down. Kindly check the Service status if it is up and running. If the status is not green, kindly wait until the service is running normally.

If the service is up and running, but you are still unable to receive any emails, it could be that the emails are received in the Junk Email folder. Kindly check the Junk Email folder in Outlook.com and see if your messages are received there. If there are messages that shouldn't have been marked as junk, kindly right-click each email and then choose Mark as not junk. The message will automatically be moved to your inbox.

If there are no messages in the Junk Email folder, kindly delete older email messages from your Inbox or from the Junk Email folder. If your Inbox is full, you won't receive any new email messages. You may delete all of the emails in your Junk Email folder by right-clicking it and selecting Delete all.

You can also use the Sweep function in Outlook.com to clean up unwanted email from your Inbox or archive folders. Sweep lets you automatically delete all incoming email from a particular sender, keep only the latest message from that sender, or delete email from that sender that's older than 10 days. To do so, kindly select the message, then select Sweep.
