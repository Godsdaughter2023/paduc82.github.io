mike foley say: OK, I finally figured out.  In the Windows\INF directory there is a file called setupapid.app.log.  This file finally gave me a clue as to what the problem was.  In there I noticed a bunch of Access Denied errors on moving files and on creating a registry key.  These errors were being generated whenever i would install the Windows Mobile Device Center Driver (the first step when installing Windows Mobile Device Center).  Without this driver being installed properly, other problems would occur when installing the actual driver for my device.  Because I was already logged in as an administrator, I had UAC completly turned off and still had the problem I decided to change the owner on the registry key (HKLM\Software\Microsoft\Windows NT\CurrentVersion\SvcHost) to Administrators group (from SYSTEM) then uninstalled and reinstalled everything one more time, starting with Windows Mobile Device Center.  Sure enough it worked without any problems.
