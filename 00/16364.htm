I would suspect that the graphics filters are missing
or damaged.

Before proceeding it would be a good idea to create a
System Restore point.

Windows 8 - How To Create a System Restore Point
http://www.groovypost.com/howto/windows-8-7-create-system-restore-recovery-point/

Since they are shared files...a simple reinstall of a
program will not work. First the graphics filters must
be found and renamed. Then a reinstall of a program
that includes graphics filters should replace them.

JPEG filter     - JPEGIM32.FLT
GIF filter          - GIFIMP32.FLT
PNG filter        - PNG32.FLT
TIFF filter        - TIFFIM32.FLT
EPS filter        - EPSimp32.FLT
PCX filter        - PCXIMP32.FLT
BMP filter         - BMPIMP32.FLT   
WMF filter        - WMFIMP32.FLT

Do a search for .flt or try the following path:
C:\Program Files\Common Files\Microsoft Shared\Grphflt

Example...rename them from :
JPEGIM32.FLT  to  JPEGIM32.OLD
GIFIMP32.FLT   to  GIFIMP32.OLD

Now install a program that includes graphics filters...

Try reinstalling PhotoDraw and it that does not work...
installing the freeware IrfanView may be worth a try..

(FWIW...it's always a good idea to create a System
Restore point before installing software or updates)

Download IrfanView
http://majorgeeks.com/IrfanView_d4253.html
(filename  :  iview435_setup.exe)
(uncheck the box if you do not want Google Chrome)

Get the plug-ins too...

IrfanView Plug-ins
http://majorgeeks.com/IrfanView_PlugIns_d4908.html
(filename  :  irfanview_plugins_435_setup.exe)

Good luck...
