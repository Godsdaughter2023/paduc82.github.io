marshall barton say:

A couple of ways to do that, here's a query that I think is the easiest to understand:

SELECT yourtable.Part, yourtable.[Date], yourtable.[Delivery Qty],
        [Starting Stock] - (SELECT Sum(X.[Delivery Qty])
                                  FROM yourtable As X
                                  WHERE  X.[Date] <= yourtable.[Date]) As Balance
FROM yourtable

If you want to do that kind of calculation in a report, you don't have to do it in the report's record source query.  You can use a text box with its RunningSum property instead.

john mcghie say:
Yeah, sorry about that: they moved a few things around in OS 10.7, and again in OS 10.8.

There are three template formats:  Normal.dotm, Normal.dotx, and Normal.dot

The latter two should not exist on your system: Word won't use them, even if they do.

Your Normal.dotm should be where Rich has sent you.  However, it can be literally anywhere on the system.

If in doubt, the easiest way to find it is to ask Word where it is:

1)  Go to Word>Preferences>File Locations

2)  Select User Templates and click Modify

Word will open a Finder window into the folder where the Normal.dotm it is using is located.

3)  Drop down the twin arrows to the right of the folder name.  That will open a drop-down showing you the complete path to the folder in question. 

Normal.dotm is the "Macro-enabled XML format" template that holds all of Word 2011's bits and pieces.

Normal.dotx should never exist: it can contain only text, and thus cannot store all the things Word needs it to store (such as macros, toolbars, auto-text etc).  You can use .dotx for text proformas (similar to Macintosh "Stationery" files), but they can't be used for Normal.

Normal.dot is the obsolete binary format for a template.  It will not store all the modern things Word 2011 can create.  Word 2011 cannot use it as the "Global" (Normal.dotm) template.  It can use .dot files as user templates to store proforma text, but there are some limits to what will work with them, so you should re-save the obsolete .dot templates as .dotm or .dotx templates as you come to them.

Hope this clears it up...
