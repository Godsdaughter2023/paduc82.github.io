bob jones aka: cybertaz say:
Yes, it is quite possible. It involves the use of Section Breaks, though, so if you aren't familiar with them see Word Help on Insert, delete, or change a section break.

One point to be aware of is that some printers do not like to change parameters for margins & orientation in mid stream. They will treat each section as a separate print job. PDF will do the same thing, so the resulting PDF files need to be joined after they're created.
