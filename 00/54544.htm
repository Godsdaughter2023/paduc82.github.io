Phone number update

How do I change/update the phone number of a contact on my list? I am using Skype for Windows desktop.
marites_p say:

Hi

Unfortunately, there is no option to change or update the existing phone number of your contact using the Skype for Desktop. You can only add a new number on their contact details.

To add, will depend on the version you're using. 

For Classic Skype version 7.40: 

Select the contact > right click > view profile > add number. 

For Skype for Windows 10 12.xx:

Select a contact > view profile > click Add number.
