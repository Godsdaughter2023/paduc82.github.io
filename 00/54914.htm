ferdz amp say:

Hi Irvine,

You can recreate thumbnail cache by using Command prompt. Kindly follow these steps to do so:

  1. Press the Windows key to launch the Start menu.
  2. Search for Command Prompt.
  3. Right-click the result and select Run as administrator.
  4. Type the following command to stop File Explorer and press Enter:

    taskkill /f /im explorer.exe
  5. Type the following command to delete all the thumbnail database files without asking for confirmation and press Enter:

    del /f /s /q /a %LocalAppData%\Microsoft\Windows\Explorer\thumbcache_*.db

  6. Type the following command to start File Explorer and press Enter:

    start explorer.exe

  7. Close Command Prompt to complete the task.

If the issue still persists, I recommend that you create a new user with administrator privileges and check if the issue is resolved. Refer to this support article for the steps on how to: Set up accounts.
