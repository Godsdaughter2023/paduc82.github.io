Filter a datefield to a particular day of the week - Microsoft Community

diana van den berg say:

Filter a datefield to a particular day of the week

I have an Access  2007 database with a date field called FromDate that is formatted to ‘dddd, d mmmm yyyy’ and I would like ultimately filter it to a particular day of the week in the form, or have in the query criteria.g. Monday or Tuesday or Friday. 

 

Please can somebody tell me how to do that. 

 

Thank you for any help.

hansv mvp say:

Option 1:

Create a query based on the table.

Add the fields that you need.

Also add a calculated column:

W: Weekday([FromDate])

This column will return 1 for Sunday, 2 for Monday etc. So to filter Mondays, enter 1 in the Criteria row.

 

Option 2:

Create a query based on the table.

Add the fields that you need.

Also add a calculated column:

W: Format([FromDate], "dddd")

This column will return "Sunday", "Monday" etc. So to filter Mondays, enter "Monday" in the Criteria row.
