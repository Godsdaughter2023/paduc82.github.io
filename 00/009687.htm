What you're doing with drag and drop is embedding "Word document objects" in the receiving document (equivalent to using Insert > Object > From file). I am not aware of any option to export an Excel file in ca file format. Your question has been posted for quite a while now with no reply so I am posting the following to see if it helps.

Many applications provide a method of importing a csv file so the following VBA code exports to a CSV file.

From there investigate whatever application you are using that requires ca file type for a method of importing a csv file.

Sub ExportTable()
    Dim wbSource As Workbook       'Source data workbook
    Dim lstObj As ListObject       'Table object
    Dim lngDefaultShts As Long      'Default number of sheets for Excel
    Dim wbDestin As Workbook       'Destination file
    Dim strFileName As String      'CSV Filename
   
    Set wbSource = ThisWorkbook    'Assuming code is in the source workbook
   
    'Following line edit "Sheet1" and "Table1" to your sheet and table name
    Set lstObj = wbSource.Worksheets("Sheet1").ListObjects("Table1")
   
    lngDefaultShts = Application.SheetsInNewWorkbook    'Save current default Number of Sheets
   
    Application.SheetsInNewWorkbook = 1     'Set to One sheet for new workbook
   
    Set wbDestin = Workbooks.Add        'Create New workbook with one sheet
   
    Application.SheetsInNewWorkbook = lngDefaultShts 'Restore to previous default number of sheets
   
    With lstObj.Range
        'Copy values only of the Table to the Destination workbook
        wbDestin.Sheets(1).Cells(1, 1).Resize(.Rows.Count, .Columns.Count).Value = .Value
    End With
   
    strFileName = "ExportedTable.csv"       'Edit to preferred file name for csv file
   
    Application.DisplayAlerts = False   'Suppress message alert re saving as csv file
   
    wbDestin.SaveAs Filename:=strFileName, _
                    FileFormat:=xlCSV, _
                    CreateBackup:=False
   
    Application.DisplayAlerts = True
                   
    wbDestin.Close SaveChanges:=False   'Close the csv file (already saved)
End Sub
