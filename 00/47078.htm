    Hi Willson,

    With regards to your lock screen concern, we suggest that you follow the steps provided below on how to enable your lock screen:

    1. Right click the Start button, run or press the Windows + R keys (run).
    2. Type regedit then, press Enter or click OK.
    3. Navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\Personalization.

    5. Reboot your PC to take effect.

    Difference Between a Full/﻿Upgrade/Retail License and an OEM/System Builder License.

    OEM licenses of Windows:

    1) Are bound to the first computer installed on and can't be moved to a different computer.

    2) Can not do an in place upgrade from older versions of Windows, only a clean/fresh install is possible.

    3) Offer no free support from Microsoft.

    andre da costa say:
    There is nothing preventing you from using an OEM System Builder Windows License in a virtual machine on a Mac or any other computers, it just that there are differences in how it is licensed.

    OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

    - OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

    - OEM licenses are tied to the very first computer you install and activate it on

    - OEM versions allow all hardware upgrades except for an upgrade to a different model motherboard

    - OEM versions cannot be used to directly upgrade from an older Windows operating system
