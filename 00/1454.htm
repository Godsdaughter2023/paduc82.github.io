You have two distinct problems to solve:

1. Permissions

2. Error in Visual Basic code

1. You most likely encountered the permissions requests when you updated Mac OS. If you use the Documents folder as the storage location, you should not be prompted for permissions.

Then quit Word. If prompted whether or not to save changes to Normal, save the changes to make the deletion permanent.

If there is a module that you actually expect to be in Normal, then you'll have to fix the code in the module to get rid of the error message.

ross jaburg say:

Before you begin the technical preview ...

Since a lot of people are (apparently) ignoring MS' warnings about only doing this if you have advanced skills and NOT using your main (or only) PC, etc., etc., I figured I would remind everyone that there is an easy way to protect yourself and get back to your previous system as it was before you installed this. And the best part is, it is built in to Windows (since Vista) and is absolutely free. (Note: the engine is there, but the utility is strangely absent from W10.)

Background: I am an IT professional with 30+ years of experience and have spent the last 18 of them working for and managing a company that does nothing but ensure companies protect and secure their data with offsite backups so that they can recover from any loss.

The last thing anyone should do before delving into this is to make a recovery image of their existing system on an external hard drive.

Use the Windows Backup And Restore functions to create a system image (could take a while) and create a recovery disk while you're at it(same menu).

That will give you a "Bare Metal" recovery that works every time.

For recovery, you can use your original Windows installation disk too, but a lot of people seem to "lose" those.

When you are finished playing with W10, you will be able to boot to the recovery disk (with external drive and image attached) and enter the recovery console to put your machine back to the way it was when you created the image. It will wipe the disk and replace the system with your old one.

Obviously, any documents or other files you have created while previewing 10 will need to have been saved somewhere outside of the system (OneDrive maybe?).
