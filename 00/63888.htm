cmdrkeene say:

It's possible -- and much more likely -- that you have told your Picture Library to display files from your Documents folder.

First, please note that libraries do not actually hold any files.  A library simply aggregates (collects) and displays files from many different places and puts them all into one view.  It's like a saved, instant search that automatically shows all the files.  Libraries are useful if you have pictures in several places, such as other hard drives or network locations, that you want to all be bunched into a single view for easy browsing.

To modify where the Pictures Library collects files from, right-click the Pictures Library icon (or link in your Start menu) and choose Properties.  Remove locations you don't want to have included in the library.  Don't worry -- you aren't removing any files or folders -- you're simply telling them not to show up when you open this library.

How to Perform a Clean Boot? (Applies to Windows 10)
