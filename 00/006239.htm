Yes, the most recent date of completion for each staff member.

SELECT Username, MAX([Date of Completion]) As LatestCompletionDate
FROM (SELECT Username, [Date of Completion]
            FROM [Table 1]
            UNION ALL
            SELECT Username, [Date of Completion]
            FROM [Table 2])
GROUP BY Username;

But I would reiterate Scott's original point that having two separate tables is bad relational database design. andreas killer say:
AFAIK all WorkSheetFunctions did not work with 3D arrays, so you have to write your own properties, that is IMHO also an elegant approach. See the example below for 1D/2D arrays.

Andreas.

Private Sub Example_ArrRowColumn()
  Dim Data(1 To 3, 1 To 2) 'As String  'Data could be an array of any type!
  Dim Arr
 
  'Write 1d array into 2d array
  ArrRow(Data, 1) = Array("Number", "Value")
  ArrRow(Data, 2) = Array(1, "a")
  ArrRow(Data, 3) = Array(2, "b")
 
  'Read a column
  Arr = ArrColumn(Data, 2)
  Debug.Print "Second column: "; Join(Arr, " ")
  'Read a row
  Arr = ArrRow(Data, 2)
  Debug.Print "Second row: "; Join(Arr, " ")
 
  'Write a string array
  Arr = Split("Data,x,y", ",")
  ArrColumn(Data, 2) = Arr
 
  Arr = ArrColumn(Data, 2)
  Debug.Print "Second column: "; Join(Arr, " ")
End Sub

Property Get ArrRow(Arr, ByVal Nth As Long) As Variant
  'Return the Nth row of a 2 dimensional array as one dimensional array
  Dim Result(), i As Long
  On Error GoTo ExitPoint
  ReDim Result(LBound(Arr, 2) To UBound(Arr, 2))
  For i = LBound(Arr, 2) To UBound(Arr, 2)
    Result(i) = Arr(Nth, i)
  Next
ExitPoint:
  ArrRow = Result
End Property

Property Let ArrRow(Arr, ByVal Nth As Long, Values)
  'Writes a one dimensional array into the Nth row of a two dimensional array
  Dim i As Long, j As Long
  On Error GoTo ExitPoint
  j = LBound(Arr, 2)
  For i = LBound(Values) To UBound(Values)
    Arr(Nth, j) = Values(i)
    j = j + 1
  Next
ExitPoint:
End Property

Property Get ArrColumn(Arr, ByVal Nth As Long) As Variant
  'Return the Nth column of a 2 dimensional array as one dimensional array
  Dim Result(), i As Long
  On Error GoTo ExitPoint
  ReDim Result(LBound(Arr) To UBound(Arr))
  For i = LBound(Arr) To UBound(Arr)
    Result(i) = Arr(i, Nth)
  Next
ExitPoint:
  ArrColumn = Result
End Property

Property Let ArrColumn(Arr, ByVal Nth As Long, Values)
  'Writes a one dimensional array into the Nth column of a two dimensional array
  Dim i As Long, j As Long
  On Error GoTo ExitPoint
  j = LBound(Arr)
  For i = LBound(Values) To UBound(Values)
    Arr(j, Nth) = Values(i)
    j = j + 1
  Next
ExitPoint:
End Property

xircal say:

Hi,

Download the free version of Malwarebytes (orange download button) and run a full scan. https://www.malwarebytes.org/

This step is primarily a precaution in case you have malware  installed which is preventing you from re-enabling WD.

After the scan finishes and anything found has been removed (you'll need to reboot if it is), do the following.

  1. Hit Windows key + X and click "Command Prompt (Admin)".
  2. Copy and then right click the following command to paste it into the command prompt and hit Enter:

DISM.exe/Online /Cleanup-image /Restorehealth

DISM doesn't display very much when it's running and all you'll see most of the time is a plain black screen. But eventually, the figure 20% will appear followed about 15 mins later by 100%. You'll also get a report of what was fixed.

If you see any error messages, please make a note of them and post those here.

felix tao msft say:

Hi David,

Please note that when an item is labeled as a record, four things happen:

  • The item can’t be permanently deleted.
  • The item can’t be edited.
  • The label can’t be changed.
  • The label can’t be removed.
     

For your situation, the item may not be a record. You may check the details in the Using labels for records management  part in this article.

For Preservation Hold library,when you include a SharePoint site or OneDrive account in a retention policy, a Preservation Hold library is created, if one doesn’t already exist. If a person attempts to change or delete content in a site that’s subject to a retention policy, first the policy checks whether the content’s been changed since the policy was applied. If this is the first change since the policy was applied, the retention policy copies the content to the Preservation Hold library, and then allows the person to change or delete the original content. Note that any content in the site can be copied to the Preservation Hold library, even if the content does not match the query used by the retention policy.
 

For details, please refer to the following article:

Overview of retention policies

Regards,
Felix
