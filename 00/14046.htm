suzanne s. barnhill say:

So there are two hyphens here, the optional hyphen inserted by Word when the word is divided and the nonbreaking hyphen at the end of "documents" that is connecting it to the next word.

 

In this situation, I can offer only two suggestions:

  1. At least in English, it is generally recommended that hyphenated words be split only at the hyphen. That means that, if the entire word "document-" can't fit on the first line, then the entire compound ("document-something") should be carried over to the next line.
  2. If Word is (not surprisingly) not understanding this, you can still prevent the word from being divided (while still keeping automatic hyphenation enabled) by formatting it as "Do not check spelling or grammar," which also suppresses hyphenation.

There are actually two ways to prevent the unwanted hyphenation. If there are no other instances of end-of-line hyphenation in the paragraph, a simple approach is just to check the "Don't hyphenate" box on the Line and Page Breaks tab of the Paragraph dialog for just that paragraph.

 

If this is not feasible, then you can select just the single word (actually you'll probably find you have to select the entire combination, both sides of the hyphen) and format it as "Do not check spelling or grammar." If you find you have to do this often, it will be worth your time to define a No Proofing character style that you can quickly apply to specific words. For more on this, see http://WordFAQs.mvps.org/MasterSpellCheck.htm#ExemptingText.

 

None of this solves the essential problem of incorrect printing/conversion to XPS, but at least it will provide a way to avoid the situation.

sheen valentine ali msft say:

Hi Tanya,

You can modify the registry manually if you would like to do it on a computer by computer instance.

You can refer to the following steps to disable personal sync using the Registry Editor:

1. Press Windows + R on your keyboard > In the text box, enter regedit > OK to open the Registry Editor.

2. Navigate to the following key:

HKEY_CURRENT_USER\SOFTWARE\Microsoft\OneDrive.

3. In this key, add a DWORD Value and name it DisablePersonalSync.

4. Modify the value to 1.

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall your operating system. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Edit at your own risk.

For the newer ISO standard:
