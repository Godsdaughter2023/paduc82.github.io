wilfredo_soto say:
Hi MarciaCrossland
I'm Wilfredo S an Independent Advisor

Repair damaged Recycle Bin in Windows

Open command prompt with elevated or administrative rights. To do so, you can right click on the Start button and choose the System Symbol (administrator) option.

Once inside the command prompt, type the command
rd /s /q C:\$Recycle.bin
and press the Enter key.

Restart the system. Once this is done, the folder of the Recycle Bin and all the files and folders contained in it will be deleted. On restarting the computer, Windows will re-create a new Recycle Bin.

I hope it helps you
