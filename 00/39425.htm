a. user say:

A clean install by definition overwrites whatever is currently on the drive, so anything that isn't backed up will be lost.

Windows includes hundreds (probably thousands) of drivers that it automatically installs. These are known as 'native drivers.' It's possible that Windows 10 has native drivers for all your computer's hardware.

If Windows doesn't have a native driver you can install a driver from the hardware manufacturer's website. These are known as third-party drivers. You can install a third-party driver for your hardware even if Windows has a native driver.

If you're thinking of a clean install, you can always go to the hardware manufacturers' websites in advance and download the driver installers. Store them on a USB flash drive, along with a text file that explains what each driver is and how to install it. That's what I did for my clean install.

My own USB flash drive has all the drivers (and firmware) I've ever installed, with complete information about each one. I never have to worry about where to find drivers or what to do with them. It's all there waiting for me.

Unless the two RAID controllers are 100% hardware compatible your system will NOT boot up, I guarantee it.  The RAID drivers are built into the boot files that start Windows and cannot be repaired or replaced short of a reinstall of Windows.  At best you'll get a blue screen and a reboot that never ends, at worst your data will be scrambled and lost forever.  A middle case would be a computer that starts up but is never completely stable again because it contains both drivers for hardware that's present and drivers for hardware that no longer exists.

Hi Marty,

Thank you for posting the query on Microsoft Community. I am glad to assist you on this.

This issue might occur due to corrupted or outdated display drivers.

I would suggest you to uninstall the current driver from device manager and install the latest display drivers for Windows 10 from manufacturer website and check if this helps.

  • Press Windows Key + X key and click Device Manager.  
  • Expand ‘Display Adapters’. 
  • Right-click on the Display Adapter listed and click on 'Uninstall'.
  • Now install the latest driver from the manufacturer website.

If there no drivers for Windows 10, you may download the drivers for lover version of Windows (Windows8/8.1) and install it in a compatibility mode.

Compatibility mode runs the program using settings from a previous version of Windows. Follow these steps to install the drivers in compatibility mode and check if that helps.

a. Download the driver from the manufacturer’s website.

b. Right-click on the driver setup file and click on ‘properties’.

c. Click on the ‘compatibility’ tab and check the box ‘Run this program in compatibility mode for’ and select Windows 8/8.1 operating system from the drop down menu and proceed with the installation.

d. Once this is done, restart the computer and check if the issue persists.
