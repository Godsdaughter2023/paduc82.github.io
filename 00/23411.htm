suzanne s. barnhill say:

Keep in mind that "chapter" is not the same thing as "section." If you have a section break within the chapter (to change page orientation or number of columns, for example), then numbering set to restart in each section will restart. I was convinced that there was no way to achieve what you want, but I seem to have had success with this procedure:

 

In the Footnote and Endnote dialog, set "Restart each section" for "Whole document." In each section that is not the first section in a chapter, set it to "Continuous" for "This section." In both cases, the starting number will be 1.

 

I was really surprised to discover this, as the sharpest minds in the Word community have struggled with this and, I thought, failed.

table split onto next page acts like a separate table - Microsoft Community

word challenged say:

table split onto next page acts like a separate table

I have a table that flows to the next page, which is what I want, but the part of the table that flows onto the next page is being treated as a completely separate table. What I change or apply to what I think is the entire table, only affects the part of the table on the first page, not the second page.
suzanne s. barnhill say:

Did you insert a manual page break in the table? If so, you actually have split it. Delete the page break and instead format the row you want to be at the top of the next page as "Page break before." To do this, select the row (or any text in it), right-click, choose Paragraph, select the Line and Page Breaks tab, and check the box for "Page break before."

If that's not the problem, make sure there's not an empty paragraph between the two tables (this will probably be easier to see in Draft view, with nonprinting characters ("formatting marks") displayed). Also make sure that both parts of the table are inline: on the Table tab of Table Properties, make sure text wrapping is set to None.

If your table has a repeating heading row that appears on the second page, you should not be able to select and edit it, as it is just "virtual" everywhere but on the first page. If that's the case, then at least you don't have a manual page break separating the parts of the table, but if you can edit the row, then it is an actual table row and may have been marked as a heading row, which will also cause the table to split. (A given table cannot have any rows marked as heading rows except the top row and any number of contiguous rows below it.)

cjlim say: Battery life is difficult to compare because it depends on usage, coverage signal strength etc. etc.

I have a friend who gets 4-5 days with his E51 while my own experience is more like 3-5 days. However, you can extend battery life by doing the following.

There are a few steps you can take to enable you to squeeze a little more standby time from your battery. These include:

1. Switch the phone from Dual Mode to GSM only
2. Turn off bluetooth when not required
3. Turn off Wlan scanning when not required
4. Reduce the backlight power saver time
5. Reduce backlight brightness
6. Exit any applications when not in use
