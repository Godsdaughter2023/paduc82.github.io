auggy say:

There is little doubt that the lpx.sys is the driver that caused the crash.

There is no doubt that the ndassvc.exe is the process that crashed.

You may be able to prevent the error by "stopping" the NDAS service prior to connecting the TomTom device:

Start > type services.msc in the "Search programs and files" box and press "Enter" > then, in the right window, scroll down to the "NDAS service" and right-click the "NDAS service" > Stop

Connect the TomTom device

Open services.msc again, right-click the "NDAS service" > Start

In any event I would still recommend to update to the latest version of the NDAS software unless you have a particular issue with that version.

bruce hagen ~ mvp say:
Are these PDF files an e-mail attachment? What e-mail program. If not an attachment, skip #1.
 
Create a file association.
 
  1.     Download and save the attachment on the desktop.
  2.     Right click on the saved file.
  3.     Select: Open With from the pop-up menu.
  4.     Click: Choose [default] Program.
  5.     Select: Adobe Reader.
  6.     Check: Always use the selected program to open this kind of file.
  7.     Click OK.
 
This should enable you to open these attachments directly from the e-mail now.
