    Folders in Outlook aren't folders as Windows folders are.  They are merely keys in a database.  Create the folders manually in Windows Explorer and then drag the messages from Outlook to the Windows folder.  Keep in mind, however, you'll still need to use Outlook to open and read those message files.  Since this is the case, I'd recommend creating new Outlook Data Files in the Windows folders, moving your related messages into the data files, then closing the data files in Outlook to disconnect them until you need them again.
