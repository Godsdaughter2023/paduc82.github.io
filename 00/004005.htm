suzanne s. barnhill say:
The default top margin for documents in Word is one inch. The default header margin is half an inch, allowing half an inch of header content before the header begins to push the top margin down. If you have multiple lines (or a graphic) in the header, you may need to set a smaller header margin or a larger top margin. Keep in mind that your header may contain extra empty paragraphs (display nonprinting characters in order to see them), or (less likely) the header could have Spacing After.

So the task here is to re-name a windows file. As noted, windows by default does hide file extensions. So to change the file extension, you have to turn on display of windows file extensions.

For windows 7, do this:

http://www.pcsnippets.com/tutorials/how-to-turn-on-file-extensions-in-windows-7/8

for windows 10:
http://www.ilovefreesoftware.com/06/windows-10/show-file-name-extensions-windows-10.html

As noted, one of the FIRST things I do with any PC I going to use for software development is turn on file extensions. So even for renaming .dif files to .txt or “many” other basic computer skills – the goal here is the ability to re-name a windows file.

So the goal and task here is to re-name a file on your computer - it not a feature related to Access.

If you want to show folders structure in metadata navigation in document library, you must choose one of the following column types as navigation hierarchies:

If you don’t select an extra column, it won’t show folders structure in metadata navigation in document library.
