This is an old question but worth publishing an answer in case it's useful to someone. I've encountered this a couple of times and both times have been able to resolve (without having to re-install or re-download email) by doing this:

- Close Outlook

- Create a new, blank Outlook profile and set it as the Default profile.

- Launch Outlook 2016. Then quit it again.

- Set your real profile as the Default profile again.

- Relaunch Outlook 2016

- The problem should be cleared.

If you're not sure how to manage profiles in Outlook 2016 for Mac, it's done by using the Outlook Profile Manager which is hidden inside the Outlook application package. To open the Outlook Profile Manager:

- From Finder, open the Applications folder.
- Ctrl+click or right-click Microsoft Outlook, and then click Show Package Contents.
- Open Contents > SharedSupport, and then double-click Outlook Profile Manager.

ron sommer mvp 2009-2017 say:

           
You can select multiple emails by holding SHIFT and clicking on them.

To select multiple emails, hold Ctrl and click on emails.

To select several adjacent emails, check on email, hold Shift key, click another email.

All emails between the two emails will be selected.

Another method to select several adjacent emails is to check an email, hold Shift key, press the Up or Down arrow on the keyboard.

To unselect a checked email and leave other emails checked, hold the Ctrl key and click on an email.
