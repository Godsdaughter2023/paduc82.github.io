chin8779 say:

Finally I found the solution... you can reset all user-configured settings using this command (use the elevated mode/run as Administrator):

          netsh int ip reset

Then do a restart of your computer for the default-settings to take effect... although you can also immediately see the user-configured APIPA is gone before the computer restart.
