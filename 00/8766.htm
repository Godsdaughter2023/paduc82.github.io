Numbers are usually represented internally using the standard 64-bit binary floating-point.   And most non-integers cannot be represented exactly.

In particular, the number 0.07 is represented internally by exactly 0.0700000000000000,06661338147750939242541790008544921875.

#include "stdafx.h"

}

jordan dom say:

Hi Jason,

By design, workbooks that are shared would have the delete options "shift cells left" and "shift cells up" greyed out. Inserting cells also have the same options greyed out. This behavior is used to assist in tracking the changes in the history. For example, inserting one row creates one entry in the change history.

However, to move one cell down one row requires one entry for the new cell value when it is populated but also two entries for the cell below that to store the original value (before the change) and the new value (after the change). Consider this on a larger scale and the change history would become massive very quickly.

rich007a say:

There are two types of Checkbox control available: Form Control and ActiveX.  The Form Control checkbox does not have the ability to be Moved and Sized with cells.  The ActiveX checkbox control does.

In the Developer tab, Controls group, click Insert and then choose the Checkbox control under ActiveX Controls.
Click in your worksheet at the desired location.
Right-click on the control > Format Control > Properties tab > Move and size with cells > OK.

To be able to use the checkbox, you need to switch off Design Mode (next to Insert on the devleoper tab).

Tip: to be able to snap the size of the control to be the same as the cell, insert a shape (Insert tab > Shapes > Rectange...) and then in the Drawing Tools > Format tab, in the Arrange group click Align > Snap to Grid.  You can then resize the checkbox control so it exactly fits in a cell.

Hope that helps.

danida_u say:

Hi

Question: How do I indicate to start Skype on startup.

Answer: There are two available Skype platforms for Windows 10 users. If you are using:

  • Skype for Windows 10 version 12.13 - This version of the app does not have the option to automatically start the application on start up or when Windows start, but you will be automatically signed in even if you reboot the device if, and only if you do not log out from your account.

  • Classic Skype for Windows desktop version 7.40: You can enable automatic start up when Windows start from this version of the application. Click Tools and then choose Options. Under General settings, tick the box for Start Skype when I start Windows.

Question: Also, where are the camera controls in this version (turn off auto focus for example?

Answer: Skype for Windows 10 does not have a detailed settings for audio and video as this version of the application uses the native settings of the device. If you need to make any changes for your video settings, you need to configure if from the system settings itself, while you can configure your video settings on the classic Skype for Windows desktop by clicking Tools then Options. Choose Video settings and make your changes.
