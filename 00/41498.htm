richardeiler say:

Hi,

If the drive is not self powered and functions properly when plugged into a self powered USB hub and/or other devices (PC's) but not when directly plugged into the USB port of the device in question this would indicate a probability that there is an excessive voltage drop problem in the 5 volt  supply leg of the devices power supply unit (PSU). Also take into consideration how many other non-self powered USB devices are pugged into the PC (other USB ports) at the same time and also if you are dealing with USB 2.0 (5v @ .5amp spec) or USB 3.0 (5v @ .9 amp spec).

Also you can check your Power setting in Control Panel, if it is set at Power Saver you can try changing the plan to Balanced or if at Balanced to Performance. If there is still no difference and the above description of the problems symptoms are a match, the PSU is suspect. It is very common for a PSU to start failing on a single leg output before and or opposed to a complete failure all at once.
