Where is the dictionary for Word 2011 on OS 10.6? - Microsoft Community

laurilt say:

Where is the dictionary for Word 2011 on OS 10.6?

I am trying to move all my custom settings etc. from Word 2011 on and older IMAC (0S 10.6) to a new IMAC (OS10.8).
I can't find the dictionary on the older mac (10.6). It is not in the Office 2011 preferences folder.

john mcghie say:
It's not the "Dictionary" you need, it's the "Custom Dictionary".  The "Dictionary" will be installed when you install Office 2011.

You should not attempt to migrate settings from the old machine to the new: Preferences are machine-specific, you can end up with a lot of grief if you migrate the preferences.  Especially between OS 10.6 and OS 10.8, because the later OS has different preferences that are stored in a different location.

Instead, take across only your Normal.dotm template, and your Custom Dictionary.  Two files.  That will take across the vast majority of your settings and customisations, without getting you into trouble.

Each of these files "can" be in "any" location, anywhere on the machine.  The most reliable way of finding them is to ask Word (on the old machine) where they are!  There may be multiple copies on a machine, by asking Word, you will be directed to the one that is really being used.

Go to Word>Preferences>File Locations.

Highlight "User Templates" and click "Modify".  This will open a dialog and Normal.dotm will appear in it.  That's the file you want.  At the top of that dialog, click the blue double-headed arrow disclosure widget to the right of the folder name.  That will show you the complete path to where the file is.

Now go to Word>Preferences>Spelling and Grammar.  Click the "Dictionaries" button, and then click "Add".  Again, this will open the folder your custom dictionary is in (it doesn't have to be called "custom dictionary", but it is the file with the ".dic" extension.  Again, drop down the disclosure arrows to get the full path.

Copy those two file over, and no others, and Word will run sweet and clean on the new machine.

Do not forget to run Help>Check for Updates ... some updates are essential to get Word running in OS 10.8.

Hope this helps

stefan blom say:
In a given document, you can prevent Word from tracking formatting changes, if that is what you want. Click the dialog launcher in the Tracking group on the Review tab and then click Advanced Options. In the Advanced Track Changes Options dialog box, clear the option "Track formatting." Note that this setting saves with the document (or template) where you set it.

ken sheridan say: I'd assumed that you were using a VBA event procedure not a macro.  The Me keyword can only be used in VBA, not in a macro, and refers to the current form (or more strictly speaking to the instance of the class in which the code is executing).  I very rarely use macros, so am not sure whether the behaviour is the same when a form is opened in dialogue mode via a macro as it is in VBA, though my guess would be that it is.  Try simply calling the Requery action in your macro without the Me keyword.  Be sure that the WindowMode argument of the OpenForm action which will precede the Requery action in the macro is set to Dialog.
