1.      Open Internet Explorer

2.      Click Tools

3.      Click Manage Add-ons

4.      Click the first disabled add-on

5.      Click Enable

6.      Click Close

7.      Surf a few pages

8.      Close Internet Explorer

9.      Open Internet Explorer

DEP is a security feature that can help prevent damage to your computer from viruses and other security threats. They are genuine Microsoft product keys for retail licenses, but not for Retail sale. They are intended for a particular product channel either the Microsoft Software Developer Network (MSDN) or TechNet for IT Professionals who pay a subscription fee. Hi Doug,

 

Had to go the route of using Win32 functions to get and set the default printer (so that I could continue using a PDF product that supported setting security).  In case someone else comes upon this thread, there are the functions I used.  They are written to be called from Visual Foxpro (VFP) but you should be able to find examples of how to call them from other languages.

 

>>>>>>>>>> Set the default printer >>>>>>>>>>>>>

 

* declare the call

 

* (note: printer name passed in as tcPrinterName prior to this)

 

DECLARE LONG

SetDefaultPrinter IN WinSpool.drv STRING pPrinterName

 

lnResult = 

SetDefaultPrinter(tcPrinterName)

 

* lnResult, 0 = error, 1 = success; set a flag if error occcurred

IF

lnResult = 0    STORE .T. TO llErrorSettingPrinter

ENDIF

 

>>>>>>>>>>> Get default printer >>>>>>>>>>>>>

 

* I used this to query the current printer - instead of changing it for every call, I would query and only

* change if needed

　

DECLARE INTEGER

GetDefaultPrinter IN winspool.drv STRING @pszBuffer, INTEGER @pcchBuffer

 

* stuff buffer string with nulls, lots of them

STORE REPLICATE(CHR(0),250) TO lcCurrentPrinter

 

lnResult = 

GetDefaultPrinter(@lcCurrentPrinter,250)

 

* if error occurred, result is zero; otherwise, non-zero

IF

lnResult = 0    STORE .T. TO llGeneralError

ENDIF

 

* otherwise, need to pull out the printer name up to the first null in the string
