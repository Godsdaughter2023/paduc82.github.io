guruprasad ra say:

By exporting the address book in outlook as .PST file it can be easily transferred to a different computer. Follow the instruction below to export the address book.

1.    Open Outlook.

2.    On the File menu, click Import and Export. And then click Import and Export.

3.    Click Export to File, and then click next.

4.    Click Personal Folder File (.pst), and then click next.

5.    Click the Contact folder to export as the .pst file to and then click next.

6.    Click Browse, and then select the location where you want the new .pst file to be saved.

7.    In the File Name box, type the name that you want to use for the new .pst file, and then click OK.

8.    Click Finish

Save the .pst file to an external hard drive or to a disc.

TO import the .pst file in to the second computer follow the instructions.

1.    Start Outlook 2007.

2.    Click Import and Export on the File menu.

3.    Click Import from another program or file, and then click next.

4.    Click Personal Folder (.PST), and then click next.

5.    Click Browse, locate the Personal Address Book that you want to import, and then click OK.

6.    Click Next.

7.    Click Finish to import the Address Book.

Excel Custom Number format for credit card - Microsoft Community

froggygremblin say:

Excel Custom Number format for credit card

I'm trying to create a custom number format for a credit card number.  I've tried #### ##### #### #### and 0000 0000 0000 0000 0000 and it almost works.  My problem is when I enter the 16 digit account number it converts the last digit to zero. Not just for display the actual number in the cell changes.  Can this be done?
joeu2004 say:

froggygremblin wrote:

I'm trying to create a custom number format for a credit card number.  I've tried #### ##### #### #### and 0000 0000 0000 0000 0000 and it almost works.  My problem is when I enter the 16 digit account number it converts the last digit to zero. Not just for display the actual number in the cell changes.  Can this be done?

No.  For data entry, Excel interprets only the first 15 significant digits (i.e. not including leading zeros), replacing any digits to the right with zero, as you noticed.

You must enter such "numbers" as text, either by formatting the cell as Text before data entry, or by typing an apostrophe (aka single-quote) before the "number".

[EDIT] I forgot to mention....  If the credit card "numbers" come from a text file, you can use the import text "wizard" to treat columns of imported data as text.

In any case, numeric formats do not work on text.  And there are no text formats to speak of -- at least, no way to insert spaces between character positions.

(Unless a text format feature was added after Excel 2010.  I don't think so.  But I don't know.)

You could implement an event macro that modifies the cell value (text) after data entry.  Is that worth considering?
