1) Copy this code.
2) Right-Click the sheet tab of interest.
3) Select "View Code"
4) Paste the code into the window that appears.
5) Save the file as a macro-enabled .xlsm file.
6) Make changes as needed

Private Sub Worksheet_Change(ByVal Target As Range)
    If Target.Address <> "$L$5" Then Exit Sub
    Dim v As Variant
    
    v = Application.Match(Target.Value, Range("A:A"), False)
    
    If Not IsError(v) Then
        Intersect(Me.Rows(v), Me.UsedRange).Font.Strikethrough = True
    End If
End Sub

mike h.. say:

Hi,

You need a macro for that.

ALT+F11 to open Vb editor, right click ThisWorkbook and insert module and paste this in and run it when required

Sub Sonic()
Dim NumCopies As Long, x As Long
NumCopies = Sheets("sheet1").Range("A2").Value
For x = 1 To 2
 Sheets(x).PrintOut Copies:=NumCopies
Next
