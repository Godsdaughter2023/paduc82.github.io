chris - support engineer say:

Hi Staite,

If you've unplugged the external hard drive, it can sometimes install a second hardware profile and take a new drive letter.

You might try plugging the drive back into the the same USB slot, or removing it and returning it to the computer while the computer is turned off.

You may be able to remove the unused profile from the Device Manager by clicking View > Show hidden devices.

You may have to remove both profiles from the Device Manager and restart the computer to get the original drive letter assigned to the device, if it's available.
