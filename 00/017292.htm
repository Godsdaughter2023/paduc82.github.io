stefan blom say:

Create two paragraph styles—one for bulleted and one for numbered paragraphs. Actually, you can use the built-in List Number and List Bullet styles as starting points.

You can modify bullet and number formats via the Modify Style dialog box. To modify indents, create an example paragraph in the style you want to modify, click to select the number (bullet) symbol; then right-click and choose Adjust List Indents from the context menu. Specify the indentation you want and click OK. When you are done, update the style to match selection (which you can do from the Styles pane).

Note that if you are trying to have a style applied as you click the Bullet button and another when you click the Numbering button, that isn't going to work. You will have to apply the styles directly, the way you apply (for example) heading styles to text.
