Hi Maribel Mendiola,

In Outlook client on PC, you can change both the default view or the current view of the calendar. The detailed methods please check https://support.office.com/en-us/article/Change-the-appearance-of-your-calendar-84efcbbb-52a5-4aab-9fbc-7e0f0520bb42 and the specific screenshots as below.




And if you are using Outlook for MAC.

Categorize an item:

On the Organize tab, click Categories.

If you are not seeing the category you would like, you will need to create a new category.

Modify Existing Categories or Create New Categories:

Assign a Color to each Item Within a Calendar :

1.Select Calendar from the bottom-left of your Outlook client.
2.Create a new event or modify an existing event, using the steps above.
3.Click on Categories.
4.Select the color category you wish to assign to the calendar event.

And in OWA( Outlook Web App), you can change it by right click the specific calendar then select color, choose the one you want.

tom ogilvy say:

this custom number format will do what you described:

00\/00\/0000

it requires 8 digit number

if does not make the number an excel date - it only makes it look like a date.

I suspect that isn't what you want.  I suspect you want to have excel convert it to an actual date serial number and format it as date.

that would require a macro.  Chip Pearson has written a macro that can do that.

http://www.cpearson.com/excel/datetimeentry.htm

ADDED:  For clarification, my answer assumes if you make your entry in cell A1, you want the formatting or actual conversion done in A1.  Or more generally for a select group of cells, any cell where you are making the entry you want the formatting or actual conversion done to that value in that cell.
