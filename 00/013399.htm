Sorry, I have to clarify this again first:

There is no "main" OneNote. Your notes are stored on OneDrive and any Client, no matter if PC, notebook, phone or tablet, iOS, windows or Android ist syncing with the notes on OneDrive

No device is syncing with another one. Not your iPad with your destop nor your iPhone with your iPad. All just sync against OneDrive.

There is no need for a OneDrive app on any device to use and sync OneNote. You just have to sign in on OneNote using your MS credentials. You should use the same MS account on every device.

There are currently 8 different OneNote versions (e.g. iOS apps, Android, Windows 10 Universal App, OneNote 2013 or 2016 for Windows desktop, OneNote for Mac...). ALL of them can only use (sync with) notes which are stored on OneDrive or OneDrive for business. Some may in addition access notes on SharePoint. 

But there is only ONE OneNote version which could also access notes which are stored locally on the device itself. This is OneNote 2010/2013/2016 for Windows, but ONLY the version that comes with MS Office, not the one you can download for free. 

So if you have notes stored locally on your desktop, the other devices could not sync with them. You would have to move the notebooks to OneDrive first (not copying anything by explorer but use the appropriate function built into OneNote instead). But this seems not the case as you mentioned that you can access your notes not only on your desktop PC but also on the iOS devices.

So the only thing you need to access your notes with your notebook (running Windows, i guess), is: Sign in with your MS credentials (the same as for the other devices) inside of OneNote to access your notes on OneDrive.

How do I get rid of long running scripts - Microsoft Community

barbarahandy say:

How do I get rid of long running scripts

How do I get rid of long running scripts and why do they pop up

don varnau say:

If this is happening only on one (or a few) website(s) there really is (by Internet Explorer standards) a long running script on the page and IE is doing you a favor by providing the "Stop script" option. Click on the Stop script button.

Again, if only a site or two are affected...

It might break a desirable feature on the site, but placing a site in Restricted Sites will disable JavaScript and the script messages will stop.

Internet Options> Security> Restricted Sites...

If you're seeing the message on all or nearly all sites, a browser add-on might be causing it.

Note: The Google Toolbar has caused script problems for some users.

Start Internet Explorer without add-ons from Start> Programs> Accessories> System tools> Internet Explorer (no add-ons)
or

from Start> Run (or Windows logo key + r) type in iexplore.exe -extoff

(iexplore.exe<space>-extoff) then hit Enter.

If the problem goes away, an add-on is causing it.

To determine which add-on is causing the problem see this old (but still relevant) article:

IEBlog  Troubleshooting and Internet Explorer’s (No Add-ons) Mode:
http://blogs.msdn.com/ie/archive/2006/07/25/678113.aspx

Another option: Other browsers like Edge, Chrome and Firefox are more forgiving of these script problems. Consider using another browser on affected sites.

Don

j.e. mcgimpsey say: Column widths are set as a multiple of the number of standard characters (i.e., a numeric character) of the standard font that will fit in a cell - so a column width of 10 will hold 10 numeric characters (this has been hidden in the last couple of versions of MacXL since it reports width in inches/cm).

First uninstall AVG2013 - please see the relevant section (FAQ 5172) here:

http://www.avg.com/gb-en/faq  .

 

Now to turn Windows Defender on:

1. On Start screen, begin typing Windows Defender until it appears on the left then left-click on it.

2. Select Settings tab.

3. Tick Turn on real-time protection (recommended)

4. Click Save changes.

5. Click Update tab then Update button then wait for the latest updates to be applied.

6. It would now be sensible to do a full-scan to start with, so click on Home tab then select Full then click Scan now.

7. Wait for scan to finish and action results as necessary.
