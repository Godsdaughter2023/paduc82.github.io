frederik long say:

The default command used by the Task Scheduler to create restore points is

%windir%\system32\rundll32.exe  /d srrstr.dll,ExecuteScheduledSPPCreation

This command creates new restore points conditionally, depending on criteria such as how long since the last restore point was created. If you want to create a restore point unconditionally then you can schedule the following command. You must run it under the System account.

wscript.exe  c:\Tools\ManualRestorePoint.vbs

And here is the code you must put into ManualRestorePoint.vbs:

sWinmgmts = "winmgmts:\\.\root\default:Systemrestore"
GetObject(sWinmgmts).CreateRestorePoint "Weekly Restore Point created on " & date(), 0, 100
