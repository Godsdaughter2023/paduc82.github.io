Hi,

Thank you posting your query on Microsoft Community.

I appreciate you for providing details about the issue and we are happy to help you.

You can move folders around within storage folders or within an account, but you can’t move them out of storage folders or an account. To get back to your previous folders, you'll have to create the folders you need under the account, then move messages from storage folders to the account folders. You can select all the messages in a folder by using Ctrl+A, then right-click and Move to folder.

If there are a lot of messages involved, it's best to do this sort of exercise while working offline to avoid interference. You might also want to consider disconnecting from the network and disabling your anti-virus meanwhile, too, for the same reason. Allow Windows Live Mail to settle down after each move before starting the next, so that the database has chance to keep up. When you're finished and all's quiet, close down Windows Live Mail and wait a few minutes before re-booting the computer and re-enabling your anti-virus. Then launch Windows Live Mail and check whether everything is in order.
