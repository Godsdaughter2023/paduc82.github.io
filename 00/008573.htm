Hi Experts,

We have introduced an incentive program at work where staff earn a bonus when they sell a specific product.

On the data worksheet I have  column of products and in the next column the corresponding $ bonus.

On the data sheet I have 2 columns

product $$$

On the associate sheet I have 5 columns

date product receipt# $$$ paid out

Under product I have been able to create the dropdown list to select what they sold.

My question is how do I fill the $$$ cell with the correct amount

As always thank you in advance for your assistance.

hansv mvp say:

Option 1: use a formula.

Assuming that product starts in B2 and $$$ starts in cell D2, enter the following formula in D2

=IFERROR(VLOOKUP(B2, Data!$A$2:$B$1000, 2, FALSE), "")

(You can adjust the row number 1000)

Fill down as far as you want.

Option 2: use VBA code.

Right-click the sheet tab of the associate sheet.

Select View Code from the context menu.

Copy the following code into the worksheet module:

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim cel As Range
    Dim rng As Range
    If Not Intersect(Range("B2:B" & Rows.Count), Target) Is Nothing Then
        Application.EnableEvents = False
        For Each cel In Intersect(Range("B2:B" & Rows.Count), Target)
            If cel.Value = "" Then
                cel.Offset(0, 2).ClearContents
            Else
               Set rng = Worksheets("Data").Range("A:A").Find(What:=cel.Value, LookAt:=xlWhole, _
                   MatchCase:=False)
                If rng Is Nothing Then
                    cel.Offset(0, 2).ClearContents
                Else
                    cel.Offset(0, 2).Value = rng.Offset(0, 1).Value
                End If
            End If
        Next cel
        Application.EnableEvents = True
    End If
End Sub

Switch back to Excel and save the workbook in a macro-enabled format (.xlsm, .xlsb or .xls, but NOT .xlsx). Users will have to enable macros.

suzanne s. barnhill say:

Assuming that your printer can duplex automatically, there are two fairly simple ways to approach this:

  1. If you have set the document up for "2 pages per sheet" on the Margins tab of Page Setup, create your two pages as shown, and then print pages 1,2,2,1 with automatic duplexing.
  2. If you've created the double page using columns, then create a two-page document with the same content on both pages but with the columns reversed on the second page, and then duplex.

If you've created this document as two full pages and were planning to use "2 pages per sheet" on the Print tab of the Backstage, then I couldn't recommend that unless you were using European A sizes (US sizes don't scale properly), which I doubt that you are given the US address. In that case you could try printing pages 1,2,2,1, duplex, but I don't think you'd be happy with the results.
