john - project say:
FramaKirk,
It sounds like you are using a template file. Add the Constraint Type field to the Gantt Chart view. My guess is that the start date of one of your tasks in the existing project file is constrained with a Must-Start-On or Start-No-Later-Than constraint. Make sure all your tasks are set for As-Soon-As-Possible. The schedule should be driven by the Project start date and the logic network links.

It's also possible that one of the tasks in the existing file has an actual start date (i.e. % complete is non-zero). In order for tasks to dynamically "float" (not to be confused with slack), they must not be constrained and they must not have actuals.

John
