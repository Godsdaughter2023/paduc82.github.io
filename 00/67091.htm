bkstevens1 say: carsmi1,

This is very easy to do, first, right click on the desktop and select the Personalize option at the bottom of the pop up window.

Next, select Desktop Background toward the bottom of the new window.

In this new window you have a pull down list called Picture location.  Inside the pull down list you will see many choices for where your pictures might be.  If your storage location is not there, select the Browse button instead of the pull down menu.  Once you have selected you storage location, all pictures are checked by default.  Simply chose the uncheck all button, and then place check marks on the pictures you want in your background list.  In this location you can also chose how long until the pictures change, and the size of the picutes. 

P.S. I you have pictures in several locations, I would suggest either moving them all to one location or using the Library function to find them all for selection.
