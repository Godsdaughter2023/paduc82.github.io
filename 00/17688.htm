girish m say:

As of for IMAP type accounts, you will not be able to set or reduce the time to delete emails on server. In an IMAP type of account a copy of the messages are kept on your computer and also on the mail server. This enables you to use web mails or access your mail items from multiple computers. Deleting emails in Outlook will automatically delete emails on the server.

The work around available is to remove and re-add the email account as a POP3 type.

 

If the email account is configured as POP3, then you can change the time for deletion of emails by going to More setting under Advance tab “Remove from server after _ days”.

steve rindsberg say:

Because of the use of Integers (rather than Longs) and User instead of User32, I'm guessing that you've found some very old (and no longer workable) VB.

I've managed to modify it to the point where it'll compile and run but it takes PPT down more reliably than Notepad.  ;-(

Give this one a try instead (substitute the exe name of the program you want to whack in place of notepad.exe):

Private Sub Command1_Click()
    TerminateProcess ("notepad.exe")
End Sub
 
Private Sub TerminateProcess(app_exe As String)
    Dim Process As Object
    For Each Process In GetObject("winmgmts:").ExecQuery("Select Name from Win32_Process Where Name = '" & app_exe & "'")
        Process.Terminate
    Next
End Sub
