rick rothstein, mvp - excel say:
I think this is more a Windows question than an Office question. It has been a long time since I have worked with WinXP (I am using Win7 now), but the procedure should be similar to this. Go into the c:\windows\fonts and find the .ttf file for the font that you want and copy it a flash drive (or use some other method to make it downloadable on your laptop). Next, go to your laptop, open Windows Explorer and locate the file on your flash drive, right click it and then Preview the file. The dialog box that displays the font should have an Install button on it... click it to install the font onto your laptop. It should now be available for use on your laptop.

bill manville say:

b - I don't think it is helpful to use the numerical value of the date rather than DATE(2016,8,1) as I suggested; makes it much harder to know what the formula is doing.

b - The reason for "too few arguments" is that you have only given Choose 1 argument; it needs 1 for the choice number (delivering 1 to N) and then the N choices.

c - I didn't suggest using a different date format, I suggested using the DATE function

I cannot complete your formula for you because you have not said what the 5 expressions are, but

“=CHOOSE(MATCH(Current_Date,{DATE(2016,8,1),DATE(2016,9,1),DATE(2016,10,1),DATE(2016,11,1),DATE(2016,12,1)}),a,b,c,d,e)

or

=CHOOSE(MONTH(Current_Date)-7,a,b,c,d,e)
