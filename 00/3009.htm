Thanks!
daniel g. grau say:
Cause: That is happening because Word needs to reference the original TOC titles in the document, which it assumes to be local. When you paste the TOC into a new document, Word no longer sees the titles to build upon.

Solution: Go to your original document, select the TOC and press Command Shift F9. This will "unlink" the TOC field and convert it to text. Then, copy it to your new document. However, unlinking a field will prevent further updates of the TOC, so keep your original document around.

How to create an Office 2016 folder icon in the OSX High Sierra dock - Microsoft Community

nicholasdrummond say:

How to create an Office 2016 folder icon in the OSX High Sierra dock

Many people have asked the question: "How do I create an Office 2016 orange folder icon in the dock of OSX."

It is actually quite straightforward. But beware older explanations of how to do his have changed as OSX has evolved. I therefore aim to provide a simple up-to-date and definitive explanation. 

Step 1 - If you haven't already done so download Office 2016 for Mac and install it in the Applications folder of your Mac.

Step 2 - Go to the Finder and select the Applications folder. Using the top navigation bar, click File and select New Folder. A new folder called "untiled folder" will appear. 

Step 3 - Select the untiled folder and rename it whatever you wish to call it, e.g. "Office 2016"

Step 4 - Select the individual Microsoft Office applications (Word, PowerPoint, Excel etc.) and place them in the new folder.

Step 5 - Find the icon art you wish to use for Office 2016 for Mac and download it to your desktop. If you want to use the same orange icon that you see on Macs in Apple stores, try here:

https://orig00.deviantart.net/dff2/f/2016/058/3/d/microsoft_office_2016_folder_icon_by_muhammadm95-d9smrff.png

Step 6 - Open the file and Save as a .PNG file (if it isn't already in this file format). 

Step 7 - Open the folder in Preview and click Edit in the top navigation bar. Select Copy.

Step 8 - Select the Office 2016 folder. Click File and select Get info. A small box will appear showing the folder icon at the bottom.

Step 9 - Right at the top of the box, you will see a very small folder icon. Click on this so that it is highlighted in blue. 

Step 10 - Click Edit and select Paste in the top navigation bar. The new folder image you have selected will now appear as the folder's new icon. Close the box and you're done.

Almost... the icon may be distorted so that it is square in shape. If this happens, it is because the image you have used is not itself square. OSX makes all folder icons square. To make a folder image square, you need to edit the canvas on which the image is placed. If you don't knowhow to do this, go to Mac App Store and download a free app called Image2icon. Follow the instructions and it will be done automatically for you. )
