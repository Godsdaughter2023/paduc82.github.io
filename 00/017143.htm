bernie deitrick say:

First, you don't need the ElseIf, since one of those conditions must be true if either of the If conditions are false.

Sub Turn_Shape_Off_And_On()

If Sheet2.Range("A1").Value >= Sheet2.Range("B2") And Sheet2.Range("A1") <= Sheet2.Range("B3") Then
Sheet1.Shapes("Star").Visible = True

Else
Sheet1.Shapes("Star").Visible = False

End If

End Sub

But, there is nothing wrong with your code - just remember that Sheet2 may not be named "Sheet2" - you are using the codename of the worksheet object, which is different from the name on the tab. The codename comes first, and the sheetname is in parentheses - here are two workbooks with 2 sheets where the sheet names and the codenames are the same: in the first, the worksheet object with the codename Sheet2 is named "Sheet2; in the second, it is named "Sheet1"   - the same goes for Sheet1 - that is the codename (the name not in parens).

mark bu say:

Hello,

When you receive the message “Your account settings are out-of-date”. You should be able to address this issue by going to Settings -> Accounts and the selecting the account you receive the message for.

 

Outlook.com Accounts

If the account is an Outlook.com account, select “Change account settings”. This will open edge to the https://account.microsoft.com page. Signing into this page should update your password settings in Mail.

NOTE: If you sign into Windows using a different MSA; you may be automatically signed into account.microsoft.com with that account and not the Outlook.com account you are attempting to update the password for in Mail. Selecting the option to sign into a different account and selecting the account you need to update in mail should address the issue in Mail.

Other Accounts

If the account was setup as an “Other” account type, you will be able to change the password in the dialog that opens.

 

Alternately, if this is not the default Outlook.com account setup in Mail, you can also attempt to remove and then re-add the account.
