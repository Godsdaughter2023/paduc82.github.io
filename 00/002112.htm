Hi Samantha.

After doing several tests, I have found that the following steps may work for you:

Export categories in Outlook 2013 (If you are using Outlook 2010 or 2007, you can also use these steps):
1. Create a new note with Ctrl-Shift-n
2. Add text to it and close it afterwards.
3. Use CTrl-5 to get to your notes folder
4. Right-click and select Categorize > All Categories
5. Select all categories that you want to export
6. Use File > Save As to save the note to the computer

Import categories into Outlook 2013 (If you are using Outlook 2010 or 2007, you can also use these steps):
1. Use Ctrl-5 to open the notes folder
2. Drag and drop the exported notes file into it
3. Switch to the mail interface with Ctrl-1
4. Right-click and select Properties
5. Select Upgrade to color categories

If you are using Outlook 2016, you can right click the display name of your mailbox while in the mail interface to choose "Data File Properties" for the above mentioned step 4.

suzanne s. barnhill say:

This really has nothing to do with the TOC per se. If you have only a single page you want unnumbered and want numbering to start at 1 on the second page, then a short cut is to enable "Different First Page." Leave the First Page Header/Footer empty and start numbering at 0. The number on the second page (in the primary Header/Footer) will be 1.

For more complex situations, you will need to insert a section break as explained in  http://word.mvps.org/faqs/formatting/front_matter_2007.htm.

You could have the person browse to the accDr file. Have the right click on it, and then choose open with. The access runtime might be listed at that option, and there will be a check box on the lower left.

 

However, if ms-access is not listed, then you can have the person browse to msaccess.exe. For 2007, that should be

 

C:\Program Files\Microsoft Office\OFFICE12\msaccess.exe

 

So, have them browse to the above and if they check that box that says "always" use this program, the accdR will be re-associated with ms-access.

 

Another way around this is to provide them a shortcut file. Create a shortcut, and then simply email them that shortcut.   That shortcut would look like:

 

"C:\Program Files\Microsoft Office\OFFICE12\MSACCESS.EXE" "c:\RidesDev\SkiRides\RidesXP\RidesXP.accDR"

 

You will of couse have to have the correct path name to the accDr file, but that is another possible solution.
