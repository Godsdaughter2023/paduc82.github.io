dhookom say:

I would create a small filter table:

tblFilters

Set the Row Source of your combo box to something like:

SELECT filTitle, filWhere

You can then use the after update event of the combo box to apply the WHERE clause:

Private Sub cboFilter_AfterUpdate()
    Me.sfrmWorkOrders.Form.Filter = Me.cboFilter.Column(1)
    Me.sfrmWorkOrders.Form.FilterOn = True
End Sub

You can use this same functionality in other forms in your application where you need to apply a filter.
