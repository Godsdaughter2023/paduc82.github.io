Marketplace is an application that requires an active internet connection for the whole time it is running on your phone. Nokia Lumia 610 is a phone that is designed to be connected to the internet at all times, either via WiFi or mobile data connection, thus the applications on the phone expect that as well. Especially Marketplace which is basically just a browser for Windows Phone Marketplace content.

 

When the phone is inactive, the connection to WiFi is automatically disconnected, when you "wake up" the phone the connection is re-established. So, first make sure that the connection is active before accessing any application (you can see the icon on the top of the screen).

ossiemac say:

The "Too many formats" possibly applies to Number Format. Custom Formats that are added by the user are saved in the workbook and do not automatically delete because they are no longer being used in the workbook.

Open Number Format and select Custom. Scroll down to the bottom of the custom formats and see how many user added custom formats you have below the built in formats.

The built in formats should be the top ones and the Custom formats are below them. You can identify the built in formats by selecting each format in turn and the Delete key should be disabled (greyed out) so you cannot delete a built in format. Therefore try deleting the custom formats that have been added by the user.
