charles kenyon say:

In word jargon, "highlight" means painting with a background that is not a shade. You mean change to a different color to make it stand out, I think?

The easiest way is to use the Replace function (Ctrl+H). Put the same word into the find and replace boxes. Then with your insertion point in the replace box, click on the More button. Click on Format >Font and pick the color you want. Then click on Replace All.

Note, in your example you picked out luminary and luminaries. These are different words, or different forms of the same word. In either case, you would have to run the replace procedure once for each variation. The color setting for the Replace will stick until you change it.

Hi Prakash Devarasu,
you can try with this code after set up folder names and range.

Andrew.

---
Public Sub copyFiles()
Dim wsSource As Excel.Worksheet
Dim sCopyFrom As String, sCopyTo As String, sFileToCopy As String
Dim lFiles As Long, lLastSourceRow As Long
Dim rFileToMatch As Range
Dim vbFile As Variant

  On Error Resume Next

  '---------- set up your data here
  sCopyFrom = "C:\CopyFromFolder\"
  sCopyTo = "C:\CopyToFolder\"
  Set wsSource = ThisWorkbook.Sheets("Sheet1")
  rFileToMatch = wsSource.Range("A2:A100")        ' range with file names to copy

  For Each vbFile In rFileToMatch
    '---------- no file extension for files to copy!
    sFileToCopy = "surface_" & Trim$(vbFile) & "_upper"
    If (Len(Dir(sCopyFrom & sFileToCopy)) > 0) Then
      lFiles = lFiles + 1
      FileCopy sCopyFrom & sFileToCopy, sCopyTo & sFileToCopy
    End If
  Next

  Call MsgBox(lFiles & " files copied.", vbInformation, "Copy Files")
End Sub
