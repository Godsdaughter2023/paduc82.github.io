How do I DELETE, no merely uncheck, an item in msconfig.exe, - Microsoft Community

beccaroo say:

How do I DELETE, no merely uncheck, an item in msconfig.exe, Startup?

The basic question is, how do I delete files from the msconfig.exe startup tab? I tried locating the file using the command prompt, but it would not display the file so I could delete them. Thanks...
burrwalnut say:

Two registry keys store the items removed (unchecked) from startup by the msconfig program. Open a Run window (Windows Logo key+R), type regedit, press Enter and in the left pane navigate to HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Shared Tools\MSConfig.

 

One or both subkeys (startupfolder and startupreg) contain entries for disabled items. To remove an unchecked item from the list in msconfig, right-click the name and delete it, e.g. skype under startupreg.

 

Create a system restore point before deleting the keys in case anything goes wrong.
