edwin tam say: Try this:

If you want to define a conditional format in A1:A10, so that if those cells equals #N/A, then font colour is white:

1) Select A1, go to HOME tab of ribbon, click "Conditional Formatting". Choose "New Rule"
2) Choose "Use a formula to determine which cells to format"
3) In "Format values where this formula is true", type =ISNA(A1)
4) Click "Format", in the Font tab, choose a white Color,
5) Click OK, and OK again.
6) Go to Conditional Formatting again, click "Manage Rules"
7) Locate the rule you just created. In "Applies to", type    =$A$1:$A$10

greginmich say:

Also, is there a way I can set a certain time and day to run automatically?  Can't seem to find that anywhere!

Windows Defender- Original Title 

Moved from Windows

Explaining how Automatic Maintenance works isn’t easy, so the Microsoft documentation authors apparently decided that it would be more expeditious to just offer people a “placebo” method for scheduling a scan by adding a trigger to the Windows Defender Scheduled Scan task. That’s a bit facetious, because Automatic Maintenance is already triggering the Windows Defender Scheduled Scan task on a daily basis without you having to do any configuration or scheduling whatsoever. That’s why the scheduling options disappeared from the Windows Defender User Interface. The “scheduled scan” now just runs automatically, in keeping with the goal of totally automated protection:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-to-schedule-windows-defender-scan/8eab1302-7326-4632-a004-8f4dce1b56d1

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_start/windows-10-how-do-i-set-windows-defenders-scan/f399bb87-37c9-4a74-bb35-1d7e3d3c2436

There’s also a slight operational problem with the scan that you'll get if you schedule it by hacking the maintenance task rather than by setting up your own independent scheduled task: The scan will get cancelled every time you start using your PC, and then wait for the next system idle state to start over again. That’s exactly how the Automatic Maintenance scan works, and although this scan-only-on-idle operational mode works very well for preventing the scan from interfering with your work – it obviously won't work very well at all if you’re looking for a scheduled scan that actually starts at the scheduled time and runs without interruptions. In order to get that old-fashioned kind of scheduled scan, you'll need to schedule your scan the old-fashioned way – with the Create Basic Task Wizard:

http://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning/how-do-i-set-windows-defender-to-a-custom-auto/ec19aab4-10ce-4f41-9990-343056fd104f

Why Call It Canary?

By calling the release Canary, Google is referring to the old technique of using canaries in coal mines. Back in the day, coal miners used to bring canaries down in the mines with them as a precaution. If there was any kind of deadly gas build-up, the canary would be the first to die. If the canary died, everyone knew it would be a good idea to leave the mine as soon as possible.

Chrome Canary offers the same set of precautions. The Canary build is less stable, but it will be used to test even more cutting-edge features than the developer’s build, which already tests new features before they are pushed to the beta build. If a change “kills” Chrome Canary, Google will block it from the developer build
