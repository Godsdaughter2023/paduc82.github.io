    If Windows crashes, because of bad drivers or any other reason, during the upgrade process we’ll fail the upgrade and restore your previous operating system. On the upgrade progress screen you'll see three steps listed along the bottom "Copying files," "Installing features and drivers", and "Configuring settings."

    During "Copying files", we actually boot into a minimal version of Windows and copy important files and drivers from the old operating system into the new one. If we fail in this first stage you’ll see an error code in the Windows Update control panel called c1900101 x200xx. For example, 20017 means we couldn’t even boot into this operating system and usually points to bad storage drivers.

    The "Installing features and drivers" phase is the trickiest part of upgrade. If you see a failure c1900101 x300xx, it means we failed during this phase. The most common failure is x30018, which generally means a driver crashed when we loaded it for the first time. Any driver could cause the failure, but it may not crash 100 percent of the time. If you try again, it may just work.

    "Configuring settings" is the final phase and failures codes look like c1900101 x400xx. The most common failure here is 40017, which means we failed to boot. Since this is the first time we boot into Windows 10 with all of the drivers in place, again, drivers are usually the cause of the crash. These issues tend to be less transient than the 30018s, so a second try is less likely to solve the issue. Still, it never hurts to try.

    OneDrive sync issues: Windows 8 and below - Microsoft Community

    a. user say:

    OneDrive sync issues: Windows 8 and below

    Technical Level : Basic

    Summary

    Self-help steps to resolve OneDrive Desktop app synchronization issues in Windows 8 (not 8.1), Windows 7 and Windows Vista.


    Details

    First thing to try

    Let's start with general troubleshooting steps:

    First of all, please review the article below and try the provided troubleshooting steps:

    • Troubleshooting Sync Problems in OneDrive in Windows 8 and older

    Please also try unlinking the account:

    1. Locate and right click the OneDrive icon in your Windows 8,7 or Vista system tray.
    2. Choose the Settings option to bring up the OneDrive Settings.
    3. On the main OneDrive Settings tab, click the Unlink OneDrive button.

    Now restart the OneDrive service:

    1. To do this, click the Start icon and type OneDrive into the Start Menu search dialog
    2. Click Microsoft OneDrive once you see it in the search results.
    3. Sign-in to your Microsoft account and follow further steps.

    If OneDrive works fine, and the sync issue affects only Office documents, or OneDrive is consuming high amount of CPU, do not use the steps above just yet – try the steps from OneDrive: Office documents first.

    If your issue falls under the following categories:

    • not able to sign-in into app
    • white / blank page during sign-in
    • script error "evt_LoginHost_onload is undefined"

    Please reset Internet Explorer, set Internet Explorer settings to medium-high and enable TLS*.

    * TLS: Transport Layer Security - Definition and understanding

    1. Reset Internet Explorer
    2. Please set your Internet Explorer Security settings to Medium-high
      1. Click Start.
      2. In search field type inetcpl.cpl, and then press Enter.
      3. On the Security tab, click Internet.
      4. Check Security level for this zone.
      5. If it is High, change it to Medium-high.
      6. Click OK.
    3. Enable TLS
      1. COG > Internet Options > Advanced.
      2. Put a check marks on Use TLS 1.0, Use TLS 1.1, Use TLS 1.2 and apply.
    4. Re-download OneDrive and try to launch it.

    Second attempt

    If it is a sign-in to app issue (see above), the next step to try is to use OpenDns:

    1. To set it up, follow steps from this article.
    2. In step 6, make sure to specify a DNS server address:
      1. Click Use the following DNS server addresses, and then, in the Preferred DNS server and Alternate DNS server boxes, type the addresses of the primary and secondary DNS servers, which are:
        • 208.67.222.222 (or resolver1.opendns.com)
        • 208.67.220.220 (or resolver2.opendns.com)
    To read more about OpenDNS, please click here to review OpenDNS Wikipedia article. To read more about OpenDNS, please click here to review OpenDNS Wikipedia article.

    For other issues (sync, crash, launch, etc.), please re-install the app:

    • Make sure OneDrive is un-installed (steps to un-install).
    • Perform clean boot.
    • Re-download OneDrive and try to launch it now.
    If this did not help, please repeat these steps again, but choose another location for your OneDrive folder in step 3. If this did not help, please repeat these steps again, but choose another location for your OneDrive folder in step 3.
       

    Third attempt

    Please create new Windows user account:

    • Create new Windows user account.
    • Re-download OneDrive and launch it.
    • If this helps, follow steps from this article in order to migrate user files from the old account to the new one.

    Contact support

    If none of the steps above help to resolve your issue, please contact OneDrive support by clicking this link. Please include as much information as possible in the very first contact; this will help to resolve your case quicker:

    • What operating system is affected?
    • Have you tried to perform all the troubleshooting steps available in this article? Please mention if you could not complete one or another troubleshooting step.
    • Does it affect another Microsoft account? (You will have to disconnect that account and re-connect with another, preferably new; see steps above)
    • Do you synchronize this account somewhere else? If yes, do you have similar issue on any of the other devices / computers?
    • Do you see error code or error text?
    • Does the sync issue affect all files or just some? Specific format, size?
    • Do you have any issues when using same account in OneDrive web version?
    • Do not hesitate to provide screenshots or even a screen capture file created by Problem steps recorder.
    • Other information that you find valuable.
