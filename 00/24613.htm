marshall barton say:

IMO, the easiest way to so that is to add a text box to the detail section.  Set its Conditional Formatting expression to this kind of thing:

   [actual date field name] >= DateAdd("m", -3, Date())

and select yellow as the text box's BackColor

 

Test that to verify it works for the right records.  Then resize the text box so it's the same size as the entire detail section and use Send To Back to put it behind all the other controls.

 

Most likely, you will also want to set all the other text and combo boxes BackStyle property to Transparent.

 

This will work in forms in single or continuous view or in reports.

 

Whether you can do that yourself or not is for you to decide.  I think it an easy thing to do and, with a little experience, can be done in a few minutes.
