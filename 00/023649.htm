Hi All,

I have an excel workbook with two work sheets (Sheet1 and Sheet2).  I am trying to create a button that when clicked will copy cells A2:B108 from Sheet1 and paste them on Sheet2, starting in cell A4, based on the value of cell D3 on Sheet2 (if D3 = 6, I need the data pasted 6 times on Sheet1).  I don't think that this Macro should be too complicated but from the research that I've done over the past couple of days I can't find any relevant help.

Scott

Try this code:

Sub Button1_Click()
   Set range_to_copy = Worksheets("Sheet1").Range("A2:B108")
   Set my_destination = Worksheets("Sheet2").Range("A4")
   Set countercell = Worksheets("Sheet2").Range("D3")
   range_to_copy.Copy
   For i = 0 To countercell.Value - 1
     ActiveSheet.Paste Destination:=my_destination.Offset(i * range_to_copy.Rows.Count, 0)
   Next i
   Application.CutCopyMode = False
End Sub
