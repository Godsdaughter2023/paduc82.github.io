There's no need to use a third party application to create a System Image. This can be done quite easily by using the "System Image Backup" option in the File History menu, the latter of which is located in Windows Control Panel.

  3. In the bottom of the three options on the right, click the "Restart now button". See screenshot.

I've used this myself and it restores everything which was on the computer on the day you created it.
