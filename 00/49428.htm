Where is the start window on Windows 8? - Microsoft Community

johnfooten say:

Where is the start window on Windows 8?

tzon say:

There is no start menu! But this is nothing to worry about. You can work in Desktop mode almost the same way as with Windows 7.

 

In the Start Screen (the screen you see when logging in to your PC) click on the tile that says "Desktop". Now you are in desktop mode, just as in Windows 7.

 

To launch a program go to the left down corner (where the start button used to be) and click. You are now back on the start screen where all of your programs are pinned (just as shortcuts used to be on the desktop):

 

- If you click on a modern app, it will open in using the new (modern) full screen interface mode. You can always transfer to the start screen by clicking the left down corner. If more than one modern programs are launched, you can manage them with the new taskbar. Move to the upper left corner and the task bar will appear. There you can see all active programs. To close a modern program you can either right click and Close from the modern (left) taskbar or just swipe it from the top all the way down when in full screen.

 

- If you click on a classic program (such as word, excel, adobe reader etc.) it will open in desktop mode which feels just as Windows 7.

 

You can ignore modern programs completely by customizing your start screen. Right click on a tile in the start screen to select a program. Then you can click on "un-pin from start". To pin a program to the start screen right click on a blank space of the start screen and then click All Apps in the bottom. You will be transferred to a list containing all installed programs. Find the programs you need, right click on them and select "pin to start".  Go back to the start screen (click left down corner). Your programs should be there as tiles. You can drag and drop them around (up n down, left or right) and move them in categories. Now you are ready to use the desktop the same way you did before and use the start screen as a large start menu.

 

Now, about the right corners... If you move your mouse to a right corner (up or down) you will see a side bar appear. It is called the "charms bar". You can see there a few functions, including Settings and Start. Start is another shortcut to the start screen. Settings is more useful. You can access from there the Control Panel (only when in desktop mode). If you're running a modern program (full screen app) and click settings from the charms bar, then the settings that open correspond to the settings of the current App.

 

To shut down your PC, view the charms bar (move to a right corner) and click on Settings. Then select Power and finally Shut Down (or Restart or Sleep).

 

For more information about how to use Windows 8, you can read the following guide (at least the chapters you find helpful).

 

http://win8manual.com/wp-content/uploads/2012/09/Getting-Started-Your-Guide-to-Windows-8.pdf

 

Good luck!
