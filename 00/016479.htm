a. user say:

RE : Some contacts have more than one email address. When I expand the Groups list in the To field, I see that the contacts have an email address that I do not want to the email to be sent to.

Outlook.com ALWAYS uses the address entered in the Personal box.
IF the Personal box is blank, then it will go looking for one in Work box.
If both Personal and Work boxes are entered, outlook.com will use the one in Personal and will ignore the one in Work box.
 
 
So, if you see an address you don't want, then go back to your group contact list and check those contacts in question.
If you see wrong address in the Personal box, EDIT it.
Also,in your group contact list,  make sure you don't have more than one entry on each contact.
I mean you don't have 2 John Doe each with a different address.
 
 
Open outlook.com > top left next to the logo Outlook, there is a down arrow > click the arrow > click People.
 

How to access a group and see the contacts within that group:
In the People window, near top left, click at the word " All v " > a drop down box will appear with a list of your groups > click the one you want > all the contacts within that selected group are now on the left side > scroll to see the whole list.

 

How to Edit contacts within a group :

On the People window, top left, click the “ All v “ > you group is on the drop down list > click the group name > now you have all the contacts within that group listed > click one to view > to edit, click at the contact’s name ( not its check box ) > click Edit at the toolbar to do editing......
Note :
1. Edit option will not appear until you have selected a contact.
2. Edit option will not appear if you click at the contact's check box instead of the contact's name

Sub Test()
  Dim C As Chart
  Dim R As Range
  Dim FName As String
 
  'Refer the cells
  Set R = Sheets(1).Range("A1").CurrentRegion
  'Refer the chart
  Set C = Sheets(1).ChartObjects(1).Chart
 
  'Delete all series
  Do While C.SeriesCollection.Count > 0
    C.SeriesCollection(1).Delete
  Loop
  'Delete all previous pictures
  Do While C.Shapes.Count > 0
    C.Shapes(1).Delete
  Loop
  'Resize the chartobject to fit the cells
  With C.Parent
    .Width = R.Width
    .Height = R.Height
  End With
  'Copy the cells as bitmap and paste into the chart
  R.CopyPicture xlScreen, xlBitmap
  C.Paste
  'Export
  FName = ThisWorkbook.Path & "\temp.gif"
  C.Export FName, "GIF", False
  'Load into the userform
  UserForm1.Image1.Picture = LoadPicture(FName, 0, 0)
End Sub
