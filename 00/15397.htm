jeovany cv say:

Hola Victor

I'm not an expert in VBA. I just learned in the same way you are probably doing.

This is my remarks from your macro. 

1- The variables in 2 lines will always be the same

result_row = Sheets(3).Range("A2:A65570").Row         it will always be row =2
result_col = Sheets(3).Range("A2:A65570").Column    it will always be column="A"

Therefore

result_row =2
result_col ="A"

2-Declare, Dim table1 and table2 as Range

3- Take HansV MVP suggestions (remarks)

4- In your VlookUp formula Search Criteria is cl.Value not cl 

5- You might get confused by setting Table1 to sheet 2

Set Table2 = Sheets(1).Range("A1:A65570")

Set Table1 = Sheets(2).Range("A1:A65570")


I'd switch them

Set Table1 = Sheets(1).Range("A1:A65570")

Set Table2 = Sheets(2).Range("A1:A65570")


So having that in mind

Your macro could be

*************************************************************************************

Sub procv3()


Dim cl As Range

Dim result_row As Long

Dim Table1, Table2 As Range

result_row = 2

Set Table1 = Sheets(1).Range("A1:A65570")

Set Table2 = Sheets(2).Range("A1:A65570")


For Each cl In Table1

        SCr = cl.Value    '' This is the Search Criteria

        Sheets(3).Cells(result_row, "A") = Application.VLookup(SCr, Table2, 1, False)

        result_row = result_row + 1

Next cl

    MsgBox "Done"

End Sub

************************************************************************************
