ken sheridan say:

So what I would like to do  is create a function where I can determine the maximum values between data in different columns within the table.  Maybe it can't be done.

It can.  The following two functions each accept an arbitrary number of values as a parameter array, returning the max or min value of the members of the array.   A value need not be a literal value of course; it can be a reference to a column or an expression:

Public Function GetMax(ParamArray aVals() As Variant) As Variant

    Dim var As Variant
    Dim varMax As Variant
    
    varMax = aVals(0)
    
    For Each var In aVals
        If var > varMax Then varMax = var
    Next var

    GetMax = varMax
    
End Function


Public Function GetMin(ParamArray aVals() As Variant) As Variant

    Dim var As Variant
    Dim varMin As Variant
    
    varMin = aVals(0)
    
    For Each var In aVals
        If var < varMin Then varMin = var
    Next var

    GetMin = varMin
    
End Function

Don't confuse this with the MAX and MIN operators used in Access.  These are set operators which return the max or min value from a column position in a set of rows of arbitrary size.
