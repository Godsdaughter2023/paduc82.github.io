cbarnhorst say:

Yes.  You can boot to uefi with Windows 10 and uefi legacy mode (emulated BIOS) with Windows 7 on the same computer.  Both are fine.  It happens automatically.  Just install each one and each one will be in its proper mode without your having to do anything special.  

The uefi standards committee has ensured that all computers using uefi firmware provide for seamless installation of legacy operating systems.  Since 2011 the Microsoft Windows Logo Program has required that all new computers shipped with the Windows Logo comply with these standards.

Windows 8 and 10 install on uefi computers with a primary partition and one two or small additional partitions.  Those are part of the UEFI format.  On the same computer you will only see a single, primary partition for Windows Vista or 7.  They are in UEFI legacy mode.  It is all part of the same UEFI firmware.

Got it now?  Install your copy of Windows 7 and enjoy.
