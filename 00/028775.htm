If you launch Excel from the Start menu and exit immediately do you see this problem?

If so, then it must be due to an add-in or a program loaded at startup.
If not, then it is probably related to a workbook you are using in the sessions in which you see the problem.

In the first case, disable ALL add-ins and remove all files from your XLSTART folders and see if the problem occurs when you run Excel normally.  If no problem then add them back in one at a time until you find the culprit.

[The following macro will list the files in your XLSTART folders in a worksheet.

Sub ListStartupFiles()
  Workbooks.Add xlWorksheet
  ActiveSheet.Name = "Startup Files"
  Range("A1:B1") = Array("Path", "File")
  ListFiles Application.StartupPath
  ListFiles Application.AltStartupPath
  ListFiles Application.Path & "\XLSTART"
  Range("A1").CurrentRegion.Columns.AutoFit
End Sub

Sub ListFiles(stPath As String)
  Dim stFile As String
  stFile = Dir(stPath & "\*.*")
  Do Until stFile = ""
    With Cells(Rows.Count, 1).End(xlUp).Offset(1)
      .Range("A1") = stPath
      .Range("B1") = stFile
    End With
    stFile = Dir()
  Loop
End Sub

]

In the second case, that it seems to be related to the workbook you have been using, you could try removing components from it until the problem goes away.
