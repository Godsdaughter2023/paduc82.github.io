RPC Server - Error message: RPC Server in unavailable - Microsoft Community

zeldapoo say:

RPC Server - Error message: RPC Server in unavailable

When I attempt to log on to my computer, I get a message saying the RPC Server in unavailable?  I usually have to make several attempts before I am able to log on.  Does anyone know what this means and how to fix it?

Thanks

rajithr say:

Hi,

 

Have you made any changes to the computer prior to this issue?

 

The RPC server, or Remote Procedure Call is a service on your computer that allows other computers to access it remotely.

Method 1: There are different scenarios when you get this error message. Please mention if this message appears when performing a specific task on the computer.

You can check Event Viewer to get detailed information about the problem. Refer the following links.

http://windows.microsoft.com/en-US/windows-vista/Open-Event-Viewer

http://windows.microsoft.com/en-US/windows-vista/What-information-appears-in-event-logs-Event-Viewer

 

 

Method 2: The RPC (Remote Procedure Call) service in Services should be started and the startup type should be set to Automatic.

a) Click Start, type services in the start search box and press Enter.

b) In the Services window, find Remote Procedure Call and right click on it to select Properties.

c) Under the General tab make sure the Startup type is Automatic and the Service status says Started.

 

 

Method 3:  If the above methods do not help, perform a system restore in safe mode.  You may restore the computer to a date where the computer was working properly.

a. Click Start, type system restore in the Start Search box, and then click System Restore in the Programs list.

 If you are prompted for an administrator password or confirmation, type your password or click Continue.

 

b. In the System Restore dialog box, click Choose a different restore point, and then click Next.

c. In the list of restore points, click a restore point that was created before you began to experience the issue, and then click Next.
