You don't create new addresses on your computer.  You create them in the mailbox.  See http://windows.microsoft.com/en-US/windows/outlook/add-alias-account

Microsoft has overloaded the term "Outlook" way beyond what is helpful.  "Outlook" can mean

  1. Outlook.com, the web site and the replacement for the web-based Hotmail.
  2. Office Outlook, the program that's part of the Office suite, along with Word, Excel, and so on.
  3. The mail program built into Windows 8/8.1, which is really Windows Mail.  It's seen as Outlook because it connects with Outlook.com using the Microsoft ID you use when signing into Windows.  It even has the same logo Outlook.com uses.

Windows Live was a Microsoft service that encompassed Hotmail (also called Windows Live Mail for a while), Windows Live Messenger, Photo Gallery, Movie Maker, and some other applications; this was also called Windows Essentials and Windows Live Essentials as a package.

Windows Live Mail is also a mail client like the old Outlook Express (there's that use of "Outlook" again) and it is part of the Windows Essentials package as well.
