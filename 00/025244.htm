You could just use a formula field coded as:

{=E2 \# ";;'O'"}

Actually it IS easy. As Daniel suggested you use the After Update event to assign a value to the textbox. 

It might help you to understand that Access is an Event driven platform. When you want something done in Access, you trigger the process using an event. So you want the date control to be populated when the combobox is updated. Hence the After Update event of the combobox. 

You need to use the Code Builder to enter a single line of VBA code like Daniel suggested.
