command-F9 (cmd-F9, or fn-cmd-F9, depending on how your keyboard is set up) inserts a pair of the special field code braces { }. Typing the { and } characters on the keyboard doesn't work.

option-F9 (also known as alt-F9), or fn-option-F9/fn-alt-F9 toggles fields between "field code view," where you see all the { }, and field results view. Sometimes you need to do that when entering field codes because the codes may disappear when you insert them (particularly if you copy/paste).

Selecting one or more field codes and pressing F9 (or fn-F9) updates the results of those fields.

Some field codes are automatically marked as "hidden text" and you may need to display hidden text to see them. But that does not apply to PAGE or NUMPAGES.

My description below assumes you do not need the "fn-"

So in this case you could do the following:

Remove what you have.

Click where you want the page numbering.

Type "Page  of  "

Click between "Page" and "of"

Click cmd-F9 so you have

Page {  } of  

Click between the  { } and type PAGE so you have

Page { PAGE } of

Click after the "of " and click cmd-F9 so you have

Page { PAGE } of {  }

Click inside the new pair of { } and type =-3 so you have

Page { PAGE } of { =-3 }

Click between the = and the - and click cmd-F9 so you have

Page { PAGE } of { ={  }-3 }

Click between the new  { } and type NUMPAGES, so you have

Page { PAGE } of { ={ NUMPAGES }-3 }

Select that lot and click F9 to update the fields, and if necessary, alt-F9 to show the results.
