To count anything (e.g. numbers, text, blanks returned by formulas) that is not truely blank, use COUNTA(...). COUNT(...) will return a count of numbers and COUNTBLANK(...) will count any blank cells, whether truely blank or a blank return by a formula. ROWS(...) will return a count of the cells in a single column. So,

=ROWS(C1:C19)-COUNT(C1:C19)-COUNTBLANK(C1:C19)-SUMPRODUCT(--ISERROR(C1:C19))
... or,
=SUMPRODUCT(NOT(ISERROR(C1:C19))*NOT(ISNUMBER(C1:C19)))-COUNTBLANK(C1:C19)

... should get you your count of cells containing true text and text that look like numbers.

Addendm: added a SUMPRODUCT(...) pseudo-count to discount error values.
