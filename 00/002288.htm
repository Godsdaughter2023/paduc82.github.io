graham mayor say:
Rich's macro addressed both headers and footers. if you merely want to address the footer you don't need to bother with the headers part as it just slows the macro down a tad. Working with ranges you don't have to open the headers or footers you merely work with the range itself, so to replace the last paragraph of the footer with the date, you merely set the range to the last paragraph of the footer and apply the date to that range


Sub DateinFooters()
Dim doc As Word.Document
Dim rng As Word.Range
Dim sec As Word.Section
Dim hfRng As Word.HeaderFooter
Set doc = Word.ActiveDocument

For Each sec In doc.Sections
    For Each hfRng In sec.Footers
        'set the range to the last paragraph of the footer
        Set rng = hfRng.Range.Paragraphs.Last.Range
        'now do something with the data in this range
        'eg add the date to the range
        rng.InsertDateTime DateTimeFormat:="MMMM d, yyyy", InsertAsField:= _
                False, DateLanguage:=wdEnglishUS, CalendarType:=wdCalendarWestern, _
                InsertAsFullWidth:=False
    Next hfRng
Next sec
End Sub
