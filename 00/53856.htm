On Thu, 15 Jul 2010 22:09:35 +0000, IBHIPRU wrote:

What are the specific steps in Windows 7?

The steps are exactly the same as copying any other kind of media and
in any other version of Windows.

And there's more than one way to do this. Here's one way:

Open the CD in Windows Explorer. Select all the files on it by
pressing Ctrl-A. Then copy all the selected files to the clipboard by
pressing Ctrl-C.

Then open the hard drive in Windows Explorer and navigate to the
folder you want these files in. Click in a blank place in the folder
and then press Ctrl-V to paste everything in the clipboard there.

spiritx say:

Hi,

This is assuming you want to COPY files and not install a program that is on a CD or DVD.

Put the disk into the CD/DVD drive. Usually a Auto-Run will come up and one option will be to
VIEW FILES - select that which is Windows Explorer.

Find the FILES (and/or FOLDERS) you want to COPY and Right Click on one and COPY. Then
go to where you want to put the FILE (FOLDER) and Right Click on and PASTE. Go back to
the disk and make the next COPY and repeat as needed.

----------------------------------------------------------------

You can also use Start - Computer   OR   Windows Explorer to navigate to the CD/DVD drive
and then use the method above to COPY and PASTE as needed.

----------------------------------------------------------------

Here are a couple of tricks to help using Start - Computer   OR   Windows Explorer.

You can HOLD the SHIFT Key and click on a FILE (and/or FOLDER) then go to the last in a series
and click on it and all of them will be HIGHLIGHTED - Right Click in the Highlighted are and then
you can COPY them all. When you go to the destination and PASTE they all will be copied in one
process.

You can HOLD the CTRL key and click on FILES (and/or FOLDERS) that are not touching, as many
as you wish, and all will be highlighted - then Right Click in the Highlighted area and COPY. Then
navigate to where you want to copy them and Right Click and PASTE and all will be copied in one
operation.

Hope this helps.
