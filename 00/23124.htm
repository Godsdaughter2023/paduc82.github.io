Word 2016 - Globally update all autotext entries in a template - Microsoft Community

ekh7000 say:

Word 2016 - Globally update all autotext entries in a template

Hi,

I have over 100 autotext entries in a template that reference my company's name.  The company's name has changed and I need to update all the entries.  Is there any way to do this globally?  Or is my only option to recreate each entry.

Any help would be appreciated.

Thanks

jay freedman say:

Hi,

I have over 100 autotext entries in a template that reference my company's name.  The company's name has changed and I need to update all the entries.  Is there any way to do this globally?  Or is my only option to recreate each entry.

Any help would be appreciated.

Thanks

The following macro is specific to your requirement. It will first let you select the template to work on, and then enter a Find What term and a Replace With term (the old company name and the new name). Note that the entries are case-sensitive; the Find What term's capitalization must match what is in the AutoText entries.

The macro will then replace the name in every AutoText entry in the General category in the selected template. (If your entries aren't in the General category, change that in the code.) The template with the changes will automatically be saved after all the replacements have been made.

Sub ReplaceInAutoTexts()

    Dim fDialog As FileDialog

    Dim strFile As String

    Dim objDoc As Document

    Dim objTemplate As Template

    Dim nCount As Long

    Dim nIdx As Long

    Dim strFind As String

    Dim strReplace As String

    Dim objAT As BuildingBlock

    

    Set fDialog = Application.FileDialog(msoFileDialogFilePicker)

    With fDialog

        .Filters.Clear

        .Filters.Add "Word Templates", "*.dot*"

        .Filters.Add "All files", "*.*"

        .InitialFileName = "C:\"

        .Title = "Select template to update AutoTexts"

        If .Show = -1 Then

            strFile = .SelectedItems(1)

        Else

            Exit Sub

        End If

    End With

    

    strFind = InputBox(Prompt:="Find what?", Title:="Replace in AutoText")

    strReplace = InputBox(Prompt:="Replace with?", Title:="Replace in AutoText")

    

    WordBasic.DisableAutoMacros 1 ' disable auto macros

    Set objDoc = Documents.Add(Template:=strFile)

    Set objTemplate = objDoc.AttachedTemplate

    With objTemplate.BuildingBlockTypes(wdTypeAutoText).Categories("General").BuildingBlocks

        For nIdx = 1 To .Count

            Set objAT = .Item(nIdx)

            objAT.Value = Replace(Expression:=objAT.Value, Find:=strFind, Replace:=strReplace)

            DoEvents

        Next nIdx

    End With

    objTemplate.Save

    objDoc.Close SaveChanges:=wdDoNotSaveChanges

    WordBasic.DisableAutoMacros 0 ' re-enable auto macros

End Sub
