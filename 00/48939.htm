You do not need to do the Control Panel thing, to change the setting.  Just open the WU window and click settings.

I advise my clients to set WU at Never, permanently.  There is a good reason for that.  If you set it at any other setting, and an update is added that is recommended and checked in the list, and you then shut down, those that are "selected" (meaning checked) will be installed.  You can stop that if you go to WU before you shut down and unselect them, but few people do that.  The only sure way to prevent installs that are unintended is to use the NEVER setting.  Never does NOT mean never update.  It simply means never update automatically.  It is the only way you gain complete control over this process.

  1. A re-install or new install
  2. A monthly update

greginmich say:

Web browsers are crawling with these Tech Support Scam pages – but they’re all fake, because no OS or AV would ever pop up a warning in your web browser. And close to 100% of these scam pages lock up your browser with JavaScript tricks that don’t actually install any malware on your system (that’s why they aren’t detected as malware by AV apps). So you can usually escape their clutches by either:

1. Cicking on the “OK” button in the popup dialog; putting a check in the “Don’t let this page create more messages” box; and then closing the page.

2. Right-clicking on the taskbar (or pressing Ctrl + Alt + Delete) to start Task Manager, and then ending the browser process.

3. But if you kill the Edge browser process with Task Manager, then you’ll have to restart Edge indirectly with a web search in the search box, and then right-click on the search result page tab and select “Close other tabs” in order to remove the scam page.

But after years of trying to educate people on how to escape from these browser lockups, the Tech Support Scammer’s still don’t have any trouble at all finding lots of unsuspecting victims. So minimizing the chances that people will encounter these Tech Support Scam sites has got to be made the focus of our efforts to protect the public from this menace. This would ideally be done with the Windows Defender SmartScreen filter on Microsoft browsers – and also by keeping high-profile websites free of malvertising. But the scammers are easily able to outpace the SmartScreen filter by simply registering new domains at a lightning pace – and we currently have unresolved malvertising issues on quite a few high-traffic sites, including the MSN portals, that are exposing potentially millions of people to malware, crypto mining, and these Tech Support Scams.

At the same time, the scammers have been shifting to multiple-dialog looping and dialog-hiding techniques in order to bypass the Edge Dialog Loop Protection feature (described above) and this has left many Edge users defenseless against these scams. So I think that now would be a good time for shifting the focus of our educational efforts from escaping these Tech Support Scam pages, to preventing them.

The current malvertising campaign on the MSN portals has provided me with the opportunity to see firsthand how a good ad-blocker extension, like uBlock Origin, will selectively prevent any compromised advertising domains from connecting to host sites – and will thus prevent these Tech Support Scam redirects from being piped in to the high-profile sites.

Installing an ad-blocker extension in Edge is safe and easy; because the Settings menu will take you directly to the Microsoft Store, where you’ll find a good selection of ad blockers to choose from. I’m very impressed with the effectiveness of uBlock Origin in mitigating this malvertising issue, and also with the way that it gives us a window into all of the domain connection activity that’s going on behind the scenes at the high-profile sites. But others have reported success with AdBlocker Ultimate, which is also very well-reviewed. And of course these extensions are all free, so if you’re not happy with the first one you pick, then you can just switch over to another one:

For Microsoft Edge, find an ad-blocker extension in the Microsoft Store:

Settings and more > Extensions > Get Extensions from the Store

The documentation for uBlock Origin (and the instructions for installing it on other browsers) is available here:

https://github.com/gorhill/uBlock
