suzanne s. barnhill say:

At File | Print, click the arrow beside Print All Pages and select Key Assignments. That will print the custom ones. For the built-in ones, see https://www.laptopmag.com/articles/see-all-word-keyboard-shortcuts. Or you could print https://support.office.com/en-us/article/keyboard-shortcuts-for-microsoft-word-on-windows-95ef89dd-7142-4b50-afb2-f762f663ceb2?ui=en-US&rs=en-US&ad=US or create a document containing its content.

To create a list using a macro, see https://gregmaxey.com/word_tip_pages/list_all_keyboard_shortcuts.html

And note that all these links were found by googling for "list of Word keyboard shortcuts."
