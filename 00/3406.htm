ken sheridan say:

You might like to take a look at FindWords.zip in my public databases folder at:

https://onedrive.live.com/?cid=44CC60D7FEA42912&id=44CC60D7FEA42912!169

Note that if you are using an earlier version of Access you might find that the colour of some form objects such as buttons shows incorrectly and you will need to  amend the form design accordingly.  

If you have difficulty opening the link, copy the link (NB, not the link location) and paste it into your browser's address bar.

This little demo file uses some custom functions to find words or phrases, as distinct from substrings, in a text column (field) in a table, using T S Eliot's famous poem The Wasteland as an example.  It allows you to search for a single word or phrase, any one of multiple words or phrases, or all of multiple words or phrases.  

While a single phrase search can include a phrase which includes a comma, multiple phase searches cannot of course, as the comma would be interpreted as a delimiter, not part of a phrase.

To use the functions in your own database you'd simply need to copy the basFindWord module from the demo into your own database, and then put code in your own search form modelled on that in the Click event procedures of the three buttons in the demo's search form.

For searching two or more separate fields you don't need to use a UNION query, you can use an OR operation.  With you example for instance you could search the Complaint Description and Investigation Report Root Cause(s) columns for:

FindAnyWord([Complaint Description],"bonnet,bsr") OR FindAnyWord([Investigation Report Root Cause(s)], "bonnet,bsr")

You'll see in the buttons' code in my demo that, when building the string expression, each literal quotes character is represented by a contiguous pair of quotes characters, so the code to build the expression for the above search would be:

strCriteria = "FindAnyWord([Complaint Description],""" & Me.txtFindAny & """)" & _
    " OR FindAnyWord([Investigation Report Root Cause(s)],""" & Me.txtFindAny & """)"

Where txtFindAny  is a text box control in the form in which the value list bonnet,bsr is entered.

ken sheridan say: It depends on whether you want to return rows in which the criteria entered in all of the two or three text boxes in which value lists have been entered are satisfied, or in any one of them.  The first is a Boolean AND operation , the second is a Boolean OR operation.  In the case of the former the code would be like this:

Dim strCriteria As String

If Not IsNull(Me.[txtComplaintDescription] Then
    strCriteria = strCriteria  & " And FindAnyWord([Complaint Description],""" & [txtComplaintDescription] & """)"
End If

If Not IsNull(Me.[txtDescription] Then
    strCriteria = strCriteria  & " And FindAnyWord([Description],""" & [txtDescription] & """)"
End If

If Not IsNull(Me.[txtComplaintDescription] Then
    strCriteria = strCriteria  & " And FindAnyWord([Complaint Description],""" & [txtComplaintDescription] & """)"
End If

If Not IsNull(Me.[txtComplaintDescription] Then
    strCriteria = strCriteria  & " And FindAnyWord([Investigation Report Root Cause(s)],""" & [txtInvestigationReportRootCauses)] & """)"
End If

' remove leading ' And '
strCriteria = Mid(strCriteria,6)

DoCmd.OpenForm "YourFormName",WhereCondition:=strCriteria

If on the other hand the operation is a Boolean OR the code would be like this:

Dim strCriteria As String

If Not IsNull(Me.[txtComplaintDescription] Then
    strCriteria = strCriteria  & " Or FindAnyWord([Complaint Description],""" & [txtComplaintDescription] & """)"
End If

If Not IsNull(Me.[txtDescription] Then
    strCriteria = strCriteria  & " Or FindAnyWord([Description],""" & [txtDescription] & """)"
End If

If Not IsNull(Me.[txtComplaintDescription] Then
    strCriteria = strCriteria  & " Or FindAnyWord([Complaint Description],""" & [txtComplaintDescription] & """)"
End If

If Not IsNull(Me.[txtComplaintDescription] Then
    strCriteria = strCriteria  & " Or FindAnyWord([Investigation Report Root Cause(s)],""" & [txtInvestigationReportRootCauses)] & """)"
End If

' remove leading ' And '
strCriteria = Mid(strCriteria,5)

DoCmd.OpenForm "YourFormName",WhereCondition:=strCriteria

You could of course open a report rather than a form by calling the OpenReport method.
