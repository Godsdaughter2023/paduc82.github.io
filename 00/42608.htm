a. user say:

Hi Stalin,

Thank you for posting your query in Microsoft Community Forum.

I understand that you want to delete a gigantic hiberfil.sys file stored in the root of your drive to free up some space.

In the first place, I would recommend you NOT to delete these files as these are Windows system files. These may seem useless occupying large memory space in your system but they are very useful in their corresponding Windows operations.

Actually, windows has two power management modes that you can choose from: one is Sleep Mode, which keeps the PC running in a low power state so you can almost instantly get back to what you were working on. The other is Hibernate mode, which completely writes the memory out to the hard drive, and then powers the PC down entirely, so you can even take the battery out, put it back in, start back up, and be right back where you were.

Hibernate mode uses the hiberfil.sys file to store the the current state (memory) of the PC, and since it’s managed by Windows, you can’t delete the file.

So, if you do NOT use the hibernate mode and want to delete the related file, you may try the following steps:

  • Press Windows logo key + X and click Command Prompt (Admin).
  • Type the following command and hit ENTER.
    powercfg -h off
  • Close the Command Prompt.

You'll notice that the hiberfil.sys file is deleted from its location in the drive.

Similarly, pagefile.sys is a Windows paging file. This helps in creating virtual memory when the RAM is out of memory and hard disk is used as a "virtual" memory. Deleting this file may lead to 'out of memory' errors. So, you should not remove this file. However, you may use the Disk Cleanup utility to reduce the number of unnecessary files on your drives, which can help your PC run faster.

Refer to the following link to do so:

Delete files using Disk Cleanup

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.

Reregister DLL files that are associated with Cryptographic Services

To register .dll files that are associated with Cryptographic Services, follow these steps:

regsvr32 /u softpub.dll

regsvr32 /u initpki.dll
regsvr32 /u dssenh.dll
regsvr32 /u rsaenh.dll
regsvr32 /u gpkcsp.dll
regsvr32 /u sccbase.dll
regsvr32 /u slbcsp.dll
regsvr32 /u mssip32.dll
regsvr32 /u cryptdlg.dll
exit
