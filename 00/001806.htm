hansv mvp say:

The Criteria argument must be a multi-cell range: the field name in the top cell and one or more conditions below that. Your setup isn't suitable for that.

Instead, use the following array formula in D2, confirmed with Ctrl+Shift+Enter:

=MAX(IF($B$1:$B$30=C2,$A$1:$A$30))

If you wish to use the DMAX function - refer column D. Enter below formula in cell D2 & copy down:

=DMAX($A$1:$B$30,"# of Guests",C$1:C2)

However, in cell D2 the formula will return Max for Room 1, in cell D2 the formula becomes - =DMAX($A$1:$B$30,"# of Guests",C$1:C3) - this will return the Max for both Room 1 & Room 2. And for DMAX you will need the same heading for Criteria (column C) and Database (A1:B30) ie. Occupied Room.

Else, you may use MAX function - refer column E. Enter below formula in cell E2 & copy dow - its a normal (non-array) formula:

=MAX(INDEX(($B$2:$B$30=C2)*($A$2:$A$30),,))
