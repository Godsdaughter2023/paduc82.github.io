If you get a black screen after signing in to Windows 10, there might be a process or service that is conflicting within Windows. Cannot install programs - Microsoft Community

phillipharris2 say:

Cannot install programs

I cannot install programs using my admin log in. I tech was helping me and created a new local admin account on my pc and that account was able to install the program. What do I need to change on my original admin account to allow me to install? The the teach just ended the session without telling me anything. 
junaid_a say:

Hi Phillip,

 

Thank you for writing to Microsoft Community forums.

 

The reason behind programs not getting installed on the main account could be the user profile corruption. Since the new user account is already created, I suggest you to transfer the data from the corrupted user profile to the working one.

 

Copy files to the new user profile:

  • Log on as the user that you want to copy files from.
  • Click the Tools menu, and then click Folder Options.
    (If you don't see the Tools menu, press Alt.)
  • Click the View tab, and then click Show hidden files, folders, and drives.
  • Clear the Hide protected operating system files check box, click Yes to confirm, and then click OK.
  • Locate the C:\Users\Old_Username folder, where C is the drive that Windows is installed on, and Old_Username is the name of the profile you want to copy files from.
  • Select and Copy all the files and folders in this folder, except the following files:

    • Ntuser.dat

    • Ntuser.dat.log

    • Ntuser.ini

  • Locate and Paste the copied content to the C:\Users\New_Username folder, where C is the drive that Windows is installed on, and New_Username is the name of the new user profile you created.
  • Log off, and then log back on as the new user.

 

Note:

If you have email messages in an email program, you must import your e mail messages and addresses to the new user profile before you delete the old profile. If everything is working properly, you can delete the old profile.

 

Also, if you’re using a Microsoft account, you may convert the newly created profile into your Microsoft account.

 

Hope it helps.
