AFAIK this is not possible.

If you want to have big squares you can create you own control using shapes and toggle the check mark with a macro.

Each of this "check boxes" have 2 shapes: A rectangle and a textbox, grouped together. The textframe of the rectangle is formatted as Wingdings size 54.

You can assign the same macro to this shapes, below is a sample.

Andreas.

Sub MyOnClick()
  Dim Parent As Shape, Sh As Shape
  'Get the shape which called us (this can be the rectangle or the textbox)
  Set Sh = ActiveSheet.Shapes(Application.Caller)
  'Get the parent group
  Set Parent = Sh.ParentGroup
  'Find the rectangle inside
  For Each Sh In Parent.GroupItems
    If Sh.Type = msoAutoShape Then
      With Sh.TextFrame2.TextRange
        'Check or uncheck (Note: Textfield is formatted as 'Wingdings'")
        .Text = IIf(.Text = "", "ü", "")
      End With
    End If
  Next
End Sub

Another way using code placed in the worksheet module ...
'---

'No controls used - faux checkboxes using font.
'For cells to have checkboxes they must have 'Wingdings 2' font preset.
'Change the column number to suit.

 Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
 'Jim Cone - Portland, Oregon USA
  If Target.Column <> 1 Then
   'Exit if Doubleclick is not in Column A

  ElseIf Target.Cells(1).Font.Name <> "Wingdings 2" Then
   'Exit if font is not Wingdings 2
    Exit Sub
  End If
 
 'Toggle Checkmarks
  With Target
    If .Value = "£" Then
       .Value = "R"

    End If
   .HorizontalAlignment = xlCenter
  End With
 
 'Exit from edit mode
  Cancel = True
 End Sub
