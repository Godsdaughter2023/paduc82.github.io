Even when you choose to delete your out-of-date security info, MS don't fully remove it for 30 days. This waiting period helps to deter hackers from updating your security info and gaining access to your account without your knowledge.

MSE Installation check List

 

Check your hidden device drivers

How to display hidden device drivers.

 

Before you do the following procedure, check the Services to make sure no old deleted services are still active. Many software utilities do a very bad job of uninstalling.

 

In order to show hidden devices:

 

  1. Right click My Computer and Select Properties
  2. Click Advanced tab
  3. Click Environment Variables
  4. Below the bottom Environment Variables window pane (System Variables), click New
  5. For Variable Name, enter devmgr_show_nonpresent_devices
  6. For Variable Value enter 1
  7. Click OK. OK to close My Computer
  8. Open Device Manager, click on View, select show hidden devices from the menu. Most of the orphans will be in non-plug and play drivers and the icons will appear grayed out. You can right click the grayed out driver and click uninstall.

Some drivers that are NOT grayed out are not needed as well, for example, AVG, Symantec and others. These should be uninstalled.

 

If you do not know what the driver is, check this reference.

 

http://www.sysnative.com/drivers/

 

If it is not in the list bing or google it. Do not delete what you do not know.

doug robbins - mvp office apps & services (word) say:

The following macro was created many years ago for someone who wanted to compile a single document from two separate files.  You may be able to use it to combine your two documents into a single file that can then be printed in the way that you want.

    Dim sourcea As Document, sourceb As Document, target As Document, Pages As Integer, Counter As Integer, targetrange As Range 'targetrange added

    Dim evenpage As Range

    Set sourcea = Documents.Open(FileName:="...")

    sourcea.Repaginate

    Pages = ActiveDocument.BuiltInDocumentProperties(wdPropertyPages)

    MsgBox Pages

    Set sourceb = Documents.Open(FileName:="...")

    Set target = Documents.Add

    target.PageSetup.LeftMargin = sourcea.PageSetup.LeftMargin

    target.PageSetup.RightMargin = sourcea.PageSetup.RightMargin

    target.PageSetup.TopMargin = sourcea.PageSetup.TopMargin

    target.PageSetup.BottomMargin = sourcea.PageSetup.BottomMargin

    target.AcceptAllRevisions

    Counter = 0

    While Counter < Pages

        sourcea.Activate

        ActiveDocument.Bookmarks("\page").Range.Copy

        Set targetrange = target.Range

        targetrange.Start = targetrange.End

        targetrange.Paste

        ActiveDocument.Bookmarks("\page").Range.Cut

        sourceb.Activate 'Assumed to be the document containing the even pages

        Selection.EndKey Unit:=wdStory 'Line of code added to start from the end of the document

        ActiveDocument.Bookmarks("\page").Range.Copy

        Set targetrange = target.Range

        targetrange.Start = targetrange.End

        targetrange.Paste

        targetrange.Start = targetrange.End

        targetrange.InsertBreak Type:=wdPageBreak

        Set evenpage = ActiveDocument.Bookmarks("\page").Range

        evenpage.Start = evenpage.Start - 1

        evenpage.Delete

        Counter = Counter + 1

    Wend

    sourcea.Close wdDoNotSaveChanges

    sourceb.Close wdDoNotSaveChanges

Center text to an image in Office 2007 Word - Microsoft Community

wild child say:

Center text to an image in Office 2007 Word

I have Office 2007 Word and I am trying to figure out how to center one line of text to an image.

I need the line spacing to remain the same as it is here (line 4 starts the main body of the text) but I need the "Prologue" to center against the image to the left of it.  I have tried all the various position options under the Picture Tools but they all seem to move the image into the center of the text body instead of keeping it on the line where it belongs.  Any help will be greatly appreciated.

john korchok say:

Normally, an in-line image will sit on the text baseline. There are several ways to change this:

  • Select the word Prologue, open the Font dialog, select the Character Spacing tab, change the Position: dropdown to Raised and set By: to around 10 points, depending on the exact size of the text.
  • Create 2-cell table, turn off the borders, put the graphic in one cell and Prologue in the other and set both cells to be centered vertically.
  • Create a text box to the right of the graphic, put Prologue in it and move the text box to your preferred position.

Let us know if you need help implementing any of these.
