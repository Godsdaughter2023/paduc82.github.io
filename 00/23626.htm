lucy thomson say:

Hi there

If you set something to Start With Previous, it will be set to start with it at the point in time you set it. If you then move the preceding item (e.g. add a .5 second delay) the second animation will not move to the new start time. That is, it will use the original start time of the predecessor for it's start time. If you re-apply Start with Previous, it will move to the new start time. OK, this is kind of hard to make clear in words :-) Let's repro!

  1. Put 2 things on a slide (thing 1 and thing 2)
  2. Give them both an animation and set both to start with previous
  3. Add delay to thing 1
  4. thing 2's start time will not move, it will start before thing 1
  5. Re-apply Start with Previous to thing 2
  6. thing 2 now starts with thing 1

One of the easiest ways to see how the animations fit together is to use the Animation Pane. I'm sorry I can't remember if that opens automatically in PPT 2007 or not (I'm on 2013) but it should be on the Animation tab somewhere...

Hope that helps,
