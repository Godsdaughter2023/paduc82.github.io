tom ogilvy say:

coffeeman22,

Here is another way to do it.   Make sure you put in the correct worksheet name in the code.   It assumes that there is nothing below your list of names.   This should be faster than looping through the cells in the sheet (but for small data lists, that isn't going to be slow. )

Private Sub UserForm_Initialize()
Dim sh As Worksheet
Dim r As Range, v As Variant, v1 As Variant
Me.ComboBox1.Clear
Set sh = Worksheets("Sheet1")    ' change to your worksheet name
Set r = sh.Range("E2", sh.Cells(sh.Rows.Count, "E").End(xlUp))
v = r.Resize(, 2)
For i = 1 To UBound(v, 1)
   v(i, 1) = v(i, 1) & " " & v(i, 2)
Next
v1 = Application.Index(v, 0, 1)
Me.ComboBox1.List = v1
End Sub
