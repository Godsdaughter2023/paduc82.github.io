Rename temporary folder SoftwareDistribution (instructions are for XP, in Vista/7, you need to get to a command prompt by typing in the search box by the start orb - command, then highlight command.exe and right click, select run as administrator)

4. Click Start, Run, type: cmd and press Enter. Please run the following command in the opened window.

net start WuAuServ

Lumia 1520 can't connect to WIFI unless I reboot the phone - Microsoft Community

janbouman say:

Lumia 1520 can't connect to WIFI unless I reboot the phone

My Lumia 1520 phone can’t connect to a known WiFi connection unless I reboot the phone. Turn on and back off Airplane mode doesn’t work. This happen every time my phone has been connected to another (known) WIFI connection for more than an hour. I use Windows 10 Mobile.

janbouman say:

Searching on forums I found the following solution:

1. Delete all known WIFI access points on phone. 

2. Set time+date to manual

After I took these actions, my phone connects automatically again to know networks without a reboot. Of course I had to reenter passwords or network keys of WIFI access points.
