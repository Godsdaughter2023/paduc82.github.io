rena yu msft say:

Hi Peter,

So, your concern is the files on the OneDrive Online server?

As the forum category here is not focus on the Sysinternals, I just suspect that the SDelete can’t control the files on the OneDrive Online server based on my research about the SDelete. The SDelete should only securely delete the data in your Windows local device and prevent the files to be restored back in some ways. To confirm that, you can use SDelete to delete a test file from your local OneDrive sync folder. Then, the OneDrive sync client will sync the changes from local to Online. The file should be deleted from the OneDrive Online also to the OneDrive Online recycle bin. If you can find the file from the OneDrive Online recycle bin, that means the SDelete can’t affect OneDrive Online server files.

If you want to ensure the files are deleted permanently, you need to delete the files from the Recycle bin > Delete the files from the Second Recycle bin.
