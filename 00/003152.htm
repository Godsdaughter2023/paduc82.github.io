julie sheets say:
Evgeny,

I'm afraid that is not going to be possible with filters.  While you can set multiple criteria (Date Range AND Using Resource for example) you can't mix and match regular filters with highlight filters.

The only option I can come up with is to combine autofilters with using the Marked field. 
  1. Add the marked field into the task view.  By default the field will show "no".
  2. Apply an autofilter on the Start and/or Finish fields to get the date range you need.
  3. Apply an autofilter on the Resource Names field to select the resource you need.
  4. With both filters applied set the Marked field on the tasks to "yes".
  5. On the Format ribbon, choose Text Styles and select Marked from the Items to Change.  Set a background color for the marked tasks.
  6. Clear the filter from the Resource Names field leaving the filter(s) for the date range.

You can fairly quickly change the Marked field using the Fill handle.  No need to change the formatting for the Marked field every time.  It will remain set in that particular view.


I hope this helps.

    Dim colAL As Outlook.AddressLists
    Dim oAL As Outlook.AddressList
    Dim colAE As Outlook.AddressEntries
    Dim oAE As Outlook.AddressEntry
    Dim oExUser As Outlook.ExchangeUser
    Set colAL = Application.Session.AddressLists
    Get_Mobile_By_eMail_from_Exchange_Address_Book = " "
    For Each oAL In colAL
        'Address list is an Exchange Global Address List
        If oAL.AddressListType = olExchangeGlobalAddressList Then
            Set colAE = oAL.AddressEntries
            For Each oAE In colAE
                If oAE.AddressEntryUserType = _
                    olExchangeUserAddressEntry _
                    Or oAE.AddressEntryUserType = _
                    olExchangeRemoteUserAddressEntry Then
                    Set oExUser = oAE.GetExchangeUser
'                    Debug.Print (oExUser.JobTitle)
'                    Debug.Print (oExUser.OfficeLocation)

                    If oExUser.PrimarySmtpAddress = iEmail Then
                        Get_Mobile_By_eMail_from_Exchange_Address_Book = oExUser.MobileTelephoneNumber

                End If
            Next
        End If

1. Activate tab named Panel Layout!
