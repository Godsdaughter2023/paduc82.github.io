I made a test XP machine and I just found the answer. It took me six hours, so I will describe it to save time to other people with the same "problem".

larryyu_msft3110 say:

Hi SatyamGupta,

You can follow the steps below to check if it addresses your concern:

1. Login to EAC (Exchange admin center).

2. Click recipients, select the user who you want to apply the policy to.

3. Click organization, type the same value such as ‘IM only’ in the Department section for all users.

4. Run the command below to apply the policy to all the selected users:

Get-CsUser -LdapFilter "Department=IM only" | Grant-CsConferencingPolicy –PolicyName xxxxxx

Note: Replace ‘IM only’ and ‘xxxxxx’ with your own value.

If this doesn’t address your concerns, there is no Microsoft method to apply the policy to all users at one time. Thanks for your understanding. About this feature, we suggest you submit feedback to the related team via the link below:
