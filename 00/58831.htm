There are several answers to your question:

  • Chkdsk does not perform any conversion. It only checks the integrity of the file system.
  • Rebooting the machine should free up the disk.
  • Storing files on a single medium is risky stuff. Important files must at all times be stored on two independent media. Before you run chkdsk you must copy these files to another disk. A 1000 GByte USB disk costs around $60 and it makes an excellent backup medium.
