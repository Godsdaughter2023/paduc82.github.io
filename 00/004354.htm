karl timmermans [outlook mvp 2012-2018] say:

Which address appears as the default when you open a <new> contact versus a pre-existing one?

Which address appearing as the default for an existing contact is governed by the internal Outlook field <SelectedMailingAddress> which has one of the following values (set when you manually select a default mailing address or alternatively is also set to the first addressblock that gets a value when creating a new contact)

0 = None

1 = Home

2 = Business

3 = Other

You basically have two options for pre-existing contacts

#1 - Write a macro to go through all the contacts and change the value of the <SelectedMailingAddress> field accordingly. You can use the sample bulk FileAs VBA code as shown on

Change Contact's File As format

http://www.slipstick.com/contacts/fileas.asp  

*** just remove the lines related to FileAs and replace them with

       .SelectedMailingAddress = 2 '(to make Business the default)

#2 - If you don't want to go through the above process - you may find the Bulk Field Change function in ContactGenie Toolkit to be of interest (http://www.contactgenie.com/toolkit_4_outlook.htm )

Note that if no address info has been entered for a contact - the value of the SelectedMailingAddress field will be zero but Outlook defaults to showing the <Business Address> as the default when creating a new contact.
