blitzen---01 say: Most of the data cables for the 3220 bought from ebay is based on the Prolific serial2usb chip. Because the 3220 actually has a serial interface (not USB like phones with memory cards) the data cable needs to have a serial to usb converter built in.

To get this to work on XP you first need to connect the cable, then install the drivers for the cable (the prolific serial2usb driver). You should also have Nokia PC Suite installed.

When you install Nokia PC Suite it will ask you to connect the phone after you specified that you are using a Cable connection. It should then pick up the phone.

Make sure that the phone has not yet been added as a modem. To check that go to Phones and Modems in Control Panel and make sure the 3220 is not listed there.

graham mayor say:

For Excel format, you want to save as XLS or XLSX. How useful that will be will depend on what the output looks like when you open the resulting file in Excel. For mail merge, ideally need a header row starting at Cell A1 with a column for each piece of

You therefore need four Mergefields on your envelope

{ Mergefield Company }

{ Mergefield "Address 1" }

{ Mergefield City }, { Mergefield "Postal Code" }

Note that where the field name has more than one word you need to enclose it in quotes..

An alternative approach would be to use http://www.gmayor.com/Envelope_Label_Add_In.htm   This can be setup to use your Excel data and you can arrange the fields in any way you require. Then select the addresses you want envelopes for and the processcreates them.

doug robbins - mvp office apps & services (word) say:

There is no real need to make any changes to the field names (headings) or to Match the Fields.

After selecting the data source, you can just click on the Insert Merge Field dropdown and you will see a list of the field names in your data source, which you can then insert into the address space of the envelope in the configuration that you want them.

The only reason that you would use the Match Fields is if you wanted to use the Address Block, which is something that I have never used.

For a detailed description of the process, see the article “Mail Merge with Word 2007/2013” on fellow MVP Graham Mayor’s website at:

http://www.gmayor.com/merge_labels_with_word_2007.htm

Guidelines to upload a workbook on OneDrive. (If you already use OneDrive and your process for saving to it is different then you can probably start at step 8 to get the link but please zip the file before uploading.)

  1. Zip your workbooks. Do not just save to OneDrive because the workbooks open with On-Line Excel and the limited functionality with the On-Line version causes problems.
  2. To Zip a file: In Windows Explorer Right click on the selected file and select Send to -> Compressed (zipped) folder.)
  3. Do not use 3rd party compression applications because I cannot unzip them. I do not clog up my computer with 3rd party apps when there are perfectly good apps supplied with windows.
  4. Go to this link.  https://onedrive.live.com
  5. Use the same login Id and Password that you use for this forum.
  6. Select Upload in the blue bar across the top and browse to the zipped folder to be uploaded.
  7. Select Open (or just double click). (Be patient and give it time to display the file after initially seeing the popup indicating it is done.)
  8. Right click the file on OneDrive and select Share.
  9. Do NOT fill in the form; Select "Get a Link" on the left side.
  10. Click the button "Create a Link"
  11. Click in the box where the link is created and it will highlight.
  12. Copy the link and paste into your reply on this forum.
