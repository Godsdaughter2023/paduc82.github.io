joeu2004 say: Tony Valko (Biff) wrote:
Still array entered.
=MEDIAN(IF(ISNUMBER(SEARCH(A1,M2:M20)),IF(EF2:EF20>=A2,
IF(EF2:EF20<A3,IF(ISNUMBER(SEARCH(A4,AW2:AW20)),EG2:EG20)))))
I believe the following is slightly more efficient in general (still array-entered):

=MEDIAN(IF(EF2:EF20>=A2,IF(EF2:EF20<A3,
IF(ISNUMBER(SEARCH(A1,M2:M20)),
IF(ISNUMBER(SEARCH(A4,AW2:AW20)),EG2:EG20)))))

Unlike real functions, IF() evaluates only the applicable parameter based on the conditional truth value.  So the searches are avoided if either the ">=A2" or "<A3" test fails.  Likewise, the 2nd search is avoided if the 1st search fails, as before.

PS:  I do not know why this returns a #N/A error for John.  I do get a #NUM error if the IF() expression fails to return at least one of EG2:EG20 because all conditions are not true at least once.

If John would like to guard against that, he could write in Excel 2007 (still array-entered):

=IFERROR(MEDIAN(IF(EF2:EF20>=A2,IF(EF2:EF20<A3,

IF(ISNUMBER(SEARCH(A4,AW2:AW20)),EG2:EG20))))),"")
