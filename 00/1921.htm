As far as I can tell by looking at each version of Word from 2003 through 2016, none of them had that command available to add to a toolbar. An alternative is to install a macro that opens the Recent File list, and add a button to the QAT to run that macro. The code is simple:

Sub FileOpenRecent()
    Application.CommandBars.ExecuteMso ("FileOpenRecentFile")

Install it in your Normal.dotm template or in any template that you store in the folder %appdata%\Microsoft\Word\Startup.

Although you can't see it, the function can easily be added to the QAT.

<mso:control idQ="mso:FileOpenRecentFile" visible="true" insertBeforeQ="mso:FileSave"/>

norman david jones say:

Hi David,

Consider using Excel's AutoCorrect feature in conjunction with  a VBA macro.

You can update the AutoCorrect list manually, entering each acronym and its replacement value once. If you have a master list of acronyms and desired replacements, it would also be possible to update the AutoCorrect list with a macro. If you would like code for this, just ask.

As you are importing the acronym text, Excel's AutoCorrect feature will not automatically effect the desired replacements; this would normally only occur with new manual entries However, the following macro will utilise your AutoCorrect list to update existing

Alt-F11 to open the VBA editor

Alt-IM to insert a new code module

In the new module, paste the following code

'==========>>
Option Explicit

'---------->>
Public Sub Tester()
    Dim WB As Workbook
    Dim SH As Worksheet
    Dim Rng As Range, rCell As Range
    Dim MyList As Variant
    Dim x As Long
    Dim CalcMode As Long

    MyList = Application.AutoCorrect.ReplacementList

    On Error GoTo XIT
    With Application
        CalcMode = .Calculation
        .Calculation = xlCalculationManual
        .ScreenUpdating = False
    End With

    Set Rng = Selection
    For Each rCell In Rng.Cells
        With rCell
            .Select
            For x = 1 To UBound(MyList)
                If InStr(1, .Value, MyList(x, 1), vbTextCompare) Then
                    .Value = Replace(.Value, MyList(x, 1), MyList(x, 2), 1, , vbDatabaseCompare)
                End If
            Next x

        End With
    Next rCell

XIT:
    With Application
        .Calculation = CalcMode
        .ScreenUpdating = True
    End With
End Sub
'<<==========

 Alt-Q to close the VBA editor

To run the macro, with the data of interest selected:

Alt-F8 to open the macro window

Select Tester | Run

If you would prefer not to use the inbuilt AutoCorrect feature, my code could readily be amended to use a master list of acronyms/replacements rather than the AutoCorrect list. The primary advantage of the AutoCorrect approach is that subsequent manual entries/edits to imported text will automatically be amended.
