Are you modifying built-in styles or adding new, user-defined styles? If the latter, I can think of no explanation. For the time being, however, you can make the changes directly in the Normal template and save it. To do this you must open the Normal template itself for editing (just as you would open a document). The easiest way to do this is as follows:

 

In Word, press Alt+F11; the Visual Basic Editor opens. Press Ctrl+G to activate the Immediate window. In that window, type

     NormalTemplate.OpenAsDocument

and press Enter. Normal opens, but note that it may not be on top of all windows. You can close the Visual Basic Editor. Make the desired changes to the Normal file, and then save and close it. If the changes you have made have not been recognized as changes (the Undo button is not activated), press Spacebar, then Backspace to "dirty" the file, thus ensuring that the template will actually be saved.

When you say that only email names and addresses were restored, where exactly are you finding those? Suspect this is in reference to your auto-complete cache - when you start to enter a name or address in the To/CC/Bcc lines of a new email - is that correct?

All actual "contact data" is stored in one or more "contact folders" which can be located in either a PST or OST file depending on your email account tyoe (POP3, Imap, Exchange).

If your contact info was contained in an .ost file associated with an IMAP account and you have not manually backed up, then you have lost that data. This could be true of all data contained in folders labeled as "This Computer Only) - i.e. contact/calendar folders.

If the OST file was associated with an Exchange account, then Outlook will recreate the OST file the next time it connects to the account and sync all the data on the server which includes contact/calendar folders.

In summary, if Carbonite doesn't back up OST files, then you have no other option but to have Outlook recreate those files.

robert__p say:

Hi Kelli,

Here are the answers to your questions:

1. Is there a way to create several inboxes?

The answer is yes; we can create several subfolders for our Inbox folder. Here are the steps to do it:

Sign into Outlook Web App (https://outlook.office365.com/owa) with your Office 365 account. Right click Inbox and select Create new subfolder. Here is the article for your reference:

https://support.office.com/en-ca/article/Working-with-message-folders-in-Outlook-Web-App-6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a

 

2. For example, one inbox for all our staff, one inbox for board members, one inbox for general emails, ect....??

We can use Inbox rules to do this. Here are the steps:

In Outlook Web App, navigate to Settings (the gear icon in the upper-right corner of the page) – Options – Mail – Inbox and sweep rules. Click + to create a new rule. Set “When the message arrives, and it matches all of these conditions” as “It was sent or received – Received from” and add the senders, set “Do all of the following” to “Move, copy or delete – Move the message to folder” and select the subfolder you just created. Click OK at last.

Click + to create more rules for other senders and subfolders. You can also create more rules based on the email content and other credentials for your subfolders. For your reference:

https://support.office.com/en-ca/article/Inbox-rules-in-Outlook-Web-App-edea3d17-00c9-434b-b9b7-26ee8d9f5622?ui=en-US&rs=en-CA&ad=CA
