I tried changing the links to the file location on my computer and it causes both Word and Excel to crash.  I tried creating new documents with new links and have the same problem. I'm at a loss as to what to try next.  Any advice would by appreciated.  Thanks in advance.
macropod say:
Because links to Excel files use an absolute path, they point to where the Excel file was when the link was created. If you link to a file opened from OneDrive, what I suspect you get is a link to a temporary copy of that file. That temporary copy gets deleted when you exit Excel - and your link dies. The same applies if you link to an unsaved Excel workbook - saving it after creating the link doesn't update the link path. If you link to a file saved locally, however, that file doesn't get deleted when you exit Excel - and your link survives.

hansv mvp say:

Try this:

Sub Tables2Text()
    Dim tbl As Table
    Application.ScreenUpdating = False
    For Each tbl In ActiveDocument.Tables
        tbl.ConvertToText Separator:=vbCr
    Next tbl
    Application.ScreenUpdating = True
End Sub

If you don't like the result, you can Undo it.
