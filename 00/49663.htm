Did you make any software/hardware changes to the computer?

As in the error message it is specified that “A recent hardware or software change might have installed a file that is signed incorrectly or damaged.” You can also go through the below article, to know more about Error Code 52.

https://technet.microsoft.com/en-us/library/ff710454(v=ws.10).aspx

I would suggest you to follow the Methods mentioned below

Method 1:

Run inbuilt Network troubleshooter and check if it works. Please follow the steps given below.

Note: When a network problem occurs, the Network Troubleshooter helps to diagnose and repair the problem. The Network Troubleshooter can automatically reset the adapter and driver. It can also detect whether a computer’s wireless networking capability is turned off.

  1. Press Windows + W keys on your keyboard to get the search option.
  2. Type Troubleshooting and click Troubleshooting.
  3. Now click on view all option.
  4. Click on Network Adapter and follow the prompts to run the troubleshooter.

If the issue still persists, follow method 2

Method 2:

How to update driver software, please follow the steps given below.

  1. Go to “device manager”.
  2. Go to “Network Adapters” and expand it.
  3. Now from the list, choose the Network adapter (wireless network adapter).
  4. Right click on your wireless network adapter and choose “Update driver software”.

If updating drivers does not help, I would suggest to uninstall and manually reinstall wireless drivers.
