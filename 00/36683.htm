Method 1:
You may run the "Windows Media Player Library troubleshooter" to check if it makes any difference:
http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-library-troubleshooter
Method 2:

Change settings for ripping music and then try to rip

Note: I would suggest you to change the rip location. To do so refer the following link.
http://windows.microsoft.com/en-US/windows7/Change-settings-for-ripping-music
 
a. Click the Start button, click All Programs, and then clickWindows Media Player.
b. If the Player is currently open and you’re in Now Playing mode, click the Switch to Library button in the upper-right corner of the Player.
c. Click Organize, and then click Options.
d. Click the Rip Music tab.
e. Under “rip to this location” check the location where the ripped files will be stored. If it is not the music library location, then click on change button to change the location
Method 3:
Rebuild windows media player.

a. Exit Windows Media Player.
b. Click Start, type %LOCALAPPDATA%\Microsoft\Media Player, and then click OK.
c. Select all the files in the folder, and then click Delete on the File menu.
Note:

You do not have to delete the folders that are in this folder.
c. Restart Windows Media Player.
Note:

Windows Media Player automatically rebuilds the database.
If this does not resolve the problem, clear the Windows Media Player database cache files. To do this, follow these steps:
a. Exit Windows Media Player.
b. Click Start, type %LOCALAPPDATA%\Microsoft, and then click OK.
c. Select the Media Player folder, and then click Delete on the File menu.
d. Restart Windows Media Player.
Note:

Windows Media Player automatically rebuilds the database.

Method 4:
Disable and enable windows media player. Follow these steps:

a. Open the Control Panel from the Start menu.
b. Open Programs and Features in the Control Panel.

Note: If Programs and Features is not available, change View by to Large icons at the top right of the Control Panel.

c. Select Turn Windows features on or off in the left column of the Programs and Features window.
d. Click on plus sign next to Media Features.
e. Uncheck Windows Media player and then click OK. (You might receive a warning message. Click on yes button.)
f. Restart your computer.
g. To re-enable Media center follow the same steps and place a check mark for Windows Media player.
h. Click on Ok button.
i. Restart your computer.
Reference:
