Not necessarily. The usual reasons for failure to send e-mail messages relate to the port settings and authentication of the SMTP server which is used to send mail.

By default Outlook will configure the SMTP server port to 25, however port 25 is reserved for your ISP's own e-mail service, and will usually be blocked to alternative e-mail accounts. However other e-mail providers are aware of this and usually provide an alternative port (often, but not always port 587).

Check your e-mail provider's port and authentication requirements from their web site and change the settings for the account accordingly.

What am I missing?

andreas killer say:

You can't use direct cell references (like =Sheet1!A1) because this references are not updated if Sheet1 is sorted.

So both sheets need an unique ID, the simplest scenario is a number, then you can pull the data from the other sheet using INDEX and MATCH or VLOOKUP, simple example:

Sheet1:

Sheet2:

B2:  =VLOOKUP(A2,Sheet1!A:B,2,0)
C2:  =MATCH(A2,Sheet1!A:A,0)
D2:  =INDEX(Sheet1!B:B,C2)

Use VLOOKUP if you want to pull just one value, otherwise use MATCH / INDEX because it's faster for more values.
