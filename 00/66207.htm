Rephrase:  Every version of Windows since Windows NT 4.0 has saved the last opened-documents and kept them in a list in the Start menu.  The only difference in Windows 7 is that it shows the recent documents grouped by the application, and not in a giant combined list.  But the actual tracking is not new, so no performance is lost in Windows 7.

Just like every version before it, you can disable the tracking of recent files and documents.  This doesn't disable jump lists (because those are nothing more than right-click command menus), but this should help you:

1 - Rightclick the Start button and choose Properties.

2 - Unselect the "Store and display recently opened items in the Start menu and Taskbar.

Click OK.  Recently opened files will no longer show in the jump menus.  The menus themselves are code inside the actual applications -- not Windows -- so they use absolutely no processor cycles or memory unless engaged by the user.  For example, iTunes was specifically programmed to have specific tasks in its Jump List.  If you don't use iTunes (or that feature of it), it will not impact your performance at all, because that code will never be run.
