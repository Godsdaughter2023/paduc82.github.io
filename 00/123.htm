jay freedman say:
As general definitions:

  • An index is a list of words or phrases (usually but not always ones that appear in the text of the main document), together with the page number where that information is found in the text.
  • A glossary is a list of words or phrases, each with a definition.

It's possible, but not common, to combine the two lists.

Microsoft Word has special tools for creating an index in a document. The help topic for that is at
http://office.microsoft.com/en-us/word-help/create-an-index-and-update-an-index-HP001226499.aspx

Word doesn't have any special tools for creating a glossary. The usual method is to start a new page (with a page break or a section break) and then, for each term that needs a definition, type the term in bold font followed by the definition in regular font. For each term, start a new paragraph.

It's possible that the document contains some corruption, maybe as little as one byte set to the wrong value that's telling Word that tracking is turned on. Unfortunately, such things are difficult to find.

Here are two things to try, with no guarantee of success. It would be best to try them on a copy of the document file, not on the original.

  • With the document open, click File > Info > Check for Issues > Inspect Document. In the dialog, uncheck all the boxes except the first one, "Comments, Revisions, Versions, and Annotations". Click the Inspect button. When the result page appears, if it says that revisions were found, click the Remove All button. Then press Esc to return to the document, and see whether you can save without the message.
  • Open a new blank document. In the problem document, click the ¶ button to turn on nonprinting characters. Then select everything except the very last paragraph mark of the document, copy it, and paste it into the blank document. Try saving the result.

The final paragraph mark "contains" most of the invisible data that affects the document as a whole, as well as the headers and footers of the last or only section.
