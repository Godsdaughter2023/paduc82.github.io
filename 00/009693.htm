brian tillman [outlook mvp 2007-2019] say:

Searching for PSTs will fail unless you enable viewing hidden files and folders in Windows.  The default location for PSTs is %UserProfile%\Local Settings\Application Data\Microsoft\Outlook, which is a hidden folder.  You can see into that folder without enabling hidden files and folders by opening Windows Explorer and enter the exact path that I've just given you, then pressing Enter.

OSTs are not PSTs and even though they have a similar internal structure, they cannot be used the same way.  Hotmail accounts are two-way and all the data is kept on the server.  The OST is a cache that contains a local copy of what's on the server so that access can be faster and you can still see data even when offline.

After deleteing the folder, your Deleted Items folder in the Hotmail folder set should have acquired a plus sign to its left.  By expanding that plus sign, you should have been able to see the folder, which you could then have dragged back to its rightful location.  If you did not do that and you exited Outlook, I suspect your deleted data was removed permanently when  the Deleted Items folder was emptied, but it can't hurt to check anyway.  If it was deleted, there is no way to recover it.

Hi,

  1. On the Home tab, click New Slide in the Slides group and then select Reuse Slides....
  2. Click the Open A PowerPoint File link in the Reuse Slides taskpane to the right.
  3. Locate the presentation that contains the slides you want to reuse.
  4. Click the slide you want to reuse or choose Insert All Slides.
