luis mahiquez say:
Hello
I am Luis, Windows expert here to help you

1- Disable update on restart option using Group Policy
Sometimes Group Policy Editor (gpedit.msc) is a life saver in these types of situations. There is an option in Windows Group Policy “Do not display ‘Install Updates and Shut Down’ option in Shut Down Windows dialog box“. Enabling this option will make Install updates and shut down option disappear. The normal Shut down only option will appear.
This is a permanent option. This means that the shut down and update or restart and update options will not appear in the Shut down dialog box. For installing update and restarting the computer, you will need to go to Settings –> Windows Update and select the install updates and restart from there.
The option is located here:
Computer Configuration –> Administrative Templates –>Windows Components –> Windows Update


2- Bypass update on restart/shutdown process using command line
If you want to bypass update on restart/shutdown process temporarily, you may do it using command line.
Basically we have to stop the Windows update service before restarting or shutting down the PC. We do it like this:
Go to Run –> net stop wuauserv
This will stop the Windows Update service.
Go to Run –> shutdown -s -t 0
This will shutdown the system immediately. If you want to restart the system, give the following command:
Go to Run –> shutdown -r -t 0
Or you can run both the commands at the same time like this:
net stop wuauserv && shutdown -r -t 0
This command will enable you to shut down or restart your system without installing Windows Updates.

the black "Acer" screen shows for about 3 seconds and that's it. The screen goes blank and black except for the light showing at the edges (no writing, no movement, just "background" lighting).  

This scenario can actually represent having the real monitor as the second of a Display which is in Extend mode, with the information that should be seen on your monitor showing only on another, possibly phantom, monitor.  Then to workaround under that hypothesis you would simply use your keyboard as if you could see stuff on the other monitor.  E.g. press Win-P, CursorUp, and Enter.  Be very observant while doing this, e.g. watch and listen for changes.  At the very least you could expect the screen to refresh, e.g. go completely black and then turn on again.  It would be even better to have another monitor to hook up and see if there actually is something being written through that port.

start again and quickly press "Fn" + "F2" or "F3"  

That "quickly" is very important.  My impression is that it is more like simultaneously press the Power on switch and press the Fn key.  In fact, the timing is so critical that some manufacturers have started providing a second Power on switch just to help with it.

Oh.  It's UEFI not BIOS.  That makes it more interesting and less understood.  Here is someone showing you that you could test Linux (if you can't boot either Windows or your bootable recovery media):

http://community.acer.com/t5/Notebooks-Netbooks/Advanced-BIOS-settings-for-Aspire-E1-522/td-p/151267

(BING Search for
    Aspire E1-522 site:acer.com user guide bios
)

Note that even Linux may have trouble with the video

<quote>

</quote>

So that would make me think that your best bet for getting going normally would be to boot from bootable media and specify Low res video for your Advanced boot option (or use bcdedit to do that more persistently.)

andre da costa say:
Summary
If you have multiple Windows licenses and have one installed but don't know which one is in use, use the following instructions to double check.
Details

Apples to: Windows Vista, Windows 7, Windows 8 and Windows 8.1

User Level: Beginner to Expert

If you have multiple PC's in your home or business running different versions of Windows and you would like to have them on the same version, you might want to check to make sure you are not using a product key that is already in use. For instance, if you bought 3 copies of Windows 7 Professional, installed one, activated it, but later wanted to upgrade another PC but can't remember which key was used, this quick tip can be very handy.

Click Start, type: cmd

Hit Enter on your keyboard

At the command prompt, type the following command:

slmgr -dli


Hit Enter on your keyboard

A dialog window will appear on screen. This will reveal the partial product key that will give a hint of the product key that is in use. Compare it with the product keys you have and it will confirm which copy is in use. When you are finish, click OK and close the Command Prompt window.

For Windows 8 and Windows 8.1 users:


Press Windows key + X

Click Command Prompt

Alternative ways to quickly launch the command prompt quickly on all versions of Windows:

Press Windows key + R

Type: CMD

Hit Enter on your keyboard

For Windows XP:

Here are four utilities, all of which will may display your product keys if they are still there:
Belarc Advisor: http://www.belarc.com/free_download.html
(It does a good job of providing a wealth of information.)

Also: http://www.magicaljellybean.com/keyfinder.shtml
and: http://www.nirsoft.net/utils/product_cd_key_viewer.html
and RockXP: http://www.majorgeeks.com/download4138.html which has additional features

azeez n say:

Hi,

I'd suggest you follow these steps and check if it helps.

Step 1:

Most laptops have a function key (FN Key) which can be used in combination with other keys for e.g. FN + F2 or FN + F7 to disable or enable the touchpad.


Since you were unable to disable the touchpad from control panel, you may disable it from device manager with your own risk and check if it helps you.

To disable touchpad in device manager, follow the steps below:

   a. Click on Start.
   b. In the start search box type "devmgmt.msc" and press enter.
   c. In device manager expand mice and other pointing device and locate touchpad.
   d. Right click on the touch pad and click on disable option.

Following the above steps should help you disable the touchpad.

Step 2:

I'd suggest you put your computer in a clean boot state and check if disabling does not help. There could be a third party software interference causing the issue to occur. Putting your computer in a clean boot state helps eliminate third party software conflict.

Note: Follow step 7 to restart your computer as usual.

I&#39;m prompted to download an update every time I insert - Microsoft Community

greypilgrim-ce541714-d291-410c-84d6-81a93517b4bd say:

I&#39;m prompted to download an update every time I insert a game disc.

First off don't worry! This is not usually a sign there is something wrong with your console or memory device.

Having to repeatedly download updates every time you play any game is usually a symptom that your cache is full. Because there is only a finite amount of space made available for updates, there is simply no room left to save these downloads. Usually this occurs when you have played a lot of different titles over time and have never cleared the cache before.

The first step in fixing this issue is to clear your cache.

(Just to let you know beforehand, you will not lose your profile, game saves, multiplayer stats or any game-related DLC you have on your memory device. Sometimes gamer pictures can be stored on the cache, so you may need to re download those from your 'download history' after clearing).

Now to clear your cache:

Sign in on your console and on the 'Home' hub of the dashboard and scroll to the far right, then select the 'Settings' hub. 
 

Then select the 'System' tile in the upper left. 


From here scroll down x2 and select 'memory.'

Highlight the HDD and press 'Y', then select 'clear system cache.'

(A message will pop up saying 'This will perform maintainance on your xbox 360 storage device, do you want to continue?'

Select 'Yes.' 

Now that you've cleared the cache, pop a game in your console and install the update when prompted. You will need to perform this for every title you play after clearing, but it should only be the once per title now (unless future updates are released).

If you're still having to constantly re-download updates then please make a post in the forums, but this has always resolved the issue when I have experienced it myself.Space.
