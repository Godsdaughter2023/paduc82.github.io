kosh vorlon say:

Hi crashater,

csrss.exe - You cannot end this process from Task Manager.

  • This is the user-mode portion of the Win32 subsystem (with Win32.sys being the kernel-mode portion). Csrss stands for client/server run-time subsystem and is an essential subsystem that must be running at all times. Csrss is responsible for console windows, creating and/or deleting threads, and some parts of the 16-bit virtual MS-DOS environment.

It's not in startup because if it were then people might actually turn it off and that would be very bad.

If you have more than one running in task manager for the same user (meaning it's normal to see one for your login user and one for SYSTEM - but as different users {even if one is the computer itself} and your view/tab is set to show all users this would be expected), or if they or others are not located in %Windows%/System32, there's a possibility you may be infected.

If so, post back and we'll discuss how to deal with that.  I do not want to go into greater detail without more information.  Otherwise it's not only normal but required. 

I'd also add that this is a commonly reported problem because it is misunderstood, but very rarely turns out to be an issue when more thoroughly analyzed.

I hope this helps.

Good luck!

suzanne s. barnhill say:

Create a new document, insert a page number, click on "different page", put in as many paragraphs as needed to make the first page header the correct size, same with the footer, save as .dotx. Voilà.

I'm still not quite clear about your comment:

use the Spacing before and after the paragraphs in the First Page Header to force the text on the first page of the document to be clear of the letterhead on your paper.

The method you used will work, of course, but it is considered more elegant just to format the existing empty header paragraph to do the job. Say your letterhead is 2" deep. So you want the first line of text on the page to be at least 2¼" or 2½" from the top of the page, allowing for a quarter or half inch between the letterhead and the text. For determining this, it's useful to have the At: x" indicator on the status bar, so right-click on the status bar and click "Vertical Position."

If your header margin is the default 0.5", then the default empty paragraph in the header will be At: 0.5". So you need to add, say, 1¾" Spacing After. Right-click on the header paragraph and choose Paragraph from the context menu. In the Paragraph dialog, under Spacing, in the After box, select the "0 pt" and type in 1.75" or 1.75 in and click OK. This will put the first line of text at 2.4" (because the baseline of the header paragraph is actually below 0.5"). If that's close enough, you can leave the paragraph spacing as is. If you want to fine-tune it, you can return to the Paragraph dialog. You'll see that Word has converted your inch measurement to 126 points. You can enter another measurement in inches or change the number of points. There are 72 points to the inch, and the spin box in Word changes the Spacing Before/After in increments of 6 points, though you can type in any amount you want.
