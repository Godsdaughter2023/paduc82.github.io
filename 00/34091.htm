Hi Bruce,

To answer your question, you cannot save anything else on your USB flash drive since it was already used for Media Creation Tool. Using this flash drive for MCT will turn it into a boot type format device, and regardless of its remaining free size, you will not be able to use it to store any other files.

Windows Vista Home Basic > Windows 7 Home Basic, Premium or Ultimate

Windows Vista Business > Windows 7 Professional, Enterprise or Ultimate

Windows Vista Enterprise > Windows 7 Enterprise

Windows Vista Ultimate > Windows 7 Ultimate.
