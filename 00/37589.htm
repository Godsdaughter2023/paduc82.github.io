gerry c j cornell say:

To set a page file locate the drive in the box in the Virtual Memory window. Place the cursor on the drive where you wish to create a page file. Check the box before Custom size, System managed size or No paging file. If you have checked Custom size you should enter an Initial size and a Maximum size. Most users who adopt a custom size set the Minimum and the Maximum at the same figure. You then click Set, OK and the Apply button in the next window. You have to restart the computer to apply the new setting.

 

Before creating a custom size page file you should fully defragment the partition where the page file is to be created. It can be difficult to create a fixed size contiguous page file where the existing partition has less than 50% free disk space. The system places the page file in the middle of the drive and if files are already where the page file is to be created the new pagefile is invariably fragmented. You need a third party defragmenter, like Defraggler, to be able to see the situation. The Defragmenter provided with Windows 7 has a poor user interface, does not defragment large files and misreports file fragmentation.

 

If you are changing page file arrangements it helps to set a No pagefile, restart your computer, clean up unwanted files and defragment before creating a fixed pagefile. You can also remove large data files temporarily from the partition, if you need to maximise the available free disk space to create a contiguous fixed page file.

 

If you are creating a System managed page file you do not need to defragment before creating the page file. However, the result of using a System managed file is that the page file is for ever causing file fragmentation in a way that a fixed page file does not.

 

Where you have two internal drives a dedicated page file partition at the beginning of the second hard drive with a small page file on C can be a simpler solution to a fixed size page on C. If you are creating a page file on a partition other than C you should also create a pagefile on C of say 2gb.

jordana_s say:

 

Hi Steffen.

Yes, you are able to change the language on your Surface and input method of the keyboard.  The initial set up will ask you to select the language you prefer; however you can select another language at anytime through your settings option. (See instructions below) 

  1. Swipe in from the right edge of the screen, and then tap Settings. (If you're using a mouse, point to the lower-right corner of the screen, and then click Settings).
  2. Tap or click Change PC settings.
  3. Tap or click General.
  4. Scroll to Language
  5. Tap or click Language Preferences

NOTE In order to see the changes made, you will need to restart your Surface.
