doug robbins - mvp office apps & services (word) say:

To use latebinding, remove the reference to the Word 12 Object Library and use the following code

    Dim appWord As Object
    Dim docWord As Object
    Dim strTemplatePath as String
    On Error Resume Next
    Set appWord = GetObject(, "Word.Application")
    If Err Then 'Word was not running
        Set appWord = CreateObject("Word.Application")
    End If
    'Get the path to the User Templates Folder
    strTemplatePath = appWord.Options.DefaultFilePath(2)
    'Determine which type of document (Agenda or Schedule) the user is creating
    appWord.Visible = True
    appWord.Activate
    'Create a new document from the required template
    Set docWord = appWord.Documents.Add(strTemplatePath & "\templatename.dot")

Note, you cannot then use the wdConstants such as

    strTemplatePath = appWord.Options.DefaultFilePath(wdUserTemplatesPath)

and must replace the wdConstant with its numeric reference as is done above. You can get the numbers by typing enumerate [whateveritis] into the Visual Basic Help facility in Word.

Rod is very passionate about not using linked structures in Project.
