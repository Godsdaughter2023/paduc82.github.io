Sub test()

Dim vispage As Visio.Page
     Set vispage = Application.ActivePage

Next visShape

End Sub

The other version of Windows you may have installed will assume subsequent drive letters in turn when using this type of configuration. So for example, let say we have a dual boot system using Windows 10 and Windows 7. If window 10 was selected at boot up time, it will show itself as being on drive "C." Let's say you now now restarted your computer and selected Windows 7, it would now assign itself as being on drive "C." In both cases, the other operating system not selected will be assigned to the next available drive letter such as "D" or whatever is next available depending on how many versions of Windows you have installed on your system. diane poremsky mvp (slipstick.com) say:

While you can export the accounts from the registry, the registry location changes in Outlook 2013 - the reg file could be edited but the profile is likely to be corrupt because a lot of machine specific data is stored in the profile.  Use Auto account setup and let Outlook 2013 set the accounts up for you.

 

older versions:

HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Windows Messaging Subsystem\Profiles\

 

Outlook 2013 profiles:

HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Outlook\Profiles

 

The steps at Moving outlook to a new computer apply to both Outlook 2010 and 2013.
