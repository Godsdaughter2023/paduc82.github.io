Change paper size in Word 2016 for Mac - Microsoft Community

yvonne lance say:

Change paper size in Word 2016 for Mac

How do I change default paper size from letter to A4 on Word 2016 on a Mac? I have Office 365.
john mcghie say:

Yvonne:

Mystifying, isn't it!

  1. Go to File>Page Setup (File menu, it's not on the ribbon...)
  2. Choose A4 as the paper size
  3. From the Settings drop-down, choose Save as Default

Good luck.  Word 2016 currently has a bad habit of "forgetting" these settings, but that should hold for you.

You may wish to update your Normal template as well.  To do so:

  1. Open the Normal.dotm template (Note: You MUST navigate to the Normal template using the File menu in Word, and choose "Open".  Don't try it from the Finder or you will make a copy of Normal each time.)
  2. Close all other documents (this is important...if you don't, Word holds a hidden instance of the old version open).
  3. Change the paper size.
  4. Add and remove a space from the text (to force Word to save the document)
  5. Save the Normal template.  (If Word prompts you for a file name, you did not open the file correctly and you now have a copy that is not a template.  Cancel out and start again...)
  6. Close Word.

Should be good when you re-open.

Hope this helps

Open the Format Paragraph dialog (tiny icon in the bottom right of the Paragraph group on the Home tab) and:
  1. Set the line spacing to something small like Single
  2. Set Spacing Before (and/or After) to something large like 12pt
  3. Uncheck the 'Don't add space between paragraphs of the same style'

stefan blom say:

As "Guessed" wrote, you will have to clear the "Don't add space..." option rather than select it.

If you frequently create numbered or bulleted lists like that, you can make use of the fact that Word (by default) applies the List Paragraph style to list items. Modify the style so that the "Don't add space..." option is cleared: In the Apply Styles pane (Ctrl+Shift+S), type in List Paragraph, and then click Modify. Click Format, and then click Paragraph. Clear the option. Click OK. Before clicking OK to close the Modify Style dialog, select "New documents based on this template."
