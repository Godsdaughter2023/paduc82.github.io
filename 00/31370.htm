bob headrick say:

There is a driver for the PSC 2110 supplied by HP that provides printing and basic scanning capability.  It is distributed through Windows Update and generally installed as shown in the previous links. There are some steps that may be out of order in HP's page describing how to apply the driver - to be successful you need to have Windows Update enabled before connecting the USB cable.  Windows Update is enabled as shown in the section titled "Configure Windows Update to automatically search for updated drivers". Click on the title to expand that section and show the directions.

Alternately, you can manually check to see the driver as follows:  go to the Devices and Printers folder, Add a Printer, pick a port such as LPT1: temporarily, Next and when the list of printers comes up click on Windows Update in the lower right.  With a high speed connection it will take several minutes to repopulate the list.  When the list is repopulated select HP as the manufacturer (not Hewlett-Packard), the driver you want is the HP PSC 2100 series toward the bottom of the list.  If this driver properly appears cancel the current installation and then plug in the USB cable and everything should happen automatically.

If for some reason you cannot use the Windows Update drivers you could also download the driver from Microsoft's Catalog Site.  This will require that you use Internet Explorer as a browser and you will need to manually find the driver you need as the search does not directly limit results my architecture.  For your x64 system you will want the 11.6Mb file from this page.  Download the file and extract it to a temporary directory, then use Add a Printer, Have Disk and point to the driver to add it to the list, then connect the USB cable.
