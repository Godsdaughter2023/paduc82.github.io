I want my office 2007 back - Microsoft Community

kenlindemann say:

I want my office 2007 back

Brand new computer.  I installed office 2007.  Now the computer has upgraded to a trial version of the latest version of office.  How do I get my office 2007 back and working.  

graham mayor say:

The trial should not have removed Office 2007, but it will have taken control of the links and the shared registry values. The application itself should still be present and usable.

You can establish that by either checking Windows Control Panel > Uninstall a Program and see if it is still in the list, or by examining the folder structure for C:\Program Files (x86)\Microsoft Office\Office12 which should contain the program files, to which you can re-establish links if the links have actually been removed.

It is possible to have both versions of Office, but it is necessary to ensure their shared components are separated as far as possible, and it is likely that the trial will become the default version after each Windows update.

If you have no plans to use the latest version now, then uninstall the trial and repair Office 2007, which should put things back as they were. If you want to keep the trial, see http://www.gmayor.com/Toolbars_in_word_2007.htm which covers the issues related to getting two or more Word versions to work together.

The standard Excel text box supports formatting:

Sub MakeTB()
    Dim wsh As Worksheet
    Dim tbx As Shape
    Set wsh = ActiveSheet
    With ActiveCell
        Set tbx = wsh.Shapes.AddTextbox(Orientation:=msoTextOrientationHorizontal, _
            Left:=.Left + .Width, Top:=.Top + .Height, Width:=100, Height:=50)

    With tbx.TextFrame2.TextRange

        .Characters(5, 5).Font.Bold = True
