Because the Windows XP OEM license will be use as a qualifying base to upgrade to Windows 7 Upgrade, it will be tied to it.

OEM versions of Windows 7 are identical to Full License Retail versions except for the following:

- OEM versions do not offer any free Microsoft direct support from Microsoft support personnel

- OEM versions cannot be used to directly upgrade from an older Windows operating system

dominicp say:

The licence for an upgrade edition of Windows 7 requires only that you own a qualifying product, not that the qualifying product is installed. The upgrade licence also specifies that once the upgrade is installed the qualifying product may not be reinstalled.

The upgrade installation instructions try to impose a requirement not sanctioned by Microsoft's licence terms - that the qualifying product be installed. A reinstallation of the upgrade following the installation instructions would involve a breach of the licence terms - reinstallation of the qualifying product when this is prohibited by the upgrade licence terms.

You qualify for an upgrade edition of Windows 7 Professional and can install it using Clean Install Windows 7 with Upgrade Media. The licence for an upgrade is a retail licence and is transferable to a new machine. The upgrade licence replaces the licence for the qualifying product and does not inherit any restrictions the original licence may have had.
