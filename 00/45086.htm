There's two ways that I use.

Firstly, if you use Wi-Fi, you can set your connection to Metered Connection. (Which is useful at work when your boss asks you why you wasted 3 gigs of company data on updates cos Microsoft thinks they know better than you)

Go to Settings>Network & Internet>Wi-Fi>Advanced Options

Then select Metered Connection for the Wi-Fi you are currently connected to.

Windows will then know you are on a connection that is limited or costly and won't auto-update.

What's really stupid is you can't do this option for physical ethernet cable connections. Somehow microsoft doesn't realize that most of the world doesn't have unlimited bandwidth.

Search for Services.

Then scroll down to Windows Update.

Right-click Windows Update>Properties.

Set Startup type to Disabled.

Then hit apply.

Then hit Stop.

You should turn it on from time to time and open updates to make sure you haven't missed any important updates.

                        net start wuauserv

                        net start cryptSvc

                        net start bits

                        net start msiserver

5. Type Exit in the Command Prompt to close it and then restart the computer.

                        net start bits

                        net start msiserver
