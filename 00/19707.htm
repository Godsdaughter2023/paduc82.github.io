  • Import-RMSTrustedPublishingDomain -RMSOnline -name "RMS Online"
  • Set-IRMConfiguration -InternalLicensingEnabled $true

The key to understanding the product key page is in the word "re-installation" that is used there.  Microsoft now considers any second and subsequent installation of the same edition as a "re-installation" no matter how it is done.  Clean installation, upgrade installation, parallel installation (multi-boot); all are re-installations once the device has been registered.  The wording in the text at the top of the product key page says that if you are reinstalling Windows click the link at the bottom.  Most folks are missing the significance of the wording.  It is a bit of a subtle difference from the common usage of the word in previous versions.

hansv mvp say:

Assuming that the option button labeled "No" has Option Value 1 and the option button labeled "Yes" has Option Value 2, set the Control Source of the option group (the frame) to


=IIf(IsNull([MyTextBox]),1,2)


where MyTextBox is the name of the text box.


Remark: the value of the option group is not updated while you type in the text box. It will be updated as soon as you tab out of the text box or click in another control.

quietman7 say:
Keep in mind that not all programs show up in Add/Remove Programs or Programs and Features. Some will have their own uninstaller, which can be found in the program's folder. This is by design as some vendors write installation routines so their program will not be included in the Add/Remove list.

If the program is not listed in Programs and Features (Add/Remove Programs) look for an uninstall icon (shortcut) from within its program group. Click on Start > All Programs (or Program Files), look for the program folder by name and use the program's uninstaller. If it's not there, the next step would be to open Window's Explorer, manually navigate to the program's folder itself, look for an uninstaller file (i.e. uninstall.exe) and double-click on it to run.

The easiest way to do this is to press the WINKEY + R keys on the keyboard or click Start > Run..., and in the Open dialog box, type: %ProgramFiles%
Click OK or press Enter to open C:\Program Files.
Also check for a readme.txt file which may contain removal instructions.

If there is no entry in Programs and Features (Add/Remove Programs) and no uninstall file in the program's folder, then it most likely is a stand-alone application which can be removed by manually deleting the folder itself (which deletes everything inside it). Again just right-click on it and choose delete. When done...be sure to empty the Recycle Bin.

JRT (Junkware Removal Tool) by thisisu is a stand-alone tool which does not require installation so there is no uninstaller. To remove JRT, just manually delete the icon on the Desktop by right-clicking on it and choosing delete.
