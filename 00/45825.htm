¡firedog say:

Windows Live Mail Error ID: 0x8007007A

The fault arises because of changes to OneDrive, where the photo album you tried to share would have been stored. The function doesn't work any more, so the message you tried to send is now stuck in your outbox and is preventing other messages from being sent. First, remove any unsent messages from the outbox (below the account folders in the folder pane). You may have to disconnect from the Internet to be able to empty the outbox. You don't have to delete the messages in the outbox; you could just drag them to a drafts folder to save having to compose them again.

When the outbox is empty, compose a new test message with a unique subject and send it to yourself (i.e. from the account that's misbehaving to the same account). Post any error messages in your reply. You can click on an error message to select it, then use right-click to copy it and paste it into the text entry box.

 

When you reply, please give your build of the program and your version of Windows.

  • To find your program version and build, in Windows Live Mail press Alt-F B.
  • To find your operating system, press Windows key + R, type winver into the box and press Enter.

The error doesn't have anything to do with the SMTP server involved nor with Windows Live Mail. It only occurs because the 'special relationship' between Windows Live Mail and OneDrive has been broken at the OneDrive end. You would see the same error whichever mail server you were connecting to.
