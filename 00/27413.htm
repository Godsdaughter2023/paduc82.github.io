joeu2004 say: fin_modler wrote:

Yes, apparently you did not read the VBA Mod help page.  It states:

``Usually, the data type of result is a Byte, Byte variant, Integer, Integer variant, Long, or Variant containing a Long, regardless of whether or not result is a whole number. Any fractional portion is truncated. However, if any expression is Null, result is Null. Any expression that is Empty is treated as 0.``

That is, the VBA Mod function is really only useful with integer operands.  You are correct to use the Int expression instead for non-integers and for integers that might be larger than type Long can handle.

graham mayor say:

The short answer is yes it can be done, but there is some confusion as to where you want to move messages to. It seems you are moving them to where they are already located.

However the following will allow you to pick the target folder and set the number of days you want to process.

Option Explicit

Sub MoveSent()
'Graham Mayor - http://www.gmayor.com - Last updated - 09 Jul 2018
Dim olItems As Outlook.Items
Dim olItem As Outlook.MailItem
Dim i As Long, j As Long
Dim olFolder As Outlook.Folder
Dim iDays As Long

    iDays = 1 'the number of days you want to process
    
    MsgBox "At the next prompt, pick the folder where you want the messages moved to"
    Set olFolder = Session.PickFolder

    Set olItems = Session.GetDefaultFolder(olFolderSentMail).Items
    olItems.Sort "[Received]", True
    j = 0
    For i = 1 To olItems.Count
        Set olItem = olItems(i)
        If Format(olItem.SentOn, "yyyymmdd") > _
           Format(Date - iDays, "yyyymmdd") Then
            j = j + 1
        Else
            Exit For
        End If
    Next i
    For i = j To 1 Step -1
        Set olItem = olItems(i)
        olItem.Move olFolder
    Next i
    MsgBox j & " messages moved to " & olFolder.Name
CleanUp:
    Set olItems = Nothing
    Set olItem = Nothing
    Set olFolder = Nothing
lbl_Exit:
    Exit Sub
End Sub

stefan blom say:
You can use a macro to convert auto numbers to text. Make sure the document you want to convert is the active document. Press Alt+F11 to display the Visual Basic Editor. Press Ctrl+G to activate the Immediate window. Type

ActiveDocument.ConvertNumbersToText

and press Enter. All auto numbers, including LISTNUM fields, will be converted to static text. Note that if numbering was applied with styles, you have to remove the numbering from the styles as well; otherwise it may resurface, for example when you press Ctrl+Q to reset paragraph formatting.
