If you simply want a button on the worksheet that users can click then, assuming you have Outlook, put an activex button on the sheet from the developer tab. Right click the button and select View Code.

Paste the following. Change the bold bits to taste

Option Explicit
Private Sub CommandButton1_Click()
ThisWorkbook.Save
CreateEmail "The Excel workbook " & ThisWorkbook.Name & " has been updated."
End Sub

Private Sub CreateEmail(MessageBody As String)
Const olMailItem = 0
Dim olApp As Object
Dim olMail As Object        ' Outlook.MailItem
Dim olInspector As Object        ' Outlook.Inspector
Dim wdDoc As Object        ' Word.Document
Dim wdRange As Object        ' Word.Range

    On Error Resume Next
    Set olApp = GetObject(, "Outlook.Application")
    If olApp Is Nothing Then Set olApp = CreateObject("Outlook.Application")
    If olApp Is Nothing Then Exit Sub
    On Error GoTo 0

    Set olMail = olApp.CreateItem(0)

    With olMail
        .To = "*** Email address is removed for privacy ***"
        .Subject = ThisWorkbook.Name
        .Display
        Set olInspector = .GetInspector
        Set wdDoc = olInspector.WordEditor
        Set wdRange = wdDoc.Range(0, wdDoc.Characters.Count)
        wdRange.InsertBefore (MessageBody)
        .send
    End With
End Sub

Hi,

either

Forms!SAP_BOM_2!Child0.Form.Filter = "Code='113941' OR Code = '113942'"

or

Forms!SAP_BOM_2!Child0.Form.Filter = "Code In('113941','113942')"

The first example is just to show the correct syntax and operator. The second method is more neat and you can add values easily.

scottgem say:

Just to expand on Karl's correct response. Your syntax was incorrect. the correct syntax for a filter is:

fieldname operator value

The Ampersand is a concatenation operator not a comparison operator so using it was inappropriate. Also, If you want to compare multiple values you need to use the AND or OR operators to specify multiple conditions (except you can use the IN clause to compare a list of values for one field).

jim gordon mvp say:

Outlook 2011 and Outlook 2016 are completely different. To rebuild the 2011 database you hold the Option key down as you start Outlook. The rebuild feature does not exist in Outlook 2016. If your database needs to be rebuilt, Outlook 2016 senses it and rebuilds it for you when you start Outlook.

There is a hidden application for Outlook 2016 named Outlook Profile Manager. You can run this app by taking these steps:

1. In Applications folder, right-click on Microsoft Outlook app

2. From the pop-up menu choose, Show Package Contents

3. In Contents > Shared Support you will find the application Outlook Profile Manager

4. Drag the Outlook Profile Manager icon to your dock so that the application becomes available on the dock. Be careful to not drag the application itself from the Shared Support folder. If you accidentally drag it out of Shared Support, drag it back in so that it works right.

You can see all your profiles when you run Outlook Profile Manager. You can use the manager to set the default profile. Once you have launched Outlook into the profile that matches the profile associated with your email message, you will be able to view it in Outlook.

Alternatively, if you want just the text, you can use Word's ability to display the text from any file to view the text of your email message file. Word won't recover graphics or formatting, just the text.
