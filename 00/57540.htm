k1234567890123456789 say:

Figured it out.  Apparently Edge has issues importing favorites from IE if you've ever redirected your IE favorites.  There was one registry location that needed to be deleted (it gets recreated when Edge is reopened).  Here's what to do:

1. Close Edge

2. Open File Explorer and go to \Users\"your username"\Favorites  (That is where your IE favorites will be unless you've setup a redirect)

3. Then open a new File Explorer window, keep #2 open also, and go to the Edge favorites folder.  This is \Users\"your username"\appdata\packages\Microsoft_microsoftedge_8weky...\AC\microsoftedge\user\default\favorites

4. Copy and paste #2 into #3.  If you're only copying your IE favorites bar, make sure to paste those in the \favorites\links folder of #3.

5. Right click on the start menu and click on 'run'

6. Type in regedit and press enter

7. In regedit go to HKEY_Current User\Software\Classes\Local Settings\Software\Microsoft\Windows\CurrentVersion\AppContainer\Storage\Microsoft_microsoftedge_8weky...\MicrosoftEdge

8. Then right click on the FavOrder folder and select Delete

9. Close regedit

10. Open Edge and your favorites should be back! 

However, you'll still get an error if you try to import again and those same steps will need to be repeated.

Hope this helps!
