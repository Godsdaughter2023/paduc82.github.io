avw3 say:

We have experienced this problem and there is a known issue that you should check for.

 

If a user has the preview pane turned on in Outlook, Windows Explorer, or in (My) Computer the PowerPoint file is held open by a hidden version of PowerPoint. No changes will be saved to the presentation while the preview pane is enabled. You will not get an error message from PowerPoint and it will look like the file was opened and closed properly, however it was never actually opened for edit by "real" PowerPoint and all changes the user has made will be lost.

 

To fix this: Hide or disable the user's preview pane in these three programs and then the user should be able to make and save changes to the PowerPoint presentation once again. This does not seem to affect Word and Excel files when the preview pane is turned on in the tests that we have done. I have read that this is a problem both in Windows 7 (which we use) and in Vista when using PowerPoint 2007.

 

Note that some users may also get a "file locked" message or a message that the PowerPoint file can only be opened read-only.  Again, turn of the preview panes and this should make the PowerPoint file usable again. The user should probably reboot their computer because I've noticed that some file-lock errors don't go away until there is a clean shutdown/restart sequence.

benishiryo say:

hi there.  can't really see the full picture here.  what i can help you to do is to understand what the formula is doing.

it finds it, it will look at the 7th column from column B (i.e. column H) and return that number.  so an eg.  for the table below, it finds "total" in row 5.  the 7th column is in column H.  Hence, it returns 100.

  B C D E G H I
2 some            
3 random            
4 text            
5 total         100  

C30 of your Summary worksheet is the number 1.  so what the formula does it count how many times the number 1 appears in DETAIL worksheet column J.  

so to summarize, what the first formula does is IF the VLOOKUP result returns 1, count how many times the number 1 appears in DETAIL worksheet column J subtract by 1.  Otherwise, count how many times number 1 appears in DETAIL sheet column J without subtracting 1.
=IF(VLOOKUP("Total",DETAIL!B:Q,7,FALSE)=1,COUNTIF(DETAIL!J:J,Summary!C30)-1,COUNTIF(DETAIL!J:J,Summary!C30))

I can see how being able to apply settings on a per sheet basis is useful but if you want the same settings on each worksheet for a workbook with for example 10+ sheets setting them individually would be very time consuming.

argel obm say:

Hi,

If you're using the default print layout settings when printing, all of your worksheets will have the same print layout, but, if you customized the print layout of one of your worksheet, it will only applicable to that worksheet. If you want to customize the print layout of all of your worksheets, follow the steps below:

  1. Press and hold the CTRL button of your keyboard.
  2. While holding the CTRL button, click all of your worksheets.
  3. When all of your worksheets are selected, release the CTRL button.
  4. Go to Page Layout tab.
  5. From the Page Layout tab, you can now customize your print layout.
  6. After customizing the print layout, click File.
  7. Select Print.
  8. Under Settings, choose Print Entire Workbook.
    • NOTE: If you'll notice, all of the settings you chose on Page Layout tab was reflected on the Settings of Print like the orientation, the paper you're going to use, and the margin. If you'll check all of your worksheets, it will have the same customized settings.
  9. Click Print.

If you still have any questions, kindly reply to this thread.
