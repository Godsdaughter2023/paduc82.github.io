Yes, it certainly looks that way as Windows is always (or should be) on C:\ . I assume when you 'explore' C:\ Windows is there ?

If Windows was on the SSD originally the Boot sector is probably still on the SSD which is why you can't boot

without the SSD connected.

That D:\ shows as full (100%) and how it became a RAW partition is also a mystery to me. If you can R/click on

- This -  "my files associated to windows on Disk 1" -  may have contributed to stuffing up your Windows if you

moved your User folder/sub-folders from the SSD to the HDD earlier.

Windows does have options to move the location of some of your User folder sub-folders, but this is far from a perfect

world scenario.

A lot of what goes into various parts of your User folder is put there by software/games and can loose the file path

and stuff things up if your re-locate the folders. Even though Windows has the option to re-located. It's just not worth the risk, imo.

Your SSD is plenty large enough for Windows, your User folder,  3rd party software, and more.

If you have large collections of music, etc. just make a new folder on Disk 1 and create a shortcut to the folder, or use a keyboard

macro key to those folders if you have a keyboard with programmable keys.

If you play games they can be installed on Disk 1 as well. I would create separate partitions for games and storage.

- Before reformatting the SSD...

Try swapping SATA ports so the HDD is on the SSD's  SATA port and vice versa. This may get a boot sector written to the HDD

when you boot into Windows after swapping SATA ports.

Once you get to where you can boot from the HDD you can clone Win10 back to the reformatted SSD.

You may also be able to build a boot sector via the Advanced Startup Options....

Advanced startup options (including safe mode) - Windows Help

You have an option to clean reinstall Win10, or clone your current install back to the SSD.

You may still be able to do this before trying to write a boot sector to the HDD, assuming there are no problems with the SSD/HDD.

noel d paton say:

The installed Key is the Default Key for Windows 7 Ultimate - and can never be activated.

If the COA sticker on your machine is for Windows 7 Ultimate, you can use the COA Key to activate the system...

 

 You need to change the Product Key to the one on the COA sticker on the machine's case (or inside the battery compartment),  using the Change Product Key link at the bottom of the System properties page.

  

You may need to activate by phone.   

If the COA sticker is for any other edition of Windows, then you will have to reformat and reinstall using legitimate media and Key.
