A password is required to synchronize this notebook. ( Error - Microsoft Community

varunsrin say:

A password is required to synchronize this notebook. ( Error code: 0x80070005 )

If you are hitting this error, you can follow the steps below to resolve the issue
varunsrin say:

SkyDrive Notebook

If your notebook is on SkyDrive, you need to provide your user name and password so that OneNote can sync to this location. If OneNote needs a password, you should see a yellow information bar at the top with this message:


"OneNote needs a password to sync some of your notebooks. Click here to see the list of sections and notebooks that require passwords"


Click on the bar, select the notebook name and then provide your credentials to begin syncing. You should also ensure that you check ‘Remember Me’ in the credential prompt, so that OneNote saves your credentials and doesn't have to prompt you again.

If this doesn’t work for you, you can always sign in by going to File -> New, selecting ‘Web’ for Step 1, and then clicking Sign In under Step 3. If you have the November 2011 updates, you can sign in from File -> Open as well.

 

 

SkyDrive Notebook - Multiple Live ID’s with different permissions to the same notebook

A slightly less common case is if you have multiple Live ID’s signed in on the same machine, and some of these accounts have different permission levels to the same OneNote notebook. Say for instance that you have a primary Live ID  <username>@yourcompany.com , where you created the notebook and then you shared it with <username>@live.com, but only provided Read Access to this secondary account.

If you are signed into both Live ID’s within OneNote, you may hit this error message because OneNote is trying to sync your changes with your secondary account, which does not have write access. OneNote tries to use credentials in order, so if your secondary account is tried first, sync will fail because it does not have write permissions.

 

To fix this issue, you can do one of the following:

  •         Give your secondary account read/write permissions to the notebook on SkyDrive

  • Delete your secondary account credentials via the credential manager, so that it always uses the primary account which has full access to the notebook.

 



SharePoint Notebook

If your notebook is on SharePoint, contact your SharePoint admin to resolve the issue.

Image / Invert Colors. Or...if you have IrfanView...go to...
