vijay a. verma say:

My take -

1. Blank option will be there only if your selection contains blanks. Hence, if you have selected e.g. A1 to A10 and there is one blank in this range and you apply filter, blank will come as an option. But if there is no blank in A1 to A10, blank will not come.

2. Second case is that you don't select A1 to A10 but you simply apply filter on A1. In this case, Excel applies filter till last value is found. Hence, if your last value was in A10, the filter will consider data till A10. Now, it becomes like point 1 discussed above.

2. If you select the entire column and then apply filter, blank option will come as it is very unlikely that all rows have data in an entire column.

If the RANDOM # column contains formulas =RAND(), Excel will recalculate the formulas when you sort the column, producing new random values, which will not be sorted.

You'd have to replace the formulas with their values if you want to sort the column successfully (select the column, copy it, then click the lower half of the Paste button and select Values)

Sub OverwriteData_In_CSV()
