The specific message you are relating to is very much a red herring. It is not referring to simple total RAM or HD space, it is talking about more limited subsets of memory.  The problem is figuring out specifically which "memory" the message is referring to.

“Microsoft Excel cannot open or save any more documents because there is not enough available memory or disk space. To make more memory available, close workbooks or programs you no longer need. To free disk space, delete files you no longer need from the disk you are saving to”

Mike, for future reference.  Often these suggestions we make are very generic. You have to pick suggestions that apply to your specific situation using "common sense" (yes, I know, common sense is very uncommon).  For example, if you do not have FireFox installed then you can ignore the step to clear the cache, and there definitely no need to install it to just clear a cache you hadn't used. 

On the other hand, don't skip most of the steps simply because you don't THINK they apply.  For example, addons in Office often cause problems. Many people want to ignore the "safe mode" trouble shooting test because they THINK they don't have any addons.  But when they actually check, they find that they do have several addons. Some are default in Office while others are added by 3rd party programs.Things like Bluetooth addons or printer related addons. The specific example I am thinking of is many Cannon printer drivers also install(ed) an addon called ABBYY Finereader.  When Office 2013 was released, current and older versions of ABBYY caused Word 2013 to crash.  ABBYY eventually released an updated version to fix the problem, but people installing older printers or using old install disks can still get the old buggy version of the addon.

You say you don't use FireFox or IE, fine. But in that case research your specific browser to find out if it uses a cache (folder) (it almost certainly does), how to clear the cache and maybe even how to increase the allocated cache size.

Note: "web history" and cache are 2 different things.  The history is generally a list of web links you have recently used.  While the cache is a downloaded copy of those pages you have looked at. The idea of the cache is to reduce the amount of stuff you need to download when you return to the same page. The browser can start by load the copy from the cache, then checking for changed elements.

This page covers the most commonly used browwers:

http://www.wikihow.com/Clear-Your-Browser's-Cache

We suggested clearing the browser cache because sometimes there are unexpected interactions between Windows applications.
