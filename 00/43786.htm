considerate_guy say:

There have been several versions of Movie Maker over the years depending on the version of Windows being used.

 

For Windows XP, you'll find Movie Maker 2.1 and it creates .mswmm project files that will only work in WMM 2.1, 2.6 or 6.0.

 

For Windows Vista, you'll find Movie Maker 2.6 and 6.0.  The 2.6 files are just like 2.1, so they will work in WMM 2.1, 2.6 or 6.0.  If they had Movie Maker 6.0, those files can work in Windows Live Movie Maker 2010, 2011 or 2012.

 

So it seems they are using WMM 2.x and that is why the project files won't open in WLMM.  The program is incompatible with the project file.

 

If you are truly trying to edit the same project on two different machines, you must have a compatible version of Movie Maker and you must have all the original source files.  A project file doesn't actually contain any video, photos or music, just references to the file names as they existed on the student's computer.  So even if you had the same version on both, you wouldn't see anything.

 

If all you want to do is to view their completed movie, the student should save their final movie, not their project file.  It's like the difference between giving someone a cake to eat (movie) or just the recipe (project).  The project can't be enjoyed without the ingredients and some "baking". 

 

If they go to File > Save movie, you'll have a .wmv file that can then be viewed on other Windows PCs.
