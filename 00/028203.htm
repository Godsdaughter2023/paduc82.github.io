shaileshmanjrekar say:

How can I create a line break with VBA in Excel 2011?

Split From: "How can I create a line break in Office for Mac 2011’s Excel?"
Any Idea how to make it possible in VBA.
I have tried "AAA" & vblf & "BBB" 
"AAA"& vbnewline & "BBB"

Nothing works :(

j.e. mcgimpsey say: So where are you trying to display this?

If a MsgBox, InputBox, etc., then vbNewLine should work as is (as will vbCr, vbLF, vbCrLf).

If you're trying to display it in a cell, you should make sure the cell is set to wrap text, e.g.,:

   With Range("A1")
      .WrapText = True
      .Value = "AAA" & vbNewLine & "BBB"
   End With

In general, you should use vbNewLine as your character to insert a line wrap.

For historical reasons, MacXL (which came first, remember - before Windows even existed) uses the Carriage Return character (ASCII 13, vbCr) to indicate that text should be wrapped, while WinXL uses the Line Feed (ASCII 10, vbLF) to do so. Of course, on an actual teletype, BOTH were required - a LF only moved the platen while the CR only moved the print head. 

Using vbNewLine will insert whichever character works for the system the code is run on.

It sounds like WMDC and Outlook are not aware of each other, which would indicate a probable Registry problem.  I would recommend uninstalling WMDC and then perform an Outlook Repair Installation (different from Laura's recommendation above).  No backup of data is necessary in this case.

  3. Click on the Outlook Installation in Control Panel > Programs.  An option should be provided to repair the existing installation.   This may require you to insert the original Installation CD or provide network access to the Installation file.  When it's done, if it requires reboot, say ok.
