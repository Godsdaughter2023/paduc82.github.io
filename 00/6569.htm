jamespitkin say: To perform this test you can use Firebind's port testing site.

If you go to www.firebind.com you can choose their Java applet client and then enter that range of ports in their UDP tester.

To go directly to the Java applet client you can click here  http://www.firebind.com/clients/applet/ but you need to make sure you have Java installed.

Once you are at the client, click on the UDP radio button (it defaults to TCP) and then paste in the ports you mentioned below in this format:

5060,50000-50100

It will then test all 102 of those ports.  Actually, they only let you test 100 at a time unless you register, so you can either register and do one test, or you can break that above into two tests such as:

5060,50000-50050

and then second test would be:

50051-50100
