Windows RE is based on Windows PE (Windows Preinstallation Environment)

When the PC restarts; Login with the existing username and password, and it will find all the installed data.

devlin sco say:

Good day,

A possible cause for this issue is a corrupted system file. If you're unable to boot into Advanced startup using Shift + F8 key, we suggest creating a Windows 10 installation media using the Media Creation Tool. Follow the steps outlined under Using the tool to create installation media (USB flash drive, DVD, or ISO file) to install Windows 10 on a different PC. After creating the installation media, follow the steps below:

  1. Insert the Windows 10 installation media and boot your PC.
  2. Boot from the disc or USB flash drive.
  3. From the Windows Setup screen, click Next.
  4. Click Repair your computer.
  5. On the Advanced startup window, select Troubleshoot.
  6. Click Advanced options.
  7. Click System Restore.
  8. Follow the on-screen instruction on restoring your PC from a system restore point.

Please update us with the results.
