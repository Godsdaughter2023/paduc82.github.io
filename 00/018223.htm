So for example if the first cell is associated with Team A and it detects the word "Team A" in any one of the drop-down lists then it should automatically change its background color (the colors themselves don't matter, they just need to stand out from each other so as to be easily recognizable from a distance).

I've got virtually no experience with VBA and despite my best effort at self-teaching haven't managed to make this work, could anyone help me with this ?

jay freedman say:

                                        

Option Base 1

Private Sub Document_ContentControlOnEnter(ByVal CC As ContentControl)
    Dim intTeam As Integer

    If Not Left(CC.Tag, 2) = "DD" Then Exit Sub

    If Not CC.ShowingPlaceholderText Then
        ' convert selected team name to a number (1 - 11)
        For intTeam = 1 To CC.DropdownListEntries.Count
            If CC.DropdownListEntries.Item(intTeam) = CC.Range.Text Then
                Exit For
            End If
        Next intTeam

        ' in the cell corresponding to the previously assigned team,
        ' set the background and the text color to white
        With ActiveDocument.Tables(1).Cell(1, intTeam)
            .Shading.BackgroundPatternColorIndex = wdWhite
            .Range.Font.ColorIndex = wdWhite
        End With
    End If
End Sub

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, _
        Cancel As Boolean)
    Dim intTeam As Integer
    Dim aryColor As Variant

    If Not Left(CC.Tag, 2) = "DD" Then Exit Sub

    aryColor = Array(wdDarkRed, wdRed, wdYellow, wdDarkYellow, _
        wdBrightGreen, wdGreen, wdTeal, wdBlue, wdDarkBlue, _
        wdViolet, wdBlack)

    If Not CC.ShowingPlaceholderText Then
        ' convert selected team name to a number (1 - 11)
        For intTeam = 1 To CC.DropdownListEntries.Count
            If CC.DropdownListEntries.Item(intTeam) = CC.Range.Text Then
                Exit For
            End If
        Next intTeam

        ' in the table cell corresponding to the chosen team,
        ' set the background to the color corresponding to the team
        ' and the text color to Automatic
        With ActiveDocument.Tables(1).Cell(1, intTeam)
            .Shading.BackgroundPatternColorIndex = aryColor(intTeam)
            .Range.Font.ColorIndex = wdAuto
        End With
    End If
End Sub

To set up the dropdowns and table in the template, run this macro once:

            

Option Base 1

Sub Initialize()
    Dim CC As ContentControl
    Dim TeamNum As Integer
    Dim TeamNames As Variant

    TeamNames = Array("PM", "SCO Orchestrator", "E-Commerce", "CSI", "Operations", _
        "OOA", "CI", "P&S", "SCS", "CHB", "ACS")

    For Each CC In ActiveDocument.ContentControls
        If CC.Type = wdContentControlDropdownList Then
            CC.DropdownListEntries.Clear
            For TeamNum = 1 To 11
                CC.DropdownListEntries.Add TeamNames(TeamNum)
            Next TeamNum
        End If
    Next CC

    Dim cl As Cell
    With ActiveDocument.Tables(1)
        For Each cl In .Rows(1).Cells
            .Shading.BackgroundPatternColorIndex = wdWhite
            .Range.Font.ColorIndex = wdWhite
        Next cl
    End With
End Sub

can i recover this document? - Microsoft Community

michellelatkovic say:

can i recover this document?

i was typing a word doc and my computer shut down.  i had not saved my paper yet.  can i recover this document?

suzanne s. barnhill say:

If you had AutoRecover enabled (which it is by default), when you next start Word,  you should be offered an AutoRecovery backup of the document. How close this will be to the state of the document when your computer crashed will depend on what version of Word you have and how long you'd been working since the last AutoRecovery save.

 

That's all assuming that the computer actually crashed. If it shut down normally (as during a forced restart after installing updates), then you're out of luck. You should always save a document within the first 5-15 minutes (depending on length and importance) and every 5-15 minutes thereafter for as long as you are working on it. Then you need never lose more than 5-15 minutes' work. Also, you should never leave an unsaved document unattended. And if you have Windows configured to download and install updates automatically, then you may find it more prudent to change the settings so that Windows prompts you to download the updates, and you can then install them (and restart the system) at your convenience, when you're not working on a Word document.

ediardp say:

you can create a helper column let' say c there enter

=A1&b1

then copy and paste this column as value and use this formula to count

=SUM(IF(FREQUENCY(IF(LEN(c2:c11)>0,MATCH(c2:c11,c2:c11,0),""), IF(LEN(c2:c11)>0,MATCH(c2:c11,c2:c11,0),""))>0,1))

this is an array formula, edit the formula holding the keys Shift + Ctrl hit enter

The above formula is thanks to CPearson if you want to learn more look here
