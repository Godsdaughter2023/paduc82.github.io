tblFromToAdder

  0             116        0.05

Public Function GetGrossAvgWeight(intDays as integer, dblRateOfGain as Double _

jay freedman say:

When you use the Envelopes button on the Mailings ribbon, Word does some special processing to extract the first group of left-aligned paragraphs from the document body for the envelope address, and to grab the return address from the box at the bottom of the Options | Advanced dialog. None of that happens when you invoke the Create Envelope dialog directly from a macro, so you have to "do it yourself".

I don't see anything in VBA that gets the return address from the Options, so you have to put that text into a string inside the macro. In the code below, I've assigned a dummy return address to the .EnvReturn property; change this to your actual return address. If there could be a choice of return addresses, you could leave it to the user to modify what appears in the dialog box, or your macro can be expanded to present a list to choose from.

The mailing address is easy to extract by referring to the form fields that contain the parts. In the code below, I've used the field names "Text1", "Text2", and "Text3", which you must change if you've assigned different names in the form fields' Properties dialogs.

Sub MakeEnvelope()
    Dim temp As String
    Dim bProtected As Boolean
    
    ' construct mailing address from fields
    With ActiveDocument.FormFields
        temp = .Item("Text1").Result & Chr(13)
        temp = temp & .Item("Text2").Result & Chr(13)
        temp = temp & .Item("Text3").Result
    End With
    
    'turn off protection
    If ActiveDocument.ProtectionType <> wdNoProtection Then
        bProtected = True
        ActiveDocument.Unprotect
    End If
    
    'display Envelope dialog
    With Dialogs(wdDialogToolsCreateEnvelope)
        .Addrtext = temp
        .EnvOmitReturn = False
        .EnvReturn = "1234 Ridge Rd." & Chr(13) & "Stonewall, AL 55555"
        .Show
    End With
    
    'restore protection
    If bProtected = True Then
        ActiveDocument.Protect Type:=wdAllowOnlyFormFields, _
            NoReset:=True
    End If
End Sub
