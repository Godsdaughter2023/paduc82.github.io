Additional information:

Computers, which are built by large manufactures that come with Windows Pre-Installed, come with two (2) Product Keys:

 

A)   OEM SLP: This key comes pre-installed in Windows, when it comes from the Factory. This key is geared to work with the special instructions found only on that Manufacturer's computer hardware. So when Windows was installed using the OEM SLP key (at the factory) Windows 7 looks at the motherboard and sees the special instructions and Self-Activates. (that's why you did not need to Activate your computer after you brought it home)

 

B)    COA SLP: This is the Product key that you see on the sticker on the side (or bottom,[ or inside the battery compartment]) of your computer. It is a valid product key, but should only be used in limited situations (such as if the OEM SLP key stops self-activating for whatever reason). The key must be activated by Phone. (Note: All manufacturers that use the OEM SLP system are required by contract to include a Certificate of Authenticity (COA) sticker, that has a COA SLP key, on the computer)

Hi Paula,

I probably can't give you any more useful information than what you've already tried.    I would have thought your name (as Author of the document) would have been there by default (in all documents), i.e. when installing Word it automatically picks up on certain computer info and records it.   Mine certainly does on all documents and the name it uses is your Computer Name.

Other things that come to mind, which you may have already checked are:

a.   in your Properties tab, under Security, that you have full permission to handle the file in the various ways it shows.

b.   you may try unticking "Always create backup copy" (in your Document Options > Advanced > Save).

c.   disabling Preview Pane.  If your documents are in a common folder, then go to the Folder's Organize tab, then to Layout, and uncheck Preview Pane.   Apparently, this can sometimes cause Word to see the document in that pane as another one.

d.   in your Properties tab, under General, make sure that Read Only is not ticked.

e.   if all else fails, including your other insightful attempts, then there is always the good 'ol  (Ctrl + Alt + Delete)  to try and get rid of the document.

f.   also, there are third party file/program cleaners that force a removal;  but this should not be used unless all hope has gone.
