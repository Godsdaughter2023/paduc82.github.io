john korchok say:

I work on both Mac and Windows and create cross-platform forms for many institutions. There is no "conversion" between Windows and Mac versions of Word files, they use the same format. While table styling does not have a visible effect on checkboxes in Windows, it does on Mac.

Creating a table, inserting a checkbox, then applying table styling in Word 2011 is a 100% reliable way to create this effect, it happens every time. Normally removing styling by applying the Table Grid style is reliable for removing it as well, but clearly in your case it's not working. Applying manual formatting to correct this issue often has the opposite effect, making it impossible to change back using the style removal technique.

Depending on the complexity of your table formatting, you can also restore the checkbox black border by selecting the table, using Table>Convert>Table to Text, followed by Table>Convert Text to Table.

You can also try saving the document or template back to Word 97-2004 (.doc) format, then back to .docx or .dotx, that often works.
