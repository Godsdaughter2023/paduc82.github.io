doug robbins - mvp office apps & services (word) say:

I am not sure if it will work on a Mac, but assuming that you have a separate table on each page, running a macro containing the following code would split each table into the four separate tables, one to a page.

Dim i As Long, j As Long

Dim crange As Range

Dim arow As Row

With ActiveDocument

    For i = 1 To .Tables.Count 'Step 4

        With .Tables(i)

            Set arow = .Rows.Add

            Set crange = .Cell(1, 2).Range

            crange.End = crange.End - 1

            arow.Cells(1).Range.FormattedText = crange.FormattedText

            Set arow = .Rows.Add

            Set crange = .Cell(2, 2).Range

            crange.End = crange.End - 1

            arow.Cells(1).Range.FormattedText = crange.FormattedText

            .Columns(2).Delete

            For j = 1 To 4

                .Cell(j, 1).Range.Paragraphs(1).PageBreakBefore = True

            Next j

        End With

    Next i

End With

If you have just the one table, first run a macro containing the following code to split it into a series of two row tables

With ActiveDocument.Tables(1)

    For i = .Rows.Count - 1 To 3 Step -2

        .Split .Rows(i)

    Next i

End With
