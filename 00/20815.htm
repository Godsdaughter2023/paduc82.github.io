rich007a say:

You could use:

Sub DeleteBlankRows()
    Dim c As Range, rng As Range
   
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual
   
    Set rng = Columns("A:A").SpecialCells(xlCellTypeBlanks)
    If Not rng Is Nothing Then
        For Each c In rng
            If c.Interior.Color <> 16643047 Then '16643047 = RGB(253, 243, 231)
                c.EntireRow.Delete
            End If
        Next c
    End If
   
    Application.ScreenUpdating = True
    Application.Calculation = xlCalculationAutomatic

End Sub

which should only consider cells in column A down to the last row of the UsedRange, but that could be quite large, so this might be better:


Sub DeleteBlankRows2()
    Dim c As Range, rng As Range
    Dim lnLastRow As Long
   
    Application.ScreenUpdating = False
    Application.Calculation = xlCalculationManual
   
    lnLastRow = Cells(Rows.Count, "A").End(xlUp).Row
    Set rng = Range("A1:A" & lnLastRow).SpecialCells(xlCellTypeBlanks)
    If Not rng Is Nothing Then
        For Each c In rng
            If c.Interior.Color <> RGB(253, 243, 231) Then '16643047 = RGB(253, 243, 231)
                c.EntireRow.Delete
            End If
        Next c
    End If
   
    Application.ScreenUpdating = True
    Application.Calculation = xlCalculationAutomatic
End Sub

smart quotes correction Word 365 - Microsoft Community

jls606 say:

smart quotes correction Word 365

In previous versions of Word there was a way to have it search the document and change all straight quotes to smart quotes. Is there a way to do that in Word 365?
suzanne s. barnhill say:

If you have smart quotes enabled on the AutoFormat As You Type tab of AutoCorrect Options (File | Options | Proofing | AutoCorrect Options...), you can use Replace (Ctrl+H) to replace " with " (or ' with '). Just type an ordinary straight quote (single or double) in the "Find what" box and again in the "Replace with" box. As the Replace operation takes place, AutoFormat As You Type will fire, correcting the straight quote to a "curly" one.

Alternatively, you can use AutoFormat. On the AutoFormat tab of AutoCorrect Options, enable only the options you want (to be on the safe side, you can clear everything except the smart quotes options). You'll have to add the AutoFormat Now button to the Quick Access Toolbar (find it in Commands Not in the Ribbon category in the Customize the Quick Access Toolbar dialog). You might not think it worth the trouble to set this up for a single instance, but AutoFormat Now (given your preferred settings for AutoFormat) can be a huge timesaver in formatting rough text.
