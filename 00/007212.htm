Do following. I have reused the code from following and modified the code https://excel.tips.net/T003135_Page_Numbers_in_...

1. Make a backup of your workbook.
2. Open your workbook and ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Right click on your workbook name > Insert > Module
5. Copy paste the Macro code given
6. Save your file as .xlsm
7. Put following in a cell to get the page number
=PageNum() 

Function PageNum()
    Dim iPages As Integer
    Dim iCol As Integer
    Dim iCols As Integer
    Dim lRows As Long
    Dim lRow As Long
    Dim x As Long
    Dim y As Long
    Dim iPage As Integer
    With ActiveSheet
        y = ActiveCell.Column
        iCols = .VPageBreaks.Count
        x = 0
        Do
            x = x + 1
        Loop Until x = iCols _
          Or y < .VPageBreaks(x).Location.Column
        iCol = x
        If y >= .VPageBreaks(x).Location.Column Then
            iCol = iCol + 1
        End If
        y = ActiveCell.Row
        lRows = .HPageBreaks.Count
        x = 0
        Do
            x = x + 1
        Loop Until x = lRows _
          Or y < .HPageBreaks(x).Location.Row
        lRow = x
        If y >= .HPageBreaks(x).Location.Row Then
            lRow = lRow + 1
        End If
        If .PageSetup.Order = xlDownThenOver Then
            iPage = (iCol - 1) * (lRows + 1) + lRow
        Else
            iPage = (lRow - 1) * (iCols + 1) + iCol
        End If
    End With
        PageNum = iPage
End Function

You need to check your account to see what you paid for and agreed to. If you no longer want a particular paid service, you need to cancel it. Typical paid services:

1. Incoming subscription - A Skype number is a landline you purchase (lease) to receive incoming phone calls over Skype. You can purchase this by the month or other periods like a year. This subscription includes no outgoing minutes. This subscripton can be cancelled in the Skype number settings of your Skype account.

2. Outgoing subscription - Calling subscriptions can be purchased to cover the cost of the places you phone call frequently. You can also purchase this by the month or other periods like a year. These can be cancelled in each of your subscription properties found on your account page under your profile picture in the left blue pane.

3. SMS texting - While Skype IM is free, using SMS to send text (outgoing only) require credit. No subscription offered covers SMS texts. Credit only gets deducted if you SMS text a number.

4. Auto-Recharge - If you have this enabled and assigned to a billing source, you will charged in $10 increments to replenish your credit balance whenever it drops below $2. You can disable Auto-Recharge if you want more control over the credit you purchase.

All cancellations will simply stop your auto renewal at the end of your billing period. To avoid be auto-renewed/billed for a future cycle, be sure to cancel before the 3 day window prior to the end of your cycle. If you do cancel your subscription it will continue functioning and finish out its current subscription cycle.
