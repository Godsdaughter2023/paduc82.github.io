cliff gu msft say:

Dear PJ_R,

 

What you mentioned should happen in Word Online. If you check out the file, open it in Office Online, edit it and the file is auto-saved. A new version will be created. After you check in the file, another new version shouldn’t be created. In other words, the process (check out the file>open the file in Office Online>edit the file>check in the file) will only create one version.

 

If you check out the file and open the file in Office 2016, AutoSave should be turned off. After editing the file, checking in the file will create one version as well.
