Try this

=SUMPRODUCT((A$2:A$10=A2)*(B2< B$2:B$10))+1

=SUMPRODUCT((MONTH(A$2:A$10)=MONTH(A2))*(B2< B$2:B$10))+1

vijay a. verma say:

RANK formula has order argument - if this is not mentioned or 0 then, it ranks the list in descending order. Hence, 20 will be ranked higher than 10.

If Order argument is any non zero value say 1, it ranks the list in ascending order. Hence, 10 will be ranked higher than 20.

Mike H..'s formulas take care of first case i.e. default.

In case, you want ascending order sorting, you will need to replace < sign with > sign. Hence, those formulas will become -

=SUMPRODUCT((A$2:A$10=A2)*(B2> B$2:B$10))+1

And

=SUMPRODUCT((MONTH(A$2:A$10)=MONTH(A2))*(B2> B$2:B$10))+1

Sub extract_MktData(Mkt1Name As String, Mkt2Name As String, Mkt3Name As String)
Dim lRow As Long
   
    lRow = .Sheets("Milestone Detail By Site").Range("A65536").End(xlUp).Row
    Rows("1:1").Select
    Selection.AutoFilter
    If Mkt1Name <> "" And Mkt2Name <> "" And Mkt3Name <> "" Then
        ActiveSheet.Range("$A$1:$CI$" & lRow).AutoFilter Field:=4, _
            Criteria1:="=" & Mkt1Name, Operator:=xlOr, Criteria2:="=" & Mkt2Name, Operator:=xlOr, Criteria3:="=" & Mkt3Name
    ElseIf Mkt1Name <> "" And Mkt2Name <> "" Then
        ActiveSheet.Range("$A$1:$CI$" & lRow).AutoFilter Field:=4, _
            Criteria1:="=" & Mkt1Name, Operator:=xlOr, Criteria2:="=" & Mkt2Name
    ElseIf Mkt2Name = "" And Mkt3Name = "" Then
        ActiveSheet.Range("$A$1:$CI$" & lRow).AutoFilter Field:=4, _
            Criteria1:="=" & Mkt1Name
    End If
End Sub

gary's student say:

Just delete the hidden rows:

 

 

Sub rowKiller()
Set rr = Intersect(ActiveSheet.UsedRange, Range("A:A"))
Set rkill = Nothing
For Each r In rr
    If r.EntireRow.Hidden Then
        If rkill Is Nothing Then
            Set rkill = r
        Else
            Set rkill = Union(r, rkill)
        End If
    End If
Next
rkill.EntireRow.Delete
End Sub

amit tandon say:

In case your query has been correctly understood, you wish to display all column values (without blanks) on a particular date with the corresponding Species, and no other columns. Below is a solution using excel formulas:

Please refer below image:

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell I3 and copy down:

=IFERROR(INDEX($A$3:$A$100,SMALL(IF(OFFSET($A$3:$A$100,,MATCH(J$2,$B$2:$G$2,0))<>"",ROW($A$3:$A$100)-ROW($A$3)+1),ROWS($A$3:$A3))),"")

Enter below formula (normal, non-array) in cell J3 & copy down:

=IF($I3="","",VLOOKUP($I3,$A$3:$G$100,MATCH(J$2,$A$2:$G$2,0),FALSE))

You may enter any date in cell J2, and the corresponding values will be returned / displayed.

Insert an empty row between rows 1 and 2.

To sort for 03/07/2016, for example, select the cell with that date (F3 after the empty row has been inserted), then click Sort > Sort Smallest to Largest (or Sort Largest to Smallest, if you prefer). That should sort the entire range, inlcluding column A (the Species column).

Do not select the column when you want to sort, just the date cell.
