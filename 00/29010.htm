Surface Pro 4 how to Boot from dvd 
steveg47 say:

Surface Pro 4 how to Boot from dvd 

Plug in the dvd drive. Start the sp4 by pressing the volume up and power buttons. This will bring you to the UEFI (bios).  If the inserted dvd is not a windows OS you will have to turn Secure Boot off in Security.  Go to the Boot Configuration and change the boot order to boot from the usb dvd first(USB Storage). Read the instructions on the Boot Configuration page in the UEFI for details on how to change the boot order and alternately how to boot with a left swipe.

Reboot. 

Note: You can boot using the sp4 usb port or one of the four usb ports on the Docking Station.
