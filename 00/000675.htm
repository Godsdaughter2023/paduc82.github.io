Function SpellNumber(ByVal MyNumber, Optional incRupees As Boolean = True)
 Dim Crores, Lakhs, Rupees, Temp
 'Dim DecimalPlace As Long, Count As Long
 Dim myLakhs, myCrores
 ReDim Place(9) As String
 Place(2) = " Thousand ": Place(3) = " Million "
 Place(4) = " Billion ": Place(5) = " Trillion "
 ' String representation of amount.
 MyNumber = Trim(Str(MyNumber))

Option Explicit
Function GetDate(dt As Date, Optional NumMonths As Long = 1) As Date
    Dim dLastWorkingDateOfMonth
    Dim dTest As Date
   
With WorksheetFunction

dLastWorkingDateOfMonth = .WorkDay(.EoMonth(dt, 0) + 1, -1)

Select Case dt
    Case Is = dLastWorkingDateOfMonth
        GetDate = .WorkDay(.EoMonth(dt, NumMonths) + 1, -1)
    Case Else
        dTest = .EDate(dt, NumMonths)
        Select Case Weekday(dTest)
            Case vbMonday To vbFriday
                GetDate = dTest
            Case Else
                If Month(.WorkDay(dTest, 1)) = Month(dTest) Then
                    GetDate = .WorkDay(dTest, 1)
                Else
                    GetDate = .WorkDay(dTest, -1)
                End If
        End Select
End Select

End With

End Function

cbarnhorst say:

It appears that your issue is not the hard drive but simply a re-formatting of Windows 10.

In 2015 Microsoft changed to the present activation method (digital entitlement) a couple of months before the release of Windows 10 to the general public.  They made the unique hardware IDs of the motherboard the method for activation instead of product keys.  At that point, the special Insider product keys no longer activated Windows 10 on new devices.  From that time forward product keys were only used to register a device's unique hardware IDs with Microsoft.  The licensing software checks for the device's registration and, if found, activates Windows 10.  If not found, a product key is requested so registration can be completed.  This registration is one-time and permanent and is specific to your motherboard.  

When the change was made in 2015, Insiders were asked to re-activate their Insider builds with a Windows 7 or Windows 8.1 product key in order to permanently register their devices.  This was called the "free upgrade."  Microsoft warned that the old activation would only remain valid as long as the user did not reformat Windows and that they should complete the free upgrade to Windows 10 in order to be guarantee activation into the future.  The free upgrade offer ended in July of 2016.

The hard drive makes no difference since registration is not recorded on the device but rather in the Microsoft Cloud so you can replace a hard drive without any change to your license.  If you did not register your device with a legacy product key you will not be able to activate a new installation of Windows 10 now and need to buy a Windows 10 license.  If you did register using the new procedure then all you need to do is install Windows 10 (same edition) and it will activate automatically upon connection to the internet.
