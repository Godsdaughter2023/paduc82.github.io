greginmich say:

The question here is “How do I schedule a Defender scan?” The answer is that the only proper way to schedule a Defender scan is with a new user-defined scheduled task. The only preexisting task is the Windows Defender Scheduled Scan task, which appears at this location (which you've captured in your second screenshot):

Task Scheduler Library > Microsoft > Windows > Windows Defender  

This is emphatically not a blank scan task that was just left there so that it could be configured and scheduled by users – this is the scheduled scan task that’s run by Windows Automatic Maintenance. It was fine-tuned for that specific purpose, and therefore properly belongs to the SYSTEM. In the previous version of Windows 10 (and maybe also in the current version) we needed to modify this task in order to get the Automatic Maintenance scan to run consistently – but this is not the place to schedule a Windows Defender scan. In order to schedule a proper scan, you need to open Task Scheduler and select “Create Basic Task…” from the Action or Actions menu, and then proceed according to the steps that I’ve outlined in the threads cited above.

Once your task has been created, it will appear when you click on the Task Scheduler Library folder – and then you can edit the schedule by double-clicking on the task, or by right-clicking on it and choosing Properties – which will bring up the windows that appear in my screenshots. These screenshots depict the Properties and Edit Trigger windows for a Defender scan task that I had just created a few minutes prior to my reply, simply for the sake of demonstrating how to edit the run time of a scheduled task.

The most common cause of this error message is that an illegal character (usually a slash /) is introduced somewhere in the file path. In particular, sometimes the startup disc gets renamed accidentally and the extraneous character is introduced. Sometimes the startup disc has no name at all, and that also causes this error message.

Rename your startup disc (your hard drive) and make sure the name has only only letters and numbers.
