Get-Mailbox -RecipientTypeDetails UserMailbox -OrganizationalUnit "<Domain name /Area>" | Get-MailboxJunkEmailConfiguration | Format-Table -Auto Identity,Enabled

Get-OrganizationalUnit

tom. say:

At the top of the page the numbering would begin at 500 and at the bottom it starts at 1, while both increment by 1 for each page, in a simple document (that is not subdivided in any way).

jeeped say:

At the top of the page the numbering would begin at 500 and at the bottom it starts at 1, while both increment by 1 for each page ... 


In the header, tap Ctrl+F9 to insert a set of field braces and type = (equals sign). Tap Ctrl+F9 again and type page. Move the insertion point outside of the nested field braces and type +499. It should look like this:
{ = { page } +499 }
Tap Ctrl+A then F9 (i.e. Select All, Update Fields) then move to the footer. In the footer, just tap Ctrl+F9 and type page. It should look like this:
{ page } 
Tap Ctrl+A then F9 (i.e. Select All, Update Fields) then move to the body of the document.

You should be getting 500, 501, 502, ... in the header and 1, 2, 3, ... in the footer.

manta---01 say: "older" PC Suite versions, like 6.5 had 2 separate functions in backup: one for traditional backup and one for transferring phone content from one phone model to another. These 2 selections were visible in PC Suite 6.5.

I have seen it in numerous forums that people tend to take the "backup" which is fine for restoring to same phone model, but it will not work to another model, there you need the content transfer.

The new PC Suite 6.8 seems to do both 2 operations at the same backup "click".

If you have a file with extension .nfb in your PC, that is a backup file that is only suitable for same phone model.


The backup files should be in
C:\Documents and Settings\USERNAME\Application Data\Nokia\ContentCopier folder in order for it to be visible in PC Suite backup.


To your question:
Does this mean the backup file format (& file extension) has changed from v6.5 to v6.8.22 and that my backups are all useless or (quite possible this) I am being really stupid and missing something obvious?

The file formats of the older version PC Suite backups are supported, but if they all are of .nfb format, it means that you can only backup to same phone model.
