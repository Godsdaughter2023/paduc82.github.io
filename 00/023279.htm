The common culprits are CCleaner (I gather there's an option you can enable/disable that relates to MRU lists) and "privacy options" settings in Windows utilities (JV16 Power Tools is one that has been mentioned).

If neither of these is likely, then it may be that you need to edit or delete a Registry key. If you scroll to the bottom of the thread at http://windowssecrets.com/forums/showthread.php/130621-MS-Office-2010.-Recent-documents-do-not-show, a couple of keys are discussed; specifically:

  • HKCU\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
  • A DWord Value named NoRecentDocsHistory with a value of 0 should restore your Recent Documents for MS Office.
  • A value of 1 will disable the Recent Documents "feature."

Again, we need to know what kind of email account you have, POP, IMAP, EAS (Hotmail/Live mail/Outlook.com), Exchange Server, something else?  If you have a POP account, Contacts are in the old .PST file on your ASUS computer.  If you have an IMAP account, there should be both an .OST file (for email, this file cannot be transferred, there is no need to, it will be rebuilt on the new computer) and a .PST file for Contacts, Calendar, etc. which needs to be transferred.  If you have either an EAS or Exchange Server account, Contacts are kept on the server and should automatically sync.
