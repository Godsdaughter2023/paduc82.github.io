Using INDEX, MATCH, and CONCATENATE simultaneously - Microsoft Community

bretmurphy say:

Using INDEX, MATCH, and CONCATENATE simultaneously

In trying to sort data categorically on a new worksheet from existing data on a separate sheet, is it possible to use multiple functions to not only sort this data but to also repopulate associated columns? I need to sort groups by organization and have only been able to get excel to identify each line as TRUE/FALSE with an associated named matched to each line. I would like it to only populate as many rows as there are individuals for said organization. I used the formula =INDEX('Main Data'!B2:B51,MATCH("VT-14",'Main Data'!A2:A51,0)) as an array formula, but the resultant cells populated recurring last names in rows where the individual belonged to another organization. I hope that by using CONCATENATE, I can get the first names to also appear, but am stumped as to how to eliminate the multiple names from appearing. Thank you to anyone who can help.
bernie deitrick say:

Array-enter (enter using Ctrl-Shift-Enter)

=IF(COUNTIF('Main Data'!$A:$A,"VT-14")>=ROW(A1),INDEX('Main Data'!B:B,SMALL(IF('Main Data'!$A$2:$A$51="VT-14",ROW($A$2:$A$51),52),ROW(A1))),"")

and copy down until it returns ""

If you need the corresponding data from other columns, just copy the formula across, or change the B:B to the column of interest.  You would probably do well to change the "VT-14" constants to a cell reference, like $B$1 so that you can change it to any value that exists in column A.

bernie deitrick say:

You could use a formula like this, to concatenate columns B, C and D:

=IF(COUNTIF('Main Data'!$A:$A,"VT-14")>=ROW(A1),INDEX('Main Data'!B:B,SMALL(IF('Main Data'!$A$2:$A$51="VT-14",ROW($A$2:$A$51),52),ROW(A1))) & ", " & INDEX('Main Data'!C:C,SMALL(IF('Main Data'!$A$2:$A$51="VT-14",ROW($A$2:$A$51),52),ROW(A1))) & " " & INDEX('Main Data'!D:D,SMALL(IF('Main Data'!$A$2:$A$51="VT-14",ROW($A$2:$A$51),52),ROW(A1))),"")

Again, array entered. I used ", " and " " between but your could use any other variation you want.
