Hi John,


Here are some of the tips for speeding up Windows 10:

1. Uninstall Crapware - That extra preinstalled software installed by PC makers continues to be an issue with new computers. When you've found junk apps you don't want, simply select them and click Uninstall.

2. Limit Startup Processes - A lot of programs install side processes that run every time you start your PC, and some of them are not things you need running on your system all the time.

3. Clean Up Your Disk - From the Start menu, type Disk Cleanup. This opens the trusty Disk Cleanup utility that's been part of Windows for several generations of the OS. Disk Cleanup finds unwanted junk and files such as temporary files, offline Web pages, and installer files on your PC and offers to delete them all at once.

4. Add More RAM - Windows 10 isn't as much of a hog as earlier versions of the OS, but more memory is always a way to speed up PC operations.

5. Install an SSD Startup Drive - This past year, I installed a solid-state (SSD) startup drive on my home desktop PC, and the result was a remarkable speedup.

6. Change Power Settings to Maximum Performance - Of course, this isn't a good choice if you want to save electricity, but it could boost your computing. Head to Control Panel / System and Security / Power Options. From here, click on the drop-down arrow on the right side and choose High Performance.

7. Use the Performance Troubleshooter - In Cortana's search box next to the Start button, type troubleshooting and under System and Security, you'll see the choice Check for performance issues. Run the troubleshooter and it may find the root cause of your slowdown.

8. Change Appearance in Performance Options Dialog - You can easily get to this by typing adjust appearance in Cortana. In the dialog, you can use the radio button at the top labeled Adjust for best performance or select which eye-candy features you can live without from the long list of check boxes below these choices.

9. Turn Off Search Indexing - Especially for lower-powered PCs, search indexing can eat up system resources, if only temporarily. If you do a lot of searching, this won't appeal to you, as some searches will be slower. To turn off indexing, open the Indexing Options Control Panel window (you can also just type index in the Start button search box, and you should see Indexing Options at the top of the result list), click Modify and remove locations being indexed and file types, too.

Let us know if you need further assistance.

aakkam22 say:

Here are my thoughts:

  • I would strongly suggest against disabling your pagefile, as the article you provided above suggests. 
  • One of the best ways to speed up any system is to install an SSD, or solid-state drive. There are lots of guides on Google that will help you, but to get started, I recommend The Complete Guide to Solid-State Drives. 
  • The more RAM you have, the smoother your experience will be when running multiple applications at the same time. Therefore, unless you open lots of apps at the same time, adding more RAM will not significantly improve performance.
  • You mentioned McAfee, installed by HP. McAfee and other bundled apps on your PC are known as "crapware". I would highly recommend removing them, including McAfee.
  • After uninstalling McAfee, you can use the built-in Windows Defender as your anti-virus program. However, only activate Windows Defender after installing McAfee as having more than one AV programs at the same time can severely impact performance. 
  • If you have a hard disk drive and won't be spending the cash for an SSD, the single best thing you can do to improve its performance is to defragment it. In the search box at the bottom of your screen, type in defragment and then click Defragment and Optimize Drives from the results. 

  • Manage your startup items, which severely impact system performance. For beginners, you can use the Task Manager to turn off startup items. Right-click the taskbar, click Task Manager, and go to the Startup tab. 
  • Finally, Windows 10 includes several services that are unnecessary and can severely hamper system performance. There are lots of guides on Google that you can use to tune your services -- just make sure to take caution when working with services because sometimes you can disable an essential Windows service.

Good luck
