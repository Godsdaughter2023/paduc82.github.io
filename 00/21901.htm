Column C Column D Column E

gord dibben say:

Add this event code to the worksheet module.

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim ws As Worksheet
    Dim cRow As Long
    Set ws = ActiveSheet
    If Intersect(Target, Range("E:E")) Is Nothing Or Target.Cells.Count > 1 Then Exit Sub
    cRow = Target.Row
    With ws
        If .Range("E" & cRow).Value <> "" Then
            .Range("C" & cRow).IndentLevel = 2
        End If
    End With
End Sub

Gord

gord dibben say:

The posted code works only on a single sheet.

If you want the code available for all worksheets you must move it to Thisworkbook module and change the code type.

Private Sub Workbook_SheetChange(ByVal Sh As Object, ByVal Target As Range)
    Dim ws As Worksheet
    Dim cRow As Long
    Set ws = ActiveSheet
    If Intersect(Target, Range("E:E")) Is Nothing Or Target.Cells.Count > 1 Then Exit Sub
    cRow = Target.Row
    With ws
        If .Range("E" & cRow).Value <> "" Then
            .Range("C" & cRow).IndentLevel = 2
            Else: .Range("C" & cRow).IndentLevel = 0   'unindent if E is deleted
        End If
    End With
End Sub

To get to Thisworkbook module hit Alt + F11 to open Visual Basic Editor

Expand your workbook/project and open the sheet module, delete that code then open Thisworkbook module.

Paste the new code in there. . . . .note also a line to cover your request to unindent has been added.

Alt + q to return to Excel sheet.

Gord
