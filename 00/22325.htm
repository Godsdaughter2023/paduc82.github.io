cindy meister say:

Hi Hans

It would probably be better to ask this in a forum that specializes in Mac, as only the Mac folks know what MacWord has (and has not).

FWIW, there is a certain standard group of styles that cannot be removed from a document. Among them, Normal, Heading 1, Heading 2, Heading 3. If your versin of Mac is similar to my WinWord 2003, you should be able to display the Styles task pane, click on the dropdown arrow next to style name, and see if "Delete" is enabled.

In WinWord 2007 it's possible to hide styles from the lists. In the Styles task pane there are three buttons at the bottom. The right-most is "Manage styles". In the "Recommend" tab you can set whether a style should be completely hidden, hidden until used, or shown. It's also possible to set priorties here (where a style should appear in the list if "Recommended" is specified as the way to list styles).

Don't forget, if you hide / don't use the built-in heading styles that this will make it more difficult (or even impossible) to do certain things in Word. For example, you won't be able to include any kind of "section" numbering in the page numbering.

Cindy Meister, VSTO/Word MVP
