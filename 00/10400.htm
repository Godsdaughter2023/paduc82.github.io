to use the team site, we can click sites icon on the app launch bar of the office portal > find team site > access it and upload the files to the document library > then share the site with other colleagues. to share a site, you can have a look at this article:
https://support.office.com/en-us/article/share-a-site-958771a8-d041-4eb8-b51c-afea2eae3658

here is an article for reference which will help you understand sharepoint better:
https://support.office.com/en-us/article/videos-for-sharepoint-online-ed074945-4ddc-4479-9efe-6b3945cf8266

if your plan doesn’t have sharepoint license or you don’t want to use the team site, as a possible workaround, for a small group such as 3 or 5 people, we can use the following ways to “work” together:
1. go to one user’s onedrive for business page (let’s say, user a)
2. go to site settings > site permissions
  
3. click grant permissions and input the user name you’d like to share (b as the user)
  
4. go to sharepoint admin center > user profiles > people > manage user profiles
5. input the username of a and find profiles
6. find the profile and choose manage site collection owners
  
7. input the username of b in the site collection administrators dialog and click ok
  
then user b will be able to access a’s onedrive for business with the url of the page. about the third user, you can repeat the steps to let all the users access the same page for work.
