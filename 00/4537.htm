a. user say:

(NB, someone has moved your question to be in a Word group - I suspect it should be in an Excel group).


The "\" probably isn't going to do what you expect on Mac - in this case, it will probably create a file in the folder *above* the workbook, with a name like "xyz\ThingsImport.txt"

What you probably need is 

Open ThisWorkbook.Path & Application.PathSeparator & "ThingsImport.txt" For Output As #1

which will use "\" for Windows and "/" for Mac Word 2016.

For the update thing you can put something like


       Cells(r, 6).Value = "x"
in your loop.

DEP is a security feature that can help prevent damage to your computer from viruses and other security threats. DEP can help protect your computer by monitoring programs to make sure they use system memory safely. If a program tries running (also known as executing) code from memory in an incorrect way, DEP closes the program.

 

First, see if a DEP-compatible version of the program is available by visiting the software publisher's website. If the publisher has not released an updated, DEP-compatible version of the program, you can turn off DEP for the program that was closed. You will be able to use the program, but it might be vulnerable to an attack that could spread to your other programs and files.

 

If you choose to turn off DEP for a program, it's a good idea to check frequently for an updated version of the program and, after you update it, to turn on DEP again. To turn off DEP for a program, see Change Data Execution Prevention settings.

PPT 2011 for Mac - Copy animation - Microsoft Community

eliusm say:

PPT 2011 for Mac - Copy animation

Hello everybody.

Is there any possibility to copy a particular set of animations connect to an object to another one without recreate it?

Thank you, EliusM.

john korchok say:

You can do this by working counter-intuitively. Instead of copying the animation to a new shape, you copy the it with animation, then change the shape:

  1. Copy the shape(s) with animations to a new slide.
  2. Select the shape to be changed.
  3. Click on the SmartArt tab (I know, the object isn't Smart Art, it's just a shape), then click on the Shape dropdown.
  4. Select your preferred shape. The shape is changed but the animation remains the same.

If this doesn't work for your objects, you can also open the file in PowerPoint 2010, 2013 or 2016 for Windows and use the Animation Painter command to copy animations to new shapes.
