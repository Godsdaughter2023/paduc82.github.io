Proper syntax is  "C:\your path to Excel\filename"

Without the double quotes Excel is treating each space as a new file name.

C:\your.xlsx

C:\path.xlsx

etc.

Best to re-register Excel.

Try this first.

Close Excel first and On the Windows Taskbar

Method 1: Drop down list on the Left navigation panel

To get the drop drown list, first you have to edit the Master Page. To do this, follow the steps below:

  1. Go to the SharePoint team site and click Settings icon (gear icon on the top right corner) > Site settings.
  2. Click on Site collection features under Site Collection Administration.
  3. Click on Activate button next to SharePoint Server Publishing Infrastructure.

Note: You usually need to wait ~5 minutes for this features to be enabled.

      4. Go back to Site settings > Web Designer Galleries > Master pages and page layouts.      5.

       

6. Rename the copy and open it with Notepad.

       8.Save it and go back to Master Page Gallery. Click FILES > Upload Documents > Save.

       9.Select Publish Major Version > OK. 

            

      10. After making these changes, go back to Site Settings> Manage site features.

      11. Click enable next to SharePoint Server Publishing.

      12. In Site Settings> Click Master Page (under Look and feel).

      13. In Site Master Page and System Master Page, in the drop down list you’ll see the name of the document uploaded> Select the name in both the drop down and click Ok.

   

      14.Back to Site settings > Under Look and Feel, click Navigation.

       15.Scroll down to Structural Navigation: Editing and Sorting, select ‘Current Navigation’ and click on Add headings.

       17. Now select Analysis and Configuration, and click Navigation Link, in title field type Dashboard Configuration, browse URL, click OK, as you want this to be under Analysis and Configuration.

       18. Repeat the steps, till you add all the sub sites.
