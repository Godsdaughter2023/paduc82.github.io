The range A1:B1000 doesn't have 3 columns, so you should either use A1:C1000 or return a value from the 2nd column.

Assuming the former, try this:

=VLOOKUP(A1,INDIRECT("'"&B1&"'!A1:C1000"),3,FALSE)
