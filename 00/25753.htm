It sounds as though you have embedded objects rather than links to external files.

PDF will support hyperlinks to external files and when you save as PDF from PPT, this type of link is supported (so long as the hyperlinked file is available to the PDF, of course)

I don't know whether PDF supports embedded Word/etc. files; after reading up on this a bit more, I get the distinct impression that it doesn't.

You *can* use Acrobat (the full product, not the free Reader) to add files as attachments to your PDF.  Attachments appear as a list in a side pane when viewing the PDF.  You can add a custom link that opens/closes the side pane but I don't offhand see any way of creating links that launch specific attachments.
