excel data sort with grouping - Microsoft Community

rebeccasimmonds say:

excel data sort with grouping

Hi, I have grouped a heap of data (into hundreds of groups) and subtotalled each group. I now want to filter the subtotals, but excel is including the data in the group.  How do I get excel to only include the subtotalled data and not the data which sits underneath it?  I have tried paste special values only and am having no luck.  Any assistance would be appreciated.
v_agarwal say:

Hi,

1.  You can use Auto outline command from Data/Group Tab. It will create auto outline based on your data and from this outline you will be able to do filtering just by selecting 1 or 2 or 3.

2. If you are willing to take only subtotaled rows to other sheet, then you can copy only visible cells(select your data, then F5, select Visible cells only) Copy and then paste special values to other sheet.

3. You may also use one helper column and there select the whole range of new column when only subtotaled rows are visible. Suppose your data with subgroups are in 'B' column and 'A' is your new helper column. And suppose your group rows which are visible are row no. 2, 5, 8 and 10. so now select the range of A column from A2 to A10. then click F5 and select visible cells only. After that just write manually "=B2" and click CTR+Enter simultanously.  Now it will show only subtotaled range in this new column. Now use filter based on column A.

john mcghie say:
Yes, you are opening the wrong "half" of the file.

A Mac file is in two parts, for historical reasons.  There is nothing in the first half, but it's still written, for compatibility purposes.

Windows File Explorer can be set to show and manage as a single file, to show both but manage as a single, or to show and manage both parts separately.

You just need to make yourself a mental note to check the file extension when opening on Windows (or the size... the blank part is only 245 bytes...)

Better yet:  Get out of the habit of "opening" files on a Flash Drive.  Copy them to your hard drive first.  There are a variety of "bad things" that can happen if you try to edit Word files on flash drives, better not to tempt fate.

Cheers
