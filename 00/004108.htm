Hi Andy,

I understand you want to remove a recurring event from a deleted user. As you already have recreated the “deleted user” and after giving full permission to this same user, calendar event is undeletable. You can try to remove the “Full” access permission for this user, and re-apply. You can do this by using the PowerShell command:

1. List the Mailbox permission for this Room Mailbox.

Get-Mailboxpermission <Resource Mailbox> -User <User Mailbox> | fl

Note: Please change the <Resource Mailbox> with you Room Mailbox address and the <User Mailbox> with the “deleted user” Email address.

2. Remove the “fullaccess” rights permission for the “deleted user”

Remove-MailboxPermission -Identity <Resource Mailbox> -User <User Mailbox> -AccessRights fullaccess

Type “y” if there is a prompt asking you to continue this action.

3. Add again the “fullaccess” rights permission for the “deleted user”

Add-MailboxPermission -Identity <Resource Mailbox> -User <User Mailbox>  -AccessRights fullaccess

Now check if you can now remove the recurring meeting in OWA or Outlook client.

If the PowerShell command doesn't work, please check with other users that have Full Access permission on the Room calendar, if they can remove the recurring entries on the Room calendar

Looking forward to you updates.

How to rank a series of Calls in excel that are not all in order - Microsoft Community

yp989 say:

How to rank a series of Calls in excel that are not all in order

Hello!

I am looking to rank a series of cells in Microsoft excel.  The issue is that not all of the cells are in a line.

I want to Rank I5 - I18 and I24 - I36. 

That is 27 Values total even though they are not all in a row.

Is it possible to do this using the 'Rank' or another relavant function??

Thank you!

vijay a. verma say:
You just need to use following formula and drag down

=RANK($I5,($I$5:$I$18,$I$24:$I$36))

If you are dragging into rows 19 to 23 also, then use IFERROR block. Hence formula would be

=IFERROR(RANK($I5,($I$5:$I$18,$I$24:$I$36)),"")
