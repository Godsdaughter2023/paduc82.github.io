bernie deitrick say:

You don't really need code - you could use a column of formulas like this starting in cell I2

=IF(COUNTIFS(Sheet2!A:A,A2,Sheet2!B:B,B2,....,Sheet2!H:H,H2)>0,"Already exists in Sheet2!","OK")

If you need to use VBA, the just transform the above formula into code:

If Application.CountIfs(Worksheets("Sheet2").Range("A:A"), ActiveSheet.Cells(ActiveCell.Row, "A").Value, ...... , Worksheets("Sheet2").Range("H:H"), ActiveSheet.Cells(ActiveCell.Row, "H").Value) > 0 Then MsgBox "Exists!"
