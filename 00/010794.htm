andreas killer say:

You are right, this is a bug. But IMHO it did not affect the result or the calculation precision of the solver itself.
 
The solver code stores the value in the name "solver_pre" in the active sheet (if no sheetname is given) by passing the value to the RefersTo property.

  Names.Add Name:="'" & ActiveSheet.Name & "'" & "!solver_pre", _
     RefersTo:=Application.Evaluate(Precision), Visible:=False

This is correct, because if we display the name with the name manager we can see the value with the correct language punctuation: "=0,000001"

But in fact the name holds a string with a dot every time! "=0.000001"

The Solver retrieves the value from the name by calling the CDBL function which is wrong, because CDBL take notice of the language settings. And in europe the dot is the thousands separator.

  SolvGet = CDbl(Mid(CStr(ActiveSheet.Names("solver_pre")), 2))

If they would use VAL instead, the result is correct. See example below.

Sub Makro1()

    SolverSolve True
  Next
End Sub

diane poremsky mvp say:
If you need to know when is scheduled to start and end, you can use PowerShell.

Get-Mailbox -ResultSize Unlimited | Get-MailboxAutoReplyConfiguration | Where-Object { $_.AutoReplyState -ne "Disabled" } | Select Identity,StartTime,EndTime,AutoReplyState

for a simple list of start and end times

for more details of a specific user, use this:

Get-MailboxAutoReplyConfiguration -identity alias

steve rindsberg say:

I'm not sure if this will do it for but it'll get you closer, I suspect:

Instead of leaving your picture placeholders  on the masters/layouts blank/unfilled, fill them with a picture (it could be any sort of dummy photo, a texture, or even an image that's completely blank white, just so it's an image of some sort).

Once you do that, the option to "Rotate with shape" appears; it's checked by default, so uncheck it.

Now when your users add a slide based on that layout and click the placeholder to insert a picture, it'll inherit the settings from your master/layout placeholder.
