If above is what you require then create 9 separate conditions; one for each value from 121 to 129 in column B.

Yellow RGB is 255, 255, 0

Orange RGB is 255, 192, 0

I got the above values by using Fill -> and select yellow from the standard colors palate. Then while still on the same cell select Fill again and select "More colors" -> Custom tab and the RGB values were displayed as 255, 255, 0.

You have 10 values from 120 to 129 that is 9 steps, I subtracted the 192 from 255 for the G component of RGB and it is 63. Dividing the 63 by 9 is 7 so by stepping the G component of RGB from 192 to 255 in steps of 7 gives the changing color.

=AND($B4=120,$C4<=80)

  • Select More colors -> and then Custom tab

=CalcMins([@[PGM Time]])

Function CalcMins(ByVal S As String) As Variant
  If InStr(S, "h") > 0 Then
    S = Replace(S, "h", ":")
    CalcMins = CDate(S) * 24 * 60
  Else
    CalcMins = Application.Evaluate("=" & S)
  End If
End Function

andreas killer say:

You must use the BeforeSave event to achieve this. Add the code below to the code module "ThisDocument"

Andreas.

Option Explicit

Dim WithEvents App As Application

Private Sub App_DocumentBeforeSave(ByVal Doc As Document, _
    SaveAsUI As Boolean, Cancel As Boolean)
  'This event runs on all documents, so be sure we get our own
  If Doc.AttachedTemplate.Name = ThisDocument.Name Then
    'If never saved or user does a "save as"
    If SaveAsUI Then
      With Application.FileDialog(msoFileDialogSaveAs)
        'The second filter is the DOCM filetype
        .FilterIndex = 2
        'Show the dialog
        .Show
        'Save the file as the user has choosen
        .Execute
      End With
      'Cancel the original dialog
      Cancel = True
    End If
  End If
End Sub

Private Sub Document_New()
  Set App = Application
End Sub

Private Sub Document_Open()
  Set App = Application
End Sub
