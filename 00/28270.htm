cortig say:

Just to be clear:

When Outlook rebuilds the identity, it copies all the data to a new, clean identity, then renames the old one adding the Backed-Up tag to the name for safe keeping.

If you have everything in the new identity (and you make backups :-> ), then you can decide to get rid of the old identities.

It won't affect Outlook in any case since it's not using them at all. They are here so that you can revert to them in case you realize you absolutely have to.

Outlook does NOT make automatic backups per se though. That's something you have to do. If you use Time Machine though, chances are that the identity folder is within your backup set.

As for the iCloud Documents folder, as you've found out since, it obviously is the cause of the email disappearance. Luckily that's easy to fix :-)
