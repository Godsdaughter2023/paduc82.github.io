    • net stop wuauserv

  4. In the Save in box, click Desktop.
  5. In the Save as type box, click All Files.

There is a difference between a sim,free (factory unlocked/unbranded) device and a carrier released version.

your challenge with updates when you have a carrier branded device is that the carrier usually has very few updates to devices. And, their firmware versions are different, which include bloatware.

it is either software they are paid to put in devices, or their own renditions of billing, WiFi calling, mapping, etc stuff. Sub Concat()
  LastRow = Cells(Rows.Count, "A").End(xlUp).Row
  mystuff = ""
  For k = 1 To 8
   For j = 2 To LastRow
    If Cells(j, "A") = k Then
      mystuff = mystuff & Cells(j, "B") & ", "
    End If
   Next j
   Cells(k + 1, "E") = Left(mystuff, Len(mystuff) - 2)
   mystuff = ""
  Next k
End Sub

Sub MakeList()
  LastRow = Cells(Rows.Count, "A").End(xlUp).Row
  mycount = 4
  For k = 1 To 8
   For j = 2 To LastRow
    If Cells(j, "A") = k Then
      mycount = mycount + 1
      Cells(k + 1, mycount) = Cells(j, "B")
    End If
   Next j
   mycount = 4
  Next k
End Sub

Sub Concat()
