marshall barton say:

Navigation form controls can only have ONE subform open at a time.  That makes it the wrong tool for what you want.


Instead, use a tab control with a different subform on each tab page.  At least the tablcontrol allows all the subforms to be open all the time, whether you can see them or not.


The LinkMaster property must be either a field or control on the main form and the LinkChild property must be a field in the subform's record source table/query.  That means you have to use some extra stuff when you want to link two subforms.  The "master" subform needs to save the current record's linking field's value in a (invisible) main form text box (named txtLink) using a line of VBA code in the "master" subform's Current event procedure:

   Parent.txtLink = Me.{name of primary key field}
