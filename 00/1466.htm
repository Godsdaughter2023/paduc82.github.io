Private Sub Workbook_Open()

dirk goldgar say:

If it's a multiselect combo box, I believe that means the form must be bound and the combo box must be bound to a multi-value field.  In that case, you can query related records by referring to the multi-value field in the record to which the form is bound, so long as the form includes the primary key of that table.

I'll clarify by example.  Suppose you have table Table1, containing (among others) fields T1ID (the primary key) and MVF (the multi-value field to which the combo box on the form is bound).  Suppose also that Table2 contains records that are related to Table1 by having T2ID (the primary key) = one of the values selected in Table1.MVF.  Finally, suppose that you have a form, frmTable1, that is bound to Table1.  If you want the query to return those records in Table2 that are selected by field MVF for the record that is current on frmTable1, then the query could have this SQL:

SELECT * FROM Table2 WHERE Table2.T2ID In

     WHERE Table1.T1ID=[Forms]![frmTable1]![T1ID])

neyagrl say:

Get rid of blank pages from a résumé

You won't be able to delete the final paragraph marker but you can "cheat" a little! Try this:

 

1. Select the final paragraph marker (the "backwards P" mark).

2. Right click on it and choose Paragraph... from the pop-up menu.
 
 
3. Under the Spacing category, make sure you have 0 in Before: and After:

4. Click OK if you made any changes to the Spacing.

5. With the paragraph marker still selected: click on the little arrow in the Font Size button in the Font category and type 1 (you won't find it to select in the list).

One more idea...

Do make sure your hidden characters are all showing. If you can not see the ⁋  symbols then you do not have the hidden characters showing as others have instructed you to do.

Now scroll to the end of page 1 (or what should be the end of the resume text) and look for a Section Break. If you find one, delete it. It maybe just a small double line attached to the end of the last paragraph on the page so look carefully.

Just last month I had a film student with the exact same problem of a blank second page on her resume. It turned out to be a Section Break attached to the last paragraph of page 1... maybe your issue has the same solution.
