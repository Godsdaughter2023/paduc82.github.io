rory archibald say:

You can use UDFs in CF but you really ought to add error handling. Even then, an error in the UDF called by the CF can result in any code that triggers the CF simply terminating without warning, which can be extremely hard to debug, so I'd avoid it generally.

FYI, I'd rewrite your UDF to use the dictionary for the results too, and avoid the WorksheetFunction call:

Function UniqueValues(ValueRange As Range) As Variant
   Dim dic  As Object 'Dictionary
   Dim c    As Range
   Set dic = CreateObject("Scripting.Dictionary")
   For Each c In ValueRange.Cells
      If c.Row Mod 2 <> 0 Then
        dic(CStr(c.Value)) = c.Value
      End If
   Next c
   UniqueValues = dic.keys
End Function
