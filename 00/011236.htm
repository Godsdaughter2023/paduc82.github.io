daniel g. grau say:

Try disabling Word > Preferences > Spelling and Grammar > Check Spelling as you type and other related options, as well as Check grammar as you type further down. Those two interactive modes are what cause the squiggly red and green underlines to appear.

Also uncheck options in Word > Preferences >AutoCorrect, in particular Capitalize first letter of sentences.

bob jones aka: cybertaz say:

Other thoughts...

  • On the Preferences page Daniel referred to, leave the automatic features active -- those checks enable/disable the features altogether. Instead, use the Preference to Hide Spelling/Grammar errors in this document on a per document basis.
  • Go a bit further by clicking the Settings button on that page, then fine tune the grammar checking criteria to suit your writing style.
  • In most poetry, a new line often does not equate with a new paragraph... Get in the habit of using Shift+return to create a line break [aka soft return] rather than just using return which creates a paragraph break. That will obviate the initial caps issue provided the preceding line has no end punctuation.

As for the markup returning, yes, that could be related to repeatedly swapping a document with Pages. Pages cannot save to .doc/.docx files types. Instead, when changes are saved to a Word document Pages creates a new file. The program's features are not comparable to Word's so some internal instruction is bound to be dropped. It's also possible that some level of corruption has been introduced into the document.

Typically, errors marked Ignore/Ignore All should retain that unless the Preference pane is used to Recheck Document. However there are some conditions that might cause the setting to be lost. A more complete & accurate history would be required in order to get a better idea of what might be happening in any given case.

eg. just type microsoft in the IE address bar.... becuase it is a single word (without a com or www parts) IE won't be able to resolve an internet address from it and will use the default Search Provider to search for it or similar terms... Since Google is you default search provider, it will automatically use the "Feeling lucky" features... because the search terms that are passed to it do not contain the domain prefix or suffixes.

However, the only drawback of this is that these Keywords will not appear in the IE Smart Address bar...

If you have not already done so... turn on the Smart Address bar autocomplete features.... Content tab of Internet Options... Autocomplete section, Settings button.. check "Address Bar" then Favorites, History and "Use windows search..."

Close all IE windows and then restart IE. Now when you start to type something in the IE Address bar you will be offered options from your internet history and favorites... for example... typing micro will give you a list of sites that you have previously visited or that are in your Favorites folders... then Pressing Shift+Enter... IE will use the item at the top of the list (the most likely match) and navigate to the site without you having to type any further...

In case you don't realize it, there is no such thing as luck.... The universe is deterministic, there is no god and computers can't read minds. Google just uses smoke and mirrors and the gullability of humans to make you think magical luck is involved... In reality it is computing logic.
