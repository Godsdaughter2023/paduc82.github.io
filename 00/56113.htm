You can turn off hibernation if it is on and you don't use it..

When you hibernate your computer, Windows saves the contents of the system's memory to the hiberfil.sys file. As a result, the size of the hiberfil.sys file will always equal the amount of physical memory in your system. If you don't use the hibernate feature and want to recapture the space that Windows uses for the hiberfil.sys file, perform the following steps:

- Start the Control Panel Power Options applet (go to Start, Settings, Control Panel, and click Power Options).
- Select the Hibernate tab, clear the "Enable hibernation" check box, then click OK; although you might think otherwise, selecting Never under the "System hibernates" option on the Power Schemes tab doesn't delete the hiberfil.sys file.
- Windows will remove the "System hibernates" option from the Power Schemes tab and delete the hiberfil.sys file.

You can control how much space your System Restore can use...

1. Click Start, right-click My Computer, and then click Properties.
2. Click the System Restore tab.
3. Highlight one of your drives (or C: if you only have one) and click on the "Settings" button.
4. Change the percentage of disk space you wish to allow.. I suggest moving the slider until you have just about 1GB (1024MB or close to that...)
5. Click OK.. Then Click OK again.

You can control how much space your Temporary Internet Files can utilize...

a. user say:

Hi John,

There's a chance that somehow, on the process of opening the contents of the card, it might have been interrupted and it became corrupted. Here are some troubleshooting steps you can follow to rectify the concern.

Method 1: Troubleshooter

Please run the the following diagnostic tools first as a start. It automatically detects and fixes most incompatibilities in your system.

  • Troubleshoot problems with devices and printers
  • Find and fix problems with devices and hardware

  1. Go to Control Panel, on the Search Control Panel bar located on the upper-right corner of the Control Panel window, type Troubleshooting.
  2. Click Troubleshooting on the results pane.
  3. On the left pane, click View all.
  4. Look for Hardware and Devices, right-click, and have it Run as administrator.
  5. Follow the onscreen instructions.

Method 2: SFC

Please run the System File Checker tool as well to be able to detect and fix concerns with your entire system files. You may refer to this article: Use the System File Checker tool to repair missing or corrupted system files.

Method 3: Clean boot

  1. Press Windows key + R, the Run dialog box should appear. Type in services.msc. Hit Enter.
  2. A new window should appear, go to Services tab.
  3. On the bottom left corner, put a check on Hide all Microsoft services.
  4. Click Disable all.
  5. Go to Startup tab, then click Open Task Manager.
  6. A new dialog box would appear. Click each item one by one, then click Disable at the bottom for each item.
  7. Close Task Manager. Then on the Startup tab of msconfig, click OK, and then restart the computer.
  8. After the computer restarts, check if issue still persists. T Scroll down and look for How to determine what is causing the problem after you do a clean boot.

Method 4: CMD

  1. Type CMD in the Cortana search box.
  2. Right-click on Command Prompt in the results. Choose Run as administrator.
  3. In the Command Prompt window, type diskpart and press Enter. A new window usually appears.
  4. Type list disk to list all the available drives and press Enter.
  5. Find which one is the SD card. You can usually base it from the memory size. Type select disk + disk number (e.g. select disk 0) to select the SD card you want to format and press Enter.
  6. Type clean to clean the SD card you have selected and press Enter.
  7. Type create partition primary to create a partition on the cleaned SD card and press Enter.
  8. Type format fs=ntfs (or format fs=exfat) and press Enter to format the drive with NTFS or exFAT.
  9. Type assign letter=drive "letter" (e.g. assign letter=E) to assign a drive letter and press Enter.

Hope this helps. If  you have any other concerns, don't hesitate to come back to us.
