geoffc1 say:

If your photos are in the JPEG format, then Windows Photo Gallery should be storing the face recognition tags (the "People Tags") in two places:

 

1) in the local database of Windows Photo Gallery (usually held in C:\Users\Your Username\AppData\Local\Microsoft\Windows Live Photo Gallery), and

 

2) as People Tag metadata in the actual photos themselves.

 

You can check for (2) by using Windows Explorer to view your photos. Select one, and if it contains People Tags, then Windows Explorer will display them as part of the photo information in the Details pane.

 

If you have photos taken in a RAW format, then Windows Photo Gallery will only store face recognition tags in its local database - it won't store them into the photos themselves.
