ruwim.b say:

This is a known bug in the CSV export file. The original timestamp in the chat database file, the main.db file, is saved as UNIX Time.

 

https://www.unixtimestamp.com/

 

Apparently, the people who created the export file didn't know how to handle UNIX Time and to convert it to a "human time".

 

Open the .csv file in Excel. The original Unix Time is in column F. In column H enter this formula:

 

=(((F2/60/1000)/60)/24)+DATE(1970,1,1)

 

Format the cells in this column to show your preferred Date format.

how to change font on whole document - Microsoft Community

lepha say:

how to change font on whole document

I would like to know how to change the font on a whole document I need to make the font smaller
suzanne s. barnhill say:

The quick-and-dirty way is to Select All (Ctrl+A) and change the font.

A better way is to right-click in any paragraph, and choose Font, which will open the Font dialog. Select the font and size you want and click Set As Default. You'll get a dialog asking whether you want to change the font for just this document or all new documents based on the attached template. In this case, you probably want to change it just for this document.

If you have a mix of fonts in the document and want to make all of them a little smaller, there are two approaches:

  1. Ctrl+< will reduce all font sizes to the next smaller size listed in the Font dropdown.
  2. Ctrl+[ will reduce all font sizes by one point.

Ctrl+> and Ctrl+] will increase font sizes correspondingly. Ctrl+A before using these shortcuts.

Still another approach is to use the ShrinkOnePage command, which is described at http://answers.microsoft.com/en-us/office/forum/office_2010-word/how-to-find-the-shrink-to-fit-option-in-word-2010/33e0e9b6-ae7e-4305-971f-5d6de6998be6. Note that you don't have to customize the Ribbon to add this command; you can add it to the Quick Access Toolbar. Or you can add the classic PrintPreviewEditMode command, which displays the classic Print Preview; the Shrink to Fit button is on the Ribbon in that view.

joanne li msft say:

Hi Michael,

I'd like to explain that with the Windows 10 Creators Update, Win+Shift+S uses the Window Clipping Tool and sends screenshot to the clipboard. We need to press Ctrl + V to complete inserting the screenshot to your notebook.

If you want to continue using the old behavior, you can right-click on the OneNote 2016 icon and select “Take screen clipping.” For power users, you can also try reassigning the behavior to a different keyboard shortcut. Please note that this is only recommended for advanced users.

Changing the keyboard shortcut for OneNote 2016:

1. Open RegEdit.exe

2. Go to HKEY_CURRENT_USER\Software\Microsoft\Office\16.0\OneNote\Options\Other\

3. Add a new DWORD with the name “ScreenClippingShortcutKey” with the value “0x41”

4. Note: You can use any character code you want. In this case, “0x41” is the ‘A’ key

5. Restart your computer

6. You can now use Win+Shift+A to take a screen clipping

Regards,

Joanne
