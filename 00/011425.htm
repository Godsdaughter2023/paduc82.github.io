teylyn say:

Hello, you can do this with an approximate match lookup formula. Consider this screenshot:

The formula in cell F2, copied down, is

=VLOOKUP(E2,$B$1:$C$5,2,TRUE)

If the formula cannot find an exact match in column B for the value in cell E2, it uses the next smallest value in column B and returns the value from that column.  So, it can't find 2.5, so it uses the next smallest, which is 0 and returns 11. In the next row, it can't find 4, so it uses the next smallest and returns 22.

The crucial argument is "TRUE" as the last parameter of the Vlookup function. With this parameter set to TRUE, the lookup table must be sorted ascending by the first column.

suzanne s. barnhill say:

The problem with the Symbol dialog suggests a possible printer driver problem, or it could be that repairing the Office 365 installation would cure it, but there are much easier ways to insert accented letters than through the Symbol dialog.

If you are typing a mass of text in French, you might want to enable an International keyboard layout for the purpose. If you just need to include the occasional French word, you can use Word's built-in keyboard shortcuts for accented letters. See a list of these in the "International characters" section at http://wordfaqs.mvps.org/InsertSpecChars.htm. When you get used to typing, say, Ctrl+', e for é, it will become second nature, so that you hardly think about it.
