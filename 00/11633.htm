jaredcheng_msft1034 say:

Hi Mike,

Generally, if we don't add additional lookup columns like the three columns you deleted when we create Classroom Notebooks, the lookup column numbers will be just 8 which meet the threshold of lookup columns in SharePoint online, and the error message will not occur. Therefore, to avoid this issue, we should not create extra lookup columns when creating Classroom Notebooks.

Meanwhile, I'd like to explain that, by default, SharePoint will only process 8 lookup columns in response to a query. The SharePoint server blocks queries that exceed this threshold rather than providing an incomplete response. It’s not feasible to change the threshold in SharePoint Online because increasing this value from the server side is likely to cause performance problems for multiple tenants. Your understanding will be appreciated.
