Hi,

 

This will return the maximum number that's an integer and ignore any number that has decimals.

 

It's an ARRAY formula, see below on how to enter it.

 

=MAX(IF(A1:A20=INT(A1:A20),A1:A20))

 

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
around the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.

shirley wang msft say:
Hi Webbrewers3,

Without Exchange Online, to create an Office365 group as a global admin, we can follow the steps below:

1. Go to the Office 365 admin center with the new UI.
2. Select Groups > Groups > View > select Office 365 Group view.
3. Click Add a group to create a new one.
4. Check if the group display in Planner page.

If you are not the global admin and just have Planner service, it is not feasible to create an Office 365 group for it. Given the situation, I’d like to invite you submit feedback to our dedicated Microsoft Planner User Voice forum. We appreciate your understanding and any ideas you may share.
