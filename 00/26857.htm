Error code 8007000e

8007000e fix when installing windows phone

a. user say:

Hello viet_553:

 

Thank you for posting to the forum regarding the error code 8007000e you are receiving.  From my experience this error code often occurs during a phone update and is caused by space issues.  Please let us know if you get this error when doing something other then trying to update the phone.

 

If this is happening when you try to update the phone we need to try and clear some space on the phone and on the PC you are using to try the update.  The first thing I would suggest is to try and clear some room on the phone.  Start by backing up all pictures, videos and music to your computer, http://support.xbox.com/en-US/zune/software/syncing-media.  Once you have completed this, begin deleting pictures, music, podcast and videos from the phone.  Once you have removed all of the media from your phone, uninstall a few apps that you do not need at this time, they can always be installed at a later time.  I would suggest you verify the app is still available on the marketplace before you uninstall it.  Once you have cleared some significant space from the phone, try the update again.

 

If the issue persists on this PC, try making some additional room on your computer.  Look for software, media and documents that can be erased or that can be saved to your SkyDrive account and then deleted. 

 

In addition I would suggest you clear your PC temp folder:

  • For a Windows 7 PC, click on the start button and then type %temp% in the search box and press enter.  Click the first file or folder and then scroll to the bottom of the list.  Press and hold ctrl. and then click the last file or folder.  Right click the selected files and folders and click the delete option.  If any files or folders refuse to delete, do not worry and simply skip those.
  • For a XP machine you will click start and then run.  In the search box type %temp% and follow the same steps provided above.

Once you have cleared some space on this PC, try the update again.  If still have the same error, try updating from a different PC, this could be a greater issue on the PC or may be the phone itself.  Trying a second computer will tell us a lot about which is the case.

 

Some additional information may be helpful if you continue to have the same experience.

  • What is the make and model of your phone?
  • Who is your mobile operator?
  • Is your phone informing you an update is available, you can find this by tapping settings > phone update?
  • Have you successfully updated this phone in the past?  If you have, when did you last update the phone?

Please let us know if you have any other questions or concerns.

 

Mark W.

You could use something like this:

Dim strFile As String
Dim strPathToUse As String
Dim xlbook As Workbook
Dim xlsheet As Worksheet
Dim strfolders As String
Dim arrfolders As Variant
Dim i As Long
arrfolders = Split("folder1,folder2,folder3,...folder15", ",")
For i = 1 To unbound(arrfolders)
    strPathToUse = arrfolders(i)
    strFile = Dir$(strPathToUse & "*.xls")
    While strFile <> ""
        'Open the workbook
        Set xlbook = Workbooks.Open(strPathToUse & strFile)
        With xlbook
            .ActiveSheet.Unprotect Password:="NOM"
            Set xlsheet = .Sheets("November 11")
            With xlsheet.Range("D4")
                .FormulaR1C1 = "='October 11'!R[1]C"
                .Protect DrawingObjects:=False, Contents:=True, Scenarios:= _
                False
                On Error Resume Next
                .Protect Password:="NOM"
            End With
            .Save
            .Close
        End With
        'Next file in folder
        strFile = Dir$()
    Wend
Next i

where folder1, folder2, etc are of the form "c"\folderpath\foldername\"
