The errors you are receiving indicate that the file locks applied by the network server are not being released.  Network servers constantly lock and unlock files, to prevent more than one user editing them at a time.

When Word recognizes a spelling error, it shows a different right-click menu than when
there is no spelling error. To get the menu you want, first right-click and choose Ignore.
Then right-click again, and you'll get the usual menu.

Menus appear based on context. There's no way around that. In Word 2003 and
