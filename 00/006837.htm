suzanne s. barnhill say:

When any setting in Word doesn't stay set, it is because the location where it is supposed to be saved is not being properly saved. In some cases, this is the Normal template; quite a few add-ins prevent the Normal template from being saved, and, although there are workarounds, the ultimate solution is to disable the problem add-in.

In this case, I would guess that the setting is saved in the Registry. It may be that you can give the Registry a kick in the pants by deleting the Word Data key (be sure to back it up first, and you may also want to make a note of any other options you have set to something other than the default).

The article at http://word.mvps.org/FAQs/AppErrors/MissingMenusEtc.htm is an old one but still mostly applicable. It suggests some troubleshooting steps and explains how to edit the Registry. The relevant key for Word 2013 would be

\\HKEY_CURRENT_USER\Software\Microsoft\Office\15.0\Word\Data

Be sure to also look at the linked article http://word.mvps.org/FAQs/Customization/DataKeySettings.htm.

Build 9926 Assesment - Microsoft Community

danwdoo say:

Build 9926 Assesment

This build seems to be another mixed bag. There are some positives, but some real negatives as well. Here are some basic thoughts after using it for a couple of days:

Start Menu

This one is being well debated in many threads. Here are my current thoughts:

Likes

  • Ability to sort modern apps into groups
  • Right clicking on non-modern items gives 'open file location' and uninstall options. Nice! 

Dislikes

  • Inability to pin programs to the left side now occupied by places. This was useful as a secondary tier taskbar list of programs that was useful, especially with apps that take advantage of jump lists.
  • Inability to resize - I know this is coming, just want to emphasize the need..
  • removal of run/search box in the start menu. Cortana takes over this functionality but having a window pop up on top of another window is clumsy. I also don't want this feature cluttered up with news and other items.
  • All programs is a horrible, endless scrolling mess. I'm not even sure I can tell you what the solution is, but this isn't it. Please spend some real time getting this one right.

Taskbar

Likes

  • Finally no transparency! I'm sure others will want this, however, so please add an option for it without allowing people like me to have a solid taskbar without having to resort to some hack like high contrast settings.
  • underlined icons that are running

Dislikes

  • Cortana search window outside of the start button.
  • Search and Cortana being mixed. Too much is going on (too busy) when I hit it to search for something.

Action Center

Likes

  • basic notification functionality
  • Shortcuts to common features

Drivers

Drivers seem to be a bit of a mess right now. I have the following devices installed with drivers, but they are not working:

  • Mobility Intel 4000 graphics processor
  • Mobility AMD 7600m graphics processor
  • Validity fingerprint reader

All three of these devices and drivers were working correctly in the last WTP build.

Misc. Problems

  • Screen brightness cannot be adjusted by keyboard or by Windows settings (volume works fine).
  • Sleep is not even an option anymore-only hibernate. Sleep worked fine in the previous build.

Misc. Suggestions

  • I much prefer the wireless network joining in Windows 7 that popped up and allowed one click joining without having to open a full screen that needed to be closed. Less is more when it comes to clicks!
  • Please make sure all options have choice for Windows 7 functionality (desktop, start screen) or Windows 8 functionality (start screen). Give users choice!
  • Please include different icon sets. Some like big and colorful. Others like high contrast or 'industrial' design. Please include a couple different sets to appeal to a broader audience. 1.5 Billion users, remember!
  • Group related items together in file explorer's right click menu. Right now 'Hide from frequent' and 'unpin from frequent' are separated into distant sections.
