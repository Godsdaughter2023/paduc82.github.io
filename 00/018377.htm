Equation Editor - Microsoft Community

susan ramsey say:

Equation Editor

I have the Microsoft Office 365 for teachers.  When I go to Insert Object I can usually insert an equation using the equation editor but now it is gone!  Help! As a math teacher I use this all the time.  

jay freedman say:

Microsoft removed the old (ancient!) equation editor because it has security holes (read https://support.microsoft.com/en-us/help/4057882/error-when-editing-an-equation-in-office).

All versions of Office starting with Office 2007 have a different equation editor, which you can use instead -- click Insert > Equation or press Alt+= to start it. Unfortunately, existing equations from the old editor currently can't be converted to the new format.

If you must work with the old equations, get the trial version of MathType as described in the article. When the trial period ends, if you haven't paid for the program, it converts to the MathType Lite version that's essentially the same as the old equation editor.
