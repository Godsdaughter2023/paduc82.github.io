amit tandon say:

Create 2 Named Lists (Formulas > Name Manager) with the names One & Two: One refers to =1, Two refers to =2

Custom Number Format cells A1 & A2 (Format > Format Cells > Custom - add this format): [=1]"Currently seeking staff";[=2]"Service break-weather";

In cells A1 & A2 enter: =One & =Two

Cells A1 & A2 will display: Currently seeking staff & Service break-weather

In cell B2, Create Data Validation(Data > Data Validation): Allow: List; Source: =$A$1:$A$2

Cell B2 will display a drop-down: selecting 'Currently seeking staff' will display 1 & selecting 'Service break-weather' displays 2

jay freedman say:

Markup (also called Track Changes) can be turned on and off by the shortcut Ctrl+Shift+E. Unfortunately, it's easy to hit that combination when you think you're typing just Shift+E. If you aren't familiar with what tracked changes look like, it's common to be confused by it.

For one thing, simply choosing No Markup doesn't remove the markup from the document, it just hides it. If you send that document to another user, they'll be able to see all the changes you made -- and under some circumstances that could be embarrassing. You must either Accept or Reject each change, using the buttons with those labels on the Review ribbon.

For more information about the feature, see http://shaunakelly.com/word/sharing/howtrackchangesworks.html
