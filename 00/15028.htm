daniel g. grau say:
What I'd recommend is AutoCorrect. Although it's intended to correct common spelling mistakes (take a look at Word's predefined list of corrections), I regularly use it for the same purpose you describe.

For further convenience, assign a shortcut to the AutoCorrect feature:
  • Select Tools > Customize Keyboard
  • Highlight All Commands on the list to the left
  • Locate ToolsAutoCorrect on the list to the right
  • Type a shortcut in the field
  • Click OK
Most Control + Character combinations are available, like Control A or even use a function key instead of a character. At any rate, Word will let you know if the combination you chose is already in use. And of course, you can combine that with the Option, Shift and Command keys.

Once a shortcut for ToolsAutoCorrect is defined, you'll be able to select some text (it's got to be the correct version of it, such as "HENRY:"), press that shortcut and define the "wrong" term to change. I recommend that you use a "terminator" symbol so you can still type isolated Hs. I prefer to use a backward slash, so in this case, I would use h\.
