Actually, I don't want any of my libraries from ANY of my user accounts to show up on other network user's computers.  How can I stop sharing all libraries?
andrew_hartley say: Open up services.msc and find the Windows Media Player Network Sharing Service. If you stop and then disable the service the media libraries should no longer show up on other computers.

But if you had a deleted user that was still showing up you might need to dig into the registry to fix it. I had the same problem with a deleted user and only just came across how to remove it.

First you need to find the SID of the deleted user before removing any items in the registry.

Look through HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\ProfileList. You will find sub keys which represent SID's of users on the system. When click on each sub key you will see on the right side ProfileImagePath which can help determine who the user is. 

Once you have the SID go to HKLM\SOFTWARE\Wow6432Node\Microsoft\MediaPlayer\PREFERENCES\HME (HKLM\SOFTWARE\Microsoft\MediaPlayer\PREFERENCES\HME if you are using 32bit Win 8). In here will be sub keys using the same SID you found under the ProfileList key. Remove the key folder that relates to the SID you want to remove. To remove the SID folder you might need to take ownership of the SID key folder.

Then go to HKLM\SOFTWARE\Microsoft\Windows Media Player NSS\3.0\Media Servers. In here will be another SID related folder that you remove. Once both keys have been removed restart the Windows Media Player Network Sharing service.
