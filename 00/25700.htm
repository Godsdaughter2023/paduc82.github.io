Sum of probabilities equal to 10 when using Excel distribution - Microsoft Community

waterandspirit say:

Sum of probabilities equal to 10 when using Excel distribution functions

When I use probability density function in EXCEL (NORM.DIST or WEIBULL.DIST), the sum of probabilities [for all possible x's] is equal to 10, instead of 1. Why does that happen? Should not the sum equal to 1? When I do the cumulative distribution function, it does end in a 1 though...
mike middleton say: A probability density function is for a continuous-valued variable. A continuous-valued variable has theoretically an infinite number of values. When you set the Excel function "cumulative" argument to FALSE for a continuous-valued variable, the function returns the height of the density function. The height of the density function is not a probability. Instead, an area under the density function for a range of values is a probability, and the total area equals one. Theoretically, the probability of a specific value is essentially zero. Examples of continuous-valued variables are the normal and Weibull distributions.

A probability mass function is for a discrete-valued variable. When you set the Excel function "cumulative" argument to FALSE for a discrete-valued variable, the function returns the probability mass function, i.e., the probability of a specific discrete value. The sum of probabilities for a discrete-valued variable equals one. Examples of discrete-valued variables are the binomial and Poisson distributions.

-  Mike Middleton, www.MikeMiddleton.com

Hi shoeic

Create an Event Procedure for the On Click event of the control.

I used a button called cmdButtonName

This is the code you need for the On Click event.

Private Sub cmdButtonName_Click()
On Error GoTo Err_cmdButtonName_Click

    If MsgBox("Are you sure you want to continue?", vbYesNo + vbQuestion, "Confirmation Required") = vbYes Then
        DoCmd.RunMacro "YuorMacroName"
    End If
   

Exit_cmdButtonName_Click:
    Exit Sub

Err_cmdButtonName_Click:
    MsgBox Err.Description
    Resume Exit_cmdButtonName_Click
   
End Sub
