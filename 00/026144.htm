bernie deitrick say:

You can do it with a User-Defined-Function:

Function Pairs(rDB As Range, rM As Integer, rN As Integer) As Integer
    Dim r As Range
    For Each r In rDB.Rows
        If rM <> rN And Application.CountIf(r, rM) = 1 And Application.CountIf(r, rN) = 1 Then Pairs = Pairs + 1
    Next r
End Function

Used like this to get a specific pair count:

=Pairs($A$1:$E$56,13,29)

or like this, to get all pairs of one versus another copied down where 1-69 are entered in F2:F70, and the one specific number is entered in F1

=Pairs($A$1:$E$56,$F$1,F2)

or like this, in cell G2

=Pairs($A$1:$E$56,$F2,G$1)

Copied down and across, where  1-69 are entered in F2:F70 and again across row 1 starting in G

amit tandon say:

Have an attempted a solution using Excel formulas. Refer below image:

Your data is in range $A$1:$E$56. Enter a number in cell F1 (29 in the above image). This number will be paired with each of the numbers in cell F2 downwards so that the number of times 29 (cell F1) appears together (ie. in the same row) with the number 1 (cell F2) will be returned in cell G2.

Enter below formula as an array formula (CTRL+SHIFT+ENTER) in cell G2 & copy down:

=SUMPRODUCT((FREQUENCY(IF(($A$1:$E$56=$F$1),ROW($A$1:$E$56)),ROW($A$1:$E$56)))*(FREQUENCY(IF(($A$1:$E$56=$F2),ROW($A$1:$E$56)),ROW($A$1:$E$56))))

In case of duplicate values, say in a row containing 29, a value appears twice in that row, say 21 appears twice, then this will counted as 2.

And note further that in case 29 (in cell F1) is paired with the same number 29 (which is in cell F30) the formula in column G will return the total number of occurrences of the number 29. You may avoid this by adding this before the formula: =IF($F$1=$F2,0,.....

suzanne s. barnhill say:

You can use a field as TrueText without quotes, but once you add anything to it, you need the quotes, so what you need here is (you can omit the FalseText since it's blank):

{ IF { STYLEREF "stylename" } <> "Error!*" "{ STYLEREF "stylename" } text" }

suzanne s. barnhill say:
All I can say is that it works as expected for me (I used Heading 1 as the referenced style). The position of the quotes is crucial. You must have quotes around the entire TrueText and around the stylename if it is more than one word but not anywhere else, lest Word interpret it as FalseText.

How to Turn Off the Calculated Columns Feature in Excel Online?

Hi there, 

How can I remove/turn off the Calculated Columns feature in Excel Online? I can find a place to turn it off in the Excel application, but not online. Please let me know. Thanks! 

 

davem121 say:
Hi,

In Excel Online, a formula result is automatically recalculated when you change data in cells that are used in that formula. You can’t turn this automatic recalculation off and you can’t calculate formula results manually by pressing F9, like you can in Excel desktop application.

You also can’t change the number of times a formula is recalculated until a specific numeric condition is met, and you can’t change the precision of calculations by using the displayed value instead of the stored value when formulas are recalculated.
