john - project say:
JMH,
Some basics on master & subprojects. When you insert a subproject into a dynamic master, the subproject is not actually part of the master. Rather, the master contains a pointer to the subproject. The only "tasks" that are actually part of the master are the summary line insertion points in the master. You can see that by looking at the ID field.

The subproject name that appears in the master is not the saved file name. It is the Title of the subproject that you see when you look at the subproject file properties. However you can change the Task Name field of the summary line insertion point for the subproject. That does not change the Title of the subproject.

I'm not quite sure what you mean by adding a note to a project. Are you referring to the Comments in file properties, the Notes field, or what?

With regard to custom fields. If you create a custom field in a subproject, that field value will be displayed in the master. However, creating the custom field at master level does NOT create the field in the subproject. It does however create the custom field for any tasks that are actually part of the master.

John

john - project say:
JMH,
If you add a note to the subproject Project Summary Task, it will not appear in the master. However a note on any other task or summary line in the subproject will appear when viewed at master level.

With regard to your custom field, after you create your custom field, go to Developer/Manage group and hit the Organizer icon. When the Organizer window appears, hit the Fields tab. Select the custom field(s) appears in the selection list on the right. Then hit "copy". Your custom field(s) will be saved in your Global file. It/they will then be available for any new project that you open.

I want macro to run when a certain cell or cell text is selected - Microsoft Community

munchkin130 say:

I want macro to run when a certain cell or cell text is selected

I know how to create macros that run off a button or an text box/object that I make look more like a hyperlink.....but I have a list that is often sorted in a different way based on what you are looking for - can I create a macro that runs when a certain cell is selected? 

hansv mvp say:

Right-click the sheet tab and select View Code from the popup menu.

If you want to react when the user select a different cell (or range of cells), create an event procedure

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    ...
End Sub

and if you want to react if the user changes the value of a cell (or range of cells), create an event procedure

Private Sub Worksheet_Change(ByVal Target As Range)
    ...
End Sub

In both cases, Target is a Range object that represents the cell(s) that have been selected c.q. changed.

So you could use something like this:

Private Sub Worksheet_SelectionChange(ByVal Target As Range)
    If Not Intersect(Range("B5"), Target) Is Nothing Then
        MsgBox "You selected cell B5!"
    End If
End Sub

jllatham say:

To add just a little to what HansV said - the Worksheet_Change() event does not trigger as the result of the change in a value determined by a formula.  It only triggers if the user enters a value or a value is placed into the cell from VBA code.

Since you mention running it by selecting a cell, that should not be a factor.

Another possible event to look at would be the
Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)

Using it allows you to select the cell without anything happening, but if you double-click in the cell, then your code would run.  As HansV said, Target refers to the cell where the DoubleClick took place.  Setting Cancel=True will then make the routine ignore the double-click to let your code function.

Code for that event might look something like this:

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)
  If Target.Address <> "$B$2" Then
    Exit Sub
  End If
  Cancel = True
  'continue with your code to run when someone
  'double-clicks in cell B2
  MsgBox "You dclicked b2"
 
End Sub
