suzanne s. barnhill say:

I can reproduce the effect you illustrate in a numbered paragraph with the following conditions:

  1. There is no hanging indent (the indent is flush left).
  2. There is consequently no left-aligned tab stop where the hanging indent would be.
  3. There is a left-aligned tab stop at the right margin.

If you have the horizontal ruler displayed, it should require only a glance to confirm whether these conditions are met.

 

The fact that you are not seeing the problem but others are suggests that possibly the others have a bizarre definition of the relevant style in their Normal template and have "Automatically update document styles" checked in the Templates and Add-ins dialog.

stefan blom say:
If this is auto numbering, note that Word 2010 (by default) does not include a tab stop at the location of the hanging indent, which could cause trouble when a document is saved in *.doc format and/or opened in an earlier version of Word. Click the auto number to select all numbers belonging to the same list. Right-click and choose Adjust List Indents; in the dialog box, make sure indentation is correctly set, with a tab stop.
