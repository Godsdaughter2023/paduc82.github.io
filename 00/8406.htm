Mail merge labels in Word 2010 - Microsoft Community

searcher_303 say:

Mail merge labels in Word 2010

I would like to mail merge lables in Office 2010, Home and Student Edition. I have established that I can use an Excel spreadsheet as the data source.

So I open Word, go to Mailings>labels and select "Single label", entering the correct number of rows and columns.

Then I click on the data source arrow and an error message tells me that I need to log on to Microsoft Exchange to access my Address Book.  How do I get ro use the.xlsx file as the data source, please?
jay freedman say:
You're in the wrong place -- the Mailings > Labels dialog is NOT mail merge, and the button you clicked is not "data source" but "Address Book".

Instead, on the Mailings ribbon, click Start Mail Merge and choose Labels from the menu. Select the correct label product in the dialog and click OK. Next, on the Mailings ribbon, click Select Recipients and choose Use Existing List in the menu. In the Select Data Source dialog (which is like the Open dialog), choose your .xlsx file (and, if needed, which worksheet in the file to use). At this point, Word creates a table with cells sized to match your label sheets.

On the Mailings ribbon, click Edit Recipient list. In the next dialog, you can check and uncheck individual recipients, use logical rules to "filter" the list (for example, choose only recipients in a particular range of zip codes), and sort the list to change the order in which the labels are printed.

On the Mailings ribbon, click Insert Merge Fields and select the first field (for example, First_Name or whatever you called it) to place on the label. Type a space and insert another field (such as Last_Name), or press Enter to go to the next line and insert a field. Continue until the top left cell of the table contains all the fields you need for the label.

On the Mailings ribbon, click Update Labels, which repeats the merge fields on all the other labels.

You can press Preview Results to see what the labels will look like. When you're ready, click the Finish & Merge button, and choose to send the result to a new document or directly to the printer. (The email choice doesn't make sense for a label merge.)

jay freedman say:
There's an easier way to get the label text vertically centered. Select the "merge to labels" and select the standard product number that corresponds to your label stock. When the table appears in the document, select the whole table (click in the table and then click the little square containing a 4-way arrow; or right-click in the table and click Select > Table). On the Table Tools ribbon, choose the Layout tab, look at the Alignment group, and click the leftmost icon in the center row.

To answer your question directly, you can type 9.91 cm into the vertical pitch box and it will be retained -- at least, that's what I see when I try it. (Many of Word's dialog boxes will round their display to the nearest tenth of a unit but will internally use two decimal places as entered; this dialog isn't one of them.)
