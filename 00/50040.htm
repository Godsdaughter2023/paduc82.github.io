Hello Bbruce,

 

Welcome to the forum!

 

Does the search results window pop up when you aren’t performing any actions (when the computer is idle)?

Does it happen when you press a certain key on the keyboard or use a certain program?

 

Try using rebooting into a clean boot to see if this is being caused by a third party program.

·         Click Start

·         Click Run

·         Type msconfig in the Open: field

·         Click OK

·         Click the Services tab

·         Click Hide All Microsoft Services

·         Click Disable All

·         Click OK

·         Click Restart

·         *wait for restart to finish*

 

Now test to see if the problem still occurs. If no windows open on their own, one of the services you disabled was causing the behavior. You can pinpoint which service by:

·         Click Start

·         Click Run

·         Type msconfig in the Open: field

·         Click OK

·         Click the Services tab

·         Click Hide All Microsoft Services

·         Enable the top half of the services

·         Click OK

·         Click Restart

·         *wait for restart to finish*

 

Once restart has finished test to see if the problem has returned. If not, then repeat the enabling process for half of the remaining disabled services. Once the problem returns, go back to the group of services you just enabled and disable them. Then enable one and restart, testing them one at a time.

 

You can also use System File Checker to make sure none of your system files have been replaced or become corrupted.

·         Click Start

·         Click All Programs

·         Click Accessories

·         Click Command Prompt

·         Enter the command sfc /scannow (you will need to have your XP disk handy)
