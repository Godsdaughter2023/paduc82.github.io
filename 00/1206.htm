diane poremsky mvp (slipstick.com) say:

Sounds like you created a POP3 account and did not leave mail on the server. You can open the Alpha's data file in the Beta profile and copy the mail over but if you want to put mail back on the server, the server needs to support IMAP - create an IMAP account in Outlook and copy the mail into the imap folders.

 

I have instructions to restore mail to the server here - http://www.slipstick.com/outlook/email/restore-pop3-mail-server/
