In case documents open in Read Mode, do the following to prevent that from happening: At File | Options | General, clear the option "Open e-mail attachments and other uneditable files in reading view."

  

Setting the Display for Review drop down to "Simple Markup" (or "All Markup," for that matter) should be sticky. In cases when it isn't, install the following macro:

    

Sub AutoOpen()

    'Edited to provide error handling suggested by Jay Freedman.
    'Edited by Stefan Blom, MVP, August 2016 (Simple Markup view)
     On Error GoTo errhdl
    With ActiveWindow.View.RevisionsFilter
      .Markup = wdRevisionsMarkupSimple   'Enable Simple Markup view
       .View = wdRevisionsViewFinal
    End With
    Exit Sub
errhdl:
    MsgBox "Error " & Err.Number & vbCr & "Couldn't set markup type", , "Error"
End Sub
