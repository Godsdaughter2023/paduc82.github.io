Publish office 365 calendar date range

poppyyang_msft6299 say:

Hi SunitaPhadnis,

Generally, the default value of the published calendar is Three Months.

To change the date range of the calendar we published out, we can run the following PowerShell cmdlet(note you will need to connect to Exchange Online Remote PowerShell by referring to: https://technet.microsoft.com/en-us/library/jj984289(v=exchg.160).aspx  to run the cmdlets):
Set-MailboxCalendarFolder -Identity <user>:\Calendar –PublishEnabled $true -PublishDateRangeFrom OneYear -PublishDateRangeTo OneYear -DetailLevel LimitedDetails

(Replace <user> with the user name of the public calendar owner)
The parameters PublishDateRangeFrom and PublishDateRangeTo in the article Set-MailboxCalendarFolder(http://technet.microsoft.com/en-gb/library/dd298124(v=exchg.150).aspx ) are for your reference.
If you encounter any error messages when running the steps above, please let us know and we will further assist you.

Best Regards,
Poppy

dbaldwin117 say: Found a solution.

1 Insert a Formatted Document into a Word 2007 or 2010 File
2 Open the Word document into which you want to insert a formatted document.
3 Place the cursor where you want to insert the formatted document.
4 Go to the "Insert" tab of the ribbon and click the "Object" button in the "Text" group. The "Object" dialog box will open.
5 Go to the "Create from File" tab. Click the "Browse" button. Find the formatted document that you want to insert into the Word file and click on it to select it.
6 Click the "OK" button. Save the changes to your Word document.

The resulting document had some margin issues which I was able to correct.
