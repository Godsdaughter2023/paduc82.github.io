dhookom say: A crosstab query requires the designation of Row Heading (Main.[Contact].[Value]), Column Heading(Main.[Title]), and Value which needs to be an aggregate like Sum, Count, Min, Max, First, etc. which you haven't set. Do you want a count of ID or Min or some other aggregate?

Maybe try:
TRANSFORM Count([ID]) AS CountOfID
SELECT Main.[Contact].[Value]
FROM Main
GROUP BY Main.[Contact].[Value]
PIVOT Main.[Title];
