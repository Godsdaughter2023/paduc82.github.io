attorraru say: This might help, it didn't for me as my computer management didn't have the Groups and Users tab, so I'm a bit stuck, but it might help you Re-enable a Windows 7 Administrator account

If you updated from Windows 7 and created a new Microsoft account when you moved, your files may be stuck in a disabled administrator account. Reenable the account and log back in to see them.

  1. In the search box, type computer management and select the Computer management app.

  2. In the left pane, expand Local Users and Groups, then open the Users folder. If the user named Administrator has a down arrow on its icon , it’s been disabled.

  3. To enable this account, double-click the Administrator icon to open the Properties dialog box. Clear the Account is disabled tick box, then select Apply to enable the account.

  4. Now log out and log back in using the Administrator account.

how do I change the user account from limited access to administrator - Microsoft Community

ct-admin say:

how do I change the user account from limited access to administrator when there's no administrator account on the machine?

how do I change the user account from limited access to administrator when there's no administrator account on the machine?

wunders say:

The built-in administrator account cannot be deleted.  It can, however, be disabled.

 

"How to disable the Local Administrator Account in Windows"

  < http://support.microsoft.com/kb/281140 >

(note the warning about making sure alternate administrator is present)

 

If you bring up a command prompt window (Start -> Run -> "cmd") and enter the command:

   net  localgroup  administrators

It will respond with a list of users currently recognized as administrators.  The built-in administrator should be one of them.  You will need to log in as this administrator to make your changes.  Assuming that administrator's password has not been changed, you should be able to log in using the steps in this KB article:

 

"How to log on to your Windows XP-based computer if you forget your password or if your password expires"

  < http://support.microsoft.com/kb/321305 >

 

That is about the best information that we are permitted to provide in this Microsoft-Supported forum...

 

"Microsoft policy about lost or forgotten passwords"

  < http://support.microsoft.com/kb/189126 >
