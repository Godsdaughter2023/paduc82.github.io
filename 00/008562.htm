Microsoft outlook 2010 psd file can't be found

I tried to open my email account today and it can't be opened due to a missing file. 

I can't find it

karl timmermans (outlook mvp 2012-2018) say:
The exact location that Outlook is looking to for that file can be determined by opening the Control Panel <Mail> app in one of 2 ways

If the email account involved is a POP3 account
#1 - From the Email tab --> highlight the POP3 account --> go down to the bottom of that window to the <Change Folder> button and take note of the folder path/file name where the POP3 is delivering the email
OR
#2 - From the Data Files tab, take note of the PST file's folder path/file name

If the email account involved is an IMAP account
#1 - From the Data file tab in Account Settings --> take note of the folder path/file name for the PST file set as the default data file for the profile

With that said, if the file no longer exists in the location specified according to the above, then it has been deleted or moved. If deleted, check your Windows recycle bin. If moved, you'll need to do a search for the specific pst file name involved. If you can't find it AND it's not in the WIndows recycle bin, unfortunately the data is lost.

If the file is lost then the simplest approach would be to
a) with Outlook closed - open the Control Panel <Mail> app
b) create a new Outlook profile and configure the email account again allowing Outlook to create the data files accordingly
c) make sure to set the new Outlook profile as the default profile for the next time Outlook is opened
*** this response assumes that only one email account is involved for the sake of simplicity

There are several things why you get the pop-up message when you tried to open the file in your OneDrive. You can follow the steps below to help you resolve the issue.

Method 1: Check the recycle bin

The files or folders might have been accidentally deleted. Go to the recycle bin on the OneDrive website. If you're syncing your OneDrive on a computer, check your desktop Recycle Bin (on a PC).

When you delete a file in OneDrive from one device, it deletes it from your OneDrive everywhere. And if you move a file from OneDrive to a different location on a device, it removes it from your OneDrive.

OneDrive automatically empties files from the recycle bin if they've been there for 30 days. If you want to keep a file that's in your recycle bin, make sure to restore it within 30 days. For more info about restoring files, see Delete or restore files and folders.

Method 2: Make sure that you are signed in with the correct Microsoft account.

If you have multiple accounts, make sure you're signed in with the one you use for OneDrive. Go to the OneDrive website, tap or click your name in the upper-right corner, and then tap or click Edit profile. The profile page shows which Microsoft account you're using.

Method 3: Search for the missing files or folders

You might have saved the files to a location you didn't intend to, or moved the files by mistake. Go to the OneDrive website and use the Search everything box in the upper-left corner.

Note: You can not only search the name of files, it also searches the contents of the files, including text in documents, as well as file metadata, photo tags and even text shown in photos.

Method 5: Check Recent for the missing items

The recent list can help you find documents that you've opened recently from anywhere in your OneDrive. Go to the OneDrive website and tap or click Recent in the left pane.
