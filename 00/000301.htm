To fix this problem create a text file somewhere you won't forget it and somewhere you won't delete it accidentally. I called mine mainstyle.css but I'm not a creative person. Inside this text file just put one line:

@-ms-viewport{width:auto!important}

In case you are wondering just add this line to your text file below the one you entered above:

body {background-color: hotpink;}

How to rank with cardinal order only - Microsoft Community

brb_eh say:

How to rank with cardinal order only

My question is about how to return an ordered ranking for a list of numerical values in which equal values don't skew the rank number of the next highest value higher.

For example, suppose I have the following list, with each number in a separate cell in a column:

8, 3, 3, 7, 2, 3, 8, 5, 9, 3

If I run the RANK (or RANK.EQ) function in the adjacent column, with each cell referencing the cell in the same row of the previous column compared to the entire group of 10 cells in that column, each cell in my ranking column returns the following list in corresponding order: 8, 2, 2, 7, 1, 2, 8, 6, 10, 2

However, I am looking to have the cells read this way: 5, 2, 2, 4, 1, 2, 5, 3, 6, 2

In other words, "2" from the original list is the first in ranking, "3" is the second, "5" is third, "7" is fourth, "8" is fifth, and "9" is sixth; each value is given a sequential position or cardinal order, without skipping any numbers the way that the RANK function does.

Is there another function, or a way to modify the RANK or RANK.EQ functions, to accomplish this task?

bernard liengme say:

You need to give a non-zero value to the third augment of RANK.EQ

best wishes

bernie deitrick say:

You can do it with a column of helper formulas....

For your values in A2:A11, in B2 enter

=IF(COUNTIF($A$2:A2,A2)=1,RANK(A2,$A$2:$A$11),"")

In C2, enter
