a. user say:

If your computer is currently activated on Windows 10, it remains activated for the rest of its life.

The way it works for Windows 10: Once Windows 10 has been activated on your computer, Microsoft only cares that you re-install Windows 10 on the same computer on which it was originally installed. So you could re-install Windows 10 every day - twice on Sunday - and as long as you re-install on the same computer you'll maintain your activation - no need to pay for a license.

Even if you go back to Windows 7. Heck, even if you back to DOS, as long as Windows 10 is re-installed on the same computer on which it was originally installed, it will remain activated.

Caveat: If you change the computer's motherboard, Microsoft considers that a 'new' computer.
