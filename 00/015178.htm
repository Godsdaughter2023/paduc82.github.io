SELECT COUNT(*) AS NameCount
FROM

ken sheridan say: To return a name which does not exist in a table you'll have to provide it as a constant, e.g.

SELECT "Tim"
FROM Table1 AS T1
WHERE NOT EXISTS
   (SELECT *
    FROM Table1 AS T2
    WHERE Column1 = "Tim")
UNION
SELECT "Basha"
FROM Table1 AS T1
WHERE NOT EXISTS
   (SELECT *
    FROM Table1 AS T2
    WHERE Column1 = "Basha")
UNION
SELECT "Sara"
FROM Table1 AS T1
WHERE NOT EXISTS
   (SELECT *
    FROM Table1 AS T2

Then you should create one.  These are data and as such should be stored in rows in a table; that's a fundamental principle of the database relational model, the Information Principle.  And it would make it a lot simpler for you.
