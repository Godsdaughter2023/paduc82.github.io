dirk goldgar say:

If your popup form "frmshortcut" is a true popup form -- that is, its PopUp property is set to Yes/True -- then this is complicated, because the main form's Activate event won't fire when the focus shifts from a popup form back to a normal form.  Here are some alternative approaches:

 

1. If you are displaying all forms in windowed mode, rather than in tabbed document mode, then you might consider turning off the PopUp property of frmshortcut, so that Access handles it as a normal form.  Then the Activate event of your main form will fire when the user clicks back on it, and you can use that event to close the pseudo-popup form.

 

2. If you don't want to take that approach, and want to keep frmshortcut as a true popup form, then you could use the GotFocus event of all the controls on the main form to close the popup form.  You wouldn't even need to create a separate event procedure for each control.  You could select all the controls at once, open their joint property sheet, and enter this on the On Got Focus property line:

 

    =[DoCmd].[Close](2,"frmPopUp2",2)

 

3. If you don't want to take either of the above approaches, I imagine you could use Windows API calls to hook the message queue of the main form's window, and trap the appropriate message(s) that would tell you that you need to close the popup form.  However, I haven't ever done this and can't give more detailed instructions at the moment.

takmilulihsan say:

Hi,

1. If I guessed correctly, you're applying filter column by column. You don't have to do that. Let's say cells A1:F1 contains all your column headers. Select A1:F1 and then apply filter. Now you'll be able to apply filter on multiple columns.

2. Since you want to "include everything" and  change page layout to landscape etc., I'm guessing you are talking about printing the desired portion in one page. In Excel 2010 onward it is easy. Just go to File>Print>No Scaling>Fit sheet on one page.

Also, if the data is always in the same format and you need to reapply the same settings repeatedly, you can try automating the task by recording a macro.
