Macro to group rows based on a value - Microsoft Community

daytonflyer2004 say:

Macro to group rows based on a value

I am looking for a macro to group rows based on a value in a cell.  For example, if cells A2 and A3 both have a '1' in them, I wan them grouped together.  Also if cells A11 through A19 each have a '1', I want them grouped too.

rick rothstein, mvp - excel say:

When you say "group", do you mean "select"? If so, give this non-looping macro a try (it should be extremely fast compared to looping through the cells individually)...

Sub SelectRowsBasedOnValuesInColumn()
  Dim Col As Variant, Word As String
  Col = InputBox("Which column has the item you want to select?")
  If Len(Col) > 0 And Not Col Like "*[!0-9]*" Then Col = Val(Col)
  Word = InputBox("What is the ""word"" whose rows you want to select?")
  With Columns(Col)
    .Replace Word, "#N/A", xlWhole
    .SpecialCells(xlCellTypeConstants, xlErrors).Select
    .Replace "#N/A", Word, xlWhole
  End With
End Sub
