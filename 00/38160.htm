bossdweeb say:

I have never used an external driver to run games from, and if using a USB connection it will probably be slow loading, but when using an external drive you do need to be sure it uses the same drive letter as when you installed the game.

However, d3dxd_43dll  is one of the most recent DirectX files ( Yes, it was released way back in 2010, but that's as new as they get). If your are using a clean install or have never updated Directx try running the DirectX Web Installer.

Download DirectX End-User Runtime Web Installer from Official Microsoft Download Center

(this is from MS and is genuinely free)

If the DX Web Installer displays a message to the effect that you have the latest version, the file may be corrupt.

In this instance the easiest fix is to delete  d3dxd_43dll from both System32 and sysWOW64 and run the

DX Web Installer again. This will replace the now 'missing' DX files with new ones.

- Using an external HDD for anything other than storage/backup is always going to be a nuisance.

If you are using a laptop a larger internal HDD is a better option. You can clone/migrate your existing Windows

to your 2TB HDD before installing it into the laptop. If you have a desktop PC install your external HDD internally.

be dealt with, but you will need the 'full' version for that task. Well they are full of something alright,

but we won't  go into that here.

- In the event that something needs doing in the Registry, only that specific issue should be dealt with.

The Registry does not require general maintenance.

System optimising tools can be handy at times, but equally not really necessary as most of the stuff they do can be

done manually, or simply by running Windows Disk Cleanup, which is built into Windows.

- CCleaner is a well regarded genuinely free system optimising tool (they do have a 'Pro' version as well, but the

free version is adequate), and while it includes a Registry cleaner, it is still not advisable to use this part of CCleaner.

While I do have CCleaner, I very rarely use it.

Even when using a tool such as CCleaner, you really should know exactly what it is wanting to 'clean'.

eg- If you use the Run Command CCleaner may erase the commands that you have typed into Run

the Advanced section).
