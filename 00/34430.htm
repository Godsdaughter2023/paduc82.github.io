1) Power up the printer, if wired unplug it's USB cable from PC.

Mail App - Selecting Multiple Messages

Is there any way to select multiple mail messages in the mail app (for deletion, moving to sub folders, etc) without using the keyboard?  You can control click with the keyboard, but I can't seem to find a spot I can tap with my finger to select more then one message at a time.

bhobg say:

It's not a really obvious gesture - what you want to do is tap the title of a message and "tug" it a little to the left or the right, then release it - it should stay highlighted and have a check mark just above the time of receipt.

 

You can do this with as many messages as you want to select, then perform the operation you want to perform on them (mark unread, move, etc.)

 

 

(There's a similar gesture in the Start screen - tap on a tile, and "tug" it down a bit and release it - now it's selected and check-marked.)
