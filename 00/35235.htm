a_user_here say:

The answer is no if your product key came from the free upgrade.  All the product keys that came from the free upgrade are the same.  This is not how Windows 10 is making sure your system is "valid".  When you upgrade it registers a "fingerprint" of your device with the Microsoft register servers.  When you start your machine it always checks to see if the "fingerprint" has changed, and if it has it invalids the license/activate status.  Since the new "fingerprint" is not going to line up with the Microsoft register servers, automatic activation is not going to work.  At this point you would have to contact Microsoft and convince them that this is in fact the same "device" and get them to provide what is needed to activate.  Unfortunately I don't know what phone number you would use to contact them.  In the past if this your system license became invalid they would provide a phone number (on: Right click Start -> System), but in my testing that doesn't come up in Windows 10.  At least not in systems that have been upgraded with the free upgrade offer.

I don't know if changing the GPU will invalidate the "fingerprint", but it certainly is possible.  Microsoft isn't going to tell us what goes into that "fingerprint" for obvious reasons.  I do know that if you change the network card it will invalidate it.  This is an easy one because every network card has an unique Id that makes if perfect for validating things like this.  And I have switched which network card is active in my machine and it does invalidate it.  BTW switching back makes it valid again.

- You are lucky to be able to run Plants vs Zombies full screen at all on Win10.

I don't use a TV so I assume this is because of the way screen scaling is done by your TV.

With my PC monitors I get black bars with P vs Z on the sides whether  I enable 3D acceleration or not.

Both with Intel HD, and Nvidia graphics.

[ EDIT: When disabling full screen the game runs in a small window that is proportionate to whatever the

games fixed resolution is. ]

- However, as you are already aware, having posted on  this issue previously - TV's are trouble when it comes

to screen scaling, even with Win7, depending on the TV.

- 3D Acceleration just means that your graphics runs from a graphics processor. Intel HD in your case.

When 3D acceleration is disabled the image is rendered via  Windows VGA graphics drivers (Software Rendering ),

and the cpu does the work.

- DirectX runs visual effects, and does not influence screen scaling.

- Win10 has taken over screen scaling. This was done (so they say)  to help make phones, TV's,

notebooks, etc. scale properly.

Graphics drivers can no longer override Windows screen scaling as they could in previous Windows versions,

so even when selecting Full Screen in the graphics control panel (Graphics options/properties with Intel HD)

you will not get full screen if Win10 does not want to use it (most of the time).

[EDIT: This is by design. Of course MS could 'fix' it so the gpu drivers do the scaling again, but they don't want to. ]

So, if you are running a game or software with a resolution that uses a 4:3 / 5:4 aspect ratio eg - 1024x768 / 1280x1024  you will have black bars on the sides when using a PC monitor, as this gives the 'correct' aspect ratio

for the games resolution without stretching the image sideways or cropping the top and bottom to make

if fit without stretching.

Win10 will most commonly only display full screen on a 16:9 aspect ratio monitor when using an appropriate

wide screen resolution such as 1920x1080. To achieve full wide screen with a low res you would need to use a

res similar to 1280x720 (divides evenly by 16:9 = 80 )...

- However there are exceptions.  Win10 may make the image with this resolution small, and surround it with

black bars  (1:1, or 'centered' scaling)- Possibly what is happening on your TV .

Some games and software do seem to be able to override the Windows and graphics driver screen scaling though.

I have a couple of games that will display full screen when using either Intel HD or Nvidia graphics, regardless

of the resolution, because the game is able to override Win10 scaling, but these are uncommon.

- There is another setting that may help (just a wild guess, I don't see it mentioned much these days).

I believe this is mostly applicable to games than use Adobe Flash or Shockwave, which your Visual Novels may do.

Try enabling or disabling Hardware Acceleration in the browser settings. I don't know where to look in

other browsers, but In Firefox this is found in - Tools > Options > Advanced.
