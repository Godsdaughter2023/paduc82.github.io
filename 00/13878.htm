Windows 10 Mobile version 1511 build 10586 / Initial Release: January 9, 2018

Windows 10 Mobile version 1703 build 15063 / Creators Update: June 11, 2019

Lumia Support Lifecycle

Lumia phones are supported for 2 years from the date they are launched. This means they get Feature Updates during those 2 years. Each Feature Update is supported for around 26 months. The last Feature Update that a Lumia phone gets at the end of the 2 years will be supported for around 2 years, so the total support period for a Lumia phone is around 4 years from the date it is launched. 

In short, a Lumia phone will get Feature Updates for 2 years and it will get Cumulative Updates for the last Feature Update for 2 more years.

Please note that the support from here onwards will consist only of bug fixes and security patches. No new features or hardware will be developed further for Windows 10 Mobile. Nobody knows what support will be offered for apps developed by Microsoft.

tryx3 say:

As Tricky said, there isn't an Outlook password as such.  It's possible to set a password on the Outlook pst file but that is only intended as a privacy aid - see Set a password to help protect your Outlook 2007 information - Outlook [this link is the Outlook 2007 equivalent of the link Tricky gave you].

 

The recommended protection method is to have a password on your user account & to lock your PC before stepping away from it [Lock it using the Win key & the l key] so the user account password is required before it can be unlocked.

 

- - - - - - - - - - -

 

You can back this up by having your screensaver lock the PC as well but that would only help you if you forgot to lock it when you stepped away.  Screensavers always seem to activate when you've just decided what to do next and they can therefore be an annoyance.

 

To set up the screensaver to lock the PC

 

1 Set the screensaver - Control panel, Personalise, Screensaver [in the bottom right-hand corner of the window].  You can alter the idle time for the screensaver to start and also set the checkbox so you need to login again after it activates.

You should then check the login part of this setting in two other places:-

2  Set this checkbox -  Control panel, Power options, [on the left-hand side of the window] Require a password on wakeup

3  Set this checkbox - Control panel, Power options, {active power plan} Change plan settings, Change advanced plan settings, [you might need to click on Show settings currently hidden] {active power plan}, Require a password on wakeup

These multiple checks are just ensuring that any alterations have been set & saved consistently - there should be no problem under normal circumstances but some users have reported that setting this in all three locations was necessary for them.
