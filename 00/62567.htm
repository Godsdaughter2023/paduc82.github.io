Login Information

  • Gateway - 192.168.10.1
  • Username – admin
  • Password – admin

 

DMZ

  1. Open web browser.
  2. Login to router by entering http://192.168.10.1 into the address bar.
  3. If prompted for a username and password use admin/admin.
  4. Select Access on the left.
  5. Select DMZ from the sub-menu on the left.
  6. Select Enable
  7. Enter the IP Address of the Xbox.
  8. Select Apply.

Disable DMZ

  1. Open web browser.
  2. Login to router by entering http://192.168.10.1 into the address bar.
  3. If prompted for a username and password use admin/admin.
  4. Select Access on the left.
  5. Select DMZ from the sub-menu on the left.
  6. Select Disabled
  7. Click Apply
