mr andrew say:

You shouldn't try to do this as described.  The concept of the "bottom of a page" does not exist in Word in the context you mean.

However, to achieve what you are after I would suggest that you first create your fax header followed by a single section break.  In the second section, set up your letterhead header and footer, insert a page break (not a section break) and then set up the header and footer for the subsequent pages.  You will need to set the "use different header/footer on first page" option for the section to do this.  Finally, remove the manual page break before saving the template (the header/footer info for subsequent pages will still be remembered).

If you need different margins on your "continuation" pages then see http://sbarnhill.mvps.org/WordFAQs/Letterhead.htm#Complex

The outcome should be that your template begins with the fax header, and then the letterhead section.  Once a page of text has been entered into the letterhead, the subsequent page will use the different header and footer

Andrew - www.stoptyping.co.uk

pamcaswell say:

Place a next page section break at the end of your cover sheet.  Set the  section 2 header to different first page.

See http://wordfaqs.mvps.org/LetterheadRibbon.htm#MultiPage for more information.


suzanne s. barnhill say:
If the same letterhead is wanted on the fax cover sheet and the first page of the letter, then both sections must be set to "Different First Page" (though the first section will have only one page). If the letterhead is created in the First Page Header of the first section and the First Page Header of the second section remains linked, they will be the same.
