john w. vinson say:

A table with no primary key is no table at all - it's an uncontrollable bucket of data. You can add an autonumber, or define a multiple field primary key... but you really, really need one.

That said, you can use the form's BeforeUpdate event to detect duplicates. The code would look like

Private Sub Form_BeforeUpdate(Cancel as Integer)
Dim rs As DAO.Recordset
Dim iAns As Integer
Set rs = Me.RecordsetClone ' assumes that the form is based on the table and not filtered
rs.FindFirst "[Inv #\] = '" & Me![Invoice No] & "' AND [Report No] = '" & Me![Report No] _
   & "' AND [PN] = '" & Me![Part Number] & "' AND [INvQty] = " & Me![Invoice qty]
If Not rs.NoMatch Then ' a record was found
   iAns = MsgBox("This record is a duplicate! Click OK to add it anyway, Cancel to erase", vbOKCancel)
   If iAns = vbCancel Then
      Me.Undo ' erase the entry
      Cancel = True ' Cancel the addition
  End If
End If
End Sub

Hi, lumia 925 user.

We'd suggest you use the Windows Recovery Tool. Please see the following steps below.

  • Start by downloading and installing Windows Device Recovery Tool (for Windows 10, 8.x and 7) from here: Windows Device Recovery Tool.  You may visit this link for more information on how to use this tool : Windows Device Recovery Tool: FAQ.

When the installation has finished, start the application and the main screen will appear:

  • The procedure to restore your phone using Windows Device Recovery Tool is as follows:
  1. Connect the phone to your PC (using the USB cable that came with the phone).
  2. After a few seconds the PC should find the phone and you'll see the Phone Selection screen. Click your phone in this screen to select it.
  3. The latest available firmware version for your device will be shown. Click the 'Install software'/'Reinstall software' button to flash your phone

Windows Device Recovery Tool will then download and install the latest firmware + OS available for your phone.

Note: Your phone will be reset to factory status when using this tool. You may also try Nokia Software Recovery Tool.
