bernie deitrick say:

In Excel, it is a little more work up front:

http://www.contextures.com/xlDataVal02.html

christianjohansen say: Is this the only way to do it in Excel? It requires me to completely rework the setup, which a lot of other variables depend on, so I would LOVE to avoid it.

bernie deitrick say:

You can create a dynamic named range with formulas that extract the correct values - for example, in H1, enter "DVList" In H2, array-enter (enter using Ctrl-Shift-Enter) the formula

=IFERROR(INDEX(TankSort,LARGE(IF(ProduktSort=Index,ROW(ProduktSort)-1),COUNTIF(ProduktSort,Index)-ROW(A1)+1)),"")

and copy down until it returns blanks.

Then create a dynamic named range named DVList, using the formula:

=OFFSET(Sheet1!$H$1,1,0,COUNTIF(Sheet1!$H:$H,">0"),1)

and as your DV list source, use

=DVList

Of course, my default sheet name is Sheet1 - change that to your actual sheet name.

And you may need to change my separator , to ;

Note, too, that if you want strings and not numbers from TankSort, then you need to change

COUNTIF(Sheet1!$H:$H,">0")

to

COUNTIF(Sheet1!$H:$H, ">""")-1

langware say:

With the new outlook.com web interface you can "flag" or "pin" a message. Hover your mouse pointer over a message in your inbox and you will see icons for "flag" and for "pin" (and a few other icons).

Pinned messages in the new interface act like flagged messages did in the old interface. I have no idea why Microsoft was not consistent (between the old and new interfaces).

If you "pin" a message in the new interface, it will appear at the top of your inbox .... but pinning only works if your inbox is sorted by date.

If you flag a message in the new interface, then in order to see the flagged messages, you will have to "filter" by flagged messages.

To select a filter, look at the top right side of your inbox. You should see a clickable label that says:    All

If you click that, a long drop down list will appear. You can click "flagged" to filter your flagged messages (if you filter by "flagged" then ONLY flagged messages will be shown .... the non-flagged messages will not be shown). To go back to the "All" filter, open the drop down list and select "All". You can also use the drop down list to select the way you want your inbox to be sorted, etc.

Again, I have no idea why Microsoft did not make the new outlook.com interface backwards compatible with the old interface.
