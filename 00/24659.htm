dirk goldgar say:

The Split function will split a string value into an array of subvalues or "tokens" based on a delimiter that is passed as an argument. So for example:

 

'----- start of example code ------

Dim astrWords() As String

Dim strInitials As String

Dim I As Integer

 

astrWords = Split("American Drilling Company", " ")

 

For I = 0 To UBound(astrWords)
    If I > 2 Then Exit For   ' maximum of 3 words, you said.

    strInitials = strInitials + Left(astrWords(I))

Next I

 

MsgBox "The initials are: " & strInitials

'------ end of example code ------

 

Note, though, that Split doesn't make allowances for multiple spacers delimiting words.  If you want to consider multiple consecutive spaces as one delimiter, it's more complicated.

Try the following Excel macro. It assumes the F/R list is on 'Sheet1' and that the Word document is open. There is some error-checking in case either Word isn't running or no documents are open. As coded, the F/R routine assumes a case-sensitive, whole-word, match is required.

Sub Demo()
Application.ScreenUpdating = True
Dim wdApp As Object, wdDoc As Object
Dim xlWkSht As Worksheet, lRow As Long, i As Long
' Test whether Word is already running.
On Error Resume Next
Set wdApp = GetObject(, "Word.Application")
'Exit Word if it isn't running
If wdApp Is Nothing Then
  MsgBox "Word isn't running." & vbCr & _
    "Please start Word and open the Word document.", vbExclamation
  Exit Sub
End If
On Error GoTo 0
Set xlWkSht = ThisWorkbook.Worksheets("Sheet1")
lRow = xlWkSht.UsedRange.Cells.SpecialCells(xlCellTypeLastCell).Row
With wdApp
  If .Documents.Count = 0 Then
    MsgBox "No documents open." & vbCr & _
      "Please open the target document, then try again.", vbExclamation
    Exit Sub
  End If
  Set wdDoc = .ActiveDocument
  'Only now can we can process the document!!!

how to read the calendar data in an Outlook file into an Excel - Microsoft Community

frankstasio say:

how to read the calendar data in an Outlook file into an Excel spreadsheet

Except for a file export, which I am doing currently, is there a way to read/query an Outlook Calendar file into Excel?
chip pearson say:

Below is some code that I wrote for another purpose, but modified to work with the calendar entries.

 

Sub ListOutlook()
    Dim OLK As Outlook.Application
    Dim NS As Outlook.Namespace
    Dim R As Range
    Dim myFolder As Outlook.Folder
    Dim ApptItem As Outlook.AppointmentItem
    Dim myFolderItems As Outlook.Items
    Dim x As Long
    Dim EarliestTime As Double
   
    EarliestTime = DateSerial(2012, 1, 1) + TimeSerial(12, 0, 0) '1-Jan-2012, Noon
   
    If OLK Is Nothing Then
        Set OLK = GetObject(, "Outlook.Application")
        If OLK Is Nothing Then
            Set OLK = CreateObject("Outlook.Application")
        End If
        If OLK Is Nothing Then
            MsgBox "Cannot attach to Outlook.Application.", vbOKOnly, "Outlook Roster"
            Exit Sub
        End If
    End If
   
    Set R = Range("A1")
    R.CurrentRegion.ClearContents
    Set NS = OLK.GetNamespace("MAPI")
    Set myFolder = NS.GetDefaultFolder(olFolderCalendar)
    Set myFolderItems = myFolder.Items
    For x = 1 To myFolderItems.Count
        DoEvents
        Set ApptItem = myFolderItems(x)
        With ApptItem
            If .StartInStartTimeZone >= EarliestTime Then
                R(1, 1).Value = .Subject
                R(1, 2).Value = .StartInStartTimeZone
                R(1, 3).Value = .Location
                Set R = R(2, 1)
            End If
        End With
       
    Next x

End Sub

 

Post back if you need help modifying it to work with your calendar items.
