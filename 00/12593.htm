Select the rows that you want to format conditionally.

In the following, I'll assume that the first row in the selection is row 2.

On the Home tab of the ribbon, click Conditional Formatting > Clear Rules > Clear Rules from Selected Cells.

Select 'Use a formula to determine which cells to format'.

Enter the formula

 

=$V2="Y"

 

where 2 is the first row in the selection.

Note that there is a $ before the column letter, but not before the row number.

Click Format...

Specify the desired formatting, then click OK twice.

Change date format to number format.

mike h.. say:

Hi,

 

So in (say) A1 we have 20111121 and that is a correctly formatted date for 21 November 2011.

 

To get that as a number you can use

 

=VALUE(TEXT(A1,"yyyymmdd"))

 

Which returns the number as text but you can still carry out mathmatical operations on that number
