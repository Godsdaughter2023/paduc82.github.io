jay freedman say:

The fault bucket is a property of an error that Microsoft uses when the error report is uploaded to their server. It helps the technicians to know how often a particular type of error occurs across all the millions of computers running their software, but it's of little or no use to anyone else.

The information that might help us to help you includes:

  • What you were doing at the time of the error, if you know it, and what happened next (program crash, failure to open or save a file, etc.).
  • Which part of the Event Viewer shows the error (is it Windows Logs > Application, or some other subset?).
  • These details:
    • Faulting application name and version
    • Faulting module name and version
    • Exception code
  • Does the same error appear in the log more than once? If so, can you describe a sequence of actions that can cause it to happen again, so someone can try to reproduce it?

Conditional Formatting Multiple Excel Cells - Dates - Microsoft Community

rfilo310 say:

Conditional Formatting Multiple Excel Cells - Dates

I have a spreadsheet that tracks when an employee was trained on a certain piece of equipment.  This spreadsheet has about 200 employees (rows) , and 5 different types of equipment to be trained on (columns).   If the employee has been trained on a piece of equipment, their date of certification is put in the cell using the format "mm/dd/yyyy".     Not all employees are trained on all equipment, so some cells are blank. 

These certifications expire in 3 years.  So, I would like each cell (with a date in it) to turn yellow when 2 years have passed, and red when 3 years have passed (and it is expired).  Any blank cell should remain blank with no formatting. 

Each cell is completely independent because even for the same employee they could have different training dates for different pieces of equipment. 

What conditional formatting formula(s) do I use to accomplish this?  I can't seem to get Excel to recognize each cell as it's own date - and most help I've found online is calculating based off of today's date, which is not the case here.   

Thank you!

roblego say:

Red Format - =AND(G7<>"",(NOW()-G7)/365>3)

Yellow Format  - =AND(G7<>"",(NOW()-G7)/365>2,(NOW()-G7)/365<3)

Change G7 to the first cell in your range.  Then use the format painter to copy the conditional format to the rest of the cells you want to apply it to.

feistyone say:

can't copy a continuous section break, it changes to section (next page)

I am making a number of slightly different files.  I want to copy a section That goes 1 column to 2 columns and back to 1 column.  The original is correct but upon copying the last section break (continuous) becomes a section (next page) break. If I delete it, the one above becomes a page break too!

suzanne s. barnhill say:

When you paste into an existing document, the section start type of the last section is determined by information stored in the final paragraph mark. (There's more information on this in http://wordmvp.com/FAQs/Formatting/WorkWithSections.htm.)

Instead of trying to work with the section break itself, change the section start type in the Page Setup dialog. If you have the horizontal ruler displayed, double-click at the top of it to open the Page Setup dialog. (Alternatively, you can access the dialog by clicking the dialog launcher in the Page Setup group on the Page Layout tab of the Ribbon.) Select the Layout tab and change the section start type to Continuous.
