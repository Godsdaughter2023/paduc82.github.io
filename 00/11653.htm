macropod say:

Word's links to Excel files use an absolute path. In this case, regardless of where you move the Word file to, it will always look for the Excel file on your network. If the web users don't have access to that, Word won't be able to find the Excel file when they open the document.

You could try moving the Excel file to the same folder as the document (if it's not already there), then setting the document's hyperlink base to point to its own folder (see https://support.office.com/en-us/article/Set-the-hyperlink-base-address-for-relative-hyperlinks-in-a-document-83a7b7aa-a3a0-4491-8575-1b0ca207f825); otherwise the users will have to edit the link path to point to wherever they download the Excel file to.

What does the crescent moon symbol mean and why is it displayed during the day?
julie fka profjulie say:

If you're seeing the crescent moon symbol during the day, chances are you've got the phone set up to turn on quiet hours whenever you have an appointment entered into the phone's calendar with a status set to "busy."  Go to Settings > quiet hours to verify or change this setting on your phone.

If you don't want the phone to ring during an appointment, then make sure you set the appointment status of the appointment to "busy" when you create the appointment.

If you want the phone to ring during an appointment, then make sure you set the appointment status to "free", "tentative," or "out of office" when you create the appointment.

To change an appointment status on the phone:

  • Open up the Calendar app and tap an appointment (to open it up).
  • Tap the "edit" icon on the bottom of the screen (looks like a pencil)
  • Scroll down the appointment information and find the "Status" box.
  • Tap the "Status" box to select the status you want assigned to the appointment.

You can also set or change the status of the appointment via logging into your outlook.com account on a computer.

Macro to Find and Replace Table Numbers - Microsoft Community

jenny_92 say:

Macro to Find and Replace Table Numbers

I work on a lot of reports with numbered tables and figures.

e.g.

Table 3: xxxxxxx
Figure 2: xxxxxxx

I often have to add/delete a table/figure, meaning the numbers are no longer consecutive. I am trying to write a macro to find all instances of "Table *:" and replace this with "Table _n_:", where _n_ is the nth occurrence of the word "Table *:" in the document, so that I no longer have to re-number all of the tables manually. 

I'm having trouble getting the number _n_ and inserting this in the 'Replace' statement though.

I would be really grateful for any help on this!

Thanks

doug robbins - mvp office apps & services (word) say:

Have you considered using Captions?  They consist of a label (such as Table) and a SEQ field that automatically increments the number of each such caption and updates the numbers if another one is inserted or deleted within the series. 

They are inserted via the Captions section of the References tab of the ribbon.

For your existing documents, running a macro containing the following code

Dim rngfound As Range
Selection.HomeKey wdStory
Selection.Find.ClearFormatting
With Selection.Find
    Do While .Execute(FindText:="Table [0-9]{1,}:", MatchWildcards:=True, Forward:=True, Wrap:=wdFindStop) = True
        Set rngfound = Selection.Range
        rngfound.Text = "Table :"
        rngfound.Collapse wdCollapseEnd
        rngfound.Start = rngfound.End - 1
        rngfound.End = rngfound.End - 1
        rngfound.Collapse wdCollapseStart
        ActiveDocument.Fields.Add rngfound, wdFieldEmpty, "SEQ Table"
    Loop
End With

Will replace the number in "Table #:" with a SEQ Table field
