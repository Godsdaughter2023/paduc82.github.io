Gosh this was too easy to figure out.. sorry for original post.

To center multiple pages in MS publisher, open .htm document in notepad (right click, open with..)

replace the </head> not <head> make sure you got right one, I had to scroll down a bit to find it.

</head><table border="0" cellpadding="0" cellspacing="2" width="880"align="center"><tr><td>

Change the "880" to whatever your page width is, i have mine at 880 and looks great on web.

When you publish your site it creates and index.htm and an index_files folder, replace the </head> with the above code on the index.htm file, now open the index_files folder and open each page of your website in notepad and do same thing in each page. Its easy to figure out which document to modify as as its extension will be a .htm

When you have the formula working correctly in the worksheet, turn on macro recording and then select the cell with the formula. Using the mouse place the cursor at the end of the formula in the formula editing bar and then press Enter. Stop the recorder and you will have the correct syntax for your formula.

In the recorded formula it will start like the following:

ActiveCell.FormulaR1C1 = "= and the rest of the formula.

Replace ActiveCell with Range("F1") like the following:

Range("F1").FormulaR1C1 = "= and the rest of your formula.

 

Note that the recorded code will probably be in a new module and you might have to select that module by double clicking it in the Project Explorer on the left of the VBA editor. You can copy the code and paste it into your main code area.

ontheflydi say:

Is there any way to create a cross-reference to another Word document?  I would like to insert hyperlinks in one "master" document (a glossary), with the page numbers referenced, for bookmarked sections in several independent documents.  Is there any way to create a cross-reference to another Word document?  I would like to insert hyperlinks in one "master" document (a glossary), with the page numbers referenced, for bookmarked sections in several independent documents. 

I've also tried to do this with hyperlinks, but I don't find any way to display the page number of the reference, like a cross-reference.
daniel g. grau say:
Better than that, you can link directly to the text in another document, so what when you click on the link, the referenced document will open and take you directly to the proper place.

To do that, you need to define a bookmark in the target document, and then link to it through a hyperlink in the original document.

The method is rather long, so I'll just refer you to Using Word’s Hyperlinks With bookmarks. Those instructions are for Windows, but the method is exactly the same on the Mac.

Then there's this Mac video: Hyperlinking in Word
, where the pertinent part starts at about 5 minutes. Unfortunately, it doesn't explain how to link to a bookmark in the target document (which is done through the "Anchor" option in the Hyperlink dialog box).

Note that in order for the hyperlink to keep on working, you need to modify the file path. Once you have your hyperlink ready, press Option F9 to show the field codes. You'll see something like this:
  • {HYPERLINK "file://localhost/Users/YOUR_NAME/Documents/Test%20Files/Linked%20File.docx" \l "BookmarkNameHere"}

That's called an "absolute" file path. You need to change that into a "relative" file path, like this:

  • {HYPERLINK "file://Linked%20File.docx" \l "BookmarkNameHere"}

That way, as long as you move the complete folder with all the files, the links will keep on working.

Finally, press Option F9 once again to hide the field codes.
