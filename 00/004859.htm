palcouk say:

You dont need to do anything, as yet with any pst file. That data file is on the local PC, and even if your current provider closes, that pst data file, and all its contents remain on your PC and its contents can be viewed in MS outlook.

Do NOT under any circumstances move a pst data file that is connected to MS Outlook to OneDrive or any other external location, as that will lead to corruption of the pst.

In MS Outlook, File>Account Settings>Email tab if you select that defunct mail address>Remove That will remove the mail account, doing so will have no impact on the pst data file, which will remain accessible/view-able in MS Outlook

PS A pst of 80gb will cause MS Outlook Performance degradation and as such you might want to use the archive options to reduce its size and create an archive pst, which will be accessible in MS Outlook. You can create, if necessary multiple archives, I used to create yearly archives.

Trying to import or copy an 80gb pst to outlook.com will take a very long time.

Hello,

the "tabs" in your screenshots look like graphics inserted into the sheet. They are not really tabs at all, but images that look like tabs with text boxes that link to another sheet. 

You can create this in Excel, too. If you want to go down the "tab" look and feel, you will need to find appropriate graphic files. Insert them into the sheet, then place a rectangular shape with the desired "tab" text on top. Select the shape and insert a link that links to another sheet in the same workbook.

If you want to make do without the "tab" look and feel, you can enter the words directly into Excel cells and insert a hyperlink to another sheet. Or use shapes with text, because these are independent of cell widths.
