Hi the first thing to try is the Windows Recovery Environment and see if any of the options there can get your PC up and running . . .

Start PC, just as Windows attempts to load (spinning dots), press and hold Power Button for 5 - 10 seconds to perform a Hard Shut Down

Do this twice

On the third start Windows will boot into the Recovery Environment and from there you can access System Repair, Safe Mode, Command Prompt . . . etc.

Please try each option one at a time, then see of Windows will boot . . .

Go to Troubleshoot - Advanced Option - Startup Repair
See if Windows startup can repair itself

Go to Troubleshoot - Advanced Option - System Restore
Check to see if you have a restore point you can go back to

If you do not have a Restore Point, go to Troubleshoot - Advanced Option - Startup Settings and click Restart
Upon restart, press 4 to enter Safe Mode
Once in Safe Mode, backup your data if you do not already have a backup
Open Device Manager and check to make sure all your Drivers are up to date
Reboot to see if Windows will start Normally



If you cannot get into the Windows Recovery Environment, you will need Windows 10 Installation Media to get you in there:

If you do not have Installation Media, click this link:
https://www.microsoft.com/en-gb/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool



Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS)

Do not install Windows 10, instead click 'Repair this Computer', the repair will fail!

Go to Troubleshoot - Advanced Options - Command Prompt
Run these commands one at a time and hit Enter:

bootrec /FixMbr
bootrec /FixBoot
bootrec /ScanOs
bootrec /RebuildBcd

Then reboot your System to see if Windows will boot . . .

a. user say:

Hello,

I understand that you are facing issues to use your 256 GB flash drive. This may be a problem with drivers.

I would suggest you to follow the following methods and see if it helps.

Method 1:

Remove and reinstall all USB controllers.

  1. Open Device Manager by pressing Windows key + X and select device manager from the list. If you're prompted for an administrator password or confirmation, type the password or provide confirmation.
  2. In the list of hardware categories, find and expand Universal Serial Bus controllers.
  3. Right-click every device under the Universal Serial Bus controllers node, and then click Uninstall to remove them one at a time.
  4. Restart the computer, and let the USB controllers get reinstalled.
  5. Plug in the removable USB device, and then test to make sure that the issue is resolved.

Method 2

  1. Try to remove the ghost drivers as there could be an interference of the other drivers previously installed. On the next command prompt line, type devmgmt.msc and press Enter. This will launch the Windows Device Manager Console.
  2. In the Device Manager Console, from the View menu, select Show Hidden Devices.

As you expand the different drivers and devices in the device manager, you will see not only the items that Windows currently detects as installed on your PC; but you will also see drivers, devices, and services which have been loaded in the past but were not uninstalled or are not currently started. You can find your offending device, right-click, and choose uninstall to remove it from the system completely.

Note that ghost devices, drivers, and services are “grayed” out, but that does not necessarily mean that you should delete all of them. Only remove items you know you do not need. Be careful that you do not change too many devices.

Once you have removed the ghost drivers completely from the system, try to re plug the USB and see if it helps.

// now create an OleDB command object with the query we built above and the connection we just opened.

ken blake - microsoft mvp say:

On Sat, 22 Dec 2012 21:49:10 +0000, Adrian Hirst 62 wrote:

Just upgraded from XP to 8 Pro with no undue issues except the download was 32 bit and doesn't use the full capabilities of my PC. Searched high and low for an upgrade from 32 to 64 but with no success. Any ideas how to sort this?

Two points:

1. An upgrade from Windows XP to Windows 8 isn't possible. Although
Microsoft calls it an upgrade, it's really a clean installation.

2. It is never possible to change the "bitness" of Windows. You can
not do an upgrade from a 32-bit version to a 64-bit version, or
vice-versa. Only a clean installation is possible.
