john korchok say:

Well, this is one of the more bizarre fixes I've discovered. I'm running Yosemite 10.10.5. I would appreciate others attempting to replicate this:

  1. If you have a brand-new installation of Word, activate it, then close it. (Closing Word for the first time creates Normal.dotm).
  2. Open Word.
  3. Check Word>Preferences>General. My experience suggests that if the option Enable background pagination in Print Layout view is greyed out, you will always get the Document Gallery when you use File>Open. This is not a cause and effect connection, that option is an flag that Normal.dotm is in compatibility mode, even though it doesn't display that in the title bar.
  4. Choose File>Open and navigate to the Normal.dotm template in /Users/YourUserName/Library/Group Containers/UBF8T348G9.Office/User Content/Templates. Open it.
  5. Choose File>Save As and click on Save. Apparently, this updates the template to be compatible with Word 2016. 
  6. Close the template.
  7. Check  Word>Preferences>General. The newly compatible template will cause Enable background pagination in Print Layout view to be clickable.
  8. Choose File>Open. The Document Gallery does not appear, but the OS X File>Open dialog does. At least it does here.

Mu apologies to anyone I misled with my earlier conclusion. To answer questions, I'm constantly messing with my Office configuration, so sometimes one thing appears to be an answer when it's really something completely wacky. I've reinstalled Office 3 times and run the above procedure and it worked every time.

bob jones aka: cybertaz say:
For a paper form, Phillip's suggestion to use an Underline Leader Tab is one of the more commonly used methods. [Although there's no need for it to be a right aligned Tab stop.] You actually need 2 custom Tab stops; 1 regular Tab stop at the point where the line is to begin, the 2nd being the Leader Tab at the point where the line is to end (probably coincident with the right margin). The result would be something like:

Question One: {TAB}_________________________________________________{Leader Tab}
Question Two: {TAB}_________________________________________________{Leader Tab}

However, if there are more than about half a dozen lines to be done or if multiple lines need to be allowed per question, it's much more expedient & effective to use Tables. Remove the Borders for all cells except where the underlining is needed.

The Paragraph Borders method is not one often used, but see this page:

http://word.mvps.org/faqs/tblsfldsfms/LinesInForms.htm
