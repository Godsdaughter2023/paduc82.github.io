In Windows Live Mail, each message is an individual text file. Without knowing why you might want to change the subject line of a message, it's difficult to suggest the best method of doing this. Windows 8 comes with its own built in Antivirus utility in addition to other security technologies. Of course, if you are not comfortable using Windows Defender, you can use a third party utility such as AVG, ESET and many others.

 

- Secure Boot/Trusted Boot - operating system cannot be tampered with

- Early Anti-Malware Scanning

- Antivirus/Anti-Spyware - Windows Defender

- Windows Smart Screen Filter

- Address Space Layout Randomization (moves around operating system code on start up fooling malware).

- Kernel Patch Guard - kernel cannot be tampered with

- User Account Control - can't accidentally install unwanted programs

- HeapStack protection - prevents buffer overflows

- Device Driver signing

- Protected Mode in IE


Add these with a little common sense and you should have an even more secure system.

 

Here is a detail of my experience using Windows Defender:

One of the core features now built into Windows 8 is early Anti-malware scanning. Windows 8 now includes Windows Defender, built up on the foundations of its successful Antivirus and AntiSpyware utility Microsoft Security Essentials. Due to the changing landscape of security, having Antivirus built into the Operating System is a logical move that guarantees a safer environment for users who either refuse to pay for commercial Antivirus utilities or do not do a sufficient job at keeping their Antivirus updated.
