If feel pretty sure there are better ways of doing this but this will work:

=IF(IFERROR(MATCH(A2,DATACL_HASWU[ITEMNUMBER],0),TEXT("0000-","00000")&IFERROR(MATCH(VALUE(A2),DATACL_HASWU[ITEMNUMBER],0),TEXT("0000","0000")))="0000-0000","no match at all",IF(LEFT(IFERROR(MATCH(A2,DATACL_HASWU[ITEMNUMBER],0),TEXT("0000-","00000")&IFERROR(MATCH(VALUE(A2),DATACL_HASWU[ITEMNUMBER],0),TEXT("0000","0000"))),5)="0000-",VLOOKUP(VALUE(A2),'1st_WU'!$A$1:$A$20000,1,FALSE),VLOOKUP(A2,'1st_WU'!$A$1:$A$20000,1,FALSE)))

If you break it down, this bit:

=IFERROR(MATCH(A2,DATACL_HASWU[ITEMNUMBER],0),TEXT("0000-","00000")&IFERROR(MATCH(VALUE(A2),DATACL_HASWU[ITEMNUMBER],0),TEXT("0000","0000")))

Two match functions look down the list and tries to find either a MATCH or a VALUE(MATCH). If there are no matches at all, this string is returned "0000-0000". If there are no matches on the left match and it finds a right match the string "0000-x" is returned (where x is the position it finds the match). Left matches are returned as x.

So we have three scenarios, "0000-0000", "0000-x" and x. These three tests are performed by the nested IF.

Not my cleanest work, and you could use a helper column to make the function easier to read but see what you think.
