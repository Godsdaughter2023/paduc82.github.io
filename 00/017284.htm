If the state = Ohio I want t insert the text from a document called second.docx stored in the My Document folder

My merge field looks like this




Unfortunately when  I run the merge, no text is inserted  from either document. I am thinking my path is not correctly identified . Can anyone point out what I am doing wrong?? I appreciate the help

Thanks

graham mayor say:
You can't paste fields into this forum, so your current syntax is not shown; however the syntax you need is

{ IF { MERGEFIELD State } = "California" { INCLUDETEXT  "C:\\Temp\\First.docx" } }{ IF { MERGEFIELD State } = "Ohio" { INCLUDETEXT  "C:\\Temp\\Second.docx" } }

If there are only the two states in the data source you could simplify that to

{ IF { MERGEFIELD State } = "Cal*" "{ INCLUDETEXT  "C:\\Temp\\First.docx" }" "{ INCLUDETEXT  "C:\\Temp\\Second.docx" }" }

Note the double slashes in the path and that the state name is case sensitive. Substitute the field name from your merge data if necessary and of course the true paths to the documents. All the bracket pairs {} are inserted with CTRL+F9

Merge to a new document or the printer (or see http://www.gmayor.com/individual_merge_letters.htm ) to view the results.

If you click Yes/Disable add-ons when you are prompted with the “Speed up startup and browsing by disabling add-ons” notification, a window will appear.
