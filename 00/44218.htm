You are mistaken! A disk is a complete drive, which may or may not be divided into partitions. The 100mb  System Reserved is a partition with no label (letter) located on the same disk / drive as the System (Windows) partition.  The System Reserved partition can be larger than 100mb. a. user say:

Hi,

Thank you for posting your query in Microsoft Community.

Sorry for the inconvenience caused. I will assist you with this.

When you upgrade to Windows 10, the previous Windows files and folders are automatically stored in C:\Windows.old folder. This folder is kept by Windows only for one month. After one month Windows 10 will automatically delete the folder. So you can restore previous Windows version only within one month after Windows 10 upgrade.

Let's try to search the files and folders using Cortana and Windows Explorer search.

Step 1: Using Cortana

  1. Press Windows key on the keyboard, type Cortana and select the top most search result. Follow the wizard to activate Cortana.
  2. Type windows.old and select the search result which shows folder & drive location.
  3. Follow the file or folder path of the program.

Step 2: Using Windows explorer

  1. Press Windows key + E keys on the keyboard
  2. Double click on This PC (select Local Disk C:).
  3. Locate windows.old folder.

Instead of clicking on windows.old, just click on the users file in the current version. It contains all the files and documents of previous Windows.
