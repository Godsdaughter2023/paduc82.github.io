brian tillman [outlook mvp 2007-2019] say:

When you upgrade from Windows XP to Windows 7, you will need to reinstall all of your applications.  They don't transfer automatically.  After installing them again, you can recover the data from the Windows.old folder.  Your Outlook data is there.  Look for *.pst.  Copy the PST to a folder in your Windows profile and use the Mail applet in Control Panel to add that PST to your mail profile and make it the default data file to restore your Outlook data to Outlook and make it available to the phone again.

ActiveSync doesn't run on Windows 7.  Instead you'll need to download and install Windows Mobile Device Center.  Here's the web page: Download Windows Mobile Device Center 6.1.  Be sure to read the caveats on that page.
