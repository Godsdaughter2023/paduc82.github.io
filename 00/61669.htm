It is normal for the drive letters to be different in Repair Mode as compared to Normal Mode. You often find something like this:
* Drive C: - System Recovery
* Drive D: - Windows System drive
* Drive E: - Data partition (if you have one)
* Drive X: - Repair Mode System partition (this is a RAM disk)

You can run commands such as chkdsk against drive D:.

If you need to recover any non-backed up files then you have two options:

  • Work through the Command Prompt while in Repair Mode.
