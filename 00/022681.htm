a. user say:

Yes, changing the password should help.

It's typically only mobile devices that credentials are cached on so as long as you don't have any tablets or cell phones missing you should be ok.

No there isn't. Just be sure to logout or not click the "remember me" when logging in on another device.

Another possibility is to use the browser's "private browsing" mode when on someone else's computer. That way it doesn't save your login info once the browser is closed.
