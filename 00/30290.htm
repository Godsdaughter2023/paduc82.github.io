squeakywheel say:

Well, I answered my own question eventually, seem that the Default Programs doesn't change everything or is not telling the full story when you update it.

After digging around in the registry a bit more I found the two keys below, which were still set to IE. Changed them to my preferred browser and the programs that weren't opening things in the right browser now are. Haven't noticed any errors yet.

HKEY_LOCAL_MACHINE\SOFTWARE\Classes\http\shell\open\command

HKEY_LOCAL_MACHINE\SOFTWARE\Classes\https\shell\open\command

If a system such as this was brought to my attention, I would likely do a full system image backup, install CLEANLY Windows 10 with the current device drivers supplied by the manufacturer (no other software at first) and fully patch it... At that point the full system image could be restored and the proper actions taken.
