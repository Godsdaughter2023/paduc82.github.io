turning a table lengthwise - Microsoft Community

dari smith say:

turning a table lengthwise

I'd like my table to be in landscape, turned lengthwise, but I can't figure out how to do it.
daniel g. grau say:
You cannot rotate a table by itself. Some alternatives:

  • Build it with rotated cells (in the Ribbon, Table Layout > Alignment > Direction).This is kind of inconvenient.
  • Build is straight up, and once it is final convert it to a picture and rotate it (the problem is, you loose editability):
    • Select the table
    • Copy (or Cut)
    • Edit > Paste Special > Picture
    • Select the picture and rotate it
  • Insert a section break, set the page in landscape and place the table there. This is only good for printing (not screen display) and if you have headers/footers, you have to recreate them in rotated text boxes
  • The easiest and meanest: 
    • Cut the table
    • Insert a text box
    • Paste the table into the text box
    • Rotate the text box
