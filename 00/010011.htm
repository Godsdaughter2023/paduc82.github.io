cindy meister say:

Hi Caeley

To have Word include these automatically, you need to have Headings in your document that are numberd automatically.

For example, at the beginning of each section, the title for the section would be formatted with the style "Heading 1". And you'd need to apply numbering to the Heading style with A, B, C, etc.

In the Insert Caption dialog box there's a button "Numbering". Here you have to activate the checkbox "Include Chapter number", select the heading style (Heading 1), then the punctuation you want to use to separate the Heading number from the caption number.

I'm sorry I can't tell you exactly where to find all these things, but you didn't tell us which version of Word you're working with. (And that's odd, because the interface for the web site does ask that.) The commands have changed from Word 2003 going forward, so that makes it difficult for us to give you exact instructions.

Quick solution:

Maybe you have the Do not check spelling or grammar option switched on. Try this to check:

 

1. Open the document and use <Ctrl>+<A> to select all text.


2. Click on the Review tab


3. Click on Language


4. Set proofing language.


5. Select the correct language in Language dialog box


6. Make sure that neither of the check boxes at the bottom is selected


7. Click on Set as default

 

8. When prompted to save changes to the global template, say Yes

 

 

More serious solution:

 

Check out this knowledge base article from Microsoft "Spell Checker does not recognize misspelled words in Word 2007":http://support.microsoft.com/kb/970826
