OK, so it could well be a issue related to corrupt files. So the next thing to do is to try and repair the files SFC couldn't.

So open another elevated command prompt and then copy/paste the following command and hit Enter:

  • DISM.exe/Online /Cleanup-image /Restorehealth

DISM.exe downloads original system files from Microsoft's servers to replace those which SFC couldn't fix so it's important to maintain your Internet connection while you run it.

When it runs, the cursor will drop one line and it may appear like nothing is happening for quite a while, but eventually you'll see the figure 20% appear followed by 100% when it finishes. It'll  also tell you whether the corruption was repaired or not.

Assuming it was, try and install Windows Update stuff again.

If the same problem rears its ugly head again, we'll have to go through the CBS.log file to see if we can track down the culprit.

However, you'll get an Access Denied if you try to open it directly, so what you have to do is to copy/paste the following command to copy the contents to an ordinary txt file which opens in Notepad by default. It also has to be run from an elevated command prompt:

  • findstr /c:"[SR]" %windir%\logs\cbs\cbs.log >%userprofile%\Desktop\sfcdetails.txt

Running that will place a file on your desktop called sfcdetails.txt. It's going to be be very long and most of it will only pertain to files which SFC examined and found nothing untoward with. But if you use the PageDown key to jump with, you'll eventually come to the data which contains the errors.

These will be easily recognisable because they'll stretch right across the page and you'll need to use the horizontal scroll bar to move across the page if you want to read everything. Fortunately, you don't need to do that.

lemp say:

Do read the articles suggested by Jsssssssss.

Windows Defender and Microsoft Security Essentials are two programs provided by Microsoft.  

Microsoft has made things confusing by recycling names so that "Windows Defender" means different things depending on what version of Windows you have.  This opening paragraph from the Wikipedia entry about "Microsoft Defender" illustrates the confusion:

Windows Defender is a software product that attempts to detect and remove malware. It was first released as a free antispyware program download for Windows XP, shipped with Windows Vista and Windows 7 and made into a full antivirus program replacing Microsoft Security Essentials as part of Windows 8 and later versions.

If you have Windows 7, you have Windows Defender already built-in, but it is only an antispyware program.  Microsoft Security Essentials detects both spyware and other forms of malware (e.g., viruses), and if you install Microsoft Security Essentials on a Windows 7 system, Windows Defender will be disabled.

Microsoft Security Essentials is not available for versions of Windows newer than Windows 7.

As far as comparing Microsoft's free offerings to those from third-parties (free or not), I do not consider myself enough of an expert on this topic to give you an opinion.  Personally, I feel that protecting one's system from malware is important enough to warrant paying for it.  For a thorough review of this topic, including suggestions for which programs you might want to obtain, read through the following:  Answers to common security questions - Best Practices -- part way down that page are comments on "Choosing an Anti-Virus Program."

Hi Shawanda Lattimore,

Method 1: Run Windows Media player settings troubleshooter and check if it helps.

Open the Windows Media Player Settings troubleshooter:

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter


Method 2: There are two ways to change rip settings while in the Player Library. You can insert a CD into your CD drive and click Rip settings, or you can click the Rip Music tab in the Options dialog box. The Rip Music tab in the Options dialog box offers additional options that aren't available on the Rip settings menu.

To rip music from a CD, do the following:

a. For best results, make sure that you are connected to the Internet before you begin.

When your computer is connected to the Internet, the Player attempts to retrieve media information about the tracks being ripped from an online database. The Player then adds the information to the files during ripping.

Note: You can rip a CD when your computer isn't connected to the Internet. However, the Player won't be able to identify the name of the CD, the artist who created it, or the names of the songs until you go online.

b. Insert an audio CD into the CD drive.

c. (Optional) If you want to select a different format (for example, MP3 instead of WMA) or bit rate (for example, 192 Kbps instead of 128 Kbps) for the files created during ripping, in the Player Library, click theRip settings menu, and then click Format or Audio Quality.

d. (Optional) if you don't want to rip every song, in the Player Library,clear the check boxes next to the songs you don't want to rip.

e. Click the Rip CD button in Now Playing mode or Rip CD in the Player Library to begin ripping.

Method 3: Enable error correction. Windows Media Player can play some CDs that contain errors if the error correction option is turned on.

a) Open Windows Media Player.
b) Go to Tools and click on Options and then click onDevices.
c) Select CD/DVD device and choose Advanced.
d) Place a checkmark on “Use error correction”.
e) Once you’ve done that, try ripping the CD.

For more information you may refer to the links provided below.

Disabling the ripple effect for tablets in Windows 8 - Microsoft Community

deweystilwell say:

Disabling the ripple effect for tablets in Windows 8

Found this on the Wacom site.

To disable the ripple effect in windows 8, run regedit, look for these two keys.

HKEY_CURRENT_USER\Control Panel\Cursors\ContactVisualization

HKEY_CURRENT_USER\Control Panel\Cursors\GestureVisualization

They should be set to 1 and 1f respectively by default.  Set them both to zero.

After installing a Wacom Tablet (CTE Graphire 4) I noticed clicking with the pen created a ripple "bubble" effect.

It's super annoying.

Searching for a solution yielded a bunch of stuff for Win7, not so much Win8.  After 2 days of discovering similar complaints, the wrong registry edits, and instructions that only worked for win7 (gpedit.msc, stuff from viziblr, suggestions to install new drivers for a tablet that hasn't seen an update in decades, allusions to a windows ink option whose existence I still can't confirm)  I finally found this solution.  So compelled was I, I've decided to share this information.  

Hopefully, even if this information proves repetitive, the additional entry will lessen the frustration for some other artist.

New Dell Latitude owners may notice an unknown device in their device manager. Here’s how to find out what it is and the solution to what mine was.

 

In Device Manager, right-click the device and click Properties. Under the details tab, choose Hardware ID from the drop box. You’ll see something like ACPI\SMO8800 (that’s what the unknown device was on mine). Google that hardware ID and you’ll find that it’s an accelerometer, a device that detects when your laptop may be falling and instantly stops the hard disk so it isn’t damaged on impact.
