beth melton say:

If you need Microsoft Equation 3.0 then it should already be installed. See this works for you:

  1. On the Insert tab, click the arrow next to Object and then click Object.
  2. On the Create New tab, Microsoft Equation 3.0 should be in the list.

Once you find the symbol you need you can create an AutoCorrect entry to make it easier to insert with these instructions:

  1. Insert the symbol
  2. Click back in your Word document to close the Equation Editor
  3. Select the symbol
  4. Press Alt, T, A (Don't hold down the keys, press each one individually. This will open the AutoCorrect dialog box)
  5. Type a name for your entry that doesn't form a real word. For example you can use something like: /div or just div
  6. Click OK
  7. Insert your symbol in a document by typing: /div or div and then press the spacebar to replace your typed characters with the symbol
  8. Double-click the inserted symbol to edit and modify it
