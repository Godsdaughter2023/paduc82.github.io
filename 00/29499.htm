andre da costa say:

Press Windows key + R

Type: msconfig

Hit Enter on your keyboard

Click the the Boot tab

There you will see the current, default OS and where exactly it is booting from.

Copying over the files from HDD to SDD will not work, in fact, this could damage system files and make your system in operable.

saltgrass say:

Well, another option.  Open Disk Management and check the labels on the partition.  If one shows as "System" it contains the boot files.  If one shows "boot" that is where the OS is.

Not knowing any particulars about your system, such as the size of the SSD it is hard to answer the other parts of your question.  But in many cases systems are set up to run from the HDD but use a small SSD or mSATA drive to work as a prefetch device which speeds up the operation from a HDD.  That is handled through an Intel application.
