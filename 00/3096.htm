stefan blom say:

The Navigation pane displays paragraphs whose "Outline level" (Paragraph dialog box) is set to something other than "Body Text." The easiest approach is to apply the built-in headings to text; you can do so via the Styles group on the Home tab of the ribbon or via the Apply Styles pane (Ctrl+Shift+S).

Note, however, that the Navigation pane does not recognize headings inside table cells, so you won't be able to use it with a document that is set up entirely with tables.
