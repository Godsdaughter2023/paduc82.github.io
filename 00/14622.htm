I was able to set up my Gmail account as the default for sending invites when I used IMAP email settings for my Outlook.com account instead of Exchange. To do this, you need to remove your Outlook.com/Hotmail.com account first from your Outlook application then add it back manually by following the steps below: 


  1. Go to File > Account Information > + Add Account. 
  2. Click Manual setup or additional server types > Next > POP or IMAP > Next. 
  3. Fill in the details for your Outlook.com account settings. Please use the following information:

    Account type: IMAP
    Incoming mail server: imap-mail.outlook.com
    Outgoing mail server (SMTP): smtp-mail.outlook.com 
          
  4. Select More Settings found on the lower right corner. 
  5. Click Outgoing Server tab and select My outgoing server (SMTP) requires authentication.
  6. Go to Advanced tab. Use the details below:

    Incoming server (IMAP)
    Port: 993
    Encryption: SSL/TLS 
      
    Outgoing server (SMTP)
    Port: 587
    Encryption: STARTTLS 
          
  7. Click OK > Next then follow the next prompt. 
  8. Restart your Outlook application and try to create an invite. 




For your reference regarding this process of adding an email account using advanced setup, you may check the articles below: 

  • Add an email account using advanced setup
  • POP and IMAP email settings for Outlook
