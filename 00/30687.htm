If you call the file SomethingOrOther.reg instead of SomethingOrOther.key then it will do open with RegEdit & write the value in the Registry.

If you really want to do it the hard way, when you see the Open with list [either doing it through Properties or just right-clicking on the file & selecting Open with], click on More apps and then scroll to be bottom of the list and click on Look for another app on this PC then browse to C:\Windows\System32\regedt32.exe and select that.

a. user say:

You can't uninstall Windows Vista conventionally - it doesn't go to Programs with an uninstall option.  See this article: http://support.microsoft.com/kb/925809. Here's another article on the process: http://www.tech-faq.com/how-to-uninstall-vista.shtml. Read them, especially the second one, and it will walk you through the process. You will have to do a clean install of whatever OS you want on that PC (you may have to format the drive first) and that will do away with Vista Ultimate.  If you don't want another OS on that PC, then just format the drive (from booting to the installation disk) and Vista ultimate will be gone. You may want to use the Easy Transfer process before you do this if there's anything on yuur old PC you want to save (in which case install it on the new PC and then run the Easy Transfer process (if you have the equipment to do so) before you remove it from the old PC. A few hours with it installed on more than one PC is certainly acceptable for you to transfer files.  Remember, you'll need to reactivate your copy of Vista ultimate when you reinstall it on the new PC.

Hello GANEWPLAYER,

Try the following steps:

 

Automatic Defrag Disable and Enable in Vista

 

Vista now includes an automatic defragment system that is enable by default. This tutorial describes how to control this new service.

 

Vista Automatic Defrag:

1. Click Start

2. Click Control Panel

3. Selct the

4. Click System and Maintenance

5. Under the Administrative Tools section at the bottom, click Defragment your hard drive

6. You may need to grant permission to open the disk fragmenter

7. Click or unclick

8. Click OK

 

How to defrag your drives in Windows Vista

  1. Open a Command Prompt as Administrator .
    Go to Start – All Programs – Accessories, right click on the Command Prompt and choose “Run as administrator”. Obviously you need admin privileges to do this.
  2. Type the following see how much your hard drive is fragmented (in this example, your C drive):
    defrag c: -a
  3. Vista will tell you a “Percent file fragmentation” and, at the bottom, if you need to defragment the drive or not.
  4. To fully defragment your C drive type the following:
    defrag c: -w
  5. Give it time to run (best to leave the computer alone) and then you’re done!
