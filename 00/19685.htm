georginatrias say:

I want to know how to have a different watermark (not in header or footer whole page) on my continuation page for a letterhead? (first page already has a whole page watermark)

I want to know how to have a different watermark (not in header or footer, whole page) on my continuation page for a letterhead? (first page of letterhead already has a whole page watermark)
suzanne s. barnhill say:

A watermark is just a graphic anchored to the page header, with wrapping set to Behind Text, and the color set to "Washout," which in Word 2013 is 85% Brightness and 15% Contrast.

First you must enable "Different First Page" in the Options group of the contextual Header & Footer Tools | Design tab (which is displayed when you access the header). I think you've already done this because you say the first page has a watermark. If you enable "Different First Page" before adding a watermark from the Design tab, you can add the watermark to either the first page or the remaining pages by placing the insertion point on the first or second page, but if you then go to the other page and try to add a watermark, it will be inserted on that page, but it will remove the watermark from the other page. Therefore, you cannot use the Watermark feature for both watermarks.

Temporarily add a second page to the document so that you can access the primary Header that starts on the second page. With the insertion point in the header, use Insert | Illustrations | Pictures to insert a picture. Change the wrapping to Behind Text and then drag the picture to the center of the page and size it as desired. Then right-click on it and choose Format Picture to get the controls that will allow you to select the brightness and contrast.

In my experience, the result is not quite the same as it would be if you selected the same picture and inserted it as a watermark. I'm not sure what the difference is, but I do note that the old (Word 2010 and earlier) Format Picture dialog comes up when you choose it from the context menu for a Word-inserted watermark, whereas for a picture inserted via the Insert tab, you get a Format Picture task pane that has different controls.

walester01 say:

I was finally able to fix the repeat header problem. The suggestion from Doug Robbins led me to the answer.

I was checking in the Table Properties to see if Text Wrapping = None was enabled (it was). I then noticed that I could look at Cell Properties as well. With only the header row highlighted, I checked the Cell Properties and then clicked on the Options button. I noticed that the option for Wrap Text was checked. I disabled it and saved the change.

I then went back to the Row Properties and rechecked the option for "Repeat as header row at the top of each page". I saved the change and that was it. The header appeared at the top of each page!
