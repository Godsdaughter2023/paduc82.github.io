luke moraga say:

Assuming the Fleematic stuff is in cell A2, and you have record data in col B, you can use this line:

Sub DeleteRows()
Range("B2").End(xlDown).Offset(1, 0).EntireRow.Resize(2).Delete
End Sub

rick rothstein, mvp - excel say:

You can do this with one line of code (either add it to your current code or make it into a stand-alone macro)...

Cells.Find("FleetMatics",,,xlpart).Offset(-1).Resize(2).EntireRow.Clear

A (small) benefit of the approach I have used over the other posted solutions, especially if you use it in a stand-alone macro, is if you accidentally run it on a worksheet that has already been processed with it, the code will not remove the bottom two rows of live data.

Note: If you choose to go with one of the other solutions offered, you do not have to Delete the rows... since they are the last two lines of code on your worksheet, you can simply Clear them..
