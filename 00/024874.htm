To clear or remove recent search history in Outlook, follow the steps below:

  1. Press Windows Key + R keys to open the Run dialog box.
  2. In the Run dialog box, type regedit in the Open box, and click the OK button.
  3. In the pop up User Account Control dialog box, click the Yes button.
  4. The Registry Editor window opens. Successively expand the key with the below path:

HKEY_CURRENT_USER > Software > Microsoft > Office > 16.0 > Outlook > Profiles > profile name > 0a0d020000000000c000000000000046.

  • By default, the profile name is called Outlook if you haven’t changed your Outlook mail profile name. If you have changed it, please replace the profile name with the name you have used to change.

5. Right click the 101f0445 key on the right window, and select Delete from the context menu. See screenshot:

6. In the Confirm Value Delete dialog box, click the Yes button.
