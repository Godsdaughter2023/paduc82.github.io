macropod say:

What you would need is a field to perform an IF test on the Discount field, coded along the lines of:

{IF{MERGEFIELD Discount}<> 0 "Less Discount: {MERGEFIELD Discount}"}

or:

{IF«Discount»<> 0 "Less Discount: «Discount»"}

Note: The field brace pairs (i.e. '{ }') for the above example are all created in the document itself, via Ctrl-F9 (Cmd-F9 on a Mac); you can't simply type them or copy & paste them from this message. Nor is it practical to add them via the standard Word dialogues. Likewise, you can't type or copy & paste the chevrons (i.e. '« »') - they're part of the actual mergefields, which you can insert from the 'Insert Merge Field' dropdown. The spaces represented in the field construction are all required.
