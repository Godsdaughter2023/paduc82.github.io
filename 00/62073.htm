  1. Press Windows key and X at the same time.
  2. Choose Command Prompt (Admin).
  3. In the Command Prompt window, type in the following command:  DISM /Online /Cleanup-Image /RestoreHealth.
  4. Press Enter.
  5. Restart your computer.

Method 3: Run the System File Checker.

  1. Open Command Prompt (Admin).
  2. In the Command Prompt window, type in the following command: sfc /scannow.
  3. Leave the Command Prompt window on until the command completes.
  4. Close the command prompt, and then run Windows Update again to see if the problem persists.

Under Windows 7 and early versions of Windows 10, the maximum length of a folder+file name path is around 260 characters. Under the current version of Windows 10 its 32,000 characters (I believe). However, File Explorer under Windows 10 is still unable to deal with such long names.

If you search the net for the phrase "Long Path Name" then you will find numerous references to this problem. There are some third-party utilities that will let you get around the limitation.
