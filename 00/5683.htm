If the command buttons are located in a sheet, use the OLEObjects collection and the name of the command button as argument. Access the button itself through the Object property.

  Dim myButton As MSForms.CommandButton
  Set myButton = Me.OLEObjects("CommandButton1").Object
  myButton.BackColor = RGB(255, 0, 0)

If the command buttons are located in a Userform, use the Controls collection same way and you get the control object directly.

  Dim myButton As MSForms.CommandButton
  Set myButton = Me.Controls("CommandButton1")
  myButton.BackColor = RGB(255, 0, 0)
