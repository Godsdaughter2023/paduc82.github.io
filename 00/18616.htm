calculate total time in excel - Microsoft Community

a. user say:

calculate total time in excel

Hi I would like to calculate the time a person spent in a task, data has 4 entries, including Start, Stop, Pause and Resume - 2 columns in between, some don't have all 4. What formula should I use to calculate all 4 in one cell and to ignore should the cells in between are empty, Thank you.

8:13:00 AM

11:40:00 AM
11:41:00 AM 11:48:00 AM 12:58:00 PM 2:03:00 PM

neha singh msft say:

Hi Anna,

Assuming Start, Pause,Resume and Stop in column A, B, C and D respectively. To calculate total time, you can use the following formula in cell E3 and E4.

=IF(B2=0,(D2-A2),(B2-A2)+(D2-C2))

=IF(B3=0,(D3-A3),(B3-A3)+(D3-C3)

After applying the formula, please make sure you change the cell format. For this, right click E3 and E4, select Format>Time.

Thanks,

Neha

Issue:  Internet Explorer does not launch properly in Windows 10.  The application launches but mouse cursor never changes from "busy" and CPU utilization hits the roof.  I have to terminate the process and am unable to run IE on Windows 10.

Resolution: 

  1. Open File Explorer and go to C:\Program Files\Internet Explorer
  2. First try running iediagcmd.exe in the Internet Explorer directory.  You may need to right click and "Run As Administrator" for it to work properly.  This is a built-in diagnostic tool that resolves common problems automatically.
  3. Right click on C:\Program Files\Internet Explorer\iexplore.exe and select "Run As Administrator".

For me, Step ③, possibly in conjunction with Step ②, resolved the problem.
