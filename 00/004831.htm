Two Questions- Count Max Consecutive Values and more... - Microsoft Community

bitterclinger say:

Two Questions- Count Max Consecutive Values and more...

I have two columns, A and B.  Column A contains either "W" or "L" for win or lose. Column B contains a two-digit number which is the predicted chance to win.

First, I want to find the longest streaks of Wins and Losses. So, if column A has nine consecutive "L" values before the next "W" is entered, the longest streak would be 9.

Second, I would like to find the lowest number in Column B where Column A has a "W". I would also like to find the highest number in Column B with an "L" in the corresponding Column A.

Any help would be greatly appreciated.

mike h.. say:

Hi,

All of these are ARRAY formula, see below for how to enter them.

Longest streak of W

=MAX(FREQUENCY(IF(A1:A20="W",ROW(A1:A20)),IF(A1:A20<>"W",ROW(A1:A20))))

Longest streak of L

=MAX(FREQUENCY(IF(A1:A20="L",ROW(A1:A20)),IF(A1:A20<>"L",ROW(A1:A20))))

Min for a W in col A

=MIN(IF(A1:A20="W",B1:B20))

Max for an L in col A

=MAX(IF(A1:A20="L",B1:B20))

This is an array formula which must be entered by pressing CTRL+Shift+Enter
and not just Enter. If you do it correctly then Excel will put curly brackets
round the formula {}. You can't type these yourself. If you edit the formula
you must enter it again with CTRL+Shift+Enter.
