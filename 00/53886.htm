Not nice, but temporarily fix the problem.

Download the free and opensource AutoHotKey software.

Save this to a file with ahk extension. (You can change the speed with Loop value changing)

; Fix Windows 10 horizontal scrolling speed

WheelRight::
; Scroll to the left
MouseGetPos,,,id, fcontrol,1
Loop 8 ; <-- Increase for faster scrolling
SendMessage, 0x114, 0, 0, %fcontrol%, ahk_id %id% ; 0x114 is WM_HSCROLL and the 0 after it is SB_LINERIGHT.
return
WheelLeft::
;Scroll to the right
MouseGetPos,,,id, fcontrol,1
Loop 8 ; <-- Increase for faster scrolling
SendMessage, 0x114, 1, 0, %fcontrol%, ahk_id %id% ;  0x114 is WM_HSCROLL and the 1 after it is SB_LINELEFT.
return

Start the ahk file with AutoHotKey and now you can change the scrolling speed!
