I have a client who is unable to pin an email template to their Outlook 2013 Jump list in Windows 8.1.  I tried this on other computers and was able to do it, so I know it works.  The template files are located in their Templates directory (under their Appdata folder) but, I also tried it from their desktop.  When I right click and drag the template over the Outlook 2013 icon in their taskbar I get a red circle with a line through it.

scottb35e0 say:

Ok, so I figured out the problem.  Thank you Karl for your assistance.  I will document the fix below in case someone runs across this in the future.

The issue was that a user could not pin a template to their Outlook "jump" list.  Using this article as a base, I was able to expand and test to find the correct file and delete it.  

https://angrytechnician.wordpress.com/2010/05/24/how-to-fix-a-brokenempty-jump-list-in-windows-7/ 

Basically, Microsoft keeps a file for each application in this location:  %AppData%\Roaming\Microsoft\Windows\Recent\AutomaticDestinations.  The file names are cryptic and of no help so it requires a little digging to figure out which one is which.  What I did was perform the action of pinning a template to the jump list on a machine that was working and then looked at the "Date Modified" to find the appropriate file.  The file names appear to be the same across different comptuers.  You can also open the file in Notepad and get some indication of what's in it.  As stated in the article above, don't edit the file directly.  

For Outlook 2013 it's:  6d2bac8f1edf6668.automaticDestinations-ms 

Once I found the appropriate file, I moved it to the desktop (as a temporary backup location) and then tried to pin the template to the Outlook on the Taskbar again.  It worked properly on one computer but, failed on a second.  To fix it on the second computer I had to unpin/re-pin the application to the taskbar.  Not exactly sure why that fixed it but, it did.  

After that, I was able to pin files to Outlook jump list without issue.  Hope this helps someone!
