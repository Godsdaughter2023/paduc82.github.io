suzanne s. barnhill say:
You can insert a manual page break using Ctrl+Enter. But, as Stefan says, pages that are centered vertically using the Page Setup setting usually look bottom-heavy. To look right, the text must be just a little above center. On a vertically center page, you can accomplish this by adding some Spacing After to the last paragraph. Alternatively, especially if you just have one or a few pages that need to be centered (as in the front matter of a book), it preferable to insert page breaks as needed, then add Spacing Before to the first paragraph on the page until it looks right.
