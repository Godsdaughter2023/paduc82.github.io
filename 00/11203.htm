Try this UDF:

 

Function BlankColsCount(rng As Range)
    Dim lnCount As Long, i As Long
    For i = 1 To rng.Columns.Count
        If WorksheetFunction.CountA(rng.Columns(i)) = 0 Then
            lnCount = lnCount + 1
        End If
    Next i
    BlankColsCount = lnCount
End Function

 

Right-click on the Sheet's tab at the bottom of Excel and View Code.
Then do Insert > Module
Then paste the code on the right.
Close the VBE.

Enter the following in a cell:

 

=BlankColsCount(B3:F8)

Does MSE protect when using public wi-fi networks?

Does MSE provide adequate protection when using public wi-fi networks for broadband?

 eg using my laptop during a 3 week stay in a relatively inexpensive hotel that provides wifi in guest rooms

stephen boots say:

MSE protects against malware -- viruses, Trojans, spyware, malicious activity. Security over a public network is not something that MSE protects against specifically. The basic security on a public network is the Windows Firewall, set to public, which restricts inbound activity and turns off file sharing and other features that are used on a private network. When using a public WiFi connection, data is easily sniffed and captured. For further questions about security concerns of a public WiFi network connection, post your question to the Windows Security or Networking forums for more advice.

brian tillman [outlook mvp 2007-2019] say:
One way I can think of to empty the Inbox is to allow a mail client to do it.  Configure the Outlook.com account to obey a mail client's commands to delete messages after downloading at Options gear>Options>Connect devices and apps with POP>Enable>Do what the device or app says-if it's set to delete messages, delete them>Save.  After this, configure a POP client to connect to the Outlook.com mailbox.  It will start to download the messages and delete them as it does.  You can start the process before going to bed and let it continue by itself.  Then, once done, you can toss the downloaded messages and uninstall the mail client.
