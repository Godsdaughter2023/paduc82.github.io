okay it's downloading something called HotFix may I ask what is that? 


Hotfix


Definition: A hotfix is a single, cumulative package that includes one or more files that are used to address a problem in a product and are cumulative at the binary and file level.

A hotfix addresses a specific customer situation and may not be distributed outside the customer's organization.


Additional information: Hotfixes are distributed by Microsoft Customer Service and Support.

 Customers may not redistribute hotfixes without written, legal consent from Microsoft.

 

•General Distribution Release (GDR)

Definition: A GDR fixes an issue that has broad customer impact or that has security implications. A GDR is determined and issued only by Microsoft. Microsoft tries to release as few GDRs as possible.

Additional information: A GDR cannot be requested by a customer. Microsoft determines whether a specific hotfix is classified and delivered as a GDR through an internal process. A GDR is released through the Microsoft Download Center. A GDR may also be released through Microsoft Update or Windows Update.

A hotfix package does not replace a service pack. A hotfix package is optional. A hotfix package can be installed or uninstalled at any time. Additionally, hotfix packages are cumulative. Therefore, the latest on-demand hotfix package or cumulative update hotfix package includes all previously released hotfixes.

a. user say:

Hi Ben,

Thank you for posting your query in Microsoft Community Forum.

Basically, we need to change the language and regional settings after installing the new language pack on the system to make every setting attuned with the applications and basic functions. So, you will need to refer to the following steps and check if you have all the settings in your preferred locale.

  1. Press Windows key + X and select Control Panel.
  2. Select Clock, Language and Region (in the Categories view).
  3. Click on Language.
  4. Click Advanced settings in the navigation pane.
  5. Under "Override for Windows display language" select English (preferred locale).
  6. Under "Override for default input method" select English (preferred locale).
  7. Click Save.
  8. Reopen Clock, Language and Region as in Step 2.
  9. Click on Region.
  10. In the Format tab, check if the Format is selected as Match Windows display language (recommended).
  11. Go the Administrative tab and select Change system locale...
  12. Select English (preferred locale) and click OK.
  13. Again, in the Administrative tab, click Copy settings...
  14. Check mark the box beside Welcome screen and system accounts.
  15. Check mark the box beside New user accounts.
  16. Click OK on the tabs and close the Control Panel.
  17. Restart your computer.

Check if the language has changed to your preferred locale.


Hope this information is helpful. If the issue still persists, please post back for further assistance, we'll be glad to assist you.
