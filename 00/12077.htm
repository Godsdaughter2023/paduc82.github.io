I'm not sure it is a good idea to do this - you'll notice that it has strange side effects.

Copy the following code into the worksheet module:

 

Private Sub Worksheet_Change(ByVal Target As Range)
    Dim r As Long
    Dim c As Long
    Dim cel As Range

    If Not Intersect(Range("G:G"), Target) Is Nothing Then
        Set cel = Intersect(Range("G:G"), Target).Cells(1)

        c = cel.Column
        If r > 1 Then

                    .BorderAround LineStyle:=xlDashDot, Weight:=xlMedium

                        .Pattern = xlSolid
                        .PatternColorIndex = xlAutomatic

                        .TintAndShade = -0.25
                    End With
                End With

        End If
    End If

karl timmermans [outlook mvp 2012-2018] say:
First, would suggest opening the recycle bin to see if you can recover the deleted PST file.

If the PST file has been permanently deleted, the simplest process would be to create a new Outlook profile using the Control Panel <Mail> app and let Outlook create a new PST for the account (assuming you only have one email account configured).. Just make sure to set the newly created profile as the default for the next time Outlook is opened

How to create profile and set up an e-mail account in Outlook
http://support.microsoft.com/kb/829918

If you have multiple email accounts configured, then use the Control Panel Mail app to remove the affected email account and then re-add it back. In this scenario, if the PST file was set as the "default data file" for the profile, you would need to first set another data store as the default (done on the Data Files tab when in Account Settings). before removing the affected email account
