An IMAP account is a copy of mail on the server, it will sync/be the same on any other device that that IMAP a/c is set up on.

A pop mail account, which uses a pst data file, downloads all mail to the local PC, unless you set the option to keep a copy on the server. A pop mail account will not therefore sync with any other device (Other than show received mail if you set the option to keep a copy)
