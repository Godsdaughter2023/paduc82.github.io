Here are some possible causes of connection problems:

The Remote Desktop port might be blocked by a firewall.

If you're using Windows Firewall, follow these steps:

1.       Open Windows Firewall by clicking the Start button, clicking Control Panel, clicking Security, and then clicking Windows Firewall.

2.       Click Allow a program through Windows Firewall. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

3.       Select the Remote Desktop check box.

Remote connections might not be enabled on the remote computer. To fix this:

4.       On the remote computer, click the Start button, right-click Computer, and then click Properties.

5.       In the left pane, click Remote Settings. If you are prompted for an administrator password or confirmation, type the password or provide confirmation.

6.       Under Remote Desktop, click either Allow connections from computers running any version of Remote Desktop or Allow connections only from computers running Remote Desktop with Network Level Authentication.

7.       Click OK.

Is this laptop or tower?  Only explanation I can think of is that you have something plugged into the card reader such as a memory card or USB device (if it offers an extra USB port through the card reader) and Windows setup is unable (or programmed not to try) to negotiate unmounting the device/volume, migrating the drivers, remounting the device/volume, and all that.  If there is nothing inserted into the reader, then try to disable the card reader depending on if you have a tower or laptop. 

I know with prior versions of Windows, when installing the OS on a PC that has a (USB) card reader plugged, the card reader slots can get assigned drive letters before the fixed hard disks due to the way that Windows enumerates storage devices/interfaces so I just made it practice to disconnect card readers prior to installing Windows, then reconnect later.  With laptops it may not be easy to do since the card reader connector could be soldered or just really difficult to access.  But with towers or desktops, it's usually easy enough to open the side panel or remove the chassis cover, follow  the cable connecting the card reader device (usually front panel) to the main circuit board connector and just unplug it (turn off the computer prior).

Otherwise you could disable that device by opening Device Manager, expanding the device categories to locate the Realtek card reader, right click on it and select "Disable".  Then try to install Windows 10.  If successful, after Windows 10 is installed, open Device Manager, locate the Realtek card reader, right-click and select "uninstall".  If you receive a prompt to 'also delete/uninstall software for this device' select Yes or check the box.  Restart the PC and let Windows redetect the card reader and install the best driver for it.

bobby mikkelson say:

Hello Christi,

 

Attempting to pin an installer to the taskbar may fail and leave a broken pinned item that doesn't show up on the taskbar. After this happens you may notice that some items pinned to the taskbar don't rearrange in the way they expect. When moving an item from left to right, the item will end up one or more positions to the left of where you intended to move it. You can work around this by moving items from right to left to achieve the desired order. You can also fix the issue by removing any broken pinned items with the steps under using the following steps:

 

1.       Browse to the following folder in your user folder under C:\Users

AppData\Microsoft\Internet Explorer\Quick Launch\User Pinned\Taskbar. Note that "User Pinned" is a hidden folder.

Note: You may need to go to AppData\Roaming\Microsoft\Internet Explorer\Quick Launch\User Pinned\Taskbar

2.       Delete any shortcuts in this directory that aren't represented on the taskbar.

3.       Click Start

4.       In the Search programs and files box, type regedit

5.       In the Programs list, click regedit.exe

 If you are prompted for an administrator password or confirmation, type the password or click Continue.

6.       Browse to and select the following registry key

HKEY_CURRENT_USER \Software\Microsoft\Windows\CurrentVersion\Explorer\

7.       Right-click Taskband!FavoritesRemovedChanges and click Modify

8.       Change the Value Data to 0

9.       Browse to and select the following registry key

HKLM\Software\Microsoft\Windows\CurrentVersion\Explorer\StartPage\

10.   Right-click FavoritesRemovedChanges and click Modify

11.   Change the Value Data make sure the value is greater than 0

12.   Log off and log back in, pinned items should be rearrange normally again.

Bobby

Bridging Guide - Microsoft Community

xb1r athena say:

Bridging Guide

Login Information

  • Gateway - 192.168.1.1 or 192.168.1.254
  • Username - user | admin | root | user | root | root
  • Password - pass | admin | 12345 | 12345 | central | ZOOM-DSL

Bridging Guide

*Bridge mode should only be used if you have a router to plug into the device.*

*Please have PPPoE information available, if you do not have this, please contact your ISP.*

  1. Plug the computer into a LAN port via an ethernet cable.
  2. Open a web browser.
  3. Enter http://192.168.1.1 or http://192.168.1.254 into the address bar. (Or click the link) When prompted enter the admin for the username and admin for the password if the gateway asks.
  4. Select Quick Setup from the menu on the left. Click NEXT
  5. Enter the PPPoE values as provided by the ISP. Click NEXT
  6. Select the radio button for “Bridging”. Click NEXT
  7. On the following Screen check the box “Enable Bridging Services”. Click NEXT
  8. Enter the LAN IP address & Subnet Mask values for your router (*192.168.1.1 is set by default*). Click NEXT
  9. Enable the Wireless Function on the following screen.
  10. Enter the Wireless Network Name (SSID). Click NEXT
  11. Ensure the information on the following screen matches the information provided to you by your internet service provider.

  12. If necessary enter the PPPoE information into the subsequently connected device. (i.e. Router/Game console/Computer)
