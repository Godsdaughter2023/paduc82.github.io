In Windows 10 the access to Control Panel is still there. You can go one or two routes.

Settings > Update and security select View update history. To see Control panel from here select Uninstall updates and it will take you to the old Control Panel view. Or you can search control panel and select the top result, select Program and Features and then View installed updates.

If you have just received version 1809 though you won't see version 1803 updates.
