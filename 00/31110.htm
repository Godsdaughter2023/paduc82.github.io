crunchydoodle say:

A Surface 2 is not what most people would call a Windows PC. The Surface 2 is more like a Chromebook than a Windows PC.

The hardware for the Surface 2 is more like an Android tablet, or even a Chromebook. It's an ARM based system that runs a special version of Windows 8.1 called Windows RT that generally looks and acts as a Windows PC, which it is not. Windows RT is a somewhat closed and highly secure system that is probably as virus-free as Chrome. For one thing, you can only download and install programs from the Microsoft App Store. You cannot install any of the millions of PC applications out there in the world.

Even with all that, a Surface 2 way more versatile than a Chromebook. The Surface 2 has real Office applications, for one.

Don't confuse the Surface 2 with the Surface Pro (any of them). The Surface Pro is a real Windows PC, warts and all. It can run most any of the millions of Windows applications out in the world. It's a real Windows Pro 8.1 PC.

Maybe you should stick with that Chromebook, if that works for you.

Bye.
