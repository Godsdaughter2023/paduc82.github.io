Hi,
 
1. Did you try to play the file using a different player?
2. What is the file format?
3. Are you trying to burn an audio CD?
4. Is the issue pertinent to this particular song?

Step 1:
 
I'd recommend you to play the file using any other media player and check if it plays fine. If the file does not play, then the file might be corrupt.
 
Step 2:
 
If the file is protected with media usage rights, those rights may allow for playback of the song on the computer but prohibit burning to a CD. Media usage rights are stored on your computer separately from your audio or video files.
 
To check if the files are protected, do the following:
 
a. In the Player, click the Library tab.
b. In the Details pane, right-click a protected file, and then click Properties.
c. Click the Media Usage Rights tab.
 
The usage rights are indicated in the Media usage rights box. If no information appears in the Media usage rights box, it might be due to one of the following reasons:
 
i. Your computer is missing the media usage rights for the file.
ii. The media usage rights for the file have expired.
iii. The file is not protected.
iv. The file has not been downloaded to your computer. For example, the file might be subscription content that is streamed to your computer from an online store.
 
You can download media usage rights from the internet.
See Windows Media Player Digital Rights management
