azam k say:

 

Hi,

Your CPU, RAM or videocard may not be able to handle restoring all the details in enough time which ends up with resulting a timeout aka "program not responding".

Games especially on full screen use quite a lot of CPU, RAM and videocard power. And in games like WoW and COD there are quite a few details happening in real-time which means the computer has to redraw/rerender quite a few details, and if your computer is not capable enough, it will not be able to respond in time and "crash".

Just to test, or if you really must multitask, you can try switching to windowed COD and see if it can maximize/restore back up. It should be able to as in windowed mode games tend to use a lot less power. Maybe not in raids or crowded cities though (more crowd, more details, more polygons for computer to draw).

 

 In few games, when you are playing in full screen there might be possible that alt tab and Windows tab option may not work. You need to minimize the screen by pressing Windows key or Windows + D key and then probably you can check.

PlayReady is a Digital Rights Management (DRM) from Microsoft for portable devices.
