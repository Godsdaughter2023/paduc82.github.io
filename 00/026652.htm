Some other options...

1) When you need to type a Subscripted character use Shift+Command+[minus sign] first, type the character, then use that keystroke again to toggle Subscript Off. Use after-the-fact, then use one of the keystrokes.

2) Select the formatted O2, add it to the Scrapbook (in the Add> Add Selection. Give it a name then drag a copy into the text whenever you need one. Items added to the Scrapbook are also available to each of the other Office apps. It means having to use the mouse, but may be worth it.
