jean-yves tfelt say:

Hi

 

Do I understand correctly that you have the same range name accross multiple sheets ?

Then this is a 3D named range and is not compatible with sql.

In the example :

strQuery = "SELECT * FROM MyRange" - MyRange should represent a range of cells on A specific sheet. By using MyRange, you tell exactly whech Table name to use.

 

Each named range should be unique for your purpose

You have to define exactly from whch table you want to extract info.

 

Another aspect is : why use spaces in object names (your sheet name)? This is avoided by all programmers. use an underscore instead to assure compatibility.

 

You can also build a dynamic query, that is changing the parameters using a Select case statement.

 

HTH
