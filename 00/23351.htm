Word Online does not have a feature to create a new envelope from a template.

You can create an envelope document in desktop Word and save it to OneDrive, where you can open it in Word Online and edit it as needed. You must put some mailing address into the original document, because Word Online can't show you the frame that holds the address if the frame is empty.

In my test, the envelope didn't print correctly -- the addresses were rotated 180 degrees from where they should appear, so the return address wasn't on the envelope at all, and only part of the mailing address printed upside-down at the bottom of the envelope. Since Word Online has nothing related to mailing at all, it also lacks the dialog to tell it which way the envelope should feed. If you're really determined to use Word Online for envelopes, you'll have to experiment with how it communicates with your printer.

Insiders must also have a qualifying upgrade OS installed for a free Win10 license--or--maintain an unbroken activation in the Insider program.  If you don't have a qualifying OS, and your Win10 license activates because you are in the Insiders program, then if you lose your activation for any reason (drive failure, reinstall, etc.) you *do not have a permanent Win10 license* sitting on a Microsoft activation server and will be unable to activate.  You only get a permanent license if and when you upgrade from a qualifying OS installation (Win7/8.1) to a like license in Win10--ie 32-bit Win7/8.1 gets you 32-bit Win10; an OEM Win7/8 license gets you gets you an OEM Win10 license--a retail Win7/8 install allows you to upgrade to a *retail* license of Win10, etc. 

Additionally, I'm pretty sure I read that a vbox install of Windows 7/8 does *not* qualify as an upgradable OS, but I am not 100% sure on that point.  (Lots of people used vboxes during the pre-release Win10 Insider Preview period as a means of installing and testing Win10, but as of build 10130 or so everything changed as per activation particulars.)

To receive the Microsoft license sitting on Microsoft's activation servers for RTM or later you must:

1) Install an upgradable version of Win7/8.1, activate it, then install the free Windows 10 upgrade on top of your activated Win7/8.1 install, which automatically builds an internal license key from your upgrade OS activation, and that in turn sits on Microsoft's activation servers and activates you during the Win10 upgrade process.  (Win10 activation uses the upgrade OS activation to build the internal activation key for Win10, and only Microsoft has this Win10 activation key on their servers.  If you do not activate your key on the OS you are upgrading, prior to upgrading to Win10, you will not receive your free Win10 license until this requirement is met.)

2) After you have completed (1) above and Windows 10 is activated, then, and only then, can you clean install a new copy of Win10 which will activate automatically from Microsoft's activation servers (into a virgin drive partition or else into a virgin vbox--if that's allowed.)

Again., you must have a qualifying OS installed to upgrade to Win10 whether you are in the Insiders program or you're not--makes no difference.  Microsoft never intended to create a situation in which all one had to do to get Win10 free was to join the Insider's Preview...;)  The benefits people get by remaining in the Insider program at present are earlier looks at builds & features long before they make it to retail product distribution through Win Update.  For some people that is no perk at all...for others it's the Holy Grail...:D

After a reboot, you should have roughly 40 MB of free ram.  If not, which of your programs run automatically at startup?  If you manually quit everything you can find (with just the Home and Menu icons remaining on the "Open Apps" list), are you still left with ~10 MB free?

 

If so, then I think something is still running.  Install a copy of NetQin's Mobile Guard from http://www.netqin.com/en/guard/symbian/﻿ .  This is a free collection of phone utilities, one of which is "Boot Manager", which lets you disable the auto-start portion of programs.  (BTW, Mobile Guard also includes a number of "helper" functions, so don't be alarmed if you start seeing pop-up messages during downloads, when installing or removing programs, etc.  Most of these can be disabled, or just remove Mobile Guard when you're finished.)

 

The only other thing I can think of that might eat into your free ram is if you had 10,000 (?) contacts...or some enormous number.  Oh, also see if the free ram changes dramatically with the memory card removed.  Worst case scenario is that you would need to back up your data and hard reset the phone, but let's not be hasty.  I hope that helps!
