Each main mail merge document (Letter, Label, etc.) is associated with a data source file of some sort. That association remains in effect unless the user chooses to specify a different source for that same document.

genevieve cri say:

Hi,

Excel file sizes have a way of increasing from creating and editing them. Your workbook usually contains text, but it may also contain charts, colorful graphics and formulas that take up a lot of bytes. Sending these huge files over email clogs your server, and storing them on your hard disk gobbles up available space. Here are the options on how to reduce the size of Excel files:

Option 1:

  • Check for formatting of the unused cells. By deleting all unused rows to the bottom of the document on each worksheet.

Option 2:

  • Save the file to Excel in binary (.xlsb). Doing these structures the workbook into a compressed folder that has many binary files. The amount of compression that you're going to get depends on the contents of the workbook. Also, you may find that opening, closing and saving the file takes longer than something saved just in regular (.xls) format. Here are the steps below:
    1. After you've saved the file as an .xlsb, highlight and delete all of the cells outside the data range (typically to the right and below).
    2. Highlight the same cells and delete all formatting (colors, lines, etc.). Delete formatting for each for each sheet within the workbook.
    3. Re-save the workbook as an .xlsx file. Check to see how much these actions reduced the file size.

Option 3:

  • Save your workbooks as the same file and type, but under an alternate file name.

Let us know if the information above helps you. Otherwise, we’ll be here if you need further assistance.

mccbleue say:

But he asked how to save them on the SIM card. You simply need to change the location where they are saved.

 

In Contacts, select Options>Settings. Firstly, check in 'Contacts to display' that SIM memory is ticked, then change 'Default saving memory' to 'SIM' and all contacts you create from then will be saved to SIM.

 

If you want to move existing contacts in the phone memory to the SIM, select Options>Marl>Mark all then Options>Copy>SIM (but be aware that the restrictions of SIM contacts mean that there can only be one name and one number per contact, and there will also be a restriction on the number of contacts that can be stored on SIM so you could find some data not transferred).
