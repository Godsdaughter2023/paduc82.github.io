Hello

It might be you're signed in to a different account under the same email address. You may check the Skype account/s under your email address using the reset password link. To do this:

1. Go to the Reset password page.

2. Enter the email address you registered when you created your Skype account, and click "Continue". From there, you'll be able to see the Skype account under your email address. If you have more than one Skype Name, reset the password for each account. Then, click on Reset password.

3. Follow the recovery process.

4. Once done resetting the password, open and sign in to the Skype application via Skype name or email address. Locate the Skype account which has your Skype contacts and Skype Credits.

Let us know if you have any further questions or clarifications.

julie fka profjulie say: Since you only changed the contacts on the computer WMDC should not detect a conflict.  The Conflict Resolution setting should only be used if you changed the same contact on both the computer and the windows mobile device.

But, if you are really worried about this, you could delete your current WMDC partnership and set it up again.  When the new partnership is set up, you may be prompted to replace all the data on the windows mobile device with the data from your computer.

How to add and show image in access 2010

scottgem say:
A standard way of attaching an image to a record is to simply store the path to the image file in a text field. Then use an unbound image control to display the image. 

To do so, you assign the path to the image as the Picture property of the image control. Thsi is usually done int he On Current event of the form:

Me.imgPicture = Me.txtImagePath
