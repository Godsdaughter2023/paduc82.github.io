How to make Month([FIELD]) return Name values instead - Microsoft Community

mohitgupta96 say:

How to make Month([FIELD]) return Name values instead of number of the month.

I am using the above function in my query. It does what I want, but it returns number values. I need to create a report and I need the names of the month, not their numbers.

Here is the SQL view:

SELECT Month([LessonRoadTestDate]) AS [Month], Year([LessonRoadTestDate]) AS [Year], Sum(Orders.Price) AS [Monthly Income]
FROM Orders
GROUP BY Month([LessonRoadTestDate]), Year([LessonRoadTestDate])
ORDER BY Month([LessonRoadTestDate]), Year([LessonRoadTestDate]);

Thanks a lot,

Mohit Gupta
ken sheridan say: You can also do it in the report without amending the computed Month column in your query, by using the expression:

=MonthName([Month])

as the ControlSource property of a text box.

As this query is the RecordSource for a report, remove the ORDER BY clause; it merely adds an unnecessary additional overhead which will be detrimental to performance.  Sort the report by means of its internal sorting and grouping mechanism, first by Year and then by Month.
