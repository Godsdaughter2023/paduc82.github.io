formula execution time vba - Microsoft Community

sst 1.0 say:

formula execution time vba

Hello,

is there a way to get vba to time the execution of a sub?

 

hansv mvp say:

You can do something like this:

 

Sub Test()
    Const n = 100
    Dim sngTime As Single
    Dim i As Long
    sngTime = Timer
    For i = 1 To n
        Call YourSub
    Next i
    sngTime = Timer - sngTime
    Debug n & " runs in " & Format(sngTime, "0.00") & " seconds."
End Sub

 

Change YourSub to the name of the procedure you want to time, and change the value of n to suit your needs. For short procedures you may need to use n = 10000 to get a good idea, for very complex procedures n = 1 may be sufficient.
