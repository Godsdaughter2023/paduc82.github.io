cmdrkeene say:

Thanks for your description. It sounds like the agent probably removed the account, but let's make sure and then get that extra folder taken care of.  To see what accounts are currently setup, find and open command prompt (use the search/Cortana to find it) and then type the command "net user" and press enter. It will list all current accounts.  For example, here's my Surface tablet's accounts. You'll notice it lists accounts that aren't shown in the control panel.

If the "old" account is there, you can remove it with another command. for example, to remove my "shawn" account, I could type net user shawn /delete
PS. Please don't try to remove the built-in accounts, it can damage your PC greatly.

As for the folder...

Now if it's not there, and only the folder remains, here's how to solve that one. There's many ways to do this but it requires preciseness.

  1. Open the C:\Users folder.
  2. Right-click the user folder you want to delete, go to Properties.
  3. On the security tab, click Advanced.
  4. In the window that opens, next to "Owner" at the top, click Change
  5. In the 'Select user or group' box, leave it empty and click Advanced
  6. In the search box that opens, leave it empty and click Find Now
  7. In the list at the bottom, find Administrators (make sure it has the S at the end). Highlight it, then click OK.
  8. Click OK again to confirm the selection.
  9. Click OK again to close the Advanced Security box
  10. Click OK again to close the folder properties box
  11. At this point, you should be back at step 1 with only the Users folder open
  12. Right-click the folder to delete, and choose Delete. It should prompt you with a continue button.
