Sub FindFromBottom()
    For Row = 10 To 1 Step -1
        If Range("B" & Row).Value >= Range("$A$1").Value Then
           Range("B" & Row).Select
            Exit Sub
        End If
    Next
    Range("$A$1").Select
End Sub

There are a couple of things you can do.  The first is to right-click the Suggested Contacts folder, choose Properties, select the Outlook Address Book tab, and uncheck the box labeled "Show this folder as an e-mail Address Book".  This will prevent the Suggested Contacts folder from being searched by the Address Book service.  The next thing you can do is click File>Options>Contacts and uncheck the box labeled "Automatically create Outlook contacts for recipients that do not belong to an Outlook Address Book".  This will prevent Outlook from adding to Suggested Contacts the non-contact addresses to which you send mail.  Then you should be able to open Suggested COntacts, select everything in it by pressing Ctrl+A, then deleting them all by pressing or clicking Delete.  It should no longer populate.

PowerPivot / Data Modelling
Power Pivot is a data modeling technology that lets you create data models, establish relationships, and create calculations. With Power Pivot you can work with large data sets, build extensive relationships, and create complex (or simple) calculations, all in a high-performance environment, and all within the familiar experience of Excel. In a nutshell, it allows you to use multiple data sources. Then you could import, merge and perform analysis on the resulting data.  Excel can “natively” only store 1 million rows of data. Draw a text box and click within > Insert > Text Group > Insert File and browse to where you store your Word file on the local hard drive and click. Publisher will convert and automatically add as many text boxes and pages as necessary to accommodate the file, auto-flowing and linking the text boxes together.

If you want two columns then go back to page one and select the first text box > Text Box tools format > Alignment Group and click the downward chevron to opt for the number of columns. Go to next page and next text box and repeat.

I would caution you about the 220 page doc as generally Word is a better choice for long documents. If you have pictures you should compress them all and be sure to back up your work...that large of file might go corrupt fairly easily. Newer versions of Publisher tend to choke on large images and large files sizes.

Edit - Alternatively create a full page text box on the first page > as above convert to two columns > Insert > Insert Page and click downward chevron > select the Insert page... option > in the insert page dialog type in perhaps 220 pages, after current page, duplicate all objects on page one and Publisher will create a 220 page publication with a two column text box on each page.

Go back to first page, first column and click within and again insert your Word file. This time Publisher will ask you if you want to autoflow the text to the next box (column), say yes and you will be asked this question again and again until the whole Word doc has been inserted into the text boxes your originally created and linking them together.

If you have extra pages that you do not need then go to the last page > right click the thumbnail view of the page in the navigation pane and delete. I found that if I held the Ctrl key down I could select a number of the blank page thumbnails, right click and delete all at once.

  • After software was installed, settings were changed, which can cause links to webpages to work incorrectly.

  • Registry keys were changed or became corrupted.
