How do I create a graph by selecting a range based on a selection from - Microsoft Community

janet_coulson say:

How do I create a graph by selecting a range based on a selection from a drop-down list?

How do I create a graph by selecting a range based on a selection from a drop-down list?  I'm using a macro.

The x axis is defined already but I need to select the y axis data based on comparing the selection of the drop-down list to another cell and selecting data relative to that cell.  I don't know how to do that or to select the data range from two different sources.

How do I allocate a title based on the selectin from the drop down list.

How to I allocate axis titles?

e.j. gun say:

Here is a method that does not require VBA (but you can still use VBA to set a cell value)

For this example, let's say we have data on Sheet1 of Book1:

X-Axis Y1 Y2 Y3
0 67 45 5
1 39 22 22
3 65 98 81
6 66 39 95
9 82 14 21
13 67 13 99
17 87 79 7

 

Note:  For this example, I assume that your drop-down list leaves an integer value in cell $Z$2 going from 0 to (however many columns of y-axis data you have) -1.  For this example, that would be 0 to 2.

Create four named ranges:

1.  Bring up the Name Manager (Formulas/Name Manager)

2.  Click on New...

3.  For your x-axis values, enter "XVALUES" for the name, and "=OFFSET(Sheet1!$A$2,0,0,COUNTA(Sheet1!$A:$A)-1,1)" for the "Refers to:"

4.  For your y-axis values, enter "YVALUES" for the name, and "=OFFSET(Sheet1!$B$2,0,Sheet1!$Z$2,COUNTA(Sheet1!$B:$B)-1,1)) for the "Refers to:"

5.  For your x-axis label, enter "XLABEL" for the name, and "=Sheet1!$A$1" for "Refers to:"

6.  For you y-axis label, enter "YLABEL" for the name, and "=OFFSET(Sheet1!$B$1,0,Sheet1!$Z$2,1,1)"

7.  Create a new (blank) chart - I'm using a Scatter chart

8.  Select the chart, right-click and choose "Select Data"

9.  Click the "Add" button

10.  Add a data series to the chart (what you add doesn't matter at this point - just create a single series)

11.  Select the data series on the chart.

12.  In the formula bar, replace the formula for the series with "=SERIES(Book1!YLABEL,Book1!XVALUES,Book1!YVALUES,1)"

13.  Insert an x-axis label into the chart.  Select that label, and in the formula bar, enter "=Book1!XLABEL"

14. Change the value in $Z$2 from 0 to however many "Y" columns you have - the chart should automatically update itself!

Whenever your VBA code changes the value in $Z$2, the chart will update to show the new y-axis data.

Adapt this example to your needs.

 

HTH,

Eric

andreas killer say:
Whatever you do and what kind of lists you always have, you have to use the Range object to add the data to the chart.

Execute the sub below with an empty sheet visible.

Andreas.

Sub Test()
  Dim C As Chart
  Dim S As Series
  Dim A As Axis
 
  'Setup some data
  Dim i As Integer
  Range("A1") = "Nr"
  Range("B1") = "Value"
  For i = 2 To 20
    Range("A" & i) = i
    Range("B" & i) = CInt(Rnd * 10)
  Next
 
  'Create a chart over this cells
  With Range("D2:F12")
    Set C = ActiveSheet.ChartObjects.Add(.Left, .Top, .Width, .Height).Chart
  End With
  C.HasLegend = False
  C.ChartType = xlXYScatterLines
 
  'Add a series
  Set S = C.SeriesCollection.NewSeries
  'The formula for the values are generated automatically if we assign a range object
  S.XValues = Range("A2:A20")
  S.Values = Range("B2:B20")
  'The "title" of the series must be a formula in R1C1 style
  S.Name = "=" & Range("B1").Address(ReferenceStyle:=xlR1C1, External:=True)
 
  'Scale the X axis
  Set A = C.Axes(xlCategory)
  With A
    .MaximumScale = 20
    .MajorUnit = 5
  End With
End Sub

Ensure the Word option 'Keep Track of Formatting' is not selected.

The simplest way to format an envelope is to add the envelope to a document with the default settings. Select All (CTRL+A). The two text areas (Envelope Address and Envelope Return) are highlighted.

Select the address area to display the outline of the address frame. Drag the frame to where you require it. Format the frame to be of sufficient width for your address (it probably already will be) and set the frame height to 'Auto'. Make whatever changes you want to the text format.

With your cursor in the frame, press ALT+CTRL+Shift+S to display the Styles selector dialog. Scroll down to Envelope Address and right click the entry. Choose 'Update Envelope Address to Match Selection'. The changes you have made are now added to the DOCUMENT style! To add the modified style to the template, open the styles dialog again. Right click Envelope Address and select 'Modify'. Check the radio button 'New documents based in this template' and save the change.  The style is now temporarily added to the template of the document to which the envelope was added (probably the normal template). To make it permanent, save the normal template when prompted on closing Word. (If you don't have the prompt it will save automatically).

For documents made from other templates, you will need to copy the style to those templates, or make different changes as above.

Envelope Return is pretty much the same except that it doesn't have the frame.

This method works for all Word versions.

stefan blom say:

The settings that control text flow would be the first thing to investigate.

By default, heading styles are set to "Keep with next"; this setting prevents a heading from ending up alone at the bottom of a page with no accompanying body text paragraph beneath it. If there are many headings with no or little "intervening" text, you may get unexpected results and headings appear to move for no good reason when Word tries to enforce the "Keep with next" setting.

An example document would allow us to tell for certain what is going on, of course.
