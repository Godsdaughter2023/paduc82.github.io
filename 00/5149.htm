Hi,

It's returning a #VALUE! error because one of the values; probably G6 is a text value and not a number. Note that if it's a text value then changing the format to GENERAL won't change it into a number, if it's a text value then it will remain as one.

The way to do it is to type 17.29 in G6 without the $ sign and format the cell as currency and select the appropriate currency symbol for your locale.

The editing language consists of the keyboard layout and proofing tools for that language. The proofing tools include language-specific features, such as dictionaries for spelling and grammar checking, or paragraph direction buttons.

Add an editing language Office 2010 programs
  1. Click Start, and then click All Programs.
  1. Click Microsoft Office, and then click Microsoft Office Tools.
  2. Click Microsoft Office 2010 Language Preferences.
  3. Under Choose Editing Languages, click the Add additional Editing Languages arrow, select the language that you want to add, and then click Add.
  4. Click OK.

Notes

  • After you change the default language, you need to close all Office programs, and then open them again for the change to take effect.

The display and Help languages are the languages used in Office for display elements, such as menu items, commands, and tabs in addition to the Help file display language.

ScreenTips are small pop-up windows that provide brief, context-sensitive help when you rest the pointer on a display element, such as a button, tab, dialog box control, or menu. Notes
