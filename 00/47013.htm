ken blake say:

Two points:

1. There is no way to upgrade from XP to 10. You would have to do a clean installation of 10, not an upgrade.

2. Even if you do a clean installation, trying to run Windows 10 on an XP-era computer typically has a host of programs:

a. The computer may not be compatible with Windows 10, and you would need to upgrade several pieces of hardware.

b. Even if it's compatible, you may need Windows 10 drivers for some of its hardware, and the needed drivers may not be available.

c. Even if a and b above are not issues, the XP-era performance of the computer may be very poor if you run Windows 10 on it.

So if you want to run Windows 10, it's likely better, and maybe even less expensive, to just buy a new computer that comes with Windows 10.

frederik long say:

You can map drives either with Group Policy or using a startup script (=batch file) that contains one or more of the following commands:

net use  /persistent:no
net  use  Q: \\ServerName\ShareName  /User:David  SomePassword

net  use  Q: \\ServerName\ShareName  /User:David  *

Note that embedding passwords in a batch file is a severe security risk which is why the second command is preferable.

    a.       Press the ‘Windows + W’ key on the keyboard.

    c.       Select Troubleshooting. Click on view all and select Internet Connections.

    d.      Click next and follow the on-screen instructions.

     

     

     

    Method 2: Perform Network adapter troubleshooter.

    a.       Press Windows + W key on the keyboard..

    b.      Type troubleshooting in the search tab and press Enter.

    c.       Click View all on left panel.

    d.      Click Network Adapter, new window will popup.

    e.       Follow the on screen instructions.

     

     

    Method 3: Update the Network adapter drivers.

    a.       Press Windows + x, select Device Manager.

    b.      Right click the Network Adapter drivers.

    c.       Click Update drivers.

     

     

    Method 4: Uncheck "Allow the computer to turn off this device to save power

    a.       Press Windows key +X.

    b.      Click on Device Manager.

    c.       Right click on Network adapter.

    d.      Click Properties.

    e.       Click on Power management tab.

    f.        Uncheck "Allow the computer to turn off this device to save power".

    g.       Click Ok.
