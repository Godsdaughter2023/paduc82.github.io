jeeped say:

While I have not experienced this failure myself, I've checked relevant threads and it seems that a System File Check (e.g. SFC.EXE) should rectify the problem. To use this command line utility properly, you should run at with an elevated command prompt.

To open an elevated command prompt. The easiest way to do this is to tap the Windows key and type cmd then hit Ctrl+Shift+Enter together, You can also locate the Command Prompt menu item in All Programs, Accessories then right-click and choose Run as administrator but the first method saves a few steps. You will have to acknowledge the UAC warning and provide administrative credentials if asked. When you have done this you will have a command prompt window with elevated privileges.

At the prompt, type,

SFC /scannow

... and press Enter. This command takes a little while to complete but if it cannot find any of the necessary system files your Windows 7 O/S requires, it will get them from a backup area, install them and register them as .DLLs.

If it noticed that files are missing or of an incorrect version but cannot fix the problem from some reason, it will notify you of the situation. Post back with any error messages received and further assistance can be offered.

Hi Andrew2011,

 

 

1.    Have you made any recent changes on the computer?

2.    Do you receive the message “no Driver Found” on all the other USB port except one?

 

Check in the device manager to know the status of the USB controller on the computer.

a)   Open Device Manager by clicking the  Start button

b)   Click  Control Panel, click  System and Security, and then, under  System, click Device Manager.‌ If you're prompted for an administrator password or confirmation, type the password or provide confirmation.

c)    Locate Universal Serial Bus controllers in the list; double click on it to know the status.

If you find any Universal Serial Bus controller entry that has any exclamation mark, then I would suggest that you reinstall all the Universal Serial Bus controllers on the computer and check if it works.

Note:  If you are using a USB mouse, please follow the "Use the Keyboard" steps to reinstall USB controllers. Windows 7 will reinstall all controllers automatically after restarting your computer. 

Use the Mouse

First, please disconnect all USB devices and then perform the following steps:

 

a)   Click "Start", type "Device Manager" (without the quote) in the Search bar and press "Enter". Click Continue if necessary.

b)   In Device Manager, double click to expand "Universal Serial Bus controllers", right click on the "Host Controller", click "Uninstall" and click "OK".

c)    Repeat the step 2 to uninstall all items under "Universal Serial Bus controllers".

 

Then, restart your computer and Windows Vista will reinstall all USB controllers automatically.

 

Use the Keyboard

First, please disconnect all USB devices. Then, go through the following steps:

 

a)   Press "Ctrl + Esc" to open the Start Menu.

b)   Type Device Manager in the Search bar and press "Enter". Click Continue if necessary.

c)    In Device Manger, press Tab and use "Up" and "Down" to move the highlight to "Universal Serial Bus Controllers".

d)   Press "Right" and expand it.

e)   Press "Down" to move to the first item and press "Delete" to remove it.

f)    Repeat the step 5 and uninstall all items under "Universal Serial Bus Controllers".

 

Hope this helps!
