john mcghie say:

Try this:

  1. Go to one of the headings that did not move, and
  2. from the Ribbon choose Clear All Formatting.

If that fixes that one, formatting has been applied as Direct Formatting, which is highly irritating.

You can clear it by selecting the whole of the text and using ResetPara. First you have to find that command.  On my system, it's Command + Shift + P, but I may have customised it.

To find out,

  1. go to Tools>Customise Keyboard.
  2. In the left window, scroll to "All commands"
  3. In the search field, type "resetpara"
  4. Select it.

When you select ResetPara, the "Current Keys" box will show you want keystroke is currently assigned to it.  Whatever it is, remember it.

Now, if you are SURE the entire document is formatted with styles, just select all the text and hit ResetPara.  That will take every paragraph in the document back to the paragraph properties of its applied style. (This is different from Clear All Formatting, which takes both paragraph and character properties back to the Normal Style).

So: every paragraph in the document will now have the formatting of the style it is set to, but the character properties (bold, italic, etc) are left alone.

You can use the sister command ResetChar to reset the character properties as well, if the result looks a little like a ransom note because people have been playing :-)

Hope this helps

rupertsland say:

How do I list all tables in database using an Access query?

I want to generate a simple list of all tables in my Access database. Just the names of the tables will do. If this can be done using an Access query, that would be helpful. How do I go about this in Access?

rupertsland say:

I found the solution...

All versions of Microsoft Access have system objects that are by default hidden. You need to display these in Access first. Then, create a Select query and choose the table MSysObjects. In your select query, you only need to choose three columns: Flags, Name, and Type.

Next, specify criteria. For Flags, enter 0. For Type, enter 1. You may sort your list by Name, if you wish.

Run the query, and you will see a list of all the tables you created in your Access database.

Screenshots for displaying system objects:

In Microsoft Access 97, 2000, 2002, and 2003, click on Tools, Options. Click on the View tab and check System objects.

If you are using Microsoft Access 2007, 2010, 2013, or 2016, right-click on the navigation pane (just above the search box) and choose Navigation Options. Then, under display options, check Show System Objects and press OK.

Hi,

Use a macro to run the spellcheck routine. You could have a button on the sheet or call with a shortcut key. Number 2053 is UK English and 1033 is US English so use the one you want

Sub Spellcheck()
ActiveSheet.Unprotect Password:="Mypass"
Cells.CheckSpelling CustomDictionary:="CUSTOM.DIC", _
IgnoreUppercase:=False, AlwaysSuggest:=True, SpellLang:=2057 '1033
ActiveSheet.Protect Password:="Mypass"
End Sub
