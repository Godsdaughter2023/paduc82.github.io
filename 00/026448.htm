j3pace say:

input mask which allows periods and ampersands

On data entry form for some of the entry fields, I want to have the input mask not only put all entry in uppercase but also allow the use of characters such as a period (i.e. middle initial with period after initial) and & (i.e. Jim & Jane). 
dirk goldgar say:
On data entry form for some of the entry fields, I want to have the input mask not only put all entry in uppercase but also allow the use of characters such as a period (i.e. middle initial with period after initial) and & (i.e. Jim & Jane). 


If you have a relatively small maximum number of characters to be entered in the field, you could use an input mask like this:

    >CCCCCCCCCC

... where you have as many "C"s as needed to represent the maximum number of characters in the field.  The above example allows up to 10 characters.

You don't necessarily have to use an input mask for this.  You could simply use code or a macro in the AfterUpdate event of the text box to convert the value to upper case.  Usualy input masks are used when the text entered must conform to a specific format.  In the case you describe, the entry seems to be more free-form.

mike h.. say:

Hi,

ALT+F11 to open VB editor. Right click 'ThisWorkboook' and insert module and paste the code below in. With the sheet you want to hide the rows as the visible sheet run the code.

You can run the code in a couple of ways but one easy way is while still in VB editor ensure the cursor is within the code and tap F5

Sub hide_rows()
Dim MyColumn As String, x As Long
MyColumn = "S"
For x = 1 To Cells(Rows.Count, MyColumn).End(xlUp).Row
    If Cells(x, MyColumn) = 0 Then Rows(x).Hidden = True
Next x
End Sub
