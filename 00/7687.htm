If find works, but replace doesn't, perhaps this will help:

 

Sub ReplaceParParSection()
    Selection.HomeKey Unit:=wdStory
    With Selection.Find
        .Text = "^p^p^b"
        .Forward = True
        .Wrap = wdFindStop
        .Format = False
        .MatchCase = False
        .MatchWholeWord = False
        .MatchAllWordForms = False

        .MatchWildcards = False

    stefan blom say:

    As far as I know, you can't display the "Default Paragraph Font" style in the Styles pane (permanently, that is).

    Note that "Default Paragraph Font" isn't really a style. It simply represents the default font formatting of a paragraph style; you can use "Default Paragraph Font" in Find and Replace operations to clear direct font formatting. See http://word.mvps.org/faqs/customization/DefParaFont.htm.

    To reset font formatting directly in the text, you can use Ctrl+SpaceBar (assigned to the ResetChar command). Word 2007 and later provide more options in the Style Inspector, which you can display by clicking the Style Inspector icon in the Styles pane (Ctrl+Alt+Shift+S).

    karma chameleon say: Interestingly, Ctrl+SpaceBar also inserts the Default Paragraph Font character style into the Styles pane. But, it remains there only until you close Word 2007. It is gone when you open Word 2007 the next time. We will use Ctrl+SpaceBar. Thank you for the prompt response.
