A file is usually showing as read only due to the following reasons:

  • If it is a shared file and the owner restricted the permission for it to be edited.

  • A second instance of the Excel (or Word) is running in the background with the same file already opened.
  • Excel (or Word) quit responding and the user who was previously editing the file was not removed.
  • The file is not saved in the same network folder.

  3. Click to highlight the correct e-mail address - mine was the default account

Hi Nithya,

No, the answer is No. A mailbox cannot exist cross two organizations (on-premises and online). Therefore, it is not feasible to use Office 365 eDiscovery to search the on-premises contents. However, if you have set up an online archive for the on-premises mailbox, the contents inside it can be searched out.
