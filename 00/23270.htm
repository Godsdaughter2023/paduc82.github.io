scottgem say:

First, is this a web app or desktop app? WEB apps do not have this ability.

In Desktop apps Listboxes have always had the ability to select multiple items. However, you need to use code to store and retrieve the selections to/from a child table. 

Starting with Access 2010, support for multi-valued fields (MVF) was added for Sharepoint compatibility. However, to make use of the capability, you have to create a lookup field on the table level and set it for Multi-select. This will create a pull down with checkboxes to select multiple items.

While MVFs have some use, they are limited. Most developers will not use them or use them in very limited circumstances. The recommeded structure is to use a lookup table. and a junction table to model what is, essentially, a many to many relationship.
