You really don't need any setup progam on CD or DVD or otherwise in order to configure your wireless router. What you can do is...

  1. Go to the router manufacturers support site and download the user manual for your device.
  2. Find out how to reset the router to the factory defaults.
  3. Connect the router to one of your PCs using an Ethernet cable.
  4. Login to the router admin pages using the default admin user ID and password.
  5. Configure your wireless SSID, encryption type (ie. WPA2 or at a minimum WPA) and encryption key/password/phrase. Document all of this step.
  6. Configure a new administrator password for the router.
  7. Configure your wireless clients for the new wireless SSID, encryption type and encryption key/password/phrase.

Hi,

Regarding your concern with Task Scheduler, you have to make sure that the Task Scheduler service is running. You can verify if the service is running by following the steps below:

  1. Click Start, click Control Panel, and then double-click Administrative tools.
  2. Click Computer Management.
  3. Expand Services and Applications, and then click Services.
  4. Right-click the Task Scheduler services, and then click Properties.
  5. On the General tab, make sure that the startup type is set to automatic, and that the service status is Started. If the service is not running, click Start.
  6. On the Log On tab, make sure that the local system account is selected, and that the Allow service to interact with desktop check box has a check mark.
  7. Click OK, and then quit Computer Management.

0x00041301     SCHED_S_TASK_RUNNING     The task is currently running  [from Task Scheduler Error and Success Constants (MSDN)]

1  If you enable all tasks history [on the right-hand side of the TS window] then a History tab will appear in your task's record.  That might tell you more.  You'll need to widen the History dialog columns to make any sense of it [just drag the dividers in the column headers].
