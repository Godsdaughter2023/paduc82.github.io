Export select excel columns to word table - Microsoft Community

debbielh say:

Export select excel columns to word table

How do you export selected columns of a large Excel Spreadsheet to a Word doc table and be able to repeat export or process/procedure weekly?
bill manville say:

Suppose you want to copy columns A, C and E of a contiguous table in the active worksheet, to replace the first table in the Word document C:\Temp\MyWord.doc

Sub CopyToWordDoc()
  Dim oWord As Object
  Dim oTable As Object
  Set oWord = CreateObject("Word.Application")
  oWord.Visible = True
  oWord.Documents.Open "C:\Temp\MyWord.doc"
  Set oTable = oWord.ActiveDocument.Tables(1)
  oTable.Select
  oTable.Delete
  Range("B1").EntireColumn.Hidden = True
  Range("D1").EntireColumn.Hidden = True
  Range("A1").CurrentRegion.Resize(, 5).Copy ' excludes hidden columns, takes columns A, C and E
  oWord.Selection.Paste
  Range("B1").EntireColumn.Hidden = False
  Range("D1").EntireColumn.Hidden = False
End Sub

e.j. gun say:

If you have that string in Cell A1, then the formula below will convert it to and Excel date/time value, which you can then format as you desire.  The formula assumes consistent format of "yyyymmddhhmm".

 

=DATE(LEFT(A1,4),MID(A1,5,2),MID(A1,7,2)) + TIME(MID(A1,9,2),RIGHT(A1,2),0)

 

HTH,

 

Eric

ronrosenfeld say: The following formula will produce an Excel date/time value, assuming your Excel understands "/" to be the date separator; and ":" to be the time separator:

--TEXT(A3,"0000\/00\/00 00\:00")

You can then either format the cell as   dddd, mmmm dd, yyyy - h:mm AM/PM

or you can place that format in a text function around the value:

=TEXT(--TEXT(A3,"0000\/00\/00 00\:00"),"dddd, mmmm dd, yyyy - h:mm AM/PM")

Again, this assumes a number in the format of yyyymmddhhmm

Function IsInvalidUser() as Boolean

sreejith_n say:

Hello Jose, 

  

Thank you for posting. 

 

  1. I work at a University, currently, we have two tenancies and two Yammer networks.

Tenancy 1 - Staff - SSO

Tenancy 2 - Students - Microsoft account

We are consolidating into one tenancy and the student domain will be added to the staff tenancy. All student will be also provisioned and will use SSO for access. My understanding is that once the domain is added to the tenancy it will synchronize to yammer and therefore all student will be new users of the Staff Yammer network, correct?

Ans: The student users will still be mapped to the old tenancy’s Yammer network. There for please follow the below mentioned steps to ensure that they access Staff tenant’s Yammer network as fresh user:

  1. Delete all users from the Yammer network of Tenant 2 student
  2. Add & verify the student domains in tenant 1
  3. Post this check if the domains have sync to Yammer of tenant 1
  4. If not then contact Yammer support team by creating a ticket to get the domains mapped to Yammer of tenant 1
  5. All these users will now be able to access Yammer of tenant 1 as fresh users.

 

  1. We are not migrating any data. So data is not a concern.

Ans: It is great that you are okay with loosing data on student’s Yammer network, as Yammer network migration or cross tenant Yammer domain shift does not allow data migration.

  1. However, Staff in the Yammer network had already created External Networks with the students added as external guest as they were in a different tenancy. My question is, will the student's in Yammer will be able to access the external networks they were part of once the tenancies are consolidated?

Ans: No. As the users will be deleted with their previous identities on Yammer, they will not be able to authenticate via tenant 2 on Yammer external network of Staff. Their posts on the external network will remain. The users will now authenticate via tenant 2 Yammer network as fresh users & shall be invited again on the Staff’s External Yammer network.

Earlier scenario of authentication:

Student à Tenant 2 à Student’s Yammer à staff’s External network

New scenario would be:

Student à Tenant 1 à Staff’s Yammer à staff’s External network (start as fresh user)

 

 

Hope above explanation will resolve your query.

Please feel free to post further queries.
