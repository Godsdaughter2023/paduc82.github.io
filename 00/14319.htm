cbarnhorst say:

One additional note to lacrumb's reply.  There is a distinction between a retail replacement and a repair replacement.  If you purchased a retail motherboard then the license is gone because the old license is based on the make, model, and unique hardware IDs of the old mobo.  A repair replacement is permitted.  A retail replacement is not.  It sounds like you did a hardware upgrade (later model CPU would require a later model mobo) and not a repair replacement.  Even if one of the old components had failed, you would have had to replace with the same makes and models to qualify for transfer of the license.

Bottom line is you must install the same edition as before and it must be a repair replacement mobo in order to transfer the old license to this device.  Otherwise, as lacrumb says, new motherboard = new device.
