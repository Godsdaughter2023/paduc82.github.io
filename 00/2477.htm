Hi Joseph,

Thank you for responding in Microsoft Community. We appreciate your interest in Windows 10.

Try the below and check if it helps.

Step 1: I suggest you to run SFC scan and check if it helps.

System File Checker is a utility in Windows that allows users to scan for corruptions in Windows system files and restore corrupted files.

  1. Press Windows + x keys, click on Command Prompt(Admin).

OR

  1. Press Windows + R keys, to open Run command.
  2. Type cmd and hit Enter to open Command Prompt.
  3. Type the command "sfc /scannow" without the quotes and hit Enter.
  4. After the scan, restart the computer and check for the issue.

Step 2: Run the DISM Tool.

If the issue persists, I would have you run the DISM tool to check the system health and will try to restore the files. 

  1. Press Windows + x keys, click on Command Prompt(Admin).
  2. In the Administrator: Command Prompt window, type the following commands. Press the Enter key after each command:

  • DISM.exe /Online /Cleanup-image /Scanhealth 

  • DISM.exe /Online /Cleanup-image /Restorehealth

    3.  To close the Administrator: Command Prompt window, type Exit, and then press Enter. 

Note: The tool might take 15-20 minutes to finish running, so please do not cancel it.

It’s not feasible to change the message deferral limit.

 

Messages will be held when a connection to the recipient server cannot be made and the recipient’s server is returning a “temporary failure” such as a connection time-out, connection refused, or a 400-series error.
