Using the System File Checker doesn't restore Registry files. Changing or tampering with the Registry files affects all other settings of the computer. If you are not completely sure what you are touching or editing on the Registry or Group Policy, you will not be able to notice the other impacts of these changes to your operating system.

The only resolution to resolve these are: revert the changes, go back to the restore point, or use the original registry backup file if you have made one. If the following is no longer applicable, resetting is the last resort.

In the future, if you have plans to make changes to the registry, we strongly suggest making a backup of that original registry file. To do this, visit this link: How to back up and restore the registry in Windows for instructions.

Please do understand that making complicated changes on your Registry files does more harm than being useful.
