  • Have you made any changes prior to the issue?
  • What game or application are you trying to download?
  • Are you experiencing the same issue with other games from Store?

In the meantime, we suggest performing these steps to troubleshoot your Windows Store issue:

  1. Check if the date and time settings on your device are correct.
  2. Disable your proxy server through Internet Explorer. Here's how: Open Internet Explorer > Tools (gear icon) > Internet options > Connections > LAN settings > uncheck "Use a proxy server for your LAN" > click OK.
  3. Run the Windows apps troubleshooter. The apps troubleshooter can help resolve issues with games and applications.
  4. Clear the Windows Store app cache. To do so, press the Windows key + R key, type WSReset.exe, then press Enter. It will launch the Windows Store app automatically.
  5. Re-register Windows Store through Windows PowerShell. To do so, follow these:
    A. From the Start menu, type Windows PowerShell. Right-click on it and click Run as administrator.
    B. Type this command: Get-AppXPackage | Foreach {Add-AppxPackage -DisableDevelopmentMode -Register "$($_.InstallLocation)\AppXManifest.xml"}. Once done, restart your computer.
