Elaine Hi, When doing LOOKUPS whether it's a simple VLOOKUP are as in this case a more complex INDEX and MATCH it's possible to include the lookup values with the formula or reference the lookup values from a cell. I prefer the latter method because it's much simpler to edit and use different lookup values by changing a cell contents rather than editing a formula. Take the very simple data set below which is in column A, B C with 2 lookup values in D1 and E1. We could use this ARRAY formula to find ddd in Col A and 44 in column B to return Col C value of vv =INDEX(C1:C20,MATCH(1,(A1:A20="DDD")*(B1:B20=44),0)) As you can see the 2 lookup values are within the formula underlined and if we wanted to change them we would need to edit the formula and then re-enter as an ARRAY with CTRL+SHIFT+Enter. This method is better where we reference the 2 cells D1 and E1 for the lookup values. =INDEX(C1:C20,MATCH(1,(A1:A20=D1)*(B1:B20=E1),0)) Now to change the lookup values we simply change E1 and/or d1 and never need to touch the formula so to answer your question, with your data D1 is the date you're looking for in Col A and E1 is the string you're looking for in Col B aaa 11 zz ddd 44 bbb 22 xx ccc 33 cc ddd 44 vv eee 55 bb fff 66 nn ggg 77 mm hhh 88 ll iii 99 pp jjj 10 ii **** 20 yy
