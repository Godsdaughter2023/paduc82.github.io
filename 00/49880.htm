a. user say:

Hello Paul,

 Thank you so much for posting your query on Microsoft Community. 

I understand that you are not able to enable the adapter in the network sharing Centre. But in properties, windows states that the device is working properly.

This issue could occur due to several reasons like firewall protection, antivirus protection, internet connection properties etc.

I would suggest you to disable the adapter and also disable the antivirus for couple of moment and then try to enable it. Please follow the steps mentioned below and try enabling the Network adapter.

1. Click on Start menu and click on Control Panel.

2. In the control panel search, type in Network.

3. In Network and sharing center click on ‘View Network Connections’.

4. Right click on Lan Connection and click on Properties.

5. Uninstall the protocol that has been installed by the security software by clicking on that protocol and clicking Uninstall button.

6. Click OK and close the box.

7. Restart the computer and try to enable the Network Adapter.
