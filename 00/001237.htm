Slow and constant buffering problem can be attributed to two factors:
1.  Internet speed.
2.  Media player.

Since your internet connection is fine, the problem might be with the windows media player. You can try chaning the buffer settings in Windows Media Player. Remember that each player has its own buffer settings. But before you do that, try using Internet Explorer Performance Troubleshooter if your current operating system is Windows 7.

Visit the link below to know how to use Internet Explorer Performance Troubleshooter.

I: The Internet Explorer Performance troubleshooter
http://windows.microsoft.com/en-US/windows7/Open-the-Internet-Explorer-Performance-troubleshooter

After making use of the troubleshooter, if you still have issues with buffering, then try changing the buffer settings in Windows Media Player.

II: To change the buffer settings in Windows media player

For Windows XP:
1. Start Windows Media Player
2. Click the "View" menu
3. Click "Options"
4. On the "Advanced" tab, click "Streaming Media"(Windows Media)
5. Click "Change" to alter the buffer settings.

For Windows Vista and Windows 7
1. Start Windows Media Player
2. Press Alt Key. Click the "Tools" menu
3. Click "Options"
4. Click on the Performance Tab. Under Network Buffering, change the Buffer value from 5 to 10, 20 or 30.

Remember that the higher you set the buffer the more of the clip will be loaded.

How to reset Internet Explorer settings
http://support.microsoft.com/kb/923737
Note: When you reset Internet Explorer settings, all add-ons and customizations are deleted, and you basically start with a fresh version of Internet Explorer.

The diagnostics utility does not reveal any personally identifiable information, does not reveal enough about your computer or software to compromise security in any way, and does not install anything onto your computer whatsoever.

 

 If you think you might have an installation of Office 2007/2003/2002 &  XP that is not genuine, follow these instructions:

Download and save to desktop - http://go.microsoft.com/fwlink/?linkid=52012
