xbr dionysus say:

Overview

Login Information



  • Gateway: 192.168.2.1
  • Username: None
  • Password: (blank)





How To Find SSID (Network Name)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Channel and SSID" from the menu on the left.
  5. The network name will be labeled as "SSID".



How to Find Wireless Passphrase (WPA/WEP)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Security" from the menu on the left.
  5. The wireless passphrase will be labeled as "Password (PSK)".



Changing Wireless Security Type/ Disabling Wireless Security



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Security" from the menu on the left.
  5. Select the desired security type from the "Security Mode" drop down menu. (To disable the wireless security, select "Disabled".)
  6. Select Apply Changes.



Change Wireless Channel　



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Channel and SSID" from the menu on the left.
  5. Select the desired channel from the "Wireless Channel" drop down menu.
  6. Select Apply Changes.



Change Wireless Mode



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Channel and SSID" from the menu on the left.
  5. Select "802.11g" from the "Wireless Mode" drop down menu.
  6. Select Apply Changes.



Change Wireless Band (Width)



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Channel and SSID" from the menu on the left.
  5. Select "20MHz" from the "Bandwidth" drop down menu.
  6. Select Apply Changes.



UPnP



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "System Settings" from the menu on the left.
  5. Select the "Enable" or "Disable" radio button for the "UPNP Enable/Disable" option, depending on current configuration.
  6. Select Apply Changes.



DHCP Reservation



This feature is not available on this device.



DMZ



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "DMZ" from the menu on the left.
  5. Check the "Enable" check box.
  6. Enter the last digit(s) of the IP address on the Xbox into the "Private IP" field.
  7. Select Apply Changes.



Disable DMZ



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "DMZ" from the menu on the left.
  5. Uncheck the "Enable" check box.
  6. Select Apply Changes.



Port Forwarding



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Virtual Servers" from the menu on the left.
  5. Locate the first available row.
  6. Check the "Enable" check box, and enter "Xbox" for the "Description".
  7. Enter "3074" for both the "Inbound port" fields and the "Private Port" fields.
  8. Select "Both" from the "Type" drop down menu.
  9. Enter the last digit(s) of the IP address on the Xbox into the "Private IP address" field.
  10. Select Apply Changes.



Disable Port Forwarding



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "Virtual Servers" from the menu on the left.
  5. Locate the desired forwarding rule.
  6. Uncheck the "Enable" check box.
  7. Select Apply Changes.



MAC Filtering



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.
  4. Select "MAC Address Control" from the menu on the left.
  5. Select the "Disable" radio button.
  6. Select Apply Changes.



Entering PPPoE



  1. Open a web browser.
  2. Log into the router by entering the default gateway (192.168.2.1) into the address bar of the internet browser.
  3. When prompted, leave the password field blank and click Submit.

  1. Connect the computer to one of the LAN ports on the router via an Ethernet cable.
