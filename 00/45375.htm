I just found out that two updates fail to install on my machines: KB3002885 and KB3000853. They just fail and systems revert. Do what I did:

 * Select all updates, right-click and hide all updates.

 * Make sure you hide all Important and Optional updates

 * Go to Windows Update / Restore hidden updates and restore one update.

 * You should see only one update in the dialog. Hide other updates, if they show up. They will.

 * Install the one update, note the KB numer in case it fails.

 * If the update fails, make a note and repeat skipping past the failed update.

 * Be careful, Windows Update will add items to your list. Hide everything, only allow one update at a time.

 * Keep repeating the process until all updates are tagged as failing or are successfully installed.
