hi bob,

it’s not feasible to achieve it via permission levels since using word or excel means users will have the edit permission, with the edit permission, users will be able to share items without approval.

about your requirement, if you do not want the files in a document library to be shared outside the organization without approval, you may consider using access request settings feature to enable access request if a user wants to share something with others.
note: this will also need approval request if we share something with internal users.

we can accomplish this via the following steps:
1. go to the site > site settings > site permissions under users and permissions > access request settings under permissions


 
2. untick the option “allow members to share the site and individual files and folders.”

more detailed information you can find in this article.
