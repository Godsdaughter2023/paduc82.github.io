How do I add the total number of pages in a Word document header automatically? The Finnish standard writes like 1 (3) meaning this is the page no 1 out of 3. The second page is 2 (3) and so forth. Help!

a. user say:

It partly depends on how you entered the page number in the first place.

However, if you 
 a. click in the header area
 b. press option-F9 to reveal the existing field code for the page number (if you can see it, it will probably look something like this: { PAGE }. If you cannot, it is possible that Word has put the field code into a small text box that will make it difficult to see.

As long as you can see the { PAGE } field, click immediately after the closing brace "}", then
 c. type a space, then a "("
 d. press cmd-F9 to insert a pair of special field braces {  } (you cannot just type them on the keyboard)
 e. type NUMPAGES between the { }
 f. click after the final "}" and type a ")"

so you have 

{ PAGE } ({ NUMPAGES })

Then press option-F9 again, and if necessary select the entire text and press F9 to update the field results.

The F9 keys used by Word may be "in use" in Mac OS X - see Daiya Mitchell's article here:
http://word.mvps.org/mac/fieldsinmacword.html

You can also insert a NUMPAGES field using the Insert->Field... menu item, selecting NUMPAGES from the right-hand list.
