Hi Ian,

The system maintenance generally doesn’t delete any thumbnail cache. I would suggest you to try these steps and check if they help:

Method 1:

The system maintenance troubleshooter can automatically detect and fix most of the incorrect settings. Refer to these steps to run this troubleshooter:

  1. Press Windows key +F to bring up the search Charms bar.
  2. Select settings in the drop down menu below search and type Troubleshooting.
  3. Click on Troubleshooting and select the view all option on the left pane.
  4. Click on System Maintenance troubleshooter and follow the prompts.

 

Method 2:

If the issue still persists, I would suggest you to add the drive or folder to indexing and check if the issue persists.

  1. Press Windows key +W, type “index’ in the search box and click on the ‘Indexing options’.
  2. In the Indexing options window, click on ‘modify’ and add the location of the images folder/drive.
  3. Now click on ‘Advanced’ button at the bottom and then click on ‘rebuild’ to rebuild the index.

Try these steps and let us know the results.
