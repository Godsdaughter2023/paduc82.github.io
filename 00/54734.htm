There are three available Skype application for Windows 10; Classic Skype (7.40), Skype for Windows 10 (UWP) and New Skype (8.13).

Possibly, the one you are referring to is the UWP (12.13) which is pre-installed in Windows 10 devices. To check the application, follow the steps below.

  • Quit Skype. If you can see the Skype icon in the task bar, right-click it and select Quit. If you can’t see it, right-click the task bar (or press Ctrl+Alt+Delete) and click Task Manager, then select Skype and click End Task.

  • Type appwiz.cpl in the Run dialog and click OK.
  • Find Skype in the list, right-click it and select Remove or Uninstall.
