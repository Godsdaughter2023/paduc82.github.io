Office365 Microsoft Teams

Hello, as an admin i would like to know how to manage members of teams created by users?

How to see list of teams created?

vanessa ouyang msft say:

Hi Lukki5,

For the first question, within Microsoft Teams there are two roles: Owner and Member. By default, a user that creates a new team is granted the Owner status. If a team is created from an existing Office 365 Group, permissions are inherited.  An owner can also promote a member to Owner status. It is also possible for an Owner to demote their own status to a Member. Owners can make other members owners in the View teams option. A team can have up to 10 owners. In this situation, it is recommended that you contact the owner to add you into this.

For the second question, you can use PowerShell to achieve this goal, A simple workaround as of now to find the newly created teams is by using Get-UnifiedGroup. By default when a new team is created its Office 365 Group will also be created. The difference is between the regular Office 365 Groups and Teams Groups is Welcome message. It is disabled in groups that are created from Teams. This logic will not work when a new Team is created using existing groups.

Command: 

Get-UnifiedGroup |Where-Object {$_.WelcomeMessageEnabled -like "False"}|select Alias,ManagedByDetails

Before that, use an admin account to connect to Exchange PowerShell.

Vanessa
