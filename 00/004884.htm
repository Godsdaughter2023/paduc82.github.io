mike h.. say:

Hi,

This will record the change time in column B. If you delete the data in column B it will start again from row 2

Right click your sheet tab, view code and paste this in

Private Sub Worksheet_Change(ByVal Target As Range)
Dim LastRow As Long
If Target.Cells.Count > 1 Then Exit Sub
If Target.Address = "$A$1" Then
If IsNumeric(Target) Then
    Application.EnableEvents = False
    Cells(Cells(Cells.Rows.Count, "B").End(xlUp).Row + 1, 2) = Now
    Application.EnableEvents = True
End If
End If
End Sub

tom ogilvy say:

It can be done use the change event (vba macro)

 right click on the sheet tab, select view code and put in code like this

Private Sub Worksheet_Change(ByVal Target As Range)
if Target.count > 1 then exit sub
if Target.Address = "$A$1" then
   lastrow = Cells(rows.count,"M").End(xlup).row
   with cells(lastrow + 1,"M")
     .Value = Now
     .Numberformat = "dd/mm/yyyy hh:mm"
     .EntireColumn.Autofit
   end with
end if
  
End Sub

some would advise disabling events since this codes make a change in the same sheet.  Any recursive calls should be essentially screened out with the second IF statement, but for completeness you might want to use this instead.

Private Sub Worksheet_Change(ByVal Target As Range)
if Target.count > 1 then exit sub
if Target.Address = "$A$1" then
   lastrow = Cells(rows.count,"M").End(xlup).row
   with cells(lastrow + 1,"M")
    on Error goto ErrHandler
    Application.EnableEvents = False
     .Value = Now
     .Numberformat = "dd/mm/yyyy hh:mm"
     .EntireColumn.Autofit
   end with
end if
ErrHandler:
Application.EnableEvents = True
End Sub

Change the "M" to whereever you want to record the changes.
