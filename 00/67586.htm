Check if the network adapter is working properly in the Device Manager.

 

Follow these steps to do so:

 

a. On the desktop, press Windows key + R and type devmgmt.msc and press Enter to open the Device Manager.

b. Expand Network Adapters and right-click on the adapter.

c. Select Properties and check if the device is working properly under Device Status.

 

If there is a yellow triangle with an exclamation mark in it, then it indicates that there are some issues with it.

 

 

Method 1: If the computer is a laptop, make sure that the network connection is enabled.

 

Method 2: Make sure that WLAN (wireless) services are running.

 

Follow these steps to start it.

 

a. On the desktop, press Windows key + R to open the Run Dialog.

b. Type services.msc in the Run Dialog, and then press Enter to open Services.

c. Right-click the WLAN service and then click Properties.

d. If the Startup type box is set to Disabled, change it to Manual.

e. Click OK to close the Properties window.

f. Right-click the WLAN service, and then click Start. The service should start without errors.

 

Check if the issue is resolved.

This indicates that Microsoft Windows or a kernel-mode driver accessed paged memory at DISPATCH_LEVEL or above.

In my PC I installed 4 GB ram and it is showing 2.97 GB usable why? Please advice.

In my PC I installed 4 GB ram and it is showing 2.97 GB usable why? Please advice.
trekdozer say:

32 bit can only see a max of 4GB - usually around 3.5GB, but if your graphics card is using shared memory it will be even less. ﻿Check to see how you can reduce your graphics card shared memory and increase available RAM, usually it's done in the BIOS, or in your GPU settings.


Also make sure your motherboard supports the amount of RAM you have installed.

The usable memory may be less than the installed memory on Windows 7-based computers:

http://support.microsoft.com/kb/978610
