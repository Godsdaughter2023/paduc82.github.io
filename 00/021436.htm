Well, during the past year there have been no changes at all to Office 2008. The fact is that updates to any specific version rarely make significant revisions to features of the software.

Office 2008 was rendered obsolete as of [IIRC] April of last year, so no features or fixes have been modified/implemented since prior to that time. In fact, no such changes have been made to the program since before the 12.3.0 update released in June of 2011.

  • Re the Track Changes warning: That is controlled by Word> Preferences> Security - Warn before printing, saving or sending... . If that preference is checked the notification will occur if the document contains any unresolved tracked changes or Comments. It's recommended that the box be checked for exactly the reason you cite -- To avoid inadvertent inclusion of content that is not to be seen by a recipient. If the notification is being triggered it's most likely because there are still unresolved tracked changes and/or Comments in the document. To verify:
  1. Display the Reviewing Toolbar
  2. Confirm that Final Showing Markup is displayed at its left end
  3. Make sure Track Changes is turned Off
  4. Use Tools> Track Changes> Accept or Reject Changes
  5. Delete any Comments
  6. Save the file
  • As for your other concern, I have no idea what you mean by "lock the formatting". If you're referring to password protection you can use Tools> Protect Document, then select the Forms option or you can use a Password to Modify in the same Security preferences mentioned above. daniel g. grau say:
Actually, "Same as previous" is not related to numbers. Rather, it means that the current header or footer will repeat from the previous section.

In general, if your document requires the same header/footer throughout, then you only need one section (its parameters being implicitly stored in the last ¶ character of the file). 

You should insert a new section only if you want to change the page size, its orientation, use a different header and/or footer, change the number of columns, alter page margins, restart page numbering and probably other stuff I now forget.

I've seen documents where sections breaks serve no purpose except changing to a new page, so as text reflows those spurious and inactive section breaks move up or down on the page, become active and cause "unexplained" changes in the looks of headers or footers. For instance, text flagged by the spell checker that cannot be seen or found in the file, if it's in an even header of a single-page document.

So make sure non-printing characters are showing (Word > Preferences > View) and delete all unwanted section breaks. Word provides a section indicator (like "Sec 5") in the status bar at the bottom of the document, and also when you edit the header/footer (like "Footer- Section 5-"). Pay attention to them.

If you want to restart numbering, then you would insert a section break, take off "Same as previous" to obtain independent headers/footers, then click on the page number and use Insert > Page number > Format > Start at to make it begin at 1.

graham mayor say:
You have a very unfortunate attitude to those from whom you seek help.

What you are asking for will not achieve your intended aim. Saving to DOCX from DOC, as you have been told, does not apply any security whatsoever. If someone had access to your documents as DOC format, they could equally access them as DOCX.

The files are not encrypted unless you encrypt them with password protection. If that is what you wanted to achieve then you should have said, instead of sneering.

The following macro will save all the DOC format files in a folder with a password that you enter in place of 'MyPassword' at the start of the macro and save them as DOCX. The macro has little error checking so it also produces a log of the files it saves. If you forget the password you will not be able to open the documents.

The macro does not delete the original documents, to allow you to ensure that you can access the changed documents before you delete or (better still) archive the originals.

Run the macro at your own risk.


Sub SaveAllAsDOCX()
Dim strFileName As String
Dim strDocName As String
Dim strPath As String
Dim oDoc As Document
Dim oLog As Document
Dim fDialog As FileDialog
Dim intPos As Integer
Const strPassword As String = "MyPassword"

Set fDialog = Application.FileDialog(msoFileDialogFolderPicker)
With fDialog
    .Title = "Select folder and click OK"
    .AllowMultiSelect = False
    .InitialView = msoFileDialogViewList
    If .Show <> -1 Then
        MsgBox "Cancelled By User", , "List Folder Contents"
        Exit Sub
    End If
    strPath = fDialog.SelectedItems.Item(1)
    If Right(strPath, 1) <> "\" Then strPath = strPath + "\"
End With
If Documents.Count > 0 Then
    Documents.Close savechanges:=wdPromptToSaveChanges
End If
Set oLog = Documents.Add
oLog.Range.Text = "Documents with Password " & strPassword
If Left(strPath, 1) = Chr(34) Then
    strPath = Mid(strPath, 2, Len(strPath) - 2)
End If
strFileName = Dir$(strPath & "*.doc")
While Len(strFileName) <> 0
    WordBasic.DisableAutoMacros 1
    Set oDoc = Documents.Open(strPath & strFileName)
    strDocName = ActiveDocument.FullName
    intPos = InStrRev(strDocName, ".")
    strDocName = Left(strDocName, intPos - 1)
    strDocName = strDocName & ".docx"
    oDoc.SaveAs2 FileName:=strDocName, Password:=strPassword, _
                 FileFormat:=wdFormatXMLDocument, CompatibilityMode:=14
    oLog.Range.InsertAfter strDocName & vbCr
    oDoc.Close savechanges:=wdDoNotSaveChanges
    WordBasic.DisableAutoMacros 0
    strFileName = Dir$()
Wend
End Sub
