As Rick noted, if you enter a time like 3:05 Excel automatically evaluates that as 3 hours 5 minutes.

You would have to enter the full time as h:m:s then you can format to only display the m:s. For example, entered in h:m:s format:

  • A1 = 0:3:05
  • A2 = 0:12:55

Format the cells as Custom m:ss. They will now display as:

  • A1 = 3:05
  • A2 = 12:55

Then you can use a simple sum formula:

=SUM(A1:A2)

Format as Custom [m]:ss.

When you enter 3:05 into a cell, Excel sees that as hours:minutes not minutes:seconds, so you cannot just sum your values to get what you want. Instead, try this formula to do your summations...

=SUMPRODUCT((A1:A1000)/60)

Change the range as appropriate and Custom Format the cell using [m]:ss as your format pattern.

jay freedman say:
If everyone who uses the form has Word 2010, you can use a checkbox content control instead of a legacy checkbox form field. That content control can be put in a text box or a frame (for the latter you need to add the Insert Horizontal Frame button to the Quick Access Toolbar), and it will operate without protecting or locking the document in any way. However, no one who uses Word 2007 or earlier can use these controls, even with the Compatibility Pack.

If you must use form fields, you must use a frame instead of a text box. Then, in order to make the form fields active and still allow editing, the document must be protected/locked in a specific way:

  • Insert the frames and place the form fields in them. Set any properties of the form fields as needed.
  • Place a section break (usually a continuous one) before and after each paragraph or group of paragraphs that contain the anchors of the frames (see #2 in http://sbarnhill.mvps.org/WordFAQs/AnchorToHeader.htm). Note that these paragraphs that contain frame anchors are going to be locked, so no text in those paragraphs will be editable. It may be best not to have any text in those paragraphs, just an empty paragraph mark.
  • On the Developer ribbon, click the Restrict Editing button to open the pane on the right.
  • Check the box under Editing Restrictions and choose "Filling in forms" in the dropdown.
  • Click the "Select sections" link under the dropdown. In the small dialog, uncheck each section that should remain editable, and leave checked each section that contains frame anchors.
  • Click the Start Enforcing Protection button, apply a password if needed, and close the pane.
