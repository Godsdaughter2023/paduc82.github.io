slance310 say:

Hi,

If you have not run a Disk Cleanup of System Files on the 10162 machine, you should still have a hidden $Windows~BT folder...

In the $Windows~BT\Sources folder there will be a Install.esd file...

This is basically the compressed ISO of build 10162...

If you have it, download the ESDtoISO Tool and copy the Install.esd into it's folder...

Right Click on the ESDtoISO batch file and select "Run As Administrator"

When asked to choose an option, select 1... that's it... when finished you will have an ISO of build 10162...   Here is the link to the forum that provides a link to the tool and more detailed instructions should they be needed...

http://www.tenforums.com/tutorials/2570-esd-iso-create-bootable-iso-windows-10-esd-file.html

Be advised that the conversion process will max out your CPU, so if you have any temperature control problems with your machine, have a extra external fan of some type ready...

The P2P feature is actually set by default... You can find them at ...

All Settings... Update and Security... Advanced Options... Choose How Updates are delivered..

I don't know if it works with new builds... I'm almost certain you will have no idea from where any data will be coming from...

Outlook 2016 set settings from previous version

Hello Outlook Gurus - We have installed Office 2016 on pilot users. They didn't like the change in their view. They want what they use to see in Outlook's previous version - SendTo view. Is it possible to set this settings and similar settings during the installation? Is it possible to set Group Policy? 

Thanks & Regards,

diane poremsky mvp say:
if you are using Exchange server, the views you used in the older version would be available in Outlook 2016 - the default view should be kept when you switch clients.

But no, you can't set views using group policy or other settings. The views are stored in the mailbox and policies won't apply. To the best of my knowledge, you can't set them on a mailbox using powershell, but i will double check.

The default Sent To view is only available in the Sent folder, but you can copy the view and make it available to all folders.
