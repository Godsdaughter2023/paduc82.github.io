Take ownership of temp folder and grant permission to it. That should give you all the rights to delete the contents of that folder.

b.    Go to C:\Users\Username\AppData\Local.

a.     Right click tem folder and select  “Properties” from Context Menu.

b.     Click on  "Security" tab.

c.     Click on  “Advanced” button.

d.     Go to  "Owner" tab in Advanced Security Settings for User windows.

e.     Click on  Edit Button and select user from given Change Owner to list.

a.    Right click on the file and select  “Properties” from Context Menu

e.     Click  Ok for changes to take effect and click  ok to exit from Properties window.

Hi Sandy. I'm Greg, an installation specialist and 9 year Windows MVP, here to help you.

When you use the Import Wizard built into Windows 10 to import the pictures, there should be a box to check in the Settings to delete the files after importing.

An SD card can be accessed the same way and files treated like any other drive, or you may have a slot to insert the card into the PC to read and edit it like any other drive.

If the SD card needs a drive letter to be read in Windows 10, right click Start to open Disk Management, scroll down to find it on the drive map, right click to assign a letter. Here's also how to format it if needed: https://www.lifewire.com/format-sd-card-using-w...
