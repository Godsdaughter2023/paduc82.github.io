windows 10 - Microsoft Community

gherpienicki say:

windows 10

When I tried to install windows 10 near completion it said my puter doesn't support Nx. What is nx ?

archangle11 say:

hello,

NX also called NX processor bit or NX bit or No eXecute bit ( I capitalized the initials) is a security feature that is an essential hardware feature to run a windows x64 bit operating system. It is a security feature that marks certain areas of memory as non executable to protect users from malicious software from running. PAE and SSE2 are also needed for running X64 bit windows OS.

Intel chipsets market it as XD bit or eXecute Disable bit, AMD uses the term Enhanced virus protection and ARM architectures refer to it as XN for eXecute Never.

To activate NX follow these steps: ( please note that most x64 bit processors support NX feature)

1) switch on the computer and then access BIOS as soon as you see the initial screen by pressing F2 or F12 or delete (function keys for accessing BIOS may differ from PC to PC). If you are using windows fast startup feature then you need to restart to get the option to access BIOS

2) go to security option in the bios and then search for the NX or no execute function and enable it. It might also be called as execute disable (XD BIT) in intel processors , enhanced virus protection in AMD and execute never in ARM chipsets.

once you enable this feature then you are good to use windows x64 bit OS versions.

But in case your processor doesn't have NX feature you won't find the option in BIOS.
