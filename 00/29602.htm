    davem121 say:
    Hi Its,

    Folders like Documents, Pictures, Videos, Camera Roll are not actually folders.

    They are library folder - and are treated differently to all normal folders on your hard drive

    Right click one of those folders and choose Properties

    You will notice they have more tabs on that Properties dialog than any normal folder

    One of those Tabs is Location

    When you dragged that folder onto your D Drive, you basically changed the Library location, rather than creating a new copy of that folder

    If you want to revert that change, on the location tab, click 'Restore Default' and click Apply and OK

    That will move that folder back into your Pictures Library

    Then you can copy and past that library folder onto your other drive to make a copy of it . . .

    a. user say:

    I decided recently to really clean up the horrendously disorganized state of my digital life because it's just driving me crazy and I can't take it anymore basically. 

    Bravo! I did the very same thing years ago, and I'm very glad that I did. I'll be glad to help you.

    Windows 10 includes two backup utilities: File History and System Image Builder. File History was designed to backup individual files as they change. It works very well, and Microsoft has announced no plans to do away with it. File History is ideal for when you want to go back to a previous version of a file you were working on -- when you don't like the changes you made to your autobiography, and wish you could go back to an earlier version.

    You can backup all your personal files at the same time with File History, but it's an awkward procedure because File History wasn't really designed for that. It will be much easier and more convenient to backup all your personal files with a third-party file-and-folder backup application. There are many great file-and-folder backup applications, like Backup4All.

    System Image Builder was designed to make image backups. An image backup is a copy of an entire drive at a point in time. For example, you can backup the entire C: drive at once. That's how you can backup Windows and all your software.

    With last year's Fall Creators Update, Microsoft deprecated System Image Builder (not depreciated, which is something accountants do.) That means SIB is no longer being developed, which is a hint from Microsoft that it will probably be removed from Windows at some time in the future. In fact, Microsoft recommends that users instead use a third-party image backup application. Here, too, there are lots of great image backup applications, like Macrium Reflect.

    Microsoft hasn't officially announced why they deprecated SIB so I can only guess their reasoning: There's so much great image backup software from organizations that specialize in this field, why try to re-create the wheel?

    After this long-winded explanation, the answer for you is: To backup effectively and conveniently, use third-party backup software. Don't bother with File History - it wasn't designed to do what you need to do. Rather, and like many other utilities added in to Windows for no extra cost, File History exists to provide a fundamental backup function for people who don't know how or are too anxious to install third-party software.
