Is there a way to turn off the cached address book for the entire - Microsoft Community

loriann chaussee say:

Is there a way to turn off the cached address book for the entire organization

Because the offline global address book is enabled the users are seeing out of date info.  Is there a way to turn it off for the entire organization?  Thanks

ruel mayoya msft say:

Hi Loriann,

An offline address book contains a snapshot of information from the Global Address List (GAL). If Outlook is left running constantly in Cached Exchange Mode, it automatically updates the offline address book one time per day (about 24 hours) depending on updates on the computer that is running Exchange.

It is not feasible to turn it off for the entire organization because whenever there are new updates from user information in the GAL, this information will not be updated (after 24 hours) to the snapshot copy in your Outlook client via Offline Address Book. For example, names, e-mail addresses, titles, and office locations. You will not see if a new member has joined or has left your department by Distribution list, if a member has been promoted with another title, if an office location of a user has changed, etc.

You can prevent users from downloading the Offline Address Book without disabling Cached Exchange Mode by modifying the Windows registry, Please refer to this article. Make sure you have backed up the registry before modifying it.

Feel free to post back if you have further questions.

Best Regards,

Ruel

jay freedman say:

Almost certainly the recipient has a different printer driver than you do. Word relies on the current printer driver for character width/spacing information, so a different driver can lead to different line breaks, and sometimes to different page breaks. See http://www.word.mvps.org/FAQs/Formatting/TextReflow.htm for more information.

If exact reproduction of page layout on different computers is important, export the Word document to PDF and send that.

If the change in layout is acceptable but the TOC has the wrong page numbers, then the recipient needs to right-click the TOC and choose to update it.

suzanne s. barnhill say:
There is no converter to allow Word 97 to read a Word 2007 .docx document, so I assume you must be sending the document in .doc format. This in itself might result in some slight difference, but the main problem will probably be a difference in printers; see http://word.mvps.org/FAQs/Formatting/TextReflow.htm.

Dollar sign $ is used in a formula to make cell reference Absolute. By default when you apply formula, it is consider as relative reference, hence no dollar sign. But sometimes we need to make our reference fixed which may absolute or mixed. In case of Absolute reference we use 2 dollar sign, one for row and another for column while in case of mix reference either of one is used.

This dollar sign can be applied either manually or by pressing function key F4 while you are in edit mode and selected the relevant portion of formula.

I don't think that dollar will be applied automatically, may be you put it by mistake by pressing F4 key.

Now in case you want to get rid of it, then either remove it manually if there are few ones and known or you may do Find and Replace where in Find box put a dollar sign and keep Replace box blank.

You may replace one by one or Replace All as per your need.

From a licensing standpoint that should not work.  The hardware profile of the vm is different from that of the host.  Imaging the virtual harddrive and transferring to the host harddrive (V2P) will work, but as soon as the computer starts and builds the device table Windows will trigger the activation wizard and your product key won't work because the host computer is not registered as a Windows 10 device.  If the issue is the vm is running Pro and the host is running a Home version of an OS then I see where you're going with this.  However, you won't get there with V2P because of the licensing.  You didn't say what the host is running now, but if it is 7 or 8 simply upgrade to 10.  That will register the host as a Windows 10 device.  By the time you reach that point, of course, you have Windows 10 for free already and you might as well continue to run the vm in the Insider Program.
