Pavan sy is correct. Windows live mail is not a MAPI email client. The Send To option in Office applications only work with a MAPI client such as Outlook, Windows Mail...Etc.

 

Note: Make sure you back up the registry. Follow the article given below to back the registry:

http://windows.microsoft.com/en-US/windows7/Back-up-the-registry

 

But you can try the workaround mentioned below to check if it works fine.

 

1. Click on Start > All Programs > Accessories > Run > Type regedit

2. Browse to HKEY_LOCAL_MACHINE > Software > Client > Click on Mail

    (Default)         REG_SZ         Windows Live Mail

Right click on (Default) and Click Modify. In the Value Data box type Windows Live Mail.

diane poremsky mvp (slipstick.com) say:
Outlook (either version) does not not have an option to only show specific signatures for a profile. If the signatures are in the folder, they are available.

Outlook (either version) use will the signatures assigned to the account, so there should never be an accidental use of the wrong signature if you only have the two signatures (or 4, if you use a different one for replies).

anula d'souza say:

Hi George,

Thank you for posting in Microsoft Community. As I understand you want to know what are subscribed folders.

 

If you use an IMAP account with Outlook 2010 and you've created additional folders outside of Outlook, you may have trouble seeing these new folders in Outlook. In order to view new folders in Outlook, you will have to subscribe to those folders.

 

You may try the below steps and check if you receive the synchronization message for subscribed folders:

1) Open Outlook. Navigate to File > Options > Advanced > scroll down to the send/receive group> click on the send/receive button.

2) In send/receive groups window click on the Edit button.

3) Un-check the option "Get folder unread count for subscribed folders"

4) Click Ok and then exit out of options and perform a send/receive to test if it is now working.

 

I hope the above suggestion helps. If you need further assistance on any Office related issues let us know and we will be happy to help you.
