If you the administrator account is unavailable, you lose administrative privileges on your system. The only way to get the admin privileges is through advanced options. I would suggest you to boot the system from a Windows 10 installation medium and then try the following methods from the advanced options.

If you do not have Windows 10 installation DVD, then follow the instructions on Windows 10 Software Download page to create one: http://www.microsoft.com/software-download/windows10

After creating the disc, refer to the following steps to enable hidden administrator from command prompt.

a) Boot from the Windows 10 installation disk.

b) In the “Windows Setup” dialog box that appears, set the Time and currency format, Keyboard or input method and the language to install and then click “Next”.

c) Click on “Repair your computer” from the lower left corner.

d) From the “Choose an option screen”, click on “Troubleshoot”.

e) Click “Advanced Options” in the “Troubleshoot” screen.

f) In the “Advanced Options” screen, select “Command Prompt”.

g) Enter the following command to enable the hidden administrator account:
net user administrator /active:yes

h) Close the command prompt, reboot and login to the account called Administrator.
{In order to disable the hidden administrator, type the above command by replacing yes with a no}

Boot to the system and check if you are able to login to the administrator account.

You can change the admin rights of your standard account with this account.

¡firedog say:

Several users have reported this problem recently, and it's not at all clear what's causing it. The only known workaround which seems to help is to delay the start of the Windows Live ID Sign-in Assistant Service. To do this,

  1. Type services.msc into the start search box and press Enter. This will open the Services snap-in of the console.
  2. In the list of services, locate the service and double-click on it to open its properties dialogue.
  3. In the Startup box, select Automatic (Delayed Start) from the dropdown, then click OK, close the services snap-in and reboot the computer.

This will prevent the service from trying to start before Windows is fully launched and seems to avoid whatever conflict it is that is causing the error you're seeing. It won't have any effect on the operation of your Windows Live programs.

a. user say:

Hello,

Thank you for posting your question on Microsoft community.

I appreciate you for providing details about your query.

DVD with Windows operating system bought from a store and Windows installation media(DVD) created using Microsoft's articles are same. They both can be used to perform clean, custom and repair installations.

Recovery partition is created by OEM(Original Equipment Manufacturer) on your computer which contains recovery image which is typically 3-6 GB in size.

To save space on your PC, you can delete the recovery image from your PC and use a recovery drive instead. Windows 8.1 includes a built-in tool to create a USB recovery drive. Windows will let you know how big the recovery partition is, and you will need a USB flash drive at least that big.

Note: Creating a recovery drive will erase anything already stored on your USB flash drive. Use an empty USB flash drive or make sure to transfer any important data from your USB flash drive to another storage device before using it to create a USB recovery drive.

I hope this information helps.

Please do let us know if you need any further assistance.
