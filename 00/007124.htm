    Top 0.5"
    Bottom 0"
    Left 1.25"
    Right 1.25"

    and click on the OK button
  6. From the Table Tools>Layout tab of the ribbon, access the Properties dialog and make the following changes:

    First Row Height 3.5"
    Second Row Height 0.5"
    Third Row Height 3.5"

    First Column Width 4"
    Second Column Width 0.5"
    Third Column Width 4"

    Then click on the OK button
  7. Select the first column of the table and use the Split Cells facility to split that cells in 2 columns and 3 rows, unchecking the box for Merge cells before split.
  8. Repeat step 7 for the third column
  9. Select the top left cell of the table and use the Text Direction facility in the Alignment section of the Table Tools>Layout tab to change the text direction so that it runs from top to bottom with the text centered in the cell (vertically and horizontally)

1.) Field Settings > Layout & Print > "Show Items with No Data"
