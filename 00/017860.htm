vijay a. verma say:

Doing through formula would be very tedious and error prone also. Below is a small VBA function to achieve the same. Just follow the steps, it is very easy.

1. Make a backup of your workbook. 
2. Open your workbook and ALT+F11
3. Locate your Workbook name in Project Explorer Window
4. Right click on your workbook name > Insert > Module 
5. Copy paste the Macro code given
6. Go back to your Workbook and ALT+F8 to display Macro Window
7. Run your Macro from here
8. Delete you Macro if the Macro was needed to be run only once.
9. Otherwise save your file as .xlsm if you intend to reuse Macro again.

10. Just put following in a cell

=FindfromSheets(D3) where D3 is reference to cell and drag down -

'*** Vijay A Verma (eforexcel.com)

Function FindfromSheets(Cell) As String
    Dim Ws As Worksheet
    Application.Volatile
    For Each Ws In Worksheets
        If Ws.Name = Cell Then
            FindfromSheets = Ws.Range("C3")

            Exit Function

        End If
    Next Ws
End Function

tucker [msft] say:

We've been doing some investigating and we believe we understand the issue. 

The primary cause is that sometimes we cache the position the pen was in when it first neared the screen, even if you move it before you ink.  This results in the system thinking that it should deposit ink in that original position instead of where you're actually inking. 

Until it is fixed you may be able to minimize the occurrences of the problem by being careful to bring your pen into range of the screen directly over where you intend to ink.  When you finish the ink, make sure to move your pen away from the screen at least a couple inches before inking elsewhere on the document.  This may not completely solve the problem, but it should make it less common. 

Our January and February updates contain some improvements to our pen in range code that will minimize the "teleporting" behavior.  We have a final fix that should eliminate it completely after the update that will go out the end of this month and be available in April.

Thanks for your help chasing down this issue.  Let us know if you continue to have problems after the updates.

You should never open Word documents directly from a flash drive. Nor should you save directly to or print from a flash drive. Instead, copy the file to the hard drive, open and work on it from there. After saving and closing the document, copy it back to the flash drive.
