ronrosenfeld say:

I guess you are using JWalkenbach's XDATE add-in, which does not include any equivalent for the edate function, which, even if wrapped in XDATE, will still not recognize dates prior to 1900.  I would suggest a simple VBA macro to generate your list.  As written below, all the dates are output as text strings, and the starting date and number of years are hard coded.  You can go back to the year 100 AD based on the Gregorian calendar extended back past it's real start.

The sequence is also defined in the code, as is the results range AD:AE, but, if it works for you, the code can easily be made more flexible.

To enter this Macro (Sub), <alt-F11> opens the Visual Basic Editor.
Ensure your project is highlighted in the Project Explorer window.
Then, from the top menu, select Insert/Module and
paste the code below into the window that opens.

To use this Macro (Sub), <alt-F8> opens the macro dialog box. Select the macro by name, and <RUN>.

Option Explicit
Sub DateSequence()
    Const dStart As Date = #12/31/2014#
    Const lYears As Long = 250
    Dim rDest As Range
    Dim vRes As Variant
    Dim I As Long

Set rDest = Range("AD1:AE1").Resize(rowsize:=lYears * 12)
ReDim vRes(1 To lYears * 12, 1 To 2)

vRes(1, 1) = dStart
vRes(1, 2) = 1

For I = 2 To UBound(vRes)
    vRes(I, 1) = CStr(DateAdd("m", -Int((I - 1) / 4) * 3, dStart) - (I - 1) Mod 4)
    vRes(I, 2) = I
Next I

rDest.EntireColumn.Clear
rDest = vRes
rDest.EntireColumn.AutoFit

End Sub

Hi,

I had the same issue as @fr51 and was able to solve by combining his post regarding removing the Birthday Calendar and then following @Alden D's post to re-add it. Went as follows:

  1. Go to your Calendar page.
  2. Select the 'Gear' symbol in the top right corner, and then 'Options.
  3. Select the 'Birthday Calendar'
  4. At the top, click 'Delete' and confirm the deletion.
  5. Go back to your Calendar page.
  6. Click the drop down arrow beside the "New" tab.
  7. Select Birthday Calendar. Then click Save.
  8. Your Facebook birthdays should now re-appear in the Birthday calendar correctly.

Hope this helps anyone else with the issue.
