Stop windows update from updating ALL drivers (including sound drivers)

1. Start Search and type Change device installation settings

Windows update will still update all other settings but not drivers (hopefully)

Later (closer to the release date) - you can reverse the steps to allow updating all devices.

What Microsoft Edge is STILL Missing

A Quick Introduction: I am Travis McEndree, A Windows Insider, Bing, Office, Xbox, etc. I use all the Microsoft Software I can because I want to be using software that I feel makes me more productive, a goal Microsoft sets in mind for all of its software. I am not a Microsoft Employee, but rather a Microsoft Enthusiast. As a Microsoft Enthusiast, I believe it is my job to Microsoft the truth about their products.

I hope you, and Microsoft, enjoy reading my thread. Let's get started.

Microsoft Edge was released with the release of Windows 10 and it has been getting better every day. Enough so, that 2 days after the release of Windows 10, I knew it would become my primary browser. I mean, being able to send pages to my devices and have them AUTOMATICALLY open? Awesome. However, with the competition getting better than ever, Microsoft needs to follow suite, otherwise, Microsoft Edge will be another IE.


1. Microsoft Edge Doesn’t Scale Correctly to The Devices it’s running on

Microsoft Edge runs on PCs, Tablets and Phones. While the big tabs and user interface makes sense on a tablet and 2 in 1, using this big interface doesn’t make too much sense on PCs and Laptops. Horizontal screen space is very precious and edge takes up quite a bit of it. Firefox with the release of quantum detects what device it’s running on and will change the UI to follow the appropriate size. Please Microsoft, Please make the UI smaller on these devices. The UI shouldn’t be an inch on a laptop or computer. 

2. Send to Devices only works on PCs

I love this feature as I can easily send pages from my phone to my PC. But there are times that I need the exact opposite. I am a student, and I frequently have to send pages from my pc to my iPhone as I’m Mobile. Microsoft needs to make this feature a two way feature.

3. Developer Tools are lacking and mediocre at best

If you’ve been to my Twitter page within the last 4 years, you would know that I spent time with Mozilla as a SUMO Volunteer. When I used Firefox, the thing I loved most about it was its Developer Tools (Especially Firebug). Microsoft’s current dev tools aren’t just mediocre implementations, but they’re slow and laggy.

4. Settings Are a Mess

As Martin Brinkmann of Beta News Puts It, The way settings uses the available screen space is ridiculous. While literally every other browser makes use of the available screen real estate to display all the options, Edge locks them to the side of the screen, which is extremely annoying. Not to mention Edge’s lack of a search bar to find settings quickly. Take a hint from Mozilla and Google and have settings open in a tab, and provide a search box. A side from that, settings are unorganized making finding a certain setting difficult.

5. Lack of Control over UI

While Firefox and Chrome allow you to place and hide extension icons, Edge locks them into place. This is useful for touch devices, but if you have a mouse, the odds of you accidentally moving an icon is low. I don’t need an icon for some of my extensions that don’t provide any options. Mozilla and Google realize this and allow users to hide these from view. I feel like the Microsoft Edge team could add this easily.

6. Scrolling doesn't feel smooth, at all.

The way Microsoft Edge scrolls is terrible. I know Microsoft Edge is trying to “snap” it for paragraphs, but this behavior is terrible. So terrible that Apple removed it from iOS in iOS 11. I feel like Edge could take a note from Firefox who’s scrolling is something many people love.

7. Cortana isn’t built into Edge for iOS/Android.

One of my favorite things about Edge, is Cortana. I love being able to go to a website and get relevant info like directions, coupons, etc. However, Edge for iOS and Android lack this. Adding this would improve the User Experience greatly as it does on Windows.

Hello Joe

TO change the password for an existing email account in Outlook desktop - follow these steps

#1 - From within Outlook - press CTRL+ALT+S which brings up the Send/Receive settings dialog
#2 - Select the <Group> (most people only have one) --> click on <Edit>
#3 - Select the <Email Account> --> Click on <Account Properties>

An alternative approach is

#4 - With Outlook closed - open the Control Panel <Mail> app
#5 - Select the <Email Account> --> select <Change>
