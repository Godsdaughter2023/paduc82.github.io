talking moose say:
If you open any Office for Mac application and select "About..." from the application menu then what version do you see installed there? To install Office 2008 SP1 your version can be no higher than 12.0.x.

If everything checks out then you can try installing SP1 using the Terminal application found in /Applications/Utilities. Enter this command while logged in as an administrator on your Mac:

sudo installer -pkg <drag the installer here> -target /

You'll drag the installer into the Terminal window where its full path will be automatically entered for you. Don't worry about any odd characters that you see.

Hit the return key and enter your administrator password to install.

Alternately, you can try installing in Safe Mode. Restarting your computer while holding the Shift key down. Your machine will take longer to start up but once it's in Safe Mode you should be able to install. Restart normally to get out of Safe Mode.
