suzanne s. barnhill say:

With the insertion point in Chapter 2, on the Insert tab (or the contextual Header & Footer Tools | Design tab, if you're in the header or footer), in the Header & Footer group, click the Page Number button and choose Format Page Numbers... from the dropdown menu.

In the Page Number Format dialog, select "Continue from previous section."

When you restarted numbering at 1 in Chapter 1, you enabled this setting for subsequent sections. If you have only one additional section (Chapter 2), then you can "nip it in the bud." Otherwise, you will have to visit each section and use the Page Number Format dialog to change the numbering to continuous. This will not affect the numbering of your footnotes.
