jay freedman say:

To automate deleting empty rows from each table in the document, install this macro and assign a keybaord shortcut or Quick Access Toolbar button to run it.

Sub DeleteEmptyRows()
    Dim tbl As Table
    Dim rw As Row
    Dim bEmpty As Boolean
    Dim cel As Cell

    For Each tbl In ActiveDocument.Tables
        With tbl
            For Each rw In .Rows
                bEmpty = True
                For Each cel In rw.Cells
                    If cel.Range.Characters.Count > 1 Then
                        bEmpty = False
                        Exit For
                    End If
                Next cel
                If bEmpty = True Then rw.Delete
            Next rw
        End With
    Next tbl
End Sub

ray gao msft say:

Hi FSpray,

 

In OneDrive for Business, we use different permission levels to restrict users’ permissions to the items. For your requirement, I would suggest you create a new permission level and share the documents using this permission level. You can follow the steps below to achieve that.

  1. Navigate to OneDrive for Business and click the Gear icon.
  2. Select Site settings > site permissions.
  3. Click Permission levels.
  4. Select Read permission level, scroll down and click Copy Permission Level.
  5. Name the copied permission level like View Only.
  6. Untick the Open Items permission. It will also untick the View Versions permission.
  7. Click Create. Then you can share documents with the custom permission level.

 

Note: To share documents with this custom permission level, we need to perform more steps than the normal ones. Select the file > click Share > click “…” and select Manage Access > click Advanced > click Stop Inheriting Permissions > select Grant Permissions > click Show Options in the Invite people window > select the custom permission level and then Share.

hansv mvp say:

You could use this:

Private Sub Workbook_Open()
    Dim wsh As Worksheet
    Dim rng As Range
    For Each wsh In Me.Worksheets
        For Each rng In wsh.UsedRange
            On Error Resume Next
            With rng.Validation
                If .Type = xlValidateList Then
                    .Modify Formula1:=.Formula1
                End If
            End With
        Next rng
    Next wsh
End Sub
