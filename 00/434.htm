You can create a formatted AutoCorrect entry by selecting text (or really anything: text boxes, tables, and so on), and then clicking Office button | Word Options | Proofing | AutoCorrect Options; on the AutoCorrect tab, you'll see a preview, and "Formatted text" should be selected.     How do I attach a document or image icon to a form in Word for Mac - Microsoft Community

    ryannebauer say:

    How do I attach a document or image icon to a form in Word for Mac 2011

    I have created a form in Word for Mac 2011 that has protected fields, as seen here:

    As you can see in the bottom cells of the above table, I would like the person filling out the form to be able to embed a PDF or JPG into the form. Ideally, this embedded file would appear as an icon that, when clicked on, opens up the relevant JPG or PDF. However I have not found a way to do this. Please advise on whether this is possible or on alternative method that fulfills this requirement.

    with thanks, Mark

    john mcghie say:

    I can't think of a way of doing that in Word 2011 because its support for HTML is not strong enough.

    What you really need is an embedded "click here to select and upload a file" widget. That needs to be written in HTML and Javascript (or in VBA).  There's quite a LOT of coding involved. Word 2011 will not support the code required.

    Word 2016 will, but you would need a professional developer to write it.

    I think the simplest way around your dilemma is just to ask the users to "Insert the picture here".  To do that, you will need to unprotect that space in the form.  To do that, add a "Continuous" type section-break above the row where you want the pictures.

    Then when you go to "Protect the document for Form Fields" you will see it now has TWO sections, and you have the option to protect only Section 1.  The form fields will work in Section 1; in Section 2 the users will be able to freely insert or type, so they can insert the pictures.

    I also suggest that you ask for "PDF, PNG or GIF" files for the pictures: JPEG is not a good choice for diagrams: it gets very blurry. JPEG removes sharpness in its compression to preserve accurate colours; PNG and GIF remove colours to preserve detail.  With that kind of diagram in black-and-white, you can afford to loose the colour, but you want to keep the detail.

    Hope this helps
