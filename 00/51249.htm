You can't import a project file from Adobe Premiere Elements because that's not an actual movie. But if you export the results from Premiere Elements into a compatible video file type, you could then use that resulting movie in Movie Maker.

The problem is, the next time I reconnect the drive to get some music, I open media player and the library is gone. I make sure that the drive always connects with the same drive letter so that can't be the problem. msesp say:

WMP12 does not store any music files. It only stores the location of the files. If you are using a temporary drive, the files go away when the source goes away. The only solution to this is to put the files on your hard drive or on a drive that is connected all the time.

To store your files on your hard drive in a location that the player automatically reads, go to the icon of <YOUR NAME> (that is: C:\users\<your name> ) when that folder opens, there is a folder named My Music. This is one of the folders that WMP reads.

If you do not have room on your hard drive to store all of the music, you can choose some favorites or rotate the music on your hard drive. You can just let it repopulate WMP each time you connect the drive.
