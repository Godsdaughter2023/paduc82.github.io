Answer:  they all are.

As I noted later, the R2 value calculated in E17 is the same as the value labeled "R Square" in the Data Analysis Regression results.

The value labeled "Multiple R" is simply the square root of "R Square".

A Google search for "adjusted rsq" turned up this formula and explanation from "Excel 2007 for Scientists and Engineers" by Gerard Verschuuren [1]:

RSQadj = 1 - (1-RSQ) * (n-1) / (n-(df+1))



And indeed, the following formula results in about -0.217576, the same value labeled "Adjusted R Square" in the Data Analysis Regression results.

=1 - (1-E17) * (COUNT(A1:A9)-1) / (COUNT(A1:A9)-COLUMNS(A1:C1))

Your document contains 119 paragraphs with line spacing set to 0. Word won't create paragraphs with that line spacing, so some other application was used for the original content's creation. A macro can be used to delete the offending paragraphs:

Sub Demo()
Application.ScreenUpdating = False
Dim Para As Paragraph
With ActiveDocument
  For Each Para In .Paragraphs
    With Para
      If .LineSpacing < 0.1 Then .Range.Delete
    End With
  Next
End With
Application.ScreenUpdating = True
End Sub

christopher waldron say:

MS ACCESS 2013 AddIns

I have downloaded and installed Access 2013 from Microsoft on Windows7.

When I try to install AddIns, none are available. How can I locate/install the Switchboard Manager, Linked Table Manager?

hansv mvp say:

The Linked Table Manager should be on the External Data tab of the ribbon. It will be greyed out if your database does not contain linked tables.

The Switchboard Manager is not displayed on the ribbon by default. To add it to the Quick Access Toolbar (QAT):

  • Click the dropdown arrow on the right hand side of the QAT.
  • Select More Commands... from the dropdown menu.
  • Select 'Commands Not in the Ribbon' from the 'Choose commands from' dropdown.
  • Scroll down the list of commands until you see 'Switchboard Manager'.
  • Select it and click Add >>.
  • Click OK.
  • You now have Switchboard Manager on the QAT.
