a. user say:

Hi,

Thank you for posting your query in Microsoft Community Forum.

With Windows 8, Microsoft does not provide Certificate of Authenticity (COA) stickers that have the product key that the user has to type in when installing the operating system. The product keys now come embedded in the BIOS. Thus, by eliminating the sticker prevents it from getting damaged while at the same time eliminating the long and irritating process of typing in various letters and numbers when installing the operating system.

If you have to reinstall the operating system on a machine that came with Windows 8 preinstalled, the installation process automatically grabs the software product key from the motherboard BIOS with no external input. As your Windows 8 came preinstalled on your system, it should automatically be reactivated.

As you like to create an installation media. You can create installation media using link given below and you would get the option to skip the product key at the time of creating the installation media.


How to create installation media for a PC refresh or reset:

http://windows.microsoft.com/en-US/windows-8/create-reset-refresh-media

Once the installation media is created your can install the windows on your system and perform phone activation.

 You may try the following steps to check.

a. Press Windows key + X then click Run, then type: slui.exe 4

b. Next press the 'ENTER' key

c. Select your 'Country' from the list.

d. Choose the 'Phone Activation' option.

e. Stay on the phone (do not select/press any options) and wait for a person to help you with activation.

f. Explain your problem clearly to the support person.

Hope this information is helpful. Do let us know if you need any further assistance, we'll be glad to assist you.

a. user say:

How could it have worked before and not work now - what has changed?

With this incident are you backing up files or creating a system image?

There is a known problem with the Windows backup program and larger drives that use 4K logical sector sizes - and there is no workaround or fix other than to reformat your external drive or partition it into smaller chunks (volumes).

There is an explanation in the Western Digital community:

 "This problem is caused by an issue with the Windows 7 Backup and Restore utility when using drives with 4K Logical sector sizes, which are found on WD 2.5 TB and larger external drives. This is different from an "Advanced Format" (AFD) drive which use 4K physical sector sizes. AFD drives should not cause this problem."
