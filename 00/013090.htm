maroguy69 say:

Ok, so I finally figured out how to perform this task and wanted to post it just in case someone else is needing it. 

"=(networkdays(h2,i2)-1)*9+(mod(i2,1)-mod(h2,1))*24>9"

This formula allows for a 24hour time frame and excludes weekends from the equation.

The N8 is extremely flexible when it comes to using it as a modem.

 

You can connect it to Ovi Suite with the N8's USB connection in 'Nokia Ovi Suite' mode and then use the Ovi Suite software to connect the PC to the internet via your 3G provider.

 

Alternatively you can put the N8's USB into 'Conn. PC to net' mode and then the phone appears to talk directly with Windows to arrange a connection that does not require Ovi Suite to be running. This is the mode which I use most of the time and it works extremely well. I like it because it does not require any software to be preloaded on the PC you're trying to connect to the Net.

 

You can also tether the phone with the Windows OS directly using Bluetooth and operate it as a Bluetooth modem.

 

You can create a Bluetooth connection with Ovi Suite and have the software sort it out in the same way it does with the USB cable. That works nicely and mean you don't need to have your cable handy.

 

You can run the stupidly named but very impressive JoikuSpot software that connects to your 3G Internet and creates a Wifi hotspot that other devices can use. Be aware that the free version of this software does not feature any security. Within moments of creating one of these Wifi hotspots in my hotel room I had some hideous Apple devices leaching off it. I think I might buy the full version because it's really very good.

 

So there you are. Many options. I doubt any other device can better them. I trust that you'll find one that meet your needs.

 I wanted to type the phrase "Okay, let's do this!", but instead, typed this by mistake:

Okay, let's this!

Okay, let's | this!

Then, I type a Ctrl-U.  The sentence immediately changes to:

Okay, let's | this!

I think, "Why did it underline word let's??"  I type Ctrl-U again, which changes it back to:

Okay, let's | this!

Okay.  I think, "Maybe I'm supposed to type the word, then hit Ctrl-U."  So I type the word "do":

Okay, let's do| this!

Okay, let's do| this!

I can type a space, then Ctrl-U, which gives me 

Okay, let's do | this!

Grrr... I want to underline just the word "do"!

Okay, let's this!

Perhaps the problem is that I started at the end of the word "let's", instead of at the beginning of the word "this".  So this time, I put my cursor just before the word "this":

Okay, let's |this!

I type Ctrl-U, then "do":

Okay, let's do|this!

Great!  My underlined word is there, and the word before it is not underlined.  So far, so good.

I then type Ctrl-U to turn off underlining so I can type a non-underlined space.  However, when I type Ctrl-U, I get:

Okay, let's do|this!

What the what??  This removed the underline from "do"!  I hit Ctrl-U again, which does this:

Okay, let's do|this!

ARGHH!! I don't want to underline both words!  So I start over:

Okay, let's |this!

I type Ctrl-U, then "do", like before:

Okay, let's do|this!

This time,  I type a space before I type Ctrl-U again to turn off underlining:

Okay, let's do |this!

However, this leaves an underlined space after the word "do", which I don't want.  

If I backspace the underlined space to get rid of it, then hit Ctrl-U in preparation to type a non-underlined space, it removes all of the underline again.  So instead, I type Ctrl-U, and type another space:

Okay, let's do  |this!

I then use the arrow key to move the cursor back one:

Okay, let's do | this!

And delete the underlined space:

Okay, let's do| this!

At last, I figured out how to insert an underlined word:
1. Start at the beginning of the next word
2. Hit Ctrl-U
3. Type the word, then a space, then Ctrl-U, then another space.
4. Use the arrow keys to back up one space, then hit the delete key to delete the unwanted underlined space.
5. Use the arrow keys to move forward one space.  
6. Now, you can insert any other non-underlined words that you wanted to insert there.

This is an unnecessarily difficult way to do what I want to do.  I should be able to simply 
1. Put the cursor where I want to insert stuff
2. Hit Ctrl-U
3. Type stuff I want to be underlined
4. Hit Ctrl-U again
5. Type stuff that I don't want to be underlined
