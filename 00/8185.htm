mikek703 say:

If you insert a "new drawing canvas" first (bottom of shapes menu) and insert your boxes and line in it, you will be able to do everything you mentioned.

ibby_ say:

Hi,

I had the exact same problem.

I found that it's easy to add shapes 'off' the canvas if you don't select the canvas before adding a shape. If you accidentally add them 'off' the canvas, then the Connectors don't work.

To make sure they are on the canvas, you can select all added shapes, Cut, then select the Canvas, then Paste.

If you use the Selection Pane (one of Office's best unknown features) you can see what is going on and if the shapes are on the canvas or not.

Then the connectors will work.
