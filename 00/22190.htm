stefan blom say:
Times New Roman should be included with the operating system. All you have to do is change the default font in Word, which you can do as follows: In a new, blank document (Ctrl+N), press Ctrl+Alt+Shift+S to display the Styles pane. Click the Manage Styles icon and go to the Set Defaults tab. Pick Times New Roman as your new font and change any other settings if you wish. Select "New documents based on this template" and click OK. Later, if you are prompted to save the Normal template, choose to do so.

suzanne s. barnhill say:

To just change the default font, there's an even easier way.

  1. Right-click in any paragraph and choose Font...
  2. In the Font dialog, select the desired font and click Set As Default in the bottom left corner of the dialog.
  3. In the ensuing dialog, select the radio button for "All documents based on the Normal.dotm template" and click OK.

Firstly Nokia Maps currently utilizes an active connection for two purposes, to allow use of Assisted GPS feature which reduces time to establish position and to download greater map detail if a "search" is made. Please note that application will function without connection albeit in a reduced capacity. It will not work inside building very well unless Assisted GPS feature is utilized.

Secondly if you don't have an active connection you need to have downloaded regional maps to your microSD unless they were included in the package (UK SIM free versions have this) using Nokia Maploader application on your PC.
