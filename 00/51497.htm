An existing primary parent account in Family Safety cannot be added to another parent account. If you want to have your wife's account as a secondary account, you must first remove her as a primary parent account in Family Safety:

how do i print a list of music in my library? - Microsoft Community

lils say:

how do i print a list of music in my library?

I have a new computer with window vista.  Using window media library version 11.  How can a print a paper copy of all the songs in my library?
robyn-- say:  

Hi lils,

Thank you for your post on the Microsoft Answers Forums.

Here's one way, but it's somewhat complicated...

Microsoft makes a Windows Media Player plug-in called Media Info Exporter that lets you export artist/album names and all the other song info into a file on your computer.  You can then format and print that file as you like.  The Exporter was designed for WMP 9, but it does work with WMP 11.

Step 1 - Download and install Media Info Exporter

1.        Download the plug-in file, mpexp.dll, directly from here.  Save it to a folder on your hard drive that you can remember, like C:\example\.

2.        Go to Start -> Run (or push Windows+R on your keyboard).  Type in:

regsvr32 C:\example\mpexp.dll

You should get a message saying "DllRegisterServer in C:\example\mpexp.dll succeeded" or something similar.  The plug-in should now be available in WMP.

Step 2 - Use the Exporter to export media info

1.        Open Windows Media Player.  A small Media Info Exporter window should already be open.  If not, you can open it manually:  From the Tools menu, go to Plug-ins and choose the Media Info Exporter.  If you don't see the Tools menu at first, just right-click somewhere on the dark blue bar nears the top of your WMP window.

2.        Click Properties to choose where the file will be saved and which application to load it into.  The file will be formatted differently depending on which application you choose, but Excel is the best option if you want to clean it up to show just the album/artist names (see below).

3.        Click Export.   The file will be exported, but it'll contain a lot more information than just the artist and album names.  The next step shows how to clean it up.

Step 3 (Optional) - Clean up the exported list in Excel

1.        Delete all the columns except for Artist and Album (columns B and C by default).

2.        Select the two remaining columns.

3.        From the Data tab or menu, choose the Remove Duplicates function.

4.        Check the My data has headers checkbox and then click OK.

5.        That's all.  Now you should have a clean list of albums along with their artists' names.  You can print this list directly or export it to Word and format it further to make it prettier.

After following the steps listed above please post back to let us know if this helped to answer your question or not so that we may further assist you if need be.
