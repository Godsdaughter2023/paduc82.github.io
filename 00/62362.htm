Hello,

 

Thank you for posting your query on Microsoft community. I understand the inconvenience caused to you. Let me help you.

 

 

  1. Since when this started happening?
  2. Is your PC up to date with Windows Updates? Press Windows logo key + R to open run command, type Winver and hit Enter key. Now check the version and build.

I Suggest you to execute below mentioned troubleshooting steps and check if they help you to fix the issue.

Method 1:

Try below mentioned troubleshooting steps to reset TCP/IP protocol to see if it sort out the issue.

a) Press Windows key + X, select Command Prompt(Admin)

b) Type the following command into it and press Enter after:

netsh int ip reset

When you run this command, it'll rewrite your computer's registry keys use for TCP/IP settings. This achieves the same result as removing and reinstalling the protocol.

Exit the command prompt and restart your computer to check if problem still persist.

Method 2:

If problem still persist, then update the Network drivers from Device Manager manually and check if this helps.

a) Open run command by pressing Windows key + R and type “devmgmt.msc” (without quotes) and hit Enter.

b) Spot and expand the “Network Adapters”.

c). Right click on the network driver and select “Update Driver Software”. When it prompts to choose an option How do you want to search driver, then go with Search automatically for updated driver software. If you have a driver available on your PC, then select Browse my computer for driver software, then select the appropriate location, where the driver located on your PC for updating it. (Ensure that PC is connected to Internet while performing this step)

d). Once this is done, restart the computer to check the problem fixed. If it doesn't fix the issue, then Right click on the same driver and select “Uninstall” then restart the system and let the system install the driver automatically on Start. Check and see if it resolve the issue. (Recommended step)
