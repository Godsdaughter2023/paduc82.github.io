Word is a word processor and not a typewriter. It is style driven whether you like it or not and there is nothing you can do to avoid styles.

The default style is the Normal style and the default formatting of that style has space included as part of its definition, for the simple reason that paragraphs with space between them are easier to read, and as it is not a typewriter you should not press the enter key twice to generate that space but include it in the style you are using.

You can modify the default style to remove the space. Right click the Normal style in the Quick Styles group and select 'modify' Make whatever changes you wish and before leaving the dialog, check the 'New documents based on this template' check box. Provided the normal template is saved when you quit Word, this will become the default for documents created from the normal template.

OR

Apply another style (Word has another style in its default set called 'No Spacing') easily accessed from the Home tab of the ribbon.

OR

You could change the style set (from the Home tab) to Word 2003, which will give you a Normal style without added space after. There are other style sets available which offer different types of formatting.

Hi,
try this code...
export data in sheet1 tab (no matter the name)

Sub Merge_CSV_Files()

'Feb 25, 2014

Const myDelim As String = ","   '<< select comma or semicolon
ActiveWorkbook.Sheets(1).Cells.ClearContents
Dim myPath
Dim myFile
myPath = "C:\Documents and Settings\david goodhand\Desktop\Reg of electors\"
myFile = Dir(myPath & "*.csv")
Dim r As Long, c As Long
r = 1

Application.ScreenUpdating = False

Do While myFile <> ""
Open myPath & myFile For Input As # 1
Do While Not EOF(1)
Input # 1, line$
c = 0
For c = 0 To UBound(Split(line$, myDelim))
Sheets(1).Cells(r, c + 1).Value = Split(line$, myDelim)(c)
Next
r = r + 1
Loop
Close # 1
myFile = Dir()
Loop
With ActiveWorkbook
.Sheets(1).UsedRange.EntireColumn.AutoFit
.Save
End With
Application.ScreenUpdating = True
End Sub

jeeped say:
Go to the Bing Search 'landing page' and before starting a search, click the Preferences icon at the top-right of your screen that looks like a small gear (e.g. ). Make sure that the Display Language is set to English and click the blue Save button at the bottom.

              

Once you are back at the landing page, continue with your search. If the results are still coming up in Spanish, immediately go back to Preferences and check that the display language has not altered. If it has, reset it to English and click Save. Browsers rely greatly upon a local cache of information for many settings. If this behavior of switching regional languages continues, clear your browser's cache. The three main browsers all use Ctrl+Shift+Del to initiate this with some minor variations after the command gets started.

If you are using a Bing profile, make sure you are logged in (profile name in upper-right) as a default log-in and a profile log-in can potentially hold two different settings.
