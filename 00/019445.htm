cjlim say: The most common causes of caller ID problems are:

- duplicate entries in your contacts - you appear to have checked this, however I wonder if your check included searching for the same number in different formats? Try searching for a PART of the number as if for example you search for +44 123 456 7890, it won't find +441234567890, whereas if you search for 7890 it would.
Because your phone is not even listing the callers name, I think it is a caller ID issue.

The most common issues with caller ID are listed below, courtesy of patc

Having different formats is a problem for caller ID.

- using your international dial code instead of "+", eg: using 00441234567890 (wrong) instead or +441234567890 (right).

- spaces in numbers (some phones, some firware versions only), eg: +44 123 456 7890 (wrong) instead or +441234567890 (right).

- MS Outlook putting "(0)" in number - if you define a country for a contact in MS Outlook it will automatically add a country code and insert a "(0)" in the number, eg: +441234567890 becomes +44 (0) 1234567890. You need to remove the "(0)"s

- there has been one report of 9 and 10 digit numbers causing problems, one doesn't work with SMS the other with phone calls. I forget which way around it was ..
