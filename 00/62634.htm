raymond p. say:

Hi CharlesLeven,

 

Thank you for posting your concern in Microsoft community. We understand that when you are trying to Send Link by E-mail, your unable to choose contacts as it does not populate in the To field section. First off, we would like to know the default mail client in your computer. Is it Windows live mail or Outlook Office? If it’s a Windows live mail, kindly make sure that you are signed or logged in before doing the said procedure as it will not populate the contact field.

 

If Windows live mail is not your default client, here's how to set it up to your computer:

 

For Windows Vista and Windows 7:

 

1. Open Default Programs by clicking the Start button, and then clicking Default Programs.

2. Click Set your default programs.

3. Under Programs, click the e mail program you'd like to use in this case Windows Live Mail, and then click Set this program as default.

4. Click OK.

 

For XP:

 

1. Log on as Administrator

2. Open Internet Explorer, click the Tools menu, select Internet Options. Click Programs tab and select your email client from the drop-down list. Once it selected, click OK/Apply button.

 

Note: The first time you start them, some e mail programs display a message asking you if you want to use that program as the default. If you choose to do so, this program will become your default e mail program; even if you've chosen a different program using Set your default programs earlier.

 

Please let us know if you have more questions.            

 

Thanks.
