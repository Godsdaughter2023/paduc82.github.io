hansv mvp say:

Let's say the first table is in A1:E7 on Sheet1.

On another sheet, enter Yes, No and N/A in B1:D1, and enter Question 1, Question 2, etc. in A2 and down.

Enter the following formula in B2:

=SUMPRODUCT((Sheet1!$A$2:$E$7=B$1)*(Sheet1!$A$1:$E$1=$A2))

Adjust the sheet name and ranges $A$2:$E$7 and $A$1:$E$1 as needed, then fill to the right and down (or vice versa)

jessen p say:  Hello,
 
Were there any changes( hardware or software) made to the computer prior to the issue?

DNS server not responding Issue.
Perform steps mentioned below and see if the issue occurs.
a :Click Start. Type cmd in the Start Search box, and then press ENTER. 
b: At the command prompt, type the following command, and then press ENTER:
ipconfig /flushdns (there is a space between ipconfig and / )
It will show the message successfully flushed the DNS resolver cache.
c: At the command prompt, type the following command, and then press ENTER:
ipconfig /registerdns (there is a space between ipconfig and / )
d:At the command prompt, type the following command, and then press ENTER:
ipconfig /all (there is a space between ipconfig and /)
 
The ipconfig /all command displays Windows TCP/IP settings for all your network adapters.
If your IP address for your Local Area Connection is either 0.0.0.0 or 169.x.x.x (where x is any number), then your computer is not getting an IP address from the router. 

If that is the case, try this:
•         ipconfig /release in the command prompt and press ENTER
•         ipconfig /renew in the command prompt and press ENTER

Test it again by typing ipconfig /all to see what the address is?  (If you are on a home router, it should start with either 192.168.1.x or 192.168.0.x or 10.x.x.x)

petrib say: A frequently asked question, it seems, but unfortunately no good answers.

On S60 based devices like the 5500, there is a setting in Contacts to choose whether to list contacts with the last name first or the first name first.

This setting is only a feature/function of the Contacts application, and has no effect whatsoever outside the Contacts app (not in Messaging, not in Log, not when showing who's calling, etc.).

Until Nokia changes the other S60 apps to either "honor" this setting in Contacts, or moves the setting to the "global" Settings app (and then changes the other apps to use it), you will see no fix for this.

What it also means, I'm afraid, is that if Nokia ever fixes this, the fix will not help users of existing device models, but only those who get a new device with this feature.
