lemp say:
Windows XP does not normally have shutdown scripts.  If you have one, someone installed it.

Open the Group Policy editor (Start > Run > gpedit.msc > OK)
In the left pane, Computer Configuration > Windows Settings > Scripts (Startup and Shutdown)
In the right pane, right click "Shutdown" and select Properties

In the window that opens, you'll see the list of shutdown scripts.  Although you can simply "Remove' the scripts, you might first want to "Edit" them and determine what they are attempting to do. You can then either fix the script so that it doesn't cause the delay or delete it entirely.

If the "cleaner program" you're using daily is a registry cleaner, you're asking for trouble -- and this may be the reason that your shutdown scripts are taking so long.
