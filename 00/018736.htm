Picture Scale Height? - Microsoft Community

david-e say:

Picture Scale Height?

What is the function of the Scale Height and Scale Width settings in the Format Picture?

john korchok say:

Scale Height and Width look at the resolution setting of the bitmap file and displays how large the picture the picture is relative to it's original size. Lots of designers think in proportions and percentages rather than absolute dimensions. So when they're making revisions, they'll spec "make that photo 80% and that one 60%."

Start with a photo that is 500 pixels wide. Save one version that is 100 pixels per inch, save another that is 250 pixels per inch. Import then to PowerPoint. The 250 d.p.i file will come in at 2 inches wide while the 100 d.p.i. edition is 5 inches wide. Both will read 100% in Scale Height and Width.

BTW, an easy way to get photos exactly aligned slide to slide is to create a custom layout with a Picture placeholders. A Picture placeholder will scale the photo to fit the smallest dimension, then crop the larger dimension. It's a good tool when you want a consistent arrangement of photos on a page. By contrast, a Content placeholder fits the large dimension to the placeholder, then shrinks the placeholder to fit the smaller dimension. You see all the image, but their arrangement on the slide gets haphazard.

smbzq say:

I have been struggling with this problem, too, and I think I've finally figured out the solution!

It turns out that the select text feature in MS Word 2011 for Mac uses the LEFT mouse button. In order to use it this way, you need to make sure that your system preferences for the mouse have "Secondary Click" unselected. When your mouse's secondary click function is turned off, you can then use the left button to select text in word.

This totally resolved the problem for me.

It's not a solution I like because I like using my secondary click feature. But it *is* a solution.

Let me know if this works for you!

Best of luck :)

If you merely want to display a message:

 

Sub ClearCellsMon()
    MsgBox "Click OK to clear a bunch of cells in the Creator sheet.", vbInformation
    Application.ScreenUpdating = False ' makes things much faster, no flickering
    With ThisWorkbook.Worksheets("Creator")
        ...

 

If you want to give the user a chance to cancel:

 

Sub ClearCellsMon()
    If MsgBox("Click OK to clear a bunch of cells in the Creator sheet", _

        ...
