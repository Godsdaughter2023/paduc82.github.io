a. user say:

Hi,

 

I have replicated your issue and I just found out that in order for you to type in your desired password on Windows Live Mail, you need to tick on Remember password first before it will allow you to type. Now one you type your password, you can untick the box again if you choose not to allow Windows Live Mail to remember it. For detailed instructions on how to do it, please refer to the steps below:

 

1. Open Windows Live Mail.

2. Click on Accounts tab then select Properties. A new dialog box will appear.

3. Click on Server tab. Under Mail server, type in your email address on the box field beside log in name.

4. As for the password, you need to click on the box beside Remember password first. Then type the password on the box field. If you're already done, you can untick the box if you want Windows Live Mail not to remember your password.

5. Click on OK.
