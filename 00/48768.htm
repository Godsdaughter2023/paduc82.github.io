that installed them will look for that particular version.

While I would probably uninstall all the extra versions if I had confirmation that the  two

experimenting with uninstalling and reinstalling again should a software I don't use often  fail to load,

btw - Just to get the terminology right. Software gets uninstalled. Files get deleted.

While software can be deleted, doing so can cause problems both with removing the title from

Programs and Features, leaving folders behind (which can happen when you uninstall anyway),

and leaving registry entries.


- If your laptop HDD/SSD is running out of space you will most likely have to move files to an external HDD, or get a larger HDD.

Music, Video are common space hogs followed by photos. Most software does not take much space,

but uninstalling the 'bloatware' that comes with many laptops can help some.

If you have games, they can use a lot of space.


Note also that an SSD will last longer if you leave 25% free space. HDD's require 15% free space for

defragmenting. If you have an HDD and have not defraged for a long time you could regain some

space and increase performance by defraging (which can take a long time if not done regularly).
