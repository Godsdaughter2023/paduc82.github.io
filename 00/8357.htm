Hi ToddLeiness,

On Windows devices, we can store a notebook in local drive, consumer OneDrive and OneDrive for Business. On Mac, we can only store the notebook in OneDrive and OneDrive for Business. My suggestion is to store a notebook in OneDrive or OneDrive for Business. So, users can access it anywhere has internet connection. And when user change to another computer, he just needs to sign in OneNote with his account and open the notebook stored in this account (actually stored in OneDrive or OneDrive for Business related to this account).

For the notebooks stored in OneDrive/OneDrive for Business, as you mentioned, they are differently from other Office documents. We open it with OneNote desktop app, and then it will start to sync the notebook. You will get error message when there are some sync issues. We can check notebook cloud version in OneNote Online to make sure all changes are synced to the cloud. After we open a notebook from cloud to OneNote desktop app, we don't have to close the notebook, just close and open the app. When OneNote is open, it will sync changes to cloud when there is internet connection.

About the error message "a system is already logged into the account", it seems that you are using one tenant account to sign into an Office app which has already signed in with an account from the same tenant.
