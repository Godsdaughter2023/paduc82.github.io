jmh3143 say:
If you are using a lower version of Internet Explorer when accessing your account, by default you must click the "Spell Check" link for it to function.

Outlook.com has a Manual Spell check button on the top next to Options.
It does not appear unless you are writing an email.




If you are using Internet Explorer 9 browser, or any updated browser, the Spell Check feature will not be found  because it is already integrated within this browser's system.
Meaning, once you have misspelled certain words, it will automatically be detected by the system. Notice that when you have a typo, you'll see it right away because it will appear with a red underline in it.
Right click the error to correct it.


Although Outlook.com is new most of the Hotmail functionality is still there, just in different places.
The first thing you need to do is to look around a bit.
Check all the menu pull downs, see what is where.
