Use Ctrl+' followed by a vowel to get an accent ' on the vowel: Ctrl+' e gives é in Word and Outlook.

Similarly, Ctrl+" followed by a vowel produces an umlaut, e.g. Ctrl+" a produces ä.

And Ctrl+` followed by a vowel produces a grave accent: Crtl+` a produces à

Changing Gridlines -Word 2010 - Microsoft Community

kb53 say:

Changing Gridlines -Word 2010

Please help. I can show and hide gridlines in a word document but I can't find a way to change the size /spacing of the gridlines as with previous versions. Does anyone know how to change the size of the spaces in the grid? Thanks
suzanne s. barnhill say:

What you're looking to change is the "grid interval." To access this setting, go to Page Layout | Arrange | Align | Grid Settings.

If you happen to have a drawing object selected (so that the contextual Drawing Tools | Format tab is displayed), you can also find the Grid Settings item under Align in the Arrange group.

For users looking for a way to turn the grid on and off, go to View | Show | Gridlines. For users who have Word 2010 Starter (which doesn't have a View tab), the toggle is at Page Layout | Arrange | Align | View Gridlines.

graham mayor say:

I don't know of any off the shelf add-in that will do this (which is not to say that there will not be one), but what you ask could be created as a macro in Outlook VBA .

With a list of 200+ keywords and you want to check for multiple occurrences in the body of the messages - even more so if the message is a reply to a message that includes the previous correspondence, there is likely to be a lot of duplication of e-mail messages across the folders (both Outlook and disc), but that part is relatively simple to achieve, using a list of keywords stored either in a text file (or better still an Excel worksheet).

What is less straightforward is the attachments. In order to process the attachments they must be saved to the hard drive, then processed using the appropriate application, which must be amenable to VBA processing, so it is essential to know what the attachments are that you want to save.

Opening all the attachments in a variety of applications in order to check them against your list in order to determine whether to keep the attachments on file is going to slow down the receipt of e-mail quite markedly.
