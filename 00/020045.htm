Highlighting Cells dependent on Checkbox - Microsoft Community

adamkerr say:

Highlighting Cells dependent on Checkbox

Hi, 

I have a questionnaire with Yes and No cells, each with an independent checkbox. I would like the row to be highlighted if either yes is checked, or neither is checked. Is there a way that this can be done?

Example:

A1 contains checkbox

A2 contains checkbox

If A1+B2 = "False" highlight Row1 red, or if A1 = "True" highlight Red. 

Any help would be amazing. 

Thank you. 

Adam

ossiemac say:

Are the CheckBoxes linked to the underlying cells? If not, then they need to be so that you can Use Conditional formatting to color the cells.

Note that the underslying cells will not contain a True of False until after the first time you click the Checkbox and make a change so you might need to go through all checkboxes and check and then uncheck them to initialize the underlying cells.

The following guidelines assumes that the first checkboxes are in cells A1 and B1. If they are actually in say A2 and B2 then change the formulas accordingly. The formulas are entered as if they apply to the first cells of the selected area for the formatting and then Excel applies the formulas to the remaining cells. Note that I have a $ sign in front of the column Id's but not in front of the row numbers.

  1. Select the area that you want to be colored.
  2. Select Conditional formatting (On Home Ribbon)
  3. Select Manage rules.
  4. Click New rule button.
  5. Select "Use a formula to determine which cells to format".
  6. In the field under "Format values where this formula is true" enter the following formula.
  7. =AND($A1=FALSE,$B1=FALSE)
  8. Click the Format button.
  9. Select the Fill tab.
  10. Select the required color (Red) and click OK and OK again.
  11. Click New rule button again.
  12. Select "Use a formula to determine which cells to format".
  13. In the field under "Format values where this formula is true" enter the following formula.
  14. =OR($A1=TRUE,$B1=TRUE)
  15. Click the Format button.
  16. Select the Fill tab.
  17. Select the required color (Green) and click OK and OK again.
  18. Click Apply and OK to finish.

If you have to go back to the conditional format and Manage to edit then when editing the formulas in the field it is necessary to click in the field and then press F2 to get edit mode otherwise if you use the arrow keys then Excel thinks you are selecting cell ranges and inserts them in the formula.

If you do not want the underlying cells to display the True or False then when you are setting the Conditional format, select the Fill color and then change the tab to the Font and set the font the same color and while the True and False will be in the cells, it will not be visible.

suzanne s. barnhill say:

You can create a custom dictionary containing the words you want to include. Name it something different from Custom.dic (which is the default user dictionary; you don't want to mess that up) but with the .dic extension. Users can then add this to their custom dictionaries (you can have as many as you want); Custom.dic would still be their default user dictionary (to which their own words would be added automatically), but your corporate one would also be enabled and used.

Find instructions for creating and sharing a custom dictionary at http://support.microsoft.com/kb/322198.

Hi

Adding a phone number on an existing Skype contact is only available for the classic version of Skype on Mac (version 7.59). Currently, this version is not offered on the new Skype on Desktop, Mac and Linux (version 8.11.0.0).

If you are using the classic Skype for Mac, you can follow the steps below on how to add a phone number to a contact's profile.

  • Sign in to Skype.
  • In the sidebar, click Skype to display your full Contact list and find the person you wish to add a number to.
  • Place your cursor over the person’s picture to see the green call button and click the down arrow.
  • Select Add Contact Number.
  • Select the type of number, country code and type in their number in the box.
  • Click Add and the number is saved.

You can edit the number you have entered anytime. Please refer to this FAQ for additional details. How do I add phone numbers to a contact’s profile in Skype for Mac OS X?
