If you are logged in with a temporary profile it's because your regular user profile has become unusable.

A temporary profile is just that - it only lasts for the current session. As soon as you sign out or shut down Windows, everything in the account is lost.

If you can determine how your regular user account became unusable, you can try to fix the problem(s). Otherwise, the easiest thing to do is to create a new user account. Once created, you can move your user data (documents, pictures, email, bookmarks, etc.) from the old user account to the new user account.

In Windows 10, syncing means that all your devices that run Windows 10 look and work the same. You can do that, if you want, by signing in to a Microsoft account on all your devices. But that's not required to use Windows 10.

Thank you for your post and giving us an opportunity to assist you.

A user logged into a standard account will be allowed to install programs that do not affect other user accounts.
 
With the Firefox example, if the password was entered instead of clicking the X, the program would've installed in the Program Files folder. Since you did not enter the password, it would be installed in a local folder that is accessible only from the standard account.
