- The two Nvidia (Nyvidia) relate to your graphics card drivers. The Backend relates to Geforce Experience,

and the Capture server Proxy, AKA shadowplay, is also a part of the 'Experience' and relates to making videos

of your gaming exploits.

You can do without either, but if you use them they probably should be in Startup.

You can uninstall the Geforce Experience via Programs and Features, but you will need to do graphics settings

that are not available in your games manually via Nvidia Control Panel

(my personal preference as the 'Experience' does occasionally cause problems, but also because I'm a control

freak and like doing things myself ;) ).

For the most part modern games don't need any settings done via Nvidia Control Panel as there are adequate

graphics options in the games setup, but the Experience will hold your hand and select 'optimised' settings, so

in theory you don't need to fiddle with the games settings if you install Geforce Experience.

However, optimised translates to settings a bit below what your graphics card can generally handle.

- Razer Synapse is, as you probably know, is profiling software for your Razer products. You need this if you use

custom settings for those peripherals, and it would be best to leave it enabled in Startup.

- Steam Client Bootstraper . Not necessary in Startup, as it will load with Steam.

- One Drive ? Shouldn't cause issues, but not necessary at Startup. I don't use it, but it shouldn't run in
