naveen_m say:

Hi Laur,

Thank you for writing to Microsoft Community Forums.

I understand that your computer wakes up from sleep and switches on after a shutdown automatically, we’ll look into this for you.

We can try a few steps and check if we can fix the system waking up from sleep automatically.

Method 1: Disable the wake timers.

Refer this article on PC automatically wakes from sleep mode (II) and follow the steps to disable the wake timers and check. This article applies to Windows 10 as well. If the issue persists, follow method 2.

Method 2: Use Command Prompt to know what had woke up Windows last and which devices are enabled to wake up the PC.

  1. Click on Start icon.

  2. Type Command Prompt.

  3. Right click on it and select Run as administrator.

  4. Type the following command and press Enter:

    powercfg –lastwake

    powercfg –devicequery wake_armed

Check the results and let us know.

When you try to shut down or hibernate the system on a computer that’s running Windows 10, the process fails and reverts to the Windows Lock screen. We can turn off hybrid sleep and fast startup and check if we can fix this.

Method 1: Turn off Hybrid sleep.

  1. Click on Start icon.

  2. Type Command Prompt and press Enter.

  3. Copy and paste the following command and press Enter.

    Shutdown /s /hybrid /t 0

Method 2: Turn off Fast Startup.

  1. Click on Start icon.

  2. Type Control Panel and press Enter.

  3. Click on Power Options.

  4. Click on Choose what the power buttons do.

  5. Click on Change settings that are currently unavailable.

  6. Under Shutdown settings, uncheck the Turn on fast startup box.

  7. Click on Save changes.

Check if the issue persists.

Regards,

Naveen M

cmdrkeene say:

That'd incredibly odd.  It's normal to have to re-activate periodically (there is no strictly set cadence, but Windows activation does expire and need re-authorized periodically, not unlike satellite TV service set-top boxes -- it can even be triggered from a system BIOS update from your manufacture (Apple in this case, but who it is doesn't matter).

Typically these are entirely seamless and the activation happens silently, usually without much notice to the user.

I can't really speculate why this one would be being blocked.  Usually it means there's a large number of people using it (usually because it got posted on the internet and a ton of people are trying to use it).  But that's really just one of the great many cases that can happen.

I think us volunteers here that discuss consumer topics in this forum can't really be of a ton more assistance, I think your best option would be to contact Microsoft's support and find out what's up.  As long as you have your key that came with Windows 7, they should be able to get you fixed up without a ton of hassle.

If that screen doesn't show a contact number or site, you can use this page.  Just tell the virtual agent you want to activate Windows 7 and after it suggests doing it on-screen (which you've tried and can't), it will give you a phone number to talk to a person.
