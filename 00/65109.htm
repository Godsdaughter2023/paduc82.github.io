bossdweeb say:

Yes, there is no easy way to list all the games that can be run on a GeForce 210, so System Requirements lab

can be useful for determining if a specific game will run on that card.

Once you have run a few games you should be able to extrapolate from the system specifications listed for

each game.

- However, 16 GB RAM is absolutely not required for games. Even the most system intensive games don't use

(including the 60 or so other processes that run with my Windows install) more than 60% of my 8GB.

- Exceptions would be if someone wants to have a heap of open browsers, or other software running

while running a game.

This would also impact on the cpu, and possibly gpu/VRAM (depending on what else is running).

In other words, multitasking is not optimal for running games, and should be avoided, particularly with a

GeForce 210 . I do understand that some people do like to have a bunch of stuff open all the time though,

so in this case 16GB may be useful. Maybe.

 A 4K monitor, or three 1080p monitors may require more system RAM than 1080p , but would also

require more VRAM.

- Using RAM Disk may also require more system RAM, but RAM Disk is somewhat redundant with an SSD,

and not really beneficial with games loading from the HDD, as the whole game has to be loaded to RAM Disk,

and that does not improve game performance, only loading times.

- Also, a GeForce 210 is not a gaming grade graphics card, so having more than 3GB system RAM

is not likely to help performance anyway, as the GeForce 210 would likely not be able to keep up.

Having said that.  3GB is imbalanced, and most likely running in single channel mode. For better performance
it would be best to buy a matched kit of two 2GB sticks so the RAM runs in dual channel mode.

If you plan on upgrading the graphics card in future (and probably a new power supply to suit) you may

want to consider buying a matched pair of 2X4GB.
