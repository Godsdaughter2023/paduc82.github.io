    Sorry if those responses seem flippant, but you seem to expect yes/no answers to questions that don't have them.

    The junk filter included with Windows Live Mail doesn't learn from your actions. It is used in conjunction with your safe and blocked senders lists to move messages to the junk email folder when appropriate.

    Clicking or selecting Mark as junk (Ctrl-Alt-J) will indeed move a message to the junk folder. If at Safety Options > Options you have Report junk email to Microsoft ... selected, your action will be reported and used in conjunction with data from other users to improve the junk filters. You won't necessarily see any effect of this unless you use Hotmail until the WLMail junk filter is updated. We were once promised years ago that the filter would be updated monthly in the same way as Outlook's is, but this has never happened and I don't expect it to.

    So it depends on your safety settings whether marking as junk does anything other than moving the message to the junk folder. Deleting a junk mail message from your inbox will not perform the public service that marking as junk would do given the appropriate settings.

    Hi,

     

    If the old Laptop came with the Windows Operating System preinstalled (Dell, HP, Acer, etc), that Product Key that came with the Laptop is for the preinstalled OEM License and is non-transferable.

     

    The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

     

    Only Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

    And Retail Licenses can also only be installed on one machine at any particular time.
