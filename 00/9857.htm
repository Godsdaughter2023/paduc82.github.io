john guin say:

Here's how I would do it.  Assume Alice and Bob are the retirees.

1. Go to Alice's machine. 

2. Plug in a USB thumb drive.

3. Open the notebook from that machine. 

4. Right click it, select Properties | Change Location

5. Change it to be on the USB drive.

6. Now go to Bob's machine and do the same thing.

Now that thumb drive will have 2 folders, one for Alice and one for Bob.

Drag and drop all the contents out of the Alice folder and drop it in the Bob folder.

Now when you open the Bob folder, the two notebooks will be combined (if they are large, this may take a few seconds).

Then you can right click the new combined notebook, select Properties and change location to wherever your manager wants.

graham mayor say:

Page background colours are a throwback to when Word was used as a tool for creating HTML web pages. If you want areas of colour (or a complete background colour) to survive a mail merge then use a full page background text box with a colour fill. Insert it in the header view and set the wrap to behind text.

Alternatively you might consider http://www.gmayor.com/ManyToOne.htm   which in One to One mode will produce a series of documents, one per record, and because it doesn't use mail merge to create the documents, they are all clones of the original with the inserted data and will thus retain the background colour.
