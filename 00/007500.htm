30% or less equals 100%

The goal is an employee turnover rate of 30% or less at each location, each month.  Each month, the turnover rate will be entered for each location.  If every location received a score of 30% or less for the month, it would equal a 100% average.   What formula do I use? Thank you.
jasles say:

If the turnover rate is entered in cell A2 and A3, use this formula in B3:

=IF(AND(A2<=30%,A3<=30%),100%,"")

Then B3 would equal 100% if both A2 and A3 are 30% or less, otherwise it will be blank.

If you need more cells to be included in the evaluation, add a comma between the 30% and the bracket and add your criteria.

norman david jones say:

Hi RocketFan,

TGry something like:

  • Alt+F11 to open the VBA editor
  • Alt+IM to insert a new code module
  • In the new module, paste the following code 
'=========>>
Option Explicit

'--------->>
Public Sub Tester()
    Dim srcWB As Workbook, destWB As Workbook
    Dim srcSH As Worksheet, destSH As Worksheet
    Dim Rng As Range
    Dim LRow As Long
    Const sSheetName As String = "Upload"
    Const sLastColumn As String = "AR"

    Set srcWB = ThisWorkbook
    Set srcSH = srcWB.Sheets(sSheetName)

    With srcSH
        LRow = LastRow(srcSH, .Columns("A:A"))
        Set Rng = .Range("A1:" & sLastColumn & LRow)
    End With

    Set destWB = Workbooks.Add
    Rng.Copy Destination:=destWB.Sheets(1).Range("A1")

    With destWB
        .SaveAs Filename:=Application.DefaultFilePath _
                        & Application.PathSeparator _
                        & sSheetName & ".csv", _
                FileFormat:=xlCSV
        .Close Savechanges:=False
    End With
End Sub

'--------->>
Public Function LastRow(SH As Worksheet, _
                        Optional Rng As Range)
    If Rng Is Nothing Then
        Set Rng = SH.Cells
    End If

    On Error Resume Next
    LastRow = Rng.Find(What:="*", _
                       after:=Rng.Cells(1), _
                       Lookat:=xlPart, _
                       LookIn:=xlFormulas, _
                       SearchOrder:=xlByRows, _
                       SearchDirection:=xlPrevious, _
                       MatchCase:=False).Row
    On Error GoTo 0
End Function
'<<=========

  • Alt+F8 to open the macro window
  • Select Tester | Run

teylyn say:

Hello,

is this really what you need to do? Typically, if weekends and holidays are excluded, the rule would be to add only business days to the date. Excel has the Workday() function that adds business days to a date, skipping over weekends and holidays.

Adding 15 days to a date, you already know what the weekday of the outcome will be. If the start day is a Friday or Saturday, then the result will be a Saturday or Sunday respectively, so you need to factor that into the formula and add fewer days to return the previous Friday. That can be done with a simple IF statement.

=A2+15-IF(WEEKDAY(A2,2)=5,1,IF(WEEKDAY(A2,2)=6,2,0))

Likewise adding 21 days. It's always the same weekday as the starting weekday, so add 2 more days for Saturday and one more for Sunday to return the following Monday:

=A5+21+IF(WEEKDAY(A5,2)=6,2,IF(WEEKDAY(A5,2)=7,1,))

Adding days to a date and then figuring out if it is a holiday, and then adding/subtracting one or more days is a lot more complex.

dicksorensen say:

I recommend examining the Excel "WORKDAY()" function, I think it suits this situation more elegantly.

All that is needed is a range of holidays, and it automatically excludes Saturdays and Sundays too. It "worries" about the conditions for you so you don't have to program so much Boolean logic ( ... and's, or's, and if's).

Please note that I may have found a sort of bug in the workday function. If you ask it to find the nearest workday 0 days ago (i.e if you want to perform the following "if today is a workday return today, if today is not a workday return the last workday"), it spits out the current date, even if it is a Saturday or Sunday ... however it can easily be fooled by adding 1 to the date, and then asking it to find the date 1 day ago (+1-1=0).

That is:

=workday(today(),0,[holiday range])

returns a Saturday if today is a Saturday, but

=workday(today()+1,-1,[holiday range])

returns a Friday if today is a Saturday, and returns Friday if it is a Friday

Microsoft support on the excel workday function is found here:
