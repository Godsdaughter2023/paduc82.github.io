per jessen say:

There is no way you can test if the initial ControlSource has been set, as the ControlSource property is cleared when the userform is closed!

 

To solve it, you have to test if the initial control source cell is emtpy or not, and based on this decide which part of the code to use...

 

Alternatively, you can use a cell (in a hidden worksheet) to hold a boolean value (true/false), to test on and determine which part of the code to run.

 

sub ufm_initialize

if Shets("Sheet1").range("A1")=false then

    sheets("sheet1").range("A1")=true

   'run initial code

else

  'run other code

endif

end sub
