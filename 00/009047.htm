rich michaels - mvp (greatcirclelearning.com) say:

Hi Brett,

I never had an occasion to do that so I looked at the KB article you reference and adapted one of their examples so that it does now work in Mac Office 2011 VBA.

Sub ShowUserForm4()
        Dim X As Object, Y As String
       'Display an input box asking for a number between 1 and 3
       '(inclusive).
       Y = InputBox("Enter 1, 2, or 3", "Select a User Form", "1")

       'Based on the number entered in the input box, X will be set to
       'the appropriate user form object.

       Select Case Y
           Case 1
               Set X = New UserForm1
           Case 2
               Set X = New UserForm2
           Case Else
               Set X = New UserForm3
       End Select

'       VBA.UserForms.Add(X).Show
       X.Show
End Sub

Hope this helps
