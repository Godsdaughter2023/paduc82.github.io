As you have discovered, the Right to Left setting applies only to the text within the columns and not to the columns themselves.

You could make a request for that feature to be added to Word via http://word.uservoice.com/

In the meantime, you would have to manually re-arrange the text and the way that I would do that would be as follows:

  1. Make a copy of the document
  2. In that copy, change to a single column layout and set the left and right margins so that the space between them is the same as the width of the columns in your two column document and use the same top and bottom margins as in that document so that then, the text on each page of the document would be the same as the text in a single column of the original
  3. Copy and paste the text from each page or the single column document into the required columns of the two column document.

Drop down menu in Word, select the display option but have the value - Microsoft Community

wepwawet78 say:

Drop down menu in Word, select the display option but have the value populate

Hello,

 

I would like to ask if anyone knows how to have a drop down menu in Word populate a secondary value.  Example:  I would like to have a drop down menu of animal names.  When a person selects 'dog' the text that populates is in Spanish, or 'perro').   The idea is that this will be used as a template and someone that does not speak Spanish will be able to customize a letter by selecting the correct English word necessary and it will populate in Spanish for printing. Any help would be appreciated!

 

Thanks!

greg maxey say: This is not possible with a dropdown as a dropdown is restricted and can only display what is in its list.  You could use a combobox content control where the display is "dog" and the value is "perro" then run the following macro on exit:

Private Sub Document_ContentControlOnExit(ByVal CC As ContentControl, Cancel As Boolean)
Dim lngIndex As Long
Select Case CC.Title
  Case "Animals"
    For lngIndex = 1 To CC.DropdownListEntries.Count
      If CC.Range.Text = CC.DropdownListEntries(lngIndex).Text Then
        CC.Range.Text = CC.DropdownListEntries(lngIndex).Value
        Exit Sub
      End If
    Next lngIndex
End Select
End Sub
