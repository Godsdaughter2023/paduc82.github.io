Since we have limited access here at the Community forum with regards to Billing concerns, I strongly recommend you to contact our chat support team in order for them to make an investigation and collate necessary information from you

3. Click Next.

4. Click Text chat support.

  • Try again after clearing the browser’s cache and cookies.

  • Try using in-private browsing mode.

  • Try using a different web browser.

  • Try using a different device.

  • Try using a different Internet connection.
