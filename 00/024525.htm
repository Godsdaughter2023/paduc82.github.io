  2. From the ribbon menu choose Project > Subproject

  4. Browser to your 90Mb monster and click Insert
  5. Expand the sub project, and select all tasks except the first one (i.e. Outline Level 2 tasks only)

  7. Delete task 1
  8. File > Save As, and give the file a name
