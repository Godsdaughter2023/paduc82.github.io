My way to make SCSI External Hard Drive work with Windows 10:

1. Plug in SCSI Hard drive to computer ( Ex: Samsung T3, All external hard drive Seagate..)

2. Open Device Manager ( Right Click on Windows Logo then choose Device Manager)

3. Explore Disk Drive we will see the SCSI device with a yellow triangle.

4. Left click to open SCSI device properties.

5. Choose tap Driver

6. Click Disable device.

7. The Device Manager window will become unresponsible.

8. Ignore this symptom then wait about 30 second.

9. Shut down computer then UNPLUG the SCSI external hard drive AFTER the computer ready shut down. ( If Windows do not shut down we might forced it shut down)

10. Turn On the computer then wait until it's ready stable.

11. Plug SCSI external hard drive back to computer.

12. Open Device Manager.

13. Explore Disk Drive, this time we will see the SCSI device with a + inside a circle.

14. Left click to open SCSI device properties.

15. Choose tap Driver.

16. Click Enable device.

17. Now the SCSI will begin working.

18. Before shut down the computer ALWAYS come back to the SCSI driver to Disable device so next time when we plug it in , it will work after we ENABLE DEVICE.

Here are my suggestions:


If some virtual CDROM or DVD devices have been seen in Device Manager, please right-click it, then click "Uninstall", then restart your computer.


In order to reduce the misoperations, you had better use batch file to perform the operation that modify registry each time, and after each time performing please restart your computer to take effect the settings, and then for checking whether the settings take effect each time you had better run a same batch file named ChkDVD.bat within only a same command, and repeatedly do the same each time, time and time again.

You could perform following specific and detailed operations:


Please use Notepad to make a batch file that reads as follows:
wmic cdrom get drive, mediatype, name, status >> %UserProfile%\Documents\GetCDROMInfo.txt


Then please save it as "%UserProfile%\Documents\ChkDVD.bat".
(You will use it in the future to check whether the settings take effect after each round trying.)

First round,


Please use Notepad to make a batch file that reads as follows and save it as "%UserProfile%\Documents\EnumDevice2.bat":
reg add "HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\atapi\Controller0" /f /v EnumDevice1 /t REG_DWORD /d 0x00000001
reg add "HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\atapi\Controller0" /f /v EnumDevice2 /t REG_DWORD /d 0x00000002


Then execute it by "Run as Administrator".

Then restart your computer.

Then execute "%UserProfile%\Documents\ChkDVD.bat" batch file by "Run as Administrator".
