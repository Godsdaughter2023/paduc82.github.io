    How to create a Refresh button for two pivots in a single sheet from - Microsoft Community

    rohit abichandani say:

    How to create a Refresh button for two pivots in a single sheet from the same data

    Hi there,

    I have created two different pivot tables in same sheet from same data source.

    I want to create a button so that both the pivot table can be refreshed from a single button.

    Thank you!

    rajesh c. say:

    @Rohit,

    - Under Developer Tab, click on Record Macro

    - Now Click on Stop Recording

    This will create a macro named "Macro1"

    Your macro code will look like:

    Sub Macro1()
    '
    ' Macro1 Macro
    '

      
    '
    End Sub

    Now you need to edit this macro and copy the following lines Dim WRKSH As Worksheet
    Dim PVTBL As PivotTable

    For Each WRKSH In ThisWorkbook.Worksheets
        For Each PVTBL In WRKSH.PivotTables
            PVTBL.RefreshTable
            PVTBL.Update
        Next
    Next
       

    Your final macro code should look like :

    Sub Macro1()
    '
    ' Macro1 Macro
    '
    Dim WRKSH As Worksheet
    Dim PVTBL As PivotTable

    For Each WRKSH In ThisWorkbook.Worksheets
        For Each PVTBL In WRKSH.PivotTables
            PVTBL.RefreshTable
            PVTBL.Update
        Next
    Next
    '
    End Sub

    You can now run this macro directly OR

    assign it to a button and then run the macro by clicking the button.

    Hope you are aware of how to create a button in Excel and How to assign a macro to the button.

    You can Google it and you will find the solution - its simple.
