sanmartin say:

Ich bin auf einen möglichen Hack ja nur deshalb gekommen, weil ein GRUB Loader wirklich nicht zur Standardausstattung von Windows gehört. Der Fehler "Windows startet nicht ohne CD im Laufwerk" bedeutet normalerweise, dass Windows keine Bootfiles auf der Platte findet, bei der Installation also was schiefgelaufen ist (z.B. falsche, nicht "aktive" Partition als Ziellaufwerk angegeben, so dass BIOS woanders hin bootet - wo dann keine Bootfiles für das Betriebssystem sind). Die sauberste Lösung wäre, Windows von CD neu zu installieren (benutzerfdefiniert) und das Ziellaufwerk bei dieser Gelegenheit zu formatieren (vom Setup aus). Darauf achten, dass es sich dabei um die Platte/Partition handelt, die als erste in der Boot-Reihenfolge des BIOS kommt.

I only pointed to a possible hack because a GRUB loader actually isn't a standard component of Windows. Usually, the error "Windows won't start without DVD inserted" means that Windows doesn't find bootfiles on the HD = something went wrong during the installation (f.e., the target drive isn't marked as "active" and BIOS boots to another partition/HD instead - where bootfiles for the OS aren't present). The cleanest solution would be to reinstall Windows from DVD (custom install) and to format the target drive from there (Drive options - advanced). Keep in mind that the target must be the drive/partition which is first in the BIOS boot order.
