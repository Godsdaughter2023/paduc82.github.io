repeated temporary profile issue windows 8

window 8 signs me into a temporary profile where i cannot access my files and any changes i make are deleted when i sign out...however i searched for the issue online and found the solution that i should create a new user and transfer my files to  it so i tried it and it worked but after a few days the issue came back again ... sometimes it is solved by signing out then in again but sometimes it is not solved ... any suggestions??

anannya podder say:

Hi Ali,

Usually when temp profile is loaded, it would create a new registry key. The new profile registry key would look like same as original profile key, but the original profile key would have .bak at the end.

Basically you need to remove the .bak from original correct profile key and rename the new temp key. Restarting the Windows 8 or 8.1 computer after modifying registry probably will bring back your old profile with your customized desktop and icons.

Steps to remove the .bak registry key.

REGISTRY DISCLAIMER:

Modifying REGISTRY settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of REGISTRY settings can be solved. Modifications of these settings are at your own risk.

a. Press Windows Key+ R, type Regedit and hit Enter.

b. Navigate the following location.

Computer\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\WindowsNT\CurrentVersion\ProfileList


You will see similar keys under profile list, but one with .bak.

c. The key with .bak is for the correct old profile. Currently your Windows computer logged in with fresh (temporary) profile with same key. So, rename the new profile key ( which is not having.bak) and remove .bak from correct profile key.

Log off and log in (or restart) with your user name and password. Check if you get your correct profile with icons and profile settings back in Windows.

Still if Windows loads with temporary profile and creates a registry key with .bak, that means user profile is corrupted and Windows is not reading it properly.

Hope it helps. Let us know how it goes on your computer.

Basic troubleshooting usually starts the same as with any other type of hardware devices. Try the following:

 

a)      Make sure that the devices are supported by your operating system and then

b)      Go to the manufacturer’s web site and download the latest version of software or drivers for those devices. While you’re there you might browse the site to see if there are known issues or other information for your device.

c)      Make sure the Bluetooth device is listed in the Bluetooth Devices Browser in the Control Panel (keep in mind that some Bluetooth devices may install a custom dialog window for their device in the control panel).

d)      Make sure that you’re system is up to date on any service packs and other updates for your operating system

 

For additional troubleshooting, I would suggest you to try these methods and check.

 

Method 1

Try to run the hardware troubleshooter and check what the issue is and fix it accordingly.

Follow the link given below open hardware troubleshooter.

http://Windows.microsoft.com/en-us/Windows7/Open-the-Hardware-and-Devices-troubleshooter

 

Method 2

Check for the Bluetooth support service status:

 

a)      Click Start, click Run, type services.msc, and then click OK.

b)      Double-click the Bluetooth Support service.

c)      If the Bluetooth Support service is stopped, click Start.

d)      On the Startup type list, click Automatic.

e)      Click the Log On tab.

f)       Click Local System account.

g)      Click OK.

h)      If you prompted to restart the computer, click Yes.
