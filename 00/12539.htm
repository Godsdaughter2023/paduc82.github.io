Hi Akiko,

The Default MRM Policy is applied automatically to new mailboxes in Exchange Online.

You can apply personal tags to folders you create or to individual items. Messages that have a personal tag applied are always processed based on the personal tag's settings. You can apply a personal tag to a message so that it's moved or deleted sooner or later than the settings specified in the DPT or RPTs applied to that user's mailbox.

I am not sure why Icalendar items present a problem, but not using them sure avoids a problem, at least for now.

How to use "Sheet1" as a formula reference when Sheet1 - Microsoft Community

hgood say:

How to use "Sheet1" as a formula reference when Sheet1 actually has a name?

Hi, I need to be able to do this in vba code, but since it has to do with a formula I'll ask it here. I have a spreadsheet that is translated into various languages. In the code I need to refer to Sheet1, Sheet2, etc. because sheet tabs will be translated. But part of the code is inserting a formula into a cell. So I need the formula to refer to Sheet1, even tho Sheet1 actually has a name. I cannot figure out how to do this.  =Sheet1!A1 does not work because Sheet1 has a name.  Any ideas?  Thanks!
jllatham say:

In the VBA Editor environment, you'll see the list of Excel objects in the VBAProject window, and there you will see the name that is on their tab in parenthesis like:

Sheet1 (Tab Name)

In VBA you can get the name like this:

aSheetsName = Sheet1.Name

which will assign "Tab Name" to the variable aSheetsName.

You can then take that and create a formula for a cell like:

newFormula = "='" & aSheetsName & "'!A1"

and assign it to some cell somewhere, like

Sheet3.Range("B4").Formula = newFormula

Note that if you spread out the quoted portions, you'll see that it is placing a single quote mark in front of and after the sheet name.  That's to handle the times when the name on the tab contains spaces.  If the name on the tab doesn't contain spaces, Excel will ignore the single quote marks when you put the formula into a cell.

I hope that helps you move forward.
