With Office365, I no longer experience the IMAP issues I've been plagued with for years.  Depending on the mail client used/prefered, here's how I solved my Hotmail/Outlook issues:

1)  If using MS Outlook (for Mac or Windows), set up a new account using Exchange (not IMAP).  Connection specifics should automatically populate once you enter your username and password.  Once successfully connected, your mail and all folders should quickly populate.  Afterwards, you can safely delete any existing IMAP accounts previously set up to access Hotmail and use only the Exchange account.
