greginmich say:

If you want to schedule a Windows Defender scan; then select the interface and scan type from the options below:

Then right-click on the Start button and select either Command Prompt (Admin) or Windows PowerShell (Admin) and copy, paste, and run the selected command line at the prompt:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning-windows_10/windows-defender-how-to-schedule-a-scan/d4ba70ce-a70c-4d82-bb26-18902b4e8f9e

To schedule a scan that runs in the Windows Defender Security Center app, use these commands:

Windows Defender Quick Scan:

SchTasks /create /sc DAILY /tn “Defender Quick Scan” /st 19:00 /IT /rl HIGHEST /tr "'C:\Program Files\Windows Defender\MSASCui.exe' -QuickScan"

Windows Defender Full Scan:

SchTasks /create /sc Weekly /tn “Defender Full Scan” /st 19:00 /IT /rl HIGHEST /tr "'C:\Program Files\Windows Defender\MSASCui.exe' -FullScan"

To schedule a scan that runs in the Command Line Interface, use these commands:

Windows Defender Quick Scan:

SchTasks /create /sc DAILY /tn “Defender Command Line Quick Scan” /st 19:00 /IT /rl HIGHEST /tr "'C:\Program Files\Windows Defender\MpCmdRun.exe' -Scan -ScanType 1"

Windows Defender Full Scan:

SchTasks /create /sc Weekly /tn “Defender Command Line Full Scan” /st 19:00 /IT /rl HIGHEST /tr "'C:\Program Files\Windows Defender\MpCmdRun.exe' -Scan -ScanType 2"

Then (optionally) click on the Task Scheduler Library folder in Task Scheduler, locate your task, and double-click on the listing for it. Then either change the scheduled time to time when you’ll normally be using the PC – or else check the Wake the computer to run this task box in the Conditions tab, and the Run task as soon as possible after a scheduled start is missed box in the Settings tab.

To set up a scheduled scan with the Create Basic Task Wizard, see this thread:

https://answers.microsoft.com/en-us/protect/forum/protect_defender-protect_scanning-windows_10/how-do-i-set-windows-defender-to-a-custom-auto/ec19aab4-10ce-4f41-9990-343056fd104f
