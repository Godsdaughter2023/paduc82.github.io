    rob koch say:

    Though the appearance of any email message could possibly be faked, a message formatted as referenced at the link you provided above may potentially be a valid security alert email from Microsoft.

    The way to be certain is to hover over the links within the message and confirm that they actually display a Microsoft.com address in the domain name portion of the URL.  In any case, the Recover Account button in the email should take your browser to the following page which you'll notice is related to Windows on Microsoft.com.

    http://windows.microsoft.com/en-us/windows/get-back-blocked-account

    If it takes you anywhere else, especially any non-Microsoft address or asks you for any personal information including account login or password, then you know the email is a fake.

    The only point at which your login information might be requested is when you are mot logged into a Microsoft Account (same as the email address the alert message was for) on the PC you are using.  In other words you are either logged into a local account on the PC or the PC being used belongs to someone else, which typically only happens if you're reading your email via a web browser using the Outlook.com web based email app.

    With a Windows 10 PC logged into a Microsoft Account, both the Mail app on the Windows start screen and the Microsoft Account page at the following URL should login automatically without being prompted for login/password information.

    https://account.microsoft.com/

    macropod say:

    Try deleting the Table of Contents control and using just a Table of Contents field - you can create one by pressing Ctrl-F9 to create a pair of field braces, then typing TOC between them, thus:

    {TOC}

    then pressing F9 to update the field. Amongst other things, the field performs much faster on its own than when inserted via the Table of Contents control.

    If you still have problems, that suggests your document has acquired some of corruption. Corrupt documents can often be 'repaired' by inserting a new, empty, paragraph at the very end, copying everything except that new paragraph to a new document based on the same template (headers & footers may need to be copied separately), closing the old document and saving the new one over it.
