Distributed Transaction Coordinator (DTC) service error code - Microsoft Community

chucksu say:

Distributed Transaction Coordinator (DTC) service error code 1073737712.

Original Title: MSDTC

a. user say:

Unless you have a need to use the DTC Service, I would set it to Manual Start or Disabled since there is no need for it to be running all day with nothing to do.

Maybe you do need to have it running or just want to be sure it will start when you need it to start without complaining, so in that case, read this:

Trying to start the Distributed Transaction Coordinator Service (MSDTC) you see a message like this:

Windows could not start the Distributed Transaction Coordinator on Local Computer. For more information, review the System Event Log. If this is a non-Microsoft service, contact the service vendor, and refer to service-specific error code -1073737712.

Look for Events like this in the Event Viewer Application log:

Event Type:    Error
Event Source:    MSDTC
Event Category:    Services
Event ID:    4163
Description:
MS DTC log file not found. After ensuring that all Resource Managers coordinated by MS DTC have no indoubt transactions, please run msdtc -resetlog to create the log file.

Event Type:    Error
Event Source:    MSDTC
Event Category:    Disk
Event ID:    4185
Description:
MS DTC Transaction Manager start failed. LogInit returned error 0x2.

Event Type:    Error
Event Source:    MSDTC
Event Category:    Devices
Event ID:    4112
Description:
Could not start the MS DTC Transaction Manager.


Usually, that means the MTDSC.log file is either missing or somehow corrupted.  The MSDTC log file is usually here so take a look:

C:\WINDOWS\System32\Msdtc

To reset and create a new MSDTC.log file, click Start, Run and in the box enter:

msdtc  -resetlog

Click OK and that will create a new MSDTC.log file, then see if you can start the MSDTC Service and look for something like this in the Event Viewer Application log:

Event Type:    Information
Event Source:    MSDTC
Event Category:    Disk
Event ID:    2444
Description:
MS DTC started with the following settings:

  Security Configuration (OFF = 0 and ON = 1):
      Network Administration of Transactions = 0,
      Network Clients = 0,
      Inbound Distributed Transactions using Native MSDTC Protocol = 0,
      Outbound Distributed Transactions using Native MSDTC Protocol = 0,
      Transaction Internet Protocol (TIP) = 0,
      XA Transactions = 0
