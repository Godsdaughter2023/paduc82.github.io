suzanne s. barnhill say:
Note that current versions of Word, which use HTML paragraph auto spacing, will honor only the larger of Before/After spacing, so when you apply 3 pts Before and 3 pts After, you're getting just 3 pts. In a given style, you can enable "Don't add space between paragraphs of the same style," but that will suppress all the spacing between the paragraphs, leaving it only before and after the entire block. So you may be better off experimenting with the Before spacing of the paragraph following the bulleted list.
