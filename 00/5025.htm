rich007a say:
I have found the instructions on adding lists to the individual ComboBox controls and should be able to use that properly. I would like to know if there is a more efficient way to use the same list for multiple controls (instead of me needing to do the same code 35+ times for the Yes/No questions).


Add a combobox to your form.

In the RowSource property in the Properties window pane, enter the range (e.g.  Sheet1!A1:A5 ).

Copy the combobox and paste as required.  The same range will be in the rowsource property.

Does that help?

 

You also asked about more efficient ways of writing very similar code for multiple controls.  Two options:

1) Use Excel to help write your code:

e.g. if you want to write the following VBA code:

If Me.Control(1).Text = "Fred" Then ...
End If

If Me.Control(2).Text = "Fred" Then ...
End If

etc..

You could put this in B1:  If Me.Control(

And this in C1:      1

And this in D1:      ).Text = "Fred" Then ...

And this in B2:     End If

In A1 enter   =B1&C1&D1

In A2 enter   =B2

Select cells A1:D2 and drag down, this will make the 1 in C1 incremement.  Now copy the text in column A into VBA.

This is a simple example of how you can use Excel to build up repetitive code with changes to each line.

 

2) Use a procedure, call the same procedure from multiple controls, passing the control as an object as an argument to the procedure.

As an example, create a userform with three comboboxes and three label controls.  Set the rowsource range for each combobox.  Add this code:

 

Private Sub ComboBox1_Change()
    ComboBoxAllChange Me.ComboBox1
End Sub

Private Sub ComboBox2_Change()
    ComboBoxAllChange Me.ComboBox2
End Sub

Private Sub ComboBox3_Change()
    ComboBoxAllChange Me.ComboBox3
End Sub

Private Sub ComboBoxAllChange(ctrl As Control)
    Dim i As Long, x As Control
    i = Right(ctrl.Name, 1)
    For Each x In Me.Controls
        Debug.Print TypeName(x)
        If TypeName(x) = "Label" Then
            If x.Name = "Label" & i Then
                x.Caption = ctrl.Text
                Exit For
            End If
        End If
    Next x
End Sub

You can create the code for the first three subs using the method described in (1) above.  In the (must be Private) sub at the bottom, the calling control is parsed as an argument.  You can then do stuff in this sub depending on which control has called it.  In this example, the number is extracted from the end of the calling control's name.  It is then combined with "Label" to make either Label1, Label2 or Label3.  You can't put a control's name in Me.Controls(xxx) so you have to loop through each control in the form and test it's name, hence the For x In Me.Controls... loop.

Note I included a double test in there, possibly unnecessary, but useful to know about If TypeName(control) = "xxxxxxx" since this can allow you to make changes to all the Label controls in one go, e.g. you could have another procedure to clear them all, but not touch the combos or other controls:

    For Each x In Me.Controls
        If TypeName(x) = "Label" Then
            x.Caption = ""
        End If
    Next x

Hope I haven't overloaded you and hope this helps.

Cheers
Rich

This alternative code for the ComboBoxAllChange macro has some comments to try to explain what's going on.

 

Private Sub ComboBoxAllChange(ctrl As Control)
' "ctrl" is a control object parsed to this procedure as an argument, e.g. Me.ComboBox1.
'
    Dim i As Long
    Dim x As Control 'x is any control object (combobox, label, etc.)
   
     'Check if the parsed control is a combobox
    If TypeName(ctrl) = "ComboBox" Then
         'Extract the number in its name by removing "ComboBox" from its name
        i = Replace(ctrl.Name, "ComboBox", "")
    Else
        'You parsed the wrong kind of control!
        Exit Sub
    End If
   
     'Loop through each control in the form:
    For Each x In Me.Controls
         'Set the text of the label that coresponds to the parsed combobox (ctrl)
        If x.Name = "Label" & i Then
            x.Caption = ctrl.Text
            Exit For 'stop looping
        End If
    Next x
End Sub
