To troubleshoot your issue regarding printing a shaded text, we suggest doing this following methods:

Method 1:

Using Printer Troubleshooter:

  1. Type troubleshooting in the search bar.
  2. Click Troubleshooting and select View all on the top left corner.
  3. Click on Printer.
  4. Follow the on-screen instructions to run the troubleshooter.
  5. Check if issue persists.

Method 2:

Restarting Printer Spooler:

  1. Press Windows + R to open Run,
  2. Type services.msc and press Enter.
  3. Find and stop Printer Spooler.
  4. Navigate to this folder locations:
    C:\Windows\system32\spool\PRINTERS
  • Click Continue or Yes and delete all files in PRINTERS folder and then close Windows explorer.
    Note: Do not delete PRINTERS folder. Only delete its contents.
  • Open services, and start Print Spooler.

Let us know how it goes.
