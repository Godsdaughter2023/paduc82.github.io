azeez n say: Hi iravid,

Windows Media Player doesn't support playing SRT subtitles by default.

To answer "how to play a SRT file on Windows Media Player", just install the codec - DirectVobSub . DirectVobSub is a Directshow filter which allows you to display subtitles in Windows Media Player. It is also compatible with all Direct Show players. The latest version of Direct Vob Sub is 2.39.

Install DirectVobSub codecs of your choice using your favorite search engine.

Also see, suggestion to install codecs: http://windows.microsoft.com/en-us/windows7/Codecs-frequently-asked-questions


Now when whenever you want to view any subtitles, copy the SRT file to the same folder as your video and rename it to the same file as the movie but with the SRT extension. If your movie file is ABC.avi, then you should have your subtitle ABC.srt in the same folder. Now, open your video file in Windows Media Player. Go to Play → Captions and Subtitles and select On if Available.

Now whenever you play that video, you should see the subtitles near the bottom of the display window
. The procedure is exactly the same for SUB subtitles as well.

Vista cannot connect to printer - Microsoft Community

xyzq say:

Vista cannot connect to printer

I have three computers, one with XP, one with Vista Home and one with Win7.  The XP has a Canon printer attached which is shared over the network.  This has worked well for a long time.

Recently I got a Kodak esp6150 wifi attached printer that I installed in addition to the old Canon.  It works well from all three computers.  Since installing it however the Vista compuer no longer can find the Canon printer on the XP computer.  So I tried to Add Printer back into Vista.  It shows the printer in the list of network attached printers, but when I select it and tell Vista to add it I get the error "cannot connect to printer".

Ok, I went back to a restore point before the Kodak and the Canon works fine again.  I reinstalled the Kodak software and both printers worked ok for maybe half an hour and the Canon disappeared from Vista again.

Does this sound familiar to anyone?  Any idea how to get Vista to recognize both printers as XP and Win7 are both happy to do?

Thanks.

Bill

 

 

 

xyzq say:

OK, I believe I have my problem resolved from info found elsewhere on the web.  The trick seems to be to tell Vista to install alocal printer for the Canon even though it's a shared network printer.  And to create a local port with the name \\XPserverName\PrinterName.

But as I stated, this is only necessary if the Kodak is installed.  Otherwise Vista sees the Canon as a shared network printer.
