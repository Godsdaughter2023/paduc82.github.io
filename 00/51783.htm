andre for directly say:
Why would you want to cancel a pending update? If your objective is not to install Windows Updates at all, then do the following:

Create a Notepad file with each of the following and save them with the .bat extension.

stop-updates.bat:

sc config wuauserv start= disabled
net stop wuauserv

sc config bits start= disabled
net stop bits

sc config dosvc start= disabled
net stop dosvc

pause

start-updates.bat:

sc config wuauserv start= auto
net start wuauserv

sc config bits start= auto
net start bits

sc config dosvc start= auto
net start dosvc

pause

Now you can simply right click on stop-updates.bat and select Run as Administrator

a. user say:

We're glad to know that everything is updated. By the way, some monitors will work with the Surface Dock, some are not, and compatibility of cables and adapters varies greatly. For rapidly flickering screen, the monitor may be receiving a dirty or corrupted signal from the display adapter, and the monitor is trying to display the signal but is not able to adjust its format. To resolve your issue, we suggest that you perform the troubleshooting steps provided below:

  1. Check all connections, including power cables and adapters. Unplug and reseat any display adapters and ensure that they are connected securely.
  2. Remove power from the dock and then reconnect it. If the power cable is connected to a power strip, plug it directly into a wall outlet instead.
  3. Try connecting the video cable directly to the Surface instead of through the dock.
  4. Whenever possible, always try using a different monitor and a different cable to see if the behavior changes.
  5. Check for third-party display software installed on your device.
  6. Adjust video settings for brightness and contrast to see if the behavior improves.
  7. Try changing display resolution settings for the external monitors.
  8. Use the keyboard hotkey combination Win+Ctrl+Shift+B to reset the graphics stack.
  9. Perform a two-button restart on the device. 
  10. Apply the latest Surface Dock firmware by installing and running the Surface Dock Updater tool from this link.

Should you need further assistance, feel free to get back to us.
