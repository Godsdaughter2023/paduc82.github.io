nortman say:

The window that appears allows you to choose any bitmap (.bmp) file to use as a mirror texture. There aren't any already available in the folders on your computer, you will need to find one. I searched for "gold textures" using the image search option on Goog: Gold Textures, and I imagine similar results could be found with other search engines.

Once you have found something that looks shiny (keep the image relatively plain: complex images will get lost in the animation) save it somewhere you can find it (eg the Pictures library)

If the image you find is .jpeg or .png you will need to convert it to .bmp - do this by opening it Paint, and choosing "Save As" and on the filetype option choose "24bit Bitmap"

Using the "Custom Reflection" option will move the image separately from the movement of the text, adding to the reflective effect. "Custom Texture" will keep the image on the same place on the text (eg for wood grain effect, or bricks, or a tie stripe)

This is my workaround to let Validity Sensor VFS300 work with Windows Hello after I updated my Dell Vostro 3700 from Windows 7 Pro x64 to Windows 10 Pro:

1. Uninstall the stock fingerprint software (mine was DigitalPersona), then reboot.

2. Uninstall and delete the original driver (mine was called "Validity Sensor VFS300") from Device Manager\Biometric Sensor, then reboot.

3. Check for the new driver (mine is called "Validity Sensor (WBF) VFS300") installation in Device Manager\Biometric Sensor.

4. Check for Windows Hello to show up in Settings\Accounts\Sign-In Options and activate it.

So, for me DELL driver doesn't work, but the auto updating one does.

Press 4 or F4 to enable Safe Mode (5 or F5 to enable Safe Mode with Networking)
Log in once Windows starts in safe mode
See whether you able to login. If yes, come out of this and start normally.

If you are not able to boot like above, You will need to boot through a Bootable USB drive. You will need to create this on another computer which is working - https://www.tenforums.com/tutorials/2376-create...

All apps will be removed and windows will be reinstalled.
