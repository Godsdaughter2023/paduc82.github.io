Hi Damian,

What it means is that the people in Marketing make promises that Engineering can't deliver.  They will point the finger saying that the device manufacturer will have to make a compatible driver.  There is no way that Windows 10 will have drivers for all hardware in the Win7/8 universe, and compatibility mode doesn't always work.

Okay,

since you want to insert more than one rows below or above....

Private Sub Worksheet_BeforeDoubleClick(ByVal Target As Range, Cancel As Boolean)

'add new rows below-above
Set Target = ActiveCell
Dim N As Long
Dim x As String, y As String
x = InputBox("please, select the number of rows", "add new rows", "2")
If x = "" Then Exit Sub
x = Val(x)
y = InputBox("please, select:" & vbCr & "Below = 1" & vbCr & "Above= 2", _
"add new rows above/below", "1")
If y = "" Then Exit Sub
Select Case y
Case Is = "1"
For N = 1 To x
Target.Offset(1).EntireRow.Insert shift:=xlUp
Next
Case Is = "2"
For N = 1 To x
Target.EntireRow.Insert shift:=xlUp
Next
End Select
Target.Offset(, 1).Select
End Sub
