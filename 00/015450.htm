%PROGRAMFILES%\Microsoft Office\Office12\scanpst.exe

... then hit <enter>. If you are running 32-bit Office 2007 on a 64-bit Windows 7 system, that location may be,

%PROGRAMFILES(X86)%\Microsoft Office\Office12\scanpst.exe


                   

You next have to locate your Outlook.pst file. The default location for your Outlook 2007 is:

%LOCALAPPDATA%\Microsoft\Outlook\Outlook.pst

The location is also noted in Outlook through Tools, Account Settings, Data Files.

You can paste the path and filename of your .PST file directly into the Enter the name of the file you want to scan box or click Browse and navigate through your folders to select it. Do not worry about Options as they only have to do with the log file. Click Start to initiate the process.

This takes a while scanpst diagnoses the .PST file. When it has completed its examination, you will be asked if you want to proceed with repair and if you want a backup copy made before repairs attempts. Answer Yes andYes to those questions.

The repair process will take some time as well and you will be notified with a popup dialog when it is complete. Close the scanpst program and start Outlook.
