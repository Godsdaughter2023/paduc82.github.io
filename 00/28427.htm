jay freedman say:
You need to insert a CREATEDATE field instead of a DATE field (see http://wordfaqs.mvps.org/DateFields.htm). You can change an existing DATE field to a CREATEDATE field by editing its field code, even months or years after the document was created, and it will show the correct date.

There is no built-in keyboard shortcut for a CREATEDATE field, but you can make one for yourself. In order to have something for the shortcut to act on, you first need to create a building block containing a CREATEDATE field.

  • In any document, edit a DATE field's code to make it a CREATEDATE field with the formatting you want, such as { CREATEDATE \@ "MMMM dd, yyyy" }. Press F9 to update the field and collapse its code.
  • Select the complete field (but nothing else).
  • Press Alt+F3 to open the Create New Building Block dialog.
  • Type a name for the building block, such as CreateDate.
  • Select a gallery, such as AutoText or Quick Parts.
  • Select a place to save the building block; probably Normal.dotm is the best choice.
  • Leave the other entries as is, and click OK.

Now you can assign a shortcut combination to the building block.

  • Click File, then Options, then Customize Ribbon. Click the "Keyboard shortcuts: Customize" button at the bottom of the dialog.
  • Near the bottom of the Categories list, choose Building Blocks.
  • In the Building Blocks list, choose the name you gave to the new building block. After the first few items, the rest of the list is in alphabetic order.
  • Click in the "Press new shortcut key" box, and press the shortcut you want to use. If that combination is already assigned to something else, the dialog will tell you, and you can decide whether to reassign the shortcut to your building block or to choose a different combination.
  • Click the Assign button to assign (or reassign) the combination to the building block. Then click the Close button, and OK the Options dialog.
  • When you shut down Word, if you're asked whether to save changes to Normal.dotm, say yes.
