Install Windows 7 OEM on a different computer

I have a computer with Windows 7 OEM installed. I want to erase Windows from this computer and install it on my main machine instead which currently has Windows Xp. I then want to install Windows XP (it's non-OEM) on the computer I erased windows 7 from as I will be donating it to a school - it's an older computer. So really I want to swap Windows XP non-OEM and Windows 7 OEM on my two computers. Would the activation work for Windows 7 or could MS provide permission manually if it didn't activate automatically considering I wlll be erasing it from the original machine?

bill smithers say:

You can't transfer the windows 7 OEM.

Microsoft won't change the licensing rules for you.

If the Laptop/Computer came with the Windows Operating System preinstalled (Dell, HP, Acer, etc) , that Product Key that came with the Laptop/Computer is for the preinstalled OEM License and is non-transferable.

The OEM License/Product Key (preinstalled Operating Systems on Dell, HP, etc machines) is tied to the first machine it is installed on and dies when that machine dies.

The above rule also applies to OEM System Builders Licenses.

Retail Operating System License, that you buy and install yourself, are allowed to be transferred to another machine.

But Retail Licenses can also only be installed on one machine at any particular time.
