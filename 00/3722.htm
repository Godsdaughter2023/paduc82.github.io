The closest I can suggest to a "shortcut" is to use the Alt accelerator sequence to navigate the ribbon.

While the cursor is in an equation, so the Equation Tools tab is visible on the ribbon,

  • Press Enter to use the selected template.

It's your call whether a (minimum) 5-keystroke "shortcut" is easier than using the mouse.
