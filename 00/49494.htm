net start wuauserv

net start cryptSvc

net start bits
net start msiserver

calgary.steven say:

Incorrect date and time of files when saved

Hello everyone,

My files always get incorrect date and time on drive K: when I save them, while I get correct date and time if I save them on local drive like C: and D:, for example, today is November 28th, 2009, I save a file ABC on D:, the date shows 28/11/2009, however if I save ABC on k:, the date shows 07/08/2009.

We have a small intranet which consists of 4 computers and another harddisk K: drive. I checked the date and time settings on 4 computers are all correct, and we use Windows Vista Ultimate version

I appreciate any help very much.

Steven

a. user say:

Because K has its OS, it has a system clock and that's what went bad for some reason.  Once you reset the system clock, the problem would be resolved as I suspected (as long as the hard drive remained powered up).  The question is why did the clock start showing the wrong date/time.  Did you have a power outage or did you turn off the power to the HDD?  When attached to a motherboard, the system clock has a very small battery that keeps time when the system is shut down.  I don't think that's the problem in this case because the hard drive is essentially operating on its own with no motherboard (but it could be the case if it was instead one of the computers).  It's possible that because it has an OS but no motherboard, the clock cannot retain its proper date/time when it is shut down but with AutoCAD it's getting the date/time from some other source on the network.  I think the solution to your problem is either to remove the OS from the drive (backup the data elsewhere, format the drive, and then restore the data - though you will have to reinstall any programs installed on the drive and attach it to a computer rather than directly to the router) or do not turn off the hard drive so it never loses power (and if it does, reset the clock).

I've never seen an external HDD with an installed and operating OS before.  This is definitely not the norm (though it's not forbidden but may have unique problems not generally discussed or known).  Most external HDDs contain no OS and are used for storage only (and get their date/time from the computer to which they are attached).  Of course, this would mean you'd need to attach the external HDD to one of the computers instead of directly to the router - and you said you didn't want to do that.  So retetting the clock when the HDD loses power may be your only option.

I hope this helps.

Good luck!

Notice popup about other users - Microsoft Community

ronik64 say:

Notice popup about other users

When I shutdown my computer, a notice sometimes appears in the lower left screen that warns me that if i shut down now, other users may lose their work. There should not be any other users. Does this mean my computer is hacked? 

a. user say:

The message means that one or more users forgot to sign out of their account.

To illustrate:  Say there are two user accounts on a computer. You start off in user account A then switch to user account B. Switching to B doesn't close A. User account A is still open and will stay open indefinitely. If you want to shut down the computer from user account B, you will be reminded that user account A is still open. You can still shut down the computer, but if A has any unsaved work it will be lost.

It's easy to see how many user accounts are on a computer. One way is to go to C:\Users. You should recognize each account. (Note: If one of the user accounts is a Microsoft account, the name of the account is truncated.)
