Value   # of Rows at Value

...  With values 10-4 calculating a # of rows to be updated in Step1 sheet.  The # of rows to be updated differs upon each run of this workbook as the total rows in Step1 is different.

Right now, I then have to manually go to Step1 and populate the first 3 rows with the value '10' in column F (first row of data is in row 5 so first cell to update is F5).  Then 3 rows later, I switch to enter '9' for the next six rows, in this example, etc.

tom ogilvy say:

GreenDriver LV

Assume the Value and # of Rows at Value data starts in A1 of LookUpTable so that 10 is in A1 of LookUpTable.  then you can use this macro.

 

Sub ABC()
Dim sh As Worksheet, sh1 As Worksheet
Dim r As Range, cell As Range, cell1 As Range
Dim j As Long, i As Long
Set sh = Worksheets("LookUpTable")
Set r = sh.Range("A1", sh.Range("A1").End(xlDown))
Set sh1 = Worksheets("Step1")
Set cell = sh1.Range("F5")
j = 0
For Each cell1 In r
   For i = 1 To cell1.Offset(0, 1)
     cell.Offset(j, 0).Value = cell1
     j = j + 1
   Next
Next
End Sub
