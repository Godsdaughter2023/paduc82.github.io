When you open Disk Defragmenter, did you notice that the system runs a Scheduled defragmentation, which is turned on by Default ?
It's at the top of the Disk Defragmenter's window.

Under Current Status, the drive shows when the scan was last run and the current percentage of fragementation of said disk.
So, you really do not need to do a manual scan of the system.

IF you are trying to run defrag with a browser open or any other programs/apps running that are constantly writing to the Hard Drive, you may receive a message that it can not be run when the HD is in use.

You can run both defrag and chkdsk from within an Elevated Command Prompt.
Click the Start orb, in the Search all programs and files field type in cmd
Right click cmd.exe and choose 'Run as administrator'. Agree to the UAC prompt.
