  10. Select all of the data and insert the table again so that the table includes all of your data.

    End With
    'Find the last used row in a column

tariqaslam say:

Hi

1 To start Setup Utility (BIOS), follow these steps: 1 Turn on or restart the computer and then press Esc while the “Press the Esc key for Startup Menu” message is displayed at the bottom of the screen now Press f10 to enter BIOS

If the laptop produce beep for around 10 to 20 seconds before screen shows, press Esc then only the system boots.

2 To return all settings in BIOS (Load Setup Defaults)  the values that were set at the factory, follow these steps: 1. Turn on or restart the computer, and then press esc while the “Press the Esc key for Startup Menu” message is displayed at the bottom of the screen. 2. Press f10 to enter Setup Utility. 3. Use the arrow keys to select Exit > Load Setup Defaults. 4. Follow the on-screen instructions. 5. To save your changes and exit, press f10

To change the boot order for clean installation or recovery 1. Restart the computer. 2. Press esc while the computer is restarting, and then press f9 for boot options. 3. Select Internal CD/DVD ROM Drive or USB drive from the boot options.
