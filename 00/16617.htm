One feature that might be of limited use to use is Shift+F5, which will take you back to the last three edit points. Beyond that, I think Track Changes might be your best bet. jay freedman say:
If you don't want to use Track Changes, or if you discover at Save time that you should have used Track Changes but didn't, there's another way.

Save the document to another name or location. Then use the Compare feature (also on the Review ribbon) to compare this version to the last save of the old version. The differences will look much like tracked changes, and you can either merge them into the older version or throw them away.
