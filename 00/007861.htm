In cell F6, enter this formula and copy down

=If(NOT(ISNUMBER(E5)),E6,F5+E6)

Hope this helps.

rich michaels - mvp (greatcirclelearning.com) say:
Hi Penny,

Before turning off Tracked Changes you have to Accept or Reject the changes and then save the document... you can save it to a different name if you want to keep the original with the marked changes. Or, make a copy first of the original document and Accept/Reject from the copy.

The indicators of the changes are held in the last paragraph mark of each section in the document. So another way to deal with this is to copy section by section but do not copy the paragraph mark where the section break is attached. In a one section document that would be the last paragraph of the document.

Turn on the none printing characters so you can clearly see each paragraph mark. The setting is on Word > Preferences > View and it also is usually on your Standard Toolbar. The icon is a backwards P.
And looks like this...¶

Hope this helps

john mcghie say:
Hi Penny:

You also need to read the Help on Track Changes.

You have turned off the DISPLAY of Tracked Changes.  You need to actually RESOLVE the changes.

To resolve them, use the Accept and Reject buttons to accept or reject each change.  (You can use "Accept all changes in document" to do the lot in one click).

Then the text will not only LOOK clean, it will actually BE clean.  The Accept/Reject commands physically remove the change marking from the text.

What you have done is uncheck the display options.  The markings are still in the text, but you can't see them.  I always recommend that people do not do that, because if you can't SEE the changes, you might forget they are there and email them to a customer, who WILL see them because they won't be hidden on his computer.

Hope this helps
