Copy Visible Cells only in Excel online - Microsoft Community

kniedler say:

Copy Visible Cells only in Excel online

Is it possible to copy only visible cells when using Office 365 Excel in edit in browser mode?
tina chen msft say:

Hi Kniedler,

 

When we copy and paste visible cells in a range of data that has hidden cells or filtering applied,  we’ll notice that the hidden cells are pasted along with the visible ones. We can’t change this when we copy and paste a range of cells in Excel Online because the option to paste only visible cells isn’t available.

 

If the data is formatted as a table with filtering applied, we can copy and paste only the visible cells.

 

As a workaround, we can open the workbook in Excel desktop application to copy and paste visible cells only.

 

You can refer to the detailed information in the following article:

Why can't I copy and paste visible cells only?
