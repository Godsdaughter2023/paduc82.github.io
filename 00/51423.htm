Check disk is a command-line tool that checks volumes on your hard disk drive for problems. The tool then tries to repair any problems that it finds. For example, Chkdsk can repair problems related to bad sectors, lost clusters, cross-linked files, and directory errors.

Steps to perform check disk:
a) Press” windows + R” key to open run window

The options for Check Disk are used as follows:
/F command + Name of the drive you want to check (e.g: /f c) Fixes errors on the disk
/V command + Name of the drive you want to check Displays the full path and name of every file on the disk (FAT16 and FAT32); displays cleanup messages if any (NTFS)
/R command + Name of the drive you want to check Locates bad sectors and recovers readable information
/X command + Name of the drive you want to check Forces the volume to dismount first if necessary (implies /F)
/I command + Name of the drive you want to check Performs a minimum check of index entries (NTFS only)
/C Skips checking of cycles within the folder structure (NTFS only)
/B command + Name of the drive you want to check Re-evaluates bad clusters on the volume (NTFS only; implies /R)

    Check the Power mode slider.

    • On applicable models, the Power mode slider is found by clicking the battery icon on the Windows taskbar. For best battery life, the slider should be set to Recommended. If you need to use the device for an extended period of time while on battery, the slider should be set to Battery Saver. 

    Run Power troubleshooter.

    • Surface devices have a configurable Power mode defines the preference between Best battery life and Best performance. The setting is configured separately for when the device is plugged and when it is on battery. This allows to set the device to Best performance while on PSU and for it to automatically change to a different setting, like Recommended, while on battery.Open Settings > Update & security > Troubleshoot > Power and then select Run the troubleshooter. 

    Check Battery saver Settings. 

    • This troubleshooter will automatically detect and fix these potential issues Open Settings > System > Battery. Check that Turn battery saver on automatically is enabled and that it is set to at least 20%. Check that Lower screen brightness while in battery saver is also enabled. 

    Check App usage.

    • Unnecessarily running programs can cause slow system behavior and affect overall stability of the PC. Open Settings > System > Battery and then select See which apps are affecting your battery life. From Time period, select 6 hours, 24 hours, or 1 week depending on which period is closest on reported problem. From Show, select Apps with usage. Looks for apps that have high battery consumption. If there are apps with high usage that is not using, should close the app. If the apps start automatically, open Task Manager and then select the Startup tab. The customer can disable any apps that they don’t need to start automatically.  

    Choose or Customize a power plan. 

    Tap the Battery icon on the taskbar > choose Power & sleep settings > Related settings > Tap Additional power settings > Tap Create a power plan > Under Create a power plan start with an existing plan and give it a name. (  Automatically balances performance with energy consumption on capable hardware). > click next > Choose the sleep and display settings that you want your Surface device to use.

    • Command Prompt > Choose Command Prompt (Admin) from the menu. Note that this must be the Admin version and not a regular command prompt. 
    • Yes to UAC prompt > A prompt will appear to which you need to give permission.
    • Command > Copy and Paste powercfg/batteryreport/output "C:\battery_report.html" into the command prompt window. Note that for the pros you can designate the output to any directory. For simplicity, we are placing it in the C:\ root folder.
    • Open file > Using Explorer navigate to C:\ aka the root directory. You should see a file labeled battery_report.html. Double-click on it to open the file in your default browser e.g. Microsoft Edge.
