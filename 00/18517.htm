The reason is that Rows.Count * Columns.Count did not fit into a Long, which is the data type of Range.Count

  Set Target = Intersect(Target, Me.UsedRange)

  If Target Is Nothing Then Exit Sub

    'Whatever

jllatham say:

Hang in there.  Sounds like you're one of those who learns much better from visuals than from the written word, and that does make it tougher than usual to learn from a book.

At the risk of adding to your confusion, I'm going to offer my own definitions of Subroutines and Functions.  They're a little different than were offered earlier.

A sub starts with its definition:

Sub NameOfSub()

or can be hidden from view as

Private Sub NameOfSub()

But you can pass variable parameters to a sub, as:

Sub AddTwoNumbers(num1 As Integer, num2 As Integer)
  'takes the two numbers passed to it, adds them togehter and
  'always places the result into cell A5 on the active sheet:
  Range("A5") = num1 + num2
End Sub

You could call it in a couple of ways, depending on where you are calling it from, either as:

AddTwoNumbers 6, 5

or

Run"AddTwoNumbers",6, 5

you could even Call it instead of Running it.

Meanwhile a Function starts with its definition, either as:

Function AddTwoIntegers(num1 As Integer, num2 As Integer) As Integer
  'adds two integers and returns the integer result of the addition
  AddTwoIntegers = num1 + num2
End Function

Then you can put that in a worksheet formula if you wanted to like:

=AddTwoIntegers(3,4)
or with cell references to the 2 numbers
=AddTwoIntegers(A5, B7)

and the cell will show the result of the addition.  You could use it in VBA with a statement something like:

myNumber= AddTwoIntegers(3,9)
of course the 3 and 9 could be variables or constants that you'd already put some value into.

 

Neither a Sub nor a Function requires any parameters, but 99% of the time a Function will have some parameters to pass to it.  An example of a built-in worksheet function that takes no functions would be Pi().  Type  =Pi() into a cell and it will return the value 3.14159265358979
