HELP NEEDED-Counting Merged Cells - Microsoft Community

excel ella say:

HELP NEEDED-Counting Merged Cells

HELP?!  Does anyone know how to create the following 2 formuals?

1.  Count merged cells (comprised of 3 cels) in a column containing both numbers and text

2.  Count blank merged cells (comprised of 3 or 2 cells) in a column.

danielco say:

It's not quite clear for me if you want one total for both columns.

Cell B35 contains the total of cells merged by three and cell B36 the total of cells merged by two. I supposed text and number are separated by a blank character :

Sub test1()

Dim TestNum As Boolean, TestAlpha As Boolean, Ctr1 As Integer, Ctr2 As Integer

For i = 1 To Selection.Columns.Count

    For Each c In Selection.Columns(i).Cells

        Var = c.Address

        If c.MergeCells Then

            If c.MergeArea.Count = 3 And c <> "" Then

                TestNum = False

                TestAlpha = False

                For Each Item In Split(c.Value, " ")

                    If IsNumeric(Item) Then

                        TestNum = True

                    ElseIf Item <> "" Then

                        TestAlpha = True

                    End If

                Next Item

                If TestNum = True And TestAlpha = True Then Ctr1 = Ctr1 + 1

            End If

            If c.MergeArea.Count = 2 And c <> "" Then

                TestNum = False

                TestAlpha = False

                For Each Item In Split(c.Value, " ")

                    If IsNumeric(Item) Then

                        TestNum = True

                    ElseIf Item <> "" Then

                        TestAlpha = True

                    End If

                Next Item

                If TestNum = True And TestAlpha = True Then Ctr2 = Ctr2 + 1

            End If

        End If

    Next c

Next i

[B35].Value = Ctr1

[B36].Value = Ctr2

End Sub
