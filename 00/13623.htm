Multiple files to use same lookup file - Microsoft Community

nhjjoubert say:

Multiple files to use same lookup file

Hi, I have a number of employees each with their own excel file to enter time. Each file have an activity lookup table. There are 2 columns in the lookup table, Code & Activities. One code can have multiple activities to choose from. I use indirect() in data validation to filter the data based on the code selection. My problem is that when the codes change or new one added, I have to add it to all the user files. And Excel does not allow me to do this kind of lookup form another file. Is there any way I can do this differently to allow a one file lookup? Any help would be much appreciated. Thanks. 

teylyn say:
Hello,

have one Excel master file with the lookup data.

In each individual user file, create a new sheet called Lookups and reference the master file content with a formula. 

Now the lookup data is in the user file and you can build your data validation based on the data in the same file.  

The formula to grab the data can be as simple as 

=[Master.xlsx]Sheet1!A1

(while the Master file is open). Copy down as many rows as needed for the current data set and any possible additions. If you have 10 rows of data in the Master file and copy the formula down 20 rows, the last rows will show a 0.

Build the range names for the data validation with a formula like this:

=Lookups!$A$1:INDEX(Lookups!$A:$A,MATCH("zzzz",Lookups!$A:$A,1))

This is a dynamic formula and will return all text rows. 

Now you can edit the Master file data, add more rows, delete rows, edit cells. As soon as a user opens her individual file, the Lookup sheet will be updated with the Master data and the dynamic range names will reflect the data on the lookup sheet.
