Inserting Multiple Images in Order - Microsoft Community

outsidaz say:

Inserting Multiple Images in Order

 

I have the following Macro to insert multiple images into a document all at once with a multi-select File Dialog box.  However, I cannot control the order that the images get inserted.  It appears that it adds the last image selected first, and then the rest.  Is there a way to sort based on file name?

 

Sub InsertMultAtt()

'

' Insert Picture Macro

' Macro created 1/20/2011 by wanga22

'

'Declare a variable as a FileDialog object.

Dim fd As FileDialog

 

'Create a FileDialog object as a File Picker dialog box.

Set fd = Application.FileDialog(msoFileDialogFilePicker)

 

'Declare a variable to contain the path

'of each selected item. Even though the path is a String,

'the variable must be a Variant because For Each...Next

'routines only work with Variants and Objects.

Dim vrtSelectedItem As Variant

 

'Add a filter that includes GIF and JPEG images and make it the second item in the list.

With fd

    .Filters.Add "Images", "*.gif; *.jpg; *.jpeg; *.png; *.bmp"

    .Filters.Add "Images", "*.*"

 

'Sets the initial file filter to number 2.

    .FilterIndex = 2

 

'Use the Show method to display the File Picker dialog box and return the user's action.

'If the user presses the action button...

If .Show = -1 Then

'Step through each string in the FileDialogSelectedItems collection.

    For Each vrtSelectedItem In .SelectedItems

'vrtSelectedItem is a String that contains the path of each selected item.

    Selection.InlineShapes.AddPicture FileName:= _

                  vrtSelectedItem _

                , LinkToFile:=False, SaveWithDocument:=True

    Selection.ParagraphFormat.Alignment = wdAlignParagraphCenter

 

'add a section break

    ActiveDocument.Characters.Last.Select

    Selection.Collapse

    Selection.InsertBreak Type:=wdSectionBreakNextPage

    ActiveWindow.ActivePane.View.SeekView = wdSeekCurrentPageHeader

    Selection.HeaderFooter.LinkToPrevious = False

    ActiveWindow.ActivePane.View.SeekView = wdSeekCurrentPageFooter

    Selection.HeaderFooter.LinkToPrevious = False

    ActiveDocument.Characters.Last.Select

    Next vrtSelectedItem

 

'If the user presses Cancel...

        Else

        End If

    End With

 

'Set the object variable to Nothing.

    Set fd = Nothing

 

End Sub

 
doug robbins - mvp office apps & services (word) say:

Oops, WordBasic.SortArray will not sort an array that is declared as a Variant.  Therefore use the following instead

Dim strPics As String
Dim arrPics() As String
Dim i As Long
Set fd = Application.FileDialog(msoFileDialogFilePicker)
Dim vrtSelectedItem As Variant
With fd
  .Filters.Add "Images", "*.gif; *.jpg; *.jpeg; *.png; *.bmp"
  .Filters.Add "Images", "*.*"
  .FilterIndex = 2
  If .Show = -1 Then
      For Each vrtSelectedItem In .SelectedItems
          strPics = strPics & "|" & vrtSelectedItem
      Next vrtSelectedItem
      strPics = Mid(strPics, 2)
      arrPics = Split(strPics, "|")
      WordBasic.SortArray arrPics()
      For i = 0 To UBound(arrPics) ' - 1
           MsgBox arrPics(i)
          Selection.InlineShapes.AddPicture filename:= _
                arrPics(i), LinkToFile:=False, SaveWithDocument:=True
          Selection.ParagraphFormat.Alignment = wdAlignParagraphCenter
          'add a section break
          ActiveDocument.Characters.Last.Select
          Selection.Collapse
          Selection.InsertBreak Type:=wdSectionBreakNextPage
          ActiveWindow.ActivePane.View.SeekView = wdSeekCurrentPageHeader
          Selection.HeaderFooter.LinkToPrevious = False
          ActiveWindow.ActivePane.View.SeekView = wdSeekCurrentPageFooter
          Selection.HeaderFooter.LinkToPrevious = False
          ActiveDocument.Characters.Last.Select
      Next i
  Else
      MsgBox "User pressed Cancel."
  End If
End With
Set fd = Nothing
 -- Hope this helps.
