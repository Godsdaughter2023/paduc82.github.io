cody c. say: Hey Paula,

A codec is a "translator" in the video world. It allows a video player to properly read and decompress a specific video format.

However, it seems you've solved most of your question on your own. The video needs to be able to play in Quicktime before it will play in PowerPoint. Simply put, PowerPoint uses Quicktime to play videos. In most cases, if it doesn't work in one, it won't work in the other.

There's a free codec package called Flip4Mac that adds several codecs to Quicktime, specifically AVI and WMV. You can download it from the following link:

Windows Media® Components for QuickTime

Once you've downloaded and installed that, Quicktime and PowerPoint should be able to properly play your AVI video.

bernd p. say:
I have copied what I think is the shortcut, but when I click on the shortcut on my XP computer, it says that OneNote cannot open the specified location.  
I'm not sure what shortcut you're talking about, but here are some facts:

  • A Onenote notebook is realized as a folder (not a file) in the Windows file system
  • So a shared notebook is a folder under the Windows shared folder
  • You should open it with Onenote using File, Open (not with a shortcut, notebooks should stay open in Onenote)
  • You must have write access to the Windows share
  • For easy access you should use the same user + password on both computers
