First rule: You can not have two files with the same name in a one specific folder.  Never could in any computer system.  It would be logically impossible. 

The correct Windows terms are Copy and Move.

It is possible and logical to have two files of the same name in different folders.  It may not be good practice because it can become quite confusing.  Exception of course is in making backups.

I suggest you adopt the practice of using the right-button to drag and drop files from one place to another.  When you let go of the right button, you will get a menu.  One choice is copy, another move and cancel.  This practice gives you complete control and the option to change your mind.

System File Checker will not "fix" this because it is basic logic
