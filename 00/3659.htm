betty gui msft say:

Hi JJentz,

 

You can schedule an email in desktop Outlook.

 

If you want to schedule an email from Outlook for Windows, you can achieve it by the following steps:

 

1. Compose a new Email.

2. Click More options arrow in the Tags group.

3. Select Do not deliver before check box under Delivery options, then choose the date and time you want to deliver message.

4. Click Close.

5. When you finished the message, click Send. The message will be saved in Outbox folder until the specified date and time.

 

If you want to schedule an email from Outlook for Mac, please follow below:

 

1. Compose a new email.

2. Navigate to Send > dropdown arrow > Send later, and enter the date and time you want.

3. Click Send. The message will be saved in the Drafts folder until the specified date and time.

 

Note: The feature will take effect when Outlook for Mac is closed but not when Outlook for Windows is closed.

rich007a say:

Method 3, which will enable you to do stuff with the Excel file from within Access's VBA:

i) Set a reference to "Microsoft Excel 12.0 Object Library"  via VBE > Tools > References.

ii) Run the following code from Access's VBA:

 

Sub UseExcelFromAccess()
' This procedure requires a reference be set to
' Microsoft Excel 12.0 Object Library
' via VBE > Tools > References.
'
    Dim XL As Excel.Application
    Dim bWeStartedExcel As Boolean
    Dim xlWb As Excel.Workbook
    
     'Get Excel if it's running:
    On Error Resume Next
    Set XL = GetObject(, "Excel.Application")
    If Err <> 0 Then
        On Error GoTo ExitPoint
         'Excel wasn't running, start it from code:
        Set XL = CreateObject("Excel.Application")
        bWeStartedExcel = True
    End If
    On Error GoTo ExitPoint
   
     'Open Excel file from Template ( use XL instead of Application):
    Set xlWb = XL.Workbooks.Add("C:\temp\TemplateName.xltm")

     'Do things with the Excel workbook like this:
    xlWb.Sheets(1).Range("A1").Value = "Hello from Access"

ExitPoint:
    On Error Resume Next
    If bWeStartedExcel Then XL.Quit
    Set xlWb = Nothing
    Set XL = Nothing
End Sub


Note, I don't have Access, so this isn't tested, but the same principal works for the other Office Applications, so I think it should work.

Hope that helps.
