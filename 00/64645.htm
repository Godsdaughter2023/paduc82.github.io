david. f say:

Hello, Narnian

You could create a second user account and change the auto run keys in the registry to run per-user – this would be a manual process.

There are two types of startup that Windows uses, one that starts programs for everyone and another is per-user.
You can leave the applications wanted for both users alone, but if you want a program to start only for a specific user you need to copy the auto run information from the “everyone” location to the per user location.


To do this, first create the second user:Create a user account

Go to Start, type  regedit in the search box and press enter. In the registry editor, browse to:

HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Run

You will see a list of startup items, this is a list of programs that startup for everyone. You will need to make two lists of startup items, one for each user.

Double click each item you want to change and copy both the value name and value data into Notepad (or another program). So that both users will be able to access this file, save it in C:\Users\Public\Documents

 

Next, make sure you are logged into the user you want to set a startup item to (to set User1’s startup items, login to User1).
In the registry editor, browse to:

HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Run

Create a new key by going to Edit, New, String Value. A new item will be created, set the name of the item then double click to change the value data.
The name and data will be from the copied information from earlier.

 

Login to the other user and open the file repeat the steps for that user.

 

WARNING:Using the Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use the Registry Editor at your own risk.

    net stop bits

    net stop msiserver

    ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

    net start wuauserv

    net start cryptSvc

    net start bits

    net start msiserver

  5. Type Exit in the Command Prompt to close it.
