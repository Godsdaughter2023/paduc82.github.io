How to extract first and last name - Microsoft Community

christiano pietro say:

How to extract first and last name

I have a field in a query named "NameD" , it contains the full names of the employees, how can i extract the first and last name only?

john w. vinson say:

Simple answer (assuming that NameD contains text like "John Doe"):

FirstName: Left([NameD], InStr([NameD], " ") - 1)
LastName: Mid([NameD], InStr([NameD], " ") + 1)

Real answer: Don't store names like that! Store them as FirstName, LastName, and perhaps MiddleName and Nickname  as well. What about names like "Mary Jo Smith"? Her first name is Mary Jo, just ask her. What about "David Bell Smith"? His first name is Dave, or if he's being formal David, but only his mother calls him David Bell, and that only when she's really mad at him. Then there's "Hans de Groot" - his first name is Hans (not Hans de!) and his last name is "de Groot". Parsing this kind of name data often requires a USB (Using Someone's Brain) interface, and even then is subject to mistakes; you'll sometimes have to ask the person concerned how they prefer to be addressed.

ken sheridan say: The following functions might help:

Public Function FirstName(varFullName)

    If Not IsNull(varFullName) Then
        On Error Resume Next
        FirstName = Left$(varFullName, InStr(varFullName, " ") - 1)
    End If
    
End Function

Public Function MiddleName(varFullName)

    Dim intFirstSpace As Integer, intSecondSpace As Integer
    
    If Not IsNull(varFullName) Then
        intFirstSpace = InStr(1, varFullName, " ")
        
        If intFirstSpace > 0 Then
            intSecondSpace = InStr(intFirstSpace + 1, varFullName, " ")
            If intSecondSpace > 0 Then
                MiddleName = Mid$(varFullName, intFirstSpace + 1, intSecondSpace - intFirstSpace)
            End If
        End If
    End If
    
End Function

Public Function Lastname(varFullName)

    If Not IsNull(varFullName) Then
        On Error Resume Next
        Lastname = Mid$(varFullName, InStrRev(varFullName, " ") + 1)
    End If
    
End Function

Parsing names tends to fall down with names like Victoria de los Angeles however.  The above would return her last name as 'Angeles' and her middle name as 'de', whereas I imagine most people would regard her last name as being 'de los Angeles'.
