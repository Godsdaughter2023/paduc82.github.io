mroset say:

Hi All,

I also experienced this issue after upgrading from my Dutch Windows 7 Pro x64 to Windows 10 Ultimate x64.

Replacing the ownership on %appdata% (as suggested by Abdul_Malik) did not work for me, but a variant on that did...

I analyzed the problem further and found that the 'Credential Manager'  Windows Service could not be started and produced error 5 - Access Denied.

In Windows Mail this resulted in the notorious "Sorry, we were unable to do that. Error: 0x80070425." error.

There were other symptoms like being unable to view the Web nor Windows Credentials.

Also, accessing Settings/Accounts/Login options (Dutch: Aanmeldingsopties) was no longer possible in my case.

Then, by using ProcMon, I found that ACCESS_DENIED results where produced for accessing the following folders in which the credential manager stores our web and windows passwords:

Dir 1.  C:\ProgramData\Microsoft\Vault

Dir 2.  C:\ProgramData\Microsoft\Vault\AC658CB4-9126-49BD-B877-31EEDAB3F204

Further analysis of the NTFS properties on these folders learned that NO permissions where assigned to these folders (Should be SYSTEM, Administrators and Users which I checked on a properly functioning reference Win 10 system).

I then successfully resolved the issue (of being unable to start the 'Credential Manager' service) as follows:

  1. Inherit permissions from the parent directory for the 1st directory
  2. On the 2nd directory, change ownership of the folder to another administrative user (mroset in my case) and indicate that the owner must be changed also for the derived/child folders/files
  3. Inherit permissions from the parent directory for the 2nd directory
  4. On the 2nd directory, change ownership of the folder BACK to the original owner (Administrators) and indicate that the owner must be changed also for the derived/child folders/files

==================

After those actions, I was able to start the 'Credential Manager' service and all mentioned problems were resolved !!

Took quite a number of hours to find the culprit, but sweet victory was my reward.
