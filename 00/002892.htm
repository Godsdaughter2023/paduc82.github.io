techfreak_ say:
Skype/Microsoft does not provide a dedicated export/download message history utility for the new Skype clients (8.x/12.x). The current versions of the Skype 12.x App store messages locally in a skype.db file. If the older messages are currently present in that file, they can be viewed/searched/extracted with a SQLite Database Viewer (from the Messages table, timestamps are in Unix time format). If the older messages are not present in that file (deleted or alternate computer), they can only be downloaded using the newer Skype clients by manually scrolling up through the history. Scrolling up through the history will initiate incremental downloads from the Skype/Microsoft cloud for a period not to exceed the up to 2 years of storage. This process will incrementally populate data into the skype.db file. As far as I can tell, this option is not available in the classic Skype client (7.x).
