Hi Eddy,

When you select a picture in your document, a contextual Format tab opens on the ribbon. The Format tab contains controls for resizing. You can also "Right Click" on the image and from the Contextual Dialog that opens, you can select the Size and Position option.

If you have many pictures to resize, then I recommend our free AuthorTec Resize add-in. There is also a free AuthorTec Insert add-in that you can use while you are inserting pictures into the document and it has a sizing function built right into it.

Finally, be aware that inserting pictures into documents really swells the size of the document. If a small file size is important, then the best way is to size the pictures using a graphics program before you even insert them.

pete_uk say:

Here's a UDF that you can use:

 

Function count_formulae(rg As Range)
    For Each c In rg.Cells
        If c.HasFormula Then count_formulae = count_formulae + 1
    Next c
End Function

 

Use it in the sheet like:

 

=count_formulae(A2:F2)

To help troubleshoot error messages and other issues, you can start Windows by using a minimal set of drivers and startup programs. This kind of startup is known as a "clean boot." A clean boot helps eliminate software conflicts.

How to perform a clean boot in Windows
