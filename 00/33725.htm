    It's hard to diagnose BSODs like that because it could be hardware or software, and those error messages aren't too informative.

    What I'd recommend is the System File Checker, to see if Windows needs any system files repaired:
    - Click Start and type cmd.
    - Right-click Command Prompt and run it as administrator.
    - Type sfc /scannow
    - Press Enter.

    If that doesn't help, you could try a system restore to a point before the crashes began:

    https://support.microsoft.com/en-au/help/12415/...

    If all else fails, a reset of your PC usually helps in cases like this. the same link above has options for resetting, and you can keep your files. Your apps will need to be reinstalled though.
