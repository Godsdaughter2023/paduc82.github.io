ossiemac say:

Hello Jorge,

The following code will set the timer provided that A1 contains a valid time in "h:m:s" format.

Public nextCalc As Date     'Must be at top of standard module before any Subs

Sub CalculateAtIntervals()

    Application.Calculation = xlManual  'Optional if already set to manual
   
    Calculate
    If Worksheets("Sheet1").Range("A1") > 0 Then
        nextCalc = Now + Worksheets("Sheet1").Range("A1")
        Application.OnTime nextCalc, _
            Procedure:="CalculateAtIntervals"

    Else
        'Do not set the next cycle
        Application.Calculation = xlCalculationAutomatic    'Optional
    End If

End Sub

To stop the timer just delete the value in cell A1.

Alternatively the following code, if run, will stop the timer without deleting the value in A1.

Sub StopTimer()

    'Cancel the scheduled procedure execution.
    On Error Resume Next    'Code will error if timer not previously set
    Application.OnTime _
        EarliestTime:=nextCalc, _
        Procedure:="CalculateAtIntervals", _
        Schedule:=False '

    Application.Calculation = xlCalculationAutomatic    'Optional

End Sub
