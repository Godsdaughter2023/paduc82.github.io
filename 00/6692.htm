suzanne s. barnhill say:

The list of paper sizes on the Paper tab of Page Setup is provided by the printer driver. By default (unless and until you select a different one) the active printer in each Word session is the printer you have selected as the Windows default. Check to make sure that the correct printer is still selected as the default and showing as the active printer in Word.

To change the printer in Word, press Ctrl+P to open the Print dialog. Note what printer is selected. You can click the dropdown list to see other available printers and select a different one.

To change the default printer, go to Start | Devices as Printers. The default printer will be indicated by a check in a green circle. If your desired printer is not selected as the default, right-click on it and choose "Set as default printer."
