MS Excel does not have a date format of its own.  It picks up the date format from the Windows machine on which it is installed.  If the Windows machine ha s mm/dd/yyyy setting in Control Panel > Regional Settings, then all dates in MS Excel will be in mm/dd/yyyy format.  To change the format to dd/mm/yyyy permanently, one will have to change this setting in Windows.

Hi Dick and CheriYoung,

You most probably signed in to Skype for Windows 10 application using your Microsoft account that is why you see only your Microsoft contacts. When you sign in, please use your Skype name and not an email address if you originally use a Skype name to create your account. To sign out from your account on Skype for Windows 10:  

  • Click your profile icon at the bottom left on the window.
  • Select Sign out.
  • Restart Skype and select Use a different account when signing in again.

Eudora filtering very slow with MSE - Microsoft Community

gh-upnorth say:

Eudora filtering very slow with MSE

Since I've switched to MSE (WinXP), email filtering in Eudora 7 has slowed dramatically.  When I was using AVG Free, the email scanner popped up during email capture, but it was fast.  Now, with MSE, capturing email seems fast, but filtering is laboriously slow.  MSE must be checking the new email files during the filtering process.  MSE is great otherwise.  Any suggestions for speeding this up?
dr. strangelove say:

MSE will only be checking attachments of email.  It is more likely that you have remnents of AVG on your system.

Verify that you have removed all other realtime antimalware products (Norton, McAfee, TM, etc). Use the removal tools here:

List of anti-malware program cleanup/uninstall tools

If you just updated Adobe Flash, you may have inadvertently installed McAfee Security Scanner.  Verify that it is not present.

Verify that Windows Defender is uninstalled (if XP), or Disabled (if Vista/W7).

Verify that you are up-to-date with patches and service packs.

Verify that your PC clock is correctly set.

Verify that you are using build 1961 of MSE (as of April 2010).

Verify that the Windows Firewall is on, and set to defaults.

In IE, reset all security zones to default level.

 

Then, restart your PC.

If the problem persists, submit a support case to Microsoft:  https://support.microsoftsecurityessentials.com/?scrx=1
