cindy meister say:

Hi T.

I don't find any way to change the default. You can use a macro to force a new equation you're inserting to be inline, and assign that to the QAT or create a custom group and assign it to a button in that. The macro:

 Sub EquationInsertNew()
    Dim rngOM As Word.Range
    Dim om As Word.OMath
   
    Set rngOM = ActiveDocument.Content.OMaths.Add(Selection.Range)
    Set om = rngOM.OMaths(1)
    om.Type = wdOMathInline
 End Sub

But I don't find any way to change how pre-fabricated equations can be inserted in-line.
