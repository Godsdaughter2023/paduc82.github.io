Automatic compression of folder/files in Windows is normal. Windows update makes sure that the space requirements are met before any critical updates are installed. To manage the free space, Windows update compresses the files/folders as per the space requirements, but they are restored back to their original state once the updates are installed.

  3. Enter key.

    fsutil behavior set disablecompression 1

    Reboot the computer after the successful execution of the command and check the status.

     

    If you are on Windows 10 Pro, you can disable automatic compression by applying group policies to file system.

     

  1. Open the Run dialog box by pressing the Windows key and the R key together.

  2. Type GPEDIT.MSC and hit the Enter key.

  3. Group Policy Editor, navigate to Computer Configuration>Administrative Templates>System>Filesystem>NTFS.

  4. Do not allow compression on all NTFS volumes.

  5. Enabled radio button, then select OK.
