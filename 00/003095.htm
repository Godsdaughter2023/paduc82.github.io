MS Project - IIF Formula with Duration - Microsoft Community

chase710 say:

MS Project - IIF Formula with Duration

I am having an issue with a couple IIF formulas is MS Project.

1 - I am trying to create a formula that returns a flag when the duration is larger than 15 days. The formula I have is below:

IIf([Duration]>15,Yes,No)

I am having an issue finding the syntax for "days". When I use that formula it returns a flag on all tasks with durations other than milestones with 0 day durations.

2 - I want to create a formula that returns a flag if a duration is missed. For example when "1 day?" is in the duration column. I'm not sure how to do this one due to the "?" and not knowing the "days" syntax.

I know an filter can be done for all this, but I am trying to make a macro to automate our QC checks when reviewing other PM schedules. Any help that can be provided would be greatly appreciated.

tom_bpc say:

Chase,

Try these in custom flag fields:

  1. IIf([Duration]>15*[Minutes Per Day],True,False)
  2. IIf([Estimated],True,False)

Durations are stored as number values with units of minutes, so you need to convert them to days as shown.

The second formula just looks for an estimated duration (the question mark).

Note that these functions work in custom field formulas.  A vba (macro) approach would be different. 

Good luck, tom
