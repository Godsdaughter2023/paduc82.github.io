My challenge is that I need my first DV item to be a blank, because I need to allow users to blank out a previous selection (without having to train them all to use the delete key)

It appeared that "Choices" was split by the DV functionality based on the commas, so I tried

and the DV list just gives the latter two options, no blank row

Edit: apparently copy/paste code from a website isn't a great idea. Here it is, cleaned up:

 

Sub AddDV()

    Dim Choices As String
    Choices = "1. Choice1, 2. Choice2, 3. Choice3"

    Sheet6.Range("A1").Select

        With Selection.Validation
        .Delete
        .Add Type:=xlValidateList, _
        AlertStyle:=xlValidAlertStop, Operator:= _
        xlBetween, Formula1:=Choices
        .IgnoreBlank = True
        .InCellDropdown = True
        .InputTitle = ""
        .ErrorTitle = ""
        .InputMessage = ""
        .ErrorMessage = ""
        .ShowInput = True
        .ShowError = True
    End With
End Sub

Here is one approach.  The first thing you have to do is define a new table that relates the existing status codes to the value amount to yield the new status codes.  The table will have four columns, as below:

 

2010 Status Names >500 Status >0 Status No Sale Status

Once you enter the data for the table, select all those cells, go to "Insert" and select "Table".  Check the box that says "My table has headers...".  Click OK.  Click on any cell in the table.  Go to the "Design" tab, and enter the name "StatusTable" in the "Table Name:" input box.

 

Now that you have defined the table, in cell C2, enter this formula:

 

=OFFSET(StatusTable,MATCH($A2,StatusTable[2010 Status Names],0)-1,IF($B2=0,1,IF($B2<500,2,3)),1,1)

 

This will find the correct status for 2011 based on the 2010 status (in Column A) and the amount of sales in 2010 (in Column B).  You will have to adjust the formula if your data are in different columns.  Copy and paste the formula down the rest of the rows in that column to get the results for all of your customers.

In October 2018, we announced several important changes for Office 365 subscriptions. As part of these changes, there is no longer a limit to the number of devices where you can install Office.
