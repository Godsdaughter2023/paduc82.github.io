bill_in_az say:

Macro to Copy Paste Only Ungrouped and Visible Cells in Excel

 After it has been updated with the latest data, I currently have a macro that will copy and paste the current 3 week period to a "dashboard".

Currently, I do this by directly referencing the range of cells which requres me to change that range weekly. Is there another way to copy and paste without having to directly call out the range so that I do not have to update the macro each week?

rick rothstein, mvp - excel say:

Your question is a little hard to answer specifically without knowing more about your layout and without seeing your macro; however, you might be able to use a concept similar to this in your code. Assuming your data area starts in Row 2 (you might have header pre-positioned to the end of your future needs, so I am going to work around them) and that there are no formulas in the data area that need to be accounted for, you can calculate the last data column and last data row like this...

Dim Data Range As Range, LastUsedRow As Long, LastUsedCol As Long, Last21ColsOfData As Range
........
........
Set DataRange = Intersect(ActiveSheet.UsedRange, Range("A2", Cells(Rows.Count, Columns.Count)))
LastUsedRow = DataRange.Find(What:="*", SearchOrder:=xlRows, SearchDirection:=xlPrevious, LookIn:=xlValues).Row
LastUsedCol = DataRange.Find(What:="*", SearchOrder:=xlByColumns, SearchDirection:=xlPrevious, LookIn:=xlValues).Column

Then you can form the range for your last 21 columns of data like this...

Set Last21ColsOfData = Range(Cells(2, LastUsedCol - 20), Cells(LastUsedRow, LastUsedCol))

daniel g. grau say:

To complement John's answer in case you don't know what VBA means, do this:

  • In Word's main menu, select Tools > Macro > VBA Editor to enter the editor
  • In the editor's main menu, select View > Immediate Window
  • Type (or paste) the following command and press Return:
  • print Application.StartupPath
  • Make a note of the result (or copy it to the Clipboard), then press Command Q to exit the editor

The result will be something like this:

Macintosh HD:Applications:Microsoft Office 2011:Office:Startup:Word

except that Applications may appear in the OS X local language and Startup in Word's localized language.
