bernie deitrick say:

If the values in any of those cells are actually blank, they will remain blank - "text to columns" ignores blank cells within the range, so there must be a 0 value in the cell.  You could try, for example:

    With Range("V:V")
        .Replace What:="0", Replacement:="", LookAt:=xlWhole
        .TextToColumns Destination:=Range("V1"), DataType:=xlDelimited, _
            FieldInfo:=Array(1, xlYMDFormat)
        .NumberFormat = "m/d/yyyy"
    End With
