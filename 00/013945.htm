Caveat:  Change parameter separators from comma to semicolon.  Read period as the decimal point (comma, for you).  Translate English function names appropriately.

Weighted mean (10.5094850948509), in C2:

=SUMPRODUCT(A2:A53, B2:B53) / SUM(B2:B53)

Weighted sample variance (54.9951543537174), in C3:

=SUMPRODUCT((A2:A53 - C2)^2, B2:B53) / SUM(B2:B53, -1)

Weighted sample standard deviation (7.41587178649398):

=SQRT(C3)

Weighted median (10):

=weightedMedian(A2:A53, B2:B53)

where weightedMedian is the following VBA code in a normal module (not a worksheet object):

Function weightedMedian(v As Range, cnt As Range)
Dim n As Long, k As Long, i As Long, j As Long
n = WorksheetFunction.Sum(cnt)
ReDim vdata(1 To n) As Double
k = 0
For i = 1 To v.Rows.Count: For j = 1 To cnt(i)
        k = k + 1
        vdata(k) = v(i)
Next j, i
weightedMedian = WorksheetFunction.Median(vdata)
End Function

Is the VBA code okay to use?

The VBA function returns an actual median.

Alternatively, there is a method for estimating the weighted median of grouped data.  It might be doable in Excel without VBA.  But it is more complicated.  And again, it is an estimate.

Vera wrote:

So i tried to calculate the median by hand and got 9,03 as a result. is this realistic?

Yes; more precisely, about 9.025.  This is due to the method of estimating the weighted median that I alluded to before.  One way to do that in Excel....

1. Construct a helper column with the cumulative counts.  In C2, enter the formula =B2.  In C3, enter the formula =B3+C2.  Copy C3 into C4:C53.

2. Then the (estimated) weighted median is (9.025):

=FORECAST(C53/2, OFFSET(A2, MATCH(C53/2, C2:C53)-1, 0, 2), OFFSET(C2, MATCH(C53/2, C2:C53)-1, 0, 2))

Of course, it would be more efficient to enter =MATCH(C53/2,C2:C53)-1 into another helper cell (e.g. D2), and replace MATCH(C53/2,C2:C53)-1 with D2 in the formula above.

Used in this manner, FORECAST interpolates between the days in A11:A12 by prorating the midpoint between the cumulative counts in C11:C12.

suzanne s. barnhill say:

Unfortunately, Word 2016 does not honor Spacing Before even after a page break. If you plan to have section breaks between chapters (and there are plenty of good reasons for doing that), then you can use the necessary amount of Spacing Before. You can experiment till you get the desired amount; the Paragraph dialog allows you to enter an amount in inches or centimeters and will convert that measurement to points. Once you have the desired amount, update the heading style to match the formatting of the paragraph.

If you will be using only page breaks rather than section breaks, then you can define the style with the required amount of Spacing Before and a frame; the Spacing Before will be preserved inside the frame. Recent versions of Word don't make it easy to insert a frame; you can add the Insert Horizontal Frame button to the Quick Access Toolbar to add one to any paragraph. In the case of a style, however, you can use Modify Style | Format | Frame.... Set the text wrapping to None and leave the other settings alone, at least to start.
