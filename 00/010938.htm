The Edit Recipient list is always greyed out for Excel data sources (as far as I know).

The Edit Recipient List tool is active only if the record source being used is stored as a Table in a Word document. To rename automaticly sheet name/ filename by follow particular - Microsoft Community

To rename automaticly sheet name/ filename by follow particular cell

HI Sir, 

How to rename automaticly sheet name/ filename by follow particular cell ?

amit tandon say:

To rename a sheet automatically, say on change of a cell value (ex. D1) in a sheet, use below code in the sheet module.

Private Sub Worksheet_Change(ByVal Target As Excel.Range)
'change sheet name to cell D1 value

If Target.Address = "$D$1" And Target.Value <> "" And Target.Value <> ActiveSheet.Name Then
    On Error GoTo ErrorHandler
    ActiveSheet.Name = Target.Value
    'may also use:
    'ActiveSheet.Name = CStr(Target.Value)
End If
Exit Sub

ErrorHandler:
MsgBox ActiveSheet.Name & " Sheet Name not changed because of Invalid New Name"

End Sub

In case you wish to change all sheet names per cell value (ex. A2) in each sheet, use below code in the standard module:

Sub RenameSheets()
'duplicate sheet names are not allowed
'some characters are disallowed for sheet names
'sheet name length is subject to a limit

Dim ws As Worksheet

For Each ws In Worksheets
    On Error Resume Next
    With ws
        If .Range("A2").Value <> "" Then
            .Name = .Range("A2").Value
        End If
        On Error GoTo 0
        If .Name <> .Range("A2").Value Then
            MsgBox .Name & " Sheet Name not changed because of Invalid New Name"
        End If
    End With
Next

End Sub

calltor say:

OK, I found a workaround for this issue!

Just edit the registry keys where Ovi Suite keeps the paths.

Start > Run > RegEdit

I found it in HKEY_CURRENT_USER\Software\Nokia\Nokia Ovi Suite\{D995DE8A-1556-45a3-BC7A-568E63663F25}\PcPath

 

If you can't find the key just create an empty folder somewhere with a certain name and configure Ovi Suite to use that folder for photos/videos. Then, search the registry for the folder's name to find where the key is hiding :)

 

So far, no problems syncing to remote folders on the network.
