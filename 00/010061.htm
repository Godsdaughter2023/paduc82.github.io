shane devenshire say: Here is another way to do it:  In Excel select one of the series on the chart, then select the whole SERIES formula on the Formula Bar and press F9, Enter.  Do this for each series.  The chart will now be static.  When the data changes, make a second chart and copy it to PP and repeat the steps for making the chart static.  This suggestion assumes you are using more that one chart from the same range.  If you only want a single chart in Excel then you should go with CallKathy's suggestion - copy the chart and paste as a picture or image.

Do the graphs in the PPT need to update ever? If not, I suggest you paste them as images instead of graphs. Each will then be an independent graphic and changing the selected data won't change the graphics. If you do need to change the graphic, you can delete it and replace it.

stefan blom say:

In Word, click File | Options | Save and select "Save to Computer by default" and clear the option to "Show additional places for saving, even if sign-in may be required."

In addition, if you select "Don't show the Backstage when opening or saving files," you can use keyboard shortcuts to directly access the classic Open and Save As dialog boxes (Ctrl+O displays the Open dialog box and F12 displays the Save As dialog box).

Now you should be able to save locally without being prompted for a password to OneDrive (or other cloud storage).
