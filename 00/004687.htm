suzanne s. barnhill say:
The headings in your document use heading styles. If you've used the built-in ones, they are Heading 1, Heading 2, etc. The entries in the TOC use a different set of styles: TOC 1, TOC 2, etc. These can be edited directly (use the Style Pane Options to set the display to "All styles," sorted alphabetically, to locate them), but you can also make changes in paragraph formatting directly in the TOC. Because the TOC styles are set to update automatically, you can change just one entry at a given level (changing indents is most easily done by the horizontal ruler), and all the other entries at that level will be updated automatically.

brian tillman [outlook mvp 2007-2019] say:
There is no way to archive old mail using the web interface to your mailbox.  The Archive button is nothing more than a move to a folder named "Archive", if that folder exists, or to another folder of your choice, the name for which you will be prompted when you click Archive the first time.  Those messages that are moved to the Archive folder still take up room in your mailbox.  The only way to truly archive messages is to use a mail client like Windows Live Mail, Office Outlook, Eudora, or Thunderbird to move the messages to a local data store and remove them from the online mailbox.

mike h.. say:

Hi,

The simple way is to use the CONVERT function so with your grams in A2 you can use this.

=CONVERT(A2,"g","lbm")

However, you appear to want it as in your example so we can do this:-

=INT(ROUND(A2/28.34952,0)/16)&" lbs " & ROUND(MOD(A2/28.34952,16),2) & " ozs"

Note because of the formatting the second formula returns text and not a number and for 1500 grams returns

3 lbs 4.91 ozs

To round the decimals to 1 decimal place use this

=INT(ROUND(A2/28.34952,0)/16)&" lbs " & ROUND(MOD(A2/28.34952,16),1) & " ozs"
