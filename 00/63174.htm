Extracting an Update - Microsoft Community

tom982 say:

Extracting an Update

Technical Level : Intermediate

Summary

Many common Windows Update errors (0x800B0100, 0x800F0900, 0x00007002 etc.) are caused by corrupt or missing files and can only be solved by replacing the corrupt/missing copy with a clean one which can be extracted from an update.


Details

Using third party tools such as 7zip, WinRAR and Universal Extractor won't properly extract all of the payload files contained in an update so we must use expand.exe - a Microsoft tool included in all Windows installations. expand.exe has no GUI and hence has to be run via the command line; the commands differ slightly when extracting service packs, but it is all explained below:

  1. Open the folder containing the update you wish to extract, then hold Shift and Right-Click on empty space in that window to bring up an extended context menu.
  2. Click on Open Command Window Here to open Command Prompt
  3. In Command Prompt, enter the following command to extract the cab file from the update:

    For standard updates

    expand -f:* {Update Name}.msu {Destination Folder}

    Where {Update File} is the update you wish to extract and {Destination Folder} is the folder (this folder must already exist) which you would like the extracted files to be saved to. For example: expand -f:* Windows6.1-KB2633952-x64.msu Extractedwill extract KB2633952 to the folder Extracted.

    For service packs

    {Service pack file} /x:{Destination Folder}

    Where {Service pack file} is the service pack you wish to extract and {Destination Folder} is the folder which you would like the extracted files to be saved to. For example: windows6.1-KB976932-X64-INTL.exe/x:Extracted will extract Windows 7 SP1 to the folder Extracted.

  4. In the Command Prompt window, type: cd {Destination Folder} (where {Destination Folder} was the folder name you chose in step 3)
  5. Type expand -f:* {Update Name}.cab {Destination Folder} to extract the files from the cabinet file.

You probably need to do a clean reinstall of Firefox with a freshly-downloaded installer file.

1.  Download the appropriate Firefox installer for your language and system:  https://www.mozilla.org/en-US/firefox/all/

2. Use Windows Explorer (file manager) to remove these folders (you'll only have Program Files (x86) if you have a 64-bit system):

  • C:\Program Files\Mozilla Firefox\
  • C:\Program Files (x86)\Mozilla Firefox\

Do not remove the Mozilla Thunderbird folder if there is one.  Do not remove any profile folders.

3.  Reboot your computer into "clean boot configuration" (directions:  https://support.logitech.com/en_hk/article/228#win7)

4.  If your antivirus program is running, temporarily disable it (see Help in your antivirus program for directions)

5.  Install Firefox using the installer from step 1.

6.  Restart the computer in normal configuration (see "After completing your troubleshooting" steps at link above).
