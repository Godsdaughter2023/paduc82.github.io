f z z say: If all phone numbers are in the same exact format, you could try

=SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(MID(A1,7,255)," (0","")," (",""),") ","")

If there could be variable numbers of spaces, change that to

=SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(MID(TRIM(A1),7,255)," (0","")," (",""),") ","")

For general parsing, better to use the most appropriate tool for the task, and in Excel that means using Laurent Longre's MOREFUNC.XLL add-in. It provides an add-in function names REGEX.SUBSTITUTE which could be used in formulas like

=REGEX.SUBSTITUTE(A1,"^\D+(\d+) *\(0?(\d+)\) *(\d+).*","[1][2][3]")

which finds the 3 groups of numerals and concatenates them.
