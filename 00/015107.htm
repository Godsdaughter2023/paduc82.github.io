Hello Alex,

Thank you for posting your query on Microsoft Community Forums.

As per description, your are unable to boot after installing an update.

  • What is the current build version installed on your computer?

I would suggest you to follow the methods given below and check.

If you don't have Windows 10 installation disk then you may download Windows 10 ISO file and burn it on a DVD to create a bootable media from a different working computer.

Refer the link: How to create an Installation DVD or USB for build 10130 - Multiple Language

Method 1:

Perform an Startup Repair through Windows 10 installation disk.

To do so, follow the steps given below:

a. Start your computer from the installation media. Insert the DVD or USB flash drive and restart your computer. If you see a message asking you to "Press any key to boot from DVD", do so. If you don’t see the message, you might need to change the boot order in your computer's BIOS settings so that it first starts from the DVD or USB. When you see the "Install Windows" page, click "Repair your computer" to start the Windows recovery environment.

b. If you don’t have installation media, use the power button to restart your computer three times. This will start the Windows recovery environment.

c. In the Windows recovery environment, on the "Choose an option" screen, click "Troubleshoot".

d. Click "Advanced options" and then click "Startup Repair". Check if it helps.

Method 2:

If the Startup repair doesn’t work, then try to rebuild the Windows boot sector from the Windows RE command prompt.

To do so, follow the steps given below:

1. Boot the computer using Windows 10 installation media.

2. In the “Windows Setup” dialog box that appears, set the Time and currency format,

Keyboard or input method and the language to install and then click “Next”.

3. Click on “Repair your computer” from the lower left corner.

4. From the “Choose an option screen”, click on “Troubleshoot”.

5. In the “Advanced Options” screen, select “Command Prompt”.

6. Now type the following commands at the Windows RE command prompt and press Enter after each command:

bcdedit /export C:\BCD_Backup

c:

cd boot

attrib bcd -s -h -r

ren c:\boot\bcd bcd.old

bootrec /RebuildBcd

7. Restart the computer and check if you are able to boot the computer successfully.

Hope this would help. If issue still persists post back with current status of your computer and result of the proposed suggestion, we will be happy to assist you.

How to display task finish date relative to other task start date - Microsoft Community

burtlamborn say:

How to display task finish date relative to other task start date

Sorry if this has been asked before, but I would like to add a column that displays the finish date of each task in terms of the start date of another task.  Specifically, I have a task at the start of my schedule: "Contract Start".  I want to display when all other tasks finish, relative to "Contract Start".  In other words, I want to see that Task A is done 5 months after contract start, Task B is done 8 months after contract start, etc.  I suspect there is an easy way already set up to do this, I just don't know what it is.  Can anyone point me in the right direction?

Thanks,
Burt
joe blake - pmp, mcp say:
Burt,

In lieu of creating a more elegant macro, vb solution or project server project field you could create a work-around using a custom project date field.

Open your project and insert a custom field > Task > Date (I use Date10) rename it to "Contract Start".  Click on formula and enter the contract start date formatted with quotation marks like this  "02/24/14".  

Next insert another custom field > Task >Number and rename it to "Contract Start Difference".  Click on formula and enter the formula DateDiff("d",[Date10],[Finish]).

Now you can enter the date of the contract in Date10 and change it as necessary and it will be reflected in each phase of your projects as your task dates change.
