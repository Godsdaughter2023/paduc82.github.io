The folder

"C:\Documents and Settings\All Users\Documents"

is linked to the folder "

"C:\Users\Public\Documents"

with a symbolic link. Any file that you save in "C:\Documents and Settings\All Users\Documents" is physically stored in "C:\Users\Public\Documents". This is done for legacy reasons.

   5.    Click OK.
   6.    When you are prompted, click Restart.
   7.    After the computer starts, check whether the problem is resolved.
