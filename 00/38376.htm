tryx3 say: This should not happen.  Here is a full set of checks & fixes.  You have already tried some but not others.

1  Run Windows in Safe mode to see if the problem occurs there – see Start your computer in safe mode.  If not then the problem is caused by a third-party application.  Clean boot procedures can help to narrow down the culprit but they do take quite some time – see How to troubleshoot a problem by performing a clean boot. 

2  Check that WMP is looking in the right folder[s] for your music by checking the folder settings in WMP, Organise, Manage libraries, Music.  Make sure that all the folders you have told it to check do actually exist.

3  Run [with WMP closed] both of the control panel audio troubleshooters or the online versions. Running any troubleshooters will reset some WMP records such as Date last played so if that is important you might want to take a set of screenshots as a record or put all the played items in a Playlist called something like PlayedBeforeProblemsAppeared just as a reminder.

Control panel, Troubleshooters, WMP Library & WMP Settings – onboard version [if you are connected to the internet then Windows should update these to be the same as the online versions when you try to run them]

3.1  Media player library troubleshooter – online version

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Library-troubleshooter

3.2  Media player settings troubleshooter – online version

http://windows.microsoft.com/en-US/windows7/Open-the-Windows-Media-Player-Settings-troubleshooter

4  Rebuild the media library in WMP by going to Tools [on the Menu bar or by pressing alt-t], Advanced, Restore media library

5  Delete the WMP library database to force a complete rebuild - Close WMP, in the Start menu search box type %LOCALAPPDATA%\Microsoft\Media Player In the Windows explorer window that appears delete all the files [but not sub-folders]. See if that has fixed the problem.

6  Delete the WMP database cache - Close WMP, in the Start menu search box type %LOCALAPPDATA%\Microsoft\Media Player In the Windows explorer window that appears, select the Media Player folder in the left-hand pane & delete it together with its sub-folders. See if that has fixed the problem.

7  If that hasn't fixed it then do both these steps [they are the closest thing possible to uninstalling then reinstalling WMP & will effectively provide a new installation of WMP]

7.1  Unload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, clear Windows Media Player checkbox, Yes, OK, Restart the PC.

7.2  Reload WMP - Control Panel, Programs and Features, [left-hand side] Turn Windows features on or off, Media Features, set the Windows Media Player checkbox, Yes, OK, Restart the PC.

8  If the problem remains then it seems to be in Windows 7 itself rather than within WMP.  Prepare for Windows 7 checks by checking the disk first –

Windows explorer, Computer, C:\ drive, right-click, Tools, Error checking - Check now, select Fix faults if found, select Yes to schedule a repair, Reboot

[Running when you next reboot is what the poorly explained question about scheduling actually means]

9  Run the System File Checker How to use the System File Checker tool to troubleshoot missing or corrupted system files on Windows Vista or on Windows 7 There is additional guidance on it in sfc.exe /scannow.  Do note that the second link provides a procedure for extracting a list of unfixable errors into a text file in one of your own folders – this is a lot easier to work with than the full SFC results file [CBS.log].

10  I have assumed above that the system is clear of malware.  Check the system, there is excellent advice from “SpiritX” in this thread - SpiritX AV cleaning advice

11  If the problem still resists resolution then have a look at the other Windows Media Player threads in this forum to see if any symptoms match yours &, if so, how they were solved.  There are a lot of threads on the subject so you might find it quicker to search for your symptom to narrow the field a bit i.e. search on Windows Media Player <symptom>

How to activate Windows 7 or Vista manually (activate by phone)
http://support.microsoft.com/kb/950929/en-us


1) Click Start, and in the Search box type:   slui.exe  4
2) Next press the 'ENTER' key
3) Select your 'Country' from the list.
4) Choose the 'Phone Activation' option.
5) Stay on the phone *** do not select/press any options *** and wait for a person to help you.
6) Explain your problem clearly to the support person.
7) The support person should give you a confirmation ID, copy it down on paper,
8) Verify the ID is correct by reading back to the support person.
9) Enter the ID number and click 'Next' to finish the activation process.
