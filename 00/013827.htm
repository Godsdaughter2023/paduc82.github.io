per jessen say:

You have to disable the OnTime Event before the workbook is closed. Use a Workbook_BeforeClose event to disable the event. Insert your original OnTime event statemet, and add ',False' which will clear the previously set procedure like below:

Private Sub Workbook_BeforeClose(Cancel As Boolean)
Application.OnTime Now + TimeValue("00:00:15"), "my_Procedure", False
End Sub
