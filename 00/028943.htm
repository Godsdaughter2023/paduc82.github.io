I know this has to do with the formatted date/time - but I have no idea how to change the field so I may extract my data for a specific date.  
john w. vinson say:

Are you using the European convention (1/2/2017 is February 1) or the American convention (January 2)? Since the developers of Access were American, your criterion will find January 2.

In addition, you say "date/time" - if the field contains a time portion (e.g. what's in the table is something like #1/2/2017 11:31:22#) then the record will NOT be found, since #1/2/2017 11:31:22# is not equal to #1/2/2017 00:00:00#. To find records for any time on the specified date you need a range, such as

>= #1/2/2017# AND < #1/3/2017#

or, better, use a Form to solicit the criteria and use

>= [Forms]![yourform]![txtSearchDate] AND < DateAdd("d", 1, [Forms]![yourform]![txtSearchDate]

dave in nola say:

Can I modify a date format to show the month in caps - 17 APR 2012

The date format I use is, dd mmm yyyy, which results in the date 17 Apr 2012.  I want the date to appear as 17 APR 2012.  I have tried DD MMM YYYY but that doesn't work.  Any suggestions?
rick rothstein, mvp - excel say:
To answer the question you asked, no, there is no Cell Format that will produce that result. While Jaeson's suggestion will work, it involves two cells... one to hold the date and another for the formula. There is a VB solution... we can construct event code that will force a format that looks like you want on any cell that is changed in the cell or cells you want to have this particular format. If a VB solution would be acceptable, then you need to tell us the range of cells that are to have this particular format so we can build the processing for them into the code.

rick rothstein, mvp - excel say:

The ranges I need to display are F3:G1048576 and J3:K1048576, if possible.


This event code should do what you want...

Private Sub Worksheet_Change(ByVal Target As Range)
  Dim RangesToMonitor As Range, Cell As Range
  Set RangesToMonitor = Range("F3:G" & (Rows.Count - 2) & ",J3:K" & (Rows.Count - 2))
  If Not Intersect(Target, RangesToMonitor) Is Nothing Then
    For Each Cell In Intersect(Target, RangesToMonitor)
      Cell.NumberFormat = "DD """ & UCase(MonthName(Month(Cell.Value), True)) & """ YYYY"
    Next
  End If
End Sub

INSTALL Event Code
-----------------------------
If you are new to event code procedures, they are easy to install. To install it, right-click the name tab at the bottom of the worksheet that is to have the functionality to be provided by the event code and select "View Code" from the popup menu that appears. This will open up the code window for that worksheet. Copy/Paste the event code into that code window. That's it... the code will now operate automatically when its particular event procedure is raised by an action you take on the worksheet itself.
