 net stop wuauserv

These are the most common problems with Windows Installer.

Windows Installer Errors troubleshooting:
Step 1: Check the Windows Installer Service.


1. Click Start Orb, in the start search box, type Services and press Enter.
2. Scroll down and find the Windows Installer.  Double-click “Windows Installer” in the Services list.
3. On the General tab, make sure the service is started under “Service status”.
4. If the service is not already running, under Service status, click Start, and then click OK. 

Step 2: Unregister and re-register the Windows Installer

1. Click Start Orb, in the start search box, type MSIEXEC /UNREGISTER, and then click OK . 

2. Click Start Orb, in the start search box, type MSIEXEC /REGSERVER, and then click OK . After you run this command, the operation is complete.

3. Now, try your installation again.  

 

 

Step 3: Clean any pending installations

Check the following registry key and clean any entries in the key that you find:

Go to the Start Orb, type in Regedit and select enter.
If you get the User Account Control dialog box that states, Do you want to allow the following program to make changes to the is computer?, Click yes.

 

Modifying REGISTRY settings incorrectly can cause serious problems that may prevent your computer from booting properly. Microsoft cannot guarantee that any problems resulting from the configuring of REGISTRY settings can be solved. Modifications of these settings are at your own risk.

 

Scroll to: HKEY_LOCAL_MACHINE \Software\Microsoft\Windows\CurrentVersion\Installer\InProgress

 

Check the following registry key and clean any entries in the key that you find:

HKEY_LOCAL_MACHINE \System\CurrentControlSet\Control\Session Manager\PendingFileRenameOperations

 

Step 4: Check the permissions

Verify the System has Full Control on the following keys:


HKEY_LOCAL_MACHINE \Software\Microsoft\Windows\CurrentVersion\Installer

 

Next, restart your computer and try again.

yottie say: Well, I finally got it to install.

Here's what finally worked for me:

1.  Download the individual update for KB 2770917.  Be careful, you'll be offered an array of downloads.  Choose the one applicable for your system.  Keep it on the desktop or elsewhere accessible.

2.  Click Windows + R.  Type in msconfig.exe, click RUN

3.  In the window that opens go to the "Services" tab and check the box at the bottom left "Hide all Microsoft Services"  then click "disable all" at the bottom right of the window.  Click "Apply" then OK and you'll be offered a restart.  Before you restart disable internet connection - if cable simply disconnect the cable or turn off WiFi.  Click restart.

4.  When machine has restarted, go to the saved individual update and double click.  Follow the prompts to install.

5.  On completion, you'll be asked to restart to make the installation complete.  Check with Windows + R "msconfig.exe" and system tab as in 3 above, and ensure the same settings remain as set in step 3.  Reconnect the internet or turn WiFi on again and restart.

6.  On restart click on "Control Panel" and go to "Windows update"  - refresh the window by clicking check for available updates.  You should again be offered KB 2770917 despite the fact that you've just installed it.  Click "install".

7.  On completion of install from Windows update, you'll be asked to restart.  Repeat Step 5 above.

8.  On restart check "Control Panel" and then "Windows update" again and (hopefully - it finally worked for me!) you should see KB 2770917 in the list of installed updates.  If so,

9.  Click Windows + R "msconfig.exe" and on the page that opens press button "Normal Start up" then "Apply" then "OK" after which you'll be offered a restart.  Click "Restart"

10.  On restart it should have all been done.

11.  BTW, I don't have a Nalperion Licensing service so it's something else that was the culprit.  Only this process worked for me in the end.

Hi,

Does your error look like this?

Assassins Creed error 000007b occurs because Microsoft .NET Framework 4.5.1 is not installed.  Framework 4.5.1 is required for the game to run.  Normally this would be downloaded and installed as part of the game install.  The only way it wouldn't be there would be for one of the following reasons:

1. The NET Framework was cancelled for some reason.

2. Your downloaded game is damaged - There should be an option to 'repair' the game from either the Steam Store application or the Ubisoft store app, depending which one you bought the game through.

3.  You have a pirated version of the game.  If this is the case, our support ends here.  Buy the game.
