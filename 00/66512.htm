Here's an ugly search option for archived Outlook files that others might try in Outlook 2013 / Windows 8.1. It's similar to the old (and odd) method used in Windows Explorer Search but much more inconvenient.

Control Panel/Mail

  • Create a new profile (archive)
  • Go to Properties/Data files and add all the archived PSTs
  • Change "When Starting Outlook" parameter to "Prompt for Profile.."
  • Start Outlook and select the newly created "archive" profile
  • Not sure from memory if you have to manually load all the PSTs into the profile or if they load automatically after creating the profile. So, if you don't see the archived PST files in "folders" - you'll need to load them.

Control Panel/Indexing

Open up indexing and check the status. When indexing is complete, you can close Outlook.

Control Panel/Mail

Change "When Starting Outlook" parameter to "Always use this profile" and select your regular Outlook profile.

When you need to search (hopefully not that often now) your archived PST files, you can:

  • Close Outlook
  • Control Panel/Mail/ Change "When Starting Outlook" parameter to "Prompt for Profile
  • Start Outlook again but select "archive" profile
  • Perform search
  • Close Outlook
  • Control Panel/Mail/Change "When Starting Outlook" parameter to "Always use this Profile
  • Restart Outlook

Very lame and incredibly weird design but at least your archived mail isn't entirely lost

a. user say:

 Hello Bhaskar,

Welcome to Microsoft Community and thank you for posting your query.
 
Sorry for the inconvenience caused, I will certainly help you with this issue.
 
Windows 10 lets you see how much data you have used over any type of internet connection Ethernet, WiFi, Dial-Up etc., so that you can stay in full control of how and when your data should be consumed. Here is how you can see how much data has been consumed overall and by individual apps in Windows 10:
 
1. Click/touch on the Start icon and select Settings from the menu.
2. In the Settings window that opens up, select Network & Internet.
3. Under the Network & Internet category, choose Data Usage from the left side. You would be able to see how much total data has been used and how much data has been used through Ethernet, WiFi or some other type of connections.
4. If you click/touch on Usage details under the data usage statistics, then it opens a new screen where you can see the data usage details for all the individual apps.
 
Knowing how much data has been used by your Windows 10 PC is great for those on limited internet connections that either stop working after certain number of gigabytes of data usage or they get extremely slow. New computers that come preinstalled with Windows often have what is called a recovery partition. This is used to reinstall the operating system in the event of a system crash. To access it, you will need to boot into when you start your computer by pressing a function key. This can be either F1, F2, F9, F10, F11, F12 or even DEL or Tab key. Consult the owners manual that came with your PC for instructions about how to reinstall Windows.

If your recovery partition is not available or damaged, you should contact the manufacturer of your computer and request a recovery disc set you can use to reinstall Windows 7. They might charge a small shipping and handling fee.

------------------------------------

This is how the recovery partition is accessed for the most popular makes..

For Dell, press CTRL + F11 directly after powering up the unit

For HP, press F11 directly after powering up the unit

For LG, press F11 directly after powering up the unit

For Toshiba, press and hold '0' BEFORE and during the power up

For Acer, press and hold ALT + F10 as soon as you see the logo

For Asus, press F9 as soon as you see the Asus logo.

For Samsung, press F4 at power up..

For Fujitsu, press the F8 key repeatedly directly after powering up

For Advent, restart your computer. Then Press F10 repeatedly until the message "Starting System Recovery" is displayed

For Sony VAIO, restart and press "F8" or "F10" repeatedly until the "Advanced Boot Options" screen appears.
