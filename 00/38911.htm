lemp say:

Some of the things you've done are inapposite or counterproductive.

Let's start with the basics.  How many physical disk drives are installed in the computer?

I'm not sure what you mean by "swap and page files."  The two words are essentially synonymous, although "swapfile" hasn't been used since Windows 9x (a file named swapfile.sys appeared in Windows 8; it isn't in Win 7).  You shouldn't, by the way, completely eliminate the page file from the system partition.  AFAIK, you need at least a small pagefile on the system partition for minidumps.  In any event, moving the pagefile to a different partition on the same physical drive will almost certainly slow down your system.

Regardless of where you install programs, most programs will install something on the system drive anyway, although you can certainly get the bulk of the installation elsewhere.

The AppData folder is what the name suggests:  data for your various installed application programs.  It's user-specific, so each user profile has its own AppData folder.  If you delete the AppData folder (it would probably be a bit safer to delete the specific application folders in ...\AppData\Local or ...\AppData\Roaming), you'll break -- or at least lose settings and saved information for -- whatever application depends on it.  See the discussion here --> https://www.quora.com/Can-I-delete-the-AppData-folder-in-Windows-7

System Restore is specific for each drive (partition).  In general, you should activate it on your system partition but not for data-only partitions.

Download TreeSize Free --> http://www.jam-software.com/treesize_free/

Run it in "Administrator" mode.

If you want to focus on your AppData folder, determine which application(s) are using the most data and then decide what you want to do about it.

For example, the AppData file on my regular user account is about 48 GB.  The two biggest culprits in that are AppData\Local\Microsoft and AppData\Roaming\Apple Computer. 

Looking a bit deeper into the latter, it's pretty clear that most of the space (11 GB) is taken up by backups of my various iDevices.  No, I don't want to delete any of that.

Looking at the Microsoft folder (29 GB) shows that most of that (28.5 GB) is from my Windows XP Mode virtual machine.  No, I don't want to delete that either.

Your situation obviously will be different.  If you find an application that's generating a large piece of AppData, decide if the application is worth keeping.  If it is, the solution is to get a bigger hard drive ... which is what you should probably do anyway.

P.S.:  I forgot to add that sometimes, changing permissions on a Windows system can result in an expanding AppData folder.  It doesn't sound as if that's your problem, but see this --> http://www.sevenforums.com/general-discussion/115149-stop-application-data-folder-replicating.html

How to fix Solitaire errors 124/ 101_107
  1. Run App troubleshooter
  2. Run Windows Update troubleshooter
  3. Check that Windows Updates aren’t ‘stuck’
  4. Check if Xbox Live is down
  5. Create new user profile and try setting up from there
  6. Solution 1: Run App troubleshooter

    This fixes some issues that prevent apps from running such as screen resolution, and incorrect account or security settings.

    • Right click Start
    • Select Control Panel
    • Go to the top right corner and change View by
    • Select Large icons
    • Click Troubleshooting

    • Click on View all option on the left panel
    • Select Windows store Apps

     

    • Follow the instructions to run the app troubleshooter

    Check whether you are able to play Solitaire again. If not, go to the next solution.

    • ALSO READ: Microsoft Solitaire is stuck on loading: Here’s how to fix it
    Solution 2: Run Windows Update troubleshooter

    This tool automatically detects, and fixes most incorrect settings on your computer that brings the Solitaire error 124/ 101_107/, and prevent you from running any updates.

    • Click Start
    • In the search field box, type Troubleshooting > hit Enter
    • Click View all in the left pane
    • Select Windows Update

     

    • Run Windows Update Troubleshooter and follow the instructions on the screen
    Solution 3: Check that Windows Updates aren’t ‘stuck’

    If you’ve checked for Windows Updates and there was no problem there, try checking if there are some updates that are stuck because some do get stuck for various reasons.

    • Click Start
    • In the search box, type Windows Updates
    • Click Check for Updates

    • Click Check Now
    • Click View Details
    • Install any updates you may find that have not been installed including optional ones
    • Restart your computer
    Solution 4: Check if Xbox Live is down

    Sometimes the network on Xbox goes down, in which case you can check and log in then click Support to see if it says “Xbox Service is up” or “Xbox Service is limited” or “Xbox Service is down”. It is down or limited, you may not be able to log in to your game till it is fixed.

    • ALSO READ: Best Solitaire apps for Windows 10, 8.1 or 7 users
    Solution 5: Create new user profile and try setting up from there

    You can create a new user profile then change settings to administrator privileges, and check if Solitaire error 124/ 101_107/ persists.

    Here’s how you can create a new user profile:

    • Click Start
    • Select Settings

    • Select Accounts

    • Click Family and other users
    • Click Add someone else to this PC

    • Fill in the form with the user name and password. Your new user account will be created.
    • Click on Change account type
    • Click the drop down arrow and choose Administrator to set the account to administrator level
    • Restart your computer
    • Login to the new account you just created

    If the issue goes away, then it may mean your other user profile is corrupted. You can try the following in case of a corrupted user profile:

    • On your new account, use it to downgrade your usual account
    • Click Apply or Ok
    • Raise back your old account to its default admin level
    • Rinse and repeat a few times as this will help to remove any corruption
    • Leave your account on as Administrator

    Please follow the below steps to reset the Windows Updates Components:

    1. Press Windows Key + X on the keyboard and then select “Command Prompt (Admin)” from the menu.

    2. Stop the BITS, Cryptographic, MSI Installer and the Windows Update Services. To do this, type the following commands at a command prompt. Press the “ENTER” key after you type each command.

                               net stop wuauserv

                               net stop cryptSvc

                               net stop bits

                               net stop msiserver

    3. Now rename the SoftwareDistribution and Catroot2 folder. You can do this by typing the following commands in the Command Prompt. Press the “ENTER” key after you type each command.

                              ren C:\Windows\SoftwareDistribution SoftwareDistribution.old

                              ren C:\Windows\System32\catroot2 Catroot2.old

    4. Now, let’s restart the BITS, Cryptographic, MSI Installer and the Windows Update Services. Type the following commands in the Command Prompt for this. Press the ENTER key after you type each command.

                            net start wuauserv

                            net start cryptSvc

                            net start bits

                            net start msiserver

    5. Type Exit in the Command Prompt to close it and then restart the computer.

    6.After restarting the computer, now retry.
