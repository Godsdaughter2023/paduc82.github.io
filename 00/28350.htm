Conditional Formatting Not Working on Merged Cells - Excel 2016 - Microsoft Community

joshexcelnoob say:

Conditional Formatting Not Working on Merged Cells - Excel 2016

As I was posing this question, I did some fiddling and actually found an answer, but the forums I was reading were locked so I decided I would share my solution, and maybe summarize other solutions that others found (that didn't work for me, but seemed to help others).

The issue is trying to apply a conditional format, specifically the STRIKETHROUGH format to a merged cell.  All other formats seemed to work (cell color, font color, etc), but the STRIKETHROUGH format just wasn't there.

One solution that helped others was to format the fill color to be just off white.  They did this by: 

Home Tab -> Styles Group -> Conditional Formatting -> New Rule -> Use a Formula... -> Format -> Fill Tab -> More Colors -> Custom Tab

Color Mode should be RGB, and with pure white, all of the RGB value should be 255

Simply change any one of them to 254, which isn't even perceivable to any normal human, but changes the color of the cell to slightly off white.

*This did not work for me but apparently helped others

The solution that worked for me was to change the color of the Strikethrough bar from automatic black to manual black:

Home Tab -> Styles Group -> Conditional Formatting -> New Rule -> Use a Formula... -> Format -> Font Tab

On the Font tab, you choose the Strikethrough condition, but change the default Color from "Automatic" to "Black".  

A weird solution but it made the magic work.

Just thought I'd share it in case others have similar issues.
