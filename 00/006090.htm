zj aj say: One thing that can happen is that your files are MP3s, but are tagged with the wrong metatag version. Zune can't read this metadata, so the songs don't show up in your collection. I use a free program called MP3Tag to check this. If the tag version is ID3v2.4, they won't show up in Zune. You want to save the metadata as ID3v2.3 (which MP3Tag can also do if you set it up that way). Then your files will show up in your collection.

ID3v2.4 isn't in favor generally, but some digital stores still use it. Most use ID3v2.3, and that's the most "standard" version.

How do I change the default font of Word documents? Currently it's Cambria and I prefer Times

how do I change the default font in Microsoft Word 2008?  Currently it's Cambria and I prefer using Times
daniel g. grau say:
Try this:
  1. Create a new blank document (Command N)
  2. In the main menu bar, selet Format > Style…
  3. In the left column, select Normal
  4. Click on Modify
  5. Change the font to Times (note that Times New Roman is identical, but is more readable)
  6. At the bottom, place a checkmark on "Add to template" (this will extend the change to your Normal template)
  7. Click OK etc.
Test by creating a new blank document with Command N.

If the above does not work, try this:
  1. In the main menu bar, select File > Open
  2. Navigate to this folder:  ~/Library/Application Support/Microsoft/Office/User Templates
  3. Open Normal.dotm
  4. Repeat steps 2-7 above.
