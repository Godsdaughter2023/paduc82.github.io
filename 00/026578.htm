jgkzdl say:

This is my suggestion:

Sub CombineColumns()
    Dim lngColumns As Long
    Dim lngRows As Long
    Dim varArr As Variant
    Dim varDest As Variant
    Dim i As Long
    Dim shS, shD As Worksheet
    lngColumns = 6
    lngRows = 9
    Set shS = Worksheets("Sheet2")
    Set shD = Worksheets("Sheet3")
    varArr = shS.Range(shS.Cells(1, 1), shS.Cells(lngRows, lngColumns))
    ReDim varDest(lngRows * (lngColumns - 2), 3)
    For i = 1 To lngRows * (lngColumns - 2)
        varDest(i - 1, 0) = varArr((i - 1) Mod lngRows + 1, 1)
        varDest(i - 1, 1) = varArr((i - 1) Mod lngRows + 1, 2)
        varDest(i - 1, 2) = varArr((i - 1) Mod lngRows + 1, Int((i - 1) / lngRows) + 3)
    Next
   
    shD.Range("A1:C" & (lngColumns - 2) * lngRows) = varDest
   
End Sub

The source tabel is on sheet Sheet2 and the destination table is on sheet Sheet3.

You have to give the columns and rows value of the source table but that can be automated.

steve_p---01 say: You can actually change the voice tag for contacts ;)

It's just slightly different from your previous 6630. Here's how to do it:

-Go to your Contact in question and click the option to edit.
-Next click option again and select Add detail, then from the long list select Nickname.
-Type the Nickname that you want for that contact.
-Close down editing the contact.
-When you next play the voice tag for that contact it should speak the Nickname and not the Contact Title as previously.

Good eh?
