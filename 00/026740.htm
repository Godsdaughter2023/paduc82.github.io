Here are some links about how to take advantage of Refresh and RecImg and some other new Win 8 backup features.  There are many more links than you strictly need (aka "overkill")

Win 8 Total Backup System: File History + RecImg + Refresh + SlimImage

Win 8 has refined your backup and restore options. Before Win 8 you could create an Image Copy backup of the full drive (or selected directories), but it was hard to separate backups of the OS, Applications and personal Data unless you created a separate data partition.  I still think it is a good option.

Out of the box, MS has been advertising the “Refresh” and “Reset” options. Personally, I think they missed a valuable selling point by not mentioning RecImg.EXE in conjunction with Refresh. It removes my main concern/objection to Refresh.

Reset is equivalent to using the “Recovery Partition” on previous versions of Windows. The factory would create a “Recovery Partition” that you could use to “reset the computer to factory fresh”. It would typically include all of the factory “crapware” and it would wipe out all of your data files and installed applications.  A less than desirable outcome but an appropriate reaction to a bad malware infection.  The “recovery partition” type of restore typically had to be accessed at boot time while Reset can be accessed from within Windows 8.

Refresh is roughly equivalent to an image copy backup.  A Reset will restore a backup of the OS and Metro applications. It will wipe out “legacy” / x86 desktop type applications BUT it will not affect your personal data files, which is much better than a Reset.  But there is still that issue of wiping out your installed desktop applications. I do not have documented confirmation, but I’m guessing that Metro apps are treated like personal data and those directories/folders are simply not reset back to the old condition.

RecImg.EXE is a backup tool that ships with Windows 8. It creates the backup “image” that is used by the Refresh tool. This image that was created at the factory is why using Refresh will wipe out installed desktop programs in the “C:\Program files” and “C:\Program Files (x86)” folders because they haven’t been backed up. However, using RecImg.EXE you can create new images that will include recently installed desktop applications.

RecImg Manager is a third party tool the provides a GUI to access the RecImg.EXE command line tool and backup files.

File History is a new Win 8 feature. It is an extension to the “Previous Versions” and “Volume Shadow Copy” features. File History continuously monitors files stored in Libraries, Desktop, Contacts, and Favorites folders. It runs on a predetermined schedule, and when it detects changes in any file, it makes a backup of that file to an External drive. It isn’t enabled by default.

Ignore what you see in the Formula Bar, for the most part.  The Formula Bar has its own formatting rules, and we have little control over it.

Mindy wrote:
The problem is columns E and F are for time ins and time outs and column G is total hours worked.  Columns E and F are formatted for hours and minutes.  In the G column I put =f1-e1.  Say E1  has 1:00 and F1 has 3:00.  The g column that has the formula displays 0.1 but it should display 2 .

The 0.1 is the fraction of 24 hours.  That is how Excel stores time.  If you format column G as h:mm, you will see 2:00.

But since you want decimal hours (2), the formula should be =(F1-E1)*24.

However, that will show fractional hours if the times were, for example, 1:23 and 3:45.  Caveat:  You might see the fractional hours if you format the cell as Number with 0 decimal places.  But they are there in the actual value, nonetheless.

If want to show only hours, you need to decide how to round the calculation.  One of the following:

=ROUND((F1-E1)*24,0)
=ROUNDDOWN((F1-E1)*24,0)
=ROUNDUP((F1-E1)*24,0)

Generally, when you have time in hh:mm form, you multiply by 24 to convert to decimal hours.

talking moose say:
Actually, Outlook for Mac 2011 always runs in cached mode when compared to Outlook for Windows. Outlook for Mac does not offer "online mode", which avoids storing copies of your messages on your computer. Instead it always works in "offline mode" and always stores copies of messages on your Mac.

If you actually need something like "online mode" in Outlook for Mac then you can go to  Outlook menu -->  Preferences... --> Accounts, select your Exchange account, click the Advanced... button and select the Download headers only option. You can get a list of your messages quickly but they will not actually download until you view or preview each message.

As for OWA, the premium version was always limited to IE for Windows only for Exchange 2007 and earlier. That meant any other web browser, such as Firefox, always connected to the "lite" version of OWA even if it was running on Windows. Since IE for Mac was discontinued years ago the Mac could never get anything but a lite version of OWA.

This did change, however, with Exchange 2010. It does offer the same rich web client to all supported browsers such as IE for Windows, Safari and Firefox.

    • Open powershell as administrator
    • Run the command Get-appxprovisionedpackage –online | where-object {$_.packagename –like “*windowscommunicationsapps*”} | remove-appxprovisionedpackage –online
    • Re-install Mail, People and Calendar from the Store (green tile)

  • Creating virtual desktops may result in missing or black thumbnails in Task View.
  • You might notice a chess knight icon on your Lock screen to the right of the screen. This was added by the Lock screen team so they could tell via screenshots if someone was using the new Lock screen or the old one, and will eventually be removed in a future build.
  • Font sizes on the Lock screen on devices with high DPI can be really large.
  • We currently have the Tablet Mode notification turned off by default to address some of the issues we’ve been seeing. The notifications can be turned back on via Settings.
  • The touch keyboard doesn’t show up on login screen which prevents you from unlocking your PC when Narrator is on.
  • Some people might see frequent prompts to restart to install updates, even though no updates need a restart. This prompt can be ignored safely.
  • When you take a picture with the camera app on your PC and tap the circular icon in the top left to view the photo you just took, the Photos app will launch and then crash immediately.
