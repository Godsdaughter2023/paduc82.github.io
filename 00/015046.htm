So the question actually is "how do I create a solid horizontal line?" Lots of ways ...

  • at the start of a new paragraph, type 3 (or more) hyphens, <enter>, you have a Horizontal line that goes across the width of the page. You can go back and enter letters over it.
  • click on the Home tab / Font group / "U " for Underline command and any letters you type subsequently have a horizontal line under them
  • Create a 1 row 1 cell table, Insert tab / Tables group / Tables command
    with cursor in table, Table Tools Design conditional tab / Table Styles group / Borders command / turn off 3 of the 4 outside borders
  • Home tab / Paragraph group / Borders command / turn on one of the borders
  • type you hyphens then, Home tab / Font group / Font Dialog Launcher button / Font dialog: Advanced tab / set Spacing: Condensed, and By: to whatever it takes to make the characters "bump" into each other, trial and error
These are the ones I remember off the top of my head. I'm sure there are others.

jean-yves tfelt say: Hi

To do in mass, you can use this macro
Press Alt + F11 to get to the VBA editor
Insert a standard module and paste this code (update the folder name in Bold) :

Sub ConvertToHyperlink()

Dim cl As Range
Dim strPdfFolder As String
' Update the folder name
strPdfFolder = "C:\Documents and Settings\My documents\"

For Each cl In Selection
    ActiveSheet.Hyperlinks.Add Anchor:=cl, Address:=strPdfFolder & cl.Value _
        , TextToDisplay:=cl.Value
Next cl
End Sub

Save your work BEFORE running any macro !!!
Select the cells containing the pdf file name and run the macro (within the VBA editor) by pressing the F5 key.
