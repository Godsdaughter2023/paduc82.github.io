stefan blom say:
You can use a macro that applies the table style to all tables in the active document:

Sub ApplySameStyleToAllTables()
Dim t As Table
For Each t In ActiveDocument.Tables
t.Style = "Table_style_name_here"
Next t
End Sub

For assistance with the macro, see http://www.gmayor.com/installing_macro.htm.
