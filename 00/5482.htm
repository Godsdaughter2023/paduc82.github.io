While you are still in Windows 7 and waiting for the Windows 10 upgrade, assuming that you did not "reserve" the Windows 10 upgrade [because doing so might override the setting in para 1]:-

1  Set Windows update to notify but not download [Control panel, View by - Small icons if necessary, Windows update, Change settings] then you can check the list of & the size of any updates to make sure that they do not include the Windows 10 upgrade without your knowledge.  When the Windows 10 upgrade does become available to you [this might still be weeks or months away, there is no way of telling], just download any other updates that have appeared in the same batch [just to get them out of the way] then you have to either

2  Stay up until midnight and then give it permission to download & install.

or

3.1  Set up a task to wake the computer at midnight [and test the task so you know it is working correctly], then

3.2  Start the download for a minute or two so you can see that it is working correctly, then

3.2  Hibernate the computer so you are not wasting your precious daytime allowance on the download.

- - - - - - - - - - - - - - - -

4  If you have reserved the upgrade, the description Microsoft gave of the meaning of this [in the Get Windows 10 app, the icon in the System tray] was that it would start downloading outside of normal Windows update control.  This might just have been a badly worded description but it might have meant what it said. 

5  In this case I think that the best thing to do would be to make a Windows 10 installation disk using Download Windows 10 [about 3GB download and you'll need a 4GB USB stick for a 32 bit version or an 8GB USB stick for a 64 bit version] - but you must then run it using "In-Place upgrade" [aka "Repair install"] procedures because, if you do not, you might never be able to recover from the situation that would be created [a reportedly unlicensed Windows 10 installation that does not revert successfully to its licensed Windows 7 predecessor].  The initial steps in the process need your participation but you'll then be able to leave it to its own devices for the main download itself.

6  Complete the preparations in para 3.3 of "In-Place upgrade" [aka "Repair install"] procedures  Do not skip these preparations because they are the most reliable tools for helping you recover your computer from any mishaps that occur during the upgrade and hence for making sure that upgrade glitches do not appear to rob you of your existing Windows 7 licence.

7  "In-Place upgrade" [aka "Repair install"] procedures- these instructions were written for Windows 7.  It is essentially the same process but you will see Windows 7 control end and Windows 10 installation begin during para 3.4.  You can choose to check for updates as part of the Repair install procedure but in your situation, I'd say it was better to just get the upgrade done and then worry about any further updates that are needed later on [after the next midnight slot starts].  Don't enter a Product Key when prompted to do so during the procedure, just skip that step.  When Windows 10 installation takes over the process, it will display a clock-like % progress display and it might seem to hang several times but you will just have to be patient [I have seen it take a couple of hours to complete sometimes and I have seen reports of users thinking it had failed partway only to discover when they repeated the process that it was merely agonisingly slow].  When installation has finished, connect to the internet and it should automatically recognise the prior activation status - see Start menu, Settings, Update & security, Activation.  It will take its own sweet time about this so you can use this dialog to make it do it straightaway but still don't put in the Product Key because it will fail - your free upgrade Windows 10 licence depends absolutely on its activation process recognising that you had a properly installed & licensed Windows 7 SP1 or Windows 8.1 installation on the computer beforehand. Note that this limitation does not affect any subsequent reinstallations because part of the successful initial Windows 10 activation process registers your hardware on Microsoft servers so subsequent reinstallations will check those servers rather than the previous OS. 

8  You could do the preparations at any time but would only want to start the Download Windows 10 download phase itself after midnight as it will involve about 3GB data.

9  You may find that Windows 10 does not initially recognise some of your hardware devices until you then perform the next Windows update [many users have found that it sets your display to a basic display driver until that first update is done at which point it will correctly identify your display card and install the proper drivers for it.  If you find that there are hardware devices that are still not performing as you expect after that first update then refer to Where to get hardware drivers.

When you defer upgrades, new Windows features won’t be downloaded or installed for several months. Deferring upgrades doesn’t affect security updates. timliu_msft1261 say:

hi gnana,
 
yes, it is possible.
 
we can add workflow to specific content type and then add this content type to list/library. only items which are related to this content type will follow the workflow.
 
in this case, we say there are content a and b, a need approval workflow but b doesn’t need:
1. we click setting gear>site settings>site content types to create content types for that site collection. here we create a and b.
2. after creating, return to the site content types page and click a>click workflow settings>add a workflow>choose approval (you may need firstly activate it) workflow>finish other settings>next>configure the workflow>save.
3. navigate to the library>library settings>advanced settings>choose yes of allow management of content types?>ok.
4. return to library settings page>click add from existing site content types in content type section>add a and b.
5. when users create new document with a, it will start approval workflow. for b, it won’t start approval workflow.


if anything is misunderstood, let me know.
