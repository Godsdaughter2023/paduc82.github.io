Hi Evonnechow,

1. Quit Skype.

2. Click on Go > Go to folder.

3. Type in ~/library/application support, then press enter

4. Under Application folder > click on Skype folder.

5. Look for user name and you will see main.db file.

erica blair say:

Hi VancouverGator,

As previously discussed, excel does not have a user-exposed option to make solid line borders any thinner than the standard line. One option is to change the borders to a slightly gray color, which can help trick the eye into thinking they're thinner. Another option would be to disable all grid-lines and only place borders on data where you need it, or use alternating row-fills instead of lines to delineate rows.

Let us know if you need further assistance.
