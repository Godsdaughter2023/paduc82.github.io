In summary, Teamviewer is a great one for PC to PC access.  However, if you are into folder/file sharing, the better option is Onedrive or Dropbox.  You get to control what you want to share be it on Folder or file level.

suzanne s. barnhill say:
The file you're seeing is the "owner" or "lock" file created whenever you open a Word document. It prevents any other user from opening the file for editing while you're using it. (See http://support.microsoft.com/kb/211632 for a description of this and other temporary files Word uses.) Ordinarily, these files are deleted when you close the document or quit Word. As Greg suggests, if they're not being deleted, it may be for the reason he cites.

greg maxey say:

It sounds like you have an instance of Word open in the background.  After closing Word, either reboot your PC or uses CTRL+ALT+Delete to open the Task Manager.  Look at the Processes tab and make sure there is no instance of winword.exe running.  If there is, end it.

You should then be able to delete your temp files.
