wunders say:

I'm not sure I exactly understand what you want other than you have an old XP machine that you would like to put to good use rather than just discard it.  I see two approaches that you might use.

 

1) Assuming that your Windows 7/8 machine has the proper extra slots/connections, you may be able to take the hard drive and CD/DVD drive out of your old computer and mount them inside your newer computer as extra drives and then throw away the rest of your old computer.

 

2) You can leave your XP machine as it is and connect it to your home network (it is assumed here that you have a internet modem and router or a modem/router combination that provides the networking to all your home computers.  Doing this, you could configure it similar to a "Network Attached Storage" device (NAS).  Using the DVD drive on the XP machine would require logging into that machine, though (either by being in front of it, or by Remote Access).

 

Just because support is going away for XP will not make it any more vulnerable to threats than it is now.  And updates to products like Microsoft Security Essentials will likely continue as they support several flavors of Windows.  If you are worried, you can manually assign it an IP address on your network and omit the Default Gateway parameter.  This will make it impossible to communicate through your router to the internet while leaving local communications intact. (Some routers have ways to block specific computers from the internet as well).

 

I don't know where you heard that XP and Win7/8 are not compatible...  I have been able to create a network share a folder on my XP machine and easily connect to it from my Win 7 machine and mount it as a drive and move files back and forth without problem.  You did mention using the drive as a backup drive...  With Windows 7, using the Windows backup tool to backup to a network-connected drive is only supported in the "Pro" versions of Windows 7. So whether you can do this depends on the version of your Windows 7 machine...  but moving individual files is not a problem with any version of Win7.

 

If you want to securely erase your hard drive before discarding your computer, you will need to boot your XP system from a CD so that files are not in use on your hard drive.  I usually recommend the free Hirens Boot CD.  Create the CD, boot your system from it, and after booting from the CD, you will have several "Nuke" options for securely wiping your hard drive.  If you would like to leave your XP machine operational but securely remove all you personal files, then simply delete all of your files to the trash can and empty the trash can.  Then download/run the program "Eraser" and tell it to erase all unused disk space.

 

Yet another option is to install a free version of Linux on your old machine...  but this takes a lot of patience and research to set it up the way you want and most likely will not work the way you might expect.

 

Using a USB cable to connect two computers is problematic.  USB likes to have one Master device and lots of slave devices.  The computer is usually the master device.  Connecting two "Masters" together requires more than just a cable with Type-A connectors at each end.

 

Hope this helps,

  JW

davem121 say:
Hi Nicolae727

Before you start, backup your personal data (files) to your D drive

You can reinstall Windows 10 at any time and it will not cost you anything !

Click this link:
https://www.microsoft.com/en-us/software-downlo...
to download the Media Creation Tool (Click on Download Tool Now), with that you can download the latest Windows 10 ISO (Select Create Installation Media for Another PC), you can create a bootable USB flash drive (min 8GB) using that tool

Then, Boot your PC from the Installation Media you just created (change Boot Order in your BIOS) to begin installing Windows 10

If you have problems booting from a boot disc, you may have UEFI BIOS:
Insert the Bootable Installation Media, then go into your BIOS and make the following changes:
1. Disable Secure Boot
2. Enable Legacy Boot
3. If Available enable CSM
4. If Required enable USB Boot
5. Move the device with the bootable disc to the top of the boot order
6. Save BIOS changes, restart your System and it should boot from the Installation Media

Since you previously had Windows 10 installed and activated on that PC during the installation process skip the steps which ask for a product key and select the option 'I am Reinstalling Windows 10 on this PC', and activation will not be an issue, your PC will have a digital entitlement stored on the Microsoft Activation Servers

Once the installation completes, you must then restore your personal data, install a set of device drivers, which you can obtain from the support page for your device on the manufacturers website and then finally re-install all your 3rd party applications . . .
