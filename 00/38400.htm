sirstephenh say:

Attepting to rule out the TV as the culprit is a good start. You should check the port by plugging in another HDMI device that you know works using a different cable just to be sure.

First off, try resetting your display settings by doing the following.

1. Remove any discs from the disc tray.

2.Turn off the console.

3. Turn on the console.

Note If you are using an Xbox 360 Wireless Controller, turn on the console by using the Guide button on the controller. Make sure that you use the player 1 controller when you do this. The player 1 controller will have the upper-left light illuminated.

4. As the console starts, on the controller, press and hold Y while pulling the right trigger. The display settings are reset to the default settings, and then the console automatically restarts.

If that doesn't work then the HDMI cable or the port on your Xbox could have been damaged under the stress.

Try using a different HDMI cable.

If a different HDMI cable doesn't work then try the AV cable that came with the Xbox. If you have the dual Component/Composite cable then make sure that you have the HD switch on the cable in the right position. If it works then your HDMI port is probably damaged and your console should be sent in for repair.

If it doesn't work then resetting your display settings again. If it works then your HDMI port is probably damaged and if it still doesn't work then your console is probably damaged in some other way. Either way your console should be sent in for repairs.

You can follow the instructions here to start the repair process.

Also remember to NEVER move your console while it's on and/or while there's a disc in the drive. Doing so can destroy your disc.

Good luck!

Boot from the Windows 7 DVD disc you created and start the installation process.

After you are satisfied all problems have been resovled then activate Windows.

I’d suggest you to disable startup programs and services temporarily, using clean boot and check if the issue resolves.

Please follow the instructions below to troubleshoot a problem using clean boot.

1.       Click Start, type msconfig in the Start Search box, and then press ENTER.

2.       If you are prompted for an administrator password or for a confirmation, type the password, or click Continue.

3.       On the General tab, click Selective Startup.

4.       Under Selective Startup, click to clear the Load Startup Items check box.

5.       Click the Services tab, click to select the Hide All Microsoft Services check box, and then click Disable All.

6.       Click OK.

7.       When you are prompted, click Restart.

To undo the changes and start your computer normally, please see the steps described below,

1.       Click Start, type msconfig.exe in the Start Search box, and then press ENTER.

2.       On the General tab, click the Normal Startup option, and then click OK.

3.       When you are prompted to restart the computer, click Restart.

4.       For more information on clean boot please see the article mentioned in the link below,

http://support.microsoft.com/default.aspx/kb/929135
