praseetha k say:

Hi Sonny2129,

 

System Error Memory Dump Files are memory files which contain dump files created when your computer stops unexpectedly because of a Stop error (also known as a "blue screen," system crash, or bug check).

When an event occurs in Windows it is immediately noted down in the System Error Memory Dump Files and it gets updated every minute. Since it gets updated every minute, it will take more time to scan the Error Memory Dump Files compared to other system files during Disk cleanup.

You may refer the following KB articles for more information:

Overview of memory dump file options for Windows Vista, Windows Server 2008 R2, Windows Server 2008, Windows Server 2003, Windows XP, and Windows 2000

http://support.microsoft.com/kb/254649

How to read the small memory dump files that Windows creates for debugging

http://support.microsoft.com/kb/315263

Note: The above mentioned articles applies to Windows 7 also.

Hope this information is helpful.
