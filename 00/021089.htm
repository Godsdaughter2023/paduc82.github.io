Duplicating emails in the Sent folder is happening for users of the Samsung Mail app.

Emails sent using the Samsung Mail app are duplicating every time the Mail app syncs with the server.

IMAP protocol is used to sync the emails.

 

Possible solutions:

Remove the account from the Samsung Mail app.  Install a different email app.

 

Remove the account and then setup the account using pop3 protocol.

 

Change the Sync settings to Never.

    Try Repairing Permissions If you are using System Prior to El Capitan, Then use method described on this website. Look for section on Repair Permissions. 

    If you have EL Capitan follow these Directions:

    Repair Permissions (EL Capitan and above):

    1. Save all work and quit all applications.
    2. Hold shift key down while restarting computer. (Safe-Mode.)
    3. On some Computers Screen may black out for a minute or two.
    4. Hold shift down until screen lights up, and you see a Progress meter.
    5. Wait a few seconds after you see this, then release Shift key.
    6. At some point you will be asked to supply password, use same one that is used to install software.
    7. Now Screen will go to a painting series from bottom to top for several cycles.
    8. When screen settles down and is completely loaded, then restart normally.

    Apple has removed ability to repair permissions from Disk Utilities.. But using this procedure forces EL Capitan and Systems newer to run permissions check and repair.
