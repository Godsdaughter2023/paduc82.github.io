Simply use Copy and Paste.  That is use CTRL+A to select everything in the document that you want to embed, then use CTRL+C to copy it to the Clipboard and then put the selection at the location in the host document where you want the "embedded document" to appear and press CTRL+V.

Another option is to use an INCLUDETEXT field

nathan li msft say:

Hi AEdmondson,

Per my test, you may use the following steps to create a Visio Stencil containing theme fonts, etc.:

1. Go to File > New >  select Organization Chart;

2. Create your Organization Chart > add shapes of different theme and texts of different font and sizes;

3. Go to File > Save As > select the path to store the file > input file name > select "Visio Stencil" in Save as type.

If you have any further concerns, please post back and let us assist you.

Best Regards,

Nathan
