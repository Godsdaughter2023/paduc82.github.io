ken sheridan say: A common way of doing this (e.g. Allen Browne's Concatenate function at http://allenbrowne.com/func-concat.html ) is to write a function which loops through a recordset of rows with a particular value in one or more columns and returns a string of the concatenated values from another column.  However, Jamie Collins once posted a clever and efficient way of doing it using the GetString method of the ADO recordset object.  Adapting this to your situation would give a function along these lines:

Public Function GetF7(strF5 As String) As String

    Const NOCURRENTRECORD = 3021
    Dim rst As ADODB.Recordset
    Dim strSQL As String
    Dim strF6s As String
   
    strSQL = "SELECT DISTINCT F6 FROM YourTable " & _
    "WHERE F5 = """ & strF5 & """ ORDER BY F6"
  
    Set rst = New ADODB.Recordset
   
    With rst
        .ActiveConnection = CurrentProject.Connection
        .Open _
            Source:=strSQL, _
            CursorType:=adOpenForwardOnly, _
            Options:=adCmdText
       
        On Error Resume Next
        strF6s = strF5 & ": " & .GetString(adClipString, , "; ", "; ")
        .Close
        Select Case Err.Number
            Case 0
            ' no error so remove trailing space and semi-colon
            ' and return string
            GetF7 = Left(strF6s, Len(strF6s) - 2)
            Case NOCURRENTRECORD
            ' no F6 values for this F5 value so return
            ' zero length string
            Case Else
            ' unknown error
            GetF7 = "Error"
        End Select
    End With
   
End Function

You can then call the function in a query:

SELECT F1,F2,F3,F4,F5,F6,
GetF7(F5) As F7
FROM YourTable;

Note that as this uses ADO you need to include a reference to the Microsoft ActiveX Data Objects Library in the database (Tools | References on the VBA menu bar).

Ken Sheridan, Stafford, England
