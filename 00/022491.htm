First, what is Two-Step Verification?

Two-step verification helps protect you by making it more difficult for someone else to sign in to your Microsoft account. Is it optional to turn on your second level of security as the account owner is the only one who has the ability to turn on his/her two step verification on the account.

If you turn on two-step verification, you’ll get a security code to your email, phone, or authenticator app every time you sign in on a device that isn't trusted. When it’s turned off, you only get and re-send security codes periodically, and when there might be a risk to your account security.

henry88 say:

How do I link an ACCESS table with a column in EXCEL

Hello friends,

 

I have an existing EXCEL sheet where there is a column of codes but no description. Howeve, there is an ACCESS table which contains those very same codes with corresponding descriptions. What I would like to do is to:

 

Insert a new column in the EXCEL sheet (next to the codes)

Import / link the description for a given code.  (coming from the ACCESS DB table)

 

This way, the EXCEL will contain the code as well as the description.

 

If this is possible, will this link be dynamic or static? I.e. if there is a change in ACCESS database, will it be reflected in the EXCEL automatically or do I have to refresh the link manualy?

 

Your help is greatly appreciated.

hansv mvp say:

You can import the Access table into your Excel workbook (into the same sheet, or into a different worksheet if you prefer) and keep a link to the Access table.

To do so, click From Access on the Data tab of the ribbon. Then follow the instructions.

In the Properties of the connection, you can specify whether the table should be refreshed wen you open the workbook, and whether it should be updated every n minutes.

You can then use VLOOKUP formulas in the column next to your codes to retrieve the descriptions.
