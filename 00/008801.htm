joeu2004 say: cardunals1985 wrote:

I am working on how to use the formula for INDEX and MATCH functions  in my homework.  I have been trying different methodis of using this, but to no avail.  I have used the example:  =INDEX(A3:F20,MATCH(G3, A3:A20, 0), 6) *Sales

The instructions state:  Enter a formula for commission that calculates the commission based upon multiplying Sales times the appropriate commission fomr the data A26:A28.  Use referencing and the INDEX and MATCH functions for looking up data in the A26:B28 cell range.


Your instructor is asking you to do this the hard way, because you could use VLOOKUP in this particular case.  But presumably the purpose of the assignment is to teach you how to use INDEX and MATCH, not necessarily write the best formula.

First, let me say that it is a good idea for you to learn as much as possible about these functions (INDEX and MATCH; forget about VLOOKUP for now) by reading the Excel Help pages.  Please do so as you read my comments below.  The help pages might not be sufficient for you to solve your problem on your own.  That is understandable.  But it is a good habit to get into.

I assume Dept is column A, Sales is column F and Commission is column G, with data starting in row 3 (Angston, Lee).

Your assignment is to put formulas into column G to compute commission.  So you would not reference column G in your formula, as you tried to do with MATCH(G3,...).

I further assume that the Dept and Rate table at the bottom is in A26:A28 and B26:B28.

So, you should construct a formula for G3 that you will copy down the column through G15 (Wyler, Len).

The MATCH expression should look up the dept in A3 in the table at $A$26:$A$28.  That will produce a row index from 1 to 3.  You must look for an exact match (i.e. last MATCH parameter zero) because the table in $A$26:$A$28 is not alphabetized.

Hint:  I use a relative reference for A3, but an absolute reference for $A$26:$A$28.  That is a hint for how you should write the MATCH expression so that you can easily copy it down the column.

The INDEX expression should select the rate from the table at $B$26:$B$28, using the MATCH result as the row index.  For the INDEX expression, you can specify a column number of 1, referring to the 1st column of the lookup table (1st INDEX parameter).

Note:  The column number is redundant in this case, since the lookup table has only one column.  But it is a "good programming practice", especially while you are learning, because there might be other instances where it is important to specify the column number.

The final part of the formula in G3 is to multilply the INDEX result -- that is, INDEX(...,MATCH(...,0),1) -- by the sales amount in F3.  That produces the commission amount, which should be 412,530.

So your formula in G3 should have the form =F3*INDEX(...,MATCH(...,0),1), where you need to fill in the parts represented by "...".

I hope this helps.  If not, post a response here showing us what you tried -- hopefully something better than your first attempt -- and we will provide further assistance.
