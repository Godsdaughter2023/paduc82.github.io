You should have a Windows 7 System Repair disk.  If you do not, go to any other Win7 computer and make one there.  The 32/64 bit must match.  You can make it by clicking on the start globe, All programs, maintenance.  It only takes a few moments, and you only need a blank CD.

Now start your computer on that disk.  If it does not go to that disk (it should), you may have to alter the BIOS to make the DVD drive the first choice to boot from.

The Repair disk loads its own Windows into the computer to begin a diagnostic process.  You may be asked for a Windows 7 install disk -- which does not have to be for the same computer.
