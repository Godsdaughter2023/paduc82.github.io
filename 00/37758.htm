bossdweeb say: Technically they're not missing. Whenever doing a clean install (the best option) you still have
to install drivers for the motherboard, dedicated graphics card (when there is one), and peripherals such as printers and keyboards with profiling options.

Get the motherboard drivers from Toshiba's web site. The drivers in your list are commonly
called the chipset drivers as they are all  functions of the motherboards chipset ( Notrh Bridge
and South Bridge).
There should also be drivers for the cpu and it's Intel HD Integrated Graphics . Windows does
install a generic VGA graphics driver, but you need the Intel drivers for full functionality of the
HD Integrated Graphics.

While you're at it Install all the Windows Updates. Setup the Windows Updates option so it
does not download and install automatically so you can install them is small batches. There is
less likelihood of corrupted updates if you do it this way. You should not need most of the optional
updates .

Also Update DirectX -
