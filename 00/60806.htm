"Deleting" a file doesn't actually delete it; it just marks the space
as available to be used. There are third-party programs that can
sometimes recover deleted files. The problem is that the space used by
the file is likely to become overwritten very quickly, and this makes
the file unrecoverable.

So your chances of successfully recovering this file are decent if you
try recovering it immediately after deleting it, and rapidly go
downhill from there. If you've been using the computer since then (for
example to write this question and read this answer), your chances may
be very poor by now.

Hi Robert,

Thank you for posting on Microsoft Communities.

I will be glad to help you with the issue you have with the computer. I understand the frustration when things don't work the way it should.

To fix Windows error code 38 is very similar to resolving various other driver errors that may occur. The first step that you should take in order to resolve this error is to restart your computer. To restart your PC, do the following:

  • Click on the Start button in the lower left hand section of your screen.
  • Click on Shutdown.
  • Choose Restart.
  • Let the computer go through the process of rebooting.

After you restart the computer, attempt to start the hardware again and see if the error returns. If there is no error message, you have resolved the problem. If the error code persists, you will need to take the next steps.

  • Launch Device Manager on your computer.
  • Locate the device that is generating the error on the list of installed devices.
  • Click on the device to open the device Properties dialog box.
  • You need to click on the General Tab
  • Click Troubleshoot.
  • The Troubleshooting Wizard will start working. Answer the questions asked in regard to the problem. It will provide some suggestions to fix. Answer any questions and follow the directions to resolve the problem.

Thank you.

lemp say:

Please do not try to change the drive letter of the partition where Windows is installed.  The only way this can safely be done is through a reinstall.

Your exe file is probably hard-coded to extract files to some temp folder on the C: drive.

One way around this is to download a free archiver such as 7-zip and use that to extract the files.  This way you can specify the destination directory.  7-zip can recognize exe files that actually are self-extracting archives.  https://www.7-zip.org/

This is the Windows Media Player Network driver. If you have music streaming from machine to machine, you need it. If you are not using Media Sharing for Media Player you can stop the process using the Services console. Try the steps mentioned below, to set the service in manual mode, instead of disable mode:

a)      Click on Start button, type services.msc in the start search window.  Right click on the services window and select Run as Administrator.

b)      Note: If you are prompted for an administrator password or for a confirmation, type the password, or click Allow.

c)      Scroll down to "Windows Media Player Network Sharing Service”, right click on it and select Properties.

d)      Click the pull-down "Startup type" dialog box and select "Manual."

e)      Click on "Apply”, and then click on "OK".

 

Configure your computer in “clean boot”, to disable startup programs and check for the issue:

How to troubleshoot a problem by performing a clean boot in Windows Vista or in Windows 7:

Note: Once you performed the “clean boot”, please follow Step 7 from the article to set your computer to normal startup.
