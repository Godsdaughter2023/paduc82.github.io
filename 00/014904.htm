  5. Select the account you want to configure the settings.

  10. Hit OK.
  11. Under Server information, you can enter the information for Incoming (POP) server, Incoming server port, Authentication, Encryption, Outgoing (SMTP) server, and Outgoing server port.
  12. Hit OK to save.

pesach shelnitz say:

I'm sorry that I didn't check that 27E1 is the Unicode number for a square. Actually, it's the Unicode number for a diamond in the few fonts, such as Cambria Math, that include this character. Therefore, as you observed, the following lines insert the diamond (27E1) from Cambria Math.

Selection.TypeText Text:=ChrW(&H27E1)

Selection.Font.Name = "Calibri"

jbrehm say:

Looks like I solved my own problem. 

I went to the Layout tab, clicked to expand the Paragraph settings, and clicked on "Tabs" at the bottom left hand side of the Paragraph settings window. Under "Default tab stops" it was set to 0.5". I changed it to 0.25" and it has seemed to solve my problem.

A long-term solution is to set this up in a multilevel list so that bullet formats are associated with paragraph styles—one style per list level. To apply the bullet format and indent of a particular level, apply the associated style. Norton (and McAfee) applications are notorious for not uninstalling (or upgrading) cleanly.

1. Download the Norton Removal Tool, saving it to your desktop:  ftp://ftp.symantec.com/public/english_us_canada/removal_tools/Norton_Removal_Tool.exe

2. Close all open applications (i.e., anything with an icon on the taskbar).

3. Right-click on the file you saved in #1 above and select Run as Administrator to run the utility. Don't TOUCH your keyboard until the run completes, then reboot.

4. Open Internet Explorer (only!) to http://support.microsoft.com/kb/923737 [ignore any DOES NOT APPLY warnings as well as the APPLIES TO section] & run the Fix It again.

5. IMPORTANT - DON'T SKIP THIS STEP! => Open Internet Explorer (only!) to http://support.microsoft.com/kb/971058 [ignore any DOES NOT APPLY or NOT RECOMMENDED warnings] & run the Fix It in DEFAULT and then AGGRESSIVE modes. [1] [2]

6. Reboot one last time & test.

===================================================
[1] Dismiss (close) the the "automated troubleshooter" Fix It pop-up that appears when you first open the webpage then run Fix It 50202 (in DEFAULT and then AGGRESSIVE modes) instead. If need be, you can right-click on the Fix It 50202 button & SAVE the download to your desktop, then run it manually (in DEFAULT and then AGGRESSIVE modes).

[2] Full Disclosure: Running the Fix It in AGGRESSIVE mode will delete your Update History but not the list of Installed Updates.
