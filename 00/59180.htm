¡firedog say:

The Windows Live Messenger service was discontinued more than four years ago, so you shouldn't have installed the program. Getting rid of it might help:

  1. Close down all running programs,
  2. Press Windows key + R to open a Run box, type control and press Enter to open Control Panel
  3. Open Programs and Features and double-click on Windows Essentials 2012 to open the Uninstall or repair Windows Essentials applet.
  4. Select Remove one or more Windows Essentials programs, then select Messenger on the screen that opens.
  5. When it's finished, restart the computer.
  6. Press Windows key and type wlmail. If Windows Live Mail is properly installed, you should see it among the search results, possibly classified as a 'Desktop app'. Right-click on it and use the options available to create the shortcuts you want.
  7. Use a shortcut to launch the program.
  8. When the program has settled down, press Ctrl-Shift-O to open the program Options dialogue. You can adjust many of the default settings here.
  9. On the General tab, ensure that Automatically log on to Windows Live Messenger is not selected. 

If you run into trouble anywhere along the line, stop and post back here explaining what's not working for you.
