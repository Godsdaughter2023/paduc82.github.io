How do I disable Automatic IP Configuration in Windows 7? - Microsoft Community

john_ha say:

How do I disable Automatic IP Configuration in Windows 7?

Can anyone please tell me how I disable Automatic IP Configuration in Windows 7?  I am happy to edit the registry to implement it if necessary.

Why?

Because my PC occasionally gives itself an IP address like 169.254.0.1.  THis is because Windows has spotted that the PC is connected to a network, but the PC has not yet been given an IP address by a network DHCP server, so Windows Automatic IP Configuration steps in and gives the PC an IP address beginning 16.254.

The reason the network DHCP server did not give an IP address was because the network comes up a few seconds before the DHCP server comes up, and Windows Automatic IP Configuration kicks in too quickly.

Of course, once the PC has an IP address beginning 169.254, it cannot communicate with my router which has an IP address beginning 192.168.  The only cure, apart from re-booting, is to issue the ipconfig/renew command to get an IP address from the router DHCP server.

If I can disable the Windows Automatic IP Configuration (I was able to disable it in XP) then I can prevent this error happening.

One option you have.

You can set the IP to a static IP address.

Go to:
Start
Control Panel
Network and Internet
Network and Sharing Center

Click on Local Area Connection (A seperate dialog box opens Local Area Connectin Status)
Click Properites (dialog box opens)
Click Internet Protocol Version 4 (TCP/IPv4)
Click Properties
