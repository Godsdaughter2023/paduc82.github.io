The first time through the loop, Range("C10") and Range("C14") will refer to Sheets("Input").

In subsequent iterations, Range("C10") and Range("C14") will refer to Sheets("Pressure Drop Chart").

I don't know which worksheet you intend to reference.  In either case, it would be prudent to put Sheets("...").Select just after the statement For i = 1 To j.

Actually, it would be better to eschew the .Select methods altogether and to use either With Sheets("...") or Sheets("...").Range("...").

I prefer the latter because using With, it is too easy to inadvertently omit the leading dot in .Range("...") and to overlook the omission.

Saving WORD doc as JPG - Microsoft Community

laurenkropp say:

Saving WORD doc as JPG

How do I save a WORD doc as JPG file?  just upgraded from 2010 to office 365 and I cannot do this anymore? 

jeeped say:
Saving a DOCX as a JPG wasn't a native Word 2010 command that I was aware of. However, there are a couple of system-wide alternatives that will save any application's contents to a JPG or PNG.
  1. With your Word document in the foreground, tap Alt+Print Srceen. This will save an image of the current application windows (e.g. Word with its document) onto the clipboard. Open Paint and use Ctrl+V to paste the clipboard's image into a new image file. You can crop, resize, etc and save in any of a variety of image formats.
  2. Your Windows 7 comes with a utility program called Snipping Tool. Open the program and you can take a screenshot of either the entire Word application window or a rectangle drawn on the screen with the mouse. You can save in PNG, JPG or GIF formats.

    Use Snipping Tool to capture screen shots

charles kenyon say:
I think Jeeped has given you the simplest method. If, though, you need your jpg to be the same size as a printed page, you might want to save as pdf and convert from pdf to jpg. This last step cannot be done by Word or Windows. It will require a third-party program. I use PaperPort because it came with my scanner. I'm sure there are other programs out there, though.

suzanne s. barnhill say:

If you have SnagIt, you can also "print" to the SnagIt printer, which renders each page as a separate image, which you can then save in any graphic format. This is the only method I know of (save the one you mentioned) that will capture the entire page.

suzanne s. barnhill say:

If you need to capture the entire page (or several pages), then Charles Kenyon's suggestion of printing to PDF or mine about using SnagIt will be what you want.

 

If, however, you just need to capture some of the page content, then the way to do it in Word (without using screen capture) is as follows:

  1. Select the content to be saved as a picture.
  2. Ctrl+C to copy.
  3. Use Home | Clipboard | Paste | Paste Special to paste as "Picture (Enhanced Metafile)."
  4. Right-click the pasted picture and choose Save as Picture.
  5. Select JPEG as the desired file format.

suzanne s. barnhill say:
Word has never offered the option to save a document as a JPEG, which is a graphic, not text, format. Individual pictures and screen clippings in Word can be saved as JPEGs, but the Save As dialog in Word does not offer this file format and never has.
