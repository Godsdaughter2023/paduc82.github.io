How do I keep (r) mark formatted as superscript within a - Microsoft Community

mharold say:

How do I keep (r) mark formatted as superscript within a drop-down in Word 2010?

I am creating a template that contains a drop-down list of several products, some of which require a service mark and some a registration mark. I've pasted in the product name with the SM and the superscript formatting is retained when I choose that product from the list. However, even though I've pasted in the registered product names with the (r) mark formatted properly, when I choose them from the list the formatting is not retained. I don't see any options for formatting text within the content control properties box where I'm adding my drop-down choices. Is it possible to keep the formatting in my lists?
jay freedman say:
In all fonts that I'm aware of, the SM and TM symbols are superscript because the glyph (the image of the character within the font's definition) is superscript. However, the registered symbol varies from one font to another. It's a full-size character in many fonts, including Times New Roman, Arial, Courier New, Franklin Gothic, and Tahoma. It's a superscript character in Calibri, Constantia, Lucida Sans, and Consolas, all of which come with Office 2010. (None of this has anything to do with the "superscript" font formatting that you can apply to regular text.)

This points to the way to get your dropdown entries to include a superscript registered mark:
  • Define a style whose only unique formatting is the specification of one of the fonts I mentioned as having a superscript registered mark. (Most easily, format a paragraph with that font, right-click it and choose Styles in the menu, then click Save Selection As New Quick Style.)
  • In the Properties dialog of the content control, check the box for "Use a style to format contents" and choose your custom style.


Every entry in the content control will be in the font specified by the style, and any registered marks will be superscripts.

pcs365_24 say:  

Performance Issues- Outlook 2007 hangs or freezes frequently


Symptoms:

    You may experience one or more of the following performance issues in Outlook 2007:

  • Outlook 2007 appears to be unresponsive.

  • Outlook 2007 appears to stop responding for a long time.


Causes/Reasons:

    The performance issues may be caused by one or more of the following:

  1. Insufficient computer specifications

  2. The latest service pack for Outlook 2007 not installed

  3. Large Personal Folders files (.pst) or Offline Folder files (.ost)

  4. Outlook .ost files or .pst files that are stored on a drive with insufficient write performance

  5. Third-party add-ins

  6. Gadgets that access Outlook data (This only applies to Windows Vista)

  7. Microsoft Office Communicator integration

  8. Antivirus software interaction

  9. Windows Desktop Search indexing

  10. Incomplete closure of .pst files or .ost files

  11. POP3 accounts on Windows Vista clients

  12. Many Really Simple Syndication (RSS) feeds

  13. To-Do Bar and Online mode with Exchange server

 

Probing questions:

  1. Have you tried any troubleshooting steps?

  2. Does the computer meet the system requirements for the 2007 Office suite that you are using?

  3. Are you running the latest version of Outlook 2007?

  4. Do you have a large .pst file or a large .ost file?

  5. Do you have many items in a single folder?

  6. Do you have any nonessential add-ins or out-of-date add-ins that are installed in Outlook?

  7. Did you install any gadgets that integrate with Outlook?

  8. Is Outlook integration with Communicator enabled?

  9. Is antivirus software running?

  10. Did the .pst file or the .ost file shut down incorrectly?

  11. Do you use a rule that moves items out of the default store?

  12. Are you connected to a POP3 server?

  13. Are you synchronizing many RSS feeds?

  14. Are you running Outlook with the To-Do Bar enabled when Outlook is running in online mode against an Exchange server?


Troubleshooting:


Step 1:

If you have nonessential add-ins installed, you may want to remove them from Outlook. To view the add-ins that are installed, follow these steps:

InOutlook, click Trust Center on the Tools menu.

ClickAdd-ins to display a list of the COM add-ins that are installed.

By default, the following COM add-ins are included in the 2007 Office system:

  1. Microsoft Access Outlook Add-in for Data Collection and Publishing

  2. Microsoft Exchange Unified Messaging

  3. Microsoft Office Groove Proxy for Outlook Add-in

  4. Microsoft Office SharePoint Server Colleague Import Add-in

  5. Microsoft Office Outlook Mobile Service

  6. Microsoft VBA for Outlook Add-in

  7. OneNote Notes about Outlook Items

  8. Windows Search E-mail Indexer



To determine whether the performance problem is caused by add-ins, run Outlook in safe mode.

To do this,press and hold CTRL when you start Outlook. When Outlook starts in safe mode, all COM add-ins and Exchange Server client extensions are disabled.

If the problem goes away when Outlook is running in safe mode, it is likely that one of the add-ins is causing the performance issue. To troubleshoot possible problems that are caused by COM add-ins ,disable any items that you do not use regularly. Then, restart Outlook. If that does not resolve the problem, disable the remaining add-ins one by one. If that does not resolve the problem, the issue may not be caused by an add-in.

To disable COM add-ins, follow these steps:

  1. On theTools menu, click Trust Center.

  2. ClickAdd-ins, click COM Add-ins in the Manage list, and then clickGo.

  3. In theCOM Add-Ins dialog box, click to clear the check boxes for any COM add-ins that you want to disable.

  4. After you disable the add-ins, clickOK, and then restart Outlook.


Step 2:

If you have installed any gadgets that integrate with Outlook follow these steps to remove gadgets:


  1. ClickStart and then Click Control Panel

  2. ClickControl Panel Home.

  3. ClickPrograms.

  4. ClickWindows Sidebar Properties.

  5. In theMaintenance section, click View list of running gadgets.

  6. Click any gadget that you want to disable, and then click Remove


Step 3:

ITo Disable Outlook integration with Communicator follow these steps:

  1. In Outlook, clickOptions on the Tools menu.

  2. Click the Other tab, click to clear the Display online status next to a person name check box under Person Names, and then click OK.

  3. Restart Outlook.


Step 4:

To avoid additional network traffic and the resulting performance effect in Outlook, you can turn off the To-Do Bar and reduce the network traffic between Outlook and Exchange in online mode.

Steps to turn off:

To turn off the To-Do Bar in Outlook, point toTo-Do Bar on the View menu, and then click Off.
