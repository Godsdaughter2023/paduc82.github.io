How do I configure Visio 2010 so that the image will be saved with the same page margin settings?

    barb way say:

    Paul is exactly right - when saving to image formats like PNG, there are no Margins.  But you can create fake margins by adding transparent objects on the page. 

    Another approach would be to create a Background Page and link it to the page with your objects to be exported to PNG.  On the Background Page, place a rectangle representing the desired 'page size' including your extra margin space, and make it white with no line (or transparent lines).  This will also create the desired 'page size' on export.  One advantage to using the Background Page is that you can create many Foreground pages and link them all to the same Background page, so that you have a standard page size for all of your exported images.

    Hope that helps :-)

    You would put the code in your "PERSONAL.XLSB" file.  This is a file that is automatically opened by Excel every time you start it up.  It is an optional file that every Excel user can maintain to store commonly used macros.  Since Excel opens it automatically at startup, those macros are reliably available to you.

    To create your PERSONAL.XLSB file, follow the instructions at this link:

    http://office.microsoft.com/en-us/excel-help/create-and-save-all-your-macros-in-a-single-workbook-HA102174076.aspx

    '========= CODE BEGIN ============

    Sub ReassKey()

    End Sub

    Sub SendF2Key()
        SendKeys "{F2}"
    End Sub
