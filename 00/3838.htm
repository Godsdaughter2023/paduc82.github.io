Get-Mailbox | Set-Mailbox -ProhibitSendQuota <Value> -ProhibitSendReceiveQuota <Value> -IssueWarningQuota <Value>

Get-Mailbox | Set-Mailbox –ProhibitSendQuota 24GB -ProhibitSendReceiveQuota 25GB -IssueWarningQuota 23GB

If you are not an Office 365 global admin, it is not feasible to set mailbox usage for all users.

rob koch say:

Since it isn't clear from your original description what might have caused the boot loop that was your original reason for refreshing the PC, it's difficult to determine the best path to try and fix it.

If the problem was purely software, then the refresh should have fixed everything, but if hardware or a deeper issue like driver, rootkit or other malware embedded in the operating system itself were involved, this might still leave issues.  There were known boot loop issues with Windows 8/8.1, but the causes were inconsistent and thus the solutions as well, often related to 3rd-party tools or rootkit malware that interfered with the already more complex secure boot process designed to intercept and avoid such items.

A key item I'd check would be whether Windows 8.1 was fully recovered and updating properly since the refresh, since if that PC originally came with Windows 8, it may need to be updated manually to 8.1 before any newer updates can install successfully.

If you can right-click the Windows Start Menu from the lower left corner of the screen and select the System option from the list that displays, this should indicate Windows 8.1 is installed, since Windows 8 didn't include this feature.  If Windows 8 is installed, then the following support article discusses how to acquire that update.

Update to Windows 8.1 from Windows 8 - Windows Help

If Widows 8.1 is already installed, then I'd check whether it's updating properly by searching for Windows Update on the Start screen, selecting that and then View update history to confirm whether the most recent 2019-06 Security updates have been successfully installed.

Performing an operating system Reset is likely to resolve the problem, but along with wiping out both more apps and personal data, it can be more difficult and require installation media you may not have.  I had to acquire this from Dell, the manufacturer of my Windows 8 based laptop a few years ago in order to fully reinstall Windows for an unrelated reason.

Something is interfering with the network connections Defender requires in order to update itself, which since this process is closely connected to Windows Update itself, implies a problem in the operation of that system.

Rob
