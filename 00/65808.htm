Without further ado, the answer is simple: pagefile. When the process "svchost" and "system" need more RAM memory to run, these turn to page file to perform their tasks.     milagrosm2020 say: Hi,

    1 - Using the Power button in the bottom right corner, restart computer once or twice and see if you can proceed.

    2- Can you use the On-screen keyboard from the Ease of Access menu at the bottom right corner of the log on screen? If so, see if you can type using it.

    3- Using the Wi-Fi button in the bottom right corner, connect to an Internet connection and then see if the PC accepts your Microsoft Account login credentials. If not, then try your PIN.

    4- If you are already connected, unplug the router, terminate the connection and try. Some have reported that this helped

    5- If you do not see the box to log in, press Ctrl+Alt+Del and see if it brings up the login box to enter your password.

    6- Restart and boot Windows 10 in Safe Mode and run ChkDsk. Restart your PC and try.

    7- In Safe Mode, Run services.msc and make sure Credentials Manager service is set on Automatic.

    8- In Safe Mode, access the Advanced Startup Options and see if you can repair the installation.

    9- Else while in Safe Mode, either Reset This PC or Rollback to your earlier OS.
