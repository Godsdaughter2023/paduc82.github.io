%AppData%\Microsoft\Windows\Recent\AutomaticDestinations

 

Right click  in the blank address bar and select Paste, Press the  Enter  Key.

a7bd71699cd38d1c.automaticDestinations-ms

 

Right click that file and select Delete.

 

 


Exit Windows Explorer.

 

 


Start working with a few Word 2010 files. The JumpList should start working again.
