Try this macro. Change the name of the worksheet to the correct one where indicated in the code. I have assumed the animal names are in Column A

 

Sub Copy_Data()
Dim r As Range, LastRow As Long, ws As Worksheet
Dim LastRow1 As Long
Dim src As Worksheet
'Change this to the sheet with the data on
Set src = Sheets("Sheet1")
LastRow = src.Cells(Cells.Rows.Count, "A").End(xlUp).Row
For Each r In src.Range("A2:A" & LastRow)
    On Error Resume Next
    Set ws = Sheets(CStr(r.Value))
    On Error GoTo 0
    If ws Is Nothing Then
        Worksheets.Add(After:=Worksheets(Worksheets.Count)).Name = CStr(r.Value)
        Sheets(CStr(r.Value)).Cells(1, 1) = "Total"
        LastRow1 = Sheets(CStr(r.Value)).Cells(Cells.Rows.Count, "A").End(xlUp).Row
        src.Rows(r.Row).Copy Sheets(CStr(r.Value)).Cells(LastRow1 + 1, 1)
        Sheets(CStr(r.Value)).Cells(1, 2) = WorksheetFunction.Sum(Sheets(CStr(r.Value)).Columns(4))
        Set ws = Nothing
    Else
        Sheets(CStr(r.Value)).Cells(1, 1) = "Total"
        LastRow1 = Sheets(CStr(r.Value)).Cells(Cells.Rows.Count, "A").End(xlUp).Row
        src.Rows(r.Row).Copy Sheets(CStr(r.Value)).Cells(LastRow1 + 1, 1)
        Sheets(CStr(r.Value)).Cells(1, 2) = WorksheetFunction.Sum(Sheets(CStr(r.Value)).Columns(4))
        Set ws = Nothing
    End If
Next r
End Sub

charles kenyon say:
See Templates in Microsoft Word. Read through it even if you don't think you need or understand everything in it.

You use a global template by:
1) putting it into Word's Startup Folder
2) making sure that folder is listed as one of Word's "trusted locations."
3) restarting Word

That's it. All keyboard shortcuts, macros, and AutoText in the template should be available in all of your documents.

You do not "open" a global template, you load it into memory. Putting it in the startup folder does that automatically. If you can't do that or only want to use its resources some times and not have them available at other times, write back.
