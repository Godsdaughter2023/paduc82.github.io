I want all the data in sheet 1  of J1 ,sheet 1 of J2, sheet 1 of J3 .........sheet 1 of J50 to be transfered to sheet J1,sheet J2,sheet J3 .....Sheet J50 respectively of an other work book named RAM.

tasos_k say: Hi,
(1) I assume that you have all Wbks j1,j2,.....j150 in one Folder
(2) The  Workbook RAM SaveAs out of  this Folder

copy and paste the code below in a module in Wbk RAM

Sub All_In_One()
'has been tested
Dim WB, myWB As Workbook
Dim myWS As Worksheet
Dim FolderPath As String
Dim FilesPath As String
Dim v As Variant
Dim r, N As Long
Application.ScreenUpdating = False
On Error Resume Next
Set myWB = ThisWorkbook
FolderPath = "C:\Folder1\Folder2\"  'change Path and Folders' Names 
FilesPath = Dir(FolderPath & "*.xls*")
N = 0
Do Until FilesPath = ""
N = N + 1
ReDim v(N)
v(N) = FilesPath
FilesPath = Dir()
Set WB = Workbooks.Open(FolderPath & v(N))
Set myWS = myWB.Worksheets.Add
myWS.Name = WB.Name
r = WB.Sheets("Sheet1").Cells(Rows.Count, 1).End(xlUp).Row
WB.Sheets("Sheet1").Rows("1:" & r).Copy
myWS.Cells(1, 1).PasteSpecial xlValues
Application.CutCopyMode = False
WB.Close False
Loop
On Error GoTo 0
Application.ScreenUpdating = True
End Sub
