=SUMPRODUCT(--(MOD(ROW(A1:A5)-ROW(A1),2)=0),--(ISNUMBER(A1:A5)),A1:A5,--(ISNUMBER(B1:B5)),B1:B5)

The correct method to handle event DocumentBeforePrint is as follows:

Add a Class Module (Defaut name: Class1), copy the code below:

 

Public WithEvents wdApp As Application

    MsgBox "This is to handle event <DocumentBeforePrint>!", vbInformation, "Event Msg"

Dim c As New Class1

  • While on the dashboard, sign out of your profile (Guide button, then X)
  • Go to Settings -> System -> Storage -> Memory Unit -> Profiles, select your GamerTag, and select Delete.  Select Delete Profile Only.
  • While under Memory Unit, go to Games and Apps, and delete Xbox Music And Video.
  • Still under Memory Unit, go to System Items and delete Marketplace System Data.
  • Go back to Storage, where it shows Memory Unit, Cloud Saved Games, and other storage devices.  Highlight Memory Unit and hit Y (Device Options) and then select Clear System Cache.
  • Restart your console.
  • Select Download Profile (all the way to the right from all of the profiles in the list).
  • Once signed in, try the Video app again and see if that fixes things for you.
