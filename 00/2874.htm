fetaa say:

Hi nightrider1981,

 

No, although using Office 365 ProPlus with shared computer activation enabled doesn’t count against the five computer limit, the user still needs the Office 365 ProPlus license to activate the Office on the shared computer.

 

Microsoft allows a single user to activate Office 365 ProPlus on a reasonable number of shared computers in a given time period. The user gets an error message in the unlikely event the limit is exceeded.

 

I need to explain that:

When a user starts an Office 365 ProPlus program, such as Word, on the shared computer, Office 365 ProPlus contacts the Office Licensing Service on the Internet to obtain a licensing token for the user. If the user is licensed for Office 365 ProPlus, a licensing token is stored on the computer in the user's profile folder, and Office 365 ProPlus is activated. The user can now use Office 365 ProPlus.

 

The licensing token that is stored on the shared computer is valid only for a few days. As the expiration date for the licensing token nears, Office 365 ProPlus automatically attempts to renew the licensing token when the user is logged on to the computer and using Office 365 ProPlus.

 

If the user doesn’t log on to the shared computer for several days, the licensing token can expire. The next time that the user tries to use Office 365 ProPlus, Office 365 ProPlus contacts the Office Licensing Service on the Internet to get a new licensing token. And if you remove the user's Office 365 ProPlus license, no licensing token will be obtained and Office 365 ProPlus won't be activated. Office 365 ProPlus will be in reduced functionality mode.

 

For the details, please refer to  the following article:

https://technet.microsoft.com/en-us/library/dn782860.aspx

Felix

The product key on the bottom of the computer is for Windows, not Office.

The Office installation that was on your computer at purchase is most likely a Trial.  Normally, to continue to use with full functionality an expired Trial, you have to buy a Product Key from Microsoft (http://store.microsoft.com) or purchase a product key from a local trusted store.
