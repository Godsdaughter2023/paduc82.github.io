Look under HKLM\Software\WOW6432Node\Microsoft\Windows\CurrentVersion\Explorer\FolderDescriptions and locate the following keys :

I have had this "unable to rename folders" problem happen several times over the years after applying Windows Updates to my system (Win 8.1). There are fixes published on the Web, usually involving modifying registry entries and sometimes they work, sometimes they don't. Microsoft knows this problem happens, it has been reported to them many times over the years.

My long-winded solution to this problem is to always create a restore point before allowing Windows Update to run. Install the suggested updates one or two at a time, making note of which ones you are choosing to install and then test for the "file or folder does not exist" problem after rebooting and logging in. If it does happen, rewind to the restore point and try some more of the suggested updates. If the problem does not occur after an update, create a new restore point and then try some of the other updates. Step and repeat until only "bad" updates are left uninstalled, and then don't install them.

a. user say:

How can I set up two Time Warner Cable Road Runner email accounts under Outlook Express?

Multiple email accounts

How can I set up two Time Warner Cable Road Runner email accounts under Outlook Express? I'd like to be able to keep emails seperate from each account but access either one at any time.
bruce hagen ~ mvp say:
You can either use two different identities, or if you want to access both accounts in one identity, set up both accounts and use message rules to move incoming messages.
 
First, create a local folder to be used as a second Inbox and name it anything you want.  Use this for one of the accounts, and the default Inbox for the other.
 
Then create a message rule.
 
Where the message is from the specified account.
Move it to the specified folder & Stop processing more rules.
Click on: Specified Account and enter the account name for the second Inbox.
Click on: Specified Folder and select the folder you created.
Name the rule.
 
Move this rule to the top of the rules list and the incoming messages will be separated into their respective folders.
