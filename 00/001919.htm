daniel g. grau say:
The two main things that affect spacing between paragraphs are:

  • Paragraph formatting: Do Command A (Select All). Then choose Format > Paragraph and change the parameters for Spacing—space After, space Before and Line spacing (the latter only if you've separated your paragraphs by using a double Return)

  • Section alignment: In Format > Document, click on the Layout tab and select a Vertical alignment (Top, Center, Bottom, Justified). "Top" is the usual choice. The rest will create gaps until you fill the page with text.


For your second problem (point size), try to select the text, then choose Format > Font and change to "14."
