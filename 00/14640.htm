I have a word doc that is restricted for "filling in the form - Microsoft Community

sjrichmond say:

I have a word doc that is restricted for "filling in the form only". I would like to add a section that will add numbers.

I have a word doc that is restricted for "filling in the form only".  I would like to add a section that will add numbers.
jay freedman say:
First, open the Properties dialog of each of the text form fields where users will enter the numbers to be added. In each dialog:
  • Set the Type box to Number.
  • Optionally, enter a default number such as 0, and a number format (the dropdown contains some typical examples, but you can enter any format you need).
  • Important: Make sure each form field has a unique bookmark name.
  • Check the boxes for "Fill-in enabled" and "Calculate on exit".


Insert another text form field to hold the total. In the Properties dialog for that field:

  • Set the Type box to Calculation.
  • In the Expression box, type the equation to sum the other fields, using their bookmark names; for example,  = Firstnum + Secondnum
  • Optionally, enter a number format.
  • Give the field a unique bookmark name.
  • If this field will be used as a subtotal in the computation of a grand total, check the box for "Calculate on exit"; otherwise leave it unchecked.

Protect the document. The field for the total will not be selectable, but it will automatically show the result of the equation in its Expression field whenever you change one of the data entry fields and then tab to another field.

jetta48 say:

Step-by-step:

1. Go to Start->All Programs->Internet Explorer (not the one that says 64-Bit)

2. Go to websitewww.adobe.com

3. Click on 'Get Adobe Flash Player'.

4. Follow prompts to download & install Flash Player.

All you need to remember now is that you must use the version of Internet Explorer mentioned in Step 1 above if you want to use Flash Player. This is what I do all the time.

There is a 64-Bit version of Flash out for beta testing, but if you just want reliable day-to-day usage do the above.
